'''二次安检'''（{{lang-en|Secondary Security Screening Selection}}，缩写'''SSSS'''）是美国[[运输安全管理局|运输安全管理局]]（TSA）对部分航空旅客要求的额外[[安全检查|安全检查]]。<ref name = "civ">[http://news.carnoc.com/list/283/283149.html 美登机牌上“SSSS”字样有何深意？] - [http://www.carnoc.com/ 民航资源网]</ref>

== 概述 ==
二次安检针对任何入境美国的航班和美国国内航班的乘客。<ref name = "net">[https://c.m.163.com/news/a/CUVBJ4BQ05149B9C.html?fromhistory=1 去美国要注意！登机牌上如果出现“SSSS”要小心] - [[网易|网易]]</ref>如果一名乘客符合TSA二次安检的要求，航空公司将禁止该名乘客进行在线值机或自助值机，而只能进行柜台值机。<ref name = "sina">[http://news.sina.com.cn/o/2017-09-25/doc-ifymenmt6805323.shtml 登机牌上如果出现这串符号，你可能摊上大事了…] - [[新浪网|新浪网]]</ref><ref name = "jiemian">[http://www.jiemian.com/article/1647539.html 登机牌上的神秘代码：这几个字母一出现，八成要被关小黑屋] - [http://www.jiemian.com/ 界面网]</ref>被选中的乘客的[[登机牌|登机牌]]上将会出现'''SSSS'''的字样（有时会是*S*的字样，美[[西南航空|西南航空]]则是一个棋盘图案<ref name=southwest-checkerboard-freetotravel>{{cite web|title=Gilmore v. Gonzales|url=http://freetotravel.org/cid.html|website=FreeToTravel.org|publisher=UNITED STATES DISTRICT COURT   NORTHERN DISTRICT OF CALIFORNIA|accessdate=2015-10-24|date=2002-07-18|quote=Southwest... clerk tore up the boarding pass, printed out another one with a checkerboard pattern on it... you become selected for secondary screening...|archive-url=https://web.archive.org/web/20160304131053/http://freetotravel.org/cid.html|archive-date=2016-03-04|dead-url=yes}}</ref>），以提示安检人员对这名乘客额外注意。<ref name = "net"/>乘客除了基本的X光扫描检查以外，还需要接受问询、<ref name = "jiemian"/>人工行李检查、爆炸物痕量探测甚至[[搜身|搜身]]。<ref name = "civ"/>这些检查过程至少会花掉30分钟。<ref name = "net"/>通过检查的乘客会被安检人员在机票上打上特殊标识予以放行，反之或拒绝进行二次安检者<ref name = "net"/>则将被拒绝登机。<ref name = "jiemian"/>

== 检查对象 ==
虽然TSA宣称二次安检的对象是随机抽取的<ref>[http://www.sohu.com/a/160496697_128409 东航北京地服部对赴美航班进行随机性二次安检] - [[搜狐网|搜狐网]]</ref>，但一般认为符合如下特点的乘客很容易受到二次安检：<ref name = "sina"/>
*高风险国家出身的乘客
*在飞行当日使用现金购票的乘客
*仅购买了单程机票或当日往返机票的乘客
*和恐怖分子重名，或位列恐怖分子名单上的乘客
*经常前往高风险国家的乘客
*有不良入境记录<ref name = "jiemian"/>
*有携带违禁品登机的记录<ref name = "jiemian"/>
*证件问题<ref name = "jiemian"/>

此外，TSA宣称手中有一个必须接受二次安检人员的名单。名单的内容并未被公布且内容一直在更新。<ref name="ACLU">{{cite web|title=Frequently Asked Questions About the No-Fly List|url=https://aclu-wa.org/news/frequently-asked-questions-about-no-fly-list|website=aclu-wa.org|author=ACLU|location=Washington State|accessdate=2015-08-07|deadurl=yes|archiveurl=https://web.archive.org/web/20140212052314/https://aclu-wa.org/news/frequently-asked-questions-about-no-fly-list|archivedate=2014-02-12}}</ref>不过，如果怀疑自己身处这个名单的话，可以通过申诉予以解除。<ref name = "sina"/>

== 社会反映 ==
二次安检是从[[911事件|911事件]]以后开始实施的。目的是从乘客中筛选出潜在的[[恐怖分子|恐怖分子]]。然而自[[特朗普|特朗普]]上台以后，二次安检的出现频率明显增高，<ref name = "net"/>自2017年以来，每天会有超过32万名乘客受到影响。<ref name = "chn">[http://news.china.com/shendu/13000808/20171026/31605731.html 赴美航班新规：登机前二次面谈 安检或达三小时] - [http://www.china.com/ 中华网]</ref>因此这个政策引致很多乘客的不满。<ref name = "sina"/>此外，尽管TSA宣称没有任何[[未成年人|未成年人]]身处二次安检名单内，至今仍有很多未成年人因为和名单上的人重名而需要接受二次安检。<ref>[http://pic.people.com.cn/GB/42590/4008861.html 因和恐怖分子重名 美国四岁男孩上了“禁飞名单”] - [http://www.people.com.cn/ 人民网]</ref><ref name= AOL>[http://news.travel.aol.com/2010/04/30/are-these-kids-terrorists/ Are These Kids Terrorists?]</ref><ref name=DALLAS>[https://www.dallasnews.com/news/transportation/2017/03/27/mom-says-tsa-agents-dfw-airport-traumatized-son-horrifying-security-check Mom says TSA agents DFW airport traumatized son horrifying security check] - [https://www.dallasnews.com Dallas News]</ref>

[[美國公民自由聯盟|美國公民自由聯盟]]认为二次安检是[[种族主义|种族主义]]行为。<ref>{{cite web|url=http://avstop.com/news_july_2010/aclu_files_lawsuit_challenging_the_no_fly_list.htm|title=ACLU Files Lawsuit Challenging The 'No Fly List'|date=July 2010|website=avstop.com|access-date=2018-04-24|archive-url=https://web.archive.org/web/20121017072127/http://avstop.com/news_july_2010/aclu_files_lawsuit_challenging_the_no_fly_list.htm|archive-date=2012-10-17|dead-url=yes}}</ref>也有一些社会活动人士相信美国政府将一些[[异见人士|异见人士]]放在二次安检名单之上，以故意为他们的生活制造不便。<ref>{{cite web| url=http://warisacrime.org/node/26399| author=Sherwood Ross| title=BUSH RESTRICTING TRAVEL RIGHTS OF OVER 100,000 U.S. CITIZENS| publisher=WarIsACrime.org| accessdate=2016-07-06| deadurl=yes| archiveurl=https://web.archive.org/web/20160814174216/http://warisacrime.org/node/26399| archivedate=2016-08-14}}</ref><ref>{{cite book| author=Naomi Wolf| title=[[The_End_of_America:_Letter_of_Warning_to_a_Young_Patriot|The End of America: Letter of Warning to a Young Patriot]]| publisher=Chelsea Green Publishing| date=September 2007| pages=93–94}}</ref><ref>{{cite web| url=http://inthesetimes.com/issue/27/02/feature3.shtml| author=Dave Lindorff| title=The No-Fly List: Is a federal agency systematically harassing travelers for their political beliefs?| publisher=In These Times| date=2002-11-22}}</ref>

由于二次安检可能导致登机时间被严重拉长，一些航空公司建议乘客至少提前三小时进行安检，以避免错过航班。<ref name = "chn"/>
