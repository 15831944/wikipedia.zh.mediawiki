'''二郎庙'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[永定门|永定门]]外，是[[道教|道教]]宫观，现已无存。

==简介==
二郎庙位于今[[马家堡路|马家堡路]]东永建里居民区一带。《北京寺庙历史资料》载：二郎庙（道庙）坐落南郊永定门外二郎庙村甲三号，建于民国二十五年（1936年）<ref name=gounian/>。

1926年4月26日凌晨四时，[[邵飘萍|邵飘萍]]被押往[[北京天桥|天桥]][[先农坛|先农坛]]二道坛门刑场枪决。[[汤修慧|汤修慧]]得知丈夫邵飘萍被害的消息后，四处打听其遗体的下落，后在永定门外的二郎庙挖出了邵飘萍尸首。汤修慧将邵飘萍安葬在[[广安门|广安门]]外[[天宁寺_(北京)|天宁寺]]旁荒地<ref name=gounian>{{Cite web|url = http://www.bjd.com.cn/pw/xq/201803/05/t20180305_11081475_1.html|title = 狗年新潮流，在老北京逛逛“狗胡同”和“狗庙”|author = |date = 2018-03-05|publisher = 京报网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180306202712/http://www.bjd.com.cn/pw/xq/201803/05/t20180305_11081475_1.html|archive-date = 2018-03-06|dead-url = yes}}</ref>。

1949年时，如今东永建里居民区、永建里居民区所在地及附近有两个村子——二郎庙东村、二郎庙西村。西村的二郎庙内有[[二郎神|二郎神]]的神像。每逢初一、十五开庙，香火旺盛。此庙早废。二郎庙西村原址现已成为马家堡路的中段。二郎庙东村在[[中华人民共和国|中华人民共和国]]成立后形成居民区，1965年将此处划为永建里和东永建里<ref>{{Cite web |url = http://www.bjdclib.com/subdb/laneculture/lane/200907/t20090717_10724.html |title = 马家堡路 |author =  |date =2009-07-17  |publisher =东华流韵  |language =  |accessdate =  |quote =  }}</ref>。
