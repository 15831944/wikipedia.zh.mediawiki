'''二氟化氙'''（[[化学式|化学式]]：[[氙|Xe]][[氟|F]]<sub>2</sub>）是一种稳定的[[氙|氙]]化合物，可长期处放在[[镍|镍]]制容器中或干燥的[[石英|石英]]和[[玻璃|玻璃]]器皿中而不发生变化。同其他氟化氙相比较，二氟化氙是一种温和的[[氧化剂|氧化剂]]和氟化剂，可生成多种氙化合物。

== 物理性质 ==
二氟化氙是一种无色固体，在室温下容易升华而形成大的透明[[晶体|晶体]]。二氟化氙蒸气是无色的，具有令人发呕的恶臭。

二氟化氙易溶于无水[[氟化氢|氟化氢]]、NOF·3HF和[[五氟化碘|五氟化碘]]中，并不发生[[电离|电离]]。固体二氟化氙晶体属于[[体心四方|体心四方]][[晶系|晶系]]，a=431.5pm，c=699.0pm，键长约为200pm。二氟化氙气态时是一种直线型的分子，对称群为D<sub>∞h</sub>，键长比固态略短，为197.7pm，其中的Xe-F键具有一定[[离子键|离子键]]的性质，这可以用[[三中心四电子键|三中心四电子键]]来解释。

== 化学性质 ==
=== 基本反应 ===
*与[[氢气|氢气]]反应定量地放出[[氙|氙]]和生成[[氟化氢|氟化氢]]，通常用这个反应来分析二氟化氙。
:<math>\rm \ XeF_2 + H_2 \xrightarrow{673K} Xe + 2HF</math>

*与[[氟气|氟气]]反应：XeF<sub>2</sub>可进一步与单质氟反应生成[[四氟化氙|XeF<sub>4</sub>]]和[[六氟化氙|XeF<sub>6</sub>]]。
:<math>\rm \ XeF_2 + F_2 \xrightarrow{473K} XeF_4</math>
:<math>\rm \ XeF_4 + F_2 \xrightarrow{673K} XeF_6</math>
:[[二氟化二氧|O<sub>2</sub>F<sub>2</sub>]]也可以将XeF<sub>2</sub>进一步氟化成为XeF<sub>4</sub>和XeF<sub>6</sub>。

*与[[水|水]]反应，中间产物可能有XeO[[自由基|自由基]]、[[过氧化氢|H<sub>2</sub>O<sub>2</sub>]]、XeF<sup>+</sup>等。
:<math>\rm \ 2XeF_2 + 2H_2O \rightarrow 2Xe + O_2 + 4HF</math>

*与[[碱|碱]]反应：
:<math>\rm \ 2XeF_2 + 4OH^- \rightarrow 2Xe + O_2 + 4F^- + 2H_2O</math>

*与[[双氧水|双氧水]]反应，中间产物有自由基O、OH、XeF，以及[[超氧酸|HO<sub>2</sub>]]。
:<math>\rm \ 2XeF_2 + H_2O_2 \rightarrow Xe + O_2 + 2HF</math>

=== 同无机物的反应 ===
==== 不生成氙化合物 ====
二氟化氙是一种具有[[氧化性|氧化性]]的氟化剂，往往会同时发生氧化和氟化反应。二氟化氙被[[还原|还原]]时放出氙，但不生成氙化合物。作为氟化剂来说，无论从[[热力学|热力学]]还是[[动力学|动力学]]的观点，二氟化氙的氟化本领不如单质氟或某些氟化物，显示出有一定的钝性：

ClF<sub>5</sub> > BrF<sub>5</sub> > IF<sub>7</sub> > ClF > BrF<sub>3</sub> > '''XeF<sub>2</sub>''' > IF<sub>5</sub> > BrF

也正因为如此，用二氟化氙作氟化剂比较安全和操作方便。

此外，二氟化氙在反应后，氙一般不会进入到产物中，释出的氙可以循环使用，所以它是一种很好的[[氟化剂|氟化剂]]。它可以氟化许多[[无机化合物|无机化合物]]，并且是一种选择性很好的无机物氟化剂。二氟化氙只有在[[气相|气相]]和非水溶液中才具有氟化性质，在水溶液中它只能起氧化剂的作用。

反应通式为：
:XeF<sub>2</sub> + HA → FXeA + HF
:FXeA + HA → XeA<sub>2</sub> + HF
其中HA是反应物，A是[[电负性|电负性]]较强的基团。

这些氙化合物的稳定性都不如二氟化氙，并且同水反应生成氙、[[氧|氧]]和相应的[[酸|酸]]。只有HOTeF<sub>5</sub>和HOSeF<sub>5</sub>形成的化合物较稳定，在100℃以上才开始分解。Xe(OTeF<sub>5</sub>)<sub>2</sub>与水反应很慢，但在碱性溶液反应剧烈。值得一提的是，[[核磁共振|核磁共振]]和[[拉曼光谱|拉曼光谱]]证实，FXeN(SO<sub>2</sub>F)<sub>2</sub>含有Xe-N键。FXeOSO<sub>2</sub>F一端与氟原子相连，另一端与氟磺酸基团中的氧相连。

=== 同有机物的反应 ===
[[有机化合物|有机化合物]]的氟化是近代化学的重要课题。用单质氟作氟化剂，由于技术和安全方面的困难，受到很大限制，反应剧烈不易控制，可引起有机化合物的断链，或者一直进行到生成全氟化合物。利用其他氟化剂，如[[氟化氢|氟化氢]]、某些金属氟化物、卤素氟化物，虽然也可以同有机化合物发生氟化取代反应，但反应物的产率一般都较低，选择性也较差。[[四氟化氙|四氟化氙]]和[[六氟化氙|六氟化氙]]同水易生成具有爆炸性的[[三氧化氙|三氧化氙]]，而且六氟化氙的氟化-氧化性质太强，可引起有机物的断链。唯独二氟化氙的氟化性比较温和，又比较稳定，操作比较安全，而且反应物的产率和氟化选择性也都比较好。

==== 非环化合物 ====
用二氟化氙作氟化剂可以使[[烯烃|烯烃]]的氟化反应得到一定的改善。气相氟化反应在室温下按下式进行：
:[[File:XeF2&C2H4.png|500px]]

二氟化氙能溶于[[乙腈|乙腈]]中，在室温下只是单纯溶解，不发生化学反应；当升高温度到乙腈的[[沸点|沸点]]时，二氟化氙就对乙腈发生氟化。二氟化氙能溶于[[五氟化溴|五氟化溴]]、[[氟化氢|氟化氢]]、[[五氟化碘|五氟化碘]]、[[二氧化硫|二氧化硫]]等之中，起初也不发生化学反应，但只要有痕量的[[路易斯酸|路易斯酸]]存在时，二氟化氙就会同这些溶剂发生氟化-氧化反应。对于路易斯酸在这里所起的作用，有人提出如下解释：二氟化氙溶解在这些溶剂里呈分子状态，没有发生电离；当加入路易斯酸时，二氟化氙电离而产生XeF<sup>+</sup>：
::XeF<sub>2</sub> + A(Lewis酸) → XeF<sup>+</sup> + AF<sup>−</sup>
::2XeF<sub>2</sub> + A(Lewis酸) → Xe<sub>2</sub>F<sub>3</sub><sup>+</sup> + AF<sup>−</sup>
XeF<sup>+</sup>和Xe<sub>2</sub>F<sub>3</sub><sup>+</sup>的反应能力比二氟化氙强得多，所以在二氟化氙的乙腈溶液中只要有痕量的路易斯酸（如[[三氟化溴|三氟化溴]]、[[五氟化砷|五氟化砷]]、[[五氟化磷|五氟化磷]]或[[四氟化钛|四氟化钛]]）存在时，二氟化氙就会同乙腈发生反应而放出氙。

氟代[[硅烷|硅烷]]是有实际意义的化合物。二氟化氙对硅烷（以乙腈为溶剂）的氟化有以下规律：含Si-Cl键的硅烷同二氟化氙反应时只需几分钟，就能完成氟同[[氯|氯]]之间的交换反应，生成氟代硅烷；但二氟化氙不同Si-C，Si-O键反应。
::2(CH<sub>3</sub>)<sub>3</sub>SiCl + XeF<sub>2</sub> → 2(CH<sub>3</sub>)<sub>3</sub>SiF (100%) + Cl<sub>2</sub> + Xe
::2(C<sub>2</sub>H<sub>5</sub>)<sub>3</sub>SiCl + XeF<sub>2</sub> → 2(C<sub>2</sub>H<sub>5</sub>)<sub>3</sub>SiF (100%) + Cl<sub>2</sub> + Xe
::(CH<sub>3</sub>)<sub>2</sub>SiCl<sub>2</sub> + XeF<sub>2</sub> → (CH<sub>3</sub>)<sub>2</sub>SiF<sub>2</sub> (100%) + Cl<sub>2</sub> + Xe
二氟化氙对仅含Si-H键的硅烷不能直接发生氟化反应，但如果硅烷中同时含有Si-Cl键时，则能很快发生氟化反应。[[反应机理|反应机理]]是二氟化氙首先同Si-Cl反应，释放出的氯随即将Si-H键转化为Si-Cl键。这样，二氟化氙就能将它进一步氟化成为二氟代硅烷了。

==== 碳环化合物 ====
二氟化氙对碳环化合物具有优良的选择性氟化性能，比传统的氟化过程简便。

;苯的氟化
二氟化氙对[[苯|苯]]直接氟化制取[[氟苯|氟苯]]有良好的效果，以[[四氯化碳|四氯化碳]]为溶剂，产率为68%。反应机理为[[自由基|自由基]]反应。除氟代苯外，还生成了少量的[[联苯|联苯]]。苯的同系物[[甲苯|甲苯]]、[[氯苯|氯苯]]等同二氟化氙反应，也生成少量的联苯。体系的电子顺磁共振谱表明有对位R(C<sub>6</sub>H<sub>4</sub>)<sub>2</sub>R<sup>+</sup> （R=CH<sub>3</sub>，Cl，F，F-C<sub>6</sub>H<sub>4</sub>-C<sub>6</sub>H<sub>4</sub>-）自由基存在。二氟化氙可将[[电极电势|电极电势]]较高的化合物变为[[阳离子|阳离子]]自由基，起到R(C<sub>6</sub>H<sub>4</sub>)<sub>2</sub>R<sup>+</sup>自由基生成剂的作用，有很重要的实际意义。

;含苯氧基的化合物
:含[[苯氧基|苯氧基]]（C<sub>7</sub>H<sub>7</sub>O）的氟化合物是有价值的[[药物|药物]]，用传统的氟化方法合成流程较长。利用二氟化氙的选择性氟化性能，直接对苯环进行氟化，只需一步反应就可完成，例如：
::C<sub>6</sub>H<sub>5</sub>OCH<sub>3</sub> + XeF<sub>2</sub> -(CH<sub>2</sub>Cl<sub>2</sub>)→ C<sub>6</sub>H<sub>4</sub>(OCH<sub>3</sub>)F (71%)

;稠环烃
:二氟化氙可用于合成很多[[稠环烃|稠环烃]]的氟化物，如1,4-二氟代[[萘|萘]]以及与[[蒽|蒽]]、[[芘|芘]]反应的产物。

==== [[杂环化合物|杂环化合物]] ====
[[5-氟尿嘧啶|5-氟代尿嘧啶]]（5-Fu, Fluorouracil）是一种有价值的药物，可用来抗癌。在生物化学方面，它是同[[核酸|核酸]]有关的一类化合物。在室温下[[尿嘧啶|尿嘧啶]]可以被二氟化氙一步直接氟化成为5-氟代尿嘧啶。

== 合成方法 ==
合成二氟化氙的方法有：
*单质合成：条件可以是加热、光合成、[[放电|放电]]、[[电子束|电子束]]照射、γ-[[辐射|辐射]]或以[[氧|氧]]或[[空气|空气]]为载气反应，将装有单质混合物的透明容器于阳光下曝晒便可合成。整个合成过程必须保持氙过量，否则会有过多的副产物[[四氟化氙|四氟化氙]]影响纯度：
:Xe + F<sub>2</sub> → XeF<sub>2</sub>

*其他[[氟化物|氟化物]]氟化，可用的氟化试剂有[[四氟化碳|四氟化碳]]、[[四氟化硅|四氟化硅]]、[[五氟化铬|五氟化铬]]、[[二氟二氯甲烷|二氟二氯甲烷]]、[[二氟化二氧|二氟化二氧]]、[[氟氧基三氟甲烷|氟氧基三氟甲烷]](CF<sub>3</sub>OF)、[[二氟化二氧|二氟化二氧]]、[[二氟化二氮|二氟化二氮]]及[[氟磺酸氟|氟磺酸氟]]等。
:Xe + 2CF<sub>4</sub> -放电→ XeF<sub>2</sub> + C<sub>2</sub>F<sub>6</sub>
:Xe + 2SiF<sub>4</sub> -放电→ XeF<sub>2</sub> + Si<sub>2</sub>F<sub>6</sub>
:Xe + CrF<sub>5</sub> → XeF<sub>2</sub> + CrF<sub>3</sub>
:Xe + CF<sub>2</sub>Cl<sub>2</sub> → XeF<sub>2</sub> （通过放电管）
:Xe + CF<sub>3</sub>OF → XeF<sub>2</sub> （250atm，500K）
:Xe + FSO<sub>3</sub>F → XeF<sub>2</sub>  （150atm，450K）
:Xe + O<sub>2</sub>F<sub>2</sub> → XeF<sub>2</sub> + O<sub>2</sub>
:N<sub>2</sub>F<sub>2</sub> + Xe → Xe的氟化物的混合物（80-150°C）

*[[四氟化氙|四氟化氙]]与[[氙|氙]]发生[[归中反应|归中反应]]：
:XeF<sub>4</sub> + Xe -(高压)→ 2XeF<sub>2</sub>
