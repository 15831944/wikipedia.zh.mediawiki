'''二氧化鉛'''或'''過氧化鉛'''，化學式[[鉛|Pb]][[氧|O]]{{sub|2}}，常溫時為棕色結晶或粉末，几乎不溶于水<ref name=cec>{{cite book|url=http://books.google.com/?id=Owuv-c9L_IMC&pg=PA590|page=590|title=Concise encyclopedia chemistry|author=Mary Eagleson|publisher=Walter de Gruyter|year=1994|isbn=3-11-011451-8}}</ref>，有强氧化性。
==制备==
二氧化鉛可由[[四氧化三鉛|四氧化三鉛]]與[[硝酸|硝酸]]作用而得。反應式：
:Pb<sub>3</sub>O<sub>4</sub> + 4HNO<sub>3</sub> → PbO<sub>2</sub> + 2Pb(NO<sub>3</sub>)<sub>2</sub> + 2H<sub>2</sub>O 
==化学性质==
===不稳定性===
二氧化铅受热分解：
::PbO<sub>2</sub>  → Pb<sub>12</sub>O<sub>19</sub> → Pb<sub>12</sub>O<sub>17</sub> → Pb<sub>3</sub>O<sub>4</sub> → PbO
===强氧化性===
由于[[6s2惰性电子对效应|6s2惰性电子对效应]]，二氧化铅具有很强的[[氧化性|氧化性]]，在常温即可将[[盐酸|盐酸]]氧化成[[氯气|氯气]]：
:PbO<sub>2</sub> + 4 HCl → PbCl<sub>2</sub> + 2 H<sub>2</sub>O + Cl<sub>2</sub><ref name=green>{{Greenwood&Earnshaw2nd|page=386}}</ref> 
也可以将浓[[硫酸|硫酸]]氧化，放出[[氧气|氧气]]：
:2 PbO<sub>2</sub> + 2 H<sub>2</sub>SO<sub>4</sub> → 2 PbSO<sub>4</sub> + 2 H<sub>2</sub>O + O<sub>2</sub><ref name=green />
二氧化铅与[[碳|碳]]共热，即可剧烈反应，产生火光和金属[[铅|铅]]：
:2 PbO<sub>2</sub> + C → 2 PbO + CO<sub>2</sub>
:PbO<sub>2</sub> + C → Pb + CO<sub>2</sub>
此外，在酸性条件下，铅单质与二氧化铅可以发生[[归中反应|归中反应]]，这也是[[铅酸蓄电池|铅酸蓄电池]]放电时的工作原理

===和碱反应===
与强碱溶液反应，得铅酸盐：
:PbO<sub>2</sub> + 2 NaOH + 2 H<sub>2</sub>O → Na<sub>2</sub>[Pb(OH)<sub>6</sub>]<ref name=cec />
和熔融的强碱反应，可以得到M<sub>4</sub>[PbO<sub>4</sub>]。

==用途==
二氧化鉛可製造[[鉛蓄電池|鉛蓄電池]]的[[極板|極板]]。加熱產生[[氧|氧氣]]。与[[硫|硫]]、[[赤磷|赤磷]]等可燃物混合研磨则发火，可用作[[火柴|火柴]]的原料。

二氧化铅也常用于实验室制备其他的高价化合物<ref>{{cite book|url=http://books.google.com/?id=PpTi_JAx7PgC&pg=PA387|page=387|title=A Text Book of Inorganic Chemistry|author=Anil Kumar De|publisher=New Age International|year=2007|isbn=81-224-1384-6}}</ref>：

:2 MnSO<sub>4</sub> + 5 PbO<sub>2</sub> + 6 HNO<sub>3</sub> → 2 HMnO<sub>4</sub> + 2 PbSO<sub>4</sub> + 3 Pb(NO<sub>3</sub>)<sub>2</sub> + 2 H<sub>2</sub>O

:2 Cr(OH)<sub>3</sub> + 10 KOH + 3 PbO<sub>2</sub> → 2 K<sub>2</sub>CrO<sub>4</sub> + 3 K<sub>2</sub>PbO<sub>2</sub> + 8 H<sub>2</sub>O
