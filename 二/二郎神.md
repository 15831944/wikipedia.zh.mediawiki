{{道教}}
'''二郎神'''，又称：'''[[川主|川主]]'''、'''顯聖二郎真君'''、'''昭惠靈顯王'''、'''灌口二郎'''、'''二郎真君'''、'''灌江神'''、'''赤城王'''、'''清源妙道真君'''。是中国[[民间|民间]]和[[道教|道教]]的神祇人物。民间多认为他是一位与[[水利|水利]]、[[農耕|農耕]]、防止[[水災|水災]]、[[旱灾|旱灾]]有关的神，甚至是[[水神|水神]]。一般認為二郎神居於[[四川|四川]]灌县（今[[都江堰市|都江堰市]]）。

二郎神信仰起源于[[唐朝|唐朝]]之前四川地区对于秦蜀郡太守[[李冰|李冰]]的信仰崇拜<ref name="明清以来四川地区川主信仰的时空分布研究">{{cite  book|author=付玉强|title=明清以来四川地区川主信仰的时空分布研究|pages=20-21|date=2011年5月}}</ref>，并在[[北宋|北宋]]时期便流行于全中国。然而[[元朝|元朝]]之后，受到戏曲话本小说的大肆渲染，中国全国范围内的二郎神信仰中的治水成分受到削弱。因而仍然基于治水传说的巴蜀地区二郎神信仰产生了分化，并在明清发展成为了[[川主|川主信仰]]，川主（二郎神）被认为是四川和重庆的鄉土神。

== 来历 ==
[[File:二郎像.jpg|thumb]]
[[张政烺|张政烺]]認为二郎神是将几种民间[[俗神|俗神]]，如[[李冰|李冰]]之次子、[[赵昱|赵昱]]、[[张仙|张仙]]、[[楊戩_(二郎神)|杨戬]]混合而成的，[[馮夢龍|馮夢龍]]《[[醒世恒言|醒世恒言]]》第十三卷《勘皮靴单证二郎神》说[[北宋|北宋]][[汴京|汴京]]已有“清源真君二郎神”庙宇，《宋史》载：“[[宋徽宗|宋徽宗]][[政和|政和]]七年（1117年）诏修神保观，俗所云二郎神者。[[京师|京师]]云，倾城男女，负土以献，不知何神。”

二郎神的來歷有很多種說法：
*[[蚕丛|蚕丛]]後裔：道书《灌江备考》上说“二郎为[[蚕丛|蚕丛]]之后，故其目纵。”

*[[川主|李二郎]]：[[秦朝|秦朝]][[李冰|李冰]]的第二个儿子，曾协助父亲斩杀[[蛟龙|蛟龙]]，平定水灾，被称为“灌口二郎”并设祠祭祀。[[朱熹|朱熹]]《朱子语类》说：“[[梓潼神|梓潼]]与灌口二郎两个神，几乎割据了[[四川|两川]]。”该书又说：“蜀中灌口二郎神，当时是李冰因开离堆有功立庙，今来现许多灵怪，乃是他第二儿子出来。”

*[[楊戩|楊二郎]]：《[[西遊記|西遊記]]》中，楊二郎，名為楊戩，是[[玉皇大帝|玉皇大帝]]的外甥，法力高強，有[[七十二變|七十二變]]，曾幫助[[天庭|天庭]]捉拿大鬧天宮的[[孫悟空|孫悟空]]。

*[[鄧遐|鄧二郎]]：[[晋朝|晋朝]][[襄阳|襄阳]]太守鄧遐，因曾經入水斬殺[[沔水|沔水]]蛟龍，被人尊称为“'''二郎神'''”。

*[[赵昱|趙二郎]]：[[隋朝|隋朝]][[嘉州|嘉州]]太守[[赵昱|赵昱]]（即[[青城山|青城山]]道士[[赵昱|赵昱]]）也因为斬殺蛟龍，被称为灌口二郎神，被[[宋真宗|宋真宗]]封为清源妙道真君。

*[[独健|独健]]：[[唐朝|唐朝]]传说，他是[[四大天王|四大天王]]中之北方多闻天王[[毗沙门|毗沙门]]的第二个儿子，名[[独健|独健]]，在[[安史之乱|安史之乱]]中曾经率天兵救护[[唐明皇|唐明皇]]。

*[[孟昶|孟昶]]：[[後蜀|後蜀]]後主孟昶投降[[宋朝|宋朝]]之後過世，被其妻[[花蕊夫人|花蕊夫人]]或懷念他的百姓以'''二郎神'''，或是[[張仙|張仙]]等其他種種名義，暗中奉祀。

== 特点 ==
艺术描写中的二郎神居住住灌江口，与梅山六兄弟为伴，掌管一千二百草头神，对于天庭“听调不听宣”，使用[[三尖两刃刀|三尖两刃刀]]为兵器，额头中间多生了一只[[第三眼|慧眼]]，身边有神犬[[哮天犬|哮天犬]]（[[嘯天犬|嘯天犬]]）跟随。

二郎神向来是正义的象征，对黎民百姓，二郎神是斩妖除魔，正直清廉的天界战神，为黎民百姓的正义公道而战，在有些电视剧里面，更被突显为执法公正的司法天神，做任何事都是为三界众生之幸福和安康。

他使用的武器是威力无比的上古神器‘三尖两刃神枪’，武功绝伦，挥舞三尖两刃神枪时出神入化，盖世无双，可以摧毁山脉，崩天裂地。

由於是人神混血，因此拥有天生的无穷神力，法术精湛，拥有滔天的强大法力，架起狂风便能不到眨眼功夫间飞跃过万里迢迢的东洋大海。他的七十二变，给予他了点石成金，呼风唤雨，永生不死，幻化猛兽的本领。

练就了九转玄功，使他拥有跟金刚不坏身相似的无可匹敌的绝顶防御力，以及极为快速的疗伤能力。练就九转玄功，二郎神等于修炼了无上金身，有超越天地洪荒的战斗实力和无边防御力。二郎神座下也有神兽哮天犬和扑天雕，都专门降妖除魔，凶猛万分。

擁有召唤天界神雷的本领，他会对邪魔歪道进行“天罚”，召唤天界的神圣雷霆彻底的轰炸消灭他们。

二郎神的第三只眼睛名叫“天眼”。这天眼拥有许多功能，一则可以辨明事物本源，无论神仙幻化或妖魔迷阵都一看便知本相。二则，天眼也有至极强大的攻击毁灭之力，能够发出彻底消灭大批敌人的强猛光束，使敌手形神俱灭，烟消云散。天眼实力足以碎波劈浪，毁灭天地，就连拥有金刚不坏身的神仙妖魔，也必会修为尽失，非残即伤。

九转玄功：阐教的护法神功,杨戬炼成了九转玄功后,拥有无比超绝坚强的肉身,有金刚不坏的圣体,无可匹敌的无边防御力。闻仲的蛟龙双鞭,余化的化血神刀都对杨戬完全无济于事。

三尖两刃神锋枪：杨戬的战斗神器,也称三尖两刃刀,三尖两刃枪,锋锐绝伦,削铁如泥。杨戬挥动三尖两刃神锋枪的武功盖世无双,出神入化,在百万妖魔军中如入无人之境,势不可当,威力大得足以崩天裂地,摧毁山脉。

天眼:杨戬的通天神目-天眼,拥有辨明事物本源,辨别真假善恶的神通,而且也可以释放出超绝强大的灭绝光柱,拥有毁灭天地的巨大力量。

== 二郎神信仰 ==

=== 福建 ===
泉州府城内有[[庆同庵|庆同庵]]，庵内供奉清源妙道真君，俗称二郎庙。二郎庙所在的社区也因之而命名为二郎巷、二郎后街。
[[廈門|廈門]][[集美區|集美區]][[灌口鎮|灌口鎮]][[鳳山祖廟|鳳山祖廟]]，建於[[明|明]][[天啟|天啟]]和[[崇禎|崇禎]]年間，祀奉「李府清源真君二郎神」。

=== 台灣 ===
在[[台灣|台灣]]，許多道觀、寺廟供奉二郎神，還有一些專門祀奉二郎神的廟宇，如[[新竹縣|新竹縣]][[新埔鎮|新埔鎮]]二郎神廟、[[彰化縣|彰化縣]][[埤頭鄉|埤頭鄉]]救世宮、[[台南市|台南市]][[佳里區|佳里區]]番仔寮應元宮 ，其中[[苗栗縣|苗栗縣]][[後龍鎮|後龍鎮]]的迴天宮
因[[親戚不計較|親戚不計較]]曾經在此取景，而一度聲名大噪，甚至有觀眾誤以為[[迴天宮|迴天宮]]就在主角居住的[[屏東|屏東]][[水底寮|水底寮]]。

== 與祆教的關聯 ==
《[[十國春秋|十國春秋]]》記載，[[五代|五代]]時，[[前蜀|前蜀]]皇帝[[王衍_(前蜀)|王衍]]「披金甲，冠珠帽，持弓矢而行。百姓望之，謂如灌口祆神」。唐末五代時[[灌口街道|灌口]]便有披甲持弓的神像，就是由這條史料獲知。《說文新附》釋「祆」，稱：「祆，胡神也。從示天聲。火千切。」[[中山大學|中山大學]]教授[[黎國韜|黎國韜]]考證此「祆神」即[[波斯|波斯]]的[[火祆教|火祆教]]神祇，他在《二郎神之祆教來源》一文中認為，二郎神之形象：三眼，手持三尖兩刃槍，牽一哮天犬，可以變化成三頭六臂。此形象與祆教維施帕卡神（{{transl|sog|Weshparkar}}）形像正相一致。維施帕卡的形像是三頭六臂，身披甲裝，手執山型叉，臂上畫有尖齒犬頭，與文學描寫中二郎神形像極為相似，這是祆神與灌口二郎神有關之一證。故推測二郎神之原型應是隋唐時從[[西域|西域]]傳入[[中原|中原]]的波斯神靈，其後才逐漸本土化。且[[五代十國|十國]]前後[[巴蜀民系|蜀人]]信奉火祆教，其形像又與元明淸以來小說、戲曲中之二郎神相似，表明祆神形像對二郎神形像之形成有很大影響。<ref>{{cite web |url=http://lib.silkroads.org.cn/Detail?article=19416&dc=QIKAN&py=ZJYJ |title=二郎神之祆教來源——兼論二郎神何以成為戲神 |author=黎國韜 |date=2004 |website=lib.silkroads.org.cn |access-date=2018年8月3日}}</ref><ref>{{cite web |url=http://xnmzxbsk.ijournal.cn/ch/reader/create_pdf.aspx?file_no=20171001&year_id=2017&quarter_id=10&falg=1 |title=二郎神崇拜與祆教「七聖刀」遺存比較研究 |author=朱靖江 |date=2017年5月5日 |website=xnmzxbsk.ijournal.cn |page=5 |access-date=2018年8月3日}}</ref>

== 艺术描写 ==
*《[[西游记|西游记]]》在西游记中說二郎神是[[玉皇大帝|玉皇大帝]]的[[外甥|外甥]]，「斧劈桃山曾救母，彈打㯶羅雙鳳凰」，亦会地煞[[七十二变|七十二变]]。 二郎神居於灌口，「聽調不聽宣」，平常不聽天庭命令，但危急時會接受天庭的派遣。二郎神第二次出場於[[唐僧|唐僧]]四眾行至「[[祭賽國|祭賽國]]」一回。唐僧師徒為救國中「[[金光寺|金光寺]]」蒙冤僧眾，自請為祭賽國奪回被妖精所盜寶物[[舍利子|舍利子]]。盜寶人為[[萬聖龍王|萬聖龍王]]之婿[[九頭蟲|九頭蟲]]。[[八戒|八戒]]、[[孫悟空|孫悟空]]力敵不過，正在想方設法時巧遇二郎神楊戩與梅山六兄弟打獵歸來，向前請求協助。二郎神與梅山六兄弟和八戒、悟空聯手將[[九頭蟲|九頭蟲]]擊敗，取回寶物，釋放了[[金光寺|金光寺]][[僧人|僧人]]，並改寺名為「[[伏龍寺|伏龍寺]]」。

*杂剧作品：元朝《二郎神醉射锁魔镜》，明朝《灌口二郎斩健蛟》、《二郎神锁齐天大圣》、《灌口二郎初显圣》。

*《[[聊齋志異|聊齋志異]]》的〈[[席方平|席方平]]〉最後欲找尋灌口二郎神伸冤。

*《[[寶蓮燈|寶蓮燈前傳]]》，[[瑤姬|瑤姬]]是[[玉皇大帝|玉皇大帝]]的妹妹，曾向[[女媧娘娘|女媧娘娘]]學藝，但愛上了凡人楊天祐，兩人結婚，生下了三個小孩。第一個孩子為楊戡（或作楊蛟、楊昭），第二個孩子為楊戩（楊二郎），第三個孩子為楊嬋（三聖母）天庭認為凡人與仙女不能通婚，故遣雷神與天將捉拿瑤姬，楊天祐與楊蛟在抵抗天兵時殉難，靈魂被扣押在天庭。王母娘娘以仙桃化作桃山鎮壓瑤姬，楊戩帶著楊嬋逃至女媧娘娘處，女媧娘娘將楊戩託付給了玉鼎真人，將楊嬋留在身邊。後來楊戩（楊二郎）學成道法，以斧劈桃山，救出母親。玉皇大帝念楊戩純孝，封他為[[灌江|灌江]]顯聖二郎真君，楊嬋封為[[華山|華山]]聖母，楊天祐、楊戡也封為神靈。

*《[[寶蓮燈|寶蓮燈]]》，情節接續寶蓮燈前傳，說是楊戩之妹三聖母楊嬋在其侍女的撮合下與凡人[[劉彥昌|劉彥昌]]成親生子，落得母親一般的下場，二郎神無奈因為職責所在施法，變出了莲花峰，鎮壓其妹[[華山女神|華山女神]]「[[三聖母|三聖母]]」。最後「[[劉沉香|劉沉香]]」成功[[劈山救母|劈山救母]]。
