《'''二十世纪电气目录'''》（{{jpn|j=二十世紀電氣目録}}），是[[日本|日本]]作者结城弘所著的一部[[轻小说|轻小说]]，由[[京都动画|京都动画]]旗下[[KA_Esuma文库|KA Esuma文库]]于2018年8月10日出版，由{{lang|ja|[[池田和美|池田和美]]}}担任插画，长谷百香负责美术和背景。<ref name=intro>{{cite web|url=https://www.kyotoanimation.co.jp/books/20thdenmoku/|title=二十世紀電氣目録|website=京都动画|access-date=2019-03-08|archive-url=https://web.archive.org/web/20180801194344/http://www.kyotoanimation.co.jp/books/20thdenmoku/|archive-date=2018-08-01|dead-url=no}}{{ja}}</ref>本作于2017年获得“第8届[[京都动画大奖|京都动画大奖]]”奖励赏。<ref>{{cite web |url=https://www.kyotoanimation.co.jp/kyoani_award/archive/ |title=過去の受賞作 京都アニメーション大賞 |website=京都动画 |access-date=2019-03-08 |archive-url=https://web.archive.org/web/20180911031931/http://www.kyotoanimation.co.jp/kyoani_award/archive/ |archive-date=2018-09-11 |dead-url=no }}{{ja}}</ref>2018年7月27日，京都动画宣布将其动画化。<ref>{{cite web |url=https://home.gamer.com.tw/creationDetail.php?sn=4072468 |title=京都動畫文庫輕小說《二十世紀電氣目錄》發表動畫化企劃 |website=巴哈姆特 |date=2018-07-27 |access-date=2019-03-11 |archive-url=https://web.archive.org/web/20180806080255/https://home.gamer.com.tw/creationDetail.php?sn=4072468 |archive-date=2018-08-06 |dead-url=no }}</ref>

== 作品简介 ==
《二十世纪电气目录》在2017年获得“第8届[[京都动画大奖|京都动画大奖]]”奖励奖。京都动画大奖素来以评判标准苛刻而闻名，每届都会出现奖项无人获得的情况，而第3届及第6届甚至出现了所有奖项无人获得的情况<ref>{{cite web|archiveurl=https://web.archive.org/web/20160308025833/http://comic.qq.com/a/20150523/011082.htm|url=http://comic.qq.com/a/20150523/011082.htm|title=不愧史上最难！京都动画奖无作品获奖|publisher=腾讯|author=阿三叔|date=2015-05-23|accessdate=2015-11-02|archivedate=2016-03-08|dead-url=no}}</ref>，从2009年设立至今仅有《[[紫罗兰永恒花园|紫罗兰永恒花园]]》获得过大奖。本作是2017年第8届京都动画大奖唯一获奖作品。京都动画于2018年7月宣布本作文库版发售和动画化的消息，并于2018年8月9日发布两个15和30秒的视频<ref>{{cite news |url=https://new.qq.com/cmsn/20180811/20180811041746.html|title=《二十世纪电气目录》小说发售CM公开 |website=腾讯网 |date=2018-08-10 }}</ref>但具体细节仍未公开。

小说以[[明治|明治]]时代20世纪初的日本为舞台，讲述住在京都伏见的平凡女孩百川稻子，在伏见稻荷神社祭拜时，意外遇到了否定神明存在的少年坂本喜八，两人踏上寻找奇书《电气目录》的旅途故事。<ref>{{cite news |url=http://finance.jrj.com.cn/tech/2018/07/27182724873076.shtml |title=轻小说《二十世纪电气目录》动画化 京都动画制作 |website=金融界资讯 |date=2018-07-27 |access-date=2019-03-11 |archive-url=https://web.archive.org/web/20180805183047/http://finance.jrj.com.cn/tech/2018/07/27182724873076.shtml |archive-date=2018-08-05 |dead-url=no }}</ref>

== 故事简介 ==

「你会和我一起死去，一起前往那片极乐净土吗？」

[[1907年|明治四十年]]，夏天。居住在[[京都|京都]][[伏见|伏见]]的百川稻子，什么事情也做不好，整天挨着父亲的责骂。对她来说，唯一的救赎就是向神灵请愿。有一天，她像往常一样来到神社参拜时，和一名说着“不要相信那些看不见的东西”的自由奔放的少年——坂本喜八相遇了。少年否认了神明的存在，并豪言今后是电气的时代。

这时，稻子突然提到了结婚的事情。虽然那是父亲单方面的决定，但是考虑到自身的价值后，稻子放弃了抵抗。喜八引导稻子发现了她心中想要逃跑的真实想法，带上稻子离开了禁锢着她的家。

要阻止结尾的唯一方法就是找到一本名叫《电气目录》的奇书。那是喜八在幼年时写下的关于电气时代的预言书，但是却被年长的哥哥坂本清六带走不知去向。喜八和稻子——两人为了寻找《电气目录》，逃往了京都、滋贺。

消失的《电气目录》，隐藏的是——<ref>{{cite web |url=https://news.dmzj.com/article/46466.html|title=获京都动画大奖鼓励奖的《二十世纪电气目录》开始动画化 |website=动漫之家新闻站 }}</ref>

== 登场人物 ==
;{{nihongo|坂本喜八|坂本 喜八（さかもと きはち）}}
:本作男主角。在新京极大道的某个蓬莱佛具店打工，同时也接受机械修理工作的热爱电气的少年，15岁。曾在[[大阪|大阪]]举办的第五届{{tsl|jp|内国勧業博覧会|内国劝业博览会}}中，被电气驱动的机械们所吸引，开始对电气产生了兴趣。他坚信“二十世纪是电气的世纪”，将自己对于未来电气发明的幻想写成了《电气目录》。虽然老家是寺庙，但是他并不信佛，为此和父亲大吵过后离家出走。十分不擅长应付黑暗的地方。

;{{nihongo|百川稻子|百川 稲子|（ももかわ いなこ）}}
:本作女主角。作为京都伏见的百川酿酒次女身份长大的少女，15岁。虽然在性格上纯粹活泼，但是做事迟钝经常犯错，故总是被父亲甚右卫门训斥。和做什么都得心应手的姐姐相比，她对自己的存在感到很悲观。深信神佛，碰到困难的时候都会跑到伏见稻荷的阿久火大明神前祈祷。口头禅是“信则优”。只有徒手抓老鼠这个技能不输给任何人。

;{{nihongo|百川规子|百川 規子（ももかわ のりこ）}}
:稻子的姐姐，20岁。与稻子形成鲜明对比，是一位既成熟又能干的才貌兼备的女性。虽然与青梅竹马的陆有婚约，但两人却是一见面就吵架的状态，所以结婚这事还在搁置中。她那白底碎花纹的衣装和光泽亮丽的长发是稻子所憧憬的。虽然在酒窖中长大，却不太能能够接受酒味。

;{{nihongo|陆健吾|陸 健吾（くが けんご）}}
:[[大津市|大津]]的米商陆恒商店的次子，24岁。为人认真正直、正义感很强，在稻子眼中如同哥哥一般仰慕的存在。与自己的婚约者规子水火不容，每次都被规子恶毒的话骂的闭嘴。[[日俄战争|日俄战争]]时，他的超过180cm的巨大身躯以及猪突猛进的战斗样子得到了“陆蒸气”这个外号。现在在热心地进行着帮助救世军这样的慈善活动。

;{{nihongo|坂本清六|坂本 清六（さかもと せいろく）}}
:坂本喜八的哥哥。虽然很精悍，但是瘦瘦的身躯一直让人替他捏把汗。喜欢捉弄别人，喜八也将他视为“撒谎者”而警戒着。把电气的魅力灌输给喜八的当事人，对神佛的怀疑和自由奔放的性格和他弟弟一模一样。喜欢高的地方，梦想是爬上埃菲尔铁塔。保管了差点被喜八的父亲烧掉的《电气目录》。<ref name=intro/>
