[[黎明|黎明]]於1995年發行[[夢、追踪|夢、追-{踪}-]][[音樂專輯|音樂專輯]]後，同年在香港推出《1995夏日演唱會：那一夜我們在霓虹下起舞》大獲成功，隨即於1995年12月推出《'''二次方Remix'''》[[音樂專輯|音樂專輯]]，專輯收錄了四首歌曲，包括《危情追蹤》和《原諒我》的混音版，同時收錄了《沒名字的歌，無名字的你》及《危情追蹤》的1995演唱會版<ref>{{Cite web |url =http://music.cdhaha.net/?p=3084|title =黎明 二次方Remix|date = 2006年4月25日 |publisher =CDHaha Music|language = 中文}}</ref><ref>{{Cite web |url =https://www.jitashe.org/album/474843/|title =二次方(Remix) 專輯的吉他譜|publisher =結他社|language = 中文}}</ref><ref>{{Cite book|title=當世事再沒完美：香港流行樂壇1974-2013|last=陳怡東|first=|publisher=法律出版社|year=2015|isbn=7511880754|location=|pages=324}}</ref>。
