'''二元制君主立憲制'''指[[国家|国家]][[政权|政权]]由[[君主|君主]]和[[议会|议会]]共同掌握，但君主为[[国家权力|国家权力]]中心的一种[[政体|政体]]形式。

在这种政体形式下，国家虽然制定了[[宪法|宪法]]、设立了议会，但君主仍然保持专制时代的[[权威|权威]]，集[[立法|立法]]、[[行政|行政]]、[[司法|司法]]和[[军事|军事]]大权于一身，是权力中心和最高的实际统治者。议会和君主共同掌握国家权力，君主是真正的权力中心，拥有全部[[政权|政权]]，[[政府|政府]]对君主负责而不对议会负责。

宪法由君主钦定，是君主意志的反映，君主的权利受到宪法的限制。议会只对君主起到[[制约|制约]]作用，是君主的[[咨询|咨询机构]]，立法权是形式的，君主不仅拥有否决议会立法的权力，而且还通过任命或指定议员控制议会；[[内阁|内阁]]是君主行使行政权的机构，[[首相|首相]]由君主任命，無需议会同意。

== 性质 ==
二元君主制产生于现代资本主义发展较晚、封建地主阶级长期拥有巨大势力的国家，是资产阶级与地主阶级联合专政的一种统治形式，较一般的[[君主立憲制|君主立憲制]]带有更多专制的色彩。

== 发展 ==
二元制君主立宪制是起源於近代歐洲現代化，從政府層面看來，在有兩個形式上矛盾的方向同時進行，便是[[民族國家|民族國家]]的[[中央集權制|中央集權制]]的成立，和相對的自由[[民主|民主]]的[[私有財產|私有財產]]的確認，在後世理解是因為[[公民社會|公民社會]]和[[理性主義|理性主義]]發達的結果，所以可以接受政府以獨立於營利事業原則外運作，而專注於國防和外交事務。

但在較後才開始現代化的歐洲國家，為了擴張政府權力又為了吸納社會上不同的思想的人士，為了晉身政府層面加強其運作，於是一部分君主制國家在統一和獨立的同時，編成了憲法成立議會，但和「虛君共和」相反是「虛位議會」制度。

一般認為二元制君主立宪制的原型是16世紀的[[波蘭立陶宛聯邦|波蘭立陶宛聯邦]]，但成熟於[[拿破崙|拿破崙]]時代的法國，而且受到十九世紀中歐洲推廣，如德國和義大利統一後憲法所仿效。

受到很多仍然保持前[[啟蒙時代|啟蒙時代]]的誠虔信仰的非歐洲社會的接受，例如[[日本帝國|日本帝國]]的[[明治維新|明治維新]]甚至[[巴列維王朝|巴列維王朝]]時代([[伊朗伊斯蘭革命|伊朗伊斯蘭革命]]前)的[[伊朗帝國|伊朗帝國]]。

歷史上施行過二元君主制的國家，有1871年後的[[德意志帝国|德意志帝国]]、1889年後的[[大日本帝国|大日本帝国]]、1908年後的[[大清帝國|大清帝國]]、以及1905年後的[[俄羅斯帝國|俄羅斯帝國]]。

目前仍保留二元君主制的国家只有[[約旦|約旦]]、[[摩洛哥|摩洛哥]]等少数君主立憲国家，这些国家具有[[开明专制|开明专制]]的性质。
