'''二氯甲烷'''（分子式：{{chem|CH|2|Cl|2}}）是低沸点无色液体，是重要的有机溶剂，广泛用于[[医药|医药]]、[[塑料|塑料]]及[[胶片|胶片]]等工业<ref>{{cite web |title=NHRI NEHRC-Methylene chloride |url=http://nehrc.nhri.org.tw/toxic/toxfaq_detail_en.php?id=68 |website=nehrc.nhri.org.tw |accessdate=2018-10-18}}</ref>。



== 物化性質 ==
無色透明易揮發液體。具有类似[[醚|醚]]的刺激性气味。溶于约50倍的水，溶于[[酚|酚]]、[[醛|醛]]、[[酮|酮]]、[[冰醋酸|冰醋酸]]、[[wikipedia:Triethyl_phosphate|磷酸三乙酯]]、[[乙酰乙酸乙酯|乙酰乙酸乙酯]]、[[環己胺|環己胺]]。与其他[[氯代烃|氯代烃]]溶剂、[[乙醇|乙醇]]、[[乙醚|乙醚]]和[[二甲基甲醯胺|N，N-二甲基甲酰胺]]可以任何比例互溶。相對密度1.3266(20/4°C)。熔點-95.1°C。沸點40°C。自燃點640°C。[[黏度|黏度]]（20°C）0.43mPa·s。折射率nD(20°C)1.4244。臨界溫度237°C，臨界壓力6.0795MPa。熱解後產生HCl和痕量的[[光氣|光氣]]。與水長期加热，生成[[甲醛|甲醛]]和HCl。进一步氯化，可得CHCl<sub>3</sub>和CCl<sub>4</sub>。

==工业制备==
一般通过[[甲烷|甲烷]]的氯化来合成。甲烷氯化得到包括四种氯甲烷在内的混合物，不过由于各自[[沸点|沸点]]相差较大，可以通过普通的[[分馏|分馏]]技术来分离提纯。

==用途==
*溶剂
溶剂是二氯甲烷的最主要用途。二氯甲烷具有广谱的溶解力、低沸点以及相对而言最低的毒性和相对而言最好的反应惰性，使其成为有机合成中使用频率位居第一的有机溶剂。作为溶剂其地位几乎跟无机盐化学中水相当<ref name="chemicalbook">{{cite web |title=二氯甲烷 {{!}} 75-09-2 |url=https://www.chemicalbook.com/ChemicalProductProperty_CN_CB7740372.htm |website=www.chemicalbook.com |accessdate=2018-10-18 |language=zh}}</ref>。

*有机合成原料
二氯甲烷因为[[立体电子效应|立体电子效应]]而显惰性，通常不参与化学反应。但在一定条件下也可以参与反应。工业上曾经发生过残留二氯甲烷与[[叠氮化钠|叠氮化钠]]在[[N,N-二甲基甲酰胺|N,N-二甲基甲酰胺]]里发生反应，生成[[叠氮甲烷|二叠氮甲烷]]导致工厂爆炸的事故<ref name="chemicalbook" />。

==危险性==
=== 毒性防护 ===
二氯甲烷毒性较大，但中毒後甦醒较快，故也有用作[[麻醉劑|麻醉劑]]。對皮膚及黏膜有刺激性。
成年大鼠经口[[半数致死量|半数致死量]]为1.6-2.0g/kg<ref name="chemicalbook" />。空气中最高容许浓度0.0005。操作时应戴防毒面具，发现中毒后立即脱离现场，对症治疗。
