'''淨土真宗本願寺派'''（じょうどしんしゅうほんがんじは）是[[淨土真宗|淨土真宗]]之一派。

== 概要 ==
[[本山|本山]]是宗祖[[親鸞|親鸞]]的墓所「[[大谷廟堂|大谷廟堂]]」發祥的[[西本願寺|「本願寺」（「西本願寺」）]]。[[淨土真宗|淨土真宗]]（真宗）最大宗派。

1592年[[淨土真宗|淨土真宗]]第十一代[[法主|法主]][[顯如|顯如]]去世後發生繼承權紛爭，其子[[教如|教如]]與[[准如|准如]]爭立。[[豐臣秀吉|豐臣秀吉]]立准如為法主。1602年[[教如|教如]]得到[[德川家康|德川家康]]的[[寄付|寄付]]而創派，脫離准如，另建新教團，是為東本願寺派，又稱[[大谷派|大谷派]]；而准如等人的一脈，稱「淨土真宗本願寺派」，簡稱「本願寺派」。

== 組織 ==
現在根據[http://www.hongwanji.or.jp/source/pdf/shuhou.pdf 浄土真宗本願寺派宗法]，分為執行宗教事務的合議制「總局」、立法和決議機關的「宗會」、關於門主教義的諮詢機關「勸學寮」以及審判監察機關「監正局」。

== 海外 ==
海外開教是第21代法主[[明如|明如]]提唱，以亞洲為中心，擴及夏威夷、北美，派遣開教使、建立寺院。但是，第二次世界大戰後，亞洲開教地全數廢止。現在的台灣開教地是在戰後復興。
