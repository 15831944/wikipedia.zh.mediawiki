'''氧化还原反应'''（{{lang-en|'''Red'''uction-'''ox'''idation reaction}}，簡稱{{lang|en|'''Redox'''}}<ref>{{Cite web|url=https://en.oxforddictionaries.com/definition/redox|title=redox - Definition of redox in English by Oxford Dictionaries|accessdate=2017-12-23|work=Oxford Dictionaries / English}}</ref>）是在反应前后元素的[[氧化数|氧化数]]具有相应的升降变化的[[化学反应|化学反应]]。这种反应可以理解成由两个半反应构成，即[[氧化反应|氧化反应]]和[[还原反应|还原反应]]。此类反应都遵守[[电荷守恒|电荷守恒]]。在氧化還原反應裡，氧化與還原必然以等量同時進行。

{{quote|
{{font|color=purple|还原剂}}　+　{{font|color=#003399;|氧化剂}}　→　{{font|color=purple|氧化产物（氧化数升高）}}　+　{{font|color=#003399;|还原产物（氧化数降低）}}
}}

一般来说，同一反应中还原产物的还原性比还原剂弱，氧化产物的氧化性比氧化剂弱，这就是所谓“强还原剂制弱还原剂，强氧化剂制弱氧化剂”。換言之：
* '''[[氧化劑|氧化劑]]'''得[[电子|电子]]，[[化合价|化合价]]降低，會「消耗」自己的氧化能力將還原劑氧化，本身被还原
* '''[[還原劑|還原劑]]'''失[[电子|电子]]，[[化合价|化合价]]升高，會「消耗」自己的還原能力將氧化劑還原，本身被氧化
[[File:Redox_reaction.png|thumb]]

舉個例子︰
:<math>Xe^- + Y \to X + Ye^-</math><br>

<math>X</math>被氧化（oxidized）， <math>Y</math>被還原（reduced）。

== 範例 ==
以酸性条件下[[過錳酸鹽|過錳酸鹽]]（<chem>{MnO4}^{-}</chem>）与[[草酸根|草酸根]]（<chem>{C2O4}^{2-}</chem>）做为例子，<chem>{MnO4}^{-}</chem>发生还原反应，故为[[氧化剂|氧化剂]]，<chem>{C2O4}^{2-}</chem>发生氧化反应，故为还原剂。其还原半反应式为：

:<chem>{MnO4}^{-}+8H+ + 5e- -> Mn^2+ + 4H2O</chem>

而氧化半反应式为：
:<chem>{C2O4}^{2-} -> 2CO2 + 2e-</chem>

将两式中的电子消除得到：
:<chem>({MnO4}^{-} + 8H+ + 5e- -> Mn^2+ + 4H2O)</chem><math>\times 2</math>
:<chem>({C2O4}^{2-} -> 2CO2 + 2e^{-})</chem><math>\times 5</math>
:<chem>2{MnO4}^{-} + 5{C2O4}^{2-} + 16H+ -> 2Mn^2+ + 10CO2 + 8H2O</chem>

'''特别例子：'''
应该注意的是，部分物质在某反应中是氧化剂，在另一个反应中却是还原剂。二氧化硫就是一个例子。以下方程式为二氧化硫与氧反应，形成三氧化硫：
:<chem>2SO2(4) + O2 -> 3SO3(6)</chem>

在这反应中，二氧化硫被氧化，因为硫的氧化数由4 增加至6，二氧化硫是还原剂。
现在再看看另一个例子。 [[二氧化硫|二氧化硫]]与[[硫化氢|硫化氢]]的反应：
:<chem>SO2(4) + 2H2S -> 3S(0) + 2H2O</chem>

在这反应中，二氧化硫被还原成硫，二氧化硫是氧化剂。

所以不可以简单地说「某物质是氧化剂或还原剂」，需要说明与什么物质进行化学反应。

== 氧化还原反应的表现 ==
=== 无机物 ===
无机物的氧化还原反应表现为一种元素与其他元素化合比例发生了变化。

很多可与[[氧|氧]]、[[氯|氯]]、[[硫|硫]]单质化合的物质在反应中都被氧化。大多数气态非金属单质都是较好的氧化剂，而碱金属都是还原剂。[[氢气|氢气]]、[[一氧化碳|一氧化碳]]等还原性气体能把金属从它们的氧化物中提炼出来，这种还原反应在工业上有重要用途。

氧化反应最早是指金属或非金属与'''氧化'''合形成氧化物的反应，而还原反应最早是指金属从其化合物中被'''还原'''成单质的反应。

有可变价态的金属元素，其高价态离子一般有氧化性，低价态离子一般有还原性。如[[重铬酸根|重铬酸根]]（Cr(VI)）、[[铁|铁]]离子（Fe(III)）等是氧化剂，2价[[锡|锡]]离子、2价[[钒|钒]]离子等是还原剂。

=== 有机物 ===
[[有机物|有机物]]因此而导致的基团变化。有机物的反应也需要氧化剂和还原剂，而且有机分子中的碳原子的氧化数一样会发生变化。确切的说，发生氧化数变化的碳原子仅限于涉及变化了的基团的少数几个碳原子，但为了计算方便，计算时可以取平均价态。

[[双键|双键]]和三键可以被氧化剂氧化而断开。含氧基团的转变也属于氧化还原反应，涉及此类反应的基团包括醇[[羟基|羟基]]、[[醛基|醛基]]、[[酮基|酮基]]和[[羧基|羧基]]。在适宜的条件下，它们可以互相转变。另外，多数有α-氢的[[芳香环|芳香环]]取代基能被高锰酸钾氧化为羧基。

这些反应一般用[[高锰酸钾|高锰酸钾]]、[[臭氧|臭氧]]、[[重铬酸钾|重铬酸钾]]等强氧化性物质作[[氧化剂|氧化剂]]，一些[[有机金属化合物|有机金属化合物]]及其他有活泼键的强还原性物质作[[还原剂|还原剂]]。

== 与电化学的关系 ==
通常氧化还原反应可以做成一个[[原电池|原电池]]。其中发生氧化反应的一极为[[阳极|阳极]]，即外电路的[[负极|负极]]；还原反应的一极为[[阴极|阴极]]，即外电路的[[正极|正极]]。两个[[电极|电极]]之间有[[电势|电势]][[电势差|差]]（[[电化学|电化学]]上通常叫[[电动势|电动势]]），因此反应可以进行，同时可以用来做[[功|功]]。
