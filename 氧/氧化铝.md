'''氧化鋁'''（'''Aluminium oxide'''）是[[白色|白色]][[固体|固体]]，是[[鋁|鋁]]和[[氧|氧]]的[[化合物|化合物]]，[[分子式|分子式]]為[[鋁|Al]]{{sub|2}}[[氧|O]]{{sub|3}}。在[[礦業|礦業]]、[[製陶業|製陶業]]和[[材料科學|材料科學]]上又稱為[[礬土|礬土]]。常见纯度为99.5%和96%。

1961年，[[通用电气|通用电气]]（GE）生产出了「Lucalox」，一种用于[[钠灯|钠灯]]中的透明矾土。

== 特性 == 
# 氧化鋁是一種白色固體，在礦業、製陶業和材料科學上又被稱為礬土
# 氧化鋁在地球上的蘊藏量只僅次於氧化矽，是一種屬於陶瓷原料之一，在氧化物中鋁與氧
有強力的鍵結，使得氧化鋁在氧化物當中有最高硬度，其 化學穩定性高且對大部份酸性、
鹼性、鹽類及熔融溶液有優秀的耐腐蝕性
#氧化铝有许多同质异晶体，目前已知的有10多种，主要有3种晶型，即α-Al<sub>2</sub>O<sub>3</sub>、β-Al<sub>2</sub>O<sub>3</sub>、γ-Al<sub>2</sub>O<sub>3</sub>。其中结构不同性质也不同，在1300℃以上的高温时几乎完全转化为α-Al<sub>2</sub>O<sub>3</sub>。
#自然界中的刚玉是α形属于六方最密堆积，熔点，硬度高，不溶于酸碱耐腐蚀，绝缘性好

== 製備 ==
在鋁[[礦|礦]]的主成份[[鐵鋁氧石|鐵鋁氧石]]中，氧化鋁的含量最高。工業上，[[鐵鋁氧石|鐵鋁氧石]]經由[[拜耳法|拜耳法]]純化為氧化鋁：

# 加热用[[氢氧化钠|氢氧化钠]]溶解矿石。氧化铁不溶，二氧化硅溶解为硅酸根（Si(OH)<sub>6</sub><sup>2−</sup>），氧化铝溶解为铝酸根（Al(OH)<sub>4</sub><sup>−</sup>）。
:<math>\rm Al_2 O_3 + 3 H_2 O + 2NaOH \longrightarrow 2 Na[Al(OH)_4]</math>
# 过滤，加酸处理，氢氧化铝沉淀出来，再过滤。
# 強熱[[氫氧化鋁|氫氧化鋁]]，可得無定形之白色氧化鋁粉末。
:<math>\rm 2Al(OH)_3 \longrightarrow Al_2 O_3 + 3 H_2 O</math>

再由[[霍尔-埃罗法|霍尔-埃罗法]]轉變為鋁金屬。

== 用途 ==
# [[紅寶石|紅寶石]]、[[藍寶石|藍寶石]]的主成份皆為氧化鋁，因為其他雜質而呈現不同的色澤。紅寶石含有[[氧化鉻|氧化鉻]]而呈紅色，藍寶石則含有[[氧化鐵|氧化鐵]]及[[二氧化鈦|二氧化鈦]]而呈藍色。
# 氧化铝是金属[[铝|铝]]在空气中不易被腐蚀的原因。纯净的金属铝极易与空气中的[[氧|氧气]]反应，生成一层薄的氧化铝薄膜覆盖在暴露于空气中铝表面。这层氧化铝薄膜能防止铝被继续[[氧化|氧化]]。这层氧化物薄膜的厚度和性质都能通过一种称为[[阳极处理|阳极处理]]（阳极防腐）的处理过程得到加强。
# 鋁為電和熱的良[[導體|導體]]，而氧化鋁則是電與熱的絕緣體。氧化鋁的晶體形態[[剛玉|剛玉]]因為硬度高，適合用作研磨材料及切割工具
# 氧化鋁的晶體也可作為磊晶用的基材。
# 氧化鋁粉末常用作[[色層分析|色層分析]]的媒介物。
# 2004年8月，在[[美國|美國]][[3M公司|3M公司]]任職的[[科學家|科學家]]開發出以鋁及[[稀土元素|稀土元素]]化合成的[[合金|合金]]製造出稱為transparent alumina的強化[[玻璃|玻璃]]。
# 可以用於[[高速鐵路|高速鐵路]]列車的緊急刹車裝置（向路軌噴灑氧化鋁粉末以增加鐵軌[[摩擦力|摩擦力]]）
# 多孔高比表面积的氧化铝称为活性氧化铝，常用作催化剂、吸附剂、脱水剂和催化剂载体。
# [[耐火砖|耐火砖]]的主要成分
