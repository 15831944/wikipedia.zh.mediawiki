'''閔泳煥'''（{{lang-ko|'''민영환'''}}，{{bd|1861年|8月7日|1905年|11月30日}}）是[[朝鮮國|朝鮮國]]的大臣，為[[驪興閔氏|驪興閔氏]]的一員，閔致久的孫子、[[閔謙鎬|閔謙鎬]]的兒子、[[朝鮮高宗|高宗國王]]的表兄弟。[[明成皇后|明成皇后]]是他的遠房姑姑。1905年日本同朝鮮簽訂[[乙巳條約|乙巳條約]]後閔泳煥自盡身亡。

閔泳煥字'''文若'''，號'''桂庭'''，高宗賜諡'''忠正'''。

== 生平 ==
1861年8月7日，閔泳煥出生。親生父親是權臣閔謙鎬。1878年闵泳焕参加科举考试，获庭试文科丙科及第；1881年升任同副承旨<ref>[http://sillok.history.go.kr/id/wza_11802019_001 《朝鮮高宗實錄》高宗18年2月19日]</ref>，后又任成均馆大司成，亦曾任禮曹<ref>[http://sillok.history.go.kr/id/wza_12410019_002 《朝鮮高宗實錄》高宗24年10月19日]</ref>、兵曹<ref>[http://sillok.history.go.kr/id/kza_12504016_001 《朝鮮高宗實錄》高宗25年4月16日]</ref>、刑曹<ref>[http://sillok.history.go.kr/id/wza_12410017_004 《朝鮮高宗實錄》高宗24年10月17日]</ref>、吏曹<ref>[http://sillok.history.go.kr/id/kza_13103001_003 《朝鮮高宗實錄》高宗31年3月1日]</ref>這四曹的判書。壬武軍亂時閔謙鎬被暴動群眾打死，於是閔泳煥辭了官。

閔泳煥1896年以特使之身分参加[[俄羅斯|俄羅斯]][[沙皇|沙皇]][[尼古拉二世|尼古拉二世]]的加冕典礼，也利用這個機會在[[英國|英國]]等地遊覽，見識到了西方的文化。

1897年時閔泳煥擔任李氏朝鮮駐6個歐洲國家的全權代表，在大韓帝國成立後，他獲頒太極章。
[[File:Min_Yeong-hwan_Portrait_02.jpg|thumb]]
閔泳煥非常反對日本。1905年，乙巳條約簽訂後他自殺身亡，留給了朝鮮人民一封遺書《訣告我大韓帝國二千萬同胞》：
{{quote|
嗚呼，國恥民辱乃至於此，我人民將且殄滅於生存競争之中矣。夫要生者必死，期死者得生，諸公豈不諒只。泳煥徒以一死仰報皇恩以謝我二千萬同胞兄弟。泳煥死而不死期助諸君於九泉之下，幸我同胞兄弟千萬億加奮勵，堅乃志氣勉其學問，決心戮力復我自由獨立即死子當喜笑於冥冥之中矣。鳴呼，勿少失望。
}}

== 榮譽 ==
閔泳煥死後[[朝鮮高宗|高宗]]追封為議政大臣並為他建了陵墓。1962年漢城當局頒發[[建國勳章|建國勳章]]給他，時任總統[[李承晚|李承晚]]還為他寫了墓誌銘。
