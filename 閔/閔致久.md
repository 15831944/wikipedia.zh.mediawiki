'''閔致久'''（{{朝|諺=민치구|羅=Min Chigu}}；{{bd|1795年||1874年|11月}}）是[[朝鮮王朝|朝鮮]]的政治人物，其長女[[純穆大院妃|純穆大院妃]]為[[興宣大院君|興宣大院君]]的元配，故他為[[朝鮮高宗|高宗]]和[[李載冕|李載冕]]的外祖父。

== 生平 ==
[[朝鮮哲宗|哲宗]]時期，閔致久擔任延安府使。<ref>[http://sillok.history.go.kr/id/kya_10103018_004 《李朝實錄》哲宗1年3月18日]</ref>1863年[[朝鮮高宗|高宗]]即位，次年特擢他為同知敦寧府事<ref>[http://sillok.history.go.kr/id/kza_10105010_003 《李朝實錄》高宗1年5月10日]</ref>，1865年1月令其擔任[[廣州市_(韓國)|廣州府]]留守<ref>[http://sillok.history.go.kr/id/kza_10201002_006 《李朝實錄》高宗2年1月2日]</ref>，同年七月封他為崇政大夫<ref>{{Cite web |url= http://sillok.history.go.kr/id/kza_10207025_001|title=高宗2年7月25日|accessdate=2016-08-20 |work= 朝鮮王朝實錄|publisher= 大韓民國國史編纂委員會|language=韓文和文言文}}</ref>。1866年，高宗同[[明成皇后|王妃閔氏]]結婚時攝政[[興宣大院君|興宣君]]任命他為嘉禮副使。<ref>{{Cite web |url= http://sillok.history.go.kr/id/kza_10302028_002|title=高宗3年2月28日|accessdate=2016-08-20 |work= 朝鮮王朝實錄|publisher= 大韓民國國史編纂委員會|language=韓文和文言文}}</ref>1868年8月，高宗以閔致久為判義禁府事<ref>[http://sillok.history.go.kr/id/kza_10508024_003 《李朝實錄》高宗5年8月24日]</ref>，隔年升他為[[工部|工曹]]判書<ref>{{Cite web |url= http://sillok.history.go.kr/id/kza_10605006_002|title=高宗6年5月6日|accessdate=2016-08-20 |work= 朝鮮王朝實錄|publisher= 大韓民國國史編纂委員會|language=韓文和文言文}}</ref>。

1874年11月，閔致久過世<ref>{{Cite web |url= http://sillok.history.go.kr/id/kza_11112014_001|title=高宗11年12月14日|accessdate=2016-08-23 |work= 朝鮮王朝實錄|publisher= 大韓民國國史編纂委員會|language=韓文和文言文}}</ref>，高宗「特施三等禮葬」並「遣中使護喪」，成服日甚至「望哭於昌慶宮禁川橋」。<ref>{{Cite web |url= http://sillok.history.go.kr/id/kza_11112017_001|title=高宗11年12月17日|accessdate=2016-08-23 |work= 朝鮮王朝實錄|publisher= 大韓民國國史編纂委員會|language=韓文和文言文}}</ref>
