'''化州市'''（[[邮政式拼音|邮政式拼音]]：{{lang|la|'''Fachow'''}}）现为[[中国|中国]][[广东省|广东省]]的一个省辖[[县级市|县级市]]，财政方面由省直管，行政方面由地级[[茂名市|茂名市]]代管。

== 地理 ==
化州位于广东省西南部，[[地理座标|地理座标]]为[[东经|东经]]110°21′～110°45′，[[北纬|北纬]]21°29′～22°13′。北交[[广西壮族自治区|广西壮族自治区]][[北流县|北流县]]，南接[[广东省|广东省]][[吴川市|吴川市]]，东面与[[高州市|高州市]]和[[茂名市|茂名市]]相连，西面与广西壮族自治区[[陆川县|陆川县]]及广东省[[廉江市|廉江市]]毗邻。化州市总面积2354平方公里。

化州北有[[浮梁山|浮梁山]]。東北有[[龍王山_(化州市)|龍王山]]，又叫[[來安山|來安山]]<ref name="ching">[[清史稿|清史稿]]。志、地理。廣東</ref>。北界[[茂名水|茂名水]]於東北，即[[竇江|竇江]]。另一支東北[[陵水|陵水]]，源出[[廣西|廣西]][[北流|北流]]，入境後，屈西南到[[合江墟|合江墟]]，[[羅水|羅水]]亦自[[陸水|陸水]]入，合稱[[罗江_(鉴江支流)|羅江]]，或者[[陵羅水|陵羅水]]。屈東南，過州治北合[[鉴江|鉴江]]，又東南流，入[[吳川|吳川]]至南海<ref name="ching"/>。

== 歷史 ==

[[南朝梁|南朝梁]]時，相传有龙潜于[[罗江|罗江]]之中，变化无穷，根據此傳說為當地取名為石龍縣、石龍郡<ref name="曾祥裕">{{cite web |url=http://blog.soufun.com/23811930/12309232/articledetail.htm |title=风尘仆仆远赴茂名  风水结缘为人祈福 |accessdate=2011-07-18 |date=2011-07-16 |language=zh-hans |author=曾祥裕}}</ref>。[[北宋|北宋]][[太平兴国|太平兴国]]五年再據“龙能变化之义”，把辯州改称化州<ref name="曾祥裕" />，開始了當地距今1500多年稱為「化州」的悠久历史。現時在化州城的東面，尚有當年[[隋代|隋代]]羅州城及[[南朝|南朝]]時石龍郡縣治所的遺址<ref name="曾祥裕" />。

[[唐朝|唐]][[貞觀|貞觀]]年間，屬[[嶺南道|嶺南道]]，名為[[辯州|辯州]]，[[北宋|北宋]][[太平兴国|太平兴国]]五年時改称化州，屬[[廣南西路|廣南西路]]管轄；
到[[元朝|元朝]]改為[[湖廣行省|湖廣行省]][[化州路|化州路]]。[[明朝|明]][[洪武|洪武]]元年（1368年）化州路被改置為[[化州府|化州府]]，二年（1369年）屬[[廣西行省|廣西行省]]，同年改屬[[廣東行省|廣東行省]]，七年（1374年）被降為化州[[直隸州|直隸州]]，並且府署所駐縣石龍縣被撤銷併入州；九年（1376年）被降為化縣，改屬[[高州府|高州府]]，十四年（1381年）復升為化州。

化州有[[特产|特产]][[橘红|橘红]]，[[明朝|明]][[清朝|清]]时列为[[贡品|贡品]]，因此，化州又称橘州。解放後，化州縣升格成為縣級市，改名化州市。

== 行政区划 ==
化州城区主要分为[[河东区_(化州市)|河东]]、[[河西区_(化州市)|河西]]、[[下郭区|下郭]]、[[北岸区|北岸]]四大区，现有两大商业街道分别为河东区的三十米街和河西区民主路的步行街。

化州市現時由地级[[茂名市|茂名市]]代管，但由於茂名城市建设一直以来都是重东轻西，一心向水东方向发展，所以有不少化州人民都希望化州脱离茂名，成立新化州地级市。

== 文化 ==
石龙岗為當地名勝之一，相傳為龍首所在，並與埋在水底的龍尾相連，在特定的環境影響下，會發出鳴叫聲或噴沙，吉凶各說紛紜，但其實是大自然的現象被神化了<ref name="曾祥裕" />。其他名勝尚有[[尖崗嶺|尖崗嶺]]及化州八景，即：宝山积翠、立石中流、鉴水飞帆、骊珠波光、千秋夜月、龙岗盘石、凤井鸣泉、登高野色，儘管這八景現時大多已不再存在<ref name="曾祥裕" />。

===方言===
化州方言包括在化州本土形成的方言[[化州话|化州话]]（俗称化州白话，是[[粤语|粤语]]的一个[[次方言|次方言]]）和[[𠊎话|𠊎话]]<ref>黎良文，谢日茂．化州𠊎话方言区婚俗趣谈[J]．岭南文史，1993，(02)，78-80．</ref><ref>李如龙，庄初升．粤西客家方言调查报告．广东广州：曁南大学出版社，1999．</ref>（是[[客家语|客家语]]的一个次方言）。
