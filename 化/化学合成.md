在[[化学|化学]]中，'''化学合成'''是以得到一种或多种[[产物_(化学)|产物]]为目的而进行的一系列[[化学反应|化学反应]]。合成通常表现为通过[[物理|物理]]或[[化学|化学]]方法操纵的一步或多部反应。在现代的[[实验室|实验室]]应用中，合成通常暗示整个过程可靠、可被重复且可于多个实验室中应用。

一个化学合成步骤由选择[[试剂|试剂]]开始。一个产物或[[中间产物|中间产物]]往往需要多种试剂合成。在合成中，产物的数量为[[产率|产率]]，往往以[[克|克]]为单位，或以实验产物质量与理论产物质量之比来衡量。[[副反应|副反应]]通常为降低所需产物[[产率|产率]]的化学反应。

在英语中，Synthesis的现代含义由[[阿道夫·威廉·赫尔曼·科尔贝|阿道夫·威廉·赫尔曼·科尔贝]]首次使用。

==策略==

==有机合成==
{{main|有机合成}}

有机合成为化学合成的特殊分支，用于合成[[有机|有机]][[化合物|化合物]]。一个复杂产物的[[全合成|全合成]]往往需要多步反应且消耗大量时间。

==其他含义==
'''化学合成'''的另一个含义相对狭窄，且局限在特定化学反应种类中，如[[化合反应|化合反应]]，此反应将两种试剂化合成一种产物。化合反应的通式为:
:A + B → AB
A和B可为[[单质|单质]]或[[化合物|化合物]]，AB为由A和B化合成的[[化合物|化合物]]。示例如下: 

:2 [[钠|Na]] + [[氯|Cl]]<sub>2</sub> → 2 [[氯化钠|NaCl]] (生成[[盐|盐]])
:[[硫|S]] + [[氧|O<sub>2</sub>]] → [[二氧化硫|SO<sub>2</sub>]] (生成二氧化硫)
:4 [[铁|Fe]] + 3 [[氧|O<sub>2</sub>]] → 2 [[三氧化二铁|Fe<sub>2</sub>O<sub>3</sub>]] (铁的锈蚀)
:[[二氧化碳|CO<sub>2</sub>]] + [[水|H<sub>2</sub>O]] → [[碳酸|H<sub>2</sub>CO<sub>3</sub>]] (二氧化碳溶于水中，与水反应生成碳酸)

四条特殊规律:
:金属氧化物 + H<sub>2</sub>O → 金属氢氧化物
:非金属氧化物 + H<sub>2</sub>O → 含氧酸
:金属氯化物 + O<sub>2</sub> → 金属氯酸盐
:金属氧化物 + CO<sub>2</sub> → 金属碳酸盐
