'''化膿性關節炎'''（{{lang-en|Pyogenic arthritis}}），是指傳染介體入侵關節引起的關節炎。傳染介體一般是微生物體，如[[細菌|細菌]]、[[病毒|病毒]]、[[分枝桿菌屬|分枝桿菌]]、[[真菌|真菌]]。微生物體可以通過血液或鄰近的受感染組織（如骨或滑囊）滲透到關節中。

== 主要病因 ==

*病原體從膿腫或受到感染的傷口进入血液；
*急性[[骨髓炎|骨髓炎]]；
*鄰近軟組受到感染；
*穿透性創傷；
*醫療性創傷。

== 常見細菌 ==

引起入侵(化膿)性關節炎最常見的細菌是：

*[[金黃色葡萄球菌|金黃色葡萄球菌]] (Staphylococcus aureus)–最常見於成年人
*[[流感嗜血桿菌|流感嗜血桿菌]] (Haemophilus influenzae)–最常見於兒童
*[[奈瑟氏球菌|奈瑟氏球菌]]，[[淋病|淋病]] (Neisseria gonorrhoea)–常見於年輕的成年人
*[[大腸桿菌|大腸埃希氏菌]] (Escherichia coli)–多見於老人，毒品使用者和患重病者
*[[結核分枝桿菌|結核分枝桿菌]](M. tuberculosis)，[[沙門氏菌屬|沙門氏桿菌]](Salmonella)和[[布魯氏菌|布魯氏菌]](Brucella)–造成化糞池脊柱關節炎

在易感染細菌的人群中，特別在有長期破損傷口的兒童中，亦發現[[綠膿桿菌|綠膿桿菌]]（Pseudomonas aeruginosa）會感染關節，這种细菌也會造成細菌性心內膜炎。

== 病徵 ==

當单個關節(monoarthritis)受到入侵性細菌感染而發炎时，最常見的病徵是發熱。若幾個關節同時發炎，则極可能是感染了[[金黃色葡萄球菌|金黃色葡萄球菌]]或[[淋病雙球菌|淋病雙球菌]]。

== 掃瞄診斷 ==

傳統上，化膿性關節炎的診斷基於臨床評估，而臨床評估仅基于几個混雜因素，缺乏具體证据；對於深層關節，如髖關節或肩膀，临床诊断可能是不正確的。掃瞄影像可用于確診化膿性關節炎，有助于臨床醫師作出診斷。

== 治療 ==

通常采用靜脈注射抗生素，鎮痛和以藥水注入關節清理进行治疗。
