'''希尔顿欢朋酒店'''（{{lang-en|Hampton by Hilton}}），是[[希爾頓全球酒店集團|希爾頓全球酒店集團]]最大的酒店品牌，也是全球最大的国际酒店品牌之一，旗下的2300多家酒店遍布21个国家。<ref>{{Cite web|url=http://www.hamptonhotels.com.cn/About/Us|title=希尔顿旗下 - 希尔顿欢朋酒店中国官方网站|accessdate=2019-01-02|work=www.hamptonhotels.com.cn|archive-url=https://web.archive.org/web/20180813092826/http://hamptonhotels.com.cn/About/Us|archive-date=2018-08-13|dead-url=no}}</ref>

== 历史 ==
[[File:Hampton_Inn_in_Gillette,_Wyoming.jpg|缩略图]]吉列的欢朋旅馆]]
欢朋酒店品牌始于1984年，原为[[假日酒店|假日酒店集团]]旗下的欢朋旅馆（{{lang|en|Hampton Inn}}）<ref name = "prom">{{Cite web|url=http://www.fundinguniverse.com/company-histories/promus-companies-inc-history/|title=History of Promus Companies, Inc. – name = "entrepreneur">{{Cite web|url=https://www.entrepreneur.com/franchises/hamptonbyhilton/284276|title=Hampton by Hilton Franchise Suites}}）、欣庭（{{lang|en|Homewood Suites}}）转让给其[[控股公司|控股子公司]]普罗姆斯酒店集团，并开始着力打造欢朋品牌。<ref name = "prom"/>因此欢朋开始从经济型酒店向中档酒店转型，得以与集团的前酒店品牌假日酒店竞争。<ref name = "prom"/>20世纪90年代，欢朋旅馆旗下已运营着超过220家旅店。<ref>{{cite |archivedate=2018-08-19}}</ref>2004年，“Make it Hampton”企划启动。<ref name="MiH">{{cite web|url=http://www.travelweekly.com/Travel-News/Hotel-News/Hampton-Inn-rolls-out-Make-it-Hampton-plan|title=Hampton == 分布 ==

希尔顿欢朋在全球拥有超过2300家酒店，<ref name="CLN">{{cite web|url=http://www.thelcn.com/lcn01/tripadvisor-recognizes-geneseos-hampton-inn-20170627|title=TripAdvisor Recognizes Geneseo's Hampton Inn|date=2017-06-27|publisher=Livingston County News|access-date=2017-11-28|archive-url=https://web.archive.org/web/20171201080930/http://www.thelcn.com/lcn01/tripadvisor-recognizes-geneseos-hampton-inn-20170627|archive-date=2017-12-01|dead-url=no}}</ref>大部分位于[[美国|美国]]。{{fact |希尔顿集团直接管理包括孟菲斯沙迪葛洛夫酒店在内的部分欢朋酒店，其余大部分为加盟店。}}{{fact |1993年，第一家位于美国境外的欢朋酒店在[[加拿大|加拿大]][[安大略省|安大略省]]的[[尼亚加拉瀑布城_(安大略省)|尼亚加拉瀑布城]]开业。}}2007年5月开始，美国境外的欢朋酒店加挂“by Hilton”字样。<ref>{{cite web|url=http://news.hampton.com/index.cfm/page/8007|title=Hampton Quick Facts|publisher=Hilton Worldwide|access-date=2015-10-05|archive-url=https://web.archive.org/web/20151006064447/http://news.hampton.com/index.cfm/page/8007|archive-date=2015-10-06|dead-url=no}}</ref>2009年，第一家欧洲欢朋酒店在英国开业。<ref>{{cite web|url=http://www.hotelnewsnow.com/Article/750/Hampton-opens-its-first-hotel-in-Europe|title=Hampton opens its first hotel in Europe|date=2009-03-03|publisher=Hotel News|access-date=2015-10-05|archive-url=https://web.archive.org/web/20151006070133/http://www.hotelnewsnow.com/Article/750/Hampton-opens-its-first-hotel-in-Europe|archive-date=2015-10-06|dead-url=no}}</ref>2015年，希尔顿集团宣布其旗下的尊盛和欢朋品牌将全部加挂“by Hilton”。<ref name="king">{{cite web|url=http://www.travelweekly.com/Travel-News/Hotel-News/Hilton-promotes-ties-to-Embassy-Suites-and-Hampton|title=Hilton promotes ties to Embassy Suites, Hampton|accessdate=2015-10-05|author=Danny King|date=2015-05-18|publisher=Travel Weekly|archive-url=https://web.archive.org/web/20151120002146/http://www.travelweekly.com/Travel-News/Hotel-News/Hilton-promotes-ties-to-Embassy-Suites-and-Hampton|archive-date=2015-11-20|dead-url=no}}</ref>
