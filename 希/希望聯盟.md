'''希望聯盟'''（{{lang-ms|Pakatan Harapan}}，缩写为PH）是[[馬來西亞|馬來西亞]]的一個[[政党联盟|政党联盟]]，由[[人民公正黨|人民公正黨]]、[[民主行动党_(马来西亚)|民主行動黨]]、[[土著團結黨|土著團結黨]]和[[國家誠信黨|國家誠信黨]]組成<ref>{{cite web |url=http://www.channelnewsasia.com/news/asiapacific/malaysia-s-opposition/2143004.html |title=Malaysia's opposition band together under new Pakatan Harapan alliance - Channel NewsAsia |work= |accessdate=2015-10-03}}</ref><ref>{{cite web|url=http://www.themalaysianinsider.com/malaysia/article/pakatan-harapan-is-new-opposition-pact-supports-anwar-for-pm|title=Pakatan Harapan is new opposition pact, supports Anwar for PM - The Malaysian Insider|format=|work=|accessdate=2015-10-03|deadurl=yes|archiveurl=https://web.archive.org/web/20151002155638/http://www.themalaysianinsider.com/malaysia/article/pakatan-harapan-is-new-opposition-pact-supports-anwar-for-pm|archivedate=2015年10月2日|df=}}</ref>，于2015年9月22日成立，现为马来西亚的执政联盟。前身为民主行動黨、人民公正黨同伊斯蘭黨于2008年4月1日組成的[[人民联盟_(马来西亚)|人民聯盟]]。

[[2018年马来西亚大选|2018年马来西亚大选]]，希望联盟除继续执政[[槟城州|槟城州]]和[[雪兰莪州|雪兰莪州]]外，还夺得[[柔佛州|柔佛州]]、[[森美兰州|森美兰州]]、[[霹雳州|霹雳州]]、[[吉打州|吉打州]]、[[马六甲州|马六甲州]]及[[沙巴州|沙巴州]]的执政权。

==历史==
===成立===
希望联盟的前身是由[[人民公正党|人民公正党]]、[[民主行动党|民主行动党]]和[[马来西亚伊斯兰党|伊斯兰党]]，三党组成的[[人民联盟_(马来西亚)|人民联盟]]。人民联盟成立于[[2013年马来西亚大选|2013年马来西亚大选]]两年后。由于伊斯兰党和民主行动党之间对{{tsl|en|Hudud|伊斯蘭刑事法}}有爭議，三成員黨之一的伊斯蘭黨在2015年6月15日決定同民主行動黨斷交，造成人民联盟瓦解，并于2015年6月16日由民主行動黨秘書長[[林冠英|林冠英]]宣佈[[人民联盟_(马来西亚)|民联]]不复存在，<ref>{{cite news|url=http://www.orientaldaily.com.my/nation/gn201441497|title=民聯不復存在　行动党將促成一新联盟|website=[[馬來西亞東方日報|東方網]]|access-date=2015-10-20|archive-url=https://web.archive.org/web/20150924074932/http://www.orientaldaily.com.my/nation/gn201441497|archive-date=2015-09-24|dead-url=yes}}</ref>而公正党也在次日宣布民聯瓦解。<ref>{{cite news|url=http://www.malaysiakini.com/news/302216|title=公正党接受民联瓦解<br>打造新联盟对抗国阵|date=2015-06-17|publisher=当今大马}}</ref><ref>{{cite web |url=http://mymassa.blogspot.my/2014/09/6-sebab-pas-keluar-pakatan-rakyat.html |title=6 Sebab PAS Keluar Pakatan Rakyat |work= |accessdate=2014-09-28 |archive-url=https://web.archive.org/web/20161117085316/http://mymassa.blogspot.my/2014/09/6-sebab-pas-keluar-pakatan-rakyat.html |archive-date=2016-11-17 |dead-url=yes }}</ref><ref>{{cite web |url=http://www.mykmu.net/2015/06/dap-umum-bubar-pakatan-rakyat/ |title=DAP Umum Bubar Pakatan Rakyat |format= |work= |accessdate=2015-06-16}}</ref><ref>{{cite web |url=http://www.bharian.com.my/node/61605 |title=Pakatan Rakyat Tidak Wujud Lagi - ==成员党==
希望联盟的成员党包括人民公正党、民主行动党、土著团结党和国家诚信党。在[[2018年马来西亚大选|2018年马来西亚大选]]获得113席，另有8席由与其合作的[[沙巴民族复兴党|沙巴民族复兴党]]贡献，還有1席由支持希望联盟的独立人士[[峇拉巴卡蘭|峇拉巴卡蘭]]（5月13日，蔡添强宣布他加入公正党）夺得。希望联盟与其盟友组成简单多数中央政府。公正党在本届大选中赢得47席，后来3位独立人士加盟而增至50席。
