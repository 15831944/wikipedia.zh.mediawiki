'''希拉里·斯旺克'''（{{lang-en|Hilary Swank}}，{{bd|1974年|7月30日||}}），生于[[美国|美国]][[内布拉斯加州|内布拉斯加州]]，童星出身的[[美国|美国]][[演员|女演员]]，两度获得[[奧斯卡最佳女主角奖|奧斯卡最佳女主角奖]]，也是奥斯卡史上屈指可数的“两提两中”得獎者。

== 影视生涯 ==

她在[[好莱坞|好莱坞]]的生涯以The Karate Kid 4（即《The Next Karate Kid》）开始，影片中她扮演了当时鼎鼎大名的“宫城师傅”所收的第一位女弟子。

在1997－1998年间她还在电视剧Beverly Hills 90210中扮演配角，后来被解雇。三个星期後她赢得了《[[男孩別哭|男孩別哭]]》的角色。在这部根据真人真事改编的电影中，她扮演了一位[[性別認同障礙|性別認同障礙]]的男子，最终被恐惧性別認同障礙者残忍杀害。出色的演技使她年仅25岁便接连摘下了金球奖影后和第72届[[奥斯卡最佳女主角奖|奥斯卡最佳女主角奖]]的殊荣。

2004年，她在[[克林·伊斯威特|克林·伊斯威特]]指导的《[[百万宝贝|百万宝贝]]》中出演一位女拳击手。她於2005年凭此片赢得金球奖剧情类电影最佳女主角，并於2005年2月27日迎来了第二个[[奥斯卡最佳女主角奖|奥斯卡最佳女主角奖]]，再一次击败[[安妮特·班宁|安妮特·班宁]]，与[[费雯丽|费雯丽]]等屈指可数的演员一样两次提名两次获奖。

== 个人生活 ==

斯旺克在[[华盛顿州|华盛顿州]]贝灵汉的Samish湖边长大，不过她并不掩饰自己穷困的过去。第二次荣登奥斯卡影后时，她说道：“不知道今生积了什么德才能有今天。我只是个在活动车房裡长大，有一个梦想的小姑娘罢了。”

她16岁时从[[高中|高中]]辍学，决定去演艺界發展。之前15岁时父母离异，母亲为了支持她，後来搬到了[[加利福尼亚州|加利福尼亚州]]，两人住在车裡度日。不久後斯旺克就出现在一些电视剧中，如Evening Shade、《[[捉鬼者巴菲|捉鬼者巴菲]]》和《[[成长的烦恼|成长的烦恼]]》，母女俩才结束了流浪的生活。

即使如此，在赢得第一次奥斯卡最佳女主角奖後，她仍然近乎一贫如洗。《[[男孩不哭|男孩不哭]]》的片酬仅有一天75美元，一共她才赚到3000美元。据《[[60分钟|60分钟]]》报道，她当时的收入竟然低到无法申请[[健康保险|健康保险]]的地步：虽然赢得了奥斯卡最佳女主角奖这颗好莱坞王冠上的明珠，买药时却仍然要自己支付全部费用。
