'''希臘軍政府時期'''，泛指右翼軍事政權統治希臘的期間。始於1967年4月21日清晨希臘軍方發動[[政變|政變]]，終於1974年7月，前後共七年。軍政府別名'''上校政权'''（{{lang-el|Το καθεστώς των Συνταγματαρχών}}）。

== 背景 ==
第二次世界大戰後不久，希臘即爆發了共產黨與政府之間的內戰。後來在美國和英國支援下，政府軍於1949年獲勝。[[希臘共產黨|希臘共產黨]]隨即被宣布為非法組織，許多黨員遭到迫害或流亡出國。從此，希臘軍方與美國[[中情局|中情局]]之間的關係變得相當密切。這個現象尤其在1952年希臘加入[[北大西洋公約組織|北大西洋公約組織]]後，更為明顯。再加上希臘在[[圍堵政策|圍堵政策]]中佔有重要的戰略地位，以及共黨隨時再度叛亂的風險仍高。因此，就在這種「恐共」的氣氛當中，極右勢力逐漸抬頭。

保守派在經過數年的執政之後，1965年，由中立派議員[[乔治奥斯·帕潘德里欧|喬治‧帕潘德里歐]]出任希臘總理，他的當選被視為改變的契機。然而，當時年紀尚輕的國王康斯坦丁二世，為了要獲得更多的政治權利，而與改革派發生了衝突，還將帕潘德里歐打發走，釀成了一場憲政危機。事後，國王指派揚.帕纳吉尔蒂斯為臨時總理，做為這段期間的過渡政府。並宣佈將於1967年5月28日舉行國會改選。由於跡象顯示帕潘德里欧所屬的中間聯盟將一舉成為國會最大黨，並且跟立場偏左的聯合民主黨組成聯合政府。此舉造成保守派人士的憂慮，深怕共產黨會死灰復燃；而這也成為軍方發動政變的理由。

== 政變 ==
1967年4月21日，就在選舉即將舉行的前幾個禮拜，陸軍[[准將|准將]]斯蒂利亚诺斯·佩塔克斯和兩名上校[[喬治·帕帕多普洛斯|喬治·帕帕多普洛斯]]以及[[尼可拉斯·馬卡瑞佐斯|尼可拉斯·馬卡瑞佐斯]]等人共同發動了兵變，以迅雷不及掩耳的速度奪取了政權。其中佩塔克斯是總部設在雅典之裝甲訓練中心的指揮官，使得整個軍團得以立刻控制整個雅典市區。同一時間，他們還派人去搜捕各主要政治領袖，以及被認為是同情共產黨的一般平民。到了上午，總理[[帕纳约蒂斯·卡内罗波罗斯|帕纳约蒂斯·卡内罗波罗斯]]遭到軟禁，全國落入軍方手中。這時，美國駐希臘的大使並不贊同這種作法，批評這是在「強姦民主」。然而中情局駐雅典分處的處長Jack Maury則以「你要怎麼強姦一個妓女」來反駁。同年12月，國王試圖反制政變失敗，被迫流亡至[[羅馬|羅馬]]，從此以後政權遂完全落入軍方手中。

== 軍政府的成立 ==
國王出亡以後，軍方乃成立攝政團以取代原先之民選政府，由喬治·帕帕多普洛斯接任首相。
