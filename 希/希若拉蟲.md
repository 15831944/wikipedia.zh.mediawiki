'''希若拉蟲'''（[[學名|學名]]：{{lang|la|'''''Ceraurus'''''}}）是生存在中[[奧陶紀|奧陶紀]]至晚[[志留紀|志留紀]][[海洋|海洋]]中的一[[屬|屬]][[三葉蟲|三葉蟲]]。其中[[模式種|模式種]][[多側肋希若拉蟲|多側肋希若拉蟲]]（''Ceraurus pleurexanthemus''）的頭部（Cephalon）有著長長的頰刺與細小的眼睛，[[胸部|胸部]]則分為11節，每節的末端都有著尖銳的棘刺，尾部也長有一對長長的尾刺，說明牠們是[[掠食性|掠食性]]的[[物種|物種]]。<ref name="Ceraurus">{{Cite web |url = http://tomleetaiwan.pixnet.net/blog/post/32085631-%E4%B8%89%E8%91%89%E8%9F%B2%E5%8C%96%E7%9F%B3%E6%A8%99%E6%9C%AC---ceraurus-pleurexanthemus-%E5%A4%9A%E5%81%B4%E8%82%8B |title = 三葉蟲化石標本 - Ceraurus pleurexanthemus 多側肋希若拉蟲 @ Tomm Happy Days :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-25 }}</ref>其化石分布於[[北美|北美]]、[[英國|英國]]、[[瑞典|瑞典]]和[[中國|中國]]等地。<ref name="Fossilworks" />
