'''希伯來注音符號'''，亦名'''尼庫德'''（{{lang-he|'''נִקּוּד'''}}），是[[希伯來語|希伯來語]]的[[附加符號|附加符號]]，用於標示[[元音|元音]]，亦用來分辨一個字的不同讀音。這個注音系統在中世紀前期出現，而現代希伯來語所使用的符號在1世紀中後期，由[[提比里亞|提比里亞]]的[[馬索拉文士|馬索拉文士]]發明。

熟悉希伯來語的人士無需使用注音符號，而根據前文後理推定該字的讀音。注音符號比[[輔音|輔音]]字母來得小，所以，它們既方便初學者學習，又不會影響母語使用者閲讀。

除了字典、詩歌及兒童讀物以外，現代希伯來語甚少使用注音符號。爲了消歧義、避免混肴意思，在1996年，[[希伯來語科學院|希伯來語科學院]]發展出了“全寫標準”<ref>[http://hebrew-academy.huji.ac.il/decision4.html ''Rules for Spelling without Niqqud''] {{webarchive|url=https://web.archive.org/web/20090227051707/http://hebrew-academy.huji.ac.il/decision4.html |date=2009-02-27 }}</ref>。

其中一個少用注音符號的原因是因爲現代希伯來語發音較少能用注音標出，例如，“צֵירֵי”及“סֶגּוֹל”（兩個注音符號）在[[提比里安發音|提比里安發音]]法的發音不同，但在現代發音沒有分別，使得注音符號的用處減少。對於新一代的希伯來語母語使用者來説，這些符號變得沒有意義。

== 鍵盤輸入 ==
希伯來語的[[輔音|輔音]]及注音符號都可以鍵盤的輸入法及網上的[[虛擬鍵盤|虛擬鍵盤]]輸入。

=== Microsoft Windows ===
在[[Windows_8|Windows 8]]或更新版本的系統的希伯來語輸入法中，按右Alt及注音符號對應的讀音便可鍵入。在Windows 8以前的版本，要先按CapsLock，並把鍵盤游標放在輔音後面，然後同時按Shift及下表所列的按鍵。

Windows用戶可以透過修改設定檔，便可按Alt及十六位元的[[Unicode|Unicode]]碼輸入符號。

用戶亦可以“微軟鍵盤界面創建程式”（Microsoft Keyboard Layout Creator）自訂鍵盤排版，以方便地輸入符號。

=== Linux ===
在[[GTK+|GTK+]] Linux系統中，有兩個方法輸入注音符號：
* 同時按下AltGR鍵及跟Windows相同的對應代碼鍵；

*按著Ctrl、Shift和U，然後輸入4個字的Unicode碼。

=== 麥金塔 ===
在Mac OS X的希伯來語輸入法中，同時按下Option鍵及對應的數字鍵便可輸入。
