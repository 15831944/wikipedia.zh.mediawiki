'''希臘－土耳其关系'''指[[希腊|希腊]]与[[土耳其|土耳其]]两国之间的关系。

== 历史 ==
=== 希腊独立战争 ===
自15世纪起，希腊地区被[[奥斯曼土耳其|奥斯曼土耳其]]占领，该地区文化、经济等遭受严重的摧残。1821年3月，希腊爆发起义，并迅速得到响应。1822年1月1日，希腊宣布独立。1830年4月，土耳其在压力之下，宣布承认希腊独立。

== 希土战争 ==
{{main|希土戰爭 (1919年-1922年)}}
一战结束后，土耳其作为同盟国方，成为战败国。希腊试图收复更多的失地，因而发动了[[第二次希土战争|第二次希土战争]]。希腊军队在初期取得胜利，但最终遭遇了失利。

== 塞浦路斯问题 ==
[[File:Map_of_Cyprus,_Greece_and_Turkey.svg|thumb]]
{{main|塞浦路斯問題}}
1960年8月，[[塞浦路斯|塞浦路斯]]宣布独立，成立[[塞浦路斯共和国|塞浦路斯共和国]]。1974年，塞浦路斯南北分裂。塞浦路斯大部分领土为塞浦路斯共和国控制，主要以希腊族为主；塞浦路斯北部则是自行独立的[[北塞浦路斯土耳其共和国|北塞浦路斯土耳其共和国]]，以土耳其族为主。目前，大多数国家承认塞浦路斯共和国，北塞浦路斯仅土耳其一国承认。塞浦路斯南北双方长期对峙，希土两国则各支持一方。

== 领土争议 ==
希土两国对[[爱琴海|爱琴海]]的部分领土、领海的主权存在争议。1913年，[[第一次巴尔干战争|第一次巴尔干战争]]后，希腊获得了爱琴海除Imbros岛、Tenedos岛之外所有的岛屿。两国曾就爱琴海部分岛屿主权问题发生过数次冲突。
