[[File:AGMA_Hérodote.jpg|thumb]]'''希羅多德'''（{{lang-grc|Ἡρόδοτος}}），前5世纪（{{bd|約前484年||前425年||H}}）的[[古希臘|古希臘]]作家，他把旅行中的所闻所见，以及波斯[[阿契美尼德帝国|阿契美尼德帝国]]的历史纪录下来，著成《[[歷史_(希羅多德)|歷史]]》一書，成為西方文學史上第一部完整流傳下來的散文作品。

== 生平 ==
尽管希罗多德留下了《历史》这样的巨著，但关于他本人情况的记载甚少。历史学家仅根据他的作品中的蛛丝马迹以及一些相当晚近的资料（比如10世纪[[拜占庭|拜占庭]]的[[苏达辞书|苏达辞书]]）来推测他的生平。

前484年，希罗多德生於[[哈利卡那索斯|哈利卡那索斯]]，即現在[[小亚细亚|小亚细亚]]的[[博德魯姆|博德魯姆]]，似乎在一次[[政变|政变]]失败后被流放，并在[[萨摩斯島|萨摩斯島]]隐居。他好像再没有回去过哈利卡那索斯，但还是在《历史》中表明对自己城邦及其女王亚特米西雅的热爱。在流放中他周游了他在《历史》中记载的各国。他去过[[古埃及|古埃及]]，并从[[尼罗河|尼罗河]]南下到[[阿斯旺|阿斯旺]]，以及[[美索不达米亚|美索不达米亚]]、[[克里米亚半岛|克里米亚半岛]]、[[黑海沿岸平原|黑海沿岸平原]]、[[亚平宁半岛|亚平宁半岛]]和[[西西里岛|西西里岛]]。希罗多德提到了他在[[斯巴达|斯巴达]]访问了一个题材提供者，并肯定在[[雅典|雅典]]住过一段时间。他在雅典认识了[[伯里克利|伯里克利]]的家族，并提及了它的演讲家传统。但雅典人并没有把外邦人当作公民：在前444年雅典殖民于亚平宁半岛的[[图利|图利]]（{{lang|en|Thurii}}）时，希罗多德成了一位殖民者。至于他是否在那儿去世則无从考知。

在一段时间内他是個說故事者，并口述了不同战役、历史及外邦的故事。他游历了希腊城邦和参加了主要的宗教和体育活动。在前431年，雅典和斯巴达间的[[伯罗奔尼撒战争|伯罗奔尼撒战争]]爆发。可能正是这一冲突引发了希罗多德撰寫《历史》，他在書中强调只有在一个雅典、斯巴达和各城邦联合了的希腊联盟的共同努力下，才可制止波斯帝国的扩张。

== 作品 ==
{{lang|el|Ἱστορίαι}}一詞原意為「調查」、「探究」，因這部著作的影响，罗马演说家[[西塞罗|西塞罗]]评价他为“历史之父”（{{lang|la|pater historiae}}）。但不少历史家和哲学家怀疑希羅多德所记内容的真实性，并称他为“谎言之父”或“欺骗者”。但近期的[[考古学|考古学]]成果證實希罗多德所記的“历史”大部分很准确。在很多没有把握的情况下，希罗多德参考了各地或对各个事件最权威的史学记载，然后才发表自己的意见去辨别真伪，并给与解释。

后世的编辑把在前430年至前424年发表的《历史》分为9章，以各[[缪斯|缪斯]]的名字命名。前六章大致讲述的是波斯帝国的崛起，从[[利底亚|利底亚]]的[[克罗伊斯|克罗伊斯]]，第一个亚细亚的希腊征服者的故事说起。波斯帝国的创始人[[居鲁士大帝|居鲁士大帝]]击败了[[克罗苏斯|克罗苏斯]]并吞并了整个亚细亚。前六章记载到前490年当波斯在[[马拉松战役|马拉松战役]]兵败于希腊联盟为止。后3章记载的是波斯王[[泽克西斯一世|泽克西斯一世]]十年后企图雪耻马拉松战役并再次侵略希腊的事迹。《历史》记载到前479年当雅典在[[萨拉米海战|萨拉米海战]]和希腊联盟在[[温泉关战役|普拉提亚战役]]再次将波斯击败，并使波斯帝国的国界退回到[[愛琴海|愛琴海]]对岸结束。

《[[歷史_(希羅多德)|歷史]]》一書有[[王以鑄|王以鑄]]的漢譯本，1959年[[商務印書館|商務印書館]]初版，1980年代起多次重印。另外亦有[[徐松岩|徐松岩]]的汉译本，2008年2月[[上海三联书店|上海三联书店]]初版。
