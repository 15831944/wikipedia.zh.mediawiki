《'''愛情守則'''》（{{lang-en|''New Rules''}}）<ref>{{cite web|url=https://www.facebook.com/WarnerMusicTaiwanIntl/videos/1502885346438744/|title=小心上癮，英倫危險甜心再度來襲！|work=華納音樂|accessdate=2017-10-17}}</ref>是[[英國|英國]]歌手[[杜娃·黎波|杜娃·黎波]]演唱的一首歌曲，出自其2017年{{le|杜娃·黎波 (專輯)|Dua Lipa (album)|首張同名專輯}}。這首歌曲由卡羅琳·愛林、{{le|埃米莉·華倫|Emily Warren (songwriter)}}和{{le|伊恩·柯克派屈克|Ian Kirkpatrick (record producer)}}創作，並由柯克派屈克製作。這首歌曲于2017年7月21日作爲專輯的第6支單曲發行至英國的[[當代流行电台|當代流行电台]]，並于2017年8月22日以相同形式發行至美國。

《愛情守則》是一首以[[鼓|鼓]]和號為樂器編成的{{le|熱帶浩室|Tropical house}}、[[電子舞曲|EDM]]和[[流行電音|電子流行]]歌曲。歌詞是關於黎波列出一系列忘記前男友的守則。歌曲獲得了主流樂評的廣泛好評，大部分樂評人將其貼上分手和女權主義聖歌的標籤。歌曲在英國、荷蘭、比利時（法蘭德斯）、愛爾蘭、以色列和羅馬尼亞奪得榜首，而在澳洲、加拿大、芬蘭、德國、意大利、紐西蘭、挪威和瑞典等國家進入了前十名。

《愛情守則》的音樂錄影帶由亨利·斯科菲尔德執導，並于2017年7月7日首映于黎波的[[Youtube|Youtube]]賬戶。影片展示了黎波和朋友身處在酒店房間中並不斷阻止她回到前男友的身邊。這部錄影帶獲得了許多樂評人的關注，評論稱讚了其關於女權主義的主題。歌曲的商業成功有部分歸功於該音樂錄影帶的流行程度。黎波曾在俄羅斯的{{le|歐羅巴加現場演唱會|Europa Plus}}和英國的[[格拉斯顿伯里当代表演艺术节|格拉斯顿伯里当代表演艺术节]]演唱《愛情守則》<ref>{{cite web |url=http://thatgrapejuice.net/2017/07/watch-dua-lipa-performs-new-rules-live-russia/ |title=Watch: Dua Lipa Performs 'New Rules' Live In Russia |date=2017-07-30 |publisher=That Grape Juice |accessdate=2017-08-26}}</ref><ref>{{cite web |url=http://www.nme.com/news/music/dua-lipa-pulls-one-biggest-crowds-glastonbury-2017-far-hit-packed-set-2092368 |title=Dua Lipa pulls one of the biggest crowds at Glastonbury 2017 so far with hit-packed set |last=Daly |first=Rhian |date=2017-06-23 |work=NME |accessdate=2017-08-26}}</ref>。

==背景和詞曲==
[[File:Dua_Lipa_04-19-2017_-16_(33970014540).jpg|175px]]
《愛情守則》由卡羅琳·愛林、{{le|埃米莉·華倫|Emily Warren (songwriter)}}和歌曲製作人{{le|伊恩·柯克派屈克|Ian Kirkpatrick (record producer)}}創作，收錄在[[杜娃·黎波|杜娃·黎波]]2017年的首張同名專輯。黎波花費了三年完成了專輯的創作，她認爲，在這一長段時間里，她可以不斷地創作和錄製歌曲，直到她感覺可以從中選出非常可觀的量放進專輯中<ref>{{cite web |url=http://diymag.com/2017/06/05/dua-lipa-self-titled-debut-interview-rabbits-whiplash-1 |title=Interview – Dua Lipa: Blow Your Mind |last=Kheraj |first=Alim |date=2017-06-05 |work=DIY |url=http://www.refinery29.com/2017/06/160092/emily-warren-hurt-by-you-music-video |title=The Drop: Exclusive Music Video Premiere For Emily Warren's 'Hurt By You' |last=Baila |first=Morgan |date=2017-06-21 |publisher=Refinery29 |accessdate=2017-08-17}}</ref>」歌曲由柯克派屈克策劃和編制、由{{le|克里斯·蓋林格|Chris Gehringer}}製作母帶並由喬希·古德温混音<ref>{{cite web |url=https://listen.tidal.com/album/74540840 |title=Dua Lipa by Dua Lipa |publisher=[[Tidal|Tidal]] |accessdate=2017-08-08}}</ref>。當談及這首歌曲時，黎波解釋說：「《愛情守則》與專輯中大部分歌曲有些不同。我想要它感覺非常新穎（……）這是一首我希望我與某人分手時就已經有的分手歌曲」以及「它是關於要跟對你不好的人保持距離。我設置一些守則所以（我）不會再回到那人身邊。」<ref name="bbvd">{{cite web |url=http://www.billboard.com/articles/news/7880956/dua-lipa-new-rules-exclusive-video |title=Go Behind the Scenes With Dua Lipa on Her 'New Rules' Video: Exclusive |last=Newman 《愛情守則》是一首有雷鬼舞廳元素的{{le|熱帶浩室|Tropical house}}、[[電子舞曲|EDM]]<ref name="trop">{{cite web |last=Davies |first=Hannah J. |url=https://www.theguardian.com/music/2017/jun/01/dua-lipa-review-debut-warner-chris-martin |title=Dua Lipa: Dua Lipa review – a pop debut high on summery nonchalance |work=[[The_Guardian|The Guardian]] |date=2017-06-01 |accessdate=2017-07-14}}</ref>和[[流行電音|電子流行]]歌曲<ref name="rs"/>，並以[[鼓|鼓]]和號為樂器編成<ref name="clash">{{Cite news |url=http://www.clashmusic.com/reviews/dua-lipa-dua-lipa |title=Dua Lipa – Dua Lipa |last=Green |first=Alex |date=2017-06-02 |work=Clash |accessdate=2017-07-16}}</ref>。根據法貝爾音樂旗下Musicnotes.com出版的電子琴譜顯示，《愛情守則》以[[A小調|A小調]]創作，為每分鐘116拍。黎波演唱這首歌曲的音域在低音[[A_(音名)|A<sub>3</sub>]]到高音[[E_(音名)|E<sub>5</sub>]]之間<ref>{{cite web |url=http://www.musicnotes.com/sheetmusic/mtd.asp?ppn=MN0176618 |title=New Rules by Dua Lipa |publisher=Musicnotes.com |accessdate=2017-08-26}}</ref>。歌曲歌詞是關於歌手設定了一系列的守則來防止自己再次回到前男友身邊。這些守則是「第一：不要接起電話，你知道他只會在喝醉和寂寞時才打來找你/第二：別讓他進來，你要將他再次踢出家門/第三：不要成爲他的朋友，你知道早上時你又會在他的床上醒來。」<ref name="strait">{{cite web |url=http://www.straitstimes.com/lifestyle/entertainment/the-most-refreshing-voice-in-pop-music |title=Music review: Singer Dua Lipa is the most refreshing voice in pop music |last=Raguraman |first=Anjali |date=2017-07-19 |work=[[The_Straits_Times|The Straits Times]] |accessdate=2017-08-09}}</ref>黎波在接受[[全國公共廣播電台|全國公共廣播電台]]訪問時承認「我不一定會遵守這些非必然的守則。然而我認爲這些規則對於告訴你自己、告訴你朋友這些是重要的……這是爲什麽人會分手，也可能同時是你不應該復合的原因。<ref name="npr">{{cite web |url=http://www.npr.org/2017/08/06/541592438/yearning-for-an-ex-heed-dua-lipa-s-new-rules |title=Yearning For An Ex? Heed Dua Lipa's 'New Rules' |last=Boyd |first=Sophia |date=2017-08-06 |publisher=[[NPR|NPR]] |accessdate=2017-08-09}}</ref>」華倫透過網站[[Genius|Genius]]歌詞中的注解來解釋歌曲其實是對於一些從分手陰霾走出來的人的一個自我提醒，不要屈服于一時的誘惑，因爲「對於長期來説這并非好事」<ref>{{cite web |url=https://genius.com/a/dua-lipa-needs-a-little-advice-in-new-video-for-new-rules |title=Dua Lipa Needs A Little Advice In New Video For 'New Rules' |last=Seabrook III |first=Robby |date=2017-07-08 |publisher=Genius |accessdate=2017-08-09}}</ref>。

==樂評==
《愛情守則》獲得了主流樂評的廣泛好評。[[AllMusic|AllMusic]]的尼爾·Z·楊稱其為是一首「受到[[浩室音樂|浩室]]影響的亮點（歌曲）」，其中黎波提出了「一個警示清單」<ref>{{cite web |last=Yeung |first=Neil Z. |url=http://www.allmusic.com/album/dua-lipa-mw0002958693 |title=Dua Lipa – Dua Lipa |publisher=[[AllMusic|AllMusic]] |accessdate=2017-08-17}}</ref>。《{{le|Clash|Clash (magazine)}}》的專欄作家亞歷克斯·格林認爲像《愛情守則》這樣的歌曲證明了「究竟爲什麽樂評人（在去年）就看好黎波。」<ref name="clash"/>{{le|The Cut|The Cut (website)}}的戴娜·埃文斯為其貼上「分手聖歌」的標籤<ref name="The Cut" />，而《[[告示牌_(雜誌)|告示牌]]》的梅琳達·紐曼認爲它是一首「女權主義聖歌」<ref name="bbvd"/>。《[[海峽時報|海峽時報]]》的安賈利·拉古拉姆還稱這首歌曲是一首「分手聖歌」，有著「令人入迷、以號聲做裝點的流行音調」，並補充道：「這首歌曲與其嫺熟的製作和傑出的聲綫流露出了自信和態度。」<ref name="strait"/>{{le|musicOMH|musicOMH}}的本·霍格伍德認爲這首歌有著「自信和完全女性力量」的主題，並感覺這比《{{le|耳目一新|Blow Your Mind (Mwah)}}》更令人心悅誠服<ref>{{cite web |url=https://www.musicomh.com/reviews/albums/dua-lipa-dua-lipa |title=Dua Lipa by Dua Lipa |last=Hogwood |first=Ben |date=2017-06-05 |publisher=musicOMH |accessdate=2017-08-17}}</ref>。節拍媒體的瑞恩·科爾評價歌曲的歌詞，並認爲其製作在整張專輯中有著「最為獨特的元素」<ref>{{cite web |url=https://beat.media/dua-lipa-dua-lipa-an-album-review |title=Dua Lipa 'Dua Lipa' – An Album Review |last=Cole |first=Ryan |date=2017-06-18 |publisher=Beat Media |accessdate=2017-08-17}}</ref>。

The Edge的詹姆士·貝克寫道「《愛情守則》那極快的合成聲」使得《杜娃·黎波》開始「歡快起來」<ref>{{cite web |url=https://www.theedgesusu.co.uk/records/albums/2017/06/02/review-dua-lipa-dua-lipa/ |title=Review: Dua Lipa – Dua Lipa |last=Barker ==音樂錄影帶==
===背景===
{{Quote box|width=30%|align=left|「（我想要在《愛情守則》的音樂錄影帶）展示你和你朋友是互相幫助的。你朋友就是守則，他們就是防止你與那人發生錯誤的那種。」|—黎波談及音樂錄影帶的主題<ref name="bbvd" />}}
《愛情守則》的音樂錄影帶由亨利·斯科菲爾德執導，他還曾拍攝黎波上一首歌曲《{{le|愛的光芒|Lost in Your Light}}》的錄影帶。她為了討論對於影片的想法，邀請斯科菲爾德在倫敦喝茶。黎波儲存了一張照片作爲「參考點」，照片是1980年代廣告中模特[[娜歐蜜·坎貝兒|娜歐蜜·坎貝兒]]依靠在另一位模特的背上，她解釋道：「我很愛女孩們像這樣子互相照顧對方的這個想法，互相守護，那種謙遜的感覺，那種充滿力量的感覺。」黎波還有一個想法是將一群[[火烈鸟|紅鶴]]放進影片中，她認爲它們很好的代表了女性友誼。導演打給德蕾莎·巴塞羅設計一套舞蹈動作並要將「親密無間和權利的敘述」帶入其中<ref name="The Cut">{{cite web|url=https://www.thecut.com/2017/07/dua-lipa-new-rules-video-interview.html|title=Dua Lipa Takes Us Behind the Scenes of the 'New Rules' Video|author=Dayna Evans|work=The Cut|date=2017-07-11|accessdate=2017-07-14}}</ref>。巴塞羅還在策劃舞蹈時以紅鶴的動作為靈感<ref name="bbvd" />。超過200個女性為影片試鏡，最終有八個獲選<ref name="bbvd" />。

音樂錄影帶于2017年6月拍攝，並在邁阿密海灘的知己酒店取景，該酒店為[[凱悅酒店|凱悅]]無極限連鎖品牌的一部分<ref name="bbvd" />。凱悅國際品牌的[[副總統|副總統]]桑德拉·米采克認爲黎波和酒店之間的這場合作對她來説是合理的，因爲「（知己酒店的）靈感和名字是來自于要做一個值得信任的朋友的觀念」<ref name="ad"/>。關於拍攝，斯科菲爾德回應道攝影機在舞蹈過程中相當接近，以便能拍攝出更多持續的單鏡<ref name="The Cut"/>。黎波在接受全國公共廣播電台采訪時表示：「我想要在這部影片中傳達出女性在形勢中互相支持、互相幫助和互相照顧的那種團結和親密無間。」<ref name="npr" />音樂錄影帶于2017年7月7日首映于黎波的[[Youtube|Youtube]]賬戶<ref name="rs">{{cite web |last=Reed |first=Ryan |url=http://www.rollingstone.com/music/news/watch-dua-lipa-walk-on-water-strut-around-hotel-in-new-rules-video-w491317 |title=Watch Dua Lipa Walk on Water, Strut Around Hotel in 'New Rules' Video |work=[[Rolling_Stone|Rolling Stone]] |date=2017-07-07 |accessdate=2017-07-14}}</ref>。在其之後，凱悅發行了兩部拍攝的幕後花絮短片<ref name="ad">{{cite web |url=http://www.adweek.com/creativity/this-buzzy-pop-singers-new-video-has-a-notable-costar-hyatts-confidante-hotel/ |title=This Buzzy Pop Singer's New Video Has a Notable Costar: Hyatt's Confidante Hotel |last=Stanley |first=T.L. |date=2017-07-13 |work=Adweek |accessdate=2017-08-24}}</ref>。

===概要===
[[File:Dua_Lipa_New_Rules_music_video.jpg|right]]
錄影帶一開始展示了知己酒店的外景和兩隻紅鶴。在下一個鏡頭，並在歌曲的環境中，黎波躺在床上，想著應該給她前男友第二次機會，而她的朋友則在其旁邊。當這首歌開始時，黎波朝向電話並想接起男友的來電，但是遭到其中一個朋友制止。接著，她不斷接受著朋友的建議，但是打算逃向大廳<ref name="rs"/>。在那裏，她們表演了舞蹈，並返回房間内。下一個鏡頭，那些女生們有一場睡衣派對，她們互相為對方梳頭髮和塗口紅<ref name="npr" />。她們接著換了衣服並走向泳池，在那裏她們走在水面上。

在歌曲的橋段部分之後，其中一個女生正打算與其前男友複合，但遭到黎波阻止，並給她一些在影片開始時所收穫的建議。接著，這群女生在泳池前表演舞蹈，其中她們「在一個牢不可破、以手臂搭出的環形中彼此靠近」。這是黎波和斯科菲爾德最喜歡的一幕，而後者説道：「這概括了我整個故事。觀衆在熒幕外面，但是那場舞蹈將你拉入她們其中，並讓你也成爲這場的關係的一份子。」<ref name="The Cut"/>影片最後女生們站在泳池前，而攝影機翻轉並展示了一群紅鶴站在同樣的位置。
