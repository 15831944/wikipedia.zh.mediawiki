'''愛麗絲·沃爾頓'''（{{Lang-en|Alice Walton}}；{{bd|1949年|10月7日| | }}），出生在[[美國|美國]][[阿肯色州|阿肯色州]][[紐波特|紐波特]]，畢業於[[聖安東尼奧三一大學|聖安東尼奧三一大學]]，2019年[[富比士|富比士]]世界億萬富翁排行榜中排名第17位，身價達444億[[美元|美元]]。<ref>{{Cite web |url = https://www.forbes.com/profile/alice-walton/?list=billionaires#212d00ba4eb2 | title =#17 Alice Walton | publisher = Forbes |language =en |accessdate = 2019-03-20 }}</ref><ref name = ' Alice ' >{{cite news| title=全球最富有的女人：68歲的她單身一人身價超過2900億元| url = http://money.jrj.com.cn/2018/06/19104224694519.shtml | accessdate=2018-12-29 |agency=金融界盈利寶|date=2018-06-19}}</ref>

愛麗絲·沃爾頓是[[沃爾瑪|沃爾瑪]]創始人[[山姆·沃爾頓|山姆·沃爾頓]]的獨生女，但她沒有參與父親的公司沃爾瑪運營，而是投入[[藝術|藝術]]愛好。麗絲·沃爾頓曾離過兩次婚，沒有子女。<ref>{{Cite web |url = https://prabook.com/web/alice_louise.walton/468418|title =Alice Louise Walton|publisher = prabook |language =en |accessdate = 2018 - 12-29 }}</ref>

==藝術愛好==
愛麗絲·沃爾頓是一位藝術愛好者，她擁有價值5億美元的藝術收藏品。<ref name="Alice2" />

在2005年為了慶祝美國藝術和教育，她於[[阿肯色州|阿肯色州]]創建了造價5000萬美元的[[水晶橋藝術博物館|水晶橋藝術博物館]]（{{Lang|en|Crystal Bridges Museum of American Art}}），館中收藏有50,000冊的藝術參考資料庫。<ref name="Alice2" /><ref>{{Cite web|url=https://news.artron.net/20110803/n181511.html|title=水晶桥美国艺术博物馆获将免费开放|accessdate=2019-01-01|author=|date=|work=news.artron.net|publisher=|agency=雅昌新闻}}</ref>

愛麗絲沃爾頓從[[紐約公共圖書館|紐約公共圖書館]]購買了價值3,500萬美元的{{Tsl|en|Asher Brown Durand|阿什·布朗·杜蘭}}的經典畫作——{{link-en|Kindred Spirites|Kindred Spirits (painting)}}。她也在[[蘇富比|蘇富比]]的拍賣會上購買了[[大衛·霍克尼|大衛·霍克尼]]和麗塔·E·弗里德的一系列作品，價值2,000萬美元。<ref name="Alice" /><ref name="Alice2">{{Cite web |url = https://moneyinc.com/10-things-didnt-know-alice-walton/ | title =10 Things You Didn't Know About Alice Walton | publisher = MoneyInc |language =en |accessdate = 2018 -12-29 }}</ref>

==慈善事業==
[[沃爾頓家族|沃爾頓家族]]的基金會主要由愛麗絲·沃爾頓經營，該基金會支持[[密西西比河三角洲|密西西比河三角洲]]和阿肯色州的經濟發展以及環境保護。1994年，阿肯色州修建機場缺乏資金，愛麗絲·沃爾頓捐出了7,950萬美元幫助。因此阿肯色州機場的航站樓以她的名字命名。另外沃爾頓家族基金會也捐助[[阿肯色大學|阿肯色大學]]捐款3億美元以支持教育事業。<ref name = ' Alice2 '/><ref >{{cite news| title=全球最幸運的女人：身價3100億超過馬雲，玩著就成了女首富| url = http://www.sohu.com/a/270367310_100280378 | accessdate=2018-12-29 |agency=搜狐|date=2018-10-21}}</ref>
