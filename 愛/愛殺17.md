== 故事大綱 ==

17岁的夏天，美好的青春，毫无征兆地刹那间结束……

青春的17岁，因为，爱，意外出现了杀机，在17岁的夏天，在刹那间告别……

徐宜真，一個無憂無慮的青春少女，當双胞胎妹妹宜靜意外死亡后，她試著在真爱的抚慰下走出傷痛...却无意中发觉妹妹的早逝其实只是某些血色真相的冰山一角...

鄒克強，一个改邪归正的回头浪子，努力想給弟弟鄒克傑跟自己一片新蓝天，却突然弟弟克傑蒙冤成為杀人犯而含冤自杀。恨之入骨的他誓言血债血还洗清冤枉；唯一能平息他胸口疯狂怒火的人，卻是被害者的姊姊宜真...

17岁少女徐宜靜的意外死亡，原本伤痛似乎可以由时间缓解；而當這個看似简单的謎團逐漸被抽絲剝繭、多年谎言逐渐破灭、盘根错节的各种真相慢慢被拼湊还原之後，眾人能接受极端殘酷的真相嗎？

== 爭議 ==
本劇在中視播出第一集時，因內容涉及[[血|血腥]]與[[暴力|暴力]]，而遭投訴「內容過於血腥暴力，會教壞小孩」；因此，中視在播第二集時，以「基於專業素養與社會良心」為由，自行刪除約40分鐘的戲。

之後，本劇在八大綜合台播出時，[[國家通訊傳播委員會|國家通訊傳播委員會]]（NCC）對本劇處以[[新台幣|新台幣]]20萬元[[罰鍰|罰鍰]]；理由是本劇內容涉及校園恐嚇、勒索，[[網路聊天|網路聊天]]內容談到[[強姦|強姦]]、[[殺人|殺人]]、[[援交|援交]]與[[自殺|自殺]]，恐對[[青少年|青少年]]有不良影響。該劇編劇小組成員陳昭如認為，中視的處理方式「粗暴地肢解並扭曲原創者的作品」，NCC的處理方式「實在讓我們對台灣的[[言論自由|言論自由]]與創作環境感到寒心」。

2006年7月13日，本劇劇組召開[[記者會|記者會]]，批評NCC「開倒車重現[[電檢|電檢]]制度」；江豐宏揚言，若NCC堅持罰款，他們將到NCC抗議。

== 金鐘獎入圍 ==
在2006年臺灣[[第41屆金鐘獎|第41屆金鐘獎]]入圍名單中，本劇一舉獲得七項入圍。本劇獲入圍最佳戲劇節目獎；張韶涵以雙重角色的演繹，首度問鼎戲劇節目最佳女主角獎；飾演女主角母親的沈時華也出現在戲劇節目最佳女配角入圍名單中。另外，其他的入圍獎項還包括戲劇節目編劇、剪輯、音效、燈光。這些獎項的入圍不僅給予劇組肯定，也為此劇之前被NCC罰款的事件出了一口氣。
