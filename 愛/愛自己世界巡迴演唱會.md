'''愛自己世界巡迴演唱會'''（{{lang-en|BTS World Tour：Love Yourself}}）是[[韓國|韓國]][[男子音樂組合|男子音樂組合]][[防彈少年團|防彈少年團]]的第三次世界巡迴演唱會，以宣傳他們的新主題「Love Yourself」的系列專輯<ref>{{Cite news|url=https://www.billboard.com/articles/columns/k-town/8390251/bts-announce-new-tour-dates-full-list|title=BTS Announces World Tour With Stops In Six North American Cities|work=Billboard|accessdate=2018-04-26}}</ref>，包括《[[Love_Yourself_承_'Her'|Love Yourself 承 'Her']]》、《[[Love_Yourself_轉_'Tear'|Love Yourself 轉 'Tear']]》和《[[Love_Yourself_結_'Answer'|Love Yourself 結 'Answer']]》等。巡演於2018年8月25日在韓國[[首爾奧林匹克主競技場|首爾奧林匹克主競技場]]開始，官方於2019年2月19日再次增加場數，總計52場。北美場及歐洲場於2019年3月1日售票，2019年3月11日巴西場售票，洛杉磯、芝加哥、紐澤西、倫敦、巴黎及聖保羅均售罄，各再追加1日，總計58場演出。

== 背景 ==
2017年在19座城市共40場演唱會門票全席售罄的《2017 BTS Live Trilogy：Episode III The Wings Tour》，是2014年開始的「Trilogy Episode」系列收尾，團隊最終以2017年12月10日於韓國首爾[[高尺天空巨蛋|高尺天空巨蛋]]作為「The Wings Tour」為期近一年的終場演出。「Trilogy Episode」系列正式落幕後，[[Big_Hit娛樂|Big Hit娛樂]]在2018年4月27日宣布防彈少年團將從2018年8月底開始進行新的名為「Love Yourself」的巡迴演唱會（BTS World Tour：Love Yourself）以宣傳「Love Yourself」系列專輯，並透過防彈少年團官方粉絲俱樂部網站、[[影音分享網站|影音分享網站]]和[[社群網站|社群網站]]公開世界巡演的預告海報及影片，展示了11座城市舉辦的22場演唱會的初始時間表，此次巡演在韓國首演後將陸續造訪[[美國|美國]]、[[加拿大|加拿大]]、[[英國|英國]]、[[荷蘭|荷蘭]]、[[德國|德國]]、法國共11座城市，而除了既定日程之外，所屬經紀公司也表示，確定追加的城市場次將在之後公告。<ref>{{cite web |url=https://www.koreastardaily.com/tc/news/104956 |title=防彈少年團《LOVE YOURSELF》巡演日程首波公開！ 北美&歐洲22場確定 |language=zh |publisher=韓星網 |date=2018-04-27 |accessdate=2018-06-25 }}</ref>

隨著首場演出場地公開為[[首爾奧林匹克主競技場|首爾奧林匹克主競技場]]，這標誌著防彈少年團在出道第五年首次在該場地單獨開演，從出道1年4個月的2014年10月18日，在首爾[[廣津區|廣津區]]{{tsl|ko|광장동|廣壯洞}}兩千人規模的{{tsl|ko|악스코리아|AX-KOREA}}舉行首場單獨演唱會《BTS 2014 Live Trilogy：Episode II. The Red Bullet》。2015年3千人規模的[[奧林匹克公園_(首爾)|奧林匹克大廳]]以及5千人規模的[[SK奧林匹克手球館|SK奧林匹克手球館]]單獨演唱會，2016年5月在[[首尔奥林匹克体操竞技场|首尔奥林匹克体操竞技场]]舉行單獨演唱會《花樣年華 On Stage: Epilogue》，再到2017年防彈少年團來到可容納2萬2千人的高尺天空巨蛋，一步一腳印地拓寬了演唱會規模，證明了粉絲群體的擴增。另外，防彈少年團是在奧林匹克主競技場舉辦單獨演唱會的第12組[[韓國流行音樂|K-Pop]]歌手，在此之前有[[H.O.T.|H.O.T.]]（1999、2001）、[[神話_(組合)|神話]]（2005）、[[g.o.d|g.o.d]]（2001、2014）、[[趙容弼|趙容弼]]（2003、2005、2008、2009、2010、2018）、[[東方神起|東方神起]]（2005、2006）、{{tsl|ko|이승환 (가수)|李承桓 (1965年生)|李承桓}}（2007）、{{tsl|ko|이승철|李承哲 (1966年生)|李承哲}}（2010）、[[JYJ|JYJ]]（2010、2014）、[[李文世|李文世]]（2013）、[[徐太志|徐太志]]（2014、2017）、[[EXO|EXO]]（2017）。<ref>{{cite web |url=http://www.newsen.com/news_view.php?uid=201805301708480410 |title=[뮤직와치]빌보드 휩쓴 방탄, 5주년 자축파티→6월중순 공식활동 마무리 |language=ko |publisher=newsen |date=2018-05-30 |accessdate=2018-06-25 }}</ref>

== 商業表現 ==
根據《體育朝鮮》報導，《Love Yourself》巡迴演唱會於2018年5月5日至6月1日進行歐美場次的門票販售，巡演規模在北美地區共18萬席（14場演唱會），以及歐洲地區10萬席（7場演唱會），總計28萬席規模，而門票在開始預售的同時全數完售，包含了美國（洛杉磯、奧克蘭、沃斯堡、紐華克、芝加哥）、加拿大（哈密爾頓）、英國（倫敦）、荷蘭（阿姆斯特丹）、德國（柏林）、法國（巴黎），共10座城市21場演唱會預售票全數售罄<ref>{{cite web |url=https://entertain.naver.com/read?oid=076&aid=0003265484 |title=방탄소년단의 위엄…북미·유럽 지역 투어 28만석 전석 매진 |language=ko |publisher=體育朝鮮 |date=2018-06-03 |accessdate=2018-07-08 }}</ref>。其中洛杉磯的場次由於大量的門票需求，促使額外場次被新增，增加的場次日期為2018年9月9日，門票於5月9日銷售。<ref name="BTS NA Tour LA day 4">{{Cite news|url=http://metro.co.uk/2018/05/06/bts-listen-army-add-extra-date-tour-tickets-sell-seconds-7524733/|title=BTS listen to their Army and add extra date to tour after tickets sell out|date=2018-05-06|work=Metro|accessdate=2018-05-09|language=en-GB}}</ref>

另外，據《韓國聯合通訊社》媒體透露，在2018年10月16日和17日於德國柏林的{{tsl|en|Mercedes-Benz Arena (Berlin)|梅賽德斯-賓士競技場 (柏林)|梅賽德斯-賓士競技場}}舉行的演出，超過3萬張門票被售出，德國日報《[[世界报_(德国)|世界報]]》則表示，兩場演唱會門票在[[歐洲中部夏令時間|歐洲中部夏令時間]]6月1日開始售票後9分鐘即宣告售罄。演唱會門票從當天早上9時開始，透過網路、電話和演唱會場地現場進行銷售。因為一部分門票是在現場銷售，所以從凌晨開始就有數百名歌迷蜂擁而至。曾在現場的韓國僑胞在電話訪問中表示「雖然在現場排隊的時候想要同時用電話和網路購票，但是因為粉絲們一下子湧入售票網站，所以連線卡住、當掉了」他還說道「有些在現場沒買到票的粉絲們也哭了」。而門票最低價為73歐元，最高價349歐元。據悉，演唱會門票完售後，在黑市的最高交易價為500歐元。<ref>{{cite web |url=http://entertain.naver.com/read?oid=001&aid=0010123499 |title=방탄소년단 베를린 공연티켓 3만장 9분만에 매진 |language=ko |publisher=韓國聯合通訊社 |date=2018-06-01 |accessdate=2018-07-08 }}</ref>
