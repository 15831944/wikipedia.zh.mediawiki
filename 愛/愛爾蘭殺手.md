{{作品名稱|name=''The Irishman''|hk=愛爾蘭殺手|cn=爱尔兰人|equiv-tw=hk}}是一部2019年美國[[史詩片|史詩]][[傳記片|傳記]][[犯罪電影|犯罪電影]]，由[[馬田·史高西斯|馬田·史高西斯]]執導，{{link-en|史蒂芬·澤里安|Steven Zaillian}}編劇，改編自查爾斯·布朗特（Charles Brandt）的書籍《聽說你刷房子了》（''I Heard You Paint Houses''）。<ref>{{Cite web |url = http://variety.com/2017/film/news/netflix-buys-martin-scorseses-the-irishman-starring-robert-de-niro-1201993446/ |title = Netflix Buys Martin Scorsese’s ‘The Irishman’ Starring Robert De Niro |author = Dave McNary |date = 2017-02-21 |publisher = Variety |language = en}}</ref>本片主演是[[羅拔·迪尼路|羅拔·迪尼路]]，飾演[[法蘭克·希蘭|法蘭克·希蘭]]，是一名勞工工會負責人和被指控為{{link-en|布法利諾犯罪家族|Bufalino crime family}}工作的殺手，而[[阿爾·柏仙奴|阿爾·柏仙奴]]飾演[[吉米·霍法|吉米·霍法]]。其他演員有[[祖·柏斯|祖·柏斯]]、[[波比·簡拿威|波比·簡拿威]]、[[夏菲·基圖|夏菲·基圖]]和[[雷·羅曼諾|雷·羅曼諾]]。本片是[[馬田·史高西斯與羅拔·迪尼路|迪尼路與史高西斯的第九次合作的電影]]、迪尼路與柏仙奴一起演出的第四部電影（之前是《[[教父2|教父2]]》、《[[盜火線|盜火線]]》和《[[世紀交鋒|世紀交鋒]]》），以及柏仙奴與史高西斯第一次合作的電影。

本片於2017年12月在[[紐約市|紐約市]]開始拍攝，至2018年3月殺青，其製作成本高達2億美元，是馬田·史高西斯的生涯中最昂貴的項目。

本片定於2019年末由[[Netflix|Netflix]]負責在戲院上映，然後是數位串流播放。

== 劇情 ==
與黑幫有聯絡的勞工工會官員[[法蘭克·希蘭|法蘭克·希蘭]]回憶起他參與殺害一名美國勞工工會領袖[[吉米·霍法|吉米·霍法]]的故事。

== 製作 ==
=== 開發 ===
[[馬田·史高西斯|馬田·史高西斯]]長期以來一直想執導《愛爾蘭殺手》，由[[羅拔·迪尼路|羅拔·迪尼路]]、[[祖·柏斯|祖·柏斯]]和[[阿爾·柏仙奴|阿爾·柏仙奴]]主演。2014年9月，柏仙奴確認本片正在開發中，這將會是史高西斯的《[[沉默_(電影)|沉默]]》之後的下一個項目。[[波比·簡拿威|波比·簡拿威]]也加入為演員。<ref>{{Cite web |url = http://www.slashfilm.com/al-pacino-the-irishman-martin-scorsese/ |title = Al Pacino Says Martin Scorsese's ‘The Irishman’ Still Happening; Bobby Cannavale Joins Cast |author = Angie Han |date = 2014-09-09 |publisher = slashfilm.com |language = en}}</ref>2015年10月，迪尼路表示本片仍然實行，可能會在2016年與史高西斯拍攝，確認{{link-en|史蒂芬·澤里安|Steven Zaillian}}會編寫劇本。<ref>{{Cite web |url = http://www.digitalspy.com/movies/news/a671493/robert-de-niro-says-hell-be-reuniting-with-martin-scorsese-next-year-for-i-heard-you-paint-houses/#~ppT2Mgjw6P9ztR |title = Exclusive: Robert De Niro says he'll be reuniting with Martin Scorsese next year for I Heard You Paint Houses |author = Ali Plumb |date = 2015-10-01 |publisher = digitalspy |language = en}}</ref><ref>{{Cite web |url = http://variety.com/2015/scene/news/martin-scorsese-thierry-fremaux-lyon-lumiere-fest-1201619829/ |title = Martin Scorsese Talks ‘Vinyl,’ ‘The Departed’ at Lyon Lumiere Fest |author = Elsa Keslassy |date = 2015-10-16 |publisher = Variety |language = en}}</ref>

=== 選角 ===
2017年7月，柏斯和柏仙奴正式加入劇組，而[[雷·羅曼諾|雷·羅曼諾]]也加入，[[鮑比·坎納瓦爾|鮑比·坎納瓦爾]]和[[夏菲·基圖|夏菲·基圖]]在最後談判中。柏斯在同意加入前據報他拒絕了50次。2017年9月，[[傑克·休斯頓|傑克·休斯頓]]、[[史提芬·格拉漢姆|史提芬·格拉漢姆]]、[[多門尼克·隆巴多西|多門尼克·隆巴多西]]、{{link-en|傑里米·盧克|Jeremy Luke}}、Joseph Russo、<ref>{{Cite web |url = http://deadline.com/2017/09/martin-scorseses-the-irishman-domenick-lombardozzi-netflix-1202169083/ |title = Martin Scorsese's 'The Irishman' Adds Domenick Lombardozzi & More |author = Amanda N'Duka |date = 2017-09-13 |publisher = Deadline Hollywood |language = en}}</ref>{{link-en|凱薩琳·納杜奇|Kathrine Narducci}}<ref>{{Cite web |url = http://deadline.com/2017/09/kathrine-narducci-martin-scorsese-netflix-the-irishman-1202173404/ |title = Kathrine Narducci Cast In Martin Scorsese's Crime Drama ‘The Irishman’ |author = Amanda N'Duka |date = 2010-09-19 |publisher = Deadline Hollywood |language = en}}</ref>、Danny Abeckaser<ref>{{Cite web |url = http://www.thewrap.com/martin-scorseses-the-irishman-casts-former-nightlife-kingpin-danny-a-abeckaser-exclusive/ |title = Martin Scorsese's ‘The Irishman’ Casts Former Nightlife Kingpin Danny A Abeckaser (Exclusive) |author = Matt Donnelly |date = 2017-09-20 |publisher = TheWrap |language = en}}</ref>、{{link-en|J·C·麥肯齊|J. C. MacKenzie}}和Craig Vincent加入劇組。10月，{{link-en|蓋瑞·巴薩拉巴|Gary Basaraba}}、[[安娜·派昆|安娜·派昆]]、Welker White和[[傑西·普萊蒙|傑西·普萊蒙]]加入劇組。

=== 編劇 ===
2017年7月，Zack Sharf報導說，本片的劇本由{{link-en|史蒂芬·澤里安|Steven Zaillian}}改編。

=== 拍攝 ===
拍攝於2017年8月在[[紐約市|紐約市]]內和周邊地區開始，並將持續至2017年12月。<ref>{{Cite web |url = http://deadline.com/2017/07/joe-pesci-the-irishman-al-pacino-robert-de-niro-martin-scorsese-deal-1202126751/ |title = Joe Pesci Joins Martin Scorsese's ‘The Irishman’; Others In Talks |author = Anita Busch |date = 2017-07-12 |publisher = Deadline |language = en}}</ref><ref>{{Cite web |url = http://www.hollywoodreporter.com//heat-vision/ray-romano-joins-robert-de-niro-al-pacino-martin-scorseses-irishman-1021007 |title = Ray Romano Joins Robert De Niro, Al Pacino in Martin Scorsese's 'The Irishman' (Exclusive) |author = Borys Kit |date = 2017-07-13 |publisher = The Hollywood Reporter |language = en}}</ref>本片的[[主要攝影|主要攝影]]於2017年8月29日在[[曼克頓|曼克頓]]的{{link-en|烏節街 (曼克頓)|Orchard Street (Manhattan)|烏節街}}開始。<ref>{{Cite web |url = http://www.boweryboogie.com/2017/08/martin-scorsese-filming-irishman-orchard-street-today-de-niro-pacino-keitel-pesci/ |title = Martin Scorsese Filming ‘The Irishman’ on Orchard Street Today (with De Niro, Pacino, Keitel, and Pesci) |date = 2017-08-29 |publisher = Bowery Boogie |language = en}}</ref><ref>{{Cite web |url = http://www.darkhorizons.com/filming-begins-on-scorseses-irishman/ |title = Filming Begins On Scorsese's “Irishman” |author = Garth Franklin |date = 2017-08-30 |publisher = darkhorizons.com |language = en}}</ref>電影在2018年3月5日拍攝完畢。<ref>{{Cite web |url = http://collider.com/martin-scorsese-the-irishman-wraps-filming/ |title = Filming on Martin Scorsese's ‘The Irishman’ Has Wrapped |author = Matt Goldberg |date = 2018-03-05 |publisher = Collider |language = en}}</ref>

=== 融資和預算 ===
在2016年5月，宣布電影的版權將在[[第69屆坎城影展|2016年坎城影展]]上出售，而墨西哥的金融家/製片人Fábrica de Cine出價1億美元資助本片，並透過這筆交易，[[派拉蒙影業|派拉蒙影業]]可保留權利。<ref name="May2016V">{{Cite web |url = http://variety.com/2016/film/markets-festivals/cannes-martin-scorseses-the-irishman-has-buyers-frustrated-and-salivating-exclusive-1201773763/ |title = Cannes: Martin Scorsese's ‘The Irishman’ Has Buyers Frustrated and Salivating (EXCLUSIVE) |author = Brent Lang |date = 2016-05-13 |publisher = Variety |language = en}}</ref>{{link-en|IM Global|IM Global}}也投標本片的國際銷售權。<ref name="May2016V" />{{link-en|STX Entertainment|STX Entertainment}}以5000萬美元的價格購得了本片的國際發行權，擊敗了其他製片商，例如[[環球影業|環球影業]]、[[二十世紀福斯|二十世紀福斯]]和[[獅門娛樂|獅門娛樂]]，而Fabrica de Cine則結束了這筆交易，派拉蒙保留了其版權。<ref>{{Cite web |url = http://deadline.com/2016/05/martin-scorsese-the-irishman-stx-huge-deal-mob-films-cannes-1201756232/ |title = Martin Scorsese's Mob Film ‘The Irishman’ In Major Deal To STX After Heated Bidding War — Cannes |date = 2016-05-15 |publisher = Deadline |language = en}}</ref>

2017年2月，據報導說派拉蒙影業公司已經放棄了《愛爾蘭殺手》的發行權，接著宣布墨西哥金融家Fabrica de Cine由於預算上升而不會融資本片。[[Netflix|Netflix]]然後以1.05億美元購買了本片，並同意為電影提供巨額1.25億美元的預算資金，安排在2019年上映。<ref>{{Cite web |url = http://deadline.com/2017/02/martin-scorsese-the-irishman-acquired-netflix-robert-de-niro-1201931848/ |title = Martin Scorsese's Robert De Niro Pic ‘The Irishman’ Heading To Netflix? |author = Liz Calvario & Anita Busch |date = 2017-02-21 |publisher = Deadline |language = en}}</ref><ref>{{Cite web |url = https://thefilmstage.com/news/netflix-picks-up-martin-scorseses-the-irishman-now-aiming-for-2019-release/ |title = Netflix Picks Up Martin Scorsese's ‘The Irishman,’ Now Aiming for 2019 Release |author = Jordan Raup |date = 2017-02-21 |publisher = The Film Stage |language = en}}</ref>

=== 視覺效果 ===
[[光影魔幻工業|光影魔幻工業]]與VFX監製人{{link-en|勞勃·勒加托|Robert Legato}}負責處理本片的特效<ref>{{Cite web |url = https://www.indiewire.com/2018/04/martin-scorsese-ang-lee-digital-aging-the-irishman-gemini-man-1201952224/ |title = Martin Scorsese's ‘The Irishman’ and Ang Lee's ‘Gemini Man’ Are Changing the Face of Digital De-Aging |author = Bill Desowitz |date = 2018-04-13 |publisher = IndieWire |language = en}}</ref>。2018年3月5日，報導說本片的預算從1.25億美元增加到1.4億美元，很大程度上是因為在整個電影的不同時刻上需要使用視覺特效把迪尼路、柏斯和柏仙奴變回年輕時的樣子<ref name="budget" />。到那年8月，成本已經增加到1.75億美元，到後期製作時已達到2億美元，成為史高西斯的職業生涯中最貴的電影。<ref>{{Cite web |url = https://www.independent.ie/entertainment/movies/movie-news/betting-on-the-irishman-37179415.html |title = Betting on The Irishman |author = Tristram Fane Saunders |date = 2018-08-04 |publisher = Irish Independent |language = en}}</ref><ref>{{Cite web |url = https://deadline.com/2019/01/hollywood-gangster-films-return-in-2019-mob-history-movies-1202528574/ |title = Peter Bart: Gangster Movies Return In Big Way In 2019, Awakening Genre From Its Hollywood Dirt Nap |author = Peter Bart |date = 2019-01-03 |publisher = Deadline Hollywood |language = en}}</ref>

== 發行 ==
根據羅拔·迪尼路說，本片會於2019年後期在戲院上映，但會是[[有限上映|有限上映]]。<ref>{{Cite web |url = https://www.livemint.com/Consumer/EwgR27cG0UupPEDmrx2C9O/Netflixs-The-Irishman-to-be-released-in-limited-theatres.html |title = Netflix's ‘The Irishman’ to be released in limited theatres, says Robert De Niro |date = 2018-12-04 |publisher = LiveMint |language = en}}</ref>2019年2月後期，據報導說在史高西斯的要求下，Netflix可能會給本片廣泛公映<ref>{{Cite web |url = https://screenrant.com/irishman-movie-theatrical-release-netflix-scorsese/ |title = Martin Scorsese's The Irishman May Get Wide Theatrical Release |author = Chris Agar |date = 2019-02-27 |publisher = Screen Rant |language = en}}</ref>。幾間影院連鎖店的負責人，包括去年11月拒絕播放《[[羅馬_(2018年電影)|羅馬]]》的[[AMC電影院|AMC電影院]]的Adam Aron，都說如果Netflix「尊重幾十年前的舊戲院窗口期，提出把電影首先搬到影院播放幾個月，然後搬回家裡播放。」他們才會公映《愛爾蘭殺手》。
