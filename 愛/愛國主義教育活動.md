'''愛國主義教育活动'''，是[[中华人民共和国|中华人民共和国]]在[[六四事件|六四事件]]後發起針對青少年的[[宣傳|宣傳]]與[[教育|教育]]運動及政策<REF name="AutoB3-1"/><REF name="AutoB3-2"/>。此政策詞彙由[[六四事件|六四事件]]後出任[[中共中央總書記|中共中央總書記]]的[[江澤民|江澤民]]宣告並執行，如其1990年對北京青年發表對紀念[[五四|五四]]運動的演說，宣告「在当代中国[[爱国主义|爱国主义]]与人民民主即[[社会主义|社会主义]]民主，本质上也是统一的」。<REF name="AutoB3-3"/>江於1989年對工人發表演說時，也強調工人的責任在於教育社會愛國主義、社會主義、集體主義、打造國家等來激發民族驕傲與自信。<REF name="AutoB3-4"/>。中國1990年代爱国主义教育系列活動，承續中國共產黨進行文化清理（cultural purge）的宣傳及教育組織對「不當文化」的清理<REF name="Davis2012"/>。此爱国主义教育系列活動為中國共產黨在政治意識型態的宣傳和教育名稱和內容上，從以黨主導的[[共產主義|共產主義]]到以[[國家|國家]]主導的務實[[國族主義|國族主義]]政策。<REF name="Zhao2004"/>

[[德育及國民教育科|德育及國民教育科]]課程在香港於2012年引發爭論與街頭示威抗議，也引發對於愛國主義教育的內容及實施方式的討論。<REF name="AutoB3-5"/><REF name="AutoB3-6"/>，爭議點在於愛國主義教育是否只是單純的像各國推行的國民教育還是政治宣傳洗腦。<REF name="AutoB3-7"/><REF name="AutoB3-8"/>

2016年《關於教育系統深入開展愛國主義教育的實施意見》要求將愛國主義精神融入中國大、中、小學的德育、語文、歷史、地理、體育和藝術等各學科課程標準、教材和考試中，愛國主義教育活動有[[新媒體|新媒體]]宣傳並輸出海外<ref>[https://theinitium.com/article/20160211-dailynews-Patriotism/ 除了用新媒體宣傳，中國還要將愛國主義教育輸出海外]</ref>，也「加大对香港、澳门和台湾青少年学生的爱国主义教育力度」<ref>[http://www.moe.gov.cn/srcsite/A13/s7061/201601/t20160129_229131.html 中共教育部党组关于教育系统深入开展爱国主义教育的实施意见]</ref><ref>[http://www.appledaily.com.tw/realtimenews/article/new/20160216/797065/ 中共下令強化台港澳「愛國」教育　倡愛黨愛國]</ref>。

== 背景 ==
在[[六四事件|六四事件]]後，時任中共中央總書記[[江澤民|江澤民]]發起並推動並制度化的一系列以“愛國主義教育”為主體的教育宣传活動<REF name="HasegawaTogo2008"/>，目標是用[[國族主義|國族主義]]替換[[共產主義|共產主義]]的意識型態，來建立在[[六四事件|六四事件]]前後[[中國共產黨|中國共產黨]]被挑戰的执政正當性，<REF name="Zhao1998"/><REF name="Hughes2005"/>承續中國共產黨由宣傳及教育組織對「不當文化」所進行文化清理的相關作法<REF name="Davis2012"/>。

此政策詞彙由[[六四事件|六四事件]]後出任[[中共中央總書記|中共中央總書記]]的[[江澤民|江澤民]]宣告並執行，江澤民於1989年對工人發表演說時，也強調工人的責任在於教育社會愛國主義、社會主義、集體主義、打造國家等來激發民族驕傲與自信。<REF name="AutoB3-9"/><REF name="AutoB3-10"/><REF name="AutoB3-11"/>。1990年5月3日，[[江泽民|江泽民]]在纪念[[五四運動|五四運動]]時以《爱国主义和我国知识分子的使命》演說時，指出[[社會主義|社會主義]]及[[愛國主義|愛國主義]]「本質上是同一的」<REF name="AutoB3-12"/>，這些政策宣示說明了[[六四事件|六四事件]]後[[國族主義|國族主義]]替換[[共產主義|共產主義]]的意識型態<ref name="Zhao1998"/>。是中共從改革開放以來、特別是[[六四事件|六四事件]]之後主張[[民族主義#國家民族主義|國家民族主義]]和[[民族主義#文化民族主義|文化民族主義]]的愛國主義教育，和原標榜[[民族主義#階級民族主義|階級民族主義]]的愛國主義教育不同<REF name="AutoB3-13"/>

愛國主義教育是一種有选择的历史教育，即一方面突出曾經的外国侵略者的残暴，一方面忽略中华人民共和国领导人自身的一些暴行和错误。愛國主義教育目的在于培养[[民族主义|民族主义]]的、反[[西方|西方]]的受害者心态，以增强中共执政的合法性<REF name="AutoB3-14"/>。

亦有中國学者认为，中华人民共和国的「爱国主义教育思想体系」則是「承前启后」的歷經三代（[[毛泽东|毛泽东]]、[[邓小平|邓小平]]、[[江泽民|江泽民]]）马克思主义爱国主义教育理论；<REF name="三代"/>或主張愛國主義教育是[[江澤民|江澤民]]的繼毛澤東及鄧小平相關思想的創新性論述<REF name="AutoB3-15"/>。

== 內容 ==
中共倡导的愛國主義教育內容雖包含廣泛，目前可见主導活動有兩大主題: 维护国家统一和领土完整<ref name="Zhao1998"/><REF name="毛泽东理论特色"/><REF name="AutoB3-16"/><ref name="三代">{{cite journal| title=论毛泽东、邓小平、江泽民爱国主义教育思想的特点| author=张宝林| journal=教育探索| year=2006| volume=8| url=http://file.lw23.com/0/07/07d/07db56e7-8829-4c4c-817e-e8cc2e60bae0.pdf| deadurl=yes| archiveurl=https://web.archive.org/web/20141006082345/http://file.lw23.com/0/07/07d/07db56e7-8829-4c4c-817e-e8cc2e60bae0.pdf| archivedate=2014-10-06}}</ref>，有學者主張[[毛泽东|毛泽东]]思想在维护国家统一和领土完整的「爱国主义传统具有重要的指导意义」。<ref name="毛泽东理论特色"/>

愛國主義教育中的強調[[中华人民共和国国旗|中华人民共和国国旗]]、國徽、國歌等，將共產符號一般化成為中國愛國主義，是其[[六四事件|六四事件]]後[[國族主義|國族主義]]替換[[共產主義|共產主義]]的意識型態宣傳手法的細膩及成功<ref name="Zhao1998"/>。

關於這兩大主軸的思想發源，有研究者主張為[[毛澤東|毛澤東]]<ref name="毛泽东理论特色"/>及[[鄧小平|鄧小平]]<ref name="三代"/>的[[愛國主義|愛國主義]]思想延伸。但有研究者持不同觀點，主張中华人民共和国政府之前雖有在宣傳及教育之中使用愛國主義，但在1989年[[六四事件|六四事件]]後單獨把愛國主義做為宣傳主題為[[中共|中共]]歷史上第一次，如1994年正式政策文件《爱国主义教育实施纲要》一出，有中共領導人率先打破中共一直以來的禁忌，向黃帝陵進行儀式活動<ref name="Zhao1998"/><ref name="Hughes2005"/>，說明由[[江澤民|江澤民]]主導推行的愛國主義教育和之前[[愛國主義|愛國主義]]思想內涵的不同處。<ref name="Hughes2005"/>

當代研究主流則主張中华人民共和国政府之前雖有在宣傳及教育之中使用愛國主義，但在1989年[[六四事件|六四事件]]後單獨把愛國主義教育做為宣傳及教育主題為中共歷史上第一次，如1994年正式政策文件《爱国主义教育实施纲要》一出，有中共領導率先打破共產黨一直以來的禁忌，向黃帝陵進行儀式活動<ref name="Zhao1998" /><ref name="Hughes2005"/>，說明由[[江澤民|江澤民]]主導推行的爱国主义教育活动和之前[[愛國主義|愛國主義]]思想內涵的不同處。<ref name="Hughes2005"/>

在[[澳門特别行政区|澳門特别行政区]]，课程原本即有[[公民教育|公民教育]]內容，在2004-2007年期間「愛國主義教育」出現並為優先受補助的項目。<REF name="GrossmanLo2008"/>在[[香港特别行政区|香港特别行政区]]，2012年因為要在[[德育及國民教育科|德育及國民教育科]]內容中加入由中國政府提供的「愛國主義教育」大綱和內容，引發一系列的抗議和民間反彈。<REF name="Auto74-4"/><REF name="Auto74-5"/>

=== 愛國主義教育基地 ===
在愛國主義教育的政策下，有許多中央、省、市級的歷史文物地點及博物館，分別成為「愛國主義教育基地」，從第一批到今已共有八批。如中央政府於北京第一批的愛國主義教育基地有：[[天安門廣場|天安門廣場]]、[[中國歷史博物館|中國歷史博物館]]、[[中國革命博物館|中國革命博物館]]、[[中國人民革命軍事博物館|中國人民革命軍事博物館]]、[[中國人民抗日戰爭紀念館|中國人民抗日戰爭紀念館]]、[[故宮博物院|故宮博物院]]、[[圓明園遺址公園|圓明園遺址公園]]、[[八達嶺長城|八達嶺長城]]、[[周口店遺址博物館|周口店遺址博物館]]。如於西藏的第一批有[[山南烈士陵園|山南烈士陵園]]、[[江孜抗英遺址|江孜抗英遺址]]。<REF name="AutoB3-17"/>

除了「愛國主義教育基地」這名稱外，還有「愛國主義示範基地」，而這些地點常被用來組織[[红色旅遊|红色旅遊]]<REF name="AutoB3-18"/>。紅色旅遊景區因此可以視為相關教育基地與示範基地及相關基地的集合。

由於近年來中國旅遊日漸發達，不少中國民眾要求愛國主義教育基地應該免費開放。<REF name="AutoB3-19"/>2008年3月，中央政府的相关部门表示将免费开放全国1400座爱国主义教育基地和[[博物馆|博物馆]]<REF name="AutoB3-20"/>。

== 评价 ==
大多數國家主要以不同方式來實施[[公民教育|公民教育]]，而以[[愛國教育|愛國教育]]或[[愛國主義教育|愛國主義教育]]為主體的國家有[[俄羅斯|俄羅斯]]及[[中華人民共和國|中華人民共和國]]<REF name="Lee2005"/>。在[[公民教育|公民教育]]（civic education）的理論及政策中，有一派人主張愛國情操教育（尤其是在歷史教學及青少年教育方面）的重要性而突出[[爱国教育|爱国教育]]（patriotic education），然而[[自由主義|自由主義]]者則會質疑愛國主義和[[民主|民主]]及[[正義|正義]]的價值是否相容，否則會有意識型態灌輸的問題<REF name="Ben-Porath2009"/>。因此，[[公民教育|公民教育]]中激發愛國心及公民意識的跨國差異，為國際[[公民權|公民權]]的差異比較關注重點<REF name="Alred2006"/><REF name="Costa2011"/>。

[[中国教育科学研究院|中国教育科学研究院]]学者張平表示，愛國主義教育是[[江澤民|江澤民]]繼[[毛澤東|毛澤東]]及[[鄧小平|鄧小平]]相關思想的創新性論述<REF name="AutoL2-2"/>。

中国异议人士[[余杰|余杰]]評論李政亮《中国课：系上红领巾的中国式青春》時提到爱国主义教育的觀察：「在中国的学校里，一方面是学术自由极度匮乏、独立思考受到压制，另一方面是当局竭力灌输主流意识形态，顺我者昌、逆我者亡。作者发现，在中国的教科书中可以清晰看到爱国主义教育的影子，包括领导人伟大形象的建构、杰出科学家与音乐家等的爱国情操、牺牲小我完成大我的革命精神，以及中国的疆界想象。小到幼儿园的小女孩、电视里才艺表演的儿童，大到课堂上的大学生，都要接受爱国教育。即使教科书从单一版本进入一纲多本时代，仍因“坚持党的基本路线，有正确的政治观点”的原则，而充斥着爱国、革命等意涵的课文。」<ref>[http://www.rfa.org/mandarin/zhuanlan/wenyitiandi-cite/gt-04092013102955.html]</ref>而异议人士、网络作家、时事评论家[[杨恒均|杨恒均]]认为，尽管国情和爱国主义教育是各国都有的教育方式，但中华人民共和国政府实施的爱国主义教育实质上是“打着爱国主义教育的旗帜来[[洗脑|洗脑]]”<REF name="AutoB3-21"/>。

台湾[[國立政治大學|國立政治大學]]黃寬裕認為，中共從[[改革開放|改革開放]]以來、特別是[[六四事件|六四事件]]之後，一方面希望融入西方[[资本主义|资本主义]]社会，另一方面又要抵挡西方价值观的渗透，因而主張[[民族主義#國家民族主義|國家民族主義]]和[[民族主義#文化民族主義|文化民族主義]]的愛國主義教育，和原標榜[[民族主義#階級民族主義|階級民族主義]]的愛國主義教育不同<REF name="AutoL2-3"/>。《[[金融时报|金融时报]]》上刊载的一篇文章認為，愛國主義教育是一種有选择的历史教育，即一方面突出外国侵略者的残暴，一方面忽略[[中华人民共和国领导人|中华人民共和国领导人]]自身的一些错误。愛國主義教育目的在于培养[[民族主义|民族主义]]的、反[[西方|西方]]的受害者心态，以增强中共执政的合法性。<REF name="AutoL2-4"/>有异议人士认为，在愛國主義教育中的強調[[中华人民共和国国旗|中华人民共和国国旗]]、國徽、國歌等，將共產符號一般化成為中國愛國主義，是其宣傳手法的細膩及成功。但同时认为，“爱国主义教育活动”实际上是在宣扬“[[国族主义|国族主义]]”。<ref name="Zhao1998" />

根據1999跨國比較[[國際公民及素養調查研究|各國公民及素養調查研究]]的研究總結，[[西歐|西歐]]國家及[[香港|香港]]絕大多數的老師認為學校教育及環境在發展「愛國並忠於國家」的愛國主義情操方面, 很少有效果（參見[[各國學生在校學到的公民教育效果列表（老師意見報告）|列表]]）<REF name="IEA2001"/>。

根據國際教育研究組織[[國際教育成績評估協會|國際教育成績評估協會]]的[[公民教育|公民教育]]諸多面向的跨國調查比較各國公民教育的研究<REF name="年智英2011"/><REF name="洪信令2007"/>，其1999跨國比較[[國際公民及素養調查研究|各國公民及素養調查研究]]的研究總結，[[西歐|西歐]]國家及[[香港|香港]]絕大多數的老師認為學校教育及環境在發展「愛國並忠於國家」的[[愛國主義|愛國主義]]情操方面, 很少有效果（參見[[各國學生在校學到的公民教育效果列表（老師意見報告）|列表]]）；但大多數國家老師則同意學生們在學校內可以且學到了瞭解他人、和他人合作、環境保護、關心社會及投票的重要性等面向的[[公民教育|公民教育]]價值目標<ref name="IEA2001"/>。
