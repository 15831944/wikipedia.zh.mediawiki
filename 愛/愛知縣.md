'''愛知縣'''（{{jpn|j=愛知県|hg=あいちけん|rm=Aichi ken|f=y}}）是位於[[日本|日本]][[中部地方|中部地方]]的[[日本行政區劃|縣]]，[[都道府縣廳所在地|首府]]與第一大城為[[名古屋市|名古屋市]]。

== 歷史 ==
[[File:090408_aichi_kenchou.jpg|thumb]]
愛知縣在歷史上，是'''[[尾張國|尾張國]]'''、'''[[三河國|三河國]]'''這兩個[[令制國|令制國]]的轄區。1871年，[[明治新政府|明治新政府]]實行[[廢藩置縣|廢藩置縣]]，除[[知多郡|知多郡]]外的尾張成為'''{{link-ja|名古屋縣|名古屋県}}'''，而三河則與知多合併成'''{{link-ja|額田縣|額田県}}'''。名古屋縣於1872年4月易名為'''愛知縣'''，之後在同年11月27日與額田縣合併成為今天的愛知縣。

縣名是由曾管轄名古屋[[市中心|市中心]]的[[愛知郡|愛知郡]]而來。爱知县和名古屋市获得[[2026年亚运会|2026年亚运会]]举办权。

== 地理 ==
愛知縣位處[[日本列島|日本列島]]中央，向南可經[[三河灣|三河灣]]及[[伊勢灣|伊勢灣]]通往[[太平洋|太平洋]]。東面與[[靜岡縣|靜岡縣]]接鄰；東北面為[[長野縣|長野縣]]；北面為[[岐阜縣|岐阜縣]]；西面則為[[三重縣|三重縣]]。縣的東西端相距106[[公里|公里]]，南北端相距94公里；全縣總面積為5153.81[[平方公里|平方公里]]，約佔全日本面積1.36％。最高點為海拔1415公尺的[[茶臼山|茶臼山]]。

== 經濟 ==
愛知縣是日本的工業大縣，是中京工業地帯的中心。愛知縣的工業產值在2013年達約42兆18億日元，連續37年排名日本第一，是第二位的神奈川縣的2倍以上。愛知縣有以[[豐田汽車|豐田汽車]]為首的眾多[[汽車|汽車]]企業。豐田汽車在1937年設立。在[[高度經濟成長|高度經濟成長]]期之後，愛知縣的[[汽車產業|汽車產業]]發展成為高度發達的產業。

=== 產業 ===
縣內總生產（2004年）有34兆6503億日圓，佔日本國內生產總值的7.0%。

==== 過去的「五攝家」 ====
'''[[五攝家_(愛知縣)|五攝家]]'''是以愛知縣名古屋市為根據地的中部圈財界的一群名門企業，包括有旧[[伊藤財閥|伊藤財閥]]系企業的[[東海銀行|東海銀行]]及[[松坂屋|松坂屋]]、基建企業的[[名古屋鐵道|名古屋鐵道]]、[[中部電力|中部電力]]及[[東邦瓦斯|東邦瓦斯]]五家商社所構成。曾經獨佔[[中部經濟聯合會|中部經濟聯合會]]（簡稱「中經聯」）、[[名古屋商工會議所|名古屋商工會議所]]（名商）等經濟團體的代表，[[名古屋觀光酒店|名古屋觀光酒店]]及[[名古屋波士頓美術館|名古屋波士頓美術館]]等主導負責的企業。

== 人口 ==
愛知縣的人口在所有[[都道府縣|都道府縣]]中排行第四，計有7,046,000人（2009年5月1日）。在縣內[[市町村|市町村]]中，名古屋市人口最多，有2,090,549人；人口最少為[[富山村_(愛知縣)|富山村]]，只有512人。

== 交通 ==
過去愛知的交通主要依靠火車連接。但自從[[中部国际机场|中部国际机场]]開放以後，對外交通就大大改善。乘搭火車從[[名铁名古屋站|名铁名古屋站]]出發，只要30分鐘就可到達中部国际机场。
