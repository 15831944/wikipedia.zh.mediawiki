《'''愛夏的鍊金工房～黃昏大地之鍊金術士～'''》（{{lang-ja|アーシャのアトリエ 〜黄昏の大地の錬金術士〜}}，{{lang-en|''Atelier Ayesha: The Alchemist of Twilight Land''}}），是由[[GUST|GUST]]所製作的一款[[PlayStation_3|PlayStation 3]][[電子角色扮演遊戲|角色扮演遊戲]]。與前三作亞蘭德之鍊金術士系列（皆任用[[岸田梅爾|岸田梅爾]]為美術設定）最明顯的分別是此作任用[[左_(插畫家)|左]]為美術設定<ref name="hidari">{{Cite web |url=http://gnn.gamer.com.tw/7/64037.html |title=《鍊金術士愛夏 ～黃昏大地的鍊金術士～》新作登場 踏上千里尋妹冒險旅程|accessdate=2012-10-17|author= GNN 記者 Sam|date= 2012-03-31|work= |publisher=[[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]]}}</ref>。

製作公司[[GUST|GUST]]為了回饋[[台灣|台灣]]與[[香港|香港]]玩家對[[煉金術士系列|鍊金工房系列]]的支持，與台灣[[索尼|索尼]]電腦娛樂的中文化中心合作進行[[正體中文|正體中文]]化、並於日文版發行的同年8月23日由台灣[[光榮特庫摩|光榮特庫摩]]代理發行<ref name="chineseVersion">{{Cite web |url=http://www.gamedb.com.tw/newscontent.aspx?newsid=18623 |title=《愛夏的鍊金工房～黃昏大地之鍊金術士～》繁體中文版2012 年8 月23 日精彩登場！|accessdate=2012-10-17|author= 記者：哈也特|date= 2012-07-26|work= |publisher=遊戲庫}}</ref>。

《愛夏的鍊金工房Plus～黃昏大地之鍊金術士～ 》於2014年3月27日發售。

== 遊戲內容 ==

本故事敘述一位孤獨的少女——愛夏。在祖父去世後、妹妹妮歐在出外採集草藥時失蹤。於是一個人孤單地在人煙罕至的工房生活著。數年後的一日，愛夏在廢墟採集草藥時見到妹妹的身影。經過多年悲傷的愛夏，對妹妹還活著之事又燃起希望。於是踏上尋找妹妹的旅程。克服傷痛是故事的主題之一<ref>http://en.wikipedia.org/wiki/Atelier_Ayesha:_The_Alchemist_of_Twilight_Land#Premises</ref>。

遊戲的主要目標是讓愛夏和失蹤的妹妹妮歐重逢。在《黃昏大地之鍊金術士》的世界裡，人們已經淡忘了鍊金術的存在，並且正漸漸地邁向「滅亡」。悲傷的氣氛使本作與前作《[[梅露露的工作室～亞蘭德的鍊金術士3～|亞蘭德之鍊金術士]]》有極大的不同的風格<ref name="background1">{{Cite web |url=http://www.siliconera.com/2012/04/02/atelier-ayesha-doesnt-put-you-on-the-clock-when-it-comes-to-finding-your-sister/ |title=Atelier Ayesha Doesn't Put You On The Clock When It Comes To Finding Your Sister|accessdate=2012-10-18|author= Spencer|date= 2012-04-02|work= |publisher=siliconera}}</ref><ref name="background2">{{Cite web |url=http://gnn.gamer.com.tw/2/64102.html |title=《鍊金術士愛夏》6 月 28 日推出 在迎向滅亡的黃昏世界尋找行蹤不明的妹妹|accessdate=2012-10-18|author= |date= 2012-04-02|work= |publisher=[[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]]}}</ref>。
