'''愛媛縣立松山中央高等學校'''（[[日語|日]]：{{lang|ja|愛媛県立松山中央高等学校／えひめけんりつまつやまちゅうおうこうとうがっこう}}，[[英語|英語]]名称：Ehime Prefectural Matsuyama Central Senior High School）位于[[日本|日本]][[爱媛县|爱媛縣]][[松山市|松山市]]井門町，愛媛縣立高等學校<ref name=":schoollist">{{cite web|url = https://ehime-c.esnet.ed.jp/koukou/itiran/itiran.html|script-title = ja:愛媛県立高等学校一覧|accessdate = 2018-11-02|publisher = {{lang|ja|愛媛県教育委員会}}|language = ja}}</ref><ref name=":introduction">{{cite web|url = https://matsuyamachuo-h.esnet.ed.jp/introduction|script-title = ja:学校紹介|accessdate = 2018-11-02|publisher = {{lang|ja|愛媛県立松山中央高等学校}}|language = ja}}</ref>。

== 概要 ==
愛媛縣立松山中央高等學校成立于 [[1986年|1986年]][[12月26日|12月26日]]，是[[爱媛县|爱媛縣]]最新的縣立普通高等學校，開校于[[1987年|1987年]][[4月1日|4月1日]]<ref name=":introduction" />。到[[2018年|2018年]][[3月|3月]]，有11,403名學生畢業<ref name=":entrance2018">{{cite web|url = https://matsuyamachuo-h.esnet.ed.jp/cabinets/cabinet_files/download/30/546c72b6243cebad96a04f70deff36de?frame_id=45|script-title = ja:2018年4月9日 入学式校長式辞（PDF）|accessdate = 2018-11-03|publisher = {{lang|ja|愛媛県立松山中央高等学校}}|language = ja}}</ref>。

== 教育 ==
第二年，學生按課程劃分。每個系統有四個[[英語|英語]]，[[人文科學|人文科學]]，[[醫療護理|醫療護理]]，[[科學|科學]]和[[數學|數學]]課程<ref name=":learningenvironment" />。在第三年，每個學生根據各自的職業喜好選修各種選修課<ref name=":learningenvironment">{{cite web|url = https://matsuyamachuo-h.esnet.ed.jp/learning-environment|script-title = ja:学習環境|accessdate = 2018-11-03|publisher = {{lang|ja|愛媛県立松山中央高等学校}}|language = ja}}</ref>。

== 沿革 ==
* [[1986年|1986年]][[12月26日|12月26日]] - 愛媛縣立學校設立條例公佈<ref name=":introduction" />。
* [[1987年|1987年]][[4月1日|4月1日]] - 愛媛縣立松山中央高等學校開校<ref name=":introduction" />。
* [[2006年|2006年]] - 愛媛縣立松山中央高等學校被指定為[[文部科學省|文部科學省]]（MEXT）的超級英語語言高等學校（SELHi，指定期限為三年）<ref name=":selhi">{{cite web|url = http://www.mext.go.jp/a_menu/kokusai/gaikokugo/1293088.htm|script-title = ja:スーパー・イングリッシュ・ランゲージ・ハイスクール（SELHi）について|accessdate = 2018-11-03|publisher = {{lang|ja|文部科学省（MEXT）}}|language = ja}}</ref>。
