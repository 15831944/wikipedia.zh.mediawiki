'''愛之里公園站'''（{{jpn|j=あいの里公園駅|hg=あいのさとこうえんえき|rm=Ainosato-Kōen eki|f=y}}）是位於[[北海道|北海道]][[札幌市|札幌市]][[北區_(札幌市)|北區]]愛之里2-條8丁目，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[札沼線|札沼線]]（學園都市線）的[[鐵路車站|鐵路車站]]。車站編號為'''G11'''。電報略號為'''{{lang|ja|アコ}}'''。

== 概要 ==
札幌市最北的車站。由於釜谷臼地區居民的請求，於1958年在現時的車站西南約600米的位置（現時札幌市立愛之里東中學的東南方用地）設立'''釜谷臼站'''。土地由當地的地主提供，而候車室則由當地居民的捐款興建。1980年由於札幌市進行新市鎮「愛之里」開發計劃，準備開發周邊的土地，因此釜谷臼站的周圍改稱為愛之里，而車站亦改稱為現在的名稱。

[[愛之里教育大車站|愛之里教育大車站]]開始營運時，由於該車站與本車站的距離過短，因此本車站向新十津川方向遷移。

== 歷史 ==
* 1958年（昭和33年）7月1日 - [[日本國有鐵道|日本國有鐵道]]（國鐵）[[札沼線|札沼線]]'''釜谷臼站'''開始營業，處理旅客業務。當時為[[側式月台|側式月台]]1面1線的車站。
* 1986年（昭和61年）11月1日 - 隨著[[愛之里教育大車站|愛之里教育大車站]]開始營業，搬遷至現時的所在地（桑園起點14.7→15.1公里）。搬遷當時為側式月台1面1線及貨物列車改建站房的車站。
* 1987年（昭和62年）4月1日 - 由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理。
* 1990年（平成2年）11月21日 - 隨著更改車站內路軌，更換成設有[[列車交會|列車交會]]設施的側式月台2面2線的車站，並改建站房。
* 1991年（平成3年）
** 3月9日 - 列車開始使用本車站進行折返。
** 3月16日 - 札沼線的暱稱設定為「'''學園都市線'''」<ref name=school>{{cite web|url=http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |language=ja |title=札沼線（学園都市線）の電化について |publisher=JR北海道 | date=2009-09-09 |accessdate=2017-08-21 |archive-url=https://web.archive.org/web/20091006234745/http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |archive-date=2009-10-06 }}</ref><ref name=school2>{{cite web |url=http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |language=ja |title=札沼線（学園都市線）の電化開業時期について |publisher=JR北海道 |date=2011-10-03 |accessdate=2017-08-21 |archive-url=https://web.archive.org/web/20111205160350/http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |archive-date=2011-12-05 |deadurl=yes }}</ref>。
* 1995年（平成7年）3月16日 - 改稱為'''愛之里公園站'''。
* 2000年（平成12年） - 札沼線[[桑園車站|桑園車站]]至[[石狩月形車站|石狩月形車站]]引入自動進路制御裝置（PRC）。
* 2007年（平成19年）10月1日 - 設定車站編號（'''G11'''）<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |language=ja |title=駅番号表示（駅ナンバリング）を実施します |publisher=JR北海道 |date=2007-09-12 |accessdate=2017-08-21 |archive-url=https://web.archive.org/web/20070930015220/http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |archive-date=2007-09-30 |deadurl=yes }}</ref>。
* 2008年（平成20年）10月25日 - 開始使用[[儲值車票|智能車票]]「[[Kitaca|Kitaca]]」<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |language=ja |title=Kitacaサービス開始日決定について |publisher=JR北海道 |date=2008-09-10 |accessdate=2017-08-21 |archive-url=https://web.archive.org/web/20080913103748/http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |archive-date=2008-09-13 |deadurl=yes }}</ref>。
* 2012年（平成24年）
** 6月1日 - 札沼線桑園站至本車站正式[[電氣化鐵路|電氣化]]（[[交流電|交流電]]20,000[[伏特|V]]、50[[赫兹|Hz]]）<ref>{{cite web|url=http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |language=ja |title=学園都市線電化開業に伴う電車の投入（第一次）について |publisher=JR北海道 | date=2012-03-14 |accessdate=2017-08-21 |archive-url=https://web.archive.org/web/20120925203437/http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |archive-date=2012-09-25 }}</ref>。
** 10月27日 - 列車時間表進行更改<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |language=ja |title=平成24年10月ダイヤ改正について |publisher=JR北海道 |date=2012-08-03 |accessdate=2017-08-21 |archive-url=https://web.archive.org/web/20120925203427/http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |archive-date=2012-09-25 |deadurl=yes }}</ref>。

== 車站構造 ==
現時為[[側式月台|側式月台]]2面2線（月台長度：135米<ref name=sapporojr>{{cite web|url=http://www.city.sapporo.jp/sogokotsu/kotsutaikei/documents/databook_3.pdf |language=ja |title=札幌の都市交通データ（3. JR） |publisher=札幌市公式ホームページ | date=2017-04-20 |accessdate=2017-08-18 |archive-url=https://web.archive.org/web/20170818094412/http://www.city.sapporo.jp/sogokotsu/kotsutaikei/documents/databook_3.pdf |archive-date=2017-08-18 }}</ref>）的地面車站。

1990年代於站房內曾設置於早上營業的售票窗口，職員由管理的車站派遣，現時為無人車站。設置簡單的[[自動售票機|自動售票機]]及[[Kitaca|Kitaca閘機]]。可以使用Kitaca但沒有設置販賣車票及Kitaca的服務。

站房旁為2號線，需要使用[[架空道路|跨線橋]]前往1號線。而往札幌（上行）方向的列車亦會於2號線出發。
