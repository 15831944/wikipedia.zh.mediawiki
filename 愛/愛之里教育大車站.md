'''愛之里教育大站'''（{{jpn|j=あいの里教育大駅|hg=あいのさときょういくだいえき|rm=Ainosato-Kyōikudai eki|f=y}}）是位於[[北海道|北海道]][[札幌市|札幌市]][[北區_(札幌市)|北區]]愛之里1條5丁目1番2號，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[札沼線|札沼線]]（學園都市線）的[[鐵路車站|鐵路車站]]。車站編號為'''G10'''。電報略號為'''{{lang|ja|アノ}}'''。

== 概要 ==
札幌市北區新市鎮「愛之里」的主要車站，亦是[[北海道教育大學|北海道教育大學]]札幌校最近的車站（並不存在稱為「愛之里教育大學」的大學）。札沼線（學園都市線）[[八軒車站|八軒車站]]至本車站為[[複線鐵路|複線]]，而本車站至[[新十津川車站|新十津川車站]]為[[單線鐵路|單線]]路段。

== 車站構造 ==
現時為[[側式月台|側式月台]]2面2線（月台長度：138米<ref name=sapporojr>{{cite web|url=http://www.city.sapporo.jp/sogokotsu/kotsutaikei/documents/databook_3.pdf |language=ja |title=札幌の都市交通データ（3. JR） |publisher=札幌市公式ホームページ | date=2017-04-20 |accessdate=2017-08-21 |archive-url=https://web.archive.org/web/20170818094412/http://www.city.sapporo.jp/sogokotsu/kotsutaikei/documents/databook_3.pdf |archive-date=2017-08-18 }}</ref>）的地面車站。最初的出入口只有位於北方的1個，由於2007年開始開發南愛之里令人口上昇，因此於2012年10月13日新設立南方入口<ref name=south/>。

由[[桑園車站|桑園車站]]管理的業務委託車站（北海道JR Servicenet）。北方入口設有[[綠窗口|綠窗口]]（營業時間5時30分至0時10分），[[自動售票機|自動售票機]]及[[驗票閘門|自動閘機]]（對應[[Kitaca|Kitaca]]及磁性票）。而南方入口則設有簡易自動售票機及簡易自動閘機（對應Kitaca及磁性票）。

位於南方入口旁為1號線，而北方入口旁為2號線。往札幌方向（上行）列車會停靠1號線，而往石狩當別方向（下行）則會停靠2號線。過去曾有以本車站為起點站的列車，但2012年10月27日進行列車時間表修訂後，往札幌方向的列車統一以[愛之里公園車站]]、[[石狩當別車站|石狩當別車站]]及[[北海道醫療大學車站|北海道醫療大學車站]]為起點站，由於現在並沒有設定於本車站折返的列車。修定時間表前，需要進行[[列車交會|列車交會]]的上行列車會於2號線進行整備及出發，但車站南方入口設置後統一由1號線進行。

由於車站興建在地基比較惡劣的地區，北方入口及站房以後的地區皆出現[[地層下陷|地層下陷]]，因此車站樓梯需要進行加建。

== 車站周邊 ==
車站周圍主要是札幌新市鎮「愛之里」的已開發地區，北方入口為「愛之里」，而南方入口則是稱呼為「南愛之里」的區域。北方入口主要是包括東光百貨及札幌合作社等的店舖，而南方入口則是溫泉。

站房外設有行人天橋連接愛之里及南愛之里。而南之里雖然是安全市鎮，但在重要地點仍設有閉路電影及警衛巡邏。
