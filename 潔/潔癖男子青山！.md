《'''洁癖男子！青山君'''》（{{lang-ja|潔癖男子!青山くん}}）是[[坂本拓|坂本拓]]創作的[[漫畫|漫畫]]，連載於[[集英社|集英社]]的青年漫画雜誌《[[周刊YOUNG_JUMP|周刊YOUNG JUMP]]》。电视动画于2017年7月2日起由[[東京都會電視台|東京都會電視台]]、[[讀賣電視台|讀賣電視台]]等電視台播出<ref>{{cite web|url=http://www.animenewsnetwork.com/news/2017-01-28/taku-sakamoto-cleanliness-boy-aoyama-kun-manga-gets-tv-anime/.111571|title=Taku Sakamoto's ''Cleanliness Boy! Aoyama-kun'' Manga Gets TV Anime|publisher=[[Anime_News_Network|Anime News Network]]|date=January 28, 2017|accessdate=March 18, 2017}}</ref><ref>{{cite web|url=http://www.animenewsnetwork.com/news/2017-02-28/cleanliness-boy-aoyama-kun-anime-main-staff-visual-unveiled/.112786|title=''Cleanliness Boy! Aoyama-kun'' Anime's Main Staff, Visual Unveiled|publisher=Anime News Network|date=February 28, 2017|accessdate=March 18, 2017}}</ref>。台灣和香港則在2018年由[[Animax|Animax]]首播。

== 故事簡介 ==

== 登場人物 ==
=== 富士美高中 ===
; {{nihongo|青山君|あおやまくん|聲：[[置鲇龍太郎|置鲇龍太郎]]<ref name="character">{{cite web|url=http://kda-anime.com/chara.html|title=CHARACTER|work=電視動畫「潔癖男子青山！」官方網站|language=ja}}</ref>、松本沙羅〈小學生〉（日本）；賴O<ref name="姓名資料待不公佈">姓名資料待不公佈</ref>、[[李昀晴|李昀晴]]〈小學生〉（台灣）}}
: 本作男主角，一年五班，身高175，足球隊，背號10號。中場球員，非常喜歡貓，是足球U-16日本代表，擁有極度潔癖，會踢足球只因為不會用到手，也因潔癖所賜使他的足球技術高超，拒絕名校邀請而就讀富士美高中而選擇富士美的原因只有那裡才有他追求的東西(免治馬桶以及白色的制服)。有青山在的比賽，目前為止是全勝紀錄，到比賽剩五分鐘時，即使身上很髒也會忍住踢完，因輸球更讓他厭惡至極。
: 看似與他人沒有什麼互動，個性也很沉默，但還是會以自己的方式鼓勵他人振作。另外因為有潔癖的關係，在沒有練習的時候會將社團休息室打掃到一塵不染的境界。
: 除了足球之外，本身也喜歡[[貓咪|貓咪]]，不過在用逗貓棒時還是會戴口罩與手套。
: 本身有在玩網路角色扮演遊戲，角色取名為青色山脈，外型與本來面貌一樣，但頭上多了貓耳，手部也是以貓爪的方式呈現出來，個性也與現實差距非常大，可以說是十分的開朗和健談，而在網路世界的房間也十分的凌亂，與現實的一塵不染形象形成強烈對比，目前知道青色山脈真實身份的只有同班的成田紫苑。
; {{nihongo|財前薰|ざいぜん かおる|聲：[[關智一|關智一]]<ref name="character"/>、古賀葵〈幼年〉（日本）；[[馬伯強|馬伯強]]、[[林筱玲|林筱玲]]〈幼年〉（台灣）}}
: 二年三班，身高190，足球隊，背號9號，前鋒球員，外型看不出是日本大企業家的有錢少爺，個性與青山相較下十分的熱血，同時有些嚮往平民的生活方式，足球實力並不算好(特別是射門這一方面)，在後藤桃夏的特訓後已經有明顯的進步。
: 本作的常識人，對比其他角色，比較不會隨著青山的節奏起舞，一開始沒辦法接受青山的行為舉止，但在相處之下慢慢能夠理解，但有時會忍不住吐嘈出來。
; {{nihongo|後藤桃夏|ごとう もか|聲：[[春野杏|春野杏]]<ref name="character"/>（日本）；[[林筱玲|林筱玲]]（台灣）}}
: 一年三班，身高151，足球隊的經理，會來當經理是因為喜歡青山，不准任何人傷害青山，會用裝滿釘子的球棒打人(但如果遇到不合理的事，一樣會拿球棒出來)，與青山小時候就認識。
: 平時個性很文靜但亦有腹黑的一面，連財前的老爸都不敢忤逆生氣時的她，另外雖然是足球隊的經理，但足球實力好過除了青山以外的所有人，甚至還為此特訓過財前薰。
; {{nihongo|坂井一馬|さかい かずま|聲：[[保志總一朗|保志總一朗]]<ref name="character"/>（日本）；賴O<ref name="姓名資料待不公佈"/>（台灣）}}
: 二年三班，身高165，足球隊，背號3號，後衛三人組的其中之一，在其中比較有常識，處於領袖地位。此外，如果髮型改變且也不搞怪的話也相當受女生歡迎。
; {{nihongo|塚本仁|つかもと じん|聲：[[阪口大助|阪口大助]]<ref name="character"/>（日本）；[[賈文安|賈文安]]（台灣）}}
: 二年三班，身高176，足球隊，背號4號，後衛三人組的其中之一，絕技是只用屁股來顛球的「屁股顛球」。
: 在國中足球部時曾有被前輩霸凌的經歷，到了富士美後才逐漸釋懷。
; {{nihongo|吉岡太一|よしおか たいち|聲：[[吉野裕行|吉野裕行]]<ref name="character"/>（日本）；[[梁群|梁群]]（台灣）}}
: 二年三班，身高186，足球隊，背號2號，後衛三人組的其中之一，性格平和，愛吃東西，尤其熱愛甜甜圈。
; {{nihongo|石川岳|いしかわ がく|聲：[[杉山紀彰|杉山紀彰]]<ref name="character"/>（日本）；[[賈文安|賈文安]]（台灣）}}
: 三年級生，足球隊隊長，背號6號，中場球員，足球部里唯一有女朋友的。
; {{nihongo|多田光|ただ ひかる|聲：[[岸尾大輔|岸尾大輔]]<ref name="character"/>（日本）；賴O<ref name="姓名資料待不公佈"/>（台灣）}}
: 三年級生，足球隊，背號5號，中場球員，自戀狂，覺得自己比青山更帥。
; {{nihongo|佐藤清|さとう きよし|聲：[[濱野大輝|濱野大輝]]<ref name="character"/>（日本）；[[張騰|張騰]]（台灣）}}
: 二年級生，足球隊，背號1號，守門員。富士美高中足球部的守護神。
; {{nihongo|武井美和|たけい みわ|聲：[[根谷美智子|根谷美智子]]<ref name="character"/>（日本）；[[李昀晴|李昀晴]]（台灣）}}
: 足球隊的教練，柔道三段，在家是個喜愛ACG的女子，以至於家裡環境髒亂。
; {{nihongo|梅屋翼|うめや つばさ|聲：[[浪川大輔|浪川大輔]]<ref name="character"/>（日本）；[[鈕凱暘|鈕凱暘]]（台灣）}}
: 一年級生，柔道社。個性十分內斂有耐性，暗戀後藤桃夏，但明白後藤喜歡的是青山後暗中支持桃夏的戀情，默默協助後藤與青山告白逐漸拉近距離。
; {{nihongo|尾崎篤夢|おざき あつむ|聲：[[森久保祥太郎|森久保祥太郎]]<ref name="character"/>（日本）；[[張騰|張騰]]（台灣）}}
: 二年級生，對周圍人保密，真實身分是天才高中生漫畫家「鷹田未來」。
: 個性看似憂鬱，長相美型也深受女生歡迎，但內心真正的個性其實很熱血，並反應在自己的漫畫角色上面。曾經以青山外型和個性在自己的漫畫中添加反派-男爵角色，卻反遭青山外型的男爵大受歡迎而讓漫畫大賣的自己怨恨不已。
; {{nihongo|成田紫苑|なりた しおん|聲：[[野島裕史|野島裕史]]<ref name="character"/>（日本）；[[梁群|梁群]]（台灣）}}
: 一年五班，另一名潔癖男子，相較於青山，他把潔癖男的形象隱藏的非常好，目前班上也只有青山知道他的真實形象，兩人也是網路遊戲上的戰友。在班上女生運動短褲不見時曾被懷疑，後被青山給解圍，並暗勸紫苑讓班上人多了解自己但被後者給否決。
; {{nihongo|田村百合|たむら ゆり|聲：[[佐藤利奈|佐藤利奈]]（日本）；未知（台灣）}}
: 足球隊隊長石川岳的女友。
; {{nihongo|小田切美緒|おだぎり みお|聲：[[中原麻衣|中原麻衣]]（日本）；[[李昀晴|李昀晴]]（台灣）}}
: 一年三班，籃球隊，身體素質很好可是投籃很差勁，是位高挑、美顏、巨乳、樂天派的學園偶像。常叫青山師父，唯一可以跟青山觸碰的人(曾讓整個足球隊感到震驚)，這點被後藤桃夏私下認定是情敵，但由於個性好相處（由平常跟青山的互動中可以看出），後藤和她的關係基本上還算是不錯。對青山的感覺只是說不上喜歡的好感，本人看似也沒有很認真。
; {{nihongo|片平老師|かたひらせんせい）|聲：松本沙羅（日本）；[[李昀晴|李昀晴]]（台灣）}}
: 數學教師。
; {{nihongo|陣内大輔|じんない だいすけ|聲：濱野大輝（日本）；[[馬伯強|馬伯強]]（台灣）}}
: 柔道部顧問。
; {{nihongo|有賀彩香|ゆうが さいこう|聲：古賀葵（日本）；未知（台灣）}}
: 一年二班，青山籃球課的同學。喜歡青山，看著小田切和青山的互動會吃醋。有一次青山為了確認自己的潔癖，嘗試與她握手，只是最後還是暈倒了。

=== 押上南高中 ===
; {{nihongo|武智彰|たけち あきら|聲：[[子安武人|子安武人]]（日本）；[[張騰|張騰]]（台灣）}}
: 押上南高中足球隊，想邀請青山來押上南來踢足球。個性很自戀，會莫名其妙的在各種場合炫耀自己的[[腹肌|腹肌]]。
; {{nihongo|伊吹誠吾|いぶき せいご|聲：[[三木真一郎|三木真一郎]]（日本）；[[B.L.|徐偉翔]]（台灣）}} 
: 押上南高中足球隊，和青山之間有種莫名的競爭關係。足球實力比青山更加厲害，但在處理與倉田梢之間的關係感到相當棘手。雖然是運動員但很討厭控制熱量的飲食方式，成為他和倉田梢吵架的原因。
; {{nihongo|倉田梢|くらた こずえ|聲：[[遠藤綾|遠藤綾]]（日本）；[[連思宇|連思宇]]（台灣）}}
: 伊吹的青梅竹馬，很喜歡伊吹，為了他做了很多控制熱量的健康料理。和青山之間似乎也相當熟識。

=== 高田學園 ===
; {{nihongo|折原洋二|おりはら ようじ|聲：[[鈴木達央|鈴木達央]]（日本）；未知（台灣）}}
: 足球隊的不良少年。
; {{nihongo|佳奈|カナ|聲：[[沼倉愛美|沼倉愛美]]（日本）；[[林筱玲|林筱玲]]（台灣）}}
: 折原的女友，嘴角下有痣。

=== 南田附属高中 ===
; {{nihongo|門-{松}-了|かどまつ りょう|聲：[[平川大輔|平川大輔]]（日本）；[[賈文安|賈文安]]（台灣）}}
: 塚本國中足球部的前輩，會故意將足球踢到塚本身上藉此欺壓他，讓塚本的國中生涯蒙上陰影。與富士美比賽也利用這點來控制賽局，但最後被青山和後衛其他兩人鼓勵的塚本給打敗。
