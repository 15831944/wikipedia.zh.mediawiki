'''潔心林炳炎中學'''（{{lang-en|Kit Sam Lam Bing Yim Secondary School}}），原名'''聖母潔心會女子中學'''，是[[天主教|天主教]]聖母潔心會於1972年創辦的一所文法女子中學，位於[[香港|香港]][[九龍|九龍]][[黃大仙|黃大仙]][[橫頭磡|橫頭磡]]。因為得到「林炳炎基金會」捐助部份建築費用，故取名為「潔心林炳炎中學」<ref>{{cite web|url=https://www.kitsam.edu.hk/website/?lang=hk |language=zh-hant |title=潔心林炳炎中學 官方網站   |accessdate=2018-10-15 }}</ref>。

==辦學宗旨== 
*營造富有福音精神的自由與仁愛的氣氛，讓學生認識[[天主教|天主教]]信仰<ref>[https://www.openschool.hk/school/details/%E4%B8%AD%E5%AD%B8/%E6%BD%94%E5%BF%83%E6%9E%97%E7%82%B3%E7%82%8E%E4%B8%AD%E5%AD%B8 OpenSchool:潔心林炳炎中學簡介]</ref>。
*培育學生均衡發展，讓學生發揮潛能、主動學習、增長智慧。
*幫助學生建立正確的價值觀及人生觀。
*注重學生學業成績

== 班級結構 ==
*中一至中六：四班

== 校園設施 ==
設有一個大花圃、一個雨天操場、一個大禮堂、一個籃球場、一個小聖堂、一個救傷室、及一個小食部。校園內有27間課室，全部安裝了電腦及投映機。此外，還附設特別室17間，包括：圖書館、電腦室、多媒體學習室、綜合科學室、生物室、物理室、化學室、地理室、美術室、音樂室、縫紉室、家政室、學生活動中心、學習室、輔導室、牧民室和學生會室。

== 學生會  ==
學生會是該校最具代表性的學生組織，成為學校與學生之間的溝通穚樑， 為同學舉辦不同大大小小的活動以及謀取福利， 按照該校傳統每年學生會會由全校同學每人一人一票投票選出該屆學生會。
