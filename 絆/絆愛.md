'''絆愛'''（{{Lang-ja|キズナアイ}}；{{Lang-en|Kizuna AI}}）是一个日本虛擬[[偶像_(日本)|偶像]]、影音[[部落客|部落客]]創作者，經營有[[YouTube|YouTube]]頻道「A.I.Channel」和「A.I.Games」，也是公認的第一個[[虛擬YouTuber|虛擬YouTuber]]<ref name = "フォルト"/>。2016年11月底，絆愛在YouTube開設第一個主播頻道「A.I.Channel」。但該頻道一度遭到停權，不得不將影片轉至[[niconico動畫|niconico動畫]]發表。2017年3月，絆愛逐漸獲得[[日本|日本]]、[[韓國|韓國]]等地廣泛關注，随后开设了專門針對遊戲實況影片的頻道「A.I.Games」。

絆愛的角色形象由森倉圓設計，三維模型由Tomitake製作，Tda負責模型製作監督，幕后劇情編輯、配音员等人的真实身分现仍不明。影片被推測是由免費軟體[[MikuMikuDance|MikuMikuDance]]製作，並透過技術捕捉表情與動作，之後由配音員為角色献声。影片的内容形式與其他真人YouTuber相似，包括主持討論、答覆問題、遊戲實況等。絆愛在影片中的聲音主要是以[[日語|日語]]為主，不過頻道中也提供有粉絲協助翻譯的多國語言字幕。

絆愛當前先後設有官方網站、[[Twitter|Twitter]]、[[新浪微博|新浪微博]]和[[Instagram|Instagram]]帳號，其頻道營運模式倾向于技術展示與粉絲經營。曾在2017年度的[[AnimeJapan|AnimeJapan]]展臺表演，也開始推動與遊戲公司間的合作。隨著絆愛逐漸受到網際網路社群關注與歡迎，許多粉絲以此主題進行[[二次創作|二次創作]]，或用MikuMikuDance製作音樂影片。作為首個虛擬YouTuber，一些評論認為這類型的影片製作模式將會在YouTuber市場開創新潮流。

儘管並非真實人物，絆愛製作的YouTube實況影片亦包括公開遊玩電子遊戲<ref name = "Nautiljon"/><ref name = "新浪"/>。作為一名遊戲實況主播的虛擬偶像<ref name = "新浪"/>，絆愛的操作技巧略顯笨拙<ref name = "爱德华"/>。不過她也會如同其他遊戲主播般<ref name = "新浪"/>，對感到興趣的遊戲內容做出類似真人的評論和反應<ref name = "Juan José Cintas"/><ref name = "Adam Bolton"/><ref name = "Scott Wilson"/>。而其播放次數最高且最受歡迎的影片，便是絆愛遊玩著名獨立遊戲《[[Inside_(遊戲)|Inside]]》體驗版的實況暨影片<ref name = "虛擬偶像進軍Youtube"/>。在影片當中，絆愛便對於遊戲部分情節感到驚訝<ref name = "Nautiljon"/>，這部影片播放次數更達到64萬人<ref name = "Sam Gutelle"/>。

而透過全方位動作捕捉技術，部分遊戲實況影片則是遊玩[[擴增實境|擴增實境]]、虛擬實境等相關遊戲<ref name = "Adam Bolton"/>，從而形成三維動漫形象人物控制遊戲三維角色的情況<ref name="書聿" /><ref name = "相源奇"/>。其中絆愛曾經遊玩過的虛擬實境遊戲，包括極限運動遊戲《{{Link-en|攀爬 (遊戲)|The Climb (video game)|攀爬}}》<ref name = "爱德华"/>、第一人稱射擊遊戲《[[Superhot|Superhot]]》<ref name = "Rich McCormick"/>、繪圖軟體《{{Link-en|Tilt Brush|Tilt Brush}}》等<ref name = "いまい"/>。另一方面，絆愛也曾經在「{{Link-en|Quick，Draw！|Quick, Draw!}}」中嘗試以繪圖方式表達英語單字，或是透過[[Google地圖|Google地圖]]安排虛擬的[[東京|東京]]遊玩行程<ref name = "Aaron Magulick"/><ref name = "新浪"/>，或是與[[蘋果公司|蘋果公司]][[iPhone|iPhone]]助理軟體[[Siri|Siri]]等人工智慧對話<ref name = "Leon"/><ref name = "Riandanu Madi Utomo"/>。

=== 語音字幕 ===
絆愛在YouTube上的所有影片-{}-語音都是日語<ref name = "Alfred Bayle"/><ref name = "Sam Gutelle">{{Cite web |url = http://www.tubefilter.com/2017/03/21/ai-kizuna-virtual-vlog-youtube/ |title = A.I. 此舉有利於觀眾對影片評論的理解，讓日本以外的YouTube使用者和觀眾大量增加<ref name = "Nautiljon"/><ref name = "Riandanu Madi Utomo"/>，也加強角色在西方世界的能見度<ref name = "Daniel Joseph Lilly"/>。例如在2017年3月時，首部介紹影片觀看人次便達到24萬人<ref name = "Furch"/>。在影片的評論語言部分，韓國語和英語則是佔了相當的比例<ref name = "勢いがすごい"/>。

== 影響 ==
=== 人物評論 ===
[[File:Kizuna_AI_cosplayer_01.jpg|thumb]]。]]

絆愛的出现使虚拟YouTuber群体得以崭露头角<ref name = "鈴木直人"/><ref name = "Aaron Magulick"/><ref name = "Alex">{{Cite web |url = https://sumikai.com/mangaanime/virtual-youtuber-kizuna-ai-erobert-die-herzen-der-anime-fans-171588/ |title = „Virtual YouTuber“ Kizuna Ai erobert die Herzen der Anime Fans |author = Alex |date = 2017年2月26日 |publisher = Sumikai |language = de |accessdate = 2017年4月13日 |quote =  }}</ref>。由於她在影片中表现出甜美、活潑和萌的形象，受到日本、韓國等地观众的廣泛歡迎<ref name = "Alex"/>，更被粉絲暱稱為「愛醬」<ref name = "虛擬偶像進軍Youtube"/>。角色設定也被視為具革命性的概念<ref name = "Nautiljon"/>，如果按这种趨勢發展，可能引领出虛擬實境、虛擬YouTube使用者乃至於直播界的新興潮流<ref name="Riandanu Madi Utomo" /><ref name = "なにこれ未来スゴい"/><ref name = "相源奇"/>。

其中有鑑於初音未來等二次元虛擬人物的紀錄，及二次元受眾廣泛等因素，使得絆愛作為YouTube主播的想法被看好，甚至被認為可能對當前直播市場造成影響<ref name = "相源奇"/>。其中直播行業普遍要求不斷給觀眾相當特別的直播內容，而二次元直播內容便可以視為一種嘗試<ref name = "相源奇"/>。另外在原本的YouTube市場中，虛擬直播可以讓因為某些原因不能在鏡頭露臉者有所表現外，亦能透過精美的人物設計彌補現實人物的不足，或是讓團隊同時建立多個人物<ref name = "Furch"/>。

{{Link-en|Tubefilter|Tubefilter}}作者山姆·居泰勒（Sam Gutelle）認為這樣的角色現在雖然被視為怪物，往後將會變得更加普遍<ref name = "Sam Gutelle"/>。[[The_Verge|The Verge]]專欄作者瑞奇·麥考密克（Rich McCormick）則認為絆愛不同於其他創作者，不需要設定嚴格的吃飯、睡覺時間表，因此在理論上能夠任意遊玩遊戲，這為頻道提供更多靈活性<ref name = "Sam Gutelle"/>，甚至這類影片可能將會是YouTube的未來<ref name = "Rich McCormick"/>。[[英國廣播公司|英國廣播公司]]則認為其最大的優點在於不像其他YouTuber，絆愛的外貌形象永遠不會衰老，而相關發展仍然在剛開始的階段<ref name = "英國廣播公司"/>。

許多粉絲也以自己的方式表達對絆愛的喜愛<ref name = "Leon"/>，亦有部分網友的[[二次創作|二次創作]]作品會被重新在官方Twitter與Instagram發表<ref name = "虛擬偶像進軍Youtube"/>。在YouTube上也有許多以「Kizuna Ai」、「{{lang|ja|キズナアイ}}」、「絆愛」為名的頻道，不過只有官方的「A.I.Channel」才具備多國字幕<ref name = "Amola"/>；也有網友將影片重新上傳至[[bilibili|bilibili]]並提供翻譯<ref name = "軟妹"/>。除了原本的影片外，也陸續出現了跳舞音樂影片<ref name = "Amola"/><ref name = "鈴木直人"/>。但也有社群粉絲製作出了涉及性交的「[[變態_(色情用語)|變態]]」（[[H_(日語)|H]]）影片<ref>{{Cite web |url = https://www.sankakucomplex.com/2017/04/01/kizuna-ais-ero-mmd-debut/ |title = Kizuna Ai’s Ero-MMD Debut |author = Rift |date = 2017年4月1日 |publisher = Sankaku Complex |language = en |accessdate = 2017年4月13日 |quote =  }}</ref>。
