'''赛音呼·钢巴特尔'''（{{lang-mn|Сайнхүүгийн Ганбаатар}}，Sainkhüügiin Ganbaatar，{{bd|1970年|7月30日||}}），[[蒙古族|蒙古族]]，[[蒙古国|蒙古国]]政治人物。

==生平==
赛音呼·钢巴特尔曾在[[英国|英国]][[伦敦|伦敦]]读书，靠打工维持生计。回国后在一家为搬家建筑服务的风水公司任职<ref name=chaozuo>{{Cite web |url =http://www.guancha.cn/Neighbors/2017_06_26_415108.shtml  |title = 蒙古大选候选人炒作中国威胁论：“中国孙子”或“俄女婿”？ |author =  |date =2017-06-26  |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>。他长期担任蒙古工会联合会主席，从未担任政府公职，只在2012年至2016年以独立身份担任[[国家大呼拉尔|国家大呼拉尔]]委员，在委员任上后期才加入[[蒙古人民革命党_(2011年)|蒙古人民革命党]]。钢巴特尔是靠批评[[力拓|力拓]]对奥尤陶勒盖煤矿的开发而在蒙古国出名，2015年前后知名度一度攀升，甚至进入最受欢迎的蒙古国政治家之列，但此后迅速滑落，在[[2016年国家大呼拉尔选举|2016年国家大呼拉尔选举]]中丢掉了国家大呼拉尔委员席位<ref name=yuanye/>。

在[[2017年蒙古国总统选举|2017年蒙古国总统选举]]中，本来蒙古人民革命党在2017年5月5日推选党主席、前总统[[那木巴尔·恩赫巴亚尔|那木巴尔·恩赫巴亚尔]]为总统候选人<ref>{{Cite web |url =http://news.xinhuanet.com/world/2017-05/06/c_1120929808.htm  |title = 蒙古国三大政党全部推选出总统候选人 |author =  |date =2017-05-06  |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref>。但[[蒙古国|蒙古国]][[最高法院|最高法院]]在申请截止前2天突然宣布拒绝恩赫巴亚尔参选，蒙古人民革命党紧急闭门协商，赶在申请截止前数小时提出赛音呼·钢巴特尔作为替代者参选<ref name=yuanye>{{Cite web |url = http://www.guancha.cn/YuanYe/2017_06_26_415063_3.shtml |title = 袁野：蒙古总统大选前瞻——内亚大陆的茶杯风暴 |author =  |date = 2017-06-26 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>。2017年5月27日，蒙古国总选举委员会正式确认了包括赛音呼·钢巴特尔在内的参加2017年总统选举的三名候选人资格，并向三人颁发了参选证书。但赛音呼·钢巴特尔在第一轮投票中被淘汰<ref>{{Cite web |url = http://news.xinhuanet.com/world/2017-05/27/c_1121051181.htm |title = 蒙古国正式确认三名总统候选人资格 |author =  |date =2017-05-27  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。
