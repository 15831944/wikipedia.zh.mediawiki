'''赛德里克·维拉尼'''（{{lang-fr|'''Cédric Villani'''}}，{{bd|1973年|10月5日|||catIdx=Villani, Cedric}}），[[法国|法国]][[数学家|数学家]]，2010年[[菲尔兹奖|菲尔兹奖]]得主。  同时也是一位政治家，他支持现任总统马克隆。

== 简历 ==
1973年10月5日，出生于[[法国|法国]]的[[布里夫拉盖亚尔德|布里夫拉盖亚尔德]]。1998年，在[[巴黎高等師範學院|巴黎高等師範學院]]获得[[博士|博士]][[学位|学位]]，博士[[论文|论文]]研究[[玻尔兹曼方程|玻尔兹曼方程]]和[[分子运动论|分子运动论]]，博士论文指导[[教授|教授]]为[[皮埃尔-路易·利翁|皮埃尔-路易·利翁]]（{{lang-fr|Pierre-Louis Lions}}）。自2000年9月，担任[[里昂高等師範學校|里昂高等師範學校]]教授。<ref name="Villani homepage">{{cite web |url=http://www.umpa.ens-lyon.fr/~cvillani/ |title=Cédric VILLANI |accessdate=2010年8月19日 |deadurl=yes |archiveurl=https://web.archive.org/web/20100808215624/http://www.umpa.ens-lyon.fr/~cvillani/ |archivedate=2010年8月8日 }}{{en icon}}{{fr icon}}</ref>

自2009年起，担任索邦大学（Sorbonne univesité）/cnrs的{{Link-en|庞加莱研究所|Institut Henri Poincaré}}（Institut Henri Poincaré）主任。<ref name="Villani homepage"/>

== 荣誉 ==
* 2008年，[[欧洲数学学会|欧洲数学学会]]奖（European Mathematical Society Prize）；
* 2009年，[[费马奖|费马奖]]（Fermat Prize）；
* 2010年，[[菲尔兹奖|菲尔兹奖]]。
