'''呼吸作用'''，又称為'''细胞呼吸'''（{{lang|en|Cellular respiration}}），是[[生物体|生物体]]细胞把[[有机物|有机物]][[氧化|氧化]]分解並转化[[能量|能量]]的[[化學|化學]]过程，也稱為'''釋放作用'''。无论是否[[自养|自养]]，[[细胞|细胞]]内完成生命活动所需的[[能量|能量]]，都是来自呼吸作用。[[真核域|真核]]細胞中，[[线粒体|线粒体]]是與呼吸作用最有關聯的[[胞器|胞器]]，呼吸作用的幾個關鍵性步驟都在其中進行。

呼吸作用是一種[[酶促反应|酶促氧化反应]]。雖名為[[氧化還原反應|氧化反應]]，不論有否氧气参与，都可称作呼吸作用（這是因為在[[化學|化學]]上，有[[電子|電子]]轉移的[[化學反應|反應過程]]，皆可稱為氧化）。有氧气参与時的呼吸作用，稱之為'''[[有氧呼吸|有氧呼吸]]'''；没氧气参与的反應，則称为'''[[无氧呼吸|无氧呼吸]]'''。

呼吸作用的目的，是透過釋放[[食物|食物]]裡之能量，以製造[[三磷酸腺苷|三磷酸腺苷]]，即[[細胞|細胞]]最主要的直接能量供應者。呼吸作用的氢與[[氧|氧]]的燃燒，但兩者間最大分別是：呼吸作用透過一連串的反應步驟產生能量（[[ATP|ATP]]），而燃燒則是將能量一次性的釋放。在呼吸作用中，三大营养物质：[[碳水化合物|碳水化合物]]、[[蛋白质|蛋白质]]和[[脂質|脂質]]的基本组成单位──[[葡萄糖|葡萄糖]]、[[胺基酸|胺基酸]]和[[脂肪酸|脂肪酸]]，被分解成更小的[[分子|分子]]，透過數個步驟，将能量转移到还原性氢（[[化合价|化合价]]为0的氢）中。最後經過一連串的[[電子傳遞鏈|電子傳遞鏈]]，[[氢|氢]]被[[氧化|氧化]]生成[[水|水]]；原本貯存在其中的能量，則转移到ATP分子上，供生命活动使用。

==[[有氧呼吸|有氧呼吸]]==
[[File:Cell_respiration.PNG|left]]
{{FileTA|Auto-and heterotrophs|svg|thumb|210px|呼吸作用（红色箭头）是动植物利用由[[光合作用|光合作用]]（绿色箭头）预先合成的化合物中的能量的主要手段。}}
有氧呼吸产生能量（[[三磷酸腺苷|ATP]]）是需要[[氧气|氧气]]的。尽管[[糖类|糖类]]、[[脂肪|脂肪]]和[[蛋白质|蛋白质]]都可以作为反应物而被处理和消耗，然而在[[糖酵解|糖酵解]]作用下降解成的[[丙酮酸|丙酮酸]]却是首选；并且[[丙酮酸|丙酮酸]]为了被[[三羧酸循环|三羧酸循环]]完全氧化，它还需要进入[[线粒体|线粒体]]。这个过程中由[[底物水平磷酸化|底物水平磷酸化]]、[[烟酰胺腺嘌呤二核苷酸|还原型烟酰胺腺嘌呤二核苷酸]]（NADH）和[[黄素腺嘌呤二核苷酸|还原型黄素腺嘌呤二核苷酸]]（FADH<sub>2</sub>）所产生的能量是在以ATP（[[三磷酸腺苷|三磷酸腺苷]]）的形式储存起来。

醣类参与的有氧呼吸简化反应如下（反應條件從略）：

: C<sub>6</sub>H<sub>12</sub>O<sub>6</sub> ([[水溶液|s]]) + 6 O<sub>2</sub> ([[气体|g]]) → 6 CO<sub>2</sub> (g) + 6 H<sub>2</sub>O<sub></sub> ([[液体|l]]) + 能量 , ΔG = -2880千[[焦耳|焦耳]](每[[摩爾|摩爾]]葡萄糖)

部分教材會在左邊加上6個H<sub>2</sub>O（[[水|水]]），右邊生成的H<sub>2</sub>O相應為12，以表示每消耗1個葡萄糖分子就有6個水分子參與中間反應，但實際上有氧呼吸的過程複雜得多，中間參與反應的也不只有水，故此處將6H<sub>2</sub>O約去。

[[热力学自由能|自由能]]变（ΔG）是负值说明此反应可自发进行。

NADH与FADH<sub>2</sub>中的还原势通过[[电子传递链|电子传递链]]将电子送达作为“最终电子受体”的氧气并被转换为更多的ATP。大多数由有氧呼吸产生出来的ATP是由[[氧化磷酸化|氧化磷酸化]]所制造。这个工作是由消耗丙酮酸所产生的能量以推动将质子泵入线粒体膜间隙所产的H<sup>+</sup>[[濃度梯度|濃度梯度]]所完成的。接下来这个化学渗透势驱使ATP合酶将[[腺苷二磷酸|腺苷二磷酸]]以及无机磷酸合成[[三磷酸腺苷|三磷酸腺苷]]。生物书上常称在细胞呼吸中每氧化一分子葡萄糖可以生成32个ATP分子（两个来自于糖酵解，两个来自于三羧酸循环以及大约34个来自于电子传递系统）。然而，这个最大产量由于质子损失（内膜渗漏）以及推动丙酮酸进入线粒体基质的因素而永远无法达到，现在估计的是每一分子葡萄糖可以生成29～30个ATP分子。

有氧代谢[[能量轉換效率|能量轉換效率]]約為40%<ref>[http://antranik.org/intro-to-cellular-respiration-the-production-of-atp/ Intro to Cellular Respiration: The Production of ATP]</ref><ref>{{illm|尼克·連恩|en|Nick Lane}}. 生命的躍升：40億年演化史上最重要的10個關鍵</ref>，大约比无氧代谢（每摩尔葡萄糖大约生成2摩尔ATP）的效率要高19倍。他们都有[[糖酵解|糖酵解]]这一起始途但有氧代谢继续进行了三羧酸循环以及氧化磷酸化步骤。糖酵解后反应发生在[[真核生物|真核细胞]]的线粒体以及[[原核生物|原核细胞]]的胞浆中。

葡萄糖是生物体内基本的能量来源，葡萄糖的有氧分解是呼吸作用的典型，因此下面用葡萄糖作为例子讲解。

===糖解作用===
{{main|糖酵解}}
在糖解作用中，一分子葡萄糖经过一系列反应，最终生成了两分子的丙酮酸（CH<sub>3</sub>COCOOH），以及两分子ATP。此反應在[[細胞質|細胞質]]中進行，並不需[[氧氣|氧氣]]參與。糖解作用又可粗略分為兩階段，第一階段是[[葡萄糖|葡萄糖]]轉變成[[3-磷酸甘油醛|3-磷酸甘油醛]]，過程中消耗了2分子[[三磷酸腺苷|ATP]]，此為投資階段。第二階段是甘油醛三磷酸轉變成[[丙酮酸|丙酮酸]]，過程中產生了4分子ATP和兩分子[[NADH|NADH]]，是為放能階段，最後反應淨得2分子ATP和2分子NADH。
:<math>\mathrm{Glucose + 2\ NAD^+ + 2\ ADP + 2\ P_i \longrightarrow}</math> <math>\mathrm{2\ Pyruvat + 2\ NADH + 2\mathbb{H}^+ + 2\ ATP + 2\ H_2O}</math><ref name="Stryer">Berg, Stryer, Tymoczko: ''Biochemie''. Spektrum Akademischer Verlag, 2007, ISBN 978-3-8274-1800-5.</ref>

===丙酮酸脱羧过程===
{{main|丙酮酸脱羧}}

在这一过程中，一分子的丙酮酸（CH<sub>3</sub>COCOOH）脱去一个[[羧基|羧基]]，生成一分子的二氧化碳和一分子的[[乙酰辅酶A|乙酰辅酶A]]。
:<math>\mathrm{2\ Pyruvate + 2\ NAD^+ + 2\ CoA \longrightarrow}</math> <math>\mathrm{2\ Acetyl\text{-}CoA + 2\ CO_2 + 2\ NADH + 2\ H^+}</math>

===三羧酸循环过程===
{{main|三羧酸循环}}
二碳的乙酰辅酶A与四碳的草酰乙酸结合，生成了六碳的柠檬酸。柠檬酸经过一系列脱羧和脱氢的酶促反应，最终仍变成四碳的草酰乙酸。草酰乙酸不被消耗，仅仅用于生成中间产物而已。
:<math>\mathrm{2\ Acetyl\text{-}CoA + 6\ NAD^+ + 2\ FAD + 2\ GDP + 2\ P_i + 4\ H_2O \longrightarrow}</math> <math>\mathrm{4\ CO_2 + 6\ NADH + 6\ H^+ + 2\ FADH_2 + 2\ GTP + 2\ CoA}</math><ref name="Stryer"/>

===有氧途径的[[電子傳遞鏈|電子傳遞鏈]]===
{{main|电子传递链|氧化磷酸化作用}}
经过上述的分解过程，产生的能量只是很少的一部分，还有大量能量随着分解过程被转移到几种[[辅酶|辅酶]]所携带的[[质子|质子]]也就是还原性氢上。这些携带还原性氢的辅酶在细胞[[线粒体|线粒体]][[质膜|内膜]]上经过一系列被称做电子传递链的酶，将氢经过不同的[[细胞色素|细胞色素]]最终传递给氧原子生成水分子。

線粒體穿梭

====[[電子傳遞鏈|電子傳遞鏈]]的实质====
電子傳遞鏈是细胞线粒体内膜上一系列[[通道蛋白|通道蛋白]]和酶的总称，在某些[[原核细胞|原核细胞]]中，细胞以细胞质膜内褶完成类似的活动。
電子傳遞鏈的实质是将[[高能电子|高能电子]]经过一系列受体传递，电位逐渐降低，最后与质子和氧原子结合生成水。化学渗透假说较好的解释了ATP合成：电子通过线粒体内膜被传递的同时，导致了膜内外的电势差，引起质子的穿膜移动推动ATP合成酶合成ATP，这一过程通过内膜上的通道蛋白和一个ADP磷酸化的过程耦联，当电子被传递，可以导致生成ATP，这一过程被称为'''氧化磷酸化'''。
:<math>\mathrm{10\ NADH+ 10\ H^+ + 2\ FADH_2 + 26\ ADP + 26\ P_i + 6\ O_2 \longrightarrow}</math> <math>\mathrm{10\ NAD^+ + 2\ FAD + 12\ H_2O + 26\ ATP}</math>

===无氧途径===
某些[[生物|生物]]{{which}}（某些[[细菌|细菌]]{{which}}）電子傳遞鏈最終的電子接受者並非氧氣，而是[[硝酸根|硝酸根]]或[[硫酸根|硫酸根]]之类的氧化態物質，以氧化磷酸化合成ATP。發酵作用沒有氧氣參與，許多版本{{which}}的高中以下教科書，將無氧呼吸與發酵作用混為一談。但是某些領域{{which}}中，發酵作用與無氧呼吸仍作為同義詞。

==[[無氧呼吸|無氧呼吸]]（发酵）==


=== 酒精发酵 ===
{{Main|酒精发酵}}
細胞行無氧呼吸的其中一條路徑，[[丙酮酸|丙酮酸]]（CH<sub>3</sub>COCOOH）在脱羧过程后不生成[[乙酰辅酶A|乙酰辅酶A]]，而是生成[[乙醛|乙醛]]，乙醛接受[[还原|还原]]性氢被还原为[[酒精|酒精]]。在該過程中，糖酵解將一分子的[[葡萄糖|葡萄糖]]分解成兩分子的丙酮酸，丙酮酸在酶的作用下釋放出[[二氧化碳|二氧化碳]]後轉變成乙醛，乙醛再被NADH還原成乙醇（酒精）。

這裡的NAD<sup>+</sup>之再生可供應糖酵解作用所需。

過程：

# C<sub>6</sub>H<sub>12</sub>O<sub>6</sub>（葡萄糖） + 2 NAD<sup>+</sup> → 2 CH<sub>3</sub>COCOOH（丙酮酸） + 2 NADH + 2 ATP + 2 H<sup>+</sup>
# 2CH<sub>3</sub>COCOOH（丙酮酸） → 2 CO<sub>2</sub> + 2 CH<sub>3</sub>CHO（乙醛）
# 2CH<sub>3</sub>CHO（乙醛） + 2 NADH + 2 H<sup>+</sup> → 2 C<sub>2</sub>H<sub>5</sub>OH（乙醇） + 2 NAD<sup>+</sup>

=== 乳酸发酵 ===
一些[[生物|生物]]的[[呼吸|呼吸]]过程，典型的是我们熟知的[[乳酸菌|乳酸菌]]，是細胞行[[無氧呼吸|無氧呼吸]]的另一條路徑。在乳酸发酵中，[[丙酮酸|丙酮酸]]直接生成[[乳酸|乳酸]]，这是一个被[[還原|還原]]的过程，同時間使NADH被氧化為NAD<sup>+</sup>，這樣糖解作用才有材料進行。[[人体|人体]]内也存在这一过程，剧烈运动时[[肌肉|肌肉]]供氧不足，便会通过这一过程還原作用所需材料，再透過糖解得到持續的能量，生成的乳酸可以在人体里短時間累积，等到又有氧气了，乳酸会被再次转化为丙酮酸盐，然后通过有氧的方式释放出全部的能量。

過程：
# C<sub>6</sub>H<sub>12</sub>O<sub>6</sub>（葡萄糖） + 2 NAD<sup>+</sup> → 2 CH<sub>3</sub>COCOOH（丙酮酸） + 2 NADH + 2 ATP + 2 H<sup>+</sup> 
# 2 CH<sub>3</sub>COCOOH（丙酮酸） + 2 NADH +  2 H<sup>+</sup> → 2 CH<sub>3</sub>CHOHCOOH（乳酸） + 2 NAD<sup>+</sup>

過程中，糖酵解將一分子的葡萄糖分解成兩分子的丙酮酸，丙酮酸直接被NADH還原形成[[乳酸鹽|乳酸鹽]]（lactate）暫存，而沒有[[二氧化碳|二氧化碳]]釋出。

== 其他营养物质的氧化==
===脂類的氧化===
脂類物質[[水解|水解]]生成[[脂肪酸|脂肪酸]]和[[甘油|甘油]]。其中，甘油經[[磷酸基|磷酸基]]的活化和[[酶促反應|酶促反應]]轉化為[[磷酸二羥丙酮|磷酸二羥丙酮]]，後脫水變為丙酮酸，丙酮酸參加上述的呼吸過程。脂肪酸[[長鏈分子|長鏈分子]]則是反覆被脫去一端的兩個[[碳原子|碳原子]]生成[[乙醘輔酶A|乙醘輔酶A]]，參加[[克氏循環|克氏循環]]。

===氨基酸的氧化===
各種[[氨基酸|氨基酸]]可以分別經過脫[[氨基|氨基]]或脫[[羧基|羧基]]作用被[[氧化|氧化]]，脫氨基生成的[[氨|氨]]對細胞有[[毒害作用|毒害作用]]，必須被排出體外或轉化為其他[[無害物質|無害物質]]。含[[碳|碳]]部分的轉化根據氨基酸的不同而各異，但最終都是通過某些途徑成為[[檸檬酸循環|檸檬酸循環]]中的物質參與循環。
可見，各種[[營養物質|營養物質]]的[[氧化路線|氧化路線]]到最後都是相同的。生物體內的諸多[[能量代謝|能量代謝]]具有相當高的[[一致性|一致性]]。
