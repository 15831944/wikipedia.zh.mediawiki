'''呼家楼站'''位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]]，[[北京地铁|北京地铁]][[北京地铁10号线|10号线]]和[[北京地铁6号线|6号线]]的一座换乘车站。

== 位置 ==
呼家楼站位于[[北京三环路|东三环北路]]（南北向）与[[朝阳北路_(北京)|朝阳北路]]（东西向）相交的[[京广桥|京广桥]]下。10号线车站为南北向，6号线车站为东西向。

== 结构 ==
呼家楼站为地下车站，10号线站台和站厅采用[[分离式岛式站台|分离式岛式站台]]设计，以避开[[立交桥|立交桥]]桥桩，左右线间距达47.2米。由于三环路没有预留地铁通过条件，10号线车站下挖很深；因此为了避免6号线施工难度过大，站厅层预留了6号线站台的位置，亦即6号线从10号线站台上部穿过。6号线车站为[[侧式站台|侧式站台]]设计，两侧和10号线站厅直接相连<ref>{{cite journal|title=北京地铁十号线呼家楼站建筑结构型式的选定|author=王正林|journal=隧道建设|year=2006|volume=26|issue=2|url=http://www.google.com.hk/url?sa=t&rct=j&q=%E5%8C%97%E4%BA%AC%E5%9C%B0%E9%93%81%E5%8D%81%E5%8F%B7%E7%BA%BF%E5%91%BC%E5%AE%B6%E6%A5%BC%E7%AB%99%E5%BB%BA%E7%AD%91%E7%BB%93%E6%9E%84%E5%9E%8B%E5%BC%8F%E7%9A%84%E9%80%89%E5%AE%9A&source=web&cd=1&ved=0CCUQFjAA&url=http%3A%2F%2Fwww.suidaojs.com%2FEN%2Farticle%2FdownloadArticleFile.do%3FattachType%3DPDF%26id%3D8958&ei=0oVyUJHDIYHkmAXc8oDIBw&usg=AFQjCNFoqxdHiDCN97mQ6FM4v642-KpImw|format=pdf}}</ref>；此外也有8条换乘通道连接6号线站台和10号线站厅，每側各有2條短通道（樓梯）和2條長通道（斜道），在高峰時間用作單向分流換乘，其餘時間則雙向換乘<ref>{{cite news|url=http://news.sohu.com/20120921/n353713009.shtml|title=北京四条地铁新线空载试运行 拟年底开通试运营|date=2012-09-21|accessdate=2012-10-10}}</ref>。6号线站台东西两端上方各设置一个站厅<ref>{{cite news|url=http://www.morningpost.com.cn/xwzx/bjxw/2010-11-21/83966.shtml|date=2010-11-21|accessdate=2012-10-10|title=磁器口呼家楼地铁换乘方案公布|deadurl=yes|archiveurl=https://web.archive.org/web/20140317060133/http://www.morningpost.com.cn/xwzx/bjxw/2010-11-21/83966.shtml|archivedate=2014-03-17}}</ref>。

== 出口 ==
目前呼家楼站有8个出口：B（东北）、C1（东南）、C2（东南）、D（西南）、E（西北）、F（东北）、G（西南）、H（西南）。

== 事件 ==
2012年7月19日晚上約9時，呼家樓站發生挾持事件，一名男子持刀劫持一名女安檢員。民警到達現場后，由談判專家與涉案男子进行對話。对话中民警提出以他自己交換人質，但被拒絕。70多分鐘的勸說没有起到效果，涉案男子持刀傷害人質，特警狙擊手開槍將疑犯擊斃，人質安全獲救。<ref>[http://rthk.hk/rthk/news/expressnews/news.htm?expressnews&20120720&55&856226 一男子持刀挾持北京地鐵安檢員 被警方擊斃] {{webarchive|url=https://web.archive.org/web/20160305120721/http://rthk.hk/rthk/news/expressnews/news.htm?expressnews&20120720&55&856226 |date=2016-03-05 }}，[[香港電台|香港電台]]，2012年7月20日</ref>
