'''呼格吉勒图案'''，又称'''呼和浩特“4·9”女尸案'''，是于1996年4月9日在[[内蒙古自治区|内蒙古自治区]][[呼和浩特市|呼和浩特市]]发生的一起死刑[[冤案|冤案]]。报案人[[呼格吉勒图|呼格吉勒图]]（{{mong|m=ᠺᠥᠭᠰᠢᠯᠠᠲᠦ|Ppp=qoγsiletu}}，意为“'''有财产的'''”<ref>内蒙古大学蒙古学研究院蒙古语文研究所 编著：《蒙汉词典》，内蒙古大学出版社1999年出版，第652页。</ref>。1979年9月21日 - 1996年6月10日）因遭到[[呼和浩特市公安局|呼和浩特市公安局]]新城分局的[[刑讯逼供|刑讯逼供]]和法院的审判不公而被认定为杀人犯，且在“[[严打|严打]]”期间根据“从重从快”的[[中华人民共和国刑事政策|政策]]在事发后62天即被判处死刑并立即执行，造成无法补救的结果<ref name="金羊网">{{cite web |url = http://www.ycwb.com.cn/news/2009-08/05/content_2211467.htm |title = 男子被执行死刑9年后真凶现身 冤案至今未昭雪 |publisher = 金羊网 |accessdate = 2012-04-18 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2014年11月起，此案进入再审程序<ref name="zaisheng">{{cite web |title = 内蒙古高院：呼格吉勒图冤案复查中，法院没有遇到阻力 |url = http://news.163.com/14/1104/16/AA7ID2OM00014SEH.html |accessdate = 2014-12-15 |newspaper = 澎湃新闻 |date = 2014-11-04}}</ref>。2014年12月15日，[[内蒙古自治区高级人民法院|内蒙古自治区高级人民法院]]判决呼格吉勒图无罪<ref>{{Cite web |url = http://www.nmg.xinhuanet.com/xwzx/2014-12/15/c_1113638465.htm |title = 内蒙古高级人民法院:撤销原判 判决呼格吉勒图无罪 |deadurl = yes |archiveurl = https://web.archive.org/web/20141215024134/http://www.nmg.xinhuanet.com/xwzx/2014-12/15/c_1113638465.htm |archivedate = 2014-12-15 }}</ref>。

== 初期侦办与判决 ==
1996年4月9日，位于[[呼和浩特市|呼和浩特市]][[赛罕区|赛罕区]]的呼和浩特市第一毛纺厂家属区[[公共厕所|公共厕所]]内，一年轻女子遭到[[强奸|强奸]]并杀害。当晚，呼格吉勒图最先发现该女子遇害死于公厕内，在发现案情后呼格吉勒图主动找到辖区民警邢芳报案。事后，呼格吉勒图被呼和浩特市公安局新城分局特警认为有作案嫌疑并实施拘捕，在调查过程中疑似存在[[刑讯逼供|刑讯逼供]]的行为。1996年4月20日，《[[呼和浩特晚报|呼和浩特晚报]]》报道了“4·9女尸案”，报道称凶手为呼格吉勒图。<ref name="冷暖人生">{{cite web |url = http://news.ifeng.com/society/lnrs/detail_2011_07/06/7476920_0.shtml | title = 呼格吉勒图报案后反遭怀疑成凶手 被判死刑 |publisher=凤凰网 |work=冷暖人生 |accessdate=2012-04-19}}</ref>

在案件仍有疑点的情况下，检方以《[[中华人民共和国刑法|中华人民共和国刑法]]》“[[流氓罪|流氓罪]]”和“[[故意杀人罪|故意杀人罪]]”两项罪名对呼格吉勒图提起[[公诉|公诉]]，提交的证据包括“证人证言、公安机关刑事科学技术鉴定书、物证检验报告、尸体检验报告以及现场勘查笔录”。<ref>{{cite web |url = http://news.sina.com.cn/s/sd/2009-08-20/144018476520.shtml |title = 当年定案证据成为“秘密” |publisher = 新浪网，三联生活周刊 |accessdate = 2012-04-19}}</ref>

1996年6月10日，在案件证据不足并有多处疑点的情况下，呼格吉勒图仍被按规定程序以[[强奸罪|强奸罪]]、故意杀人罪判处死刑立即执行，在当日遭[[枪决|枪决]]，至此该案被以“[[严打|严打]]”期间“从重从快”的原则迅速结案。也正是因为在“严打”期间的快速结案，办理案件的冯志明等四名警察荣立[[二等功|二等功]]，并被媒体誉为“神探”。<ref name="冷暖人生" />

== 后期平反与再审 ==
{{See also|赵志红连环杀人案}}
2005年10月23日，[[赵志红|赵志红]]因涉嫌“2·25系列强奸、抢劫、杀人案”而被捕。在交代曾经犯罪记录的过程时，赵志红曾先后四次向警方供述1996年在呼市毛纺大院厕所内奸杀一名受害妇女，即呼和浩特“4·9”女尸案的详细经过。由于赵志红对呼格吉勒图案中强奸杀人过程可以准确描述，并事后经过[[中华人民共和国公安部|中国公安部]]专家多种手段的调查，经[[测谎|测谎]]认定其证言的可信度达98%以上，证实其确实是该起案件真正凶手。因此，呼格吉勒图并非“4·9女尸案”的凶手，而该案也属于[[冤假错案|冤假错案]]。2005年12月16日，《[[内蒙古法制报|内蒙古法制报]]》报道了“4·9女尸案”的后续和经过，指出真正凶手应为赵志红而非呼格吉勒图。2006年12月5日，赵志红递出一份写在[[卫生纸|卫生纸]]上的“偿命申请书”，再次重申在公厕内强奸杀害一名年轻女子的凶手是他而非呼格吉勒图，并希望可以偿命<ref>{{cite news|title=赵志红“偿命申请”曝光 愿为呼格案负责|url=http://news.qq.com/a/20141215/041604.htm|accessdate=2014-12-15|newspaper=腾讯网}}</ref>。该份文件被当地媒体报道后引起社会舆论的热议和震动。2007年1月1日起，由[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]统一行使[[中华人民共和国死刑制度|死刑核准权]]。同时，因需调查呼格吉勒图案，赵志红死刑执行被延期，被关押在呼和浩特的看守所中<ref>{{cite news|author=邢世伟、谷岳飞|title=呼格错案公检法办案人员全部被启动调查|url=http://news.ifeng.com/a/20141217/42731977_0.shtml|accessdate=2014-12-17|newspaper=新京报|date=2014-12-17}}</ref><ref>{{cite news|author=朱顺忠|title=内蒙古“呼格吉勒图冤杀案”将启动重审程序 被执行死刑时仅18岁|url=http://www.rmzxb.com.cn/yw/fd/2014/10/30/400238.shtml|accessdate=2014-12-15|newspaper=法制日报|date=2014年10月30日}}</ref>。

2009年5月29日，长期关注呼格吉勒图案的[[中华人民共和国公安部|公安部]]刑侦专家[[杨承勋|杨承勋]]和[[吴国庆|吴国庆]]向媒体表示，呼格吉勒图绝非“4·9女尸案”的真正凶手<ref name="南风窗">{{cite news|title=男子被以奸杀罪执行死刑9年后疑似真凶现身|url=http://news.sina.com.cn/s/sd/2009-08-04/143518362244.shtml|accessdate=2014-12-15|newspaper=南风窗}}</ref>。此外，内蒙古公检法系统某位不具名的重要官员就呼格吉勒图案所牵涉的问题，向媒体表示：
{{cquote|这个案件的事实部分早已经明确，不应该存在任何悬念。但是，如果按照法律规定进行，必然就有多名责任人被追究刑责和党纪政纪处分。而当年公检法三家单位的办案人都已经有了相应职务的调整，有的甚至已经升职或调迁，难度可想而知！<ref name="南风窗" />}}

直到2010年，内蒙古公检法部门尚未将此案平反或给出正式说法，受害者呼格吉勒图的家属被迫继续[[上访|上访]]，虽然得到接访人员的同情，但案件没有进展<ref name="冷暖人生" />。根据《[[中华人民共和国刑事诉讼法|中华人民共和国刑事诉讼法]]》第205条，对于已经生效的判决有三个途径启动再审程序：一是做出判决的人民法院（即[[内蒙古自治区高级人民法院|内蒙古自治区高级人民法院]]），将已经发生法律效力的判决提交本院审判委员会处理；二是[[最高人民法院|最高人民法院]][[提审|提审]]或者指令下级人民法院再审；三是[[最高人民检察院|最高人民检察院]]按照[[审判监督程序|审判监督程序]]向最高人民法院提出抗诉<ref>{{cite journal|author=汤计、董瑞丰|title=“偿命申请”拷问错案|journal=《瞭望》新闻周刊|year=2007年|issue=5期|url=http://www.cnki.com.cn/Article/CJFDTotal-LWZZ200705011.htm|accessdate=2014-12-15}}</ref>。

2014年11月4日，[[内蒙古自治区高级人民法院|内蒙古自治区高级人民法院]]决定启动再审程序，另行组成合议庭并依法进行审理<ref name="zaisheng"/>。12月15日，内蒙古高级人民法院舉辦新闻发布会宣布再审结果，决定撤销原判、宣告呼格吉勒图无罪<ref>{{Cite web |url = http://www.nmg.xinhuanet.com/xwzx/2014-12/15/c_1113638465.htm |title = 内蒙古高级人民法院:撤销原判 判决呼格吉勒图无罪 |deadurl = yes |archiveurl = https://web.archive.org/web/20141215024134/http://www.nmg.xinhuanet.com/xwzx/2014-12/15/c_1113638465.htm |archivedate = 2014-12-15 }}</ref>。内蒙古高级人民法院发布会公布了呼格吉勒图无罪的具体理由：呼格吉勒图多次供述的犯罪手段与尸体检验报告不符；血型鉴定结论不具有排他性、唯一性，不能证实呼格吉勒图实施了犯罪行为；呼格吉勒图的有罪供述不稳定，且与其他证据存在诸多不吻合之处<ref>{{cite news|author=罗沙、贾立君|title=内蒙古自治区高级人民法院对呼格吉勒图案作出再审判决  撤销原一审判决和二审裁定并宣告无罪|url=http://www.nmg.xinhuanet.com/xwzx/2014-12/15/c_1113639596.htm|accessdate=2014-12-15|newspaper=新华社|date=2014-12-15|deadurl=yes|archiveurl=https://web.archive.org/web/20141215062943/http://www.nmg.xinhuanet.com/xwzx/2014-12/15/c_1113639596.htm|archivedate=2014-12-15}}</ref>。

内蒙古高院副院长[[赵建平|赵建平]]当面向呼格父母道歉<ref name="apolo">{{cite news |title=内蒙古高院向呼格吉勒图父母道歉表示：对不起！ |url = http://sn.people.com.cn/n/2014/1215/c186331-23227538.html |accessdate=2014-12-15 |newspaper=新华社 |date=2014-12-15}}</ref>：{{cquote|我代表自治區高級人民法院，向你們表示真誠的道歉。對不起。對於呼格吉勒圖的錯判，對我們來說，也是一件非常痛心的事情，我們從今以後，一定會吸取教訓，深刻反思，在辦這個案子當中，法院存在的問題，我們會針對這些問題，採取相應的措施，堅守底線，防止冤假錯案的發生，絕不能讓呼格吉勒圖這種悲劇再重演，所以我再次說，對不起。|赵建平<ref name="trad">{{cite news|url=http://www.chinatimes.com/realtimenews/20141216005102-260402|title=內蒙古高院改判呼格吉勒圖無罪並道歉|publisher=新华社|date=2014-12-16|accessdate=2014-12-17}}</ref>}}

內蒙古自治區高級人民法院院長[[胡毅峰|胡毅峰]]同时託趙建平交予呼格吉勒圖父母三萬元慰问金，并向他们表示可以申請[[國家賠償|國家賠償]]<ref name="trad" />。

2014年12月30日，內蒙古高院做出國家賠償的決定，決定支付[[呼格吉勒圖|呼格吉勒圖]]的父母賠償金，總共人民幣貳百〇伍萬玖千陸百貳拾壹元肆角整（2,059,621.4元）<ref>{{cite news|url=http://www.sc.xinhuanet.com/content/2014-12/31/c_1113838944.htm|title=内蒙古高院决定赔偿呼格吉勒图家人200余万|date=2014-12-31|publisher=新华社|accessdate=2015-01-01|deadurl=yes|archiveurl=https://web.archive.org/web/20141231193439/http://www.sc.xinhuanet.com/content/2014-12/31/c_1113838944.htm|archivedate=2014-12-31}}</ref>。12月31日，呼格吉勒图的父母领取了国家赔偿决定书<ref>{{cite news|url=http://news.xinhuanet.com/2014-12/31/c_1113846407.htm|title=呼格吉勒图父母获赔205万元：内心伤痛无法抹平|date=2014-12-31|publisher=新华网|accessdate=2015-01-01}}</ref>。

2015年11月11日，呼格吉勒图骨灰迁入新坟，新墓地位于离呼和浩特市区近30公里远的和林格尔县安佑生态园内的宝珠山半山腰处，新墓地和旧坟地相距约20多公里。

{{cquote|呼格吉勒图，内蒙古呼和浩特人。一九七七年九月二十一日生。十八岁时，危难攸降。蒙冤而死。

一九九六年四月九日夜，一女子被害身亡。呼格报案。被疑为凶手。后不堪严刑而屈招。被判死刑，六月十日。毙。

呼格负罪名而草葬于野。父母忍辱十年。哀状不可言。二零零五年十月。命案真凶现身。呼格之冤方显于天下。令华夏震惊。然案牍尘封无所动，又逾九年。内蒙古自治区高级人民法院再审。二零一四年十二月十五日宣布呼格无罪。

优良的司法，乃国民之福。呼格其生也短，其命也悲。惜无此福。然以生命警示手持司法权柄者，应重证据，不臆断。重人权，不擅权，不为一时政治之权益而弃法治与公正。

今重葬呼格。意在求之，以慰冤魂。

特立此碑。|呼格吉勒图墓志铭，由中国著名法学家[[江平_(学者)|江平]]撰写}}

== 追责程序 ==
在撤销原判的文书送达的当天，内蒙古自治区党委宣布已启动呼格吉勒图案追责程序<ref>{{cite news |author=邢世伟 |title=内蒙古高院：呼格案追责程序已启动 |url = http://news.sohu.com/20141215/n406957533.shtml |accessdate=2014-12-15 |newspaper=新京报|date=2014-12-15}}</ref>。据《法制晚报》2015年9月20日报道，呼和浩特市公安局及赛罕分局的多名警官因涉及冯志明涉嫌职务犯罪案件，接受办案机关调查，其中呼和浩特市公安局赛罕分局大学东路派出所原所长张文，被办案机关采取司法强制措施<ref>{{cite web|url=http://www.fawan.com/Article/fwkx/2015/09/20/100846307419.html|publisher=法制晚报|accessdate=2015-09-20|title=“呼格冤案”续——内蒙多名警官涉冯志明案被调查|archive-url=https://web.archive.org/web/20151010203121/http://www.fawan.com/Article/fwkx/2015/09/20/100846307419.html|archive-date=2015-10-10|dead-url=yes}}</ref>。2016年1月31日，新华社报道，对呼格吉勒图错案负有责任的27人被有关机关和部门进行了追责<ref name="追责结果">{{cite web|url=http://news.xinhuanet.com/legal/2016-02/01/c_128688928.htm|title=内蒙古公布呼格吉勒图案追责结果|accessdate=2016-02-01|publisher=新华网}}</ref>。
