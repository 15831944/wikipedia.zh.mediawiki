'''缅甸掸邦东部第四特区'''<ref>[http://www.4tzx.com/portal.php?mod=view&aid=1084 关于春节期间销售和燃放烟花爆竹的通告]</ref>，[[缅甸|缅甸]]政府称'''勐拉乡'''（前称'''缅甸[[掸邦|掸邦]]第四特区'''），通称'''勐拉'''，是位于[[缅甸|缅甸]][[掸邦|掸邦]]东部邊陲山区的一个[[事实上|事实上]]的[[自治区|自治区]]，成立于1989年，首府[[勐拉|勐拉]]，辖区面积约4952平方公里，人口约8.5万人，民風民俗和生活習慣與[[中华人民共和国|中國]][[雲南省|雲南省]]大體相同。2008年缅甸新宪法修订后，缅甸政府将此地编制为勐拉乡，但勐拉当局仍使用特区名号。

== 历史沿革 ==
该地原为[[缅甸共产党|缅甸共产党]]815军区驻地。1989年4月19日，[[林明贤|林明贤]]领导的815军区宣布脱离缅共，随后与缅甸军政府达成协议成立自治特区，原军事力量改组成[[掸邦东部民族民主同盟军|掸邦东部民族民主同盟军]]。

== 地理 ==
特区位於緬甸撣邦東北部，北与[[中国|中国]][[云南省|云南省]][[西双版纳|西双版纳]]接壤，东与[[老挝|老挝]]接壤，西与[[佤邦|佤邦]]相邻。

==行政区划==
缅甸掸邦东部第四特区下设三个地区：[[勐拉|勐拉地区]]、南板地区、色勒地区。

== 政治军事 ==
缅甸掸邦东部第四特区成立以来一直维持军政自治，仅在名义上隶属于掸邦政府。[[和平与团结委员会|和平与团结委员会]]（原名掸邦东部民族民主同盟党）是该地的执政党，主席是[[吴再林|吴再林]]（汉名林明贤），书记处总书记是[[吴腾林|吴腾林]]（汉名林道德，吴再林之子）。[[掸邦东部民族民主同盟军|掸邦东部民族民主同盟军]]是该地的武装力量。

== 對外交流 ==
特區外交事務現時有緬甸聯邦政府代表。

但{{Fact|实际上，缅甸依然无法对特区进行有效控制，包括外交。外事交流頻繁地區主要為特區周邊地區的中國[[雲南省|雲南省]]、[[佤邦|佤邦]]、[[果敢|果敢]]等地區。特区和中国关系密切，两者近乎一体|time=2019-01-30}}。

==经济==
该地毒品、色情交易猖獗，走私现象严重且治安混乱。当地政府完全放开了赌博和色情，并为其提供安全保障。2017年，勐拉有8家赌城。最早的赌场是建立于1995年的金三角赌场。各大赌场使出各种手段吸引赌客上门参赌<ref>[http://mini.eastday.com/mobile/171003110654995.html 走进神秘缅甸小勐拉，赌场大街：中国人比当地人还多]</ref><ref>[http://world.huanqiu.com/exclusive/2014-03/4893311.html 美媒：缅甸勐拉小城治理混乱 危及中缅边境]</ref>。
