'''缅茄'''（[[学名|学名]]：{{lang|la|''Afzelia xylocarpa''}}）是[[豆科|豆科]][[缅茄属|缅茄属]]的植物。

==形态==
常绿[[乔木|乔木]]，高达40米。偶数羽状复叶，2－4对宽椭圆形或宽卵形的小叶，先端圆或微凹；总状花序排列成圆锥状，密生灰色短毛，萼管状花偏向花序轴的一侧，裂片四枚，两大两小，仅一枚淡紫色花瓣，其余退化；长椭圆形褐色的木质[[荚果|荚果]]肥厚，有种子2－3个，种子有角质种柄。

===種籽===
緬茄的種籽上有一種黃色的附著物，那種附著物像是種籽上的一層帽子，極其堅硬。

== 分布 ==
分布在[[越南|越南]]、[[老挝|老挝]]、[[柬埔寨|柬埔寨]]、[[泰国|泰国]]、[[缅甸|缅甸]]以及[[中国大陆|中国大陆]]的[[广东|广东]]、[[广西|广西]]、[[海南|海南]]、[[云南|云南]]等地。

== 别名 ==
木茄（广东高州）、沔茄、细茄（广东误写）

== 緬甸國寶 ==
緬茄原產於緬甸。為稀珍樹種，喬木，樹皮灰褐色，有灰白斑黃，葉偶數羽狀複葉，莢果木質，棕褐色而光滑，種子木質黑色，卵形長橢圓形或，乾時堅如石。緬茄樹為高級家具用材，果實用於刻印或手雕藝術擺飾，被人們視為象徵平安幸運的珍品。
