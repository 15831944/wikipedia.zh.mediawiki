'''緬甸國旗'''是[[東南亞|東南亞]]國家[[緬甸|緬甸]]所使用的旗幟。緬甸政府在2010年根據2008年通過的《緬甸聯邦共和國憲法》<ref>[http://news.sina.com.cn/w/2010-10-21/202421325022.shtml 缅甸启用新国旗国徽 现军政府下月移交国家权力]，新浪网，2010年10月21日{{zh-hans}}</ref>有關國家標誌的規定，修改國旗圖案。並於2010年10月21日正式啟用新國旗，國旗樣式為黃綠紅三色加一顆白色[[五角星|五角星]]<ref name="bcc_2010Oct21">[http://news.sina.com.tw/article/20101021/3870566.html 緬甸更改國旗圖案　與我國國旗相似度減少]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}{{zh-tw}}</ref>。

== 樣式與含義 ==
外觀為黃綠紅三條線，中間有一顆白色巨大五角星<ref name="bcc_2010Oct21"/>。黄色象征团结，绿色象征和平、安宁以及代表青葱翠绿的国家，红色象征勇敢和决心。白色五角星代表[[聯邦|聯邦]]永久长存。此[[三色旗|三色旗]]的樣式和[[第二次世界大戰|第二次世界大戰]]中，在[[日本|日本]]支持下建國的[[緬甸國|緬甸國]]國旗相同，唯一的差異在於，緬甸國國旗的中心不是白星，而是象徵[[貢榜王朝|貢榜王朝]]的[[綠孔雀|綠孔雀]]。

==更换国旗==
2006年11月10日的一次宪法会议上提出新国旗方案，新国旗由等宽的绿、黄、红三色横条带和左上角一颗白星组成。<ref name="globalflag"/>2007年9月又提出另一个方案，该方案中白星放在旗面中央并被放大，色条排列也变为黄、绿、红。<ref name="globalflag">{{cite web|url=http://www.globalflag.idv.tw/zy/mmr.htm|title=緬甸 Myanmar/Burma|last=Tai|first=Yu-liang|language=zh-hant|accessdate=2010-10-23}}</ref>

===采用新国旗===
2007年9月提议的新国旗包含在2008年缅甸宪法中，并在2008年缅甸宪法公投中获得承认。<ref name="Robert Birsel">{{cite news|url=http://ca.reuters.com/article/topNews/idCATRE69K2HM20101021|title=Myanmar gets new flag, official name, anthem|last=Birsel|first=Robert|date=2010-10-21|work=[[路透社|路透社]]|accessdate=2010-10-23|location=http://www.webcitation.org/5thEflNof}}</ref>政府官员被要求在当地时间2010年10月21日下午3点前降下旧国旗，<ref name="st"/>还传出要求保证焚毁所有旧国旗的命令。<ref name="Robert Birsel" /> 直到国旗更换后不久，国家媒体才宣布采用新国旗。<ref name="st">{{cite news|title=Myanmar unveils new flag|url=http://www.straitstimes.com/BreakingNews/SEAsia/Story/STIStory_593613.html|accessdate=2010-10-23|publisher=[[海峡时报|海峡时报]]|newspaper=AP|date=2010-10-21}}</ref>

== 國旗歷史 ==
前任國旗採用於1974年1月3日，[[吳奈溫|吳奈溫]]執政時期。是一面紅底旗幟，左上角為藍底，上有白色十四五角星，象徵十四個行政單位。中間為[[齒輪|齒輪]]和[[稻|稻]]的圖案，是工農的象徵。紅色象徵人民的英勇和果敢，白色弘揚了緬甸人純潔的天性和美德，藍色則是和平與統一的象徵。

== 旧旗帜类似旗帜 ==
1948年-1974年，以及1974年－2010年間採用的缅甸国旗，其樣式与[[萨摩亚国旗|萨摩亚国旗]]、[[中华民国国旗|中华民国国旗]]（青天白日滿地紅旗）、[[台灣民眾黨|台灣民眾黨]]黨旗相似，有时易導致混淆。<ref>[https://www.nownews.com/news/20100523/701251 沖繩車站介紹台灣，誤植緬甸國旗]，[[NOWnews|NOWnews]]，2010年5月23日{{zh-tw}}</ref>依照[[TVBS|TVBS]]新聞報導，[[淡江大學|淡江大學]]東南亞所副教授林若雩表示，緬甸獨立前，與中華民國政府走得近，原因是需要外力支持，把英國趕走，所以類似的國旗設計是有這樣的淵源。<ref>[https://news.tvbs.com.tw/politics/93377 日本爆烏龍！錯把緬甸國旗當我國旗]{{zh-tw}}</ref>

在[[2008年北京奧運|2008年北京奧運]]期間，由於青天白日滿地紅旗不能被攜帶進入會場，有部分台灣棒球迷攜帶当时的緬甸國旗進入[[2008年夏季奧林匹克運動會棒球比賽|棒球賽]]現場為[[中華成棒隊|中華成棒隊]]加油。<ref>[http://sports.ltn.com.tw/news/paper/235611 球迷揮緬甸國旗 挑戰官方尺度] [[自由時報|自由時報]]{{zh-tw}}</ref>
