'''张宏民'''（{{bd|1961年|3月13日||}}），男，[[北京|北京]]人，祖籍[[河北|河北]][[唐县|唐县]]西城子村。[[中央电视台|中央电视台]]新闻[[播音员|播音员]]、[[主持人|主持人]]。1982年毕业于北京广播学院（现[[中国传媒大学|中国传媒大学]]），即调入中央电视台主持[[新闻联播|新闻联播]]节目。<ref>{{cite web |url=http://news.cctv.com/china/20090611/103544.shtml |title=《新闻联播》主持人张宏民出席罗京追悼会(图) |accessdate=2009 |publisher=CCTV |work= |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20160308001931/http://news.cctv.com/china/20090611/103544.shtml |archivedate=2016-03-08 }}</ref><ref>{{cite web |url=http://ent.qq.com/a/20080326/000024.htm |title=央视主播张宏民掷20万过49岁生日引发热议(图) |accessdate=2008 |publisher=QQ |work= |language=zh}}</ref><ref>{{cite web |url=http://media.people.com.cn/n/2013/0809/c40606-22499709.html |title=薛飞罗京张宏民 央视《新闻联播》三大帅哥旧照 |accessdate= 2013 |publisher=People.com |work= |language=zh}}</ref>主持期间，2009年曾有“饱嗝门”出镜事件。

2014年5月28日，有媒体报道张宏民与[[李瑞英|李瑞英]]退出《新闻联播》，后得到李瑞英本人证实<ref>{{cite book | url=http://ent.ifeng.com/a/20140528/40089166_0.shtml | title=李瑞英证实与张宏民告别《新闻联播》 | publisher=凤凰网 | accessdate=2014-05-28}}</ref>，从而正式结束其32年的播音生涯。

== 轶闻 ==
1989年6月4日[[六四事件|六四事件]]天安门清场当天，张宏民与[[杜宪|杜宪]]一同担任当晚新闻联播的播音员。杜宪穿着肃穆，语速缓慢、沉重地播报了关于[[中國人民解放軍|中國人民解放軍]]戒嚴部隊開入[[天安門廣場|天安門廣場]]清场等新闻。张宏民穿着同样肃穆、庄重，但表情自然，播音流畅，并未过于明显地表露出对学生的同情。此举既不违反播音纪律，也委婉地表达了对学生遇难的哀悼。尽管如此，张宏民仍然与杜宪和[[薛飞_(电视主播)|薛飞]]一同领受了一纸停止出镜播音的组织决定。虽然杜宪和薛飞稍后被调离新闻联播栏目组，但张宏民最后获解禁，并继续主持新闻联播节目，直至2014年4月13日最后一次出镜。
