'''张一一'''{{BD|1981年|1月1日|||catIdx= Z张}}出生于[[湖南省|湖南省]][[岳阳市|岳阳市]][[湘阴县|湘阴县]]，为青年作家、评论家，与韩寒、[[郭敬明|郭敬明]]并列为中国大陆当代三大青年作家<ref name="fish " >{{cite web |url= http://news.xhby.net/system/2011/07/06/011160857.shtml|title=《党赋》作者张一一个人简介 |accessdate= 2011-07-06|publisher=新华报业网 |work= |language =zh-hans}}</ref>，同[[李敖|李敖]]、[[王朔|王朔]]、[[韩寒|韩寒]]并列为“中国四大文霸”<ref name="bull " >{{cite web |url=http://book.ifeng.com/yeneizixun/detail_2012_02/23/12731286_1.shtml |title=争议作家张一一新作《带三只眼看国人》叫板柏杨 |accessdate=2012年2月23日 |publisher=凤凰网 |work= |language =zh-hans}}</ref>。張一一因其牽涉諸多文化、娛樂事件以及炒作而備受爭議。他曾自封“炒作教主”和“炒家学派创始人”，在網絡上亦被稱為「炒作帝」。<ref>{{cite news|title=张一一 自娱自乐自我“炒作帝”|url=http://finance.591hx.com/article/2011-11-04/0000088090s.shtml|accessdate=2015-10-10|newspaper=华讯财经|date=2011-10-04}}</ref>

== 生平 ==
1981年，张一一出生在[[湖南省|湖南省]][[岳阳市|岳阳市]][[湘阴县|湘阴县]]。<ref name="fish " /><ref name="bull " />

2005年，张一一出版《我不是人渣》。

2011年，张一一出版《反红楼梦》，后为歌手[[杨臣刚|杨臣刚]]写作歌曲《兔子爱萝卜》，又创作了《男人的苦恼》，在[[农历|农历]][[春节|春节]]创作《兔赋》。4月份，张一一出版《可爱的中国人》。<ref name="fish " />
2014年，张一一编写的《新百家姓》在网上引起网民热议。因其将现任国家领导人的姓氏排在众多姓氏的前列，被网友们痛批为“走狗、奴才”。

== 爭議事件 ==
2004年4月30日，[[長沙|長沙]]《[[當代商報|當代商報]]》刊登了張一一的征婚廣告，應征條件中提到「你一定要沒有看過[[中国国家足球队|中國隊]]和[[中国国家奥林匹克足球队|國奧隊]]的任何一場比賽」。當時中韓即將於長沙進行奥运會男子足球賽，因此其做法引來爭議。<ref>{{cite news|title=中韓大戰即將上演 長沙一征婚廣告『叫板』|url=http://sports.big5.enorth.com.cn/system/2004/05/01/000777701.shtml|accessdate=2014-12-07|newspaper=红网|date=2004-05-01|deadurl=yes|archiveurl=https://web.archive.org/web/20141210113922/http://sports.big5.enorth.com.cn/system/2004/05/01/000777701.shtml|archivedate=2014年12月10日|df=}}</ref>球賽結果是國奧隊以0:2的比分敗於韓國隊。之後在2004年5月8日，張一一以“中国足球的槽糕表现”為由提出民事起訴，向[[中国足球协会|中國足協]]索赔精神损失费一分钱。有報導稱這是首次有中國球迷起訴足協。<ref>{{cite news|title=长沙球迷受伤太深 状告中国足协索赔一分钱(图)|url=http://news.rednet.cn/c/2004/05/09/559146.htm|accessdate=2014-12-07|newspaper=红网|date=2004-05-09}}</ref>

2006年11月27日，張一一在博客上刊登文章，公開向當時剛宣布已離婚的[[湖南衞視|湖南衞視]]主持人[[李湘_(主持人)|李湘]]求婚，更列舉出李湘「下嫁」的六大理由。<ref>{{cite news|title=80後作家向李湘求婚 列舉"下嫁"六理由(圖)|url=http://news.qq.com/a/20061128/000680.htm|accessdate=2014-12-07|newspaper=華商報|date=2006-11-28}}</ref>李湘後來發短信婉拒<ref>{{cite news|title=李湘发短信婉拒张一一求婚 称其早已心有所属|url=http://media.people.com.cn/GB/40606/5116468.html|accessdate=2014-12-07|newspaper=上海青年报|date=2006-12-02}}</ref>，不過張一一並未就此放棄，之後還曾作出各種行徑以令對方回應，比如威脅要公佈她和緋聞男友的照片、在博客發表情書、聲稱會到對方家裡或工作場所裸奔等。<ref>{{cite news|title=张一一向李湘求婚 称其从初中开始暗恋李湘(图)|url=http://news.sohu.com/20061207/n246863583.shtml|accessdate=2014-12-07|newspaper=青年周末|date=2006-12-07}}</ref><ref>{{cite news|title=作家張一一致李湘“21世紀最偉大的情書”(組圖)|url=http://news.qq.com/a/20061206/002179.htm|accessdate=2014-12-07|newspaper=中國經濟網|date=2006-12-06}}</ref><ref>{{cite news|title=李湘遭张一一最后通牒 不表态情人节将裸奔|url=http://www.hubei.xinhua.org/newscenter/2007-01/18/content_9075661.htm|accessdate=2014-12-07|newspaper=红网|date=2007-01-17|deadurl=yes|archiveurl=https://web.archive.org/web/20141210175641/http://www.hubei.xinhua.org/newscenter/2007-01/18/content_9075661.htm|archivedate=2014-12-10}}</ref>於擾攘一個多月後，張一一在博客稱放棄追求李湘，并以“日薪一万元求租女友回家过年”。<ref>{{cite news|title=作家放弃裸奔向李湘求婚 日薪万元租女友过年(图)|url=http://news.xinhuanet.com/society/2007-01/22/content_5634649.htm|accessdate=2015-10-10|newspaper=新华网|date=2007-01-22}}</ref>

2012年8月25日，張一一在[[新浪微博|新浪]]和[[腾讯微博|腾讯微博]]上聲稱，[[瑞典文學院院士|瑞典文學院院士]]、[[諾貝爾文學獎|諾貝爾文學獎]]評審委員[[馬悅然|馬悅然]]從翻譯中國作家[[莫言|莫言]]的作品中收取了好處費。8月27日，莫言在新浪和腾讯微博上作澄清，並稱會對造謠者保留诉诸法律的权利。張一一其後轉發了該聲明並繼續追討。<ref>{{cite news|title=莫言微博回应张一一质疑 陈安娜称别理小疯子|url=http://www.chinanews.com/cul/2012/08-30/4145579.shtml|accessdate=2014-12-07|newspaper=每日新报|date=2012-08-30}}</ref>及後他也不時在微博等网络媒体上稱莫言不配得諾貝爾文學獎，即使在頌獎前也發微博稱：「莫言获奖，我裸奔」。之後結果揭曉，莫言果真成為中國大陸首位獲得諾貝爾文學獎的作家，網民於是要求張一一兌現裸跑承諾。張一一最後履行承諾，在12月8日現身於[[浏阳飞天温泉|浏阳飞天温泉]]並當眾裸奔。<ref>{{cite news|title=莫言获颁诺奖 张一一兑现承诺当众“裸奔”/组图|url=http://365.voc.com.cn/article/201212/201212101035176218.html|accessdate=2014-12-07|newspaper=华声在线|date=2012-12-10|deadurl=yes|archiveurl=https://web.archive.org/web/20141209224851/http://365.voc.com.cn/article/201212/201212101035176218.html|archivedate=2014-12-09}}</ref>
