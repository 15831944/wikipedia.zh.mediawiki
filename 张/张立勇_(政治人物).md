'''张立勇'''（{{bd|1955年|7月|catIdx=Zhang張}}），[[山东省|山东省]][[荣成市|荣成市]]人。现任[[河南省高级人民法院|河南省高级人民法院]]院长，二级大法官。[[第十二届全国人民代表大会|第十二届全国人民代表大会]]河南省代表。

==生平 ==
1982年毕业于[[山东大学|山东大学]]哲学系。1982年-1993年期间历任中共中央纪律检查委员会干事、副处长、处长；其间曾挂职河北省[[河间市|河间市]]委副书记一年。1993年底转任陕西省[[咸阳市|咸阳市]][[秦都区|秦都区]]委书记，1997年6月担任咸阳市委常委，1999年兼任咸阳市委副书记、常务副市长。2002年底升任咸阳市市长。2003年获[[西北政法学院|西北政法学院]]法律硕士；2005年9月升任咸阳市委书记，次年3月兼任咸阳市人大常委会主任。2007年12月转任[[河南省高级人民法院|河南省高级人民法院]]党组书记，次年1月任[[河南省高级人民法院|河南省高级人民法院]]院长、党组书记<ref>{{Cite book|title=司法公正的进程 2015年两会期间新闻宣传及报道选编|last=最高人民法院新闻局编|first=|publisher=北京：人民法院出版社|year=2015.11|isbn=978-7-5109-1372-3|location=|pages=369}}</ref>。2008年起担任全国人大代表<ref>{{cite web |title = 全国人大代表信息-张立勇 |url = http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=111466 |accessdate = 2013-04-05 |website = 全国人大网 }}</ref>。2013年，担任全国人大代表<ref>{{cite web |title=全国人大代表信息-张立勇 |url = http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=121437 |accessdate = 2013-04-05 |website = 全国人大网 }}</ref>。2008年12月，当选河南省法官协会会长。
