'''张立昌'''（{{bd|1939年|7月|2008年|1月10日|catIdx=Z张}}），[[河北|河北]][[南皮|南皮]]人，[[天津冶金工业学校|天津冶金工业学校]]毕业，步入仕途后在[[北京经济函授大学|北京经济函授大学]]经济管理专业取得大专学历。1966年2月加入[[中国共产党|中国共产党]]，是中共第十二、十三届[[中央候补委员|中央候补委员]]，第十四、十五、十六届[[中央委员|中央委员]]，第十六届[[中央政治局|中央政治局]]委员。曾任[[中共天津市委|中共天津市委]]书记，任职期间口碑甚差，常遭到[[天津人|天津市民]]的调侃和讽刺，后调任[[国务院振兴东北地区等老工业基地领导小组|国务院振兴东北地区等老工业基地领导小组]]副组长<ref>[http://chinaneast.xinhuanet.com/2007-03/26/content_9611897.htm 张立昌兼任振兴东北等老工业基地领导小组副组长] {{webarchive|url=https://web.archive.org/web/20071123100505/http://chinaneast.xinhuanet.com/2007-03/26/content_9611897.htm |date=2007-11-23 }}</ref>。2008年1月10日，张立昌死于[[天津|天津]]，终年68岁<ref>[http://news.enorth.com.cn/system/2008/01/11/002630035.shtml 中共天津市委原书记张立昌逝世]，天津北方网 </ref>。

== 早年经历 ==
张立昌19岁时，考入[[天津冶金工业学校|天津冶金工业学校]]学习。1960年毕业后，张立昌一直工作在[[天津|天津]]。他先是被分配到“天津无缝钢管厂”工作；历任厂团委副书记、动力科科长等职；到“[[文化大革命|文化大革命]]”开始的时候，27岁的张立昌已经是“天津无缝钢管厂”的副厂长了。在[[文革|文革]]中，张立昌受到冲击，先是被下放劳动，接着是被降职为车间主任，后又调任厂基建科科长。直到1972年，张立昌才重新获得使用，并升任厂长兼厂党委副书记。

== 从政经历 ==
=== 进入政界 ===
[[改革开放|改革开放]]以后，中共不断提拔培养中青年干部。而正是在这一大背景下，张立昌进入了[[中共天津市委|中共天津市委]]、[[天津市人民政府|市政府]]所选定的提拔干部的名单中。1980年6月，在[[胡启立|胡启立]]出任[[天津市市长|天津市市长]]的两个月后，时年41岁的张立昌因为尚属年富力强、专业对口，以及在“[[文革|文革]]”中没有参与“[[造反派|造反]]”的政治表现，被提拔为天津市冶金工业局副局长。1982年5月，[[李瑞环|李瑞环]]接替了[[胡启立|胡启立]]的[[天津市市长|天津市市长]]的职务，并继续执行干部年轻化的政策。于是，1983年7月，张立昌晋升为天津市冶金工业局局长；一年后，又调任天津市经委副主任；不久，就晋升为主任。1985年9月，张立昌和[[吴邦国|吴邦国]]、[[李长春|李长春]]、[[吴官正|吴官正]]、[[贺国强|贺国强]]、[[刘云山|刘云山]]等人一起被增补为中共第十二届中央候补委员。

=== “扎根”天津 ===
1985年10月，张立昌被任命为[[天津市人民政府|天津市人民政府]]副市长，兼任市委工业工委书记；1989年9月，升任[[中共天津市委|中共天津市委]]副书记、副市长，兼市口岸委主任；1993年6月正式当选[[天津市市长|天津市市长]]。五年后，张立昌又出任中共天津市委书记、市人大常委会主任。张立昌任职中共天津市委、市政府主要领导工作二十年。2002年11月，在中共十六届一中全会上，他当选为[[中共中央政治局|中共中央政治局]]委员。

张立昌离职前，天津爆出了数件震动全国的腐败案。原天津检察长[[李宝金|李宝金]]因受贿和挪用公款于被判处死刑，缓期两年执行。李宝金情妇[[王小毛|王小毛]]亦被判刑。2007年3月，中共中央决定张立昌不再兼任[[天津|天津]][[市委书记|市委书记]]、常委、委员的职务；但同时被任命为[[国务院振兴东北老工业基地领导小组|国务院振兴东北老工业基地领导小组]]的副组长。其被调离天津的消息一经发出，引发天津市民的热议和欢庆，当天市区多个地方出现反常的[[鞭炮|放鞭炮]]等庆祝现象。2007年6月3日，张立昌刚刚从天津离职后天津市政协原主席宋平顺自杀身亡。中共中央纪委经过调查称，宋平顺道德败坏，包养情妇；滥用手中权力，为情妇谋取巨额不正当利益，将其开除出党。事发后，中国官方媒体则说，宋平顺的贪腐长达10年，身为天津一把手的张立昌也难逃其咎。

在2007年10月召开的[[中共十七大|中共十七大]]上，张立昌并没有再次连任中央委员，表明68岁的张立昌已经“[[退居二线|退居二线]]”。2008年1月10日晚22时20分，张立昌因病去世。消息传出当天，天津市大街小巷再次传出鞭炮声，当天{{Fact|面条旺销|time=2014-05-06T12:37:33+00:00}}。1月16日，官方在天津为他举行了丧礼和追悼仪式。

== 执政情况 ==
=== 语录 ===
在天津市一次反腐工作会议上，张立昌曾说：“天津市有腐败的现象，但是没有腐败的官员。”<ref>{{cite web |url = http://news.sohu.com/20060830/n245071286.shtml |title = 评论：“天津有腐败现象，但没有腐败官员？” |accessdate = September 3, 2011}}</ref>此语一出引发社会各界的热议和嘲讽。不久便爆发天津市检察长[[李宝金|李宝金]]及情妇[[王小毛|王小毛]]被捕，市政法委书记、政协主席[[宋平顺|宋平顺]]自杀等一系列腐败案件。

=== 政绩 ===
* 在其主导下，[[天津市|天津市]]在1990年代做出工业东移的战略，并提出“三五八十”的计划，力争十年基本建成[[滨海新区|滨海新区]]，为日后滨海新区上升为国家战略和中国的第三增长极做出了准备工作。
* 主导[[海河综合开发工程|海河综合开发工程]]，为日后[[海河|海河]]两岸的繁荣奠定了基础。

=== 消极 ===
* 张立昌执政天津的10余年，是天津自近代发展成为中国北方最大的工商业城市后经济发展最为缓慢的时期，直接导致天津老百姓在其执政期间工资和生活水平低，对市政府颇多怨言。直至张立昌被调离天津后，天津市的经济增长才重新开始高速增长，经济增速连续多年位于全国第一。
* 在其将天津市全市财政收入大量投入到滨海新区建设的情况下，造成天津市中心城区固定投资减少，市容面貌在九十年代滑坡。
* 在其执政下天津市政府[[裙带关系|裙带关系]]严重，贪污腐败、徇私枉法现象不绝，官商勾结受到庇护，其在任和刚离任时曾发生[[永基非法集资案|永基非法集资案]]、[[天津地铁腐败案|天津地铁腐败案]]、[[李宝金|李宝金落马案件]]、[[王小毛|王小毛案件]]、[[宋平顺|宋平顺自杀事件]]等一些列大案要案，贪污腐败、徇私枉法等现象直至其下台才稍有改善。2004年3月15日，曾有1000多名来自天津的民众在北京[[中国银行业监督管理委员会|中国银行业监督管理委员会]]门前打出“打倒张立昌，打倒大恶霸[[宋平顺|宋平顺]]！”的横幅表示对永基非法集资案的处理方式的不满。

== 评价 ==
与曾主政天津的[[李瑞环|李瑞环]]不同，张立昌在[[天津|天津]]执政期间的口碑之差难以形容，对于张立昌的评价，大体上可以用“官褒民贬”来形容，尤其在民间恶评如潮，调侃讽刺之语不胜枚举。

=== 官方评价 ===
官方对其的评价为“'''中国共产党的优秀党员，忠诚的共产主义战士，卓越的党的工作领导者'''”<ref>[http://news.xinhuanet.com/newscenter/2008-01/11/content_7401813.htm 中共天津市委原书记张立昌逝世]，新华网，2010年11月28日查阅</ref>。

=== 民间评价 ===
张立昌在[[天津|天津]]执政期间的口碑之差难以形容，调出天津和逝世的两个时间，曾引发天津市民燃放鞭炮、吃喜面庆祝的狂潮，同时天津当地流传着大量讽刺张立昌的民谣。
* 天津民谣称“两顺一昌，唬弄中央”（一作“糊弄中央”；“两顺”指2007年6月3日畏罪自杀的前天津政法委书记、政协主席[[宋平顺|宋平顺]]和2015年2月13日因涉嫌严重违纪违法被双开的公安局长[[武长顺|武长顺]]）<ref>[http://news.creaders.net/china/newsViewer.php?nid=331896&id=774510 天津书记张立昌病逝 官褒民贬]，2010年11月27日查阅</ref>。
* “张立昌，给碗汤”以讽刺其执政时期，天津市下岗职工多、失业率高、工资水平低。 
* 另有一则民谣称张立昌有个“三不走：百姓轰不走，中央调不走，[[阎王爷|阎王爷]]叫不走。”{{Fact|time=2014-05-06T12:37:33+00:00}}亦是对其实施的“三步走”政绩工程的讽刺。
* 天津民众还有一则“四大张”的民谣用于调侃张立昌，即“天津四大张：[[泥人张|泥人张]]，[[果仁张|果仁张]]，[[崩豆张|崩豆张]]，瞎话张”，天津本有“三大张”其中最后加上的一个“瞎话张”指的就是张立昌。把他和天津几个著名特产放在一起，凸现了天津市民的幽默与无奈。与此相关的有，[[天津电视台|天津电视台]]在2000年时曾现场采访一名天津儿童京剧演员，天津的“三大张”有哪些。该名儿童面对镜头竟公开说出了“瞎话张”在当时引起了议论。
* 因[[天津市第一殡仪馆|天津市第一殡仪馆]]位于[[北仓镇|北仓]]，天津还有一则民谣“张立昌进北仓，天津人民奔小康”，表达出天津市民对张立昌的不满。而张立昌2008年病故后，遗体确实在天津市第一殡仪馆火化。<ref>{{cite news|url=http://news.cctv.com/xwlb/20080116/106193.shtml|title=张立昌同志遗体在天津火化|work=[[新闻联播|新闻联播]]|publisher=[[中国中央电视台|中国中央电视台]]|date=2008-01-16|accessdate=2015-01-02|deadurl=yes|archiveurl=https://web.archive.org/web/20150102081746/http://news.cctv.com/xwlb/20080116/106193.shtml|archivedate=2015-01-02}}</ref>
* 天津另有一则民谣“张立昌，真能干，要把天津变成县”，用以讽刺其在位期间天津发展缓慢。
* 张立昌停职后不到半年即病情恶化死亡。病逝消息传出后，当天天津市区内有市民燃放鞭炮庆祝，亦有大量市民吃[[喜麵|喜麵]]庆祝，当天很多超市、市场的面条一度脱销。
