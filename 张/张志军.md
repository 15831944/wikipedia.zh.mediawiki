'''张志军'''（{{bd|1953年|2月|||CatIdx=Zhang张}}），[[江苏|江苏]][[南通|南通]]人，[[北京大學|北京大學]]畢業。[[中国共产党|中国共产党]][[中国共产党第十八届中央委员会委员列表|第十八届中央委员]]<ref name=shibada>{{cite news|url =http://politics.people.com.cn/n/2012/1115/c105667-19595018.html|title =中国共产党第十八届中央委员会委员名单|work =人民日报社|publisher =[[人民日报|人民日报]]|pages =|date =2012-11-15|accessdate =2014-06-27|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20140903213224/http://politics.people.com.cn/n/2012/1115/c105667-19595018.html|archivedate =2014年9月3日|df =}}</ref>，曾任[[中共中央台湾工作办公室|中共中央台湾工作办公室]]主任、[[国务院台湾事务办公室|国务院台湾事务办公室]]主任。原任[[中华人民共和国外交部|外交部]]常务副部长，负责外交部常务工作、主管政策规划事务<ref name=chusheng/>。現任[[全国人民代表大会外事委员会|全国人民代表大会外事委员会]]副主任委員、[[海峽兩岸關係協會|海峽兩岸關係協會]]會長。

==學歷==
张志军1971年進入[[北京大学|北京大学]]就讀，后赴[[英国|英国]]留学<ref name="生平履历">{{cite news  |author =   |url = http://paper.takungpao.com/html/2013-03/19/content_18_1.htm |title = 張志軍盼與台胞交朋友 |publisher = [[大公报|大公报]] |pages = A18 |date = 2013-03-19 |accessdate = 2014-06-27|quote=張志軍是江蘇人，......山東省淄博市委副書記。 }}</ref>。

==經歷==
*1975年进入中華人民共和國外交部（[[中共中央对外联络部|中共中央对外联络部]]）任職，曾任七局干部、副处长、美大北欧局处长。
*1991年起在[[中華人民共和國驻英国大使馆|中華人民共和國驻英国大使馆]]任一等秘书。<ref name="生平履历"/>
*1994年出任中联部美大北欧局副局长<ref name="生平履历"/>。
*1996年出任中共山东省[[淄博市|淄博市]]委副书记<ref name="生平履历"/>。
*1997年起又历任中联部研究室主任、美大北欧局局长、副秘书长、副部长等职<ref name="生平履历"/>。
*2000年起出任中联部副部长。
*2009年调任中华人民共和国外交部常务副部长<ref>{{cite web |url = http://news.ifeng.com/mainland/detail_2010_12/20/3597619_0.shtml|title=张志军任外交部常任副部长 外交精英梯队渐成型|publisher=凤凰网}}</ref>。
*2010年出任中华人民共和国外交部党委书记<ref>{{cite web|title=张志军任外交部党委书记 |url = http://cpc.people.com.cn/GB/164113/13535047.html|publisher=中国共产党新闻网}}</ref>。
*2013年3月，接替出任外交部长的[[王毅|王毅]]担任[[国务院台湾事务办公室|国务院台湾事务办公室]]主任<ref>[http://www.apdnews.com/news/23347.html 张志军就解决两岸政治问题提出叁点看法] {{webarchive|url=https://web.archive.org/web/20130325045442/http://www.apdnews.com/news/23347.html |date=2013-03-25 }}，亚太日报，2013年3月22日</ref><ref>{{cite news|title=陆东福任国家铁路局局长 张业遂任外交部副部长 |url = http://news.163.com/13/0326/14/8QT9AA5J0001124J.html|accessdate=2013-03-26|newspaper=网易}}</ref><ref name=chusheng>{{cite news |url = http://jsnews.jschina.com.cn/system/2013/03/18/016591393.shtml |title = 南通人张志军任中共中央台办、国务院台办主任 |work=新华网|publisher = 中国江苏网 |pages =  |date = 2013-03-18 |accessdate = 2014-06-27|quote=}}</ref>。
*2018年3月，轉任[[全国人民代表大会外事委员会|全国人民代表大会外事委员会]]副主任委員。
*2018年4月，接替[[陳德銘|陳德銘]]出任[[海峡两岸关系协会|海峡两岸关系协会]]会长。

==事蹟與軼事==
===台湾事务===
*2013年10月6日，在[[印尼|印尼]][[峇里島|峇里島]]举办之前的第二十一届[[亚太经合组织|亚太经合组织]]领导人非正式会议，张志军參與两岸事务协调工作，讓當時的[[中華民國總統|中華民國總統]][[马英九|马英九]]特使[[萧万长|萧万长]]能与[[中共中央总书记|中共中央总书记]][[习近平|习近平]]会晤，並首次让[[中華民國|中華民國]][[陆委会|陆委会]]参与APEC会议，加上贊同並邀請[[中華民國|中華民國]][[行政院大陸委員會|行政院大陸委員會]]主任委員的[[王郁琦|王郁琦]]在适当时间访问[[中國大陸|中國大陸]]，被認為是[[两岸关系|两岸关系]]进展的一大突破。

===2014年-{zh-cn:张王会;zh:张王会;zh-tw:王張會;}- ===
{{main|张王会谈}}
2014年2月11日，张志军与陆委会主委[[王郁琦|王郁琦]]在南京召开首次两岸事务首长会议，双方互以官衔称呼。张志军同意具体落实台生在大陆的相关医疗保险，陆委会表示肯定。王郁琦在「张王会」後记者会上表示，双方同意建立常态沟通机制<ref>[http://www.cna.com.tw/topic/popular/4297-1/201402110045-1.aspx (影音) 王张会互称官衔 建沟通机制] {{Wayback|url=http://www.cna.com.tw/topic/popular/4297-1/201402110045-1.aspx |date=20140221235330 }}，中央社，2014年2月11日</ref><ref>[http://www.apdnews.com/news/67976.html “张王会”成功举行 两岸关系再获重大突破] {{webarchive|url=https://web.archive.org/web/20140221232442/http://www.apdnews.com/news/67976.html |date=2014-02-21 }}，亚太日报，2014年2月12日</ref>。
<br>2014年6月25日，为了建立两岸互访常态化机制，以及作为对陆委会主委[[王郁琦|王郁琦]]访问南京的回访。张志军首度访台，成為[[中華人民共和國|中華人民共和國]]第一位訪問[[台灣|台灣]]的國臺辦主任，并将深入台湾基层走访<ref>[http://www.taihainet.com/news/twnews/bilateral/2014-06-26/1272817.html 台湾舆论聚焦张志军首度访台并与王郁琦会面,台海网，2014年06月26日]</ref>。[[中華民國總統|中華民國總統]][[马英九|马英九]]对此表示：“张志军访台显示两岸关系确实有相当的进展。”<ref>[http://news.ifeng.com/a/20140626/40907438_0.shtml 马英九：张志军访台显示两岸关系存实质进展，凤凰网，2014年06月26日]</ref>
