'''张忠培'''（{{bd|1934年||2017年|7月5日}}）[[中华人民共和国|中华人民共和国]][[湖南|湖南]][[长沙|长沙]]人，考古学家，北京[[故宫博物院|故宫博物院]]院长<ref name=gao/><ref name=zxw/>。

==生平==
1952年，张忠培自长沙[[长郡中学|长郡中学]]考入[[北京大学|北京大学]]于同年创办的考古专业。1956年，张忠培自北京大学考古专业本科毕业。1957年获选拔就读北京大学考古专业副博士研究生，师从[[苏秉琦|苏秉琦]]。1961年研究生毕业后，被分配到[[吉林大学|吉林大学]]历史系任教。1972年主持创办吉林大学考古专业，历任考古专业副主任、主任，历史系副系主任、主任，研究生院常务副院长，参与创办了吉林大学研究生院。1984年获吉林省有突出贡献科技专家称号，任吉林大学学位评定委员会副主席、学术委员会副主任、教师职称评定五人领导小组成员，协助吉林大学校长主持重点学科及其学术梯队建设工作<ref name=gao>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1725318 |title = 原故宫博物院院长张忠培辞世，考古人生八十三载 |author =  |date = 2017-07-05 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

1958年，当时还是副博士研究生的张忠培带着北京大学考古专业本科生在[[陕西省|陕西省]][[华县|华县]]、[[渭南县|渭南县]]率先开展了中国考古史上“拉网式”大规模区域系统考古调查，在华县元君庙遗址全面揭露出一处[[仰韶文化|仰韶文化]]墓地，即[[元君庙仰韶墓地|元君庙仰韶墓地]]。张忠培不采用过去考古学以一座墓葬或房屋为单位的做法，而将整座墓地作为研究单元，开创了中国考古史上全面揭露并研究墓地的新方法。1959年起，张忠培逐渐研究出元君庙仰韶墓地的社会组织及其社会性质。1983年，他撰写的《元君庙仰韶墓地》报告出版<ref name=gao/>。

1979年[[中国考古学会|中国考古学会]]成立，张忠培成为中国考古学会最为年轻的理事。此后担任理事34年、常务理事19年、副理事长9年，2008年当选为中国考古学会理事长。张忠培还曾任全国哲学社会科学规划委员会考古学科组成员、[[国家文物局|国家文物局]]考古专家组成员、国务院长江三峡工程验收委员会委员、国家[[南水北调工程|南水北调工程]]考古专家组组长、[[第七届全国人大代表|第七届全国人大代表]]等职<ref name=gao/>。

1987年，张忠培被国务院任命为故宫博物院院长。1991年被定为国家有突出贡献专家，享受国务院[[政府特殊津贴|政府特殊津贴]]。离任后担任[[故宫研究院|故宫研究院]]名誉院长、故宫博物院学术委员会副主任委员<ref name=gao/><ref name=zxw/>。

2017年7月5日，张忠培在北京病逝，享年83岁<ref name=gao/><ref name=zxw>{{Cite web |url =  http://www.chinanews.com/cul/2017/07-05/8269502.shtml|title = 考古学家、故宫原院长张忠培因病去世 享年83岁 |author =  |date =  2017-07-05|publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。7月11日，遗体告别仪式在[[八宝山殡仪馆|八宝山殡仪馆]]东礼堂举行。7月25日，[[国家文物局|国家文物局]]、[[北京大学|北京大学]]、[[吉林大学|吉林大学]]、[[中国社会科学院|中国社会科学院]]考古研究所、[[故宫博物院|故宫博物院]]、[[中国考古学会|中国考古学会]]、[[中国文物学会|中国文物学会]]、[[中国博物馆协会|中国博物馆协会]]联合在故宫博物院召开“张忠培先生追思会”<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1730323 |title = 故宫博物院原院长张忠培遗体告别仪式在北京举行 |author =  |date = 2017-07-11  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
