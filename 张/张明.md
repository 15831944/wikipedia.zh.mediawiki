'''张明'''（{{bd|1957年|6月||}}），[[河北|河北]][[蠡县|蠡县]]人，[[中华人民共和国|中华人民共和国]]外交官。

==生平==
1975年参加工作，1983年加入[[中国共产党|中国共产党]]。毕业于[[北京外国语学院|北京外国语学院]]，获学士学位。2001年，任[[中华人民共和国外交部|外交部]]办公厅副主任。2006年，接替[[郭崇立|郭崇立]]，担任[[中国驻肯尼亚大使列表|驻肯尼亚大使]]<ref>{{cite web|author=中华人民共和国外交部|title=中国驻肯尼亚历任大使|url=http://www.mfa.gov.cn/chn/pds/ziliao/wjrw/2167/2170/t9197.htm|publisher=中华人民共和国外交部|accessdate=2012-03-07}}</ref>。2009年，任[[中华人民共和国外交部|外交部]]非洲司司长。2010年，任外交部办公厅主任。2011年，任外交部部长助理（主管办公厅、干部人事和档案工作的部长助理）。2013年12月，任外交部副部长<ref>{{cite news|title=【一周人事信息】张明任外交部副部长 高云龙任光大副总|url=http://renshi.people.com.cn/n/2013/1206/c368940-23770551.html|accessdate=2013-12-07|newspaper=人民网}}</ref>。2017年10月接替[[杨燕怡|杨燕怡]]出任[[中国驻欧盟使团大使列表|驻欧盟使团团长]]<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1824129  |title =张明已担任中国驻欧盟使团团长 |author =  |date =2017-10-13  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1823886  |title = 张明不再担任外交部副部长职务 |author =  |date =  2017-10-13 |publisher =澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。
