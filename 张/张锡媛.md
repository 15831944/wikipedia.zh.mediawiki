'''张锡媛'''（{{bd|1907年|10月28日|1930年|1月|Z}}<ref name="a1">{{Cite web |url=https://www.chinatimes.com/cn/newspapers/20160831001084-260306 |title=《两岸史话－邓小平》：踏上革命之路 留学归来 走向革命（十二）|accessdate=2018-12-18 |author=文：亚歷山大‧潘佐夫、梁思文，译：吴润璇|date=2016-08-31 |work= |publisher=中天电子报，来源：旺报 | quote =他只知道那个女孩的真名是张锡媛，于1907年10月28日出生在直隶省房山县良乡站的工人之家（她的父亲是名铁道工人）。1925年11月27日，张锡媛与陈绍禹一同从中国来到了中山大学。在抵达莫斯科的一个月之后，加入了共产党。|language=Zh-hans }}</ref>），[[邓小平|邓小平]]的第一任妻子。

== 生平 ==
張茜本名張錫瑗，1907年10月28日生於[[直隶省|直隶省]][[顺天府|顺天府]][[房山县|房山县]]<ref name="a1"/>。<ref name="寒山碧">寒山碧原著，伊藤潔縮譯，唐建宇、李明翻譯：《鄧小平傳》，香港：東西文化事業公司，1993年1月</ref>{{rp|22}}父親張鏡海是鐵路工人，姐妹三人，她是其中最大，两位妹妹分别是张锡瑞、[[张晓梅_(1911年)|张锡珍（张晓梅）]]<ref name="a2">{{Cite web |url=http://history.sohu.com/a/165668840_717266 |title=房山“第一红色家庭”的故事您想知道吗|accessdate=2018-12-18 |author=|date=2017-08-18 |work= |publisher=搜狐网 |language=Zh-hans }}</ref>。父亲張鏡海因[[二·七大罷工|二·七大罷工]]被通緝<ref name="寒山碧"/>{{rp|22}}。1924年，就读于[[保定第二女子师范学校|保定第二女子师范学校]]。1925年，全家遷居[[北京|北京]]，三姐妹同時加入[[共青團|共青團]]，在[[中国共产党|中国共产党]]北方區委創辦學校學習，深受[[李大釗|李大釗]]教誨，不久張錫瑗正式加入中國共產黨<ref name="寒山碧"/>{{rp|22－23}}<ref name="a1"/>。1925年11月<ref name="a1"/>，張錫瑗与妹妹张锡瑞<ref name="a2"/>、[[陈绍禹|陈绍禹]]<ref name="a1"/>进入苏联[[莫斯科中山大学|莫斯科中山大学]]学习，与邓小平相识。

1927年张锡瑗回国<ref name="a2"/>，受中共中央指示進入中共中央秘書處，在上海與[[鄧小平|鄧小平]]一起工作<ref name="寒山碧"/>{{rp|23}}。1928年，兩人因相愛而结婚。<ref name="寒山碧"/>{{rp|23}}1929年夏，鄧小平趕赴廣西策劃[[百色起義|左右江起義]]，張留在上海，鄧經常往返廣西和上海向中共中央匯報工作。<ref name="寒山碧"/>{{rp|23}}1930年1月，張产子后患[[产褥热|产褥热]]，因難產而死時，鄧小平在廣西還不知道<ref name="寒山碧"/>{{rp|23}}。
