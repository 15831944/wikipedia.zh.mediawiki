'''张元勋'''（{{bd|1933年|7月2日|2013年|4月12日|catIdx=Z}}），祖籍[[江苏省|江苏省]][[赣榆县|赣榆县]][[青口镇_(连云港市)|青口镇]]，曾任[[曲阜师范大学|曲阜师范大学]]教授<ref name="linzhao2">{{cite web |author1=張元勛 |title=北大往事與林昭之死（續） |url=http://ywang.uchicago.edu/history/big5/Linzhao_text2.htm |website=ywang.uchicago.edu |accessdate=2018-10-30|language=zh-hant}}</ref><ref name="songbie">{{cite news |author1=李岩峰 |title=送别张元勋教授 |url=http://www.infzm.com/content/89770 |accessdate=2018-10-31 |work=南方周末 |date=2013-04-19 |archive-url=https://web.archive.org/web/20180203180946/http://www.infzm.com/content/89770 |archive-date=2018-02-03 |dead-url=no |language=zh-hans }}</ref>，中国楚辞学家。

== 生平 ==
1954年，张元勋于[[青岛一中|青岛一中]]高中毕业，后考入[[北京大学|北京大学]][[北京大学中文系|中文系]]文学专业。1956年秋，《北大诗刊》停办，他与[[林昭|林昭]]等人成为《红楼》编委<ref>{{cite web |author1=張元勛 |title=北大往事與林昭之死 |url=http://ywang.uchicago.edu/history/big5/Linzhao_text.htm |website=ywang.uchicago.edu |accessdate=2018-10-31|language=zh-hant}}</ref>。1957年[[中国共产党|中国共产党]]发起[[大鸣大放|大鸣大放]]运动，而后发展为[[反右|反右]][[整风运动#1957年“开门整风”|整风运动]]。因担任北京大学学生自办的刊物《广场》的主编，撰写《广场发刊词》、《编后记》，以及与[[沈泽宜|沈泽宜]]等合写短诗《是时候了》等[[大字报|大字报]]而被划为“极右派”，遭受轮番批斗。1957年12月25日张元勋被逮捕，北京市中级人民法院以[[反革命罪|反革命罪]]判处有期徒刑八年，剥夺政治权利五年。张元勋被押送于北京南郊之[[团河农场|团河农场]]、[[河北省|河北省]][[宁河县|宁河县]]之[[茶淀|茶淀]][[清河农场|清河农场]]等地区进行[[劳动改造|劳动改造]]。1965年刑满，继续留在清河农场，接受“[[无产阶级专政|无产阶级专政]]”，顶着“[[反革命分子|反革命分子]]”的“帽子”在劳动机关的管制下劳动<ref name="linzhao2"/>。

[[文化大革命|文化大革命]]开始后，张元勋被押送到[[山东省|山东省]]劳改队所属的[[章丘|章丘]][[埠村|埠村]]煤矿，在煤井下挖煤四年；1966年5月6日，刑满释放但仍在管制中的张元勋由山东奔赴[[上海|上海]]，以“未婚夫”之名给在[[提篮桥监狱|提篮桥监狱]]中的[[林昭|林昭]]探监<ref>{{Cite book|title=今天怎样改革|last=《读者参考丛书》编辑部编著|first=|publisher=上海：学林出版社|year=2013.02|isbn=978-7-5486-0480-8|location=|pages=198-199}}</ref><ref name="cannotstop">{{cite web |author1=李木生 |title=“总不能停止写吧” 探访张元勋 |url=http://www.infzm.com/content/81473 |publisher=南方周末 |accessdate=2018-10-31 |date=2012-09-29 |archive-url=https://web.archive.org/web/20180404230613/http://www.infzm.com/content/81473 |archive-date=2018-04-04 |dead-url=no |language=zh-hans }}</ref>。受此牵连，张元勋被单独关押138天，险些丧命<ref>{{cite news |author1=李木生 |title=张元勋的最后出行 |url=http://www.infzm.com/content/89769 |accessdate=2018-10-31 |work=南方周末 |date=2013-04-19 |archive-url=https://web.archive.org/web/20150516235610/http://www.infzm.com/content/89769 |archive-date=2015-05-16 |dead-url=no |language=zh-hans }}</ref>。1970年又被押送至山东[[济宁|济宁]]“六四”劳改农场，继续劳动改造。1977年，“[[四人帮|四人帮]]”倒台，对张元勋的监管得到政策性的放松。同年秋天，张元勋经其组织批准与来自山东农村的女青年马用强结婚<ref name="cannotstop"/>。1979年11月24日，北京市中级人民法院作出平反裁决，推翻1957年的判决，宣告张元勋无罪<ref>{{Cite book|title=追寻林昭|last=许觉民编|first=|publisher=武汉：长江文艺出版社|year=2000.11|isbn=7-5354-1976-3|location=|pages=92}}</ref>。

张元勋后经由北京大学党委决定，依照“就地安置”的原则被安排到曲阜师范大学中文系任教，历任助教、讲师、副教授、教授，并担任先秦两汉六朝文学专业硕士研究生导师，直至1994年退休。张元勋在[[先秦|先秦]]文学[[楚辞|楚辞]]研究领域有颇高造诣，所著《关于屈原放逐的研究》、《论二湘、二司及九歌的篇章结构》两文，分获山东省社会科学优秀成果奖、[[山东省教育厅|山东省教育厅]]社会科学优秀成果奖，并被《[[人民日报|人民日报海外版]]》转载<ref>{{Cite book|title=赣榆文史资料 第15辑 今古名人与赣榆|last=政协赣榆县委员会学习文史委员会|first=|publisher=政协赣榆县委员会学习文史委员会|year=|isbn=|location=|pages=252}}</ref>。其著有23万字的《九歌十辨》成为楚辞学研究的重要资料<ref>{{Cite book|title=赣榆文史资料 第10辑 赣榆现代人物专辑|last=政协赣榆县文史资料研究委员会编；宋世亮主编|first=|publisher=政协赣榆县文史资料研究委员会|year=1994.12|isbn=|location=|pages=125}}</ref><ref>{{Cite book|title=山东社会科学人名辞典|last=梁自洁主编|first=|publisher=济南：山东人民出版社|year=1990.10|isbn=7-209-00709-1|location=|pages=70}}</ref>。2013年4月12日，张元勋去世。其遗体告别仪式于14日上午在曲阜师大举行，北大中文系挽祭<ref>{{cite web |author1=朱毅 |title=千古的浩叹：痛别张元勋先生 |url=http://beijingspring.com/bj2/2010/240/2013418195402.htm |publisher=北京之春 |accessdate=2018-10-31 |archive-url=https://web.archive.org/web/20130627044239/http://beijingspring.com/bj2/2010/240/2013418195402.htm |archive-date=2013-06-27 |dead-url=no }}</ref>。
