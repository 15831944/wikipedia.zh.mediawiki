'''张敏'''（{{bd|1925年||2017年|7月20日|catIdx=}}），[[河北|河北]][[定州|定州]]人，[[中国人民解放军中将|中国人民解放军中将]]<ref name=shishi/>。

==生平==
张敏是河北定县（今定州）人，1937年10月参加[[中国共产党|中国共产党]]领导的革命，1938年4月加入中国共产党。[[抗日战争|抗日战争]]时期，历任宣传员、干事、分队长、连副政治指导员、参谋等职，参加了讨伐顽军、反“扫荡”、反“蚕食”等等战役战斗。[[第二次国共内战|解放战争]]中，历任干事、大队总支书记、师直属队特派员、营政治教导员等职，先后参加了[[辽沈战役|辽沈战役]]、[[平津战役|平津战役]]、[[广州战斗_(1949年)|广州战斗]]等战役战斗。[[中华人民共和国|中华人民共和国]]成立后，历任团政治处副主任、主任，团副政委、政委、团长，副处长、处长、工区主任、[[中国西昌卫星发射中心|第二十七试验训练基地]]司令员等职，参加了[[抗美援朝战争|抗美援朝战争]]。后任[[中华人民共和国国防科学技术工业委员会|国防科学技术工业委员会]]参谋长<ref name=shishi/>。

张敏是[[第八届全国政协委员|第八届全国政协委员]]<ref name=shishi/>。1955年被授予少校军衔，1988年被授予中将军衔<ref name=shishi/><ref>{{citenews|author=徐平 |title=重新实行军衔制 |url=http://chn.chinamil.com.cn/jfdg/2011-07/13/content_4473591.htm |accessdate=2012-05-05 |newspaper=中国军网 |deadurl=yes |archiveurl=https://web.archive.org/web/20130511192908/http://chn.chinamil.com.cn/jfdg/2011-07/13/content_4473591.htm |archivedate=2013-05-11 }}</ref>。曾获三级[[独立自由勋章|独立自由勋章]]、三级[[解放勋章|解放勋章]]、[[独立功勋荣誉章|独立功勋荣誉章]]<ref name=shishi/>。

2017年7月20日，张敏在[[北京|北京]]病逝，享年92岁<ref name=shishi>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1763469  |title = 原国防科工委参谋长张敏中将逝世，享年92岁 |author =  |date =  2017-08-15 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
