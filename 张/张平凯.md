'''张平凯'''（{{bd|1910年|12月26日|1990年|11月22日}}）[[湖南省|湖南省]][[平江县|平江县]]人，[[中国人民解放军开国少将|中国人民解放军开国少将]]<ref name=hxrw>{{Cite web |url = http://www.library.hn.cn/hxrw/xdrw/jfj/zhangpingkai.htm |title =  张平凯|author =  |date =  |publisher = 湖湘人物 |language =  |accessdate = 2017-04-30 |quote =  }}</ref><ref name=cpc>{{Cite web |url = http://dangshi.people.com.cn/GB/165617/165618/166491/167909/12101682.html |title = 张平凯 |author =  |date =  |publisher = 中国共产党新闻网 |language =  |accessdate = 2017-04-30  |quote =  }}</ref>。

==生平==
张平凯生于平江县龙门镇枫树下村的一个贫苦农民家庭。1925年参加秘密农会。1926年加入[[中国共产主义青年团|中国共产主义青年团]]，1927年加入[[中国共产党|中国共产党]]。1928年由地方游击队转入[[中国工农红军|中国工农红军]]。历任[[红五军|红五军]]第三纵队士兵委员会委员长、红五军[[士兵委员会|士兵委员会]]副委员长、红五军党委代理书记、[[红三军团|红三军团]]炮兵营政委兼军团直属队总支书记、军委炮兵团政委、[[红一方面军|红一方面军]]后委副书记、[[福建|福建]]独立十师政委、中南作战军一纵队政委<ref name=hxrw/>。

张平凯先后参加了保卫[[井冈山|井冈山]]、两次攻打[[长沙|长沙]]、[[中央苏区|中央苏区]]反“围剿”斗争和[[二万五千里长征|二万五千里长征]]。1931年3月，红三军团后方留守处主任[[何长工|何长工]]被调往前线指挥作战，张平凯接替工作后，组织了三个战时流动后方医院，建立了五个固定医疗点，还组建了新兵训练营及俘虏官兵训练营,建立了一个修械所，受到[[彭德怀|彭德怀]]赞扬<ref name=hxrw/>。

[[抗日战争|抗日战争]]时期，张平凯任[[八路军|八路军]]总部随营学校党总支书记、[[中国人民抗日军政大学|中国人民抗日军政大学]]二分校政治部副主任、[[晋察冀军区|晋察冀军区]]直属政治部主任、军区生产民运部部长。抗日战争初期，张平凯在敌后开展部队教育工作，与学员一起挖[[窑洞|窑洞]]，用土坯垒成课桌，并且开荒种菜。1940年到1944年，在晋察冀军区工作，负责在敌后游击区开展[[大生产运动|大生产运动]]<ref name=hxrw/>。

[[第二次国共内战|第二次国共内战]]时期，张平凯任[[冀察热辽军区|冀察热辽军区]]纵队政治部主任、[[冀热辽军区|冀热辽军区]]后勤司令部参谋长、[[第四野战军|第四野战军]]后勤司令部参谋长、第四野战军南下工作团行政秘书长、铁道部办公厅主任、[[中南军政大学|中南军政大学]]湖南分校副政委、代理党委书记。第二次国共内战中，张平凯主要从事军需后勤工作，参与领导[[辽沈战役|辽沈战役]]、[[天津战役|天津战役]]的后勤保障工作<ref name=hxrw/>。

[[中华人民共和国|中华人民共和国]]成立后，张平凯任[[东北军区|东北军区]]后勤部副政委、[[中国人民志愿军|中国人民志愿军]]后勤部副政委兼政治部主任。[[朝鲜战争|朝鲜战争]]爆发后，张平凯协助[[李聚奎|李聚奎]]等人准备粮秣、油料、军械，很快便将参战部队两个月的粮食、10个基地的弹药以及大量饲料、油料补充齐，并且组建、扩建了许多后方医院，组织了中朝联运铁路指挥部。此后，历任[[中国人民解放军后勤学院|中国人民解放军后勤学院]]副政委、[[山西省军区|山西省军区]]副政委等职。他是[[第五届全国政协委员|第五届全国政协委员]]<ref name=hxrw/>。1955年被授予少将军衔<ref name=hxrw/><ref>{{cite news|title=揭秘：中共首批开国将军的级别是怎样划分的？ |url=http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |accessdate=2012-06-15 |newspaper=凤凰网（转自摘自《党史博览》2010年第5期） |date=2010-06-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20120620091504/http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |archivedate=2012-06-20 }}</ref>。

1990年11月22日，张平凯在[[北京|北京]]病逝，享年80岁<ref name=hxrw/>。
