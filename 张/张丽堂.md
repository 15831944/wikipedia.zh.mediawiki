'''張麗堂'''（{{bd|1935年|12月20日|||catIdx=Zhang张}}），字'''裕淳'''，[[台湾|台湾]][[台南|台南]]人，中華民國政治人物，[[佛教|佛教]]徒。曾任[[臺南市市長|臺南市市長]]、台灣省民政廳廳長、台灣省政府物資局局長。

== 早年生平 ==
張麗堂於[[昭和|昭和]]十年（1935年）出生於今[[臺南市|臺南市]]。在[[台灣日治時期|日治時期]]進入鹽水小學校附設幼稚園就讀，小學時因家人搬至臺南市番薯崎（今[[忠義路|忠義路]]），張麗堂進入明治公學校（今成功小學）就讀。在三年級時因適逢[[二戰|二戰]]期間[[美軍|美軍]]空襲，全家疏散到[[鹽水區|鹽水]]鄉下。戰後才搬回台南市，進入永福國小。{{r|公|page=2-3}}

1948年，張麗堂進入[[臺南一中|臺南一中]]初中部，畢業後，張麗堂於1951年考入[[臺南師範學校|臺南師範學校]]。{{r|公|page=14}}1954年畢業，服預備士官役後，先於雲林縣飛沙國小、台南市永福國小擔任[[教師|教師]]，後來因為理想，1958年考入省立法商學校（後來為[[國立中興大學|國立中興大學]]法商學院、今[[國立臺北大學|國立臺北大學]]法律系）社會系，大二轉系至法律系。畢業後，於1962年考上[[律師|律師]]，成立[明正誠財經法律事務所]擔任律師。

== 政途 ==
1964年，張麗堂出任[[國民黨|國民黨]]臺南市黨部委員。{{r|公|page=63}}1968年，張麗堂獲國民黨提名參選[[台灣省議員|台灣省議員]]，成為國民黨提名的三名候選人之一。然而卻因[[黨外|黨外]]的[[蔡介雄|蔡介雄]]投入選舉，最後張麗堂以第四名在應選三席的臺南市選區落選。

== 臺南市市長 ==
1972年9月，張麗堂獲[[國民黨|國民黨]]提名參選[[1972年中華民國縣市長選舉|第七屆臺南市長選舉]]，以105,106票擊敗脫黨參選的市議員[[蘇南成|蘇南成]]75,124票當選市長。

張麗堂就任第七屆台南市長後，推動「臺南觀光年」，大規模整修市內[[古蹟|古蹟]]。完成開拓[[臺南車站|臺南車站]]後站、建設[[東門陸橋|東門陸橋]]、並改善及重建道路等改善交通政績。以及爭取興建[[安平|安平]]新港，取代淤塞的安平舊港，配合安平工業區的開發完成安平市區規劃。並且完成直達[[臺南縣|臺南縣]]的過境道路二等一路（今[[中華東路|中華東路]]）的開闢。{{r|公|page=83-84}}在文化上復刊了《[[臺南文化|臺南文化]]》，保留文獻。並興建中正圖書館及育樂文化中心、成立仁愛之家、國民住宅、創辦全台第一家啟智學校。<ref>〈張麗堂重獲提名競選市長 黨方揚棄舊觀念各方稱道〉，1977年8月12日，《中國時報》六版。</ref> 

然而張麗堂在任內也發生在填平[[臺南運河|臺南運河]]事情上與[[臺南市議會|市議會]]引起爭執，<ref>〈如何填平臺南運河 臺南府會引起爭執〉，1976年12月6日，《自立晚報》六版。</ref> 而張麗堂又與議長[[王奕棋|王奕棋]]交惡，由於王奕棋議長議事主持受議員支持，使國民黨籍議員傾向支持王奕棋，對張麗堂成為掣肘。{{r|大|page=61}}

在市政方面，臺南觀光年遭議員[[蘇南成|蘇南成]]質疑淪為口號，任內工程弊案遭到揭發。此外二等一路的開闢也因位置臨近[[臺南紡織|臺南紡織]]後甲總廠土地，使得與南紡關係密切的張麗堂遭到外界質疑圖利財團。而1976年發生建設局長王欽誠涉入詐賭案，也使張麗堂聲望受到波及，一度遭到省府民政廳提案申誡。<ref>〈[http://catalog.digitalarchives.tw/item/00/36/6b/7a.html 民政廳提報臺南市市長張麗堂，對該市政府建設局前局長王欽誠詐賭案，雖經事先加以防範處置，但仍使案情發生，影響政府聲譽，擬從輕予以申誡一次處分，並依照民選縣市長經考核辦法第十七條規定提報府會核議後，函報內政部備查案] {{webarchive|url=https://web.archive.org/web/20160314013759/http://catalog.digitalarchives.tw/item/00/36/6b/7a.html |date=2016-03-14 }}</ref>

張麗堂後來與台南幫大老吳三連等數人共同創辦[[南台工專|南台工專]]，即現今之南台科技大學，張麗堂為原始創辦人代表人暨當然董事、也因為台灣退出聯合國而創辦 P2P台南市東區[[扶輪社|扶輪社]]社長、正一纸器工業公司董事长等职。
日後張麗堂投入[[南台工專|南台工專]]的創辦，出任南台工專副校長一職。<ref>〈張麗堂獲國民黨器重脫穎而出 有其獨特條件〉，1972年9月23日，《[[中國時報|中國時報]]》六版。</ref>


1977年[[1977年中華民國縣市長選舉|第八屆市長選舉]]中，張麗堂再獲[[國民黨|國民黨]]提名。而[[蘇南成|蘇南成]]則以無黨籍身分再度捲土重來。最後張麗堂以113,627票敗給蘇南成的131,504票，落選市長。

== 卸任後 ==
1978年，張麗堂出任南台工專副校長。同年，蒙蔣經國院長徵召，擔任[[行政院|行政院]]研究發展考核委原會副主委。1981年8月出任国民党台南县党部主委，其后歷任国民党中央委员会社会工作会专任委员，台湾省政府委员、民政厅厅长等职。1992年当选第二届[[国大代表|国大代表]]，翌年任台湾省选举委员会主委。1994年又任台湾省政府物资局局长。<ref name="kmt">{{cite book|title=《中国国民党百年人物全书》|publisher=团结出版社|year=2005}}</ref><ref>{{cite web|url=http://www.huaxia.com/lasd/hxrwk/ddrw/tw/2003/07/506794.html|title=张丽堂|publisher=华夏经纬}}</ref>2001年7月，張麗堂一度因物資局任內發生代購礦砂弊案遭調查局台北市調查處約談。<ref>[http://old.ltn.com.tw/2001/new/jul/18/today-c3.htm 礦砂弊案 物資局前局長張麗堂到案]，2001年7月18日，《[[自由時報|自由時報]]》</ref>

張麗堂还是国民党第十三、十四届中央委员，十五届中央评议委员。<ref name="kmt" />

1974年皈依密宗白教傳人貢噶老人，1977奉 上師之命，成立財團法人台灣省台南市噶瑪噶居法輪中心(後改名為財團法人貢噶寺)，設董事會，作為密宗白教噶瑪噶舉的法脈傳承之組織，擔任董事長，推行弘法、利生等工作。於安平區億載金城近鄰，興建莊嚴之金剛梵宇—貢噶寺，顯密雙修。

1981年，台南幫大老吳尊賢先生創辦[吳尊賢公益文教基金會]，張麗堂擔任董事長至今，對教育界、學術界、醫療界、社會各公益團體都有長久以來的捐助貢獻。

1989年，擔任[道院台灣總主院[首席統掌]]，改建兩層舊樓房為地下四層，地上十七層之[道慈大樓]為宏揚大道之道場。

1995年，擔任[社團法人世界紅卍字會台灣總會]理事長，至2008年。

1996年，成立[張麗堂文教助學基金會]，每年頒發 [台南啟智學校]、[永福國小]獎助學金，對創辦的啟智學校一直頒發獎助學金至今。

2002年，道院成立 [道慈事業發展基金會]發展慈善事業，以捐助孤兒助學為主要項目，張麗堂擔任董事長至今。

2010年，創辦[台灣世界道慈學會]，擔任理事長至今，弘道行慈未曾中斷。

== 家族 ==
*張麗堂父親張金來是名木匠，在鹽水與張麗堂母親張羅世結婚後至臺南市學習木匠，再回到[[鹽水區|鹽水]]開業。
*張麗堂妻子林芙美為[[麻豆區|麻豆]]人，兩人是在張麗堂就讀省立法商學院時認識，婚後育有三個兒子。
