'''张海'''（{{bd|1941年|9月||catIdx=Z张}}），出生于[[中国|中国]][[河南省|河南省]][[偃师市|偃师市]]，中国当代著名[[书法|书法]]家。现任[[中国书法家协会|中国书法家协会]]主席，[[郑州大学|郑州大学]]美术学院院长。[[中国人民政治协商会议全国委员会常务委员会|全国政协常委]]，[[中华人民共和国国务院|国务院]]批准有突出贡献的[[专家|专家]]。曾任中国第八、九、十届[[全国人大代表|全国人大代表]]，河南省文联主席，中国[[河南省书法家协会|河南省书法家协会]]主席，河南省书画院院长等。

== 人生经历 ==
1960年7月就读于河南省[[新乡师范专科学校|新乡师范专科学校]]。

1962年10月参加工作，毕业于河南省新乡师专理化系，大专学历。

1962年10月任[[河南省|河南省]][[安阳市|安阳市]]西大街小学教师。1964年3月任[[河南省安阳市总工会职工学校|河南省安阳市总工会职工学校]]教师、工人文化宫干事。1975年10月任[[河南省安阳市群众艺术馆|河南省安阳市群众艺术馆]]副股长、股长、副馆长。

1981年加入[[中国共产党|中国共产党]]。

1982年3月任河南省书协干事、副秘书长。1985年6月任河南省书协副主席兼秘书长。1986年7月任河南省文联副秘书长、省书协副主席兼秘书长。1991年11月任河南省书协主席。1994年1月任河南省文联副主席、党组成员、省书协主任兼书画院院长。1998年8月任河南省文联副主席、党组成员、省书协主席。

2000年5月任河南省文联主席、党组成员、省书协主席。2000年12月任河南省文联主席、党组成员、省书协主席、中国书协第四届副主席。2001年任河南省文联主席、党组副书记、省书协主席、中国书协第五届主席。

2010年12月29日再次当选中国书协第六届主席。<ref>{{Cite news|url = http://news.ifeng.com/mainland/detail_2010_12/29/3761560_0.shtml|title = 中国书法家协会全国代表大会闭幕 张海当选主席|last = |first = |date = |work = |accessdate = 2010年12月29日}}</ref>

2015年12月8日，任中国书法家协第七届名誉主席。<ref>{{Cite web|url = http://www.thepaper.cn/newsDetail_forward_1406573|title = 中国书法家协会换届：全国政协常委苏士澍当选主席，张海卸任|accessdate = 2016年2月8日|author = 岳怀让|date = 2015-12-08|publisher = }}</ref>
