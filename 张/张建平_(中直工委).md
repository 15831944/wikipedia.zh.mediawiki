'''张建平'''（{{bd|1953年|3月||}}）[[河北|河北]][[平山|平山]]人，[[中国共产党|中国共产党]]官员，[[中国共产党第十七届中央纪律检查委员会|中国共产党第十七届中央纪律检查委员会]]委员，[[第十二届全国政协|第十二届全国政协]]常委<ref name=wy>{{Cite web |url = http://news.163.com/17/0502/15/CJEN00ND00018AOQ_all.html |title = 孟祥锋任中直机关工委常务副书记(正部长级) 张建平不再担任 |author =  |date = 2017-05-02 |publisher = 网易|language =  |accessdate =  |quote =  }}</ref>。

==生平==
1972年1月参加工作，1976年5月加入中国共产党。[[中共中央党校|中央党校]]研究生院经济管理专业毕业，中央党校研究生学历。1972年1月至1975年2月，为[[中共中央办公厅|中共中央办公厅]]管理局北戴河管理处工人。1975年2月至1976年7月，在[[江西|江西]][[进贤|进贤]]中央办公厅[[五七干校|“五七”学校]]劳动。1976年7月至1988年7月，为[[中共中央直属机关事务管理局|中直管理局]]党办、局办干部，研究室副主任。其间，1981年4月至1984年7月在北京市职工业余大学中文专业学习<ref name=wy/>。

1988年7月至1996年8月，任中共中央办公厅调研室政治组、群团组正处级调研员、室机关党委书记。其间，1988年9月至1990年7月在[[中南海业余大学|中南海业余大学]]行政管理专业学习；1993年9月至1996年7月为[[中国人民大学法学院|中国人民大学法学院]]法学专业研究生；1994年4月至1996年8月在[[广东省|广东省]]挂职担任中共[[东莞市|东莞市]]委列席常委、副秘书长<ref name=wy/>。

1996年8月至2000年10月，任中直管理局副局长、局机关党委书记。2000年10月至2005年3月，任中直管理局常务副局长（正局长级）、局机关党委书记。其间，1999年9月至2002年6月在中央党校研究生院经济管理专业学习。2005年3月至2007年1月，任中直管理局局长、局机关党委书记。2007年1月至2011年3月，任中共中央办公厅副主任兼中直管理局局长、局机关党委书记。2011年3月至2013年1月，任中共中央办公厅副主任。2013年1月至2017年4月，任[[中共中央直属机关工委|中共中央直属机关工委]]主持常务工作的副书记（正部长级）、中直党校校长，[[中央党的建设工作领导小组|中央党的建设工作领导小组]]成员<ref name=wy/><ref>{{Cite web |url =http://renshi.people.com.cn/n/2013/0427/c139617-21309261.html  |title = 张建平任中直工委常务副书记 |author =  |date = 2013-04-27 |publisher =中国共产党新闻网  |language =  |accessdate =  |quote =  }}</ref>。

他是[[中国共产党第十七届中央纪律检查委员会|中国共产党第十七届中央纪律检查委员会]]委员，第十二届全国政协常委<ref name=wy/>。
