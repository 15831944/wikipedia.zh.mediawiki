'''张少琴'''（{{bd|1953年|5月||catIdx=}}），[[山西|山西]][[新绛|新绛]]人，是一位中国材料学家、政治人物<ref name=zhonggw>{{Cite web |url =  http://www.workercn.cn/16da/28210/201310/22/131022072403005_5.shtml|title =  中华全国总工会新当选的领导机构成员简历|author =  |date =2013-10-22  |publisher = 中工网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1972年1月参加工作。1972年到1974年，任山西省运城公路段测量设计队工人。1974年到1977年，在[[太原重型机械学院|太原重型机械学院]]工程机械专业学习。1977年到1985年，任太原重型机械学院教师（其间，1983年到1985年在[[英国|英国]][[曼彻斯特大学|曼彻斯特大学]]理工学院做访问学者）。1985年到1987年，任太原重型机械学院力学实验室主任。1987年到1989年，在[[美国|美国]][[奥本大学|奥本大学]]复合材料专业学习，获得博士学位。1989年到1990年，任美国[[俄亥俄州立大学|俄亥俄州立大学]]复合材料专业[[博士后|博士后]]。1990年到1997年，任[[新加坡|新加坡]]国家标准与工业研究院主任研究员、首席工程师、复合材料研究中心副主任。1997年到1999年，任太原重型机械学院院长助理、数理系主任、应用科学系主任。1999年到2000年，任太原重型机械学院常务副院长。2000年到2003年，任太原重型机械学院院长<ref name=zhonggw/>。

2003年1月到2008年1月，任[[山西省人民政府|山西省人民政府]]副省长。2003年5月加入[[中国民主建国会|中国民主建国会]]（民建），后来至2007年任民建山西省委主委。2007年，当选民建第九届中央副主席。2007年到2008年，任民建中央副主席、山西省委主委。2008年到2013年，任[[第十一届全国人大常委会|第十一届全国人大常委会]]副秘书长，民建中央专职副主席。2013年10月，兼任[[中华全国总工会|中华全国总工会]]副主席<ref name=zhonggw/><ref>{{cite news|title=民建中央副主席张少琴来衡考察|url=http://www.hstongzhou.com/TPXW/2011/07/15/content_210009.shtml|accessdate=2012-09-23|newspaper=衡水统战部}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite news|title=全国人大代表信息-张少琴|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=110259|accessdate=2013-04-05|newspaper=全国人大网}}</ref>。2013年至2014年，任[[第十二届全国人大常委会|第十二届全国人大常委会]]副秘书长<ref name=zhangguo>{{Cite web |url = http://hn.ifeng.com/hnzhuanti/hnztc/zuixin/detail_2014_02/28/1915798_0.shtml |title =郭雷任全国人大常委会副秘书长  |author =  |date =2014-02-28  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

他是第十届全国政协委员，第十一届、十二届全国人大常委会委员<ref name=zhonggw/>。
