==生平==
1973年张长东入伍，参加[[中国人民解放军上海警备区|中国人民解放军上海警备区]]警卫师。1975年加入[[中国共产党|中国共产党]]。不久提干，担任宣传干事、干部干事。1978年，参加“[[文革|文革]]”结束后的第二届[[高考|高考]]，考入[[上海师范大学|上海师范大学]]，带职带薪学习。1982年毕业，任宣传教育干事，负责给整个上海地区的部队干部补习初中、高中文化。后因工作成绩突出，由宣传教育干事直接提拔成宣传文化处副处长，以后又任处长，还任[[北京大学|北京大学]]上海武警函授站站长。1990年底，调到[[武警总部|武警总部]]任正团职秘书<ref name=tao/>。

1994年张长东调回上海，历任[[中国人民武装警察部队政治学院|武警上海指挥学校]]党委书记及校长。同年，张长东经请示上级同意，在武警上海指挥学校试办大专班，成为武警部队全国20余所初级指挥学校里率先试办大专班的学校。1995年至1997年，第一届大专班办完后，武警部队在该校召开“武警部队教育改革座谈会”，[[中央军委副主席|中央军委副主席]][[迟浩田|迟浩田]]亲临会场接见与会代表。1997年，张长东和该校党委其他同志提出开办本科班，经上级主管部门支持，该校在武警部队全国20余所指挥学校中率先开办武警指挥专业本科班，1998年开始招生。2000年，该校更名为武警上海指挥学院，全面招收本科生，成为[[中国人民解放军总参谋部|中国人民解放军总参谋部]]和[[中华人民共和国教育部|中华人民共和国教育部]]批准的本科院校<ref name=tao/>。2005年，张长东升任[[武警上海总队|武警上海总队]]副政委<ref name=tao/><ref>{{Cite web |url = http://news.hexun.com/2008-02-04/103522253.html |title = 武警上海总队与新闻界迎新座谈 |author =  |date = 2008-02-04 |publisher =和讯  |language =  |accessdate =  |quote =  }}</ref>。[[2010年上海世博会|2010年上海世博会]]期间，张长东任世博园区武警前指的一名指挥员<ref name=zhengxie/>。2011年至2012年间，卸任武警上海总队副政委职务<ref>{{Cite web |url = http://cpc.people.com.cn/n/2012/0701/c87228-18417252.html |title =武警百名将军签名纪念封展出  |author =  |date =2012-07-01  |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。

张长东先后参加过[[北京大学|北京大学]]法律专业函授学习、[[中国人民解放军空军政治学院|中国人民解放军空军政治学院]]经济管理本科学习。1996年至1999年，成为上海首批攻读[[中共中央党校|中共中央党校]]政治学在职研究生的学生，在毕业典礼上作为优秀学员讲话。后到[[复旦大学|复旦大学]]国际关系学院担任访问学者一年。2002年，到[[上海交通大学|上海交通大学]]学习硕士外语课程。随后考上[[中国人民解放军国防大学|中国人民解放军国防大学]]作战指挥系硕士研究生，2007年1月毕业<ref name=tao/>。

他曾任第三、四、五届上海市[[宝山区|宝山区]]人大代表<ref name=tao/>。[[上海市政协|上海市政协]]委员<ref name=zhengxie>{{Cite web |url = http://sh.eastday.com/qtmt/lh2011/u1a848597.html |title =政协委员:应把世博无线数字电视信息平台常态化  |author =  |date = 2011-01-19 |publisher =东方网  |language =  |accessdate =  |quote =  }}</ref>。中国人民解放军军事管理研究会副会长，中国人民解放军教育学会副会长。他是上海美术家协会会员，其油画作品曾展览并获奖<ref name=tao/>。
