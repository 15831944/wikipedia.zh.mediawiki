'''张泉灵'''（{{bd|1973年|6月8日|||catIdx=Z张}}），祖籍[[浙江|浙江]][[宁波|宁波]]，出生于[[上海市|上海市]]，毕业于[[北京大学|北京大学]][[德语|德语]]文学系，曾任[[中国中央电视台|中国中央电视台]]新聞中心[[记者|记者]]、[[节目主持人|节目主持人]]<ref>{{cite web|title=网传主持人张泉灵离职央视 知情人:还没结果|url=http://news.163.com/15/0522/20/AQ8DHEKH00011229.html?baike|website=网易|accessdate=2015-08-29|date=2015-05-22}}</ref>，目前担任[[傅盛战队|傅盛战队]]顾问<ref>{{cite web|title=张泉灵加盟傅盛战队 担任紫牛基金合伙人|url=http://money.163.com/15/0829/10/B2693JME00253G87.html|website=网易|accessdate=2015-08-29}}</ref>，紫牛基金合伙人。

== 简介 ==
1996年，张泉灵毕业于[[北京大学|北京大学]][[德语|德语]]文学系。<ref name="smex " >{{cite web |url= http://media.people.com.cn/GB/7347165.html|title=张泉灵简介 |accessdate= 2008年|work=|publisher=人民网 |language = zh-hans}}</ref>

1997年，张泉灵任职中央电视台海外中心专题部[[主持人|主持人]]和[[编导|编导]]。<ref name="smex " />

2000年，张泉灵担任新版《[[东方时空|东方时空]]》的[[主持人|主持人]]。<ref name="smex " />

== 事业 ==
张泉灵是个性格大气的女人。<ref name="gmxsw " />她曾经担任[[张健|张健]]横渡[[英吉利海峡|英吉利海峡]]、[[雅典奥运会|雅典奥运会]]、[[连战|连战]]及[[宋楚瑜|宋楚瑜]]大陆之行、抗击[[非典|非典]]第一线、[[罗布泊|罗布泊]]无人区、[[阿富汗|阿富汗]]战乱地震灾区、[[汶川地震|汶川地震]]灾区的直播主持人。<ref name="gmxsw " />

2015年7月上旬，张泉灵确认从央视离职，7月13日[[傅盛战队|傅盛战队]]官方微博对外公开宣布，张泉灵以顾问形式加盟[[傅盛战队|傅盛战队]]。张泉灵还加盟傅盛旗下[[紫牛基金|紫牛基金]]合伙人。

2015年9月9日早上8点，张泉灵发长文《生命的后半段》。

2017年開始播映的中國網絡辯論綜藝《[[奇葩說|奇葩說]]》中，張泉靈擔任奇葩議長的角色。

== 个人生活 ==
不明媒体和网站曾疯传张泉灵的丈夫是[[曲向东|曲向东]]，此事遭到曲向东本人亲自出面澄清，两人只是同行并非夫妻。<ref>{{cite web |url=http://ent.163.com/09/1121/11/5OL0A9OV00031H2L.html |title=央视主持曲向东发声明:与张泉灵是同行非夫妻 |accessdate= 2009年|work=|publisher=网易 |language = zh-hans}}</ref>

张泉灵的丈夫是李铁（1955年10月24日-），现任中国国家发改委城市和小城镇改革发展中心主任，比她大18岁，两人2000年结婚。

2006年7月23日，张泉灵生下一个男孩李晨华（2006年7月23日-），取名叫做“晨晨”。<ref>{{cite web |url=http://ent.ifeng.com/idolnews/mainland/detail_2012_03/23/13394914_0.shtml |title=央视美女主播张泉灵六岁儿子萌照曝光 |accessdate= 2012年|work= |publisher=凤凰网 |language =zh-cn}}</ref>
