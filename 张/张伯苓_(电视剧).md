《'''张伯苓'''》，中国大陆电视剧，导演[[周友朝|周友朝]]，主演唐国强、刘冠雄、赵亮、马晓伟、雷镇语、傅晶。

== 剧情 ==
{{See Also|张伯苓}}
该剧从张伯苓弃武从文放弃海军职业转投教育职业开始，讲述张伯苓教育救国的人生事迹。<ref>{{cite web |url=http://ent.sina.com.cn/v/2005-03-28/2243688359.html |title=资料：电视剧《张伯苓》剧情简介 |accessdate=2005 |publisher=新浪网 |work= |language =zh-cn}}</ref><ref>{{cite web |url= http://edu.sina.com.cn/y/news/2005-09-09/191643524.html|title= 传世精品剧《张伯苓》亮相CETV-1|accessdate=2005年9月9日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

== 花絮 ==
