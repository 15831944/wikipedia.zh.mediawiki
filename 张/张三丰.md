'''張三-{丰}-'''（{{Fact|1247年—1458年|time=2018-11-21}}），本名'''全一'''，字'''君宝'''，号'''三-{丰}-'''，[[元朝|元]]末[[明朝|明]]初[[道士|道士]]，[[辽西|辽西]][[懿州（辽朝）|懿州]]人（或為[[福建省|福建省]][[邵武|邵武]]人<ref>福建省邵武市博物館學者傅喚民指出張三-{丰}-是福建省邵武市人</ref>）<ref name="MingShi">[[s:明史/卷299|明史•卷二百九十九•列传第一百八十七•方伎]]</ref>。<!-- 不明確的斷語～他的生平、事迹、著作等的真实性存在很大争议。-->

據[[武當山|武當山]][[道教|道教]]中人相傳，張三-{丰}-<ref>{{cite web|url=http://www.chinawudang.com/wdjhb/wdws_wgcs.asp?WebShieldDRSessionVerify=4RjNoeixt8vPKgYigtpu&Language=ft|title=「中國武當網」有關張三-{丰}-的簡介|deadurl=yes|archiveurl=https://web.archive.org/web/20171202203111/http://www.chinawudang.com/wdjhb/wdws_wgcs.asp?WebShieldDRSessionVerify=4RjNoeixt8vPKgYigtpu&Language=ft|archivedate=2017年12月2日|df=}}</ref>於[[湖北|湖北]][[均县|均县]][[武当山|武当山]]修道，供奉[[玄天上帝|玄天上帝]]为主神。另據現存[[武當派|武當派]]武術各分支派傳人及弟子相傳，亦是武當派武術的開山祖師。<ref>{{cite web|url=https://www.douban.com/note/360442844/|title=國家體委科研課題《武當拳派源流、拳系及內容研究》工作研究員之論文 -- 太极拳的源与流 (文／欧阳学忠) -- 转载、原文自 "武谈、太极拳源流之谜"。山西科技出版社|deadurl=yes|archiveurl=https://web.archive.org/web/20171231155451/https://www.douban.com/note/360442844/|archivedate=2017年12月31日|df=}}</ref>

== 生平簡介 ==
按《明史·方伎传》<ref name="MingShi"/>记载，张三丰曾在[[武当山|武当山]]结庐而居，后来在[[宝鸡|宝鸡]]死而复活，还和明朝[[朱椿_(明朝)|蜀献王]]见过面。[[明太祖|明太祖]]、[[明成祖|明成祖]]派人到处寻找张三丰但都没有找到。后来[[明英宗|明英宗]]给他赠了个“通微显化真人”的封号。据说活到了211岁，他191岁时与41岁妻子结婚。

== 主要思想 ==
張三丰抱持[[三教合一|三教合一]]思想，學貫[[儒家|儒]]、[[佛教|釋]]、[[道教|道]][[三教|三教]]， 更精研於[[宋明理學|理學]]。張三丰認為“[[道|道]]”為三教共同之源，道生天地萬物，含[[陰陽|陰陽]]動靜之機，具造化玄微之妙，統無極，生[[太極|太極]]，是萬物的根本、本始和主宰。

張三丰認為正邪之別比三教之分重要。他說，儒、佛、道都講道，它們的功用都是“修身利人”，“儒離此道不成儒，佛離此道不成佛，仙離此道不成仙”，[[儒家|儒家]]“行道濟時”，[[佛家|佛家]]“悟道覺世”，[[道教|道家]]“藏道渡人”，同[[孔子|孔子]]一樣，[[老子|老子]]所傳的也是“正心、修身、齊家、治國、平天下”的理論。儒家修養人道，仙家修煉[[仙道|仙道]]。張三丰以修人道為煉仙道的基礎，強調無論貴賤賢愚，老衰少壯，只要素行陰德，仁慈悲憫，忠孝信誠，全於人道，離仙道也就自然不遠了。他把道家的內煉思想同儒家的修正身心學說合在一起，說：“人能修正身心，則真精真神聚其中，大才大德出其中。”

他的修煉方法主張[[性命雙修|性命雙修]]，當時另有一名道士「張三峰」提出[[男女雙修|男女雙修]]，倡導[[房中術|房中術]]，此張三峰非武當張三丰，「三峰」不同「三丰」此多被後人混淆。

== 著作 ==
有《張三丰先生全集》，由[[清朝|清朝]][[李西月|李西月]]编著，是明清道教[[隐仙派|隐仙派]]的作品汇集，主要章节有：《大道論》、《玄機直講》、《玄譚全集》、《玄要篇》、《無根樹詞》、《云水前集》、《云水后集》等。

== 榮譽 ==
[[明英宗|明英宗]]赐號“通微显化真人”；[[明宪宗|明宪宗]]特封“韬光尚志真仙”；[[明世宗|明世宗]]赠为“清虚元妙真君”。

== 衍生教派 ==
明時出現的道教派別多與張三丰有所聯繫，如：[[武當派|武當玄武派]]、千师仙道派,[[寶雞三丰派|寶雞三丰派]]、[[武當派|武當三丰派]]、[[王屋山三丰派|王屋山三丰派]]、[[武當派|三丰自然派]]、[[三丰蓬萊派|三丰蓬萊派]]、[[三丰日新派|三丰日新派]]等相繼出現。

清[[道光|道光]]年間，出現一位名叫[[李西月|李西月]]的人，自稱遇張三丰親授秘訣，講道納徒，活躍於[[四川|四川]][[樂山|樂山]]一帶，創立了當時最大的道派之一。

張三丰晚年把火龍真人隱仙派法脉传承，传予太和四仙中的杨善登，杨善登亦是千师仙道派的開山祖師，直到清末千师仙道派部分传承因为战乱丢失，幸好苏齐把民间道教混合进来，并易名千师仙馆，現时千师仙馆由千师三子发展到有五个分馆。

在[[金庸|金庸]]的[[武俠小說|武俠小說]]中，張三丰是最絕頂的高手之一，武功造詣達出神入化、深不可測的地步，招牌武功為自創的「[[武當太極拳|太極拳]]」和「[[太極劍|太極劍]]」。雖然年事已高鮮有出手，實已達從心所欲、無不如意的最高境界。

張三丰原名「張君寶」，最先出現於《[[神鵰俠侶|神鵰俠侶]]》結尾，是[[少林|少林]]藏经阁僧人[[覺遠|覺遠]]的徒弟，碰上「神雕大侠」[[楊過|楊過]]指導了几招掌法，名為「推心置腹」、「四通八達」、「鹿死誰手」。后来到《[[倚天屠龍記|倚天屠龍記]]》的开头，他随覺遠逃離少林，因緣際會聽到了覺遠臨終时背誦《[[九陽真經|九陽真經]]》。日后他改名‘'''-{三丰}-'''’，以所記得的《九陽真經》，及[[峨嵋派|峨嵋派]]創派祖師[[郭襄|郭襄]]所贈之一對少林鐵羅漢內所藏之少林羅漢拳為基礎，創建了[[武當派|武當派]]一派武功，成為一代宗師。張三丰共收有七名弟子：[[宋遠橋|宋遠橋]]、[[俞蓮舟|俞蓮舟]]、[[俞岱巖|俞岱巖]]、[[張松溪|張松溪]]、[[張翠山|張翠山]]、[[殷梨亭|殷梨亭]]、[[莫聲谷|莫聲谷]]，合稱[[武當七俠|武當七俠]]。其中[[張翠山|張翠山]]是《[[倚天屠龍記|倚天屠龍記]]》男主角[[張無忌|張無忌]]的父亲，因此[[张无忌|张无忌]]尊称张三丰为“太师父”。

在金庸筆下，張三丰是[[南宋|南宋]]末年的人（生於[[淳祐|淳祐]]六年四月初九，西元1246年5月15日），[[元順帝|元順帝]]至元二年時滿九十歲，至張翠山自刎時剛好足百歲。

=== 武功 ===
;[[太極拳|太極拳]]
[[武當派|武當派]]鎮派之寶，集道家武學之大成的拳法，講究太極圓轉，永無止境，用意而不用力。

;[[太極劍|太極劍]]
[[武當派|武當派]]鎮派之寶，集道家武學之大成的劍法，講究神在劍先，綿綿不絕，以不變應萬變。

;純陽無極功
[[武當派|武當派]]絕頂內功心法，道家功法之精華，也是護體防身必須修煉的功法。

;神門十三劍
傳弟子殷梨亭，這十三劍專攻對手的神門穴使對方失去戰力，因而得名。小說中，殷以此劍幫助回歸的張翠山夫婦。

;武當九陽功
年輕時曾聽取覺遠背誦部分九陽神功心法而自創的高級內功，由於張君寶悟性為三人最高外加當時無武功底子，故對心法片段別有心得，創出後比峨嵋九陽功及少林九陽功略勝一籌。
