'''张存浩'''（{{bd|1928年|2月23日||}}），[[中国|中国]][[物理化学|物理化学]]家。中国高能化学激光奠基人、分子反应动力学奠基人之一，长期从事催化、火箭推进剂、化学激光、分子反应动力学等领域的研究，取得多项国际先进成果。

==求学经历==
籍贯[[山东|山东]][[无棣|无棣]]，生于[[天津|天津]]书香世家。父亲[[张铸|张铸]]早年留学美国，曾任天津化工局高级工程师。母亲龙文瑗系[[云南|云南]][[哈尼族|哈尼族]]人。张存浩2岁起，母亲就开始教他识字。[[抗日战争|抗日战争]]爆发时，张存浩正在读小学。因不愿自己的长子接受日本奴化教育，母亲龙文瑗将9岁的张存浩交给在[[重庆大学|重庆大学]]任教的姑父[[傅鹰|傅鹰]]、姑母[[張錦_(化學家)|張錦]]抚养。1939年，应[[厦门大学|厦门大学]]校长[[萨本栋|萨本栋]]邀请，傅鹰来厦大任教，张存浩因此跟随姑父姑母到了[[长汀|长汀]]。张存浩在[[长汀中学|长汀中学]]完成了中学学习，1943年，15岁的张存浩考入[[厦门大学|厦门大学]]化学系，一年之后，转入抗战期间陪都[[重庆|重庆]][[中央大学|中央大学]][[南京工业大学|化学工程系]]，1947年毕业。1950年获美国[[密歇根大学|密歇根大学]]硕士学位。<ref>{{cite web |url = http://news.xmu.edu.cn/s/13/t/542/34/78/info144504.htm |title = 国家最高科学技术奖获得者、校友张存浩：厦大是一段难忘的美好时光 |publisher=厦门大学新闻网 |date=2014-01-12| language = }}</ref>

==荣誉==
1992年当选为[[第三世界科学院|第三世界科学院]]院士。[[中国科学院大连化学物理研究所|中国科学院大连化学物理研究所]]研究员，曾任该所所长，国家自然科学基金委员会主任，[[中国科学院|中国科学院]]化学部主任，国际纯粹和应用化学联合会执行局成员。1980年当选为[[中国科学院学部委员|中国科学院学部委员]]（院士）。<ref>[http://sourcedb.cas.cn/sourcedb_ad_cas/zw2/ysxx/hxb/ 中国科学院学部与院士·院士信息·化学部] {{webarchive|url=https://web.archive.org/web/20120427015018/http://sourcedb.cas.cn/sourcedb_ad_cas/zw2/ysxx/hxb/ |date=2012-04-27 }}</ref>

2014年1月10日和物理学家[[程开甲|程开甲]]获[[中国国家最高科学技术奖|中国国家最高科学技术奖]]。<ref>{{cite web |url = http://news.163.com/14/0110/10/9I7IP7400001124J.html|title=2013年国家最高科学技术奖得主：张存浩、程开甲|publisher=网易|date=2014-01-10  | language = }}</ref>
