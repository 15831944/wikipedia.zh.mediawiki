'''张茅'''（{{bd|1954年|2月||catIdx=Zhang张}}），[[山东|山东]][[巨野|巨野]]人；[[北京师范大学|北京师范大学]]历史系毕业，[[中国人民大学|中国人民大学]]中共党史专业硕士研究生，[[北京大学|北京大学]]经济思想史专业博士，高级经济师。是中聯辦社會工作部部長[[張鐵夫|張鐵夫]]之子，原国务院副总理[[谷牧|谷牧]]的女婿、[[刘燕远|刘燕远]]之夫<ref name="测试">[http://news.takungpao.com/mainland/zgzq/2013-03/1510225.html 谷牧女婿掌工商总局]，大公网。</ref>。1973年9月加入[[中国共产党|中国共产党]]。曾任[[北京市人民政府|北京市人民政府]][[北京市市长列表|副市长]]、[[第29届奥林匹克运动会组织委员会|北京奥运会组委会]]副主席，[[中华人民共和国卫生部|卫生部]]党组书记、副部长等职。现任[[中华人民共和国国家市场监督管理总局|中华人民共和国国家市场监督管理总局]]局长、党组书记。中国共产党第十八届、<ref>{{cite news|title=中国共产党第十九届中央委员会委员名单|url=http://cpc.people.com.cn/19th/n1/2017/1024/c414305-29606192.html|accessdate=2017-10-24|agency=中国共产党新闻}}</ref>。

==早年经历==
1969年，15岁的张茅受“[[文化大革命|文化大革命]]”的影响被迫中断学业，进入北京玻璃仪器厂当工人。[[高考|高考]]制度恢复后，于1978年考入[[北京师范大学|北京师范大学]]历史系学习；1982年本科毕业后，考入[[中国人民大学|中国人民大学]]，攻读中共党史专业硕士学位。1984年，回到北京玻璃总厂，任党委副书记；后又兼任玻璃器皿厂厂长；1986年，出任北京玻璃总厂副厂长；1990年，升任北京市一轻总公司副经理、党委常委。

==仕途==
1992年，张茅进入政府部门任职；曾先后担任北京市经贸委副主任，[[海淀|海淀]]区委副书记、副区长 、区长。1997年，出任北京市市长助理、兼北京市新技术产业开发实验区管委会主任。1998年1月，44岁的张茅当选[[北京市人民政府|北京市人民政府]]副市长。在任副市长期间，张茅曾参与[[2008年夏季奥林匹克运动会|北京奥运会]]的筹备工作，先后担任北京奥运组委会副主席、执行副主席。2003年5月，北京“[[非典|非典]]”肆虐，张茅又兼任北京市防治非典联合工作小组医疗组组长，首次在从政履历中触及医疗卫生领域<ref>[http://politics.people.com.cn/n/2013/0323/c1001-20888158.html 新任工商总局局长张茅：挺过非典扛起奥运大旗]</ref>。

2006年6月，张茅转任[[国家发展改革委员会|国家发展改革委员会]]副主任，主导[[中华人民共和国医疗体制改革|医疗卫生体制改革]]。2009年1月，接替[[高强|高强]]出任[[中华人民共和国卫生部|卫生部]]党组书记、副部长<ref>{{cite news|title=张茅任卫生部党组书记 高强另有任用|url=http://www.gov.cn/rsrm/2009-02/12/content_1228562.htm|accessdate=2012-11-16|newspaper=卫生部新闻办公室}}</ref>，晋升正部级。

2013年3月，[[第十二届全国人民代表大会第一次会议|第十二届全国人民代表大会第一次会议]]通过《[[国务院机构改革#2013年|国务院机构改革和职能转变方案]]》，决定将原[[国家人口和计划生育委员会|国家计生委]]和[[中华人民共和国卫生部|卫生部]]合并，张茅离开卫生部，调任国家工商行政管理总局局长、党组书记<ref>{{cite news|title=张茅任国家工商行政管理总局党组书记、局长|url=http://news.qq.com/a/20130322/001140.htm|accessdate=2013-03-22|newspaper=腾讯网}}</ref>。

2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。3月，任第十三届全国政协常委、教科卫体委员会副主任。同月21日，[[国家工商行政管理总局|国家工商行政管理总局]]、[[国家质量监督检验检疫总局|国家质量监督检验检疫总局]]、国家食品药品监督管理总局被撤销，新组建的[[国家市场监督管理总局|国家市场监督管理总局]]正式成立。64岁的原国家工商行政管理总局党组书记、局长张茅任国家市场监督管理总局局长、党组副书记<ref>{{cite web|title=国家市场监督管理总局正式成立|url=http://m.thepaper.cn/newsDetail_forward_2036753=|accessdate=2018-03-21}}</ref>。2018年9月，中共中央决定任命张茅为市场监管总局党组书记，接替因[[长生生物|吉林长春长生公司问题疫苗案件]]而引咎辞职的原党组书记[[毕井泉|毕井泉]]<ref>{{cite web |url = http://samr.saic.gov.cn/xw/zyxw/201809/t20180925_276095.html |title = 中共中央决定张茅同志任市场监管总局党组书记 |author =  |date = 2018-09-25 |publisher = 国家市场监督管理总局官网 |language = zh-cn |accessdate = |quote =  }}</ref>。
