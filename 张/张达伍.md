'''张达伍'''（{{bd|1931年||1982年|5月16日}}）[[江苏省|江苏省]][[邗江县|邗江县]]人，[[中国人民解放军海军东海舰队|中国人民解放军海军东海舰队]]某防救船大队大队长，被[[中央军委|中央军委]]授予“献身海军事业的模范干部”称号<ref name=guhui>{{Cite web |url =  http://www.81.cn/2015hj/2015-04/21/content_6452339.htm|title =张达伍——“把骨灰放在靠海的地方！” |author =  |date =2015-04-21  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1949年夏，刚满17岁的张达伍参加[[中国人民解放军|中国人民解放军]]。1952年，自大连海校毕业，到华东海军航运大队的一艘打捞防救船上工作。从此直到逝世前夕一直从事海上打捞防救工作<ref name=shan/>。1958年加入[[中国共产党|中国共产党]]<ref name=guhui/>。在30多年间，他安全指挥一千多人次下潜作业，参加过打捞[[阿波丸|阿波丸]]沉船、探摸太平轮沉船等急难险重任务上百次<ref>{{Cite web |url =http://www.81.cn/hj/2014-04/14/content_5862738.htm  |title =东海舰队某防救船大队 运用张达伍烈士事迹激励官兵投身强军实践  |author =  |date = 2014-04-14 |publisher =  中国军网|language =  |accessdate =  |quote =  }}</ref>。1980年，张达伍被《[[解放军报|解放军报]]》赞誉为“海上‘老防救’”<ref name=shan/>。

1982年，因长期带病出海，张达伍额头常冒虚汗，并常用手指顶住肝区，忍着病痛指挥打捞沉船，战友们反复催促他住院检查，经检查确诊为[[肝癌|肝癌]]晚期。1982年5月16日，张达伍病逝。在清理其遗物时，发现病历上医生共开病假510天，他却一天都没休息<ref name=shan>{{Cite web |url =  http://www.81.cn/2015hj/2015-05/20/content_6499528.htm|title = 山坳里，父亲的塑像面海而立 |author =  |date =2015-05-20  |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref>。

1984年3月21日，[[中央军委主席|中央军委主席]][[邓小平|邓小平]]签署命令，授予张达伍“献身海军事业的模范干部”称号。防救船大队在大队办公楼前树立了用黑色[[大理石|大理石]]雕成的张达伍像，面海而立<ref name=shan/>。“把骨灰放在靠海的地方！”是张达伍临终前最后一个心愿<ref name=guhui/>。张达伍葬在[[奉化市|奉化市]]烈士陵园，此处已成为[[宁波市|宁波市]]爱国主义教育基地<ref name=shan/>。
