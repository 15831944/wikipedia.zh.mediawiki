'''张塞'''（{{bd|1931年|5月||}}）祖籍[[山东省|山东省]][[莱阳市|莱阳市]]，生于[[黑龙江省|黑龙江省]][[哈尔滨市|哈尔滨市]]，[[中华人民共和国|中华人民共和国]]统计学家、[[中华人民共和国国家统计局|国家统计局]]局长<ref name=shi>{{Cite web |url =http://money.163.com/12/1206/20/8I2OAQ5E00252G50.html  |title =新中国经济学人史——张塞  |author =  |date =2012-12-06  |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1931年5月，张塞生于哈尔滨市。1948年9月参加[[中国共产党|中国共产党]]领导的革命工作。1950年5月加入中国共产党。1954年入[[中国人民大学|中国人民大学]]经济系学习，1958年毕业。此后历任[[山西省计划委员会|山西省计划委员会]]科长、副处长、办公室主任，[[山西省统计局|山西省统计局]]副局长，山西省计划委员会主任，[[中共山西省委|中共山西省委]]常委<ref name=shi/>。1984年6月到1997年2月，任国家统计局局长<ref name=shi/><ref>{{Cite web |url =http://news.qq.com/a/20160126/057429.htm  |title = 王保安2015年5月接任国家统计局局长一职 |author =  |date = 2016-01-26 |publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref><ref>国务院任免人员，中华人民共和国国务院公报1984(18)</ref>。在任国家统计局局长期间，兼任国民经济核算协调委员会主任、[[中华人民共和国国家计划委员会|国家计划委员会]]委员、[[联合国|联合国]]统计委员会委员等职。张塞是高级统计师，曾任[[首都经贸大学|首都经贸大学]]、中国人民大学、[[西安统计学院|西安统计学院]]兼职教授，[[中国社会科学院|中国社会科学院]]数量经济与技术经济研究所兼职研究员<ref name=shi/>。张塞是第七、八届全国人大代表，中共十三大、十四大、十五大代表，第九届全国政协委员<ref>{{Cite web |url =http://www.stats.gov.cn/ztjc/zthd/50znjn/200206/t20020617_35560.html  |title =国家统计局历任局长  |author =  |date =2004-10-18  |publisher = 中华人民共和国国家统计局 |language =  |accessdate =  |quote =  }}</ref>。

张塞是中华人民共和国投入产出表编制及应用开发的先驱。在任山西省统计局领导期间，1979年组织了中华人民共和国第一张地区投入产出表——山西省投入产出表的编制及应用开发，因此获得山西省社会科技进步一等奖。张塞和李强（1954年生）等合作编著《投入产出地区表编制方法》、《投入产出地区表的编制与应用》、《投入产出基层调查表编制方法》等专著。出任国家统计局局长后，倡导编制了全国及各省、自治区、直辖市的投入产出表。张塞组织研究的国家重大软科学应用研究项目《中国1987年投入产出模型应用》，在1991年获得国家统计科研成果特等奖。在国家统计局局长任内，他还积极推动统计工作适应[[社会主义市场经济|社会主义市场经济]]要求，完善国民经济核算体系，推进中华人民共和国统计改革及统计现代化<ref name=shi/>。
