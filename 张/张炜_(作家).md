'''張煒'''（{{bd|1955年|11月|catIdx=Z}}<ref>出生日期一作1956年11月。</ref>），出生於[[山東省|山東省]][[龍口市|龍口市]]，祖籍[[山東|山東]][[棲霞|棲霞]]。原[[煙台|煙台]]師範專科學校（即現在鲁东大学中文系畢業，專業作家；1982年加入[[中國作家協會|中國作家協會]]。現任中国作协副主席、[[山東省|山東省]]作家協會主席，[[山東師範大學|山東師範大學]]兼職教授，[[万松浦书院|万松浦书院]]创办人、院长<ref>{{cite web |url=http://culture.people.com.cn/n/2013/0903/c172318-22793633.html |title=作家张炜：非专业的作家才是最好的作家 |accessdate= 2013 |publisher=人民网 |work= |language=zh-hans}}</ref>。

== 简介 ==
張煒1976年高中毕业后，回原籍在农村参加工副业劳动，<ref name="multiple1">{{cite web|url=http://www.shm.com.cn/yantai/2004-05/03/content_1985.htm|title=《著名作家、山东作协主席　张炜》|accessdate=|work=|publisher=烟台网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20070810082924/http://www.shm.com.cn/yantai/2004-05/03/content_1985.htm|archivedate=2007年8月10日|df=}}</ref>在[[棲霞|棲霞]]縣寺口橡膠廠當技術員、工人；並於同年開始發表文學作品。<br>

1978年考入山东烟台师专中文系。<ref name="multiple1">{{cite web|url=http://www.shm.com.cn/yantai/2004-05/03/content_1985.htm|title=《著名作家、山东作协主席　张炜》|accessdate=|work=|publisher=烟台网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20070810082924/http://www.shm.com.cn/yantai/2004-05/03/content_1985.htm|archivedate=2007年8月10日|df=}}</ref><br>大学毕业后，被分配到[[中共山東省委|中共山東省委]]辦公廳檔案編研處工作；此后历任[[山東省|山東省]]文聯專業作家、創作室副主任，[[山東省|山東省]]作家協會副主席，[[山東師範大學|山東師範大學]][[煙台|煙台]]師範學院中文系教授，文學創作一級；[[中共|中共]][[山東省|山東省]][[龍口市|龍口市]]委副書記、副市長，全國青聯委員，[[山東省|山東省]]青聯副主席、青年文體委主任等職。<br>

1980年張煒開始發表小說、散文、文論等。1987年，第一部長篇小說《[[古船|古船]]》，由[[人民文学出版社|人民文学出版社]]出版。自1999年起，《古船》一書還被[[法國|法國]]教育部和[[巴黎|巴黎]]科學中心列入「[[法國|法國]]高等考試教材及必考書目」。

2007年6月，張煒獲得了由[[美國|美國]][[總統|總統]]亞太顧問委員會頒發的傑出成就獎；成為目前[[亞洲|亞洲]]地區獲得此獎項的唯一一位代表性作家，也是第一位獲得該獎項的[[中國作家協會|中國作家協會]]會員。此外，被評為「世界華語小說百年百強」的長篇代表作《古船》和[[沈從文|沈從文]]的中篇小說《[[边城|边城]]》、[[老舍|老舍]]名著《駱駝祥子》同時入選[[美國|美國]]哈柏·柯林斯出版集團「中國當代作家作品」的翻譯項目。

2011年8月20日，第八届[[茅盾文学奖|茅盾文学奖]]获奖名单公布，张炜历时20余年完成的450万字、十卷本鸿篇巨著《你在高原》，通过实名投票获得了61位评委中的58票，位居第一。授奖词说：《[[你在高原|你在高原]]》是“长长的行走之书”，在广袤大地上，在现实与历史之间，诚挚凝视中国人的生活和命运，不懈求索理想的“高原”。张炜沉静、坚韧的写作，以巨大的规模和整体性视野展现人与世界的关系，在长达十部的篇幅中，他保持着饱满的诗情和充沛的叙事力量，为理想主义者绘制了气象万千的精神图谱。《你在高原》恢宏壮阔的浪漫品格，对生命意义的探寻和追问，有力地彰显了文学对人生崇高境界的信念和向往。<ref>{{cite web |url=http://book.sina.com.cn/news/c/2011-09-19/1503291094.shtml|title=《第八届茅盾文学奖获奖作品授奖词》|accessdate=2011年9月19日|work=|publisher=新浪网|language= zh-hans}}</ref><ref>{{cite web|title=http://www.chinanews.com/cul/2011/08-21/3272422.shtml|url=http://www.chinanews.com/cul/2011/08-21/3272422.shtml|title=何建明:61位茅奖评委绝大多数看完了《你在高原》|publisher=中新网|accessdate=2011-08-21}}</ref>

2016年12月，当选中国作协副主席<ref>{{cite web|accessdate=2016-12-05|url=http://www.thepaper.cn/newsDetail_forward_1572959|title=铁凝第三次当选中国作协主席，贾平凹首次当选副主席|publisher=澎湃新闻}}</ref>。
== 作品风格 ==
张炜早期作品主要着墨于两性之间纯淡而朦胧的柔情，显得纤巧柔美。后期转入对农村现实的揭示，表达对人性的深入思考。文风从纤细敏感走向深厚沉郁。后期作品更多地在思考中国文化的命运和出路的问题，包括传统文化的现代化改造问题和知识分子的精神自救问题。<ref name="multiple1">{{cite web|url=http://www.shm.com.cn/yantai/2004-05/03/content_1985.htm|title=《著名作家、山东作协主席　张炜》|accessdate=|work=|publisher=烟台网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20070810082924/http://www.shm.com.cn/yantai/2004-05/03/content_1985.htm|archivedate=2007年8月10日|df=}}</ref>

有的文学评论家评点他为“乡土文学作家”、“寻根派作家”。<ref>{{cite web|url=http://www.shm.com.cn/yantai/2004-05/03/content_1983.htm|title=《张炜：创作的丰富多样》|accessdate=2004年5月3日|work=|publisher=烟台网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20041029220600/http://www.shm.com.cn/yantai/2004-05/03/content_1983.htm|archivedate=2004年10月29日|df=}}</ref>
