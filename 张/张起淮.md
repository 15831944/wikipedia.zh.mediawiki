'''张起淮'''（{{bd|1955年|||}}）<ref name=nanhua>{{Cite web |url =  http://uscnews.usc.edu.cn/info/1037/11255.htm|title =  北京著名法学家张起淮教授来我校学术交流并被聘为客座教授|author =  |date =2006-11-13  |publisher = 南华大学新闻网 |language =  |accessdate =  |quote =  }}</ref>，籍贯不详，[[中华人民共和国|中华人民共和国]]律师。

==生平==
张起淮从小在军队大院长大。14岁实现了自幼的愿望，成为一名[[中国人民解放军空军|中国人民解放军空军]]战士。18岁加入[[中国共产党|中国共产党]]。27岁便成为当时中国人民解放军空军最年轻的航空兵“团参谋长”。他申请到[[空军指挥学院|空军指挥学院]]进修，1987年自该院指挥系毕业。自空军指挥学院进修毕业回部队后，张起淮奉命处理一起“飞机运输羊毛走私案”。在[[内蒙古大学|内蒙古大学]]的两位法学教授鼓励下，已是中校军衔的张起淮学习法律知识，并参加全国律师资格考试。备考期间，全军在张起淮所在基地进行战备训练，召开装备现场会，身为参谋长的张起淮一边频繁到37公里外的[[呼和浩特市|呼和浩特市]]参加全国律师资格考试培训，一边准备迎接装备现场会。装备现场会开得很成功，张起淮所在基地获三个“先进单位称号”。1990年，张起淮在[[内蒙古自治区|内蒙古自治区]]参加了全国律师资格考试，[[中华人民共和国司法部|中华人民共和国司法部]]公布的分数线为320分，张起淮考了311分，随后接到[[内蒙古自治区司法厅|内蒙古自治区司法厅]]通知：司法部发文，少数民族地区考生加10分录取，张起淮乃以321分取得律师资格证<ref name=jingcai>{{Cite web |url =  http://www.sohu.com/a/116436952_455412|title = 专访律师张起淮：为“梦”活出精彩 |author =  |date =2016-10-18   |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

不久，张起淮调任[[中国人民解放军南京军区|中国人民解放军南京军区]]法律顾问处主任。任内经手大批案件，张起淮由此成为中华人民共和国首批军队律师。在南京军区服役数年，张起淮取得了军队律师资格证。1996年，张起淮由军委空军司令员点名调到[[北京|北京]]任军委空军法律顾问处主任律师。此后张起淮赢得多起诉讼，在军队名声鹊起，当时他已是上校军衔。此后他多次出国，成为首位在[[剑桥大学|剑桥大学]]做学术演讲的中华人民共和国律师；后又到[[瑞典|瑞典]][[斯德哥尔摩大学|斯德哥尔摩大学]]、[[美国|美国]][[宾夕法尼亚大学|宾夕法尼亚大学]]、[[澳大利亚|澳大利亚]][[悉尼大学|悉尼大学]]等国外高校进行学术交流。张起淮曾率[[中国民航大学|中国民航大学]]队参加[[迪拜|迪拜]]国际航空法模拟法庭，获团体第三名、单项第一名。张起淮还代理多起涉外诉讼案件，成为[[中国国际经济贸易仲裁委员会|中国国际经济贸易仲裁委员会]]首位来自[[中国人民解放军|中国人民解放军]]的国际仲裁员，以及[[英国|英国]]皇家仲裁员协会御准仲裁员<ref name=jingcai/>。

离开军队后，张起淮创立蓝鹏律师事务所并担任主任律师。同时兼任[[中国民航大学|中国民航大学]]兼职教授，[[北京理工大学|北京理工大学]]、[[中国民航管理干部学院|中国民航管理干部学院]]、[[南华大学|南华大学]]等院校客座法学教授，[[中国政法大学|中国政法大学]]航空与空间法研究中心研究员，[[中国法学会|中国法学会]]会员，北京法学会航空法研究会创始人、常务副会长兼秘书长，中国仲裁法学研究会常务理事，中国犯罪学研究会常务理事，[[国际保护知识产权协会|国际保护知识产权协会]]中国分会理事，[[国际商会|国际商会]]中国国家委员会律师团律师，[[全国妇联|全国妇联]]法律帮助中心专家，[[北京市人大常委会|北京市人大常委会]]立法咨询专家，中国航空法律服务中心首席专家等职务<ref name=jingcai/>。

==代理案件==
张起淮律师出身空军，所以代理过大量航空业案件。例如“水污染空难索赔案”，“拒载断足少女登机案”，逾千起民航飞行员辞职案，[[俄罗斯|俄罗斯]]艾迪兰航空公司[[图-154|图-154]]飞机案，中华人民共和国首例飞机走私案，[[敦煌|敦煌]]少年坠机案，[[昆明|昆明]]攀机案，中华人民共和国首例“零赔付”、“低赔付”辞职案，[[嘉峪关|嘉峪关]]拒载案、[[厦门航空|厦门航空]]拒载“黑名单”第一案，[[盘锦空难|盘锦空难]]索赔案，[[伊春空难|伊春空难]]索赔案等<ref name=jingcai/>。

此外，张起淮代理过国内外、军内外的大量仲裁、诉讼案件，除民用航空法等领域，还涉及刑法、保险法、城市房地产管理法、婚姻法、合同法等领域。在婚姻家庭案件方面，张起淮代理过某上将离婚案，中华人民共和国首例婚内诈骗案，某核电企业高管离婚案等等，以及[[王宝强|王宝强]]离婚案、[[苏享茂|苏享茂]]自杀案等知名案件<ref name=jingcai/><ref>{{Cite web |url = http://news.163.com/17/0917/17/CUI7RTNP0001875P.html |title =  程序员被骗婚自杀案律师:曾有人婚内诈骗被判10年|author =  |date = 2017-09-17 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。

在刑事案件方面，张起淮代理过[[厦门|厦门]][[远华集团走私案|远华集团走私案]]，[[辽宁|辽宁]][[盘锦|盘锦]]5亿元集资诈骗案，[[哈尔滨|哈尔滨]]特大团伙贩毒案，以及[[李天一|李天一]]强奸案等知名案件<ref name=jingcai/>。

在知识产权案件方面，张起淮代理过[[毛泽东|毛泽东]]著作权侵权案，“珊拉娜”商标侵权案，“[[小肥羊|小肥羊]]”商标纠纷案，《秘密专机上的领袖们》著作权侵权案，[[高敏|高敏]]著作权、名誉权侵权案，[[杨红缨|杨红缨]]名誉权侵权案，[[游江雄|游江雄]]著作权侵权案，[[庄则栋|庄则栋]]著作权、姓名权侵权案，[[京剧|京剧]]《[[沙家浜|沙家浜]]》著作权侵权案，《[[乔家大院|乔家大院]]》等电视剧的著作权侵权案等知名案件<ref name=jingcai/>。[[徐静蕾|徐静蕾]]、[[赵薇|赵薇]] 、[[王宝强|王宝强]]等明星都曾请他担任代理律师<ref>{{Cite web |url = http://www.artsbj.com/show-114-5276-1.html |title =王宝强起诉离婚，为何选择李天一代理律师张起淮？  |author =  |date = 2016-08-16 |publisher = 北京文艺网 |language =  |accessdate =  |quote =  }}</ref>。
