'''張温'''（{{bd||？|191年|catIdx=Z張}}），字'''伯慎'''。[[東漢|東漢]]末期[[南阳郡|南阳郡]][[穰县|穰县]]（今河南省[[邓州市|邓州市]]）人，位至[[三公|三公]]，曾經為[[董卓|董卓]]、[[孫堅|孫堅]]、[[陶謙|陶謙]]等人的上司。

== 生平 ==
=== 京中為官 ===
張温曾受[[曹操|曹操]]祖父[[曹騰|曹騰]]所提拔。[[中平|中平]]元年（184年），張溫當時為[[大司農|大司農]]，受封為[[司空|司空]]。同年張溫舉薦[[討虜校尉|討虜校尉]][[蓋勳|蓋勳]]為[[京兆尹|京兆尹]]。

=== 奉命討賊 ===
中平二年（185年），為[[車騎將軍|車騎將軍]]，與副將[[執金吾|執金吾]][[袁滂|袁滂]]討伐胡人[[北宮伯玉|北宮伯玉]]。當時[[董卓|董卓]]為[[破虜將軍|破虜將軍]]、[[蕩寇將軍|蕩寇將軍]][[周慎|周慎]]、別駕從事[[孫堅|孫堅]]與議郎[[陶謙|陶謙]]皆被攬至張溫帳下。張溫領兵十餘萬駐紮[[美陽|美陽]]，與[[邊章|邊章]]、[[韓遂|韓遂]]等會戰，初期戰勝後派出周慎、董卓等追擊，但因不聽孫堅建言而無法取得有利形勢，後來戰況陷入膠著的局面而撤兵，班師回朝後陶謙亦不滿張溫的無能多次私下批評抱怨。

=== 位至三公 ===
中平三年（186年），於[[長安|長安]]拜張溫為[[太尉|太尉]]，成為第一位不在朝廷的三公，有傳言張溫以行賄方式得到此位。同年張溫受徵召回京師。

=== 事業滑落 ===
中平四年（187年），張溫因未能平亂被罷免。[[初平|初平]]二年（191年），當時張溫為[[衛尉|衛尉]]，並不與董卓結交，為董卓所怨恨，所以董卓誣蔑張溫與[[袁術|袁術]]私通，於市街上被拷打致死。

== 演義記載 ==
登場於第八回「王司徒巧使連環計 董太師大鬧鳳儀亭」，董卓於長安宴請百官時，因[[呂布|呂布]]接獲袁術與張溫私通書信，乃登堂密告董卓；董卓獲報後，立刻命令呂布將張溫從宴席中拖出斬首，並將首級呈上，百官見之俱感顫慄，唯董卓一人飲食談笑自若。

== 部下 ==
* [[董卓|董卓]]：為[[破虜將軍|破虜將軍]]，一同出征邊章等人。
* [[陶謙|陶謙]]：為[[參軍|參軍]]，認為張温才能不及[[皇甫嵩|皇甫嵩]]而輕視張温。
* [[孫堅|孫堅]]：為[[參軍|參軍]]，認為張温應殺怠慢軍機的董卓以立軍威，不為張温採納。
* [[張純_(漢靈帝)|張純]]：張純自薦為將討伐邊章不為張温採納，心生不憤而叛亂。

== 評價 ==
* [[晉|晉]][[傅玄|傅玄]]曰：「溫有傑才。」
* [[東漢|東漢]][[劉陶|劉陶]]曰：「將軍張溫，天性精勇。」
