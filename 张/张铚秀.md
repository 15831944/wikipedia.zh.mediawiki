'''张铚秀'''（{{bd|1915年|7月26日|2009年|8月14日|catIdx=Z张}}），[[汉族|汉族]]，[[江西|江西]][[永新|永新]]人，中国现代[[军事家|军事家]]、[[革命家|革命家]]。[[中国人民解放军|中国人民解放军]]高级将领。1955年被授予[[中国人民解放军少将|中国人民解放军少将]]军衔，并被同时授予二级[[八一勋章_(1955年)|八一勋章]]、二级[[独立自由勋章|独立自由勋章]]和一级[[解放勋章|解放勋章]]。1988年被授予一级[[红星功勋荣誉章|红星功勋荣誉章]]。另外，他还因为在[[抗美援朝战争|抗美援朝战争]]中的贡献被朝鲜授予[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]二级独立自由勋章、二级[[国旗勋章|国旗勋章]]。

== 早年与第一次国共内战时期 ==
张铚秀1915年7月26日生于[[江西省|江西省]][[永新县|永新县]]怀忠乡虹桥村，1929年加入[[少先队|少先队]]，参加虹桥村农民起事，1930年加入[[中国共青团|中国共青团]]，1933年加入[[中国工农红军|中国工农红军]]，历任副班长、班长，当时正值[[第一次国共内战|第一次国共内战]]时期。1934年2月加入[[中国共产党|中国共产党]]。之后再任排长、副连长。参加了[[长征|长征]]。其间升任营长。<ref>[[#军旅生涯（张铚秀回忆录）|军旅生涯（张铚秀回忆录） (1998)]]，第474页</ref>

== 抗日战争时期 ==
[[抗日战争|抗日战争]]时期，张铚秀于1937年进入[[红军大学|红军大学]]和[[抗日军政大学|抗日军政大学]]二期学习。1938年2月，任湘赣游击大队第2中队中队长，3月任[[新四军|新四军]]第1支队2团侦察参谋，4月28日任新四军先遣支队第一中队中队长、支队司令部侦察参谋。其后调任2营营长，后又转任1营营长。1939年任2团团长，参加[[韦岗战斗|韦岗战斗]]。1941年进入[[华东党校|华东党校]]学习。<ref>[[#军旅生涯（张铚秀回忆录）|军旅生涯（张铚秀回忆录） (1998)]]，第474页</ref>

== 第二次国共内战时期 ==
[[第二次国共内战|第二次国共内战]]时期，张铚秀历任[[山东野战军|山东野战军]]第7师19旅副旅长兼参谋长，[[华东野战军|华东野战军]]第七纵队9师副师长兼参谋长，第九纵队26师副师长兼参谋长、师长，[[第三野战军|第三野战军]]九兵团27军80师师长，率部参加[[涟水战役|涟水战役]]、[[莱芜战役|莱芜战役]]、[[孟良崮战役|孟良崮战役]]、[[济南战役|济南战役]]、[[淮海战役|淮海战役]]、[[渡江战役|渡江战役]]、[[上海战役|上海战役]]。1949年4月24日率部攻克[[宣城|宣城]]。<ref>[[#军旅生涯（张铚秀回忆录）|军旅生涯（张铚秀回忆录） (1998)]]，第474页</ref>

== 中华人民共和国成立后 ==
=== 建国初期与朝鲜战争 ===
[[中华人民共和国|中华人民共和国]]成立后，张铚秀于1950年第一批入朝参加[[朝鲜战争|朝鲜战争]]，历任[[中国人民志愿军|中国人民志愿军]]27军80师师长、26军副军长，参加第一到第五次战役、平金淮防御战，获[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]二级独立自由勋章、二级[[国旗勋章|国旗勋章]]。朝鲜战争[[第二次战役(抗美援朝)|第二次战役]]的新田里战斗，张铚秀率部全歼美军第七师三十一团级战斗队3000余人，开创了中国人民志愿军在朝鲜战场上全歼现代化装备美军一个加强团的唯一一例。1952年，张铚秀在将防区交给志愿军第15军后回国，进入[[华东军区军事干部学校|华东军区军事干部学校]]高干队学习。1953年任26军代军长。1955年被授予[[中国人民解放军少将|中国人民解放军少将]]军衔，并被同时授予二级[[八一勋章_(1955年)|八一勋章]]、二级[[独立自由勋章|独立自由勋章]]和一级[[解放勋章|解放勋章]]。1958年入[[高等军事学院|高等军事学院]]学习，于1960年毕业，任68军军长。<ref>[[#军旅生涯（张铚秀回忆录）|军旅生涯（张铚秀回忆录） (1998)]]，第474页</ref>

=== 中越战争与晚年 ===
{{Main|中越战争|1979－1990中越边境冲突}}

张铚秀于1969年被任命为[[济南军区|济南军区]]副司令员，1975年任[[昆明军区|昆明军区]]副司令员，1979年他在时任昆明军区司令员杨得志因病离开后任西线战役总指挥，实际指挥了[[中越战争|中越战争]]西线作战，他制订了作战方案和10项作战指导原则，对保证云南方向解放军作战的胜利起到了重要作用。1980年，张铚秀升任昆明军区司令员，指挥了1980年至1985年的[[1979－1990中越边境冲突|中越边境冲突]]，他先是指挥云南方向部队拔点作战和部队轮战，取得了胜利，1984年，他又亲自指挥了[[两山战役|两山战役]]，取得了胜利，收复了老山、者阴山。他是解放军从事作战指挥年限最长的将领之一。1988年，张铚秀获一级[[红星功勋荣誉章|红星功勋荣誉章]]。2009年8月14日，张铚秀于[[北京|北京]]逝世，享年95岁。<ref>[[#军旅生涯（张铚秀回忆录）|军旅生涯（张铚秀回忆录） (1998)]]，第474页</ref>

张铚秀晚年著有回忆录《[[军旅生涯|军旅生涯]]》（张铚秀回忆录）。
