'''张勇'''（{{bd|1953年|9月||catIdx=Z张}}），[[河北|河北]][[平山|平山]]人，生于[[北京|北京]]，1969年8月参加工作，1974年3月加入中国共产党，发展经济学硕士，高级经济师。曾任[[国务院副秘书长|国务院副秘书长]]、[[国家食品药品监督管理总局|国家食品药品监督管理总局]]局长兼党组书记，现任[[国家发展和改革委员会|国家发展和改革委员会]]副主任、党组成员（正部长级）。中共十七大、十八大、十九大代表，中共十八届中纪委委员。

==生平==
曾在[[黑龙江生产建设兵团|黑龙江生产建设兵团]]、解放军部队、北京真空仪表厂、北京微波通讯设备厂工作。1979年至1983年在[[中国人民大学|中国人民大学]]学习。1983年后，在[[国家计委|国家计委]]固定资产投资司和国家开发银行综合计划局、国际金融局以及国家计委经济政策协调司工作，先后任副处长、处长、副局长、局长、司长职务。2001年1月起，任国务院办公厅秘书二局政务专员兼副局长、局长。2003年，担任[[国务院副秘书长|国务院副秘书长]]。2010年，后兼任[[国务院食品安全委员会办公室|国务院食品安全委员会办公室]]主任<ref>{{cite news|title=国务院副秘书长张勇主持会议|url=http://www.gov.cn/zhibo53/content_507857.htm|accessdate=2012-06-10|newspaper=中国政府网}}</ref>。2013年，任[[国家食品药品监督管理总局|国家食品药品监督管理总局]]局长、党组书记<ref>{{cite news|title=国家食品药品监督管理总局今晨“摸黑”挂牌|url=http://news.qq.com/a/20130322/000107.htm|accessdate=2013-03-22|newspaper=腾讯网}}</ref>。2015年2月，转任[[国家发展和改革委员会|国家发展和改革委员会]]副主任（正部级）<ref>{{cite news|title=张勇任发改委副主任 解振华朱之鑫徐宪平卸任|url=http://wap.ce.cn/yw/201502/26/t20150226_4656422.html|accessdate=2015-02-26|newspaper=中国经济网|deadurl=yes|archiveurl=https://web.archive.org/web/20150402091629/http://wap.ce.cn/yw/201502/26/t20150226_4656422.html|archivedate=2015年4月2日|df=}}</ref>。2015年4月24日，被免去国务院副秘书长职务<ref>{{cite web|title=江泽林任国务院副秘书长 聂辰席兼任CCTV台长|url=http://news.163.com/15/0424/09/ANV622050001124J.html|publisher=网易|accessdate=2014-04-24}}</ref>。2018年任[[第十三届全国政协委员|第十三届全国政协委员]]、常委<ref>{{Cite web |url =  http://www.gov.cn/guoqing/2018-03/14/content_5274165.htm |title = 中国人民政治协商会议第十三届全国委员会 |author =  |date = 2018-03-14 |publisher =中国政府网 |language =  |accessdate =  |quote =  }}</ref>、全国政协农业和农村委员会副主任。
