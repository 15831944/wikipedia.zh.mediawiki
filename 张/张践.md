'''张践'''（{{bd|1963年|||}}），[[湖南省|湖南]][[益阳市|益阳]]人，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
长期在[[广州军区|广州军区]]服役，1997年任[[中国人民解放军驻香港部队|中国人民解放军驻香港部队]]司令部办公室主任。后任驻香港部队步兵旅旅长、驻香港部队副参谋长，广州军区司令部作战部部长。2007年任[[中国人民解放军陆军第四十二集团军|陆军第四十二集团军]]参谋长，后升任副军长。2013年5月任[[中国人民解放军湖北省军区|湖北省军区]]司令员，2014年9月任[[中国人民解放军海南省军区|海南省军区]]司令员<ref>{{Cite news|url=http://china.caixin.com/2014-09-11/100727383.html|title=张践调任海南省军区司令员|author=|date=2014-09-11|work=财新网|accessdate=2017-10-01}}</ref><ref name=san/>。2016年2月任新成立的[[中国人民解放军东部战区陆军|东部战区陆军]]参谋长<ref>{{Cite news|url=http://china.caixin.com/2016-02-05/100907607.html|title=东部战区陆军领导层亮相 多来自南京军区|author=|date=2016-02-05|work=财新网|accessdate=2017-10-01}}</ref><ref name=san/>。2017年3月任[[中央军委联合参谋部作战局|中央军委联合参谋部作战局]]局长<ref>{{Cite web|url=http://www.thepaper.cn/newsDetail_forward_1649932|title=张践少将出任中央军委联合参谋部作战局局长|accessdate=2017-10-01|author=|date=|work=|publisher=澎湃新闻}}</ref>。2017年9月升任[[中国人民解放军南部战区陆军|南部战区陆军]]司令员<ref>{{Cite web|url=http://news.163.com/17/0930/22/CVK60JRC000187VE.html|title=彭清华陈武及南部战区陆军司令员张践等在南宁向烈士敬献花篮|accessdate=2017-10-01|author=|date=2017-09-30|last=|work=|publisher=网易}}</ref><ref name=san/>。

2010年7月晋升[[中国人民解放军少将|少将]]军衔<ref name=san>{{Cite web |url = http://news.ifeng.com/a/20171001/52261822_0.shtml |title = 解放军重要人事调整：三战区级新亮相 |author =  |date = 2017-10-01 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
