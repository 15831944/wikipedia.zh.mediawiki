'''张子谦'''（1899年10月3日-1991年1月5日），名'''益昌'''，字'''子謙'''，祖籍[[江苏|江苏]][[仪征|仪征]]，出生於[[揚州|揚州]]。[[广陵琴派|广陵琴派]]音乐家。善弹《[[龙翔操|龙翔操]]》，故得“'''张龙翔'''”的称号，與[[査阜西|査阜西]]、[[彭祉卿|彭祉卿]]並稱“浦東三傑”。<ref name="貢獻">{{Cite web|url=http://www.cglgq.com/wz/39.htm|title=張子謙先生古琴藝術初探|accessdate=2014-12-31|author=[[成公亮|成公亮]]|date=2008|work=|publisher=}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==生平==

張子謙祖籍儀征，光緒二十五年八月廿九（1899年10月3日）出生於[[揚州|揚州]]東關街的一個富裕家庭之中，然而在七八歲時父母雙亡，他則被伯父等親戚養大。1911年師從[[孫紹陶|孫紹陶]]學習[[古琴|古琴]]。<ref name="初探">{{Cite web |url=http://big5.china.com.cn/chinese/zhuanti/309371.htm |title= 张子谦先生在近现代琴史上的贡献|accessdate=2014-12-31 |author=[[李鳳雲|李鳳雲]] |date=2003-02-05|work= |publisher=[[中國網|中國網]] }}</ref><ref name="中音">{{Cite web|url=http://ccmusic.sres.bjedu.cn/drms/portal/yinyue/index109.113_list_qr.jsp?currPath=%B9%C5%C7%D9/5%A1%A2%C7%D9%C8%CB/02%BB%A6%A1%A2%C4%FE%A1%A2%BA%BC/03%D5%C5%D7%D3%C7%AB|title=张子谦|accessdate=2014-12-31|author=|date=|work=|publisher=[[中國音樂學院|中國音樂學院]]|deadurl=yes|archiveurl=https://web.archive.org/web/20150323054815/http://www.blayney.nsw.gov.au/ArticleDocuments/329/3h%20-%20Hobbys%20Yards%20Heritage%20Items.pdf.aspx|archivedate=2015-03-23}}</ref>

1930年代張子謙前往[[上海|上海]][[浦東|浦東]]的公茂盐栈工作，後來擔任了主任一職18年之久。1934年結識査阜西、彭祉卿，<ref name="初探"/> 也曾向査阜西請教古琴技法，後來他們三人被稱為“浦東三傑”。古琴家[[谢孝苹|谢孝苹]]談到張子謙的曲風時說：<ref name="中音"/>
{{cquote|张子谦演奏风格清新委婉，节奏明快中跌荡时出，指法上不避偏锋，右手有时疾进至七八徽间，几与左手相遇在毫发间，将广陵派的独特风格发挥得淋漓尽致。}}

1936年他和査阜西等人在[[蘇州|蘇州]]成立了中國近代著名的[[今虞琴社|今虞琴社]]，後來今虞琴社的上海分部“滬社”就開設在張子謙於浦东春江路17号的家中。[[抗日戰爭|抗日戰爭]]爆發之後，原先琴人所用的“回回堂琴弦”斷貨，導致琴人陷入無弦可用的尷尬境地，張子謙便和[[吳景略|吳景略]]等人依古法在1943年試製成“今虞琴弦”。此外，在抗戰期間，張子謙著成了記載當時文化界狀況的《操縵瑣記》。<ref name="貢獻"/>

1956年他加入[[上海民族樂團|上海民族樂團]]，常和[[孫裕德|孫裕德]]琴簫合奏，頗為時人稱道。<ref name="廣陵">{{Cite web |url=http://info.guqu.net/guqindashi/20060730113459_570.html |title=“广陵派”古琴演奏家张子谦|accessdate=2014-12-31 |author= |date=2006-07-30|work= |publisher=中國古曲網 }}</ref> 1950年代末到[[上海音樂學院|上海音樂學院]]授課。<ref name="大師">{{Cite web |url=http://www.yznews.com.cn/yzwb/html/2011-08/20/content_302250.htm |title=广陵派古琴大师张子谦|accessdate=2014-12-31 |author= |date=2011-08-20|work= |publisher=[[揚州晚報|揚州晚報]] }}</ref><ref name="廣陵"/><ref name="中音"/> 他雖然是[[廣陵琴派|廣陵琴派]]的代表人物，張子謙本身並不讚成古琴分派的說法，反而認為“海内为一家、南北无二派”，而之所以古琴當時仍有眾多派別是因為古代交通不便之故。<ref name="中音"/>

在[[文化大革命|文化大革命]]期間許多古琴演奏家都遭到排擠打壓，而張子謙只得離開上海音樂學院，今虞琴社也不得不廢社。1980年復社時，張子謙出任社長，之後再度受聘於上海音樂學院，[[龔一|龔一]]、[[成公亮|成公亮]]、[[李禹贤|李禹贤]]和[[戴晓莲|戴晓莲]]等都曾是他的學生。<ref name="中音"/>

1988年將近90歲高齡的張子謙因身體欠安，而移居長子张家镇在[[天津|天津]]的家中。到達天津的第二天接受[[天津音樂學院|天津音樂學院]]聘請，成為該校的客座教授，在該校任職期間收下最後一個弟子[[李鳳雲|李鳳雲]]。1991年在天津的家中逝世。<ref name="中音"/>
