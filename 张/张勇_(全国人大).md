==生平==
张勇是[[中国共产党|中国共产党]]党员<ref name=elysan/>。早年曾在[[辽宁省|辽宁省]][[凤城市|凤城市]]第一中学学习。后成为法律专家，曾任[[国务院港澳事务办公室|国务院港澳事务办公室]]综合司司长<ref>{{Cite web|url = http://www.chinaxq.com/html/2007-8/14/content_27512.shtml|title = 辽宁省凤城市第一中学130周年校庆公告|author = |date = 2007-08-14|publisher = 中国校庆网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180314042341/http://www.chinaxq.com/html/2007-8/14/content_27512.shtml|archive-date = 2018-03-14|dead-url = yes}}</ref>。2016年2月，[[第十二届全国人大常委会|第十二届全国人大常委会]]第十九次会议任命张勇为全国人大常委会法制工作委员会副主任<ref>{{Cite web |url = http://china.caixin.com/2016-02-26/100913281.html |title = 刘源履新全国人大财经委副主任委员 向宪法宣誓 |author =  |date = 2016-02-26 |publisher = 财新网|language =  |accessdate =  |quote =  }}</ref>。2018年当选[[第十三届全国人大|第十三届全国人大]]江苏代表<ref name=elysan/>。2018年3月，任[[全国人大宪法和法律委员会|全国人大宪法和法律委员会]]委员<ref name=xianfa>{{Cite web |url = http://www.npc.gov.cn/npc/xinwen/2018-03/13/content_2048058.htm |title = 第十三届全国人民代表大会宪法和法律委员会主任委员、副主任委员、委员名单 |author =  |date = 2018-03-13 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>。
