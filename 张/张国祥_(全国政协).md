'''张国祥'''（{{bd|1940年|8月||}}）[[河北省|河北省]][[固安县|固安县]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中华全国总工会|中华全国总工会]]副主席，第八、第九届[[全国人大常委会|全国人大常委会]]委员，第十届全国政协常委。

==生平==
张国祥是大学文化。1964年9月参加工作，1966年3月加入[[中国共产党|中国共产党]]。历任北京工商管理专科学校教师，北京第三通用机械厂干部，《北京工人报》编辑，北京第三通用机械厂宣传科副科长，北京市总工会办公室干部、副主任、主任，政研室主任，北京市总工会常委、副秘书长，北京市总工会副主席兼秘书长、党组成员，北京市[[崇文区|崇文区]]区委副书记、区长<ref name=diyi>{{Cite web |url = http://news.china.com/zh_cn/focus/zhibo/2h01.html   |title =全国政协十届二次会议第一次新闻发布会 |author =  |date =2004-03-02 |publisher = 中华网 |language =  |accessdate =  |quote =  }}</ref>。

1989年12月，任[[中华全国总工会|全国总工会]]主席团委员、书记处书记兼政策研究室主任。1993年10月，任全国总工会副主席、书记处书记、党组成员。1999年3月到2003年3月，任[[全国政协|全国政协]]副秘书长、[[全国政协机关党组|机关党组]]成员<ref name=diyi/>。2003年和2004年担任全国政协大会新闻发言人<ref>{{Cite web |url =http://cppcc.people.com.cn/n/2015/0303/c34948-26623786.html  |title = 张国祥：新闻发布会越来越好看 |author =  |date =2015-03-03  |publisher =中国政协新闻网  |language =  |accessdate =  |quote =  }}</ref>。他是[[第八届全国人大常委会|第八届全国人大常委会]]委员、[[全国人大环境与资源保护委员会|全国人大环境与资源保护委员会]]委员，[[第九届全国人大常委会|第九届全国人大常委会]]委员、[[全国人大法律委员会|全国人大法律委员会]]委员，第十届全国政协常委、[[全国政协外事委员会|全国政协外事委员会]]副主任<ref name=diyi/>。
