'''张绍基'''（{{bd|1935年|2月11日|2012年|10月15日|catIdx=Zhang张绍基}}），[[中华人民共和国|中国]]著名的[[寄生虫病|寄生虫病]]防治研究专家、原[[江西省|江西省]]寄生虫病防治研究所所长。1982年加入[[中国共产党|中国共产党]]，是中国共产党第十三次代表大会代表、十四次代表大会代表，第五届、第六届中国[[政治协商会议|政治协商会议]]江西省委员。1991年起享受[[国务院|国务院]]政府特殊津贴。历任[[中华人民共和国卫生部|中华人民共和国卫生部]]血吸虫病专家咨询委员会委员、[[中华预防医学会|中华预防医学会]]寄生虫病专业委员会常委、江西省预防医学会寄生虫病专业委员会主任委员、江西省血吸虫病研究委员会主任委员。<br />

== 生平 ==
1935年2月出生于[[江苏省|江苏省]][[太仓县|太仓县]]，1957年毕业于[[上海第一医学院|上海第一医学院]]，同年9月至1958年在江西省卫生防疫站工作，1958年9月在江西省卫生厅除害灭病办公室工作，1959年11月在江西省寄生虫病研究所工作，1968年11月在[[铜鼓县|铜鼓县]]三溪大队行医，1969年9月借调铜鼓县卫生局，1973年在江西省医学科学研究所工作，1981年2月在江西省寄生虫病研究所工作，1984年起担任所长，1999年9月退休。2012年10月15日于江西南昌逝世。<br />

== 主要学术观点 ==
张绍基是第一位在中国国内提出了以[[钉螺|感染螺密度]]为主要指标的[[日本血吸虫|血吸虫病]]易感地带分类标准的学者，为中国血吸虫病疫区有螺环境灭螺提供了依据。他提出并起草的《鄱阳湖大区域控制血吸虫病的总体规划》，加速促进了鄱阳湖区血吸虫病防治的进程。

== 国际交流和影响 ==
张绍基于1989年通过[[南京医科大学|南京医科大学]]，首次与[[美国|美国]][[布朗大学|布朗大学]]国际卫生研究所和南京医科大学联合签订了一项WHO／TDR和美国Clark基金资助的“重型日本[[血吸虫病|血吸虫病]]流行区人群感染率、感染度与疾病相关关系及化疗对日本血吸虫病的流行性和临床发病影响的研究” 的三年现场研究计划。他与他领导的江西省寄生虫病防治研究所先后与[[菲律宾|菲律宾]][[马尼拉|马尼拉]]热带病研究所、[[日本|日本]]ASSCA(中国和东南亚血吸虫病消灭援助团)、日本[[琦玉医科大学|琦玉医科大学]]、日本[[东京大学|东京大学]]医学部、日本[[顺天堂大学|顺天堂大学]]医学部、日本[[产业医科大学|产业医科大学]]、[[日本长崎大学|日本长崎大学]]、美国Symter制药公司、美国[[费城自然科学院|费城自然科学院]]、[[澳大利亚|澳大利亚]][[昆士兰热带医学研究中心|昆士兰热带医学研究中心]]、[[瑞士|瑞士]][[热带病|热带病]]研究所、[[韩国|韩国]]国际协力团、韩国健康管理协会和韩国[[国立汉城大学|国立汉城大学]]医学院等单位建立了良好科研合作关系。
