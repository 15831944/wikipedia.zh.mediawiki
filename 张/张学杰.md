'''张学杰'''（{{bd|1962年|||}}），[[安徽省|安徽省]][[寿县|寿县]]人，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
张学杰曾任[[中国人民解放军陆军第七十三集团军|陆军第三十一集团军]]摩步某师政委，陆军第三十一集团军政治部副主任。2009年任陆军第三十一集团军政治部主任。2013年任[[中国人民解放军陆军第七十一集团军|陆军第十二集团军]]政治委员。2014年任陆军第三十一集团军政治委员<ref>{{Cite web |url = http://china.caixin.com/2014-10-01/100735146.html |title = 周皖柱升任第12集团军政委 |author =  |date = 2014-10-01 |publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://mil.sohu.com/20141024/n405425914.shtml |title = 南京军区3集团军换将频繁 张学杰上任31军政委 |author =  |date = 2014-10-24 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。2017年，任[[中国人民解放军东部战区|中国人民解放军东部战区]]政治工作部副主任<ref name=zhxj>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1740096 |title = 原31集团军政委张学杰已改任驻南京部队领导 |author =  |date =2017-07-23  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{cite web |title=我市召开2018年军政座谈会 |url=http://wemedia.ifeng.com/48974667/wemedia.shtml |website=南京新闻 |accessdate=2018-09-03 |date=2018-02-12}}</ref>。2018年5月，升任[[西藏军区|西藏军区]]政治委员<ref>{{cite web |title=西藏军区张政委到校视察指导工作 |url=http://zswd.cdqz.net/Cdqzby/Front/Detail.jsp?articleid=5988&menuid=52 |website=成都七中八一学校 |accessdate=2018-09-03 |date=2018-05-29}}</ref><ref>{{cite web |title=张学杰少将升任西藏军区政委，曾任原第31集团军政委 |url=http://www.thepaper.cn/newsDetail_forward_2404652 |website=澎湃新闻 |date=2018-09-03|accessdate=2018-09-03}}</ref>。

2010年7月晋升少将军衔<ref name=zhxj/>。
