'''张守攻'''（{{bd|1957年|7月||}}），[[安徽|安徽]][[怀远|怀远]]人，[[中华人民共和国|中华人民共和国]]林学家，[[中国工程院院士|中国工程院院士]]，[[中国林业科学研究院|中国林业科学研究院]]院长<ref name=yuan/><ref name=db>{{Cite web |url = http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=132356 |title = 代表信息：张守攻 |author =  |date =  |publisher =中国人大网  |language =  |accessdate = 2018-03-19 |quote =  }}</ref>。

==生平==
张守攻是[[中国共产党|中国共产党]]党员。博士研究生指导教师，研究员，享受[[中华人民共和国国务院|国务院]][[政府特殊津贴|政府特殊津贴]]。1982年，自[[安徽农学院|安徽农学院]]毕业并留校任教。1986年到1990年，在[[北京林业大学|北京林业大学]]森林经理专业学习，获农学博士学位。1990年被分配到中国林业科学研究院林业研究所工作，历任林业研究所经营室副主任、主任。1994年，任林业研究所副所长。1995年任林业研究所所长。1997年任中国林业科学研究院副院长。2002年任中国林业科学研究院常务副院长。2007年任中国林业科学研究院院长。他还兼任[[中国林学会|中国林学会]]副理事长，[[福建省人民政府|福建省人民政府]]科技顾问，当选[[中共十七大|中共十七大]]代表、中央国家机关党代表等<ref name=yuan>{{Cite web |url = http://www.caf.ac.cn/news/yqjj/0926122951CI45581ICHHHAKHB248A.htm |title = 张守攻 |author =  |date = 2016-01-01 |publisher = 中国林业科学研究院 |language =  |accessdate =  |quote =  }}</ref>。2017年11月27日，[[中国工程院|中国工程院]]公布2017年院士增选结果，共选举产生67位新当选院士，中国林业科学研究院张守攻当选中国工程院院士<ref>{{Cite web|url = http://www.greentimes.com/green/news/renwu/xwrw/content/2017-11/28/content_368201.htm|title = 张守攻蒋剑春新当选中国工程院院士|author = |date = 2017-11-28|publisher = 中国林业新闻网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180319214205/http://www.greentimes.com/green/news/renwu/xwrw/content/2017-11/28/content_368201.htm|archive-date = 2018-03-19|dead-url = yes}}</ref>。2018年当选[[第十三届全国人大代表|第十三届全国人大代表]]、[[全国人大环境与资源保护委员会|全国人大环境与资源保护委员会]]副主任委员<ref name=db/>。

==学术贡献==
张守攻是中国人工林培育集成技术体系和森林可持续经营研究领域主要学术带头人之一，长期研究森林培育和森林经理，开展[[落叶松|落叶松]]品质改良、繁殖工程、林分经营技术及应用基础研究。第一次在森林经营模型中实现了林分生长模型系统与产品预估模型重构；完成了落叶松生态育种区的划分，建立了长期育种和良种生产体系；构建了落叶松培育集成技术；建成了落叶松遗传改良、良种繁育、定向培育工程体系<ref name=yuan/>。

截至2017年，张守攻获国家科技进步二等奖3项，省部级科技进步一等奖2项、二等奖1项；国际发明专利1项、国家发明专利13项；发表学术论文263篇（其中SCI收录42篇），出版专著7部<ref name=yuan/>。
