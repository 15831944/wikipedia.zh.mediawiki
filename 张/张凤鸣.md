'''张凤鸣'''（{{bd|1963年|7月||}}）今[[重庆市|重庆市]][[梁平区|梁平区]]人<ref>{{Cite web|url =http://www.lpxedu.gov.cn/contentview.asp?nodecode=msfc&id=6236|title =梁平学子，归去来兮|author =蓝振中|date =2013-09-12|publisher =重庆市梁平区教育委员会|language =|accessdate =|quote =}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，[[中国人民解放军空军少将|中国人民解放军空军少将]]，[[中华人民共和国|中华人民共和国]]装备系统工程学家<ref name=elysi/>。

==生平==
张凤鸣生于1963年7月<ref name=elysi>{{Cite web |url = http://www.csaa.org.cn/a/zhongguohangkongxuehuidijiuciquanguohuiyuandaibi/2014/1029/1379.html |title =第九届理事会理事候选人基本信息表  |author =  |date =2014-10-03  |publisher = 中国航空学会 |language =  |accessdate =  |quote =  }}</ref>。曾任[[中国人民解放军空军工程大学|中国人民解放军空军工程大学]]工程学院院长。2010年出任中国人民解放军空军工程大学训练部部长<ref name=liangxiang>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1746377 |title = 空军工程大学军政主官亮相：张凤鸣、朱瑞分任校长、政委 |author =  |date = 2017-07-29 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。同时兼任[[中国航空学会|中国航空学会]]第九届理事会理事<ref name=elysi/>。2017年7月，出任新调整组建的中国人民解放军空军工程大学校长<ref name=liangxiang/>。
