'''张铁生'''（{{bd|1904年||1979年|11月19日}}）又名'''张铭鼎'''，[[江苏|江苏]][[高邮|高邮]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员<ref name=zts>{{Cite web |url =http://news.163.com/08/0309/18/46K61LDV0001126S.html  |title = 张铁生 |author =  |date = 2008-03-09 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1927年加入[[中国共产党|中国共产党]]。1929年到[[德国|德国]][[柏林|柏林]]学习。1936年重新加入中国共产党。在[[中华民国|中华民国]]时期，长期从事中国共产党领导下的文化界、新闻界的革命工作。先后主编或编辑过《国民公论》、《青年知识》周刊（1941年8月6日在[[香港|香港]]创刊，张铁生主编）等多种刊物。[[中华人民共和国|中华人民共和国]]成立后，历任[[中华人民共和国对外文化联络委员会|中华人民共和国对外文化联络委员会]]副秘书长、亚非研究所副所长、亚非学会秘书长、中非友协副会长，西亚非研究所副所长，[[中共中央对外联络部|中共中央对外联络部]]三局顾问，[[第五届全国政协委员|第五届全国政协委员]]等职务<ref name=zts/><ref>{{Cite web |url =http://www.zgqxb.com.cn/pressman/cmlw/201211/t20121130_33872.htm  |title =《中国需要什么样的记者》与范长江的记者素质观  |author =  |date = 2012-11-30|publisher = 新气象 |language =  |accessdate =  |quote =  }}</ref>。

1979年11月19日，张铁生在北京病逝<ref name=zts/>。
