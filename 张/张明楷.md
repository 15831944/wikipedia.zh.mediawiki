'''张明楷'''（{{bd|1959年|7月||}}），[[湖北|湖北]][[仙桃|仙桃]]人，[[中华人民共和国|中华人民共和国]][[刑法学|刑法学]]家，[[中国刑法学研究会|中国刑法学研究会]]副会长。

== 生平 ==
1985年取得[[湖北财经学院|湖北财经学院]]（今[[中南财经政法大学|中南财经政法大学]]）[[法学硕士|法学硕士]]，毕业后留校任教。1989年在[[东京都立大学|东京都立大学]]法学部研修；1995年任[[东京大学|东京大学]]法学部客座研究员，1996年任东京都立大学法学部客座研究教授。

1985年到1998年任[[中南政法学院|中南政法学院]]法律系助教、讲师、副教授和教授<ref name="张明楷">{{cite web |url=http://afxfy.chinacourt.org/public/detail.php?id=6910 |language= |title=张明楷 |publisher=安福县人民法院网 |date=2009-09-22 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20160304102048/http://afxfy.chinacourt.org/public/detail.php?id=6910 |archivedate=2016-03-04 }}</ref>。1998年2月到[[清华大学|清华大学]]任教授，担任[[博士生导师|博士生导师]]至今<ref name="介绍">{{cite web|url=http://www.douban.com/group/105671/ | language= | title=张明楷 |publisher=豆瓣网 |date=2008-05-07 |accessdate= }}</ref>。

== 著作 ==
主要著作有：《'''犯罪论原理'''》（[[武汉大学出版社|武汉大学出版社]]，1991年版）、《刑事责任论》（中国政法大学出版社，1992年版）、《刑法的基础观念》（中国检察出版社，1995年版）、《市场经济下的经济犯罪与对策》（中国检察出版社，1995年版）、《未遂犯论》（[[法律出版社|法律出版社]]，1997年版）、《'''刑法学'''》（法律出版社，1997年版，司法部第三届部级优秀教材一等奖<ref name= "张明楷"/>）、《外国刑法纲要》（[[清华大学出版社|清华大学出版社]]，1999年版）、《刑法格言的展开》（法律出版社，1999年版）、《法益初论》（中国政法大学出版社，2000年版）、《刑法的基本立场》（中国法制出版社，2002年版）。<ref name= "介绍"/>

曾在《[[中国社会科学|中国社会科学]]》、《[[中国法学|中国法学]]》、《[[法学研究|法学研究]]》等重点刊物上发表120余篇[[论文|论文]]。<ref name= "介绍"/>
