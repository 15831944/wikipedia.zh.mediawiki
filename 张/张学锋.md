'''张学锋'''（{{bd|1961年|4月||}}）[[辽宁|辽宁]][[沈阳|沈阳]]人，[[中国人民解放军少将|中国人民解放军少将]]<ref name=zhxf/>。

==生平==
毕业于[[中国人民解放军国防大学|中国人民解放军国防大学]]联合战役指挥专业，研究生学历。是[[中国共产党|中国共产党]]党员。担任[[沈阳军区|沈阳军区]][[中国人民解放军陆军第十六集团军|陆军第十六集团军]]装甲四师师长。2010年任沈阳军区[[中国人民解放军陆军第四十集团军|陆军第四十集团军]]参谋长。2013年1月，接替[[彭勃|彭勃]]任沈阳军区第四十集团军军长。2008年起担任[[第十一届全国人民代表大会|第十一届全国人民代表大会]]解放军代表<ref>{{cite news|title=全国人大代表信息-张学锋|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=112863|accessdate=2013-04-05|newspaper=全国人大网}}</ref><ref>{{cite web|author=新华网|title=第十一届全国人大代表|url=http://news.xinhuanet.com/ziliao/2008-03/03/content_7708230.htm?node=36628|publisher=新华网|accessdate=2014-01-07}}</ref><ref name=zhxf/>。2017年，任[[中国人民解放军东部战区陆军|中国人民解放军东部战区陆军]]副司令员<ref name=zhxf>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1758734 |title = 原第40集团军军长张学锋已出任东部战区陆军副司令员 |author =  |date = 2017-08-10  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>

2010年12月晋升少将军衔<ref name=zhxf/>。
