'''张超'''（{{bd|1986年|8月|2016年|4月27日}}）[[湖南省|湖南省]][[岳阳市|岳阳市]]人，[[中国人民解放军海军|中国人民解放军海军]]少校、一级飞行员、[[烈士_(中华人民共和国)|烈士]]<ref name=yazi>{{Cite web |url = http://hn.rednet.cn/c/2017/02/09/4208503.htm |title =  岳阳伢子张超烈士当选“感动中国2016年度人物”|author =  |date =  2017-02-09|publisher = 红网 |language =  |accessdate =  |quote =  }}</ref><ref name=ss>{{Cite web|url =http://www.hnxw.cn/news/0801/shsh/2016/278568.html|title =80后湖南岳阳籍舰载机飞行员张超生死4.4秒间|author =|date =2016-08-01|publisher =湖南新闻网|language =|accessdate =|quote =}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。牺牲后被追授一级[[英雄模范勋章|英雄模范勋章]]。

==生平==
2004年9月，张超自岳阳市七中毕业后参加[[中国人民解放军|中国人民解放军]]。2009年5月加入[[中国共产党|中国共产党]]。大学本科学历。2009年，张超谢绝了当飞行教员的机会，加入驻[[中国南海|南海]]一线的某飞行团暨烈士[[王伟_(飞行员)|王伟]]生前所在部队服役。2015年3月，张超转为舰载机飞行员。后担任[[中国人民解放军海军舰载航空兵部队|中国人民解放军海军舰载航空兵部队]]战斗机团正营职中队长，海军少校军衔。2016年4月27日，张超在驾驶[[歼-15战斗机|歼-15舰载战斗机]]开展陆基模拟着舰接地时，发生电传故障，张超推杆无效，被迫跳伞，坠地身受重伤，经抢救无效牺牲。他是海军舰载航空兵部队成立后第一位牺牲的飞行员<ref>{{Cite web |url =http://mil.huanqiu.com/photo_china/2016-08/2839915.html#p=4  |title =  歼15飞行员张超弹射瞬间模拟图|author =  |date = 2016-08-28 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref><ref name=yazi/><ref name=ss/>。2016年11月，[[中央军委|中央军委]]主席[[习近平|习近平]]签署命令，追授海军舰载航空兵部队一级飞行员张超“逐梦海天的强军先锋”称号<ref>{{Cite web |url = http://news.xinhuanet.com/politics/2016-11/14/c_1119909045.htm |title = 习近平签署命令 追授张超“逐梦海天的强军先锋”荣誉称号 |author =  |date =  2016-11-14|publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。2016年11月30日，[[中央军委|中央军委]]追授张超同志“逐梦海天的强军先锋”荣誉称号命名大会在海军舰载航空兵部队举行，中央军委委员、海军司令员[[吴胜利|吴胜利]]向张超的妻子张亚颁发了一级[[英雄模范勋章|英雄模范勋章]]及证书，[[中共辽宁省委|中共辽宁省委]]书记、[[辽宁省人大常委会|辽宁省人大常委会]]主任[[李希|李希]]出席大会<ref>{{Cite web |url =http://news.xinhuanet.com/politics/2016-08/02/c_129198906.htm  |title = 真英雄张超：冒再大的险也要飞歼-15 |author =  |date = 2016-08-02 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://liaoning.nen.com.cn/system/2016/12/01/019497844.shtml |title =中央军委追授张超同志“逐梦海天的强军先锋”荣誉称号命名大会举行  |author =  |date = 2016-12-01 |publisher =  东北新闻网|language =  |accessdate =  |quote =  }}</ref>。2017年2月，张超当选“[[感动中国年度人物评选|感动中国2016年度人物]]”<ref name=yazi/>。
