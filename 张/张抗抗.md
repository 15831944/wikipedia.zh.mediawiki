'''张抗抗'''（{{bd|1950年|7月3日|||Z張}}），中华人民共和国当代女作家。其早期作品多描写知青生活，后来也写作了一批以都市女性为主角的小说。其作品远离了民族、国家和革命的话语，以女性主义的视角审视这个世界。

== 简历 ==
原名'''張抗美'''，1950年出生于[[杭州|杭州]]，祖籍[[广东|广东]][[新会|新会]][[杜阮镇|杜阮镇]][[长乔乡|长乔乡]]。1963年至1966年，张抗抗就读于[[杭州高级中学|杭州一中]]初中。1969年前往[[黑龍江省|黑龙江]]鹤立河农场劳动。1972年在《[[解放日报|解放日报]]》发表了第一篇短篇小说《灯》。1977年-1979年，在[[黑龙江省艺术学校|黑龙江省艺术学校]]学习[[编剧|编剧]]。1979年进入[[黑龙江省作家协会|黑龙江省作家协会]]，后出任任黑龙江省作家协会副主席，黑龙江省第六、七、八届政协委员，第十届、十一届、十二届全国政协委员。张曾赴[[美国|美国]]、[[加拿大|加拿大]]、[[德国|德国]]、[[法国|法国]]、[[俄罗斯|俄罗斯]]、[[南斯拉夫|南斯拉夫]]、[[马来西亚|马来西亚]]进行交流。2009年被聘为国务院参事。<ref>{{cite web |url = http://www.fox2008.cn/Article/2011/20110324000329_51222.html |title=《中外作家辞典》张抗抗生平与创作简介-中学生读书网 www.fox2008.cn |format= |work= |accessdate=}}</ref>有媒体报道称，张抗抗的丈夫就是小说《[[狼图腾|狼图腾]]》的化名作者姜戎，即[[吕嘉民|吕嘉民]]。<ref>[http://www.dw-world.de/dw/article/0,2144,2307458,00.html]</ref>

== 作品 ==
张抗抗以发表于《[[收穫_(雜誌)|收获]]》1979年2期的短篇《爱的权利》成名。小说通过对文革中一个家庭悲剧的描写，以恢复人的尊严为主题。小说中，舒贝、舒莫姐弟的母亲在文革中不堪迫害自杀。不久，舒贝做护士时遇到李欣产生感情。但李欣激进开放的政治态度让舒贝离去。最后舒贝在弟弟劝说下解开心结，获得了爱的权利。<ref>黄红妹.维护人的尊严，恢复人的价值——小说《爱的权利》浅析.曹成杰主编.校友文集. 海南出版社,2005.页385</ref>她其他的短篇小說還有获得1980年全国优秀短篇小说奖的《夏》、《白罌粟》等。

1975年的《[[分界線|分界線]]》是其長篇成名作。1986年的知青文学作品《隐形伴侣》，时代设定在红卫兵运动和上山下乡运动时期，通过心理描写反思人自身的深层问题。书中写实同回忆、梦境、幻觉相交叠，心态同情节融为一体。1994年寫完的《赤彤丹朱》书中主要塑造了“母亲”朱小玲和“父亲”张恺之两个人物，灵感来自于作者自己父母的经历，有些地方引用了历史性文本。该书解构了红色经典，批判阶级斗争学说。<ref>童庆炳，陶东风主编.文学经典的建构、解构和重构.北京大学出版社,2007.页242.</ref>1995年的《情爱画廊》，写以为躶体画画家周由曾有许多裸体模特做情人，可她们最后纷纷离去。之后在苏州遇到阿霓相遇，与她和她的母亲水虹均发生情爱纠葛。这部作品受到了很大争议。<ref>白烨,热读与时评——90年代以来的长篇小说.中国社会科学出版社,2005.页150.</ref>2002年的《作女》探讨了都市女性的欲望。<ref>[http://www.qikan.com.cn/Article/kjsj/kjsj201331/kjsj201331203.html 张抗抗作品《作女》中卓尔人物形象解析]</ref>

张的其他作品还有中篇小说《淡淡的晨雾》（获第一届全国优秀中篇小说奖）、《北极光》、《在丘陵和湖畔有一个人》；中篇童话《翔儿和他的氢球》、散文集《橄榄》等等。
