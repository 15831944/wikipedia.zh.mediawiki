'''张隽轩'''（{{bd|1903年|7月|1988年|9月19日}}），原名'''廷选'''，[[山西省|山西省]][[五台县|五台县]]南沟村人。[[中华民国大陆时期|中华民国大陆时期]]及[[中华人民共和国|中华人民共和国]]政治人物。

== 生平 ==
=== 早年革命 ===
张隽轩早年考入[[山西省立第一中学|山西省立第一中学]]<ref name="sxa1989">{{cite book |editor=《山西年鉴》编辑部 |title = 山西年鉴 1989 |date=1990年7月 |location = 太原  |publisher=山西人民出版社 |ISBN = 7-203-01727-3 |page = 479 }}</ref>。1924年，考入[[北京大学|北京大学]]经济系。受[[李大钊|李大钊]]影响，积极参加学生反帝反封建的爱国运动。1926年，积极参加[[三一八运动|三一八运动]]。1927年加入[[中国共产党|中国共产党]]。他在[[胡熙庵|胡熙庵]]的直接领导下，与[[王伯唐|王伯唐]]等山西籍学生组成新晋学社，并亲自创办和主编革命刊物《[[风棱|风棱]]》杂志，同时，他还参与创办北大平民夜校。1931年，张隽轩回到山西，在阎锡山的清乡督办办公室给[[杨爱源|杨爱源]]当私人秘书，通过职业掩护，从事中共党的地下工作<ref>{{Cite book |title = 山西政协工作亲历记 |authors = 胡富国 等 |editor = 山西省政协文史资料委员会 |location=北京 |publisher= 中国文史出版社 |date = 1999年9月 |ISBN = 7-5034-1032-9 |page = 95 }}</ref>。

30年代中期，因[[阎锡山|阎锡山]]急于重振旗鼓。张隽轩向其推荐[[張友漁_(法學家)|张友渔]]、[[邢西萍|邢西萍]]、[[温健公|温健公]]、[[韩幽桐|韩幽桐]]、[[張友漁_(法學家)|张晓梅]]、[[杨绍萱|杨绍萱]]、[[黄松龄|黄松龄]]、[[邓初民|邓初民]]、[[许德珩|许德珩]]等共产党人和进步人士来山西讲学，宣传民主、进步和统一战线；他还介绍张友渔、胡熙庵、[[赖若愚|赖若愚]]、[[冀云程|冀云程]]、[[侯俊岩|侯俊岩]]、邢西萍等人加入阎锡山官办的[[山西青年救国团|山西青年救国团]]的核心组织，并让中共地下党员戎子和担任特别小组组长<ref>{{Cite book |title = 晋冀豫抗战纪实 |authors = 刘干才、李奎 编著 |location=北京 |publisher=团结出版社 |year=2015年7月 |ISBN = 978-7-5126-3545-6 |page = 8 }}</ref>。1934年，通过[[杜任之|杜任之]]、张友渔等人与苏联塔斯社驻华记者[[刘尊棋|刘尊棋]]取得联系，促成刘尊棋与杨爱源会谈。山西成立[[牺牲救国同盟会|牺牲救国同盟会]]，张隽轩被选为执行委员<ref>{{Cite book |title = 山西通志 |volume = 第48卷 人物志 |editor = 山西省史志研究院  |location=北京 |publisher= 中华书局 |date=2001年5月 |ISBN =7-101-02011-9 |pages = 636-638 }}</ref>。

=== 抗日战争 ===
抗战期间，张隽轩先后担任[[离石县|离石县]]县长、山西省第四行政督察专员兼保安司令等职。他利用自己的公开身份和地位，为八路军[[120师|120师]]、[[山西新军|山西新军]]筹粮筹款，动员新兵，发展和壮大了革命力量<ref>{{Cite book |title = 父辈的抗战 |editors = 张铁柱、曹智、陶德言 主编 |location = 武汉 |publisher=长江文艺出版社 |date = 2015年9月 |ISBN = 978-7-5354-8363-8 |page = 189 }}</ref>。他将[[刘墉如|刘墉如]]、[[常芝青|常芝青]]等一大批中共党员安排在政府和保安司令部工作。他积极支持战地总动员委员会的工作，为实行[[减租减息|减租减息]]、合理负担、[[民主政治|民主政治]]<ref>{{Cite book |title = 山西政协年鉴 1992 |editor = 山西省人民政协研究所 |location=太原 |publisher = 山西人民出版社  |date = 1992年11月 |ISBN = 7-203-02393-4 |page = 586 }}</ref>。

1939年[[晋西事变|晋西事变]]爆发前夕，他从[[赵承绶|赵承绶]]处得知阎锡山的反共部署后，向中共晋西北区党委作汇报，使中共党组织能及时作出相应的对策。事变爆发后，他冒着生命危险以[[山西省_(中华民国)|山西省]]第四[[行政督察区|行政督察区]]行政督察专员的名义，给中共地下党员以专署视察员、巡视员的身份，开上护照和路条通过了阎军的封锁线，使大批中共地下党员和抗日干部脱离危险。当赵承绶命令张隽轩和第四专署的人员、物资撤离离石[[白文镇|白文镇]]，到临县集中时，他召开专署紧急会议研究决定，四专署由[[常芝青|常芝青]]、[[武汝扬|武汝扬]]等共产党员负责向兴县转移。他本人只身带着一名警卫人员去见赵承绶，被软禁<ref name="qw" />；后在南迁动乱中脱险回到临县。抗日战争结束后，张隽轩任[[晋中行署|晋中行署]]秘书长，他积极协调各方力量，动员民工和物资支援[[太原战役|太原战役]]前线<ref>{{Cite book |title = 山西社会主义学院史稿 1956-1996 |editor = 山西社院院史编纂委员会 |publisher=山西社院院刊编辑部 |page = 48 }}</ref>。

=== 共和国时期 ===
中华人民共和国成立后，1949年2月，晋中行署与太原市政府合并，成立新的太原市政府，张隽轩先后任太原市民政局长<ref>{{Cite book |title=山西省档案馆指南 |editor = 山西省档案馆 |location = 北京 |publisher=中国档案出版社 |date = 1996年8月 |ISBN
 = 7-80019-632-1 |location= |page = 298 }}</ref>。之后升任[[山西省人民政府|山西省人民政府]]委员兼民政厅长、省政府政法委员会副主任、[[中国人民保卫世界和平委员会|中国人民保卫世界和平委员会]]山西省委员会副主席。1955年2月，山西省政治协商委员会成立后，他被先后选为第一、二、三、四届政协副主席兼前三届秘书长，政协二、三届党组副书记，[[民革|民革]]第三、四、五届中央委员。1988年9月19日，病逝<ref name="qw">{{Cite book |title=晋绥风云人物 党政人物卷 |editors = 王波、李迎选 |location = 北京 |publisher = 中央文献出版社 |year=2007年12月 |ISBN = 978-7-5073-2448-8 |pages = 325-326 }}</ref>。
