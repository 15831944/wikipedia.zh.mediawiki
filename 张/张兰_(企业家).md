'''张兰'''{{BD|1958年|4月7日|||catIdx=Z张}}，[[圣基茨和尼维斯|圣基茨和尼维斯]]籍华人企业家。出生于[[中国|中国]][[湖北省|湖北省]][[孝感市|孝感市]]，毕业于[[北京工商大学|北京工商大学]]、[[长江商学院|长江商学院]]，曾留学[[加拿大|加拿大]]，与人合作建立了“[[俏江南|俏江南]]”公司。儿子[[汪小菲|汪小菲]]。张兰曾荣获“中国餐饮界十大最具影响力人物”、“中国十大最具影响力首席执行官（CEO）”、“北京市[[三八红旗奖章|三八红旗奖章]]”等荣誉。<ref name=" fish" >{{cite web |url=http://finance.people.com.cn/n/2012/1121/c70846-19645230.html |title= 俏江南董事长已加入外国国籍|accessdate=2012年11月21日 |publisher=2012年11月23日 |archiveurl=http://finance.people.com.cn/n/2012/1121/c70846-19645230.html |archivedate=2012年11月23日 |work= |language =zh-cn}}</ref><ref name=" tiger" >{{cite web |url= http://www.hunantv.com/c/20121121/0826495110.html|title=张兰已加入外国国籍 法院或考虑涉外公告送达 |accessdate= 2012-11-21|publisher=金鹰网 |archiveurl=http://www.hunantv.com/c/20121121/0826495110.html |archivedate=2012年11月23日 |work= |language =zh-cn}}</ref>

2012年中，张兰加入圣基茨和尼维斯国籍，并于2012年9月17日注销其[[中华人民共和国国籍|中国国籍]]；但相关事宜并未公之于众，并仍就身兼该年新一届[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[政协委员|政协委员]]的身份，此事因另案商业纠纷而曝光，引起社会哗然。<ref name="bee "/><ref name="163com121128"/>

== 生平 ==
1958年，张兰出生在[[湖北省|湖北省]][[孝感市|孝感市]]，曾在[[北京工商大学|北京工商大学]]企业管理专业读书，大学毕业，留学[[加拿大|加拿大]]，後在加拿大餐館打工。1991年，张兰回到中国大陆, 以兩萬[[美元|美元]]在首都北京市開設「阿蘭餐館」，由餐厅做起, 後逐渐升级扩大。2000年与人合作建立了“[[俏江南|俏江南]]”餐饮公司。俏江南在中国国内享受外商投资企业优厚待遇。<ref name=" tigerman" /><ref name=" ooos" />
