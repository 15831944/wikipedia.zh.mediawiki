'''张偲'''（{{bd|1963年|10月3日|}}）<ref>{{Cite web|url=http://ysg.ckcest.cn/cae/ysgDetailsAction.do?method=baseInfo&acInfoId=3878&pagename=grxx_jbxx&flag=0|title=张偲_基本信息_中国工程院院士馆|accessdate=2018-11-18|work=ysg.ckcest.cn}}</ref>，[[海南|海南]][[文昌|文昌]]人，中国[[海洋学|海洋学]]家，主要从事海洋生物、海洋药物、海洋生物化学、生态学等领域的研究工作。中学就读于[[文昌中学|文昌中学]]<ref>{{cite web |url=http://www.hq.xinhuanet.com/news/2013-12/21/c_118652301.htm |language= |title=海南籍海洋生物专家张偲当选中国工程院院士 |publisher=海南日报 |date=2013-12-21 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150924031602/http://www.hq.xinhuanet.com/news/2013-12/21/c_118652301.htm |archivedate=2015-09-24 }}</ref>，1981年9月至1985年7月在[[山东海洋学院|山东海洋学院]]（今[[中国海洋大学|中国海洋大学]]）海洋生物系就读，毕业后进入[[中国科学院|中国科学院]][[南海|南海]]海洋研究所攻读硕士和博士，毕业后留所工作至今，历任所长助理、党委委员、职代会主席。2013年当选[[中国工程院|中国工程院]]院士。<ref>{{cite web |url=http://edu.ifeng.com/gaoxiao/detail_2013_12/19/32270320_0.shtml |language= |title=中国海洋大学1981级校友张偲当选中国工程院院士 |publisher=中国海洋大学 |date=2013年12月19日 |accessdate= |archive-url=https://web.archive.org/web/20160304191831/http://edu.ifeng.com/gaoxiao/detail_2013_12/19/32270320_0.shtml |archive-date=2016-03-04 |dead-url=yes }}</ref>
