'''张效曾'''（{{bd|1917年||2011年|8月10日}}）[[河北|河北]][[井陉|井陉]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中华人民共和国第一机械工业部|中华人民共和国第一机械工业部]]副部长兼政治部主任。

==生平==
1937年11月张效曾参加革命工作，1938年6月加入中国共产党。1937年11月，张效曾任[[井陉县|井陉县]]抗日民主政府宣教科科长、秘书。1938年6月后，先后在晋东南军政干部学校、[[延安|延安]][[抗日军政大学|抗日军政大学]]、延安马列学院学习，历任马列学院校务处教育干事，[[中共中央党校|中央党校]]三部总务处副处长、校务部办公室主任等职。1947年2月后，任晋绥土改工作团支部书记、区特派员。1948年9月后，任[[中共中央华北局|中共中央华北局]]党校副处长，[[华北人民革命大学|华北人民革命大学]]校务处副主任、代主任。1949年9月后，历任[[中共中央组织部|中共中央组织部]]秘书处第一副处长、处长，[[国家机关事务管理局|中央人民政府政务院机关事务管理局]]副局长、党委书记，政务院分党委会委员等职。1957年3月后，任[[中华人民共和国第一机械工业部|中华人民共和国第一机械工业部]]办公厅主任，部党委委员、部机关党委书记、部政治部主任。1978年4月，任中华人民共和国第一机械工业部副部长兼政治部主任、部党组成员。1982年11月离休<ref name=sina>{{Cite web |url = http://news.sina.com.cn/c/2011-09-04/030523101132.shtml |title = 张效曾同志逝世 |author =  |date =2011-09-04  |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref name=qq>{{Cite web |url =https://news.qq.com/a/20110825/000968.htm  |title = 原第一机械工业部副部长张效曾同志逝世 |author =  |date = 2011-08-25 |publisher =  腾讯|language =  |accessdate =  |quote =  }}</ref>。

2011年8月10日，张效曾在[[北京|北京]]病逝，享年94岁<ref name=sina/><ref name=qq/>。
