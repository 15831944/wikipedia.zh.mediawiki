'''张继平'''（{{bd|1958年|7月11日|}}），[[中国|中国]][[山东省|山东省]][[成武县|成武县]]人，中国数学家，[[北京大学数学科学学院|北京大学数学科学学院]]教授，专业为[[代数学|代数学]]，主要研究现代{{le|模表示论|Modular representation theory}}。由于[[文化大革命|文化大革命]]，张继平高中毕业后无法继续学业，成为了小学教员。[[恢复高考|恢复高考]]后，他考入[[山东大学|山东大学]]数学系。本科毕业后，他在[[北京大学|北京大学]]获得硕士、博士学位，师从数学家[[段学复|段学复]]。博士毕业后张继平留校任教，先后被北京大学破格提拔为副教授、教授。1998年至2008年，张继平任北京大学数学科学学院院长。

== 求学 ==
张继平生于鲁西南的乡村。据报道，中学时代张继平语文、数学、物理三科的成绩都很优秀，语文尤其突出，数次在作文比赛中获奖。由于时代的缘故，高中毕业之后，张继平无法继续求学，而是成为了一名小学教师。1977年，[[中华人民共和国|中华人民共和国]][[恢复高考|恢复高考]]。张继平在文革之后的第一次高考中考上了[[山东大学|山东大学]]数学系。本科毕业之后，他进入[[北京大学|北京大学]]，师从[[中国科学院院士|中国科学院院士]]、[[北京大学数学力学系|北京大学数学力学系]]前主任[[段学复|段学复]]教授。段学复是中国模表示论研究的开路人。张继平在北大攻读硕士、博士学位的五年时间里，一直跟从段学复学习、研究模表示论。1987年，他以论文《亏数零P块的存在性》获得博士学位。<ref name=pkunews>{{cite news|title=北京“十大杰出青年”之张继平：用睿智演绎生命的精彩|url=http://pkunews.pku.edu.cn/zdlm/2007-05/29/content_114521.htm|accessdate=2017-04-21|work=大才精诚|publisher=北京大学新闻网|date=2007-05-29|deadurl=no|archiveurl=https://web.archive.org/web/20180123152732/http://pkunews.pku.edu.cn/zdlm/2007-05/29/content_114521.htm|archivedate=2018-01-23}}</ref>

== 任教 ==
张继平1987年博士毕业后留校任教。1988年，他被北京大学破格聘为副教授。1990年，北京大学破格提拔他为教授、博士生导师，张继平成了当时北大最年轻的教授博导之一。<ref name="pkunews" />在此期间，张继平出国访学，1989年至1990年间在[[美国|美国]][[佛罗里达大学|佛罗里达大学]]担任访问学者，1991年至1993年在[[法国|法国]][[巴黎高等师范学院|巴黎高等师范学院]]数学与计算机系任研究员，1993年至1994年受[[洪堡基金|洪堡基金]]资助在[[德国|德国]][[美因茨大学|美因茨大学]]从事研究。1994年至1995年，他在美国[[明尼苏达大学|明尼苏达大学]]担任访问教授。1995年，他还在美国[[俄亥俄州立大学|俄亥俄州立大学]]任访问教授。<ref name="MathUnionCV" />

1995年，张继平返回中国。1998年，张继平出任北京大学数学科学学院院长。院长任上，他改变了之前奖金平均分配的制度，还设立教学与科研相结合的五年工作周期，即教师完成四年教学任务后，可用一年时间集中从事科研而免于教学任务。2000年北京市优秀成果评比上，北大数院摘得五项第一名。他任职期间，北大数院还被评为国家级人才培养优秀基地，赢得国家教学成果特等奖。<ref name=head>{{cite news|title=数学家院长学者——记北京大学数学学院院长张继平教授|url=http://www.china.com.cn/zhuanti2005/txt/2001-07/02/content_5042348.htm|accessdate=2017-04-21|work=中国网|date=2001-07-02|deadurl=no|archiveurl=https://web.archive.org/web/20180123152732/http://www.china.com.cn/zhuanti2005/txt/2001-07/02/content_5042348.htm|archivedate=2018-01-23}}</ref><ref name="pkunews" />2000年5月，北大数院迁入新落成的理科一号楼。2001年，北大数院基础数学、计算数学、应用数学、概率统计四个专业被评为国家重点学科，其中前三个专业是再度入选，概率统计专业为首次入选。<ref>{{cite news|title=北大110周年校庆：数学科学学院成立|url=http://edu.sina.com.cn/exam/2008-04-22/0107135482.shtml|accessdate=2018-01-20|work=北京大学新闻中心|date=2008-04-22|deadurl=no|archiveurl=https://web.archive.org/web/20180123152732/http://edu.sina.com.cn/exam/2008-04-22/0107135482.shtml|archivedate=2018-01-23}}</ref>

1995年回国之后，张继平还曾于1997年在美国[[普度大学|普度大学]]任访问学者，1999年在[[香港中文大学|香港中文大学]]访学，2001年在美国[[耶鲁大学|耶鲁大学]]访学，2004年在[[香港科技大学|香港科技大学]]访学。<ref name="MathUnionCV" />

此外，张继平在学术期刊《[[数学学报|数学学报]]》《[[代数集刊|代数集刊]]》担任副主编，在《[[中国科学_(期刊)|中国科学]]》《[[科学通报|科学通报]]》《[[中国科学基金|中国科学基金]]》等杂志任编委。学术兼职包括教育部科技委数理部常务副主任，[[国际数学家联盟|国际数学家联盟]]交流与发展委员，[[中国青年科技协会|中国青年科技协会]]副会长，[[中国数学会|中国数学会]]副理事长，[[国家自然科学基金委|国家自然科学基金委]]咨询委员等。<ref name="pkunews" />

== 研究 ==
张继平主要研究现代{{le|模表示论|Modular representation theory}}。读博士期间，他解决了布劳尔39号问题（Brauer 39）<ref name="pkunews" />。他首次给出了亏数零p-块的充分必要条件。他在约化有限单群中研究了普维格（Puig）猜想。他还发展了群的算术理论。2007年，他由块分离性质得到了新p-幂零准则。<ref name=symmetry>{{cite news|title=张继平：流连数学对称之美|url=http://tech.sina.com.cn/d/2009-05-06/15043067870.shtml|accessdate=2017-04-21|work=科技日报|date=2009-05-06|deadurl=no|archiveurl=https://web.archive.org/web/20180123152732/http://tech.sina.com.cn/d/2009-05-06/15043067870.shtml|archivedate=2018-01-23}}</ref>

== 获奖 ==
张继平曾获1996年教育部科技进步一等奖、1997年国家自然科学三等奖。1998年，他被列入国家级有突出贡献的中青年专家。2000年，他被授予第四届北京十大杰出青年称号<ref name="pkunews" />，同年被聘为[[长江特聘教授|长江特聘教授]]<ref name="MathUnionCV" />。2009年，中国数学会授予他[[陈省身数学奖|陈省身数学奖]]。此外，他还荣获首届[[中国青年科技奖|中国青年科技奖]]、[[中国青年科学家奖|中国青年科学家奖]]、[[“求是”杰出青年奖|“求是”杰出青年奖]]、[[全国五一劳动奖章|全国五一劳动奖章]]和[[全国优秀科技工作者|全国优秀科技工作者]]荣誉称号。<ref name="Chern">{{cite news|title=北大教授张恭庆、张继平分别获“华罗庚数学奖”和“陈省身数学奖”|url=http://pkunews.pku.edu.cn/zdlm/2009-04/23/content_145573.htm|accessdate=2018-01-20|work=北京大学新闻网|date=2009-04-23|deadurl=no|archiveurl=https://web.archive.org/web/20180123152731/http://pkunews.pku.edu.cn/zdlm/2009-04/23/content_145573.htm|archivedate=2018-01-23}}</ref>

张继平曾在2003年<ref>{{cite journal|title=2003年中国科学院院士增选第二轮候选人名单|journal=科学新闻|date=2003|issue=19|pages=24}}</ref>、2007年<ref>{{cite news|title=2007年中国科学院院士增选有效候选人名单|url=http://news.sina.com.cn/o/2007-05-29/040911913947s.shtml|accessdate=2018-01-20|work=光明网-光明日报|date=2007-05-29|deadurl=no|archiveurl=https://web.archive.org/web/20180123152732/http://news.sina.com.cn/o/2007-05-29/040911913947s.shtml|archivedate=2018-01-23}}</ref>、2009年<ref>{{cite news|title=中国科学院公布2009年院士增选初步候选人名单|url=http://www.edu.cn/edu/jiao_yu_zi_xun/edu_liter/jyzl_zhzl/200909/t20090901_403751.shtml|accessdate=2018-01-20|work=中国科学院|date=2009-09-01|deadurl=no|archiveurl=https://web.archive.org/web/20180103101322/http://www.edu.cn/edu/jiao_yu_zi_xun/edu_liter/jyzl_zhzl/200909/t20090901_403751.shtml|archivedate=2018-01-03}}</ref>、2011年<ref>{{cite news|title=中科院院士增选有效候选人314名（附全名单）|url=http://www.caijing.com.cn/2011-05-25/110728792.html|accessdate=2018-01-20|work=财经网|date=2011-05-25|deadurl=no|archiveurl=https://web.archive.org/web/20150109175340/http://www.caijing.com.cn/2011-05-25/110728792.html|archivedate=2015-01-09}}</ref>、2015年<ref>{{cite web|title=关于公布2015年中国科学院院士增选初步候选人名单的公告|url=http://www.casad.cas.cn/doc/16228.html|website=中国科学院|accessdate=2018-01-20|deadurl=no|archiveurl=https://web.archive.org/web/20170630155603/http://www.casad.cas.cn/doc/16228.html|archivedate=2017-06-30}}</ref>、2017年<ref>{{cite web|title=关于公布2017年中国科学院院士增选初步候选人名单的公告|url=http://casad.cas.cn/doc/17271.html|website=中国科学院|accessdate=2018-01-20}}</ref>多次参加[[中国科学院院士增选|中国科学院院士增选]]，但从未当选。
