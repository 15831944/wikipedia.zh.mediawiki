'''张轩'''（{{bd|1958年|5月|catIdx=Zhang張}}），女，[[河北省|河北省]][[滦县|滦县]]人；[[西南政法大学|西南政法学院]]法律系毕业，经济法专业研究生，高级政工师。1979年4月加入[[中国共产党|中国共产党]]；是中共第十六至十八届中央候补委员。曾任[[中国共产党重庆市委员会|中共重庆市委]]副书记，[[重庆市高级人民法院|重庆市高级人民法院]]院长、[[中华人民共和国大法官|国家二级大法官]]。现任[[重庆市|重庆市]][[人民代表大会制度|人大常委会]]主任。

== 早年经历 ==
1975年，中学毕业的张轩在[[云南省|云南省]][[昆明市|昆明市]][[官渡区|官渡区]]插队劳动。1976年，通过招工进入云南滇中石油勘探处当工人，后转任团委干事。1981年考入[[西南政法大学|西南政法学院]]法律系学习；在校期间，曾担任学校学生会主席，[[全国学联|全国学联]]副主席。1985年大学毕业后，被提拔为[[中国共产主义青年团|共青团]][[四川省|四川省]]重庆市委组织部部长。虽为法律科班出身，但此后张轩一直在[[重庆市|重庆市]]党务机关、社团单位工作；曾历任共青团四川省重庆市委副书记，重庆市（[[省辖市|省辖市]]）青联副主席、主席，[[大渡口区|大渡口区]]委副书记，四川省重庆市妇联主席等职。1997年，重庆市升格为直辖市后，张轩续任重庆市妇联主席。<ref>{{cite web |accessdate=2012-12-01 |url = http://cpc.people.com.cn/GB/64162/123659/123804/7330867.html |title=张轩同志简历 |publisher=[[人民网|人民网]]}}</ref>

== 重庆高院院长 ==
2001年1月，张轩被任命为[[重庆市高级人民法院|重庆市高级人民法院]]副院长；当年6月，前院长[[赵俊如|赵俊如]]由于“违反组织纪律”被处理，并辞去了院长职务；重庆市人大遂任命张轩为市高院代理院长；并于次年1月，正式出任重庆市高院院长。<ref name="FB">{{cite web|accessdate=2012-12-01|url=http://www.yuandaep.com/%E6%94%BF%E5%B7%A5%E5%B7%A5%E4%BD%9C/%E7%BA%AA%E6%A3%80%E7%9B%91%E5%AF%9F/tabid/93/articleType/ArticleView/articleId/1726/CategoryID/31/.aspx|title=重庆高院的腐败案件|publisher=中电投远达环保工程有限公司官网}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>  2002年3月，44岁的张轩获任[[中华人民共和国大法官|中华人民共和国二级大法官]]，成为首批任命的41位大法官中年龄最小的一位；同时也是仅有的三位女大法官之一（其她两位分别是，前[[安徽省高级人民法院|安徽省高院]]院长[[韩云萍|韩云萍]]，前[[宁夏回族自治区高级人民法院|宁夏自治区高院]]院长[[黑俊英|黑俊英]]）<ref>{{cite web |accessdate = 2012-12-01 |url = http://big5.china.com.cn/policy/txt/2002-03/22/content_5122132.htm|title=我国首次任命中国大法官、大检察官（附名单） |publisher=[[中国网|中国网]]}}</ref>。

== 重庆市委副书记 ==
2007年5月27日，在[[中共重庆市委|中共重庆市第三届委员会]]第一次会议上，张轩当选市委副书记；当年7月起，兼任市委党校校长<ref>{{cite web |accessdate=2012-05-01 |url = http://news.xinhuanet.com/local/2007-05/27/content_6159440.htm |title = 汪洋任重庆市委书记 王鸿举张轩任副书记 |publisher=[[新华网|新华网]]}}</ref>；12月，辞去了重庆市高院院长的职务<ref>{{cite web|accessdate=2012-12-01|url=http://www.cq.xinhuanet.com/2007-12/22/content_12014298.htm|title=张轩辞去市高院院长职务 钱锋被任命代理院长|publisher=[[新华网|新华网]]|deadurl=yes|archiveurl=https://web.archive.org/web/20150923211408/http://www.cq.xinhuanet.com/2007-12/22/content_12014298.htm|archivedate=2015-09-23}}</ref>。2013年1月，当选为[[重庆市|重庆市]][[人大常委会|人大常委会]]主任；当年4月起，不再担任市委副书记职务。<ref>[http://district.ce.cn/newarea/sddy/201304/24/t20130424_24319938.shtml 张国清接替张轩任重庆市委副书记]</ref>

迄今为止，张轩连续当选了三届中共候补中央委员，已和四位重庆市委书记共事，[[汪洋_(政治人物)|汪洋]]、[[薄熙来|薄熙来]]、[[张德江|张德江]]、[[孙政才|孙政才]]。
