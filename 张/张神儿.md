'''张神儿'''（{{bd|1995年|9月9日}}），湖北武汉人，中国内地[[女歌手|女歌手]]、网络[[主播|主播]]<ref name ="甲">{{cite web |title=从优秀主播成长历程 看泛娱乐时代如何打造星主播 |url=http://games.ifeng.com/a/20170929/44703543_0.shtml |website=凤凰网 |accessdate=2018-10-25}}</ref>，浙江卫视《[[2018中国好声音|中国好声音2018]]》学员。<ref>{{cite web |title=张神儿个人资料 |url=http://www.edushi.com/renwu/info/1-7-n3884.html |website=E都市 |accessdate=2018-09-25}}</ref><ref>{{cite news |title=好声音学员杭州远洋乐堤港见面会 张神儿赵家豪欢乐开唱 |url=http://ent.ifeng.com/a/20180903/43096607_0.shtml|work=凤凰网 |accessdate=2018-10-25 |date=2018-09-03}}</ref>
==演艺经历==
张神儿在中学阶段就开始参加歌唱比赛，并取得优异成绩。<ref name="乙" />

到了大学阶段，张神儿组建了乐队，并以嘉宾乐队的身份参加明星见面会活动，甚至和[[黑豹乐队|黑豹乐队]]、[[动力火车|动力火车]]等知名艺人同台演出过。<ref name ="乙">{{cite web |title=音乐主播《母系社会》燃炸全场 李健：不转身就转世 |url=http://ent.rednet.cn/c/2018/08/08/4700736.htm |website=红网 |accessdate=2018-10-25}}</ref>与此同时，她又因为常在歌唱比赛中翻唱[[阿黛尔|阿黛尔]]的歌曲，被学校里的人称作“武汉阿黛尔”。

2016年，入驻中国大陆网络直播平台[[熊猫TV|熊猫TV]]，以表演唱歌为主。

2017年，参加《华谊新星盛典》，最终进入全国十强。<ref name="甲" />

2018年，参加浙江卫视大型音乐节目《[[中国好声音|中国好声音]]2018》<ref>{{cite news |title=《中国好声音》四大战队团体赛落幕 恭喜哈林的“哈士奇”战队获“最佳战队” |url=http://www.yangtse.com/app/ent/2018-10-14/627042.html |work=扬子晚报|accessdate=2018-10-25 |date=2018-10-14}}</ref>，在盲选阶段凭借一首《母系社会》获得四位导师的转身，并成功地加入了周杰伦战队，进入全国24强。<ref>{{cite web |title=《中国好声音3》：张神儿《母系社会》 |url=http://n.cztv.com/wy/12968005.html |website=新蓝网 |accessdate=2018-09-25}}</ref><ref>{{cite web |title=《中国好声音》热播 选手们完美实力强吸粉！ |url=http://ent.163.com/18/0731/11/DO1RVGRV000380EO.html |website=网易 |accessdate=2018-09-25}}</ref>在七强争夺赛中，张神儿不敌李健战队学员[[旦增尼玛|旦增尼玛]]，无缘全国七强，最终停留在周杰伦战队3强、全国14强。<ref>{{cite web |title=《中国好声音》七强争霸启示 勇敢跳出“安全区”，才是突破与成长 |url=http://www.zjstv.com/zcloud/4906735.html |website=浙江卫视 |accessdate=2018-10-25}}</ref>由于登顶“好声音人气榜”，张神儿获得了返场表演机会，并于2018年10月7日在《中国好声音2018》鸟巢总决赛的第一轮竞演（导师帮唱环节），与导师[[周杰伦|周杰伦]]共同帮唱学员[[周兴才让|周兴才让]]。<ref>{{cite news |title=中国好声音2018鸟巢总决赛直播！旦增尼玛夺冠 宿涵第一个淘汰 |url=http://news.chinaxiaokang.com/dujiazhuangao/2018/1007/518286_5.html |work=中国小康网|accessdate=2018-10-25 |date=2018-10-07}}</ref><ref>{{cite news |title=李健队旦增获《好声音》冠军 周杰伦再次梦碎 |url=http://ent.sina.com.cn/music/zy/2018-10-07/doc-ihkvrhps9466410.shtml?cre=tianyi&mod=pchp&loc=6&r=0&doct=0&rfunc=12&tj=none&tr=12|work=新浪娱乐 |accessdate=2018-10-25 |date=2018-10-07}}</ref>
