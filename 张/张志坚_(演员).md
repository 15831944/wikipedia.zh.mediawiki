'''张志坚'''（{{bd|1955年|10月5日||}}）生于[[江苏省|江苏省]][[南京市|南京市]]，[[中华人民共和国|中华人民共和国]]话剧、影视演员。

==生平==
张志坚出生在一个军人家庭，父亲在文工团任职。高中毕业后，张志坚当锅炉工约四年。1979年，考入江苏人民艺术剧院学员班。此后成为江苏人民艺术剧院（2001年改为[[江苏省演艺集团|江苏省演艺集团]]话剧院）演员。1983年开始进入影视界。1996年，妻子从[[南京军区|南京军区]]前线话剧团转业到[[北京|北京]]工作，张志坚也随之到北京，成为“[[北漂|北漂]]”。此后主要接拍各类影视剧。但张志坚仍是江苏人民艺术剧院（2001年改为[[江苏省演艺集团|江苏省演艺集团]]话剧院）的国家三级演员<ref>{{Cite web|url =http://yanglao.china.com.cn/lehuo/20170421/26378.html|title =最新:55介绍至大结局 37、38集剧情介绍 生活中的高育良介绍|author =|date =2017-04-21|publisher =中国网|language =|accessdate =|quote =}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url =http://news.ifeng.com/a/20170418/50956340_0.shtml  |title = “育良书记”至今是三级演员 称不在意头衔职称 |author =  |date =2017-04-18  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.1905.com/mdb/star/2993606/life/|title = 张志坚的星路历程 |author =  |date =  |publisher =电影网  |language =  |accessdate =2017-04-30  |quote =  }}</ref>。
