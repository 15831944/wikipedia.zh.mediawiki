'''瘀點'''（petechia）是在身體上的一個紅色或紫色的血點塊，[[医学|医学]]上也称作'''出血点'''。

== 定義 ==
2毫米（mm）以下的紅色或紫色的血點塊。其他大小的紅色或紫色的血點塊請參見：[[紫斑|紫斑]]，[[瘀斑|瘀斑]]

== 造成原因 ==
造成瘀點的最基本原因是当皮肤下黏膜下微血管爆裂时，血流出来，而呈现的小血块形状。

* 过分用力[[咳嗽|咳嗽]]或[[呕吐|呕吐]]会造成脸上呈现脸部瘀點，尤其眼部周围。
* [[新生婴儿|新生婴儿]]脸部通常会呈现瘀點。
* 是[[血小板低下症|血小板低下症]]的一个症状。
* 可能是[[敗血症|敗血症]]的一个症状。
* 身体缺乏[[维他命C|维他命C]]，[[维他命K|维他命K]]，[[维他命B9|维他命B9]]或[[维他命B12|维他命B12]]。
* 患上[[骨痛熱症|骨痛熱症]]的病人。

== 診斷測驗 ==
* [[血液|血液]]所需的凝固时间。
* 可以测试血凝固功能的测验。
* [[完全血球計數|完全血球計數]]（Complete blood count）。
* [[血小板計數|血小板計數]]。
