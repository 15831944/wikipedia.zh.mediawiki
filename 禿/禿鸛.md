'''禿鸛'''（''Leptoptilos crumeniferus''），又称'''非洲秃鹳'''，是分佈在[[撒哈拉以南非洲|撒哈拉以南非洲]]的一種[[鸛|鸛]]。牠們棲息在潮濕及乾旱的環境，包括接近[[人類|人類]]的地方，如垃圾站。

== 特徵 ==
禿鸛是大型的[[鳥類|鳥類]]，最大的標本高達1.5米，重超過9公斤，翼展闊3.2米。一般而言，牠們長1.1-1.4米，翼展闊2.25-2.87米，重4.5-8公斤。牠們飛行時的頸部會縮起，彷彿[[鷺科|鷺科]]。

[[File:Marabu_8415.jpg|thumb]]
禿鸛的[[頭部|頭部]]及[[頸部|頸部]]沒有毛，背部黑色，[[腹部|腹部]]白色。牠們的喙也很大，[[喉嚨|喉嚨]]附近有粉紅色的囊。雙腳及翼都是黑色的。雄鳥及雌鳥很像，但雛鳥較為褐色，喙較細小。雛鳥要達4歲才達至完全成熟。

== 行為 ==
禿鸛是群居及有地盤的。在旱季，牠們會在樹上築巢，每胎產2-3隻蛋。

禿鸛很少出聲，示愛時其喙會發出咔嗒咔嗒聲。喉嚨上的囊亦可以發出不同的[[聲音|聲音]]。

禿鸛是[[食腐動物|食腐動物]]。牠們的[[頭部|頭部]]及[[頸部|頸部]]沒有[[羽毛|羽毛]]，就是一種適應性，因為若有羽毛的話，羽毛會黏上屍體的[[血|血]]及其他[[物質|物質]]，很難保持清潔。牠們也會吃其他細小的[[哺乳動物|哺乳動物]]及[[爬行動物|爬行動物]]，包括[[白蟻|白蟻]]、[[魚類|魚類]]、[[蝗蟲|蝗蟲]]、[[草蜢|草蜢]]、[[毛蟲|毛蟲]]、[[青蛙|青蛙]]、[[齧齒目|齧齒目]]、[[鱷魚|鱷魚]]蛋及幼鱷、[[紅嘴奎利亞雀|紅嘴奎利亞雀]]雛鳥、[[鴿|鴿]]、[[火烈鳥|火烈鳥]]、[[鸕鶿|鸕鶿]]雛鳥及[[鵜鶘|鵜鶘]]雛鳥。

== 用途 ==
禿鸛絨羽經常用來裝飾衣服及帽子，還有[[路亞|擬餌]]。<ref name="OED">[http://www.encyclopedia.com/doc/1O999-marabou.html ''The Oxford Pocket Dictionary of Current English 2008''] (Oxford University Press, 2008)</ref>
