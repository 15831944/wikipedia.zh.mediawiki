'''{{lang|en|Nutella}}（能多益）'''是[[意大利|意大利]]厂商[[Ferrero|Ferrero]]生產的棕榈油[[巧克力|巧克力]][[榛子|榛子]]酱，可如[[牛油|牛油]]、[[花生酱|花生酱]]般塗在[[麵包|麵包]]、[[餅乾|餅乾]]等食物上來增添美味，也可以用作烘烤糕點的馅料。

== 历史 ==
1940年代正值[[第二次世界大戰|第二次世界大戰]]時期，當時[[意大利|意大利]]西北一帶的[[可可豆|可可豆]]供應短缺，[[Ferrero|Ferrero]]公司的創辦人Pietro Ferrero先生便採用[[榛子|榛子]]代替[[可可豆|可可豆]]，發明了Nutella的前身「[[榛子棒|榛子棒]]」，並以當地家傳戶曉的[[嘉年華|嘉年華]]角色人物[[Gianduja|Gianduja]]，配合廣告宣傳，把榛子棒命為''pasta gianduja''。榛子棒是固體狀，以[[錫紙|錫紙]]包裝，食用時得用刀切片。及後[[Ferrero|Ferrero]]把配方改為「榛子醬」，命名為''supercrema gianduja''，意即「可塗抹的gianduja」。

1963年，[[Ferrero|Ferrero]]為了讓产品推向更广大的[[欧洲|欧洲]]市场，便採用不同歐洲語言併合而成的新品牌名稱'''Nutella'''：以當中的「Nut」代表產品的材料「[[榛子|榛子]]」（hazelnut，英語可略作nut；[[德语|德语]]則是nuss），而「ella」則取自[[意大利语|意大利语]]的通用[[詞綴|後綴]]，意思是「甜的」。这样，品牌名字不但讓人知產品的材料和味道，而且还暗含了产品的原產地[[意大利|意大利]]。厰商希望新名字會为广大消费者认可，並在1963年／1964年年注册了'''Nutella'''的品牌以及商标，一直沿用至今。1983年，Nutella引入[[美國|美國]]，並在約十年後在[[新泽西州|新泽西州]]的Somerset開設生產厰房。

== Nutella的普及 ==
Nutella的[[產品|產品]]在不同地區[[銷售|銷售]]，在[[欧洲|欧洲]]、[[澳大利亚|澳大利亚]]和[[巴西|巴西]]等地非常[[流行|流行]]。在很多[[欧洲|欧洲]]的[[城市|城市]]（比如[[巴黎|巴黎]]）有一种很受欢迎的[[街頭小吃|街頭小吃]]，用Nutella塗在[[可麗餅|可麗餅]]上，再加上[[香蕉|香蕉]]和[[草莓|草莓]]等[[水果|水果]]片混在一起吃。在[[意大利|意大利]]的很多地方，当[[麵包节|麵包节]]到来的时候，很多展台都會提供抹着Nutella的麵包供路人品尝。Nutella在1983年才引入[[美國|美國]]，由於售價較貴而未能流行起來，及至2000年代售價才調低。後來廠商請了在意大利長大的[[NBA|NBA]]明星[[科比·布莱恩特|科比·布莱恩特]]為Nutella作形象代言人。

「Nutella」在很多地區是[[巧克力|巧克力]][[榛子|榛子]]酱的代名词。但是在一些[[国家|国家]]，还是有别的[[巧克力|巧克力]][[榛子|榛子]]酱[[品牌|品牌]]比它更[[流行|流行]]，比如：[[希腊|希腊]]的''Merenda''和[[西班牙|西班牙]]本土出產的[[Nocilla|Nocilla]]（Nutella最近才进入西班牙市场）。在前[[东德|东德]]的地區，人們把Nutella產品稱作''Nudossi''。

除了[[產品銷售|產品銷售]]外，Nutella更在不同[[文化|文化]]和[[社会層面|社会層面]]中[[流行|流行]]起來。曾有不少書[[描写|描写]]过Nutella這產品，在一部[[意大利电影|意大利电影]]《Bianca》的其中一个著名的[[场景|场景]]中，也有以這產品作為故事主線。




== 成分 ==
不同地區的Nutella產品成分有異，[[欧洲|欧洲]]（以及[[波蘭|波蘭]]）的成分是：[[糖|糖]]、[[菜籽油|菜籽油]]、[[榛子|榛子]]（13%）、[[可可|可可]]（7.4%）、[[脱脂牛奶|脱脂牛奶]]（5%）、[[乳糖|乳糖]]、[[大豆|大豆]][[卵磷脂|卵磷脂]]和[[香草醛|香草醛]]。美国的成分是：[[糖|糖]]、[[棕榈油|棕榈油]]、[[榛子|榛子]]、[[可可|可可]]、[[脱脂牛奶|脱脂牛奶]]、纯[[乳清|乳清]]，部分[[氢化|氢化]][[花生油|花生油]]、[[大豆|大豆]][[卵磷脂|卵磷脂]]、[[香草醛|香草醛]]。

== 棕櫚油爭議 ==
[[歐洲食品安全局|歐洲食品安全局]]於2016年5月發表研究報告，[[棕櫚油|棕櫚油]]在超過攝氏超過200度下會產生[[致癌物質|致癌物質]] - 「縮水[[三酸甘油酯|甘油脂肪酸酯]]」（Glycidyl fatty acid esters），棕櫚油的[[三酸甘油酯|甘油脂肪酸酯]]含量是植物油中最高，比含量次高的人造奶油還要高5至6倍，因此呼籲兒童避免食用。由於Nutella榛子醬使用棕櫚油調配而成，因而其致癌風險備受關注，Nutella榛子醬在意大利更被大型超市等零售商下架。根據[[路透社|路透社]]報導，費列羅公司每年消耗18.5萬噸棕櫚油製造Nutella榛子醬等食品，棕櫚油是較低成本的植物油，如果轉換其他植物油將會令成本上升，而歐洲食品安全局的棕櫚油高溫加熱致癌報告，使大量採用棕櫚油的廠家備受壓力<ref>[http://orientaldaily.on.cc/cnt/china_world/20170115/00180_009.html 棕櫚油易致癌 Nutella榛子醬意下架]，東方日報，2017年1月15日</ref>。費列羅公司回應指旗下產品為了避免使用含[[反式脂肪酸|反式脂肪酸]]的氫化植物油才會採用棕櫚油，並聲稱透過低溫工藝使成品產生的可能致癌物質含量降至最低<ref>[https://udn.com/news/story/7266/2248134 高溫加工棕櫚油恐致癌？Nutella榛果巧克力背黑鍋]，聯合新聞網，2017-01-26</ref>。

== 趣闻 ==
在Chloé Doutre-Roussel的著作《The Chocolate Connoisseur》（意譯為「巧克力鑑賞家」；ISBN 1-55788-503-6）的一个场景中曾提及Nutella：故事讲及作者母亲在[[墨西哥|墨西哥]]的一段趣事：她带了好多瓶Nutella酱到墨西哥，为了骗过边境检查人员，假說該些酱只是护肤用品而不是违禁的食品，她居然往自己的脸上抹了一些Nutella；書中還記載了其他有關Nutella的趣聞，包括下文所述數項：

* 在[[二战|二战]]以后的意大利，由於'''supercrema gianduja'''太流行，以致一些商店提供了特別的服务，给小朋友们带来的面包片上面抹上Nutella.

* 意大利每年生产17.9万吨Nutella.

* 尽管Nutella的基本配方是公開的，但是精确的配方却像[[可口可乐|可口可乐]]的配方那样保密。

* 在欧洲，Nutella是分装在400克和750克的玻璃瓶中出售，也有200克的玻璃杯裝潢，杯口没有螺口，盖子是简单的压上去的，所以人們吃完了Nutella酱，杯子可以留起當作普通的杯子使用。很多人用杯子來裝牛奶，容量正好是配合一份Nutella夹心面包的量。Nutella还推出過不同款式的玻璃杯，刺激了顧客收集玻璃杯的需求。

* 欧洲的Nutella的用玻璃瓶裝潢，美国則用[[塑膠|塑膠]]瓶，瓶子的外觀設計申請了[[專利|專利]]。

* 根据[https://web.archive.org/web/20060902015402/http://www.nutellausa.com/boutique.htm Nutella美国网站]，Nutella的发音应该是「new-tell-uh」，但是在英国和其他一些国家，发音是「nut-ella」，在德国和荷兰的发音則是「nuu-tella」，而英語中是沒有「nuu」这个音的。

* [[Gnutella|Gnutella]]是一種文件共享协议，他的名字是根据[[GNU|GNU]]和「Nutella」共同得来的。

* [[Plumpy'nut|Plumpy'nut]]是種一种在援助饥荒时使用的食品，生產這種食品作救援用途的靈感源自Nutella.
