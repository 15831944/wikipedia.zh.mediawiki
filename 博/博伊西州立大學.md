'''博伊西州立大学'''（Boise State University），又称'''樹城州立大學'''<ref>{{Cite web|url=http://carnegieclassifications.iu.edu/lookup/view_institution.php?unit_id=142115|title=Carnegie Classifications {{!}} Institution Lookup|website=carnegieclassifications.iu.edu|access-date=2018-09-11}}</ref>，是一所位于美国[[爱达荷州|爱达荷州]][[博伊西|博伊西]]的公立研究型大学，由[[美国圣公会|美国圣公会]]创立于1932年，1934年成为独立的两年制学院，1974年成为该州的第三所州立大学。<ref>{{cite web |url=http://www.idahostatesman.com/2010/09/13/1338680/boise-state-enrollment-breaks.html|title=Boise State Enrollment Breaks Record |accessdate=December 23, 2017}}</ref>

按学生数算，它是爱达荷州最大的大学<ref>{{Cite web|url=https://www.campusexplorer.com/college-advice-tips/AF81E5A2/6-Largest-Colleges-in-Idaho/|title=6 Largest Colleges in Idaho|website=www.campusexplorer.com|language=en|access-date=2018-09-19}}</ref>，提供的本科学位数量在州内位于第一<ref>{{cite web |url=https://focus.boisestate.edu/article/presidents-report2017/|title=2017 President’s Report - Focus |website=focus.boisestate.edu |accessdate=December 23, 2017}}</ref>。2018年《[[美國新聞與世界報道|美國新聞與世界報道]]》將其列在全美大学排名中的第281-301位<ref>{{cite web|url=https://www.usnews.com/best-colleges/boise-state-university-1616|title=Boise State University|publisher=USNews |date=|accessdate=2018-09-18}}</ref>。
