'''哈奇的博尚领主爱德华·西摩'''（{{bd|1561年|9月21日|1612年|7月21日|catIdx=S}}），对英格兰王位有理论上的强权利，但事实上因身份合法性存疑，沦为无足轻重的人。

== 家世 ==
爱德华·西摩是[[第一代赫特福德伯爵爱德华·西摩|第一代赫特福德伯爵]]（1539年—1621年）和其妻、“九日女王”[[简·格雷|简·格雷]]之妹[[凯瑟琳·格雷|凯瑟琳·格雷]]（死于1568年）之子。他的祖父[[第一代薩默塞特公爵爱德华·西摩|第一代薩默塞特公爵爱德华·西摩]]（1552年伏诛），在外甥[[爱德华六世|爱德华六世]]年间（1547年—1553年）因被国会剥夺财产和公民权，全部爵位被废除。他的父亲被女王[[伊丽莎白一世|伊丽莎白一世]]（1558年—1603年在位）于1559年重新提拔为贵族，为哈奇的博尚男爵和赫特福德伯爵。他在父亲生前去世，在父亲生前以父亲较低的爵位仪礼称号称“博尚领主”。

他的母亲因违抗伊丽莎白一世的意愿秘密与其父结婚，被囚于[[伦敦塔|伦敦塔]]，他就在那里出生。他的母亲进入伦敦塔时已经身怀有孕，被提供糟糕的生活条件，显然意在希望她流产或死亡。多年来，因为没有合法结婚的证据，凯瑟琳诸子被认为私生子。

== 王位权利 ==
1603年，爱德华是[[亨利八世|亨利八世]]遗言规定的第一王位继承人。按遗言约定，都铎长系[[玛格丽特·都铎|玛格丽特·都铎]]的后人斯图亚特系的继承权被跳过，以便幼系、亨利八世最喜欢的妹妹[[玛丽·都铎|玛丽·都铎]]的后人继位。按遗言，爱德华唯一可能的竞争对手是[[卡斯尔哈文伯爵夫人安妮·斯坦利|卡斯尔哈文伯爵夫人安妮·斯坦利]]，一旦爱德华被认为私生子，安妮将成为王位继承人。但王位继承最终由长系继承，[[苏格兰|苏格兰]]国王[[詹姆斯一世_(英格兰)|詹姆斯六世]]坐上了英格兰的王座。如亨利八世遗言的约定得到实施，爱德华将成为英格兰国王爱德华七世，英格兰和苏格兰也不会成为共主邦联。伊丽莎白一世从未官方取消其父的遗言，但詹姆斯在英格兰议会支持下继承了王位。

== 家庭和子女 ==
1572年，爱德华娶了霍诺拉·罗杰斯，她是多塞特选区下院议员多塞特的布莱安斯顿的理查·罗杰斯爵士（1527年—1605年）和索默塞特的邓斯特城堡的邓斯特封建领主安德鲁·卢特雷尔爵士（1484年—1538年）的女儿塞西莉亚·卢特雷尔的女儿。<ref>http://www.historyofparliamentonline.org/volume/1558-1603/member/rogers-richard-1527-1605</ref>霍诺拉是布莱安斯通的下院议员安德鲁·罗杰斯的姐妹（大约死于1599年），<ref>http://www.historyofparliamentonline.org/volume/1558-1603/member/rogers-andrew-1599</ref>安德鲁之前娶了博尚领主的姑母玛丽·西摩女领主（1552年生）为第二任妻子。博尚的父亲伯爵强烈反对这场婚姻，派了一位乔治·卢德洛去和罗杰斯家讨论态度。卢德洛回报霍诺拉是“一个荡妇”、其父理查·罗杰斯是“一个笨蛋”、博尚领主最初只想和她“一夜情”。伯爵相信理查爵士已鼓励这场婚姻，对于反对或推动这场婚姻“虽然表面上他什么也没做”，其女也有“秘密的善意”。<ref>http://www.historyofparliamentonline.org/volume/1558-1603/member/rogers-richard-1527-1605</ref>他们有三子三女：

*博尚领主爱德华·西摩（1586年 - 1618年），娶[[第二代多塞特伯爵罗伯特·萨克维尔|第二代多塞特伯爵罗伯特·萨克维尔]]之女安妮·萨克维尔，先于祖父而死，无子女。
*[[第二代索默塞特公爵威廉·西摩|第二代索默塞特公爵威廉·西摩]]（1588年 - 1660年），后为第二代赫特福德伯爵，最终于1660年重获曾祖父[[第一代索默塞特公爵爱德华·西摩|第一代索默塞特公爵爱德华·西摩]]（约1500年—1552年）生前获授予的索默塞特公爵头衔。
*马尔博罗城堡的[[特罗布里奇的第一代西摩男爵弗朗西斯·西摩|特罗布里奇的第一代西摩男爵弗朗西斯·西摩]]（约1590年 - 1664年），后进爵为特罗布里奇的西摩男爵。[[第五代索默塞特公爵弗朗西斯·西摩|第五代索默塞特公爵弗朗西斯·西摩]]（1658年—1678年）和其弟“骄傲的公爵”[[第六代索默塞特公爵查尔斯·西摩|第六代索默塞特公爵查尔斯·西摩]]（1662年—1748年）的祖父。
*霍诺拉·西摩（1594年之前 - 1620年），嫁[[第五代达德利男爵爱德华·萨顿|第五代达德利男爵爱德华·萨顿]]子费迪南多·萨顿爵士。
*安妮·西摩
*玛丽·西摩

== 过世和继承 ==
他先于其父第一代赫特福德伯爵去世，仪礼称号由长子博尚领主爱德华·西摩（1586年—1618年）继承，他也比第一代赫特福德伯爵先去世，故由他的次子威廉·西摩于1621年继承祖父为第二代赫特福德伯爵，并于1660年被重新封为第二代索默塞特公爵。他的黄铜纪念碑碑文保存在大贝德文教堂（[[狼厅|狼厅]]和[[托特纳姆|托特纳姆]]的教区教堂)，用拉丁文写着：<ref>Frederic Madden, Bulkeley Bandinel, John Gough Nichols, (Eds.), ''[https://books.google.co.uk/books?id=JS4EAAAAIAAJ&pg=PA21 Collectanea Topographica Et Genealogica]'', Vol.5, p.24 </ref>
