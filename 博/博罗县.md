'''博罗县'''为[[中国|中国]][[广东省|广东省]]下辖[[县|县份]]，隶属[[惠州市|惠州市]]。博罗于前214年置县，名“'''傅罗'''”，280年改“傅罗”称“'''博罗'''”县至今。博罗县位于广东省中南部、[[珠江三角洲|珠江三角洲]]东北端，[[东江|东江]]下游北岸，处北纬23°03′50″至23°43′20″，东经113°49′50″至114°45′50″之间。县境东北与[[河源市|河源市]][[源城区|源城区]]、[[东源县|东源县]]和[[紫金县|紫金县]]交界，东南与[[惠阳区|惠阳区]]、[[惠城区|惠城区]]相连，南隔东江与[[东莞市|东莞市]]相望，西与[[广州市|广州市]][[增城区|增城区]]相连。博罗县面积2,795平方公里，2009年年末时户籍人口有813,730人。博罗为广东省较具实力的[[县级行政区|县（市）]]，综合竞争力[[中国百强县#县域经济论坛评价|2010年位居全国百强县（市）第89位]]。2009年，全市[[地区生产总值|GDP总量]]252.6705亿元（折合36.9888亿美元），[[广东各县市地区生产总值列表|居广东各县（市）第3位]]；人均GDP为26,398元（合3,864美元），[[广东各县市地区生产总值列表|居各县（市）第6位]]。博罗下辖17个镇，政府驻地[[罗阳镇_(博罗县)|罗阳镇]]<ref>2009年末户籍人口与经济数据来源于《广东统计年鉴2010》，辖境面积来源于[http://xzqh.cn/ONEWS_zq.asp?id=983 民政部-行政区划网] {{webarchive|url=https://web.archive.org/web/20101202023324/http://xzqh.cn/ONEWS_zq.asp?id=983 |date=2010-12-02 }}</ref>。

== 历史 ==
[[File:Guangdong_Sheng_Bowuguan_2012.11.18_11-04-36.jpg|缩略图]]]]
[[秦始皇|秦始皇]]三十三年（前214年）設傅羅縣，[[晉武帝|晉武帝]]太康元年（280年）改為[[博羅縣|博羅縣]]。1963年属于惠阳专区管辖，1988年属于[[惠州市|惠州市]]管辖。是[[廣東省|廣東省]]四大古縣之一。

== 行政区划 ==
博罗县下辖2个街道和15个镇：罗阳街道、龙溪街道、公莊镇、[[柏塘镇|柏塘镇]]、杨村镇、麻陂镇、石坝镇、泰美镇、[[园洲镇|园洲镇]]、[[石湾镇|石湾镇]]、龙华镇、福田镇、长宁镇、湖镇镇、杨侨镇、观音阁镇、横河镇。

== 民族 ==
[[File:博罗大鼓_5964.jpg|thumb]]博罗本地居民以[[汉族|汉族]][[客家人|客家人]]为主。博罗亦是少数民族分布较多的广东县份之一，有[[瑶族|瑶族]]、[[畲族|畲族]]等少数民族。

== 語言 ==
博罗通行客家话，[[客家话|客家话]]是博罗的主要方言；靠近东莞的石湾、园洲两镇则多用粤语；另有闽南语、畲语，但分布较少且分散。

* [[客家话|客家话]]：主要以惠中片客家话、本地話和平婆話為主。惠中片客家话分布于博罗大部分乡镇，和新惠小片的惠阳话十分接近； 博罗本地话和客家话相比差异较大但可以互通，和惠城话有近似之处但有一定的区别，从语音特点上语言界普遍的解释是“一种兼具粤语特点的客语”<ref>{{Cite journal|title=客家方言的分区（稿）：|author=谢留文，黄雪贞|url=|journal=中国语言地图集|issue=2007：238-249|doi=|others=|year=2007|volume=|page=12|pmid=}}</ref>；平婆话属客家话汀州片。实际上来源福建的漳潮客家话群，属闽南客家话的一种。受[[闽语|闽语]]的影响颇深。 
* [[闽语|闽语]]：主要是以鹤佬话为主，在靠近惠城的地区有较大范围的分布。鹤佬话使用者兼用客家话。  
* [[粤语|粤语]]：主要是以广府话为主，属粤语广府片，与从化、增城的粤语接近，而与广州话有所不同。广府话分布于靠近东莞的园州、石湾、九潭等地。  
* [[畲语|畲语]]：属汉藏语系苗瑶语族，不是汉语，亦不是畲族普遍使用的类似于客家话的畲话。分布于畲族地区。

== 经济 ==
博罗县的[[农业|农业]]比较有特色的是花卉、水果、蔬菜、生猪、三乌等“[[三高农业|三高农业]]”。博罗的[[工业|工业]]以[[外向型经济|外向型经济]]为主，工业比较有特色的是建材、制药、纺织、服装、化工及电子等轻工业，境内有多个大型工业区，包括宝盛电子城、宏博工业城、梅花工业园、侨兴工业城、福兴工业城。

== 旅游 ==
[[File:Boluo_County_from_Luofu_Shan.JPG|thumb]]
博罗县旅游资源丰富，县内的[[罗浮山|罗浮山]]，被誉为“岭南第一山”，是中国[[道教|道教]]十大名山之一。博罗县内有四星级酒店三家，是罗浮山大酒店、富华大酒店及嘉豪御景酒店。

== 交通 ==
[[205国道|205国道]]、[[324国道|324国道]]、[[京九铁路|京九铁路]]、[[广梅汕铁路|广梅汕铁路]]贯穿博罗全境，[[广惠高速公路|广惠高速公路]]和[[惠河高速公路|惠河高速公路]]、[[博深高速公路|博深高速公路]]也在境内交汇，东江上有6座大桥与[[东莞|东莞]]、[[惠州|惠州]]相连，交通十分便利。

== 著名人物 ==

張宋卿： 宋朝绍兴二十七年（1157）以“春秋魁为天下第一”，擢进士，历任秘书省正字，曾在肇慶擔任教育方面的官員。
*[[张萱|张萱]]：明萬曆十年[[举人|举人]]。著有《汇雅》、《疑耀》《西園聞見錄》、《西園題跋》。匯雅前集二十卷明張萱編。萬曆三十四年廣東省博羅縣張萱刊本卷二十後有“萬曆乙巳秋八月於金臺清真館刊”牌記
*[[鄧子瑜|鄧子瑜]]：[[同盟會|同盟會]]成員，1907年「[[惠州七女湖起義|惠州七女湖起義]]」領導者
*[[梁慕光|梁慕光]]：[[興中會|興中會]]成員, 義軍首領。
*[[徐子珊|徐子珊]]：香港著名女歌手及演員。
*[[黃日華|黃日華]]：香港著名男演員。
*[[葛民輝|葛民輝]]：香港男演員、導演及電台節目主持。
*[[陳秀中|陳秀中]]：大聖劈卦門宗師。
*[[鍾楚紅|鍾楚紅]]：香港著名女演員。
*[[林耀桂|林耀桂]]：龍形宗師。
*[[謝嬰白|謝嬰白]]：又名膺白，黃埔軍校教育長，國民政府軍事參議院中將。
