'''博客來網路書店（'''{{lang-en|'''books.com.tw'''}}，縮寫為'''books）'''，經常簡稱為'''博客來'''，是一家總部位於[[臺灣|臺灣]]的網路[[零售业|零售業者]]，亦是台灣第一家[[網路購物|網路]][[書店|書店]]。2001年，[[統一超商|統一超商]]投資博客來並取得過半[[股權|股權]]，博客來加入[[統一流通次集團|統一流通次集團]]。2007年，博客來[[總經理|總經理]][[張天立|張天立]]被[[董事會|董事會]]撤換並退出經營團隊。2009年，博客來成為臺灣圖書市場銷售規模第一大通路。

== 沿革與發展 ==
[[File:Books-com-tw_box_W020_20130125.jpg|thumb]]
*1995年，博客來由[[張天立|張天立]]等人成立，是臺灣最早成立的網路書店。 
*1995年12月27日，[[張天立|張天立]]於[[臺北市|臺北市]]成立博客來數位科技股份有限公司。
*1996年7月30日，博客來網路書店正式上網營運。
*1999年，博客來網路書店成立自營的物流中心。
*1999年7月12日，《[[商業周刊_(臺灣)|商業周刊]]》評選博客來網路書店為「臺灣25大值得投資的潛力網站」零售網站類股前三名。
*2000年，與統一超商合作，展開「博客來訂書，7-ELEVEN取貨再付款」服務機制。 
*2000年，成立CD音樂館。與統一超商合作，開創「博客來購買，7-11取貨付款」機制。
*2000年6月1日，博客來網路書店與[[統一超商|統一超商]]合作，展開「博客來訂書，7-11付款取貨」通路。
*2000年10月27日，博客來網路書店榮獲（[[中國時報|中國時報]]、[[PChome電腦家庭出版集團|PChome電-{}-腦家庭出版集團]]、[[中時網路科技|中時網-{}-路科技]]及[[PC_home_Online網路家庭|PC home Online網路家庭]]合辦）第二屆[[金手指網路廣告獎|金手指網路廣告獎]]「最佳電子商務獎」銀牌獎及「最佳網站內容獎」佳作。
*2000年12月，統一超商宣佈將投資博客來網路書店新臺幣1億40萬元，取得50.025%的過半股權。
*2001年，統一超商投資博客來。 
*2001年，成立電子雜誌、中英日文雜誌館。 
*2002年，[[大智通文化行銷|大智通文化行銷]]代管博客來網路書店物流中心。
*2002年，成立MOOK館。商品頁呈現實際庫存量。
*2003年，推出全球首創「即時庫存顯示」機制。 
*2003年，成立DVD館。開始獲利，連續三年50%以上的正成長。 
*2004年，海外遞送全面升級為「快遞」服務。 
*2004年，成立簡體字館。
*2004年10月11日，博客來網路書店承辦[[臺北市政府文化局|臺北市政府文化局]]第七屆[[台北文學獎|臺北文學獎]]「築夢臺北」系列活動。
*2004年11月9日，博客來網路書店簡體字館上線銷售；同年，博客來網路書店海外遞送全面升級為[[快遞|快遞]]服務。
*2005年，宅配服務全面提升為[[黑貓宅急便|黑貓宅急便]]。
*2005年，成立美妝、親子、精品、玩藝、百貨、軟體3C館。
*2005年7月8日，博客來網路書店第六屆金手指網路獎「最佳電子商務獎」佳作。
*2005年7月26日，博客來網路書店「2005博客來十週年慶」活動。
*2006年，成立博客來售票網、美食館、外文館與相機館。 
*2007年5月15日，博客來網路書店「松鼠窩」社群網站開通；同年，博客來網路書店入選《[[天下雜誌|天下雜誌]]》2006年500大服務業及營運績效100強，博客來網路書店與[[統一客樂得|統一客樂得]]合作貨到付款，創世際網購滿意度調查博客來網路書店是網友評價最高的網站。
*2008年，與統一超商合作「24小時隔日取貨、全年無休」。 
*2008年1月10日，博客來網路書店與統一超商合作「24小時隔日取貨、全年無休」。
*2009年，連續三年每年維持30%的營收成長。
*2009年，博客來網路書店推出會員分級制度，差異化經營、提升會員的黏著度。
*2010年，深化閱讀媒體之經營，建立「OKAPI」平臺。同時建立「三魚網」平臺，深耕中學生閱讀族群。 
*2010年，整併百貨商品為美妝保健、服飾配件、美食飲品、創意生活、玩藝玩具、3C、家電等7館。
*2010年11月8日，博客來OKAPI網站成立；同年，三魚網上線，深耕中學生閱讀族群。
*2011年，推出「行動博客來」手機購物服務。榮獲《數位時代》2011臺灣電子商務TOP50強第一名。
*2011年1月4日，博客來網路書店推出新[[企業識別|企業識別]]系統（CIS），更名「博客來」，為消費者提供更多元的消費選擇。連續三年入選《[[數位時代|數-{}-位時代]]》評比之臺灣網站100強，並推出「行動博客來」手機購物服務。榮獲《數-{}-位時代》2011臺灣電子商務50強第一名。
*2012年，榮獲《天下雜誌》第二屆金牌服務大賞-「網路購物中心」類別第一名。[[資策會|資策會]]舉辦「2012科技創新力調查」企業「卓越獎」。《壹週刊》「第9屆服務第壹大獎」網路商城及拍賣類第三名。 
*2012年，打造質感生活，成為質感生活的新入口！ 
*2013年，連續五年入選《數位時代》評比之臺灣網站100強。榮獲《工商時報》臺灣服務業大評鑑大型購物網站「銅獎」。
*2013年，會員數突破500萬人。10月份美食館成立「安心食材專賣店」打造臺灣最大安心食材網購通路。  
*2014年，啟動「博客來訂購、香港7-ELEVEN門市取貨」新服務，博客來首次將「到店取貨服務」延伸至海外地區，打造成為香港最大的網路書店。
*2014年，資策會「2014科技創新力調查」企業「卓越獎」。
*2015年3月16日，宣布將海外門市取貨服務從香港拓展至澳門地區，成為博客來第二個海外店取服務據點；同步開放取貨商品限制，從圖書擴及臺灣設計文具、開架彩妝、美食乾貨、沖泡飲品、3C周邊等4萬種百貨商品，成為第一個提供百貨商品海外超商門市取貨服務的臺灣網購平臺。
*2015年8月25日，於成立20週年記者會宣佈，於年底推出[[電子書|電子書]]服務<ref>{{Cite web |url = http://www.inside.com.tw/2015/08/25/books-dot-com-dot-tw-announce-that-its-ebook-store-is-coming |title = 終於！博客來宣佈年底推出電子書服務 |author = Titan |date = 2015-08-25 |publisher = Inside 硬塞的網路趨勢觀察 |language = zh-hant |accessdate = 2015-09-29 |quote = 只比亞馬遜晚幾個月推出、以網路書店起家的博客來，在成立20週年之際，於今日宣佈年底將推出電子書服務。}}</ref>。
*2016年，海外門市取貨服務從港澳地區拓展至新加坡，成為博客來第三個海外店取服務據點。
*2017年9月4日，正式推出電子書服務，迎接行動數位閱讀世代。
*2018年8月7日，宣布海外門市取貨服務拓展至馬來西亞，成為博客來第四個海外店取服務據點。<ref>{{Cite web |url = https://www.chinatimes.com/newspapers/20180808000366-260204 |title = 博客來 新增大馬店取服務 |date = 2018-08-08 |publisher = 中時電子報 |language = zh-hant |accessdate = 2018-08-08 }}</ref>。
