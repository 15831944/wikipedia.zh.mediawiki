{{日本作品|博多豚骨拉面团|博多豚骨ラーメンズ}}是木崎千秋著作，一色箱负责插画的小说作品，是以[[博多|博多]]為舞台、描寫殺手的群像劇。MediaWorks文库（KADOKAWA）于2014年2月起发行，第20届（2013年）[[电击小说大奖|电击小说大奖]]大赏作品。<ref>{{Cite web |url = http://ascii.jp/elem/000/000/842/842693/ |title = 2作品が大賞に！　第20回電撃大賞贈呈式が開催 |date = 2013-11-14 |work=ASCII.jp|publisher = [[KADOKAWA|KADOKAWA]] |language = ja |accessdate = 2018-01-13 }}</ref>

2017年7月10日宣布動畫化。同年10月1日在「電擊文庫2017秋之祭典」宣布電視動畫於2018年1月12日起於[[AT-X|AT-X]]電視台晚上首播。

==登场角色==
=== 博多豚骨拉麵團 ===
; {{nihongo|馬場 善治|馬場 善治|声：[[小野大辅|小野大辅]]}}
:熱愛博多的私家偵探。在博多經營“馬場偵探事務所”。同時也是殺手，擔任殺手時會戴上博多傳統面具並使用武士刀，而被稱為“仁和加武士”，同時因連續接下殺掉殺手的委託而有了刺客殺手（殺手殺手）(殺手誅殺者)的名稱。
:實力強大，個性樂天，有點懶散，自從林來之後，事務所的衛生環境都是他在打掃。
:喜歡豚骨拉麵、明太子和棒球。
:棒球位置：二壘手。

; {{nihongo|林憲明|林 憲明|声：[[梶裕贵|梶裕贵]]}}
:喜歡穿女裝的男性殺手。因為覺得自己穿女裝看起來和妹妹很相似，以此來思念妹妹；也認為自己穿女裝比其他女生漂亮而有優越感。
:會講流利日文的中國人。資料上稱台灣人，實際上出生[[雲南|雲南]][[昆明|昆明]]鄉下，本名：林貓梅。
:母親（声：[[天野真實|天野真實]]）貧病不肯鬻兒賣女，林因此對家人謊稱工作而自賣。從九歲就開始接受殺人訓練，並在訓練場時認識了緋狼，與緋狼有著深厚的情誼，但訓練場的畢業條件卻是要他們互相殘殺，但緋狼卻毫不考慮偷襲將林刺傷，也讓他對於同伴之間的信任有著一定之間的恐懼。專長是使用刀具，尤其是匕首類型的小刀。原本受僱於新興跨國地下組織“華九會”，但組織將妹妹林僑梅（声：[[石上靜香|石上靜香]]）進行人口販賣給市長的兒子而使她遭殺害。林為了報仇借助馬場的力量，並因為馬場開的代價——五年份的明太子，而繼續留在馬場的事務所中借住。經過多次事件也漸漸信任了馬場。
:棒球位置：游擊手。

; {{nihongo|齊藤|斉藤|声：[[小林裕介|小林裕介]]}}
:殺人承包公司“Murder Inc.”的菜鳥社員。入社半年，從東京本部調職到福岡支店，而來到博多。
:高中時當過棒球投手，曾經在比賽時差點用棒球砸死人，因此有一段時間無法投球，直到某次為了救林才又能再次投球。
:沒什麼特別突出的殺手能力，個性稍微膽小又容易被人牽著鼻子走。
:運氣很差，常替人背黑鍋，但都能巧妙的化險為夷。
:在某次事件後小整形怕被認出。
:棒球位置：投手。

; {{nihongo|榎田|榎田|声：[[小野賢章|小野賢章]]}}
:以網咖為據點的天才駭客情報商，為某議員的兒子，8年前曾被父親下令殺掉，因此資料上顯示已死亡，但實際上父親是想讓他自由。曾從屬於海外的駭客集團。
:常以用小動作偷裝蜘蛛造型的追蹤竊聽器在他人身上耍小巧思，實際幾乎沒有打鬥能力。
:白金色的蘑菇頭和顯眼的服飾是其特徵。
:棒球位置：中外野手。

; {{nihongo|次郎|ジロー|声：[[浪川大輔|浪川大輔]]}}
:曾經是美容師，在戀人被殺手所殺後轉職成復仇代理人。與助手美咲共同行動。
:講話溫柔且有耐性，比較有著母親的屬性。
:能夠一拳讓職業殺手倒地不起。
:棒球位置：左外野手。

; {{nihongo|美咲|ミサキ|声：[[悠木碧|悠木碧]]}}
:小學生。復仇屋的助手。由於過去的一些原因而被次郎帶走。
:擅長使用還是小女孩的身分協助次郎的任務。
:有著成熟冷靜的談話及思考模式，甚至還嗆離家出走的林幼稚。
:球隊經理。

; {{nihongo|何塞·馬丁內斯|ホセ・マルティネス|声：[[前野智昭|前野智昭]]}}
:整體師兼拷問師。出身於多明尼加共和國。
:似乎是同性戀，1號。
:棒球位置：一壘手。

; {{nihongo|大和|大和|声：[[松岡禎丞|松岡禎丞]]}}
:牛郎俱樂部“Adam”的人氣牛郎。外表輕浮，但扒手手段高超。
:棒球位置：右外野手。

; {{nihongo|佐伯|佐伯|声：[[平川大輔|平川大輔]]}}
:黑暗業界的醫生。美容外科診所的院長。擁有高超的外科技術，擅長屍體的處理、籌備工作的黑暗清掃人。
:棒球位置：三壘手。

; {{nihongo|重松|重松|声：[[濱田賢二|濱田賢二]]}}
:刑警。與馬場等人有所來往，對於黑暗社會很熟悉的老練刑警。也經常處理法律無法制裁的事件。
:棒球位置：捕手。

; {{nihongo|剛田源造|剛田 源造|声：[[廣田行生|廣田行生]]}}
:馬場經常光顧位於中州的拉麵店的店主，也擔任殺手仲介。曾經為傳說中的殺手“G.G”，現已退休。
:某次事件重出江湖，手持雙槍打前線拯救馬場與林憲明。
:球隊教練。

=== 其他角色 ===
; {{nihongo|猿渡俊助|猿渡 俊助|声：[[中村悠一|中村悠一]]}}
:白頭髮，穿著寬鬆的褲子。
:北九州的危險殺手。殺人承包公司“Murder Inc.”的前王牌。渴望有個強大的對手而辭職，後與新田重逢，使用苦無和手裡劍，以“潛水艇忍者”的稱呼活動。對馬場有敵意。
:與馬場對戰時平分秋色，因差點摔下大樓被馬場救上來而中斷打鬥，但堅持是平手並非戰敗，也與緋狼有過簡短的打鬥，也是五五開但又摔下港口落海中斷打鬥，但因為是個旱鴨子差點溺斃，被馬場救起甚至還被嘲笑。
:個性有點大男孩，不服輸，喜歡爭輸贏。
:高中時曾擔任棒球投手。

; {{nihongo|新田巨也|新田 巨也|声：[[松風雅也|松風雅也]]}}
:殺手顧問。高中時和猿渡是棒球投捕組合。
:雖說過殺手只是工具，但在緊要時刻選擇的還是俊助。
; {{nihongo|小百合|小百合|声：[[大原沙耶香|大原沙耶香]]}}
:與新田合作的殺手，擅於毒殺。馬場的前女友，受其委託暗殺華九會王會長。

; {{nihongo|緋狼|緋狼|声：[[島崎信長|島崎信長]]}}
:林的前搭檔。來自中國的殺手，有吹口哨的習慣。
:曾經和林在同一個殺手訓練場，並與林結下深厚的感情，但在最終考試中背叛林，但反被林用小刀刺傷左眼，之後到博多尋找林，將住在博多叫林憲明的人全殺害並針對左眼惡意刺傷，找到林後說服林回到他身邊再一起作戰，但林以喜歡博多為理由果斷拒絕，拒絕後緋狼宣稱要將林最喜歡的一切毀掉，但最後被林親手所殺。
