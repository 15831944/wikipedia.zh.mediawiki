'''勒阿弗尔'''（{{lang-fr|Le Havre}}，{{IPA-fr|lə ɑvʁ|pron|Fr-Paris--Le Havre.ogg}}）是[[法国|法国]]北部[[诺曼底大区|诺曼底]]地区继[[鲁昂|鲁昂]]之后的第二大城市，位于[[塞纳河|塞纳河]]河口，濒临[[英吉利海峡|英吉利海峡]]，以其作为“[[巴黎|巴黎]]外港”的重要的航运地位而著称，在法国经济中具有独特的地位。勒阿弗尔是法国[[海岸線|海岸線]]上横渡[[大西洋|大西洋]]航线的远洋船舶到欧洲的第一个挂靠港，也是离开欧洲前的最后经停港。勒阿弗尔是法国第二大输出港（仅次于[[马赛|马赛]]），[[集装箱|集装箱]]货运量则居法国第一位。

勒阿弗尔除了港口以外，还拥有藏品丰富的博物馆和一所大学。该市在体育方面，拥有法国最古老的职业[[足球俱乐部|足球俱乐部]]——[[勒哈費爾體育會|勒哈費爾體育會]]。2005年，奥古斯特·佩雷主持重建的勒阿弗尔城被列为[[世界遗产|世界文化遗产]]。

== 政区 ==

勒阿弗尔在行政上属于[[上諾曼底|上諾曼底]]（Haute-Normandie）大区的[[滨海塞纳省|滨海塞纳省]]（Seine-Maritime）。勒阿弗尔的现任市长是[[爱德华·菲利普|爱德华·菲利普]]。

== 人口 ==
在2010年，人口为175,497。是[[诺曼底大区|诺曼底]]地区人口最多的城市，全法国排名13位。

== 地理与气候 ==

勒阿弗尔的地理位置非常重要，它濒临[[英吉利海峡|英吉利海峡]]，位于法国四大河流之一[[塞纳河|塞纳河]]入海口的北岸，是首都巴黎的海上门户。勒阿弗尔的邻近城镇有[[蒙堤維利耶|蒙堤維利耶]]（Montivilliers）、{{link-en|哈弗勒尔|Harfleur}}（Harfleur）、[[贡夫勒维尔|贡夫勒维尔]]（Gonfreville）等。塞纳河的对岸则是[[翁弗勒尔|翁弗勒尔]]。地理坐标：{{coord|49.5|N|0.12|E}} 。市区面积46.95 km²。

勒阿弗尔的气候类型属于[[温带海洋性气候|温带海洋性气候]]，终年盛行西和西南风，冬季温和，1月平均气温高于0 °C；夏季凉爽，7月平均气温25 °C。常年湿润多雨有雨，全年平均降雨量约800毫米。

== 历史 ==

该市创建于1517年，当时称为''Franciscopolis''，得名于[[弗朗索瓦一世_(法兰西)|弗朗索瓦一世]]，后来改名为“恩典勒阿弗尔”（Le Havre-de-Grâce），意即“恩典港”，“勒阿弗尔”的意思就是港口或码头。建设勒阿弗尔是为了取代已被淤塞的古代港口翁弗勒尔（Honfleur）和哈弗勒尔（Harfleur）。该市的历史与港口的联系极为密切。在18世纪，勒阿弗尔不仅从事与法国和欧洲的贸易，又增加了与[[西印度群岛|西印度群岛]]的贸易，因此勒阿弗尔开始增长。到19世纪，它发展成一个工业中心。 

在[[诺曼底战役|诺曼底战役]]期间，该市受到严重破坏：5,000人死亡，12,500栋房屋被彻底摧毁，8万人无家可归。战后，市中心由[[奥古斯特·佩雷|奥古斯特·佩雷]]（Auguste Perret）重建，采用现代风格。2005年，奥古斯特·佩雷主持重建的勒阿弗尔城被列为[[世界遗产|世界文化遗产]]。

== 经济 ==

勒阿弗尔的主要工业有造船、机械、石油化工、木材加工、电工器材及食品等。

== 交通 ==
[[File:PortHavre.jpg|170px]]
[[File:Port_Havre_2.jpg|170px]]

勒阿弗尔拥有完善的公路、铁路和航空（Octeville 机场）联系，由[[勒阿弗尔站|勒阿弗尔站]]乘火车2小时到达巴黎（距离216千米），并可通过A131、A29高速公路与巴黎、鲁昂、亚眠等地相连，前往巴黎也需时2小时。机场距港口约7千米，有定期航班飞往巴黎等地。

勒阿弗尔的市内交通主要依靠庞大的公共汽车系统。该市也计划兴建一个电车系统。从总站有通往[[英国|英国]][[朴次茅斯|朴次茅斯]]的渡轮，由LD Lines经营。

勒阿弗尔港是一个深水海港，航道水深14米，可以昼夜不停地接待大型船舶，无需候潮进入。又面临繁忙的英吉利海峡，世界上1/4的货物在此通过。因此长期以来一直是欧洲的重要港口之一，不仅承担法国与南、北美洲之间的货物转运，并且是来往西班牙、葡萄牙、爱尔兰和苏格兰的理想中转港口。按照港口吞吐量（5529万吨，1992年），勒阿弗尔是法国仅次于[[马赛|马赛]]的第二大港口，但按照集装箱货运量（87.5万TEU，1994年），勒阿弗尔则居法国第一位，高于[[马赛|马赛]]、[[鲁昂|鲁昂]]和[[南特|南特]]。勒阿弗尔是法国[[海岸線|海岸線]]上[[横渡大西洋航线|横渡大西洋航线]]的远洋船舶到欧洲的第一个挂靠港，也是离开欧洲前的最后经停港。它是法国第二大输出港。能源（原油和煤）占港口运输总量的65%。按照码头类别，港区主要码头泊位依次是：集装箱码头（CONTAINER，13个）、油码头（OIL QUAY、10个）、杂货码头（GENERLALCARGO，9个）、散货码头（BULK CARGO，3个）、油突堤（OIL PIER，1个）；占有岸线长度较长的则依次是：集装箱码头（3700米）、杂货码头（2205米）、油码头（1582米）、散货码头（654米）和油突堤（400米）。散货码头最大可靠25万载重吨的船舶，油突堤最大可靠55万载重吨的超级油船。码头装卸效率：煤炭每小时卸1200吨；散糖每小时600吨；矿石每小时卸2000吨。

== 文化与体育 ==

勒阿弗尔大学于1986年创办，主校区是在城市中心，靠近火车站和有轨电车站 。校园包括大学图书馆（2006年） ，体育馆，几个食堂和学生中心。在2010-2011年，有6,914名学生注册，其中包括5,071名本科生，1651名硕士研究生和192博士研究生。该大学还包括高等物流研究学院（ ISEL ）的317位工程师 。它的科学与技术学院，国际交流学院，艺术与人文学院开办120个不同专业。该大学拥有17 ％的外国留学生。勒阿弗尔大学也是一个拥有 9个实验室的研究中心。与其他高等教育机构合作密切。
Institut universitaire de technologie du Havre(IUT DU HAVRE)有两个校区： 一个在Caucriauville区，成立于1967年，另一个在EURO区，成立于2011年以来。 IUT共有1881名学生。
其他高等教育部门：
诺曼底管理学校（l'École de Management de Normandie）
巴黎政治研究学院（l'Institut d'études politiques de Paris）
海事高等学校（L'École nationale supérieure maritime ）


由于与[[英国|英国]]隔海相望，勒阿弗尔拥有法国最古老的职业[[足球俱乐部|足球俱乐部]]——[[勒阿弗尔足球俱乐部|勒阿弗尔足球俱乐部]]（成立于1872年）。
