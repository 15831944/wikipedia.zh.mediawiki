《'''黄埔'''》是[[黄埔军校同学会|黄埔军校同学会]]主办、向海内外公开发行的综合性刊物<ref>2007年《黄埔》杂志开始征订，统一论坛2006年06期</ref>。'''《黄埔》杂志社'''由黄埔军校同学会举办<ref>{{Cite web |url = http://www.gjsy.gov.cn/cxzl/ |title = 《黄埔》杂志社 |author =  |date =  |publisher =事业单位在线  |language =  |accessdate =  2018-02-10 |quote =  }}</ref>，是[[全国政协办公厅|全国政协办公厅]]直属[[事业单位|事业单位]]。

==简介==
在[[邓颖超|邓颖超]]、[[徐向前|徐向前]]、[[聂荣臻|聂荣臻]]等人的关心下，《黄埔》杂志于1988年6月创刊。黄埔军校同学会首任会长徐向前提出了“为黄埔同学立言，为祖国统一尽力”的办刊宗旨。该刊物作为黄埔军校同学会的对外宣传窗口，起到了联系海内外黄埔同学及亲友的作用，深入宣传[[中共中央|中共中央]]的对[[台湾|台]]方针政策，报道[[黄埔军校|黄埔军校]]在中国现代史上的地位和作用，宣传黄埔师生在[[国民革命军东征|东征]]、[[北伐|北伐]]和[[抗日战争|抗日战争]]中的功绩，反映[[海峡两岸|海峡两岸]]和海外黄埔同学及亲友在经贸、文化等方面的交流交往情况，系统介绍优秀中华传统文化，在促进[[两岸关系|两岸关系]]和平发展，推进祖国和平统一进程中发挥了独特作用<ref name=eshw>{{Cite web |url =  http://www.chinanews.com/mil/2013/06-14/4928701.shtml|title = 《黄埔》杂志创刊25周年 |author =  |date = 2013-06-14 |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref><ref name=hpzz>{{Cite web |url =http://www.huangpu.org.cn/hpzz/  |title = 黄埔杂志 |author =  |date =  |publisher = 黄埔军校同学会 |language =  |accessdate = 2018-02-10 |quote =  }}</ref>。

《黄埔》杂志自1988年创刊之初成立了编委会，1989年停止活动<ref>晓琼，《黄埔》杂志编委会举办新春联谊会，黄埔2003(1)</ref>。2003年，黄埔军校同学会决定恢复成立《黄埔》杂志编辑委员会，聘请[[李赣驹|李赣驹]]等22人为编辑委员会委员<ref>《黄埔》杂志编辑委员会恢复成立，黄埔2003(1)</ref>。

2008年到2013年，《黄埔》杂志陆续推出“我的军校生活”、“[[中国远征军|中国远征军]]”、“两岸关系展望”、“我心中的黄埔前辈”、“黄埔老人的幸福晚年”、“[[台儿庄大捷|台儿庄大捷]]”等专题策划。2013年，杂志扩增至96页，并实现全彩印刷。《黄埔》杂志还加强了同网络媒体的合作，完善了数字化版本，扩大了发行范围<ref name=eshw/>。
