'''黄永刚'''（{{bd|1962年|11月2日|||catIdx=黄}}），[[美国国家工程院|美国国家工程院]]院士、[[中国科学院外籍院士|中国科学院外籍院士]]，美国[[西北大学_(伊利诺伊州)|西北大学]]机械工程、土木与环境工程、材料科学与工程系Walter P. Murphy教授，力学家。中国科学院院士[[黄克智|黄克智]]之子。与[[高华健|高华健]]、[[卢天健|卢天健]]、[[锁志刚|锁志刚]]一同被称作华人固体力学界“两健两刚”。

==经历==

黄永刚1962年11月出生于北京，1984年毕业于[[北京大学|北京大学]]力学系，之后跟随[[胡海昌|胡海昌]]院士作研究生两年,1986年前往[[哈佛大学|哈佛大学]]留学，1990年获得博士学位，导师为{{tsl|en|John W. Hutchinson|约翰·W·哈钦森}}，之后留在哈佛大学随{{tsl|en|Bernard Budiansky|伯纳德·巴蒂安斯基}}以及{{tsl|en|James R. Rice|詹姆斯·R·莱斯}}做博士后，为期一年。

1991年，黄永刚在[[亚利桑那大学|亚利桑那大学]]担任助理教授。1995年，前往[[密西根理工大學|密西根理工大學]]担任副教授。1998年，前往[[伊利诺伊大学厄巴纳-尚佩恩分校|伊利诺伊大学厄巴纳-尚佩恩分校]]任副教授，2001年升任正教授，2003年，黄永刚开始任[[伊利诺伊大学厄巴纳-尚佩恩分校|伊利诺伊大学厄巴纳-尚佩恩分校]]的Grayce Wicall Gauthier讲席教授，2004年，担任Shao Lee Soo讲席教授。2007年起，在美国西北大学任Joseph Cummings讲席教授。2015年起，担任Walter P. Murphy讲席教授。<ref>{{Cite web |url = http://web.mech.pku.edu.cn/subpage.asp?id=94 |title = 黄永刚 |author =  |date =  |publisher = 北京大学力学与工程科学系 |language = 中文 |accessdate = 2017-09-25 |quote =  |archive-url = https://web.archive.org/web/20171210233242/http://web.mech.pku.edu.cn/subpage.asp?id=94 |archive-date = 2017-12-10 |dead-url = yes }}</ref>

2004年，受聘为国家教育部长江学者奖励计划讲座教授。2014年，黄永刚受聘为[[西安交通大学|西安交通大学]]名誉教授。2015年，受聘为西南交通大学名誉教授<ref>{{Cite web |url = http://news.swjtu.edu.cn/ShowNews-11223-0-1.shtml |title = 美国西北大学黄永刚教授受聘为西南交通大学名誉教授 |author = 李宗文 |date =  |publisher = 西南交通大学新闻中心 |language = 中文 |accessdate = 2017-12-11 |quote = }}</ref>。2016年，受聘为[[南京工业大学|南京工业大学]]名誉教授、[[华中科技大学|华中科技大学]]名誉教授<ref>{{Cite web |url = http://mse.hust.edu.cn/info/1100/4076.htm |title = 我校名誉教授-西北大学黄永刚教授入选美国工程院院士 |author = 黄永安 |date =  |publisher = 华中科技大学机械学院信息化建设办公室 |language = 中文 |accessdate = 2017-12-11 |quote = }}</ref>。2017年，受聘为[[南京航空航天大学|南京航空航天大学]]名誉教授、[[西北工业大学|西北工业大学]]顾问教授。2019年，受聘为[[上海大学|上海大学]]名誉教授。

2010年，当选为{{tsl|en|European Academy of Sciences and Arts|欧洲科学与艺术院}}院士。2017年当选为[[美国国家工程院|美国国家工程院]]院士、{{tsl|en|European Academy of Sciences|欧洲科学院}}院士与[[中国科学院|中国科学院]]外籍院士，成为继[[高华健|高华健]]、[[锁志刚|锁志刚]]之后固体力学领域第三位华人美国国家工程院院士。

==成就==

与其父黄克智相同，黄永刚主要研究领域为固体力学，尤其是材料和电子器件的力学行为，他开创了可延展无机电子器件的研究领域并创建了基于微观机制的应变梯度理论。

在从1998年到2017年近二十年内，黄永刚为世界固体力学领域文章引用率最高的学者。截至2017年，黄永刚为世界固体力学领域文章引用率最高的华人学者。

黄永刚为中国力学学科和柔性电子学科发展做出了重要贡献，截至2017年，黄永刚培养的17名中国博士生和博士后均已回国，另外还与国内院校联合培养了9名博士后和28名博士。同时，黄永刚每年在国内工作三个月，截至2017年，已与国内学者联合发表论文三百余篇。<ref>{{Cite web |url = http://www.casad.cas.cn/aca/315/wjysmd-201711-t20171130_4625320.html |title = 黄永刚（Yonggang Huang） |author =  |date =  |publisher = 中国科学院 |language = 中文 |accessdate = 2017-12-10 |quote = }}</ref>
