'''黄木岗站'''是[[深圳地铁7号线|深圳地铁7号线]]的车站，黄木岗站位于黄木岗立交桥下，沿泥岗西路及華富路東側南北向布置。车站装修以草绿和米白为主色调，以LED奥运环造型灯具为设计特色。<ref>[http://www.szmc.net/page/service/service.html?code=2103&site=0601 地铁7号线站点介绍]</ref><ref>{{Cite web |url=http://www.sznews.com/news/content/2016-09/15/content_13862745_3.htm |title=7号线站点 深圳新闻网 |access-date=2016-11-01 |archive-url=https://web.archive.org/web/20161028153850/http://www.sznews.com/news/content/2016-09/15/content_13862745_3.htm |archive-date=2016-10-28 |dead-url=yes }}</ref>

== 车站结构 ==
本站設有兩個獨立的收費區，均位於B1層，中間由車站機電設施相隔，兩者由車站通道連接。

==車站設施及藝術牆==
本站設有洗手間，位於B出口旁。

本站商店有佳田麵包、[[7-11便利店|7-11便利店]]，另設有自助售賣機。

本站所在的黃木崗，曾出現安置區，後在2004年拆除<ref>新浪網 - [http://news.sina.com.cn/o/2004-05-26/10102633187s.shtml 深圳黄木岗安置区后天拆除(图)]</ref>，藝術牆根據這段歷史創作，模擬深圳拆除城中村進行城市改造。

==出口==
黄木岗站为地下车站，共设有4个出入口。4個出入口都設有一對扶手電梯，是深圳地鐵站中少有出入口全部提供上下行電梯的地鐵站。

== 未來發展 ==
[[深圳地鐵14號綫|深圳地鐵14號綫]]將在黃木崗設站。為騰出空間興建車站及改善周邊景觀，當局在2019年2月春節期間拆除黃木崗立交。
