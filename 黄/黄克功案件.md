《'''黄克功案件'''》（{{lang-en|'''A Murder Beside Yanhe River'''}}），中国大陆电影，导演王放放，主演成泰燊、王凯、毛孩、黄海冰、戴江、马薇薇、杨佳音。青年[[胡耀邦|胡耀邦]]的电影形象首次出现在荧屏。<ref>{{cite web |url=http://news.163.com/14/1201/23/ACDQJC580001124J.html |title=红军将领黄克功延安杀女学生案搬上银幕(图) |accessdate=2014 |publisher= 网易|work= |language=zh-hans}}</ref>

==剧情==
1937年10月，[[抗日战争|抗日战争]]刚刚开始，[[陕西省|陕西省]][[延安|延安]][[抗日军政大学|抗日军政大学]]第六队队长“[[黄克功|黄克功]]”逼婚并且枪杀16岁女青年“[[刘茜|刘茜]]”。黄克功自恃自己是红军团长战斗功臣，恃宠而骄，目无法纪。此案由[[陕甘宁边区|陕甘宁边区]]高等法院审批，由于院长因事不在，审判长[[雷经天|雷经天]]负责主审。雷经天也是一位久经考验战功卓绝的战士，他参加过[[南昌起义|南昌起义]]、[[广州起义|广州起义]]、[[百色起义|百色起义]]和[[长征|长征]]。当时，中共中央领导人[[毛泽东|毛泽东]]（妻子[[贺子珍|贺子珍]]）、[[张闻天|张闻天]]、[[胡耀邦|胡耀邦]]都对该案表现出重视。国际舆论和学生要求枪毙黄克功，而老红军则请求轻判他，雷经天苦思出方法，决定“公开审理，民主判决”。2000多人在陕北公学的操场上参看审理，胡耀邦作为代表公诉人发言，先后有12位群众代表发言，黄克功当庭辩论，最后审判庭五人展开了激烈争辩。黄克功在最后仍然抱着生的希望，认为毛泽东会赦免自己的死罪。不久毛泽东的复信在大会上宣读，毛泽东赞成处黄克功以极刑，{{Link-ja|揮淚斬馬謖|泣いて馬謖を斬る}}。

==上映==
2014年10月23日，该片在首届“丝绸之路电影节”上举行了首映新闻发布会，导演和主演都悉数参加。

2014年12月1日下午，《黄克功案件》在[[北京市|北京市]][[人民大会堂|人民大会堂]]举办首映式。<ref>{{cite web |url=http://ent.163.com/photoview/00B50003/546710.html#ACDPLO2700B50003&p=ACDPLO2700B50003 |title=法治电影《黄克功案件》人民大会堂首映 |accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref>该片被誉为“首个国家宪法日献礼电影”，将会在12月4日“[[国家宪法日|国家宪法日]]”全国上映。<ref>{{cite web |url=http://ent.163.com/photoview/00B50003/544559.html#p=AAELI42200B50003 |title=《黄克功案件》定档12月4日 |accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref>

==评价==
编剧王兴东称赞毛孩出演的胡耀邦说:“胡耀邦当年22岁,是非常有政治经验的领导干部。我在剧本中刻意描写他行动敏捷、雷厉风行的个性，一身正气，又不失人性，毛孩演得生动感人。”<ref>{{cite web |url=http://news.163.com/14/1024/00/A99I1GH600014Q4P.html |title=《黄克功案件》打造“红色商业电影”新模式 |accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref>
