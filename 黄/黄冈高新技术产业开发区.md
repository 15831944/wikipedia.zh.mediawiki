'''黄冈高新技术产业开发区'''是位于[[湖北省|湖北省]][[黄冈市|黄冈市]]的[[国家级高新技术产业开发区|国家级高新技术产业开发区]]<ref name="a1">{{Cite web |url=http://gxq.hg.gov.cn/cols/2/index.html |title=基本概况|accessdate=2018-11-18 |author= |date= |work= |publisher=湖北黄冈高新技术产业开发区管委会网站|language=Zh-hans }}</ref>。地处黄冈市新城区，周边交通便捷：距[[武汉天河国际机场|武汉天河国际机场]]90公里，与正在规划建设的[[鄂州机场|鄂州顺丰机场]]仅一江之隔。距[[京九铁路|京九铁路]][[黄州站|黄州站]]18公里，紧邻[[沪蓉高速|沪蓉高速]]、[[京珠高速|京珠高速]]，建有长江国家二级港，距离[[武汉|武汉]]中心城区60公里。辖区以'''[[黄冈经济开发区|黄冈经济开发区]]'''之名列为[[黄州区|黄州区]]的一个[[类似乡级单位|类似乡级单位]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|42/11/02/470/000}}}}</ref>。

== 简介 ==
1990年6月，经湖北省人民政府批准，黄州科技经济开发区挂牌成立。2000年7月，省政府批准成立黄冈高新技术产业开发区。2006年，省政府规范园区名称为湖北黄冈经济开发区。2015年7月，湖北黄冈经济开发区更名为“湖北黄冈高新技术产业园区”<ref name="a1"/>。

2017年2月，经中国务院批复升格为[[国家级高新技术产业开发区|国家级高新技术产业开发区]]<ref>{{cite web |url=http://hb.qq.com/a/20170216/024513.htm|title=黄冈高新区晋升国家级鄂国字号数量居中部第一 |accessdate=2017-02-16 |publisher=腾讯大楚网}}</ref>。
