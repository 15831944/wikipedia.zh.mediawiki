'''黄晨'''（{{bd|1914年|5月27日|1994年|3月14日}}）女，原籍[[江苏|江苏]][[苏州|苏州]]，生于[[上海|上海]]，[[中华民国|中华民国]]及[[中华人民共和国|中华人民共和国]]电影演员、导演<ref name=shiguang>{{Cite web |url =  http://people.mtime.com/1201973/details.html#biography|title =黄晨  |author =  |date =  |publisher = 时光网 |language =  |accessdate = 2017-11-21 |quote =  }}</ref>。

==生平==
《娜拉》在上海大剧院首次公演时，[[鲁迅|鲁迅]]第一次看到了郑君里的表演并表示赞赏。同日，郑君里在后台第一次认识了黄晨。黄晨当时是上海启秀女子中学学生，喜欢戏剧。两人后来结婚，因为郑君里的关系，黄晨与[[中共|中共]]方面关系也很密切<ref name=fuqin/>。1931年，黄晨毕业于[[上海|上海]]启秀女子中学。1937年参加抗战演剧四队，开始舞台生涯<ref name=shiguang/>。后来黄晨到[[延安|延安]]，路条是由[[周恩来|周恩来]]亲自开的<ref name=fuqin/>。1939年，黄晨在[[延安|延安]][[陕北公学|陕北公学]]学习<ref name=shiguang/>。在延安期间，黄晨曾亲眼见到中共党内斗争的残酷<ref name=fuqin/>。

[[抗日战争|抗日战争]]期间在[[重庆|重庆]]，左翼电影人的文艺活动是在周恩来直接领导下进行。郑君里当时常去[[八路军重庆办事处|八路军重庆办事处]]。那时，郑君里与[[史东山|史东山]]、[[洪深|洪深]]、[[郭沫若|郭沫若]]、[[胡风|胡风]]等人住在一起，周恩来常把他们叫去讲形势，让他们看[[苏联|苏联]]电影。在重庆时郑君里、黄晨夫妇与周恩来的文艺秘书[[张颖_(章文晋夫人)|张颖]]（前[[中华人民共和国外交部|中华人民共和国外交部]]副部长[[章文晋|章文晋]]的夫人）往来密切，那时张颖经常突然敲开郑君里、黄晨夫妇家门，递个包给黄晨说：“黄晨你帮我藏起来。”黄晨也不知是什么东西，就把它藏在米缸里，过几天有人来取<ref name=fuqin/>。

后来黄晨曾经加入[[中国电影制片厂|中国电影制片厂]]任演员。1947年，进入[[昆仑影业公司|昆仑影业公司]]，参演电影《八千里路云和月》、《希望在人间》等。[[中华人民共和国|中华人民共和国]]成立后，任[[上海电影制片厂|上海电影制片厂]]演员，作品有《不夜城》等<ref name=shiguang/>。她还是上海电影制片厂创作室的中共党总支书记<ref name=fuqin>{{Cite web |url = http://ent.sina.com.cn/r/m/2008-03-19/14211953643.shtml |title =《三联生活周刊》：我的父亲郑君里  |author =  |date =2008-03-19  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

郑君里和[[江青|江青]]（原先艺名为蓝苹）1930年代在上海熟识。1936年4月[[赵丹|赵丹]]与[[叶露茜|叶露茜]]、[[唐纳|唐纳]]与蓝苹、[[顾而已|顾而已]]与[[杜小鹃|杜小鹃]]轰动一时的[[六和塔|六和塔]]下的婚礼，就是由郑君里主持、[[沈钧儒|沈钧儒]]证婚。中华人民共和国成立后，郑君里、黄晨夫妇与江青关系曾经很密切。江青的丈夫[[毛泽东|毛泽东]]知道郑君里爱抽烟，有一次还曾对江青说：“你拿几包我的烟给君里！这都是[[美国|美国]]‘茄里克’香烟！”[[文革|文革]]初期的1960年代某日，江青曾来到上海的郑君里、黄晨夫妇家找黄晨，跟黄晨聊了一两个小时，当时江青是来上海了解“革命”情况的<ref name=fuqin/>。

文革爆发不久，郑君里、[[赵丹|赵丹]]等上海文艺界人士便因和江青的密切关系招致大祸。很快有不明身份的人来郑君里家抄家，同时被抄的还有赵丹、[[周信芳|周信芳]]、[[童芷苓|童芷苓]]、[[陈鲤庭|陈鲤庭]]这几家，其中郑君里家被抄得最厉害，抄走两卡车的文字材料。被抄家后郑君里说：“这些人虽然戴的是[[红卫兵|红卫兵]]袖章，但抄的手法非常专业，他们肯定不是红卫兵。”事后证实，来抄家的都是[[中国人民解放军空军第四军|空四军]]的人。郑君里、黄晨夫妇分别被隔离审查，长子在[[南海舰队|南海舰队]]当兵，家里只剩下幼子郑大里和郑大里的姑妈这两人。1969年郑君里在医院病重，专案组派人到家里通知郑大里以及回上海探亲的哥哥去医院探望，当时他们已有一年多没见到郑君里了，黄晨也在接到通知后从被隔离的上海电影制片厂来到医院，一家四口在医院“团聚”。两天后，郑君里因[[肝癌|肝癌]]在[[上海中山医院|上海中山医院]]病逝。在殡仪馆里，郑君里直接躺在烧尸体用的铁板车上，家属送去一身灰色[[中山装|中山装]]，郑君里头底下垫了两块砖头，身上蒙了一块白布，黄晨、两个儿子及儿子的姑妈，四人与郑君里的遗体最后匆匆待了10分钟后，郑君里随即被拉去火化，四位家属还被关照：出去绝对不能说郑君里已经死了<ref name=fuqin/>。

1980年，在[[中华人民共和国最高人民法院特别法庭|中华人民共和国最高人民法院特别法庭]]审判[[林彪、江青反革命集团案|林彪、江青反革命集团案]]时，从上海前往北京的出庭者有黄晨和[[秦桂贞|秦桂贞]]<ref>{{Cite web |url =http://www.china.com.cn/culture/txt/2008-04/08/content_14561311.htm  |title = 上海"保姆"忆江青旧事 害苦前夫唐纳跟章泯同居 |author =  |date = 2008-04-08 |publisher =中国网  |language =  |accessdate =  |quote =  }}</ref>。

1994年3月14日，黄晨在上海逝世<ref name=shiguang/>。
