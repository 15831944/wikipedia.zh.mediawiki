'''黄皮'''（[[學名|學名]]：''{{lang|la|Clausena lansium}}''），（[[英文|英文]]名稱：''{{lang|en|Chinese Wampi、 Wampi,、Punctate Wampee,、Wampee}}''）<ref  name = "浸會大學">[http://libproject.hkbu.edu.hk/was40/detail?record=1&channelid=1288&searchword=%28name_formal_latin_sort%3D%60Clausena+lansium%60%29+&extension=all&sortfield=%2Bname_formal_latin_sort 藥用植物圖像數據庫]香港浸會大學中醫藥學院</ref>，別稱'''黃彈'''、'''黃彈子'''、'''黃段'''、'''黃皮子'''、'''黃皮果'''、'''黃皮葉'''、'''毛黃皮'''、'''果子黃 '''、'''油梅'''及'''油皮'''等<ref name = "中國自然標本館"/><ref  name = "浸會大學"/>，為[[芸香科|芸香科]][[黃皮屬|黃皮屬]]植物<ref  name = "浸會大學"/>，为常绿小乔木，高5至10[[米|米]]。黃皮的[[果實|果實]]中含有多種人體所需的[[微量元素|微量元素]]及豐富的[[氨基酸|氨基酸]]，是[[熱帶|熱帶]]及[[亞熱帶|亞熱帶]]地區一種水果<ref name="當代藥用植物典">{{cite book|title=《當代藥用植物典 4》|author=|series=蕭培根|publisher=香港賽馬會中藥研究院|page=第136頁|isbn=978-988-99226-3-4|url=http://www.hkjcicm.org/cm_database/plants/detail.aspx?herb_id=421|date=2007年8月}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==分佈==
黃皮原產於[[中國|中國]]，現分佈於[[福建|福建]]、[[廣東|廣東]]、[[廣西|廣西]]、[[海南|海南]]、[[貴州|貴州]]南部、[[雲南|雲南]]及[[四川|四川]][[金沙江|金沙江]]河谷以及[[台灣|台灣]]等地區，多長於山坡、荒地及疏林之中，[[美洲|美洲]]、[[亞洲|亞洲]]及[[澳洲|澳洲]]等地也有引種栽培<ref  name = "浸會大學"/><ref name="中國植物志">[http://www.cvh.org.cn/zhiwuzhi/page/43(2)/132c.PDF 中國植物志]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}卷冊43(2) ，第132 -133頁</ref><ref name="海南植物志">[http://www.cvh.org.cn/difangzhi/hainan/page/3/042.pdf 海南植物志]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}卷冊3 ，第42頁</ref><ref name="四川植物志">[http://www.cvh.org.cn/difangzhi/sichuan/page/9/145.pdf 四川植物志]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}卷冊9 ，第145-147頁</ref><ref name="贵州植物志">[http://www.cvh.org.cn/difangzhi/guizhou/page/2/273.pdf 贵州植物志]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}卷冊2，第273頁</ref><ref name="浙江植物志">[http://www.cvh.org.cn/difangzhi/zhejiang/page/3/418.pdf 浙江植物志]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}卷冊3，第418頁</ref><ref name="福建植物志">[http://www.cvh.org.cn/difangzhi/fujian/page/2/368a.pdf 福建植物志]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}卷冊2，第368-369頁</ref> 。模式種採自於[[廣州|廣州]]近郊<ref name = "中國植物志"/>。

==形態特徵==
黃皮是一種常綠[[灌木|灌木]]或小[[喬木|喬木]][[植物|植物]]， 高約4－6米，高可達12米。[[樹皮|樹皮]]淡黃褐色，嫩枝披微柔毛快脫落，幼枝具多數瘤狀突起腺體，突起上集生成簇的短毛，其餘披柔毛。

===葉===
[[葉|葉]]互生，[[奇數|奇數]][[羽狀複葉|羽狀複葉]]，小葉5－13片，互生，頂端1枚最大，其餘的向下逐漸變小；小葉卵狀橢圓形或卵形，薄革質，表面深綠色，背面淡綠色，兩面均披細小的褐色腺點，頂端短漸尖或急尖，基部寬楔形或近圓形，兩側不對稱，常一側偏斜，全緣或呈波浪狀或具淺的圓裂齒，稍向下反卷，兩面無毛，背面油點明顯；中脈於表面平或稍凸起，披疏生成簇的短毛和柔毛，中脈於背面隆起，尤以未張開的小葉背脈上散生明顯凸起的半透明細油點及密披短直毛，長約2.5－14.5厘米，寬約1－6厘米；[[葉柄|葉柄]]連[[葉軸|葉軸]]長約7－12厘米，側生小葉柄長約5毫米，披生成簇的短毛和柔毛。

===花===
花為聚傘狀圓錐[[花序|花序]]，頂生或腋生，多花，密披柔毛；花兩性；花黃白色，芳香；[[花蕾|花蕾]]圓球形，星芒狀，有5條稍凸起的縱脊梭；花[[直徑|直徑]]約5毫米；[[花序軸|花序軸]]長約13厘米，小[[花梗|花梗]]長約1－2毫米；[[花萼|花萼]]裂片5枚，闊卵形，基部稍連合，外密披短柔毛，內密披長柔毛，長不及1毫米；[[花瓣|花瓣]]5枚，長圓形，端鈍，黃白色，外披黃色柔毛及淡褪色腺點，內有時無毛或披黃色疏短毛，長不超過5毫米，開放時反展，背部有隆起的中線；[[雄蕊|雄蕊]]10枚，長短相間，長的與花瓣等長；[[花絲|花絲]]線狀，無毛，基部稍增寬，不呈曲膝狀；[[花藥|花藥]]寬卵形，黃色，背着藥；[[雌蕊|雌蕊]]短於雄蕊；[[子房|子房]]圓球形，柄短小，密披淡黃色的直長毛，4－5室，每室有略呈上下迭生的[[胚珠|胚珠]]，直徑約1毫米；[[花柱|花柱]]柱狀，比子房為短，有時披疏短柔毛；[[柱頭|柱頭]]扁圓呈盤狀，膨大，無毛；花盤細小。

===果===
[[果|果]]為[[漿果|漿果]]，形狀多樣，闊卵形、橢圓形或圓形，成熟時淡黃至暗黃色，表面密披細毛或略披毛，具油點，[[果肉|果肉]]乳白色多透明，2－5室，長約1－3厘米，寬約1－2厘米，內有[[種子|種子]]1－4顆，稀有多至5顆；種子扁長圓形至近圓球形，[[種皮|種皮]]薄膜質，[[子葉|子葉]]深綠色；胚根直生，圓柱形。

==醫藥用途==
{{medical}}
===樹皮===
黄皮以[[樹皮|樹皮]]入藥，味苦，性溫，中藥名為黃皮皮，載於《[[生草藥性備要|生草藥性備要]]》，原文記載為「消風腫，去疳積，散熱癪。煲酒服，通小便，解污穢。核，治疝氣」，具利水消腫、通利尿道等功效<ref name = "生草藥性備要"/>。

===葉===
黄皮以[[葉|葉]]入藥，味苦、辛，性平，[[藥材|藥材]]主產於[[廣西|廣西]]、[[雲南|雲南]]及[[四川|四川]]等地。[[中藥|中藥]]名為黄皮葉，始載於《[[嶺南採藥錄|嶺南採藥錄]]》，具解表散熱、行氣化[[痰|痰]]、解毒、利尿等功效，主治[[咳嗽|咳嗽]]痰喘、溫室發熱、瘧疾、[[流行性腦膜炎|流行性腦膜炎]]、黃腫、[[小便|小便]]不利、[[腹脘|腹脘]]腹疼痛、[[風濕|風濕]]痹痛、熱毒疥癬、蛇蟲咬傷等，現代[[藥理|藥理]]研究表明黄皮葉具益智、保[[肝|肝]]、降[[血糖|血糖]]、降[[血脂|血脂]]、[[抗氧化|抗氧化]]等作用，[[臨床應用|臨床應用]]於風濕骨痛、[[感冒|感冒]]、[[急性腸胃炎|急性腸胃炎]]、狂犬咬傷、[[木薯|木薯]]中毒等治療上，民間則認為黄皮葉具行氣化痰、利[[尿|尿]]、解表散熱、解毒等的功效<ref name = "浸會大學"/><ref name = "當代藥用植物典"/>。

===果===
黄皮以[[果|果]]入藥，味辛、甘、酸，性微溫， [[藥材|藥材]]主產於[[廣西|廣西]]、[[雲南|雲南]]及[[四川|四川]]等地。[[中藥|中藥]]名為黄皮果，始載於《[[本草綱目|本草綱目]]》，具化[[痰|痰]]、理氣、消食等功效，主治痰飲咳喘、胸膈滿痛、食精不化等<ref name = "浸會大學"/><ref name = "當代藥用植物典"/>。

===果核===
黄皮以[[果核|果核]]入藥，味苦，性溫，[[藥材|藥材]]主產於[[廣西|廣西]]、[[雲南|雲南]]及[[四川|四川]]等地。中藥名為黃皮核，始載於《[[嶺南採藥錄|嶺南採藥錄]]》，為[[嶺南|嶺南]]民間[[草藥|草藥]]，具解毒散結、行氣止痛、健胃消腫等功效，主治氣滯脘腹疼痛、[[睪丸|睪丸]]疼痛、[[疝氣|疝氣]]痛、經痛、風濕骨痛、小兒頭瘡、[[蜈蚣|蜈蚣]]咬傷等。本品種載錄於《[[廣東省中藥材標準|廣東省中藥材標準]]》，定為[[中藥|中藥]]黃皮核的原植物[[來源種|來源種]]<ref  name = "浸會大學"/><ref name = "當代藥用植物典"/><ref name = "生草藥性備要"/>。
