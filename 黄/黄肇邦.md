== 舞回路 ==
《舞回路》记录了由一群80后组成的香港街舞组合，他们2005年踏上国际街舞大赛决赛，成为首队踏上世界舞台的中国街舞组合。队员们个性鲜明，对事物敢于批评，对舞蹈有着自己独特的见解和狂热。<ref name=":0">{{Cite web|url=https://www.westkowloon.hk/tc/freespacefest2013/hongboy|title=Hongboy|accessdate=2018-03-15|last=西九文化區管理局|work=www.westkowloon.hk|language=zh-Hant}}</ref>

=== 制作背景 ===
《舞回路》是导演黄肇邦拍摄的第一部纪录片，这同时也是他当时的一份大学功课。在2003年至2005年间，黄肇邦在实习期间认识了香港街舞组合“节拍攻击”。后来，在大学教授的建议下，黄肇邦寻找当年的街舞年轻人，并作为拍摄故事的背景，拍摄了《舞回路》。在拍摄过程中，通过与“节拍攻击”队员们的接触，黄肇邦被他们用舞蹈表达自身的热情和激情所感染。在影片中，他也在不断的挖掘新的元素，并希望能够与观众产生互动与共鸣。<ref name=":0" /><ref name=":1">{{Cite news|url=http://www.dcfever.com/news/readnews.php?id=11870|title=紀錄片「有心」就唔怕悶‧ 導演黃肇邦分享拍攝心得 - DCFever.com|newspaper=DCFever|accessdate=2018-03-15}}</ref>

== 子非鱼 ==
《子非鱼》记录了在油麻地鲜鱼行学校读书的余伟豪家庭和他好友的故事。故事里，这些孩子们有的是“单非”，有的是“双非”，为了接受更好的教育，在香港读书。他们生活条件贫穷，有只能靠综媛维生的单亲家庭，有越南新移民… … 电影从小孩视角出发，用他们稚任的语言讲述他们眼里的贫穷和家庭问题。<ref>{{Cite web|url=http://www.bastillepost.com/hongkong/article/295982-%E6%B2%92%E6%9C%89%E5%8A%87%E6%9C%AC%E7%9A%84%E5%95%8F%E8%99%9F-%E7%B4%80%E9%8C%84%E7%89%87%E5%B0%8E%E6%BC%94%E7%9A%84%E4%B8%8D%E6%AD%B8%E8%B7%AF|title=%E6%B2%92%E6%9C%89%E5%8A%87%E6%9C%AC%E7%9A%84%E5%95%8F%E8%99%9F+%E7%B4%80%E9%8C%84%E7%89%87%E5%B0%8E%E6%BC%94%E7%9A%84%E4%B8%8D%E6%AD%B8%E8%B7%AF+|accessdate=2018-03-15|work=www.bastillepost.com}}</ref>

=== 制作背景 ===
《子非鱼》导演黄肇邦用了三年的时间，拍摄在鲜鱼行学校一个班级的孩子生活。影片用孩子的眼睛看这个世界，在拍摄中除了导演的录影机外，还给孩子们一部手提DV，为了从孩子们的第一角度拍摄日常所见。《子非鱼》拍摄半年期间，黄肇邦不断写计划书申请资金，最终该影片得到专门资助潜质华人纪录片导演的CNEX基金会的支持，获批12万港币。黄肇邦将这笔有限的资金，大部分集中在中后期制作上。而后，《子非鱼》意外获得奥斯卡得奖导演杨紫烨担任监制，国际级剪辑师Mary Stephen为剪辑顾问。<ref name=":1" />

== 伴生 ==
《伴生》记录了一个公众普遍都会面对和思考的问题，面对父母日渐衰老，如何照顾和赡养父母？片中记录了三个家庭各自不同的看法。有人不舍得老父亲离开人世，却又不忍心父亲受病魔折磨，愿他早日解脱；有人因长期照顾患病父母，压力重大，开始质疑人生的意义；有人因母亲病重，重建疏远多年的关系。

=== 制作背景 ===
黄肇邦用了两年的时间制作了《伴生》中的三个故事，拍摄一部与家庭和老人家有关的话题一直是黄肇邦的愿望。剧中的人物，大多是他做义工的时候认识的。在两年的拍摄中，他意外经历了松哥和松哥太太梅婆婆两人离世。原本只想拍摄家庭的黄肇邦，没有想过会拍摄死亡，这一切也让黄肇邦花更多的时间去思考和消化。<ref>{{Cite news|url=https://thestandnews.com/society/%E6%87%B5%E6%87%82%E5%B0%91%E5%B9%B4%E7%9A%84%E7%94%9F%E5%91%BD%E8%AA%B2%E5%A0%82-%E5%B0%88%E8%A8%AA%E7%B4%80%E9%8C%84%E7%89%87-%E4%BC%B4%E7%94%9F-%E5%B0%8E%E6%BC%94%E9%BB%83%E8%82%87%E9%82%A6/|title=懵懂少年的生命課堂 — 專訪紀錄片《伴生》導演黃肇邦 {{!}} 立場報道 {{!}} 立場新聞|newspaper=立場新聞 Stand News|accessdate=2018-03-15|language=en}}</ref>

在被问及拍摄对象是否经过刻意挑选时，黄肇邦表示每个家庭都是独一无二的，被拍摄者一定要是自己喜欢的人，建立一种关系和感应才能够讲述别人的故事。<ref name=":4" />

黄肇邦表示，希望观众看完电影后，能够与父母提前开放的讨论晚年安排，有时候在选择上，子女和老人也许有不一样的想法，提前商量与沟通，可以提供更多更佳的方案，并且达成共识。“人一生都在学习，去到最后一天，可能都不知怎样面对家人。中国人不懂说话，有时伤害了挚亲都不知道。”<ref name=":5">{{Cite web|url=https://hk.lifestyle.appledaily.com/lifestyle/special/daily/article/20170211/19924249|title=被迫送病母住老人院　71歲女風雨不改探望餵早餐|accessdate=2018-03-15|work=Apple Daily 蘋果日報}}</ref>

== 有敬 ==
《有敬》记录了一个行动不便，膝下无子女的104岁老人“敬婆婆”，她信奉基督教，有一位牧师作为监护人照顾，退休后30年都住在养老院。面对孤独漫长的生命旅程，厌世的她祈求早日离开这个世界。影片抛出“长命百岁是否就是福气”的话题。<ref name=":6">{{Cite web|url=https://www.hk01.com/%E8%97%9D%E6%96%87%E5%89%B5%E6%84%8F/76842/%E5%BE%9E-%E4%BC%B4%E7%94%9F-%E5%88%B0-%E6%9C%89%E6%95%AC-%E9%BB%83%E8%82%87%E9%82%A6%E5%BE%9E%E7%99%BE%E6%AD%B2%E8%80%81%E4%BA%BA%E8%BA%AB%E4%B8%8A%E5%AD%B8%E6%87%82%E7%94%9F%E5%91%BD%E5%83%B9%E5%80%BC|title=從《伴生》到《有敬》　黃肇邦從百歲老人身上學懂生命價值|accessdate=2018-03-15|last=鄺曉恩|work=香港01|language=en}}</ref>

=== 制作背景 ===
《有敬》与《伴生》在同时期拍摄。在两年的制作中，共有四个个案，《伴生》讲述了三个故事，而《有敬》中的“敬婆婆”就是第四个，原本黄肇邦想将“敬婆婆”放入《伴生》中，经过思考认为，两部影片的方向不同，《伴生》是讲述照顾者与被照顾者的关系，而《有敬》则是探讨生命价值，于是最终决定将“敬婆婆”的故事独立制作成片。<ref name=":6" />

== 黄肇邦早期个人经历 ==
黄肇邦出身草根家庭，从小住在没有升降机的旧式屋邨，和邻居们共用厕所。这让他从小接触到很多低下阶层，每天看到听到很多看似荒诞，却又实实在在发生在身边的故事，这样的成长环境，训练了黄肇邦的观察力。<ref name=":4" />

发生在童年时期的两件事，让他印象深刻。在小学时期，黄肇邦踢球时，将球踢入了一位独居婆婆的住所，他敲门时，无人应门。几日后，因为白车开到楼下，才得知独居婆婆已经过世，却没有人立刻知道。另一件事是在黄肇邦十几岁时，面对爷爷奶奶相继因病离世，不懂世情的他，对于生和死，表示了自己的“懵懂”与“无知”。他在学生时代，黄肇邦就喜欢影像，在20岁时，想创作一部与家庭、关系、生命有关的作品，但那时还未接触过纪录片。<ref name=":5" />

之后，在大学期间，因一次功课，他开始接触纪录片，并拍摄了第一部作品《舞回路》。<ref name=":3" />
