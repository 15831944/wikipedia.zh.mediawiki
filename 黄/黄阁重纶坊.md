'''“黄阁重纶”坊'''位于中国[[福建省|福建省]][[福清市|福清市]]城区利桥路，距[[瑞云塔|瑞云塔]]仅百余米，1985年被列为[[福建省文物保护单位|福建省文物保护单位]]。

“黄阁重纶”坊建于明[[崇祯|崇祯]]元年(1628年)，是叶长青为[[叶向高|叶向高]]而建，因叶向高曾在[[万历|万历]]、[[天启|天启]]朝两度入阁任[[首辅|首辅]]，故而得名。坊为四柱三间石牌坊，用花岗石砌筑，仿木楼阁式，重檐四坡顶，宽11米，进深3米，通高10.07米。<ref>{{cite web |title=七、经幢、牌坊 |author= |url=http://www.fjsq.gov.cn/ShowText.asp?ToBook=152&index=35& |publisher=福建省情资料库 }}</ref><ref>{{cite web |title=“黄阁重纶”石坊 |author= |url=http://www.fjww.com/news1.asp?id=216&pd=51 |publisher=福建省文物局 }}</ref>
