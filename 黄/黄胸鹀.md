'''黄胸鹀'''（学名：'''''Emberiza aureola'''''），又名'''-{zh-cn:金鵐;zh-tw:黃胸鹀;}-'''、'''黄胆'''、'''禾花雀'''、'''黄肚囊'''、'''黄豆瓣'''、'''麦黄雀'''、'''老铁背'''，是[[鵐屬|鵐屬]]的一种鸟类，在中国遭到大肆捕杀而在2017年列入[[國際自然保護聯盟瀕危物種紅色名錄|IUCN红色目录]]极危物种。

== 生态环境 ==
黄胸鹀喜欢在平原的灌丛、苇丛、农田等低矮植物构成的生境活动，常常结成较大的群，穿梭于农田苇丛之间，群的大小不一，大者逾500只以上，小群则仅有50只左右，在迁徙季节甚至可以看到上万只一起飞过的壮观场面。由于大量的非法捕捉，这种场景已不多见。

== 分布地域 ==
黄胸鹀分布于[[亚洲|亚洲]]东部和[[欧洲|欧洲]]东北部，繁殖区东抵[[朝鲜半岛|朝鲜半岛]]和[[日本列岛|日本列岛]]、千岛群岛一线；越冬区则在南亚和东南亚，几乎覆盖整个印度支那半岛和大部分的南亚次大陆；在[[中国|中国]]，本物种分布于整个中国东部地区，包括东北、华北、华中、华东各省区以及西北的部分省区均可见本物种在迁徙季节过境，亦于越冬季节见于西南和华南各省，其在中国的分布西线可抵[[内蒙古|内蒙古]]西部、[[青海|青海]]、[[甘肃|甘肃]]至[[西藏|西藏]]东部一线，在[[台湾|台湾]]岛和[[海南|海南]]岛等里岛，本物种均为偶见冬[[候鸟|候鸟]]，此外在[[新疆|新疆]]天山山脉附近也有黄胸鹀指名亚种的分布。另据研究，黄胸鹀的繁殖地一般在北纬50度以北而越冬地一般在北纬20度以南，分界比较明显。

== 特征 ==
[[File:Yellow-breasted_bunting_(26233364138).jpg|thumb]]

黄胸鹀体形中等，约15厘米长，全身色彩鲜明，以黄色为主。雄鸟[[换羽|繁殖羽]]头顶、颈及背部栗红色；脸和喉部为黑色；颈下有一宽边黄色环，与黄色的胸部间有一条栗色胸带相隔；背部棕褐色有黑褐色的纵纹；胁部有栗褐色的纵纹；肩上覆羽有明显的白色横斑纹，白斑后有一道较黯淡且细的翅斑；下体鲜黄色；非繁殖期的羽色较黯淡，颊和喉部黄色，[[羽毛|耳羽]]黑色且杂有白斑。

雌鸟及亚成体顶纹沙色，两侧冠纹略深；眉纹皮黄色较明显；背部颜色和纵文教雄鸟的略浅；肩上的白斑和翅斑较雄鸟的灰暗，下体黄色较黯淡。无论雌雄，腰和尾上覆羽都为栗红色；外测两对尾羽外测具楔状斑。飞行时翼上的白色斑块明显可见，配合体色，是辨识的主要特征。上[[喙|喙]]灰色，下喙粉褐色；脚是淡褐色。

== 食物 ==
与其他鹀属鸟类一样黄胸鹀的[[喙|喙]]形适合咬开谷物的壳，是一种以植物性食物为主的鸟类，他们取食的食物主要为稻子、麦子。<ref>{{Cite book|url=https://books.google.com.tw/books?id=8rQrBgAAQBAJ&pg=PA88&lpg=PA88&dq=%E7%A6%BE%E8%8A%B1%E9%9B%80&source=bl&ots=Gb7pGKKiIX&sig=-3QoSv0GZTIwxH2XSRZCagCyQIk&hl=zh-TW&sa=X&ved=0ahUKEwj-tKOLnejSAhUFnpQKHQcrCk84HhDoAQhJMAg#v=onepage&q=%E7%A6%BE%E8%8A%B1%E9%9B%80&f=false|title=辜卡兵的禮物: 翻閱童年|last=冰谷|first=|date=2015-02-01|publisher=獨立作家－釀出版|year=|isbn=9789865696597|location=|pages=|language=zh-hant|chapter=}}</ref>高梁、谷子等农作物，因而曾经被认为是有害农业生产的害鸟遭到捕杀，但是随着人们对自然现象的进一步认识，认为原有的以有益有害区分[[野生动物|野生动物]]的二分法有较大的片面性和局限性，因而现在已经不再强调黄胸鹀所谓害鸟的身份了，另外具野外剖检的结果，黄胸鹀在[[繁殖季节|繁殖季节]]会大量取食各种[[昆虫|昆虫]]，这一习性与其他食-{zh-hans:谷;zh-hant:穀;}-鸟一致。

== 繁殖与保护 ==
[[File:Emberiza_aureola_ornata_MHNT.ZOO.2011.11.219_China.jpg|thumb]] 
黄胸鹀在每年的5－7月间繁殖，营巢与草原或接近草原的林缘地带，巢筑于草丛间，巢呈碗状，以马尾、草根、纤维等柔软材料缠绕而成，每巢产卵4－5枚，孵化期12－13天。

黄胸鹀在2004年被[[国际自然保护联盟|国际自然保护联盟]]列為[[近危|近危]]，在2008年被列入[[易危|易危]]，在2013年11月1日被列入[[濒危|濒危]]物种，2017年12月5日被列入[[極危|極危]]，十四年内连升四级，距[[野外滅絕|野外滅絕]]只剩一步之遙，消失速度之快不容乐观。

在中国，黄胸鹀在鸟类贸易中占有很大份额，中国南方和北方本物种进入贸易市场的形式也有着极大的差异，在中国北方，本物种作为一种宠物进入贸易市场，因其雄性外形优美叫声悦耳而受到欢迎；而在中国南方，本物种作为食品进入市场<ref>{{Cite web|url=http://eastweek.my-magazine.me/main/23254|title=記憶中的禾花雀 - 專欄 - May姐美食情|accessdate=2017-03-21|work=東周網【東周刊官方網站】Eastweek.com.hk|language=zh-hant}}</ref>，由于中医理论认为本物种有滋补强壮的作用，因而在[[广东|广东]]民间人们将中医理论加油添醋，错误地宣传食用以禾花雀为主要原料煲制的汤可以补肾壮阳<ref>{{Cite book|url=https://books.google.com.tw/books?id=6YauAAAAQBAJ&pg=PT35&lpg=PT35&dq=%E7%A6%BE%E8%8A%B1%E9%9B%80&source=bl&ots=Y70lEcCBCr&sig=xXRZ2ttNtOPi1SVQdaye0Ca0uOg&hl=zh-TW&sa=X&ved=0ahUKEwiT2vy1nOjSAhVBvZQKHUCiCKE4FBDoAQgnMAI#v=onepage&q=%E7%A6%BE%E8%8A%B1%E9%9B%80&f=false|title=養生進補湯水|last=關之義|first=|date=2009-01-01|publisher=明窗|year=|isbn=9789628994298|location=|pages=|language=zh-hant|chapter=}}</ref>，极大地提高男性的性能力。<ref>{{Cite book|url=https://books.google.com.tw/books?id=O1QfCgAAQBAJ&pg=PA37&lpg=PA37&dq=%E7%A6%BE%E8%8A%B1%E9%9B%80&source=bl&ots=jJ3rF1Bv42&sig=mudbgMQgR2Ka4fo5xkUCkPjmByk&hl=zh-TW&sa=X&ved=0ahUKEwj4i4ahnujSAhWIjpQKHQ-kDsQ4KBDoAQhQMAk#v=onepage&q=%E7%A6%BE%E8%8A%B1%E9%9B%80&f=false|title=男科‧偏方大全《最新版》|last=老中醫|first=王揚|date=2015-06-20|publisher=丹陽文化|year=|isbn=9789869197274|location=|pages=|language=zh-hant|chapter=}}</ref><ref>{{Cite web|url=http://m.jibingnet.com/jiankang/jksl/46408.html|title=禾花雀肉的功效和吃法_健康食疗_疾病网|accessdate=2017-03-21|work=m.jibingnet.com}}</ref>虽然这一说法没有得到现代药理学的实验支持，但这一传统仍然随着粤菜和广东食文化的传播而广泛传播，由于始终未能实现人工繁殖，其所需的个体均-{zh-hans:系;zh-hant:係;}-野外捕捉。<ref>{{Cite web|url=http://paper.wenweipo.com/2012/01/30/CH1201300001.htm|title=禾花雀悲鳴：瀕危逾十年  仍是盤中餐 -
