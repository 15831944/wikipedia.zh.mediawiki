'''黄桂章'''（{{bd|1962年|3月||}}），[[福建|福建]][[闽清|闽清]]人，[[中华人民共和国铁路|中华人民共和国铁路]]高管。

==生平==
毕业于[[长沙铁道学院|长沙铁道学院]]铁道运输专业，是[[中国共产党|中国共产党]]党员。2008年，当选为[[第十一届全国人大代表|第十一届全国人大代表]]<ref>{{cite news|title=黄桂章代表信息|url=http://www.npc.gov.cn/delegate/dbmd.action?id=a1|accessdate=2013-03-01|newspaper=全国人大代表信息}}</ref><ref name=ling/>。

黄桂章一直在[[长江|长江]]以南的铁路系统工作，先后任福州铁路分局分局长、[[南昌铁路局|南昌铁路局]]局长。2008年9月，黄桂章从[[南昌铁路局|南昌铁路局]]局长调任[[北京铁路局|北京铁路局]]局长，这是他首次到长江以北地区工作<ref name=wuguan>{{Cite web |url = http://news.cntv.cn/china/20111017/100282.shtml  |title =北京铁路局：黄桂章调离与事故无关系 另有任用  |author =  |date =2011-10-27  |publisher =中国网络电视台  |language =  |accessdate =  |quote =  }}</ref>。

2011年10月10日，[[陕西|陕西]][[韩城|韩城]]开往[[北京西站|北京西站]]的1164次列车在[[石景山南站|石景山南站]]发生机车车头脱轨事故，事故未造成人员伤亡，事故发生后北京铁路局工务处长和工务段长随即被撤职。10月11日下午，铁道部召开全路运输安全电视电话会议，铁道部部长[[盛光祖|盛光祖]]强调了安全问题，要求坚决杜绝客车事故，会议将10月10日1164次列车脱轨事故作为典型通报批评，因为该事故，盛光祖给了黄桂章一个处分。10月13日，在铁道部召开的电视电话会议上，铁道部部长[[盛光祖|盛光祖]]宣布免去黄桂章的北京铁路局局长职务，另有任用，由铁道部党组成员、铁路总工会主席[[何玉华|何玉华]]主持北京铁路局全面行政工作，会上盛光祖谈到1164次列车脱轨事故，但未将该事故与黄桂章的任免联系起来。正局级干部任免一般由铁道部政治部主任宣布，由铁道部部长宣布还是第一次<ref name=ling>{{Cite web |url =http://roll.sohu.com/20111016/n322318899.shtml  |title =北京铁路局长被免另有任用 曾因脱轨事故被处分  |author =  |date = 2011-10-16 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。但北京铁路局表示，黄桂章被调离与1164次列车脱轨事故无关，因为在事故发生前，铁道部已就黄桂章的调任征求了北京铁路局的意见<ref name=wuguan/>。

随后黄桂章任[[京沪高速铁路|京沪高速铁路]]股份有限公司副总经理<ref name=ling/>。2015年8月3日，[[中国保监会|中国保监会]]发出《关于黄桂章等任职资格的批复》（保监许可〔2015〕786号），核准黄桂章担任[[中国铁路财产保险自保有限公司|中国铁路财产保险自保有限公司]]董事长任职资格<ref>{{Cite web |url = http://www.circ.gov.cn/web/site0/tab5168/info3970158.htm |title =关于黄桂章等任职资格的批复  |author =  |date = 2015-08-12  |publisher = 中国保险监督管理委员会 |language =  |accessdate =  |quote =  }}</ref>。2016年8月黄桂章调任[[中国铁路总公司|中国铁路总公司]]资本运营和开发部主任<ref>{{Cite web |url =   http://www.chnrailway.com/html/20160826/1415365.shtml |title =总公司直属机关举办“转闯增”第五场报告会  |author =  |date = 2016-08-26 |publisher = 中华铁道网 |language =  |accessdate =  |quote =  }}</ref>。
