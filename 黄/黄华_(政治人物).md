'''黄华'''（{{bd|1913年|1月25日|2010年|11月24日|catIdx=Huang黃华}}），[[河北|河北]][[磁县|磁县]]人，[[中华人民共和国|中國]]外交家，[[燕京大学|燕京大学]]毕业。1936年加入[[中国共产党|中国共产党]]。曾是中共第十、十一、十二屆中央委員，第十三屆[[中共中央顧問委員會|中共中央顧問委員會]]常委。曾任[[中华人民共和国外交部|中华人民共和国外交部]][[中国外交部长列表|部长]]、[[国务院副总理|国务院副总理]]、[[国务委员|国务委员]]、[[全国人大常委会副委员长|全国人大常委会副委员长]]等职。

== 生平 ==
黄华曾任[[燕京大学|燕京大学]][[学生会|学生会]]主席。[[一二九运动|一二九运动]]的[[学生联合会|学生联合会]]总交际。1936年1月，在燕京大学秘密加入[[中国共产党|中国共产党]]。6月，陪同燕京大学教师[[斯诺|斯诺]]赴[[延安|延安]]采访毛泽东。后留在延安，曾担任翻译、秘书等工作。1949年主持接收[[中华民国外交部|中华民国外交部]]，同年5月5日，奉[[中共中央|中共中央]]命与时任[[美国驻华大使|美国驻华大使]][[司徒雷登|司徒雷登]]接触未果<ref>{{cite book |title = Decisive Encounters |publisher= Stanford University Press |author = Odd Arne Westad |page = 306 |date = 2003 |accessdate = 2013-11-24}}</ref>。1953年任[[朝鲜停战|朝鲜停战]]政治谈判中方代表和外交部西欧非洲司司长。1954年，作为中华人民共和国代表团一员参加[[日内瓦会议_(1954年)|日内瓦会议]]。<ref>{{cite web |author1=李亚平 |title=周恩来在日内瓦会议上的外交艺术 |url=http://dangshi.people.com.cn/GB/85038/11921000.html |date=2010年06月21日|website=人民网-中国共产党新闻网 | accessdate=2019-01-11}}</ref>1960年至1971年相继任驻[[加纳|加纳]]、[[埃及|埃及]]、[[加拿大|加拿大]]大使。1971年中华人民共和国获得[[联合国|联合国]]席位后，为首任常驻联合国及其[[聯合國安全理事會|安全理事会]]的代表。1976年12月任[[外交部长|外交部长]]，后兼任[[国务院副总理|国务院副总理]]、[[國務委員|國務委員]]。

黄华曾随[[周恩来|周恩来]]总理出席[[日内瓦会议|日内瓦会议]]和[[万隆会议|万隆会议]]，任中国代表团顾问和发言人。出席过多届[[联合国大会|联合国大会]]。1978年与日本外相[[园田直|园田直]]签署了《[[中日和平友好条约|中华人民共和国和日本国和平友好条约]]》。1978年主持与美国代表的建交谈判，1982年与[[美国国务卿|美国国务卿]][[黑格|黑格]]签署关于解决美售台武器问题的[[八一七公报|八一七公报]]。1979年随[[邓小平|邓小平]]副总理访问美国。

黃華卸任外交部部長後，1983年當選為第六屆[[全国人大常委会副委员长|全国人大常委会副委员长]]，多次以副委員長身份率人大代表團到各國訪問和出席各種國際會議。

2010年11月24日凌晨3時8分在北京逝世，享年97歲。

==评价==
官方稱讚黃華為「中國共產黨的優秀黨員，久經考驗的忠誠的共產主義戰士，無產階級革命家，我國外交戰線的傑出領導人」。

==家人==
*夫人：[[何理良|何理良]]，法学家[[何思敬|何思敬]]的女儿，1925年生，1940年参加革命,1944年经朱德夫人[[康克清|康克清]]介绍与时任朱德政治秘书的黄华在延安结婚，1945年毕业于延安中央军委俄文学校。1958年毕业于莫斯科国立国际关系学院历史系。1960年至1976年先后在驻加纳、埃及和加拿大大使馆以及常驻联合国代表团任外交官至参赞，1977年至1986年任外交部国际司副司长，曾任五、六、七、八届全国政协委员和全国妇联执委。
