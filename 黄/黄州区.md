'''黄州区'''是中国[[湖北省|湖北省]][[黄冈市|黄冈市]][[政府|政府]]驻地，人口34.2万（2004年）。是黄冈市[[政治|政治]]、[[文化|文化]]、[[经济|经济]]中心，鄂东北重要的物资集散地。

== 地理位置 ==
黄州区位于[[湖北省|湖北省]]东部，[[长江|长江]]北岸。

== 气候 ==
亚热带湿润季风气候，年均气温16.3℃，无霜期264天，年均降水量1280毫米。

== 行政区划 ==
黄州区现辖4个街道、3个镇、1个乡。共有33个居委会、89个村委会。另有一个[[类似乡级单位|类似乡级单位]]——[[黄冈经济开发区|黄冈经济开发区]]在其境内。

== 经济 ==
[[种植业|种植业]]以[[粮食|粮食]]、[[养殖|养殖]]、[[蔬菜|蔬菜]]为主。[[第二产业|第二产业]]形成了[[轻工|轻工]]、[[纺织|纺织]]、[[机械|机械]]、[[建材|建材]]、[[医药化工|医药化工]]、[[食品饮料|食品饮料]]等支柱产业。

== 交通 ==
106国道穿过城区，[[京九铁路|京九铁路]]从东北角穿过，有位于王家店的[[黃州站|黄州火车站]]；[[武冈城際鐵路|武冈城際鐵路]]快速與武漢都市圈連接，並設有[[黃岡西站|黃岡西站]]、[[黃岡站|黃岡站]]和[[黃岡東站|黃岡東站]]。黄州港是国家二类开放口岸。[[鄂黄长江大桥|鄂黄长江大桥]]也已经建成通车，交通更加便捷。[[黄冈长江大桥|黄冈长江大桥]]位于长江上的湖北黄冈市黄州区唐家渡上游位置，上距阳逻长江大桥约37公里，下距鄂黄长江大桥约17公里，桥的东岸是黄冈市黄州区唐家渡开发区西、西岸是鄂州市华容区和武汉洪山葛店，集城铁、国铁、高速公路三位为一体，该桥于2010年2月开建，2014年已建成通车。
