'''黃浦江'''（简称'''黄浦'''、'''浦江'''），旧称'''[[黄歇|黄歇]][[浦|浦]]'''（简称'''歇浦'''）、'''[[春申君|春申]]江'''（简称'''申江'''），是[[上海市|上海市]]界内最大河流，被称为「'''上海的母亲河'''」<ref name="中国水利报">{{cite web |url = http://www.envir.gov.cn/info/2001/11/1126286.htm |title = 上海母亲河 |page=第五版 |date = 2001-11-26 |publisher = [[中国水利报|中国水利报]]}}</ref><ref name="解放日报">{{cite web |url = http://news.163.com/09/1215/08/5QIERJA5000120GR.html |title = 苏州河是上海“母亲河”引异议 |author=李君娜 |date = 2009-12-15 |publisher = [[解放日报|解放日报]]}}</ref>。黄浦江将上海分割成了[[浦西|浦西]]和[[浦东|浦东]]，两岸分别形成了举世知名的[[外滩|外滩]]以及[[陆家嘴金融贸易区|陆家嘴金融中心]]等上海重要地标。

==概况==
黄浦江全长约114千米，河宽300-700米，终年不冻，是[[上海|上海]]的重要水道。它始于[[上海市|上海市]][[青浦区|青浦区]][[朱家角镇|朱家角镇]]淀峰的[[淀山湖|淀山湖]]，在[[吴淞口|吴淞口]]注入[[长江|长江]]，是长江入海之前的最后一条[[支流|支流]]。黄浦江上游接纳太湖流出的诸河，是[[太湖|太湖]]向[[东海|东海]]泄水的主要通道，现时78%的太湖入海径流通过黄浦江排入东海，主要有淀山湖、[[太浦河|太浦河]]、[[红旗塘|红旗塘]]、[[上海塘|上海塘]]等四大主要水系。

主要支流包括[[吴淞江|吴淞江]]、[[蕰藻浜|蕰藻浜]]、[[川杨河|川杨河]]、[[淀浦河|淀浦河]]、[[大治河|大治河]]、[[斜塘|斜塘]]、[[园泄泾|园泄泾]]、[[大泖港|大泖港]]。浦东、浦南地区临海，以川杨河、大治河、金汇糖、紫竹塘等河道引黄浦江水灌溉当地土地。

黄浦江市区段水深10米左右，最深处17米左右，曾是上海航运业的核心水道，有著名的[[十六铺|十六铺]]码头。如今其航运地位逐渐被深水海港取代，不过[[巨潮港|巨潮港]]（奉贤西北部）至[[吴淞口|吴淞口]]段航道深度仍维持8.2米以上，[[松浦大桥_(上海)|松浦大桥]]以下常年通行3000吨级海轮，5000吨级海轮可直达[[闵行|闵行]]，2万吨级海轮可抵[[吴泾|吴泾]]，5万吨级海轮可通过[[徐浦大桥|徐浦大桥]]。
==历史==
黄浦江在松江以上也被称为「拦路港」，在下游曾被称为'''黄歇浦'''，'''春申江'''，有说法认为是因为上海曾为[[楚国|楚国]][[春申君|春申君]]黄歇的封地。历史上黄浦江曾是[[吴淞江|吴淞江]]（苏州河）的支流，经过明代「[[黄浦夺淞|黄浦夺淞]]」之后，苏州河反而成为了黄浦江的最大支流。

===黄浦江两岸综合开发===
黄浦江两岸综合开发规划控制范围从[[吴淞口|吴淞口]]至[[徐浦大桥|徐浦大桥]]，两侧岸线长度约85公里，规划控制面积约74平方公里。规划控制范围内设核心区和协调区，其中核心区分为中心段、南延伸段和北延伸段，涉及[[浦东|浦东]]、[[宝山区_(上海市)|宝山]]、[[杨浦|杨浦]]、[[虹口|虹口]]、[[黄浦|黄浦]]和[[徐汇|徐汇]]等6个行政区。<ref name="浦江开发">{{cite web|url=http://www.hpjbanks.com/plan/forword.asp|title=浦江规划前言|author=|date=|publisher=上海市黄浦江两岸开发工作领导小组办公室网站}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2018年1月1日，黄浦江两岸从杨浦大桥到徐浦大桥45公里岸线[[公共空间|公共空间]]全线贯通，并开放给全体市民<ref>[http://www.shobserver.com/news/detail?id=75551 黄浦江两岸45公里公共空间今贯通，新一轮三年行动计划启动].上观.</ref><ref>[http://www.yicai.com/image/5388130.html 还江于民 黄浦江两岸45公里公共空间全部贯通].第1财经.</ref>。

===黄浦江上游情况===
淀山湖是太湖东部湖荡区主要泄水通道，内河航运为苏申外港线，维持航道深度3.2米，1000吨级内河船舶常年可由上海抵[[苏州|苏州]]。

太浦河为1949年后开挖的太湖直达黄浦江的重要排涝水道，水深可达5米以上，1000吨级内河船舶由上海至[[湖州|湖州]]。

红旗塘—园泄泾为1949年后开挖的吸收杭嘉湖地区来水排入黄浦江的重要水道，水深2.7米以上。1991年太湖洪水期间曾炸开青浦的一段大坝分洪。内河航运为杭申线、湖嘉申线，现建设三级航道，维持深度3.2米，1000吨级内河船舶由上海至湖州及杭州。

上海塘曾是古代太湖向大海泄水的东江水道，现时为杭嘉湖地区向下游黄浦江泄水的主要水道，水深在5米以上，内河航运为杭平申线，500吨级内河船舶常年由上海经平湖、海盐、海宁抵杭州。

吴淞江曾是太湖向大海泄水的主要通道。近代元明以后吴淞江下游淤积严重，吴淞江上游水流被分流至太湖东部湖荡区，并由淀山湖排入黄浦江。1949年后打通浏河、[[蕰藻浜|蕰藻浜]]、大盈港等河道，使吴淞江中下游水流也被分走。黄浦江为太湖径流出海的主要通道，上海市在(安吉县)龙王山设立黄浦江源头碑。

==越江设施==
=== 黄浦江上的桥 ===
黄浦江水深港阔，是上海的主要通海、通江、内河航运干线，因而建桥不易。清代黄浦江形成“长八渡”，1930年代就有建桥动议，但始终无法实施。1970年代配合上海石化工程，建成第一座黄浦江大桥——松浦大桥。该桥系多孔钢桁梁公铁两用桥，但建桥时没有预见到未来通海航道会向黄浦江上游延伸，桥下净通航高度仅10米。1980年代，南浦大桥开工，系单孔双塔斜拉桥，一跨过江。此后黄浦江上的大桥如雨后春笋般矗立起来。
