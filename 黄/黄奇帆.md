'''黄奇帆'''（{{bd|1952年|5月|catIdx=H}} ），[[浙江|浙江]][[绍兴|绍兴]][[诸暨|诸暨]]人；原[[上海理工大学|上海机械学院]]仪器仪表系毕业，[[中欧国际工商学院|中欧国际工商学院]][[工商管理硕士|工商管理硕士]]，研究员。1976年3月加入[[中国共产党|中国共产党]]。曾任[[中共重庆市委|中共重庆市委]]副书记，[[重庆市人民政府|重庆市人民政府]]市长、党组书记，中共重庆市委企业工委书记。[[中国共产党第十八届中央委员会|中国共产党第十八届中央委员会]]委员、第五届重庆市政协委员。

== 早年经历 ==
在16岁这年，由于受[[文化大革命|文化大革命]]的影响，黄奇帆被迫辍学，到当时的[[上海市焦化厂|上海市焦化厂]]焦炉车间当了一名普通工人。由于工作表现好，1974年他被推荐到原[[上海机械学院|上海机械学院]]（今[[上海理工大学|上海理工大学]]）仪器仪表系自动化仪表专业学习，成为了一名“工农兵学员”。1977年大学毕业后，黄奇帆又回到上海焦化厂，在设备科任技术员。后来，他又升任助理工程师、工程师等职。1983年7月，他被提升为上海市焦化厂副厂长，时年31岁。

== 从政生涯 ==
自1983年12月，黄奇帆进入上海市党政机关工作，并由此步入政界。他曾历任上海市经委综合规划室副主任，市经济咨讯中心主任，上海市第六届青年联合会副主席，浦东开发办公室副主任，浦东新区管委会副主任等职。自1995年起，黄奇帆更是身兼[[中共上海市委|中共上海市委]]副秘书长，和市政府副秘書長的双重职务。其间，他曾兼任市委研究室主任、市体改委副主任、市经委主任、市工业工作党委副书记等职<ref>[http://big5.china.com.cn/zhuanti2005/txt/2004-12/24/content_5737773.htm 黄奇帆简历]</ref>。有媒体认为，在1998年上海本地股重组工作中，黄奇帆曾做出不小的贡献，曾创造性地推出了“净壳”概念。他虽然没有直接领导证券工作的经历，但对资本运作却经验丰富。1999年，他还在[[中欧国际工商学院|中欧国际工商学院]]在职高层管理人员[[工商管理硕士|工商管理硕士]]课程班，获得工商管理硕士学位。

2001年10月，黄奇帆转任[[重庆市|重庆市]]副市长；次年5月，又当选第二届[[中共重庆市委|中共重庆市委]]常委。自2002年10月起，他一直担任中共重庆市委常委，市政府副市长、党组副书记的职务，主要分管工业、教育、财税、金融等方面的工作。

2004年中国股市频创新低，外界有传言，黄奇帆有可能会代替[[尚福林|尚福林]]出任[[中国证监会|中国证监会]]主席的职务。重庆市政府的官员还曾出来辟谣<ref>[http://news.xinhuanet.com/stock/2004-11/24/content_2253108.htm 新华网：重庆官员称副市长黄奇帆出任证监会主席属谣传] {{webarchive|url=https://web.archive.org/web/20090428195931/http://news.xinhuanet.com/stock/2004-11/24/content_2253108.htm |date=2009-04-28 }} 原载《第一财经日报》</ref>。随着时间的推移，这种传言也就不攻自破了。然而，在进入2008年后，中国股市再次大幅下挫，民间对于[[尚福林|尚福林]]即将下台的消息又开始流传，而黄奇帆则一直被视为是最佳继任者。

2009年11月30日，[[中共中央|中共中央]]任命黄奇帆为[[中共重庆市委|中共重庆市委]]副书记，提名黄奇帆为[[重庆市市长|重庆市市长]]候选人，2010年1月當選市長。2013年1月31日，黄奇帆在重庆市四届人大一次会议上，如外界预期再次当选为重庆市人民政府市长。2015年9月，陪同中华人民共和国主席[[习近平|习近平]]访问美国<ref>{{cite news|title=习近平陪同访问正部以上官员超15人 外交部2位|url=http://news.qq.com/a/20150926/009039.htm|accessdate=2015-09-26|work=腾讯|agency=南方都市网}}</ref>。2016年12月30日，重庆市四届人大常委会第三十二次会议经表决决定接受黄奇帆辞去重庆市人民政府市长职务的请求<ref>[http://finance.ifeng.com/a/20161230/15114070_0.shtml 凤凰财经：黄奇帆卸任重庆市市长职务] 原载华龙网</ref>，在其任内重庆的GDP增长速度在全国领先，而且房价得到了有效的控制，据报道黄奇帆的工作得到了党中央与市委书记孙政才的高度评价。2017年2月24日，黄出任[[全国人民代表大会财政经济委员会|全国人大财经委员会]]副主任委员<ref>{{Cite web|url=http://news.21cn.com/hot/cn/a/2017/0224/16/32014870.shtml|title=黄奇帆杨雄等任全国人大财经委员会副主任委员|accessdate=2017-02-24|author=|date=2017-02-24|publisher=21CN.COM|deadurl=yes|archiveurl=https://web.archive.org/web/20170224215423/http://news.21cn.com/hot/cn/a/2017/0224/16/32014870.shtml|archivedate=2017年2月24日|df=}}</ref>。2018年1月当选第五届重庆市政协委员<ref>{{Cite web|url=http://www.zaobao.com.sg/special/report/politic/cnpol/story20180130-831123|title=原市长黄奇帆回调重庆政协 但未担任领导职务|accessdate=2018-01-30|author=|date=2018-01-30|publisher=联合早报}}</ref>。
