'''黄浦江死猪漂流事件'''是指自2013年3月初起，在[[黄浦江|黄浦江]]上海松江段，打捞起数千头高度腐败的死猪的[[水污染|水污染]]事件。<ref name="sina0309" >{{cite web |url=http://news.sina.com.cn/c/2013-03-09/181026481356.shtml|title= 黄浦江上游松江水域发现死猪929头 未影响供水  |accessdate=2013年3月9日  |publisher= 新浪网|work= |language=zh-cn}}</ref>

==新闻动态==

===3月11日之前===
3月初，中国大陆各大网站开始关注此事件，3月9日的新闻称，死猪并未影响到当地供水。<ref name="sina0309"/> 官方也曾一直重申，并未影响到上海居民的饮用水[[水质|水质]]。在3月11日的新闻中，爆出漂浮的死猪带有[[致病原|致病原]]，但不属于人畜共患病。<ref>{{cite web |url=http://news.sina.com.cn/c/2013-03-11/133226496693.shtml|title= 黄浦江死猪检出病毒病原 不属人畜共患病  |accessdate=2013年3月11日  |publisher= 新浪网|work= |language=zh-cn}}</ref>

另有环卫工人称，[[黄浦江|黄浦江]]时常出现死猪尸体，十年间从未间断，每年春夏之交、夏秋之交均是高发期。<ref>{{cite web|url=http://www.news365.com.cn/cq/201303/t20130309_997047.html|title=黄浦江常出现死猪尸体 十多年来一直未断过|accessdate=2013年3月9日|publisher=文新传媒|work=|language=zh-cn}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

===3月12日至3月14日===
最新消息表示，黄浦江中死猪来源自上游[[浙江|浙江]][[嘉兴|嘉兴]]，官方表示猪的死因为[[低体温症|失温致死]]，因为冻死猪太多无法处理，故将其丢入河道。<ref>{{cite web |url=http://news.sina.com.cn/c/2013-03-13/093926517234.shtml|title= 媒体探访浙江嘉兴死猪地:总量大处理池不够用  |accessdate=2013年3月13日  |publisher= 新浪网|work= |language=zh-cn}}</ref> 并有一些养殖户被立案调查。<ref>{{cite web |url=http://www.chinanews.com/gn/2013/03-13/4641216.shtml|title= 浙江嘉兴称今年共死猪7万头 一养殖户被立案  |accessdate=2013年3月13日  |publisher= 中国新闻网|work= |language=zh-cn}}</ref>另据新闻报道，养殖户由于死猪埋葬处理费用太高，所以才会将其抛入黄浦江。<ref>{{cite web |url=http://news.sina.com.cn/c/2013-03-14/025926524140.shtml|title= 民政部刊文释疑死猪殡葬费答问：部长理解无误|accessdate=2013年3月14日  |publisher= 新浪网|work= |language=zh-cn}}</ref>在此事件爆出的几天内，[[中华人民共和国民政部|民政部]]长初次回应时表示，死猪[[火葬|火葬]]费过高是造成此事件的原因，几天后，他又说死猪无需火葬，只需要进行埋葬等[[无害化处理|无害化处理]]。<ref>{{cite web |url=http://www.chinanews.com/gn/2013/03-14/4642750.shtml|title= 李立国回应“人猪穿越答问”：猪不存在火化问题  |accessdate=2013年3月14日  |publisher= 中国新闻网|work= |language=zh-cn}}</ref>经媒体报道几日后，上海河道漂浮死猪仅剩零星几头，恶臭环境改善。<ref>{{cite web |url=http://sh.sina.com.cn/news/g/2013-03-14/160338189.html|title= 上海河道漂浮死猪仅剩零星几头 恶臭环境改善|accessdate=2013年3月14日  |publisher= 新浪网|work= |language=zh-cn}}</ref>

===后续===
*截至15日15时，上海相关区域内共打捞起漂浮死猪8354头。<ref name="xxcb">{{cite web|url=http://www.yangtse.com/system/2013/03/15/016576095.shtml|title=视频：上海死猪事件最新进展 8天打捞死猪8354头|accessdate=2013年3月15日|publisher=扬子江晚报网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160305112339/http://www.yangtse.com/system/2013/03/15/016576095.shtml|archivedate=2016年3月5日}}</ref>

*截至3月18日下午3时，上海市累计打捞的死猪共达到9795头，嘉兴则共收集打捞乱弃的死猪4664头。<ref>{{cite web|url=http://epaper.jinghua.cn/html/2013-03/19/content_1975898.htm|title=黄浦江已打捞死猪近万头|accessdate=2013年3月19日|publisher=京华时报|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20130328062028/http://epaper.jinghua.cn/html/2013-03/19/content_1975898.htm|archivedate=2013年3月28日}}</ref>两地打捞死猪尸体合计已近一万五千具。
*截至3月20日，上海市已累计打捞死猪10395头。<ref>{{cite news|url=http://news.qq.com/a/20130329/000069.htm|title=农业部调查死猪事件半月 上海方面不知进展|date=2013-03-29|publisher=腾讯新闻}}</ref>

=== 禽流感 ===
對於是否與幾乎同時期在[[長三角|長三角]]爆发的[[H7N9|H7N9]][[禽流感|禽流感]]有關連，復旦大學公共衛生學院院長姜慶五表示，從以往全世界爆發的情況上來看，H7N9的病毒主要是在禽類裡面發生，沒有在豬裡面發生過<ref>{{cite web |url=http://china.cnr.cn/xwwgf/201304/t20130402_512281046.shtml |title=世卫组织称H7N9病源或来自猪 专家称怪猪牵强 |accessdate=2013年4月2日  |publisher=中广网 |work= |language=zh-cn}}</ref>，而上海市公共衛生臨床中心黨委書記盧洪洲則在同日表示：[[流行病学|流行病学]]研究證明，[[黃浦江|黃浦江]]打撈的死豬中，致死病毒是[[猪圆环病毒|豬圓環]]，只傳染豬不傳染人，「禽流感病毒再怎麼分裂變異，也不會和豬圓環病毒扯上關係」，但豬隻本身可以成為病毒雜交的宿主<ref>{{cite web |url=http://time-weekly.com/story/2013-04-04/129354.html |title=新禽流感來襲 |accessdate=2013年4月4日  |publisher=[[時代周報|時代周報]] |work= |language=zh-cn}}</ref>。

==官方回应==

===农业部===
[[中华人民共和国农业部|农业部]]表示高度关注，并迅速派出调查组赴浙江、上海实地了解情况，督导、协调处置工作。国家首席兽医师于康震14日带领农业部督导组赴浙江，督促指导地方科学开展处置工作。<ref>{{cite web |url= http://finance.sina.com.cn/nongye/nyhgjj/20130315/071914839891.shtml|title= 农业部派调查组赴浙沪实地督导死猪漂浮事件|accessdate=2013年3月15日  |publisher=新浪网|work= |language=zh-cn}}</ref>
