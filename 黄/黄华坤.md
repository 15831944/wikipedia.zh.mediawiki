==生平==
*1966年生于福建省南安市罗东镇；
*1991年，黄华坤第二次创业，在深圳蔡屋围村建立丽日家私厂<ref name="从几间铁皮厂房到年产值数十亿：左右黄华坤创业三十年风雨">{{cite web | url=http://home.163.com/16/1214/14/C88K0NG6001081H8.html | title=从几间铁皮厂房到年产值数十亿：左右黄华坤创业三十年风雨 |publisher=网易家居|}}</ref>；
*1995年，黄华坤注册"左右"品牌商标<ref name="从几间铁皮厂房到年产值数十亿：左右黄华坤创业三十年风雨">{{cite web | url=http://home.163.com/16/1214/14/C88K0NG6001081H8.html | title=从几间铁皮厂房到年产值数十亿：左右黄华坤创业三十年风雨 |publisher=网易家居|}}</ref><ref name="一个家具匠人用30年打造“幸福企业”">{{cite web | url=http://sz.jiaju.sina.com.cn/news/20161215/6215103961108908810.shtml | title=一个家具匠人用30年打造“幸福企业” |publisher=新浪家居|}}</ref>；

==与[[董明珠|董明珠]]对话==
在2015年举行的"首届中国家居建材品牌发展论坛"上，黄华坤对话格力董事长[[董明珠|董明珠]]。董明珠认为，企业的发展追求是为了整个社会、国家、人民以及自己员工生活得更好。黄华坤则认为，企业存在的不单只是为了赚钱，以及赚了钱如何去花钱，更重要是为社会和人民创造共同的幸福生活<ref name="董明珠和黄华坤约了，他们究竟说了什么？">{{cite web | url=http://jiaju.sina.com.cn/news/20150202/402508.shtml | title=董明珠和黄华坤约了，他们究竟说了什么？ |publisher=新浪家居|}}</ref>。

==出版专著==
2012年，黄华坤与赵龙二人合著的《左右幸福的力量》出版发售，在书中，黄华坤将幸福视角聚焦于传统行业、民营企业以及工人的幸福生活，他认为企业的追求是“打造幸福企业，成就幸福员工，让更多人过上幸福的生活”<ref name="《左右幸福的力量》看企业的力量">{{cite web | url=http://theory.people.com.cn/n/2012/1203/c40538-19776785.html | title=《左右幸福的力量》看企业的力量 |publisher=人民网|}}</ref>。
