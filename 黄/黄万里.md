黄万里（1911年8月20日—2001年8月27日），中国著名[[水利工程|水利工程]]学专家、[[清华大学|清华大学]]教授。近代著名教育家、革命家[[黄炎培|黄炎培]]第三子，早年毕业于唐山交通大学（现[[西南交通大学|西南交通大学]]），后获得美国[[伊利诺伊大学|伊利诺伊大学]]香槟分校工程博士学位，是第一个获得该校工学博士学位的中国人。曾因反对[[黄河三门峡水利工程|黄河三门峡水利工程]]而被错划为右派。黄万里一生主要反对过两项水利工程，一是[[三門峽水利樞紐|三门峡工程]]，另外是[[三峡工程|三峡工程]]。2001年8月27日病逝于北京。

== 生平 ==
黄万里祖籍[[江苏省|江苏省]][[川沙县|川沙县]]，是[[黄炎培|黄炎培]]（父）和王纠思（母）的第三个儿子，1911年出生于[[上海|上海]]，2001年因[[肺癌|肺癌]]逝于[[北京|北京]][[清华园|清华园]]。

1924年黄万里入无锡实业学校学习，1927年进入[[唐山|唐山]]交通大学（现[[西南交通大学|西南交通大学]]），1932年毕业。1933年任[[杭江铁路|杭江铁路]]见习工程师，参与[[江山|江山]]江铁桥的建造。1934年赴美留学，1935年获得美国[[康奈尔大学|康奈尔大学]]硕士学位，1937年获得美国[[伊利诺伊大学香槟分校|伊利诺伊大学香槟分校]]工程博士学位。

1945年在[[南京|南京]]出任[[中华民国|中华民国]]水利部视察工程师。1947年至1949年4月出任[[甘肃省|甘肃省]]水利局局长兼总工程师，[[黄河|黄河]]水利委员会委员。1949年3月，赴[[香港|香港]]。1949年4月至上海，5月辞去兰州职务，经[[广州|广州]]到[[香港|香港]]。1949年9月到[[沈阳|沈阳]]，出任[[中国共产党|中国共产党]][[中国东北|东北]]水利总局顾问。1950年6月回到唐山交通大学任教。1953年被调至[[清华大学|清华大学]]任教。在随后的年间，他编写了重要的学术专著《洪流估算》和《工程水文学》。

1955年4月，[[黄河|黄河]]三门峡大坝工程动工。同月，[[中华人民共和国水利部|中华人民共和国水利部]]召集学者和水利工程师70多人就已开工的[[黄河|黄河]][[三门峡|三门峡]]水利规划方案进行讨论。在当时流传“圣人出，黄河清”的言论下，黄万里是唯一反对建造三门峡水库的与会者，并与其他专家在会上进行了七天的辩论。

1957年6月19日，黄万里在清华大学校刊上发表散文《花丛小语》，被中共中央主席[[毛泽东|毛泽东]]批示“这是什么话？”，并在《[[人民日报|人民日报]]》以“什么话”为标题发表。“什么话”也被作为以后《人民日报》刊登供批判的[[右派|右派]]文章的专栏题目。1969年[[文化大革命|文化大革命]]期间被下放[[江西|江西]][[鄱阳湖|鄱阳湖]]劳动，1974年被揪回清华大学批斗，并在此期间完成《论治理黄河方略》。1980年2月26日被清华大学党委“[[平反冤假错案|平反]]”。1998年，黄万里被清华大学获准给[[研究生|研究生]]授课。

== 学术贡献 ==
黄万里主张从江河及其流域地貌生成的历史和特性出发，全面、整体地把握江河的运动态势；认识和尊重自然规律，把因势利导作为治河策略的指导思想。他的这一理论，在学术界有广泛的影响。

== 社会责任 ==
黄万里一生坚持反对修建[[黄河三门峡水利工程|黄河三门峡水利工程]]及[[长江三峡水利工程|长江三峡水利工程]]，但他的治黄策略及对于三峡工程的意见均未被决策者采纳。在被“平反”以后，他多次向中共中央总书记[[江泽民|江泽民]]和国务院总理[[李鹏|李鹏]]去信，阐述自己的观点，但没有得到中央领导人任何答复；他又向[[中国共产党中央纪律检查委员会|中纪委]]举报[[中华人民共和国国务院|国务院]]对他的申诉置之不理的行为[[违宪|违宪]]，但也没有得到任何答复。

== 家庭 ==
黄万里的妻子丁玉隽是[[中国国民党|中国国民党]]元老[[丁惟汾|丁惟汾]]之女。他们育有三子三女，其中长女黄且圆的丈夫是知名[[数学家|数学家]]、[[中国科学院|中国科学院]]院士[[杨乐|杨乐]]。
