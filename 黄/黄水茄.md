'''黄水茄'''（[[学名|学名]]：{{lang|la|'''''Solanum incanum'''''}}），又名野茄、苦茄，俗稱「黃金茄子」<ref name ="mag-udn-122985"/>、「白絨毛茄」<ref name="herb-nat-tpe-gov-890">[http://www.herb.nat.gov.tw/b2b_cpinfo01.asp?id=890 黃水茄（Solanum incanum Linn. ）]{{dead link|date=十一月 2017 |bot=InternetArchiveBot }} ，[[臺北市政府|臺北市政府]][[臺北市政府工務局|工務局]]大地工程處</ref>，为[[茄科|茄科]][[茄属|茄属]]下的一个种，是食用[[茄子|茄子]]的野生祖先。

== 藥物特性（毒性） ==
{{medical small}}
[[衛生福利部豐原醫院|衛生福利部豐原醫院]]中醫師林煥欽說，黃水茄若吃過量，嚴重會致死。根據醫學文獻記載，黃水茄雖可治療[[肝炎|肝炎]]、[[黃疸|黃疸]]、肝腫脹、[[肝硬化|肝硬化]]、[[水腫|水腫]]等，但不可過量服用，否則會造成口乾、口渴、吞嚥困難、體溫升高、皮膚乾燥發紅、瞳孔擴大、視力模糊等中毒的症狀，重者出現呼吸、循環抑制，甚至[[呼吸|呼吸]]衰竭致死。[[草藥|草藥]]，一定要經過[[醫師|醫師]]依病人病症、體質加減給藥，才不會威脅生命。衛生福利部豐原醫院肝膽腸胃科（內科）何明印主任強調，服用野生黃金茄子後會導致[[鉀|鉀]]離子過高。<ref name ="mag-udn-122985">[http://mag.udn.com/mag/life/storypage.jsp?f_ART_ID=122985 黃水茄當水喝 治肝病反送醫]，[[聯合報|聯合報]]／記者陳秋雲／[[豐原區|豐原]]報導，2008-04-30，{{zh-tw}}</ref>

==產地==
*臺灣，南部平野、[[小琉球|小琉球]]、[[澎湖|澎湖]]。<ref name="herb-nat-tpe-gov-890"/>

== 外觀 ==
全株密佈白色毛，葉廣卵形，先端鈍，基部圓形，葉緣波狀淺裂，葉被灰色毛。花淺紫色，花冠鐘形。漿果球形，熟時黃色。<ref name="herb-nat-tpe-gov-890"/>台灣草藥市場（店）常與[[印度茄|印度茄]]<small>（別名鈕仔茄、柳仔茄、紫花茄、刺天茄等；漿果與「[[破布子|破布子]]」<small>（破布仔）</small>一樣小）</small>混用；功效相當，鈕仔茄的刺很多，而且殘留的果實多而小；黃水茄基本上沒有刺或很少，殘留果實少而大。<ref name="herb-nat-tpe-gov-890"/>

== 性味 ==
味辛苦，性寒，有小毒。入肝經。<ref name="herb-nat-tpe-gov-890"/>
