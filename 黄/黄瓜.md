'''黃瓜'''（[[学名|学名]]：{{lang|la|''Cucumis sativus''}}）也称'''胡瓜'''、'''青瓜'''、'''刺瓜'''、'''瓜仔哖'''，属[[葫芦科|葫芦科]][[甜瓜属|甜瓜属]]植物。广泛分布於中國各地，是中国夏季主要蔬菜，且为温室产品之一。

==名稱==
黄瓜原产印度，[[西汉|西汉]]时期[[张骞|张骞]]出使[[西域|西域]]时将其引入中原，称为「胡瓜」。一說因[[隋炀帝|隋炀帝]]忌讳胡人，将其改为“黄瓜”；<ref>《贞观政要·卷六·慎所好》：隋炀帝性好猜防，专信邪道，大忌胡人，乃至谓胡床为交床，胡瓜为黄瓜，筑长城以避胡。</ref>另一說五胡十六国时后赵皇帝石勒忌讳'''胡'''字，汉臣襄国郡守樊坦将其改为'''黄瓜'''。

== 形态 ==
一年生[[攀缘植物|攀缘]][[草本|草本]]。茎蔓生，卷须不分岐。五角状心形叶子互生，莖和葉都長有細毛。雌雄同株异花，雌花单生或簇生于叶腋，花黄色，雌花數目較少，雄花數目多。[[子房|子房]]深埋在花托裡，所以在雌花下部所結成的細小胡瓜，就是花托和子房聯合變成的果實，其果实圆柱形，通常有刺，刺基常有瘤状突起，所以，[[閩南語|閩南語]]又稱胡瓜為「刺瓜」就是這個原因，此外有的果實有棱或无棱，成熟时墨绿或黄绿色。

== 品种 ==
黄瓜有地黄瓜、架黄瓜、夏黄瓜、秋黄瓜之分。良种有宁阳大刺话瓜、[[北京|北京]]刺瓜等，南方品种大都无刺。

== 烹饪 ==
嫩果（小黃瓜、青瓜）可生吃也可榨汁，入馔常作凉菜。如拍黄瓜、炝黄瓜皮、酸辣黄瓜等。黄瓜也可和[[炸酱面|炸酱面]]同食，或者作汤或热菜。另外黄瓜可切丝作菜码和用作拼盘、菜肴的雕刻和装饰材料。热菜多用于炒、熘等菜的配料，也作酿菜，或用于氽汤。吃入口中脆酥嫩爽。又常用作酸渍、酱浸、腌制菜品原料。

如果在小胡瓜開花結果後，在花朵剛凋萎，尚未完全脫落時，就被採收下來，這種帶枯花的小胡瓜就叫做「花胡瓜」或「花瓜」，常將它製成一般早餐佐[[稀飯|稀飯]]吃的[[醬瓜|醬瓜]]。

黄瓜含有抗炎黄酮醇，如“菲西汀”，以及各种其他抗氧化剂，包括葫芦素。

含水極高，且含[[丙醇二酸|丙醇二酸]]，可抑-{制}-[[糖类|糖类]]转化为[[脂肪|脂肪]]，被视为[[减肥食品|减肥食品]]。嫩籽含[[维生素E|维生素E]]较多，如非必要，可勿去除。中医认为其味甘、性凉，可除热，利尿、解毒。

黄瓜含有一种维生素C分解酶，会破坏其他蔬菜中的维生素C，达不到补充营养的效果。一般认为黄瓜有美肤的作用，常有女性将黄瓜片贴在脸上意图改善面部皮肤。
