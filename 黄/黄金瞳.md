'''黄金瞳'''，2018年中国都市探险电视剧。改编自打眼[[黄金瞳_(小说)|同名小说]]，由[[爱奇艺|爱奇艺]]、灵河传媒、[[腾讯影业|腾讯影业]]出品，爱奇艺龚宇担任出品人<ref>{{Cite web |url=https://www.yidianzixun.com/article/0I7aVoyq?share_count=3&from=groupmessage&isappinstalled=0|title=爱奇艺年度大剧《黄金瞳》开机 张艺兴加盟传奇鉴宝之旅|accessdate=2018-03-05 |author= |date=2018-01-12 |work= |publisher=一点号|language=Zh-hans }}</ref>，铁佛担任总监制，[[南派三叔|南派三叔]]担任总策划，[[林楠|林楠]]担任总导演，[[白一骢|白一骢]]担任总制作人，张鸢盎任编剧<ref>{{Cite web |url=http://ent.qq.com/a/20170917/038831.htm |title=网络小说《黄金瞳》将拍电视剧 张艺兴出演男主角|accessdate=2018-03-05 |author= |date=2017-09-17 |work=leonhaiwang |publisher=腾讯娱乐|language=Zh-hans }}</ref>。由[[张艺兴|张艺兴]]、[[王紫璇|王紫璇]]、[[陳泇文|陳泇文]]及[[王栎鑫|王栎鑫]]領銜主演。2018年1月12日于北京怀柔开机<ref>{{Cite web |url=http://ent.qianlong.com/2018/0112/2324059.shtml|title=《黄金瞳》开机 张艺兴加盟传奇鉴宝之旅|accessdate=2018-03-05 |author= |date=2018-01-12 |work=高骞（QN037） |publisher=千龙网|language=Zh-hans }}</ref>，6月5日全劇殺青。於2019年2月26日在[[爱奇艺|爱奇艺]]首播。

== 剧情简介 ==
原本平凡无奇的典当行职员庄睿在一个午夜突遇劫案，在与劫匪的周旋之中被一块碎玉击伤眼睛，伤愈后却发现自己已然来到了人生的分水岭。被命运选中的庄睿自此走上了一段传奇的冒险之旅。在探究宿命真相的道路上，庄睿持续努力，几乎攀上了人生巅峰，在经历了公盘赌石、丛林冒险、圣彼得堡游历期间，跟髮小皇甫云、结交了彭飞等一干生死之交，与女警苗菲菲、香港珠宝集團千金设计师秦萱冰之间产生了若有似无的情愫。随着一桩四十年前的考古秘闻逐渐被揭开，庄睿等人也遭遇了敌对势力的打压和追击，生死抉择之间，庄睿不忘初心，在弘扬传统文化的同时，坚定信念，不向厄运低头，最终打破宿命的诅咒，收获了至亲挚爱，将暗黑势力一网打尽，真正完成了人生的逆袭，站上了巔峰<ref>{{Cite web |url=http://dsj.sarft.gov.cn/tims/site/views/include/quickSearch/checkBlueprint/result.shanty?ids=015e9f022e4459314028e4a05e503e86|title=电视剧拍摄制作备案公示表|accessdate=2018-06-12 |author=腾讯影业 |date=2017-10 |work= |publisher=|language=Zh-hans }}</ref>。
