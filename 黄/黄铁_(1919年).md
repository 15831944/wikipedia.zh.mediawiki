'''黄铁'''（{{bd|1919年|10月|2014年|11月21日}}），原名'''黄宏世'''<ref>{{cite web |url=http://www.hbdfz.com.cn/detail/index/10236/135392 |title=叙述长诗《阿诗玛》第一作者黄铁 |last= |first= |date=2014-12-18 |website=www.hbdfz.com.cn |publisher=大武汉方志网 |access-date=2018-03-19 |quote=}}</ref>，女，[[武汉|武汉]]人，[[中国共产党|中国共产党]]党员，[[作家|作家]]、[[诗人|诗人]]，[[云南|云南]][[石林县|石林]]荣誉公民，以“《[[阿诗玛|阿诗玛]]》第一作者”著称，曾参与《阿诗玛》的整理。其父是[[中国共产党|中国共产党]]早期领导人[[黄负生|黄负生]]<ref>{{cite web |url=http://www.chinanews.com/cul/2010/07-01/2375277.shtml |title=女作家黄铁为父捐赠 纪念《阿诗玛》作者黄负生 |author=管菁 |date=2010-07-01 |website=www.chinanews.com |publisher=[[长江日报|长江日报]] |access-date=2018-03-19 |quote=}}</ref>，前夫是著名作家[[冯牧|冯牧]]。

==生平==
黄铁1919年出生于[[武汉|武汉]]，1938年开始，先后进入[[延安|延安]][[抗日军政大学|抗日军政大学]]、[[中共中央党校|中共中央党校]]、[[中国女子大学|中国女子大学]]、[[鲁迅艺术学院|鲁迅艺术学院]]学习。中华人民共和国成立后，参与创建了武汉市文联和云南省文联，1952年进入云南工作，任[[云南省文化局|云南省文化局]]局长，曾发掘了著名傣族舞蹈家[[刀美兰|刀美兰]]、著名彝族演员[[杨丽坤|杨丽坤]]。1982年加入[[中国作家协会|中国作家协会]]。1983年调回武汉工作，曾任武汉文联副主席、[[国家民委|国家民委]]文化司副司长兼[[中央民族大学|中央民族大学]]少数民族文学艺术研究所所长等职。2014年11月21日在武汉因肾器官衰竭逝世，享年95岁，葬于[[江夏区|江夏区]][[龙泉山孝恩园公墓|龙泉山孝恩园公墓]]，墓地盖板上刻有“阿诗玛永远陪伴你”字样，左上角刻着电影里的[[阿诗玛_(电影)|阿诗玛]]头像。主要作品有《黄铁文集》、《[[黄负生|黄负生]]纪念文集》、《[[黄钢|黄钢]]文集》等，最著名的成就是倡议、领导、执笔参与了1953年[[云南文工团|云南文工团]]组织的彝族民间叙事长诗《[[阿诗玛|阿诗玛]]》整理（时任[[中共云南省委|云南省委]]宣传部文艺处处长）。<ref>{{cite web |url=http://www.chinanews.com/cul/2014/11-25/6811659.shtml |title=95岁诗人黄铁去世 曾发表叙事长诗《阿诗玛》(图) |author=钟磬如 |date=2014-11-25 |website=www.chinanews.com |publisher=[[武汉晚报|武汉晚报]] |access-date=2018-03-19 |quote=}}</ref><ref>{{cite web |url=http://www.chinanews.com/cul/2014/11-25/6811469.shtml |title=叙述长诗《阿诗玛》第一作者黄铁逝世 享年95岁 |author=蒋太旭 |date=2014-11-25 |website=www.chinanews.com |publisher=[[长江日报|长江日报]] |access-date=2018-03-19 |quote=}}</ref><ref>{{cite web |url=http://news.cjn.cn/gnxw/201412/t2580528.htm |title=北有白毛女南有阿诗玛 两名鲁艺女生的传奇艺术人生 |author=彭莹羿 |date=2014-12-08 |website=news.cjn.cn |publisher=[[长江网|长江网]] |access-date=2018-03-19 |quote=}}</ref><ref>{{cite web |url=http://www.chinawriter.com.cn/zxhy/member/8981.shtml |title=黄铁 (1919～)  |website=www.chinawriter.com.cn |publisher=中国作家网 |access-date=2018-03-19 |quote=}}</ref>
