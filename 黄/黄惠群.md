'''黄惠群'''（{{bd|1930年|||}}<ref>江苏省丹阳高级中学《1949届校友、中央电视台原台长黄惠群》称1937年其7岁，故推知其生年约为1930年。</ref>）女，[[浙江|浙江]][[杭州|杭州]]人，原[[中国中央电视台|中国中央电视台]]台长<ref name=danyang/>。

==生平==
1937年[[抗日战争|抗日战争]]全面爆发，七岁的黄惠群随父母逃难来到[[四川|四川]][[重庆|重庆]]。1944年，黄惠群考入四川[[巴县|巴县]]青木关的[[国立社会教育学院|国立社会教育学院]]附属中学念初中。抗日战争胜利后，国立社会教育学院附属中学迁至江苏[[丹阳市|丹阳]]，黄惠群初中毕业升入高中。1949年5月，黄惠群参军，先后入[[华东军政大学|华东军政大学]]、[[中国人民解放军第三野战军|中国人民解放军第三野战军]]政治部俄文专科学校、上海俄语专科学校学[[俄语|俄语]]。毕业后被分配至[[北京|北京]]的中央广播事业局担任翻译。1959年8月调到[[北京广播学院|北京广播学院]]外语系，历任秘书、组长、副系主任等职务，还曾被调到中国驻[[阿尔巴尼亚|阿尔巴尼亚]]大使馆担任翻译<ref name=danyang/>。

1984年6月，黄惠群调任中央广播电视局干部司副司长。1985年7月，调任[[中国中央电视台|中国中央电视台]]副台长。1988年至1991年任台长<ref name=danyang/><ref name=liren/>。自1984年中央电视台设总编辑职位后，一般均由台长兼任总编辑，但黄惠群未兼任总编辑。另外，黄惠群直至卸任台长时，一直都是司局级干部，未提升<ref name=liren>{{Cite web |url =http://www.guancha.cn/wenzhengming/2015_04_11_315504_s.shtml  |title =从开创春晚到反腐风暴 盘点历任央视台长背后的故事  |author =  |date = 2015-04-11 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>。1992年1月后，黄惠群任中央电视台[[梅地亚中心|梅地亚中心]]董事长、中塔有限责任公司总经理。她还曾兼任中国广播电视学会副会长、[[中华全国妇女联合会|中华全国妇女联合会]]委员、[[中国残疾人福利基金会|中国残疾人福利基金会]]理事等职务<ref name=danyang>{{Cite web |url = http://www.sdz.net.cn/xxgk/ShowArticle.asp?ArticleID=431   |title =1949届校友、中央电视台原台长黄惠群  |author =  |date = 2009-02-13 |publisher = 江苏省丹阳高级中学 |language =  |accessdate =  |quote =  }}</ref>。
