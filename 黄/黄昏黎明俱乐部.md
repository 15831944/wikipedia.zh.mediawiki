'''黄昏黎明俱乐部'''（{{lang|en|Dusk Dawn Club}}，缩写DDC），位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[山老胡同|山老胡同]]14号，是北京市一家知名的小型现场演出场所（Live House）。

==简介==
2014年7月15日，位于山老胡同一座[[四合院|四合院]]内的黄昏黎明俱乐部开业。老板为张锦灿（绰号为“69”），自[[中國科學技術大學|中国科技大学]]毕业后曾在[[厦门|厦门]]、北京经营过三家音乐酒吧，2014年创办黄昏黎明俱乐部。该俱乐部主要承接独立音乐现场演出。2015年，233支境外乐队在该俱乐部演出。2016年全年组织活动383场，演出310场，境内外演出乐队783个，摇滚、布鲁斯、嘻哈、爵士等十多种音乐在此演出。该俱乐部与互联网结合密切<ref>{{Cite web |url =  http://money.163.com/16/0630/08/BQPV49QL00253B0H.html|title = 摇滚四合院丨钛媒体影像《在线》 |author =  |date =2016-06-30  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.163.com/17/0208/17/CCP58J9500018AOQ.html  |title =  张锦灿：做世界独立音乐的忠诚服务者|author =  |date = 2017-02-08 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://mt.sohu.com/20150424/n411812196.shtml |title = 4/28-5/3 五一假期来DDC从黄昏玩到黎明 |author =  |date =2015-04-24  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://ent.163.com/photoview/28NP0003/573611.html#p=B5DR330F28NP0003 |title = 黄昏黎明俱乐部DDC  |author =  |date =  |publisher = 网易 |language =  |accessdate = 2017-02-13 |quote =  }}</ref>。
