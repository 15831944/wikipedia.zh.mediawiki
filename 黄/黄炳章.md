'''黄炳章'''（{{bd|1967年|1月||catIdx=黃炳章}}），[[广东省|广东省]][[惠来县|惠来县]]人，[[天主教汕头教区|天主教汕头教区]]主教（[[中國自選自聖主教列表|自选自圣]]），[[中国天主教爱国会|中国天主教爱国会]]副主席。

==生平==
黄炳章生於1967年1月。1985年高中毕业考入[[天主教中南神哲学院|天主教中南神哲学院]]神哲学专业，获哲学学士学位<ref>{{cite news|title=全国人大代表信息-黄炳章|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=121899|accessdate=2013-04-05|newspaper=中国人大网}}</ref><ref name=zhusheng/>。1991年5月毕业并晋铎（晋升为神父）。同年7月被委派在[[汕头市|汕头市]][[若瑟堂_(汕头)|若瑟堂]]（汕头教区主教座堂）服务至今。1998年当选为[[中国天主教爱国会|中国天主教爱国会]]常委，广东省天主教爱国会副主席、广东省天主教教务委员会秘书长，汕头市天主教爱国会主席，[[第九届全国人大代表|第九届全国人大代表]]，汕头市政协常委。2002年11月补选为广东省天主教爱国会主席。2003年当选为[[第十届全国人大代表|第十届全国人大代表]]。2004年12月当选为广东省天主教爱国会主席。2008年当选为[[第十一届全国人大代表|第十一届全国人大代表]]。2010年12月当选为中国天主教爱国会副主席<ref name=zhusheng>{{Cite web |url = http://roll.sohu.com/20110715/n313501763.shtml |title = 天主教汕头教区举行黄炳章主教祝圣典礼|author =  |date = 2011-07-15 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。2004年在中国天主教第七届代表会第一次“一会一团”负责人会议上，被任命为[[中国天主教主教团|中国天主教主教团]]副秘书长。2010年在中国天主教第八届代表会上，当选中国天主教爱国会副主席。同年，在中国天主教八届第一次“一会一团”负责人联席会议上，担任“公益慈善与社会服务委员会主任”<ref name=jl/>。2011年晋牧前，正在担任中国天主教爱国会副主席，中国天主教公益慈善与社会服务委员会主任，[[中国宗教界和平委员会|中国宗教界和平委员会]]委员，天主教中南神哲学院副董事长，广东省天主教爱国会主席，广东省天主教教务委员会副主席，汕头市天主教爱国会主席，汕头教区“五人领导小组”召集人，汕头教区牧灵培训中心负责人，汕头教区主教座堂主任司铎和汕头市[[圣家堂_(汕头)|圣家堂]]主任司铎，第十一届全国人大代表，汕头市政协常委，汕头市慈善总会副会长等职务<ref name=zhusheng/>。

2011年5月11日，黄炳章被推选为汕头教区候任主教<ref name=zhusheng/>。2011年7月14日晋牧，[[中國自選自聖主教列表|自选自圣]]成为[[天主教汕头教区|天主教汕头教区]]主教<ref name=jl>{{Cite web |url = http://www.chinacatholic.cn/html1/report/1405/668-1.htm |title = 爱国会负责人简历 |author =  |date =  |publisher = 中国天主教 |language =  |accessdate = 2018-02-07 |quote =  }}</ref>。2011年7月14日，黃炳章在未有獲得[[教宗本篤十六世|教宗本篤十六世]]認可的情况下，经自選自聖晉牧，成為经[[中国天主教主教团|中国天主教主教团]]批准的[[天主教汕頭教區|汕頭教區]]正權主教<ref name=tys/>。[[聖座|聖座]]“不承认其身份”，要求其“拒绝接受任命”，但黄炳章坚持履职<ref name=yq>{{Cite web |url = http://www.guancha.cn/YuanQuan/2018_02_07_446115.shtml |title = 看来，中梵关系正常化这事，梵蒂冈这次是真着急了 |author = 原泉 |date =2018-02-07  |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>。同月16日，聖座發出聲明，指黃炳章接受主教聖秩是正處於[[教會法_(天主教會)|教會法典]]第1382條指定的[[自科绝罰|自科绝罰]]當中，並指出[[莊建堅|莊建堅]]已是汕頭教區正權主教，不承認黃炳章為汕頭教區主教<ref name=tys>{{cite web | url=http://china.ucanews.com/2018/01/26/%E6%95%99%E5%BB%B7%E8%A6%81%E6%B1%82%E5%90%88%E6%B3%95%E4%B8%BB%E6%95%99%E8%AE%93%E4%BD%8D%E6%96%BC%E9%9D%9E%E6%B3%95%E4%B8%BB%E6%95%99%EF%BC%8C%E4%B8%BB%E6%95%99%E6%84%9F%E9%9B%A3%E9%81%8E%E8%A2%AB/ | title=教廷要求合法主教讓位於非法主教，主教感難過被出賣 | publisher=[[天主教亞洲通訊社|天主教亞洲通訊社]] | date=2018年1月26日 | accessdate=2018年1月26日 | language=中文}}</ref>。当时庄建坚已81岁，已超过75岁的履职年龄。另外，2011年黃炳章晉牧时，在教廷和地下教会的支持下，中国各地部分天主教徒在[[沈阳|沈阳]]聚集，使本计划主持黄炳章主教祝圣典礼的[[天主教辽宁教区|天主教辽宁教区]]主教[[裴军民|裴军民]]受到压力未能成行<ref name=yq/>。

2013年，当选为[[第十二届全国人大代表|第十二届全国人大代表]]<ref>{{Cite web |url = http://www.npc.gov.cn/wxzl/gongbao/2013-07/24/content_1810912.htm |title = 中华人民共和国第十二届全国人民代表大会代表名单 |author =  |date = 2013-07-24 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>。2016年，在中国天主教第九次全国代表会议上，当选为中国天主教爱国会副主席，公益慈善与社会服务委员会主任。他还继续兼任中国宗教界和平委员会委员，广东省青联副主席，汕头市政协常委，汕头市慈善总会副会长等职务<ref name=jl/>。2018年，当选为[[第十三届全国人大代表|第十三届全国人大代表]]。

2017年，在[[中華人民共和國與梵蒂岡關係|中梵关系正常化]]进程中，聖座在主教任命问题上让步，聖座派出的赴华代表团劝说汕头教区地下主教庄建坚“退休”，把主教席位讓給黃炳章<ref>{{cite web|author=   |title= 外媒称主教认可让步大陆 台湾又忧心梵蒂冈“断交”|url=http://www.guancha.cn/local/2018_01_24_444375.shtml |publisher= 观察者网 |date=2018-01-24 |accessdate= |pages=}}</ref><ref>{{cite web|author=   |title= 梵蒂冈要求合法主教让位于“自选自圣”的非法主教 |url=http://www.asianews.it/news-zh/梵蒂冈要求合法主教让位于自选自圣的非法主教-42896.html |publisher= AsiaNews |date=2018-01-22 |accessdate= |pages=}}</ref><ref name=yq/>。

2018年9月22日，中梵簽署臨時協議，他在协议下获得赦免。<ref>
亚洲新闻，[http://www.asianews.it/news-zh/七位非法主教的绝罚被解除；成立新的承德教区-45012.html 七位非法主教的绝罚被解除；成立新的承德教区]，2018年9月22日。</ref>
