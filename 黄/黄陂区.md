'''黄{{ruby-py|陂|-{zh-cn:pí;zh-tw:ㄆㄧˊ;zh-hk:pei<sup>4</sup>}-}}区'''是[[中华人民共和国|中华人民共和国]][[湖北省|湖北省]][[武汉市|武汉市]]下辖的[[市辖区|市辖区]]。总面积为2261平方公里，人口113.28万（2016年），為武漢市面积最大、人口最多、生态旅游环境最好的新城区，是长江新城（新区）和武汉临空经济区核心区。区政府驻黄陂大道380号，下辖15个街道（这里的街道似镇级单位）、1乡、1场、3个开发区。

黄陂区地处武汉市北部，湖北省东部偏北，黄陂区南邻东西湖区、江岸区，东连洪山区、新洲区，西北交孝感市，东北交黄冈市。
黄陂境内商代盘龙城遗址被誉为武汉城市之根，5A级景区木兰文化生态旅游区被誉为武汉之肺。武汉天河国际机场坐落于此被誉为武汉之窗。

黄陂是一座有4400年筑城史、1800余年建制史的中国历史文化名城，也是楚文化的重要发祥地之一，有“无陂不成镇”的深厚文化积淀。素有“千年古郡、木兰故里、滨江花都、孝信之城”的美誉，拥有盘龙城文化、木兰文化、二程文化、首义文化、四大文化名片。
黄陂是湖北省第一台乡、第二侨乡，华侨华裔和港澳台同胞超过30万人。

== 历史 ==
{{main|黄陂县}}
原为[[黄陂县|黄陂县]]。1933年，部分地区划入新设置的[[礼山县_(中华民国)|礼山县]]，今[[大悟县|大悟县]]。1949年[[中华人民共和国|中华人民共和国]]成立后，归入新设置的[[孝感专区|孝感专区]]。1960年，孝感专区撤销，所属县份划归武汉市。1961年，孝感专区恢复，再属孝感专区。1970年，孝感专区再度撤销，属新置的[[孝感地区|孝感地区]]。1983年起归武汉市管辖，1998年撤县建区。

== 行政区划 ==
黄陂区辖11个街道、3个镇、2个乡：前川街道、六指店街道、长轩岭街道、王家河街道、横店街道、滠口街道、罗汉寺街道、祁家湾街道、武湖街道、天河街道、蔡家榨街道、姚家集镇、李家集镇、三里镇、蔡店乡、木兰乡。其他：木兰山风景管理处、大潭原种场、盘龙城经济发展区。共有43个居委会、611个村委会。[[武汉天河国际机场|武汉天河国际机场]]位于天河街道境内。

== 旅游 ==
=== 自然风景 ===
黄陂境内座落有武汉市最高的山——双峰尖及道教名山——[[木兰山|木兰山]]。相传此山是中国古代传说中的女将军[[花木兰|花木兰]]的家乡。木兰山及其周边地区已经被黄陂区政府规划为旅游发展区。

为纪念北宋理学家程颢、程颐出生地而修建于北宋末年的“双凤亭”,被喻为“二程”理学文化的萌芽地。

=== 人文景观 ===
黄陂区滠口街道有盘龙城遗址，是该地区[[商代|商代]]人类活动留下来的遗迹。同時黄陂区是非物質文化遺產[[僵獅子|僵獅子]]發源地。

在黄陂区鲁台地区，有一座亭子叫做双凤亭，为了纪念中国[[宋|宋]]代[[理学|理学]]家[[程颐|程颐]]程灏而建。

== 方言 ==
黄陂话是黄陂区的独特方言，比起武汉话来说有些许不同，主要是音调不同，但是大体上与武汉话相同，因此两地的人都很容易的互相沟通。 黄陂使用的方言称为[[黄陂话|黄陂话]]，是一种与武汉中心城区差别较大的方言。
武汉中心城区使用的是[[武汉话|武汉话]]，隶属于[[官话|官话]]方言的西南片（与重庆话、成都话、昆明话属于[[西南官话|西南官话]]），具有四个[[声调|声调]]，[[入声|入声]]消失，整片归为[[阳平|阳平]]。与此不同的是，黄陂话属于[[官话|官话]]方言的[[江淮|江淮]]片（与孝感话、黄冈话、扬州话、南京话、南通市区话同属于[[江淮官话|江淮官话]]），具有六个声调，保留了[[入声|入声]]，但入声不分阴阳。<br />
在湖北，黄陂话用来演唱[[湖北大鼓|湖北大鼓]]、[[楚剧|楚剧]]的曲艺形式。
