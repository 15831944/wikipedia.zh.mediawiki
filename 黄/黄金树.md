'''黄金树'''（[[学名|学名]]：{{lang|la|''Catalpa speciosa''}}）为[[紫葳科|紫葳科]][[梓属|梓属]]的植物。

== 形态 ==
落叶[[乔木|乔木]]，高可达30米。单叶对生，叶宽卵形或卵状长圆形，先端渐尖，背面密生柔毛，基出三脉。圆锥花序顶生，夏初开白色两唇形花，下唇内面有黄色条纹和淡紫色斑点。[[蒴果|蒴果]]细长，果壁厚，果期9至10月。 

== 分布 ==
分布在[[台湾|台湾]]、[[美国|美国]]以及[[中国大陆|中国大陆]]的[[广东|广东]]、[[山西|山西]]、[[河南|河南]]、[[河北|河北]]、[[浙江|浙江]]、[[云南|云南]]、[[广西|广西]]、[[山东|山东]]、[[江苏|江苏]]、[[新疆|新疆]]、[[福建|福建]]、[[陕西|陕西]]等地，目前已由人工引种栽培。

== 别名 ==
白花梓树（广西）
