'''黄鶺鸰'''（[[学名|学名]]：{{lang|la|''Motacilla flava''}}）为[[鶺鸰科|鶺鸰科]][[鶺鸰属|鶺鸰属]]的[[鸟类|鸟类]]。分布于古北区、阿拉斯加、非洲、亚洲等地，一般生活于河谷、林缘、原野、池畔及居民点附近以及从平原至海拔4000米以上的高原地区均可见其踪迹。该物种的模式产地在瑞典<ref name='id2630'/>。

== 生态环境 ==
河谷、林缘、原野、池畔及居民点附近，从平原至海拔4000米以上的高原地区均可见其踪迹。

== 分布地域 ==
[[中国|中国]]东南部地区至[[新疆|新疆]]、[[广东|广东]]、[[广西|广西]]、[[福建|福建]]、[[海南|海南]]、[[台湾|台湾]]；古北区、[[阿拉斯加|阿拉斯加]]西部、[[埃及|埃及]]、[[非洲|非洲]]、[[印度|印度]]、[[亚洲|亚洲]]东南部。

== 特征 ==
身体纤细，体长15-16cm，带褐色或橄榄色；虹膜－褐色；嘴－褐色；脚－褐至黑色。 叫声－为重复的叫声间杂颤鸣声；並非直線飛行，而是一上一下地飛行，行走時尾巴會不斷上下擺動。

== 食物 ==
以[[昆蟲|昆蟲]]及種子为食。

== 繁殖 ==
繁殖期3月開始，一次产4-8卵。
