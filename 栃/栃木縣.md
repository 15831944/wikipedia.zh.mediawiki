'''栃木縣'''<ref>普遍認為「{{lang|ja|栃}}」是[[和製漢字|和製漢字]]，故沒有對應中文字。有研究指中文「櫔」與日文「{{lang|ja|栃}}」指同一類樹木，若將「-{萬}-」簡化為「-{万}-」，則兩字形體便幾乎相同，雖然「{{lang|ja|栃}}」右上方是一撇，「櫔」右上方是一橫，但「櫔」在字形和字義上還是較接近「{{lang|ja|栃}}」。不過仍沒有證據證明「{{lang|ja|栃}}」源自「櫔」字。不少人追溯「{{lang|ja|栃}}」的字源時張冠李戴，曲解「{{lang|ja|栃}}」的字形字義，誤以為「{{lang|ja|栃}}」的對應中文字是「槴」、「梔」、「朸」、「枋」、「櫪」，這些都不是正確用法。</ref>（{{jpn|j=栃木県|hg=とちぎけん|rm=Tochigi ken|f=y}}）是位於[[日本|日本]][[關東地方|關東地方]]北部的一個內陸[[縣|縣]]。東面連接[[茨城縣|茨城縣]]、西面為[[群馬縣|群馬縣]]、南面為[[埼玉縣|埼玉縣]]、北面則接鄰[[福島縣|福島縣]]。縣政府所在地為[[宇都宮市|宇都宮市]]。「栃」在日語的意思即是[[日本七叶树|日本七葉樹]]。

栃木縣與[[群馬縣|群馬縣]]的地域在古代為「[[毛野國|毛野國]]（{{lang|ja|けの の くに/け の くに}}）」的一部份。[[仁德天皇|仁德天皇]]時，以地理上的上下分為「{{link-ja|上毛野國|上毛野国}}（{{lang|ja|かみつけの の くに/かみつけ の くに}}）」和「{{link-ja|下毛野國|下毛野国}}（{{lang|ja|しもつけの の くに/しもつけ の くに}}）」。此兩國於[[奈良時代|奈良時代]]改稱「[[上野國|上野國]]（{{lang|ja|かみつけの の くに/かみつけ の くに/かむつけ の くに/かうつけ の くに}}，即現時群馬縣）」、「[[下野國|下野國]]（{{lang|ja|しもつけの の くに/しもつけ の くに}}）」。

== 地理 ==
栃木縣位於關東地方北部，面積約6,408.28[[平方公里|平方公里]]，在日本全國排名第20位，是關東最大的一個縣，2012年總人口 1,994,199人。栃木縣東西距離約84公里、南北距離約98公里。縣都[[宇都宮市|宇都宮市]]距離[[東京|東京]]約100公里、從[[東京站|東京站]]乘搭[[東北新幹線|東北新幹線]]至宇都宮市約需50分鐘、也是北關東最高人口地區。縣內的[[日光市|日光市]]為日本著名的觀光勝地。
