'''館山自動車道'''（{{jpn|j=館山自動車道|hg=たてやまじどうしゃどう|rm=Tateyama jidōshadō|f=y}}），[[日本高速公路|日本高速公路]]（[[高速自動車國道|高速自動車國道]]），自[[千葉縣|千葉縣]][[千葉市|千葉市]][[中央區_(千葉市)|中央區]][[京葉道路|京葉道路]]至[[富津市|富津市]][[富津竹岡交流道|富津竹岡交流道]]。略稱為'''館山道'''（{{lang|ja|館山道／たてやまどう}}，TATEYAMA EXPWY）。雖名為館山自動車道，但本道路與其連接的[[富津館山道路|富津館山道路]]皆未進入[[館山市|館山市]]境內。高速公路路線編號與京葉道路、富津館山道路共編為「'''E14'''」<ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/list/index.html|title=高速道路ナンバリング一覧|accessdate=2017-09-03|publisher=国土交通省}}</ref>。

== 概要 ==
於北側與[[京葉道路|京葉道路]]、南側與[[富津館山道路|富津館山道路]]連接，為連結[[東京都|東京都]]至千葉縣[[南房總市|南房總市]]的[[自動車專用道路|自動車專用道路]]。[[木更津JCT|木更津JCT]]有[[東京灣AQUA-LINE|東京灣AQUA-LINE連絡道]]分岐，可至[[神奈川縣|神奈川縣]]。在2007年3月21日與[[首都圈中央連絡自動車道|圈央道]]連接。

日本國內沒有與其他高速自動車國道（含預定路線區間等）直接連接的路線只有[[沖繩縣|沖繩縣]]的[[沖繩自動車道|沖繩自動車道]]與館山道。

國土開發幹線自動車道建設法法定路線名稱為[[東關東自動車道|東關東自動車道]]館山線，[[高速自動車國道路線指定政令|高速自動車國道路線指定政令]]指定路線名稱為東關東自動車道千葉富津線。

* BS為巴士站、○/●是使用中、◆是停用中，未註記者即未設置。
* 停车区：停車場，SA：服務區，IC：交流道，JCT：系統交流道。

== 歷史 ==
* 1995年4月26日 : 千葉市中央區濱野町（京葉道路）-姉崎袖浦IC間通車。
* 1995年7月18日 : 姉崎袖浦IC-木更津南IC間通車。
* 1996年3月28日 : 木更津JCT通車，並與東京灣AQUA-LINE連絡道連接。
* 2003年4月29日 : 木更津南JCT-君津IC間通車。
* 2005年3月19日 : 富津中央IC-富津竹岡IC間通車。
* 2007年3月21日 : 木更津JCT與圈央道連接。
* 2007年7月4日 : 君津IC-富津中央IC間通車，經過24年'''全線通車'''。
* 2009年3月29日 : 君津停车区 SMART IC使用開始（千葉縣最初的SMART IC）
