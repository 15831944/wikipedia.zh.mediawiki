'''薩摩今和泉站'''（{{jpn|j=薩摩今和泉駅|hg=さつまいまいずみえき|rm=Satsuma-imaizumi eki|f=y}}）是一座位於[[鹿兒島縣|鹿兒島縣]][[指宿市|指宿市]]的[[九州旅客鐵道|九州旅客鐵道]]（JR九州）[[指宿枕崎線|指宿枕崎線]]的鐵路車站。

== 車站構造 ==
*一個1面2線島式月台，可以[[列車交會|列車交會]]。
*[[無人站|無人站]]。
*[[地面車站|地面車站]]。

== 車站周邊 ==
* 指宿市立指宿商業高等學校
* 指宿市役所今和泉分室
* 今和泉郵局

== 歷史 ==
* 1934年（昭和9年）12月19日：車站啟用。
* 1987年（昭和62年）4月1日：因[[國鐵分割民營化|國鐵分割民營化]]，本站成為[[九州旅客鐵道|九州旅客鐵道]]的車站。
