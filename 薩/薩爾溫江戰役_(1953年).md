'''薩爾溫江戰役'''又稱'''血戰薩爾溫江'''、'''猛撒保衛戰'''<ref name="泰緬地區華人孤軍奮戰史績記要">{{Cite web |url = http://www.yunnan.tw/index.php/literature/list-4/yunnanliterature33/413-article3308.html |title = 泰緬地區華人孤軍奮戰史績記要 |accessdate = 2018/07/29  }}</ref>，是1953年[[緬甸聯邦_(1948年－1962年)|緬甸聯邦]]與[[泰緬孤軍|泰緬孤軍]]的戰役，結果孤軍擊潰緬軍。事後緬甸政府向[[聯合國|聯合國]]提出控訴，使[[中華民國政府|中華民國政府]]將部分泰緬孤軍撤回[[臺灣|臺灣]]。

== 背景 ==
1951年[[泰緬孤軍|泰緬孤軍]]反攻[[雲南|雲南]]，一度收復[[滄源|滄源]]、[[鎮康|鎮康]]、[[耿馬|耿馬]]等縣，但[[解放軍|解放軍]]的增援部隊抵達後，考量反攻軍大多未經過訓練，無法與經過訓練的解放軍抗衡，加上雙方兵力懸殊，不得不撤回[[緬甸|緬甸]]。

反攻雲南失敗後，[[李彌|李彌]]記取教訓，重新整頓部隊，1951年10月5日孤軍在[[猛撒|猛撒]]基地成立[[雲南省反共抗俄大學|雲南反共抗俄大學]]，培養反共武力，加入者眾多，孤軍的兵力迅速擴充至三萬以上，勢力遍布[[薩爾溫江|薩爾溫江]]流域，預備[[反攻大陸|反攻大陸]]。<ref name="泰北孤軍—他們的故事"/><ref name="泰緬地區華人孤軍奮戰史績記要"/>

[[中共|中共]]將孤軍的整備視為芒刺在背，於是中共向緬甸政府施壓，要求緬甸驅逐孤軍，並表示願意派兵協助緬軍驅逐孤軍。另外，孤軍進入緬北[[撣邦|撣邦]]後，受到當地少數民族支持，引起緬方不安。孤軍甚至與反緬的[[克倫族|克倫族]]及[[孟族|蒙族]]（又譯為[[孟族|孟族]]）革命軍合作，緬方大為不滿，遂決定進攻孤軍。<ref>{{cite book|author=覃怡輝|title=金三角國軍血淚史：1950-1981|url=https://books.google.com/books?id=1VHBBwAAQBAJ&pg=PA122-123|date=2009-08-28|publisher=聯經出版|isbn=978-986-01-9491-3|pages=122-123}}</ref>
== 戰役經過 ==
1953年3月11日，緬軍在[[東枝|東枝]]、[[景棟|景棟]]、[[大其力|大其力]]等地，調集5,000人的部隊，在中共[[戰車|戰車]]、[[飛機|飛機]]的支援下，攻打位於猛撒北邊的孟布（Mong Pu）等地的[[李國輝|李國輝]]部。另一支2,000人的緬軍部隊，攻打位於猛撒南邊，距離猛撒較近的沙拉（Wan Hsala，又譯為薩拉、溫沙拉）渡口，夾擊猛撒。

在沙拉渡口方面，由於孤軍各部多駐防僻遠，難以在短時間內馳援，只好臨時動員受訓中的反共大學學生，由教育長[[李則芬|李則芬]]中將統一指揮，兵分三路馳援沙拉渡口。21日李則芬帥援軍抵達，與緬軍激戰後大獲全勝，血染薩爾溫江，擊斃緬軍403人，俘虜71人，繳獲野砲1門，小砲21門，81迫擊砲6門，機槍42挺，步槍、卡賓、衝鋒、手槍共542枝，望遠鏡15具，橡皮船21艘。孤軍陣亡官兵41人，傷54人。

在孟布方面，緬軍在陸軍副總司令左蘇將軍指揮下，向駐李國輝部行包圍攻擊，17日馳援部隊抵達，兩面夾擊，最終擊潰緬軍，擊斃緬軍111人，俘虜49人，繳獲81迫擊砲2門，機槍21挺，步槍201枝，衝鋒槍33枝，緬刀300把，彈藥無數。孤軍傷21餘人，反共大學學生隊長孫占奎英勇犧牲。<ref name="泰北孤軍—他們的故事"/><ref name="泰緬地區華人孤軍奮戰史績記要"/><ref>{{cite book|author=覃怡輝|title=金三角國軍血淚史：1950-1981|url=https://books.google.com/books?id=1VHBBwAAQBAJ&pg=PA123|date=2009-08-28|publisher=聯經出版|isbn=978-986-01-9491-3|pages=123}}</ref>
==結果==
{{see also|泰緬孤軍#「雲南反共救國軍」時期#第一次撤退}}
緬軍第二次敗給孤軍，使緬甸政府惱羞成怒，在[[中共|中共]]、[[蘇聯|蘇聯]]支持下，向[[聯合國|聯合國]]控訴中華民國入侵，1953年4月23日聯合國以59比0票（中華民國棄權）通過第707號決議案<ref name="中華救助總會">{{Cite web |url = http://www.cares.org.tw/s_2440_detail.asp?sn=502 |title = 中華救助總會 |accessdate = 2019/07/30  }}</ref>，要求孤軍撤回臺灣。5月[[美國|美國]]、中華民國、[[泰國|泰國]]、緬甸派出代表在泰國[[曼谷|曼谷]]成立四國委員會進行撤軍，計自11月18日開始至1954年3月30日先後由緬甸經泰國[[南梆|南梆]]飛抵臺灣官兵及眷屬共7,288人，為泰緬孤軍第一次撤退。<ref name="泰北孤軍—他們的故事"/><ref name="泰緬地區華人孤軍奮戰史績記要"/>

第一次撤軍時，孤軍總指揮李彌將軍被調回台灣，中華民國政府對國際宣布，未撤台的部隊與中華民國政府已無關係。實際上，中華民國政府下令孤軍「明撤暗留」，將1萬多人的精銳部隊繼續留在緬甸，並指派[[柳元麟|柳元麟]]將軍接替孤軍總指揮，1954年7月柳元麟奉命將孤軍改組為「雲南人民反共志願軍」，以志願軍的名義繼續作戰。<ref name="中華救助總會"/><ref>{{cite book|author=覃怡輝|title=金三角國軍血淚史：1950-1981|url=https://books.google.com/books?id=1VHBBwAAQBAJ&pg=PA7|date=2009-08-28|publisher=聯經出版|isbn=978-986-01-9491-3|pages=7}}</ref>
== 中共的參與 ==
此戰役中參戰的緬軍部隊，每個營安排中共幹部3至5人擔任顧問，中共亦派遣戰車、飛機協助緬軍作戰，並在轟炸孤軍陣地時空投招降宣傳單，勸孤軍向緬軍投降，此外緬軍部隊中夾雜許多共軍。<ref name="金三角國軍血淚史：1950-1981（124）"/>
