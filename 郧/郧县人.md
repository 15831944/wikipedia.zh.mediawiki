== 发现 ==
1989年，[[郧阳|郧阳]]举行第三次[[文物普查|文物普查]]。因为此前有人反应[[郧县|郧县]][[青曲镇|青曲镇]]弥陀寺村村民在汉江边淘金时，曾从沙砾堆中发现过[[化石|化石]]。5月18日，来自郧县博物馆和郧西县文化馆的工作人员来到该村进行现场勘察。当日，在弥陀寺村学堂梁子发现了骨化石。第二次挖掘时将[[头骨|头骨]]的两块枕部找回，并在附近挖掘出100多块[[石器|石器]]。<ref>{{cite news

“郧县人”2号头骨“青曲”是中国目前发现的“[[直立人|直立人]]”阶段人类骨化石中，保存最为完整头骨化石。<ref name=CCTV /> 

== 复原 ==
1990年代，自“郧县人”头骨化石出土后，考古学家一直在寻找复原方法，但由于技术限制，头骨复原工作一直无法开展。因为“郧县人”头骨被挤压变形，脑腔里的软物质已经被坚硬的[[鈣質|鈣質]]胶结物替代，使对头骨化石的复原、观察和测量的难度加大。<ref name=success /> 
直到2003年5月，[[湖北省文物考古研究所|湖北省文物考古研究所]]与[[武汉大学中南医院|武汉大学中南医院]][[计算机断层成像|计算机断层成像]]室进行合作，对郧县人头骨的255个层面用计算机断层成像扫描仪扫描，并使用扫描得到的资料建立[[二维动画|二维动画]]和[[三维动画|三维动画]]，<ref name=CCTV>{{cite web

}}</ref><ref name=success>

== 再次发掘 ==
2003年12月31日，[[南水北调中线工程|南水北调中线工程]]开始施工建设。计划2010年向中国北方供水，到时[[丹江口水库|丹江口水库]]周边方圆300公里的部分区域将被永久淹没，影响涉及学堂梁子遗址所处的[[湖北省|湖北省]][[十堰市|十堰市]][[郧县|郧县]]。虽遗址核心区在坡上，且学堂梁子遗址总计124200平方米，但工程将淹没遗址核心区的外围，也就是坡下部分地区，为防止会对核心区的文物遗址造成影响，2008年2月中旬，湖北省文物局对学堂梁子遗址进行深度发掘，以保护这一重要遗址。<ref>{{cite web|title=湖北将深度发掘“郧县人”遗址|url=http://news.xinhuanet.com/newscenter/2008-01/14/content_7415048.htm|publisher=新华网|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20080223193008/http://news.xinhuanet.com/newscenter/2008-01/14/content_7415048.htm|archivedate=2008-02-23}}</ref>

== 新发现 ==
2012年2月，郧县弥陀寺村青曲中学的一位教师杨正林在学堂梁子遗址上发现一把长约15厘米，宽约10厘米，厚约3厘米的石斧，之后杨正林又在附近发现其他石器。经[[武汉大学|武汉大学]]考古专家初步鉴定，杨正林发现的这些石器，应该就是[[新石器时代|新石器时代]]中晚期人类使用的石器。
