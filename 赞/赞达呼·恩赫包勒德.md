'''赞达呼·恩赫包勒德'''（{{lang-mn|Зандаахүүгийн Энхболд}}，{{bd|1966年|5月23日||}}）[[蒙古族|蒙古族]]，[[蒙古国|蒙古国]][[乌兰巴托|乌兰巴托]]人，[[蒙古国|蒙古国]]政治人物。<ref name=xinhua>[http://news.xinhuanet.com/2012-07/24/c_112518660.htm 蒙古国议会选举赞·恩赫包勒德为新议长，新华网，2012年07月24日] {{webarchive|url=https://web.archive.org/web/20130130060203/http://news.xinhuanet.com/2012-07/24/c_112518660.htm |date=2013年1月30日 }}</ref>

==生平==
1966年5月23日，恩赫包勒德生于蒙古国首都乌兰巴托。1984年毕业于[[色楞格省|色楞格省]]苏赫巴特尔十年制中学。1989年，毕业于俄罗斯乌拉尔综合技术学校自动化机械工程专业。后来，恩赫包勒德在[[蒙古国立大学|蒙古国立大学]][[法学|法学]]专业学习，1996年毕业于蒙古国立大学法学专业。2004年自[[美国|美国]][[丹佛大学|丹佛大学]]获得国际[[工商管理硕士|工商管理硕士]]学位。<ref name=xinhua/><ref name=zxw>[http://www.chinanews.com/gj/2012/07-15/4033326.shtml 蒙古民主党主席阿勒坦呼亚格获蒙古国总理提名，中国新闻网，2012年07月15日]</ref>

1999年至2000年，任[[国家大呼拉尔|国家大呼拉尔]]主席顾问。2000年至2001年，任[[民主党_(蒙古国)|民主党]]总书记<ref>{{Cite web |url = http://politics.people.com.cn/n1/2016/0708/c1001-28537418.html |title = 蒙古国家概况 |author =  |date =  2016-07-08|publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。1996年至1999年、2004年至2005年，恩赫包勒德曾任国有资产委员会主席。2008年当选为[[国家大呼拉尔|国家大呼拉尔]]委员，曾任国家大呼拉尔外交与安全常设委员会主席。2012年再度当选为[[国家大呼拉尔|国家大呼拉尔]]委员。2012年7月24日，恩赫包勒德当选为[[国家大呼拉尔|国家大呼拉尔]]主席。他是由[[民主党_(蒙古国)|民主党]]提名为国家大呼拉尔主席的，由于民主党的提名未同[[蒙古人民党|蒙古人民党]]正式协商，蒙古人民党议会党团全体议员未参加选举国家大呼拉尔主席的会议。<ref name=xinhua/><ref name=zxw/>
