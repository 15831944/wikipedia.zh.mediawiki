'''林穎彤'''（英文：'''Bella Lam'''，{{bd|1992年|10月21日|catIdx=L林}}），原名'''林頴彤'''，[[香港|香港]]女藝人，曾是[[童星|童星]]、[[演員|演員]]、 主持、歌手及[[模特兒|模特兒]]，現為[[香港無綫電視|無綫電視]]基本藝人合約女藝員。

== 簡介 ==
林穎彤來自[[單親家庭|單親家庭]]，自幼在[[葵涌|葵涌]][[葵芳邨|葵芳邨]]長大，一歲時林父因有[[婚外情|婚外情]]拋妻棄子<ref>{{Cite news |url=http://hk.on.cc/hk/bkn/cnt/entertainment/20160805/bkn-20160805140253947-0805_00862_001.html |title=曾跟母乞食！單親林穎彤演「鑫鑫」勁感觸 |date=2016-08-05 |work=東網}}</ref>，留下她與母、姊相依為命。由於生活陷入困境，其母在走投無路下，唯有帶同當時一歲半的她在[[葵芳站|葵芳站]]街頭[[行乞|行乞]]九個月<ref>{{cite news|url=http://orientaldaily.on.cc/cnt/entertainment/20160825/00282_002.html |title=林穎彤保護阿媽淡化行乞 |date=2016-08-25 |work=東方日報}}</ref>，後憑領取[[綜援|綜援]]長大。

她曾就讀於[[佛教林金殿紀念小學|佛教林金殿紀念小學]]、[[葵涌蘇浙公學|葵涌蘇浙公學]]和[[棉紡會中學|棉紡會中學]]。她曾經在[[葵青劇院|葵青劇院]]憑演唱《花花宇宙》贏得歌唱比賽冠軍<ref>{{Cite news |url=http://orientaldaily.on.cc/cnt/entertainment/20160810/00282_038.html |title=林穎彤重遊成長地感觸爆喊 |date=2016-08-10 |work=東方日報}}{{zh-yue}}</ref>。2003年，她寫信到電視台參加兒童節目獲發掘為童星，並能演出[[香港亞洲電視|亞洲電視]]電視劇《[[萬家燈火|萬家燈火]]》。劇中她表現不俗，及後又推出過兒歌唱片，更有機會於香港各地商場和中國大陸演出。

2006年，她參加雜誌《[[Yes!|Yes!]]》的校花校草選舉，獲得甜美校花獎及才藝大獎。

2007年，她受到賞識及邀請，回復幕前演出，拍攝[[香港有線電視|有線電視]]劇集《[[補習天后_(電視劇)|補習天后]]》。

2009年，其後經朋友介紹簽約成為JamCast旗下模特兒。同年參加[[電視廣播有限公司|TVB]]跟[[香港商業電台|商業電台]]合辦的節目比賽《[[十人巷|十人巷]]》，成為蛋白組成員「{{僻字|[[wiktionary:zh:𡃁|𡃁]]|o靚}}模Bella」，最後更於比賽中勝出而加入[[香港無綫電視|無綫電視]]成為旗下藝員。入行初期收入不穩，曾在眼鏡店當售貨員<ref>{{Cite web |url=https://www.hk01.com/%E7%9F%A5%E6%80%A7%E5%A5%B3%E7%94%9F/286871/%E6%9E%97%E7%A9%8E%E5%BD%A4-%E5%85%92%E6%99%82%E8%A1%8C%E4%B9%9E-%E5%85%A5%E8%A1%8C9%E5%B9%B4%E6%9C%AA%E4%B8%8A%E9%AB%98%E5%B3%B0%E5%B7%B2%E7%9C%8B%E9%80%8F-%E6%88%91%E4%BD%8F%E5%85%AC%E5%B1%8B%E5%8F%88%E6%80%8E%E6%A8%A3 |title=林穎彤　兒時行乞、入行9年未上高峰已看透：我住公屋又怎樣 |work=香港01 知性女生| date=2019-01-24 |accessdate=2019-01-28}}</ref>。

2010年起，林穎彤擔任住宅資訊節目《[[安樂蝸|安樂蝸]]》的主持至今。

2016年，她演出《[[城寨英雄|城寨英雄]]》內悲慘孖辮妹角色「李懷鑫」一角，人氣急升<ref>[http://www.hk01.com/%E5%A8%9B%E6%A8%82/35363/-%E4%BA%BA%E7%A5%9E%E5%85%B1%E6%86%A4-%E6%9E%97%E7%A9%8E%E5%BD%A4-%E5%9F%8E%E5%AF%A8%E8%8B%B1%E9%9B%84-%E6%85%98%E8%A2%AB%E7%84%A1%E9%99%90%E6%AC%BA%E5%87%8C-%E5%90%8C%E8%87%AA%E5%B7%B1%E8%BA%AB%E4%B8%96%E5%A5%BD%E7%9B%B8%E4%BC%BC 林穎彤《城寨英雄》慘被無限欺凌：同自己身世好相似]《[[香港01|香港01]]》，2016-08-04</ref>。

2017年8月，林穎彤於個人網絡社交平台中宣佈加盟[[星夢娛樂|星夢娛樂]]<ref>{{Cite web |url=http://news.modia.com.hk/2017/08/23/35119/%E5%8A%A0%E7%9B%9F%E6%98%9F%E5%A4%A2%E5%81%9A%E6%AD%8C%E6%89%8B%E3%80%80%E6%9E%97%E7%A9%8E%E5%BD%A4%E6%84%9F%E8%AC%9D%E4%BD%95%E5%93%B2%E5%9C%96%EF%BC%9A%E7%B5%A6%E6%88%91%E6%A9%9F%E6%9C%83%E5%8E%BB/ |title=加盟星夢做歌手 林穎彤感謝何哲圖：給我機會去追夢 |accessdate=2017-11-05 |date=2017-08-23 |publisher=流動新聞 娛樂頭版}}</ref><ref>{{Cite news |url=https://www.hk01.com/article/114148/ |title=林頴彤加入星夢做吳若希接班人 又係何哲圖賞識：唔會辜負佢 |accessdate=2017-11-05 |date=2017-08-24 |work=香港01 娛樂}}</ref>。

== 軼事 ==
傳媒和公眾人士一直以來將她的名字寫作「林穎彤」，至2016年她透過經理人通知傳媒正確寫法應為「林頴彤」<ref>{{cite news|title=林頴彤自爆宅到中秋冇人約|url=http://hk.on.cc/hk/bkn/cnt/entertainment/20160916/bkn-20160916135208859-0916_00862_001.html|accessdate=2016-10-01|agency=東網|date=2016-09-16}}</ref>，簽約星夢娛樂之後，改回「林穎彤」。

2017年10月，林穎彤在藝人[[吳若希|吳若希]]的[[Instagram|Instagram]]帖文上的留言「臭[[妓女|雞]]」上錯按「讚好」，引起小風波，她即時向對方傳短訊道歉，免遭誤會。而吳若希則稱相信對方是一時「手誤」才會錯按，不需要計較<ref>{{Cite news |url=https://news.mingpao.com/ins/instantnews/web_tc/article/20171016/s00007/1508150285125 |title=貼合照解圍 吳若希信林穎彤錯手LIKE負評 不會介懷 |date=2017-10-16 |work=明報}}</ref>。隨後，二人各自在Instagram貼出二人的合照，以示友好，並稱「多謝師姐大方幫我粉碎不和傳聞」<ref>{{Cite web |url=https://www.instagram.com/p/BaTfSfUlwNn/?taken-by=bella_lam |title=林穎彤的Instagram帖文 |date=2017-10-16}}</ref><ref>{{Cite web |url=https://www.instagram.com/p/BaTeQ0CBgQN/?taken-by=jinnyng |title=吳若希的Instagram帖文 |date=2017-10-16}}</ref>。
