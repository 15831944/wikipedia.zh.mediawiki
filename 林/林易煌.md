== 早年參政市議員 ==
就讀大學上街頭要求國會全面改選，擔任醫師第3年就選市議員，邀請同學賴清德從政摻與國民代表選舉，屬新潮流系。<ref>{{Cite web|url = http://magazine.chinatimes.com/ctweekly/20150814003275-300106|title = 成大同學林易煌爆料 揪逆向行駛賴清德慘遭球棒痛毆|author = 劉建宏|date = 2015年8月14日}}</ref>

1993選市議員。

1996年賴清德決定出馬參選國代。

== 退出民進黨參選市長 ==
2001年台南市長選舉，民進黨直接徵召許添財，當時許添財是無黨籍，林易煌跟許添財競選結果慘敗，便退出民進黨。<ref>{{Cite news|url = http://news.ltn.com.tw/news/politics/breakingnews/1364520|title = 國民黨徵召賴清德同學林易煌 對戰王定宇|last = |first = |date = 2015-06-30|work = |accessdate = }}</ref>

== 轉入國民黨 ==
2005年馬英九邀請他加入國民黨，也成為他的入黨推薦人。
