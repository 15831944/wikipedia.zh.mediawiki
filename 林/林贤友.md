'''林贤友'''{{BD|1962年||catIdx=L}}，籍贯[[温州|温州]]，[[清华大学|清华大学]]工商管理硕士毕业。

== 头衔 ==
林贤友现为甘肃省大陆桥房开发有限公司董事长。甘肃省慈善总会荣誉会长。兰州市工商业联合会温州企业项目发展商会理事会会长。甘肃浙江企业联合会常务副会长。
== 简历 ==
1980年代初从温州来到甘肃寻找商机，并且发财致富<ref>{{Cite web |url=http://news.dahe.cn/2011/08-04/100797652.html |title=温商林贤友 传统生意做成现代化 |accessdate=2011-08-04 |publisher=大河网 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，成为甘肃富商之一<ref>{{Cite web |url=http://finance.qq.com/a/20110727/004293.htm |title=林贤友：成功的人生需要非凡的梦想 |accessdate=2011-07-27 |publisher=腾讯财经(来源:新华网)}}</ref>。事业有成后开始参与各种慈善<ref>{{Cite web |url=http://baike.soso.com/v47595346.htm |title=林贤友:慈善简介 |accessdate=2011-12-31 |publisher=搜搜百科 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 活动<ref>{{Cite web |url=http://news.163.com/11/0819/20/7BRJUNN600014AEE.html |title=林贤友：慈善人生的代表 |accessdate=2011-08-19 |publisher=网易新闻(来源:云南网[昆明])}}</ref>，林贤友个人捐款总数为甘肃之首<ref>{{Cite web |url=http://news.xinhuanet.com/life/2011-08/01/c_121755427.htm |title=林贤友----甘肃慈善第一人 |accessdate=2011-08-01 |publisher=新华网(来源:中国甘肃网)}}</ref>。
