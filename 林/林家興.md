'''林家興'''（Alfred W. Lin，1990年6月8日-），籍貫[[浙江省|浙江省]][[溫州市|溫州市]]，戶籍在[[桃園市|桃園市]]，現居[[美國|美國]][[紐約市|紐約市]]。臺灣學生領袖及政治人物。
曾任第九屆[[中國國民黨青年團|中國國民黨青年團]]總團長兼[[中國國民黨|中國國民黨]]中央常務委員<ref>{{Cite news|url = http://www.nownews.com/n/2014/09/12/1410568|title = 國民黨青年團總團長出爐：台大國發所林家興|author = 王鼎鈞|date = 2014-09-12|work = [[今日新聞網|今日新聞網]]|accessdate = 2015-06-01}}</ref>。前任[https://www.facebook.com/USUNT/?fref=ts 臺灣北部大專院校學生自治聯合協會（北學聯）]協會理事長及創會召集人。2015年底由[[中國國民黨|中國國民黨]]提名為第九屆不分區立法委員候選人，現於美國[[紐約大學|紐約大學]]攻讀國際關係（International Relations）碩士。其政治意識形態為現今臺灣較為少見的藍統派，曾多次協助組織並參與臺灣學生紀念六四天安門事件的活動，並主張以中華民國憲法所揭示的自由、民主、法治、人權、均富等價值統一全中國。

== 生平 ==
民國79年出生於臺灣台北市，父親林德蒼為[[中華民國國軍|中華民國國軍]]退伍老兵，具榮民身份，後曾擔任中視製作人父親林德蒼是溫州林家村人，1929年出生，2011年逝世；母親為印尼華僑<ref>{{Cite news|url=http://news.xinhuanet.com/2011-08/11/c_131042985.htm|title=林家興：從"國共合作的産物"到"兩岸青年交流領袖生"|author = 黃玥 任沁沁|date= 2011-08-11||work = [[新華網|新華網]]||accessdate=2011-08-11}}</ref>。

成長於[[桃園市|桃園市]]，就讀[[國立中壢高商|國立中壢高商]]資料處理科，後考取並畢業於[[國立政治大學|國立政治大學]]歷史學系與政治學系雙學位，大學時期曾任[[國立政治大學|國立政治大學]]第11屆[[國立政治大學學生會|學生會]]會長、校務會議學生代表，畢業後考取[[中華民國陸軍|中華民國陸軍]]預備軍官並服役於[[陸軍關渡地區指揮部|陸軍關渡地區指揮部]]擔任機械化步兵少尉排長。退伍後進入[[國立臺灣大學|國立臺灣大學]]國家發展研究所大陸與兩岸關係組碩士班就讀。曾任臺灣北部大專院校學生會聯盟創始召集人、[[中華民國教育部|中華民國教育部]]第四屆學生自治諮詢小組諮詢委員、[[中國國民黨青年團|中國國民黨青年團]]總團長兼黨指定中央常務委員、[[中華民國總統府|中華民國總統府]]聘用諮議。

2015年5月[[朱習會|朱習會]]中，以[[中國國民黨青年團|中國國民黨青年團]]總團長的身份出席，為國民黨方面與會者中最年輕的一員<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20150504/603870/ 朱習會登場 國共兩黨領導人6年來首次會面]</ref>。
