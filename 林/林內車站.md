== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]，停靠車種以[[臺鐵區間車|區間車]]、[[台鐵區間快車|區間快車]]為主，少部分[[莒光號列車|莒光號]]亦有停靠。
*[[莒光號列車|莒光號]]停靠班次
**南下(逆行)：507次、561次。 
**北上(順行)：506次、554次。
*對號列車停靠率：7.8%；所有車種停靠率：55.1%。
*鄰近[[南投縣|南投縣]][[竹山鎮_(台灣)|竹山鎮]]，有公車客運行駛其間，車程10分鐘。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*[[石渣線|石渣線]]載運砂石，是中部道渣主要的儲存場

== 站內設施 ==
*列車資訊自動播音服務系統客家語、閩南語、國語等三種。
*售票窗口：設有兩個。(一個備用)
*自動售票機：設有兩台
*列車資訊LED顯示系統：剪票口一面、各月台一面。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。
*電梯:已設置兩部，已啟用。

== 歷史 ==
*1907年6月20日：設'''林內驛'''。
*2013年9月30日：啟用多卡通刷卡機。

==流行文化==
1987年上映的台灣電影《[[芳草碧連天_(電影)|芳草碧連天]]》曾在本站取景，並以在本站拍攝的送別場景作為影片片尾。
