'''林衣倩'''（{{bd|1986年|7月24日}}）藝名'''蔡羽芃'''台灣女演員，台灣嘉義市人，2015年畢業於[[台北市立大學|台北市立大學]]體育學系，曾就讀國立復興劇藝實驗學校（現為[[國立台灣戲曲學院|國立台灣戲曲學院]]），因此擁有[[戲曲|戲曲]]表演的特長。<ref>{{Cite web |url = https://www.facebook.com/profile.php?id=1700905510&lst=100001777037988%3A1700905510%3A1546710969&sk=about |title = 林衣倩facebook關於 |author = 林衣倩  }}</ref>目前擔任《[[戲說台灣|戲說台灣]]》電視劇演員<ref>{{Cite web |url = http://gn89850623.pixnet.net/blog/post/36646521-%E8%94%A1%E7%BE%BD%E8%8A%83 |title = 蔡羽芃@ 戲說台灣的家:: 痞客邦::  }}</ref>以及[[國立台灣戲曲學院|國立台灣戲曲學院]]兼任教師。<ref>{{Cite web |url = https://www.facebook.com/profile.php?id=1700905510&lst=100001777037988%3A1700905510%3A1546710969&sk=about |title = 林衣倩facebook關於 |author = 林衣倩  }}</ref>
