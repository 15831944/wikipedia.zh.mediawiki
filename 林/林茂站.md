'''林茂站'''（{{lang-ms|Stesen Kereta Api Rembau}}）位於[[马来西亚|马来西亚]][[森美兰州|森美兰州]]{{tsl|ms|Daerah Rembau|林茂县}}，是[[KTM通勤铁路|KTM通勤铁路]][[芙蓉线|芙蓉线]]和[[KTM电动列车服务|KTM电动列车服务]]的停站之一。在2015年10月10日之前，本站是电动列车服务的南端终点站，而该服务也在之后南延至{{站|金马士}}，并在数年之后计划延伸至[[新山中央车站|新山中央车站]]<ref>{{cite news|url=http://www.chinapress.com.my/20151001/%E6%9E%97%E8%8C%82%E7%81%AB%E8%BB%8A%E7%AB%99%E6%96%B0%E8%B7%AF%E7%B7%9A%E6%B7%A1%E9%82%8A%E5%BE%80%E8%BF%94%E9%87%91%E9%A6%AC%E5%A3%AB%E5%B1%95%E8%87%B3%E6%97%A5/|title=林茂火車站新路線 淡邊往返金馬士 展至10日開跑|publisher=[[中国报|中国报]]}}</ref>。

== 历史 ==
在本站启用之前，位于芙蓉的{{站|新那旺}}与{{站|双溪加笃}}的火车站也于2011年5月中投入使用，而本站则在2013年8月30日启用，车站位于圣泉福圣宫前方。<ref>{{cite news|url=http://www.sinchew.com.my/node/938178|title=林茂火车站30日启用．搭火车无需舟车劳顿了！|date=2013-08-30|publisher=[[星洲网|星洲网]]}}</ref>这使到从[[吉隆坡|吉隆坡]]前往林茂的服务需时1小时45分钟，而从林茂至[[芙蓉市|芙蓉市]]则需25分钟<ref>{{cite news|url=http://www.sinchew.com.my/node/938060|title=林茂电动火车站启用．隆市至林茂1小时45分钟|date=2013-08-30|publisher=[[星洲网|星洲网]]}}</ref>。
