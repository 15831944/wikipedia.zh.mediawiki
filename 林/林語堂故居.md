'''林語堂故居'''座落於[[台北市|台北市]][[士林區|士林區]][[陽明山|陽明山]][[仰德大道|仰德大道]]2段141號，是[[林語堂|林語堂]]生前最後十年的居所<ref>[http://www.linyutang.org.tw/big5/house.asp 關於故居]</ref>。目前由[[東吳大學|東吳大學]]經營管理，執行長為東吳大學中文系系主任鍾正道教授。

== 歷史沿革 ==

位於台北[[陽明山|陽明山]]上，建於1966年，由[[林語堂|林語堂]]親自設計，是其最後十年(1966年-1976年)定居台灣的住所。建築體現中西文化的融合，整體是以中國[[四合院|四合院]]佈局，東方特色的建築元素如藍色的[[琉璃瓦|琉璃瓦]]及窗櫺，而西方元素則如西班牙式白牆及[[螺旋|螺旋]]柱等。

台北市政府為紀念林語堂的文學成就，得到林夫人廖翠鳳女士捐贈大師藏書、著作、手稿及其遺物於大師舊居，成立「林語堂先生紀念圖書館」1985年5月開放。

[[台北市文化局|台北市文化局]]成立後，有意擴充活化原有圖書館功能，以「名人故居」及「文學生活館」之方向規劃。經過公開招標評選，由[[佛光人文社會學院|佛光人文社會學院]]（後改名為「[[佛光大學|佛光大學]]」）取得經營管理權，於2002年3月26日全新開館，成立「林語堂故居」。 林語堂故居結合展示參觀、藝文講座、餐飲休憩的多元化空間，完整呈現了語堂先生的格調思想、發明創意、生活態度與文學成就。2005年10月1日由[[東吳大學|東吳大學]]接受[[台北市政府文化局|台北市政府文化局]]委託經營林語堂故居。

2018年5月9日，經台北市文資審議委員審議通過，[[臺北市政府|臺北市政府]][[文化局|文化局]]公告林語堂故居為市定紀念建築，成[[台北市|台北市]]第一座為了對歷史、文化、藝術等有重要貢獻人物而保存的建物。

== 建築介紹 ==

此建築形式是以中國[[四合院|四合院]]為架構，設計取向為[[西班牙|西班牙]]式的風格，可為是兼具東西方特色以及現代感與古典美。其屋頂為藍色[[琉璃瓦|琉璃瓦]]，白色的拉毛粉牆是為[[地中海|地中海]]建築特色。另外，牆上深紫色的圓角窗櫺，透天中庭旁西班牙式螺旋廊柱，都再再呈現東西建築元素的結合。
林語堂先生愛竹、愛石，所以他刻意在中庭一角，由翠竹、[[楓香|楓香]]、蒼蕨、藤籮等植物，與造型奇特的石頭，營造出可愛的小魚池，他常坐在池邊的大理石椅上，享受「持竿觀魚」之樂。<ref>[http://www.linyutang.org.tw/big5/house.asp 關於故居]</ref>

== 空間簡介==  
===有不為齋餐廳===
原本是林語堂家的餐廳與客廳，如今開放為對外營業之餐廳。推開木門，延伸出的陽台是林語堂生前最喜愛的空間，他曾寫道：「黃昏時候，工作完，飯罷，既吃西瓜，一人坐在陽台上獨自乘涼，口銜煙斗，若吃煙，若不吃煙。看前山慢慢沉入夜色的朦朧裡，下面[[天母|天母]]燈光閃爍，若無所思。不亦快哉！」<ref>＜來臺二十四快哉＞一文</ref>

===閱讀研討室===
此空間原是車庫及客房，現為閱讀研討空間，展示林語堂的各種中外著作，包括小說、傳記、散文、月刊等八十餘種，以及廣獲國際推崇的《生活的藝術》之中文、英文、韓文、德文、法文、義大利文、西班牙文、葡萄牙文、丹麥文、挪威文、瑞典文、芬蘭文等十二國語言之譯本。此空間角落並設有故居紀念品及林語堂著作販賣處。<ref>林語堂故居導覽簡介手冊</ref>另，每週六上午此空間皆定期舉辦「讀原著學英文」之課程，閱讀《生活的藝術》英文版。<ref>[http://www.linyutang.org.tw/big5/event1.asp?idno=293 每周﹝六﹞讀原著學英文－「生活的藝術」開課中]</ref>

===書房展示間===
此空間將林語堂的書房以原貌呈現，陳列他的手稿、藏書、文具及英文打字機等。他依照自身發明的「上下形檢字法」，以及他改良的「國語羅馬字拼音法」，歷時五年，在1972年由[[香港中文大學|香港中文大學]]出版了《林語堂當代漢英辭典》，是為其在故居時最重要的著作。

===臥房展示間===
其臥室展示林語堂家庭生活照片、衣衫、家飾等。其空間中僅置有一張單人床，是因林語堂怕打擾其夫人廖翠鳳的生活作息，故設置兩個臥房。後來因為空間的改置，目前僅保存林語堂的房間。

===客廳及餐廳展示間===
此空間原為夫人的臥房及兩臥房間的共用衛浴空間，因後來空間改置，目前做為餐、客廳的展示空間。其空間展有林語堂設計的餐桌組，椅背上皆刻有「鳳」字的[[小篆|小篆]]體，是為感謝夫人廖翠鳳的辛勞而設計。另外，牆上掛有多幅林語堂墨寶及朋友贈送之書畫，如：[[蔣中正|蔣中正]]先生置贈之墨寶「壽」字、[[蔣宋美齡|蔣宋美齡]]女士所繪之蘭花水墨畫。此外，餐酒櫃上「有不為齋」四字為林語堂親自所寫，是其為上海的書房所取的名字。所謂的「有不為」者，正代表了林語堂不隨流俗的處事哲學，意思是世上有些事是他所不屑做的。<ref>林語堂故居導覽簡介手冊</ref>
