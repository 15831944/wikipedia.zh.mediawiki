== 車站構造 ==
林鳳營車站內設有[[岸式月台|岸式月台]]、[[島式月台|島式月台]]各一座。建造於1933年（[[昭和|昭和]]8年）的木造車站站房與[[後壁車站|後壁車站]]造形類似，為[[廡殿頂|廡殿頂]]構造建築，是台灣僅存的幾座日式風格車站建築。站內原有一條通往霖牧公司工廠的專用側線，但已停用。

林鳳營車站於2005年時依據[[文化資產保存法|文化資產保存法]]，公告為受保護的歷史性建築{{R|HACH}}。

== 營運狀況 ==
林鳳營車站是一座僅停靠[[臺鐵區間車|區間車]]的甲種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]。站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。站內設有可對應[[智慧卡|智慧票卡]]的驗票閘門，接受[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[icash|icash 2.0]]以及HappyCash進出站乘車。

== 歷史 ==
*1901年12月16日：設'''林鳳營乘降場'''。
*1902年6月1日：改稱'''林鳳營停車場'''。
*2000年6月1日：由三等站降為甲種簡易站，並指定由[[隆田車站|隆田站]]管理。
*2003年：配合拼音標準的改變，外文站名由原本的'''Linfungying'''改為'''Linfongying'''。
*2009年：
**配合政府地名拼音政策的調整，外文站名再度更改為'''Linfengying'''。
**8月8日：[[八八水災|八八水災]]之後，部份班次[[自強號列車|自強號]]臨時增停本站至8月13日止。
*2013年5月30日：啟用多卡通刷卡機。
*2016年5月12日：林鳳營車站前轉乘環境優化改善工程完工啟用，公車停靠處改遷移至候車亭，原林鳳營火車站公車站牌廢除。
