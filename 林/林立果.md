'''林立果'''{{BD|1945年|12月23日|1971年|9月13日|catIdx=Lin林}}<ref>[https://books.google.com/books?id=70xxDAAAQBAJ&pg=PT283&lpg=PT283&dq=%E6%9E%97%E7%AB%8B%E6%9E%9C1945%E5%B9%B4&source=bl&ots=7cmjuVc448&sig=ESaIH2gZ-7Hm2HydXd_yjm-z01g&hl=en&sa=X&ved=0ahUKEwjt5-XEldjUAhVGID4KHdHlDEU4FBDoAQgnMAE#v=onepage&q=%E6%9E%97%E7%AB%8B%E6%9E%9C1945%E5%B9%B4&f=false 《林彪傳》（1945年12月23日,葉群生林立果。）]</ref>，[[林彪|林彪]]与[[叶群|叶群]]之子，小名“老虎”。

== 生平 ==
林立果曾就读于[[北京市第四中学|北京市第四中学]]，后升入[[北京大学|北京大学]]物理系，[[文化大革命|文化大革命]]時輟學，1967年3月參加[[中國人民解放軍空軍|中國人民解放軍空軍]]，當上了[[中國人民解放軍空軍司令部|中國人民解放軍空軍司令部]]辦公室秘書，不久加入[[中國共產黨|中國共產黨]]。

== 政治活动 ==
中共称，1969年10月17日，[[吴法宪|吴法宪]]按林彪的授意，指示林立果任空軍司令部辦公室副主任兼作戰部副部長，这是个关键职位，所有文件都要经过林副主任的手。根据吴法宪在1980年审判中的证词，「事实上从1970年7月6日起，空军的所有事情都向林立果报告，所有事情都得经他处理、听他的命令」<ref>《中国历史上的一次大审判》，第93页</ref>。由於獲得多次提拔，當時有人稱之为“'''一年兵，二年黨，三年副部長，四年太上皇'''”。空四軍政委[[王維國|王維國]]一再吹捧林立果是“天才”。

1970年10月，林立果協助林彪組成秘密組織“聯合艦隊”，其成员绝大多数是人民解放军空军的军官，並制定武裝政變計劃《[[五七一工程紀要|“571工程”纪要]]》（意思是“'''武装起義'''”）。

1971年2月，林立果带着[[中国人民解放军空军政治部|中国人民解放军空军政治部]]副主任[[于新野|于新野]]到杭州；而后又将另一位副主任[[周宇驰|周宇驰]]自北京召到上海，从3月20日至24日在上海与他们及空4军政治部秘书处副处长[[李伟信|李伟信]]等召开秘密会议，根据林彪的命令，多次密谋<ref>《剑桥中华人民共和国史第15卷》，第327页，中国社会科学出版社</ref>。

1971年，林立果與林彪及母親[[葉群|葉群]]策動政變，意圖暗害[[毛泽东|毛泽东]]，結果事败。同年9月13日，林立果与林彪、叶群等九人乘「三叉戟」飛機出逃；据事后中共的官方说法，因当时情况紧急，飞机内油料仅够支撑飞行兩小时，经过一段时间超低空飞行进入蒙古后，燃料即将耗尽，随即遂寻找较平坦处进行迫降，结果蒙古[[温都尔汗|温都尔汗]]附近折戟沉沙，機上九人全部死亡。是為[[九一三事件|九一三事件]]。

后来，林立果的未婚妻[[張寧_(政治人物)|張寧]]女士出版回憶錄《塵劫》，[[唐德刚|唐德刚]]為序。

==业余爱好==
有传言说，1960年代末期，北京高干子弟间流传着[[披头士|披头士]]的唱片，林立果就是此乐队的歌迷<ref name="gs"> {{Cite web |url= http://qkzz.net/article/514380cb-06a1-4c74-a4f1-e8f0f3679882_3.htm|title=  “滚石”40年|accessdate=2014-01-24 |work= 林丽玲 |publisher= |date= }}</ref>｡ 未婚妻張寧也回忆说，林立果播放过摇滚音乐<ref name="zn">{{cite book |author=張寧 |authorlink= |coauthors= |title= 塵劫|year=1997 |publisher=明报出版社 |location=香港|isbn=9789623579407 }}</ref>。

作家[[朱大可|朱大可]]在《流氓的盛宴》一书中认为，[[崔健|崔健]]身穿军装演唱[[摇滚音乐|摇滚音乐]]是源于童年时北京空军大院中业余歌手林立果演唱披头士歌曲的回忆<ref name="zdk" />。林立果在摇滚音乐中注入的“叛逆精神”，在崔健音乐中获得了“秘密传承”<ref name="zdk">{{cite book |author= 朱大可 |authorlink= |coauthors= |title=流氓的盛宴 |year=2006 |publisher=新星出版社 |location= |isbn= 9787802251922}}</ref>。然而崔健在锵锵三人行（20141009）中否定了朱大可的说法。
