'''林虎'''（{{bd|1927年|12月26日|2018年|3月3日}}），原名'''林根生'''，[[汉族|汉族]]<ref name=d7/><ref name=d8/><!-- 虽然有俄罗斯血统，但是至少当人大代表的时候民族成分还不是俄罗斯族。《飞过天安门》里面第45页第5行最后 --><ref name=p>共产党是我再生父母《飞过天安门 寻访新中国开国大典受阅飞行人员》 萧邦振等著 第45-48页 ISBN 7-5065-5172-1</ref>，[[山东|山东]][[招远|招远]]人，原中国人民解放军空军副司令员，[[中国人民解放军中将|中国人民解放军空军中将]]<ref name=hainan>{{cite journal | url=http://hnrb.hinews.cn/html/2010-03/29/content_205050.htm | title=骁勇善战的“飞将军”林虎 | author=陈耿 | journal=海南日报 | pages=B12|date=2010-03-29}}</ref><ref name=pp>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_2017171  |title =空军原副司令林虎中将昨晚去世，享年91岁  |author =  |date = 2018-03-04  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref name=ifeng>{{Cite web |url = http://news.ifeng.com/a/20180304/56450519_0.shtml |title =空军原副司令林虎中将逝世 曾驾机越开国大典上空  |author =  |date = 2018-03-04 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

== 生平 ==
林虎祖上姓曹<ref name=cao> 《中国王牌飞行员》 耿龙武主编 第22页</ref>，其生母为俄罗斯人<ref name=p /><ref name=hainan/><ref name=ifeng/>。他的父亲年轻时[[闯关东|闯关东]]来到[[哈尔滨|哈尔滨]]，结识了一位俄罗斯姑娘，两人生下三个孩子，除他之外，还有一个姐姐和一个弟弟。一年冬季，他的父亲在运煤的火车上睡着后被冻死。不久，母亲和弟弟因霍乱病死，姐姐被别人领走，仅有几岁大的他被林之祯收为养子，改姓林，回到山东后私塾老师为他取名“林根生”<ref name=cao23> 《中国王牌飞行员》 耿龙武主编 第23页</ref>。

[[抗日战争|抗日战争]]全面爆发后，1938年10月，林根生志愿参加[[山东人民抗日救国军第三军|山东人民抗日救国军第三军]]第4路21旅<ref name=p />，被分配到某单位当勤务，部队领导给他起了“林虎”这个名字<ref name=p />。1939年初，他分配到山东省中国少年先锋队，先后参加了胶东、鲁中地区历次反“扫荡”、讨[[吴化文|吴化文]]、解放临沂等战役战斗<ref name=p /><ref name=autogenerated1>{{Cite web |url = http://www.xinhuanet.com/politics/2018-03/13/c_1122530421.htm |title =林虎同志逝世 |author = |date = 2018-03-13 11:35:16  |publisher = 新华网|language =  |accessdate =  |quote =  }}</ref>。后来又被分配到[[山东|山东]][[沂蒙山|沂蒙山]]区抗日根据地。1945年，林虎加入[[中国共产党|中国共产党]]。同年12月，由山东[[抗日军政大学|抗日军政大学]]一分校选送到[[东北民主联军航空队|东北民主联军航空队]]学航空技术。1949年，自[[东北人民解放军航空学校|东北人民解放军航空学校]]毕业，奉命调往[[南苑航空队|南苑航空队]]<ref name=ifeng/>。

1949年10月1日，在[[中华人民共和国开国大典|中华人民共和国开国大典]]上，林虎驾驶[[P-51战斗机|P-51战机]]飞过[[天安门|天安门]]上空。1951年，参加[[抗美援朝战争|抗美援朝战争]]，任[[中国人民志愿军|中国人民志愿军]]空军副团长，在朝鲜战场上击落和击伤[[美军|美军]]的[[F-86佩刀战斗机|F-86“佩刀”战斗机]]各一架。1954年初，到驻[[广州|广州]]沙堤机场的空军歼击航空兵第十八师任副师长。1958年，指挥部队空战，一次击落[[中华民国空军|中华民国空军]]飞机两架、击伤一架<ref name=ifeng/>。

林虎后历任[[广州军区空军|广州军区空军]]军训部部长、空军副军长、[[空军兴宁指挥所|空军兴宁指挥所]]主任、广州军区空军副司令员、[[空军第九航空学校|空军第九航空学校]]副校长、[[昆明军区空军指挥所|昆明军区空军指挥所]]副主任、空军学院副院长<ref name=autogenerated1 />。1985年，任[[中国人民解放军空军|中国人民解放军空军]]副司令员<ref name=ifeng/>。1988年，获授中国人民解放军空军中将军衔<ref name=ifeng/><ref>{{citenews|author=徐平 |title=重新实行军衔制 |url=http://chn.chinamil.com.cn/jfdg/2011-07/13/content_4473591.htm |accessdate=2012-05-05 |newspaper=中国军网 |deadurl=yes |archiveurl=https://web.archive.org/web/20130511192908/http://chn.chinamil.com.cn/jfdg/2011-07/13/content_4473591.htm |archivedate=2013-05-11 }}</ref>。荣获三级[[独立自由勋章|独立自由勋章]]和三级[[解放勋章|解放勋章]]<ref name=p62>共产党是我再生父母《飞过天安门 寻访新中国开国大典受阅飞行人员》 萧邦振等著 第62页</ref>。

林虎是中华人民共和国第七<ref name=d7>《中国人物年鉴 1989》主编: 李方诗 第407页 ISBN 7-80039-301-0</ref>、八届全国人民代表大会代表<ref name=d8>《第八届全国人民代表大会及其常务委员会会议大事记（1993年3月至1998年2月）》 全国人大常委会办公厅秘书局编 371页 ISBN 7-80078-425-8</ref>，[[全国人大外事委员会|全国人大外事委员会]]委员<ref name=ifeng/>。

1994年10月退役。

1997年8月23日，在[[莫斯科航展|莫斯科航展]]上，林虎体验飞行了[[苏-30战斗机|苏-30战斗机]]并进行[[眼镜蛇机动|眼镜蛇机动]]<ref>{{Cite journal|last=|first=|date=|year=1999|title=戎马一生为蓝天——访原空军副司令员林虎中将|url=|journal=现代军事|volume=6|pages=28-29|via=}}</ref><ref>{{Cite news|url=http://www.cannews.com.cn/epaper/zghkb/2012/11/08/A08/story/327816.shtml|title=一次难忘的飞行|last=|first=|date=|work=中国航空报 |access-date=|archive-url=|archive-date=|dead-url=}}</ref><ref name=ifeng/>。

2018年3月3日，林虎与原空军中将、生前挚友[[耀先|耀先]]同日病逝，享年91岁<ref name=ifeng/>。
