'''林俊德'''（{{bd|1938年|3月13日|2012年|5月31日}}），[[福建|福建]][[永春|永春]]人，中国爆炸力学与核试验工程专家，[[中国工程院|中国工程院]]院士，[[中国人民解放军少将|中国人民解放军少将]]。

== 生平 ==
林俊德1960年毕业于[[浙江大学|浙江大学]]机械系，分配到[[国防科委|国防科委]]下属研究所工作；之后，到[[哈尔滨|哈尔滨]][[中国人民解放军军事工程学院|中国人民解放军军事工程学院]]进修两年。从1964年到1996年，参与了中国的全部45次[[核试验|核试验]]<ref>{{cite journal |author1=郑流年 |title=林俊德将军：从永春大山走出的院士 |journal=福建党史月刊 |date=2002 |issue=8 |url=http://roll.sohu.com/20120913/n353053716.shtml}}</ref>。林俊德长期从事空中爆炸冲击波、地下爆炸岩体应力波、爆炸地震波、爆炸安全工程技术、强动载实验设备与实验测量技术等研究工作。参加过众多重大国防科研试验任务，带领项目组解决了多项关键技术课题。为测量核爆炸冲击波参数，林俊德团队独立创新制作了钟表式压力自记仪。曾获国家发明奖2项，国家科技进步奖2项，获二等以上省部级科技进步奖12项。1990年获国家人事部颁发的“有突出贡献的中青年专家”证书<ref>{{cite web |title=林俊德 |url=http://www.cae.cn/cae/html/main/colys/21322066.html |website=中国工程院 |accessdate=2018-09-23}}</ref>。1993年，晋升少将军衔。2001年，当选[[中国工程院|中国工程院]]土木、水利与建筑工程学部院士。生前担任[[中国人民解放军总装备部|解放军总装备部]][[中国核试验基地|第21试验训练基地]]研究员。2012年5月31日，因患[[胆管癌|胆管癌]]在[[西安|西安]]逝世<ref>{{cite web |title=永春籍院士林俊德病逝 亲友眼中“两弹一星“功臣 |url=http://news.cntv.cn/china/20120913/103247.shtml |website=中国网络电视台 |accessdate=2018-09-23 |date=2012-09-13}}</ref>。逝世前数小时，还在众人的搀抬下走向数步之外的办公桌继续工作<ref>{{cite web |author1=余建斌 |title=大漠铸核盾 生命写忠诚——追记中国工程院院士、总装某基地研究员林俊德 |url=http://cpc.people.com.cn/n/2012/0924/c64104-19085591.html |website=人民网 |accessdate=2018-09-23 |date=2012-09-24}}</ref>。

2013年1月17日，中央军委主席[[习近平|习近平]]签署命令，追授林俊德“献身国防科技事业杰出科学家”荣誉称号，同时追授一级[[英雄模范勋章|英雄模范勋章]]。2013年2月19日，荣获“2012年度[[感动中国|感动中国]]十大人物”荣誉称号<ref>{{cite web |title=2012感动中国人物：林俊德 |url=http://news.cntv.cn/special/gdzg2012/linjunde/index.shtml |website=央视网 |accessdate=2018-09-23}}</ref>。2018年9月20日，中央军委决定，增补林俊德、[[张超_(飞行员)|张超]]为全军挂像英模<ref>{{cite web |title=经中央军委批准 增加林俊德、张超为全军挂像英模 |url=http://www.xinhuanet.com/politics/2018-09/20/c_129957876.htm |website=新华网 |accessdate=2018-09-23 |date=2018-09-20}}</ref>。
