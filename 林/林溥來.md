'''林溥來'''（{{lang-en|'''Patrick Lam'''}}，{{bd|1977年|2月15日<ref>電視廣播有限公司：[http://artiste.tvb.com/main/artistedetail/patricksir 男藝人'''林溥來'''基本資料]</ref>}}），人稱'''Patrick Sir'''，香港通識科及英文科補習教師、補習社 EDFLOW Learning Center（EDFLOW）創辦人、[[無綫電視|無綫電視]]經理人合約男藝員。

== 背景 ==
林溥來畢業於[[喇沙小學|喇沙小學]]（1989年）及[[喇沙書院|喇沙書院]]（1996年）<ref>{{web reference|URL=http://std.stheadline.com/daily/news-content.php?id=173418&target=2|title=喇沙校友創辦國際幼園|work=星島日報|year=2014年|date=6月23日}}</ref>，之後分別於[[香港科技大學|香港科技大學]]及[[香港中文大學|香港中文大學]]獲得學士及碩士學位。在大學讀書時，林溥來曾是香港甲一組[[發景籃球隊|發景]]及[[永倫籃球隊|永倫]]籃球隊隊員。另一方面，他又兼職拍攝電視及雜誌廣告。由於網民認為他比起明星有型及帥氣，所以獲得『補習型男』的稱號。

2005年，林溥來開辦 EDFLOW Learning Center。在辦學初期，他兼教英文、數學、科學等科目。其後因為補習社的業務不斷擴展，至2010年已擴展至三間分校。2009年，他獲[[無線電視|無線電視]]邀請，於無線電視收費兒童台節目《[[通識通通識|通識通通識]]》擔任專家主持，也為無線電視節目《[[放學ICU|放學ICU]]》中「Patrick Sir英語通通識」的環節及《[[智激校園|智激校園]]》中分別擔任專家主持及首席評判。除了學術領域之外，林溥來亦參與其他如文化、旅遊及綜藝等電視節目，亦有為一些大型公開活動當嘉賓及評判。此外，林溥來也為多間雜誌及報章撰寫專欄如《[[太陽報|太陽報]]》的「通識攻略」專欄、《「兒童快報」》的通識專欄等，並且為學生編寫教材用書。<ref>{{cite news|title=林溥來熱血籃球火|url=http://the-sun.on.cc/cnt/lifestyle/20120328/00485_002.html|accessdate=2016-05-05|newspaper=《[[太陽報|太陽報]]》|date=2012-03-28}}</ref>

目前，林溥來是私立幼稚園「伽利利國際幼稚園」的校監。<ref>{{cite news|title=簽約經理人 林溥來望主持多元化|url=http://hd.stheadline.com/ent/ent_content.asp?track=news&contid=294812&srctype=n|accessdate=2016-05-05|newspaper=《[[頭條日報|頭條日報]]》|date=2014-07-25}}</ref>同時於無綫電視主持節目《[[東張西望|東張西望]]》以及《[[Think_Big_天地|Think Big 天地]]》。

2018年4月由基本藝人轉為經理人合約。

== 爭議 ==
2016年5月23日，林溥來在其有份主持的綜藝節目《[[東張西望|東張西望]]》內，就近日有關《[[壹週刊|壹週刊]]》與[[霸王集團|霸王集團]]官司案件報道後，以「就算幾自由都好，都要有適當嘅自我審查，按當地文化同環境講適當嘅嘢」形容和反思「言論自由」而被網民唾罵，被認為是政治立場親[[梁振英|梁振英]]的[[余詠珊|余詠珊]]旗下藝人操控市民的[[意識形態|意識形態]]和個人思想。<ref>{{cite news|title=《東張西望》主持談言論自由：幾自由都要有適當嘅自我審查|url=http://hk.apple.nextmedia.com/realtime/news/20160524/55142055|accessdate=2016-05-25|newspaper=《[[蘋果日報|蘋果日報]]》|date=2016-05-24}}</ref>
