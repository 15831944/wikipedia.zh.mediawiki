林慧如，[[民主進步黨|民進黨]]雲林縣政治人物。曾任[[雲林縣政府|雲林縣政府]]農業處長、[[雲林縣政府|雲林縣政府]]勞工處長、[[古坑鄉|古坑鄉長]]、亦曾任第12、13、14、15屆[[雲林縣議會|雲林縣議員]]。[[國立政治大學|國立政治大學]]國際貿易學系學士。

== 經歷 ==
林慧如(1962年6月7日-)，台灣[[雲林縣|雲林縣]]人，[[民主進步黨|民進黨]]籍政治人物。[[國立政治大學|國立政治大學]]國際貿易學系畢業，曾任[[雲林縣政府|雲林縣政府]]農業處長、[[雲林縣政府|雲林縣政府]]勞工處長、[[古坑鄉|古坑鄉長]]、(第12、13、14、15屆)[[雲林縣議會|雲林縣議員]]、亦曾在[[鄭南榕|鄭南榕]]與友人創辦的[[自由時代雜誌|自由時代雜誌社]]協助編輯的工作。

== 家庭背景 ==
外祖父黃朝文曾任[[古坑鄉長|古坑鄉長]]。

父親[[林國華_(臺灣政治人物)|林國華]](1935年－2016年)，為1988年[[1988年臺灣農民運動|520農民運動]]總指揮、前[[立法院|立法委員]]、前[[農委會副主委|農委會副主委]]。

母親[[黃富美|黃富美]](1936年－2011年)，為退休國小老師、前[[國大代表|國大代表]]。

丈夫羅吉丞是台灣新竹縣人，婚後育有1子2女。

== 政治經歷 ==
'''<big>踏入政壇</big>'''

1988年，剛從政大畢業的林慧如，留在台北準備考研究所，當時適逢[[1988年臺灣農民運動|520農民運動]]，其父親[[林國華_(臺灣政治人物)|林國華]]擔任總指揮，因為當天台北天氣炎熱，他心繫家鄉的農民便趕到活動現場沿路發放茶水，一起聲援農民運動，不料在晚間七點，林慧如與父親[[林國華_(臺灣政治人物)|林國華]]遭警方逮捕，坐了七個月又二十三天的黑牢。

出獄後，林慧如至自由時代雜誌社，向[[鄭南榕|鄭南榕]]先生致謝，感謝他對[[1988年臺灣農民運動|520農民運動]]的義助，得知[[鄭南榕|鄭南榕]]先生正為捍衛[[言論自由|言論自由]]與[[台灣獨立運動年表|台灣獨立]]理想，

準備以身殉道。當下林慧如決定留在雜誌社幫忙。

1989年4月7日，林慧如親眼見證了[[鄭南榕|鄭南榕]]先生以身殉道，那堅持理想的身影，已深深影響了林慧如，之後他決定返回雲林家鄉投入政治。

'''<big>雲林縣議員</big>'''

1988年林慧如隨著他的父親林國華，參與當年的「[[1988年臺灣農民運動|五二零農民運動]]」，並因參與[[農民運動|農民運動]]而入獄。林慧如在出獄後於1990年間，

以[[民主進步黨|民進黨]]員身分投入[[雲林縣第二選區縣議員|雲林縣第二選區縣議員]]選舉，就以出生之犢之勢，最高票當選。

林慧如擔任縣議員時期的服務都是騎著摩托車到處奔波，她一步一腳印的精神與的服務態度，使她很快建立起口碑。

因為她在第14屆爭取連任時，以同額競選之姿，穩獲當選。第12、13、15屆競選均以最高票當選，其中有一次還是獲得全縣最高票10334票當選<ref>{{Cite web|url=http://db.cec.gov.tw/histQuery.jsp?voteCode=20091201D1D1&qryType=ctks&prvCode=03&cityCode=009&deptCode=020x|title=中選會選舉資料庫網站|accessdate=2018-07-04|work=db.cec.gov.tw}}</ref>。

'''<big>古坑鄉鄉長</big>'''

2005年全國鄉鎮市長選舉，林慧如在支持者的勸進下，競選雲林縣[[古坑鄉鄉長|古坑鄉鄉長]]。林慧如憑藉其擔任議員時期累積的雄厚基層實力，

在三位參選人中，林以8781票，48.79%的得票率<ref>{{Cite web|url=http://db.cec.gov.tw/histQuery.jsp?voteCode=20051201D1D1&qryType=ctks&prvCode=01&cityCode=009&deptCode=007x|title=中選會選舉資料庫網站|accessdate=2018-07-04|author=中選會|date=2018/7/4|work=db.cec.gov.tw|publisher=http://db.cec.gov.tw/histQuery.jsp?voteCode=20051201D1D1&qryType=ctks&prvCode=01&cityCode=009&deptCode=007x}}</ref>，當選鄉長。

並於2009年尋求連任時，因其備受肯定的清廉形象與執政能力，沒有對手與之競爭，而同額競選順利連任。

'''<big>雲林縣政府</big>'''

林慧如在卸任鄉長後，全心照料臥病在床的父親[[林國華_(臺灣政治人物)|林國華]]，伴隨[[林國華_(臺灣政治人物)|林國華]]走人生的最後一哩路。

在[[林國華_(臺灣政治人物)|林國華]]逝世後，2017年[[雲林縣縣長|雲林縣縣長]][[李進勇|李進勇]]邀請林慧如擔任[[雲林縣勞工處處長|雲林縣勞工處處長]]一職，同年六月再轉任[[農業處長|農業處長]]。

隨後林慧如接受[[民主進步黨|民進黨]]徵召，轉換跑道參選[[斗六市市長|斗六市市長]]，已於2018年4月底辭職，專注投入選戰及選民服務，為年底選舉做準備。

== 事蹟 ==
'''<big>縣議員時期</big>'''

'''為社會貧戶請命 編列預算'''

•林慧如在[[縣議會|縣議會]]第十二屆第三次定期大會中，審查到社會科輔助民間團體(如婦聯會、商業團體等)二四二萬八千元時，林慧如認為社會上有更多低收入戶身心障礙者、精神疾患者，需要照顧希望能將該預算移做社會救濟，後來議長室與社會科人員當面溝通後，同意由社會救濟基金頂下給予補助。<ref>{{Cite journal|title=為社會貧戶請命 編列預算|author=葉子綱|url=|journal=民眾日報|issue=|doi=|others=|year=1991/5/29|volume=|page=|pmid=}}</ref>

'''審慎審查各項預算'''

•1993年5月26日，林慧如議員提出質疑不滿下年度總預算成長15億，農業局卻負成長高達一億六千萬，要求縣政府農業局說明。

•林慧如在擔任議員期間認真審查各項預算，為了把關工程決算，還曾向縣政府索取工程發包資料，花三天兩夜整理後，在縣議會開會時公開指稱八十二年度的工程決算案件起碼浪費公帑多達二億五千多萬。

•雲林縣政府因財政困難原本在民國七十九年元月要發放第一款低收入戶救濟金，每月一千二百元，到該年五月都尚未發放，擔任縣議員的林慧如聲稱發揮人飢己飢的精神，五月十一日中午在縣府門口絕食靜坐，在社會科長張琛保證十二日發放下，及議員同仁勸說下，林慧如才同意離開。<ref>{{Cite journal|title=雲縣未發貧戶救濟金 人飢己飢 議員林慧如絕食抗議|author=林銘忠|url=http://dx.doi.org/10.1002/bies.v12:12|date=1990-12|journal=BioEssays|issue=12|doi=10.1002/bies.v12:12|others=|year=|volume=12|page=|issn=0265-9247|pmid=}}</ref>

'''為清廉形象樹立標竿'''

•林慧如對於議會的金錢文化，他卻不曾被汙染，堅持理念尤勝過金錢的林慧如對於鉅額的金錢視之如流水，在1994年3月1號縣[[議會正、副議長|議會正、副議長]]選舉中再度以行動實現了崇高的精神。林慧如在議會質詢時，人高馬大的縣府官員也不敢小覷個子小的林慧如，因為理直氣壯的他，常以咄咄逼人的語氣，夾中時弊的質詢，令官員們招架不住。縣議員任內，林慧如被議員同仁們公認為最敬業的縣議員；也由於他堅持「眾人皆醉、我獨醒」的理念，因此官員們遇到他的質詢時都心服口服；其執著與擇善固執，連「護航派」的議員都對她讚譽有加。<ref>{{Cite journal|title=⟨一步一腳印 林慧如誰與爭鋒⟩|author=蔡水星|url=|date=1997年12月28日|journal=(中國時報，|issue=12|doi=10.1002/chin.v28:12|others=|year=|volume=28|page=|issn=0931-7597|pmid=}}</ref>

'''<big>古坑鄉鄉長時期</big>'''

•林慧如以「清廉、勤政」自許，任內政績斐然，積極重視[[古坑鄉|古坑]]交通建設問題、活絡地方觀光休閒產業，並針對外配子女教育及當地人口老化等問題進行改善。

他在任期內每年舉辦古坑咖啡節與柳丁嘉年華以此帶動地方產業以及觀光發展。

•林慧如不僅致力於帶動古坑觀光產業發展也將鄉村的老年人口問題視為當務之急，率領[[古坑鄉公所|古坑鄉公所]]成立全國第一個老人福利所<ref>{{Cite web|url=http://m.ltn.com.tw/news/local/paper/737526|title=〈中部〉古坑公所開先河 設老人福利所 - 地方 - 自由時報電子報|accessdate=2018-07-04|work=m.ltn.com.tw}}</ref>，整合全鄉各地現有老人健康、醫療、服務及福利措施，配合硬體樂齡據點，提供老年人娛樂、休閒、再學習機會，激發老年人潛力。以因應人口日益老化、人民對休閒需求提升，林慧如也主動向[[教育部|教育部]]申請樂齡學習計畫，鼓勵老人家走出家門且學習新事物，真正落實[[老人福利|老人福利]]及照護政策。此外，規劃孩童課後集中輔導工作且積極幫助許多面臨困境孩子，林慧如開辦了全國唯一 一個由[[鄉鎮市公所|鄉鎮市公所]]主辦的「陽光學園」<ref>{{Cite web|url=http://m.ltn.com.tw/news/local/paper/679597|title=〈中部〉古坑陽光學園 家教式輔導邊緣生 - 地方 - 自由時報電子報|accessdate=2018-07-04|work=m.ltn.com.tw}}</ref>，讓在學校適應不良的學生，進行「家教式」的輔導，讓孩子偏差行為及觀念有所修正後，再返回學校繼續常態性教育。

卸任時，林慧如也不忘為地方發聲，先前[[921大地震|九二一地震]]造成[[古坑清水溪大橋|古坑清水溪大橋]]毀損，嚴重影響當地交通及居民生活，林慧如特別當面提請省府[[林政則|林政則]]主席，代為協助反映地方之需。<ref>{{Cite news|url=http://www.fingermedia.tw/?p=228767|title=雲林縣古坑鄉清水溪大橋修復案，將進行復建工程先期規劃可行性評估作業|date=2015-01-09|newspaper=指傳媒 Fingermedia|accessdate=2018-07-04|language=zh-TW}}</ref><ref>{{Cite web|url=http://www.macrocosm.tw/indexmain.php?mmenu=18000&sid=51279&cateid=10&imp=1|title=大宇宙能量 

                -焦點新聞 : 消失的149甲線清水溪橋路段|accessdate=2018-07-04|work=www.macrocosm.tw}}</ref>

'''<big>雲林縣政府處長時期</big>'''

林慧如先以勞工處長職務，半年後再調任[[農業處長|農業處長]]，在這期間，林慧如服務勞工或農民，不論是勞基法[[一例一休|一例一休]]修法業務、農業天然災損爭取等等，都有一番作為。

先前的北農休市風波，林慧如更是從前往日本參加食品展的搭機途中折返回縣政府召開緊急會議<ref>{{Cite news|url=http://www.chinatimes.com/newspapers/20180306000549-260114|title=不處理不安心 林慧如跳車回雲林|last=中時電子報|newspaper=中時電子報|accessdate=2018-07-04|language=zh-TW}}</ref>，心繫著如何幫忙受波及的農民，立即祭出[[預冷|預冷]]及[[耕鋤|耕鋤]]之政策<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/2360784|title=北農開市菜價穩住 雲縣農業處長喊話：不要再連休了 - 生活 - 自由時報電子報|accessdate=2018-07-04}}</ref>，以減少農民的損失；而在蒜價事件中，林慧如也緊急啟動[[烘乾機補助|烘乾機補助]]及[[烘乾費補助|烘乾費補助]]等措施<ref>{{Cite news|url=http://www.cna.com.tw/news/aloc/201803220306-1.aspx|title=平穩大蒜價格 雲林縣府將補助烘乾機 {{!}} 地方 {{!}} 中央社 CNA|accessdate=2018-07-04}}</ref>，適當的應變處置，讓農民損失不再擴大。

此外，林慧如也十分關心農業缺工及非法農業外勞問題<ref>{{Cite news|url=http://www.chinatimes.com/realtimenews/20171115004312-260407|title=取締非法農業外勞 農業處長：罰到認真做事的人|last=中時電子報|newspaper=中時電子報|accessdate=2018-07-04|language=zh-TW}}</ref>，並積極呼籲中央試辦農業外勞計畫，以改善缺工問題。<ref>{{Cite news|url=http://www.chinatimes.com/realtimenews/20171122003662-260402|title=警掃蕩非法農業外勞  雲農民怒吼：立委只會打架作秀！|last=中時電子報|newspaper=中時電子報|accessdate=2018-07-04|language=zh-TW}}</ref><ref>{{Cite news|url=http://www.chinatimes.com/newspapers/20171212000571-260107|title=雲縣府籲中央 試辦農業外勞|last=中時電子報|newspaper=中時電子報|accessdate=2018-07-04|language=zh-TW}}</ref>
