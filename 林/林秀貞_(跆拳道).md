'''林秀貞'''（{{bd|1986年|8月20日}}），[[韓國|韓國]]女子[[跆拳道|跆拳道]]運動員，是[[2008年夏季奧林匹克運動會跆拳道比賽—女子57公斤級|2008年北京奧運]]女子57公斤級金牌得主。

== 亞洲運動會 ==
'''林秀貞'''在十六歲時，在[[2002年亞洲運動會|2002年 釜山亞運會]]跆拳道女子51公斤級奪得金牌。

== 世界大學生運動會 ==
2007年，'''林秀貞'''在[[2007年夏季世界大學生運動會|2007年曼谷大運會]]跆拳道女子59公斤級奪得金牌。

== 夏季奧林匹克運動會 ==
'''林秀貞'''在[[2008年夏季奧林匹克運動會跆拳道比賽|2008年北京運動會跆拳道比賽]]女子57公斤級奪得金牌。
