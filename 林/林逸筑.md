'''林逸筑'''（{{bd|1984年|1月26日|2012年|4月6日|catIdx= L林}}）出生于[[中华民国|中华民国]][[新北市|新北市]]，为著名的[[模特儿|平面模特儿]]。<ref name=" shws" >{{cite web |url= http://roll.sohu.com/20120407/n339999341.shtml|title=艳模林逸筑跳楼身亡 生前写真尺度惊人 |publisher=搜狐网 |accessdate= 2012-04-07|work= |language =zh-cn}}</ref><ref name=" szw">{{cite web|url=http://www.sznews.com/ent/content/2012-04/08/content_6633429.htm|title=台湾模特林逸筑跳楼身亡|publisher=深圳新闻网|accessdate=2012-04-08|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120411011208/http://www.sznews.com/ent/content/2012-04/08/content_6633429.htm|archivedate=2012年4月11日|df=}}</ref>

== 生平 ==
=== 杂志模特 ===
林逸筑，女生，曾担任限制级性感男性杂志的封面写真艳模特林逸筑，《尤物》当时写着，即使像小筑这种“拥有绝杀外型的女恐怖份子”，其实“只是需要被聆听的”。对照林逸筑以自杀结束生命，令人格外感慨。。<ref name=" shws" /><ref name=" szw" /><ref>{{cite web|url=http://www.redrs.net/index.php?doc-view-150.html|title=台红人榜林逸筑|publisher=红人榜|accessdate=2012-04-08|work=|language=zh-cn}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
  
=== 逝世 ===
林逸筑患有轻度[[忧郁症|忧郁症]]，加上近日来和男友吵架，她发完短信“你超屌，有人为你死”、“掰了我的爱”之后，在[[新北市|新北市]][[芦洲区|芦洲区]]住处楼顶跳楼逝世。<ref name=" shws" /><ref name=" szw" /><ref>{{cite web |url=http://ent.163.com/12/0407/04/7UF9DUB500032DGD.html |title=台湾28岁艳模林逸筑疑因情变跳楼身亡 |publisher=网易 |accessdate=2012-04-07 |work= |language=zh-cn }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url=http://ent.ifeng.com/idolnews/hk/detail_2012_04/06/13700380_0.shtml |title=台湾模特林逸筑跳楼身亡 曾患抑郁症因感情问题寻死 |publisher=凤凰网 |accessdate= 2012-04-07|work= |language =zh-cn}}</ref><ref>{{cite web |url= http://eladies.sina.com.cn/news/2012/0406/17191140189.shtml|title=台杂志艳模跳楼身亡 最后短信：掰了我的爱 |publisher=新浪网 |accessdate= 2012-04-06|work= |language =zh-cn}}</ref>

== 评价 ==
林逸筑自谓：“我也没坏到像[[恐怖份子|恐怖份子]]一样，我比较像[[蔷薇|蔷薇]]，带刺但是不伤人，花茎上的刺仅是伪装。”，“所以玩咖就别找我了，我才是脆弱容易受伤的人，嘻！”<ref name=" shws" /><ref name=" szw" />

《[[尤物|尤物]]》称她是：“外表冶艳、作风大胆的[[新生代|新生代]]都会女子”。<ref name=" shws" /><ref name=" szw" />
