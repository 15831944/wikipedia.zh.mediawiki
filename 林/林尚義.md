'''林尚義'''（'''Spencer Lam Sheung Yee'''，{{bd|1934年|12月14日|2009年|4月23日|catIdx=Lin林尚義}}<ref name="mp_death">{{cite news|title=資深足球評述員林尚義逝世 |url=http://hk.news.yahoo.com/article/090423/4/btu6.html|publisher=明報|date=2009-04-23|accessdate=2009-04-23}}</ref>），[[香港|香港]]退役[[足球|足球]][[運動員|運動員]]、 著名[[電視|電視]]足球評述員暨[[教師|教師]]，和影視兩棲演員。林於[[聯合書院|香江書院]]（現併入香港中文大學）經濟系畢業，曾代表[[中華民國|中華民國]]參加多項國際足球賽，有'''重炮手'''及'''䱽魚'''等稱號，晚年被尊稱為'''阿叔'''。

== 生平經歷 ==
林尚義生於一個殷實家庭，有8個姊姊及一個哥哥，是么子，父母在[[九龍城|九龍城]]有一幢大廈收租，在[[青山公路|青山公路]]擁有6塊[[土地|土地]]<ref>「阿叔倦了林尚義」《壹週刊》，2006年6月29日</ref>。經濟環境許可，令他自小參與足球運動無後顧之憂。

== 球員生涯 ==
[[File:Kitchee_1959.jpg|thumb]]於1959年到[[澳門|澳門]]作籌款義賽，前排右二為林尚義]]

林尚義在13歲開始參與沙地小型足球，於1953年加入[[中華體育會|中華]]乙組隊，翌年獲得[[東方足球隊|東方]]賞識，展開甲組球員生涯，之後轉投[[傑志體育會|傑志]]，於1958年重返[[東方足球隊|東方]]。<ref>「國腳訪問記(十九) 鴻運當頭的林尚義」《星島日報》，1958年5月4日第5版</ref>

林尚義曾就讀[[威靈頓英文中學|威靈頓英文中學]]，後來畢業於香江書院經濟系及[[香港教育學院|葛量洪教育學院]]，為了賺錢供養兒子升學，他一度同時擔任教師、足球員及足球評述員，先後在[[聖士提反小學|聖士提反小學]]及[[文理書院|文理書院]]任教體育，社會及中國語文等科目<ref>「阿叔倦了林尚義」《壹週刊》，2006年6月29日</ref>。

1958年，他代表[[中華台北足球代表隊|中華民國]]參加[[1958年亞洲運動會|東京亞運會]]，中華民國成功衛冕足球金牌<ref>參見：國際足協[http://www.fifa.com/worldfootball/statisticsandrecords/players/player=289797/index.html FIFA.com - Seung Yee LAM]</ref>。當年的陣容還包括「香港之寶」[[姚卓然|姚卓然]]、[[莫振華|莫振華]]、[[何應芬|何應芬]]等人，而執教的是有「中國球王」美譽的[[李惠堂|李惠堂]]<ref>{{cite news|url=http://hk.news.yahoo.com/article/090423/3/bu4l.html|title=「重炮」震球壇 亞運摘金|publisher=星島|date=2009-04-24|accessdate=2009-04-24|deadurl=yes|archiveurl=https://web.archive.org/web/20090426172557/http://hk.news.yahoo.com/article/090423/3/bu4l.html|archivedate=2009-04-26}}</ref>。兩年後，他再度代表[[中華台北足球代表隊|中華民國]]參加[[1960年夏季奧林匹克運動會|羅馬奧運會]]。

1971年，林尚義帶領[[香港流浪足球會|流浪]]奪得[[香港甲組足球聯賽|甲組聯賽]]及[[香港足球高級銀牌賽|銀牌]]雙料冠軍後原打算退役，其後獲[[東方足球隊|東方]]邀請加盟一年，至1972年正式掛靴，並擔任該隊教練一職。

== 掛靴生活 ==
掛靴之後，林尚義曾先後在[[香港電台|香港電台]]及[[商業電台|商業電台]]擔任足球評述員，其中最為人熟悉的賽事可算1985年5月19日舉行的[[1986年世界盃|1986年世界盃]]外圍賽亞洲區第四組A，[[香港足球代表隊|港隊]]在[[北京工人體育場|北京工人體育場]]以二比一擊敗主場的[[中國國家足球隊|中華人民共和國隊]]出線，當時任職商台的林尚義與拍擋[[蔡文堅|蔡文堅]]現場透過長途電話作評述<ref>「特稿：難忘現場評述五‧一九」《蘋果日報》，2008年1月3日</ref>。

在八十年代後期，他兼任[[亞洲電視|亞洲電視]]的足球評述，後來並與[[黃霑|黃霑]]等一同在[[亞洲電視|亞洲電視]]主持旁述[[1990年世界杯足球賽|1990年世界杯足球賽]]<ref>「阿叔倦了林尚義」《壹週刊》，2006年6月29日</ref><ref>參見：亞洲電視，《[http://www.youtube.com/watch?v=PihP0IUkLu0 90世界盃全接觸]》1990年7月4日</ref>。當年[[亞洲電視|亞洲電視]]的世界杯評述贏盡口碑，林尚義的專業評述功不可沒。

翌年林尚義轉投[[無綫電視|無綫電視]]<ref>「藝人讚講波專業風趣」《大公報》，2009年4月24日</ref> 擔任足球節目《[[球迷世界|球迷世界]]》主持，其中他所主持的環節「䱽魚話當年」，訪問不少香港足球名宿，回顧球壇歷史。

林尚義獨特的聲線及評述風格亦漸為人熟悉，令他不時獲邀客串廣告、電視劇與電影演出，其中以《[[古惑仔電影系列|古惑仔電影系列]]》飾演林[[牧師|牧師]]一角，最為人津津樂道。

== 婚姻生活 ==
林尚義曾有過兩段婚姻，23歲時與首任妻子結婚，而第二任妻子任職《[[成報|成報]]》法庭版編輯的劉蕙芳，2005年因急性[[肝硬化|肝硬化]]引致內出血身故，終年43歲。妻子突然離世，令林尚義大受打擊，並決定於2005年8月中與[[無綫電視|無綫電視]]合約屆滿後退休，及後被無綫挽留至[[2006年世界盃|2006年世界盃]]<ref>「阿叔講波成絕響 傳奇球評家林尚義病逝終年74歲」《星島日報》，2009年4月24日</ref>。2006年7月10日凌晨，林尚義在[[奧海城|奧海城]]完成[[2006年世界盃|2006年世界盃]]決賽旁述後，正式宣佈退休，當晚無綫在直播球賽前後均以他榮休為節目內容。同年，林尚義接受醫生建議成功戒煙，結束數十年的吸煙習慣。

2009年4月23日早上11時，林尚義被兒子發現在[[天后_(香港)|天后]][[僑興大廈|僑興大廈]]住所內昏迷不醒，被送往[[律敦治醫院|律敦治醫院]]搶救，延至11時58分證實不治，終年74歲<ref name="mp_death" /><ref>「一代講波宗師別了」《東方日報》，2009年4月24日</ref>。

== 軼事 ==
1998年7月3日，林尚義、[[韓毓霞|韓毓霞]]及[[郭家明|郭家明]]在[[法國|法國]][[法蘭西體育場|法蘭西體育場]]，現場旁述[[1998年世界盃足球賽|1998年世界盃足球賽]]八強賽由[[法國國家足球隊|法國]]迎戰[[意大利國家足球隊|意大利]]的賽事。中場休息期間，身處[[香港|香港]][[電視城|電視城]]的主持[[李克勤|李克勤]]與[[郭家明|郭家明]]連線評述兩隊上半場的表現，當時電視鏡頭拍攝到旁邊的林尚義從口袋裡拿出煙包，並抽出一根含在口中準備點燃。當時身處電視城的主持及嘉賓見狀即時起哄，林尚義可能因為聽到起哄聲以及郭家明的示意而意識到正在直播，愕了一愣即放下煙支。李克勤在廣告時段開始前，向法國現場的三人說了一句：「家明！家明！我哋俾啲時間阿叔食少少嘢！睇睇現場嘅精華片段。」（家明！家明！我們給點時間阿叔「吃」點東西！看看現場的精華片段）揶揄林尚義在直播節目吸煙，此事曾引起觀眾熱烈討論<ref>參見：無綫電視翡翠台[https://www.youtube.com/watch?v=94R-qm8MvkI 《98世界盃-列強爭霸》]1998年7月3日</ref>。

== 悼念 ==
[[無綫電視|無綫電視]]於2009年4月25日播出的體育節目《[[體育世界|體育世界]]》及5月3日播出的《球迷世界》在節目中，剪輯了林尚義主持體育節目的片段作最後致敬。
