'''林宇軒''' {{BD|1999年|}}，[[臺灣|臺灣]][[新北市|新北市]]人。臺灣新生代詩人。[[臺北市立成功高中|臺北市立成功高中]]畢業<ref>{{Cite web |url= https://issuu.com/xihan/docs/______19_________ |title= 有荷文學雜誌第19期〈文學鮮人：林宇軒〉 |author=  喜菡主編  |date=2016年 |publisher= 高雄:大憨蓮文化工作室 |language=  |accessdate=  |quote=  }}</ref>，現就讀於[[國立臺灣師範大學|國立臺灣師範大學]]社會教育學系。作品以現代詩為主，曾獲全國優秀青年詩人獎、香港青年文學獎、金車現代詩獎等。現任喜菡文學網新詩版主、台灣詩學吹鼓吹詩論壇分行詩版主、臺師大噴泉現代詩社文字編輯。文學創作散見[[聯合報|聯合報]]副刊、[[中國時報|中國時報]]副刊、[[自由時報|自由時報]]副刊、人間福報副刊、[[創世紀詩刊|創世紀詩刊]]、野薑花詩刊等<ref>{{Cite web |url= https://www.mirrormedia.mg/story/20180726cul001/ |title= 鏡文化副刊-為你朗讀林宇軒〈迴圈〉 |author= 鏡週刊 |date=2018年 |publisher= 鏡傳媒 |language= 中文 |accessdate=  |quote=  }}</ref>
<ref>{{Cite web |url= https://www.mirrormedia.mg/story/20190201cul001/ |title= 鏡文化副刊-為你朗讀林宇軒〈人間動物園〉 |author= 鏡週刊 |date=2019年 |publisher= 鏡傳媒 |language= 中文 |accessdate=  |quote=  }}</ref>。著有詩集《泥盆紀》。
