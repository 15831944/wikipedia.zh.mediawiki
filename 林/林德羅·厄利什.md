林德羅·厄利什（{{lang|en|'''Leandro Erlich'''}}，{{bd|1973年|4月8日|||catIdx=E}}），出生於[[阿根廷|阿根廷]]的首都[[布宜諾斯艾利斯|布宜諾斯艾利斯]]，是阿根廷的[[觀念藝術|觀念藝術]]家。

== 生平 ==
曾在2001年的[[威尼斯雙年展|威尼斯雙年展]]於阿根廷國家館展出作品《泳池》（The Swimming Pool），並在2005年被邀請至主題館展出<ref>http://www.skny.com/artists/leandro-erlich/</ref><ref>{{Cite web  |title = 泳池天台爆爭議 無限期關閉 |publisher = 中國時報 |url = http://www.chinatimes.com/newspapers/20150720000403-260115 |date = 2015-07-20 |author= 吳垠慧、吳江泉 |accessdate = 2015-07-21 |language = zh-hant}}</ref>。並在2008年永久陳列於美國的{{tsl|en|MoMA PS1|MoMA PS1}}<ref>http://momaps1.org/exhibitions/view/207</ref>。以及在2000年{{tsl|en|Whitney Biennial|惠特尼雙年展}}和2001年{{tsl|en|Istanbul Biennial|伊斯坦堡雙年展}}邀請展出<ref>http://www.artnet.com/artists/leandro-erlich/</ref>。而臺灣首次邀請厄利什是在2015年於[[衛武營國家藝術文化中心|衛武營國家藝術文化中心]]園區<ref>{{Cite web  |title = 體驗當蛛蜘人 衛武營童樂節7/3招手 |publisher = 自由時報 |url = http://news.ltn.com.tw/news/life/breakingnews/1366024 |date = 2015-07-01 |author= 陳文嬋 |accessdate = 2015-07-21 |language = zh-hant}}</ref>。
