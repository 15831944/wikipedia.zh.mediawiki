'''林昭'''（{{bd|1932年|1月23日|1968年|4月29日|catIdx=L}}<ref>{{cite news |title=采访圣女林昭的“未婚夫”张元勋 |url=http://blog.dwnews.com/post-385007.html |accessdate=2018-10-31 |work=多维新闻 |date=2013-11-16}}</ref>{{efn|张元勋在接受采访时表示，林昭是生于1932年农历的12月16日，即公历1月23日}}{{efn|林昭的出生日期尚存疑问。在1964年上海市静安区人民检察院起诉书中写林昭当时为32岁，但林昭本人对此曾批注“应为三十岁”。}}），原名'''彭令昭'''，中国[[苏州|苏州]]人。林昭1954年入北京大学新闻系学习，在1957年的[[反右运动|反右运动]]中因公开支持[[北京大學|北京大學]]学生[[張元勛|張元勛]]的[[大字報|大字報]]“是时候了”而被劃為[[右派|右派]]，后因“攻击[[无产阶级专政|无产阶级专政]]罪、[[反革命|反革命]]集团罪”<ref>1968年4月19日的《中国人民解放军上海市公检法军事管制委员会刑事判决书》</ref>于1962年起被关押於[[上海市提篮桥监狱|上海市提篮桥监狱]]，在獄中書寫了<!--二十万字(?)的-->反对[[毛泽东|毛泽东]]的血書與日記<!--（20万字血书相当于50个成年人的血量）-->。<ref>《枉言正道是沧桑》</ref>1968年4月29日林昭在狱中当着众多在押人员的面被宣判死刑，同日被[[中国人民解放军|中国人民解放军]]上海市公检法[[军事管制委员会|军事管制委员会]]枪决于[[上海龙华机场|上海龙华机场]]（[[张春桥|张春桥]]时任[[南京军区|南京军区]]兼[[上海警备区|上海警备区]]第一政治委员）。

== 家庭 ==
1932年，林昭出生于[[江苏|江苏]][[苏州|苏州]]。其父亲[[彭国彦|彭国彦]]曾到[[英国|英国]]留学，1922考入[[國立中央大學_(南京)|国立東南大學]]，主修政治經濟，1926年畢業論文是《愛爾蘭自由邦憲法述評》，1928年9月，在[[國民政府|國民政府]]舉辦的第一屆縣長考試中獲第一名，被任命為蘇州吳縣縣長。做过[[吴县|吴县]]、[[江阴|江阴]]等县[[县长|县长]]。

母亲[[许宪民|许宪民]]曾任苏州《[[大华报|大华报]]》总经理，[[民盟|民盟]]成员，支持中共，曾秘密为中共捐款，建立地下电台，还曾被日本人逮捕。1946年，国民政府举行[[國民大會|國民大會]]代表竞选，许宪民当选國大代表。林昭父母时常为了该给女儿教授哪种政治价值观而争吵。

大舅舅[[许金元|许金元]]曾任[[中共江苏省委|中共江苏省委]]青年部长，[[四·一二事件|四·一二事件]]中，被国民党处决沉尸长江。

== 革命 ==
[[File:Linzhao-1950.jpg|thumb]]石门]]
林昭1943年考入[[蘇州|蘇州]]萃英中學。<ref>{{Cite web | url=http://www.oac.cdlib.org/findaid/ark:/13030/kt9c60388k/admin/#a2 | title=''Biographical/Historical Note'' | publisher=The Online Archive of California}}</ref>她受母親的影响，对[[中国共产党|中国共产党]]的革命行为抱有很大的热情，在景海高中毕业后，林昭不顾父亲反对，于1949年7月考進“革命摇篮”[[苏南新闻专科学校|苏南新闻专科学校]]，决心“与家庭生不来往，死不吊孝”，投身到革命中去，甚至曾经无中生有地揭发过自己的父亲，多年后，林昭对此感到很不安：“他们要我井裡死也好，河裡死也好，逼得我没办法，写了些自己也不知道的东西，我不得不满足他们……我没存心诬陷你”。<ref>{{cite web |url = http://www.aisixiang.com/data/46673.html |title = 林昭之死 |author = 陈伟斯 |date = 1981年第3期 |language = zh-hans |publisher = 《[[民主与法制|民主与法制]]》}}</ref>

毕业后林昭随[[苏南|苏南]]农村工作团参加苏南农村[[土改|土改]]。1952年开始在《[[常州民报|常州民报]]》、[[常州|常州]][[文联|文联]]工作。2004年8月11日《[[冰点_(中国青年报)|冰点]]》的《寻找林昭》文章称：土改队努力地工作着。为了让农民看到工作队的权威和力量，他们将地主放在冬天的水缸裡，冻得彻夜嚎叫。林昭把这称为“冷酷的痛快”，认为只有这样的斗争，才能够显示改革的决心，灭掉地主的威风。

為了與身為國民黨的父親劃清界線，本名彭令昭的林昭抛棄父姓，改名為林昭。

== 北大 ==
1954年，林昭以江苏省第一名的成绩考入了[[北京大学|北京大学]]中文系新闻专业，立志作[[毛澤東|毛澤東]]時代最好的[[記者|記者]]。林昭在北大瘋狂地閱讀許多她喜歡的書，同學常看到她從圖書館抱出許多線裝書。她觀察到現實并非她想像般美好，因而陷入“爱与恨的一盆糨糊”。但在北大自由的學風中，她開始成長，開始思考。在想到自己曾亲自揭发母亲的“罪行”时，她痛苦得哭出来，写信给母亲发誓说：“今后宁可到河里、井里去死，絕不再说违心话！”

由于林昭勤学多思，受到[[游国恩|游国恩]]教授的赞赏，建议林昭调入文学专业，未果。林昭与另一位才女[[张玲|张玲]]任校刊编辑，负责[[副刊|副刊]]《未名湖》。1955年春，林昭参加了[[北大诗社|北大诗社]]，任《北大诗刊》编辑。1956年秋，《北大诗刊》停办后，林昭成为综合学生文艺刊物《[[紅樓_(雜誌)|紅樓]]》的编委会成员之一，被称为“红楼裡的林姑娘”。该刊物主编是[[乐黛云|乐黛云]]。《红楼》第2期的责任编辑是林昭和[[张元勋_(曲阜师范大学)|张元勋]]。

1957年5月19日，张元勋等贴出大字报《是时候了！》，这是为了响应中央的[[百花運動|鸣放号召]]，随后几天北大的大字报越来越多，学生互相辩论，有人认为大字报中的右倾言论是反革命煽动。5月22日，林昭在辩论中公开反对那些上纲上线的批评，并说：「我料到一旦说话也就会遭到像今晚这样的讨伐！我一直觉得组织性与良心在矛盾着！」{{efn|據胡-{杰}-考证，林昭并非赞同张元勋言论而替他辯護。她感到痛苦而无法理解的是，一些有思想、敢作敢为的同学被说成“疯子”和“魔鬼”。这段期間她在日记裡說：“党啊，你是我们的母亲！母亲应该最知道孩子的心情！尽管孩子过于偏激，说错了话，怎么能说孩子怀有敌意呢？”}}<ref>{{cite web |url = http://www.taosl.net/dir2/memo_linzhao_moluo.htm |title = 林昭年表 |author = 摩罗 }}</ref>5月29日，《红楼》编辑部举行会议，宣布將张元勋与[[李任|李任]]开除出《红楼》编委会，原因是他们参加右派刊物《广场》编委会，林昭在发言批判时对张元勋说：「我有受骗的感觉！」<ref>{{cite web |url = http://www.taosl.net/wcp/memo_linzhao_zhangyx.htm |title = 北大往事与林昭之死|author =张元勋 |language = zh-hans}}</ref>6月8日，《[[人民日报|人民日报]]》发表[[社论|社论]]《这是为什么？》，将提意见的言论说成是右派分子乘机向黨进攻。<ref>{{cite news |url = http://news.xinhuanet.com/ziliao/2004-12/31/content_2401225.htm |title = 这是为什么？|accessdate = 2009年2月 |newspaper = 《[[人民日报|人民日报]]》 |language = zh-hans |date = 1957年6月8日}}</ref>

1957年秋，张元勋、林昭等人[[反右運動|被打成右派分子]]，林昭吞服大量[[安眠药|安眠药]][[自杀|自杀]]，但被及时抢救过来。于是她被认定为对抗组织、“态度恶劣”，遭到加重处分：[[劳动教养|劳动教养]]三年。林昭不服，跑到团中央质问：“当年[[蔡元培|蔡元培]]先生在北大任校长时，曾慨然向[[北洋政府|北洋军阀政府]]去保释‘五四’被捕的学生，现在他们（指北大领导）却把学生送进去，良知何在？”<ref>{{cite web |url = http://www.taosl.net/wcp/memo_linzhao_zhangyx.htm |title =北大往事与林昭之死 |author = 张元勋 |language = zh-hans }}</ref> 

1957年12月25日，张元勋被秘密逮捕，判处有期徒刑8年。北大当时约有八千学子，其中约有1500名师生被打成右派，他们中的许多人，被开除公职与学籍，发配到边疆荒野，20多年後才得以平反。<ref>{{cite web |url = http://www.21ccom.net/articles/lsjd/lccz/article_2010082616807_3.html |author=张元勋 |title = 北大往事与林昭之死(3) |date=2010-08-26|publisher=[[共识网|共识网]]|archiveurl=https://web.archive.org/web/20160304195500/http://www.21ccom.net/articles/lsjd/lccz/article_2010082616807_3.html|archivedate=2016-03-04|deadurl=yes}}</ref><ref>{{cite web |url = http://news.k618.cn/xda/201304/t20130429_3167251_4.html |title=林昭是谁？哪里人？个人资料 林昭案件细节始终 |editor = 小文 |date=2013-04-29 |publisher=未来网}}</ref>

== 人民大学 ==
[[File:Linzhao-1959-Beijing.png|thumb]]]]
1958年6月北京大学中文系新闻专业并到[[中国人民大学|中国人民大学]]新闻系，林昭也从北京大学到了中国人民大学。北京大学中文系新闻专业主任[[罗列_(新闻学家)|罗列]]到人大新闻系来後是副主任，正主任是[[安岗|安岗]]。林昭是被打成右派分子后由罗列带过来的，她是北大第一批右派分子。后因罗列怜其体弱多病，冒险为之说情，林昭得以留在中国人民大学新闻专业资料室接受群众「监督改造」。林昭在新闻系资料室监督劳动期间与同在资料室“劳动考察”的人大学生「右派分子」[[甘粹|甘粹]]产生爱情，他们提出结婚申请，但上级批评他们谈情说爱是抗拒改造，不准他们结婚。<ref>{{Cite news |url = http://news.ifeng.com/history/zhongguoxiandaishi/201001/0125_7179_1524406.shtml |title=林昭情人的口述：两个右派份子从相识 |date=2010-01-25}}</ref>

1959年9月，甘粹被发配到[[新疆|新疆]]进行[[劳动改造|劳动改造]]。林昭病情加重，冬天咳血加剧，请假要求回上海休养。1960年春，[[中国人民大学|中国人民大学]][[校长|校长]][[吴玉章|吴玉章]]批示准假，林昭由母亲接回上海。

== 上海 ==
{{Wikisource|上海市静安区人民检察院（64）沪静检诉字第423号起诉书|上海市静安区人民检察院起诉书（1964年）}}

1959年左右，通过调养，林昭病情渐有好转。在上海养病期间，受到江浙一代结社文化的影响，并认识了[[兰州大学|兰州大学]]的[[研究生|研究生]]顾雁、[[徐诚|徐诚]]，当时兰大的[[张春元|张春元]]{{efn|[[张春元|张春元]]，男，在[[兰州大学|兰州大学]]读书期间被打成“[[右派|右派]]”，1958年被下放到[[甘肃|甘肃]]的偏远[[农村|农村]]参加苦力劳动。在下放期间与右派同学自办“《[[星火|星火]]》杂志”，1960年9月30日被[[逮捕|逮捕]]，被关在专门收押重刑犯的[[甘肃省第三监狱|甘肃省第三监狱]]。1968年被诬指“密谋暴动越狱”，被判处死刑，立即枪决。}}等人，正在准备筹办针砭时弊的《[[星火|星火]]》杂志，随后林昭的长诗《海鸥之歌》和《普鲁米修斯受难之日》，在《星火》第一期上发表。但很快涉及《星火》的人员，都被抓捕。1960年10月，林昭被逮捕入狱。<ref>{{cite news |url =http://news.163.com/12/0429/11/808MHI1200012Q9L.html |title=有的人永远不会被忘记，纪念林昭逝世44周年 |author=江菲 |publisher=《[[中國青年報|中國青年報]]》}}</ref>

1962年初，林昭得以[[保外就医|保外就医]]。9月，林昭在苏州与黄政商量并且起草了“中国自由青年战斗同盟”的纲领和章程。期间还曾要求上海的[[无国籍|无国籍]]侨民阿诺，将《我们是无罪的》、《给北大校长陆平的信》等带到海外发表。<ref>{{cite news |url = http://www.infzm.com/content/4325 |title = 【往事】读林昭十四万言书 |author=傅国涌 |location = 杭州 |publisher = 《[[南方周末|南方周末]]》 |date=2008-04-30}}</ref>

林昭的母親在六十年代[[受洗|受洗]]，有一批[[基督徒|基督徒]]會到林昭家唸聖經。林昭成為[[基督徒|基督徒]]的確切時間不明。林昭1963年的寫作使用「[[主曆|主曆]]」一詞，可以證明林昭在1963年的时候已經信[[耶稣|耶稣]]。<ref name="彭令范訪談錄">{{cite web|url=http://www.chinesepen.org/Article/sxsy/200804/Article_20080429040845.shtml|title=林昭胞妹彭令范訪談錄|author=張敏|publisher=自由亞洲電台|date=2008-04-29|quote=張：您信主耶穌是在出國以後，大概是在哪年？彭：1985年。我來美國後不久。|deadurl=yes|archiveurl=https://web.archive.org/web/20101227071811/http://www.chinesepen.org/Article/sxsy/200804/Article_20080429040845.shtml|archivedate=2010-12-27}}</ref>

1962年12月，林昭又被捕入狱。在狱中林昭曾多次[[绝食|绝食]]、自杀，并且分别两次给当时的上海市长[[柯庆施|柯庆施]]、《人民日报》写信，反映案情并表达政治见解，都没有-{回}-音。林昭在狱中，因为没有笔纸，便用血在白色的被单上写作。<ref>{{cite web |url = http://www.taosl.net/wc102.htm |title = 我的姐姐林昭 |accessdate = |date = |author = 彭令范，林昭之妹 |language = zh-hans |quote = 姐姐每次来信，总是要白被单，我们实在百思不得其解。到后来才知道，送去的白被单她都撕成条条用来写血书。”……“姐姐一度保外治疗时，我们问她为什么要这么多的白被单，她支吾其辞。当我们看到她手腕部血迹斑驳的伤痕时，母亲立即把她衣袖拉起来，手臂上也全是小的切口疤痕。母亲当时放声大哭：‘你为什么要这样作贱自己？这也是我的血肉呀！’”}}</ref>另外，由于林昭拒绝违心地服从，被狱卒视为表现恶劣，遭受较严重的虐待，林昭在血书中写到：“光是镣铐一事人们就玩出了不知多少花样来：一副反铐，两副反铐，时而平行，时而交叉，等等不一。臂肘之上至今创痕犹在不消说了，最最惨无人道酷无人理的是：不论在我绝食之中，在我[[胃炎|胃炎]]发病痛得死去活来之时，乃至在[[月经|月经]]期间，不仅从未为我解除过镣铐，甚至从未有所减轻！--比如在两副镣铐中暂且除去一副”。<ref>{{cite web |url = http://www.oioj.net/blog/user1/8879/archives/2005/47469.shtml |title = 林昭在为我们寻找——《寻找林昭的灵魂》观后 |accessdate = 2009年2月 |date =  |author = 林达 |language = zh-hans |deadurl = yes |archiveurl = https://web.archive.org/web/20130708082224/http://www.oioj.net/blog/user1/8879/archives/2005/47469.shtml |archivedate = 2013年7月8日 |df =  }}</ref>1965年3月23日，林昭开始在狱中写《告[[人类|人类]]》。<ref>{{cite web |url= http://b5.secretchina.com/news/13/04/30/495627.html?%E8%BF%BD%E6%80%9D%E6%9E%97%E6%98%AD |title=追思林昭 |author=唐師曾 |publisher=看中國}}</ref>

1965年5月31日，开庭审判，林昭被判有期徒刑20年。林昭随后血书《判决后的申明》。《判决后的申明》部分内容：{{quote|……这是一个可耻的判决，但我骄傲地听取了它！这是敌人对于我个人战斗行为的一种估价，我为之由衷地感到战斗者的自豪！……我应该作得更多，以符合你们的估价！除此以外，这所谓的判决与我可谓毫无意义！我蔑视它！看着吧！历史法庭的正式判决很快即将昭告于后世！你们这些极权统治者和诈伪的奸佞--歹徒、恶赖、窃国盗和殃民贼{{efn|指[[毛泽东|毛泽东]]（外界推测）}}）将不仅是真正的被告更是公诉的罪人！公义必胜！自由万岁！林昭 主历 一九六五年六月一日}}

1966年5月6日，北大同学[[张元勋_(曲阜师范大学)|张元勋]]来到上海，同林昭母亲许宪民到[[上海市提篮桥监狱|上海市提篮桥监狱]]看望她。<ref>{{cite news|url=http://www.worldjournal.com/view/full_news/23428590/article--%E9%81%A0%E6%9D%B1%E7%AC%AC%E4%B8%80%E7%9B%A3%E7%8D%84-%E5%92%8C%E5%AE%83%E7%9A%84%E7%99%BE%E5%B9%B4%E9%81%8E%E5%AE%A2|title=「遠東第一監獄」和它的百年過客|author1=雷磊|author2=李純|author3=廖梅|date=August 25, 2013 06:00 AM|publisher=南方週末}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>[[文化大革命|文化大革命]]爆发后，在中央的直接批示和干预之下，林昭被列为应该[[处决|处决]]的[[反革命分子|反革命分子]]。

1968年4月29日，林昭接到被[[上海市高级人民法院|上海市高级人民法院]]改判死刑的判决书后，随即被绑赴[[上海龙华机场|上海龙华机场]]执行[[枪决|枪决]]，终年36岁。5月1日，[[公安|公安]]人员来到林昭母亲家，索取5分钱子弹费<ref>{{cite news |url = http://news.ifeng.com/history/zhongguoxiandaishi/detail_2013_04/29/24804544_0.shtml |publisher=新华网 |title=林昭文革中被杀害 其母被逼交5分钱子弹费 |date=2013年4月29日 |author1=穆青 |author2=郭超人 |author3=陆拂}}</ref>（事件在1981年報道[[林彪、江青反革命集团案|林彪、江青反革命集团案]]审判的《歷史的審判》一文中提到）。

林昭被捕一個月后，其父服药自杀；其母则精神失常，后因醫院拒絕醫治，1975年在上海[[外灘|外灘]]自殺。<ref>據《蘇州市志》第三冊1222頁</ref>林昭弟弟彭恩华，2004年8月3日逝世于美国[[犹他州|犹他州]][[桑迪_(猶他州)|桑迪市]]，享年59岁。<ref>{{Cite web |url = http://www.legacy.com/obituaries/saltlaketribune/obituary.aspx?n=edward-enhua-peng&pid=2485987 |title=''Edward Enhua Peng Obituary'' |publisher = Salt Lake Tribune |date=2004-08-05}}{{en}}</ref><ref>Salt Lake Tribune(盐湖城论坛报), August 5-6, 2004.</ref>林昭的妹妹彭令范1980年後移居美国。<ref name="彭令范訪談錄"/><ref>{{cite news |url = http://www.rfa.org/mandarin/zhuanlan/xinlingzhilyu/shengminghuiyinbi/linzhao-20050228.html |title = 林昭：回忆、纪念与研究 |date=2005-02-28 | publisher=自由亞洲電台}}</ref>

1980年8月22日，上海市高级人民法院“沪高刑复字第435号”刑事判决书，撤销[[上海市静安区人民法院|上海市静安区人民法院]]1962年度静刑字第171号和中国人民解放军上海市公检法军事管制委员会1967年度沪中刑字第16号两次判决，宣布林昭以精神病為由平反為无罪，结论为“这是一次冤杀无辜”；而1981年1月25日上海高院的“沪高刑申字第2346号”刑事判决书中认为80年的判决书宣告无罪的理由为精神病不妥，“在病发期间的行为不应以反革命罪论处……林昭的行为既不构成犯罪……沪高刑复字第435号判决在适用法律上亦属不当，均应与前两个判决一并予以纠正”，撤销了1980年“沪高刑复字第435号”判决书，但依旧对林昭宣告无罪，内有1981年沪高刑申字第2346号判决书。

2004年4月22日，林昭骨灰由苏南新闻专科学校與北大部分師生集資立碑，並被安葬在江苏省[[苏州市|苏州市]][[木渎镇|木渎镇]][[灵岩山|灵岩山]]的安息公墓。林昭的尸体至今下落不明，墓里只保留着林昭生前的一件衣服和一缕头发。<ref>{{cite news |url=http://www.worldchineseweekly.com/a/renwu/redianrenwu/2013/0513/21875.html |title=黑暗中點燃的那一星火光--紀念林昭逝世四十五週年 |date=2013-05-13 |publisher=世界華人週刊 |author=楚寒 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url = http://boxun.com/news/gb/pubvp/2005/05/200505140212.shtml |author=小乔 |title=追寻一个圣洁的灵魂——感悟林昭(图) |publisher=博讯 |date=2005年5月14日)}}</ref>林昭墓碑背面鐫有她1964年寫的詩句「自由無價，生命有涯，寧為玉碎，以殉中華。」<ref>{{Cite web|url=http://www.mingpaomonthly.com/cfm/Archive2.cfm?File=200407/feature/01a.txt|title=解讀林昭之死|author=[[丁子霖|丁子霖]]|date=2004年7月|work=明報月刊|deadurl=yes|archiveurl=https://web.archive.org/web/20150318073803/http://www.mingpaomonthly.com/cfm/Archive2.cfm?File=200407%2Ffeature%2F01a.txt|archivedate=2015-03-18}}</ref>

2003年，中国独立制片人[[胡杰|胡杰]]把他過去五年中亲自尋訪認識林昭本人的80人的錄像纪录，更通过特別途徑拍攝到林昭獄中文稿，完成了[[纪录片|纪录片]]《尋找林昭的靈魂》；其相關採訪《寻找林昭》刊登於2004年8月11日《[[冰点_(中国青年报)|冰點]]》，<ref>{{cite web |url = http://zqb.cyol.com/content/2004-08/11/content_926077.htm |title =寻找林昭| date = 2004-08-11|publisher=中国青年报}}</ref>年終時因題材關係，另外「文氣豐沛，貫穿首尾」，被[[章詒和|章詒和]]評為她所看過描寫林昭最好的一篇，因而得2004年度《冰点》周刊“最佳特稿奖”。

2005年，[[独立中文笔会|独立中文笔会]]特设“林昭纪念奖”，纪念这位思想先驱和自由斗士。<ref>{{Cite web |url = http://www.canyu.org/n15282c12.aspx | title=“因作家畅所欲言”五十年中国第一案：林昭（1960）|publisher=独立中文笔会}}</ref>

林昭的档案，包括在狱中写的大量血书，1980年代曾一度开放，但不久又被封存。2009年4月，林昭的档案原件由胞妹彭令范捐赠給[[史丹福大學|史丹福大學]][[胡佛研究所圖書檔案館|胡佛研究所圖書檔案館]]，2009年10月26日首次公开档案電子版。<ref>{{cite news |url = http://www.rfa.org/mandarin/yataibaodao/linzhao-10282009093317.html |title=RFA独家：林昭遗稿原件由胞妹彭令范捐赠斯坦福大学（组图）| publisher=自由亞洲電台}}</ref>档案原件包括狱中血书，公開信，私人信件，以及家庭照等。<ref>{{cite web |url = http://www.hoover.org/news/letters-and-diaries-chinese-political-activist-lin-zhao-opened-hoover-institution-archives |title = ''Letters and diaries of Chinese political activist Lin Zhao opened at the Hoover Institution Archives | publisher=Hoover Institution |date=November 11, 2009}}{{en}}</ref>網上有档案目錄，档案電子版可供研究。<ref>{{cite web |url = http://www.oac.cdlib.org/findaid/ark:/13030/kt9c60388k/ |title=Inventory of the Lin Zhao papers |publisher=Online Archive of California}}{{en}}</ref>

2012年12月16日，各地网友12月16日在苏州灵岩山林昭墓前举行活动，纪念林昭的80冥诞。据现场照片显示，有十多位[[公安部国内安全保卫局|国保]]便衣在网友纪念过程中监视并錄影。<ref>{{cite news |author=海彦 |title=网友聚集苏州纪念公民先驱林昭八十冥诞 |url = http://www.voachinese.com/content/chinese-activists-celebrate-the-birthday-of-lin-zhao-20121216/1565894.html#hash=relatedInfoContainer |accessdate=2013-05-09 |publisher = [[美国之音|美国之音]] |date=2012-12-16}}</ref>

2013年4月29日，林昭遇难45周年之际，有人拜祭林昭受到当地政府人员阻挠。<ref>{{cite web |title=广东律师刘士辉拜祭林昭遭人殴打 |url = http://www.bbc.co.uk/zhongwen/simp/china/2013/04/130429_liushihui_beaten_linzhao.shtml |publisher=BBC中文网 |accessdate=2013-04-29}}</ref><ref>{{cite news |title=“林昭精神表达了被凌辱中国人的追求”|url = http://www.bbc.co.uk/zhongwen/simp/china/2013/04/130429_li_datong_lin_zhao.shtml |accessdate=2013-04-30 |newspaper=BBC中文网 |date=2013-04-29}}</ref>

2016年4月28日，林昭祭日前一天，中国各地网民、维权人士等为避开政府干扰，选择提前一天来墓地祭林昭，但遭到公安封山及抓人。<ref>{{Cite web|url=http://www.rfa.org/mandarin/yataibaodao/renquanfazhi/ql2-04302016100946.html|title=林昭周年祭日网民灵岩山祭奠受阻|accessdate=2017-02-23|work=Radio Free Asia|language=zh-cn}}</ref>
