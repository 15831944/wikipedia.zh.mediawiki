'''林書緯'''（{{bd|1992年|6月21日|catIdx=林|catIdx=Lin, Joseph}}），生於[[美國|美國]]。[[男子|男子]][[籃球|籃球]][[運動員|運動員]]，是現役[[NBA|NBA]]球星[[林書豪|林書豪]]的弟弟，現效力於SBL聯盟的[[富邦勇士|富邦勇士]]，場上位置為[[控球後衛|控球後衛]]。

==籃球生涯==
林書緯大學時期就讀漢米爾頓學院經濟系並代表學校參加NCAA第三級籃球賽，主要位置是控球後衛，曾入選聯盟的年度第二隊<ref>{{cite web |language=zh-hant |url=http://www.setn.com/News.aspx?NewsID=88924| title=SBL／「豪弟」林書緯加盟勇士　為聯盟注入新活力| publisher=三立新聞 |date=2015-08-07 |accessdate=}}</ref>。

2015年自漢米爾學院畢業後加盟[[SBL|SBL]][[富邦勇士|富邦勇士]]，於SBL夏季聯盟以平均12.1分、4.1籃板、4.4助攻獲得夏聯助攻王、夏聯最佳新秀及夏聯最佳五人<ref>{{cite web |language=zh-hant |url=http://sports.ltn.com.tw/news/breakingnews/1443633| title=SBL夏季聯賽》台版林來瘋 林書緯獲夏聯三大獎（影音）| publisher=自由時報 |date=2015-09-13 |accessdate=}}</ref>。開季後林書緯也延續夏季聯盟優異的表現，在[[第十三季超級籃球聯賽|SBL第十三季]]以平均12.1分、4.1籃板、4.4助攻、1.4抄截的獲得成績年度助攻王、年度新人王及年度第一隊的榮譽<ref>{{cite web |language=zh-hant |url=http://sports.ltn.com.tw/news/breakingnews/1653571| title=SBL》新人王入選最佳5人 林書緯史上頭一遭| publisher=自由時報 |date=2016-04-03 |accessdate=}}</ref>。

林書緯在[[第十四季超級籃球聯賽|SBL第十四季]]的明星賽與富邦隊友魏維攜手贏得技術挑戰賽冠軍<ref>{{cite web |language=zh-hant |url=http://www.tsna.com.tw/tw/news/show.php?num=7582| title=技術挑戰賽　林書緯、魏維奪冠| publisher=TSNA運動新聞 |date=2017-01-21 |accessdate=}}</ref>。
