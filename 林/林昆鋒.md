'''林昆鋒'''（{{bd|1975年|11月20日|||林}}），台灣新聞主播，曾在[[年代新聞|年代新聞]]擔任《[[年代高峰會|年代高峰會]]》節目主持人。

== 家庭 ==
2006年3月，林昆鋒在[[公關公司|公關公司]]春酒會場上認識[[黃文華|黃文華]]，經過一段時間的交往後，林昆鋒在9月主動向黃文華求婚。<ref name="許晉榮"/>2007年1月8日，林昆鋒前往[[新竹市|新竹市]]迎娶黃文華，並於[[新竹國賓大飯店|新竹國賓大飯店]]完成訂婚。<ref name="竹市府"/>2007年1月20日，兩人在[[台北華漾飯店|台北華漾飯店]]結婚宴客。<ref name="許晉榮">{{cite web |url=http://ent.appledaily.com.tw/enews/article/entertainment/20070121/3196868/ |title=主播黃文華 林昆鋒婚了 義大利蜜月連做11天 |work=蘋果日報 娛樂新聞 |author=許晉榮 |publisher=台灣蘋果日報 |accessdate=2014-02-20 |language=zh-tw |date=2007-01-21 }}</ref>現育有二子一女。<ref name="黃鋒府">{{cite web |url=http://ent.appledaily.com.tw/enews/article/entertainment/20110729/33561507/ |title=黃文華主播夫妻育兩兒 4管齊下 調教動靜雙修 |work=蘋果日報 娛樂新聞 |author=黃鋒府 |publisher=台灣蘋果日報 |accessdate=2014-02-20 |language=zh-tw |date=2011-07-29 }}</ref>
== 經歷 ==
2015年5月，林昆鋒以第一名成績，録取「台大國家發展研究所碩士班」，老婆、東森主播黃文華隔空祝賀，甜蜜送飛吻<ref>{{cite news|author= |title=老公考上研究所 女主播大方獻愛|url=http://ent.ltn.com.tw/news/breakingnews/1317322|language=zh-hant|accessdate=2015-05-14|newspaper=《[[自由時報|自由時報]]》 |date=2018-12-22}}</ref>。
== 從政經歷 ==
2017年5月，根據《[[蘋果日報_(台灣)|蘋果日報]]》獨家報導，台北市政府內部不少人盛傳林昆鋒將入府負責新聞媒體操作，甚至擔任發言人工作<ref>{{cite news|author=陳思豪 |title=《獨家》柯Ｐ新幕僚就位？　新聞主播、蘇系子弟兵入北市府|url=https://tw.appledaily.com/new/realtime/20170505/1112591/|language=zh-hant|accessdate=2017-05-05|newspaper=《[[蘋果日報_(台灣)|蘋果日報]]》 |date=2018-12-22}}</ref><ref>{{cite news|author= |title=年代主播林昆鋒 傳任北市府發言人|url=http://news.ltn.com.tw/news/politics/breakingnews/2058336|language=zh-hant|accessdate=2017-05-05|newspaper=《[[自由時報|自由時報]]》 |date=2018-12-22}}</ref>。

2018年9月3日，林昆鋒、[[黃瀞瑩|黃瀞瑩]]向柯文哲競選辦公室報到<ref>{{cite news|author= 林縉明 |title=台北》柯文哲邁向連任之路　學姊、林昆鋒今競辦報到|url=https://www.chinatimes.com/realtimenews/20180903001392-260407|language=zh-hant|accessdate=2018-09-03|newspaper=《[[中國時報|中國時報]]》 |date=2018-12-22}}</ref><ref>{{cite news|author= 林縉明 |title=學姊、林昆鋒 進駐柯P競辦|url=https://www.chinatimes.com/newspapers/20180904000622-260107|language=zh-hant|accessdate=2018-09-04|newspaper=《[[中國時報|中國時報]]》 |date=2018-12-22}}</ref>。
