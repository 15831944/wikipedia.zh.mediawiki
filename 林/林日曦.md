'''林日曦'''（{{lang|en|Roy Tsui}}，{{bd|1980年|9月30日}}），本名'''徐家豪'''<ref name="蘋果日報 果籽 文化 20130810">{{cite news |language = |author = |coauthors = |url = http://hk.apple.nextmedia.com/supplement/culture/art/20130810/18371769 |title = 火熱悶棍 林日曦 |work = |publisher = 蘋果日報 |pages =  | date = 2013-08-10 |accessdate = 2015-07-31}}</ref>，[[香港|香港]][[填詞人|填詞人]]、[[作家|作家]]及創作單位《[[黑紙|黑紙]]》創辦人，創辦[[毛記電視|毛記電視]]、雜誌《[[黑紙|黑紙]]》、雜誌《[[100毛|100毛]]》及[[白卷出版社|白卷出版社]]，別稱'''林日A'''<ref>{{cite web|url=https://www.facebook.com/blackpaperclub/posts/192802140750275|title=傳媒餵屎論|publisher=黑紙|date=2011-03-02}}</ref>或'''腦細'''<ref>[http://hk.apple.nextmedia.com/nextmag/art/20160129/19469940 堅•搵錢！毛記年賺800萬　腦細唔上市]</ref>，居於[[屯門|屯門]]<ref>[http://www.hkexnews.hk/app/SEHK/2017/2017072502/Documents/SEHK201707260012_c.pdf 毛記葵涌有限公司上市申請書 第44頁] {{Wayback|url=http://www.hkexnews.hk/app/SEHK/2017/2017072502/Documents/SEHK201707260012_c.pdf |date=20170728200818 }}</ref>。

==簡歷==
[[File:Lam_Yat_Hei_in_Hong_Kong_Book_Fair_2016.jpg|thumb]]為讀者簽名]]

林日曦早年在[[葵涌|葵涌]][[大窩口邨|大窩口邨]]成長<ref>{{web reference|URL=http://www.metrohk.com.hk/pda/pda_detail.php?id=303018|title=不只黑面 林日曦|work=都市日報|year=2016年|date=2月24日}}</ref>，為家中獨子。他曾就讀[[佛教林炳炎紀念學校|佛教林炳炎紀念學校]]，後來分別入讀過四所中學，包括[[聖公會林護紀念中學|聖公會林護紀念中學]]（中一至中三）、[[葵涌循道中學|葵涌循道中學]]（中四）、[[循道衞理聯合教會李惠利中學|循道衞理聯合教會李惠利中學]]（中五）以及[[珠海學院|珠海學院]]（中學部）。在學時多無心向學。<ref>{{web reference|URL=http://www.metrodaily.hk/metro_columns/%E6%9E%97%E6%97%A5%E6%9B%A6%E9%BB%91%E9%9D%A2%E6%9B%B8%E5%9F%BA%E5%9C%B0/|title=林日曦##黑面書##基地|work=都市日報|year=2015年|date=11月2日}}</ref>林日曦是香港著名[[填詞|填詞]]人[[林夕|林夕]]的徒弟之一<ref>商業電台節目《叱咤強——填詞新力軍!! 》。2009年12月22日</ref>，其配偶為香港獨立一人樂隊[[The_Pancakes|The Pancakes]]成員蔡明麗，兩人於2006年結婚。

林日曦於[[香港專業教育學院|香港專業教育學院]]（IVE）[[設計|設計]]系中途輟學加入[[商業電台|商業電台]]負責聲音剪接及網站管理，及後自薦參與電台雷霆881及叱吒903的幕後創作<ref name="Recruit封面故事。2010年2月2日">{{cite news|language =|author =|coauthors =|url =http://www.recruitonline.com/article/ArticleDetail.aspx?article_id=25813|title =80後　三人行|work =|publisher =Recruit|pages =|date =2010-02-02|accessdate =2010-02-02}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，曾擔任[[天比高創作伙伴|天比高創作伙伴]]創作總監<ref name="《新七大奇蹟》">{{cite news |language = | author = |coauthors = |url = http://www.cp1897.com.hk/product_info.php?BookId=9789888134670 |title = 新七大奇蹟 |work = |publisher = 日閱堂 |pages = |date = 2010-02-02 |accessdate = 2010-02-02 }}</ref>。

2007年林日曦參與電影《[[東京鐵塔：我的母親父親#电影|東京鐵塔：我的父親母親]]》宣傳，《(東京鐵塔下) gulugulu》的歌詞成為他第一首發表作，歌曲入選當年叱吒十大。至今發表作品近100首。

2009年林日曦與[[陳強|陳強]]及[[阿Bu|阿Bu]]創辦創作單位《[[黑紙|黑紙]]》，開始參與不同類型的創作，如出版雜誌、MV導演及平面設計<ref name="有關黑紙">{{cite web |language = |author = |coauthors =  | url = http://www.blackpaper.com.hk/?page_id=2 |title = 有關黑紙 |work = |publisher = 黑紙 |pages =  |ate = |accessdate =  }}</ref>。翌年1月推出首度只得一頁紙的《黑紙》雜誌。

2011年起林日曦於報紙撰寫專欄。2012年出版第一本著作《白痴》<ref name="明報網上書店">{{cite web|language =|author =|coauthors =|url =http://books.mingpao.com/cfm/bookinfo.cfm?TitleID=4208|title =明報網上書店|work =|publisher =明報|pages =|date =|accessdate =|deadurl =yes|archiveurl =https://web.archive.org/web/20120626125219/http://books.mingpao.com/cfm/bookinfo.cfm?TitleID=4208|archivedate =2012-06-26}}</ref>2013年出版第二本著作《青筋》，2014年出版《黑面》及第一本小說《快樂有限》。

2013年3月《[[100毛|100毛]]》雜誌創刊，不足半年已達收支平衡。同年並創辦[[白卷出版社|白卷出版社]]，專門出版小說、散文、圖文集等流行讀物。<ref name="80後老闆-「無為」管人法">{{cite web|language =|author =|coauthors =|url =http://www.jiujik.com/resources/article/80後老闆-「無為」管人法|title =80後老闆「無為」管人法|work =|publisher =Recruit|pages =|date =|accessdate =|deadurl =yes|archiveurl =https://web.archive.org/web/20160304204304/http://www.jiujik.com/resources/article/80%E5%BE%8C%E8%80%81%E9%97%86-%E3%80%8C%E7%84%A1%E7%82%BA%E3%80%8D%E7%AE%A1%E4%BA%BA%E6%B3%95|archivedate =2016-03-04}}</ref> 在2013年之前亦常為[[無綫電視|無綫電視]]劇集主題曲填詞。

2014年起林日曦於[[Now_TV|Now TV]]擔任《電視節目有好多種》主持，同年接受商業電台節目訪問，因面露難色被指為經常「黑面」。<ref>{{cite web | url = http://hk.apple.nextmedia.com/financeestate/art/20150520/19153354 | title = 【職場秘笈】 100毛係地獄？！林日曦：我係惡魔老細 | publisher = apple daily hong kong | date = 2015-05-20}}</ref>

2015年9月15日，香港近一兩年備受年輕讀者和觀眾追捧的《[[100毛|100毛]]》雜誌以及毛記電視的大本營，當中的核心人物，創辦人之一林日曦就在其中一間沒有大班椅，沒有L型檯，更加沒有海景view的老闆房內接受jobsDB的專訪。 <ref>{{cite web | url = http://hk.jobsdb.com/en-hk/articles/%E6%9E%97%E6%97%A5%E6%9B%A6%E5%B0%88%E8%A8%AA%E4%B8%80%E6%88%91%E6%88%90%E5%8A%9F%E5%9B%A0%E5%B0%9A%E6%9C%AA%E5%9F%B7%E7%AC%A0%E4%BD%86%E4%BF%82%E9%81%B2%E6%97%A9%E7%9A%84%E4%BA%8B | title = 林日曦專訪（一）：我成功，因尚未執笠，但係遲早的事 | publisher = jobsdb | date = 2015-09-15}}</ref>

2017年5月12日，由《[[100毛|100毛]]》及[[毛記電視|毛記電視]]製作、[[香港四大天王|香港四大天王]]之一[[黎明|黎明]]代言的[[雀巢|雀巢正宗越南咖啡]]廣告，在《[[100毛|100毛]]》、等多個[[facebook|facebook]]專頁廣傳。黎明親自演繹改編自其經典作品《對不起，我愛你》的改編歌曲《對不起，我愛越南啡》，MV更邀得堅．黎明<ref name="real">根據毛記電視傳統，現實公眾人物會於名稱前方加上「真」字或「堅」字，以便與毛記幕前偽員區分。</ref>、林日曦、[[劉江_(香港)|堅．劉江]]<ref name="real"></ref>、林夕 及[[毛記電視#偽員|毛記電視偽員]]「肥Ben」、「黃慘盈」、「龍志權」、 合作拍攝。林日曦飾演「腦細」，其他人飾演職員。<ref>[https://www.facebook.com/100most/videos/902063266587988/ facebook：100毛 - 黎明《情深咖啡未曾飲》]</ref><ref>[http://hk.apple.nextmedia.com/enews/realtime/20160514/55104057 蘋果日報即時新聞：【識唱跟住唱】毛記搵黎明開金口改歌詞 《情深咖啡未曾飲》]，2016年5月14日。</ref>廣告在《[[100毛|100毛]]》Facebook專頁版本已有近三百萬瀏覽量，網民爭相分享，{{fact|使香港各大地區超級市場的[[雀巢|雀巢正宗越南咖啡]]幾乎售罄|time=2018-07-21}}，掀起熱潮。
