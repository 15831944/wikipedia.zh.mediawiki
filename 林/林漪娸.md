'''林漪娸'''（{{lang|en|'''Yvonne Lam Yi Kei'''}}，{{bd|1959年|4月19日|catIdx=L林}}），原名'''林綺雯'''，人稱漪娸姐，[[外號|外號]]為'''港產[[美魔女|美魔女]]'''<ref>{{cite web | language = zh-hant | publisher = Yahoo 娛樂圈 | title = 大姐大手記﹕林漪娸30年不變港產「美魔女」 | url = http://hk.ent.yahoo.com/news/%E5%A4%A7%E5%A7%90%E5%A4%A7%E6%89%8B%E8%A8%98-%E6%9E%97%E6%BC%AA%E5%A8%B830%E5%B9%B4%E4%B8%8D%E8%AE%8A%E6%B8%AF%E7%94%A2-%E7%BE%8E%E9%AD%94%E5%A5%B3-%E5%BC%B5%E5%8F%AF%E9%A0%A4%E4%BF%9D%E9%A4%8A%E5%BE%97%E5%AE%9C%E8%A5%AF%E5%BE%97%E8%B5%B7%E9%99%B3%E8%B1%AA-212658730.html | date = 2012年10月3日 | deadurl = yes | archiveurl = https://web.archive.org/web/20121011062412/http://hk.ent.yahoo.com/news/%E5%A4%A7%E5%A7%90%E5%A4%A7%E6%89%8B%E8%A8%98-%E6%9E%97%E6%BC%AA%E5%A8%B830%E5%B9%B4%E4%B8%8D%E8%AE%8A%E6%B8%AF%E7%94%A2-%E7%BE%8E%E9%AD%94%E5%A5%B3-%E5%BC%B5%E5%8F%AF%E9%A0%A4%E4%BF%9D%E9%A4%8A%E5%BE%97%E5%AE%9C%E8%A5%AF%E5%BE%97%E8%B5%B7%E9%99%B3%E8%B1%AA-212658730.html | archivedate = 2012年10月11日 }}</ref>，現為[[電視廣播有限公司|無綫電視]]基本藝人合約女藝員，1980年度[[香港小姐競選|香港小姐競選參賽者]]之一。因參與[[香港小姐|香港小姐]]而加入[[無綫電視|無綫電視]]至1983年，1986年重返[[無綫電視|無綫電視]]至今。

== 背景 ==
林漪娸在[[九龍|九龍]][[深水埗區|深水埗區]]長大，於8至9歲時搬到[[美孚新邨|美孚新邨]]居住。她家中共有兩名胞兄、一名胞姊和一名胞弟，排行第四。而父親是一名商人，經營成衣生意。林漪娸曾就讀[[瑪利諾修院學校|瑪利諾修院學校]]至中二時被家人送到[[英國|英國]][[伯恩茅斯|伯恩茅斯]]留學。直至21歲時獲得表姐鼓勵回港參加香港小姐競選<ref>{{web reference|URL=http://yule.sohu.com/20070413/n249417113.shtml|title=無綫藝人專訪 林漪娸：胸無大志輕鬆對待人生|work=搜狐娛樂|year=2007年|date=4月13日}}</ref>。此前，她在英國修讀商科。本來有意回港協助父親經營成衣生意。之後因參加香港小姐競選而加入娛樂圈。{{noteTag|關於林漪娸早年生活與入行經過的資料出自香港電台第二台廣播節目《守下留情》於2018年7月23日播出的訪問錄音〈Madam Lam林漪娸訪問第一集 + 學校訂飯定帶飯 ，重溫！〉。}}

1980年，林漪娸以原名'''林綺雯'''參加1980年度[[香港小姐競選|香港小姐競選]]。奪得第四名，憑著參選而入行，加入無綫電視至1983年，1986年重返無綫電視至今。她仍然活躍電視圈，曾演出多部電視劇。

漪娸參演多部電視劇，由於演技精湛而大獲好評，包括首部担正電視劇為[[生锈橋王|生锈橋王]]、《[[真情|真情]]》中的小女人李小梅、《[[皆大歡喜_(古裝電視劇)|皆大歡喜 (古裝版)]]》的諸事八婆雲影姬（僵屍）、《[[皆大歡喜_(時裝電視劇)|皆大歡喜 (時裝版)]]》中的精明秘書雲影姬（OK姐）、《[[ID精英|ID精英]]》的汪明慧、《[[老友狗狗_(電視劇)|老友狗狗]]》中的潑辣妻子陳美蓮（蟹妖蓮）、《[[宮心計|宮心計]]》里的高傲自大的孫家碧（萬夫人）、處境喜劇《[[誰家灶頭無煙火|誰家灶頭無煙火]]》中的孫瑪利（Mary）及《[[愛·回家_(第一輯)|愛·回家]]》等。林漪娸於《[[愛·回家_(第一輯)|愛·回家]]》亦擔任第一女主角，並與年輕15歲的[[徐榮|徐榮]]合演夫妻。

林漪娸曾多次憑《[[皆大歡喜_(古裝電視劇)|皆大歡喜]]》[[皆大歡喜_(時裝電視劇)|系列]]的「雲影姬」一角獲提名“[[萬千星輝賀台慶|萬千星輝賀台慶]]”的「本年度我最喜愛的女主角」及「本年度我最喜愛的拍檔（戲劇組）」。2011年，漪娸姐憑著處境喜劇《[[誰家灶頭無煙火|誰家灶頭無煙火]]》中飾演的孫瑪利（Mary）一角，而獲提名“[[2011年萬千星輝頒獎典禮|2011年萬千星輝頒獎典禮]]”「'''最佳女配角'''」。

值得一提的是漪娸姐曾參演多部[[處境喜劇|處境喜劇]]，參演的處境喜劇包括《[[我愛玫瑰園|我愛玫瑰園]]》、《[[開心華之里|開心華之里]]》、《[[真情|真情]]》、《[[皆大歡喜_(古裝電視劇)|皆大歡喜（古裝版）]]》、《[[皆大歡喜_(時裝電視劇)|皆大歡喜（時裝版）]]》、《[[誰家灶頭無煙火|誰家灶頭無煙火]]》以及《[[愛·回家_(第一輯)|愛·回家]]》，因參演多部處境喜劇而被人稱之為「'''處境劇女王'''」。

== 家庭生活 ==
林漪娸已婚。丈夫為前高級警司譚德榮。兩人育有一子一女譚紹偉及譚詠欣。
