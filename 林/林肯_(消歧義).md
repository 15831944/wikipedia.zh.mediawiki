'''林肯'''（'''Lincoln'''）一字源自[[英國|英國]]的[[林肯郡|林肯郡]]和[[林肯_(林肯郡)|林肯城]]，現時可指下列：

== 人物 ==
* [[亞伯拉罕·林肯|亞伯拉罕·林肯]]：美國第十六任[[美國總統|總統]]，任內美國爆發[[南北戰爭|南北戰爭]]。林肯廢除了[[奴隸制度|奴隸制度]]，擊敗了[[美國南部|美國南部]][[分離主義|分離主義]]勢力，維護了國家的統一。由於林肯的成就，美國有不少事物均以林肯命名。
* [[布蘭琪·林肯|布蘭琪·林肯]]：[[民主黨_(美國)|民主黨]]人，曾任美國[[阿肯色州|阿肯色州]]參議員。
* [[老利瓦伊·林肯|老利瓦伊·林肯]]：美國政治家。
* [[林肯_(足球运动员)|林肯]]：是一名[[巴西|巴西]][[足球|足球]]運動員。
* [[伊芙琳·林肯|伊芙琳·林肯]]：美国第三十五任总统[[约翰·肯尼迪|约翰·肯尼迪]]的[[秘书|私人秘书]]。
* [[爱德华·林肯|爱德华·林肯]]：亚伯拉罕·林肯的次子。
* [[威廉·林肯|威廉·林肯]]：亚伯拉罕·林肯的第三子。
* [[塔德·林肯|塔德·林肯]]：亚伯拉罕·林肯的幼子。
* [[亚伯拉罕·林肯二世|亚伯拉罕·林肯二世]]：罗伯特·林肯的儿子，亚伯拉罕·林肯唯一的孙子。

== 地方 ==
* [[林肯郡|林肯郡]]：
* [[林肯縣|林肯縣]]：[[美國|美國]]有二十四個州的行政單位以林肯命名。
* [[林肯中心|林肯中心]]：位於[[美國|美國]][[紐約市|紐約市]]的綜合藝術表演中心。
* [[林肯港|林肯港]]：[[澳洲|澳洲]]南部一著名漁港。
* [[林肯座堂|林肯座堂]]：座落[[英國|英國]]的[[林肯郡|林肯郡]]，屬於[[英國國教會|英國國教會]]。
* [[林肯紀念堂|林肯紀念堂]]：為紀念美國總統[[亞伯拉罕·林肯|亞伯拉罕·林肯]]而設立的紀念堂，位於[[美國|美國]][[華盛頓特區|華盛頓特區]]。
* [[林肯大郡|林肯大郡]]：位於[[台灣|台灣]][[新北市|新北市]][[汐止區|汐止區]]汐萬路二段228巷的複合功能住宅社區。
* [[林肯隧道|林肯隧道]]：位於[[美國|美國]]的隧道，跨越[[哈德遜河|哈德遜河]]底以連接[[新澤西州|新澤西州]]威活肯與[[紐約市|紐約市]][[曼哈頓|曼哈頓]]中城。

== 大學 ==
* [[林肯大學|林肯大學]]：位於[[紐西蘭|紐西蘭]]，創立於1990年，是[[紐西蘭|紐西蘭]]八所大學之一，前身是坎特伯雷大學林肯學院。
* [[內布拉斯加-林肯大學|內布拉斯加-林肯大學]]：位於[[美國|美國]][[內布拉斯加州|內布拉斯加州]]的[[林肯市|林肯市]]。

== 體育 ==

== 電影 ==

== 其他 ==
* [[林肯汽車|林肯汽車]]：[[福特汽車|福特汽車]]旗下高級汽車品牌，名稱是以美國總統[[亞伯拉罕·林肯|亞伯拉罕·林肯]]的名字命名。
* [[亞伯拉罕·林肯號航空母艦|亞伯拉罕·林肯號航空母艦]]：[[美國|美國]][[尼米茲級|尼米茲級]]核動力[[航空母艦|航空母艦]]，為紀念美國總統[[亞伯拉罕·林肯|亞伯拉罕·林肯]]而命名。
* [[林肯一分鋼幣|林肯一分鋼幣]]：簡稱林肯分，美國錢幣，1909年第一次鑄造並發行，是美國有史以來流通最久的硬幣。
