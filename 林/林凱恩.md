'''林凱恩'''（{{Lang|en|'''Iris Lam Hoi Yan'''}}，{{bd|1992年|8月17日}}），香港女主持及演員，現為[[無綫電視藝員列表|無綫電視經理人合約]]藝員<ref>[http://artiste.tvb.com/main/artistedetail/irislam 林凱恩 Lam Hoi Yan - TVB藝人資料 - tvb.com]</ref>。

== 簡歷 ==
林凱恩有一兄長<ref>[https://www.instagram.com/p/BMJLnfDh0zL/?taken-by=iris_lammm 我最敬重的媽媽終於畢業了]，Instagram - 林凱恩（iris_lammm），2016年10月29日</ref>，父母在她就讀中學期間分開，母親便成為了家庭支柱。她畢業於[[仁濟醫院王華湘中學|仁濟醫院王華湘中學]]，因其母年輕時須照顧家庭而未能參加選美比賽以圓心願，遂2014年修畢[[香港城市大學專上學院|香港城市大學專上學院]]公共關係及廣告社會科學副學士課程後代母參加《[[2015年度香港小姐競選|2015年度香港小姐競選]]》<ref name=IL>{{cite web |title=【開心速遞】林凱恩單親家庭長大入行為家人：想令媽咪無咁辛苦 |url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/130479/%E9%96%8B%E5%BF%83%E9%80%9F%E9%81%9E-%E6%9E%97%E5%87%B1%E6%81%A9%E5%96%AE%E8%A6%AA%E5%AE%B6%E5%BA%AD%E9%95%B7%E5%A4%A7%E5%85%A5%E8%A1%8C%E7%82%BA%E5%AE%B6%E4%BA%BA-%E6%83%B3%E4%BB%A4%E5%AA%BD%E5%92%AA%E7%84%A1%E5%92%81%E8%BE%9B%E8%8B%A6 |website=[[香港01|香港01]] |date=2017-11-09}}</ref>，並奪得「友誼小姐」獎項<ref>{{cite web |title=林凱恩贏友誼小姐超感動 |url=http://std.stheadline.com/daily/news-content.php?id=42252&target=2 |website=[[星島日報|星島日報]] |date=2015-08-28}}</ref>，隨後亦加入無綫電視。

2017年，林凱恩在該台處境劇《[[愛·回家之開心速遞|愛·回家之開心速遞]]》裡飾演「譚育英（Bonnie）」一角漸為人所認識<ref name=IL/>，同年於台慶劇集《[[降魔的|降魔的]]》內飾演「張佩玲」也獲網民讚賞<ref>{{cite web |title=《降魔的》惹關注　網民讚林凱恩似初出道佘詩曼 |url=http://news.modia.com.hk/2017/11/10/42458/%E3%80%8A%E9%99%8D%E9%AD%94%E7%9A%84%E3%80%8B%E6%83%B9%E9%97%9C%E6%B3%A8%E3%80%80%E7%B6%B2%E6%B0%91%E8%AE%9A%E6%9E%97%E5%87%B1%E6%81%A9%E4%BC%BC%E5%88%9D%E5%87%BA%E9%81%93%E4%BD%98%E8%A9%A9%E6%9B%BC/ |website=[[流動新聞|流動新聞]] |date=2017-11-10}}</ref>。
