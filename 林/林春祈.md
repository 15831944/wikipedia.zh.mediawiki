'''林春祈'''{{BD|1836年|四月六日丑時|1890年四月七日卯時||catIdx=林}}，字'''锡爵'''、諡'''穆忠'''，人稱'''林部爺'''<ref name="林姓宗廟">{{cite web |author1=林黃河 |title=南投草屯林氏家廟 |url=http://www.linajf1901.com/information_db1.php?information_id=44&action=new_information |website=財團法人全國林姓宗廟 |accessdate=2018-08-11}}</ref>，清台灣[[南投廳|南投廳]][[北投堡|北投堡]][[北投埔庄|北投埔庄]]（今台灣[[南投縣|南投縣]][[草屯鎮|草屯鎮]]北投埔）人，林姓宗族地方領導，在[[戴潮春事件|戴潮春事件]]中協助官府對抗戴軍，在[[北勢湳|北勢湳]]一役中出力甚多<ref name="謝貴文">{{cite journal |author1=謝貴文 |title=傳說、歷史與集體意識 － 從草屯龍德廟 的「刑期無刑」匾談起 |journal=高雄師大學報 |date=2011-06-01 |issue=30 |pages=23-42 |url=https://ir.nknu.edu.tw/ir/retrieve/18213/%E5%82%B3%E8%AA%AA%E3%80%81%E6%AD%B7%E5%8F%B2%E8%88%87%E9%9B%86%E9%AB%94%E6%84%8F%E8%AD%98%EF%BC%8D%E5%BE%9E%E8%8D%89%E5%B1%AF%E9%BE%8D%E5%BE%B7%E5%BB%9F%E7%9A%84%E3%80%8C%E5%88%91%E6%9C%9F%E7%84%A1%E5%88%91%E3%80%8D%E5%8C%BE%E8%AB%87%E8%B5%B7.pdf |accessdate=2018-08-11}}</ref>。

== 宗族 ==
林春祈是月眉林氏宗族「鵝房」第十八世孫。月眉林氏宗族是[[元朝|元朝]][[至正|至正]]年間林長清的後裔，林長清生於象山東坑仔，徙至永豐里和溪墟麟埜社後，成當地林姓宗族開基祖。其八世孫林文渙，因為居於鵝墌坪而號稱「鵝房」；清初，林長清的十三、十四、十五世孫共六房，自今[[嘉義縣|嘉義縣]][[布袋港|布袋港]]處登陸臺灣，先在今[[梅山鄉|梅山鄉]]一帶暫居，而後遷徙到月眉一帶，以月眉厝龍德廟為家族的信仰中心。六房中奉祀[[關公|關公]]與[[觀音|觀音]]的各半、每年輪流主祭，而「鵝房」屬奉祀觀音的一方<ref name="林姓宗廟" />。

林春祈誥授奉政大夫四品藍翎知州，胞弟[[林春梗|林春梗]]也侍贈五品軍功，兩人均為清代族中賢達。其後代地方頭人有二十世[[林野|林野]]，[[臺灣日治時期|臺灣日治時期]]任[[草屯街協議會|草屯街協議會]]議員<ref name="林姓宗廟" />。

== 生平 ==
林春祈生於1836年（清[[道光|道光]]十六年）四月初六丑時，卒於1890年(清[[光緒|光緒]]十六年)四月初七卯時 [[咸豐|咸豐]]年間於[[北投堡|北投堡]]擔任職員，加入[[登瀛書院|登瀛書院]]的玉峰社<ref name="張家綸" />。

1862年（清同治元年），林春祈與林文翰、邱石莊、簡榮卿、洪玉崑和南投大家族的頭人聚於倚南軒，為當時有擴大之勢的[[戴潮春事件|戴潮春事件]]做準備。次年4月28日，林春祈和多名彰化、北投堡、[[南投堡|南投堡]]的舉人、生員參與[[陳肇興|陳肇興]]所組的六堡同盟，發動鄉勇響應官兵，共同對抗戴潮春勢力。其中多人初交戰即遭陳永明、陳輝、黃丁目、蕭金泉等戴黨軍隊擊敗；而林春祈則是因為遭內賊連同戴黨截斷水道，使農作歉收，導致家中財產盡空，全部用於軍火，但仍勉強守住鄉里。6月16日，陳捷三與戴黨楊目丁在[[濁水溪|濁水溪]]一帶大戰，獲得陳貞元、陳上治合兵支援，生擒了楊目丁後斬首摘心；自此中部反戴聯軍開始反攻，林春祈也從北投埔引兵助攻<ref name="張家綸">{{cite journal |author1=張家綸 |title=草屯社會發展與地方菁英(1751-1945) |journal=國立台灣師範大學歷史研究所碩士論文 |date=2008-06 |pages=67-68 |url=https://www.ntl.edu.tw/public/ntl/4216/%E5%BC%B5%E5%AE%B6%E7%B6%B8%E5%85%A8%E6%96%87.pdf |accessdate=2018-08-11}}</ref><ref name="戴案紀略">{{cite book |author1=蔡青筠 |title=戴案紀略 |url=https://ctext.org/wiki.pl?if=gb&chapter=928828 |accessdate=2018-08-11}}</ref>。

1864年11月，官軍[[丁曰健|丁曰健]]攻打在[[北勢湳|北勢湳]]的[[洪欉|洪欉]]，北勢湳由於道路曲折、易守難攻，官軍攻勢受阻，李光輝、林春甚至中伏身亡。林春祈為鄭榮擔任嚮導，挖地道深入[[洪欉|洪欉]]宅底，以火藥炸坍了房屋，使洪軍軍心動搖。後來洪欉被火炮擊殺、其弟洪璠遭其妾劉氏綁縛後交給丁曰健，洪軍遂平<ref name="戴案紀略" />。林春祈因此受封「誥授奉政大夫藍翎知州」<ref name="張家綸" />。

== 「刑期無刑」匾 ==
「刑期無刑」匾是台灣[[南投縣|南投縣]][[草屯鎮|草屯鎮]][[月眉厝龍德廟|月眉厝龍德廟]]中保存的國家重要古物，於1865年（清同治四年）由丁曰健獻立<ref>{{cite web |title=月眉厝龍德廟 |url=https://www.taiwangods.com/html/cultural/3_0011.aspx?i=57 |website=臺灣宗教文化資產 |publisher=中華民國內政部 |accessdate=2018-08-11}}</ref>。該匾除了「刑期無刑」四字之外，也敘述了北勢湳一役的民間傳說。文中提及丁曰健行軍經龍德廟時，廟內乩童以[[保生大帝|保生大帝]]之名攔其轎。丁只能留宿林春祈宅，次日獲神明指示需駐兵在北勢湳五里外候機進攻。丁曰健聽旨按兵不動，並派人威脅洪欉姻親，策動洪氏母舅毒害洪欉，待洪營內亂後進兵。平亂後，丁氏獲皇帝賜「刑期無刑」四字，因此製匾獻於廟中<ref name=" 謝貴文" />。

此匾所述的北勢湳之役，林春祈出力甚多，但在保生大帝顯靈傳說中僅提供丁宿於其家中，未提及挖地道事。據學者謝貴文推論，林春祈極有可能是[[洪欉|洪欉]]的姻親，依《[[大清律例|大清律例]]》，是謀反罪可誅連的親族之一；因此林春祈加入官軍對抗洪欉是「大義滅親」的自保之舉，而丁曰健則認為其功僅足以折過，所以在表功奏折中只提及挖掘地道的[[呂偉徠|呂偉徠]]與[[施福|施福]]，未提及林錫爵<ref name=" 謝貴文" />。
