'''總統'''（{{lang-en|President}}），[[日本|日本]]及[[朝鲜半岛|朝鲜半岛]]称为“大總統”（{{jpn|j=大統領|hg=だいとうりょう|rm=Daitōryō|f=h}}；{{kor|k=대통령|hanja=大統領|rm=Daetongnyeong|f=h}}），是[[共和制|共和制]][[國家|國家]]的[[國家元首|元首]]稱呼之一。

== 名稱起源與演變 ==
總統的[[英文|英文]]譯稱「{{lang|en|President}}」，在英文中是指[[領導者|領導者]]或[[領導人|領導人]]（由[[拉丁文|拉丁文]] pre-“之前”+sedere“坐”而來），在[[羅馬帝國|羅馬帝國]]時期為副執政官或行省總督的職位，並非[[執政官|執政官]]。

但「President」不一定是政治領袖，某些機構或團體的[[主席|主席]]、負責人，[[公司|公司]]的[[總裁|總裁]]和[[大學|大學]]的[[校長|校長]]也使用這稱謂。

== 中文舊稱 ==
[[清朝|清朝]]末年曾音译為“伯理玺天德”、「柏理璽天德」、「普利玺天德」<ref>清·薛福成，《出使四国日记·光绪十六年十二月二十九日》：「美洲各国及欧洲之瑞士与法国皆民主之国也，其政权全在议院，而伯理璽天德无权焉。」；严復《＜法意＞按语》：「虽伯理由于公推，议院有其聚散，而精神之贯彻始终则一而已。」</ref>。

舊稱統兵的首領為總統。民國初年及北洋政府時期，皆冠以「大」字以區別或彰顯之；如臨時大總統、非常大總統。

1931年制訂的[[中華民國訓政時期約法|中華民國訓政時期約法]]中，僅設國民政府主席。

1936年制訂的[[五五憲草|五五憲草]]中，始稱「總統」，不再冠以「大」字，沿用至今。

== 各國制度 ==
* 在[[總統制|總統制]]國家，總統兼任[[政府首脑|政府首脑]]，直接領導行政機關，不設[[總理|總理]]，由人民直接或間接選舉，如{{USA}}、{{MEX}}、{{ARG}}、{{BRA}}、{{CHI}}、{{INA}}（1966年以后）、{{PHI}}（1978年以后）、{{TUR}}（2018年以后）以及[[拉丁美洲|拉丁美洲]]和[[非洲|非洲]]的不少[[國家|國家]]，皆實行[[總統制|總統制]]。部分国家设总理，但實際上是總統制，如{{KOR}}、{{PER}}。
* 在[[议会制|议会制]]國家，总统作为国家元首通常只是[[虛位元首|虛位元首]]，真正行使行政職權的是[[總理|總理]]（部分共和國或稱[[部長會議主席|部長會議主席]]），總統由議會組成的選舉團選出，如{{GER}}、{{GRE}}、{{HUN}}、{{IND}}、{{ISR}}、{{ITA}}、{{LAT}}、{{PAK}}、{{BAN}}。
* 在[[议会制|议会制]]國家，总统作为国家元首通常只是象徵性職位，但由[[直選|直選]]產生，因此擁有部分外交权力，如{{AUT}}、{{BUL}}、{{CZE}}、{{SVK}}、{{FIN}}、{{ICE}}。
* 在总統领導的[[议会制|议会制]]國家，总统兼任[[政府首脑|政府首脑]]，直接領導行政及立法機關，不設[[總理|總理]]，總統由民選的[[議會|議會]]選出，如{{MYA}}、{{RSA}}、{{BWA}}。
* 在[[半总统制|半总统制]]（又称[[雙首長制|雙首長制]]）國家，总统与[[总理|总理]]职权的划分，依其國情的不同，实行「合法性二元化」措施，总理代表[[行政机关|行政机关]]向议会负责，但总统的权力普遍较总理大，總統擁有国防及外交权力，且擁有解散国会的权力。最典型的例子是{{FRA}}，其他如{{POL}}、{{POR}}、{{RUS}}、{{UKR}}、{{ROM}}、{{SER}}、{{ROC}}等也可视为半总统制。
* 在[[委员制|委员制]]国家，“[[瑞士聯邦主席|瑞士聯邦主席]]”（也有翻译成總統）在法律上与[[总统制|总统制]]类似，是国家元首和政府首脑；但其实际权力及地位都不比其他委員高，如同[[议会制|议会制]]下的國家元首。现今实行这一制度的只有{{SUI}}和{{SMR}}两国。
* 在[[社会主义国家|社会主义国家]]，国家元首一般不称总统，因为[[一党执政|一党执政]]的原因，其地位和权力比不上[[执政党|执政党]]的[[政党领袖|最高领导人]]（[[总书记|总书记]]、[[第一书记|第一书记]]，除非兼任），部分國家元首甚至是[[虚位元首|虚位元首]]。{{CHN}}、{{VIE}}、{{LAO}}称其国家元首为[[国家主席|国家主席]]，{{CUB}}、{{GDR}}、{{flag|保加利亚人民共和国}}等称[[国务委员会主席|国务委员会主席]]。曾经使用“总统”为国家元首名称的社会主义国家有{{flag|古巴}}（1959-1976）、{{flag|苏联}}（1990-1991）、{{flag|捷克斯洛伐克}}（1948-1989）、{{DDR}}（1949-1960）、{{flag|罗马尼亚社会主义共和国}}（1974-1989）、{{flag|南斯拉夫社会主义联邦共和国}}（1945-1990）和{{flag|波兰人民共和国}}（1947-1952、1989-1990）。


*在不同國家設定上，總統職位可任職一定年期，可連任或不可連任，或任職一定年期後不能再出任。
