'''总主教'''，也翻譯為'''大主教'''，是[[基督教|基督教]]传统教会的一种高階聖職人員，根据[[天主教|天主教]]及[[聖公會|聖公會]]的制度，数个[[教区|教区]]组成一个[[教省|教省]]，其中會設有一個[[總教區|總教區]]，總教區的[[正權主教|正權主教]]則稱為總主教；而教省總主教在省属教区中有某些权限<ref>《[[天主教法典|天主教法典]]》第435條</ref>。通常兼任省屬教區主教組成的[[主教團|主教團]]團長。不過這一職銜在[[聖秩聖事|聖秩聖事]]三階級的分類上和一般的主教是同等的。頭銜簡稱為『''abp.''』<ref name="字典">[http://www.catholic.org.tw/katalina/ycw/BLESSING/dict/A.htm 天主教字典A] {{webarchive|url=https://web.archive.org/web/20101101214643/http://www.catholic.org.tw/katalina/ycw/BLESSING/dict/A.htm |date=2010-11-01 }}</ref>

== 教省總主教 ==
在早期的天主教受到[[羅馬帝國|羅馬帝國]]的[[羅馬帝國行省|行省]]制度影響，行省都會的教區主教因為所轄教區在該地區的中心性，往往會成為該地區主教們的領袖，因此延伸出[[都主教|都主教]]（Metropolitan）這個職位，獲得教會法賦予特殊職責。而當時有德望的主教常常被尊稱為''Archbishop''。由於這兩種主教稱呼的性質相近，以至於這兩種職銜逐漸合而為一，教省總主教（Metropolitan archbishop）所轄的教區跟著被稱為''Archdiocese''<ref>[http://dcc.catholic.org.hk/Testing/Useful%20Knowledge/uk3.pdf 各種主教的分別] {{webarchive|url=https://web.archive.org/web/20111225211324/http://dcc.catholic.org.hk/Testing/Useful%20Knowledge/uk3.pdf |date=2011-12-25 }}天主教香港教區教理中心及教理委員會網頁資料</ref>。

《[[天主教法典|天主教法典]]》第436条規範了教省總主教在所轄教省所屬教區中的監督，以及在所轄教省內的教區主教出缺時選任署理等職權，[[東儀天主教|東儀天主教]]則另將教省總主教的職責規範在[[東儀天主教會法典|東儀天主教會法典]]157條。

教省總主教對於教省所屬的其它教區並無監督以外的管轄權。不過除了主教座堂需通知該教區主教外，教區總主教能夠在教省內各天主教教堂內舉行聖禮。

== 非教省的總主教 ==
有77個總教區並無自己的附屬教區，這些總教區大多是因為當教區本身所擁有的歷史因素使之獲得總教區頭銜，或是受到行政區域影響，一個國家國土面積過小而無法涵蓋多個教區形成教省，此單一教區則會被給予總教區的頭銜，如[[盧森堡|盧森堡]][[天主教盧森堡總教區|總教區]]。

部分總教區本身就是一個教省總教區的附屬教區。如[[天主教艾克斯總教區|艾克斯總教區]]就是從屬於[[天主教馬賽總教區|馬賽總教區]]。其他的非教省首都的總教區則直屬於[[羅馬教廷|羅馬教廷]]，但他們與鄰近的教省依然會保持合作的關係。

== 領銜總主教 ==
{{Main|領銜主教}}

一些擔任[[教廷|教廷]][[圣部|部]]、會首長或是外交官員的的神職人員，會自一些因取消或改組易名而不再存在的教區名稱中選一個作為其所轄總教區，此主教銜並未對該地區有正式的牧職，稱為領銜總主教，如香港籍神父[[韓大輝|韓大輝]]接任教廷[[萬民福音部|萬民福音部]]秘書長一職後，同時被授予錫拉教區的領銜總主教一職<ref>{{cite news |title = 韓大輝晉牧服務教廷 淺談傳信部歷史職能 |url = http://kkp.catholic.org.hk/gl/gl3494/gl3494_02.htm |accessdate=2012-12-31|newspaper=公教報|date=2011年2月6日}}</ref>。而一名自總主教職務上榮休的總主教則以他原先所管理的總教區作為領銜。

此外一個總教區的[[助理主教|助理主教]]或稱助理總主教也可被視為一個領銜總主教，因為他並非一個實際統領一個教區。如同其他助理主教，總教區的助理主教亦有在現任總主教榮休或辭世而真除總主教的權利。

== 個人加銜 ==
有些非總教區的正權主教，由於個人貢獻而由[[教宗|教宗]]特別給予總主教頭銜，如台灣[[高雄教區|高雄教區]]的[[劉振忠|劉振忠總主教]]，這類型的總主教銜為「個人加銜」（{{lang|la|archbishop ad personam}}）所以不會由下一任的教區主教繼承。

== 總主教的象徵符號 ==

=== 披帶 ===
教省總主教必須向[[教宗|教宗]]取得[[披帶|披帶]]以作為在教省內的職權表徵以及象徵世界各地天主教教會的合一，此外總主教披帶只能在他所管轄的教省內舉行儀式時配掛，在其他教省則不能使用<ref>《天主教法典》第437條</ref>。由於披帶為教省總主教職的象徵，因此總主教的牧徽上往往會加上披帶的符號。不過不是所有的教省總主教都有在牧徽上加上披肩圖像。

至於非教省的總主教和領銜總主教並未實際管理一個總教區，因此也不會領取總主教披帶。

=== 雙槓十字架 ===
總主教習慣上會在牧徽的盾徽上方加上一個雙槓的十字架，有別於一般主教的單槓拉丁十字。

=== 穗帶 ===
總主教牧徽上的帽穗左右各十個，以1-2-3-4的序列排成四行。
