'''總統科學獎'''，為[[中華民國|中華民國]]層級最高的[[基礎科學|基礎科學]]研究獎勵，從2001年開始，每2年頒發一次，每屆得獎人上限為4名。
[[File:3_trophies_of_ROC_Presidential_Science_Prize_20151104.jpg|thumb]]

== 目的 ==

總統科學獎的宗旨在於提升[[臺灣|臺灣]]在國際學術界之地位，獎勵數理科學、生命科學、社會科學及應用科學在國際學術研究上具創新性且貢獻卓著之學者，尤以對臺灣社會有重大貢獻之基礎學術研究人才為優先獎勵對象。

== 評選 ==

總統科學獎委員會由[[中央研究院|中央研究院]]院長、[[中華民國科技部|科技部]]部長、[[總統府|總統府]]副秘書長、[[中華民國教育部|教育部]]部長、[[國家衛生研究院_(中華民國)|國家衛生研究院]]院長及五位大學校長、五位卓越科學研究人員代表共同組成，由中央研究院院長擔任召集人。

候選人需要經過[[中央研究院院士|中央研究院院士]]或總統科學獎得主3人以上聯名推荐，也可以由總統科學獎委員會邀請學術、研發單位或團體推薦。

總統科學獎委員並將聘請數理科學、生命科學、社會科學、及應用科學4個小組的委員，負責審查，之後再由總統科學獎委員會及4個小組的召集人組成的聯席會議，決定得獎名單。

== 獎勵 ==

得獎人將由[[中華民國總統|中華民國總統]]親自頒發[[新臺幣|新臺幣]]200萬獎金、獎狀、獎座，並將會替得獎人舉辦演講等活動。
