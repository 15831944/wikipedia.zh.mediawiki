《'''總有一天'''》（{{lang-ja|いつだって僕らは}}）是日本樂隊[[生物股長|生物股長]]的第22張[[單曲|單曲]]，於2012年1月18日由[[Epic_Records_Japan_Inc.|Epic Records Japan Inc.]]發售。

== 概要 ==
本作跟前作《[[向前走_(單曲)|向前走]]》相隔2個月發售，是[[生物股長|生物股長]]在2012年發行的第1張單曲。單曲在[[限定版|初回期間]]附送「生物卡片027」一張。單曲封面有一個紅色的立方體，其中有「四角」和「資格」的[[雙關語|音義雙關]]。<ref>{{ja icon}}{{Twitter|mizunoyoshiki/status/158887874656337921|水野良樹2012年1月16日在Twitter的訊息}}</ref>另外，上一次單曲封面出現組合成員已是前5作《[[鄉愁_(生物股長單曲)|鄉愁]]》。

== 歌曲 ==
單曲收錄了兩首歌曲，分別A面曲《總有一天》（{{lang|ja|いつだって僕らは}}）及B面曲《紅傘》（{{lang|ja|赤いかさ}}）。另外，除了《總有一天》的純音樂外，亦跟前作《向前走》一樣收錄了B面曲的純音樂版本。

A面曲《總有一天》是一首支持向着目標努力的人和挑戰些甚麼的人的加油歌。<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/61621 |title=いきものがかり、ユーキャンCMに前向き応援ソング提供 |publisher=[[Natalie_(網站)|Natalie]] |date=2011-12-22 |accessdate=2012-05-22}}</ref>這首歌由以往只寫B面曲的[[山下穗尊|山下穗尊]]作曲和填詞，是他首次負責製作A面曲。山下說雖然希望能在歌中表現「無論如何都帶着氣勢去挑戰」，但第二段[[副歌|副歌]]卻完全不浮現出來，最後第二段副歌是錄音途中才寫好的。<ref>{{ja icon}}{{cite web |url=http://www.u-can.co.jp/cm/2012/int_ikimono.html |title=いきものがかりインタビュー |publisher=U-CAN |date=2012 |accessdate=2012-05-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20120329212108/http://www.u-can.co.jp/cm/2012/int_ikimono.html |archivedate=2012-03-29 }}</ref>另外，這首歌有不少押韻的部份，如B段的「try!」和「fly!!」、副歌的「{{lang|ja|いつだって 最高の感情を描いて'''みた'''／僕らそうやって純粋に夢を'''みた'''}}」<ref group="註">歌詞翻譯：無論何時 我們都試過描繪最棒的情感／我們都試過這樣追尋純真的夢。句尾都是「{{lang|ja|みた}}」（Mita）。音樂樣本的第一句。</ref>和「{{lang|ja|果てしなく広がるこの空の'''下（した）'''／僕らはその答えを見つけま'''した'''}}」<ref group="註">歌詞翻譯：在這廣闊無垠的天空下／我們發現了答案。句尾都是「{{lang|ja|した}}」（Shita）。音樂樣本的第二句。</ref>等等。

另外，《總有一天》是{{link-ja|U-CAN|ユーキャン}}2012年度的廣告歌曲。其中生物股長的成員參與演出的「開學編」只於2012年1月1日至3日期間播映。<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/62120 |title=いきものがかり2年ぶりアルバム発売＆3日間限定CM出演 |publisher=[[Natalie_(網站)|Natalie]] |date=2012-01-01 |accessdate=2012-05-22}}</ref>組合上次參與廣告演出約為2年前[[江崎固力果|江崎固力果]][[百奇|百奇]]的廣告，廣告歌曲是《[[YELL/JOYFUL|JOYFUL]]》。

B面曲《紅傘》原本收錄於生物股長[[獨立製作|獨立製作]]時期的第2張專輯《{{lang|ja|[[生物股長#地下時期專輯|七色こんにゃく]]}}》，重新編曲後再收錄到這張單曲中<ref>{{ja icon}}{{cite web|url=http://www.music-lounge.jp/v2/articl/pickup/detail/?articl=2012/01/16-10:10:00_805f2546e93acf1904d4a7462c59223f |title=新年第１弾シングルは未来へ向けて頑張る人へ贈るアッパーチューン！  |publisher=MUSIC LOUNGE |date=2012-01-16 |accessdate=2012-05-22}}</ref>，為首次有這張專輯的歌曲收錄到主流作品內。這首歌是生物股長高校時期的歌曲，是組合最舊的歌曲之一。<ref>{{ja icon}}{{Twitter|mizunoyoshiki/status/149667330807443456|水野良樹2011年12月22日在Twitter的訊息}}</ref>另外，這是自《[[Planetarium|Happy Smile Again]]》以來第二次[[水野良樹|水野良樹]]寫的歌成為B面曲。
