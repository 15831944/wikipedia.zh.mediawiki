《'''欲望者'''》（{{jpn|j=|hg=よくぼうもの|rm=Yokubōmono}}）是日本女子[[日本偶像|偶像]]組合[[NMB48|NMB48]]的第18張[[單曲|單曲]]，於2018年4月4日由[[YOSHIMOTO_R_and_C|laugh out loud! records]]發行。自第16張單曲《[[除我之外的人|除我之外的人]]》以來，再次由[[山本彩|山本彩]]出任中心成員，5期生[[梅山戀和|梅山戀和]]則首次成為選拔成員<ref>{{cite web|url=http://news.nmb48.com/release/20180219/205923.html |language=ja |script-title=ja:NMB48、18thシングル発売決定！ センターは山本彩、梅山恋和が初選抜入り |publisher={{Link-ja|M-ON! MUSIC|M-ON! MUSIC}} |date=2018-02-19 |accessdate=2018-04-13}}</ref>。

== 概要==
單曲分為TYPE-A、TYPE-B、TYPE-C、TYPE-D和劇場盤。《欲望者》和《Thinking time》收錄於所有版本，其中《欲望者》的音樂錄影帶和製作特輯收錄於劇場盤以外的所有版本，劇場盤則收錄了《誤解》。《阪急電車》和其音樂錄影帶、演唱會片段「NMB48 LIVE 2017 in Summer 〜最最最 最棒〜@神戶世界紀念會堂（2017.8.6） 第一輯」收錄於TYPE-A；《成語女孩》和其音樂錄影帶、演唱會片段「NMB48 LIVE 2017 in Summer 〜最最最 最棒〜@神戶世界紀念會堂（2017.8.6） 第二輯」收錄於TYPE-B；《別無藥可救》和其音樂錄影帶、演唱會片段「NMB48 LIVE 2017 in Summer 〜最最最 最棒〜@神戶世界紀念會堂（2017.8.6） 第三輯」收錄於TYPE-C；《Good Timing》和其音樂錄影帶、影片「矢倉楓子 畢業紀念!Golden Hour」收錄於TYPE-D。劇場盤以外的所有版本均附有活動參加券和[[生寫真|生寫真]]，劇場盤則是個別[[握手會|握手會]]的參加券<ref>{{cite web |url=http://nmb48.com/discography/18th_single.html |language=ja |script-title=ja:18th Single 「欲望者」 |publisher=[[NMB48|NMB48]] |accessdate=2018-04-12 |deadurl=no |archiveurl=https://web.archive.org/web/20180412201247/http://nmb48.com/discography/18th_single.html |archivedate=2018-04-12 }}</ref>。2018年2月19日，NMB48在[[NMB48劇場|NMB48劇場]]舉行的[[Team_A_5th_Stage「戀愛禁止條例」|Team BII 4th Stage「戀愛禁止條例」]]公演中宣佈推出單曲<ref>{{cite web |url=https://www.m-on-music.jp/0000255172/ |language=ja |script-title=ja:【発売決定】NMB48 18thシングル「タイトル未定」2018年4月4日（水）発売決定！ |publisher=NMB48 |date=2018-02-19 |accessdate=2018-04-13 |deadurl=no |archiveurl=https://web.archive.org/web/20180412201247/https://www.m-on-music.jp/0000255172/ |archivedate=2018年4月12日 }}</ref>。3月6日，NMB48公佈了單曲的收錄內容<ref>{{cite web |url=https://www.oricon.co.jp/news/2107011/full/ |language=ja |script-title=ja:NMB48、新曲「欲望者」全容公開 太田夢莉が参加 矢倉&白間デュエット曲も |publisher=[[Oricon|Oricon]] |date=2018-03-06 |accessdate=2018-04-13 |deadurl=no |archiveurl=https://web.archive.org/web/20180307022824/https://www.oricon.co.jp/news/2107011/full/ |archivedate=2018-03-07 }}</ref>。

== 反應 ==
在《[[告示牌_(雜誌)|告示牌]]》Top Singles Sales榜中，單曲銷出207,448張，取得週榜第1位<ref>{{cite web |url=https://dot.asahi.com/billboardnews/2018041000008.html?page=1 |language=ja |script-title=ja:【ビルボード】NMB48『欲望者』が207,448枚を売り上げシングル・セールス首位獲得 |date=2018-04-10 |publisher=[[告示牌_(雜誌)|告示牌]] |accessdate=2018-04-13 |deadurl=no |archiveurl=https://web.archive.org/web/20180412201247/https://dot.asahi.com/billboardnews/2018041000008.html?page=1 |archivedate=2018-04-12 }}</ref>，[[日本熱門100金曲榜|日本熱門100金曲榜]]亦同樣取得第1位<ref>{{cite web |url=https://dot.asahi.com/billboardnews/2018041100075.html?page=1 |language=ja |script-title=ja:【ビルボード】NMB48「欲望者」が207,448枚を売り上げ総合首位　米津玄師「Lemon」8週連続ダウンロード1位 |date=2018-04-11 |publisher=告示牌 |accessdate=2018-04-13 |deadurl=no |archiveurl=https://web.archive.org/web/20180412201247/https://dot.asahi.com/billboardnews/2018041100075.html?page=1 |archivedate=2018-04-12 }}</ref>。在[[Oricon公信榜|Oricon公信榜]]中，單曲以約19.4萬銷量，取得週榜第1位，連續7張單曲取得第1位，總計則是16張單曲取得冠軍，僅次於[[AKB48|AKB48]]（38張）、[[早安少女組。|早安少女組。]]、[[乃木坂46|乃木坂46]]和[[SKE48|SKE48]]（18張），在女子組合取得週榜冠軍次數中排第5位<ref>{{cite web |url=https://www.oricon.co.jp/news/2109242/full/ |language=ja |script-title=ja:【オリコン】NMB48、シングル通算16作目首位 女性グループ歴代5位記録更新 |publisher=Oricon |date=2018-04-10 |accessdate=2018-04-12 |deadurl=no |archiveurl=https://web.archive.org/web/20180412201247/https://www.oricon.co.jp/news/2109242/full/ |archivedate=2018-04-12 }}</ref>。
