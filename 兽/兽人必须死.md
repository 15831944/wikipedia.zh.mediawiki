{{vgname|兽人必须死|en=Orcs Must Die!}}是由[[罗伯特娱乐|罗伯特娱乐]]开发的动作[[塔防游戏|塔防游戏]]。与其他的塔防游戏不同，该游戏使用第三人称视角而非俯视视角。<ref>{{cite web |url=http://www.gamepro.com/article/previews/218504/pax-east-orcs-must-die-preview/ |title=PAX East: Orcs Must Die Preview |publisher=[[GamePro|GamePro]] |date=2011-03-13 |accessdate=2011-03-15 |first=Dennis |last=Scimeca |archiveurl=https://www.webcitation.org/query?url=http://www.gamepro.com/article/previews/218504/pax-east-orcs-must-die-preview/&date=2011-11-30+22:18:43 |archivedate=2011-11-30 |deadurl=yes }}</ref>在Penny Arcade Expo East 2011证实该游戏的存在，2011年10月5日在[[Xbox_Live_Arcade|Xbox Live Arcade]]发行，2011年10月12日发行[[Windows|Windows]]版本。<ref>{{cite web |url=http://www.destructoid.com/hands-on-orcs-must-die--194575.phtml |title=Hands-on: Orcs Must Die! |publisher=[[Destructoid|Destructoid]] |date=2011-02-25 |accessdate=2011-03-15 |first=Colette |last=Bennett}}</ref>
