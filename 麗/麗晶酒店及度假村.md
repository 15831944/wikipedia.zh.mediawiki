'''麗晶酒店及度假村'''（{{lang-en|Regent Hotels & Resorts}}）或稱'''麗晶國際酒店'''（{{lang-en|Regent International Hotels}}），是[[洲際酒店集團|洲際酒店集團]]與[[晶華國際酒店集團|晶華國際酒店集團]]旗下的合資跨國[[酒店|酒店]]公司，目前據點分布[[亞洲|亞洲]]、[[歐洲|歐洲]]等地。

== 沿革 ==
麗晶酒店由旅館大亨[[羅伯特·伯恩|羅伯特·伯恩]]與日本[[東急|東急集團]]於1970年共同成立，曾在[[紐約|紐約]]、[[巴里島|巴里島]]、[[米蘭|米蘭]]、[[伊斯坦堡|伊斯坦堡]]等大城與旅遊勝地拓點。

1992年售予[[四季酒店|四季酒店集團]]後，將所有原屬麗晶品牌的酒店項目改以四季酒店品牌經營或管理。

1998年，[[卡尔森酒店集团|卡尔森酒店集团]]買下麗晶酒店經營及管理權。四季酒店可以選擇以麗晶品牌營運此前已經管理的酒店<ref>四季酒店已悉數將其原來持有的麗晶酒店管理合約改以四季品牌經營。新加坡麗晶酒店則為特例，由四季酒店繼續予以保留麗晶酒店的品牌截至2018年12月31日該管理合約正式終止為止。目前新加坡麗晶酒店的管理合約由洲際酒店集團簽訂，與四季酒店無任何關係</ref>。

2010年4月16日，卡爾森集團再將麗晶酒店售予[[晶華麗晶酒店集團|晶華麗晶酒店集團]]（即台北晶華酒店業主），創下台灣本土旅館業者收購國際品牌首例<ref name="晶華收購麗晶">[http://www.appledaily.com.tw/appledaily/article/finance/20100417/32443683/ 晶華18億收購麗晶酒店品牌 - 台灣《蘋果日報》 2010年4月17日]</ref>。晶華酒店收購麗晶酒店品牌後，規劃將台北晶華酒店塑造為麗晶酒店連鎖體系的品牌[[旗艦店|旗艦店]]<ref name="晶華收購麗晶" />。

2018年3月14日晶華國際酒店董事會，通過與[[洲際酒店集團|洲際酒店集團]]成立合資公司—'''麗晶酒店及度假村'''，負責經營開發麗晶酒店於台灣以外地區的全球性業務，合資公司晶華將持股49%，51%股權以3930萬美元轉讓給洲際酒店集團。計劃將現時6間酒店規模，逐步拓展至40間，共逾萬間客房；更會將[[香港洲際酒店|香港洲際酒店]]大翻新，在2020年農曆年後，進行為期逾一年的裝修，完工之時改用原名「麗晶酒店」。<ref>{{cite web |author1=蘋果日報 |title=洲際隔20年變回「麗晶」 擬作大翻新　定位更豪華 |url=https://hk.finance.appledaily.com/finance/daily/article/20181108/20541333 |website=https://hk.finance.appledaily.com/finance/daily/article/20181108/20541333 |publisher=蘋果日報 |accessdate=2018-11-15}}</ref>

== 旗下據點 ==
除酒店經營業務外，[[晶華麗晶酒店集團|晶華麗晶酒店集團]]亦擁有[[加勒比海|加勒比海]][[麗晶郵輪|麗晶郵輪]] ''Regent Seven Sea Cruises'' 的品牌特許權（由麗笙酒店（Radisson Blu Hotels）以麗晶品牌特許經營）{{Fact|time=2019-01-23T13:53:45+00:00}}。
