{{engname|麗瑤邨|Lai Yiu Estate}}是[[香港|香港]]的一個[[公共屋邨|公共屋邨]]，位於[[新界|新界]][[葵青區|葵青區]]，由[[香港房屋委員會|香港房屋委員會]]發展，並由雅居物業管理有限公司負責屋邨管理。

{{engname|翠瑤苑|Tsui Yiu Court}}是由[[香港房屋委員會|香港房屋委員會]]發展的[[居者有其屋計劃|居者有其屋計劃]]屋苑，位於[[香港|香港]][[新界|新界]][[葵涌|葵涌]][[荔景山|荔景山]]，在1981年落成，是香港最早落成的居屋屋苑之一，由[[巴馬丹拿|巴馬丹拿]]建築及工程師有限公司設計，以[[穗禾苑|穗禾苑]]的錯層式十字型設計作藍本，採用了3層共用一部電梯，以及以3個單位為一組共用一條樓梯的設計，目的是為了加強鄰里守望相助的精神以減低罪案的發生，而開放式大堂也加強了自然通風和採光，各翼不同高度令立面更具特色。

翠瑤苑鄰近[[華員邨|華員邨]]及[[青山公路|青山公路葵涌段]]。屋苑只有一座，大部份單位景觀開揚，中層以上向南的單位更可遠晀中環一帶及維港海景。

跟一般居屋不同，翠瑤苑所有單位均不用[[補地價|補地價]]，而且一律都可以在公開市場自由轉讓。<ref>{{Cite web |url=http://valuers.hk/property-valuation-hos/1523.html# |title=毋需補地價的居屋 |access-date=2015-08-13 |archive-url=https://web.archive.org/web/20140222022320/http://valuers.hk/property-valuation-hos/1523.html# |archive-date=2014-02-22 |dead-url=yes }}</ref>

== 介紹 ==
麗瑤邨位於[[葵涌|葵涌]]第11C區，建在[[下葵涌|下葵涌]]一處陡峭的山坡上，鄰近[[青山公路|青山公路]]葵涌段。麗瑤邨是按前[[香港屋宇建設委員會|香港屋宇建設委員會]]的規格與標準興建，是首個獲得煤氣供應的公共屋邨<ref>華僑日報，1977年8月20日</ref>。邨內原有3座24層高[[雙塔式|雙塔式]]大廈（樂瑤樓、富瑤樓、貴瑤樓）及1座12層高[[舊長型|舊長型]]大廈（華瑤樓），華瑤樓更是70年代公屋中罕有地在各單位備配備兩個梗房與浴缸的房委會公屋大廈，這4座在1976至1977年期間落成。麗瑤邨是[[荃灣新市鎮|荃灣新市鎮]]首條擁有雙塔式大廈的[[公共屋邨|公共屋邨]]，大廈單位較舊式屋邨寬敞，很多住戶都願由舊型屋邨調遷至該邨，所以該邨對於減輕當時舊型屋邨的居住擠迫情況，佔有重要地位。

1999年，[[房委會|房委會]]為求增加公屋供應，遂委托鍾華楠建築師事務所於本邨室內停車場對出的山邊位置設計及加建設計獨特的榮瑤樓，榮瑤樓外立面設計充滿不同顏色及裝飾，不同方向的單位有不同的樓宇高度，形成了大廈錯落有緻，因此天台花園分佈於不同高度的位置；1-5樓為長者住屋，而6-20樓為小單位大廈配小量家庭式單位，主要安置區內的獨居長者，亦為葵涌邨2，3及4期重建主要接收資源，此大廈亦令本邨樓宇增加至5座。

本邨設施確實不多，現時只有一所[[中學|中學]]（[[葵涌循道中學|葵涌循道中學]]）、一所[[小學|小學]]（[[東華三院高可寧紀念小學|東華三院高可寧紀念小學]]）、一所[[幼稚園|幼稚園]]（[http://www.fy.edu.hk 富瑤幼稚園]）、一所[[國際學校|國際學校]]（[[宣道會劉平齋紀念國際學校|宣道會劉平齋紀念國際學校]]（3年級或以下））、一所家舍/宿舍（[https://www.fuhong.org/PageInfo.aspx?md=20000&cid=264 扶康會麗瑤成人訓練中心]）、巴士總站，2008年增設一間[[7-11|7-11]]、紅色小巴站（設在[[麗瑤街|麗瑤街]][[麗瑤商場|麗瑤商場]]側）、綠色小巴（包括90M往[[美孚新邨|美孚新邨]]、47M往[[荔景站|荔景站]]、91A往[[葵芳|葵芳]][[新都會廣場|新都會廣場]]、91往[[荃灣|荃灣]]鹹田街、411往[[深水埗|深水埗]][[九江街|九江街]]）、麗瑤商場、室內停車場等。本邨榮瑤樓旁邊設有燒烤場，只供本邨居民租用，屬於全港唯一一個設有燒烤場的公共屋邨。至於其他設施多依賴山下的[[荔景|荔景]]及[[葵芳|葵芳]]。

本邨的位置比較隔涉，交通頗為不便，[[巴士總站|巴士總站]]雖有4條巴士線，但除前往[[天恆邨|天恆邨]]的路線265M外，其餘三條路線（30線往長沙灣、45線往[[土瓜灣|土瓜灣]]、46線往[[佐敦|佐敦]]）的班次都是非常疏落，而且這些路線也是[[葵青區|葵青區]]班次最疏落巴士路線之一。不過，由於本邨有較多長者居住，所以他們都會刻意等候巴士，因為可以享有2元乘車優惠。由於榮瑤樓附近有升降機塔及小徑連接山腳的[[下葵涌村|下葵涌村]]，居民為節省等車時間，選擇先沿這小徑步行到[[葵芳|葵芳]]，再在葵芳轉乘其他交通工具（如[[港鐵|港鐵]]），因此步行漸變成居民出入的主要方法。此外，本邨鄰近[[青山公路|青山公路]]葵涌段，有多條巴士線途經，分別前往[[東九龍|東九龍]]、[[深水埗|深水埗]]及[[油尖旺|油尖旺]]、[[青衣|青衣]]及[[葵涌|葵涌]]各地。

== 區域分界爭議 ==

葵青區南面以[[呈祥道|呈祥道]]為界，一些被人視為九龍的地區，如[[華荔邨|華荔邨、荔欣苑]]、荔灣花園、華豐園等位於[[荔園|荔園]]原址的住宅以及[[九華徑|九華徑]]及鐘山臺一帶，都是全屬於新界'''葵青區'''。反而位於上述地區以西的[[清麗苑|清麗苑]]是屬於九龍[[深水埗區|深水埗區]]。

荔景山區居民概念爭議：
 
由於麗瑤邨（包括由[[荔景|荔景]]、[[祖堯|祖堯]]及[[瑪嘉烈醫院|瑪嘉烈醫院]]一帶），過往未取消電話地區字頭時是跟九龍區的「3」字區域，居民早已根深柢固，在寫地址時常常把荔景、葵芳等寫成九龍荔景、九龍華景山莊、華員邨等並不罕見，但這其實是錯誤的，因為電話公司只是商業機構。最切實的是九龍不可能建有[[丁屋|丁屋]]，但葵青區仍存有丁屋。
