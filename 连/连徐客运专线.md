== 线路 ==
当前线路由连云港站引出，分别通过[[东海|东海]]、[[新沂|新沂]]和[[邳州|邳州]]，最终接入[[京沪高铁|京沪高铁]]徐州东站。连徐客运专线共设车站7座：连云港站、东海站、新沂南站、邳州东站、大许南站（预留站）、后马庄站和徐州东站。

== 技术标准 ==
* 铁路等级：客运专线。
* [[正线|正线]]数目：双线。
* 限制[[坡度|坡度]]：一般路段12‰，困难路段20‰。
* 路段[[旅客列车|旅客列车]]设计行车速度：350[[公里/小时|公里/小时]]，局部限速。
* [[最小曲线半径|最小曲线半径]]：一般路段7000米，困难路段5500米。
* 牵引种类：[[电力机车|电力]]牵引。 	
* [[到发线|到发线]]有效长度：
* [[闭塞_(铁路)|闭塞]]类型：[[自动闭塞|自动闭塞]]。
