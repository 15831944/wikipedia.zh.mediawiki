'''連珠'''是人們對“[[五子棋|五子棋]]”遊戲做了改進，增加了一些規則，使其具有較強競技性後，形成的一套類似五子棋的遊戲體系。

== 歷史由來 ==
[[File:Renju.jpg|thumb]]
五子棋在日本曾被稱為“五石”、“{{lang|ja|五目並べ}}”、“格五”、“五法”、“五联”、“京棋”等。但中國古代文獻中的“[[格五|格五]]”並不是五子棋的一種。“[[朝鮮五目|朝鮮五目]]”、“[[朝鮮碁|朝鮮碁]]”專指連珠的一種變體[[二拔連珠|二拔連珠]]<ref>[http://i-alpha.com/renju/gametop.htm 二抜き連珠(NINUKI)] {{webarchive|url=https://web.archive.org/web/20110526192658/http://i-alpha.com/renju/gametop.htm |date=2011-05-26 }}</ref>與現代連珠不同。

1856年，《五石定式集》出版，随后各种五子棋的书籍纷纷出现，各种流派在[[日本|日本]]发展起来。1899年在[[大阪|大阪]]、[[东京|东京]]、[[北陆|北陆]]([[富山县|富山县]]和[[石川县|石川县]])等地出现的高手举行了与其他流派的比赛。當時的比赛规则里，黑白双方禁止走“[[雙活三|雙活三]]”，至此五子棋里出现了“[[禁手|禁手]]”的概念。 

《万朝报》于1899年9月连载了社长黑岩泪香（后成为初代名人）所写的《五子棋必胜法》，引起很大反响。执黑必胜使五子棋失去了比赛和研究的意义。五子棋规则已經到了必须改革的時候。黑岩泪香对将五子棋称为“{{lang|ja|五目並べ}}”的这种含轻视的称谓表示遗憾，同年12月6日《万朝报》向全日本民眾發起公开征名，最終以“聯珠”这一名称替代“{{lang|ja|五目並べ}}”，“聯珠”語出《[[汉书|漢書·律历志]]》“日月如合璧，五星如联珠”。1946年，日本更改當用漢字，以“連珠”替代“聯珠”，現代“連珠”的稱為因此而來。

== 沿革 ==

连珠的规则多年不断改良，進程如下：

* 1903年提出了按珠形分为桂、间、连的21种开局，并规定禁止黑方走“三三”，白方無行棋限制。
* 1912年增加规定，黑方被迫走“三三”也算输。
* 1916年增加规定，黑方禁止走“长连”（超過五連）。
* 1918年增加规定，黑方禁止走“四三三”。
* 1931年提出，黑方禁止走“四四”，同年有些人開始提出，將棋盤由十九路改為十五路盘。
* 1960年，經過長期爭論，正式将十九路盘改为十五路盘。
* 1988年增加规定，為削弱黑棋，第五步需走兩個位置，白方擇其一個，簡稱“五手兩打”。
* 1995年增加规定，下完前三步，執白方有交換權，即“三手可交换”。
* 1996年增加规定，執黑方開局下三子，“指定開局”。

==術語==

*連五：縱、橫、斜線同色五子的緊連。
*活四：兩端都可形成連五的四。
*沖四：僅一端能形成連五的四。
*活三：指可形成活四的三。
*休三：指因禁手點無法形成活四的三。
*眠三：僅能形成沖四的三。
*四四：又稱雙四，一子出現兩條或以上活四或沖四。
*三三：又稱雙活三，一子出現兩條或以上活三。
*長連：縱、橫、斜線同色形成超過五子的連線。
*禁手：黑子禁止下出雙活三、雙四、長連，下出則判敗<ref>[http://i-alpha.com/renju/johobox/yougo01.htm 連珠の用語集] {{Wayback|url=http://i-alpha.com/renju/johobox/yougo01.htm |date=20090516055010 }}</ref>。

== 規則 ==
為適應不同層次的連珠愛好者參與比賽，現代連珠比賽有多套規則並行。

; 一般連珠规则（日式规则）
# 黑子先行，白子後行，輪流落子。
# 黑棋有禁手，黑棋禁止双活三，双四，三個三，三個四，四三三，四四三，长连。

; [[RIF规则|RIF规则]]（三手交換，五手兩打規則）
# 开局方（假先方）下前三手，限定在26種[[連珠開局|連珠開局]]<ref name="RIF">[http://www.renju.net/study/rifrules.php 但在1996年之前RIF規則第二手是由假後方選擇]</ref>。（指定開局）
# 另一方（假后方）選擇持黑或白。（三手可交换）
# 白方下出第四手。
# 第五手黑方必须提供不同型的兩個打点。（五手兩打）
# 白方从中选择一点，当第五手。
# 此后按连珠规则继续进行。（黑子有禁手）
# ※為[[国际连珠联盟|国际连珠联盟]]在第一屆到第十屆[[世界连珠锦标赛|世界连珠锦标赛]]的使用規則。

由於三手交換五手兩打規則變化有限，為豐富競技比賽，相繼衍生出[[山口规则|山口规则]]、[[索索夫规则|索索夫规则]]、[[坂田规则|坂田规则]]、[[塔拉-山口規則|塔拉-山口規則]]、[[塔拉尼科夫规则|塔拉尼科夫规则]]、[[連換規則|連換規則]]等。目前國際連珠聯盟認可上述前四種規則<ref>[http://www.renju.net/organizations/ga2008.php The certified rules in alphabetical order are: Soosõrv, Taraguchi and Yamaguchi.]</ref><ref>[http://www.renju.net/organizations/ga2009.php The meeting decided to certify Sakata rule as well.]</ref>。

; [[坂田规则|坂田规则]]：
# 开局方下前三手，限定在26種連珠開局。
# 另一方下第四手（中央7×7區域內）和第五手（中央9×9區域內）。
# 开局方選擇持黑或白。
# 此后按连珠规则继续进行。（黑子有禁手）

; [[山口規則|山口規則]]：
# 开局方下前三手棋，限定在26種連珠開局，同时提议五手打点的数目。
# 另一方選擇持黑或白。
# 白方下第四手。
# 第五手黑方必须提供不同型的提议数目打点。
# 白方从中选择一点，当第五手。
# 此后按连珠规则继续进行。（黑子有禁手）
# ※為第十一屆到第十四屆連珠世錦賽的使用規則。

; [[索索夫規則|索索夫規則]]：
# 开局方下前三手棋，限定在26種連珠開局。
# 另一方選擇持黑或白。
# 白方下出第四手，同时提议五手打点需1、2、3或4個。
# 另一方選擇持黑或白。
# 第五手黑方必须提供不同型的提议数目打点。
# 白方从中选择一点，当第五手。
# 此后按连珠规则继续进行。（黑子有禁手）
# ※索索夫-N規則代表其五手打點最多N個。
# ※索索夫-8規則為目前[[世界连珠锦标賽|世界连珠锦标賽]]使用規則，自2017年第十五屆[[世界连珠锦标賽|世界连珠锦标賽]]開始試行<ref>[http://www.renju.net/media/news.php?news_no=371 Decided: the opening rule since 2017 will be Soosorv-8. ]</ref>。

; [[塔拉尼科夫规则|塔拉尼科夫规则]]：
# 開局方在天元放第一手（黑子），由另一方選擇持黑或白。
# 白方在中央3×3區域內放第二手（白子），由另一方選擇持黑或白。
# 黑方在中央5×5區域內放第三手（黑子），由另一方選擇持黑或白。
# 白方在中央7×7區域內放第四手（白子），由另一方選擇持黑或白。
# 黑方在中央9×9區域內放第五手（黑子），由另一方選擇持黑或白。
# 此后按连珠规则继续进行。（黑子有禁手）

; [[塔拉-山口规则|塔拉-山口规则]]：
# 開局方在天元放第一手（黑子），由另一方選擇持黑或白。
# 白方在中央3×3區域內放第二手（白子），由另一方選擇持黑或白。
# 黑方在中央5×5區域內放第三手（黑子），由另一方選擇持黑或白。
# 白方在中央7×7區域內放第四手（白子）。
# 另一方有兩種選擇。
## 選擇持黑或白。之後黑方在中央9×9區域內放第五手（黑子），由另一方選擇持黑或白。
## 第五手黑方必须提供不同型的五個打点。白方从中选择一点，当第五手。
# 此后按连珠规则继续进行。（黑子有禁手）
# ※是[[山口规则|山口规则]]和[[塔拉尼科夫规则|塔拉尼科夫规则]]的混合。塔拉-山口-N規則代表其五手打點有N個。

== 國際組織 ==
1988年8月8日瑞典成立了[[国际连珠联盟|国际连珠联盟]]（Renju International Federation，RIF）这样的一个连珠国际性组织，创始成员国有日本、苏联、瑞典。
自1989年起，连珠世界錦標賽（個人賽）每两年举办一次，分资格赛（QT）及之後的冠军赛（AT）、女子赛（WT）和公开赛（BT）四个比赛项目。至第十五屆，日本獲得四次冠軍，愛沙尼亞六次，中國三次，俄羅斯两次。第十一屆同時也復辦五子棋（Gomoku）世錦賽。

自1996年起，每兩年舉辦一次世界團體錦標賽（團體賽）。还有利用e-mail的方式进行的[[世界连珠通讯战|世界连珠通讯战]]（World Championship in Correspondence Renju），按水平分组，每年举行一次。

== 计算机与连珠 ==
原始的没有附加规则的连珠被证明存在完美对策使得先手必胜<ref>{{cite journal | url=https://pdfs.semanticscholar.org/0cc4/0a7750b14ccf417faa39ac292e8d096617bc.pdf | author=J. Wágner and I. Virág | title=Solving Renju | journal=ICGA journal | volume=24 | number=1 | pages=30—35 | date=Mar 2001 | access-date=2017-02-26 | archive-url=https://web.archive.org/web/20170226212908/https://pdfs.semanticscholar.org/0cc4/0a7750b14ccf417faa39ac292e8d096617bc.pdf | archive-date=2017-02-26 | dead-url=yes }}</ref>。然而，对于专业比赛中采用的平衡开局规则，如目前世錦賽使用的Soosõrv-8的規則，并没有被证明存在必胜策略。

世界电脑连珠锦标赛（The Renju World Computer Championship）从1991年开始，至2004年终止共举办了4届<ref>{{Cite web|url=http://www.5stone.net/wc/worldcup_en.html|title=Renju Computer World Championship|website=www.5stone.net|access-date=2016-06-02}}</ref><ref>{{Cite web|url=http://nosovsky.narod.ru/forth.html|title=4-th World Championship among Computer programs|last=|first=|date=|website=Nosovsky Japanese Games Home Page|publisher=|access-date=2016-06-03}}</ref>。自2016起，五子棋人工智能比赛[[Gomocup|Gomocup]]增设了连珠组，比赛每年举行一次。

2000年，计算机程序Meijin-2000参与了莫斯科公开赛（Moscow Open Tournament），使得它成为首个在公开比赛中与人类棋手较量的程序<ref>{{Cite web|url=http://www.renju.net/media/news/news00a.htm|title=Renju Newsletter Januari 2000 - June 2000|last=|first=|date=|website=renju.net|publisher=|access-date=}}</ref>。然而，2017年前，并没有计算机程序在公开比赛中战胜人类顶尖高手的记录。2017年7月，人工智能冠军程序[[弈心|弈心]]<ref>{{Cite web|url=http://www.aiexp.info/pages/yixin.html|title=Yixin, the Strongest Gomoku/Renju Engine in the World|last=|first=|date=|website=AIEXP|publisher=|access-date=}}</ref>与台湾名人林书玄进行了4局比赛，弈心以3胜1负获胜<ref>{{Cite web|url=http://www.aiexp.info/ai-vs-human-renju-2.html|title=Lin Shu-Hsuan versus Yixin|last=|first=|date=|website=AIEXP|publisher=|access-date=}}</ref>。2018年4月，[[弈心|弈心]]与前世界冠军[[祁观|祁观]]进行了5局比赛，并以2.5:2.5的成绩战平。<ref>{{Cite web|url=http://wuziqi.org/records/|title=AI vs Human LIVE|website=wuziqi.org|access-date=2018-11-27}}</ref>

== 连珠活动在中国的情况 ==
1990年，由那威等十几名五子棋爱好者，共同发起并成立了中国第一个现代专业五子棋的民间组织——京都五子棋队。那威于1995年7月只身赴[[爱沙尼亚|爱沙尼亚]]，参加第四届[[世界连珠锦标赛|世界连珠锦标赛]]國際公開賽（BT），并排名第28位。

[[中华人民共和国|中华人民共和国]]于1996年正式加入[[国际连珠联盟|国际连珠联盟]]。[[CCTV|CCTV]]-5于同年开播了由那威、彭建国主讲的连珠五子棋电视讲座。1999年7月，在中国北京成功地举办了第六届[[世界連珠锦标赛|世界連珠锦标赛]]。来自11个国家70多选手参加了三个组别的角逐。中国选手张进宇在A组取得了第九名，白涛在公開賽（BT）獲得第一名，少年选手陆瑶獲得女子组第四名。2001、2002年中国棋手张进宇（freefish）连续两次获得世界连珠通信錦標赛亚军。2003年，世界连珠通信錦標赛冠军最终被中国棋手陈伟（chinastar）夺得。

2007年8月15日，[[吴镝|吴镝]]夺得第10届[[世界连珠锦标赛|世界连珠锦标赛]]A组冠军，这是中国棋手首次夺得五子棋世界冠军。2011年，[[曹冬|曹冬]]夺得第12届[[世界连珠锦标赛|世界连珠锦标赛]]A组冠军。2012年，世界连珠团体锦标赛于北京举办。2015年，[[祁观|祁观]]夺得第14届连珠世锦赛A组冠军。
