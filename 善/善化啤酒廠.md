'''善化啤酒廠'''為[[臺灣菸酒公司|臺灣菸酒公司]]旗下的啤酒生產工廠，廠址位於[[臺灣|臺灣]][[臺南市|臺南市]][[善化區|善化區]]。該廠也是臺灣菸酒公司於[[臺灣南部|臺灣南部]]最重要的[[啤酒|啤酒]][[生產線|生產線]]，生產量佔臺灣菸酒啤酒總產量約38%<ref name="善化區公所"/>，總佔地面積為17.787公頃。<ref name="善化高中"/>

== 沿革 ==
善化啤酒廠前身為[[臺灣省菸酒公賣局|臺灣省菸酒公賣局]]成功啤酒廠。1960年代臺灣啤酒市場銷售量逐步成長，鑑於成長之迅速，公賣局即決定進行[[臺灣北部|臺灣北部]][[建國啤酒廠|建國啤酒廠]]的擴建以及中部增建一座新啤酒廠以應付逐漸擴大的啤酒需求量。<ref name="善化高中"/><ref name="善化啤酒廠"/>

然而後續因建國啤酒廠及中部[[烏日啤酒廠|烏日啤酒廠]]因廠區限制無法再繼續擴建增加產能。公賣局逐決定於臺灣南部尋覓適當廠址來興建啤酒生產工廠。經尋找臺南、[[高雄|高雄]]以及[[屏東|屏東]]地區後，公賣局認為臺南縣善化鎮（今臺南市善化區）鄰近[[臺1線|臺1線]][[縱貫公路|縱貫公路]]、鎮內多為[[農田|農田]]，用地單純，以及鄰近灌溉用[[水圳|水圳]][[水源|水源]]取得容易，加上當時的善化鎮鎮長與鎮民代表極力支持公賣局進駐設廠。<ref name="善化高中"/><ref name="善化啤酒廠"/>

因此公賣局決議於臺南縣善化鎮興建成功啤酒廠，1972年7月公賣局率先派員前往[[西歐|西歐]]各國之現代化啤酒生產工廠進行考察，了解生產設備之概況與生產技術上的問題。1973年12月成功啤酒廠正式動工，歷經2年施工於1975年12月完工開始生產啤酒，年總生產量為啤酒700萬打<ref name="臺南市政府經發局"/>，總投資金額為[[新臺幣|新臺幣]]10億7千萬元。<ref name="善化高中"/><ref name="善化啤酒廠"/>

1980年公賣局因應啤酒供應市場持續擴大之需求進行成功啤酒廠之二期擴建工程，並引進新的啤酒釀製技術，於1984年完工投產<ref name="善化高中"/>，爾後第三度擴建後，年總生產量到達啤酒2,700萬打<ref name="臺南市政府經發局"/>。2002年7月配合臺灣省菸酒公賣局改制為臺灣菸酒股份有限公司，因此成功啤酒廠改名為善化啤酒廠，全名改為「臺灣菸酒股份有限公司善化啤酒廠」。<ref name="善化啤酒廠"/>

2004年起善化啤酒廠經營方向改變，增設啤酒文化園區供遊客來訪參觀<ref name="善化啤酒廠"/>，成為臺灣第一座以啤酒為主題的觀光酒廠<ref name="臺南市政府"/>。2011年善化啤酒廠榮獲[[工業技術研究院|工業技術研究院]]民國100年度節約能源績優優等獎<ref name="工研院"/>。2017年7月8日於善化啤酒廠停辦6年的啤酒節重新舉辦。<ref name="中時電子報"/>

== 生產設備 ==
[[File:TTL_Shanhua_Beer_Factory01.jpg|thumb]]
善化啤酒廠當前廠房總建築面積為39,206坪，廠內設備包含兩座[[麥芽|麥芽]]工廠、糖化設備三套、1,000公石室內發酵槽26座、1,500公石室內儲酒槽104座、5,600 公石屋外發酵儲酒單一槽30座、瓶裝啤酒包裝機3套、罐裝啤酒包裝機2套、40公升生啤酒包裝機1套、[[變電所|變電所]]一座，以及[[汙水處理場|汙水處理場]]1座。<ref name="善化高中"/>

== 生產項目 ==
善化啤酒廠主要生產臺灣菸酒公司出產之罐裝0.33公升與0.5公升[[臺灣啤酒|臺灣啤酒]]、臺灣金牌啤酒、瓶裝0.6公升臺灣啤酒、臺灣金牌啤酒、18天臺灣生啤酒，以及40公升桶裝臺灣生啤酒等8項啤酒產品。<ref name="臺南市政府經發局"/>
