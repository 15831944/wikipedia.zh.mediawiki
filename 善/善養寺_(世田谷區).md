'''善養寺'''（[[日語|日語]]：ぜんようじ）是位於日本[[東京都|東京都]][[世田谷區|世田谷區]][[野毛_(世田谷區)|野毛]]的寺院。屬於[[真言宗智山派|真言宗智山派]]，最初是在[[江戶時代|江戶時代]]初期的深澤村（現在[[深澤_(世田谷区)|深澤]]地區的一部分），在[[慶安_(年號)|慶安]]年間（1648年-1651年）移轉至現在地<ref name="ふるさと世田谷野毛上野毛54-56">『ふるさと世田谷を語る 野毛・上野毛』54頁-56頁。</ref>。在本堂前方的[[日本榧樹|大榧木]]（{{tsl|ja|善養寺のカヤ|善養寺榧木}}），其樹齡約700年~800年，在1964年被指定為東京都的[[自然紀念物|自然紀念物]] <ref name="東京都の文化財3">『東京都の文化財 3』84頁。</ref>。

 
== 歷史 ==

善養寺在[[多摩川|多摩川]]畔、沿著[[國分寺崖線|國分寺崖線]]的位置。正式名稱為「'''影光山佛性院 大毘廬遮那殿 善養密寺'''（日語：ようこうさんぶっしょういんだいびるしゃなでんぜんようみつじ）」，是 [[京都市|京都市]][[東山區_(日本)|東山區]][[智積院|智積院]]的末寺<ref name="せたがやの散歩道">『改訂・せたがやの散歩道』163頁。</ref><ref name="ふるさと世田谷野毛上野毛116118">『ふるさと世田谷を語る 野毛・上野毛』116頁-118頁。</ref>。

在江戶時代以前，此寺據說是在深澤村<ref name="史料に見る江戸時代の世田谷">『史料に見る江戸時代の世田谷』106-108頁。</ref>。寺院的僧侶--祐榮阿闍梨（慶安5年7月26日遷化<ref>遷化，佛教用語，指僧侶的死亡、入滅。</ref>）<ref name="史料に見る江戸時代の世田谷"/>去本山智積院修業後，再回到深澤村，但他不在寺院的期間，寺院已經被毀壞。傳說慶安年間祐榮將寺院移轉至下野毛村<ref>下野毛村是從野毛一丁目起至三丁目和野良田（1932年玉川仲町的改稱，1969年的住居表示之際時成為中町）的[[飛地|飛地]]所構成。1889年與用賀村、瀬田村、上野毛村、等等力村、奥澤村、尾山村、野良田村等等合併成[[東京府|東京府]][[荏原郡|荏原郡]]玉川村。</ref>字根通路，但在《新編武藏風土記稿》卷之四十九，〈荏原郡世田谷領菅刈庄下野毛村〉之條目，紀載著「那一年的紀載不明」，因此關於移轉至現在地以前的善養寺之史實文獻知之甚少<ref name="せたがやの散歩道"/><ref name="ふるさと世田谷野毛上野毛116118"/><ref name="史料に見る江戸時代の世田谷"/>。

雖然曾經歷一段沒有住持的時間，但根據統計，至2012年包括創始人佑榮，歷代住持約有20人<ref name="ふるさと世田谷野毛上野毛116118"/>。在新編武藏風土記稿卷之四十九記載著「改為真言宗新義宗、等等力村[[滿願寺_(世田谷区)|滿願寺]]的末寺」<ref name="史料に見る江戸時代の世田谷"/>。根據1872年（明治5年）的《古義真言新義真言本末一派寺院明細帳II》的資料，善養寺住持也兼任谷澤川沿岸的'''淨音寺'''（日語：じょういんじ）此無住持寺院的の住持<ref name="世田谷区寺院台帳161">『世田谷区寺院台帳』、161頁。</ref>。淨音寺的山號是「慈眼山」（日語：じがんさん），當時和善養寺同屬於新義真言宗派，也是滿願寺的末寺<ref name="史料に見る江戸時代の世田谷"/><ref name="世田谷区寺院台帳161"/>。之後淨音寺因火災全燒毀而廢寺，合併至滿願寺<ref name="ふるさと世田谷野毛上野毛54-56"/>。從遺址附近坡道「淨音寺坂」可看到仍保留了淨音寺的名稱。<ref name="ふるさと世田谷野毛上野毛54-56"/>。

祭祀的本尊是金剛界大日如來，是木彫塗漆，高約4尺餘的坐像<ref name="ふるさと世田谷野毛上野毛54-56"/><ref name="史料に見る江戸時代の世田谷"/>。毎月1日是本尊緣日，而進行[[護摩|大護摩供養]]，正月元旦也進行元旦護摩。其他則有[[節分|節分]]會、[[施孤|施餓鬼]]、春秋兩季的[[彼岸_(節日)|彼岸]]等等的修業<ref name="ふるさと世田谷野毛上野毛116118"/>。在世田谷區野毛和中町附近，從[[川崎市|川崎市]][[高津區|高津區]]下野毛開始，都内和近縣都擁有{{tsl|ja|檀家制度|檀家}}<ref name="ふるさと世田谷野毛上野毛54-56"/><ref name="ふるさと世田谷野毛上野毛116118"/>。

善養寺也是玉川八十八箇所32番和玉川二十一箇所14番的[[朝聖|禮所]] <ref name="ふるさと世田谷野毛上野毛54-56"/><ref name="ふるさと世田谷野毛上野毛116118"/>。巡禮歌是「肅穆的 如來大悲 多摩川邊 在榧御寺（日語：おごそかに 如来の大悲 かがやけり たまの川辺の 榧のみてらに）」<ref name="ふるさと世田谷野毛上野毛116118"/>。另外，從境内也出土了[[鎌倉時代|鎌倉時代]]至[[室町時代|室町時代]]製作的板碑共18個<ref name="ふるさと世田谷野毛上野毛54-56"/>。

== 境内和文化資產 ==

[[File:Japanese_nutmeg_in_Zenyōmitsu-ji_201504_02.jpg|thumb]]
穿過{{tsl|ja|丸子川 (東京都)|丸子川}}上的紅橋（大日橋)石階，迎面而來有一對[[獬豸|獬豸]]坐像<ref name="せたがやの散歩道"/>。[[獬豸|獬豸]]是架空的神獸，也是防範火災的神，罕見的只有5個存在世上<ref name="せたがやの散歩道"/>。本堂是模仿奈良[[唐招提寺|唐招提寺]]金堂，擁有一對[[鴟尾|鴟尾]]、瓦葺的[[廡殿頂|廡殿頂]]屋頂為特徵<ref name="せたがやの散歩道"/>。

本堂前方則有被稱為'''{{tsl|ja|善養寺のカヤ|善養寺榧木}}'''的大榧木。此榧木有著澤蟹親子報答豪族女兒幫助的報恩傳說，一旁設置了澤蟹和青蛙，以及河童的石像<ref name="珍寺訪問">{{cite web |title=珍寺訪問・世田谷区野毛「善養密寺」 - 東京DEEP案内 |url=https://tokyodeep.info/post_396/ |website=tokyodeep.info |accessdate=2018-08-29 |language=ja}}</ref>。境內還包括了[[象頭神|象頭神]]、石羊、[[布袋和尚|布袋像]]等等雕像<ref name="珍寺訪問"/>。

在本堂内，除了本尊大日如來坐像外，也祭祀著秘佛[[不動明王|不動明王]]立像、[[制多迦|制多迦]]、[[矜羯羅|矜羯羅]]<ref name="ふるさと世田谷野毛上野毛116118"/>。古代境內也存在著鎮守此地的六所神社之神輿堂和閻魔堂<ref name="ふるさと世田谷野毛上野毛116118"/>。1998年（平成10年），為紀念善養寺中興400年記念而落成梵鐘和梵鐘堂。<ref>{{cite web |title=善養寺 |url=http://www004.upp.so-net.ne.jp/reizi/zenyouzi.htm |website=www004.upp.so-net.ne.jp |accessdate=2018-08-29}}</ref>
