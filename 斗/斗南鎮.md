'''斗南鎮'''（{{lang-nan-tw|Táu-lâm-tìn}}），舊稱「他里霧」，位於[[臺灣|臺灣]][[雲林縣|雲林縣]]南端，行政屬於[[中華民國|中華民國]][[臺灣省|臺灣省]]，是雲林的交通中心，氣候溫和且境內交通四通八達，為雲林地區重要農業鄉鎮之一。

== 歷史 ==
斗南鎮是由[[台灣原住民|台灣原住民]][[洪雅族|洪雅族]]（平埔族之一支）的他里霧社（Dalivoe）社名名音譯而來，而猴悶社（Cunaumui）是現在本鎮將軍里溫厝角，位於[[嘉南平原|嘉南平原]]。

斗南於[[明鄭|明鄭]]時期便已經有漢人前來開墾，乾隆年間逐年市街。是雲林地方最早開發地方。[[臺灣清治時期|清治時期]]時先屬[[諸羅縣|諸羅縣]]轄下的他里霧社，清光緒十二年因[[雲林縣|雲林縣]]建縣改隸[[雲林縣|雲林縣]]，為十六保之一，轄本鎮及古坑鄉湳仔、麻園、虎尾鎮惠來厝過溪仔，大埤鄉埔羌崙等計五十八-{庄}-。 

[[台灣日治時代|台灣日治時代]]初期，初置他里霧支廳，在日本統治早期的[[兒玉源太郎|兒玉源太郎]]總督，[[後藤新平|後藤新平]]民政長官時代，曾發生過稱為他里霧事件的殺害降俘事件（又名[[歸順會場事變|歸順會場事變]]）。

1897年（明治三十年）因應街-{庄}-長設置辦法之訂定改隸斗六廳。

1907年（明治四十年）改隸嘉義廳。

1910年（明治四十三年），設他里霧區，置他里霧區長役場。

1920年10月1日（大正九年），地方制度改革，改名為「斗南」，隸屬於[[臺南州|台南州]][[斗六郡|斗六郡]]。斗南名稱的由來，因位於[[斗六|斗六]]以南，故稱斗南，剛好日本也有[[斗南藩|斗南藩]]（{{lang|ja|となみはん}}，現[[青森縣|青森縣]][[陸奧市|陸奧市]]斗南丘），所以被日本人改為斗南。

[[日本投降|日本投降]]，[[國民政府|國民政府]]接收臺灣以後，1946年（民國35年）1月11日成立斗南鎮公所。

1950年（民國39年）10月25日改屬於[[雲林縣|雲林縣]]。

== 地理 ==
*地理位置
斗南鎮位於[[雲林縣|雲林縣]]東南方，斗六台地和雲嘉沖積平原交接處，東面與[[斗六市|斗六市]]江厝里和[[古坑鄉|古坑鄉]]湳子村和麻園村為鄰，西邊與大埤鄉(三結村，豐田村)以大湖口溪為界，南鄰[[嘉義縣|嘉義縣]][[大林鎮_(台灣)|大林鎮]]，以石龜溪為界，北側為[[虎尾鎮|虎尾鎮]]惠來厝和平和厝。斗南鎮面積48.12平方公里。

*地形地質
屬於平原地形，共有新虎尾溪、雲林溪、石牛溪、崙仔溪、大湖口溪四條溪，由東向西貫穿斗南鎮。斗南鎮的地質屬於現代沖積層，土壤為河川近代沖積土，占地最廣的是砂頁岩新沖積土，占全鎮面積58%，黏板岩和砂頁岩混合沖積土次之，占全鎮面積13%，黏板岩沖積土共209公頃，分布於新虎尾溪一帶，紅壤最少，約73.4公頃。母岩為[[第三紀|第三紀]]之沙岩。

*氣候
氣候屬於亞熱帶氣候區，年平均溫度高於攝氏22℃，平均降雨量在1400公厘，主要集中在5至9月，11月下旬至翌年3月下旬為乾季，雨水來源以熱帶雨和颱風雨為主。

== 經濟 ==

=== 產業 ===
本鎮經濟結構以農業型經濟為主，農牧業非常發達，一級產業就業人口相當多，
另一方面，二、三級產業相當有限，工商業發展也極為有限，所以成長是相當程度受到限制的，
鎮內大多工商就業人口，主要以[[斗六市|斗六市]]與[[虎尾鎮|虎尾鎮]]為主。

近來由於斗六市與虎尾鎮的工商業的成長，為雲林地區經濟成長較為迅速的城鎮，
算是縣內生活機能比較好的地區，其提供周遭鄉鎮基本的就業、教育、醫療、消費等，
例如斗六之雲林科技工業區及斗六工業區、虎尾之中科虎尾園區，
吸引本鎮不少人前往該地就業，境內居住人口有減少趨勢。

== 交通 ==

其中延平路(台一線)、中山路、中正路、大成路、文元路、南昌西路等，
為鎮內重要市區道路，其主要市區發展以台一線及中山路為發展主軸。
