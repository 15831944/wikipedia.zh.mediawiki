'''斗南車站'''位於[[台灣|台灣]][[雲林縣|雲林縣]][[斗南鎮|斗南鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]及[[臺灣糖業鐵路|臺灣糖業鐵路]][[斗南線|斗南線]]（廢止）的[[鐵路車站|鐵路車站]]，是日本人戰敗前為徵用[[戰略物資|戰略物資]]在台灣最後建造的大型車站。本站為配合[[臺鐵捷運化|臺鐵捷運化]]政策，將改建為[[跨站式站房_(台鐵)|跨站式站房]]<ref>{{cite news|author=|title=禁建16年鬆綁 斗南車站跨站工程有譜|url=http://news.ltn.com.tw/news/local/paper/810575|accessdate=2014-09-05|newspaper=自由時報|date=2014-09-05}}</ref>。於106年12月12日舉行動土儀式，預計109年12月31日完工。

[[雲林縣政府|雲林縣政府]]計畫興建臺鐵[[虎尾線|虎尾線]]，由斗南至虎尾，串連[[高鐵雲林站|高鐵雲林站]]。<ref>[https://udn.com/news/story/7326/2330597 雲林推動「雙鐵一纜」 五分車復出]</ref>

== 利用狀況 ==
[[File:Running_in_board_at_TRA_Dounan_Station.jpg|thumb]]
*目前為[[臺灣鐵路管理局車站等級#二等站|二等站]]，停靠車種以[[臺鐵區間車|區間車]]、[[臺鐵區間快車|區間快車]]、[[莒光號列車|莒光號]]為主（觀光列車除外）。
*[[推拉式自強號|自強號]]：停靠南下（逆行）：101次、105次、115次、121次、149次、175次；北上（順行）：108次、116次、122次、134次、142次、150次、170次）。
*停靠起站的[[台鐵EMU300型電聯車|自強號]]6126次(逢週日行駛，由[[EMU300|EMU300]]擔當)。
*177次/6177次自強號斗南為終點站。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1903年12月15日：設「他里霧驛」。
*1920年10月1日：改名「斗南」。
*1945年：站房改建。本站為[[臺灣日治時期|臺灣日治時期]]改建的最後一個火車站。<ref>{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>
*2013年8月29日：康芮颱風影響台灣，雨勢使雲林縣斗南鎮淹水，淹到只剩月台，鐵軌整個淹沒。
*2013年9月30日：啟用多卡通刷卡機。
*2017年12月12日：本站新跨站式站房動工。
