'''斗姥元君'''（又写作'''斗母'''或'''斗姆'''）是[[中国|中国]][[道教|道教]]的[[女神|女神]]，也是[[北斗七星|北斗]]众星之母。名字的“斗”指的是北斗众星。而“姥”字（古字也寫成姆），意思就是母親。斗姥元君在道教信仰中出现得很晚，但地位却很高；原本是佛教的[[摩利支天|摩利支天]]（[[觀音菩薩|觀音菩薩]]或[[准提菩薩|准提菩薩]]化現）信仰，摩利支天在[[唐代|唐代]]以前由[[佛教|佛教]]传入中国后，被道教所敬奉，尊為“斗姥”，即「北斗众星之母」。对斗姥元君的信仰当然也源自于古人的[[星宿|星宿]]崇拜，但由于斗姥是北斗众星的长辈，所以比群星更为尊贵。

她的尊号则有“斗姥天尊”、“先天道姥天尊”、“北斗九真圣德天后”、“中天梵气斗姥元君”以及全称的“先天斗姥紫光金尊摩利支天大圣圆明道姥天尊”等。在《太上玄灵斗姆大圣元君本命延生心经》<ref>[http://ctext.org/wiki.pl?if=gb&chapter=583390 《太上玄靈斗姆大聖元君本命延生心經》]</ref>中则给斗姥元君加上了“九灵太妙白玉龟台夜光金精祖母元君”这样一个称号，將其地位上抬，索性与[[西王母|西王母]]混为一谈。

== 来历与文献记载 ==
據《玉清無上靈寶自然北斗本生真經》<ref>[http://ctext.org/wiki.pl?if=gb&chapter=616983 《玉清無上靈寶自然北斗本生真經》]</ref>中记载，斗姥元君原本是龙汉年间的周御王的玉妃，名号为“'''紫光夫人'''”。于尘劫中发至愿，愿生圣子，辅佐乾坤。由于一个春季在花园里游玩时而突发感悟，使得金莲花温玉池里的九朵[[莲花|莲花]]化生为九位圣子，大儿子与二儿子就是[[四御|四御]]中的[[天皇大帝|天皇大帝]]（[[勾陈大帝|勾陈大帝]]）以及[[紫微大帝|紫微大帝]]，其余的七个儿子则是[[北斗七星|北斗七星]]，总共号称为“[[九皇大帝|九皇大帝]]”。就因为这样，北斗众星的母亲紫光夫人也就成为了地位可以与王母娘娘同等的“斗姥”。

《斗姥宝诰》是斗姥的重要经典文献。其言：“志心皈命礼。西天竺国。大智光中。真空妙相法王师。无上玄元天母主。金光烁处。日月潜辉。宝杵旋时。鬼神失色。显灵踪于尘世。卫圣驾于阎浮。众生有难若称名。大士寻声来救苦。大悲大愿。大圣大慈。圣德巨光天后。摩利支天大圣。圆明道姥天尊。”

《太上玄灵斗姆大圣元君本命延生心经》中记载：“玉池自化现金身，生九苞，放光明，毫光闪闪飞上无庭。左太阳，右太阴，东斗启明星，而斗号长庚，南有萁星注福寿，北斗七元注长生。紫微主，玉皇尊，二十八宿镇乾坤，十二宫辰安天下，四圣天君把天门，三元三品三官帝，四圣四府四天丁。周无诸斗府，河汉众星真，生天生地生万物，保家保国保皇民，百万雷兵常拥护，五千甲将尽随行。

== 形象 ==
[[成都|成都]][[青羊宫|青羊宫]]内安设有斗姥殿，配神为西王母（右侧）及[[后土|后土]]（左侧）。其神像放置于殿内正中，形象为四首、三目、八臂，乘七豕所拉之车，手里分别拿着[[太阳|太阳]]、[[月亮|月亮]]、[[铃|宝铃]]、[[印|金印]]、[[弓|弯弓]]、[[矛|矛]]、[[戟|戟]]等兵器或法器。

[[明清|明清]]时[[广东|广东]][[肇庆|肇庆]]的[[七星岩_(肇庆)|七星岩]]中也有斗姥像，则是佛教菩萨的装扮，又叫“摩利支菩萨”，华冠璎珞，赤足，两手合掌，两手持日月，两手握剑；左右有两位天女捧盘，盘中一为羊头，一为兔头。

== 民间信仰 ==
在道教中，斗姥元君是个法力无边的女神，能护持信徒吉利平安。据说斗姥元君是“先天之气”所成的大神。所以對斗姥元君的崇拜十分普遍，许多道教宫观都建有“斗姥殿”、“斗姥阁”，专门供奉斗姥。她的诞辰为旧历[[九月初九|九月初九]]的[[重阳节|重阳节]]。斗姥主宰着天上星辰，[[台湾道教|台湾道教]]、[[台湾民间信仰|台湾民间信仰]]中，祭拜[[太岁星君|太岁星君]]等诸星辰者，多半先礼拜斗姥。

== 脅侍 ==
有寺廟以[[殷郊|殷郊]]、[[殷洪|殷洪]]兩元帥，或甲子太歲[[金辨|金辨]]、癸亥太歲[[虞程|虞程]]為斗姥元君左右[[脅侍|脅侍]]，以示斗姥為太歲之母。

亦有寺廟以[[天皇大帝|天皇大帝]]、[[紫微大帝|紫微大帝]]兩大帝；亦或[[太陽星君|太陽星君]]、[[太陰星君|太陰星君]]兩星君；亦或妙梵天尊、玉梵天尊；又或左輔洞明星君（[[密跡金剛|密跡金剛]]）、右弼隱光星君（[[那羅延天|那羅延天]]）脅侍，以示斗姥為眾星之母。

或以[[難陀龍王|難陀龍王]]，[[跋難陀龍王|跋難陀龍王]]為脅侍，以示斗姥為[[準提佛母|準提佛母]]化現。

== 文學創作 ==
《[[封神演义|封神演义]]》中的斗姥称之为“金灵圣母”，也就是[[闻太师|闻太师]]的师父。她道行高深，一人就能对抗[[慈航道人|慈航]]（观音）、[[文殊|文殊]]、[[普贤|普贤]]三位大士，最后打了个平手，但遭[[燃灯道人|燃灯道人]]使用定海神珠偷袭而死。后来[[姜子牙|姜子牙]]在封神台封神时，将金灵圣母册封为“坎宫斗母正神”，让她掌管五斗群星、吉曜恶煞，也就是[[北斗|北斗]]、[[南斗|南斗]]、[[东斗|东斗]]、[[西斗|西斗]]以及[[中斗|中斗]]的母親。
