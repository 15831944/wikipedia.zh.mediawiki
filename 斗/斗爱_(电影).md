《'''斗爱'''》（{{lang-en|'''Duel For Love'''}}）2010年上映于中国大陆的剧情片，是中国大陆首部“[[拉丁舞|拉丁舞]]”主题电影。<ref>{{cite web |url=http://ent.qq.com/a/20090612/000354.htm |title=热舞电影《斗爱》火爆影城 诱惑拉丁以舞励志 |accessdate=2009 |publisher=腾讯网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/c/2009-10-09/12022723433.shtml |title=解秘电影《斗爱》 导演：中国无纯正歌舞片 |accessdate=2009 |publisher=新浪网 |work= |language=zh-hans}}</ref>

==剧情==
“小曼”看到“楚思南”和“黎莉莉”跳拉丁舞之后，为他们精湛的技艺而感动。楚思南和黎莉莉原本是中国大陆拉丁舞绝配，两人打败所有对手之后，互相比赛，从此分道扬镳，两人赌咒发誓，热泪盈眶，今生今世不再愿意见到对方。楚思南从此和“小曼”合作，去舞蹈世界寻找自己的梦想。<ref>{{cite web |url=http://data.ent.163.com/movie/152148.html |title= 斗爱 Duel For Love (2009)  |accessdate=2009 |publisher=网易 |work= |language=zh-hans}}</ref>

==花絮==
钟汉良是[[香港|香港]][[TVB|TVB]]舞蹈艺员身份出道，他娴熟于“[[探戈|探戈]]”、“[[桑巴|桑巴]]”、“[[恰恰|恰恰]]”，在电影里表现出拉丁舞火辣、狂野、诱人的魅力。

爱戴作为女歌手也有舞蹈功底，其专辑常展现其性感热辣的身材，蛮腰翘臀在该电影中穿着火红舞裙表现。

孙田莉子是中国大陆90后拉丁舞实力派舞者，曾荣获“全国体育舞蹈冠军”，她娴熟于“[[斗牛舞|斗牛舞]]”、“桑巴”、“恰恰”、“[[现代舞|现代舞]]”等舞种。
