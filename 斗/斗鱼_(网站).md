'''斗鱼'''是一家隶属于武汉斗鱼网络科技有限公司的以游戏直播为主的[[视频弹幕网站|弹幕式视频直播分享网站]]。其前身为ACFUN生放送直播，2014年正式更名为斗鱼。该网站主要以游戏视频直播为主，也包括体育、综艺、娱乐等内容。

== 运营方式 ==
斗鱼网的站内直播视频主要来自网友和知名主播，有时也会有游戏国际赛事甚至体育赛事、综艺节目的直播内容。注册用户通过实名认证申请成为主播。

站内的流通货币为“鱼丸”和“鱼翅”，鱼翅通过充值获得，充值比例为鱼翅：人民币=1：1，鱼丸通过充值鱼翅或者完成相应任务获得。

主播通过粉丝赠送赚取鱼丸超过50万后，可以与斗鱼TV签署协议将其按兑现，从而盈利。兑现比例为鱼丸:人民币=1000:1，之后还要扣除20%的个人所得税。

== 融资历史 ==
2016年3月15日，斗鱼网完成1亿美元的B轮融资，此次的融资由腾讯出资4亿人民币领投，天神娱乐以及斗鱼前一轮的投资方南山资本进行了跟投。<ref>[http://tech.sina.com.cn/i/2016-03-15/doc-ifxqhmve9204620.shtml 斗鱼获1亿美元新一轮融资 腾讯领投]</ref>

===直播模仿吸毒===
2016年10月25日凌晨，有网友举报斗鱼直播平台上的一名黄姓男主播毒瘾发作，面向30万观众直播吸毒行为。据网友上传的视频截图显示，主播坐在车内的驾驶座位上，左手拿一张纸片，右手拿一根管状物，用管状物靠近鼻子，做出疑似吸食毒品的动作。事件曝光后引起社会热议，有网友呼吁报警，希望警方介入调查，并表示“直播吸毒会对很多人产生不良影响，会引起社会上他人的模仿”。次日，该主播在微博上就自己的行为向公众道歉，称当天直播时只是为了增加节目效果，表演了一个影响很不好的动作。这位主播是一名健身模特，当时直播只是为了哗众取宠、取悦观众。该主播的微博被认证为2010上海国际模特大赛T台男模组季军。该主播在接受《[[京华时报|京华时报]]》记者采访时表示，当时只是模仿动作，他在视频中使用的道具是两张发票，纸上没有任何东西，只是为了节目效果。事后他在民警的陪同下去医院做了尿检，检测结果全部为阴性。虽然主播没有吸毒，但在直播中连续做出模仿吸毒的动作，带来不良社会影响，主播因扰乱公共秩序被警方处以行政拘留5天，且在斗鱼平台的直播账号已被永久封禁<ref>{{cite news|title=斗鱼主播为增加节目效果直播吸毒 回应:只是模仿 |url=http://money.163.com/16/1101/01/C4OI557O002580S6.html |publisher=京华时报(北京) |date=2016-11-01 |accessdate=2016-11-01 }}</ref><ref>{{cite news|title=直播模仿“吸毒” 斗鱼主播被拘5天 |url=http://money.163.com/16/1107/07/C58KHGGN002580S6.html#from=keyscan |publisher=《创业邦》杂志(北京)|date=2016-11-07 |accessdate=2016-11-07}}</ref>。

===客户端遭下架===
2018年10月[[中华人民共和国国庆节|国庆假期]]期间，有网友透露斗鱼客户端已经在各大应用市场下架。据《[[南方都市报|南方都市报]]》报道，斗鱼客户端已经在苹果[[App_Store|App Store]]中无法搜索到，而在Android平台的应用市场则只能搜索到“极速版”。10月4日，斗鱼在[[虎扑|虎扑]]论坛回应网友就斗鱼客户端下架的提问时称“由于苹果商店问题目前暂时下线了斗鱼APP。我们正在积极协调中，请您耐心等待后续重新上线。安卓APP可以在官网扫码进行下载，给您造成的不便请您谅解。”10月12日，斗鱼发出声明称下架原因与[[陈一发儿|陈一发儿]]直播间于8月被封禁后仍然有大量礼物打赏有关，另有分析指出，斗鱼可能因未取得《信息网络传播视听节目许可证》而遭到下架<ref>{{cite news|title=斗鱼APP国庆悄然下架 |url=http://tech.caijing.com.cn/20181009/4522397.shtml|publisher=南方都市报|author=蔡辉|date=2018-10-09 |accessdate=2018-10-14}}</ref><ref>{{cite news|title=下架8天后斗鱼发声明：永久关闭陈一发直播间|url=https://news.sina.com.cn/s/2018-10-13/doc-ihmhafir3110727.shtml|publisher=环球时报 |author=|date=2018-10-13 |accessdate=2018-10-14}}</ref>。
