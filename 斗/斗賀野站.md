'''斗賀野站'''（{{jpn|j=斗賀野駅|hg=とがのえき|rm=Togano eki|f=y}}）是一位於[[日本|日本]][[高知縣|高知縣]][[高岡郡|高岡郡]][[佐川町|佐川町]]東組、隸屬於[[四國旅客鐵道|四國旅客鐵道]]（JR四國）的[[鐵路車站|鐵路車站]]。車站編號為'''K15'''，本站只停靠普通列車。附近有散落的民居，但仍以農地為主。

車站曾是主要貨運站，尤其是附近採礦場的[[石灰|石灰]]，有專用的路軌負責運送。礦場關閉後，貨物路軌亦棄用。

==車站結構==
站舍為單層木製建築，設有票務處及候車室。現時本站為業務委託站，由非JR職員負責售票事宜。

[[File:Togano_station_01.jpg|thumb]]
設有側式月台及島式月台各1個，提供2線2面的配置。近站舍為1號月台。島式月台現時只使用其中一邊作2號月台，另一邊為原貨物線道，部份路軌已被拆走，只留下少部份作為避線或工程車停泊之用。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/dosan/st-togano_fuka.jpg |title=存档副本 |accessdate=2012-08-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304082504/http://www.shikoku.org.uk/JR/LINE/dosan/st-togano_fuka.jpg |archivedate=2016-03-04 }}</ref>島式月台及站舍以平交道連接月台末端向[[襟野野站|襟野野站]]方向。另外,在2號月台中央設有有蓋候車室。

列車入站限速為50km/h，這是因為本站由單軌路段轉至雙軌路段採用Y型的[[轉轍器|轉轍器]]，亦即是說，雙軌路段並無主副線／側線之分，因此列車入站時，須減速通過車站。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/dosan/ST-TOGANO.htm |title=存档副本 |accessdate=2012-08-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304073513/http://www.shikoku.org.uk/JR/LINE/dosan/ST-TOGANO.htm |archivedate=2016-03-04 }} 非官方車站介紹，第2及3幅圖片可見兩邊的Y型轉轍器。</ref>

== 歷史 ==
[[File:JNR_df50_18_43_togano_senyou.jpg|thumb]]
* 1924年3月30日：開業，兼辦客運和貸運服務。<ref name="hensen2">《停車場變遷大事典－國鐵・JR編》2，JTB，1998年，p.669。ISBN 4-533-02980-9（日語書籍）</ref>。
* 1942年：土佐石灰工業大平山礦山専用線開始使用。
* 1960年6月28日：專用貨運列車改在専用線上辦理。<ref name="hensen2"/>
* 1969年10月1日：旅客手提行李及小型行李再托運停止。<ref name="hensen2"/>
* 1970年6月1日：小型貨物托運停止。只辦理旅客手提行李及小型行李（只限當站處理）及專用貨運列車。<ref name="hensen2"/>
* 1984年2月1日：行李托運停止。<ref name="hensen2"/>
* 1987年4月1日：日本國鐵分割民營化，車站的營運由JR四國繼承。
* 1992年10月1日：貨運服務停止。土佐石灰工業専用線亦停止運作。
* 2010年（[[平成|平成]]22年）10月1日：無人化。<ref>{{Cite web|url=http://www.jr-shikoku.co.jp/03_news/press/10-06-21/02.htm |title=JR四国　車站業務體制變更 |date=2010-06-21 |accessdate=2010-06-22 |publisher=[[四國旅客鐵道|四國旅客鐵道]] |deadurl=yes |archiveurl=https://web.archive.org/web/20100625120429/http://www.jr-shikoku.co.jp/03_news/press/10-06-21/02.htm |archivedate=2010-06-25 }}</ref>
