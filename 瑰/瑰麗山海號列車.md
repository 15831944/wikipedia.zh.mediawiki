'''瑰麗山海'''（{{lang-ja|ベル・モンターニュ・エ・メール}}）是由日本的[[西日本旅客鐵道|西日本旅客鐵道]]公司所營運，在特定時段裡行經[[城端線|城端線]]或[[冰見線|冰見線]]的觀光用途快速列車。<ref name="Watch">{{cite web |url=https://travel.watch.impress.co.jp/docs/news/720235.html |script-title=ja:JR西日本、キハ40をベースにした観光列車「べるもんた」を報道公開 |work=Impress Corporation |author= |language=ja |date=2015年9月8日 |accessdate=2018年6月17日}}</ref>

此列車的原名「{{lang|ja|ベル・モンターニュ・エ・メール}}」，是採用[[法文|法文]]語句「{{lang|fr|Belles montagnes et mer}}」所對應的日語讀音，該法文語句的原始含意則是「美麗的山與海」，<ref>{{cite web |url=https://www.nikkei.com/article/DGXMZO31539140Y8A600C1962M00/ |script-title=ja:富山の観光列車「ベるもんた」、すしや地酒も提供 |work=[[日本經濟新聞|日本經濟新聞]] |language=ja |date=2018年6月8日 |accessdate=2018年6月17日}}</ref>而在日本當地多將此列車以「{{lang|ja|べるもんた}}」來簡稱之。<ref name="Watch"/>

== 歷史 ==
2015年在[[北陸新幹線|北陸新幹線]]開通後，為了讓利用新幹線來到[[北陸地方|北陸]]一帶的其他日本縣市旅客能更方便體驗當地觀光，西日本旅客鐵道開始著手在[[城端線|城端線]]與[[冰見線|冰見線]]安排觀光列車。<ref>{{cite web |url=http://www.sankei.com/west/news/151010/wst1510100039-n1.html |archiveurl=https://web.archive.org/web/20151014233550/http://www.sankei.com/west/news/151010/wst1510100039-n1.html |deadurl=yes |script-title=ja:景色魅力の観光列車　べるもんた運行開始、富山 |work=[[產經新聞|產經新聞]] |language=ja |date=2015年10月10日 |archivedate=2015年10月14日 |accessdate=2018年6月17日}}</ref>

基於城端線一帶是面臨靠山的景色，冰見線是行經面向海洋的景色，便決定取用法文裡含意代表著「美麗的山與海」的句子「{{lang|fr|Belles montagnes et mer}}」，來作為此觀光列車的名稱。此觀光列車則在2015年10月10日正式啟用。<ref name="MyNavi">{{cite web |url=https://news.mynavi.jp/article/20150731-a365/ |script-title=ja:JR西日本「べるもんた」城端線・氷見線観光列車の愛称決定 - 10/10運転開始  |work=My Navi |language=ja |date=2015年7月31日 |accessdate=2018年6月17日}}</ref>

== 車輛 ==
[[File:Toyamasushi_horoyoi.jpg|thumbnail]]
使用的車輛是改造自[[國鐵Ki40系|國鐵Ki40系]]款號，<ref>{{cite book |script-title=ja:《JR現役鉄道図鑑》 |publisher=My Navi |location=日本 |isbn=978-483-995-698-1 |language=ja |chapter=〈{{lang|ja|愛称は「ベるもんた」キハ40形 ベル・モンターニュ・エ・メール}}〉 |page=148頁 |date=2016年8月1日}}</ref>並僅使用一節車廂來乘載旅客。

車內裝飾是採用[[南砺市|南砺市]]所生產的木雕手工藝品為主，藝品是以列車所在地[[富山縣|富山縣]]一帶的人文風情作為主題，包含有[[五箇山|五箇山]]的民謠表演[[筑子舞|筑子舞]]，以及代表南砺市的花草，如[[香椿|香椿]]、[[日本百合|日本百合]]等等。<ref>{{cite web |url=https://www.westjr.co.jp/press/article/2015/09/page_7603.html |script-title=ja:城端線・氷見線観光列車「べるもんた」の車両が完成しました |work=[[西日本旅客鐵道|西日本旅客鐵道]] |language=ja |date=2015年9月8日 |accessdate=2018年6月17日}}</ref>

== 營運狀況 ==
瑰麗山海是在各週的週末兩日行駛，且兩日各有不同的行駛路線。以城端線為主的路線，是從城端線起點[[高岡車站|高岡車站]]開向終點[[城端車站|城端車站]]，讓遊客能在城端車站搭乘開往[[合掌造|合掌造]]形式住屋的[[五箇山|五箇山]]、[[白川鄉|白川鄉]]等山區觀光地的巴士。<ref name="Asahi">{{cite web |url=https://dot.asahi.com/dot/2015101900106.html?page=1 |script-title=ja:大人気！　北陸の観光列車「べるもんた」に乗ってみた！ |work=[[朝日新聞|朝日新聞]] |author=若林朋子 |language=ja |date=2015年10月25日 |accessdate=2018年6月17日}}</ref><ref name="MyNavi"/>另一以冰見線為主的路線，則是先從城端線的[[新高岡站|新高岡車站]]開至高岡車站後，改走向從高岡車站前往[[冰見車站|冰見車站]]的冰見線，讓遊客欣賞面向[[雨情海岸|雨情海岸]]另一端的[[立山連峰|立山連峰]]景色。<ref name="Asahi"/>
