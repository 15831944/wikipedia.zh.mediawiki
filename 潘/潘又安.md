'''潘又安'''，[[紅樓夢|《紅樓夢》]]中人物，[[司棋|司棋]]的姑舅兄弟，賈府小廝，初次登場於《紅樓夢》第七十一回<ref>{{cite web |title=第七十一回　嫌隙人有心生嫌隙　鴛鴦女無意遇鴛鴦 |url=http://freereddream.blogspot.com/search/label/%E7%AC%AC%2071%E5%9B%9E%20%E5%AB%8C%E9%9A%99%E4%BA%BA%E6%9C%89%E5%BF%83%E7%94%9F%E5%AB%8C%E9%9A%99%20%E9%B4%9B%E9%B4%A6%E5%A5%B3%E7%84%A1%E6%84%8F%E9%81%87%E9%B4%9B%E9%B4%A6}}</ref>，書中描述其當時正與司棋在大觀園內[[私通|私通]]，為[[鴛鴦_(紅樓夢)|鴛鴦]]所撞見，當場向鴛鴦磕頭請求替他掩埋和司棋的私情。
卻在第七十二回中因害怕被揭發私情而逃跑，讓司棋因而病倒，認為自己誤信了這個男人。<ref>{{cite web |title=第七十二回　王熙鳳恃強羞說病　來旺婦倚勢霸成親 |url=http://freereddream.blogspot.com/search/label/%E7%AC%AC%2072%E5%9B%9E%20%E7%8E%8B%E7%86%99%E9%B3%B3%E6%81%83%E5%BC%B7%E7%BE%9E%E8%AA%AA%E7%97%85%20%E4%BE%86%E6%97%BA%E5%A9%A6%E5%80%9A%E5%8B%A2%E9%9C%B8%E6%88%90%E8%A6%AA}}</ref>書中描述其品貌風流，與司棋自小一起玩耍長大而產生感情。
而直至第七十四回才知其姓名為潘又安，確切身份為司棋的表弟。<ref>{{cite web |title=第七十四回　惑奸讒抄檢大觀園　矢孤介杜絕寧國府 |url=http://freereddream.blogspot.com/search/label/%E7%AC%AC%2074%E5%9B%9E%20%E6%83%91%E5%A5%B8%E8%AE%92%E6%8A%84%E6%8F%80%E5%A4%A7%E8%A7%80%E5%9C%92%20%E7%9F%A2%E5%AD%A4%E4%BB%8B%E6%9D%9C%E7%B5%95%E5%AE%81%E5%9C%8B%E5%BA%9C}}</ref>
