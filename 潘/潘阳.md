'''潘阳'''{{BD|1983年|2月13日|||catIdx= P潘}}籍贯为[[吉林省|吉林省]][[怀德县|怀德县]]范家屯镇，毕业于[[中国人民解放军艺术学院|中国人民解放军艺术学院]]，著名的影视演员和歌手，其父亲为小品演员[[潘长江|潘长江]]。

== 生平 ==
27岁，潘阳在中国人民解放军艺术学院毕业，随后踏入影视圈和音乐圈。

== 个人生活 ==
2012年7月29日，潘阳和“江南石肆”会所总裁石磊在[[北京|北京]]万达索菲特酒店举行婚礼<ref>{{cite web |url= http://ent.ifeng.com/photo/hot/detail_2012_07/29/16387369_0.shtml#p=1|title= 潘长江女儿嫁“高富帅” 婚礼内场曝光|accessdate=2012-07-29  |publisher=凤凰网 |work= |language =zh-cn}}</ref>，在恋爱三年后正式结婚。<ref>{{cite web |url=http://ent.ifeng.com/zz/detail_2012_07/29/16384679_0.shtml |title= 潘长江女儿出嫁 女婿被曝身家已达1.35亿美元|accessdate=2012年7月29日 |publisher= 凤凰网|work=曾灵 |language =zh-cn}}</ref> 石磊为[[香港人|香港人]]，出生于音乐世家，私企北京武氏集团老板，个人总资产达1.35亿美元。<ref>{{cite web |url=http://ent.163.com/12/0729/12/87J4CPMJ00031H2L.html |title=潘长江女儿潘阳今日大婚 新郎身价过10亿 |accessdate= 2012-07-29|publisher= 网易|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://gb.cri.cn/27564/2012/07/29/1326s3788379_1.htm |title=潘阳老公武石磊资料背景揭秘 |accessdate=2012-07-29  |publisher=国际在线 |work= |language =zh-cn}}</ref>
