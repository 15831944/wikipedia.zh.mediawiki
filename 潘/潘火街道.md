'''潘火街道'''是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[鄞州区|鄞州区]]一个街道，辖区面积14.25平方公里，户籍人口1.9万人，流动人口5.2万人。<ref name="main">[http://news.cnnb.com.cn/system/2012/08/17/007423976.shtml 鄞州新城区城市型街道再添新成员 潘火街道成立]</ref>

== 沿革 ==
潘火街道原为[[下应街道|下应街道]]的一部分。2012年4月10日，划下应街道北部成立潘火街道。<ref name="main"/>

== 地理位置 ==
潘火街道位于鄞州区南部，北与[[福明街道|福明街道]]相接，西隔[[甬台温高速公路|甬台温高速公路]]与[[中河街道|中河街道]]相望，南至[[鄞县大道|鄞县大道]]与[[下应街道|下应街道]]相连，东与[[东钱湖镇|东钱湖镇]]相邻。<ref name="main" />

== 下辖 ==
* 社区：东南、雅苑、东莺、世纪、紫郡、同欣、尚诚、星苑、泗港
* 村：殷家坑、小花园、潘火桥、妙胜、曹隘、王家弄、土桥

== 经济 ==
鄞州投资创业中心位于潘火街道。<ref name="main" />

== 交通 ==
鄞县大道为街道的东西向主干道。<ref name="main" />
