'''潘敦仔'''（[[漢名|漢名]]'''潘敦'''，「仔」為[[俗稱|俗稱]]，{{Bd|1705年||1771年|}}<ref name=":1">{{Cite web|url=http://webcache.googleusercontent.com/search?q=cache:havXfKxQLYkJ:nrch.culture.tw/twpedia.aspx?id=11164+&cd=1&hl=zh-TW&ct=clnk&gl=tw|title=潘敦仔|accessdate=2017-10-30|author=|date=|work=webcache.googleusercontent.com|publisher=}}</ref>），亦作'''潘墩仔'''，[[原名|原名]][[音譯|音譯]]'''敦仔阿打歪'''，是[[岸裡社|岸裡社]]第三任總[[土官|土官]]。

== 生平 ==
[[康熙|康熙]]四十四年（1705年），敦仔阿打歪[[出生|出生]]於[[岸裡社|岸裡社]]；敦仔阿打歪的[[祖父|祖父]]阿穆（亦作「阿莫」）在康熙年間協助[[清軍|清軍]]鎮壓平埔族通宵社亂事，因而於康熙五十四年（1715年）被封為岸裡九社第一任總土官。

康熙六十年（1721年）[[農曆五月|五月]]，敦仔阿打歪在[[朱一貴事件|朱一貴事件]]爆發時與通事[[張達京|張達京]]率領岸裡社族人於[[大肚溪|大肚溪]]防守並攔截起事分子。

[[雍正|雍正]]元年起，敦仔阿打歪與張達京及其所代表的六館業戶進行「割地換水」，以將[[水田|水田]]耕種技術引進岸裡社，並依各社[[人口|人口]]平均分配[[田地|田地]]，進而奠定岸裡社土地[[私有化|私有化]]，復使岸裡社人生活型態由[[狩獵|狩獵]][[游耕|游耕]]變成[[定居|定居]][[農耕|農耕]]；另外，敦仔阿打歪又招聘[[業戶|業戶]]及[[佃戶_(中國)|佃戶]]開墾岸裡社部分土地。

雍正四年（1726年），敦仔阿打歪參與圍剿[[水沙連|水沙連社]]。

雍正九至十年（1731年 - 1732年），敦仔阿打歪協助清軍鎮壓[[大甲西社抗清事件|大甲西社抗清事件]]與[[吳福生事件|吳福生事件]]<ref>{{cite book |author=洪麗完|title=《熟番社會網絡與集體意識：臺灣中部平埔族群歷史變遷（1700-1900）》 |date=2009-02|publisher=聯經出版公司|location=臺灣|language=zh|isbn=9789570831764}}</ref>，並因而奉[[詔令文書|詔]]由[[王郡|王郡]]帶領前往[[紫禁城|紫禁城]]晉見[[雍正帝|雍正帝]]，復獲其賞賜御衣及[[號|號]]「大由仁」。

雍正十二年（1734年），敦仔阿打歪敘封[[正七品|正七品]]官銜，並被賜與[[蟒袍|蟒袍]]等寶物。

[[乾隆|乾隆]]二年（1737年），敦仔阿打歪帶領岸里社人隨北路參將[[靳光瀚|靳光瀚]]、淡水廳同知[[趙奇芳|趙奇芳]]搜山圍捕[[後壠|後壠]]及加志閣社反叛族人。

乾隆五年（1740年），敦仔阿打歪被[[赐姓名|賜以潘姓]]；[[農曆十月|十月]]，巡臺御史[[楊二酉|楊二酉]]賞賜潘敦「忠勇可嘉」[[匾額|匾額]]。

乾隆七年（1742年），南北投社王永興計謀反叛，潘敦率領族眾前往圍剿。

乾隆二十三年（1758年）[[農曆十一月|十一月]]，潘敦被任命為岸裡社[[通事|通事]]，並擔任該職直至乾隆三十六年（1771年）[[農曆四月|四月]]；他是岸裡社首位非[[漢人|漢人]]通事。

乾隆三十一年（1766年），潘敦招撫[[東勢區|東勢]]十三社[[泰雅族|泰雅族]]人歸順[[清廷|清廷]]。

潘敦名下[[田產|田產]]分布於後[[神岡|神岡]]與[[豐原區|豐原]]周遭，當時每年[[租金|租金]]收入達穀數千[[石_(容量單位)|石]]。

乾隆三十六年（1771年，或乾隆四十一年（1776年）<ref name=":0">{{Cite web|url=http://memory.ncl.edu.tw/tm_cgi/hypage.cgi?HYPAGE=toolbox_figure_detail.hpg&project_id=twpeop&dtd_id=15&subject_name=%E8%87%BA%E7%81%A3%E4%BA%BA%E7%89%A9%E8%AA%8C(1895-1945)&subject_url=toolbox_figure.hpg&xml_id=0000300573&who=%E6%BD%98%E6%95%A6%E4%BB%94|title=臺灣記憶 Taiwan Memory--國家圖書館|accessdate=2017-10-30|work=memory.ncl.edu.tw|language=zh-TW}}</ref>），潘敦逝世，隨後被葬在大社楓仔林，後復遷葬大社潘家墓園。<ref name=":1" />

== 家族 ==
潘敦的[[姑丈|姑丈]]是[[張達京|張達京]]；張達京自雍正元年（1723年）起任岸裡五社總通事，但因稍後發生多起通事入侵[[番界|番界]]開墾事件，且有通事聯繫高山族[[出草|出草]]殺害他人，當局於是整肅革去張達京等具極大勢力的漢人通事，並將其遣回其戶籍所在地。<ref name=":0" />

== 宅第 ==
潘敦的家族[[宅第|宅第]]屬彷漢式大型[[三合院|三合院]]形式，其單元包含[[客廳|正廳]]、虎平、龍平、後陵等規制。<ref>{{Cite web|url=http://ndmmc2.iis.sinica.edu.tw/System/FileManage/pes/Search/View.jsp?id=678473|title=岸裡社潘敦仔第宅模型|accessdate=2017-10-30|work=ndmmc2.iis.sinica.edu.tw}}</ref>
