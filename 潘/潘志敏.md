'''潘志敏'''（{{bd|1992年|7月28日||}}），[[浙江|浙江]][[温州|温州]]人，[[咖啡师|咖啡师]]。现于[[北京|北京]]SOE咖啡店任[[店长|店长]]。2017年，获得{{tsl|en|World Barista Championship|世界咖啡师大赛}}中国总赛区冠军。

==咖啡生涯==
潘志敏接触[[咖啡|咖啡]]事业的时间较晚<ref name='pzm'>{{cite web |author1=Original 咖报 |title=咖报丨潘志敏，距离2018WBC世界咖啡师大赛冠军最近的人 |url=https://mp.weixin.qq.com/s/gIAwKoE4CYhfYMVGWeOdjg |website=好康视频 |accessdate=2018-11-14}}</ref>。潘志敏原本在大学本科修习[[金融|金融]]专业，在一次尝试了手冲的单一产区[[意式咖啡|意式咖啡]]后，开始对咖啡产生兴趣。2014年，潘志敏与其女友一同决定放弃较为平稳安逸的家业，尝试进入咖啡行业。最初，潘志敏在[[北京|北京]]进入[[雕刻时光|雕刻时光]]咖啡店工作，很快便在内部考核中升到A级员工，同年便转而加入北京咖啡沙龙<ref>{{cite web |title=2017CBC咖啡大赛冠军团队专访-潘志敏_重庆百瑞斯特咖啡西点学院 |url=http://www.brst.cn/news/kfzx/1991.html |website=百瑞斯特 |accessdate=2018-11-14}}</ref>。

2015年，潘志敏首次代表北京咖啡沙龙参加[[中国咖啡师大赛|中国咖啡师大赛]]（{{lang|en|China Barista Championship}}）的北京分赛，出乎意料地获得分赛区冠军。然而由于首次参赛而准备不足，潘志敏在2016年驾车赶赴下一场位于[[上海|上海]]的{{tsl|en|World Barista Championship|世界咖啡师大赛}}中国赛区的初赛途中舟车劳顿，甚至发生轻微车祸，赛场上精神不济，未能进入复赛<ref>{{cite web |title=勝利沒有意外 – Roasters News |url=https://webcache.googleusercontent.com/search?q=cache:NOI79rlcUsYJ:https://roasters.news/cbc-champion-f7dda09c010%3Fsource%3D---------0+&cd=1&hl=en&ct=clnk&gl=jp |website=Roasters News |accessdate=2018-11-14 |date=2017-04-02}}</ref>。同年年底，潘志敏蝉联了中国咖啡师大赛的北京分赛区冠军，并一举获得了2017年世界咖啡师大赛中国赛区的冠军，被誉为[[黑马|黑马]]<ref>{{cite web |title=CBC冠军兴起新概念处理法 |url=http://www.kafeippjm.com/news/dongtai/1416.html |website=中国咖啡品牌网 |accessdate=2018-11-14}}</ref>。

潘志敏现任北京SOE咖啡店店长<ref>{{cite web |author1=北京人民广播电台 |title=老址在后海的SOE COFFEE 尝尝冠军的咖啡-北京时间 |url=https://item.btime.com/400bmiha2ei9ug8rfrnpe5m8c3p |website=北京时间 |accessdate=2018-11-14}}</ref>，并在CREMA咖啡培训机构担任培训师<ref>{{cite web |title=Crema克丽玛咖啡培训机构-导师团队 |url=http://crema.cn/team.php |website=crema.cn |accessdate=2018-11-14 |language=zh}}</ref>。他亦取得了[[国际咖啡质量研究所|国际咖啡质量研究所]]（{{lang|en|Coffee Quality Institute}}）认证的[[咖啡品质鉴定师|咖啡品质鉴定师]]（{{lang|en|Q-Grader}}）资格<ref name='pzm'/>。
