'''潘岳'''（{{bd|1960年|4月|||catIdx=P潘}}），[[江苏省|江苏省]][[南京市|南京市]]人；历史学博士，副研究员。曾任[[中华人民共和国环境保护部|中华人民共和国环境保护部]]副部长、党组副书记兼直属机关党委书记，现任[[中央社会主义学院|中央社会主义学院]]党组书记、第一副院长（正部长级）。中共第十九届中央委员会候补委员。

== 简历 ==
1976年，时年十六岁的潘岳就参加了[[中国人民解放军|中国人民解放军]]；在[[中国人民解放军第38集团军|陆军第三十八集团军]]、[[中国人民解放军铁道兵|铁道兵]]第十三师服役。1982年退役后，被分配到《经济日报》任资料员；而后还在《中国环境报》社当记者。退役后的十多年间，潘岳的工作变动地很频繁；曾工作过的部门和工作领域也各有不同。1986年，他就曾转任国家空中交通管制局，担任研究室副主任，还兼任机关[[共青团|共青团]]委书记。1988年间，潘岳则又在中共北京市房山区委短暂任职，曾担任外联处处长，兼外经委副主任。同年的12月，潘岳又回到了新闻界，出任《中国技术监督报》社副总编辑；一年后，又转任《[[中国青年报|中国青年报]]》社副总编辑。1993年2月，潘岳进入共青团中央，出任中国青年研究中心主任。1994年5月，被任命为[[国家国有资产管理局|国家国有资产管理局]]副局长。1998年，时年38岁的潘岳出任[[国家技术质量监督局|国家技术质量监督局]]副局长，成为当时中国最年轻的副部级官员之一。2000年，任[[国务院经济体制改革办公室|国务院经济体制改革办公室]]副主任。2003年3月，任[[国家环境保护总局|国家环境保护总局]]副局长，党组成员兼新闻发言人。2008年，[[国家环境保护总局|国家环境保护总局]]升格为[[中华人民共和国环境保护部|中华人民共和国环境保护部]]，潘岳任环保部副部长。2015年8月，出任空缺长达8年的环保部党组副书记<ref>{{cite web|url=http://news.163.com/15/0805/00/B07DE6O20001124J.html|title=潘岳任环保部党组副书记 该职已空缺8年|publisher=网易|accessdate=2015-08-05}}</ref>。2016年3月，转任[[中央社会主义学院|中央社会主义学院]]党组书记、第一副院长（正部长级）<ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1439336|title=潘岳接棒叶小文任中央社院党组书记、第一副院长，跻身正部级|publisher=澎湃新闻|accessdate=2016-03-04}}</ref>。

== 家庭 ==
潘岳的父亲[[潘田|潘田]]，是原[[中国人民解放军|中国人民解放军]]铁道兵司令部副参谋长兼总工程师；曾历任苏南区《苏南日报》社记者、华东军区兵站部交通学校教育科科长、铁道兵团第三师总工程师、铁道兵干部学校副校长兼训练部长、铁道兵学院副院长兼训练部长、铁道兵司令部副总工程师、铁道兵司令部总工程师、[[中国人民解放军铁道兵|铁道兵]]司令部副参谋长兼总工程师等职。他的母亲[[周兰|周兰]]，生前是解放军的一位知名军医。

潘岳的前妻是原[[中共中央政治局常委|中共中央政治局常委]]、[[中国共产党中央军事委员会|中央軍委]]副主席[[劉華清|劉華清]]上將的女兒[[劉超英|劉超英]]，但两人後來離婚。

== 政坛新星 ==
潘岳因近年来行事高调，频频掀起“环评风暴”而广受关注。曾提出“环保社会主义”理论，一般人认为这是[[欧洲绿党|欧洲绿党]]的理念在中国的借鉴。

2010年潘岳因推動環境保護的貢獻獲得菲律賓的[[拉蒙·麦格塞塞奖|拉蒙·麦格塞塞奖]]，但因正逢[[馬尼拉人質事件|馬尼拉人質事件]]，中菲關係緊張，潘岳未出席頒獎典禮<ref>{{cite web |url=http://orientaldaily.on.cc/cnt/news/20100830/00176_024.html |title=兩環保精英拒赴菲領獎 |accessdate= 2010|publisher=東方日報 |work= |language=zh}}</ref>。
