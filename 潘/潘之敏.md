'''潘之敏'''（{{lang-en|'''Vicci Pan'''}}，{{bd|1月19日}}），[[台灣|台灣]][[女演員|女演員]]，台灣國立清華大學經濟系畢，曾接受北藝大電影所「即興表演」演員課程。2011年首挑大樑出演李啟源導演電影《河豚》<ref>{{Cite web |url = http://www.nownews.com/n/2011/06/29/28517
 |title =清大高材生潘之敏為《河豚》　零防護全裸激戰吳慷仁|date = 2011年6月29日 |publisher = NOWnews今日新聞 |language = zh-tw |accessdate = 2017年5月20日 |quote =  }}</ref>女主角，和男主角[[吳慷仁|吳慷仁]]有大膽精彩的對手戲；本片除入圍台北電影節、德國曼漢姆國際影展、波蘭華沙長片競賽片單元，並獲瑞士「日內瓦國際影展」國際影評人費比西獎與發行獎。除了電影作品，近年來亦參與多部電視劇的演出，如《[[植劇場－戀愛沙塵暴|植劇場－戀愛沙塵暴]]》、《[[深夜食堂_(亞洲華語版)|深夜食堂 (亞洲華語版)]]》、《[[一把青_(電視劇)|一把青]]》等。
