== 简介 ==
1957年，潘长江出生于[[黑龙江省|黑龙江省]][[牡丹江市|牡丹江市]][[东宁县|东宁县]]，他的父母都是[[戏曲|戏曲]]演员。<ref name=" duck" /><ref name="fox " >{{cite web |url=http://eladies.sina.com.cn/qg/p/2008/0205/1002656829.shtml |title=潘长江：我是这样“追”到漂亮老婆的 |publisher=新浪网 |accessdate= 2008年2月5日|work= |language =zh-cn}}</ref>
1979年，潘长江考入[[辽宁省|辽宁省]]的[[铁岭县|铁岭县]]评剧团。<ref name="fox " />
1986年，潘长江和[[赵本山|赵本山]]合作演出小品《大观灯》，轰动[[东北三省|东北三省]]，一蹴而红。<ref name=" duck" />

== 个人生活 ==
1979年，潘长江认识了[[东宁县|东宁县]]政府机关[[公务员|公务员]][[杨云|杨云]]，1981年8月31日，两人结婚。<ref name=" duck" /><ref name="fox " />
1985年，杨云生下女儿[[潘阳|潘阳]]。<ref name=" duck" /><ref name="fox " />

== 趣闻 ==
因为潘长江很矮，一些小品演员经常在小品表演中调侃潘的[[身高|身高]]，潘自己也多次在小品中自嘲。

== 奖项 ==
1989年，《猪八戒拱地》荣获第三届国际青年戏剧节“个人表演金奖”。

2003年，《明天我爱你》荣获第九届[[中国金鸡百花电影节|中国金鸡百花电影节]]第23任[[百花奖|百花奖]]最佳男主角；<ref name=" ant" >{{cite web |url=http://www.cctv.com/specials/gxgh/kk1118b.html |title= 张思涛说，潘长江获奖是众望所归|publisher=中国网络电视台 |accessdate= 2003年|work= |language =zh-cn}}</ref>

2011年，《能人冯天贵》荣获第三届新农村电视艺术节“优秀农村题材电视剧一等奖”。<ref>{{cite web |url=http://ent.163.com/11/1108/05/7IAK77K900031GVS.html |title= 潘阳农村艺术节获奖 潘长江颁奖谈春晚三缄其口|publisher= 网易|accessdate= 2011-11-08|work= |language =zh-cn}}</ref>
