'''潘炕'''（{{bd|9世纪||10世纪|catIdx=P}}<ref name=dd/>），字'''凝梦'''，[[五代十国|五代十国]]时期[[前蜀|前蜀]]官员，任内[[枢密使|枢密使]]。

== 背景及效力王建 ==
潘炕出生时间和地点都无载，祖上来自[[河西|河西]]地区。潘炕为人有器量，家人不曾看到他喜怒。<ref name=SGCQ41>《[[十国春秋|十国春秋]]》[http://archive.org/stream/06080952.cn#page/n160/mode/2up 卷四十一]</ref>

[[武成_(前蜀)|武成]]三年（910年）二月，潘炕在前蜀开国皇帝[[王建_(前蜀)|王建]]手下任宣徽南院使时，王建之子太子[[王宗懿|王宗懿]]和内枢密使[[唐道袭|唐道袭]]的冲突公开化，都弹劾对方有罪，王建意图派唐道袭离开都城[[成都|成都]]任山南节度使以化解纠纷，要唐道袭推荐替任者，唐道袭最初提名潘炕的同僚宣徽北院使[[郑顼|郑顼]]。但郑顼受到委任的第二天就开始着手调查唐道袭的兄弟们盗用内库金帛之事。唐道袭惊惧，对王建奏称郑顼性格褊急，不可大任。王建听信了，出郑顼为[[果州|果州]]刺史，任潘炕为内枢密使。<ref name=ZZTJ267>[[:s:資治通鑑/卷267|《资治通鉴》卷二百六十七]]</ref><ref>《十国春秋·前蜀高祖本纪下》作三月事。</ref>

[[永平_(前蜀)|永平]]元年（911年），王建任潘炕为武泰节度使兼[[侍中|侍中]]，改任其弟宣徽南院使[[潘峭|潘峭]]为内枢密使。<ref name=ZZTJ268>[[:s:資治通鑑/卷268|《资治通鉴》卷二百六十八]]</ref>潘炕、潘峭同掌机要，号大枢小枢。<ref name=STW>《蜀梼杌》</ref>

三年（913年），已被召回成都的唐道袭和已改名[[王元膺|王元膺]]的王宗懿之间的旧怨白热化。[[七夕节|七夕节]]前夕，王元膺召诸王大臣宴饮，王建养子集王[[王宗翰|王宗翰]]、潘峭、翰林学士承旨[[毛文锡|毛文锡]]没到场。王元膺发怒了，认为是潘峭和毛文锡让王宗翰疏远了自己。次日，他向王建弹劾二人离间兄弟，王建起初信了，下令贬逐二人，再任潘炕为内枢密使。但后来，唐道袭弹劾王元膺谋反，两人最终兵戎相见，唐道袭被杀，但王建在潘炕劝说下面谕大将，王元膺军崩溃，王元膺本人也被一士兵所杀。<ref name=ZZTJ268/>

王元膺死后，十月，潘炕一直请求王建立新太子。王建误以为幼子郑王[[王宗衍|王宗衍]]得绝大多数官员支持，选立为太子。此事后，潘炕称朝廷无事，请求退休。王建本不同意，但潘炕哭泣再请，王建同意了。但即使潘炕退休了，王建也常派使者咨询他重要事务。<ref name=ZZTJ268/><ref>《十国春秋·前蜀高祖本纪下》</ref>

== 王建死后 ==
[[光天_(王建)|光天]]元年（918年），王建崩，王宗衍继位，随即改名王衍。<ref name=ZZTJ270>[[:s:資治通鑑/卷270|《资治通鉴》卷二百七十]]</ref>尽管其子内皇城使[[潘在迎|潘在迎]]成为王衍的知名狎客之一，潘炕却没有回归朝廷，也没有他继续介入国政的记载。<ref name=SGCQ41/><ref name=ZZTJ272>[[:s:資治通鑑/卷272|《资治通鉴》卷二百七十二]]</ref>[[咸康_(前蜀)|咸康]]元年（925年），前蜀被东北邻国[[后唐|后唐]]所灭后，<ref name=ZZTJ274>[[:s:資治通鑑/卷274|《资治通鉴》卷二百七十四]]</ref>潘炕也降后唐，被任为[[蜀州|蜀州]]刺史。<ref name=SGCQ41/>大约[[天成_(后唐)|天成]]二年（927年），潘炕在[[夔州|夔州]]刺史任上被罢官。<ref name=ZZTJ275>[[:s:資治通鑑/卷275|《资治通鉴》卷二百七十五]]</ref>这是史籍对潘炕的最后记载，他的卒年不详。<ref name=dd/>

== 軼事 ==
潘炕有一至爱美妾名叫赵解愁，有国色，善为新声。王建曾来潘炕官邸，对他说：“朕的宫里没有她这样的人。”希望潘炕把她献给他。潘炕答：“这只是臣下的贱人，不敢让她玷污了至尊。”潘峭举西晋[[石崇|石崇]]和其寵妾[[绿珠|绿珠]]的例子劝他献出解愁以免灾祸，潘炕说：“人生贵在适意，岂能因为怕死而不随心所欲？”人们因此敬仰他的操守。<ref name=SGCQ41/>然而潘炕也正是因为宠爱解愁而得了[[花柳病|花柳病]]。<ref name=STW/>

== 评价 ==
*《十国春秋》论曰：两潘敏于吏事，颇知大体，洵所谓玉友金昆者也。
