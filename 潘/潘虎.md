'''潘虎'''（{{lang|en|Tiger Pan}}，{{Bd|1973年|11月1日}}），中华人民共和国美术[[设计师|设计师]]，出生于[[湖北|湖北]][[武汉|武汉]]，目前是[[深圳市|深圳市]]潘虎包装设计有限公司创始人兼首席设计师、盛行传播机构首席执行官。

== 生平 ==
{{fact|1992年以中南五省考区专业第一名的成绩}}应届考入[[清华大学美术学院|清华大学美术学院]]的前身[[中央工艺美术学院|中央工艺美术学院]]装潢设计系。在校期间，获得了较为系统的设计基础教育。由于个人爱好，经常游走于工业设计系、环境艺术设计系乃至陶瓷艺术系，学习设计观念与思考方法。

1995年获得了首届[[英国|英国]]刚古纸（Conqueror Paper）机构形象设计大赛铜奖，并在王国伦教授和林桂枝的支持和推荐下，率先加入刚刚进入中国的全球4A旗舰广告公司——智威汤逊（[[J._Walter_Thompson|J. Walter Thompson]]）北京公司进行了最早的社会实践。

2010年，创立仅有12位成员的潘虎包装设计实验室在5年内，成为[[中国大陆|中国大陆]]地区的高端消费产品包装设计的领先团队。

2012年，潘虎设计的两款卷烟包装获得Pentawards 两项银奖，这是当时[[中国|中国]]设计师在烟标设计领域所获的最高奖项。

2015年，潘虎包装设计实验室设计出品的封老爷以及阿里山烟草分别获得Pentawards饮品类银奖及其他市场类银奖。<ref>{{Cite web |url = http://www.sznews.com/news/content/2015-07/02/content_11841370.htm |title = 潘虎包装设计实验室三款设计再获Pantawards奖提名 |author = 深圳新闻网 |date = 2015年7月2日 |language = 中文 }}</ref>

2016年，潘虎包装设计实验室设计出品的褚橙、极燕孕盏、极燕冰淇淋、天子壹号与龙凤呈祥硬道理在[[德国|德国]]红点设计大奖（[[Red_Dot|Red Dot]] Design Award）上一举夺得5项红点大奖。潘虎包装设计实验室此次斩获的红点奖中含金量最高的是红点最佳设计奖（Red Dot Award Best Of The Best）。<ref>{{Cite web |url = http://hn.qq.com/a/20161105/016046.htm |title = 潘虎携新作褚橙包装荣获设计界“奥斯卡”红点奖 |author = 腾讯大湘网 |date = 2016年11月05日发表 |language = 中文 }}</ref>
