'''潘震宙'''（{{bd|1942年|||}}）[[江苏|江苏]][[常州|常州]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中华人民共和国文化部|中华人民共和国文化部]]副部长，[[中国国家博物馆|中国国家博物馆]]馆长、党委书记<ref name=changzhou>{{Cite web |url = http://news.cz001.com.cn/2011-02/24/content_1477518.htm|title =访原文化部副部长、国家博物馆首任馆长潘震宙 |author =  |date = 2011-02-24 |publisher =中国常州网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1957年考入苏州第二中学。1960年考入[[南京师范学院|南京师范学院]]中文系。1964年毕业留校参加工作。1966年加入[[中国共产党|中国共产党]]。历任南京师范学院中文系助教、中文系党总支秘书。1971年，调入[[中共江苏省委|中共江苏省委]]写作小组工作。1972年，到中共江苏省委宣传部写作组工作。1979年，任中共江苏省委机关期刊《群众》杂志编辑室副主任。1983年9月到1985年7月，在[[中共中央党校|中共中央党校]]干部培训班学习。1986年，任中共江苏省委宣传部理论处处长。1987年，任中共江苏省委宣传部副部长。1990年，任中共江苏省委宣传部副部长、中共江苏省作家协会[[党组|党组]]书记。1993年，任中共江苏省委宣传部副部长、[[江苏省文化厅|江苏省文化厅]]厅长。1996年3月，任[[中华人民共和国文化部|中华人民共和国文化部]]副部长、中共文化部党组成员。2003年到2005年9月，任新成立的[[中国国家博物馆|中国国家博物馆]]馆长、党委书记<ref name=changzhou/>。后任中国国家博物馆名誉馆长<ref>{{Cite web |url = http://news.163.com/11/0517/15/7490ON9H00014JB5.html |title = “古代中国”基本陈列在国博预展 |author =  |date = 2011-05-17 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>，中国戏曲现代戏研究会名誉会长<ref>潘震宙，在中国戏曲现代戏研究会2012年年会暨河南省豫剧三团成立六十周年纪念会上的讲话，南腔北调·河南戏剧2013(1)</ref>等职。他是[[第十届全国政协委员|第十届全国政协委员]]。
