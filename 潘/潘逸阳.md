'''潘逸阳'''（{{bd|1961年|8月||catIdx=P潘}}），[[广东省|广东]][[惠阳区|惠阳]]人，生于[[广州市|广州]]。汉族。1982年1月参加工作，1984年8月加入[[中国共产党|中国共产党]]。[[中山大学|中山大学]]数学力学系力学专业毕业，中山大学哲学系[[马克思主义哲学|马克思主义哲学]]专业[[在職研究生|在职研究生]]学历，[[哲學博士|哲学博士]]。[[中国共产党第十八届中央委员会候补委员列表|中共第十八届中央候补委员]]。

== 簡歷 ==
历任[[中国共产主义青年团|中国共青团]]广东省[[佛山市|佛山市]]委书记、广东省委副书记、书记、共青团中央常委，[[中国共产党广州市委员会|中共广州市委]]常委、[[从化区|从化]][[市委書記|市委书记]]，[[新余市|新余]]市委书记等职。

2001年12月，任[[中国共产党江西省委员会|中共江西省委]]常委、省委农村工作部部长。2003年9月，改兼[[赣州市|赣州]]市委书记，

2010年10月，任[[中国共产党内蒙古自治区委员会|中共内蒙古自治区党委]]常委，次月任[[内蒙古自治区人民政府|内蒙古自治区人民政府]]常务副主席。2014年9月，任内蒙古自治区党委常委、自治区政府常务副主席、党组副书记、自治区委党校（[[内蒙古自治区行政学院|内蒙古自治区行政学院]]）院长。

2014年9月17日，潘逸阳因涉嫌严重违纪违法，而被组织调查<ref>{{cite web |accessdate= |url = http://politics.people.com.cn/GB/41223/13117986.html |title=潘逸阳任内蒙古自治区党委常委、副主席(图/简历) |publisher=人民网}}</ref>。因涉嫌严重违纪违法，2014年9月27日，内蒙古自治区第十二届人大常委会第十二次会议决定罢免其[[第十二届全国人民代表大会代表名单|第十二届全国人民代表大会代表]]职务<ref>{{cite news|title=第十二届全国人民代表大会常务委员会代表资格审查委员会关于个别代表的代表资格的报告（2014年11月1日第十二届全国人民代表大会常务委员会第十一次会议通过）|url=http://www.npc.gov.cn/wxzl/gongbao/2014-12/23/content_1892465.htm|accessdate=2016-07-03|newspaper=中国人大网}}</ref>。2015年10月16日被双开，移送司法机关<ref name=PanYu>{{cite web |url =http://www.ccdi.gov.cn/xwtt/201510/t20151016_63607.html|title =内蒙古自治区党委原常委、自治区政府原副主席潘逸阳，广西壮族自治区党委原常委、南宁市委原书记余远辉严重违纪被开除党籍和公职 |accessdate = 2015年10月16日 |publisher = 中央纪委监察部网站}}</ref>。

潘逸阳曾是落马的原江西省委书记[[苏荣|苏荣]]的属下。[[令计划|令计划]]案判决中显示，他曾向令某妻子[[谷丽萍|谷丽萍]]行贿761万余元。

2017年4月18日，天津市第一中级人民法院公开宣判：潘逸阳在[[江西省|江西]]、[[内蒙古自治区|内蒙古]]担任领导职务期间，非法收受他人财物，折合人民币共计8601万余元，并多次向时任[[中国共产党中央书记处|中共中央书记处]]书记、[[中共中央办公厅|中央办公厅]]主任[[令计划|令计划]]行贿，折合人民币共计761万余元。依法以受贿罪判处其有期徒刑15年，并处罚金人民币400万元；以行贿罪判处有期徒刑7年；数罪并罚，决定执行有期徒刑20年，并处罚金人民币400万元；对其受贿所得及其[[孳息|孳息]]予以追缴，上缴国库<ref>{{cite web |url = http://www.ccdi.gov.cn/yw/201704/t20170418_97520.html |title = 内蒙古自治区原副主席潘逸阳受贿、行贿案一审宣判 |accessdate = 2017年4月18日 |publisher = 中央纪委监察部网站}}</ref>。

== 语录 ==
潘逸阳在中共中央纪委给予其开除党籍处分的决定书上所写的意见是：“同意。我表示服从中央的决定。我深知，我的行为已构成大错，严重背离了党的宗旨，辜负了党的信任和培养，已严重违纪违法，深感愧疚，痛心疾首。在此，我向党表示真诚忏悔，负荆请罪。祈求组织能够给予宽大处理，给我一个重新做人的机会。我决心痛改前非，做一个安分守己的好公民。我将永远感激党的恩情，一辈子跟党走。最后，衷心祝愿党的事业兴旺发达，衷心祝愿祖国繁荣昌盛！”<ref>{{cite web |url = http://news.sina.com.cn/c/nd/2017-09-27/doc-ifymeswe0379474.shtml|}}</ref>
