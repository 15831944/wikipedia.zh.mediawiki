== 简介 ==
1950年12月11日，潘绥铭出生于一个高干家庭。<ref name="性学第一人潘绥铭：闯入红灯区与小姐一起吃住 " />九岁时，他的父亲就被发配，到三十岁时父亲才平反离休。<ref name="性学第一人潘绥铭：闯入红灯区与小姐一起吃住 ">{{cite web |url=http://news.163.com/41110/2/14QV1FHO00011246_3.html|title=《性学第一人潘绥铭：闯入红灯区与小姐一起吃住》|accessdate=2004-11-10 |work=|publisher=网易|language=zh-hans}}</ref>

1966年[[文革|文革]]开始，1968年，潘绥铭作为[[知青|知青]]来到[[黑龙江|黑龙江]]国营农场。1973年，潘先生离开了[[黑龙江|黑龙江]]，回到了[[内蒙古|内蒙古]]，上[[研究生|研究生]]时，他的女儿已经8个月了。知青期间，潘绥铭当过[[红卫兵|红卫兵]]、[[狗崽子|狗崽子]]、[[农场|农场]]工人、大集体镀锌工、工农兵中专生和机关小职员。<ref name="性学第一人潘绥铭：闯入红灯区与小姐一起吃住 " />

1980年，潘绥铭在[[东北师范大学|东北师范大学]]读研时，他选择了世界中世纪史。1984年，潘大学毕业后，在[[中国人民大学|中国人民大学]]当教师。1985年，他第一次开设了性社会学课。听课女学生骂他“脸皮厚”，中国人民大学的教授们对他不屑一顾。学校教师劝他停课，学校领导审查他的身份，这也间接地造成了潘绥铭在争议中一举走红。

1988年，潘绥铭的第一本性学专著《神秘的圣火：性的社会学史》出版发行。2000年，潘绥铭一共出版了《中国性现状》、《性，你真的懂了吗？》、《存在与荒谬：中国地下性产业考察》、《生存与体验》、《当代中国大学生的性观念与性行为》六本专著，还有《金西报告：人类男性性行为》等三本译著出版发行。

自1997年起，潘绥铭数度一个人入住[[红灯区|红灯区]]，和“[[小姐|小姐]]们”吃住一起，做[[学术|学术]]研究和考察，后来出版了[[中国|中国]]地下性产业的专著《存在与荒谬》。

2012潘绥铭教授因“科研资金使用不明”而遭到了行政处分，从二级教授降为三级教授，退休年龄也降为60岁，故被迫提前前年退休。潘绥铭长期主持对中国性工作者的大型访谈，其中牵涉到给性工作者访谈报酬的问题，而此次事发的原因也正与此笔报酬因无法开发票而导致使用不明有关。<ref name="性学教授或因有偿访谈小姐不能开发票 被处分 ">{{cite web |url=http://d.youth.cn/shrgch/201410/t20141028_5921690.htm|title=《性学教授或因有偿访谈小姐不能开发票 被处分》|publisher=中国青年网|language=zh-hans}}</ref>

== 思想 ==
潘绥铭是[[性学|性学]]家，曾自谓：“[[饭岛爱|饭岛爱]]永远活在我的硬盘里，我也看[[成人漫画|耽美漫画]]。”<ref name="著名性学专家：不能把“A片”当作性知识看 " /><ref name=" 潘绥铭：饭岛爱永远活在我的硬盘里" />潘绥铭不认同“[[处女情结|处女情结]]”，认可“[[婚前性行为|婚前性行为]]”鼓励年轻人恋爱，“在校期间没有谈过恋爱等于白念四年，这是教育部长的失职，该为此道歉！”<ref name=" 潘绥铭：饭岛爱永远活在我的硬盘里" />并且撰文提倡青少年婚前性行为，“性教育只应该有一个‘终极关怀’的目标：帮助所有的个体，尤其是下一代，都尽可能多地获得‘性福’。性教育面对的是人，是青春。这些，我们牺牲不起，也没有权利去牺牲。”<ref>{{cite web|url=http://news.cn.yahoo.com/051205/1038/2gclr.html|title=《性学家潘绥铭：不让青少年性交叫什么性教育?》|accessdate=2005-12-05|work=|publisher=雅虎|language=zh-hans}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

=== 爱情论 ===
潘绥铭诠释“什么是[[爱情|爱情]]”时说：“中国传统上只有夫妻恩爱。浪漫情爱，是从[[西方|西方]]传过来的，公元11世纪，[[法国|法国]][[普罗旺斯省|普罗旺斯省]]出现了第一个歌颂浪漫爱情的游吟诗篇《[[罗兰骑士之歌|罗兰骑士之歌]]》（实际上“La Chanson De Roland”中几乎完全没有女性角色，仅在诗歌最后数行简单交代了罗兰的恋人闻其死讯悲痛而亡），在那之前没有这东西，在西方经过1000年的成长，[[五四|五四]]之后传到[[中国|中国]]。现在但凡读书人，受过点教育的人，都信仰这个。那么在浪漫情爱出产地西方，为什么出现了性爱分离？其中的一个原因就是爱情难于实现哪，这不就是这么简单吗？[[西方|西方]][[性革命|性革命]]为什么首先从[[大学生|大学生]]中刮起来，尤其在[[美国|美国]]特别明显，跟这个有相当大的关系——爱情的幻灭。[[美国|美国]]进入[[性革命|性革命]]的第三阶段，性和爱分离了。”<ref name="潘绥铭：中国的性革命已经基本成功 " />

=== 处男处女情结 ===
2010年，潘绥铭在[[广州大学|广州大学]]演讲中解说了自己的“[[处男|处男]][[处女情结|处女情结]]”观点：“我们过去把[[性欲|性欲]]定义太窄了，只定义为男女上床，性幻想这些算不算啊？如果你把这些精神活动加起来的话，男性[[性欲|性欲]]最旺盛的是[[青春期|青春期]]的第一年。其实对于[[男性|男性]]来说，你就想到[[处女|处女]]的问题，有[[处男|处男]]吗？男生从[[青春期|青春期]]第一天开始就[[遗精|遗精]]，[[性梦|性梦]]至少三天一回吧，[[性幻想|性幻想]]至少一天五回吧？”他指着男大学生们问：“其实你就经常这样过着性生活，都十来年了，你还说你是[[处男|处男]]？！”<ref name="著名性学专家：不能把“A片”当作性知识看 ">{{cite web |url=http://news.xinhuanet.com/edu/2010-10/28/c_12712310.htm|title=《著名性学专家：不能把“A片”当作性知识看》|accessdate=2010年10月28日|work=|publisher=新华网|language=zh-hans}}</ref>

=== 一夜情 ===
2004年11月，潘绥铭在[[广东省|广东省]]科学馆作“21世纪中国的性问题”性学讲座， 为“[[一夜情|一夜情]]”等最新性词语做了自己的注解。潘绥铭认为“[[一夜情|一夜情]]”的主要特征是“邂逅相逢”、“就性论性”和“互不纠缠”，而并非很多人认为的买[[卖淫|卖淫]]。<ref name=" 潘绥铭：饭岛爱永远活在我的硬盘里" />

=== 性工作者 ===
2004年11月，潘绥铭在[[广东省|广东省]]科学馆作“21世纪中国的性问题”性学讲座，对于“[[性工作者|性工作者]]”，潘绥铭表示了自己的观点：“[[小姐|小姐]]也是人，对她们进行的研究，也必须是在平等和尊重的基础上进行”。<ref name=" 潘绥铭：饭岛爱永远活在我的硬盘里" />
