'''潘复故居'''建于1919年，是清末[[举人|举人]]、原山东省实业司司长、北洋政府财政次长兼盐务署署长、边业银行创办者、[[中华民国军政府|中华民国军政府]]内阁总理[[潘复|潘复]]的住宅。位于当时的[[天津英租界|天津英租界]]的马厂道（Race Course Road）（今[[和平区_(天津市)|和平区]][[马场道|马场道]]2号），该故居作为[[五大道|天津五大道近代建筑群]]的重要组成部分，被[[中华人民共和国国务院|中华人民共和国国务院]]批准为[[天津全国重点文物保护单位列表|全国重点文物保护单位]]<ref>[[中华人民共和国国务院|中华人民共和国国务院]]，二零一三年五月三日公布，[[天津市人民政府|天津市人民政府]]立</ref>。现为[[天津市第二十中学|天津市第二十中学]]行政楼。

== 历史 ==
该建筑是潘复于1919年委托[[开滦煤矿|开滦煤矿]]董事长[[庄乐峰|庄乐峰]]请法国建筑师设计并建造的，在1919年至1939年期间为潘复的居住地，[[中华人民共和国|中华人民共和国]]成立后，该建筑曾先后为天津市农林局、交通银行天津分行办公使用。2005年，该建筑因建[[天津地铁一号线|天津地铁一号线]][[小白楼站|小白楼站]]拆除，2006年，潘复旧居复建并作为天津市第二十中学行政楼使用。<ref>[http://www.tianjinwe.com/tianjin/tbbd/201001/t20100113_483436.html 快报“小洋楼”之潘公馆]</ref>
　　
== 建筑风格 ==
潘复故居占地面积十余亩，建筑面积为3827．99平方米，院内共有楼房和平房共117间。院西北角建有平房，院子以铁栅围墙并种有各种花木和草坪绿地，院子中间设有甬道直通主楼正门。潘复故居分为东西两楼，东楼为砖木结构三层楼房，屋顶为大瓦、瓦垄铁顶，外立面为水泥抹面，设有圆形和五面形阳台，建筑内部设有11处卫生间和洗澡间，门窗地板均用菲律宾木料。西楼为砖木结构三层楼房，一楼设有两间大会客室，原故居内部的墙壁和顶棚均装饰有油画，墙壁装饰有花、木、鱼、虫图案，顶棚绘制有四个西方美女形象。该建筑是一座具有西欧风格的花园式楼房。<ref>{{Cite web |url=http://www.tjwh.gov.cn/shwh/lywh/mrgj/pan-fu/jzfg---pf.htm |title=潘复故居 |access-date=2012-04-24 |archive-url=https://web.archive.org/web/20130512072850/http://www.tjwh.gov.cn/shwh/lywh/mrgj/pan-fu/jzfg---pf.htm |archive-date=2013-05-12 |dead-url=yes }}</ref>
