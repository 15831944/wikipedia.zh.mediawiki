'''腓尼基'''（[[腓尼基语|腓尼基语]]：{{script|Phnx|𐤊𐤍𐤏𐤍}}；{{lang-en|Phoenicia}}；{{lang-el|Φοινίκη}}；{{lang-ar|فينيقية}}；[[埃及语|埃及语]]：''fnḥw'' (''fenkhu'')<ref>Françoise Briquel-Chatonnet and Eric Gubel, ''Les Phéniciens : Aux origines du Liban'' (Paris: Gallimard, 1999), 18.</ref>）是古代[[地中海|地中海]]东岸的一个地区，其范围接近于如今的[[黎巴嫩|黎巴嫩]]和[[叙利亚|叙利亚]]。腓尼基人是[[闪米特人|闪米特人]]的一支，乃[[犹太人|犹太人]]的邻。腓尼基人善于[[航海|航海]]与[[经商|经商]]，在全盛期曾控制了西地中海的贸易。他们的[[腓尼基字母|腓尼基字母]]，與[[希伯来字母|希伯来字母]]、[[希腊字母|希腊字母]]和[[拉丁字母|拉丁字母]]同源。

腓尼基人是[[闪族|闪族]]的一支。为了便于修筑堡垒以及防御，大部分的腓尼基城镇都建在沿海地带，而且是各自独立的。

== 历史 ==
[[File:Griechischen_und_phönizischen_Kolonien.jpg|thumb]]
他們於公元前十四、五世紀時定居於地中海濱，建立了許多城邦。根據[[聖經|聖經]]記載，古腓尼基城市包括現今在[[以色列|以色列]]境內的[[阿什杜德|阿什杜德]]、[[阿什凯隆|阿什凯隆]]和[[以革倫|以革倫]]；在[[黎巴嫩|黎巴嫩]]境內的[[泰尔|泰尔]]和[[赛达_(黎巴嫩城市)|西頓]]。其中泰爾和西頓二城曾先後為各城邦的盟主。但他們並不曾組織過近代式的國家，在政治上不過是[[古埃及|埃及]]、[[亞述|亞述]]等帝國的附庸而已。

== 工商业与航海贸易 ==
腓尼基社会主要以商业、玻璃器皿制造业、纺织业以及金属制造业为主，而其中最著名的、同时使他们获利最多的，是从地中海海螺身上提取酱汁进而加工成一种紫色颜料。[[File:PhoenicianTrade.png|thumb]]
腓尼基人擅长航海和贸易。腓尼基所处的地理环境面朝大海，交通发达，而在东面的[[黎巴嫩|黎巴嫩]]山区盛产轻质木材，为造船业提供了便利条件。所以腓尼基人从事航海贸易有着得天独厚的优势。

环绕[[地中海|地中海]]，腓尼基人建立了往返于[[希腊|希腊]]、[[西西里岛|西西里岛]]、[[撒丁岛|撒丁岛]]、[[巴利阿里群岛|巴利阿里群岛]]、[[伊比利亚半岛|伊比利亚半岛]]、[[加那利群岛|加那利群岛]]的[[航线|航线]]，他们的定期船只往來於北非、[[累范特|累范特]]、希臘和[[地中海|地中海]]各島嶼。为了贸易他们曾冒险进出[[直布罗陀海峡|直布罗陀海峡]]到[[加那利群岛|加那利群岛]]进行[[锡|锡]]的采挖和买卖。甚至有证据表明他们的商船曾远至[[不列颠群岛|不列颠群岛]]，尤其是在前600年左右在[[古埃及|古埃及]][[新王國時期|新王國時期]]晚期的[[法老|法老]]支持下的一支由四十艘船组成的舰队首先从[[西奈半岛|西奈半岛]]的[[亚喀巴湾|亚喀巴湾]]出发沿着[[红海|红海]]岸走，穿过[[曼德海峡|曼德海峡]]进入[[亚丁湾|亚丁湾]]；其次沿着今天的索马里海岸线一直南下到达今[[南非|南非]]南海岸；再次从南非和今[[纳米比亚|纳米比亚]]的非洲西海岸北上到[[几内亚湾|几内亚湾]]，最后沿着西非的海岸线转入[[直布罗陀海峡|直布罗陀海峡]]进入地中海回到家乡腓尼基。历时三年，航程三万公里並環繞[[非洲|非洲]]。

腓尼基人也是一群優秀的軍事家，他们为了争夺地中海航线上的利益与当时活跃在地中海的其他势力之间曾发生过多次战事。

== 殖民地 ==
由于腓尼基地狭人稠，他们便向其在航线上建立的贸易据点殖民。地中海的西海岸，特别是[[西班牙|西班牙]]和[[北非|北非]]都遍布他们的殖民地。这些殖民地中有一些在日后的历史里发展壮大，其中比较著名的就是后来与[[古罗马|古罗马]]发生过多次战争的[[迦太基|迦太基]]。有意见以为，腓尼基人是部分[[突尼斯人|突尼斯人]]的祖先。

=== 伊比利亚半岛 ===
腓尼基人一直试图垄断伊比利亚半岛南部的金属开采（锡、铜、银）以及相应的贸易路线。从公元前十一世纪开始，腓尼基人就在伊比利亚半岛上建立商业据点，其中较早且重要的举措之一就是在直布罗陀海峡建立城市加地尔{[[腓尼基语|腓尼基语]]：גדר，意为“被墙围绕的城市”，今名[[加的斯|加的斯]]（西班牙语：Cádiz）}。

除此之外，腓尼基人也在地中海沿岸的马拉加（Malaka，现在的西班牙马拉加 Málaga）、塞西（Sexi，现今的西班牙[[阿尔穆涅卡尔|阿尔穆涅卡]]）和阿德拉（Abdera，现在的西班牙[[阿尔梅里亚|阿尔梅里亚]]）建立了商业据点。

在腓尼基人殖民时期，[[安達盧西亞|安达卢西亚]]地区不仅有矿产、金属等商业活动，也盛行陶器、紫色布、编织品、酒、油、玻璃等手工业产品的贸易，也不乏鱼、腌制品以及奴隶贸易。

尽管腓尼基人与半岛当地的塞尔特伊比利亚人之间并无巨大的冲突，然而腓尼基人的势力仍无法渗透入半岛内部。腓尼基人只是藉由商业活动，向半岛南部的居民传播文化，教导当地人使用钱币、使用字母、金属制造技术以及编织技术，其影响力也反映在了伊比利亚人的宗教信仰上。

== 腓尼基字母 ==
{{main|腓尼基字母}}
腓尼基人最初使用[[苏美尔人|苏美尔人]]创制的[[楔形文字|楔形文字]]。后来为了提高效率，在公元前1000年左右他们以原始[[迦南|迦南]]字母为基础，合并埃及的部分[[象形文字|象形文字]]和简化后的楔形文字设计出了22个腓尼基字母。腓尼基字母牺牲了前人文字的华丽外形来换取更高的书写效率。

== 文化交流 ==
腓尼基人對外營商發達，與其他民族通婚，因此他們失去了他們的文化特色，漸漸受異族同化了。雖然如此，腓尼基人在[[新約聖經|新約]]時代仍是一群強大的民族。
=== 埃及 ===
他們深受埃及文化的影響，奉的是[[多神教|多神教]]。所以他們學會了製作[[木乃伊|木乃伊]]的技巧，並把死者遺體製成為木乃伊後放入人形的棺槨內；也朝拜[[埃及|埃及]]的神祇。
=== 希臘 ===
腓尼基也是[[希臘文明|希臘文明]]的来源之一。希臘人從他們的商船上得到了許多東方的製造品，學到了許多工藝美術。但其中最重要的乃是腓尼基人的[[字母|字母]]。希臘人學會了那些字母之後，又加上了幾個[[母音|母音]]，完成了字母的作用，成為後來歐洲各類字母之祖。腓尼基的商船上又載有埃及等處運來的筆墨紙等文具，在這個情形之下，希臘的文學也就漸漸地萌芽起來了。

== 名考 ==
近代的语言学家确认“phoinix”这个词源自[[印欧语系|印欧语系]]，意为“紫色/紫红色”。希腊人应是将住在腓尼基地区“拥有面部黝黑肤色”的居民与紫色联想在一起（“Phoínik”是指腓尼基人著名的工业，可能与紫红色染布工业有关，而复数“phoínikes”指的是腓尼基人）。即便如此，“腓尼基”这个词的来源依然存在混淆，在其他的语言里有着不同的含义。
