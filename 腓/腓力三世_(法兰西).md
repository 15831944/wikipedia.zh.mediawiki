'''腓力三世（勇敢者）'''（{{lang-fr|Philippe III le Hardi}}，{{bd|1245年|4月30日|1285年|10月5日|catIdx=P}}）[[卡佩王朝|卡佩王朝]]第10位国王（1270年—1285年在位）。

== 事迹 ==

腓力三世为法蘭西國王[[路易九世_(法蘭西)|路易九世]]之子，生于[[普瓦西|普瓦西]]，母亲是[[普罗旺斯|普罗旺斯]]公主玛格丽特。他在很小的时候就获得了[[奥尔良|奥尔良]]伯爵的爵位。1270年，腓力与父亲一起参加[[第八次十字军|第八次十字军]]，前往[[突尼斯|突尼斯]]进行远征；结果路易九世在这次远征中因瘟疫死于突尼斯，腓力三世遂在当地被宣布为法兰西国王，时年25岁。

腓力三世是一个性格软弱、优柔寡断的国王。他获得“勇敢者”这个外号并非由于他性格勇敢，而是因为他善于骑马打仗。腓力三世是一个虔诚的[[基督教徒|基督教徒]]，但缺少文化；在他统治时期，他在政策方面往往受到权臣的影响。这些权臣中最重要的是[[皮埃尔·德·拉·布罗切|皮埃尔·德·拉·布罗切]]。腓力的叔叔、[[那不勒斯|那不勒斯]]國王[[查理一世_(那不勒斯)|查理一世]]也对腓力有巨大的影响力。

腓力三世在即位之后立刻请求他的叔叔查理一世与[[穆斯林|穆斯林]]的领袖谈判，以期结束这次东征，而他本人却返回了法蘭西。结果十字军与穆斯林签署了一份10年休战协议。1271年8月12日，腓力三世正式加冕为法蘭西國王。不久腓力的另一个叔叔[[圖盧茲|圖盧茲]]伯爵[[阿方斯_(图卢兹)|阿方斯]]去世，由于他没有子嗣，腓力兼并了他的大部分领地。这些新获领地使王室的力量大为扩充。

1279年，腓力三世与[[英格蘭|英格蘭]]國王[[爱德华一世|爱德华一世]]签署了[[1279年亚眠条约|1279年亚眠条约]]。根据此条约，腓力三世把[[阿让奈|阿让奈]]割让给爱德华一世。这个损失后来由于他兼并了其无嗣的弟弟皮埃尔的领地[[阿朗松|阿朗松]]而获得了部分补偿（1284年）。

腓力三世积极支持他叔叔查理一世在[[意大利|意大利]]的事业，但收效甚微（[[西西里晚祷起义|西西里晚祷起义]]之后，查理一世就失去了对[[西西里|西西里]]的控制权）。另一方面，腓力三世在[[納瓦拉|納瓦拉]]国王[[恩里克一世_(纳瓦拉)|恩里克一世]]去世后企图为他的儿子[[腓力四世_(法兰西)|腓力]]（后来的腓力四世）谋取納瓦拉王位，于是让儿子与纳瓦拉女继承人[[胡安娜一世|胡安娜一世]]结婚。

1284年，腓力三世在[[教宗|教宗]]的劝诱下（教宗承诺如法国进攻[[阿拉贡|阿拉贡]]就把阿拉贡王位赠给腓力的儿子[[瓦卢瓦的查理|瓦卢瓦的查理]]）率领大军进入[[鲁西永|鲁西永]]，企图从阿拉贡王室（他们受到教皇的敌视）手中夺取这个地区。这次战争有时被称为“反阿拉贡十字军”，一些历史学家称其为“卡佩王朝进行过的最非正义，最不必要和最灾难性的军事冒险”<ref>Chaytoe，阿拉贡和加泰罗尼亚历史，105页</ref>。腓力三世开始时取得不小的进展；但法军之中突然爆发了一场[[痢疾|痢疾]]，而在那个时代这种病是极其危险的。腓力本人也染上了疾病。法军被迫撤退，在途中被阿拉贡军堵截，遭到惨败。腓力三世本人没能返回领地就在他的盟友、[[马略卡|马略卡]]国王[[海梅二世_(马略卡)|海梅二世]]位于[[佩皮尼昂|佩皮尼昂]]的宫廷里去世了。他的儿子腓力四世继承了王位。

== 家庭 ==

腓力三世于1262年5月28日与[[阿拉贡|阿拉贡]]国王[[海梅一世|海梅一世]]的女儿[[阿拉贡的伊莎贝拉|阿拉贡的伊莎贝拉]]结婚。他们生了3个孩子：
# 路易（未活到成年）
# [[腓力四世_(法蘭西)|腓力四世]]
# [[查理_(瓦盧瓦伯爵)|查理]] ，[[瓦盧瓦領主列表|瓦盧瓦伯爵]]、[[:en:Counts_and_Dukes_of_Alençon|阿朗松伯爵]]、[[:en:Duc_de_Chartres|沙特爾伯爵]]、[[安茹伯爵|安茹伯爵]]、[[:en:Counts_and_Dukes_of_Maine|曼恩伯爵]]，瓦卢瓦王朝始祖。

在伊莎贝拉去世后，腓力三世于1274年8月21日再次结婚，对象是[[布拉班特|布拉班特]]公爵[[亨利三世_(布拉班特)|亨利三世]]的女儿玛丽亚。他们生了以下子女：
# [[埃夫勒的路易|埃夫勒的路易]] 
# 布朗歇
# [[法兰西的玛格丽特|法兰西的玛格丽特]]
