'''腓力二世·奥古斯都'''（[[法语|法语]]：Philippe II Auguste，{{bd|1165年|8月21日|1223年|7月14日|catIdx=P}}）[[卡佩王朝|卡佩王朝]]國王（1180年—1223年在位），法國名君之一。

因為饒富謀略與政治手腕而有「尊嚴王」的美稱。

== 早年事迹 ==

腓力二世生于[[巴黎|巴黎]]，是[[西法兰克王国|西法蘭克國]]王[[路易七世_(法兰西)|路易七世]]之子，母亲为香槟的阿黛勒。其父因健康欠佳，在1179年便让14岁的腓力在[[兰斯|兰斯]]加冕。1180年4月28日，腓力与埃诺伯爵之女埃诺的伊莎贝拉结婚，这使得阿图瓦作为嫁妆被并入法国王室领地。

路易七世于1180年9月18日去世后，15岁的腓力二世成为唯一的国王。他的叔父香槟伯爵亨利一世、兰斯大主教纪尧姆·德·布卢瓦和沙特尔伯爵蒂博五世被宣布为國家的[[摄政|摄政]]。

== 他的统治 ==

腓力二世是卡佩王朝的第一位强大的君主。在执政初期，他首先力图摆脱摄政的束缚。与埃诺的伊莎贝拉的婚姻在客观上有利于腓力与香槟家族的斗争。尤其是1180年其父路易七世病危时，腓力二世抛开摄政独自与来访的[[英格蘭|英格蘭]]國王[[亨利二世_(英格兰)|亨利二世]]会谈并签订条约，使香槟家族陷于孤立。后来他迫使香槟的亨利割让了大量领地，包括[[阿米耶努瓦|阿米耶努瓦]]、[[维尔芒杜瓦|维尔芒杜瓦]]和[[瓦卢瓦|瓦卢瓦]]。这些领地成为王室领地的一部分，充实了腓力二世的力量。然而不久，香槟家族与[[佛兰德伯爵|佛兰德伯爵]][[菲利普·德·阿尔萨斯|菲利普·德·阿尔萨斯]]联合，发起一场反对腓力二世的战争。经过几年艰苦的努力，腓力于1185年与菲利普达成协议，解决了双方的领土纠纷。

腓力二世奉行明显的使国家[[集权|集权]]化的政策，力图抑制使法国王权陷于瘫痪的强大诸侯。这主要通过以下办法来实现：争取小封建领主和[[基督教|教会]]的支持，与经济繁荣的[[城市|城市]]维持良好关系（腓力二世大大扩展了[[路易六世|路易六世]]给予城市自治权的制度，并且把市政宪章实行到诸侯领地内的城市中去，这就使得大贵族受到自己领地内城市的掣肘），极力拓展王室领地。这其中最重要的还是王室领地的直接扩大，这使腓力二世有能力与桀骜不驯的贵族进行对抗。通过4次大规模的合并领地，腓力二世事实上成为國內最大的封建领主。

王室领地的最大扩展来自于腓力二世与英格蘭[[金雀花王朝|金雀花王朝]]诸国王的斗争。金雀花王朝的始祖亨利二世由于出身西法蘭克贵族而在法蘭西境内拥有大量领地，包括[[安茹|安茹]]和[[诺曼底|诺曼底]]，又通过婚姻获得对[[阿基坦|阿基坦]]的统治权。为了削弱亨利二世的势力，腓力二世竭力挑拨亨利和他的几个儿子（[[亨利_(幼王)|亨利]]、[[理查一世|理查]]、[[若弗鲁瓦二世_(布列塔尼)|乔弗雷]]、[[约翰_(英格兰)|约翰]]）之间的关系。腓力二世支持理查和约翰在亨利二世晚年发动的叛乱；在[[理查一世|理查一世]]继承王位后，他又积极策划打击理查。

腓力二世是1189年~1192年的[[第三次十字军东征|第三次十字军东征]]的领袖之一。[[欧洲|欧洲]]的另外两位重要君主，[[神聖羅馬帝國皇帝|神聖帝國皇帝]][[腓特烈一世_(神聖羅馬帝國)|腓特烈一世]]和英格蘭國王[[理查一世|理查一世]]也参加了这次十字军东征。这次东征毫无成果：腓特烈在[[小亚细亚|小亚细亚]]渡河时淹死，腓力二世与理查矛盾明显而分道扬镳（腓力于1191年返回法蘭西）。理查单独与穆斯林领袖[[萨拉丁|萨拉丁]]议和后，在返回途中被新任神聖帝國皇帝[[亨利六世_(神聖羅馬帝國)|亨利六世]]的封臣、[[奧地利統治者列表|奧地利公爵]][[利奥波德五世_(奥地利)|利奥波德五世]]俘虏。亨利六世后来成为腓力二世的重要盟友，他们在反对理查的问题上是意见一致的。腓力二世不断蚕食金雀花王朝领地的行为终于引起理查的反击：在[[佛兰德伯爵|佛兰德伯爵]][[博杜安九世|博杜安九世]]和[[纳瓦拉国王|纳瓦拉国王]][[桑乔六世_(纳瓦拉)|桑乔六世]]的支持下，理查一世在法蘭西本土向腓力发起攻击。一时间腓力二世似乎腹背受敌，因为桑乔六世从南方侵袭法蘭西。但是理查一世的运气很不好。在与腓力二世艰难地达成了一个为期5年的停战协议（1199年1月13日）之后，理查去围攻其宿敌[[利摩日子爵|利摩日子爵]]的领地[[沙吕斯堡|沙吕斯堡]]。他在是役中被[[箭|箭]]射中，不久因伤重去世。理查对腓力的战役没有取得任何重要结果。

在理查一世去世后，腓力二世继续反对他的继任者[[约翰_(英格兰)|约翰]]（亨利二世最小的儿子）。腓力与约翰的第一次矛盾爆发于1200年，起因是后者与[[昂古莱姆的伊莎贝拉|昂古莱姆的伊莎贝拉]]的婚姻。昂古莱姆的伊莎贝拉原本是[[马尔什伯爵|马尔什伯爵]][[于格十世·德·吕济昂|于格十世·德·吕济昂]]的未婚妻，她与约翰的婚姻使[[吕济昂家族|吕济昂家族]]失去了可能的领地嫁妆，于格·德·吕济昂遂请求腓力二世主持公道。腓力二世抓住机会打击约翰，他以领主的身份要求其封臣、[[阿基坦公爵|阿基坦公爵]]约翰到法蘭西应诉。在遭到约翰当然的拒绝后，腓力宣布金雀花家族在法蘭西的所有领地都被法蘭西國王没收。腓力并于1202年与[[布列塔尼公爵|布列塔尼公爵]][[阿尔蒂尔一世|阿尔蒂尔一世]]结盟。阿尔蒂尔是约翰的侄子，他拥有对英格兰王位的继承权，金雀花王朝的许多法蘭西附庸都希望用他来取代刚愎自用的约翰。腓力二世利用这一点诱使阿尔蒂尔一世向他效忠，约定夹击约翰的领地（诺曼底和[[普瓦图|普瓦图]]）。然而阿尔蒂尔在1203年7月31日于[[米拉博|米拉博]]附近遭到约翰的奇袭，被俘虏后带往[[鲁昂|鲁昂]]。不久阿尔蒂尔就神秘地失踪了，普遍认为他被约翰下令杀害。腓力二世于是挑明了反对约翰的立场，于是年再次进攻诺曼底；他在摧毁了许多城堡之后，集中兵力攻打[[加亚尔堡|加亚尔堡]]。尽管英格兰士兵英勇奋战，这座工事还是于1204年3月6日陷落。整个诺曼底随后都落入腓力二世手中。令人惊异的是，在腓力二世侵入诺曼底时，约翰却返回了英格兰。

1206年10月13日，腓力二世与约翰在[[图阿尔|图阿尔]]签订和约。这个和约对法蘭西王室压倒性地有利：约翰被迫放弃[[罗亚尔|罗亚尔]]以北所有原属金雀花王朝的领地，包括诺曼底、阿基坦、安茹和[[图赖讷|图赖讷]]。按照条约，约翰应保有普瓦图；但腓力于1207年就又侵入了这一地区。

约翰希望夺回失去的领地，与[[欧洲|欧洲]]大陆上另一位重要君主、[[神聖羅馬帝國|神聖帝國]][[皇帝|皇帝]][[奥托四世|奥托四世]]结盟，约定从东西夹击腓力二世的法蘭西王國。他可谓是找错了结盟对象；奥托四世本身正受到[[教廷|教廷]]史上最强大的[[教宗|教宗]][[英诺森三世|英诺森三世]]的孤立。1214年2月，约翰最终在[[拉罗歇尔|拉罗歇尔]]登陆，很快进入安茹。1214年7月21日，腓力二世领导法军与约翰和奥托四世的联军在[[布汶战役|布汶战役]]中展开决战。法军在战斗中获得了决定性的胜利。

布汶战役具有历史意义。它不仅是[[罗马帝国|罗马帝国]]灭亡以来[[西欧|西欧]]发生的第一场伤亡重大的战役，而且还标志着法兰西取代德意志成为欧洲大陆上最主要的国家。德意志在以后的历史中不断衰弱、分裂，而法蘭西则在[[百年战争|百年战争]]后成为西欧的头号强国。

尽管腓力二世与[[基督教|教会]]一直保持着良好关系，但他从未受制于他们。他对教宗关于清剿[[朗格多克|朗格多克]]地区的异端的要求充耳不闻。他也拒绝了为英诺森三世发动一场反对[[阿尔比派|阿尔比派]]异端的十字军。1223年7月14日，腓力二世去世于[[芒特|芒特]]，遗体安葬在[[圣但尼教堂|圣但尼教堂]]。

腓力二世是第一个没有在生前就给[[王储|王储]]提前加冕的卡佩王朝国王。因为此时，法蘭西王室的力量已经强大到不需要再用这种方式来确保王位了。

== 在巴黎的建设 ==

在腓力二世统治时期，[[巴黎|巴黎]]市区周围建起了坚固的[[城墙|城墙]]。约在1200年，腓力二世在巴黎中心[[城岛|城岛]]西端建立一座方形[[城堡|城堡]]；这座城堡后来成为[[卢浮宫|卢浮宫]]的前身。同年，腓力二世正式批准[[巴黎大学|巴黎大学]]的成立。于是从那时开始，许多学校被建立起来，包括现代巴黎大学的前身[[索邦大学|索邦大学]]（1257年，时为[[路易九世|路易九世]]统治时期）。
