'''腓尼基字母'''是[[腓尼基|腓尼基人]]用以书写[[腓尼基语|腓尼基语]]（屬北[[闪含语系|闪含语系]]）的一套字母，在公元前1000年出现，由[[原始迦南字母|原始迦南字母]]演化出来，现在的[[希伯来字母|希伯来字母]]、[[阿拉伯字母|阿拉伯字母]]、[[希腊字母|希腊字母]]、[[拉丁字母|拉丁字母]]等，都可追溯至腓尼基字母。腓尼基字母跟希伯来字母、阿拉伯字母一样，都是[[辅音音素文字|辅音音素文字]]（abjad），没有代表[[元音|元音]]的字母或符号，字的读音須由上下文推断。

腓尼基铭文曾在考古遗址中发现，包括一些腓尼基城市及[[地中海|地中海]]周边的殖民地，例如[[比布鲁斯|比布鲁斯]]（今[[黎巴嫩|黎巴嫩]]）和[[迦太基|迦太基]]（今[[突尼斯|突尼斯]]）。

腓尼基字母被认为是当今所有[[字母|字母]]的祖先，起源于[[古埃及|古埃及]]的只有辅音用声旁表示的[[象形文字|象形文字]][[圣书体|圣书体]]。腓尼基人把源于古埃及的辅音声旁，在改造字形后，变成了纯表辅音的字母。

腓尼基字母跟[[乌加里特字母|乌加里特字母]]（[[叙利亚|叙利亚]]沿海的一种[[楔形文字|楔形文字]]演化而来的字母文字）何者先被[[发明|发明]]仍未有定论。

== 腓尼基字母 ==
原本的[[原始西奈字母|原始西奈字母]]是[[象形文字|象形文字]]，然而到腓尼基字母时代，有些字的意思已改变了，例如gimel是抛拐杖的形状。

在下表中：
* “意义”一栏是指该腓尼基字母的来源。
* 由于字母本来是刻在石上，所以多数字母都是直线和方形的，就像古日耳曼字母一样。虽然之后有多一些较曲线的版本，成了罗马时代的北非新迦太基字母。
* 腓尼基语通常由右到左写，而有些文字使用了[[牛耕式轉行書寫法|牛耕式轉行書寫法]]。
* 部分字母有另一种书写表达方法，例如taw可以写成似一个“+”号而不用“×”号，heth可以有两条横线穿过。
* 用括号括住的希腊字母是古字母，有些字体并不支援。
* 拉丁字母[[X|X]]来自希腊字母[[Χ|Χ]]的发音，而不是直接來自[[Ξ|Ξ]]。但是，希腊字母Χ可能来自腓尼基字母samekh的二次变种。

== Unicode编码 ==
腓尼基字母在2006年7月加到[[Unicode|Unicode]] 5.0版之內，範圍在U+10900–U+1091F。除上表的字母外，U+10916–U+10919分別是腓尼基數字1、10、20和100。U+1091F是詞分隔符。之前曾有一个把腓尼基字母当成为希伯来字母的字体变种的提案，已被否决。（参看[http://www.unicode.org/charts/PDF/U10900.pdf PDF]的摘要）

== 由腓尼基字母衍生的字母 ==
[[File:Phönizisch-5Sprachen.svg|thumb]]、[[希臘字母|希臘字母]]、'''腓尼基字母'''、[[希伯來字母|希伯來字母]]及[[阿拉伯字母|阿拉伯字母]]。]]
古希伯来字母是用来书写早期[[希伯来语|希伯来语]]。它几近于腓尼基字母。[[撒马利亚人|撒马利亚人]]用的撒马利亚字母，是古希伯来字母的一个版本。

[[亞蘭字母|阿拉米字母]]（亞拉姆字母）是用来书写[[阿拉米语|阿拉米语]]。阿拉米语是中东地区的[[混合语|混合语]]，在该时被广泛使用。阿拉米语之后变成了数种字母，包括现代希伯来字母、叙利亚字母和纳巴泰字母。纳巴泰字母曲线很多，是阿拉伯字母的来源。

[[希腊字母|希腊字母]]被认为是直接来自腓尼基字母，或与原始迦南语从同一个来源而来。[[希腊语|希腊语]]保留了所有符号，但把其中用不到的腓尼基辅音字母标成了表示希腊语的元音字母。希腊语就像不少[[印欧语系|印欧语系]]语言一样，不像[[闪含语系|闪族语言]]全是依靠辅音。

希腊字母是拉丁字母及[[西里尔字母|西里尔字母]]的来源。而[[如尼字母|古日耳曼字母]]（又称“卢恩字母”）似是来自拉丁字母的早期形式。

很多历史学者相信[[婆罗米文|婆罗米文]]及随後的印度语系文字，同是来自腓尼基字母。腓尼基字母可说是除了[[汉字|汉字]]及相近书写系统外(如[[平假名|平假名]]、[[片假名|片假名]]或[[諺文|諺文]])，现今众多书写系统的起源。
