'''（謹慎的）西班牙的腓力二世'''（{{Lang-es|Felipe II de España}}，又譯'''费利佩'''、'''菲利普'''、'''菲利波'''；{{bd|1527年|5月21日|1598年|9月13日|catIdx=F}}）[[哈布斯堡王朝|哈布斯堡王朝]]的[[西班牙帝國|西班牙]]国王（1556年—1598年在位）和[[葡萄牙帝国|葡萄牙]]国王腓力一世（1581年起在位）。他的执政时期是西班牙历史上最强盛的时代。在腓力二世治下，西班牙帝国的国力达到颠峰，历史学家常以这段时间为[[哈布斯堡王朝|哈布斯堡王朝]]之称霸[[欧洲|欧洲]]。西班牙的腓力二世雄心勃勃，试图维持西班牙帝国，但最终未能成功。

== 哈布斯堡王朝巨大遗产的继承者 ==
{{Main|西班牙帝國|哈布斯堡王朝}}
腓力二世是强大的[[神圣罗马帝国|神圣罗马帝国]]皇帝[[查理五世_(神圣罗马帝国)|查理五世]]（西班牙的卡洛斯一世）的儿子，母为[[葡萄牙的伊莎贝拉|葡萄牙的伊莎贝拉]]。在查理五世皇帝于1556年宣布退位后，腓力二世继承了哈布斯堡帝国除家族起源地[[奥地利|奥地利]]和[[德意志|德意志]]之外的其余所有部分。查理五世的弟弟[[斐迪南一世_(神圣罗马帝国)|斐迪南]]继承了神圣罗马帝国的[[皇帝|皇帝]]称号和有名无实的在德意志的最高地位，而哈布斯堡王朝的军事与经济实力来源—西班牙和[[尼德兰|尼德兰]]—都归于腓力二世。

腓力二世总共继承了下列领地：西班牙、尼德兰、[[西西里|西西里]]与[[那不勒斯|那不勒斯]]、[[弗朗什孔泰|弗朗什孔泰]]、[[米兰|米兰]]及全部西属[[美洲|美洲]]和[[非洲|非洲]]殖民地。

== 在西班牙和尼德兰的统治 ==
{{Main|八十年戰爭}}
腓力二世继续执行其父在西班牙执行的一切强化[[中央集权|中央集权]]的制度。在他统治时期，中央政府真正开始剥夺一些历史上的王国和民族地域（[[阿拉贡|阿拉贡]]，[[卡斯蒂利亚|卡斯蒂利亚]]，[[巴伦西亚|巴伦西亚]]，特别是[[加泰罗尼亚|加泰罗尼亚]]）的独立性。腓力二世扑灭了1590年至1591年阿拉贡为保持其自治地位而发动的暴动。他并且取消一些城市的自治法规，可能是为了更有利于王室获得直接的税收。在腓力二世时代，西班牙的集权程度和官僚体系的臃肿可謂在歐洲中頗高。

腓力二世对低地国家实行毫不妥协的铁腕统治；1567年他任命著名的将领和刽子手[[阿尔瓦-德托梅斯|阿尔瓦]]公爵[[费尔南多·阿尔瓦雷斯·德·托莱多|费尔南多·阿尔瓦雷斯·德·托莱多]](Álvarez de Toledo，1507年－1582年）为尼德兰总督。后者严厉镇压尼德兰人民的一切反抗活动，处死了大约8000人。

腓力二世为坚定而狂热的天主教徒，他的很多政策中都能看到宗教信仰的影子。这大大影响了他在一些问题上的判断，并使政府卷入一系列宗教纷争。特别是由于腓力二世希望铲除各种[[异端|异端]]或异教的决心，[[中世纪|中世纪]]的[[宗教审判|宗教审判]]和[[迫害异端|迫害异端]]在他统治时期达到登峰造极的程度。腓力二世大力支持天主教[[宗教裁判所|宗教裁判所]]，使大批持异端邪说的人（无论是真有其事还是受到陷害）在[[火刑|火刑柱]]上化为灰烬。他的另一项政策则使1568年－1570年被强制迁徙到西班牙内地的[[摩尔人|摩尔人]]受到残酷迫害。从阿拉贡国王[[斐迪南二世_(阿拉贡)|费尔南多二世]]开始的对[[穆斯林|穆斯林]]和[[犹太人|犹太人]]的迫害和驱逐最终使西班牙的种族构成趋于“纯化”。

== 国际舞台上的腓力二世 ==
{{Main|日不落帝國}}
腓力二世执行了积极的对外政策，似乎他有决心也有能力推动一场把整个欧洲统一到一面旗帜下的运动。在即位之初，腓力二世曾因为希望维持他对[[意大利|意大利]]领地的支配和削弱[[教会|教会]]力量而与教皇[[保罗四世|保禄四世]]发生一场不小的冲突。他也继续进行其先辈为之努力的[[意大利战争|意大利战争]]。1557年，西班牙军队在[[圣康坦战役|圣康坦战役]]中击败了[[法国|法国]]军队。腓力二世于1559年和法国国王[[亨利二世_(法兰西)|昂利二世]]签订了对西班牙有利的《[[卡托-康布雷齊和約|卡托-康布雷齊和約]]》，从而结束了绵延半个世纪的意大利战争。

腓力二世领导了对严重威胁欧洲的[[奥斯曼帝国|奥斯曼帝国]]的回击。在他牵头之下，组成了反对土耳其的“神圣同盟”，同盟成员除西班牙外，还包括[[教皇|教皇]]、[[萨伏依|萨伏依]]、[[威尼斯|威尼斯]]和[[热那亚|热那亚]]。1571年，腓力二世的异母弟弟（卡尔五世皇帝的私生子）[[奧地利的唐胡安|奥地利的唐·胡安]]在[[勒班陀战役|勒班陀战役]]中全歼了奥斯曼帝国舰队。这次胜利因其终结了土耳其人在[[地中海|地中海]]上的扩张而永垂史册。

1580年，腓力二世派遣[[费尔南多·阿尔瓦雷斯·德·托莱多，第三代阿尔瓦公爵|阿尔瓦公爵]]率军强行[[葡萄牙王位繼承戰爭|合併葡萄牙]]。

== 对英国和法国的干涉 ==
{{Main|英西戰爭|亨利四世 (法蘭西)}}
1554年，当时还是[[王储|王储]]的腓力与[[英格兰|英格兰]]女王[[玛丽一世_(英格兰)|玛丽一世]]结婚，但其婚后几乎不曾在英格兰住过。腓力二世与玛丽均为天主教徒，他们共同对国内的[[新教徒|新教徒]]实行恐怖政策。然而在玛丽一世死后，同情新教的[[伊丽莎白一世|伊丽莎白一世]]登上英格兰王位。腓力二世曾向伊丽莎白求婚未果。伊丽莎白对新教的明显偏爱促使腓力二世作出入侵英国的决定，他于1588年建立了历史上赫赫有名的[[无敌舰队|无敌舰队]]。

1588年5月，拥有130艘舰只和3万余名战士的无敌舰队在[[麦迪纳-西多尼亚|麦迪纳-西多尼亚]]率领下向英格兰发起远征。在著名的[[英吉利海峡|英吉利海峡]]遭遇战中，这支舰队被机动灵活的英国海军击散。在重組隊形與靠岸補給的途中，又由于劇烈的风暴被一路吹往[[北海_(大西洋)|北海]]，因此在[[苏格兰|苏格兰]]海域折损了大量舰只與人馬。

在无敌舰队对英国的进攻遭受巨大的挫折之后，西班牙大伤元气。尽管其强大实力还未倒塌，英格兰也不可能立刻成为与之平起平坐的国家，但从此西班牙在海上的威力就趋于衰落。

腓力二世的另一项错误决定同样源于他的宗教热情。在法国发生可怕的[[法国宗教战争|宗教战争]]之后，腓力二世催生了法國內部的[[天主教联盟_(法国)|天主教联盟]]，该联盟由[[吉斯家族|吉斯家族]]领导，致力于消灭法国的[[胡格诺派|胡格诺派]]教徒，並從1576年開始越發仰賴腓力的金援及西班牙精兵。1591年形势最混乱的时候，西班牙的军队甚至在一代名將[[亚历桑德罗·法尔内塞|亚历桑德罗·法尔内塞]]的指揮下开进了[[巴黎|巴黎]]。腓力二世希望他的女儿与第四代吉斯公爵[[吉斯公爵_(第四)|夏尔·德·洛林]]结婚，并使后者继承法国王位，但最终被法國上下的愛國反西輿論給否定了；多次擊敗[[賢君亨利|纳瓦拉的昂赫伊]]的[[亚历桑德罗·法尔内塞|亚历桑德罗·法尔内塞]]，也在1592年病逝，西班牙軍頓失支柱。1595年，腓力二世被[[賢君亨利|纳瓦拉的昂赫伊]]（此时已是法國國王[[賢君亨利|昂赫伊四世]]）击败。后来腓力二世终于承认了昂赫伊四世的国王地位。

== 评价 ==
{{review}}
腓力二世统治时期使西班牙国力昌盛，同时在军事方面开支巨大，因此腓力二世不得不多次增税。尽管西班牙帝国有美洲殖民地的强力支撑，但仍留下了巨大的赤字，以至腓力二世在1557年、1575年和1598年3次宣布国家破产。腓力二世严厉的宗教政策则使腓力二世四处树敌。在他去世后，西班牙帝国很快衰落了。
