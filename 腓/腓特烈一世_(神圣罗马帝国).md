'''腓特烈一世'''（[[德语|德语]]：{{lang|de|Friedrich I}}，1122年－1190年6月10日），也被称为'''红胡子腓特烈'''（{{Lang-it|Federico Barbarossa}}）'''，'''或者'''巴巴罗萨'''（{{Lang-de|Barbarossa}}，音译自意大利语的“红胡子”）'''，'''[[霍亨斯陶芬王朝|霍亨斯陶芬王朝]]，[[德国君主列表|罗马人民的国王]]（1152年－1190年在位）和[[神圣罗马帝国|神圣罗马帝国]]皇帝（1155年加冕）。他也是[[德意志王国|德意志]]的[[士瓦本公爵|士瓦本公爵]]（称'''腓特烈三世'''，1147年起）和[[意大利王国_(中世纪)|意大利]]的国王（1154年－1186年）。他的父亲是[[士瓦本公爵|士瓦本公爵]][[腓特烈二世_(士瓦本)|腓特烈二世]]，母亲是巴伐利亚的朱迪丝（属[[韦尔夫家族|韦尔夫家族]]）。按亲属关系，他是國王[[康拉德三世_(神聖羅馬帝國)|康拉德三世]]的侄子。

== 国内政策 ==
腓特烈一世像所有有才干的皇帝一样，试图在德意志境内驾驭桀骜不驯的诸侯。他在1158年颁布采邑法令，要求所有接受采邑者为皇帝服兵役。他的另一个重大举措是把大的诸侯领地分割成多块，因此在1156年将[[奥地利|奥地利]]从[[巴伐利亚|巴伐利亚]]公国分出来，使之成为独立公国。腓特烈一世企图把王室直辖的领地连成一片，并仰赖家臣进行统治。

== 入侵意大利==
腓特烈一世对[[意大利|意大利]]的强烈兴趣使意大利事务成为他生命中最主要的部分。腓特烈一世的綽號是紅鬍子，在義大利語中，紅鬍子的譯音為巴巴羅薩，意思是這位入侵者在義大利殘殺無辜，讓義大利人的血染紅他的鬍子，由此可見意大利人與腓特烈之間的血海深仇。

为征服[[伦巴第|伦巴第]]诸城市，使这些富饶地区成为帝国的税收来源，腓特烈一世长期执行侵略意大利的政策。他对意大利进行过5次军事入侵（1154年、1158年、1163年、1166年和1174年）。为巩固帝国在意大利统治，恢复德意志各公国的秩序。1153年与教皇达成康士坦斯协议，承诺保护教皇国，并帮助教皇夺取罗马的控职权。取得教皇的支持后，腓特烈一世对意大利进行六次远征并于1155年拿下罗马。1158年第二次占领米兰，并在意大利北部地区建立政权。然而这一举动导致他与[[歷山三世|教皇亚历山大三世]]的长期斗争，西西里和[[教皇国|教皇国]]組成[[伦巴第同盟|伦巴第同盟]]。1167年腓特烈一世在图斯库卢姆指战中大败联军，然而一场瘟疫席卷他的军队，导致他不得不班师回朝。1176年，伦巴第联盟（此时威尼斯和拜占庭也加入到联盟中）在[[莱尼亚诺战役|莱尼亚诺战役]]中彻底打败腓特烈一世，迫使他在1183年的[[康斯坦茨和约|康斯坦茨和约]]中，承认地方城市拥有自由选举执行官的权利。不过腓特烈一世的意大利政策也取得一定成效。通过安排儿子亨利与[[西西里王国|西西里王国]]的继承人[[科斯坦察_(西西里女王)|科斯坦察]]结婚，霍亨斯陶芬王朝取得对[[西西里岛|西西里]]的统治权。然而，正是西西里领地成为后来[[霍亨斯陶芬王朝|霍亨斯陶芬王朝]]与教皇残酷争斗的根源，这场争斗最终使[[霍亨斯陶芬王朝|霍亨斯陶芬家族]]被斩尽杀绝。

== 与亨利的斗争 ==
虽然腓特烈一世和他的帝国威名赫赫，但在业已形成诸侯割据的德意志，他的中央集权政策仍然受到强有力的挑战。他的主要敌人和竞争对手是[[萨克森公爵列表|萨克森公爵]][[狮子亨利|狮子亨利]]，亨利並在1156年兼领了巴伐利亚。1174年，由于狮子亨利拒绝参加对意大利的军事行动，腓特烈一世向亨利进攻并将之打败。亨利的大部分土地都被剥夺，只保住了[[不伦瑞克|不伦瑞克]]和[[吕讷堡|吕讷堡]]。但腓特烈一世并未将狮子亨利被没收的领地划入王室领地，而是将它们分给了其他公爵。从此这就在德意志成为一种惯例，即被没收的公爵领地只能短期为皇帝所拥有。

== 十字军东征和去世 ==
1189年，腓特烈一世在与教皇和解后，与英国国王[[理查一世|狮心王理查]]和法国国王[[腓力二世_(法兰西)|腓力二世·奥古斯都]]一起领导了[[第三次十字军东征|第三次十字军东征]]。然而，他毕竟未能到达圣地。在帶領德軍東征的途中，因為決意認為游過河比翻山越嶺更省時間，於是真的跳進了河中，想證明給士兵看，結果在幾分鐘後淹死了。
