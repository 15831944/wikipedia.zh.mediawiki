'''被忽略的热带疾病''' (Neglected Tropical Diseases，简称NTD) 是一类在发展中地区（如[[漠南非洲|漠南非洲]]、[[東南亞|東南亞]]及[[拉丁美洲|拉丁美洲]]）的低收入人群中，常见的由许多种不同疾病组成的热带感染疾病。这些疾病的病原体十分多样，包括[[病毒|病毒]]、[[细菌|细菌]]、[[原生生物|原生生物]]、[[寄生虫|寄生虫]]等。不同组织对于这些疾病的定义不同，在漠南非洲，这些疾病的影响力与三大疾病[[愛滋病|愛滋病]]、[[肺结核|肺结核]]及[[疟疾|疟疾]]相当。<ref>{{Template:Cite journal|author = Hotez PJ, Kamath A|title = Neglected Tropical Diseases in Sub-Saharan Africa: Review of Their Prevalence, Distribution, and Disease Burden|journal = PLoS Negl Trop Dis|volume = 3|issue = 8|pages = e412|year = 2009|doi = 10.1371/journal.pntd.0000412|pmid = 19707588|editor1-last = Cappello|editor1-first = Michael|pmc = 2727001}}</ref> 一些被忽略的热带疾病在发达地区有已知的预防方法和治疗方法，但是这些治疗并没有在贫穷地区實踐。对于很多疾病，治疗并不昂贵，例如一个小孩治療[[血吸虫病|血吸虫病]]的每年费用只需$0.20美元。<ref>{{Template:Cite web|url = http://www.npr.org/templates/story/story.php?storyId=7806977|title = Making the Case to Fight Schistosomiasis|publisher = [[National_Public_Radio|National Public Radio]]|accessdate = 2008-12-01}}</ref> 据估计，在未来的5到7年中控制被忽略的热带疾病需要大概20至30亿美元的资助。<ref>{{Template:Cite journal|author = Hotez PJ|title = How To Cure 1 Billion People? — Defeat Neglected Tropical diseases|journal = [[Scientific_America|Scientific America]]|volume = 302|issue = 1|pages = 90–4, 96|date = January 2010|pmid = 20063641|url = http://www.scientificamerican.com/article/a-plan-to-defeat-neglected-tropical-diseases/|doi = 10.1038/scientificamerican0110-90}}</ref>

这些疾病与三大疾病（愛滋病、肺结核及疟疾）不同，因为后者得到了更多的治疗和研究资助。被忽略的热带疾病令愛滋病和肺结核變得更加致命。<ref>{{Template:Cite web|publisher = SciDev.Net|title = Beat neglected diseases to fight HIV, TB and malaria|author = Mike Shanahan|date = 31 January 2006|url = http://www.scidev.net/News/index.cfm?fuseaction=readNews&itemid=2629&language=1}}</ref> 但是，一些制药公司投身于捐献需要的治疗药品，并且在很多国家大量输入药品（如除蠕虫药品）已经实施。<ref>{{Template:Cite journal|author = Reddy M, Gill SS, Kalkar SR, Wu W, Anderson PJ, Rochon PA|title = Oral drug therapy for multiple neglected tropical diseases: a systematic review|journal = JAMA|volume = 298|issue = 16|pages = 1911–24|date = October 2007|doi = 10.1001/jama.298.16.1911|pmid = 17954542|url = http://jama.jamanetwork.com/article.aspx?doi=10.1001/jama.298.16.1911|ref = {{harvid|Reddy|Gill|2007}}}}</ref>

WHO将17个被忽略的热带疾病放在优先顺序。这些疾病在149个国家非常常见，影响着超过14亿人（包括超过5亿儿童），每年花费发展中经济体几十亿美元。在2013年，被忽略的热带疾病造成了142,000人死亡—相比1990年的204,000人死亡已有减少。<ref name="GBD204">{{Template:Cite journal|last1 = GBD 2013 Mortality and Causes of Death|first1 = Collaborators|title = Global, regional, and national age-sex specific all-cause and cause-specific mortality for 240 causes of death, 1990-2013: a systematic analysis for the Global Burden of Disease Study 2013.|journal = Lancet|date = 17 December 2014|pmid = 25530442|doi = 10.1016/S0140-6736(14)61682-2|volume = 385|issue = 9963|pages = 117–71|pmc = 4340604}}</ref>在这17个优先的被忽略的热带疾病中，有两个是完全根除的目标：龙线虫病（2015年）和雅司病（2020年）；有四个是在2020年前根除的目标：致盲性[[沙眼|沙眼]]、[[非洲人类锥虫病|非洲人类锥虫病]]、[[痲瘋病|痲瘋病]]和[[淋巴丝虫病|淋巴丝虫病]]。<ref name="who.int">http://www.who.int/neglected_diseases/about/en/</ref>

== 疾病类型  ==
在WHO, CDC和一些传染性疾病专家中有一些关于究竟哪些疾病应被分类于被忽略的热带疾病的分歧。 Feasey，一个被忽略的热带疾病的研究者，认为13个被忽略的热带疾病尤为突出: 
* 蛔虫病 (ascariasis)
* 布如里氏溃疡 (Buruli ulcer)
* 美洲锥虫病 (Chagas disease)
* 龙线虫病(dracunculiasis, guinea-worm disease)
* 钩虫传染 (hookworm infection)
* 非洲人类锥虫病 (human African trypanosomiasis)
* 利什曼病 (leishmaniasis)
* 麻风(leprosy)
* 淋巴丝虫病(lymphatic filariasis)
* 蟠尾丝虫症(onchocerciasis)
* 血吸虫病(schistosomiasis)
* 沙眼(trachoma)
* 鞭虫病(trichuriasis)。  <ref name="Feasey">{{Template:Cite journal|author = Feasey N, Wansbrough-Jones M, Mabey DC, Solomon AW|title = Neglected tropical diseases|journal = Br. Med. Bull.|volume = 93|issue = 1|pages = 179–200|year = 2010|pmid = 20007668|doi = 10.1093/bmb/ldp046|url = http://bmb.oxfordjournals.org/content/93/1/179.full}}</ref>

Fenwick 认定了 12个 "核心的" 被忽视热带病: [[麦地那龙线虫病|麦地那龙线虫病]]、 [[非洲人类锥虫病|非洲人类锥虫病]]、[[利什曼病|利什曼病]]、[[麻风病|麻风病]]、[[淋巴丝虫病|淋巴丝虫病]]、[[象皮病|象皮病]]、[[蟠尾絲蟲症|蟠尾絲蟲症]]、[[血吸虫病|血吸虫病]]、[[沙眼|沙眼]]和[[鞭虫病|鞭虫病]]<ref name="Fenwick">{{Template:Cite journal|author = Fenwick A|title = The global burden of neglected tropical diseases|journal = Public Health|volume = 126|issue = 3|pages = 233–6|date = March 2012|pmid = 22325616|doi = 10.1016/j.puhe.2011.11.015|url = http://linkinghub.elsevier.com/retrieve/pii/S0033-3506(11)00367-2}}</ref>。

这些被忽略的热带疾病有四种不同的病原体:
(i) 原生动物(美洲锥虫病 , 非洲人类锥虫病 , 利什曼病) 
(ii) 细菌(布如里氏溃疡, 麻风, 沙眼,雅司病)
(iii)寄生虫或蠕虫(猪囊虫病/绦虫病, 龙线虫病, 包虫病, 淋巴丝虫病, 蟠尾丝虫症, 血吸虫病); (v)病毒(登革热,切昆贡亚热,狂犬病)。

==经济效应==
康斯, 恩格斯, 和莫邻诺(音译) 将被忽视热带病的低成本治疗归因于大规模的项目, 制药公司提供的免费药物, 药物的分发, 和分发药物的无偿志愿者们. 他们同时认为被忽视热带病的经济负担被低估了, 因而被忽视热带病降低的发病率对应的经济影响和成本节约被低估了。控制被忽视热带病的措施的投资回报据估计大概在14–30%, 依据不同疾病和地区有所不同。除蠕虫的长期好处包括了学校缺勤率降低了25%, 成年人的工资收入增加了20%.

一些被忽视热带病，如布鲁里溃疡，的治疗成本可以超过在最低收入地区的家庭的平均年收入的两倍，然而对于最高收入的地区，负担微小于平均年收入。这样巨大的金融花销常常导致治疗推迟和家庭破产, 但是经济负担的贫富差距也有存在。这些疾病也增大了了政府的开销，对于医疗保险，和由于死亡和寿命缩短导致的劳动者效率下降。例如在肯尼亚，据估计除蠕虫将承认平均收入提高了40%，反映了收益:花销＝100的比例。 据估计，每一个没有被治疗的沙眼病例会导致相当于 $118 的劳动力损失。每一个血吸虫病病例会导致每年减少45.4天工作。大部分的被忽视热带病会造成发展中国家几百万美金的经济损失。据预测，大规模的提倡预防的活动会提升农业产出和教育水平。



==社会效应==
一些被忽视热带病，例如麻风，会导致严重的畸形，因而带给患者明显的患病症状。淋巴丝虫病会导致严重的畸形，从而造成悔婚和无法工作。 在加纳和斯里兰卡的研究表明，对于淋巴丝虫患者的支持小组可以通过社会支持和提供关于如何应对他们的疾病的建议增加患者的自信，提高他们的生活质量，改善他们的人际关系。



==预防前景==
芬威客(音译)认为预防和根除非常重要，因为“在被忽视热带病的症状中, 畸形, 失明和残疾最为骇人“。一篇于2013年发布的研究表明，在未来十年中我们有潜能根除或完全根除龙线虫病，麻风，淋巴丝虫病， 盘尾丝虫病，沙眼，昏睡病，内脏利什曼病以及狂犬病。


== 常见NTD  ==
1. 血吸虫病（Schistosomiasis）

一种由在水里生活的寄生虫感染而引起的疾病，常见于热带与亚热带国家；

2. 美洲锥虫病（Chagas disease）

恰加斯病也称南美锥虫病，是原生动物寄生虫克氏锥虫引起的一种可威胁生命的疾病。该病主要见于21个拉丁美洲国家的流行地区，多数通过接触锥蝽臭虫的粪便传播给人类；

3. 蛔虫病（Ascariasis）

蛔虫寄生于人体小肠所引起的常见寄生虫病。儿童感染率高。传染源为肠道蛔虫感染者及病人；

4. 十二指肠钩虫感染（Hookworm infection）

十二指肠钩虫成虫寄生在小肠上段，以口囊内的钩齿咬附肠粘膜，以血液、淋巴液，脱落上皮细胞为食，使患者长期慢性失血而导致贫血；

5. 沙眼（Trachoma）

沙眼是由沙眼衣原体引起的一种慢性传染性结膜角膜炎，因其在睑结膜表面形成粗糙不平的外观，形似沙粒，故名沙眼；

6. 登革热（Dengue fever）

由受染于四种登革热病毒中任何一种病毒的伊蚊叮咬所传播。该病发生在世界上的热带和亚热带地区。被具传染性的蚊虫叮咬之后3-14天出现症状。
