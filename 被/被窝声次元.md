'''被窝声次元'''（又称'''被窝app'''、'''被窝worm'''）是中国大陆一个以[[ACG|ACG]]和声音为主题的[[同人_(ACG)|同人]]创作分享平台，是北京尖果网络科技有限公司旗下的产品，于2014年9月推出。[[有妖氣原創漫畫夢工廠|有妖氣原創漫畫夢工廠]]和[[中国中央电视台新科动漫频道|中国中央电视台新科动漫频道]]的前员工段金玖、[[獨立製作|独立动画]]创作者兼[[导演|导演]]董淳以及技术负责人王辰是被窝声次元的三位联合创始人<ref name="天眼查" />。被窝娘是被窝声次元的[[吉祥物|吉祥物]]，是一个粉红色头发的女性动漫角色<ref name="ZOL" />。

== 运营 ==
被窝声次元的定位是一个[[二次元|二次元]]声音分享交流平台，采用了[[用户生成内容|用户生成内容]]的运营模式，作为创作者的用户可以在被窝声次元上传自己[[翻唱|翻唱]]的歌曲或者是给[[日本漫画|漫画]]配音制作成有声漫画，也可以制作[[广播剧|广播剧]]上传到平台上，或者是进行其他跟声音和ACG有关的创作<ref>{{cite web |author1=卫闫森 |title=被窝APP：二次元声音爱好者的集散地 |url=http://www.lieyunwang.com/archives/64032 |website=猎云网 |accessdate=2018-07-18 |date=2014-12-03 |archive-url=https://web.archive.org/web/20180807040639/http://www.lieyunwang.com/archives/64032 |archive-date=2018-08-07 |dead-url=no }}</ref>。作为观看者的用户可以使用电脑网页版或者[[智能手机|智能手机]]上的[[app|app]]观看被窝声次元平台上的作品并进行评论，也可以使用[[弹幕|弹幕]]进行[[吐槽|吐槽]]。由于这种模式降低了普通用户创作的门槛，作为观看者的用户也可以自己创作内容上传到平台上，成为创作者。这种模式跟[[AcFun|AcFun]]和[[嗶哩嗶哩彈幕網|嗶哩嗶哩彈幕網]]类似，即用户可以上传自己的原创作品或者对他人的ACG作品进行[[演绎作品|演绎]]和[[二次创作|二次创作]]，同时也提供一个用户之间交流的社交平台。<ref>{{cite web |author1=荔闽 |title=被窝，一个好看、动听的二次元声音社区 |url=https://36kr.com/p/218610.html |website=[[36氪|36氪]] |accessdate=2018-07-18 |date=2015-01-16 |archive-url=https://web.archive.org/web/20150511171030/http://36kr.com/p/218610.html |archive-date=2015-05-11 |dead-url=no }}</ref>

除了用户主动投稿之外，被窝声次元曾与一些ACG创作者和创作团队合作开展活动。例如被窝声次元曾与中国大陆漫画创作者使徒者合作，对其创作的漫画《一条狗》进行配音，制作成有声漫画<ref>{{cite web |author1=王新宇 |title=二次元创业：看被窝如何玩转声音+图片轻视频 |url=http://news.newhua.com/2015/0608/298245.shtml |website=牛华网 |accessdate=2018-07-18 |date=2015-06-08 |archive-url=https://web.archive.org/web/20180807040640/http://news.newhua.com/2015/0608/298245.shtml |archive-date=2018-08-07 |dead-url=no }}</ref><ref name="ZOL">{{cite web |author1=席龙飞 |title=每日佳软:二次元懒床必备"被窝声次元" |url=http://4g.zol.com.cn/504/5047174.html |website=[[中关村在线|中关村在线]] |accessdate=2018-07-18 |date=2015-01-28 |archive-url=https://web.archive.org/web/20150704160107/http://4g.zol.com.cn/504/5047174.html |archive-date=2015-07-04 |dead-url=no }}</ref>，也曾跟中国大陆[[美少女游戏|美少女游戏]]同人制作组SP-time合作，通过被窝声次元平台向用户征集游戏《[[真恋～寄语枫秋～|真恋～寄语枫秋～]]》的片头曲{{单双书号转换|永恒的等待}}的演唱版本<ref>{{cite web |author1=飘雪幻幻 |title=SP-time × 被窝 系列活动第二弹！真恋声募计划活动结果公布！ |url=http://sp05.sp-time.com/sm47.html |website=SP-time |accessdate=2018-08-07 |archive-url=https://web.archive.org/web/20180603160238/http://sp05.sp-time.com/sm47.html |archive-date=2018-06-03 |dead-url=no }}</ref>。
