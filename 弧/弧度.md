'''弧度'''又稱'''弳度'''，是[[平面角|平面角]]的單位，也是[[國際單位制導出單位|國際單位制導出單位]]。單位弧度定義為圓弧長度等於[[半徑|半徑]]時的圓心角。角度以弧度給出時，通常不寫弧度單位，或有時記為'''rad'''（㎭）。平面角和立體角皆無[[因次|因次]]。

一個完整的圓的弧度是<math>2\pi</math>，所以<math>2\pi</math>rad = 360°，<math>\pi</math>rad = 180°，1°=<math>\frac{\pi}{180^\circ}</math>rad，1 rad = <math>\frac{180^\circ}{\pi}</math>（約57.29577951°）。以度數表示的角度，把數字乘以<math>\frac{\pi}{180^\circ}</math>便轉換成弧度；以弧度表示的角度，乘以<math>\frac{180^\circ}{\pi}</math>便轉換成度數。

:<math>\frac{\pi}{180}\times deg = rad</math>

同樣地︰

:<math>deg = rad\times\frac{180}{\pi}</math>

[[微積分|微積分]]的[[三角函數|三角函數]]中，角度以弧度為單位，以獲得簡潔的結果。例如以弧度為單位，有如下簡單等式：

: <math>\lim_{h\to0}\frac{\sin h}h=1</math>，

從這等式可以推導出很多漂亮的數學等式。

把三角函數寫成[[泰勒级数|泰勒级数]]，則必須以弧度表示。
:<math>\sin x = \sum^{\infty}_{n=0} \frac{(-1)^n}{(2n+1)!} x^{2n+1} = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots\quad\!</math>
