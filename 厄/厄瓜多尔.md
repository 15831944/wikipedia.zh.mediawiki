'''厄瓜多共和国'''（{{lang-es|República del Ecuador}}）  通稱'''厄瓜多'''，是一個位於[[南美洲|南美洲]]西北部的國家，北與[[哥倫比亞|哥倫比亞]]相鄰，南接[[祕魯|祕魯]]，西濱[[太平洋|太平洋]]，與[[智利|智利]]同為南美洲不與[[巴西|巴西]]相鄰的國家，另轄有距厄瓜多本土1,000公里的[[加拉巴哥群島|加拉巴哥群島]]。

厄瓜多於1809年8月10日時脫離[[西班牙|西班牙]]的统治獨立建國。由於[[赤道|赤道]]橫貫了厄瓜多的國境，所以西班牙文中以赤道（ecuador）作为国名，因此該國又擁有「'''赤道国'''」的別稱，或因為盛產[[香蕉|香蕉]]而又被稱為「'''香蕉之国'''」。首都[[基多|基多]]位於[[皮欽查火山|皮欽查火山]]的山麓，[[海拔|海拔]]高達2,850公尺，使該市成為全世界第二高的[[首都|首都]]（僅次於[[玻利維亞|玻利維亞]]首都[[拉巴斯|拉巴斯]]）。為[[南美洲國家聯盟|南美洲國家聯盟]]的成員國。

== 历史 ==
{{main|厄瓜多尔历史}}
據現今文獻，厄瓜多爾的歷史可追溯至三千五百年前。當時在今日沿海地區已有部落散居。1463年成為[[印加帝國|印加帝國]]一部分。在[[西班牙|西班牙]]人發現新大陸後，厄瓜多爾於1532年被西班牙人佔領，自此淪為[[西班牙|西班牙]]殖民地數百年。

1809年8月10日厄瓜多爾宣佈獨立，但很快被西班牙佔領回。掙扎了十數年，1822年才徹底擺脫西班牙之控制。1825年加入[[大哥倫比亞共和國|大哥倫比亞共和國]]，但此國於1830年解體。而厄瓜多爾共和國也因此成立。

立國以來厄瓜多爾政治一直都上不了軌道，文人與軍人政府輪替了足足19次，至1979年正式由文人政府統治，確立憲法，並沿用了二十多年。

2007年4月15日，举行全民公决<ref>赵晖，[http://news.xinhuanet.com/world/2007-04/15/content_5980111.htm 厄瓜多尔就成立制宪大会举行全民公决]，新华网</ref>，决定成立制宪大会，制订新宪法。<ref>赵晖，[http://news.xinhuanet.com/world/2007-04/16/content_5983996.htm 初步结果显示多数选民支持厄瓜多尔成立制宪大会]，新华网</ref>

== 地理 ==
{{main|厄瓜多地理}}
[[File:Ecuador_cajas_national_park.jpg|thumb]]境內形狀特殊的對稱湖面]]
厄瓜多西境瀕臨[[太平洋|太平洋]]，国土的一半以上为山区，靠近海岸的東部地區为[[平原|平原]]地形，高原盆地相间分布。中部海拔较高，其中海拔6310米的[[钦博拉索山|钦博拉索山]]屬於[[安地斯山脈|安地斯山脈]]的一部份，是座[[死火山|死火山]]。它除了是全国最高点（海拔6,267公尺），也是地球表面上離地心最遠之處（因為自轉的[[離心力|離心力]]，使[[地球|地球]]成為一個南北縱軸較東西橫軸略短的微扁球體，因此钦博拉索山雖然不是地表上最高的山，卻因位置正處赤道上而成就此第一頭銜）。厄瓜多尔境内多火山，有20多座5000米以上的[[火山锥|火山锥]]。
厄瓜多尔总面积为28万3520平方公里，含[[加拉帕戈斯群岛|加拉帕戈斯群岛]]。其中，土地面积27万6841平方公里，水域面积6720平方公里。在南美洲，厄瓜多尔的面积大于[[乌拉圭|乌拉圭]]，[[苏里南|苏里南]]，[[圭亚那|圭亚那]] 和[[法属圭亚那|法属圭亚那]]。
[[File:(lava_tunnel)_El_Chato_Reserve,_Santa_Cruz_Highlands,_Galápagos),_inside_view_pic_i.JPG|thumb]]   El Chato Reserve [[熔岩管|熔岩管]] ]]

== 经济 ==
{{main|厄瓜多經濟}}
厄瓜多爾屬於中等收入的[[發展中國家|發展中國家]]，經濟高度依賴於類似石油，農牧业產品的[[原物料|原物料]]。2000年時該國推行貨幣[[美元|美元]]化政策，雖然導致人民激烈的反抗讓當時的執政者被迫下台，但仍成功地用美元取代掉因為嚴重的[[通貨膨脹|通貨膨脹]]而價值極低的舊貨幣[[厄瓜多尔苏克雷|蘇克雷]]。厄瓜多以[[白花修女兰|白花修女兰]]為國花，以[[安地斯神鷹|安地斯神鷹]]為國鳥。

== 人口 ==

[[File:Escalinatas_de_Las_Peñas.jpg|thumb]]
[[File:Submarino_Shyri.jpg|thumb]]]]
{{main|厄瓜多人口}}
厄瓜多國土面積28万3560平方公里，人口1522万3680人 （2011年数据），由於曾經是西班牙的殖民地，境內人種複雜，其中印欧混血人口（[[麥士蒂索人|麥士蒂索人]]，Mestizos）占总人口的41%，[[印第安人|印第安人]]占34%，[[白种人|白种人]]占15%，除此之外還有被從[[非洲|非洲]]送至南美洲作為勞動人力的[[黑人|黑人]]與黑白混血人口。如同[[拉丁美洲|拉丁美洲]]大部分國家，厄瓜多的居民信奉的是一種融入部分印地安文化色彩的[[天主教|天主教]]，官方語言是西班牙語，但印地安族群間則通用[[克丘亚语|克丘亚语]]。
