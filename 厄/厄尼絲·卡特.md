'''厄尼絲·羅貝塔·亨頓·卡特'''（{{lang-en|'''Eunice Roberta Hunton Carter'''}}，{{bd|1899年|7月16日|1970年|1月25日}}）是一名美國律師。她是[[紐約州|紐約]]首位[[非裔美國人|非洲裔美國]]女律師之一，也是美國首位有色人種的檢察官之一。她積極參與{{link-en|泛非大會|Pan-African Congress}}和[[聯合國|聯合國]]委員會，以提高世界婦女的地位。她還幫助紐約[[地方檢察官|地方檢察官]][[湯瑪斯·杜威|湯瑪斯·杜威]]以[[性交易|強迫賣淫罪]]成功指控[[美國黑手黨|黑手黨]]老大[[查理·盧西安諾|查理·「幸運」·盧西安諾]]。<ref>{{Cite web |url = https://themobmuseum.org/notable_names/eunice-carter/ |title = Eunice Carter |publisher = The Mob Museum |language = en}}</ref><ref>{{Cite web |url = https://themobmuseum.org/blog/eunice-carter-key-player-in-luciano-conviction/ |title = Eunice Carter: Key player in Luciano conviction |author = Jeff Burbank |date = 2016-10-06 |publisher = The Mob Museum |language = en}}</ref>

== 婚姻和家庭 ==
她嫁給了老利斯爾·卡特（Lisle Carter, Sr.），老利斯爾是紐約首位非裔美國牙醫之一。他們在曼哈頓的[[哈林區|哈林區]]生活了很多年。這對夫婦唯一的孩子是{{link-en|利斯爾·C·卡特|Lisle C. Carter|小利斯爾·卡特}}，他畢業於大學和法學院。他從事法律工作，後來在[[約翰·甘迺迪|約翰·甘迺迪]]和[[林登·莊遜|林登·莊遜]]的政府中擔任政治任命官員。
