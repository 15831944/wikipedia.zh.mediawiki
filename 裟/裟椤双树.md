'''裟椤双树'''（{{bd|1983年|12月9日}}），本名'''童玲'''<ref>{{Cite web|url = http://comic.sina.com.cn/n/2011-05-10/0905854.shtml|title = 古风动漫幻想小说《浮生物语》上市|accessdate = 2015-08-03|author = |date = 2011年5月10日|publisher = 新浪动漫|language = zh-cn}}</ref>，生于四川成都，女，中国幻想小说家。因写作系列幻想小说《[[浮生物语|浮生物语]]》而知名于大众。

== 主要作品 ==
*《[[浮生物语|浮生物语]]》系列（连载中）
**《[[浮生物语|浮生物语]]》
**《浮生物语·贰》
**《浮生物语·叁》（上、下册）
**《浮生物语外传·七夜》
**《浮珑·浮生物语前传》
*《三界宅急送》
*《三界宅急送外传之异人学园》
*《降灵家族》
*《雌雄怪盗》
*《与魅共舞》
*《不停·X人事档案》
*《陌上桑》
*其它作品：《白色苹果树》、《预言石》、《半兽24小时》
