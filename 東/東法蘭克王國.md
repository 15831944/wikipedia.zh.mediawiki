'''東法蘭克王國'''（[[德語|德語]]：'''{{lang|de|Ostfrankenreich}}'''）為[[中歐|中歐]]的一個[[君主制|君主制]]國家，存在時間為843年至10世紀。

== 建立 ==
843年，[[法蘭克王國|法蘭克王國]][[查理曼|查理曼]]的三個後人，[[洛泰爾|洛泰爾]]、[[日耳曼人路易|日耳曼人路易]]（{{lang|de|Ludwig II der Deutsche}}）及[[禿頭查理|禿頭查理]]簽署《[[凡爾登條約|凡爾登條約]]》，共同瓜分整個法蘭克王國。當中，日耳曼人路易分得了王國東部的領土，建立起一個稱為「東法蘭克王國」的[[君主制|君主制]]國家。另外，法蘭克王國的剩餘領土則分裂成[[西法蘭克王國|西法蘭克王國]]及[[中法蘭克王國|中法蘭克王國]]。然而，路易所處的東部領土，是一片被征服不久的日耳曼與波西米亞地區，較其他地方落後。

== 發展 ==
870年，路易與西法蘭克國王禿頭查理簽署《[[墨爾森條約|墨爾森條約]]》，瓜分洛泰爾後人所遺留下來的中法蘭克王國領土。這條條約使得東法蘭克王國領土得以擴張，併吞[[洛林|洛林]]成為之後[[神聖羅馬帝國|神聖羅馬帝國]]的雛型。
{{德国历史}}
雖然東法蘭克王國保存了[[加洛林王朝|加洛林王朝]]的統治，但是當地的文化、經濟及政治相對落後。此外，[[薩克森|薩克森]]、[[法蘭克尼亞|法蘭克尼亞]]、[[巴伐利亞|巴伐利亞]]及[[士瓦本|士瓦本]]四大公國相繼崛起，使得加洛林的王權旁落，大權頓失。另外，當時東法蘭克王國的外患也相當嚴重，有來自東邊的[[馬扎爾人|馬扎爾人]]入侵，以及北歐[[諾曼人|諾曼人]]的侵擾，都令東法蘭克王國受到影響，王權不能集中。

899年，國王[[阿努尔夫_(东法兰克)|阿努爾夫]]病逝，其子[[孩童路易|孩童路易]]繼位。由於他年少，因此權力落入美因茲大主教的手上。不久後，路易在911年病死，東法蘭克的加洛林王朝絕嗣。後來，法蘭克尼亞公爵[[康拉德一世_(德意志)|康拉德一世]]（{{lang|de|Konrad I.}}）繼位，完全取代加洛林王朝的統治。數年後，康拉德去世，王位輾轉落入薩克森公爵[[亨利一世_(德意志)|捕鳥者亨利]]的身上，开始[[薩克森王朝|薩克森王朝]]，也称'''非加洛林的東法蘭克王國'''（[[:en:Kingdom_of_Germany|regnum Teutonicum]]）。

== 終結 ==
[[亨利一世|亨利一世]]在936年去世後，其子[[奥托一世_(神聖羅馬帝國)|奥托]]繼位。他力主建立一個得以凌駕於各公國的王權，因此大力鎮壓各[[公國|公國]]的叛亂。962年，[[教宗|教宗]][[約翰十二世|約翰十二世]]為他加冕，成為「'''羅馬人的皇帝'''」，後世的史學家稱奧托為[[神聖羅馬帝國|神聖羅馬帝國]]的建立者。
而王國到12世紀[[法蘭克尼亞王朝|薩利安王朝]]結束後，實質上便已瓦解，1356年[[卢森堡王朝|卢森堡王朝]]的查理四世颁布《[[金璽詔書|金璽詔書]]》，讓東法蘭克王國正式瓦解，空存王號。
