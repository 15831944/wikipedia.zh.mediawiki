'''東普魯士'''（[[德語|德語]]：Ostpreußen；{{lang-lt|Rytų Prūsija或Rytprūsiai}}；{{lang-pl|Prusy Wschodnie}}；[[俄語|俄語]]：Восточная Пруссия）是[[普魯士|普魯士王國]]及后来[[德意志帝国|德意志帝国]]的一个省，位于[[普魯士公国|普魯士公国]]的领地上。今天东普鲁士的北部，分别属于[[立陶宛|立陶宛]]的[[克萊佩達|默麥爾]]地区，及[[俄罗斯|俄罗斯]]的[[加里宁格勒|加里寧格勒州]]（舊稱[[哥尼斯堡|哥尼斯堡]]）。而南部大部份被納入波兰的[[瓦爾米亞-馬祖里省|瓦爾米亞-馬祖里省]]。东普鲁士包含了古[[普鲁士人|普鲁士人]]在[[波罗的海|波罗的海]]的领地。在历史上被普遍认为是[[德国|德国]][[军国主义|军国主义]]势力的支柱—[[容克|容克]][[贵族|贵族]]的发源地。

东普鲁士位于波罗的海的东南海岸，首府是[[哥尼斯堡|哥尼斯堡]]。第二次世界大战后，[[苏联|苏联]]占领东普鲁士北部并将其改名为[[加里宁格勒州|加里宁格勒州]]，而东普鲁士南部则划归[[波兰|波兰]]，当地的德意志居民大多被遣送回[[东德|东德]]或被流放[[西伯利亚|西伯利亚]]。

== 从騎士團统治到藩属 ==
於13世紀時，東普魯士本是信奉[[多神教|多神教]]的古普魯士人聚居地。於[[十字軍東征|十字軍東征]]的宗教狂熱浪潮下，諸如[[教廷|教廷]]、[[波蘭|波蘭]]及[[神聖羅馬帝國|神聖羅馬帝國]]等天主教勢力紛紛派遣[[傳教士|傳教士]]，意圖基督教化此鄰近的異教橋頭堡，但對彪悍的普魯士人作用不大，反而令其更常受到普魯士人的襲擾掠奪。於是，於1226年，[[條頓騎士團|條頓騎士團]]應波蘭公爵康拉德一世之邀、得教廷及神聖羅馬帝國御準，發動[[北方十字軍|北方十字軍]]征服普魯士並建立[[條頓騎士團國|條頓騎士團國]]。此後，騎士團的宗教戰爭不斷擴大，無遠弗繼，更將目標放至波蘭的友邦[[立陶宛大公國|立陶宛大公國]]，導致與波蘭齟齬日深，最終爆發[[波蘭-立陶宛-條頓戰爭|波蘭-立陶宛-條頓戰爭]]。於戰事末段，騎士團於[[格倫瓦爾德之戰|坦能堡大敗]]，雖然戰爭最後以雙方和約作結，但騎士團自此已由盛極轉衰，國運江河日下，無力再發動大規模戰爭。此後的一個世紀內，條頓騎士團國飽受內亂、強鄰波蘭-立陶宛擴張、甚至[[宗教改革|新教思潮]]的影響，傳至末代團長[[阿尔布雷希特_(普鲁士)|阿尔布雷希特·霍亨索伦]]時，騎士團已無足夠軍力維持團國內的軍事統治。於是於1525年，阿尔布雷希特毅然將條頓騎士團國世俗化、改宗[[新教|新教]][[路德宗|路德宗]]、並向強鄰[[波蘭立陶宛聯邦|波蘭立陶宛聯邦]]俯首稱臣，是為[[普鲁士公国|普鲁士公国]]。

1618年，普魯士大公阿爾伯-腓特烈逝世，其兩子均早夭，而爵位亦不可由女性繼承，故此傳位於長女婿[[勃兰登堡选帝侯|勃兰登堡选帝侯]]約翰-西吉斯蒙，史称[[勃兰登堡-普鲁士|勃兰登堡-普鲁士]]。此後由於普魯士大公同時以共主邦聯身份擁有[[勃兰登堡|勃兰登堡]][[選帝侯|選侯國]]（其隸屬神聖羅馬帝國而非波蘭）的領地，國土面積大幅增加之餘亦開始擁有有半壁自主的江山，從而得以開始擺脫波蘭的控制繁榮發展。

於1657年，波蘭-立陶宛泥足深陷於[[第二次北方戰爭|第二次北方戰爭]]戰敗、[[瑞典帝國|瑞典帝國]]入侵後的[[大洪水時代|大洪水]]泥沼中，國運開始衰亡，已無力再制約普魯士公國。普魯士趁機與波蘭簽訂[[布隆堡條約|布隆堡條約]]取得完全獨立地位，終於得以擺脫波蘭近半個世紀以來的束縛，正式走上強國崛起之路。

普魯士末代大公[[腓特烈一世_(普魯士)|腓特烈三世]]一直雄心勃勃立志於稱王。其任內勵精圖治，富國強兵，國力日益強盛；並積極參與神聖羅馬帝國諸邦事務，於1686年加入[[奧格斯堡同盟|奧格斯堡反法同盟]]，並於2年後參與[[大同盟戰爭|大同盟戰爭]]，與帝國諸邦共同攜手對抗迅速冒起的法國太陽王[[路易十四|路易十四]]。1701年，腓特烈大公因於[[西班牙王位繼承戰爭|西班牙王位繼承戰爭]]中覲皇有功，終得[[神聖羅馬帝國皇帝|皇帝]][[利奧波德一世|利奧波德一世]]賜封王號，得以成功登基稱王，即位稱腓特烈一世，正式建立[[普魯士王國|普魯士王國]]。

在1772年[[第二次瓜分波兰|第二次瓜分波兰]]后，普魯士王國重奪公國立國時割讓與波蘭的失土—「[[皇家普魯士|皇家普魯士]]」。1773年1月31日，[[腓特烈大帝|腓特烈大帝]]把瓜分得来的土地重新命名为[[西普鲁士|西普鲁士]] (Westpreußen)，前普鲁士公国地區成为'''东普鲁士''' (Ostpreußen)。

== 德意志帝国 ==
东普鲁士作為普魯士王國的領地，在1871年被一同并入[[德意志帝国|德意志帝国]]。在1875年，在该地的人口中73.48%為德国人，18.39%是波兰人和8.11%為立陶宛人。1900年，总人口是1,996,626人，包括1,698,465名基督教徒、269,196名天主教徒及13,877名犹太人。波兰人（Masuren）及立陶宛人（Litauer）的人数，因为德国的德国化政策而减少。波兰居民集中在南部，立陶宛居民则在东北部。普鲁士人一直德国化，所以在18世纪时，古普鲁士语已变成[[死語|死語]]。

非德裔人口是指非德国公民，并不包括非德裔的公民。根据德国法律，人口有居民（Einwohner）与公民（Bürger）之分。

1885年至1890年之间，[[柏林|柏林]]的人口增长是20%，[[布蘭登堡|布蘭登堡]]和[[莱茵兰|莱茵兰]]是8.5%，[[威斯特法倫|威斯特法倫]]是10%；东普鲁士减少0.07%而[[西普魯士|西普魯士]]减少0.86%。

== 魏玛共和時期 ==

1918年，德皇[[威廉二世_(德國)|威廉二世]]退位，德国变成[[共和国|共和国]]。由1919年至[[第二次世界大战|第二次世界大战]]爆发为止，东普鲁士与部分[[西普鲁士|西普鲁士]]是德国的[[外飛地|外飛地]]。基于[[凡尔赛条约|凡尔赛条约]]，德国割让大部分的西普鲁士以及[[波森省|波森省]]予[[波兰|波兰]]，亦即[[波兰走廊|波兰走廊]]及[[但泽自由市|但泽自由市]]。

1920年，西普鲁士西面和东普鲁士南部都举行[[公投|公投]]，让居民决定是否把土地归于波兰还是德国。96.7 %的投票者支持土地归于德国。1923年，在没有举行公投之下，[[立陶宛|立陶宛]]强占[[梅梅尔|梅梅尔]]地区（亦即[[克萊佩達區|克萊佩達區]]）。

== 纳粹时期 ==

纳粹党将三分之一的地名改掉，又在1938年扫除一切由波兰或立陶宛色彩的名字。不合作的少数族裔活跃分子就被送到[[集中营|集中营]]。

根据[[苏德互不侵犯条约|苏德互不侵犯条约]]，[[苏联|苏联]]容许德国占领波兰西部以及[[克莱佩达|克莱佩达]]。

== 第二次世界大战 ==
[[File:East_Prussia_1939.JPG|thumb]]

战争期间，德军征服波兰西部，令东普鲁士扩大。[[1939年德國人口普查|1939年普查]]，此区人口共249万人。虽然纳粹党强烈宣传民族主义，宣称他们征服的地区都住着大量德国居民、都想回归德国，但事实恰恰相反：根据国家的统计，1939年时德军占领区的994,092居民当中，只有31,000人是德国人。很多居民在战争中死亡，其中大部分青年被征召到军队、战死沙场。

1945年初，有部分德国人在战后想要返回东普鲁士，但全部都被[[苏联|苏联]]军队驱逐，也有很多人被发配到苏联当苦工。当地所有德式地方名字都改成俄式或波兰式。

1946年4月，东普鲁士的北部被并入苏联。7月，首府[[哥尼斯堡|哥尼斯堡]]被改名为[[加里寧格勒|加里寧格勒]]，州名是[[加里寧格勒州|加里寧格勒州]]。1947年，苏军驱逐当地所有德意志人，让俄罗斯人、白俄罗斯人及乌克兰人迁到此地。

在領土重劃分中[[寇松線|寇松線]]以东大片波兰领土被劃入苏联，该地的波兰人被驱逐到波兰占领的东普鲁士南部，亦即[[瓦爾米亞-馬祖里省|瓦爾米亞-馬祖里省]]。

1947年，无论作为[[普鲁士自由州|州领地]]或[[普鲁士邦|邦国]]，[[普鲁士|普鲁士]]被完全废除。
