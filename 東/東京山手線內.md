'''东京山手线内'''（{{jpn|j=東京山手線内|hg=とうきょうやまのてせんない|rm=Tōkyō Yamanote sennai}}），是根据[[日本铁路公司|日本铁路公司（JR）]]的旅客营业规则第78条第1项第1号的规定划分的区间。

指的是作为运行组织的[[山手线|山手线]]，以及其内侧的[[中央本线|中央本线]][[神田站_(東京都)|神田站]]至[[代代木站|代代木站]]区间，[[总武本线|总武本线]][[秋叶原站|秋叶原站]]至[[御茶之水站|御茶水站]]区间的区域。区域内的车站的指示牌上有“山”（[[File:JR_area_YAMA.png|16px]]）字的记号。

== 关于运营距离，票价计算里程的起止车站的特例 ==
在'''东京山手线内'''的车站与距离中心车站（东京站）100至200公里区间内的车站间乘车时的票价，将根据将东京站作为起点或终点的情况下的运营距离进行计算<ref>旅客营业规则第87条本文</ref><ref>{{lang|ja|本特例は[[特定都区市内|特定都区市内]]の派生規定である。ただし、特定都区市内に関する規定のうち本特例にも適用されるものは「特定都区市内または東京山手線内発着の場合は〜」や「〜は東京山手線内発着の場合に（特定都区市内に関する規定を）準用する」と規定されている。そのため、特定都区市内のみを対象に規定されていて東京山手線内に関する準用規定がない事項については、本特例に対しては適用されない}}。</ref>。

* 在东京近郊区间内，从东京站出发使用非最短路线乘车且运营距离超过了100公里，但最短路线的运营距离未超过100公里的情况下，可以不使用此特例计算票价<ref>2009年3月14日以降の旅客営業取扱基準規程第115条第2項（ただし、2008年3月31日まで存在していた同条項とは内容が異なる）。</ref><ref>例えば、東京近郊区間内である秋葉原駅から浪花駅までの運賃について、秋葉原⇒浪花の最短経路となる総武本線、外房線経由（98.3km）では東京駅からの営業キロが100km超（100.3km）のため、本来は本特例が適用され「東京山手線内⇒浪花（東北本線、総武本線、外房線経由）」の1890円となる。ただし、東京駅から浪花駅までの最短経路の営業キロが100km以下（99.7km）のため、「秋葉原（単駅）⇒浪花（総武本線、外房線経由）」の1620円とすることができる。</ref>。
* 适用于本特例的车票可以在东京山手线内任意一站乘车及下车，但是不能在中途下车<ref>旅客营业规则第156条第3号</ref>。
* 与东京站间的运营距离超过200公里的情况下，将参照比本特例范围更广的东京都区内的规定进行计算。<ref>[http://www.jreast.co.jp/ryokaku/ 旅客营业规则]第86条第1号</ref>
* 如果使用适用于本条例的车票从东京山手线内区域内的车站乘坐到区域外的车站，将计算下车车站到最近的东京山手线内车站间的距离并收取相应车费<ref>旅客営業取扱基準規程第275条第1号。ただし、元の乗車券が東京近郊区間内相互発着の普通乗車券で区間変更後も東京近郊区間内相互発着となる場合を除く。</ref>。

== 关于票价的特例 ==
在山手线内的的车站之间乘坐的场合，根据乘坐的区间来进行相对便宜的票价计算<ref>根据[http://www.jreast.co.jp/ryokaku/ 旅客营业规则]第78条第1项第1号</ref>。详细的请参考{{link-ja|电车特定区间|電車特定区間}}。

== 统一票价的车票 ==
在2000年1月31号之前发售过在山手线内能在一定次数内使用的[[回数券|回数券]]<ref>{{lang|ja|旅客営業規則第38条の2。詳細は、[[定期乘車券#特殊均一定期乘車券|東京山手線内均一定期券]]を参照のこと}}。</ref>，在之后发售的是在山手线内一定时间内能使用的[[定期票|定期票]]<ref>{{lang|ja|旅客営業規則第41条、ただし現在は削除されている。詳細は、[[回数乘車券|回数乘車券]]を参照のこと}}。</ref>。

== 历史 ==
虽然从1925年（大正14年）开始成为东京电车环状线<ref>{{lang|ja|内部用語の「東京電車環状線」の略で通称でもある}}。</ref>，但是真正叫做“山手线”是从1972年（昭和47年）[[池袋站|池袋站]]到[[赤羽站|赤羽站]]的区间剥离之后<ref>{{lang|ja|元々は、こちらも山手線だった}}。</ref>并入[[埼京线|埼京线]]开始的。
