'''東京都會電視台'''（{{lang|ja|東京メトロポリタンテレビジョン株式会社}}，Tokyo Metropolitan Television Broadcasting Corp.）是[[日本|日本]]一家以[[東京都|東京都]]為播送範圍的[[電視台|電視台]]，簡稱'''TOKYO MX'''，又稱'''MXTV'''。[[无线电台呼号|識別訊號]]為JOMX-DTV。東京都會電視台設立於1993年4月30日，是東京都的第六家民營無線電視台。於1995年11月1日正式開始播出，並於2003年12月1日開始播出數位電視。東京都會電視台是[[全國獨立放送協議會|全國獨立放送協議會]]的成員，並未加入[[電視聯播網|電視聯播網]]。為強調東京都的電視台色彩，和東京的其他民營電視台相比，東京都會電視台的特定報社影響力較弱。但現在[[東京新聞|東京新聞]]的發行者[[中日新聞社|中日新聞社]]持有東京都會電視台部分股份，並有在東京都會電視台播出廣告。

東京都會電視台的簡稱「MX」取自於識別訊號「JO'''MX'''-TV」。在公司成立之初，東京都會電視台曾使用「'''T'''okyo '''M'''etropolitan '''T'''elevision」的首字母「TMT」作為簡稱，但開播時已改為「MXTV」或「MX電視台」（{{lang|ja|MXテレビ}}）。東京都會電視台在2006年7月搬遷至現在的[[千代田區|千代田區]][[麹町|麹町]]半藏門媒體中心之後，開始改用「TOKYO MX」作為簡稱並持續至今。在搬遷的同時，東京都會電視台還開始啟用[[村上隆|村上隆]]設計的吉祥物{{link-ja|Yume Lion|ゆめらいおん}}<ref>{{cite web |url=http://s.mxtv.jp/yumelion/ |script-title=ja:ゆめらいおんのページ |accessdate=2017-07-17|publisher={{lang|ja|東京メトロポリタンテレビジョン}} |language=ja}}</ref>。2009年5月之後，為將電視數位化之後定頻'''9'''頻道一事廣而告之，東京都會電視台使用「{{lang|ja|キュ〜ト!9ch}}（Cute!9ch）」（日語中「9」和「cute」的「cu」發音相同）等宣傳標語<ref>{{cite web |url=http://www.mxtv.co.jp/company/press/090430_cute9ch.pdf |script-title=ja:“地デジ９ｃｈキャンペーン”について |accessdate=2017-07-17|publisher={{lang|ja|東京メトロポリタンテレビジョン}} |language=ja}}</ref>。

== 沿革 ==
雖然日本的媒體業中心是東京都，但長期以來並無以東京都為播出地區的電視台，前[[東京都知事|東京都知事]][[鈴木俊一_(東京都知事)|鈴木俊一]]和{{link-ja|東京商工會議所|東京商工会議所}}因此有意設立以主要服務東京都的電視台。1991年，[[郵政省|郵政省]]決定發出東京地區的電視台執照；同年3月，東京都會電視台的發起人開始申請電視台執照<ref name="history">{{cite web |url=http://s.mxtv.jp/company/history.php |script-title=ja:会社沿革 |accessdate=2017-07-17|publisher={{lang|ja|東京メトロポリタンテレビジョン}} |language=ja}}</ref>。1993年2月1日，東京都會電視台獲得電視播出的預備執照；並於4月30日正式設立「東京都會電視台株式會社」，設立之初的簡稱是「TMT」（{{lang|en|'''T'''okyo '''M'''etropolitan '''T'''elevision}}），當時總部位於[[虎之門|虎之門]]<ref name="reader"></ref>。1995年10月，東京都會電視台正式獲得電視播出執照，總部搬遷至[[江東區_(東京都)|江東區]][[青海_(江東區)|青海]]2丁目{{link-ja|電信中心大樓|テレコムセンター}}<ref name="reader"></ref>。10月15日，東京都會電視台開始試播，最初播出的節目有「東京NEWS」、「MX電視台收視指南」、「莫札特的東京」等。11月1日，東京都會電視台正式開播，首個節目是「東京NEWS」<ref name="history"></ref>。東京都會電視台在播出之初，為和其他電視台進行[[市場區隔|市場區隔]]，而以播出新聞節目為主，並未播出任何綜藝節目。當時電視是利潤率極高的業界，因此有眾多企業出資東京都會電視台。但東京都會電視台開播之後因使用UHF波段導致眾多家庭無法收看，因此經營陷入苦戰，公司內部陷入混亂，眾多高層辭職<ref>{{cite web |url=http://www.maroon.dti.ne.jp/mamos/tv/mxtv2.html |script-title=ja:蝕まれてゆくMXテレビ|accessdate=2017-07-17|publisher={{lang|ja|坂本衛}} |language=ja}}</ref>。

1997年6月，[[FM東京|FM東京]]前社長{{link-ja|後藤亘|後藤亘}}就任東京都會電視台社長，並對經營體制和節目編排進行大幅調整，開始充實[[電視購物|電視購物]]及各類娛樂節目。2002年度之後，東京都會電視台已實現扭虧為盈。2003年12月1日，東京都會電視台開始播出數位電視<ref name="history"></ref>。2006年，東京都會電視台總部搬遷至千代田区麹町半藏門媒體中心<ref name="history"></ref>。2012年10月1日，東京都會電視台改自[[東京天空樹|東京天空樹]]發射信號<ref>{{cite web |url=http://www.mxtv.co.jp/company/press/20120927_skytree_housou.pdf |script-title=ja:東京スカイツリーからの試験電波発射について |accessdate=2017-07-17|publisher={{lang|ja|東京メトロポリタンテレビジョン}} |language=ja}}</ref>。2015年，東京都會電視台與綜合情報公司{{link-ja|里庫路特|リクルート}}推出了在網路同步播出東京都會電視台節目的免費[[行動應用程式|行動應用程式]]{{link-ja|M-CAS|エムキャス}}<ref>{{cite web |url=http://s.mxtv.jp/company/press/pdf/press2015_230001.pdf |script-title=ja:番組の同時配信アプリ「エムキャス」実証実験スタート! |accessdate=2017-07-17|publisher={{lang|ja|東京メトロポリタンテレビジョン}} |language=ja}}</ref>；同年東京都會電視台迎來開播20週年，並舉辦「{{lang|ja|9レイジーだぜ。}}」活動，在11月1日播出了開播20週年特別節目<ref>{{cite web |url=http://s.mxtv.jp/company/press/pdf/press2015_380001.pdf |script-title=ja:9月9日より「９レイジーだぜ。」キャンペーンスタート |accessdate=2017-07-17|publisher={{lang|ja|東京メトロポリタンテレビジョン}} |language=ja}}</ref>。

=== 體育 ===
東京都會電視台在開播之初播出[[千葉縣|千葉縣]][[柏太陽王|柏太陽王]]的比賽，但自1999年開始播出東京當地球隊[[FC東京|FC東京]]球隊[[日本職業足球聯賽|日本職業足球聯賽]]比賽。東京都會電視台自開播之初就播出[[大井競馬場|大井競馬場]]的賽馬實況。和其他獨立電視台一樣，東京都會電視台在每年高中棒球預選球季也會播出當地高中的比賽。1998年開始，東京都會電視台亦播出[[日本職棒|日本職棒]]比賽，最初播出[[東京養樂多燕子|東京養樂多燕子]]及[[阪神虎|阪神虎]]的比賽。但自2007年開始，由於[[軟銀|軟銀]]的贊助，東京都會電視台開始大量播出[[福岡軟銀鷹|福岡軟銀鷹]]的比賽。現在東京都會電視台僅直播福岡軟銀鷹的職棒比賽<ref>{{cite web |url=http://s.mxtv.jp/sports/hawks/ |script-title=ja:2017年7月放送スケジュール |accessdate=2017-07-18|publisher={{lang|ja|東京メトロポリタンテレビジョン}} |language=ja}}</ref>。
=== 動畫節目 ===
和其他獨立電視台相比，東京都會電視台在開播之初播出的動畫節目較少。但在2006年數位電視開始普及之後，[[日本電視台|日本電視台]]出身的製作人（現為顧問）提案「播出動畫的話可獲得收視率」<ref name="otapol">{{cite web |url=http://otapol.jp/2015/12/post-5150.html |script-title=ja:今年も来年もアニメはMX！今年もアニメ攻勢がすごいTOKYO MXの年末年始特番、そして2016年について聞いてみた!! |accessdate=2017-07-18|publisher={{lang|ja|おたぽる}} |language=ja}}</ref>，此後東京都會電視台開始逐漸增加動畫節目播出時間。2012年10日之後，隨著東京都會電視台改用東京天空樹為信號發射地點使得收視範圍擴大，有不少UHF動畫節目在關東地方只在東京都會電視台播出。現在東京都會電視台是日本少數在晚間黃金時段（19點至22點）播出動畫節目的電視台之一。2014年，東京都會電視台設立動畫事業部<ref>{{cite web |url=http://www.oricon.co.jp/news/2037774/full/ |script-title=ja:「アニメを見るならMX」 TOKYO MXの挑戦 |accessdate=2017-07-18|publisher={{lang|ja|オリコン}} |language=ja}}</ref>，並表示希望能舉辦「TOKYO MX動畫祭」，繼續擴大動畫事業<ref>{{cite web |url=https://ddnavi.com/anime-bu/196498/ |script-title=ja:在京アニメファンを支えるTOKYO MXに､気になること全部聞いてみた! |accessdate=2017-07-18|publisher={{lang|ja|ダ・ヴィンチニュース}} |language=ja}}</ref>。

== 電視數位化及多頻道化 ==
2006年7月，為對應電視數位化及提高前往東京都內各地便利性，東京都會電視台總部搬離電信中心大樓，並開始播出高畫質電視和手機電視。2006年7月開始，東京都會電視台成為東京的電視台中除了NHK以外的首個進行常規多頻道播出的電視台<ref name="channel">{{cite web |url=http://s.mxtv.jp/company/press/20140206_hdsd_allmulti.pdf |script-title=ja:2014 年 4 月スタート決定！ＨＤ（091ｃｈ）ＳＤ（092ｃｈ）による２４時間マルチチャンネル編成をスタート |accessdate=2017-07-18|publisher={{lang|ja|東京メトロポリタンテレビジョン}} |language=ja}}</ref>。最初東京都會電視台主要在直播賽馬時進行多頻道播出（一個頻道播出通常節目，另一頻道播出賽馬），之後多頻道播出時間逐漸擴大。2014年4月1日開始，東京都會電視台正式實現全部時間雙頻道播出，並採用新壓縮技術，使得雙頻道中的91頻道（TOKYO MX 1）實現常時高畫質播出，而92頻道（TOKYO MX 2）則使用標準畫質播出<ref name="channel"></ref>。

== 收視地區 ==
東京都會電視台雖然以東京都為播出範圍，但實際上[[神奈川縣|神奈川縣]][[川崎市|川崎市]]幾乎全市、[[横濱市|横濱市]]大部分地區、[[相模原市|相模原市]]全市、[[座間市|座間市]]全市、[[大和市|大和市]]全市、[[横須賀市|横須賀市]][[東京灣|東京灣]]沿岸、[[千葉縣|千葉縣]]西北部及東京灣沿岸、[[埼玉縣|埼玉縣]]中部西部東部、[[茨城縣|茨城縣]]南部部分地區也可以收看到東京都會電視台<ref>{{cite web |url=http://s.mxtv.jp/jyushin/antenna.php |script-title=ja:アンテナで直接受信し視聴する |accessdate=2017-07-18|publisher={{lang|ja|東京メトロポリタンテレビジョン}} |language=ja}}</ref>。然而另一方面，東京都中心也有部分地區（[[葛飾區|葛飾區]]、[[東京國際機場|羽田機場]]等地）較難收看東京都會電視台，為此東京都會電視台在2008年和[[東武鐵道|東武鐵道]]簽約，預約使用[[東京晴空塔|東京晴空塔]]發射信號<ref>{{cite web |url=http://av.watch.impress.co.jp/docs/20081204/tower.htm |script-title=ja:TOKYO MXテレビが新タワーの利用予約契約を締結 |accessdate=2017-07-18|publisher={{lang|ja|AV Watch}} |language=ja}}</ref>。然而因這將使得東京都會電視台的信號發射範圍大幅越過東京都，引發[[神奈川電視台|tvk]]等其他關東地方獨立電視台的不滿，也使得關東地方統一各獨立電視台的遙控器頻道計劃中只有東京都會電視台被改為9頻道。關東地方以外的[[山梨縣|山梨縣]]、[[靜岡縣|靜岡縣]]等地也可以通過有線電視收看東京都會電視台<ref>{{cite web |url=http://s.mxtv.jp/jyushin/catv.php |script-title=ja:CATVでの受信について |accessdate=2017-07-18|publisher={{lang|ja|東京メトロポリタンテレビジョン}} |language=ja}}</ref>。

== 網際網路 ==
東京都會電視台自開播時就開始積極使用互聯網，自2006年8月開始就在[[YouTube|YouTube]]上傳部分自公司製作的節目，是日本的電視台中在視頻網站免費實時播出自公司節目的先驅。2007年，東京都會電視台在youtube開設了自公司的頻道。2009年，東京都會電視台及主要節目還開始了官方Twitter賬戶。2015年7月開始，東京都會電視台推出經由網路實時播出自公司電視節目的app{{lang|ja|エムキャス}}，使得播出區域以外地區的民眾也能收看東京都會電視台的節目，但播出節目僅限部分動畫節目和信息類節目<ref>{{cite web |url=http://av.watch.impress.co.jp/docs/news/709688.html |script-title=ja:TOKYO MXのアニメなどを全国同時に無料スマホ配信。「エムキャス」実証実験 |accessdate=2017-07-18|publisher={{lang|ja|AV Watch}} |language=ja}}</ref>。
