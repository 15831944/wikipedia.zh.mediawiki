'''東川口站'''（{{jpn|j=東川口駅|hg=ひがしかわぐちえき|rm=Higashi-Kawaguchi eki|f=y}}），是一個位於[[埼玉縣|埼玉縣]][[川口市|川口市]]，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）<ref name="JR">{{Cite web |url=http://www.jreast.co.jp/estation/station/info.aspx?StationCd=1277 |title=JR東日本：各駅情報(東川口駅) |publisher=東日本旅客鉄道 |accessdate=2015-01-01}}</ref>、[[埼玉高速鐵道|埼玉高速鐵道]]<ref name="SR">{{Cite web |url=http://www.s-rail.co.jp/line/higashikawaguchi.php |title=時刻表・運賃｜東川口 |publisher=埼玉高速鉄道 |accessdate=2015-01-02}}</ref>的[[鐵路車站|鐵路車站]]。JR東日本的[[車站編號|車站編號]]是「'''JM 23'''」、埼玉高速鐵道是「'''SR 25'''」。

此站有JR東日本的[[武藏野線|武藏野線]]與埼玉高速鐵道的[[埼玉高速鐵道線|埼玉球場線]]2條路線停靠，是互相轉乘站<ref name="JR"/>。

== 使用情況 ==
* '''JR東日本''' - 2016年度的一日平均上車人次為'''34,954人'''<ref group="利用客数">[http://www.jreast.co.jp/passenger/2016_01.html JR東日本 各駅の乗車人員（2016年度）]</ref>，在武藏野線內25個車站當中排行第8位。使用人數有增加傾向，2012年度突破30,000人。
* '''埼玉高速鐵道''' - 2016年度的一日平均上車人次為'''15,225人'''<ref group="利用客数">[http://www.s-rail.co.jp/about/corporate/financial.html 埼玉高速鉄道 経営状況・輸送状況]</ref>。此站是埼玉高速鐵道線的中途站當中上車人次最多的，該公司的車站僅次於[[赤羽岩淵站|赤羽岩淵站]]排行第2位。
