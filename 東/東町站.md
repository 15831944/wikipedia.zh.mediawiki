'''東町站'''（{{jpn|j=東町駅|hg=ひがしちょうえき|rm=HIgashichō eki|f=y}}）位於[[北海道|北海道]][[浦河郡|浦河郡]][[浦河町|浦河町]]東町牛尾1丁目，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|ヒチ}}'''。

== 車站構造 ==
* [[側式月台|側式月台]]1面1線的地面站。
* 僅設有候車室，沒有站房。

== 車站周邊 ==
* [[國道236號|國道236號]]
* 浦河東町簡易郵局
* 北海道浦河高等學校
* 浦河紅十字醫院
* 道南巴士浦河總站
* JR北海道巴士「東町」站牌

== 歷史 ==
* 1977年9月1日 - 設立'''東町臨時車站'''。只處理旅客業務。
* 1987年4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。同時升級為車站。
