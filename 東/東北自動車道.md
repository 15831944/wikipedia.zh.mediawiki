'''東北自動車道'''（{{jpn|j=東北自動車道|hg=とうほくじどうしゃどう|rm=Tōhoku jidōshadō|f=y}}）是起於[[日本|日本]][[埼玉縣|埼玉縣]][[川口市|川口市]]{{link-ja|川口系統交流道|川口ジャンクション}}，經[[群馬縣|群馬縣]]、[[栃木縣|栃木縣]]、[[福島縣|福島縣]]、[[宮城縣|宮城縣]]、[[岩手縣|岩手縣]]、[[秋田縣|秋田縣]]至[[青森縣|青森縣]][[青森市|青森市]]{{link-ja|青森交流道|青森インターチェンジ}}的[[日本高速公路|高速公路]]（[[高速自動車國道|高速自動車國道]]）。略稱為'''東北道'''（{{lang|ja|東北道／とうほくどう}}，TOHOKU EXPWY）。高速公路路線編號為{{Ja Exp Route Sign|E4}}{{efn|但是下列路段有重複編號，[[岩舟系統交流道|岩舟系統交流道]] - [[栃木都賀系統交流道|栃木都賀系統交流道]]間編為{{Ja Exp Route Sign|E50}}、[[福島系統交流道|福島系統交流道]] - {{link-ja|桑折系統交流道|桑折ジャンクション}}
間編為{{Ja Exp Route Sign|E13}}、{{link-ja|村田系統交流道|村田ジャンクション}} - {{link-ja|仙台南交流道|仙台南インターチェンジ
}}間編為{{Ja Exp Route Sign|E48}}、{{link-ja|北上系統交流道|北上ジャンクション}} - {{link-ja|花卷系統交流道|花巻ジャンクション}}編為{{Ja Exp Route Sign|E46}}<ref>{{Cite web |url=http://www.mlit.go.jp/road/sign/numbering/list/index.html |title=高速道路ナンバリング一覧 |publisher=国土交通省 |accessdate=2018-01-27}}</ref>。}}。

== 概要 ==
由[[川口JCT|川口JCT]]至[[瀧澤IC|瀧澤IC]]之間的範圍與[[國道4號_(日本)|國道4號]]及[[東北新幹線|東北新幹線]]並行，連接關東地方及東北地方，是[[東日本|東日本]]的「大動脈」。亦是日本最長的高速自動車國道，全線為[[東日本高速道路|東日本高速道路]]株式會社管理路線。

== 設施 ==
* 交流道（IC）編號欄背景色<span style="color:#BFB">■</span>區間是已通車區間，設施欄背景色<span style="color:#CCC">■</span>區間是未通車區間與未使用設施，未開通區間名稱全部是暫稱。
* [[智能IC|智能IC]]背景色為<span style="color:#eda5ff">■</span>。
* 下表之起點為[[川口JCT|川口JCT]]。
* BS（巴士站）：○為使用中，◆為停用中，未註記者即未設置。
* 停车区：休息區，SA：服務區，IC：交流道，SIC：智能IC，JCT：系統交流道，TB：設於主線之[[公路收費站|收費站]]（料金所）。

=== 速率限制 ===
基本上直線路段占該條高速道路全程的大多數，但於仙台近郊及岩手縣南部、青森縣南部附近有連續急彎道、隧道及陡坡的路段，因此在這些路段的部分道路線型並未完全達到高速道路的標準。在這些區間的最高速率限制除了臨時特殊的限制情形之外，平常規定為80[[km/h|km/h]]，而經常發生事故的隧道區間（如[[坂梨隧道|坂梨隧道]]）最高速率限制則為70km/h。
*川口JCT～浦和本線（收費站）、藏王停车区～泉IC、一關IC～前澤SA間：80km/h
*西根IC～大鰐弘前IC間：80km/h（一部70km/h）
*上述以外的路段除了臨時特殊的限制情形外，速率限制均為100km/h
