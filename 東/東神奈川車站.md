'''東神奈川車站'''（{{jpn|j=東神奈川駅|hg=ひがしかながわえき|rm=Higashi-nagawa eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]][[神奈川區|神奈川區]]{{link-ja|東神奈川|東神奈川}}一丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）的[[鐵路車站|鐵路車站]]。
== 停靠路線 ==
停靠此站的路線在軌道名稱上有[[東海道本線|東海道本線]]與[[橫濱線|橫濱線]]2條路線（詳情參見路線條目與{{link-ja|鐵道路線的名稱|鉄道路線の名称}}），當中此站屬於東海道本線的[[日本鐵路車站#所屬線|所屬線]]，東海道本線只有行走[[電車線與列車線|電車線]]的[[京濱東北線|京濱東北線]]列車停靠，旅客導覽不顯示「東海道（本）線」。因此此站的導覽顯示為京濱東北線與橫濱線的轉乘站。雖然橫濱線以此站為起點，一部分列車途經京濱東北線和[[橫濱站|橫濱站]]直通運行至[[根岸線|根岸線]]。

依據[[特定都區市內|特定都區市內]]，本站屬於「[[特定都區市內#設定區域一覧|橫濱市內]]」。

== 車站構造 ==
車站月台採[[島式月台|島式月台]]2面4線設計，所有月台均能容納較長的京濱東北線電車（10輛編組）。由於橫濱線的設計關係，本站內側2個月台由橫濱線電車使用，外側2個月台則由京濱東北線電車使用。

== 使用情況 ==
2016年度的1日平均'''上車'''人次為'''35,984人'''。雖是橫濱線的起點站，使用人數並不多。理由之一是此站的鄰站是[[橫濱站|橫濱站]]，使用者多流到該站。但是，此站到發的橫濱線列車使用人數的大半為轉乘京濱東北、根岸線的橫濱方向與[[川崎站|川崎]]方向，繁忙時段特別擠迫。因此，橫濱線內的20個車站當中排行第10位。
