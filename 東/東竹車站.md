東竹車站位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[富里鄉|富里鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[島式月台|島式月台]]一座。

== 利用狀況 ==
*目前為乙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]。
*為極少數有停靠對號列車之簡易站。
== 停靠狀態 ==
**[[莒光號列車|莒光號]]：停靠南下602次、642次(週六，日停駛)/640次(週六行駛)/6642次(週日行駛)；北上611次
**[[臺鐵區間車|區間車]]：皆停靠。
*對號列車停靠率：12.0%；所有車種停靠率：29.0%。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。
*目前營業時間為08：00～12：00、12：30～14：30、15：00～17；20,其餘時間無站務員服務。

== 歷史 ==
週邊包含閩南、客家、原住民及平埔族聚落，過去以農業昌盛見長，街市熱鬧充滿朝氣欣欣向榮，卻因民國60年代（1970年代）青年外移都市就業，如今人口稀少村景寥落。
*1924年5月1日：設站，初設時站名為「頭人埔驛」，據說此地昔日有一酋長強佔此地，不許外人來此放牧，故將此地稱為「頭人埔」。
*1937年11月1日：為與[[屏東線|屏東線]][[竹田車站|竹田站]]有所區別，站名改稱「東部竹田」。
*1945年：站名簡化為「東竹」。
*2014年7月15日：配合「花東新車站運動」計畫進行的改建工程完成。除了外觀立面的更新外，改建的項目也包括了跨月台[[行人地下道|行人地下道]]、月台候車雨棚與[[公共廁所|公共廁所]]的增建{{R|SL301}}。
*2016年6月28日：啟用多卡通刷卡機。
*2017年9月16日：本站由[[三等站|三等站]]降為乙種[[簡易站|簡易站]]，並指定由[[富里車站|富里站]]管理。
