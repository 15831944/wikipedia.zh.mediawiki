'''東澳車站'''位於[[臺灣|臺灣]][[宜蘭縣|宜蘭縣]][[南澳鄉|南澳鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[北迴線|北迴線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
東澳車站內設置有[[島式月台|島式月台]]兩座，雖然是設置在地面上的車站，但月台區實際上是位在一個土堤構造的上方，部分月台跨越[[蘇花公路|蘇花公路]]（[[台9線|台9線]]）。由於同時兼有水泥貨運集散的功能，因此站區內用地寬廣，設有多條主要供貨運使用的側線。

== 營運狀況 ==
*目前為二等站。 
*[[莒光號|莒光號]]:停靠南下(順行)642/640/6642、554，北上(逆行)605、653/6653次。
*[[復興號|復興號]]:停靠南下(順行)684、688/6688次，北上(逆行)689次。
*[[區間快車|區間快車]]：停靠南下(順行)4032、4042/4092、4046次，北上(逆行)4033/4083、4043次。
*[[區間車|區間車]]:皆停靠。
*站內設施
**自動售票機
**[[中華電信|中華電信]]公用電話服務
**有[[I-Taiwan|I-Taiwan]][[WIFI|WIFI]][[無線網路|無線網路]]服務
除了客運服務外，東澳也是[[幸福水泥|幸福水泥]]的貨運基地，站內經常可見到由散裝水泥車組成的貨運列車。
*本站可以使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]乘車。

== 歷史 ==
*本站是在1980年2月1日，伴隨[[蘇澳新站|蘇澳新站]]至[[花蓮新站|花蓮新站]]之間的北迴線全線通車，所一同啟用的車站之一。
*1985年7月1日：由三等站升為二等站。
*2016年3月1日：啟用多卡通刷卡機。
