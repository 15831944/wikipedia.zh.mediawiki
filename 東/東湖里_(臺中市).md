'''東湖-{里}-'''，位於[[大里區|大里區]]南境，與[[西湖里_(大里區)|西湖-{里}-]]並稱「草湖」。

== 里名由來 ==
因在「草湖」之東而得名。

== 地理 ==
東湖-{里}-劃分為36鄰，4,067戶，總人口11,723人，在本區內排行第五。本里東隔[[中興路_(大里區)|中興路一段]]與[[仁化里|仁化-{里}-]]、[[仁德里_(臺中市大里區)|仁德里]]、[[金城里_(大里區)|金城-{里}-]]、[[瑞城里|瑞城-{里}-]]為鄰，西以草湖路與[[西湖里_(大里區)|西湖-{里}-]]相毗連，北鄰[[大里溪|大里溪]]與[[大元里|大元里]]對望，南以[[草湖溪|草湖溪]]隔[[霧峰區|霧峰區]]吉峰里、甲寅里。

== 行政區劃沿革 ==
東湖-{里}-於[[清代|清代]]時隸屬'''[[藍興堡|藍興堡]]草湖庄'''；[[明治34年|明治34年]]（1901年）時，改隸'''[[臺中廳|臺中廳]]大里杙區草湖庄'''，至[[大正9年|大正9年]]（1920年）則隨行政區調整，隸屬'''[[臺中州|臺中州]][[大屯郡|大屯郡]][[大里庄|大里庄]]草湖大字'''。[[二次大戰|二次大戰]]結束後，[[國民政府|國民政府]]接收臺灣，以原草湖大字範圍分置'''東湖村'''與'''西湖村'''，隸屬[[臺中縣|臺中縣]][[大里區|大里鄉]]。[[民國82年|民國82年]]（1993年）大里鄉升格為縣轄市'''大里市'''，隨之改制為'''東湖-{里}-'''。
