'''東靜內站'''（{{jpn|j=東静内駅|hg=ひがししずないえき|rm=Higashi-shizunai eki|f=y}}）位於[[北海道|北海道]][[日高郡|日高郡]][[新日高町|新日高町]]東靜內，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|ヒナ}}'''。

== 車站構造 ==
[[側式月台|側式月台]]1面1線的地面站。以前是島式月台1面2線，目前上行線已被拆除。

== 車站周邊 ==
附近是日本首屈一指的[[純種馬|純種馬]]產地，同時也是[[自衛隊|自衛隊]]少數的對空靶場之一。
由於近海，附近有東靜內漁港和朝里濱（{{lang|ja|アサリ浜}}）。
* [[北海道道637号西川東靜內停車場線|北海道道637号西川東靜內停車場線]]
* [[國道235號|國道235號]]
* 靜內警察署東靜內派出所
* 東靜內郵局
* 日高漁業協同組合東靜內事業所
* 靜內營區
* 新日高町立東靜內小學
* 新日高町立靜內第二中學
* 道南巴士「東靜內」站牌

== 歷史 ==
* 1933年（[[昭和|昭和]]8年）12月15日 - 設立。
* 1977年（昭和52年）2月1日 - 廢止貨物和行李處理的業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
