'''東哥德王國'''（英文：'''The Ostrogothic Kingdom'''，489年－553年），是日耳曼民族的一支——[[東哥德人|東哥德人]]（Ostrogoths）建立的國家。隨著東哥德人在[[西羅馬帝國|西羅馬帝國]]滅亡後遷入義大利，東哥德王國的勢力約在義大利半島及附近地區。

== 日耳曼民族大遷徙 ==
考古學家及語言學家從北歐銅器時代文化的證據中推斷日耳曼民族的發源地應在今日的[[斯堪地納維亞半島|斯堪地納維亞半島]]南部、[[日德蘭半島|日德蘭半島]]及[[德國|德國]]北部。日耳曼民族在西元前五世紀時已經從他們的發源地向外擴展了。

剛開始靠近[[羅馬帝國|羅馬帝國]]邊境的日耳曼部族與羅馬帝國的接觸包括了貿易、移民，有些日耳曼人成了奴隸，有些日耳曼人則以個人或團體聯盟的方式加入羅馬帝國的軍隊，羅馬帝國也與一些日耳曼部族有過征戰。到了四世紀，羅馬帝國的軍隊其實常常是羅馬人指揮之下的日耳曼族軍隊，日耳曼人在羅馬軍隊裡的地位也逐漸重要起來。

四世紀末，中亞地區來的[[匈人|匈人]]向西移動，住在[[羅馬帝國|羅馬帝國]]邊境的日耳曼人爲求避難又再度大量湧入了羅馬帝國境內，即[[日耳曼民族大遷徙|日耳曼民族大遷徙]]。五世紀時，移入羅馬帝國的日耳曼人在羅馬帝國境內建立了數個王國，如北非的[[汪達爾王國|汪達爾王國]]、伊比利半島的[[西哥德王國|西哥德王國]]、高盧的[[法蘭克王國|法蘭克王國]]等等。

== 西羅馬帝國的滅亡 ==
如前所述，四世紀起，日耳曼人在羅馬軍隊裡的地位日增。到了五世紀中，[[羅馬帝國|羅馬帝國]]的皇帝已經沒有權力了。456年到475年日耳曼人[[李希梅爾|李希梅爾]]統治[[義大利|義大利]]，這期間，在466年到476年之間的五位羅馬皇帝都只是他任意拔擢、罷黜的傀儡。繼里西梅爾之後掌權的是[[奧多亞塞|奧多亞塞]]，他在476年罷黜了上述皇帝中的第五位皇帝，也就是[[西羅馬帝國|西羅馬帝國]]的最後一位皇帝——[[罗慕路斯·奥古斯都|羅慕路斯·奧古斯圖盧斯]]<ref>[http://www.livius.org/a/1/emperors/coin_romulus_augustulus.JPG 印有羅慕路斯·奧古斯圖盧斯像的錢幣]</ref>。至此，西羅馬帝國滅亡。

== 狄奧多里克的統治 ==
當時[[東羅馬帝國|東羅馬帝國]]的皇帝[[芝诺|芝诺]]任命東哥德族領袖[[狄奧多里克大帝|狄奧多里克]]為帝國官員去驅逐篡位的[[奧多亞塞|奧多亞塞]]。狄奧多里克率領東哥德人在489年越過阿爾卑斯山，打敗了[[奧多亞塞|奧多亞塞]]，並成為[[亚平宁半岛|亚平宁半岛]]的統治者。

狄奧多里克雖然是東哥德族領袖，但是是一個受過教育的人。他的童年在君士坦丁堡以人質的身分度過，並在當地接受了教育。488年，一方面是東哥德族國王一方面又有[[東羅馬帝國|東羅馬帝國]]官員身分的狄奧多里克被東羅馬帝國皇帝[[芝诺_(罗马帝国)|芝诺]]派去重新征服義大利。493年，狄奧多里克征服了[[拉文納|拉文納]]，並且殺了[[奧多亞塞|奧多亞塞]]。理論上，狄奧多里克只是君士坦丁堡管轄下的一個副王（viceroy）。然而，在實際上，東羅馬帝國對狄奧多里克並沒有約束的能力。

當時[[西羅馬帝國|西羅馬帝國]]雖然已經不存在了，在伊比利半島、高盧、義大利等地，[[西羅馬帝國|西羅馬帝國]]的遺民的人數也遠遠多過他們的統治者——[[哥德人|哥德人]]。在這些地方，法律的適用性通常是依據種族背景來分的。西羅馬帝國的遺民還是適用羅馬法，而日耳曼人則適用日耳曼人的法律。在狄奧多里克統治之下的義大利也是如此：西羅馬帝國的遺民適用羅馬法，而東哥德族人則可以說是獨立於西羅馬帝國遺民之外，保持自己的風俗習慣，受自己同族之中的貴族治理；哥德武士成为了当地的军事贵族，除此之外的大部分帝國时期的社會制度在他们的统治下保存了下來。

狄奧多里克是一位出色的統治者，在他的統治之下[[義大利|義大利]]比五世紀的任何一位皇帝統治之下的情況還要好。对于罗马留下来的财政制度，哥特政府未曾作出什么新的修改。罗马度量衡制和罗马驿站制继续下去。各种旧直接税甚至那在东罗马已由阿那斯塔细亚取消的"金银税"也保留着，哥特的赋税不如过去罗马赋税那样的苛重。政府的开支更是少得多；教会和公共教育机关都没有领到什么定额补助费；军费的开支在和平时期原是很少的；官吏俸给看来是远低于罗马官僚制政府的官俸；而且其中大部分可能是由行政费内开支的。可以說，[[西羅馬帝國|西羅馬帝國]]的文明在狄奧多里克的統治之下在義大利延續下來了。在城市裡，古典時代的法律學校與修辭學校仍然興旺。狄奧多里克改變稅制，試著使賦稅更為公平，穷人少缴税，富人多缴税。相較於五世紀的兵荒馬亂来说，狄奧多里克的統治給了義大利三十年的和平。

對外政策方面，狄奧多里克透過一系列的聯姻把一些其它日耳曼人建立的王國組成聯盟。狄奧多里克把自己的妹妹嫁給阿兰-汪達爾王國的國王，一個女兒嫁給[[西哥德王國|西哥德王國]]的國王，另一個女兒嫁給了[[勃艮第王國|勃艮第王國]]（約在今日[[法國|法國]][[羅納河|羅納河]]河谷一帶）的國王，狄奧多里克自己則是娶了[[法蘭克王國|法蘭克王國]]國王[[克洛維|克洛維]]的妹妹。如此一來，東哥德王國成了日耳曼人建立的王國彼此之間因聯姻而建立的關係的中心。

狄奧多里克还成为了[[西哥德王國|西哥德王國]]的攝政王。當時的西哥德王國據有高盧西南部及大部分的伊比利半島。因此，狄奧多里克的影響力從今日的葡萄牙、西班牙、法國西南部一直延伸到義大利。

== 東哥德王國統治機制的問題 ==
古典文明要在[[狄奧多里克大帝|狄奧多里克]]統治之下的[[義大利|義大利]]復興看起來似乎是有希望的。然而，東哥德王國的統制機制有很大的問題。

狄奧多里克統治之下的東哥德王國對[[西羅馬帝國|西羅馬帝國]]的遺民跟[[東哥德人|東哥德人]]有雙重的制度。一方面，狄奧多里克是東哥德人的國王，另一方面，他是[[東羅馬帝國|東羅馬帝國]]的官員，治理著為數眾多的西羅馬帝國的遺民，而東哥德人的角色像是他們的保護者。信仰方面，西羅馬帝國的遺民信奉的是正統派的基督教，而東哥德人信奉的是基督教裡有爭議性的[[阿里烏派|阿里烏派]]。

[[東哥德人|東哥德人]]要成功地統治[[義大利|義大利]]，必須要得到[[西羅馬帝國|西羅馬帝國]]的遺民的合作。但是如前所述，這兩群人的社會在許多方面明顯不同。東哥德人與西羅馬帝國遺民之間的互信基礎也不夠。可以說，東哥德王國在義大利成功的統治，靠的其實是狄奧多里克個人的能力。一旦狄奧多里克不在位了，這樣的統制機制其實並不穩定。

== 狄奧多里克到維蒂吉斯之間的時期 ==
[[狄奧多里克大帝|狄奧多里克]]在526年去世。繼位的是狄奧多里克未成年的外孫[[阿塔拉里克|阿塔拉里克]]（Athalaric），而阿塔拉里克的母親，也就是狄奧多里克的女兒[[阿瑪拉遜莎|阿瑪拉遜莎]]（Amalasuntha）則為攝政王。

527年，[[查士丁尼一世|查士丁尼一世]]成為[[東羅馬帝國|東羅馬帝國]]的皇帝。查士丁尼一世是一位有野心的皇帝，進行了一系列復興[[羅馬帝國|羅馬帝國]]的戰爭。首先他派將軍[[貝利薩留|貝利薩留]]對汪達爾人出兵，在534年滅亡了[[汪達爾王國|汪達爾王國]]，汪達爾人從此在歷史舞台上消失。535年，又派貝利薩留攻打東哥德王國。

貝利薩留很快地在535年攻下了西西里島。536年，貝利薩留攻下了[[那不勒斯|那不勒斯]]及[[羅馬|羅馬]]。

== 維蒂吉斯的統治 ==
在貝利薩留的攻打之下，東哥德王國的王位從[[狄奧多里克大帝|狄奧多里克]]的家族轉移到了另外一位領袖[[维蒂吉斯_(东哥特)|维蒂吉斯]]（Witigis）的手中。536年，維蒂吉斯在對抗貝利薩留的戰爭中即位為東哥德王國國王。維蒂吉斯是[[阿瑪拉遜莎|阿瑪拉遜莎]]（Amalasuntha）在當時唯一存活的小孩——瑪瑟遜莎（Mathesuentha）的丈夫。不過這樁婚姻只是支持維蒂吉斯的王位繼承的政治婚姻而已。

維蒂吉斯統治之下的東哥德王國還是無法抵擋[[東羅馬帝國|東羅馬帝國]]的攻擊。貝利薩留繼續向北攻下了[[米蘭|米蘭]]，並且在540年攻下了東哥德王國的首都[[拉文納|拉文納]]。維蒂吉斯與瑪瑟遜莎都被俘擄。

這個時候，[[查士丁尼一世|查士丁尼一世]]給了[[東哥德人|東哥德人]]一個“慷慨”的協議：東哥德人交出他們一半的財物給[[東羅馬帝國|東羅馬帝國]]，則可在[[義大利|義大利]]西北部保有一個獨立的王國。貝利薩留將消息告知東哥德人。東哥德人並不信任[[查士丁尼一世|查士丁尼一世]]而比較信任貝利薩留，他們提出一個條件：只要貝利薩留背書，他們就接受協議。然而，這個條件卻使得協議陷入僵局。

== 維蒂吉斯之後的東哥德王國 ==
540年，因為維蒂吉斯的失利，一群貴族認為他們需要一位新的領袖。這群貴族的首領[[艾拉里克_(东哥特)|艾拉里克]]（Eraric）支持貝利薩留。在徵得其它人的同意之後，他們決定要將王位讓給貝利薩留。貝利薩留還是對[[查士丁尼一世|查士丁尼一世]]忠心的。他假裝同意這項提議，前往[[拉文納|拉文納]]進行加冕，然後出其不意地逮捕東哥德人的領袖，並且將全部東哥德王國置於[[東羅馬帝國|東羅馬帝國]]的統治之下——這次沒有“慷慨”的協議了。

接下來，被選為東哥德王國國王而在540年繼位的是[[伊尔迪巴德_(东哥特)|伊狄巴德]]（Ildibad）（又寫為Hildebad或Heldebadus）。

一方面，[[東羅馬帝國|東羅馬帝國]]的東邊正在與[[波斯帝國|波斯帝國]]作戰，因此[[查士丁尼一世|查士丁尼一世]]希望東羅馬帝國西邊能有一個緩衝國家把[[法蘭克王國|法蘭克王國]]與東羅馬帝國隔開。另一方面，東哥德人對貝利薩留的提議讓查士丁尼一世起了疑心。查士丁尼一世對貝利薩留之舉大為不滿，將他調往東邊對抗[[波斯帝國|萨珊王朝]]，而將[[亚平宁半岛|亚平宁半岛]]置於東羅馬帝國另一位地方官員約翰（John）的管理之下。

伊狄巴德在位僅約一年（540年－541年）。接下來繼位東哥德王國國王的的是[[艾拉里克_(东哥特)|艾拉里克]]（541年），但是旋即被殺害。再接下來在541年被選為東哥德王國國王繼位的是[[托提拉|托提拉]]（Totila）。托提拉與立場偏向[[東羅馬帝國|東羅馬帝國]]的艾拉里克不同，是一個立場偏向[[東哥德人|東哥德人]]的國王，同時也是一位出色的領導者。他帶領東哥德人反抗[[東羅馬帝國|東羅馬帝國]]。

[[File:Byzantium550.png|thumb]]征服的土地。]]
當貝利薩留在545年回到[[義大利|義大利]]的時候，他發現情勢已經改變了。艾拉里克已經被殺害，親[[東羅馬帝國|東羅馬帝國]]的東哥德貴族也被推翻，整個義大利北部又被[[東哥德人|東哥德人]]佔據，東哥德人甚至還將[[東羅馬帝國|東羅馬帝國]]的军队逐出了[[羅馬城|羅馬城]]。

貝利薩留再度奪回[[羅馬|羅馬]]，但是與貝利薩留已有嫌隙的[[查士丁尼一世|查士丁尼一世]]沒有給貝利薩留足夠的支援與補給，使得貝利薩留只能採取守勢，並且失去了羅馬。548年，查士丁尼一世以他信任的[[納西斯|納西斯]]（Narses）將軍代替貝利薩留。

納西斯沒有讓[[查士丁尼一世|查士丁尼一世]]失望。在納西斯的征服行動中，托提拉於552年戰死。552年繼位為東哥德王國國王的[[泰雅_(东哥特)|德亞]]（Teia）也於553年戰死。德亞是最後一位東哥德王國國王。

歷史上有記載的[[東哥德人|東哥德人]]的抵抗一直持續到550年代末。被擊敗之後，東哥德人從此自歷史舞台上消失。

== 東哥德王國的貢獻 ==
對義大利人而言，[[東哥德人|東哥德人]]也許只是歷史上一群侵略者。但是[[狄奧多里克大帝|狄奧多里克]]給了[[義大利|義大利]]三十年的和平，在狄奧多里克的統治之下，義大利的情況不比五世紀時動亂的時代來得差，古典文明和大多數羅馬的社會制度在[[亚平宁半岛|亚平宁半岛]]也保留下來了。

反而是[[東羅馬帝國|東羅馬帝國]]擊潰東哥德王國之後，繼位的皇帝[[查士丁二世|查士丁二世]]卻又無力保衛[[義大利|義大利]]。於是，當另一批日耳曼人——[[倫巴底人|倫巴底人]]從568年開始入侵[[亚平宁半岛|亚平宁半岛]]時，東羅馬帝國阻止不了倫巴底人對[[義大利|義大利]]施加的毁坏。

* [[阿塔拉里克|阿塔拉里克]]（Athalaric）526年－534年
