'''東非大裂谷'''（Great Rift Valley），位於[[非洲|非洲]]東部，是一個在3500萬年前由非洲板塊的[[地殼|地殼]]運動所形成的地理奇觀，縱貫東非的大裂谷是世界上最大的斷裂帶，属于生长边界。其所形成的生態、地理和人類文化都相當獨特，目前觀光的主要景點則由[[肯亞|肯亞]]進入。

== 地理 ==
東非大裂谷的整個形狀可畫成不規則[[三角形|三角形]]，最深達2000公尺，寬30～100公里，全長6000公里（3700英里），是世界最長的不連續谷，由探險家[[約翰·華特·古格里|約翰·華特·古格-{里}-]]所命名。東非大裂谷的詳細地理位置以三角形的三個點來描述的話，南點在[[莫三比克|莫三比克]]入海口，西北點則遠到[[苏丹共和国|蘇丹]][[約旦河|約旦河]]，北點則可進入[[死海|死海]]。中間有相當多個湖泊、[[火山|火山]]群。

== 東部及西部 ==
一般會將大裂谷區分成東部和西部，在西部裂谷，又稱為[[艾伯丁裂谷|艾伯丁裂谷]]，主要由高山群組成，包括有Virunga山，Mitumba山，Ruwenzori山區和Rift Valley湖區，其中有世界最深淡水湖之一的[[坦干依喀湖|坦干依喀湖]]（1470公尺深），以及世界上第二大的[[淡水湖|淡水湖]]－[[維多利亞湖|維多利亞湖]]。西部裂谷的形狀是比較破碎的。

位於肯亞的東部裂谷部分，則是整個東非大裂谷區較深的部份，其中最深處在肯亞的[[奈洛比|奈洛比]]北部，這一部份的裂谷區沒有出海，因此所造成的湖泊則較淺、所含礦物和鹽份也較高。如[[馬加迪湖|馬加迪湖]]幾乎都是碳酸鈉組成。[[埃爾門泰塔湖|埃爾門泰塔湖]]（Lake Elmenteita）、[[巴林哥湖|巴林哥湖]]、[[柏哥利亞湖|柏哥利亞湖]]和[[納庫魯湖|納庫魯湖]]（Lake Nakuru）則鹼性相當高。[[奈瓦夏湖|奈瓦夏湖]]則是小規模的淡水湖。

== 火山 ==
目前東非大裂谷依然在活動當中，地質學家預測幾百萬年後，東非可能會分裂成不同的版塊形狀。目前地殼活動較旺盛的區域在肯亞和坦桑尼亞的火山區，包括有非洲最高峰[[吉力馬扎羅山|吉力馬扎羅山]]、[[卡里辛比山|卡-{里}-辛比山]]（[[卡里辛比火山|卡里辛比火山]]，非洲中東部雅龍加火山山脈最高峰）、[[尼拉貢戈火山|尼拉貢戈火山]]（Nyiragongo）、[[梅盧山|梅盧山]]（[[梅魯火山|梅魯火山]]）、[[埃爾貢山|埃爾貢山]]、世界上其中一座位海平面以下的[[爾塔阿雷火山|爾塔阿雷火山]]（[[尔塔阿雷火山|Erte Ale]]）、著名的[[倫蓋火山|倫蓋火山]]（[[:en:Ol_Doinyo_Lengai|Ol Doinyo Lengai]]，高2890米，世界上唯一一座[[熔岩|熔岩]]溫度不高，並以[[鈉|鈉]]和[[碳酸鹽|碳酸鹽]]成份居多的火山）等。

== 大裂谷底部的湖泊 ==
{{See also|東非大裂谷湖泊}}
東非大裂谷底部的[[湖泊|湖泊]]依成因主要分為兩種：
# [[斷層湖|斷層湖]]：是東非大裂谷斷裂時形成的，水深狀長，如[[坦干依喀湖|坦干依喀湖]]
# [[構造湖|構造湖]]：因地殼運動使地面下沉而形成的集水盆地，湖廣水淺，如[[維多利亞湖|維多利亞湖]]

== 原始人類 ==
東非大裂谷的另一個特色是，它可能是人類文明最早的發源地，1950年代到1970年代考古學家曾在東非大裂谷中發現了200萬年前、290萬年前人類的頭骨，而引起世人注目的則是1975年，在[[坦桑尼亞|坦桑尼亞]]和肯尼亞交界挖出了350萬年前的人類遺骨，以及足跡化石，這是目前為止所發現最古老的史前人類證據。
