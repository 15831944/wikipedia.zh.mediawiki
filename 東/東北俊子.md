'''東北俊子'''是2011年[[2011年日本東北地方太平洋近海地震|东北地区地震]]后，SSS合同会社于同年十月发表的东北地方支援角色，亦是SSS合同会社委托{{tsl|ja|AH-Software}}開發的[[日语|日语]][[语音合成|语音合成]]女性声音资料库。角色的人物設計師為江戸村ににこ，聲優為[[佐藤聰美|佐藤聰美]]。

== 形象简介 ==
東北俊子拥有绿色的长发，头戴毛豆状的头带，身穿绿色短小的和服，拥有富有弹性的肌肤，持有一把叫做「ずんだアロー」的弓。是高校二年级的学生。在官方小说中，提到東北俊子（東北 ずん子）的本名叫做东北顺子（東北じゅん子）。

== 产品簡介 ==
=== 历史 ===
最初只是作为[[2011年日本東北地方太平洋近海地震|东北地区地震]]的支援角色，后来有其推出VOICEROID的计划，从而在2012月1月開始了為東北俊子的VOICEROID化募集资金.2012年9月28日，AHS正式发售「VOICEROID+ 東北俊子」，还有相对应的exVOICE扩展包。2013年6月20日，東北毛豆子VOCALOID化启动，在一个月中，已募集的资金已达到目标，2014年6月5正式发售。2016年也通过同样的方式推出了VOCALOID4版本。

== 版權申明 ==

東北毛豆子為標榜支援復興受到[[東日本大震災|東日本大震災]]造成損害的東北地方，而以[[毛豆麻糬|毛豆麻糬]]（{{lang|ja|ずんだ餅}}）為原型所設計的虛構角色。只要是總部登記在日本[[东北地方_(日本)|東北地方]]的企業，都可以不經任何手續，以商業目的免費使用在{{tsl|ja|POP広告|POP廣告}}或商品包裝、網站上。若為非商業使用，則不限於東北地方，都可以不經手續免費使用。
