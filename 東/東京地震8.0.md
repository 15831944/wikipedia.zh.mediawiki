《'''東京地震8.0'''》（{{lang-ja|'''東京マグニチュード8.0'''}}）是2009年7月於日本[[富士電視台|富士電視台]]的[[noitaminA|noitaminA]]播放的以虛構[[地震|地震]]為主要情節的電視動畫<ref>{{cite web|url=http://www.animenewsnetwork.com/news/2009-03-17/tokyo-magnitude-8.0-anime-to-air-in-japan-in-july/|title=Tokyo Magnitude 8.0 Anime to Air in Japan in July (Updated)|publisher=[[動畫新聞網|動畫新聞網]]|date=2009年3月17日|accessdate=2014年1月16日|language=en}}</ref>。香港地區於2010年11月5日起，由[[J2|J2]]播出。台灣於2011年3月19日起，由[[衛視電影台|衛視電影台]][[日語|日語]]原音放送。另外，台灣[[衛視電影台|衛視電影台]]與[[公視|公視]]播放時是將全11話的TV版動畫全部編成上下兩集的電影形式播出，而[[衛視中文台|衛視中文台]]則是一小時連播2話日語原音播出。此外[[臺灣|臺灣]]播出該動畫的時機點正逢2011年日本發生[[2011年日本東北地方太平洋近海地震|311大地震]]的時候，因此警示和教育的意味很濃厚。

臺灣[[公視|公共電視臺]]於2011年7月以[[華語|國語]]、[[日語|日語]]雙語播出，同樣編成上下兩集電影。2011年8月公共電視臺再次播出，因時間分配而改為以三集的電影形式播出。

==故事簡介==
正處於反抗期的初中一年級生小野澤未來，與弟弟悠貴一起來到[[台場|台場]]遊玩，卻被捲進[[震级|震级]]8.0的大[[地震|地震]]。二人與偶然相識的成人女性日下部真理從建築物中逃出來，卻發現眼前出現一個完全陷落，變成自己不認識的都市。

不擅長與人交往，認定所有大人都是污穢的未來；不知世途險惡，與任何人都能成為朋友的悠貴；還有擔當二人的保護者，單身母親真理。為了再與因地震而分開的家人再會，三人一同展開了這次的崩壞東京之旅。

== 概要 ==
這是一部模擬真實地震發生後，挑戰社會性的題材。故事以發生大地震受到災害的東京作為舞台的原創動畫。故事中透過主角們的互動，將體驗過大震災的人痛切感受到的想法融入作品，兼具正面教育意義，喚起大家對地震災害的防災意識。

影像製作除了向東京消防廳、[[海上保安廳|海上保安廳]]、[[陸上自衛隊|陸上自衛隊]]等進行綿密的取材外，更深入受災地區，以龐大綿密的採訪資料與人員訪問為基礎，包括街道、救助、支援的狀況，真實描繪地震與二次災害的恐怖，傳達不只是故事中的世界，也有可能是未來現實的動畫影像。不僅僅是單純的災難作品，更多的是連繫人與人心的牽絆，細膩刻畫角色性格與人性，從小孩子的觀點講述如何跨越發生在自己身邊的巨大變化，家人的重要帶來的感動。

2009年7月在日本富士電視台「noitaminA」播出，第一話創下日本深夜動畫首集最高收視率5.8%的驚人收視率，同時《東京地震8.0》更是登上了日本地區的防災海報於日本全國發出外，並由電視系列重新再剪輯的《東京地震8.0總集篇》的影片，預定往後在各地區的防災宣傳活動時舉行上映。獲選為2009年[[日本|日本]][[文部省|文部省]][[文化廳媒體藝術祭|文化廳媒體藝術祭]]動畫部門第十三回動畫優秀獎。

== 地震詳細資訊 ==

* 發生時間 - 2012年7月21日 下午3點46分
* [[地震規模|地震規模]] - 8.0
* 地震震央 - 東京灣北部
* 震源深度 - 約地下25公里
* 地震形態 - 海溝型{{noteTag|name=e1|1=官方網站的宣傳為海溝型(板塊間地震)，在第3集為首都直下型(板塊內地震)，在第7集為都市直下型。}}
* 死亡人數 - 約18萬人(截止至2012年7月23日)
* 失蹤人數 - 約15萬人(截止至2012年7月23日)
* 輕重傷者 - 約20萬人(截止至2012年7月23日)
* 受困無法返家人數 - 約650萬人

; 災害的第一天
: 多處發生[[土壤液化|土壤液化]]、火災等[[二次災害|二次災害]]。三軒茶屋站周邊發生大規模火災。
: 多處高速公路和鐵路等的高架橋倒塌。全部的交通工具暫停行駛。第一次[[交通管制|交通管制]]實施，自[[國道16號|國道16號]]西側往都心方向禁止車輛通行。[[多摩川|多摩川]]、[[國道246號|國道246號]]以及[[東京都道318號環狀七號線|環狀七號線]]連線內側的範圍全面禁止車輛通行。另外在國道16號以東的都縣境，往都內方向禁止車輛出入。
: 手機由於通信限制無法通話，而[[災用傳話板服務|災用傳話板服務]]可以使用，[[1seg|1seg]]移动电视可以接收信号。
: [[荒川區|荒川區]]在颱風的影響下提防潰堤，水淹進地下鐵。
; 災害的第二天
: [[彩虹大橋|彩虹大橋]]、[[東京鐵塔|東京鐵塔]]倒塌，周邊損害擴大。由於[[地上數位廣播|地上數位廣播]]等廣播設施已轉移至[[東京天空樹|東京天空樹]]，電視、收音機、[[3G|3G]]手機等不受影響。
; 災害的第三天
: [[美國海軍|美國海軍]][[第七艦隊_(美國海軍)|第七艦隊]]的碼頭型[[兩棲突擊艦|登陸艦]]在各地開始運輸受災者。
: 中午12點5分發生[[日本氣象廳震度等級|震度]]6以下的餘震。
; 災害的第四天
: [[三軒茶屋站|三軒茶屋站]]周邊火災已撲滅。收容罹難者遺體達40名以上。
