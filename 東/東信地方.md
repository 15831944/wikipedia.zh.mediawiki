'''東信地方'''（{{jpn|j=東信地方|hg=とうしんちほう|rm=Tōshin chihō|f=y}}）是指[[日本|日本]][[長野縣|長野縣]]的東部千曲川（[[信濃川|信濃川]]）流域，也稱'''東信州'''（{{jpn|j=東信州|hg=ひがししんしゅう}}）或'''上田、佐久地方'''（{{jpn|j=上田・佐久地方|hg=うえださくちほう}}）。

== 地理 ==
東信地方北以[[淺間山|淺間山]]（[[關東山地|關東山地]]）、南以[[八岳|八岳]]、東以[[碓氷峠|碓氷峠]]（関東山地）、西以{{link-ja|和田峠 (長野縣)|和田峠 (長野県)|和田峠}}（北八岳、美原）為界。

[[上信越自動車道|上信越自動車道]]、[[國道18號|國道18號]]、[[北陸新幹線|北陸新幹線]]途徑此地，佐久地方則正在進行[[中部橫斷自動車道|中部橫斷自動車道]]的整備。水系方面，佐久地方的市町村屬於[[太平洋|太平洋]]側地域，其他地域屬[[日本海|日本海]]側。這一地區的氣候呈現內陸性氣候的特徵，部分地區屬於日本海側氣候，是豪雪地帶。除以以外的地域屬中央高地式氣候，降水量較少，輕井澤、野邊山等高原地帶較為嚴寒<ref>[https://web.archive.org/web/20070521205954/http://www.hrr.mlit.go.jp/omachi/wind/issue/16.htm 第16号 長野県の雨] 国土交通省大町ダム 平成14年7月</ref>。

* 河川：[[信濃川|千曲川]]

* 盆地：{{link-ja|佐久盆地|佐久盆地}}（佐久平）、{{link-ja|上田盆地|上田盆地}}（上田平、{{link-ja|鹽田平|塩田平}}）
* 高原：{{link-ja|野邊山高原|野辺山高原}}、{{link-ja|湯之丸高原|湯の丸高原}}、{{link-ja|菅平高原|菅平高原}}、{{link-ja|高峰高原|高峰高原}}、輕井澤高原

== 地域 ==
; {{link-ja|上小地域|上小地域}}
中心都市為[[上田市|上田市]]，另還包含[[東御市|東御市]]、[[小縣郡|小縣郡]]。而東御市有時僅計入舊東部町地域。

本區域有[[信濃鐵道線|信濃鐵道線]]、[[國道18號|國道18號]]通過。另由於皆位在[[國道144號|國道144號]]沿線，鄰接的群馬縣[[吾妻郡|吾妻郡]]（嬬戀村、長野原町）也屬於[[上田市|上田市]]（舊真田町）的都市圈內。

[[江戶時代|江戶時代]]初期以前，本地是以「十勇士」聞名的[[真田氏|真田氏]]地盤。之後成為[[上田藩|上田藩]]領土。本地的{{link-ja|別所溫泉|別所温泉}}因與[[北條氏|北條氏]]有淵源，又稱「信州之[[鎌倉|鎌倉]]」。

; {{link-ja|佐久地域|佐久地域|北佐久地域}}（[[北佐久郡|北佐久郡]]）
本區域有[[信濃鐵道線|信濃鐵道線]]、[[國道18號|國道18號]]通過，兩路線在中心都市[[小諸市|小諸市]]分歧出[[小海線|小海線]]與{{link-ja|國道141號|国道141号}}。

有時本地域還包含以前同屬[[北佐久郡|北佐久郡]]的[[東御市|東御市]]一部分（舊北御牧村）與佐久市一部分（舊望月町、淺科村）。

本地位於[[淺間山|淺間山]]南麓，有知名避暑勝地[[輕井澤町|輕井澤]]與[[島崎藤村|島崎藤村]]淵源地小諸。受惠於[[碓氷峠|碓氷峠]]，與[[群馬縣|群馬縣]]的聯繫也較為密切。

; {{link-ja|佐久地域|佐久地域|南佐久地域}}（[[南佐久郡|南佐久郡]]）
本區域有小海線、國道141號通過。[[國道254號|國道254號]]可通往[[群馬縣|群馬縣]]。

佐久市雖跨越北佐久地域與南佐久地域，但一般多劃入南佐久地域。

南部是{{link-ja|野邊山高原|野辺山高原}}，以[[萵苣|萵苣]]等高地蔬菜栽培聞名。南部與相鄰的[[山梨縣|山梨縣]]聯繫較為密切。

== 交通 ==
佐久地方透過[[國道18號|國道18號]]可北上至日本海側（[[北信地方|北信地方]]、[[上越地方|上越地方]]），穿過[[碓冰峠|碓冰峠]]則可至[[群馬縣|群馬縣]]。

另外，[[宇都宮市|宇都宮]]～[[高崎市|高崎]]～[[小諸市|小諸]]～[[山梨縣|山梨縣]]（[[韮崎市|韮崎]]、[[甲府市|甲府]]）～[[駿河灣|駿河灣]]岸（[[清水市|清水]]、[[富士市|富士]]）的'''碓冰峠、千曲川、[[富士川|富士川]]路線'''則做為[[東京都|東京]]與其周邊堵塞時的「迂回路」。

=== 交通史 ===
[[戰國時代_(日本)|戰國時代]]以前，本地作為[[東山道|東山道]]沿線發展，[[江戶時代|江戶時代]]起成為[[中山道|中山道]]的途經之地。戰國時代以前的東山道是從[[松本市|松本]]經[[上田市|上田]]至[[碓冰峠|碓冰峠]]的路線，江戶時代的中山道則是從[[塩尻市|塩尻]]經{{link-ja|和田峠 (長野縣)|和田峠 (長野県)|和田峠}}至碓冰峠的路線。此地設有{{link-ja|追分宿|追分宿}}（[[輕井澤町|輕井澤]]郊外）作為[[中山道|中山道]]與[[北國街道_(信越)|北國街道]]的分岐點。

之後進入[[明治|明治]]的鐵道敷設期，連結追分宿與{{link-ja|下諏訪宿|下諏訪宿}}的中山道（{{link-ja|國道142號|国道142号}}）沿線並未鋪設鐵道。從此，佐久地方逐漸失去連結關東與近畿的內陸路線角色，而轉變成關東與[[日本海|日本海]]側的要道（國道18號、JR信越本線、上信越道、北陸新幹線）。
