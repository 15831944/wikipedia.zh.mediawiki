'''東山文化'''（ひがしやまぶんか）是[[室町時代|室町時代]]中期的[[文化|文化]]。八代將軍[[足利義政|足利義政]]（1436年-1490年）的'''東山山莊'''為中心，融合武家、公家、禪僧等文化而誕生。[[京都|京都]]的[[慈照寺|慈照寺]]銀閣是東山文化的代表建築。

== 概要 ==
[[應仁之亂|應仁之亂]]（1467年）以降，雖然在亂世之中，但是，一方面也是[[能|能]]、[[茶道|茶道]]、[[華道|華道]]、[[庭園|庭園]]、[[建築|建築]]、[[連歌|連歌]]等多樣藝術開花的時代。這些藝術次第向庶民浸透，延續到今日產生許多日本的文化。又，[[京都|京都]]遭遇戰火，許多文化人・知識人寄身到地方的[[守護大名|守護大名]]，文化也由此向地方傳播。相對於貴族的・華麗的足利義滿的[[北山文化_(日本)|北山文化]]，東山文化是由[[幽玄|幽玄]]、[[侘・寂|侘・寂]]的美意識所形成。村田珠光所創的「侘茶」，成為後世茶道的基礎；立花(插花)的誕生；善阿彌把禪的精神帶入庭園造景，確立了「枯山水」的風格；完成了日後日本家屋基礎的「書院造」，這些深深影響後來日本文化的藝術。

== 建築 ==
*[[慈照寺|慈照寺]]銀閣：正式名稱是慈照寺觀音殿。一層是住宅風的[[書院造|書院造]]、二層是禪宗樣（唐樣）的佛殿所構成。
*慈照寺東求堂：持佛堂。四疊半的書院造建築。茶室的起源、近代和風建築的原型。
*[[大笹原神社|大笹原神社]]本殿

== 庭園 ==
*[[龍安寺|龍安寺]]方丈庭園
** [[長方形|長方形]]的庭內敷白砂，配置15個石。不用一木一草，以極度象徵的表現手法描寫自然。[[枯山水|枯山水]]的代表庭園之一。
*[[大德寺|大德寺]]大仙院庭園
** 枯山水的代表庭園之一。以石和白砂象徵的表現發自[[深山幽谷|深山幽谷]]的水，成為落瀑、大河的流動全景。

== 繪畫 ==
*[[狩野正信|狩野正信]]（1434年-1530年）：[[狩野派|狩野派]]之祖。[[小栗宗湛|小栗宗湛]]之後成為幕府的[[御用繪師|御用繪師]]。
*[[土佐光信|土佐光信]]（生没年不詳）：[[土佐派|土佐派]]之祖。融合水墨畫和[[大和繪|大和繪]]。
*[[雪舟|雪舟]]（1420年-1506年）：日本風的[[水墨畫|水墨畫]]之集大成。受[[大內氏|大內氏]]庇護。

== 工藝 ==
*[[後藤祐乘|後藤祐乘]]（1440年-1512年）：金工。仕於義政，製作刀劍金具。
*幸阿彌家：漆工
*五十嵐家：漆工
*相阿彌：[[盆庭|盆庭]]
