'''東京高速道路'''（{{jpn|j=東京高速道路|hg=とうきょうこうそくどうろ|rm=tōkyō kōsokud ōro|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[中央區_(東京都)|中央區]]的[[高架道路|高架]][[快速道路|快速道路]]。由於其經營者為'''東京高速道路株式會社'''（{{jp|j=東京高速道路株式会社|hg=とうきょうこうそくどうろかぶしきかいしゃ|rm=tōkyō kōsoku dōro kabushiki kaisha|f=y}}），因此此道路通常也被稱為'''東京高速道路株式會社線'''、'''會社線'''或'''KK線'''<ref>{{Cite web|url=http://www.shutoko.jp/use/network/kk/|title=首都高ドライバーズサイト KK線のご利用について|publisher=首都高速道路|accessdate=2017-8-27|archive-url=https://web.archive.org/web/20170814165112/http://www.shutoko.jp/use/network/kk/|archive-date=2017-08-14|dead-url=no}}</ref>。

== 概要 ==
東京高速道路長約2公里，起點位於蓬萊橋（東京都中央區[[銀座|銀座]]8丁目），終點位於新京橋（東京都中央區銀座1丁目）<ref>{{Cite web|url=http://www2.odn.ne.jp/nihon-yuryodoro/kaiin/tokyo/index.html|title=日本收費道路協會－東京高速道路|accessdate=2014-11-08|archiveurl=https://web.archive.org/web/20160304113659/http://www2.odn.ne.jp/nihon-yuryodoro/kaiin/tokyo/index.html|archive-date=2016-3-4}}</ref>，路線大致上環繞[[銀座|銀座]]西側。此道路在{{link-ja|京橋系統交流道|京橋ジャンクション}}和[[汐留系統交流道|汐留系統交流道]]與[[首都高速都心環狀線|(C1)環狀線]]連接（並於{{link-ja|西銀座系統交流道|西銀座ジャンクション}}與[[首都高速八重洲線|(Y)八重洲線]]連接），形成一條[[繞行道路|繞行道路]]。其中，由東京高速道路株式會社真正管理的範圍是白魚橋乘繼所<ref>{{Cite web |url=http://www.shutoko.jp/~/media/Images/customer/use/network/jct/routeguide/jct_kyobashi |title=首都高 系統交流道 複雜交流道 示意圖－京橋 |accessdate=2014-11-08 |archive-url=https://web.archive.org/web/20141108051212/http://www.shutoko.jp/~/media/Images/customer/use/network/jct/routeguide/jct_kyobashi |archive-date=2014-11-08 |dead-url=no }}</ref>－汐留乘繼所<ref>{{Cite web |url=http://www.shutoko.jp/~/media/Images/customer/use/network/jct/routeguide/jct_shiodome |title=首都高 系統交流道 複雜交流道 示意圖－汐留 |accessdate=2014-11-08 |archive-url=https://web.archive.org/web/20141108053634/http://www.shutoko.jp/~/media/Images/customer/use/network/jct/routeguide/jct_shiodome |archive-date=2014-11-08 |dead-url=no }}</ref>及西銀座系統交流道－西銀座乘繼所路段。

在道路指示標誌中很少使用「東京高速道路」此稱呼。雖然稱為「高速道路」，但實際上此道路並非完全的自動車專用道路：根據[[道路法_(日本)|道路法]]及道路運送法，此道路既是自動車専用道路（重型機車不可上路），也是一般自動車道（重型機車可上路）。

此外，由於[[東京都廳|東京都廳]]規定使用河川的建設不須付費，因此使用此道路不須支付通行費<ref>講談社　図解・首都高速の科学　川辺謙一著　P61 </ref>。而東京高速道路株式會社藉由租賃高架道路下方房屋維持營運（由於有管理高速道路且有收入，該公司也有加入日本收費道路協會）。

== 歷史 ==
[[1951年|1951年]]（昭和26年），為了緩和[[日本歷史#二戰後修憲|戰後]]銀座復興後飽和的車流量，商界23名人士成立東京高速道路株式會社<ref>通稱「KK線」的由來（「株式（K）会社（K）線」）[https://www.nikkei.com/article/DGKKZO35240970R10C18A9L83000/ 【点照】東京高速道路と日本橋] {{Wayback|url=https://www.nikkei.com/article/DGKKZO35240970R10C18A9L83000/ |date=20180914022852 }}『日本経済新聞』朝刊2018年9月13日（首都圏経済面）</ref><ref name="horie">{{Cite journal en|title=東京の高速道路計画の成立経緯|date=1996|journal=土木計画学研究・論文集|publisher=土木学会|doi=10.2208/journalip.13.1|volume=13|pages=1-22|author=堀江興}}</ref>。工程將汐留川、京橋川及銀座的護城河填平，並在上方興建高架高速道路及商店<ref name="TKD about">{{Cite web|url=http://www.tokyo-kousoku.co.jp/about/index.html|title=會社資訊|access-date=2017-8-27|publisher=東京高速道路|archive-url=https://web.archive.org/web/20170827091720/http://www.tokyo-kousoku.co.jp/about/index.html|archive-date=2017-08-27|dead-url=no}}</ref>。部分路段於1959年開通，全線於1966年通車<ref name="TKD history">{{Cite web|url=http://www.tokyo-kousoku.co.jp/about/history.html|title=沿革|access-date=2017-8-27|publisher=東京高速道路|archive-url=https://web.archive.org/web/20160519230951/http://www.tokyo-kousoku.co.jp/about/history.html|archive-date=2016-05-19|dead-url=no}}</ref>。

== 通行規定 ==
根據'''「道路交通法」'''，不能通行的車種如下：

* [[輕型車|輕型車]]<ref>道路交通法規定：沒有引擎的車輛</ref>
* [[迷你車|迷你車]]<ref>道路交通法規定：普通汽車</ref>
* [[摩托化自行車|摩托化自行車]]
* 容積小於125 cc的摩托車（超過125cc摩托車除[[邊車|邊車]]外不得載運超過1人<ref>{{Cite web|url=http://www.keishicho.metro.tokyo.jp/kotsu/jikoboshi/highway/regu2seat_bike.html|title=首都高速道路における自動二輪車の二人乗り規制について|access-date=2017-8-27|publisher=警視庁|archive-url=https://web.archive.org/web/20170827091113/http://www.keishicho.metro.tokyo.jp/kotsu/jikoboshi/highway/regu2seat_bike.html|archive-date=2017-08-27|dead-url=no}}</ref>）
* 特定中型汽車（總重量為8噸以上不滿11噸、最大載重量為5噸以上不滿6.5噸的貨車和[[公共小型巴士|公共小型巴士]]）
* 大型車輛

== 首都高速與東京高速道路乘繼 ==
從首都高速（收費道路）的3個乘繼所（{{link-ja|汐留乘繼所|汐留乗継所}}、{{link-ja|西銀座乘繼所|西銀座ジャンクション}}和{{link-ja|白魚橋乘繼所|白魚橋乗継所}}）進入到東京高速道路（免費）區間，若在10分鐘內持乘繼券回到首都高速就不必支付2次過路費<ref>{{Cite web|url=http://www.shutoko.jp/use/network/kk/about/|title=關於如何乘繼KK線|access-date=2017-8-27|publisher=首都高速道路|archive-url=https://web.archive.org/web/20170908071744/http://www.shutoko.jp/use/network/kk/about/|archive-date=2017-09-08|dead-url=no}}</ref>。

2013年12月15日以後，由於[[首都高速八重洲線|(Y)八重洲線]]解除長期通行禁止，乘繼的時間從30分鐘縮短至10分鐘<ref>{{Cite web|url=http://www.shutoko.jp/news/2013/data/12/02_noritsugijikan/|title=KK線乘繼時間更改（12月15日下午8時開始）|accessdate=2013-12-15|date=2013-12-02|publisher=首都高速道路株式会社|archive-url=https://web.archive.org/web/20131215023553/http://www.shutoko.jp/news/2013/data/12/02_noritsugijikan/|archive-date=2013-12-15|dead-url=no}}</ref>。裝有[[電子道路收費系統|ETC]]的車輛不用索取乘繼券，不須在乘繼所停車。

== 不動產租賃 ==
東京高速道路株式會社利用高架道路下方的空間興建14座建物（共10,000 m{{sup|2}}）並招租。這些建物大多作為商業用途及停車場<ref>{{Cite web|url=http://www.tokyo-kousoku.co.jp/estate/index.html|title=不動産賃貸事業|access-date=2017-8-27|publisher=東京高速道路|archive-url=https://web.archive.org/web/20170909172244/http://www.tokyo-kousoku.co.jp/estate/index.html|archive-date=2017-09-09|dead-url=no}}</ref>（其總部就設在其中之一的北有樂大樓2樓）<ref name="TKD accessmap">{{Cite web|url=http://www.tokyo-kousoku.co.jp/about/access.html|title=アクセスマップ|access-date=2017-8-27|publisher=東京高速道路|archive-url=https://web.archive.org/web/20170827085709/http://www.tokyo-kousoku.co.jp/about/access.html|archive-date=2017-08-27|dead-url=no}}</ref>。

由於此道路下方的建物為河川填平騰出來的土地，又其中一些河川為[[中央區_(東京都)|中央區]]、[[千代田區|千代田區]]和[[港區_(東京都)|港區]]的邊界，因此當店家想要收寄郵件、包裹時需要與面向店家入口方向的房屋共用地址<ref>{{Cite web|url=http://www.tbsradio.jp/stand-by/2011/10/post_4038.html|title=TBSラジオ 森本毅郎・スタンバイ 2011年10月27日（木）|access-date=2014-11-08|archive-url=https://web.archive.org/web/20141108053909/http://www.tbsradio.jp/stand-by/2011/10/post_4038.html|archive-date=2014-11-08|dead-url=no}}</ref>。
