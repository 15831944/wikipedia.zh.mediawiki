'''東方明珠總站'''（{{lang-pt|'''PÉROLA ORIENTAL/ TERMINAL'''}}、{{lang-en|'''Oriental Pearl Terminal'''}}）位於[[澳門|澳門]]黑沙環新區東方明珠街的一個終點站，該站是配合澳門第二階段[[巴士路線調整計劃|巴士路線調整計劃]]新增的3條巴士路線<ref>{{Cite web |url=http://www.tcm.com.mo/station2/station.asp?ST=M2 |title=澳巴網站>東方明珠總站  |access-date=2014-12-25 |archive-url=https://web.archive.org/web/20141225131448/http://www.tcm.com.mo/station2/station.asp?ST=M2 |archive-date=2014-12-25 |dead-url=yes }}</ref>而設立，現時共有3條巴士路線使用該站。<ref> [http://www.dsat.gov.mo/show_news.asp?id=633751513162#  2009年4月12日 - 交通事務局《第二階段巴士路線調整26日實施》]</ref>

== 沿革 ==
本站的設立是為配合[[澳門|澳門]][[交通事務局|交通事務局]]第二階段[[巴士路線調整計劃|巴士路線調整計劃]]新增的3條巴士路線而設置。設置目的主要是為了回應公眾對往返[[北區_(澳門)|東北區]]及[[中區_(澳門)|中區]]的巴士服務需求。

== 歷史 ==
* 2009年4月26日，本站正式投入使用。

== 設施 ==
* [[澳巴|澳巴]]服務台。

== 各車道安排 ==
每線各佔一車道。
