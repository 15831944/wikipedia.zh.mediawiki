'''東北亞非核地帶條約'''（''draft Northeast Asia Nuclear-Weapons-Free Zone Treat''） ，全名「'''東北亞非核兵器地帶條約'''」，是由[[東北亞|東北亞]]三國以非核化訂定的非核地帶（NFZ）條約案。2008年8月，由[[日本|日本]][[民主黨_(日本)|民主黨]]核軍縮促進議員聯盟正式於國內提案，呼應從2007年到2012年在[[長崎市|長崎市]]發表的「長崎和平宣言」。<ref>

{{核武器}}

== 概要 ==
[[大韓民國|大韓民國]]、[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]、[[日本|日本]]三國以[[非核三原則|非核三原則]]簽訂非核兵器地帶條約，[[中華人民共和國|中華人民共和國]]、[[俄羅斯|俄羅斯]]、[[美國|美國]]三個相關國家亦有參加。

== 目的 ==
使[[東北亞|東北亞]]成為非核地帶，減緩區域間衝突，促進世界非核地帶擴展。

== 主要內容 ==
* 位於東北亞非核兵器地帶內的國家，禁止[[核爆炸|核爆炸]]裝置的研究、開發、實驗、製作、生產、接受、保有、貯藏、配備、使用。
* 曾遭[[原子彈|原子彈]]攻擊而承受巨大傷害的城市的真相，現在與未來將世世代代傳達下去。
* 「鄰近擁有核兵器的國家」禁止於東北亞非核兵器地帶使用核兵器恐嚇。

== 特點 ==
* 以區域內國家和位於國內的他國軍事設施為針對對象。
* [[非核三原則|非核三原則]]在多國之間實踐。
* 記取[[原爆|原爆]]的慘痛教訓，訂定義務教育核軍縮。
