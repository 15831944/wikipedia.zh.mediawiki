'''東金交流道'''（平假名：とうがねインターチェンジ）是[[日本|日本]][[千葉縣|千葉縣]][[東金市|東金市]]內屬於[[千葉東金道路|千葉東金道路]]及[[首都圈中央聯絡自動車道|首都圈中央聯絡自動車道]]之[[交流道|交流道]]（圈央道東金交流道松尾横芝交流道間與千葉東金道路共線）。

== 連接道路 ==
=== 直接道路 ===
* [[國道126號|國道126號]]
* 東金市道

== 歷史 ==
*1979年3月8日 - [[千葉東金道路|千葉東金道路]][[千葉東JCT|千葉東JCT]]至'''東金交流道'''/JCT之間開通，此交流道也同時啟用。
*1998年3月30日 - 千葉東金道路東金交流道/JCT至[[松尾橫芝交流道|松尾橫芝交流道]]/JCT之間開通。
*2013年3月 - 首都圈中央聯絡自動車道東金交流道/JCT至[[木更津東交流道|木更津東交流道]]之間開通。

== 交流道周邊 ==
