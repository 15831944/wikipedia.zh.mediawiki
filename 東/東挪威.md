'''東挪威'''（[[書面挪威語|書面挪威語]]：'''Østlandet'''、[[新挪威語|新挪威語]]：'''Austlandet'''），是文化與語意上的挪威東部地域，地理上位於[[挪威|挪威]]的東南部。依[[自然地理學|自然地理學]]觀點，[[北挪威|北挪威]]才是挪威最東邊的地域。

東挪威共有八個[[挪威行政區劃#郡|郡]]，依[[ISO_3166-2:NO|ISO 3166-2:NO]]標準編號順序，包括有：[[東福爾|東福爾]]、[[奧斯陸|奧斯陸]]、[[海德馬克|海德馬克]]、[[奧普蘭|奧普蘭]]、[[布斯克魯|布斯克魯]]、[[西福爾|西福爾]]與[[泰勒馬克|泰勒馬克]]。

== 地理 ==
東挪威總面積約{{convert|94576|sqkm|sqmi|0}}<ref>{{cite web |url=http://www.statkart.no/?module=Files;action=File.getFile;ID=24831 |work=Statens kartverk |title=Arealstatistikk for 2006 |date=2006-01-01 |language=no |accessdate=2008-08-28 |archive-url=https://www.webcitation.org/68TtuyvAK?url=http://www.statkart.no/filestore/Profil/Areal_milj_og_plan_gml_dok/Arealstatistikk-2006-01-092.xls |archive-date=2012-06-17 |dead-url=yes }}</ref>，依2008年7月所公告統計結果<ref>{{cite web |url= http://www.ssb.no/folkendrkv/ |work=Statistisk Sentralbyrå |title=Kvartalsvise befolkningsendringer |language=no |accessdate=2008-08-28}}</ref>，人口共有2,380,463人。

東挪威的北部與西部滿佈山脈，北邊為[[特倫德拉格|特倫德拉格]]，西邊為[[西挪威|西挪威]]，西南邊為[[南挪威|南挪威]]，東邊與鄰國[[瑞典|瑞典]]相接，南部為'''灣區'''（{{lang-no|[[:no:Viken|Viken]]}}）與[[斯卡格拉克海峽|斯卡格拉克海峽]]為鄰。

這個地域眾多的山脈中，其中[[尤通黑門山脈|尤通黑門山脈]]（{{lang-no|Jotunheimen}}）的[[格利特峰|格利特峰]]（{{lang-no|[[:no:Galdhøpiggen|Galdhøpiggen]]}}），海拔{{convert|2469|m|ft|0}}，為[[斯堪的納維亞半島|斯堪的納維亞半島]]的最高峰，其他著名的山區還有[[多夫勒高原|多夫勒高原]]（{{lang-no|[[:no:Dovrefjell|Dovrefjell]]}}）、[[龍達訥國家公園|龍達訥國家公園]]（{{lang-no|Rondane}}）與[[哈當厄高原|哈當厄高原]]（{{lang-no|[[:no:Hardangervidda|Hardangervidda]]}}）。

山脈間有許多落差很大的山谷，由東至西有[[厄斯特達爾山谷|厄斯特達爾山谷]]（{{lang-no|[[:no:Østerdalen|Østerdalen]]}}）、[[古德布蘭德斯達爾峽谷|古德布蘭德斯達爾峽谷]]（{{lang-no|[[:no:Gudbrandsdal|Gudbrandsdal]]}}）、[[華德雷斯山谷|華德雷斯山谷]]（{{lang-no|[[:no:Valdres|Valdres]]}}）、[[侯林達山谷|侯林達山谷]]（{{lang-no|[[:no:Hallingdal|Hallingdal]]}}）、[[諾美達爾山谷|諾美達爾山谷]]（{{lang-no|[[:no:Numedal|Numedal]]}}）與[[泰勒馬克|泰勒馬克]]（{{lang-no|Telemark}}），除了厄斯特達爾山谷有較平坦的松木林外，其他都是縱深大的山谷地。

從奧斯陸峽灣周圍向東北方延申地帶，是比較平坦的陸地，散布著較多的可耕地，著名有[[海德馬克郡|海德馬克郡]]（{{lang-no|Hedmark}}）、[[奧普蘭郡|奧普蘭郡]]（{{lang-no|Toten}}）、[[費德列斯達(挪威)|費德列斯達]]（{{lang-no|[[:no:Hadeland|Hadeland]]}}）、[[靈厄里克|靈厄里克]]（{{lang-no|[[:no:Ringerike|Ringerike]]}}），全國約百分四十的人口，集中在這一帶居住，沿岸散布著許多小島，為挪威夏季的游泳與汎船勝地。

== 人文風俗 ==
[[File:Borgund.jpg|thumb]]教堂<br />攝影：Nina Aldin Thune]]
[[File:Findalslaft.jpg|thumb]]小屋<br />攝影：Arnstein Rønning]]
東挪威東南方一帶的[[方言|方言]]，彼此間擁有相同的發聲，但在[[文法|文法]]、[[語彙|語彙]]或[[讀音|讀音]]上會有一些差異；內陸山區的方言與其他方言有著明顯的差異，就連挪威國內其他地區的居民，都有理解上的困難；而沿海地帶的方言則非常地接近[[書面挪威語|書面挪威語]]。

東邊的[[芬士柯根|芬士柯根]]（{{lang-no|[[:no:Finnskogen|Finnskogen]]}}），原是少數民族的宿地，17世紀[[芬蘭人|芬蘭人]]開始移居至此，故受到[[芬蘭|芬蘭]]的語言與文化影響，特別是風俗與飲食習慣。東北邊的[[恩格達|恩格達]]（{{lang-no|[[:no:Engerdal|Engerdal]]}}）居住著挪威境內最南的一支[[薩米人|薩米人]]。

大部份山區聚落，保留著許多傳統文化，著名的有[[木架構教堂|木架構教堂]]（{{lang-no|[[:no:stavkirke|stavkirke]]}}）、[[原木小屋|原木小屋]]（{{lang-no|[[:no:lafteverk|lafteverk]]}}）、民俗樂曲與飲食；惟在滑雪度假區，因大量都市人口移入，興建了許多現代的商店、飯店或度假屋，而帶入了都市文化。

沿岸人口稠密的地區，為標準的挪威或歐洲風俗，也是[[工業化|工業化]]最早的地區，過去是以[[木材|木材]]出口與[[航運|航運]]為主，現在[[工業|工業]]部門衰退，大部份人民改為在[[服務業|服務]]導向的公司工作。沿海地區風貌多樣，從繁華的奧斯陸[[大都會|大都會]]，到寧靜賦有田園詩意的海洋城市[[德勒巴克|德勒巴克]]，也有挪威最古老的城市[[滕斯貝格|滕斯貝格]]。
