'''東京國際論壇大樓'''（{{lang-ja|東京国際フォーラム}}）是位於[[日本|日本]][[東京都|東京都]][[千代田區|千代田區]][[丸之內|丸之內]]三丁目的[[會議展覽中心|會議展覽中心]]，由株式會社東京國際論壇大樓（完成當時為[[財團法人|財團法人]]東京國際交流財團）營運，是公共綜合文化設施，也是[[東京|東京]]的主要國際會議場所之一。

東京國際論壇大樓原為舊[[東京都廳舍|東京都廳舍]]所在地，在都廳搬遷至[[新宿副都心|新宿副都心]]後開始改建，鄰近[[有樂町站|有樂町站]]、[[京葉線|京葉線]][[東京站|東京站]]等[[鐵路車站|鐵路車站]]。整個設施分為7個會堂、展覽廳、33個會議室、商店、餐廳、[[相田光男|相田光男]]美術館、[[太田道灌|太田道灌]]雕像等，供各種節目及展覽等活動舉行。

[[2020年夏季奧林匹克運動會舉重比賽|2020年夏季奧林匹克運動會舉重比賽]]將於此舉行<ref>{{cite web|title=Venue Plan|url=http://tokyo2020.jp/en/plan/venue/index.html|publisher=Tokyo 2020 Bid Committee|accessdate=11 September 2013|deadurl=yes|archiveurl=https://web.archive.org/web/20130727040849/http://tokyo2020.jp/en/plan/venue/index.html|archivedate=27 July 2013|df=}}</ref>。
== 建築 ==
以[[國際建築師協會|國際建築師協會]]（UIA）之基準為基礎舉行國際公開比賽，是日本首次採用此種方式進行競圖。

勝出設計者為[[建築師|建築師]]{{link-en|拉斐爾·維諾利|Rafael Viñoly}}。[[玻璃|玻璃]]建成的大堂（玻璃棟）以[[船|船]]為題材，巨大壯觀的外觀成為建築物的象徵，與已搬遷的新都廳同樣被批評為「[[泡沫經濟|泡沫經濟]]之遺產」、「税金的浪費」。

=== 概要 ===
* 設計者：{{link-en|拉斐爾·維諾利|Rafael Viñoly}}（Rafael Viñoly）
* 構造設計：{{link-ja|渡邊邦夫|渡辺邦夫}}
* 設備設計：{{link-ja|森村設計}}
* 所在地：[[東京都|東京都]][[千代田區|千代田區]][[丸之内|丸之内]]3-5-1
* 開館日：1997年1月10日
* 層數：地上11層、地下3層
* 建築面積：21,000[[平方公尺|m<sup>2</sup>]]
* 建物延床面積：145,000m<sup>2</sup>
* 總工費：1,647億[[日圓|日圓]]（用地費除外）

=== 各設施 ===
; A廳
: 成為全球屈指可數的室內音樂廳，客席數為2層構造5,012席。8種語言同聲傳譯系統、AV網路、音響設備、舞台結構等設備完善。
; B7廳
: 位於B地段最上部7・8層，面積1,400m<sup>2</sup>平土間形式綠地。亦可將可動式間壁分割使用。擁有設備完善的8種語言同聲傳譯系統等。
; C廳
: 音樂會專用廳，客席數為3層構造1,502席。Concert Hall Shaver及[[混響|混響]]可變裝置等，最新音響設備完善。
; D7廳
: 採用抽屜式座椅等之多用途大廳。

== 營運 ==
2003年7月，為提高民間企業的資金、人材、知識等的積極利用，及為提高收益及經營效率，由財團法人東京國際交流財團轉移至「第三類法人株式會社東京國際論壇大樓」。
