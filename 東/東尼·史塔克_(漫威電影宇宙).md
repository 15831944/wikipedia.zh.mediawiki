'''安東尼·愛德華·史塔克'''（{{lang-en|Anthony Edward Stark}}），別稱'''[[鋼鐵人|鋼鐵人]]'''，是基於同名[[漫威漫畫|漫威漫畫]][[角色|角色]]，在[[漫威电影宇宙|漫威电影宇宙]]（MCU）系列電影中由[[小勞勃·道尼|小勞勃·道尼]]飾演的角色。電影中，東尼·史塔克是一名[[企業家|企業家]]、天才發明家兼極度自戀的花花公子，擔任{{tsl|en|Stark Industries|史塔克工業}}執行長。他在電影最初作為[[美军|美军]]的主要武器供應商，隨後，為了保衛世界和平，運用其技術知識發展[[鋼鐵人動力服|動力服]]。{{As of|2018}}，這個角色曾在九部系列電影中登場（包含在《[[無敵浩克|無敵浩克]]》的短暫客串），是漫威電影宇宙的核心人物之一。

作為其中一位受集結的英雄，史塔克在《[[復仇者聯盟|復仇者聯盟]]》中共同抵禦由邪神[[洛基_(漫威漫畫)|洛基]]領軍的外星人入侵地球。最初，道尼希望導演[[喬斯·溫登|喬斯·溫登]]讓史塔克在這部電影中擔任主角：「當時，我提到『我不知道你有什麼打算，但是我必須讓東尼在電影片頭登場，主導全局。』導演回覆說『好吧，讓我們試試看。』我們嘗試過後成效不彰，因為故事、想法、主題各自就像章魚的每一條腕一樣，完全是不同的思路。」<ref name="EW4">{{cite web |last=Breznican |first=Anthony |date=2011-09-29 |title='Avengers': Thor, Hulk, Iron Man, Captain America — EXCLUSIVE PICS |url=https://www.ew.com/article/2011/09/29/the-avengers-dis-assembled-exclusive-cast-portraits-revealed/ |work=[[娱乐周刊|Entertainment Weekly]] |accessdate=2011-07-29 |archiveurl=https://www.webcitation.org/624KaS4rJ?url=http://insidemovies.ew.com/2011/09/29/the-avengers-dis-assembled-exclusive-cast-portraits-revealed/ |archivedate=2011-07-29 |deadurl=no}}</ref>關於這個角色歷經前幾部電影的轉變，道尼表示「他在《鋼鐵人》中自我領悟並獲得救贖，《鋼鐵人2》則在於脫離自我孤立、解決遺留的問題並為他人騰出空間，最終在《復仇者聯盟》與大夥結盟一同作戰。」<ref name="Filmink">{{cite journal |last=Adamek |first=Pauline |date=January–February 2012 |title=Avengers Assemble! |journal=Filmink |pages=70–75 |publisher=FKP International Exports}}</ref>到了電影高潮片段，史塔克引導一枚核彈穿越星際傳送門，摧毀了外星人的指揮船艦，展現出願意犧牲自我以拯救地球的決心。

==與漫畫的差異==
漫威電影宇宙中的鋼鐵人在許多細節上與漫畫版本有所差異。在電影中，史塔克在[[阿富汗|阿富汗]]經歷一場實驗後成為鋼鐵人，漫畫則是[[越南|越南]]進行的。<ref name="Rant-Erao">{{cite web |url=https://screenrant.com/avengers-movies-compared-to-comics-characters/ |title=The Avengers: How 15 Movie Heroes Compare To The Comics |first=Matthew |last=Erao |publisher=Screen Rant |date=2017-01-06 |deadurl=no |archiveurl=https://web.archive.org/web/20180114020419/https://screenrant.com/avengers-movies-compared-to-comics-characters/ |archivedate=2018-01-14 |df= }}</ref>[[艾德恩·賈維斯|賈維斯]]在漫畫中是一名管家，在電影中，史塔克則是受到童年管家的啟發，<ref name="Rant-Erao"/>設計出人工智慧「賈維斯」（J.A.R.V.I.S.）。<ref name="Beat 2014">{{cite web |url=http://www.comicsbeat.com/iron-man-in-comic-vs-movies/ |title=Iron Man in Comic vs Movies |publisher=Comics Beat |date=2014-08-31 |deadurl=no |archiveurl=https://web.archive.org/web/20180113150308/http://www.comicsbeat.com/iron-man-in-comic-vs-movies/ |archivedate=2018-01-13 }}</ref>史塔克在早期便將其裝甲漆成紅色及金色版本。<ref name="Rant-Erao"/>

人工智慧賈維斯最終被史塔克植入一具人造軀體，成為[[幻視_(漫畫)|幻視]]，以對付由史塔克及[[浩克_(漫畫)|布魯斯·班納]]創造出的[[奧創|奧創]]；不同的是，漫畫版本的幻視是由奧創獨自創造，奧創則為另一名復仇者[[漢克·皮姆|漢克·皮姆]]創造。<ref name="Rant-Browne">{{cite web |url=https://screenrant.com/iron-man-worst-changes-from-the-comic-books-to-movies/ |title=Iron Man: 15 Worst Changes From The Comic Books To The Movies |first=Ben |last=Browne |publisher=Screen Rant |date=2017-01-18 |deadurl=no |archiveurl=https://web.archive.org/web/20180114020316/https://screenrant.com/iron-man-worst-changes-from-the-comic-books-to-movies/ |archivedate=2018-01-14 }}</ref>史塔克在電影中與{{tsl|en|Pepper Potts|小辣椒·波茲}}交往，然而，漫畫中的波茲則是單戀著史塔克，最終與史塔克的保鑣{{tsl|en|Happy Hogan (comics)|快樂·霍根 (漫威漫畫)|快樂·霍根}}交往。<ref name="Beat 2014"/><ref name="Rant-Browne"/>

史塔克遭遇的反派大多僅於一部電影登場便遭解決，不同的是，像是[[鐵霸王|奧比戴亞·史坦]]及[[滿大人_(漫威漫畫)|滿大人]]等漫畫角色則是長期對抗的敵人。滿大人在電影中以演員的身分登場，其背後主謀為{{tsl|en|Aldrich Killian|奧德利奇·齊禮安}}，然而，齊禮安一角在漫畫中卻不是一名重要反派。<ref name="Rant-Browne"/><ref>{{cite web |url=http://www.mtv.com/news/1706745/iron-man-3-comic-books/ |title=Does 'Iron Man 3' Stay True To The Comic Books? |publisher=MTV News |first=Kevin |last=Sullivan |date=2013-05-02 |deadurl=no |archiveurl=https://web.archive.org/web/20180113203029/http://www.mtv.com/news/1706745/iron-man-3-comic-books/ |archivedate=2018-01-13 }}</ref>
