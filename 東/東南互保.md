'''東南互保'''，或作'''東南自保'''，指[[清末|清末]][[八國聯軍|八國聯軍]]時大清東南各[[行省|行省]][[督撫|督撫]]不理會朝廷命令，不與[[列強|列強]][[宣戰|宣戰]]，避免與大清一起陷入戰爭局面。。

[[慈禧太后|慈禧太后]]向八國宣戰後，[[兩廣總督|兩廣總督]][[李鴻章|李鴻章]]、[[兩江總督|兩江總督]][[劉坤一|劉坤一]]、[[湖廣總督|湖廣總督]][[張之洞|張之洞]]、[[閩浙總督|閩浙總督]][[許應騤|許應騤]]、[[山東巡撫|山東巡撫]][[袁世凱|袁世凱]]等人，即和外國達成和平協議，他們在東南各省違抗“支持義和團”的命令。他們的理由是清廷的[[宣戰|宣戰]]詔書，是義和團「[[矯詔|矯詔]]」，脅持朝廷所下的“亂命”。

== 事发经过 ==
其實，[[太平天国|太平天国]]后，[[两江|两江]]、[[闽浙|闽浙]]等地區已非[[皇室|皇室]]势力所及，实为[[湘军|湘军]]、[[淮軍|淮軍]]系统把持。（参见[[刺马案|刺马案]]）

[[庚子拳亂|庚子拳亂]]發生後，在[[清廷|清廷]]尚未向各國[[宣戰|宣戰]]時，[[兩江總督|兩江總督]][[劉坤一|劉坤一]]（[[湘军|湘军]][[江忠源|江忠源]]旧[[将|将]]）、[[湖廣總督|湖廣總督]][[張之洞|張之洞]]（與[[湘軍|湘軍]]、[[淮軍|淮軍]][[關係|關係]]俱佳）、[[兩廣總督|兩廣總督]][[李鴻章|李鴻章]]（淮軍[[領導人|領導人]]）、[[礦務鐵路總局|鐵路大臣]][[盛宣懷|盛宣懷]]（[[父亲|父亲]][[盛康|盛康]]是湘军[[胡林翼|胡林翼]]的[[军需官|军需官]]，盛宣懷自己與李鴻章關係甚佳）、[[山東巡撫|山東巡撫]][[袁世凱|袁世凱]]（淮軍[[吳長慶|吳長慶]]舊部）、[[閩浙總督|閩浙總督]][[許應騤|許應騤]]（族叔[[许祥光|许祥光]]曾与湘军[[江忠源|江忠源]]一起镇压[[太平天国|太平天国]]）等即商議如何保存[[中国东部|東南]]的穩定，避免[[列強|列強]]有藉口入侵；同時密議盤算倘若[[北京|北京]]失守而[[光绪帝|光绪帝]]、[[慈禧太后|慈禧太后]]不測，當改建[[共和|共和]]，由李鴻章作[[大總統|大總統]]支撐局面。

1900年6月21日，清廷以[[光緒帝|光緒帝]]名義，向[[大不列顛及愛爾蘭聯合王國|英]]、[[美國|美]]、[[法蘭西第三共和國|法]]、[[德意志帝國|德]]、[[義大利王國_(1861年–1946年)|-{zh-hans:意; zh-hant:義;}-]]、[[日本帝國|日]]、[[俄罗斯帝国|俄]]、[[奥匈帝国|奧]]八國同時宣戰{{NoteTag|實際上皇室只曾向清室臣下下達「宣戰詔書」，而該詔書未曾按現代外交慣例，交予外國使節。}}。

當時宣戰詔書下至各地方時，鐵路大臣盛宣懷下令各地電信局將清廷召集義和團民及宣戰詔書扣押，只給各地督撫觀看，並且電告各地督撫，不要服從此命令。[[张謇|张謇]]、[[赵凤昌|赵凤昌]]等人分別勸告两江总督[[刘坤一|刘坤一]]、湖广总督[[张之洞|张之洞]]、两广总督[[李鸿章|李鸿章]]，倡议抗命，李鴻章覆電朝廷：「此亂命也，粵不奉詔」<ref>{{Wikisource-inline|庚子國變記 (羅惇曧)}}</ref>，此一電報也鼓勵了東南各省督撫。時局日益糜爛，湖廣總督張之洞隨後提出「李鴻章大總統」方案：一旦北京不保，就共同推舉李鴻章出任中國「總統」以主持大局<ref>{{cite book |author=张社生 |title=绝版李鸿章 |url = http://books.google.com/books?id=C4gQPwAACAAJ |year=2009 |location=上海 |publisher=文汇出版社 |ISBN = 978-7-80741-428-5 }}</ref>，李鴻章[[幕僚|幕僚]][[劉學詢|劉學詢]]並去信[[孫中山|孫中山]]謂：「傅相{{NoteTag|[[李鴻章|李鴻章]]有「[[大學士|大學士]]」、「[[太子太傅|太子太傅]]」銜，被尊稱為「傅相」。}}因北方拳亂，欲以粵省獨立，思得足下為助，請速來粵協同進行。」<ref>{{cite book |author=陳錫祺 |title=孫中山年譜長編 |volume=上冊 |url = http://books.google.com/books?id=pLlTAAAAIAAJ |year=1991年 |page = 第206页 |location=北京 |publisher=中华书局 |ISBN = 9787101006858 }}</ref><ref>{{cite journal |url = http://news.xinhuanet.com/mrdx/2013-11/08/c_132870676.htm |title = 李鴻章與兩廣獨立 |publisher = [[新華每日電訊|新華每日電訊]] |author=唐荒 |journal=《新歷史》 |year = 2009年 |issue = 第2期 |accessdate = 2009-10-04 |language = zh-cn }}</ref><ref>{{cite book |editor=羅家倫 |publisher=[[中國國民黨黨史史料編纂委員會|中國國民黨黨史史料編纂委員會]] |title=革命文獻 |volume=64 |url = http://books.google.com/books?id=o3EwAAAAMAAJ }}</ref>

劉坤一、張之洞於6月26日，以[[上海道|上海道]][[道員|道員]][[余联沅|余联沅]]為代表，邀約各国駐[[上海|上海]]領事商訂所謂《[[東南保護約款|東南互保條約]]》九條。主要內容為：

一、上海租界归各国共同保护，长江及苏杭内地均归各督抚保护，两不相扰，以保全中外商民人命产业为主。

二、上海租界共同保护章程，已另立条款。

三、长江及苏杭内地各国商民教士产业，均归南洋大臣刘坤一、两湖总督张之洞切实保护，并移知各省督抚及严饬各该文武官员一律认真保证。现已出示禁止谣言，严拿匪徒。

四、长江内地中国兵力已足使地方安静，各口岸已有的外国兵轮者仍照常停泊，惟须约束人等水手不可登岸。

五、各国以后如不待中国督抚商允，竟至多派兵轮驶入长江等处，以致百姓怀疑，借端启衅，毁坏洋商教士的人命产业，事后中国不认赔偿。

六、吴淞及长江各炮台，各国兵轮不可近台停泊，及紧对炮台之处，兵轮水手不可在炮台附近地方练操，彼此免致误犯。

七、上海制造局、火药局一带，各国允兵勿往游弋驻泊，及派洋兵巡捕前往，以期各不相扰。此军火专为防剿长江内地土匪，保护中外商民之用，设有督巡提用，各国毋庸惊疑。

八、内地如有各国洋教士及游历洋人，遇偏僻未经设防地方，切勿冒险前往。

九、凡租界内一切设法防护之事，均须安静办理，切勿张皇，以摇人心<ref>摘录自：盛宣怀的《愚斋存稿》</ref>

随后，劉坤一、張之洞、李鴻章、閩浙總督許應騤、山東巡撫袁世凱、浙江巡撫劉樹棠、安徽巡撫王之春和廣東巡撫德壽（後李鴻章北上議和，由德壽[[政治代理|署理]][[兩廣總督|兩廣總督]]），即與各参战國達成協議，稱'''東南互保'''，另外[[陝西巡撫|陝西巡撫]][[端方|端方]]、[[四川總督|四川總督]][[奎俊|奎俊]]雖然沒有加入東南互保，但是亦是支持東南互保。他們稱皇帝敕令是義和團拳民脅持下的“矯詔、亂命”，“亂命”一詞是李鴻章精心選擇的政治術語，李鴻章深知“若不量力而輕於一試，恐數千年文物之邦，從此已矣”，故稱“此亂命也，粵不奉詔。”，在東南各[[行省|行省]]違抗「支持義和團」的命令，以為「[[拳民|亂民]]不可用、[[邪術|邪術]]不可信，[[戰爭|兵釁]]不可開」「無論北方情形如何，請列國勿進兵長江流域與各省內地；各國人民生命財產，凡在轄區之內者，決依條約保護。」

== 影响 ==
东南互保，是清末各地官方实力派人物首次公然聚众反抗[[朝廷|朝廷]]，此事件虽使[[河北|河北]]、[[山西|山西]]以外的地区得以免于[[义和团运动|义和团运动]]及[[八国联军|八国联军]]战乱的波及，但同时此事也使得清廷威信扫地：東南眾[[大臣|大臣]]抵制的慈禧太后，事後不但不敢處罰他們，甚至還表扬他們“度势量力，不欲轻构外衅，诚老成谋国之道”<ref>《义和团档案史料》上，187</ref>，[[太后|太后]]與[[皇室|皇室]]的虚弱乏力表露无遗；而各省亦因[[八国联军|八国联军]]入侵而自觉需有自卫的武力需要，因此清廷治下各省及各实力人物由此开始[[军阀|军阀]]化（典型例子即为[[淮军|淮军]]发展而来的袁世凯系[[北洋军阀|北洋军阀]]诸部及原清朝[[广西|广西]][[绿营|绿营]]及[[巡防营|巡防营]]发展而来的[[旧桂系|旧桂系]][[陆荣廷|陆荣廷]]部），至[[辛亥革命|辛亥革命]]时，各省外籍主官均被驱逐，即为此军阀化过程的结束，并由此展开[[民国军阀|民国军阀]]时代的序幕。
