'''東京都交通局12-000型電車電力動車組'''（{{jpn|j=東京都交通局12-000形電車|hg=とうきょうとこうつうきょく12-000がたでんしゃ|rm=Tōkyōto kōtsūkyoku 12-000 data densha|f=y}}）是[[東京都交通局|東京都交通局]]（[[都營地下鐵|都營地下鐵]]）[[大江戶線|大江戶線]]用的[[直流電|直流]][[通勤型電力動車組|通勤型電動列車]]，並於1991年12月正式投入服務<ref name="railfan579">{{cite journal| last = Kajiwara| first = Hisashi| title = 公営地下鉄在籍両数ビッグ3 東京都交通局|trans_title= Top 3 Subway Vehicle Owners: Toei| journal = [[铁道爱好者_(杂志)|Japan Railfan Magazine]] | volume = 49| issue = 579| page = p.128–139| publisher = Kōyūsha| location = Japan | date = July 2009| url = | doi = | id = | accessdate = }}</ref>。這款電動列車由[[日本車輛製造|日本車輛]]、[[日立制作所|日立制作所]]及[[東急車輛製造|東急車輛公司]]負責製造。

== 概况 ==
本款列车共分4批，截止至2009年4月，一共有53组共424辆正在运营中。尽管列车已经充分利用[[车辆限界|车辆限界]]，不过仍较其他车型相比显得狭小许多。列车两端拥有[[紧急出口|紧急出口]]，每节车厢拥有3对车门，车门为[[内崁式门|内崁式门]]。采用[[直线电动机|直线电动机]]进行驱动，是[[都营地下铁|都营地下铁]]中第1款使用[[直线电动机|直线电动机]]的列车。其中直线电动机为TLIM-12型，功率120kW，额定电压1,100V，额定電流170A，额定频率21Hz。

== 车辆配置 ==
=== 样本车 ===
==== 设计背景 ====
1986年3月，由[[东急车辆制造|东急车辆制造]]制作的2辆样本车，本列车是在[[大江户线|大江户线]]的隧道规格作出修改后，进行试制的。列车在3月25日运送至马込车辆检修场进行试验，本批列车遵循如下设计要求：
* 21世纪通用的车辆
* [[节约能源|节约能源]]、减少维护
* 引进最新的技术对车辆系统集成化
* 车体、机器的小型轻量化
==== 列车外观 ====
列车车身采用[[不锈钢|不锈钢]]车体，列车两端拥有[[紧急出口|紧急出口]]，其车门为[[内崁式门|内崁式门]]。[[前照灯|前照灯]]位于驾驶室玻璃下方，[[尾灯|尾灯]]位于驾驶室玻璃上方。列车[[方向幕|方向幕]]使用[[液晶显示器|液晶显示器]]，可显示红、白、蓝三色，本列车车头形状和后期生产的列车有很大的不同。

==== 列车设备 ====
列车控制装置为[[可关断晶闸管|GTO]]-[[VVVF|VVVF]]，为[[都营地下铁|都营地下铁]]中首次采用，并安装有监视设备。制动方式为[[踏面制动|踏面制动]]。电动机采用单机功率为120kW的[[鼠笼式电动机|鼠笼式电动机]]，其粘附性能较其他旋转电机有所提高。为了确保列车的[[起动加速度|起动加速度]]为3.0km/h/s，列车的[[动拖比|动拖比]]设置为了1:1。<br />列车转向架采用无摇枕转向架，由[[近畿车辆|近畿车辆]]制造，为桶形多层结构并附带橡胶套管，以支持轴箱系统<ref>{{lang|ja|この軸箱方式の台車は、ボルスタ付ではあるが、後の[[浅草線|浅草線]]用[[東京都交通局5300型電力動車組|5300形]]や[[三田線_(都營地下鐵)|三田線]]用[[東京都交通局6300型電力動車組|6300形]]においても採用されている}}。</ref>。其中动力转向架为T-12X型（厂家形式为KD91型），无动力转向架为T-12Y型（厂家形式为KD91A型）。为容纳列车的低地板要求，列车的转向架采取小型轻量化设计，车轮轮径为660mm。<br />列车辅助电源装置采用了电容为100kVA的静止型逆变器，[[集电弓|集电弓]]则分别使用了菱形和Z型进行测试，最终确认最低折叠高度为170mm。列车搭载有[[自動列車控制裝置|ATC]]和[[列車自動運行系統|ATO]]设备。<br />由于[[车辆限界|车辆限界]]的限制，列车的空调无法完全安装在车顶，而是安装在了地板上。其中空调单机功率为14.53kW，每节车厢各有2台，列车的空调控制室在列车端头。这种空调设置模式参考了[[都营地下铁|都营地下铁]][[东京都交通局10-000型电力动车组|10-000型]]的方式，不过[[东京都交通局10-000型电力动车组|10-000型]]只有样板车如此采用，后期的列车并未如此采用。

==== 马込车辆检修场的试验 ====
本批列车在1986年5月-9月在[[马込车辆检修场|马込车辆检修场]]进行旋转电机驱动试验，为适应运营要求，厂区内将10号股道作为列车试车线，其中当时的300米延长线在目前已经作为新的检修场用地，另外还专门布置了刚性接触网区段。在[[浅草线|浅草线]]的末班车运营结束后，列车开始在[[浅草线|浅草线]]的[[户越站|户越]]-[[西马込站|西马込]]区段内进行试验<ref>{{lang|ja|浅草線で試運転を行ったことは「東京都交通局80年史」ならびに「大江戸線放射部建設史」という東京都交通局発行の文献において公式に記載されている}}。</ref>。其行驶速度由在车辆基地内的限速40km/h提升至最大时速70km/h，以对列车的振动、噪音等方面进行考核，最后经确认反映良好。另外，本车还一度计划在[[大江户线|大江户线]]投入运营后继续行驶<ref>{{lang|ja|鉄道ジャーナル社「鉄道ジャーナル」1986年8月号89ページ記事参照}}。</ref>。

==== 驱动装置改造 ====
在1987年3月开始讨论了使用[[直线电动机|直线电动机]]的可行性，并建议制造直线电机列车作为运营车辆的可能。1987年6月，地下鉄12号線建設推進本部決定了12號線([[大江户线|大江户线]])的列车以直线电机作驱动。所以因此，[[东京都交通局|东京都交通局]]作出了对样本车的驱动装置改造决定，并从安全性、经济性方面进行确认。驱动方式改造试验由社团法人日本地铁协会承担。并在VVVF、制动装置、驱动装置进行改装，加装了电磁吸附装置<ref name="TOKYO">东京都交通局「大江戸線放射部建設史」参照。</ref>。<br />列车改装后，装备的新型[[直线电动机|直线电动机]]采用的输出功率为120kW，12-001号车厢也进行了相关改造。另外，该款[[直线电动机|直线电动机]]分为两种方式加装：一种装备在转向架下端，另一种装备在主轴上。另外车轮轮径更改为610mm，由[[住友金属工业|住友金属工业]]和[[日本车辆制造|日本车辆制造]]负责。

==== 直线电机试验 ====
在列车驱动装置改造后，为适应列车的新的试验要求。[[马込车辆检修场|马込车辆检修场]]专门在部分试车线铺设了感应板，并在1988年4月-11月进行了试验，由于反应良好。故在1988年12月21日正式决定在[[大江户线|大江户线]]全线铺设感应板。
==== 列车的停用 ====
本车的诸多性能在后期制造的4批列车中均有所体现，不过由于应用价值已经达到，列车最终于1990年停用，后于1991年2月送至[[千早花公园|千早花公园]]进行[[静态保存|静态保存]]<ref name="railfan617">{{cite journal| last = Okada| first = Seiichi| title = 魅惑の公園保存車両 4|trans_title= Fascinating park-preserved rolling stock (Part 4)| journal = [[铁道爱好者_(杂志)|Japan Railfan Magazine]] | volume = 52| issue = 617| page = p.96–97| publisher = Kōyūsha Co., Ltd.| location = Japan | date = September 2012}}</ref>。

=== 第1批 ===
==== 投入使用 ====
1990年9月至10月，由[[日本车辆制造|日本车辆制造]]制造的第1批列车生产完毕，本批列车共5组30辆。列车在1990年10月至1991年10月期间于[[大江户线|大江户线]][[光丘站|光丘]]-[[练马站|练马]]之间进行高速行驶试验，以确认其整体性能和安全性能<ref>{{lang|ja|これは馬込検車場内でのリニアモーター試験は試験線の都合上、速度は40km/hに抑えられ、高速走行時の安全性などが確認できないためであった。回転形モーター車では浅草線を走行することは可能であるが、リニアモーター車両ではリアクションプレートを敷設しない限り、浅草線本線の走行は不可能である（大江戸線放射部建設史参照）}}。</ref>。并最终于1991年12月正式投入服务。列车在1998年全部扩编为8编组。
[[File:Model_12-000_of_Toei_Transportation.jpg|300px]]]]
==== 列车外观 ====
本批列车采用轻量化[[铝合金|铝合金]]设计，外涂装为白色。车头采用流线型设计，[[紧急出口|紧急出口]]设置在列车行进方向左侧。[[前照灯|前照灯]]设置于车窗下方，[[尾灯|尾灯]]设置在[[方向幕|方向幕]]两端。
==== 列车客室 ====
列车客室内壁为白色，地板在中央过道内为米黄色，在两端则为茶色。列车的空调设备与样本车不同，采用超薄型空调，并安装在天花板上，以确保车内的净空高度能达到至少2,100mm。车门高度为1,850mm，车门开度达1,300mm。行李架采用[[不锈钢|不锈钢]]焊接而成。而且自第4批车开始，在地板上设置了电动机检查盖。列车座位的普通席颜色为粉红色，优先席为蓝色。由于列车全线运行于地下区间，所以不设置窗帘。另外，列车的贯通门的玻璃也较其他车门的大。<br />列车轮椅位设置于4号车厢，旁边还设置有安全扶手和[[紧急警报装置|紧急警报装置]]（每节车厢设置6套）。在紧急状态按下按钮之后，可与司机进行无线通话。车门一侧上方安装有LED显示幕，另一侧安装有[[路线图|路线图]]<ref>該系統與[[帝都高速度交通营团|帝都高速度交通营团]]（現[[東京地下鐵|東京地下鐵]]）[[銀座線|銀座線]][[營團01系電力動車組|01系]]、[[丸之内線|丸之内線]][[營團02系電力動車組|02系]]的設置相似。</ref>。
==== 驾驶室 ====
列车驾驶室进深为2,200mm，由于[[大江户线|大江户线]]的所有车站月台均为[[岛式月台|岛式月台]]结构，故列车驾驶台设置于驾驶室行进方向的右侧。驾驶室与客室之间的门采用电磁技术锁闭。<br />驾驶室内壁颜色为白色，驾驶台为深蓝色，拥有无线广播、数字速度计、压力表和电压表。列车操纵杆为手柄式，分为牵引4段+制动7段+紧急制动。其中还设置有[[抬头显示器|抬头显示器]]（由于容易和外界图像混淆，故在第2批列车以后不再设置）<ref name="RW-P">「铁道画刊」2001年7月号臨時増刊号参照。</ref>。<br />列车驾驶室的[[红外线|红外线]]监控系统可通过无线方式传送至车站或控制中心。此外列车采用[[车辆信息控制装置|车辆信息控制装置]]一体化控制模式，信息显示器采用触屏模式。[[车内向导表示器|车内向导表示器]]、[[方向幕|方向幕]]、[[自动广播装置|自动广播装置]]可以自动设定<ref name="TOKYO" />。
==== 安装设备 ====
由于列车轮径为610mm，列车地板高度为800mm，使得地板之下的机器安装高度需要控制在500mm以内。控制装置是由[[日立制作所|日立制作所]]制造的[[可关断晶闸管|GTO]]-[[VVVF|VVVF]]（T-INV12型），其中1、2批次列车的[[磁励音|磁励音]]与[[大阪市营地下铁|大阪市营地下铁]][[長堀鹤见绿地线|長堀鹤见绿地线]]的[[大阪市交通局70系电力动车组|70系]]相似。<br />列车的[[压缩机|空气压缩机]]采用交流驱动形式，在3、6号车厢各搭载2台。辅助电源装置材料为[[晶体管|晶体管]]，采用[[靜式變流器|靜式變流器]]，供应至分散搭载在第3、6车厢的空调（35kVA）、[[压缩机|空气压缩机]]（23kVA）、控制设备（25kVA）。为适应[[刚性接触网|刚性接触网]]的要求，列车的[[集电弓|集电弓]]的最小折叠高度为160mm。<br />列车采用具有[[空气弹簧|空气弹簧]]的[[转向架|转向架]]，能够在曲线区段安全通过。第1批次的列车曾在[[马込车辆检修场|马込车辆检修场]]进行转向架比较试验，不过规格均为轴距1,900mm，制动方式均为[[碟式制动|碟式制动]]。
;第1批次列车的转向架配置情况：
* T-12D型（[[住友金属工业|住友金属工业]]制FS545D型）第05编成转向架装载
* T-12S型（[[住友金属工业|住友金属工业]]制FS540型）第01、04编成主轴承负载
* T-12N型（[[日本车辆制造|日本车辆制造]]制ND325型）第02、03编成主轴承负载

T-12D型采用外轴箱方式，但T-12S型和T-12N型采用内軸箱方式。其中内轴箱的轻量化采用，在日本国内较为罕见<ref>{{lang|ja|日本国内における鉄道車両は[[窄轨|狭軌]]車両が主流であり、狭軌車両に内軸箱方式を採用することは主電動機等の取り付けの関係から難しい。ただし、大江戸線のような[[標準軌|標準軌]]であればスペースに余裕があることから採用はしやすい}}。</ref>。[[碟式制动|碟式制动]]刹车片安装在外侧也是其中的明显特征之一。在[[大江户线|大江户线]]通车后，列车在行进在曲线区段时，经常会发生异常大的噪音。经调查，这与
T-12D型的负载过大有关，因此在后期便逐渐以T-12S、T-12N型来取代现有的T-12D型。<br />列车内部搭载有[[列车自动控制系统|ATC]]和[[列车自动运行系统|ATO]]，并采用[[模糊控制|模糊控制]]，提高列车的驾驶和停靠精度。制动方式采用[[电气指令式制动|电气指令式制动]]和[[再生制动|再生制动]]，制动段数分为7段+紧急制动，在[[列车自动运行系统|ATO]]状态下可达到31段，[[紧急制动|紧急制动]]情况下会由[[再生制动|再生制动]]承担一部分。

=== 第2批 ===
==== 投入使用 ====
由于部分第1批次列车开始进行检修，为确保列车的正常运营，在1994年12月，由[[日本车辆制造|日本车辆制造]]生产的一组6节列车开始运至[[光丘车辆检修场|光丘车辆检修场]]，并于1995年2月投入使用，作为备用车辆（[[光丘站|光丘]]-[[练马站|练马]]之间只需4组列车，1组备用车即可）。列车在1998年扩编为8编组。
==== 列车特性 ====
本批次列车在外形上与第1批次列车相似。其中[[排障器|排障器]]面积较第1批次有所增大，车钩也有所修改。转向架则全部采用了T-12D型。列车客室的[[荧光灯|荧光灯]]照度较之前有所增加，车厢之间的贯通通道变得较之前狭窄。自本批列车开始，列车的英语广播开始改为自动播放。列车的驾驶台外观与第1批次和之后的第3、4批次有很大不同。

=== 第3批 ===
==== 投入使用 ====
为应对[[练马站|练马]]-[[新宿站|新宿]]区间的开通。在1997年3月至9月，第1207-1215编成的共72辆列车及第1、2批次的12辆扩编用车厢运送至[[光丘车辆检修场|光丘车辆检修场]]，这些列车由[[日本車輛製造|日本車輛製造]]和[[日立製作所|日立製作所]]负责制造<ref>{{lang|ja|一部の雑誌では全車両が日立製作所製となっているが これは誤りで、第07 - 12編成と中間増備車は日車、それ以外が日立製なのが正しい}}。</ref>。最初的第1207-1212编成开始以8编组形式于5月25日开始投入运营<ref>{{lang|ja|これは当時の練馬 - 光が丘間の最大運用本数が4本と予備2本のため、同数の3次車で1・2次車を一時的に編成替えのため置き換えるためであった}}。</ref>。在此期间，第1、2批次列车开始进行了扩编工程。直至扩编结束后，已开通的线路区间可容纳12组列车、3组备用车。
==== 不同点 ====
第3批次列车有如下变更：
* 列车外涂装采用无涂装形式
* 驾驶室面积变大，列车车头形状改变
* 改为使用[[IGBT|IGBT]]-[[VVVF|VVVF]]
* 节约辅助电源装置
* [[自動列車控制裝置|ATC]]、[[列車自動運行系統|ATO]]装置作出修改

此外，列车的[[磁励音|磁励音]]较之前的批次有所降低，其[[IGBT|IGBT]]-[[VVVF|VVVF]]为T-INV12A型，空调控制装置为集中化设计。
==== 列车客室 ====
客室内部的样式与第2批次相同，行李架取消了钢管模式，改为了铁丝网模式。[[紧急警报装置|紧急警报装置]]也由6套缩减至了3套，轮椅位设置在第4、5节车厢。
==== 造价 ====
第1批次列车的每节车厢制造费用约1亿5,300万日元，第2批次列车约1亿7,200万日元。第3批次列车约1亿1万日元，较之前的列车节约了不少成本<ref name="TOKYO" />。

=== 第4批 ===
==== 投入使用 ====
为应对全线区间的开通。在1999年2月22日和2000年2月17日，第1216-1253编成的共304辆列车运送至[[木场车辆检修场|木场车辆检修场]]，这些列车中的第1216-1228编成由[[日本車輛製造|日本車輛製造]]负责制造，第1229-1253编成由[[日立製作所|日立製作所]]负责制造。其中，1216和1233编成在较晚时间才运送至[[木场车辆检修场|木场车辆检修场]]。在[[新宿站|新宿]]-[[国立竞技场站|国立竞技场]]区间开通时，已有2组列车率先投入使用。在全线开通后，可运行43组列车，预备车10组。
==== 列车外观及设备 ====
本批次列车外观与第3批次列车外观相似。[[VVVF|VVVF]]由[[东芝|东芝]]制造，辅助电源装置由[[东洋电机制造|东洋电机]]制造。

== 列车大修 ==
由于第1、2批次列车服役时间均超过10年，所以在2007年开始进行大修。由于本列车制式和其他线路列车制式不同，故使用2005年3月制成的[[东京都交通局E5000型电力机车|东京都交通局E5000型电力机车]]进行牵引[[回送|回送]]，经[[浅草线|浅草线]]到达[[马込车辆检修场|马込车辆检修场]]。其中主要进行列车的外涂装、地板、[[荧光灯|荧光灯]]及「银杏」标志的变更。<br />在2009年，第3批次的列车也开始了大修工程，主要内容为增加车门处的识别标志<ref>铁道图书刊行会「铁道画刊」2010年10月臨時増刊号铁道车辆年鉴参照。</ref>。
