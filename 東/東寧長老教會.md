'''台灣基督長老教會東寧教會'''位於[[臺灣|臺灣]][[臺南市|臺南市]][[東區|東區]]，地址為臺南市東區東寧路120巷12號，是[[台灣基督長老教會|台灣基督長老教會]]台南中會所轄的教會之一。現任主任牧師為陳淳佳牧師。

== 沿革 ==
[[美國|美國]]籍宣教師杜佐志牧師（[[英語|英語]]：George Todd）來台任教於[[台南神學院|台南神學院]]期間，建議訓練大專事工人才，並認為應在[[成功大學|成功大學]]附近建立教會，作為傳道據點。時任[[英國|英國]]籍神學院教授彌迪理牧師（[[英語|英語]]：Dan Beeby）與吳振坤長老依此構想，向[[東門長老教會|東門教會]]提議，於1960年1月17日和會通過分設支會。當時租借勝利路與東寧路交叉口店舖為佈道所，定名為東寧教會，於1961年4月2日復活節開始聚會禮拜。<ref name=":0">{{Cite web|url=http://churches.chhstw.net/dongn/About.asp|title=關於東寧-東寧教會|accessdate=|author=東寧教會|date=|publisher=}}</ref>

== 建築 ==
1964年4月5日起向[[長榮女中|長榮女中]]租借禮拜堂聚會，並於同年7月15日升格為堂會，隨即購買現址土地二百坪，於1966年11月開工建堂，工程於1967年夏天完成。1979年購買教會後面約廿坪土地、1984年興建教育館、1995年購買教會隔壁五十一坪土地及房屋，而有今日的面貌，本會禮拜堂無論外觀與內部富有本地風格<ref name=":0" />。

== 附屬機構 ==
* 文化研究室：1990年成立，出版「東寧雙月刊」<ref name=":0" />。
* 台南市終生學習推廣協會：1996年向[[臺南市政府|臺南市政府]]立案成立，開辦「社區學苑」及「東寧社區報」等事工。
* 台南市家庭關懷協會：1998年由[[臺南市政府|臺南市政府]]委託，承辦政府委託「老人居家服務」、「獨居老人午餐外送服務」等事工。

== 歷史 ==
* 1961年4月2日：舉行第一場禮拜。
* 1964年年7月15日：從支會升格為堂會。
* 1966年11月：禮拜堂工程開工。
* 1967年夏：禮拜堂完工。
* 1976年4月18日：舉行設教15週年感恩禮拜，並補行禮拜堂獻堂。
* 1977年：張德謙牧師接任本會第四任牧師。
* 1984年：興建教育館。
* 1990年：張德謙牧師離任，由劉錦昌傳道接任本會。本會成立「文化研究室」。
* 1990年9月25日：劉錦昌傳道於本會封牧，是為本會第五任牧師。
* 1993年：劉錦昌牧師離任，由呂秉衡牧師接任本會，是為本會第六任牧師。
* 1996年：本會向[[臺南市政府|臺南市政府]]立案成立「台南市終生學習推廣協會」。
* 1998年：[[臺南市政府|臺南市政府]]委託台南市家庭關懷協會至本會，本會承辦政府委託「老人居家服務」、「獨居老人午餐外送服務」等事工。
* 2007年：呂秉衡牧師離任，當年9月由林瑞隆牧師接任本會，是為本會第七任牧師。
* 2011年4月3日：本會舉行設教50周年感恩禮拜。<ref>{{Cite web|url=http://blog.xuite.net/cssc8326/wretch/101163679-0403%E6%9D%B1%E5%AF%A7%E6%95%99%E6%9C%83%E4%BA%94%E5%8D%81%E5%91%A8%E5%B9%B4%E6%84%9F%E6%81%A9%E7%A6%AE%E6%8B%9C|title=0403東寧教會50周年感恩禮拜|accessdate=2017-12-05|author=JOHNSON LEE|date=2011-04-09|publisher=}}</ref>
* 2012年9月：郭仲晃傳道於本會封牧<ref>{{Cite web|url=http://www.pct.org.tw/news_church.aspx?strBlockID=B00001&strContentID=C2012100800004&strDesc=&strSiteID=&strPub=&strCTID=&strASP=news_church|title=2012年9月牧者異動消息|accessdate=2017-12-05|author=台灣基督長老教會|date=2012-10-15|publisher=台灣基督長老教會}}</ref>，成為本會青年牧師。
* 2014年9月：林瑞隆牧師、郭仲晃牧師同時離任本會。
* 2015年9月27日：陳淳佳牧師接任本會，是為本會第八任牧師。
