'''東出昌大'''（{{lang-ja|{{jp|j=東出 昌大|hg=ひがしで まさひろ|rm=Higashide Masahiro|help=no}}}}，{{bd|1988年|2月1日|||Masahiro Higashide}}），生於[[日本|日本]][[埼玉縣|埼玉縣]]，男[[演員|演員]]、[[模特兒|模特兒]]、首飾設計師。經紀公司為{{link-ja|humanitè Inc.|ユマニテ (企業)}}。其妻為知名演員[[渡邊謙|渡邊謙]]的女兒[[杏_(演員)|杏]]。

== 簡歷 ==
埼玉縣朝霞高中畢業。擅長劍道、籃球、足球。由於父親是劍道老師，從小開始便每日練習劍道。<ref>[http://www.inlifeweb.com/reports/report_4022.html 男の履歴書 東出昌大編｜インライフ]</ref>程度為三段。有空的話也會到道場指導小朋友。<ref name="nikkansports20131201">[[日刊スポーツ|日刊スポーツ]] 2013年12月1日芸能面（30面）「日曜日のヒーロー880・東出昌大」より。</ref>此外，父親是[[日本料理|日本料理]]的[[廚師|廚師]]<ref name="nikkansports20131201"/>。

高中時在《MEN'S NON-NO》專屬模特兒選拔中得獎而獲得出道資格。大學退學後為了成為首飾設計師，就讀於珠寶專門學校<ref name="nikkansports20131201"/>。

2006年到2011年間出席Paris Fashion Week。作為ZUCCa與山本耀司的模特兒十分活躍。

2012年，於電影《[[聽說桐島退社了|聽說桐島退社了]]》作為演員出道，以此作品為契機，將身份由模特兒轉為[[演員|演員]]。以飾演本片角色菊池宏樹獲得第67屆[[每日電影獎|每日電影獎]]日本體育新聞大獎的新人奬<ref>{{Cite web |author=鈴木隆 |coauthors=広瀬登 |date=2013-01-18 |url=http://mainichi.jp/select/news/20130118k0000m040112000c.html |title=毎日映画コンクール:大賞は「終の信託」 |work=[[毎日jp|毎日jp]] |publisher=[[毎日新聞社|毎日新聞社]] |pages=1-2 |accessdate=2013-01-18}}</ref>、[[第36屆日本電影金像獎|第36屆日本電影金像獎]]最佳新人獎。

在2013年連續出演兩部[[日本放送協會|NHK]]的[[晨間小說連續劇|晨間劇]]上半期（四月到九月底）、下半期（十月到隔年三月底）。於晨間劇出演兩部以上的演員有許多，但在同一年度連續出演兩部則是相當罕見的例子。由於參演晨間劇，在[[Oricon|Oricon]]雜誌所做的「2013年爆紅男演員排行榜」中獲得第四名<ref>{{Cite web |date=2013-11-27 |url=http://www.oricon.co.jp/entertainment/special/page/831/ |title=2013年 ブレイク俳優ランキング |work=ORICON STYLE |publisher=オリコン |accessdate=2013-11-27}}</ref>。正式出演《[[多謝款待|多謝款待]]》前，便先搬到[[大阪市|大阪市]]居住，邊在天滿天神繁昌亭打工，邊聽[[落語|落語]]來塑造角色形象<ref name="nikkansports20131201"/>。

== 個人生活 ==
*2013年，東出昌大和[[杏_(演員)|杏]]因拍攝[[連續電視小說|NHK晨間劇]]《[[多謝款待|多謝款待]]》而交往，共同話題為「歷史」等；2015年1月1日，兩人正式登記結婚。<ref>[http://www.cna.com.tw/news/amov/201412260039-1.aspx 日本銀幕情侶 東出昌大和杏元旦結婚]，中央社，2014年12月26日</ref><ref>[http://www.nikkansports.com/entertainment/news/p-et-tp0-20150103-1416770.html 杏、東出昌大が元日に婚姻届を提出]</ref>2016年1月，妻子杏表示懷有[[雙胞胎|雙胞胎]]。同年5月16日，妻子杏生下雙胞胎女兒。2017年11月，杏再生下了一個兒子。
*喜歡[[日本酒|日本酒]]和[[歷史|歷史]]，對於[[時代劇|時代劇]]也有相當興趣<ref name="nikkansports20131201"/>。
