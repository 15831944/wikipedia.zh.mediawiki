'''東開聞站'''（{{jpn|j=東開聞駅|hg=ひがしかいもんえき|rm=Higashi-kaimon eki|f=y}}）是一座位於[[鹿兒島縣|鹿兒島縣]][[指宿市|指宿市]]的[[九州旅客鐵道|九州旅客鐵道]]（JR九州）[[指宿枕崎線|指宿枕崎線]]的鐵路車站。

== 車站構造 ==
*一個1面1線單式月台，不可以[[列車交會|列車交會]]。
*[[無人站|無人站]]
*[[地上車站|地上車站]]

== 利用狀況 ==
* 在2007年，平均1日的乘車人數為27人。

== 历史 ==
* 1960年（昭和35年）3月22日：車站啟用。
* 1987年（昭和62年）4月1日：因[[國鐵分割民營化|國鐵分割民營化]]，本站成為[[九州旅客鐵道|九州旅客鐵道]]的車站。
