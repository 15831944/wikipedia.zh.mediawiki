'''東海環狀自動車道'''（{{jpn|j=東海環状自動車道|hg=とうかいかんじょうじどうしゃどう|rm=Tōkai Kanjō jidōshadō|f=y}}）是起於[[愛知縣|愛知縣]][[豐田市|豐田市]]，經[[岐阜縣|岐阜縣]]，終於[[三重縣|三重縣]][[四日市市|四日市市]]的[[高規格幹線道路|高規格幹線道路]]，略稱'''東海環狀道'''（{{lang|ja|東海環状道}}／{{lang|en|TOKAI-KANJO EXPWY}}）。高速公路路線編號被編為{{Ja Exp Route Sign|C3}}（[[東京外環自動車道|東京外環自動車道]]亦編號為{{Ja Exp Route Sign|C3}}，但與本道路並無關聯）<ref>[http://www.mlit.go.jp/road/sign/numbering/files/numbering_tsuchi002.pdf 「高速道路ナンバリングの導入について」（平成29年２月14日付け国道企第55号）] {{webarchive|url=https://web.archive.org/web/20170217063651/http://www.mlit.go.jp/road/sign/numbering/files/numbering_tsuchi002.pdf |date=2017年2月17日 }}、国土交通省道路局長通知、2017年2月14日</ref>。

== 概要 ==
1994年名稱募集時取「'''M'''IE」（三重）「'''A'''ICHI」（愛知）「'''G'''IFU」（岐阜）之頭一字成愛稱'''{{lang|ja|MAGロード}}'''。

[[名古屋市|名古屋市]]周邊30-40[[km|km]]都市圈的環狀道路，連繫愛知縣豐田市、[[瀨戶市|瀨戶市]]，[[岐阜縣|岐阜縣]][[土岐市|土岐市]]、[[可兒市|可兒市]]、[[美濃加茂市|美濃加茂市]]、[[關市|關市]]、[[岐阜市|岐阜市]]、[[大垣市|大垣市]]、三重縣四日市市等[[都市|都市]]，與[[東名高速道路|東名高速道路]]、[[新東名高速公路|新東名高速公路]]、[[中央自動車道|中央自動車道]]、[[東海北陸自動車道|東海北陸自動車道]]、[[名神高速道路|名神高速道路]]、[[新名神高速道路|新名神高速道路]]等廣域的交通網形成長度約160km的[[自動車專用道路|自動車專用道路]]。

路線名稱為'''[[#國道475號|國道475號]]'''。

此道路被[[東海地方|東海地方]]政財界寄予厚望，理由如下：
# 緩和東名高速[[名古屋IC|名古屋IC]]附近和名神高速[[一宮JCT|一宮JCT]]附近發生的交通阻塞。
# 豐田市（[[自動車|自動車]]）、瀨戶市、土岐市、[[多治見市|多治見市]]（[[瓷器|瓷器]]、[[陶瓷材料|陶瓷材料]]）、關市（[[刃物|刃物]]、[[金屬加工|金屬加工]]）、大垣市（[[電子元件|電子元件]]）、[[員辨市|員辨市]]（自動車、電子元件）、四日市市（[[油脂|油脂]]、[[家庭用電氣機械器具|家電]]、[[信息技术|IT]]製品）等[[製造業|製造業]]之集積與都市連繋，增進製造效率。
# 經伊勢灣岸道、[[知多半島道路|知多半島道路]]、[[知多橫斷道路|知多橫斷道路]]、[[中部國際機場連絡道路|中部國際機場連絡道路]]（[[Centrair_Line|Centrair Line]]）至[[中部國際機場|中部國際機場]]的貨物便利與活性化。
# 愛知縣集中的自動車部品製造業分散化至他縣（特別是岐阜縣）。
# 經東海北陸道至岐阜縣奥美濃，[[飛驒國|飛驒地方]]之[[觀光|觀光]]活性化。

因為是環狀道路，故沒有「上行、下行」而有「外回、內回」之稱。

部份路標為防誤認成東海北陸道，故有「'''<small>東海</small>環狀道'''」與「'''<small>東海</small>環狀'''」標識。

=== 國道475號 ===
[[File:Japanese_National_Route_Sign_0475.svg|left]]
基於一般國道之路線指定政令與一般國道之指定區間指定政令之'''一般國道475號'''概要如下（政令記載為[[市町村|市町村]]名）：{{-}}
* 起點 : 豐田市（琴平町宮久後685番1）
* 終點 : 四日市市（北山町字中之山1909番之5）
* 重要經過地 : 愛知縣[[西加茂郡|西加茂郡]][[藤岡町_(愛知縣)|藤岡町]]、瀨戶市、土岐市、可兒市、[[美濃加茂市|美濃加茂市]]、關市、岐阜市、岐阜縣[[山縣郡|山縣郡]][[高富町|高富町]]、同縣[[本巢郡|本巢郡]][[糸貫町|糸貫町]]、大垣市、三重縣[[員辨郡|員辨郡]][[大安町|大安町]]、同郡[[東員町|東員町]]
* [[指定區間|指定區間]] : 豐田市琴平町宮久後685番1（豐田東JCT）- 四日市市北山町字中之山1909番之5（四日市北JCT）

一般國道475號於1993年4月1日政令改正後，是一般國道中唯一沒有共用區間的路線，2005年3月19日豐田東JCT-美濃關JCT區間通車。

== 歷史 ==
* 1988年6月30日 : [[第四次全國總合開發計劃|第四次全國總合開發計劃]]（四全總）有高規格幹線道路構想。
* 1989年度 : 土岐-關間開工。
* 1989年12月1日 : 都市計劃決定（土岐-關）。
* 1990年度 : 北勢-四日市間開工。
* 1991年3月4日 : 都市計劃決定（豐田-瀨戶）。
* 1992年1月21日 : 都市計劃決定（北勢-四日市）。
* 1993年4月1日 : 指定為一般國道475號路線。
* 1994年度 : 西部區間路線全面施工。
* 1996年10月4日 : 都市計劃決定（關-養老）。
* 1998年度 : 養老-北勢間開工。
* 1998年4月10日 : 都市計劃決定（瀨戶-土岐）。
* 2000年8月 : 豐田東JCT-美濃關JCT間 一般收費道路工程許可。
* 2005年3月19日 : 豐田東JCT-美濃關JCT間通車。
* 2007年4月24日 : 都市計劃決定（養老-北勢）。
* 2007年12月16日 : 五斗蒔停车区開業。
* 2009年4月18日 : 美濃關JCT-關廣見IC間通車。
* 2012年9月15日 : 大垣西IC-養老JCT間通車。
