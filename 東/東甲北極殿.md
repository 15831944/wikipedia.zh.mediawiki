'''東甲北極殿'''，又稱'''東甲宮'''，舊稱'''真武殿'''、'''上帝廟'''，主祀北極[[真武上帝|真武上帝]]。<ref name=":0">{{Cite book|title=《澎湖廳志．規制》|last=林|first=豪|publisher=成文|year=1983|isbn=|location=台北市|pages=58}}</ref>{{efn|真武上帝（玄天上帝）乃北方正神，五行屬水，顏色尚黑，具鎮妖伏魔之銳氣，曾獲[[明成祖|明成祖]]大力崇奉。<br/>根據民間五行學說，八字命理中喜用神為水者，適宜參拜真武上帝以及祈求光明燈庇佑，添補命數<ref>{{Cite web|url=https://jason168chiu.pixnet.net/blog/post/52126644-%E5%85%AB%E5%AD%97%E5%96%9C%E7%94%A8%E4%BA%94%E8%A1%8C%E7%9A%84%E9%96%8B%E9%81%8B%E6%B3%95|title=八字喜用五行的開運法|accessdate=2019-01-27|author=太極易研舘|date=2012-04-17|publisher=|language=zh-tw}}</ref>。<br/> 由於真武上帝肅穆之氣稍重，若從事軍警消人士且用神為水祈福，更形適宜。}}澎湖四大古廟之一，亦是馬公市區三大甲頭廟之一，更是唯一見載於清代地方志之中的甲頭廟（角頭廟）。<ref name=":2">{{Cite book|title=《2010澎湖縣文化資產手冊》|last=陳英俊、高啟進、林文鎮、郭金龍|first=|publisher=澎湖縣文化局|year=2010|isbn=9789860262797|location=澎湖縣|pages=114-117}}</ref>北極殿位於媽宮社之東南（今馬公市啟明里），清季此區屬於媽宮東甲，是以當地人慣稱北極殿為「東甲宮」。

== 沿革 ==
實際建廟時間不詳，一說明朝末年、一說是康熙廿九年（1690年）<ref name=":1">{{Cite web|url=https://penghu.info/OB3E62F7CE02DA2D1EAB|title=北極殿|accessdate=2018/03/23|author=|date=|publisher=Penghu.Info｜澎湖知識服務平台}}</ref>，總之於清乾隆、嘉慶年間便有可考之捐修紀錄。<ref name=":0" />根據廟內昭和年間改建碑文紀載，廟內祀有真武上帝與[[准提菩萨|準提菩薩]]，而在遭逢兵燹與時疫之際[[溫王爺|溫王]]曾顯現過靈蹟。光緒元年（1875年），當地仕紳感念於此，又見北極殿殘破，[[武秀才|武庠生]]高其華（後出任[[澎湖水師協|澎湖水師協]]左營[[千總|千總]]）、[[舉人|舉人]]鄭步蟾與郭鶚翔再度鳩資重建。

昭和二年（1927年），當時董事以舊有規制狹小，乃增建川亭、鐘鼓樓於殿內立碑謹記。{{efn|見附圖石碑紀載。}}大正十年（1921年）受「[[一新社|一新社]]」影響，開辦鸞堂「聿新社寶善堂」。<ref>{{Cite book|title=《續修澎湖縣志．卷12．宗教志》|last=余|first=光弘|last2=黃|first2=有興|publisher=澎湖縣政府|year=2005|isbn=9860015198|location=澎湖縣|pages=51–54|language=zh-tw}}</ref>民國58年（1969年），北極殿降壇神諭指示，[[澎湖三官殿|澎湖三官殿]]由是開辦鸞堂「自新社三善堂」。<ref>{{Cite web|url=https://penghu.info/OBE112D59871E1513B41|title=三官殿|accessdate=2018-05-04|author=|date=|publisher=}}</ref>

惟可惜第二次世界大戰期間，盟軍轟炸媽宮，北極殿廟身嚴重受損，另於1949年再度修建，繼續沿用昭和二年（1927年）擴建的規制<ref>{{Cite web|url=http://nrch.culture.tw/twpedia.aspx?id=8747|title=臺灣大百科|accessdate=2018/03/23|author=李孟淑|date=2009/09/09|publisher=}}</ref>，修建落成後卻被國民政府軍隊進駐，爾後又有多次翻修的記錄。<ref name=":1" />北極殿因廟舍莊嚴典雅，在2002年被登錄為[[臺灣歷史建築|歷史建築]]。今日所見北極殿，氣象新穎且色彩鮮豔，不見古樸舊貌，乃2005年遭逢祝融之災後再度整建的樣貌<ref>{{Cite book|title=《續修澎湖縣志．卷13．文化志》|last=楊|first=麗祝|publisher=澎湖縣政府|year=2005|isbn=9860015201|location=澎湖|pages=94}}</ref>，也是澎湖縣政府登錄歷史建築以來，首座以自籌經費完成整修的宮廟。<ref name=":2" />

== 特色 ==
清代以降，文人雅士如舉人鄭步蟾和官員郭石頭、陳拱照皆多居東甲，該處文風鼎盛，是以有「東甲好筆尾」之稱，與「南甲好櫓尾」、「北甲好龜粿」互為對應。其他諸如高恭、陳材蕊或紀雙抱等文人的水墨、書法以及對聯作品亦存於廟宇之內。<ref name=":3" />山門玄武岩立面上，尚有題「東方呈秀氣，甲第起文人」之對聯，以及1927年高恭「蟹蘭圖」作品，充分展現東甲北極殿之文人雅致。<ref name=":2" />

更難能可貴的是，有別於其他市區的角頭廟（[[南甲海靈殿|南甲海靈殿]]與[[北甲北辰宮|北甲北辰宮]]），東甲北極殿在改建之後仍然有保留傳統廟宇的建制。<ref name=":3">{{Cite web|url=http://blog.xuite.net/hikarirage0425/twblog/165365406-%E6%9D%B1%E7%94%B2%E5%AE%AE+%28+%E9%A6%AC%E5%85%AC%E5%8C%97%E6%A5%B5%E6%AE%BF+%29Part1|title=東甲宮（馬公北極殿）Part1|accessdate=2018/03/23|author=|date=2009/010/25|publisher=}}</ref> 東甲宮內的四尊[[門神|門神]]彩繪與[[二十四節氣|二十四節氣]]宿神咸出於國寶大師[[黃友謙|黃友謙]]手筆，而棟架工程則由主持過[[澎湖天后宮|澎湖天后宮]]整修的[[藍木|藍木]]與其弟子陳六甲先後擔當。<ref name=":2" /><ref>{{Cite book|title=《台灣古建築圖解事典》|last=李|first=乾朗|publisher=遠流|year=2006|isbn=957324957X|location=台北市|pages=162}}</ref><ref name=":3" />

內藏「藍木製銅鑄宣爐」，造型作三片蓮瓣狀，刻有「大正壬戌年孟秋鑄」、「-{R|炉}-下弟子藍木敬-{R|献}-」及「辛酉澎湖藍木工場製」等落款，於2018年9月3日經[[文化部文化資產局|文化部文化資產局]]公告為「生活及儀禮器物」類別的「古物」。<ref>{{Cite web|url=https://nchdb.boch.gov.tw/assets/overview/antiquity/20181031000002|title=馬公東甲北極殿-藍木製銅鑄宣爐|accessdate=2018-11-29|author=|date=2018-09-03|publisher=文化部文化資產局--國家文化資產網|language=zh-tw}}</ref>

== 遶境活動 ==
2018年4月7日（農曆二月廿二），為慶祝[[西衛宸威殿|西衛宸威殿]]重建落成十五周年與[[屏東縣|屏東]][竹田慈后宮]結盟六周年<ref name=":4">{{Cite news|url=http://www.taiwanhot.net/?p=562704|title=〈慶祝重建15週年　西衛宸威殿展開繞境祈福〉|author=地方中心|date=2018-04-08|work=台灣好新聞|accessdate=2018-04-16}}</ref>（註：屏東竹田慈后宮係於1988年從[[澎湖天后宮|澎湖天后宮]]分靈出去的媽祖廟<ref>{{Cite news|url=https://news.housefun.com.tw/news/article/451236127671.html|title=〈曾信基督教的她　為澎湖媽祖在台建廟〉|author=林良齊|date=2016-04-24|work=聯合新聞網|accessdate=2018-04-16}}</ref>），澎湖地區特別舉辦祈求國泰民安的遶境活動。澎湖公廟[[澎湖水仙宮|臺廈郊水仙宮]]、[[媽宮城隍廟|馬公城隍廟]]、[[澎湖三官殿|澎湖三官殿]]、澎湖天后宮、[[澎湖觀音亭|澎湖觀音亭]]，或者各地角頭廟如[[靈光殿|光榮里靈光殿]]、[[重光威靈殿|重光威靈殿]]、將軍永安宮、中屯永安宮，媽宮三甲的[[東甲北極殿|東甲北極殿]]、[[南甲海靈殿|南甲海靈殿]]與[[北甲北辰宮|北甲北辰宮]]，甚至移靈至高雄吉貝武聖殿、高雄西衛宸威殿皆來澎共襄盛舉。<ref name=":4" />

東甲北極殿立於昭和二年（1927年）之碑文。

本殿建自何時誌書不詳無可考焉據諸父老相傳云為有明旅澎先民所建若然距今四百有餘年誠吾澎一古廟也主神祀真武大帝後增祀準提菩薩及溫王兵燹時疫之際輒著靈跡遐邇咸帡幪焉有清光緒乙亥年董事里人武庠生高其華與舉人鄭步蟾郭鶚翔等鳩資重建迄今五十餘載迭遭兵燹及風雨漂削寖形頹圯甲子年夏我民爰議改建之事各自慨捐巨款計得金一萬餘圓此雖民樂趨義舉實則神恩奕世淪浹之深有以使然也于越年丁卯年春興工改建因舊制稍狹議拓而廣之由是改舊址築新基加川亭增前進建鐘鼓樓及兩偏掖庭塑神像制廟器無不煥然一新是歲葭冬完峻計費貳萬有奇其不敷者以所積公項及民丁緣金各千餘圓充之餘即由諸董事湊填事乃克濟爰書其梗概泐石以不朽|昭和丁卯年季冬之月吉旦改建事立石}}
