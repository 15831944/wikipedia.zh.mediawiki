'''東龍洲'''又名'''東龍島'''或者'''南堂島'''（英文：Tung Lung Island），是[[香港|香港]][[西貢區|西貢區]]最南端的[[島嶼|島嶼]]，位於[[清水灣半島|清水灣半島]]以南，[[香港島|香港島]][[小西灣|小西灣]]之東南，島嶼面積2.42平方公里。

由於東龍洲位於[[佛堂門|佛堂門]]以南，古時被稱為'''南佛堂'''<ref name=新安縣志 />。島嶼西南的[[南堂海峽|南堂海峽]]（今多寫作[[藍塘海峽|藍塘海峽]]）也是因為南堂島的名稱而得名。另一方面，其北部的海峽被稱為佛堂門，後來因為[[填海|填海]]關係，佛堂門以北的[[佛堂洲|佛堂洲]]（[[北佛堂|北佛堂]]）已經與清水灣半島連在一起。

==景點==
[[東龍洲石刻|東龍洲石刻]]是[[香港史前時期|香港史前時期]]的石刻。東龍洲上設有[[東龍島炮台特別地區|東龍島炮台特別地區]]，內有[[東龍洲炮台|東龍洲炮台]]遺址；亦是[[漁農自然護理署|漁農自然護理署]]的[http://www.afcd.gov.hk/tc_chi/country/cou_vis/cou_vis_cam/cou_vis_cam_cam/cou_vis_cam_cam.html 指定露營地點]之一，最少可以容納7個[[露營|露營]]單位。

[[洪聖宮|洪聖宮]]<ref>{{Cite web |url=http://www.openlife.com.hk/%E9%A6%99%E6%B8%AF%E5%BB%9F%E5%AE%87%E5%90%8D%E5%86%8A/%E6%9D%B1%E9%BE%8D%E6%B4%B2%E6%B4%AA%E8%81%96%E5%AE%AE/%E5%8F%A4%E5%BB%9F |title=存档副本 |access-date=2017-07-23 |archive-url=https://web.archive.org/web/20170905051900/http://www.openlife.com.hk/%E9%A6%99%E6%B8%AF%E5%BB%9F%E5%AE%87%E5%90%8D%E5%86%8A/%E6%9D%B1%E9%BE%8D%E6%B4%B2%E6%B4%AA%E8%81%96%E5%AE%AE/%E5%8F%A4%E5%BB%9F |archive-date=2017-09-05 |dead-url=yes }}</ref>是众多[[洪聖廟|洪聖廟]][[香港洪聖廟列表|之一]]。洪聖宮的西边的海岸自[[南宋|南宋]]<ref name=大廟灣刻石碑文 />和清朝曾有[[天后廟|天后廟]]。<ref name=新安縣志>[https://zh.wikipedia.org/w/index.php?title=%E6%9D%B1%E9%BE%8D%E6%B4%B2%E7%82%AE%E5%8F%B0&oldid=35180004 「佛堂門在鯉魚門之東南……其北曰北佛堂，其南曰南佛堂，兩邊皆有天后古廟…… 《[[新安縣志|新安縣志]]》．卷之四．山水略]</ref>該[[香港天后廟列表|天后廟]]現已不存。

[[北宋|北宋]]时，東龍洲上有石塔{{dubious}}<ref>大廟灣刻石碑文：「攷南堂石塔建於大中祥符五年」。語譯：據考證所得，南佛堂的石塔在宋[[大中祥符|大中祥符]]五年（[[1012年|1012年]]）建成。</ref><ref>{{Cite journal| last = Jen| first = [[簡又文|Yu-wen]] | title = The Travelling Palace of Southern Sung in Kowloon| journal = Journal of the '''[[皇家亞洲學會|Royal Asiatic Society]] Hong Kong Branch'''| volume = 7| pages = 21–38 | year = 1967| url = http://hkjo.lib.hku.hk/archive/files/86488813852b55347564ada1c6256ae9.pdf| issn = 1991-7295| doi = | id = | accessdate =}}</ref>，現已不存。南宋时，九龍[[蒲岡村|蒲岡村]]林姓[[林松堅|林松堅]]、[[林柏堅|林柏堅]]駕駛艚船出海遇到[[颱風|颱風]]，船毀貨失。他們兩人力挽船篷，緊抱船上祀祭的林氏大姑神主，浮到東龍洲（南佛堂），安全脫險。他們認為這是神靈保佑，便在南佛堂修建了「祭祀林氏大姑的神廟」。這個林氏大姑即後來人們所稱的天后。<ref>[http://www.chiculture.net/0217/html/c27/1205c27.html 中國文化研究院:香港天后廟]</ref>林松堅的兒子[[林道義|林道義]]後來又在北佛堂修建了[[佛堂門天后古廟|一座同類神廟]]。<ref name=大廟灣刻石碑文>大廟灣刻石碑文：「土人林道義継之道義又能宏其規求再立石以紀咸淳甲戌六月十五日書」。語譯：本地人林道義，又再修繕南北佛堂；林道義更擴充其規模，同時請求再次書文立石，記述南北佛堂修廟的經過。[[咸淳|咸淳]][[甲戌|甲戌]]年六月十五日（即[[1274年|1274年]]7月20日）。</ref>

東龍洲最南端的半島南堂尾有南堂尾燈塔，為[[香港日佔時期|日軍佔領香港]]時建造的防衛設施，約建於1941至1945年，原用作炮台以保衛[[維多利亞港|維多利亞港]]東南面水道，燈塔以混凝土建成，炮房則在1965年改建為[[海事處|海事處]]燈塔機房<ref>{{Cite web |url=http://www.hkmemory.hk/collections/lighthouses/geo/LH6/index_cht.html |title=南堂尾燈塔 |accessdate=2018-01-20 |publisher=香港記憶}}</ref><ref>{{Cite web |url=http://www.hk-place.com/view.php?id=258 |title=建設及建築物－防衛設施(五)日軍防衛設施 |accessdate=2018-01-20 |publisher=香港地方}}</ref>。

==交通==
東龍洲對外交通完全依靠[[渡輪|渡輪]]，而且只在星期六、星期日及公眾假期提供服務<ref name=transport>{{cite web |language=zh-hk |url=http://www.td.gov.hk/tc/transport_in_hong_kong/public_transport/ferries/service_details/#k02|title=渡輪 服務詳情|publisher=運輸署}}</ref>。在救急需要等特殊情況下，可利用在南堂頂附近設有的[[直升機|直升機]]坪。

* 街渡：[[三家村_(九龍)|三家村]]至東龍洲，由[[珊瑚海船務|珊瑚海船務]]營運<ref name=transport/>。
* 街渡：[[西灣河|西灣河]]至東龍洲，原来由林記街渡營運，後由梁記街渡（[[碧海船務|碧海船務]]）接手營運<ref name=transport2>{{cite web |language=zh-hk |url=http://www.td.gov.hk/tc/transport_in_hong_kong/public_transport/ferries/service_details/#k15|title=西灣河- 東龍島 服務詳情|publisher=運輸署}}</ref>。
