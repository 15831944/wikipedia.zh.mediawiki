'''東海堂高級麵包西餅專門店'''，簡稱'''東海堂'''（英語：Arome Bakery，日語：アローム・ベーカリー），是[[香港|香港]]一家售賣[[麵包|麵包]]及[[西餅|西餅]]的[[連鎖店|連鎖店]]，成立於1980年代，主攻中高檔市場。其總部現位於[[九龍西|九龍西]][[長沙灣|長沙灣]][[長順街|長順街]]17號[[美心集團中心|美心集團中心]]，廠房則於2012年元旦由長沙灣遷往[[大埔工業邨|大埔工業邨]]。東海堂近年積極在香港擴展市場，目前在香港擁有74間分店。

公司主要負責人之一是[[于文瑛|-{于}-文瑛]]，{{#if: 據悉她擁有東海堂約四分之一股份|<mark class="template-facttext" title="需要提供文献来源">據悉她擁有東海堂約四分之一股份</mark>}}<sup class="noprint Template-Fact">[[Wikipedia:列明来源|<span style="white-space: nowrap;" title="来源请求{{#if:]]{{namespace||[[Category:{{#if:|自{{#time:Y年F]]}}</sup>，其父[[于鏡波|-{于}-鏡波]]則是「[[香港建設|香港建設]]」（前稱[[熊谷組|熊谷組]]）的創辦人，父女倆均是東海堂董事。於2008年3月被[[美心集團|美心集團]]收購，原設有的積分計劃於年底終止及會員折扣亦至會籍到期日止。

== 竄改食用日期事件 ==
2007年12月初，有報章揭發東海堂部份分店的職員，在晚上竄改賣剩麵包的「[[最佳食用日期|最佳食用日期]]」標籤，把本來售3天的袋裝麵包的最佳食用日期，在第3天晚延長到4天，目的是將大批隔夜麵包當新鮮麵包出售，加快去貨<ref>《東海堂竄改麵包食用日期 隔夜麵包當新鮮麵包賣》，《明報》，2007年12月7日</ref>。事件引起公眾廣泛關注，但當時[[香港法例|香港法例]]並未將該等行為列為違法<ref>《最佳食用日 法律管不嚴》，《明報》，2007年12月7日 </ref>。

== 濫用日文事件 ==
2016年1月，[[AKB48|AKB48]][[小嶋陽菜|小嶋陽菜]]曾為東海堂拍攝一則題為「美人の赤苺」的代言廣告，東海堂官方也在廣告上使用日文做為宣傳詞，但文宣內容夾雜港式中文和日文，通篇文章不成文法，遭到強烈的批評。<ref>{{cite web|author1=劍心日劇廣告娛樂情報|url=https://www.facebook.com/KenshinFB/photos/a.173852862633685.39792.172541882764783/1103838919635070/?type=3&theater|accessdate=2017-03-25}}</ref>

== 小註 ==
於[[中华人民共和国|中國]][[廣州市|廣州市]]開設分店售賣麵包和西餅的[[东海堂_(广州)|廣州東海堂]]，與香港的東海堂並沒有關係。
