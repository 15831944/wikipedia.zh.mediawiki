'''東村山站'''（{{jpn|j=東村山駅|hg=ひがしむらやまえき|rm=Higashi-murayama eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[東村山市|東村山市]]本町二丁目，屬於[[西武鐵道|西武鐵道]]的[[鐵路車站|鐵路車站]]。包括特急「{{link-ja|小江戶號列車|小江戸 (列車)|小江戶}}」號的所有列車都會停靠。

== 停靠路線 ==
此站有[[新宿線_(西武鐵道)|新宿線]]、[[西武園線|西武園線]]、[[國分寺線|國分寺線]]的3條路線停靠，使用4個方向的列車。西武園線以此站為起點、國分寺線以此站為終點。但是國分寺線的一部分列車會與新宿線本川越方向以及西武園線直通運行。

新宿線的車站編號是'''SS21'''，國分寺線與西武園線的車站編號是'''SK05'''。

== 車站構造 ==
[[島式月台|島式月台]]3面6線的[[地面車站|地面車站]]，設有[[跨站式站房|跨站式站房]]。

各月台以2條跨線天橋聯絡，與近所澤、西武園的站舍結合成一體<ref name="higashimurayamamap">[http://www.seibu-group.co.jp/railways/railway/ekimap/higashi-murayama/index.html 東村山駅 駅構内マップ] {{webarchive|url=https://web.archive.org/web/20130807060706/http://www.seibu-group.co.jp/railways/railway/ekimap/higashi-murayama/index.html |date=2013-08-07 }} - 西武鉄道</ref>。各月台與閘機層，東口、西口地面部分與閘機層各自設置了[[升降機|升降機]]、[[扶手電梯|扶手電梯]]聯絡<ref name="higashimurayamamap"/>。

[[廁所|廁所]]位於閘內大堂部分，併設多機能廁所<ref name="higashimurayamamap"/>。
