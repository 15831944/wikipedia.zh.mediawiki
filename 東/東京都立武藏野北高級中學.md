'''東京都立武藏野北高級中學'''（{{Lang-ja|'''東京都立武蔵野北高等学校'''}}，{{Lang-en|'''Tokyo Metropolitan Musashinokita High School'''}}）是一所位於[[日本|日本]][[東京都|東京都]][[武藏野市|武藏野市]]八幡町二丁目的[[男女共學|男女共學]][[公立學校|公立學校]]。

== 學校簡介 ==
1978年，在當地人民對高中的強力需求和呼籲下，設立該校。該校學生、教職員工及當地人都稱呼該校的暱稱“武北（{{lang|ja|ムサキタ}}）”。武藏野北高級中學被東京都教育機關指定為“進学指導推進校”，其大學入學成績一直在穩步提升中。除了學風盛行之外，該校的課外活動也開展得非常熱鬧。

學校毗鄰{{Link-ja|武藏野中央公園|武蔵野中央公園}}，這是東京都少有佔地面積龐大的公園。放學後，可以在這遇見不少正進行課外活動的武藏野北高級中學學生。<ref name="about">{{cite web|author1=東京都立武藏野北高級中學|title=東京都立武蔵野北高等学校沿革|url=http://www.musashinokita-h.metro.tokyo.jp/gaiyou-enkaku.html|publisher=東京都立武藏野北高級中學|accessdate=2017-11-12}}</ref>

== 學校沿革 ==
* 1978年：東京都立武藏野北高級中學開校（於東京都立砂川高級中學教學樓授課）。<ref name="about" />

* 1979年：東京都立武藏野北高級中學二期學生入校，學校遷往現址授課。<ref name="about" />

* 1980年：成為全國[[軟式棒球|軟式棒球]]大賽東京都代表。<ref name="about" />

* 1982年：引入{{Link-ja|東京都集體聯合選拔制度|グループ合同選抜制度}}，被編為第九學區的91小組。<ref name="about" />

* 1994年：開闢{{Link-ja|日本高中招生制度|高校受験|單獨招生}}通道。<ref name="about" />

* 2007年：獲選成為東京都“{{Link-ja|進学指導推進校|進学指導推進校}}”<ref name="about" />

== 學校制服 ==
東京都立武藏野北高級中學所使用校服於1995年制訂，上裝男女均為深藍色西裝外套，下裝男生為同色單排釦西褲，女生為雙排釦格子[[百褶裙|百褶裙]]。而自2005年學年開始，女生下裝裙子的長度修改為通常長度。<ref>{{cite web|author1=東京都立武藏野北高級中學|title=東京都立武蔵野北高等学校制服|url=http://www.musashinokita-h.metro.tokyo.jp/gaiyou-seihuku.html|publisher=武藏野北高級中學官方網站|accessdate=2017-11-12}}</ref>

== 學校活動 ==
* 校運動會每年五月舉行。

* 校合唱節每年六月舉行。

* 校文化節（也被稱之為“綠光節”）每年九月舉行。主要內容有班級劇場、餐飲販售及教室展示等。

== 課外活動 ==
游泳部、圍棋將棋部與田徑部擁有參加[[關東地區|關東地區]]乃至全國大賽的實力；男子排球部等體育部門則在市級比賽中擁有較好成績。此外，理科部也有諸多研究報告發佈。
