== 回應 ==
=== 赞成意見 ===
前中华人民共和国副主席[[习近平|习近平]]在2009年12月14日访问日本前表示，日本首相[[鸠山由纪夫|鸠山由纪夫]]提出建立东亚共同体，显示了日本政府重视东亚区域合作的积极姿态。这一构想符合亚洲一体化进程的大趋势，也是包括中日两国在内本地区各国共同追求的目标。构建东亚共同体是一个系统工程，既要立足现实，又要着眼长远。当前最重要的是各国应加强对话沟通，形成共识。<ref>{{Cite web|url=http://news.163.com/09/1214/09/5QG2B6P6000120GR.html|title=建东亚共同体是中日共同目标}}<!--2009-12-14 09:47:00　来源: 东方早报(上海)　跟贴 0 条 手机看新闻
早报讯http://news.sina.com.cn/w/2009-12-14/025316767357s.shtml?from=wap--></ref>

==== 經濟方面的優點 ====
* 若只限於貨幣來說，導入亞洲共通貨幣將能壓抑[[外匯市場|外匯市場]]的影響，使東亞地區的經濟長期安定。亞洲共通貨幣將會與[[美元|美元]]及[[歐元|歐元]]並列存在，使亞洲在國際經濟上地位向上提升。

==== 對地域的政治穩定、融和的期待 ====
* 隨著共同體經濟交流發展，各國政治對立能漸漸消除，維持地域安定。

==== 文化價值觀的相同性 ====
* 支配東亞地區的[[儒教|儒教]]已不是本地區共同官方學說，但是[[儒家思想|儒家思想]]依然是東亞社會共同的基本文化和倫理思想。

==== 其他 ====
* 日本贊成者認為，如果日本不參加共同體，共同體將會以中國為中心，中國對與會各國的影響力將會日益增大。所以作為牽制中國角色的日本，有必要在建設共同體上擔任領導角色。

=== 反對的意見 ===
==== 價值觀差異 ====
* 區內並沒有一致的價值觀。[[中華人民共和國|中華人民共和國]]和[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]仍然實行法律及事实上的[[一黨專政|一黨專政]]并奉行[[共产主义|共产主义]]，而诸如[[新加坡共和国|新加坡共和国]]及[[马来西亚|马来西亚]]等东南亚国家也存在强人政治的局面。在民主國家裡的[[大韓民國|大韓民國]]的[[國會|國會]]議會中，經常出現國會議員們強烈質詢政務官員對於公共事務處理的問題和責任歸屬，以及因[[政黨|政黨]]意見不同形成的激烈對立場面。此外，由于靠近[[中華人民共和國|中華人民共和國]]的[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]與[[日本國|日本國]]和[[美利堅合眾國|美利堅合眾國]]對立，因此比起[[自由|自由]]及[[民主主義|民主主義]]，[[民族主義|民族主義]]的價值觀亦會造成相當程度的影響。
* 中、日、韓三國之間，對各自及相互的[[歷史|歷史]]認識雖有部份的不同，但2009年10月開始進行歷史教科書共同編輯，增進東亞各國民族間的歷史共識，建立亞洲經濟共同繁榮的局面。

==== 政治及領土紛爭 ====
*把日本南韓的盟國[[美國|美國]]排除，將令美國感到不快，因此美國積極遊說澳大利亞等盟國共同加入該共同體。
*國際主流社會不承認[[中華民國|中華民國]]（[[臺灣|臺灣]]）的國際地位，可能導致其人民無法加入。
*[[中華人民共和國|中華人民共和國]]堅持獨立自主的國家性，韓國[[盧武鉉|盧武鉉]]政府具有反美性質，因此被認為這個構想帶有反美色彩。
*中日民眾的[[反韩情绪|反韩情绪]]、中韓民眾的[[反日情緒|反日情緒]]、以及日韓民眾的[[反华情緒|反华情緒]]日渐增长使日本國內反對者眾多。
*韓國與日本在[[獨島|獨島]]／[[竹島|竹島]]、[[中華人民共和國|中華人民共和國]]和[[中華民國|中華民國]]與[[日本|日本]]在[[釣魚台列嶼|釣魚台列嶼]]／[[尖閣諸島|尖閣諸島]]，[[中華人民共和國|中華人民共和國]]與[[中華民國|中華民國]]及[[東南亞|東南亞]]各國之間有[[南沙群島|南沙群島]]的領土問題未解決。

==== 經濟學 ====
* 日本與其他各國在經濟水平上差異甚大，低素質勞工將大量流入日本。就算是將[[社會|社會]]因素考慮在內，日本與其他國家之間的[[自由貿易協定|自由貿易協定]]仍然相當之多。
* 若導入共通貨幣，各國將失去在[[經濟政策|經濟政策]]及[[財政政策|財政政策]]的[[自主權|自主權]]，各國將不能實行符合自身市場的金融及財政政策，反將使經濟惡化。事實上，沒有加入歐元區的[[英國|英國]]、[[瑞典|瑞典]]及[[丹麥|丹麥]]等地的經濟良好，而加入了歐元區的[[德國|德國]]及[[法國|法國]]的經濟則陷入低迷。
* 美國作為中國第二大貿易伙伴、日本最重要的貿易伙伴，成立共同體後被排除在外顯得不合理。
* 東亞地區成立共同體後，日本將被強勢的中國經濟圈蠶食，經濟變得弱體化，日本反對者認為日本應加入[[北美自由貿易協定|北美自由貿易協定]]與價值觀相近的美國形成共同體，而非東亞共同體。

==== 國防 ====
* 若成立類似[[快速反應部隊|快速反應部隊]]的聯合軍隊，[[中國人民解放軍|中國人民解放軍]]將可能駐留日本。日本的[[自衛隊|自衛隊]]亦有可能駐留在中國及朝鮮半島，將招致各國的反對聲音。

==== 其他 ====
* 反對方認為，與[[大日本帝国|大日本帝国]]1940年代提倡成立的[[大東亞共榮圈|大東亞共榮圈]]過於相似，違反讓東亞各國在對等立場下均能參加的國際組織的主張。而贊成方認為，國際政治的現實就是強權政治，應從開始就信任「國家之間的對等立場」的樂觀想法。
* 中国国内的发展程度不均衡，日本本身排外的心态，韩国的殖民地历史都成为东亚共同体成长的障碍。东亚虽然在文化区域上都是儒家文化的范围，但是本国的意识形态和西化浪潮的影响，凝聚力显然不够。尤其是北韓和中国的共产政权与日本南韩的民主政治模式之间的矛盾。<ref>{{Cite_web |url=http://www.ftchinese.com/story/001057909 |title=从东亚共同体到TPP：日中渐行渐远 - FT中文网 |author=徐一睿 |date=2014年8月27日<!-- 06:08--> |editor=徐瑾 |department= |website=中日关系 |series= |publisher=FT中文网 |location=日本 |page=1 |at= |language=zh-Hans |script-title= |trans-title= |type= |format=shtml |archive-url= |archive-date= |dead-url=no |access-date=2017-02-27 |quote=本文仅代表作者观点，作者为日本庆应大学经济学博士，亦为近期在日出版《中国财政调整制度的新局面》 |ref=日本嘉悦大学经营经济学部学者 徐一睿 为英国《金融时报》中文网撰稿 |postscript= |subscription= |registration=}}</ref>
