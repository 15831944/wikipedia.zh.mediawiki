'''彭佳学'''{{BD|1965年|5月}}，[[安徽省|安徽省]][[安庆市|安庆市]][[潜山县|潜山县]]人，1987年6月加入[[中国共产党|中国共产党]]，1989年7月参加工作。[[武汉水利电力学院|武汉水利电力学院]]农水系农田水利工程专业毕业，研究生学历。曾任浙江省海洋与渔业局局长、党组书记，中共绍兴市委书记。现任[[浙江省人民政府|浙江省人民政府]]副省长<ref name="Bio1">{{Cite web |url=http://ldzl.people.com.cn/dfzlk/front/personPage15127.htm |title=彭佳学 简历 |accessdate=2018-01-20 |publisher=人民网地方领导资料库}}</ref>、党组成员。

== 简历 ==
曾任[[浙江省|浙江省]]水电建设投资总公司筹备组副组长，省水利厅规划计划处副处长、处长，省水利厅副厅长、党组成员，温州市委常委、常务副市长，省水利厅副厅长、党组副书记等职务<ref name="Bio1"/>。2013年3月，任浙江省海洋与渔业局局长、党组书记。2015年10月，任中国共产党[[绍兴|绍兴]]市委员会书记<ref name="Bio2">{{Cite web |url=http://pol.zjol.com.cn/201703/t20170302_3232715.shtml |title=绍兴市第八次党代会闭幕 彭佳学当选市委书记 |accessdate=2018-01-20 |author=记者 方力 |date=2017-03-02 |publisher=浙江在线}}</ref>。

2018年1月30日，53岁的彭佳学当选[[浙江省人民政府|浙江省人民政府]]副省长<ref>{{cite web|accessdate=2018-01-31|url=http://www.xinhuanet.com/politics/2018-01/30/c_1122341919.htm|title=车俊当选浙江省人大常委会主任 袁家军当选浙江省省长|publisher=[[新华网|新华网]]}}</ref>。2月13日，不再担任绍兴市委书记职务。
