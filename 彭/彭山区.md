'''彭山区'''是[[中国|中国]][[四川省|四川省]][[眉山市|眉山市]]所辖的一个[[市辖区|市辖区]]，位于[[成都市|成都市]]南约45公里。总面积465平方公里，总人口约33万。2014年10月21日国务院批准撤销彭山县设立眉山市彭山区<ref>[http://202.108.98.30/defaultQuery?shengji=%CB%C4%B4%A8%CA%A1%28%B4%A8%A1%A2%CA%F1%29&diji=%C3%BC%C9%BD%CA%D0&xianji=-1 中华人民共和国民政部2014年眉山市行政区划]</ref>。

== 历史 ==
[[汉|汉]]置武阳县，[[南朝|南朝]]·[[梁|梁]]改犍为县，[[西魏|西魏]]改隆山县，[[唐|唐]][[先天|先天]]元年（712年），避[[李隆基|李隆基]]讳，改名彭山县。[[明|明]][[洪武|洪武]]十年（1377年），彭山县并入[[眉县|眉县]]。洪武十三年（1380年）眉县升为眉州，十一月复置彭山县，属眉州。[[清|清]][[康熙|康熙]]元年（1662年）彭山县并入眉州。[[雍正|雍正]]六年（1728年）复设彭山县，仍隶属眉州。1959年4月，彭山县并入眉山县，为眉山县彭山区。1962年11月，恢复彭山县。<ref name="彭山概况">[http://www.scps.gov.cn/news/aboutps/ 彭山概况] {{webarchive|url=https://web.archive.org/web/20111006222324/http://scps.gov.cn/news/aboutps/ |date=2011-10-06 }}</ref>。2014年10月21日国务院批准撤销彭山县设立眉山市彭山区<ref>[http://sichuan.scol.com.cn/ggxw/content/2014-11/03/content_9514358.htm?node=7220 长寿之乡眉山市彭山县撤县改区]</ref>。

== 地理 ==
地处[[成都平原|成都平原]]西南缘，地势东、西较高，中部为[[锦江|锦江]]、岷江河谷平原。

最高点在[[保胜乡|保胜乡]]石柱村北平观，海拔711.6米。最低点在[[江口镇|江口镇]]尖山子的石牛栏，海拔410米。<ref name="彭山概况"/>

== 历代名人 ==
彭山是古代寿星[[彭祖|彭祖]]修行和逝世的地方，也是[[晋代|晋代]]《陈情表》作者[[李密_(西晋)|李密]]的家乡。为了纪念李密的孝心，在牌坊正前方有李密和祖母的雕像。
