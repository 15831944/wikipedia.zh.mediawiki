'''彭加木'''（{{bd|1925年||1980年|06月17日（失蹤）|catIdx=P彭}}），原名'''彭家睦'''，[[中国|中国]][[广东|广东]][[番禺区|番禺]]人。中国著名生物化学家。

== 生平 ==
1947年毕业于[[國立中央大學|國立中央大學]][[南京农业大学|农学院]]农业化学系，当年在[[北京大学|北京大学]][[中国农业大学|农学院]]任助教。1949年任[[中国科学院|中国科学院]][[中国科学院上海生命科学研究院|生理生化研究所]]助理员、助理研究员。20世纪50年代开始，彭加木参加科学院综合考察委员会的工作，十余次进入[[新疆|新疆]]协助建立科研基地，后筹建中国科学院新疆分院，为新疆的研究起到巨大的推动。1961年任中国科学院[[中国科学院上海生命科学研究院|上海生物化学研究所]]副研究员、研究员。1979年兼任[[中国科学院新疆分院|中国科学院新疆分院]]院长。主要从事[[植物病毒|植物病毒]]研究及防治工作。

== 失踪 ==
<!-- [[File:彭加木字条.jpg|thumb]] -->
1980年5月，彭加木率队考察[[罗布泊|罗布泊]]，首次成功自北向南纵穿罗布泊干涸湖底。考察队缺水、缺油。6月17日，彭加木不顾个人安危，独自去寻找水源，並留下字條：
{{quote|“我往東去找水井。彭。6月17日10時30分。”}}
后其不幸迷途，可能葬身罗布泊。实践了他早年的誓言：「我准备用自己的骨头，让新疆的土壤多添一点有机质。」往后，一直未找到彭加木的踪迹。对他的失踪，全国曾风传过各种说法和猜测。多年来，官方和民间曾多次发起寻找，均一无所获。

=== 踪迹尋找 ===
2005年冬，[[中国科学院|中国科学院]]寒区旱区环境与工程研究所的[[董治宝|董治宝]]研究员在[[罗布泊|罗布泊]]进行野外考察时，在彭加木失踪地点附近发现一具[[干尸|干尸]]，可能是彭加木的遗体。一支科学探险队2006年4月13日在距[[罗布泊|罗布泊]]东缘最近的城市[[甘肃|甘肃]][[敦煌|敦煌]]集结，出发前往[[罗布泊|罗布泊]]试图确认。后发现并不属于彭加木。

2007年6月2日，探险爱好者在位于[[哈密盆地|哈密]][[大南湖|大南湖]][[戈壁|戈壁]]与[[罗布泊|罗布泊]]相接部位的[[雅丹地貌|雅丹地貌]]群拍照时，发现一具干尸。由于尸身多处特征与彭加木相符，他们怀疑这就是彭加木的遗体。<ref>{{cite news|url=http://www.china.com.cn/city/txt/2007-06/25/content_8435531.htm|title=探险者在罗布泊发现干尸 与彭加木有五大相似处|publisher=[[新疆都市报|新疆都市报]]|date=2007-06-25|accessdate=2016-10-31}}</ref>由于彭加木父母早已去世，DNA采样测试只能选择兄弟姐妹或者子女。2007年7月9日，一直关注此事的彭加木生前好友、中国科学院新疆分院的专家[[阎鸿建|阎鸿建]]查看了从罗布泊带回的6件干尸遗物取样，“我发现没有一件物品是属于彭加木的，我可以肯定的说这不是彭加木的遗体。”<ref>{{cite news|url=http://tech.sina.com.cn/d/2007-07-11/17281610748.shtml|title=罗布泊干尸不是彭加木|publisher=[[北京晚报|北京晚报]]|date=2007-07-11|accessdate=2016-10-31}}</ref>

== 紀念 ==
[[廣州市|廣州市]][[白云区_(广州市)|白云区]]松洲街槎龙村建有[[彭加木公園|彭加木公園]]以紀念他的貢獻，公園由槎龙村和熱心人士出資建設，佔地約4萬平方米，於2002年12月18日向公眾開放，是白雲區青少年愛國主義教育基地。園內除了有彭加木的紀念館和銅像外，還建有來自該村的[[林成佑|林成佑]]、[[張悅楷|張悅楷]]等人的紀念館。<ref>{{Cite web |url=http://www0.by.gov.cn/cn/citizen/content.asp?id=200304161514513317 |title=彭 加 木 公 园 |accessdate=2014-03-20 |author=区委宣传部 |work= |publisher=白云信息网 |date=2003-04-16 |language=zh-hans }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
