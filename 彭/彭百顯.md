'''彭百顯'''{{BD|1949年|6月14日|||P彭}}，[[臺灣|臺灣]][[政治人物|政治人物]]，[[南投縣|南投縣]][[國姓鄉|國姓鄉]]人。

==生平==
彭百顯為南投政壇名人[[彭華英|彭華英]]的曾孫。1968年從[[國立臺中第二高級中學|臺灣省立臺中第二中學]]（今[[國立臺中第二高級中學|國立臺中第二高級中學]]）高中部[[畢業|畢業]]後，進入[[文化大學|文化大學]][[經濟學|經濟學]]系就讀，並於文化大學經濟學[[研究所|研究所]]取得[[碩士|碩士]][[學位|學位]]。往後彭百顯除了在文化大學經濟-{系}-任職之外，也在[[國立中興大學法商學院|國立中興大學法商學院]](今[[國立臺北大學|國立臺北大學]])任教，且出任文化大學銀行-{系系}-主任。之後彭百顯受聘為[[合作金庫|合作金庫]]高級專員；於[[美國|美國]][[國際學士院大學|國際學士院大學]]（INTERNATIONAL ACADEMY OF EDUCATION）取得經濟學[[博士|博士]]。

1986年[[民主進步黨|民主進步黨]]成立後，彭百顯於南投縣參選[[國大代表|國大代表]]，僅以三千票之差距落敗；1989年彭百顯參選末代增額[[立法委員|立法委員]]並且當選，接著於1992年、1995年獲得連任。擔任立法委員期間獲得甚高評價，曾獲選1993年《[[天下雜誌|天下雜誌]]》「新世代領導者」；歷年國會記者評鑑中，彭百顯也獲得「預算審查表現」第一名與「專業素養」的第一名，與[[許添財|許添財]]都是民進黨內的財經智囊。在[[陳定南|陳定南]]參選[[臺灣省省長|臺灣省省長]]與[[彭明敏|彭明敏]]參選總統時，彭百顯也是重要的輔選幹部。

1997年彭百顯未受民進黨提名參選南投縣長，便[[退黨|退黨]]參選南投縣長，擊敗[[中國國民黨|中國國民黨]]提名的[[許惠祐|許惠祐]]與民進黨提名的[[林宗男_(政治人物)|林宗男]]。1998年，彭百顯出任[[新國家連線|新國家連線]]召集人。1999年的[[921大地震|921大地震]]中，南投縣受創嚴重，彭百顯開始進行救災工作；為了獲得重建經費，於[[2000年中華民國總統大選|2000年中華民國總統大選]]中支持[[連戰|連戰]]。2001年1月12日，彭百顯由於涉及[[貪污|貪污]]罪名受到[[臺灣南投地方法院檢察署|南投地檢署]]羈押六十一天，並且被求處[[有期徒刑|有期徒刑]]二十年，一直到2004年11月下旬才由[[臺灣高等法院|臺灣高等法院]]判處無罪。<ref>[http://udn.com/NEWS/SOCIETY/SOC6/6466379.shtml 弊案纏訟10年 前南投縣長彭百顯無罪確定]{{dead link|date= |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/paper/512299 |title = 貪瀆冤案 纏訟11年無罪 彭百顯嘆公道難求 |author = |date = 2011-07-29 |publisher = 自由時報 |language = zh-hant |accessdate =  |quote = |via = }}</ref>

2002年6月15日，在當時[[財經立法促進社|財經立法促進社]]會長[[吳東昇|吳東昇]]邀請下，彭百顯在[[立法院|立法院]]表示，臺灣是一個[[主權國家|主權國家]]，只要做好「有效管理」，不應害怕[[三通|三通]]；三通可以使臺灣[[國際化|國際化]]、可以提升臺灣的國際地位、可以幫助臺灣加入[[聯合國|聯合國]]；若臺灣害怕三通，將給[[中華人民共和國政府|中華人民共和國政府]]更多藉口認定臺灣是中華人民共和國的一省；在三通談判上，他主張[[臺海兩岸|臺海兩岸]]政府都有權行使各自的[[國家主權|國家主權]]，暫時拋棄「含蓋對方」的主權主張，以「港對港」、「內對內」方式進行三通。  

2008年1月4日，彭百顯在「挺台聯，正本土」[[記者會|記者會]]提出「新[[黨外運動|黨外運動]]」、「[[臺灣人|新臺灣人]]運動」，主張認同臺灣者都是新臺灣人，臺灣應該要有[[第三勢力|第三勢力]]，由[[台灣團結聯盟|台灣團結聯盟]]（台聯）[[精神領袖|精神領袖]][[李登輝|李登輝]]擔任臺灣第三勢力精神領袖，以台聯為臺灣第三勢力平台。

目前彭百顯為[[開南大學|開南大學]]財務金融學系主任暨財務金融研究所所長。

2010年3月13日，[[李登輝民主協會|李登輝民主協會]]舉行成立大會，彭百顯出任李登輝民主協會首任秘書長。<ref>{{Cite web |url = https://presidentlee.tw/李登輝民主協會成立過程/ |title = 李登輝民主協會成立過程 |author =  |date = 2010-07-09 |publisher = 李登輝基金會 |language = zh-hant |accessdate = 2018-10-23 |quote = |via = }}</ref>

2011年2月24日，彭百顯擔任[[國家展望文教基金會|國家展望文教基金會]]中國研究中心產業組召集人。
2011年7月14日，[[中華民國最高法院|中華民國最高法院]]判決，彭百顯涉及的多項921重建工程弊案無罪定讞；同月28日，彭百顯在南投召開說明會感謝鄉親十一年來含淚相挺，南投縣長[[李朝卿|李朝卿]]到場致意並說「老天有眼」。

2014年10月，彭百顯表態支持受民進黨提名參選南投縣長的[[李文忠_(臺灣)|李文忠]]。

2017年7月27日，彭百顯接任台銀證券獨立董事，不到4個月，隨即轉任台銀獨董。<ref>{{Cite web |url = https://www.cna.com.tw/news/afe/201711170344.aspx |title = 任台銀證獨董3個多月 彭百顯轉任台銀獨董 |author =  |date = 2017/11/18 |publisher = 中央通訊社 |language = zh-hant |accessdate =  |quote = }}</ref>

2018年5月，彭百顯公開支持民進黨提名南投縣長參選人洪國浩。<ref>{{Cite web |url = https://udn.com/news/story/7325/3159717 |title = 前南投縣長彭百顯 公開支持縣長參選人洪國浩 |author = |date = 2018-05-24 |publisher = 聯合新聞網 |language = zh-hant |accessdate = 2018-10-23 |quote = }}</ref><ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/2436653 |title = 綠營大團結挺洪國浩 前縣長彭百顯：盼他再創新歷史 |author = |date = 2018-05-24 |publisher = 自由時報 |language = zh-hant |accessdate = 2018-10-23 |quote = }}</ref><ref>{{Cite web |url = https://udn.com/vote2018/story/10958/3434717 |title = 藍綠南投輔選戰／朱立倫挺林明溱 蘇嘉全讚洪國浩 |author =  |date = 2018-10-22 |publisher = 聯合新聞網 |language = zh-hant |accessdate = 2018-10-23 |quote = }}</ref>
