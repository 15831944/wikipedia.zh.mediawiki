'''彭亨州'''（{{lang-ms|Pahang}}），全稱{{lang|ms|Pahang Darul Makmur}}（意為“彭亨-繁荣的家邦； Darul:家邦，Makmur：繁荣），是[[西马来西亚|西马来西亚]]最大的州属，州[[首府|首府]]为[[关丹|关丹]]（{{lang|ms|Kuantan}}），皇城位于[[北根|北根]]（{{lang|ms|Pekan}}）。马来西亚半岛最高山峰[[大汉山|大汉山]]（{{lang|ms|Gunung Tahan}}）位于彭亨州北部，面积35,964平方公里，人口1,448,000。州最高统治者为苏丹阿末沙（{{lang|ms|Sultan Haji Ahmad Shah Al-Mustain Billah ibni Almarhum Sultan Sir Abu Bakar Riayatuddin Al-Muadzam Shah}}）。

== 历史 ==
在中國史書上，彭亨最早见于《[[唐书|唐书]]》，称为婆凤。{{fn|1}}彭亨自[[南宋|南宋]]即与[[福建|福建]]省通商，1206年的《[[云麓漫钞|云麓漫钞]]》卷第五载“佛囉安、'''朋豐'''、達囉啼、達磨國則有木香。”“朋豐”就是彭亨。{{fn|2}}

南宋時期的《[[諸蕃志|諸蕃志]]·三佛齐国》作“蓬豐”，为[[三佛齐|三佛齐]]属国。{{fn|3}}元朝的 《[[島夷誌略|島夷誌略]]》作“彭坑”。“石崖週匝崎嶇，遠如平寨，田沃，谷稍登。氣候半熱。男女穿長衫，繫單布捎；富貴女頂帶金圈數四；常人以五色硝珠為圈以束之。民煮海為鹽。釀椰槳為酒，有酋長。地產黃熟香、頭沉速、打白香、腦子、花錫、粗降真。貿易之貨用諸色絹、[[闍婆|闍婆]]布、銅鐵器、漆瓷器、鼓、板之屬。”{{fn|4}}

明朝的《星槎胜览》作“彭坑”，《[[郑和航海图|郑和航海图]]》作“彭杭”，《[[东西洋考|东西洋考]]》作'''彭亨'''。

[[洪武|洪武]]十一年（1379年）彭亨王''麻哈剌惹答饒''遣使奉金表朝貢。[[永乐|永乐]]十年（1413年），[[鄭和|鄭和]]出使彭亨國。[[永樂|永樂]]十二年（1415年）彭亨王遣''蘇麻固門的里''來朝並貢方物”。

[[万歷|万歷]]年間，[[柔佛|柔佛]]國副王子娶彭亨王女，婚前，柔佛國副王送王子到彭亨，彭亨王設宴款待，親戚畢會。彭亨王妹婿是婆羅國王子，當時也在座，舉杯向柔佛國副王道賀。當時婆羅國王子手指上帶著一顆大珍珠，閃閃發光。柔佛國副王願出重金換取大珠，但婆羅國王子不肯割愛，柔佛國副王惱羞成怒，歸國後立刻發兵來攻打彭亨。彭亨人大出所料，不戰自潰，彭亨王和婆羅王子逃奔金山。彭亨王妃的哥哥是[[浡泥|浡泥]]國王，聞訊率兵來援救。柔佛國副王在彭亨大肆焚掠而去。 當時，彭亨國中鬼哭三日，人民半死。浡泥王將妹妹和彭亨王迎回浡泥，彭亨王命長子攝國。后來彭亨王复位，彭亨王次子素來凶悍，毒殺父王，並弒兄自立為彭亨王。{{fn|5}} 

清朝出版的《[[明史|明史]]》作彭亨。

== 地理 ==
彭亨州位于[[马来西亚|马来西亚]]半岛东海岸，共35,964 平方公里，由高山及海岸所组成。约80% 土地为原始森林，大马主干山脉蒂迪旺沙山脉穿越此州，使此州拥有著名的[[金马仑高原|金马仑高原]]、[[云顶高原|云顶高原]]、[[国家公园|国家公园]]等。

== 旅游名胜地 ==
* [[金马仑高原|金马仑高原]]：马来西亚最大的高原度假胜地，盛产茶叶，它也是马来西亚最大的温带蔬菜与花朵种植区。
* [[云顶高原|云顶高原]]：由丹斯里[[林梧桐|林梧桐]]所建立，是马来西亚唯一的合法[[赌场|赌场]]，也是[[主题公园|主题公园]]，拥有世界上最大的[[酒店|酒店]]。
* [[福隆港|福隆港]]：是以看鸟闻名的高原度假胜地。
* [[大汉山国家公园|大汉山国家公园]]：全马最大的原始森林公园，大马半岛最高峰[[大汉山|大汉山]]座落于此。
* [[比拉湖|比拉湖]]：大马最大的自然湖。
* [[珍妮湖|珍妮湖]]：富有神秘色彩与美丽传说的湖泊，也有莲花湖之称。
* [[刁曼岛|刁曼岛]]：被喻为世界十大最美丽的岛屿之一。
* [[遮拉汀|遮拉汀]]：以環境優美的熱帶雨林生態著稱，有多家知名飯店在此。
