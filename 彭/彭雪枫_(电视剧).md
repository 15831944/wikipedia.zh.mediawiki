《'''彭雪枫'''》，中国大陆电视剧，导演[[贾钢|贾钢]]，主演孙洪涛、秦梵翔、张琳、王霙、王伍福、沈保平。

== 剧情 ==
该剧时间跨度从1937年[[抗日战争|抗日战争]]开始一直到1944年彭雪枫中流弹牺牲，讲述了彭雪枫在抗日战争中的英雄事迹。

== 花絮 ==
该剧是“纪念彭雪枫诞辰100周年”献礼剧。<ref>{{cite web |url=http://news.xinhuanet.com/newscenter/2007-10/09/content_6852977.htm |title=电视剧《彭雪枫》研讨会在北京举行 |accessdate=2007年10月9日 |publisher=新华网 |work= |language =zh-cn}}</ref>
