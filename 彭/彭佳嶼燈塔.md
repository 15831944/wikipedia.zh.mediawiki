'''彭佳嶼燈塔'''是一座位於[[臺灣|臺灣]]基隆市[[彭佳嶼|彭佳嶼]]的[[灯塔|燈塔]]，該燈塔落成於1909年。該建物為[[臺灣地區|臺灣地區]]第三高燈塔，並在2001年[[臺灣歷史建築百景列表|歷史建築百景徵選活動]]中被票選為第12名。彭佳嶼目前為軍事管制區，故該建物無對外開放參觀。

== 歷史 ==
彭佳嶼燈塔開始動工於[[臺灣日治時期|日治時期]]的1906年，原本預定在1908年完工，但是因為製造廠方面延遲交貨，所以無法安裝燈器等零件。到了1909年5月上旬，燈器等零件才送到，並在該年7月完成後續工程，最後在9月20日正式啟用。在此同時，為了指引搬運材料的船隻而於1907年所設的臨時燈桿也予以拆除。

[[二次世界大戰|二次世界大戰]]時，燈塔屢遭[[盟軍|盟軍]]轟炸。在1945年1月9日被首次轟炸後，燈塔便因燈具受損而失去作用。後來在該年的1月11日、5月15日與5月16日燈塔又被轟炸而造成燈塔與房舍的嚴重毀損。戰後，燈塔由[[中華民國|中華民國]]海關接收並於1946年6月19日修復，後來在1948至1949年期間，又將附近原本為四坡式木造鋪瓦屋頂的磚造辦公室及宿舍改建，而成為現在的磚牆水泥平頂建築。<ref name="文"/>

== 燈塔結構諸元 ==
*塔高：26.2公尺。
*塔身塗色・構造：白色磚造圓塔，底座則呈八角形，與一棟方形屋舍連接。
*燈高：145.5{{〈}}以高潮面至燈火中心計{{〉}}。
*燈器：白熱石油燈，配備一等旋轉透鏡與[[霧笛|霧笛]]（過去是[[霧砲|霧砲]]）。
*燈質（頻率）：每15秒閃一次白光。
*[[發光強度|光力]]：白光864,000[[燭光|燭光]]。<ref name="文">{{cite book |title=《台灣歷史建築百景專輯》 |date=2004-06-30 |edition=二版 |publisher=[[行政院文化建設委員會|行政院文化建設委員會]] |location=臺北市 |isbn=957-01-2749-X |pages=30頁 }}</ref>
*公稱[[光程|光程]]：25.3[[浬|浬]]。
*明弧：四週皆見燈光，但在距離該島2浬內264度-316度被本島遮蔽，不見燈光。

===燈塔行政諸元===
*管轄機關：燈塔原為[[中華民國|中華民國]][[財政部關務署|財政部關務署]]管理，已於2013年1月1日轉交由中華民國[[交通部航港局|交通部航港局]]管轄。
*人員編制：一名主任管理員一名與五名技工。所需的生活物資仰賴每月來自基隆的補給船供應，可輪流返回臺灣休假。
*開放參觀與否：否。

== 周圍設施 ==
彭佳嶼上除了燈塔之外，還有軍事設施與氣象臺。<ref name="文"/>
