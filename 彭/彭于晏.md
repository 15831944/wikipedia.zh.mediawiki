'''彭于晏'''（{{bd|1982年|3月24日||catIdx=P}}），知名[[男演員|男演員]]，出生於[[台灣|台灣]][[澎湖縣|澎湖縣]]，13歲和家人[[移民|移民]]到[[加拿大|加拿大]]，進入加拿大[[英屬哥倫比亞大學|英屬哥倫比亞大學]]，主修[[经济学|經濟學]]。大二暑假外婆過世，他回台奔喪，被導演邀請參加甄選演出偶像劇《[[愛情白皮書|愛情白皮書]]》。彭于晏回憶說：「我抱持打工的心情去嘗試，當時也沒簽任何合約就去拍戲的。」因進入演藝圈而休學專心拍片。<ref>[https://bka.mpweekly.com/interview/%E5%A8%9B%E6%A8%82123/20180901-129346 【彭于晏專訪】彭于晏讚「彭太太」可愛]</ref>。

== 公益活動 ==
*2009年11月電影《聽說》在香港上映期間，曙光影視發行公司與慈善機構[[龍耳|龍耳]]合作舉辦了慈善首映禮<ref>{{cite web |language=zh-cn |url=http://v.ent.163.com/video/2009/11/E/G/V5O1QH3EG.html | title=《聽說》香港慈善首映 為聽障人士籌集愛心 |publisher=[[網易|網易]] |date=2009-11-25 |accessdate=2011-08-14}}</ref><ref>{{cite web | language=zh-hk | url=http://silence.org.hk/File/HearMe.pdf | title=龍耳呈獻﹕《聽說》首映禮 | work=[[龍耳|龍耳]] | date=2009-11-24 | accessdate=2010-08-14 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在此次籌辦首映禮，彭于晏聯同[[聽障|聽障]]隊員與[[香港明星足球隊|香港明星足球隊]]進行足球友誼賽<ref>{{cite web |language=zh-hk |url=http://www.youtube.com/watch?v=yngQmhC_Dlk | title=4-11-2009 友誼賽 龍耳足球隊 + 彭于晏 VS 明星足球隊 |publisher=[[Youtube|Youtube]] |date=2009-11-20 |accessdate=2011-08-14}}</ref><ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20091105&sec_id=462&art_id=13386668 | title=彭于晏樂得黐實蔡依林 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2010-11-05 |accessdate=2011-08-14}}</ref>，並得到[[勞工及福利局|勞工及福利局]]局長[[張建宗|張建宗]]、[[電影演員|影視明星]][[曾志偉|曾志偉]]、[[羅慧娟|羅慧娟]]及[[立法會議員|立法會議員]]梁耀宗及正言匯社[[張超雄|張超雄]]等人的致函支持，亦得到多家媒體報道<ref>{{cite web |language=zh-hk |url=http://www.youtube.com/watch?v=o_PMdfC_1i0 | title=「聽說」香港宣傳 彭于晏踢球為聽障募款  |publisher=[[中天新聞|中天新聞]] |date=2009-11-04 |accessdate=2011-08-14}}</ref><ref>{{cite web |language=zh-hk |url=http://www.hkheadline.com/ent/ent_content.asp?contid=93219&srctype=n | title=許志安極速減磅增賣點 |publisher=[[頭條日報|頭條日報]] |date=2010-11-05 |accessdate=2011-08-14}}</ref>，向社會大眾推廣「聾健共融」的理念。
*彭于晏在香港宣傳電影《聽說》期間，慈善機構[[龍耳|龍耳]]派出手語導師協助下，拍攝一系列十集《手語教室》片段<ref>{{cite web |language=zh-hk |url=http://www.youtube.com/watch?v=12_Q1miwNno | title=港版 聽說 手語教學  |publisher=[[Youtube|Youtube]] |date=2010-12-18 |accessdate=2011-08-14}}</ref>，於2009年11月每天早、午、晚時段，於[[亞洲電視|亞洲電視]]平台上播放，藉此加強公眾對手語的認識。
