'''圭亞那合作共和国'''（{{lang-en|Cooperative Republic of Guyana}}<ref name="gyn">[http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/nmz_608635/1206_608832/ 圭亚那国家概况]</ref><ref>{{Cite web|url=http://www.motp.gov.gy/|title=Ministry of the Presidency|accessdate=2017-02-07|work=www.motp.gov.gy|language=en-gb}}</ref><ref>[http://www.guyanaembassybeijing.cn/indexch.asp 圭亚那驻华大使馆] {{webarchive|url=https://web.archive.org/web/20150923132802/http://www.guyanaembassybeijing.cn/indexch.asp |date=2015-09-23 }}</ref>），通称'''圭亚那'''，位於[[南美洲|南美洲]]北部，是南美洲唯一以[[英語|英語]]為[[官方語言|官方語言]]的[[国家|国家]]，也是[[英联邦|英联邦]]成员國。東鄰[[蘇利南|蘇利南]]，南臨[[巴西|巴西]]，西鄰[[委内瑞拉|委内瑞拉]]，北鄰[[大西洋|大西洋]]。圭亞那與蘇利南和委内瑞拉有國界爭議。尤其委内瑞拉一再聲稱[[埃塞奎博河|埃塞奎博河]]以西的土地─[[西屬圭亞那|西屬圭亞那]]（約佔圭亞那面積的三分之二）屬於委內瑞拉。圭亞那雖地處南美洲，為[[南美洲國家聯盟|南美洲國家聯盟]]的成員國，但傳統上及歷史上與加勒比海諸島的關係較为密切。

== 历史 ==
{{main|圭亞那历史}}
1498年[[克里斯托弗·哥伦布|克里斯托弗·哥伦布]]發現蓋亞那及其鄰近地區後，歐洲移民紛至沓來。隨後三個世紀該地區輪流由英國、荷蘭、法國佔領，直到19世紀初1814年英、法、荷三國訂立界約，蓋亞那地區正式由英國殖民統治，稱為英屬圭亞那（British Guiana）。1961年獲得自治，改稱圭亞那（Guyana）。1966年5月26日獨立，1970年2月23日成立圭亚那合作共和国<ref name="gyn" />。

== 人口 ==
蓋亞那山深林密，人口稀少約80萬人，英國殖民者最初利用由非洲購得之奴隸來開墾，1834年奴隸制度廢除，殖民當局轉由南亞的印度與東南亞地區，以契約模式引進廉價勞工，使蓋亞那的農業及礦業能持續發展，此舉也使蓋亞那成為多民族地區。735,554人（2014年）
印度裔約佔49%，黑非裔32%，印非混血12%，餘為土著、華裔及歐洲裔(約7%)。

== 地理 ==

位於南美大陸北緣，北濱大西洋，東為蘇利南，南與巴西接壤，西接委內瑞拉。

== 經濟 ==
財政收支：税入2億8千7百萬美元；税出3億7百萬美元（2004年）。輸出總值：4.994億美元（2007年）；輸入總值：8.358億美元（2007年）。主要輸出項目：鐵礬土、糖、蝦、木材、黃金及米等。主要輸入項目：資本財、燃料、機械、化學品、建材及消費品等。主要出口國：除[[加勒比共同體|加勒比共同體]]之會員國外，另為美、英及加拿大等國。主要進口國：美、英及加拿大等國。

== 其他 ==
圭亞那是唯一仍舊視[[同性戀|同性戀]]為非法地位的[[南美洲|南美洲]]國家。
