'''幕張站'''（{{jpn|j=幕張駅|hg=まくはりえき|rm=Makuhari eki|f=y}}）是位於[[日本|日本]][[千葉縣|千葉縣]][[千葉市|千葉市]][[花見川區|花見川區]]{{link-ja|幕張|幕張|幕張町}}五丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[總武本線|總武本線]]的[[鐵路車站|鐵路車站]]。

此站只有[[中央·總武緩行線|中央·總武線各站停車]]停靠。

== 車站結構 ==
本站是一個[[地面車站|地面車站]]，站房為[[跨站式站房|跨站式站房]]設計，橫跨總武本線[[多重複線|雙複線]]軌道。車站出口設於軌道兩旁，稱為北口和南口，其中北口由於只以無簷篷遮蓋的樓梯連接跨站式站房和地面，使用上不及南口便利。站內設有[[綠色窗口|綠色窗口]]（營業時間：06:00－22:00），[[Suica|Suica]]自動剪票口和對號座位自動售票機等設備。

車站月台採[[島式月台|島式月台]]2面3線設計，其中2、3號月台共用中間供列車折返用的軌道。以前曾有在本站折返的班次，但經歷年多次訂正時間表後，現在只剩下一班早上自本站開出的班次。此外，當班次因事故而延誤時，部分班次會臨時以本站為折返站。而本站西側是[[幕張車輛中心|幕張車輛中心]]，部分進入該中心的列車也會停靠本站。

在[[成田國際機場|成田國際機場]]啟用初期，航空燃油均由[[京葉臨海鐵道|京葉臨海鐵道]]經本站折返運往成田機場。當時為了防止反對興建機場的激進派人士攻擊停靠本站運送燃油的列車（當時快速線尚未開業），2號月台在1978年至1983年間全面鋪上[[鐵絲網|鐵絲網]]封閉，不作客運用途。

== 其他 ==
因為站名的問題，部分前往[[幕張新都心|幕張新都心]]地區、[[美濱區|美濱區]]和[[幕張展覽館|幕張展覽館]]的乘客會誤認本站是最近的車站而下車。早期由於沒有巴士連接，所以誤乘的乘客需要步行20至30分鐘或乘[[計程車|計程車]]前往。現時則有[[千葉海岸巴士|千葉海岸巴士]]和[[京成巴士|京成巴士]]來往本站和幕張新都心地區所在的[[海濱幕張站|海濱幕張站]]。

此外，本站由月台往大堂的樓梯上方均有避免乘客誤乘的告示：「前往幕張新都心、駕駛執照中心的乘客請在海濱幕張站下車」。

== 歷史 ==
* 1894年（[[明治|明治]]27年）12月9日－以[[總武鐵道|總武鐵道]]車站的名義開業，兼營客貨運業務。
* 1907年（明治40年）9月1日－政府根據[[鐵路國有法|鐵路國有法]]收購總武鐵道，成為[[日本國有鐵道|日本國有鐵道]]車站。
* 1973年（[[昭和|昭和]]48年）11月1日－停辦貨運業務。
* 1987年（昭和62年）4月1日－[[國鐵分割民營化|國鐵分割民營化]]，本站由JR東日本接手管理。
* 2001年（[[平成|平成]]13年）11月18日－智能卡系統[[Suica|Suica]]正式投入服務。
