'''幕末'''，即'''江户末年'''，是指[[日本|日本]]历史上[[德川幕府|德川幕府]]统治时代（[[江户时代|江户时代]]）的末期。一般意义上的“幕末”时期从1853年[[黑船來航|黑船来航]]开始，终于1869年[[戊辰戰爭|戊辰战争]]。

== 概述 ==
虽然对于“幕末”的起止时间没有统一、严密的定义，但一般认为始于1853年7月8日（[[嘉永|嘉永]]6年旧历6月3日）[[黑船|黑船]]（即[[马休·佩里|马休·佩里]]所率领的美国海军舰队）来航。即使在《[[王政复古|王政复古大号令]]》（1868年）中，这次“抑癸丑（1853年）以来未曾有之国难”也被看作开启了国家体制改革的序幕。

而“幕末”的结束标志，有观点认为是1867年11月9日（[[慶應|庆应]]3年旧历10月14日）幕府[[征夷大將軍|征夷大将军]][[德川庆喜|德川庆喜]]进行[[大政奉还|大政奉还]]，也有观点认为应为1868年5月3日（旧历4月11日）[[江戶開城|江户无血开城]]。此外，也有人将1869年[[箱館戰爭|箱馆战争]]结束、旧幕府军投降新政府军，1871年8月29日（[[明治|明治]]4年旧历7月14日）实行[[廢藩置縣|废藩置县]]、幕藩体制完全终结，以及明治政府采用西欧式太阳历（[[格里高利历|格里高利历]]）的前一天——1872年12月31日（明治5年旧历12月2日）等不同时间作为幕末的结束标志。

幕末时期，受到所谓“西洋的冲击”，以[[水戶學|水户学]]等华夷思想为基础，国防意识高涨，国家[[民族主義|民族主义]]也逐渐兴起；另一方面，德川幕府所支配的幕藩体制政治正走向解体，国内政治势力正面临着重组的格局，处于政治中心的是[[薩摩藩|萨摩藩]]、[[长州藩|长州藩]]、[[土佐藩|土佐藩]]、[[佐賀藩|肥前藩]]等西南雄藩。在这一时期，幕府放弃了从二代将军[[德川秀忠|秀忠]]开始延续下来的“[[锁国_(日本)|锁国]]”政策，被迫开港，开始与外国实现自由贸易，卷入世界范围内的资本主义市场经济与大国[[殖民主义|殖民主义]]的浪潮之中。尽管不同形式的排外主义（[[尊王攘夷|尊王攘夷论]]）不断抬头，但当时的整个社会仍忌惮于西方列强压倒性的军事实力，继而在西方文明的强烈影响下发生了巨大的变化。

== 政治史 ==
'''条约缔结与将军继嗣问题（1853年 - 1858年）'''
{{main|黑船来航}}
[[File:Shinagawa_Baidai_cannon.jpg|thumb]]台场所设立的80磅铜制加农砲（口径250mm、炮身长3830mm）]]
1853年（嘉永6年）7月8日（旧历为6月3日），[[美利坚合众国|美国]]海军提督[[马休·佩里|马休·佩里]]率4艘[[黑船|黑船]]来到[[浦賀|浦贺]]港<ref>同日、[[久里浜|久里浜]]で[[浦賀奉行|浦賀奉行]]以下60人が大筒[[ボンベン・モルチール砲|ボンベン・モルチール砲]]の砲撃訓練をしていた。訓練は三日目に入っていた。夕方、異国船4隻が沖合を通り過ぎ、浦賀沖に投錨した。そのうち2隻が蒸気軍艦であった。（井上勝生『幕末・維新』シリーズ日本近現代史① 岩波書店 〈岩波新書1042〉 2006年 2ページ）</ref>，并带来了要求[[江户幕府|江户幕府]]开国通商的[[美国总统|总统]]国書。[[老中|老中]]首座[[阿部正弘|阿部正弘]]（[[備後福山藩|备后福山藩]]藩主）向[[海防参与|海防参与]][[德川齊昭|德川齐昭]]（前[[水戶藩|水户藩]]藩主）、[[松平春嶽|松平庆永]]（春岳、[[越前藩|越前藩]]藩主）・[[島津齊彬|岛津齐彬]]（[[萨摩藩|萨摩藩]]藩主）等[[亲藩大名|亲藩]]及[[外樣大名|外样大名]]征求对策，诸大名得以介入幕政。在这场黑船事件引起的外交风波中，12代[[征夷大將軍|将军]][[德川家慶|'''德川家庆''']]去世，世子'''[[德川家定|德川家定]]'''继任第13代将军。

次年（1854年，嘉永7年）2月13日（旧暦为1月16日 ）马休·佩里再度前来重申对幕府开国的要求。在大学頭[[林復斎|林復斋]]等与佩里等进行交涉之后，于1854年（嘉永7年）3月31日]]（旧暦为3月3日）签订[[神奈川條約|日美和亲条约]]，德川幕府的“锁国”体制就此终结。此后，[[英國皇家海軍|英国皇家海军]]上将[[詹姆士·史特靈|詹姆士·史特灵]]与水野忠德在1854年（嘉永7年）10月14日签订[[日英和亲条约|日英和亲条约]]，[[俄罗斯帝国|俄罗斯帝国]]海军上将[[普提雅廷|普提雅廷]]与[[川路圣謨|川路圣謨]]等在1855年（安政元年）2月7日签订[[日俄和亲条约|日俄和亲条约]]，[[日荷和亲条约|日荷和亲条约]]也于1856年1月30日缔结生效。为了在开国之后改革幕政，阿部正弘开始加大人才启用力度，并创设高等教育设施，重整幕府军事体制，这场从开国以前就一直持续的改革也被称作“安政改革”。值得一提的是，[[胜海舟|胜海舟]]也在这次外交风波中崭露头角，开始登上政治舞台。

'''安政大狱与樱田门外之变（1858年 - 1860年）'''

1858年（安政5年）6月4日（旧历为4月23日），就任大老的[[井伊直弼|井伊直弼]]（[[彥根藩|彦根藩]]主）意图通过强硬手法一鼓作气地解决所面临的两个幕政难题：悬而未决的将军继嗣问题以及愈发严峻的开国问题。具体而言，对于新任幕府将军一职，就在大老就任之后的1858年（安政5年）6月11日（旧历为5月1日），幕府将[[紀州藩|纪州藩]]的[[德川家茂|庆福]]定为次任将军，同时庆福改名为'''德川家茂'''，进入江户城，并于安政5年10月25日就任将军一职。而对于条约签署问题，井伊直弼虽然自己认为得到天皇勅许是有必要的，但由于受到松平忠固的强烈反对，遂于7月29日（安政5年6月19日），在还没有得到天皇敕许的情况下就命井上清直与岩濑忠震全权负责所谓“'''日美修好通商条约”'''的签订。此后幕府又与英、法、荷、俄四国签署了同样的通商条约（安政五国条约）。通商城市与港口的开放是分阶段进行的（开市开港延期），但这是由井伊的后继者安藤信正所派遣的文久遣欧使节在签署所谓“伦敦备忘录”时才得以实现的。
