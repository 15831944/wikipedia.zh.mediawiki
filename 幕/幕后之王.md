== 剧情概述 ==
布小谷（周冬雨 饰）是传媒学院学生，她心怀梦想，立志成为优秀的节目制作人，把顶级制作人淳于乔（罗晋 饰）视为自己的榜样，通过不断努力去追随淳于乔的脚步。两人开始在工作中并肩战斗，共同应对来自嘉宾、播出平台、广告商等纷繁复杂的问题和挑战，携手奋斗，也在感情上互相吸引。他们也曾站在对立面，却从没有忘记制作节目的初衷，最终他们收获了事业、爱情，也为他们所热爱的传媒行业探索出更美好的明天。

== 制作 ==
2017年12月1日，梦幻星生园影视2018项目发布会上宣布《幕后之王》由李骏导演<ref>{{cite news|url=http://www.dzwww.com/yule/tt/201712/t20171204_16740230.htm|title=《幕后之王》将开机 热播缔造者李骏开职场剧新题材|date=2017-12-04|publisher=大众网|accessdate=2017-12-24}}</ref>，其官方微博也宣布由演员[[周冬雨|周冬雨]]、[[罗晋|罗晋]]主演，但之后被删除。从骅威文化子公司（其全资子公司为梦幻星生园影视）发布的新剧计划，可以查看到“公司计划在2018年完成拍摄、制作及发行的《幕后之王》请到了影星周冬雨、罗晋加盟主演”<ref>{{cite news|url=http://www.cs.com.cn/sylm/jsbd/201712/t20171201_5603079.html|title=骅威文化子公司发布新剧计划 大制作+大IP助力影视“精品”升级|date=2017-12-01|publisher=中国证券报|accessdate=2017-12-24}}</ref>。

12月22日本剧在[[杭州市|杭州市]]开机。

据骅威文化公告显示，本剧成本拟定为人民币2.8亿元<ref>{{cite news|url=http://finance.sina.com.cn/stock/s/2018-01-05/doc-ifyqiwuw6648963.shtml|title=周冬雨新剧单集价867万 骅威文化非公开发行股尚未批|date=2018-01-05|publisher=新浪|accessdate=2018-01-05}}</ref>，还未拍摄完成便以3.9亿元的价格向浙江天猫技术有限公司转让了独家信息网络传播权(全球)<ref>{{cite news|url=http://data.p5w.net/t1204299097.html|title=骅威文化：关于签署重大协议的公告|date=2018-01-03|publisher=全景网|accessdate=2018-01-05}}</ref>。

2018年4月14日正式杀青。

2019年1月3日首发由[[张靓颖|张靓颖]]献声的主题曲MV《我爱你三个字能倒过来》<ref>{{cite news|url=http://ent.ifeng.com/a/20190103/43161462_0.shtml|title=《幕后之王》张靓颖献声 周冬雨罗晋诠释新型情感观|date=2019-01-03 |publisher=凤凰网|accessdate=2019-01-04}}</ref>。
