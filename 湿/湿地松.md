'''湿地松'''（[[学名|学名]]：{{lang|la|''Pinus elliottii''}}），又稱'''愛氏松'''，为[[松科|松科]][[松属|松属]]的植物。

== 形态 ==
常绿[[乔木|乔木]]，高可达36米；黄褐色粗壮的小枝，叶子2-3针一束，长15-30厘米，2-10内生的树脂管；椭圆状圆锥形红褐色毬果，长7.5-17厘米，有短柄，种鳞的鳞脐有尖刺。

== 分布 ==
原產於[[美國|美國]]東南部[[阿拉巴馬州|阿拉巴馬州]]，[[佛羅里達州|佛羅里達州]]等地，分布于[[台湾岛|台湾岛]]、[[美国东南部|美国东南部]]以及[[中国大陆|中国大陆]]的[[江浦|江浦]]、[[福建|福建]]：[[闽侯|闽侯]]、[[广东|广东]]：[[广州|广州]]、[[江西|江西]]：[[吉安|吉安]]、[[江苏|江苏]]：[[南京|南京]]、[[桂林|桂林]]、[[台山|台山]]、[[湖北|湖北]]：[[武汉|武汉]]、[[浙江|浙江]]：[[安吉|安吉]]、[[安徽|安徽]]：[[泾县|泾县]]、[[广西|广西]]：[[柳州|柳州]]、[[香港|香港]]等地，目前已由人工引种栽培。由於在[[香港|香港]]廣泛被種植，因此與[[台灣相思|台灣相思]]，[[紅膠木|紅膠木]]並稱為「植林三寶」。

== 用途 ==
湿地松材质比较松软，耐朽力强；亦为优良的产脂树种，产脂多，质量好。
