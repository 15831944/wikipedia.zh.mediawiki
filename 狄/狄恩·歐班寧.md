'''查理·狄恩·歐班寧'''（'''Charles Dean O'Banion'''，{{bd|1892年|7月8日|1924年|11月10日}}）是一名美國的黑幫，他是1920年代殘酷的[[芝加哥|芝加哥]]{{link-en|朗姆酒飛車|Rum-running|私酒}}戰爭期間[[強尼·托里奧|強尼·托里奧]]和[[艾爾·卡彭|艾爾·卡彭]]的主要對手。他時期的報紙令他被稱為'''迪翁·歐班寧'''（'''Dion O'Banion'''），儘管他從來沒有用這個名字。他領導{{link-en|北邊幫|North Side Gang}}直到1924年，當時據說他被[[法蘭基·耶魯|法蘭基·耶魯]]、{{link-en|約翰·斯卡萊斯|John Scalise}}和{{link-en|艾伯特·安塞爾米|Albert Anselmi}}謀殺。

== 流行文化 ==
在「頭號公敵」時代的早期，前十年的狄恩·歐班寧和其他愛爾蘭黑幫，都是1930年代許多黑幫電影的基礎。例如，[[占士·格尼|占士·格尼]]在1931年電影《[[國民公敵_(1931年電影)|國民公敵]]》中的角色是基於歐班寧和[[海米·魏斯|海米·魏斯]]的。<ref>{{Cite web |url = http://www.filmsite.org/publ.html |title = The Public Enemy (1931) |publisher = American Movie Classics |language = en}}</ref>[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》的第三季也加入了歐班寧的虛構版本。
