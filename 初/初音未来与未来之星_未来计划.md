《'''初音未来与未来之星 未来计划'''》（{{lang-ja|初音ミク and Future Stars Project mirai }}）是由[[世嘉|世嘉]]开发及发行的[[音乐游戏|音乐游戏]]，游戏也是[[初音未来系列|初音未来系列]]的首款[[任天堂3DS|任天堂3DS]]游戏。游戏于2012年3月8日在日本发售。如同系列前作，游戏使用了歌曲合成器[[VOCALOID|VOCALOID]]。

2013年11月28日，發售本作的续作《[[初音未来_未来计划2|初音未来 未来计划2]]》（{{lang-ja|初音ミク Project mirai 2}}）。

2015年5月28日，發售續作《[[初音未來_未來計畫DX|初音未來 未來計畫DX]]》（{{lang-ja|初音ミク Project mirai でらっくす}}）。

== 开发与发行 ==

游戏于2010年中期开始开发，直到2011年9月官方才正式宣布。在2011年11月，《[[Fami通|Fami通]]》正式公布了游戏的正式名字《初音未来与未来之星 未来计划》，此举标志着系列首个改变名称的游戏发布。2012年3月8日，普通版和包含Nendoroid Puchi图的限定版发布<ref>{{cite web |url=http://andriasang.com/comz1j/ |title=3DS Hatsune Miku Named and Dated |author=Anoop Gantayat |date=November 22, 2011 |work= |publisher=Andriasang |accessdate=April 13, 2012 |archiveurl=https://archive.is/20120701141642/http://andriasang.com/comz1j/ |archivedate=2012年7月1日 |deadurl=yes }}</ref><ref>{{cite web |url=http://andriasang.com/comznt/miku_complete/ |title=3DS Hatsune Miku Is Complete |author=Anoop Gantayat |date=January 16, 2012 |work= |publisher=Andriasang |accessdate=April 13, 2012 |archiveurl=https://archive.is/20120701175444/http://andriasang.com/comznt/miku_complete/ |archivedate=2012年7月1日 |deadurl=yes }}</ref>。

根据Media Create的统计，截至2012年4月19日，游戏一共销售了132,689套<ref>{{Cite web|date=2012-04-19|url=http://www.4gamer.net/games/117/G011794/20120418046/|title=30万本目前の「第2次スーパーロボット大戦Z 再世篇」が先週に続いて1位となった「ゲームソフト週間販売ランキング＋」|publisher=4Gamer.net|accessdate=2012-04-19|language=ja}}</ref>。
