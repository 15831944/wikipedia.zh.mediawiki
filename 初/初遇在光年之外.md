'''初遇在光年之外（光年）'''，2018年中国大陆青春偶像剧。该剧由[[剧酷传播|剧酷传播]]出品，[[汪远|汪远]]、[[邹杰|邹杰]]担任编剧，[[徐娇|徐娇]]、[[陈若轩|陈若轩]]等主演<ref>{{Cite web |url=http://ent.ifeng.com/a/20170213/42831335_0.shtml?_zbs_baidu_bk |title=《光年》曝预告 徐娇朱然本色演绎“发光少年”|accessdate=2017-12-02 |author= |date=2017-02-13 |work=邓莹 PK064 |publisher=凤凰娱乐|language=Zh-hans }}</ref>。2016年底<ref>{{Cite web |url=http://character.workercn.cn/361/201612/27/161227073458735.shtml |title=陈若轩：外表小鲜肉，内心老干部|accessdate=2017-12-02 |author=章琰 |date=2016-12-27 |work=朱晶晶 |publisher=羊城晚报|language=Zh-hans }}</ref>于新加坡科学与设计大学拍摄<ref>{{Cite web |url=http://ent.sina.com.cn/2017-02-20/doc-ifyarrcc8124699.shtml |title=《光年》曝新加坡特辑 陈若轩徐娇玩转校园|accessdate=2017-12-02 |author= |date=2017-02-20 |work=得得 |publisher=新浪娱乐|language=Zh-hans }}</ref>，2017年1月24日转场珠海<ref>{{Cite web |url=http://yule.sohu.com/20170124/n479494154.shtml |title=《光年》国内开机 陈若轩徐娇颠覆套路有神器|accessdate=2017-12-02 |author= |date=2017-01-24 |work=王昱妍 |publisher=搜狐娱乐|language=Zh-hans }}</ref>，同年5月2日杀青。於2018年1月31日在[[优酷|优酷]]首播
