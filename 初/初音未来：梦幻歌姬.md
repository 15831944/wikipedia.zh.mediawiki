《'''初音未来：梦幻歌姬'''》是由[[掌趣科技|掌趣科技]]和[[腾讯游戏|腾讯游戏]]联合开发的正版授权[[音乐游戏|音乐]][[养成游戏|养成游戏]]，于2018年9月26日公测首发<ref>{{cite web |url=http://miku.qq.com/webplat/info/news_version3/31712/31713/31714/31715/31716/m18699/201809/762635.shtml |title=《初音未来：梦幻歌姬》9.26不删档，全球巡演正式开启! |last= |first= |date=2018-09-26 |website= |publisher=初音未来运营团 |access-date= |quote=}}</ref>。与传统音乐游戏不同的是，《梦幻歌姬》引入了对战模式，游戏中包含了[[Niconico动画|Niconico动画]]的百万点击量殿堂曲目<ref>{{cite web |url=http://sports.eastday.com/a/180926211812551000000.html |title=《初音未来：梦幻歌姬》今日开启 网友：腾讯出品必属氪品！ |last= |first= |date=2018-09-26 |website= |publisher= |access-date= |quote=}}</ref>。

角色可以更换演出服装<ref>{{cite web |url=http://shouji.tgbus.com/201707/66035145692.html |title=《初音未来：梦幻歌姬》唱响腾讯移动四周年 |last= |first= |date=2017-07-26 |website= |publisher= |access-date= |quote=}}</ref>，赠送礼物以增加好感度。
