'''初鹿車站'''位在[[台灣|台灣]][[台東縣|台東縣]][[卑南鄉|卑南鄉]]，曾為[[台灣鐵路管理局|台灣鐵路管理局]][[台東線|台東線]]（現已廢止）的[[鐵路車站|鐵路車站]]。

== 概況 ==
初鹿站設於1919年，在1982年[[台東線#台東線拓寬計畫|東拓工程]]完成時廢止。

== 車站構造 ==


== 利用狀況 ==
* 東拓工程進行改線，完工後，現已廢止。

== 車站周邊 ==
*初鹿國小
*初鹿派出所

== 歷史 ==
*1919年2月1日：台東製糖株式會社鐵道卑南─北絲鬮間開業，設立「北絲鬮停車場」。
*1922年4月20日：台灣總督府鐵道部收購台東開拓株式會社（原台東製糖株式會社）鐵道，改稱「初鹿尾乘降場」，後升等為「初鹿尾驛」。
*1937年11月1日：改為「初鹿」。
*1982年6月27日：廢止。
