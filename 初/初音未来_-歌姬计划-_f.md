《'''初音未来 -歌姬计划- f'''》（{{lang-ja|初音ミク -Project DIVA- f}}，{{lang-en|''Hatsune Miku: Project Diva f''}}）是一款由[[世嘉|世嘉]]与[[Crypton_Future_Media|Crypton Future Media]]为[[PlayStation_Vita|PlayStation Vita]]和[[PlayStation_3|PlayStation 3]]平台开发的[[音乐游戏|音乐游戏]]。<ref>{{cite web |url=http://andriasang.com/con13r/miku_f/ |title=PlayStation Vita Hatsune Miku Project Diva Gets a Final Name |author=Anoop Gantayat |date=April 23, 2012 |work= |publisher=Andriasang |accessdate=May 22, 2012 |archiveurl=https://archive.is/20120707070832/http://andriasang.com/con13r/miku_f/ |archivedate=2012年7月7日 |deadurl=yes }}</ref>这是[[初音未来_-歌姬计划-|初音未来 -歌姬计划-]]系列的第五款游戏，也是歌姬计划系列中首款[[跨平台|跨平台]]的游戏。《初音未来 -歌姬计划- f》于2012年8月30日发布于PlayStation Vita平台，2013年3月7日发布于PlayStation 3平台。<ref name="Release">{{cite web |url=http://gematsu.com/2012/04/next-hatsune-miku-project-diva-japanese-date-set |title=Next Hatsune Miku Project Diva Japanese date set |author=Sal Romano |date=April 23, 2012 |work= |publisher=Gematsu |accessdate=April 23, 2012}}</ref><ref name="E32012">{{cite web |url=http://www.siliconera.com/2012/06/06/why-is-there-a-translated-demo-of-hatsune-miku-project-diva-f-at-e3/ |title=Why Is There A Translated Demo Of Hatsune Miku Project Diva f At E3? |author=Kris |date=June 6, 2012 |work= |publisher=Siliconera |accessdate=June 7, 2012}}</ref><ref>{{cite web |url=http://www.joystiq.com/2012/06/05/hatsune-miku-bringing-synthesized-vocal-music-gaming-to-north/ |title='Hatsune Miku' bringing synthesized vocal music gaming to North American Vita |author=JC Fletcher |date=June 5, 2012 |work= |publisher=[[Joystiq|Joystiq]] |accessdate=June 3, 2012}}</ref>

==影响==
凭借《-歌姬计划- f》的影响，PSV在游戏的PSV版本发行当周的周销量榜中首次以50,070的销量超过3DS和3DSLL。<ref>[http://www.jpbeta.net/2012/09/psv-is-no1-first/ 救机神作初音终到来，PSV历史初击败3DS/LL！]</ref>
