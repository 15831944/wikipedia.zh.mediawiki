《'''初戀魔法電擊'''》（{{lang-ja|初恋マジカルブリッツ}}）， [[明日香正太|明日香正太]]著作，[[天廣直人|天廣直人]]插畫的日本[[輕小說|輕小說]]作品，目前共有十五卷，預計共有十六卷<ref >明日香正太 (2011)，《初恋マジカルブリッツあなた、ときめき、恋、もっと！》後記</ref>。日文版第一卷由[[集英社|集英社]]於2005年6月24日<ref >{{ja}}スーパーダッシュ文庫(2005)，[http://books.shueisha.co.jp/CGI/search/syousai_put.cgi?isbn_cd=4-08-630244-6&mode=1 スーパーダッシュ文庫 初恋マジカルブリッツシリーズ]</ref>出版發行。台灣繁體中文版由[[青文|青文]]於2007年10月3日<ref>青文(2007)，[http://www.ching-win.com.tw/?sinfo=detail&b_unit=2464 青文出版] {{Wayback|url=http://www.ching-win.com.tw/?sinfo=detail&b_unit=2464 |date=20140203050017 }}</ref>代理發行。[[漫畫|漫畫]]日文版由集英社發行，中文版[[玉皇朝|玉皇朝]]代理。

== 作品概要 ==
作品是一部校園戀愛喜劇，故事包含[[舊約|舊約]][[聖經|聖經]]的神話及戰鬥幻想內容，內裏包括各種性及殘酷的表達。

== 故事簡介 ==
初戀學園的普通高校一年生藤井鼓太郎鼓起勇氣向溫柔內向、身材高挑的女孩白鳥祈梨告白。突然出現的美少女鈴蘭及一眾魔物的影響，使得兩人的戀情無法順利前進。鼓太郎為了從性命攸關的狀況中逃脫，唯一的方法竟然是跟祈梨結合！

== 登場人物 ==
=== 主要角色 ===
; {{nihongo|藤井 鼓太郎|藤井 鼓太郎（ふじい こたろう）}}
: 本作主角。15歲，戴眼鏡，身高不足一米六，長相像女孩的普通高校一年級學生，容易臉紅。喜歡祈梨，但同班一個月來從來沒跟祈梨說過話。

; {{nihongo|白鳥祈梨|白鳥 祈梨（しらとり いのり）}}
: 本作女主角之一。個性溫柔，有着一頭濃密的長髮、長長的睫毛與又大又明亮的眼睛，身高約一七五公分，雖然身材高大但卻十分膽小。

; {{nihongo|鈴蘭|鈴蘭（すずらん）}}
: 本作女主角之一。突然出現在鼓太郎面前的魔法師，一度想要取鼓太郎的性命，後來成為了鼓太郎的使魔。

=== 魔法相關 ===
; {{nihongo|莉莉斯|リリス}}
: 從神話中被消除的亞當第一任妻子。

; {{nihongo|克羅諾斯|クロリリス}}
: 鈴蘭的創造者。

; {{nihongo|弗朗西絲卡|フランシスカ}}
: 梵蒂岡的使者。

=== 其他角色 ===
; {{nihongo|藤井健次郎|藤井 健次郎（ふじい けんじろう）}}
: 鼓太郎的父親。

; {{nihongo|藤井琴子|藤井 琴子（ふじい ことこ）}}
: 較鼓太郎小數個月的妹妹，與鼓太郎同年級，活潑好動。與鼓太郎身高差不多，雖然是妹妹但總是掌握主動權，把鼓太郎玩得團團轉。曾以轉交東西為由，把鼓太郎騙到正被女生作為臨時更衣室的美術室。
