'''初台站'''（{{jpn|j=初台駅|hg=はつだいえき|rm=Hatsudai eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[澀谷區|澀谷區]][[初台|初台]]一丁目，屬於[[京王電鐵|京王電鐵]][[京王線|京王線]]（[[京王新線|京王新線]]）的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KO02'''。

== 年表 ==
* 1914年6月11日，作為京王電氣軌道的'''改正橋站'''啟用。
* 1919年9月，改稱'''初台站'''。
* 1944年5月31日，京王與[[東京急行電鐵|東京急行電鐵]]合併，初台站作為京王線的車站運行。
* 1948年6月1日，東急與京王帝都電鐵分拆，初台站成為京王的車站。
* 1964年6月7日，改作地下站。
* 1978年10月31日，，京王新線開通後，初台站搬至現址。
* 2013年2月22日，以'''KO02'''作主為初台站的編號。

== 車站構造 ==
本站位於[[首都高速4號新宿線|首都高速4號新宿線]]與[[國道20號|國道20號]]（[[甲州街道|甲州街道]]）地下，[[月台|月台]]為上行線與下行線分開的二層構造，地下2樓為上行線，地下3樓為下行線。樓梯分為新宿側與幡谷側2處，並設有[[電扶梯|電扶梯]]。新宿側另有[[電梯|電梯]]。幡谷側樓梯可至地下一樓的中央口，付費區內設有廁所。中央口外的地下通道往右為甲州街道北側的北口，往左為南側的南口。北口直接連結[[日本新國立劇場|日本新國立劇場]]，並有地下式腳踏車停車場。新宿側閘口的東口直接連結[[東京歌劇城|東京歌劇城]]。

本站在1978年10月31日京王新線開通時由京王線移轉至此。[[新宿站|新宿站]] - [[笹塚站|笹塚站]]之間的隧道內仍保留舊月台作為軌道維護資材存場、資材搬入口、隧道緊急避難通道與車站維護用通道<ref>[[NHK綜合頻道|NHK綜合頻道]]『{{link-ja|BURATAMORI|ブラタモリ}}』「新宿誕生編」（2012年3月1日撥出）</ref>。

車站月台側的通風口是因為車站開設當時因風壓過大而設置。

== 車名由來 ==
車站原名「'''改正橋'''」，源自附近[[玉川上水|玉川上水]]橋名。

現行站名「'''初台'''」則是以當時車站南側地名「''澀谷區[[初台|代代木初台町]]''」命名。
