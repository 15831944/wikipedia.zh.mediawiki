《'''初吻'''》（{{lang-ja|はじめてのチュウ}}）是[[日本|日本]]歌手{{link-ja|安心爸爸|実川俊晴}}的單曲。1990年5月1日由[[日本古倫美亞|日本古倫美亞]]發行。

該歌曲後來在1993年5月1日（規格編號：CODC-182）、1995年11月1日（規格編號：CODA-826）以[[Mini_CD|8cmCD]]形式再度發盤。

== 概要 ==
《初吻》是[[富士電視台|富士電視台]][[電視動畫|電視動畫]]《[[奇天烈大百科|奇天烈大百科]]》的第4代片頭主題曲<ref group="注">使用時間從1990年4月15日（第87話）至1990年10月21日（第108話）為止。</ref>。也是同名動畫後來的第6代片尾主題曲，原本在一時期間換其它歌曲使用，但又換回該歌曲使用到動畫完結為止<ref group="注">使用時間從〔1990年11月4日（第109話）至1992年3月29日（第170話）、1993年4月25日（第213話）至1995年3月19日（第291話）、1995年10月29日（第311話）至1996年6月9日（第331話／最終話）。</ref>，還有該歌曲也是《奇天烈大百科》所有片尾主題曲中使用最久的。

至於B面歌曲「我和瑪莉只是普通朋友」也被《奇天烈大百科》的電視動畫系列當第5代片尾主題曲使用，主唱由作品主角木手英一的聲優[[藤田淑子|藤田淑子]]擔任，使用時間從1990年4月15日（第87話）至1990年10月21日（第108話）為止。

《初吻》是自從序奏的主題之後，以描述一個剛體驗[[親吻|初吻]]經驗的少年為主題的歌曲。而主唱者「安心爸爸（<small>{{lang-ja|あんしんパパ}}</small>）」是虛擬人物，實際是由負責作詞、作曲的{{link-ja|實川俊晴|実川俊晴}}主唱。在錄音時是用原曲一半的速度進行配唱，而在播放時則用正常速度的手法錄製，最後經由特效處理製作出聽了帶給人一種很可愛的歌聲。

此外，製作過程中有公開是使用{{link-ja|變音器|ボイスチェンジャー}}將聲音調整成聽起來很可愛的歌聲，再用{{link-ja|開盤磁帶錄音機|Reel-to-reel audio tape recording}}等特殊錄音器材進行速度的必要調整。由於製作過程比較費工夫，所以實川將其視為「'''企業機密'''」<ref name="anshin-papa web-three">{{Cite web|date=|url=http://www.real-river.com/rec_3.html|title=【あんしんパパの製作現場エピソード3】あんしんパパ公式Web|work=|publisher={{link-ja|實川俊晴|実川俊晴}}的官方個人網站|accessdate=2017-07-29|language=ja}}</ref>。 

《初吻》也是實川他第一次以現成的歌曲為點子，然後用[[鼠|老鼠]]、[[倉鼠|倉鼠]]及[[河狸|河狸]]等鼠科動物的「啾」叫聲配上愛情故事進行歌詞填寫的歌曲<ref>「奇妙{{lang|ja|キテレツ}}“早回{{lang|ja|し}}歌手”」《[[日本經濟新聞|日本經濟新聞]]》早報，1995年12月6日版，第40面。</ref>。

1993年1月，根據發行唱片公司[[日本古倫美亞|日本古倫美亞]]的統計（包含1993年的銷售超過1萬張的[[動畫歌曲|動畫歌曲]]紀錄），《初吻》曾經創下銷售出2萬張CD與卡式錄音帶的最高紀錄<ref>「『{{lang|ja|ねぇ、チューして}}』子{{lang|ja|らにもウケてます}} 流行語部門{{lang|ja|で}}銀賞{{lang|ja|の}}余勢」《[[朝日新聞|朝日新聞]]》早報1993年1月13日版，第16面。</ref>。到了1995年，單曲已經發行飆升超過10萬張。成為實川的知名代表歌曲。

1995年11月1日（日本古倫美亞發行，規格編號：COCC-13006）發行的專輯後來收錄了新的錄製版本。

== My First Kiss ==
2000年最早由日本的搖滾樂團{{link-ja|Hi-STANDARD|Hi-STANDARD}}將「初吻」歌名及歌詞翻譯成英文歌名「'''My First Kiss'''」及英文歌詞詮釋之後（後在單曲《{{link-ja|Love Is A Battlefield|Love Is A Battlefield}}》的當B面歌曲收錄）而廣為人知。在那之後該英文歌曲版本被多位海外歌手翻唱。

2006年，日本搖滾樂團{{link-ja|GAGAGA SP|ガガガSP}}的翻唱版本被汽車公司[[日產|Nissan]]選用當作{{link-ja|Nissan SERENA|日産・セレナ|SERENA}}廠牌的廣告歌曲使用。

2010年，瑞典音樂家{{link-en|瑞斯摩·菲柏|Rasmus Faber}}的翻奏版本被日本啤酒公司[[朝日啤酒|朝日啤酒]]當自家新商品「{{lang|ja|くつろぎ}}仕{{lang|ja|込}}」的廣告歌曲使用。2016年也被汽車製造公司[[鈴木_(公司)|SUZUKI]]選用當作[[鈴木Baleno|鈴木Baleno]]廠牌的廣告歌曲使用。
