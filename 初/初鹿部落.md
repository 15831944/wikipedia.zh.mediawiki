'''初鹿部落'''（{{lang-pyu|Ulivelivek}}）位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[卑南鄉|卑南鄉]]，行政區畫分上，屬於卑南鄉初鹿村，舊稱'''北絲鬮社'''<ref name="文化部"/>，是[[臺灣原住民|臺灣原住民]][[卑南族|卑南族]]的傳統部落之一，也是卑南族八社之中，地理位置上最北邊的部落，更北方便是[[布農族|布農族]]的傳統領域<ref name="原民資訊資源網"/><ref name="臺東縣政府"/>。

== 歷史 ==
初鹿部落形成據部落長老口述歷史表示，古代的卑南族從[[太麻里鄉|太麻里鄉]]華源海岸登陸後，首先來到現今的卑南鄉[[建和村|建和村]]定居，後來再往知本深山遷移，來到一處稱為「撒里亞量」的地方定居，之後再輾轉遷徒到「[[巴蘭遺址|巴蘭]]」、以及現稱五家坡，初鹿部落西南方山坡地的「普拉撲拉」<ref name="文化部"/><ref name="原民資訊資源網"/><ref name="初鹿國民小學"/>。

初鹿部落舊稱北絲鬮社，據部落長老表示，[[臺灣清領時期|臺灣清領時期]]末期，[[卑南廳|卑南廳]]的[[清兵|清兵]]進行[[理番|理番]]的駐紮兵營位於北方的新開園（現稱為[[池上鄉|池上]]），因此官兵要前往新開園時，須從由卑南廳徒步穿越花東縱谷後才可抵達，而當時的官兵身上都會背著以麻絲做成的背包來裝載長途步行所需的糧食與生活用品，在當時，官兵時常招募原住民來代為背運厚重的行囊，當步行隊伍來到今日的初鹿部落北側山坡，將開始攀登龍過脈山麓的時候，原住民則趁機將官兵殺害後，奪取其行囊，爾後此地開始有[[閩南語|閩南語]]「拔絲鬮」、「絲鬮」等稱呼，意旨為背包的意思<ref name="初鹿國民小學"/>。

直到[[十九世紀|十九世紀]]末期，臺灣自[[馬關條約|馬關條約]]後開始進入[[臺灣日治時期|日治時期]]，經過[[日本人|日本人]]的規劃下，以一分地一戶的交換方式，卑南族人再度遷至昔日稱為初鹿通谷的部落現址，在卑南族語中，這座山谷被稱為「烏利敷利敷庫」，又叫做「巴哈利彎」，意旨為休憩的地方<ref name="初鹿國民小學"/>。

臺灣[[臺灣戰後時期|戰後]]初鹿部落於北方布農族傳統領域交界處另外分出一個新的部落，稱為「[[龍過脈部落|龍過脈部落]]」<ref name="原民資訊資源網"/>。

== 部落概況 ==
初鹿部落的地理環境上，正好位在卑南鄉的北端以及[[花東縱谷|花東縱谷]]的南端，屬於[[中央山脈|中央山脈]]支系及[[海岸山脈|海岸山脈]]之間的[[丘陵|丘陵]]坡地。由於[[省道|省道]][[台9線|臺九線]]花東縱谷公路筆直地穿過初鹿部落的中心，鄰近著名的[[初鹿牧場|初鹿牧場]]也須經由該村落的路口進入，因此初鹿部落已形成來往過客的中繼休息站。<ref name="巴蘭遺址"/>

在卑南鄉戶政事務所2018年2月的人口統計中，全村共有32鄰，總戶數有542戶，總人口1,469人，其中原住民人口有510人<ref name="population"/>。卑南族原住民主要居住在初鹿部落的聚落主體中，但也有少數[[漢人|漢人]]定局於此，而在主要聚落北方的的河灘地以及省道沿線的居民多半以漢人為主，原住民則占較少部分。<ref name="巴蘭遺址"/>

== 祭典 ==
初鹿部落在每年7月會舉行收穫祭，12月時會舉行猴祭以及大獵祭、除喪、成年禮、訪少女，統稱年祭。其中初鹿部落所舉辦的收穫祭還會有[[盪鞦韆|盪鞦韆]]的活動，也是除了[[魯凱族|魯凱族]]傳統祭典以外，另一個擁有盪鞦韆活動的原住民部落<ref name="臺東縣政府"/>。

== 巴蘭遺址 ==
{{main|巴蘭遺址}}

巴蘭遺址位在初鹿部落西南方[[海拔|海拔]]約600公尺的山坡上，遺址的年代鑑定上，先住民的居住時期大約是西元600到1400年之間。這座遺址被初鹿部落的卑南族人視為聖地，並且也是該部落重要的水源地。巴蘭遺址已在2006年4月14日由[[臺東縣政府|臺東縣政府]]公告為縣定遺址。<ref name="臺東縣政府"/><ref name="巴蘭遺址"/>
