{{engname|《車手》|''Motorway''}}是一部2012年上映的香港電影，導演是[[鄭保瑞|鄭保瑞]]，主要演員包括[[黃秋生|黃秋生]]、[[余文樂|余文樂]]、[[郭曉冬|郭曉冬]]。動作指導[[錢嘉樂|錢嘉樂]]。獲得2012年[[香港電影評論學會大獎|香港電影評論學會大獎]]最佳電影、最佳導演。入圍[[第32屆香港電影金像獎|第32屆香港電影金像獎]]6項提名。

== 剧情 ==
[[隱形戰車|隱形車隊]]是[[香港警務處|香港警務處]][[交通總部|交通部]]的精英隊伍，熱血滿腦的[[隱形戰車|隱形戰車]]司機陳翔（[[余文樂|余文樂]]）與拍擋盧峰（[[黃秋生|黃秋生]]）因為互不對盤，經常使到任務失敗收場。在一次執勤中，陳翔竟輕易地緝拿了黑道傳奇車手蔣薪（[[郭曉冬|郭曉冬]]），讓盧峰覺得有內情。原來他打算入警署羈留室救友，最後更讓他逃之夭夭。而且陳翔更發現盧峰竟與蔣薪有一段不為人知的過去。

== 製作 ==
[[錢嘉樂|錢嘉樂]]是此片的動作導演，盡量少用[[電腦合成影像|電腦合成影像]]，真正飛車硬碰硬。在港版海報中一句「8000轉2咪車」是此戲的賣點，此項絕技是吳海堂親自嘗試給[[鄭保瑞|鄭保瑞]]看，令他看傻了眼，並決定加入電影橋段中，成戲中重要轉捩點。<ref name="cannes">[http://ol.mingpao.com/cfm/star5.cfm?File=20120522/saa01/mcb1.txt 《車手》康城成功「試水」] ''明報''. 2012年5月22日.</ref>
[[錢嘉樂|錢嘉樂]] 、[[吳海堂|吳海堂]]及 [[黃偉輝|黃偉輝]]憑此片獲得第49屆[[金馬獎|金馬獎]]最佳動作設計。

== 發行 ==
《車手》在康城影展進行試映獲好評，並獲邀參展威尼斯電影節。法國、日本、星馬、澳紐與泰國等地已買下電影版權。《車手》受日本發行商力捧，評價《車手》比香港早年拍攝的《[[頭文字D|頭文字D]]》更勝一籌，是一部真正展示車手狀態與汽車動感文化的賽車電影。<ref name="cannes" />

== 影評 ==
[[爽報_(香港)|香港爽報]]的善人給予3個讚，他說：「部份劇情充滿犯駁，不過整體拍出型格，只要觀眾不入錯「筍位」觀賞，絕對是合格有餘的。」<ref>[http://www.sharpdaily.hk/article/entertain/20120622/104422 爽影評：《車手》衝動𡃁警VS老差骨] ''爽報 (香港)''. 2012年6月22日.</ref>

[[蘋果日報_(香港)|香港蘋果日報]]的黃國兆說：「影片以這種特殊駕駛技術大做文章，頗能配合警匪追逐的緊張情況，亦恰當地描寫劇中人物的性格和背景。<ref>[http://hk.apple.nextmedia.com/entertainment/art/20120625/16455531 蘋影話：《車手》 警匪飆車電影緊張刺激] ''蘋果日報''. 2012年6月25日.</ref>影片在劇情和細節上有頗多紕漏，最明顯的敗筆是結尾多層停車場內的汽車追逐，警方竟不作封鎖停車場出口的部署，奇怪！」<ref>[http://hk.apple.nextmedia.com/entertainment/art/20120626/16459010 蘋影話：《車手》汽車窄巷過彎有新意] ''蘋果日報''. 2012年6月26日.</ref>

[[頭條日報|頭條日報]]的祈佳仕說：「《車》片在描繪角色心態也見不俗。導演沒有說教，只以劇情啟發觀眾的思維。惟電影沒交代主角追求速度的原因，以及沒有給飆車者一些沉重的教訓，有點遺憾。」<ref>[http://blogcity.me/blog/reply_blog_express.asp?f=X5U3QSP2LC209597&id=423687 速度感強烈的警匪片] ''頭條日報''. 2012年6月18日.</ref>

[[明報|明報]]的石琪說：「嚴格來說，編劇Joey O'Bryan、司徒錦源和馮日進的佈局仍屬警匪公式，好在以交通警為主，更重要是導演鄭保瑞拍得認真，不落俗套。現在《車手》進一步專業和成熟了，不愧為當今最值得注意的香港導演之一。」<ref>[http://ol.mingpao.com/cfm/star20.cfm?File=20120624/sae03/myb1.txt 石琪說戲﹕詠春式車技本格派推理] ''明報''. 2012年6月24日.</ref>

中国大陆首周4天收获1680万人民币，位列第六名。
