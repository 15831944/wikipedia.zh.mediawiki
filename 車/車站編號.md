'''車站編號'''（又稱車站代碼）是對[[鐵路車站|鐵路車站]]除常見的站名以外，增設的英文字母（[[拉丁字母|拉丁字母]]）和[[阿拉伯數字|阿拉伯數字]]等代號的制度。因拉丁字母和阿拉伯數字已於全世界普及，因此加入車站編號以便旅行者識別。除鐵路車站外亦有[[巴士站|巴士站]]導入的例子。通常出現於母語並非為[[拉丁文字|拉丁文字]]的地區。
{{TOC limit|3}}

== 車站編號的組成 ==
車站編號通常由記號部分與數字部分（編號部分）組成。

; 只使用數字的例子
:* [[金澤海岸線|金澤海岸線]]
:* [[大糸線|大糸線]]
:* [[豐橋鐵道|豐橋鐵道]][[渥美線|渥美線]]、[[東田本線|市內線]]

; 只使用字母的例子
:* [[港鐵|香港鐵路]]（只用作員工內部溝通之用）
:* [[美鐵|美鐵]]（Amtrak）

==== 桃園捷運 ====
2017年3月2日通車時採用。並不與路線同時開業的[[二重站|二重站]]與[[輔大醫院站|輔大醫院站]]不設編號，啟用時將以分支編號（分別是A2a與A5a），[[機場旅館站|機場旅館站]]啟用時起就採用分支編號的A14a，未啟用的[[機場第三航廈站|機場第三航廈站]]則預留了A14的編號。原因是計劃時使用的編號沿用至旅客資訊的車站編號，而二重、輔大醫院、機場旅館等站並不在當初的計劃之中，而是後來追加而採用分支編號。隨著桃園機場捷運通車，一部份車站週邊有建設公司進行土地開發、建築物建設，因此車站編號常出現在建案銷售時的廣告文宣。
