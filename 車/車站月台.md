'''月台'''，或稱'''-{zh-hans:月台; zh-hant:站台;}-'''，是指[[路軌|路軌]]傍邊高出的台面。一般建筑在[[鐵路車站|鐵路車站]]以方便乘客上下車。

== 意思变迁 ==
-{zh-hans:站台原先称作“月台”; zh-hant:月台}-，原意是「赏月的高台」。中国传统建筑中，大殿、正房前方连接台阶的向前方突出的平台称为“-{月台}-”。站在-{月台}-上，一般前无遮拦，适于赏月。古人登-{月台}-触景生情，心情舒畅，思绪万千，会引发出各种人生感悟，并留下众多诗词佳作：“明从海上来，皎皎入我牖。何如登高台，对月把尊酒”（[[宋敏_(宋代)|宋敏]]）“夜色何苍苍，月明久未上。不上倚城台，无奈东南嶂。”（[[苏辙|苏辙]]）等。

现代-{月台}-成为送迎之地，别离时的伤感，重逢时的喜悦，都汇集于此。

== 稱呼法 ==
月台通常是用[[数_(数学)|號碼]]去命名，如一號月台、二號月台，通常是照順序編號，也有以軌道編號作月台編號者；有些地方使用[[英文字母|英文字母]]或開行駛方向、地點命名。

而島式月台的兩側，有用同一月台的A、B側命名，也有當作不同月台各別命名的。

也有同一月台的同一面，拆為兩個月台使用者。

在[[中國大陸|中國大陸]]，铁路、地铁、[[轻轨运输|轻轨]]等车站的标识一般使用“-{站台}-”一词，“-{月台}-”一词已经很罕见，仅偶尔出现在[[文学|文学]]作品中。[[臺灣|臺灣]]、[[香港|香港]]及[[新加坡|新加坡]]則仍一律採用「-{月台}-」用詞。

== 長度 ==
月台能停放列車的長度相較於{{link-ja|月台有效長|有効長}}，通常要比線上營運的客運列車長，否則會發生月台無法完全容納列車，旅客需要跳下軌道面才能下車的情況，此時可以用[[選擇性車門操作|部分車門開啟]]解決，不過部分鐵道業者（如[[台鐵|台鐵]]）並未使用此種對策。

<!--中国铁路站台標準有效長，高站台（高度1250mm）400m、低站台（高度300mm）或一般站台（高度500mm）560m。-->

全世界最長的月台為[[印度|印度]][[北方邦|北方邦]]之 Gorakhpur，全長1,366.33 公尺(4,483[[英尺|英尺]])。

==高度==
[[中国铁路|中国铁路]]系统中对站台高度分为低站台、一般站台和高站台，分别为高出相邻线路轨面300mm、500mm和1250mm。<ref>{{cite web|title=课程学习>>第五章 第二节 客运站的站场设施|url=http://zy.swust.net.cn/15/1/lkyszz/kcxx/chapter5/0502.html}}</ref>
九龙深圳广州昆明系统站台高度均为高站台（1250mm）、他系统站台高度分为低站台（300mm）和一般站台（500mm）。

朝鲜民主主义人民共和国直流电气化铁路站台高度均为高站台（1250mm）。

德国铁路站台高度分为低站台（300mm）、一般站台（500mm）和中站台（760mm）。

荷兰铁路站台高度分为中站台（760mm）和高站台（1250mm）。

西班牙铁路站台高度分为低站台（300mm）、一般站台（500mm）、中站台（760mm）和高站台（1250mm）。

前苏联和芬兰铁路站台高度分为超低站台（200mm）、一般站台（500mm）和高站台（1100mm），在非电气化或交流电气化铁路路段站台高度分为超低站台和一般站台。
