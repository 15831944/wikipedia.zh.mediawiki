'''車淑梅'''（{{lang|en|'''Candy Chea Shuk Mui'''}}，{{bd||1957年10月14日|catIdx=C車}}），資深傳媒從業員，曾為[[香港電台|香港電台第二台]]晨早節目《[[晨光第一線|晨光第一線]]》的第一代主持人之一，並長時間主持。2002年7月18日主持最後一集《晨光第一線》後 <ref>{{cite web|url=http://rthk9.rthk.hk/classicschannel/2000s_radio_0006_86.htm |title=香港電台經典重溫頻道——晨光第一線 車淑梅臨別秋波，重溫昔日片段 |accessdate=2018-09-16 }} </ref>，調往[[香港電台|香港電台第一台]]主持《[[開心日報|開心日報]]》。2011年年尾開始更只主持[[第一台|第一台]]的《[[舊日的足跡_(電台節目)|舊日的足跡]]》，過著半退休生活。

== 履歷 ==
1967年透過參加《[[空中小園地|空中小園地]]》加入電台兒童節目播音。中學畢業後，車淑梅擁有真正屬於自己的節目《車天車地》。

八九十年代開始連續多年蟬聯[[香港大學民意研究計劃|香港大學民意研究計劃]]《[[全港電台收聽調查|全港電台收聽調查]]》十大最受歡迎電台節目主持人第一位，因此被外界封為香港金牌DJ<ref>{{cite web |url=http://trans.wenweipo.com/gb/paper.wenweipo.com/2008/11/02/EN0811020008.htm |title=存档副本 |accessdate=2010-03-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20090427072304/http://trans.wenweipo.com/gb/paper.wenweipo.com/2008/11/02/EN0811020008.htm |archivedate=2009-04-27 }}</ref>。1992年被一名電台聽眾提名，成功獲選[[香港十大傑出青年|香港十大傑出青年]]。2009年接受美國[[CNN|CNN]]訪問 <ref>[http://www.youtube.com/watch?v=ox_iU-fsOYQ]</ref>。車淑梅更於2010年《[[讀者文摘|讀者文摘]]》舉行的香港「最受信任的人物」網上調查中上榜。

車淑梅多年來致力於義工事務，2001年被委任為[[香港腎臟基金會|香港腎臟基金會]]董事局成員，亦為多個香港服務團體的顧問，曾參與過百個慈善機構舉辦的活動，包括司儀工作，活動嘉賓，出任代言人等，於2005年榮獲第一屆[[香港傑出義工獎|香港傑出義工獎]]。<ref>http://www.avs.org.hk/hkva/winners/</ref>

車淑梅專長於人物專訪節目，於全港最受歡迎電台節目之一的《[[舊日的足跡(電台節目)|舊日的足跡]]》中曾訪問多為社會傑出人士，訪問內容常受傳媒廣泛報導。2006年在[[香港有線電視|香港有線電視]]擔任《[[玄機解碼|玄機解碼]]》節目主持人，訪問了十多位圈中好友。至2008年，再為有線主持主題相近的節目《[[人生命書|人生命書]]》。2012年在[[Roadshow|Roadshow路訊通]]，分別曾和尹師傅及于師傅主持烹飪節目《煮場作客》，因而人氣再度飈升。

車淑梅於2012年獲頒發行政長官社區服務獎狀。2018年，她獲嶺南大學頒授榮譽院士銜以表揚對廣播界、社會服務方面貢獻良多。

==個人背景==
車淑梅小時候在[[馬頭圍邨|馬頭圍邨]]居住，在[[馬頭涌官立小學|馬頭涌官立小學]]就讀。中學畢業於[[伯特利中學|伯特利中學]]（英文部）及預科畢業於已結束辦學的[[萃華英文書院|萃華英文書院]]<ref>{{cite news|title=李龍基返母校開唱|url=http://orientaldaily.on.cc/cnt/entertainment/20120629/00282_074.html|accessdate=2015-06-27|newspaper=《[[東方日報|東方日報]]》|date=2012-06-29}}</ref>。雖在香港出生，但因從小與[[客家人|客家人]]乾媽同住，加上父親為[[蒙古人|蒙古人]]的關係，故此不甚會說[[廣東話|廣東話]]，當[[傳媒工作者|傳媒工作者]]，遇上[[文化差異|文化差異]]，經努力，成為[[語言|語言]]流利的港台首席主持人。<ref>[http://www.youtube.com/watch?v=X1mSzVMjGKE 港台我有份：故事姐姐成為「十大傑出青年」 (香港電台 / RTHK)]</ref>

[[丈夫|丈夫]]是香港電台電視部助理廣播處長[[張文新|張文新]]，姊姊[[車森梅|車森梅]]亦是電台[[主持人|主持人]]及[[播音員|播音員]]。車淑梅善於培育子女，家有長子張家瑋<ref>[https://hk.celebrity.yahoo.com/news/%E8%BB%8A%E6%B7%91%E6%A2%85%E8%AE%9A%E6%96%B0%E6%8A%B1%E5%8F%88%E5%8F%BB%E5%8F%88%E4%B9%96%E5%8F%88%E5%AD%9D%E9%A0%86-140400038.html 車淑梅讚新抱又叻又乖又孝順]</ref>及幼女張家晞<ref>[http://hk.next.nextmedia.com/article/858/6224842 這個阿媽真爆炸　車淑梅]</ref>，皆成績優異。長子會考3A5B，首先入讀[[南加州大學|南加州大學]]，再轉入[[美國|美國]][[費城|費城]][[賓夕法尼亞大學|賓夕法尼亞大學]]，修讀[[經濟學|經濟學]]及[[心理學|心理學]]，2007年畢業回港。二女會考5A4B，SAT數學科滿分，2012年在[[美國|美國]][[麻省理工學院|麻省理工學院]]畢業，修讀[[環境工程|環境工程]]及[[管理學|管理學]]。<ref>[http://baby.esdlife.com/features_parent/1122/dj-mui/: 淑梅姐姐教您培育孩子]</ref>
