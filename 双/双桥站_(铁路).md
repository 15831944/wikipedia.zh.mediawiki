'''双桥站'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[三间房地区|三间房地区]]双桥路东柳西里甲1号，是[[京哈铁路|京哈铁路]]（[[京秦铁路|京秦铁路]]）和[[京承铁路|京承铁路]]的车站，北京铁路枢纽内[[双沙铁路|双沙铁路]]、[[丰双铁路|丰双铁路]]起点站，距[[北京站|北京站]]12公里，是[[北京铁路枢纽|北京铁路枢纽]]东部辅助[[编组站|编组站]]。等级为[[一等站_(中国)|一等站]]。该站不办理客运。

== 简介 ==
双桥站建于[[清朝|清朝]][[光绪|光绪]]二十七年（1901年）<ref name=cyqz/><ref name="difangzhi"/>。是通州支线一个中间站，因站北侧北双桥村而得名。1900年[[庚子事变|庚子事变]]，[[八国联军|八国联军]]占领北京。1901年[[英军|英军]]占领北京期间，为适应战争需要及加强对北京的控制，将[[京奉铁路|关内外铁路]]北京端的终点从[[马家堡站|马家堡站]]展长至[[北京南站|永定门]]再经[[东便门_(北京)|东便门]]至[[东交民巷|东交民巷]]外国使馆区附近的[[正阳门东站|正阳门]]，同时修建了自东便门通向[[漕运|漕粮]]仓储重镇[[通州区_(北京市)|通州]]运河西岸长23公里的通州支线，该线从正阳门起向东经东便门至双桥村进入通州，是关内外铁路的支线。1902年8月24日，通州支线由中国收回国有<ref name=cyqz/>。

双桥站建站初期，仅办理通过，无客货运业务。[[抗日战争|抗日战争]]时期，1938年[[日本|日本]]修建[[京承铁路|京古线]]时，重建双桥站站舍，站线增至两股半道，扩建为中间站。[[中华人民共和国|中华人民共和国]]成立后，在北京枢纽总布置图中，将双桥站规划为北京枢纽东部地区辅助编组站<ref name=cyqz/><ref name="difangzhi"/>。1956年，北京东南环线[[百子湾站_(铁路)|百子湾站]]与双桥站接通，双桥站站线扩建为三股半道。1958年部分土石方完成后停工。1965年增建一股线。1973年增铺六、七道两股卸车线<ref name=cyqz/>。1975年，为了配合[[京坨铁路|通坨线]]开通，双桥站扩建，作为过渡措施<ref name=cyqz/><ref name="difangzhi"/>。1976年，双桥编组站施工设计完成。1979年11月24日，双桥站全部迁到新站。1983年1月21日，双桥站直属北京铁路分局，1983年3月1日定为三等站<ref name=cyqz/>。1975年到1984年，双桥编组站工程分批完工投入使用<ref name=cyqz/><ref name="difangzhi"/>。1979年北京铁路枢纽又扩建，双桥站自1984年后继续扩建，至1988年建成126条股道的车站<ref name="difangzhi">{{cite book|title=北京志·市政卷·铁路运输志|year=2004|publisher=北京出版社|location=北京|isbn=7-200-05098-9}}</ref>。1985年12月15日，电气化在双桥站通电试车成功。1987年底，双桥站扩建为二级三场辅助编组站，累计完成投资1.7亿元人民币。1988年4月1日，双桥站被定为二等站<ref name=cyqz/>。

双桥站是双丰线、星双线、京承线、京秦线的汇集点。截至1990年代中期，车站上行场有8股线，下行场有7股线，调车场有17股线，设自动化驼峰1座，有3台调车机担当编解取送调车作业。1990年代中期，全年日均出入车3536.2辆，日均编组29.4列，日均解体27.7列。日均装车13辆，日均卸车41辆。日接发货车91.5对、旅客列车39对。1995年出入车135.6万辆，载客15.1万人次，发送货物38.5万吨，年收入1912万元<ref name=cyqz>{{Cite web |url =  http://www.bjchy.gov.cn/chaoyang/qznj/8a24f09a24d90f1f0124d94c73fe0019.html |title =朝阳区志第七编交通邮电  |author =  |date = 2009-11-09 |publisher =北京朝阳  |language =  |accessdate =  |quote =  }}</ref>。

== 下辖车站 ==
双桥站位于丰双线、星双线、京承线、京哈线的汇集点，管辖[[大红门站_(铁路)|大红门站]]、[[百子湾站_(铁路)|百子湾站]]、[[广安门站|广安门站]]、[[北京东站|北京东站]]、[[通州站|通州站]]、[[李营站|李营站]]、[[小红门站_(铁路)|小红门站]]等。
