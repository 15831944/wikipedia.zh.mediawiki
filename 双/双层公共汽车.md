'''雙層巴士'''，或稱作'''-{zh-hans:双层巴士; zh-hant:雙層巴士; zh-hk:雙層公車;zh-sg:双层公交车}-'''、'''-{zh-hans:双层公交; zh-hant:雙層公交; zh-hk:雙層公交;zh-sg:双层公车}-'''，是指載客車廂由上下兩層組成的[[公共汽车|公共汽车]]。

==构造==
雙層巴士由兩层构成，两层之间有[[楼梯|楼梯]]相连接，楼梯在车厢的中部或后部。大多的巴士有兩扇车[[门|门]]，分别设在车的前部和中部。虽然汽车的长度和宽度与一般巴士并无差别，但由于多了一层，車輛高度比其他的高，这使得此种车型的运載量比其它车要大，比[[單層巴士|單層巴士]]能接載多一倍的乘客。雖然雙層巴士較容易受到建築物的高度限制，但相比起要達到相同載客量的掛接式巴士，雙層巴士在轉向及上落斜坡方面會較掛接式巴士靈活。

雙層巴士起源於[[英國|英國]]，其中行走[[倫敦|倫敦]]的紅色雙層巴士，更是英國的「國寶」。在一些（前）[[英聯邦|英聯邦]]國家和地區，諸如[[印度|印度]]、[[香港|香港]]、[[新加坡|新加坡]]、[[南非|南非]]等地，雙層巴士是主要的[[通勤|通勤]]交通工具之一。另外[[德国|德国]]，特别是北部德国城市也大量使用。由於車體龐大，重心較高，驾驶员和乘客都要特别注意乘车安全，雙層巴士的設計要特別考慮穩定性。因此，双层巴士在設計上需更慎重考慮其車體和物理特性對機械設計和運行的影響。而為免更進一步將車輛[[重心|重心]]升高，乘客一般也不能站在上层，以免巴士翻側。在一些地方例如[[香港|香港]]，新型號巴士在獲准載客前必須通過[[傾斜測試|傾斜測試]]。

==利用情况==
不少地方例如[[中國大陸|中國大陸]]、[[台灣|台灣]]和[[澳門|澳門]]，都曾引進過雙層巴士，但由於行車安全、經營成本，甚至是[[人行天橋|人行天橋]]的高度限制等因素，最後還是以失敗告終。在目前一些[[歐洲|歐洲]]、北美城市及中國大陸、[[馬來西亞|馬來西亞]]、[[日本|日本]]一些城市，雖也有雙層巴士，惟主要還是作为观光巴士以及長途客運。这些车通常有兩種，分別為車身低矮，設有後置樓梯及行李艙的雙層遊覽車，以及上层是露天或半露天的雙層观光巴士。英國、香港及新加坡因為很早已普及使用雙層巴士，並透過建築物及道路的相關法規，新建的建築物須要保留雙層巴士能夠通過的高度，確保雙層巴士可以通行無阻。

現時，中國城市[[北京|北京]]、[[上海|上海]]、[[廣州|廣州]]、[[南寧|南寧]]、[[西安|西安]]、[[深圳|深圳]]、[[天津|天津]]、[[杭州|杭州]]等都有雙層公車行走。长途汽车的上層甚至設置铺位。

[[上海|上海]][[好運道書局|好運道書局]]發行的一張[[明信片|明信片]]，黑白照片顯示1920年代中國[[南京路|南京路]]已經出現雙層巴士。<ref>{{cite web|url=http://www.flickr.com/photos/china-postcard/3732563106/|title=上海南京路上的双层公交车 Shanghai 1920s|publisher=}}</ref>

[[臺灣|臺灣]][[臺北市|臺北市]]早在1978年有過引進雙層巴士的計畫，當時[[臺北市政府建設局|臺北市政府建設局]]屬意引入12米長、4米高的德國製雙層巴士，總載客量達126人，但由於超過法令的高度限制問題（臺灣法令規定大客車車高不能超過3.8米），因此需要修改法令才可引入，引進雙層巴士計畫因而不了了之<ref name=台灣慢慢爬/>。到了1990年臺北市引入了雙層巴士試行投入路線營運，為兩輛在香港裝嵌車身的左-{zh-hant:軚;zh-hans:舵}-[[利蘭奧林比安|利蘭奧林比安]]11米雙層巴士（台灣以Volvo作登記）<ref>{{Cite web |url=http://www.busfocus.info/greaterchina/taiwan/taipei-opentop |title=雙層巴士再現臺北 |publisher=Bus Focus 巴士之訊 |date=2017-02-02}}</ref><ref>{{youtube|j92qID2xPxI|懷舊台北市利蘭雙層公車試行上路}}</ref>，在1990年3月1日早上6時首航[[台北聯營公車74路線|74號路線]]，該計畫預定試行3個月。新的雙層巴士按香港標準設計，上層車廂只有在樓梯扶手設下車鐘，下層配用長條形膠帶按鐘<ref name=台灣慢慢爬/>。巴士上路後引來許多臺北市民好奇目光，更有不少民眾因新鮮感而試搭，在尖峰時段、上學時間等全車爆滿，惟部分人對搭乘這種雙層巴士的方法並不熟悉，如不知危險性地於上層站立，也有反映樓梯過於狹窄且陡峭不易上落<ref>facebook.com/groups/193180350726223/permalink/1695459933831583/ 台灣巴士文化協會facebook專頁 </ref>，更有人因深怕來不及按下車鐘而不敢在上層坐，引致下層客滿，上層仍有座位的情況出現<ref name=台灣慢慢爬/>。後來[[臺北捷運|臺北捷運]]工程導致74號需要改道而行，路段因此不再適合雙層巴士行駛，之後巴士試行了504、[[台北聯營公車505路線|505線]]，另也曾建議過為巴士安裝空調，但代理商稱技術問題而未能成事。試驗結果認為在提升運載能力及減低日常營運成本方面雙層巴士有優勢，惟最後因購車成本預算問題、巴士繞路避開過矮之陸橋（香港稱天橋）徒添行車時間等，計畫最終無疾而終<ref name=台灣慢慢爬>{{Cite web |url=https://www.facebook.com/taiwanbustraveler/posts/1829418380648112:0 |title=台灣慢慢爬facebook專頁 |date=2017-01-17}}</ref><ref>{{Cite web |url=http://www.taiwanhot.net/?p=406668 |title=北市雙層觀光巴士2017年1月15日上路　票價300元起 |date=2016-12-23 |publisher=台灣好新聞}}</ref>，車輛則運回香港，輾轉至2000年代才拆毀<ref>{{Cite web |url=http://www.chinamotorbus.com/retire/disappeared/retire-left-drive-olympian/ |title=尋找退役巴－移民又回歸的左軚Olympian |accessdate=2018-01-10 |date=2016-08-02 |publisher=中華巴士紀念館}}</ref>。

马来西亚的双层巴士则可分为公共和旅游两种，其中公共双层巴士可以在[[巴生谷|巴生谷]]和[[槟城|槟城]]找到。其中[[槟城快捷通|槟城快捷通]]目前共使用3辆双层巴士川行，并将在不久后投入30辆双层巴士以容纳更多乘客<ref>{{cite news|url=http://www.sinchew.com.my/node/1610941|title=缓和交通拥挤·槟城快捷通将增30双层巴|date=2017-02-03|publisher=[[星洲网|星洲网]]}}</ref>。吉隆坡的双层巴士则有100余辆，每辆可容纳108名乘客<ref>{{cite news|url=http://www.sinchew.com.my/node/1034109|title=每辆可容纳108乘客．rapid KL双层巴士月中开跑|date=2015-05-11|publisher=[[星洲网|星洲网]]}}</ref>。作为旅游用途的双层巴士服务则有槟城的[[随乘随下|随乘随下]]巴士和吉隆坡的随乘随下巴士，然而收费较高，主要于旅游景点川行。
