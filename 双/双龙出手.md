{{作品名稱|name=''2 Guns''|hk=孖Gun雙雄|tw=2槍斃命|cn=双龙出手}}是一部2013年美国喜剧动作片，[[冰岛|冰岛]]导演[[巴塔萨·科马库|巴塔萨·科马库]]执导，[[丹泽尔·华盛顿|丹泽尔·华盛顿]]和[[马克·沃尔伯格|马克·沃尔伯格]]主演<ref name ="notes">{{cite web  | url =http://www.2guns.net/pdf/2GUNScastCrew.pdf| title = Production notes|publisher = [[Universal_Studios|Universal Studios]]| accessdate = August 3, 2013}}</ref>。根据 Steven Grant 所著的同名[[漫画|漫画]]改编而成，美国于2013年8月2日上映<ref name="ComicBook, Kick-Ass 2, 2 Guns Get New Release Dates; R.I.P.D. ==剧情==
一名美国缉毒局探员和一位[[美国海军|海军]][[卧底|卧底]]上士在一次任务中不期而遇，两人都发现自己陷入深不可测的圈套，无奈之下只能联手以共同找出幕后真相。

==评价==
[[烂番茄|烂番茄]]新鲜度57%，平均分6分；[[Metacritic|Metacritic]]上平均得分59，好评比负评要多一些。

代表性评价：“对一部动作电影而言，除了动作本身之外，最好的调味剂就是演员自身的魅力。而丹泽尔-华盛顿与马克-沃尔伯格的联手，正是将这点发挥到极致。对于这个到处都能碰见超级英雄哑弹的暑期而言，他们不啻于一个惊喜”，“导演在片中设置了大量的起承转合，符合暑期档对一部商业影片的所有娱乐要求”，“说句老实话，从影片本身的角度来看，《双龙出手》实在是太过平庸。若不是两位主演的完美發挥和魅力使然，可看性几乎为零”， “虽然有很好的可切入点和发挥空间，但导演并没有尝试挖掘，导致从各个角度而言都只流于表面且乏善可陈”。<ref>[http://ent.sina.com.cn/m/f/2013-08-05/ba3980171.shtml 新片《双龙出手》《蓝精灵2》《欧罗巴》]</ref>

== 票房 ==
美国首周末收获2736万美元名列第一位。<ref>[http://www.apdnews.com/news/31087.html “双龙”欢喜登顶票房榜] {{webarchive|url=https://web.archive.org/web/20130813164247/http://www.apdnews.com/news/31087.html |date=2013-08-13 }}，亚太日报，2013年8月6日</ref>次周末收入1112万排名第五。全球票房为1.31亿。
