{{Vgname|双点医院|en=Two Point Hospital}}是由{{link-en|双点工作室|Two Point Studios}}所开发，[[世嘉|世嘉]]于2018年8月30日发行的，以[[医院|医院]]为背景的[[模拟经营游戏|模拟经营游戏]]，支持[[Microsoft_Windows|Windows]]、[[MacOS|MacOS]]及[[Linux|Linux]]平台。<ref name="uYWge" /><ref name="UDTyn" /><ref name="yVvOP" />

因为游戏的制作人{{link-en|马克·韦伯利|Mark Webley}}、设计师{{link-en|加里·卡尔|Gary Carr (video game developer)}}及程序师本·荷姆斯都是1997年发行的同类游戏《[[主题医院|主题医院]]》的开发者，《双点医院》被视为《主题医院》的精神续作。<ref name="EuroGamer" /><ref name="PCGamer_management" /><ref name="VentureBeat" /><ref name="GameSpot" /><ref name="PolyGon" /><ref name="IGN" />

== 玩法 ==

故事发生在虚构的“双点县”，玩家饰演了“双点基金会”（默认名，可自行修改）旗下多间医院的经营者。在实际游戏中，玩家需要对各医院的科室规划、人才培养、技术研发、营销策略等方面进行部署。<ref name="GamesTMDev" />

== 发行 ==

2018年1月16日<ref name="Kotaku_co_uk_successor" />，《双点医院》制作组发布一段[[YouTube|YouTube]]短片，以[[英式幽默|诙谐]]的形式讲述了一位“光头症”（头部变成了发光大灯泡）患者的故事。<ref name="destructoid" /><ref name="EuroGamer" />

2018年2月17日，制作组通过《[[PC_Gamer|PC Gamer]]》周刊首度公开游戏画面，展示了游戏操作界面和可调整视角，同时宣布将于同年8月正式发行。<ref name="Kotaku_debut" /><ref name="PCGamer_Ailments" /><ref name="Eteknix" />

2018年7月，游戏于[[Steam|Steam]]平台开始预售。商店页面显示游戏将自带[[繁体中文|繁]][[简体中文|简]]中文界面，正式发行日为2018年8月30日。<ref name="euro_TwoP" />
