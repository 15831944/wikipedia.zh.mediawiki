'''双绞线'''（'''Twisted pair'''）是由两條相互绝缘的导线按照一定的规格互相缠绕（一般以顺时针缠绕）在一起而制成的一种通用配线，属于信息通信网络传输介质。

双绞线过去主要是用来传输[[類比信号|類比信号]]的，但现在同样适用于[[数字信号|数字信号]]的传输。

== 原理 ==
把两根绝缘的铜导线按一定规格互相绞在一起，可降低[[電磁干擾|信号干扰]]的程度，每一根导线在传输中辐射的电波会被另一根线上发出的电波抵消。其中外皮所包的导线两两相绞，形成双绞线对。

== 电缆类型 ==
* [[屏蔽双绞线|屏蔽双绞线]]（STP）於線外有金屬網以屏蔽電磁干擾。
* [[非屏蔽双绞线|非屏蔽双绞线]]（UTP）

== 接頭类型 ==
* [[8P8C|RJ-45]]接头：双绞线与网卡，双绞线与集线器的接口，俗称水晶头。

== 规格型号 ==
[[電子工業聯盟|EIA]]/{{le|電信行業協會|Telecommunications Industry Association|TIA}}为双绞线电缆定义了五种不同质量的型号。计算机网络综合布线使用第三、四、五类。
这五种型号如下：
* 1类（CAT-1）：主要用于传输语音，用于[[数据|数据]]传输。
* 2类（CAT-2）：传输频率为1MHz，用于语音传输和最高传输速率4Mbps的数据传输，常见于使用4Mbps规范[[令牌传递|令牌传递]]协议的旧的[[令牌環|令牌環]]。
* 3类（CAT-3）：指目前在[[ANSI|ANSI]]和[[EIA/TIA568|EIA/TIA568]]标准中指定的电缆。该电缆的传输频率为16MHz，用于语音传输及最高传输速率为10Mbps的数据传输，主要用于[[10BASE-T|10BASE-T]]。
* 4类（CAT-4）：该类电缆的传输频率为20MHz,用于语音传输和最高传输速率16Mbps的数据传输，主要用于基于[[令牌|令牌]]的局域网和[[10BASE-T|10BASE-T]]/[[100BASE-T|100BASE-T]]。
* 5类（[[CAT-5|CAT-5]]）：该类电缆增加了绕线密度，外套一种高质量的绝缘材料，传输频率为100MHz,用于语音传输和最高传输速率为100Mbps的数据传输，主要用于[[100BASE-T|100BASE-T]]和[[10BASE-T|10BASE-T]]网络，这是最常用的[[以太网|以太网]]电缆。
* 超5类（CAT-5e）：超5类具有衰减小，[[串扰|串扰]]少，并且具有更高的衰减与串扰的比值（ACR）和信噪比（Structural Return Loss）、更小的时延误差，性能得到很大提高。
* 6类（[[CAT-6|CAT-6]]）：10BASE-T/100BASE-T/1000BASE-T。傳輸頻率為250MHz傳輸速度為10Gbps標準外徑6mm。  
* 擴展6類（CAT-6A）：10GBASE-T。傳輸頻率為500MHz傳輸速度為10Gbps標準外徑9mm。
* 擴展6類（CAT-6e）傳輸頻率為500MHz傳輸速度為10Gbps標準外徑6mm。
* 7類（CAT-7）：傳輸頻率為600MHz傳輸速度為10Gbps單線標準外徑8mm多芯線標準外徑6mm。  
*8類 (CAT)：40GBASE-T。傳輸頻率為2000MHz傳輸速度為40Gbps。
