「双勾填墨」又称「双钩填墨」，原为[[工笔画|工笔画]]画法中的一种技法，简称'''双钩法'''，一般为一个部分采用两笔勾勒进行描边封闭，再采用相应的颜色进行填充。

经过演变后，在书画[[臨摹|临摹]]方面得到了广泛的应用，很多古籍的书法临摹的精本均采用了双钩法，作品一般称为'''双钩摹本'''。

雙鉤填墨最為接近真跡，保持原本的筆墨情趣和神態風韻。

== 歷史 ==
雙鉤填墨源於東西晉[[南北朝|南北朝]]時天師道的「畫符郭填」。天師道的畫符郭填，先用細勁的線條勾勒成籙文，迂迴曲折，極相近[[元代|元代]]的「八思巴文」[[篆體|篆體]]，然後再在空心處填上顏色。後來書法中的雙鉤填墨與這種手法同出一轍。
雙鉤填墨在隋唐相當流行。[[唐太宗|唐太宗]]篤尚王字，獨倡右軍，在[[越州|越州]]辯才處賺得[[蘭亭序|《蘭亭》]]后，命歐陽詢、虞世南、褚遂良、馮承素、趙模、湯普徹、諸葛貞、薛稷等能書者和內廷拓書手摹之，賜給皇太子、諸王及近臣。
[[北宋|北宋]]時期，[[印刷術|印刷術]]發明，古人法書多賴此以傳承。刻帖摹勒逼真，精神完足，為學書者之最佳範本。「雙鉤填墨」至此而告終止。

== 具体方法 ==
將原作向光放置，如貼在窗戶上，然後用很薄的油紙(不透墨，防止污損原作)覆蓋在原作上將其摹下來，然後再以油紙作為中介，轉到普通的紙上。

== 代表作品 ==
雙鉤填墨運用到[[中國|中國]]書壇可謂源遠流長。流傳至今的「[[蘭亭序|蘭亭文集序]]」所有版本，包括最為有名的「神龍本」、「定武本」等，絕大部分出源於雙鉤填墨。其它有藏於[[台北|台北]][[國立故宮博物院|國立故宮博物院]]的[[王羲之|王羲之]]「平安三帖」，即[[平安何如奉橘三帖|《平安》、《何如》、《奉橘》]]帖，真跡無留下，人們所見的也是唐人的雙鉤填墨本。[[奈良時代|奈良時代]]流入[[日本|日本]]的《喪亂》、《二謝》、《邵兒》、《左邊》、《得示》五帖，與《哀禍》、《孔侍中》、《憂懸》三帖也是唐人之摹本。
