《'''双面胶'''》，[[新加坡|新加坡]][[小说家|小说家]][[六六|六六]]的著作，讲述[[中国大陆|中国大陆]]普通家庭婆媳之间矛盾的故事。<ref name=" sit" >{{cite book |author=六六 |title=《双面胶》|year=2005年 |publisher=上海人民出版社 |location= |pages=前言 |isbn=9787208058224 |language = zh-cn}}</ref>

== 故事 ==
小说讲述了[[中国大陆|中国大陆]]普通家庭[[上海|上海]]媳妇和[[中国东北地区|东北]]婆婆的婆媳之间矛盾的故事。<ref name=" sit" />

胡丽娟是土生土长的上海女人，后来她嫁给了东北小伙子李亚平，两人在丈母娘家的帮助下在上海买房安了家。李亚平从一个粗夯鲁莽的东北人变成了一个体贴温柔的[[上海男人|上海男人]]，不过好景不成，自从婆婆搬过来之后，婆婆决心用中华传统文化的[[三纲五常|三纲五常]]君君臣臣父父子子来改造媳妇胡丽娟，婆媳之间的矛盾自此爆发并且闹得不可开交，最后使得这个温馨的家庭从此破裂。

== 风格 ==
小说用平淡朴实的家庭生活表现了中国上千年的婆媳关系主题，故事类似远古诗歌《[[孔雀东南飞|孔雀东南飞]]》，作品中流露出作者对[[封建|封建]][[伦理|伦理]]和[[礼教|礼教]]的对抗精神。<ref name=" sit" />

== 书中人物 ==
*胡丽娟，上海女人，温柔贤惠，后嫁给了东北人李亚平。
*李亚平，东北男人，粗夯鲁莽，娶胡丽娟为妻子。
*亚平妈，[[东北人|东北人]]，传统保守，重视[[三纲五常|三纲五常]]封建伦理道德。

== 作者 ==
