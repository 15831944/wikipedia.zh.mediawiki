《'''双城计中计'''》（{{lang-en|'''Scheme With Me '''}}）2012年2月24日上映于[[中国大陆|中国大陆]]的战争背景喜剧，电影背景为民初，日寇军队侵略中华大地。有抄襲《[[飛天大盜|飛天大盜]]》的爭議。

== 剧情 ==
中国民国初年，中华大地饱受[[日本人|日本人]]的侵略和凌辱，

日本人委托老千陈少卿送给[[军阀|军阀]]王大帅20箱黄金，这个消息马上在[[上海市|上海市]]流传开来，[[上海滩|上海滩]][[黑帮|黑帮]]头子林啸东得知后，打起了黄金的主意。另一个人千面鬼脸也想把黄金捞到手。于是，鬼脸经常去林啸东的[[赌场|赌场]]赌博，顺便打听消息，同时又可以赚些[[大洋|大洋]]。千面鬼脸因为在唐探长身上狠狠的赚一笔，被林啸东知道后带人把他抓了起来，随后把他押送到监狱。鬼脸在监狱认识了不动石佛。20箱黄金的消息牵引着各方的欲念，他们都开始谋划如何算计对方，最后自己成为赢家。

日本人把黄金弄到了[[西北|西北]]。没有多久，他们都追到了西北边境的荒凉小镇。石佛在荒漠中的西北小镇遭遇到了美丽的女土匪金三娘，不久之后，金三娘就爱上了他，并且开始追恋石佛。

林啸东的计划最先破产，被王大帅率兵围歼，林啸东被擒获，随后，王大帅和日本人展开了战争，日本人和王大帅都两败俱伤，而精明的鬼脸却趁机假扮王大帅把黄金劫持走了。

== 花絮 ==
[[导演|导演]][[潘安子|潘安子]]向媒体解说时强调说“该片是喜剧片，片中有[[爱情|爱情]]、兄弟情；有计谋、有诈骗；有笑话，有幽默；有古装，有现代，是一部多元素电影。”<ref>{{cite web |url=http://news.xinhuanet.com/local/2012-02/29/c_111584269.htm |title=《双城计中计》：元素齐全的解压电影 |publisher= 新华网|accessdate=2012年2月29日  |work= |language =}}</ref>

电影中[[腾格尔|腾格尔]]和[[翁虹|翁虹]]有床戏，对此[[任贤齐|任贤齐]]表示羡慕不已，而翁虹表示：“我们演[[床戏|床戏]]那场等了很久才拍上。他不太习惯等好久，我俩就一直聊天。其实拍摄床戏的拍摄过程中我们并未碰面，戏里的内容是我被他灌醉了。”，“他越斯文我就越喜欢。”<ref>{{cite web |url=http://ent.163.com/12/0221/20/7QQK6992000300B1.html |title=《双城计中计》首映 任贤齐羡慕腾格尔有床戏 |publisher=网易 |accessdate=2012-02-21  |work= |language =}}</ref>

[[熊乃瑾|熊乃瑾]]柔中带刚、搞笑幽默的风格赢得了观众的认可。<ref>{{cite web |url=http://yule.sohu.com/20120307/n336953549.shtml |title=《双城计中计》票房走高 熊乃瑾展露喜剧天赋 |publisher= 搜狐网|accessdate=2012年3月7日 |work= |language =}}</ref>

电影的语言和[[黑色幽默|黑色幽默]]效果类似《[[让子弹飞|让子弹飞]]》，播出首周票房就达到了1800万。<ref>{{cite web |url= http://news.cntv.cn/20120301/120650.shtml|title=《双城计中计》首周票房1800万 黑色幽默获好评 |publisher=中国网络电视台 |accessdate= 2012年3月1日|work= |language =}}</ref>
