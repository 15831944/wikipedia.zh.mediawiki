'''重庆市双桥经济技术开发区'''是[[重庆市|重庆市]]的一个市级经济技术开发区，位于重庆市[[大足区|大足区]]南部，地处成渝之心，同时位于重庆1小时经济圈和成都1小时经济圈，是重庆市产城融合发展示范开发区和重要的汽车及机械装备制造基地。<ref name=":2">{{Cite web|url=http://sq.cq.gov.cn/jkqgk/2/|title=双桥经开区-重庆市双桥经济技术开发区网站-经开区概况|accessdate=2018-03-17|work=sq.cq.gov.cn}}</ref><ref>{{Cite web|url=http://sq.cq.gov.cn/jkqgk/29/|title=双桥经开区-重庆市双桥经济技术开发区网站-地理位置|accessdate=2018-03-17|work=sq.cq.gov.cn}}</ref>

== 历史沿革 ==
双桥经济技术开发区前身为重庆市[[双桥区_(重庆市)|双桥区]]。1965年，为响应[[三线建设|三线建设]]，经[[第一机械工业部|第一机械工业部]]批准，[[四川汽车制造厂|四川汽车制造厂]]（后更名为[[重庆红岩汽车公司|重庆红岩汽车公司]]，现为[[上汽集团|上汽]][[依维柯|依维柯]]红岩商用车有限公司）于[[大足区|大足县]]（当时属[[江津地区|江津地区]]）双路公社建成投产。1974年，为便于国家拨专款改善设施条件，经[[四川省革命委员会|四川省革命委员会]]申请，国务院发布《国务院关于建立[[大足汽车制造厂|大足汽车制造厂]]行政区问题的批复》（国发〔1974〕100号文件），批准从大足县划出双路公社和元通公社的青春、天星、白鹤、天桥及土桥公社的茅店、新民六个生产大队（六个生产大队另外组建通桥公社），建立重庆市双桥区，隶属于重庆市<ref>{{Cite book|url=https://www.worldcat.org/oclc/40231756|title=大足县志|last=李|first=传授|last2=张|first2=划|last3=大足县县志编修委员会|first3=|date=(1997 printing)|publisher=方志出版社|year=|isbn=7801221605|edition=第1版|location=北京市|pages=74|chapter=|oclc=40231756}}</ref>。在重庆直辖后，仍然属重庆市。

2011年10月24日，经国务院《关于同意重庆调整部分行政区划的批复》（国函〔2011〕129号文件）批准，撤销大足县、双桥区，设立大足区，原双桥区管辖区域由大足区管辖，同时设立重庆市双桥经济技术开发区，管辖原双桥区管辖区域与邮亭镇，由重庆市委、市政府委托大足区委区政府管理。2011年12月25日，大足区与双桥经济技术开发区同时挂牌成立。<ref name=":3">{{cite news|url=http://cq.sina.com.cn/news/snews/2011-12-26/13414_2.html|title=双桥经济技术开发区挂牌揭幕|date=2011-12-26|work=人民网|accessdate=2018-03-17}}</ref>

在2016年，国务院重新启动[[国家级经济技术开发区|国家级经济技术开发区]]的申报后，双桥经济技术开发区已被重庆市纳入进入培育期的省级经开区名单{{NoteTag|根据国务院办公厅下发的《关于完善国家级经济技术开发区考核制度促进创新驱动发展的指导意见》（国办发〔2016〕14号），“ 对申请新设立或升级为国家级经开区的，给予2年培育期，待培育期满后进行实地考察，经综合评价其各项指标在被培育的省级经济开发区中位居前列的，启动新设或升级办理程序”。}}，进行国家级经济技术开发区的申创的准备工作。<ref>{{cite web|url=http://www.cq.gov.cn/publicmail/citizen/ViewReleaseMail.aspx?intReleaseID=951484|title=双桥国家级经开区_重庆市政府公开信箱|accessdate=2018-03-17|author=|date=2017-11-28|publisher=|website=}}</ref>

== 行政管辖 ==
双桥经济技术开发区并不是[[中华人民共和国民政部|民政部]]在册的行政区，开发区域在行政上隶属于大足区。中共双桥经济技术开发区工作委员会和双桥经济技术开发区管理委员会是[[中共重庆市委|中共重庆市委]]、[[重庆市人民政府|重庆市人民政府]]的正厅级派出机构，行使市级行政权力，位于重庆市双桥经济技术开发区双龙西路106号（行政上隶属于重庆市大足区双路街道）。

双桥经济技术开发区管辖原双桥区管辖的[[龙滩子街道|龙滩子街道]]、[[双路街道|双路街道]]（原双路镇）、[[通桥街道|通桥街道]]（原通桥镇）和原大足县管辖且与双桥区邻近的[[邮亭镇|邮亭镇]]。其中，双桥经济技术开发区主要负责经济发展和开发建设工作，党务、社会事务由大足区委、区政府统筹管理。<ref name=":3" />

辖区内另有重庆双桥工业园区、重庆双桥邮亭工业园区、重庆双桥车城工业园区。

== 经济 ==
2017年，双桥经济技术开发区实现地区生产总值160亿元，增长13%，其中工业增加值130亿元，增长13%；实现工业总产值625亿元，增长20.3%，其中规上工业总产值完成480亿元，增长19.6%；全社会固定资产投资120亿元、增长13%, 其中工业投资100亿元、增长15%；一般地方公共预算收入7.7亿元，其中税收收入4.1亿元；社零总额完成45亿元，增长14.9%；实际利用外资8000万美元，进出口总额1.2亿美元。<ref name=":2" />

== 交通 ==
双桥经济技术开发区位于[[成渝经济区|成渝经济区]]腹地，交通便利。[[银昆高速公路|银昆高速公路]]横穿开发区，设有大足出口。[[成渝铁路|成渝铁路]]、[[成渝客运专线|成渝客运专线]]也横穿开发区，共设有[[大足站|大足站]]、[[长河碥站|长河碥站]]、[[大足南站|大足南站]]3个车站，抵达重庆只需20分钟，抵达成都只需40分钟。。
