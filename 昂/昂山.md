'''翁山'''（[[緬甸語|緬甸語]]：ဗိုလ်ချုပ် အောင်ဆန်，[[罗马化|罗马化]]：{{lang|la|Aung San}}；{{bd|1915年|2月13日|1947年|7月19日|catIdx=Aung San}}）是帶領[[緬甸|緬甸]]脫離[[英國|英國]]獨立的軍事領袖。

1942年，昂山接受[[日本|日本]]的援助，協助日本攻打緬甸的[[英國|英國]][[英屬緬甸|殖民政府]]，以推行民主政治來結束[[英國|英國]]的[[殖民統治|殖民統治]]。不久後，翁山懷疑日本會否給予緬甸獨立，於是改為與[[盟軍|盟軍]]合作。[[第二次世界大戰|第二次世界大戰]]結束[[大日本帝國|大日本帝國]]軍[[無條件投降|無條件投降]]後，翁山作為英屬緬甸的最後一任[[緬甸總理|總理]]，與英國展開有關緬甸獨立的談判。但在緬甸正式獨立前，翁山被暗殺身亡，此後被緬甸人民尊稱為[[國父|國父]]。

翁山的女兒是[[緬甸民主運動|緬甸民主運動]]領導人、政黨[[全國民主聯盟|全國民主聯盟]]主席[[翁山蘇姬|翁山蘇姬]]<ref>[http://finance.gucheng.com/201502/2954500.shtml 昂山将军：昂山素季之父] {{webarchive|url=https://web.archive.org/web/20150428104947/http://finance.gucheng.com/201502/2954500.shtml |date=2015-04-28 }}</ref>。

== 學生時代 ==
1915年，翁山生於緬甸中部[[馬圭省|馬圭省]][[那卯鎮|那卯鎮]]（又譯「納特曼克鎮」）。1933年，入讀[[仰光大學|仰光大學]]，很快成為學生領袖，爾後擔任仰光大學學生會執委並與校內刊物《[[孔雀之聲|孔雀之聲]]》（Peacock's Call）主編。1936年2月，因拒絕透露《孔雀之聲》上一篇反英文章（Hell Hound At Large）的作者姓名，連同[[吳努|吳努]]同時受校方開除學籍。翁山、吳努兩人因而領導大學生罷課；後來仰光大學讓步，兩人復課。1938年，翁山當選學生會主席和緬甸全國學生聯合會主席。同年10月，翁山離開仰光大學法律課程，轉而投身政界。

== 反英運動 ==
1930年代，緬甸民族主義日益高漲。

一些愛國青年知識分子，在反對英國[[殖民主義|殖民主義]]統治的運動中，成立了[[德欽黨|德欽黨]]（又稱「我緬人黨」）。

1938年，翁山加入德欽黨，改稱德欽翁山（Thakin Aung San），並當選總書記。同年領導石油工人罷工，各地工會、民眾、學生亦相續加入，工運最後演變為全國性的反英運動，即[[緬甸歷史|緬甸歷史]]著名的“[[1300運動|1300運動]]”。1939年8月15日，[[緬甸共產黨|緬甸共產黨]]成立，翁山任總書記。同年10月1日，翁山聯合德欽黨、緬甸共產黨、貧民黨等組成自由聯盟（亦稱緬甸出路派組織），由[[巴莫|巴莫]]出任主席，翁山出任總書記。1940年3月，翁山參加印度國民代表大會。後來，緬甸殖民地政府因翁山的反英活動通輯翁山，他為了逃避追捕及尋求[[中國共產黨|中國共產黨]]援助而來到中國，結果到達[[廈門市|廈門]]時，被日本特務攔截，[[鈴木敬司|鈴木敬司]]建議昂山改道日本，從此昂山投向日本的怀抱。

當時日本計劃攻打緬甸以破壞印緬滇國際援華的通道[[滇緬公路|滇緬公路]]，與[[大本營_(大日本帝國)|日本大本营]]陆军部直属特务机关——[[南机关|南机关]]（[[鈴木敬司|鈴木敬司]]大佐化名南益世而得名）達成協議後，翁山秘密回到緬甸並帶29青年（史稱「{{le|三十志士|Thirty Comrades}}」）1941年2月1日到日本占领的[[海南岛|海南岛]]接受軍事訓練，准备组织武装起义配合日军侵入英属缅甸，1942年3月8日，昂山攻佔首都仰光，同年3月至8月间，昂山又协助日军击败[[中国远征军|中国远征军]]，曾受过中国人帮助的昂山对待中国战俘的手段非常残忍，約1500名中国远征军伤员不愿受被俘虏之辱，慨然于1942年5月21日凌晨集体引火自焚，含恨而终。事实上，昂山此时也已经脱离了[[緬甸共產黨|緬甸共產黨]]，[[德欽索|德欽索]]任[[緬甸共產黨|緬甸共產黨]]第二任总书记。

== 緬甸國成立 ==
1941年12月27日，翁山在[[泰國|泰國]][[曼谷|曼谷]]組建[[緬甸獨立軍|缅甸独立义勇军]]，引領日軍進軍緬甸。1942年3月，日本佔領[[仰光|仰光]]。同年7月，全緬英軍被擊退，緬甸獨立軍改組為[[緬甸军事|緬甸國防軍]]，翁山被授予[[大佐|大佐]]軍銜。1943年3月，翁山晉升少將，並到[[東京|東京]]覲見[[昭和天皇|天皇]]，被授予[[旭日章|旭日勛章]]。1943年8月1日，[[緬甸國|緬甸國]]成立，巴莫出任總理，翁山出任國防部長。缅甸独立义勇军改称改组为缅甸国民军（Burma National Army）。在此時期，翁山一度使用'''面田纹次'''（''omota monji''）作為自己的[[日本人名|和名]]。

== 抗日战争 ==
随着太平洋战争的推进，翁山对日本所承诺的缅甸独立事務乃至日本自身能否维持局面都产生了怀疑；爾后[[英帕尔战役|英帕尔战役]]裡，日军惨败的结果使翁山决意背弃日本。1943年11月，翁山向英军传达了“正在考虑转投阵营”的意向；1944年8月1日，翁山又在独立一周年的演说中做出“缅甸独立是场骗局”的发言。8月底，缅甸共产党、人民革命党等联合建立了“反法西斯组织”，翁山出任其军事指挥。1944年月英国达成协议：缅甸军队参与盟军的对日作战；在结束战斗的地方，缅甸军队解散。

1945年3月，缅甸北方一部分国防军与日军发生冲突；3月27日，翁山在以镇压叛军的名义将缅甸国防军召集至[[仰光|仰光]]后，突然调转枪口向日军开火。“反法西斯组织”所属的各势力亦同时蜂起，与盟军站在同一战线上的缅甸抗日运动就此开始。5月仰光光复，6月15日发表了抗日胜利的宣言，同时“反法西斯组织”亦扩大规模并改成[[反法西斯人民自由同盟|反法西斯人民自由同盟]]（AFPFL）。人民自由同盟內部在對待一系列與英人合作與獨立形式的問題上，出現了嚴重的分歧。[[緬甸共產黨|緬甸共產黨]]對於昂山的“中間路線”進行了強烈的抨擊。1946年10月10日，由於共產黨批評昂山鎮壓罷工，人民自由同盟執行委員會通過決議，把緬共排除出了人民自由同盟。這個行動，表明緬甸向英國表明獨立後不會加入[[共產主義|共產主義]]陣營，使緬甸更順利走向獨立。

== 暗殺 ==
1945年8月15日[[第二次世界大戰|第二次世界大戰]]结束[[大日本帝國|大日本帝國]][[無條件投降|無條件投降]]后，缅甸恢復战前英国殖民地的状态。翁山所率领的缅甸军队与英国指挥下的缅甸军合併；翁山本人则于1946年9月出任英属缅甸政府行政議会的副议长（[[政府首腦|政府首腦]]），负责国防和外交政策。同时，作为独立主义者的翁山仍然以反法西斯人民自由同盟黨总裁的身份，持续着以完全独立为目标的政治活动。

1947年1月27日，翁山与当时的[[英国首相|英国首相]][[克莱门特·艾德礼|克莱门特·艾德礼]]定下了保证缅甸在一年内完全独立的“昂山-艾德礼协定”。2月，试图调和国内各派冲突的[[彬龙协议|彬龙协议]]发表，然而在对英交涉艰难和对内调和反复之中的独立路上，缅甸国内的混乱并未平息。

1947年4月制宪议会选举，反法西斯人民自由同盟黨取得了202议席中的196席；尽管选举取得压倒性胜利，但翁山於1947年7月19日，經由保守派前總理[[吴苏|吴苏]]指使下，与6名阁僚于仰光被暗杀，得年32岁，未能成為[[緬甸|緬甸]]獨立後的第一任總理。他的領導權及繼任人由[[吳努|吳努]]繼承。

由於昂山生前是緬甸軍方的領導人並統領軍隊，他的突然離世導致軍方的權力真空。獨立後軍方和文人官僚的政治矛盾加劇，最終為後來導致軍方於1962年發動政變全面接管政權埋下伏線。
