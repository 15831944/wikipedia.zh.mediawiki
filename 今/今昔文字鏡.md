'''今昔文字鏡'''（{{jpn|j=今昔文字鏡|hg=こんじゃくもじきょう}}）是日本[[AINet|AINet]]開發的一款東亞文字檢索軟件，在[[Windows|Windows]]平臺上運行。收字量龐大，最新版本收錄文字達17萬以上。所收字類型有[[諸橋轍次|諸橋轍次]]編《[[大漢和辭典|大漢和辭典]]》中的全部漢字、[[和製漢字|和製漢字]]、[[簡化字|簡化字]]、[[方言字|方言字]]、[[甲骨文|甲骨文]]、[[篆書|篆書]]等各類漢字，[[喃字|喃字]]、[[水书|水族文字]]、[[悉曇文字|悉曇文字]]、[[西夏文字|西夏文字]]、[[變體假名|變體假名]]、[[臺灣語假名|臺灣語假名]]等其他文字。<ref>[http://www.mojikyo.com/info/about/index.html 今昔文字鏡とは]</ref>

== 歷史 ==

今昔文字鏡開發於1985年。1997年，[[大修館書店|大修館書店]]發佈其1.0版本，當時只收有不到8萬個字。1999年，在[[TrueType|TrueType]]上搭載的2.0版本發售。

2001年收錄西夏文字，并發佈“今昔文字鏡　單漢字10万字版”，收錄字數約102,300。同年收錄楷書。“文字鏡Web”網站也開始出現。

2006年發佈“今昔文字鏡　單漢字15万字版”（4.0版），發佈專業版“Indexfont Ver. 1.0”。<ref>XP[32ビット版](NT5.1)・Vista(NT6.0)[32ビット版]にのみ対応。「Windows XP Mode」上での動作についてはサポート対象外。</ref>

2008年3月末“文字鏡Web”關閉。<ref>{{cite web|url=http://www.mojikyo.com/info/web/index.html|title=文字鏡WEB概要|accessdate=2008-04-01}}</ref>6月初“今昔文字鏡 UnicodeEdition”發售。<ref>{{cite web|url=http://www.vector.co.jp/soft/winnt/writing/se458414.html|title=今昔文字鏡UnicodeEdition(WindowsNT-2000-XP-Vista - 文書作成)|accessdate=2008-11-23|deadurl=yes|archiveurl=https://web.archive.org/web/20090306201502/http://www.vector.co.jp/soft/winnt/writing/se458414.html|archivedate=2009-03-06}}</ref>

2010年3月15日，發佈專業版“Indexfont Ver. 2.0”。<ref>XP(NT5.1)・Vista(NT6.0)[32bit版]・'''7(NT6.1)[32 / 64bit版]'''にのみ対応。XP x64 Edition(NT5.2)・Vista(NT6.0)[64bit版]および「Windows XP Mode」上での動作についてはサポート対象外。</ref>

至今(2017年)，今昔文字鏡的網路版都還在重新構築當中。<ref>  [http://www.mojikyo.org/#Banner] </ref>
