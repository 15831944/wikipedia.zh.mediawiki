'''今敏'''（{{jpn|j = 今 敏 |hg = こん さとし |rm = Kon Satoshi }}，{{bd|1963年|10月12日|2010年|8月24日|catIdx = Kon, Satoshi }}<ref name = "永眠のお知らせ">{{Cite web |url = http://konstone.s-kon.net/modules/info/index.php?page=article&storyid=37 |script-title = ja:今 敏　永眠のお知らせ |author =  |date = 2010年8月25日 |publisher = {{lang|ja|今 敏 オフィシャル·サイト}} |language = ja |accessdate = 2015年9月29日 |quote =  }}</ref>）是[[日本|日本]][[動畫|動畫]][[導演|導演]]、[[動畫師|動畫師]]、[[編劇|編劇]]和[[日本漫畫家|漫畫家]]，同時也是{{Link-en|日本動畫從業人員協會|Japanese Animation Creators Association}}會員<ref>{{Cite web |url = http://www.animenewsnetwork.com/news/2008-06-06/japan-animator-labor-group-legally-incorporates |title = Japan's Animator Labor Group Legally Incorporates |author = 《[[每日新聞|每日新聞]]》 |date = 2008年6月6日 |publisher = [[動畫新聞網|動畫新聞網]] |language = en |accessdate = 2015年9月29日 |quote =  }}</ref>。今敏出生於[[北海道|北海道]][[札幌市|札幌市]]，自[[武藏野美術大學|武藏野美術大學]]造形研究科視覺傳達設計系畢業後，先是以漫畫家的身分出道。今敏早期主要跟隨[[大友克洋|大友克洋]]的工作團隊，曾因而擔任過電影的共同編劇，他也在這個時候接觸動畫工作。除了和大友克洋有著密切交流外，他在1993年時曾與押井守合作製作《{{Link-en|機動警察Patlabor 2 the Movie|Patlabor 2: The Movie}}》。今敏在參與多部動畫作品的製作後，決定由其執導動畫電影，並交由[[MADHOUSE|MADHOUSE]]負責動畫製作。

1997年今敏發表他的第一部動畫長篇作品《[[藍色恐懼|藍色恐懼]]》，在這部驚悚片中他以極為寫實的風格傳達故事內容。透過處理日本偶像文化的題材，今敏準確地傳達角色的情感以及人們內心的反思，這對當時剛接觸日本動畫作品的西方世界造成轟動。之後他以不屬於任何動畫製作公司的獨立導演身分，分別在2002年發表《[[千年女優|千年女優]]》、2003年發表《[[東京教父|東京教父]]》等重要代表作，而這些作品都由MADHOUSE製作<ref>{{Cite web |url = http://konstone.s-kon.net/modules/interview/index.php?content_id=26 |script-title = ja:Interview 24 2005年4月、海外から監督作全般に関して |author =  |date = 2005年4月 |publisher = {{lang|ja|今 敏 オフィシャル·サイト}} |language = ja |accessdate = 2015年9月29日 |quote =  }}</ref>。另外今敏還執導電視系列動畫《[[妄想代理人|妄想代理人]]》，在作品中他以「鰐淵良宏」作為自己的筆名。隨後今敏發表自己完成的最後一部動畫電影《[[盜夢偵探|盜夢偵探]]》，這部電影也讓他在國際影壇獲得廣泛認可。

今敏的多部作品中儘管處理不同的題材，不過都有個性鮮明但貼近現實生活的人物、角色精神層面的探討描寫、夢境與現實間的曖昧關係等元素，這讓他成為具有獨特創作風格的日本動畫導演，並在國際間獲得眾多獎項<ref>{{Cite web |url = http://www.nytimes.com/2010/08/26/arts/design/26kon.html?_r=0 |title = Satoshi Kon, Anime Filmmaker, Dies at 46 |author = {{Link-en|A·O·斯科特|A. O. Scott}} |date = 2010年8月26日 |publisher = 《[[紐約時報|紐約時報]]》 |language = en |accessdate = 2015年9月29日 |quote =  }}</ref>。除了動畫製作工作外，今敏還協助創立了日本動畫從業人員協會，希望能藉此改善年輕動畫師的工作條件。他的哥哥{{Link-ja|今剛|今剛}}，則是日本流行音樂界著名的[[吉他手|吉他手]]和伴奏[[音樂家|音樂家]]<ref>{{Cite web |url = http://konstone.s-kon.net/modules/works/index.php?content_id=2 |script-title = ja:魂の玉座 |author =  |date =  |publisher = {{lang|ja|今 敏 オフィシャル·サイト}} |language = ja |accessdate = 2015年9月29日 |quote =  }}</ref>。

== 早年 ==
[[File:Hokkaido_Kushiro_Koryo_High_School.JPG|thumb]]

今敏於1963年10月12日出生在[[日本|日本]][[北海道|北海道]][[札幌市|札幌市]]<ref name="MADHOUSE PEOPLE FILE">{{Cite web |url=http://www.madhouse.co.jp/column/mhpf/mhpf_no035.html |script-title=ja:今 敏　- SATOSHI KON - |author= |date= |publisher=[[MADHOUSE|MADHOUSE]] |language=ja |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20110616212422/http://www.madhouse.co.jp/column/mhpf/mhpf_no035.html |archivedate=2011年6月16日 }}</ref>，不過因為父親工作的緣故，使得他4歲到小學四年級期間生活在位於北海道東部的[[釧路市|釧路市]]。爾後又因父親工作遭到調動，今敏在小學四年級到中學二年級的這段期間，全家搬至札幌市生活。在札幌市居住時，他和漫畫家{{Link-ja|滝澤聖峰|滝沢聖峰}}是同班同學，後來兩人仍是相當親密的朋友。在中學四年級時，今敏一家又從札幌市搬回到釧路市。之後他就讀{{Link-ja|北海道釧路湖陵高等學校|北海道釧路湖陵高等学校}}，畢業前決定要成為一名[[原畫師|原畫師]]<ref name = "Konstone 公式プロフィール">{{Cite book | author =  | script-title = ja:Konstone 公式プロフィール | location = 日本東京 | publisher = {{lang|ja|今 敏 オフィシャル·サイト}} | date =  | pages =  | ISBN =  | accessdate = 2015年9月29日 | url =  | language = ja | quote =  }}</ref>。連同往後前往[[東京都|東京都]]讀書的經歷，這樣不斷來往各處對他後來作品的節奏有很大影響，促成結合想像與現實的作品風格<ref name = "北海道新聞">{{cite news |language = ja |author =  |coauthors =  |url =  |script-title = ja:哀惜·今敏さん（アニメーション映画監督、8月24日死去、46歳） - 名声得ても釧路に愛着 |publisher = 《[[北海道新聞|北海道新聞]]》 |pages = 第6頁 |date = 2010年9月25日 |accessdate = 2015年9月29日 }}</ref>。

今敏在高級中學時便喜歡觀看動畫，特別是1974年的《[[宇宙戰艦大和號|宇宙戰艦大和號]]》和《[[小天使|小天使]]》、1978年的《[[未來少年柯南|未來少年柯南]]》以及1979年的《[[機動戰士鋼彈|機動戰士鋼彈]]》等作品<ref name = "Aimee Giron">{{Cite web |url = http://www.soundandvision.com/httalksto/1205talks/#WIOix20wZV74lxmf.97 |title = HT Talks To……FilmMaker Satoshi Kon |author = Aimee Giron |date = 2005年12月7日 |publisher = 《{{Link-en|有聲視野|Sound & Vision (magazine)}}》 |language = en |accessdate = 2015年9月29日 |quote =  }}</ref>。他也買了很多漫畫作品，其中特別喜歡[[大友克洋|大友克洋]]創作的漫畫《{{Link-en|童夢 (漫畫)|Domu: A Child's Dream|童夢}}》，而成為大友克洋的忠實愛好者<ref name="Director of Perfect Blue">{{Cite web|url=http://www.perfectblue.com/interview.html |title=Interview with Satoshi Kon, Director of Perfect Blue |author= |date=1998年9月4日 |publisher=PERFECT BLUE |language=en |accessdate=2015年9月29日 |quote= |deadurl=no |archiveurl=https://web.archive.org/web/20120402165610/http://www.perfectblue.com/interview.html |archivedate=2012年4月2日 }}</ref>。他自己亦受到小說作家[[筒井康隆|筒井康隆]]的短篇故事影響，甚至以此創作漫畫，這也影響他後來發表的漫畫和動畫風格。1982年從北海道釧路湖陵高等學校畢業後，今敏考上位於東京都郊區的[[武藏野美術大學|武藏野美術大學]]造形研究科視覺傳達設計系，主要學習專業設計等領域<ref name = "Konstone 公式プロフィール"/>。大學就讀期間，他對於許多外國電影有著高度興趣，並熱衷閱讀筒井康隆的小說著作<ref name = "Director of Perfect Blue"/><ref name = "Emmanuel Pettini">{{Cite web |url = http://www.catsuka.com/focuson/itw_satoshikon |title = Interview Satoshi Kon |author = Emmanuel Pettini |date = 2006年10月18日 |publisher = CATSUKA |language = fr |accessdate = 2015年9月29日 |quote =  }}</ref>。

今敏就讀大學期間先是從漫畫創作開始，1984年他以學生身分向[[講談社|講談社]]著名的漫畫雜誌《[[週刊Young_Magazine|週刊Young Magazine]]》投稿個人短篇作品《虜》（{{Lang|ja|虜 -とりこ-}}），並以該漫畫獲得《週刊Young Magazine》舉辦的第10屆年度[[千葉徹彌賞|千葉徹彌賞]]優秀新人獎項<ref name = "Konstone 公式プロフィール"/><ref>{{Cite web |url = http://www.chibapro.co.jp/index.php?startIndex=30&tbl=award_yng&key= |script-title = ja:ちばてつや賞（ヤング部門） |author =  |date =  |publisher = {{lang|ja|ちばてつや}} |language = ja |accessdate = 2015年9月29日 |quote =  }}</ref><ref name = "Justin Sevakis">{{Cite web |url = http://www.animenewsnetwork.com/interview/2008-08-21/satoshi-kon |title = Interview: Satoshi Kon |author = Justin Sevakis |date = 2008年8月21日 |publisher = [[動畫新聞網|動畫新聞網]] |language = en |accessdate = 2015年9月29日 |quote =  }}</ref>。以此為契機，他在隔年於講談社的雜誌上正式出道成為職業漫畫家<ref name = "Justin Sevakis"/>，也曾擔任過著名漫畫家大友克洋的{{Link-ja|漫畫助手|アシスタント (漫画)}}<ref name = "Justin Sevakis"/><ref>{{Cite web |url=http://konstone.s-kon.net/modules/interview/index.php/content0007.html |script-title=ja:Interview 06 1998年3月 フランスから「パーフェクトブルー」に関するインタビュー |author= |date=1998年3月 |publisher={{lang|ja|今 敏 オフィシャル·サイト}} |language=ja |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20120311213135/http://konstone.s-kon.net/modules/interview/index.php/content0007.html |archivedate=2012年3月11日 }}</ref>。1987年今敏自武藏野美術大學畢業後<ref name = "Konstone 公式プロフィール"/>，講談社於1990年推出今敏個人第一部漫畫單行本《[[海歸線|海歸線]]》（{{lang|ja|海帰線}}）<ref name = "COMIC BOOK">{{Cite book | author =  | script-title = ja:COMIC BOOK | location = 日本東京 | publisher = {{lang|ja|今 敏 オフィシャル·サイト}} | date =  | pages =  | ISBN =  | accessdate = 2015年9月29日 | url =  | language = ja | quote =  }}</ref>。

== 從業 ==
=== 早期工作 ===
[[File:Musashino_Art_University.jpg|thumb]]。]]

1990年，今敏擔任大友克洋的動畫電影《[[老人Z|老人Z]]》的美術設定、原畫和構圖，這也是他第一次接觸動畫製作<ref name = "Konstone 公式プロフィール"/><ref name = "Justin Sevakis"/>。大友克洋在這部動畫中，也同時擔任原作者、編劇、[[機械設計_(動畫)|機械設計]]三項重要職務，這項工作促使今敏与大友克洋關係更為密切。在這之後，大友克洋先是讓今敏為他拍攝的真人電影《{{Link-en|國際恐怖公寓|World Apartment Horror}}》撰寫故事劇情<ref name="Justin Sevakis" /><ref name="COMIC BOOK" />。1991年，大友克洋則請今敏將其執導的真人電影《國際恐怖公寓》改編成漫畫，透過發行單行本來讓電影增加不少話題性。另外一方面，今敏還在[[押井守|押井守]]的《{{Link-en|機動警察Patlabor 2 the Movie|Patlabor 2: The Movie}}》擔任分鏡構圖，陸續投入動畫電影的製作<ref name = "Konstone 公式プロフィール"/>。

1993年今敏參與了《[[JoJo的奇妙冒險|JoJo的奇妙冒險]]》的製作工作，分別在第二話擔任原畫；第五話擔任編劇、分鏡和導演；以及第六話擔任構成協力<ref name = "Director of Perfect Blue"/>。在原創動畫錄影帶《JoJo的奇妙冒險 第三部 星塵鬥士》的《DIO的世界 花京院 結界的死鬥》中，擔任演出的今敏則與後來成為《[[薔薇少女|薔薇少女]]》導演的同事[[松尾衡|松尾衡]]首次認識<ref>{{Cite web |url=http://clappa.jp/Special/79/1/ |script-title=ja:今 敏監督×松尾 衡監督　対談 |author= |date= |publisher=clappa！ |language=ja |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20071011221818/http://clappa.jp/Special/79/1/ |archivedate=2007年10月11日 }}</ref>。1994年時，今敏和押井守合作在[[德間書店|德間書店]]的漫畫雜誌《[[Animage|Animage]]》上發表《{{Link-ja|六翼天使2億6661萬3336之翼|セラフィム 2億6661万3336の翼}}》，由其負責漫畫繪製的部分，但這部作品最後並未完結<ref name = "Andrew Osmond 17">{{Cite book | author = Andrew Osmond | title = Satoshi Kon: The Illusionist | location = 美國柏克萊 | publisher = {{Link-en|石橋出版社|Stone Bridge Press}} | date = 2009年12月1日 | pages = 第17頁 | ISBN = 978-1933330747 | accessdate = 2015年9月29日 | url =  | language = en | quote =  }}</ref>。在刊載《六翼天使2億6661萬3336之翼》後，今敏逐漸將工作重心放在動畫製作上<ref name = "UNRELEASED COMIC">{{Cite web |url = http://konstone.s-kon.net/modules/works/index.php?content_id=5 |script-title = ja:UNRELEASED COMIC |author =  |date =  |publisher = {{lang|ja|今 敏 オフィシャル·サイト}} |language = ja |accessdate = 2015年9月29日 |quote =  }}</ref>。

1995年，今敏開始連載新的漫畫作品《[[OPUS_(漫畫)|OPUS]]》，但後來因為雜誌休刊而沒有完結。同樣在這一年，今敏在大友克洋擔任總監督、以三部曲方式製作的電影《[[MEMORIES|MEMORIES]]》中，擔任第一部短片《[[她的回憶|她的回憶]]》的動畫編劇、美術設定和構圖​​<ref name = "Konstone 公式プロフィール"/><ref name = "Justin Sevakis"/>。在《[[她的回憶|她的回憶]]》這部作品中，今敏第一次使用了幻想和現實間反覆變換作為主要的元素，這在他之後推出的作品中成為重要的表現風格，著重想像和現實相互融合的概念<ref>{{Cite book | author = [[大友克洋|大友克洋]] | title = 《[[MEMORIES|MEMORIES]]》 | location = 日本東京 | publisher = [[松竹|松竹]] | date = 1995年12月23日 | pages =  | ISBN =  | accessdate = 2015年9月29日 | url =  | language = ja | quote =  }}</ref>。在發表《她的回憶》後，今敏也開始準備執導下一部動畫電影的計畫，並投入製作工作。

=== 擔任導演 ===
[[File:Masao_Maruyama.jpg|thumb]]成為重要的工作夥伴。]]

1997年，今敏順利推出擔任動畫導演後的處女作《[[藍色恐懼|藍色恐懼]]》，動畫基於小說作家{{Link-en|竹內義和|Yoshikazu Takeuchi}}創作的同名小說改編而成<ref name = "Director of Perfect Blue"/><ref name="Bill Aguiar">{{Cite web |url=http://www.tokyopop.com/Robofish/tp_article/688419.html |title=Interview with Satoshi Kon |author=Bill Aguiar |date=2007年4月25日 |publisher=[[TOKYOPOP|TOKYOPOP]] |language=en |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20110414164311/http://www.tokyopop.com/Robofish/tp_article/688419.html |archivedate=2011年4月14日 }}</ref>。這部是[[MADHOUSE|MADHOUSE]]首次製作今敏執導的動畫作品<ref name = "MADHOUSE PEOPLE FILE"/>，後來受邀在[[柏林國際影展|柏林國際影展]]作為非競賽影片播放。《藍色恐懼》劇情以一名流行偶像為中心，描寫所發生的一連串懸疑故事。今敏最初不滿意竹內義和的原始劇本，並要求修改原作故事<ref name = "Tom Mes">{{Cite web |url = http://www.midnighteye.com/interviews/satoshi-kon/ |title = Satoshi Kon |author = Tom Mes |date = 2002年2月11日 |publisher = Midnight Eye |language = en |accessdate = 2015年9月29日 |quote =  }}</ref><ref name="Interview 07">{{Cite web |url=http://konstone.s-kon.net/modules/interview/index.php/content0008.html |script-title=ja:Interview 07 2004年6月 アメリカから、監督作品全般に関するインタビュー |author= |date=2004年6月 |publisher={{lang|ja|今 敏 オフィシャル·サイト}} |language=ja |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20120311213158/http://konstone.s-kon.net/modules/interview/index.php/content0008.html |archivedate=2012年3月11日 }}</ref>。最後在保证製作主軸須維持小說的偶像、恐怖和跟蹤狂三個要素後，今敏獲准改编脚本内容<ref name = "Tom Mes"/>。電影的劇本後來是今敏和編劇[[村井貞之|村井貞之]]共同改寫而成<ref name = "Director of Perfect Blue"/><ref name = "Tom Mes"/><ref name = "Interview 07"/>，在這之前村井貞之便以現實世界和想像世界間的模糊邊界為作品題材<ref name = "Tom Mes"/>。在發表《藍色恐懼》後，作為筒井康隆粉絲的今敏計劃下部電影製作，便是改編過去非常喜歡、筒井康隆在1993年發表的小說《{{Link-en|盜夢偵探|Paprika (novel)|盜夢偵探}}》。

但因為《藍色恐懼》的發行公司雷克斯娛樂（Rex Entertainment）破產，使得這個製作想法被迫拖延<ref name = "Jason Gray">{{Cite web |url = http://www.midnighteye.com/interviews/satoshi-kon-2/ |title = Satoshi Kon |author = Jason Gray |date = 2006年11月20日 |publisher = Midnight Eye |language = en |accessdate = 2015年9月29日 |quote =  }}</ref>；與此同時，些許失意的今敏仍希望下部作品的主題，繼續探索想像與現實的交界，因此轉向製作原創的動畫電影<ref name = "Tom Mes"/>。1999年他開始投入下部電影的創作，並在2000年時製作完成。不過一直到2002年，今敏的的第二部電影《[[千年女優|千年女優]]》才正式上映，還在同年出版了著作《KON'S TONE?「千年女優」之道》（{{lang|ja|KON’S TONE?「千年女優」への道}}）。《千年女優》的動畫劇本同樣是今敏和編劇村井貞之共同創作<ref name = "Tom Mes"/>，故事的主軸描寫一名具傳奇色彩的女演員選擇在事業生涯的最高巔峰時，突然神秘地向社會大眾宣布隱退。電影中除了保留虛幻和現實間的無縫連接外，同時還增添了許多[[視覺陷阱|視覺陷阱]]和時間感喪失等概念<ref name="DVJ 2.0">{{Cite web|url=http://www.dvdvisionjapan.com/actress2.html |title=Director Satoshi Kon Interview |author= |date= |publisher=DVJ 2.0 |language=en |accessdate=2015年9月29日 |quote= |deadurl=no |archiveurl=https://web.archive.org/web/20120308142511/http://www.dvdvisionjapan.com/actress2.html |archivedate=2012年3月8日 }}</ref>。

在《[[千年女優|千年女優]]》製作過程中，今敏還首次邀請了他十分喜歡的作曲家[[平澤進|平澤進]]參與音樂製作<ref name="Interview 23">{{Cite web|url=http://konstone.s-kon.net/modules/interview/index.php/content0025.html |script-title=ja:Interview 23 2007年6月 アメリカから『パプリカ』について |author= |date=2007年6月 |publisher={{lang|ja|今 敏 オフィシャル·サイト}} |language=ja |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20120311213205/http://konstone.s-kon.net/modules/interview/index.php/content0025.html |archivedate=2012年3月11日 }}</ref>；後來平澤進持續參與今敏的後續作品，成為今敏的固定音樂製作人<ref name = "Interview 23"/>。大體上《千年女優》和《藍色恐懼》同樣花費約1.2億日圓的低預算製作<ref name = "Emmanuel Pettini"/>，然而前者除了獲得[[好萊塢|好萊塢]]知名公司[[夢工廠|夢工廠]]破例買下電影版權，由其負責全球總代理外，《千年女優》在電影評價和財務收入上也比前作《藍色恐懼》更為成功。同時《千年女優》也獲得多項獎項，且因為與[[宮崎駿|宮崎駿]]的《[[神隱少女|神隱少女]]》共同獲得2001年度第5屆日本[[文化廳媒體藝術祭|文化廳媒體藝術祭]]的動畫大賞，而讓今敏的名聲更加著名<ref>{{Cite web |url=http://plaza.bunka.go.jp/english/festival/2001/animation/000372/ |title=Grand Prize Millenial Actress |author= |date= |publisher=[[文化廳媒體藝術祭|文化廳媒體藝術祭]] |language=en |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20071018082323/http://plaza.bunka.go.jp/english/festival/2001/animation/000372/ |archivedate=2007年10月18日 }}</ref>。

早在2001年，今敏便決定了下一部電影的製作內容；到了2003年，其第三部作品《[[東京教父|東京教父]]》正式上映。《東京教父》的電影劇本是今敏與{{Link-en|信本敬子|Keiko Nobumoto}}共同創作，故事主軸講述三名無家可歸的露宿者在聖誕節前夕，於東京街頭發現一個棄嬰，三人隨後為了棄嬰而出發尋找她的親生父母<ref>{{Cite web |url=http://www.sonypictures.jp/archive/movie/worldcinema/tgf/ |script-title=ja:東京ゴッドファーザーズ：オフィシャルサイト |author= |date=2003年 |publisher=[[索尼影視娛樂|索尼影視娛樂]] |language=ja |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20110720181414/http://www.sonypictures.jp/archive/movie/worldcinema/tgf/ |archivedate=2011年7月20日 }}</ref>。電影將焦點集中在無家可歸的露宿者、被拋棄的嬰兒等社會題材，並用喜劇的方式聯繫各主題的故事發展<ref name = "Justin Sevakis"/><ref name = "Tom Mes"/>。《東京教父》花費的製作成本相較於前兩部電影還要來得更多，總計花費約3億日圓的預算<ref name = "Emmanuel Pettini"/>。《東京教父》後來和《千年女優》同時獲得2003年[[第75屆奧斯卡金像獎|第75屆奧斯卡金像獎]]的[[奧斯卡最佳動畫片獎|奧斯卡最佳動畫片獎]]第一次入圍，但並沒有獲選至最終入圍名單。

=== 獲得歡迎 ===
《東京教父》於2003年正式上映後不久，今敏在同年投入自己首部電視動畫的製作上。2004年，《東京教父》的[[數碼多功能影音光碟|數碼多功能影音光碟]]發售後不久，由今敏初次執導、共計13集的動畫作品《[[妄想代理人|妄想代理人]]》正式在電視上播出。今敏在《妄想代理人》的故事內容中，重新強調想像與現實間的交互關係，並加入有關工作等種種社會性題材的描寫<ref>{{Cite web |url=http://www.gamestar.com/11_04/pause/pause_disc_satoshikon.shtml |title=Interview with Satoshi Kon |author= |date= |publisher=Gamestar |language=en |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://archive.is/20110711071742/http://www.gamestar.com/11_04/pause/pause_disc_satoshikon.shtml |archivedate=2011年7月11日 }}</ref>。這些故事的來源許多是今敏過去覺得故事和設定不錯，但因沒有適合表現的時機而未採納的電影構想，經長期擱置後將這些想法重新整理，並納入電視動畫製作中<ref name="妄想の二">{{Cite web |url=http://konstone.s-kon.net/modules/moso/index.php/content0005.html |script-title=ja:妄想の二「総監督の謎」 |author= |date= |publisher={{lang|ja|今 敏 オフィシャル·サイト}} |language=ja |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20120311213225/http://konstone.s-kon.net/modules/moso/index.php/content0005.html |archivedate=2012年3月11日 }}</ref>。而在經過數年的籌畫和長期的對外宣傳，今敏在2006年推出改編自筒井康隆同名作品的《[[盜夢偵探|盜夢偵探]]》，這也實現了今敏長年以來的構想。

《[[盜夢偵探|盜夢偵探]]》的電影製作獲得原作者筒井康隆的支持<ref name = "北海道新聞"/>，故事中心圍繞在一種新形式的心理治療，能透過儀器分析患者的夢境，進一步治療精神疾病患者。《盜夢偵探》很快便被視為今敏非常成功的電影作品，認為這部作品成功將幻想與現實相互融合的構想表現至極致，也因而獲得多項電影獎項和參加多次影展。這包括獲選成為2006年角逐[[金獅獎|金獅獎]]的[[第63屆威尼斯國際電影節|第63屆威尼斯國際電影節]]影展競賽片之一，這也是該屆唯一一部入選競賽影片的動畫作品；以及入圍[[第80屆奧斯卡金像獎|第80屆奧斯卡金像獎]]的奧斯卡最佳動畫片獎第一次入圍，但並沒有獲選至最終入圍名單。《盜夢偵探》的故事內容引起多方的討論和解釋，今敏在總結這部電影時表示「一切但根本的故事已經改變」<ref>{{Cite web |url = http://www.premiere.fr/Bandes-annonces/Video/Satoshi-Kon-ITW |title = Satoshi Kon -ITW : bandes-annonces |author =  |date = 2006年12月6日 |publisher = 《[[首映_(雜誌)|首映]]》 |language = fr |accessdate = 2015年9月29日 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20100828225907/http://www.premiere.fr/Bandes-annonces/Video/Satoshi-Kon-ITW |archivedate = 2010年8月28日 }}</ref>；同時該電影也類似今敏過去的構想，在作品中同樣著重夢境與現實間的互動與影響<ref name = "Emmanuel Pettini"/>。

在《盜夢偵探》發表後，今敏於2007年和押井守、[[新海誠|新海誠]]等多位著名的日本導演，共同參與[[日本放送協會|日本放送協會]]提出的《[[Ani*Kuri15|Ani*Kuri15]]》製作計畫。《Ani*Kuri15》計畫分成三季播出，並在2007年開始在日本放送協會的節目間插入1分鐘的動畫。今敏便執導當中一個單元，並藉此機會發表1分鐘長的短篇作品《早安》（{{lang|ja|オハヨウ}}）<ref>{{Cite web |url = http://www.animenewsnetwork.com/news/2007-05-01/nhk-ani-kuri-15 |title = NHK Launches Ani-Kuri 15 |author = Saiani |date = 2007年5月1日 |publisher = [[動畫新聞網|動畫新聞網]] |language = en |accessdate = 2015年9月29日 |quote =  }}</ref>。就在同一年，今敏還協助成立{{Link-en|日本動畫從業人員協會|Japanese Animation Creators Association}}，並成為該協會的會員<ref>{{Cite web |url = http://www.animenewsnetwork.com/news/2008-06-06/japan-animator-labor-group-legally-incorporates |title = Japan's Animator Labor Group Legally Incorporates |author = 《[[每日新聞|每日新聞]]》 |date = 2008年6月6日 |publisher = [[動畫新聞網|動畫新聞網]] |language = en |accessdate = 2015年9月29日 |quote =  }}</ref>。而在短篇動畫《早安》推出後不久，今敏自2009年著手製作他的下一部動畫電影《[[夢想機械|夢想機械]]》，這也是他首部為孩童製作、內容較為童趣的動畫。

== 逝世 ==
=== 罹癌之後 ===
{{Quote box| width = 300px | align = left | quoted = true | bgcolor = #FFFFF0 | salign = right | quote = 我要懷著對世上所有美好事物的謝意，放下我的筆了，先走一步了。 | source= —今敏<ref>{{Cite web |url = http://konstone.s-kon.net/modules/notebook/archives/565 |script-title = ja:さようなら |author = 今敏 |date = 2010年8月25日 |publisher = {{lang|ja|今 敏 オフィシャル·サイト}} |language = ja |accessdate = 2015年9月29日 |quote = {{lang|ja|世界中に存する善きものすべてに感謝したい気持ちと共に、筆をおくことにしよう。じゃ、お先に。}} }}</ref> }}

2010年，今敏開始出現身體不適，同年5月確診罹患晚期[[胰臟癌|胰臟癌]]<ref name = "さようなら">{{Cite web |url = http://konstone.s-kon.net/modules/notebook/archives/565 |script-title = ja:さようなら |author = 今敏 |date = 2010年8月25日 |publisher = {{lang|ja|今 敏 オフィシャル·サイト}} |language = ja |accessdate = 2015年9月29日 |quote =  }}</ref>。今敏認為這可能是工作期間長期仰賴[[酒精|酒精]]、[[咖啡因|咖啡因]]、[[尼古丁|尼古丁]]而工作所造成的結果，考慮醫生診察後告知僅可能存活半年的時間，今敏選擇在自己住家中度過剩餘的人生，並繼續從事更新官方部落格等活動。至此之後，除在部落格上敘述日常生活，他還開始整理身旁物品。不過癌症在確診後數月便迅速惡化<ref name="passaway">{{Cite web |url = http://www.animenewsnetwork.com/news/2010-08-24/award-winning-director-satoshi-kon-passes-away |title = Perfect Blue/Paprika Director Satoshi Kon Passes Away (Updated) |author = 《[[東京新聞|東京新聞]]》 |date = 2010年8月24日 |publisher = [[動畫新聞網|動畫新聞網]] |language = en |accessdate = 2015年9月29日 |quote =  }}</ref>，對於自己的病情，今敏還是積極面對。不過他不想讓社會大眾或新聞媒體知道自己的病情發展，部分原因在於對於自己的身體逐漸消瘦、和死後的遺體可能被討論而感到尷尬。

其中他僅在7月23日表示自己已經剃了光頭<ref>{{Cite web|url=http://konstone.s-kon.net/modules/notebook/archives/519 |script-title=ja:坊主頭にする |author=今敏 |date=2010年7月23日 |publisher={{lang|ja|今 敏 オフィシャル·サイト}} |language=ja |accessdate=2016年6月25日 |quote= |deadurl=no |archiveurl=https://web.archive.org/web/20110105211717/http://konstone.s-kon.net/modules/notebook/archives/519 |archivedate=2011年1月5日 }}</ref>，並在8月6日時說已經在整理周邊的東西，陸續將其處理或送給他人<ref>{{Cite web|url=http://konstone.s-kon.net/modules/notebook/archives/538 |script-title=ja:捨て神様 |author=今敏 |date=2010年8月6日 |publisher={{lang|ja|今 敏 オフィシャル·サイト}} |language=ja |accessdate=2016年6月25日 |quote= |deadurl=no |archiveurl=https://web.archive.org/web/20110105081928/http://konstone.s-kon.net/modules/notebook/archives/538 |archivedate=2011年1月5日 }}</ref>。在8月17日和8月18日，今敏則在官方網站上發布《夢想機械》的劇組人員向電影愛好者推薦的100部佳片名單<ref>{{Cite web|url=http://konstone.s-kon.net/modules/notebook/archives/551 |script-title=ja:「夢みる機械」班が選ぶ映画100・前編 |author=今敏 |date=2010年8月17日 |publisher={{lang|ja|今 敏 オフィシャル·サイト}} |language=ja |accessdate=2016年6月25日 |quote= |deadurl=no |archiveurl=https://web.archive.org/web/20110104154343/http://konstone.s-kon.net/modules/notebook/archives/551 |archivedate=2011年1月4日 }}</ref><ref>{{Cite web|url=http://konstone.s-kon.net/modules/notebook/archives/558 |script-title=ja:「夢みる機械」班が選ぶ映画100・後編 |author=今敏 |date=2010年8月18日 |publisher={{lang|ja|今 敏 オフィシャル·サイト}} |language=ja |accessdate=2016年6月25日 |quote= |deadurl=no |archiveurl=https://web.archive.org/web/20110104154053/http://konstone.s-kon.net/modules/notebook/archives/558 |archivedate=2011年1月4日 }}</ref>。而在8月20日，他還在個人[[Twitter|Twitter]]上更新消息，稱讚《夢想機械》故事中的兩位主角都非常可愛<ref>{{Cite web |url = https://twitter.com/konsatoshi/status/21725864891 |script-title = ja:「夢みる機械」のラッシュをいくつか見る。ロビンもリリコも可愛い。ウフ。
 |author = 今敏 |date = 2010年8月20日 |publisher = [[Twitter|Twitter]] |language = ja |accessdate = 2016年6月25日 |quote =  }}</ref>。到了8月23日，今敏在自己的官方部落格上更新日記，並補充之前的電影名單，但仍沒有提到自己身體不舒服或生病的字樣<ref>{{Cite web|url=http://konstone.s-kon.net/modules/notebook/archives/562 |script-title=親切なお節介 |author=今敏 |date=2010年8月23日 |publisher={{lang|ja|今 敏 オフィシャル·サイト}} |language=ja |accessdate=2016年6月25日 |quote= |deadurl=no |archiveurl=https://web.archive.org/web/20110104170928/http://konstone.s-kon.net/modules/notebook/archives/562 |archivedate=2011年1月4日 }}</ref>。

2010年8月24日凌晨6時20分，今敏因為罹患胰臟癌而病逝，享年46歲<ref name="passaway"/>。由於病情惡化的相關消息不曾對外公布，罹患癌症的今敏也常發表近期活動的公開消息，這使得他逝世的消息引起不少人震撼和意外，甚至一度誤傳是因為[[肺癌|肺癌]]逝世。在他逝世後不久，他的家人在隔天下午4時，於今敏的個人部落格上發表生前親自撰寫、題為《再見》的遺書，向長期關注自己和所執導電影的粉絲告知已經病逝的消息；在數千字的文章中提到自己的病情、和家人與朋友的關係以及動畫創作等，並確認死因是胰腺癌<ref name = "さようなら"/>。

=== 後續發展 ===
最快發表逝世消息者是動畫公司[[GAINAX|GAINAX]]的工作人員{{Link-en|武田康廣|Yasuhiro Takeda}}，他在自己的Twitter上表示：「得知今敏先生去世的消息，年僅47歲。今後再也無法和他在吉祥寺的小酒吧相見了，感到十分傷心<ref>{{Cite web |url = https://twitter.com/takedayasuhiro/status/22018243608 |script-title = ja:今敏さんの訃報をさっきmixiで読んだ。４７歳。もう吉祥寺の飲み屋で会う事がないのか。悲しすぎる。 |author = 武田康廣 |date = 2010年8月24日 |publisher = [[Twitter|Twitter]] |language = ja |accessdate = 2016年6月25日 |quote =  }}</ref>。」與他合作多年的[[丸山正雄|丸山正雄]]也說：「如此了不起的導演就這麼去世了……<ref>{{Cite web |url = https://twitter.com/MasaoMaruyama/status/21985724413 |script-title = ja:すいません!すいません、夕方ということで時間が決まってましたっけ？いまマッドの大事な監督が急逝して… |author = 丸山正雄 |date = 2010年8月24日 |publisher = [[Twitter|Twitter]] |language = ja |accessdate = 2016年6月25日 |quote =  }}</ref>」。今敏逝世後不久，[[Animax|Animax]]和{{Link-en|家庭劇場|Family Gekijo}}為了紀念他的貢獻，接連播出今敏曾參與製作的動畫作品。相關人士也舉辦了「今敏導演告別會」（{{lang|ja|今 敏監督を送る会}}），包括{{Link-en|鈴木慶一|Keiichi Suzuki (composer)}}、平澤進等曾與今敏合作者都出席告別會。在2010年下半年，今敏被《[[時代_(雜誌)|時代]]》選為該年度十位逝世的全球偉大人物之一<ref>{{Cite web |url = http://content.time.com/time/specials/packages/article/0,28804,2036683_2036477_2036995,00.html |title = Satoshi Kon |author =  |date = 2010年9月13日 |publisher = 《[[時代_(雜誌)|時代]]》 |language = en |accessdate = 2015年9月29日 |quote =  }}</ref>。電影《[[黑天鵝_(電影)|黑天鵝]]》導演[[戴倫·艾洛諾夫斯基|戴倫·艾洛諾夫斯基]]還為今敏寫下{{Link-en|贊詞|Eulogy}}以茲紀念，後來贊詞印在回顧今敏的紀念書籍《今敏動畫全工作》（{{Lang|ja|今敏アニメ全仕事}}）<ref>{{Cite web |url = http://www.animenewsnetwork.com/news/2011-05-26/satoshi-kon-book-adds-message-by-black-swan-aronofsky |title = Satoshi Kon Book Adds Message by Black Swan's Aronofsky (Updated) |author = 《Cinema Today》（{{lang|ja|シネマトゥデイ}}） |date = 2011年5月26日 |publisher = [[動畫新聞網|動畫新聞網]] |language = en |accessdate = 2015年9月29日 |quote =  }}</ref>。今敏逝世前主要投入在動畫電影《[[夢想機械|夢想機械]]》製作上，這部作品也成為其生前的遺作。

由於《{{Link-en|機動警察劇場版3|WXIII: Patlabor the Movie 3}}》的導演[[遠藤卓司_(動畫導演)|遠藤卓司]]在2009年7月也因病逝世，接連動畫製作團隊裡的兩位重要導演逝世，讓MADHOUSE公司內部一度感到不安。2010年11月12日，MADHOUSE宣布將重啟今敏生前未完成、已進入後期製作的《夢想機械》計劃，並由曾多次在今敏作品中負責角色設定和作畫監督的[[板津匤覧|板津匤覧]]擔任代理導演投入作品的製作工作，但並未公布電影正式的上映日期<ref>{{Cite web |url = http://www.madhouse.co.jp/news/news_2010_11.html |script-title = ja:2010年11月のニュース |author =  |date = 2010年11月29日 |publisher = [[MADHOUSE|MADHOUSE]] |language = ja |accessdate = 2015年9月29日 |quote =  }}</ref>。不過在2012年舉辦的{{Link-en|OTAKON 動漫展|Otakon}}中，MADHOUSE創始人丸山正雄表示：「不幸的是，我們還沒有足夠的錢。我個人的目標是他逝世後，在5年內完成這部作品，我仍然努力實現這個目標。<ref>{{Cite web |url = http://www.animenewsnetwork.com/convention/2012/otakon/11 |title = Masao Maruyama/MAPPA Q&A |author = Justin Sevakis |date = 2012年7月28日 |publisher = [[動畫新聞網|動畫新聞網]] |language = en |accessdate = 2015年9月29日 |quote =  }}</ref>」截至2013年為止，《夢想機械》仍無法確定能否完成，主要原因是製作方在募資層面遭遇困難，在1,500個畫面中只有600個分鏡已經製成動畫。

製作人丸山正雄在2016年中的訪談<ref>https://akiba-souken.com/article/27618</ref>提到：「我在今敏過世後的四到五年間，曾一心堅持著到處找人來接手、完成他的遺作。當時腳本與分鏡都早已完成，連原畫都做好一部分了。但，就算有人真能模仿今敏的風格，做出來的也不過就是仿作而已，沒辦法有今敏的味道。我舉個例子，若讓細田守來接下導演，我相信成品也一定會很好看。但這樣的電影，就會是細田守的了，並不是今敏的。做夢機器只能是今敏的作品，我們不該只是為了完成作品，而在各種地方進行妥協。與其這麼做，不如放棄完成它…我花了好多年，才得到這個苦澀的答案。比起硬是要完成這份遺作，我更傾向讓人來接手今敏的原作構想，將其改編成動畫電影。這樣的話，完成的電影就會100%是那個導演的作品了。我願意幫忙完成這樣的計劃，除此之外也想拍部今敏的紀錄片。」

據此可知，目前做夢機器的企劃正在變更製作方向。

== 風格 ==
=== 動畫製作 ===
[[File:今敏作品.jpg|thumb]]

今敏在發想《[[千年女優|千年女優]]》、《[[東京教父|東京教父]]》、《[[妄想代理人|妄想代理人]]》等原創作品時，會先從故事表現開始思考<ref>{{Cite web |url = http://twitchfilm.com/2008/07/exclusive-interview-with-satoshi-kon.html |title = Exclusive Interview with Satoshi Kon |author = Simon Abrams |date = 2007年7月16日 |publisher = {{Link-en|Twitch Film|Twitch Film}} |language = en |accessdate = 2015年9月29日 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20151001052500/http://twitchfilm.com/2008/07/exclusive-interview-with-satoshi-kon.html |archivedate = 2015年10月1日 }}</ref>，判斷內容是否適合採取電影呈現的方式，而不會直接將電影視為敘述故事的手法<ref name = "Aimee Giron"/>。而如果是《盜夢偵探》等有可供參考的作品時，今敏仍然不會完全忠實於故事的發展，而會另外加入自己的詮釋。今敏經常和村井貞之、信本敬子、[[水上清資|水上清資]]等編劇共同推敲故事的走向<ref name = "Aimee Giron"/>，然後向負責動畫製作的MADHOUSE公司提交計畫案<ref name = "Aimee Giron"/>，一旦獲得通過後便開始創造各式的作品設定<ref name = "Aimee Giron"/>。不過在《妄想代理人》中，擔任作畫監督和角色設定的[[安藤雅司|安藤雅司]]扮演著非常重要的角色<ref>{{Cite web |url = http://archive.filmdeculte.com/entretien/satoshikon.php |title = FILMDECULTE : CONFERENCE DE PRESSE SATOSHI KON |author = Danielle Chou |date = 2012年7月28日 |publisher = FilmDeCulte |language = fr |accessdate = 2015年9月29日 |quote =  }}</ref>。原因在於依照原本製作的流程完成故事架構後，透過安藤雅司的居中協調，可以讓其他編劇在個別話數中保留一些額外的插曲<ref name = "妄想の二"/>。

當作品的構想完成後，製作流程便進入逐步繪製動畫[[分鏡|分鏡]]的階段，這個過程需要1年半至2年的時間<ref name = "Aimee Giron"/>。許多MADHOUSE主要的工作人員都長期參與今敏執導的作品製作，包括美術總監[[池信孝|池信孝]]、音效指導[[三間雅文|三間雅文]]以及音樂製作人[[平澤進|平澤進]]等人。整個製作成本大約是幾億日圓，然而相較於常見日本動畫的製作預算來說，這樣的成本花費仍然是偏少<ref name = "Emmanuel Pettini"/>。對於動畫成本問題，今敏自己曾表示過：「在低預算的情況下製作出高品質的作品，實際上是工作人員的禮饋。<ref name = "DVJ 2.0"/><ref>{{Cite web |url=http://konstone.s-kon.net/modules/interview/index.php/content0003.html |script-title=ja:Interview 02 2002年12月 イタリアから、主に「千年女優」に関するインタビュー |author= |date=2002年12月 |publisher={{lang|ja|今 敏 オフィシャル·サイト}} |language=ja |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20090424071117/http://konstone.s-kon.net/modules/interview/index.php/content0003.html |archivedate=2009年4月24日 }}</ref>」

=== 類型影響 ===
今敏在接受訪問時表示自己文學方面，受到科幻小說作家[[菲利普·狄克|菲利普·狄克]]的許多影響<ref name = "Bill Aguiar"/>，不過對於[[筒井康隆|筒井康隆]]的作品有著深刻印象<ref name = "Emmanuel Pettini"/><ref name = "Bill Aguiar"/>。他在高級中學畢業前，亦看過許多漫畫和動畫，並深深影響後續的創作<ref name = "Director of Perfect Blue"/>。他特別喜歡1974年的《宇宙戰艦大和號》、1978年的《未來少年柯南》和《[[銀河鐵道999|銀河鐵道999]]》、以及1979年的《機動戰士鋼彈》<ref name = "Aimee Giron"/><ref name="Interview 05">{{Cite web |url=http://konstone.s-kon.net/modules/interview/index.php/content0006.html |script-title=ja:Interview 05 1998年2月 アメリカから「パーフェクトブルー」に関するインタビュー |author= |date=1998年2月 |publisher={{lang|ja|今 敏 オフィシャル·サイト}} |language=ja |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20120311213049/http://konstone.s-kon.net/modules/interview/index.php/content0006.html |archivedate=2012年3月11日 }}</ref>，另外大友克洋創作的漫畫作品《童夢》也給他許多靈感<ref name = "Aimee Giron"/><ref name = "Interview 05"/>。歐美國家拍攝的電影亦對今敏有著種種影響，其中讓他獲得啟發的重要作品有[[喬治·羅伊·希爾|喬治·羅伊·希爾]]在1972年執導的《{{Link-en|第五屠場|Slaughterhouse-Five (film)}}》，他形容這部電影帶給他許多震撼，另外他也喜歡{{Link-en|馬克·卡羅|Marc Caro}}和[[尚-皮爾·桑里|-{zh-hk:尚-彼亞·桑里; zh-tw:尚-皮耶·居內; zh-cn:让-皮埃尔·热内;}-]]在1995年執導的《{{Link-en|驚異狂想曲|The City of Lost Children}}》<ref name="Bill Aguiar" />。電影導演[[泰瑞·吉連|泰瑞·吉連]]的作品也深深影響著今敏的創作<ref name="Director of Perfect Blue" />，他特別喜歡吉連在1981年執導的《[[時光大盜|時光大盜]]》、1985年執導的《[[妙想天开|-{zh-hk:妙想天開; zh-tw:巴西; zh-hans:妙想天开;}-]]》和1988年執導的《{{Link-en|終極天將|The Adventures of Baron Munchausen}}》<ref name = "MADHOUSE PEOPLE FILE"/><ref name = "Director of Perfect Blue"/><ref name = "Bill Aguiar"/>。除了泰瑞·吉連外，今敏也在部落格上提到曾觀賞過[[蒙提·派森|蒙提·派森]]的表演<ref>{{Cite web |url = http://konstone.s-kon.net/modules/notebook/archives/396 |script-title = ja:先月の傾向 |author =  |date = 2009年3月3日 |publisher = {{lang|ja|今 敏 オフィシャル·サイト}} |language = ja |accessdate = 2015年9月29日 |quote =  }}</ref>。

同時他也受到一部分日本電影的影響，今敏便非常喜歡電影導演[[黑澤明|黑澤明]]的作品，甚至在製作《盜夢偵探》時故意安排和黑澤明[[友情客串|相像]]的角色<ref name = "Jason Gray"/>；儘管他曾表示並不常接觸一般的日本電影<ref name = "DVJ 2.0"/><ref name="Interview 03">{{Cite web|url=http://konstone.s-kon.net/modules/interview/index.php/content0004.html |script-title=ja:Interview 03 2002年12月　カナダから、主に「千年女優」に関するインタビュー |author= |date=2002年12月 |publisher={{lang|ja|今 敏 オフィシャル·サイト}} |language=ja |accessdate=2015年9月29日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20120311213055/http://konstone.s-kon.net/modules/interview/index.php/content0004.html |archivedate=2012年3月11日 }}</ref>，但在接受採訪時強調自己的作品是「日本的」作品<ref name = "DVJ 2.0"/><ref name = "Interview 03"/>。今敏認為自己基本上受到以[[美國電影|美國電影]]為首的西方電影影響，希望參考這些結構性強、故事簡潔且容易理解的電影，進一步形成自己的電影風格。不過雖然深受美國電影影響，他認為自己的電影風格卻出乎意料地具有東方世界的味道<ref name = "DVJ 2.0"/>。今敏還是自己電影的音樂製作人平澤進的熱衷粉絲，這些音樂作品也對他的電影創作帶來許多影響。他曾公開表示自己除了收藏電影原聲帶外，也曾前往觀賞平澤進的現場表演，平常家中播放的背景音樂也都是平澤進的曲子。而在今敏出殯時，便是播放了平澤進為《千年女優》創作的主題曲《{{Link-en|Philosopher's Propeller|Philosopher's Propeller|旋轉}}》（{{lang|ja|ロタティオン}}）<ref>{{Cite web |url = https://twitter.com/hirasawa/status/22186794536 |script-title = ja:本日も告別式ではずっとヒラサワ曲が流れ、出棺は「ロタティオン」だったそうだ。 |author = [[平澤進|平澤進]] |date = 2010年8月26日 |publisher = [[Twitter|Twitter]] |language = ja |accessdate = 2015年9月29日 |quote =  }}</ref>。

=== 作品分析 ===
當被問到對於女性角色的興趣時，今敏指出女性角色更容易被描寫，因為他不知道男性角色如何以同樣方式呈現，並認為「這可以投射我迷戀上的角色，並擴展我想要形容的部分」<ref name = "Steven T. Brown 23-43">{{Cite book | author = Steven T. Brown | title = Cinema Anime | location = 英國倫敦 | publisher = [[帕爾格雷夫·麥克米蘭|帕爾格雷夫·麥克米蘭]] | date = 2008年9月15日 | pages = 第23頁至第43頁 | ISBN = 978-0230606210 | accessdate = 2015年9月29日 | url = https://books.google.com.tw/books?id=EV_HAAAAQBAJ&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false | language = en | quote =  }}</ref>。{{Link-en|蘇珊·納皮爾|Susan J. Napier}}則是參考《東京教父》的故事框架後，指出有關{{Link-en|男性凝視|Male gaze}}的概念是需要更深入探討的議題，而她認為今敏的作品風格中存在著明顯的共同性。納皮爾表示今敏在電影凝視部分的改變，從原本在《她的回憶》和《藍色恐懼》中使用的限制性和消極性的凝視，在《千年女優》中轉變成為共同審視的觀點，到了《東京教父》則是朝向沉浸於不確定性和幻想的新角度進行<ref name = "Steven T. Brown 23-43"/>。

對於今敏的作品表現，[[迪恩·德布洛斯|迪恩·德布洛斯]]表示：「今敏透過手工繪製探索社會汙點和人類心靈，試圖探索我們自身的複雜性，而這可能無法透過真人現場表演的方式呈現。即便像是在噩夢般，大部分仍然是堅韌不拔、激烈的、和變動的樣貌。今敏在他的作品並沒有迴避成熟的題材或真人的情感，他的電影總是會處在『卡通』和世界之間的迷人中間地帶，這也是我們已經知道的特點。<ref>{{Cite web |url = http://articles.latimes.com/2010/aug/26/local/la-me-satoshi-kon-20100826 |title = Satoshi Kon dies at 46; Japanese anime director |author = Charles Solomon |date = 2010年8月26日 |publisher = 《[[洛杉磯時報|洛杉磯時報]]》 |language = en |accessdate = 2015年9月29日 |quote =  }}</ref>」
