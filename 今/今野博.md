'''今野博'''（{{bd|1919年||1944年|}}），曾用化名'''中野博'''、'''金野博'''，日本[[秋田县|秋田县]]人。1939年春入伍，隶属[[侵华日军|侵华日军]][[第32师团_(日本陆军)|第32师团]]，1939年8月在山东[[梁山|梁山]]附近的独山庄被[[八路军第115师|八路军第115师]]俘虏，1941年7月参加[[八路军|八路军]]，后任反战同盟鲁中支部支部长，因遭遇日伪军[[掃蕩|扫荡]]被俘，1944年被秘密杀害于济南。

== 生平 ==
1919年，今野博出生于[[日本国|日本国]][[本州岛|本州岛]]东北部的[[秋田县|秋田县]]，中小学均在本县学习，中学毕业于秋田县实业学校。<ref name=jingqing/>1939年春应征入伍，编入[[侵华日军|侵华日军]][[第32师团_(日本陆军)|第32师团]]长田大队，其大队长为[[长田敏江|长田敏江]]步兵少佐。1939年5月，今野博随长田大队从[[青岛|青岛]]登陆，驻[[山东省|山东省]][[汶上县|汶上县]]。8月2日，{{efn|一说8月3日。<ref name=bei>金野博同志生平简介. 抗日山烈士陵园</ref>}}在[[梁山歼灭战|梁山歼灭战]]独山庄战斗中，今野博等14名日军被[[八路军第115师|八路军第115师]]俘虏。9月13日夜，今野博伙同其他战俘逃跑，再次被俘。被俘后，今野博曾企图自杀，但被因被及时发现未遂。<ref name=qlwb>{{cite web |author1=邓撰相 |title=今野博,为抗日献身的日本八路 |url=http://epaper.qlwb.com.cn/qlwb/content/20150416/ArticelA21002FM.htm |website=齐鲁晚报 |access-date=2018-08-04 |archive-url=https://web.archive.org/web/20180804153321/http://epaper.qlwb.com.cn/qlwb/content/20150416/ArticelA21002FM.htm |archive-date=2018-08-04 |dead-url=no }}</ref>

在八路军指战员和反战日本人的耐心教育下，今野博逐渐认识到[[日本侵华战争|日本侵华战争]]给中日两国人民带来的严重灾难，思想上变得[[反战主义|反战]]。1941年6月2日，今野博与其他几位日本人发起成立了“[[在华日人反战同盟|在华日人反战同盟]]山东支部”，今野博任宣传委员。7月7日，<ref name=bei/><ref>{{cite web |title=曾在连云港战斗过的11位烈士入选第二批抗日英烈名录 |url=http://www.lyg01.net/news/lygxw/2015/0825/23998.shtml |website=连网 |access-date=2018-08-04 |archive-url=https://web.archive.org/web/20151013094352/http://www.lyg01.net/news/lygxw/2015/0825/23998.shtml |archive-date=2015-10-13 |dead-url=no }}</ref>{{efn|一说7月1日。<ref name=qlwb/>}}今野博与另外2名日本人在[[八路军山东纵队|八路军山东纵队]]驻地宣誓参加[[八路军|八路军]]。不久后，到[[日本工农学校|日本工农学校]]山东分校学习中文、政治常识与时事等。结业后，化名中野博到[[中共鲁中区委|鲁中区]]从事反战活动，1943年初任反战同盟鲁中支部支部长。<ref name=bei/><ref name=qlwb/>

1943年7月，今野博被[[滨海军区|滨海军区]]派遣到[[日照|日照]]，随滨海军区第四武工队（队长[[于镜清|于镜清]]）在日照城南一带日军据点做反战工作。为便于工作，于镜清让今野博改名金野博，因此今野博在武工队常被喊作金部长或老金。1944年7月，今野博由武工队护送回根据地向“在华日人反战同盟山东支部”汇报工作，夜宿[[高兴镇_(日照市)|高兴区]]汪家庄子（今日照市岚山区高兴镇汪家庄子村）。次日与到当地扫荡的日伪军遭遇，在突围时被俘。武工队多次营救无效，今野博被押到日照城，后转到日军驻济南宪兵队部，后被秘密杀害于[[济南|济南]]。<ref name=qlwb/>

== 纪念 ==
1944年8月25日，中共山东分局、八路军山东军区在江苏省[[赣榆县|赣榆县]]抗日山召开追悼抗日阵亡将士大会。在华日人反战同盟山东支部编演了歌颂今野博的[[话剧|话剧]]《我们在一起》。<ref name=qlwb/>

1983年5月，[[中共赣榆县委|中共赣榆县委]]编纂出版《抗日山志》，序言中[[开国上将|开国上将]][[萧华_(上将)|肖华]]评价今野博称“他与中国人民的胜利同在，与中国的高山流水同在。”<ref>{{cite web |title=金野博 |url=http://www.rzsq.gov.cn/menu_show3_2.php?menu_cont_id=729&menu_id=7&up_id= |website=日照市情网}}</ref><ref>{{cite book |editor=中共赣榆县委《抗日山志》编辑委员会 |title=抗日山志 |date=1983-05 |publisher=内部发行}}</ref>

1991年4月，赣榆县人民政府在苏鲁交界的[[抗日山烈士陵园|抗日山烈士陵园]]修建一座手榴弹形的“日本国友人金野博同志纪念碑”，碑右另立一白色[[大理石|大理石]]卧碑，碑文为《金野博同志生平简介》。<ref name=jingqing>{{cite journal |author1=镜清, 金科, 孟明 |title=日本国友人金野博 |journal=党史博采(纪实) |date=1994 |issue=4 |pages=43-46}}</ref>

[[中华人民共和国民政部|中华人民共和国民政部]]于2015年8月24日公布第二批《[[著名抗日英烈和英雄群体名录|著名抗日英烈和英雄群体名录]]》，今野博的名字被收录其中。<ref>{{cite web|url=http://politics.people.com.cn/n/2015/0825/c1001-27512283.html|title=民政部公布第二批600名著名抗日英烈和英雄群体名录|date=2015-08-24|publisher=中華人民共和國民政部|access-date=2016-07-21|archive-url=https://web.archive.org/web/20151017224402/http://politics.people.com.cn/n/2015/0825/c1001-27512283.html|archive-date=2015-10-17|dead-url=no}}</ref>
