'''桂林东站'''为[[桂海铁路|桂海铁路]]车站，位于[[中国|中国]][[广西壮族自治区|广西壮族自治区]][[桂林市|桂林市]][[七星区|七星区]]东站路，桂海铁路12.03千米处，距[[桂林南站|桂林南站]]8.4千米，距[[大圩站|大圩站]]9.16千米。该站只承担货运，不承担客运。

== 历史 ==
1974年2月，为开发桂林市东部经济，广西壮族自治区投资60万元，开设桂林东站，设有2股道和卸煤线等。<ref>[http://www.gxdqw.com/bin/mse.exe?seachword=%20&K=b&A=12&rec=643&run=13 桂林市志·交通志·第三章 铁路·第四节 地方铁路]</ref>

== 现状 ==
目前火车东站设有5股道，2009年之后由[[南宁铁路局|南宁铁路局]]管辖。2014年桂林东站停办货运业务，改作为“'''东旺物流园'''”使用，但是站内的股道并未被撤除。目前路线状况不佳，铁路两侧杂草相掩，轨道锈蚀情况较严重<ref name="gxsina">{{Cite web |url=http://gx.sina.com.cn/news/gx/2017-09-22/detail-ifymfcih2358262.shtml |title=桂林桂海线将开发小火车观光旅游 串联漓东精华山水 |accessdate=2017-11-07 |publisher=新浪广西 |date=2017-09-22 }}</ref>。

桂林东站至大圩镇东岸菜园村铁路尽头的线路实际已废弃。

== 机车 ==
在归属广西地方铁路管辖期间，桂林东站先后配属[[建设型蒸汽机车|建设型蒸汽机车]]和[[东风型柴油机车|东风型柴油机车]]各两台以及一台[[GKD1型柴油机车|GKD1型柴油机车]]，以之作为自备机车<ref name="railography">[http://www.railography.co.uk/info/cn_steam/lists/locations/gx_guilin_local.htm Railography : Chinese Locomotive Lists : Guilin Nan - Guilin Dong - Dayu Local Railway]</ref>。其中一台标记“'''桂东'''”字样的东风型柴油机车废弃在桂林东站直至2017年10月被南宁铁路局出售<ref name="crpn">{{Cite web |url=http://www.crpn.cn/news.php?id=54410 |title=南宁铁路局2017年9月22日报废机车网上公开竞价销售公告 |accessdate=2017-10-30 |publisher=南宁铁路局物资交易所 |date=2017-09-22 }}</ref>。而GKD1型柴油机车在被南宁铁路局接收后，调往柳州机务段本部作为调车机车使用，后被封存。

在被南宁铁路局接管后直至改做为东旺物流园这一期间，桂林东站的调车任务由南宁铁路局[[柳州机务段|柳州机务段]]桂林运用车间的[[东风5型柴油机车|东风5型柴油机车]]负责。
