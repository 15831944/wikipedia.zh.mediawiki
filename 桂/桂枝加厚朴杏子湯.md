'''桂枝加厚朴杏子湯'''，又名'''桂枝加厚朴杏仁湯'''，為溫辛解表劑，主治外感傷風帶有微咳，以現代的話語來說，就是類似於[[气管炎|氣管炎]]，[[支气管炎|支氣管炎]]、[[百日咳|百日咳]]等病症。

== 出處 ==
《[[傷寒雜病論|傷寒論]]》．辨太陽病脈證并治法中：「太陽病下之，微喘者，表未解也，桂枝加厚朴杏子湯主之。」<ref>[https://ctext.org/shang-han-lun/zh 《傷寒論》]中國哲學書電子化計劃.</ref>

== 功用 ==
主治外感風寒，伴有不算嚴重的咳嗽；氣管炎暨支氣管炎。 

== 辨症要點 ==
*基本上病人不怕冷也不怕熱，但帶有咳嗽的現像，症狀上既不是[[小青龍湯|小青龍湯]]症，也不是[[大青龍湯|大青龍湯]]症。
*吹風會覺得不舒服，有一點流汗。
*本方所指稱的咳嗽，並不是由於腹滿或心下的水氣所引起，乃是誤用攻下的方法後，以致內氣上逆迫胸所引起的咳嗽。

== 應用時機 ==
#太陽中風表症因有腹滿或便秘的狀況，誤用攻下劑所生之咳喘現象。
#若病人素來便有喘咳的毛病，一旦感冒（中風），因為造成太陽表氣不和，致使肺氣的宣發、肅降失調，從而引發了原有的咳嗽，此時仍要以新病為主，對於舊病應該安排在新病獲癒之後<ref>此為中醫治病原則之一，見{{cite book|title=精譯傷寒雜病論∶傷寒論 金匱要略合訂本|last=柳|first=術軍|publisher=中醫古籍出版社|year=2003|pages=241| isbn=9789864710713}}</ref>，是故以[[桂枝湯|桂枝湯]]為主，治療新病<ref>[http://www.theqi.com/cmed/class/class1/note_14.html 郝萬山講《傷寒論》 第14講 太陽傷寒與麻黃湯的適應症]</ref>，用厚朴來乾燥脾臟的濕氣，用[[杏仁|杏仁]]來潤肺<ref>{{cite book|title=解讀傷寒論-藥物篇|last=張|first=步桃|publisher=遠流出版公司|isbn=957-32-5663-0|pages=124, 141.}}</ref>。
#幼童身體較弱，常於感冒之時伴有喘咳的現像，若服用麻黃劑則會引起食慾不振，或身體倦怠，此時宜用本方<ref name=”矢數道明”>矢數道明著，吳家鏡譯，傷寒論解說，大眾書局，頁90。</ref>。

== 方劑組成 (漢制) ==
[[桂枝|桂枝]]三兩、[[芍藥|芍藥]]三兩、[[甘草|甘草]]三兩（炙）、[[大棗|大棗]]十二枚（擘）、[[厚朴|厚朴]]二兩（炙、去皮）、[[杏仁|杏仁]]五十個（去皮尖）。以水七千，微火煮取三升，去滓，溫服一升，覆取微汗。

== 方義 ==
*本方是說，太陽病仍有表症尚未痊癒時，如果兼有陽明症或是假的內實之症時，都要先處理表症，不能置表邪不顧而先攻下。換句話說，太陽症若伴有腹滿、便秘等應該攻下的症狀時，仍然需要先用桂枝湯驅解表邪，然後才能攻下<ref name=”矢數道明” />。
*桂枝能發汗，鬆弛肌肉。芍藥亦能鬆弛肌肉，有止痛的效果。甘草能解毒，大棗能保護腸胃，厚朴能去濕氣，杏仁潤肺<ref>{{cite book|title=解讀傷寒論-藥物篇|last=張|first=步桃|publisher=遠流出版公司|isbn=957-32-5663-0|pages=48, 128, 76, 124, 141.}}</ref>。
