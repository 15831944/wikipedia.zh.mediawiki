'''桂园'''，位于[[重庆市|重庆市]][[渝中区|渝中区]]中山四路65号，是原[[中华民国国军|中华民国国军]][[上将|上将]][[张治中|张治中]]的公馆，因院内有两株[[桂树|桂树]]而得名。1945年8月[[重庆谈判|重庆谈判]]期间，桂园成为毛泽东与[[周恩来|周恩来]]等人代表[[中国共产党|中国共产党]]同[[国民政府|国民政府]]代表进行谈判和签订《[[政府与中共代表会谈纪要|政府与中共代表会谈纪要]]》（即[[双十协定|双十协定]]）的地点<ref name=xiangyang>{{Cite web|url =http://www.hj.cn/2013/0731/491331.shtml|title =陪都记忆——那些正在消失的往事|author =|date =2013-07-31|publisher =襄阳日报网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20160709051641/http://www.hj.cn/2013/0731/491331.shtml|archivedate =2016-07-09}}</ref>。现为[[红岩革命纪念馆|红岩革命纪念馆]]桂园展览室<ref name=cqwwj/>。

==历史==
桂园原为属于[[孔祥熙|孔祥熙]]系统的[[关吉玉|关吉玉]]的产业，关吉玉时任[[国民政府|国民政府]]财政部司长，后来擢任次长、部长。1937年12月，国民政府内迁重庆后，[[陈诚|陈诚]]租用桂园作为官邸。1938年，张治中自[[湖南省政府|湖南省政府]]主席调任[[国民政府军事委员会|国民政府军事委员会]]委员长侍从室一处主任（分管军事），因为桂园距离侍从室近，乃和陈诚商议将桂园转租给张治中，直到[[抗日战争|抗日战争]]胜利。1945年8月28日，[[毛泽东|毛泽东]]自[[延安|延安]]抵达重庆，参加[[重庆谈判|重庆谈判]]。[[周恩来|周恩来]]虑及毛泽东的安全，乃向张治中借用桂园，自此桂园成为毛泽东、周恩来等人在重庆谈判期间的主要活动场所。毛泽东和[[蒋介石|蒋介石]]面谈六次。在国共双方指定的代表周恩来、[[王若飞|王若飞]]、[[王世杰_(中華民國)|王世杰]]、[[张群|张群]]、张治中、[[邵力子|邵力子]]谈判时，有时毛泽东也在场。多数接触是在德安里101号的蒋介石旧居举行，但会外接触多在桂园。经过43天谈判，1945年10月10日《政府与中共代表会谈记要》（《[[双十协定|双十协定]]》）签字仪式在桂园举行<ref name=cqwwj/>。

1962年2月18日，“桂园”列為重慶市第一批文物古蹟保護單位<ref>《重慶市文物古跡情況調查和今後工作意見》，重慶市檔案館藏，1085/1/729</ref>。1980年7月7日，“桂园”列為四川省重新確定公布的第一批省級文物保護單位<ref>關於重新公布全省文物保護單位名單的通知（川府發〔1980〕154號），四川省人民政府，1980年7月7日</ref>。1983年12月1日，“桂园”列為重慶市第一批市級文物保護單位<ref>「重慶市人民政府關於重新公布重慶市文物保護單位名單的通知」，重慶市人民政府文件，重府發〔1983〕208號，1983年12月1日</ref>。1977年，在桂园举办陈列展览，正式对外开放<ref name=cqwwj/>。2000年9月7日，作为“[[重慶談判舊址|重慶談判舊址]]”之一列為第一批[[重慶市文物保護單位|重慶市文物保護單位]]<ref>重慶市人民政府關於公布第一批重慶市文物保護單位的通知（渝府發〔2000〕83號），2000年9月7日</ref>。2001年6月25日，“桂园”公佈為[[第五批全國重點文物保護單位|第五批全國重點文物保護單位]]<ref>國務院關於公布第五批全國重點文物保護單位和與現有全國重點文物保護單位合併項目的通知（國發〔2001〕25號），2001年6月25日</ref>。

==建筑==
[[File:Guiyuan_Introduction.jpg|thumb]]
桂园位于今重庆市渝中区中山四路65号（原德安里107号）。总占地面积800平方米。园内有中西式砖木结构楼房一幢，坐南朝北，一楼一底，高约11米，面阔约13米，进深约13米，占地面积273平方米。进门为走道及衣帽间，右侧是客厅，厅后两小间是秘书办公室及洗手间，右侧是餐厅，餐厅后面是厨房。楼上是卧室，同楼下房间对称。楼后北边的多间平房是传达室，左侧是汽车房<ref name=cqwwj>{{Cite web |url =http://www.cqww.gov.cn/contents/59/1376.html  |title = 桂园 |author =  |date = 2015-04-28 |publisher =  重庆市文物局|language =  |accessdate =  |quote =  }}</ref>。

重庆谈判期间，在楼下右侧餐厅里，毛泽东、周恩来曾宴请过各国驻华团体负责人和各界人士。二楼是毛泽东、周恩来办公、休息的地方<ref name=xiangyang/>。
