'''桂东县'''属于[[湖南省|湖南省]][[郴州市|郴州市]]行政区，位于湖南省东南部，[[湘|湘]][[赣|赣]]交界地带，毗邻[[井冈山市|井冈山市]]，在[[毛泽东|毛泽东]]发起的[[秋收起义|秋收起义]]失利整编后在境内的沙田颁布《[[三大纪律八项注意|三大纪律八项注意]]》，属于中华人民共和国称之为的[[革命老区|革命老区]]。由于地处[[罗霄山|罗霄山]]脉深处，县城亦位于海拔2000米以上的地方，为湖南省海拔最高的县城，境内亦有湖南省海拔最高的山峰‘[[八面山|八面山]]’。同时由于地处深山，所以森林覆盖率相对较高，同时整个的生态环境亦相对好于全国普通水平，但随着近年的商业开发生态环境堪忧。

== 历史 ==

桂东县“本[[郴县|郴县]]地”。[[南宋|南宋]][[嘉定_(年号)|嘉定]]四年（1211年），{{来源请求|因为桂阳地区匪患猖獗}}，以[[桂阳县|桂阳县]]的'''零陵、宜城'''二乡于上犹砦置桂东县，属[[郴州_(古代)|郴州]]<ref>《[[宋史|宋史]]·卷八十八·志第四十一》地理四……郴州，中，桂阳郡，军事……南渡后，增县二：兴宁，嘉定二年，析郴县资兴、程水二乡置资兴县，后改今名。桂东。本郴县地。嘉定四年，析桂阳之零陵、宜城二乡置今县于上犹砦。宝庆府……</ref>，此为桂东县置县之始。明清时，属[[郴州直隶州|郴州直隶州]]。[[清朝|清朝]][[康熙|康熙]]十七年（1678年），因为[[吴三桂|吴三桂]]于[[衡阳|衡阳]]称帝，因避讳更名‘阳平县’次年复旧名<ref>彼时因为上属郴州路为吴周所辖，但桂东地处山区，彼时交通不畅通，是否为之所辖县志并未有所载，故无从考</ref>。

由于桂东地区地处山区，又处罗霄山脉深处，故而在[[二战|二战]][[抗日战争|日本侵略时期]]，虽郴州大部分区域被占领，但桂东县与[[汝城县|汝城县]]并未成为日寇占领区，因而桂东地区的共产党在此期间保留的相对完整，进而发展壮大，使之在之后的解放战争中早于湖南的和平起义成为湖南省最早被解放的县城<sub>（1949年7月间）</sub>。

1959年3月至1961年期间因为中国进入[[大跃进|大跃进时期]]桂东县与汝城县进行了短暂的合并，称‘汝桂县’。

桂东县治始为寨前，明后迁于今治城关（现更名为‘'''沤江镇'''’）。

=== 语言 ===
桂东土话为[[客家话|客家方言]]，但由于历史原因，同时存在有客家方言的南、北片，主流方言为北片[[于桂片|于桂片]]客话。

桂东主体属民大约为明清时的移民，大多由[[江西|江西]]迁入，而所属土著早已随历史的衍变或融入移民或向西外迁。但移民的来源并非单一从江西一地迁入，虽然迁入民众以客家人为主，但由于来源的丰富，所以造成了语言上的分化，但因主体的民众语言属于客家北片区，所以形成了今时县域的语言以北区于桂片为主，同时因为移民的迁入，也将北区客话的地域从江西西扩于湖南。

桂东因处山区，故亦有[[苗族|苗]]、[[瑶族|瑶]]等民族生活，但其言语以汉话为主。

=== 关于共产党 ===
因秋收起义失利，毛泽东率部南下退避井冈山时曾驻留桂东县城作战指挥，并在随后整编过程中在下属沙田颁发《三大纪律六项注意》。

1935年8月因中共军队开始战略转移，即[[长征|长征]]，红六军团于桂东誓师西征，近而拉开长征序幕。

== 文化 ==
虽然桂东境内属民多为客家人，但因为主体部分为北区客家人，客家文化形成较晚，故并无多的保留有相应的文化痕迹，但属于山区的南片客家移民却保持有相对完好的客家文化。

但婚丧嫁娶等习俗与客家文化区相同，稍有差异。

相对的亦有发展出属于本土的乡土文化。

== 地理 ==
地理坐标处东经113°37′—114°14′、北纬25°44′—26°13′之间，东西宽约61公里，南北长约54公里。

县境北部以山地为主，南部以盆地为主，境内多山，水系发达，各主要城镇均有穿城江河，是湘江的水源地之一。

=== 行政区划 ===
2012年5月开始县域行政区划进行了大量整合归并调整以期达到发展地方经济之要，其中包括对县城，城关镇的归并更名，故今县治名为‘[[沤江镇|沤江镇]]’。

=== 景点 ===
由于经济发展需求，近年亦有发展出多处旅游景点，但目前仍属于开发的初级阶段，而且多为人造景观，虽然境内多山，亦有数处自然景观，但旅游发展的导向却是以[[红色旅游|红色旅游]]为主，故而在有关于共产党革命的历史发生地为主建设有大批革命文化纪念设施。

* 自然景观：[[青鹅仙|青鹅仙]]、[[八面山|八面山]]、草山、县志八景<ref>以秀山、容洞为主，但并未得到相应的开发及保护。</ref>等。
* 历史遗迹：三台山<ref>山顶的廉溪学院遗迹早已无存，早年增将之开植为烈士纪念园现今废止，但始建于道光年间的宝塔仍耸立原止，并已成为县域标志性建筑物。</ref>、沙田[[城隍庙|城隍庙]]戏如及周遭附属建筑物、革命历史博物馆<ref>以唐家大屋为主扩除其它建设用以纪念革命的博物馆，馆藏有相应的革命文物。</ref>。
* 新景点：三台山<sub>人工建有湖区及疗养中心，以及于山上建立有寺院等设施</sub>、寨前<sub>（红色纪念设施）</sub>、寨前梦花园、沙田军规广场、其它新建景点。

== 经济 ==
由于地处山区，因为限止了桂东经济的发展，所以并未在改革开放中获得多少的改某红利，也因些在相对长的一段时间内（包括现在）桂东的经济发展水平一直处于湖南省的未尾，长期低于湖南的整体经济发展水平。

但随着近年来中央政府的大工程投入使桂东通向外界的道路得到很大的改善，但整体经济仍不佳。

地方主要经济以服装代工、农业、旅游业为主，同时因为境内矿藏的品种相对丰富，煤储量尤其丰富，但因为通路问题，主要为内销，所以采矿业虽然相对于县域经济发达，但亦有限。由于近年的开发，所以县域内的建筑业亦得到发展，随同带动相关的附属行业，但由于主要的工和建筑被外包（以江西建筑企业为主）所以实质的发展并不大。

县域总面积1451.56平方公里，居全省县市的第73位。常住人口230,948人（2010普查），居全省第83位；年末户籍总人口20.97万人（2011年）。2011年，全市GDP总量17.30亿元，人均GDP为7,567元。财政总收入15,998万元，其中地方财政收入9,000万元。全年农村居民人均纯收入2,695元，城镇居民人均可支配收入10,851元<ref name="2011年桂东县经济与社会发展指标">2011年桂东县国民经济与社会发展指标出自《[http://www.hntj.gov.cn/tjgb/xqgb/czgb/201204/t20120419_92610.htm 桂东县2011年国民经济和社会发展统计公报] {{webarchive|url=https://web.archive.org/web/20130902190133/http://hntj.gov.cn/tjgb/xqgb/czgb/201204/t20120419_92610.htm |date=2013-09-02 }}》</ref>。

== 人物 ==
桂东由于地处湘赣山区，彼时为[[中华苏维埃共和国|中华苏维埃政权]]属地，故在共产党之中出现过数名桂东籍人物，其中以彼时有左王之称的‘[[邓力群|邓力群]]’为主，但在早期共产党中亦有数名桂东籍人物。

虽然桂东地处山区，但并非封闭，故亦有大批年青人才于革命年代冲破群山参与革命，因而亦有桂东男儿成为[[黄埔军校|黄埔]]生参与革命。
