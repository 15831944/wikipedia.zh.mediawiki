'''桂花'''（[[学名|学名]]：''{{lang|la|Osmanthus fragrans}}''），又名'''木樨'''、'''梫'''、'''月桂'''，常绿[[灌木|灌木]]或小[[乔木|乔木]]；叶子对生，多呈椭圆或长椭圆形，叶面光滑，革质，叶边缘有锯齿；秋季开花，花簇生于叶腋，花冠分裂至基乳有乳白、黄、橙红等色，极芳香；[[核果|核果]]成熟后为紫黑色。常见于南方。

另外桂花容易與[[樟科|樟科]]的喬木[[肉桂|肉桂]]混淆，在辨識時須小心。

== 分布 ==
[[File:Osmanthus_Fragrans_chinese_Guihua.JPG|thumb]]一株开花的桂树]]
[[亞熱带|亞熱带]]树种，桂花原产于亞洲。從喜馬拉雅地區、中国大陸西南部、台灣、日本南部等地均有野生，[[印度|印度]]、[[尼泊尔|尼泊尔]]、[[柬埔寨|柬埔寨]]也有分布。中国大陸有包括[[广西|广西]][[桂林市|桂林市]]、[[江蘇省|江蘇省]][[蘇州市|蘇州市]]、[[浙江省|浙江省]][[杭州市|杭州市]]、浙江省[[衢州市|衢州市]]、[[湖北省|湖北省]][[咸宁市|咸宁市]]及[[安徽省|安徽省]][[合肥市|合肥市]]在内的20多个城市以桂花为市花。

== 变种 ==
*[[金桂|金桂]]（''Osmanthus fragrans'' var. ''thunbergii''）：金桂花朵金黄，香气浓郁，叶片较厚。
*[[银桂|银桂]]（''Osmanthus fragrans'' var. ''latifolius''）：银桂花朵颜色较白，稍带微黄，香气较淡，叶片较薄。
*[[丹桂|丹桂]]（''Osmanthus fragrans'' var. ''aurantiacus''）：丹桂花朵颜色橙黄，气味浓郁，叶片厚，色深。
*[[四季桂|四季桂]]（''Osmanthus fragrans'' var. ''semperflorens''）：四季桂别称月月桂。花朵颜色稍白，或淡黄，香气较淡，叶片薄。长年开花。

== 生長習性 ==
桂花喜溫暖環境，宜在土層深厚、排水良好、疏鬆肥沃、富含腐殖質的偏酸性砂質土壤中生長。不耐乾旱瘠薄，在淺薄板結貧瘠的土壤上，生長特別緩慢，枝葉稀少，葉片瘦小，葉色黃化，不開花或很少開花，甚至有週期性的枯頂現象，嚴重時桂花整株死亡。幼樹時需要有一定的蔽蔭，成年後要求要有相對充足的光照，才能保證桂花的正常生長。

== 食用 ==
[[File:Kueh_hrua_lau_cau.jpg|thumb]]]]
[[File:Sweet_Tea_Olive_Cake.jpg|thumb]]
桂花味辛，可入药。有化痰、止咳、生津、止牙痛等功效。

桂花味香，持久，可制[[糕点|糕点]]（[[桂花糕|桂花糕]]）、[[糖果|糖果]]，并可酿[[酒|酒]]。此外，亦常制成[[桂花糖|桂花糖]]、[[桂花湯圓|桂花湯圓]]、[[桂花釀|桂花釀]]（與蜂蜜泡制而成）、桂花酱、桂花滷等。在[[中国|中国]][[南京|南京]]，加工盐水鸭的过程中，加入桂花，增加香味，也称为[[桂花鸭|桂花鸭]]。
过去[[台湾|台湾]]桂花多半种植在茶园旁，作为提升茶香气陪衬，也因此易染病需施农药，不能食用居多。

中國國家林業局有訂定林業行業標準LY/T 1910-2010食用桂花栽培技術規程，其附表有列出可用的主要食用桂花品種。<ref>[http://www.forestrysci.gov.cn/portal/lykj/s/1716/content-663048.html 林業行業標準目錄]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}國家林業局</ref><ref>[http://new.sac.gov.cn/was5/web/outlinetemplet/hbSearch.jsp 行業標準備案目錄查詢] {{Wayback|url=http://new.sac.gov.cn/was5/web/outlinetemplet/hbSearch.jsp |date=20161220180933 }}國家標準委</ref>
