'''桂花西路站'''（工程名为桂花路站）是建设中的[[杭州地铁6号线|杭州地铁6号线]][[杭富城际铁路|富阳段]]的一个车站，位于[[杭州市|杭州市]][[富阳区|富阳区]]金桥北路桂花西路<ref>{{cite web |title=杭富城际铁路11个站点的具体位置出炉 |url=http://hznews.hangzhou.com.cn/chengshi/content/2016-11/07/content_6371321.htm |website=杭州网}}</ref>

==结构==
桂花西路站为地下二层岛式站台，为地下两层双柱三跨钢筋混凝土箱形结构。站台宽度12.6米。站前设单渡线，站后设两道停车折返线，车站总建筑面积约为32800平方米。
<ref>{{cite web |title=省发展改革委关于杭州至富阳城际铁路工程初步设计的批复 |url=http://www.zjdpc.gov.cn/art/2016/11/4/art_808_1717507.html |website=浙江省发展和改革委员会办公室}}</ref>
