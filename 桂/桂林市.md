'''桂林市'''（[[邮政式拼音|邮政式拼音]]：{{lang|la|Kweilin}}）是[[中国|中国]][[广西壮族自治区|广西壮族自治区]]下辖的[[地级市|地级市]]，位于广西东北部。市境西界[[柳州市|柳州市]]，南临[[来宾市|来宾市]]、[[梧州市|梧州市]]、[[贺州市|贺州市]]，东接[[湖南省|湖南省]][[永州市|永州市]]，北邻湖南省[[邵阳市|邵阳市]]、[[怀化市|怀化市]]。地处[[南岭山脉|南岭山脉]]西南部桂东北山地区，[[湘桂走廊|湘桂走廊]]南端，北部为[[越城岭|越城岭]]，东部为[[海洋山|海洋山]]、[[都庞岭|都庞岭]]，西部为[[天平山_(桂林)|天平山]]。主要为丘陵地貌，中部为典型的岩溶峰林区。[[漓江|漓江]]、[[湘江|湘江]]为境内两大河流，中间有[[灵渠|灵渠]]相连，全市总面积2.77万平方公里，人口496.16万。桂林是首批[[国家历史文化名城|国家历史文化名城]]、[[中国旅游城市|中国旅游城市]]，享有“桂林山水甲天下”之名誉。

== 历史沿革 ==
距今约一万年前，根据市区[[宝积岩|宝积岩]]和[[甑皮岩|甑皮岩]]洞穴发现的遗物考证，甑皮岩人已进入母系氏族社会阶段。 夏商周时期这里是[[百越|百越]]人的居住地。

[[秦朝|秦]]（前214年）开凿[[灵渠|灵渠]]沟通湘漓水系，属[[桂林郡|桂林郡]]。

[[汉朝|汉]][[元鼎|元鼎]]六年（前111年）设始安县，为县治所在，隶属荆州[[零陵郡|零陵郡]]。[[东汉|东汉]](30年)时又设始安侯国，为始安侯国治所。

[[三国|三国]]先属[[蜀汉|蜀]]，后归[[孫吳|吴]]。甘露元年（265年）建[[始安郡|始安郡]]，郡县治所都在今之桂林，属荆州始安郡。

[[南北朝|南北朝]]（540年）置桂州于湘州始安郡，受湘州督。

[[隋朝|隋朝开皇十年]]（590年）置桂州总管府。大业元年（605年），总管府被改为都尉府。<ref>{{cite book|author1=张湛平|title=中国历史文化名城小辞典|year=1991|publisher=中國工人出版社}}</ref>

[[唐朝|唐朝]]（634年）改名临桂县，属桂州始安郡。（692年）筑[[相思埭|相思埭]]，即今[[桂柳运河|桂柳运河]]。（757年）改始安郡为建陵郡。（868年）[[庞勋|庞勋]]率桂州戍卒反唐。（879年）[[黄巢|黄巢]]领导的农民起义军陷桂州。（900年）属静江军节度。

[[北宋|北宋]]（997年）置广南西路，治所在今桂林，属桂州<ref>{{cite book|editor=广东省地方史志编纂委员会|title=广东省志：地理志|publisher=广东人民出版社|year=1999|volume=51|page=31}}</ref>。
[[南宋|南宋]]（1133年）升桂州为静江府，治今桂林。静江府本桂林，始安郡，静江军节度。

[[元朝|元朝]]（1277年）3月置广南西路宣抚司于静江（今桂林），5月改为广南西路宣尉司。（1278年）7月增设岭南广西道，改静江府为静江路总管府。（1295年）并左右两江宣尉司都元帅府为广西两江道宣尉司都元帅府，分司邕州。（1363年）3月立广西等处行中书省，省治在今桂林。

[[明朝|明朝]]（1370年）[[朱元璋|朱元璋]]封侄孙[[朱守谦|朱守谦]]为[[明朝藩王列表_(仁祖系)#靖江國|靖江王]]，7月建[[靖江王府|靖江王府]]于[[独秀峰|独秀峰]]前。明朝洪武年(1367年)設[[廣西承宣布政使司|廣西承宣布政使司]]將[[靜江路|靜江路]]改[[桂林府|桂林府]]因而沿用至今。（1372年）6月改为桂林市府，府治在今桂林。

[[清朝|清朝]]初建广西省，於桂林置巡抚，属广西省桂林府。

[[中華民國|中華民國]]成立后（1911年）被改为广西省省会。民国元年（1912年）10月，[[广西军政府|广西军政府]]从[[桂林|桂林]]迁往南宁，南宁成为广西省会；民国25年（1936年）10月，广西省会从南宁迁往桂林；民国29年（1940年）划桂林县县城及附廓成立桂林市，余地复称临桂县。民国33年（1944年）11月10日，[[桂林保卫战|桂林保卫战]]失利，[[日军|日军]]占领了桂林。民国34年（1945年）7月28日，桂林被收复<ref>{{cite book|title=大辞海: 军事卷|publisher=上海辞书出版社|year=2007|page=370}}</ref><ref>{{cite book|author1=景敏|title=二次大战中国战区统帅部內幕纪实|publisher=甘肃人民出版社|pages=2|year=1993}}</ref>。民国38年（1949年）10月，广西省会再次从桂林迁往南宁。

1949年11月22日，[[中国人民解放军第四野战军|中国人民解放军第四野战军]]攻占桂林<ref>{{Cite web |url=http://www.people.com.cn/GB/historic/1122/3974.html |title=1949年11月22日 桂林解放 |accessdate=2014-09-05 |author= |date= |work= |publisher=人民网}}</ref>。1958年改称广西壮族自治区桂林市，辖桂林城区和郊区。

桂林市原分为桂林市和[[桂林地区|桂林地区]]两部分。1981年7月1日，[[阳朔县|阳朔县]]划归桂林市。1983年10月8日，[[临桂县|临桂县]]划归桂林市。1990年2月3日，撤销[[恭城县|恭城县]]，设立恭城瑶族自治县。1996年12月2日，桂林市郊区更名为[[雁山区|雁山区]]。1998年8月27日，桂林市和桂林地区合并。2013年，撤销临桂县，设立桂林市临桂区，市政府迁驻临桂区西城中路69号。<ref name="one">{{cite news|title=临桂"撤县改区"获国务院批复 桂林政府驻地将迁往|url=http://news.gxnews.com.cn/staticpages/20130409/newgx51634bd6-7326917.shtml|newspaper=广西新闻网|date=2013年4月9日}}</ref>2014年7月正式搬迁。<ref>[http://news.focus.cn/gl/2014-07-12/5258575.html 桂林市政府正式迁址临桂新区 12日隆重揭牌]</ref>

== 地理 ==
=== 临近地区 ===
[[File:Guilin3.jpg|left]]
[[桂林市区|桂林市区]]与[[临桂区|临桂区]]、[[灵川县|灵川县]]、[[阳朔县|阳朔县]]相邻。

桂林市西与本省[[柳州市|柳州市]]、西南与[[来宾市|来宾市]]、南与[[梧州市|梧州市]]、东南与[[贺州市|贺州市]]相邻，西北与[[湖南省|湖南省]][[怀化市|怀化市]]、北与[[邵阳市|邵阳市]]、东与[[永州市|永州市]]相邻。

=== 地貌 ===
桂林属典型的[[喀斯特地貌|喀斯特地貌]][[石灰岩|石灰岩]]地形，大部分为[[丘陵|丘陵]]，北部和东部分别为[[越城岭|越城岭]]和[[海洋山|海洋山]]、[[都庞岭|都庞岭]]山区，有很多面积较小的山间[[盆地|盆地]]。桂林市区坐落在[[湘桂走廊|湘桂走廊]]中的一个相对较大的[[平原|平原]]里。桂林市南部有破碎的丘陵平原，西部、北部和东部是大面积的山地。[[猫儿山|猫儿山]]海拔2142米，称“[[华南|华南]]第一峰”。

=== 气候 ===
桂林属中[[亚热带季风气候|亚热带季风气候]]，温和多雨，雨热同期，气候适宜。又有“三冬少雪，四季常花”的说法。冬季阴凉，偶尔寒冷。夏季炎热潮湿。1月平均气温7.9℃，7月平均气温28.0℃，年平均气温18.8℃。无霜期为309天。年均[[降水量|降水量]]1901.3mm。桂林雨美可与重庆雾美相映，有“雾重庆，雨桂林”之说。每年4月至10月为最佳旅游季节。

桂林属于“[[梅雨|梅雨]]”地区，夏季降水集中，雨量大。降梅雨时，天气闷热，空气潮湿，河水暴涨，经常形成[[洪灾|洪灾]]，对桂林的[[经济|经济]]特别是[[旅游业|旅游业]]造成极大的危害。

桂林冬季气温在零度及以下的情况并不少见，但很少降雪，大雪更是凤毛麟角。降雪是学生们的最爱，此时学校往往会停课，但是大雪也会造成巨大的经济损失。在2002年末至2003年初下大雪的时候，一些工厂停工，由于路面结冰，交通几近瘫痪，大量农作物、牲畜和绿化植物冻死，由于缺少防雪的经验，罕见的大雪给桂林市造成了巨大的损失。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为474.80万人<ref>{{cite book |author=桂林市统计局|title=《桂林市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加2.82万人，增长0.60%，年平均增长率为0.06%。其中，男性人口为245.66万人，占51.74%；女性人口为229.14万人，占48.26%。人口性别比（以女性为100）为107.21。0－14岁人口为76.66万人，占16.15%；15－59岁人口为328.01万人，占69.08%；60岁及以上人口为70.13万人，占14.77%；其中65岁及以上人口为48.66万人，占10.25%。汉族人口为401.33万人，占84.53%；各[[少数民族|少数民族]]人口为73.47万人，占15.47%。

== 经济 ==
[[File:Flickr_-_archer10_(Dennis)_-_China-7648.jpg|缩略图]]
桂林属[[南方|南方]]水稻产区，[[耕地|耕地]]多是[[稻田|水田]]，山区[[梯田|梯田]]颇多。[[粮食|粮食]]作物主要是[[水稻|水稻]]。桂林的[[经济作物|经济作物]]很多，主要的经济作物有[[罗汉果|罗汉果]]、[[马蹄|马蹄]]、[[毛竹|毛竹]]和[[茶树|茶树]]等。

== 交通 ==
[[File:Guilinxi.jpg|缩略图]]
{{Main|桂林交通}}
桂林交通发达，是[[广西|广西]]乃至[[华南地区|华南地区]]的重要交通枢纽。[[桂林两江国际机场|桂林两江国际机场]]是全国为数不多的能起降[[空中客车A380|空中客车A380]]的4F级区域性干线枢纽机场，拥有60多条国内、国际航线。主要铁路有[[湘桂铁路|湘桂铁路]]、[[衡柳铁路|衡柳铁路]]和[[贵广客运专线|贵广客运专线]]，桂林主要的火车站有[[桂林站|桂林站]]、[[桂林北站|桂林北站]]和[[桂林西站|桂林西站]]。公路方面有[[321国道|321国道]]、[[322国道|322国道]]和[[323国道|323国道]]，还有[[泉南高速公路|泉南高速公路]]、[[包茂高速公路|包茂高速公路]]、[[厦蓉高速公路|厦蓉高速公路]]、机场高速公路和[[桂林绕城高速公路|桂林绕城高速公路]]等高速公路。水运可以通过漓江的磨盘山码头和竹江码头乘船游览[[漓江|漓江]]，也可乘船直达珠江上的[[梧州|梧州]]、[[广州|广州]]、[[香港|香港]]等城市。

== 城市改造 ==
[[File:Guilin_liberation_bridge.jpg|thumb]]]]

桂林从[[二十世纪|二十世纪]]九十年代起进行大规模的城市改造，主要是改造小街小巷，改造道路，整治市容。有“百条小巷整改工程”。要把桂林建设成为一个环境优美的，设施优良的，人文优秀的城市。另外为发展[[旅游业|旅游业]]大规模建设城市景点，进行[[生态环境|生态环境]]改造。另外还增加了免费的公益[[公共汽车|公交车]](已撤销)。

为了方便交通及桂林整体的形象，市政府决定重修和扩建一部分桥梁，这部分桥梁有[[解放桥_(桂林)|解放桥]]、[[龙隐桥|龙隐桥]]、[[文昌桥|文昌桥]]等。后来这部分桥梁和漓江、桃花江、杉湖、榕湖、桂湖、新开掘的木龙湖、襟江形成著名的两江四湖体系。

=== 旅游 ===
[[File:Guilin_lijiang.jpg|thumb]]

桂林是一个典型的旅游型城市，是中国知名度最高的旅游胜地之一。[[桂林山水|桂林山水]]因“山青、水秀、洞奇、石美”而驰名中外。南宋诗人[[范成大|范成大]]在《桂海虞衡志》中说：“余尝评桂山之奇，宜为天下第一。”南宋[[王正功|王正功]]在其诗作中有云:“桂林山水甲天下。”唐代大诗人[[韩愈|韩愈]]曾在其《送桂州严大夫同用南字》中称赞桂林山水为：“江作青罗带，山如碧玉篸。”1985年，《[[中国旅游报|中国旅游报]]》发起了历时半年多的评选“[[中国十大风景名胜|中国十大风景名胜]]”的活动。“桂林山水”排名“[[万里长城|万里长城]]”之后，名列“中国十大风景名胜”第二位。

主要旅游景点：
* 生态公园和人文山水艺术有[[七星公园|七星公园]]，[[象鼻山|象鼻山]]，[[伏波山|伏波山]]，[[叠彩山|叠彩山]]，[[西山公园_(桂林)|西山公园]]，[[穿山公园|穿山公园]]，[[虞山公园_(桂林)|虞山公园]]，[[世外桃源_(阳朔)|阳朔世外桃源]]，[[大圩古镇|大圩古镇]]，[[南溪山公园|南溪山公园]]，[[雁山植物园|雁山植物园]]等。
* 古代近代文化有[[靖江王陵|靖江王陵]]及王府，[[桂林石刻|桂林石刻]]，[[灵渠|灵渠]]，[[桂海碑林|桂海碑林]]等。
* 喀斯特溶洞景观有[[芦笛岩|芦笛岩]]，[[冠岩|冠岩]]，[[七星岩_(桂林)|七星岩]]，[[荔浦|荔浦]][[银子岩|银子岩]]，[[丰鱼岩|丰鱼岩]]等。
* 大型综合景区有[[两江四湖|两江四湖]]（[[漓江|漓江]]、[[桃花江_(桂林)|桃花江]]、[[榕湖|榕湖]]、[[杉湖|杉湖]]、[[桂湖_(桂林)|桂湖]]、[[木龙湖|木龙湖]]），[[漓江|漓江]]景区，[[乐满地|乐满地]]等。

=== 小吃 ===
[[file:Guilin_mifan.jpg|thumb]]桂林最有名的食品莫过于[[桂林米粉|桂林米粉]]。此外，民间手工制作的[[马蹄糕|马蹄糕]]、[[板栗粽|板栗粽]]等也颇受欢迎。十二县也有名吃如[[恭城|恭城]]的[[油茶_(小吃)|油茶]]，用[[荔浦|荔浦]]名产[[荔蒲芋|荔蒲芋]]制作的[[扣肉|扣肉]]、[[阳朔镇|阳朔]]的[[啤酒鱼|啤酒鱼]]和[[灵川|灵川]][[狗肉|狗肉]]等。

享誉海内外的三种制品即[[辣椒酱|辣椒酱]]、[[豆腐乳|豆腐乳]]、[[三花酒|三花酒]]被合称为桂林三宝，后来还有“桂林四宝”一说在“桂林三宝”的基础上加入了[[西瓜霜|西瓜霜]]。
