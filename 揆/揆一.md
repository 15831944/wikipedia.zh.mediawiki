'''揆一'''（{{lang-nl|'''Frederick Coyett'''}}，{{bd|1615年||1687年|10月17日|catIdx=Coyett, Frederick}}），直譯為'''弗雷德瑞克·柯耶特'''，是一名[[瑞典|瑞典]]貴族、[[荷蘭東印度公司|荷蘭東印度公司]]官員，也是[[台灣荷蘭統治時期|台灣荷蘭統治時期]]的最後一任[[台灣長官|台灣長官]]。

== 生平 ==
揆一1615年生於[[瑞典|瑞典]][[斯德哥爾摩|斯德哥爾摩]]。進入[[荷蘭東印度公司|荷蘭東印度公司]]後，經歷各種職階，1645年在[[巴達維亞|巴達維亞]]升任為高級商務專員，1647年被任命為[[日本|日本]][[出島|出島]]商館館長，曾於[[尼可拉斯·維堡|尼可拉斯·維堡]]擔任[[臺灣長官|臺灣長官]]時批評費爾勃格[[郭懷一事件|郭懷一反荷事件]]鎮壓政策。

1656年6月升為[[臺灣長官|臺灣長官]]，5個月後[[臺灣島|臺灣島]]遭[[颱風|颱風]]侵襲，損失慘重。1662年於[[鄭成功攻台之役|鄭成功攻台之役]]為[[明鄭王朝|明鄭王朝]][[鄭成功|鄭成功]]軍隊包圍而[[締和|締和]]。揆一带着1000多名荷兰人与其眷属離開臺灣，陆续回到[[巴達維亞|巴達維亞]]（今雅加达），揆一立即受到[[軍事審判|軍事審判]]，以失去公司重要財產之罪，被判終身[[流放|流放]]到[[班達群島|班達群島]]以西的[[艾一島|艾一島]]（Pulau Ai），直到12年後（1674年）揆一的子女、朋友向[[威廉三世_(英格兰)|威廉三世]][[陳情|陳情]]，並以25000[[荷蘭盾|荷蘭盾]]（約合[[银两|白銀]]7000[[兩|兩]]，[[銀元|銀元]]9700元）贖出揆一，在威廉[[特赦|特赦]]下回到荷蘭，同时要求揆一不得再过问任何与东印度公司有关事务，以後住在[[阿姆斯特丹|阿姆斯特丹]]至於逝世。

== 著作 ==
1675年以「'''C.E.S'''」的署名<ref>現今學界一般認定應指「Coyett et Socii」，即「揆一及其同僚」</ref>出版《[[被遺誤的臺灣|被遺誤的臺灣]]》（{{lang|nl|'t verwaerloosde Formosa}}，英文：Neglected Formosa）一書為自己辯護，譴責[[荷蘭東印度公司|東印度公司]]高層怠忽職守、拒派援軍、遺誤時機，才使他失去臺灣<ref>翁佳音，《被遺誤的臺灣》[http://nrch.cca.gov.tw/ccahome/website/site20/contents/011/cca220003-li-wpkbhisdict002686-0802-u.xml]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}，臺灣歷史辭典（十一畫），2004-05-18</ref><ref>{{cite web|author1=國立臺灣歷史博物館|title=斯土斯民-臺灣的故事：《被遺誤的福爾摩沙》，荷蘭使節與鄭成功談判圖|url=http://the.nmth.gov.tw/nmth/zh-TW/Item/Detail/0057e407-7c1e-4534-aba9-9cd9b6ac700d|agency=國立臺灣歷史博物館|work=斯土斯民展品列表 - 斯土斯民-臺灣的故事}}</ref>。

該書最新中文譯本（第五本）於2011年12月6日由台灣[[前衛出版社|前衛出版社]]出版，全名《被遺誤的台灣：荷鄭台江決戰始末記》。並特地請中研院台史所研究員[[許雪姬|許雪姬]]寫導讀<ref>{{Cite web |url=http://www.avanguard.com.tw/ch/CH1/1838/MD/MD0000001838001588.html |title=被遺誤的台灣：荷鄭台江決戰始末記  |access-date=2013-01-20 |archive-url=https://web.archive.org/web/20140115231139/http://www.avanguard.com.tw/ch/CH1/1838/MD/MD0000001838001588.html |archive-date=2014-01-15 |dead-url=yes }}</ref>。

== 後人 ==
揆一後代今居於[[比利時|比利時]]，其14代孫後曾於2006年訪臺，並遊覽[[安平區|安平]]等地且祭祀鄭成功感念300年前不殺先祖之恩<ref>《[http://www.asiahotel.tw/2006/06/blog-post.html 揆一的後代]》，東亞樓大飯店</ref>。

== 揆一像 ==
[[安平古堡|安平古堡]]中設有揆一像。[[中央研究院|中央研究院]]研究員[[翁佳音|翁佳音]]則認為此像應為[[審訊|審訊]]並[[流放|流放]]揆一的[[巴達維亞總督|巴達維亞總督]]{{le|約翰·馬特索科爾|Joan Maetsuycker}}（Joan Maetsuycker），而非揆一本人<ref>《臺灣史十一講》，ISBN 9860085021，國立歷史博物館</ref>。
