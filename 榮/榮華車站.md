'''榮華車站'''位於[[台灣|台灣]][[新竹縣|新竹縣]][[竹東鎮|竹東鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[內灣線|內灣線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]一座。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]。
<!-- *行駛車輛為[[冷氣柴客|DR1000]]型[[台鐵柴油客車|柴油客車]]，等級為[[復興號列車|復興號]]。 -->
*內灣線在2007年3月1日因[[新竹車站|新竹]]－[[竹東車站|竹東]]段配合[[六家線|六家線]]興建與內灣線改善工程而暫時停駛，於2011年11月11日復駛。
*為了讓車票上的起迄站名稱能夠組成「榮華富貴」四字而成為[[吉祥語車票|吉祥語車票]]，原名為「南河」的[[富貴車站|富貴車站]]因而改名，以求與本站站名搭配。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*2001年11月24日：設站。
*2007年3月1日：隨著內灣線新竹－竹東段停駛而暫停營運。
*2011年11月11日：隨著工程完工而復駛。
*2013年9月30日：啟用多卡通刷卡機。
