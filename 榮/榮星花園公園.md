'''榮星花園公園'''（簡稱'''榮星花園'''）位於[[台北市|台北市]][[中山區_(台北市)|中山區]][[行仁里|行仁里]]民權東路三段與建國北路、龍江路、五常街所圍繞之區域，面積6.8[[公頃|公頃]]。這塊土地裡，有一半以上的面積[[產權|產權]]歸屬於「榮星企業股份有限公司」，其餘為市有地及九十一名地主所有的私人土地。1968年由[[鹿港辜家|鹿港辜家]]開闢經營，原名'''榮星花園'''，採收費入園制，是台北市著名的花園綠地之一；西北角另設有小型游泳池一座；過去更曾有[[螢火蟲|螢火蟲]]現蹤於園內。

榮星花園於1989年由[[台北市政府工務局公園路燈工程管理處|台北市政府工務局公園路燈工程管理處]][[徵收|徵收]]接管，列為[[都市計畫|都市計畫]]中山232號公園。2003年底整建完成後，改稱榮星公園，並且設置螢火蟲復育區（可惜未有成效）。2011年再改名為榮星花園公園，簡稱又回到榮星花園，英文譯名維持Rongxing Park。

此處為電影《[[我的少女時代|我的少女時代]]》取景地點之一。

== 名稱來歷 ==
榮星花園的「榮星」二字，來自辜氏家族對[[辜顯榮|辜顯榮]]的紀念。辜顯「榮」，字耀「星」，所以命名為「榮星」花園。

== 歷史淵源 ==
1966年，辜振甫在工商界有了一定地位後，在廟旁建了一小亭供奉牌位立碑，並請一位[[中國國民黨中央評議委員會|中國國民黨中央評議委員會]]委員寫文紀念父親辜顯榮。此一事件被報刊披露，稱有一名人為[[漢奸|漢奸]]立碑，引起一陣震動。辜家趕忙拆亭毀碑，事態才未擴展。 

1968年，辜振甫之弟[[辜偉甫|辜偉甫]]興建榮星花園，並成立「榮星企業股份有限公司」來管理。又被輿論提起，說：「辜家紀念漢奸[[忠烈祠|忠烈祠]]不成，現在又蓋花園紀念辜顯榮了。」此時輿論似乎對[[意識形態|意識形態]]的問題不大關心，而未引起多大轟動。但後來爆發了「[[榮星花園弊案|榮星花園弊案]]」。 

1985年，榮星企業由辜偉甫之子[[辜晏宏|辜晏宏]]接手，以[[新台幣|新台幣]]8億元價格三次公告拍賣榮星花園，均無人問津。最後辜偉甫之弟[[辜寬敏|辜寬敏]]接手並出任榮星企業董事長，後來找[[僑福建設_(臺灣)|僑福建設]]董事長[[黃週旋|黃週旋]]合作，計劃開發投資達新台幣200億元的榮星花園。 

1987年6月，榮星企業將榮星花園投資計畫送到[[台北市政府|台北市政府]]審批，先後7次被退回，也無人問津。1988年，榮星花園開發案被揭發涉嫌[[官商勾結|官商勾結]]與[[索賄|索賄]]，在社會上引起極大轟動。結果在纏訟將近十五年後，包括開發案當時的[[民主進步黨|民主進步黨]][[台北市議員|台北市議員]][[周伯倫|周伯倫]]與[[中國國民黨|中國國民黨]]台北市議員[[陳俊源|陳俊源]]在內的多位政界與商界名人分別被判刑。<ref>[http://news.pts.org.tw/detail.php?NEENO=29417 周伯倫遭限制出境 最快年後發監].[[公視新聞|公視新聞]].2003-01-30</ref>

== 憲法爭議 ==
榮星企業代表人[[辜寬敏|辜寬敏]]，因其所有榮星花園土地為台北市都市計畫指定為公共設施保留地，長期未予徵收，經向[[行政法院|行政法院]]提起撤銷之訴，敗訴確定後，向[[司法院|司法院]][[大法官會議|大法官會議]]聲請釋憲；大法官於1994年2月4日作成[[司法院釋字第336號解釋|司法院釋字第336號解釋]]，認定辜寬敏所指牴觸《[[中華民國憲法|憲法]]》第十五條之《都市計畫法》第五十條並無違憲<ref>[http://www.judicial.gov.tw/constitutionalcourt/P03_01_detail.asp?expno=336&showtype=%AC%DB%C3%F6%AA%FE%A5%F3 榮０企業股份有限公司代表人辜０敏聲請書]</ref><ref>[http://www.judicial.gov.tw/constitutionalcourt/p03_01.asp?expno=336 釋字第336號]</ref>。
