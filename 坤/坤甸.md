'''坤甸市'''（{{lang-id|Kota Pontianak}}）簡稱「'''坤甸'''」（{{lang|id|Pontianak}} 或 {{lang|id|Khuntien}}），是[[印度尼西亚|印度尼西亚]][[西加里曼丹省|西加里曼丹省]]的[[首府|首府]]，臨近[[赤道|赤道]]，地處{{link-id|曼帕瓦縣|Kabupaten Mempawah}}境內。該市被[[卡普阿斯河|卡普阿斯河]]分隔兩岸，一邊（東面）為[[市區|市區]]，另一邊（西面）則為[[郊區|郊區]]。該市總[[人口|人口]]約達62.8萬人，乃[[西加里曼丹省|西加省]]的[[經濟|經濟]]、[[政治|政治]]中心和最大[[城市|城市]]。

'''坤甸'''建市於1771年10月23日，目前[[市中心|市中心]]為{{link-id|曼帕瓦鄉|Mempawah Hilir, Mempawah}}。早在西元18世紀，該地就被定為[[蘭芳共和國|蘭芳共和國]]的[[首都|首都]]<ref>朱杰勤《东南亚华侨史》</ref>。由於該市市內[[水渠|水渠]]縱橫交錯，故[[住房|屋舍]]大多被興建在水上。

==氣候==
按[[柯本气候分类法|柯本气候分类法]]，坤甸屬於[[熱帶雨林氣候|熱帶雨林氣候]]。全年多雨，年降雨量約{{convert|3210|mm}}。只有8月降雨量低{{convert|200|mm}}。氣溫全年不變，最高溫平均{{convert|30|°C}}，最低溫平均{{convert|23|°C}}。

==人口==
全市人口約628,000人。[[華人|華人]]人口占三分之一。種族主要為[[馬來族|馬來族]]（26.05%）、[[爪哇族|爪哇族]]（11.67%）、[[華裔|華裔]]（31.24%）、[[馬都拉|馬都拉]]（6.35%）、[[達雅族|達雅族]]等（3.57%）。

== 語言 ==
坤甸市有很多[[華人|華人]]聚居，約佔全市總[[人口|人口]]的三分之一。受[[中華文化|中華文化]]影響，當地的一些其他[[種族|族裔]]的[[印尼人|印尼人]]在使用[[印尼語|印尼語]]之外，偶爾也會以一兩句[[潮州話|潮州話]]和[[客家話|客家話]]這兩種比較強勢的[[漢語方言|漢語方言]]來跟[[華社|華社]]居民進行溝通。以[[卡普阿斯河|卡普阿斯河]]为[[分水岭|分水岭]]，该市[[东面|东面]]的[[市区|市区]]（老埠头）一带通行[[潮州話#分类|潮州话]]，以[[揭阳|揭阳]]口音为主；[[西面|西面]]的[[工业区|工业区]]（新埠头）一带则通行[[客語方言|客家话]]，以[[梅縣話|梅县口音]]和[[海陸客家語|陆丰口音]]为主。除此之外，當地亦有不少[[福建人|閩籍居民]]，同時略懂[[普通話|普通話]]。

== 教育 ==
坤甸市內有[[公立|公立]]的[[丹戎布拉大學|丹戎布拉大學]]，以及數所[[私立大學|私立大學]]。

==交通==
[[File:Kapuas_bridge.jpg|thumb]]

坤甸有兩條[[大橋|大橋]]連接市內兩岸。

===對內===
主要依靠[[小型巴士|小型巴士]]和人力[[三輪車|三輪車]]。當地居民則大多擁有[[電單車|電單車]]和[[腳踏車|腳踏車]]，方便日常出入。

===對外===
主要依賴[[飛機|飛機]]和[[船隻|船隻]]。[[空運|空運]]方面，[[坤甸機場|坤甸機場]]連接[[馬來西亞|馬來西亞]]和國內其它[[機場|機場]]；[[水運|水運]]方面，市內有一個中型[[貨櫃|貨櫃]]及[[客運|客運]][[碼頭|碼頭]]。
