'''沧州市'''是[[中华人民共和国|中华人民共和国]][[河北省|河北省]]下辖的[[地级市|地级市]]，位于河北省东部偏南。北界[[天津市|天津市]]、[[廊坊市|廊坊市]]，西北接[[保定市|保定市]]，西南连[[衡水市|衡水市]]，东南邻[[山东省|山东省]][[德州市|德州市]]、[[滨州市|滨州市]]，东临[[渤海|渤海]]。地处[[华北平原|华北平原]]东部，地势平坦。[[南运河|南运河]]、[[子牙新河|子牙新河]]流经境内。全市面积1.43万平方公里，人口744.3万。沧州盛产[[海盐|海盐]]，并拥有丰富[[石油|石油]]资源，亦是[[环渤海经济圈|环渤海经济圈]]和[[京津冀城市群|京津冀城市群]]的重要的区域中心城市。

==历史==
古为[[幽州|幽]]、[[兖州_(古代)|兖]]二州地。[[西汉|西汉]][[汉文帝|文帝]]置[[浮阳县|浮阳县]]，为[[勃海郡|勃海郡]]治。[[南北朝|南北朝]]时，分属[[北魏|北魏]][[瀛州|瀛州]]、[[冀州_(古代)|冀州]]，浮阳县为[[浮阳郡|浮阳郡]]治。[[北魏|北魏]][[元诩|孝明帝]][[熙平|熙平]]二年（517年）析瀛、冀二州设立[[沧州_(古代)|沧州]]，取沧海之意，治饶安县（今[[盐山县|盐山县]]西南），辖浮阳、安德、乐陵三郡。[[北周|北周]][[大象_(北周)|大象]]二年（580年）置[[长芦县|长芦县]]，治今市区西，属[[章武郡|章武郡]]。[[隋朝|隋]][[开皇|开皇]]十八年（598年）浮阳县更名[[清池县|清池县]]，属沧州。

[[唐朝|唐]][[开元|开元]]十六年（728年）移沧州治至今城区。《[[元和郡县志|元和郡县志]]》记曰：沧州“后魏孝明帝熙平二年分瀛州、冀州置沧州，以沧海为名。”清池、长芦两县属之。[[北宋|北宋]][[乾德_(北宋)|乾德]]二年（964年）省长芦县入清池县，清池县为沧州治。[[金朝|金朝]]仍置沧州。[[元朝|元]]徙沧州及清池县治至长芦镇，属[[河间路|河间路]]。[[明朝|明]][[洪武|洪武]]初省清池县入沧州，属[[京师|京师]][[河间府|河间府]]。[[清朝|清]]初沧州属河间府，[[雍正|雍正]]九年（1731年）改隶[[天津府|天津府]]。

[[民国|民国]]二年（1913年）废府制，改沧州为[[沧县|沧县]]，属[[直隶省|直隶省]][[渤海道|渤海道]]，次年改属[[津海道|津海道]]。1928年废道制。民国二十六年（1937年）沧县属河北省第六行政督察区。1947年6月，[[中国共产党|中国共产党]]军队发动[[青沧战役|青沧战役]]占领沧州。

[[中华人民共和国|中华人民共和国]]成立后，1949年8月，降沧州为沧镇，划沧县、青县、黄骅、建国（今沧县、河间交界）、任丘、河间、献县、肃宁、交河、泊头镇、沧镇为[[沧县专区|沧县专区]]。1958年撤销沧县专区，并入[[天津专区|天津专区]]。1961年复设沧州专区，并设沧州市，为专署驻地。1968年沧州专区改称[[沧州地区|沧州地区]]。1983年，沧州市升为省辖市，并划入沧州地区的沧县。1986年将[[青县|青县]]划归沧州市。1993年撤销沧州地区，所属县市划归沧州市。

== 地理 ==
沧州位于河北省东部，接近渤海，与[[天津市|天津市]]、[[河北省|河北省]]的[[廊坊市|廊坊市]]、[[保定市|保定市]]、[[衡水市|衡水市]]以及[[山东省|山东省]]的[[德州市|德州市]]、[[滨州市|滨州市]]接壤。沧州地处环渤海中心地带，是河北省确定的“两环” (环京津、环渤海)开放一线地区，也是京津通往东部沿海地区的交通要冲。[[大运河|大运河]]也穿过沧州。

===气候===
沧州属半湿润的暖[[温带季风气候|温带季风气候]]，四季分明，温度适中，光照充足，雨热同季，降水集中。1月平均气温−3.0℃，7月平均气温26.5℃。年平均气温12.9℃。全年降水量605毫米，主要集中在夏季，7和8月占60%左右。年日照时数2662.7小时。

===水资源===
沧州水资源极为短缺。由于成陆时间短，沧州的浅表地下水为苦咸水；深井地下水高氟，超标3到6倍，造成严重的地区性氟骨症流行。1960年代以前，沧州[[大运河|大运河]]沿岸使用运河水。但是自[[南运河|南运河]]断流以后，就只能引用高氟地下水。沧州人均水资源占有量只有192立方米，为全国的8%。沧州市区，40万人长期饮用高氟水。为此，[[大浪淀水库|大浪淀水库]]1957年、1977年、1995年三次上马，1997年1月建成蓄水，每年可向市区供水8121万立方米，基本解决了沧州市区的地区性饮用水氟中毒问题。

沧州东部12县（市、区）各城区、以及222万农村人口到“十二五”末全部告别苦咸水、高氟水，通过集中供水工程饮用上达标的引黄河水。 沧州西部四县市，计划2014年启动引长江水集中供水工程，主要工程是修建80多公里的干渠和若干水库。远期，沧州市计划建成日产20万立方米淡化海水。

2000年起，以国家、河北省开始解决人饮安全工程，在沧州农村联村建设采取深井水、通过反渗透净化设备的降氟水厂。

=== 行政区划 ===

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为7134053人<ref>{{cite book |author=沧州市统计局|title=《沧州市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加419661人，增长6.25%。年平均增长率为0.61%。其中，男性人口为3647200人，51.12%；女性人口为3486853人，占48.88%。总人口性别比（以女性为100）为104.60%。0－14岁人口为1278628人，占17.92%；15－64岁人口为 5265381人，占73.81%；65岁及以上人口为590044人，占8.27%。

== 交通 ==
[[File:沧州西站.jpg|thumb]]
沧州市是[[京沪铁路|京沪线]]上重要的城市之一，[[京九铁路|京九铁路]]也贯穿其间，[[京杭大运河|京杭大运河]]南支纵贯市内以煤码头[[黄骅港|黄骅港]]为终点的[[朔黄铁路|朔黄铁路]]正在成为中国的一条主要输煤铁路线路。<br />

于2011年6月30日正式商业运营的京沪高铁穿过沧州，沧州高铁站位于西郊，直达北京仅需50分钟

== 文化 ==
沧州是中国的[[武术|武术]]和[[杂技|杂技]]之乡，2009年获得中国[[书法|书法]]之乡的称号。有[[五代|五代]]时期建造的，至今屹立的雄伟的[[沧州铁狮子|沧州铁狮子]]。以吴桥命名的“中国吴桥国际杂技艺术节”每两年举办一次，有十几个国家和地区的杂技团体参加。[[吴桥杂技大世界|吴桥杂技大世界]]汇粹了所有中国传统杂技表演。沧州的“金丝小枣”和“泊头鸭梨”（以出口港的名称天津鸭梨闻名）是著名的特产水果，现在新培育的黄骅冬枣已经名闻全国。

== 人物 ==
*[[贾庆林|贾庆林]] （1940年-）河北泊头人，1959年12月加入[[中国共产党|中国共产党]]，1962年10月参加工作，河北工学院电力系电机电器设计与制造专业毕业，大学学历，高级工程师。曾任[[中共中央政治局常委|中共中央政治局常委]]，十一届[[全国政协|全国政协]]主席。
*[[扁鹊|扁鹊]] 传说中的[[战国|战国]]时名医，鄚（今[[任丘|任丘]]市）人。年轻时因奇遇得到了肉眼透视人体的特异功能，云游列国，广施医术，救治了许多患急难病症的百姓，成为天下众口称赞的神医。著有《[[内经|内经]]》、《[[外经|外经]]》、《[[难经|难经]]》等医著，并成为中医经典。
*[[冯道|冯道]]（882年－954年），字可道，号长乐老，瀛州景城（今河北沧县相国庄）人。生于唐僖宗中和二年（882），卒于后周显德元年（954）。中国大规模官刻儒家经籍的创始人。曾经侍奉五朝、八姓、十三帝，“累朝不离将相、三公、三师之位”。为官四十多年，是中国官场历史的一个不倒翁。
*[[张邦昌|张邦昌]]（1081年-1127年），[[北宋|北宋]]末大臣。字子能，永静军东光人（今[[河北|河北]]东光）。进士出身，[[宋徽宗|徽宗]]、[[宋钦宗|钦宗]]朝时，历任[[尚书|尚书]]右丞，左丞，[[中书|中书]][[侍郎|侍郎]]，[[少宰|少宰]]，[[太宰|太宰]]兼[[门下|门下]][[侍郎|侍郎]]等职务。金兵围[[开封|开封]]时，他力主议和，与康王[[赵构|赵构]]作为人质前往金国，请求割地赔款以议和。归宋后，任[[河北|河北]]路割地使。靖康二年（1127年），金兵攻陷[[汴京|汴京]]，掳走徽钦二帝及皇族470多人，及文武百官2000多人北歸，又立张邦昌为大楚皇帝，定都[[金陵|金陵]]（今[[江蘇省|江蘇省]]南京市），做了32天的皇帝。史稱“[[靖康之變|靖康之變]]”。后[[金国|金国]]退兵之后，迫于众怒，去除帝号，迎[[元祐皇后|元祐皇后]]垂帘听政，封郡王。后来南下归德（今[[河南|河南]][[商丘|商丘]]）。[[宋高宗|宋高宗]][[赵构|赵构]]即位后，宰相[[李纲|李纲]]力主严惩，建炎元年六月（1127年7月），被貶至潭州“安置”，“令監司守臣常切覺察”。不久金兵又來犯。同年九月下詔將張邦昌賜死，並誅[[王時雍|王時雍]]，殿中侍御史馬伸前來長沙執行。據稱張邦昌讀罷詔書，“徘徊退避，不忍自盡”，執行官逼他就死，最後登上潭州城內天寧寺的平楚樓，自縊身亡。
*[[马致远|马致远]]（1250年?-1323年?），字'''千里'''，號'''东篱'''。中国[[元代|元代]]初期[[杂剧|杂剧]]作家，河北[[东光县|东光]]人。作品以反映退隐山林的田园题材为多，风格兼有豪放、清逸的特点。有描述[[王昭君|王昭君]]传说的《[[汉宫秋|汉宫秋]]》以及《任风子》等。《汉宫秋》被后人称做[[元曲|元曲]]的最佳杰作。作品收入《东篱乐府》。与[[关汉卿|关汉卿]]，[[白仁甫|白仁甫]]，[[王实甫|王实甫]]（一说[[郑光祖|郑光祖]]）等人并称[[元曲四大家|元曲四大家]]。
*[[纪昀|纪昀]]（1724年－1805年） 字晓岚，清直隶河间府献县（今沧县）崔尔庄人。历经雍正、乾隆、嘉庆三朝，官至礼部尚书、加太子少保兼国子监事。一生诗文创作颇丰，有笔记小说《阅微草堂笔记》、诗文合集《纪文达公遗集》、撰著《评文心雕龙》等。编撰的《[[四库全书|四库全书]]》被誉为中国和世界文化史上规模最大的一部百科全书。
*[[张之洞|张之洞]]（1837年－1909年）清末洋务派首领，直隶省南皮县双庙村人，同治进士，曾任翰林院侍讲学士、内阁学士等职。1884年中法战争时，由山西巡抚升两广总督。1889年调任湖广总督，开办汉阳铁厂和湖北枪炮厂，设立纺纱、织布、缫丝、制麻四局，创办两湖书院，筹办芦汉铁路。1894年为两江总督。1907年任军机大臣。著有《张文襄公全集》。
*[[冯国璋|冯国璋]]（1859年－1919年），字华甫，直隶河间（今河北河间）人，直系军阀，毕业于北洋武备学堂，曾任北洋步兵学堂总办兼督练营务处总办。1903年中央练兵处任军学司正使。辛亥革命时率领北洋军镇压武昌起义。后出任江苏都督，坐镇东南。袁世凯死后，黎元洪继任大总统，经过国会补选冯为副总统，在南京办公。后黎元洪与段祺瑞府院之争，引发张勋复辟，为段祺瑞所镇压。黎元洪辞职，冯国璋进京任代理总统。
*[[荀慧生|荀慧生]]（1900年1月5日—1968年12月26日），祖籍[[河北|河北]][[东光|东光]]。中国著名[[京剧|京剧]]表演家。“[[京剧四大名旦|四大名旦]]”之一。荀慧生初名秉超，后改名秉彝，又改名“词”，字慧声，号留香，艺名白牡丹，与[[余叔岩|余叔岩]]合演《打渔杀家》起，改用荀慧生的名字。
*[[马本斋|马本斋]]（1903年－1944年） 回族，献县东辛庄人，出生于一个贫苦的农民家庭。1938年，在家乡组织回民义勇军，抗击日本侵略者，同年与进驻河间的中共河北游击军取得联系，组织回民教导总队，马本斋担任总队长。1939年7月，回民教导总队改编为八路军三纵队回民支队，马本斋任司令员。1938年初组织抗日队伍到1944年，六年多的时间里，率部驰骋在冀中平原和鲁西北地区，身经大小战斗五六百次，沉重地打击了日本侵略军。
*[[孙悦_(篮球运动员)|孙悦 (篮球运动员)]] (1985年)
*著名大學校長及教授[[郭位|郭位]]祖籍[[河北|河北]][[滄州|滄州]]。
