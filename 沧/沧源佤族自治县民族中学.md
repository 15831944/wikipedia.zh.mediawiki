'''沧源佤族自治县民族中学'''简称'''沧源民族中学'''（[[佤语|佤语]]：Qang Yuan Min Jux Jong Sioux）或'''沧源民中'''，是[[云南省|云南省]][[临沧市|临沧市]][[沧源佤族自治县|沧源佤族自治县]]唯一的一所完全中学，2000年被评定为云南省二级一等完全中学，位于[[勐董镇|勐董镇]]。现有校园面积220余亩，校舍面积26,503平方米。学校共有教职工207人， 50个教学班，其中高中38个班，初中12个班，在校学生2,720人，[[佤族|佤族]]学生占78℅。<ref>{{cite web |url=https://www.liuxue86.com/a/1956886.html |title=云南省沧源县民族中学(县一中)2014年公开招聘应届优秀大学毕业生 公 告 |last= |first= |date=2014-03-17 |website=www.liuxue86.com |publisher=出国留学网 |access-date=2017-12-09 |quote=}}</ref><ref>{{cite web |url=http://cymz.30edu.com.cn/Article/154b7bad-07cf-43e5-918b-7bd674e655ca.shtml |title=沧源佤族自治县民族中学简介 |last= |first= |date=2008-10-14 |website=cymz.30edu.com.cn |publisher=沧源佤族自治县民族中学 |access-date=2017-12-09 |quote=}}</ref>

==历史==

1958年7月12日，创办'''沧源中学'''，最初与勐董小学共用校园（后勐董小学迁出）。1961年，第一届初中毕业生共15人毕业。1966至1969年响应[[文化大革命|文化大革命]]运动，学校暂停招生，保持5个教学班。1971年8月，学校开始招收高中班。1981年8月，沧源中学改为'''省立民族中学'''。1985年由于实行“9年一贯制”教学，民族中学改名'''城区学校'''。<ref>{{cite book |author=沧源佤族自治县地方志编撰委员会 |date=1998年12月 |title=《沧源佤族自治县志》 |url= |location=[[昆明|昆明]] |publisher=云南民族出版社 |page=713 |isbn=7-5367-1498 |ignore-isbn-error=true |author-link= }}</ref>

2002年，沧源佤族自治县民族中学地理室编辑出版了《沧源地理》一书（ISBN 7-5416-1753-9）。

2010年5月1日，[[富滇银行|富滇银行]]出资110万元，在校内建成“富滇科技楼”，包括学生生物实验室、物理实验室、化学实验室和电话教育室。<ref>{{cite web |url=http://www.fudian-bank.com/html/fddt/content/4471.html |title=“富滇科技楼”揭牌仪式在临沧市沧源县民族中学举行 |last= |first= |date=2010-06-09 |website=www.fudian-bank.com |publisher=[[富滇银行|富滇银行]] |access-date=2017-12-08 |quote=}}</ref>

==助学金==

沧源佤族自治县民族中学部分学生能够享受国家助学金，助学金分发给一等资助学生和二等资助学生两个等次，2014年前者每人每年可获得2,000元国家助学金，后者为1,000元。<ref>{{cite web |url=http://220.164.135.131/Jrlc/Qxkx/Cyx/201403/50667.html |title=沧源自治县民族中学普通高中学生享受国家助学金90.9万元 |last= |first= |date=2014-03-26 |website=临沧市政府公众信息网 |publisher=临沧市人民政府办公室 |access-date=2017-12-08 |quote= }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
