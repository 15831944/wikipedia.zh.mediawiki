'''敵友識別系統'''、'''敵我辨識系統'''（{{lang-en|Identification Friend or Foe}}、{{abbr|IFF|Identification Friend or Foe}}）是為了指揮和控制而設計的識別系統。能使得軍隊和國家（民用空中交通管制）的訊問系統能夠識別飛機、車輛或友軍部隊，並確定受詢問方的方位和位置。軍用和民用飛機都可以使用敵友識別系統。

== 北大西洋公約組織 ==
[[北大西洋公約組織|北大西洋公約組織]]將在2020年以前採用Mode 5標準的敵友辨識系統，[[英國|英國]]採用系統廠商為李奧納多·亨索爾特公司相容於Mode 5標準的產品：M428飛機、直升機與航艦詢答機、SIT 2010密碼電腦，與MSSR 2000 I、MSSR 1000 I詢問器。<ref name = "英軍將改用Mode 5敵我識別系統_2017"> {{ cite web | title = 英軍將改用Mode 5敵我識別系統 | url = https://www.ydn.com.tw/News/242448 | publisher = 青年日報 | author = 崔敬熙 | language = zh-tw | date = 2017-06-25 }} </ref><ref name = "Leonardo in the UK_2017"> {{ cite web | title = Leonardo in the UK | url = http://www.uk.leonardocompany.com/-/leonardoiff | language = en | date = 2017-03-21 }} </ref>
