'''魏-{謩}-'''<ref name=BT176>[[:s:舊唐書/卷176|《旧唐书》卷一百七十六]]</ref>，一作'''魏谟'''<ref name=NBT97>[[:s:新唐書/卷097|《新唐书》卷九十七]]：徵五世孙谟。</ref>（{{bd|793年||858年|catIdx=W}}），[[表字|字]]'''申之'''，[[唐朝|唐朝]]官员，[[唐宣宗|唐宣宗]]年间拜为[[宰相|宰相]]。

== 家世 ==
魏谟生于[[唐德宗|唐德宗]][[贞元_(唐朝)|贞元]]九年（793年），为唐初[[唐太宗|唐太宗]]传奇名相[[魏徵|魏徵]]五世孙。曾祖[[魏殷|魏殷]]为[[汝阳|汝阳]]令，祖父[[魏明|魏明]]为县令，父[[魏冯|魏冯]]或[[魏凭|魏凭]]<ref>{{cite web |url=http://www.sidneyluo.net/a/a17/072.htm |title=存档副本 |accessdate=2008-10-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20081120085821/http://www.sidneyluo.net/a/a17/072.htm |archivedate=2008-11-20 }}[http://www.sidneyluo.net/a/a17/table/form57.htm 《新唐书》卷七十二] {{webarchive|url=https://web.archive.org/web/20100620180729/http://sidneyluo.net/a/a17/table/form57.htm |date=2010-06-20 }}</ref>为献陵台令。<ref name=BT176/>

== 唐文宗年间 ==
[[唐文宗|唐文宗]][[太和_(唐朝)|太和]]七年（833年），魏谟通过[[进士|进士]][[科举|科举]]。[[杨汝士|杨汝士]]任[[同州|同州]]刺史时（834年—835年），邀请魏谟任防御判官、同州长春宫巡官、秘书省校书郎。后杨汝士被召回[[长安|长安]]，九年（835年）十月，文宗读《[[贞观政要|贞观政要]]》，感念魏征贤明，下诏访求其后人，杨汝士就推荐魏谟任[[右拾遗|右拾遗]]。<ref>《[[全唐文|全唐文]]》</ref>文宗得知他是魏徵之后，待他异于旁人。宰相[[李固言|李固言]]、[[李珏|李珏]]、[[杨嗣复|杨嗣复]]也都赏识他。<ref name=BT176/><ref name=CFYG>《[[册府元龟|册府元龟]]》</ref>

前[[邕管经略使|邕管经略使]][[董昌龄|董昌龄]]枉杀录事参军衡方厚，坐贬[[溆州|溆州]]司户。[[开成|开成]]元年（836年）正月，改[[硖州|硖州]]刺史，魏谟上疏：“王者施恩赦有罪，但故意杀人是无赦的。昌龄以前因微功被授方镇，不能谨慎于荣宠，却恣其狂暴，无辜专杀，事迹显彰。受害人妻儿衔冤，万里诉讼。等审讯伏罪，还让他活下来，中外议论，以为用法不够。今若再授之牧守之职，以理疲人，则杀人者被升官，而冤苦者如何伸冤？”<ref>[[:s:论董昌龄量移硖州刺史疏|:s:论董昌龄量移硖州刺史疏]]</ref>于是改董昌龄为[[洪州_(隋朝)|洪州]]别驾。<ref name=BT176/>

先前太和九年（835年）文宗与近臣[[李训|李训]]、[[郑注|郑注]]图谋屠杀当权宦官的计划[[甘露之变|甘露之变]]事败后，李训下属[[御史中丞|御史中丞]]宗室[[李孝本|李孝本]]伏诛，家属充军为奴，但文宗命将李孝本二女带入宫中，人们以为文宗要纳她们为妃。开成元年（836年）七月，魏谟上表称文宗应该三思避开做下儒家反对的[[内婚制|内婚]]行为的嫌疑。<ref>[[:s:谏纳李孝本女疏|:s:谏纳李孝本女疏]]</ref>文宗得表，立刻送李孝本女出宫，升魏谟为[[右补阙|右补阙]]，<ref>[[:s:答魏谟谏疏诏|:s:答魏谟谏疏诏]]</ref>命中书省作诏书大赞魏谟，将其与先祖魏徵相比。<ref>[[:s:資治通鑑/卷245|《资治通鉴》卷二百四十五]]：李孝本二女配没右军，上取之入宫。秋，七月，右拾遗魏谟上疏，以为：“陛下不迩声色，屡出宫女以配鳏夫。窃闻数月以来，教坊选试以百数，庄宅收市犹未已；又召李孝本女入宫，不避宗姓，大兴物论，臣窃惜之。昔汉光武一顾列女屏风，宋弘犹正色抗言，光武即撤之。陛下岂可不思宋弘之言，欲居光武之下乎！”上即出孝本女。擢谟为补阙，曰：“朕选市女子，以赐诸王耳。怜孝本女宗枝髫龀孤露，故收养宫中。谟于疑似之间皆能尽言，可谓爱我，不忝厥祖矣！”命中书优为制辞以赏之。谟，征之五世孙也。</ref>

教坊副使云朝霞善吹笛，新声变律，深得文宗意旨，从左骁卫将军得授兼扬府司马。宰相们奏：“扬府司马品级高，是郎官刺史所任，不可授伶官。”文宗意欲授之，趁与宰相们对话时称云朝霞的好处。魏谟闻之，多次上疏陈论，文宗才改授云朝霞为[[润州|润州]]司马。[[荆南|荆南]]监军使吕令琮的从人擅入江陵县毁骂县令韩忠，观察使[[韦长|韦长]]将诉状送到[[枢密使|枢密使]]处。魏谟上疏称韦长没有便宜处理，也不将诉状送到门下省，违反常典，应该处理，希望惩戒。<ref>[[:s:论吕令琮毁骂江陵县令疏|:s:论吕令琮毁骂江陵县令疏]]</ref>但此疏不被受理，时论以为可惜。<ref name=BT176/>

二年（837年）五月，[[李德裕|李德裕]]代[[牛僧孺|牛僧孺]]为[[淮南节度使|淮南节度使]]，牛僧孺将军事交付副使[[张鹭|张鹭]]后离任。淮南府钱八十万缗，李德裕奏称只有四十万，被张鹭用了一半。牛僧孺对文宗称冤，魏谟与同僚补阙[[王绩|王绩]]、谏官[[姚合|姚合]]等都弹劾李德裕挟私怨诬告牛僧孺，文宗诏李德裕核实，发现牛僧孺留下的钱确实是八十万缗。李德裕称诸镇被代任时按例以半数备水旱、助军费，自己刚到任时染病，不知道这个惯例，被下属官吏欺瞒，待罪请求处罚。魏谟、王绩与补阙[[崔谠|崔谠]]、[[韦有翼|韦有翼]]、拾遗[[令狐綯|令狐綯]]、[[韦楚老|韦楚老]]、[[樊宗仁|樊宗仁]]等上疏弹劾李德裕，但李德裕最终受到包庇被下诏赦免。<ref>[[:s:旧唐书/卷174|《旧唐书》卷一百七十四]]</ref><ref>[[:s:新唐书/卷180|《新唐书》卷一百八十]]</ref>

三年（838年）正月，文宗以魏谟论事忠切有魏征之风，不顾提拔月限，提拔魏谟为[[起居舍人|起居舍人]]。魏谟面圣致谢，文宗得知他家的旧诏书都丢了，只剩下簪笏，就命他献上先祖魏徵的[[笏|笏]]板。宰相[[郑覃|郑覃]]说：“在人不在笏。”文宗答：“我这是遵循《[[甘棠|甘棠]]》的意旨。”魏谟将要退下，文宗又召诫之：“事有不当，即须奏论。”魏谟说：“臣为谏官时，应该规劝进谏。今居史职，职在记录言论，臣不敢逾越职分。”文宗说：“只要两省官一起论事，不要拘于此言。”杨嗣复想提拔[[衡州|衡州]]司马[[李宗闵|李宗闵]]，文宗得知后也对宰相们说想任李宗闵以官，郑覃与另一宰相[[陈夷行|陈夷行]]反对，并与杨嗣复互相指责对方结党。文宗说不妨任李宗闵为州刺史。郑覃等退下后，文宗对起居郎[[周敬复|周敬复]]和魏谟说：“宰相喧争如此，可以吗？”答：“确实不可以。但郑覃等尽忠愤激，自己没察觉罢了。”二月，李宗闵终被用为[[杭州|杭州]]刺史。<ref name=ZZTJ246>[[:s:資治通鑑/卷246|《资治通鉴》卷二百四十六]]</ref>不久，魏谟以本官直弘文馆。<ref name=BT176/>四月，文宗于阁内召魏谟问魏征政绩，访求魏征后裔，以邓州南阳县尉[[魏可则|魏可则]]为栎阳县尉。<ref name=CFYG/>

四年（839年），魏谟被授[[谏议大夫|谏议大夫]]，仍兼起居舍人，<ref>[[:s:授魏谟谏议大夫制|:s:授魏谟谏议大夫制]]</ref>判弘文馆事。魏谟在起居注中记录下[[华州|华州]][[镇国军|镇国军]]防御使[[陈夷行|陈夷行]]论[[忠武军节度使|忠武军节度使]][[王茂元|王茂元]]所请不许与吐蕃交马事。<ref>[[:s:为濮阳公上陈相公状|李商隐《为濮阳公上陈相公状》]]</ref>十月，文宗派中使向魏谟索看起居注，称可检视自己的行为。魏谟指出史家据实修史，文宗若有善事，不需要怕他不记录；如有错，即使他不写，天下之人也写，他以文宗为文皇帝（李世民），文宗也当比他如[[褚遂良|褚遂良]]。<ref>[[:s:请不取注记奏|:s:请不取注记奏]]</ref>文宗又说自己曾取看起居注。魏谟说那是以前的史官不守职分，但他不敢陷文宗于非法。若文宗看了起居注，史官便要兼顾皇帝对史家对关于他们的记录所持态度的好恶，就无法据实记录了，无以取信于后人。文宗没再坚持。<ref name=BT176/><ref name=ZZTJ246/>

魏谟曾奏请令判官推劾诉事、<ref>[[:s:请令判官推劾诉事奏|:s:请令判官推劾诉事奏]]，但该奏是在哪位皇帝任内，待考。</ref>奏请放朝期间如公事繁忙可继续办公，<ref>[[:s:权放一两月朝参奏|:s:权放一两月朝参奏]]，但该奏是在哪位皇帝任内，待考。</ref>也曾与[[高元裕|高元裕]]等论奏请重审妖人[[贺兰进兴|贺兰进兴]]及其党羽。<ref>[[:s:请将贺兰进兴等重付台司覆勘疏|:s:请将贺兰进兴等重付台司覆勘疏]]</ref><ref>[[:s:令台司覆勘贺兰进兴党与诏|:s:令台司覆勘贺兰进兴党与诏]]</ref>

== 唐武宗年间 ==
五年（840年），文宗崩，弟[[唐武宗|唐武宗]]继位。随即李珏、杨嗣复被罢相，其政敌李德裕成为首相。<ref name=ZZTJ246/>[[会昌_(唐朝)|会昌]]元年（841年），魏谟还曾作为使者出使[[回鹘|回鹘]]，<ref>[[:s:赐背叛回鹘敕书|李德裕《赐背叛回鹘敕书》]]</ref>但因得宠于李、杨，也被遣出京城任[[汾州|汾州]]刺史。后李、杨被进一步贬谪，魏谟也被贬为[[信州_(江西)|信州]]长史。<ref name=BT176/>

== 唐宣宗年间 ==
六年（846年），武宗崩，皇叔唐宣宗继位，李德裕因此失势，[[白敏中|白敏中]]成为首相。<ref>[[:s:資治通鑑/卷248|《资治通鉴》卷二百四十八]]：甲子，上崩。……丁卯，宣宗即位。宣宗素恶李德裕之专，……壬申，以门下侍郎、同平章事李德裕同平章事，充荆南节度使。德裕秉权日久，位重有功，众不谓其遽罢，闻之莫不惊骇。……以翰林学士、兵部侍郎白敏中同平章事。</ref>随后魏谟被提升为[[郢州_(西魏)|郢州]]刺史，[[大中_(唐朝)|大中]]元年（847年）又被提拔到离京城更近的[[商州|商州]]为刺史。二年（848年），被召回长安任[[给事中|给事中]]，迁[[御史中丞|御史中丞]]，谢日面赐金紫之服。在任上上表弹劾宣宗姐妹夫驸马都尉[[杜中立|杜中立]]贪赃，<ref>[[:s:新唐書/卷083|《新唐书》卷八十三]]：真源公主，始封安陵。下嫁杜中立。</ref>权贵震动且害怕他。后他兼任[[户部侍郎|户部侍郎]]，判本司事。他上表指出既监督财货又任御史中丞不合适，在他的请求下，他被免除了御史中丞一职。<ref name=BT176/>

五年（851年）十月，宣宗授魏谟[[同中书门下平章事|同中书门下平章事]]，拜为宰相，仍判户部。<ref name=NBT8>[[:s:新唐书/卷8|《新唐书》卷八]]</ref><ref>[[:s:魏謩拜相制|沈询《魏謩拜相制》]]</ref><ref>《旧唐书·宣宗纪》作五月事。《册府元龟》作四年（850年）十一月事。</ref>当时，唐宣宗四十二岁，相对而言已是高龄，仍未立[[太子|太子]]，群臣慑于政治敏感性，也不敢提及。当魏谟面圣感谢任命时，说自己没有[[夔|夔]]、[[契|契]]之才却担当夔、契之任，并趁机指出如今海内无事却未立太子及以正人辅导是他对宣宗为政的主要担忧，并且哭了。宣宗也有所感动。虽然此后宣宗仍然未立太子，但时人都被魏谟感动了。<ref name=ZZTJ249>[[:s:資治通鑑/卷249|《资治通鉴》卷二百四十九]]：戊辰，以户部侍郎魏谟同平章事，仍判户部。时上春秋已高，未立太子，群臣莫敢言。谟入谢，因言：“今海内无事，惟未建储副，使正人辅导，臣窃以为忧。”且泣。时人重之。    河东节度使李业纵吏民侵掠杂虏，又妄杀降者，由是北边扰动。闰月，庚子，以太子少师卢钧为河东节度使。业内有所恃，人莫敢言，魏谟独请贬黜。上不许，但徙义成节度使。……    二月，辛巳，以门下侍郎、同平章事魏谟同平章事，充西川节度使。谟为相，议事于上前，它相或委曲规讽，谟独正言无所避。上每叹曰：“谟绰有祖风，我心重之”。然竟以刚直为令狐綯所忌而出之。</ref>不久，兼任集贤殿大学士。[[詹毗国|詹毗国]]进献大象，魏谟认为大象的性格不能安居中原，请求还给使者，宣宗同意了。<ref name=BT176/>

六年（852年）二月，右卫大将军[[郑光|郑光]]请免所赐田租税。魏谟奏：“郑光以国舅之亲，赐田是可以的，免税就不能让百姓心服了。”宣宗下敕按例供税。<ref name=BT18>[[:s:旧唐书/卷18下|《旧唐书》卷一十八下]]</ref>[[河东节度使|河东节度使]][[李业|李业]]放纵吏民劫掠胡人，滥杀投降胡人，致使北疆不宁。但李业在朝中有靠山，未被处罚。闰四月，只有魏谟敢于弹劾李业，要求完全罢免其职务；宣宗没有照做，只是派太子少师[[卢均|卢均]]取代李业，调李业至[[义成|义成]]。<ref name=ZZTJ249/>魏谟被加[[中书侍郎|中书侍郎]]。大理卿[[马曙|马曙]]被从人[[王庆|王庆]]告发私藏兵甲，马曙因此被贬官，王庆却无罪。魏谟引用法律，杖杀王庆。<ref name=BT176/>

魏谟进阶[[银青光禄大夫|银青光禄大夫]]，兼礼部尚书，以银青光禄大夫、守中书侍郎、兼礼部尚书、同中书门下平章事、集贤殿大学士、[[上柱国|上柱国]]被授监修国史，<ref>[[:s:授令狐绹太清宫使魏謩监修国史裴休集贤殿大学士制|:s:授令狐绹太清宫使魏謩监修国史裴休集贤殿大学士制]]</ref>负责监修文宗实录，八年（854年）三月修成四十卷并呈上，<ref name=SS203>[[:s:宋史/卷203|《宋史》卷二百零三]]</ref>因此和下属给事中[[卢耽|卢耽]]、太常少卿[[蒋偕|蒋偕]]、司勋员外郎[[王沨|王沨]]、右补阙[[卢吉|卢吉]]、膳部员外郎[[牛丛|牛丛]]都获颁赐银器、锦彩有差，这些下属们也得到升迁。七月，以银青光禄大夫、守[[门下侍郎|门下侍郎]]、同平章事兼户部尚书。魏谟仪容魁伟，其他宰相在宣宗面前议政时，为了不引起皇帝不悦，都委曲进言，只有魏谟敢直言。宣宗常说：“魏谟有祖上的风范，我很敬重他。”但魏谟也因此为首相[[令狐绹|令狐绹]]所忌。十一年（857年）二月，他被遣出长安为检校户部尚书、同平章事，兼成都尹，充剑南西川节度事大使、知节度副、管内观察处置统押近界诸蛮及西山八国云南安抚等使。<ref name=NBT8/><ref name=ZZTJ249/><ref name=BT18/><ref>[[:s:授魏謩西川节度使平章事制|:s:授魏謩西川节度使平章事制]]</ref>十二年（858年），魏谟染病，请求接任。<ref>《两唐书》本传及《全唐文》作十年出镇西川，《旧唐书》作十一年求代，《[[册府元龟|册府元龟]]》卷三百三十二同，疑误。《[[唐大诏令集|唐大诏令集]]》卷五十四《魏謩西川节度平章事制》可证出镇西川事在十一年二月。</ref>宣宗准备召他回长安任[[吏部尚书|吏部尚书]]，但魏谟称病，只求闲职，因而被任为检校[[右仆射|右仆射]]、太子少保。<ref>但当时仍然没有太子，这完全是荣誉职位。</ref>同年十二月，魏谟卒，赠[[司徒|司徒]]。<ref name=BT176/>

== 轶闻 ==
《[[唐摭言|唐摭言]]》载：太和初年[[李回|李回]]任京兆府参军时，不送刚中进士的魏谟，魏谟很记恨。会昌年间李回为刑部侍郎，魏谟为御史中丞，提起当初李回不送魏谟之事，李回说如今也不送，魏谟更愤恨。李回被贬为[[建州_(唐朝)|建州]]刺史，魏谟拜相，李回的上疏魏谟都不采纳。后来李回将一衙官杖责停职，衙官愤恨自己因停职损失收入，逃到京城找宰相喊冤，宰相们都不问，旁人告知魏谟素与李回有隙，可找魏谟。正好魏谟的导骑从中书省出来，衙官望尘而拜，导骑问他何事，他自称建州百姓诉冤。魏谟闻之，倒持塵尾，看到衙官的状子论事二十余件，第一件是李回娶同姓女入门。当时李回已迁[[邓州|邓州]]刺史，行到[[九江|九江]]，被御史问罪，只得回建州，最终因罪被贬为[[抚州|抚州]]司马，卒于任上。

《北梦琐言》载：宰相[[夏侯孜|夏侯孜]]尤其厌恶名家子弟[[薛保逊|薛保逊]]，薛保逊被贬为[[澧州|澧州]]司马，长达七年得不到代任。夏侯孜罢相出镇，改由魏谟拜相后，薛保逊才得到征拜，但已在任所去世。魏谟并未在夏侯孜之后拜相，疑误。
