'''魏国运'''（{{bd|1914年|5月8日|2017年|6月30日}}）[[湖北省|湖北省]][[监利县|监利县]]人，[[中国人民解放军开国少将|中国人民解放军开国少将]]<ref name=shishi>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1722515 |title = 开国少将再陨一员：103岁原武汉军区空政主任魏国运逝世 |author =  |date = 2017-07-01 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref name=zhibu>{{Cite web |url =http://www.rmzxb.com.cn/c/2016-09-23/1049895.shtml  |title = 魏国运：103岁的支部书记 |author =  |date = 2016-09-23 |publisher = 人民政协网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1914年5月8日，魏国运出生在监利县一个贫农家庭。幼年读过两年[[私塾|私塾]]，曾给地主放牛，还曾乞讨，学木匠，当裁缝。1930年，魏国运参加[[中国共产党|中国共产党]]领导的打土豪、分田地革命活动，并配合游击队攻打国民党国防队。1931年，到[[中国工农红军|红军]]被服工厂当青工，同年在工厂加入[[中国共产主义青年团|中国共产主义青年团]]。1932年参加[[中国工农红军|中国工农红军]]，同年加入[[中国共产党|中国共产党]]。先后在[[红三军|红三军]]九师警卫营、八师二十四团、七师二十一团当警卫员、班长<ref name=shishi/><ref name=zhibu/>。

1935年，魏国运随[[红二方面军|红二方面军]]参加[[长征|长征]]，任四师十团党总支书记。1936年4月底，红二方面军占领[[金沙江|金沙江]][[石鼓镇_(玉龙县)|石鼓镇]]渡口，魏国运所在团掩护主力渡过金沙江<ref name=shishi/>。

[[抗日战争|抗日战争]]时期，魏国运历任[[八路军|八路军]]前方总指挥部参谋处参谋，[[中国人民抗日军政大学|中国人民抗日军政大学]]第一分校队长，[[八路军|八路军]]前方总指挥部参谋股长、科长，军委总参谋部一处五科副科长。在八路军前方总指挥部工作期间，魏国运参加了[[平型关战斗|平型关战斗]]和[[百团大战|百团大战]]<ref name=shishi/>。1938年，魏国运所在的八路军总部司令部决定成立一个党支部，魏国运当选为支部委员兼党小组长，[[朱德|朱德]]被编在该小组<ref name=zhibu/>。

[[第二次国共内战|第二次国共内战]]时期，魏国运历任[[中原军区|中原军区]]司令部参谋处副处长、[[江汉军区|江汉军区]]鄂中军分区副司令员兼参谋长、[[湖北省军区|湖北军区]]荆州军分区副司令员兼参谋长<ref name=shishi/>。

[[中华人民共和国|中华人民共和国]]成立后，魏国运任湖北军区荆州军分区副政治委员，空军师政治委员，[[广州军区空军|广州军区空军]]指挥所政治部主任，[[武汉军区空军|武汉军区空军]]政治部主任等职。1961年晋升少将军衔<ref name=shishi/>。1983年7月离休。1986年7月16日，魏国运所在干休所成立第一党支部，首任书记是[[杨思禄|杨思禄]]，魏国运任组织委员。后来杨思禄身体不好，魏国运接替杨思禄担任该干休所党支部书记。2014年，魏国运被[[中国人民解放军总政治部|中国人民解放军总政治部]]评为“全军先进离休干部”<ref name=zhibu/>。

2017年6月30日，魏国运在[[北京|北京]][[中国人民解放军空军总医院|中国人民解放军空军总医院]]病逝，享年103岁<ref name=shishi/>。
