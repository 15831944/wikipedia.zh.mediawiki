'''魏霸'''{{BD|？年||？年||catIdx=W}}，字'''喬卿'''，[[濟陰郡|濟陰郡]][[句陽縣|句陽縣]]（今[[山東省|山東省]][[菏澤市|菏澤市]]）人。家中世代都懂禮儀，<ref>《[[:s:後漢書/卷25|後漢書·魏霸傳]]》：魏霸字喬卿，濟陰句陽人也。世有禮義。</ref>官至[[東漢|東漢]][[光祿大夫|光祿大夫]]、[[長樂衛尉|長樂衛尉]]。
== 生平 ==
魏霸小時候喪親，[[兄弟|兄弟]]同居，州裡的人們都很羨慕他們兄弟感情和睦。<ref>《[[:s:後漢書/卷25|後漢書·魏霸傳]]》：霸少喪親，兄弟同居，州裏慕其雍和。</ref>

[[建初_(東漢)|建初]]年間，魏霸被舉薦爲[[孝廉|孝廉]]，八次遷官，[[漢和帝|漢和帝]]時爲[[鉅鹿|鉅鹿]]太守。爲政簡樸寬恕。[[掾史|掾史]]有過錯，魏霸先教誨其過失，不願更改的才罷免職務。有些官吏向魏霸闡述其他官吏的壞話，魏霸總是表彰官吏的優點，始終不說他們的缺點，毀訴的人都感到慚愧，於是這種誣告的風氣漸漸平息。<ref>《[[:s:後漢書/卷25|後漢書·魏霸傳]]》：建初中，舉孝廉，八遷，和帝時為巨鹿太守。以簡樸寬恕為政。掾史有過，霸先誨其失，不改者乃罷之。吏或相毀訴，霸輒稱它吏之長，終不及人短，言者懷慚，譖訟遂息。</ref>

[[永元_(漢)|永元]]十六年（公元104年），魏霸被任命為[[將作大匠|將作大匠]]。隔年（公元105年），[[漢和帝|漢和帝]][[駕崩|駕崩]]，魏霸掌管建造[[漢慎陵|順陵]]。當時天寒地凍，朝廷派來使者督責催促，經常懲罰縣吏以厲魏霸。魏霸只有安撫而已，並不責怪下位的人，還安慰他們說現在諸位被侮辱，都是他的過錯。官吏們對魏霸的恩德心存感激，全都盡全力加快施工進度。<ref>《[[:s:後漢書/卷25|後漢書·魏霸傳]]》：永元十六年，征拜將作大匠。明年，和帝崩，典作順陵。時盛冬地凍，中使督促，數罰縣吏以厲霸。霸撫循而已，初不切責，而反勞之曰：『令諸卿被辱，大匠過也。』吏皆懷恩，力作倍功。</ref>

[[延平_(漢殤帝)|延平]]元年（公元106年），魏霸替代[[尹勤|尹勤]]爲[[太常|太常]]。隔年（公元107年），魏霸因病請求[[致仕|致仕]]，後被任命為光祿大夫。<ref>《[[:s:後漢書/卷25|後漢書·魏霸傳]]》：延平元年，代尹勤為太常。明年，以病致仕，為光祿大夫。</ref><ref>《[[:s:後漢紀/卷14|後漢紀·卷十四]]》：後拜太常，以病致仕，為光祿大夫。</ref>[[周章_(東漢)|周章]]接任太常。<ref>《[[:s:後漢書/卷33|後漢書·周章傳]]》：代魏霸為太常。</ref>

[[永初_(漢)|永初]]五年（公元111年），魏霸被任命為長樂衛尉，以病請求辭職，後來複任光祿大夫，在任內逝世。<ref>《[[:s:後漢書/卷25|後漢書·魏霸傳]]》：永初五年，拜長樂衛尉，以病乞身，復為光祿大夫，卒於官。</ref>

== 軼事 ==
魏霸任鉅鹿太守時，魏霸的妻和子沒有在[[官舍|官舍]]居住，魏霸常想念兄嫂在家勤勞受苦，認為自己家則都在享樂。於是經常穿粗布衣服，不吃魚肉，妻子親自養[[蠶|蠶]]織布，兒子則親自耕田農作，魏霸兄弟和孩子們同甘共苦。鄉里的人們都很羨慕，受到魏霸一家感化。<ref>《[[:s:東觀漢記_(四庫全書本)/卷19|東觀漢記·魏霸傳]]》：魏霸，字喬卿，為鉅鹿太守，妻子不到官舍。常念兄嫂在家勤苦，己獨專樂，故掌服癞糲，不食魚肉之味，婦親蠶桑，服機杼，子躬耕農，與兄弟子同苦樂，不得有異。鄉里慕其行，化之。</ref>

魏霸任光祿大夫時。魏霸的妻子去世，其兄為照顧弟弟為他挑選一位妻子，送到官舍。魏霸笑說他兒子都有了，還有必要養一位婦女？隨即入拜其妻，手奉案前。跪說：「夫人視老夫復何中空，而空失計義，不敢讓您有所委屈。」即拜而出。妻子感到羞慚，請求離去，於是送還。<ref>《[[:s:太平御覽/0542|太平御覽·卷五百四十二]]》：魏霸爲光祿大夫。霸妻死，長兄伯爲霸娶妻，送至官舍。霸笑曰：「兒子備具，何空養他家姥爲？」即自入拜其妻，手奉案前。因跪曰：「夫人視老夫復何中直而空失計義，不敢相屈。」即拜而出。妻慚，求去，遂送還之。</ref>

《[[:s:太平御覽/0512|太平御覽]]》引[[謝承|謝承]]《後漢書》言魏霸字嶠卿，《[[:s:初學記_(四庫全書本)/卷17|初學記]]》引用謝承《後漢書》則是喬卿。[[范曄_(劉宋)|范曄]]《後漢書》亦是喬卿。
== 魏霸玉印 ==
魏霸[[玉印|玉印]]是漢朝玉印中的精品，也是現存漢朝形制最大的一方私印。尺寸：纵2.8cm，横2.9cm，高1.9cm 。現今館藏於[[上海博物館|上海博物館]]。<ref>[https://museum.eastday.com/shanghai%20museam/collection/seal/zhanpin/8.htm 上海博物館館藏魏霸玉印]</ref>
== 惠應廟 ==
魏霸去世後，村里的人們將他奉為[[土地神|土地神]]祭祀，<ref>《[[:s:吳興備志_(四庫全書本)/卷14|吳興備志]]》：及卒里人祀之為土神</ref>惠應廟位於[[新市鎮_(德清縣)|新市]]西郊古村梅林柏林坪，是為了紀念魏霸而建。<ref>《[[:s:吳興備志_(四庫全書本)/卷14|吳興備志]]》：恵應廟在德清縣東北梅林村祀漢鉅鹿太守魏覇神</ref>[[嘉泰|嘉泰]]三年（公元1203年），賜廟額惠應。
