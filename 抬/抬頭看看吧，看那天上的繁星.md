== 故事簡介 ==
就讀星野第一高中的男學生宙見曉斗雖然是天文社員但卻不觀星，因此被學校的學生稱為「不觀星的天文社員」。某一天曉斗收到了明光學院以天文社員交流為主的活動「六連星會」的邀請，原本不打算參加的曉斗，在一次的觀星活動中與昔日的青梅竹馬箒星光再次相遇，也讓曉斗再次找回觀星的樂趣<ref>{{Cite web|url=https://www.game-style.jp/soft/201511/06/01mia_pu.php|script-title=ja:『この大空に、翼をひろげて』のスタッフが贈る新作は、神秘的な星空の下で紡がれる甘く切ない青春群像劇！ 『見上げてごらん、夜空の星を』12月18日発売！|publisher=Game-Style|date=2015-11-06|accessdate=2018-04-22|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|宙見曉斗|宙見 暁斗（そらみ あきと）}}
: 就讀星野第一高中一年級，是天文社唯一的社員，因為不觀星而有「不觀星的天文社員」之稱。自称留在天文社的目的是为了住在天文部的活动室。
: 小时候常常与光、沙夜共同用父亲留下的望远镜观星，但在光离开后，不再观星。
: 目前在早乙女美晴家的便利店兼职以赚取生活费。
: 兴趣是登山，最喜欢在山顶喝速溶咖啡的感觉。
; {{nihongo|箒星光|箒星 ひかり（ほうきぼし ひかり）|聲：萌花ちょこ}}
: 就讀天之中高中一年級，曉斗的青梅竹馬，是个個性開朗且富有行動力的[[元气少女|元气少女]]，也是小學時在曉斗轉學進來時最先跟他成為朋友的人，在曉斗的介紹下開始對觀星產生興趣。小學畢業後搬離了村子，同時也與曉斗疏遠，但在六連星會的觀星活動上與曉斗再度相遇。
: 搬家後在天文網站上的投稿被曉斗看到，並化名「駒子」跟曉斗用通訊軟體互相交流，也因此知道六連星會的事。
: 小學畢業前知道沙夜要跟曉斗告白的事，為了讓曉斗接受沙夜的心意而騙曉斗說自己已經對觀星沒興趣了，也因此讓曉斗覺得被背叛，開始不再觀星。
; {{nihongo|天之川沙夜|天ノ川 沙夜（あまのがわ さや）|聲：雪菜めぐり}}
: 就讀靜泉女子學園一年級，曉斗的青梅竹馬。小學刚入学時對自己擁有[[虹膜異色症|蓝色和黄色的異色瞳]]感到厭惡，并曾戴墨镜来掩饰，但在被曉斗說自己的眼睛像是[[辇道增七|辇道增七]]一樣美麗後，開始不再抱有自卑感，並與曉斗和光一起觀星。长大后经常给曉斗送饭，被同学们戏称为“送饭妻”。个人路线结局成为了一名教师。
: 最喜欢的星星是“冬季的辇道增七”（即大犬座H3945，与辇道增七一样是美丽的天蓝-橙黄双星）。
; {{nihongo|白鳥織姬|白鳥 織姫（しらとり おりひめ）|聲：[[遠野そよぎ|遠野そよぎ]]}}
: 明光學院的三年級女學生，學院的前天文社長，也是六連星會的發起人。
: 家里很有钱，是个大小姐。
; {{nihongo|日下部可樂娜|日下部 ころな（くさかべ ころな）|聲：小鳥居夕花}}
: 小曉斗一歲的國中生，曉斗的友人日下部武一的妹妹，有着小恶魔般的性格。喜歡曉斗，經常對他積極採取攻勢，也因此对天之川沙夜抱有戒心。後來就讀明光學院一年級。

=== 其他角色 ===
; {{nihongo|早乙女美晴|早乙女 美晴（さおとめ みはる）|聲：優希芹果}}
: 曉斗小學時的班導師，是孩子们小时候十分憧憬的对象和天文启蒙教师，小學廢校後一直在家裡過著[[尼特族|尼特族]]生活。Fine Days中為可攻略角色。
; {{nihongo|吉岡帆乃香|吉岡 帆乃香（よしおか ほのか）|聲：白月かなめ}}
: 明光學院的一年級女學生，現任的天文社長，個性冷靜且一本正經。Fine Days中為可攻略角色。
; {{nihongo|山田陽南|山田 陽南（やまだ ひなみ）|聲：百瀨ぽこ}}
: 科羅娜的友人，對天文有興趣。後來就讀星野第一高中並加入天文社。經常說出奇特的發言。崇拜沙夜並稱她為輦道增七的公主。根据人气投票结果，Fine Days中為可攻略角色。
; {{nihongo|日下部武一|日下部 武一（くさかべ たけいち）|聲：須賀紀哉}}
: 星野第一高中的男學生，籃球社社員。孩子王，小學時因为經常恶作剧与晓斗不和，但长大后兩人成為了好友。
; {{nihongo|加納紀子|加納 紀子（かのう のりこ）|聲：羽鳥いち}}
: 天之中高中的天文社長，光的學姐。雖是天文社員，但對天文的事情並沒有很了解。
; {{nihongo|陣野鳴枝|陣野 鳴枝（じんの なるえ）|聲：[[川島梨乃|川島梨乃]]}}
: 天之中西高中的地學社社長。
; {{nihongo|田尻啓介|田尻 啓介（たじり けいすけ）|聲：增岡祐二}}
: 男子校諸美澤工業天文社的社長。
; {{nihongo|豐田|豊田（とよだ）|聲：夏村伊介}}
: 天之中西高中的地學社社員，陣野的學弟。興趣是[[天文攝影|天文攝影]]。
; {{nihongo|川中島|川中島（かわなかじま）|聲：小次狼}}
: 諸美澤工業天文社的社員，田尻的學弟。擅長創作跟星座有關的詩。
