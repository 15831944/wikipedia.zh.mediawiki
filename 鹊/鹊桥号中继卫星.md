'''鹊桥号中继卫星'''（又名'''嫦娥四号中继星'''）是一颗服务[[嫦娥四号|嫦娥四号]]的地月间通讯中继卫星。

== 发射入轨 ==
[[File:20180912_6258TPS-TPR-2018Q3-18-09-04-p14legacy.png|thumb]]的地月间通讯中继卫星。]]
鹊桥号中继卫星在2018年5月21日5点28分由[[长征四号丙运载火箭|长征四号丙运载火箭]]在[[西昌卫星发射中心|西昌卫星发射中心]]发射<ref>{{Cite web |url=http://www.xinhuanet.com/politics/2018-05/22/c_1122867606.htm |title=嫦娥四号中继星发射成功 人类迈出航天器月背登陆第一步 |accessdate=2018-05-26 |author=科技日报 |date=2018-05-22 |archive-url=https://web.archive.org/web/20180526190527/http://www.xinhuanet.com/politics/2018-05/22/c_1122867606.htm |archive-date=2018-05-26 |dead-url=no }}</ref>。这是西昌卫星发射中心首次发射长征四号丙运载火箭<ref>{{Cite web |url=http://zz.81.cn/content/2018-05/21/content_8037281.htm |title=首次发射CZ-4C火箭，西昌发射场干了这些事 |accessdate=2018-05-26 |author=军报记者 |date=2018-05-21 |archive-url=https://web.archive.org/web/20180526190516/http://zz.81.cn/content/2018-05/21/content_8037281.htm |archive-date=2018-05-26 |dead-url=no }}</ref>。

由于[[嫦娥四号|嫦娥四号]]将在月球背面着陆，而因为月球自转周期与绕地球公转周期相同，故嫦娥四号的通讯将会受到月球本身的阻挡。鹊桥号中继卫星将承担地球与嫦娥四号之间的通讯中继任务<ref name="ce4relay_xinhua">{{Cite web |url=http://pic.cnr.cn/pic/nativepic/20180521/t20180521_524241429.shtml |title=嫦娥四号中继星成功发射 将搭建地月“鹊桥” |accessdate=2018-05-26 |author=新华网 |date=2018-05-21 |archive-url=https://web.archive.org/web/20180526190634/http://pic.cnr.cn/pic/nativepic/20180521/t20180521_524241429.shtml |archive-date=2018-05-26 |dead-url=no }}</ref>。鹊桥号中继星是世界上首颗运行于地月拉格朗日L<sub>2</sub>点的通信卫星<ref>{{Cite web |url=http://china.cnr.cn/yaowen/20180521/t20180521_524240247.shtml |title=嫦娥四号中继通信卫星今晨发射 搭建地月信息联通“鹊桥” |accessdate=2018-05-26 |author=央广网 |date=2018-05-21 |archive-url=https://web.archive.org/web/20180526201402/http://china.cnr.cn/yaowen/20180521/t20180521_524240247.shtml |archive-date=2018-05-26 |dead-url=no }}</ref>。

鹊桥号中继卫星发射后进入近地点高度200公里，远地点高度40万公里的[[地月转移轨道|地月转移轨道]]<ref name="ce4relay_xinhua" />，2018年5月25日21时46分成功实施近月制动，进入月球至地月拉格朗日L<sub>2</sub>点的转移轨道<ref>{{Cite web |url=https://3w.huanqiu.com/a/a-XDHTCG545CC993A81ACE6A |title=“鹊桥”中继星成功实施近月制动 |accessdate=2018-05-26 |author=环球时报 |date=2018-05-25 |archive-url=https://web.archive.org/web/20180526192409/https://3w.huanqiu.com/a/a-XDHTCG545CC993A81ACE6A |archive-date=2018-05-26 |dead-url=no }}</ref>。随后再进行数次变轨，在2018年6月14日時进入环绕[[拉格朗日点|地月拉格朗日L<sub>2</sub>点]]的[[晕轮轨道|晕轮轨道]]。<ref>{{Cite news|first=Andrew|last=Jones|title=Queqiao update: Chang'e-4 lunar relay satellite establishing halo orbit after approaching Lagrange point|url=https://gbtimes.com/queqiao-update-change-4-lunar-relay-satellite-establishing-halo-orbit-after-approaching-lagrange-point|publisher=GB times|date=1 June 2018|accessdate=2018-06-14|language=en}}</ref><ref>{{Cite news|title=Relay satellite for Chang'e 4 lunar probe enters orbit|url=http://europe.chinadaily.com.cn/a/201806/14/WS5b221389a31001b8257222e6.html|newspaper=[[チャイナデイリー|チャイナデイリー]]|date=14 June 2018|accessdate=2018-06-14|language=en}}</ref>

== 设备 ==
鹊桥号中继卫星搭载了一口展开后直径达5米的伞状天线，是发射时人类深空探测器史上携带的最大口径通信天线<ref>{{Cite web |url=http://www.chinanews.com/gn/2018/05-21/8518942.shtml |title=鹊桥号发射成功 将成为世界首颗连通地月中继卫星 |accessdate=2018-05-26 |date=2018-05-21 |archive-url=https://web.archive.org/web/20180527070317/http://www.chinanews.com/gn/2018/05-21/8518942.shtml |archive-date=2018-05-27 |dead-url=no }}</ref>。其与地面之间的通信带宽为2M，与嫦娥四号着陆器和巡视器间的带宽在100k以上。<ref name="guokr-442961">{{cite web |title=鹊桥号启程，为嫦娥四号登陆月球背面架设通信桥梁 |url=https://www.guokr.com/article/442961/ |author=Lsquirrel |website=[[果壳网|果壳网]] |date=2018-05-20 |accessdate=2019-01-04 }}</ref>

伴随鹊桥号升空的科学仪器还有:
*荷兰－中国低频探测器（NCLE）和激光反射器。它由荷兰[[奈梅亨拉德伯德大学|奈梅亨拉德伯德大学]]和[[中国科学院|中国科学院]]开发，用于探测早期宇宙的[[宇宙微波背景|低频射电辐射]]。其观测任务包括绘制全天射电图、检测[[太阳风暴|太阳风暴]]和行星射电脉冲。<ref name="guokr-442961" />
*由中山大学研制的激光反射器，为空间探测引力波计划“[[天琴计划|天琴]]”做先导性的激光测距研究。<ref name="guokr-442961" />
== 搭載器具 ==
在發射時，[[哈爾濱工業大學|哈爾濱工業大學]]開發的兩個小型航天器也一同搭載。

這些衛星是50x50x40厘米，質量約45公斤，並且是3軸穩定的。兩個線性極化天線沿飛行方向安裝並垂直。

=== 龍江一號 ===
別名DSLWP-A1。DSLWP的月球編隊飛行任務是開展低頻射電天文、業餘無線電和教育方面的研究實驗，該任務由兩顆微型衛星組成。但龍江一號衛星發射不久後出現了故障，未能成功入軌。雖然一個[[射電天文衛星|射電天文衛星]]不能只用一架衛星建造，但未來的觀測計劃當時還未確定。<ref>{{Cite news|url=https://news.mynavi.jp/article/20180601-639992/|title=中国、月の裏側に通信衛星を打ち上げ - 年末には探査機も着陸|publisher=マイナビニュース|date=2018-06-01|accessdate=2018-06-19}}</ref>

{{fact|DSLWP-A1的下行鏈路的頻率為435.425 MHz和436.425 MHz。}}

=== 龍江二號 ===
別名DSLWP-A2。世界上第一顆小型衛星從繞地球軌道到達月球軌道的。它也是中國提出的[[一帶一路|一帶一路]]的一部分，是與[[沙特阿拉伯|沙特阿拉伯]]的國際合作項目。<ref>{{Cite news|url=http://jp.xinhuanet.com/2018-06/16/c_137257973.htm|title=宇宙の国際協力　中国小型衛星がサウジ製カメラで月を撮影|publisher=新華網日本語|date=2018-06-16|accessdate=2018-06-19}}</ref>

下行鏈路頻率435.400 MHz，使用10K0F1DCN或10K0F1DEN的下行鏈路為436.400 MHz。使用turbo碼或JT4G的250/500 bps GMSK。
