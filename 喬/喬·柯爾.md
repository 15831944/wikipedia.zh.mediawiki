'''喬·柯爾'''（{{lang-en|'''Jon Cor'''}}，{{bd|1984年|11月17日}}）是一名[[加拿大人|加拿大]]男演員，最著名於電視劇《[[闇影獵人_(電視劇)|闇影獵人]]》中飾演Hodge Starkweather。

== 早年 ==
柯爾在{{link-en|范莎學院|Fanshawe College}}修讀戲劇藝術。<ref name=writing>{{Cite web |url = https://lithiummagazine.com/2016/01/20/an-in-depth-interview-with-actor-jon-cor/ |title = An In Depth Interview With Actor Jon Cor |date = 2016-01-20 |publisher = Lithium Magazine |language = en}}</ref>

== 生涯 ==
他以飾演《{{link-en|比弗瀑布|Beaver Falls (TV series)}}》中的Jake、《{{link-en|孤岛生存大乱斗之生化危机|Total Drama: Revenge of the Island}}》的Brick、《{{link-en|重回昨日|Being Erica}}》的Zach Creed、《''The Boy She Met Online''》的Jake Byers<ref>{{Cite web |url = http://www.radiotimes.com/film/pp97f/the-boy-she-met-online |title = The Boy She Met Online |author = Sloan Freer |publisher = Radio Times |language = en}}</ref>和《[[闇影獵人_(電視劇)|闇影獵人]]》的Hodge Starkweather而著名<ref>{{Cite web |url = http://variety.com/2015/tv/news/shadowhunters-jon-cor-being-human-1201507541/ |title = ABC Family's 'Shadowhunters' Casts 'Being Human' Alum |author = Whitney Friedlander |date = 2015-05-28 |publisher = Variety |language = en}}</ref>。他曾在《[[奪魂鋸3D_(第7集)|奪魂鋸3D]]》中飾演Ryan<ref name=writing />。

=== 寫作 ===
柯爾寫過一本小說《''In Heat''》。<ref name=writing />
