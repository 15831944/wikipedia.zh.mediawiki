'''喬奧·卡里奧卡'''（José Carioca）是以[[鸚鵡|鸚鵡]]為原型，擬人化的[[迪士尼|迪士尼]]動畫的角色之一。

首次亮相為在1942在[[里約熱內盧|里約熱內盧]]首播（美國1943年、日本1957年播出）的動畫電影《[[致候吾友|致候吾友]]》。

== 概要 ==
來自[[里約熱內盧|里約熱內盧]]。

在《[[致候吾友|致候吾友]]》、《[[西班牙三紳士|西班牙三紳士]]》中，喬奧帶領著[[唐老鴨|唐老鴨]]介紹[[巴西|巴西]]的[[森巴|森巴]]和[[巴伊亞州|巴伊亞州]]等。

已成為[[巴西|巴西]]的代表角色，有喬奧為主角的連載漫畫《Zé Carioca》。而漫畫的設定之中有七位以巴西各地為原型，擬人化的[[侄子|侄子]]。

他常以開朗的紳士形象出場。（在跟[[唐老鴨|唐老鴨]]第一次見面時遞出[[名片|名片]]，或是在『[[米老鼠群星會|米老鼠群星會]]』中受邀教導[[高飛|高飛]]餐桌禮儀。）

常與[[班基德|班基德]]一同出場。

上海首映時的翻譯為'''喬鸚哥'''。

== 外貌 ==
整身的羽毛為綠色，眼睛是偏橘的紅色。

常穿著白色立領襯衫加上黑色[[領結|領結]]與奶油色西裝外套，常帶著[[平頂帽|平頂帽]]與黑[[傘|傘]]。

不管在哪個作品中都常吸咄著[[雪茄|雪茄]]，第一次在《[[致候吾友|致候吾友]]》中登場時有描寫著喬奧用[[火柴|火柴]]盒敲打出[[森巴|森巴]]旋律的畫面。

== 關於別名 ==

在英語圈生活的[[唐老鴨|唐老鴨]]常常暱稱喬奧為「喬」，或是跟[[葡萄牙語|葡萄牙語]]接近的「喬傑」。

[[西班牙語|西班牙語]]為母語的[[班基德|班基德]]則是從頭到尾都稱呼他為「喬奧」。

不過，2001年英文版的《[[米老鼠群星會|米老鼠群星會]]》中則只被稱呼為「喬奧」，有可能在近幾年統一稱呼。

漫畫中的名字「Zé」是「喬奧（José）」在葡萄牙語中的暱稱。
