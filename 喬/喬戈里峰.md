'''乔戈里峰'''（Chhogori / Qogir）為世界[[世界高山列表|第二高峰]]，属于[[喀喇昆仑山脉|喀喇昆仑山脉]]，位于[[巴基斯坦|巴基斯坦]]与中国边界上，[[海拔|海拔]]8,611-{zh-hans:米; zh-hant:公尺;}-，僅次於[[珠穆朗瑪峰|珠穆朗瑪峰]]。喬戈里峰為[[巴基斯坦|巴基斯坦]]的[[各国最高点列表|最高峰]]（[[印度|印度]]方面则认为乔戈里峰为[[克什米尔|其最高峰]]），也是[[中国|中国]]第二高峰。登山者通稱喬戈里峰為K2，因位置偏遠及山勢陡峭，喬戈里峰通常被認為是最難攀登的8000米以上高峰之一。

== 位置 ==

位于东经76.5度，北纬35.9度地處[[中國|中國]][[新疆|新疆]][[塔什库尔干塔吉克自治县|塔什库尔干塔吉克自治县]]縣与[[巴控克什米尔|巴控克什米尔]]之界山。

== 名称 ==
此峰有多種名稱。中國方面正式名稱「喬-{戈里}-」（''Qogir''，衍生自''Chogori''）為塔吉克语「高大雄伟」之意。而'''K2'''是國際上最常見的名稱，源自1856年西方探险队首次考察此地区时，标出了[[喀喇昆仑山脉|喀喇昆仑山脉]]自西向东的5座主要山峰，各以K1至K5命名。其餘四座分别是[[玛夏布洛姆峰|玛夏布洛姆峰]]（K1），[[布洛阿特峰|布洛阿特峰]]（K3），[[加舒尔布鲁木II峰|加舒尔布鲁木II峰]]（K4）與[[加舒尔布鲁木I峰|加舒尔布鲁木I峰]]（K5）。

早期曾先後稱為Mount Godwin-Austen、Lambha Pahar、Chogori、Kechu V、Dapsang等，唯俱未獲到廣泛應用。

== 气候 ==
每年5月至9月，是雨季，从印度洋吹来的西南季风送来暖湿的气流，化为雨水。9月中旬以后至下一年4月中旬，强劲的西风凛冽而至，带来严酷的冬季。峰顶的最低气温可达-50度，最大风速可达到5米/秒以上，是登山的气候禁区。在5月至9月间，西南季风送来温湿气流，这是雨季，融雪和降水，会导致河谷水位猛涨，人畜都难以进入。因此，登山活动的最佳时机应安排在5月至6月初进山，其时河水虽涨，但不太严重；7月至9月，山顶气温稍高，好天气持续时间较长，是登顶的好时间。海拔7,000米以上经常刮着8级以上的高空风，风速达每小时60公里以上，有时一秒就可以达到25米，降雪时会连续降4，5天，温度最低时达到-50多度，峰顶常年被浓雾笼罩。

== 攀登历史 ==
1902年，英国人奥斯卡·艾肯斯坦與Aleister Crowley率領隊伍首次尝试攀登乔戈里峰，經過五次嘗試後，仍無隊員能成功登頂，或許是體能訓練不足所致。

1954年，阿迪托·迪塞奥帶領一支意大利登山隊向峰頂出發，隊中2人里諾·雷斯德里和阿奇里·科帕哥諾尼在7月31日成功登上峰頂。

1979，莱茵霍尔德．梅斯纳尔成功登顶K2峰（首次[[阿尔卑斯式攀登|阿尔卑斯式攀登]]该峰）。

1982年，日本山-{岳}-協會的一支登山隊首次從乔戈里峰中国境内的北坡成功登顶。

1995年，[[台灣|台灣]]登山家[[陳國鈞|陳國鈞]]跟[[江秀真|江秀真]]登上K2峰，但未登頂<REF>[http://www.taiwangoodlife.org/taxonomy/term/566 高山酷爸寶貝女| 台灣好生活電子報]</REF>。

2004年7月27日，[[西藏登山探险队|西藏登山探险队]]于当地时间6时50分（北京时间9时50分）成功登顶。这次成功也创造了3人同时登顶13座海拔8000以上高峰的中国登山新记录。成立于1993年的“中国西藏攀登世界14座海拔8000米以上高峰探险队”也成为世界上首支集体登完13座高峰的登山队。<ref>{{cite web|url=http://news.xinhuanet.com/travel/2004-07/28/content_1659894.htm|title=中国人首次登顶世界第二高峰乔戈里峰|publisher=新华社|date=2004年7月28日}}</ref>

2012年7月30日，[[杨春风|杨春风]]与其队友[[张京川|张京川]]及[[饶剑峰|饶剑峰]]沿南坡成功登顶成功。这是首支中国民间登山队登顶成功。

=== 登顶死亡率 ===
2003年9月统计数字：共登顶198人，共死亡53人，总体死亡率26.77%（死亡率＝死亡人数／登顶人数），1990年之前的死亡率41%，1990年以来死亡率为19.7%。

2008年8月2日，17名登山者从[[巴基斯坦|巴基斯坦]]一侧登顶返回时遭遇雪崩，当中11人遇难，成为[[2008年喬戈里峰登山事故|乔戈里峰登山史上死亡人数最多的灾难]]。
