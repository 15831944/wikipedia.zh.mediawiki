'''劉偉明'''（{{bd|1969年|2月6日}}），筆名'''喬靖夫'''，英文名'''Jozev'''，出生于[[香港|香港]]，香港小說作家及流行曲作詞人，大多數作品皆以武俠及格鬥為主。

在未用喬靖夫為筆名時，曾以'''雷義'''為筆名。亦曾因其作品《[[殺禪|殺禪]]》推出十年才得以完結，被讀者冠以「喬年鑑」、「香港第一遲筆」之名。

==生平==
喬靖夫十五歲開始習[[空手道|空手道]]，達至啡帶級別，2009年開始學習{{link-en|菲律賓魔杖|Eskrima}}作器械搏擊。早年就讀[[長沙灣天主教英文中學|長沙灣天主教英文中學]]時與填詞人[[周博賢|周博賢]]為同屆中學同學。畢業於[[香港城市大學|香港城市大學]]翻譯系，曾先後涉足於新聞翻譯、電腦遊戲及編劇的工作，曾擔當《[[快報|快報]]》翻譯員，現專注於文字及跨媒體的創作，偶爾亦會為香港流行曲填詞。

2000年憑《深藍》一曲，喬靖夫奪得[[香港作曲家及作词家協會|香港作曲家及作词家協會]](CASH)最佳歌詞獎。<ref>[http://www.cash.org.hk/hk/content/web.do?id=ff80808110e292b70110e55b83980027 香港作曲家及作词家協會官網]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2008年喬靖夫為電影《[[十分鍾情|十分鍾情]]》內其中一個故事《[[紙皮爸爸|紙皮爸爸]]》擔當演員，飾演爸爸一角。

2010年，喬靖夫為[[香港電台|香港電台]]電視節目《[[功夫傳奇|功夫傳奇]]》擔任第五集「[[太極|太極]]」節目主持人。<ref>[http://app3.rthk.org.hk/press/main.php?id=359 香港電台《功夫傳奇》新聞稿]</ref>2012年再為[[香港電台|香港電台]]電視節目《[[功夫傳奇II之再戰江湖|功夫傳奇II之再戰江湖]]》擔任第三集「威風八極」節目主持人。<ref>[http://programme.rthk.hk/rthk/tv/programme.php?name=tv/kungquestii&p=5258 香港電台《功夫傳奇II之再戰江湖》網頁]</ref>。

2013年12月，喬靖夫繼續多元化嘗試，除了繼續為《功夫傳奇》電視系列第三輯《[[功夫傳奇之決戰邊疆|功夫傳奇之決戰邊疆]]》擔任第四集「勇闖天竺」節目主持人外<ref>[http://programme.rthk.hk/rthk/tv/programme.php?name=tv/kungfuquest2013&p=6052 香港電台《功夫傳奇之決戰邊疆》網頁]</ref>，還與鞋商「[[Nike|Nike]]」合作，推出與籃球員[[高比拜仁|高比拜仁]]有關的短篇小說系列。

==著作==
《武道狂之詩》卷一【風從虎.雲從龍】及卷六【任俠天下】分別被列入第21及22屆[[「中學生好書龍虎榜」|「中學生好書龍虎榜」]]60本候選好書之一。<ref>http://www.hkpl.gov.hk/english/ext_act/ext_act_ra/ext_act_ra_sgbss/files/21thSelectedGoodBookdownload.pdf 香港公共圖書館文件</ref><ref>{{cite web |url=http://www.hkptu.org.hk/reading/v3/goodbook/19goodbook_60.php |title=存档副本 |accessdate=2011-06-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20101121134844/http://www.hkptu.org.hk/reading/v3/goodbook/19goodbook_60.php |archivedate=2010-11-21 }} 教協普及閱讀獎勵計劃網站</ref> 另外同系列的卷六【任俠天下】、卷七【夜戰廬陵】、卷八【破門六劍】及卷十【狼行荊楚】亦在[[三聯書店_(香港)|出版之門]]第355、369、370、389及426期榮登香港每周暢銷書榜首位。<ref>{{cite web |url=http://www.publishing.com.hk/bookrank/Default.asp?AreaCode=F |title=存档副本 |accessdate=2011-06-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20110419223123/http://publishing.com.hk/bookrank/Default.asp?AreaCode=F |archivedate=2011-04-19 }} 出版之門每周暢銷書榜</ref>

2005年，[[樂文社|樂文社]]得到喬靖夫授權，以香港薄裝漫畫形式出版其作品《冥獸酷殺行》，故事一共十期，[[曹志豪|曹志豪]]主編，週刊，售價港幣十四元正。

2011年 3月下旬，喬靖夫結婚<ref>[http://paper.udn.com/udnpaper/POI0028/191895/web/ 明日武俠電子報2011/04/01 第76期]</ref>。《[[武道狂之詩|武道狂之詩]]》漫畫創刊號正式於4月15日於[[香港|香港]]及[[台灣|台灣]]同時推出，[[夢馬工作室|夢馬工作室]]製作，[[葉偉青|葉偉青]]主篇，[[袁建滔|袁建滔]]編劇，經由[[東立出版社|東立出版社]]出版，漫畫首三回故事曾在三月及四月份的《[[新少年週刊|新少年週刊]]》雜誌連載。<ref>{{cite web |url=http://www.tlcomics.com/blood.html |title=存档副本 |accessdate=2011-04-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20110308170006/http://www.tlcomics.com/blood.html |archivedate=2011-03-08 }}</ref><ref>ISBN 9789629558499</ref>漫畫版第一輯「[[武當|武當]]野望篇」於2011年至2013年期間出版，在2011年獲得國內[[金龍獎|第八屆金龍獎]]最佳成人漫畫獎<ref>[http://www.comicfans.net/oacc/OACC8/zxxx/2011/09/30/01523724933.html 香港成人漫畫內地動漫大獎奪魁]</ref>

2012年9月7日《武道狂之詩》少年版漫畫在內地漫畫雜誌《[[漫畫SHOW|漫畫SHOW]]》第40期開始彩色連載，由金龍獎得主[[白驍|白驍]]執筆。其後喬靖夫與漫畫家[[江康泉|江康泉]]合作，製作漫畫《獵魂之翼》，於2013年7月在漫畫雜誌《熱血少年》中連載。
