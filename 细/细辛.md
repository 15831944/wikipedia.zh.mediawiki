'''细辛'''（[[学名|学名]]：{{lang|la|''Asarum sieboldii''}}）是[[马兜铃科|马兜铃科]][[细辛属|细辛属]]的植物。分布在[[日本|日本]]、[[朝鲜|朝鲜]]以及[[中国大陆|中国大陆]]的[[浙江|浙江]]、[[河南|河南]]、[[山东|山东]]、[[河北|河北]]、[[江西|江西]]、[[陕西|陕西]]、[[四川|四川]]、[[安徽|安徽]]等地，生长于海拔1,200米至2,100米的地区，多生在林下阴湿腐殖土中，属中国原产的植物（非从国外引种）。

== 别名 ==
华细辛（文献通称） 盆草细辛（四川）

== 性味归经 ==
性温，味辛。归[[肺_(臟腑)|肺]]，[[肝_(臟腑)|肝]]经<ref name=":0">{{Cite book|title=《中药学》|last=|first=|publisher=上海科学技术出版社|year=1984|isbn=9787532304974|editor-last=凌一揆|editor-link=凌一揆|editor2-last=颜正华|edition=1984年第一版|location=上海|pages=}}</ref>。

== 功效 ==
*祛风，散寒止痛，温肺化饮，宣通鼻窍<ref name=":0" />
*流感或風濕引起之頭痛、身痛。
*牙痛、口瘡、鼻淵。
*本品有麻醉作用。
*可幫助腸道蠕動。
*氣血兩虛者不宜多用。

== 毒性作用 ==
*古人就認為細辛是毒藥，只能少量使用，有「細辛不過錢」的說法，但實際上是「細辛…若單用末，不可過一錢，多則氣悶塞不通則死，雖死無傷」。<ref>[[本草綱目|本草綱目]]</ref>
*含有[[黃樟醚|黃樟醚]]毒性較強，係致癌物質，高溫易被破壞。
*細辛含有強力致癌及腎毒性物質[[馬兜鈴酸|馬兜鈴酸]]，但仍有辦法安全使用，可以僅使用含量極低的根部（古方也僅使用根部）、並利用馬兜鈴酸幾乎不溶於水的特性，若僅採用根部製藥，並採取水煎，再經由嚴格的檢驗複查，則不含馬兜鈴酸；因此並未禁用。<ref>[http://news.ltn.com.tw/news/life/paper/67401 再傳馬兜鈴腎病變]</ref>
