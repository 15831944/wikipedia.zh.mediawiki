{{中共指导思想}}
《'''实践是检验真理的唯一标准'''》，是由[[南京大学|南京大学]]哲学系教师[[胡福明|胡福明]]原作，经过多人修改，最终由時任[[中共中央组织部|中共中央组织部]]部长[[胡耀邦|胡耀邦]]审定的一篇文章。该文标志着中华人民共和国[[真理标准大讨论|真理标准大讨论]]的开始。

== 经过 ==
该文于1978年5月10日首次发表于[[中共中央党校|中共中央党校]]内部刊物《[[理论动态|理论动态]]》第60期上。次日，《[[光明日报|光明日报]]》以“特约[[评论员|评论员]]”的署名全文发表于头版。当日，[[新华社|新华社]]转发了这篇文章。12日，《[[人民日报|人民日报]]》和《[[解放军报|解放军报]]》同时转载。数日之内，该文传遍全国。

该文全文7000余字，主张只能依靠社会实践检验真理；[[马克思主义|马克思主义]]最基本的原则之一就是理论与实践的统一；[[馬克思|马]][[恩格斯|恩]][[列寧|列]][[斯大林|斯]][[毛澤東|毛]]诸位革命导师都坚持用实践检验真理；任何理论都要不断接受实践的检验<ref>[http://www.chinaelections.org/NewsInfo.asp?NewsID=102051 中国选举与治理·我国三次思想大解放的历史回望]</ref>。

从此文的发表开始，真理标准大讨论席卷全国，最终动摇了[[华国锋|华国锋]]执政的理论基础——所谓“两个凡是”。随着华国锋在中共中央工作会议上作检查，以及1978年底[[中共十一届三中全会|中共十一届三中全会]]的召开，[[邓小平|邓小平]]逐渐取代华国锋掌握了党和国家的最高权力<ref>[http://www.people.com.cn/GB/historic/0510/6267.html 人民网·1978年5月10日《实践是检验真理的唯一标准》发表]</ref>。

== 争议 ==
《光明日报》原副总编兼理论部主任[[马沛文|马沛文]]临终前，发布《必得澄清的一桩重大史实》，回应了中央党史中一派认为此文是由邓小平授意完成的作品<ref>{{cite journal|author1=[[马沛文|马沛文]]|coauthors=责任编辑[[杨继绳|杨继绳]]|title=必得澄清的一桩重大史实|journal=[[炎黄春秋|炎黄春秋]]|date=2015年|issue=第1期|url=http://www.yhcqw.com/34/9723.html|accessdate=2018-05-10}}</ref>。
