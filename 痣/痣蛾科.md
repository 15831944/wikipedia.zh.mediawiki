'''痣蛾科'''（學名：{{lang|la|Blastobasidae}}）{{r|TaiBNET}}，又名'''粗角蛾科'''，是[[鱗翅目|鱗翅目]][[有喙亞目|有喙亞目]][[異脈下目|異脈下目]]之下一個有爭議的[[科_(生物)|科級]][[分类级别|分类级别]]，是{{tsl|en|Gelechioidea|麥蛾總科}}之下的一種[[蛾|蛾]]。本科[[物种|物种]]幾乎在全球均可見其踪影，但有不少地方都是因為被人類攜帶，而非原生於當地。因此，不少在大中華地區已被辨識的物種都未有通用的中文名稱{{r|TaiBNET}}。
在部分文獻，本科被列為同一總科之下{{tsl|en|case-bearer|Coleophoridae}}科的[[亚科|亚科]]（即 '''痣蛾亞科'''；Blastobasinae）{{r|ITIS_694002}}}。
{{tsl|en|Symmocidae|}}這個科亦有時會被列為本科的亞科或[[族_(生物)|族]]<ref name="AFD">{{cite web |author={{tsl|en|Australian Biological Resources Study|}} |work={{tsl|en|Australian Faunal Directory|}} |url=http://www.environment.gov.au/biodiversity/abrs/online-resources/fauna/afd/taxa/BLASTOBASIDAE |title=Blastobasidae |date=2008-10-09 |access-date=2010-04-28 |publisher={{tsl|en|Department of the Environment, Water, Heritage and the Arts|}} |language=en }}</ref><ref name="FE">{{cite web |publisher={{tsl|en|Fauna Europaea|}} |url=http://www.faunaeur.org/full_results.php?id=2901 |title=Blastobasidae |work=Version 2.1 |date=2009-12-22 |access-date=2010-04-28 |language=en }}</ref><ref name="TOL">{{cite web |work={{tsl|en|Tree of Life Web Project|}} |url=http://tolweb.org/Coleophoridae/12108 |title=Coleophoridae |date=2008-05-01 |access-date=2010-04-28 |language=en }}</ref>。

此外， Holcocera周圍的群體經常被分離為家族Holcocerinae （或部落Holcocerini ）與Blastobasis譜系（相應地成為亞科或Blastobasini部落）。
此外，環繞''{{tsl|en|Holcocera|}}''屬的物種有時會從組成'''Holcocerinae'''亞科或'''Holcocerini'''族，而餘下的物種則組成痣蛾亞科或痣蛾族。
儘管這個編排比起在麥蛾總科內某些極端的[[生物分类学|生物分类学]]和[[系統分類學|系統分類學]]編排更合理，但痣蛾科各屬間的關係至今仍未有足夠的研究去支持{{r|AFD|FE}}。

==屬==
痣蛾科現時已辨認了約30個[[属|属]]，主要都是體型細小的物種；當中有不少只有少數物種，甚至是[[單型|單型]]的屬。例外的有物種多樣性很高的''{{tsl|en|Auximobasis|}}''屬和 ''{{tsl|en|Pigritia|}}''屬，以及體型較大的[[痣蛾屬|痣蛾屬]]（''{{tsl|en|Blastobasis|}}''）和 ''{{tsl|en|Holcocera|}}''屬物種。這些屬在[[ITIS|ITIS]]被分為[[Blastobasini|Blastobasini]]族和[[Holcocerini|Holcocerini]]族這兩個族{{r|ITIS_694002}}，但未有紀錄各族的成員。
