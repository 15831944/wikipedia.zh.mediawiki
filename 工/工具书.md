'''工具书'''指一种在[[学习|学习]]中和[[劳动|工作]]中可以作为工具使用的特定类型的[[书籍|书籍]]，專供查考資料，以解决工作或学习过程中所遇到的某些疑难问题。這些書籍經過彙集、編著或譯述的資料，把相关领域的问题与其有关的知识资料按特定的编排方法汇集在一起，形成特定的編排和檢索方式，可以讓讀者於短時間內查出答案，以供參考。

因為這些書籍是解決問題的好工具，因此稱為「參考書」或「工具書」。

== 功用 ==
# 解決疑難問題
# 指示讀書途徑
# 提供參考資料
# 掌握學術信息
# 節省時間精力
