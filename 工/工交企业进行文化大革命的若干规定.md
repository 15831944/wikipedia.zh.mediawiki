《'''工交企业进行文化大革命的若干规定'''》，简称《'''工交十五条'''》，是1966年11月文化大革命初期的一个政府规定，目的是否定《[[关于工厂文化大革命的十二条指示|关于工厂文化大革命的十二条指示]]》<ref name=as />。

== 概述 ==
1966年11月17日，[[中央文革小组|中央文革小组]]和首都革命职工代表举行座谈期间，由[[王力_(政治人物)|王力]]宣读了《[[关于工厂文化大革命的十二条指示|关于工厂文化大革命的十二条指示]]》（《工厂十二条》），内容是鼓励在工厂范围内进行[[文化大革命|文化大革命]]推广活动。但该草案受到中央和地方一些主观经济工作的领导人的抵制，因此在11月16日，按中共中央指示，[[中华人民共和国国务院|国务院]]在北京召开工交企业座谈会，讨论工业交通企业“文化大革命”问题，并围绕该草案进行讨论。国务院下属五部（[[中华人民共和国冶金工业部|冶金部]]、[[中华人民共和国水利电力部|水电部]]、[[中华人民共和国铁道部|铁道部]]、[[中华人民共和国化学工业部|化工部]]、[[中华人民共和国第一机械工业部|机械部]]）、七个城市（[[北京市|京]]、[[上海市|沪]]、[[天津市|津]]、[[沈阳市|沈]]、[[哈尔滨市|哈]]、[[武汉市|汉]]、[[广州市|穗]]）及各大区有关负责人参加座谈会<ref name="a1">{{Cite web |url=http://cpc.people.com.cn/GB/64162/64164/4416081.html |title=中国共产党大事记·1966年|accessdate=2018-12-19 |author= |date= |work= |publisher=中国共产党新闻网|language=Zh-hans }}</ref>。

与会者对“文化大革命”危及工矿行业表示揪心，对在工矿企业如同学校、机关一样地开展“文化大革命”提出异议。还有表示工厂的“文化大革命”不能全面铺开，否则生产肯定要受到影响，且当时生产已有停顿趋势，科研、尖端项目、协作项目已基本上停止，基本建设也开始受到影响。有官员指出，当时生产已经靠吃老本，库存材料已经很少，问题越来越多，马钢、武钢都有停产危险。有的指出，工厂里无论如何要由党委领导，下面再分两摊，一摊抓革命，一摊抓生产，不然生产非受影响不可。有的指出，工厂中不要建立群众组织，担心建立群众组织会出现对立。最终工矿企业表示不同意中央文革小组在工交部分开展文化大革命<ref name=as>{{Cite book|title=从一大到十七大 第3册 图文版|last=张静如主编|first=|publisher=沈阳：万卷出版公司|year=2008.08|isbn=978-7-80601-874-3|location=|pages=338}}</ref>。

会议在[[陶铸|陶铸]]主持下，[[余秋里|余秋里]]、[[谷牧|谷牧]]组织草拟与十二条相对立的《工交企业进行文化大革命的若干规定》。后者主张工矿企业仍然按照[[四清运动|四清运动]]部署，分批分期，证明教育，不搞串联，减持八小时生产）<ref>{{Cite book|title=中国共产党党史全鉴 第4卷|last=党史理论研究室编；刘海藩主编|first=|publisher=党史研究出版社|year=2006.11|isbn=978-7-80112-612-2|location=|pages=1592}}</ref>。不久，[[毛泽东|毛泽东]]否定了《工交十五条》。中央文革小组对《十二条》进行修改，形成了《[[中共中央关于抓革命、促生产的十条规定|中共中央关于抓革命、促生产的十条规定]]（草案）》，即《工业十条》<ref>{{Cite book|title=中国共产党全国代表大会史丛书 从一大到十七大 第3册 图文版|last=张静如主编；赵勇民，李颖副主编|first=|publisher=沈阳：万卷出版公司|year=2012.10|isbn=7-5470-1983-8|location=|pages=306-307}}</ref>。

12月4日至6日，[[林彪|林彪]]主持召开[[1966年12月中共中央政治局扩大会议|中共中央政治局扩大会议]]，听取工交座谈会情况汇报并讨论通过《工业十条》。林彪说：“这次工交会议开了二十多天，会议开得不好，是错误的，思想很不对头。”,“[[刘少奇|刘]]、[[邓小平|邓]]不仅是五十天的问题，而是十年、二十年的问题。工交战线受刘、邓的影响很大。”他还说：这次“文化大革命”是“对全党的批判运动，批判干部的运动。”会议最后一天，中央文革王力等人对陶铸发动攻击，指责陶铸坚持抓生产是用生产压革命。最终，会议一致通过经中央文革修改定稿的《工业十条》<ref name="a1"/>。
