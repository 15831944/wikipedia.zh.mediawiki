'''工人黨'''（{{lang-en|Workers' Party}}）是[[新加坡|新加坡]]的一个[[中间偏左|中间偏左]]政党。该党是现在新加坡最大的反對黨，但它在新加坡的影響力遠不及執政黨[[人民行動黨|人民行動黨]]。目前，该党在[[新加坡國會|國會]]的89个选举席位和3个非选区议席中佔有9席，包含[[阿裕尼集選區|阿裕尼集選區]]的[[劉程強|劉程強]]、[[林瑞莲|林瑞莲]]、[[陳碩茂|陳碩茂]]、[[畢丹星|畢丹星]]、[[莫哈默費沙|莫哈默費沙]]5席、[[後港單選區|後港單選區]]的[[方榮發|方榮發]]1席與3席非選區議員，是歷來最多。

== 历史 ==
工人黨在1957年由律师[[大卫·马绍尔|大卫·马绍尔]]创立。2001年5月，刘程强出任党的[[秘书长|秘书长]]。2002年，[[林瑞莲|林瑞莲]]担任該[[党主席|党主席]]。

[[2011年新加坡大選|2011年新加坡大選]]，工人黨既保住既有的[[後港單選區|後港]]議席，並由刘程强轉戰鄰近的[[阿裕尼集選區|阿裕尼集選區]]，搭配林瑞蓮與知名[[台灣|台灣]]移民[[律師|律師]][[陳碩茂|陳碩茂]]，擊敗現任外交部長[[楊榮文|楊榮文]]的團隊，在新加坡史上創下第一次由反對黨攻下集選區的記錄，席次一舉提升到八席，其中六席为国会议席，两席為非選區議員。但後港選區議員饒欣龍，因牽涉桃色糾紛而被工人黨開除黨籍，議席因而出缺，在懸缺數月後，5月9日宣佈在5月26日進行補選，而工人黨考慮派出[[方榮發|方榮發]]參選。工人党的方荣发发挥了该党的主场优势，在后港区补选中以62％的得票率力挫人民行动党的朱倍庆。

2013年1月的榜鹅东区补选中，工人党的[[李丽连|李丽连]]以54.52%（16,038票）的得票率战胜人民行动党的[[许宝琨|许宝琨]]（43.71%的得票率，12,856票），她因此成为自1968年以来第一位赢得单选区的[[女性|女性]]反对党议员<ref>{{cite web |url = http://www.zaobao.com/sp/sp130127_001.shtml |language= |title = 胜行动党10.81个百分点 3182票 工人党夺下榜鹅东 |publisher=联合早报 |date = 2013-01-27 |accessdate= }}</ref>。在国会议员中，执政党人民行动党还是保住了80个席位，但工人党却史无前例夺走了7个国会和2个非选区席位，席次由2011年的8席提升到9席。
