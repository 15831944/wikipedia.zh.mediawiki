'''工程師節'''存在於以下幾個國家

== 阿根廷 ==

在[[阿根廷|阿根廷]],  工程師節在6月15日, 是為了慶祝1870年6月15日, [[Luis_Huergo|Luis Augusto Huergo]] 成為[[阿根廷|阿根廷]]第一位工程師.

== 哥倫比亞 ==

在 [[哥倫比亞|哥倫比亞]],  工程師節在八月17日.

== 印度==

在 [[印度|印度]],  工程師節是在九月15日.

== 伊朗 ==

在 [[伊朗|伊朗]] 工程師節在每年的二月24日.是為了慶祝[[伊朗|伊朗]]最偉大的科學家 [[Nasir_al-Din_al-Tusi|Nasir al-Din al-Tusi]].<ref>http://calendar.ut.ac.ir</ref>

== 墨西哥 ==

在 [[墨西哥|墨西哥]],  工程師節在七月1日.<ref>{{cite web |url=http://www.cicm.org.mx/institu.php |title=存档副本 |accessdate=2011-07-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20110630180310/http://www.cicm.org.mx/institu.php |archivedate=2011-06-30 }}</ref>

== 委內瑞拉==
在 [[委內瑞拉|委內瑞拉]],  工程師節在十月28日.

== 巴拿馬==
在 [[巴拿馬|巴拿馬]], 工程師節在一月26日.

== 義大利==

在 [[義大利|義大利]],  工程師節在六月15日.

== 中華民國 ==
在 [[臺灣|臺灣]](中華民國)，工程師節在6月6日，目的是為了紀念治水的[[大禹|大禹]]，此日也是水利節。<ref>{{cite web |url=http://www.rical.net/blog/article-1118025939.html |title=存档副本 |accessdate=2012-06-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20120609053944/http://www.rical.net/blog/article-1118025939.html |archivedate=2012-06-09 }}</ref>

== 美國==
在 [[美國|美國]], 工程師節在每年的六月4日.<ref>http://sarcastictestguy.blogspot.com/2012/05/engineers-day-june-4.html</ref>
