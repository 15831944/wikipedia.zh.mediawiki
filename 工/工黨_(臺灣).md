'''工黨'''為[[中華民國|中華民國]]的一個[[勞工|勞工]][[中華民國政黨|政黨]]，成立於1987年11月1日，是臺灣宣布[[解嚴|解嚴]]後第四個登記的政黨（前三個是[[中國國民黨|中國國民黨]]、[[中國青年黨|中國青年黨]]、[[中國民主社會黨|中國民主社會黨]]）<ref name=zheng>{{Cite web |url = http://www.crntt.com/doc/1045/6/7/7/104567714.html?coluid=93&kindid=5670&docid=104567714&mdate=0208000757 |title = 郑昭明：郑清谈判条件愈来愈差最后武统收场 |author =  |date =2017-02-08  |publisher =中評網  |language =  |accessdate =  |quote =  }}</ref><ref name=nfw>{{Cite web |url =http://www.southcn.com/news/hktwma/zhuanti/2004xuanju/bg/200402120553.htm  |title = 台湾其他党派 |author =  |date =2004-02-12  |publisher =南方网  |language =  |accessdate =  |quote =  }}</ref>。

==简介==
1987年11月1日，工黨在[[臺北市|臺北市]]成立。1987年12月6日，举行第一届党员代表大会，通过党纲、党章、党旗，选举产生工党中央委员会，成立了党的组织。工黨是以台湾中下层出身、谋求劳工权益的知识分子为主要力量组成。工党主席[[王義雄|王義雄]]称工党是以“劳工为主体”，“联合其它阶级人民的政党”<ref name=nfw/>。

工黨組織的背景主要是：原[[民主進步黨|民主進步黨]]的[[立法委員|立法委員]][[王義雄|王義雄]]對民進黨[[勞工運動|勞工運動]]的發展以及[[臺灣獨立|臺灣獨立]]主張方面與黨內主流發生歧見，進而退黨另組工黨。而工黨也有不少勞工團體以及[[知識分子|知識分子]]參與，其中以[[陳映真|陳映真]]為主的[[夏潮聯合會|夏潮聯合會]]更曾經掌握黨機器。

工黨黨旗為綠底黃星七星旗，墨绿色底，中間一顆大星代表以工人为主体，下邊六顆小星則為農民、漁民、婦女、少數民族、中小企業者及知識分子，七星象徵七大群體的團結<ref name=nfw/>。在政黨主張方面，主要包括擁護廣大勞動民眾利益的獲得，實現勞動大眾的民主、自由、平等，以及臺灣與[[中國大陸|中國大陸]]的和平統一。首任黨主席為王義雄，副主席為[[羅美文|羅美文]]，秘書長為[[蘇慶黎|蘇慶黎]]。

工黨成立時曾經獲得相當程度的支持。1988年5月內部發生分裂，[[王義雄|王義雄]]只傾向給予工運法律支援，而夏潮派，主張社會主義、統一色彩(反帝反殖)較為濃厚、強調勞工大眾屬性並且與《夏潮》雜誌系統較為有關係的黨員，在羅美文率領下先成立幹部聯誼會，最後於1988年底脫黨，1989年另組[[勞動黨_(臺灣)|勞動黨]]。工黨與勞動黨的糾紛，導致工黨的發展受阻；1989年的選舉中，工黨只當選一席[[高雄市議員|高雄市議員]][[張益郎|張益郎]]，立法委員與縣市長方面全軍覆沒，工黨主席王義雄與藝術家[[許曉丹|許曉丹]]等人都在該年的選舉中落選。1992年工黨提名許曉丹在[[高雄市|高雄市]]參選立法委員，仍告落敗，王義雄也為此負責而退出政壇，工黨主席一職改由[[鄭昭明|鄭昭明]]出任。

之後工黨長期未推舉候選人參選民意代表，在台灣著重關注勞工運動、勞工服務、勞資爭議協調；對外促進兩岸和平發展與交流。工黨主席鄭昭明是第一位访问中國大陸的臺灣政党领袖。工党为落实大陆政策，联合各界成立“台湾两岸关系发展促进会”，并由工黨主席鄭昭明担任创会理事长<ref name=zheng/>。
