{{日本作品|工作細胞|はたらく細胞}}是[[清水茜|清水茜]]所著的[[日本漫畫|日本漫畫]]作品，主要講述經[[萌擬人化|擬人化]]後人體內各種[[細胞|細胞]]於人體內的日常工作，例如負責物流作業（運送氧氣及二氧化碳）的[[紅血球|紅血球]]和日常警備工作（清除惡菌）的[[白血球|白血球]]。

本作的[[衍生作品|衍生作品]]有：在《[[Nakayoshi|Nakayoshi]]》上連載，{{lang|ja|吉田はるゆき}}所畫的{{日本作品|工作細菌|はたらく細菌}}；於《[[月刊少年天狼星|月刊少年天狼星]]》上連載，[[杉本萌|杉本萌]]作畫的{{日本作品|不工作細胞|はたらかない細胞}}；於《[[Morning_(漫畫雜誌)|Morning]]》上連載，[[原田重光|原田重光]]擔任原作、[[初嘉屋一生|初嘉屋一生]]作畫的{{日本作品|工作細胞BLACK|はたらく細胞BLACK}}；於《[[別冊FRIEND|別冊FRIEND]]》上連載，{{lang|ja|黒野カンナ}}擔任原作，{{lang|ja|和泉みお}}作畫的{{日本作品|工作細胞FRIEND|はたらく細胞フレンド}}；皆由清水茜擔任監修。

== 登場角色 ==
=== [[細胞|細胞]] ===
==== [[血球|血球]] ====
; {{nihongo|[[紅血球|紅血球]]|赤血球（せっけっきゅう）}}
: 主要工作是負責日常運送養分、[[氧氣|氧氣]]或[[二氧化碳|二氧化碳]]。制服外套與帽子可以雙面使用，運送氧氣與養分時鮮紅色面朝外，運送二氧化碳時則是暗紅色。由於是沒有[[粒線體|粒線體]]的細胞，所以仰賴[[葡萄糖|葡萄糖]]作為養分來源。一旦受物理、化學、生物等因素影響死亡，就會觸發[[溶血反應|溶血反應]]。
:; {{nihongo|AE3803|AE3803|聲：[[花澤香菜|花澤香菜]]<ref name="character">{{cite web|url=http://hataraku-saibou.com/character/|title=CHARACTER|work=電視動畫《工作細胞》官方網站|language=ja|access-date=2018-05-20|archive-url=https://web.archive.org/web/20180521020901/http://hataraku-saibou.com/character/|archive-date=2018-05-21|dead-url=no}}</ref><ref name="staffcast">{{cite web|url=https://hataraku-saibou.com/staffcast/|title=STAFF/CAST|work=電視動畫《工作細胞》官方網站|language=ja|access-date=2018-07-29|archive-url=https://web.archive.org/web/20180716194728/https://hataraku-saibou.com/staffcast/|archive-date=2018-07-16|dead-url=no}}</ref>（日本）；[[楊詩穎|楊詩穎]]（台灣）}}
:: 本作的主要角色之一，形象為紅短髮的少女。個性溫和開朗富朝氣，故事開始時剛脫核不久，是手握《身體地圖》時也能迷路的新人。在肺炎球菌侵入事件中結識了「U-1146」，兩者開始經歷連串的人體反應事件，隨著劇情發展逐漸培養出堅強的性格，和「U-1146」交情匪淺。實則於骨髓時期已和「U-1146」有過一面之緣。
:: 在本作的初始單回原型作品「細胞之話（細胞の話）」中，邂逅和經歷白血球「U-1116」壽限到期的事件，該短篇故事尾聲以AE3803和轉生為「U-1117」的白血球作結。
:; {{nihongo|紅血球母細胞|赤芽球（せきがきゅう）|聲：[[花澤香菜|花澤香菜]](AE3803)、[[泊明日菜|泊明日菜]]（日本）；[[連思宇|連思宇]](AE3803)、[[汪世瑋|汪世瑋]]（台灣）}}
:: 紅血球的幼年型態。[[祖細胞|前驅細胞]]生成後，由造血幹細胞選別轉化為紅血球母細胞，在骨髓中由巨噬細胞照料成長。當紅血球母細胞足夠成熟時，巨噬細胞會取下在帽子上的細胞核，讓紅血球母細胞轉為紅血球，這個過程被稱為脫核。
:: 「AE3803」在骨髓時期就曾經因為迷路而遭到細菌（綠膿桿菌）襲擊，以此契機遇見還是骨髓球的「U-1146」。
:; {{nihongo|AA5100|AA5100|聲：[[遠藤綾|遠藤綾]]<ref name="character"/><ref name="staffcast"/>（日本）；[[陳貞伃|陳貞伃]]（台灣）}}
:: 形象為棕色長髮的女性。為「AE3803」剛脫核時負責帶她熟悉環境的前輩，雖然已經沒有責任關係，但是還是經常陪伴總是迷路的「AE3803」送貨。
:; {{nihongo|NT4201|NT4201|聲：[[石川由依|石川由依]]<ref name="character"/><ref name="staffcast"/>（日本）；[[葉于嘉|葉于嘉]]（台灣）}}

; {{nihongo|[[白血球|白血球]]／[[嗜中性球|嗜中性球]]|白血球/好中球（はっけっきゅう/こうちゅうきゅう）}}
: [[免疫系統|免疫系統]]中的最前線戰力，主要負責「雜菌偵察」的警備工作。全員形象為白色制服、白髮、膚色蒼白的警備人員，發現雜菌後帽子（白血球[[受體|受體]]）會彈起等同「正解」的 O 型圖案，性情也會變得極為兇暴。若情況許可，白血球會當場擊殺惡菌。可以在身體組織之間自由遊走，穿越血管壁到達病菌的所在；也可以透過吞噬細胞（惡菌）判斷出細菌及過敏原。當人體受傷遭遇細菌[[感染|感染]]的時候，戰死的白血球會成為[[膿|膿]]的成分流出傷口。
:; {{nihongo|U-1146|U-1146|聲：[[前野智昭|前野智昭]]<ref name="character"/><ref name="staffcast"/>（日本）；[[張騰|張騰]]（台灣）}}

:: 在本作的初始單回原型作品「細胞之話」中，以「U-1116」的身分邂逅紅血球AE3803，最終因為壽限到期回歸骨髓分化，轉生為白血球「U-1117」和紅血球AE3803再會。
:; {{nihongo|骨髓球|骨髄球（こつずいきゅう）|聲：[[村中知|村中知]]<ref name="character"/><ref name="staffcast"/>（日本）；[[汪世瑋|汪世瑋]]（台灣）}}

:: 「U-1146」在骨隨時期憑藉天分發現了入侵骨髓、正在攻擊「AE3803」的綠膿桿菌，雖然挺身而出但是因為過於幼小而不敵，但是造成的騷動已經足以引起巨噬細胞與嗜中性球的注意，趕來清除綠膿桿菌。
:; {{nihongo|U-2001|U-2001|聲：[[江越彬紀|江越彬紀]]<ref name="character"/><ref name="staffcast"/>；[[喬資淯|喬資淯]]（台灣）}}

:; {{nihongo|U-2048|U-2048|聲：{{link-ja|佐藤健輔}}<ref name="staffcast"/>（日本）；[[馬伯強|馬伯強]]（台灣）}}

:; {{nihongo|U-2626|U-2626|聲：[[柳田淳一|柳田淳一]]<ref name="character"/><ref name="staffcast"/>（日本）；[[梁群|梁群]]（台灣）}}
:: 「U-1146」的同僚，形象為雙眼被劉海蓋住、神情平淡的男性。在擦傷事件時前去支援陷入苦戰「U-2048」。
:; {{nihongo|U-4989|U-4989|聲：[[熊谷健太郎|熊谷健太郎]]<ref name="character"/><ref name="staffcast"/>（日本）；[[賈文安|賈文安]]（台灣）}}

; {{nihongo|[[血小板|血小板]]|血小板（けっしょうばん）|聲：[[長繩麻理亞|長繩麻理亞]]<ref name="character"/><ref name="staffcast"/>（小隊長）、[[石見舞菜香|石見舞菜香]]、[[春野杏|春野杏]]、[[赤尾光|赤尾光]]、[[森永千才|森永千才]]、[[立花芽惠夢|立花芽惠夢]]（日本）；[[林筱玲|林筱玲]]、[[汪世瑋|汪世瑋]]（台灣）}}
{{main|血小板 (工作细胞)}}
: 修補血管的維修工。主要工作是以[[纖維蛋白|纖維蛋白]]編織[[凝血|凝血]]塊以止血，若傷口過大會強行將附近的血細胞粘上纖維蛋白形成二次血栓，以加速止血。形象皆為穿著[[幼稚園|幼稚園]]制服的孩童。
: 在本作中和嗜中性球U-1146以及紅血球AE3803之間關係非常密切，其小隊長甚至會主動親近嗜中性球U-1146，與其他害怕嗜中性球U-1146的一般細胞或紅血球形成強烈對比。

==== [[淋巴球|免疫細胞]] ====
; {{nihongo|[[殺手T細胞|殺手T細胞]]|キラーT細胞（キラーティーさいぼう）|}}
: [[免疫系統|免疫系統]]中最強大的戰力，主要負責對付特殊頑強的惡菌和受病毒感染的細胞。居住在淋巴管中，只有得到輔助性T細胞的指令時才可以離開淋巴管。
:; {{nihongo|班長||聲：[[小野大輔|小野大輔]]<ref name="character"/><ref name="staffcast"/>、[[東內麻里子|東內麻里子]]<ref name="character"/><ref name="staffcast"/>（未熟胸腺細胞時代）（日本）；[[馬伯強|馬伯強]]（台灣）}}
:: 形象凶狠，穿着黑色工作服，多以赤手空拳作戰。如果說白血球是一般的警察，那麼殺手T細胞則可被視為軍人。

: 會引發痤瘡的一種常見細菌，以人體皮脂維生，具厭氧性。

; {{nihongo|[[登革熱病毒|登革熱病毒]]|デングウイルス}}
: 以蚊子作為媒介進行傳染的病毒，感染後會產生發燒、頭痛等症狀。

; {{nihongo|[[機會性感染|伺機性感染菌]]|悪玉菌}}
: 對人體無害也無益的細菌總稱，佔腸道菌落的70%以上。在益菌與害菌之間，會傾向佔優勢的那一邊。

=== 其他 ===
; {{nihongo|杉樹花粉過敏原|スギ花粉アレルゲン（スギかふんアレルゲン）|聲：[[興津和幸|興津和幸]]<ref name="character"/><ref name="staffcast"/>（日本）}}
: [[日本柳杉|日本柳杉]]的花粉中大量存在的物質，不會致病也沒有毒性。雖然不會對身體造成危害，但因為是外來異物，仍會被免疫系統攻擊，因而造成過敏反應，即花粉症（或稱花粉熱）。

; {{nihongo|[[類固醇|類固醇]]|ステロイド|聲：不明}}
: 現代西藥的一種，作品中以機械人的形象登場。具有強效抗炎與免疫抑制作用，能有效控制組織胺所引起的各種症狀與過敏反應。由於會對所有細胞進行無差別攻擊，使用的劑量與時間長度需要特別注意。

; {{nihongo|[[癌細胞|癌細胞]]|がん細胞}}
: 在細胞分裂的過程中因為不明原因突變而出現的異常細胞，正常人體每天都會產生癌細胞，但是絕大多數會被免疫細胞殺死。癌細胞會入侵周圍的正常細胞並不斷增殖，這種現象稱為浸潤。當浸潤嚴重到一定程度後，癌細胞就會隨著血管與淋巴管轉移到全身，到這個階段，所有重要器官都會遭到侵襲，威脅生物的生命。
:; {{nihongo|癌細胞首腦|がん細胞の親玉|聲：[[石田彰|石田彰]]<ref name="character"/><ref name="staffcast"/>、[[諏訪彩花|諏訪彩花]]（幼年）（日本）；[[賈文安|賈文安]]、[[楊詩穎|楊詩穎]]（幼年）（台灣）}}
:: 躲過免疫細胞追殺的癌細胞。因為複製失誤而產生的癌細胞之一，自誕生後就被免疫細胞追殺而憎恨著這樣的命運，試圖增殖自己侵占人體。能偽裝成一般細胞欺騙免疫系統，在和NK細胞、殺手T細胞、白血球的交手中大佔上風，但因為紅血球叫來的援軍使其力量大減，最後敗在因此活性化的NK細胞手下。
:: 對自己究竟是為何要如此誕生這點表示了恨意，並向唯一曾保護過偽裝為一般細胞的他的白血球表示自己有天定會回來進行報復。
:: 乳酸菌事件中再度復活，並利用遺傳訊息欺騙抑制性T細胞，讓抑制性T細胞阻擋了NK細胞、記憶T細胞（原殺手T細胞班長），而不受抑制T細胞節制的白血球則被他用陷阱關起來。最後因為乳酸菌的活躍，供給癌細胞力量的的腸道內病毒毒氣下降，讓白血球找到機會突破陷阱，殺死癌細胞。
