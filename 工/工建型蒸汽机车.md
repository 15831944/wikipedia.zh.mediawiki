'''工建型蒸汽机车'''是一种[[轴式|轴式]]为0-6-0T的[[蒸汽机车|蒸汽机车]]，用于厂矿[[调车机车|调车]]，最大时速为35km/h。该型机车由[[大连机车|大连机车车辆工厂]]设计，在1958年到1961年间由[[南车成都机车车辆有限公司|成都机车车辆工厂]]和[[太原轨道交通装备|太原机车车辆工厂]]生产，共计122台。有些使用到2000年代。<ref name=railography>[http://www.railography.co.uk/info/cn_steam/profiles/gj.htm Railography : Class Profiles : GJ Class 0-6-0T]</ref>

== 出口 ==
=== 朝鲜铁道省100型 ===
1959年7月，成都机车车辆工厂向朝鲜提供7台工建型蒸汽机车，朝鲜铁道省将此7台机车编为'''100型'''。

=== 越南太原钢铁 ===
越南北部的[[太原市_(越南)|太原]]钢铁亦有此型机车的使用记录，<ref name=railvn>[http://railvn.byethost3.com/GJ.html GJ Class 0-6-0T_Chinese built 1435mm gauge]</ref>关于本车型交付太原钢铁的时间不得而知，但可确认太原钢铁至少配属四台本型机车，其中一台编号为1037的本型机车在2016年尚有目击记录。<ref name=yakuzenshin>{{Cite web|url=https://www.youtube.com/watch?v=qsatOkMuhvk|title=GJ 1037, Thai Nguyen Steel Works, Vietnam 2016|accessdate=|author=Yakuzenshin|date=|publisher=}}</ref>这些机车沿用原车型及车号记述，而不采用越南铁路的蒸汽机车型号命名形式。<ref name=dsvn>按[[越南铁路|越南铁路]]的铁路机车命名规则，蒸汽机车车型以蒸汽机车轴式确定。但中国在[[越南战争|越南战争]]期间援助给[[越南民主共和国|越南民主共和国]]的解放6型蒸汽机车，同样亦不按蒸汽机车轴式定机车车型，而被越南铁路命名为'''Giải Phóng 6'''型。</ref>
