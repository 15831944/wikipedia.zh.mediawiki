'''列昂尼德·伊里奇·勃列日涅夫'''（{{lang-ru|Леони́д Ильи́ч Бре́жнев}}{{Audio|Ru-Leonid Ilich Brezhnev.ogg|听}}，{{lang-uk|Леоні́д Іллі́ч Бре́жнєв}}，{{bd|1906年|12月19日|1982年|11月10日|Бре́жнев}}），[[苏联|苏联]]领导人、[[苏联元帅|苏联元帅]]，曾任[[苏联共产党中央委员会总书记|苏联共产党中央委员会总书记]]（1964至66年间为第一书记）、[[蘇聯最高蘇維埃主席團|蘇聯最高蘇維埃主席團]]主席（國家元首），掌權共18年。

== 早年經歷 ==
儒略历1906年12月6日（公元1906年12月19日），勃列日涅夫出生于[[葉卡捷琳諾斯拉夫省|葉卡捷琳諾斯拉夫省]][[卡緬斯科耶|卡緬斯科耶]]（今[[乌克兰|乌克兰]][[卡緬斯科耶|卡緬斯科耶]]）。1935年5月5日，勃列日涅夫从{{link-ru|第聂伯罗彼得罗夫斯克冶金学院|Днепровский_государственный_технический_университет}}获得了热力工程师的证书。1935年至1936年間被徵入兵役，在坦克學校學習後，被派到一所坦克工廠擔任[[政治委員|政治委員]]。1939年成為[[第聶伯羅彼得羅夫斯克|第聶伯羅彼得羅夫斯克]]的黨委書記，主管該市的重要軍事工業。

1941年德國入侵蘇聯後，勃列日涅夫被徵入軍隊，在多支部隊內擔任政治職務。1944年升为[[少将|少将]]。1946年退役，二戰期間他在軍中的全部時間是負責政委工作，而不是當軍隊指揮官。

==进入领导层==
他在烏克蘭的重建項目工作，後來再次成為第聶伯羅彼得羅夫斯克的第一書記。後來調任摩爾達維亞的黨第一書記。1952年當選苏共中央书记。1953年[[斯大林|斯大林]]逝世后，改任苏军总政治部副主任。後任苏共中央第二书记。1955年兼任[[哈萨克|哈萨克]]党中央第一书记，积极推行[[赫鲁晓夫|赫鲁晓夫]]的垦荒政策（[[處女地運動|處女地運動]]）。1956年留任为党中央主席团候补委员和中央书记。1957年开除[[莫洛托夫|莫洛托夫]][[反黨集团|反黨集团]]成員后，升任主席团正式委员。1960年任苏联最高苏维埃主席团主席。1964年10月14日，勃列日涅夫与[[阿列克谢·柯西金|阿列克谢·柯西金]]、[[苏斯洛夫|苏斯洛夫]]、[[亚历山大·尼古拉耶维奇·谢列平|谢列平]]和[[波德戈尔内|波德戈尔内]]等人一起发动推翻赫鲁晓夫的[[十月政變|十月政变]]，逼迫赫鲁晓夫提前退休并就任苏共中央第一书记，勃列日涅夫之后与众人一起解除谢列平职务。1966年4月8日勃列日涅夫改任蘇共中央總書記。执政之初他待人温和，遇事总是与周围人商量后才决定，并不大权独揽，与[[部长会议主席|部长会议主席]][[柯西金|柯西金]]、波德戈尔内、苏斯洛夫等人一起实现了“[[列宁|列宁]]之后第一次[[集体领导|集体领导]]”。进入1970年代后期，勃列日涅夫的专权才渐渐浮现。<ref>[http://www.newsmth.net/nForum/#!article/History/327866 真实的勃列日涅夫时代 第一节：机遇赐予了还没准备好的没头脑]</ref>

在结束了[[赫鲁晓夫|赫鲁晓夫]]的无序而带有自由化色彩的[[改革|改革]]之后，勃列日涅夫的政策总体趋向于保守和僵化，其领导期间腐化和[[裙带关系|裙带风]]盛行。他在任期间，苏联的军事力量大大增强，[[核武器|核武器]]的数量超过美国。在他统治后期，由于超高的军费开支和失败的[[计划经济|计划经济]]，苏联经济已经停滞。

==总书记任内==
1964年10月，[[勃列日涅夫|勃列日涅夫]]和[[柯西金|柯西金]]通过政变推翻[[赫鲁晓夫|赫鲁晓夫]]上台执政，实行“静悄悄的斯大林化”，开始发表言论肯定斯大林之历史作用，并于1966年初决定终止“非斯大林化”。著名作家[[西尼亚夫斯基|西尼亚夫斯基]]和[[丹尼尔|丹尼尔]]因在国外发表反斯大林作品，而被控犯“诽谤 ”罪投入劳改营，《古拉格群岛》作者[[索尔仁尼琴|索尔仁尼琴]]被驱逐出境，《让历史来审判》作者[[罗伊·麦德维杰夫|罗伊·麦德维杰夫]]被开除出党，《斯大林大清洗内幕》作者[[列夫·费尔德宾|列夫·费尔德宾]]受到[[克格勃|克格勃]]追杀。勃列日涅夫时代拍摄之[[电影|电影]]《解放》（1972年上映）和《围困》（1977年上映）中，都出现了斯大林的形象，赞美斯大林在苏联卫国战争指挥中的功绩。勃列日涅夫还在1978年斯大林诞辰100周年时发行照相纸年历纪念册，并在1968年斯大林诞辰90周年时，在列宁墓背后的-{克里姆林宫}-城墙下中立了一座斯大林半身铜像，与[[基洛夫|基洛夫]]、[[伏罗希洛夫|伏罗希洛夫]]等人并列。但勃列日涅夫并没有重新把伏尔加格勒恢复为赫鲁晓夫[[去斯大林化|去斯大林化]]之前的名字--斯大林格勒，所有被赫鲁晓夫改名的城市、工厂、企业也都没有恢复原来带有斯大林的名字。勃列日涅夫也承认“[[苏共二十大|苏共二十大]]对破除[[个人崇拜|个人崇拜]]和个人迷信的贡献”，除了斯大林故乡[[格鲁吉亚|格鲁吉亚]]的[[哥里|哥里]]和首都莫斯科，苏联其他城市没有保留任何斯大林塑像。<ref>[http://www.newsmth.net/nForum/#!article/History/327866 真实的勃列日涅夫时代 第四节：亲斯大林感情与个人崇拜的重现]</ref>

赫鲁晓夫在1964年被勃列日涅夫赶下台，虽然在[[毛泽东|毛泽东]]领导下的[[中华人民共和国|中华人民共和国]]、[[霍查|霍查]]领导下的[[阿尔巴尼亚|阿尔巴尼亚]]和[[金日成|金日成]]领导下的[[朝鲜|朝鲜]]看来是延续[[修正主义|修正主义]]，是“没有赫鲁晓夫的赫鲁晓夫主义”（朝鲜是在内部文件中谴责苏联为修正主义，但在公开场合中于苏联和中国之间左右逢源，来回摇摆，保证自身的最大利益。<ref>[http://news.ifeng.com/history/gundong/detail_2012_05/21/14688135_0.shtml 凤凰网：[[中朝关系|中朝关系]]60年来多次面临重大考验]</ref>直到[[邓小平|邓小平]]和[[阿利雅|阿利雅]]于1980年代分别在中国和阿尔巴尼亚上台执政并进行经济改革，[[苏修|苏修]]这一说法才被废止不用）<ref>[http://page.renren.com/601004243/note/815962901 把反对赫鲁晓夫修正主义的斗争进行到底]</ref>但在西方各国和苏联国内主流观点看来，勃列日涅夫上台却被视为恢复斯大林主义，主要体现在[[勃列日涅夫主义|勃列日涅夫主义]]和苏联共产党官僚“干部队伍的稳定”，直到1980年代末期[[戈尔巴乔夫|戈尔巴乔夫]]实施改革开放政策，最后苏共下台为止。<ref>[Robert Conquest Reflections on a Ravaged Century (2000) ISBN 0-393-04818-7, page 101]</ref> 

勃列日涅夫时期，苏联政府与[[东正教会|东正教会]]的关系比之[[斯大林|斯大林]]时代的变化多端（在1930年代的打压消灭、[[卫国战争|卫国战争]]期间为打败[[希特勒|希特勒]]进行的结盟）渐趋成熟，由於[[馬克思主義與宗教|馬列主義的宗教觀點]]是[[無神論|無神論]]，[[苏联共产党|苏联共产党]]党员仍旧严禁信仰[[宗教|宗教]]，但[[苏联|苏联]]政府对非党员的人民宗教信仰的干涉已大为减少，此时苏联政府与东正教会的关系被称为“处于[[十月革命|十月革命]]以来最好的时期”。<ref>[http://bbs.tianya.cn/post-worldlook-569927-1.shtml 侧写勃列日涅夫时代的苏联]</ref>

对外方面他注重[[外交|外交]]，推行“[[主权有限论|主权有限论]]”，声称当[[华沙条约|华沙条约]]成员国的[[社会主义|社会主义]]政权受到威胁时，苏联可以进行武力干涉，此为著名的[[勃列日涅夫主义|勃列日涅夫主义]]。他在1968年派军队[[布拉格之春|鎮壓捷克斯洛伐克]]。1976年5月，他成为[[苏联元帅|苏联元帅]]。1977年至1982年他去世前最後五年，任[[苏联最高苏维埃|苏联最高苏维埃]]主席。1979年，由于[[阿富汗|阿富汗]]新政府取消了亲苏联的政策，他发动了[[阿富汗战争_(1979年)|阿富汗战争]]，成为导致苏联衰落和最终解体的重要因素之一。

尽管[[苏联|苏联]]在[[勃列日涅夫|勃列日涅夫]]时期极力在[[中东|中东]]地区扩张，扩大了苏联在中东的[[势力范围|势力范围]]，但当苏联支持的[[埃及|埃及]]和[[叙利亚|叙利亚]]在1973年的[[第四次中东战争|第四次中东战争]]中再次败给[[以色列|以色列]]之后，借助战后[[亨利·基辛格|亨利·基辛格]]在以、阿之间的穿梭外交谈判，[[哈菲兹·阿萨德|哈菲兹·阿萨德]]领导下的叙利亚开始谋求在[[美国|美国]]与苏联之间进行平衡外交，不再专一依赖苏联。[[穆罕默德·安瓦尔·萨达特|穆罕默德·安瓦尔·萨达特]]领导下的埃及与苏联彻底闹翻、反目成仇。<ref>[http://www.tianyayidu.com/article-a-614270-2.html 1974年基辛格通过对叙利亚的外交活动结束了阿盟对西方的石油禁运]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

任内他与[[毛泽东|毛泽东]]领导下的[[中华人民共和国|中华人民共和国]]于1969年在[[珍宝岛|珍宝岛]]、[[铁列克提|铁列克提]]等地爆发了[[中苏边界冲突|中苏边界冲突]]，之后苏联与中国关系在其任内长期没有改善，同为[[社会主义国家|社会主义国家]]的[[中国|中国]]逐渐成为对苏联来说仅次于[[北大西洋公约组织|北大西洋公约组织]]的死敌。但1982年3月24日，勃列日涅夫在紧临中国[[新疆维吾尔自治区|新疆维吾尔自治区]]的苏联[[乌兹别克加盟共和国|乌兹别克加盟共和国]]首府塔什干发表「[[塔什干講話|塔什干講話]]」，表示愿意改善对华关系，被認為是勃列日涅夫政府的對外政策變動的訊號。<ref>{{cite web |title=从"塔什干讲话"到"海参崴讲话":中苏关系正常化历程 |url=http://dangshi.people.com.cn/GB/16098926.html |website=人民网-中国共产党新闻网 |accessdate=2018-10-31}}</ref>

1955年，[[赫鲁晓夫|赫鲁晓夫]]访问[[南斯拉夫|南斯拉夫]]，[[铁托|铁托]]与[[苏联|苏联]]恢复正常外交关系。1956年[[匈牙利事件|匈牙利事件]]后，苏联与南斯拉夫关系一度又有恶化，后来随着[[中苏交恶|中苏交恶]]与决裂，苏南关系又迅速升温。<ref>[http://cpc.people.com.cn/GB/64184/64186/66672/4493667.html 1963年中共三评苏共中央公开信：南斯拉夫是真正的社会主义国家吗？]</ref>苏联政府在1968年镇压[[捷克斯洛伐克|捷克斯洛伐克]][[布拉格之春|布拉格之春]]后，苏联与南斯拉夫关系一度又出现紧张，铁托派特使[[爱德华·卡达尔|爱德华·卡达尔]]到北京面见毛泽东，中国与南斯拉夫关系改善。为防止中国与南斯拉夫亲密，勃列日涅夫宣布[[勃列日涅夫主义|勃列日涅夫主义]]宣称的苏联可以为了安全武装干预“[[社会主义国家|社会主义大家庭]]中任何一国”的范围不包括[[南斯拉夫|南斯拉夫]]，苏南关系才再度改善。<ref>[http://www.cccpism.com/moskow/viewussr/2/3/boke/b36.jpg 1969年勃列日涅夫会见铁托]</ref><ref>{{Cite web |url=http://www.2vi.org/w/waidianpingwenge/1971/09/143.htm |title=南斯拉夫发言人武伊察说：勃列日涅夫访问南斯拉夫“有助于推进南苏友好合作” |access-date=2013-09-08 |archive-url=https://web.archive.org/web/20131002170900/http://www.2vi.org/w/waidianpingwenge/1971/09/143.htm |archive-date=2013-10-02 |dead-url=yes }}</ref>尽管中国与南斯拉夫关系改善，但在内部文件中中国仍然谴责南斯拉夫为[[修正主义|修正主义]]国家，直到[[邓小平|邓小平]]上台，这一说法才停止使用。

==逝世==
{{主条目|勃列日涅夫之死}}

1982年11月10日，勃列日涅夫在苏联首都[[莫斯科|莫斯科]]逝世，享年76歲。死後苏联經歷長達三年的政治鬥爭、領導人更替的混亂局面。

11月12日，[[蔣經國|蔣經國]]函電[[宋美齡|宋美齡]]：「俄共[[勃列日涅夫之死|布里茲湼夫死亡]]，[[蘇聯|蘇俄]]本質不至驟變，但終必於[[世界|世界]]局勢產生影響。而匪俄關係之縱橫詭譎，尤其值得密切注意」<ref>[[周美華|周美華]]、[[蕭李居|蕭李居]]編，《蔣經國書信集——與宋美齡往來函電》(下)，[[台北|台北]]「[[國史館|國史館]]」出版，2009年，第330-331頁。</ref>。

==影响==
{{main|勃列日涅夫的影响}}
[[赫鲁晓夫|赫鲁晓夫]]在苏联国内的经济改革在1968年勃列日涅夫镇压[[捷克斯洛伐克|捷克斯洛伐克]][[布拉格之春|布拉格之春]]后陷入停滞。进入1970年代，在对市场社会主义掀起批判浪潮的情况下，苏联的经济改革从强调改革传统体制到强调[[计划经济|计划经济]]的优越性，由扩大企业自主权到加强对企业的控制，改革重心由调整国家与企业的关系转移到合并企业，建立联合公司。<ref>[http://euroasia.cass.cn/news/60875.htm 中国社会科学院俄罗斯东欧中亚研究所：论“布拉格之春”对苏联内政的影响] {{webarchive|url=https://web.archive.org/web/20130927023439/http://euroasia.cass.cn/news/60875.htm |date=2013-09-27 }}</ref>但改革在苏联在东欧的[[卫星国|卫星国]]（[[傀儡政权|傀儡政权]]），如[[卡达尔|卡达尔]]的[[匈牙利|匈牙利]]、[[爱德华·吉瑞克|盖莱克]]的[[波兰|波兰]]仍有延续，如卡达尔的匈牙利政府在1979年进一步允许把亏损的国营小型工商企业改由职工承包或租给私人经营、把[[私营企业|私营企业]]的合法雇工人数限制放宽到30人、除税收外政府不干预企业资金的流动、允许企业自主在社会上公开招聘管理人员等等。<ref>[http://bbs.tianya.cn/post-no05-221103-1.shtml 天涯：1988年苏联东欧市场化程度可能比当时的我国高]</ref>1983年，苏联[[人均国民生产总值|人均国民生产总值]]达到3710美元，是美国同时期的三分之一，而其卫星国[[东德|东德]]则达到了5670[[美元|美元]]，是全世界最富裕的自称遵循“[[马克思列宁主义|马克思列宁主义]]”为指导原则的国家。<ref>中国人民大学出版社2012年出版，（美国）罗伯特·A·达尔：《现代政治分析》 ISBN 9787300148724</ref><ref>[http://bbs.tianya.cn/post-no05-246564-1.shtml 天涯：侧写勃列日涅夫时代]</ref>但当苏联在勃列日涅夫命令下入侵[[阿富汗民主共和国|阿富汗民主共和国]]导致美国总统[[吉米·卡特|吉米·卡特]]对苏实施粮食禁运后，尤其是在苏联耗费大量财力应对美国总统[[罗纳德·里根|罗纳德·里根]]的“[[星球大战计划|星球大战计划]]”后，加之苏联[[石油|石油]]出口也受国际原油价格下落而缩水，苏联人民的[[日用品|日用品]]消费和[[副食|副食]]（如[[奶酪|奶酪]]、[[黄油|黄油]]）等的采购都受到明显影响，生活质量下降，1985年苏联人民每人每天要耗费2个小时的时间排长队购买食物。[[戈尔巴乔夫|戈尔巴乔夫]]改革也因此开启。<ref>[http://tieba.baidu.com/p/1448553319?pn=2 飞毛腿导弹的前世今生：戈尔巴乔夫为解除美国粮食禁运销毁“奥卡”导弹]</ref>

== 轶闻  ==

勃列日涅夫晚年开始大搞[[个人崇拜|个人崇拜]]，并嗜好勋章成癖，具有浓重的“勋章情结”<ref>{{cite news|url=http://www.chinanews.com.cn/2000-09-06/26/45077.html|title=勋章、石油和勃列日涅夫 |author=中国新闻网}}</ref>，有“勋章大王”之戏称，一生获得至少114枚[[勋章|勋章]]。1976年12月，70岁生日时获得“[[蘇聯英雄|蘇聯英雄]]”（一般只授予在军事上为苏联国家和社会做出伟大贡献的人士，苏联历史上只有[[朱可夫|朱可夫]]及[[勃列日涅夫|勃列日涅夫]]本人先后4次获得该[[勋章|勋章]]）以及[[苏联元帅|苏联元帅]][[军衔|军衔]]。同样出于庆生，他还获得[[列宁勋章|列宁勋章]]（苏联最高国家勋章）8次。1978年他还获得军队最高勋章“[[胜利勋章|胜利勋章]]”，成为二战结束后唯一获得该勋章人士。他死后，1989年苏联取消了他的这枚勋章资格。
