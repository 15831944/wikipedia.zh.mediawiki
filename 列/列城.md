'''列城'''（{{lang-en|Leh}}，{{Bo|z=གླེ་|w=Gle}}），是[[印度|印度]][[查谟-克什米尔邦|查谟-克什米尔邦]][[列城县|列城县]]的一个城镇。总人口30870人（2011年）。

== 人口 ==
该地2001年总人口27513人，其中男性16746人，女性10767人；0—6岁人口2440人，其中男1241人，女1199人；识字率75.00%，其中男性为81.72%，女性为64.54%。

===佛教===
列城人口比例最高的宗教为佛教，市域内有许多佛教建筑。
[[File:Thikse_Monastery_Front.jpg|缩略图]]
*[[提克西寺|提克西寺]]是一座[[藏传佛教|藏传佛教]][[格鲁派|格鲁派]]的寺院，位于[[列城|列城]]以东19公里的提克西村山顶<ref>Singh (2006), p.91</ref>。该寺与[[西藏|西藏]][[拉萨|拉萨]]的[[布达拉宫|布达拉宫]]相似而著称，是拉达克中部最大的寺庙。<ref name=Willis>{{cite web|last1=Willis|first1=Jan|title=The Cho-mos of Ladakh: From Servants to Practitioners|url=http://archive.thebuddhadharma.com/issues/2004/summer/willis.htm|publisher=Buddhadharma: The Practitioner's Quarterly|accessdate=2014-09-29|deadurl=yes|archiveurl=https://archive.is/20140929170643/http://archive.thebuddhadharma.com/issues/2004/summer/willis.htm|archivedate=2014-09-29}}</ref><ref>Singh （2006），p.91 </ref>
[[File:Shanti_Stupa,_Leh_-_2.jpg|缩略图]]
*[[香提佛塔|香提佛塔]]由日本僧人兴建于1983年-1991年，以促进世界和平和繁荣，以及纪念佛教2500年，[[第十四世达赖喇嘛|第十四世达赖喇嘛]]于1991年8月为香提佛塔揭幕。<ref name="Temples">{{cite web|url=http://www.buddhist-temples.com/buddhist-monastery/ladakh/shanti-stupa.html|title=Shanti Stupa|publisher=Buddhist-temples.com|accessdate=2009-10-19}}</ref><ref name = "bhasin">{{cite book|last= Bhasin|first=Sanjeev Kumar|title=Amazing land Ladakh: places, people, and culture|pages=119–20|chapter=Shanti Stupa|url=https://books.google.com/books?id=8IZloNzI8BgC&pg=PA120&dq=shanti+stupa+Ladakh&as_brr=3&client=firefox-a#v=onepage&q=shanti%20stupa%20Ladakh&f=false | isbn=978-81-7387-186-3 | year=2006 | publisher=Indus Publishing}}</ref><ref name="bajwa">{{cite book|last= Jagir Singh Bajwa, Ravinder Kaur|title=Tourism Management|pages=117|url=https://books.google.com/books?id=TVXCiwdwPxsC&pg=PA117&dq=shanti+stupa+Ladakh&lr=&as_brr=3&client=firefox-a#v=onepage&q=shanti%20stupa%20Ladakh&f=false | isbn=978-81-313-0047-3 | year=2007 | publisher=APH Publishing}}</ref>《[[印度教徒报|印度教徒报]]》称它是列城周围“最著名的旅游胜地”。

==交通==
===机场===
[[File:IXL_Front_2010.jpg|缩略图]]
{{le|列城巴库拉仁波切机场|Kushok Bakula Rimpochee Airport}}位于列城，它是世界上[[世界最高海拔机场列表|海拔第22高的商业机场]]，位于海平面以上3256米（10682英尺）。机场以[[印度|印度]]政治家兼僧侣{{le|库肖克·巴库拉·仁波切|19th Kushok Bakula Rinpoche}}的名字命名，他的[[贝图寺|贝图寺]]就在机场附近。该机场有航班通往印度境内的[[德里|德里]]、[[孟买|孟买]]、[[昌迪加尔|昌迪加尔]]、[[查谟|查谟]]、[[斯利那加|斯利那加]]。

==旅游==
===列城老城===
[[File:Leh_Old_Town_in_2009.jpg|缩略图]]
老城保存非常完整，由古老泥砖房和[[佛塔|佛塔]]、[[清真寺|清真寺]]组成。风格如[[布达拉宫广场|布达拉宫广场]]改建前拆除的那片村子一样。被称为复古版的[[拉萨|拉萨]]老城。<ref>{{Cite web |url = https://m.qyer.com/place/poi/V2UJYlFgBz5TZlI3/ |title = 列城老城 |language = zh |accessdate = 2019-03-08 }}</ref>老城内有许多重要的历史建筑，如贾玛清真寺（Jama Masjid），[[列城王宫|列城王宫]]（Leh Palace），{{le|纳杰东珠提西莫修道院|Namgyal Tsemo Monastery}}（Namgyal Tsemo Monastery）以及列城的第一个电影院。<ref>{{Cite web |url = https://www.lehladakhindia.com/old-town/ |title = OLD TOWN: LADAKH TRAVEL GUIDE |language = en |accessdate = 2019-03-08 }}</ref>

====贾玛清真寺====
[[File:Leh_Old_Town_18_(Friar's_Balsam_Flickr).jpg|缩略图]]
贾玛清真寺位于列城市中心，是[[拉达克|拉达克]]的主要历史清真寺之一。这座清真寺被认为是拉达克地区最大的清真寺，由一座名为沙希哈姆丹（Shahi Hamdan）的纪念碑组成。这座纪念碑是为了纪念伊斯兰教[[苏非派|苏非派]]圣徒{{le|米尔·赛义德·阿里·哈马达尼|Mir Sayyid Ali Hamadani}}。清真寺建于公元1666~1667年。最初修建的清真寺已被拆除，目前的清真寺采用了新技术重新修建。<ref>{{Cite web |url = https://www.lehladakhindia.com/jama-masjid-ladakh/ |title = JAMA MASJID: LADAKH TRAVEL GUIDE |language = en |accessdate = 2019-03-08 }}</ref>

====列城王宫====
{{主条目|列城王宫}}
[[File:Leh_Palace_2011.jpg|缩略图]]
列城王宫由[[南嘉王朝|南嘉王朝]]国王[[僧格南嘉|僧格南嘉]]在17世纪建造，造型模仿了[[西藏|西藏]][[拉萨|拉萨]]的[[布达拉宫|布达拉宫]]，高九层，上部是皇家住所，下部是马厩和储藏室。19世纪中叶多加拉部队控制拉达克后，宫殿废弃。现在宫殿已被印度考古部门修复，向公众开放，在屋顶可欣赏列城及周边地区群山的全景。<ref>{{cite web  | last = Staff |language = en | title = Monuments of India : Leh Palace  | publisher = ''iloveIndia.com''  | url = http://www.iloveindia.com/indian-monuments/leh-palace.html  | accessdate = 2010-02-04}}</ref>
