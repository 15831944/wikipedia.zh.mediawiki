{{托洛茨基主义}}
'''列夫·达维多维奇·托洛茨基'''（{{lang-ru|Лев Давидович Троцкий}}，{{IPA-ru|ˈlʲɛf ˈtrotskʲɪj|發音|ru-Leon Trotsky.ogg}}；{{bd|1879年|11月7日|1940年|8月21日|catIdx=T}}），本名'''列夫·达维多维奇·布隆施泰因'''（{{lang|ru|Лев Давидович Бронштейн}}），是[[布尔什维克|布尔什维克]]主要领导人、[[十月革命|十月革命]]指挥者、[[苏联红军|苏联红军]]缔造者和[[第四国际|第四国际]]精神领袖，是一名[[革命家|革命家]]、[[军事家|军事家]]、[[政治哲學|政治理论家]]和[[作家|作家]]。

托洛茨基原先在[[俄國社會民主工黨|俄國社會民主工黨]]中的[[布尔什维克|布尔什维克]]（多数派）和[[孟什维克|孟什维克]]（少数派）之间摇摆不定。在1917年[[十月革命|十月革命]]之前，他最终决定加入[[布爾什維克|布爾什維克]]，并成為[[俄国社会民主工党（布尔什维克）|俄国社会民主工党（布尔什维克）]]的主要領導人之一。苏俄建國之初，他擔任[[蘇聯外交部|苏俄外交人民委员]]，後任苏俄軍事和海軍事務人民委员，建立了[[苏联红军|苏联红军]]，並出任總司令。他帶領布爾什維克赢得了[[俄國內戰|俄國內戰]]（1918年－1923年），並成為这一勝利的精神象徵。[[列宁|列宁]]死后他被排挤出苏共领导核心，后流亡海外，1940年被[[斯大林|斯大林]]派人刺杀于[[墨西哥城|墨西哥城]]。

他提出的政治理论被称为[[托洛茨基主义|托洛茨基主义]]。

== 早年 ==
[[File:LeonTrotsky1897.jpg|thumb]]
[[File:Leon_trotsky.jpg|thumb]]
列夫·达维多维奇·布隆施泰因，1879年生于[[沙俄|沙俄]][[赫尔松|赫尔松]]省亚诺夫卡村（今属于[[乌克兰|乌克兰]][[别列斯拉夫卡_(博布里涅茨区)|别列斯拉夫卡]]）一个[[犹太|犹太]][[富农|富农]]家庭。在[[敖德萨|敖德萨]]与工业城市[[尼古拉耶夫|尼古拉耶夫]]受中等教育，因参加革命活动于1898年1月被捕，並放棄了原本取得數學學位的目標。在等待沙俄法庭判决的两年时间内，被关押在布特尔监狱，并举行了由[[拉比|拉比]]主持的按犹太宗教仪式进行的婚礼，与年长7岁的[[亚历山德拉·索科洛夫斯卡娅|亚历山德拉·索科洛夫斯卡娅]]结婚。婚后索科洛夫斯卡娅跟随列夫·布隆施泰因去了西伯利亞[[伊尔库茨克|伊尔库茨克]]省的流放地，1901年、1902年长女、次女出生。

1902年夏天，列夫·布隆施泰因离开妻子与两个女儿，使用了一位來自[[敖德萨|敖德萨]]姓“托洛茨基”的狱友的护照逃离流放地。他的两个幼女后来由他在故乡亚诺夫卡村生活富裕的父母抚养长大。托洛茨基在[[萨马拉|萨马拉]]、[[基辅|基辅]]、[[哈尔科夫|哈尔科夫]]、[[波尔塔瓦|波尔塔瓦]]等地从事革命活动，并给《[[火星报|火星报]]》撰稿。1902年年底去[[巴黎|巴黎]]从事争取俄国侨民活动，并在[[英国|英国]][[伦敦|伦敦]]加入了由[[普列汉诺夫|普列汉诺夫]]、[[列宁|列宁]]、[[马尔托夫|马尔托夫]]主持的《[[火星报|火星报]]》编辑部。1903年托洛茨基在[[巴黎|巴黎]]结识了欧洲大银行家日沃托夫斯基的女儿[[娜塔莉娅·谢多娃|娜塔莉娅·谢多娃]]。不久二人在巴黎结婚成为终身伴侣，在1906年与1908年生了两个儿子。

== 革命生涯 ==
[[File:Leo_Trotzki_Oktober_1917.jpg|thumb]]
[[File:Leo_Trotzki_1918.jpg|thumb]]的缔造者托洛茨基1922年身着军装]]
1903年7月托洛茨基以代表身份参加了[[俄国社会民主工党|俄国社会民主工党]]第二次代表大会，会后不久[[俄国社会民主工党|俄国社会民主工党]]分裂为[[布尔什维克|布尔什维克]]与[[孟什维克|孟什维克]]两个派别时，托洛茨基是孟什维克成员，并积极参加为[[马尔托夫|马尔托夫]]主持领导的新《[[火星报|火星报]]》撰稿，称列宁为“马克西米利安·列宁”（马克西米利安是[[罗伯斯庇尔|罗伯斯庇尔]]的名字）、批评列宁的路线是“{{link-en|雅各宾主义|Jacobinism|雅-{各}-宾主义}}”。1904年9月托洛茨基宣布脱离与俄国自由派结盟的[[孟什维克|孟什维克]]。从此直到1917年托洛茨基一直是[[布尔什维克|布尔什维克]]与[[孟什维克|孟什维克]]之外的中间派。

1905年一得知沙俄即将爆发革命消息，托洛茨基立即于二月潜回，先在[[基辅|基辅]]，很快来到[[彼得堡|彼得堡]]从事报纸新闻工作推动革命形势发展，后以扬努夫斯基的化名当选为[[圣彼得堡苏维埃|圣彼得堡苏维埃]]副主席，苏维埃主席被捕后，11月26日到12月3日当选为苏维埃主席。12月3日被捕，1906年被判终身流放。

1907年1月托洛茨基逃亡伦敦，开始了十年国外生涯。这时期托洛茨基的演讲和文章在设计革命运动的理论问题和实际问题上都同列宁进行论战。1913年托洛茨基在写给孟什维克领袖[[齐赫泽|齐赫泽]]的信中这样写道“目前列宁主义的大厦整个建筑在谎言和伪证之上，包含着自身腐败的有毒因素”，“列宁是俄国工人运动中一切落后现象的职业得利者”。

[[第一次世界大战|第一次世界大战]]中，托洛茨基站在反战的国际主义一边，但反对列宁的“变现代帝国主义战争为国内战争”“使本国政府在帝国主义战争中失败”的主张，反对列宁提出的退出[[第二国际|第二国际]]成立[[第三国际|第三国际]]的主张。1916年9月，托洛茨基被法国政府驱逐，年底抵达美国，加入在[[纽约|纽约]]由[[布哈林|布哈林]]主办的俄语报纸《新世界报》编辑部。

[[俄國二月革命|二月革命]]的消息传来后，1917年3月27日，托洛茨基离美回国，但被[[加拿大|加拿大]]的[[英国|英国]]当局扣留了一个月，后[[俄国临时政府|俄国临时政府]]总理[[克伦斯基|克伦斯基]]、外交部长[[米留可夫|米留可夫]]要求[[英国|英国]]政府释放托洛茨基。回国后，参加“[[区联派|区联派]]”。

== 领导苏俄 ==
之后由于公开表示支持[[列宁|列宁]]等[[布尔什维克|布尔什维克]]领导人的主张而被临时政府逮捕入狱，入狱期间[[区联派|区联派]]加入[[俄国社会民主工党（布尔什维克）|俄国社会民主工党（布尔什维克）]]，托洛茨基以第三高票当选布尔什维克党中央委员（少[[列宁|列宁]]两票）、之后又当选布尔什维克党第一届[[政治局|政治局]]委员，出狱后当选彼得格勒苏维埃主席。是[[十月革命|十月革命]]的主要领导人之一。[[十月革命|十月革命]]胜利之后历任[[外交人民委员|外交人民委员]]、[[军事人民委员|军事人民委员]]、[[革命军事委员会主席|革命军事委员会主席]]、[[铁道人民委员|铁道人民委员]]。

1918年到1922年间，在[[苏联国内战争|苏联国内战争]]中组织、领导[[苏联红军|苏联红军]]對抗支持沙皇的[[白军|白军]]，对内战胜利和[[苏联|苏联]]反抗[[美国|美国]]、[[英国|英国]]、[[法国|法国]]、[[波兰|波兰]]、[[日本|日本]]、[[芬兰|芬兰]]、[[爱沙尼亚|爱沙尼亚]]、[[拉脱维亚|拉脱维亚]]、[[立陶宛|立陶宛]]和[[中华民国|中国]]等[[协约国武装干涉俄国内战|十四国武装干涉]]作出了无与伦比的贡献。内战和武装干涉结束后，托洛茨基与[[布尔什维克|布尔什维克]]中央组织策劃了对波兰的反攻，结果在[[华沙|华沙]]城下遭到了巨大的失败。

== 路線之爭 ==
[[托洛茨基|托洛茨基]]主張[[十月革命|十月革命]]後應以先支援其他國家共產黨革命為最優先路線，[[斯大林|斯大林]]則是以先鞏固建設[[蘇聯|蘇聯]]這世界上第一個[[社会主义国家|社会主义国家]]為優先路線，後行有餘力再支援[[世界革命|世界革命]]。

== 流亡海外及被杀 ==
[[列宁|列宁]]去世后，托洛茨基和[[布哈林|布哈林]]、[[斯大林|斯大林]]、[[季诺维也夫|季诺维也夫]]、[[加米涅夫|加米涅夫]]、[[李可夫|李可夫]]并列为[[俄国共产党（布尔什维克）|俄国共产党（布尔什维克）]]主要的六名领导人。但在[[列宁|列宁]]病重和逝世期间，托洛茨基遭到了其他[[政治局|政治局]]委员的共同反对而失势；至1927年10月，因为他极力反对[[斯大林|斯大林]]的[[独裁|独裁]]政策，主张[[世界革命|世界革命]]而被开除出党，斯大林大整肅后被流放、驱逐。先后流亡[[土耳其|土耳其]]、[[法国|法国]]、[[挪威|挪威]]直到[[墨西哥|墨西哥]]。1938年，他在[[墨西哥|墨西哥]]与支持者创建了[[世界社会主义革命党|世界社会主义革命党]]，即第四国际，与斯大林控制的[[共产国际|共产国际]]（[[第三国际|第三国际]]）对抗。他先是住在[[墨西哥共产党|墨西哥共产党]]成员、著名画家[[迭戈·里维拉|迭戈·里维拉]]和[[弗里达·卡罗|弗里达·卡罗]]夫妇的家中。因为和弗里达有染而搬到了几条街之外的科瑶坎另一处房子，期间躲过多次暗杀。1940年，他被蘇聯情報單位[[特務|特務]][[拉蒙·麥卡德|拉蒙·麥卡德]]用[[冰镐|冰镐]]凿入后脑，第二天不治身亡。

== 著作 ==
[[File:MyLife_Trotsky.jpg|right]]》書影]]
著有《不断革命》、《俄国革命史》、《文学与革命》、《[[斯大林评传_(托洛茨基的书)|斯大林评传]]》、《他们的道德与我们的道德》、《我的生平》等書，其中以《[[俄国革命史|俄国革命史]]》最为出名。而波兰托派作家多伊彻所著《[[先知三部曲|先知三部曲]]》是研究托洛茨基生平与思想最权威的著作。
