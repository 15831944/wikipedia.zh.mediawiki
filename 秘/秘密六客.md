{{作品名稱|name=''The Secret Six''|hk=秘密六客|cn=六个秘密}}是一部1931年的美國{{link-en|前編碼荷里活|Pre-Code Hollywood|前編碼}}犯罪電影，由{{link-en|喬治·希爾 (導演)|George Hill (director)|喬治·W·希爾}}執導，[[弗朗西絲·馬里恩|弗朗西絲·馬里恩]]編劇。主演是[[華理士·勃利|華理士·勃利]]，飾演"Slaughterhouse Scorpio"，這個角色非常鬆散地基於[[艾爾·卡彭|艾爾·卡彭]]，其他演員有{{link-en|劉易斯·史東|Lewis Stone}}、{{link-en|強尼·梅克·布朗|Johnny Mack Brown|約翰·梅克·布朗}}、[[珍·哈露|珍·哈露]]、[[奇勒·基寶|奇勒·基寶]]、{{link-en|雷夫·貝拉米|Ralph Bellamy}}和{{link-en|馬喬里·蘭比奧|Marjorie Rambeau}}。<ref>{{Cite web |url = http://www.tcm.com/tcmdb/title/3175/The-Secret-Six/ |title = THE SECRET SIX(1931) |publisher = Turner Classic Movies |language = en}}</ref>

== 劇情 ==
私酒販Johnny Franks招募一個名叫Louis "Louie" Scorpio的粗魯工人，成為黑幫老大Richard "Newt" Newton幫派的一員。Scorpio最終成為了組織的領導者。然後，他被一個受報紙記者Carl Luckner和Hank Rogers協助下的六名蒙面的打擊犯罪鬥士組成的秘密組織起訴。
