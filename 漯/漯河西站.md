'''漯河西站'''为[[京广高速铁路|京广高速铁路]]的一座中间站，也是规划中的[[洛周客运专线|洛周客运专线]]的中间车站，于2012年9月28日投入使用。

== 简介 ==
西站总建筑面积1.6万平米，站房主体一层、局部三层，内设4个站台。其中候车厅5530平米，分一等区、二等区和贵宾区，可容纳1000人；售票厅1200多平米，设公安办证窗口、自动售票机。<ref name="通车">{{cite web |url=http://news.shangdu.com/luohe/20120920/128_5700897.shtml |language= |title=石武高铁通车在即 漯河西站随时迎客 |publisher=漯河名城网 |date=2012年9月20日 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20120926164335/http://news.shangdu.com/luohe/20120920/128_5700897.shtml |archivedate=2012年9月26日 }}</ref>

西站广场面积9万平米。广场南侧为公众停车场、[[出租车|出租车]]停靠点，西侧为预留的[[城际轻轨|城际轻轨]]，北侧规划有长途客运总站。<ref name= "通车"/>

预计高铁运营后，西站日发送旅客在一千人左右。<ref>{{cite web |url=http://news.shangdu.com/luohe/20120928/128_5704656.shtml |language= |title=漯河西站高铁首发日：预定车票300多张 |publisher=漯河日报 |date=2012年9月28日 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20130610055246/http://news.shangdu.com/luohe/20120928/128_5704656.shtml |archivedate=2013年6月10日 }}</ref>

==事故==
*2017年3月18日，漯河西站的站名LED灯短路着火。
