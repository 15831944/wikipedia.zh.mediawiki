'''漯河经济技术开发区'''是位于[[河南省|河南省]][[漯河市|漯河市]]的[[国家级经济技术开发区|国家级经济技术开发区]]。2009年工业销售收入450亿元，在国内生产总值、工业总产值、上缴税金、合同利用外资、实际利用外资等5项指标均位列[[河南省|省内]]第二位。<ref name="授牌">{{cite web |url=http://henan.people.com.cn/news/2010/12/15/516797.html |language= |title=漯河经济技术开发区授牌 总规划面积41平方公里 |publisher=人民网河南频道 |date=2010年12月15日 |accessdate= }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 简介 ==
开发区创建于1992年，位于漯河市区东南部<ref name="简介">{{cite web |url=http://www.lhjk.gov.cn/list/1952/1952.html |language= |title=开发区简介 |publisher=漯河经济技术开发区 |date= |accessdate=2012年9月 |deadurl=yes |archiveurl=https://web.archive.org/web/20160307051629/http://www.lhjk.gov.cn/list/1952/1952.html |archivedate=2016年3月7日 |df= }}</ref>。总规划面积41平方公里，建成区面积10平方公里，下辖一个[[后谢乡|后谢乡]]，31个行政村（居委会），总人口近10万，2007年更名为'''河南漯河经济开发区'''。<ref name= "授牌"/>

== 著名企业 ==
到目前已吸引了600多家企业在此投资<ref name= "简介"/>，形成了以生物[[食品|食品]]为主导，高新技术、机械加工、商贸物流共同发展的产业格局。
*有美国[[杜邦|杜邦]]、美国[[高盛|高盛]]、[[日本火腿|日本火腿]]株式会社、日本丸红株式会社等8家[[世界500强|世界500强]]企业<ref>{{cite web |url=http://henan.people.com.cn/news/2011/09/12/566988.html |language= |title=漯河经济技术开发区：延伸产业链条 坚持招大引强 |publisher=人民网河南频道 |date=2011年9月12日 |accessdate= }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
*[[双汇|双汇]]、[[银鸽集团|银鸽]]、[[中粮|中粮]]、[[可口可乐|可口可乐]]、[[旺旺|旺旺]]、[[统一企业|统一企业]]<ref>{{cite web|url=http://news.xinmin.cn/rollnews/2011/07/06/11319021.html | language= | title=台湾统一集团入驻河南漯河经济技术开发区 |publisher=中国台湾网 |date=2011-07-06 |accessdate= }}</ref>、[[康师傅|康师傅]]、[[东方华垦粮油工业有限公司|华垦]]为代表的10多个专业园区<ref name= "授牌"/>
