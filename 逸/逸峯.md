'''逸峯'''（{{lang-en|'''Green Code'''}}）位處於[[香港|香港]][[新界|新界]][[粉嶺|粉嶺]][[馬適路|馬適路]]1號，是[[香港小輪|香港小輪]]與[[恒基兆業|恒基兆業]]合作發展的住宅項目。

==歷史==
2005年2月，[[香港房屋委員會|香港房屋委員會]]在[[北區區議會|北區區議會]]中，建議該地段興建單幢式的[[公屋|公屋]]<REF>[http://www.districtcouncils.gov.hk/archive/north_d/pdf/2005/N_2005_005_TC.pdf 香港房屋委員會擬在粉嶺沙頭角道興建之公共房屋]</REF>，預計可興建1,000個單位，但因被[[綠悠軒|綠悠軒]]、[[御庭軒|御庭軒]]及[[帝庭軒|帝庭軒]]的居民強烈反對而擱置，最終一直用作為臨時停車場<REF>[http://www.districtcouncils.gov.hk/archive/north_d/pdf/2005/N_2005_017_TC.pdf 強烈要求房委會擱置於馬適路與沙頭角道交界興建房屋計劃 北區區議會 2005年]</REF>。根據2014年[[審計署|審計署]]第62號報告，房署為此發展項目共損失超過400萬元。

土地於2010年4月被發展商以10.538億元勾出<REF>[http://orientaldaily.on.cc/cnt/finance/20100417/00204_001.html 粉嶺地低市價45%勾出 東方日報 2010年4月17日]</REF>，最終由[[香港小輪|香港小輪]]以13.3億元投得。<ref>[http://www.property.hk/news_content.php?author=PHK_NEWSPROP&id=9074 粉嶺細地搶上天價 東方日報 2010年9月30日]</ref>

==住宅==
逸峯物業共分為4座，每座樓高29層，單位總數為728伙，於2014年第3季落成。<ref>[http://orientaldaily.on.cc/cnt/finance/20130311/00204_003.html 最快明日推396伙 逸峯勁收千票]《東方日報》2013年03月11日</ref><ref>[http://www.hkej.com/template/property/php/detail.php?title_id=14217 新盤巡禮22:粉嶺逸峯]《信報》2013年2月6日</ref>

逸峯設住客會所「逸峯薈Club Delight」，樓高兩層，面積約4萬方呎，設施包括長約38米音樂泳池、宴會廳、兒童遊戲室、瑜伽室、舞蹈室、豪華3D電影院、桑拿室、音樂室、卡拉OK室、閱覽室、網絡空間等。

==商場==
商場名為逸峯廣場（Green Code Plaza），樓高兩層，面積約為12萬平方呎，原預計2015年6月開業，至同年12月18日正式開業，現時已進駐的主要商戶包括[[759阿信屋|759阿信屋]]、[[佳宝超級市場|佳宝超級市場]]、7-11便利店及[[薩莉亞|薩莉亞]]意式餐廳等，不少商戶都在[[粉嶺中心|粉嶺中心]]設有分店。商場設有行人天橋連接[[聯和墟市政大廈|聯和墟市政大廈]]，經行人天橋網絡可前往[[聯和墟|聯和墟]]及鄰近建築。

2015年12月，[[洲立影藝|洲立影藝]]證實，地下將開設戲院，設3個影院，以照顧粉嶺及鄰近新界東地區居民的娛樂需要，於2017年1月21日正式開幕。<ref>[http://hk.on.cc/hk/bkn/cnt/news/20151228/bkn-20151228154518871-1228_00822_001.html 闊別6年戲院重臨北區　粉嶺MCL明秋開幕 on.cc東網 2015-12-28]</ref><ref>https://hk.news.yahoo.com/粉嶺好戲星期六-浪漫草地免費睇好戲-mcl-終於開幕喇-072954678.html Yahoo新聞</ref>
