'''辜莞允'''（{{bd|1992年|2月16日|||}}），台灣女藝人，英文名Nono。原就讀[[東吳大學|東吳大學]]日文系，重考後就讀[[世新大學|世新大學]]新聞系。於2014年拍攝警察宣導桌曆，於2016年受[[新北市政府警察局|新北市政府警察局]]三峽分局及[[臺北市政府警察局|臺北市政府警察局]]中山分局邀約於三峽分局北大派出所拍攝犯罪預防影片。2019年4月，因為在外景時偷拍同房藝人[[徐瑋吟|徐瑋吟]]淋浴照並傳送照片給男友而被徐瑋吟所屬經紀公司告上法院。

==偷拍爭議==
2019年4月17日，《鏡周刊》於網路刊登報導，稱接獲匿名人士爆料，指辜莞允在錄製三立電視《愛玩客》節目帛琉外景期間，以手機偷拍同房藝人、《愛玩客》固定主持人[[徐瑋吟|徐瑋吟]]（鮪魚）的淋浴照片並傳給男友分享。由於該趟外景中，徐瑋吟不慎弄丟了手機，借用了辜莞允的手機才意外得知此事。徐瑋吟返台後將此事告知經紀公司，經紀公司決定提告。<ref>{{Cite web |url = https://www.mirrormedia.mg/story/20190415ent027/ |title = Nono偷拍鮪魚露點照　傳給男友看遭控告 |author = 鏡周刊娛樂組 |date = 2019-04-17 |publisher = 鏡周刊 |language = zh-tw |accessdate = 2019-04-18}}</ref>

報導刊出後，辜莞允第一時間在臉書貼文否認，並指「我方將保留法律追訴權，謝謝大家關心。」<ref name="A">{{Cite web |url = https://www.chinatimes.com/realtimenews/20190417001176-260404?chdtv|title = 辜莞允偷拍入浴照給男友看 徐瑋吟：心受傷是事實 |author = 蘇士亨 |date = 2019-04-17 |publisher = 中時電子報 |language = zh-tw |accessdate = 2019-04-18}}</ref>。但是徐瑋吟在同日稍晚於臉書貼文，稱全案已進入法律程序，接下來將統一由公司回應<ref name="A" />。辜莞允經紀人則稱是玩笑開過火，已對徐瑋吟致歉並談和解<ref>{{Cite web |url = https://star.ettoday.net/news/1424465 |title = 辜莞允經紀人：都是誤會一場！　偷拍後已簽切結書「在談和解」|author = Ettoday娛樂中心 |date = 2019-04-17 |publisher = Ettoday新聞雲 |language = zh-tw |accessdate = 2019-04-18}}</ref>。4月18日，徐瑋吟錄製狼谷競技台《GAME什麼東西》節目，在節目主持人[[阿達|阿達]]與現場嘉賓的陪同下，在事發後首度受訪，在面對「是否接受辜莞允道歉」的提問時說：「誠心道歉是必要」<ref>{{Cite web |url = https://www.nownews.com/news/20190418/3330977/|title = 「閨密偷拍風波」後首露面　徐瑋吟淚崩：誠心道歉是必要 |author =臺孝婷|date = 2019-04-18 |publisher = 今日新聞網 |language = zh-tw |accessdate = 2019-04-18}}</ref>。同日稍晚，該次外景同行者之一，《愛玩客》固定主持人[[小鐘|小鐘]]出面證稱，當天是他在幫忙拍照時不慎讓徐瑋吟的手機落海，無手機可用的徐瑋吟回房後借用辜莞允的平板，才發現她在房內換泳裝的時候被偷拍<ref>{{Cite web |url = https://www.nownews.com/news/20190418/3330977/|title = 小鐘作證「鮪魚裸照外洩」兇手是她！　還原偷拍過程…證據全都在|author =林彥君、劉宜庭|date = 2019-04-18 |publisher = Ettoday新聞雲 |language = zh-tw |accessdate = 2019-04-19}}</ref>。

2019年4月19日，辜莞允在IG以及臉書發表八點聲明，指照片是在房間內拍攝，只是穿著比較清涼，絕無露點，除徐瑋吟以外，其本人也有入鏡。雙方在當地有在第三人見證下簽協議書，只是返台後雙方經紀公司對和解條件一直談不攏，才拖延至今。辜莞允亦表示，不會回應扭曲事實的爆料，將會主動配合司法單位調查，不會逃避責任。<ref>{{Cite web |url = https://star.ettoday.net/news/1425430|title = 辜莞允發八點聲明 「絕對沒有露點照」 |author = 蘇士亨|date = 2019-04-19 |publisher = 中時電子報 |language = zh-tw |accessdate = 2019-04-19}}</ref>
