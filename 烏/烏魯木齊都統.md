'''烏魯木齊都統'''，[[清代|清代]]官名，是駐防於[[新疆|新疆]][[烏魯木齊|烏魯木齊]]的[[都統|都統]]、[[駐紮大臣|駐紮大臣]]。官秩為從一品。為清代所設的四個[[都統|駐防都統]]之一。

[[乾隆|乾隆]]二十年（1755年）清兵[[準噶爾之役|滅準噶爾]]，之後於烏魯木齊城置[[鎮迪道|安西]]提標綠旗五營。乾隆二十四年（1759年）十一月，置烏魯木齊辦事大臣，首任大臣為[[安泰|安泰]]。乾隆三十六年（1771年）十一月，改滿兵駐防，置參贊大臣二人，由甘肅安西提督[[覺羅巴彥弼|覺羅巴彥弼]]任。三十八年（1773年），再增置領隊大臣二人。乾隆四十八年（1783年）五月，裁烏魯木齊參贊大臣、領隊大臣，置烏魯木齊都統、副都統各一人。以原任參贊大臣[[索諾穆策凌|索諾穆策凌]]為都統。同時，改烏魯木齊同知為[[迪化州|迪化州]][[知州|知州]]。

烏魯木齊都統統轄烏魯木齊（[[迪化州|迪化州]]）及[[吐魯番廳|吐魯番廳]]、[[哈密廳|哈密廳]]、[[鎮西府_(清朝)|巴里坤]]、[[古城|古城]]、[[庫爾喀喇烏蘇|庫爾喀喇烏蘇]]等地駐防官軍。其下有協領六人，佐領、防禦、驍騎校各二十四人。吐魯番領隊大臣、哈密辦事大臣、巴里坤領隊大臣、古城領隊大臣、庫爾喀喇烏蘇領隊大臣歸其節制。[[光緒|光緒]]八年（1882年）籌建[[新疆省|新疆省]]，次年裁撤烏魯木齊都統等各處大臣。

== 歷任都統 ==
# [[索諾穆策凌|索諾穆策凌]]，蒙古鑲黃旗。乾隆三十七年（1772年）三月任烏魯木齊參贊大臣。次年任都統。四十五年（1780年）遷[[盛京將軍|盛京將軍]]。四十七年（1782年）因[[甘肅冒賑案|甘肅冒賑案]]發，被乾隆帝賜死。
# [[奎林|奎林]]，滿洲鑲黃旗。乾隆四十五年（1780年）任烏魯木齊都統。
# [[明亮|明亮]]，滿洲鑲黃旗。乾隆四十六年（1781年）任。
# [[海祿|海祿]]，乾隆四十八年（1783年）署理。
# [[長清_(人名)|長清]]，海祿子，乾隆四十九年（1784年）任。
# 奎林，乾隆五十年（1785年）再任。
# [[永鐸|永鐸]]，乾隆五十年（1785年）代奎林為烏魯木齊都統。
# [[尚安|尚安]]，乾隆五十二年（1787年）任。
#明興（富察氏，[[明亮|明亮]]兄弟，喀什噶爾參贊大臣），乾隆五十九年（1794年）署理<ref>（乾隆朝實錄 卷1445）○又谕曰、尚安奏请入觐。伊自往乌噜木齐。将及十载。朕欲面询该处情形。尚安即著来京。所有乌噜木齐都统印务。著明兴署理。</ref>。
