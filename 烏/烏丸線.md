'''烏丸線'''（{{jpn|j=烏丸線|hg=からすません|rm=Karasuma sen|f=y}}）是自日本[[京都府|京都府]][[京都市|京都市]][[左京區|左京區]]的{{link-ja|國際會館車站|国際会館駅}}至京都市[[伏見區|伏見區]]的{{link-ja|竹田車站 (京都府)|竹田駅 (京都府)|竹田車站}}的[[京都市營地下鐵|京都市營地下鐵]]的路線。烏丸線是京都最初的市營地下鐵路線。

== 概要 ==
北大路車站 - 十條車站區間位於[[烏丸通|烏丸通]]的地下，南北縱貫京都市中心。現在所有列車均為6車廂編成運行。開業当初暫以4車廂編成，之後恢復成現行編制。

=== 路線資料 ===
* 路線距離（{{link-ja|營業距離|営業キロ}}）：13.7公里
* [[軌距|軌距]]：1435毫米
* 站數：15個（包括起終點站）
* 複線路段：全線
* 電氣化路段：全線電氣化（直流1500V、[[高架電纜|高架電纜]]）
* 地面路段：竹田站附近
* [[閉塞_(鐵路)|閉塞方式]]：車內信號式（[[自動列車制御裝置|ATC]]）
* 最高速度：75公里/小時
* 總建設費：3054億日圓<ref group="注釋">北大路、京都間 1236億日圓，京都、竹田間 1115億<!--12百万-->日圓，北山、北大路間 247億<!--54百万-->日圓，國際會館、北山間 456億<!--42百万-->日圓。<!--億円以下の部分を足すと、1億円加算。--></ref>

=== 路線特徴 ===
沿線有以下大學，因此烏丸線的學生乘客頗多。
* [[京都精華大學|京都精華大學]]（自國際会館車站有穿梭巴士。但更多學生利用[[叡山電鐵|叡山電鐵]][[叡山電鐵鞍馬線|鞍馬線]][[京都精華大前車站|京都精華大前車站]]）
* [[京都工藝繊維大学|京都工藝繊維大学]]（也有較多利用松崎車站、叡山電鐵[[修學院車站|修學院車站]]的學生）
* [[京都府立大學|京都府立大學]]、[[京都ノートルダム女子大學|京都ノートルダム女子大學]]（北山車站）
* [[京都產業大學|京都產業大學]]（主要在北大路車站、國際會館車站換乘巴士。）
* [[大谷大學|大谷大學]]、[[大谷大学短期大學部|短期大學部]]（北大路車站）
* [[同志社大學|同志社大學]]、[[同志社女子大學|同志社女子大學]]（今出川校地：今出川車站）
* [[平安女學院大學|平安女學院大學]]（京都校區：丸太町車站）
* [[佛教大學|佛教大學]]（紫野校區：自北大路車站換乘巴士、四條中心：四條車站）
* [[龍谷大學|龍谷大學]]、[[龍谷大學短期大學部|短期大學部]]（深草校舎：くいな橋駅、大宮校舎：京都車站。單深草校舎距[[京阪本線|京阪本線]][[深草車站|深草車站]]較近）

也有眾多觀光客乘坐烏丸線，沿線的主要[[觀光地|觀光地]]有[[京都御所|京都御所]]（今出川車站、丸太町車站）和[[東本願寺|東本願寺]]（[[京都車站|京都車站]]、[[五條車站_(京都府)|五條車站]]）。前往其他觀光地則多換乘[[京都市營地下鐵東西線|東西線]]，或自地下鐵各車站換乘[[京都市營巴士|京都市巴士]]、[[京都巴士|京都巴士]]等公共汽車。

== 運行形態 ==
除了在線內往返運行之外，也有在竹田車站直通運行至[[近鐵京都線|近鐵京都線]]的[[直通運行|相互直通運行]]列車。2012年3月20日修改時刻表之後，在白天每小時運行8班列車（7分30秒間隔。線内往返5班列車、新田邊直通普通2班列車、奈良直通急行1班列車）。無區間運行列車，所有列車均途徑國際會館 - 竹田區間運行。

=== 線内往返 ===
在白天每小時運行五班線內往返的列車。車站的行駛方向介紹和[[方向幕|方向幕]]均只表示「國際會館」或「竹田」。除了京都市交通局的列車之外，也有近鐵的列車運行。其中3200系不顯示列車種別只顯示行駛行駛方向，3220系則不顯示種別表示（黑色）。

=== 新田邊直通普通 ===
在白天每小時運行兩班（30分間隔）自[[近鐵京都線|近鐵京都線]][[新田邊車站|新田邊車站]]發車的列車。方向幕顯示為「{{Colors|white|#0033ff| 普通 }} 新田辺」「{{Colors|white|#0033ff| 普通 }} 国際会館」。

早晨則僅運行有一班自[[近鐵宮津車站|近鐵宮津]]發車開往國際會館的列車（並無開往近鐵宮津的列車）。另外，急行列車并不在距[[同志社大學|同志社大學]]京田邊校區較近的[[興戶車站|興戶車站]]停車。因烏丸線直通普通列車在新田邊站往返，近鐵普通列車的運行間隔為7分半 - 22分半。

=== 近鐵奈良直通急行 ===
在2000年3月15日時刻表修改之後登場。自[[近鐵奈良車站|近鐵奈良車站]]發車，為急行列車（在烏丸線內為各站停車），白天每小時有一班列車（60分間隔），早晨則只有國際會館發車的兩班列車（禮拜六、休息日為一班列車）運行。方向幕顯示為「{{Colors|white|#ff8000| 急行 }} 奈良」「{{Colors|white|#ff8000| 急行 }} <small>[京都]</small>国際会館」。在列車到達竹田車站后進行線內返程運行時顯示變更為後者。另外，2012年3月20日時刻表修改之前，白天每小時運行兩班列車（30分間隔）。

== 歷史 ==
* 1974年（昭和49年）：烏丸線開工（着工）。開業試運行時期，可憑在這一年發售的「記念章及車票」乘車。
* 1981年（昭和56年）5月29日：北大路車站 - 京都車站區間開業。10系電車開始投入營業運行。
* 1988年（昭和63年）6月11日：京都車站 - 竹田車站區間開業。
* 1988年（昭和63年）8月28日：開始直通運行至近鐵京都線新田邊車站。近鐵3200系電車開始營業運行。
* 1990年（平成2年）10月24日：北山車站 - 北大路車站區間開業。
* 1997年（平成9年）5月22日：[[京都市營地下鐵東西線|東西線]]開業之前，御池車站改稱為烏丸御池車站。
* 1997年（平成9年）6月3日：國際會館車站 - 北山車站區間開業，全線開通（自正午開始營業）。
* 2000年（平成12年）3月15日：開始進行國際会館車站 - 近鐵奈良車站曲間的直通急行運行。近鐵3220系電車開始營業運行。
* 2011年（平成23年）3月25日：自這天起更新接近廣播和接近旋律，并導入入線旋律。<ref>[http://www.city.kyoto.lg.jp/kotsu/page/0000096642.html 地下鉄烏丸線におけるホームの安全対策の実施について（京都市情報館 交通局より）] {{webarchive|url=https://web.archive.org/web/20120120023641/http://www.city.kyoto.lg.jp/kotsu/page/0000096642.html |date=2012-01-20 }}（2011年9月9日閲覧）</ref>。

== 延伸計劃 ==
[[油小路通|油小路通]]被指定為高度集積地區，政府對當地的活性化頗具期待，也有將烏丸線自竹田車站和近鐵京都線分岐，延伸至横大路附近（全長4.4km）的計劃<ref name="tosin2004">{{PDFlink|[http://wwwtb.mlit.go.jp/kinki/shingi/toshin/bessi.pdf 平成16年近畿地方交通審議会答申第8号「京阪神圏において、中長期的に望まれる鉄道ネットワークを構成する新たな路線」]}} - 国土交通省近畿運輸局</ref>。計劃中的線路將建設在大手筋的地下，到達[[京阪本線|京阪本線]]的[[中書島車站|中書島車站]]和[[淀車站|淀車站]]之間的附近地區<ref name="tosin2004" /><ref>{{PDFlink|[http://wwwtb.mlit.go.jp/kinki/shingi/pdf/8-5-4.pdf 平成16年近畿地方交通審議会答申第8号「京阪神圏において、中長期的に望まれる鉄道ネットワークを構成する新たな路線」（付図2）]}} - 国土交通省近畿運輸局</ref>。近畿地方交通審議會（[[國土交通省|國土交通省]]的諮問機關）在2004年10月8日，作為中長期希望建設的鐵道網，報告了延伸至竹田車站以南的計劃<ref name="tosin2004" />。

並且也有越過[[淀川|宇治川]]南進，延伸至計劃在宇治川南部建設的[[洛南新都市|洛南新都市]]<ref>{{PDFlink|[http://www.mlit.go.jp/tetudo/toshitetu/pdf/03_10_08.pdf 平成元年運輸政策審議会答申第10号 答申図]}} - 国土交通省鉄道局</ref>的構想，但近畿地方交通審議會的報告中並未提及。

在北端也有延伸至岩倉地區的構想，但近畿地方交通審議會的報告中也未提及這個計劃。

和近鐵的連絡車票只能在京都線的竹田以南的各車站及奈良線、[[近鐵橿原線|橿原線]]的部份車站購買。

烏丸線的一些車站隔[[鴨川_(淀川水系)|鴨川]]和[[京阪本線|京阪本線]]相鄰，兩者間雖然有約1站左右的距離，卻和京阪的車站名稱相同。有不少觀光客和不習慣的乘客均稱非常混亂。京阪在[[京阪中之島線|中之島線]]開業時變更了車站名稱，混亂情況有所改善。 
* 松崎車站 - [[近鐵山田線|近鐵山田線]]也有[[松崎車站_(三重縣)|松崎車站]]。
* 丸太町車站 - 和京阪[[神宮丸太町車站|神宮丸太町車站]]的旧站名同名。
* 四條車站 - 和京阪[[祇園四條車站|祇園四條車站]]的旧站名同名。
* 五條車站 - 和京阪[[清水五條車站|清水五條車站]]的旧站名同名。JR[[和歌山線|和歌山線]]也有[[五條車站_(奈良縣)|五條車站]]。
* 九條車站 - 直通運行的[[近鐵橿原線|近鐵橿原線]]有[[九條車站_(奈良縣)|九條車站]]。並且近鐵線路的直通運行線路[[阪神難波線|阪神難波線]]和[[大阪市營地下鐵中央線|大阪市營地下鐵中央線]]也有[[九條車站_(大阪府)|九條車站]]。另外，在近鐵九條車站可購買前往另外兩個九條車站的車票，票價也幾乎一樣。
* 十條車站 - 和近鐵[[十條車站_(近鐵)|十條車站]]近隣。JR[[埼京線|埼京線]]也有[[十條車站_(東京都)|十條車站]]。

== 歷史遺跡的發現 ==
在烏丸線建設之前曾進行遺跡的發掘調查，并發現了眾多遺跡。一些遺跡在烏丸線開通時曾在御池車站（現在的烏丸御池車站）的畫廊展示。其中最大的是旧二條城遺構。
