《'''烏克蘭悲歌'''》（[[英語|英語]]：{{lang|en|''Bitter Harvest''}}）是一部由{{tsl|en|Ukrainian Canadians|烏克蘭裔加拿大人|烏克蘭裔加拿大}}導演{{tsl|en|George Mendeluk|喬治·門德路克}}執導的[[2017年電影|2017年]][[劇情片|劇情片]]，背景設置於[[1930年代|1930年代]]初的[[蘇維埃烏克蘭|蘇維埃烏克蘭]]，當時正値[[史達林|史達林]]推行[[農業集體化|農業集體化]]而造成[[烏克蘭大饑荒|烏克蘭大饑荒]]，以至數百萬[[烏克蘭人|烏克蘭人]]死於這一事件。此片的劇本由理查·巴秦斯基·霍華（Richard Bachynsky Hoover）編寫，而最終拍攝階段的劇本是由巴秦斯基和導演喬治共同完成。電影由伊安·伊納多維奇（Ian Ihnatowycz）製片，理查·巴秦斯基·霍華擔任[[執行製作人|執行製作人]]。[[英國人|英國]]演員[[麥斯·艾恩斯|麥斯·艾恩斯]]和[[薩曼莎·巴克斯|薩曼莎·巴克斯]]出演主角。

== 製作 ==
此片的編劇理查·巴秦斯基·霍華也是烏克蘭裔加拿大人，他於1999年遊歷[[烏克蘭|烏克蘭]]時萌生了拍攝這樣一部電影的想法，<ref name="HfRFE427">{{cite web |url=http://www.rferl.org/a/ukraine-holodomor-film-famine-love-story-soviet-union-stalin/28278871.html |title=A Love Story Set Amid The Holodomor, Ukraine's 20th-Century Famine, Hits The Big Screen |publisher=[[自由歐洲電台／自由電台|Radio Free Europe]] |date=2017-02-04 |accessdate=2017-09-14}}</ref>他之後一直圍繞這個題材進行研究。他於2004年烏克蘭爆發[[橘色革命|橘色革命]]期間造訪[[基輔|基輔]]，在那裡了解到[[烏克蘭大饑荒|烏克蘭大饑荒]]還從未被改編成電影。<ref name="HfRFE427"/>2008年，巴秦斯基開始向烏克蘭政府以及烏克蘭寡頭政治首領為電影尋求拍攝資金，但無人對此感興趣。<ref name="HfRFE427"/>2011年，巴秦斯基接觸到一位同是烏克蘭裔加拿大人的投資者伊安·伊納多維奇，後者為影片開出2100萬美元的拍攝資金。<ref name="HfRFE427"/>

這部電影最初命名為「惡魔的豐收」（{{lang|en|''The Devil's Harvest''}}），<ref name=Finishes>{{cite news|last=Barraclough|first=Leo|title=''White Queen'' Star Max Irons Finishes Ukraine Shoot for ''Devil's Harvest''|url=http://variety.com/2014/film/news/white-queen-star-max-irons-finishes-ukraine-shoot-for-devils-harvest-1201087866/|accessdate=2017-09-14|newspaper=[[Variety_(magazine)|Variety]]|date=2014-02-05}}</ref><ref>{{cite web |url=http://collider.com/the-devils-harvest-images-max-irons/ |title=First-Look Images from THE DEVIL'S HARVEST Starring Terence Stamp, Max Irons, and Barry Pepper |first=Dave |last=Trumbore |publisher={{link-en|Collider.com}} |date=2014-02-04 |accessdate=2017-09-14}}</ref>在[[烏克蘭|烏克蘭]]取景攝製。製片人伊安·伊納多維奇希望能藉此片撥開籠罩在這段歷史上的迷霧，他說：「鑑於[[烏克蘭大饑荒|大饑荒]]的重要性，但在烏克蘭以外很少有人知道這場並非天災，而是人禍引起的饑荒，因為很多史實都被克里姆林宮政權所掩蓋。這段歷史應以英文首次搬上大銀幕。」<ref name="Finishes" /><ref name="AC151014">{{cite news |url= http://www.atlanticcouncil.org/blogs/new-atlanticist/new-movie-reveals-russia-s-attempts-to-destroy-ukraine |title= New Movie Reveals Russia's Attempts to Destroy Ukraine |first= Diane |last= Francis |publisher= Atlantic Council |date= 2015-10-14 |deadurl= yes |archiveurl= https://web.archive.org/web/20151208152329/http://www.atlanticcouncil.org/blogs/new-atlanticist/new-movie-reveals-russia-s-attempts-to-destroy-ukraine |archivedate= 2015-12-08 }}</ref>

影片攝製於2013年11月15日在烏克蘭開機，<ref name=underway>{{cite web|last=Mitchell|first=Wendy|title=Max Irons, Samantha Barks go for Harvest|url=http://www.screendaily.com/news/production/max-irons-samantha-barks-go-for-harvest/5063716.article|accessdate=2014-02-08|publisher={{link-en|Screendaily.com}}|date=2013-11-15|archiveurl=https://web.archive.org/web/20131119112154/http://www.screendaily.com/news/production/max-irons-samantha-barks-go-for-harvest/5063716.article|archivedate=2013-11-19|deadurl=yes}}</ref>美國雜誌《[[綜藝_(雜誌)|綜藝]]》於2014年2月5日報導說電影已在基輔拍攝完畢。<ref name=Finishes/>數名當地的工作人員還參加了[[烏克蘭親歐盟示威運動|烏克蘭親歐盟示威運動]]。<ref name="HfRFE427"/>
