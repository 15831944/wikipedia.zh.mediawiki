'''烏拉圭超级盃'''（'''Supercopa Uruguaya'''），是[[烏拉圭|烏拉圭]]的一項足球賽事，由上一季的[[烏拉圭足球甲級聯賽|烏拉圭足球甲級聯賽]]冠軍與季中錦標賽（Torneo Intermedio）冠軍之間角逐。賽事會以一場過形式進行，於每年一月下旬或二月上旬舉行，即新一季聯賽展開前的一個星期。<ref>{{cite web |url=http://www.tenfield.com.uy/reforma-torneo-intermedio-en-dos-grupos-entre-apertura-y-clausura-y-crearan-la-recopa-uruguaya/ |title=REFORMA: “TORNEO INTERMEDIO” ENTRE EL APERTURA Y EL CLAUSURA QUE SUMARÁ PARA LA TABLA ANUAL; EL CAMPEÓN JUGARÁ LA SUPERCOPA |publisher=Tenfield |date=2015年8月12日}}</ref>

賽事會於中立場的[[世紀球場|世紀球場]]舉行。假如有球隊同時奪得兩項賽事錦標，則會由季中錦標賽亞軍補上出賽。<ref>{{cite web |url=http://www.tenfield.com.uy/aprobado-a-partir-de-2017-el-uruguayo-ira-desde-enero-a-diciembre-crearon-torneo-intermedio-y-la-supercopa/ |title=Aprobado: A partir de 2017 el Uruguayo irá desde enero a diciembre; crearon Torneo Intermedio y la Supercopa |publisher=Tenfield |date=2015年8月28日}}</ref>
