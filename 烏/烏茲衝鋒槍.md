'''烏茲衝鋒槍'''（{{lang-en|Uzi submachine gun}}或{{lang|en|Uzi}}，{{lang-he|עוזי}}），也叫烏奇衝鋒槍（Uzi 的常見中文稱呼源自於不同地區的民間音譯結果）<ref>{{Cite web|url=http://www.hnqwq.com/aspcms/news/2012-8-9/184.html|title=巴拉贝鲁姆弹-湖南兵器轻武器研究所有限责任公司--官方网站（将 UZI 称作“乌奇”）|accessdate=2018-08-11|author=|date=|work=www.hnqwq.com|publisher=}}</ref>。烏茲是[[以色列軍事工業|以色列軍事工業]]的一种輕型[[衝鋒槍|衝鋒槍]]，由[[烏茲·蓋爾|烏茲·蓋爾]]（Uziel Gal）於1948年設計。

== 歷史 ==
烏茲衝鋒槍由[[以色列國防軍|以色列國防軍]][[上尉|上尉]]（後升至[[少校|少校]]）[[烏茲·蓋爾|烏茲·蓋爾]]（{{lang|en|Uziel Gal}}）於[[第一次中東戰爭|第一次中東戰爭]]後的1948年開始設計，1951年生產，在1956年[[第二次中東戰爭|第二次中東戰爭]]中服役並取得令人滿意的效果，其後開始[[大量生產|量產]]。

當時的烏茲衝鋒槍是[[軍官|軍官]]、[[車輛|車組]]成員及[[炮兵|炮兵]]部隊的[[個人防衛武器|自衛武器]]，亦是精英部隊的前線武器，[[第三次中东战争|六日戰爭]]時的以色列士兵認為烏茲衝鋒槍的緊湊外型及火力十分適合清除[[叙利亚|叙利亚]]及[[约旦|约旦]]士兵所建的[[碉堡|碉堡]]。[[雷根遇刺案|雷根遇刺案]]發生時[[美國特勤局|美國特勤局]]特工Robert Wanko拔出一把烏茲，使烏茲的名聲大振。

烏茲是一款有著簡單的結構及易於生產的衝鋒槍，該武器以其設計者烏茲·蓋爾的名稱來命名，雖然當時他表明並不希望以其名字作武器名稱，但這請求被駁回。

== 採用 ==
烏茲衝鋒槍目前為全世界廣泛地使用，輕便、操作簡易及低成本令烏茲衝鋒槍成為一種十分有效的近戰武器，尤其是用於清除室內、[[碉堡|碉堡]]及[[堑壕战|戰壕]]裡有生目標，亦是常見在機械化部隊的自衛武器。

直至今天，以色列的常規部隊已將烏茲衝鋒槍除役，但[[特種部隊|特種部隊]]仍然在任務時採用作近戰武器之一，而[[以色列軍事工業|以色列軍事工業]]（IMI）仍有生產部件作出口用途。

烏茲衝鋒槍在部分國家及一些軍火商獲授權生產或私自仿製及使用，[[Fabrique_Nationale|FN]]、[[德國|德國]]及[[辛巴威|羅德西亞]]（名為「Rhuzi」）也同時取得合法生產，有些國家更為其產品重新命名（但槍枝本身沒有改變的），但這些獲授權生產或私自仿製品大都遠遠地比不上其原型。

除以色列外，美國一些執法部門、特警隊、私人保安部隊及[[美軍|美軍]]以至[[美國特勤局|美國特勤局]]（已被[[FN_P90衝鋒槍|P90]]及[[HK_MP5衝鋒槍|MP5]]取代）都裝備烏茲，{{link-en|Zim船務公司| Zim Integrated Shipping Services}}的[[傭兵|傭兵]]也有裝備。

除標準型烏茲衝鋒槍外，烏茲亦有多種不同長度及半自動的衍生型，這些衍生型現在仍是多個[[特種部隊|特種部隊]]及執法部門的常用武器之一<ref>{{cite web |url=http://www.priu.gov.lk/news_update/features/20031017naval_surveillance_millstone_LTTE.htm |title=存档副本 |accessdate=2003-10-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20031120105840/http://www.priu.gov.lk/news_update/features/20031017naval_surveillance_millstone_LTTE.htm |archivedate=2003-11-20 }}</ref>。

[[德國聯邦國防軍|德國聯邦國防軍]]在1959年起裝備合法生產型的烏茲，命名為MP2，直至現在才開始被[[HK_MP7衝鋒槍|MP7]]取代。[[斯里蘭卡|斯<nowiki>里</nowiki>蘭卡]]在1990年代入口了數百把迷你烏茲及烏茲卡賓槍，現在仍是斯<nowiki>里</nowiki>蘭卡陸軍特種部隊及[[特警隊|特警]]的武器。

== 設計 ==
[[File:Uzi_silencer.jpg|250px]]的烏茲衝鋒槍]]

烏茲衝鋒槍採用來自[[捷克|捷克]][[CZ_25冲锋枪|CZ 23至CZ 26衝鋒槍]]的[[開放式槍機|開放式槍機]]、[[後座作用|後座作用]]設計<ref name="Modern Firearms">[http://world.guns.ru/smg/smg17-e.htm Modern Firearms-UZI / Mini UZI / Micro UZI submachine gun (Israel)]</ref>，而CZ衝鋒槍系列是第一種採用[[包絡式槍機|包絡式槍機]]的衝鋒槍，這種設計把彈匣位置改為在握把內，部份槍管會被[[機匣|機匣]]覆蓋，令總長度大幅下降，重量分佈更加平衡。

機匣採用低成本的金屬沖壓方式生產，以減小生產成本及所需的金屬原料，亦縮短了生產所需的時間，而且更容易進行維護及維修，但對沙塵的相容性較低，當[[擊鎚|擊鎚]]釋放時，退殼口會同時關上以防止沙塵進入機匣造成故障。

烏茲衝鋒槍採用握把式保險（位於握把背部，必需保持按壓才可發射），減低了走火機會，握把內藏彈匣的設計令射手在黑暗環境時仍可快速更換彈匣。

不過握把內藏彈匣的設計令槍械高度影響了伏姿射擊時所需的空間，在[[沙漠|沙漠]]或[[風沙|風沙]]較大的地區作戰時須經常分解清理以避免射擊時出現如卡彈等的情況發生，這種缺點在所有開放式槍機的[[槍械|槍械]]亦有存在。

==口徑==
烏茲衝鋒槍最常見的口徑為[[9×19mm鲁格弹|9×19毫米]]，民用市場中有些會被改為[[.22_LR|.22 LR]]、[[.41_Action_Express|.41 AE]]或[[.45_ACP|.45 ACP]]，甚至是[[.40_S&W|.40 S&W]]及[[10毫米Auto|10毫米Auto]]<ref>[http://files.uzitalk.com/reference/pages/caliberconversions.htm]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

9毫米口徑常見的彈匣容量為20、25、32、40及50發，甚至100發彈鼓，.22 LR及.41 AE則是10發，而.45 ACP為16發。<ref name="Modern Firearms"/>

==衍生型==
* '''迷你烏茲（Mini Uzi）'''－1980年推出，以標準型作縮短而成，按某些分類上剛好被歸入了[[衝鋒手槍|衝鋒手槍]]範圍的，[[9mm鲁格弹|9毫米]][[口徑|口徑]]，打開[[槍托|槍托]]後全長600毫米（23.62寸），槍托向右摺疊時只有360毫米（14.17寸），槍管長197毫米，初速為每秒375米，槍機開放時射速為每分950發（約每秒16發），槍機封閉時射速為每分1700發（約每秒28發）<ref name="uzi.com">[http://www.uzi.com/ 烏茲官方網頁]</ref>，裝25發彈匣時重3.15公斤，槍機開放及封閉時皆可發射<ref name="google">{{cite web |url= http://books.google.ru/books?id=z7nW8LpDOBoC&pg=PA55&lpg=PA55&dq=Uzi+manufactures+closed-bolt&source=web&ots=fYtGhC3sTN&sig=ewv6r1Cmnw_fYYv9fMm3CbQV6xg&hl=ru |title= Mini-Uzi and Micro-Uzi specifications |publisher= Twenty-First Century's Small Arms, MBI Publishing |accessdate= 2008-04-02 |deadurl= yes |archiveurl= https://web.archive.org/web/20150402120641/http://books.google.ru/books?id=z7nW8LpDOBoC&pg=PA55&lpg=PA55&dq=Uzi+manufactures+closed-bolt&source=web&ots=fYtGhC3sTN&sig=ewv6r1Cmnw_fYYv9fMm3CbQV6xg&hl=ru |archivedate= 2015-04-02 }}</ref>，具半自動或[[自動火器|全自動射擊]]模式。
* '''微型烏茲（Micro Uzi）'''－1990年代推出，比迷你烏茲進一步縮小短，類似較大型的[[衝鋒手槍|衝鋒手槍]]，同樣9毫米口徑，打開槍托後全長460毫米（18.11寸），槍托向右摺疊時全長只有250毫米（9.84寸），槍管長117毫米，初速為每秒350米，射速為每分1250發（約每秒21發）<ref name="Modern Firearms"/>或每分1700發（約每秒28發）<ref name="uzi.com"/>，裝25發彈匣時重2.2公斤，封閉式槍機<ref name="google"/>，同樣具半自動或全自動射擊模式，[[扳機護環|扳機護環]]及握把的外型亦有所不同。微型烏茲設計上亦可作為[[個人防衛武器|個人防衛武器]]（PDW）使用。
* '''傘兵微型烏茲（Para Micro Uzi）'''－為以色列反恐部隊如Shabak及[[YAMAM|YAMAM]]等特別設計，為9毫米口徑，[[機匣|機匣]]頂部及底部加裝[[戰術導軌|戰術導軌]]，改為傾斜式握把以對應[[格洛克18|格洛克18]][[全自動手槍|全自動手槍]]的33發彈匣。[[扳機護環|扳機護環]]及握把的外型亦有所不同。
* '''烏茲手槍（Uzi Pistol）'''1984年推出，－微型烏茲的[[半自動手槍|半自動手槍]]版本，為9毫米口徑，初速為每秒345米，主要出口至各地民用市場，設有[[砲口制動器|槍口制退器]]。
* '''烏茲卡賓槍（Uzi Carbine）'''－標準型烏茲衝鋒槍的半自動版本，為9毫米口徑，槍管長400毫米（16寸），主要出口至各地民用市場。
* '''迷你烏茲卡賓槍（Uzi Mini Carbine）'''－迷你烏茲衝鋒槍的半自動版本，同樣為9毫米口徑，槍管長450毫米（18寸），主要出口至各地民用市場。
*'''Uzi PRO'''是最新版微型迷你衝鋒槍。
