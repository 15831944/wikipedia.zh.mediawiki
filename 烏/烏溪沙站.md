'''烏溪沙站'''（{{lang-en|'''Wu Kai Sha Station'''}}），是[[港鐵|港鐵]][[馬鞍山綫|馬鞍山綫]]的上行終點站，位於[[烏溪沙|烏溪沙]][[西沙路|西沙路]]旁側，是該線其中一個架空車站，取名自鄰近的[[烏溪沙|烏溪沙]]和[[烏溪沙村|烏溪沙村]]，於2004年12月21日啟用。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Wu Kai Sha Station 2017 11 part6.jpg | caption1 = 大堂

}}
大堂入付費區設有洗手間供乘客使用。車站出入口設有客務中心、增值機和售票機，另外亦有為盲人指導的錄音提示器。此外，出入口亦設有特闊閘機，方便有需要人士出入閘。

因站內原先預計人流不多，烏溪沙站只有少量商店，包括[[便利商店|便利店]]、[[銀行|銀行]]、[[唐記包點|唐記包點]]、勝豐壽司及兩間地產代理，站內亦設有兩部[[自動櫃員機|自動櫃員機]]及4個報紙櫃供乘客於早上免費取閱。<ref name="shops"/>。

==鄰接車站==
烏溪沙站是馬鞍山綫的下行起點，乘客可在此乘坐往[[大圍站|大圍]]列車，往返新界東北或九龍各區。

==歷史==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Wu Kai Sha Station.jpg | caption1 = 兩鐵合併前的月台（2005年）
}}
馬鞍山綫通車前曾命名為'''利安站'''（{{lang-en|'''Lee On Station'''}}），因為車站位於利安邨的東面<ref>{{cite web |url=http://www.districtcouncils.gov.hk/archive/st_d/chinese/doc/tt_minutes_7_00.doc|title=交通及運輸委員會第六次會議|publisher=沙田區議會|date=2000年11月14日|accessdate=2014年4月30日|language=zh-hant}}</ref>。

==未來發展==
當沙中綫東西走廊工程完成後，新建路段將會與西鐵綫和馬鞍山綫合併，並易名為屯馬綫，烏溪沙站將成為屯馬綫的總站，日後在本站前往大圍至九龍城以至紅磡以西或尖沙咀、西九龍、深水埗、荃灣、元朗、天水圍及屯門一帶，將毋須轉乘其他路綫就可以直接到達。
