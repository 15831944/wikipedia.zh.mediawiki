'''烏日站'''位於[[臺中市|臺中市]][[烏日區|烏日區]]，為[[臺中捷運|臺中捷運]][[臺中捷運綠線|綠線]]（[[烏日文心北屯線|烏日文心北屯線]]）之[[捷運車站|捷運車站]]<ref>[http://news.ltn.com.tw/news/local/paper/291933 新增3站…臺中捷運綠線確定設18站]，自由時報，2009-04-01</ref>，與[[臺中都會區鐵路高架捷運化計畫|臺鐵捷運化]]後的通勤鐵路車站（[[烏日車站|烏日車站]]）將為[[站外轉乘|站外轉乘]]之捷運車站。

== 車站概要 ==
[[File:臺中捷運烏日站2015年2月工程.jpg|thumb]]
*車站位於[[建國路_(臺中市)|建國路]]與[[光日路|光日路]]口附近（建國路上）；車站代碼為'''G16'''<ref>[http://www.dorts.gov.taipei/ct.asp?xItem=660998&CtNode=20285&mp=115001 臺北市政府捷運工程局 >捷運興建中路網 >臺中捷運烏日文心北屯線 >路線說明]，臺北市政府捷運工程局</ref>。
*本站與[[烏日車站|臺鐵烏日車站]]站外轉乘。

== 車站構造 ==
高架車站，兩座[[側式月台|側式月台]]，兩個出入口。
