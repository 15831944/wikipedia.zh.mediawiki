'''烏克蘭獨立新聞社'''，簡稱'''UNIAN''' （'''Ukrainian Independent Information Agency'''、{{lang-uk|Українське Незалежне Інформаційне Агентство, УНІАН}}; ''Ukrayins'ke Nezalezhne Informatsiyne Ahentstvo'') 是烏克蘭最大的新聞通訊社，總部位於[[基輔|基輔]]。它以烏克蘭文、俄文、英文產出並提供政治、商業和財務信息、新聞照片報導服務；擁有自己的電視及廣播頻道。


==歷史==
UNIAN通訊社成立於1993年3月，其新聞被主流媒體包括報紙、電視頻道、廣播電台作為主要信息來源之一。UNIAN擁有全球約500家媒體客戶，圖片庫有逾20萬張新聞圖片。UNIAN還主辦了全國最熱門的新聞發布會廳。

該通訊社擁有自己承繼自蘇聯時代的建物，位於首都[[基輔]的]赫雷夏蒂克街。
== UNIAN 電視==
UNIAN有自己的電視頻道，UNIAN TV。內容包括新聞、分析節目、紀錄片、體育和電影。<ref>{{Cite web|url = http://media.1plus1.ua/en/group/broadcasting/unian|title = <nowiki>UNIAN TV | 1+1 Media</nowiki>|date = |accessdate = |website = |publisher = 1+1 Media Group|last = |first = }}</ref>

===廣播===
該頻道可透過衛星，有線(Cable)和IPTV網絡收聽。
