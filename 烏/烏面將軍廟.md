'''烏面將軍廟'''，是位於[[臺灣|臺灣]][[彰化縣|彰化縣]][[秀水鄉_(台灣)|秀水鄉]][[陝西_(臺灣)|陝西村]]的[[陰廟|陰廟]]，有說法是主祭[[鄭成功|鄭成功]][[陝西|陝西]]籍的部下[[馬信_(東寧)|馬信]]，並傳說當地人為[[明鄭|明鄭]]陝西軍人的後裔，而引起[[臺灣戰後時期|臺灣戰後時期]]陝西籍移民的注意。

==歷史沿革==
===建廟傳說===
{{Multiple image
|align=right
|direction = horizontal
|width  = 150
|image1 =烏面將軍廟斜照.jpg|caption1 =廟身
}}
此廟由來傳說是陝西村{{efn|學者[[戴寶村|戴寶村]]、黃雅慧於《[[台灣風物|台灣風物]]》雜誌認為秀水一帶居民多係福建漳、泉移民，因[[漳泉械鬥|漳泉械鬥]]，部份民眾閃躲遷往西邊，稱地名為「閃西」，可能官員認為不雅而以同音的「陝西」名之<ref name="賴淑姬"/>。}}先民開荒墾地時發現一堆骨骸，其中夾雜著一小方磚石，上書「烏面將軍」四字，其中一付骨骸特別大、顏色也比較沉，便加以整理供奉，尊稱名曰「烏面將軍」<ref name="潘嘉珠"/>。但其神像面孔並非黑面<ref name="潘嘉珠"/><ref name="1978-04-27"/>，耆宿吳永宗解釋該稱號是傳說該神姓「烏」名「面」<ref name="1978-04-27"/>。廟原為[[百姓公廟|百姓公廟]]改建，另說「烏面將軍」即「鬼將軍」之意<ref name="賴淑姬"/>。樓梯口還設有一座石烏龜當辟邪物<ref name="黃志亮"/>。廟址與[[花壇鄉|花壇]][[戰備跑道|戰備跑道]]近<ref name="林宜慈、江良誠"/><ref name="王百鍊、劉曉欣、湯世名"/>。村民病痛或對上蒼有所祈求時，便前往禱祝，還願者促使廟身翻修又擴大<ref name="潘嘉珠"/>。

===引起注意===
{{Multiple image
|align=right
|direction = horizontal
|width  = 150
|image1 =彰化縣秀水鄉陝西村刻有「陝西」的墓碑.jpg|caption1 =刻有「陝西」的墓碑。
}}
1976年9月3日，彰化縣[[田中鎮|田中鎮]]田中國小的全縣國小校長會議，陝西省籍的台灣省教育廳督學徐秉琰發現學校手冊有陝西國民小學，台灣各地命名「長安」的學校有許多，而「陝西」是他第一次看到，因好奇便立即向縣教育局人員打聽該國小在何處。1977年6月3日他視導花壇國中時，順便請人帶路前往陝西國小。當日，他見到國小旁邊還有烏面將軍廟，然後看到當地村-{庄}-北邊墓地墓碑上刻有「陝西」{{efn|後來學者調查該些墓碑都是1951年以後的新墓，且用省名也不合臺灣人習慣<ref name="賴淑姬"/>。}}，及查覺村民下顎骨都較寬{{efn|下顎骨較寬能否證明即陝西人後裔，在[[體質人類學|體質人類學]]缺乏根據<ref name="賴淑姬"/>。}}，就斷定烏面將軍與這村民祖先都來自陝西省，很高興對村民說他們都是陝西省人。<ref name="顏文閂、陳錫龍"/>

返回台中後的徐秉琰，即向同鄉報告此消息，約好四、五人於該年7月初再到陝西村訪問<ref name="顏文閂、陳錫龍"/>。對被徐秉琰說是陝西省人，該村長老吳太山感到驚訝，猜測或許是年代久遠、日本人以皇民化迫害，所以村內並沒有證明祖籍來自陝西的相關文物，但可能是日本人以為居民祖先不會來自陝西省，遂沒禁止在墓碑刻陝西，因而墓碑成了尋「根」的證據<ref name="1978-04-28"/>。同年農曆八月廿三，烏面將軍誕辰，一位陝西籍住在[[臺中市_(省轄市)|台中市]]的王姓醫生趕到廟宇，激動地講聞到故鄉土地的芳芬<ref name="1978-04-27"/>。徐秉琰在該日看到戲台後台用的[[胡琴|胡琴]]，認為不論外形和拉法，都和[[秦腔|秦腔]]用的沒有兩樣<ref name="顏文閂1"/>。

===神明說法===
{{Multiple image
|align=right
|direction = horizontal
|width  = 150
|image1 =烏面將軍廟門牌.jpg|caption1 =門牌
}}
對傳出烏面將軍乃鄭成功陝西籍部下，成功大學歷史系教授[[黃典權|黃典權]]表示可能確有烏姓的軍人，但不是主將，因而史料上查不到，但確實有一名陝西籍將領馬信，並帶陝西子弟兵來臺灣<ref name="考證烏面將軍"/>。至於以「烏面將軍」代替馬信的說法，一說避免滿清對馬信的仇恨，另說馬信面貌黝黑，遂有此稱號<ref name="潘嘉珠"/>。台北市文獻委員會執行秘書[[王國璠|王國璠]]也認為是怕清朝反感，於是稱此名<ref name="郭淑敏"/>。馬信於鄭成功死後五天亦亡故，從不曾到過彰化縣境<ref name="賴淑姬"/>。

陝西村前任村長張泉益回憶說他十幾歲時自父親聽聞，有一位將軍率軍戰死，感嘆若早一點知道祂是鄉親該多好<ref name="1978-05-01"/>。在烏面將軍被從認定是陝西村先人後，村民更加禮遇，在1977年特地做了一隻重達二百台斤的麵龜作祭品<ref name="1978-05-01"/>。徐秉琰認為烏面將軍的酬神戲，和陝西地方戲類似，說可見陝西習俗和戲劇{{efn|但戲班都是外地請來，並非居民自組<ref name="賴淑姬"/>。}}也在陝西村流傳下來<ref name="1978-04-27"/>。

===尋根熱潮===
{{Multiple image
|align=right
|direction = horizontal
|width  = 150
|image1 =烏面將軍廟與陝西文物館.jpg|caption1 =烏面將軍廟與陝西文物館
}}
1978年4月26日，陝西國小校長蘇鴻猷、陝西村耆宿吳永宗、秀水鄉老鄉長張進登、秀水國小校長張江，北上到[[台北市|台北市]][[仁愛路_(臺北市)|仁愛路]]與[[敦化南路|敦化南路]]交叉口的圓環，向陝西籍的[[于右任|于右任]]銅像行禮，獲得《聯合報》記者[[顏文閂|顏文閂]]、陳錫龍的報導<ref name="顏文閂、陳錫龍"/>。這四位也受到陝西籍立法委員[[楊大乾|楊大乾]]及其他陜西人接待，到台北市的國軍文藝活動中欣賞秦腔{{efn|上演的是陝西梆子的《斷橋》，《聯合報》寫說這四位客人聽不懂，但重要的它是「鄉音」<ref name="1978-04-30"/>。至於這四人的感想是發現自己的「根」真好，希望趕快[[光復大陸|光復大陸]]<ref name="1978-04-27"/>。}}，由馬靜亭表演<ref name="1978-04-27"/>。新聞在次日於《聯合報》見報後，陝西村民在當天碰面均以此為話題，津津樂道，並到烏面將軍廟參加討論<ref name="1978-04-28"/>。

次月15日，陝西同鄉會理事長陳建中和總幹事王保民等九人，在徐秉琰帶路下，自台北到陝西村參觀{{efn|陳建中一行人參觀時認為村中許多四合院住宅的建築均與陝西省的建築型式相同<ref name="1978-05-16"/>。徐秉琰曾說，經過他細心查究，發現村民的生活起居仍依稀承襲中原的開朗習性，住屋建築格局更隱含著黃土高原的風範<ref name="潘嘉珠"/>。後來徐秉琰又說在彰化沿海地區發現永安村和長安村的居民，分別來自河北、河南、陝西、甘肅、山東等省，《[[黑白集|黑白集]]》因此撰文讚揚徐秉琰有助於中國人的團結<ref name="1979-11-06"/>。}}，歡迎的隊伍足有半公里長，有人還因此感動落淚。隨後，陳建中等人在烏面將軍廟內列隊，向烏面將軍行三鞠躬禮，再請陝西村內年齡最高，時年八十歲的長者吳太山上坐。<ref name="1978-05-16"/>

[[華視|華視]]也於次日節目《[[中華電視公司節目列表|地平線]]》報導陝西村的風土人情<ref name="1978-05-16"/>。該年[[臺灣光復節|光復節]]特別節目，[[中視|中視]]《[[愛心_(公益節目)|愛心]]》在下午一點{{efn|同時段的[[華視|華視]]，則放電影《吳鳳》，描寫[[吳鳳|吳鳳]]如何感化原住民<ref name="1978-10-25"/>。}}以片名「根」來介紹陝西村的故事<ref name="1978-10-25"/><ref name="1978-10-22"/>，製作人周志敏還特率領製作小組拍攝[[五福臨門神木|五福臨門神木]]作為片頭<ref name="1978-10-22"/>。

同年11月14日，中國地方文獻學會上，陝西同鄉會決定在烏面將軍廟旁邊，籌設陝西文物館<ref name="1978-11-15"/>。他們除以文化中心來使陝西村成為陝西人聯誼之處，也捐錢作為陝西村學童的獎學金之用<ref name="顏文閂2"/>。

在1978年，來陝西村的陝西省籍人士就有五千多人次<ref name="顏文閂2"/>。陝西村長林欽回憶，政府未開放大陸探親之前，陝西村是陝西省籍旅台人士的「根」{{efn|譬如投效[[中華民國|中華民國]]的[[孫天勤|孫天勤]]從同鄉呂協恭聽說該村民是隨馬信來臺的陝西子弟後裔，也來此村參觀<ref name="黃玉峰"/>；中國攝影學會會員馮廷珩說騎單車跑六遍全台灣，卻從未遇見陝西村，這回找到了這個有一千多老鄉的地方，以後可以常來，也在此村拍攝許多照片<ref name="高惠宇"/>。}}，每年烏面將軍誕辰，來自全台灣各地的陝西省同鄉會都到陝西村聚會{{efn|像是湖南的同鄉會則曾去[[昭忠廟|昭忠廟]]祭祀與聚會<ref name="台中訊"/>。}}，人數可達七、八千人次<ref name="簡國書"/>。

1979年10月4日，烏面將軍誕辰，由陝西籍人士樂捐、耗資新台幣一百二十萬元、佔地三百坪的陝西文物館與靈骨塔破土，參與陝西籍同鄉多達二千人，由陝西耆宿馬雲河、高世榮、王孔祥主持<ref name="李湘聲"/>。土地為村長林月潭和張進登所捐出<ref name="陳志成"/>。1981年5月31日，烏面將軍廟後的陜西文物館落成，一樓供奉神像面孔為黑面的烏面將軍，二樓文物館，三樓規畫為圖書館<ref name="潘嘉珠"/>。至於靈骨塔除供陝西籍人百年後安放外，陝西村民必耍時也可利用<ref name="李湘聲"/>。當時會長陳建中、上將[[劉玉章|劉玉章]]之外，副總統[[謝東閔_(台灣)|謝東閔]]、內政部長[[邱創煥|邱創煥]]親臨剪綵，力言臺灣與大陸血脈相連，勉勵眾人以光復神州為職志<ref name="陳志成"/>。

自1981年起，中華民國的國小四年級上學期第七冊社會課本在名為「我們都從大陸來」單元中，編入陝西村的故事，稱陝西村是鄭成功部將的馬信陝西籍部下來臺後所開墾，為了紀念馬信，在村裡興建烏面將軍廟<ref name="賴淑姬"/>。

===兩岸交流===
{{Multiple image
|align=right
|direction = horizontal
|width  = 150
|image1 =烏面將軍廟匾額.jpg|caption1 =「光耀兩岸」匾額
}}
繼[[吳鳳|吳鳳]]在教材被刪除後，陝西村的故事因有爭議{{efn|台灣省文獻委員會及一些歷史學者，到陝西村進行田野調查，完全推翻徐秉琰的論斷。然而，他們當年並沒有出面懷疑，也不願替徐督學背書。曾任職文獻會的台灣史專家[[洪敏麟|洪敏麟]]及另一位時任職省府的楊姓官員都表示，當時政治氣氛，省文獻會不願也不敢做這種「殺風景」的事。<ref name="賴淑姬"/>}}，也被[[國立編譯館|國立編譯館]]於1990年決定刪除<ref name="陳碧華1"/>。兩岸開放交流後，[[中國大陸|中國大陸]]的陝西旅客依然會到陝西村參觀、並到烏面將軍廟膜拜<ref name="陳志成"/><ref name="董順隆"/><ref name="劉明岩"/>。時任陝西省代省長的[[赵正永|赵正永]]還贈送「光耀兩岸」的匾額{{efn|匾額為雷珍民所書，見照片。}}給烏面將軍廟，同時添五萬元香油錢<ref name="劉明岩"/>。[[親民黨|親民黨]]主席[[宋楚瑜|宋楚瑜]]至中國大陸作搭橋之旅時，也會談及陝西村<ref name="簡國書"/>。

2005年1月28日，《中國時報》記者陳志成來此廟訪問老人。當地人說臺灣的陝西省同鄉會只來此熱鬧幾年，之後很少來，理由是赴陝西省尋根、會親都很方便。對於村民是否相信自己是烏面將軍、是陝西省人的後代，則表示以往相信，現在將信將疑。至於陝西文物館，一到三樓全部空空洞洞。<ref name="陳志成"/>

2008年，秀水鄉公所爭取到[[中華民國內政部|內政部]]補助一千六百多萬元，對烏面將軍廟周圍環境、生態池等美化工程<ref name="林宛諭"/>。
