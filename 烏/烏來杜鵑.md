'''烏來杜鵑'''（[[学名|学名]]：{{lang|la|''Rhododendron kanehirae''}}）又名'''台北杜鵑'''、'''柳葉杜鵑'''、'''金平氏杜鵑'''<ref>{{cite web |url=http://www.taipei-expopark.tw/ct.asp?xItem=101450&ctNode=7838&mp=16 |title=花博公園：烏來杜鵑 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>以紀念[[金平亮三|金平亮三]]<ref name="龔招健">{{Cite news|url=|title=植物園 名人後裔緬懷先人事蹟 |language=zh-tw|publisher=《中國時報》|author=薛雲峰|date=2003-09-26|accessdate=}}</ref>，是[[杜鹃花科|杜鹃花科]][[杜鹃属|杜鹃属]]下的一个种，為台灣特有種、唯一一種岩生性杜鵑花科植物。

== 型態 ==
多年生灌木植物，灌叢高1至3公尺，生長於海拔200至800公尺的岩壁。葉片為互生、橢圓披針形，嫩葉覆有剛毛。花小型，色紫紅，單瓣，於莖條的芽冠頂端單生或簇生，花冠直徑約3至4公分，呈闊鐘型，花萼五裂，花期為每年三月下旬至四月上旬；結卵形蒴果。<ref>{{cite web |url=http://www.ymsnp.gov.tw/index.php?option=com_content&view=article&id=158:2011-07-20-05-49-21&catid=233:2011-07-14-05-51-26&Itemid=386&limitstart=8 |title=陽明山國家公園：烏來杜鵑}}</ref><ref>{{cite web |url=http://nrch.culture.tw/twpedia.aspx?id=6506 |title=台灣大百科：烏來杜鵑}}</ref>。

== 分布 ==
烏來杜鵑的野生棲地位於[[新北市|新北市]][[烏來區|烏來區]][[北勢溪|北勢溪]]上游，分布狹隘。[[民國73年|民國73年]]（1984年）[[翡翠水庫|翡翠水庫]]開始蓄水，將其唯一的原始棲地淹沒，[[民國77年|民國77年]]依[[文化資產保存法|文化資產保存法]]公告為法定珍貴稀有植物，野外已數十年無植株發現的紀錄。

== 保育 ==
烏來杜鵑分布區域狹促，因水庫蓄水淹沒棲地，為避免永久滅絕，由[[行政院農業委員會特有生物研究保育中心|行政院農業委員會特有生物研究保育中心]]與翡翠水庫管理局進行保育，於[[民國84年|民國84年]]在翡翠水庫大壩下游河床開設復育樣區，種植250株烏來杜鵑；但因多次豪雨沖刷，目前僅存30餘株；其後再以[[扦插|扦插]]繁殖，於水庫庫區、原鷺鷥潭址地種植，目前已有人工植株千餘棵。此外尚有植栽保存於台北市各級學校與公園內，如[[陽明山國家公園|陽明山國家公園]]、[[台灣大學|台灣大學]]校園內等；烏來、小坪林的地方人家亦保存有自行種植的烏來杜鵑盆栽<ref>{{cite web |url=http://www.feitsui.gov.tw/ct.asp?xItem=117358&CtNode=31452&mp=122011 |title=翡翠水庫管理局：烏來杜鵑在翡翠水庫之復育歷程}}</ref>。但廣植的人工復育樹苗係以扦插單性繁殖，基因多樣性恐有所不足。[[民國101年|民國101年]]由特生中心與[[嘉義大學|嘉義大學]]合作，開發出以種子培育的烏來杜鵑實生苗，將有助以兼顧基因多樣性的方式，進行烏來杜鵑的復育工作<ref>{{cite web |url=http://e-info.org.tw/node/75772 |title=環境資訊中心：真有種！ 原地絕跡28年 烏來杜鵑實生苗種出來了}}</ref>。
