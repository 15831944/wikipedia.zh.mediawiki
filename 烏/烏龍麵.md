'''烏龍麵'''（{{jpn|j=饂飩|hg=うどん|rm=udon}}），是一種日本的以[[小麥|小麥]]為原料製造的麵體，在粗細和長度方面有特別的規定。

== 起源 ==
# 根據香川縣的口頭傳說，從大唐中土歸還的[[空海|空海]]大師（[[弘法大師|弘法大師]]，於西元774─835）帶回烏龍麵的製法，拯救了[[讚岐|讚岐]]當地的貧民。因為[[瀨戶內海|瀨戶內海]]雨水稀少，很難種米而傳授給讚岐人，一般認為，這就是現在的讚岐烏龍麵的原型。
# 起源自[[奈良時代|奈良時代]]就已興盛的唐果子（即唐朝傳來的糕點）──索餅（さくべい、Sakubei）、餺飥（はくたく、Hakutaku，中文音念“博拖”）、餛飩（こんとん、Konton）。
#* 索餅：用麵粉及米粉揉製成細長交叉的油炸餅。陰曆7月7日用以在驅除[[瘧疾|瘧疾]]的儀式食用。索餅的系統，發展為現今的素麵、五島烏龍麵、稻庭烏龍麵。
#* 餺飥：用麵粉揉製成繩狀或是團狀切割的糕點。演變為今日的餺飥（ほうとう、Houtou）。
#* 餛飩：麵粉內包肉餡等，或煮或蒸的糕點。演變為今日的[[餃子|餃子]]或[[餛飩|餛飩]]。
# 另一種認為是因為[[伊勢|伊勢]]生產的米不足，但在很長時間裡這個被認為是烏龍麵的起源。

現今日本烏龍麵的形狀誕生於[[室町時代|室町時代]]（西元1336─1573）中期，[[元祿|元祿]]時代（西元1688─1703）也出現在[[江戶|江戶]]、[[京都|京都]]、[[大阪|大阪]]及[[琴平町|琴平]]的烏龍麵店。

== 規格 ==
[[File:Kakeudon.jpg|thumb]]
[[File:Yaki-udon.jpg|thumb]]
根據現行[[日本農業規格|日本農業規格]]<ref>日本農林水產省第1371號告示：《[http://www.maff.go.jp/j/jas/hyoji/pdf/kijun_25.pdf 乾麵類品質表示基準] {{webarchive|url=https://web.archive.org/web/20101008170214/http://www.maff.go.jp/j/jas/hyoji/pdf/kijun_25.pdf |date=2010-10-08 }}》，2007年11月6日最終修訂。</ref>，烏龍麵圓面直徑要1.7毫米以上，角面的寬度在1.7毫米以上；以下的則為「'''涼麵'''」（<span lang="ja">ひやむぎ</span>，寬度1.2－1.7毫米，厚度為1.0－2.0毫米），以此區分。除此以外，社會上通用的觀念裡，還有細面的「細烏龍麵」和「涼麵」的明確區別。

== 製作 ==
通常，用中筋性（相当于欧美标准的低筋性）小麦粉加入若干的鹽而製作，食用時多採用鱼干、海带以及醬油製作的[[湯底|湯底]]。

簡單來說作為平民主食、大米主食的代用食物，在喜慶時則作為「熱」食物，在古代的日本各處都非常受歡迎。以消費量來說[[四國|四國]][[香川縣|香川縣]]生產的[[讚岐|讚岐]]烏龍麵排第一，在乾烏龍麵的產地[[群馬縣|群馬縣]]排第二。烹調方法和配料強烈反映了地區差別，所以存在各種各樣的類型存在。
