'''钱江七桥'''，又名'''之江大桥'''，是[[杭州市|杭州市]]西南部的一座跨[[钱塘江|钱塘江]]公路桥梁，是[[杭新景高速公路|杭新景高速公路]]延伸线，西与杭新景高速公路及[[杭州绕城高速公路|绕城高速]]相连，东与滨江区[[彩虹大道|彩虹大道]]城市快速路相接。<ref name="通车">{{cite web|url=http://hznews.hangzhou.com.cn/chengshi/content/2012-05/10/content_4190477_2.htm | language= | title=钱江七桥、八桥预计今年通车，钱江隧道明年竣工 |publisher=杭州网 |date=2012-05-10 |accessdate= }}</ref>

== 建设 ==
2008年12月18开工，2012年8月底主桥合龙，于2013年1月18日通车。杭州市政府下属国企——杭州市交通投资集团为主要投资方，集团下属的之江大桥开发有限公司、杭新景高速公路延伸线(之江大桥)工程建设指挥部负责之江大桥的建设及管理；大桥共投资26亿元，其中65%是靠[[贷款|贷款]]，大桥收费年限已从25年调整到15年；按成本核算，每天至少要收30万元过路费才能保本，即每天要保证3万辆车通行才不亏本。官方解释“之江大桥属于由交投集团投资建设，用收费的方式来偿还建设资金的一条经营性的[[高速公路|高速公路]]，即国企‘直营’的高速公路。”<ref name="通行"/>

== 概况 ==
大桥全长1724米，双向六车道的[[高速公路|高速公路]]标准，设计速度80公里/小时。由于机动车过桥要交10元的过桥费导致过桥车辆偏少，开通至2013年9月这8个多月内日均车流量仅8000辆<ref name="通行">{{cite web|url=http://hznews.hangzhou.com.cn/chengshi/content/2013-09/26/content_4907225.htm | language= | title=10元通行费，冷了之江大桥 |publisher=杭州网 |date=2013-09-26 |accessdate= }}</ref>。大桥主干道两边，各有一条人性化的2.5米宽的慢行辅道，供[[自行车|自行车]]、[[电动车|电动车]]和行人使用，上下桥路程约2公里<ref name="纪录">{{cite web|url=http://hznews.hangzhou.com.cn/chengshi/content/2012-12/18/content_4524259.htm | language= | title=之江大桥有望年底通车 桥上可步行也可骑车 |publisher=杭州网 |date=2012-12-18 |accessdate= }}</ref>。

该收费措施已于2014年8月25日起取消。<ref>{{cite web|url=http://zj.qq.com/a/20140823/005605.htm|title=之江大桥25日起取消收费 2吨以上货车仍禁行|publisher=钱江晚报|date=2014-08-23}}</ref>

== 意义 ==
之江大桥的开通，方便了[[滨江区|滨江区]]、[[双浦镇|双浦镇]]、[[转塘街道|转塘街道]]、[[富阳区|富阳区]]等地的交通往来，也可以有效缓解[[之江路|之江路]]的交通压力。还改善了杭州与[[千岛湖|千岛湖]]、[[黄山|黄山]]之间的交通条件，为“旅游西进”提供交通保障。<ref name= "通车"/>

== 国内纪录 ==
钱江七桥上的两座主桥钢塔——“象牙门”是中国桥梁史上最大的拱形钢塔，刷新国内拱形钢塔斜拉桥三项纪录，分别为跨度最大（247米）、塔高最高（90.5米）、桥面最宽（40.5米）。<ref name= "纪录"/>
