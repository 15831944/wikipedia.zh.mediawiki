'''钱旺集团'''<ref name="a1"/><ref name="a2"/>、'''江苏钱旺信息产业控股集团'''<ref name="a1"/><ref name="a2"/>、'''江苏钱旺集团'''<ref name="a1"/>，是[[张小雷|张小雷]]等人在2010年创建的'''江苏钱旺智能系统有限公司'''（以下简称江苏钱旺）使用的一系列代称，2016年时，公司更名为'''成都钱坤智能系统有限公司'''。张小雷在创建江苏钱旺后，又创建、控股数十家公司。以其多层控股、通过高息吸储的社交型网络交易平台——“'''钱宝网'''”最为知名，而张小雷控股的众多企业又被称为“钱宝系”<ref name="b5"/>。

2017年12月27日，南京警方称张小雷在12月26日向南京市公安机关投案自首。12月28日的报道<ref name="b1"/>，即认为这将是近年来规模仅次于[[e租宝|e租宝]]的投资骗局。2018年2月2日，南京市公安局发布的通报中，初步统计张小雷等人未兑付的[[本金|本金]]为300亿元<ref name="b5"/>。在2015年以来的[[网络借贷|网络借贷]]产生的非法集资案中，非法集资数额仅次于[[e租宝|e租宝]]、[[上海快鹿投资集团|“快鹿系”集资诈骗案]]。

== 钱旺集团 ==

江苏钱旺由张小雷与黄希妍于2010年12月在[[南京市|南京市]][[浦口区|浦口区]]创建，主营业务为软件和信息技术服务业<ref name="b4"/>。张小雷是公司法定代表人和第一大股东，持股94.6%<ref name="b3"/>。2012年7月<ref name="b4">{{Cite web|url=http://www.bjnews.com.cn/finance/2018/01/03/470966.html|title=钱宝网坠落 百分之六七十收益背后的庞氏骗局|accessdate=2018-02-20|author=记者：罗亦丹、王全浩、宓迪|date=2018-01-03|work=编辑：倪雪莹|publisher=新京报网|language=Zh-hans}}</ref>，成立的南京钱宝信息传媒有限公司（以下简称南京钱宝，2016时由张小雷和江苏钱旺完全控股），是钱宝网最早的运营方<ref name="b3"/>。

2015年的报道宣称<ref name="a1">{{Cite web |url=http://www.cqn.com.cn/news/zgzlb/dier/1091149.html |title=江苏钱旺集团旗下吉信甘油公司有望成为全球最大的甘油生产商 |accessdate=2018-02-14 |author=记者：徐建华 |date=2015-11-03 |work= |publisher=中国质量新闻网|language=Zh-hans }}</ref>，“钱旺集团已发展成为23家控股、参股及关联子公司的企业集团”。其投资的江苏吉信甘油科技有限公司拥有亚洲第一、世界第二的[[甘油|甘油]]生产基地。

钱宝网对外宣称，钱旺集团拥有的十余家子公司，主要子公司如下：上海钱柳电子商务有限公司、上海友稔智能科技有限公司（Qbike）、成都钱淼计算机科技有限公司（钱宝、微商）、上海藤榕网络科技有限公司（数据中心）、上海冰穹网络科技有限公司（冰穹互娱）、四川雷神空天科技有限公司等。业务涉及[[共享单车|共享单车]]、[[无人机|无人机]]、[[大数据|大数据]]、[[网络游戏|网络游戏]]、[[电子票证|电影票务]]等领域<ref name="b2">{{Cite web|url=http://epaper.bjnews.com.cn/html/2017-12/28/content_707103.htm?div=-1|title=钱宝网张小雷投案自首旗下泡宝网涉非法发行证券|accessdate=2018-02-20|author=记者：王全浩、陈鹏、黄鑫雨|date=2017-12-28|work=|publisher=新京报电子报|language=Zh-hans}}</ref>。

2016年11月<ref name="b1"/><ref name="b2"/>，江苏钱旺更名为'''成都钱坤智能系统有限公司'''（以下简称成都钱坤），注册地迁至[[四川省|四川省]][[成都市|成都市]][[金堂县|金堂县]]<ref name="b3"/>。同时，江苏钱旺及张小雷控股的南京钱宝及张小雷控股的数家公司一同迁出南京，南京钱宝改名为成都钱乾信息技术有限公司<ref name="b3">{{Cite web|url=http://www.asiafinance.cn/jmnc/113584.jhtml#|title=2017年最后一雷？钱宝网实控人张小雷涉嫌违法向南京警方投案|accessdate=2018-02-20|author=楼云|date=|work=|publisher=亚洲财经|language=Zh-hans}}</ref>。

2017年7月6日，成都市金堂县市场和质量监督管理局将成都钱坤列入经营异常名录<ref name="b3"/>。

== 钱宝网 ==

张小雷于2012年创建钱宝网<ref name="b1">{{Cite web|url=http://finance.sina.com.cn/money/bank/bank_hydt/2017-12-28/doc-ifypyuve1071887.shtml|title=钱宝网500亿的“庞氏骗局”崩塌始末|accessdate=2018-02-20|author=曹天鹏|date=2017-12-28|work=|publisher=新浪网，钛媒体|language=Zh-hans}}</ref>，最初由南京钱宝负责钱宝网运营，后由上海洋井网络科技有限公司（2014年成立）负责运营。上海洋井股东为张小雷及其控股的成都钱乾信息技术有限公司（原南京钱宝）<ref name="b2"/>。

钱宝网于2017年12月26号被南京公安突然关闭，之前已公开运行5年，获得央视及本地媒体正面宣传，自称“平台流水超过500亿元，注册用户超过2亿”<ref>{{Cite web|url=http://www.xinhuanet.com/comments/2017-12/28/c_1122177515.htm|title=钱宝网崩盘，博傻游戏当休-新华网|accessdate=2017-12-31|work=www.xinhuanet.com}}</ref>。缴纳保证金后，完成指定广告任务可获取收益<ref>{{Cite web|url=http://tech.sina.com.cn/i/2014-01-23/11369122256.shtml|title=解密钱宝网商业模式：用户赚钱需先交保证金|accessdate=2017-12-31|work=tech.sina.com.cn}}</ref>，并在高校和各大媒体公开推广5年，<ref>{{Cite web|url=http://finance.sina.com.cn/chanjing/gsnews/2017-12-27/doc-ifyqcsft7301855.shtml|title=疯狂的钱宝网：500亿流水2亿用户 到高校拉学生充值|accessdate=2017-12-31|work=finance.sina.com.cn}}</ref>。2014年钛媒体质疑为其为[[庞氏骗局|庞氏骗局]]<ref>{{Cite web|url=http://tech.163.com/14/0814/10/A3JQ2CAE000915BF.html|title=揭秘钱宝网：宣称看广告赚外快 类似"庞氏骗局"_网易科技|accessdate=2017-12-31|last=网易|work=tech.163.com}}</ref>，但5年来钱宝网未发生兑付违约，多年致力国内慈善事业和服务三农，在[[天津爆炸事故|天津爆炸事故]]发生后向[[天津|天津]]捐款<ref>{{Cite web|url=http://www.chinatimes.cc/article/51153.html|title=钱宝网CEO张小雷重返故里天津  20万宝粉捐赠善款终落实处_财讯_华夏时报网|accessdate=2017-12-31|last=华夏时报|work=www.chinatimes.cc|language=zh-CN}}</ref>。

2015年11月5日，钱旺集团在南京市[[栖霞区|栖霞区]]紫东国际创意园举行全球研发总部启用仪式<ref name="a2">{{Cite web |url=http://digitalpaper.stdaily.com/http_www.kjrb.com/kjrb/html/2015-11/11/content_322798.htm?div=-1 |title=钱旺集团启用全新全球研发总部 |accessdate=2018-02-14 |author=陈杰 |date=2015-11-11 |work= |publisher=中国科技网，科技日报——数字报|language=Zh-hans }}</ref>。钱旺集团董事局主席、钱宝网CEO张小雷出席启用仪式<ref>{{Cite web |url=http://finance.longhoo.net/2015-11/05/content_11517210.htm |title=另一段奇迹的开始——钱旺信控集团全球研发总部今日启用 |accessdate=2018-02-14 |author= |date=2015-11-05 |work=编辑：倪瑞 |publisher=龙虎网|language=Zh-hans }}</ref>。年底，[[e租宝|e租宝]]出事后，[[南京市|南京市]]有关部门要求钱宝网迁出南京<ref>{{Cite news|url=http://finance.sina.com.cn/money/lczx/2016-03-21/details-ifxqnskh1039209.shtml|title=钱宝网争议两重天：创富微商or圈钱网站|date=2016-03-21|accessdate=2017-12-31}}</ref>。

2016年，钱宝网总部搬至[[上海|上海]]，先后在[[紫竹高新区|紫竹高新区]]、[[漕河泾|漕河泾]]、[[周家嘴路|周家嘴路]]等地办公<ref>{{Cite web|url=http://www.sohu.com/a/213372227_632979|title=视线｜实探钱宝网上海总部，狡兔三窟频换地址|accessdate=2017-12-31|date=2017-12-28|work=www.sohu.com}}</ref>，2017年8月有投资人发现钱宝网总部已人去楼空，但遭张小雷否认<ref>{{Cite web|url=http://www.xinhuanet.com/fortune/2017-12/28/c_1122176704.htm|title=钱宝网张小雷被主动投案自首-新华网|accessdate=2017-12-31|work=www.xinhuanet.com}}</ref>。

== 钱宝案发及司法进程 ==

2016年8月，既有“钱宝网涉嫌诈骗，[[张小雷|张小雷]]向南京警方自首”的传言<ref name="c1"/>。而经过公众、媒体的多年质疑、南京官方半公开的“封杀”后。最终于2017年12月26日，钱宝网实际控制人张小雷向[[南京市公安局|南京市公安机关]]投案[[自首|自首]]<ref>{{Cite web|url=http://www.fj.chinanews.com/news/fj_jsxw/2017/2017-12-27/398657.html|title=钱宝网实际控制人张小雷涉嫌违法犯罪投案自首-中新网福建|accessdate=2018-02-21||last=|first=|author=|date=2017-12-27|work=|publisher=中国新闻网|language=Zh-hans}}</ref>。27日上午，南京市公安局通过[[新浪微博|新浪微博]]的官方帐号“平安南京”发布张小雷自首的消息。此则消息最初被部分钱宝网投资人质疑真实性，怀疑是“平安南京”被盗号所发<ref name="c2">{{Cite news|url=http://finance.caijing.com.cn/20171228/4385907.shtml|title=本年度最忧伤骗局：钱宝网暴雷、老板自首，宝粉们却选择“不相信”_互联网金融_金融频道首页_财经网 - CAIJING.COM.CN|accessdate=2017-12-31|work=编辑：胡伟桐|publisher=财经网|language=zh-CN|archive-url=https://web.archive.org/web/20171228044553/http://finance.caijing.com.cn/20171228/4385907.shtml#|archive-date=2017-12-28|dead-url=yes}}</ref>。

张小雷自首后，由南京市公安局江北新区分局（以下简称江北分局）立案侦查<ref name="b5"/>。随后，张小雷因涉嫌[[非法集资罪|非法集资罪]]以及其他人士被江北新区分局采取刑事强制措施<ref name=police>{{Cite web|url=http://www.xinhuanet.com/fortune/2017-12/29/c_129779005.htm|title=张小雷被采取刑事强制措施　警方吁请钱宝网用户配合调查取证-新华网|accessdate=2017-12-31|work=www.xinhuanet.com|language=zh-CN}}</ref>。<ref name="c1">{{Cite web|url=http://finance.sina.com.cn/money/bank/gsdt/2017-12-29/doc-ifyqcwaq5650284.shtml|title=忠粉程序员稳赚5年一朝赔光 钱宝网2亿用户如何维权|accessdate=2018-02-21|author=|date=2017-12-29|work=责任编辑：杨畅|publisher=网易|language=Zh-hans}}</ref>。[[江苏省公安厅|江苏警方]]称追回的钱款将按比例退还<ref>{{Cite web|url=http://finance.sina.com.cn/chanjing/gsnews/2017-12-29/doc-ifyqcwaq5524270.shtml|title=钱宝网最新进展：江苏公安厅称追回钱按比例退还|accessdate=2017-12-31|work=finance.sina.com.cn}}</ref>。

此前，钱旺集团在2013年投资成立[[南京钱宝足球俱乐部|南京钱宝足球俱乐部]]（后更名为[[成都钱宝足球俱乐部|成都钱宝足球俱乐部]]）。而2014年起，钱宝网与[[皇家社会|皇家社会]]合作，成为其赞助商<ref>{{Cite news|url=http://sports.sina.com.cn/g/laliga/2015-03-31/15527560664.shtml|title=钱宝网与皇家社会完成续约 下赛季继续亮相西甲|accessdate=2017-12-31}}</ref>。张小雷自首后，皇家社会称将与钱宝网解约<ref>{{Cite web|url=http://we.sportscn.com/viewnews-2429637.html|title=皇家社会官方：钱宝网事件若属实 将与其解除合同_国际足球_五大联赛_华体即时比分直播网|accessdate=2017-12-31|work=we.sportscn.com}}</ref>，[[成都钱宝足球俱乐部|成都钱宝足球俱乐部]]称正进一步了解情况<ref>{{Cite web|url=http://www.xinhuanet.com/sports/2017-12/28/c_1122177068.htm|title=成都钱宝投资人投案自首，俱乐部、足协：正了解情况-新华网体育|accessdate=2017-12-31|work=www.xinhuanet.com}}</ref>。

2018年1月22日，约两千名钱宝网投资人从[[南京艺术学院|南京艺术学院]]出发，向[[南京市人民政府|南京市人民政府]]驻地游行。投资人要示南京政府确保钱宝网正常运营，并与南京警方发生冲突。有投资人认为“当初‘钱宝网’能吸引大批投资人与政府造势有关，当局理应负起责任。”“我们是响应政府的号召，有多位高官给我们‘钱宝网’站台，还有一些媒体的渲染，我们才会倾家荡产地投入到‘钱宝网’里面的，因为我们相信政府，相信中央电视台，相信官媒，相信党，相信国家。我们投入了以后，你现在又说我们是非法的。所以我们现在必须要有一个说法。<ref>{{Cite web|url=https://www.rfa.org/mandarin/yataibaodao/renquanfazhi/gf2-01232018103520.html|title=“钱宝网”非法集资受害者南京游行 爆发警民冲突|accessdate=2018-02-25|author=特约记者：高锋|date=2018-01-23|work=责编：石山、寇天力，网编：郭度|publisher=自由亚洲电台|language=Zh-hans}}</ref>”

1月30日<ref>{{Cite web|url=http://finance.ifeng.com/a/20180202/15965920_0.shtml|title=终于等到你｜钱宝网官方案件受理登记系统正式开通|accessdate=2018-02-20|author=|date=2018-02-02|work=|publisher=凤凰网，来源：AI财经社|language=Zh-hans}}</ref>，南京市公安局官网开通“钱宝网用户配合调查取证受理登记平台”（钱宝网案件受理登记系统）。2月2日，南京市公安局通过“平安南京”，发布钱宝案详情。此前一日，经南京市浦口区人民检察院批准，江北新区分局以涉嫌[[非法吸收公众存款罪|非法吸收公众存款罪]]对张小雷、周某富、康某斌、于某建、端某飞等12人依法逮捕<ref name="b5">{{Cite web|url=http://tech.163.com/18/0202/09/D9KM1LLG00097U7R.html|title=南京警方公布钱宝案详情：张小雷非法集资养情妇|accessdate=2018-02-20|author=|date=2018-02-02|work=责任编辑：白鑫_NT4464|publisher=网易|language=Zh-hans}}</ref>。

7月27日，南京市公安局江北新区分局依法对“钱宝系”企业实际控制人张小雷以涉嫌集资诈骗罪移送检察机关审查起诉<ref>{{Cite web|url=http://www.xinhuanet.com/fortune/2018-07/27/c_1123187323.htm|title=“钱宝系”非法集资超千亿元 张小雷被移送起诉 |accessdate=2018-07-27||last=|first=|author=|date=2018-07-27|work=|publisher=新华网|language=Zh-hans}}</ref>。
