'''钱江四桥'''，又称'''复兴大桥'''，是[[杭州市|杭州市]]一座跨[[钱塘江|钱塘江]]特大型城市桥梁，2002年3月28日开工建设<ref name= "复兴"/>，于2004年10月16日竣工通车<ref name="通车">{{cite web|url=http://www.gxnews.com.cn/staticpages/20041017/newgx417159d2-265621.shtml | language= | title=壮乡彩虹架到"天堂" 广西路桥人精建钱江四桥通车 |publisher=新华社 |date=2004年10月17日 |accessdate= }}</ref>。它位于[[钱江一桥|钱江一桥]]与[[钱江三桥|钱江三桥]]之间（一桥下游4.3公里处）。将杭州南部的[[滨江区|滨江区]]与北部的[[上城区|上城区]]相连接。

== 简介 ==
总投资7.6亿元，由[[广西|广西]]路桥工程总公司的人员设计建设。<ref name= "通车"/>

全长1376米，宽26.4米，分上下两层，是世界上第一座双层钢拱形桥梁。上层设双向6条[[机动车|机动车]]快速行车道，连接老城区的中河高架与[[滨江区|滨江区]]的时代高架；下层设4条机动车道及2条行人、非机动车通道和8个观景平台。<ref name= "通车"/>它是连接杭州老城区与滨江区和[[萧山区|萧山区]]的市内交通要道，是钱塘江上过江能力最强的桥梁之一<ref name="复兴">{{cite web|url=http://www.hangzhou.com.cn/20081124/ca1621060.htm | language= | title=钱江四桥（复兴大桥） |publisher=杭州网 |date=2008年12月17日 |accessdate= }}</ref>。

复兴大桥在世界桥梁史上的创新意义，在于它是大小拱结合的钢管混凝土系杆拱新型拱桥，多跨连拱，双层路面，其中有两座190米跨径的主拱，9座85米跨径的小拱。


== 历史 ==
*2002年3月28日，大桥开工。<ref>{{cite web |title=钱江四桥今天开工 建成后不收过桥通行费 |url=http://news.sina.com.cn/s/2002-03-28/1743526547.html |website=新浪新闻}}</ref>
*2002年9月8日，受到[[风暴潮|风暴潮]]影响，建设中的桥墩被[[钱江潮|钱江潮]]冲垮，300米施工便桥被摧毁。<ref>{{cite web |title=今年会不会出现2002年那样的大潮？ - 杭州新闻中心 - 杭州网 |url=http://hznews.hangzhou.com.cn/chengshi/content/2010-07/30/content_3376121.htm |website=杭州网}}</ref>
*2004年10月16日，大桥通车<ref>{{cite web |title=2004年10月16日 “复兴大桥”全线通车-科普中国 |url=http://www.xinhuanet.com/science/2017-10/16/c_136682092.htm |website=新华网}}</ref>
*2012年，钱江四桥将首次启动大型维修养护工程，工期大约2-3个月，预计11月份开工。最主要的工程就是对桥面重新铺设，施工面积有7000多平方米。重新铺设的[[沥青|沥青]]选择了强度大、稳定性更好、更耐磨、寿命更长的SMA改性沥青，以代替原先的普通沥青。<ref>{{cite web|url=http://n.cztv.com/zhejiang/2012/09/2012-09-143563647.htm?shidian20120914 | language= | title=杭州：通行9年 钱江四桥11月起首次大修 |publisher=浙江卫视 |date=2012-09-14  |accessdate= }}</ref>

== 轶事 ==
复兴大桥下层设计为“按照公路桥开通，但预留改建为轨道交通通道”，并作为[[杭州地铁1号线|杭州地铁1号线]]预留过江通道。但由于1号线技术突破，改为隧道形式过江，地铁5号线在设计过程中也放弃此处预留，因而此预留从未得以实施。<ref>{{cite web |title=青年科学家揭秘：杭州地铁1号线为何改走过江隧道？-地铁1号线-浙江在线-住在杭州 |url=http://zzhz.zjol.com.cn/05zzhz/system/2009/12/15/016153517.shtml |website=浙江在线}}</ref>
