'''钱大昕'''（{{bd|1728年||1804年|catIdx=Q钱}}），[[表字|字]]'''晓-{徵}-'''，一字'''辛楣'''，[[号|号]]'''竹汀'''，[[江苏|江苏]][[嘉定縣|嘉定縣]]（今[[上海|上海]][[嘉定区|嘉定区]]）人，[[清代|清代]][[史学家|史学家]]，[[语言学家|语言学家]]。

== 生平 ==
幼時為神童，<ref>钱氏有《初春怀乡杂咏》詩云：“人日元宵兴不孤，小时了了记欢娱。烧灯巧解藏头谜，对局偷拈谒选图。”</ref>周岁能认字，三岁诵匾联，五岁受经书，十岁学[[八股文|八股文]]。十五岁应童子试，为内阁学士[[刘藻|刘藻]]所赏識，谓“吾视学一载，所得惟王生鸣盛、钱生大昕两人耳”。[[王艮斋|王艮斋]]阅其课义，诧为“此天下才也”。早年以詩賦聞名[[江南|江南]]，獻賦獲賜[[舉人|舉人]]，又精通[[经学|经学]]、[[史学|史学]]、[[天文|天文]]、历算、[[音韵|音韵]]、[[训诂|训诂]]、[[金石|金石]]，“不专治一经而无经不通，不专攻一艺而无艺不精。”<ref>[[江藩|江藩]]：《汉学师承记》卷三。</ref>錢氏於《[[元史|元史]]》用功极深，在百卷《[[二十二史考異|二十二史考異]]》中，《元史考异》独占十五卷。[[乾隆|乾隆]]十九年中[[進士|進士]]，選[[翰林院庶吉士|翰林院庶吉士]]，[[散館|散館]]授[[編修|編修]]，歷升[[侍講學士|侍講學士]]，三十四年，入直上書房，授皇十二子書。升[[少詹事|少詹事]]，不久[[提督|提督]][[廣東|廣東]][[學政|學政]]。[[乾隆|乾隆]]四十年（1775年）因[[丁憂|父喪]]去職，服阕後又遇母喪，以病不再复出。好讀書，亦善治學，未嘗一日廢書。<ref>《严久能娱亲雅言序》云：“今海内文人学士，穷年累月，肆力于铅椠，孰不欲托以不朽？而每若有不敢必者，予谓可以两言决之，曰：‘多读书而已矣，善读书而已矣。’胸无万卷书，臆决唱声，自夸心得，纵其笔锋，亦足取快一时。而沟浍之盈，涸可立待。小夫惊而舌挢，识者笑且齿冷，此固难以入作者之林矣。”</ref>曾在[[鐘山|鐘山]]、婁東、[[紫阳书院_(苏州)|紫陽]]等各書院講學多年，桃李满天下，“门下士积二千余人，其为台阁、侍从、发名成业者，不可胜计。”[[嘉庆|嘉庆]]九年（1804年）卒。<ref>《清史稿·列传二百六十八·儒林二》：钱大昕，字晓徵，嘉定人。乾隆十六年召试举人，授内阁中书。十九年进士，选翰林院庶吉士，散馆授编修。大考二等一名，擢右春坊右赞善。累充山东乡试、湖南乡试正考官，浙江乡试副考官。大考一等三名，擢翰林院侍讲学士。三十二年，乞假归。三十四年，补原官。入直上书房，迁詹事府少詹事，充河南乡试正考官。寻提督广东学政。四十年，丁父艰，服阕，又丁母艰，病不复出。嘉庆九年，卒，年七十七。</ref>

==學術==
钱大昕对于[[音韵学|音韵学]]和[[训诂学|训诂学]]很有创见。首先注意到了古声母的研究，证明了古代没有[[唇齒音|轻唇音]]和[[三十六字母#王力拟音音表|舌上音]]（[[卷舌音|卷舌]][[塞音|塞音]]）的分别。錢大昕考証，上古無輕唇音（唇齒音），“無”發成“模”，又轉為“毛”。“毛”的古音為“謀”，所以模能轉為毛。錢大昕撰《[[廿二史考異|廿二史考異]]》一百卷，以[[顧炎武|顧炎武]]之歷史考據方法，拾遺規過，最享時譽。[[曹聚仁|曹聚仁]]說：“錢大昕推許[[戴東原|戴東原]]‘實事求是，不主一家’，儼然是科學家的頭腦了。假如他們研究的對像是自然科學的話，他們便是[[達爾文|達爾文]]、法布耳那樣的科學家了。”他還稱讚說：“這（考據學）便是[[牛頓|牛頓]]、達爾文的治學態度。”<ref>[[曹聚仁|曹聚仁]]：《中國學術思想史隨筆》</ref>錢大昕也寫過（奕喻）這樣的文章，他在下棋時，領悟了一些道理：「今之學者，讀古人書，多訾古人之失；與今人居，亦樂稱人失。人固不能無失，然試易地以處，平心而度之，吾果無一失乎？吾能知人之失，而不能見吾之失；吾能知人之小失，而不能見吾之大失」。

钱大昕轻视西方[[数学|数学]]，[[江永_(人物)|江永]]因学习西方数学，钱大昕却讥其“为西人所用”<ref>钱大昕：《潜研堂文集》卷三十三，《与戴东原书》</ref>。钱大昕说《[[诗经|诗经]]》已有反切，並且驳斥反切受[[佛经|佛经]]的影响，“岂古圣贤之智乃出梵僧下耶”，“吾于是知六经之道，大小悉备，后人詹詹之智，早不出圣贤范围之外也”<ref>钱大昕：《潜研堂文集》卷十五，《答问十二》。</ref>。錢大昕面對西學所採取的態度是：“西士之術固有勝於中法者，習其術可也，習其術而為所愚弄不可也。”

[[趙翼|趙翼]]的《[[廿二史劄記|廿二史劄記]]》、[[王鳴盛|王鳴盛]]《[[十七史商榷|十七史商榷]]》與錢大昕的《廿二史考異》合稱「清代三大史學名著」。

== 評價 ==
* [[戴震|戴震]]说：“当代学者，吾以晓徵（即钱大昕）为第二人”，“盖东原毅在以第一人自居。然东原之学，以肄经为宗，不读汉以后书。若先生（钱大昕）学究天人，博综群籍。自开国以来，蔚然一代儒宗也。以汉儒拟之，在高密（郑康成）之下，即贾逵、服虔，亦瞠乎后矣”<ref>[[江藩|江藩]]：《汉学师承记》卷三。</ref>

* 近代史學家[[陳寅恪|陳寅恪]]獨服錢大昕，稱之為“清代史家第一人”。陳寅恪《元白詩箋證稿》依錢大昕《十駕齋養新錄》所謂「唐制服色不視職事官，而視階官之品」，考證當時“江州司馬青衫濕”的[[白居易|白居易]]，雖名義上為「從五品下」，然卻是「將仕郎守江州司馬」；據此，陳寅恪得出以下結論：「樂天此時止為州佐，固唯應依將仕郎之階品著青衫也。」。

* [[陈垣|陈垣]]曾说：“《[[日知录|日知录]]》在清代是第一流的，但还不是第一，第一应推钱大昕的《[[十驾斋养新录|十驾斋养新录]]》。”<ref>《励耘书屋问学记》，三联书店，1982，第159页。</ref>

== 貢獻 ==
=== 音韻學 ===
在錢之前，研究古音眾多，但都衹注重古韻，真正注意到古紐問題者當推錢氏。關於古紐，錢氏提出四個要點：
* [[古無輕脣音|古無輕脣音]]
* [[古無舌上音|古無舌上音]]
* [[古無正齒音|古人多舌音]]
* 古[[影母|影]][[喻母|喻]][[曉母|曉]][[匣母|匣]]雙聲
* 此外，錢氏對于古音也提出許多看法，如主張詩經有正音，有轉音。正音，就是從偏旁得聲；轉聲，就是「聲隨義轉或「雙聲假借」

主要著作：
* 《[[恒言录|恒言录]]》
* 《[[十驾斋养新录|十驾斋养新录]]》
* 《[[二十二史考異|二十二史考異]]》
* 《[[潜研堂文集|潜研堂文集]]》
