'''钱江世纪城'''位于[[中国|中国]][[浙江省|浙江省]][[杭州市|杭州市]][[钱塘江|钱塘江]]东南岸的[[萧山区|萧山区]][[宁围街道|宁围街道]]，是正在建设中的[[杭州|杭州]]未来的一个[[中央商务区|中央商务区]]和新城区。总规划面积22.27平方公里，规划居住人口16万。<ref name="未来">{{cite web|url=http://news.qq.com/a/20080830/000291.htm | language= | title=未来的钱江世纪城令人向往 |publisher=杭州网 |date=2008年8月30日 |accessdate= }}</ref>其中核心区块为9.6平方公里。<ref name= "开工"/>

== 简介 ==
世纪城位于萧山城北江滨地区，与[[钱江新城|钱江新城]]隔江相望。西北至钱塘江滨，西南至七甲闸——利民河，东北到[[杭甬高速公路|杭甬高速公路]]，南到前解放河，东至利群河和市心路。<ref>[http://www.qjcbd.com/news_show.asp?newsid=663 规划范围]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>2008年开工建设。<ref name="开工">{{cite web |url=http://news.163.com/10/0517/08/66SDJ3EC00014AEE.html |language= |title=钱江世纪城有50幢高楼开工 |publisher=杭州网 |date=2010-05-17 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150502160246/http://news.163.com/10/0517/08/66SDJ3EC00014AEE.html |archivedate=2015-05-02 }}</ref>

世纪城将主要引进金融商贸、商务办公、科技信息、会展咨询、体育休闲等现代服务业，积极吸引[[世界五百强|世界五百强]]企业地区总部、国际金融机构地区分行和国内著名民营企业总部，以争取建设成杭州的“[[陆家嘴|陆家嘴]]”。<ref name= "未来"/>

== 规划结构 ==
世纪城由以下区域组成：
*商务综合功能区
*中心特色产业区
*[[杭州奥体博览城|休闲体育博览区]]
*沿江公建区
*居住综合功能区
*居住区

== 交通 ==
[[钱江三桥|钱江三桥]]与[[庆春路过江隧道|庆春路过江隧道]]连接世纪城与对岸的杭州老城区，同时位于世纪城南部的[[杭州机场高速公路|杭州机场高速公路]]将新城与杭州市区和[[杭州萧山国际机场|杭州萧山国际机场]]连接起来。[[杭州地铁2号线|杭州地铁2号线]]在此设有[[钱江世纪城站|钱江世纪城站]]，可前往萧山和江干。
