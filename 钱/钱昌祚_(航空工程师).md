'''钱昌祚'''（{{bd|1901年||1988年|}}）字'''莘觉'''、'''惺觉'''，[[江苏省|江苏]][[常熟市|常熟]][[鹿苑街道|鹿苑]]（今[[张家港市|张家港市]]）人，[[中华民国|中华民国]]航空工程师及政治人物，其堂哥是[[钱昌照|钱昌照]]。

钱昌祚1913年入上海私立浦东中学就读，1917年考入[[清华大学|清华学校]]，1919年毕生后被保送赴美留学，入[[麻省理工学院|麻省理工学院]]机械系就读，1922年获机械工程学士学位，后转入航空机械工程系，1924年获航空工程硕士学位。回国后在[[浙江公立工业专门学校|浙江工业专门学校]]及[[清华大学|清华大学]]任教，曾任[[杭州市|杭州市]]公用局科长。1928年进入航空领域，1933年7月18日任[[國民政府軍事委員會|國民政府軍事委員會]]航空署技术处处长，1934年5月任国民政府航空委员会技术处处长。1934年4月1日担任中国航空工程学会会长。1936年3月任[[空军航空技术学院|空军机械学校]]校长，1938年3月任航空委员会技术厅副厅长、代理厅长。1940年任[[南川飞机制造厂|南川第二飞机制造厂]]厂长。1941年6月接替[[邢契莘|邢契莘]]成为[[雷允飞机制造厂|中央雷允飞机制造厂]]监理，为[[中央飞机制造厂|中飞厂]]第三任监理。1944年12月任航空委员会参事。1946年6月5日任[[中华民国国防部|中华民国国防部]]第六厅厅长，1947年4月任国防部国防科学委员会委员。1948年9月22日晋任空军机械监。1949年前往[[台湾|台湾]]，历任[[中华民国经济部|中华民国经济部]]常务次长，[[行政院經濟安定委員會|行政院經濟安定委員會]]秘书长，[[行政院经济建设委员会|行政院美援运用委员会]]委员，外汇贸易审议委员会副主任委员，[[中央信托局|中央信托局]]常务理事，台湾手工业中心董事长。曾任“中国航空工程学会”、“中国机械工程学会”、“中国工程师学会”会长、理事长。曾任[[美国|美国]][[南加州大学|南加州大学]]客座教授。1971年退休，1988年病逝。著有《惺觉论选》及《浮生百记》。<ref>{{cite web |url=https://m.sohu.com/n/481486272/ |title=珍贵史料：民国时期中国飞机制造业三杰 |last= |first= |date=2017-02-23 |website= |publisher=航空制造网 |access-date=2017-11-30 |quote=}}</ref><ref>{{cite web |url=http://www.tsinghua.edu.cn/publish/xsg/8348/2014/20140411090953231740230/20140411090953231740230_.html |title=抗战前清华航空研究考察 |author=金富军 王向田 |date= |website=www.tsinghua.edu.cn |publisher=清华大学校史馆 |access-date=2017-11-30 |quote=}}</ref>
