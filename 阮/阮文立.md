== 早年生活 ==
科斯塔斯·萨兰蒂蒂斯1927年生于[[希腊|希腊]][[塞萨洛尼基|塞萨洛尼基]]，父母是[[小亚细亚|小亚细亚]]难民。1943年秋和[[第二次世界大战|第二次世界大战]][[軸心國佔領希臘時期|軸心國佔領希臘時期]]，萨兰蒂蒂斯因贩卖走私[[烟草|烟草]]被捕，后被徒步押往德国[[納粹德國|納粹]]强制劳动营。后来他设法在[[維也納|維也納]]附近逃脱，还偷了一套军装。战争结束前，他一直都把自己伪装成一名德国人。

== 越南生涯 ==
战争结束后，萨兰蒂蒂斯去往[[罗马|罗马]]，想被遣返回希腊。然而这是不可能的，因为他没有任何[[身分證|身份证明]]。萨兰蒂蒂斯被诱骗进[[法国外籍兵团|法国外籍兵团]]，想着过一种充满冒险的生活，邂逅一些漂亮女人<ref name="kathimerini"/>。加入军团后，萨兰蒂蒂斯先转到[[阿尔及利亚|阿尔及利亚]]，1946年转到[[中南半島|中南半島]]，被告知部署时间不长，任务就是解除日军武装，恢复秩序。萨兰蒂蒂斯反感法国殖民军队压迫当地居民的行为，待了两个月后，他联系了越南[[间谍|间谍]]，带着他的步枪和一挺机枪前去[[投誠|投誠]]。他有了个越南名字阮文立（Nguyễn Văn Lập），参加了多场战役，最后做到了[[上尉|上尉]]军衔。1949年，他获准加入[[越南共产党|越南共产党]]。

1954年战争结束后，越南南北分裂，萨兰蒂蒂斯退役，搬至[[北越|北越]]。他的妻子是一名护士，其时被指控为[[反动|反动分子]]入狱<ref name="kathimerini"/>。萨兰蒂蒂斯当了一名[[德语|德语]]翻译，后来又当了一名[[矿工|矿工]]。他和一越南女子再婚，与她有三个孩子。

== 希腊晚年 ==
自在塞萨洛尼基被捕到50年代早期，萨兰蒂蒂斯都没有和家人联系，因为他们以为他已经死了。战争快结束时，他才开始与他们写信。1965年，他决定返回希腊。在他的一个兄弟帮助下，萨兰蒂蒂斯一家人弄到护照，搬到了塞萨洛尼基。彼时他已失业数月，面临严重经济困难。最后，他终于在彼施涅（Pechiney，一译贝奈、普基）找了份司机的工作，几年后退休。在希腊，萨兰蒂蒂斯加入了[[希臘共產黨|希臘共產黨]]，致力于帮助越南并促进希越关系。此外，萨兰蒂蒂斯还积极帮助那些受[[橙劑|橙劑]]毒害的越南儿童。

2008年10月，萨兰蒂蒂斯陪同时任希腊总统[[卡罗洛斯·帕普利亚斯|卡罗洛斯·帕普利亚斯]]访问越南。2010年，萨兰蒂蒂斯获得越南公民身份和护照。2013年，萨兰蒂蒂斯获[[人民武装力量英雄|人民武装力量英雄]]称号。<ref>{{Cite news|url=http://cn.nhandan.org.vn/friendshipbridge/vietnam-and-the-world/item/1150201-%E8%B6%8A%E5%8D%97%E4%B8%8E%E5%9F%83%E5%8F%8A%E6%8E%A8%E5%8A%A8%E5%85%A8%E9%9D%A2%E5%90%88%E4%BD%9C%E5%85%B3%E7%B3%BB.html?PageSpeed=noscript|title=英勇的国际主义战士阮文立荣获武装力量英雄称号|newspaper=英勇的国际主义战士阮文立荣获武装力量英雄称号|accessdate=2018-11-09|language=zh-cn}}</ref><ref name="vietnamplus">{{Cite news|url=http://en.vietnamplus.vn/greek-citizen-receives-vietnamese-honour/48623.vnp|title=Greek citizen receives Vietnamese honour|date=2013-08-30|newspaper=vietnamplus.vn|accessdate=2018-11-08}}</ref>
