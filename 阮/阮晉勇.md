'''阮晉勇'''（{{lang-vi|'''Nguyễn Tấn Dũng'''}}，{{bd|1949年|11月17日}}），生於[[南越|南越]][[金甌省|金甌省]][[金瓯市|金瓯市]]，[[越南|越南]]政治人物，政治傾向屬南方亲美派，[[越共中央政治局|越共中央政治局]]委員（黨內位列第三，在總書記和國家主席之後），2006年6月至2016年4月任[[越南政府总理|越南政府总理]]、党组书记，还担任国防与安全委员会副主席。<ref>[http://global.dwnews.com/news/2014-05-16/59471984-2.html 南海危机急转向 越南暴力反华变谨慎防美] {{webarchive|url=https://web.archive.org/web/20140517121536/http://global.dwnews.com/news/2014-05-16/59471984-2.html |date=2014-05-17 }}</ref>

== 從政簡歷 ==
阮晉勇出生於在越南南部（當時仍屬[[越南国|法國附庸]]）[[金瓯市|金瓯市]]。據稱他1961年至1981年在[[越南人民军|越南人民军]]服役，任文书、通信员、卫生员（战地救伤）、护士、医士。历任班长(Tiểu đội bậc trưởng，[[中士|中士]]级别）、排长(Trung đội bậc trưởng，准尉级别)、副连长(Đại đội bậc phó, 少尉级别)、连长(Đại đội bậc trưởng, 中尉级别)-手术队队长、 [[坚江省|迪石省队]]军医连连长兼政治员长(Đại đội trưởng và Chính trị viên trưởng Đại đội Quân y thuộc Tỉnh đội Rạch Giá)。

1967年6月10日加入越南劳动党，1968年3月10日转为正式党员。1969年下半年的美军的金瓯省-坚江省反游击队大扫荡中，阮晋勇身负重伤几乎阵亡。1975年越南战争结束后，阮晋勇以上尉军衔选择了继续留在军队工作，任第207步兵营的政治员长（Chính trị viên trưởng Tiểu đoàn Bộ binh 207）、第152步兵团大尉政治主任(Chủ nhiệm Chính trị Trung đoàn Bộ binh 152)。 参加了1976-1977年的[[保卫越南西南边界战争|保卫越南西南边界战争]]和1978年开始的[[柬越战争|柬越战争]]。最后一个军队职务是坚江省军事指挥部少校干部部长，曾在阮爱国高级党校学习。

1981年转业地方后担任[[坚江省|坚江省]]省委组织与干部部副部长；坚江省委常委、坚江省[[河仙市|河仙]]县委书记；1990年任坚江省委副书记兼省人民委员会主席、越军第9军区坚江省军事委员会书记等职。在[[越南共产党第六次全国代表大会|越共六大]](1986年)当选为中央候补委员、越共七大(1991年)当选为中央委员。1995年1月，调入中央工作，出任越南内务部副部长、党中央检查委员会委员。1996年6月在越共八大上当选为政治局委员、政治局常委、中央书记处书记、中央经济委员会负责人，负责解决金融问题。1997年9月29日在越南第十届国会当选为政府常务副總理，出任政府金融与货币委员会主席。1998年1月不再担任政治局常委。1998年5月国会任命阮晋勇兼任[[越南国家银行|越南国家银行]]行长，推动金融系统的改革与整顿，直至1999年12月卸任。2001年4月越共九大上当选政治局委员排名第5（从这届开始越共不再设政治局常委）。2002年8月，越南政府换届，继续担任副总理。

== 总理 ==
2006年4月，越共十大上当选[[越南共产党中央政治局|中央政治局]]委员，排名第3，5月16日，獲時任總理的[[潘文凱|潘文凱]]提名阮晋勇为总理候选人。 2006年6月24日，潘文凯宣布辞职。6月27日越南国会选举阮晋勇为总理。2007年6月25日，第12届国会选举新一任政府，阮晋勇继续担任总理，得票率96.96%。

2011年6月25日，第13届[[越南国会|越南国会]]选举新一届政府，阮晋勇继续留任总理。7月25日，新当选的越南国家主席[[张晋创|张晋创]]根据[[越南共产党中央委员会|越共中央]]的建议，向国会推荐阮晋勇为越南新一届总理候选人；7月26日下午，越南第十三届国会第一次会议投票通过提名，现任总理阮晋勇以94%得票率获得连任<ref>[http://news.163.com/11/0726/16/79TDFHGR00014JB6.html 阮晋勇连任越南总理]</ref>，任期將至2016年。

在2016年1月的[[越南共產黨第十二次全國代表大會|越共十二大]]前，阮晉勇一度被外界認為能更進一步當選[[越共中央总书记|越共中央总书记]]，但最終[[阮富仲|阮富仲]]连任总书记职务，且未能當選進入越共中央委員會，被認為是權力斗爭失敗。<ref name="VN12">{{cite web|url=http://www.bbc.com/zhongwen/trad/world/2016/01/160127_vietnam_partycongress |language= |title=越共新領導出爐「親中」阮富仲連任總書記 |publisher=[[bbc|bbc]]|date=2016-01-27 }}
</ref><ref name="mingpao"></ref>

2016年4月6日，阮晋勇总理任期届满，越南国会投票同意阮晋勇卸任政府总理一职，次日经投票由唯一候选人[[阮春福|阮春福]]接任。<ref>{{cite news|url=http://www.881903.com/Page/ZH-TW/newsdetail.aspx?csid=261_367&itemid=864402|date=2016-04-07|publisher=[[香港商業電台|香港商業電台]]|title=越南國會通過副總理阮春福接任總理}}</ref>

===經濟政策===
阮晉勇任内主導下，越南實施經濟改革，2007年協助越南加入[[世界貿易組織|世界貿易組織]]（WTO），推進與歐盟和韓國等的自由貿易協定（FTA），加上引進大量外資，令越南人均GDP增長3倍，2015年經濟增長達6.7%。阮晉勇領導越南參与了由美國主導的[[跨太平洋戰略經濟夥伴關係協議|跨太平洋戰略經濟夥伴關係協議]]（TPP），將對越南帶來重大經濟利益，被認為是阮晉勇任内的重大政績。<ref name="mingpao">{{cite web|url=http://news.mingpao.com/pns/dailynews/web_tc/article/20160127/s00014/1453831707297|title=越南權力洗牌 「最強總理」出局 中央委員會改選 保守派總書記留任|publisher=明報|date=2016-01-27}}</ref>

但是，阮晉勇實施的經濟改革面臨反對激進改革的保守派領導層的反對。保守派擔心，經濟過份開放會令市場被外資奪走，將令國內經濟陷入疲弊。保守派指摘他處理經濟不當，令國營公司倒閉、無力控制龐大公共債務、縱容貪污等。<ref name="japan news"></ref>

阮晉勇任内越南也面對嚴重的經濟問題和一系列金融腐敗事件，导致他也受到许多批评，甚至有议员要求他下台<ref>{{cite web|url=http://www.21ccom.net/articles/qqsw/qqgc/article_2013021877167.html |language= |title=崔向升：越南“政改”改几许——总理在国会挨批 |publisher=[[共识网|共识网]]  |date=2013-02-18 |accessdate=2016-01-28|archiveurl=https://web.archive.org/web/20160205051336/http://www.21ccom.net/articles/qqsw/qqgc/article_2013021877167.html|archivedate=2016-02-05|deadurl=yes}}</ref>。阮晋勇任内同時還面對隨金融危機而來的銀行壞帳問題，以及越南必須面對欠缺管理現代銀行體系的實際操作經驗的問題<ref>{{Cite web |url=http://www.takungpao.com/world/content/2012-09/12/content_1087648.htm |title=越南银行体系坏帐超一成 新兴市场逐步丧失魅力 |access-date=2016-01-28 |archive-url=https://web.archive.org/web/20160203235200/http://www.takungpao.com/world/content/2012-09/12/content_1087648.htm |archive-date=2016-02-03 |dead-url=yes }}</ref>。2012年越共第十一屆中央委員會第六次會議時，阮晋勇因經濟與金融問題受到不點名批評，但最終越共中央委員會沒有處分總理阮晉勇<ref>{{cite web|url=http://www.bbc.com/zhongwen/trad/world/2012/10/121015_vietnam_pm.shtml|title=越共全會沒有處分總理阮晉勇|publisher=[[bbc|bbc]]|date=2012-10-15}}</ref>。

===外交关系===
阮晉勇在任期間，主張對中國在南海的領土主張採取強硬姿態。2014年5月，中國將石油勘探裝置移至西沙群島，阮晉勇曾稱「我們不會用主權與合法權利來換取與中國之間的虛偽而從屬性的友情」。<ref name="japan news">{{cite web|title=阮富仲為何能打敗阮晉勇|url=http://zh.cn.nikkei.com/politicsaeconomy/politicsasociety/18038-20160128.html|publisher=[[日本經濟新聞|日本經濟新聞]]|date=2016-01-28}}</ref>

阮晉勇也被認為是亲美派，在阮晉勇任内，他帶領越南與美國建立良好關係。<ref name="mingpao"></ref>

== 家属 ==
长子阮清谊。生于1976年8月16日，2011年成為越共中央委員會的候補委員，兼任建設部副部長。39岁当选越南[[坚江省|坚江省]]省委书记。2016年[[越南共產黨第十二次全國代表大會|越共十二大]]後進入越共中央委員會<ref name="VN12"></ref>。

小儿子阮明哲（Nguyen Minh Triet）1990年出生，25岁当选越共[[平定省|平定省]]执行委员会委员，是该省55名省委干部中最年轻的一个<ref>[http://www.vnlook.com/htm/1235.html 越南最年轻省委书记 市委第一书记 平定省执行委员会委员 省委委员]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

女儿阮芳清(Nguyen Thanh Phuong)执掌着投资基金公司“越南资本管理”(Viet Capital Asset Management)和证券公司“越南资本证券”(Viet Capital Securities)。女婿亨利·阮（阮宝黄），[[美国|美国]]籍，是原[[南越|南越]][[阮文绍|阮文绍]]政权内阁副部长、1975年出逃美国的阮邦的儿子<ref>[http://www.szhgh.com/Article/news/world/201405/52440.html 揭秘越南买办总理阮晋勇：女婿是前南越政权高官之子]</ref>。

== 傳聞 ==
2006年有报道称，阮晉勇曾就讀於[[中華人民共和國|中国]][[广西师范大学|广西师范大学]]的育才校区<ref>{{cite news|author=费洪海|title=越南新总理阮晋勇毕业于中国“育才班”(附图)|url=http://news.xinhuanet.com/herald/2006-06/30/content_4771847.htm|accessdate=2010-08-05|newspaper=国际先驱导报|date=2006-06-30|deadurl=yes|archiveurl=https://web.archive.org/web/20140720200952/http://news.xinhuanet.com/herald/2006-06/30/content_4771847.htm|archivedate=2014-07-20}}</ref>。但此后广西师范大学有教师表示，该校从来没有一个叫做阮晋勇的学生，“校友”之说为误传<ref>{{cite news|author=王佴|title=寻访越南总理阮晋勇的中国母校 广西师大高材生 |url=http://news.sohu.com/20060630/n244014851.shtml|accessdate=2010-08-05|newspaper=第一财经日报|date=2006年6月30日}}</ref>。

2014年6月，有华文媒体传出阮晋勇因反对越共中央将[[2014年越南排华暴动|越南排华暴乱]]评价为“有关反共反社会主义的、以民族主义为借口发生的暴力行为”而遭到调查的消息，据传阮晋勇表示“不排除以个人身份与党外政治团体合作”<ref>{{cite web |url= http://m4.chinaiiss.com/html/20146/10/w5febb.html |title=反华先锋阮晋勇被查 越共清理门户讨好中国 |publisher=战略网 |accessdate=2014-06-14}}</ref>。其后中国大陸[[央视|央视]]亦引述了这一消息<ref>{{cite web |url= http://xm.ifeng.com/baoliao/detail_2014_06/12/2422183_0.shtml |title=央视曝光越南党内斗争激烈 总理阮晋勇被查 |publisher=凤凰网 |accessdate=2014-06-14}}</ref><ref>{{cite web |url= http://news.cntv.cn/2014/06/11/VIDE1402501147345370.shtml |title=《环球视线》 20140611我向联大发照会 越还将有何动作？ |publisher=CNTV |accessdate=2014-06-14 |date=2014-06-11}}</ref>。但此消息尚未得到越南官方证实。

2016年1月[[越南共产党第十二次全国代表大会|越南共产党第十二次全国代表大会]]举行前夕，据闻已经出任两届总理的阮晋勇谋求升任[[越共中央总书记|越共中央总书记]]，成为越南的[[党和国家最高领导人|党和国家最高领导人]]。不过越共内部的保守派劝服[[阮富仲|阮富仲]]超龄留任总书记，最终阮晋勇放弃竞逐，离开中央委员会和政治局，但其长子阮清谊成功当选越共第十二届中央委员。<ref name="japan news"></ref>
