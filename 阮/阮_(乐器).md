'''阮'''，又名'''阮咸'''、'''阮琴'''，[[中國|中國]]的[[撥弦樂器|撥弦樂器]]，得名於[[西晉|西晉]]“[[竹林七贤|竹林七贤]]”中的[[阮咸|阮咸]]。古代，此种乐器被称为'''秦琵琶'''，后因[[中國琵琶|曲项琵琶]]传入，占用了“琵琶”这一名称，才以擅长演奏这种乐器的阮咸为名。近代，[[阮咸|阮咸]]得到了许多改革，現代阮音色中正浑厚，依其形制不同而有高音阮、小阮、中阮、大阮、低音阮，甚至拉阮的出現，廣泛使用於獨奏及合奏。如同[[笙|笙]]在[[國樂團|國樂團]]吹管樂器中扮演的角色，阮以其出色的音响融合性和便于演奏[[和声|和声]]的长处，在國樂團彈撥樂器中占有一席之地。

== 沿革 ==
[[File:Reproduction_of_Ruan_and_Pipa_in_Tang_Dynasty_2011-07.jpg|缩略图]]時候的阮咸和[[中國琵琶|曲項琵琶]]。]]
據漢至魏晉時期的文史資料所載，[[汉武帝|漢武帝]]元鼎二年（西元前115年），[[張騫|張騫]]出使[[烏孫|烏孫國]]（今烏孜別克民族），烏孫王昆彌與漢通婚，[[烏孫公主|烏孫公主]]出嫁前，漢武帝命懂得音樂的工匠參考[[琴|琴]]、[[箏|箏]]、[[筑|筑]]、臥[[箜篌|箜篌]]等創制了一種能在馬上彈奏的樂器，有圓形音箱、直柄、十二柱、四弦，這種樂器便是阮，當時稱“'''秦琵琶'''”{{Efn|“琵琶”在中国古代是拨弦乐器的统称。}}（[[唐代|唐代]]杜佑的《通典》、《舊唐書》、《新唐書》中稱秦琵琶或'''秦漢子'''）。傳至[[西晉|西晉]]的時候，因“[[竹林七贤|竹林七贤]]”中的[[阮咸|阮咸]]擅彈此種琵琶而得名阮咸。約到了唐朝，西元350年前後由西域傳入中原的曲項琵琶大為流行，遂獨佔了“琵琶”这一名称；這類直柄圆形的琶琶则称阮咸。直到[[清代|清代]]，阮虽仍保持着古制，但音域只有1个多[[八度|八度]]，在民间流行的旧式阮，则退化为两条或三条弦。

1950年代以来，改良古阮咸为4根弦24品，並用鋼弦，并且形成阮乐器系列，包括高音阮、小阮、中阮、大阮和低音阮。[[香港中樂團|香港中樂團]]並以高音阮取代柳琴。由于阮的古曲谱均失传，所以没有传统乐曲。但是现代作曲家为演奏阮，尤其中阮创作了很多乐曲；1987年[[刘星_(音乐家)|劉星]]創作的《雲南回憶》中阮協奏曲，奠定了阮獨奏樂器的地位。

== 演奏方法 ==
[[File:Zhongruan.jpg|缩略图]]
阮咸一般为坐姿演奏，琴身置於大腿上且抱于胸前，左手按弦，右手持用尼龙等材料制成的拨片弹奏。在1980年代之前以假甲演奏为主流，但由于琴弦张力较大，逐渐被用拨片弹奏的方法替代。但是由于阮的复兴仍在起步阶段，所以各家的演奏方法均有差异，目前还未形式统一的演奏方法。

阮的演奏技巧，原來比較簡單。隨著樂器的改革，如今逐步得到豐富。經過演奏者的繼承、實踐、移植和創造，右手的彈法有彈、挑、勾、抹、扣、劃、輪、拂、分、搖、掃、滾等30多種， 左手指法有泛、打、勾、帶、滑、推、拉、吟、紋、揉等10多種。

== 拉阮 ==
[[File:La_ruan.png|缩略图]]
拉阮是因為國樂團中缺乏低音樂器而創造，實際上是一種仿效大提琴的形制的拉弦樂器，後因效果不理想，少有續用。
