'''阮文紹'''（{{vie|v='''Nguyễn Văn Thiệu'''}}；{{bd|1923年|4月5日|2001年|9月29日}}）[[越南共和國|越南共和國]]（南越）政治、軍事人物。[[越南共和國陸軍|越南共和國陸軍]]中將。1965年至1967年期間擔任[[國家領導委員會_(越南共和國)|越南共和國國家領導委員會]]主席。1967年至1975年期間擔任[[越南共和國總統|越南共和國總統]]。1975年辭職流亡[[臺灣|臺灣]]。後病逝於[[美國|美國]]。

== 早年经历 ==
1923年4月5日出生于[[宁顺省|宁顺省]][[潘郎市社|潘郎市社]]的小地主家庭。少年时代，曾加入了[[胡志明|胡志明]]领导的[[越盟|越盟]]，致力于使[[越南|越南]]脱离[[法国|法国]]的[[殖民|殖民]]统治。但当他发现越盟的目标是在越南建立[[共产主义|共产主义]]后，便于1946年脱离了。

在族中长辈的资助下，阮文绍于1947年进入海事学校学习，但因为时局变化，转入越南自由邦军官学校第1期，这所学校位于大叻，是由法国支持的[[保大皇帝|保大皇帝]]所建立的。1949年毕业后，他成为越南自由邦陆军中的一名[[少尉|少尉]]，与越盟[[游击队|游击队]]作战，阻擋越南被赤化。

== 军事生涯 ==
1954年，法国在越南的军事力量，因为[[奠边府战役|奠边府战役]]的失败而瓦解。而后，根据[[日内瓦会议|日内瓦会议]]的决议，越南实行南北分治。1955年，[[越南共和国|越南共和国]]（[[南越|南越]]）政府成立，将前越南自由邦陆军收编。此时，阮文绍已经是一名[[中校|中校]]了。而后，他历任陆军军官学校校长、步兵第1师师长、步兵第5师师长。并因在1963年参与推翻总统[[吳廷琰|吳廷琰]]的[[军事政变|军事政变]]而晋升为[[准将|准将]]。

在[[楊文明|楊文明]]、[[阮慶|阮慶]]相繼執政的两年中，阮文紹被任命为陆军[[越南共和國第4軍|第4軍]]军长、[[总参谋长|总参谋长]]，而后被提升为陆军[[中将|中将]]并担任[[国防部长|国防部长]]。

1965年6月，阮慶被迫下臺，阮文绍被军事强人们推选为军事革新会议的首领，并成为国家领导委员会主席，换言之，就是[[国家元首|国家元首]]。

== 政治生涯 ==
[[File:Nguyen_Van_Thieu_with_map_(cropped).jpg|thumb]]
1967年9月3日，他在竞选中赢得了38%的选票，成功当选新一任南越总统，同時也是各屆總統中任期最長的領導人之一。

阮文绍统治的贪腐程度被控远甚他的前任。和1963年军事政变的领袖[[阮高祺|阮高祺]]不同，阮文绍建立了一个政党，通过架空國會，使政治权力极大地集中于政府的行政分支。为了防备总统的统治受到威胁，他的同盟者被安插进重要的军政岗位。

在1971年，阮文绍参加了改选，但是他贪腐的恶名，使他的政治对手认为选举將会被操纵，从而拒绝参加。作为唯一的候选人，阮文绍顺利当选。

在1975年4月21日[[春祿戰役|春祿失守]]使得首都圈門戶洞開後，阮文紹宣布辞去總統職務並出國前往[[台灣|台灣]]，迅速将权力移交给副总统[[陈文香|陈文香]]，后者在1975年4月21日接任，也就是1975年4月30日南越向北越无条件投降的九天之前。
