'''梅尔·布鲁克斯'''（{{lang-en|Mel Brooks}}，{{bd|1926年|6月28日|||catIdx= B布鲁克斯}}）出生于[[美国|美国]][[纽约州|纽约州]][[纽约市|纽约市]][[布鲁克林区|布鲁克林区]]，是一个美国电影导演、编剧、作曲家、作词家、喜剧演员和制片人。布鲁克斯一生多次获得奥斯卡奖，托尼奖，艾美奖，劳伦斯·奥利弗奖以及其他奖项。<ref>[http://www.afi.com/tvevents/100years/laughs.aspx AFI's list]</ref>

== 生平 ==
1926年，布鲁克斯出生于[[美国|美国]][[纽约州|纽约州]][[纽约市|纽约市]][[布鲁克林区|布鲁克林区]]，父亲叫做马克西米利安·卡敏斯基，母亲叫做凯特。<ref>[http://www.filmreference.com/film/35/Mel-Brooks.html "Mel Brooks Biography (1926-)" at Filmreference.com]</ref> 
布鲁克斯他父亲的家庭是[[德国犹太人|德国犹太人]]，他们来自[[波罗的海|波罗的海]]港口丹岑（今[[波兰|波兰]][[格但斯克|格但斯克]]），他母亲的家族是[[乌克兰犹太人|乌克兰犹太人]]。<ref>[http://www.jewish-theatre.com/visitor/article_display.aspx?articleID=970 Mel Brooks is on a roll. As his hit revival of The Producers comes to London] {{webarchive|url=https://web.archive.org/web/20111130164324/http://www.jewish-theatre.com/visitor/article_display.aspx?articleID=970 |date=2011-11-30 }}</ref> 
布鲁克斯他有三个哥哥：欧文，莱尼和伯尼；当布鲁克斯才两岁时，他的父亲死于[[肾|肾]]病。<ref>{{cite web|url=http://www.brookslyn.com/print/Adelina1980/Adelina1980.php/kidney |title=A Mel Brooks Fan Site :: Print :: Adelina Magazine Feb 1980 |publisher=Brookslyn |date= |accessdate=2012-05-04}}</ref>
布鲁克斯曾聊起他父亲的死讯，说“那是一个愤怒。我可能是愤怒上帝，或者这世界。我确信我的许多喜剧是基于愤怒和敌意。生长在威廉斯堡，我学会了给它在喜剧备用自己问题——就像一拳打在脸上。”<ref name="Wakeman, John 1988">Wakeman, John. World Film Directors, Volume 2. The H. W. Wilson Company. 1988. 162-167.</ref>

儿时，布鲁克斯是一个小个子的、心理病态的男孩，他经常被他的同学们欺负。<ref>{{cite web|url=http://www.brookslyn.com/print/Adelina1980/Adelina1980.php |title=A Mel Brooks Fan Site :: Print :: Adelina Magazine Feb 1980 |publisher=Brookslyn |date= |accessdate=2012-05-04}}</ref>
当布鲁克斯14岁时，他就在[[威廉斯堡|威廉斯堡]]教巴迪·里奇如何打鼓来挣钱。<ref name="Wakeman, John 1988"/>
[[高中|高中]]后，他花了一年的时间在[[布鲁克林学院|布鲁克林学院]]学习[[心理学|心理学]]，以方便将来服兵役入伍。<ref name="Wakeman, John 1988"/>
后来他参加了隶属于[[弗吉尼亚军事学院|弗吉尼亚军事学院]][[陆军|陆军]]专业培训计划。<ref>{{cite web|url=http://www.vmi.edu/uploadedFiles/Archives/Records/World_War_II_ASTP/ASTPRoster.pdf |title=ASTP Program and Roster, World War II, VMI. |format=PDF |date= |accessdate=2012-05-04}}</ref> 
[[第二次世界大战|第二次世界大战]]，布鲁克斯参加了援助[[法国|法国]]的战争。<ref>[http://www.military.com/veterans-day/these-guys-served.html These Guys Served?! Who Knew?]</ref>

== 个人生活 ==
1964年，[[安妮·班克罗夫特|安妮·班克罗夫特]]和梅尔·布鲁克斯结婚，1972年，两人育有儿子马克思。<ref name=" wolf">{{cite web|url= http://bjyouth.ynet.com/article.jsp?oid=5522371|title= 奥斯卡影后安妮·班克罗夫特去世享年73岁|accessdate= 2005-06-09|publisher= 北青网|work= |language= zh-cn|deadurl= yes|archiveurl= https://archive.is/20130706021236/http://bjyouth.ynet.com/article.jsp?oid=5522371|archivedate= 2013-07-06}}</ref><ref name=" bear" >{{cite web |url=http://www.cctv.com/news/entertainment/20050608/101952.shtml |title= 美国著名女影星安妮·班克罗夫特逝世 |accessdate= 2005年6月8日|publisher=CCTV新闻频道 |work= |language =zh-cn}}</ref>
两人的相识也极具有传奇色彩，他们是在一次音乐剧的彩排上认识的，班克罗夫特后来回忆说：“那时候我就在想，我遇到了真命天子。我从来没有和一个人相处得那么愉快，我也希望他喜欢我。”<ref name="rat " >{{cite web |url=http://ent.sina.com.cn/r/m/2005-06-09/1133747194.html |title=东方早报：再见，罗宾逊太太 |accessdate=2005年6月9日 |publisher= 新浪网|work= |language =zh-cn}}</ref>
