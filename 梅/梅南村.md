梅南村是中華民國臺灣[[嘉義縣|嘉義縣]][[梅山鄉|梅山鄉]]轄下的行政區。位於[[嘉義縣|嘉義縣]]東北方，北鄰[[梅山鄉|梅山鄉]][[梅北村|梅北村]]，西鄰[[梅山鄉|梅山鄉]][[過山村|過山村]]，南鄰[[梅山鄉|梅山鄉]][[永興村|永興村]]，東鄰[[梅山鄉|梅山鄉]][[梅東村|梅東村]]。
面積約0.896998平方公里，行政區包含15個鄰、403戶、人口數1034人。梅山公園、橫山鎮安宮、大將宮為該村景點。

==地理==
[[嘉義縣|嘉義縣]][[梅山鄉|梅山鄉]]的一個行政區域，與[[梅北村|梅北村]]、[[梅東村|梅東村]]為[[梅山鄉|梅山鄉]]的鄉鎮中心。<ref name="cite1">{{cite web |title=地名資訊服務網 |url=http://gn.moi.gov.tw/geonames/index.aspx}}</ref>

==聚落歷史==
目前區域由北側補崙及南側鼎敢閂兩個聚落組成。<ref name="cite1">{{cite web |title=地名資訊服務網 |url=http://gn.moi.gov.tw/geonames/index.aspx}}</ref>

==教育==
本區域內無設立國中小，國小學區為梅山國小；國中學區為梅山國中。<ref>{{cite web |title=嘉義縣教育處─國小學區劃分表、國中學區劃分表 |url=http://edu.cyhg.gov.tw/News4.aspx?n=8B04CBD4471A6D45&sms=745127AC0E5C0036}}</ref>
梅山國小創立於1907年，早期稱為「梅仔坑公學校」。<ref>{{cite web |title=嘉義縣政府全球資訊網 |url=https://www.cyhg.gov.tw/News_Content.aspx?n=20C1A3DAF6A74FCE&s=45BC60B3A1212036}}</ref>
梅山國中籌設於1956年，建成於1956年，初名為「嘉義縣立大林中學梅山分部」。1960年與大林中學拆分，獨立為「嘉義縣立梅山初級中學」。<ref>{{cite web |title=梅山國中 |url=http://www.msjh.cyc.edu.tw/modules/tadnews/page.php?nsn=4}}</ref>

==景點==
===梅山公園===
位於嘉義縣梅山鄉，梅山公園內有「草嶺兵工殉難紀念碑」、「介壽亭」、「現代文學步道」、「樹屋平台」、「天空廊道」等景。「草嶺兵工殉難紀念碑」為紀念七十四位建設草嶺潭因天災殉職的軍人而立的紀念碑。「介壽亭」為梅山公園的至高點。「現代文學步道」於園區設立共30做景石，加入32位作家詩作所建成。「樹屋平台」樹屋上下設有相同號碼的喇叭，號碼相同可進行通話。「天空廊道」兩旁，栽有老梅樹。<ref>{{cite web |title=嘉義縣文化觀光局-文化觀光資源 |url=https://www.tbocc.gov.tw/Default.aspx?lang=tw}}</ref>
