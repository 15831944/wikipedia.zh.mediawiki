《'''梅花三弄'''》，又名《'''梅花引'''》、《'''梅花曲'''》、《'''玉妃引'''》，是一首[[古琴曲|古琴曲]]，採用[[正調|正調]]定弦。樂曲通過梅花的潔白芬芳和耐寒等特徵，借物抒懷，歌頌具有高尚節操的人。全曲共有10個段落，因為主題在琴的不同[[徽位|徽位]]的[[泛音|泛音]]上彈奏三次（上准、中准、下准三個部位演奏），故稱「三弄」。

这首乐曲的一部分经过改编后，自2004年起成为了中央人民广播电台中国之声的ID音乐。

==歷史==
現存最早的琴曲樂譜見於《[[神奇秘譜|神奇秘譜]]》（1425年），据其記載此曲原為是[[晉朝|晉朝]][[王子猷|王子猷]]向[[桓伊|桓伊]]討得的一首笛曲（一说为箫曲），後來改編為[[古琴|古琴]]曲，但實際笛曲曲譜已不復存在。《[[太古遺音|太古遺音]]》沿襲此説法，而1609年[[楊倫|楊倫]]的《[[伯牙心法|伯牙心法]]》中則載“按是曲為[[顏師古|顏師古]]所作也”，而顏師古是唐朝琴師，或許是這一時期笛曲才被改編成琴曲的，同時書中評價道此曲為“梅為花之最清，琴為聲之最清。以最清之聲寫最清之物”。<ref name="戴曉蓮"/>

現存最早的梅花三弄錄音是[[今虞琴社|今虞琴社]]在1940年代錄製的45轉“琴瑟簫”合奏版，1955年[[中國唱片总公司|中國唱片总公司]]出版慢轉的琴簫合奏版，迄今已有十多種唱片出版。<ref name="戴曉蓮"/>

==版本==
梅花三弄現存的琴譜有四十多種，其中流傳最廣的包括：<ref name="戴曉蓮">[[戴曉蓮|戴曉蓮]]，[http://www.cnki.com.cn/Article/CJFDTOTAL-ZYYX199904009.htm 琴曲《梅花三弄》研究]，中國音樂學，1999年第4期</ref>

* [[吴景略|吴景略]]《[[琴譜諧聲|琴譜諧聲]]》（1820年刻本）的琴簫版本，其技巧加花較多，風格洒脫，節奏規整。
* [[張子謙|張子謙]]《蕉庵琴谱》（1868年刊本），節奏較跌宕自由，被稱為《老梅花》。
* [[溥雪齋|溥雪齋]]的版本其節奏規整，風格清麗，被稱為《新梅花》。
* [[卫仲乐|卫仲乐]]用譜由《老梅花》的谱所打，节奏紧凑。

此外现在還有作曲家[[王建中_(作曲家)|王建中]]于1973年依吴景略提供古琴曲改寫的[[钢琴|钢琴]]曲《梅花三弄》，<ref>徐偉，[http://www.cnki.com.cn/Article/CJFDTotal-DWUT201403058.htm 钢琴曲《梅花三弄》的艺术赏析]， 《大舞台》 2014年03期 </ref> 以及[[陸春齡|陸春齡]]改編的[[笛子|笛子]]獨奏曲和[[彭修文|彭修文]]編寫的[[編鐘|編鐘]]協奏曲。

==結構==

全曲共10段，每段都有小標題，不同琴譜名稱有出入。有琴譜將主題出現的3個段落稱為「一弄叫月」、「二弄穿雲」、「三弄橫江」。

明代浙派《[[梧岡琴谱|梧岡琴谱]]》的標題：

# 溪山夜月（引子，以[[散音|散音]]演奏）
# 聲入太霞（主題，以[[泛音|泛音]]演奏，左手放在下准，10徽、12徽）
# 青鳥啼魂（插段，以[[按音|按音]]演奏）
# 玉笛聲（主題，以泛音演奏，左手放在中准，7徽、9徽）
# 玉簫聲（插段2，以按音演奏）
# 讀書聲
# 凌風嘎玉（主題，以泛音演奏，左手放在上准，4徽、5徽）
# 鐵笛聲（插段3，以按音演奏）
# 風蕩梅花
# 欲罷不能（尾聲）
