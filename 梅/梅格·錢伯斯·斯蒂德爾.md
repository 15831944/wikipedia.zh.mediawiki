'''梅格·錢伯斯·斯蒂德爾'''（'''Meg Chambers Steedle'''）是一名美國女演員。她在[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》第三季中飾演主要配角Billie Kent。<ref>{{Cite web |url = http://www.imdb.com/name/nm3499572/ |title = Meg Steedle: Filmography |publisher = IMDb |language = en}}</ref>在該劇之前，她經常在劇場演出，包括《{{link-en|鬥氣夫妻 (舞台劇)|Barefoot in the Park|鬥氣夫妻}}》<ref>{{Cite web |url = http://www.nytimes.com/2010/02/21/nyregion/21playnj.html |title = Simon's Dueling Newlyweds Return |author = Naomi Siegel |date = 2010-02-19 |publisher = The New York Times |language = en}}</ref>和《''The Heir Apparent''》。錢伯斯·斯蒂德爾在NBC的《[[蘿拉的秘密|蘿拉的秘密]]》第一季中飾演偵探Francesca "Frankie" Pulaski。
