'''梅貽琦'''（{{bd|1889年|12月29日|1962年|5月19日|catIdx=Mei梅貽琦}}），字'''月涵'''，[[天津市|天津市]]人，中国物理学家和教育家。毕业于[[南开中学|南开中学]]、[[國立清華大學|清華學校]]。曾任[[中華民國教育部|中華民國教育部]]部長、[[中央研究院|中央研究院]]院士和[[清华大学|清华大学]]校长。祖籍[[江蘇|江蘇]][[武進|武進]]，祖先於[[明成祖|明成祖]]時由[[江南|江南]]遷居[[北京|北京]]，後於天津落籍。熟讀[[史書|史書]]，喜愛[[科學|科學]]。初起研究[[電機工程|電機工程]]，後轉為專攻[[物理學|物理]]。

== 經歷 ==
1904年，15岁的梅貽琦初入天津南开学堂（其前身是由严范孙创办的“严氏家塾”，当时称“私立中学堂”，后又曾改称“敬业中学堂”），成为著名教育家[[张伯苓|张伯苓]]的学生。

1908年入保定直隶高等學堂。

1909年8月在北京史家胡同游美学务处临时办公处考取[[庚子賠款|庚子賠款]][[獎學金|獎學金]]留學生（首批“直接留美生”）。在630名考生中，考得第六名（总分834分，平均分75.11）。

1911年入[[美國|美國]][[伍斯特理工学院|伍斯特理工学院]]研究[[電機工程|電機工程]]，1914年夏毕业，获工学士学位，并获美国{{tsl|en|Sigma Xi|Sigma Xi荣誉学会}} 嘉奖（美国一种专为奖励优秀大学生的制度，获奖者由个大学推荐，奖品为一金质钥匙，故称“金钥奖”）。 

1914年回國在[[天津|天津]]青年會工作，1915年到[[清華學校|清華學校]]執教英文、幾何；1916年即擔任清華大學物理[[教授|教授]]。

1931年起擔任清華大學[[校長|校長]]。對日[[中國抗日戰爭|八年抗戰]]期間，清華、[[北京大学|北大]]、[[南开大学|南開]]合組[[國立西南聯合大學|國立西南聯合大學]]，以校務委員會常委身份主持校務。梅貽琦先生是清華大學（含[[北京清華大學|北京]]、[[國立清華大學|新竹]]）歷史上任期最長的校長（1931年 - 1948年、1955年 - 1962年），清華人尊稱為「永遠的校長」。

国民党六大召开前夕，列入朱家骅与陈立夫联名向蒋介石推荐的98名“最优秀教授党员”之一<ref>[http://news.ifeng.com/history/zhongguojindaishi/detail_2013_04/16/24258575_0.shtml 45年国民党“最优秀教授党员”：华罗庚陈寅恪冯友兰]</ref>。

1948年12月29日，梅貽琦堅辭[[教育部|教育部]]長，[[陳雪屏|陳雪屏]]代理教育部務<ref name="民國史大事記">李新 總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室 編，韓信夫、姜克夫 主編：《中華民國史大事記》（全十二卷，共十二冊），[[北京|北京]]：[[中華書局|中華書局]]，2011年7月</ref>{{rp|8762}}。1953年，任[[中華民國教育部|教育部]][[在美文化事業顧問委員會|在美文化事業顧問委員會]]主任委員。

1955年奉召返[[臺灣|臺]]，在今臺灣[[新竹市|新竹市]][[赤土崎|赤土崎]]将[[國立清華大學|清華大學]]復校，並籌辦清華[[原子科學|原子科學]]研究所（参[[國立清華大學工程與系統科學系|國立清華大學工程與系統科學系]]）。

1958年7月任教育部部長，兼清華大學校長。

1959年兼任中華民國國家長期發展科學委員會副主席。

1960年5月患病入[[台大醫院|台大醫院]]療養。 1961年2月奉准辭教育部長，仍兼[[原子能委員會|原子能委員會]]主任委員。

1962年2月當選中華民國[[中央研究院|中央研究院]]第4屆(數理科學組)院士<ref>[https://academicians.sinica.edu.tw/index.php?func=1-D 梅貽琦 逝世院士一覽表] 中央研究院</ref>。

1962年5月擔任[[國立中央大學|國立中央大學]]地球物理研究所籌備委員會主任委員。同月19日上午11時病逝台大醫院。23日举行追思會公祭，[[蔣中正|蔣中正]][[總統|總統]]特頒「勳昭作育」輓額并派[[張群|張群]]至祭，[[陳誠|陳誠]][[副總統|副總統]]亦頒輓聯：崇樸學以黜浮華，實大聲宏，盛績久為文苑重；樹良規而垂教澤，薪傳火盡，道徽猶繫國人思。妻子韓詠華，天津人，為[[津門八大家|津門八大家]]之一韓家之族裔，1919年與梅先生結婚，育有一子四女。<ref>{{cite web |url = http://www.tsinghua.edu.cn/publish/news/4208/2011/20110225231447703706893/20110225231447703706893_.html |title = 清华“永远的校长”梅贻琦：寡言 嗜酒 真君子 |deadurl = yes |archiveurl = https://web.archive.org/web/20131026080349/http://www.tsinghua.edu.cn/publish/news/4208/2011/20110225231447703706893/20110225231447703706893_.html |archivedate = 2013-10-26 }}</ref>

==兄弟及其成就==
其四弟[[梅貽寶|梅貽寶]]曾先後任職[[燕京大學|燕京大學]]等院校，並於1970-73年間出任[[香港中文大學|香港中文大學]][[新亞書院|新亞書院]]校長。

==為人作風==
梅貽琦個性沉靜，寡言、慎言，他的學生曾作打油詩：“大概或者也許是，不過我們不敢說，可是學校總認為，恐怕仿佛不見得。”[[葉公超|葉公超]]用“慢、穩、剛”三個字形容他。“身教重於言教”及“所謂大學者，非謂有大樓之謂也，有大師之謂也”的教育名言深為世人推崇。梅校長對清華大學的成長貢獻巨大，受到所有清華人的崇敬。

== 紀念 ==
民國52年（西元1963年），梅貽琦葬於國立清華大學校園，其墓地名為「[[梅園|梅園]]」<ref>{{cite web |url = http://archives.lib.nthu.edu.tw/history/doc/4/4-03-05-04.pdf |title=余火炎先生、余河益先生訪談稿 |accessdate=2011-11-04  |deadurl=yes |archiveurl=https://web.archive.org/web/20160305050050/http://archives.lib.nthu.edu.tw/history/doc/4/4-03-05-04.pdf |archivedate=2016-03-05 | work=國立清華大學圖書館 |date=2010-07-29}}</ref>。

[[民國58年|民國58年]]（西元1969年）[[國立清華大學|國立清華大學]]與[[國立交通大學|國立交通大學]]共同舉行的「[[梅竹賽|梅竹賽]]」，紀念兩校梅貽琦前校長（1962年過世）與交大前校長[[淩鴻勛|凌竹銘]]先生（當時任[[中國石油公司|中國石油公司]]董事長）。

梅貽琦受到清華師生校友很高的評價：「提到梅貽琦就意味著清華」，「梅貽琦是清華'''永遠的校長'''」。
