'''梅貝爾'''（{{lang-en|Maybell}}）是位於[[美國|美國]][[科羅拉多州|科羅拉多州]][[莫弗特縣_(科羅拉多州)|莫弗特縣]]的一個[[人口普查指定地區|人口普查指定地區]]。

== 地理 ==
梅貝爾的座標為{{coord|40|31|02|N|108|05|10|W|type:city_region:US-AL_source:GNS-enwiki|display=inline}}<ref name=GNIS />，而該地最高點為海拔高度1805米（即5922英尺）。

== 人口 ==
根據[[2010年美國人口普查|2010年美國人口普查]]的數據，梅貝爾的面積為1.30平方千米，當中陸地面積為1.30平方千米，而水域面積為0.00平方千米。當地共有人口72人，而人口密度為每平方千米{{formatnum: {{rounddown|{{#expr: 72 / 1.30}}}} }}人。<ref name="GR8">{{cite web|url=http://factfinder2.census.gov|publisher=[[美国人口调查局|美国人口调查局]]|accessdate=2011-05-14|title=American FactFinder}}</ref>
