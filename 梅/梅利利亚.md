'''梅利利亚'''（{{lang-es|'''Melilla'''}}，发音：[meˈliʎa]）是[[西班牙|西班牙]]兩個[[自治市|自治市]]之一，為西班牙位於[[北非|北非]]的海外[[属地|属地]]，它位于[[地中海|地中海]]沿岸和[[马格里布|马格-{里}-布]]的最北部。

== 行政規劃 ==
在1995年3月14日自治法-{zh-hans:出台; zh-hant:訂立;}-之前，它作为[[马拉加省|马拉加省]]的一部分被管辖，它在西班牙加入[[欧盟|欧盟]]前是一个[[自由港|自由港]]；它最主要的产业是[[渔业|渔业]]。其它收来源有跨国界商业贸易（合法或走私的）、西班牙和欧盟赠予的基金。据1994年统计，它的人口为63,670，人口由[[基督教|基督教]]徒、[[穆斯林|穆斯林]]、[[犹太教|犹太人]]和少部分[[印度教|印度教]]徒组成。

== 領土爭端 ==
在从[[法国|法国]]和[[西班牙|西班牙]]的统治下独立之后，[[摩洛哥|摩洛哥]]曾多次宣布对梅利利亚、[[休达|休达]]、[[加那利群岛|加那利群岛]]和许多属西班牙的位于[[非洲|非洲]]海岸的小岛的主权。为此，西班牙政府声明休达和梅利利亚是西班牙的重要组成部分，也说明了[[直布罗陀|直布罗陀]]海峽的主权应属于自己，而不是属于[[英国|英国]]。

==气候==
梅利利亚属于[[地中海气候|地中海气候]]，由于靠近海洋，相比非洲内陆地区夏天更凉爽，降雨量也偏多。梅利利亚的气候同欧洲大陆的西班牙南部海岸地区有很大相同点，不同季节的温差非常小。

== 移民問題 ==
由於地理位置特殊，梅利利亞可以作為一道非洲難民逃往歐洲的跳板。但實際上，他們要進入作為歐盟一部分的（西屬）梅利利亞是相當困難，因為邊境被六米高的柵欄及瞭望台重重封鎖，亦有許多難民被西班牙警察逮捕，以非法入境罪名遣返其祖國。西班牙警方以鐵絲網、催淚氣體噴射裝置、雷達及二十四小時閉路監系統來加強梅利利亞邊防保安。2005年10月，有超过700名[[西撒哈拉|西撒哈拉]]移民试图从边境的摩洛哥一侧进入到西班牙领地。其中许多人都被摩洛哥[[宪兵|宪兵]]从背后击中。[[国际特赦组织|国际特赦组织]]和[[无国界医师|无国界医师]]曾经谴责摩洛哥政府在撒哈拉沙漠中滞押超过500名难民并使其处于饥饿缺水状态的行为。
