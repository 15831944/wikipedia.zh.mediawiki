梅爾文·厄普顿（Melvin Emanuel Upton, Jr）出生於(1984.8.21)是一位[[美國職棒大聯盟|美國職棒大聯盟]]的選手，於1984年8月21日出生於[[美國|美國]][[維吉尼亞州|維吉尼亞州]]的[[諾福克|諾福克]]，現正效力於[[美國聯盟|美國聯盟]]的[[亞特蘭大勇士|亞特蘭大勇士]]隊，守備位置為[[中外野手|中外野手]]，原登陸名('''B. J. Upton''')在勇士隊2年的低潮被交易到[[聖地牙哥教士|聖地牙哥教士]]隊前決定改登錄回原本名字。

他的暱稱 B.J.是"Bossman Junior"的縮寫—而他的父親的暱稱則是"Bossman"，其弟[[賈斯汀·厄普頓|賈斯汀·厄普頓]]目前效力於[[美國職棒大聯盟|美國職棒大聯盟]][[美國聯盟|美國聯盟]]的[[洛杉磯天使|洛杉磯天使]]隊。

2009年10月2日於光芒隊主場對戰[[紐約洋基|紐約洋基]]隊的比賽中，B.J.以四個打數完成了[[完全打擊|完全打擊]]，成為光芒隊隊史中第一個出現完全打擊的選手<ref> {{cite web |url=http://tampabay.rays.mlb.com/news/article.jsp?ymd=20091002&content_id=7288010&vkey=news_tb&fext=.jsp&c_id=tb |title=存档副本 |accessdate=2014-11-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20130617164508/http://tampabay.rays.mlb.com/news/article.jsp?ymd=20091002&content_id=7288010&vkey=news_tb&fext=.jsp&c_id=tb |archivedate=2013-06-17 }} </ref>。
有位弟弟賈斯汀·厄普頓也是大聯盟球員，兩人在勇士隊及教士隊皆同隊。

== 球員特點 ==
厄普顿是一位速度快的跑者,在08~09球季共跑出86次盜壘成功,但近年打擊率,保送和長打率持續下滑,在09年擔任第一棒表現不理想,間接造成光芒成績下滑,厄普顿原本是內野手,由於新隊友的加入而改守中外野,才發揮他的速度優勢,能夠防守大範圍,傳球臂力強且時常有美技演出,他經常守得很淺,因為他可以急速倒退並精準判斷球向
