'''梅林站'''（{{jpn|j=梅林駅|hg=うめばやしえき|rm=Umebayashi eki|f=y}}）是一個位於[[日本|日本]][[福岡縣|福岡縣]][[福岡市|福岡市]][[城南區_(日本)|城南區]]梅林四丁目<ref name="ud135">『公共交通機関のユニバーサルデザイン』、135頁。</ref>，屬於[[福岡市地下鐵|福岡市地下鐵]][[七隈線|七隈線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是N05。

== 歷史 ==
*2005年（平成17年）2月3日 - 開業。

== 車站結構 ==
[[島式月台|島式月台]]1面2線的[[地底車站|地底車站]]。

== 使用情況 ==
2016年（平成28年）度的1日平均'''上車'''人次為'''1,508人'''<ref>{{PDFlink|[https://subway.city.fukuoka.lg.jp/subway/about/pdf/subway_05.pdf 駅別乗車人員の推移（年度別1日平均）]}} - 福岡市交通局</ref>。開業以後一直有增加傾向，但是作為七隈線車站，卻是[[福岡市地下鐵|福岡市地下鐵]]車站當中最少。
