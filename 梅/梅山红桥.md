'''梅山红桥'''，原名为春晓大桥、梅山春晓大桥，是一座位于[[浙江省|浙江省]][[宁波市|宁波市]][[北仑区|北仑区]]的跨海大桥。该桥跨越梅山水道，西起春晓[[洋沙山|洋沙山]]，终于[[梅山岛|梅山岛]]，全长约1.971公里，总投资11.7亿元，是梅山岛和[[梅山保税港区|梅山保税港区]]对外联系的重要通道，也是继续[[梅山大桥|梅山大桥]]后梅山岛与陆地相连接的第二座跨海大桥<ref name="construction2">{{Cite web |url=http://zjnews.zjol.com.cn/system/2016/04/26/021126050.shtml |title=春晓大桥成功合龙 建成后宁波到梅山只需40分钟 |accessdate=2018-07-19 |author=甬派 |date=2016-04-26 |publisher=浙江在线 |archiveurl=https://web.archive.org/web/20180719100103/http://zjnews.zjol.com.cn/system/2016/04/26/021126050.shtml |archivedate=2018-07-19 |deadurl=no }}</ref>，大桥专门用于客运车辆通行。

大桥于2014年1月3日开工<ref name="construction1">{{Cite web |url=http://news.163.com/14/0527/13/9T8LSA0C00014AEE.html |title=梅山春晓大桥建设正酣 目前正准备主墩钻孔桩施工 |accessdate=2018-07-19 |author=张跃 谢挺 |date=2014-05-27 |publisher=中国宁波网 |archiveurl=https://web.archive.org/web/20180719100102/http://news.163.com/14/0527/13/9T8LSA0C00014AEE.html |archivedate=2018-07-19 |deadurl=no }}</ref>，2016年4月26日主拱合龙<ref name="construction2"/>，2017年9月29日建成通车<ref name="construction3">{{Cite web |url=http://blnews.cnnb.com.cn/system/2017/09/28/011607095.shtml |title=梅山红桥正式通车 |accessdate=2018-07-19 |author=林萍 |date=2017-09-28 |publisher=北仑新闻网 |archiveurl=https://web.archive.org/web/20180719100104/http://blnews.cnnb.com.cn/system/2017/09/28/011607095.shtml |archivedate=2018-07-19 |deadurl=no }}</ref>。
==历史==

==地理位置==
大桥两端连接位于宁波梅山岛的梅山保税港区和北仑区[[春晓镇|春晓镇]]。根据规划，大桥起自春晓镇东八路，接洋沙山东六路，终于梅山港湾路，接规划盐湖路<ref name="design1"/>，是梅山岛五条跨梅山港交通通道中最南端的一条、两条城市客运通道之一<ref name="plan1"/><ref name="plan2"/>。大桥对接北仑城区太河路，经太河路可直接从大桥驶入梅山岛，使得宁波主城区到梅山的交通时间缩短为40分钟，进一步便利化了梅山保税港区的对外交通<ref name="construction2"/>。

==结构与设计==
梅山红桥及其连接线全长约1.971公里，大桥主跨长336米，采用钢桁拱桥结构，大桥建成时为世界最大跨海钢桁拱桥<ref name="construction2"/>。

大桥按功能分为上、下两层，上层桥面为主线，宽25米，采用一级公路标准按双向六车道设计，设计速度60公里/小时；下层桥服务慢行休闲系统，宽12米，满足行人及非机动车辆通行；外侧辅道采用城市次干道标准，设计速度40公里/小时<ref name="construction3"/><ref name="design1"/>，路基断面分为西岸引桥、主桥、东岸引桥、连接线四种形式<ref name="design2"/>。桥梁通航标准按双层桥分别设计，上层按500吨级海轮通航净空标准设计；下层按游艇通航净空标准设计。主桥中部108米范围内设置下层纵移桥架，通过向两侧伸缩的方式实现开启与闭合，纵移打开后可满足500吨级轮船通航要求，平时下层桥面闭合状态时梁底约300m范围满足游艇通航净空高度<ref name="construction3"/><ref name="design2"/>。108米的下层纵移式开启桥跨度在大桥建成时为世界第一<ref name="construction2"/>。
