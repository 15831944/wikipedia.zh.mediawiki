'''次氯酸'''（{{Lang-en|hypochlorous acid}}）是一种化学式为[[氢|H]][[氯|Cl]][[氧|O]]的不稳定[[弱酸|弱酸]]，仅能存在于溶液中，一般用作[[漂白剂|漂白剂]]、[[氧化剂|氧化剂]]、[[除臭剂|除臭剂]]和[[消毒剂|消毒剂]]。

== 製取 ==

当纯净的[[氯气|氯气]]通入[[水|水]]中时，会形成[[盐酸|盐酸]]和次氯酸：

:Cl<sub>2</sub> + H<sub>2</sub>O ↔ HCl + HClO　　K<sub>W</sub>=1.56×10<sup>-4</sup>

[[一氧化二氯|一氧化二氯]]溶於水會轉變為次氯酸：

:Cl<sub>2</sub>O + H<sub>2</sub>O → 2HClO

== 化學性質 ==
在[[水|水]]溶液中，次氯酸部分电离为次氯酸根离子（ClO<sup>−</sup>，也称为“次氯酸盐阴离子”）和[[氢离子|氢离子]]（H<sup>+</sup>）：

:HClO {{eqm}} H<sup>+</sup> + ClO<sup>−</sup>

次氯酸中性溶液是一種比[[標準狀況|標準狀況]]下的氯更強的氧化劑：

:2 HOCl(aq) + 2 H<sup>+</sup> + 2 e<sup>−</sup> {{eqm}} Cl<sub>2</sub>(g) + 2 H<sub>2</sub>O　　E=+1.63V
注：标准状况指0℃，标准大气压
次氯酸不穩定，會慢慢發生[[自身氧化還原反應|自身氧化還原反應]]而分解，降低[[pH|pH]]會促進該反應的進行：
: <math>\rm 2HClO {=\!=\!=\!=} 2HCl + O_2 \uparrow </math>
: <math>\rm 3HClO {=\!=\!=\!=} 2HCl + HClO_3 </math>

含有次氯酸根的[[盐|盐]]被称为[[次氯酸盐|次氯酸盐]]。最广为人知的一种家用次氯酸盐消毒剂是[[次氯酸钠|次氯酸钠]]（NaClO）。

在[[生物学|生物学]]中，次氯酸被[[嗜中性白细胞|嗜中性白细胞]]（Neutrophil）用来杀灭[[细菌|细菌]]，也被广泛用于[[游泳池|游泳池]]的含氯消毒剂产品。
