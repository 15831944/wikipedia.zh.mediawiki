'''翻译'''（{{Lang-en|Translation}}；轉譯），又称'''转译'''，是[[蛋白質生物合成|蛋白質生物合成]]（[[基因表达|基因表达]]中的一部分，基因表达还包括[[转录|转录]]）过程中的第一步。翻译是根據[[遺傳密碼|遺傳密碼]]的[[中心法則|中心法則]]，将成熟的[[信使RNA|信使RNA]]分子（由{{lang|en|DNA}}通过[[转录|转录]]而生成）中「[[碱基|碱基]]的排列顺序」（核苷酸序列）解码，并生成对应的特定[[氨基酸|氨基酸]]序列的过程。但也有许多转录生成的[[RNA|RNA]]，如[[转运RNA|转运RNA]]、[[核糖体RNA|核糖体RNA]]和[[小核RNA|小核RNA]]等并不被翻译为氨基酸序列。

翻译的過程大致可分作三個階段：起始、延長、终止。翻译主要在[[细胞质|细胞质]]内的[[核糖体|核糖体]]中进行，氨基酸分子通过转运{{lang|en|RNA}}被带到核醣體上。生成的多肽链（即氨基酸链）需要通过正确[[蛋白质折叠|折叠]]形成蛋白质，许多蛋白质在翻译结束后还需要进行[[翻译后修饰|翻译后修饰]]才能具有真正的生物学活性。

在[[原核生物|原核生物]]的蛋白質合成中，通常可以使用某些[[抗生素|抗生素]]（如[[茴香素|茴香素]]、[[放线菌酮|放线菌酮]]、[[氯霉素|氯霉素]]、[[四環黴素|四環黴素]]）来抑制或阻断翻译的进行；其基本原理是[[酶#競爭性抑制作用|競爭性抑制作用]]或是[[共價|共價]]结合而佔據了核糖体的[[活性位点|活性位点]]。由于原核生物的核糖体结构与真核生物中的不同，这些抗生素可以特异性消灭感染真核宿主的原核生物而不会对宿主造成影响。

== 分子機制 ==

{{lang|en|mRNA}}的遗传信息是來自於{{lang|en|DNA}}，經由[[核糖体|核糖体]]被各种[[tRNA|tRNA]]所识别。{{lang|en|tRNA}}可以识别{{lang|en|mRNA}}上以三个[[核苷酸|核苷酸]]为代码的[[密码子|密码子]]，与它们相配的{{lang|en|tRNA}}上的三个核苷酸被称为[[反密码子|反密码子]]。带有特定反密码子的{{lang|en|tRNA}}携带特定的氨基酸。因此通过翻译机制，{{lang|en|mRNA}}上的密码子就可以被“翻译”为对应的氨基酸。

[[氨酰tRNA合成酶|氨酰tRNA合成酶]]是催化将与{{lang|en|tRNA}}相连的氨基酸连到一起组成蛋白质的[[酶|酶]]。

== 原核翻译 ==
{{main|原核翻译}}
原核生物没有[[细胞核|细胞核]]，因此它们的{{lang|en|mRNA}}在转录的同时就可以被翻译。假如在翻译时有多个核糖体同时工作的话，那么蛋白质的组成部分可以比较快地建成和连接到一起。

=== 起始 ===
轉译开始时，核糖体的一个小单位与{{lang|en|mRNA}}的起始密码子结合，而細胞中的氨基酸被激活，和{{lang|en|tRNA}}結合，而{{lang|en|tRNA}}將氨基酸帶至核糖體。{{lang|en|mRNA}}的起始密码子标志着{{lang|en|mRNA}}上蛋白质的信息的开始位置。一般起始密码子的顺序是{{lang|en|AUG}}，不过在原核生物中有不少其它的码。[[细菌|细菌]]的蛋白质以一个改变了的[[N-甲酰甲硫氨酸|''N''-甲酰甲硫氨酸]]（{{lang|en|f-Met}}）开始。在甲酰甲硫氨酰中，[[胺基|胺基]]被一个甲酰代替而形成了一个[[酰胺|酰胺]]，这个改变使得这个码无法与一个氨基酸相结合，但这不是问题，因为这个码标志着一个蛋白质的开始。在原核细胞中{{lang|en|mRNA}}与核糖体的结合由一个被称为[[夏因-达尔加诺序列|夏因-达尔加诺序列]]的基组导入，这个序列一般位于开始位置前8到13个核苷酸的地方。

=== 延長 ===
一个激活的{{lang|en|tRNA}}进入核糖体的A位，{{lang|en|tRNA}}上的反密碼子和{{lang|en|mRNA}}上的密碼子進行互補鹼基配對，{{lang|en|DNA}}上的胸腺嘧啶(T)和{{lang|en|mRNA}}上的腺嘌呤(A)配對;DNA上的腺嘌呤(A)和{{lang|en|mRNA}}上的尿嘧啶(U)配對;而胞嘧啶(C)和鳥嘌呤(G)互相配對。{{lang|en|DNA}}上的編碼氨基酸。肽酰转移酶在邻近的氨基酸间建立一个肽键，此后在P位上的氨基酸离开它的{{lang|en|tRNA}}与A位上的{{lang|en|tRNA}}结合，核糖体则相对于{{lang|en|mRNA}}向前滑动，原来在A位上的{{lang|en|tRNA}}移动到P位上，原来在P位上的空的{{lang|en|tRNA}}移动到E位上，然后在下一个{{lang|en|tRNA}}进入A位之前被释放，將氨基酸遂一加上。这个过程被称为易位。

=== 终止 ===
以上的过程不断重复直到核糖体遇到三个结束密码子之一，翻译过程终止。相鄰的氨基酸在編碼後以肽鍵連接，產生多肽，多個多肽再透過氫鍵，離子鍵等弱鍵摺曲，形成有三維構象的蛋白质。同時蛋白质不再延长，一种模仿{{lang|en|tRNA}}的蛋白质进入核糖体的A位将合成的蛋白质从核糖体内释放出来。

== 真核翻译 ==
{{main|真核翻译}}
在真核细胞中转录是在细胞核中进行的，然后{{lang|en|mRNA}}被运输到[[细胞质|细胞质]]进行翻译。在运输过程中{{lang|en|mRNA}}受到特别的结构的保护。但需要注意的是，在真核细胞中[[粒線體|粒線體]]和[[葉綠體|葉綠體]]中的轉錄與轉譯行為都與原核生物類似（参见[[內共生理論|內共生理論]]）。

=== 起始 ===
在真核细胞中核糖体与{{lang|en|mRNA}}中的起始密码子结合，在[[真核生物|真核生物]]和[[古细菌|古细菌]]中起始密码子的码与[[蛋氨酸|蛋氨酸]]的码相同，与蛋氨酸相连的{{lang|en|tRNA}}是核糖体的一个组成部分。

延长和终止的过程与原核细胞相似。

== 手动翻译 ==
生物学家和化学家用手或计算机来模拟翻译过程来理解一个基因所编码的蛋白质的结构。

首先要从{{lang|en|DNA}}导出{{lang|en|RNA}}：

 DNA -> RNA
  A  ->  U
  T  ->  A
  G  ->  C
  C  ->  G

然后将每三个{{lang|en|RNA}}码组合为一个密码子，最后察看表格将每个密码子转化为氨基酸。

这是蛋白质的氨基酸顺序。按照蛋白质内[[亲水性|亲水]]和[[疏水性|斥水]]部分的排列可以对[[蛋白质折叠|蛋白质折叠]]方式作出一些推断。但要完全预言一个蛋白质的[[三级结构|形状]]是相当不容易的。{{citation_needed|今天的软件可以推测出70%的蛋白质的形状|time=2007-10-15T16:59:10Z}}。
