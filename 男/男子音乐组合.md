'''男子組合'''（意思和英文的Boyband不盡相同，男子組合是對男子和組合的一個廣泛稱呼，而英文的Boyband則是指由一群年青人組成的歌唱組合）或直譯為'''男孩團'''，1990年代時一些著名男子音樂組合，例如，英国的[[接招|Take That]]，美國的[[超級男孩|NSYNC]]，爱尔兰的[[男孩特區|Boyzone]]。

== 歷史 ==
[[馬瑞斯·史達|馬瑞斯·史達]]（{{lang|en|Maurice Starr}}）和[[街頭頑童_(男孩團體)|New Kids on the Block]]通常被認為是開創起男子音乐组合潮流的先鋒，雖然當時「{{lang|en|Boy band}}」這個名詞還不存在。史達的想法是將當時[[節奏藍調|節奏藍調]]的原型利用到在流行音樂中。這個模式後來被一些歐洲的經紀人例如尼加·馬丁-史密斯（{{lang|en|Nigel Martin-Smith}}）和[[路易·華許|路易·華許]]重新定義、並且佔據當時的英國流行音樂市場。 

[[頑童合唱團|頑童合唱團]]（{{lang|en|The Monkees|}}）一般被認為是第一個男子音乐组合。電視節目製作人[[伯特·史耐德|伯特·史耐德]]（{{lang|en|Bert Schneider}}）和（{{lang|en|Bob Rafelson}}）找了四位歌手演唱了一些容易上口的流行音樂和在演出[[電視劇|電視劇]]。猴子男孩成立於1965年並在1970年解散、而「猴子男孩」這四個字後來也常被用來形容成立於1990年代前的男子音乐组合。[[披頭四|披頭四]]、[[海灘男孩|海灘男孩]]（{{lang|en|The Beach Boys}}）和[[誘惑合唱團|誘惑合唱團]]（{{lang|en|The Temptations}}）、以及1960年代的[[比吉斯|比吉斯]]（{{lang|en|The Bee Gees}}）、{{lang|en|The Jackson 5}}、與在1970年代相當受歡迎的[[大地、風和火|大地、風和火]]（{{lang|en|Earth, Wind and Fire}}）被部分人認為也是另一種類型的男子音乐组合。 

在[[美國|美國]]最具知名度的男子音乐组合經紀人[[羅·佩爾曼|羅·佩爾曼]]（{{lang|en|Lou Pearlman}}）經營過一些非常成功的團體比如說[[後街男孩|後街男孩]]以及[[超級男孩|超級男孩]]（{{lang|en|NSYNC}}）。英國方面製作人[[西蒙·高維爾|西蒙·高維爾]]因為曾經製作過[[西城男孩|西城男孩]]而成名。

== 中心概念 ==
一個男子音乐组合除了銷售成績以外最重要的就是形象。男子音乐组合的形象通常藉由嚴格控制團員衣著造型、廣告行銷（通常利用青少年讀物）、以及[[音樂錄影帶|音樂錄影帶]]等。銷售成績等同於這個團體的。最常見的是每個團員都會有一些特徵來描寫、例如說人格特質像是「鄰家男孩」、「壞男孩」、或著是「好好先生」等。雖然刻意維持流行[[歌手|歌手]]形象的這種行為幾乎和[[流行音樂|流行音樂]]的歷史一樣老，但是經紀公司對團員的特別呵護是男子和女子音乐组合的一種特徵。

在大部分的狀況中，男子團體的音樂作品通常是已經由負責經營團體的製作人編寫、安排和發布、如果必要的話甚至會僱用合音歌手來個別執導團員。

一個典型的男子音乐组合的表演包括精心製作的舞蹈和成員輪流演唱（雖然有時是用預先錄製好的音樂[[對嘴|對嘴]]）。男子音乐组合通常並不編寫他們自己的音樂，除非團員們願意（例如猴子男孩和[[超級男孩|超級男孩]]）。

男子音乐组合最重要的要素就是流行。這表示一個男子音乐组合必須要能夠跟隨著當時流行的音樂和衣著來改變。雖然一些符合[[刻板印象|刻板印象]]的男子音乐组合都紛紛解散，這個概念延伸至現今的話可以用來形容一些比較極端的團體例如[[我的另類羅曼史|我的另類羅曼史]]以及Hellogoodbye。

== 用辭爭議 ==
「樂團（{{lang|en|band}}）」在字典中的定義是一個演奏[[樂器|樂器]]的團體<ref>[http://www.merriam-webster.com/dictionary/band%5B3%5D]</ref>，因此有時“男子樂團（{{lang|en|boy band}}）這個名詞的用法具有爭議，因為大部分的男子樂團並不演奏樂器。

== 批評 ==
男子音乐组合因其市場目標為年青的小女生，與及過度包裝的關係而經常遭到批評。基於男子音乐组合極端公式化的特質，在美國，某些批評常將男子音乐组合大量發行的流行音樂作品和小說家[[喬治·奧威爾|喬治·奧威爾]]所寫的[[反烏托邦|反烏托邦]]小說《[[一九八四|一九八四]]》中的情結作比較。另外有些人則是將男子音乐组合當作是種[[後現代主義|後現代主義]]和[[商業主義|商業主義]]的研究。

== 在流行文化中的影響 ==
男子音乐组合不僅僅經常受到批評、同時也是諷刺的目標。

* [[挪威|挪威]][[電影|電影]]《{{tsl|en|Get Ready to be Boyzvoiced||Get Ready to be Boyzvoiced}}》 [http://us.imdb.com/Title?0248036]諷刺男子音乐组合{{tsl|en|Boyzvoice||Boyzvoice}}的支持者和經紀公司。

* 《[[南方公園|南方公園]]》, 阿ㄆ一ㄚˇ（{{lang|en|Cartman}}、中國大陸譯卡特曼、港譯蛋散）組成了一個叫做指叉（{{lang|en|Fingerbang}}）的男子音乐组合。

* [[阿姆|阿姆]]曾在專輯The Real Slim Shady中表示「我對你們這些小男子和女子團體感到噁心、你們只會讓我感到厭煩」。

== 音樂方面的成就 ==
雖然一些男子音乐组合的支持者對他們的音樂表示瘋狂的支持，但這些男子音乐组合的銷售成績通常不會維持太久。當支持者（通常是年紀相當小的女生）長大並且改變喜好的音樂類型時，通常會逐漸對這些團體失去興趣。如果當一個團體持續暢銷的話，通常團體的成員會選擇離開團體（尤其是當他們有創作歌曲的能力時）、例如[[賈斯汀·提姆布萊克|賈斯汀]] （{{lang|en|Justin Timberlake}}）、[[瑞奇·馬丁|瑞奇·馬丁]]（{{lang|en|Ricky Martin}}）等。

某些男子團體在成員步入中年後依然受到歡迎，例如[[美國|美國]]的[[新好男孩|新好男孩]]、[[愛爾蘭|愛爾蘭]]的[[西城男孩|西城男孩]]、[[中華民國|臺灣]]的[[5566|5566]]、[[日本|日本]]的[[SMAP|SMAP]]。[[韓國|韓國]]的[[SHINHWA|SHINHWA]]
