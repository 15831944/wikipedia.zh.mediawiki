'''男性乳房發育症'''({{lang|en|gynecomastia}})，亦作'''男性乳腺增生'''或'''女性化乳房'''，是指男性胸部因為[[乳腺|乳腺]]不正常發育造成的增大狀況。這種疾病的英文名稱Gynecomastia的兩個詞根gyne-(γυνή)和-mastos(μαστός)在[[古希臘語|古希臘語]]中分別指代“女性”和“[[乳房|乳房]]”。這種狀況可能會發生在初生嬰兒、[[青春期|青春期]]和成年男性身上：[[新生兒|新生兒]]的症狀可能是因為體內仍留有母親的一部分雌性[[荷爾蒙|荷爾蒙]]而引起的；青春期男性的症狀在患者沒有過度[[肥胖症|肥胖症]]的情況下，通常會在幾年內逐漸自愈<ref>{{cite web |title=Gynecomastia in Adolescent |url=http://www.keepkidshealthy.com/adolescent/adolescentproblems/gynecomastia.html |language=en |date=2001-05-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20100210023757/http://www.keepkidshealthy.com/adolescent/adolescentproblems/gynecomastia.html |archivedate=2010-02-10 }}</ref>。男性乳房發育症的病因比較難以判斷，一般認為是荷爾蒙分泌不平衡或者是胸部組織對荷爾蒙刺激反應異常所致。乳房組織、[[皮下脂肪|皮下脂肪]]和胸部[[皮膚|皮膚]]的過度生長都會造成患者胸部異常增大，臨床上常見幾種狀況綜合出現。純綷因為肥胖症或其他因為身體有過多脂肪而產生的乳房，會被稱為'''假性乳房發育症'''（{{lang|en|'''pseudogynecomastia'''}}）<ref>{{cite journal| last=Braunstein| first=GD| title=Gynecomastia| journal=N Engl J Med| date=1993-02-18 | volume=328| issue=7| pages=490–5| pmid=8421478| doi=10.1056/NEJM199302183280708}}</ref>或'''脂肪性乳房發育症'''（{{lang|en|'''lipomastia'''}}）<ref>{{cite web
  | last = Allee

== 治療 ==

儘管醫學界對於是否需要治療男性乳房發育症而莫衷一是，對於比較輕微的男性乳房發育症，可以透過穿著有壓力的男性[[胸圍|胸圍]]來把突出的乳房隱藏；但較嚴重的病況，則需要透過[[外科手術|外科手術]]進行[[縮胸|縮胸]]。
