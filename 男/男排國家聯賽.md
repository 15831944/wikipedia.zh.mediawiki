'''男排國家聯賽'''，或稱'''世界男排聯賽'''（[[英語|英語]]: '''FIVB Volleyball Men's Nations League'''，英文簡稱：'''VNL'''），是[[國際排球總會|國際排聯]]（[[FIVB|FIVB]]）舉辦的一年一度大型世界級男子排球比賽，首屆賽事於2018年舉行，以取代已舉辦28年的[[世界排球聯賽|世界排球聯賽]]。賽事有16支隊伍參賽，其中12支為核心隊伍、4支為挑戰隊伍。

與世界排球聯賽不同的是，國家聯賽不再劃分三個檔次組別，統一以一個組別進行比賽，但參與隊伍的總數會由去年世界聯賽的36隊降低至16隊，其中12隊核心隊伍大部分來自以往第一組別的隊伍，擁有至2024年的參賽資格；餘下4隊為挑戰隊伍，其中一個名額每年均會作出調整，需參加國家挑戰杯競爭。<ref>{{Citeweb|url=https://www.fivb.org/viewPressRelease.asp?No=73085&Language=en#.WpUIIB3miEc|title=FIVB announces the Volleyball Nations League}}</ref>另外分站賽的數目由世界聯賽時的三站增加到五站，以及避免球隊之間重複交手的現象。

== 起源 ==
在2017年10月13日，國際排聯推出了一項名為排球國家聯賽 (VNL) 的新比賽，並於2018年首次亮相。通過先進技術，創新和數字廣播的界限，觀眾將目睹多元化的角度，展現每個球員的世界級運動能力。

FIVB, IMG和21個國家聯合會合作男子排球國家聯賽和女子排球國家聯賽的比賽。每隊至少打15場比賽，整個賽季總共打130場比賽。每個核心隊伍都將舉辦至少一次分站賽，以向世界各地展示新的競賽。<ref>{{Citeweb|url=http://www.sportspromedia.com/news/fivb-to-launch-volleyball-nations-league|title=FIVB to launch Volleyball Nations League}}</ref>這是國際排聯首次與IMG合作，將全面控制內容製作，最大限度地提高質量，並推出場內講述，對球迷來說更容易接觸和理解。

2018年2月22日，國際排聯宣佈首屆男排國家聯賽的決賽舉辦城市，最終由[[法國|法國]][[里爾|里爾]]承辦。

== 賽制 ==
賽事分為兩輪，分站賽及總決賽，決賽主辦國的球隊會自動獲得六強名額。國家聯賽比賽歷時六週，前五週屬分站賽賽事，最後一週屬總決賽。分站赛主要在周五、周六、周日进行，分站賽每個分站均有四隊參加單循環賽事。對每一隊而言，每個分站的对手都不一樣，每支球隊都會與其他15支球隊交手，而同一週均有四個分站賽事進行。每隊都要先經5個分站共15場比賽，<ref>{{Citeweb|url=http://sports.sina.com.cn/others/volleyball/2018-02-25/doc-ifyrwsqi1148957.shtml|title=国家联赛中国女排将辗转五地 15个对手由弱到强}}</ref>
最後成績最佳的首5名（不包括總決賽主辦國）連同總決賽主辦國隊伍進入總決賽輪。 <ref>{{Citeweb|url=http://www.worldofvolley.com/News/Latest_news/FIVB/88188/this-is-the-official-format-of-new-volleyball-league-2018-both-mens-and-womens.html|title=official format of New Volleyball League 2018, both men's and women's}}</ref>总决赛採用分组加交叉的赛制。

===預賽輪（分站賽）===
16支球队经过抽签后，比赛将会分20組進行，每支球队都与其他15支球队交手。而每支球队都是分站賽东道主之一，均会承繼至少一個分站賽事，其中部分核心球队将不止有一次成为分站賽东道主的机会。所有比赛完成之后，排名前5名的球队，与决赛的东道主球队参加最终的决赛。<ref name=format/>在所有比賽結束後排名最後的挑戰球隊將被淘汰，其名額由國家挑戰杯的冠軍球隊取代。<ref>{{cite web |url=http://www.fivb.com/en/about/news/fivb-executive-committee-embraces-digital-transformation?id=73296 |title=FIVB Executive Committee embraces digital transformation |date=7 December 2017 |publisher=[[Fédération_Internationale_de_Volleyball|FIVB]] |accessdate=8 February 2018}}</ref>

===决赛轮===
6支晋级决赛的球队将分为两个小组，每个小组3支球队。小组前两名直接晋级半决赛。每个小组第一将对阵小组第二，并最终决出冠军。

== 參賽資格 ==
賽事有16支球队参加。 其中十二队的核心队伍無須经历降级，大部分均是世界排名靠前的隊伍。 其他四支球队被选为挑战队伍，可能会被淘汰。12支核心隊無論排名如何都將是2018年至2024年的參賽球隊，4支挑戰隊排名前3名將保留參賽資格，最後一名的名額將由國家挑戰杯的冠軍取代。

== 國家挑战杯 (Challenger Cup)  ==
國際排聯的排球國家挑戰杯與排球國家聯賽同時舉行，旨在讓更多隊伍能以挑戰球隊身份參與VNL。 挑戰杯由不參加本屆排球國家聯賽的球隊組成，包括來自五大洲地區聯合會的五支球隊和一支主辦國主隊。賽事會採用小組賽形式進行，之後經過半決賽，最後在決賽的勝者將獲得參加下屆排球國家聯賽的參賽資格。

地區及洲份聯合會可通過組織洲份資格賽，或使用現有的比賽來確定參加排球國家挑戰杯的球隊，其中歐洲獲得兩個席位。
