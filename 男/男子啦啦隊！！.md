'''男子啦啦隊！！'''（{{lang-ja|チア男子!!}}）為[[朝井遼|朝井遼]]的[[日本|日本]]青春[[小説|小説]]作品。2010年開始在[[集英社|集英社]]發表。中譯版2016年在台灣翻譯出版。電視動畫在2016年7月5日至9月27日首播。

== 故事簡介 ==
{{expand section}}
柔道場長男坂東晴希決定放棄柔道，宣布退出大學柔道部的那天，和一起退出柔道部的青梅竹馬橋本一馬組成男子啦啦隊，吸引各具特色的成員加入，沒有經驗、土法煉鋼的他們，卻以全國大賽為目標。

== 登場角色 ==
=== 主要人物 ===
; {{nihongo|坂東晴希|坂東 晴希（ばんどう はるき）|聲：[[米内佑希|米内佑希]]（日）／[[巫哲棋|巫哲棋]]（港）}}
: 命志院大學社會學系一年級生。BREAKERS草創時成員之一，擔任頂端人員。因為肩膀受傷而退出柔道社。本身有懼高症，對於被拋到高處的動作感到懼怕。出身柔道世家，家中開有道館，姐姐晴子是柔道社的主將，但和姐姐及一馬相比，從沒有在正式大賽比賽的經歷，覺得自己沒有柔道天賦，在肩膀受傷後正式放棄柔道。在好友一馬的邀約下加入男子啦啦隊。生日是5月9日。
; {{nihongo|橋本一馬|橋本 一馬（はしもと かずま）|聲：[[岡本信彦|岡本信彦]]（日）／[[胡家豪|胡家豪]]（港）}}
: 命志院大學社會學系一年級生。BREAKERS的發起者，擔任頂端人員。晴希的青梅竹馬。父母從小就過世而與奶奶生活，小學轉學到晴希的班級，本來孤僻的他因為晴希主動接近，逐漸開啟心房而成為朋友。從小就和晴希一同學習柔道，相當的有天賦，曾打入全國高中聯賽。在晴希退出柔道社後也一起退出。由於父母年輕時都是啦啦隊員，母親是頂端人員，父親是教練，退出柔道社後拉著晴希組建男子啦啦隊，家中還有父母親年輕時比賽的影帶。生日是8月10日。
; {{nihongo|溝口涉|溝口 渉（みぞぐち わたる）|聲：[[杉田智和|杉田智和]]（日）／[[黃啟昌|黃啟昌]]（港）}}
: 命志院大學商學系一年級生。BREAKERS草創時的成員之一，擔任保護人員。20歲，因為重考兩次所以仍然是一年級。時常引用知名典故和名句。節奏感差，肢體僵硬。家裡開設料亭所以非常富有，負責支出練習設備並將其運至晴希家頂樓空地。直到高中都是田徑隊員。由於個性較異於常人，所以遇上晴希他們前都沒有朋友。生日是12月25日。
; {{nihongo|德川翔|徳川 翔（とくがわ しょう）|聲：[[小野友樹|小野友樹]]（日）／[[黃積權|黃積權]]（港）}}
: 命志院大學社會學系一年級生。BREAKERS成員之一，是隊伍中唯一有啦啦隊經驗的人，起初反對只有男生的啦啦隊。極喜歡動畫中卡通人物《松鼠次郎》。高中就讀陽明大學附設高中部，曾擔任SPARKS的底層人員，但因夏季盃害教練的妹妹受傷而對啦啦隊感到退卻。生日是10月31日。
; {{nihongo|遠野浩司|遠野 浩司（とおの こうじ）|聲：[[林勇|林勇]]（日）／[[李安邦|李安邦]]（港）}}
: 命志院大學文學系一年級生。BREAKERS成員之一，擔任底層人員。綽號噸。為向日葵食堂巨大咖哩豬排飯最快速吃完的紀錄保持人。沒有運動經驗，且缺乏自信，因為想要改變而加入BREAKERS。為隊中唯一有女朋友的人。生日是3月14日。
; {{nihongo|鈴木總一郎|鈴木 総一郎（すずき そういちろう）|聲：{{link-ja|桑野晃輔|桑野晃輔}}（日）／[[關令翹|關令翹]]（港）}}
: 命志院大學商學系一年級生。BREAKERS成員之一，擔任底層人員。綽號ICHIRO。弦的青梅主馬。關西人。國小及國中時期皆與弦加入棒球隊，是隊上的王牌投手，高中時仍為棒球隊主力之一。被弦稱做"不懂得人們懊惱"的人。上大學後因想加入比較時尚的社團而參加網球同好會，但發現不太搭。因稱想嘗試不同的挑戰(實則是吸引女生的注意)而與弦一同加入BREAKERS。生日是4月2日。
; {{nihongo|長谷川弦|長谷川 弦（はせがわ げん）|聲：[[小西克幸|小西克幸]]（日）／[[林筠翔|林筠翔]]（港）}}
: 命志院大學商學系一年級生。BREAKERS成員之一，擔任保護人員。關西人。國小及國中時期皆加入棒球隊，高中時為足球隊，常說自己不如總一郎。曾挑戰巨大咖哩豬排飯但未在時限內吃完而失敗。上大學後後因想加入比較時尚的社團而參加網球同好會，但發現不太搭。因稱想嘗試不同的挑戰(實則是吸引女生的注意)而加入BREAKERS。生日是9月27日。
