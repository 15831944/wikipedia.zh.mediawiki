《'''男兒本色'''》是[[電視廣播有限公司|電視廣播有限公司]]在1987年播出的電視劇，由李添勝監製，[[林俊賢|林俊賢]] <ref>{{Cite web |url = http://enjoy.eastday.com/e/tv/u1ai9875426.html |title =林俊賢:男神未滿 每個人要走的路各有不同|author =甘鵬|date =2016年10月31日|publisher =上海電視週刊|language =中文|via=東方網娛樂 }}</ref>、[[黎明|黎明]] <ref>{{Cite web |url =https://www.mpweekly.com/entertainment/focus/local/20170605-60480|title =金句王黎明經典劇集| date =2017年6月5日|publisher =明報周刊|language =中文}}</ref>、[[謝寧|謝寧]]、[[陳敏兒|陳敏兒]]、[[蘇永康|蘇永康]]、[[蔡嘉利|蔡嘉利]]、[[吳孟達|吳孟達]]、[[高雄|高雄]]、[[林建明|林建明]]等演出，於1987年3月9日至4月7日期間在[[無綫電視|無綫電視]][[翡翠台|翡翠台]]播出<ref>{{Cite book|author =陳善之、翟浩然|title =善緣滿佈 - 陳善之回首前塵四十載|publisher =明窗出版社|ISBN = 9789888338689|date =2016年7月11日| url =https://books.google.com.hk/books?id=xbtgDQAAQBAJ&vq=%E9%BB%8E%E6%98%8E&hl=zh-TW&source=gbs_navlinks_s=false|language =中文|}}</ref>。

==劇情簡介==
劇情講述徐湘自幼由余威收養，撫養長大。余威從事偏門行業，經常惹禍，徐湘為了報答余威的養育之恩，經常獨力承擔余威闖下的禍。徐湘有一弟妹徐永發及徐美麗，由徐湘賺錢供養。殷紹文是一名社工，偶然認識了徐湘，彼此被對方吸引，與此同時，徐湘的好友謝森也對殷紹文產生了愛慕之情，兩人因此反目成仇，後來謝森和徐湘在同一間銀行工作，在工作上出現紛爭，大家的關係更趨緊張。此外，徐湘一直想念著小時候的玩伴高雪媚，眾人展開了錯綜複雜的恩怨情仇故事<ref>{{Cite web |url =https://programme.mytvsuper.com/tc/103224/%E7%94%B7%E5%85%92%E6%9C%AC%E8%89%B2|title = 男兒本色| publisher =myTV SUPER|language =中文}}</ref><ref>{{Cite web |url = http://img2.ph.126.net/v6e1nO2quVENpUxfT2Rwrw==/6630522407932918865.jpg |title =港劇評析－論港劇男兒本色|publisher =歡樂無線雜誌|language =中文}}</ref><ref>{{Cite web |url = http://img1.ph.126.net/hA2OvI1WlSp9LRUD8CZwMw==/6630391566049212934.jpg |title =港劇評析－愛情利刃 頭破血流|publisher =歡樂無線雜誌|language =中文}}</ref>。
