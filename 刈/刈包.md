'''割包（英文：Gua bao)'''，為[[臺灣|臺灣]]當地知名[[小吃|小吃]]{{r|辭典割包}}{{r|外交部}}之一。蒸饅頭內夾滷肉的概念，原型源起於[[中國|中國]][[福建省|福建省]][[福州|福州]]的「虎咬豬」<ref>{{cite book |author1=江韶瑩 |title=臺灣民俗文物辭彙類編 |date=2009-12-01 |publisher=國史館臺灣文獻館 |location=中華民國南投市 |isbn=9860203997 |page=568 |edition=初版 |url=https://books.google.com/books?id=MUdha-AR2nEC&q=%E8%99%8E%E5%92%AC%E7%8C%AA+%E7%A6%8F%E5%B7%9E&dq=%E8%99%8E%E5%92%AC%E7%8C%AA+%E7%A6%8F%E5%B7%9E&hl=en&sa=X&ved=0ahUKEwjzkYbVsbzeAhUxh-AKHdeCAtMQ6AEIXTAI}}</ref><ref name="chuanyi">{{cite magazine |author=<!--Staff writer(s); no by-line.--> |title=傳統藝術 |issue=74-76 |page=103 |url=https://books.google.com/books?id=_ZURAQAAMAAJ&dq=%E8%99%8E%E5%92%AC%E7%8C%AA+%E7%A6%8F%E5%B7%9E&focus=searchwithinvolume&q=%E8%99%8E%E5%92%AC%E7%8C%AA+%E5%88%88%E5%8C%85+%E7%A6%8F%E5%B7%9E |magazine=傳統藝術 |location=中華民國宜蘭縣 |publisher=國立傳統藝術中心 |date=2008 |ISSN=1680-1490}}</ref><ref>又稱「割包」，教育部《[[重編國語辭典修訂本|重編國語辭典修訂本]]》以為三者為同種概念，{{cite web |title=《重編國語辭典修訂本》刈包 |url=http://dict.revised.moe.edu.tw/cgi-bin/cbdic/gsweb.cgi?o=dcbdic&searchid=Z00000150975}}</ref>，但經臺灣在地化口味改良而讓普羅大眾更廣為接受<ref>{{cite web |author1=魚夫 |title=飲食文化大不同──來說潤餅、刈包、燒賣和粽子 |url=https://opinion.cw.com.tw/blog/profile/194/article/6570 |website=獨立評論@天下 |accessdate=2018-11-05 |language=zh-TW}}</ref>。在日本[[長崎新地中華街|長崎新地中華街]]稱之為「[[角煮|角煮]][[饅頭|饅頭]]」（{{lang|ja|角煮まんじゅう}}），取其為滷[[五花肉|五花肉]]夾饅頭之意。

== 簡介 ==
刈包類似[[饅頭|饅頭]]。兩者差別是饅頭為立體半球形，但刈包是長橢圓扁形，再對摺起來包餡料。

傳統餡料通有：片狀五花滷肉、[[酸菜|酸菜]]、花生粉及[[香菜|香菜]]等。花生粉一般與糖混合，有甜味。另外，也有健康導向，以[[豬肉|瘦肉]]代替[[五花肉|五花肉]]的做法{{r|外交部}}。因為其形狀及內餡似[[老虎|老虎]]的嘴咬住一塊豬肉，所以在[[臺灣|臺灣]]亦被稱作'''虎咬豬'''（hóo-kā-ti）<ref name="chuanyi" />，其吃法如同[[漢堡_(食物)|漢堡]]般，故有「臺式漢堡」、「臺灣漢堡」之名{{r|外交部}}，也常被暱稱為「包包」。

近年，[[臺灣|臺灣]]業者將這種傳統美食再改良及包裝後引領一波小吃創新的風潮，除了傳統白麵皮外，甚至也有了黑糖麵皮等，在內餡口味也變化出許多花樣，例如：加入[[萬巒豬腳|萬巒豬腳]]、[[香腸|香腸]]、[[筊白筍|筊白筍]]（美人腿）、[[剝皮辣椒|剝皮辣椒]]、[[天婦羅|天婦羅]]、鴨胸、[[煎蛋|煎蛋]]、[[炸雞排|雞排]]、[[鱈魚|鱈魚]]排、[[日式叉燒|日式叉燒肉]]、[[鹹豬肉|鹹豬肉]]、爐烤[[杏鮑菇|杏鮑菇]]、照燒[[雞腿|雞腿]]、[[牛肉丸|牛肉丸]]，甚至[[豬舌|豬舌]]等新式刈包。因為這個緣故，不少人也稱刈包為「臺灣[[漢堡包|漢堡]]」（''Taiwanese hamburger''）。在[[臺灣|臺灣]]，早期刈包經常會搭配[[四神湯|四神湯]]食用，後來開始出現以精釀啤酒、果汁或氣泡飲料佐餐的潮流並逐漸發展出「台酒配台食」的餐酒品飲美學。

經典口味的刈包在[[臺灣|臺灣]]同時也是「[[尾禡|尾禡]]」（尾牙，或稱「尾迓」）宴席上經常出現的菜色之一。因為外型長得像錢包，裡面夾一大塊滷肉、酸菜、花生粉、還有香菜，因形狀像一個飽滿的錢包，取其財富滿足的喻意，因此在尾牙吃刈包，象徵來年發大財、錢包滿滿用不完。另一說是，刈包的另外一個名字為「虎咬豬」，與閩南語音「福咬住」相近，內夾的酸菜被客家人視為是福菜，所以有「留住福氣」之意。其實，尾牙吃刈包，對生意人還有另一層意義，商家由於生意上的應對，常會說些善意的謊言，吃刈包代表商家象徵性地把今年曾說過謊言包起來吃掉的意思。這些都是[[臺灣|臺灣]]當地的演進出的應景風俗，與源起地並無相關。

== 讀音、寫法 ==
[[File:Steamed_Bun_Burger.jpg|thumb]]、[[生菜|生菜]]等美式漢堡配料，故被稱為「美式叉包」。]]
「刈」字國語音為-{zh-cn:yì;zh-tw:ㄧˋ;}-，意為「割取」，以刀切開之意。該字閩南語唸做ngāi，與[[艾姓|姓氏艾]]的發音相同{{r|甘字典}}。但日常中，「刈包」慣以[[臺灣閩南話|臺灣閩南話]]「kuah-pau」發音，[[台文|台文]]寫做「'''割包'''」{{r|辭典割包}}，因為「割」字在臺灣閩南話中才有kuah的音{{r|辭典割}}。也因此也常寫為國、閩語皆同音的「掛包」。

另外，香港有發售這種包的店舖稱之為「'''叉包'''」。
