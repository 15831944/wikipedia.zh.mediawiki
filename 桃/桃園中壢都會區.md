'''桃園都會區'''，其涵蓋範圍包括[[桃園市|桃園市]]（不含[[復興區_(台灣)|復興區]]）以及[[新北市|新北市]][[鶯歌區|鶯歌區]]，都會區內以[[桃園區|桃園區]]和[[中壢區|中壢區]]為[[雙城|雙核心]]發展，都會區人口約225萬人，為次於[[臺北都會區|臺北都會區]]、[[臺中都會區|臺中都會區]]、[[高雄都會區|高雄都會區]]的第四大都會區，如在世界人口組織裡，排名與[[臺北都會區|臺北都會區]]分開，為世界排名第'''277'''大都會區<ref>http://www.populationdata.net/index2.php?option=palmares&rid=4&nom=grandes-villes</ref>

依據主計處在2010年5月18日發布的新聞稿，上述定義範圍連同全[[中華民國|臺灣]]各種統計地區的分類定義，已在當年12月25日停止適用。<ref>{{zh-tw}}{{cite web|title=「中華民國統計地區標準分類」自99年12月25日起停止適用|url=http://www.dgbas.gov.tw/public/Attachment/051910361871.pdf|accessdate=2010-05-18|publisher=[[行政院主計處|行政院主計處]]}}</ref>

== 形成 ==
早期[[中華民國內政部|內政部]]將桃園市、八德市、鶯歌鎮等規劃為[[臺北都會區|臺北都會區]]內，中壢市、平鎮市等則為中壢次都會區。後續修訂版本中，將桃園市及鄰近行政區由臺北都會區分出另劃'''桃園中壢都會區'''，新都會區人口2,130,526人。而根據世界人口組織網站的統計，「桃園中壢都會區」被劃入[[臺北都會區|臺北都會區]]內，若與[[臺北都會區|臺北都會區]]併計，人口將達905萬人，在全球都會區排名第49位；[[臺中彰化都會區|臺中彰化都會區]]第132位；[[高雄都會區|高雄都會區]]第152位；[[臺南都會區|臺南都會區]]第318位<ref>http://www.populationdata.net/index2.php?option=palmares&rid=4&nom=grandes-villes</ref>。

== 現況 ==
桃園中壢都會區和其他[[中華民國|臺灣]]的大都會區不同的地方點有二：
# 兩個等大的中心都市：[[桃園區|桃園區]]的人口有44萬，[[中壢區|中壢區]]有40萬，兩者構成[[雙城|雙城]]格局，與其他都會區相對獨大的單核心發展不同。周邊的[[平鎮區|平鎮區]]的人口有22.6萬，[[八德區|八德區]]有20.3萬，[[楊梅區|楊梅區]]有17.1萬，[[蘆竹區|蘆竹區]]有16.4萬，[[龜山區|龜山區]]有16.1萬。
# 因磁吸效應，桃園中壢都會區高度依賴[[臺北都會區|臺北都會區]]，異於與其他都會區經濟獨立自主。例如2009年直轄市改制案中，桃園即遭中央認為核心都市「尚非北部地區之主要發展中心」<ref>{{zh-tw}}{{cite news|title=臺南縣市行政院核定改制|url=http://www.ey.gov.tw/News_Content.aspx?n=F8BAEBE9491FC830&sms=99606AC2FCD53A3A&s=2F7B7933857FCFEB|archiveurl=https://archive.is/20170810005933/http://www.ey.gov.tw/News_Content.aspx?n=F8BAEBE9491FC830&sms=99606AC2FCD53A3A&s=2F7B7933857FCFEB|website=[[行政院|行政院]]全球資訊網|publisher=[[行政院新聞局|行政院新聞局]]|date=2009-06-29|archivedate=2017-08-10|deadurl=no}}</ref>，這也是早期將[[桃園市|北桃園]]直接劃入[[臺北都會區|臺北都會區]]的原因之一。

==中壢次都會區==
1985年出版之《中華民國臺灣地區聚居地、都市化地區及都會區分布圖》內之中壢次都會區。以[[中壢區|中壢市]]為中心都市，都會區包含[[中壢區|中壢市]]、[[楊梅區|楊梅鎮]]、[[八德區|八德鄉]]、[[新屋區|新屋鄉]]、[[龍潭區_(台灣)|龍潭鄉]]、[[平鎮區|平鎮鄉]]、[[關西鎮_(台灣)|關西鎮]]。當時總區域人口約62.3萬人。
