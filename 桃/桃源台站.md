'''桃源台站'''（{{jpn|j=桃源台駅|hg=とうげんだいえき|Tōgendai eki}}）及'''桃源台港'''是一座位於[[神奈川縣|神奈川縣]][[足柄下郡|足柄下郡]][[箱根町|箱根町]]{{link-ja|元箱根}}的車站及港口。該站編號為'''OH 65'''<ref>{{cite press release|url=http://www.odakyu.jp/program/info/data.info/8052_1284200_.pdf|script-title=ja:小田急線・箱根登山線・箱根ロープウェイ・箱根海賊觀光船にて２０１４年１月から駅ナンバリングを順次導入します！|publisher=小田急電鐵||language=ja|date=2013-12-24|accessdate=2018-11-22}}</ref>，[[箱根空中纜車|箱根空中纜車]]、[[箱根海賊觀光船|箱根海賊觀光船]]及[[箱根登山巴士|箱根登山巴士]]和[[小田急箱根高速巴士|小田急箱根高速巴士]]等交通在此交匯，並共用同一站體設備。

== 簡史 ==
* [[1960年|1960年]][[9月7日|9月7日]] - 啟用。
* [[2006年|2006年]][[6月1日|6月1日]] - 空中纜車翻新工程暫停營業<ref name="kaisetsu">{{Cite web |url=http://www.hakoneropeway.co.jp/about/|script-title=ja:箱根ロープウェイをもっと知ろう！|language=ja|publisher=箱根ロープウェイ|date=2015-08-27|accessdate=2015-10-18}}</ref>。車站於停業期間同時進行擴建工程。
* [[2007年|2007年]]6月1日 - 恢復營業。
* [[2015年|2015年]]
** [[5月6日|5月6日]] - 因箱根山的火山性地震頻發<ref>{{Cite web |url=http://www.onken.odawara.kanagawa.jp/modules/mysection1/item.php?itemid=44|script-title=ja:箱根のやや活発な地震活動（4月26日〜）|publisher=神奈川縣温泉地学研究所|language=ja|accessdate=2015-10-18}}</ref>、氣象廳針對箱根山地區發出[[火山噴發預警級別|火山噴發預警第二級警報]]<ref>{{Cite web|url=http://www.jma.go.jp/jp/volcano/forecast_03_20150506060003.html|script-title=ja:火山名 箱根山 噴火警報（火口周辺）|publisher=[[気象庁|気象庁]]地震火山部|language=ja|date=2015-05-06|accessdate=2015-10-18}}</ref>，箱根空中纜車全線停駛<ref>{{Cite web|url=http://www.kanaloco.jp/article/101152|script-title=ja:折り紙でおもてなし　箱根・桃源台窓口の女性奮闘：カナロコ|publisher=[[神奈川新聞|神奈川新聞]]|language=ja|date=2015-06-08|accessdate=2015-10-18}}</ref>。
** [[5月20日|5月20日]] - 往返早雲山站的替代運輸巴士投入營運。
** [[6月30日|6月30日]] - 因氣象廳改發出火山噴發預警第三級警報<ref>{{Cite web |url=http://www.jma.go.jp/jp/volcano/forecast_03_20150630123029.html|script-title=ja:火山名 箱根山 噴火警報（火口周辺）|publisher=気象庁地震火山部|language=ja|date=2015-06-30|accessdate=2015-10-18}}</ref>，替代運輸巴士亦要停駛。
** [[7月6日|7月6日]] - 往返[[強羅站|強羅站]]的替代運輸巴士開始營運<ref>{{Cite web |url=http://www.worldplan.co.jp/goura/file/150716_bus_unkou.pdf|format=PDF|script-title=ja:箱根ロープウェイ代行バス 迂回ルートによる運行開始について|publisher=小田急箱根ホールディングス、箱根ロープウェイ|date=2015-07-03|language=ja|accessdate=2015-10-18}}</ref>。
** [[9月14日|9月14日]] - 火山噴發預警級別降至第二級，往返早雲山站的替代運輸巴士恢復營運<ref>{{PDFlink|[http://www.hakoneropeway.co.jp/info_img/206_pdf.pdf 箱根ロープウェイ代行バス 県道開通による運行経路の変更について]}}</ref>。
** [[10月30日|10月30日]] - 往返[[姥子站|姥子站]]的纜車運行再開<ref>{{Cite news|script-title=ja:箱根ロープウェイ、きょうから一部再開　「紅葉に合わせ」町も許可|newspaper=東京新聞|date=2015-10-30|url=http://www.tokyo-np.co.jp/article/kanagawa/list/201510/CK2015103002000181.html|language=ja|accessdate=2015-11-07|archive-url=https://web.archive.org/web/20151101151112/http://www.tokyo-np.co.jp/article/kanagawa/list/201510/CK2015103002000181.html|archive-date=2015-11-01|dead-url=yes}}</ref>，替代運輸巴士繼續運行。

== 車站 ==
=== 構造 ===
桃源台站依山而建，目前共有四層<ref name="stn">{{Cite web|url=https://www.hakonenavi.jp/transportation/station/togendai/|script-title=ja:桃源台駅・港：箱根の駅構内・周辺図 - 箱根ナビ|publisher=箱根導航|language=ja|accessdate=2018-11-22}}</ref>。二樓為箱根空中纜車月台<ref name="stn"/>，一樓（地面）為出入口及大堂<ref name="stn"/>，箱根空中纜車驗票處、纜車和巴士售票處、商店及旅客服務中心等主要設施均設於此層。地下一樓為餐廳，地下二樓毗鄰蘆之湖，[[箱根海賊觀光船|箱根海賊觀光船]]售票處及候船室等設施位於此層<ref name="stn"/>，各樓層均有以電梯或手扶梯連接<ref name="stn"/>。該站與早雲山站同為箱根空中纜車的終點站，故此兩者都在站內安裝了調校纜索張緊的設備<ref name="kaisetsu" />。

== 車站週邊 ==
桃源台站為小田急集團的大型據點，[[箱根空中纜車|箱根空中纜車]]、[[箱根海賊觀光船|箱根海賊觀光船]]及[[箱根登山巴士|箱根登山巴士]]和[[小田急箱根高速巴士|小田急箱根高速巴士]]均在此交匯<ref name="stn"/>。西武集團[[伊豆箱根鐵道|伊豆箱根鐵道]]的相關交通不會駛進該站，從伊豆箱根鐵道管理的湖尻港步行至該站需約10分。

* [[芦之湖|芦之湖]]
* 芦之湖露營村
* 桃源台港：可換乗[[箱根海賊觀光船|箱根海賊觀光船]]。
* [[湖尻港|湖尻港]]：可換乗伊豆箱根鐵道[[蘆之湖遊覧船|蘆之湖遊覧船]]。
