〈'''桃花泣血記'''〉，是一首由[[詹天馬|詹天馬]]與[[王雲峰_(作曲家)|王雲峰]]創作於[[1932年臺灣|1932年]]的[[電影配樂|電影配樂]]和[[流行歌曲|流行歌曲]]。該曲是[[臺灣|臺灣]]目前已知最早的[[現代|現代]][[商業性|商業性]]流行歌曲。<ref name=":0">{{Cite web|url=https://market.cloud.edu.tw/content/local/tainan/kunhwa/music/90.html|title=桃花泣血記|accessdate=2017-10-18|work=market.cloud.edu.tw}}</ref>

== 簡介 ==
[[1931年臺灣|1931年]]後，片商自[[中國|中國]]進口黑白默片《[[桃花泣血記|桃花泣血記]]》，並委由詹天馬與王雲峰[[創作|創作]]詞曲以作配樂進行[[廣告|宣傳]]；隨後，兩人將[[劇情|劇情]]寫入該曲[[歌詞|歌詞]]中。詞曲完成後，片商[[雇用|雇用]]小型[[樂隊|樂隊]]在[[永樂町_(臺北市)|永樂町]]及[[太平町|太平町]]沿街演奏該曲，以吸引[[人民|大眾]]前往[[戲院|戲院]]觀賞該片。後來，隨著該片受到歡迎，該曲也因此聞名各地，[[古倫美亞唱片|古倫美亞唱片]]更將該曲灌錄成[[黑膠唱片|唱片]]發行。<ref name=":0" />

== 演唱 ==
{{expand section}}
該曲屬[[七字仔調|七字仔調]]形式，以在[[演唱|演唱]]時配合劇情作出變化。<ref name=":0" /><ref>{{Cite web|url=http://webcache.googleusercontent.com/search?q=cache:Tnm3_Jn6EPwJ:mocfile.moc.gov.tw/files/201701/9e250c52-af50-4316-8441-92cac61ac5c9.pdf+&cd=1&hl=zh-TW&ct=clnk&gl=tw|title=多元文化的拼貼―〈桃花泣血記〉|accessdate=2017-10-18|work=webcache.googleusercontent.com}}</ref>

== 歌詞 ==

[[戀愛|戀愛]]無分[[階級|階級]]性，第一要緊是真情，琳姑出世歹[[環境|環境]]，親像桃花彼薄命。

[[禮教|禮教]]束縛非現代，最好[[自由|自由]]的世界，德恩老母無理解，雖然[[有錢人|有錢]]也真害。

德恩無想是富戶，專心實意愛琳姑，免驚日後來相誤，我是男子無糊塗。

琳姑本正也愛伊，信用德恩無懷疑，每日作陣真歡喜，望要作伊的[[妻子|妻兒]]。

[[愛情|愛情]]愈好事愈濟，頑固老母真囉嗦，富男貧女蓋不好，強制平地起風波。

紅顏自本多薄命，拆散兩人的真情，[[運命|運命]]作孽真僥倖，失意斷送過一生。

離別愛人蓋艱苦，親像鈍刀割腸肚，心啼哭病倒鋪，悽慘生意行無路。

壓迫子兒過無理，[[家庭|家庭]][[革命|革命]]隨時起，德恩走去欲見伊，可憐見面已經死。
