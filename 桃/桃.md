'''桃'''（[[学名|学名]]：''{{lang|la|Prunus persica}}''），是起源于中国的一种植物，其果实可以作为[[水果|水果]]食用。

== 分布 ==
桃原产于[[中国|中国]]，最遠可以追溯到[[周朝|周朝]]，《[[诗经|诗经]]·国风·周南》即有〈桃夭〉一詩，逐渐传播到[[亚洲|亚洲]]周边地区，从[[波斯|波斯]]传入西方，桃的[[种小名|种小名]]“爲{{lang|la|persica}}。

桃是一種營養價值很高的水果，含有蛋白質、脂肪、糖、鈣、磷、鐵和維生素B、C等。桃富含果膠，經常食用可預防便秘。中醫認為，桃仁味甘酸，性微溫，具有補氣養血、養陰生津、止咳殺蟲等功效。桃仁的藥用價值，主要在於桃仁，桃仁中含有苦杏仁甙、脂肪油、揮髮油、苦杏仁酶及維生素B1等。《神農本草經》上有“桃核仁味苦、平。主瘀血血閉，癥瘕邪氣，殺小蟲”之功效。桃仁對治療肺病有獨特功效，唐代名醫孫思邈稱桃仁為“肺之果，肺病宜食之”。{{FACT}}

== 种植 ==
桃是一种[[温带|温带]][[植物|植物]]，需要一定的夏日光照但晚上要比较凉爽的气温及充足的水分和肥料，而且不耐旱。一般用嫁接法种植。把树型要修整为张开的手掌形以使各部分能获得充分的阳光（和[[苹果|苹果]]要有一个向上的中心领导枝不同），桃园和苹果园可以从树型上很容易地区别。

== 品种 ==
桃有多种品种，一般果皮有毛，“油桃”的果皮光滑；“蟠桃”果实是扁盘状；“碧桃”是观赏花用桃树，有多种形式的花瓣，[[北京|北京]]植物园還有几百个不同的品种。

食用的桃还有大久保桃、水蜜桃、绿化九、十四号、白凤等不同品种。<ref>[http://digifruit.tari.gov.tw/content.php?category=%E6%A1%83 農業試驗所果樹種原標本數位化] {{Wayback|url=http://digifruit.tari.gov.tw/content.php?category=%E6%A1%83 |date=20120302183103 }}</ref>

== 文化 ==
桃在[[亚洲|亚洲]]文化中占很大的地位，[[中国|中国]]古代传说经常提到桃是一种可以延年益寿的水果，[[神仙|神仙]]多食用桃，《[[西游记|西游记]]》中[[孙悟空|孙悟空]]在[[天國|天界]]看管的桃园，出产的[[蟠桃|蟠桃]]人吃了可以立刻[[成仙|成仙]]<ref>《西游记》描寫[[孙悟空|孙悟空]]在烂桃山“吃了七次饱桃矣”，祖师道：“你既吃七次，想是七年了”，《大唐三藏取经诗话》记载：猴行者曰：“我因八百岁时，偷吃十颗，被王母捉下，左肋判八百，右肋判三千铁棒，配在花果山紫云洞。至今肋下尚痛。我今定是不敢偷吃也。”。</ref>。[[日本|日本]]神话中有“[[桃太郎|桃太郎]]”。

桃木在中国文化中有避邪的意义<ref>《典術》云：“桃者，五木之精也，故壓伏邪氣者也。桃之精生在鬼門，制百鬼，故今作桃人梗著門以壓邪，此仙木也。”</ref>，相传东海东少山上住着神荼、郁垒二位神仙，手持桃枝，专司捉妖拿邪。通常在门外挂有桃木符或桃木做的对联<ref>《[[说邪|说邪]]》卷十引[[马鉴|马鉴]]《续世始》：“《玉烛宝典》曰：‘元旦造桃板著户，谓之仙木……’即今日桃符也，其上或书神茶、郁垒之字”。</ref>。《[[淮南子|淮南子]]·诠言》说：“羿死于桃棓”。[[许慎|许慎]]注：“棓，大杖，以桃木为之，以击杀羿，由是以来鬼畏桃也”。戰國楚人认为桃树是可以驱鬼辟邪<ref>《左传·昭公十二年》：“昔我先王熊绎辟在荆山，荜露蓝蒌，以处草莽。跋涉山林，以事天子。唯是桃弧、棘矢，以共御王事。”</ref>。《棘楚岁时记》记载南朝楚地风俗：“元旦服桃汤，桃者五行之精，能厌服邪气，制御百鬼。”“正月一日……造桃板著户，谓之仙木，百鬼所畏。”《[[晋书|晋书]]·礼志》记晋代：“岁旦，常设苇索、桃梗、磔鸡于宫及百寺之门以禳恶气。”《[[本草纲目|本草纲目]]》说：“桃味辛气恶，故能厌邪气”。[[道士|道士]]做法时亦常用桃木[[剑|剑]]降妖伏魔，《[[封神榜|封神榜]]》載[[姜子牙|姜子牙]]用桃木劍降妖興[[周朝|周]]。

[[桃花|桃花]]在中國文化中則多與愛情、人緣相關，《[[詩經|詩經]]·国风·周南·桃夭》云：“桃之夭夭，灼灼其華，之子于歸，宜室宜家”，這是一首送新嫁娘歌，用美麗的桃花比喻[[新娘|新娘]]。[[春節|春節]]時會擺放桃花以求行桃花運，參見[[犯桃花|犯桃花]]詞條。而「桃」與「圖」於[[粵語|粵語]]和部份南方方言同音，春節時做生意的人亦喜歡於家中或公司擺放大棵桃花，取其「大展鴻圖」之意。另外在中國形容[[男同性戀|男同性戀]]者的其中一個成語為「[[斷袖|斷袖]][[分桃|分桃]]」<ref>《[[韩非子|韩非子]]·说难》：“异日，与君游于果园，食桃而甘，不尽，以其半啖君。君曰：‘爱我哉！忘其口味以啖寡人。’”《[[汉书|汉书]]·佞幸传第·六十三》：“常与上卧起。尝昼寝，偏藉上袖，上欲起，贤未觉，不欲动贤，乃断袖而起。”</ref>。中国俗语裡的“桃李满天下”则以“桃花”、“[[李树|李花]]”比喻生徒济济，教育的硕果累累<ref>[[武则天|武则天]]称赞[[狄仁杰|狄仁杰]]曰：“天下桃李，悉在公门矣。”[[白居易|白居易]]《奉和令公绿野堂种花》：“ 令公桃李满天下，何用堂前更种花？”</ref>。
