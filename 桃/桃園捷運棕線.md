'''桃園捷運棕線'''為[[桃園捷運|桃園捷運]]計畫路線之一，自1990年代便展開推動，路線能夠銜接[[桃園捷運機場線|桃園捷運機場線]]、[[桃園捷運綠線|桃園捷運綠線]]、[[台鐵捷運紅線|台鐵捷運紅線]]、[[台北捷運中和新蘆線|台北捷運中和新蘆線]]、[[台北捷運萬大樹林線|台北捷運萬大樹林線]]，並串連[[臺北捷運|臺北捷運]]路網。

== 路線 ==
桃園捷運棕線（迴龍線）由桃園車站起，沿桃園市桃園區萬壽路三段與龜山區萬壽路二段、東萬壽路、萬壽路一段、新北市新莊區中正路佈設，最後與[[台北捷運中和新蘆線|台北捷運中和新蘆線]]、[[台北捷運萬大樹林線|台北捷運萬大樹林線]]於[[迴龍站|迴龍站]]交會，路線長11.5公里，共設8座高架車站，軌道機電系統採用[[輕軌鐵路|輕軌捷運系統]]。

目前設有桃園客運及中壢客運營運之[[桃園捷運棕線先導公車|桃園捷運棕線先導公車]]，其起訖站為捷運[[迴龍站|迴龍站]]－[[桃園車站_(臺鐵)|桃園車站]]（代號：'''BR'''）以及延伸服務並有站站停車的601路，起訖點為捷運迴龍站－[[內壢|內壢]]。

== 系統 ==
迴龍線曾考慮採[[單軌鐵路|單軌捷運系統]]，不過後來決定使用[[輕軌鐵路|輕軌運輸系統]]。<ref name=捷運棕線路線規劃>{{cite web|author=桃園市政府捷運工程處|title=捷運棕線路線規劃|url=http://traffic.tycg.gov.tw/Rapid/mrt/brown/intro3.asp=桃園市政府捷運工程處}}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 歷史 ==
*2009年12月：桃園縣（現為桃園市）政府委託臺北市政府捷運工程局辦理「桃園與臺北都會區捷運路網銜接研究」，分析結果認為不宜以捷運新莊線採地下化方式延伸，建議配合捷運新莊線營運通車，行駛桃園-龜山-迴龍之快速直達公車，逐步改變交通行為，吸引私人運具旅次，培養大眾運輸客源。
*2012年12月31日：台鐵林口線年底停駛後，在地方爭取下，先以自行車專用道活化，後再決定是否規劃為棕線輕軌系統。
*2015年10月29日：交通部原則核定棕線（BRH1－BRH8站）之可行性研究，預計2021年通車。
*2016年3月8日：桃園市議會要求桃園市政府交通局就桃園捷運棕線進行專案報告，交通局表示，捷運棕線可行性報告書預計今年底送行政院核定，2019年核定綜合規劃報告書，預計2026年全線通車。不過交通局去年底到市議會報告捷運棕線期程時，是預計2021年通車，相隔3個多月後再度報告，通車日卻要延5年，引發議員林正峰不滿，怒斥交通局信口開河，隨便說說；交通局長高邦基解釋，預計期程會因計畫等因素變動。
*2016年3月21日：捷運棕線可行性研究報告獲交通部原則同意，全線採用高架化，因受限於台一線省道寬度，市府只能採用單軌捷運，將爭取今年底前核定，2026年全線通車營運。
*2018年3月7日：[[中華民國國家發展委員會|國發會]]同意支持本案並送行政院核定，後續交通局將啟動綜合規劃、基本設計、環境影響評估等工作，希望在2019年底獲行政院核定，預計2026年可完工通車<ref>[http://news.ltn.com.tw/news/life/breakingnews/2358392 國發會同意支持桃捷棕線 預計115年完工通車] 2018.3.7 自由時報</ref>。
*2018年5月1日: 行政院核定捷運棕線可行性研究報告。
*2018年5月11日：桃園市議會進行桃園市政總質詢，中國國民黨籍桃園市議員[[林正峰|林正峰]]質詢時，提出桃捷棕線市區市區路段地下化，而龜山萬壽路自救會成員們旁聽見證，並直接邀請市長鄭文燦簽署承諾書，獲得自救會成員一致掌聲；鄭文燦當場簽署承諾書後重申，桃園捷運棕線從桃園縣政府時代即規畫高架化版本，為了考量地方發展，桃園市府決定綜合規畫評估市區路段地下化。<ref>[https://www.youtube.com/watch?v=Buk059ZdAyE 20180511寰宇整點新聞》桃園捷運棕線地下化 鄭文燦簽承諾書]寰宇新聞
2018年5月11日</ref><ref>[http://www.epochtimes.com/b5/18/5/11/n10382643.htm 鄭文燦簽承諾書 桃園捷運棕線納地下化] 大紀元2018年05月11日記者徐乃義</ref>
