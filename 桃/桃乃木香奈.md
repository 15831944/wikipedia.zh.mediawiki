'''桃乃木香奈'''（{{jpn|j=桃乃木かな|rm=Momonogi Kana}}，{{bd|1996年|12月24日}}），[[日本|日本]][[AV女優|AV女優]]。所属于{{tsl|jp|ライフプロモーション|LIFE PROMOTION}}旗下，女子團體[[惠比壽★麝香葡萄|惠比壽★麝香葡萄]]成員，[[東京都|東京都]]出身。

==簡歷==
2015年02月19日，桃乃木用艺名“松嶋真麻”出道，隶属于いもうとChannel公司，当时片商形容她是“肥美大奶萝莉”，但人气不高。

2015年9月24日，桃乃木成為惠比壽★麝香葡萄成員<ref>{{Cite web |author= |date= |url=http://www.oricon.co.jp/news/2059735/full/ |script-title=“2代目”恵比寿★マスカッツお披露目 明日花キララら33人|publisher=ORICON STYLE |accessdate=2016-03-08|language=ja}}</ref>，10月19日以[[IdeaPocket|IdeaPocket]]専属女優身份在AV界出道。

2017年3月3日，桃乃木獲得2017年[[成人廣播獎|成人廣播獎]]新人女優賞獎及[[東京體育|東京體育]]獎<ref name="スカ17">{{Cite web|url=http://news.dmm.co.jp/article/993939/ |script-title=新人女優賞は桃乃木かな!号泣スピーチにファンも感涙。誰からも愛された大食い美少女が桃の節句に戴冠!「今まで応援してくださった方にトロフィーを持ってる姿を見てもらえて嬉しい」【スカパー!アダルト放送大賞2017】|work=DMMニュースR18|date=2017-03-04|accessdate=2017-03-05|language=ja}}</ref><ref>{{Cite web|url=http://dxbeppin-r.com/archives/37487/2 |script-title=【スカパー!アダルト放送大賞2017】女優賞はAIKA! 新人女優賞は桃乃木かな、熟女女優賞は羽月希! リベンジ、ジャイアントキリングと今年も数々のドラマが生まれた春の祭典を詳細レポート! 頁2|work=デラべっぴんR|date=2017-03-05|accessdate=2017-03-06|language=ja}}</ref>。

桃乃木的興趣為一人[[卡啦OK|卡啦OK]]，亦為一個[[大胃王|大胃王]]<ref>{{Cite web |author= |url=http://dxbeppin-r.com/archives/11000 |script-title=【SEXY♥フードファイター降臨!!】AV女優・桃乃木かなさん【特技：大食い】を動画で実証!! 顔よりデカい特大カレー1.5キロに挑戦!!（前編）  |publisher=デラべっぴんR |date=2016-02-09|accessdate=2016-05-13|language=ja}}</ref><ref>{{Cite web |author= |url=http://news.dmm.co.jp/article/986269/?page=all |script-title=桃乃木かな(くいしんぼう)の華麗なる食生活！2月上旬からバレンタインデー14日までお腹の桃袋に入っちゃったごはんをプレイバック♪ラーメン二郎からなんと｢スーパーサイズミー!!｣まで!? |publisher=DMMニュースR18 |date=2016-02-15|accessdate=2016-05-13|language=ja}}</ref>。就讀中學的六年間，她亦參與過[[排球|排球]]部，並曾任該部的部長<ref>{{Cite web |author= |date= |url=http://momonogi.com/questions |script-title=桃乃木かなに100の質問 |work=桃乃木.com |accessdate=2016-03-08|language=ja}}</ref>。
