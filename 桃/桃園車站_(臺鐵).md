'''桃園車站'''位於[[台灣|台灣]][[桃園市|桃園市]][[桃園區|桃園區]]，[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(北段)|縱貫線]][[基隆車站|基隆車站]]起點57.503公里處，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(北段)|縱貫線]]的[[臺灣鐵路管理局車站等級#一等站|一等站]]，旅客進出人次全國第二名，僅次於[[台北車站|台北車站]]，每日往返[[台北|台北]]的通勤旅客極多。因路線進行立體化工程，桃園車站目前使用的站房為臨時站房，總樓地板面積約1萬1917[[平方公尺|平方公尺]]，建造經費[[新臺幣|新臺幣]]4.6億元；未來[[桃園捷運|桃園捷運]][[桃園捷運綠線|綠線]]、[[桃園捷運棕線|棕線]]計畫將與立體化完成後的本站共構。

==概要==
*桃園車站設站始於1893年[[清代|清代]]的縱貫鐵路，稱為桃仔園停車場。
*1905年[[日治時代|日治時代]]將縱貫線改線，在現址建造木造站體，並更名為桃園驛。桃園驛的位置是當時二十廳時代警察哨的所在地，也是一個極為熱鬧且富麗的農產品買賣集散地。
*1962年改建為混凝土建築站房，使用至2015年7月，因新臨時站房啟用後停用。
*2010年開始[[台鐵|台鐵]][[桃園鐵路地下化|桃園鐵路地下化]]動工，由台灣世曦設計監造，榮金營造承攬，桃園車站也將隨之立體化，將興建2座[[島式月台|島式月台]]4股道<ref>{{Cite web |url= http://search.railway.gov.tw/EAY980521/doc/tyA110.pdf |date=2012-12-24 |title= 桃園市用地徵收公聽會說明簡報資料 |publisher= 臺灣鐵路管理局 |language= zh-tw }}</ref>。[[桃園捷運|桃園捷運]][[桃園捷運綠線|綠線]]、[[桃園捷運棕線|棕線]]亦計畫在地下層與本站共構。
*為了因應車站立體化施工，2015年7月26日原本的桃園車站啟用[[跨站式站房|跨站式站房]]的臨時車站<ref>[http://www.cna.com.tw/news/firstnews/201507265003-1.aspx 台鐵桃園舊站熄燈 新站啟用 中央社即時新聞]</ref>，新建臨時車站主體建築位於原後站方向（大林路、延平路），原前站方向則設立兩處出入口。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#一等站|一等站]]。
**[[普悠瑪列車|普悠瑪號]]：南下停靠111次、271次(週日停駛)、6271次(週日行駛)，北上停靠136次、288次。
***不停靠南下127次、273次(週日停駛)、283次(週日停駛)、6273次(週日行駛)、6283次(週日行駛)；北上110次、280次、282次(週日停駛)、6282次(週日行駛)。
**其餘各級列車除每周五南下143次[[自強號列車|自強號]]過站不停外其餘各級列車(包含[[太魯閣列車|太魯閣號]]和[[觀光列車|觀光團體列車]])每班都有停靠。
**[[臺鐵區間車|區間車]]1105次以本站為起站、4209次以本站為終點站
*2015年旅客人次在所有台鐵車站排名第二（僅次於[[台北車站|台北車站]]）；而在營收則排名第十（次於[[台北車站|台北]]、[[花蓮車站|花蓮]]、[[臺中車站|台中]]、[[高雄車站|高雄]]、[[臺南車站|台南]]、[[新竹車站|新竹]]、[[臺東車站|台東]]、[[板橋車站_(臺灣)|板橋]]、[[中壢車站|中壢]]）。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。
*本站原另有「新生麵粉公司支線」專辦貨運，但因臺鐵都會區捷運化桃園段高架化建設計畫，目前已廢線並拆除。

== 歷史 ==
[[File:1904台灣堡圖.png|250px]]
[[File:桃園驛.jpg|250px]]
[[File:桃園驛模型.jpg|250px]]]]
*1893年10月30日：清代-[[台北車站|臺北]]－[[新竹車站|新竹]]間鐵路通車時即設置'''桃仔園火車碼頭'''，日治後改稱'''桃仔園停車場'''。
*1905年7月1日：縱貫線改線、車站站體重建，並更名'''桃園停車場'''；<ref>臺灣史料稿本明治三十八年七月一日</ref>站體建築在站房正反兩端立面增蓋[[唐式閣樓|唐式閣樓]]，形狀如一頂帽子。由十對雙併木柱連成九座拱門，是為洋風特徵；但屋頂又採日式結構，為[[明治|明治]]三十年代全[[日本|日本]]木造驛舍建築的典型，1918年落成的[[萬華車站|艋舺車站]]也是同類型的建築風格，相較同時期的同類型日本車站為早<ref>{{Cite web |url= http://www.railway.gov.tw/Taoyuan/CP.aspx?SN=13480 |title= 桃園車站懷舊之旅 |publisher= 台灣鐵路管理局 |language= zh-tw }}</ref>。
*1920年：桃園車站改隸新竹州桃園郡，桃園站成為從台北進入新竹的第一大站。
*1962年：拆除原先木造站房，重建為今日所見之站房。
*1963年2月8日：重新落成啟用。
*1981年5月1日：改為一等甲站，至今仍為一等站。
*2005年10月27日：林口線開始試辦免費客運<ref>{{Cite web|url=http://railnews.tw/rn_index/210|title=RN210 2012 年 11 ~ 12 月號|accessdate=2015-08-01|publisher=[[鐵道情報|鐵道情報]]}}</ref>。
