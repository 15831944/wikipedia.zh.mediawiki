'''叶夢'''（原名：'''ゆいの、仓持结爱、桃濑友梨奈'''，{{bd|1990年|8月19日}}），出生於[[东京都|东京都]]，前[[日本|日本]][[AV女優|AV女優]]、現任為寫真女星。<ref name="prime">{{Cite web|author=|url=http://www.prime-recruit.com/girlInfo.php?id=75|title=公式プロフィール|publisher=プライムエージェンシー|accessdate=2013-12-02|archiveurl=https://web.archive.org/web/20131203083312/http://www.prime-recruit.com/girlInfo.php?id=75|archivedate=2013年12月3日}}</ref>

== 簡歷 ==
2013年4月7日，在[[S1_(AV片商)|S1]]以《新人NO.1 STYLE H-cup アイドルデビュー》AV作为S1专属女优“'''仓持结爱'''”出道。

2014年10月改名为“'''ゆいの'''”，并成为[[E-BODY|E-BODY]]专属女优，同时也参与出演[[Attackers|Attackers]]所制之片。

2015年6月15日，在[[推特|推特]]上宣布改名为“'''桃濑友梨奈'''”。所属经纪公司改为{{link-ja|ディアスグループ}}<ref>{{Cite web |author= |date= |url=http://www.lantana-pro.com/momoseyurina.html |title=桃瀬友梨奈 |work=MODELS |publisher=LANTANA |accessdate=2016-09-10}}</ref>。改名后仍作为E-BODY专属女优而出演。

2016年12月2日，在推特上宣布引退，并作为前AV女优而继续活跃。

2018年7月27日，已 叶夢 復出並且發表寫真新作《First dream》

== 爱好与特长 ==
喜爱逛街、烹饪、观赏演唱会。特长是制作甜点。<ref name="S1"/>
