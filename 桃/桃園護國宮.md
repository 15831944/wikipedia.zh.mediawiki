'''桃園護國宮'''，是[[桃園市|桃園市]][[桃園區|桃園區]]國際路一段401巷的一座廟宇。該宮舊址位於桃園市民安路，其後遷至此地。主祀「中壇元帥哪吒三太子爺公」。

== 建廟歷程 ==
民國81年遷廟到現址、興建臨時大殿。護國宮現今建地3500坪，以座東朝西的方位興建。民國95年農曆四月初八，正好是『太子爺公蓮花化身顯聖紀念日』舉行動土開工儀式。民國98年舉行大廟上樑大典。<ref>{{Cite news|url=http://blog.xuite.net/o927603252/twblog/512146876-%E5%AF%BA%E5%BB%9F%E5%B7%A1%E7%A6%AE-%E6%A1%83%E5%9C%92%E5%B8%82%E6%A1%83%E5%9C%92%E5%8D%80%E8%AD%B7%E5%9C%8B%E5%AE%AE%E4%B8%AD%E5%A3%87%E5%85%83%E5%B8%A5|title=寺廟巡禮／台灣最大哪吒三太子廟　桃園護國宮|author=|date=|work=臺灣導報|accessdate=}}</ref>

民國99年農曆11月13日由法道心師父<ref>{{Cite web|url=http://hkk.org.tw/html/index/3.html|title=法道心師父的故事|accessdate=|author=桃園護國宮|date=|publisher=桃園護國宮}}</ref>開廟門，舉辦遷火入廟儀式。近百尊神佛由600多位善男信女吃素3天，以接力方式恭請神尊入新廟，代表延壽、納福、除穢之意義。<ref>[http://news.ltn.com.tw/news/local/paper/453180 〈北部〉護國宮遷火 600人接力迎百尊神像]</ref>

== 太子遊樂園 ==
護國宮在每年農曆大年初一到初五舉辦全台唯一「太子遊樂園」。民眾攜帶小孩到護國宮新春參拜時，可讓小孩免費體驗旋轉木馬、旋轉鞦韆、海盜船、熱氣球、地震屋、大型充氣氣墊屋、釣鴨子池、投籃機等遊樂設施，讓大朋友、小朋友放鬆身心，這是宗教信仰的轉換，讓信仰生活化，並結合文創產業，也是寺廟文化未來發展的新方向。<ref>[https://www.tycg.gov.tw/ch/home.jsp?id=9&parentpath=0,1&mcustomize=news_view.jsp&dataserno=201701230006&aplistdn=ou=news,ou=chinese,ou=ap_root,o=tycg,c=tw&toolsflag=Y 護國宮太子遊樂園新年走春記者會 鄭市長：重現傳統文化，邀請市民走春參香]</ref><ref>{{Cite news|url=https://www.youtube.com/watch?v=6xuAijCWTpw|title=桃園護國宮『太子廟』2018戊戌年【中天娛樂台-生活萬事通】Yuki 主持|author=|date=|work=|accessdate=}}</ref>

== 先鋒聖典蟠龍宴 ==
護國宮在每年農曆四月初六至初八日舉辦《先鋒盛典蟠龍宴 》歡喜迎太子 · 坐客聚神威，敬邀各宮廟及善信大德府上有供奉『中壇元帥哪吒三太子公』一同來到桃園護國宮太子廟參與，恭請至本宮作客，共享萬年香火，同沾神恩聖德。

三日蟠龍大宴各有不同主題

《翌日大宴 :《[[凤凰|鳳凰]]鳴[[岐山县|岐山]]》》

== 太子忠孝文化季 ==
[[File:夜巡暗訪行經桃園高城社區盛況.jpg|缩略图]]
有別於一般宮廟神明白天出巡，護國宮太子爺是「夜巡暗訪」。自2014年起，廟方於每年秋季舉辦「太子忠孝文化季」節慶活動，護國宮太子爺神像一連數天夜巡[[桃園區|桃園區]]、[[中壢區|中壢區]]及[[八德區|八德區]]等鄰近地區，並駐駕相關宮廟。<ref>[https://udn.com/news/story/7324/2753443 桃園護國宮 18日起連5天夜巡暗訪]</ref>
