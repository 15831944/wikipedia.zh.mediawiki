'''町村信孝'''{{BD|1944年|10月17日|2015年|6月1日|Machimura, Nobutaka}}，[[日本|日本]][[政治家|政治家]]，曾任第75任[[眾議院_(日本)|眾議院]]議長。

町村信孝出生在[[靜岡縣|靜岡縣]][[沼津市|沼津市]]，籍貫是[[北海道|北海道]]的[[江別市|江別市]]，曾就讀[[東京學藝大學附屬世田谷小學校|東京學藝大學附屬世田谷小學校]]、[[東京學藝大學附世田谷中學校|東京學藝大學附世田谷中學校]]、[[東京都立日比谷高等學校|東京都立日比谷高等學校]]，並於1969年自[[東京大學|東京大學]]經濟學部畢業。在東京大學就讀期間，曾經以[[交換學生|交換學生]]身份前往[[美國|美國]][[衛斯連大學|衛斯連大學]]留學。

他是[[日本自由民主党|自由民主党]][[清和政策研究會|町村派]]的要角。父親是前[[日本眾議院|日本眾議院]]、[[日本參議院|參議院]]議員及[[北海道|北海道]][[知事|知事]][[町村金五|町村金五]]，前[[日本參議院議長|日本參議院議長]][[原文兵衛|原文兵衛]]是他的從兄。他曾八次參選眾議院選舉，都成功當選。1969年，大學畢業後，他進入日本政府的[[經濟產業省|通產省]]工作。1974年，任職於[[國土廳|國土廳]]，1969年，任職於日本貿易振興會。1983年，首次當選眾議員。後任文部省政務次官，在[[第2次橋本改造内閣|第2次橋本改造内閣]]中首次入閣擔任文部大臣。其後，在[[小淵惠三|小淵惠三]]內閣中擔任外務政務次官。

他從2004年9月27日開始擔任日本外相。任內成功與[[俄羅斯|俄羅斯]]簽署有關[[北方四島|北方四島]]（南千島群島）的邊境糾紛條約，以及調查日本在1970年代及1980年代被[[北韓|北韓]]擄走的日本人的事件。2005年4月17日，他在中日关系低潮的时刻访问[[中国|中国]]。

2007年9月出任[[内閣官房長官|内閣官房長官]]。2014年12月24日擔任眾議院議長。

2015年4月，因健康因素請辭議長。同年6月1日在東京都逝世，享年70歲。

== 簡歷 ==
*1969年6月：大學畢業；7月：[[經濟產業省|通商産業省]]入省。
*1974年：任職於國土廳。
*1979年：任職於日本貿易振興會。
*1982年4月 ：自通商産業省退職。
*1983年：首次參選[[日本眾議院|日本眾議院]]議員，並成功當選。
*1989年6月－1990年2月：文部省政務次官。
*1990年3月：自由民主党文化局長。
*1997年9月11日－1998年7月30日：[[文部科學省|文部大臣]]。
*1998年7月31日－1999年10月5日：外務省政務次官（從1999年9月20日擔任外務省總括政務次官）。
*2000年3月1日－7月4日：內閣總理大臣補佐官（教育改革擔當）。
*2000年12月5日－2001年1月5日：文部大臣兼科學技術廳長官。
*2001年1月6日－4月26日：[[中央省廳再編|中央省廳再編]]，擔任首位[[文部科學省|文部科学大臣]]。
*2001年4月下旬－2002年10月上旬：自由民主党幹事長代理。
*2002年10月－2004年9月：自由民主党総務局長。
*2004年9月27日－[[2005年10月|2005年10月]]：出任第二.三次小泉内閣[[外務省|外務大臣]]。
*2006年10月：出任[[清和政策研究会|清和政策研究会]]会長 。
*2007年8月－9月：出任安倍改造内閣的外務大臣 。
*2007年9月－2008年9月：出任福田内閣的官房長官。
*2008年9月：代理農林水産大臣。
*2012年9月：[[2012年自由民主黨總裁選舉|參選自民黨總裁]]，結果落選。
*2014年12月：出任眾議院議長。
*2015年4月：請辭眾議院議長。
*2015年6月1日：逝世。

== 家族、親族 ==
*祖父：[[町村金彌|町村金彌]]、外祖父：[[桑田熊蔵|桑田熊蔵]]（學者）。
*父：[[町村金五|町村金五]]（官僚､政治家）。
*母：二葉（[[鳥取縣|鳥取縣]]，學者[[桑田熊藏|桑田熊藏]]的女兒）。
*妻：淳子。
*伯父：[[町村敬貴|町村敬貴]]（政治家、[[牧場|牧場]]主：[[まちむら農場|町村農場]]創設者）。
*其他親戚：[[原文兵衛|原文兵衛]]（官僚、政治家：元[[参議院議長|参議院議長]]）、[[河合良一|河合良一]]（實業家）等。

== 世系圖 ==
*町村氏：祖父[[町村金彌|町村金彌]]本家為[[武士|武士]]家系，先祖曾入仕為[[江戶時代|江戶時代]]､[[越前藩|越前藩]]之支藩･府中領主[[本多氏|本多家]]為武士。金彌曾在[[札幌農學校|札幌農學校]]就學､之後在[[北海道|北海道]]開拓事業生涯、以｢北海道酪農之父｣的稱號聞名。父親[[町村金五|町村金五]]（原[[衆議院議員|衆議院議員]]、[[北海道|北海道]][[知事|知事]]、[[參議院議員|參議院議員]]、[[自治大臣|自治大臣]]）、堂姐夫[[原文兵衛|原文兵衛]]（原[[參議院議長|參議院議長]]､[[警視總監|警視總監]]）。
