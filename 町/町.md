'''町'''意為田界、田間小路、田地<ref>《重編國語辭典修訂本》網路版</ref>，在[[中國|中國]]亦用於地名，如[[湖南省|湖南省]][[衡阳县|衡阳县]][[三湖鎮_(衡陽縣)|三湖镇]]，即出自当地存在上湖町、中湖町和下湖町，当地人又合称“三湖町”<ref>{{cite web |url=http://www.hyx.gov.cn/xshz/link2.htm |title=存档副本 |accessdate=2011-01-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20111123233939/http://www.hyx.gov.cn/xshz/link2.htm |archivedate=2011-11-23 }} 三湖镇介绍，衡阳县政府公众信息网</ref>。《南史·列傳·侯景傳》有載「先是江陵謠言：『苦竹町，市南有好井。荊州軍，殺侯景。』及景首至，元帝付諮議參軍李季長宅，宅東即苦竹町也。」其中的「苦竹町」即地名之例{{r|劉}}。古代又将其作为[[地积|地积]]单位，《左传·襄公二十五年》有“町原防，牧隰皋，井衍沃”；《禮記註疏》有「偃豬之地，九夫為規，四規而當一井；原防之地，九夫為町，三町而當一井。」{{r|劉}}

== 日文中的“町” ==
今天“町”这一汉字在[[日文|日文]]中的使用频率远超过[[中文|中文]]，日文的“町（{{lang-ja|通常[[訓讀|訓讀]]まち，個別情況下[[音讀|音讀]]ちょう}}）”主要有以下意涵：
* [[町_(行政區劃)|町 (行政區劃)]]
* [[面積單位|面積單位]]。1町=10[[反|反]]=3000[[坪|坪]]，約9917[[平方公尺|平方公尺]]，約等於一[[公頃|公頃]]（一甲約2934坪，一公頃約3025坪）。
* [[長度單位|長度單位]]。1町=60[[間|間]]，約109.09[[公尺|公尺]]。
