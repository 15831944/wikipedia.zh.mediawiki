《'''蠟筆小新：宇宙人Pi力來襲！！'''》（{{lang-ja|クレヨンしんちゃん 襲来!! 宇宙人シリリ}}）是于[[日本|日本]]2017年4月15日上映的《[[蜡笔小新|蜡笔小新]]》第25部电影版。本作並為蠟筆小新電影版25周年紀念作品。

==概要==
*本作繼《[[蠟筆小新：動感超人大戰泳裝魔王|動感超人大戰泳裝魔王]]》、《[[蠟筆小新：小白的屁屁炸彈|小白的屁屁炸彈]]》及《[[蠟筆小新：我和我的宇宙公主|我和我的宇宙公主]]》後再度以「'''宇宙人'''」作為主題的作品。
*導演及腳本由2015年的《[[蠟筆小新：我的搬家物語_仙人掌大襲撃|我的搬家物語 仙人掌大襲撃]]》導演[[橋本昌和|橋本昌和]]擔任<ref>{{Cite web |url=http://www.cinematoday.jp/page/N0086757 |title=『映画クレヨンしんちゃん』25周年　最新作は宇宙人襲来！某光る棒もあるよ|publisher=cinematoday|date=2016-10-14|language=ja}}</ref>。
*由於本作為蠟筆小新電影版25周年紀念作品，過去24部電影版的人物和相關物品皆以客串方式出現在電影中<ref>{{Cite web |url=https://www.cinematoday.jp/news/N0091110|title=隠れしんちゃん…？映画『クレしん』最新作に過去24作キャラがゲスト出演！|publisher=cinematoday|date=2017-04-20|language=ja}}</ref>。
*'''野原廣志'''的配音員[[藤原啟治|藤原啟治]]仍在生病中，本作為[[森川智之|森川智之]]代理後首次在电影版中飾演野原廣志。
*一般小新劇場版登場的角色到最後都會跟野原一家道別，Pi力是少數在主線劇情結束後還跟野原一家生活過一段時間的劇場版角色。

==故事==
有一天，突然出現一個神秘的小小宇宙人「Pi力」，因照射到宇宙人的神秘光線，廣志和美冴竟然返老還童！為了變回大人，萌孩版的野原一家人踏上冒險旅程，小新甚至為了Pi力封印了屁屁，為的就是幫Pi力找到爸爸…
