《'''蠟筆小新：功夫小子〜拉麵大亂鬥〜'''》（{{lang-ja|クレヨンしんちゃん 爆盛!カンフーボーイズ ～拉麺大乱～}}）是《[[蠟筆小新|蠟筆小新]]》第26部[[動畫電影|動畫電影]]，[[日本|日本]]於2018年4月13日上映。

== 概要 ==
* 導演由曾執導《[[蠟筆小新：大對決！機器人爸爸的反擊|大對決！機器人爸爸的反擊]]》和《[[蠟筆小新：爆睡！夢世界大作戰|爆睡！夢世界大作戰]]》的[[高橋涉_(動畫演出家)|高橋涉]]擔任<ref>{{Cite web |url=https://www.cinematoday.jp/news/N0095276|title=映画クレしん新作は『カンフーボーイズ』！ロボとーちゃん監督が手掛けるアクション作に|publisher=Cinematoday|date=2017-10-13|language=ja}}</ref>，腳本由曾負責《[[蠟筆小新：超級美味！B級美食大逃亡！！|超級美味！B級美食大-{}-逃-{}-亡！！]]》和《[[蠟筆小新：我的搬家物語_仙人掌大襲擊|我的搬家物語 仙人掌大襲擊]]》的[[上野貴美子|上野貴美子]]擔任。
* 本作是首次以「功夫」為主題，並以[[春日部市|春日部市]]的[[中華街|中華街]]「哎呀街」為舞台的作品<ref>{{Cite web |url=http://eiga.com/news/20171013/20/|title=映画「クレしん」第26弾はカンフーが題材！しんのすけが“ノースタントアクション”|publisher=映画.com|date=2017-10-13|language=ja}}</ref>。
* 本作是繼2016年《爆睡！夢世界大作戰》，相隔2年再度以「春日部防衛隊」為故事主角的電影動畫。
* 本作是首次以「春日部防衛隊」的正男和小新成為共同主角。
* 本作是矢島晶子最後一次聲演野原新之助的蠟筆小新劇場版。

== 票房 ==
在日本方面，總票房為'''18.3億'''日圓,並在首日上映得到接近'''4億'''(3.9億)票房佳績,在'''日本黄金周'''(5月12日-5月13日)更達到'''5.19億'''的票房,更達到了'''動員124萬人'''觀看該電影,是歷史上第五高的蠟筆小新電影票房紀錄。

在香港方面，首周票房'''83萬'''港元（含優先場）,一周動員'''2.4萬人'''觀看該電影。

在台灣方面,總票房為'''614萬'''新台幣,首周票房'''41萬''',一周動員'''2萬人'''觀看該電影。
