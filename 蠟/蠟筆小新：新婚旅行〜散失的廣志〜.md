《'''蠟筆小新：新婚旅行〜散失的廣志〜'''》（{{lang-ja|クレヨンしんちゃん 新婚旅行ハリケーン ～失われたひろし～}}）是《[[蠟筆小新|蠟筆小新]]》第27部[[動畫電影|動畫電影]]，[[日本|日本]]於2019年4月19日上映。

宣傳標語有: 「傳說的秘密, 危機100出發!!!」「我沒有聽到這樣的事情.」「野原一家, 愛的試練.」等。

== 概要 ==
* 本作是首次以'''家庭愛'''為主題的電影，並主要想發揮出'''野原一家'''的感情生活和點滴的電影！
*導演由曾執導《[[蠟筆小新：宇宙人Pi力來襲！！|宇宙人Pi力來襲！！]]》的[[橋本昌和|橋本昌和]]擔任<ref>{{Cite web |url=https://www.fashion-press.net/news/44471|title=『映画クレヨンしんちゃん 新婚旅行ハリケーン ～失われたひろし～』野原一家がまさかのトレジャーハント|publisher=Fashion Press|date=2018-11-02|language=ja}}</ref>。腳本由[[上野貴美子|上野貴美子]]和{{link-ja|水野宗徳|水野宗徳}}擔任<ref>{{Cite web |url=https://natalie.mu/eiga/news/306391|title=「映画クレヨンしんちゃん」第27弾の公開決定、野原家が今さら新婚旅行|publisher=[[Natalie_(網站)|Natalie]]|date=2018-11-01|language=ja}}</ref>。
* 本作是首次以「[[新婚旅行|新婚旅行]]」為主題，並以[[澳洲|澳洲]]為舞台的作品<ref>{{Cite web |url=http://eiga.com/news/20181102/21/|title=「映画クレしん」第27作は野原一家初の新婚旅行＆ひろしが“お宝のカギ”に？　19年4月19日公開|publisher=映画.com|date=2018-11-02|language=ja}}</ref>。
* 本作為[[小林由美子|小林由美子]]接任後首次在電影版中聲演野原新之助。
* 本作為蠟筆小新系列電影在[[平成時代|平成時代]]上映的最後一部劇場版。
* 本作是繼2017年《蠟筆小新：宇宙人Pi力來襲！！》，相隔2年再度以「野原一家」為故事主角的電影動畫。

== 故事==
美冴和廣志結婚後注意到他們沒有度過蜜月，而美冴曾經有一個澳大利亞朋友，所以他們與小新一起乘坐飛機並找到澳大利亞家庭陪同一起的廉價蜜月之旅！

但是，這是一個危險的冒險，原來小新的爸爸受到假面族的威脅並帶走！小新的媽媽和小新正式去拯救爸爸去解決假面族的威脅⋯⋯
