'''闻一多'''（{{bd|1899年|11月24日|1946年|7月15日|闻一多}}），本名'''家骅'''，[[表字|字]]'''友三'''，[[湖北省|湖北省]][[黄冈市|黄冈市]][[浠水县|浠水县]]人，中国[[诗人|诗人]]、[[学者|学者]]、[[爱国主义|爱国主义]]者和[[民主主义|民主主义]]者。因家传渊源，自幼爱好古典诗词和[[美术|美术]]。

== 生平 ==
*1910年闻一多与堂兄们来到武昌求学，就读于[[两湖总师范学堂|两湖总师范学堂]]附属高等小学校。[[中华民国|中华民国]]建立后，闻一多短期就读于武昌[[民国公校|民国公校]]和[[实修学校|实修学校]]。
*1912年以湖北籍第二名考入北京[[清华学校|清华学校]]中等科一年级，喜读中国古代诗集、诗话、史书、笔记等。1916年开始在《清华周刊》上发表系列读书笔记，总称《[[二月庐漫记|二月庐漫记]]》。同时创作[[旧体诗|旧体诗]]。1919年[[五四运动|五四运动]]时积极参加[[学生运动|学生运动]]，曾代表学校出席全国学联会议（上海）。

*1920年4月，发表第一篇[[白话文|白话文]]《旅客式的学生》。同年9月，发表第一首[[新诗|新诗]]《西岸》。

*1921年清华学校毕业，11月与[[梁实秋|梁实秋]]等人发起成立清华文学社，次年3月，写成《[[律诗的研究|律诗的研究]]》，开始系统地研究新诗格律化理论。

*1922年7月赴[[美国|美国]][[芝加哥艺术学院|芝加哥艺术学院]]学习。年底出版与梁实秋合著的《[[冬夜草儿评论|冬夜草儿评论]]》，代表了闻一多早期对新诗的看法。1923年出版第一部诗集《[[红烛|红烛]]》，把[[反帝國主義|反帝]]爱国的主题和[[唯美主义|唯美主义]]的形式典范地结合在一起。1925年5月回国后，历任[[国立第四中山大学|国立第四中山大学]]（1928年更名为中央大学，1949年更名为[[南京大学|南京大学]]）、[[国立武汉大学|国立武汉大学]]（任文学院首任院长并设计校徽）、[[国立青岛大学|国立青岛大学]]（现[[山东大学|山东大学]]）、[[北平艺术专科学校|北平艺术专科学校]]、[[政治大学|政治大学]]、[[清华大学|清华大学]]、[[西南联合大学|西南联合大学]]教授，曾任北京艺术专科学校教务长、南京第四中山大学外文系主任、[[武汉大学|武汉大学]]文学院长、国立青岛大学（后改为[[国立山东大学|国立山东大学]]）文学院长。

*1925年7月4日发表《[[七子之歌|七子之歌]]》，写出中国被列强掳掠的七片土地的声音。

*1928年出版第二部诗集《[[死水_(詩集)|死水]]》，在颓废中表现出深沉的[[爱国主义|爱国主义]]激情。此后致力于[[古典文学|古典文学]]的研究。对《[[周易|周易]]》、《[[诗经|诗经]]》、《[[庄子|庄子]]》、《[[楚辞|楚辞]]》四大古籍的整理研究，被[[郭沫若|郭沫若]]称为“前无古人，后无来者”。　

*1937年[[中國抗日戰爭|中國抗日戰爭]]开始，他在昆明西南联大任教。抗战八年中，他蓄起一把胡子，发誓不取得抗战的胜利不剃去。1943年后，目睹了[[国民政府|国民政府]]的一些腐败现象，遂积极参加反对[[独裁|独裁]]，争取[[民主|民主]]的斗争。

*1943年，时在中共南方局任宣传部长的[[华岗|华岗]]，被派到昆明，担任[[共产党|共产党]]南方局与“云南王”[[龙云|龙云]]之间的联络人。不久，[[华岗|华岗]]与闻一多正式见面并成为朋友。 1945年10月，[[华岗|华岗]]返回重庆前，对闻一多说，现在形势变了，要注意保存革命力量，不要采取极端措施。闻一多记住了这些叮嘱，在[[一二一惨案|一二一惨案]]之前，并不赞成学生罢课，因为他认为这会过于暴露自己，不符合华岗说的保存力量的指示。<ref>{{cite web |url=http://news.china.com/zh_cn/history/all/11025807/20100708/16015763.html |title=闻一多把共产党看得神秘：从华岗开始有所认识 -- 《大众日报》}}</ref>

[[File:闻一多01.jpg|thumb]]

*1945年为[[中国民主同盟会|中国民主同盟会]]委员兼[[云南省|云南省]]负责人、昆明《[[民主周刊|民主周刊]]》社长。[[一二一事件_(中国)|一二一惨案]]发生后，他投身民主运动，反对[[蔣中正|蔣中正]]的独裁统治。

*1946年7月15日在悼念被暗杀的[[李公朴|李公朴]]的大会上发表了《[[最后一次讲演|最后一次讲演]]》，当天下午在西仓坡宿舍门口即被[[中國國民黨|國民黨]]昆明警备司令部下级军官[[汤时亮|汤时亮]]和[[李文山|李文山]]枪杀，聞一多之子[[闻立鹤|闻立鹤]]亦身受重傷。聞一多被暗殺後，举世震惊，當時在[[庐山|庐山]]的蔣中正也知道問題的嚴重性，下令[[唐縱|唐縱]]徹查，暗殺事件很快就破案，李文山和汤时亮由宪兵司令部举行公开军法审讯，两人经审讯后被枪决<ref>{{cite book|
title=特工秘闻: 军统活动纪实|
url=http://books.google.com/books?id=n6mAAAAAIAAJ|
year=1990|publisher=中国文史出版社 | author=程一鸣 | chapter=李公朴、闻一多暗杀案内幕 | pages=181-184}}</ref>。昆明警察局長龔少俠也因此被撤職。事實上是雲南警備總司令[[霍揆彰|霍揆彰]]安排了兩名死囚充當凶手被槍決，真兇早已逍遙法外<ref>[[谢泳|谢泳]]：《闻一多之死》</ref>。台灣學者[[陳永發|陳永發]]表示：“闻一多遭暗杀事件，是[[国共内战|国共内战]]转折的重要关键。当时国民政府处理不当，被批为[[法西斯主義|法西斯]]独裁，让红色政权赢得知识分子、学生支持，甚至连国际舆论、支持也开始转向。”<ref>《联合报》2009年11月25日</ref>陳永發表示：整個暗殺事件前後浮現出特務嚴重的問題，蔣中正已無法精準的掌控整個特務情報系統<ref>《[[传记文学|传记文学]]》雜誌，2010年8月</ref>。

*1948年8月，闻一多的遗著——《闻一多全集》出版，由[[朱自清|朱自清]]主编，[[雷海宗|雷海宗]]、[[潘光旦|潘光旦]]、[[吴晗|吴晗]]、[[浦江清|浦江清]]、[[许维啭|许维啭]]、[[余冠英|余冠英]]协助完成。<ref>{{cite web |url=http://www.edualpha.net/article-39356-1.html |title=朱自清与《闻一多全集》 |deadurl=yes |archiveurl=https://web.archive.org/web/20110909144053/http://www.edualpha.net/article-39356-1.html |archivedate=2011-09-09 }}</ref>

==文學主張==
聞一多主張[[新詩|新詩]]的格律詩要講求音樂美，重視音尺整齊，韻律諧協；要講求繪畫美，務使句雕字琢，嚴謹精煉；要講求建築美，追求節的勻稱，句的均齊。

==詩風==
聞一多[[新詩|新詩]]具強烈民族意識，流露出濃厚的愛國感情，從超現實而充滿浪漫氣息的生活中，轉到滿腔熱血，全部奉獻給他所忠於的事業。[[新月派|新月派]]作家中，只有聞一多才顯出強烈而迫切的愛國精神和正義感。

聞一多早期詩歌多屬自由體，後期詩歌形式整齊。他較早提出新詩的[[格律|格律]]問題。他的新詩具音樂美，講究音尺、[[平仄|平仄]]和[[押韻|押韻]]，重視音尺的節奏和旋律的美，押韻方式豐富多采；具繪畫美，形象豐富，色彩穠麗；具建築美，重視節的對稱和句的均齊，主張詩節勻稱和詩句整齊。

聞一多新詩善於創造[[比喻|比喻]]，想象力豐富，善用[[擬人法|擬人法]]，強調暗示，意在言外，善於使用典故，增強了詩的繁富，語言精鍊，精於鍊字，善於自鑄新詞，並在實踐中作可貴嘗試。他依西洋[[格律|格律]]而創作，但跟[[徐志摩|徐志摩]]不同的，是他的詩歌氣勢雄渾豪邁，善用北方口語。

== 评价 ==
[[File:闻一多纪念馆入口.jpg|right]]。]]瑞典著名[[汉学家|汉学家]]、[[诺贝尔文学奖|诺贝尔文学奖]]终审评委[[马悦然|马悦然]]认为：“闻一多，1946年被枪杀了。他的《死水》和《红烛》很好，非常好。他的《死水》我认为是非常伟大的作品——这是／一沟／绝望的／死水，清风／吹不起／半点／漪沦。《死水》是闻一多在诗歌构建方面最成功的实验，是[[五四运动|五四运动]]期间诗歌中最悲哀的一首诗，是现代中国文学中韵律最完美的挽歌式的诗歌。他的诗有一种建筑的美，他是个诗歌建筑家，他的诗歌都有一个美丽的形式，非常好。他有一些短诗诗意很像唐朝时代的绝句。闻一多不光是伟大的诗人，也是一位杰出的学者，他是五四运动之后非常杰出的作家。他还有一首诗《闻先生的书桌》，写得非常好，写他书桌上的笔墨、纸砚，他看着那些东西就开始发牢骚。他的诗歌都是用民间语言写出来的，像《飞毛腿》，完全是用北京的拉车夫的语言写的——“我说飞毛腿那小子也真够别扭，管保是拉了半天车半天歇着”，这首诗非常好。”<ref>{{cite web |url=http://cul.sohu.com/20071010/n252580088.shtml|title=《沈从文如果活着就肯定能得诺贝尔文学奖》|accessdate=2007年10月10日16|work=|publisher=搜狐网|language= zh-hans}}</ref>
