'''牟宗三'''（{{bd|1909年|6月12日|1995年|4月12日|catidx=Mou牟}}）（{{lang|en|Prof. MOU Tsung San }}），字'''離中'''，現代思想家、哲學家、教育家，是新儒家學派代表人物。

== 生平 ==
牟宗三1909年生於[[山東省|山東省]][[棲霞縣|棲霞縣]]牟家疃，祖籍[[湖北省|湖北省]][[公安縣|公安縣]]。牟氏9歲入私塾讀書，11歲入新制小學，15歲入棲霞縣立中學。1928年，考入[[北京大學|北京大學]]預科，兩年後升哲學系，1933年畢業。大學時代先後受學於[[張申府|張申府]]、[[金岳霖|金岳霖]]，受[[熊十力|熊十力]]影響最大，受其《新唯識論》義理之震撼，並維持幾十年的師生情誼。<ref name="hkuhongrads">{{ cite web |url=https://www4.hku.hk/hongrads/index.php/chi/archive/graduate_detail/176 |title=第130屆頒授典禮 (1987)牟宗三讚詞 |date= |publisher=香港大學 |author= }}</ref>

北大哲學系畢業後，牟氏先後任教於山東[[壽張縣|壽張縣]]鄉村師範、廣州學海書院、山東鄒平村治學院、廣西梧州中學、南寧中學、[[華西大學|華西大學]]、[[中央大學|中央大學]]、[[金陵大學|金陵大學]]、[[浙江大學|浙江大學]]，以講授邏輯與西方文化為主。三十年代，曾主編《歷史與文化》、《再生》雜誌等。1949年，牟氏到台灣[[國立臺灣師範大學|國立臺灣師範大學]]與[[東海大學_(台灣)|東海大學]]任教。1954年受聘為台灣教育部學術審議委員。<ref name="twcultural">{{ cite web |url=http://cultural-award.moc.gov.tw/index03_03_1.html |title=行政院文化獎第3屆得獎人介紹 |access-date=2018-02-09 |archive-url=https://web.archive.org/web/20180210002137/http://cultural-award.moc.gov.tw/index03_03_1.html |archive-date=2018-02-10 |dead-url=yes }}</ref><ref name="cugallery">{{ cite web |url=http://www.cuhk.edu.hk/ugallery/tc/story/mou_tsung_san.htm |title=大師身影：牟宗三先生 |date= |publisher= |author= }}</ref>

1960年應聘至[[香港大學|香港大學]]主講中國哲學。1968年，由香港大學轉任[[香港中文大學|香港中文大學]][[新亞書院|新亞書院]]哲學系主任。1974年自香港中文大學退休，任教[[新亞研究所|新亞研究所]]。其後又任教[[國立臺灣大學|國立臺灣大學]]、[[國立臺灣師範大學|國立臺灣師範大學]]、[[東海大學_(台灣)|東海大學]]、[[國立中央大學|國立中央大學]]。<ref name="cugallery" />

1995年4月12日下午三時，牟氏因[[器官衰竭|器官衰竭]]逝於[[國立台灣大學醫學院附設醫院|國立台灣大學醫學院附設醫院]]，享壽86歲。

==學術==

牟宗三的思想受[[熊十力|熊十力]]的影響很大，他不僅繼承而且發展了熊十力的哲學思想。牟氏較多地著力於哲學理論方面的專研，謀求儒家哲學與康德哲學的融通，並力圖重建儒家的「道德的形上學」。代表作包括：《心體與性體》《才性與玄理》《中國哲學十九講》《中西哲學之匯通》《現象與物自身》《佛性與般若》等。

牟氏曾獨力翻譯康德的《三大批判》，融合康德哲學與[[孔子|孔]]、[[孟子|孟]]、[[陸九淵|陸]]、[[王陽明|王]]的[[心學|心學]]，以中國哲學與康德哲學互相詮解。也曾參與編寫雜誌，主編《再生》、創辦《歷史與文化》、編輯《理想歷史文化》。作為[[新儒家|新儒家]]代表人物，牟氏認為當代[[新儒學|新儒學]]的任務為「道統之肯定，即肯定道德宗教之價值，護住孔孟所開闢之人生宇宙之本源。」以「反省中華民族的文化生命，以重開中國哲學的途徑。」為己任。 他的著作，都是針對時代或學術問題，而提供一解決之道。<ref name="twcultural" />

牟氏的學術研究範圍包括：
*[[邏輯學|邏輯學]]
*[[伊曼努尔·康德|康德]]哲學
*[[宋明理學|宋明理學]]
*[[魏晉玄學|魏晉玄學]]
*[[佛學|佛學]]

==榮譽==

* 1987年，獲[[香港大學|香港大學]]頒授榮譽文學博士。<ref name="hkuhongrads" />

==評價==
*牟宗三治喪委員會的評價：“綜觀先生一生，無論講學論道，著書抒義，莫不念念以光暢中國哲學之傳統、昭蘇民族文化之生命為宗趣。其學思之精敏，慧識之弘卓，與夫文化意識之綿穆強烈，較之時流之內失宗主而博雜歧出者，敻乎尚矣。”<ref>[http://groups.msn.com/sgip55u2n4ipcjm9146n91lu30/page99.msnw 牟宗三先生學行事略]</ref>
