'''坦帕'''（{{lang-en|Tampa}}）是位于[[美国|美国]][[佛罗里达州|佛罗里达州]]西海岸的一座城市，也是[[希尔斯伯勒县_(佛罗里达州)|希尔斯伯勒县]]的县治所在。根据[[美国人口调查局|美国人口调查局]]2000年统计，共有[[人口|人口]]303,447，是仅次于[[杰克逊维尔_(佛罗里达州)|杰克逊维尔]]和[[迈阿密_(佛罗里达州)|迈阿密]]的佛罗里达州第三大城市，其中[[白人|白人]]占64.22%、[[非裔美国人|非裔美国人]]占26.07%、[[亚裔美国人|亚裔美国人]]占2.15%。

当地华人居民将坦帕译作天柏。

== 历史 ==
{{main|坦帕历史}}坦帕市目前所在的位置曾一度被{{le|平安港文化|Safety Harbor culture}}的原住民所居住，特别是{{le|Tocobaga}}和{{le|Pohoy|Pohoy}}两个族群曾活跃在坦帕湾沿岸。这一地区被[[西属佛罗里达|西班牙探险者]]于16世纪发现，随后爆发了残酷的暴力冲突和[[哥倫布大交換|欧洲疾病的肆虐]]，导致了本地原始文化的灭绝。尽管西班牙人声称佛罗里达是[[新西班牙|新西班牙]]的一部分，他们却没有在坦帕湾地区建立殖民地。现在坦帕市 区的范围内在很长一段时间内并没有来自美洲或欧洲的定居者，直到美国于1819年从西班牙手中获得了佛罗里达。

1824年，[[美国陆军|美国陆军]]在{{tsl|en|Hillsborough_River_(Florida)||希尔斯伯勒河}}的河口建立了一个叫{{tsl|en|Fort_Brooke||Fort Brooke}}的前哨，地点离现在的{{tsl|en|Tampa_Convention_Center||坦帕市会展中心}}很近。坦帕最早的居民是在这个陆军前哨附近定居的拓荒者，他们向陆军哨所寻求保护来抵御附近的[[塞米諾爾人|塞米諾爾人]]。他们建立的 小村庄于1849年被首次以坦帕（Tampa）的名字划入编制。在1880年代之前，这个小村子发展很缓慢，然而随后发生的三件事：铁路的开通，磷酸盐矿的发现，雪茄工业的兴起，让这个1880年人口还不到800人的小村子快速发展，在1900年代，坦帕已发展为人口超过30000人的繁华城市。

=== 气候 ===
坦帕属[[副热带湿润气候|副热带湿润气候]]，接近[[热带季风气候|热带季风气候]]，大体上终年暖和，雨热同季。冬季温和，微潮，时而偏向温暖，日最高气温低于或等于{{convert|15|C}}的平均日数为8.2天，日最低气温低于或等于{{convert|0|C}}的平均日数为1.6天（大约40%的冬季无霜），低于{{convert|5|C}}的平均日数为14天；夏季炎热潮湿，日最高气温超过{{convert|30|C}}的日数年均有161天，超过{{convert|35|C}}的只有5.5天；日低温大于或等于{{convert|25|C}}的年均日数为30天。<ref name = "NWS Ruskin, FL"/>最冷月（1月）均温{{convert|60.8|F|1|disp=flip}}，极端最低气温{{convert|18|F|0|disp=flip}}（1962年12月13日）。<ref name = "NWS Ruskin, FL"/>最热月（8月）均温{{convert|83.2|F|1|disp=flip}}，极端最高气温{{convert|99|F|0|disp=flip}}（1985年6月5日）。<ref name = "NWS Ruskin, FL"/>年均降水量约{{convert|46.3|in|sigfig=3|disp=flip}}，全年大约60%的降雨集中在6月至9月；年极端最少降水量为{{convert|28.89|in|disp=flip}}（1956年），最多为{{convert|76.57|in|disp=flip}}（1959年）。<ref name = "NWS Ruskin, FL"/>降雪为罕见：1893年以来的唯一一场可测量的降雪出现于1899年2月13日。<ref name = "NWS Ruskin, FL"/>
