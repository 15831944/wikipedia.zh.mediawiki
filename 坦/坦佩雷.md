'''坦佩雷'''（{{lang-fi|Tampere}}，{{lang-sv|Tammerfors}}），人口225,485人（2016年3月），[[芬兰|芬兰]]第三大城市，主要交通枢纽之一。

== 历史 ==
[[File:Tampere_city_hall_-_panoramio.jpg|thumb]]]]
坦佩雷原为一个小市场，1775年由[[瑞典国王|瑞典国王]][[古斯塔夫三世|古斯塔夫三世]]所建，四年后正式建城。到19世纪下半叶，坦佩雷已经成为芬兰最主要的工业中心，集中了全国几乎一半的工业，号称“北方的[[曼彻斯特|曼彻斯特]]”。

== 地理 ==
坦佩雷位于芬兰西南部，南距[[赫尔辛基|赫尔辛基]]170公里。全市位于两个湖之间，由于两湖有18米的落差，所以水力为坦佩雷提供了主要的动力来源，直到今日，水力发电还是该市重要的电力来源之一。

坦佩雷市西側接鄰[[諾基亞市|諾基亞市]]，為世界著名流動電話生產商[[諾基亞公司|諾基亞公司]]（Nokia Corporation）的诞生地。

== 對外交通 ==
[[File:Keskustori_bus_terminals1.jpg|thumb]]
[[坦佩雷-皮尔卡拉机场|坦佩雷-皮尔卡拉机场]]位于坦佩雷西南方向的皮尔卡拉市，距离坦佩雷市区约17千米。该机场开通了前往[[赫尔辛基|赫尔辛基]]、[[图尔库|图尔库]]、[[奥卢|奥卢]]等国内目的地以及[[斯德哥尔摩|斯德哥尔摩]]、[[哥本哈根|哥本哈根]]、[[伦敦|伦敦]]、[[法兰克福|法兰克福]]、[[米兰|米兰]]等欧洲目的地的航线。

坦佩雷火車站位於市中心旁，是芬蘭铁路系统的“十字路口”。北部[[拉普蘭|拉普蘭]]地區南下首都赫尔辛基列車及中部內陸地區的列車都會經過坦佩雷。由首都赫尔辛基及西部港口城市图尔库乘芬蘭國鐵火車到坦佩雷車程分別約2小時及1.8小時。

另外坦佩雷與諾基亞市雖有火車連結，但班次疏落；相反坦佩雷市內有頻密的市營巴士服務連結兩地。

== 教育事业 ==
=== 坦佩雷理工大学 ===
[[File:TUT_Festia.JPG|thumb]]
有11,600在校学生，10000名本科和研究生，1600名博士生。几乎所有的硕士毕业生都能就业。<ref name="TUT1">[http://www.tut.fi/public/index.cfm?MainSel=1607&Sel=1690&Show=1579&Siteid=32]TUT in a Nutshell, http://www.tut.fi, 30.04.2010</ref>大学设有七个国家级博士生院。大学还拥有自己的博士生院。在2008年，有超过2,200份论文刊登于芬兰或国际期刊。2009年， 获得科研经费1.55亿欧元。

[[坦佩雷理工大学|坦佩雷理工大学]](TUT)是芬兰最国际化的大学之一。 2008年，接近1200教授，教师，研究人员和学生都从国外聘用或在大学学习。 外国学生的兴趣日益高涨，是推动TUT国际学习交流的动力，并且得到了交换学生的良好反馈。交换学生可以选择世界各地270所大学进行合作交流。 在2008年，有184名交换学生。<ref name="TUT2">[http://www.tut.fi/public/index.cfm?MainSel=1607&Sel=15108&Show=22136&Siteid=32]International interest, http://www.tut.fi, 27.10.2009</ref>

TUT被国际公认的前沿领域是信号处理，纳米光子学和智能机械。<ref name="TUT3">[http://www.tut.fi/public/index.cfm?MainSel=1604&Sel=14964&Show=21554&Siteid=32]Leading-edge fields of research, http://www.tut.fi, 23.10.2009</ref>

=== 坦佩雷大学 ===
[[坦佩雷大学|坦佩雷大学]](UTA)位坦佩雷的市中心，是芬兰最古老的大学之一，坦佩雷大学(UTA)现有1.5万在校生，大学下属六个学院：经济与管理，教育，人文，信息科学，医学院，社会科学。坦佩雷大学校区，周边设施完善，交通便利。学生宿舍<ref name="CSSA-Tampere">[http://sites.google.com/site/tamperecssa/sheng-huo-chang-shi-hui-zong#TOC-4]CSSA-Tampere官方网站</ref>坐落在市中心，步行十五分钟的地方就有学生宿舍。该大学在其他城镇也有校区，离坦佩雷约70公里的[[海门林纳|海门林纳]]为学校的教育部，在距离坦佩雷约200公里的地方也有学校的一些研究中心。<ref name="UTA1">{{cite web |url=http://www.uta.fi/english/about.html |title=存档副本 |accessdate=2011-04-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20110514220348/http://www.uta.fi/english/about.html |archivedate=2011-05-14 }}Facts about the University, http://www.uta.fi, 31.12.2010</ref>

UTA的科学前沿领域是线粒体生物合成与疾病研究（国家级重点研究所）。 教育优势领域有语言中心，医学基础教育(硕士)，幼儿教育。<ref name="UTA2">{{cite web |url=http://www.uta.fi/english/keyfacts/index.html |title=存档副本 |accessdate=2011-04-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20110610212008/http://www.uta.fi/english/keyfacts/index.html |archivedate=2011-06-10 }}Key Facts 2009, http://www.uta.fi, 29.12.2010</ref>

=== 坦佩雷应用科技大学 ===
坦佩雷应用科技大学(TAMK)有在校生10000多名，有坦佩雷以及[[皮尔卡拉|皮尔卡拉]]地区的几所学院合并而成。学校授予应用技术方面的本科与硕士学位。并且设有继续教育学院。<ref>[http://www.tamk.fi/cms/tamken.nsf/$all/DD54E6FCC996B233C225759000250591]About TAMK, http://www.tamk.fi, 09/28/2010</ref>本科学位领域有文学、媒体、经济、技术，其中计算机和商学学科比较有名。
