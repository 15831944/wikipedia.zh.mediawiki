'''屌丝'''，字面意思為[[男性|男性]]的[[阴毛|阴毛]]，後引申為「人生失败者」、「人生输家」、「愛情輸家」的泛稱。该词流行于中国大陆的[[互联网|互联网]]，用作称呼由[[农村|农村]]进入[[城市|城市]]的体制外男女年輕人或在一定程度上，撸瑟和屌丝是同义。房及车对于屌丝来说更是遥不可及的梦想<ref name="et">{{cite web | url=http://www.ettoday.net/news/20120426/42116.htm#ixzz1tCuxiYcy | title=「屌丝」大战白富美 自我解嘲唱反高富帅 | publisher=ETtoday 新闻云 大陆新闻 | accessdate=2012-04-27 | language=zh-hant}}</ref><ref>[http://news.163.com/12/1111/04/8G0M4HPJ00014AED.html 《人民日报》回应“屌丝”、“给力”、“元芳”等见报：属很正常报道]</ref>。2012年初，「屌丝」在中国大陆地区广泛流行起来，年轻人群体间的语言文化中更被广泛应用，義近於[[台灣|台灣]]網路用語[[輪家男|魯蛇]]或中國大陸的[[擼瑟|擼瑟]]<ref name="fhws">{{cite web |url=http://news.ifeng.com/opinion/special/diaosi/ |title=屌丝，一个专有词汇的诞生 |accessdate= 2012年|work=|publisher=凤凰网 |language = zh-cn}}</ref>，也義近於[[香港|香港]]網路用語[[廢青|廢青]]、[[毒男|毒男]]。

== 起源 ==
该词起源于2011年10月[[李毅_(足球运动员)|李毅]][[贴吧|贴吧]]与[[雷霆三巨头吧|雷霆三巨头吧]]的一次对骂<ref>[http://v.ifeng.com/news/society/201303/c976ee26-201a-4633-983b-0e203b959986.shtml “屌丝”一词的来源，见视频第6分50秒。]</ref>，雷霆三巨头吧的吧友骂李毅“是个[[屌|屌]]”，其粉丝就是“屌丝”，表示对[[李毅吧|李毅吧]]成员的蔑视<ref>{{Cite book|title=泺尚创意中国调研报告 2013|last=昝胜锋，刘欣主编|first=|publisher=济南：[[山东大学出版社|山东大学出版社]]|year=2013.05|isbn=978-7-5607-4785-9|location=|pages=383}}</ref>。然而，當時[[李毅吧|李毅吧]]网友却觉得这个词听起来很[[酷|酷]]，之后拿来作為自嘲用语<ref>{{Cite book|title=网络那些词儿|last=马中红，陈霖主编|first=|publisher=北京：[[清华大学出版社|清华大学出版社]]|year=2014.07|isbn=978-7-302-36287-6|location=|pages=41-42}}</ref>，并以李毅吧为据点向外影响。

== 影响 ==
2011年底，屌丝一词在[[Google|Google]]的搜索结果达到4110万，[[新浪微博|新浪微博]]搜索有超过220万条微博，该词也成为众多[[中国大陆|中国大陆]]男青年自嘲的常用语。网络上更是产生了一大批以屌丝为题材的影音作品，如《[[屌丝男士|屌丝男士]]》等。<ref>{{cite web|url=http://v.youku.com/v_show/id_XMzM0MzUxMTQ4.html|publisher=[[优酷网|优酷网]]|title=音乐视频：屌丝与黑木耳|date=2011-12-21}}</ref><ref>{{cite web|url=http://v.youku.com/v_show/id_XMzMyMjMyMDYw.html|title=视频: 毅丝和黑木耳的故事：备自修 剧组 正片前期预告MV |publisher=[[优酷网|优酷网]]|date=2011/12}}</ref><ref>{{cite web|url=http://v.youku.com/v_show/id_XMzIyOTI2MjI4.html|publisher=[[优酷网|优酷网]]|title=网络电影：屌丝的爱情|date=2011-12-08}}</ref><ref>{{cite web|url=http://tv.sohu.com/s2012/diorsman/|publisher=[[搜狐视频|搜狐视频]]|title=电视剧：屌丝男士第1季|date=2012-10-10}}</ref><ref>{{cite web|url=http://tv.sohu.com/s2013/diorsman2/|publisher=[[搜狐视频|搜狐视频]]|title=电视剧：屌丝男士第2季|date=2013-06-26}}</ref>

2012年3月初，中国大陆地区[[防火长城|防火长城]]短暂失效，使中国大陆网民忽然可以不用[[代理服务器|代理]]访问[[Google+|Google+]]，许多自称是屌丝的人在Google+上给美国总统[[奥巴马|奥巴马]]留言，一时间[https://plus.google.com/110031535020051778989/posts 奥巴马页面]上充满了中文留言，被称为“占领奥巴马”，也有媒体称为中国屌丝“杀出国门”。<ref>{{cite web|url=http://view.news.qq.com/a/20120301/000018.htm|publisher=[[腾讯网|腾讯网]]|title=中国屌丝“杀出国门”|date=2012-03-01}}</ref>

[[德国|德国]]的喜剧集Knallerfrauen《炸弹妞》被网友译为《[[屌丝女士|屌丝女士]]》，在网络上广受欢迎。<ref>{{cite web|url=http://movie.douban.com/subject/10537919/|publisher=[[豆瓣网|豆瓣网]]|title=豆瓣网《屌丝女士》页面|date=2012-02-03}}</ref>日本电视剧、电影《[[桃花期_(电影)|桃花期]]》也被网友取名为《屌丝的逆袭》。<ref>{{cite web|url=http://movie.douban.com/subject/4862213/|publisher=[[豆瓣网|豆瓣网]]|title=豆瓣网《桃花期》电视剧版页面|date=2010-07-16}}</ref> <ref>{{cite web|url=http://movie.douban.com/subject/6041222/|publisher=[[豆瓣网|豆瓣网]]|title=豆瓣网《桃花期》电影版页面|date=2011-09-23}}</ref>

网络主播[[MC石头|MC石头]]因其形象和生活被认为典型的屌丝。他生活在[[四川|四川]][[泸州|泸州]]，初在网络上传自己的喊麦视频，后被[[凤凰卫视|凤凰卫视]]等多家媒体报导，被[[北京|北京]][[上海|上海]]多家[[迪厅|迪厅]]请去做现场DJ。<ref>{{cite web|url=http://weibo.com/mcshitou|publisher=[[新浪微博|新浪微博]]|title=MC石头新浪微博|date=2011-04-14}}</ref>

2012年6月10日，即将毕业的[[电子科技大学|电子科技大学]]的金雪涛和[[四川教育学院|四川教育学院]]的吴桔吉以及他们的朋友罗杰在[[西南财经大学|西南财经大学]]大门的对面开了一家名为“舌尖上的屌丝”的小餐馆，开业三天就净赚了上万元。<ref>{{cite web|url=http://news.qq.com/a/20120616/000499.htm|title=临毕业大学生开“舌尖上的屌丝”餐馆 生意兴旺|date=2012-06-16}}</ref>

2012年11月3日，《人民日报》十八大特刊刊发《激发中国前行的最大力量》一文，文中写道：“市场经济的冲击余波未了，全球化、民主化、信息化的浪潮又不期叠加。分配焦虑、环境恐慌，拼爹时代、屌丝心态，极端事件、群体抗议，百姓、社会、市场、政府的关系进入‘敏感期’。”<ref>{{cite web|url=http://news.china.com.cn/live/2012-11/11/content_17092692.htm|title=人民日报回应用“屌丝”:属正常报道无特别关注|date=2012-11-11}}</ref>

但2012年12月30日，《[[咬文嚼字|咬文嚼字]]》杂志发布了“2012年十大流行语”中该词未入选，专家认为，“它的不雅、恶俗的趣味是很明显的”。<ref>{{cite web|url=http://zj.qq.com/a/20121231/000037.htm|title=“屌丝”落选2012年十大流行语 专家称该词不雅|date=2012-12-31}}</ref>

== 评价 ==
* [[凤凰网|凤凰网]]的评论认为屌丝貌似[[阿Q|阿Q]]而并不[[意淫|意淫]]。屌丝族群和屌丝文化深深反映了时代背景、贫富之差，“小人物”获取成功得付出比别人多的艰辛；同时也是争夺[[政治|政治]]、[[经济|经济]]、[[社会生活|社会生活]]中话语权的一种表达方式，而且又没有触及[[中国大陆|中国大陆]][[法律|法律]]的底线，他们选择了一种温凉性的表述。<ref name="fhws " />

* [[中山大学|中山大学]]亚太研究院教授[[朱崇科|朱崇科]]认为：“‘屌丝’的传播具有庶民叙事的特征，由下而上地对主流话语、[[价值观|价值观]]进行反抗。自称‘屌丝’让很多人找到自我解嘲和宣泄压力的途径，这也是它传播之快的原因。”[[中山大学|中山大学]]中文系副教授[[柯倩婷|柯倩婷]]把这个网络亚文化现象概括为“一场语言的狂欢”，“屌丝”的狂欢背后有明显的失落情绪，“婚恋中的现实压力，加剧了这个群体的焦虑情绪。”<ref>{{cite web|url=http://www.shm.com.cn/newscenter/2012-03/24/content_3728586.htm|publisher=[[羊城晚报|羊城晚报]]|title=“屌丝”一词爆红中国网络 专家称反映集体焦虑|date=2012-03-24|language = zh-hans}}</ref>

* [[金羊网|金羊网]]评论说：“‘屌丝’这个词的诞生，并不只是‘集体焦虑’，而是社会价值观开始崩塌、人伦底线逐渐烂陷的标志。倘若还将嘲笑‘屌丝’、追捧‘高富帅’当作网络狂欢的乐趣，那么，这个社会的灵魂确实应该被狠狠、彻底地改革一下了。”<ref>{{cite web|url=http://sp.ycwb.com/2012-03/25/content_3753558.htm|publisher=金羊网|title=“屌丝”蹿红反映社会改革之迫切|date=2012-03-25|language=zh-hans|access-date=2012-03-25|archive-url=https://web.archive.org/web/20120329223043/http://sp.ycwb.com/2012-03/25/content_3753558.htm|archive-date=2012-03-29|dead-url=yes}}</ref>

==批评==
“屌丝”一词虽然在主流媒体上铺天盖地，同时却也引起了许多的反对和厌恶。尤其是“屌丝‘文化’”中对女性的物化和[[性歧视|性歧视]]，以及[[拜金主义|拜金主义]]更是导致[[女性主义|女性主义]]者和独立知识分子一致的声讨。有评价称其为“一种对社会拜金风潮的无奈掺杂的产物，亦充满了格调不高的低俗气”。亦如“但如果当‘黑木耳’之类随意且带有性暗示和侮辱性的称呼在网络上大行其道，对女性起码的尊重已经荡然无存”，并道出“毫无疑问，在屌丝的语境中，女性已经被物化为男性的附属”。<ref>{{cite web|url=http://www.guokr.com/blog/97963/|title=【女性主义解读“吊丝文化”】亚文化的轻蔑与厌恶|date=2012-02-19|language = zh-hans}}</ref>

有女生写出了《防屌丝手册》，并且指出“屌丝有[[大男子主义|大男子主义]]倾向，对女性缺乏基本的尊重。”<ref>{{cite web|url=http://www.douban.com/group/topic/30185795/|title=防屌丝手册|date=2012-06-05|language = zh-hans}}</ref>，但这篇文章对屌丝的草率定义亦缺乏对部分被这篇文章無辜波及到的男性的尊重。

一篇《北大才女：为什么反对“屌丝”亚文化》的文章也在网络上广为流传，并被不少贴吧所转载。文中指出“旗帜鲜明反对『屌丝』亚文化，是因为它具有[[犬儒主義|犬儒性]]和排他性”，亦有提及屌丝对女性的物化和排斥。<ref>{{cite web|url=http://ibeidou.net/?p=19625|title=为什么反对“屌丝”亚文化|date=2012-03-09|language=zh-hans}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>类似的文章还有《我为什么反对屌丝文化》等。<ref>{{cite web|url=http://luo.bo/24557/|title=我为什么反对屌丝文化|date=2012-05-11|language = zh-hans}}</ref>

“复旦人类学”的新浪博客不仅指出了“屌丝”文化所蕴含的[[双重标准|双重标准]]和处女崇拜，更是发出如下的警告：“一旦这种话语模式广泛地渗透入大众传媒，顶着『解构、自嘲』的招牌的『屌丝文化』就会使歧视性的性别观念变得堂而皇之起来，甚至可能麻痹社会性别意识，钝化女性的生命感觉。更可怕的是，在『屌丝文化』的戏谑自嘲的基调下，一旦有人提出质问，就会被真真假假的屌丝以一句『认真你就输了』弄得哑口无言”。<ref>{{cite web|url=http://blog.sina.com.cn/s/blog_6593f65301013ujy.html|title=【2012年春季文化人类学课程论文】：浅析“屌丝文化”的性别观 |date=2012-07-08|language = zh-hans}}</ref>

不过以上的引用来源都是网页内容可信程度不高，有学者发的论文研究认为类似「[[擼瑟|擼瑟]]」（loser）、“屌丝”这一类贬义单词容易让人和男性联想起来实际上是对男性的歧视。“作为养家糊口者不甚成功或不尽人意的话，英语中自然不乏诽谤之辞: loser，deadbeat，bum/freeloader，leech，parasite，goldrick，sponge 等等。这样的境遇伴随男性一生。如果他背离了社会所界定的男性的行为准则就会受到惩罚。而女孩只要愿意，她就可以表现得很『boyish』，没人觉得不可接受; 但如果男孩表现得很『girlish』，就会让人难以接受。”<ref>{{cite journal|title=性别歧视语的两面性——微探男性性别歧视语|author=蒋婷媛|journal=外国语文|issue=S1|year=2012|url=http://www.cnki.net/KCMS/detail/detail.aspx?QueryID=2&CurRec=1&recid=&filename=SCWY2012S1021&dbname=CJFD2012&dbcode=CJFQ&pr=&urlid=&yx=|accessdate=2013-03-06|deadurl=yes|archiveurl=https://web.archive.org/web/20160128172436/http://www.cnki.net/KCMS/detail/detail.aspx?QueryID=2&CurRec=1&recid=&filename=SCWY2012S1021&dbname=CJFD2012&dbcode=CJFQ&pr=&urlid=&yx=|archivedate=2016-01-28}}</ref>

大洋网的评论《回顾百度贴吧屌丝节 以网络亚文化反哺主流社会》中的评论“诚如[[百度贴吧|百度贴吧]]所宣扬的，爱因斯坦、乔丹、梵高、扎克伯格乃至奥巴马，这些改变世界的人，在功成名就之前，也曾是屌丝。”和“在更早之前，网易、金融界、新浪微博等主流门户、财经、SNS网站，都对屌丝节做了深度的报道与传播。”也说明了“屌丝‘文化’”的突然膨胀背后离不开百度等商业力量的推波助澜。

2013年2月26日，著名电影导演[[冯小刚|冯小刚]]在微博上称“屌丝就是[[阴毛|雞巴毛]]，是对境遇不堪者的蔑称，自称屌丝的是不以为耻反以为荣。”<ref>{{cite web|url=http://www.yangtse.com/system/2013/02/27/016383805.shtml|title=冯小刚：屌丝是自贱 中国人不以为耻反以为荣|date=2013-02-27|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20140116222050/http://www.yangtse.com/system/2013/02/27/016383805.shtml|archivedate=2014年1月16日|df=}}</ref>
