《'''皿三昧'''》（{{lang-ja|さらざんまい}}）是由[[MAPPA|MAPPA]]与Lapin Track联合制作的日本原创动画。于2019年4月11日起于[[富士電視台|富士电视台]]的[[noitaminA|noitaminA]]时段播放。

== 故事简介 ==
中學二年級生矢逆一稀在一天慣常地上學時，與轉學生久慈悠於未盡了解下起了衝突，意外打碎了城鎮的一尊金河童像。剎那間，二人短暫失去記憶，城中亦無人察覺他們的所作所為。他們在思緒混亂下被廟宇之聲喚回舊地，驚見酷似青蛙的河童神明。神明未幾施出難以置信的「尻子玉」榨取技，將兩人和無故被牽扯進來的陣內燕太變成河童，一同進入欲望與秘密具現化的結界，同樣以榨取「尻子玉」的形式，解決干擾現實世界秩序的河童殭屍。至此，「皿三昧」的前設全數集齊，眾人為了保持「聯繫」的所作所為，亦將會在「聯繫」中被一一洩漏。同一時間，高調吸取欲望的黑手，與擁有相近力量的地元偶像，也準備加入混亂的戰團中。

; {{nihongo|矢逆一稀|矢逆 一稀||聲優：[[村瀨步|村瀨步]]}}
: 中學二年級生。為了討好弟弟，有著時刻帶著箱子，方便扮演偶像吾妻沙羅的習慣，但本人平日並不突高調。
; {{nihongo|久慈悠|久慈 悠||聲優：[[内山昂輝|内山昂輝]]}}
: 剛轉學到城鎮的中學二年級生，實質是跟隨哥哥步伐來到的社團成員，已經曉得種植大麻、用槍和對成年人使用暴力。主要武器是哥哥皆有配備的長尺。
; {{nihongo|陣内燕太|陣内 燕太||聲優：[[堀江瞬|堀江瞬]]}}
: 中學二年級生，一稀的青梅竹馬，喜歡足球。似乎也喜歡一稀。有名總是會鬥氣的姊姊。
; {{nihongo|河童|ケッピ||聲優：[[諏訪部順一|諏訪部順一]]}}
: 顯現於三人面前的神明，要求眾人榨取並回收河童殭屍的尻子玉，以希望之皿作為回禮。行為舉止像青蛙，但本人極為討厭被如此形容。
; {{nihongo|新星玲央|新星 玲央||聲優：[[宮野真守|宮野真守]]}}
; {{nihongo|阿久津真武|阿久津 真武||聲優：[[細谷佳正|細谷佳正]]}}
: 兩個城鎮分局的警察，以儀式吸收罪犯的欲望。
