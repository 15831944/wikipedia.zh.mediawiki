'''穎娃站'''（{{jpn|j=頴娃駅|hg=えいえき|rm=Ei eki|f=y}}）是一座位於[[鹿兒島縣|鹿兒島縣]][[南九州市|南九州市]]的[[九州旅客鐵道|九州旅客鐵道]]（JR九州）[[指宿枕崎線|指宿枕崎線]]的鐵路車站。

== 車站構造 ==
*一個1面1線單式月台，不可以[[列車交會|列車交會]]。
*[[無人站|無人站]]。曾經是有人站，設有站舍。
*[[地上車站|地上車站]]。

== 历史 ==
* 1960年（昭和35年）3月22日：車站啟用。
* 1987年（昭和62年）4月1日：因[[國鐵分割民營化|國鐵分割民營化]]，本站成為[[九州旅客鐵道|九州旅客鐵道]]的車站。

== 車站周邊 ==
* 穎娃郵局
* 南九州市立穎娃小學
* 南九州警署穎娃派出所
* [[國道226號|國道226號]]

== 其他 ==
此車站的羅馬拼音為[ei]，與[[粟生站|粟生站]][ao]、[[小江站|小江站]][eo]及[[飯井站|飯井站]][ii]成為全日本羅馬拼音最短的車站。而在音節數中最短的車站則為[[津車站|津站]]，其羅馬拼音[tsu]，但平假名則只是[つ]。
