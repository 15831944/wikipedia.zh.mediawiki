'''文德站'''位於[[台灣|台灣]][[台北市|台北市]][[內湖區|內湖區]]，為[[台北捷運|台北捷運]][[文湖線|文湖線]]（[[內湖線|內湖線]]）沿線的[[捷運車站|捷運車站]]。站名來自車站所在路名文德路。

== 車站構造 ==
高架二層車站，兩個[[側式月台|側式月台]]，兩個出入口。

=== 車站出口 ===
出口1位於車站東北側，出口2位於車站西南側，所有出口均設有無障礙電梯
* 出口1：碧湖公園[[File:Accessibility-directory.svg|20px]]
* 出口2：內湖高中[[File:Accessibility-directory.svg|20px]]

== 利用狀況 ==
車站位於文德路上，[[臺北市立內湖高級中學|內湖高中]]西北側；車站代碼為'''BR18'''。站內公共藝術：鳥飛。

== 歷史 ==
*2009年2月22日：文德站實質完工。
*2009年7月4日：隨著內湖線通車而啟用。
*2009年7月28日：捷運局東工處長陳耀維表示，會導致車廂漏水的文德、葫洲、南港展覽館站，將在一個月內裝設鍍鋅排水管，徹底解決問題。
