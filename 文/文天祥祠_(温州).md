'''文天祥祠'''又名'''文信国公祠'''、'''文丞相祠'''，位于[[中国|中国]][[浙江省|浙江省]][[温州市|温州市]][[鹿城区|鹿城区]][[江心屿|江心屿]]东首、[[江心寺|江心寺]]东侧，为祭祀南宋文学家、政治家[[文天祥|文天祥]]的祠堂，[[杜浒_(南宋)|杜浒]]、[[徐臻|徐臻]]从祀<ref>清·李琬修，齐召南、汪沆纂，《乾隆温州府志·卷九·祠祀》：“文天祥祠（江心寺志）在沙门左，祀宋右丞相文天祥，以台郡少卿杜浒、永嘉正将徐臻配。”</ref>。南宋[[德祐|德祐]]二年（1276）正月，文天祥受命出使元营谈判被扣留，在解送途中逃脱，后辗转泛海南行，四月至温州，留居江心屿一个月，期间曾作《北归宿中川寺》。该祠始建于明成化十八年（1482），后多次重修，现存为晚清建筑，坐北朝南，由门厅、正厅和东西廊屋组成，门厅与正厅面阔均为三间，门厅为硬山顶，正厅为悬山顶<ref>{{cite web |url=http://culture.zjol.com.cn/05culture/system/2006/08/07/007791135.shtml|title=《江心寺文天祥祠》|accessdate=2018-10-13|work=|publisher=浙江在线}}</ref><ref>{{cite web |url=http://www.zjww.gov.cn/subject/2006-02-16/1446480.shtml|title=《温州·文天祥祠》|accessdate=2018-10-13|work=|publisher=浙江省文物局}}</ref>。
