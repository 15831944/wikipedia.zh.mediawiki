《'''文匯报'''》[[香港|香港]]版於1948年9月9日創刊，該報原是[[中國|中國]][[上海|上海]]一份歷史悠久的報紙，經歷多番轉變后落户香港。《文匯報》的言論一向緊貼[[中國共產黨|中國共產黨]]的政策，親中共的政治立場鮮明，在中國大陸還未[[改革開放|開放]]前，外界往往憑其報導或言論揣測[[北京|北京]]某些政策動向。《文匯報》、《[[大公報|大公報]]》和《[[香港商報|香港商報]]》被稱為「香港三大[[左派|左]]報」<ref>{{cite journal |author1=《明鏡月刊》編輯部 |title=中共媒體在香港的運作 |journal=《明鏡月刊》 |date=2017 |issue=86}}</ref>。

== 創辦背景 ==
《文匯報》於1938年1月25日在[[上海市|上海市]][[福州路_(上海)|福州路]]436号創刊，由[[严宝礼|严宝礼]]等一批「爱国人士」集资创办<ref name=shsdfz>{{cite web|url=http://www.shtong.gov.cn/Newsite/node2/node2245/node4522/node5501/node5526/node63723/userobject1ai8672.html|title=上海新闻志→第一编报纸→第三章上海解放后报纸（1949年5月～1996年12月）→第一节 三家主要报纸|publisher=上海市地方志办公室|accessdate=2016年9月2日 }}</ref>。《文匯報》在[[中国抗日战争|抗日戰爭]]時期，立場鮮明的號召抗日及反對日本侵華<ref>{{Cite web|url=http://hk.apple.nextmedia.com/supplement/special/art/20151103/19357515|title=【贈閱文匯報】你不知道的愛國報章抗爭史|accessdate=2017-09-21|author=|date=2015-11-03|work=|publisher=蘋果日報}}</ref>。《文匯報》在[[中国抗日战争|抗日戰爭]]之後，立場鮮明地反對[[第二次国共内战|國共內戰]]及号召支持[[中国共产党|中國共產黨]]，令當時的[[中華民國|中华民国]][[國民政府|国民政府]]十分不滿，迫使上海《文匯報》於1947年5月24日停刊。香港《文匯報》由[[徐铸成|徐铸成]]等人，把創辦民革機關報與文匯報復刊計劃結合起來，與當時在香港的亲共人士共同創辦<ref name=shsdfz/>。1948年9月9日，《文匯報》在香港創刊。创刊之初一度成为[[中国国民党革命委员会|中国国民党革命委员会]]的机关报。1957年后，香港文汇报由[[中国共产党香港工作委员会|中共港澳工委]]接管，港澳工委常委[[吴荻舟|吴荻舟]]被任命为首任社长，主持日常工作<ref>{{Cite web|url=http://sp.wenweipo.com/wwp60/about-4_2.php|title=社史拾遺 - 創刊歷史|accessdate=2017-09-21|author=|date=|work=|publisher=文匯報}}</ref>。香港《文匯報》在[[文化大革命|文化大革命]]期间，曾一度改爲橫排，並使用[[简化字|簡體字]]發行，但其後恢復使用[[繁體字|繁體字]]。

== 市場定位 ==
香港《文匯報》與[[中共中央宣传部|中共中央宣傳部]]有密切关系，多年來，中華人民共和國官方媒體經常引用《文匯報》的報導；文章首段會以「 据香港《文汇报》报道」等聲明<ref>{{Cite web|url=http://www.hkcna.hk/content/2012/0423/138564.shtml|title=香港各界重歸大團結 齊撐“香港營”|accessdate=2017-09-21|author=|date=2012-04-23|work=|publisher=香港新闻网}}</ref>，《文匯報》以香港報紙自居，聲稱讀者定位主要是香港社會各界精英，由于該報擁有支持[[中共|中共]]的悠久歷史背景，在報道國際兩岸事务時，往往站在中共立場，香港事務方面亦親親中建制派，以「反對派」來稱呼及批評[[泛民主派|泛民主派]]，其政治色彩濃厚等因素，使得該報在香港缺乏公信力，根據2009年針對香港市民對傳媒可信度的調查顯示，《文匯报》和《[[大公報|大公報]]》可信度分數排行最低<ref>{{Cite web|url=http://www.rthk.org.hk/mediadigest/20090817_76_122335.html|title=甚麼是RT？HK何處去？|accessdate=2017-09-21|author=|date=2009-08-17|work=|publisher=传媒透视|archive-url=https://web.archive.org/web/20160314045359/http://www.rthk.org.hk/mediadigest/20090817_76_122335.html|archive-date=2016-03-14|dead-url=yes}}</ref>。到了2010年，一項香港市民對傳媒公信力評分的調查顯示，《文匯报》可信度分數下降至5.46，然而可信度排行卻增至14<ref>{{Cite web|url=http://rthk.hk/mediadigest/20110113_76_122704.html|title=傳媒公信力評分因年齡層而變化|accessdate=2017-09-21|author=|date=2011-01-13|work=|publisher=传媒透视|archive-url=https://web.archive.org/web/20170308205715/http://rthk.hk/mediadigest/20110113_76_122704.html|archive-date=2017-03-08|dead-url=yes}}</ref>。

在香港報章市場激烈競爭的下，一般認為《文匯報》在香港的銷售量頗低<ref>{{Cite news|url=http://www.epochtimes.com/b5/3/11/22/n415695.htm|title=金鐘：回歸後的香港傳媒|author=|date=2003-11-22|work=|newspaper=大紀元|accessdate=2017-09-21|language=zh-TW}}</ref>。

== 政治立場 ==
《文匯報》親[[中国大陆|中國大陸]]立場鮮明，在1997年[[香港回歸|香港主權移交]][[中华人民共和国|北京當局]]後，《文匯報》亦傾向大多數支持香港特區政府及相關政策，較少提出一些對於香港特區政府的批評，和另外一份香港親中共報章的《[[大公報|大公報]]》相同。《文匯報》也會因為其與中央政府的特殊關係，被認為較其他香港媒體率先得到或獨家發佈中央政府與香港特區政府的一些政策資訊。

《文匯報》亦曾對中國社會問題作出相關報導，例如；2006年3月2日以「走近北京『告御狀』的人」為題的專題報導，更獲得獎項，該文章報導有中國內地民眾受到地方政府權勢欺壓，而上訪到北京投訴，該文章結尾歸究於社會有腐敗和不公，但點到即止，該報對中國共產黨及相關制度的問題均採用輕描淡寫方式，不會直接批評<ref>{{Cite web|url=http://paper.wenweipo.com/2006/03/02/CH0603020004.htm|title=走近「告御狀」的人群|accessdate=2017-09-21|author=|date=2006-03-02|work=|publisher=香港文匯報}}</ref>。在2006年，[[臺灣|台灣]]前[[總統|總統]][[陳水扁|陳水扁]]引發的「[[百萬人民倒扁運動|百萬人民倒扁運動]]」；而撰寫的「30萬人怒吼　阿扁下台」的報導中，大幅報導台灣民眾遊行及「倒扁」情況，而文中的「總統府」及「国旗」等詞彙均用上引號，以示不承認台灣政權。

《文匯報》常以「反對派」稱呼[[香港|香港]][[泛民主派|泛民主派]]，對其報道傾向負面，評論版中有不少不使用真實姓名的署名文章，抨擊香港民主派及批評反中共的人士。《文匯報》亦會報導每年香港的「[[六四事件|六四]]」燭光晚會等新聞，類似這些被中共視為敏感的新聞，篇幅會較小，而文中如[[香港市民支援愛國民主運動聯合會|支聯會]]、[[民主女神|民主女神像]]、[[六四事件|六四]]等詞彙或組織名稱，《文匯報》會用上引號，或在前面加上「所謂」以示不認同<ref>{{Cite web|url=http://paper.wenweipo.com/2010/06/05/HK1006050033.htm|title=「支聯會」在維園  舉行六四晚會|accessdate=2017-09-21|author=|date=2010-06-05|work=|publisher=香港文匯報}}</ref><ref>{{Cite web|url=http://hk.apple.nextmedia.com/news/art/20120606/16402046|title=隔牆有耳：中聯辦攞景　左報贈興|accessdate=2017-09-21|author=|date=2012-06-06|work=|publisher=蘋果日報}}</ref>。

《文匯報》、《[[大公报|大公報]]》、《[[香港商報|香港商報]]》被持[[反共主義|反共]]立場的《[[蘋果日報_(香港)|蘋果日報]]》等稱為「[[中共|中共]]喉舌」等<ref>{{Cite web|url=http://hk.apple.nextmedia.com/news/art/20100609/14116140|title=蘋論：中國的平等定義是有些動物更平等|accessdate=2017-09-21|author=|date=2010-06-09|work=|publisher=蘋果日報}}</ref>，與《文匯報》則反稱對方為「反對派喉舌」<ref>{{Cite web|url=http://paper.wenweipo.com/2011/03/10/PL1103100009.htm|title=反對派喉舌美化暴力|accessdate=2017-09-21|author=|date=2011-03-10|work=|publisher=香港文匯報}}</ref>，兩者對中共立場可謂相差悬殊。

== 「六四」引發的集體辭退事件 ==
[[File:痛心疾首！.jpg|thumb]]
1989年「[[六四事件|六四事件]]」爆發前夕，中共在5月20日下令[[中国人民解放军|解放軍]]進京執行[[戒严|戒嚴]]令，此舉當時在香港社會引起很大迴響。《文匯報》遂於翌日（5月21日）以「開天窗」形式發表[[社論|社論]]，刊登了「痛心疾首」四個大字；報社社長[[李子誦|李子誦]]更公開在大會及電視上發言譴責鎮壓。有評論認為，《文匯報》的帶頭「造反」間接帶動中間立場報紙一致譴責中共。

「六四」後，中共秋後算賬，令社長李子誦、副總編輯[[程翔|程翔]]、駐北京特派員[[劉銳紹|劉銳紹]]先後離開《文匯報》；總編輯[[金堯如|金堯如]]後來更移居[[美国|美國]]，並退出中共，生前透過發表評論文章呼籲中共當局推行政治改革<ref>{{Cite web|url=http://www.epochtimes.com/b5/nf4028.htm|title=金堯如|accessdate=2017-09-21|author=|date=|work=|publisher=大紀元|language=zh-TW}}</ref>。

== 發行與網絡 ==
[[File:HK_HingWaiCentre.JPG|thumb]]興偉中心的2-4樓]]
香港《文匯報》主要版面有：重要新聞、香港新聞、中國新聞、國際新聞、財經新聞、娛樂、體育、教育、通識、副刊、馬經等。

《文匯報》除在香港本地發行外，也在中國大陸發行，香港報刊極小數能經中國政府批准在中國內地發行，包括香港《文匯報》的網站，也是中國政府少數「解禁」的香港媒體網站。香港《文匯報》還在東南亞各國、北美洲出版多個不同的海外版本〔美洲版、印尼「東南亞版」、加拿大版《今日中國·文匯報》、泰國「亞洲·文匯合作專版」、菲律賓版，歐洲航空版、馬來西亞「南洋商報·文匯專版」、北馬版、東馬版、緬甸版〕。並聲稱日發行量共計超過40萬份，讀者總人數高達150萬。但這並沒有得到獨立機構調查的確切數據。

《文汇报》最早在2006年3月中在中国[[珠江三角洲地区|珠三角地区]]开始发行试验性质的“铁路通”版，不到半月后就开始试刊发行“商务精华版”。2006年6月28日正式创刊香港《文汇报》商务精华版。《文汇报》商务精华版售价是3元[[人民币|人民币]]，起初试刊是出纸7张，后正式试刊后改为出纸6张。

2016年，《文匯報》開設[[Facebook|Facebook]]專頁，並成立網絡媒體“[[點新聞|點新聞]]”，官方網站於2016年8月17日正式運作。
