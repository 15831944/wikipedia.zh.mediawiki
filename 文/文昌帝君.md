'''文昌帝君'''，即'''文昌武烈梓潼帝君'''，簡稱'''梓潼帝君'''、'''文昌君'''，是保護文運與[[考試|考試]]的[[神祇|神祇]]。中國有「北[[孔子|孔子]]、南文昌」之說，可見南方文昌帝君信仰之盛。文昌帝君是'''蜀王張育'''、'''梓潼神亞子'''與'''文昌星君'''等三個神靈合併而成的[[神明|神明]]。時人認為「[[張育|蜀王張育]]」即為[[梓潼神|梓潼神]]「亞子」的[[轉世|轉世]][[化身|化身]]，梓潼神為[[文昌星|文昌星君]][[轉世|轉世]]，三神合併，故稱其「張亞子」，受士人學子敬奉<ref>[http://www.twowin.com.tw/lucky/wen1_1.php 認識文昌-考得廟] .育達教育網 .[2016-03-20] {{zh-tw}}</ref>。文昌帝君與[[大魁帝君|大魁帝君]]、[[朱衣帝君|朱衣帝君]]、[[孚佑帝君|孚佑帝君]]、[[關聖帝君|關聖帝君]]，合稱為「[[五文昌|五文昌]]」。

[[巴蜀|巴蜀]]地區（今[[四川|四川]]、[[重慶|重慶]]），認為文昌帝君為[[武神|武神]]、[[境主|境主神]]、[[瘟神|克瘟之神]]。

==封敕==
[[唐玄宗|唐玄宗]]封'''左丞相'''、[[唐僖宗|唐僖宗]]封'''濟順王'''，賜[[尚方寶劍|尚方寶劍]]，[[宋代|宋代]]封為'''神文聖武孝德忠仁王'''。[[元朝|元朝]]封為'''輔元開化文昌司祿宏仁帝君'''。

[[道教|道教]]尊為'''梓潼英顯武烈濟順王'''、'''七曲山雷澤神龍濟渡大王'''、'''九天去邪除瘟宏仁教忠布孝祖師'''、'''天曹桂籍司祿真君'''、'''九天六陽七曲三宮文昌武烈梓潼帝君'''、'''司祿主者權職貢舉真君'''、'''開化保德宏仁大帝'''、'''消劫行化更生永命天尊'''等，可見其有護持文運、保命護身、教忠教孝等功能。

==源流==
===蜀王張育===
{{Main|張育}}
[[東晋|東晋]][[寧康|寧康]]二年（374年），[[蜀|蜀]]人[[張育|張育]]起兵抗擊[[前秦|前秦]][[苻堅|苻堅]]，自立為蜀王，英勇[[戰死|戰死]]。蜀人在[[梓潼|梓潼]][[七曲山大庙|七曲山]]建張育祠，認為他是[[龍神|龍神]]的化身，尊奉他爲「雷澤[[龍王|龍王]]」。

===梓潼神亞子===
當時，梓潼七曲山另有梓潼神亞子祠，就在「張育祠」的附近，逐漸相傳「張育」就是「張亞子」的化身。

傳說，亞子姓張，又名「堊子」，或「惡子」。《[[華陽國志|華陽國志]]》記梓潼縣有“善板祠”，供張亞子。《[[太平寰宇記|太平寰宇記]]》又記有張亞子顯靈的故事。張亞子曾經在[[長安|長安]]見到姚萇，張亞子對他說：“九年後，君當入蜀，若至梓潼七曲山，希望您來找我。”。《十六國春秋輯補·後秦錄》則說，[[姚萇|姚萇]]於[[前秦|前秦]][[建元|建元]]二年（366年）果然來到七曲山，見到一神人，神人說：「回[[秦|秦]]地去罷！秦地無[[君主|主]]，[[君主|君主]]大概就是您罷！」[[姚萇|姚萇]]請問那神人的姓名，神人說他叫「張亞子」，說罷就不見了。姚萇回到[[秦|秦]]地後果然建立[[后秦|后秦]]，[[登基|登基]]稱[[帝|帝]]，于是就在秦地立「張相公廟」以祀張亞子。

[[唐朝|唐朝]]張亞子的信仰非常盛行。[[安史之亂|安史之亂]]時，[[唐玄宗|唐玄宗]]走避蜀地，宿[[七曲山|七曲山]]，曾被張亞子[[託夢|託夢]]，說玄宗不久將做[[太上皇|太上皇]]。後來[[唐肅宗|唐肅宗]]果然自行[[即位|即位]]，夢境靈驗，故玄宗對張亞子舉行隆重[[祭祀|祭祀]]，追封「[[左丞相|左丞相]]」。現七曲山尚有唐玄宗“應夢仙台”遺迹。[[廣明|廣明]]二年（881年），[[唐僖宗|唐僖宗]]避[[黃巢之亂|黃巢之亂]]到蜀地，經過七曲山，也親自祭祀梓潼神，追封張亞子爲'''濟順王'''，幷將自己的[[尚方劍|尚方劍]]解下來贈廟。由于[[唐朝|唐朝]][[天子|天子]]的推崇，張亞子的影響迅速擴大，逐漸由地方神演變爲全國性的大神。

===蜀王、梓潼神、文昌星信仰混合===
[[File:明德化何朝宗作文昌帝君坐像.jpg|thumb]][[德化|德化]][[何朝宗|何朝宗]]作文昌帝君坐像]]
[[中國|中國]]各地參加[[科舉考試|科舉考試]]的學子，都會[[祭祀|祭祀]]自己家鄉的[[神靈|神靈]]，祈求自己的[[考試|考試]]順利。在[[北宋|北宋]]時，[[梓潼神|梓潼神]]轉化為保佑[[四川|四川]]地區學子，考試順利的人格神祇。[[南宋|南宋]]時，梓潼神被認為較靈驗，而取代了其它地方[[科舉|科舉]]之神，成為[[中國內地|中國內地]]學子，共同的保佑科舉順利的神祇。由於信仰的傳佈，使得原本的梓潼神信仰，與傳統觀念中[[掌管|掌管]]文學的[[文昌星|文昌星]]相混合，（或曰：梓潼神被視為文昌星[[轉世|轉世]]、[[化身|化身]]），而梓潼神被稱為文昌神、文昌帝君或文昌梓潼帝君。而蜀地靈驗之下，[[宋朝|宋朝]][[帝王|帝王]]多有[[敕|敕]]封，如[[宋真宗|宋真宗]]封為'''英顯武烈王'''，[[宋光宗|宋光宗]]時封為'''忠文仁武孝德聖烈王'''，[[宋理宗|宋理宗]]時封為'''神文聖武孝德忠仁王'''。

[[元仁宗|元仁宗]]延祐三年（1316年）終於正式[[詔|詔]]封張亞子為'''輔元開化文昌司祿宏仁帝君'''。於是'''梓潼神張亞子'''遂被稱為'''文昌帝君'''，與[[文昌星|文昌星]]信仰完全合併。《[[明史|明史]]》禮志稱，「[[梓潼帝君|梓潼帝君]]，姓[[張|張]]，名亞子，居蜀七曲山，仕晋戰歿，人為立廟祀之」。講的其實是張育，不過民間俗信張育是張亞子的化身，並無對信仰造成妨礙。

《文昌帝君陰騭文》稱，文昌帝君曾一十七次[[化身|化身]]人間，世為[[士大夫|士大夫]]，為官[[清廉|清廉]]，從未酷民，同秋霜白日之高潔不可侵。「濟人之難，救人之急，憫人之孤，容人之過，廣行陰騭，上格蒼穹」。因此，[[天帝|天帝]]命文昌帝君掌天曹桂籍文昌之事。凡[[人間|人間]]之鄉舉里選，服色祿秩，封贈奏予等等，都歸文昌帝君管理。

[[巴蜀|巴蜀]]地區相傳文昌帝君化身瘟祖大神，降伏五瘟神，能克服[[瘟疫|瘟疫]]。

===其他職能===
文昌帝君戎馬[[起義|起義]]，抗擊[[胡人|胡人]]，為國戰死，有忠君救民之功績，被當地視為[[武神|武神]]。

文昌帝君為梓潼縣的[[境主|境主]]神，許多[[鄉民|鄉民]]在遇到冤屈不平時，會向文昌帝君申訴，一如[[城隍爺|城隍爺]]。

《梓潼帝君化書》稱，文昌帝君「生及冠，母病疽重，乃為吮之，並於中夜自割股肉烹而供，母病遂痊。」因此，文昌帝君也是慈祥孝親的楷模。

文昌帝君在[[瘟疫|瘟疫]]流行時，得到「神授以《大洞仙經》並法箓，謂可治邪祛瘟，行之果驗」。另外，[[四川|四川]]地區相傳'''文昌帝君'''在梓潼七曲山祖廟隱修時，遇見五[[瘟神|瘟神]]欲來散行[[瘟疫|瘟疫]]，為了保護民眾，於是他化身為'''瘟祖大神'''，面貌威武果敢，一手持[[寶劍|寶劍]]（或[[如意|如意]]），另一手變為鷹爪，因而降伏五[[瘟神|瘟神]]。

==造像與陪祀神座騎==
[[File:明末黄杨木雕文昌帝君坐像.jpg|thumb]][[黄杨木雕|黄杨木雕]]文昌帝君坐像]]
一般文昌帝君以文官[[大臣|大臣]]面貌出現，持[[玉板|玉板]]或[[如意|如意]]，亦有持筆、持書、持扇、甚至執[[拂塵|拂塵]]的造型。[[新莊文昌祠|新莊文昌祠]]有文昌帝君作為[[儒生|儒生]]模樣，戴笠乘[[祿馬|祿馬]]趕路的造型。

陪祀神為「[[天聾|天聾]]」、「[[地啞|地啞]]」兩位[[書僮|書僮]]打扮的神祇，一人持[[毛筆|毛筆]]，一人持簿冊。代表著「天機不可洩漏」、「文運人不能知」、「文人須謙卑少言」等意義。《[[歷代神仙通鑑|歷代神仙通鑑]]》曰：「[[梓潼真君|梓潼真君]]道號六陽，每出駕白騾，隨二童，曰天聾、地啞。真君為文章之司命，貴賤所繫，故用聾、啞於側，使其知者不能言，言者不能知，天機弗洩也 。」這裏還提到了文昌帝君駕「白騾」。

另外，文昌帝君的前身張育，乃[[蜀地|蜀地]]的[[將領|將領]]，故亦有[[戎裝|戎裝]]披甲，騎[[戰馬|戰馬]]，世稱「祿馬」，相傳能帶福祿而來，俗言：「祿馬得得跑，官位步步升。」 在四川，這座騎被稱為「[[驢特|驢特]]」，是一隻似馬而非馬的神獸，又稱「[[四不像|四不像]]」，載著帝君往來四方作戰，講學雲遊。

[[四川人|四川人]]認為，文昌帝君化身為'''瘟祖大神'''時，面貌威武果敢，一手持[[寶劍|寶劍]]（或[[如意|如意]]），另一手為鷹爪。

== 出生地 ==
[[四川|四川]][[越巂|越巂]]是文昌帝君張亞子的誕生地，亦是其故鄉。其影響深遠，[[歷史|歷史]]悠久。據統計各地建造的文昌宮、閣、樓、殿，數以千計，遍佈[[中國大陸|中國大陸]]、[[港澳|港澳]]和[[臺灣|臺灣]]、[[東南亞|東南亞]]一帶。

據《越西廳志》載：「張亞子晉太康八年，七十一化降生在中所盧林溝的張老夫婦家中，後勤學苦練，羽化成神」。張亞子出生地--越西中所水鎮：古時水觀音泉邊山崖上，生長有一株婆娑古樹，開的花兒如　桃花，春天嫣紅，秋天翠碧，掩映著水潭。此樹人們稱血滕樹，傳說枝條折斷後，帶淌出血漿樣的汁，甚是奇異。樹的石崖另有一處刻字「勝景清絕」。

越西金馬山上曾留有勝跡「紫府飛霞洞」、「上馬石」等。上馬石石上曾留有腳印，據說是踏石，上「驢特」留下的印跡。石上鐫刻有「文昌勝跡」四字。另刻字「泉湧月明」相傳為帝君親書。又有鐫字於帝君賞憩處：「金闕化身」守「泉湧月明」數字。

== 相關著作 ==
{{维基文库|文昌帝君陰騭文}}

成書於宋元時的《文昌帝君陰騭文》簡稱“[[陰騭|陰騭]]文”，是一篇[[駢文|駢文]]，以文昌帝君本人的口吻來講述其化身與訓示，列舉古代士人行善得福的事例，說明善有善報、惡有惡報，「近報則在自己，遠報則在兒孫」的[[因果|因果]][[報應|報應]]，勸人[[行善|行善]]積德。《文昌帝君陰騭文》在傳統社會幾乎無人不曉。[[鲁迅|鲁迅]]在《二十四孝圖》等作品中，就提到了本书。在清代，《文昌帝君陰騭文》與《[[太上感应篇|太上感应篇]]》及《關聖帝君覺世真經》合稱「三聖經」。

== 奉祀 ==
[[File:台灣桃園市文昌帝君聖像.jpg|thumb]][[桃園市|桃園市]]文昌帝君聖像]]
元明以後，隨著[[科舉制度|科舉制度]]的規模化和制度化，對於文昌帝君的[[奉祀|奉祀]]也漸普遍。各地都建有[[文昌宫|文昌宫]]（梓潼宮）、文昌閣或文昌祠等等，其中以[[四川|四川]][[梓潼|梓潼]]七曲山的[[七曲山大廟|七曲山文昌宮]]規模最大。[[明|明]]末[[萬曆帝|萬曆帝]]甚至在[[京師|京師]]敕建[[燕京梓潼宮|燕京梓潼宮]]，供入京[[會試|會試]]的[[舉人|舉人]]禮拜求福。

不僅於各大[[都會|都會]]，鄉間[[書院|書院]]和[[私塾|私塾]]也都供奉文昌[[香火|香火]]或神像、[[神位|神位]]，其間雖時有興廢，但因文章司命，是世人貴賤所繫，所以一直奉祀不衰。舊時每年[[農曆|農曆]]二月初三日為文昌帝君神誕之日，[[公家|公家]]和當地文人學士都要到供奉文昌帝君的廟宇奉祀，或吟[[詩|詩]][[作文|作文]]，舉行'''文昌會'''。

在文昌帝君的七曲山祖廟中，也奉祀了文昌帝君的化身'''瘟祖大神'''造像。每逢疫病流行，時禱時應。

在臺灣地區文昌信仰廣布，或因應考生應試祈願之需要，一般具規模的大型廟宇通常設有文昌殿，或配祀文昌帝君。
