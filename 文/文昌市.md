'''文昌市'''是[[中国|中国]][[海南省|海南省]]东北部沿海下辖的一個[[县級市|县級市]]，政府所在地在文城镇。面积2403平方公里，人口55.8万。东南北三面临海，海岸线长210千米，沿海多天然良港和沙滩。出產之[[文昌鷄|文昌鷄]]被稱爲[[海南四大名菜|海南四大名菜]]之首。

文昌市属海南省省直辖县级行政单位。

== 历史 ==
在[[漢武帝|漢武帝]]元封元年（公元前110年）时建'''紫贝县'''，是[[海南島|海南島]]三大歷史古邑之一，亦是海南閩南文化發源地。隋大業三年（公元607年）改稱'''武德縣'''。唐高祖武德五年，改稱'''平昌縣'''。[[唐太宗|唐太宗]]贞观元年（公元627年）才定名為'''文昌縣'''，历经2000多年，直至1995年11月7日撤县建市。

== 地理 ==
文昌地处热带边缘，属[[热带季风气候|热带季风气候]]。沿海地区风较大，夏秋季影响海南岛的[[台风|台风]]有75%是从文昌登陆，故有“台风走廊”之称<ref>[http://www.chinaquhua.cn/hainan/wenchang.html 海南文昌市簡介] {{webarchive|url=https://web.archive.org/web/20111128133002/http://www.chinaquhua.cn/hainan/wenchang.html |date=2011-11-28 }}</ref>。

== 旅遊 ==
文昌市东南北三面临海，海岸线长210千米，沿海多天然良港和沙滩，擁有豐富旅遊資源，如：馮家灣、高隆灣、月亮灣、后港灣、八門灣紅樹林、七星嶺、銅鼓嶺、抱虎嶺、[[東郊镇|東郊]]椰林、[[椰子大觀園|椰子大觀園]]、[[七洲列島|七洲列島]]等都是著名風景區<ref>[http://www.hiwenchang.com/forum-38-1.html 文昌景點]</ref>。 

文昌市具有深厚歷史文化底蘊，主要人文景點有：[[文南老街|文南老街]]、[[陳策將軍祖居|陳策將軍祖居]]、[[宋氏祖屋|宋氏祖屋]]、[[溪北書院|溪北書院]]、[[孔廟|孔廟]]等。

飲食文化方面，文昌市是海南四大名菜<ref>{{cite web |author1=海南香 |title=海南四大名菜 |url=http://hncx.anhees.com/HainanFood/hnsdmc |website=http://hncx.anhees.com}}</ref>中文昌鷄的發源和主要出產地。文昌鷄的概念，廣義上包含文昌鷄這種本地雞種、文昌鷄的烹飪文化。文昌鷄的烹飪文化，包含了文昌鷄飯的做法，這種做法，是聞名世界的海南鷄飯的源頭。

[[文昌航天发射场|文昌航天发射场]]地處[[龍樓鎮|龍樓鎮]]，為中國四大航天發射場之一，是文昌市的新兴航天旅游景点。

== 著名人物 ==
*[[宋氏家族|宋氏家族]]：[[宋嘉澍|宋嘉澍]]（1863年－1918年），[[實業家|實業家]]及[[傳教士|傳教士]]，他是[[宋靄齡|宋靄齡]]、[[宋慶齡|宋慶齡]]、[[宋子文|宋子文]]、[[宋美齡|宋美齡]]、[[宋子良|宋子良]]、[[宋子安|宋子安]]之父。
*[[陳策_(民國)|陳策]]（1892年－1949年）：中華民國海軍中將，國民黨海軍中廣東艦隊的名將、戰後首任任[[廣州市|廣州市]]市長。曾協助[[孫中山|孫中山]]在廣州對抗[[陳炯明|陳炯明]]，抗戰時負傷截斷一腿，有「獨腳將軍」之稱。日軍攻陷香港時，陳策帶領數十名英軍成功突圍，獲英皇頒授[[大英帝國勳章|大英帝國爵級司令勳章]]。
*[[鄭介民|鄭介民]]（1897年9月18日－1959年12月11日）：[[中華民國陸軍|中華民國陸軍]][[一級上將|一級上將]]。
*[[张云逸|张云逸]]（1892年－1974年）：[[中国人民解放军大将|中国人民解放军大将]]。
*[[陈序经|陈序经]]（1903年－1967年）：著名[[思想家|思想家]]、[[社会学家|社会学家]]、[[历史学家|历史学家]]，在社会学、经济学、民族学、历史学等方面都有独到研究。
