'''文素勤'''{{BD|1867年|2月12日|1919年|10月5日|catIdx=W文素勤}}，[[小名|小名]]'''七妹'''（'''其美'''），出生于[[清朝|清朝]][[湖南省|湖南省]][[长沙府|长沙府]][[湘乡县|湘乡县]]棠佳阁，[[毛贻昌|毛贻昌]]的妻子，[[毛泽东|毛泽东]]的母亲。<ref>{{cite web |url=http://news.xinhuanet.com/theory/2009-03/13/content_11003015.htm |title=毛泽东与父亲毛贻昌、母亲文素勤 |accessdate= 2009年3月13日|publisher= 新华网|work= |language =zh-cn}}</ref>

== 生平 ==

1867年，她出生在[[湖南省|湖南省]][[长沙府|长沙府]][[湘乡县|湘乡县]]棠佳阁，父亲文芝仪，母亲贺老太太。根据其子毛泽民填写的《履历表》显示，她的[[本名|本名]]是文素勤<ref>{{Cite web |url=http://culture.people.com.cn/GB/40479/40480/8980792.html|title=严父慈母:毛泽东与父亲毛贻昌、母亲文素勤 |accessdate=2014-12-14 |author= |date=2009年3月18日 |work= |publisher=人民网，来源：中国共产党新闻网|language=Zh-hans }}</ref>，而非文七妹（文其美）。她的母亲贺老太太生育两子三女，长子文玉瑞，次子文玉钦，大女嫁给钟家，二女嫁给王家，-{么}-女七妹嫁给韶山土地冲毛家毛贻昌。

1919年，文素勤因为淋巴腺炎来到长沙治疗，居住在长沙河西刘家台子[[蔡和森|蔡和森]]的家里，10月5日，因为病情发作，文素勤逝世。<ref name=" fish" />
