'''文艺报'''，报社社址[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]]农展馆南里10号6层，是由[[中国作家协会|中国作家协会]]主管主办的综合性文学艺术类报纸<ref name=shige/>。

==简介==
1949年5月4日，《文艺报》创刊于[[北平|北平]]（今北京），是文学周刊；1949年7月28日出到第13期停刊<ref name=shige/>。该报是[[中华全国文学艺术工作者代表大会|中华全国文学艺术工作者代表大会]]会刊，由中华全国文学艺术工作者代表大会筹备委员会文艺报编辑委员会编辑，主编是[[茅盾|茅盾]]，编委有[[胡风|胡风]]、[[丁民|丁民]]（严辰）。创刊号的套红报头由[[丁聪|丁聪]]设计<ref name=shoucang>{{Cite web |url =http://roll.sohu.com/20120419/n341023938.shtml |title = 一波三折的收藏经历 |author =  |date =  2012-04-19|publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

1949年6月30日至7月28日，第一次中华全国文学艺术工作者代表大会召开。1949年7月19日，[[中华全国文学艺术界联合会|中华全国文学艺术界联合会]]（简称中国文联）正式成立<ref>{{Cite web |url = http://guoqing.china.com.cn/2014-12/02/content_34203691.htm |title = 中国文学艺术界联合会简介 |author =  |date =2014-12-02  |publisher = 中国网 |language =  |accessdate =  |quote =  }}</ref>。1949年7月23日，中华全国文学工作者协会（简称全国文协，是中国作家协会的前身）在北平成立<ref>{{Cite web |url = http://www.chinawriter.com.cn/n1/2016/0703/c403958-28519041.html |title =  历史沿革|author =  |date =2016-07-03  |publisher =中国作家网  |language =  |accessdate =  |quote =  }}</ref>。1949年9月25日，《文艺报》复刊，是半月刊，为中华全国文学艺术界联合会的机关刊物。1957年，中国文学艺术界联合会主席团决议将《文艺报》委托中国作家协会主办。1957年4月改成周刊。1958年1月改成半月刊。1961年改成月刊。1966年，因[[文革|文革]]爆发，出到第342期停刊。1978年7月复刊，是月刊。1981年1月改成半月刊。1982年恢复成月刊。1985年7月改成周报。到2010年代，《文艺报》已成为对开大报，每周出版三期<ref name=shige>{{Cite web |url =  http://zgshige.com/c/2016-06-15/1378947.shtml|title = 文艺报社简介 |author =  |date = 2016-06-15 |publisher = 中国诗歌网 |language =  |accessdate =  |quote =  }}</ref><ref name=shoucang/>。

[[中华人民共和国|中华人民共和国]]建国初期的文艺风云均和《文艺报》有关系，[[毛泽东|毛泽东]]、[[邓小平|邓小平]]等[[党和国家领导人|党和国家领导人]]对《文艺报》的工作有过指示。《文艺报》在中国文艺界具有历史形成的影响力<ref name=shige/>。
