'''文化霸權'''（{{lang-it|Egemonia culturale}}，{{lang-en|Cultural hegemony}}）也称为'''文化领导权'''、'''领导权'''，是[[義大利|義大利]]马克思主义者[[安東尼奧·葛蘭西|安東尼奧·葛蘭西]]所提出的[[哲学|哲学]]和[[社会学|社会学]]理论。它指的是：一个[[社会阶层|社会阶层]]可以通过操纵社会文化（信仰、解释、认知、价值观等），支配或统治整个[[多元文化|多元文化]]社会；[[统治阶级|统治阶级]]的[[世界观|世界观]]会被強制作為唯一的[[社会规范|社会规范]]，并被认为是有利于全社会的普遍有效的思想，但实际上只有统治阶级受益。階級都要透過其知識分子在公民社會爭取到知性與道德上的領導權。

== 歷史追溯 ==
文化领导权最初由[[普列汉诺夫|普列汉诺夫]]在1883年-1884年间作为推翻[[沙皇|沙皇]]的策略之一提出的。其后，[[列宁|列宁]]使用这个概念指出，[[无产阶级|无产阶级]]不要躲避[[资产阶级|资产阶级]]的革命，而要领导社会各阶级去争取推翻沙皇的胜利，同时不能放弃对资产阶级革命的领导权。之后，经过葛蘭西的完善，才成为了一个[[概念|概念]]。

== 葛蘭西的文化霸權理論 ==
葛蘭西在[[二十世紀|二十世紀]]初期，見到[[十月革命|十月革命]]的成功，以及西方[[資本主義|資本主義]]國家並未發生如[[馬克思|馬克思]]所預期的發生[[革命|革命]]，因此提出了文化霸權的概念來作進一步的探討，希望能夠對資本主義社會作更深入的了解，並依此來訂立更有效的革命方針。

根據馬克思與[[恩格斯|恩格斯]]的理論，資本主義將不可避免地導向[[無產階級革命|無產階級革命]]，而且必然先在[[工業化|工業化]]的歐洲爆發；而由於資本主義生產方式的本質，在這過程中將產生與革命運動相應的文化與政治上層建築。雖然馬克思早在1848年就如此預言，百年後的工業化國家工人仍然沒能達成這個任務。

葛蘭西針對馬克思主要強調[[社會|社會]][[經濟|經濟]]結構、而對[[思想|思想]][[文化|文化]]等[[上層建築|上層建築]]的影響以及個人的主動性較不重視的情形加以修正。他認為下層結構是「制約」而非「決定」上層結構。他提出的文化霸權理論，指出了一個[[政權|政權]]的維持，是需要政治的強制力加上霸權文化的力量配合；而後者來自於在[[市民社會|市民社會]]的配合之下，以包括如[[教育|教育]]、大眾[[傳播媒體|傳播媒體]]等對於大眾的潛移默化，造成了工人階級的[[虛假意識|虛假意識]]，使此一霸權得以維持。在所謂先進工業化國家中，文化霸權的力量尤其隱密而強大。因此，要霸權有所轉變，必須由[[知識分子|知識分子]]先以所謂的[[陣地戰|陣地戰]]進行長期準備，與社會上的許多集團合作以及並逐漸促使其[[意識形態|意識形態]]的轉化，取得了此一基礎之後，方能進一步使霸權轉移。

葛蘭西認為在朝與在野階級透過各自的知識份子，必須爭取到對公民社會在心智上與道德上的領導權，他甚至認為一定先要有霸權，才能進一步取得政權，也就是在西方政治思想中合法壟斷武力的權利。每個利益團體(意識形態)要透過爭霸來讓他人任肯。

儘管葛蘭西的文化霸權論起源於對[[階級鬥爭|階級鬥爭]]狀況的分析，這套理論可以應用得更廣。根據葛蘭西的理論，當前盛行的文化範式既非自然的，也非必然的；相反地，我們必須從包括機構、實踐及信念等等的文化範式中分析其支配結構。

文化霸權並不是單獨的或統一的，它是社會結構的分層展現。其中的每一種結構都有各自的「使命」與內在邏輯。然而一如軍事組織，每個結構都假設了其它結構的存在，並彼此結合以達成更高的全局使命。霸權便是如此運作。社會中的每個人都有各自的生活方式，對於個人，社會的其它部份貌似無關緊要。然而整體而言，每個人的生活都構成霸權結構的一環。在日常生活的瑣碎活動中，多樣性與自由意志看似存在；而正是藉著無數這樣的情境，巨觀的霸權結構得以維持，而不為生活在其中的人們所察覺（見葛蘭西《獄中札記》，頁233-38）

葛蘭西的這種理論能更進一步解釋社會文化與政權變遷的過程，並且更強調人尤其是知識分子的主動性作用，可以彌補馬克思過於重視社會經濟結構的不足之處。

== 影響 ==
文化霸權理論對[[歐洲共產主義|歐洲共產主義]]、[[社會科學|社會科學]]與社會運動有深遠影響。在社會科學領域，[[米歇尔·福柯|米歇尔·福柯]]等人應用霸權的概念，深入研究了主流論述。在[[教育學|教育學]]中，霸權理論導向了[[批判性教學|批判性教學]]。

此后，[[后马克思主义|后马克思主义]]的代表人物[[厄尼斯特·拉克劳|厄尼斯特·拉克劳]]和[[尚塔尔·墨菲|尚塔尔·墨菲]]对葛蘭西的這種理論重新进行了思考，批评葛蘭西的理論中存在[[本质主义|本质主义]]问题，坚持领导权的阶级主体性，而忽视了非阶级力量之间的斗争，过分强调领导权在社会斗争中的中心性和单一性。
