'''文澳祖師廟'''，主祀[[清水祖師|清水祖師]]陳昭應，今為臺灣[[澎湖縣|澎湖縣]][[馬公市|馬公市]]西文里的公廟，聚落四周設有六營以保境安民。<ref name=":0">{{Cite web|url=https://penghu.info/OB31D2DA20E259DF29FD|title=祖師廟|accessdate=2018-04-07|author=|date=|publisher=Penghu.Info｜澎湖知識服務平台}}</ref>根據乾隆卅二年（1767年）[[胡建偉|胡建偉]]總編的《[[澎湖紀略|澎湖紀略]]》紀載，祖師廟建造於康熙年間，相傳當時有一僧侶自[[泉州市|泉州]]清水巖東渡，來到文澳地區替居民治病，且不收取分文，故設立此廟感念。<ref>{{Cite book|title=《澎湖紀略》|last=胡|first=建偉|publisher=遠流|year=2004|isbn=9789570193718|location=臺北市|pages=}}</ref>

== 沿革 ==
祖師廟建造於原澎湖廳署西側，坐北朝南、面海背山，風水甚佳。雖今日已是新式廟宇的建築外觀，究其歷史淵源，祖師廟同[[澎湖水仙宮|澎湖水仙宮]]、[[澎湖觀音亭|澎湖觀音亭]]、[[施公祠|施公祠]]與[[東甲北極殿|東甲北極殿]]等，皆為澎湖在康熙年間已有文獻紀錄的祠廟。<ref>{{Cite book|title=《澎湖廳志．規制》|last=林|first=豪|publisher=成文|year=1983|isbn=|location=台北市|pages=}}</ref>

根據胡建偉《[[澎湖紀略|澎湖紀略]]》和[[蔣鏞|蔣鏞]]《澎湖續編》所載，關於文澳祖師廟的興建起源以及於乾隆廿九年（1764年）與嘉慶十六年（1811年）修建的紀錄中，其整修多為民間里人募捐出資<ref name=":0" />，庶民色彩更形濃厚。

民國48年（1959年）祖師廟三度重建，民國54年（1963年）里人葉順鴻等再勸捐重修，民國75年（1986年）遭逢[[颱風韋恩_(1986年)|韋恩颱風]]侵襲，廟舍漏水嚴重，里人陳明廷、林財達與林永安等籌組重建委員會，亦延請文澳清水祖師分靈至[[大港埔祖師廟|高雄大港埔祖師廟]]的負責人蔡進宗為委員。1993年動工，1997年12月21日歷時四年八個月終告竣工。落成後新廟亦採坐北朝南，廟宇樣式即為現狀。<ref name=":0" /><!-- 引用參酌碑文紀錄 --><gallery mode="nolines" widths="150" heights="150">

== 祖師廟與鸞堂的淵源 ==
1895年（光緒21年），時逢[[乙未戰爭|乙未戰爭]]，澎湖鸞堂始祖「[[一新社|一新社]]」為躲避兵燹，遷至西文澳的民宅扶鸞宣講，另於1902年（明治35年）在祖師廟東鄰創辦「聖真寶殿」。文澳祖師廟受一新社與[[文澳聖真寶殿|文澳聖真寶殿]]的影響，亦於1921年（大正10年）開辦鸞堂，即「登岸社禮善堂」。直到民國55年（1966），一新社另行於馬公市區籌辦廟宇「一新社聖真寶殿」，「登岸社禮善堂」隨之遷移到文澳聖真寶殿。<ref>{{Cite web|url=https://penghu.info/OB19C319B55971FD030A|title=聖真寶殿|accessdate=2018-04-07|author=許玉河|date=|publisher=Penghu.Info｜澎湖知識服務平台}}</ref><ref>{{Cite book|title=《續修澎湖縣志．卷十二．宗教志》|last=余|first=光弘|last2=黃有興|first2=有興|publisher=澎湖縣政府|year=2005|isbn=9860015198|location=澎湖縣|pages=48|language=zh-tw}}</ref>

此外，1993年祖師廟修葺期間，暫駐行館於[[文澳聖真寶殿|文澳聖真寶殿]]，原廟四垂亭亦移架聖真寶殿使用。<ref>〈文澳祖師廟重建碑記〉，甘村吉撰於1987年。</ref><!-- 引用參酌碑文紀錄 -->

文澳祖師廟正廳尚存有《喚醒善導》手冊四本，亦為鸞堂善書。
== 匾額文物 ==
澎湖地區盛行恭請四方庇佑的小法儀式時時會在祖師廟前廣場舉行。此外，文澳祖師廟內懸掛三個清代便保存下來的匾額文物，分別是「清溪垂庇匾」、「一方之庇匾」與「千江明月匾」。除了「清溪垂庇匾」之外，「一方之庇匾」與「千江明月匾」皆有落款明確的立匾年份。

=== 清溪垂庇匾 ===

置於祖師廟正殿上方的「清溪垂庇匾」，由於僅落款「乙巳年」，廟方立碑之際，便將祖師廟立廟歷史推至康熙四年（1665），即[[明鄭時期|明鄭]]永曆19年便已立廟。<!-- 引用參酌碑文紀錄 -->澎湖文史工作者許玉河則認為此匾所指的乙巳年應係雍正三年（1725）或乾隆五十年（1785），推論之依據在於清溪垂庇匾下款「四甲眾弟子敬叩」之「四甲」二字；「甲」常見於清代澎湖聚落的自治單位，具有一定的人口規模才能形成四個甲的單位組織，但依照[[明鄭時期|東寧政權時期]]治澎政策，[[延平郡王|延平郡王]][[鄭經|鄭經]]係將澎湖島佈署為軍事基地，兵多於民，並不太可能形成高達「四甲」的村莊規模<ref>{{Cite web|url=https://penghu.info/OBDA8F98AA6630D1DD59|title=清溪垂庇匾|accessdate=2018-04-07|author=許玉河|date=|publisher=Penghu.Info｜澎湖知識服務平台}}</ref><ref>{{Cite book|title=《續修澎湖縣志．卷2．地理志》|last=顏|first=尚文|publisher=澎湖縣政府|year=2005|isbn=9860015104|location=澎湖縣|pages=}}</ref>，故提出立匾之乙巳年為1725年或1785年的主張。<gallery mode="packed-hover" widths="150" heights="150">

== 文澳祖師廟夜市 ==
[[File:文澳祖師廟｜廟埕廣場、夏季夜市.jpg|缩略图]]
馬公市文澳地區周遭為海埔新生地，早期多為住宅區、商店不多，文澳廟埕前腹地廣大，便開放些許攤販在此擺攤兼賣點心、宵夜。近年來由於大型飯店與民宿進駐林立，觀光人潮湧入文澳地區，祖師廟因應地方需求，於2016年起開辦夏季觀光夜市<ref>{{Cite news|url=http://www.penghutimes.com/modules/phnews/index.php?nsn=10639|title=文澳祖師廟觀光夜市規模不輸士林夜市|last=鄧|first=至傑|date=2016-04-24|work=澎湖時報|accessdate=2018-04-07|language=zh-tw}}</ref>，在澎湖宮廟中亦屬創舉。
