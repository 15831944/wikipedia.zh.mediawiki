{{汶莱政治}}
'''文莱'''的'''政治'''制度，始於其沿袭自14世纪以来相同的苏丹王朝，是国家和政府的领袖。雖然汶萊一度淪為[[英國|英國]]的[[保護地|保護地]]，但汶萊蘇丹在當地的統治並未有中止過。汶萊的政治體制源自英國的承傳，所以亦奉行[[行政|行政]]、[[立法|立法]]及[[司法|司法]]三權分立，儘管現時當地的立法權力已被蘇丹完全接管。

== 政府 ==
根據當地於1959年訂定的憲法，[[汶萊蘇丹|汶萊蘇丹]]是[[國家元首|國家元首]]，並有完全的行政權力，包括自1962年開始被賦與的緊急權力。
== 行政 ==

== 立法 ==
根據當地於1959年訂定的憲法，當地的立法機關''Majlis Masyuarat Negeri''需由選舉產生，但自從1962年之後，未有任何新的選舉。當時政府為了禁制[[汶萊人民黨|汶萊人民黨]](''Parti Rakyat Brunei''，簡寫''PRB'')而頒布[[緊急狀態令|緊急狀態令]]，下令解散立法議會。此後，從1970年開始，立法會議的所有成員都有[[汶萊蘇丹|汶萊蘇丹]]任命。

現時汶萊政府積極推行政制改革，當中包括恢復立法會的選舉，但預計數年之內都未能成事。

== 領土糾紛 ==
現時汶萊與周邊國家裡的[[馬來西亞|馬來西亞]]和[[中華人民共和國|中華人民共和國]]有領土糾紛。

由於汶萊的國土被馬來西亞一分為二，之間的水域誰屬，存在著爭議的空間。另外，汶萊亦聲稱擁有[[砂拉越|砂拉越]]的主權，不过已经放弃。

此外，汶萊與[[南沙群島|南沙群島]]接鄰，雙方的界線在哪裡，還未有定案。中國聲稱其領土最南端的[[曾母暗沙|曾母暗沙]]，目前由汶萊控制。
