'''文科'''，全名'''人文社会科学'''，是以人类社会独有的[[政治|政治]]、[[经济|经济]]、[[文化|文化]]等为研究对象的学科。文科原於[[英語|英語]]中的'''Arts'''或'''The arts'''（-s不能省略；見英文維基中[[藝術種類|藝術種類]]條目的解釋）。在一些教育體系中，并不是所有学生都学习相同科目，因此有分科（如中學[[文理分科|文理分科]]）的情況出現，文科是指其中的一大類科目。但在不同的教育系統中，文科所包括的學科可能會有所不同。

== 不同教育系統中的「文科」==
*在[[中國高等教育|中國大陸的高等教育]]中，文科包括[[語言|語言]]、[[文學|文學]]、[[歷史學|歷史學]]、[[哲學|哲學]]、[[心理學|心理學]]、[[社會學|社會學]]、[[教育學|教育學]]、[[藝術學|藝術學]]、[[傳播學|傳播學]]等。某些學科具有[[學科交叉|學科交叉]]的成分，譬如心理學的重要分支[[認知心理學|認知心理學]]便屬於理科。[[中華人民共和國教育部|教育部]]把學科分成四大類，即文科、[[財經|財經]][[政法|政法]]、[[理工|理工]]科、[[農業|農業]][[醫藥|醫藥]]。每一類學科在教育部[[高等教育司|高等教育司]]轄下設有一個教育處<ref name = "moe_hi">[http://www.moe.edu.cn/publicfiles/business/htmlfiles/moe/s3372/201001/82437.html 高等教育司领导介绍内设处室介绍]</ref>，負責管理此類學科的事務。在[[普通高等學校招生全國統一考試|中國的高考]]中，有理科考生與文科考生之區分。文科考生需要學習的學科有語文、外語、數學（文史哲類）、政治、歷史、地理。中國大部分地區的中學多在高中二年級分為文科班和理科班，即[[文理分科|文理分科]]制度，與[[普通高等學校招生全國統一考試|高考]]對接。
*在[[台灣|台灣]]，[[普通型高級中學|普通型高級中學]]自第二學年開始分為文科及[[理科|理科]]，即社會組與自然組，另[[綜合型高級中學|綜合型高級中學]]學術學程分流之社會學程與自然學程亦同。

== 分类 ==

文科分为[[人文科学|人文科学]]与[[社会科学|社会科学]]。
* 人文科学研究人类文化遗产，其经典学科是文学、历史学、哲学；“史”包括历史、考古等；哲学是一种方法，美学、宗教学等都属于哲学范畴。
* 社会科学研究社会发展、社会问题、社会规律，是法学、教育学、经济学、管理学4个学科门的统称，共有19个学科类，120个本科目录内专业。2004年，设立社会科学本科专业的大学共有597所。社会科学是人类认识和改造人类社会的科学，它研究的对象是人类社会。

大学文科设置有[[政治学|政治学]]、[[经济学|经济学]]、[[法学|法学]]、[[哲学|哲学]]、[[历史学|历史学]]、[[文学|文学]]、[[艺术学|艺术学]]、[[外国语言|外国语言]]与[[文学|文学]]、[[新闻学|新闻]][[传播学|传播学]]、[[人类学|人类学]]、[[社会学|社会学]]、[[民族学|民族学]]、[[管理学|管理学]]、[[教育学|教育学]]等。
