'''文幼章'''（英文名：'''{{lang|en|James Gareth Endicott}}'''，音譯'''詹姆士·艾迪科特'''<ref>{{cite news |url = http://www.people.com.cn/GB/guoji/14549/1987088.html |title = 內幕：美在朝鮮戰爭中對朝中兩國進行細菌戰 |date = 2003年7月28日 |publisher = 新華社 |access-date = 2013-11-24 |archive-url = https://web.archive.org/web/20131203012552/http://www.people.com.cn/GB/guoji/14549/1987088.html |archive-date = 2013-12-03 |dead-url = no }}</ref>；{{bd|1898年||1993年||}}），出生於[[中国|中国]][[四川省|四川省]]，父母均为[[加拿大|加拿大]][[傳教士|傳教士]]。他在[[第一次世界大戰|第一次世界大戰]]期間在法國的加拿大軍隊中服務，並從[[多倫多大學|多倫多大學]]畢業，之後擔任{{link-en|加拿大聯合教會|United Church of Canada}}傳教士，帶著新娘瑪麗·奧斯汀回到中國。文家在1925年落戶[[重慶|重慶]]，他很快就學會中文。

== 生平 ==
1939年，加拿大聯合教會回應[[蔣中正|-{zh-cn:蒋介石;zh-tw:蔣中正}-]]夫人的邀請，推薦文幼章為[[新生活運動|新生活運動]]的政治顧問。

美國對日本開戰之後，美國[[戰略情報局|戰略情報局]]要求文幼章為它服務，專門負責了解中國的共產主義運動。文幼章因此結識了[[周恩來|周恩來]]，[[喬冠華|喬冠華]]等中共領導人。

文幼章回到加拿大之後，1949年擔任加拿大和平會的主席，1957年擔任[[世界和平理事會|世界和平理事會]]的副主席。他為[[中華人民共和國|中華人民共和國]]的成立而歡呼。

文幼章因為「促進基督徒與共產主義者的和平共存」，1952年獲得[[斯大林和平獎|斯大林和平獎]]<ref name="彭小年">{{cite journal |url = http://wap.cnki.net/qikan-CQSJ201118014.html |title = 人民友好使者文幼章先生 |author = 彭小年 |journal =  |access-date = 2013-11-27 |archive-url = https://web.archive.org/web/20131202231929/http://wap.cnki.net/qikan-CQSJ201118014.html |archive-date = 2013-12-02 |dead-url = no }}</ref><ref name="obituary">"OBITUARY: James Gareth Endicott Chinese-born missionary was peace activist", by Donn Downey, Globe and Mail, November 29, 1993</ref>。1965年，[[世界和平理事會|世界和平理事會]]授予[[約里奧-居里金質獎章|約里奧-居里金質獎章]]<ref name="彭小年" />。

[[冷戰|冷戰]]期間，他和妻子出版《加拿大遠東新聞通訊報》逾40年，提倡對中華人民共和國的了解和友誼。他的活動導致他與教會與[[加拿大政府|加拿大政府]]公開衝突，加拿大政府一度考慮以[[叛國罪|叛國罪]]起訴他。加拿大媒體稱他「頭號{{link-en|公敵|public enemy}}」<ref name="obituary" /><ref>{{cite web |url = http://www.canadashistory.ca/Magazine/Online-Extension/Articles/Helping-Heroes.aspx |title = ''Helping Heroes'' |at = James Gareth Endicott (1898–1993): Peacemaker and pariah |publisher = Canada's History Magazine |deadurl = yes |archiveurl = https://web.archive.org/web/20131213110953/http://www.canadashistory.ca/Magazine/Online-Extension/Articles/Helping-Heroes.aspx |archivedate = 2013-12-13 }}</ref>。

1993年，[[中国人民对外友好协会|中国人民对外友好协会]]於授予文幼章「[[人民友好大使|人民友好大使]]」[[荣誉称号|荣誉称号]]。
