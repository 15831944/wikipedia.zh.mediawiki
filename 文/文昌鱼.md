'''文昌鱼'''（[[學名|學名]]：''{{lang|la|Branchiostoma lanceolatum}}''）属于[[头索动物亚门|头索动物亚门]] [[文昌魚科|文昌魚科]] (舊分類為 [[鳃口科|鳃口科]])，沒有[[脊椎骨|脊椎骨]]，不属于[[鱼类|鱼类]]。文昌魚是[[演化论|生物演化]]研究中的[[模式生物|模式生物]]<ref>[http://homepage.ntu.edu.tw/~ntucdbrm622/1000108.pdf 似魚非魚 – 從文昌魚看脊椎動物的發育機制演化]</ref>，它揭示了现存[[脊椎动物|脊椎动物]]的起源。

文昌魚不同于香港称之为「白飯魚」的一種魚。香港白飯魚實是[[銀魚科|銀魚科]]的[[白肌銀魚|白肌銀魚]]（{{lang|la|''Leucosoma chinensis''}}）, 及[[有明銀魚|有明銀魚]]（{{lang|la|''Salanx ariakensis''}}）, 屬脊椎動物的魚類。

== 特征 ==
體長3-5厘米，外表看起來像魚類。身體半透明。同脊椎动物一样，文昌鱼具有一条沿背部下行的[[神经索|神经索]]，并具有呈条带状的[[肌节|肌节]]。然而，和脊椎动物不同的是，文昌鱼的背神经索不是由[[骨骼|骨骼]]所保护，而是由许多[[柱状细胞|柱状细胞]]所在组成的具有韧性的膜状结构所紧密包围，这种起支撑作用的结构比脊椎简单得多，又被称为[[脊索|脊索]]（notochord）。文昌鱼的脊索和脊椎动物的脊椎不同，它一直延伸进入头部，故文昌鱼所属的亚门被称作[[头索动物亚门|头索动物亚门]]（{{lang|la|''Cephalochordata''}}），其中“cephalo-”在[[拉丁文|拉丁文]]的意思就是“与头部相连”。文昌魚於頭部被稱為[[腦室|腦室]]的神經索部份比較粗大，但並不是[[腦部|腦部]]。並文昌鱼嘴的前端长有[[口笠触手|口笠触手]]（oral cirri），起到感觉器官和过滤进入口中的海水的作用。文昌魚擁有[[循環系統|血管系統]]，但沒有[[心臟|心臟]]，血液由一部份血管的脈動帶動。

== 习性 ==
文昌鱼成熟个体平均体长大约5厘米左右，寿命约2年8个月左右。广泛分布于各大洋低、中纬度海域的近岸浅水区。它们对地质要求比较严格，通常仅在有机质含量较低的粗中砂和中砂中大量出现。幼体运动能力弱，营浮游生活；成体在沙质的浅海中营半穴居生活，身体埋入砂中，仅前端外露，用于进行呼吸和滤食水体中的硅藻。夜间较为活跃，在海水中做短暂的游泳。

== 分布状况 ==
[[中国|中国]][[青岛|青岛]] (''Branchiostoma belcheri tsingtauense'')、[[厦门|厦门]] (''Branchiostoma belcheri xiameng'')是文昌鱼的两个主要栖息地。此外，整个[[东南亚|东南亚]]沿海均有发现文昌鱼。在美国[[加利福尼亚州|加利福尼亚州]]和[[佛罗里达州|佛罗里达州]]也有分布。

[[香港城市大學|香港城市大學]]曾經研究文昌魚，調查發現香港有四種文昌魚，包括:

[[白氏鰓口文昌魚|白氏鰓口文昌魚]]

白氏鰓口文昌魚青島亞種、

馬來鰓口文昌魚

短刀側殖文昌魚。

其中以白氏鰓口文昌魚最多，可見於[[西貢|西貢]][[大浪灣|大浪灣]]一帶海域，通常在水深十米的近岸海床棲息。<ref>{{cite news|url=http://hk.apple.nextmedia.com/news/art/20051016/5314398|title=
遠古生物被當白飯魚吃 文昌魚瀕危學者籲列管|newspaper=蘋果日報 (香港)|date=2005-10-16}}</ref>

== 保护现状 ==
由于对栖息地的底质及水质的要求较高，人为破坏所造成的栖息地环境改变有可能造成文昌鱼种群的消失。例如[[高集海堤|高集海堤]]的修建就直接导致了曾经年产60吨文昌鱼的刘五店渔场的消失。此外，由于文昌鱼具有丰富的营养价值，为[[东南亚|东南亚]]一带的传统食物，商业性的捕捞也加速了文昌鱼资源的枯竭。文昌鱼现为[[中华人民共和国|中国]]的[[中国国家二级保护动物|国家二级保护动物]]。
