'''文萃'''是1984年10月5日由[[湖南日报社|湖南日报社]]创办的[[周刊|周刊]]报纸，为湖南省重要报刊之一。<ref name="webfh1 " >{{cite web |url=http://news.ifeng.com/gundong/detail_2010_10/14/2779501_0.shtml |title=文萃的前世今生 |accessdate=2010年10月14日 |work=|publisher=凤凰网 |language=zh-hans}}</ref>

== 影响 ==
[[湖南省|湖南省]]委宣传部将《文萃报》列为2010年“全省首家发行过百万工程报刊”。<ref name="webfh1 " />

[[中国邮政集团|中国邮政集团]]公司2011年正式将《文萃报》列为“中国邮政重点发行畅销报刊”。<ref name="webfh1 " />

== 评价 ==
“湖南省十佳报纸”。<ref name="webfh1 " />
