'''《文藝春秋》'''（{{jpn|j=文藝春秋|kk=ぶんげいしゅんじゅう|rm=Bungeishunjū|f=no}}）是由[[日本|日本]][[文藝春秋|文藝春秋出版社]]所出版發行的一份綜合性[[月刊|月刊]][[雜誌|雜誌]]。

== 雜誌概要 ==
1923年1月，[[菊池寬|菊池寬]]私人投資創立了《文藝春秋》。<ref name="Kikuchi">{{cite web |author1=菊池寛 |language=ja|script-title=ja:『文藝春秋』創刊の辞 |url=https://web.archive.org/web/19990502212227/http://www.honya.co.jp/contents/archive/kkikuchi/kikuchimsg.html |publisher=文藝春秋 |accessdate=2018-12-31}}</ref> 但是《文藝春秋》的實際發行日期卻是在1922年的年底。雜誌初期定價為10錢，發行量為3000份，而發行商則為春陽堂。<ref>{{cite web |author1=和田篤太郎 |title=出版社不運録～春陽堂書店を中心に - 大震災前の大正は |url=http://www.asahi-net.or.jp/~bh3h-smjy/shunyo/shunyo015.htm |accessdate=2019-01-03}}</ref><ref>{{cite web |title=【1923年】出版（大正12年）▷「月刊 文藝春秋」（創刊号、春陽堂・発売元） |url=https://jaa2100.org/entry/detail/053858.html |publisher=ジャパンアーカイブズ |accessdate=2019-01-03}}</ref> 雜誌名源自菊池寬在《[[新潮_(日本雜誌)|新潮]]》上所連載的[[文藝評論|文藝評論]]專欄名。<ref name="Kikuchi" /> 隨後，受[[關東大地震|關東大地震]]的影響，已經印刷好的同年9月號雜誌被焚燒殆盡，因而不得不宣佈暫時休刊。但此時《文藝春秋》的發行量已經取得了長足進步，而且還首次開創性地使用了“座談會”這個名詞。1930年代，《文藝春秋》從文藝類雜誌轉變為綜合類雜誌。同時由於它是[[芥川龍之介獎|芥川龍之介獎]]獲獎文章的刊登雜誌，因此權威性得到了極大的提高。[[同盟國軍事佔領日本|盟總時期]]，由於菊池寬擔負戰爭責任而被“[[公職追放|公職追放]]”，《文藝春秋》一度面臨停刊危機。《文藝春秋》的主編{{Link-ja|池島信平|池島信平}}、{{Link-ja|鷲尾洋三|鷲尾洋三}}與社長{{Link-ja|佐佐木茂索|佐佐木茂索}}另行成立“文藝春秋新社”來接管《文藝春秋》的編輯、發行工作。

《文藝春秋》現在的發售日為每個月的10號（雜誌上標註的發行日為每月1號），雜誌採用的版型為[[ISO_216|A5紙]]，目錄為折疊式，而每期雜誌的頁數多為450頁。《文藝春秋》的報道覆盖包括[[政治|政治]]、[[經濟|經濟]]、[[經營|經營]]、[[社會|社會]]、[[歷史|歷史]]、[[娛樂|娛樂]]、[[軍事|軍事]]、[[日本皇室|皇室]]、[[教育|教育]]、[[醫學|醫學]]、[[體育|體育]]等多种领域。而日本知名的[[政治家|政治家]]、[[研究員|研究員]]、[[企業家|企業家]]、[[新聞記者|新聞記者]]、[[紀實文學|紀實文學]][[作家|作家]]、[[評論家|評論家]]會為《文藝春秋》提供各色新聞報道及評論，每期刊登的文章多達三十餘篇。而除了日本本土讀者之外，在日本以外也有海外日本人及知日派海外人士定期購買并閱讀《文藝春秋》。

創刊初期，《文藝春秋》上會刊登許多在女性讀者中擁有較高人氣的作家，如{{Link-ja|倉田百三|倉田百三}}和{{Link-ja|島田清次郎|島田清次郎}}等人的新聞報道。而到了1970年代初期，由於從菊池時代遺留的報道習慣，有關作者的報道也略有出現。但是到了1980年代，由於發行商的壓力，《文藝春秋》的商業色彩開始變得濃厚，此類文章變得非常難以見到。但時至今日，或許是為了顧及讀者高齡化及[[嬰兒潮|嬰兒潮]]讀者的需要，有關[[昭和|昭和]]時期作者的報道和回憶又開始流行起來。

《文藝春秋》負責主辦或運營多個文學獎項，例如：每年2月號的“{{Link-ja|文藝春秋讀者獎|文藝春秋読者賞}}”；3月號和9月號的“[[芥川龍之介獎|芥川龍之介獎]]”；6月號的“{{Link-ja|大宅壯－紀實文學獎|大宅壮一ノンフィクション賞}}”；7月號的“{{Link-ja|松本清張獎|松本清張賞}}”和12月號的“[[菊池寬獎|菊池寬獎]]”。特別是每年刊登“[[芥川龍之介獎|芥川龍之介獎]]”獲獎文章的時候，《文藝春秋》的銷量都會大漲。例如刊登剛滿20歲的獲獎作者[[金原瞳|金原瞳]]的作品《[[蛇信與舌環|蛇信與舌環]]》；芥川獎最年輕獲獎者[[綿矢莉莎|綿矢莉莎]]的作品《[[欠踹的背影|欠踹的背影]]》；以及搞笑藝人組合[[Piece_(搞笑組合)|PEACE]]成員[[又吉直樹|又吉直樹]]的作品《[[火花_(小説)|火花]]》時，雜誌的銷量都創下了記錄。雖然也有人批評這樣會導致獎項評委考慮“獲獎者的話題性”而引發獎項本身的公平性。

《文藝春秋》在書店及零售店里的新刊預告宣傳單多為紅色與黑色的雙色印刷，而使用的字體則為手寫毛筆字。

《文藝春秋》是[[文藝春秋|文藝春秋出版社]]的招牌出版物，故而在出版社內有“本誌”的稱呼。

=== 立場與傾向 ===
《文藝春秋》是一本持[[保守主義|保守主義]]立場的雜誌。<ref>{{cite journal |author1=佐藤都 |language=ja|script-title=ja:日本の総合雑誌3誌の数量・内容分析からみる日本人の中国に対する関心の変遷 |journal=北海道大学大学院国際広報メディア・観光学院院生論集 |date=2012-03 |issue=8 |pages=100-108 |hdl=2115/49173 |url=https://eprints.lib.hokudai.ac.jp/dspace/bitstream/2115/49173/1/Sau8_010.pdf |accessdate=2018-12-31 |author= |archive-url=https://web.archive.org/web/20161221052559/http://eprints.lib.hokudai.ac.jp/dspace/bitstream/2115/49173/1/Sau8_010.pdf |archive-date=2016-12-21 |dead-url=no }}</ref> 《文藝春秋》對[[日本共產黨|日本共產黨]]、[[社會民主黨_(日本)|社會民主黨]]等[[左派|左派]][[政黨|政黨]]持批判態度；因此即使是非時事類專欄也很少刊登這些政黨所屬政治家的文章。{{efn|1=例外的個案包括2005年《文藝春秋》9月號的“終戰50週年特輯”里刊發了[[日本社會黨|日本社會黨]]及[[社會民主黨_(日本)|社會民主黨]]原黨領袖[[土井多賀子|土井多賀子]]的文章《戰爭體驗談》以及[[蘇聯解體|蘇聯解體]]時，也刊登過日本共產黨黨員[[不破哲三|不破哲三]]的見解。}} 而它同時對[[公明黨|公明黨]]、[[創價學會|創價學會]]則更具批判性，因此連相關人士（主要是這些政黨的幹部）都不願提及。

但與此同時，《文藝春秋》並不拒絕[[瀨戶內寂聽|瀨戶內寂聽]]、{{Link-ja|澤地久枝|澤地久枝}}、[[坂本龍一|坂本龍一]]、[[森村誠一|森村誠一]]等左派作家及[[進步主義|進步主義]]文化人（[[九條會|九條會]]支持者）的文章。

== 雜誌地位 ==
《文藝春秋》在日本有著“國民雜誌”的美譽，{{Sfn|竹内|2011|p=81}}<ref>{{cite web |author1=張宜 |title=回首 2017 年最 chill 選書——浮光、朋丁、亞典、誠品 R79 推薦 |url=https://www.shoppingdesign.com.tw/post/view/2559 |publisher=Shopping Design |accessdate=2018-12-31}}</ref><ref>{{cite web |author1=張文聰 |title=日本書房》明治維新150週年 回顧・反省・未來 |url=https://readingpass.openbook.org.tw/article/p-18515 |publisher=Openbook閱讀通 |accessdate=2018-12-31}}</ref><ref name="udn">{{cite web |author1=我愛羅119 |title=【日本】《文藝春秋》 |url=http://blog.udn.com/jason080/3085411 |website=udn部落格 |publisher=扎誌 |accessdate=2018-12-31 |archive-url=https://web.archive.org/web/20130421082406/http://blog.udn.com/jason080/3085411 |archive-date=2013-04-21 |dead-url=no }}</ref> 同時對於[[台灣|台灣]][[台灣日治時期|日據時代]]成長起來的知識分子也有著深遠影響。<ref name="udn" /> 根據[[每日新聞社|每日新聞社]]所實施的《全國閱讀情況調查》，1950年或1951年之後，《文藝春秋》無論是在“購買量”和“閱讀量”都處於“出類拔萃”的地位。{{Sfn|竹内|2011|p=81}} 《文藝春秋》在當時的平均發行量約在50萬份到80萬份左右，而同類雜誌《[[中央公論|中央公論]]》在1954年之後的最高發行量還不到14萬份。{{Sfn|竹内|2011|p=81}}

== 經典報道 ==
* 1975年11月的《文藝春秋》特別號上，時任雜誌主編的{{Link-ja|田中健五|田中健五}}製作了名為“重新審視田中政權（{{lang|ja|田中政権を問い直す}}）”的特輯。特輯中包含由{{Link-ja|立花隆|立花隆}}撰寫的《[[田中角榮|田中角榮]]研究——他的金脈與人脈》和由{{Link-ja|兒玉隆也|児玉隆也}}撰寫的《孤獨的越山會女王》兩篇有關“{{Link-ja|田中金脈問題|田中金脈問題}}”的報道。這成為迫使[[第2次田中角榮內閣_(第2次改造)|田中內閣]]總辭職的契機。

* 1986年《文藝春秋》10月號上，時任日本[[文部大臣_(日本)|文部大臣]]的{{Link-ja|藤尾正行|藤尾正行}}發表了有關[[日韓關係|日韓關係]]等歷史問題的言論。隨後他的言論成為話題，執政黨[[自由民主黨_(日本)|自由民主黨]]的一部分成員要求其辭職。但該要求遭到其拒絕，最後根據其意願被罷免。本期雜誌被追加增印，同時在次月的11月號上發表了續篇。藤尾正行在第二年獲得了“文藝春秋讀者獎”。

* 1990年《文藝春秋》12月號上刊載了《昭和天皇的八小時獨白 太平洋戰爭的全貌》。當期雜誌發行量突破105萬份。次年，[[文藝春秋|文藝春秋出版社]]發行了《昭和天皇獨白錄 附寺崎英成的御用日記》。后又在文春文庫以《昭和天皇獨白錄》的名字發行。

* 2004年《文藝春秋》3月號刊登了第130屆[[芥川龍之介獎|芥川龍之介獎]]的獲獎名單及文章，[[金原瞳|金原瞳]]與[[綿矢莉莎|綿矢莉莎]]共獲此獎。由於綿矢莉莎創下芥川龍之介獎最年輕得主的新聞引起話題，使得該期雜誌初次印刷量就突破80萬份，最後創下雜誌史上最高的118萬5000份的記錄。

* 2015年《文藝春秋》9月號刊登了第153屆芥川龍之介獎的獲獎名單及文章，搞笑組合[[Piece_(搞笑組合)|PEACE]]的成員[[又吉直樹|又吉直樹]]的作品《[[火花_(小說)|火花]]》獲獎。該期雜誌初次印量為92萬3000份，而最後印量則是110萬3000份。<ref>{{cite web |language=ja|script-title=ja:“又吉さん「火花」掲載の「文芸春秋」が再増刷。１１０万部を突破” |url=http://www.sankei.com/life/news/150824/lif1508240026-n1.html |publisher=[[產經新聞|產經新聞]] |accessdate=2018-12-31 |archive-url=https://web.archive.org/web/20160509185031/http://www.sankei.com/life/news/150824/lif1508240026-n1.html |archive-date=2016-05-09 |dead-url=no }}</ref>

== 相關醜聞 ==
* 1999年的《文藝春秋》12月號上刊載了一篇名為《第一勸銀不良債券巨額虧損》的報道，揭露當時{{Link-ja|第一勸業銀行|第一勧業銀行}}因為投資大量不良[[債券|債券]]而出現巨額虧損的狀況。然而經過《周刊新潮》、《周刊POST》等其他雜誌的檢查，這篇報道中的部分內容並未經過嚴格的來源審查，而報道中的錯誤多達六十餘處。另據報道，負責當日新聞撰寫的《[[朝日新聞|朝日新聞]]》記者在利用“朝日新聞”名義採訪相關素材之後即為《文藝春秋》供稿，而他在供稿中修改了部分數據。

* 2005年的《文藝春秋》12月號上刊登了由{{Link-ja|笹幸惠|笹幸恵}}撰寫的《“[[巴丹死亡行軍|巴丹死亡行軍]]” 女人獨自完成》的文章。該文章因為內容“嚴重歪曲歷史”而遭受[[猶太人|猶太人]]人權組織[[西蒙·維森塔爾中心|西蒙·維森塔爾中心]]等機構的抗議。<ref>{{cite web |author1=古沢由紀子 |language=ja|script-title=ja:「バターン死の行進」記事、ユダヤ人団体が文春に抗議 |url=http://www.yomiuri.co.jp/world/news/20060114i504.htm |publisher=[[讀賣新聞|讀賣新聞]] |accessdate=2018-12-31}}</ref>
