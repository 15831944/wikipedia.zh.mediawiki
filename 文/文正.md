'''文正'''是[[日本|日本]][[年號|年號]]之一。在[[寬正|寬正]]之後、[[應仁|應仁]]之前，指的是1466年的期間，此時期的天皇是-{[[後土御門天皇|後土御門天皇]]}-。[[室町幕府|室町幕府]]的將軍是[[足利義政|足利義政]]。

== 改元 ==
*寛正七年二月二十八日（西元1466年3月14日）因-{後}-土御門天皇開始執政而改元。
*文正二年三月五日（西元1467年4月9日）改元為應仁。

== 出典 ==
出自《[[荀子|荀子]]·王制篇》的「積文學，正身行」之句。

== 文正年間大事 ==

=== 出生 ===
*文正元年七月三十日（1466年9月9日）：[[足利義材|足利義材]]、[[室町幕府|室町幕府]]第十代-{[[征夷大將軍|征夷大將軍]]}-（[[大永|大永]]三年過世）。
*文正元年（1466年）：[[細川政元|細川政元]]、室町幕府[[管領|管領]]（[[永正|永正]]四年死去）。
