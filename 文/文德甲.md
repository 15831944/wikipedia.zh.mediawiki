'''文德甲'''（{{lang-ms|Mentakab}}），是[[马来西亚|马来西亚]][[彭亨州|彭亨州]][[淡马鲁县|淡马鲁县]]城镇和一个[[巫金|巫金]]，隶属于[[淡马鲁市议会|淡马鲁市议会]]。

==名称来源==
根据记载，文德甲于一百多年开埠，其原名为巴西拉哇（Pasir Lawa），只是一个小村落，约为现今阿布巴卡国民小学周边一带，後來英殖民政府将巴西拉哇改名為Mentakab，而外來人稱它“文章道德甲天下”，“文德甲”之名便因此誕生。<ref>{{cite web|url=http://www.guangming.com.my/node/379883|title=一杯濃咖啡•一則愛情故事 傳頌浪漫文德甲|date=2007-12-09|publisher=}}</ref>

== 气候 ==
这个城镇在[[柯本气候分类法|柯本气候分类法]]下被归类为[[热带雨林气候|热带雨林气候]]（“Af”级）。<ref name="Climate-Data.org">{{Cite web |url=https://en.climate-data.org/location/184392/|title=Climate: Mentakab - Climate graph, Temperature graph, Climate table |publisher=Climate-Data.org |accessdate=2017-08-28 }}</ref>文德甲平均气温约为{{convert|26|°C}}，降水量则平均约有{{convert|2014|mm|in}}。

==交通==
文德甲可通过从关丹至巴生的{{JKR|2}}联邦公路2号抵达。近年来新建的{{MES-E|8}}[[东海岸大道|东海岸大道]]也行经于此。

目前文德甲设有铁路车站[[文德甲站|文德甲站]]，文德甲站为[[东海岸铁路线|东海岸铁路线]]的其中一站，往北可前往[[道北站|道北站]]，往南则可抵达[[金马士站|金马士站]]并转乘至[[西海岸铁路线|半岛西海岸]]的其他主要城市。而在建中的[[东海岸衔接铁道|东海岸衔接铁道]]（ECRL或东铁）也将会在文德甲设站，使文德甲将成为两条铁路的交汇处。<ref>{{cite web|url=http://www.spad.gov.my/ms/pengangkutan-awam-darat/rel/projek-landasan-keretapi-pantai-timur-ecrl|title=Projek Landasan Keretapi Pantai Timur  (ECRL) - Laman Web Rasmi Suruhanjaya Pengangkutan Awam Darat (S.P.A.D.)|work=www.spad.gov.my}}</ref>

==政治==
文德甲属于[[淡马鲁国会议席|淡马鲁国会议席]]，该议席在[[马来西亚下议院|马来西亚下议院]]的代表为来自[[希盟|希盟]]的[[Tuan_Haji_Mohd_Anuar_bin_Mohd_Tahir|Tuan Haji Mohd Anuar bin Mohd Tahir]]。<ref>{{cite web|title=Ahli Parlimen|url=http://www.parlimen.gov.my/profile-ahli.html?uweb=dr&id=3247|website=Portal Rasmi Parlimen Malaysia|accessdate=2017-08-28}}</ref>
同时，文德甲属于以其名称命名的文德甲州议席，该议席在[[彭亨州议会|彭亨州议会]]的代表为来自[[民主行动党|民主行动党]]的[[胡智云|胡智云]]。<ref>{{cite web|title=ADUN N22-33|url=http://www.pahang.gov.my/ahli-dewan-undangan-negeri?p_p_id=56_INSTANCE_Kn4g&p_p_lifecycle=0&p_p_state=normal&p_p_mode=view&page=3|website=Portal Kerajaan Negeri Pahang Darul Makmur|accessdate=2017-08-28|archive-url=https://web.archive.org/web/20170828104017/http://www.pahang.gov.my/ahli-dewan-undangan-negeri?p_p_id=56_INSTANCE_Kn4g&p_p_lifecycle=0&p_p_state=normal&p_p_mode=view&page=3|archive-date=2017-08-28|dead-url=yes}}</ref>
