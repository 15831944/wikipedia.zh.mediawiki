'''文京区'''（{{jpn|j=文京区|hg=ぶんきょうく|rm=Bunkyō ku|f=y}}）是[[日本|日本]][[東京都|東京都]]的23个特别區之一，劃分上屬於[[東京都區部|23區]]西部，實際位於中央偏北的位置，現任區長是成澤廣修<ref>{{cite web|url=http://www.city.bunkyo.lg.jp/sosiki_busyo_senkyo_19toitsu_kugikucho_sokuho.html|title=文京區議會議員選舉・文京區長選舉の結果|publisher=文京區|accessdate=2010-03-01|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20111126071255/http://www.city.bunkyo.lg.jp/sosiki_busyo_senkyo_19toitsu_kugikucho_sokuho.html|archivedate=2011-11-26}}</ref>。文京區的面積11.31平方千米，在23區中位列倒数第4，佔東京都區部的50分之一<ref>{{cite web |url=http://www.city.bunkyo.lg.jp/profile_three_minutes_location.html |title=文京区の位置と地形 |publisher=文京区 |accessdate=2010-03-01 |language=ja}}</ref>。該區以文化教育機關和住宅街區為主，有“文之京”的美名。從[[明治|明治]]時期以来，這裡居住過[[夏目漱石|夏目漱石]]、[[森鷗外|森鷗外]]、[[宮澤賢治|宮澤賢治]]等著名的文人、學者和政治家。除此以外出版、印刷業也很繁盛，並设有先進醫療設施和大型醫院。區內著名的設施包括日本最高學府[[東京大學|東京大學]]本部（本鄉校區）、[[東京巨蛋|東京巨蛋]]和[[講道館|講道館]]等，[[六义园|六义园]]、[[小石川后乐园|后乐园]]和[[小石川植物園|小石川植物園]]等古老而寬廣的日本庭園也坐落於文京區内。

==地理==
文京區位於[[東京都|東京都]]東部、[[皇居|皇居]]北側。區內有[[武藏野台地|武藏野台地]]斜坡、[[神田川_(東京都)|神田川]]等沿岸低地、5台地（關口台、小日向台、小石川台、白山台、本鄉台）與谷地等地形。海拔最低處是[[後樂|後樂]]一丁目，約3.1公尺。最高處在[[大塚_(文京區)|大塚]]五丁目附近，約34公尺。面積11.31平方公里，是第四小的特別區。

== 历史 ==
考古学家认为目前文京区的位置开始有人类居住的历史可以溯及18,000年前的[[旧石器时代|旧石器时代]]，区内有多处[[繩文時代|繩文時代]]的遗迹出土确认，尤其在本乡高地周边发现了大量[[贝塚|贝塚]]和繩文式土器<ref>{{cite web |url=http://www.syougai.metro.tokyo.jp/iseki0/iseki/list/ruins/13105/105ruins.htm |title=文京区-遺跡一覧 |publisher=東京都教育庁 |accessdate=2010-03-01 |language=ja}}</ref>。1884年（明治17年），在本乡向丘弥生町（现在的弥生一丁目）的贝塚中发现的土器被依发现地而称为“弥生式土器”，这即是后来演变而成的[[弥生时代|弥生时代]]称谓的由来<ref>{{cite web |url=http://www.maibun.or.jp/qa/a31.html |title=弥生時代の弥生の由来は何ですか？ |publisher=栃木県埋蔵文化財センター |accessdate=2010-03-01 |language=ja}}</ref>。1896年（明治29年）在六义园至[[本乡通|本乡通]]的东侧（现在的本驹込五丁目）又出土了弥生后期的遗迹。至今为止，在文京区和[[丰岛区|丰岛区]]的接壤地区已经确认了大片的历史遗迹出土。

进入[[战国时代_(日本)|战国时代]]后曾是[[上杉氏|上杉氏]]统治的地区，并受到太田氏、丰岛氏等豪族的影响，而后又成为[[後北條氏|後北條氏]]的统治范围。1590年（[[天正|天正]]18年）[[德川家康|德川家康]]的[[江户幕府|江户幕府]]打败後北條氏之后，在文京区的地界设立了大量的[[武家|武家]]住宅及寺庙、神社，其中包括著名的[[德川御三家|德川御三家]]之一，[[水户黄门|水户黄门]]的[[德川光圀|德川光圀]]在此修造的水户德川家屋敷<ref>{{cite web |url=http://teien.tokyo-park.or.jp/contents/outline030.html |title=小石川後楽園·概要 |publisher=東京都公園協会 |accessdate=2010-03-01 |language=ja}}{{zh-hans}}[http://teien.tokyo-park.or.jp/cn/koishikawa/index.html]{{zh-hant}}[http://teien.tokyo-park.or.jp/hk/koishikawa/index.html]</ref>，以及埋葬了[[德川家康|德川家康]]生母的[[傳通院|傳通院]]<ref>{{cite web |url=http://www.denzuin.or.jp/ |title=傳通院 |publisher=無量山·傳通院·寿経寺 |accessdate=2010-03-01 |language=ja}}</ref> 和由[[德川綱吉|德川綱吉]]之母[[桂昌院|桂昌院]]創建的[[護國寺_(東京)|護國寺]]<ref>{{cite web|url=http://www.gokokuji.or.jp/NewFiles/engi.html|title=護国寺略縁起|publisher=真言宗豊山派·大本山·護国寺|accessdate=2010-03-01|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20010405051323/http://www.gokokuji.or.jp/NewFiles/engi.html|archivedate=2001-04-05}}</ref>。幕府第三代将军[[徳川家光|徳川家光]]的乳母[[春日局|春日局]]和[[大奧|大奧]]中的女官[[音羽|音羽]]等都在文京区地界内受领过土地，“春日”、“音羽”等地名也一直沿用至今。[[大政奉还|大政奉还]]后，第十五代将军[[德川庆喜|德川庆喜]]在区界内的旧小日向地区居住。日本的学校教育发祥地[[汤岛圣堂|汤岛圣堂]]和幕府直辖的教学机构[[昌平坂学问所|昌平坂学问所]]也设在文京区的地界内。

进入[[江户时代|江户时代]]的文京区，起初由于区内大量的武家领地，使得[[町人|町人]]街市的发展受到了很大的妨碍。在中期后，很多地区都进行了市街化的改造，后期在[[中山道|中山道]]的街道两侧商店林立，商业得到了很大发展。在进入明治时期以后，广大的武家宅地遗址大部分转为教育机构和军事设施使用。1877年（明治10年）4月12日，[[东京大学|东京大学]]在加贺[[前田氏|前田氏]]的领地遗址上设立<ref>{{cite web |url=http://www.u-tokyo.ac.jp/gen03/b03_01_j.html |title=東京大学の歴史·沿革 |publisher=東京大学 |accessdate=2010-03-01 |language=ja}}</ref>，四周集结了大量的[[出版社|出版社]]等机构，[[坪内逍遥|坪内逍遥]]、[[森鷗外|森鷗外]]、[[夏目漱石|夏目漱石]]、[[樋口一葉|樋口一葉]]等著名文人也在附近居住。连同先前设立的东京师范学校（现[[筑波大学|筑波大学]]大塚校区）和东京女子师范学校（现[[御茶水女子大學|御茶水女子大學]]），文京区体现出鲜明的文教地区特色。1878年（明治11年），根据[[郡区町村編制法|郡区町村編制法]]成立了属于[[東京15區|東京15區]]的小石川区和本乡区两个区划单位。

1937年（昭和12年），在东京炮兵工厂的遗址上设立了后乐园球场，是今天[[东京巨蛋|东京巨蛋]]的前身，二战后成为文京区职业棒球复兴和市民娱乐的重要地点。1947年（昭和22年），根据市街地编成计划，小石川区和本乡区合并成立了文京区，区名“文京”作为“文教之府”的形象被原有两区的区议会一致同意采用。

== 经济 ==
印刷及其关联行业在文京区的工业中占有决定性的地位，其销售营业额超过文京区内所有行业总计的80%。除了该行业外，由于区内有较多医疗设施，又邻近[[秋叶原|秋叶原]]，电子和精密仪器在文京区的工业中也占有比较大的比重<ref>{{cite web |url=http://www.toukei.metro.tokyo.jp/kougyou/2008/kg08q20000.htm |title=東京都工業統計調査結果（平成20年調査） |publisher=東京都総務局 |accessdate=2010-03-01 |language=ja}}</ref>。
