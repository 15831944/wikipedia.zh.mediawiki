'''文鼎科技開發股份有限公司'''（Arphic Technology Co., Ltd.），簡稱'''文鼎科技'''，為[[台灣|台灣]][[字型|字型]]廠商。

文鼎科技接受[[日本|日本]][[C&G|C&G]]公司（{{lang|ja|株式会社C&Gシステムズ}}）委託，開發設計[[日文|日文]]字型[[明瞭體|明瞭體]]中的[[漢字|漢字]]。字体被[[微軟|微軟]][[Windows_Vista|Windows Vista]]使用。著名的[[網路遊戲|網路遊戲]]《[[魔獸世界|魔獸世界]]》的繁體中文字型也是採用文鼎科技所設計開發之字型。

近年文鼎科技針對[[嵌入式系统|內嵌式系統]]用的[[多國語言|多國語言]]字型技術進行發展，專門為記憶體容量有限的小型设备設計耗用[[記憶體|記憶體]]更少的Arphic Tiny Layout Engine。其搭配文鼎的Mobile Font、Bitmap Font等字型技術，解决了[[MP3|MP3]]、Multimedia Player等小型多媒體/音樂撥放器对于[[印地文|印地文]]、[[阿拉伯文|阿拉伯文]]、[[泰文|泰文]]等複雜語言字型的排版显示问题。

另外也開發完成Arphic Tiny Layout Engine，就是簡化版的Layout Engine，特別為某些無法使用Layout Engine的小型Device設計的，因為這些設備的[[CPU|CPU]]效能與記憶體有限，或受限於它們使用平台上的GUI framework，無法正常的顯示阿拉伯文、泰文等複雜語言字型。如MP3、Multimedia Player等小型設備等嵌入式系統，採用Tiny Layout Engine後就可以支援阿拉伯文、泰文等複雜語言字型，使這些廠商的產品順利進入[[阿拉伯|阿拉伯]]、泰國等市場，目前已獲得[[飛利浦|飛利浦]]、日本大廠等採用。

==捐獻==
文鼎最為人所知的是於1999年捐贈了四套字型予[[開源軟件|開源軟件]]社群：'''文鼎PL细上海宋'''（[[繁體中文|繁體]][[Big5碼|Big5碼]]）、'''文鼎PL中楷'''（繁體Big5碼）、'''文鼎PL简报宋'''（[[簡體中文|簡體]][[GB碼|GB碼]]）、'''文鼎PL简中楷'''（簡體GB碼，即是繁簡[[宋體|宋]][[楷體|楷]]齊備）。後經[[螢火飛|螢火飛]]加入[[點陣字|點陣字]]而成[[文鼎PL新宋|文鼎PL新宋]]及經[[高盛華|高盛華]]改進成[[uming|uming]]及[[ukai|ukai]]，已包括在各大[[Linux發行版|Linux發行版]]。这些字体所用的《文鼎公众授权书》獲[[自由软件基金会|自由软件基金会]]承认为[[自由软件|自由软件]]许可。<ref>{{cite web|url=http://www.gnu.org/licenses/license-list.html#Arphic|title=Various Licenses and Comments about Them|publisher=Free Software Foundation|accessdate=7 December 2010|archiveurl=https://web.archive.org/web/20101209220802/http://www.gnu.org/licenses/license-list.html#Arphic|archivedate=2010年12月9日|deadurl=no}}</ref>

2010年又再提供兩套新的公眾授權字型：'''文鼎PL明體U20-L'''及'''文鼎PL報宋2GBK'''（新的中文公眾授權字型，採用[[Unicode|Unicode]]及[[GBK|GBK]]編碼，漢字字數分別為20902字及21003字，較舊版本的[[GPL|公眾授權]]字型多出7000多字及14000多字）<ref>{{cite web|url=http://www.arphic.com/tw/news/2010/20100420.html|title=文鼎科技發佈新的公眾授權字型|year=2010|publisher=文鼎科技|language=zh|accessdate=7 December 2010|archiveurl=https://web.archive.org/web/20101111045358/http://www.arphic.com/tw/news/2010/20100420.html|archivedate=2010年11月11日|deadurl=yes}}</ref>。不过这两套字体所用的是新修改的《文鼎公众授权书》，禁止商業使用，因而不属于[[自由软件|自由软件]]许可。<ref>{{cite web|title=文鼎科技提供2套新的中文公眾授權字型(Public License Font)|url=http://www.openfoundry.org/tw/forum?func=view&catid=8&id=484|accessdate=2015-08-18}}</ref>
