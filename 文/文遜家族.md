'''文遜家族'''（{{lang-en|'''Manson Family'''}}）是於1960年代末在[[加利福尼亞州|加利福尼亞州]]建立的一個[[公社|公社]]和[[邪教|邪教]]<ref>{{Cite web |url = https://www.nbcnews.com/news/us-news/what-ever-happened-other-manson-family-cult-members-n556561 |title = What ever happened to the other Manson family cult members? |author = Corky Siemaszko |date = 2016-04-16 |publisher = nbcnews.com |language = en}}</ref><ref>{{Cite web |url = https://www.biography.com/people/groups/manson-family |title = Manson Family |publisher = Biography |language = en}}</ref>，由[[查理斯·文遜|查理斯·文遜]]所領導的，該團體由約100名「追隨者」組成，他們過著非常規的生活並習慣性使用[[致幻劑|致幻毒品]]。大多數的團體成員都是來自中產階級背景的年輕女性，其中許多人被文遜的教導而變得激進，並被[[嬉皮士|嬉皮士文化]]和集體生活所吸引。<ref>{{Cite web |url = http://www.bbc.com/news/world-us-canada-42056252 |title = Charles Manson death: Where are the Family members now? |date = 2017-11-20 |publisher = BBC |language = en}}</ref>

文遜在1967年因輕罪而出獄後，文遜家族便搬到了三藩市，後來到了聖費爾南多谷（San Fernando Valley）一個廢棄的牧場。文遜的追隨者也包括一個小而忠誠的小組，其中大部分是令人印象深刻的年輕女人和女孩。他們毫不懷疑地開始相信文遜所聲稱他是耶穌的具體化和他對種族戰爭的預言。

在查理斯·文遜的指令下，{{link-en|泰克斯·華生|Tex Watson}}和另外三名家族成員在1969年8月9日謀殺了女演員[[莎朗·蒂|莎朗·蒂]]和另外四人之後，他們得到了國內和國際性的惡名昭彰。組織的成員還負責其他一些謀殺、毆打、輕微犯罪和偷竊事件。<ref>{{Cite web |url = https://www.crimemuseum.org/crime-library/serial-killers/charles-manson-and-the-manson-family/ |title = Charles Manson and the Manson Family |publisher = Crime Museum |language = en}}</ref>
