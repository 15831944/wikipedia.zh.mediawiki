'''文革遗毒'''，是对[[文化大革命|文化大革命]]持反对与批评态度的人士对该运动结束后的持续影响，尤其是对人们的观点、语言习惯和政治思维方面影响的称呼。该用语并无官方定义，具体包括哪些内容随具体场合和使用者主观意愿而异。在[[中国大陆|中国大陆]]当前的语境中，'''文革遗毒'''一词较多地被用于形容人们的以下观点或行为：
*将[[毛泽东|毛泽东]][[神化|神化]]，拒绝容忍对其非正面的评价；
*[[大字报|大字报]]式的辩论语言，用喊口号、[[扣帽子|扣帽子]]、[[恐吓论据|恐吓威胁]]、[[上纲上线|上纲上线]]、[[假两难推理|非黑即白]]等[[谬误|谬误]]代替逻辑理性的讨论；
*政治挂帅，将一切问题归结为政治路线问题；
*全面肯定文革十年中的中国，全面否定文革后的中国。

1981年6月27日，[[中国共产党第十一届中央委员会第六次全体会议|中国共产党第十一届中央委员会第六次全体会议]]一致通过的《[[关于建国以来党的若干历史问题的决议|关于建国以来党的若干历史问题的决议]]》，对文化大革命进行了官方正式否定，但是文化大革命的遗留下來的思想和物质并没有因此泯灭。2012年3月14日，[[中华人民共和国国务院总理|中华人民共和国国务院总理]][[温家宝|温家宝]]在回答[[记者|记者]]提问时说：“[[粉碎四人帮|粉碎四人帮]]以后，我们党作出了关于新中国成立以来若干历史问题的决议，实行了[[改革开放|改革开放]]。但是‘文革’错误的遗毒和封建的影响并没有完全清除。”<ref name=" 21cn">{{cite web |url=http://news.21cn.com/caiji/roll1/2012/03/15/11154416.shtml |title=文革遗毒没有完全清除 |publisher=[[信息时报|信息时报]] |accessdate=2012-03-15 |work= |language=zh-cn |archive-url=https://web.archive.org/web/20150326035054/http://news.21cn.com/caiji/roll1/2012/03/15/11154416.shtml |archive-date=2015-03-26 |dead-url=yes }}</ref><ref name="bfw " >{{cite web |url= http://china.nfdaily.cn/content/2012-03/15/content_40528001_4.htm|title=温家宝：“文革”错误遗毒并未完全清除 |publisher=[[南方日报|南方日报]] |accessdate= 2012-03-15|work= |language =zh-cn}}</ref><ref name="bookzh " >{{cite book |author=王桧林 |title=《中国现代史》|year=2004 |publisher=[[北京师范大学出版社|北京师范大学出版社]] |location= |pages= |isbn=9787303010844 |language = zh-cn}}</ref><ref name="while " >{{cite book |author=席宣、金春明 |title=《“文化大革命”简史》|year= 2005|publisher=[[中共党史出版社|中共党史出版社]] |location= |pages= |isbn=9787800239601 |language = zh-cn}}</ref>

== 历史 ==
=== 背景 ===
1966年到1976年，[[中国大陆|中国大陆]]进行了十年[[文化大革命|文化大革命]]，使中国大陆的工作重心从[[经济|经济]]转向了[[政治活动|政治活动]]，社会生活的各个层面，[[经济|经济]]、[[教育|教育]]、[[科研|科研]]、[[民生|民生]]、[[道路|道路]]、[[文物|文物]][[古迹|古迹]]都不同层面的被破坏，一大批[[干部|干部]]被打倒，[[知识分子|知识分子]]被[[上山下乡|上山下乡]][[劳动改造|劳动改造]]，道路桥梁等[[基础设施|基础设施]]被损毁，[[文物|文物]][[古迹|古迹]]也在[[破四旧|破四旧]]中损失殆尽，科研工作除受保护部门也被进行思想整顿。<ref name="bookzh " /><ref name="while " />

文革期間，[[四人帮|四人帮]]利用毛泽东的威望，以[[红色歌曲|唱红]]的形式製造[[个人崇拜|个人崇拜]]，加其利用排除政治异己。同时，国内民粹主义泛滥，严重延误了经济发展并导致了生产力落后的现象。<ref>{{cite book| language =zh-hans| author =钱其琛| title =《回忆邓小平》上| url =http://www2.gxtc.edu.cn/Threepage/llxt/200611/19152.html| date =1998年2月| publisher =中央文献出版社| deadurl =yes| archiveurl =https://web.archive.org/web/20071023201943/http://www2.gxtc.edu.cn/Threepage/llxt/200611/19152.html| archivedate =2007年10月23日| df =}}</ref>

1981年6月27日，中国共产党第十一届中央委员会第六次全体会议一致通过的《[[关于建国以来党的若干历史问题的决议|关于建国以来党的若干历史问题的决议]]》，对文化大革命进行了官方正式否定。<ref name="bookzh " /><ref name="while " />

=== 改革开放时期 ===
[[中共十一届三中全会|中共十一届三中全会]]之后，[[邓小平|邓小平]]领导的中央第二代领导集体实行了“[[改革开放|改革开放]]”的基本国策，将国家工作重心由政治运动转移到经济建设上，同时对文化大革命中的部分人物进行了法律制裁和审判，也为无辜受害者进行了[[平反|平反]]。<ref name="bookzh " /><ref name="while " />

民间此时也开始了对文化大革命的反思和研究。文化大革命中的受害者著名作家[[巴金|巴金]]后半生一直呼吁建立“文革博物馆”，巴金生前曾说：“建立文革博物馆是件非常必要的事。惟有不忘过去，才能做未来的主人；……最好是建立一座文革博物馆，用具体的实在的东西、用惊心动魄的情景说明20年前在中国这块土地上究竟发生了什么事情，让大家看看它的全部过程，想想个人在10年间的所作所为，脱下面具、掏出良心，弄清自己的本来面目，偿还过去的大小欠债。没有私心，才不会受骗上当；敢说真话，就不会轻信谎言。”但是由于文革遗留势力的阻扰和其他各种各样的阻力，直到巴金逝世他的愿望也未能达成。<ref name="atc " >{{cite web |url=http://www.21ccom.net/articles/dlpl/szpl/2012/0320/article_55824.html |title= 西蒙周：再改革，从肃清文革遗毒开始|publisher=[[共识网|共识网]]|date=2012-03-20|accessdate=2012-03-20 |work= |language =zh-cn|archiveurl=https://web.archive.org/web/20160305011815/http://www.21ccom.net/articles/dlpl/szpl/2012/0320/article_55824.html|archivedate=2016-03-05|deadurl=yes}}</ref><ref>{{cite web |url= http://www.gmw.cn/content/2005-10/21/content_320560.htm|title=巴金最后的愿望：建立文革博物馆 |publisher=光明网 |accessdate=2005-10-21 |work= |language =zh-cn}}</ref>

随着改革开放经济建设的成就，中国大陆社会出现了各种各样的深层矛盾，如分配不公、[[贫富悬殊|贫富悬殊]]、诚信缺失、[[贪污腐败|贪污腐败]]等问题；于是社会上的文革遗留势力开始死灰复燃。文革遗毒由于对现实社会的不满，开始追溯想像中的[[毛泽东时代|毛泽东时代]]的清廉和公平；由于民主改革会夺走其手中的权力，用唱[[红色歌曲|红色歌曲]]来追思文革的记忆；其认为，保持中共的本色就是[[意识形态|意识形态]]向左走，经济上则主张政府分配资源、实行[[计划经济|计划经济]]，民生问题则希望实行[[大锅饭|大锅饭]]分蛋糕的形式。<ref name="atc " />

=== 新时代时期 ===
2010年11月份，[[北京大学|北京大学]]教授[[孔庆东|孔庆东]]公开表态支持[[重庆市|重庆市]]公安局长[[王立军|王立军]]的“双起记者论”，同时提出“记者国家公害论”，对[[南方报业传媒集团|南方报业传媒集团]]旗下的[[南方报系|南方报系]]说：“我觉得全国人民应该起诉南方报系，天天在诬蔑我们[[革命|革命]]先烈，诬蔑我们的党和国家领导人，诬蔑全体中国人民。像南方报系还有[[凤凰网|野鸡网]]这些媒体首先就应该遭到起诉。”孔庆东的言论被[[肖鹰|肖鹰]]认为是典型的文革遗毒。<ref>{{cite web |url=http://news.sina.com.cn/c/2010-12-05/141321585005.shtml |title=评论批评孔庆东记者公害论系文革遗毒 |publisher=新浪网 |accessdate=2010-12-05 |work= |language =zh-cn}}</ref><ref>{{cite web|url=http://www.ycwb.com/ePaper/ycwb/html/2010-12/05/content_986842.htm|title=“记者公害论”与“文革”遗害|publisher=羊城晚报|accessdate=2010-12-05|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20101209002703/http://www.ycwb.com/ePaper/ycwb/html/2010-12/05/content_986842.htm|archivedate=2010-12-09}}</ref><ref>{{cite web |url=http://www.zhgpl.com/crn-webapp/mag/docDetail.jsp?coluid=0&docid=101518720 |title= 孔庆东：全国人民都应该起诉南方报系|publisher=中国评论月刊 |accessdate=2010-11-26 |work= |language =zh-cn}}</ref><ref>{{cite web |url= http://finance.ifeng.com/opinion/xuejie/20101125/2948555.shtml|title=孔庆东：南方报系天天在诬蔑全体中国人民 |publisher=凤凰网 |accessdate=2010-11-25 |work= |language =zh-cn}}</ref>

2012年3月14日，[[中华人民共和国总理|中华人民共和国总理]][[温家宝|温家宝]]在答记者问时说道：“粉碎‘[[四人帮|四人帮]]’以后，我们党作出了关于新中国成立以来若干历史问题的决议，实行了[[改革开放|改革开放]]。但是‘文革’错误的遗毒和封建的影响并没有完全清除。”这是官方首次出面说明文革遗毒的存在和活跃。<ref name=" 21cn" /><ref name="bfw " /><ref>{{cite web |url=http://house.people.com.cn/GB/17398338.html |title=十一届全国人大五次会议记者会温家宝答记者问 |publisher= 人民网|accessdate=2012-03-15 |work= |language =zh-cn}}</ref>

2012年6月6日，[[武汉|武汉]][[华中师范大学|华中师范大学]]毕业生身着类似红卫兵的服装，手拿“[[红宝书|红宝书]]”在学校的[[图书馆|图书馆]]前面拍毕业照。<ref>{{cite web|url= http://news.21cn.com/today/topic/2012/06/07/12086120.shtml|title= 邸朔：“文革”是我们用来调侃和恶搞的么？|accessdate= 2012-06-07|publisher= 21CN|work= |language= zh-cn|archive-url= https://web.archive.org/web/20150326065740/http://news.21cn.com/today/topic/2012/06/07/12086120.shtml|archive-date= 2015-03-26|dead-url= yes}}</ref> 此事引发社会极大的关注，震动中国大陆和[[香港|香港]]，[[腾讯网|腾讯网]]列出专题网《文明社会怎容忍“红卫兵”？》深刻批评学校的学生逆流；<ref>{{cite web |url= http://view.news.qq.com/zt2012/hwb/index.htm|title=文明社会怎容忍“红卫兵”？ |accessdate=2012年6月9日 |publisher= 腾讯网|work= |language =zh-cn}}</ref> 香港[[凤凰卫视|凤凰卫视]]主持人[[闾丘露薇|闾丘露薇]]说：“有的东西是不能拿来调侃和消费的。不过，这些年轻人，对于那个时代，只有这样的印象吧？谁的责任？他们的下一代又会怎样？”表示对未来大陆一代的深刻担忧。<ref>{{cite web |url=http://news.21cn.com/hot/focus/2012/06/07/12082378.shtml |title=华中师大学生红卫兵毕业照遭炮轰 |accessdate=2012-06-07 |publisher=21CN |work= |language=zh-cn |archive-url=https://web.archive.org/web/20120611132643/http://news.21cn.com/hot/focus/2012/06/07/12082378.shtml |archive-date=2012-06-11 |dead-url=yes }}</ref>
