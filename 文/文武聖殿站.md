'''文武聖殿站'''位於[[高雄市|高雄市]][[鼓山區|鼓山區]]，為[[高雄捷運環狀輕軌|高雄捷運環狀輕軌]]的捷運車站。本站預計於2019年初通車<ref>[http://news.ltn.com.tw/news/local/paper/1077045 環狀輕軌二階動土 拚後年底完工]</ref>。

== 車站概要 ==
車站代碼為'''C16'''。站區位於大公路與鼓山一路口。預定興建為[[側式月台|側式月台]]。<ref>[http://mtbu.kcg.gov.tw/cht/project_LRT_circle_contents.php 高雄市政府捷運工程局 環狀輕軌計畫內容] {{webarchive|url=https://web.archive.org/web/20120814130739/http://mtbu.kcg.gov.tw/cht/project_LRT_circle_contents.php |date=2012-08-14 }}</ref>

== 車站構造 ==
