'''文艺复兴音乐'''约为公元1400年－1600年，在[[中世纪|中世纪]]“新艺术”的基础上，更加追求人性的解放与对人的内心情感的抒发与表达。这时的[[音乐家|音乐家]]在[[人文主义|人文主义]]思潮的推动下，对[[复调|复调]]音乐进行了发展和变革，[[声乐|声乐]]与[[器乐|器乐]]逐渐分离而独立发展。这一时期[[五线谱|五线谱]]已得到完善，[[印刷术|印刷术]]也运用到曲谱上，这都使音乐的传播更加便利和广泛。这一时期有以下几个较有影响力的乐派：

== 尼德兰乐派 ==
主要音乐活动在[[尼德兰|尼德兰]]的一批音乐家。创作内容多为弥撒曲与经文歌等宗教音乐，也有世俗音乐。代表人物有[[迪费|迪费]]、[[若斯坎|若斯坎]]、[[班舒瓦|班舒瓦]]、[[奥克冈|奥克冈]]等。

== [[威尼斯|威尼斯]]乐派 ==
在1530-1620年间的一个器乐乐派，其特点是音响气势宽广宏大、对比效果鲜明。创作内容有铜管乐与弦乐的重奏曲、[[管风琴|管风琴]]的前奏曲、幻想曲与[[托卡塔|托卡塔]]等。代表人物有[[维拉尔特|维拉尔特]]、A.加布里埃利等。

== [[罗马|罗马]]乐派 ==
此时期的一个专门创作服务于宗教作品的乐派，以[[无伴奏合唱|无伴奏合唱]]的形式为主。代表人物有[[帕莱斯特里纳|帕莱斯特里纳]]、[[G.M.纳尼诺|G.M.纳尼诺]]、[[F.索里亚诺|F.索里亚诺]]等。
