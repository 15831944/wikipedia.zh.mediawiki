'''文冬'''（{{lang-ms|Bentong}}），是[[马来西亚|马来西亚]][[彭亨州|彭亨州]]第三大城，隶属于[[文冬市议会|文冬市议会]]。其面积为867.69平方公里，人口于2010年为114,397。该市北临[[劳勿|劳勿]]，东临[[淡马鲁|淡马鲁]]和[[百乐|百乐]]，南临[[雪兰莪州|雪兰莪州]][[安邦|安邦]]、[[加影|加影]]和[[瓜拉格拉旺|瓜拉格拉旺]]，西临[[雪兰莪州|雪兰莪州]][[士拉央|士拉央]]和[[新古毛|新古毛]]。文冬因其历史悠久而以旅游业为主，市内拥有许多华人会馆和一间文化馆，还有当地特产榴莲、豆腐卜等。<ref>{{cite web|url=http://www.enanyang.my/news/20150927/%E5%88%B0%E6%96%87%E5%86%AC%E6%80%80%E6%97%A7%E4%B8%80%E7%95%AA/|title=到文冬怀旧一番|publisher=[[南洋商报|南洋商报]]}}</ref>文冬以及其周边生态环境优美，因此也被马来西亚旅游部列为「清肺游景点」，放眼成为吉隆坡的后花园。<ref>http://www.sinchew.com.my/node/1676938</ref>

==历史==
根据古人记载，文冬原名为加邦（Kapong），后来于二战后才改为現名。现今的文冬市中心早期原是两条河的交汇之处。在当时最重要的交通工具是一种称为Perahu的小船，并在市内设有码头。

文冬后来被发现蕴含丰富的[[锡|锡]]矿，吸引了许多人前来挖矿。而那时候人们在文冬附近建了很多用来挖矿的“Ban”，而挖出很多矿的Ban就被叫做「Ban Untung」（马来文原意为赚很多钱的Ban的意思），Ban Untung慢慢就变成了现今的Bentong（文冬的马来文名）。<ref>{{cite web|url=http://tastylilworld.blogspot.my/2016/08/blog-post_6.html?m=1|title=文冬吃吃吃之旅：手作云团面 ＋ 早晨市集|work=tastylilworld.blogspot.my}}</ref>

另一个说法文冬是源自于马来名字“Bapong”（意译为漂浮），因这里曾是拿督巴哈曼与英国军队战斗后，英军在战败后雇佣的[[廓尔喀人|廓尔喀人]]被当地人抓住全部杀害并把他们的尸体丢下河内任其漂浮在水中。也因如此，真正从Bapong更名至文冬的日期是在马来亚联合邦独立当天（1957年8月31日）。

== 气候 ==
这个城镇在[[柯本气候分类法|柯本气候分类法]]下被归类为[[热带雨林气候|热带雨林气候]]（“Af”级）。<ref name="Climate-Data.org">{{Cite web |url=https://en.climate-data.org/location/25945/|title=Climate: Bentong - Climate graph, Temperature graph, Climate table |publisher=Climate-Data.org |accessdate=2017-08-28 }}</ref>文冬平均气温约为{{convert|26.8|°C}}，降水量则平均约有{{convert|2419|mm|in}}。

==交通==
文冬共有两条干道，{{JKR|68}}鹅唛文冬大道將其同[[馬來西亞|馬來西亞]]首都[[吉隆坡|吉隆坡]]相連。此外，外地人也能以贯穿[[马来西亚半岛|马来西亚半岛]]东西海岸的{{MES-E|8}}[[东海岸大道|加叻大道]]。目前文冬没有铁路车站，但在建中的[[东海岸衔接铁道|东海岸衔接铁道]]（ECRL或东铁）将会在文冬设站。<ref>{{cite web|url=http://www.spad.gov.my/ms/pengangkutan-awam-darat/rel/projek-landasan-keretapi-pantai-timur-ecrl|title=Projek Landasan Keretapi Pantai Timur  (ECRL) - Laman Web Rasmi Suruhanjaya Pengangkutan Awam Darat (S.P.A.D.)|work=www.spad.gov.my}}</ref>市区的主要道路为陆佑街，该街是以马来西亚锡矿业者[[陆佑|陆佑]]命名，而[[吉隆坡|吉隆坡]]和[[新加坡|新加坡]]也有一条陆佑街。
