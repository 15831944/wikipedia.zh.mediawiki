'''文忠民'''（{{bd|1950年|3月||}}）[[陕西省|陕西省]][[长武县|长武县]]彭公乡方庄村人，[[中国人民解放军少将|中国人民解放军少将]]<ref name=sanci>{{Cite web |url = http://mil.news.sina.com.cn/2009-01-05/0701537420.html |title = 三次部队任职磨炼 |author =  |date = 2009-01-05 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref><ref name=wenzm>{{Cite web |url = http://www.shanximingren.com/index.php?m=content&c=index&a=show&catid=182&id=2489 |title = 文忠民 |author =  |date =  |publisher = 陕西名人网 |language =  |accessdate = 2017-05-23 |quote =  }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==生平==
1968年3月入伍<ref name=sanci/><ref name=wenzm/>。1969年7月<ref name=wenzm/>（一说1970年<ref name=sanci/>）加入[[中国共产党|中国共产党]]<ref name=wenzm/>。毕业于[[中国人民解放军国防大学|中国人民解放军国防大学]]基本系。入伍后，历任第二十一军第六十二师后勤部战勤科参谋，师政治部宣传科干事。1974年起，历任连队指导员，团组织股股长，[[中国人民解放军兰州军区|中国人民解放军兰州军区]]政治部组织部干事、副处长、处长，[[中国人民解放军总政治部组织部|中国人民解放军总政治部组织部]]于事、副处长等职。1992年9月，任中国人民解放军总政治部组织部组织局局长。1996年2月，任[[中国人民解放军陆军第二十六集团军|陆军第二十六集团军]]第七十七师第16任政委（师长为[[姚春阳|姚春阳]]）。1998年8月，任[[中国人民解放军南京政治学院|中国人民解放军南京政治学院]]副院长。1998年12月，任[[中国人民解放军总政治部直属工作部|中国人民解放军总政治部直属工作部]]副政治委员。2004年，任中国人民解放军南京政治学院政治委员<ref name=sanci/><ref name=wenzm/>。

1998年3月晋升为大校军衔，2001年7月晋升为少将军衔。曾被中国人民解放军总政治部直属党委评为“优秀共产党员”<ref name=wenzm/>。
