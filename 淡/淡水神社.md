'''淡水神社'''為[[臺灣日治時期|臺灣日治時期]][[台北州|台北州]][[淡水郡|淡水郡]][[淡水街|淡水街]](今[[新北市|新北市]][[淡水區|淡水區]])的[[神社|神社]]。落成於昭和14年（1939年），二戰後廢止，改為[[新北市忠烈祠|忠烈祠]]至今。

== 歷史 ==
淡水神社興建於昭和10年（1936年），昭和14年（1939年）6月1日鎮座，[[近代社格制度|社格]]是無格社，祭祀[[北白川宮能久親王|北白川宮能久親王]]、[[明治天皇|明治天皇]]、[[大物主命|大物主命]]、[[崇德天皇|崇德天皇]]，例祭日為11月10日，1939年時任社掌為小笠原清禧<ref>{{Cite book | author = 臺灣總督府文教局社會課 | title = 臺灣圖書館，日治時期圖書影像系統(需密碼登入後才可閱覽掃描後的檔案)，[昭和十四年度] 臺灣總督府文教局社會課 臺灣ニ於ケル神社及宗敎 | location = 臺北市錦町 | publisher = 臺灣總督府文教局社會課 | date = 昭和15年1月19日 | pages = 原書第10頁 | url = http://hyerm.ntl.edu.tw:2135/cgi-bin/gs32/gsweb.cgi/ccd=QsffPI/record?r1=50&h1=0 | language = ja }}</ref>。
神社選址於[[台北州|台北州]][[淡水郡|淡水郡]][[淡水街|淡水街]]油車口字油車口(今[[新北市|新北市]][[淡水區|淡水區]]中正路一段6巷31號之[[新北市忠烈祠|新北市忠烈祠]])，恰好在[[滬尾砲台|滬尾砲台]]旁，此地曾為[[清法戰爭|清法戰爭]][[西仔反|滬尾之役]]的古戰場。而昔日神社前有成排的松樹，為[[淡水八景|淡水八景]]之一的「松濤落日」，淡水各機關學校也會定時參拜<ref>{{Cite web |url = http://blog.xuite.net/apex.cheng/wretch/117809626-%E7%A5%9E%E7%A4%BE%E9%81%BA%E8%B7%A1%28%E4%B9%9D%29+++%E6%B7%A1%E6%B0%B4%E7%A5%9E%E7%A4%BE |title = 神社遺跡(九)淡水神社 |author = 時空旅人 |date = 2010-03-03  |language=zh-TW }}</ref>。
二戰後淡水神社廢止，1953年成為台北縣忠烈祠。1975年依原有神社基座改建為現貌之中國北方宮殿式建築，2011年隨台北縣更名新北市而改名為[[新北市忠烈祠|新北市忠烈祠]]<ref>{{Cite web |url = https://martyrs.ntpc.gov.tw/history.aspx |title = 新北市忠烈祠官網-沿革 |author = 新北市忠烈祠 |language = zh-TW |access-date = 2018-02-14 |archive-url = https://web.archive.org/web/20180214202846/https://martyrs.ntpc.gov.tw/history.aspx |archive-date = 2018-02-14 |dead-url = yes }}</ref>。

==  [[神社建築|神社建築]]   ==
淡水神社原配置有[[參道|參道]]、混凝土造[[鳥居|明神鳥居]]2基、[[燈籠|燈籠]](石燈籠、大石燈籠、混凝土半木造春日燈籠)數對、揭示場、社號標、[[切妻造|切妻造]]手水舍、具唐破風或切妻破風入口的[[入母屋造|入母屋造]]社務所、[[切妻造|切妻造]]或[[校倉造|校倉造]]倉庫(緊鄰社務所)，主體建築部分與[[桃園神社|桃園神社]]類似，有[[切妻造|切妻造]]神門、[[入母屋造|入母屋造]]拜殿、與拜殿左右兩側相接的神饌所與祭器庫、[[切妻造|切妻造]]渡殿/幣殿、[[流造|流造]]本殿等<ref>{{Cite web |url = http://www.5819375.idv.tw/phpbb3/viewtopic.php?f=12&t=1137&p=63314&hilit=%E6%B7%A1+%E6%B0%B4+%E7%A5%9E+%E7%A4%BE#p63314 |title = 鞠園論壇中"阿六"提供的淡水神社日治舊照(發表於17:58者) |author = 阿六 |date = 2007-02-12 |language = zh-TW }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==  現況  ==

淡水神社原址在1975年改建後保留原神社參道路徑(部分路段位於淡水區中正路一段8巷)與原有玉垣、駁坎等舍殿基座部分。其餘遺跡目前被保存於新北市淡水區忠山里椿水林18號的[[行忠堂|行忠堂]]内，有16基之石燈籠、若干石燈籠構件、揭示場、手水舍之屋頂部分與手水缽<ref>{{cite news |language=zh-TW |author = 記者黃其豪  |url = http://news.ltn.com.tw/news/local/paper/49011 |title = 〈台北都會〉淡水神社燈座 差點成廢棄物 |publisher = 自由時報 |date = 2005-12-20 }}</ref>。
