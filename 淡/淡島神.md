'''淡島神'''（{{lang|ja|アワシマノカミ}}）或'''淡島明神'''為[[日本|日本]][[民间宗教|民間信仰]]的[[女神|女神]]。

== 概要 ==
日本各地的'''淡島神'''皆自[[和歌山縣|和歌山縣]][[和歌山市|和歌山市]]加太的[[淡嶋神社|淡嶋神社]]分靈而來，該[[神社|神社]]亦是全國淡島、淡路神社的總本社。大多數的淡島、淡路神社在[[明治|明治]]初年[[神佛分離|神佛分離]]的政策逼迫下，改奉[[少彥名|少彥名]]神。而各地[[寺廟|寺廟]]祭祀淡島神的地方一般稱作「淡島堂」，其中以[[東京都|東京都]][[台東區|台東區]][[淺草|淺草]][[淺草寺|淺草寺]]裡的淡島堂最為出名。

[[江戶時代|江戶時代]][[元祿|元祿]]時期，淡島信仰開始在全國流行起來，這歸功於背負著淡島明神小廟的[[苦行僧|苦行僧]]「淡島願人（{{lang|ja|あわしまがんにん}}）」（也有人說是[[廚師|廚子]]），行走全國詠唱經文，宣揚淡島明神的各種神蹟<ref>請看{{ja}}[http://www.geocities.jp/seijiishizawa/NewFiles/hina-okuri2.html 淡島信仰と流し雛 ～流し雛は雛人形の源流か～]，「淡島願人の活動」之段落。</ref>。相傳淡島明神可以治癒[[白帶|白帶]]、[[不孕|不孕]]等婦女病，也可保佑安胎順產、幫助增長[[裁縫|裁縫]]技藝等。至於淡島明神的本體由來眾說紛紜，主要有下述幾點：
# [[少彥名|少彥名]]命之說：據《[[日本書紀|日本書紀]]》卷第一神代上的第六種說法記載，少彥名命為[[醫藥|醫藥]]之神，幫助[[大己貴神|大己貴神]]建設[[國家|國家]]。後來行至熊野之御碕，遂往[[常世國|常世國]]；亦曰至'''淡島'''，乘坐粟莖彈跳渡至常世國。而《伯耆國風土記》軼文也有同樣之記載<ref>參考[http://shinku.nichibun.ac.jp/kojiruien/pdf/chib_2/chib_2_0453.pdf 地部二十四：伯耆國]，第二句。</ref>，提到少日子命（即少彥名命）往粟島（{{lang|ja|あわしま}}，發音同淡島）彈跳渡至常世國。很多自加太淡島神社分靈出去的淡島神社皆採此說法，故[[神佛分離|神佛分離]]時改奉[[少彥名|少彥名]]神。
# 「淡島」之說：按《[[古事記|古事記]]》上卷之記載，[[伊奘諾尊|伊邪那岐命]]、[[伊奘冉尊|伊邪那美命]]二神相媾生國，所生子名為[[蛭子神|水蛭子]]，祂們將此子放入以[[蘆葦|蘆葦]]編成的船而流去；次生'''淡島'''，亦不計入子之例。
# [[住吉三神|住吉明神]]后神之說：傳說淡島神為[[天照大神|天照大神]]的第6個孩子<ref>天照大神和[[素戔嗚尊|建速須佐之男命]]至天安河誓約所生之子乃：[[天忍穗耳尊|正勝吾勝勝速日天之忍穗耳命]]（《日本書紀》作正哉吾勝勝速日天忍穗耳尊）、[[天菩比神|天之菩卑能命]]（《日本書紀》作天穗日命）、[[天津日子根命|天津日子根命]]（《日本書紀》作天津彥根命）、[[活津日子根命|活津日子根命]]（《日本書紀》作活津彥根命）、[[熊野久須毘命|熊野久須毘命]]（《日本書紀》作熊野櫲樟日命）等五男神。</ref><ref>請見《日本の神々―多彩な民俗神たち》，戶部民夫著，新紀元社，1998年11月，ISBN 978-4883173242。</ref>，下嫁住吉明神，卻因染上婦女病遭流放至淡島，因此發誓要救助為婦女病所苦的女性。至於和歌山市加太對岸的[[友島|友島]]上有一座住吉神社，則是後世穿鑿附會而設置<ref>詳見《神道大辞典》，佐伯有義、宮地直一監修，臨川書店，1986年8月，ISBN 978-4653013471。</ref>。
# 婆利塞女之說：這個傳說和上一個有點關聯性，據說婆利塞女（{{lang|ja|ばりさいじょ}}）在16歲那年嫁給住吉明神，卻不幸染上婦女病，被放在小船裡流放，3月3日在淡島擱淺。由於同為婦女病所苦，便發願救助類似遭遇的女性，修禊時使用紙人除去障礙，這便是現在信徒在加太淡島神社裡貢獻女性人偶的由來。

再者，目前加太淡島神社也有祭祀息長足姬命（即[[神功皇后|神功皇后]]），該神祇也有安胎順產、治癒疾病之神格。

== 信仰 ==
主祀淡島神的[[神社|神社]]除各地之淡島、淡路神社及淡島堂外，另計有下述：
* [[淡嶋神社|淡嶋神社]]（[[和歌山縣|和歌山縣]][[和歌山市|和歌山市]]加太）：全國淡島、淡路、粟島神社的總本社，又稱加太淡島神社。
* 粟島神社（[[熊本縣|熊本縣]][[宇土市|宇土市]]新開町）：其「迷你[[鳥居|鳥居]]」<ref>參考{{ja}}[http://www.awashima.or.jp/awashima/awashima.htm 粟嶋神社 - 粟嶋神社とは]。</ref>相當出名。
* 宗德寺粟島堂（[[京都市|京都市]][[下京區|下京區]]）
* 八幡山淨光院森巖寺淡島堂（[[東京都|東京都]][[世田谷區|世田谷區]]）
* [[淺草寺|淺草寺]]淡島堂（東京都[[台東區|台東區]]）
