'''淡水紅樹林'''位[[新北市|新北市]][[淡水區|淡水區]]竹圍，是[[淡水河|淡水河]]與[[基隆河|基隆河]]交匯的出海口處。

== 背景 ==
因地緣的關係，竹圍河段是淡水及鹹水交匯處，於河灘上長了許多的[[水筆仔|水筆仔]]，就是俗稱的紅樹林植物，而這些植物伴隨著帶來了許多沼澤生態系，例如: [[招潮蟹|招潮蟹]]、[[彈塗魚|彈塗魚]]及水鳥等。

1864年，英國[[:en:Richard_Dixon_Oldham|Richard Dixon Oldham]]在此地採集到水筆仔<ref name="郭智勇">{{cite journal |author=郭智勇|title=〈台灣的紅樹林植物〉|journal=《滬尾街》|issue=第九期|date=1996-05|publisher=滬尾文史工作室|location=臺灣|language=zh-tw|isbn=}}</ref>。

== 紅樹林自然保留區 ==
[[FILE:紅樹林自然保留區.jpg|right]]
本區位於新北市淡水區竹圍附近之淡水河北岸，距離淡水河口約5公里，隸屬台灣省林務局羅東林區管轄之區外保安林<ref>{{Cite web|url=http://conservation.forest.gov.tw/ct.asp?xItem=59725&ctNode=174&mp=10|title=淡水河紅樹林自然保留區|publisher=行政院農委會林務局|date=2012-04-10|accessdate=2012-12-28|deadurl=yes|archiveurl=https://web.archive.org/web/20121025134645/http://conservation.forest.gov.tw/ct.asp?xItem=59725&ctNode=174&mp=10|archivedate=2012-10-25}}</ref>。

== 特色 ==
淡水紅樹林是世界分佈最廣的水筆仔純林，在保護區外緣自[[捷運紅樹林站|捷運紅樹林站]]至[[捷運淡水站|捷運淡水站]]間有建置自行車步道，可清楚地觀查水筆仔的胎生過程。
