'''淡水禮拜堂'''，是[[台灣|台灣]][[新北市|新北市]]市定古蹟之一，位於新北市[[淡水區|淡水區]]馬偕街上。目前的建物改建於1932年，由[[偕叡廉|偕叡廉]]設計，為仿[[歌德式建築|歌德式建築]]的紅磚建築，有一方型鐘塔，內部為木架天花板，且保存一個自1909年開始使用的古風琴。淡水禮拜堂今日依然由[[台灣基督長老教會|台灣基督長老教會]]淡水教會使用中，是淡水地區最大的台灣基督長老教會[[聚會所|聚會所]]，約可容納三百人。此教堂曾在1986年修建，將屋頂由陶製屋瓦改為鋁制浪板。<ref>{{cite web |url = https://nchdb.boch.gov.tw/assets/advanceSearch/monument/19980829000004 |title = 淡水禮拜堂 |publisher = 文化部文化資產局 |language = zh-tw |accessdate = }}</ref>

== 建築經過 ==
[[馬偕|馬偕]]牧師於1872年租下一間預備做為馬廄的房子，以此傳教。<ref>馬偕著，陳宏文譯《馬偕博士略傳、日記》，台南：台灣教會公報社，1972，頁37。</ref>
至1873年，馬偕牧師租到稍前的一間民房做為新的禮拜堂。<ref>蘇文魁編，《滬尾江河》，台北：淡水基督長老教會，1992，頁66。</ref>
在1875年（光緒元年）時，淡水教會購下今日淡水教會的土地作為宿舍使用；約1890年淡水教會翻修其上的醫生宿舍做為禮拜堂，並於翌年8月在教堂後面建造一所義塾。<ref>周宗賢，《淡水：輝煌的歲月》，臺北市 : 臺灣商務印書館，2007[民96]，頁121。</ref>

1915年（大正四年）改建禮拜堂，使用磚造材料，建築物中間建有方形鐘樓突出於正立面，下設有入口正門，外表牆面則粉飾為白色。<ref>周宗賢，《淡水：輝煌的歲月》，臺北市 : 臺灣商務印書館，2007[民96]，頁122。</ref>

1932年（昭和七年）11月18日由偕叡廉親自設計、監工，並由鍾天枝牧師協助重建，呈現了現今所見淡水禮拜堂之樣貌。
