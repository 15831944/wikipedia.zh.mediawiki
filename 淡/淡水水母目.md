'''淡水水母目'''（{{lang|la|''Limnomedusae''}}）属[[刺胞动物门|刺胞动物门]]、[[水螅纲|水螅纲]]，顾名思义是一類生活在淡水中的水母。水螅蟲裸露，並在水螅的莖分枝，屬於一種小型的水母，身長約1毫米到30毫米<ref name=mhng />。据记载：欧、亚、美洲和大洋洲的温带地区，都曾发现淡水水母目動物的存在。現時淡水水母目大約有54個不同的屬<ref name=mhng>{{cite web|url=http://www.ville-ge.ch/mhng/hydrozoa/limno/limnomedusae.htm|title=The Hydrozoa Directory: Order Limnomedusae Kramp, 1938|publisher=[http://www.ville-ge.ch/mhng/ 瑞士日內瓦市立自然歷史博物館]|author=Peter Schuchert|accessdate=2008-06-09}}</ref>。

== 特徵 ==
淡水水母目的口部一般有四片唇，但亦有六片唇的<ref name=mhng />。有兩種不同類型的觸鬚。

== 分類 ==
淡水水母目旗下有四個科：
* [[笠水母科|笠水母科]] (Olindiasidae)
* [[微水螅科|微水螅科]] (Microhydrulidae)
* [[Monobrachiidae|Monobrachiidae]]
* [[Armorhydridae|Armorhydridae]]
