'''高雄港港史館'''位於[[臺灣|臺灣]][[高雄市|高雄市]][[鼓山區|鼓山區]]，於民國九十二年（[[2003|2003]]）年2月6日公告為歷史建築{{r|文}}。該建築是[[高雄港|高雄港]]第二期築港工程時興建，曾作為稅關、港務局辦公處所使用{{r|文}}。另外此建築在某方面來說其實是未完成建築，由於財政因素，該建築只完成了計劃的三分之一便宣佈完工{{r|高}}。

[[高雄港務局|高雄港務局]]局本部遷走後，此建築改為港史館，於2002年11月28日正式開放{{r|高}}。

== 沿革 ==
高雄港港史館之建築於大正三年（1914年）開始動工，但由於[[第一次世界大戰|第一次世界大戰]]爆發、物價波動等因素，該建築在昭和十二年（1937年）時由於政府財政緊縮直接宣佈完工{{r|高}}。之後該建築作為稅關使用，昭和十八年（1943年）為高雄港港務局使用，港務局由[[中華民國交通部|中華民國交通部]]接收後仍繼續使用到民國八十三年（1994年）為止，而後一度有拆除打算，於民國八十六年（1997年）決定作為港史館{{r|高}}。港史館於2002年11月28日正式開放{{r|高}}。

== 建築 ==
港史館是兩層樓高的折衷式樣磚造建築，其正立面並不對稱，大門偏左，而大門右邊開有兩個長方形窗，二樓則是兩個弧拱窗，至於大門左邊一樓則開了一扇圓拱窗，二樓則是三扇細長條窗{{r|高}}。

而建築物的西邊牆壁為洗石子，窗戶數少，東邊牆壁則是磚牆，有數扇窗戶{{r|高}}。
