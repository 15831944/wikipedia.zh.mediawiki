'''高雄市文化中心'''（舊名'''高雄市立中正文化中心'''）是位於[[高雄市|高雄市]][[苓雅區|苓雅區]][[五福路_(高雄市)|五福一路]]的一座大型表演展覽場地，佔地約十三[[台甲|甲]]餘（13[[公頃|公頃]]），建築形式為有[[中式建築|中式建築]]特徵的[[粗野主義|粗野主義]]建築。正門對面為[[高雄市立五福國民中學|高雄市立五福國民中學]]；[[高雄師範大學|高雄師範大學]]和平校區則位在其東面。此文化中心同時也是[[高雄市政府文化局|高雄市政府文化局]]的辦公地點。在高雄縣市合併以前，高雄市文化中心是高雄市唯一的大型表演展覽專用場地。

== 歷史 ==
[[File:慈湖紀念雕塑公園.jpg|Right]]組裝展示]]
今高雄市文化中心土地的前身是一所未完工的棒球場<ref>{{cite web|author=蔡高明|url=http://collection.khm.gov.tw/search_results.aspx|title=興建中的中正文化中心|publisher=高雄市立歷史博物館典藏查詢系統}}</ref>。高雄市文化中心籌設於1974年，選定地點是[[林德官|林德官]]的「17號公園」預定地<ref name="yanyun">{{cite book|title=苓雅寮煙雲|author=杜劍鋒|year=2005|publisher=高雄市文獻委員會|isbn=9860034699}}</ref>。原規畫北至五福路，南至四維路，東至和平路，西至光華路，預定地內大多為[[違建|違建]]，在違建戶籍該選區之[[高雄市議員|高雄市議員]]的要求下，[[高雄市政府|高雄市政府]]決定「減半徵收」；時任議員[[湯阿根|湯阿根]]隨即表示，進步的城市就要有大型的公園。原本命名為「中正堂」，但由於1975年[[蔣中正|蔣中正]]總統逝世，曾改名為「中正紀念堂」，而[[高雄市苓雅區四維國民小學|高雄市苓雅區四維國民小學]]校歌第3句至今仍保持著「東鄰中正紀念堂」。工程於1976年開始，<ref name="yanyun"/>主建築師是王昭藩。<ref>{{cite web|url=http://humanitiescenter.nsysu.edu.tw/webplan_04_02.php|title=台灣建築大師 王昭藩 (1931-2006)|publisher=國立中山大學人文研究中心|accessdate=2011-01-18|deadurl=yes|archiveurl=https://web.archive.org/web/20150925125603/http://humanitiescenter.nsysu.edu.tw/webplan_04_02.php|archivedate=2015-09-25}}</ref>1980年時，因配合[[十二項建設|十二項建設]]中的「文化中心」項目，將設施正式命名做「高雄市立中正文化中心」，並且在1981年4月5日（蔣中正過世六週年）時啟用。<ref>{{cite news|title= 高雄文化中心 正式啟用|date=1981-04-05|newspaper=聯合報}}</ref>

主建築內原有有一座高6.4公尺的蔣中正銅像，銅像出自[[林木川|林木川]]之手，但在2007年3月時，因為民進黨陳水扁中央主政，[[去蔣化|去蔣化]]運動勃興，由[[高雄市政府|高雄市政府]]拆除，<ref>[http://big5.huaxia.com/zt/rdzz/07-088/538926.html 高雄中正文化中心 夜拆蔣介石銅像].華夏經緯網.2007-03-14 </ref>拆解之後送至[[桃園市|桃園]][[慈湖_(桃園)|慈湖]]旁公園，主建築匾額上的「永懷領袖」四字去除，中心名稱也改為「高雄市文化中心」。<ref>{{cite news|url=http://tw.people.com.cn/BIG5/14812/14875/5519257.html|title=高雄市文化中心完成改名挂牌 "中正"兩字被取消|date=2007-03-26|accessdate=2011-01-18|publisher=人民網}}</ref>

== 主建築 ==
該中心主體建築有4200[[坪|坪]]，館內的「'''至德堂'''」可容納1702人，是高雄市藝文演出的場地；而「'''至善廳'''」可容納483人，可作音樂、戲劇或舞蹈等表演藝術節目演出使用。除此還有「'''至真堂'''」、「'''至美軒'''」等美術展覽室，是此地藝術家及學生的展覽舞台。

因應展覽需求，前庭兩側的文物陳列室也改作展出場地之用，建築物後方一至四樓則為展覽室及附設圖書館。建築物本身還有可供活動的室外空間。文化局的入口設於至善廳。

== 其他設施 ==
[[File:Square_of_the_cultural_center.JPG|thumb]]
[[File:Sidewalk_around_the_cultural_center.JPG|thumb]]
[[File:Stone_Drums_along_sidewalks_of_Kaohsiung_Culture_Center.JPG|thumb]]
在主建築與正門之間有圓形露天廣場，四周則有綠地，是自由的活動場地。在建築後方則設有地下停車場，是1990年代後才挖設的。

文化中心的外圍原本是以弧狀的欄杆式圍牆與外圍人行道隔開。2002年時，市政府將正門整修成為「市民藝術大道」<ref>[http://www.khcc.gov.tw/home02.aspx?ID=$1607&IDK=2&DATA=-1&EXEC=L 市民藝術大道].高雄市政府文化局</ref>，同時拆除了一部分圍牆。之後逐年將四周圍牆拆除，並且重新鋪設人行道。

== 活動 ==
除了展覽及表演外，文化中心也是一些節慶活動的舉辦地點。文化中心在2000年以前曾是[[高雄燈會|高雄燈會]]的所在地，之後移往[[愛河|愛河]]。[[跨年|跨年]]晚會也曾於文化中心舉辦，近年則移往其他新的活動地點。

自2006年6月開始，每個週末在文化中心有藝術市集，供手工藝品設攤或[[街頭藝人|街頭藝人]]表演。

== 交通 ==
[[高雄市公車|高雄市公車]]在文化中心的[[五福路_(高雄市)|五福一路]]及[[和平路_(高雄市)|和平一路]]兩面設有公車站，附近也有[[高雄捷運|高雄捷運]][[高雄捷運橘線|橘線]]的[[文化中心站_(高雄市)|文化中心站]]。此外，[[高雄市公共腳踏車租賃系統|高雄市公共腳踏車租賃系統]]也在此設了租用點。
