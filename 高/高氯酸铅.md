'''高氯酸铅'''（{{lang-en|Lead perchlorate}}）是[[铅|铅]]的[[高氯酸|高氯酸]]盐，分子式为Pb(ClO<sub>4</sub>)<sub>2</sub>。

== 制备 ==
像[[高氯酸铜|高氯酸铜]]、[[高氯酸锌|高氯酸锌]]等大部分其他高氯酸盐一样，高氯酸锌容易[[潮解|潮解]]<ref name="chemicalbook"/>。

== 化学性质 ==
高氯酸铅与脱镁[[叶绿素|叶绿素]]在[[乙醇|乙醇]]溶液中能形成铅叶绿素[[配合物|配合物]]<ref name="yelvsu"/>，也能与4,13-二癸基-1,7,10,16-四氧-4,13-二氮环十八烷(cryptand[2,2,D,D])形成配合物<ref name="guanmi"/>。

== 应用 ==
高氯酸铅溶液可用于准确测定矿石或精矿中的[[金|金]]、[[银|银]]含量<ref name="jinyin"/>。
