'''高鸿宾'''（{{bd|1953年|5月|||catIdx=}}）籍贯不详，[[中华人民共和国|中华人民共和国]]官员。

==生平==
1975年12月加入[[中国共产党|中国共产党]]，1969年9月参加工作。研究生学历，高级经济师。自1980年代初开始从事农业、农村经济政策研究，曾长期从事国家扶贫开发工作<ref>{{Cite web |url =http://www.moa.gov.cn/leaders/gaohongbin/jianli/201009/t20100920_1661011.html  |title = 高鸿宾简历 |author =  |date =2010-09-20  |publisher =  中华人民共和国农业部 |language =  |accessdate =  |quote =  }}</ref>。1995年2月至1996年4月，任[[国务院扶贫开发领导小组办公室|国务院扶贫开发领导小组办公室]]外资项目管理中心主任。1996年4月至2003年3月，任[[国务院扶贫开发领导小组|国务院扶贫开发领导小组]]成员、办公室副主任。2003年3月至2007年7月，任国务院扶贫开发领导小组办公室副主任、党组副书记<ref>{{Cite web |url =http://finance.ifeng.com/people/official/gaohongbing.shtml  |title =高鸿宾  |author =  |date =  |publisher =凤凰网  |language =  |accessdate =2017-11-05  |quote =  }}</ref>。2007年7月至2013年7月，任[[中华人民共和国农业部|中华人民共和国农业部]]党组成员、副部长<ref>{{cite news|title=高鸿宾副部长会见欧盟委员会卫生和消费者政策委员约翰·达利先生一行|url=http://www.moa.gov.cn/leaders/gaohongbin/huodong/201203/t20120320_2513240.html|accessdate=2012-04-22|newspaper=中华人民共和国农业部}}</ref><ref>{{cite news|title=财政部原副部长丁学东被任命为国务院副秘书长|url=http://finance.ifeng.com/news/people/20100618/2323131.shtml|accessdate=2012-06-09|newspaper=凤凰网}}</ref><ref>{{cite news|title=国务院任免国家工作人员 宫蒲光任民政部副部长|url=http://www.chinanews.com/gn/2013/07-23/5072868.shtml|accessdate=2013-07-27|newspaper=中国新闻网|date=2013年7月23日}}</ref>。

他是[[第十二届全国政协委员|第十二届全国政协委员]]<ref>{{Cite web |url = http://cppcc.people.com.cn/n/2014/0226/c34948-24468039.html |title = 中国人民政治协商会议第十二届全国委员会委员名单 |author =  |date =  2014-05-21|publisher = 中国政协新闻网 |language =  |accessdate =  |quote =  }}</ref>。
