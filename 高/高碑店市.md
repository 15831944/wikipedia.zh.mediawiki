'''高碑店市'''是[[中华人民共和国|中华人民共和国]][[河北省|河北省]]的一个[[县级市|县级市]]，由[[保定市|保定市]]代管。位于保定市北部，北距[[北京|北京]]82公里，东距[[天津|天津]]134公里，南距保定68公里。与[[河北省|河北省]][[涿州市|涿州市]]、[[定兴县|定兴县]]、[[涞水县|涞水县]]相邻，具有较为悠久历史。市政府驻城区东大街18号。

== 历史 ==
[[西汉|西汉]]置[[新昌侯国|新昌侯国]]，[[东汉|东汉]]废；[[唐朝|唐朝]][[大曆|大历]]四年（769年）析[[固安县|固安县]]地，复置[[新昌县_(唐朝)|新昌县]]。[[大和_(唐朝)|太和]]六年（832年），于古[[督亢|督亢]]地区置[[新城县_(唐朝)|'''新城县''']]，两县均属[[河北道|河北道]][[涿州_(古代)|涿州]]，后均废置；[[后唐|后唐]][[天成_(后唐)|天成]]四年（929年），复析[[范阳县|范阳县]]置新城县；[[辽朝|辽]][[金朝|金]]因之，属涿州；[[明朝|明]][[清朝|清]]属[[保定府|保定府]]；[[民国|民国]]二年直属[[直隸省|直隸省]]。

1970年，县人民政府迁至高碑店。1993年4月9日撤销新城县，建立'''高碑店市'''，属[[河北省|河北省]]，由[[保定市|保定市]]代管；2013年财政由[[河北省人民政府|河北省人民政府]]直管。

== 经济 ==
2008年，全市生产总值完成94亿元。城镇居民人均可支配收入达到12867元，农民人均纯收入达到5026元。出口创汇完成800万美元。

== 特产 ==
最享誉盛名的要属几百年历史的高碑店[[豆腐丝|豆腐丝]]，其口味细腻独特，别具特色。
