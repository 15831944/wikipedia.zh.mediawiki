== 概要 ==
本系列作品是[[高橋留美子|高橋留美子]]從1987年發表的短篇「浪漫{{lang|ja|の}}商人」開始，原本是每個月以前篇、後篇方式發表，後來才轉以每年會在連載其它作品期間定期發表一部。

2003年7月至9月，由[[東京電視台|東京電視台]]系列在深夜時段播出全13話的電視動畫（另外又稱做「第1系列」）。並於第1系列播出之後，緊接著以前以[[OVA|OVA]]方式製作的「[[人魚之森|人魚之森]]」跟著被製作成全11話的電視動畫，同樣在東京電視台的深夜時段播出，標題名稱叫做「'''高橋留美子劇場 人魚之森'''（<small>{{lang-ja|高橋留美子劇場 人魚の森}}</small>）」（同樣被稱做第2系列，由[[山寺宏一|山寺宏一]]、[[高山南|高山南]]擔當主演）。

每一集的主角與要角配音員大多起用以前參加過高橋留美子作品動畫版的陣容。

== 電視動畫 ==
首播時間於2003年7月5日至同年9月27日、每週六深夜24時50分至25時20分（[[日本標準時間|日本標準時間]]）。以「'''高橋留美子劇場'''（第1系列）」作為標題，在[[東京電視網|東京電視網]]播出，全13話。

另外，該動畫在故事中加入了原作沒出現的「在這集登場的角色會在其它集數中友情客串」等娛樂要素。

該動畫於東京電視網播畢之後，至今仍不定時在[[全國獨立放送協議會|獨立局]]聯播網進行實質的再放送，2014年7月4日起在[[京都放送|KBS京都]]的深夜時段、2015年3月30日起每週一在[[千葉電視台|千葉電視台]]的晚間19時時段、2016年10月4日至12月27日每週二在[[SUN電視台|SUN電視台]]的深夜24時時段，最近一次是在2018年2月2日至6月15日每週五在[[中京電視台|中京電視台]]（[[日本電視網|日本電視網]]）的深夜時段播出。

「第2系列」內容請參照[[人魚之森#電視動畫|人魚之森（動畫版）]]。

== 電視劇 ==
2012年7月8日和7月15日，於[[NHK_BS_Premium|NHK BS Premium]]的「Premium Drama」時段，播出上下總共2集的電視劇，標題分別名為『紅色花束（<small>{{lang-ja|赤い花束}}</small>）』和『命運之鳥（<small>{{lang-ja|運命の鳥}}</small>）』。

還有，每一集的故事架構都有重新調整，上集『紅色花束』是由「紅色花束」「盆栽中的秘密」「迷失家庭F」這3部短篇所組成。

而下集『命運之鳥』是由「命運之鳥」「專務之犬」「只要有你在」這3部短篇所組成。
