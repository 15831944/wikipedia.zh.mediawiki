'''高{{ruby-py|要|-{zh-cn:yāo;zh-tw:ㄧㄠ;zh-hk:jiu<sup>1</sup>}-}}区'''是[[中华人民共和国|中华人民共和国]][[广东省|广东省]][[肇庆市|肇庆市]]下辖的[[市辖区|市辖区]]，位于[[广东省|广东省]]西北部。[[面积|面积]]約2,196[[平方千米|平方-{zh:公里;zh-hans:千米;zh-hk:公里;zh-tw:公里;}-]]。[[人口|人口]]约78万。

== 历史 ==
据《[[水经注|水经注]]》、《高要[[县志|县志]]》记载，[[漢武帝|漢武帝]][[元鼎|元鼎]]六年（公元前111年）置[[高要縣|高要縣]]，因高要峽（今[[羚羊峽|羚羊峽]]）居高險要而得名。1956年置[[高要專區|高要專區]]。1993年撤縣建市（縣級）。

2015年4月28日，国务院关于同意广东省调整肇庆市部分行政区划的批复（国涵[2015]76号）。撤销高要市，设立[[肇庆市|肇庆市]][[高要区|高要区]]，以原高要市的行政区域为高要区的行政区域。

== 地理 ==
高要地处[[粤西|粤西]]山地东缘，地势西北高东南低。

== 交通 ==
[[广茂铁路|广茂铁路]]斜贯市境；内河航运可至[[广州|广州]]及[[广西|广西]][[梧州|梧州]]等地；[[广肇高速公路|广肇高速公路]]、[[江肇高速公路|江肇高速公路]]及[[广梧高速公路|广梧高速公路]]通广州、梧州、珠海等地。

== 经济 ==
工业有精密压铸、食品、轻纺、电子、机械、建材、森工等。农业耕地4.87万公顷，为广东省重要粮食产地之一，有林地10.59万公顷，林木有松、杉、竹、桂木，产松香、桂皮、竹叶球等。

== 物产 ==
* 矿藏有金、银、锡、石灰岩、云母等。
* 土特产有金渡端砚、大湾镇麦溪鲤、金渡花席，禄步、水南、乐城等地区的桂皮 、松香、桂油、巴戟、佛手等，双马菠萝、古迳马蹄、蚬岗西瓜，莲塘、白土、新桥等镇的裹蒸；剑花。

== 文化 ==
民间节日有河台的开耕节，莲塘、白土一带农村的“茶裹节”。金渡行村节（放炮节）。金利龙舟节。

== 电视广播 ==
长年以来，高要市当局会每天中午12时35分及逢周六、日晚23：15分在[[翡翠台|翡翠台]]、[[本港台|本港台]]插播由高要市广播电视台制作的《高要新闻》。广东省广播电影电视局声称，该台经[[中共|中共]]市委、市政府批准确有在港台节目中插播《高要新闻》，根据“有关协议”（无法得知协议名称及内容），插播新闻均是在广告时段插播，并无影响电视连续剧的正常播放，偶尔会因重大新闻（宣传任务）造成“些许超时”。广电当局亦没有解释为何《高要新闻》一定要在翡翠台、本港台，而不在自己的高要市广播电视台频道播放，或者在广东省的电视台，如[[广东新闻频道|广东新闻频道]]中插播。<ref>{{cite web|title=投诉高要电视插播新闻节目|url=http://www.rftgd.gov.cn/wsbs/jzxx_view.dy?mid=18201|publisher=广东广播影视网 - 咨询投诉|date=2012-08-04|accessdate=2012-09-05|deadurl=yes|archiveurl=https://web.archive.org/web/20160304073448/http://www.rftgd.gov.cn/wsbs/jzxx_view.dy?mid=18201|archivedate=2016-03-04}}</ref>
