'''高井田中央站'''（{{jpn|j=高井田中央駅|hg=たかいだちゅうおうえき|rm=Takaida-chūō eki|f=y}}）位於日本[[大阪府|大阪府]][[東大阪市|東大阪市]]川俣一丁目，是[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）[[大阪東線|大阪東線]]的[[鐵路站|鐵路站]]，[[車站編號|車站編號]]是JR-F09。

==歷史==
大阪東線是「城東貨物線」（確切地說是[[片町線|片町線]]的貨運支綫）旅客線化改造而成的客運線。

* [[2007年|2007年]][[8月23日|8月23日]] - 站名決定為「高井田中央站」。之前的暫定站名為「高井田站」<ref name="jrw_20070823">{{cite web|url=http://www.westjr.co.jp/news/newslist/article/1173226_799.html|title=大阪外環状線（仮称）の路線名・駅名などについて|accessdate=2018-10-30|date=2007-08-23|work=[[西日本旅客鐵道|西日本旅客鐵道]]|archiveurl=https://web.archive.org/web/20080617234029/http://www.westjr.co.jp/news/newslist/article/1173226_799.html#|archivedate=2008-06-17|deadurl=yes}}</ref>。
* [[2008年|2008年]][[3月15日|3月15日]] - 開業。先行導入大阪環状、大和路線運行管理系統<ref>{{Cite web |url=http://www.westjr.co.jp/news/newslist/article/1174388_799.html |title=大阪環状・大和路線運行管理システムの使用開始について |publisher=[[西日本旅客鐵道|西日本旅客鐵道]] |accessdate=2018-10-31 |deadurl=yes |archive-url=https://web.archive.org/web/20091003010754/http://www.westjr.co.jp/news/newslist/article/1174388_799.html# |archive-date=2009-10-03 }}</ref>。
* [[2018年|2018年]][[3月17日|3月17日]] - 導入車站編號<ref>{{Cite web |url=http://www.westjr.co.jp/press/article/2016/07/page_8973.html |title=近畿エリアの12路線 のべ300駅に「駅ナンバー」を導入します！ |publisher=[[西日本旅客鐵道|西日本旅客鐵道]] |accessdate=2018-10-31}}</ref><ref>{{Cite web |url=http://www.westjr.co.jp/press/article/items/160720_03_ekinumber.pdf |title=JR線 駅ナンバー路線図 |publisher=[[西日本旅客鐵道|西日本旅客鐵道]] |accessdate=2018-10-31}}</ref><ref>{{Cite web |url=http://www.westjr.co.jp/press/article/items/160720_01_ekinumber.pdf |title=「駅ナンバー」一覧表 |publisher=[[西日本旅客鐵道|西日本旅客鐵道]] |accessdate=2018-10-31}}</ref>。
* [[2019年|2019年]][[3月16日|3月16日]] - 隨大阪東線延伸新大阪，與隣隔的[[JR河内永和站|JR河内永和站]]升格為直通快速列車的停車站。同時成為[[特定都区市内|大阪市内]]站<ref name="jrw_20181227">{{Cite press release |title= おおさか東線開業に伴う運賃に関わる認可申請などについて |url= https://www.westjr.co.jp/press/article/2018/12/page_13598.html |publisher=西日本旅客鐵道 |date=2018-12-27 |accessdate=2019-1-12}}</ref>。

==站名由来==
大阪東線的車站暫定名為'''高井田站'''，但[[關西本線|關西本線]]（[[大和路線|大和路線]]）上已經有{{link-ja|高井田站 (大阪府柏原市)|高井田駅 (大阪府柏原市)|同名站}}（[[柏原市|柏原市]]），為了區別，車站名稱使用'''高井田中央站'''。 “中央”源於車站位於中央大通之上<ref name="jrw_20070823"></ref>。

此外，本站可與地鐵「'''中央'''線」換乘。但行政區劃上，本站不在高井田地區的「中央」位置，而是位於東北邊。

==車站構造==
[[島式月台|島式月台]]，1面2線的[[架空車站|架空車站]]，月台長度可對應8輛編成列車，月台屋頂長度為6節車廂。本站沒有[[轉轍器|轉轍器]]或[[日本铁路信号#主信号机|絕對信號機]]，所以被分類為[[停車場_(鐵路)|停留所]]。設有一台升降機、和上下行各1部共2部扶手電梯連接月台。[[大阪市高速電氣軌道|大阪地鐵]][[中央線_(大阪市高速電氣軌道)|中央線]]的[[高井田站_(東大阪市)|高井田站]]位於正下方，該站的車站入口位於高井田中央站2號入口附近。本站沒有設置綠窗口，出入閘口的職員窗口亦不售票；改爲設置了綠色售票機。

由[[放出站|放出站]]管理，車站業務委托予[[JR西日本交通服務|JR西日本交通服務]]<ref>{{cite web| url = http://jrw-kids.jp/jk-nishiko/company.html　| title = 株式会社 JR西日本交通サービス　会社概要　|  publisher = [[JR西日本交通服務|JR西日本交通服務]]| accessdate = 2018-10-31 }}</ref>，一部分時間無人駐守。出入閘機、售票機、精算機附近設有内綫電話，在無人駐守的時間帶可與呼叫中心的操作員聯絡，並由操作員遠程控制各種設備。

本站以「製造業的中心—先進性、能源」為設計概念。外墻有不銹鋼所組成的波浪紋，以表現高井田地區高科技的製造業<ref>{{cite web| url = http://www.city.higashiosaka.lg.jp/cmsfiles/contents/0000003/3501/080315.pdf| format = PDF| title = 東大阪市政だより 2008年3月15日号| date = 2008-03-15 |  publisher = [[東大阪市|東大阪市]]經營企劃部廣報廣聽室廣報課| accessdate = 2018-10-31 }}</ref>。大阪東線的中途站均導入了車站色系，其中本站的顔色是紅色。
