'''高露'''{{BD|1982年|10月8日|||catIdx=G高}}出生于[[北京市|北京市]]，毕业于[[中央戏剧学院|中央戏剧学院]]表演系2000级本科，为中国大陆影视演员。

== 生平 ==
1982年，高露出生在[[北京市|北京市]]，曾在[[中央戏剧学院|中央戏剧学院]]就读。由于高露和[[高圆圆|高圆圆]]长相颇像，经常被误以为是高圆圆。<ref>{{cite web |url=http://yule.sohu.com/20050524/n225685161.shtml |title=多次被误认是高圆圆 高露笑称：圆圆是我姐姐 |accessdate= 2005|publisher=搜狐网 |archiveurl=http://yule.sohu.com/20050524/n225685161.shtml |archivedate=2012年12月30日 |work= |language=zh-cn}}</ref>

=== 感情生活 ===
高露已经结婚，老公為攝影師，育有一女一子。<ref>{{cite web |url=http://news.yule.com.cn/html/201110/99218.html |title=高露首度回应感情问题：我的婚姻用心经营 |accessdate= 2011-10-28 |publisher= 中国娱乐网|archiveurl=http://news.yule.com.cn/html/201110/99218.html |archivedate=2012年12月30日 |work= |language=zh-cn}}</ref>
