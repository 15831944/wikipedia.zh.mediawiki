'''高國輝'''（'''Kuo Hui KAO'''，1985年9月26日─），[[臺灣|臺灣]][[花蓮縣|花蓮縣]]人，父親是[[客家人|客家人]]，母親是[[阿美族|阿美族]]人，[[棒球|棒球]][[运动员|選手]]，曾效力於[[美國職棒|美國職棒]][[西雅圖水手|西雅圖水手]]隊[[小聯盟|小聯盟]]體系，現為[[富邦悍將|富邦悍將]][[外野手|外野手]]，於2002年至2003年間將姓氏改為'''[[羅|羅]]'''，2012年10月改回父姓，外號為「棒球情人」和「桃園王」。

「桃園王」的由來是來自其在[[桃園國際棒球場|桃園國際棒球場]]出賽時的高全壘打率。諷刺的是，[[中華職棒歷年選秀會_#職棒二十四年（2013年）|2012年季末選秀]]時，主場在[[桃園國際棒球場|桃園國際棒球場]]的[[Lamigo桃猿|Lamigo桃猿]]總教練[[洪一中|洪一中]]原本有機會選其入隊，卻選擇投手[[曾琮萱|曾琮萱]]（目前已釋出），因此才在第二輪第五順位被[[義大犀牛|義大犀牛]]選入。中華職棒生涯面對[[Lamigo桃猿|Lamigo桃猿]]的打擊三圍為0.357/0.422/0.708，[[整體攻擊指數|OPS]]高達1.130，明顯高於生涯平均<ref>打擊三圍即為打擊率/上壘率/長打率，數據統計於2017年球季結束</ref>。

有三位弟弟，目前也都加入[[中華職棒|中華職棒]]，依年齡長幼次序為[[羅國華_(1986年出生)|羅國華]]、[[羅國龍|羅國龍]]、[[高國麟|高國麟]]，其中[[高國麟|高國麟]]與'''高國輝'''在同隊一同效力[[富邦悍將|富邦悍將]]。

==生涯歷程==
*其最為台灣人所熟知的一役應屬[[2008年夏季奧運棒球最終資格排名賽|2008年北京奧運八搶三資格賽]]中對[[加拿大棒球代表隊|加拿大隊]]的逆轉三分[[全壘打|全壘打]]<ref>{{cite web |language = zh-tw |url=http://www.youtube.com/watch?v=sReHfX8-iyE |title = 八搶三對加拿大隊的逆轉三分砲 |publisher =    |date = 2008-03-10}}</ref>，這支全壘打直接讓當時[[緯來體育台|緯來體育台]]主播[[蔡明里|蔡明里]]直接大喊「[https://www.youtube.com/watch?v=9cEBv394ugc 羅國輝我愛你]」。
*2011年11月3日[[MLB明星台灣大賽|MLB明星台灣大賽]]第二戰於[[台中洲際棒球場|台中洲際棒球場]]比賽中，因滑回本壘時左腳尖不幸正好卡在打擊區的土坑導致滑壘動作異常，造成'''左腳腳踝脫臼'''及'''小腿閉鎖性骨折'''的嚴重傷勢，手術後評估至少需休養三個月以上。<ref>{{cite web |language = zh-tw |url = http://www.youtube.com/watch?v=v1Ezpe3_2Ss |title = 羅國輝滑回本壘因而受傷 |publisher =    |date = 2011-11-03}}</ref>經過治療與復健之後，於2012年8月已痊癒。
*2012年末返台參加[[中華職棒歷年選秀會_#職棒二十四年（2013年）|中華職棒選秀會]]，並在第二輪第五順位被[[義大犀牛|義大犀牛]]選入。據他個人在《四百英尺外的王者：高國輝》一書中提到，原本當年自認為只有[[胡金龍|胡金龍]]、[[羅錦龍|羅錦龍]]、[[陳鴻文|陳鴻文]]的順位會在他前面，因此預估自己將會被[[Lamigo桃猿|Lamigo桃猿]]隊選中<ref>[http://magazine.chinatimes.com/ctweekly/20160408004731-300106 無緣的桃園王 高國輝用全壘打飆身價]</ref>，結果桃猿隊當年主導選秀的[[洪一中|洪一中]]總教練卻選擇[[曾琮萱|曾琮萱]]（職業一軍生涯0勝，現已暫離職棒）。
*2014年動[[脊椎|脊椎]]滑脫手術，原預計全年報銷，但下半季就復出，擊出18支[[全壘打|全壘打]]，逆轉拿下全壘打王。
*2015年9月26日單季34轟打破中職紀錄，當年季賽全壘打數39支締造聯盟新紀錄，並打破原[[統一獅|統一獅]][[提爾森·布雷托|布雷]]於2007年季賽所保持的33支全壘打紀錄。
*2016年5月6日於[[桃園國際棒球場|桃園球場]]對戰[[Lamigo桃猿|Lamigo桃猿]]，五局上胞弟[[羅國麟|羅國麟]]先敲出生涯首轟，自己在同局面對投手[[鄭承浩|鄭承浩]]時也開轟，創造兄弟檔同場同半局皆敲出全壘打的紀錄。
*2016年5月10日在[[澄清湖棒球場|澄清湖棒球場]]面對[[統一7-ELEVEn獅|統一7-ELEVEn獅]]，由於[[羅國麟|羅國麟]]於三局上接替原[[游擊手|游擊手]][[陳凱倫|陳凱倫]]出賽，對方[[羅國龍|羅國龍]]也先發，創造三兄弟首次同時出賽紀錄；四局上羅國龍上場打擊時更創造三人同時在場中的紀錄<ref>[http://sports.ettoday.net/news/695908 影／兄弟鬩牆！ 高國輝、羅國龍、羅國麟首度同在場中]</ref>。
*2016年9月18日，也就是[[陳金鋒|陳金鋒]]引退賽當日，以中華職棒生涯第364場出賽達成百轟，打破陳金鋒保持的中職最快百轟紀錄，且在回本壘時比出52向陳金鋒致敬<ref>[http://sports.ltn.com.tw/news/breakingnews/1829699 中職》高國輝改寫陳金鋒最快百轟紀錄 比出52手勢向鋒哥致敬]，[[自由時報|自由時報]]，20160918</ref>。
*2016年賽季以34支全壘打達成聯盟史上首次的全壘打王三連霸紀錄<ref>以較少打數擊敗同為34支全壘打的[[中信兄弟|中信兄弟]][[林智勝|林智勝]]</ref>。
*2018年5月13日，於[[桃園國際棒球場|桃園國際棒球場]]對戰[[Lamigo桃猿|Lamigo桃猿]]隊，於處理[[王柏融|王柏融]]的深遠飛球時，不慎用手套將球托過全壘打牆，造成對方擊出兩分打點全壘打。
