'''高鐵桃園站'''是[[台灣高速鐵路|台灣高速鐵路]]、[[桃園機場捷運|桃園機場捷運]]的一座[[鐵路車站|鐵路車站]]，位於[[桃園市|桃園市]][[中壢區|中壢區]][[青埔_(桃園市)|青埔]]，故俗稱'''高鐵青埔站'''<ref>{{cite news|title=鄭文燦：高鐵青埔站是台灣櫥窗、機場入口|url=http://www.mygonews.com/news/detail/news_id/115576/%E9%84%AD%E6%96%87%E7%87%A6%EF%BC%9A%E9%AB%98%E9%90%B5%E9%9D%92%E5%9F%94%E7%AB%99%E6%98%AF%E5%8F%B0%E7%81%A3%E6%AB%A5%E7%AA%97%E3%80%81%E6%A9%9F%E5%A0%B4%E5%85%A5%E5%8F%A3|publisher=mygonews|date=2015-12-18|accessdate=2016-05-09}}</ref><ref>{{cite news|title=台灣醫療專區 相中高鐵青埔站|url=http://mag.udn.com/mag/digital/storypage.jsp?f_ART_ID=308285|publisher=聯合新聞網|date=2011-03-22 |accessdate=2014-08-17|archiveurl=https://web.archive.org/web/20140819082410/http://mag.udn.com/mag/digital/storypage.jsp?f_ART_ID=308285|archivedate=2014-08-19}}</ref>。除了車站大廳與出入口外，另還設有運務管理中心、高鐵行控中心。至2016年為止，桃園站營運人次為第四名，前三名分別為台北站、台中站與左營站，第五名為新竹站<ref name = ridership/>。

車站因鄰近[[空軍桃園基地|空軍桃園基地]]而有航空地域建築限制，且設站地點地勢較高，在經費允許下設計為地下車站。由於月台與軌道皆設於地下，為了解決列車高速通過引起的[[活塞效應|活塞效應]]而需設置解壓井，但初通車時解壓井設計不當，列車時速被限制為120公里；直到2011年改善工程完成，列車才得以以時速300公里通過<ref>{{cite news|url=http://www.tvbs.com.tw/NEWS/NEWS_LIST.asp?no=arieslu20070407130140|title=怕玻璃碎！ 高鐵過桃園站先減速|author=游皓婷|publisher=TVBS新聞|accessdate=2012-11-04|language=zh-tw|date=2007-04-07}}</ref>。航空限高因空軍桃園基地解編而於2013年6月解除<ref>{{Cite news|url=http://www.chinatimes.com/newspapers/20130715000113-260204|title=青埔航高解禁 推案爆量|last=中時電子報|first=|date=2013-07-15|work=|newspaper=中時電子報|accessdate=2017-02-09|language=zh-TW}}</ref>。

[[桃園機場捷運|桃園機場捷運]][[高鐵桃園站_(桃園捷運)|高鐵桃園站]]位於高鐵車站北邊高鐵北路一段上，除了經由戶外通道與高鐵車站一樓連接外，另設有專用通道和閘門與高鐵車站地下一樓收費區連接，高鐵旅客可藉由此來往[[臺灣桃園國際機場|臺灣桃園國際機場]]及桃園市、新北市、臺北市等沿線地區。

== 歷史 ==
*2002年5月30日：開始動工。
*2006年11月10日：正式啟用。
*2007年1月5日：高鐵'''[[板橋車站_(臺灣)|板橋]]－[[新左營車站|左營]]'''段通車，開始停靠列車。
*2009年12月16日：增設[[中華航空|中華航空]]報到櫃檯<ref>[https://tw.news.yahoo.com/%E5%A0%B1%E5%88%B0-%E9%A0%98%E7%99%BB%E6%A9%9F%E8%AD%89-%E6%89%98%E9%81%8B%E8%A1%8C%E6%9D%8E%E5%85%8D%E5%88%B0%E6%A9%9F%E5%A0%B4%E6%9C%8D%E5%8B%99-%E8%8F%AF%E8%88%AA%E7%8D%A8%E5%AE%B6-055132064.html 報到、領登機證、托運行李免到機場服務！華航獨家]</ref>。
*2010年3月1日：增設[[長榮航空|長榮航空]]報到櫃檯。
*2014年2月28日：撤除[[長榮航空|長榮航空]]報到櫃檯。
*2015年1月1日：撤除[[中華航空|中華航空]]報到櫃檯<ref>[http://www.appledaily.com.tw/appledaily/article/headline/20141208/36253697/ 使用率低 華航將撤高鐵櫃台]</ref>。
*2017年3月2日：[[桃園捷運機場線|桃園捷運機場線]][[高鐵桃園站_(桃園捷運)|高鐵桃園站]]啟用。

== 車站構造 ==

*[[高鐵桃園站|高鐵桃園站]]地下車站的[[側式月台|側式月台]]兩座，總樓地板面積為81,516平方公尺<ref>[http://www.hsr.gov.tw/icons/2sets/station_taoyuan.htm 高鐵車站簡介－桃園站]</ref>。
*與{{rint|tw|tym}}[[桃園捷運|桃捷系統]]{{rint|taoyuan|airport}}[[高鐵桃園站_(桃園捷運)|高鐵桃園站]]共站。
