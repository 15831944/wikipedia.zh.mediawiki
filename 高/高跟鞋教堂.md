'''高跟鞋教堂''' ，又稱'''玻璃高跟鞋教堂'''，是位於[[臺灣|臺灣]][[嘉義縣|嘉義縣]][[布袋鎮|布袋鎮]]海景公園內的大型[[裝置藝術|裝置藝術]]造景，由[[雲嘉南濱海國家風景區|雲嘉南濱海國家風景區]]管理處建造。雖以[[教堂|教堂]]為名，實際上並非[[基督教|基督教]]宗教場所，並沒有[[牧師|牧師]]、[[神父|神父]]等神職人員駐在，而是一個無宗教的西式[[結婚禮堂|結婚禮堂]]。

== 建築特色 ==
外觀為[[藍色|藍色]][[透明|透明]][[玻璃|玻璃]][[高跟鞋|高跟鞋]]造型，主體結構總高約17公尺，長達11公尺，以1269根鋼架和320片[[玻璃|玻璃]]拼組而成，每片[[玻璃|玻璃]]的大小與拼裝角度各異，總工程費用耗資新台幣2317萬元。

== 金氏世界紀錄 ==
2016年4月申請了[[吉尼斯世界纪录大全|吉尼斯世界纪录]]<ref>{{cite news|title=全世界最大高跟鞋教堂 申請金氏紀錄認證 |url=http://udn.com/news/story/7314/1611177-全世界最大高跟鞋教堂-申請金氏紀錄認證 |language=中文 |author=卜敏正 |publisher=聯合新聞 |date=2016-04-06 |accessdate=2016-06-25 }}</ref>，同年6月23日獲得金氏世界紀錄認證為全世界最大高跟鞋型建築物。<ref>{{cite news|title=全球最大高跟鞋教堂　獲金氏世界紀錄認證 |url=http://m.appledaily.com.tw/realtimenews/article/new/20160623/892663/ |language=中文 |author= |publisher=蘋果即時 |date=2016-06-23 |accessdate=2016-06-25 }}</ref>

經查先前世界紀錄最大的高跟鞋建築高度僅9公尺，但高跟鞋教堂為17.76公尺，幾乎超過一倍，因此吸引超過300多家國外媒體報導。

== 建造背景 ==
[[台灣|台灣]]嘉南沿海地帶，過去曾經因[[烏腳病|烏腳病]]盛行，導致一名將軍鄉的王姓女子在婚前發病，而必須截斷足部，無法在結婚時穿上[[高跟鞋|高跟鞋]]展開全新的人生。<ref>[http://www.chinatimes.com/newspapers/20151009000422-260102 高跟鞋教堂動土 布袋新地標] [2015.10.09]</ref>雲嘉南濱海國家風景管理處認為，在[[布袋鎮|布袋鎮]]建造一座象徵未來[[幸福|幸福]]的玻璃高跟鞋教堂，可藉此彌補當年許多因[[烏腳病|烏腳病]]而被迫截去雙足女子的心中遺憾，完成她們的夢想。<ref name="烏腳" />同時，期待藉由高跟鞋教堂的設立，作為[[雲嘉南|雲嘉南]]地區告別過去貧瘠時光的象徵，希望以此開創當地[[觀光產業|觀光產業]]，推動[[嘉義縣|嘉義縣]]沿海地區發展進入一段嶄新未來。<ref name="觀光局" />
== 爭議 ==
高跟鞋教堂在建造期間就不斷飽受外界質疑，擔心將來可能淪為蚊子館等種種問題而一度停工，後來經各方協調才又低調復工。<ref name="烏腳">[http://udn.com/news/story/7314/1438473-嘉義布袋高跟鞋教堂-背後是烏腳病故事 嘉義布袋高跟鞋教堂 背後是烏腳病故事] [2016.01.13]</ref><ref>[http://udn.com/news/story/7314/1274179-嘉義布袋「高跟鞋教堂」-溝通後復工   嘉義布袋「高跟鞋教堂」 溝通後復工] [2015.10.27]</ref><br />

建築本身及造型美感則引發台灣建築界討論，認為玻璃高跟鞋教堂的建築設計，與當地以[[濕地|濕地]]、漁村景色為主的沿海環境差異過大，欠缺建築必須與在地連結的審慎思考。<ref>[http://udn.com/news/story/7314/1243084-嘉義布袋海景中-冒出「高跟鞋教堂」  嘉義布袋海景中 冒出「高跟鞋教堂」] [2015.10.12]</ref><ref>[http://e-info.org.tw/node/113210 回應高跟鞋教堂爭議 雲管處長：人潮先來，談文化才會成功] [2016.02.05]。</ref>[[東海大學建築系|東海大學建築系]]主任邱浩修更指出，高跟鞋教堂的建造，不論在歷史面、文化面、環境面、藝術面或宗教面的考量上，都有所不足。<ref>[http://www.chinatimes.com/newspapers/20160131001129-260102 拍照、打卡...然後呢？ 高跟鞋教堂幸福的災難] [2016.01.31]</ref>以致缺乏遊客。<ref>https://tw.appledaily.com/new/realtime/20181206/1474779/</ref>

亦有人認為把高跟鞋與當地烏腳病史扯上關係說法牽強，因為當地烏腳病史相關文獻並沒有任何關於高跟鞋的紀錄。<ref>[http://www.new7.com.tw/NewsView.aspx?t=03&i=TXT20160413173410C6A 【社會事】從一隻高跟鞋看台灣的觀光夢]</ref><ref>[http://ourisland.pts.org.tw/content/%E8%AA%B0%E7%9A%84%E9%AB%98%E8%B7%9F%E9%9E%8B#sthash.9cEIUZPV.dpbs 誰的高跟鞋]</ref>
