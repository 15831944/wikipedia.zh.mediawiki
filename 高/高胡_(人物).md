'''高胡'''（{{bd|1927年|1月|2017年|10月16日}}），原籍[[江苏省|江苏省]][[金湖县|金湖县]]，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中共福建省委|中共福建省委]]常委、中共福建省委组织部部长、[[中共福建省纪委|中共福建省纪委]]书记，[[福建省政协|福建省政协]]副主席。

==生平==
1940年7月参加[[中国共产党|中国共产党]]领导的革命工作，历任[[新四军|新四军]]战士、见习医生、医务员，1942年3月加入中国共产党<ref>{{Cite book|title=淮安文史资料 第19辑 淮安名人 下|last=淮安市政协文史资料委员会编|first=|publisher=淮安市政协文史资料委员会编|year=|isbn=|location=|pages=46}}</ref>。[[第二次国共内战|第二次国共内战]]时期，历任[[华东军区|华东军区]]第三后方医院室长、所长，华东军区卫生部第一重伤医院所长、队长，[[福建省|福建省]][[福州市|福州市]]军管会卫生处军代表等职，参加了[[淮海战役|淮海战役]]、[[渡江战役|渡江战役]]、上海战役、杭州战役、福州战役等<ref>{{Cite book|title=高邮文史资料 十二辑 高邮当代人物 续辑|last=政协江苏省高邮市文史资料委员会编|first=|publisher=|year=1993年12月|isbn=|location=|pages=42}}</ref>。[[中华人民共和国|中华人民共和国]]成立后，历任[[福建省卫生厅|福建省卫生厅]]医学教育科、医学预防科副科长，福建卫生干校校长、党总支书记，中共福建省委政策研究室干部，[[福建医学院|福建医学院]]马列主义教研室主任、党委办公室主任，[[福建医科大学|福建医科大学]]教务处处长，中国援[[塞内加尔|塞内加尔]]医疗队队长，福建医科大学党委副书记、副校长等职。1981年8月调到中共福建省委机关工作，历任中共福建省委组织部副部长，中共福建省委常委、组织部部长，中共福建省委常委、中共福建省纪委书记，福建省政协副主席等职。1998年12月离职休养<ref name=shishi>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1834816 |title =福建省委组织部原部长、省纪委原书记高胡逝世，享年91岁  |author =  |date = 2017-10-23 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

2017年10月16日，高胡在福州病逝，享年91岁。10月22日，高胡同志遗体告别仪式在福州举行<ref name=shishi/>。
