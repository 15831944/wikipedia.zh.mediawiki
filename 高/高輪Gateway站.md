'''高輪Gateway站'''（{{jpn|j=高輪ゲートウェイ駅|hg=たかなわゲートウェイえき|rm=Takanawa-Gētowei eki|f=y}}，華語圈又譯作「高輪入口」<ref name="cna">{{cite news|author=黃名璽|editor=蔡佳敏|title=JR東日本公布山手線新站名　拚東奧前啟用|url=https://www.cna.com.tw/news/aopl/201812040172.aspx|agency=[[中央通訊社|CNA 中央社]]|language=zh-tw|date=2018-12-04|location=東京|accessdate=2018-12-04|archive-url=https://web.archive.org/web/20181204142515/https://www.cna.com.tw/news/aopl/201812040172.aspx|archive-date=2018-12-04|dead-url=no}}</ref><ref name=Konwaku>{{cite news|author=|editor=|script-title=ja:「高輪ゲートウェイの中国語名はどう翻訳」　中国で困惑の声|language=ja|url=https://www.sankei.com/world/news/181205/wor1812050017-n1.html|agency=[[產經新聞社|產經新聞社]]|date=2018-12-05|accessdate=2018-12-05|archive-url=https://web.archive.org/web/20181205103820/https://www.sankei.com/world/news/181205/wor1812050017-n1.html|archive-date=2018-12-05|dead-url=no}}</ref>）是[[日本|日本]][[東京都|東京都]][[港區_(東京都)|港區]][[港南_(東京都港區)|港南]]地區一座正在建設的[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）車站，再開發計劃展開時曾暫名「'''品川新站'''」<ref name="JE"/>。車站預定於2020年啟用，屆時將供[[山手線|山手線]]與[[京濱東北線|京濱東北線]]使用。

== 歷史 ==
2014年6月3日，JR東日本宣布將在距田町車站約1.3公里，[[品川車站|品川車站]]約0.9公里處建設新車站，並計劃趕在[[2020年|2020年]][[2020年夏季奧林匹克運動會|東京奧運]]、[[2020年夏季帕拉林匹克運動會|帕運]]實現暫定開業。車站用地中有20公頃是取自{{lj|東京綜合車輛中心|東京総合車両センター}}田町中心用地（其中13公頃是通過使用舊田町車輛中心，以及重劃列車停留場所而獲得的[[再開發|再開發]]用地）<ref name="tk">{{Cite web|language=ja|script-title=ja:山手線新駅から読み解く品川再開発の行方|url=http://toyokeizai.net/articles/-/39267|date=2014-06-04|accessdate=2017-09-13|publisher=[[東洋經濟新報社|東洋經濟新報社]]|archive-url=https://web.archive.org/web/20170913135829/http://toyokeizai.net/articles/-/39267|archive-date=2017-09-13|dead-url=no}}</ref><ref>{{Cite web|url=https://www.jreast.co.jp/press/2014/20140604.pdf|language=ja|script-title=ja:田町〜品川駅間に新駅を設置し、まちづくりを進めます|format=PDF|date=2014-06-03|accessdate=2017-09-13|publisher=[[東日本旅客鐵道|東日本旅客鐵道株式會社]]|archive-url=https://web.archive.org/web/20160304003642/http://www.jreast.co.jp/press/2014/20140604.pdf|archive-date=2016-03-04|dead-url=no}}</ref>。新車站位於[[泉岳寺站|泉岳寺站]]東南約300米處<ref name="tk" />，是山手線自1971年[[西日暮里站|西日暮里站]]開業以來再次設站，也是京濱東北線自2000年開業[[埼玉新都心站|埼玉新都心站]]以來再次設立新站<ref name="tk" />。

2016年9月6日，JR東日本發布了該站的發展大綱，並將該站定位為品川再開發計劃「Global Gateway品川」的核心設施<ref name="JE">{{Cite web |date=2016-09-06 |url=https://www.jreast.co.jp/press/2016/20160903.pdf |language=ja |script-title=ja:品川開発プロジェクトにおける品川新駅（仮称）の概要について |format=PDF |publisher=東日本旅客鐵道株式會社 |accessdate=2017-09-13  |archive-url=https://web.archive.org/web/20160910005431/http://www.jreast.co.jp/press/2016/20160903.pdf |archive-date=2016-09-10 |dead-url=no }}</ref>。2017年2月10日，JR東日本開始了該車站的施工<ref name="sk">{{Cite web |date=2017-02-10 |url=http://www.sankei.com/life/news/170210/lif1702100024-n1.html |language=ja |script-title=ja:「品川新駅」起工式 山手線で49年ぶり30駅目 |work=[[產經新聞|產經NEWS]] |publisher=產經新聞社 |accessdate=2017-09-13  |archive-url=https://web.archive.org/web/20170913135704/http://www.sankei.com/life/news/170210/lif1702100024-n1.html |archive-date=2017-09-13 |dead-url=no }}</ref><ref name="as">{{Cite web |date=2017-02-11 |url=http://www.asahi.com/articles/ASK283TVJK28UTIL00N.html |language=ja|script-title=ja:品川新駅、20年開業へ起工式 周辺に高層ビル7棟計画 |work=電子版[[朝日新聞|朝日新聞]] |publisher=朝日新聞社 |accessdate=2017-09-13}}</ref>。

JR東日本在2018年6月5日至6月30日公開募集該站的站名<ref>{{Cite web |date=2018-06-05 |url=http://www.jreast.co.jp/shinagawa_shineki/ |language=ja |script-title=ja:新しい駅の駅名大募集 |publisher=[[東日本旅客鉄道|東日本旅客鉄道株式会社]] |accessdate=2018-06-05  |archive-url=https://web.archive.org/web/20180605083650/http://www.jreast.co.jp/shinagawa_shineki/ |archive-date=2018-06-05 |dead-url=no }}</ref>。這也是JR東日本首次公開募集車站名稱<ref>{{Cite web |date=2018-06-05 |url=http://www.jreast.co.jp/press/2018/20180604.pdf |language=ja |script-title=ja:品川新駅(仮称)の駅名を募集します。 |format=PDF |publisher=[[東日本旅客鉄道|東日本旅客鉄道株式会社]] |accessdate=2018-06-05  |archive-url=https://web.archive.org/web/20180605110844/http://www.jreast.co.jp/press/2018/20180604.pdf |archive-date=2018-06-05 |dead-url=no }}</ref>。2018年8月29日，JR東日本對媒體公開了該站的建設現場<ref>{{Cite news|script-title=ja:山手線新駅、工事現場公開|date=2018-08-29|url=https://www.yomiuri.co.jp/stream/?id=09476|accessdate=2018-08-29|language=ja|website=YOMIURI ONLINE（読売新聞）|agency=讀賣新聞}}</ref>。2018年12月4日，JR東日本宣布將該站名稱定為「高輪Gateway」，名稱是來源於「此地自古便有[[街道|街道]]通過，是古代[[江戶|江戶]]的[[玄關|玄關口]]」<ref name="jreast20181214">{{Cite web |date=2018-12-04 |url=http://www.jreast.co.jp/press/2018/20181201.pdf |language=ja |script-title=ja:田町～品川駅間の新駅の駅名決定について |format=PDF |publisher=東日本旅客鐵道株式會社 |accessdate=2018-12-04  |archive-url=https://web.archive.org/web/20181204081543/http://www.jreast.co.jp/press/2018/20181201.pdf |archive-date=2018-12-04 |dead-url=no }}</ref>。

JR東日本計劃在2020年春季暫定開通該站<ref name="JE" />，在2024年實現正式開業<ref name="JE" />。

== 停靠路線 ==
[[File:Lines_from_Tokyo_Station.svg|thumb]]
該站在線路名稱上屬於[[東海道本線|東海道本線]]，以運行系統而言該站將有京濱東北線和山手線兩系統的列車停靠。該站也是JR[[特定都区市内|特定都区市内]]制度內「[[特定都区市内#設定区域一覧|東京都区内]]」及「[[東京山手線内|東京山手線内]]」範圍車站。

== 車站結構 ==
新站站體主要為鋼骨結構的2面4線[[島式月台|島式月台]][[地面車站|地面車站]]<ref name="JE"/>，車站設計由曾經手[[新國立競技場|新國立競技場]]及[[澀谷站|澀谷站]]的建築師[[隈研吾|隈研吾]]操刀<ref name="JE" /><ref name="sk"/>，外觀以折紙作為構想<ref name="as"/>，是其一大特徵。車站燈光設計則由建築照明設計師[[面出薰|面出薰]]主理<ref name="jreast20181214"/><ref>{{Cite news|url=https://trafficnews.jp/post/82261|language=ja|script-title=ja:山手線の新駅「高輪ゲートウェイ」に名称決定　JR東日本|date=2018-12-04|accessdate=2018-12-04|newspaper=乗り物ニュース|publisher=mediavague|archive-url=https://web.archive.org/web/20181204111016/https://trafficnews.jp/post/82261|archive-date=2018-12-04|dead-url=no}}</ref>。車站地上3層、地下1層，樓高約30公尺<ref name="sk" />；月台設於地面一樓，剪票口和出入口及活動廣場則是設於二樓，二樓和三樓有店鋪進駐。

== 車站週邊 ==
車站土地原為JR品川車輛基地，佔地約{{convert|13|ha|km2}}，JR東日本與[[都市再生機構|都市再生機構]]（UR）合作進行約5,000億日圓再開發計畫<ref>{{Cite web|url=https://style.nikkei.com/article/DGXMZO12950230V10C17A2000000|language=ja|script-title=ja:品川新駅再開発に着手 JR東、民営化30年目の挑戦|date=2017-03-21|accessdate=2017-09-13|publisher=日經BP社|work=[[日本經濟新聞|NIKKEI STYLE]]|archive-url=https://web.archive.org/web/20170913135755/https://style.nikkei.com/article/DGXMZO12950230V10C17A2000000|archive-date=2017-09-13|dead-url=no}}</ref>，山手線和京濱東北線軌道將向東移設最大約120公尺，騰出的土地用將興建辦公、住宿、零售等複合商業設施共7棟高層大樓<ref name="as" />。

再開發第一期計画涵蓋東京都港区港南二丁目、芝浦四丁目、高輪二丁目及三田三丁目各地，区域總面積約{{convert|9.5|ha|km2}}。以田町起至車站周圍的街区畫分為四大區，預定工期為2019年度至2024年度共6年。<ref>{{Cite web |date=2018-09-25 |url=https://www.jreast.co.jp/press/2018/20180923.pdf |language=ja |script-title=ja:品川開発プロジェクト（第Ⅰ期）に係る都市計画について |format=PDF |publisher=東日本旅客鐵道株式会社 |accessdate=2018-09-25  |archive-url=https://web.archive.org/web/20181120221109/https://www.jreast.co.jp/press/2018/20180923.pdf |archive-date=2018-11-20 |dead-url=no }}</ref>。

車站預定地直線距離約300米即[[泉岳寺站|泉岳寺站]]。沿該站西側的国道15号線而行，可前往過去江戶往東海道的門關{{lj|高輪大木戶遺跡|高輪大木戸跡|高輪大木戶}}。

== 車站命名 ==
根據日本中學生娛樂門戶網站ing在[[澀谷|澀谷]]訪問100位女高中生的「JR山手線第30個車站站名調查」調查結果顯示，排名第一的是「新品川站」，其次是「SPACE STATION」<ref>{{Cite web|url=http://www.sankei.com/economy/news/170627/prl1706270081-n1.html|language=ja|script-title=ja: ”流行の最先端”女子高生が選んだJR山手線30番目の新駅の名称2位は「SPACE STATION」、1位は…？|publisher=產經新聞|accessdate=2017-06-27|date=2017-06-27|archive-url=https://web.archive.org/web/20171023083015/http://www.sankei.com/economy/news/170627/prl1706270081-n1.html|archive-date=2017-10-23|dead-url=no}}</ref>。另外，根據JTown Net的的網路問卷調查結果，739票當中得票第一為「[[高輪|高輪]]」、次名為「[[芝浦|芝浦]]」、第三為「[[泉岳寺|泉岳寺]]」<ref>{{Cite news|url=https://j-town.net/tokyo/research/results/233251.html?p=all|language=ja|script-title=ja:山手線の新駅、名前は何がいい？→4位「新品川」 3位「泉岳寺」 2位「芝浦」 1位は...|newspaper=Jタウンネット 東京都|accessdate=2016-10-07|date=2018-11-16|archive-url=https://web.archive.org/web/20161011145919/http://j-town.net/tokyo/research/results/233251.html?p=all|archive-date=2016-10-11|dead-url=no}}</ref>。

JR東日本舉辦的徵名活動64,052件投稿中，「高輪」為第1位、「芝浦」為第2位、「{{link-ja|芝濱|芝浜}}」為第3位<ref name="jreast20181214"/>，而遴選委員会決議採用的「高輪Gateway」則僅有36人推薦，排名第130位<ref>{{Cite news|url=https://www3.nhk.or.jp/shutoken-news/20181204/0022141.html|language=ja|script-title=ja:山手線新駅「高輪ゲートウェイ」|newspaper=首都圏 NEWS WEB|publisher=[[日本放送協会|NHK]]|accessdate=2018-12-04|date=2018-12-04|archive-url=https://web.archive.org/web/20181204081209/https://www3.nhk.or.jp/shutoken-news/20181204/0022141.html|archive-date=2018-12-04|dead-url=no}}</ref>。JR東日本表示雖採用站名排第130位，但遴選委員会認為新站以建立國際交流的橋頭堡為目標<ref name="jreast20181214"/>，加上該站所在地區過往是通往江戶的玄關等因素<ref name="jreast20181214"/>，最後通過選擇使用帶有「Gateway」一詞的「高輪Gateway」，新站站名公布後各界褒貶兩極<ref>{{Cite news|url=https://www.sankei.com/life/news/181204/lif1812040017-n1.html|language=ja|script-title=ja:新駅「高輪ゲートウェイ」片仮名駅名に賛否　利用者の評価さまざま|agency=產經新聞|date=2018-12-04|accessdate=2018-12-05|archive-url=https://web.archive.org/web/20181205104142/https://www.sankei.com/life/news/181204/lif1812040017-n1.html|archive-date=2018-12-05|dead-url=no}}</ref><ref>{{Cite news|url=https://www.sponichi.co.jp/society/news/2018/12/05/kiji/20181204s00042000586000c.html|language=ja|script-title=ja:山手線新駅名「高輪ゲートウェイ」に決定　“カタカナ単語入り”に賛否|newspaper=Sponichi Annex|date=2018-12-05|accessdate=2018-12-05|archive-url=https://web.archive.org/web/20181205022152/https://www.sponichi.co.jp/society/news/2018/12/05/kiji/20181204s00042000586000c.html|archive-date=2018-12-05|dead-url=no}}</ref>。

在中文使用者中，因Gateway同時也是IT用語「[[網關|網關]]」<ref name=appear>此為中國大陸用語，即港澳台所講的閘道器。</ref>，因此有網友將該站翻譯為「高輪網關站」<ref name=Konwaku/>。
