'''高秀敏'''（{{bd|1959年|1月28日|2005年|8月18日|catIdx=Gao,高}}），[[吉林省|吉林省]][[松原市_(吉林省)|松原市]][[扶余市|扶余市]]人，国家一级演员，著名[[小品|小品]][[演员|演员]]。

1982年进入扶余县民间艺术团做演员。1992年辞职做个体演员。她与[[赵本山|赵本山]]、[[范伟|范伟]]合作演出了多个小品、[[东北二人转|东北二人转]]、[[电视剧|电视剧]]等，至今仍经久不衰。2005年8月18日凌晨3时左右，[[心肌梗塞|心肌梗塞]]导致窒息，在[[长春|长春]]去世。<ref>{{cite web |url=http://yule.sohu.com/20050818/n226717682.shtml |title= 医生初步鉴定 高秀敏死于心肌梗塞导致窒息|accessdate=2005-08-18 |work=|publisher=搜狐 |language= zh-hans}}</ref>

== 婚姻 ==
高秀敏生前有过一次婚姻，育有一女。另外，她与[[何庆魁|何庆魁]]的一段感情则是广为人知。她的女儿在接受《新京报》采访时<ref>{{cite web|url=http://news.163.com/05/1221/08/25G1I33E0001122D.html|title=高秀敏女儿丹丹回应父母情感话题：继父"够份"了|publisher=网易社会新闻|date=2005-12-21|accessdate=2010-02-03 |language=Zh-hans}}</ref>提及，在《[[艺术人生|艺术人生]]》录制节目时高秀敏曾经说过，她和何庆魁是先有了感情后来才离开了各自的家庭，是互为第三者。

高秀敏生前与何庆魁生活近十三年<ref>{{cite web|url=http://news.163.com/05/1219/09/25AV2OBM0001122D.html|title=媒体披露何庆魁婚姻状况 "前妻"早已原谅高秀敏|publisher=网易社会新闻|date=2005-12-19|accessdate=2010-02-03 |language=Zh-hans}}</ref>，在事业上被认识是夫妻搭挡<ref>{{cite web|url=http://gzdaily.dayoo.com/gb/content/2005-04/01/content_1997556.htm|title=何庆魁高秀敏夫妻档亮相《圣水湖畔》|publisher=大洋网大洋新闻|date=2005年4月1日|accessdate=2010-02-03|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20070709113253/http://gzdaily.dayoo.com/gb/content/2005-04/01/content_1997556.htm|archivedate=2007年7月9日}}</ref>，但实际上两人未正式登记<ref>{{cite web|url=http://news.163.com/05/1225/10/25QH0L1A0001122D.html|title=何庆魁高秀敏涉嫌重婚罪 根据刑法最高可判刑2年|publisher=网易社会新闻|date=2005-12-25 |accessdate=2010-02-03 |language=Zh-hans}}</ref>，仅是以夫妻名义对外进行演义事业。同时何庆魁与妻子张艳茹亦未离婚。

高秀敏和[[赵本山|赵本山]]、[[范伟|范伟]]被誉为“铁三角”。<ref>{{cite web |url=http://yule.sohu.com/20050819/n226720543.shtml |title= 高秀敏突然病逝 留下三大遗憾|accessdate=2005-08-19 |work=|publisher=搜狐 |language= zh-hans}}</ref>

[[陶泽如|陶泽如]]：“她私底下是一个比较热情的人，对人特别好。”<ref name="msj " />

[[朱时茂|朱时茂]]：“她是一个很认真的人，每到演出，总能见她提前赶到做准备。她没有演过话剧，却能做到这一点，提前10分钟脑子里像过电影一样。作为一个老演员，不是很多人能做得到的。这一点值得我们学习。她的形象和观众缘特别好，演小品很认真，做人比较成功。虽然她和我们同龄，但我们都称她为老师。”<ref name="msj " />

[[朱军|朱军]]：“高秀敏老师真的是戏如其人，她的表演非常生活化，她的生活和她的作品一样，非常简单，她为人大大咧咧，性格非常爽朗。怎么也没有想到会走得这么急。”，“最后一次见到高秀敏老师是在2个月前北京的一次活动上吧，她在台上非常精彩地唱了一段二人转，台下的观众非常喜欢。”，“那次我很感动，他们是由两个离异家庭组织在一起的，但是他们非常恩爱。高秀敏找何庆魁写小品，然后两人一起努力创业。当时还讲到了何庆魁对高秀敏的帮助，也讲到了高秀敏对何庆魁的照顾，很是恩爱。”<ref name="msj " />
