'''高建国'''（{{bd|1954年|10月|}}），[[山东省|山东省]][[青岛市|青岛市]][[黄岛区|黄岛区]]人，[[中国人民解放军中将|中国人民解放军中将]]。[[中国共产党第十八届中央委员会候补委员列表|中国共产党第十八届中央委员会候补委员]]。

==生平==
[[中国共产党|中国共产党]]党员，1999年毕业于[[中国人民解放军国防大学|中国人民解放军国防大学]]，军事学博士学位。1972年参军，曾任宣传干事、军区报社记者、军区党委秘书、军区宣传部宣传处长、军区组织部副部长等职<ref name=elyq>{{Cite web |url = http://www.chinawriter.com.cn/zxhy/member/8220.shtml |title =高建国 (1954～) |author =  |date =  |publisher =中国作家网 |language =  |accessdate = 2017-04-29 |quote =  }}</ref>。

2000年任[[中国人民解放军陆军第五十四集团军|第五十四集团军]]一六二师政委，2003年任[[中国人民解放军陆军第二十集团军|第二十集团军]]政治部主任，2006年任第二十集团军副政委，2006年任第二十集团军政委，2008年任第五十四集团军政委，2010年任[[沈阳军区|沈阳军区]]政治部主任，2013年12月任沈阳军区副政委<ref>{{Cite web |url =  http://news.163.com/14/0218/02/9LB66M1500014AED.html|title = 多军区高层调整 33位高级将领履新 |author =  |date = 2014-02-18 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref name=ifeng/><ref>{{cite news|title=沈阳军区政治部主任高建国中将 对松原军分区工作给予高度评价|url=http://syrbszb.chinajilin.com.cn/html/2012-02/22/content_2494521.htm|accessdate=2012-05-21|newspaper=中国吉林网|deadurl=yes|archiveurl=https://web.archive.org/web/20141025010208/http://syrbszb.chinajilin.com.cn/html/2012-02/22/content_2494521.htm|archivedate=2014-10-25}}</ref>。2016年1月，出任[[济南军区|济南军区]]善后工作办公室政委、党委书记<ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1422013|title=吉文明出任济南军区善后工作办公室主任，高建国任政委|publisher=澎湃新闻|accessdate=2016-03-06}}</ref><ref name=ifeng/>。

2004年7月晋升为[[中国人民解放军少将|中国人民解放军少将]]军衔。2011年晋升为[[中国人民解放军中将|中国人民解放军中将]]军衔<ref name=ifeng>{{Cite web|url =http://renwuku.news.ifeng.com/index/detail/322/gaojianguo|title =高建国|author =|date =|publisher =凤凰网|language =|accessdate =2017-04-29|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20160527010441/http://renwuku.news.ifeng.com/index/detail/322/gaojianguo|archivedate =2016-05-27}}</ref>。

==著作==
1976年开始发表作品。1993年加入[[中国作家协会|中国作家协会]]以及中国报告文学协会。主要作品有：
*《当代中国军人婚姻透视》
*《地球第一雕塑》
*《东方写真录》
*《本世纪无大战》（中篇报告文学，获1988年“中国潮”征文二等奖）
*《二十一世纪中国军事人才发展战略构想》（博士学位论文，被评为2001年全国优秀博士论文）
*《改革开放与当代中国军人形象塑造》（论文，被分别评为全国和全军纪念党的十一届三中全会召开30周年优秀论文）<ref name=elyq/>
