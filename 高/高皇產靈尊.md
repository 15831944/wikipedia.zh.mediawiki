'''高皇產靈尊'''（{{lang|ja|たかみむすひ の みこと}}）為《[[日本書紀|日本書紀]]》裡的記述，《[[古事記|古事記]]》則記作高御產巢日神（{{lang|ja|たかみむすひ の かみ}}），祂是[[日本神話|日本神話]]裡的[[別天神|別天神]]，別名作'''高木神'''（{{lang|ja|たかき の かみ}}）。不過在《[[延喜式|延喜式]]》卷第八神祇八祝詞的出雲國造神賀詞篇中，卻寫成'''神王高御魂命'''。

高皇產靈尊的[[兒子|兒子]]為[[思兼神|思兼神]]（{{lang|ja|おもひかね の かみ}}），[[女兒|女兒]]為[[栲幡千千姬命|栲幡千千姬命]]（{{lang|ja|たくはたちぢひめ の みこと}}）。

== 概要 ==
以下為各典籍記載之別天神・[[神世七代|神世七代]]神名對照表：

{{神世七代對照表}}

依照《[[記紀|記紀]]》二書之記載，[[天照大神|天照大神]]之子[[正哉吾勝勝速日天忍穗耳尊|正哉吾勝勝速日天忍穗耳尊]]娶了'''高皇產靈尊'''的女兒[[栲幡千千姬命|栲幡千千姬命]]，生下[[天津彥彥火瓊瓊杵尊|天津彥彥火瓊瓊杵尊]]，所以高皇產靈尊算是天津彥彥火瓊瓊杵尊的[[祖父母|外公]]。

《[[日本書紀|日本書紀]]》卷第二神代之記載，高皇產靈尊賜天鹿兒弓（{{lang|ja|あめのかごゆみ}}）及[[天羽羽矢|天羽羽矢]]給[[天稚彥|天稚彥]]，派遣祂下凡平定[[葦原中國|葦原中國]]之邪鬼。但祂甫抵葦原中國即娶[[下照姬|下照姬]]為妻，因為久未回報結果，高皇產靈尊派雉鳥前往一探究竟。想不到天稚彥取出天鹿兒弓及天羽羽矢射死雉鳥，箭矢直達天上落在高皇產靈尊面前。祂見到箭矢染血，以為天稚彥正與邪鬼激戰，遂投還天羽羽矢。不料天稚彥正臥床休息，胸膛中矢身亡。

《[[古事記|古事記]]》卷第二[[神武東征|神武東征]]的故事中，[[高倉下|高倉下]]夢見[[天照大神|天照大神]]及'''高木神'''對[[建御雷|建御雷神]]說：「你不是平定[[葦原中國|葦原中國]]了嗎？為何讓我等之子孫遭受紛擾呢？還不趕快下凡拯救他們！」建御雷神卻回覆：「輪不到我沁自下凡，只要一把能平定戰火的橫刀就可以了。」所以高倉下向[[神武天皇|神武天皇]]獻上橫刀，隨即斬殺熊野山的凶神，受大熊之毒倒地的神武軍士卒亦隨著起身醒來。

== 解說 ==
以其別名「高木神」而言，乃是將「高木」神格化；而「產靈（{{lang|ja|むすひ}}）」則含有生產、生成之意義。[[日本神話|日本神話]]裡提到[[日本神話#天地開闢時期|開天闢地]]的[[別天神|別天神]]，最初出現的乃[[天御中主尊|天御中主尊]]（{{lang|ja|あまのみなかぬし の みこと}}），代表[[宇宙|宇宙]]的根本、支配[[高天原|高天原]]中心的主宰；其次出現的是'''高皇產靈尊'''和[[神皇產靈尊|神皇產靈尊]]（{{lang|ja|かむみむすひ の みこと}}），此二神相對為[[陰陽|陰陽]]兩儀。雖然[[獨神|獨神]]應該沒有[[性別|性別]]可言，不過一般認為'''高皇產靈神'''是[[男性|男性]]，而[[神皇產靈尊|神皇產靈尊]]是[[女性|女性]]。

高皇產靈尊和神皇產靈尊供奉在{{link-ja|八神殿|八神殿}}裡，每逢[[日本皇室|日本皇室]]重大祭典活動時由[[神祇官|神祇官]]負責祭祀。
