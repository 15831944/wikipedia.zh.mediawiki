'''高樹勛'''（{{bd|1897年||1972年|1月19日|catIdx=Gao高}}），字'''建侯'''，[[直隶省|直隶省]][[天津府|天津府]][[鹽山县|鹽山县]]人。中華民國国軍中将。<ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>

==生平==
1915年（[[民国紀元|民国]]4年），高树勋成为[[北京政府|北京政府]]方面的軍人，后来在[[馮玉祥|馮玉祥]]领导的部隊中逐步升迁。1926年（民国15年），升任[[国民革命軍第二集团軍|国民革命軍第二集团軍]]的師長。1928年（民国17年），率軍入[[青海省_(中華民国)|青海]]，降伏了支配青海的[[馬麒|馬麒]]。後来，一度代理青海省政府主席。<ref name=xu/><ref>魏明章「馬麒」{{Cite book<!--|和書-->|author = 中国社会科学院近代史研究所|title = 民国人物传 第12卷|year = 2005|publisher = 中華書局|isbn = 7-101-02993-0}}</ref>

1930年（民国19年）[[中原大战|中原大战]]中，高树勋同[[蒋介石|蒋介石]]軍作战。敗北後，高树勋投降蒋介石，被任命为第26路軍第27師師長。1931年（民国20年），在[[江西省_(中華民国)|江西省]]参加对[[中国工农红军|中国工农红军]]的[[第三次江西剿共戰爭|第三次围剿]]中，突然脱离部队，逃往[[天津市|天津]]。以後，高树勋同[[中国共产党|中国共产党]]秘密联系。1933年（民国22年），高树勋参加了馮玉祥在[[察哈爾省|察哈爾]]組織[[察哈爾民众抗日同盟軍|察哈爾民众抗日同盟軍]]，任察哈爾民众抗日同盟軍第2挺進軍司令。<ref name=xu/>

1937年（民国26年）[[抗日战争|抗日战争]]全面爆发后，高树勋任[[河北省_(中華民国)|河北省]]保安处处長、河北游击总指揮。1939年（民国28年）2月，任察哈爾省政府委員。1940年（民国29年）12月，由于[[石友三|石友三]]企图投降日本，高树勋奉蒋介石的秘密命令，逮捕并活埋了[[石友三|石友三]]。1941年（民国30年）1月，升任第39集团軍副总司令。同年5月，兼任新編第8軍軍長<ref>徐友春主編『民国人物大辞典 増訂版』作此。沈慶生「石友三」作1940年12月处决石友三時，高树勋就任新編第八軍軍長。</ref>。其後，历任第39集团軍总司令、冀察战区总司令、[[抗日战争第十一战区|第11战区]]副司令長官。<ref name=xu/><ref>沈慶生「石友三」{{Cite book<!--|和書-->|author = [[中国社会科学院|中国社会科学院]]近代史研究所|title = 民国人物传 第1卷|year = 1987|publisher = [[中華書局|中華書局]]|isbn = }}</ref>
[[File:Gao_Shuxun1.jpg|thumb]][[武安县|武安县]][[伯延镇|伯延镇]]举行建军大会。图为大会主席台。图中主席台上左一为高树勋。]]
抗日战争结束後的1945年（民国34年）10月30日，在[[邯郸战役|邯郸战役]]期间，高树勋率领新編第八軍、[[河北民军|河北民军]]1万多人于阵前在[[磁县|磁县]][[马头镇|马头镇]]宣布投向[[中国共产党|中国共产党]]方面。[[中共中央|中共中央]]对此加以赞扬，不久[[毛泽东|毛泽东]]还发动了“[[高树勋运动|高树勋运动]]”，以促使国民党军队起义、投诚、接受改编。1945年11月10日，高树勋部改编成的民主建国軍成立，高树勋被任命为民主建国軍总司令，[[王定南|王定南]]为总政治部主任，[[范龙章|范龙章]]为第一军军长，[[乔明礼|乔明礼]]为第二军军长。1945年11月13日，经[[中共中央书记处|中共中央书记处]]批准，由[[邓小平|邓小平]]、[[薄一波|薄一波]]介绍，高树勋正式加入[[中国共产党|中国共产党]]。<ref name=gcd>[http://dangshi.people.com.cn/GB/85039/11789663.html 庾莉萍，高树勋：起义第一将军，中国共产党新闻网，2010年06月04日]</ref><ref name=xu/><ref>[http://dangshi.people.com.cn/GB/85039/10786278.html 毕铭，揭秘：“高树勋运动”兴起前后，中国共产党新闻网，2010年01月18日]</ref>

民主建国軍成立后，[[晋冀鲁豫军区|晋冀鲁豫军区]]派出一批政治工作干部到民主建国军开展政治工作，但这批干部同民主建国军将领相处并不愉快。1947年6月14日，传出所谓民主建国军“阴谋暴动”。1947年6月15日，高树勋因民主建国军“阴谋暴动”而被逮捕审查，民主建国军番号被撤销。审查一年后，未发现证据，高树勋旋即被任命为[[华北军区|华北军区]]副司令员。后来，直到[[中共十一届三中全会|中共十一届三中全会]]以后，所谓民主建国军“阴谋暴动”的冤案才最终平反。<ref>刘刚范主编，高树勋将军，团结出版社，1995年</ref><ref name=gcd/>
{{Wide image|Gao Shuxun2.jpg|600px|1946年2月，[[邓小平|邓小平]]（左一）、[[刘伯承|刘伯承]]（左四）、[[薄一波|薄一波]]（左七）、[[杨秀峰|杨秀峰]]（左二）、高树勋（左五）与[[邯郸战役|邯郸战役]]中被俘的国军第11战区副司令长官[[马法五|马法五]]（左三，着长衫者）合影。随后马法五被送回[[新乡|新乡]]。}}
[[中華人民共和国|中華人民共和国]]成立後，高树勋历任[[河北省人民政府|河北省人民政府]]副主席、[[河北省|河北省]]副省長、[[全国人民代表大会|全国人民代表大会]]代表、[[中华人民共和国国防委員会|中华人民共和国国防委員会]]委員、[[中国人民政治協商会議|中国人民政治協商会議]]全国委員会委員。<ref name=xu/>

1972年1月19日，高树勋在[[北京市|北京]]逝世。享年76岁。<ref name=xu/>
