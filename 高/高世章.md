'''高世章'''（[[:en:Leon_Ko|Leon Ko]]），[[音樂劇|音樂劇]]及電影作曲家。曾獲「Richard Rodgers Development Award」、[[金馬獎|金馬獎]]及多個音樂獎項。作品曾搬上紐約卡內基音樂廳的舞台，並為美國公眾廣播電視台節目《The Puzzle Place》創作音樂。母親是兩屆「亞洲影后」及第一屆「金馬影后」[[尤敏|尤敏]]，外祖父是[[粵劇|粵劇]]名伶[[白玉堂_(粤剧演员)|白玉堂]]，而「曼波女郎」[[葛蘭|葛蘭]]則是他的伯娘。高氏於[[美國|美國]][[西北大學|西北大學]]修讀音樂，及後到[[紐約大學|紐約大學]]專攻音樂劇作曲。

==創作年表==
2001年，憑音樂劇《Heading East》獲「Richard Rodgers Development Award」。<ref>{{cite web |url=http://www.artsandletters.org/awards2_popup.php?abbrev= |title=存档副本 |accessdate=2009-11-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20110617061058/http://www.artsandletters.org/awards2_popup.php?abbrev= |archivedate=2011-06-17 }}</ref>

2002年，為[[中英劇團|中英劇團]]話劇《咖喱盆菜釀薯條》譜寫音樂。

2003年，為[[演戲家族|演戲家族]]粵語音樂劇《[[四川好人|四川好人]]》作曲、編曲及擔任音樂總監，並憑該劇獲[[香港戲劇協會|香港戲劇協會]]舉辦的第十三屆[[香港舞台劇獎|香港舞台劇獎]]「最佳創作音樂」，該劇其後兩度公演。 

2004年，為[[張學友|張學友]]創意音樂劇《[[雪狼湖|雪狼湖]]》國語版擔任音樂總監。

2005年，與[[金培達|金培達]]合作，為電影《[[如果·愛|如果·愛]]》作曲及配樂，並憑該片獲[[2006年香港電影金像獎|第二十五屆香港電影金像獎]]「最佳原創電影音樂」、第五十一屆[[亞太影展|亞太影展]]「最佳音樂獎」及第十一屆[[香港電影金紫荊獎|香港電影金紫荊獎]]「最佳音樂獎」。憑《十字街頭》獲[[CASH_金帆音樂獎|CASH 金帆音樂獎]]「最佳另類作品」及[[第43屆金馬獎|第四十三屆台灣金馬獎]]「最佳原創電影歌曲」。

2006年，為[[香港藝術節|香港藝術節]]及[[演戲家族|演戲家族]]粵語音樂劇《[[:en:Legend_of_the_White_Snake|白蛇新傳]]》作曲、編曲及擔任音樂總監，並憑該劇獲[[香港戲劇協會|香港戲劇協會]]舉辦的第十六屆[[香港舞台劇獎|香港舞台劇獎]]「最佳創作音樂」。同年，為[[任白慈善基金會|任白慈善基金會]]主辦，[[雛鳳鳴劇團|雛鳳鳴劇團]]演出之粵劇《[[帝女花|帝女花]]》譜寫新序曲及過場音樂。

2007年，為《[[學友光年世界巡迴演唱會|學友光年世界巡迴演唱會]]》譜寫音樂劇環節及為[[香港舞蹈團|香港舞蹈團]]歌舞劇《夢傳說》作曲、編曲及擔任音樂總監。同年，為紀念香港回歸十週年的電影《[[老港正傳|老港正傳]]》配樂。與[[陳光榮|陳光榮]]、[[金培達|金培達]]及[[Chatchai_Pongprapaphan|Chatchai Pongprapaphan]]合作，為電影《[[投名狀|投名狀]]》配樂，並憑該片獲提名[[2008年香港電影金像獎|第二十七屆香港電影金像獎]]「最佳原創電影音樂」及[[第45屆金馬獎|第四十五屆台灣金馬獎]]「最佳原創電影音樂」。

2008年，為[[香港話劇團|香港話劇團]]原創音樂劇《頂頭鎚》作曲、編曲及擔任音樂總監，並憑該劇獲[[香港戲劇協會|香港戲劇協會]]舉辦的第十八屆[[香港舞台劇獎|香港舞台劇獎]]「最佳創作音樂」，該劇其後在香港及北京再度公演。同年，其改編自美國電影《[[:en:Please_Don't_Eat_the_Daisies_(film)|Please Don't Eat The Daisies]]》的百老匯音樂劇在紐約預演。<ref>http://www.playbill.com/news/article/122473-Edelman_Fowler_Korbich_and_Ward_Set_for_Don't_Eat_the_Daisies_Workshop{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2009年，為[[汪明荃|汪明荃]]歌舞劇《真係阿姐》譜寫新曲。為[[演戲家族|演戲家族]]粵語音樂劇《[[一屋寶貝|一屋寶貝]]》(改編自日本小說《[[在雨和夢之後|在雨和夢之後]]》) 作曲、編曲及擔任音樂總監/樂隊領班，慶祝[[香港文化中心|香港文化中心]]二十周年誌慶，並憑該劇獲[[香港戲劇協會|香港戲劇協會]]舉辦的第十九屆[[香港舞台劇獎|香港舞台劇獎]]「最佳創作音樂」，該劇其後四度公演及以音樂劇演唱會形式和音樂廳版演出。

2010年，為[[香港藝術節|香港藝術節]]及「[[毛俊輝|毛俊輝]]戲劇計劃」聯合製作的原創越界音樂劇場《情話紫釵》擔任音樂總監，並在北京、深圳及上海世界博覽會演出。其音樂劇《Heading East》以演唱會形式在紐約上演。<ref>{{cite web |url=http://www.playbill.com/news/article/139267-Heading-East-Concerts-to-Feature-BD-Wong-and-Manu-Narayan |title=存档副本 |accessdate=2010-05-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20121015075417/http://www.playbill.com/news/article/139267-Heading-East-Concerts-to-Feature-BD-Wong-and-Manu-Narayan |archivedate=2012-10-15 }}</ref> 此外，於[[國際金融中心_(香港)|國際金融中心]]商場舉辦《尋香記》《Time In A Bottle》古董香水瓶展覽，把珍藏的香水瓶歸納起來，想出十三個場景，用香水瓶充當角色去訴說一對戀人的傳說，將古董香水瓶與戲劇兩種藝術共冶一爐。同年，為國際小提琴家[[姚珏|姚珏]]《All About Love全為愛》聖誕演奏會擔任音樂顧問。

2011年，為倫敦[[:en:Theatre_Royal_Stratford_East|Theatre Royal Stratford East]]原創音樂劇[https://web.archive.org/web/20110511191942/http://www.stratfordeast.com/whats_on/takeaway.shtml 《Takeaway》]作曲。同年，為電影《[[大魔術師|大魔術師]]》配樂及譜寫主题曲《猜情人》。

2012年，憑音樂劇《[[一屋寶貝|一屋寶貝]]》獲[[香港戲劇協會|香港戲劇協會]]舉辦的第二十一屆[[香港舞台劇獎|香港舞台劇獎]]「最佳原創曲詞」。為[[陳寶珠|陳寶珠]]及國家一級演員[[蔣文端|蔣文端]]演出之粵劇《[[紅樓夢|紅樓夢]]》譜寫新序曲音樂。同年，為電影《[[大上海|大上海]]》譜寫主题曲《[[定風波_(歌曲)|定風波]]》，並憑該曲獲[[第32屆香港電影金像獎|第32屆香港電影金像獎]]「最佳原創電影歌曲」。

2013年，與[[演戲家族|演戲家族]]合作，為粵語音樂劇《穿Kenzo的女人》(改編自「[[鄧小宇|錢瑪莉]]」的小說《穿Kenzo的女人》) 作曲及編曲。

2014年，為[[任白慈善基金會|任白慈善基金會]]主辦之粵劇《[[再世紅梅記|再世紅梅記]]》譜寫過場音樂、為舞台劇《[[杜老誌|杜老誌]]》擔任音樂總監、為[[一舖清唱|一舖清唱]]之無伴奏合唱劇《大殉情》提供意念及作曲及為[[澳門文化中心|澳門文化中心]]十五周年誌慶之壓軸原創音樂劇《我要高八度》作曲及擔任音樂總監。同年，為電影《[[魔警|魔警]]》及《[[親愛的_(2014年電影)|親愛的]]》配樂。

2015年，為電影《[[暴瘋語|暴瘋語]]》配樂及譜寫主題曲《暴風雨》，並憑該曲獲提名[[第34屆香港電影金像獎|第34屆香港電影金像獎]]「最佳原創電影歌曲」。同年，為電影《[[捉妖記_(2015年電影)|捉妖記]]》配樂及創作四首歌曲。

2016年，為電影《脱皮爸爸》配樂。

2017年，憑無伴奏合唱劇《大殉情》(重演)獲[[香港戲劇協會|香港戲劇協會]]舉辦的第二十六屆[[香港舞台劇獎|香港舞台劇獎]]「最佳原創曲詞」。同年，為[[任白慈善基金會|任白慈善基金會]]主辦之粵劇《[[蝶影紅梨記|蝶影紅梨記]]》譜寫新序曲及過場音樂。與[[利希慎基金|利希慎基金]]合作，為粵語音樂劇《奮青樂與路Sing Out》 作曲、編曲及擔任音樂總監，並憑該劇獲[[香港戲劇協會|香港戲劇協會]]舉辦的第二十七屆[[香港舞台劇獎|香港舞台劇獎]]「最佳原創曲詞」及「最佳配樂」。

2018年，為電影《[[捉妖記2|捉妖記2]]》配樂。同年，聯同[[香港小交響樂團|香港小交響樂團]]舉辦《高世章的神奇電影畫布》，將其創作電影樂章成為段段交響樂章。

==表演==
2007年，為[[香港舞蹈聯盟|香港舞蹈聯盟]]主辦的「[[香港舞蹈年獎|香港舞蹈年獎]]」頒獎禮擔任表演嘉賓。

2009年，為慶祝[[香港戲劇協會|香港戲劇協會]]成立二十五週年，高氏在「第十八屆[[香港舞台劇獎|香港舞台劇獎]]頒獎禮」與林澤群、劉榮豐、姚詠芝聯合演出[http://www.leonko.net/news/view.php?ID=70&CID=1 其過往創作的音樂劇選段]。同年，與小提琴演奏家[[姚珏|姚珏]]合作，在香港同胞慶祝中華人民共和國成立六十週年文藝晚會壓軸環節《[http://www.leonko.net/news/view.php?ID=66&CID=1 星空閃耀照未來]》為[[香港流行音樂|香港流行音樂]]巨星[[張學友|張學友]]及北京奧運會開幕式《歌唱祖國》的原聲[[楊沛宜|楊沛宜]]伴奏，演出由[[中華人民共和國總理|中華人民共和國總理]][[溫家寶|溫家寶]]的詩作《仰望星空》改編的歌曲及《愛是永恆》。

2010年起，為「[[邵逸夫獎|邵逸夫獎]]頒獎禮」擔任司儀。<ref>http://img.tvb.com/p/shawprize.org/pdf/Brochure_2010.pdf http://img.tvb.com/shawprize/upload/en/20110928-booklet.pdf http://img.tvb.com/shawprize/upload/en/20120917-booklet.pdf {{cite web |url=http://mpaper.org/Story.aspx?ID=39664 |title=存档副本 |accessdate=2011-10-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20131021052724/http://mpaper.org/Story.aspx?ID=39664 |archivedate=2013-10-21 }} http://img.tvb.com/shawprize/upload/en/Booklet_2013.pdf http://img.tvb.com/shawprize/upload/en/apf9tmj5ekkr.pdf</ref>

2016年，獲政府委任為[[香港演藝學院|香港演藝學院]]校董會成員。
