'''高铭暄'''{{BD|1928年|5月24日}}，[[浙江省|浙江省]][[玉环市|玉环市]]人，中国法学家和法学教育家，现任中国人民大学法学院荣誉一级教授，博士研究生导师，北京师范大学刑事法律科学研究院名誉院长、博士研究生导师。<ref>{{Cite web|url = http://www.criminallawbnu.cn/criminal/Info/ShowLS.asp?programid=580&lawstar=%B8%DF%C3%FA%EA%D1&pkID=83|title = 高铭暄教授|accessdate = 2015-07-13|author = |date = |publisher = }}</ref>1951年毕业于北京大学法律系，后任教于中国人民大学法学院，是中国大陆第一位刑法学博士生导师。与马克昌教授一起为中国刑法学与世界刑法学，尤其是日本刑法学的交流作出了重大的贡献。<ref>{{Cite book|title=我的刑法研究|last=西原春夫|first=|publisher=北京大学出版社|year=2016|isbn=|location=|pages=}}</ref>

==生平==
长期参与《[[中华人民共和国刑法|中华人民共和国刑法]]》的制定和修改，<ref>{{Cite news|url = http://www.npc.gov.cn/npc/zgrdzz/2012-06/05/content_1724103.htm|title = 高铭暄：通过刑法学实现的人生价值|last = |first = |date = 2012-06-05|work = |accessdate = 2015-07-13|publisher = 中国人大网}}</ref>为中国的刑事司法从文化大革命时期的乱象走到今天作出了不可磨灭的贡献。高铭暄教授还撰写了《中华人民共和国刑法的孕育发展和发展完善》，为中国刑法立法史提供了珍贵的研究资料。

从苏联引进四要件犯罪构成理论。虽然四要件犯罪构成理论存在不足，相比德日三阶层犯罪论体系而言不够精细，但是当时中国废除了“旧法”，四要件为刑法学提供了重要的理论框架，且至今仍然为许多司法实务人员运用。

==著作==
与武汉大学的马克昌教授合作主编的《刑法学》教科书（高等教育出版社）至2017年已经出至第8版。作为中国曾经最为权威的刑法学教科书，高、马版《刑法学》几乎成为法学生人手一本的“红宝书”（因该书的封面为大红色而得名）。<ref>{{Cite book|title=我与刑法七十年|last=高铭暄|first=|publisher=北京大学出版社|year=2018|isbn=|location=|pages=}}</ref>

==争议==
曾经在沈阳刘涌案、嫖宿幼女罪存废问题中发表过一些争议性观点，并因此在网络上遭到中国大陆某些网民的批评。

==学生==
学生如陈兴良（北京大学教授）、黄京平（中国人民大学教授）、邱兴隆（湖南大学教授，2017年去世）、张智辉（最高人民检察院司改办主任）、张军（最高人民检察院检察长，首席大检察官）、姜伟（最高人民法院副院长）、周振想（2004年去世）、齐文远（中南财经政法大学教授）、赵秉志（北京师范大学教授）、李翔（华东政法大学教授）等人已成为中国大陆刑法实务界和学界的中流砥柱。高铭暄教授曾与武汉大学法学院的马克昌教授并称为中国刑法学界的“北高南马”。
