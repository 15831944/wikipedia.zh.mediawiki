'''社團法人高雄市關懷流浪動物協會'''（{{lang-en|Kaohsiung Concern Stray Animal Association}}）是一個位於[[台灣|台灣]][[高雄市|高雄市]]的[[動物保護|動物保護]]組織，由[[王小華|王小華]]創立於2000年3月31日，以關心街頭[[流浪動物|流浪動物]]生命權益為宗旨<ref>{{Cite web|url=http://www.kcsaa.org.tw/context_about.jsp?page=1&abt=1|title=協會宗旨|accessdate=2017-11-12|publisher=高雄市關懷流浪動物協會}}</ref>，在高雄市持續推行流浪貓狗救援，其會址設於高雄市[[鼓山區|鼓山區]]。

至2016年其所屬的保育場共收容了兩千隻以上的流浪犬貓<ref name=":4">{{Cite web|url=http://www.kcsaa.org.tw/%e5%8d%94%e6%9c%83%e7%8b%97%e8%88%8d/|title=協會狗舍|accessdate=2019-01-15|publisher=高雄市關懷流浪動物協會|language=zh-TW}}</ref>。

== 創會緣起 ==
王小華原名王春金，本在一家[[汽車旅館|汽車旅館]]擔任[[會計|會計]]，因下班時見到流浪狗會主動餵食，便進而開始沿路餵食多隻流浪狗、及將流浪狗帶回家養。之後王小華認識一批志同道合的愛心媽媽，決議設立一個動保組織。於是2000年3月31日王小華與16位創始會員向[[高雄市政府社會局|高雄市政府社會局]]申請登記，正式成立「高雄市關懷流浪動物協會」，性質為「[[公益性社團法人|公益性社團法人]]」，當時由她就任[[理事長|理事長]]<ref name=":1">{{Cite news|url=http://www.tanews.org.tw/info/4392|title=毛孩當兒女  王小華不惜搏命捍衛|newspaper=台灣動物新聞網|accessdate=2017-11-11|language=zh-hant}}</ref>。

== 歷史 ==
2005、2006年間，高雄市關懷流浪動物協會開始承接[[高雄市政府|高雄市政府]]的捕犬業務<ref name=":3" />。

2007年2月，高雄市關懷流浪動物協會數名志工前往救援在[[國道一號|國道一號]][[分隔島|分隔島]]生活了近兩年的一隻知名流浪黑狗「209」，埋伏在[[高速公路|高速公路]]兩側五個小時才成功以[[吹箭|吹箭]][[麻醉|麻醉]]「209」，完成救援任務，但遭[[高速公路局|高速公路局]]開罰<ref name=":1" /><ref name=":2">{{Cite news|url=https://www.mirrormedia.mg/story/20170110soc013/|title=收容浪浪年募2800萬　高協爭議多|author=|date=2017-01-10|work=|newspaper=鏡週刊|accessdate=2017-11-12|language=zh-TW}}</ref><ref>{{Cite news|url=https://tw.appledaily.com/forum/daily/20070205/3234712/|title=誇張 國道救狗狂奔6小時|author=|date=2007-02-05|newspaper=蘋果日報|accessdate=2017-11-27|language=zh-TW}}</ref><ref>{{Cite news|url=http://news.ltn.com.tw/news/society/paper/114932|title=國道跑跳碰 成功搶救209|author=侯承旭、林志青、劉曉欣|date=2007-02-05|work=自由時報電子報|accessdate=2017-11-27}}</ref>。

2008年起，高雄市關懷流浪動物協會定期在其[[官方網站|官方網站]]上公告與救援[[流浪狗|流浪狗貓]]有關的新聞，及透過[[臉書|臉書]][[粉絲專頁|粉絲專頁]]和[[LINE_(應用程式)|LINE]][[官方帳號|官方帳號]]邀請民眾認購愛心商品<ref name=":0">{{Cite news|url=https://www.mirrormedia.mg/story/20170110soc006/|title=收容犬隻數目膨風　高雄市關懷流浪動物協會爆離譜帳冊|author=|date=2017-01-10|work=|newspaper=鏡週刊|accessdate=2017-11-11|language=zh-TW}}</ref>。

2013年3月27日，高雄市關懷流浪動物協會串聯臺灣各地動保人士到[[民主進步黨|民主進步黨]][[中央黨部|中央黨部]]，抗議該黨[[高雄市議員|高雄市議員]][[林芳如|林芳如]]公開侮辱他們「像[[黑社會|黑社會]]」，要求林芳如公開道歉，也要求[[民主進步黨主席|黨主席]][[蘇貞昌|蘇貞昌]]道歉並介入處理；民主進步黨副秘書長[[林育生|林育生]]出面接見抗議群眾，[[台灣貓狗人共和國協會|台灣貓狗人共和國協會]]理事長[[黃泰山|黃泰山]]、藝人[[陳珮騏|陳珮騏]]也在抗議群眾行列<ref>{{Cite web|url=http://www.crntt.tw/crn-webapp/doc/docDetailCreate.jsp?coluid=98&kindid=2995&docid=102484068|author=鄒麗泳|language=zh-tw|title=高雄綠議員辱罵動保人士 社團赴民進黨抗議|publisher=[[中評社|中評社]]|date=2013-03-27|accessdate=2018-06-04}}</ref>。

2016年6月，高雄市關懷流浪動物協會與民主進步黨高雄市議員[[陳信瑜|陳信瑜]]聯手揭發[[中華民國海軍陸戰隊|中華民國海軍陸戰隊]]士兵虐殺流浪狗「小白」事件；協會還發起「著白衣悼念軍中受虐犬」活動，集合百位民眾在[[中華民國國防部|中華民國國防部]]外抗議，提出「國防部長道歉」等四項要求，但否認要求[[國家賠償|國家賠償]]<ref>{{Cite news|url=http://www.tanews.org.tw/info/10737|title=小白狗案 抗議民眾不散 要求懲兇|newspaper=台灣動物新聞網|accessdate=2017-11-11|language=zh-hant}}</ref><ref>{{Cite news|url=https://pets.ettoday.net/news/724723|title=牠犯了什麼錯？動團將圍國防部　王小華淚求「站出來」|date=2016-06-28|newspaper=[[ETtoday新聞雲|ETtoday新聞雲]]|accessdate=2018-04-30|language=zh-TW}}</ref><ref>{{Cite web|url=http://www.setn.com/News.aspx?NewsID=159535|title=國軍虐殺小白 愛狗人士穿白衣到國防部討公道|accessdate=2018-04-30|date=2016-06-28|publisher=三立新聞網|language=zh-TW}}</ref><ref>{{Cite news|url=http://www.chinatimes.com/realtimenews/20160628001986-260402|title=國軍虐狗案  網友串連北上國防部抗議|author=中央社|date=2016-06-28日|newspaper=中時電子報|accessdate=2018-04-30|language=zh-TW}}</ref><ref>{{Cite news|url=https://tw.appledaily.com/new/realtime/20160628/895757/|title=為小白伸冤　愛狗人士北上國防部抗議|author=吳慧芬、侯昌騰|date=2016-06-28|newspaper=臺灣蘋果日報|accessdate=2018-04-30|language=zh-TW}}</ref><ref>{{Cite news|url=https://pets.ettoday.net/news/725849|title=國防部前抗議被批太超過　動保團體澄清：沒要求國賠|date=2016-06-29|newspaper=ETtoday新聞雲|accessdate=2018-04-30|language=zh-TW}}</ref>。

== 保育場 ==
高雄市關懷流浪動物協會收容犬貓的保育場因地主逼遷的原因，歷經了數次搬遷。2015年，保育場「[[糖果屋|糖果屋]]」建成，收容兩千隻流浪犬貓<ref name=":4" />。協會希望犬貓們擁有舒適的居住環境、而不是像住在狗舍或監獄一樣的感覺，每天都會清潔糖果屋，維持環境乾淨。飲食方面，則用經攪碎機攪碎過的雞肉跟[[白米|白米]]混在一起給犬貓吃，雖然花費成本頗高，但犬貓吃了比較有營養<ref>{{Cite web|url=https://www.ettoday.net/news/20181230/1337904.htm|title=給毛孩一個永遠的家！ 「糖果屋」為流浪動物打造幸福園地|accessdate=2019-01-15|author=李彥慧|date=2018-12-30|publisher=ETtoday新聞雲|language=zh-Hant}}</ref>。

== 爭議事件 ==
2007年11月，民眾指控，高雄市關懷流浪動物協會承接高雄市政府捕犬業務，致使「每年至少讓一千四百隻流浪狗斷送性命」。該會理事長王小華澄清，當初是不忍見其他承接單位以殘暴方式濫捕流浪狗來增加業績，才承接捕犬業務<ref name=":3">{{Cite news|url=https://tw.appledaily.com/headline/daily/20070321/3333417/|title=流浪協會年殺1400隻狗|author=|date=|work=|newspaper=蘋果日報|accessdate=2017-11-11|language=zh-TW}}</ref>。

2010年3月，高雄市鼓山區美術南二路一名陳姓男子時常以[[空氣槍|空氣槍]]虐狗、用車撞狗，王小華帶著高雄市關懷流浪動物協會20名志工去質問陳姓男子，見他並無悔意，便向他丟[[雞蛋|雞蛋]]、推倒他的[[摩托車|摩托車]]。事後，高雄市關懷流浪動物協會遭[[起訴|起訴]]涉及《[[中華民國刑法|中華民國刑法]]》[[公然侮辱罪|公然侮辱罪]]、[[妨害名譽罪|妨害名譽罪]]、[[妨害秘密罪|妨害秘密罪]]<ref>{{Cite news|url=http://news.ltn.com.tw/news/local/paper/435445|title=〈南部〉關懷流浪動物協會理事長王小華 起訴|author=鮑建信、葛祐豪|date=2010-10-14|work=自由時報電子報|accessdate=2018-04-30}}</ref><ref>{{Cite news|url=http://www.ipet.tw/content/93108.html|title=動物保護者用雞蛋砸虐狗疑犯|date=2010-03-26|newspaper=愛寵物咨詢網|accessdate=2017-11-12|language=zh-TW}}</ref><ref>{{Cite web|url=http://www.kcsaa.org.tw/context_single.jsp?sga=2111|title=2010年03月18日槍殺你‧撞死你|accessdate=2017-11-12|author=|date=|publisher=高雄市關懷流浪動物協會}}</ref><ref name=":1" /><ref name=":2" />。

2011年7月，高雄市關懷流浪動物協會在[[臉書|臉書]]貼文指責，[[高雄市政府消防局|高雄市政府消防局]]仁武分隊把卡在水溝裡的[[哈士奇|哈士奇]]救上來卻放在水溝旁，是「把生命當作垃圾」；引來不少[[網友|網友]]撻伐該會，全台[[消防員|消防員]]也感到不滿<ref>{{Cite news|url=https://www.nownews.com/news/20110705/506029|title=仁武消防隊救狗還挨罵！網友痛批高市關懷流浪動物協會|newspaper=NOWnews 今日新聞|accessdate=2017-11-12|language=zh-TW}}</ref><ref>{{Cite web|url=http://www.fet.org.tw/index.php?option=com_content&view=article&id=1608:%E6%95%91%E7%8B%97%E5%8F%8D%E6%8C%A8%E7%BD%B5%20%E6%B6%88%E9%98%B2%E5%93%A1%E4%B8%B2%E9%80%A3%E6%80%92%E5%90%BC&catid=1:%E5%9C%8B%E5%85%A7%E6%96%B0%E8%81%9E&Itemid=107|title=救狗反挨罵 消防員串連怒吼|accessdate=2017-11-12|author=|date=|publisher=新北市消防設備士公會|language=zh-tw|archive-url=https://web.archive.org/web/20171113112925/http://www.fet.org.tw/index.php?option=com_content&view=article&id=1608:%E6%95%91%E7%8B%97%E5%8F%8D%E6%8C%A8%E7%BD%B5%20%E6%B6%88%E9%98%B2%E5%93%A1%E4%B8%B2%E9%80%A3%E6%80%92%E5%90%BC&catid=1:%E5%9C%8B%E5%85%A7%E6%96%B0%E8%81%9E&Itemid=107|archive-date=2017-11-13|dead-url=yes}}</ref>。

2014年，因高雄市三名[[越南|越南]]籍勞工將流浪貓抓來食用的事件曝光，高雄市關懷流浪動物協會對此製作了「吃貓越勞，滾出台灣！」文宣，引起爭議，並遭質疑帶有[[種族歧視|種族歧視]]意涵；王小華在事後澄清，該標語只針對那三名殺貓的越勞<ref>{{Cite news|url=http://www.storm.mg/article/35869|title=吃貓越勞滾出去？高雄動保團體文宣惹議|author=王立柔|date=2014-09-15|work=風傳媒|accessdate=2017-11-12|language=zh-TW}}</ref><ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/1107089|title=吃貓越勞滾出台灣？ 高雄動保團體文宣引爭議|author=|date=2014-09-15|work=自由時報電子報|accessdate=2017-11-12}}</ref>。

2015年11月，[[壽山國家自然公園|壽山國家自然公園]]籌備處為保護當地野生動物，誘捕了26隻流浪犬送到[[高雄市動物保護處|高雄市動物保護處]]壽山動物收容所；但高雄市關懷流浪動物協會不滿這項安置，前往壽山國家自然公園籌備處抗議、要求安置在民間收容單位，甚至揚言要將這些流浪犬領養後再放回柴山，此行徑隨即引發爭議<ref>{{Cite news|url=https://tw.news.yahoo.com/%E6%9F%B4%E5%B1%B1%E7%A7%BB%E7%BD%AE%E6%B5%81%E6%B5%AA%E7%8A%AC-%E6%8A%97%E8%AD%B0%E8%80%85%E6%8F%9A%E8%A8%80%E8%A6%81%E5%86%8D%E9%87%8E%E6%94%BE%E7%8A%AC%E9%9A%BB%E5%85%A5%E5%B1%B1-011639724.html|title=柴山移置流浪犬 抗議者揚言要再野放犬隻入山|author=林憲源|date=2015-11-19|work=中廣新聞網|accessdate=2017-11-12}}</ref>。

2016年10月，一名74歲陳姓[[拾荒|拾荒老人]]誤取高雄市關懷流浪動物協會擺放在騎樓的一袋空[[寶特瓶|寶特瓶]]，遭王小華控告[[竊盜罪|竊盜罪]]；陳姓老人表示覺得委屈、活了大半輩子第一次被當賊對待，事發後引起爭議<ref>{{Cite news|url=https://tw.appledaily.com/new/realtime/20161018/970250/|title=翁顧癌兒拾荒　為1元寶特瓶遭法辦|author=|date=2016-10-18|work=|newspaper=蘋果日報|accessdate=2017-11-11|language=zh-TW}}</ref><ref>{{Cite web|url=http://www.setn.com/News.aspx?NewsID=190842|title=為1元寶特瓶！顧癌兒病妻拾荒翁 遭關懷流浪動物理事怒告|accessdate=2017-11-11|author=翁郁雯、田居達|date=2016-10-18|publisher=三立新聞網}}</ref>。王小華表示事後才知道老人的辛酸背景，但因竊盜是[[公訴罪|公訴罪]]、無法撤告，只能盼在[[檢方|檢方]]傳喚時告知始末，期盼檢方不要追究阿伯<ref>{{Cite web|url=https://tw.appledaily.com/new/realtime/20161019/970601/|title=【洋蔥動畫】為寶特瓶挨告 拾荒伯身世堪憐|accessdate=2019-01-15|date=2016-10-19|publisher=臺灣蘋果日報|language=zh-TW}}</ref>。
