'''高角羚'''（[[学名|学名]]：''{{lang|la|Aepyceros melampus}}''），别名'''黑斑羚'''、'''飞羚'''，是一种中等体型的[[羚羊|羚羊]]，分布于非洲东部和南部。高角羚由德国动物学家[[辛里奇·利希滕斯坦|辛里奇·利希滕斯坦]]于1812年命名，是高角羚[[属|属]]的唯一物种，下有高角羚[[亚种|亚种]]和黑面高角羚亚种。高角羚肩高约70－92厘米，体重约40－76千克，体表覆盖着光滑而略带红棕色的毛。雄性高角羚的[[角_(动物)|角]]可长达45－92厘米。

高角羚主要在昼间活动，根据地理位置和气候不同，可能表现出群居性，也可能表现出领地意识。存在三种不同的社会群体：有[[领域_(动物)|领地]]意识的雄性，待交配的雄性群体，雌性群体。高角羚善以跳跃来躲避捕食者，并以其特有的两种跳跃方式著称。高角羚的食谱包括各种草、嫩叶和果实。每年雨季结束前，通常是在五月，高角羚会有为期三周左右的繁殖季节。发情的雄性会为了{{link-en|支配地位|Dominance (ethology)}}而争斗，获胜者会向雌性求爱。雌性怀孕期为六至七个月，通常为单胎。幼崽接受哺乳四至六个月，之后年轻的雄性会被赶出群体，加入待交配的雄性群体；年轻的雌性则可以继续留在本群体中。

高角羚通常在[[疏林|疏林]]或者疏林和[[热带草原|热带草原]]间的{{link-en|群落交错区|ecotone}}活动，栖息地靠近水源。高角羚亚种分布较为广泛，黑面高角羚亚种的分布则限于较小的区域中。[[国际自然保护联盟|国际自然保护联盟]]将高角羚亚种归于[[无危物种|无危物种]]，将黑面高角羚亚种归于[[易危物种|易危物种]]。截至2008年，野生的黑面高角羚仅有1000余只。

== 分类和演化 ==

高角羚由德国动物学家[[辛里奇·利希滕斯坦|辛里奇·利希滕斯坦]]于1812年命名，是高角羚[[属|属]]的唯一物种，下有高角羚[[亚种|亚种]]和黑面高角羚亚种<ref name =MSW3 />。1984年，古生物学家{{link-en|伊丽莎白·弗尔巴|Elisabeth Vrba}}基于高角羚和[[狷羚|狷羚]]的相似之处，认为高角羚和[[狷羚亚科|狷羚亚科]]同为[[牛科|牛科]]下的亚科<ref name="vrba">{{cite book |last1 = Vrba |first1 = E.S. |editor1-last = Eldredge |editor1-first = N. |editor2-last = Stanley |editor2-first = S.M. |title = Living Fossils |date = 1984 |publisher = Springer |location = New York, USA |isbn = 978-1-4613-8271-3 |chapter = Evolutionary pattern and process in the sister-group Alcelaphini-Aepycerotini (Mammalia: Bovidae) |oclc = 10403493 |language=en}}</ref>。1999年[[法国国家科学研究中心|法国国家科学研究中心]]进行的一项{{link-en|系统发育|phylogenetic}}研究对[[线粒体DNA|线粒体DNA]]和{{link-en|核DNA|Nuclear DNA}}的分析显示，高角羚和[[桑岛新小羚|桑岛新小羚]]（[[岛羚属|岛羚属]]）共同形成一个新的[[演化支|演化支]]，同时[[骝毛小羚羊|骝毛小羚羊]]和[[山羚|山羚]]是另一个演化支，这两个演化支在演化树上处于并列位置<ref name="Hassanin">{{cite journal |last1 = Hassanin |first1 = A. |last2 = Douzery |first2 = E.J.P. |title = Evolutionary affinities of the enigmatic saola (''Pseudoryx nghetinhensis'') in the context of the molecular phylogeny of Bovidae |journal = Proceedings of the Royal Society B: Biological Sciences |date = 1999 |volume = 266 |issue = 1422 |pages = 893–900 |doi = 10.1098/rspb.1999.0720 |pmid = 10380679 |pmc = 1689916 |language=en }} {{open access}}</ref>。2003年的一项研究利用[[核糖体核糖核酸|核糖体RNA]]和[[血影蛋白|β-血影蛋白]]对核序列进行了分析，结论也证实了高角羚属和岛羚属的演化上的关联<ref>{{cite journal |last = Kuznetsova |first = M.V. |last2 = Kholodova |first2 = M.V. |title = Revision of phylogenetic relationships in the Antilopinae subfamily on the basis of the mitochondrial rRNA and β-spectrin nuclear gene sequences |journal = Doklady Biological Sciences |year = 2003 |volume = 391 |issue = 1–6 |pages = 333–6 |doi = 10.1023/A:1025102617714 |issn = 1608-3105 |language=en}}</ref>。下图是按照1999年的研究绘制的[[演化树|演化树]]<ref name="Hassanin" />：

已有六个高角羚[[亚种|亚种]]被描述，但是根据线粒体DNA数据而被广泛认可的只有两个亚种<ref name="Nersting">{{cite journal |first = L.G. |last = Nersting |first2 = P. |last2 = Arctander |title = Phylogeography and conservation of impala and greater kudu |journal = Molecular Ecology |volume = 10 |issue = 3 |pages = 711–9 |year = 2001 |doi = 10.1046/j.1365-294x.2001.01205.x |pmid = 11298982 |language=en}}</ref>。尽管在形态上很相似，这两个亚种间存在显著的[[遗传距离|遗传距离]]，并且两个亚种间没有[[杂交种|杂交种]]被发现<ref name=kingdon>{{cite book |last1 = Kingdon |first1 = J. |last2 = Happold |first2 = D. |last3 = Butynski |first3 = T. |last4 = Happold |first4 = M. |last5 = Hoffmann |first5 = M. |last6 = Kalina |first6 = J. |title = Mammals of Africa |year = 2013 |publisher = Bloomsbury Publishing Plc |location = London, UK |isbn = 978-1-4081-8996-2 |pages = 479–84 |url={{Google books|id=B_07noCPc4kC|page=RA4-PA479|plainurl=yes}} |oclc = 854973585 |language=en}}</ref><ref name="Lorenzen">{{cite journal |last1 = Lorenzen |first1 = E.D. |last2 = Arctander |first2 = P. |last3 = Siegismund |first3 = H.R. |title = Regional genetic structuring and evolutionary history of the impala (''Aepyceros melampus'') |journal = Journal of Heredity |date = 2006 |volume = 97 |issue = 2 |pages = 119–32 |doi = 10.1093/jhered/esj012 |url = http://jhered.oxfordjournals.org/content/97/2/119.full.pdf+html |language=en}} {{open access}}</ref>。

* ''A. m. melampus'' <small>Lichtenstein, 1812</small>：高角羚指名亚种，分布于非洲东部和南部，包括北至[[肯尼亚|肯尼亚]]中部，南至[[南非|南非]]，西至[[安哥拉|安哥拉]]东南部的广袤区域。
* ''A. m. petersi'' <small>{{link-en|Bocage|José Vicente Barbosa du Bocage}}, 1879</small>：黑面高角羚亚种，分布于非洲西南部，在[[纳米比亚|纳米比亚]]西北部和安哥拉西南部。

根据弗尔巴的研究，高角羚自狷羚亚科的祖先演化而来，虽然狷羚亚科的形态历经至少18次{{link-en|遗传分歧|Genetic divergence|变化}}，高角羚维持了其基本形态至少五百万年没有重大变化<ref name=vrba /><ref>{{cite book|last1=Arctander|first1=P.|last2=Kat|first2=P.W.|last3=Simonsen|first3=B.T.|last4=Siegismund|first4=H.R.|editor1-last=Smith|editor1-first=T.B.|editor2-last=Wayne|editor2-first=R.K.|title=Molecular Genetic Approaches in Conservation|date=1996|publisher=Oxford University Press|location=Oxford, UK|isbn=978-0-19-534466-0|pages=399–412|chapter=Population genetics of Kenyan impalas – consequences for conservation|url={{Google books|id=QI8sdCHszY0C|page=399|plainurl=yes}} |oclc = 666957480|language=en}}</ref>。历史上还曾发现过高角羚的数个化石种，包括在[[埃塞俄比亚|埃塞俄比亚]]发现的[[上新世|上新世]]化石种''[[Aepyceros_datoadeni|A. datoadeni]]''<ref name="Geraads2012">{{cite journal |last1 = Geraads |first1 = D. |last2 = Bobe |first2 = R. |last3 = Reed |first3 = K. |title = Pliocene Bovidae Mammalia from the Hadar Formation of Hadar and Ledi-Geraru, Lower Awash, Ethiopia |journal = Journal of Vertebrate Paleontology |date = 2012 |volume = 32 |issue = 1 |pages = 180–97 |doi = 10.1080/02724634.2012.632046 |language=en}}</ref> 。目前发现的最早的化石表明，高角羚的远古祖先相较其目前的体型略小，但是其他方面都很相似。这表示高角羚自人类的史前时代以来已经有效地适应了环境。高角羚的群居性、食谱的多样性、种群数量扩大的趋势、对[[真蜱科|蜱虫]]的抵抗力以及与{{link-en|啄牛鸟|oxpecker}}的[[共生|共生]]关系共同保证了高角羚在形态和行为上的相对稳定<ref name=vrba />。

== 形态 ==

[[File:ImpalaSerengetiNationalParkMar2011.jpg|thumbnail]]]]

高角羚是一种中等体型的[[羚羊|羚羊]]，体型类似[[赤羚|赤羚]]和[[葛氏瞪羚|葛氏瞪羚]]<ref name="estes">{{cite book |last = Estes |first = R.D. |title = The Behavior Guide to African Mammals: Including Hoofed Mammals, Carnivores, Primates |year = 2004 |publisher = University of California Press |location = Berkeley, US |isbn = 978-0-520-08085-0 |pages = 158–66 |edition = 4th |url = {{Google books|page=158|id=g977LsZHpcsC|plainurl=yes}} |oclc = 19554262 |language=en}}</ref>。高角羚的头身长约130厘米<ref name="Liebenberg">{{cite book |last1 = Liebenberg |first1 = L. |title = A Field Guide to the Animal Tracks of Southern Africa |date = 1990 |publisher = D. Philip |location = Cape Town, South Africa |isbn = 978-0-86486-132-0 |pages = 275–6 |url = {{Google books|id=3Aco4wVvpdUC|page=275|plainurl=yes}} |oclc = 24702472 |language=en}}</ref>，雄性肩高约75－92厘米，雌性肩高约70－85厘米；雄性体重约53－76千克，雌性体重约40－53千克。高角羚[[两性异形|两性异形]]，雌性没有角，并且体型显著小于雄性；雄性的[[角_(动物)|角]]可长达45－92厘米<ref name="estes" />。雄性的角起皱、分叉，有部分弯曲，根部中空；这一弓状结构能够使得两只高角羚的角互相锁定在一起，帮助雄性在争斗中将对手甩出去，同时还起到防止[[颅骨|颅骨]]受伤的功能<ref name="kingdon" /><ref name="estes" />。

高角羚光滑的皮毛表现出两种颜色：背部多为红棕色，侧面多为黄褐色，腹部却多为白色。脸部的特征包括眼部周围的白环，浅色的下巴和{{link-en|口鼻部|snout}}。高角羚的耳朵长约30厘米，形状较尖，有黑色出现<ref name="kingdon" /><ref name=Stuart />。臀部至后腿上部有黑色条纹，尾巴多毛，长约30厘米，白色为主，带有一道黑色条纹<ref name="Stuart">{{cite book |last1 = Stuart |first1 = C. |last2 = Stuart |first2 = T. |title = Field Guide to Mammals of Southern Africa |date = 2001 |publisher = Struik Publishers |location = Cape Town, South Africa |isbn = 978-1-86872-537-3 |page = 210 |edition = 3rd |url = {{Google books|id=sdY_sa1FPw0C|page=210|plainurl=yes}} |oclc = 46643659 |language=en}}</ref>。高角羚的颜色和[[长颈羚|长颈羚]]颇为相似，后者的角较短，没有高角羚腿上的黑色条纹<ref name=kingdon />。高角羚后腿上有{{link-en|气味腺|scent gland}}，被一丛黑毛覆盖。对于有支配地位的雄性，[[皮脂腺|皮脂腺]]集中于前额和躯干部位，在繁殖季节分泌最为旺盛<ref name="estes" /><ref name="cavendish">{{cite book |last = Armstrong |first = M. |title = Wildlife and Plants |volume = 9| year = 2007 |publisher = Marshall Cavendish |location = New York, US |isbn = 978-0-7614-7693-1 |edition = 3rd |pages = 538–9 |url = {{Google books|id=W7TJGrHh0yIC|page=538|plainurl=yes}} |oclc = 229311414 |language=en}}</ref>。对于雌性，皮脂腺则较不发达，活跃程度没有显著的季节性差异<ref>{{cite journal |last1 = Welsch |first1 = U. |last2 = van Dyk |first2 = G. |last3 = Moss |first3 = D. |last4 = Feuerhake |first4 = F. |title = Cutaneous glands of male and female impalas (''Aepyceros melampus''): seasonal activity changes and secretory mechanisms |journal = Cell and Tissue Research |date = 1998 |volume = 292 |issue = 2 |pages = 377–94 |doi = 10.1007/s004410051068 |language=en}}</ref>。高角羚有四个[[乳腺|乳头]]<ref name="estes" />。

较之高角羚亚种，黑面高角羚亚种体型更大，皮毛颜色更深。这一特征是由[[隐性基因|隐性基因]]决定的<ref name="Hoven">{{cite book|last1=Hoven|first1=W.|editor1-last=van der Duim|editor1-first=R.|editor2-last=Lamers|editor2-first=M.|editor3-last=van Wijk|editor3-first=J.|title=Institutional Arrangements for Conservation, Development and Tourism in Eastern and Southern Africa|date=2015|publisher=Springer|location=Dordrecht, Netherlands|isbn=978-94-017-9528-9|pages=101–18|chapter=Private game reserves in southern Africa|url={{Google books|id=Tn5qBQAAQBAJ|page=101|plainurl=yes}} |oclc = 895661132 |language=en}}</ref>。黑面高角羚最显著的特征是鼻子左右各有一条深色条纹<ref name=Liebenberg /><ref name=Stuart />。其他特征包括耳朵上更大的黑色尖端，更多毛、更长的尾巴等<ref name="kingdon" />。

高角羚下颚的牙齿排列相当特别，其具有类似[[原猴|原猴]]的{{link-en|齿梳|toothcomb}}的结构<ref name="mckenzie">{{cite journal |last = McKenzie |first = A.A. |title = The ruminant dental grooming apparatus |journal = Zoological Journal of the Linnean Society |date = 1990 |volume = 99 |issue = 2 |pages = 117–28 |doi = 10.1111/j.1096-3642.1990.tb00564.x |language=en}}</ref>，用于互相梳理头部和脖子上的皮毛，清除[[皮外寄生虫|皮外寄生虫]]<ref name="kingdon" /><ref name="Mills">{{cite book |last1 = Mills |first1 = G. |last2 = Hes |first2 = L. |title = The Complete Book of Southern African Mammals |date = 1997 |publisher = Struik Publishers |location = Cape Town, South Africa |isbn = 978-0-947430-55-9 |page = 271 |edition = 1st |url = {{Google books|id=CavgCweI1nMC|page=271|plainurl=yes}} |oclc = 37480533 |language=en}}</ref>。

== 生态和行为 ==

[[File:Impala_AdeFrias.jpg|thumbnail]]

高角羚是[[昼行性|昼行动物]]，主要在正午最热的时段以外的昼间时段活动，夜间休息为主<ref name=estes />。存在三种不同的社会群体：有[[领地_(动物)|领地]]意识的雄性，待交配的雄性群体，雌性群体<ref name=schenkel>{{cite journal |last = Schenkel |first = R. |title = On sociology and behaviour in impala (''Aepyceros melampus'') Lichtenstein |journal = [[非洲生態學雜誌|African Journal of Ecology]] |year = 1966 |volume = 4 |issue = 1 |pages = 99–114 |doi = 10.1111/j.1365-2028.1966.tb00887.x |language=en}}</ref>。有领地意识的雄性会以尿和粪便划分领地的界线，驱赶其他幼年个体或雄性侵入者以守卫领地，并可能在其领地上形成由雌性组成的{{link-en|后宫 (动物)|Harem (zoology)|后宫}}<ref name=estes />。待交配的雄性群体一般比较小，只有不到30只高角羚，两两之间相隔2.5－3米左右，除了年龄差距较大的个体间以外，基本不进行互动。雌性群体的规模则从6只到100只都有可能，群体在约80－180[[公顷|公顷]]的范围内活动。高角羚母子间的关系不强，断奶后即告终结，未成年的个体离开母亲的群体，加入其它群体。雌性群体组织较为松散，没有明显的领导<ref name=estes /><ref name=murray>{{cite journal |last = Murray |first = M.G. |title = Structure of association in impala, ''Aepyceros melampus'' |journal = Behavioral Ecology and Sociobiology |date = 1981 |volume = 9 |issue = 1 |pages = 23–33 |doi = 10.1007/BF00299849 |language=en}}</ref>。在待交配的雄性群体和雌性群体中，互相梳理皮毛是一种重要的社交互动。事实上，高角羚是目前已知仅有的既能自己梳理皮毛，又能互相梳理皮毛的[[有蹄类|有蹄类]]动物。雌性通常会为和自己相关的个体梳理，雄性则偏好不相关的个体<ref name="hart">{{cite journal |last1 = Hart |first1 = B.L. |last2 = Hart |first2 = L.A. |title = Reciprocal allogrooming in impala, ''Aepyceros melampus'' |journal = Animal Behaviour |year = 1992 |volume = 44 |issue = 6 |pages = 1073–83 |doi = 10.1016/S0003-3472(05)80319-7 |language=en}}</ref>。

高角羚的社会行为会受到气候和地理位置影响。高角羚在某些时候表现出群居性，另外一些时候则表现出领地意识，且其领地意识持续时间在不同种群间差异很大。例如，非洲南部的种群会在[[繁殖季节|繁殖季节]]中表现出领地意识，仅持续很短的几个月；而非洲东部的种群，领地意识不强，繁殖的季节却更长。此外，有领地意识的雄性有时也会容忍待交配雄性的存在，甚至在一年中的不同时候，其自身也可能转换成为待交配雄性。有研究表明，在[[塞伦盖蒂国家公园|塞伦盖蒂国家公园]]的高角羚中，94%的雄性领地意识仅持续不足四个月<ref name=estes />。

高角羚是[[猎豹|猎豹]]、[[豹|豹]]、[[狮|狮]]等肉食动物的重要猎物。高角羚具有其特有的两种跳跃方式：一种跳跃方式可以跃过高达3米的草木障碍甚至其他高角羚，水平位移可达10米；另一种跳跃方式由一连串的跳跃组成，其中每次跳跃均由前腿先落地，后腿在半空中做出踢的动作，四腿落地后再反弹。两种跳跃方式均会跳向不同方向，以迷惑捕食者<ref name=kingdon /><ref>{{cite web |url = http://animals.nationalgeographic.com/animals/mammals/impala/ |title = Impala: ''Aepyceros melampus'' |publisher = [[国家地理学会|National Geographic]] |language=en|accessdate = 2014-05-06 }}</ref>。有时，高角羚也会隐藏在草木之中，避免被捕食者发现<ref name=Nowak>{{cite book |last = Nowak |first = R.M. |title = Walker's Mammals of the World |volume= 2 |year = 1999 |publisher = Johns Hopkins University Press |location = Baltimore, USA |isbn = 978-0-8018-5789-8 |pages = 1194–6 |edition = 6th |oclc = 39045218 |language=en}}</ref>。高角羚会发出大声的呼啸，先是闭嘴发出一至三声喷鼻声，然后张嘴发出两至十声咕哝声，同时伴有尾巴竖起；呼啸声可以传出2千米之远<ref name=estes />。高角羚可通过气味腺分泌物识别有领地意识的雄性<ref name="Kingdon1989">{{cite book |last1 = Kingdon |first1 = J. |title = East African Mammals: An Atlas of Evolution in Africa |volume = 3 |date = 1989 |publisher = Academic Press |location = London, UK |isbn = 978-0-226-43725-5 |pages = 462–74 |url = {{Google books|id=JiHqC9ROZTQC|page=462|plainurl=yes}} |oclc = 48864096 |language=en}}</ref>，其领地范围相对稳定，对于较年长的雄性，领地可以保持数年不变<ref name="estes" />。

=== 寄生和共生 ===

[[File:Impala_mutualim_with_birds_wide.jpg|thumb]]

多种蜱虫可寄生于高角羚<ref name=Mooring1995>{{cite journal |last1 = Mooring |first1 = M.S. |title = The effect of tick challenge on grooming rate by impala |journal = Animal Behaviour |date = 1995 |volume = 50 |issue = 2 |pages = 377–92 |doi = 10.1006/anbe.1995.0253 |url = http://www.pointloma.edu/sites/default/files/filemanager/Biology/Faculty/Mooring/Publications/1995-Tick_challenge_impala_grooming_Anim_Behav.pdf |format = pdf |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20161012005200/http://www.pointloma.edu/sites/default/files/filemanager/Biology/Faculty/Mooring/Publications/1995-Tick_challenge_impala_grooming_Anim_Behav.pdf |archivedate = 2016-10-12 }}</ref><ref>{{cite journal |last1 = Gallivan |first1 = G.J. |last2 = Culverwell |first2 = J. |last3 = Girdwood |first3 = R. |last4 = Surgeoner |first4 = G.A. |title = Ixodid ticks of impala (''Aepyceros melampus'') in Swaziland: effect of age class, sex, body condition and management |journal = South African Journal of Zoology |date = 1995 |volume = 30 |issue = 4 |pages = 178–86 |doi = 10.1080/02541858.1995.11448385 |url = http://www.tandfonline.com/doi/pdf/10.1080/02541858.1995.11448385 |language=en }} {{open access}}</ref><ref>{{cite journal |last1 = Horak |first1 = I.G. |title = Parasites of domestic and wild animals in South Africa. XV. The seasonal prevalence of ectoparasites on impala and cattle in the Northern Transvaal |journal = The Onderstepoort Journal of Veterinary Research |date = 1982 |volume = 49 |issue = 2 |pages = 85–93 |pmid = 7177586 |language=en}}</ref>。在[[津巴布韦|津巴布韦]]，蜱虫的大量侵扰造成的{{link-en|蜱瘫痪|tick paralysis}}已被证实是有蹄类死亡的重要原因之一。高角羚利用其类似齿梳的牙齿排列，通过梳理皮毛来清理蜱虫。然而，为了控制蜱虫的数量所需进行的大量梳理活动在夏季易造成脱水，降低对捕食者的警惕，还会对牙齿造成磨损。有研究表明，高角羚会根据蜱虫的流行程度改变梳理皮毛的频率和花费的时间<ref name=Mooring1995 />。

啄牛鸟可和高角羚共生，以蜱虫为食，同时帮助高角羚清理无法自行触及的部位（如耳部、脖子、眼皮、前额、肚皮等）的蜱虫。高角羚是和啄牛鸟共生的最小的有蹄类动物。一项研究发现，{{link-en|柯氏狷羚|Coke's hartebeest}}、葛氏瞪羚、[[汤氏瞪羚|汤氏瞪羚]]、[[转角牛羚属|转角牛羚]]等其他动物共同出现时，啄牛鸟仍偏好选择高角羚共生。可能的解释是高角羚主要生活于[[疏林|疏林]]地带，体表蜱虫密度较高<ref>{{cite journal |last1 = Hart |first1 = B.L. |last2 = Hart |first2 = L.A. |last3 = Mooring |first3 = M.S. |title = Differential foraging of oxpeckers on impala in comparison with sympatric antelope species |journal = [[非洲生態學雜誌|African Journal of Ecology]] |date = 1990 |volume = 28 |issue = 3 |pages = 240–9 |doi = 10.1111/j.1365-2028.1990.tb01157.x |url = https://www.researchgate.net/profile/Ben_Hart/publication/229520029_Differential_foraging_of_oxpeckers_on_impala_in_comparison_with_sympatric_antelope_species/links/554529cd0cf234bdb21d5192.pdf |format=pdf|language=en}} {{open access}}</ref>。另一项研究表明，相较于其他部位，啄牛鸟更偏好耳部，这可能是由于耳部受蜱虫侵扰程度最高<ref>{{cite journal |last1 = Mooring |first1 = M.S. |last2 = Mundy |first2 = P.J. |title = Interactions between impala and oxpeckers at Matobo National Park, Zimbabwe |journal = [[非洲生態學雜誌|African Journal of Ecology]] |date = 1996 |volume = 34 |issue = 1 |pages = 54–65 |doi = 10.1111/j.1365-2028.1996.tb00594.x |url = https://www.researchgate.net/profile/Mike_Mooring/publication/227814876_Interactions_between_impala_and_oxpeckers_at_Matobo_National_Park_Zimbabwe/links/55084f0a0cf2d7a281284433.pdf |format=pdf|language=en}} {{open access}}</ref>。

[[虱子|虱子]]和其他寄生虫也存在于高角羚身上<ref>{{cite journal |last1 = Horak |first1 = I.G. |last2 = Boomker |first2 = J. |last3 = Kingsley |first3 = S.A. |last4 = De Vos |first4 = V. |title = The efficacy of ivermectin against helminth and arthropod parasites of impala |journal = Journal of the South African Veterinary Association |date = 1983 |volume = 54 |issue = 4 |pages = 251–3 |pmid = 6689430 |language=en}}</ref>。南非的一项研究显示，寄生虫的数量在高角羚一岁时达到峰值，部分类型的寄生虫密度会随着季节变化<ref>{{cite journal |last1 = Horak |first1 = I.G. |title = Parasites of domestic and wild animals in South Africa. X. Helminths in impala |journal = The Onderstepoort Journal of Veterinary Research |date = 1978 |volume = 45 |issue = 4 |pages = 221–8 |pmid = 572950 |language=en}}</ref>。

=== 食性 ===

[[File:Aepyceros_melampus_(Masai_Mara,_Kenya).jpg|thumbnail]]]]

取决于周边的资源条件，高角羚既会取食低处的草，也会取食高处的嫩叶和果实<ref name="Skinner">{{cite book |last1 = Skinner |first1 = J.D. |last2 = Chimimba |first2 = C.T. |title = The Mammals of the Southern African Subregion |date = 2005 |publisher = Cambridge University Press |location = Cambridge, UK |isbn = 978-0-521-84418-5 |pages = 703–8 |edition = 3rd |oclc = 62703884|language=en}}</ref>。高角羚的食谱包括各种[[单子叶植物|单子叶植物]]、[[双子叶植物|双子叶植物]]、{{link-en|非禾本草本植物|forb}}、果实，例如[[金合欢族|金合欢果]]等。高角羚偏好距离水源近的区域，其次偏好[[多肉植物|多肉植物]]丰富的区域<ref name=estes />。一项研究显示，高角羚的食谱包括45%的单子叶植物，45%的双子叶植物，10%的果实；食谱中草的比例在雨季的第一场雨后显著增加，可高达90%，在旱季回落<ref>{{cite journal |last1 = Meissner |first1 = H.H. |last2 = Pieterse |first2 = E. |last3 = Potgieter |first3 = J.H.J. |title = Seasonal food selection and intake by male impala ''Aepyceros melampus'' in two habitats |journal = South African Journal of Wildlife Research |year = 1996 |volume = 26 |issue = 2 |pages = 56–63 |issn = 0379-4369 |language=en}}</ref>。在雨季结束前后和旱季期间，高角羚多数时候取食高处的嫩叶，特别是在旱季中，高角羚主要取食木本双子叶植物，营养较为缺乏<ref name="kingdon" /><ref>{{cite journal |last = Dunham |first = K. M. |title = The diet of impala (''Aepyceros melampus'') in the Sengwa Wildlife Research Area, Rhodesia |journal = Journal of Zoology |date = 2009 |volume = 192 |issue = 1 |pages = 41–57 |doi = 10.1111/j.1469-7998.1980.tb04218.x |language=en}}</ref>。另一项研究显示，待交配的雄性和雌性取食多子叶植物的比例远高于有领地意识的雄性<ref>{{cite journal |last = van Rooyen |first = A.F. |last2 = Skinner |first2 = J.D. |title = Dietary differences between the sexes in impala |journal = Transactions of the Royal Society of South Africa |date = 1989 |volume = 47 |issue = 2 |pages = 181–5 |doi = 10.1080/00359198909520161 |language=en}}</ref>。

在不同种类的草中，高角羚主要取食柔软而富有营养的草，很少选择[[扭黄茅|扭黄茅]]、[[黄背草|黄背草]]等硬而高的草<ref name=Sinclair>{{cite book |last1 = Jarman |first1 = P.J. |last2 = Sinclair |first2 = A.R.E. |editor1-last = Sinclair |editor1-first = A.R.E. |title = Serengeti, Dynamics of an Ecosystem |chapter = Feeding strategy and the pattern of resource partitioning in ungulates |year = 1984 |publisher = University of Chicago Press |location = Chicago, US |isbn = 978-0-226-76029-2 |pages = 130–63 |editor2-last = Norton-Griffths |editor2-first = M. |url = {{Google books|id=ffLvZbZYpr0C|page=130|plainurl=yes}} |oclc = 29118101 |language=en}}</ref>。高角羚群体进食时，外围的高角羚通常会对捕食者更为警觉；正在进食的个体会采取低头的方式来防备食物被抢占<ref>{{cite journal |last1 = Blanchard |first1 = P. |last2 = Sabatier |first2 = R. |last3 = Fritz |first3 = H. |title = Within-group spatial position and vigilance: a role also for competition? The case of impalas (''Aepyceros melampus'') with a controlled food supply |journal = Behavioral Ecology and Sociobiology |date = 2008 |volume = 62 |issue = 12 |pages = 1863–8 |doi = 10.1007/s00265-008-0615-3 |language=en }}</ref>。一项研究表明，高角羚一天中花在进食上的时间在旱季即将结束时达到最大值75.5%，雨季到来后逐渐减少，直到旱季再次来临时达到最小值57.8%<ref>{{cite journal |last = Wronski |first = T. |title = Feeding ecology and foraging behaviour of impala ''Aepyceros melampus'' in Lake Mburo National Park, Uganda |journal = [[非洲生態學雜誌|African Journal of Ecology]] |date = September 2002 |volume = 40 |issue = 3 |pages = 205–11 |doi = 10.1046/j.1365-2028.2002.00348.x |language=en }}</ref>。

=== 繁殖 ===
[[File:Fighting_impalas_brighten.jpg|thumbnail]]

雄性高角羚一岁时[[性成熟|性成熟]]，但是成功的交配一般发生于至少四年后。成熟的雄性高角羚开始建立领地，尝试接近雌性。雌性一岁半之后可以怀孕，[[发情|发情]]持续24至48小时，周期12-29天<ref name="Nowak" />。每年雨季结束前，通常是在五月，高角羚会有为期三周左右的繁殖季节。雄性在此之前数月，[[生殖腺|生殖腺]]生长，分泌更多[[激素|激素]]，导致更强的攻击性和领地意识<ref name="estes" />。这一期间，有领地意识的雄性[[尿道球腺|尿道球腺]]增重，[[睾酮|睾酮]]水平几乎可达待交配雄性的两倍<ref>{{cite journal |last = Bramley |first = P.S. |last2 = Neaves |first2 = W.B. |title = The relationship between social status and reproductive activity in male impala, ''Aepyceros melampus'' |journal = Journal of Reproduction and Fertility |year = 1972 |volume = 31 |issue = 1 |pages = 77–81 |pmid = 5078117 |doi = 10.1530/jrf.0.0310077 |language=en}}</ref>。

繁殖季节的雄性会为了{{link-en|支配地位|Dominance (ethology)}}而争斗，通常会发出大声呼啸，互相追逐，故意来回踱步，展示他们的脖子和角。雄性停止互相梳理皮毛，投入更多时间取悦发情的雌性，甚至会检查雌性的排泄物来确认雌性是否发情<ref name=hart2>{{cite journal |last = Mooring |first = M. S. |author2 = Hart, B. L. |title = Differential grooming rate and tick load of territorial male and female impala |journal = Behavioral Ecology |year = 1995 |volume = 6 |issue = 1 |pages = 94–101 |doi = 10.1093/beheco/6.1.94 |language=en }}</ref>。当雄性遇到雌性时，兴奋的雄性会追逐雌性求爱，保持3－5米的距离。雄性弹出舌头，同时可能会拼命点头示意；雌性会允许雄性舔其[[外阴|外阴]]，将自己的尾巴偏向一边。雄性尝试骑上雌性，前腿抱紧雌性的躯干，头部高抬。如果一次骑上的动作失败，雄性会隔几秒至一两分钟重复尝试。交配成功一次后，雄性即对雌性失去兴趣，而雌性可能兴趣仍未消退，继续和其他雄性交配<ref name="estes" /><ref name="schenkel" />。

雌性高角羚怀孕期为六至七个月，如果环境过于恶劣，生产的日期可能推迟一个月。生产通常是在日间，雌性会选择产痛开始时离开群体，独自生产<ref name=jarman>{{cite book |last = Jarman |first = M.V. |title = Impala Social Behaviour: Territory, Hierarchy, Mating, and the Use of Space |year = 1979 |publisher = Parey |location = Berlin, Germany |isbn = 978-3-489-60936-0 |pages = 1–92 |oclc = 5638565 |language=en}}</ref>。高角羚通常为单胎，刚出生的高角羚会被隐藏保护若干周，然后加入群体中的保育小组。高角羚幼崽接受哺乳四至六个月，之后年轻的雄性会被赶出群体，加入待交配的雄性群体；年轻的雌性则可以继续留在本群体中<ref name="estes" />。

== 分布和栖息地 ==
[[File:Aepyceros_melampus_Impala_in_Tanzania_3490_Nevit.jpg|thumb]]

高角羚主要栖息于[[疏林|疏林]]中，以便利用树荫遮阳，也会在疏林和[[热带草原|热带草原]]间的{{link-en|群落交错区|ecotone}}活动，偏好离水源近的地方。在非洲南部，高角羚种群的数量与{{link-en|可乐豆木|Colophospermum mopane}}和金合欢族疏林相关<ref name=estes /><ref name=Skinner />。栖息地的选择随季节变化：在雨季，高角羚偏好[[阿拉伯胶树|阿拉伯胶树]]疏林；旱季则偏好{{link-en|镰荚金合欢|Vachellia drepanolobium}}疏林。另一个影响栖息地选择的因素为是否易受捕食者攻击；高角羚倾向于远离较高的草丛，因为捕食者可能藏在其中<ref name=Sinclair />。一项研究发现，普通非洲象的活动会对高角羚的数量有正面影响：普通非洲象的活动会造成疏林的减少，并且促进[[山柑属|山柑属]]等[[灌木|灌木丛]]形成，从而在旱季为高角羚提供了更多食物<ref>{{cite journal |last = Rutina |first = L.P. |last2 = Moe |first2 = S.R. |last3 = Swenson |first3 = J.E. |title = Elephant ''Loxodonta africana'' driven woodland conversion to shrubland improves dry-season browse availability for impalas ''Aepyceros melampus'' |journal = Wildlife Biology |date = 2005 |volume = 11 |issue = 3 |pages = 207–13 |doi = 10.2981/0909-6396(2005)11[207:ELADWC]2.0.CO;2 |language=en}}</ref>。高角羚一般不会选择[[山地生态学|山地灌丛草甸]]作为栖息地<ref name=kingdon />，仅在[[夸祖鲁-纳塔尔省|夸祖鲁-纳塔尔省]]有出现于海拔1,400米的记录<ref name=Skinner />。

高角羚历史上广泛分布于非洲东部到南部，这一范围很大程度上保持完整，仅在[[布隆迪|布隆迪]]等地消失。高角羚生活的范围东至肯尼亚中部和南部、[[乌干达|乌干达]]东北部，南至夸祖鲁-纳塔尔省北部，西至纳米比亚和安哥拉南部，同时被引入[[加蓬|加蓬]]和非洲南部的多个自然保护区。黑面高角羚则限于安哥拉西南部和纳米比亚西北部的{{link-en|卡奥科兰|Kaokoland}}<ref name=iucn />。

== 威胁和保护 ==

[[File:Aepyceros_melampus_petersi_female_8014.jpg|thumbnail]]

[[国际自然保护联盟|国际自然保护联盟]]将高角羚亚种归于[[无危物种|无危物种]]<ref name=iucn />，将黑面高角羚亚种归于[[易危物种|易危物种]]<ref name=iucn2 />。截至2008年，高角羚约有200万只，而野生的黑面高角羚仅有1000余只<ref name=iucn /><ref name=iucn2>{{IUCN2014.3|assessors=IUCN SSC Antelope Specialist Group |year=2008|id=549|title=Aepyceros melampus petersi|downloaded=2016-05-09}}</ref>。高角羚亚种的生存没有重大威胁，而黑面高角羚亚种的生存则面临着偷猎和自然灾害等威胁。根据[[保护生物学|保护生物学]]领域的研究，实施{{link-en|物种迁移|Species translocation}}可能有益于对黑面高角羚的保护<ref name=green>{{cite journal |last = Green |first = W.C.H. |last2 = Rothstein |first2 = A. |title = Translocation, hybridisation, and the endangered black-faced impala |journal = Conservation Biology |year = 2008 |volume = 12 |issue = 2 |pages = 475–80 |doi = 10.1111/j.1523-1739.1998.96424.x |language=en}}</ref><ref name=matson>{{cite journal |last = Matson |first = T. |last2 = Goldizen |first2 = A.W. |last3 = Jarman |first3 = P.J. |title = Factors affecting the success of translocations of the black-faced impala in Namibia |journal = Biological Conservation |date = 2004 |volume = 116 |issue = 3 |pages = 359–65 |doi = 10.1016/S0006-3207(03)00229-5 |language=en}}</ref>。

约四分之一的高角羚种群生活在自然保护区中，例如[[波札那|波札那]]的[[奥卡万戈三角洲|奥卡万戈三角洲]]，肯尼亚的[[马赛马拉|马赛马拉]]和[[卡贾多|卡贾多]]，南非的[[克留格尔国家公园|克留格尔国家公园]]，[[坦桑尼亚|坦桑尼亚]]的[[鲁阿哈国家公园|鲁阿哈国家公园]]、塞伦盖蒂国家公园、[[塞卢斯禁猎区|塞卢斯禁猎区]]，[[赞比亚|赞比亚]]的[[卢安瓜河|卢安瓜河谷]]，[[津巴布韦|津巴布韦]]的[[万基国家公园|万基国家公园]]、[[赞比西河|赞比西河谷]]等等。较为珍稀的黑面高角羚主要生存于纳米比亚的[[埃托沙国家公园|埃托沙国家公园]]和私人农场中。种群密度在不同地区差异较大，坦桑尼亚的{{link-en|姆科马齐国家公园|Mkomazi National Park}}每平方千米不足一只，津巴布韦的[[卡里巴水库|卡里巴水库]]附近则高达每平方千米135只<ref name=iucn /><ref name="East">{{cite book |last1 = East |first1 = R. |title = African Antelope Database 1998 |date = 1999 |publisher = IUCN Species Survival Commission |location = Gland, Switzerland |isbn = 978-2-8317-0477-7 |pages = 238–41 |url = {{Google books|id=59Ci3C8BCmEC|page=238|plainurl=yes}} |oclc = 44634423|language=en}}</ref>。
