'''高田憂希'''（{{bd|1993年|3月16日|catIdx=Takada Yuuki}}<ref>{{Cite web |url=https://twitter.com/sunflower930316/status/577358017542287360 |title=2015年3月15日付、高田のツイートより|accessdate=2015-07-05}}</ref><ref name="profile">{{Cite web |url=https://mausu.net/talent/takada-yuuki.html|title=マウスプロモーション公式サイト：所属タレント|accessdate=2013-07-05}}</ref>）是[[日本|日本]]的女性[[聲優|聲優]]，[[福岡縣|福岡縣]][[北九州市|北九州市]]出身。血型AB型<ref>{{Cite web |url=https://twitter.com/sunflower930316/status/582478310552174592|title=2015年3月30日付、高田のツイートより|accessdate=2013-07-05}}</ref><ref>{{Cite web |url=https://twitter.com/sunflower930316/status/582480091185184768|title=2015年3月30日付、高田のツイートより|accessdate=2013-07-05}}</ref>。[[Mausu_Promotion|Mausu Promotion]]所屬<ref name="profile" />。

== 人物 ==
曾就讀[[代代木動畫學院|代代木動畫學院]]福岡校，並是Mausu Promotion附屬養成所第28期生，2015年開始為Mausu Promotion的成員。

在劇場版動畫《[[魔兵驚天錄|魔兵驚天錄BLOODY FATE]]》中飾演維格利德的人們、乘客、魔界住人等角色為動畫出道作<ref name="debut">[https://twitter.com/sunflower930316/status/380719153385988096 2013年9月19日、高田憂希Twitter]</ref>。

喜歡的作品是[[美少女戰士|美少女戰士]]，尤其喜歡[[愛野美奈子|水手金星]]<ref>{{Cite journal|journal=[[娘TYPE|娘TYPE]]|title=2016年3月號|publisher=Newtype|date=2016-01-30|page=69}}</ref>。
