'''高莽'''{{BD|1926年||2017年|10月6日|catIdx=G}}<ref>{{cite news|url=http://www.thepaper.cn/newsDetail_forward_1816717|title=91岁翻译家高莽去世，一生执着于俄罗斯文学|agency=澎湃|date=2017-10-07}}</ref>，中国翻译家、肖像画画家，[[俄罗斯科学院|俄罗斯科学院]]远东研究所名誉博士。<ref name="fish" >{{cite web |url=http://www.chinawriter.com.cn/zxhy/member/8231.shtml |title=高莽 |accessdate=2012 |publisher=中国作家网 |work= |language=zh-hans}}</ref><ref name="cat" >{{cite web |url=http://news.hexun.com/2013-11-26/160014975.html |title=高莽：人生的最后一次冲刺 |accessdate=2013 |publisher=和讯网 |work= |language=zh-hans}}</ref><ref name="bee">{{cite web|url=http://bjyouth.ynet.com/3.1/1306/28/8103130.html|title=翻译家高莽的绘画生活|accessdate=2013|publisher=北青网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20141030131155/http://bjyouth.ynet.com/3.1/1306/28/8103130.html|archivedate=2014-10-30}}</ref>

==笔名==
高莽的笔名包括乌兰汗、雪客、小四、肖尔、竹马、何马、何焉等。

==生平==
1926年，高莽出生在[[哈尔滨|哈尔滨]]，母亲是文盲，但是重视对他的教育。

1933年，高莽在哈尔滨的一所[[俄语|俄语]]学校读书，从此掌握了俄语。

1943年，高莽从[[美国|美国]]哈尔滨基督教青年会学校毕业。
同年，高莽在《大北新报》翻译了屠格涅夫的一篇散文诗《曾是多么美多么鲜的一些玫瑰》。

1945年，高莽在哈尔滨中苏友好协会工作，陆续担任翻译员、编辑等。

1947年，高莽翻译了《保尔·柯察金》电影剧本。

1949年，高莽在[[沈阳市|沈阳市]]东北中苏友好协会工作，翻译了[[乌克兰|乌克兰]]作家冈察尔的《永不掉队》。

1954年，高莽前往[[北京|北京]]中苏友好协会工作，同年加入了[[中国作家协会|中国作家协会]]。

1964年，高莽到[[中国社会科学院|中国社会科学院]]外国文学研究所工作。

1989年，高莽退休。

1997年，高莽加入[[俄罗斯作家协会|俄罗斯作家协会]]。

==家庭==
高莽和妻子有一个女儿，曾在[[南美洲|南美洲]]国家[[巴西|巴西]]工作，后回国照顾父亲高莽。
