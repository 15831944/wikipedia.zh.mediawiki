'''高登榜'''（{{bd|1914年|12月4日|1997年|7月28日}}）又名'''高玉成'''，[[陕西省|陕西省]][[延川县|延川县]]高家屯乡高家屯村人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中共中央办公厅|中共中央办公厅]]副主任，第六、七届全国人大常委会委员<ref name=sqyw/>。

==生平==
1932年参加革命工作，同年11月加入[[中国共产党|中国共产党]]。历任中共延川县委秘书，西北工委技术处党支部书记，中央国民经济部、[[陕甘宁边区|陕甘宁边区]]政府建设厅科长，油矿特派员（支部书记）。1938年10月，奉派到[[新疆|新疆]]工作，历任[[喀什|喀什]]税务局、财政局副局长，[[沙雅县|沙雅县]]税务局局长。1942年11月，遭军阀[[盛世才|盛世才]]软禁，1943年2月入狱，在狱中组织狱友先后三次绝食。1946年6月，经[[中共中央|中共中央]]营救出狱，返回[[延安|延安]]。[[第二次国共内战|第二次国共内战]]时期，历任[[中共中央西北局|中共中央西北局]]财务副秘书长兼运输大队队长，西北军区后勤运输部长、党委书记，陕甘宁边区政府交通厅副厅长。[[中华人民共和国|中华人民共和国]]成立后，历任[[西北军政委员会|西北军政委员会]]交通部副部长，西北铁路干线工程局副局长、副政委，[[中华人民共和国铁道部|中华人民共和国铁道部]]办公室主任、设计总局局长，[[国务院机关事务管理局|国务院机关事务管理局]]局长，[[国务院副秘书长|国务院副秘书长]]，[[中共中央国家机关委员会|中央国家机关党委]]书记处书记，[[中国科学院|中国科学院]]副秘书长，[[人民大会堂|人民大会堂]]管理局局长兼[[中共中央直属机关事务管理局|中共中央直属机关事务管理局]]局长，[[全国人大常委会|全国人大常委会]]副秘书长，[[中共中央办公厅|中共中央办公厅]]副主任，[[全国人大华侨委员会|全国人大华侨委员会]]副主任、主任。他是[[中共十二大|中共十二大]]代表，[[第五届全国政协委员|第五届全国政协委员]]，第六、七届全国人大常委会委员<ref name=sqyw>{{Cite web |url = http://www.sanqinyou.com/mingren/info/127216294095882.html |title =延川县政坛人物——高登榜  |author =  |date =2012-07-02  |publisher = 三秦游网 |language =  |accessdate =  |quote =  }}</ref>。

在中共中央办公厅副主任的任内，1981年[[宋庆龄|宋庆龄]]病逝，中央成立了执行宋庆龄同志遗嘱的临时小组，由高登榜、[[汪志敏_(临清)|汪志敏]]、[[李家炽|李家炽]]、[[杜述周|杜述周]]、[[沈粹缜|沈粹缜]]、[[童小鹏|童小鹏]]、[[廖承志|廖承志]]、[[邓颖超|邓颖超]]八人组成，由高登榜召集会议。该临时小组对执行宋庆龄同志遗嘱作出了原则决定<ref>{{Cite web |url =  http://news.163.com/08/0323/16/47O01NMN000120GU.html|title = 执行宋庆龄遗嘱秘闻 |author =  |date =2008-03-23  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

1997年7月28日，高登榜在北京病逝，享年83岁。
