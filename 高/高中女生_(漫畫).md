《'''高中女生'''》（{{lang-ja|女子かう生}}，又譯：-{zh-hans:高中女生;zh-hant:默繪女高}-）为日本漫畫家{{tsl|ja|若井ケン|若井堅}}，以女高中生日常為題材，所创作的漫畫作品。于2013年9月27日在[[双叶社|双叶社]]的[[網路漫畫|網路漫畫]]平台{{tsl|ja|WEBコミックアクション|WEB Comic action}}开始连载，2014年4月10日开始发行[[单行本|单行本]]<ref name="natalie"/><ref name="mantan-web"/>。台湾中文版由[[长鸿出版社|长鸿出版社]]代理，簡體中文電子版由[[布卡漫畫|布卡漫畫]]代理。

2018年9月12日发售第8卷时宣告改编为电视动画<ref name="natalie_vol8">{{Cite web |url=https://natalie.mu/comic/news/299112 |title=ひと言もしゃべらないJKコメディ「女子かう生」がTVアニメ化！2019年放送 - コミックナタリー |accessdate=2018-09-12 |last=Inc. |first=Natasha, |work=コミックナタリー |language=ja}}</ref>，並於2019年4月8日播出<ref>{{cite news |title=サイレントJKアニメ「女子かう生」放送情報解禁、OPはあま津うに |url=https://news.nicovideo.jp/watch/nw4832625 |accessdate=2019-02-23 |work=nicovideo}}</ref>。

==簡介==
本作围绕以桃子为中心的三位高中女生搞笑而温馨的日常生活为故事内容，没有出现任何角色对话，只依靠角色动作和[[拟声词|拟声词]]来推动故事的表达。<ref>{{Cite web |url=http://konomanga.jp/guide/11763-2 |title=『女子かう生』第2巻 若井ケン【日刊マンガガイド】  |accessdate=2018-09-20 |work=konomanga.jp |language=ja}}</ref>

== 登场角色 ==
;{{nihongo|富户桃子|富戸 もも子||聲：[[立花理香|立花理香]]<ref name="staffcast">{{Cite web|url=http://joshikau-anime.com/#staffcast|title=Staff/Cast|work=TVアニメ『女子かう生』公式サイト|accessdate=2019-01-22}}</ref>}}
:本作女主角，外貌靓丽，性格开朗天然搞怪，经常做一些不可思议的行为而引发故事的发生，被老师评价为“遗憾的美女”（{{lang|ja|残念な美人}}），腳很臭。
:有一个姐姐。
:和涉美成为好友，是因为在涉美小学时被其他女生发卡弄坏时，用自己姐姐送的挂饰修复了，而感动了涉美，两人因此成为好友。

;{{nihongo|涩泽涉美|渋沢 しぶ美||聲：[[嶺內智美|嶺內智美]]<ref name="staffcast"/>}}
:桃子的同班好友，性格认真，经常负责斥责桃子不可思议的愚蠢行为。[[贫乳|贫乳]]，所以对胸的事情十分敏感。
:和桃子是小学同班。两人相识是因为小学时作为班长的她曾反对同班女生在书包上带挂饰，而自己生日时得到了一个发卡，在学校被书包带挂饰的女生嫉妒而弄坏，桃子主动用自己的挂饰修复了发卡，最终被桃子感动而在挂饰提议上让步，并和桃子成为好友，而她一直使用那只发卡。<ref>漫画12话</ref>

;{{nihongo|古井真由美|古井 まゆみ||聲：[[久保百合花|久保百合花]]<ref name="staffcast"/>}}
:桃子的同班好友，转学生，仍穿著轉學前的校服，个子矮小，性格比较腼腆，容易被桃子捉弄。
:和桃子成为好友，是因为刚刚转学时被同学孤立，之后偶然介入到桃子怪诞的行为中而和桃子交好，之后加入桃子和涉美的朋友群中。
