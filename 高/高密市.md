'''高密市'''位于[[中国|中国]][[山东省|山东省]]东部，是[[潍坊市|潍坊市]]代管的一个县级市。地处[[胶东半岛|胶东半岛]]和[[山东|山东]]内陆的中间，东临山东省最大城市[[青岛|青岛]]。高密市西北与[[安丘市|安丘市]]，西南与[[诸城市|诸城市]]，东部与[[胶州市|胶州市]]，北部与[[平度市|平度市]]接壤。

高密市是[[国务院|国务院]]最早批准的沿海地区对外开放县（市）之一，也是[[中国肉类生产百强县列表|山东省肉类生产第四大、中国第13大县市]]。

== 地理 ==
===气候===
高密地处暖温带东部季风区，气候宜人，四季分明，雨量集中，雨热同期。年均气温12.7℃，无霜期平均212天，日照时数2453小时，年均降雨量619.6毫米。

== 历史 ==
[[秦朝|秦]]置高密县，治今高密市西北。属[[胶东郡|胶东郡]]。《漢武故事》言漢武帝数歲，封膠東王。[[西汉|西汉]]建元，為[[高密国|高密国]]治。[[唐|唐]][[武德|武德]]六年（623年）移治今高密市，属[[密州|密州]]。元属[[胶州|胶州]]，明初属[[青州府|青州府]]，[[洪武|洪武]]九年（1376年）后属[[莱州府|莱州府]]，清光绪三十一年，县属[[胶县|胶州府]]。[[抗日战争|抗日战争]]及[[國共內戰|國共內戰]]时期，县境曾先后析置[[胶高县|胶高]]、[[胶河县|胶河]]等县。1994年5月18日撤销高密县，改设高密市。

===高密事件===
清[[光绪|光绪]]二十五年（1899年），德国强占民地民房，修筑[[胶济铁路|胶济铁路]]；6月高密县大吕庄民众阻修铁路，被德军枪杀20余人；1900年1月，高密武生李金榜等率众阻止铁路，要求改线，时任山东巡抚[[袁世凯|袁世凯]]下令逮捕；9月高密数百民众在南流阻修铁路，又被德军枪杀3人。

== 行政区划 ==
下辖3个街道、7个镇，993个行政村：
*街道办事处：[[朝阳街道|朝阳街道]]、[[醴泉街道|醴泉街道]]、[[密水街道|密水街道]]。
*镇：[[柏城镇|柏城镇]]、[[夏庄镇|夏庄镇]]、[[姜庄镇|姜庄镇]]、[[大牟家镇|大牟家镇]]、[[阚家镇|阚家镇]]、[[井沟镇|井沟镇]]、[[柴沟镇|柴沟镇]]。

== 经济 ==
2009年GDP达到274亿元，列[[濰坊市|濰坊市]]第四。产业以纺织、化纤、建材、机械制造、农产品加工为主。

== 人口 ==

2010年[[中国第六次人口普查|中国第六次人口普查]]时，高密市共有人口895582人。共有家庭262670户，平均每户3.23人。14岁以下的少年儿童共143702人，占总人口16.04%；15-64岁人口共653610人，占总人口72.98%；65岁及以上的老年人共98270人，占总人口的10.97%。男性共计447238人，占总人口49.93%；女性共计448344，占总人口50.06%。本地居住的人口中，拥有本地户籍的人口为778962人，占86.97%。<ref name="人口普查2010">{{cite web| url =http://www.yearbookchina.com/navipage-n2013030152000017.html| title =《中国2010年人口普查分乡、镇、街道资料》| author =中国国务院人口普查办公室、中国国家统计局人口就业统计司等| year =2010| website =统计年鉴分享平台| page =表15 山东省乡、镇、街道人口| format =收费| access-date =2018-07-19| archive-url =https://web.archive.org/web/20180719084356/http://www.yearbookchina.com/navipage-n2013030152000017.html| archive-date =2018-07-19| dead-url =no}}</ref>

== 名人 ==
*[[晏婴|晏婴]]，東周時期諸侯國齊國的行政官員，侍奉景公，是著名的思想家、外交家。曾多次出使楚國，與楚王有精辟的對話。
*[[郑玄|郑玄]]（127年－200年）字康成，北海高密（今山東省高密市）人，東漢經學家。鄭益（字益恩）之父，鄭小同（字子真）祖父。
*[[刘墉|刘墉]]，清朝乾隆時期政治人物。
*[[莫言|莫言]]，原名管謨業，中国当代作家，2012年[[诺贝尔文学奖|诺贝尔文学奖]]得主，作品有《[[红高粱家族|红高粱]]》等。
*[[刘世文|刘世文]]，人称“欧洲哥”，香港著名欧洲旅游专家顾问、专栏作家。
* [[林建华|林建华]]，2015年出任[[北京大学校长列表|北京大学校长]]。
