'''高昌景教寺院壁畫'''（[[德語|德語]]：{{lang|de|Wandbilder aus einem christlichen Tempel, Chotscho}}）是指[[德國吐魯番考察隊|德國吐魯番考察隊]]於20世紀初在[[高昌古城|高昌古城]]郊外發現的一座廢棄的[[唐朝|唐朝]]基督教[[景教|景教]]寺院內出土的三幅公元9世紀末的壁畫殘片，現藏德國{{Link-en|柏林亞洲藝術博物館|Museum of Asian Art}}，亦名柏林印度藝術博物館（{{lang|de|Museum für Indische Kunst}}）。

== 歷史背景 ==
[[北宋|北宋]]年間，使者[[王延德|王延德]]於公元981年出使[[高昌回鶻|高昌回鶻]]。他見到當地仍有[[摩尼教|摩尼教]]寺院和波斯僧，即景教僧，各持其法，就是被當時的佛教稱為外道的僧侶。[[東方教會|東方教會]]初入唐帝國時稱「波斯經教」，因當時教會大本營位於[[波斯|波斯]][[薩珊帝國|薩珊帝國]]，所以唐朝時來到中土的景教教士[[阿羅本|阿羅本]]和[[景淨|景淨]]又被稱做「波斯僧」。《宋史·天竺傳》記載：「太平興國九年（公元984年）五月三日，西州回鶻與波斯外道來朝貢。」這裡所謂的「波斯外道」指[[高昌回鶻|高昌回鶻]]王國的景教信徒<ref name=景教藝術在西域之發現 />。

== 發現經過 ==
20世紀初，德國{{Link-en|柏林民族人類學博物館|Ethnological Museum of Berlin}}的{{Link-en|艾伯特·克倫威德爾|Albert Grünwedel}}和[[艾伯特·馮·李·寇克|艾伯特·馮·李·寇克]]率領的考察隊在[[吐魯番窪地|吐魯番]]進行了三次考察，期間發現了兩座[[摩尼教|摩尼教]]寺院和一座景教寺院，皆屬廢寺遺址。其中出土了一批摩尼教和景教的壁畫、紙畫、絹畫及麻布畫，包括色彩鮮麗奪目，帶有濃郁波斯風格的手抄本和絲織品、[[掛軸|掛軸]]等<ref name=德國考察隊>{{cite web |url=http://www.zgmsbweb.com/index.php?g=Home&m=index&a=detail&relaId=12813 |title=德國考察隊在高昌故城收集的摩尼教、景教繪畫殘片 |author=張惠明 |date=2016-05-30 |newspaper=中國美術報 |accessdate=2018-07-03}}</ref>。

這座小型景教寺院位於高昌古城東城牆北端與小佛塔群之間，現存三個殿堂，長約20米，寬近7米。李·寇克在寺內發現殘缺的壁畫，主要繪於東廳北牆、東牆和西廳，大多都非常殘破，他揭取了兩幅較為完整的壁畫：「做懺悔的紅衣女子」和「聖枝主日祭」<ref name=景教藝術在西域之發現>{{cite web |url=http://dsr.nii.ac.jp/narratives/discovery/09/index.html.zh |title=景教藝術在西域之發現 |author1=林梅村 |author2=宋妮雅 |date=2007-01-31 |website=dsr.nii.ac.jp |publisher= |accessdate=2018-07-04}}</ref>。克倫威德爾則以寫生畫方式臨摹了另一幅壁畫「基督榮進耶路撒冷城」<ref name=ICE>{{cite journal |author=K. Parry |year=1996 |title=Images in the Church of the East: The Evidence from Central Asia and China |url=https://www.escholar.manchester.ac.uk/api/datastream?publicationPid=uk-ac-man-scw:1m2404&datastreamId=POST-PEER-REVIEW-PUBLISHERS-DOCUMENT.PDF |journal=Bulletin of the John Rylands Library |publisher=John Rylands University Library |location=Manchester |volume=78 |issue=3 |pages={{p.|159}} |language=en |accessdate=2018-07-04}}</ref>。

== 壁畫概述 ==
* '''聖枝主日祭'''：也叫「'''棕枝主日'''」或「'''聖枝圖'''」，此畫高約63釐米，寬約70釐米。左邊一較大人像描繪的是一位教士，顯然是個[[伊朗人|伊朗人]]，留着捲曲的黑髮，身穿[[圓領袍|圓領袍]]，正在向一群[[突厥人|突厥人]]傳教；他右手拿聖水杯，也可能是聖餐杯<ref name=達·伽馬以前中亞和東亞的基督敎>{{cite book |last=Klimkeit |first=Hans-Joachim |author-link=漢斯-約雅金·克里木凱特 |others=[[林悟殊|林悟殊]]/翻譯增訂 |date=1995年4月10日 |chapter=上篇 第五章：絲路北道的基督敎遺跡 |title=《達·伽馬以前中亞和東亞的基督敎》 |series=世界文化叢書 |volume=31 |location=臺北市 |publisher=淑馨出版社 |page={{p.|35}} |isbn=957-531-421-2}}</ref>，行祝福禮的左手還持有一隻金色[[香爐_(基督教)|香爐]]<ref>{{Cite web |url=http://dsr.nii.ac.jp/toyobunko/LFc-42/V-1/page-hr/0046.html.en |title=Wandbilder aus einem christlichen Tempel, Chotscho |accessdate=2018-07-24 |website=dsr.nii.ac.jp |language=de |quote=In der linken Hand schwingt er ein goldenes (gelbes) Weihrauchfaß.}}</ref><ref>{{cite web |url=http://www.christianweekly.net/2010/ta20551.htm |title=中國境外有豐富景教遺物 |author=浩然 |date=2010年4月11日 |newspaper=基督教週報 |accessdate=2018年7月5日}}</ref>；右邊三名較小人像手執棕枝列隊而行，其中兩名男子穿着具有[[西域|西域]]風格的翻領長袍，頭上的圓形帽冠似乎也是胡帽；最右邊的女子足蹬綠色翹頭履，頭頂圓髻，腦後還梳有大髮髻，上身似乎穿着{{Link-ja|半臂}}，可見綠色衣袖，下垂至[[襦裙|襦裙]]上的織物可能是披帛。李·寇克原認為這幅畫是描繪景教祭司施[[洗禮|洗禮]]的場景，但經[[京都大學|京都大學]]教授[[羽田亨|羽田亨]]考証，此畫實際是表現信徒慶祝基督教節日[[聖枝主日|聖枝主日]]。另一名日本學者[[吉村大次郎|吉村大次郎]]則認為此畫左邊較大人物是[[耶穌|耶穌]]，右邊三名較小人物分別是[[彼得_(使徒)|彼得]]、[[約翰_(使徒)|約翰]]和[[抹大拉的馬利亞|抹大拉的馬利亞]]<ref name=景教藝術在西域之發現 />。畫中手持棕櫚枝的年輕女信徒形象被認為有[[回鶻|回鶻]]女子的特徵<ref name=德國考察隊 />。德國宗教學家[[漢斯-約雅金·克里木凱特|漢斯-約雅金·克里木凱特]]認為這幅畫證實了當時「教會的領導成員是伊朗人，包括[[粟特|粟特]]人；而突厥人，或許是來自社會的較低層，則構成了會衆」<ref name=達·伽馬以前中亞和東亞的基督敎 />。
* '''做懺悔的紅衣女子'''：此畫高43釐米，寬21釐米，描繪一名站立的女子，身着紅衣，正在做{{Link-en|悔改 (基督教)|Repentance (Christianity)|懺悔祈禱}}。相較「聖枝主日祭」裡的人物，這名女子的着裝和髮型都顯得簡單樸素。
* '''基督榮進耶路撒冷城'''：這幅畫描繪了蓄着濃密鬍鬚的基督正騎驢進入[[耶路撒冷|耶路撒冷]]。基督帶頭光的頭部佩戴着一個內嵌十字架的頭飾，向後飄拂的短上衣，手持遊行用的十字架長杖，十字架四臂末端飾有珍珠。這種珍珠十字架和同在高昌發現的一件摩尼教繪幡上描繪的[[夷數|夷數]]像[[:File:Drawing_of_Manichaean_Fragment_with_Jesus_Figure.jpg|所持十字架]]極為相似<ref name=ICE />。基督身後可見一名穿着唐朝服飾的女信徒像。此畫原作已佚失，現僅有白描複製圖。
