'''{{PAGENAMEBASE}}'''是{{PRC admin/fullname|42/90/05/006/000|3}}下辖的一个[[街道办事处|街道办事处]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|42/90/05/006/000}}}}</ref>。

[[高场原种场|高场原种场]]是潜江市的一个[[国有农场|国有农场]]<ref>{{Cite web |url=http://nkj.hubei.gov.cn/xwdt/nkxw/34508.htm |title=省农垦局印发简报推广潜江市农垦国有农场办社会职能改革经验做法|accessdate=2018-10-29 |author=杨卫东 |date=2018-06-06 |work=责任编辑：董友胜 |publisher=湖北省农垦事业管理局网站|language=Zh-hans }}</ref>。2011年时<ref>{{Cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2011/42/90/05/429005456.html |title=高场原种场|accessdate=2018-10-29 |author= |date= |work= |publisher=中华人民共和国国家统计局网站|language=Zh-hans }}</ref>，国家统计局网站中以'''高场原种场'''之名列为潜江市的一个[[类似乡级单位|类似乡级单位]]。2012年，[[湖北省民政厅|湖北省民政厅]]批复同意设立高场街道办事处，以原高场原种场的地域范围为新设的高场街道办事处行政区域范围。同时，保留“'''潜江市高场原种场'''”的牌子，实行“[[一个机构两块牌子|一套班子、两块牌子]]”的管理方式<ref>《湖北省民政厅关于潜江市设立高场街道办事处的批复》（鄂民政发〔2012〕?号）</ref>。此后<ref>{{Cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2012/42/90/05/429005006.html |title=高场办事处|accessdate=2018-10-29 |author= |date= |work= |publisher=中华人民共和国国家统计局网站|language=Zh-hans }}</ref>至2017年<ref>{{Cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2017/42/90/05/429005006.html |title=高场办事处|accessdate=2018-10-29 |author= |date= |work= |publisher=中华人民共和国国家统计局网站|language=Zh-hans }}</ref>，在国家统计局网站以'''高场办事处'''之名列为潜江市的一个类似乡级单位。
