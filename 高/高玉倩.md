'''高玉倩'''（{{bd|1927年||2018年|12月23日|}}），女，原名'''高晨'''、'''高永蒨'''，中国[[京剧|京剧]]表演艺术家，京剧[[旦角|旦角]]演员。

==生平==
1927年生于北京，工[[京剧|京剧]][[旦角|旦角]]。8岁进入[[北平|北平]]中华戏曲专科学校永字科学习，科名“高永蒨”，先后师从[[王瑶卿|王瑶卿]]、[[于连泉|于连泉]]、[[韩世昌|韩世昌]]、[[雪艳琴|雪艳琴]]、[[欧阳予倩|欧阳予倩]]、[[程玉菁|程玉菁]]等名家。1947年，加入[[焦菊隐|焦菊隐]]主办的北平艺术馆。[[北平和平解放|北平和平解放]]后，1949年7月28日，中国戏曲改进会发起人大会在[[北京饭店|北京饭店]]举行，高玉倩作为中国戏曲改进会发起人参加。[[中华人民共和国|中华人民共和国]]成立后，1950年调入[[中国京剧院|中国京剧院]]工作。1964年起，改工[[老旦|老旦]]。在现代京剧《[[红灯记|红灯记]]》（八个[[样板戏|样板戏]]之一）中饰演李奶奶，成为[[文革|文革]]期间家喻户晓的知名演员。高玉倩参演过许多戏曲及影视剧。代表京剧剧目有《[[桃花扇|桃花扇]]》、《九件衣》、《新蝴蝶梦》、《凤凰二乔》、《彩楼记》、《人面桃花》、《吕布与貂蝉》等等。在电视剧《[[西游记_(1982年电视剧)|西游记]]》中饰演[[高老庄|高老庄]]的高老太<ref>{{Cite web |url =  http://www.jingju.com/jingjurenwu/yuantuanmingjia/2011-06-03/225.html|title = 高玉倩 |author =  |date = 2011-06-03 |publisher = 京剧艺术网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://history.xikao.com/|title =  高玉倩|author =  |date =  |publisher = 中国京剧戏考 |language =  |accessdate = 2017-04-17 |quote =  }}</ref>。

2018年12月23日在[[北京|北京]]逝世，享年92岁。<ref>{{Cite web|url=https://news.sina.com.cn/2018-12-23/doc-ihmutuee1856114.shtml|title=“红灯记李奶奶”扮演者高玉倩逝世 享年92岁|accessdate=2018-12-23|date=2018-12-23|last=北京青年报|work=news.sina.com.cn}}</ref>
