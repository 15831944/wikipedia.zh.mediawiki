'''高野川'''是流經[[京都府|京都府]][[京都市|京都市]][[左京区|左京区]]的[[淀川|淀川]]水系的[[日本一級水系列表|一級河川]]。總長17公里。

== 地理 ==
發源自位在京都市左京區和[[滋賀縣|滋賀縣]][[大津市|大津市]]的邊境的[[途中峠|途中峠]]南部。之後和[[國道367號|國道367號]]並行南進。在三谷口和從左京區[[大原_(京都市)|大原]]北部的三谷峠（ミタニ峠）發源的河川匯合，在小出石町和從左京區[[鞍馬|鞍馬]]和左京區大原的邊境的天岳發源的'''高谷川'''匯合。在三谷口或[[八瀨|八瀨]]流域也各自被稱為'''大原川'''或'''八瀨川'''。

之後和從[[三千院|三千院]]深山發源的'''呂川'''及[[寂光院|寂光院]]深山發源的'''草生川'''匯合、南下流過[[比叡山|比叡山]]的西邊山麓。於上高野的花園橋和[[白川通|白川通]]交叉並與'''岩倉川'''匯合，一面和[[川端通|川端通]]並行一面在[[出町柳車站|出町柳]]的加茂大橋和[[鴨川_(淀川水系)|賀茂川]]合流形成[[鴨川_(淀川水系)|鴨川]]。

在京都市的地圖上，鴨川經常被畫為Y字樣，高野川就是其右側的部分。

== 自然 ==
* [[日本大鯢|日本大鯢]] - 有屬於[[天然記念物|特別天然記念物]]的日本大鯢棲息在此。有時會在大雨之後等情況下被沖至京都市區，嚇壞當地居民。

== 歷史 ==
若根據地質調查的節果，在古時候平安遷都之前的高野川，可能並未像現在於出町柳和鴨川匯流。該結果顯示，當時可能是在更南方的流域才和鴨川匯流。雖然有各種說法，但於[[平安時代|平安時代]]成為現在河道的可能性較高。

根據「[[日本後紀|日本後紀]]」，在平安時代高野川被稱為'''埴川'''。根據「雍州府志」，其名稱的由了是因為河川流經了高野村。

從古時候開始高野川被當作農業用水而受到重用，沿岸的村莊們享受了其所帶來的利益，但每當缺水的時候則發生了多次村莊之間的對力。

此外，高野川在現在被大原地區的部份居民當作自來水的水源。
