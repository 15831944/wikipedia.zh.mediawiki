'''高等教育'''（{{lang-fr|Études Supérieures}}；{{lang-de|Höhere Bildung}}；{{lang-en|Higher Education}}；{{lang-es|Educación Superior}}），又称'''专上教育'''（{{lang-en|Post-secondary Education}}<ref>{{cite web |url=https://www.edb.gov.hk/en/edu-system/postsecondary/index.html |title=Post-Secondary Education: Overview |work=Education Bureau, Hong Kong SAR |accessdate=2019-01-20 |date=2017-09-29 |language=en}}</ref>）是一个[[教育|教育]]层级的概念，广义上是指一切建立在[[中等教育|中等教育]]基础上的[[专业教育|专业教育]]。这个层级涵盖了[[学园|学园]]、[[大学|大学]]、[[學院_(學校)|學院]]、[[神学院|神学院]]、[[理工学院|理工学院]]、[[技術學院|技術學院]]以及其他颁授[[学位|学位]]或[[专业证书|专业证书]]的学院级机构（例如[[專科學校|專科學校]]）。

== 欧洲人权公约 ==
根据1950年起草的[[欧洲人权公约|欧洲人权公约]]第一号议定书第二条，所有签约的[[欧洲委员会|欧洲委员会]]成员国均有义务保障民众的[[受教育权|受教育权]]。

== 形式类型 ==
高等教育是[[大学|大学]]、文理学院、[[理工学院|理工学院]]和[[师范|师范]]学院等机构所提供的各种类型的教育，包括专科教育、本科教育和研究生教育，其基本入学条件为完成[[中等教育|中等教育]]，学完课程后授予学位、文凭或证书，作为完成高等学业的证明。

高等教育形式多样，有全日制的和业余的，面授的和非面授的，学校形式的和非学校形式的等等层次和形式。

高等教育所授予的学位、文凭和证书：
*[[專科學校|专科文凭]]
*[[本科學校|本科文凭]]、[[学士学位|学士学位]]
*[[硕士学位|硕士学位]]
*[[博士学位|博士学位]]

高等教育作为一种教育方式，是人生存方式的一个阶段，从时间上说，是人生存过程中的终生学习选择阶段，从空间上说，高等教育是人进入完全社会生活的最后一个准备场所。
高等教育还是高深文化知识传播创新的社会组织机构；在传播高深文化的同时，高等教育还复制或者再制了社会的价值、行为模式和道德规范。
