'''高黎贡山隧道'''是[[大瑞铁路|大瑞铁路]]位于[[怒江站|怒江车站]]和[[龙陵站|龙陵车站]]之间的一座隧道<ref name="www.cae.cn"></ref>，跨越[[保山市|保山市]][[龙陵县|龙陵县]]全境后进入[[德宏傣族景颇族自治州|德宏傣族景颇族自治州]][[芒市|芒市]]<ref name="YouTube"></ref>，是[[中缅铁路|中缅铁路]][[大理市|大理]]至[[瑞丽市|瑞丽]]段的重点控制性工程<ref name="www.cae.cn">{{cite web|url=http://www.cae.cn/cae/html/main/col35/2016-07/05/20160705162255591334118_1.html|title=大瑞铁路高黎贡山隧道施工方案暨长大深埋隧道建设方法专家咨询会在云南举行|author=中国中铁二局土木水利与建筑工程学部办公室|date=2016-07-05|website=www.cae.cn|publisher=[[中国工程院|中国工程院]]|access-date=2017-07-31|deadurl=yes|archiveurl=https://web.archive.org/web/20170903212415/http://www.cae.cn/cae/html/main/col35/2016-07/05/20160705162255591334118_1.html|archivedate=2017-09-03}}</ref>，总长34.586千米<ref name="crecgec.com"></ref><ref group="注" name="长度">另一说为34.538千米</ref><ref name="www.cae.cn"></ref>，是目前[[中华人民共和国|中华人民共和国]]在建的最长[[单线铁路|单线铁路]]隧道<ref name="www.cae.cn"></ref>、亚洲最长山岭铁路隧道和世界第7长隧道<ref name="ctg.ha">{{cite web|url=http://www.ctg.ha.cn/sdsc/tabid/630/InfoID/29176/frtid/630/Default.aspx|title=卢春房视察大瑞铁路高黎贡山隧道|author=李航、熊滔|date=2016-02-19|website=www.ctg.ha.cn|publisher=中铁隧道集团三处有限公司|access-date=2017-07-31}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="guancha.cn"></ref><ref name="fei123.com">{{cite web |url=http://www.fei123.com/1820000/1814204.shtml |title=探秘!走进亚洲山岭第一长铁路隧道——大瑞铁路高黎贡山隧道 |date=2017-02-09 |website=www.fei123.com |publisher=飞123网 |access-date=2017-07-31 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。高黎贡山隧道于2015年12月1日开工，计划竣工日期为2022年5月31日，合同总造价36.28亿[[人民币|元]]，增建的Ⅱ线工程将在2025年11月30日竣工<ref name="yn.xinhuanet.com"></ref>。高黎贡山隧道工程规模、建设难度与工程风险较大，被视为铁路建设的又一里程碑工程<ref>{{cite web |url=http://www.baoshan.cn/2017/0706/10013.shtml |title=龙陵境内亚洲第一长隧——大瑞铁路亮相央视《新闻联播》 |date=2017-07-06 |website=www.baoshan.cn |publisher=龙陵新闻网 |access-date=2017-07-31}}</ref>。2017年8月28日，随着“彩云一号”隧道掘進机在高黎贡山隧道工地始发，标志着高黎贡山隧道进入全面施工阶段<ref>{{cite web |url=http://www.gov.cn/xinwen/2017-08/28/content_5221068.htm#1 |title=我国铁路第一长隧高黎贡山隧道进入全面施工阶段 |last= |first= |date=2017-08-28 |website=中国政府网 |publisher=新华社 |access-date=2017-09-03 |quote=}}</ref>。

==地质环境==

[[File:龙瑞高速芒市段08.jpg|thumb]]与[[龙陵县|龙陵]]海拔相差约1,700米，直线距离仅15公里，图为隧道附近的[[高黎贡山|高黎贡山]]南脉以及采用[[架空道路|高架]]和[[隧道|隧道]]盘旋方式到达低海拔芒市坝的[[杭瑞高速|杭瑞高速]]公路]]

高黎贡山隧道地理位置特殊，位于[[歐亞地震帶|喜马拉雅地震带]]，地质条件复杂，具有“三高”<ref group="注">高[[地热能|地热]]、高[[地应力|地应力]]、高[[中国地震烈度表|地震烈度]]</ref>和“四活跃”<ref group="注">活跃的[[新构造期|新构造运动]]、活跃的[[地热能|地热水环境]]、活跃的[[外动力地质作用|外动力地质条件]]、活跃的岸坡浅表改造过程</ref>的特征，施工难度高<ref>{{cite web |url=http://yn.yunnan.cn/html/2016-05/22/content_4349755.htm |title=云南高黎贡山隧道加快建设 |date=2016-05-22 |website=云南网 |publisher=云南日报 |access-date=2017-07-31 |archive-url=https://web.archive.org/web/20160523102321/http://yn.yunnan.cn/html/2016-05/22/content_4349755.htm |archive-date=2016-05-23 |dead-url=yes }}</ref>。隧道穿越[[高黎贡山|高黎贡山]]南延段，属高黎贡山[[古生界|古生界]][[变质岩|变质岩]]紧密褶皱和[[花岗岩|花岗岩体]]高山区，山脉大体为南北走向，地表沟谷纵横，山河相间，地形起伏大<ref name="crecgec.com"></ref>。隧道进口位于[[怒江|怒江]]大峡谷西岸的悬崖峭壁上，隧道出口从1,100余米深的地下穿过[[龙陵县|龙陵县]]后进入[[芒市|芒市]]境内<ref name="YouTube">{{cite AV media |people=中铁隧道集团 大瑞铁路工程项目经理部 |date=2016年7月10日 |title=大瑞铁路高黎贡山隧道建设宣传片 |medium=视频 |language=中文 |url=https://www.youtube.com/watch?v=4B0GTz3BKLc |access-date=2017-07-30 |time=1分42秒 |quote=中铁隧道集团承建的大瑞铁路保瑞段二标高黎贡山隧道，自怒江西岸横穿高黎贡山脉，跨龙陵县全境，进入德宏州芒市境内。}}</ref>，出口处山体陡峭，属凹子地[[古滑坡|古滑坡体]][[地层|地层]]<ref name="yn.xinhuanet.com"></ref>。由于芒市坝平均海拔在640米左右，远低于临近的龙陵县，使得隧道需要从山底无间隙贯穿，隧道施工将穿过18种不同的[[地层|岩层]]、19条[[断层|断层]]<ref name="www.cae.cn"></ref>，其中2条为活动断层<ref name="fei123.com"></ref>，存在高温热害、[[突水|突水]][[突泥|突泥]]、{{le|岩爆|Rock burst}}、[[岩溶|岩溶]]、[[蚀变岩|蚀变岩]]、高[[地应力|地应力]]造成的软岩大变形、[[断层|断层破碎带]]、[[节理密集带|节理密集带]]、[[地震带|高烈度地震带]]、[[放射性|放射性物质]]、[[毒气|有害气体]]、[[滑坡|滑坡]]、[[偏压隧道|偏压]]、[[顺层断层|顺层]]等工程技术难题<ref name="guancha.cn">{{cite web ==设计施工==

[[File:高黎贡山隧道08.jpg|thumb]]预制区]]

高黎贡山的地质条件使得普通钻爆法难以施工，于是高黎贡山隧道主要使用大直径敞开式[[隧道掘進機|隧道掘進機]]施工<ref name="fei123.com"></ref><ref name="www.cae.cn"></ref>，全隧道采用“贯通[[隧道平导|平导]]+1座[[斜井|斜井]]+2座[[竖井|竖井]]”的辅助坑道设置方案<ref name="dh.gov.cn"></ref>。高黎贡山隧道的建设也将是[[中国西南地区|中国西南地区]]铁路修建史上首次采用[[隧道掘進機|隧道掘進機]]施工<ref name="dh.gov.cn"></ref>。2017年5月27日，直径6.39米的隧道掘進機在[[重庆|重庆]]下线，此台隧道掘進機曾参与[[重庆轨道交通6号线|重庆轨道交通6号线]]的修建，累计掘进6,652米<ref name="dehong.gov.cn">{{cite web |url=http://www.dehong.gov.cn/news/dh/content-16-35541-1.html |title=大瑞铁路高黎贡山隧道将添一“重器”！ |author=人民网 |date=2017-06-02 |website=www.dehong.gov.cn |publisher=德宏网 |access-date=2017-07-31}}</ref>。该台“彩云一号”为中国首台再制造隧道掘进机，2017年8月28日在高黎贡山隧道顺利始发，标志高黎贡山隧道正式进入全面施工阶段<ref name="2017-08/28">{{cite web |url=http://news.xinhuanet.com/fortune/2017-08/28/c_129690968.htm |title=我国铁路第一长隧高黎贡山隧道进入全面施工阶段 |author=齐中熙、丁怡全 |date=2017-08-28 |website=新华网 |publisher=新华社 |access-date=2017-09-03 |quote=}}</ref>。2017年11月27日，“彩云一号”掘进机开始掘进，每个月能掘进600米左右，隧道建设再次提速，同时直径达9.03米的“彩云号”掘进机也已在施工现场进行安装调试，计划将于2018年初投入正洞施工<ref>{{cite web |url=https://www.chinanews.com/cj/2017/11-27/8386515.shtml |title=大瑞铁路高黎贡山隧道建设再提速 |author1=陈静 |author2=何平阳 |date=2017-11-27 |website=www.chinanews.com |publisher=中国新闻网 |access-date=2017-12-05 |quote=}}</ref><ref>{{cite web |url=http://news.sina.com.cn/c/2017-11-27/doc-ifypacti8603479.shtml |title=新闻背景 高黎贡山隧道 中国铁路第一长隧 |author= |date=2017-11-27 |website=news.sina.com.cn |publisher=央视网 |access-date=2017-12-05 |quote=}}</ref>。

设计上，高黎贡山隧道在正洞的洞口设2根[[抗滑桩|抗滑桩]]，沿线路基设105根[[锚固桩|锚固桩]]。2016年5月1日出口平导进洞，[[隧道平导|平导隧道断面]]为37平方米；2016年8月1日出口正洞进洞，[[隧道正洞|正洞隧道断面]]为179平方米。平导与正洞的洞内[[隧道掌子面|掌子面]]设计均为{{le|围岩|Wall rock|V级围岩}}，都采用[[台阶法|台阶法]]挖掘。<ref name="yn.xinhuanet.com">{{cite web |url=http://www.yn.xinhuanet.com/2016original1/20161025/3501950_c.html |title=大瑞铁路高黎贡山隧道工程顺利推进 |author=刘馨蔚 |date=2016-10-25 |website=www.yn.xinhuanet.com |publisher=新华网 |access-date=2017-07-31}}</ref>2017年6月30日，高黎贡山隧道出口正洞双线段贯通，较预期时间提前两个月，使[[隧道掘进机|隧道掘进机]]能够按预期进场施工<ref>{{cite web|url=http://www.dh.gov.cn/Web/_F0_1_170704111438d682b199.htm|title=大瑞铁路高黎贡山隧道出口正洞双线段贯通|author=张仁韬、段晨阳、何宗政、王亚峰|date=2017-07-04|website=www.dh.gov.cn|publisher=德宏团结报|access-date=2017-08-01}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。截至2017年9月，高黎贡山隧道1号斜井施工已突破1,000米，预计2019年5月进入正洞；1号竖井施工突破500米，预计2018年1月进入正洞<ref name="2017-08/28"></ref>。

高黎贡山隧道进口里程为 D1K192+302 ，进口紧邻[[怒江特大桥|怒江特大桥]]，[[怒江站|怒江站]]部分进入隧道进口段，进口段形成双线及三线车站隧道498米；出口里程 D1K226+840 ，位于龙陵县南凹子地，[[龙陵站|龙陵站]]部分进入隧道出口段，出口段形成双线车站隧道538米。全线为直线，隧道内线路[[纵坡|纵坡]]为人字坡，进口段除198米的怒江车站范围为平坡外，其余地段上坡21.6公里，最大线路坡度23.5‰；出口段除240米的龙陵车站范围为平坡外，其余地段下坡长12.5公里，最大线路坡度9‰。高黎贡山隧道进口 D1K192+302 至 D1K213+500 段共21.198km采用[[鑽炸法|钻爆法]]施工，隧道出口 D1K213+500 至 D1K226+840 段共13.34公里以及平导 PDZK215+250 至 PDZK226+838 段共11.588公里主要使用[[隧道掘進機|隧道掘進機]]施工，其中隧道掘進機施工困难段及洞口段均采用钻爆法施工，隧道出口段[[隧道正洞|正洞]]的隧道掘進機直径为9.03m，[[隧道平导|平导]]的隧道掘進機直径为6.36m。<ref name="crecgec.com">{{cite web |url=http://www.crecgec.com/article-66415-1-1.html |title=中铁隧道集团有限公司大瑞铁路工程项目经理部二分部项目轨线材料招标公告 |author=中铁隧道集团有限公司大瑞铁路工程项目经理部二分部 |date=2017-04-09 |website=www.crecgec.com |publisher=中铁鲁班商务网 |access-date=2017-07-31}}</ref><ref name="《隧道建设》">{{cite web |url=http://m.chaoxing.com/mqk/read_38502727e7500f264d43b11113ae452dac6b0826c7d418281921b0a3ea255101fc1cf1fbb4666ae688759607a10f5d7c58d85279554aceea0daba7fb27e23e2ba2bd955bac1a3bbf174f3255f0fa88d2 |title=高黎贡山隧道复杂地质条件下敞开式TBM施工关键技术研究 |author=宋法亮、赵海雷 |publisher=《隧道建设》2017年第A1期 |access-date=2017-08-01}}</ref>
