'''高圓宮憲仁親王'''（{{bd|1954年|12月29日|2002年|11月21日||catIdx=Takamado, Prince}}）<ref>1954年（昭和29年）12月29日[[宮内廳|宮内廳]]告示第8號「崇仁親王妃百合子殿下親王御出產」</ref>是一位已逝世的[[日本皇室|日本皇室]]成員，為[[三笠宮崇仁親王|三笠宮崇仁親王]]與王妃[[崇仁親王妃百合子|百合子]]所生的第三個兒子。也為[[昭和天皇|昭和天皇]]的侄子、[[明仁天皇|明仁天皇]]的堂弟。[[身位|身位]]是[[親王_(東亞)#日本的親王|親王]]，[[御印|御印]]為[[柊樹|柊]]。勳等是大勳位，學位為[[學習院大學|學習院大學]][[法學士|法學士]]。

[[高圓宮|高圓宮]]的宮號由[[昭和天皇|昭和天皇]]所授予，原因是父親宮號[[三笠宮|三笠宮]]的由來[[三笠山|三笠山]]，與同位於[[奈良縣|奈良縣]][[奈良市|奈良市]]的[[高圓山|高圓山]]距離相近，因而命名。

== 生平 ==
1954年（[[昭和|昭和]]29年）出生，是[[三笠宮崇仁親王|三笠宮崇仁親王]]和[[三笠宮妃百合子|三笠宮妃百合子]]的幼子。就讀松濤幼稚園、學習院初等科、學習院中等科、學習院高等科，高校時期參加攝影社，之後攝影也成為生活的嗜好之一。

1978年自[[學習院大學|學習院大學]]法學部畢業後，前往[[加拿大|加拿大]]的[[皇后大學_(加拿大)|皇后大學]]留學，至1981年才返國。隨後，於與[[外務省|外務省]]相關的[[國際交流基金|國際交流基金]]工作。

1984年4月23日，於加拿大駐日本大使館舉辦的酒會上，認識了[[實業家|實業家]][[鳥取滋治郎|鳥取滋治郎]]的長女[[憲仁親王妃久子|鳥取久子]]，5月上旬時父親崇仁親王即向鳥取久子確認是否對憲仁親王存有好感。5月20日，鳥取久子以[[英語|英語]]「Will you marry me?」向憲仁親王求婚，得到親王的承諾回應。

8月1日，雙方結為連理獲得[[皇室會議|皇室會議]]的承認，9月17日進行[[納采之儀|納采之儀]]，12月6日舉行結婚的相關儀式，同日也代表[[高圓宮|高圓宮]]家正式成立。之後，共育有三名女兒，分別是[[承子女王|承子女王]]、[[千家典子|千家典子]]與[[守谷絢子|守谷絢子]]。

此外，久子王妃的曾祖母和[[貞明皇后|貞明皇后]]為[[表親|表親]]關係。

雖然在皇位繼承順位上與眾皇族相比，排位相對後方，不過也因如此個性較不為拘謹，自認是「皇室的發言人」，曾上電視拉近國民與皇室的關係，或發表自身的看法，對於面對新時代的皇室貢獻良多。

另外，憲仁親王也投入於日本[[足球|足球]]的發展當中，1987年起擔任[[日本足球協會|日本足球協會]]的名譽總裁。由於擅長[[英語|英語]]與[[西班牙語|西班牙語]]，因此對[[南美洲足球協會|南美洲足球協會]]（CONMEBOL）和日本足球協會的合作提供許多幫助。在申辦[[2002年世界盃足球賽|2002年世界盃足球賽]]期間時，也不遺餘力地接待來訪的貴賓，積極奔走。

2001年7月，就任為社團法人日本水難救濟會的名譽總裁。2002年，[[2002年世界盃足球賽|2002年世界盃足球賽]]正式舉辦，5月底夫妻抵達[[韓國|韓國]]進行官方訪問，並出席開幕式。此次出訪也是[[二戰|二戰]]後首次日本皇族以官方訪問的名義前往韓國，在這之前[[雍仁親王妃勢津子|秩父宮妃勢津子]]、[[高松宮宣仁親王|高松宮宣仁親王]]與王妃[[宣仁親王妃喜久子|喜久子]]、[[三笠宮崇仁親王|三笠宮崇仁親王]]與王妃[[崇仁親王妃百合子|百合子]]、[[寛仁親王|寬仁親王]]曾經非官方形式訪問。當屆憲仁親王共觀看了19場的賽事。<ref>{{cite web|url=http://jeremy.footballjapan.jp/2002/11/post_0d27.html|title=高円宮さまの急逝：日本にとって大きな損失|work=ジェレミー・ウォーカーの A View From a Brit|date=2002-11-24|accessdate=2012-08-03}}</ref>

憲仁親王對[[足球|足球]]的熱愛與投入廣為人知，被期待在壯年貴族當中能有活躍的表現。同年11月21日，於加拿大駐日本大使館正要練習[[壁球|壁球]]時，身體[[心室顫動|心室顫動]]引起[[心衰竭|心衰竭]]而倒地。

隨後，被直接送往[[慶應義塾大學醫院|慶應義塾大學醫院]]救治，但已經進入[[心肺停止|心肺停止]]的狀態。進行復甦術後一度具有心跳，不過至半夜狀況惡化，在得到久子妃的同意下取出人工心肺裝置，確認無生命反應後宣告逝世，享年47歲。

11月29日舉行[[斂葬之儀|斂葬之儀]]（皇族喪禮），在[[落合齋場|落合齋場]]進行[[火葬|火葬]]，埋葬於[[豐島岡墓地|豐島岡墓地]]。喪禮動員了學習院初等科、中等科的學生。另外，因為[[世界盃足球賽|世界盃足球賽]]的影響，一般民眾與足球迷也特別關注。

憲仁親王生前所擔務的眾多職位，主要由久子妃繼承，例如[[日本足球協會|日本足球協會]]的名譽總裁一職。

此外，自憲仁親王驟逝以來，[[厚生勞動省|厚生勞動省]]和[[消防廳|消防廳]]針對心室顫動採取相關對應措施。逝世兩年後，2004年開始推廣、普及[[自動體外心臟去顫器|自動體外心臟去顫器]]（AED）的使用，[[去顫|去顫]]以於緊急時刻即時治療。

== 人物 ==
* 除了體育之外，對[[芭蕾舞|芭蕾舞]]等相關造詣也有一定的深度，曾著和芭蕾舞者[[森下洋子|森下洋子]]的對談集「謝幕的此端」（{{lang-ja|カーテンコールのこちら側}}）。 
* [[攝影|攝影]]是興趣之一，此外還包括蒐集[[根付|根付]]、[[社交舞|社交舞]]等，
* 昭和天皇在世時，與[[皇太子德仁親王|浩宮德仁親王]]有如兄長的存在<ref>宮内廳・平成14年皇太子妃誕生日會見</ref>。另外，1987年浩宮德仁親王和[[皇太子德仁親王妃雅子|小和田雅子]]尚未結婚時，就邀請憲仁親王至自邸作客，可知交情深厚。過世前皇太子夫妻依然會至高圓宮邸拜訪，於結婚後仍有交情。
* 盡力於振興體育和國際交流中，特別在[[足球|足球]]方面，但對小型比賽也未忽視，自己組織足球隊「王子高圓Eleven」（{{lang-ja|プリンス高円イレブン}}），每年都會參加比賽<ref>[[岡野俊一郎|岡野俊一郎]]日本足球協會名譽會長談</ref>。在網路上又被稱為「足球宅殿下」（{{lang-ja|サカヲタ殿下}}）及「踢球殿下」（{{lang-ja|玉蹴り殿下}}）的親近暱稱。

== 軼事 ==
* 於皇族中少見地擔任[[國際交流基金|國際交流基金]]的職員，與其他一般職員一同工作。
* 在國際交流基金事務局被暱稱為「小憲」（{{lang-ja|りのちゃん}}）。
* 於[[朝日電視台|朝日電視台]]的節目[[NEWS_STATION|NEWS STATION]]中登場，是首位在[[現場直播|現場直播]][[電視節目|電視節目]]出場的日本皇族。主持人[[久米宏|久米宏]]問到「為何答應上電視節目？」時，以「今天是什麼都可以說的日子（4月1日為[[愚人節|愚人節]]），所以覺得肯定可以出場。」回答。
* 小時候喜歡收看由[[岡野俊一郎|岡野俊一郎]]解説的每週足球情報節目「三菱鑽石・足球」。中學時，每天必定花一小時踢足球。
