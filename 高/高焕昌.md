'''高焕昌'''（{{bd|1924年||2012年|10月30日|catIdx=}}）[[河北省|河北省]][[深县|深县]]（今[[深州市|深州市]]）人，[[中国人民解放军中将|中国人民解放军中将]]<ref name=gao>{{Cite web |url =  http://mil.news.sina.com.cn/2012-11-20/0708707201.html|title =  新疆军区原司令员高焕昌逝世|author =  |date = 2012-11-20 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref><ref name=ifeng>{{Cite web |url = http://news.ifeng.com/gundong/detail_2012_11/12/19038596_0.shtml |title = 新疆军区原司令员高焕昌同志逝世 |author =  |date =2012-11-12  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1924年9月生于深县。1939年8月参加深县抗日武装游击队。1940年5月参加[[八路军|八路军]]。1942年2月加入[[中国共产党|中国共产党]]。[[抗日战争|抗日战争]]时期，历任八路军冀中警备旅2团侦察连通信员、晋绥二支队班长、合作社主任。先后参加了冀中“[[五一反扫荡|五一反扫荡]]”、冀鲁豫临南战役等战役战斗<ref name=gao/><ref name=ifeng/>。

[[第二次国共内战|第二次国共内战]]期间至[[中华人民共和国|中华人民共和国]]初期，历任[[山西省|山西省]][[吕梁军区|吕梁军区]]11团通信参谋，八路军[[359旅|359旅]]719团参谋，山西省[[岚县|岚县]]晋绥军政干部学校学员，[[西北野战军|西北野战军]]2军5师15团作战参谋、作战股长、副参谋长。参加了[[延安保卫战|延安保卫战]]、[[青化砭战役|青化砭战役]]、[[羊马河战役|羊马河战役]]、[[沙家店战役|沙家店战役]]、[[扶眉战役|扶眉战役]]等战役战斗<ref name=gao/><ref name=ifeng/>。

中华人民共和国初期，到新疆工作，历任[[中国人民解放军南疆军区|中国人民解放军南疆军区]]军训科科长，4师10团团长、11团团长、副师长、师长，[[中国人民解放军北疆军区|中国人民解放军北疆军区]]副参谋长，中国人民解放军南疆军区副司令员、司令员，[[中国人民解放军新疆军区|中国人民解放军新疆军区]]司令员<ref name=gao/><ref name=ifeng/>。

高焕昌是[[中国共产党第十三届中央委员会|中国共产党第十三届中央委员会]]委员，[[第六届全国人大代表|第六届全国人大代表]]，[[第八届全国政协委员|第八届全国政协委员]]。1955年获二级独立自由勋章和二级解放勋章<ref name=gao/><ref name=ifeng/>。1988年获授中将军衔，获独立功勋荣誉章<ref name=gao/><ref name=ifeng/><ref>{{citenews|author=徐平|title=重新实行军衔制|url=http://chn.chinamil.com.cn/jfdg/2011-07/13/content_4473591.htm|accessdate=2012-05-05|newspaper=中国军网|deadurl=yes|archiveurl=https://web.archive.org/web/20130511192908/http://chn.chinamil.com.cn/jfdg/2011-07/13/content_4473591.htm|archivedate=2013-05-11}}</ref>。1998年9月离职休养<ref name=gao/><ref name=ifeng/>。

2012年10月30日，高焕昌在[[乌鲁木齐|乌鲁木齐]]病逝，享年89岁<ref name=gao/><ref name=ifeng/>。
