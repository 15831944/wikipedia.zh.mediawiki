'''高野麻里佳'''（{{bd|1994年|2月22日|catIdx=Kono Marika}}<ref>{{Cite web|url=https://twitter.com/marika_0222|title=本人推特（簡介欄）|accessdate=2016-10-08|language=ja}}</ref>）是[[日本|日本]]的[[女性|女性]][[聲優|聲優]]。[[Mausu_Promotion|Mausu Promotion]]所属。[[東京都|東京都]]出身。[[聲優組合|聲優組合]]「TEAM OHENRO。」、「{{tsl|ja|イヤホンズ|EARPHONES}}」的成員。

== 簡介 ==
代代木動畫學院高等部大宮校畢業<ref>{{Cite web|url=http://ameblo.jp/kouno222marika/entry-11774465763.html|title=コミュニケーション講座|accessdate=2016-10-08|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140513120517/http://ameblo.jp/kouno222marika/entry-11774465763.html|archivedate=2014-05-13}}</ref><ref name="seiyuzukan">{{Cite web | publisher= ダ・ヴィンチニュース |url = http://ddnavi.com/news/191491/a/|title = 「高野麻里佳」新人声優のインタビュー＆ミニグラビア！【新人声優図鑑】|accessdate=2016-10-08|language=ja}}</ref>。MAUSU PROMOTION附屬養成所第27期畢業。2012年入所，2014年開始為[[Mausu_Promotion|Mausu Promotion]]正式成員。興趣是游泳、籃球、搖滾、{{link-ja|搖頭|ヘッドバンギング}}<ref>{{Cite web|url=https://mausu.net/talent/kouno-marika.html|title=高野麻里佳|work=マウスプロモーション|accessdate=2016-10-08|language=ja}}</ref>。

2014年，在[[ufotable|ufotable 德島Studio]]以[[四國八十八箇所|四國八十八箇所]]為主題所製作的情報節目《{{tsl|ja|おへんろ。|遍路。}}》中擔任主角之一，並與該節目的主要演員[[山下七海|山下七海]]、{{tsl|ja|江原裕理}}組成聲優組合「TEAM OHENRO。」，於同年10月8日以該組合名義發行前述節目的主題曲「{{lang|ja|千と二百の物語}}」CD出道<ref>{{Cite web|url=http://animeanime.jp/article/2014/07/12/19422.html|title=ufotable制作の個性派アニメ「おへんろ。～八十八歩記～」主題歌CD発売、上映イベント決定|date=2014-07-12|work=アニメ!アニメ!|accessdate=2016-10-08|language=ja}}</ref>。

2015年，在電視動畫《[[那就是聲優！|那就是聲優！]]》飾演主要角色，並與一同演出的[[高橋李依|高橋李依]]、[[長久友紀|長久友紀]]組成該作品的衍生聲優組合「EARPHONES」，於同年6月18日發售單曲「{{lang|ja|耳の中へ}}」<ref name="biography" />。

2016年，以EARPHONES成員身份在《[[Animelo_Summer_Live#2016年|Animelo Summer Live 2016 刻-TOKI-]]》的第二天（8月27日）演出<ref>{{Cite web|url=http://anisama.tv/2016/news/detail.php?id=15|title=出演アーティスト追加発表！|work=Animelo Summer Live 2016 刻-TOKI-|date=2016-05-13|accessdate=2016-10-08|language=ja}}</ref>。

== 人物 ==
家中三姐妹的二女<ref>{{Cite web|url=https://twitter.com/marika_0222/status/358246934999339008|title=本人推特|date=2013-07-20|accessdate=2016-10-08|language=ja}}</ref>，其姐[[高野璃奈|高野璃奈]]是女演員和舞者<ref>{{Cite web|url=https://twitter.com/marika_0222/status/539071432102146048|title=本人推特|date=2014-12-01|accessdate=2016-10-08|language=ja}}</ref>。

小學時朗誦練習中調整各種角色自得其樂並獲得母親的讚賞後，開始覺得這是好玩的事情。當認識到《[[神奇寶貝_(動畫)|神奇寶貝]]》的皮卡丘與《[[ONE_PIECE_(動畫)|ONE PIECE]]》的[[多尼多尼·喬巴|多尼多尼·喬巴]]配音的聲優同是[[大谷育江|大谷育江]]後，覺得聲優很厲害並萌生了從事聲優這一職業的意識<ref name="seiyuzukan"/>。

喜歡的食物是牛肉乾，前往錄音現場等地方時總是會放在自己的包包內。
