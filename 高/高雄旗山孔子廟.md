'''高雄旗山孔子廟'''位於[[臺灣|臺灣]][[高雄市|高雄市]][[旗山區|旗山區]][[鼓山公園|鼓山公園]]（原稱中山公園）內，佔地約有4.1[[公頃|公頃]]，為臺灣、也是全東南亞佔地面積最大的[[孔子廟|孔子廟]]<ref name="高雄市政府觀光局"/><ref name="蘋果日報"/>。旗山孔子廟在[[2010年中華民國縣市改制直轄市|2010年高雄市縣市合併]]前為[[高雄縣|高雄縣]]境內唯一的孔子廟，於縣市合併後與原高雄市[[左營區|左營區]][[高雄左營孔子廟|孔子廟]]形成鬧雙包的現象，因此目前每年舉行的祭孔大典，改成兩間輪流舉辦。<ref name="蘋果日報"/>

== 沿革 ==
旗山孔子廟的前身為[[臺灣日治時期|臺灣日治時期]]興建的[[旗山神社|旗山神社]]，後於[[太平洋戰爭|太平洋戰爭]]結束後神社建築遭到破壞。直到1983年由時任高雄縣縣長[[蔡明耀|蔡明耀]]推動之下<ref name="自由時報2"/>，將旗山神社改建為高雄縣孔子廟，歷經兩年施工其於1985年完工啟用。<ref name="李文環"/>

[[2010年中華民國縣市改制直轄市|2010年高雄市縣市合併]]後，使得合併後的高雄市擁有原高雄市設立的左營孔子廟，以及原高雄縣的旗山孔子廟，為此合併後的高雄市政府將每年都會舉辦的祭孔大典改成兩邊輪流舉辦，以解決孔子廟鬧雙包的狀況。<ref name="蘋果日報"/>

2016年2月高雄市[[美濃區|美濃區]]發生[[2016年高雄美濃地震|芮氏規模6.6地震]]，造成[[臺灣南部|臺灣南部]]地區多處災情，旗山孔子廟也因[[地震|地震]]發生而受損，供俸孔子牌位的大成殿有18根柱梁柱[[油漆|油漆]]碎裂掉落，南面牆壁大片剝落並[[鋼筋|鋼筋]]明顯外露，外圍設施則花格磚牆倒塌、地板隆起等災情。<ref name="自由時報"/>

旗山區公所於地震災後會同高雄市土木技師公會人員前往旗山孔子廟進行災害受損鑑定。經建築技師評估後認定大部分損壞為非結構性損壞，因此[[高雄市政府|高雄市政府]]民政局動用災害準備金，於2016年4月底發包修復旗山孔子廟，總修復經費約為[[新臺幣|新臺幣]]120萬元，修復工程於該年8月上旬，928[[教師節|教師節]]之前完工重新啟用。<ref name="自由時報"/>

== 建築 ==
旗山孔子廟方位座西朝東依山勢興建，因此孔子廟佈局上與以往傳統設計不同，是臺灣所有的孔廟中較特殊的規劃<ref name="旗山區公所"/><ref name="客家委員會"/>。旗山孔子廟有兩座櫺星門、大成門、大成殿、東西廡、崇聖祠等建築，但無萬仞宮牆與泮池的設計<ref name="旗山區公所"/>。自旗山市區的山腳往旗山孔子廟須經由500階的階梯前往<ref name="美美網"/>，至第一平臺，進入第一座櫺星門，設有五開牌坊，門採七開間，門楣橫書「道貫古今」，為[[中華民國總統|中華民國總統]][[蔣經國|蔣經國]]篆書，左右兩側的門牌，以石板封飾並刻有旗山孔子廟之建廟歷史。通過第一座櫺星門後進入大進階，左右側各有一條階梯進入第二平臺及孔子廟主體建築。<ref name="旗山區公所"/>

主體建築遵循[[宋朝|宋朝]]建築樣式而建，以黃色琉璃瓦、朱紅樑柱<ref name="高雄市政府觀光局"/>，以及白玉欄柵襯飾，包含櫺星門、大成門、大成殿、東西廡、崇聖祠等設施。大成殿位於獨立臺基上，屋頂採重簷歇山式，簷下以柱列支撐，形成柱廊。前方設有丹墀，是祭孔典禮時跳[[佾舞|佾舞]]的地方。<ref name="旗山區公所"/>

== 祭典 ==
由於旗山孔子廟因占地廣大之因素，因此每年孔子誕辰的[[祭孔|釋奠大典]]中所跳的佾舞，是採用[[天子|天子]]專用的[[八佾舞|八佾舞]]。旗山孔子廟的八佾舞共有8行8列共64人，與其他孔子廟的六佾舞，場面較為盛大，並且旗山孔子廟的釋奠大典採用改良式拔智慧毛、過聰門活動，以[[毛筆|毛筆]]取代傳統牛毛。<ref name="美美網"/>
