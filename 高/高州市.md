'''高州市'''（Gaozhou，[[邮政式拼音|邮政式拼音]]：{{lang|la|'''Kochow'''}}）是[[中华人民共和国|中国]][[广东省|广东省]]西南部的[[县级市|县级市]]，由[[茂名市|茂名市]]代管，位于[[鉴江|鉴江]]中上游，西北与[[广西壮族自治区|广西壮族自治区]]毗邻，历史上曾是粤西地区的政治、经济、军事、文化中心。

高州是广东省历史文化名城及文化旅游目的地<ref>{{Cite web|url=http://www.gaozhou.gov.cn/gaozhou/jrgz/ViewTextInfo.aspx?InfoID=48349|title=高州市城市总体规划（2015-2035 年）公示 - 高州市人民政府公众网|accessdate=2018-01-26|work=www.gaozhou.gov.cn}}</ref>，广东十大传统美食之乡之一<ref>{{Cite web|url=http://jiuban.moa.gov.cn/fwllm/qgxxlb/gd/201403/t20140319_3821267.htm|title=广东评出“十大传统美食之乡” 荔湾番禺榜上有名|accessdate=2018-08-23|work=jiuban.moa.gov.cn}}</ref>，中国南方重要的水果生产基地，盛产荔枝、龙眼、香蕉。

===新石器时代===
高州境内人类居住的历史可以上溯到新石器时代，在南部的[[石鼓镇|石鼓镇]]祥山茶亭岭出土的两件穿孔石球是目前高州境内发现最早的石器工具，为当时旧石器时代晚期制作和使用的捕猎工具。在[[新垌镇|新垌镇]]红珠岭和[[云潭镇|云潭镇]]的黄鳝坡上，亦发现新石器时代文化遗址，出土有石斧、石锛、砺石等磨制石器。此外，在曹江、泗水、根子、谢鸡等镇发现新石器文化遗存点共11处，出土一批石器工具。这些出土文物，证明了在4500年前的新石器时代，高州的土著民族已生活在境内的大片土地上，创造了高州古代人类文明<ref>{{Cite web |url=http://www.gd-info.gov.cn/shtml/gzs/lanmu01/lanmu02/2009/07/06/7502.shtml |title=高州县历史沿革，高州市地情网 |access-date=2015-04-05 |archive-url=https://web.archive.org/web/20140316194811/http://www.gd-info.gov.cn/shtml/gzs/lanmu01/lanmu02/2009/07/06/7502.shtml |archive-date=2014-03-16 |dead-url=yes }}</ref>。

===先秦至明清===
先秦时期，高州归属于西瓯、骆越部族。这两大部落都是中国南方百越民族的组成部分，统称为南越。春秋战国时期，高州属楚地。秦统一岭南后，设南海、桂林和象郡，高州一带区域当时处于桂林和象郡之间，即中北部属桂林郡，南部属象郡。 

汉代，岭南地区重新分设为南海、交趾、合浦等7郡，高州属合浦郡。元鼎六年（前111年），合浦郡下设高凉县，县治在现在的[[长坡镇|长坡镇]]旧城村。今高州长坡以南当时属合浦郡高凉县，余部属苍梧郡端溪县。汉灭南越国后，部分南越民族开始汉化，原南越民族则史称为俚族。

[[南朝梁|南朝梁]]大通二年（528年），在俚人聚居地区中心置立高州，此为高州之始，州治设于高凉郡（今[[长坡镇|长坡镇]]旧城村）。从南朝至唐代，先后在境内设立电白郡和良德县、茂名县，又设置高州和潘州。至此，在今高州境内，同时存在着两州（高州、潘州）、三县（茂名、良德、电白）的行政治所，高州治所也历经三址二迁，隋治高凉县（今[[阳江市|阳江市]][[阳东县|阳东县]]境内），唐贞观二十三年迁治良德县（原[[东岸镇|东岸镇]]良德一带，现为[[高州水库|高州水库]]），唐大历十一年迁治电白县（今[[长坡镇|长坡镇]]旧城村）。

宋开宝五年（公元972年），合并州县机构，地方行政区划进行了较大的调整，潘州并入高州，良德县并入电白县，境内仍保留有高州和茂名、电白一州两县建制。元至元十七年（1280年）改高州为高州路，明洪武元年（1368年）改路为府，高州路为高州府。洪武七年降为州，茂名县撤销并入高州。洪武九年高州复府，其后由治电白县城（今长坡镇旧城村）迁治茂名县城（今高州市城区）。明成化三年（公元1467年），电白县迁神电卫（今[[电白区|电白区]][[电城镇|电城镇]]）。原高州及电白治所附近一带地域，划为茂名县管辖，这便成了今天高州的基本地域。至此高州府领茂名县、电白县、信宜县、化州及其辖县吴川、石城（今[[廉江|廉江]]）等“一州五县”。

清代沿明制，高州府治、茂名县治以及高州总兵机构同设于今高州市城区一带。由于高州辖地较大，地理位置又处于战略要冲地带，因此便把高州府称为广东下四府（高州府、雷州府、廉州府、琼州府）之首府。

===民国时期===
中华民国元年（1912），撤高州府，广东省派出高州绥靖处，驻茂名县高州城，领茂名、信宜、电白、化县、吴川、廉江六县。民国三年，撤高州绥靖处，设高雷道，治所在茂名县，辖原高州府、雷州府、阳江直隶州属下11县。九年（1920），撤道，派出高雷善后处，治所与高雷道同。民国十四年（1925），撤善后处，派出南路行政公署，驻茂名县。十五年，又改为南路绥靖专员公署。民国二十五年（1936）11月，广东派第七行政督察专员公署驻茂名县，辖茂名、信宜、电白、化县、吴川、廉江、阳江、阳春和梅菉管理局。民国二十七年派出南路行署驻茂名县，管理第七、第八行政督察专员公署。民国三十六年（1947），原第七行政督察专员公署改为第八专署行政督察区，治所和辖区不变<ref>[http://www.mm111.net/2009/0731/354307.html 茂名高州市近代建置沿革，茂名新闻网] {{webarchive|url=https://web.archive.org/web/20150411021032/http://www.mm111.net/2009/0731/354307.html |date=2015-04-11 }}</ref>。 

===中华人民共和国时期===
1949年10月底，茂名县属南路专区。1950年属高雷专区。1952年属粤西行政区。1956年属湛江专区。

1958年8月，原茂名县南部成立茂名工矿区市（茂名市）；1958年10月，茂名县南部的原公馆乡和袂花乡、鳌头乡的部分地域划入茂名工矿区市；将原小良乡划入电白县；将原石鼓乡的南盛、鳌头乡的兰石和龙首乡划入化州县。同年11月，茂名县与信宜县合并为茂信县，12月茂信县定名为高州县。1961年4月茂信县分署，原茂名部分仍称高州县，原信宜县管辖的深镇、古丁、马贵三个公社划归高州县管辖。

1983年，湛江专区撤销，高州县由已升格成地级市的茂名市管辖。

1987年，分界公社的山阁、烧酒、霞池、黄杰、禄村、合益、金塘岭、福居等八个大队划归茂名市茂南区，成立山阁公社（现为[[茂名市|茂名市]][[茂南区|茂南区]][[山阁镇|山阁镇]]）。

1993年6月，撤县设市，高州县改为高州市。同年12月正式举行撤县建市挂牌仪式。

== 地理 ==
高州境内主要河流有[[鉴江|鉴江]]、曹江、大井河、凌江、黄塘河、南塘河，河网密度0.28公里/平方公里。主要山峰有界山棉被顶，东部的平云山。年平均气温22.8℃，年均降水量1794.1毫米。主要自然资源有[[金|金]]、[[铝|铝]]、[[银|银]]、[[锌|锌]]、[[硫铁|硫铁]]、[[铁|铁]]、[[高岭土|高岭土]]、[[辉长岩|辉长岩]]、[[石灰石|石灰石]]、[[绢云母|绢云母]]、[[滑石|滑石]]、[[石英|石英]]、[[磷|磷]]、[[煤|煤]]、[[膨润土|膨润土]]、[[钾长石|钾长石]]、[[漂白土|漂白土]]、[[玛瑙|玛瑙]]、[[水晶|水晶]]等，其中[[高岭土|高岭土]]的质量和储量均居全国首位。主要[[野生动物|野生动物]]有[[山猪|山猪]]、[[黄猄|黄猄]]、[[果子狸|果子狸]]、[[穿山甲|穿山甲]]、[[乌龟|乌龟]]等。主要自然灾害为[[台风|台风]]、[[暴雨|暴雨]]、[[寒露风|寒露风]]、[[霜冻|霜冻]]。主要[[中药|中药]]材有[[砂仁|砂仁]]、[[益智|益智]]、[[巴戟|巴戟]]、[[田七|田七]]、[[白术|白术]]、[[防党|防党]]、[[熟地|熟地]]、[[霍香|霍香]]、[[琥珀|琥珀]]等等。

== 文化 ==

=== 方言 ===
高州境内通用粤语（白话），其他方言有客家话（涯话）和闽南话（俚话）。高州白话，属于汉藏语系下的汉语族[[粤语|粤语]]支高阳片；高州涯话，属于汉藏语系汉语族[[客语|客语]]支，之前属于不分片客语，后来经比较可算粤台片[[涯话|涯话]]小片，使用者散布在高州东部和东南部、东北部的泗水、谢鸡、根子、分界、新垌、云潭、大坡、马贵、古丁、深镇等镇。

=== 木偶戏 ===
高州木偶戏，民间称“傀戏”、“傀仔戏”，由明朝万历年间从福建传入的布袋戏发展而起，20世纪七八十年代盛极一时，木偶戏班发展到两千多班，年演出一万多场，成为广东演出单位中演出剧种最多的文艺团体<ref>{{Cite web|url=http://www.chinaqw.com/news/2006/0612/68/32054.shtml|title=高州木偶戏名扬海内外(图)|accessdate=2018-01-26|work=www.chinaqw.com}}</ref>。2003年，高州被评为中国民族民间艺术(木偶)之乡，作为木偶戏的重要分支，木偶戏（高州木偶戏）被入选为第一批国家级非物质文化遗产。

== 经济 ==

=== 农业 ===
高州的三高农业发展迅猛，农业总产值连续多年名列广东省各县（市）第一，其中香蕉产量占全国1/6。其主要的土产特品品种有：
* [[荔枝|荔枝]]：[[白糖罂|白糖罂]]、[[妃子笑|妃子笑]]、[[黑叶|黑叶]]、[[白腊|白腊]]、[[桂味|桂味]]；
* [[龙眼|龙眼]]：[[储良|储良]]、[[石硖|石硖]]；
* [[香蕉|香蕉]]：[[遁地雷|遁地雷]]。
* [[缅茄|缅茄]]：[[中国|中国]]唯一的一棵珍贵的缅茄树，国家一级古树，树龄将近500年。

=== 商圈 ===

==== 中山路商圈 ====
粤西地区历史最悠久的商圈之一。广义的中山路商圈以原百货大楼（已倒闭）为核心，除了中山路商业步行街，还包括附近的永镇街、解放街、南华路一带。中山路以往就是高州城区重要的商业活动中心，全长884米，占地4万多平方米，商业氛围浓厚，以浓郁的岭南骑楼风格为特色，被评为第三批广东省历史文化街区<ref>{{Cite web|url=http://www.gdcic.net/HTMLFile/shownews_messageid=122692.html|title=广东建设信息网(行业服务版)|accessdate=2017-11-23|work=www.gdcic.net}}</ref>。

==== 金港湾商圈 ====
位于城区文明路，亦是目前人气最旺的商业区，商业配套相对集中完善，以文明路金港湾百货为核心，扩至广场路、公园路、教育路一带，汇集大型超市、品牌服饰、数码产品、金器首饰、休闲饮食、电影娱乐等商业业态。

==== 太阳城商圈 ====
2010年后随着城市南拓新兴的商圈，以高凉路太阳城广场为核心，涵盖超市、服装、饮食、儿童乐园以及精品书店等业态。

==== 桂圆食坊 ====
位于市区桂圆东路两侧，于2016年元旦正式开业，定位为粤西地区最大的美食城，占地面积近5万平方米，一期工程汇聚超过200间食肆、商铺、酒吧、小吃档。
