'''高雄市政府'''(簡稱：'''高市府''')是[[中華民國|中華民國]][[高雄市|高雄市]]的最高[[行政機關|行政機關]]，在[[中華民國政府|中華民國政府架構]]中屬於[[直轄市|直轄市]][[政府|政府]]自治位階，自治監督機關為[[中華民國行政院|行政院]]，並轄有31個一級機關、138個二級機關、35個派出機關、1所市立大學及356所高級中等以下學校。

== 沿革 ==

高雄市政府的組織架構，最早可追溯至[[台灣日治時期|日治時代]]，於1924年高雄[[臺灣市制|設市]]時所成立的「'''高雄市役所'''<ref>轄區為今[[高雄市|高雄市]][[鹽埕區|鹽埕]]、[[鼓山區|鼓山]]、[[左營區|左營]]、[[楠梓區|楠梓]]、[[三民區|三民]]、[[新興區_(高雄市)|新興]]、[[前金區|前金]]、[[苓雅區|苓雅]]、[[前鎮區|前鎮]]、[[旗津區|旗津]]等十區（不含小港區、前鎮區的南側、三民苓雅前鎮三區的東側）</ref>」，隸屬於[[高雄州|高雄州]]。

1945年11月8日，[[國民政府|國民政府]]接管高雄市；12月6日正式成立「'''高雄市政府'''<ref>轄區為今[[高雄市|高雄市]][[鹽埕區|鹽埕]]、[[鼓山區|鼓山]]、[[左營區|左營]]、[[楠梓區|楠梓]]、[[三民區|三民]]、[[新興區_(高雄市)|新興]]、[[前金區|前金]]、[[苓雅區|苓雅]]、[[前鎮區|前鎮]]、[[旗津區|旗津]]等十區（不含小港區）</ref>」，隸屬於[[臺灣省|臺灣省]]。

1979年7月1日，「'''高雄市政府'''」升格為[[直轄市|直轄市政府]]，脫離省政府所轄<ref>轄區除原有十區以外，將高雄縣小港鄉併入'''高雄市'''改為[[小港區|小港區]]</ref>。

2010年12月25日，高雄市和臺灣省轄[[高雄縣|高雄縣]]合併改制，重新組建成新的同名直轄市政府組織「'''高雄市政府'''」。

2017年1月1日，成立「高雄市專業文化機構」，為高雄市政府監督之[[行政法人|行政法人]]，原文化局下轄二級機關[[高雄市立歷史博物館|高雄市立歷史博物館]]及[[高雄市電影館|高雄市電影館]]移交該法人機構，[[高雄市立美術館|高雄市立美術館]]於同年7月1日移交該法人機構；「高雄市政府兵役局」併入高雄市政府民政局，成為民政局下轄之「高雄市兵役處」。

2017年9月1日，原文化局下轄二級機關[[高雄市立圖書館|高雄市立圖書館]]改制為行政法人，直接由高雄市政府監督。

2018年1月1日，成立「高雄流行音樂中心」，為高雄市政府監督之[[行政法人|行政法人]]；成立「高雄市政府毒品防治局」，為高雄市政府下轄一級機關。

2018年9月1日，原高雄市政府秘書處改制為「[[高雄市政府行政暨國際處|高雄市政府行政暨國際處]]」，機關位階不變，同為高雄市政府下轄一級機關<ref>8月31日（星期五）舉行揭牌典禮</ref>。原教育局下轄二級機關「高雄市體育處」升格改制為「[[高雄市政府運動發展局|高雄市政府運動發展局]]」，為高雄市政府下轄一級機關<ref>9月3日（星期一）上班首日舉行揭牌典禮</ref>。

== 組織 ==
高雄市政府所轄組織，在[[高雄市市長|市長]]及副市長之下，轄有31個一級機關（包括24個局，4個處，3個委員會）及35個區公所，另外還包括處/院/所/館/中心等138個二級機關以及1所市立大學與356所學校（高級中等學校至幼兒園），並監督3個具有地方自治權的山地原住民區。高雄市政府並置'''[[市政會議|市政會議]]'''為市政最高決策機構，市長在重要決策或向[[高雄市議會|高雄市議會]]提交[[法案|法案]]前均先由市長會同市政會議通過。市政會議每[[週|週]]舉行一次，由市長、副市長、秘書長、副秘書長、局長、處長、委員會主任委員、與市長指定之人員組成。市長為市政會議之[[主席|主席]]，當市長因故不能出席時，由市長指派其中一位副市長代理市長擔任主席；當三位副市長也不能出席時，則指派一級機關單位其中一位首長包括秘書長代理市長擔任主席，必要時市長得邀請或指定其他有關人員列席。 

高雄市政府另依[[s:zh:行政法人法|行政法人法]]成立了3個專業[[行政法人|行政法人]]機構，機構首長由法人董事會任命，並由董事會審議核定營運方針、發展目標、預算及績效等；行政法人機構並非市府下轄機關，不受市府管轄，但市府依法為行政法人機構之監督機關，進行營運績效之評鑑，並就專業人員遴選聘任為董、監事會成員。

===任務編組===
高雄市目前共有138個由高雄市政府府本部設立的任務編組單位如下<ref>{{Cite web |url=https://kpd.kcg.gov.tw/News.aspx?n=E7E9694A71DD2E9E&sms=DE04465881CCC4E8= |title=高雄市政府人事處-人事法規  }}</ref>

遇中央法令規定、政策需求應成立任務編組；階段性任務涉及兩個以上機關協調事項，透過會議或聯繫方式無法有效處理而有設置任務編組之必要時，得成立任務編組。

任務編組名稱除法令另有規定者，得按其性質定名為會、小組、中心、籌備會、會報。組成成員，通常稱為委員，由府內相關首長及外聘專家、學者及團體代表派（聘）兼之，由市長圈選決定；其中主其事者稱召集人及副召集人，通常由[[高雄市市長|市長]]、副市長或權責一級機關正（副）首長擔任；單一性別比例應不低於成員人數三分之一，開會時得邀請有關機關（構）派員列席。

任務編組對外行文時，多以「'''高雄市政府'''」名義行之，並指派一一級機關為幕僚單位；部分任務編組單位以權責一級機關名義對外行文；此外尚有數個由一級機關或二級機關設立的任務編組單位未列於下表。

== 市政府大樓 ==
*1924年（[[大正|大正]]13年），[[日本|日本]][[台灣總督府|執政當局]]廢[[高雄郡|高雄郡]]與[[高雄街|高雄街]]，設[[高雄市_(州轄市)|高雄市]]。高雄設市時成立的高雄市役所，當時的辦公廳舍即為今日的[[鼓山區|鼓山區]][[哈瑪星代天宮|哈瑪星代天宮]]址。
*1939年（[[昭和|昭和]]14年），鑒於市中心東移，當局乃將市役所遷至榮町，即今之'''[[高雄市立歷史博物館|高雄市立歷史博物館]]'''。
*1992年（民國81年）1月18日，高雄市政府搬遷至四維路、苓雅區[[高雄市苓雅區四維國民小學|四維國小]]西南方新築之市政大樓，並命名為「'''高雄市政府合署辦公大樓'''」，俗稱「新市府」。
*2010年（民國99年）12月25日，[[高雄市_(1979年-2010年)|原高雄市]]與[[臺灣省|臺灣省]][[高雄縣|高雄縣]][[五都改制|合併升格]]為[[高雄市|高雄市]]，位於苓雅區的高雄市政府合署辦公大樓稱'''四維行政中心'''，而位於鳳山區的前高雄縣政府辦公大樓稱'''鳳山行政中心'''。
*由於合併升格後的高雄市政府公務員額有1萬5400名，目前兩處辦公廳舍均不能單獨容納市府所有人員，加上分散兩地辦公恐影響辦公效率，未來將擇地新建市政大樓，目前提議地點有高雄新市鎮二期（位於橋頭[[青埔站|捷運青埔站]]東側）、[[台灣中油|中油]][[高雄煉油總廠|高雄煉油總廠]]遷廠後原址、[[凹子底|凹子底]](龍華國小舊校區)、[[國防部軍備局生產製造中心第205廠|205兵工廠]]等處<ref>[http://www.epochtimes.com/b5/12/4/19/n3569674.htm%E9%AB%98%E9%9B%84%E5%B8%82%E6%94%BF%E5%BA%9C%E6%98%AF%E5%90%A6%E6%9C%83%E8%A8%AD%E7%AB%8B%E6%96%B0%E5%B8%82%E6%94%BF%E4%B8%AD%E5%BF%83 高雄市政府是否會設立新市政中心].大紀元.2012-04-19 </ref>，但2012年1月30日高雄市長陳菊指出高雄市新市政中心大樓，興建經費至少需要[[新台幣|新台幣]]150億元，以現今高雄市政府的財政來說，再過十年、二十年恐怕都有困難，因此，有關興建新的行政中心大樓，市府將再研究<ref>[http://tw.news.yahoo.com/%E9%99%B3%E8%8F%8A-%E8%8B%A5%E5%BB%BA%E6%96%B0%E7%9A%84%E8%A1%8C%E6%94%BF%E4%B8%AD%E5%BF%83-%E8%87%B3%E5%B0%91%E8%A6%81150%E5%84%84-072711367.html 陳菊：若建新的行政中心　至少要150億].中廣新聞網.2012-01-30</ref>。
*2012年（民國101年）12月21日，鳳山行政中心前棟大樓因為建物老舊，市府委請專業技師鑑定，該建築未達安全耐震標準，且部分建物未領有使用執照，為了顧及辦公人員的安全，市府決定拆除重建動工。
*2013年（民國102年）12月，鳳山行政中心前棟整建工程完工。計有水利局、教育局、海洋局、觀光局、原住民事務委員會進駐合署辦公。
*2018年4月18日，高雄市都發局長李怡德表示市府現階段並無興建新市政中心規劃，未來仍需經過不斷溝通，來形成共識<ref>葛祐豪.[http://news.ltn.com.tw/news/local/paper/1193379 〈南部〉市政中心遷到亞灣？ 市府：無計畫].自由時報電子報.2018-04-18</ref>。
