'''高愷蔚''' ({{lang-en|APE KAO}}）（{{bd|1993年|8月8日||}}），是一名[[臺灣|臺灣]][[創作歌手|創作歌手]]，現為[[KKFARM|KKFARM]]旗下藝人<ref>{{cite web |title=高愷蔚「好朋友野餐日」開唱擠爆　細膩嗓音讓民眾好沈醉 |url=https://star.ettoday.net/news/918797 |website=ETtoday星光雲 |accessdate=2018-10-24}}</ref><ref>{{cite web |author1=李妙音 |title=《The Voice决战好声》盲选赛 高恺蔚、杨碧琪让导师“秒转” |url=https://www.zaobao.com.sg/zentertainment/music/story20171009-801635 |website=联合早报 |accessdate=2018-10-24}}</ref>。

== 經歷 ==
* {{start_date|2011|08|14}}於[[超級偶像第六屆校際爭霸戰|超級偶像第六屆校際爭霸戰]]海選通過，演唱[[Mariah_Carey|Mariah Carey]]的《Hero》，表現不俗，蒙黃國倫老師讚為奇葩。
* {{start_date|2012|4|06}}於恩師錄音室，協助好友Kimberley陳芳語錄製《愛你》錄音室版本，發布後大受好評，而他配唱的部分也受到極高的評價。
* {{start_date|2012|12|09}}於[[華人星光大道2|華人星光大道2]]以踢館者身份演唱孫楠的《燃燒》，結果獲得15分，PK失敗。
* {{start_date|2016|11|}}主唱105學年度 HBL高中籃球聯賽專屬主題曲 Official MV《勝戰》，庭竹 Ting Suyama作詞 ，陶山 Skot Suyama作曲。
* {{start_date|2017|02|19}}發行首張個人EP《讓一切重來》，高愷蔚、庭竹作詞，陶山、Suphoria作曲。
* {{start_date|2017|10|08}}現身[[The_Voice_決戰好聲_(第一季)|The Voice 決戰好聲 (第一季)]]盲選賽演唱[[林俊傑|林俊傑]]的《那些你很冒險的夢》，獲得[[陳奐仁|陳奐仁]]、 [[丁噹|丁噹]]及[[伍思凱|伍思凱]]三位導師轉身，並選擇加入陳奐仁隊伍。值得一提的是，[[陳奐仁|陳奐仁]]老師在愷蔚演唱第一個音符時即秒轉，隔天登上當地媒體新聞，受到大肆讚譽。
* {{start_date|2017|11|19}}進行隊內PK賽，演唱[[方大同|方大同]]的《BB88》，結果敗給闇鴻麟，但獲曹格使用搶奪權利，成功加入曹格隊伍。
* {{start_date|2017|12|03}}和林文蓀PK，演唱[[盧廣仲|盧廣仲]]的《今天睡在這裡》，結果失敗，卻步於淘汰賽。
