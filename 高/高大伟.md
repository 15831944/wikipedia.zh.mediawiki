'''高大伟'''（{{lang-en|David Gosset}}，{{bd|1970年|5月23日}}) 出生于法国[[巴黎|巴黎]]，法国籍全球事务和[[国际关系|国际关系]]方面的专家，同时也是一位[[汉学家|汉学家]]。
 
== 个人背景 ==
[[File:2010年12月1日，高大伟在伊斯坦布尔，讨论新丝绸之路战略.png|thumb]]


高大伟出生于[[法国|法国]][[巴黎|巴黎]]，在[[索邦大学|索邦大学]]完成了学习的主要部分。他在[[中国|中国]]以“高大伟”<ref>[http://v.ifeng.com/view/dazhengjing/special/gaodaweilunzhongmeiguanxi/detail_2011_01/12/4247598_0.shtml]</ref>这个名字为人所知([[拼音|拼音]] : GAO DAWEI)。高大伟可以用法语、英语及中文讲课和撰写文章。<ref>[http://www.huffingtonpost.com/david-gosset/]
</ref><ref>[http://news.nankai.edu.cn/zhxw/system/2014/07/31/000196138.shtml]</ref>

== 研究方向  ==
作为一位汉学家，高大伟关注并书写中国的转型及其对21世纪[[亚洲|亚洲]]及全球的影响<ref>[http://www.guancha.cn/indexnews/2011_05_13_56965.shtml]</ref>，涉及经济、社会政治、文化、战略动力学等多个方面。

高大伟着力研究 “中国复兴”这一概念<ref>[http://www.sinolatamforum.com/opiniones_detalle/2-m170-83/]</ref>,在他对中国的研究中，他强调“中”这一概念的重要性：“中”是最古老最常见的汉字之一，如果从类比的角度看，这对解释中国在国际上的表现提供了线索，这个字在中国历史和文化背景下有多重含义，也是[[图论|图论]]领域内“中”的三个重要尺寸的写照。
 
中国称[[习近平|习近平]]时期是“中国复兴”的重要时刻。对于作为习近平时期关键词的[[中国梦|中国梦]]，高大伟认为: 这是一个处于变革的国家的理念和计划、定义和定位，习近平的中国梦可以分为三部曲，既“现代化中国”、“全球化中国”、“文明中国”。<ref>[http://news.nankai.edu.cn/nkyw/system/2014/08/04/000196374.shtml]</ref>

高大伟认为习近平提出的“一带一路”是推动和平发展的大国方略，是一条合作共赢之路<ref>[http://www.guancha.cn/RomanoProdi/2015_10_20_338116.shtml]</ref>，其战略构想有利于欧洲的繁荣发展，欧盟应多加参与“一带一路”的发展规划，如果欧洲不抓住这个机会，将是历史性的错误。<ref>[http://www.guancha.cn/GaoDaWei/2015_01_12_305975.shtml]</ref>

== 主要活动 ==
高大伟于2001年在[[中欧国际工商学院|中欧国际工商学院]]成立了(中欧文苑)<ref>[http://www.ceibs.edu/]</ref>,一年后又成立了中欧论坛，属于同类论坛中最早建立者，其中第九届中欧论坛曾与联合国[[教科文组织|教科文组织]]联合举办。<ref>[http://www.unesco.org/new/en/unesco/events/programme-meetings/?tx_browser_pi1%5BshowUid%5D=4105&cHash=10d6657733]</ref>

为研究并支持“新丝路”,高大伟于2015年1月提出了“新丝绸之路行动计划”(NSRI)。

== 个人荣誉 ==
[[西班牙国王|西班牙国王]][[胡安·卡洛斯一世|胡安·卡洛斯一世]]曾授予高大伟“十字国民勋章”<ref>[http://www.ee-iese.com/96/noticia16.php]</ref>，以表彰其中欧文苑的成果。[[保加利亚共和国|保加利亚共和国]]也曾授予高大伟特殊荣誉，以表达对中欧论坛的肯定。
 
2014年12月31日，法兰西共和国总统弗朗索瓦•奥朗德颁布法令，高大伟被提名成为“荣誉军团骑士”。
 
2015年9月，高大伟被天津市政府授予“中华人民共和国友谊奖”。<ref>[http://www.ceibs.edu/ceibs-david-gosset-receives-tianjin-friendship-award]</ref>

2015年9月，高大伟被天津市政府授予“中华人民共和国友谊奖”。<ref>[http://www.ceibs.edu/ceibs-david-gosset-receives-tianjin-friendship-award]</ref>
