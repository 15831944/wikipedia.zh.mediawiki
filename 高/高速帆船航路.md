[[Image:ClipperRoute.png|thumb]]
在[[航海|航海]]中，'''高速帆船航路'''是高速帆船的傳統航路之一，此航路以[[歐洲|歐洲]]及[[遠東|遠東]]或[[澳洲|澳洲]]及[[新西蘭|新西蘭]]作兩端，途經[[大西洋|大西洋]]及[[南冰洋|南冰洋]]，某些船隻可能會滯留在[[熱帶輻合帶|赤道無風帶]]數日。<ref>《沿着高速帆船航路》，第44-45頁</ref>此航路由西至東航行，以借用[[咆哮西風帶|咆哮西風帶]]的強風來航行。很多船隻因航路本身顛簸，而在大海中迷失，尤其是在[[合恩角|合恩角]]。經過合恩角後，高速帆船要拐彎以進入回到歐洲的航道。<ref>《沿着高速帆船航路》，弗朗西斯·奇切斯特，1966年</ref>

高速帆船航路在[[蒸汽船|蒸汽船]]出現和[[巴拿馬運河|巴拿馬運河]]及[[苏伊士运河|苏伊士运河]]開通後正式被商界废弃。但是，高速帆船航路仍是全球最快的航道，並且仍為某些航海比賽作賽道，如[[VELUX_5海洋競赛|VELUX 5海洋競赛]]和[[旺迪單人不靠岸航海賽|旺代环球]]比賽。


== 澳洲和新西蘭航路 ==
[[File:Unidentified_tall_ship_near_Cape_Horn_-_Nla.pic-vn3299637-v.jpg|thumb]] 
由[[英國|英國]]經合恩角到[[澳洲|澳洲]]和[[新西蘭|新西蘭]]高速帆船航路提供了环游世界的最快航道，因此商船航行此航路能獲取巨大利潤。航行此航路的商船主要運送粮食、[[羊毛|羊毛]]和[[黃金|黃金]]。但是，此航線經過三個大海岬的南方，包括[[好望角|好望角]]、[[露纹角|露纹角]]以及合恩角，並因而提高了航行的風險。經此航道的船隻要越過烈風、巨浪，以及冰山。雖然此航路危險，但因經此航道能獲取巨大利潤，很多商家仍然願意在此航道上經商。<ref>《沿着高速帆船航路》，弗朗西斯·奇切斯特，第15-16頁  霍德和斯托顿，1966年 ISBN 0-340-00191-7</ref>

== 航道的變數 ==
[[File:StateLibQld_1_100620.jpg|right]]
高速帆船航路總長度約為3275英里，航行時間約為21日。但是在此航線上航行的船隻在很大程度上是由風決定其去向，而風也會改變方向，因此高速帆船航路的航道和總長度實際上要視乎當時的風向和風速。

曾經有些帆船欲逆道而行，但結果大多為失敗。於1922年，帆船加思華爾嘗試在航道於大西洋南部向東拐彎時，故意向西拐彎航向[[智利|智利]]。但在兩次失敗的嘗試後，船長決定放棄，並回歸原本的航道。<ref name=atcw-72>《沿着高速帆船航路》第72-73頁</ref>

而更著名的逆道嘗試則為1919年的加思奥尼尔。她嘗試由[[墨爾本|墨爾本]]航行至[[班伯利_(西澳)|班伯利]]，但無法抗衡澳洲南方強烈的海風。因此船長決定向北航行，並且於[[托勒斯海峽|托勒斯海峽]]再一次面對強風。加思奥尼尔最後重回航道，經過太平洋、合恩角、大西洋、好望角和印度洋並在76天後回到回到班伯利。<ref name=atcw-72/><ref>[http://www.dehaa.sa.gov.au//heritage/ships_graveyards/pdfs/garthneill.pdf 加思奥尼尔–花园之岛''] {{webarchive|url=https://web.archive.org/web/20060302050405/http://www.dehaa.sa.gov.au/heritage/ships_graveyards/pdfs/garthneill.pdf |date=2006-03-02 }} 來自 [http://www.dehaa.sa.gov.au/ 南澳大利亚环境和遗产部]  {{webarchive|url=https://web.archive.org/web/20051210040654/http://www.dehaa.sa.gov.au/ |date=2005-12-10 }}</ref>

== 现代用途 ==

隨著[[蒸汽船|蒸汽船]]的出現和[[苏伊士运河|苏伊士运河]]及[[巴拿马运河|巴拿马运河]]的開通，標誌著高速帆船航路的廢棄。但是，高速帆船航路仍保持著其全球最快的帆船航路，因此人們仍選擇此航路作休闲航行，並因此「复兴」了此航路。

[[弗朗西斯·奇切斯特|弗朗西斯·奇切斯特]]是首個嘗試在高速帆船航路上高速環球航行的人。奇切斯特本已是一个有名的航海先驱，曾經獨自由倫敦航行至悉尼。他又是[[单人艇航行|单人艇]][[帆船比赛|比赛]]的先驱，並且是[[单人艇跨大西洋赛|单人艇跨大西洋赛]]的創辦人之一。 當奇切斯特成功完成跨大西洋的挑戰後，他開始尋找能夠讓他進行環球航行的航路。他有意超越单人艇由倫敦到悉尼航行100天的紀錄。<ref>《吉卜赛环行世界》， 弗朗西斯·奇切斯特爵士，第6頁，国际海运，2001年 ISBN 0-07-136449-8</ref>他於1966年出發，並於107天後成功到達悉尼。經過48天的休息後，他開始了歸程，並在119天後回到倫敦。<ref>《吉卜赛环行世界》第105頁</ref><ref>《吉卜赛环行世界》第218頁</ref>

奇切斯特的成功激勵了更多人挑戰高速帆船航路。而[[周日泰晤士报黃金環球赛|周日泰晤士报黃金環球赛]]亦因此開始舉辦。黃金環球赛除了是首個单人艇帆船環球比赛，還是首個允許參賽者以不同設計的单人艇帆船來參加比賽。此賽事難度高，因此成功不上岸休息並完成環球的只有一人，此人就是[[罗宾·诺克斯·约翰斯顿|罗宾·诺克斯·约翰斯顿]]。他亦成為了首個經高速帆船航路航行，不上岸休息並完成環球的人。

在現代，高速帆船航路仍有幾項定期举行的比賽。[[沃尔沃环球帆船赛|沃尔沃环球帆船赛]]是一個可中途上岸休息的比賽，每四年舉辦一次。而在兩個单人艇帆船比赛中，VELUX 5海洋競賽可以讓選手中途上岸休息，而旺代環球則不可以讓選手中途上岸休息。

於2005年3月，[[布鲁诺·佩罗恩|布鲁诺·佩罗恩]]和他的團隊乘著[[雙體船|雙體船]]橙Ⅱ，刷新了高速帆船航路的最快環球航行紀錄。其紀錄為50天，16小时，20分4秒。<ref>[http://www.yachtsandyachting.com/news/?article=16425 ''橙Ⅱ刷新了環球航行记录'']，游艇和帆船</ref>

於2005年，[[艾伦·麦克阿瑟|艾伦·麦克阿瑟]]亦刷新了單人艇不上岸休息並完成環球的紀錄。她的紀錄為71天，14小时18分33秒。<ref name="WSSRC Ratified Passage Records">[http://www.sailspeedrecords.com/content/view/19/21 WSSRC批准的航道记录] — 《环游世界，不停止，独力》 [[世界航行速度纪录理事会|世界航行速度纪录理事会]]</ref>

於2008年，[[弗朗西斯·茹瓦永|弗朗西斯·茹瓦永]]又刷新了麦克阿瑟的紀錄。茹瓦永的紀錄為57天，13小时，34分6秒。<ref name="WSSRC Ratified Passage Records"/>
