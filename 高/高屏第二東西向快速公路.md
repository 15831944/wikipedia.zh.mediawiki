'''高雄－屏東間東西向第二快速公路'''，簡稱'''高屏第二快速公路'''或'''高屏二快'''或'''屏中快道'''，是規劃中的[[台灣|台灣]]東西向[[省道快速公路|快速公路]]之一。西起[[高雄市|高雄市]][[仁武區|仁武區]]接[[國道十號_(中華民國)|國道十號]]，東至[[屏東縣|屏東縣]][[長治鄉|長治鄉]]接[[福爾摩沙高速公路|國道三號]]，計畫長度23公里<ref name=ltn>[http://news.ltn.com.tw/news/local/paper/1126215 高屏第二條東西向快速路 審查通過]</ref>。完工後將納編省道快速公路。

2018年7月，行政院長[[賴清德|賴清德]]至屏東縣視察表示，[[行政院|行政院]]已經核定計畫，將全力縮短[[台灣的南北差距|台灣的南北差距]]。<ref name="NOWnews 今日新聞 2018">{{cite web |author=陳宗傑|location=屏東報導 | title=縮短南北差距　賴清德視察屏東交通、醫療建設 | website=NOWnews 今日新聞 | date=2018-07-07 | url=https://www.nownews.com/news/20180707/2785108 | language=zh | access-date=2018-07-09}}</ref>

== 緣由 ==
[[高屏|高屏]]兩地往來頻繁，早為同一生活圈，但因高屏往來唯一東西向快速公路[[台88線|台88線]]面臨交通流量飽和、動輒塞車，因此交通部公路總局計畫興建第二條東西向快速公路因應，即高屏二快。預期完工後，對屏北地區及屏東市居民，來往高雄市區及[[高鐵左營站|高鐵左營站]]，將更加便利。

== 建設計畫 ==
[[交通部公路總局|交通部公路總局]]綜合分析以[[高雄市|高雄市]][[仁武區|仁武區]]到[[屏東縣|屏東縣]][[長治鄉|長治鄉]]B1路廊最佳，B1路廊總長20.2公里，以仁武區[[國道十號_(中華民國)|國道十號]][[仁武交流道|仁武交流道]]為起點，往東北以橋樑型式跨越[[國道七號|國道七號]]。東行穿越鳳梨山群入平原區，以橋樑型式跨越省道[[台29線|台29線]]並設交流道；續東行跨越高屏溪，以地下道型式穿越堤防路（[[縣道189甲線|縣道189甲線]]）及設置西側匝道後，以地下道穿越屏東機場，往東出土以橋樑型式跨越省道[[台3線|台3線]]與省道[[台27線|台27線]]設交流道，再往東北行於台27線南側，至[[福爾摩沙高速公路|國道三號]][[鹽埔交流道|鹽埔交流道]]至[[長治交流道|長治交流道]]間設置系統交流道做為路廊終點。

[[屏東縣政府|屏東縣政府]]於2017年3月23日立法院交通委員會考察屏東地區交通及觀光建設時建議，屏東端路線宜作調整，路線過高屏溪後往北偏移，沿[[屏東機場|屏東機場]]北側堤防路行經台3線，過台3線後沿既有台電高壓鐵塔路線布設。

[[高雄市政府|高雄市政府]]要求，從原本國道十號的起點，延長到高鐵平面道路，而屏東端也有從台27線，再連接到國道三號[[鹽埔交流道|鹽埔交流道]]的需求，使得整體計畫益趨複雜。長度24.8公里，總經費360多億元的高屏第二快速公路，交通部公路總局將於2017年6月份完成修正成果，再報請[[中華民國行政院|行政院]]核定。

== 最新計畫 ==
新路線的路廊規劃，西起高雄左營高鐵路，向東延伸，沿線設有國道10號系統交流道、義大二路交流道、台29交流道、台3交流道、台27交流道，在鹽埔附近採平面銜接縣道187丙線（國道3號側車道），全長23公里870公尺<ref>[https://udn.com/news/story/7327/2635440 屏東第二條東西快速道路通過可行性研究]</ref>。主要採高架橋樑興建，包括高雄端一座零點六公里長的隧道，雙向四線道設計<ref name=ltn/>。

屏東縣政府表示，第二東西快速道路彌補屏北地區長達13公里跨河缺口，節省屏東往來左營區20分鐘車程，提升高屏溪兩岸產業競爭力，紓解台1線與台88線長久以來交通壅塞。

2018年7月7日，屏東縣政府向行政院簡報，提出「一捷、二鐵、三快速」交通路網需求，第二東西向快速道路新左營到國道三號，也能規劃延伸到正建置中的內埔水門轉運站。<ref>[https://www.atanews.net/?taiwan-pingtung=46419 屏中快道延伸]</ref>
