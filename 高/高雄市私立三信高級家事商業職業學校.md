'''高雄市私立三信家事商業職業學校'''，簡稱'''三信家商'''。是一所位於[[高雄市|高雄市]][[苓雅區|苓雅區]]的技術型高級中等學校。該校名稱來自其創辦者[[高雄市第三信用合作社|高雄市第三信用合作社]]<ref>[http://www.sanhsin.edu.tw/master/a/a11/a11.htm 校史簡介]</ref>。

== 建校沿革 ==
該校創於1958年，創辦人為[[林瓊瑤|林瓊瑤]]先生，現任董事長為林昶彤先生，現任校長為林岑。該校的母企業為[[高雄市第三信用合作社|高雄市第三信用合作社]]，於1956年1月10日召開的社員代表大決議創辦三信高級商業職業學校，其後幾經折衝協商，終於1958年4月27日，由三信合信社理事主席，該校創辦人林瓊瑤先生於現址五塊厝親自主持動土，興建校舍。並定是日為該校之校慶。


現任董事長已改

== 建築特色 ==
校內有兩座被[[中華民國文化部|中華民國文化部]]列為[[台灣|台灣]][[二次大戰|二次大戰]]後的重要建築-波浪大樓及圖書館(創學中心)，均為[[陳仁和|陳仁和]]建築師的作品，常吸引許多民眾參觀。

== 校名沿革 ==
* 1958年8月9日奉准設立高商校名為「台灣省高雄市私立三信高級商業職業學校  」
* 1959年7月18日奉准改為完全商職校名「台灣省高雄市私立三信商業職業學校     」
*1960年6月25日奉准撤銷改制專辦高中「台灣省高雄市私立三信高級商業職業學校 」
*1964年6月9日奉准改為完全商職校名為「台灣省高雄市私立三信高級商業職業學校」
*1974年3月15日初級部廢止改校名為「台灣省高雄市私立三信高級商業職業學校   」
*1979年8月1日高雄市改為院轄市更正校名「高雄市私立三信高級商業職業學校     」
*1988年1月18日該校增設家科，更改校名為「高雄市私立三信高級家事商業職業學校」

== 樂旗儀隊 ==
[[File:San_Sin_High_School_Marching_Band.JPG|thumb]]
三信家商的樂儀隊成立1964年，早期樂儀隊都由男生担任，由於男生停招的更迭；70年代後完全由女生担綱。80年代更有旗隊的加入。90年代旗隊更替代儀隊。樂旗隊在崔華傑教練的精心指導下，能在變化的隊形中演奏各型樂器及舞動旗姿；年年榮獲全國音樂比賽的優等或特優獎的殊榮。樂旗隊不但是招生的尖兵，更是三信家商的榮耀；不但經常受邀在國內各地，各項大小慶典中參與演出，各聞遐邇。其精湛的表演除頻添各式盛會的光釆外，且深獲觀眾的喜與讚賞；更曾代表高雄市政府遠赴美國、加拿大、香港、澳門等地協助有關文化的，教育的、城市的外交使命。對於提昇台灣在國際舞台的能見度，著實頁獻良多。

===棒球隊===
1958年9月成軍，於1960年代初期成為南部兩大勁旅(另一隊為[[南英商工|南英商工]])，但至1970年代，由於兵源不繼，歷年的全國青棒選拔賽大多缺席，1980年代初，因網羅[[郭進興|郭進興]]，[[涂鴻欽|涂鴻欽]]等好手，曾一度有復興的蹟象，可惜在1983年因球隊開銷過大以及教練許春權因病過世後，球員管理出現問題，導致學校董事會決議解散球隊，剩下的在校生如[[陳順在|陳順在]]，[[楊章鑫|楊章鑫]]，[[黃森泰|黃森泰]]，[[陳昭安|陳昭安]]，[[連德政|連德政]]，[[陳政賢|陳政賢]]等球員在棒壇人士安排之下，由台中市[[新民商工|新民商工]]接手。

2003年6月，因應高中棒球的興盛，球隊重新成立，並由高雄市棒壇耆宿[[王正輝|王正輝]]掌軍，三信家商已逐漸成為台灣強力新猷。到了2009年，[[王躍霖|王躍霖]]、[[王梓安|王梓安]]、[[羅華韋|羅華韋]]三名應屆畢業生接連赴美，更是證明三信家商已在台灣青棒占有一席之地。

2016年，因校方和董事會決議，因少子化及經費問題之下，三信家商棒球隊於105學年度、暑假起暫時停止招生；2018年6月在19名高三球員畢業後球隊等同宣告解散。

===籃球隊===
[[107學年度高中籃球聯賽|107學年度]]三信男籃3年前在乙組地區複賽遭到淘汰，首度在2018年挑戰HBL甲級，目前由由[[高苑工商|高苑工商]]前總教練[[孫晁彰|孫晁彰]]帶隊，分在A組跟兩支甲組強隊[[強恕高中|強恕高中]]和[[三民家商|三民家商]]一組，也確定直接晉級外卡戰第一輪不過第一戰抱著學經驗以35：123慘敗結束賽季
