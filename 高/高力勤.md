'''高力勤'''（{{lang|en| Derek Corrigan}}，{{bd|1951年|11月21日}}），是一个[[加拿大|加拿大]]的政治家和[[卑诗省|卑诗省]][[本拿比|本拿比]]市长。

== 早年经历 ==
高力勤毕业于[[温哥华|温哥华]][[查尔斯塔珀爵士中学|查尔斯塔珀爵士中学]]，并在[[卑诗大学|卑诗大学]]学习政治学和哲学，1977年获得法学学位，1978年加入卑诗省律师协会。高力勤是温哥华律师事务所林賽·肯尼（Lindsay Kenney）的助理律师。

== 政治生涯 ==
高力勤于1987年成为[[本拿比|本拿比]]市议会的市议员，共任職15年。他于2002年当选市长，并在2005年、2008年、2011年和2014年连任5届。下届市长选举将在2018年10月进行。

== 家庭 ==
高力勤与妻子柯瑞根1976年结婚。<ref>{{Cite web |url=http://www.burnabycitizens.ca/meet-the-mayor/ |title=了解市长 |accessdate=2017-09-11 |work=本拿比公民协会 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>柯瑞根2009年至2017年任[[本拿比鹿湖选区|本拿比鹿湖选区]]的第1任、第2任省议员。他们有4个孩子，包括西恩（Sean）、達西（Darcy）、派崔克（Patrick）和凱爾西（Kelsey）。
