'''高平陵之變'''，亦称'''高平陵政變'''、'''高平陵事變'''或'''正始之變'''，发生在[[三国|三国]]时期的[[曹魏|魏国]]，是魏國建立以後的一次重大政變。事件源自[[曹魏宗室|曹魏宗室]]大將軍[[曹爽|曹爽]]和朝中重臣太傅[[司马懿|司马懿]]之間的权力鬥争，最后司马懿趁著[[曹爽|曹爽]]與魏帝[[曹芳|曹芳]]到[[高平陵|高平陵]]謁陵時發動[[政變|政變]]，控制京城，[[族滅|族滅]]曹爽而结束，自此司馬氏正式全面掌權，此後曹氏皇帝皆淪為司馬家的[[傀儡政權|傀儡]]。

== 起因 ==
魏明帝[[曹叡|曹叡]]於[[景初|景初]]三年（239年）逝世，遺詔由年僅八歲的皇太子[[曹芳|曹芳]]繼位，並由[[大將軍_(東亞)|大將軍]][[曹爽|曹爽]]和[[太尉|太尉]][[司馬懿|司馬懿]]輔政。曹爽是[[曹真|曹真]]之子，辅政之初，曹爽因为司馬懿年龄和威望较高，侍奉司馬懿如父親一般，凡事不敢专行。[[何晏|何晏]]等人向曹爽进言「权力不宜委之于人」，后曹爽开始专权<ref>《三国志卷九 诸夏侯曹传》：初，爽以宣王年德并高，恒父侍之。及晏等进用，咸共推戴，说爽以权重不宜委之于人。初，宣王以爽魏之肺腑，每推先之，爽以宣王名重，亦引身卑下，当时称焉。丁谧、毕轨等既进用，数言于爽曰：“宣王有大志而甚得民心，不可以推诚委之。”由是爽恆猜防焉。礼貌虽存，而诸所兴造，皆不复由宣王。</ref>，重用[[何晏|何晏]]、[[鄧颺|鄧颺]]、[[李勝|李勝]]、[[畢軌|畢軌]]、[[丁謐|丁謐]]等人，排斥司马懿；且不久即晉升司馬懿為[[三國太傅、太保列表|太傅]]而奪去了他的[[軍權|軍權]]。之後又任命弟弟[[曹羲|曹羲]]和[[曹訓|曹訓]]為[[中領軍|中領軍]]及武衛將軍，曹爽集團於是完全掌握宮中[[禁軍|禁軍]]。從此曹爽和何晏等心腹控制了朝廷的運作，權傾朝野，甚至迁[[郭太后|郭太后]]于永宁宫，曹爽更以魏明帝才人为[[歌伎|歌伎]]，僭用皇帝仪仗，而同為[[輔政大臣|輔政大臣]]的司馬懿則被架空。

正始五年（244年），曹爽意圖消滅蜀漢，發動了[[興勢之戰|興勢之戰]]。4月，曹爽提拔[[夏侯玄|夏侯玄]]為征西將軍。中護軍一職，改由[[司馬師|司馬師]]擔任。正始六年八月，曹爽又毀中壘中堅營，以其營兵歸中領軍曹羲，更加強了中領軍的權力，而中護軍的權力則進一步削弱。但曹爽以司馬氏掌握禁軍，帶來了意想不到的後果。

司马懿無法參與政令決策，為了等待時機，凝聚反擊力量，於[[正始_(曹魏)|正始]]八年（247年）藉故生病辭職以迴避曹爽。次年，李勝到[[荊州|荊州]]上任[[刺史|刺史]]前向司馬懿辭行，司馬懿更在他面前裝出重病的樣子<ref>《三國志》註引《魏末傳》：「宣王令兩婢侍邊，持衣，衣落；復上指口，言渴求飲，婢進粥，宣王持杯飲粥，粥皆流出沾胸。」又多次將荊州說成并州，似乎已經病重得思緒混亂。</ref>，因此令曹爽對他更為鬆懈；但於此同時，司馬懿卻与儿子[[司马师|司马师]]（當時為中護軍）暗中准备發動[[政變|兵变]]，司马师甚至为政变阴养死士三千<ref>《晋书》卷2：宣帝之将诛曹爽，深谋秘策，独与帝潜画....初，帝阴养死士三千，散在人间。</ref>。

== 經過 ==
[[正始_(曹魏)|正始]]十年正月初六<ref>《三国志·魏书四·齐王纪》：嘉平元年春正月甲午</ref>（249年2月5日），少帝曹芳拜谒[[魏明帝|魏明帝]]之墓[[高平陵|高平陵]]，曹爽兄弟及其亲信們皆随同前往。司馬懿和中护军司馬師以及三千死士在皇宮內城司馬門聚集，前往擺放武器的武庫，途徑曹爽府門，曹爽帐下督严世上楼，用弩瞄準司馬懿準備射殺，曹爽門人孙谦卻拉拽嚴世的肘制止并說：「天下事未可知！」導致嚴世無法襲擊司馬懿。司马懿控制武库后，藉曾被曹爽奪權的[[明元郭皇后|皇太后郭氏]]之诏令，关闭[[洛阳市|洛阳]]所有城门，率兵佔領[[洛河_(黄河)|洛水]]浮橋。接著任命[[司徒|司徒]][[高柔|高柔]][[符节|假節]]行[[大將軍_(東亞)|大將軍]]事，接管曹爽的軍權；并以[[桓范|桓范]]为中领军，[[桓范|桓范]]在其子的劝说下拒绝<ref>《三国志》卷9注引《魏略》：及宣王起兵，閉城門，以範爲曉事，乃指召之，欲使領中領軍。範欲應召，而其子諫之，以車駕在外，不如南出。範疑有頃，兒又促之。</ref>，改以[[王观_(三国)|王觀]]行[[護軍|中領軍]]事，接管[[曹羲|曹羲]]的軍隊。

司马懿控制城內，隨即派人上奏皇帝曹芳，宣称奉[[皇太后|皇太后]][[詔書|詔書]]，罷免曹爽兄弟。詔書先传至曹爽手中，曹爽惶然不知如何是好，也不敢送給曹芳。[[大司農|大司農]][[桓範|桓範]]在政變發生後不顧下屬勸阻，与曹爽司馬[[魯芝|魯芝]]、主簿楊綜等出城勸曹爽前往[[许昌|许昌]]，然後以皇帝為號召擁兵抵抗司马懿。司馬懿接連派侍中[[許允_(曹魏)|許允]]、尚書[[陳泰_(三國)|陳泰]]、[[尹大目|尹大目]]等人勸說曹爽[[投降|投降]]，並指着洛水发誓，允诺只要曹爽罷兵息馬，交出兵權，仍可保留[[爵位|爵位]]<ref>《三国志》卷9：宣王使許允、陳泰解語爽，蔣濟亦與書達宣王之旨，又使爽所信殿中校尉尹大目謂爽，唯免官而已，以洛水爲誓。</ref>。曹爽犹豫了一夜，最後認為投降雖然會失去政治權力，但以[[侯爵|侯爵]]的身份應仍能享受榮華富貴；於是放棄抵抗，而請皇帝罷免自己，並向司馬懿認罪。曹爽兄弟罷官後隨即回到府邸，并遭到司马懿的监视<ref>《三国志》卷9注引《魏末传》：爽兄弟歸家，勑洛陽縣發民八百人，使尉部圍爽第四角，角作高樓，令人在上望視爽兄弟舉動。</ref>。

正月初十<ref>《三国志·魏书四·齐王纪》：戊戌，有司奏収黄门张当付廷尉</ref>（249年2月9日），與曹爽往來甚密的朝中侍從[[张当|張當]]，因私自将宫女送给曹爽被抓捕，在[[廷尉|廷尉]]嚴刑拷問之下供稱曹爽和[[何晏|何晏]]計劃在三月造反，於是曹爽與其同夥都被捕<ref>《三国志》卷9：初，張當私以所擇才人張、何等與爽。疑其有姦，收當治罪。當陳爽與晏等陰謀反逆，並先習兵，須三月中欲發，於是收晏等下獄。</ref>，而桓範亦因曾經揚言司馬懿謀反，被視為誣告<ref>《三國志》註引《魏略》：桓範出城後與門候[[司蕃|司蕃]]說：「太傅圖逆，卿從我去！」事後司蕃向司馬懿自首，司馬懿於是控告桓範誣人謀反之罪。</ref>而下獄，與曹爽等人一同[[死刑|處死]]，並且[[族誅|誅滅三族]]。后封曹真的族孙[[曹熙|曹熙]]为新昌亭侯，邑三百户，延續對曹真的祭祀<ref>《三国志卷九·诸夏侯曹传》：嘉平中，绍功臣世，封真族孙熙为新昌亭侯，邑三百户，以奉真后。</ref><ref>《晋书》卷1：嘉平元年春正月甲午，天子謁高平陵，爽兄弟皆從。是日，太白襲月。帝于是奏永寧太后廢爽兄弟。時景帝為中護軍，將兵屯司馬門。帝列陣闕下，經爽門。爽帳下督嚴世上樓，引弩將射帝，孫謙止之曰：「事未可知。」三注三止，皆引其肘不得發。大司農桓範出赴爽，蔣濟言於帝曰：「智囊往矣。」帝曰：「爽與範內疏而智不及，駑馬戀短豆，必不能用也。」於是假司徒高柔節，行大將軍事，領爽營，謂柔曰：「君為周勃矣。」命太僕王觀行中領軍，攝羲營。帝親帥太尉蔣濟等勒兵出迎天子，屯于洛水浮橋，上奏曰：「先帝詔陛下、秦王及臣升於御牀，握臣臂曰『深以後事為念』。今大將軍爽背棄顧命，敗亂國典，內則僭擬，外專威權。羣官要職，皆置所親；宿衞舊人，並見斥黜。根據槃互，縱恣日甚。又以黃門張當為都監，專共交關，伺候神器。天下洶洶，人懷危懼。陛下便為寄坐，豈得久安？此非先帝詔陛下及臣升御牀之本意也。臣雖朽邁，敢忘前言。昔趙高極意，秦是以亡；呂霍早斷，漢祚永延。此乃陛下之殷鑒，臣授命之秋也。公卿羣臣皆以爽有無君之心，兄弟不宜典兵宿衞；奏皇太后，皇太后敕如奏施行。臣輒敕主者及黃門令罷爽、羲、訓吏兵，各以本官侯就第。若稽留車駕，以軍法從事。臣輒力疾將兵詣洛水浮橋，伺察非常。」爽不通奏，留車駕宿伊水南，伐樹為鹿角，發屯兵數千人以守。桓範果勸爽奉天子幸許昌，移檄徵天下兵。爽不能用，而夜遣侍中許允、尚書陳泰詣帝，觀望風旨。帝數其過失，事止免官。泰還以報爽，勸之通奏。帝又遣爽所信殿中校尉尹大目諭爽，指洛水為誓，爽意信之。桓範等援引古今，諫說萬端。終不能從，乃曰：「司馬公正當欲奪吾權耳。吾得以侯還第，不失為富家翁。」範拊膺曰：「坐卿，滅吾族矣！」遂通帝奏。既而有司劾黃門張當，并發爽與何晏等反事，乃收爽兄弟及其黨與何晏、丁謐、鄧颺、畢軌、李勝、桓範等誅之。蔣濟曰：「曹真之勳，不可以不祀。」帝不聽。

初，爽司馬魯芝、主簿楊綜斬關奔爽。及爽之將歸罪也，芝、綜泣諫曰：「公居伊周之任，挾天子，杖天威，孰敢不從？舍此而欲就東市，豈不痛哉！」有司奏收芝、綜科罪，帝赦之，曰：「以勸事君者。」</ref>。

== 影響 ==
司馬懿因為這次政變，清除了以曹爽为首的曹氏[[宗室|宗室]]在朝中的勢力，曹氏宗室力量日漸薄弱，司馬氏得以作为辅政大臣全面掌握權力，逐步控制[[曹魏|曹魏]]朝政，為日後[[司馬炎|司馬炎]]代魏立[[晉朝|晉]]奠下了根基。

[[蔣濟|蔣濟]]因此事件認為自己失信於曹爽而自責病死。

[[王凌|王凌]]和[[令狐愚|令狐愚]]因為高平陵之变，認為魏帝曹芳年幼平庸而司馬懿獨攬大權，於是於兩年後發動兵變企圖推翻曹芳和司馬懿，另立年紀較長的[[曹彪|曹彪]]（[[曹操|曹操]]兒子）為帝，即[[壽春三叛#王淩之叛|壽春三叛]]中的第一次。

駐守[[雍州_(九州)|雍州]]的征蜀護軍[[夏侯霸|夏侯霸]]因與曹爽有親戚關係，與同時身為[[征西大将军|征西將軍]]的侄兒[[夏侯玄|夏侯玄]]被徵召入[[雒邑|洛邑]]，由於恐懼會遭司馬氏逼害；同時與自己不和的[[郭淮|郭淮]]又出任征西將軍，令他十分不安，因而逃入[[蜀漢|蜀漢]]。
