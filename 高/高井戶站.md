'''高井戶站'''（{{jpn|j=高井戸駅|hg=たかいどえき|rm=Takaido eki|f=y}}）是一個位於[[東京都|東京都]][[杉並區|杉並區]][[高井戶西|高井戶西]]二丁目，屬於[[京王電鐵|京王電鐵]][[井之頭線|井之頭線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''IN12'''。

== 歷史 ==
* 1933年（[[昭和|昭和]]8年）8月1日 - 隸屬於帝都電鐵的車站開始營運。當時為為建造在土提上的車站。
* 1940年（昭和15年）5月1日 - 與小田原急行鐵道合併，成為該公司帝都線的車站。
* 1942年（昭和17年）5月1日 - [[小田急電鐵|小田急電鐵]]與[[東京急行電鐵|東京急行電鐵]]（大東急）合併。
* 1948年（昭和23年）6月1日 - 從東急分離，成立京王帝都電鐵，成為該公司井之頭線的車站。
* 1972年（昭和47年）3月15日 - 	伴隨都道311號環狀八號線的通車，車站進行高架化工程。

== 車站構造 ==
設有1面2線[[島式月台|島式月台]]，為井之頭線上少數的[[高架車站|高架車站]]。車站構築橫跨於{{link-ja|東京都道311號環狀八號線|東京都道311号環状八号線}}（環八通）上。

於2006年12月完成站房及車站高架下的附設商店街整修工程，商店街以「京王retnode高井戶」開幕。

== 車站周邊 ==
[[東京都區部|東京23區]]有數的住宅區之一，車站面向{{link-ja|東京都道311號環狀八號線|東京都道311号環状八号線}}，亦有定期巴士路線的停留所。

北側為{{link-ja|杉並區立高井戶小學校|杉並区立高井戸小学校}}和高井戶温泉美湯、{{link-ja|日本年金機構|日本年金機構}}本部。

東北側有{{link-ja|杉並清掃工場|杉並清掃工場}}及利用廢熱加熱的溫水游泳池、杉並區高井戶地域區民中心、高井戶保健所。杉並清掃工場曾經在前東京都知事[[美濃部亮吉|美濃部]]執政時代，成為{{link-ja|東京垃圾戰爭|東京ゴミ戦争}}的導火線。

西南側為住宅區，有大片的[[公共房屋|都營住宅]]，[[神田川_(東京都)|神田川]]流經此地。另外徒步約5分鐘的距離，是電視連續劇經常取景的{{link-ja|社會福祉法人|社会福祉法人}}{{link-ja|浴風會|浴風会}}本館。

東南側有高井戶站前[[郵便局|郵便局]]、[[大相撲|大相撲]]的{{link-ja|芝田山部屋|芝田山部屋}}（師傅為第62代[[橫綱|橫綱]]的[[大乃國康|大乃國康]]）。另外日本惠普（HP）總公司及分支出的[[安捷倫科技有限公司|安捷倫]]等事業所分布於此，2011年5月HP的事業所遷移集中至[[江東區_(東京都)|江東區]]大島，當車站周邊的事業所已經全數關閉。
