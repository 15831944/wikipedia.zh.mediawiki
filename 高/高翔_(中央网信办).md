'''高翔'''（{{bd|1963年|10月|}}），男，[[汉族|汉族]]，[[四川|四川]][[南部|南部]]人，生于[[四川|四川]][[盐亭|盐亭]]。1988年7月参加工作，1994年10月加入[[中国共产党|中国共产党]]。[[中国人民大学|中国人民大学]]清史系中国古代史专业毕业，[[研究生|研究生]][[学历|学历]]，[[历史学|历史学]][[博士学位|博士学位]]，[[研究员|研究员]][[职称|职称]]。

== 生平 ==
1988年9月起，高翔先后担任[[中国人民大学|中国人民大学]]清史所助教、讲师、副教授。1996年10月，担任[[中国社会科学院|中国社会科学院]]历史研究所副研究员、研究员，1999年起，兼任明清史研究室主任。

2002年8月，担任中国社会科学院直属机关党委副书记。2006年7月，任中国社会科学院《中国社会科学》杂志社总编辑。2012年7月，任[[中国社会科学院|中国社会科学院]]副秘书长，兼《中国社会科学》杂志社总编辑<ref>{{cite web |url = http://renshi.people.com.cn/n/2013/0716/c139617-22212906.html |title=高翔任中国社科院秘书长、党组成员（图/简历）--组织人事-人民网 |accessdate=2017-11-04 |website = 人民网－中国共产党新闻网 }}</ref>。2013年7月，担任中国社会科学院党组成员、秘书长。

2016年3月，担任[[中共福建省委|中共福建省委]]常委、宣传部部长<ref>{{Cite web |url = http://news.xinhuanet.com/politics/2016-11/27/c_1119999099.htm |title = 尤权当选中共福建省委书记-新华网 |accessdate=2017-11-04|website=新华网}}</ref>，兼省社科联主席<ref>{{cite web |url = http://www.fjtv.net/folder546/folder841/2016-04-01/623720.html |title=两位新任福建省委常委 亮相省委常委会议_最新推荐_福建网络广播电视台-福建省最大音视频新闻门户www.fjtv.net |accessdate=2017-11-04 |work=福建新闻 }}</ref>。2017年12月，出任[[中央网络安全和信息化领导小组|中央网络安全和信息化领导小组]]办公室副主任，次月任[[国家互联网信息办公室|国家互联网信息办公室]]副主任。2018年12月，任中国社会科学院副院长。
