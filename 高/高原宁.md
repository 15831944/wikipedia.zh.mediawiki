'''高原宁'''{{BD|1963年|4月}}，生于[[中华人民共和国|中华人民共和国]][[黑龙江省|黑龙江省]][[牡丹江市|牡丹江市]]，祖籍黑龙江省[[齐齐哈尔市|齐齐哈尔市]]，[[北京大学|北京大学]]教授，[[北京大学物理学院|北京大学物理学院]]院长，[[高能物理|高能物理]]实验学家。

== 生平 ==
高原宁高中毕业于[[牡丹江市第一高级中学|牡丹江市第一高级中学]]。<ref>{{cite web |title=市一中迎来70周年校庆 |url=http://www.mdj12z.com/djgz/521.html |website=牡丹江日报 |publisher=牡丹江中学校园网 |accessdate=2018-11-14 |date=2015-09-30 |archive-url=https://web.archive.org/web/20161027185828/http://mdj12z.com/djgz/521.html# |archive-date=2016-10-27 |dead-url=no }}</ref>1979年，他考入[[北京大学物理系|北京大学物理系]]，1983年本科毕业，获理学学士学位。本科毕业后，他在北京大学物理系理论物理专业攻读研究生，1989年获理学博士学位。<ref name="ihep.cas">{{cite web |title=粒子物理前沿卓越中心骨干人才 |url=http://www.ihep.cas.cn/zdsys/ccepp/cceppprogress/201408/t20140804_4170772.html |website=中科院粒子物理前沿卓越创新中心 |accessdate=2018-11-14 |archive-url=https://web.archive.org/web/20150602220312/http://www.ihep.cas.cn/zdsys/ccepp/cceppprogress/201408/t20140804_4170772.html# |archive-date=2015-06-02 |dead-url=no }}</ref>

博士毕业后，高原宁进入[[中国科学院高能物理研究所|中国科学院高能物理研究所]]工作。1992年起，他先后在[[英国|英国]][[伦敦大学皇家霍洛威学院|伦敦大学皇家霍洛威学院]]、[[美国|美国]][[威斯康星大学麦迪逊分校|威斯康星大学麦迪逊分校]]任职，期间在[[欧洲核子研究中心|欧洲核子研究中心]]进行粒子物理实验。2000年底，高原宁入选[[清华大学|清华大学]]百人计划，被聘为[[清华大学工程物理系|清华大学工程物理系]]教授，负责在清华大学的组建团队参加[[大型强子对撞机|大型强子对撞机]]国际合作。2002年，他获得[[国家杰出青年基金|国家杰出青年基金]]<ref>{{cite web |title=国家杰出青年科学基金获得者 |url=http://www.tsinghua.edu.cn/publish/kyy/10046/index.html |website=清华大学科研院 |accessdate=2018-11-14 |archive-url=https://web.archive.org/web/20170720093537/http://www.tsinghua.edu.cn/publish/kyy/10046/index.html# |archive-date=2017-07-20 |dead-url=no }}</ref>。2004年至2018年，高原宁担任了[[清华大学高能物理研究中心|清华大学高能物理研究中心]]主任。2010年，他接受了癌症手术，顺利康复<ref>{{cite news |title=高原宁：躲进“粒子”成一统 管它春夏与秋冬 |url=http://news.sciencenet.cn/htmlnews/2017/8/385678.shtm |work=科技日报 |accessdate=2018-11-14 |date=2017-08-21 |archive-url=https://web.archive.org/web/20170914040408/http://news.sciencenet.cn/htmlnews/2017/8/385678.shtm# |archive-date=2017-09-14 |dead-url=no }}</ref>。2017年，为表彰他领导实验发现[[五夸克粒子|五夸克粒子]]，[[中国物理学会|中国物理学会]]授予他2016-2017年度[[王淦昌物理奖|王淦昌物理奖]]<ref>{{cite web |title=2016-2017年度中国物理学会各项物理奖获奖者名单及工作介绍 }}</ref>。同年，他入围了[[中国科学院院士|中国科学院院士]]增选初步候选人名单（数学物理学部）<ref>{{cite web |title=关于公布2017年中国科学院院士增选初步候选人名单的公告 |url=http://www.casad.cas.cn/doc/17272.html |website=中国科学院 |accessdate=2018-11-14 |accessdate=2018-11-14 |work=新华网 |agency=新华社 |date=2017-07-07}}</ref>2018年11月13日，中共北京大学党委组织部发布公告，拟任命高原宁为北京大学物理学院院长，公示期自11月14日至20日。<ref name="dangwei">{{cite web |title=关于物理学院院长拟任人选的公示 == 学术 ==
高原宁主要研究[[高能对撞机|高能对撞机]][[粒子物理实验|粒子物理实验]]与[[粒子物理理论|粒子物理理论]]，研究领域包括[[τ子|τ]]-[[粲夸克|粲]]物理、寻找新粒子、[[量子色动力学|量子色动力学]]的实验检验、[[底夸克|b物理]]、[[CP破坏|CP破坏]]等。<ref name="ihep.cas" />
