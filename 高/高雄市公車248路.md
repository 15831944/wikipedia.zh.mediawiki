'''高雄市公車248路'''，又稱'''中正幹線'''。行走[[捷運衛武營站|捷運衛武營站]]及[[鼓山輪渡站|鼓山輪渡站]]之間<ref>[http://www.iot.gov.tw/fp.asp?xItem=103914&ctNode=1066&mp=2 高雄市公車動態資訊系統 18日啟用] {{webarchive|url=https://web.archive.org/web/20150923163220/http://www.iot.gov.tw/fp.asp?xItem=103914&ctNode=1066&mp=2 |date=2015-09-23 }}，轉載至中國時報，2002年4月15日。</ref><ref name="tbkc">[http://www.tbkc.gov.tw/Files/service/2014CITYBUS.pdf 《2014年高雄市公車路線導覽手冊》，高雄市政府交通局出版，第67頁。]</ref><ref name="tbkc2">[http://www.tbkc.gov.tw/Files/service/2015CITYBUS.pdf 《2015年高雄市公車路線導覽手冊》，高雄市政府交通局出版，第76頁。]</ref>，路線多處與[[高雄捷運橘線|高雄捷運橘線]]重疊<ref>[陳婷芳(2012)：《新大高雄72小時這樣最好玩》。墨刻出版，第124頁。（[https://books.google.com.hk/books?id=dsOuAAAAQBAJ&pg=PA124&lpg=PA124&dq=248%E5%B9%B9%E7%B7%9A%E5%85%AC%E8%BB%8A+%E9%AB%98%E9%9B%84&source=bl&ots=iks1Kjv3jC&sig=Zba8bL4jchQDNKlR-jebynwSCp8&hl=zh-TW&sa=X&ei=ybaIVNrvHY6A8QXRiICwBw&ved=0CDgQ6AEwBQ#v=onepage&q=248%E5%B9%B9%E7%B7%9A%E5%85%AC%E8%BB%8A%20%E9%AB%98%E9%9B%84&f=false 網上存檔]）。</ref>，在橘線通車前被視為[[捷運先導公車|捷運先導公車]]路線<ref>[https://buslover.tian.yam.com/posts/37482093 20020607 高雄台南嘉義斗六公車隨拍]</ref>。本路線由於途經包括[[高雄火車站|高雄火車站]]、[[西子灣|西子灣]]<ref>[http://tnews.cc/07/newscon1_12723.htm 西子灣交通資訊，]高雄新聞網，2008年7月4日。</ref>、[[鼓山輪渡站|鼓山輪渡站]]、[[高雄港|高雄港]]<ref>[http://www.railway.gov.tw/Kaohsiung-Harbor/CP.aspx?SN=11391 台鐵高雄港車站資訊。]</ref>等主要地標；[[國立中山大學|國立中山大學]]<ref>[http://www.nsysu.edu.tw/files/90-1000-7.php 國立中山大學交通資訊。]</ref>、[[國立高雄師範大學|國立高雄師範大學]]<ref>[http://adms.web.nthu.edu.tw/ezfiles/72/1072/img/342/103map_NKNU.pdf 前往國師大交通圖。]</ref>、高雄文化中心及高雄歷史博物館等政府設施，以及包括[[澄清湖|澄清湖]]、[[打狗英國領事館官邸|打狗英國領事館官邸]]<ref>[http://superspace.moc.gov.tw/hall/local_culture_page.aspx?oid=d481491b-d880-4586-a816-a9dac608faa0 台灣文化部地方文化館網頁中介紹。]</ref>在內的多個高雄著名的觀光點<ref>[http://library.taiwanschoolnet.org/cyberfair2003/C0336800360/guide2.htm 如何暢遊高雄古蹟。]</ref><ref>[http://www.tzengs.com/travel/Kaohsiung_bustour/kaohsiung_citybus.htm 如何在高雄搭公車。]轉載至中國時報，2005年9月。</ref>，屬高雄市其中一條主要的公共運輸路線。

過往[[高雄縣|高雄縣]]未跟原高雄市合併時，本路線已是來往[[鳳山區|鳳山市]]及高雄市之間的主要路線。

==路線基本資料==

單程行車里數約14.5公里，由[[公車建軍站|建軍站]]駛出起，會先繞駛建軍路、[[中山西路|中山西路]]、[[澄清路|澄清路]]、光復路、曹公路、[[中山西路|中山西路]]、[[澄清路|澄清路]]、[[中正路(高雄市)|中正路]]回[[公車建軍站|建軍站]]；再沿[[中正路(高雄市)|中正路]]、[[林森路_(高雄市)|林森路]]、[[建國路_(高雄市)|建國路]]到[[高雄火車站|高雄火車站]]；再沿[[中山路_(高雄市)|中山路]]、[[中正路(高雄市)|中正路]]、大公路、七賢路、臨海新路、臨海二路、濱海二路至[[鼓山輪渡站|鼓山輪渡站]]為止。2016年5月16日起新增區間車(高雄火車站-鼓山輪渡站)。2018年10月14日配合[[高雄市區鐵路地下化計畫|高雄市區鐵路地下化計畫]][[鳳山車站_(臺鐵)|鳳山車站]]改建完工，248繞駛[[鳳山車站_(臺鐵)|台鐵鳳山站]]<ref>[http://southeastbus.com/index/kcg.html 臺鐵鳳山站已改建完成，為便利民眾轉乘火車，自107年10月14日起，248路線班次繞駛臺鐵鳳山站，歡迎乘客多加利用。]</ref>。

==更改路線==
本路線曾作出過以下改動：
*原本繞駛[[鳳山區|鳳山]]路線為：[[公車建軍站|建軍站]] → 行仁路 → 國泰路 → 南京路 → 新富路 → 國泰路 → 五甲路 → 中山路 → 中山東路 → 鳳林路 → 仁愛路 → 中山東路 → 光遠路 → 中山西路 → 澄清路 → 回[[公車建軍站|建軍站]]，繞駛一圈約30分鐘。後來改為：[[公車建軍站|建軍站]] → 建軍路 → 中山西路 → 澄清路 → 光復路 → 曹公路 → 中山西路 → 澄清路 → 回[[公車建軍站|建軍站]]，繞駛一圈約15分鐘。
<ref>[http://news.ltn.com.tw/news/local/paper/46794/print 〈南部〉高捷隧道坍陷 周邊交通大管制。自由時報，2005年12月6日。]</ref>
*因臨海新路開通，[[高雄市248路公車|248]]不行駛公園陸橋 → 臨海一路 → 臨海二路 → 濱海二路 → 濱海一路 → 鼓山一路 → 公園路橋（接回程）；而改駛直行七賢三路 → 臨海新路 → 臨海二路 → 濱海二路 → 濱海一路 → 臨海一路 → 臨海二路 → 臨海新路（接回程）
