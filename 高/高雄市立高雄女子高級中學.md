'''高雄市立高雄女子高級中學'''（{{lang-en|Kaohsiung Municipal Kaohsiung Girls' Senior High School}}，KGHS），簡稱'''雄女'''、'''高雄女中'''、'''南高一女'''，舊稱'''省高女'''，為[[中華民國|中華民國]][[台灣|台灣]][[高雄市|高雄市]]一所[[女校|女子]][[高級中學|高級中學]]。高雄女中的學生稱學校是'''斑城'''，自稱為'''斑馬'''。

== 校史 ==
===日治時期===
* 1924年：高雄女中創校於[[台灣日治時期|日治]]時期，1924年4月1日，當時名為「[[高雄州|高雄州]]立高雄高等女學校」，由高雄州理事官[[名和仁一|名和仁一]]處理校長事務。<ref name="takao">{{cite book|title= 高雄州立高雄高等女學校一覽　昭和十三年|year=1938}}</ref>由於學校草創尚無校舍，因此暫借[[高雄市立鼓山國民小學|高雄第一尋常高等小學校]]（今鼓山國小）<ref>鄭德慶/編，《看見老高雄》，高雄：高雄市政府新聞處，2001年，頁93。</ref>為臨時校舍。<ref name="takao">{{cite book|title= 高雄州立高雄高等女學校一覽　昭和十三年|year=1938}}</ref>同年4月25日舉行第一回入學式，5月16日舉行開校式，8月時教諭[[本田喜八|本田喜八]]兼任校長職。<ref name="takao"/>
* 1925年：位在[[哨船頭|哨船頭]]的宿舍落成。
* 1926年：本田喜八免改為專任校長。
[[File:Kōshō,_Takao_Kōtō_Jōgakkō.jpg|thumb]]
* 1927年：位在[[苓雅寮|苓雅寮]]三番地（今屬前金區）的新築校舍部分完工。
* 1928年：3月20日舉辦第一屆畢業典禮；同年6月校舍增築完成。
* 1930年：4月補習科（補校）獲設置認可修業一年，同年7月位在[[前金區|前金]]的宿舍完成。
* 1931年：舉辦補習科第一屆修業式。
* 1933年：新設游泳池完工。
* 1934年：舉辦創立十週年紀念式。
* 1937年：[[龜山相次|龜山相次]]接任校長。<ref name="takao"/>
* 1942年：小川七十二擔任校長。
* 1943年：改名為'''高雄州立高雄第一高等女學校'''。
* 1944年：岩本義烈擔任校長。

===戰後時期===
* 1945年：[[臺灣戰後時期|中華民國接管臺灣]]，11月接管委員會派陳東榮到校接收，暫代校長職務，12月改名為'''台灣省立高雄第一女子中學'''。<ref name="history">[http://w3.kghs.kh.edu.tw/~hsyyang/80/h1.htm 雄女文史]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}，高雄女中全球資訊網，2009年9月5日查閱。</ref>。
* 1946年：省立高雄第二女子中學之台灣籍學生併入該校，日籍學生移入第二女子中學。教育廳派吳伯俊擔任校長，學校分設初高中兩部修業各三年。
* 1947年：第二女子中學併入該校遂改名為'''台灣省立高雄女子中學'''。
* 1949年：陳頤擔任校長，初中部已經擴增至14班，高中部7班。
* 1953年：薛佩琦擔任校長。
* 1964年：成為完全三年制之完全高級中學。
* 1970年：改名為'''台灣省立高雄女子高級中學'''。
* 1971年：附設空中補校（1975年停辦）。
* 1974年：談太儁擔任校長。
* 1979年：師蔚霞擔任校長。因[[高雄市|高雄市]]升格[[直轄市|直轄市]]改為'''高雄市立高雄女子高級中學'''。雄女儀隊創立。
* 1984年：開始舉辦雄中、雄女聯合大露營活動。
* 1991年：[[雄中|雄中]]、[[雄女|雄女]]校長對調，朱迺武擔任校長。
* 1995年：陳有華擔任校長。
* 2002年：余碧芬擔任校長。
* 2009年7月：林全義擔任校長。
* 2013年8月：黃秀霞擔任校長。
* 2018年8月：林香吟擔任校長。

== 校服 ==
{{Unreferenced section|time=2013-08-21T20:13:10+00:00}}
:制服長短袖皆為白色國民領式的襯衫，左前胸部份皆有一個口袋，口袋上方繡上紅色的高雄女中字樣，口袋上面繡上年級槓(紅槓)，年級槓下方則繡上學號及班別，年級槓的數量會隨著年級增加。
:制服左胸口前口袋，年級與學號的辨識方法除了學號年份以外，在口袋處會以紅緞帶辨認年級，一槓為一年級、二槓為二年級、三槓為三年級，與他校不同。
:夏季制服為黑色百摺裙，冬季制服為黑色長褲，以及冬天特有的紅或深藍兩色無袖針織背心，與黑色西裝外套。

=== 斑城與斑馬 === 
:高雄女中稱學校是斑城，稱雄女學生為斑馬。其原因就是來自-{制服}-顏色組成為黑色及白色。穿上制服的時候，頭髮是黑色的，上衣是白色的，裙子或褲子又是黑色的，而又規定得穿[http://www.appledaily.com.tw/appledaily/article/headline/20100927/32843142/ 過踝白襪]（已開放黑或白短襪），視覺上看起來像黑白相間的斑馬。

:運動服長短袖上衣以白色為底色，左胸上有著書法字體的雄女二字，運動服長短褲皆為藍紫色。

===流傳===
:雄女的運動服在許多學生中流傳著，因質地舒適，就算升上大學也有許多校友在大學的宿舍也會穿著，甚至出了社會也會作為平日居家服。

===規定===
:雄女目前比較嚴格的校規就屬一定得穿著制服或繡著學號的運動服出入校門。
:雄女校長[[黃秀霞|黃秀霞]]女士表示，男生不可以任意穿著雄女校服，尤其是雄中男同學，否則將採取法律途徑處理<ref>[http://www.libertytimes.com.tw/2014/new/jan/9/today-south20.htm]</ref>。

== 學生生活 ==
=== 雄中雄女聯合大露營 ===
[[雄中|雄中]]雄女聯合大露營又被稱為雄中雄女大露營，為台灣唯一一項由男女兩校共同聯合舉辦的[[露营|露營]]活動。<ref name = "google">{{Cite web |url= https://docs.google.com/viewer?a=v&q=cache:BXUGFICVqCoJ:w2.kshs.kh.edu.tw/bulletin_kshs/files/6217/%25E9%259B%2584%25E4%25B8%25AD%25E9%259B%2584%25E5%25A5%25B3%25E9%25A3%25B2%25E6%2596%2599%25E6%2597%25A5(%25E5%25A5%25B6%25E8%258C%25B6%25E6%2597%25A5)%25E8%25AA%25AA%25E6%2598%258E.doc+%E9%9B%84%E4%B8%AD%E9%9B%84%E5%A5%B3%E5%A4%A7%E9%9C%B2%E7%87%9F&hl=zh-TW&gl=tw&pid=bl&srcid=ADGEESizUbXEo7mdKy9T8cLVzjxbRxbUMDTlrPtixq4czcY1KW5pTBmEedHK2346r5a1YcN_os31OgvHdPKW7_3EmzZysYpv3ef5pr7w9Z2aX_HYjBJ428waoE8x9hqnjTUeY7CJh2N2&sig=AHIEtbSb6XXLyPlnpJbkHPzVViyuYRl9gQ |title= 有關雄中雄女「飲料日」（奶茶日）說明 |author=  |work= [[Google文件|Google文件]] |date=  |publisher= 高雄市立高雄高級中學 |language=  |accessdate= 2011年8月7日 |quote=  }}{{zh-tw}}</ref>雄中雄女大露營最早是在1985年4月在當時擔任[[高雄市市長|高雄市市長]][[許水德|許水德]]的支持之下，由高雄中學校長[[朱迺武|朱迺武]]與[[高雄市立高雄女子高級中學|高雄女中]]校長[[師蔚霞|師蔚霞]]的合作之下開始舉辦。<ref name = "google"/><ref>{{Cite web |url= http://www.kghs.kh.edu.tw/kghshistory/album7493.html |title= 雄 女 大 事 紀 - 民國74年 ~ 民國93年 |author=  |date=  |publisher= [[高雄市立高雄女子高級中學|高雄市立高雄女子高級中學]] |language=  |accessdate= 2011年8月7日 |quote=  }}{{zh-tw}}</ref>高雄中學校方將此列為[[國防通識|國防通識]]課程的「[[公民|公民]]訓練」課程，因此露營活動兩天的期間也被歸類為正常上課時段。<ref name = "google"/>

雄中雄女大露營為每年性的活動，其主要對象為高雄中學與高雄女中的二年級學生，於二年級下學期第一次[[期中考試|期中考試]]前在[[高雄|高雄]][[澄清湖|澄清湖]]進行兩天一夜的露營活動。<ref name = "gsrat">{{Cite web |url= http://gsrat.net/news2/newsclipDetail.php?pageNum_RecClipData=11&&ncdata_id=713 |title= 雄中、雄女大露營20歲了 |author= 徐如宜 |work= [[台灣性別人權協會|台灣性別人權協會]] |date= 2004年2月24日 |publisher= 《[[聯合報|聯合報]]》 |language=  |accessdate= 2011年8月7日 |quote=  }}{{zh-tw}}</ref>露營時主要活動領導社團分別為高雄中學學聯會和高雄女中班聯會（位於和平廣場）、高雄中學童軍團和高雄女中童軍團（位於中心廣場）、高雄中學康輔社和高雄女中康輔社（位於忠孝廣場），並會於第一天晚上進行多項表演活動。<ref name = "gsrat"/><ref>{{Cite web |url= http://www.youtube.com/watch?v=h0B_jsN_nGo |title= 2011雄中雄女聯合大露營之康輔社晚會火之精靈火球表演 |author= rabbitscanfly |date= 2011年2月14日 |publisher= [[YouTube|Youtube]] |language=  |accessdate= 2011年8月7日 |quote=  }}{{zh-tw}}</ref>

=== 雄中雄女飲料日 ===
在雄中雄女大露營後兩校學生也會有相關聯誼行為，為了表達對露營時夥伴的問候，而利用段考結束後下午的溫書假買飲料送給對方。<ref name = "nextmedia">{{Cite web |url= http://tw.nextmedia.com/applenews/article/art_id/33250682/IssueID/20110316 |title= 雄中奶茶日 專車會雄女 |author= 涂建豐 |date= 2011年3月16日 |publisher= 《[[蘋果日報_(台灣)|蘋果日報]]》 |language=  |accessdate= 2011年8月7日 |quote=  }}{{zh-tw}}</ref>隨著這個非正式的活動漸成常態，兩校校方便規定僅有第一次段考結束後才可贈送飲品給對方學校。<ref>{{Cite web|url=http://takau.org/news/Article_Show.asp?ArticleID=5499|title=雄中雄女奶茶日 純粹為自發性活動|author=|date=2011年3月15日|publisher=大高雄新聞網|language=|accessdate=2011年8月7日|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20131213112643/http://takau.org/news/Article_Show.asp?ArticleID=5499|archivedate=2013年12月13日}}{{zh-tw}}</ref>由於高雄中學與高雄女中的段考次數並不同，因此在每學期第一次段考後便會有學生互送[[饮料|飲料]]，也有人送飲料以外的食物。<ref>{{cite news |language =  | author =  | coauthors =  | url =  | title = 雄中雄女午餐約會，奶茶傳情 | publisher = [[聯合新聞網|聯合新聞網]] | pages =  | date = 2000年8月24日 | accessdate = 2011年8月7日 }}{{zh-tw}}</ref>對於這一情況，部分媒體稱呼其為「飲料日」或「奶茶日」。<ref>{{Cite web|url=http://mag.udn.com/mag/campus/storypage.jsp?f_ART_ID=307050|title=禁放溫書假 擋不住雄中雄女「飲料日」|author=徐如宜|date=2011年3月16日|publisher=[[聯合新聞網|聯合新聞網]]|language=|accessdate=2011年8月7日|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20110425121318/http://mag.udn.com/mag/campus/storypage.jsp?f_ART_ID=307050|archivedate=2011年4月25日}}{{zh-tw}}</ref><ref>{{Cite web |url= http://www.libertytimes.com.tw/2011/new/mar/31/today-south9.htm |title= 雄中雄女奶茶日 紛擾半個月 道歉落幕 |author=  |date= 2011年3月31日 |publisher= 《[[自由時報|自由時報]]》 |language=  |accessdate= 2011年8月7日 |quote=  }}{{zh-tw}}</ref>不過2011年時因應[[中華民國教育部|中華民國教育部]]中禁止於期中考試當天下午放假，高雄市教育局宣佈取消段考後的「自主學習」時間，也使得3月15日時，高雄中學校方基於保護學生的立場，租借了16輛[[遊覽車|遊覽車]]搭載學生前往高雄女中。<ref name = "nextmedia"/>現今兩校學生之間這個「常態」的活動已經被迫中止，但仍有不少學生利用段考下午放學後的時段送飲料給在友校的朋友們。2013年後再次恢復飲料日，目前飲料日之確切時間點已定為高雄中學校慶之後的週一補假，當日高雄女中的女學生依然正常上學，中午午休時間便是浩大壯觀的飲料日。目前這個兩校之間的交流活動已被含校方在內的多數人視為一定要參加一次的傳統。
