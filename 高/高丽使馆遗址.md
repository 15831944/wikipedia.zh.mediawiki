'''高丽使馆遗址'''，位于[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[海曙区|海曙区]][[月湖_(宁波)|月湖]]风景区镇明路宝奎巷内，为[[北宋|北宋]]时期迎接[[高丽国|高丽]]使节和商团所建，后于1130年因战火毁损。1984年10月高丽使馆遗址被公布为海曙区文物保护单位，现在原址宝奎巷建筑上开设'''明州与高丽交往史陈列室'''。

== 历史 ==
高丽使馆遗址因宋徽宗在明州设置“高丽司”，始建于北宋政和七年（1117年），后因金兵南下，原建筑毁于1130年。<ref>{{cite web|url=http://zw.nbwh.gov.cn/art/2013/2/21/art_608_4996.html|title=高丽使馆遗址|date=2013-02-21|publisher=宁波市文化广电新闻出版局}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>原建筑以东以“宝奎巷故里”的西南墙角南北走向的延长线为界、南以大樟树东西走向的延长线为界、西以月湖东岸的人行道为界、北以现少儿音像馆为界。后在原址宝奎巷附近建有酒都务、[[史浩|史浩]]故居、宝奎庙等建筑，作为宝奎巷历史街区（海曙区文物保护点）保留至今。1984年10月，高丽使馆遗址被公布为海曙区文物保护单位。1999年于原址进行考古挖掘，发现了宋代建筑遗址和大量的[[越窑|越窑]][[青瓷|青瓷]]、[[韩瓶|韩瓶]]<ref>{{cite web |url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=922&type=2 |title=高丽使馆遗址|date=2006-02-10 |website= |publisher=宁波市文物保护管理所 |access-date= |quote=}}</ref>。 目前于宝奎巷建筑设立明州与高丽交往史陈列室。<ref>{{cite web |url=http://www.nbwb.net/pd_bwg/info.aspx?Id=24480&type=2 |title=明州与高丽使馆交往陈列室（高丽使馆遗址）|date=2012-09-24 |website= |publisher=宁波市文物保护管理所 |access-date= |quote=}}</ref>

有学者认为，高丽使馆遗址和[[宋代|宋代]]瓷器贸易有一定联系，部分高丽瓷器进入南宋宫廷的路线应是海运至明州“高丽司”后，再海运至杭州。<ref>{{cite web |url=http://www.zhejiangmuseum.com/filestorefg/a1/2017/201703/20170301/43d3fc60fe4a11e6990000163e024754.pdf?ori_name=%E8%BF%91%E5%B9%B4%E6%9D%AD%E5%B7%9E%E5%9F%8E%E5%8C%BA%E5%87%BA%E5%9C%9F%E5%AE%8B%E4%BB%A3%E5%90%8D%E7%AA%91%E7%93%B7%E7%89%87.pdf |title=近年杭州城区出土宋代名窑瓷片|author=蔡乃武 |date= |website= |publisher=浙江省博物馆 |access-date= |quote=至于高丽青瓷进入南宋宫廷的路线，应该是从海道到达宁波，那里有高丽国使馆，然后再从宁波经海路到达杭州。今年年初宁波博物馆拟征集的一件高丽瓷刻划花梅瓶，高达40厘米，修复率达90%以上，它的形貌特征与凤山门出土的梅瓶完全一样，宁波方面告诉我，这件梅瓶的出土地点就是宁波的南宋高丽使馆遗址。这便是一件重要的物证。}}</ref>

== 现状 ==
高丽使馆遗址位于现[[月湖_(宁波)|月湖]]公园内，而宝奎巷一带则成为海曙区文物保护点，周边有[[史浩|史浩]]故居、[[平桥水则碑|平桥水则碑]]等文物保护单位，但高丽使馆遗址存在石碑文字模糊不清，铜像掉色严重，且存在乱贴广告的现象。<ref>{{cite web |url=http://www.whcn.org/Detail.aspx?Id=1776 |title=海洋文化遗迹亟待保护|date=2016-07-26|publisher=苏州市世界文化遗产古典园林保护监管中心（联合国教科文组织亚太地区世界遗产培训与研究中心苏州分中心）|quote=再如位于海曙区月湖风景区镇明路宝奎巷内的高丽使馆遗址石碑文字模糊不清，铜像掉色严重，且存在乱贴广告现象。}}</ref>
