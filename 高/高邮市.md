'''高邮市'''位于[[中国|中国]][[江苏省|江苏省]]北部，[[京杭大运河|京杭大运河]]沿岸，[[高邮湖|高邮湖]]畔，为[[扬州市|扬州市]]代管的县级市。高邮是汉唐时期的各地传信机构，也是中国唯一以“邮”命名的城市。<ref>[http://www.gaoyou.gov.cn/zjgy/content.php?id=50617 走进高邮-中国高邮政府门户网站] {{webarchive|url=https://web.archive.org/web/20110520190410/http://gaoyou.gov.cn/zjgy/content.php?id=50617 |date=2011-05-20 }}</ref>

2016年11月22日，国务院批复同意将江苏省高邮市列为国家历史文化名城。

== 历史 ==
前223年，秦王[[嬴政|嬴政]]在境内筑高台、置邮亭，故名高邮，别称秦邮。后人又称高沙、珠湖、盂城。[[西汉|西汉]]武帝元狩五年（前118年）由[[东阳县_(秦朝)|东阳县]]析置'''[[高邮县|高邮县]]'''，属[[广陵国|广陵国]]。[[南北朝|南北朝]]时曾置广邺郡、神农郡，又析临泽、三归、竹塘三县。[[隋|隋]]初郡废，三县并入高邮县。[[宋代|宋代]]先后筑有二城，名旧城、新城。宋、元时期，因置[[高邮军|高邮军]]、[[承州|承州]]、[[高邮路|高邮路]]、[[高邮府|高邮府]]，领辖过[[高邮|高邮]]、[[兴化|兴化]]、[[宝应|宝应]]三县，为淮扬间繁华之地。[[明代|明代]]撤县建[[高邮州|高邮州]]，辖[[兴化|兴化]]、[[宝应|宝应]]两县。自[[清代|清代]][[乾隆|乾隆]]时起，遂为[[散州|散州]]。古代高邮，史称“江左名区、广陵首邑”。

[[民国|民国]]元年（1912年）废州为县。在[[抗日战争|抗日战争]]和[[国共内战|国共内战]]期间，高邮县[[澄子河|澄子河]]以南地区曾属[[江高行政区|江高行政区]]、[[江都县|江都县]]，高邮县湖西地区曾先后属[[天高县|天高县]]([[天长县|天长县]])、[[甘泉县_(清朝)|甘泉县]]、[[东南县|东南县]]、[[仪扬县|仪扬县]]、[[仪征县|仪征县]]。[[中华人民共和国|中华人民共和国]]成立后设高邮县。1991年2月经国务院批准撤县设市（县级），4月1日正式建[[高邮市|高邮市]]。

== 地理 ==
高邮位于北纬32°38'~33°05',东经119°13'~119°50'，地处江淮平原南端，东邻[[兴化市|兴化市]]，南连[[江都区|江都区]]、[[邗江区|邗江区]]、[[仪征市|仪征市]]，西接[[天长市|天长市]](安徽省)、[[金湖县|金湖县]]，北界[[宝应县|宝应县]]。市区西南经扬州市达[[南京市|南京市]]165千米，东南经[[无锡市|无锡市]]、[[苏州市|苏州市]]抵[[上海市|上海市]]331千米,北距[[北京市|北京市]]980千米。高邮属于水乡平原，地势平坦，海拔一般在2～3.3米左右。土质主要为粘土，土层较厚，工程地质条件较好。[[高邮湖|高邮湖]]为江苏第三大湖，依傍着[[京杭运河|京杭运河]]，众多湖滩分布东西，数百条河流交错。高邮属亚热带温润气候区，主导风为东南风，平均风速3.6米/秒；年均温15℃；年均降雨1030毫米，无霜期为217天。气候温和、雨量充沛、四季分明、日光充足。 

== 经济 ==
养[[鸭|鸭]]业发达，特产[[高邮鸭蛋|高邮鸭蛋]]（高邮咸鸭蛋），系用高邮麻鸭的蛋制成，双黄、多油，是受国家保护的[[中国地理标志产品|原产地域产品]]。其他农产品还有小麦、水稻等。工业主要有机械、电器、燈具制造等门类。

== 物产 ==
*矿产资源有[[石油|石油]]、[[泥炭|泥炭]]、[[矿泉水|矿泉水]]等。
*野生动物有[[秧鵽|秧鵽]]、[[麻鸭|麻鸭]]、[[银鱼|银鱼]]等。
*土特产品有双黄鸭蛋、[[秦邮董糖|秦邮董糖]]、[[界首茶干|界首茶干]]、[[水晶月饼|水晶月饼]]。

== 民族 ==
高邮下属的[[菱塘回族乡|菱塘回族乡]]是江苏省唯一的[[民族乡|民族乡]]。

== 古迹 ==
高邮是江苏省的历史文化名城，古迹较多，[[马可·波罗|马可·波罗]]曾在其游记中记载过这里。
*城南的[[盂城驿|盂城驿]]兴建于[[明|明]][[洪武|洪武]]年间，地处[[京杭大运河|京杭大运河]]旁[[南京|南京]]通往北方的交通要道上，其规模仅次于[[北京|北京]]以西的[[鸡鸣驿城|鸡鸣驿城]]，是中国目前仅有的保存较完整的古代驿站。
*[[龙虬庄遗址|龙虬庄遗址]]则是一处大约5000-7000年前的[[新石器时代|新石器时代]]古人类文化遗址。
*东西双塔（[[高郵淨土寺塔|高郵淨土寺塔]]、[[镇国寺塔|镇国寺塔]]）
*[[全国重点文物保护单位|全国重点文物保护单位]]：[[盂城驿|盂城驿]] [[龙虬庄遗址|龙虬庄遗址]]
*[[文游台|文游台]]

== 著名人物 ==
*[[cmn:秦观|秦观]]，字太虚，又字少游，汉族，高邮人，称苏门四学士，别号邗沟居士、淮海居士，世称淮海先生。官至太学博士，国史馆编修。是北宋文学史上的一位重要作家，著名《鹊桥仙》。
*马俊，明朝洪武十五年（1382年）在任上去世后，被追封为武毅将军。官授管军千户。
*王磐，江苏高邮人，明代著名画家，曲作，被称为南曲之冠。
*高邮王氏：[[王安國_(雍正進士)|王安国]]、[[王念孙|王念孙]]、[[王引之|王引之]]等。
*[[cmn:汪曾祺|汪曾祺]]（1920年－1997年），中国当代作家、散文家、戏剧家、京派作家的代表人物。
*乔登江（1928年－2015年）， 两弹功臣、著名核物理学家、核技术应用专家，中国工程院院士。
*秦怡（1922年－），祖籍江苏高邮，中国著名电影、话剧、电视剧女演员、艺术家。获得金鸡百花终生成就奖。
*吴登云，被评为100位新中国成立以来感动中国人物之一。曾荣获全国五一劳动奖章和白求恩奖章。

== 语言 ==
[[高邮话|高邮话]]：高邮方言属于江淮官话洪巢片扬州小片，是扬州地方经典的江淮官话。
