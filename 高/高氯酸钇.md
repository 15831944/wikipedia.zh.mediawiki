'''高氯酸钇'''（{{lang-en|Yttrium perchlorate}}）是[[钇|钇]]的[[高氯酸|高氯酸]]盐，分子式为Y(ClO<sub>4</sub>)<sub>3</sub>。

== 制取 ==

将[[氧化钇|氧化钇]]溶于稀[[高氯酸|高氯酸]]溶液，可制得八水合高氯酸钇；[[五氧化二磷|五氧化二磷]]存在时，将八水合高氯酸钇置于72℃下、真空中脱水，可制得三水合高氯酸钇。<ref name="journal_1"/>

== 化学性质 ==

25℃下，三水合高氯酸钇与[[18-冠-6|18-冠-6]]在[[丙酮|丙酮]]中形成摩尔比为1:1和1:2的2种[[配合物|配合物]]，在[[甲醇|甲醇]]或[[乙醇|乙醇]]中形成摩尔比为4:3、1:1和1:2的3种配合物；20℃下，在[[乙腈|乙腈]]中形成摩尔比为1:1和1:2的2种[[配合物|配合物]]；15℃下，在乙醇中形成摩尔比为2:3、4:3、1:1和1:2的4种配合物。各种配合物都带有一定数目的[[有机溶剂|有机溶剂]]分子。<ref name="journal_1"/>

从[[红外光谱|红外光谱]]中可以看出，[[18-冠-6|18-冠-6]]参与了对[[钇|钇]]离子的配位，并且在不同的配合物中18-冠-6的配位有些差别。<ref name="journal_1"/>

由于高氯酸钇与[[水杨酸甲酯|水杨酸甲酯]]的共同作用能促使[[高氯酸根|高氯酸根]]离子将[[一氧化氮|一氧化氮]]氧化为[[二氧化氮|二氧化氮]]并进一步将其转变为[[三氧化二氮|三氧化二氮]]，[[一氧化氮|一氧化氮]]与高氯酸钇与水杨酸甲酯的[[配合物|配合物]]在一定温度下能发生有特征性的显色反应，形成稳定的[[蓝绿色|蓝绿色]]产物。<ref name="thesis_1"/>

== 用途 ==

高氯酸钇与[[水杨酸甲酯|水杨酸甲酯]]的混合物可以用来捕获、稳定储存和测定[[一氧化氮|一氧化氮]]，也可以用来保护生物体内[[蛋白质|蛋白质]]等有机大分子不被过量的一氧化氮损害。<ref name="thesis_1"/>
