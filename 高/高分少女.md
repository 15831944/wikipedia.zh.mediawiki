《'''高分少女'''》（{{lang-ja|ハイスコアガール}}）是[[日本漫畫家|日本漫畫家]][[押切蓮介|押切蓮介]]創作的[[漫畫|漫畫]]。
在2012年的「{{tsl|ja|テレビブロス#ブロスコミックアワード|Bros Comic Award}}（{{lang|ja|ブロスコミックアワード}}）」獲頒大賞。在2013年的《[[這本漫畫真厲害！|這本漫畫真厲害！]]》得到男生編第二名的榮譽。

== 概要 ==
以1990年代的[[對戰格鬥遊戲|對戰格鬥遊戲]]熱潮為舞台，由引發熱潮的遊戲《[[快打旋風Ⅱ|快打旋風Ⅱ]]》造成男主角「矢口春雄」與女主角「大野晶」邂逅的戀愛喜劇作品<ref>{{Cite web |url=http://comic.qq.com/a/20131223/007351.htm |title=获奖漫画《高分少女》动画化决定 |work=腾讯动漫 |date=2013-12-23 }}</ref>。

== 故事 ==
世界動盪的1991年，日本長崎縣[[雲仙岳|雲仙普賢火山]]爆發、[[波灣戰爭|波灣戰爭]]開打、[[日本泡沫經濟|泡沫經濟]]的末期……一個嗜電玩成癡的小學生「矢口春雄」時不時就會去遊戲中心打街機，他把遊戲中心視為自己崇高的殿堂，只有「豪指春雄」能稱霸這裡。然而不幸地，他的一切幻想終被玷污。一個可怕的女人出現了－－「大野晶」，在一次電玩對戰中與春雄大打出手。本是不同次元的兩人，從此結下了一段孽緣。

電玩冒險與校園戀愛的交織，開啟了一段非凡的旅程－－

==人物介紹==
; {{nihongo|矢口春雄|矢口 春雄（やぐち はるお）|聲：[[天崎滉平|天崎滉平]]}}
: 初登場時小學六年級。
: 是出了名的電玩狂，自稱「豪指春雄」，視大野晶為勁敵，且漸漸重視晶的存在。之後為了跟晶上同一所高中，曾一度放棄電玩努力備考，可惜最後仍以失敗告終，但還是和宮尾一起考上了第二志願。
; {{nihongo|大野晶|大野 晶（おおの あきら）|聲：[[鈴代紗弓|鈴代紗弓]]}}
: 初登場時小學六年級。
: 文武雙全的千金小姐。在第二學期搬家至洛杉磯（第9話），於中三時歸國。從不說話，常以攻擊春雄來表達各種情緒和想法，相當喜歡與春雄在一起，但迫於萌美老師的規範而常常無法如願。當春雄身邊有女性與之親近時會明顯表現出不悅。
; {{nihongo|日高小春|日高 小春（ひだか こはる）|聲：[[廣瀨有紀|廣瀨有紀]]}}
: 初登場時初中二年級。
: 原本對電玩一竅不通，被春雄拉進電玩的世界後開始磨練自己的技巧，只為了更接近春雄。常被男同學告白但都拒絕了，內心只喜歡著春雄。在知道春雄與大野之間的曖昧關係後，以交往為條件對春雄提出挑戰，但最終在三戰兩勝的比賽中輸給了春雄，不過仍未放棄和春雄交往。
; {{nihongo|宮尾光太郎|宮尾 光太郎（みやお こうたろう）|聲：[[興津和幸|興津和幸]]}}
: 初登場時初中二年級。
: 春雄的好友。曾經打算向大野晶告白，但是在證見了春雄與晶的不尋常關係後，決定放棄。常常在重要關頭時給春雄意見，也是第一個注意到小晶喜歡春雄的人。
; {{nihongo|鬼塚千尋|鬼塚 ちひろ（おにづか ちひろ）|聲：[[御堂ダリア|御堂ダリア]]}}
: 初登場時小學六年級。
: 春雄、晶小六時的同班同學、升上同一所中學。自認是美人，化妝後能讓宮尾認不出來。
; {{nihongo|沼田|沼田（ぬまた）|聲：[[中村悠一|中村悠一]]}}
: 春雄所就讀的中學的生活指導教師。
; {{nihongo|業田 萌美|業田 萌美（ごうだ もえみ）|聲：[[伊藤靜|伊藤靜]]}}
: 大野家的家庭教師，做事一板一眼，對於晶的管教相當嚴格，連管家爺爺也拿她沒輒。曾一度企圖拆散春雄與晶。
; {{nihongo|土井玄太|土井 玄太（どい げんた）|聲：[[山下大輝|山下大輝]]}}
: 春雄、晶小六時的同班同學，高中與春雄、宮尾同校。非常喜歡大野晶，追求行動時常碰壁。
; {{nihongo|大野真|大野 真（おおの まこと）|聲：[[赤崎千夏|赤崎千夏]]}}
: 晶的姐姐，大野家的頭痛人物，就連業田都拿她沒轍。雖然身為大野家的長女卻不願接受大野家的教育方針，也因此讓晶受到過於嚴格的管教，對此真有些自責。

== 侵權爭議 ==
由於漫畫內容使用了許多過去有名的遊戲畫面或內容，但由於作者實際上沒有取得授權，剛開始遊戲公司並沒有提告，但直到決定動畫化時，[[SNK_Playmore|SNK Playmore]]遊戲公司終於提侵權告訴，導致漫畫被暫停連載，並回收了部分已出版的單行本<ref>[http://www.animen.com.tw/Page/News/NewsContent.aspx?ID=NS14090018 漫畫《高分少女》侵權事件持續延燒，遊戲大廠「SEGA」傳出也遭侵權！]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://news.aniarc.com/news/98686/snk%E5%90%91square-enix%E6%8F%90%E5%87%BA%E8%A8%B4%E8%A8%9F%EF%BC%81-%E3%80%8A%E9%AB%98%E5%88%86%E5%B0%91%E5%A5%B3%E3%80%8B%E7%9A%84%E4%BE%B5%E6%AC%8A%E5%8D%B1%E6%A9%9F SQUARE ENIX因違反著作權遭警察搜索 SNK對《高分少女》使用遊戲角色侵權提出訴訟]</ref><ref>[http://www.qoos.com/tech/%E3%80%8A%E9%AB%98%E5%88%86%E5%B0%91%E5%A5%B3%E3%80%8B%E6%B6%89%E4%BE%B5%E6%AC%8A-1737251.html 《高分少女》涉侵權]- Qoos 科技頻道</ref>。但是到了2015年8月24日，SNK Playmore願意與SQUARE ENIX和解，並檢討展開繼續進行連載和發行單行本的作業<ref name="sqexrel20150826">{{ja}}[http://www.jp.square-enix.com/company/ja/news/2015/html/e9d94051bc90936cadf6b66e9e7e5604.html 和解による紛争の解決に関するお知らせ],スクウェア・エニックス,2015年8月26日</ref><ref>{{ja}}[http://nlab.itmedia.co.jp/nl/articles/1508/26/news112.html 「ハイスコアガール」めぐる紛争でスクエニとSNKプレイモアが和解],ITmedia,2015年8月26日</ref>。

另外，並非所有知名遊戲及內容未經過許可，在這之前作者和出版社已經有向[[卡普空|卡普空]]、[[萬代南夢宮娛樂|南夢宮]]（今併入改名為萬代南夢宮娛樂）取得許可，並於連載開始之後防止發生事後遊戲公司正式提出告訴，
有開出條件經過許可<ref>{{Cite news |title=「高分少女」問題，卡普空、萬代南夢宮娛樂、世嘉等遊戲角色的使用許可|newspaper=ITmedia News|date=2014-08-06|url=http://www.itmedia.co.jp/news/articles/1408/06/news130.html |accessdate=2016-02-26|language=ja}}</ref>。
