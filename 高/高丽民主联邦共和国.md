'''高丽民主联邦共和国'''（{{kor|k='''고려민주연방공화국'''|hanja='''高麗民主聯邦共和國'''}}）是[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]已故最高領導人[[金日成|金日成]]在1980年10月10日[[朝鲜劳动党第六次代表大会|朝鲜劳动党第六次代表大会]]上所作的[[朝鲜劳动党中央委员会|朝鲜劳动党中央委员会]]工作总结报告中提出的[[兩韓統一|-{zh-cn:朝鲜半岛统一; zh-sg:两韩统一; zh-hant:兩韓統一;}-]]後的國名<ref>{{Cite web |url = http://naenara.com.kp/ch/one/nation.php?1+koryo |title = 建立高丽民主联邦共和国的方案 |publisher = Naenara |language = zh-Hans |accessdate = 2014-02-15 }}</ref>，以一个民族、[[一国两制|一个国家、两种制度]]和两个政府为基础。<ref>{{cite news |author=记者杨子岩 |last= |first= |date=2014年10月14日 |title=韩国会接受朝鲜的统一方案吗？ |url=http://paper.people.com.cn/rmrbhwb/html/2014-10/14/content_1487567.htm |newspaper=人民日报海外版 |location= |access-date=2017-05-18 }}</ref>

[[金日成|金日成]]主張朝鮮国自主和平统一，並使用朝鮮半島歷史上第一個统一國家[[高麗國|高麗國]]作為國號。

1997年8月4日，[[金正日|金正日]]发表了《切实贯彻伟大领袖金日成同志关于统一祖国的遗训》，将《建立高丽民主联邦共和国方案》与《统一祖国三项原则》《为统一祖国，实现全民族大团结的十大纲领》一起，定为“统一祖国的三大宪章”。<ref>{{Cite web |url = http://www.naenara.com.kp/ch/great/guid.php?7 |title = 祖国统一政策 |publisher = Naenara |language = zh-Hans |accessdate = 2014-02-15 }}</ref>

2013年10月10日，[[朝鲜劳动党|朝鲜劳动党]]建党68周年纪念日，[[朝鲜劳动党|朝鲜劳动党]]官方报纸《[[劳动新闻|劳动新闻]]》登载文章，强调联邦制度才是最光明正大的民族共同统一方案。<ref>{{Cite web |url = http://news.ifeng.com/world/detail_2013_10/11/30241444_0.shtml |title = 朝鲜官媒：联邦制度才是最光明正大的民族统一方案 |date = 2013-10-11 |publisher = 凤凰网 |language = zh-Hans |accessdate = 2014-02-15 }}</ref>
