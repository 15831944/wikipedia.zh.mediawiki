'''高頻寬記憶體'''（英文：'''{{lang|en|High Bandwidth Memory}}'''，縮寫'''HBM'''）是[[超微半導體|超微半導體]]和[[SK_Hynix|SK Hynix]]發起的一種基於3D堆疊工藝的高效能[[DRAM|DRAM]]，適用於高[[内存带宽|記憶體頻寬]]需求的應用場合，像是[[圖形處理器|圖形處理器]]、網路交換及轉發裝置（如[[路由器|路由器]]、[[交換器|交換器]]）等。<ref>[http://isscc.org/doc/2014/2014_Trends.pdf ISSCC 2014 Trends] {{webarchive|url=https://web.archive.org/web/20150206093927/http://isscc.org/doc/2014/2014_Trends.pdf |date=2015-02-06 }} page 118 "High-Bandwidth DRAM"</ref>首個使用HBM的裝置是[[Radeon_Rx_300|AMD Radeon Fury系列]]顯示核心。<ref name='amd_fiji'>{{cite news|last1=Smith|first1=Ryan|title=The AMD Radeon R9 Fury X Review|url=http://www.anandtech.com/show/9390/the-amd-radeon-r9-fury-x-review|accessdate=1 August 2016|publisher=Anandtech|date=2 July 2015}}</ref><ref>{{cite news|url=http://www.enterprisetech.com/2014/03/25/future-nvidia-pascal-gpus-pack-3d-memory-homegrown-interconnect/|title=Future Nvidia ‘Pascal’ GPUs Pack 3D Memory, Homegrown Interconnect |last=Morgan|first=Timothy Prickett |date=March 25, 2014|publisher=EnterpriseTech|accessdate=26 August 2014|quote= Nvidia will be adopting the High Bandwidth Memory (HBM) variant of stacked DRAM that was developed by AMD and Hynix}}</ref>2013年10月HBM成為了[[JEDEC|JEDEC]]通過的工業標準，<ref name="HBM_JEDEC">[http://www.jedec.org/standards-documents/results/jesd235 HIGH BANDWIDTH MEMORY (HBM) DRAM (JESD235)], JEDEC, October 2013</ref>第二代HBM —— HBM2，也於2016年1月成為工業標準，<ref name="HBM2_JEDEC">{{cite web |url=https://www.jedec.org/news/pressreleases/jedec-updates-groundbreaking-high-bandwidth-memory-hbm-standard |title=JESD235a: High Bandwidth Memory 2 |date=2016-01-12}}</ref>[[NVIDIA|NVIDIA]]在該年發表的新款旗艦型[[NVIDIA_Tesla|Tesla運算加速卡]] —— Tesla P100、AMD的[[AMD_Radeon_Vega_系列|Radeon RX Vega系列]]、[[Intel|Intel]]的[[Xeon_Phi|Knight Landing]]也採用了HBM2。
