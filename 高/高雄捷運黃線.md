'''高雄捷運黃線'''，又稱'''高雄捷運都會線'''，是[[高雄捷運|高雄捷運]]規劃中路線之一，設置23座車站，全線皆位於[[高雄市|高雄市]]。黃線是地下化的中運量捷運系統，2019年1月，通過國發會核定並建請行政院同意可行性報告。

== 歷史 ==
黃線最早見於2002年的規劃案，但今黃線北段路線，早在1984年高雄市政府工務局都市計畫科委託[[交通部運輸研究所|交通部運輸計畫委員會]]進行的[[高雄捷運#規劃歷程|《高雄都會區大眾捷運系統長期發展建議路網》專案]]中，即已出現，在該專案中，中運量綠線自澄清湖起，沿著澄清路、本館路、建工路、民族路行駛，與今黃線相同。但該綠線於民族九如路口後即轉向九如路，南段路線就與今黃線不同。而至1990年高雄市政府委託顧問公司進行的《高雄都會區大眾運輸系統調查、分析、改善及捷運系統可行性研究規劃》專案中，民族、建工、本館路已無捷運路線經過。

=== 輕軌規劃時期 ===
[[File:KMRT_Yellow_Line_2000.png|thumb]]
高雄市政府捷運局於1998年12月委託顧問公司進行《高雄都會區大眾捷運系統長期路網運輸規劃》專案，並在2001年4月完成路網建議報告報行交通部，隔年8月核轉行政院，黃線首見於該計畫。當時計畫從[[高雄捷運環狀輕軌|輕軌環線]][[經貿園區站|正勤國宅站]]發車，經預定的[[高雄多功能經貿園區|多功能經貿園區]]園道四號（現為五號船渠、光華三路）、成功二路口向東行銜接[[光華路_(高雄市)|光華路]]，經光華路、[[民族路_(高雄市)|民族路]]，再轉入建工路，終於高速公路西側，系統建議採用輕軌，路線全長9公里，規劃設置11座車站。其一心光華路口至民族建工路口段與當時規劃的[[高雄捷運藍線_(廢線)|藍線]]共線。機廠與藍線、輕軌環線規畫共同使用，設於瑞南街、凱興街 、前鎮河間的土地（今為75期重劃區）。另外專案中也有建議在中山路轉南，從還未興建的[[時代大道|多功能經貿園區中央大道]]至成功路的替代方案。而在該專案中，黃線是市區路網五路線優先度排第四，僅優先於綠線，建議待多功能經貿園區開發後才興建。

2003年12月捷運局再委託顧問公司辦理後續細節設計的《高雄都會區大眾運輸系統工程計畫長期路網規劃作業》專案，並於2005年2月交通部審定；該計畫中藍線改走大順路，不再與黃線共線，僅交會於建工大順路口。該計畫中將[[鳳山本館線|鳳山線]]、棕線延伸至鳥松，在神農路、神農路150巷、大同路、坔埔大排間設置鳥松機廠；也一併將黃線向東延伸，經本館路，並在澄清路、大埤路與此二線共線，抵達[[澄清湖|澄清湖]]和[[長庚醫院|高雄長庚醫院]]，最後於大埤、中正路口路線終止並進入鳥松機廠。路線全長13.574公里，規劃設置17座車站。在此專案中，黃線優先度排在第五位，評估經濟效益低落，但因[[益本比|益本比]]為正，所以仍建議興建。

[[File:KMRT_Yellow_Line_2010.png|thumb]]
後考量於光華路過窄，並且於夜間設有[[光華夜市|光華夜市]]，因此正勤國宅站發車後改經正勤路，小轉彎過[[獅甲站|獅甲站]]，再沿[[民權路_(高雄市)|民權路]]至[[民生路_(高雄市)|民生路]]小轉彎接回[[民族路_(高雄市)|民族路]]。此時藍線也已與臨港輕軌整併為環狀輕軌，黃線在建工大順路口與之交會。2009年12月捷運局又委託顧問公司進行《高雄都會區大眾捷運系統路網黃線、棕線及水岸輕軌規劃作業》專案，專案中黃線改由環狀輕軌[[夢時代站|夢時代站]]發車，經時代大道轉中山路轉乘[[凱旋站_(高雄市)|凱旋站]]及獅甲站，其餘路線不變<ref>{{cite journal|author1=高雄市政府捷運工程局|title=黃線及棕線規劃說明（圖） |journal=高雄捷運雙月刊 |date=2009-10-01|issue=22|pages=3|url=http://mtbu.kcg.gov.tw/cht/publish/report/9810/index.htm}}</ref>。由於大埤路棕線、黃線、鳳山線三線共線共站，但該段位於郊區，共線效益低；該專案亦另有於本館澄清路口以東改走青年路二段、文龍東路、鳳松路、中正路、大同路進鳥松機廠的方案。然而2011年12月該專案因其中的水岸輕軌併入環狀輕軌，導致整個專案中途停止，並未完成；隔年4月捷運局依法補償顧問公司。

由於輕軌推行進度緩慢，高雄市交通局另做了[[高雄公車捷運|高雄公車捷運]]的可行性評估；2011年11月《大高雄地區整體公車捷運系統路網可行性研究案結案報告書》中，黃線被列為優先路線之一。因為公車捷運不需要機廠，因此不需要延伸到鳥松來與公車捷運棕線共線使用機廠；加上公車捷運所需路幅較輕軌大，而本館路路幅不大；因此在該報告中，決定將黃線截短，並且因為公車捷運雙節車廂無法像輕軌直接反向行駛，因此在建工大昌路口轉進大昌路，並在大昌、皓東、正忠三條路形成一個環狀，再接回大昌路回程；而南端則多延伸到公車總站[[前鎮站|前鎮站]]來轉向。

2015年3月2日，[[高雄市政府捷運工程局|高雄市政府捷運工程局]]局長吳義隆表示：黃線輕軌正在進行可行性評估，部分路段重新規劃為高架化路段；上述路線在抵達澄清湖一帶後，再回到[[亞洲新灣區|亞洲新灣區]]，並構成環狀輕軌之後的第二環，並將正式名稱改為「都會延伸環線」<ref>{{cite news|title=雙軌雙環 串起高雄經濟脈絡|url=http://www.chinatimes.com/newspapers/20150302000143-260204|accessdate=2015-04-24|agency=中時電子報|date=2015-03-02}}</ref>。4月20日，吳義隆局長於[[高雄市議會|高雄市議會]]向議會作業務報告，根據其簡報資料，可看出第二環大致由先前計畫的黃線與棕線構成，並在長庚一帶改走公園路<ref>{{cite web|author1=高雄市議會|title=第2屆第1次定期大會捷運工程局業務報告|url=http://cissearch.kcc.gov.tw/Upload/Attachment/BusinessReport/886/ec4edec4-c86c-4c56-9f73-9a44e85a49e3.ppt|accessdate=2015-04-24|format=ppt|date=2015-04-20}}</ref>。其中黃線改為用MA做編號，設25站，棕線改為編號MB，設13站。2015年底，與顧問公司簽約進行《高雄都會區大眾捷運系統都會延伸環線建設及周邊土地開發計畫可行性研究》專案，該專案還包含了都會延伸環線的兩條配套連接線[[高雄捷運青線|青線]]（信義國小站至高鐵左營站）、[[鳳山本館線|藍線]]（時代凱旋路口站至鳥松機廠）。2016年9月，專案提報交通部；11月交通部審查意見表示運量預測偏低，建議僅提報最優先路線。2017年1月，該專案去除兩連接線的可行性研究報告書原則同意通過。

=== 中運量規劃時期 ===
[[File:KMRT_Yellow_Line_2017.png|thumb]]

2017年2月14日，四名民進黨立委[[劉世芳|劉世芳]]、[[李昆澤|李昆澤]]、[[賴瑞隆|賴瑞隆]]、[[許智傑|許智傑]]聯合召開記者會，發表重新規劃的高雄捷運黃線路線，預定採用[[地下車站|地下型式]]之[[中型鐵路系統|中運量捷運系統]]，路線也與過去的黃線不同，結合了棕線、[[鳳山本館線|鳳山線]]以及[[高雄捷運綠線|綠線]]五甲段，成Y字型路網，共線段以Y為編號，西側段以YA編號，東側以YB編號，總計26站；預估經費1430億，向行政院爭取本計畫納入「前瞻基礎建設計畫」。3月23日，行政院正式宣布高雄捷運黃線納入「[[前瞻基礎建設計畫|前瞻基礎建設計畫]]」，並命名為「高雄捷運都會線」<ref>{{cite news|title=砸千億！高捷新增黃線　紅線延伸到路竹|url=http://www.appledaily.com.tw/realtimenews/article/new/20170323/1082864/|accessdate=2017-03-23|agency=蘋果即時新聞|date=2017-03-23}}</ref>。3月27日，捷運局將修改為地下化捷運系統的黃線可行性研究報告送至交通部審核。

2017年3月27日，第一次提報高雄捷運都會線可行性研究予交通部審查，其中民權五福、五甲南昌、澄清青年等路口站點及前鎮區公所、澄清湖棒球場站（Y14）取消，五甲南正路口站點北移到鳳南路口，澄清湖風景區大門口站點（Y12）南移至正修科大，原走鳥松濕地北側改為南側，並改為全以Y編號，總計21站。4月5日，行政院核定前瞻基礎建設軌道建設納入「高雄捷運都會線計畫」<ref>院臺經字第1060009184號函</ref>。5月12日，交通部函送審查意見予市府，經依審查意見完成修正，5月19日，第二次提報予交通部審查。由於路線倉促設計，所有交會路線均設計為站外徒步轉乘，造成大眾不滿，捷運局表示將會再修正為站內轉乘<ref>[https://news.ltn.com.tw/news/life/breakingnews/2096866 1430億的高捷黃線要站外轉乘 網友昏了]</ref>。10月5日，交通部函送審查意見予市府，經依審查意見完成修正，11月10日，第三次提報予交通部審查。12月20日，交通部辦理黃線現勘；12月22日，交通部召開黃線初審會議。2018年1月2日，交通部函送初審會議紀錄予高雄市府，經依審查意見完成修正。年初，Y6站東移至高速公路以東以便轉乘[[高雄捷運銀線|銀線]]，Y11站從四維路口略北移至苓雅路口，並增設Y11A站，Y14從文衡路口南移至寶業滯洪池西側，Y15南移至台鐵以南，Y16南移至橘線以南。2月27日第四次提送予交通部審查。3月7日，交通部函請高鐵局辦理審查作業。

2018年7月19日，將鳥松機廠東移1.36公里至位於坔埔-{里}-的鳥松第三公墓，並新增Y1A站，總經費估算調整為1438億元，比原1455億元少17億元。捷運局表示，新規劃機廠用地多為公有，大幅減少徵收費用，所以整體費用可望降低，相關計畫正由交通部審核中，設定期程比原先預期晚一年，如果可行性評估過關，會設法在綜合規劃、環評、都市計畫變更等層面同步進行，以縮短整體工期<ref>[http://m.ltn.com.tw/news/local/paper/1217728 高捷黃線路段延伸 將新增一站]自由時報 </ref>。9月12日，交通部原則通過可行性研究，待完成報告後轉報請行政院核定<ref>{{cite news|title=高捷黃線可行性報告書 審議通過|url=http://www.idn.com.tw/news/news_content.php?catid=5&catsid=6&catdid=0&artid=20180912abcd008|accessdate=2018-09-12|agency=自立晚報|date=2018-09-12}}</ref>。2019年1月8日，[[國發會|國發會]]召開審查會議，會議結論同意通過，並建議行政院同意黃線可行性研究報告<ref>{{Cite news|url=https://www.chinatimes.com/realtimenews/20190108004272-260410|title=韓國瑜爭取續推的高雄捷運黃線 國發會建議政院同意|author=|date=2019-01-08|work=|accessdate=}}</ref>。

== 延伸線 ==
; 前鎮漁港延伸線
在《高雄都會區大眾捷運系統都會線建設及周邊土地開發計畫可行性研究》中規劃有前鎮漁港延伸線，分為兩方案。方案A延著鎮中路以及其直線延伸的民房地下穿越，至新生路設站（大致為今大博鋼鐵公司處），再延著新生路往南，在漁港東二路轉進前鎮漁港設站。路線採疊式設計，共2.3公里，經費131.3億。方案B則為進Y21站前就轉進中山路，並轉進漁港路至新生路，並在約今漁港路加油站處設站，其後轉進新生路，剩餘路線與方案A相同。路線亦採疊式設計，共2.5公里，經費137.4億元。由於兩線經過人口均只有五千人左右，且造價昂貴，以及路幅問題，因此目前暫不推動。

; 屏東延伸線
2018年11月，高雄市捷運局委託顧問公司進行《高雄捷運延伸屏東整體路網、可行性研究、綜合規劃及環境影響評估》專案，評估四條路線，包含已有的[[大寮屏東線|大寮屏東線]]、[[林園東港線|林園東港線]]，並增加[[高屏第二東西向快速公路|高屏第二東西向快速公路]]興建時共構捷運，以及由黃線延伸的方案。由黃線延伸的方案，從Y1站還未進Y1A站前岔出往東北延伸，繞過仁美、水寮北側，從竹寮跨過[[高屏溪|高屏溪]]，穿過[[屏東機場|屏東基地南機場]]，終於台3線，轉乘屏東市區輕軌。

== 先導公車與接駁車 ==
