'''高富IC'''（平假名：たかとみインターチェンジ）是位於[[岐阜縣|岐阜縣]][[山縣市|山縣市]]的[[東海環狀自動車道|東海環狀自動車道]]之[[交流道|交流道]]。此交流道還未啟用。

== 連接道路 ==
* [[國道256號|國道256號]]（[[高富繞道|高富繞道]]）

== 年表 ==
* 2020年（預計） - [[東海環狀自動車道|東海環狀自動車道]][[大垣西IC|大垣西IC]]至[[關廣見IC|關廣見IC]]開通，此交流道也同時啟用<ref>{{cite web|title = 再評估結果（平成24年度事業継続箇所） 一般国道475号東海環状自動車道（関-養老）|url = http://www.mlit.go.jp/road/ir/ir-hyouka/23sai/2_h23_087.pdf|format=PDF|publisher = [[国土交通省|国土交通省]]|date = 2012年| accessdate = 2012年3月11日}}</ref>。
