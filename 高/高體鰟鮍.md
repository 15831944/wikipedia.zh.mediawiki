'''高体鰟鮍'''（学名：{{lang|la|''Rhodeus ocellatus''}}）为[[輻鰭魚綱|輻鰭魚綱]][[鯉形目|鯉形目]][[鲤科|鲤科]][[鰟鮍属|鰟鮍属]]的[[鱼类|鱼类]]，俗名'''土扁屎'''、'''鱊'''或'''牛糞鯽'''、'''牛屎鯽仔'''、'''紅目鯽仔'''<ref>{{cite web |url=http://tw.myblog.yahoo.com/kming-blog/article?mid=623&sc=1 |title=牛屎鯽(高體鰟鮍) |date=2008-06-05 |accessdate=2013-09-09 |language=zh-hant |author=[http://tw.myblog.yahoo.com/kming-blog/profile 小明] }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name=pchome>{{cite web |url=http://mypaper.pchome.com.tw/068060/post/1323600830 |title=認識臺灣最豔麗原生淡水魚-高體鰟鮍 |date=2012-12-15 |accessdate=2013-09-09 |language=zh-hant }}</ref>。分布于[[朝鲜|朝鲜]]以及[[南盘江|南盘江]]、[[长江|长江]]、[[香港|香港]]、[[台灣|台灣]]、[[福建|福建]]和[[海南|海南]]等。该物种的模式产地在[[上海|上海]]。<ref name='id5905'/>雌性的體長約4到5公分，雄性的體長約5到8公分。體扁，身體銀灰色，但到了發情季節時，雄性的身體會轉成紫紅色到玫瑰紅色，以吸引異性<ref>{{cite journal |author=[[加納義彥|加納義彥]] |title=ニッポンバラタナゴの保護と環境保全 |journal=第５回日本水大賞受賞活動集 |publisher=日本水大賞顕彰制度委員会 |pages=42－45 |year=2003 |language=ja }}</ref>。

高體鰟鮍依賴水中的巨型淡水河蚌及河蜆作為寄主，在貝內的鰓瓣上產卵<ref name=pchome>{{cite web |url=http://mypaper.pchome.com.tw/068060/post/1323600830 |title=認識臺灣最豔麗原生淡水魚-高體鰟鮍 |date=2012-12-15 |accessdate=2013-09-09 |language=zh-hant }}</ref><ref>{{cite news |url=http://www.singtao.com.hk/yesterday/loc/0907ao09.html |title= 新界東北居民促否決環評 |date=2013-09-07 |accessdate=2013-09-09 |language=zh-hant |publisher=[[星島日報|星島日報]] }}</ref><ref>{{cite web |url=http://www.wretch.cc/blog/freenight2/11059931 |title=高體鰟鲏(缸內繁殖成功) |publisher=[http://www.wretch.cc/blog/freenight2/ freenight2 's Blog - 無名小站] |date=}}</ref>。因此，保護高體鰟鮍的同時，保護與其共同棲息的河蚌及河蜆也很重要，例如：[[粉嶺北|粉嶺北]][[梧桐河|梧桐河]][[河曲|河曲]]的[[二枚貝|二枚貝]]<ref>{{cite web |url=http://mapopo.wordpress.com/2013/08/26/eia_fish/ |title=[東北環評] 荒謬的「高體鰟鮍」原區安置措施 !|date=2013-08-26 |accessdate=2013-09-09 |language=zh-hant |publisher=[http://mapopo.wordpress.com/ 馬寶寶社區農場]}}</ref>。
