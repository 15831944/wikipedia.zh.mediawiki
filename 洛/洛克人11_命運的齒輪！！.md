== 開發 ==
本作為洛克人系列誕生30 周年的最新作，卡普空在2017年12月的洛克人30周年紀念直播上，首次公布了遊戲資訊。此次洛克人新作，追加了全新遊戲系統等多項要素。畫面自洛克人1～6、9、10的[[八位元組|八位元]]復古風，轉為較接近「[[洛克人X系列|洛克人X系列]]」後期作品的2.5D遊戲風格：宣傳影片中，遊戲角色和物體均是在2D背景上以[[多边形造型|3D多邊形]][[渲染|渲染]]，從而呈現出看似[[三维计算机图形|3D畫面]]的2.5D效果，但同時有八位元畫面的質感<ref>{{Citation|last=CapcomChannel|title=ロックマン30周年記念&『ロックマン11』アナウンス映像|date=2017-12-04|url=https://www.youtube.com/watch?v=u885gjZciQw|accessdate=2017-12-05}}</ref>。

== 故事 ==
西元20XX年，人類與機器共存的世界裡，擁有人工智慧的機器人已經融入社會中。邪惡科學家威利博士卻在此時因年輕時的一場噩夢而想起當年遭凍結的研究，並決定以此研究作為再度襲擊和平的機器人社會的武器。

而該研究是『雙重齒輪系統』。是讓機器人的能力能夠飛躍性提升的裝置，最初是威利博士在學生時代所架構的裝置，用以達成自身研究的最終目的，但此裝備擁有過高性能並伴隨危險性，結果遭到身為同窗的萊特博士反對，此研究後來更遭大學的教授們宣告凍結。結果此事以後造成兩人立場相左，可算是威利博士如何走上罪惡的不歸路起點。

隨後，威利博士便帶著裝有此系統的飛行器，到萊特博士的研究所對其宣戰，並奪走了在研究所檢修中的八名機器人。萊特博士為了對抗此事態，而決定把過去威利製作的『雙重齒輪系統』試作機裝到洛克人身上。究竟洛克人能否阻止威利博士的野心呢？決定命運的齒輪將再度轉動。

== 系統 ==
本作不僅完整保留以射擊、跳躍為中心的動作要素以及各關卡的豐富機關等各項系列作特徵，據官方釋放圖片，本作洛克人武器轉換時，除了顏色樣貌也會改變。

=== 雙重齒輪系統 ===
“雙重齒輪系統”是全新的洛克人基本能力，有 “速度齒輪”、“威力齒輪” 兩種。遊戲中萊特博士將威力博士過去製作的試作機，經過調整裝進洛克人身體。因此，不論哪種都在遊戲開始時就能使用。但由於對機器人傷害過大，只要超過使用限制時間就會導致過熱，到計量條回復前都無法使用。

*速度齒輪
一旦發動速度齒輪後將使周遭時間流動變得緩慢，相對加快洛克人的速度。不僅可作為初學者攻略時的支援，就連高手都能發揮其威力達成各種創造性的通關手法。可說是種可以開拓遊玩層面的能力。

*威力齒輪
發動中將增加洛克砲的射擊速度，連充能射擊都會升級為雙重充能射擊。另外，發動威力齒輪時也會大幅強化特殊武器，在不同武器下能獲得的升級效果各有不同。

*雙重齒輪
只有在陷入絕境時才能發動的（血量格數處於4格或更低），是讓兩個齒輪同時發動的危險能力。同時使用能提升速度與破壞力，但與其他齒輪不同無法中途解除。發動到極限洛克人就會強制陷入過熱狀態，暫時無法使用任何齒輪，充能射擊也暫時不能用。

===難度設定===
為了初次遊玩洛克人的玩家，以各種輔助功能減少失誤次數的「NEW COMER」，到比過往作品更有挑戰性的「EXPERT」為止，為本作提供豐富的難度設定。

遊戲可以選擇4個難度級別
*NEW COMER
針對新玩家的設計，難度極低，敵人的耐用性保持低。重生點數量也增加，不只以各種輔助功能減少失誤次數，可能連落入陷阱或坑也沒有。

*ADVANCED
比NEW COMER更難的模式。對於過去有本遊戲經驗卻空白期太久的人來說，這是適合輕鬆體驗的選項。

*ORIGINAL SPEC
遊戲的原始規格，是最標準的難度。

*EXPERT
難度最高的模式，能帶給經驗豐富的老玩家們比過往作品更有挑戰性的樂趣。

===獎盃挑戰===
本作共有51個獎盃，其中34個銅獎盃，15個銀獎盃，1個金獎盃和1個白金獎盃。

:GO GO GUM與MOMOS ROBOT共同開發的衝擊測試用機器人，負責指導室內運動設施。會活用以特殊膠材製造的身體一邊到處跳躍，一邊又能伸出很長的拳頭。個性天真無邪又樂觀，因此把戰鬥視為遊戲。雖然高大，但體內安裝了加速齒輪。加速齒輪發動時，身體會變得更有彈性，能夠以一閃即過的高速在四處跳躍。

=== 其他頭目 ===
*'''黃惡魔Mk-III（イエローデビルMk-III/Yellow Devil Mk-III）'''
:齒輪城第一個頭目。改良自第一代的黃惡魔，正常情況下的攻擊方式跟第一代一樣，但因為這次安裝了加速齒輪的關係，故此可分裂成9隻小黃惡魔對洛克人進行撞擊。
*'''蒙巴恩（モンバーン/Mawverne）'''
:齒輪城第二個頭目。是威利博士在開發威利機器11號試作時的副產品，可利用自身的球狀外殼防禦一切攻擊，並在球體打開時發射多重能量彈攻擊外，亦可啟動自身所安裝的動力齒輪進行強力的鐳射攻擊。
*'''威利機器11號(第一型態)（ワイリーマシン11號(第一形態)/Wily Machine 11(V1)）'''
:齒輪城最終頭目。屬飛行型威利機器之一，外觀設計有如裝上骷髏頭的[[四軸飛行器|四軸飛行器]]。可發出反重力能源彈及長型導彈作為攻擊手段（取自威利機器2號及10號），底部的齒輪更能推動洛克人所站著的履帶式地板將洛克人拉近自己並進行攻擊。
*'''威利機器11號(第二型態)（ワイリーマシン11號(第二形態)/Wily Machine 11(V2)）'''
:實為前代各威利機器所設備的威利膠囊逃生裝置，但這次仍以威利機器的名義命名。外層滿布運行中的齒輪並可發出4道大型齒輪飛彈對洛克人作出攻擊，甚至在危急關頭時發動雙重齒輪能力，在這狀態下不但能隨意傳送到另一位置外，更在傳送完成的利用大型齒輪所形成的錘子高速撞向洛克人，而且這能力並不如洛克人般會在一段時間內解除（即在被打敗前此能力會一直在啟動狀態）。
