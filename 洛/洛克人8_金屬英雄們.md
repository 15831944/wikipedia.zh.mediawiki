《'''洛克人8 金屬英雄們'''》（{{lang-ja|ロックマン8 メタルヒーローズ}}，{{lang-en|''Mega Man 8''}}）是遊戲開發商[[卡普空|卡普空]]於1996年12月17日在日本發售於[[PlayStation|PlayStation]]以及1997年1月17日在日本發售於[[世嘉土星|世嘉土星]]的[[動作遊戲|動作遊戲]]，[[洛克人元祖系列|洛克人元祖系列]]的第八作。

== 概要 ==
*平台從[[超級任天堂|超級任天堂]]轉變為[[PlayStation|PlayStation]]以及[[SEGA_SATURN|SEGA SATURN]]，畫面與從[[16位元|16位元]]來到[[32位元|32位元]]，變得更加鮮明，亦首次加入動畫劇情。關卡由開頭、前四大頭目關卡、中盤關卡、後四大關卡和威利城四關卡共14關組成。
*首次使用聲優配音，並加入片頭曲及片尾曲。
*本作是在[[日本|日本]]境內唯一沒有發行在[[任天堂|任天堂]]平台的一代。而在歐美發行的『洛克人週年系列』中則發行在[[Game_Cube|Game Cube]]上。

== 系統變更 ==
*特殊武器的剋制分為前段與後段的兩個循環。
*從[[洛克人2：威利博士之謎|2代]]時開始採用的[[密碼|密碼]]存檔和遊戲結束時必須重頭開始的模式廢止，取而代之的是本代的[[存檔|存檔]]功能。
*本作廢止如E罐等可任意回復全部血量的道具，武器介面也全部更新，中繼存檔點通過後便會回復血量及所有武器能量，在頭目房間房口也會有回復武器能量的道具可以使用。
*保留前作的螺絲收集系統，收集到一定數量後便可以購買加強能力的道具。

== 劇情 ==
洛克人接到來萊特博士的消息，前來調查一顆來自[[宇宙|宇宙]]的神祕[[隕石|隕石]]。但是威利已經搶先一步拿走了隕石，隕石中有著強大的邪惡能量，為了調查宇宙之謎和阻止威利的野心，洛克人又再次出動。

== 登場頭目 ==
本作的頭目挑戰與前代相同，採前半部分(天狗人、小丑人、手榴彈人、冰霜人)及後半部分(星象人、水人、巨劍人、搜索人)兩大分類，打完前半和中間關卡才能挑戰後半關卡，在武器相剋中也是採前半和後半的獨立迴圈。
本代除天狗人和星象人是由製作團隊親自設計外，其餘六個頭目都是透過向玩家公開招募來改良。
;DWN.057 天狗人(テングマン)(Tenguman)（声優：[[長嶝高士|長嶝高士]]）
:原為用來開發人造[[颱風|颱風]]的實驗用機器人，被威利盜走後改造成戰鬥用機器人。就如其名，不管是外表還是個性都像個孤傲的天狗，對自己的長鼻子感到驕傲。特殊武器為往地面發射葉片，藉由旋轉來產生[[龍捲風|龍捲風]]的「龍捲風球」，藉由高速移動也能產生強烈的風暴。而雖然是氣象實驗用機器人，但非常怕冷。
:目前住在[[日本|日本]]，並佔領空中基地做為自己的據點，PS版與SS版使用的BGM不一樣。
;DWN.058 星象人(アストロマン)(Astroman)（声優：[[二又一成|二又一成]]）
:原為在[[天文台|天文館]]使用[[異次元|異次元]][[星象儀|星象儀]]來教育兒童的機器人，被威利盜走後改造成戰鬥用機器人。守護威利塔屏障的四部機器人之一。
:使用[[失重|無重力]]裝置飄浮在空中。個性膽小害羞，所以常常將自己隱藏起來，很怕被追蹤，也是[[躲貓貓|躲貓貓]]的世界冠軍。特殊武器為從太空中吸引大量的隕石，並粉碎自己週邊敵人的「隕石雨」，但其武器的空隙非常的大。
:個性單純善良，有著會像流星許願的一面。所在據點位於[[英國|英國]]。
;DWN.059 巨劍人(ソードマン)(Swordman)（声優：[[高木渉|高木渉]]）
:威力利用從[[博物館|博物館]]偷來古代大劍加上身體後製作成的機器人。守護威利塔屏障的四部機器人之一。
:有著相當高明的劍術，而因為身上的劍過重導致無法平衡，所以威利在其上半身安裝了反重力裝置來保持平衡。特殊武器為將劍纏繞火焰後將敵人一刀兩斷的「火焰劍」。得意技術為藉由上半身快速旋轉來斬擊敵人的「火焰切割」。
:喜歡堂堂正正的決勝負，據點在[[中亞|中亞]]。 
:最初招募時的名稱為「遠古人」(エインシェントマン)，使用武器為「守護之劍」(ガーディアンブレード)。
:投稿者為秋田縣的松尾佳悟。
;DWN.060 小丑人(クラウンマン)(Clownman)（声優：[[坂本千夏|坂本千夏]]）
:原為遊樂園表演的機器人，被威利改造成戰鬥用機器人，有著[[小丑|小丑]]的造型，身體很小且輕盈。擁有相當長度的手臂，但遇到強風會打結。因為能輕鬆做到[[空中飛人|空中飛人]]的動作所以感到驕傲，利用長手臂在空中迴轉來愚弄對手。特殊武器為用手臂抓住敵人後放出電流攻擊的「霹靂爪」，以及將身體包成帶電的球狀並來回彈跳的得意技「雷電嘉年華」。
:性格惡劣且瞧不起洛克人，認為洛克人又笨又弱。目前以[[澳洲|澳洲]]的遊樂園作為自己的據點。
:名稱和長手臂在當初招募時就已出現，差別是最初有著類似死神的面具。
:投稿者為岐阜縣的阪井裕茂。
;DWN.061 搜-{索}-人(サーチマン)(Searchman)（声優：[[長嶝高士|長嶝高士]]）
:原為安全監控的機器人，被威利改造成戰鬥用機器人。守護威利塔屏障的四部機器人之一。
:為了加強監視，所以威利自作聰明的給予加裝了兩個頭腦，但兩個頭有點類似上級與下屬的關係且關係極差，導致變成互相在監視對方。
:特殊武器為右手迫擊砲所發射出來能自動狙擊敵人的「追蹤飛彈」。
:由於在[[非洲|非洲]]的雨林內作為自己的據點，故此能夠用樹葉堆隱藏自己，並製作出替身以分散洛克人的注意力並偷偷的作出偷襲。
:最初招募時的名稱為「巴爾幹人」(バルカンマン)。
:投稿者為島根縣的田裕太。
;DWN.062 冰霜人(フロストマン)(Frostman)（声優：[[高木涉|高木涉]]）
:由於製造小丑人後有剩餘的零件而製成的戰鬥用機器人，為歷代八大頭目中體型最大的。
:因為思考回路不完全所以腦袋不太靈光，夢想為吃掉用洛克人製成的刨冰。
:特殊武器為「冰凍波」（實際上為洛克人5當中的頭目波浪人的特殊武器「水柱波浪」的冰凍版）。
:目前以[[阿拉斯加|阿拉斯加]]作為自己的據點。
:最初招募時的名稱為「葉堤人」(イエティマン)(葉堤(yeti)為聖母峰的傳說生物)，特殊武器為「暴雪護盾」(ブリザードシールド)，但除了體積比較巨大的設定外，其他設定皆沒有被採用。
:投稿者為島根縣的野村明史。
:在[[洛克人10：來自宇宙的威脅！！|洛克人10：來自宇宙的威脅！！]]中，作為8代頭目的代表，以可以重現過去機器人頭目招式的「武器存檔機」的造型代表再次登場。
;DWN.063 手榴彈人(グレネードマン)(Gernademan)（声優：[[二又一成|二又一成]]）
:以手榴彈為設計原型的戰鬥用機器人。內設的強化程式能暫時提升手榴彈人的戰鬥力。
:有著喜歡炸毀物品的癖好，因為被威利派去管理炸彈工廠而非常高興。
:特殊武器為爆炸時會有閃光的「閃光炸彈」，卻也很享受被自己的炸彈反彈後波及的快感。
:目前以[[南美洲|南美洲]]的炸彈工廠作為自己的據點。
:最初招募時的名稱為「引爆人」(エクスプロードマン)，使用武器為「爆發炸彈」(エクスプロードボム)。
:投稿者為神奈川縣的小牧健一郎。
;DWN.064 水人(アクアマン)(Aquaman)（声優：[[坂本千夏|坂本千夏]]）
:威利製造的戰鬥用機器人，似乎跟小丑人的設計有出入。守護威利塔屏障的四部機器人之一。
:個性低級，喜歡開下流的玩笑，身體為一個大水缸，可用手上的水炮抽水進入身體以便使用特殊武器「水氣球」。
:目前以[[馬達加斯加|馬達加斯加]]的海底實驗室作為自己的據點。
:最初招募時的名稱為「生化人」(バイオマン)。特殊武器為能融化金屬，並入侵機器人身體使其故障的「生化溶液」(バイオリキッド)，原本設定為能將對手機器人溶解，身體極為精密且智慧極高的機器人，與後來定案的水人相去甚遠。
:投稿者為福島縣的栗成盛登。

=== 其他 ===
;寄居蟹(ヤドカルゴ)(Yadocargo)
:開頭關卡的頭目。威利用於秘密基地警戒的戰鬥用機器，但由於島上一直處於沒有任何戰鬥的和平狀態，所以殼上長了鳥巢和[[青苔|青苔]]。有著類似鉗子的戰鬥武器，殼下為最大的弱點。
;阿提提米諾(アテテミー)(Atete Mino)
:威利關卡的第一頭目，守衛著基地入口的[[蓑衣蟲|蓑衣蟲]]型機器人。有著喜歡惡作劇使人嚇一跳的個性。戰鬥時會吊在空中向人投擲炸彈，或是丟出小阿提提米諾來攻擊對手。
;鰤王(ブリキング)(Bliking)
:威利關卡的第二頭目，威利為了打倒洛克人而拿出全部技術力設計的[[戰鬥機|戰鬥機]]型機器。因為是緊急出擊的緣故所以來不及[[塗装|塗装]]。戰鬥時會使用頭上的炸彈、翅膀的雷射和導彈來攻擊。
;綠惡魔(グリーンデビル)(Green Devil)
:威利關卡的第三頭目，黃惡魔的改良型。身體是由厚度極高的[[果凍|果凍]]狀的物質構成，並由眼珠造型的本體控制。除了使用基本的分裂攻擊外，也會分裂後用類似波浪的方式攻擊對手。
;威利機器8號(ワイリーマシーン8号)(Wily Machine 8)
:威利關卡的第四頭目。對付洛克人用的[[飛行船|飛行船]]造型機，搭載著強力的惡能源，總使主砲已被迪歐破壞，但仍擁有著不可小看的戰鬥力
:本體被破壞後，會出現最終頭目"威利偉大膠囊"。
;威利偉大膠囊(ワイリーカプセルグレート)(WILY GREAT CAPSULE)
:威利機器八號內建的逃脫裝置，在本體被破壞後出現，藉由由裝填的惡能量可進行多樣的攻擊及瞬間移動。
