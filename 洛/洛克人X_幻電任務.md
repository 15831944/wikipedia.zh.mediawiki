{{Vgname|洛克人X 幻電任務|ja=ロックマンX サイバーミッション，Rockman X Cyber Mission|en=Mega Man Xtreme}}是[[洛克人X系列|洛克人X系列]]的外傳故事，2000年10月20日發表於[[Game_Boy_Color|Game Boy Color]]上的專用[[動作遊戲|動作遊戲]]。

== 概要 ==
[[洛克人X系列|洛克人X系列]]發行在的[[Game_Boy_Color|Game Boy Color]]第一部外傳故事。本作採用了與[[洛克人世界|洛克人世界]]相同的模式，將《[[洛克人X_(遊戲)|洛克人X]]》與《[[洛克人X2|洛克人X2]]》混合移植後加入新角色，並將模式分成一般模式、困難模式與完全模式，誕生出了全新的故事，本作也是[[卡普空|卡普空]]發行的作品中最後一部能使用於舊型[[Game_Boy|Game Boy]]的遊戲。

== 故事 ==
時間點發生在『洛克人X2』的事件之後，某日非正規獵人基地中的'''馬薩電腦'''突然受到網路攻擊，各項網路數據被竄改導致了混亂，為此艾克斯與非正規獵人中的天才機器少年'''米迪 '''合作，潛入馬薩電腦中調查此事，而在馬薩電腦的資料中必須與過去打倒的敵人對戰。

但調查後卻發現此事件的犯人竟是米迪的哥哥-已經非正規化的機器人'''提克諾'''，而幕後的指使人又是西格瑪，艾克斯又將出動阻止他的野心。

== 主要角色 ==
=== 非正規獵人 ===
;[[艾克斯_(洛克人)|艾克斯]]（{{lang|ja|エックス}}，{{lang|en|X}}）
:本作的主角之一，擔任非正規品獵人第十七部隊隊長。本自為了調查馬薩電腦被入侵的事件，潛入了馬薩電腦進行調查。
;[[傑洛|傑洛]]（{{lang|ja|ゼロ}}，{{lang|en|Zero}}）
:本作的主角之一，有著特A級獵人的實力，本作不能直接遊玩，而是做為救援角色。本作為X3的時間軸之前，並未開始接手第0特殊部隊的隊長職務，也未說明在這之前的所屬職位。
; 米迪（{{lang|ja|ミディ}}，{{lang|en|Midi}}）
: 凱因博士的助手，有如天才般的少年思考型機器人，與提克諾的電子頭腦為相同[[CPU|CPU]]的雙胞胎弟弟，負責協助艾克斯與傑洛在電子空間的調查，因為與提克諾為生命共同體的緣故，在提克諾消逝之後，也隨之跟著死去。
; 凱因博士（{{lang|ja|ケイン博士}}，{{lang|en|Dr.Cain}}）
: 22世紀機器人工學的權威，非正規獵人的創立者，也是當前最了解艾克斯和傑洛的人，本次和米迪同為艾克斯和傑洛的協助者。

=== 闇影獵人 ===
過去曾經為了名聲而大膽挑戰西克瑪的二人組，但卻被西格瑪壓倒性的實力打敗，之後便對西克瑪完全服從，成為直屬的祕密部下，兩者的風格都是中世紀風，續作[[洛克人X2_Soul_Eraser|洛克人X2 Soul Eraser]]的貝爾卡娜和卡雷斯也是相同的風格。
; 斬因（{{lang|ja|ザイン}}，{{lang|en|Zain}}）
: 神秘戰鬥思考型機器人集團'''闇影獵人'''的其中一人，聽從西格瑪的指示進入電子空間擊倒艾克斯。身上覆蓋[[中世紀|中世紀]]裝甲及巨劍作為主力的戰鬥型機器人，致力於強化自己身上的戰力，是個沉默寡言的強者。
; 基梅爾（{{lang|ja|ギーメル}}，{{lang|en|Geemel}}）
: '''闇影獵人'''的其中一人，斬因的同夥，與沉默寡言的斬因相反，自身聰明、狡猾與邪惡，有著自己獨有的笑聲。武器為大型[[手裡劍|手裡劍]]和滑翔用的[[風箏|風箏]]、是如同[[忍者|忍者]]般的角色。

=== 其他角色與頭目 ===
; 提克諾（{{lang|ja|テクノ}}，{{lang|en|Techno}}）
: 米迪的雙胞胎哥哥，被洗腦後非正規化，聽從西格瑪的命令協助闇影戰警入侵非正規獵人基地的馬薩電腦，本作不需與其戰鬥，與米迪為一個實驗室中誕生的產物，其目的是要讓兩枚電腦晶片在組合時能發揮出加倍的威力，但最後實驗失敗，只讓兩兄弟可以個別運作，但兩枚晶片之間的連結沒有並除去，讓他們變成了生命共同體。
; 霸法({{lang|ja|ヴァヴァ}}，{{lang|en|VAVA}}）
: 艾克斯過去的同事，為艾克斯印象中最深的反派，本作做為過去的資料再次登場。
; 薩加斯({{lang|ja|サーゲス}}，{{lang|en|Sagesse}}）
: 過去修羅戰警的其中一人，與霸法同樣作為過去的資料登場。
; 西格瑪({{lang|ja|シグマ}}，{{lang|en|Sigma}}）
: 艾克斯的上司，前第17精銳部隊的隊長，本次事件的幕後主使。
