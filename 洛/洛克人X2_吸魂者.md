{{Vgname|洛克人X2 吸魂者|ja=ロックマンX2 ソウルイレイザー，Rockman X2 Soul Eraser|en=Mega Man Xtreme 2}}是[[洛克人X系列|洛克人X系列]]的第二部外傳故事，2001年7月19日發表於[[Game_Boy_Color|Game Boy Color]]上的專用[[動作遊戲|動作遊戲]]。

== 概要 ==
前作《[[洛克人X_Cyber_Mission|洛克人X Cyber Mission]]》的續集，同為[[Game_Boy_Color|Game Boy Color]]專用的遊戲，將前作系統與《[[洛克人X2|洛克人X2]]》的系統延伸，再加入《[[洛克人X1|洛克人X1]]》與《[[洛克人X3|洛克人X3]]》的要素與頭目組合後所製作的移植型外傳作品。

本作修正了前作的缺失，引進了一些新的系統，如系列作首創的即時換人模式、改進版的中途儲存裝置、零件系統等，使遊玩性大增。

== 故事 ==
時間點發生於[[洛克人X3|洛克人X3]]的事件結束之後<!-- 來源請求? -->，某日發生了思考型機器人們的程式突然消失的大規模事件，這些思考型機器人彷彿就像是失了魂一樣，如同廢鐵般動彈不得，尤其是在位發生在南洋之島「拉格斯島」的情況最為嚴重，艾克斯、傑洛與在非正規戰警總部中擔任實習生的愛麗絲共同展開調查，但卻發現過去打倒的敵人相繼復活的現象。

此時，此事件的主謀'''貝爾卡娜'''對艾克斯與傑洛下了挑戰書，兩人便接受了挑戰，來執行奪回思考型機器人們靈魂的任務。

== 主要角色 ==
;[[艾克斯_(洛克人)|艾克斯]]（{{lang|ja|エックス}}，{{lang|en|X}}）
:本作的主角之一，擔任非正規品獵人第十七部隊隊長。

;[[傑洛|傑洛]]（{{lang|ja|ゼロ}}，{{lang|en|Zero}}）
:本作的主角之一，擔任非正規品獵人第零特殊部隊的隊長。

;愛麗絲（{{lang|ja|アイリス}}，{{lang|en|Iris}}）
:卡尼爾的妹妹，《[[洛克人X4|洛克人X4]]》的女主角。
:本作為仲裁部隊派來非正規戰警總部中研習的實習生，因剛好碰到這次的事件，所以這次臨時擔任兩人的領航員，由故事劇情的進行可得知喜歡傑洛。
:由對話得知與艾克斯為舊識。
; 貝爾卡娜 （{{lang|ja|ベルカナ}}，{{lang|en|Berkana}}）
: 本作事件的主謀，有著[[魔女|魔女]]的外型，原先是拉格斯島負責思考型機器人開發的女性研究員機器人，是製造時間、過去的經歷等，一切都成謎的神祕女性角色，也是系列作中少見的女性頭目之一。
:名稱由來為[[盧恩字母|盧恩字母]]中的[[樺木|樺木]]。<!-- 來源請求 -->
; 卡雷斯（{{lang|ja|ガレス}}，{{lang|en|Gareth}}）
: 貝爾卡娜製作，[[騎士|騎士]]造型的思考型機器人。是她其信賴的左右手，擁有強力的攻擊與優秀防禦力，有著一把號稱能貫穿一切的長槍作為主武力。
;西格瑪（{{lang|ja|シグマ}}，{{lang|en|Sigma}} ）
: 本代經由貝爾卡娜之手，利用其DNA和大量靈魂合體後復活。第二形態的「'''(西格瑪魔獸)シグマビースト'''」是與[[獅子|獅子]]型的巨大機器合體後的型態。

== 主要頭目 ==
; 「深海的武裝將軍」魚雷烏賊（{{lang|ja|ランチャー・オクトパル}}(Launcher Octopuld)╱ {{lang|en|LAUNCH OCTOPUS}} ）
: 原第六艦隊隊員的[[章魚|章魚]]思考型機器人，登場於《X1》的頭目。
;「灼熱的油箱」石油豪豬（{{lang|ja|バーニン・ナウマンダー}}(Burnin' Noumander)╱ {{lang|en|FLAME MAMMOTH}} ）
: [[猛瑪象|猛瑪象]]外型的思考型機器人。前第四陸上部隊隊長，登場於《X1》的頭目。
;「綠林之小惡魔」長鏈藤蔓（{{lang|ja|ワイヤー・ヘチマール}}(Wire Hetimarl)╱ {{lang|en|Wire Sponge}} ）
: 西格瑪所製作，[[絲瓜|藤蔓]]形思考型機械人，登場於《X2》的頭目。
;「砂漠之奔跑家」音速鴕鳥（{{lang|ja|ソニック・オストリーグ}}(Sonic Ostreague)╱ {{lang|en|Sonic Ostrich}} ）
: [[鴕鳥|鴕鳥]]外型的思考型機器人。原為非正規機械人獵人第七隊（空艦部隊）的特A級隊員，登場於《X2》的頭目。
;「影之飛忍」 大黃蜂(エクスプローズ・ホーネック (Explose Horneck) /Blast Hornet)
: [[黃蜂|黃蜂]]造型的思考型機器人，原為第0特殊部隊的副隊長，登場於《X3》的頭目。
;「援助發電站」 高壓電鯰(エレキテル・ナマズロス (Electel Namazuros) / Volt Catfish)
: 用於發電的[[鯰魚|鯰魚]]思考型機器人，登場於《X3》的頭目。
; 「閃耀的守護神」 閃光猛虎(シャイニング・タイガード (Shining Tigerd) / Neon Tiger)
: [[虎|虎]]型的思考型機器人，過去為負責逮捕盜獵者以保護瀕臨絕種動物的森林巡察員，登場於《X3》的頭目。
;「地底的野蠻人」電鑽犀牛(スクリュー・マサイダー (Screw Masaider) / Tunnel Rhino)
: [[犀牛|犀牛]]造型的思考型機器人，前能量水晶礦的礦工，登場於《X3》的頭目。
