'''洛塔尔·马特乌斯'''（{{lang|de|'''Lothar Matthäus'''}}，{{bd|1961年|3月21日|catIdx=Matthäus, Lothar}}）出生於[[巴伐利亞|巴伐利亞]]的[[埃爾朗根|埃爾朗根]]，前德國足球運動員，曾在1990年以隊長身份帶領[[德國國家足球隊|德國國家隊]]贏得[[1990年世界盃|世界盃]]，被評選為當年度世界杯银球，[[歐洲足球先生|歐洲足球先生]]和[[德國足球先生|德國足球先生]]。一年後，他被評選為首位[[世界足球先生|世界足球先生]]。世界足坛史上最佳德国巨星之一。2004年，他位列球王[[比利|比利]]評選的[[FIFA_100|125位最出色的在世球星（FIFA 100）中。]]

== 球會生涯 ==

馬圖斯的球會生涯是從位於埃爾朗根市郊的小城[[黑措根奧拉赫|黑措根奧拉赫]]足球會開始的，1979-84年轉入[[慕遜加柏|慕遜加柏]]足球會。他職業生涯的最高峰是在1984-88年、和1992-2002年這段時間他轉投的規模更大的[[拜仁慕尼黑|拜仁慕尼黑]]，以及1988-92年間效力的意大利的[[國際米蘭|國際米蘭]]足球會。

馬圖斯於1988年轉投意甲球會國際米蘭，於1989年贏得[[意甲|意甲]]冠軍。在其球員生涯早期，馬圖斯主要司職[[中場|中場]]，在30歲之後，他轉型為一名[[自由人|自由人]]。1992年他重返拜仁慕尼黑，贏得了幾屆[[德甲|德甲]]聯賽冠軍。雖然此時他已年逾三十，仍是球會主力，差不多奪得球員生涯中所有重要獎項，唯一的遺憾是失落[[歐洲聯賽冠軍盃|歐洲聯賽冠軍盃]]，1999年他為拜仁打入決賽，但尚差兩分鐘到手的獎盃最終被[[英超|英超]]球隊[[曼聯|曼聯]]奪得。

2000年馬圖斯轉投[[美國職業足球大聯盟|美國職業足球大聯盟]]的[[紐約紅牛|紐約紅牛]]隊（2000年），但不足一年卻宣佈退役。2004-05球季，馬圖斯曾短暫參與了[[萊比錫火車頭足球俱樂部|萊比錫火車頭]]足球會的城市半決賽比賽。

== 國家隊生涯 ==
馬圖斯在國家隊成名很早，早於1980年已隨隊出賽，贏得了[[1980年歐洲足球錦標賽|1980年歐洲杯]]。其後[[1984年歐洲足球錦標賽|84年]]、[[1988年歐洲足球錦標賽|88年歐洲杯]]也有份出賽。同時他一共參加過1982-98年之間的五屆[[世界盃足球賽|世界盃]]，至今保持着參加世界盃決賽週出場最多的紀錄（25場）。其中[[1990年世界盃|1990年世界盃]]他更贏得了冠軍。

馬圖斯雖然腳法稱不上華麗，但控球能力強勁，不作無謂的花巧表現，簡單實在有效率，助攻助守意識強，活動範圍大，完全是一個高機動性的全能中場示範，而且鬥志拚勁強橫更兼有遠射本領，簡單實在中見功力。刻苦敬业，大器晚成，1987年舒马赫被开除才被委任德国队队长，年紀稍大後（30歲後），馬圖斯轉踢自由人位置，雖然體力下降，但球場上的閱歷仍是一樣強。

[[2000年歐洲國家盃|2000年歐洲國家盃]]馬圖斯以39歲高齡出賽，到他退役為止，一共為德國國家隊出賽150場，攻入23個進球。

== 执教生涯 ==
馬圖斯首支執教的球隊是奧地利[[維也納迅速|維也納迅速]]（SK Rapid Wien），成績平平。2002年12月中途接手[[游擊隊足球俱樂部|柏迪遜]]（FK Partizan），成功帶領球隊晉身歐聯，在第三圈外圍賽淘汰[[英超|英超]]的[[紐卡素|紐卡素]]，分組賽被編入強隊林立的一組，對手包括[[皇家馬德里|皇家馬德里]]、[[馬賽足球俱樂部|馬賽]]及最終冠軍[[波圖|波圖]]，飲恨出局。2003年12月馬圖斯突然宣佈辭職，翌日簽約出任[[匈牙利國家足球隊|匈牙利國家隊]]的主教練，爭取[[2006年世界盃足球賽|2006年世界盃]]決賽周的出線權。匈牙利被編於有六隊的歐洲第八組外圍賽，10戰4勝2和4負，只能失望地排於[[克羅地亞國家足球隊|克羅地亞]]、[[瑞典國家足球隊|瑞典]]及[[保加利亞國家足球隊|保加利亞]]之後的第四位，比賽後馬圖斯辭職，但雙方仍維持友好關係。

2006年1月11日，馬圖斯與巴西的[[帕拉尼恩斯體育會|巴拉那競技]]（Atlético Paranaense）簽約一年，但只任教了7場比賽（5勝2和），於3月告訴球隊急需返回歐洲處理私人事務，但一去不返，兩周後更傳真其辭職通知書。

2006年5月，馬圖斯接受奧地利[[薩爾斯堡紅牛|薩爾斯堡紅牛]]（Red Bull Salzburg）的邀請，出任其2006-07球季主教練。一個賽季之後馬圖斯收到了炒他魷魚的通知，理由是因為和當時的總教練[[查帕東尼|查帕東尼]]發生爭執而被解職。

2008年4月，馬圖斯被以色列球會[[内坦亚马加比足球俱乐部|尼坦耶馬卡比]]（Maccabi Netanya）任命為2008-09球季主教練。時隔一年，球會高層宣布由於財政方面的原因，球季結束後不再與馬圖斯續約。尼坦耶馬卡比最終聯賽排名第四。

[[File:Lothar_Matthäus_in_frendly_mach_Bulgaria-Serbia.JPG|thumb]]的一場友誼賽中。]]
2010年9月21日，繼2004年執教匈牙利之後，馬圖斯再一次拿起了國家隊的教鞭。這一次他選擇的是[[保加利亞國家足球隊|保加利亞國家隊]]。但是由於預選賽戰績不佳，在數字上，保加利亞幾可肯定已無緣出線明夏舉行的歐錦賽決賽周。2011年9月19日，保加利亞足總宣布解除與馬圖斯的合同，馬圖斯在任僅僅一年。

== 個人生活 ==
目前為止，馬圖斯有3個孩子，其中兩個女兒是與他的第一任前妻Sylvia（婚姻維持期：1981-92年）所生，另外的一個兒子是與第二任前妻Lolita Morena（婚姻維持期：1994-99年）所生。他與第三任前妻Serbin Marijana Kostić在2007年11月末離婚。之後不久，他與年僅26歲的烏克蘭人Kristina Liliana Tchoudinova於2009年1月1日在[[拉斯韋加斯|拉斯韋加斯]]結婚，這是他的第四任妻子。
