《'''洛克人與佛魯迪'''》（{{lang-ja|ロックマン&フォルテ}}，{{lang-en|''Mega Man & Bass''}}）是日本遊戲公司[[卡普空|卡普空]]於1998年4月24日於[[超級任天堂|超級任天堂]]、2002年8月10日於[[Game_Boy_Advance|Game Boy Advance]]平台上發售的[[動作遊戲|動作遊戲]]，為[[洛克人元祖系列|洛克人元祖系列]]的外傳作品之一。

== 概要 ==
洛克人與佛魯迪繼在[[PlayStation_(遊戲機)|PlayStation]]及[[世嘉土星|世嘉土星]]平台發售的《[[洛克人8|洛克人8]]》後，又重回與《[[洛克人7|洛克人7]]》相同的超級任天堂平台。據製作人稻船敬二表示，此舉是讓當時無法負擔次世代主機價格的入門玩家也能遊玩新一代的洛克人。因此遊戲與洛克人8十分接近，擁有相同的[[2D|2D]][[精灵图|精靈圖]]及動畫風格，甚至包含了兩個來自洛克人8的頭目天狗人和星象人。

== 特色 ==
採用雙主角制，在遊戲的最初可以選擇使用洛克人或者佛魯迪。除了能力以外，雙方的劇情也會有部份的差異。關卡選擇的方式打破了過去的模式，採用樹狀圖的構成模式，分為前半與後半部份，玩家需要打倒一個前半的頭目和中間關卡才能開啟部份的後半關卡。遊戲中追加了特殊的道具CD，分散在各個關卡中，共有100枚，在取得之後能夠開啟相對應的角色圖鑑資料。

== 故事 ==
跟洛克人的戰鬥失敗而失蹤的威利博士，為了實現自己的野心而建造新的研究基地，卻被自稱國王（King）的機器人佔領。他搶走威利製造的基地並從博物館開始偷機器人的數據，編成了自己的軍團。企圖征服世界，而威利博士狼狽地逃出求助於萊特博士，於是洛克人和佛魯迪暫時停戰，合作調查事件真相。

== 登場人物 ==
; DRN.001 洛克人(ロックマン)(Rockman)
: 原本是家事用的機器人，因為看到威利博士作亂，所以要求萊特博士將他改造成戰鬥機器人。
; DRN.002 蘿露(ロール)(Roll)
: 當洛克人成為戰鬥機器人之後，萊特博士用之替補打掃的工作，如同洛克的妹妹，總是為前線的洛克感到擔心。本作的造型與服裝更新。
; DRN.000 布魯斯(ブルース)(Blues)
: 戴著墨鏡、持有盾牌的神秘紅色機器人。過去曾數次幫助洛克人，實際上是萊特博士的作品之一。
; 萊特博士(Dr.ライト)(Dr. Right)
: 被世人稱為「機器人工學之父」，他和藹可親、厭惡戰爭，希望人類能與機器人和平相處。
; 萊德多(ライトット)(Rightot)
: 自稱萊特博士首席弟子的機器人。
; 威利博士(Dr.ワイリー)(Dr. Wily)
: 自稱「邪惡天才科學家」，優秀的頭腦總不用在正途上。本作幕後主謀。
; SWN.001 佛魯迪(フォルテ)(Forte)
: 威利參考洛克人所製作的戰鬥用機器人，洛克人的勁敵。本作中與洛克人合作。
; 國王('''キング''')(King)
: 自機器人博物館中竊取了機器人的藍圖，自稱要建立一個只有機器人的國度。實際上是威利博士為了要超越不聽自己指示的佛魯迪而製作的戰鬥型機器人，並賦予可以自己生產機器人的智能。國王興趣是西洋棋，但對於人類非常的憎惡且不信任。當國王完成後，威利便想到利用他來對付洛克人，並用來考驗佛魯迪能力。

== 登場頭目 ==
本作的頭目挑戰與前代相同，採前半部分及後半部分兩大分類，打完前半和中間關卡才能挑戰後半關卡，在武器相剋中也是採前半和後半的獨立迴圈。 本代除天狗人和星象人是由製作團隊親自設計外，其餘六個頭目都是透過向玩家公開招募來改良。
