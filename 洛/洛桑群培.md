'''洛桑群培'''（{{Bo|z=བློ་བཟང་ཆོས་འཕེལ་|w=blo bzang choe 'phel}}；{{bd|1957年||||catIdx=L}}），[[中國|中國]]男演員，出生于[[西藏|西藏]][[桑日县|桑日县]]，藏族，毕业于[[上海戏剧学院|上海戏剧学院]]。<ref name=" tiger" >{{cite web |url=http://www.tibet.cn/rwxw/zyth/201206/t20120618_1753528.htm |title=洛桑群培：西藏走出来的金马影帝  |accessdate= 2012|publisher=中国西藏网 |work= |language=zh-cn}}</ref>

== 生平 ==
1957年，洛桑群培出生在[[西藏|西藏]][[桑日县|桑日县]]一个农民家庭，父母在解放前从是[[农奴|农奴]]，因为家境贫寒，读小学时曾辍学。<ref name=" tiger" />14岁时，他在中学里学习[[会计|会计]]，后来考上中央民族学院（今[[中央民族大学|中央民族大学]]），但是被人走后门顶包抢了名额，他被分配到西藏自治区话剧团学习表演，1977年，他被学校送入[[上海戏剧学院|上海戏剧学院]]学习。<ref name=" tiger" />1998年，以電影《[[天浴|天浴]]》獲[[第35屆金馬獎|第35屆金馬獎]]最佳男主角獎。
