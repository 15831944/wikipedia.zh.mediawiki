《'''洛克人5：布魯斯的陷阱！？'''》（{{lang-ja|ロックマン5：ブルースの罠！？}}，{{lang-en|''Mega Man 5''}}）是遊戲開發商[[CAPCOM|CAPCOM]]於1992年12月4日發售的[[FC遊戲機|FC遊戲機]]專用[[動作遊戲|動作遊戲]]。

== 概要 ==
[[洛克人元祖系列|洛克人元祖系列]]的第五作。，由八大頭目關卡，加上布魯斯城四關卡和威利城四關卡共16關卡。
類似宇宙及反重力的上下顛倒關卡等特殊關卡，皆在本作登場。
== 劇情 ==

20XX年，經過2個月後，「布魯斯到底是什麼人呢？」洛克人對這種不知是敵還是友的謎之機器人的事非常在意。突然布魯斯帶領神秘的機器人軍團在世界各地叛亂中，這時看到布魯斯把萊特博士拐走。然後洛克人看到布魯斯的黃色領巾掉落在他的手上，決心投入戰場尋找真相。

== 系統變更 ==
*《[[洛克人4|洛克人4]]》中洛克炮聚氣威力的覆蓋範圍增加，武裝也從'''新洛克砲'''改名'''超級洛克砲'''，聚氣時間也比前一代更短，但缺點是若聚氣時受到傷害則必須重新聚氣<ref name="r20"/><ref name="steps">{{cite web | author=Thomas, Lucas M. | date=February 16, 2010 | title=The 10 Steps to ''Mega Man 10'' | url=http://wii.ign.com/articles/106/1069184p3.html | publisher=[[IGN|IGN]] | accessdate=April 11, 2010}}</ref>。
*特殊武器除了對特定頭目有極大傷害外，對非弱點頭目也會有基本的傷害。
*彈簧拉休跳躍方式改變，跳躍的高度變得更高。
*噴射拉休可在水中使用，所以自本作起廢除潛艇拉休。
*八大頭目的關卡中各藏著「R」「O」「C」「K」「M」「A」「N」「5」等八個字母電路板，若全部收齊便可使用本代新登場的援助機械「比特」。
*可一次補滿生命及武器能量的M罐登場，但與E罐不同的是只能帶一罐，在身上擁有M罐的前提下不會在關卡中遇到第二罐。
*在武器切換介面中，特殊武器所使用的名稱也改為頭目的英文字首加上招式名稱。 
*八大再戰時離開用的傳送器取消，再次打完八大後會自動離開舞台。

== 主要角色 ==

; {{nihongo|[[洛克人|洛克人]]|ロックマン|Rockman}}:原本是家事用的機器人，因為看到威利博士作亂，所以要求萊特博士將他改造成戰鬥機器人。
; {{nihongo|萊特博士|ライト博士|Dr. Right}}:被世人稱為「機器人工學之父」，他和藹可親、厭惡戰爭，希望人類能與機器人和平相處。
; {{nihongo|拉休|ラッシュ|RUSH}}:多功能犬型支援機器，是洛克人的拍檔。新增新彈簧拉休。
; {{nihongo|艾迪|エディ|EDDIE}}:萊特博士開發出道具運輸用的輔助型機器人。在關卡特定位置裡會隨機給予洛克人各種道具的補給。
; {{nihongo|比特|ビート|BEAT}}:柯薩克博士為了感謝洛克人於[[洛克人4_新的野心！！|前作]]幫助他以及他的女兒脫離威利博士的陰謀而為他所開發的支援用機器鳥，能用身體攻擊敵人。收集洛克人金屬板R.O.C.K.M.A.N.5後出現。
; {{nihongo|布魯斯|ブルース|BLUES}}:戴著墨鏡、持有盾牌的神秘紅色機器人。過去曾數次幫助洛克人，實際上是萊特博士的作品之一。
; {{nihongo|威利博士|Dr.ワイリー|Dr. Wily}}:自稱「邪惡天才科學家」，這次威利博士仍是幕後主謀。
; {{nihongo|柯薩克博士|Dr.コサック|Dr. Cossack}}:全名「米哈伊爾‧謝爾蓋維奇‧柯薩克」﹝ミハイル・セルゲイビッチ・コサック﹞，是洛克人的協助者，於本作帶來比特作為送給洛克人的獎勵。

== 八大頭目 ==
; DWN.033 重力人(グラビティーマン/Gravityman)
: 佔領反[[重力|重力]]研究所的機器人，個性開朗。武器為同一個研究所所開發重力控制裝置「重力拋」，在狹小的房間中控制重力，喜好研究物理學。當不喜歡的同伴造訪時會利用反重力躲到天花板上。有著再反轉重力時毫無防備的弱點，所以配備了基本武力，在反轉重力時以自己為中心給予對手射擊。
: 舞台的設計靈感來自同為[[卡普空|卡普空]]公司所出品的《[[出擊飛龍|出擊飛龍]]》系列遊戲，關卡中配備了反轉重力的裝置。
: 最初角色設計的性別為女性。
; DWN.034 水流人(ウェーブマン/Waveman)
: 水陸兩用戰鬥用機器人，威利以「征服世界的第一步就是要確保水源控制」的前提製造，用來佔領水質管理局，討厭人群。特殊武器為沿著地面行進的高壓水柱「水柱波浪」。脾氣暴躁，憤怒時會用魚叉刺向對手，不喜歡下水，且不擅長近距離戰鬥。
: 過去歐美發表的同人遊戲《[[洛克人3_(DOS版)|洛克人3 (DOS版)]]》中有同名的角色登場。
; DWN.035 石頭人(ストーンマン/Stoneman)
: 用[[石頭|石頭]]打造的巨型機器人，用在攻佔空中庭園時，順便佔領附近的高山作為前線基地。身體很重，材質則為便宜的石頭，容易解體 ，在著地分解時會觸動身上的重組裝置來回復，特殊武器為使用三顆往外擴張的石頭，以漩渦軌道飛出的攻擊的「岩石護盾」。性格如同石頭般頑固，與《[[洛克人_(游戏)|洛克人1]]》的氣力人是好友，兩人常一起喝油。
; DWN.036 螺旋人(ジャイロマン/Gyroman)
: 空中戰鬥用機器人。奉威利的命令佔領空中庭園來收取門票錢，但占領後客人反而銳減，威利索性改造成基地後讓他看守。原本搭載著噴射用的飛行裝置和攻擊用飛彈，但礙於資金不足所以改成攻擊與飛行兩用的[[螺旋槳|螺旋槳]]，但本人並不在意這回事。擅長躲在雲中攻擊對手，特殊武器為丟出螺旋槳型的刀片進行攻擊的「螺旋飛刀」。
; DWN.037 星辰人(スターマン/Starman)
: 宇宙戰鬥用機器人，奉威利的命令佔領[[地球|地球]][[軌道_(力學)|軌道]]上[[衛星|衛星]][[基地|基地]]。有著「華麗貴公子」的稱號，是個極度自戀及喜歡美麗物品的機器人，興趣為鑑賞[[歌劇|歌劇]]和[[音樂劇|音樂劇]]。雖然負責監督衛星基地的改造工程，但只顧著打扮自己而導致工程落後。夢想是在宇宙找到自己理想的[[戀人|戀人]]並一起旅行。特殊武器為星星狀的強力屏障「星辰防護罩」。與《[[洛克人3：威利博士的末期！？|洛克人3]]》的忍者人同樣使用地球以外的材質製造。
; DWN.038 充氣人(チャージマン/Chargeman)
: 從[[輸送|輸送]]用機器人改造。將物資供應給太空站基地而製作的運輸機器人。因為資金不足的緣故，所以動力系統使用蒸汽裝置，對自己能省錢感到得意，本身也很愛錢。有著拼命三郎的個性，做事為一直線往前衝的。特殊武器為滑行時腳底能放出衝擊波的「充氣撞」。
: 最初收到的投稿名稱為「蒸氣人」，特殊武器為變形時發動的必殺技「移動衝擊」，但是沒有被採用。
; DWN.039 汽油人(ナパームマン/Napalmman)
: 威利製造，全身藏滿重火器武力的戰鬥用機器，負責管理祕密兵工廠。在工場中來回地看著自己新得到的兵器為每日的必備功課。身上裝著包括特殊武器「汽油彈」等強力武器，同伴都不太敢接近他，因為朋友很少所以有著膽小的一面。
: 在《[[洛克人大賽車|洛克人大賽車]]》為了用賞金來建造兵器[[博物館|博物館]]而來參加比賽。
: 在《[[洛克人10：來自宇宙的威脅！！|洛克人10：來自宇宙的威脅！！]]》中，作為5代頭目的代表，以可以重現過去機器人頭目招式的「武器存檔機」的造型代表再次登場。
; DWN.040 水晶人(クリスタルマン/Crystalman)
: 體內可以製造[[水晶|水晶]]原石的機器人，最初開發的目的是因為要賺取資金，體內可生產人工水晶。身體包括[[關節|關節]]在內的不少部位都是水晶做成，所以有著媲美石頭人的硬度。擁有比其他機器人跳更高的高度。特殊武器為碰到牆壁等障礙時會分裂為三顆小水晶，且會繼續反射的「水晶眼」。
: 與前作登場的吸塵人相同，原設計者為[[漫畫|漫畫]][[光速蒙面俠21|光速蒙面俠21]]的作者[[村田雄介|村田雄介]](當時14歲)。最初投稿名稱為「鏡面人」，可以集中凸透鏡的光來攻擊。

=== 其他頭目 ===
; 黑暗人I(ダークマンI/Darkman I)
: 布魯斯城的第一頭目，擔任布魯斯的守衛，黑暗人系列中唯一的戰車型。有著高攻擊力，受到損傷達一定程度時會加快速度。
; 黑暗人II(ダークマンII/Darkman II)
: 布魯斯城的第二頭目，黑暗人系列中唯一的完整人型。其強力的電磁波可使敵人的攻擊偏轉，但因威利忘了裝上其武器的緣故，導致他除了衝撞以外沒有任何攻擊手段。
: 雖然名為II，但實際上為黑暗人系列的基本型。
; 黑暗人III(ダークマンIII/Darkman III)
: 布魯斯城的第三頭目，在黑暗人中只有他的右手安裝強大威力的砲。有著在跳上空中狙擊洛克人的高機動性。胸部射出的光線可以使洛克人暫時停滯。
; 黑暗人IV(ダークマンIV/Darkman IV)
: 布魯斯城的第四頭目，冒牌布魯斯的真面目，有著黑暗人系列的最強戰鬥力。可以藉由移動自己身上零件來改變自己的巨大體型，利用這特性來假扮布魯斯，最後被布魯斯本人拆裝偽裝，口哨聲與布魯斯本尊也稍有不同。具備了黑暗人II的電磁波和黑暗人III的武力，以及比黑暗人I更強的攻擊力。
; 大培茲(ビッグペッツ/Big Pez)
: 威利城的第一頭目，原本是在遊樂園指點遊客路徑的柱狀機器人，但被威利改造。會以類似[[不倒翁跌倒了|不倒翁跌倒了]](日本的一種遊戲，用木槌將柱狀不倒翁的身體部分敲出)的方式進行攻擊。
; 圓環Q9(サークリングQ9)（ - キューナイン）(Circling Q9)
: 威利城的第二頭目，有著厚實裝甲防禦的大型機器人。其裝甲可以阻擋任何攻擊，但為了機體散熱而做的兩個換氣口卻成了重大的弱點。
; 威利壓縮機(ワイリープレス/Wily Press)
: 威利城的第三頭目。威利發現洛克人對上下攻擊不拿手而設計的幽浮狀機械，雖然威利本人認為這是個了不起的發明，但說穿了只是個會壓頂攻擊的壓縮機罷了。
; 威利機器5號(ワイリーマシーン5号/Wily Machine 5)
: 威利城的第四頭目，為大型戰車型態的對洛克人作戰用機器。本來有變形功能，但由於製造時的配線錯誤，導致原本第二型態的變形動作變成了發射逃生艙，但其逃出裝置已有不俗的戰鬥力。
; 威力膠囊II(ワイリーカプセルII/Wily Capsule II)  
:新加入了瞬間移動的能力，攻擊方式比前作還多，也比前作的更強，在打完威利機器後接著上場。
