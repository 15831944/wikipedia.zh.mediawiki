'''洛天依'''是[[上海禾念|上海禾念]]推出的一套采用[[Vocaloid|Vocaloid]][[语音合成|语音合成]]技术的[[库_(计算机)|数据库]]及[[虚构角色|虚构角色]]。歌声样本取自中国[[配音演員|配音演員]][[山新|山新]]和日本[[歌手|歌手]][[鹿乃|鹿乃]]两位，ideolo为「Vocaloid™ China Project」设计出该角色。上海禾念在2012年7月12月发布其首个产品。五年后，其升级产品于2017年12月30日发售，日语产品在2018年5月21日发售。

作为第一位发布的中文Vocaloid歌手，在ACG圈内拥有较多粉丝及人气。在2016年，洛天依登上湖南卫视小年夜春晚，这是其第一次走上主流舞台。2017年6月举办的Vsinger Live则是洛天依目前规模最大的商业演出<ref>{{cite web |url= http://blog.sina.cn/dpool/blog/ArtRead.php?nid=13762eb7d0102xe0j |title= 洛天依：进击的中国虚拟偶像_刺猬公社_新浪博客 |author= 彦东|date= 2017-06-26|language= zh-hans }}</ref>。

== 背景 ==
{{Quote box
| quote  = “新人VOCALOID，VOCALOID学院的优秀新生，出身于保留着中国传统风貌的领“神州领”。对过去曾经在人类世界创造了历史的传说中的VOCALOID前辈们非常憧憬，也期待着自己有朝一日能够来到人类世界，成为新的VOCALOID歌姬，这样的她在某日突然获得了VOCALOID学院的传召，并且带着某个重要的任务，作为新的VOCALOID使者来到了人类的世界。

洛天依是一名被设定为十五岁的少女，出生在远离人类世界“[[人类世界_(北欧神话)|米德嘉特]]”（Midgard）的世界“[[華納海姆|瓦纳海姆]]”（Vanaheim）上，为了某个任务而来到人类世界并与[[乐正绫|乐正绫]]等其他小伙伴相遇，天依利用自己的特殊能力“共鸣”唱出了绫心中的歌曲《step on your heart》。她有一只从小相伴的音之精灵，叫做天钿，它被设定为十岁且身长三十厘米，能化身麦克风，“会代替无法交谈的天依表现情绪”，来自第三世界“[[亞爾夫海姆|精灵乡]]”（Álfheim）。天依的象征物是[[中国五声音阶|中国五声音阶]]中的“羽”，代表符号是梵文“{{lang|sa|म}}”<ref name="artbook"/>。

Vocaloid™ China Project制作人王杲引用《汉书·律历志》中的“羽为水声，居北方，时序为冬”来体现洛天依的性格和她作为角色给人的印象<ref>{{cite web |url= https://tieba.beidu.com/p/3823728760 |title= [细节+科普]关于DEMO曲和声库版绫帅_乐正绫吧_百度贴吧 |author= jarryestel |date= 2015-06-13 |language= zh-hans }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。也在[[百度贴吧|百度贴吧]]上解释了名字的含义：“洛”是一个拥有中国地理（[[洛阳市|洛阳]]，[[洛河_(黄河)|洛水]]）和中华文化（[[洛神|洛神]]）含義的漢字，同时也是常见姓氏；“天”也是国人常用来指代自身的字（比如天朝），洛天依本身的设定也是天降的歌姬；“依”谐音“一”，表示第一位中文歌姬，同时拥有依靠互相支持之意，同时拥有“伊人”之意，中国文化中非常常见的对美丽少女的称谓<ref>{{cite web |url= https://tieba.bidu.com/p/2307125252 |title= 【科普向】洛天依名字的由来_洛天依吧_百度贴吧 |author= 龙皇优阳 |date= 2013-05-04 |language= zh-hans }}</ref>。

ideolo称早期公式服的设计灵感是来自游戏《[[怪物猎人|怪物猎人]]》的角色海龙。<ref>{{cite web |url= https://weibo.com/1009398820/DARw1AHwf |title= STAFF采访面对VC国民岳父i大，你想知道... 来自VC相关推广站 - 微博 |author= VC相关推广站 |date= 2016-06-18 |language= zh-hans }}</ref>并且在最初的设定中，天依拥有两种服饰风格，普通模式（TYPE B，即公式服）和热唱模式（TYPE A），分别适用于慢歌和快歌环境下。<ref name="artbook"/><ref>{{cite web |url= https://tieba.beidu.com/p/1731412396 |title= 【授权转】vocaloid china详细人设图解=====洛天依_洛天依吧_百度贴吧 |author= jarryestel|date= 2012-07-17|language= zh-hans }}</ref>

== 宣传 ==
Vocaloid™ China在2012年6月6日至11月12日推出了五篇宣传视频，由七灵石动漫画有限公司负责制作，视频中大致地讲述了洛天依的背景和相遇乐正绫的事。制作人王杲考虑到同人精神而未给视频配音{{来源请求}}。

'''VOCANOVA'''是VOCALOID CHINA第一个中文虚拟偶像洛天依在[[iOS|iOS]]平台的音乐互动游戏。玩家可以跟随洛天依的歌舞进行游戏。

VOCANOVA于2012年7月13日在[[App_Store_(iOS)|App Store]]上线，目前仅支持iPad 2和New iPad。

在游戏中，玩家可以根据歌曲节奏，在画面上根据下落的节奏符号进行点击、滑动、拖动等操作。
