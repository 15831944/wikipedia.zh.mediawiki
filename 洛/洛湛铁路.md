'''洛湛铁路'''是[[中国|中国]]一条连接[[河南省|河南省]][[洛阳市|洛阳市]]与[[广东省|广东省]][[湛江市|湛江市]]的铁路，北起河南省洛阳市，经[[平顶山市|平顶山市]]、[[南阳市|南阳市]]，[[湖北省|湖北省]][[襄阳市|襄阳市]]、[[荆门市|荆门市]]，[[湖南省|湖南省]][[常德市|常德市]]、[[益阳市|益阳市]]、[[娄底市|娄底市]]、[[邵阳市|邵阳市]]、[[永州市|永州市]]，[[广西壮族自治区|广西壮族自治区]][[贺州市|贺州市]]、[[梧州市|梧州市]]，在[[岑溪市|岑溪市]]分西、南两路，其中西路经[[玉林市|玉林市]]至湛江市，南路经[[广东省|广东省]][[茂名市|茂名市]]至湛江市。洛湛铁路是中西部地区至华南地区及沿海地区深水港口的重要出海通道，是[[中华人民共和国铁路运输|中华人民共和国铁路运输]]的“[[八纵八横|八纵八横]]”铁路干线之一大（同）湛（江）通道之一段。洛湛铁路通道与京广、京九、成昆铁路共同构成中国铁路纵向路网骨架，从而缓解焦柳铁路南段和京广铁路南段的运能紧张状况，同时形成中西部地区通往华南各港口的重要出海通道。

== 线路构成 ==
洛湛铁路实际修建路段为益阳东站至娄底站、邵阳站至马路圩站、马路圩站至玉林站、马路圩站至电白站四段，其他路段中关林站至石门县北站区间使用[[焦柳铁路|焦柳铁路]]既有线，石门县北站至益阳区间使用[[石长铁路|石长铁路]]既有线，娄底站至邵阳站区间使用[[娄邵铁路|娄邵铁路]]既有线，玉林站至湛江站区间使用[[黎湛铁路|黎湛铁路]]既有线，电白站至湛江站之间使用[[广茂铁路|广茂铁路]]、[[茂湛铁路|茂湛铁路]]既有线。

为了平衡广西和广东两省的利益<ref>[http://news.sina.com.cn/c/2004-10-12/11173899320s.shtml 洛湛铁路玉林段今冬明春动工(图)]</ref>，洛湛铁路在岑溪马路圩站分为两个走向：西向经[[容县|容县]]、[[北流|北流]]，与黎湛铁路相接，这段线路被部分资料称为马玉铁路；南向则经[[信宜|信宜]]、[[高州|高州]]、[[电白|电白]]与广茂铁路、茂湛铁路相连，这段线路被部分资料称为马电铁路。

洛湛铁路河南段属[[郑州铁路局|郑州铁路局]]管辖。湖北段属[[武汉铁路局|武汉铁路局]]管辖。湖南段总长321公里，投资41.5亿元，属[[广州铁路集团|广州铁路集团]]运营。广西段线路经过[[南岭|南岭]]山脉中段[[紫金山|紫金山]]、[[大桂山|大桂山]]、[[小平山|小平山]]三大分水岭，跨越[[潇水|潇水]]、[[贺江|贺江]]、[[桂江|桂江]]、[[浔江|浔江]]等河流，总长380公里，投资约84亿元，属[[南宁铁路局|南宁铁路局]]管辖。广东境内的马路圩到[[电白|电白]]区间线路属[[广州铁路集团|广州铁路集团]]管辖。

== 修建历史 ==
2000年12月27日益阳至永州段开工。2004年12月15日全线开工。2006年益阳至永州段开通，当时称为益永铁路。2008年12月31日广西段贯通<ref>[http://www.gxnews.com.cn/staticpages/20090101/newgx495c07b5-1840138.shtml 广西新闻网]，洛湛铁路全线贯通</ref>，2009年7月1日开行货车<ref>[http://news.xinhuanet.com/fortune/2009-07/01/content_11634348.htm 新华网]，洛湛铁路湖南永州至广西玉林段开通运营</ref>，同年9月9日开行客车<ref>[http://www.wuzhoudaily.cn/html/2009-09/10/content_93845.htm 梧州日报多媒体数字报刊平台] {{webarchive|url=https://web.archive.org/web/20160304220820/http://www.wuzhoudaily.cn/html/2009-09/10/content_93845.htm |date=2016-03-04 }}，洛湛铁路永州至玉林段旅客列车在我市举行首发仪式</ref>。

洛湛铁路全线通车，这标志着中国“八纵八横”铁路网建设顺利实现，也标志着中国铁路“八纵”中最后“一纵”正式通车。
