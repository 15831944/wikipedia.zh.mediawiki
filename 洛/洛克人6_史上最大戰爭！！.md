== 概要 ==
[[洛克人元祖系列|洛克人元祖系列]]的第六作，此代頭目破例採用海外國家的募集資料。
由八大頭目關卡，加上X先生城四關卡和威利城四關卡共16關卡，由四大關卡的分叉路線首度登場。<ref name="Nintendo" /><ref name="PS配信" /><ref name="GA" />

== 劇情 ==
為了對抗數度征服世界的威利博士，各國發起成立了世界機器人聯盟，並主辦了第一屆世界機器人大會，其中選出了能對抗威利的八具最強機器人。但就在比賽如火如荼進行時，八具機器人突然暴動，而身為贊助者的X先生卻在此時宣布要代替威利，用這八具機器人征服世界。

為了阻止X先生的野心，洛克人又將再次出動。

== 系統變更 ==
* 八大頭目為人馬人、騎士人、戰斧人以及武士人的四種關卡分叉路線各藏著「B」「E」「A」「T」四個字母電路板，若全部收齊便可使用前作登場的援助機器鳥「比特」。
* 首次追加道具「能源平衡器」。武器能源可以直接拿，而無須切換才能補。
* 取消歷代以來所使用的彈簧拉休與噴射拉休，取而代之的是可以與拉休合體所誕生的噴射洛克人與威力洛克人。

== 主要角色 ==
; {{nihongo|洛克人|ロックマン|Rockman}}:原本是家用的清潔機器人，因為看到威利博士作亂，所以要求萊特博士將自願改造成戰鬥機器人。
; {{nihongo|萊特博士|ライト博士|Dr. Right}}:被世人稱為「機器人工學之父」，他和藹可親、厭惡戰爭，希望人類能與機器人和平相處。
; {{nihongo|布魯斯|ブルース|BLUES}}:不知是敵是友的神秘機器人，總是暗中協助洛克人。
; {{nihongo|X先生|Mr.X}}:世界機器人大會的贊助者，真面目為威利，利用舉辦世界機器人大會來控制八位機器人。
; {{nihongo|威利博士|Dr.ワイリー|Dr. Wily}}:X先生的真面目，自稱「邪惡天才科學家」。

== 八大頭目 ==
; DWN.041 暴雪人(ブリザードマン/Blizzardman)
: 雪人造型的[[南極|南極]]觀測用機器人，特殊武器為將產生人工[[雪|雪]]的氣象觀測裝置改造而來的「暴雪攻擊」，成為戰鬥用機器人。另一個身分為[[冬季運動|冬季運動]]的知名選手，[[冬季奧林匹克運動會|冬季機器人奧林匹克]]比賽中連續三屆取得[[金牌|金牌]]的優秀好手。最近在從事[[氣象|氣象]][[主播|主播]]的工作，動力源為太陽能。

; DWN.042 人馬人(ケンタウロスマン/Centaurman)
: 古代歷史[[博物館|博物館]]的嚮導機器人，為了得到優勝獎金來整修老舊的博物館而參加比賽，能力不明。參考[[希臘神話|希臘神話]]中登場的[[半人馬|半人馬]]所製作，有著控制時間與空間的能力，可以在任何場所瞬間移動。
: 特殊武器為技術上應用了閃光人的「時間暫停」以及燈泡人的「閃光燈泡」，成為可以扭曲時空的武器「人馬閃光」，但缺點是武器的能量消耗頗高。能量為[[電池|電池]]（未來的乾電池可以用非常久），走路時有獨特的馬蹄聲。
: 在[[池原威利|池原威利]]的漫畫版中，是與騎士人為戀人關係的女性機器人，頭盔下的真面目為金色長髮。

; DWN.043 烈火人(フレイムマン/Flameman)
: 頭上捲著[[頭巾|頭巾]]的舊式[[火力發電|火力發電]]型機器人，安定性卻遠比舊式的[[核能|核能]]及現行的[[太陽能|太陽能]]發電型機器人更為安定。光自身發電機就可以供應整座發電廠的電力。特殊武器為體內的火力發電機發射出來的「爆破火焰」。一天要喝三次油，且非常注重準時。認為「核能及太陽能機器人毫無用處可言」。
: 在『[[洛克人10：來自宇宙的威脅！！|洛克人10：來自宇宙的威脅！！]]』中，作為6代頭目的代表，以可以重現過去機器人頭目招式的「武器存檔機」的造型再次登場。

; DWN.044 騎士人(ナイトマン/Knightman)
: 以[[中世紀|中世紀]]的[[騎士|騎士]]為範本打造的戰鬥型機器人，已經打敗超過1000具機器人。有著重視攻擊力與防禦力的設計，左手攜帶著鐵球型特殊武器「騎士裂鎚」右手則裝備著盾牌，但過重的身體讓他失去機動性，有著喜歡正當決勝負的騎士精神。
: 在[[池原威利|池原威利]]的漫畫版中，與人馬人為戀人關係。

; DWN.045 花人(プラントマン/Plantman)
: 在[[植物園|植物園]]管理[[植物|植物]]的管理機器人，在隊伍中有如[[吉祥物|吉祥物]]般的存在，之後塗上特殊迷彩並改造成戰鬥用。能與植物溝通，痛恨踐踏花草的人。特殊武器為將能量濃縮成花瓣型後發射出的「苗木阻擋層」。對蜜蜂和自然環境的急遽變化沒有抵抗力，尤其非常怕冷。
: 原始設定為「花朵人」。

; DWN.046 戰斧人(トマホークマン/Tomahawkman)
: 為了戰鬥而開發的戰士機器人，[[北美洲|北美洲]]的代表選手，造型參考自[[蘇族|蘇族]]的[[印地安|印地安]][[酋長|酋長]]。有著不畏懼死亡的勇敢性格及優秀的高攻擊力，特殊武器為右手裝備，可讓100公尺外的[[蠟燭|蠟燭]]熄火的「銀色戰斧」，以及彌補死角的羽毛型飛彈攻擊，對接近戰很不拿手。美國精神濃厚，討厭說謊。

; DWN.047 風扇人(ウインドマン/Windman)
: 農業用機器人，工作為釋放有助農作物生長的風。與[[洛克人2：威利博士之謎|2代]]登場的空氣人使用相同的技術，但與空氣人相比較偏向人型。同時並用[[螺旋槳|螺旋槳]]動力和[[噴射機|噴射機]]動力來操縱[[風|風]]，有著吹起200公里風速的威力。可以利用自身的風力移動，腳底也應用了氣墊技術，所以能夠短距離飛行。特殊武器為發射突風的武器「颶風風暴」，被同類型的空氣人視為競爭對手。頭上的帽子如同[[拉麵|拉麵]]碗，對料理拿手。
: 原國籍不明、在[[有賀等|有賀等]]的漫畫版中為[[中華人民共和國|中國]]製的機器人。

; DWN.048 武士人(ヤマトマン/Yamatoman)
: 全身包裹著[[武士|武士]][[鎧甲|鎧甲]]，如同日本武士的戰鬥用機器人。在設計時參考了騎士人，雖然看起來很重，但追求輕量化的同時也將鎧甲給輕薄化。特殊武器為將[[長槍|長槍]]的前端射出攻擊的「武士飛槍」，但槍頭發射後不能回收。將磨槍列為每日必做的功課。
: 有著武士道精神，尊敬有著騎士道精神的騎士人。使槍的好手，但令人意外的是一個日本刀狂熱分子。不擅英文，閒暇時會在英語教室附近出沒。

=== 其他頭目 ===
; 繞行者II(ラウンダーII/Rounder II)
: X城的第一頭目。由[[洛克人5：布魯斯的陷阱！？|前作]]登場的雜魚'''繞行者'''改良而來。因為重量的緣故無法飛行，以兩台一組在軌道上移動。
; 強力活塞(パワーピストン/Power Piston)
: X城的第二頭目，巨大的活塞型機械。利用威力強大活塞來撞擊要塞牆壁 ，利用地震來使要塞的天花板崩塌，進而把侵入者給活活壓死，有著很高的防禦力。
; 梅德路金剛Z(メットンガーZ/Mettenger Z)
: X城的第三頭目，由梅德路操縱，除了駕駛槍外接包覆著超合金裝甲的戰車機器人，形狀為威利參考『[[無敵鐵金剛Z|無敵鐵金剛Z]]』的頭部而製作。
; X粉碎機(Xクラッシャー/X Crasher)
: X城的第四頭目，X先生親自操縱的球型機器人。以單擺的方式來進行移動式的撞擊，但因為移動速度過快和擺動角度太大，導致X先生在上頭很容易頭暈。最近X先生似乎有把它改造成一般交通工具的打算。
; 機械雷龍(メカザウルス/Mecha Zaurus)
: 威利城的第一頭目，參考恐龍圖鑑做出來的恐龍型機器人。口中可發射火焰，腹部可發射圓盤來攻擊，且包覆者可擋住多數攻擊的裝甲。
; 坦克CSII型(タンクCS-II型/Tank CS-II)
: 威利城的第二頭目，對洛克人專用的戰鬥型三輪坦克。可用紅外線精準感應對方的攻擊動作後，再進行強大的反擊。
; 威利機器6號(ワイリーマシーン6号/Wily Machine 6)
: 威利城的最終頭目，以前作的威利壓縮機改良而成。將壓制力改良，在往下跳之後可以立刻彈起再次攻擊，同時也加裝了一次三發的能源彈。但缺點是攻擊時無法移動。被打破上部裝甲後的第二型態的威力減低很多，一次只能發射單發能源彈。
; 威利膠囊III (ワイリーカプセルIII╱Wily Capsule III)  
:在歷代的威利膠囊中是最大的，一次可以連發四顆能源彈，在打完威利機器之後由上部的駕駛艙分離並接著上場。
