'''蓬榮站'''（{{jpn|j=蓬栄駅|hg=ほうえいえき|rm=Hōei eki|f=y}}）位於[[北海道|北海道]][[日高郡|日高郡]][[新日高町|新日高町]]三石蓬榮，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）
[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|ホウ}}'''。站名由來是附近有一座「蓬萊岩」，祈求昌盛的「榮」所合成。

== 車站構造 ==
[[側式月台|側式月台]]1面1線的地面站。沒有站房，只有一座小型的候車室。

== 車站周邊 ==
附近多為牧場和田地。
* 三石大壩
* シカルベ山
* 蓬萊岩
* 三石川

== 歷史 ==
* 1958年（[[昭和|昭和]]33年）7月15日 - 設立，只處理旅客業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
