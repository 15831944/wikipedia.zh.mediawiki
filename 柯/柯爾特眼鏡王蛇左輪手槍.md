'''柯特眼鏡王蛇'''（{{Lang-en|'''Colt King Cobra'''}}，以下簡稱為“眼鏡王蛇”）是一款由[[美国|美国]][[康乃狄克州|康乃狄克州]][[哈特福|哈特福]]的{{Link-en|柯爾特生產公司|Colt's Manufacturing Company|柯爾特}}所[[設計|設計]]和[[制造业|生產]]並且向[[警察|執法機關]]和[[平民|民間]]槍械愛好者[[目標市場|銷售]]的6發式中等[[机匣|底把]][[扳機|雙動操作]][[左輪手槍|左輪手槍]]，於1986年首次推出，被認為是[[柯爾特蟒蛇左輪手槍|柯爾特蟒蛇]]（{{Lang-en|Colt Python}}）的後繼槍。它具有{{Link-en|烤藍|Bluing (steel)}}[[钢|钢]]和[[不鏽鋼|不鏽鋼]]兩種表面處理，可以使用不同的槍管長度，發射[[中央式底火|中央底火式]][[.357麥格農|.357 S&W馬格南]]和[[.38_Special|.38特種彈]]等子彈。<ref name="kr">[http://www.kitsune.addr.com/Firearms/Revolvers/Colt_King_Cobra.htm “Colt King Cobra”]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}, Kitsune Reference Web site. Accessed November 18, 2008.</ref><ref>[http://www.gunsworld.com/colt/king_us.htm “Colt King Cobra”], Guns World Web site. Accessed November 18, 2008.</ref><ref name="cr">[http://www.coltrevolvers.net/trooper.html “Colt Model 357 and Trooper”] {{webarchive|url=https://web.archive.org/web/20090213131721/http://coltrevolvers.net/trooper.html |date=2009-02-13 }}, Colt Revolvers Web site. Accessed November 18, 2008.</ref>這是一把現代武器，可靠，火力強大，用途非常廣泛：它被用作為瞄準射擊、自我防衛和狩獵。

== 研發和歷史 ==
眼鏡王蛇的設計是以柯爾特早期型型號，歷史悠久的[[柯爾特騎兵型左輪手槍|騎兵MK V型]]為藍本。改進的內容除了裝上增加強度的重型[[槍管|槍管]]，還使用了在槍管底部的全尺寸型[[弹巢|弹巢]]退殼桿保護凸耳（{{Lang-en|Underlug}}），以及在槍管頂部的一條厚實堅固的散熱[[肋條|肋條]]。眼鏡王蛇在1986年推出，1992年首度[[產品壽命結束|停产]]；1994年，柯爾特重開眼鏡王蛇的生產線，然後1998年第二度停产，以後都不再生產。

直到2019年1月，柯爾特宣布他們將第二度重開眼鏡王蛇的生產線，並且銷售。其唯一的配置是[[不鏽鋼|不鏽鋼]]製，配有3英寸的槍管（類似於為蒙特利爾警察局生產的6英寸型）。

== 設計細節 ==
眼鏡王蛇是由柯爾特的中等尺寸的Ⅴ式[[机匣|底把]]為基礎所設計的，在1986年至1992年之間使用非常高級的[[碳鋼|碳鋼]]製造，並且使用當時柯爾特招牌的明亮而且高級[[拋光|拋光]]深[[皇室藍|皇家藍色]]（{{Link-en|烤藍|Bluing (steel)}}處理）處理。在1987年至1992年和1994年至1998年之間還推出了[[磨砂|磨砂]][[不鏽鋼|不鏽鋼]]製版本，以及在1988年至1992年之間有[[光澤|光澤]]的[[拋光|拋光]]型[[不鏽鋼|不鏽鋼]]可以選擇。眼鏡王蛇可以選擇配備超大尺寸型[[胡桃|核桃]]製瞄準型樣式或硬[[橡膠|橡膠]]製[[黑色|黑色]]連[[手指凹槽|手指凹槽]][[戰鬥|戰鬥]]樣式[[手槍握把|握把]]，以及一個加大版瞄準型{{Link-en|擊錘|Hammer (firearms)}}。<ref name="kr"/>眼鏡王蛇的瞄準系統是由固定的鐵製紅色刀片型[[照準器|準星]]以及一個完全可調的鐵製白色輪廓的[[照門|照門]]。<ref name="kr"/>隨著時間的推移，眼鏡王蛇推出各種[[槍管|槍管]]長度到表面處理的選擇。<ref name="kr"/>

在1986年，眼鏡王蛇推出了4英吋（101.6毫米）和6英吋（152.4毫米）長度[[槍管|槍管]]{{Link-en|烤藍|Bluing (steel)}}型號，直到烤藍鋼型槍械在1992年停止生產，而在1990年至1992年之間也推出了使用烤藍處理的2.5英吋（63.5毫米）長度槍管型號。而在[[磨砂|磨砂]][[不鏽鋼|不鏽鋼]]製型號方面，2.5英吋（63.5毫米）長度槍管型號從1987年推出開始，直到眼鏡王蛇在1992年停止生產以前都在[[產品目錄|產品目錄]]上；而2英吋（50.8毫米）長度槍管型號從1988年推出開始，直到眼鏡王蛇在1992年停止生產，然後它們再在1994年至1998年之間一度重新回到了產品線上。[[磨砂|磨砂]][[不鏽鋼|不鏽鋼]]製型號也在1990年至1992年之間推出了其4英吋（101.6毫米）、6英吋（152.4毫米）和8英吋（203.2毫米）槍管長度型號，然後它們再在1994年至1998年之間一度重新回到了產品線上。而在有如鏡子般明亮的[[拋光|拋光]]型[[不鏽鋼|不鏽鋼]]處理型號方面，在1988年至1992年之間推出了4英吋（101.6毫米）和6英吋（152.4毫米）長度[[槍管|槍管]]型號，在1990年至1992年之間推出了2.5英吋（63.5毫米）長度槍管型號，以及在1990年至1992年之間推出了8英吋（203.2毫米）長度槍管型號。

眼鏡王蛇的[[膛室|膛室]]（[[弹巢|弹巢]]）可以裝填以及發射火力強大的[[.357麥格農|.357 S&W馬格南]][[手枪|手枪]][[子彈|子彈]]，或是就像所有的.357[[口徑|口徑]]左輪手槍那樣發射火力相對地較弱的[[.38_Special|.38 S&W特種]][[彈藥|彈藥]]。<ref name="kr"/>
