《'''柯里昂家族'''》（{{lang-en|'''''The Family Corleone'''''}}）是一本由{{link-en|艾華·法爾科|Edward Falco}}所寫的2012年小說，改編自[[馬里奧·普佐|馬里奧·普佐]]未製作的劇本，而馬里奧·普佐於1999年去世。<ref>{{Cite web |url = https://archive.is/20130205100729/http://usatoday30.usatoday.com/life/books/news/story/2012-05-07/the-family-corleone/54794844/1 |title = Prequel lays out life before 'The Godfather' |author = Craig Wilson |date = 2012-05-07 |publisher = USA Today |language = en}}</ref>該小說是普佐所寫的《[[教父_(小說)|教父]]》的前傳。該小說在2012年5月8日發售。

== 有聲書 ==
《柯里昂家族》的有聲書由{{link-en|Hachette Book Group|Hachette Book Group|Hachette Audio}}公司製作，[[波比·簡拿威|波比·簡拿威]]朗讀。
