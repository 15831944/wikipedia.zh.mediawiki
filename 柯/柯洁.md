'''柯洁'''（{{bd|1997年|8月2日||}}），[[中國|中國]][[围棋|围棋]]职业[[围棋段位制|九段]]棋手，[[浙江省|浙江]][[丽水市|丽水]]人。父亲柯国凡，母亲周柳萍。启蒙老师李守胜、周宗强<ref>{{cite web |url=http://sports.sina.com.cn/go/2016-01-07/doc-ifxnkmaw2083266.shtml |title=恩师:想在丽水找下一个柯洁 儿时的他下棋不怕输 |publisher=Sports.sina.com.cn |date= |accessdate=2017-05-23}}</ref>。七次世界冠军得主（史上最年輕七冠王）。在近3年的时间内，柯洁在韩国棋院排名委员会委员裴太日博士的世界排名<ref>{{Cite web |title=韩版围棋世界排名：与欧洲版差别大 连笑进前三 |url=http://sports.qq.com/a/20170407/044794.htm}}</ref>中、雷米·库隆（Rémi Coulom）创建的[[Go_Ratings|Go Ratings]]排名<ref>{{Cite web |url=http://www.goratings.org/ |title=World's Go Player Ratings |date=2017年5月}}</ref>中，均位列第一。按照Go Ratings排名，自2014年8月开始至2017年11月止，柯洁一直保持世界第一，随后被韩国棋手、前世界第一[[朴廷桓|朴廷桓]]九段重新超越<ref>{{Cite web |title=GoRatings历史排名 |url=https://www.goratings.org/zh/history/ |date=Jan 2018}}</ref>。2017年12月20日获得中国十佳劳伦斯冠军奖最佳非奥运动员奖。<ref>{{cite web |url=http://sports.sina.com.cn/go/2017-12-20/doc-ifypxrpp3085291.shtml |title=柯洁获劳伦斯最佳非奥运动员 围棋首获此殊荣 |first= |last=梓沫 |date=2017-12-20 |website=sports.sina.com.cn}}</ref>柯潔棋風靈活多變，善於轉換建立優勢，形勢判斷能力極強，若是局面落後，其攪棋翻盤能力也是一絕，尤善執白棋，號稱「白棋不敗」<ref>{{cite web |url=https://www.yikeweiqi.com/banner/52650/ |title=柯洁世界大赛猜先异常、白棋不败、三星杯战绩好的统计性分析-柯洁世界大赛的猜先异常与白棋不败集中在四强战之前 |website=弈客围棋-多一个维度发现世界}}</ref><ref>{{cite web |url=https://passport.weibo.com/visitor/visitor?entry=miniblog&a=enter&url=https%3A%2F%2Fweibo.com%2Fttarticle%2Fp%2Fshow%3Fid%3D2309404317179607651357&domain=.weibo.com&ua=php-sso_sdk_client-0.6.28&_rand=1547867624.0488 |title=Sina Visitor System |website=passport.weibo.com}}</ref>，番棋勝率極高，又稱「番棋不敗」<ref>{{cite web |url=https://news.sina.com.tw/article/20181109/28774786.html |title=柯潔世界大賽戰績靠番棋 猜先異常與白棋不敗統計 |date=2018-11-09 |website=新浪新聞中心}}</ref>。另外，柯潔通常落子極快，到了決定勝負的後半盤常常佔據絕對的時間優勢。唯一的短板是容易出現隨手棋。偶像是歌神[[張學友|張學友]]。关注足球運動。

== 职业大事記 ==
* 2008年7月初段 2010年7月二段 2011年7月三段 2012年7月四段。
* 2015年 [[百灵杯|百灵杯]] 3-2 勝 [[邱峻|邱峻]]九段奪冠(1月)，世界大賽首冠，中国大陆第16位围棋世界冠军，升九段<ref>{{citeweb|url=http://sports.sina.com.cn/go/2015-01-14/15397482667.shtml|title=17岁少年加冕围棋世界冠军|publisher=新浪|date=2015-01-14|accessdate=2015-01-14}}</ref>。超車[[时越|時越]]九段，登頂中國等級分第一(10月)。[[三星杯|三星杯]] 2-0 零封 時越奪第二座世界冠軍(12月)。
* 2016年 [[梦百合杯|夢百合杯]] 3-2 險勝[[李世石|李世石]]九段奪第三座世界冠軍(1月)(最年輕三冠王)、三星杯 2-1 勝[[柁嘉熹|柁嘉熹]]九段奪第四座世界冠軍(12月)(最年輕四冠王)。
* 2016年 [[百靈盃|百靈盃]] 1-3 負於[[陳耀燁|陳耀燁]]九段首次得世界亞軍 (世界大賽四冠一亞)。
* 2017年 [[中国乌镇围棋峰会|中国乌镇围棋峰会]] 0-3 被 [[AlphaGo|AlphaGo]] 零封，之后取得22连胜，打破[[李世乭|李世乭]]九段及[[吳清源|吴清源]]大国手的记录，僅次於[[李昌鎬|李昌鎬]]九段的24連勝<ref>{{Cite web|url=http://sports.sina.com.cn/go/2017-07-17/doc-ifyiakur9080584.shtml?cre=sportspagepc&mod=f&loc=1&r=9&doct=0&rfunc=15&tj=none&s=0|title=破李世石吴清源记录 柯洁再虐韩国棋手取22连胜|accessdate=2017-07-18|author=|date=|publisher=}}</ref>。[[新奥杯|新奧杯]] 3-2 勝 [[彭立尧|彭立嶢]]五段奪第五座世界冠軍(12月)(最年輕五冠王)。年獎金破千萬人民幣<ref>{{cite web|url=http://sports.sina.com.cn/go/2018-01-13/doc-ifyqrewh5359183.shtml|title=2017柯洁奖金已超千万 KO朴廷桓比肩井山裕太|first=|last=谢天舒|date=2018-01-13|website=sports.sina.com.cn}}</ref>。
* 2018年 三星杯 2-1 勝 [[安國鉉|安國鉉]]八段奪第六座世界冠軍(12月)(最年輕六冠王)。
* 2019年 百靈杯 2-0 零封 [[申真諝|申真諝]]九段奪第七座世界冠軍(1月)(最年輕七冠王)。
* 2019年 世界最強棋士決定戰 0-1 負於[[朴廷桓|朴廷桓]]九段獲第二次世界亞軍(世界大賽七冠兩亞)。

==网棋战绩==
自2017年3月1日起，腾讯野狐围棋网正式施行新的升段规则，9段棋手在最近20盘有效升降级对局中，有18盘以上（含18盘）战胜银冠以上（含银冠）棋手，则由9段升为10段，且上不封顶。3月15日，柯洁战胜[[孙腾宇|孙腾宇]]七段、[[尹畯相|尹畯相]]九段、[[范蕴若|范蕴若]]五段、[[辜梓豪|辜梓豪]]五段、[[元晟溱|元晟溱]]九段、[[檀啸|檀啸]]七段、[[连笑|连笑]]八段、[[李钦诚|李钦诚]]九段、[[范胤|范胤]]六段等等，成为[[绝艺|绝艺]]之后的第二个野狐10段，也是人类棋手中第一个10段<ref>[http://www.ttplus.cn/publish/app/data/2017/03/17/44832/share1.html 柯洁升上10段自嘲：虚名而已]</ref>。

2017年7月17日，柯洁开始以让二子的棋份和职业棋手下网棋<ref>{{cite web|url=http://weiqi.qq.com/news/8684.html|title=柯洁让职业棋手两子，胜多负少不服来战|website=weiqi.qq.com}}</ref>。

2014年10月24日第16届阿含桐山杯快棋公开赛夺冠后，柯洁获得与日本的阿含桐山杯冠军井山裕太进行中日冠军对抗赛的资格。赛后柯洁表示在对抗赛中，“我估计自己稍微好点吧，让他‘血溅五步’”<ref>{{Cite web|url=http://sports.sina.com.cn/go/2014-10-24/17457382487.shtml|title=柯洁：夺得职业首冠很激动 要让井山裕太血溅五步}}</ref>。后来果然获胜<ref>{{Cite web|url=http://sports.sina.com.cn/go/2015-03-14/15217542332.shtml|title=井山胜负手遭屠 柯洁实现“血溅五步”豪言}}</ref>。2016年3月1日，柯洁在与井山裕太共同接受采访时表示“当初是在井山说出日本棋手该转运的时候，他在表达他的信心，我当时刚好拿了冠军心态比较复杂，所以说出了那句话，也挺不好意思的，不过要是现在拿出来提就有点伤了大家的和气了”<ref>{{Cite web|url=http://sports.sina.com.cn/go/2016-03-01/doc-ifxpvysx1801622.shtml|title=柯洁：数学不好四连胜概率存在 提血溅五步伤和气}}</ref>。

2015年11月25日，柯洁在梦百合杯半决赛获胜后，展望与李世石（即[[李世乭|李世乭]]）的决赛时表示“李世石之前说他有五成希望取胜，我想如果一共是一百成的话，他有五成。另外我想说，传奇是时候落幕了。”<ref>{{Cite web|url=http://sports.sina.com.cn/go/2015-11-25/doc-ifxkwuwv3602470.shtml|title=柯洁：决赛李世石5%胜机 传奇是时候落幕了}}</ref> 三星杯夺冠后又确认了一次<ref>{{Cite web|url=http://sports.sina.com.cn/go/2015-12-09/doc-ifxmifzh4443443.shtml|title=柯洁：实现儿时愿望 对李世石95%胜算站得住脚}}</ref>。此言在中韩围棋爱好者中引起轩然大波，不过柯洁最后凭借粘劫收后3:2战胜李世石夺冠<ref>{{Cite web|url=http://sports.sina.com.cn/go/2016-01-05/doc-ifxncyar6369783.shtml|title=梦百合杯柯洁3比2胜李世石 夺个人世界第三冠}}</ref>。比赛也创下了空前的关注度<ref>{{Cite web|url=http://sports.sina.com.cn/go/2016-01-12/doc-ifxnkkuy7964488.shtml|title=柯洁决赛创围棋转播纪录 堪比NBA重量级比赛热度}}</ref>。

赛后李世石在韩国国内接受采访时表示：“柯洁年纪还小，棋的深度还不够，说他已具备引领时代的能力，难以令人信服。”而柯洁看到报道后当即发微博反击：“都6比2了还不服？好傲娇啊。”<ref>{{Cite web|url=http://www.chinanews.com/ty/2016/01-28/7736712.shtml|title=柯洁称李世石“傲娇” 两人“口水战”再升级}}</ref>

2016年3月5日，在第17届“农心辛拉面杯”世界围棋团体锦标赛最后阶段比赛中，李世石作为韩国主将击败了日本副将村川大介、中国副将连笑和日本主将井山裕太，再次和作为中国主将的柯洁对弈。赛前柯洁在微博上表示“最后还是得靠我”<ref>{{Cite web|url=http://weibo.com/u/2865101843?is_hot=1|title=棋士柯洁的微博}}</ref>。结果果然击败对手为中国夺得冠军<ref>{{Cite web|url=http://sports.sina.com.cn/go/2016-03-05/doc-ifxqafha0395675.shtml|title=农心杯决战柯洁再胜李世石 中国豪取三连冠}}</ref>。柯洁个人和李世石的历史战绩也改写为了8:2。

2016年3月，在[[AlphaGo李世乭五番棋|AlphaGo李世乭五番棋]]第一局过后，柯洁通过[[新浪微博|新浪微博]]表示“就算阿法狗战胜了李世石，但它赢不了我”。这立即引起了媒体和网友的关注，其微博粉丝数暴涨<ref>{{Cite web|url=http://tech.sina.com.cn/it/2016-03-10/doc-ifxqhmve9048527.shtml|title=揭秘谷歌围棋战大赢家：柯洁微博粉丝翻20倍|accessdate=2016-03-13|author=|date=2016-03-10|publisher=新浪科技}}</ref>。但10个月之後柯潔在非正式的网-{zh-cn:络;zh-tw:路}-快棋中三次敗於化名為「Master」的升级版[[AlphaGo|AlphaGo]]。

2016年中柯洁接连在LG杯和应氏杯中折戟。之后他表示“之前高估自己”<ref>{{Cite web|url=http://sports.sohu.com/20160628/n456645491.shtml|title=柯洁：之前高估自己下得太快 百灵杯不去想卫冕}}</ref>、“过去我太浮躁”<ref>{{Cite web|url=http://tech.sina.com.cn/it/2016-08-30/doc-ifxvixeq0746453.shtml|title=柯洁：不相信人工智能能让两子 过去我太浮躁}}</ref>。

2016年5月23日，柯潔回應最近輸給[[朴廷桓|朴廷桓]]表示：朴廷桓棋不见得怎么样，输他因自己问题<ref>{{cite web|url=http://sports.sina.com.cn/go/2016-05-23/doc-ifxsktkr5925146.shtml|title=柯洁：朴廷桓棋不见得怎么样 输他因自己问题|first=|last=笑|date=2016-05-23|website=sports.sina.com.cn}}</ref>。

2017年5月25日，在[[中国乌镇围棋峰会|中国乌镇围棋峰会]]柯洁与AlphaGo三番棋第二局赛后发布会上，[[中国国际电视台|中国国际电视台]]记者用英语向柯洁提问，并且语速非常快，现场翻译甚至无法及时准确翻译，柯洁以“如果你是中国记者，而我是中国棋手，我认为你应该用中文向我提问”回应而拒绝回答问题。随后CGTN记者不得不又用中文向柯洁重复了问题，他也解释称，CGTN是一个以英语为工作语言的平台，所以提问题需要用英语。同日晚，CGTN官方微博发文表示，英语是CGTN的工作语言，同时也向柯洁和现场翻译表达了歉意。该争议发生后，随即成为了网络热议话题<ref>{{cite web|url=http://tech.ifeng.com/a/20170525/44626228_0.shtml|title=央视旗下CGTN就英文提问向柯洁道歉：工作语言是英语|publisher=凤凰科技 |accessdate=2017-05-28|date=2017-05-25|archiveurl=|archivedate=}}</ref><ref>{{cite web|url=http://business.sohu.com/20170526/n494610598.shtml|title=央视因记者英文提问向柯洁道歉，但这次中国人就别太苛责CCTV了|publisher=搜狐 |accessdate=2017-05-28|date=2017-05-26|archiveurl=|archivedate=}}</ref>。
