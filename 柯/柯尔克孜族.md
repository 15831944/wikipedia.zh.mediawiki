'''柯尔克孜族'''（中国标准罗马字母：{{lang|en|Kirgiz}}<ref>GB3304-91 [http://www.moe.edu.cn/edoas/website18/22/info16422.htm 中国各民族名称的罗马字母拼写法和代码]  {{webarchive|url=https://web.archive.org/web/20100503143709/http://www.moe.edu.cn/edoas/website18/22/info16422.htm |date=2010-05-03 }}</ref>，[[柯爾克孜語|柯爾克孜語]]：<span style="font-family:arial">Кыргыз / قىرعىز</span>），也译作'''吉尔吉斯族'''，[[中亚|中亚]]的一支民族，为[[吉尔吉斯斯坦|吉尔吉斯斯坦]]的主体民族，多信仰[[伊斯兰教|伊斯兰教]]。中国境内有超過18万柯尔克孜人（2010年），主要分布于[[新疆|新疆]]西部的[[克孜勒苏|克孜勒苏柯尔克孜自治州]]。

== 历史 ==
突厥語 Qırğız，即中国史籍记载的黠戛斯，就是今天的柯尔克孜人，《[[史记|史记]]》中最早记录了柯尔克孜人的情况，称之为“鬲昆”。[[兩漢|兩漢]]时称“坚昆”，[[魏晋南北朝|魏晋南北朝]]至[[隋代|隋代]]称“结骨”、“契骨”、“纥骨”、“护骨”、“居勿”等。[[唐代|唐代]]时根据[[汉语|汉语]]的音译通称为“黠戛斯”，也有称“纥扢斯”。[[五代十國|五代十國]]、[[遼|遼]]、[[宋|宋]]、[[金|金]]时又称“纥里迄斯”，[[大蒙古國|大蒙古國]]和[[元代|元代]]时称“吉利吉斯”等，这些名称其实都是各时期各种不同的音译。清代時按准噶爾人的稱呼，稱其為布魯特。乃蛮、钦察、[[契丹|契丹]]等来自[[哈萨克汗国|哈萨克汗国]]的部族属于吉尔吉斯的“色尔特克勒克”（意为“外部之人”），不属于吉尔吉斯本部。

吉爾吉斯人原本世代居於[[西伯利亞|西伯利亞]]南部、[[葉尼塞河|葉尼塞河]]上游的盆地。在汉代受匈奴统治，二世纪中叶后受[[鲜卑|鲜卑]]和[[柔然|柔然]]统治。[[北齐|北齐]]、[[隋唐|隋唐]]时期先后受[[突厥汗国|突厥汗国]]、[[回紇|回紇]]统治，后在840年灭[[回鹘|回鹘]]后独立过一段时间，一直到924年才被耶律阿保机的[[大契丹国|大契丹国]]统治。[[蒙古帝国|蒙古帝国]]时期划入[[益兰州|益兰州]]，到17世紀，受到[[准噶尔汗国|准噶尔汗国]]蒙古人和[[俄罗斯帝国|俄罗斯帝国]]人排擠，漸漸遷移到[[天山|天山]]北麓的[[伊塞克湖|伊塞克湖]]一帶，即今日[[吉爾吉斯共和國|吉爾吉斯共和國]]之地。[[清高宗|清高宗]][[乾隆|乾隆]]二十二年（1757年）平定準噶爾蒙古後，吉爾吉斯人被[[清朝|清朝]]統治。後來清朝衰落，在1864年的[[北京條約|北京條約]]將[[伊犁河|伊犁河]]及天山一大部份割讓俄國，因為人隨地歸，吉爾吉斯人從此被俄罗斯帝国統治。一部分因為被准噶尔裹脅叛亂被[[乾隆帝|乾隆帝]]送交今[[黑龙江省|黑龙江省]][[富裕縣|富裕縣]]，又称[[富裕柯尔克孜人|富裕柯尔克孜人]]。

柯尔克孜人最早信仰原始宗教萨满教，早期生活在天山南部及中亚等地的柯尔克孜人西元10世纪前后就皈依伊斯兰教。

在吉尔吉斯斯坦共和国境内约361万柯尔克孜族人，与现今的在中国新疆的16万柯尔克孜人均信仰伊斯兰教。而生活在中国黑龙江的富裕柯尔克孜人则信仰萨满教，新疆[[额敏|额敏]]因長期與厄魯特生活因此信仰[[藏传佛教|藏传佛教]]。吉尔吉斯斯坦因[[浩罕汗国|浩罕汗国]]而伊斯兰化，南部吉尔吉斯更为伊斯兰化。

== 中国柯尔克孜人口分布 ==
[[现代汉语|现代汉语]]为表区分多对居于中国境内的这一民族常称“柯尔克孜族”，对居于中亚的这一群体称为“吉尔吉斯族”。

== 語言文字 ==
{{main|柯爾克孜語}}

柯爾克孜族有自己的[[语言|语言]]和文字，语言为[[柯尔克孜语|柯尔克孜语]]（[[吉尔吉斯语|吉尔吉斯语]]），屬于[[突厥語族|突厥語族]]钦察语支。歷史上，柯爾克孜族曾經使用過[[古突厥文|古突厥文]]与[[察合台文|察合台文]]，目前，吉爾吉斯斯坦則使用以[[西里爾字母|西里爾字母]]為基礎的拼音文字，而中國新疆的柯爾克孜族則使用[[阿拉伯字母|阿拉伯字母]]拼寫柯爾克孜語。

== 文化藝術 ==
柯爾克孜族有豐富的文化遺產，最著名的是英雄史詩《[[瑪納斯_(史詩)|瑪納斯]]》。

== 風俗習慣 ==
=== 民族服饰 ===
[[File:Kyrgyz_Manaschi,_Karakol.jpg|thumb]]

柯尔克孜族的女子多穿长连衣裙，上身穿黑色小马甲，有些地方的妇女穿小竖领的白色衬衫。妇女用头巾包头，外佩饰品。男子的高顶方形帽是一大特色，多用皮子或毡子做成，又一个卷沿，两侧有护耳。男子的上衣多是圆领，绣有花边。腰间束皮带，佩有小刀。

== 史籍描述 ==

=== 族属与容貌 ===
《唐书·回鹘传·下》附《黠戛斯》条中说：“（黠戛斯）人皆长大，白发、皙面、绿履，以黑发为不祥，黑瞳者必曰〔李〕陵苗裔也。”唐段成式的《酉阳杂俎》卷四中也说“坚昆部非狼种”，日本历史学家羽田亨据此推断，黠戛斯人在唐代仍不属于[[突厥人|突厥人种]]，而是属于[[雅利安人種|雅利安人]]种，以后随着时间的推移才逐渐[[突厥化|突厥化]]的。<ref>{{Cite book|title=《西域文化史》|last=羽田亨 著|first=耿世民 译|publisher=新疆人民出版社|year=1981年9月|isbn=11098·13|location=新疆乌鲁木齐|pages=第二章“西域的民族”第一页}}</ref>
