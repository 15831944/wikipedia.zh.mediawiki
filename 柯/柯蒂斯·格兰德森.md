'''柯蒂斯·古蘭德森'''（'''Curtis Granderson, Jr.'''，{{bd|1981年|3月16日|||Granderson, Curtis}}）出生于[[美国|美国]][[伊利诺伊州|伊利诺伊州]][[蓝岛_(伊利诺伊州)|蓝岛]]，是[[美国职棒大联盟|美国职棒大联盟]][[密爾瓦基釀酒人|密爾瓦基釀酒人]]的[[中外野手|中外野手]]。

== 早年生活 ==
=== 大学时期 ===
2000年格兰德森进入[[伊利诺伊大学芝加哥分校|伊利诺伊大学芝加哥分校]]，就读工商管理和广告专业。大一那年他击出7支[[全垒打|全垒打]]，并获得45次[[保送|保送]]，均为全队最高。他在大二继续着优异的表现，得分、全垒打、保送都领先于全队。大三时他被[[美国棒球|美国棒球]]杂志和[[今日美国|今日美国]]的体育周刊选入全美第二队。

=== 小联盟时期 ===
格兰德森在[[2002年美国职棒大联盟选秀|2002年大联盟选秀大会]]上被[[底特律老虎|底特律老虎]]在第三轮选中。他从1A的[[安尼安塔老虎|安尼安塔老虎]]开始小联盟生涯，第一个赛季他在52场比赛中打击率达到了3成44。{{by|2003年}}他升上高阶1A的[[莱克兰老虎|莱克兰老虎]]。{{by|2004年}}他在2A的[[伊利海狼|伊利海狼]]打出了他在小联盟最好的一个赛季，打击率为3成01，并有21支全垒打和93分打点。他入选了[[东方联盟|东方联盟]]全明星赛，并被[[美国棒球|美国棒球]]杂志选入2A全明星阵容。格兰德森在9月13日首次登上大联盟。

{{by|2005年}}格兰德森多数时间仍留在3A的[[托利多泥母鸡|托利多泥母鸡]]。季末利用大联盟名单扩编的机会再度登上大联盟。

== 大联盟生涯 ==
[[Image:grandersonclip.jpg|200px]]比赛时格兰德森在打击准备区]]
===2005年至2006年===
{{MLBy|2005年}}格兰德森在为数不多的大联盟出场机会中表现出色。9月15日击出一支[[场内全垒打|场内全垒打]]，9月18日单场击出5支安打，9月26日在对[[芝加哥白袜|芝加哥白袜]]的比赛中更是击出[[再见全垒打|再见全垒打]]。

{{MLBy|2006年}}春训格兰德森击败队友[[努克·洛根|努克·洛根]]（[[:en:Nook_Logan|Nook Logan]]），占据了老虎队主力[[中外野手|中外野手]]的位置。


===2007年===
尽管格兰德森没被放入[[2007年美国职棒大联盟全明星赛|2007年全明星赛]]候选名单中，但他获得了最多的手写票<ref>{{cite web |first=Tim |last=Kirby |url=http://detroit.tigers.mlb.com/news/article.jsp?ymd=20070701&content_id=2061505&vkey=news_det&fext=.jsp&c_id=det |title=Notes: Granderson lost in the crowd |accessdate=2008-09-01 |date=July 1, 2007 |publisher=[[MLB.com|MLB.com]] |deadurl=yes |archiveurl=https://web.archive.org/web/20070819133719/http://detroit.tigers.mlb.com/news/article.jsp?ymd=20070701&content_id=2061505&vkey=news_det&fext=.jsp&c_id=det |archivedate=2007年8月19日 }}</ref>。在全明星赛之后的一周他被选为美联单周最佳球员<ref>{{cite web |url=http://detroit.tigers.mlb.com/news/press_releases/press_release.jsp?ymd=20070716&content_id=2089596&vkey=pr_mlb&fext=.jsp&c_id=mlb |title=Detroit's Curtis Granderson named Bank of America Presents the American League Player of the Week |accessdate=2008-09-01 |date=July 16, 2007 |publisher=[[MLB.com|MLB.com]] |deadurl=yes |archiveurl=https://web.archive.org/web/20110615103244/http://detroit.tigers.mlb.com/news/press_releases/press_release.jsp?ymd=20070716&content_id=2089596&vkey=pr_mlb&fext=.jsp&c_id=mlb |archivedate=2011年6月15日 |df= }}</ref>。

9月9日 他完成了赛季第20次盗垒成功，从而成为了历史上第三位达到单赛季20支[[二垒安打|二垒安打]]、20支[[三垒安打|三垒安打]]、20支[[全垒打|全垒打]]和20次[[盗垒|盗垒]]成功的球员。21天后[[费城费城人|费城费城人]]的[[吉米·罗林斯|吉米·罗林斯]]也达到了这一纪录<ref>体育画报2007年9月24日第51页</ref>。第一位达到此成就的球员是{{MLBy|1911年}}[[芝加哥小熊|芝加哥小熊]]的[[弗兰克·舒尔特|弗兰克·舒尔特]]（[[:en:Frank_Schulte|Frank Schulte]]），第二位是1957年[[旧金山巨人|纽约巨人]]的[[威利·梅斯|威利·梅斯]]<ref>{{cite web | publisher = [[美联社|美联社]] | url = http://sports.espn.go.com/mlb/news/story?id=3011789 | title = Granderson's 20th steal puts Tigers center fielder in select company | work = [[ESPN|ESPN]] | date = September 9, 2007 | accessdate = September 9, 2007}}</ref>。

这个赛季格兰德森打击率为3成02，并有23支全垒打和26次盗垒成功。他击出23支三垒安打，成为自{{MLBy|1949年}}以来第一位击出23支以上三垒安打的球员<ref>{{cite web | publisher = baseball-reference.com | url = http://www.baseball-reference.com/leaders/3B_season.shtml | title = Single-Season Leaders & Records for Triples}}</ref>。他的23支三垒安打甚至比6支球队（[[芝加哥白袜|芝加哥白袜]]、[[辛辛那提红人|辛辛那提红人]]、[[洛杉矶安那罕天使|洛杉矶安那罕天使]]、[[奥克兰运动家|奥克兰运动家]]、[[西雅图水手|西雅图水手]]和[[圣路易红雀|圣路易红雀]]）全队击出的三垒安打都多。他之前一直被人诟病的高三振率<ref name="Beck">{{cite web |first=Jason |last=Beck |url=http://mlb.mlb.com/news/article.jsp?ymd=20080819&content_id=3333661&vkey=news_det&fext=.jsp&c_id=det |title=Granderson's two-strike approach better |accessdate=2008-09-01 |date=August 19, 2008 |publisher=[[MLB.com|MLB.com]]}}</ref>也降低了许多，{{MLBy|2006年}}时他以174次三振在美联领先，这个赛季降到了141次，排名美联第七<ref>{{cite web |first=Jason |last=Beck |url=http://www.mlb.com/news/article.jsp?ymd=20080219&content_id=2380694&vkey=spt2008news&fext=.jsp&c_id=mlb |title=Notes: Granderson learning from Sheff |accessdate=2008-09-01 |date=February 19, 2008 |publisher=[[MLB.com|MLB.com]] }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

===2008年至今===
{{MLBy|2008年}}2月4日，格兰德森和球队续签了一份为期五年总价值3025万美元的合同，球队拥有{{MLBy|2013年}}的选择权<ref>{{cite web |first=Jason |last=Beck |url=http://mlb.com/news/article.jsp?ymd=20080204&content_id=2363704&vkey=news_det&fext=.jsp&c_id=det |title=Tigers, Granderson agree to deal |accessdate=2008-09-01 |date=February 4, 2008 |publisher=[[MLB.com|MLB.com]] }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

3月23日在春训的一场比赛中，格兰德森被[[特拉维斯·布莱克利|特拉维斯·布莱克利]]（[[:en:Travis_Blackley|Travis Blackley]]）的快速球击中右手，导致他错过了开季的比赛<ref>{{cite web |author=Gary Bond & Danny Knobler |url=http://www.mlive.com/news/index.ssf/2008/04/tigers_granderson_to_play_with.html |title=Tigers' Granderson to play for Whitecaps Friday |accessdate=2008-09-01 |date=April 17, 2008 |publisher=MLive.com}}</ref><ref>{{cite web |first=Gary |last=Bond |url=http://www.mlive.com/tigers/index.ssf/2008/04/granderson_takes_baby_steps_to.html |title=Granderson takes baby steps toward Tigers return |accessdate=2008-09-01 |date=April 21, 2008 |publisher=The 大急流报}}</ref>。康复后回到大联盟的格兰德森继续着不错的表现，这个赛季他打击率2成80，并有13支三垒安打和22支全垒打。他在8月份就击出了6支三垒安打，其中还包括在对[[德州游骑兵|德州游骑兵]]的一场比赛中连续两局击出三垒安打<ref>{{cite web |url=http://www.iht.com/articles/ap/2008/08/19/sports/BBA-Tigers-Rangers.php |title=Sheffield, Granderson help Tigers rally for win |accessdate=2008-09-01 |date=August 19, 2008 |publisher=[[国际先驱论坛报|国际先驱论坛报]]}}</ref>。他的三振数也下降到111次，并获得了生涯新高的71次保送<ref name="Beck"/>。

{{MLBy|2009年}}格兰德森入选了[[2009年美国职棒大联盟全明星赛|全明星赛]]，这是他首次入选[[美国职棒大联盟全明星赛|全明星赛]]。比赛中他在8局上半击出一支三垒安打，并最终回到本垒得到制胜分。这个赛季他的打击率有所下降，三振数也增加到141次，但长打能力有所提高，30支[[全垒打|全垒打]]是生涯至今为止最高的一个赛季。2009年12月9日，老虎、[[紐約洋基|紐約洋基]]、[[亞利桑那響尾蛇|亞利桑那響尾蛇]]達成三方交易，格蘭德森被交易到洋基<ref>[http://newyork.yankees.mlb.com/news/article.jsp?ymd=20091208&content_id=7774692&vkey=news_nyy&fext=.jsp&c_id=nyy Granderson joins Yanks in three-way trade | yankees.com: News]</ref>。

{{MLBy|2013年}}古蘭德森在今年春訓的第一場比賽因觸身球打中手，進入傷兵名單，直到5月14日[[洋基隊|洋基隊]]在主場迎戰[[水手隊|水手隊]]的比賽才歸隊，該比賽[[洋基隊|洋基隊]]終場以4:3擊敗[[水手隊|水手隊]]，但是古蘭德森3打數無[[安打|安打]]，緊靠著一次[[四死球|四死球]][[保送|保送]]上壘，使的他本季第一場比賽表現得不盡理想。

2017年6月14日對[[芝加哥小熊隊|芝加哥小熊隊]]達成生涯第300轟，成為史上第143位、現役第10位生涯300轟的球員，同年8月19日被交易到[[洛杉磯道奇隊|洛杉磯道奇隊]]。


== 球场之外 ==
球场外，格兰德森是大联盟的国际大使，他曾到[[意大利|意大利]]、[[中华人民共和国|中国]]和[[南非|南非]]访问。为此大联盟的总裁[[巴德·塞利格|巴德·塞利格]]（[[:en:Bud_Selig|Bud Selig]]）还给他写了一封感谢信：“如今我们大联盟中有很多不错的年轻球员，但我认为过去的这一段时间没有人比你更适合代表我们国家。”<ref name="Curtis Granderson—A Humanitarian, Philanthropist, and Educator">{{cite web | url = http://bleacherreport.com/articles/128525-curtis-granderson-the-humanitarian-philanthropist-and-educator | title = Curtis Granderson—A Humanitarian, Philanthropist, and Educator}}</ref>

格兰德森还获得了2009年的马文·米勒年度最佳球员的荣誉，以表彰他在球场外所做的工作<ref>{{cite web | url = http://mlb.mlb.com/news/article.jsp?ymd=20091030&content_id=7582518&vkey=news_det&fext=.jsp&c_id=det | title = Granderson named Miller Award winner | author = Jason Beck | date = 2009-10-30 | accessdate = 2009-11-17 | publisher = [[MLB.com|MLB.com]]}}</ref>。
