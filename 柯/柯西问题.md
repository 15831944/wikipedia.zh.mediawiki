'''柯西问题'''（{{lang-en|Cauchy problem}}）在数学中是指，在一区域内的[[超曲面|超曲面]]上给定特定初始条件的情况下求[[偏微分方程|偏微分方程]]的解。柯西问题由[[初值问题|初值问题]]推广而来，与[[边值问题|边值问题]]相对。该问题以法国数学家[[奥古斯丁·路易·柯西|奥古斯丁·路易·柯西]]的名字命名。

假定偏微分方程定义在'''R'''<sup>''n''</sup>上，有一(''n''-1)维的[[光滑流形|光滑流形]]''S'' ⊂ '''R'''<sup>''n''</sup>（''S''称为[[柯西曲面|柯西曲面]]）。那么柯西问题是指求偏微分方程的解''u''，满足

:<math> \begin{align}
u(x) &= f_0(x) \qquad && \text{for all } x\in S; \\
\frac{\partial^k u(x)}{\partial n^k} &= f_k(x) \qquad && \text{for } k=1,\ldots,\kappa-1 \text{ and all } x\in S,
\end{align} </math>

其中<math>f_k</math>是曲面''S''上的给定函数（合称为该问题的柯西数据），''n''是''S''的[[法向量|法向量]]，κ则表示微分方程的阶数。

[[柯西－柯瓦列夫斯卡娅定理|柯西－柯瓦列夫斯卡娅定理]]表明柯西问题在某些特定条件下有唯一解，其中最重要的条件是柯西数据与偏微分方程的系数为[[实解析函数|实解析函数]]。
