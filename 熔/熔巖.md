《'''熔巖'''》（{{jpn|j='''マグマ'''}}）是[[日本|日本]][[音樂組合|音樂組合]][[B'z|B'z]]主唱[[稻葉浩志|稻葉浩志]]的SOLO出道[[錄音室專輯|專輯]]。[[1997年|1997年]][[1月29日|1月29日]]由[[VERMILLION_RECORDS|Rooms RECORDS]]於日本發行。

== 概要 ==
根據稻葉的說法表示，由於自B'z的16th單曲「{{lang|ja|[[ねがい|ねがい]]}}」起自身亦會參與[[編曲|編曲]]，因此同時自己也開始製作樂曲。本作即是在那些樂曲的庫存積攢到足以製作1張專輯時所發表的作品<ref name="be with!">{{Cite journal|language=ja|title=be with! |publisher=[[B'z_Party|B'z Party]]}}</ref>。

並陳述了日後亦會持續進行SOLO活動的原因，是由於在作為B'z製作等各種經驗中，會對於在B'z以外表達自己時會發生什麼感興趣。

在本專輯中，稻葉不僅作詞，還挑戰了[[作曲|作曲]]及[[口琴|口琴]]演奏等各式各樣的事物。

作為本專輯發售紀念，製作了限量500張的宣傳用[[黑膠唱片|黑膠唱片]]。唱片封面亦使用了與CD不同的寫真。此外，由於將本黑膠唱片送給了承蒙關照的工作人員等人，因此現在已成為非常難以得手的稀有物件（在2006年舉辦的B'z粉絲俱樂部活動「B'z Treasure Land」上，展示了此黑膠唱片）。

亦是有親密交情的演員[[勝新太郎|勝新太郎]]畢生都持續聆聽的專輯。

既無收錄單曲，亦無收錄[[商業搭配|商業搭配]]曲，卻在[[Oricon公信榜|Oricon]]首發銷量突破62萬張，獲得100.1萬張的百萬銷量紀錄。

此外，除了「{{lang|ja|冷血}}」「{{lang|ja|そのswitchを押せ}}」「{{lang|ja|Soul Station}}」「{{lang|ja|Chopsticks}}」「{{lang|ja|愛なき道}}」「{{lang|ja|Little Flower}}」以外，全部皆製作了有稻葉本人歌唱的MUSIC VIDEO<ref>雖然有如同「{{lang|ja|冷血}}」那樣無直接歌唱，而是作為形象片製作的情況，或是將正在錄音中的模樣作為影像使用等，亦有僅是製作了數秒鐘的影像剪輯，但仍已製作了全曲的MUSIC VIDEO。</ref>，在「{{link-ja|NO.}}」「{{link-ja|CD NEWS}}」等音樂節目上，以及1997年專輯發售當時的「[[COUNT_DOWN_TV|COUNT DOWN TV]]」專輯全曲介紹中，播放了全收錄曲的MUSIC VIDEO。

== 收錄曲 ==
# '''冷血''' (2:15)
#: 演奏時長2分15秒的短樂曲。在2004年SOLO巡迴演唱會「[[LIVE_2004_〜en〜|稻葉浩志 Inaba Koshi LIVE 2004 〜en〜]]」亦是用本曲點綴了開場。在MUSIC VIDEO中並無直接歌唱，但收錄了在佈滿多層印刷上黃色「CAUTION」的禁止進入膠帶中，稻葉本人身著黑色夾克佇立的模樣。
# '''唇（{{lang|ja|くちびる}}）'''(4:15)
#: 原曲始於B'z專輯『[[LOOSE|LOOSE]]』時<ref name="WHAT's IN? 1996">{{Cite journal|language=ja|title={{link-ja|WHAT's IN?}} |date=1996年12月號 |publisher={{link-ja|M-ON! Entertainment Inc.|エムオン・エンタテインメント}}}}</ref>。
# '''按下那個switch（{{lang|ja|そのswitchを押せ}}）'''(3:33)
#: 根據稻葉表示「{{link-en|鼓機Loop|Drum machine}}與{{link-ja|山木先生|山木秀夫}}的爵士鼓之間連結得很棒{{#tag:ref|{{lang|ja|「ループと山木さんのドラムとの絡みがすばらしい」}}|group="原文"}}」<ref name="be with!" />。
#: 在「[[羽_(稻葉浩志單曲)#「初回限定盤」附屬影像軟體|Koshi Inaba LIVE 2014 〜en-ball〜]]」上是首次於演唱會上演奏。
# '''波''' (4:46)
#: 在[[樣本唱片|DEMO]]階段是僅演奏了吉他，但最終已更改為民族調的節奏模式<ref>在「NO.」1000集紀念播出時，播放到「波」的MUSIC VIDEO之際，在介紹軼事的反射式字幕中透露。</ref>。
#: 亦製作了Full size的[[音樂錄影帶|MUSIC VIDEO]]，收錄了身著黑色西裝與紅色襯衫，戴著墨鏡的稻葉本人，一邊乘坐在由外國女性所駕駛的敞篷車內一邊歌唱的模樣，以及身著黑色皮草大衣，一邊靠在敞篷車上一邊歌唱的模樣等。在1997年的「{{link-ja|NO.}}」1000集紀念中，播放了此MUSIC VIDEO。
#: 經常在SOLO演唱會上演奏，亦有在「[[LIVE-GYM|B'z LIVE-GYM '98 "SURVIVE"]]」上披露。
# '''無法入眠是誰的錯（{{lang|ja|眠れないのは誰のせい}}）'''(4:02)
#: 根據稻葉表示「想用[[爵士|爵士]]吉他的八度音奏法來做連複段（Riff）{{#tag:ref|{{lang|ja|「ジャズ・ギターのオクターブ奏法をリフにしたい」}}|group="原文"}}」。此外，亦談及了「用海綿作為[[貝斯|貝斯]][[弱音器|弱音器]]的點子很有趣{{#tag:ref|{{lang|ja|「ベースをスポンジでミュートするアイデアが面白い」}}|group="原文"}}」<ref name="be with!" />。
# '''Soul Station''' (6:36)
#: 前奏由[[管風琴|管風琴]]獨奏揭開序幕。演奏時長6分34秒是在稻葉的樂曲中最長的一首。這是[[1995年|1995年]]時在[[巴黎|巴黎]]完成的曲子，據說製作時便已是現在的長度<ref name="be with!" />。
# '''arizona''' (5:00)
#: 稻葉本人將在[[亞利桑那州|亞利桑那州]]獨自旅行所獲得的體驗作為基礎<ref name="be with!" />。
# '''氣球（{{lang|ja|風船}}）'''(3:53)
#: 採用了簡單的鋼琴[[伴奏|伴奏]]與稻葉的低音主唱，並且能隨處聽見[[大提琴|大提琴]]及[[法國號|法國號]]。在「B'z LIVE-GYM Pleasure'97 "FIREBALL"」上披露時，稻葉自身用電子琴替代了鋼琴，在前奏、間奏等亦演奏了藍調口琴。在SOLO巡演上是於「[[Koshi_Inaba_LIVE_2016_〜enIII〜|Koshi Inaba LIVE 2016 〜enIII〜]]」的安可曲中被首次披露。
# '''乾脆來個颱風好了（{{lang|ja|台風でもくりゃいい}}）'''(3:09)
#: 歌詞中出現的伊織（{{lang|ja|イオリ}}），是稻葉從前飼養過的寵物[[貓|貓]]。MUSIC VIDEO收錄了在深夜中一邊窺視著房間裡的冰箱一邊歌唱的模樣。
# '''燃燒之人（{{lang|ja|灼熱の人}}）'''(3:57)
#: 前奏由[[電吉他|吉他]]{{link-ja|連複段|リフ}}（Guitar Riffs）揭開序幕，在間奏中稻葉的歌聲臨摹了吉他連複段。
# '''一無所有的城市（{{lang|ja|なにもないまち}}）'''(2:49)
#: 小調的[[巴薩諾瓦|巴薩諾瓦]]。在結尾使用了[[聲碼器|聲碼器]]。
# '''Chopsticks''' (2:15)
#: [[器樂|器樂]]曲。在4分7拍子這樣的[[節拍#复节拍（混合节拍）|複節拍]]中，樂曲前半僅由[[打擊樂器|打擊樂器]]及人的吆喝聲所構成。採用了猶如非洲音樂般的[[複節奏|複節奏]]，以許多打擊樂器所組成。自樂曲後半，開始刻入帶有失真的電吉他連複段，亦緊密結合了搖滾鼓。
# '''JEALOUS DOG''' (4:32)
#: 是一首單詞很多，樂譜亦相當精密的樂曲。曲中出現的[[狗|狗]]叫聲，是稻葉所飼養的狗「BUB」的聲音<ref name="be with!" />。
# '''無愛之路（{{lang|ja|愛なき道}}）'''(4:32)
#: 使用了一段稻葉出身地岡山的方言。
#: 在2004年的SOLO巡演上作為自新潟公演以後的最後一首安可曲，在「Koshi Inaba LIVE 2016 〜enIII〜」上作為每日替換的最後一首安可曲。
# '''Little Flower''' (6:12)
#: 結尾的[[滑音管吉他|滑音管吉他]]，據說是來自受到[[傑夫·貝克|Jeff Beck]]的樂曲影響<ref name="WHAT's IN? 1996" />。
