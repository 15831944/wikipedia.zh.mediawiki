'''红岩魂陈列馆'''，位于[[重庆市|重庆市]][[沙坪坝区|沙坪坝区]]政法三村63号，隶属[[重庆歌乐山烈士陵园|重庆歌乐山烈士陵园]]管理处。

==简介==
1963年，“重庆中美合作所集中营美蒋罪行展览馆”建成，原址是“[[中美合作所|中美合作所]]”阅兵台<ref name=lianxian/>。该馆由[[重庆中国三峡博物馆|重庆市博物馆]]筹建，1964年脱离重庆市博物馆独立<ref>{{Cite web |url = http://news.163.com/11/1027/04/7HBJKN1900014AED.html |title = 弘懿一甲子 奋进六十年 |author =  |date = 2011-10-27 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。随着该馆举办的《红岩魂》展览在全国影响扩大，1999年该馆更名为“红岩魂陈列馆”。展览名称也从《歌乐忠魂 世代英华——中美合作所暨军统集中营史实展览》改为《红岩魂——白公馆、渣滓洞革命烈士斗争事迹展览》<ref name=lianxian>{{Cite web |url =  http://www.hongyan.info/gb/news/news_detail.asp?id=2316|title = 红岩魂陈列馆 |author =  |date =2007-07-04   |publisher =红岩联线 |language =  |accessdate =  |quote =  }}</ref>。

2008年7月9日，历时一年多兴建的红岩魂陈列总馆举办开馆仪式，[[中共重庆市委|中共重庆市委]]书记[[薄熙来|薄熙来]]、市委副书记[[张轩|张轩]]等领导出席，同1500多名群众共同参观了新馆<ref>{{Cite web |url =http://news.sina.com.cn/c/2009-09-02/143718565119.shtml  |title = 重庆红岩：历史带给一座城市的永久记忆 |author =  |date =2009-09-02  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

该馆展览分为：（1）[[一一·二七大屠杀|11·27大屠杀]]半景画馆；（2）图片、文物展览<ref name=lianxian/>。
