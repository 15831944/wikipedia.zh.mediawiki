'''红黄蓝教育'''（{{lang-en|RYB Education}}；{{nyse|RYB}}）是[[中华人民共和国|中华人民共和国]]的一家私立儿童[[早期教育|早期教育]]企业，拥有红黄蓝亲子园、红黄蓝幼儿园、竹兜育儿三大教育品牌，与[[美国|美国]]多个儿童早期教育机构有战略合作关系，并在[[纽约证券交易所|纽约证券交易所]]上市<ref name="gy" />。公司在[[开曼群岛|开曼群岛]]注册，总部位于[[北京市|北京市]]<ref>{{cite web |url = http://usstock.jrj.com.cn/2017/11/24070623688473.shtml |language = zh |title = 红黄蓝：人间失格 |publisher=金融界 |date=2017-11-24 |accessdate= }}</ref>。

==发展历史==
红黄蓝教育由曹赤民、史燕来成立于1998年<ref name=gy/><ref name=chuangshi/>。1979年，曹赤民从[[桑植|桑植]]一中考入[[西安|西安]][[中国人民解放军第二炮兵工程学院|二炮工程学院]]。1993年转业后下海经商，起初到[[中关村|中关村]]从事推销电脑业务，后与朋友注册成立华一电子有限公司。1995年，曹赤民获得授权，将“翻斗乐”游乐项目引入[[北京|北京]]，并于1996年在北京成立[[中国内地|中国内地]]第一家“翻斗乐”大型室内游乐场。这给曹赤民带来了可观的经济效益<ref name=chuangshi/>。在引进翻斗乐项目时，曹赤民、史燕来已合作。2007年曹赤民获得了[[澳大利亚国立大学|澳大利亚国立大学]]和[[清华大学|清华大学]]的联合[[管理学|管理学]][[硕士|硕士]]学位<ref name=chuangshi>{{Cite web |url = http://finance.sina.com.cn/roll/2017-11-23/doc-ifypacti7526142.shtml |title =  起底扎针幼儿园红黄蓝:讲情怀的创始人和神秘大股东|author =  |date =2017-11-23  |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref name=wmzl/>。

史燕来畢業於[[北京大学|北京大学]]，是[[马克思主义|马克思主义]]学院2000级學生<ref name=bd>{{Cite web |url=http://www.jwb.com.cn/qy/201710/t20171031_4810249.html|title =120年120人，史燕来受邀与北大校友分享红黄蓝成长之路  |author =  |date =2017-10-31  |publisher = 今晚网 |language =  |accessdate =  |quote =  }}</ref>。和曹赤民一样曾获澳大利亚国立大学和清华大学的联合管理学硕士学位<ref name=chuangshi/>。1998年，史燕来做了母亲，同时开始创业，创办了中华人民共和国第一家亲子园——红黄蓝亲子园<ref name=wmzl/>。该园是曹赤民在“翻斗乐”游乐场辟出两间教室开办<ref name=chuangshi/>。当时中华人民共和国国内尚无“亲子园”概念，该园创办后，“亲子园”概念逐渐广为人知，到2010年代已成为中国早期教育的行业代名词。该园以“红黄蓝”为名，史燕来认为“红黄蓝”是[[三原色|三原色]]，所有颜色都是用“红黄蓝”调配出，“红黄蓝”预示着未来发展的无限可能，象征着充满亲情友情大家庭<ref name=wmzl>{{Cite web|url =http://www.mbxq.org/shownews.asp?Fld_ProductId=3759|title =红黄蓝教育机构总裁史燕来： 从0到1000，我们做了什么|author = |date =2017-11-13|publisher =中国民办教育协会学前教育|language = |accessdate = |quote = }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。史燕来表示，自己创办该园的初衷就是为让自己的儿子及更多孩子享受适合的早期教育<ref name=bd/>。

创业初期，红黄蓝亲子园邀请了[[北京师范大学|北京师范大学]]的梁志焱等儿童早期教育领域专家于1999年共同组建了教学研究中心，自主研发早期教育课程<ref name=gy/><ref name=wmzl/>。1999年底研发出首套完整的亲子课程，这也是中华人民共和国首套亲子课程体系。此后又研发出多个年龄段的具体课程<ref name=nge/>。2000年，红黄蓝取得了教育部门颁发的0-6岁亲子教育办学许可证<ref name=gy/>。在开业后的前三年里，红黄蓝亲子园实行直营，未放开[[特许加盟|特许加盟]]<ref name=wmzl/>。

2001年，红黄蓝38个股东联合创业，正式组建北京红黄蓝儿童教育科技发展有限公司，董事长为史燕来<ref name=bd/><ref name=wmzl/><ref>{{Cite web |url = http://qyxy.baic.gov.cn/xycx/queryCreditAction!qyxq_view.dhtml?reg_bus_ent_id=2520C7D69E234FA98D43A6D26008BF59&credit_ticket=CD9B1D737A23234BB58FC11701AB4BF4 |title =北京红黄蓝儿童教育科技发展有限公司  |author =  |date =  |publisher =北京市企业信用信息网  |language =  |accessdate = 2017-11-24 |quote =  }}</ref>。当时38人筹集了180万启动资金。其中史燕来是借债创业，将全家人的钱都借空了。同年，红黄蓝开始特许加盟业务。2001年到2005年是红黄蓝的“定位”阶段，红黄蓝开创了亲子园和幼儿园互动衔接的0～6岁学前一体化教育，改变了过去中国学前教育将0～3岁和3～6岁割裂开的状况。2003年创办了首家红黄蓝幼儿园——北京红黄蓝[[右安门|右安门]]幼儿园。2005年到2008年，红黄蓝对运营模式进行了研究，先在北京实现40多家亲子园和幼儿园和谐互动，资源互补共享。该模式随后在其他各城市复制<ref name=wmzl/>。2002年，红黄蓝获首批北京市名优特许经营品牌。2006年，建立客户关系管理系统，开启信息化之路。2007年，成立红黄蓝培训学校，创建幼儿教师培训体系。2008年，红黄蓝引进了[[美国|美国]]学乐英语课程（Scholastic Inc.），并获得战略投资<ref name=gy/>。2007年注册成立北京红黄蓝科技发展有限公司，位于[[北京市|北京市]][[丰台区|丰台区]]科学城航丰路8号，公司董事长为曹赤民，投资人为[[开曼群岛|开曼群岛]]顶辉公司<ref name=nue/><ref>{{Cite web |url = http://qyxy.baic.gov.cn/xycx/queryCreditAction!qyxq_view.dhtml?reg_bus_ent_id=A30FB8ADD996469594DDBD65C2D8386D&credit_ticket=BFD76E2A7ACBC9761D1633C7A04DC171 |title =北京红黄蓝科技发展有限公司  |author =  |date =  |publisher = 北京市企业信用信息网 |language =  |accessdate =2017-11-24  |quote =  }}</ref>。

2008年底，为在产品研发、人才引进、信息化建设和标准化管理等方面实现优化提升，红黄蓝暂停特许经营8个月，直到2009年7月恢复。2009年底，红黄蓝在全国各地建立16个总部直接投资运营的直营中心<ref name=wmzl/>。2009年，中国民办教育协会学前教育专业委员会向红黄蓝颁发“优秀民办幼儿园”称号。2010年，获中国连锁经营协会“AAA级信用企业”。2011年，红黄蓝创造第三品牌“竹兜育儿”，同年完成B轮融资。2013年，获中国连锁经营协会颁“中国特许奖”。2014年，与美国{{le|埃里克森儿童发展研究院|Erikson Institute}}达成中国区独家战略合作。2015年，与蜜芽合资成立跨境母婴O2O电商平台品牌“青田优品”。同年，与美国音乐教育品牌TMC签署中国独家战略合作。2016年举办国际学前教育高峰论坛。同年成立北京红杉优幼联盟<ref name=gy/>。

红黄蓝教育在中国儿童早期教育中的开创性工作也受到各级政府重视。早在2004年，作为中国儿童早期教育的领军企业，红黄蓝便承接了中国“[[十五计划|十五]]”教育科学规划项目，研发红黄蓝立体教育方案。2012年，红黄蓝成立ECD（婴幼儿早期发展项目部），探索与政府合作推进婴幼儿早期发展事业<ref name=gy/>。2016年以后，各地政府官员对红黄蓝旗下幼儿园视察增加<ref name=shicha/><ref>{{Cite web |url =http://www.sohu.com/a/61248822_249916  |title =厦门红黄蓝幼儿园喜迎各级领导视察与关怀  |author =  |date =  2016-03-01 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。其中红黄蓝海南生态国际幼儿园等具有创新理念的幼儿园尤受重视。2016年11月28日，[[中共中央政治局委员|中共中央政治局委员]]、[[中央政法委|中央政法委]]书记、[[中央社会治安综合治理委员会|中央社会治安综合治理委员会]]主任[[孟建柱|孟建柱]]、[[中共海南省委|中共海南省委]]书记[[罗保铭|罗保铭]]共同考察海南生态软件园，并视察了位于软件园内的红黄蓝海南生态国际幼儿园，称赞该幼儿园“环境好、理念新”<ref>{{Cite web |url =http://www.rybbaby.com/detail/ryb/4563.html   |title = 孟建柱视察红黄蓝海南国际幼儿园 |author =  |date = 2016-12-09 |publisher = 红黄蓝教育 |language =  |accessdate =  |quote =  }}</ref>。2017年5月24日，[[全国人大华侨委员会|全国人大华侨委员会]]副主任委员、原[[中共海南省委|中共海南省委]]书记罗保铭到红黄蓝海南生态国际幼儿园参观调研，赞扬了该幼儿园的办园条件和科学的教育理念<ref>{{Cite web |url = http://www.sohu.com/a/143483733_177818 |title = 全国人大华侨委员会副主任委员、原海南省委书记罗保铭一行视察海南红黄蓝国际幼儿园 |author =  |date = 2017-05-25  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。在2017年11月23日虐童事件曝出后，网络有传言称孟建柱是[[孟亮|孟亮]]的父亲，[[新加坡|新加坡]]《[[联合早报|联合早报]]》则称据知情人士介绍，孟建柱无儿无女，该传言并不属实<ref name=lhzaobao/>。

2010年代中期，红黄蓝教育已发展为中国学前教育知名品牌，宣称在营业规模、市场份额等方面均居中国学前教育行业首位<ref name=gy/>。红黄蓝搭建起0-6岁儿童教育生态圈，业务体系涵盖幼儿园、亲子园等园所服务以及幼教服务平台等领域，先后打造了红黄蓝亲子园、红黄蓝幼儿园、竹兜早教套装三个教育品牌<ref name=shangshi/>。其中，红黄蓝幼儿园和红黄蓝亲子园分别是中国幼儿教育和早期教育领域各自唯一的[[中国驰名商标|中国驰名商标]]<ref name=yizhi>{{Cite web |url = http://bj.news.163.com/17/1019/15/D14DPMRI043897TC.html |title =办好学前教育，红黄蓝一直在行动  |author =  |date = 2017-10-19 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。截至2017年6月30日，红黄蓝有853所亲子园、255所幼儿园，在建及筹备园所724家，遍布全国307座大中城市<ref name=shangshi/>，平均每周有近30万孩子及家庭走进红黄蓝，为数百万家庭提供学前教育服务及指导<ref name=gy/>。其中数十家幼儿园获评为“一级一类园”、“省市级示范园”、“优秀民办幼儿园”<ref name=yizhi/>。2017年9月27日，红黄蓝（NYSE：RYB）在[[美国|美国]][[纽约证券交易所|纽约证券交易所]]挂牌上市，成为中华人民共和国首家独立上市的学前教育企业<ref name=gy>{{Cite web |url = http://www.rybbaby.com/about |title =关于红黄蓝  |author =  |date =  |publisher = 红黄蓝 |language =  |accessdate =2017-11-24  |quote =  }}</ref>。9月27日上午9时30分在纽约证券交易所，红黄蓝教育机构董事长曹赤民、总裁史燕来和其他代表一起敲响了开盘钟<ref name=shangshi/>。最终发行价格每股18.5美元，高于IPO定价区间。最终发行量780万股计算，累计募集资金1.4亿美元。上市承销商是[[瑞士信贷|瑞士信贷]]集团、[[摩根士丹利|摩根士丹利]]和[[中金公司|中金公司]]<ref name=nge/>。

在2017年上市前，红黄蓝曾三度融资，曹赤民、史燕来早已让出第一大股东地位。2008年9月23日，从Hagerty融资1000万美元，后全部退出。2011年11月18日，纪源资本领投，银瑞达亚洲和和通集团跟投了2000万美元，后也全部退出<ref name=nge/>。战略投资人之一是真格基金创始人[[徐小平|徐小平]]。2011年徐小平在某论坛上表示，学前教育领域的“红黄蓝”就是外语培训领域的“[[新东方学校|新东方]]”。同年他对外称红黄蓝是他以个人身份成功投资的品牌。红黄蓝教育VIE架构中在国内成立的北京红黄蓝科技发展有限公司，徐小平曾任董事，但在2015年10月退出了董事名单。后来在一家[[自媒体|自媒体]]发布的采访中，[[孟亮|孟亮]]提及该电商业务称：“红黄蓝之前的亏损，很大程度上与此有关。叫停之后，营运利润就开始回升了。”<ref name=chuangshi/>
