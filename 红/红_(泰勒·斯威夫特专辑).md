《'''紅色'''<ref name="專輯介紹">{{cite web |url=http://www.umusic.com.tw/album.php?id=4608 |title=Red / 紅色 (普通盤) |accessdate=2016-11-13 |work= |publisher=台灣環球音樂 |date=2012-11-06 |deadurl=no |archiveurl=https://web.archive.org/web/20161113182346/http://www.umusic.com.tw/album.php?id=4608 |archivedate=2016-11-13 }}</ref><ref name="中国">{{cite web|url=http://www.amazon.cn/dp/B00ANJ2K8I?keywords=Red |title=泰勒史薇芙特:红(CD)-正版泰勒史薇芙特:红(CD)音乐在线销售 |publisher=Amazon.cn |date=2012-12-31 |accessdate=2013-10-05}}</ref>》（{{lang-en|''Red''}}）是美國創作型歌手[[泰勒·斯威夫特|泰勒絲]]的第4張錄音室專輯，由[[大機器唱片|大機器唱片]]於2012年10月22日發行。專輯從2010年開始製作，到2012年製作完成。專輯的名稱源於泰勒絲在構思專輯時經歷的“半陶醉的交往關係”，由於這種關係強烈而動蕩，因此她感到這是一種“紅色的情感”。專輯中有3首[[流行_(音樂類型)|流行]]，16首[[乡村音乐|鄉村]]歌曲，鄉村歌曲中也混雜了一點[[摇滚乐|搖滾]]，專輯曲風較為多樣。

《紅色》獲得了樂評人的好評，讓專輯入圍了[[葛萊美獎|葛萊美獎]]年度專輯和最佳鄉村專輯，單曲《[[我们再也回不去了|我们再也回不去了]]》入圍年度製作，《[[重新開始_(泰勒絲歌曲)|重新開始]]》入圍最佳鄉村歌曲。專輯以121萬份首周銷量空降美國[[公告牌二百强专辑榜|公告牌二百强专辑榜]]冠軍，是泰勒絲在美國的第3張冠軍專輯，此外也成为了当时首周销量第二高的由女歌手发布的专辑。在美国外，專輯亦獲得了巨大的商業成功，是泰勒絲在英國的首张冠軍專輯，在澳大利亞、加拿大、愛爾蘭和紐西蘭亦奪冠，並在全球幾乎所有主要唱片市場都進入了前10名。《紅色》是美國唱片業協會認證的4白金唱片，其全球销量已超過600萬。

專輯一共發行了7支單曲，首支單曲《[[絕對絕對分定了|絕對絕對分定了]]》和第3支單曲《[[我知道你是大麻煩|我知道你是大麻煩]]》進入十多個國家單曲榜前十名。而《[[重新開始_(泰勒絲歌曲)|重新開始]]》、《-{zh-tw:[[22_(歌曲)|二十二歲]];zh-cn:[[22_(歌曲)|22]]}-》、《[[都變了|都變了]]》等其它單曲都在美國獲得了白金及以上的認證。為宣傳專輯，泰勒絲启动了[[紅色巡迴演唱會|紅色巡迴演唱會]]，巡演在2014年6月12日結束，以1.5億美元的票房收入成為史上鄉村歌手舉辦的票房收入最高的巡演。

== 背景與發行 ==

2010年，泰勒絲開始創作新及製作的录音室專輯<ref name="RS1">{{cite web|url=http://www.rollingstone.com/music/news/taylor-swift-to-release-new-red-album-in-october-20120813#ixzz3JwllXJtP|title=Taylor Swift to Release New 2012年8月13日，泰勒絲面向72,500名觀眾開啟了直播網路聊天，回答粉絲各種問題<ref name="Live EW"/>。在聊天中，她透露了首支單曲的名稱“[[絕對絕對分定了|絕對絕對分定了]]”，并公佈了她第四張录音室專輯的名稱“紅色”及其發行日期<ref name="Live EW"/><ref name="newalbum"/>。泰勒絲亦揭示了專輯名稱背後的含義，她說專輯裡表達的都是自己這兩年經歷的“半陶醉的交往關係”，而這種關係強烈而動蕩，让她产生了一些感觉上是“紅色”的情感<ref name="Live EW">{{cite web|url=http://music-mix.ew.com/2012/08/14/taylor-swift-we-are-never-ever-getting-back-together-red/|title=Taylor Swift releases single 'We Are Never Ever Getting Back Together', announces new album 'Red'|author=Grady Smith|work=[[Entertainment_Weekly|Entertainment Weekly]]|date=2012-08-14|accessdate=2012-08-14|deadurl=no|archiveurl=https://www.webcitation.org/6Bq6eRlHB?url=http://music-mix.ew.com/2012/08/14/taylor-swift-we-are-never-ever-getting-back-together-red/|archivedate=2012-11-01}}</ref>。

2012年9月24日至10月15日期間，泰勒絲每週一都在《[[早安美國|早安美國]]》釋出一首專輯中的歌曲<ref name="Red GMA">{{cite web|url=http://abcnews.go.com/blogs/headlines/2012/09/good-morning-america-exclusive-taylor-swift-launches-red-album-release-with-4-week-song-preview-countdown-exclusively-on-abcs-good-morning-america/|title="Good Morning America" Exclusive: Taylor Swift Launches Red Album Release With 4-Week Song Preview Countdown – Exclusively on ABC’s "Good Morning America"|date=2013-09-21|accessdate=2013-02-18|publisher=American Broadcasting Company|deadurl=no|archiveurl=http://archive.wikiwix.com/cache/20130218052058/http://abcnews.go.com/blogs/headlines/2012/09/good-morning-america-exclusive-taylor-swift-launches-red-album-release-with-4-week-song-preview-countdown-exclusively-on-abcs-good-morning-america/|archivedate=2013-02-18}}</ref>。10月22日，專輯的普通盤和精裝盤正式在意大利、紐西蘭和美國等地發行，精裝盤比普通盤多出第2張CD，内附三首附送歌曲、兩首歌曲的取樣版本和一首歌曲的原音版<ref name="italy"/><ref name="nzl"/><ref name="usrelease"/>。為宣傳專輯，泰勒絲在專輯發行當日和次日在《早安美國》进行了现场表演。<ref name="Red GMA"/>2013年2月5日，專輯的卡拉OK版本在[[iTunes_Store|iTunes Store]]發售，卡拉OK版本收錄了普通盤全部曲目的伴奏<ref name="Karaoke">{{cite web |url=https://itunes.apple.com/us/album/taylor-swift-karaoke-red/id595751813 |title=Taylor Swift Karaoke: Red |accessdate=2013-02-19 |publisher=[[iTunes_Store|iTunes Store]]. === 乐评 ===
《红》在发行后获得了乐评的普遍好评。[[Metacritic|Metacritic]]综合23家乐评给予了专辑77分的评分<ref name="MC">{{cite web |url=http://www.metacritic.com/music/red/taylor-swift |title=Reviews for Red by Taylor Swift |work=Metacritic |publisher=[[CBS_Interactive|CBS Interactive]] |accessdate=2012-10-02 |deadurl=no |archiveurl=https://www.webcitation.org/6CqwpXQ5M?url=http://www.metacritic.com/music/red/taylor-swift |archivedate=2012-12-12 }}</ref>。《[[滾石_(雜誌)|滾石]]》编辑喬恩·杜蘭认为泰勒絲这张专辑受到了[[琼尼·米歇尔|琼尼·米歇尔]]和[[U2樂團|U2]]风格的影响，并写道“她的自我探索是在流行音乐史上最棒的故事之一。”<ref name="Rolling Stone" />《[[告示牌_(雜誌)|告示牌]]》则称《红》是泰勒絲“迄今最有趣的一张专辑”，并称专辑超越了泰勒絲原本乡村风格，转为多元化风格，让泰勒絲这位歌手达到巅峰位置<ref name="Track by track"/>。《[[AllMusic|AllMusic]]》的[[史蒂芬·托馬斯·艾爾維恩|史蒂芬·托馬斯·艾爾維恩]]则认为《红》是一张几乎没有乡村痕迹的流行专辑，虽然歌词仍然是关于爱情和社交焦虑的，听起来有点愚蠢，但它们丝毫没有减少组成音乐的“原有的流行甜蜜”<ref name="am"/>。而《[[纽约时报|纽约时报]]》的喬恩·卡拉曼尼卡则认为专辑中每一个纯流行的突破都有乡村风格来缓和<ref name="The New York Times">{{cite web|last=Caramanica|first=Jon|url=http://www.nytimes.com/2012/10/28/arts/music/no-more-kid-stuff-for-taylor-swift.html?_r=0|title=No More Kid Stuff for Taylor Swift|publisher=[[The_New_York_Times|The New York Times]]|date=2012-10-24|accessdate=2016-06-29|deadurl=no|archiveurl=https://web.archive.org/web/20140908124743/http://www.nytimes.com/2012/10/28/arts/music/no-more-kid-stuff-for-taylor-swift.html?_r=0|archivedate=2014-09-08}}</ref>。[[羅伯特·克里斯特高|羅伯特·克里斯特高]]在《[[MSN音乐|MSN音乐]]》中评论道，《红》是弱化版的磁场乐队1999年专辑《[[69首情歌|69首情歌]]》。克里斯多偏爱专辑中更加欢快的歌曲，但也赞赏了一些较为感伤的歌曲，例如《重新开始》和《留下來》<ref name="MSN Music"/>。

《[[偏锋杂志|偏锋杂志]]》的编辑約納森·基夫写道，《红》作为一张流行专辑，其整體一致性不佳，因此不是“真正的好”专辑，许多歌曲缺乏泰勒絲“通常的火花”。然而他也写道专辑的“亮点作品可称得上泰勒絲职业生涯最好的作品，它们听起来就像泰勒絲注定要成为的流行巨星一般。”<ref name="Slant"/>《[[影音俱樂部|影音俱樂部]]》的编辑迈克尔·蓋路奇认为，泰勒絲在这张专辑中的歌词“更加有深度且更加黑暗”，音乐听起来比她之前的作品更加雄心勃勃。他称赞《恩宠状态》和《回憶太清晰》两首歌曲是“吸引人的作品”。但他也总结道，《红》总体是一张“曲风繁复并有点缺少焦点的”流行专辑，并批评了专辑中二人合作的歌曲较为索然无味<ref name="A.V. Club"/>。《[[PopMatters|PopMatters]]》的编辑阿諾德·潘认为，泰勒絲在这张专辑中丢下了她原本以乡村流行为基础的风格而转为加强的现代摇滚风格，而这张过渡性专辑中的部分歌曲应该要删除<ref name="Pan">{{cite web|last=Pan|first=Arnold|url=http://www.popmatters.com/review/164856-taylor-swift-red/|work=[[PopMatters|PopMatters]]|title=Taylor Swift: Red|accessdate=2015-10-18|date=2012-10-30|deadurl=no|archiveurl=https://web.archive.org/web/20151003075653/http://www.popmatters.com/review/164856-taylor-swift-red/|archivedate=2015-10-03}}</ref>。
