== 创始人 ==
红芯研发公司为'''红芯时代（北京）科技有限公司'''。公司创始人兼CEO[[陈本峰|陈本峰]]，为中国[[千人计划|千人计划]]特聘专家<ref name="redcore" />，自赋头衔为“国际云安全联盟CSA专家委员、企业级[[HTML5|HTML5]]产业联盟主席、HTML5国际标准制定者之一”<ref name="sina" />。据称，他曾任[[微软|微软]]美国总部[[Internet_Explorer|Internet Explorer]]浏览器研发团队的成员，声称“成功发布了[[IE8|IE8]]、[[IE9|IE9]]、[[IE10|IE10]]”，并编写“从IE8一直沿用至今”的IE [[HTTP_404|404]]页面<ref>{{cite web |title=“千人计划”专家陈本峰：从“404无法访问”到“多屏合一” |url=http://www.sohu.com/a/132631107_661347|website=搜狐网|accessdate=2018-08-16 |date=2017-04-07|language=zh-hans|dead-url=no|archive-date=2018-08-16|archive-url=https://web.archive.org/web/20180816143338/http://www.sohu.com/a/132631107_661347}}</ref><ref>{{cite web |author1=李菡丹 |title=陈本峰 一行代码改变世界 |url=http://cpc.people.com.cn/n/2015/0123/c179979-26439342.html |website=中国共产党新闻网 |accessdate=2018-08-16 |date=2015-01-23|language=zh-hans|dead-url=no|archive-date=2018-08-16|archive-url=https://web.archive.org/web/20180816143540/http://cpc.people.com.cn/n/2015/0123/c179979-26439342.html}}</ref>。

红芯国产浏览器官方网站解释的自己软件的架构，与这一解释矛盾。<ref>{{cite web|title=红芯国产浏览器|url=https://browser.redcore.cn/|language=zh-hans}}</ref>

目前，在红芯浏览器的安装包中有两个浏览器插件可以看到[[源代码|源代码]]。不少互联网工程师通过阅读代码发现，红芯的代码存在过度注释、代码冗余、语言陈旧等问题。甚至还有工程师测试后发现，它在[[插件|插件]]的保密性上存在虚假宣传。一位从事数据相关业务的工程师表示，红芯浏览器插件的源代码中存在大量的[[注释_(计算机语言)|注释]]，显得注释过度，产品上线后大量注释都没有处理，另外其插件代码中，大量使用[[常量|常量]][[字符串|字符串]]，这使得之后可能的重构变得非常困难，以及不利于国际化等进一步开发。除了代码的被指显得不够专业，更值得注意的是，红芯浏览器插件在数据安全方面也值得商榷。一位程序员试验发现，红芯的密码管家插件中所存储的密码仅存储于本地并且没有加密，即用户可知道使用同一台计算机的其他人的密码。<ref>{{cite web|editor=张艳红|title=红芯调查：“山寨”谷歌竟称“自主研发”，创始人履历涉嫌造假！|url=http://toutiao.chinaso.com/cj/detail/20180817/1000200033137081534477361685184650_1.html|website=国搜头条|date=2018-08-17|access-date=2018-08-17|language=zh-hans|dead-url=no|archive-date=2018-08-17|archive-url=https://web.archive.org/web/20180817095452/http://toutiao.chinaso.com/cj/detail/20180817/1000200033137081534477361685184650_1.html}}</ref>

紅芯時代亦公開表示他們開發的瀏覽器已經被國務院、國資委、中車集團、國家電網、比亞迪、海信、中糧可口可樂等公私營機構採用為工作平台，但中車集團及海信已經否認此事。{{Fact|time=2018-12-10T03:13:11+00:00}}

公司創辦人陳本峰曾入讀香港科技大學，獲微軟錄用，在中港均曾獲獎，並入選中共「[[千人計劃|千人計劃]]」。「千人計劃」早前被美國批評是中共藉此轉移、竊取美國技術，多名入選學者均因此被美方拘捕或者拒發簽證。而大陸也多次爆出「千人計劃」學者學術造假等醜聞。有網民戲稱「千人計劃」變「老千計劃」。{{Fact|time=2018-12-10T03:13:11+00:00}}

至於另一名創辦人高婧則被指學歷造假，在官方網站上她自稱畢業於「美國哈佛大學」及「香港科技大學」，但被揭發她只是曾以交換生身分短暫留學。其後官方網站亦將高婧的學歷改為「畢業於香港科技大學」。<ref>{{cite web|title=关于红芯|url=https://www.redcore.cn/about/index.html|website=红芯官网|access-date=2019-03-08|language=zh-hans|dead-url=no|archive-date=2018-08-16|archive-url=http://web.archive.org/web/20180816073113/https://www.redcore.cn/about/index.html}}</ref>

== 历史 ==
2018年8月15日，红芯公司宣布完成2.5亿C轮系列融资<ref name="ifeng" />。投资方包括大型上市公司、政府客户，以及晨兴资本、达晨创投和IDG资本等。<ref name="caixin">{{cite web|url=http://companies.caixin.com/2018-08-16/101315451.html|title=“国产”浏览器红芯融资2.5亿元 谷歌内核遭曝光|access-date=2018-08-16|date=2018-08-16|website=财新网|editor1=覃敏|editor2=王影|language=zh-hans|dead-url=no|archive-date=2018-08-16|archive-url=https://web.archive.org/web/20180816144000/http://companies.caixin.com/2018-08-16/101315451.html}}</ref>在融资新闻稿中则称内核为“红芯Redcore”。

2018年8月16日，媒體曝光，融资2.5亿的“自主国产”红芯浏览器，其实是个套壳Chrome。<ref name="chrome_dabao" /><ref name="taoke" />后来红芯公司回应，称此项目没有拿国家基金。<ref>{{cite web|url=http://yuanchuang.caijing.com.cn/2018/0816/4501631.shtml |title=红芯陈本峰回应：没有拿国家基金 被比喻成“汉芯”冤枉|website=财经网|publisher=《财经》杂志|date=2018-08-16|access-date=2018-08-16|language=zh-hans}}</ref><ref>{{cite web |author1=DoNews |title=涉嫌造假的红芯浏览器是第二个汉芯吗？ |url=http://tech.caijing.com.cn/20180817/4502290.shtml |website=财经网 |accessdate=2018-08-17 |date=2018-08-17}}</ref><ref>{{cite news |title=钱江晚报：从汉芯到红芯，警惕造假误了创新 |url=http://www.xinhuanet.com/fortune/2018-08/17/c_129934782.htm |accessdate=2018-08-17 |work=钱江晚报 |publisher=新华网 |date=2018-08-17}}</ref>
