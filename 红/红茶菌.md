'''紅茶菌'''，又名'''紅茶菇'''、'''茶菇'''，因普遍在加糖的紅茶中培養而得名。紅茶菌不是單一的菌種，更不是菇類，經由分析其為[[酵母屬|酵母屬]]、[[醋桿菌屬|醋桿菌屬]]和[[乳桿菌屬|乳桿菌屬]]的共生物，其中以[[醋酸菌|醋酸菌]]的含量最多。這些菌利用茶湯中的糖分及紅茶的[[單寧酸|單寧酸]]作為養分產生[[酒精|酒精]]與[[醋酸|醋酸]]，顯微鏡下可看見桿狀的[[乳酸菌|乳酸菌]]、[[醋酸菌|醋酸菌]]和顆粒狀的[[酵母菌|酵母菌]]附著在一層薄膜上，若培養得宜會看見薄膜漸漸形成大朵菇狀的「茶菇」。紅茶菌[[飲料|飲料]]味道微酸，和酸奶一樣是一種活菌飲品。<ref>有趣的真菌（三版 2005）頁33，[[國立自然科學博物館|國立自然科學博物館]] 發行</ref>

==功效==
红茶菌是一种发酵茶饮料。主要功能是饮用后能够帮助人体排便正常，增强体抗力，帮助美颜。可治疗多种疾病，包括糖尿病、高血压、便秘、胃痛炎等。<ref>{{cite web |title=Kombucha红茶菌饮料 |url=http://www.sofunsd.com/kombucha-%E7%B4%85%E8%8C%B6%E8%8F%8C%E9%A3%B2%E6%96%99%EF%BC%8C%E4%BD%A0%E9%82%84%E6%B2%92%E5%96%9D%E9%81%8E%E5%B0%B1%E4%B8%8D%E7%AE%97%E8%B5%B0%E5%9C%A8%E6%99%82%E4%BB%A3%E5%B0%96%E7%AB%AF%EF%BD%9E/ |website=http://www.sofunsd.com/?variant=zh-cn |accessdate=2018-09-03}}</ref>

==做法==
靠一种名为“菌膜”的菌类，加入红茶，根据喜好加入糖过后，让它根据环境发酵，大约7至12天后即可饮用。除了红茶之外，其它茶类例如绿茶和普洱茶，都能用来发酵。成功发酵后，会增生新的菌膜，新菌膜能够分享给其他人，也能够用做新的茶菌。<ref>{{cite web |author1=陳　倢 |title=【科健籽】自製益力多？排毒紅茶菌 |url=https://hk.lifestyle.appledaily.com/lifestyle/wellbeing/daily/article/20160126/19466236 |website=https://hk.appledaily.com/ |publisher=黃仲兒 |accessdate=2018-09-03 |date=2016-01-26}}</ref>
