'''红砖美术馆'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[崔各庄乡|崔各庄乡]]何各庄，是展示当代艺术的私立美术馆。

==简介==
红砖美术馆位于北京市朝阳区东北部一号地国际艺术区，由收藏家闫士杰和曹梅夫妇创立<ref name=yisi>{{Cite web |url =http://dy.163.com/v2/article/detail/CI0DGEMM051496PC.html  |title =不看不知道！北京竟然有这么多有意思的艺术馆  |author =  |date =2017-04-14  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。2012年12月，红砖美术馆初步建成并试运营。2014年5月正式对外开放。红砖美术馆的馆舍及园林由[[北京大学|北京大学]]建筑研究中心教授[[董豫赣|董豫赣]]设计，是山水庭院的园林式美术馆，主要建筑材料为红砖<ref name=sohu>{{Cite web |url = http://www.sohu.com/a/158264784_558509 |title = 北京红砖美术馆 让你刷新对红砖材料的定义 |author =  |date = 2017-07-19  |publisher =  搜狐|language =  |accessdate =  |quote =  }}</ref>。园区占地面积大约二万平方米，展馆面积将近一万平方米<ref name=yisi/>。开馆后，红砖美术馆举办过各种大型展览，例如2017年3月举办的馆藏展“[[陈界仁|陈界仁]]：残响世界”<ref name=sohu/>。
