[[File:ChoeungEk-Darter-9.jpg|thumb]][[杀戮战场_(柬埔寨)|杀戮战场]]的[[万人坑|万人坑]]]]

'''红色高棉大屠杀'''，或称'''红色高棉种族灭绝'''、'''柬埔寨种族灭绝'''，指的是[[柬埔寨共产党|柬埔寨共产党]]总书记[[波尔布特|波尔布特]]领导的[[红色高棉|红色高棉]][[共产主义|共产主义]]政权在1975年至1979年期间对[[柬埔寨|柬埔寨]]进行[[种族灭绝|种族灭绝]]的事件，据估计造成了150万至300万柬埔寨人的死亡。{{sfn|Frey|2009|p=83}}红色高棉计划创建一个{{link-en|农业社会主义|Agrarian socialism}}的社会。当局将城市人口全部迁移到乡下。由于酷刑、大量处决、[[强迫劳动|强迫劳动]]和[[营养不良|营养不良]]，造成了将近当时25%人口（约200万人）的死亡。{{sfn|Etcheson|2005|p=119}}{{sfn|Heuveline|1998|pp=49-65}}这一大屠杀在[[越柬战争|越柬战争]]之后结束，全国境内发现了超过两百个[[万人坑|万人坑]]，这些地方后来被人称为“[[杀戮战场_(柬埔寨)|杀戮战场]]”。{{sfn|DeMello|2013|p=86}}

红色高棉的领导人宣称要“洗净平民”，因而开始了这场杀戮。{{sfn|Hannum|1989|pp=88-89}}2001年1月2日，[[柬埔寨政府|柬埔寨政府]]通过立法，对有限数量的前红色高棉领导人进行审判。审判开始于2009年2月17日。{{sfn|Mendes|2011|p=13}}2014年8月7日，柬埔寨共产党原副书记[[农谢|农谢]]和国家主席团原主席[[乔森潘|乔森潘]]因为[[反人类罪|反人类罪]]，被判处[[终身监禁|终身监禁]]。

== 强制迁移 ==
早在1973年，波尔布特与农谢等就决定在解放金边后把全城300万人口疏散到农村，以解决粮食问题，并观察美国或越南是否会插手柬埔寨。1975年4月17日，波尔布特藉口美軍即將[[空襲|空襲]]金邊，将首都居民疏散至鄉下，並以三日後將可以返回為由，要求居民不必帶任何財產。所有居民被迫紧急撤离，部分不愿意的人被军队[[槍殺|槍殺]]，一些没有能力离开的人如[[残疾人士|残疾人士]]被遗弃。三天时间内金边[[人口|人口]]由原有300万人，变成几万人，撤离过程中造成大量無辜百姓伤亡。

== 政治清算 ==
红色高棉的政治清算和镇压的对象主要是前[[龍诺|龍诺]]政权的军政人员，包括一般士兵、警察和公务员，也包括[[朗诺|朗诺]]政变前的[[王室|王室]]成员。处决方式一般是用[[卡车|卡车]]将大量此类人员运至某个地点，用[[棍棒|棍棒]]、[[斧頭|斧頭]]、[[鋤頭|鋤頭]]、[[榔頭|榔頭]]、[[十字镐|十字镐]]、[[圓鍬|圓鍬]]、[[鐮刀|鐮刀]]、[[弯刀|弯刀]]等各種[[農具|農具]]打死或是直接[[枪决|枪决]]。

== 强制劳动 ==
幸存的城市遣散的人员往往和农民一起被迫从事修筑水渠、农田和道路的工作。由于经济状况的恶化，粮食和生活物资缺乏保障，大量的人口在这种强制劳动下死亡。

== 内部清洗 ==
红色高棉对自身组织的“纯洁”追求近乎偏执，波尔布特喜欢用[[细菌|细菌]]来形容党内的异己思想，“它们”无处不在，所以党的眼睛必须时刻睁开。红色高棉从一建国开始就以[[肃清|肃清]][[亲越|亲越]]分子（高棉身越南心）、蘇聯[[克格勃|克格勃]]（KGB）间谍、美国[[中央情报局|中央情报局]]特务和新混入党内的异己分子为藉口开始了内部清洗。在1975年10月宣布的民族阵线的13个领导人中，有5个在1977年的清洗中被处决，包括内政部长胡荣、两任商务部长、新闻和宣传部长、国家主席团第一副主席等等。各大区的党政军领导人被处决的更多。

1976年9月20日首先开始对东北大区的清洗。东北大区书记奈沙朗（Nay Saran，别名Yan）、盖敏（Kev Meas）、盖莫尼（Kev Mony）相继被捕。这些人都是原印度支那共产党成员。

1976年底到1977年初，农业部长农笋（Noun Suon）、商业部长贵敦（Koy Thun）、建筑部长Thuch Pourn等先后被捕。1977年3月，福财（Phoukh Cai）被捕。1977年4月文化与新闻部长符宁被捕。

由於1977年底在柬越边界冲突中失败，1978年1月底到2月间，由宋成负责对东部大区展开行清洗，400名干部被捕。1978年5月西南区领导人塔莫负责清洗东部大区，东部大区书记索平开枪自杀，数千东部大区的部队逃入森林，越界逃入越南。1978年7月就处决了万余人。而在金边郊外建立的[[S-21集中营|S-21集中营]]，主要用来进行内部清洗。

1978年秋，主管经济的副总理温威被清洗处决。

== S-21集中营 ==
{{main|S-21集中营}}
当上百万柬埔寨人在集体农庄裡慢慢走向死亡，另外一些人和他们的家属则被贴上[[政治犯|政治犯]]的标签，在红色高棉的审查中心里面临更为直接的恐怖。所有这些审查中心中，最著名的是S-21集中营，这是金边郊外的一栋砖石结构的法式建筑，以前是一所中学。

S-21集中营全称第21号保安监狱。1975年至1979年间，据估计有14,000至15,000人被囚禁在S-21集中营（部分人相信总数超过20,000人）。该集中营的犯人从柬埔寨全国选送而来，前期的犯人主要是朗诺政权时期的政府官员、军人以及学者、[[医生|医生]]、[[教师|教师]]、[[僧侣|僧侣]]、[[藝人|藝人]]等，后期的犯人主要是红色高棉政权的[[党员|党员]]、[[士兵|士兵]]甚至一些高级官员，如外务部副部长[[沃维|沃维]]、新闻部长符宁等，囚禁者中仅有7人幸免于难。

大部分进入S-21的人实际上是无罪的。S-21的运作模式是，首先抓住一批“叛徒”，然后用[[酷刑|酷刑]]逼迫他们承认他们的罪行。因为不认罪是不被接受的，即使是最忠诚的红色高棉黨員也会最终承认他们为[[美国|美国]][[中情局|中情局]]做间谍、当[[越南共產黨|越南共產黨]]的走狗、暗中反对党中央，甚至是[[非礼|非礼]]幼女。接下来，他们被要求供出同党的名字，然后犯人们以及他们的家人会被带往琼邑克灭绝中心用各種[[鋤頭|鋤頭]]、[[斧頭|斧頭]]、[[鐮刀|鐮刀]]等各種農具或其他工具杀害。而被招认的同党又有了新的罪名，于是再被带进来，重复这一过程。<ref>{{cite web|url = http://news.ifeng.com/history/1/jishi/200902/0219_2663_1023042.shtml|title =审判红色高棉：康克由和他的集中营 |date=2009年2月19日|author =凤凰资讯 |publisher =  |language =zh-hans}}</ref>

== 否认==
{{main|否认柬埔寨种族灭绝}}
在波特布特于1998年4月15日死亡之前的几个月，{{sfn|Chan|2004|p=256}}他接受了Nate Thayer的采访。在采访中，他表示他有良知，并且否认对种族灭绝负责。波尔布特宣称他“来开展斗争，而不是杀人”。据亚历克斯·阿尔瓦雷斯（Alex Alvarez），波尔波特“把自己描绘成一个被误解和被不公正地诽谤的人物”。{{sfn|Alvarez|2001|p=56}}2013年，柬埔寨首相[[洪森|洪森]]通过立法，禁止否定红色高棉所犯的柬埔寨种族灭绝和其他战争罪行。尽管柬埔寨救国党副主席、反对派领导人[[金速卡|金速卡]]表示反对，立法仍获通过。金速卡表示，[[吐斯廉屠殺博物館|吐斯廉屠杀博物馆]]的展品是虚构的，而且这些文物是在1979年入侵后被越南人伪造的。柬埔寨救国党声称金速卡的评论被断章取义了。{{sfn|Buncombe|2013}}
