'''红头潜鸭'''（学名：''Aythya ferina''），又名红头鸭、矶凫、矶雁。

== 生态环境 ==
红头潜鸭是一种内陆性[[候鸟|候鸟]]，很难在沿海滩涂见到，常栖息于芦苇丛生视野不开阔的湖泊，因为这样的水面通常隐蔽条件好，适合胆小的红头潜鸭。过了[[繁殖季节|繁殖季节]]，红头潜鸭会聚集成群，并与潜鸭属的其他物种如凤头潜鸭混群，活动于开阔水面。

== 分布地域 ==
繁殖于[[西伯利亚|西伯利亚]]南部、[[蒙古|蒙古]]、[[欧洲|欧洲]]中部以及[[英国|英伦三岛]]，在[[中国|中国]]，他们繁殖于[[新疆|新疆]][[天山山脉|天山山脉]]一带，越冬于[[日本|日本]]、[[印支半岛|印支半岛]]、[[缅甸|缅甸]]、[[南亚次大陆|南亚次大陆]]、[[伊朗|伊朗]]、[[阿富汗|阿富汗]]、[[欧洲|欧洲]]南部[[非洲|非洲]]北部等地区，在中国他们的越冬地在[[长江流域|长江流域]]，向南一直分布到[[福建|福建]]、[[广东|广东]]、[[香港|香港]]沿海及[[台湾|台湾]]。

== 特征 ==
红头潜鸭属于体形较小的鸭子，与绿头鸭相比，体长尽是后者的2/3左右，雄性红头潜鸭的头颈部栗红色，上背、胸部和尾部为地道的黑色，而下背和两肩为污白色，并有幼细的黑色横纹，喙灰蓝色。雄性红头潜鸭从头到尾颜色由蓝到红到黑到白再到黑，变化鲜明，颇为抢眼。

雌性红头潜鸭是雄性的哑色版本，颜色分布与雄鸟近似，但以棕色调为主，不似雄鸟那般色彩鲜明。

== 食物 ==
红头潜鸭有很好的潜水技能，他们的食谱也从水草到小鱼小虾分布甚广，据研究红头潜鸭爱吃[[马来眼子菜|马来眼子菜]]，少食谷粒；动物性食物有[[软体动物|软体动物]]、[[鱼|鱼]]、[[蛙|蛙]]等。
