'''红岩革命纪念馆'''，位于[[重庆市|重庆市]][[渝中区|渝中区]]红岩村52号，隶属[[重庆红岩联线文化发展管理中心|重庆红岩联线文化发展管理中心]]（重庆红岩革命历史博物馆）。

==简介==
早在1951年至1955年间，[[重庆中国三峡博物馆|西南博物院]]便筹设“五馆一所”即历史馆、自然馆、革命馆、美术工艺馆、民族馆以及独立的研究所。1954年底，西南大区撤销。1955年6月西南博物院更名为[[重庆中国三峡博物馆|重庆市博物馆]]。1955年，重庆市博物馆在[[八路军重庆办事处旧址|八路军重庆办事处旧址]]筹建红岩革命纪念馆。历经3年筹建和重庆市博物馆代管，1958年5月1日红岩革命纪念馆正式对外开放。该馆的建成开放使原西南博物院规划的“五馆”中的“革命馆”变为现实。馆名“红岩革命纪念馆”是1959年由[[董必武|董必武]]题写。1964年，红岩革命纪念馆脱离重庆市博物馆独立，成为重庆市独立的文博单位<ref>{{Cite web |url = http://news.163.com/11/1027/04/7HBJKN1900014AED.html |title = 弘懿一甲子 奋进六十年 |author =  |date = 2011-10-27 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref><ref name=sohu/>。

1999年3月，红岩革命纪念馆新馆动工建设<ref name=hylxgu/><ref name=sohu>{{Cite web |url = http://www.sohu.com/a/140371536_747538 |title = 红岩革命纪念馆 |author =  |date =  2017-05-13|publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。2000年，该工程被[[重庆市人民政府|重庆市人民政府]]定为为民办实事十大重点建筑项目之一，被重庆市质监总站评为优良工程<ref name=hylxgu>{{Cite web |url = http://www.hongyan.info/gb/news/news_detail.asp?id=2327 |title = 红岩革命纪念馆新陈列馆 |author =  |date =  |publisher = 红岩联线 |language =  |accessdate = 2017-07-17 |quote =  }}</ref>。2001年6月29日，红岩革命纪念馆新馆开馆暨《千秋红岩——中共中央南方局历史陈列》开展<ref>{{Cite web |url = http://news.eastday.com/epublish/gb/paper5/20010630/class000500006/hwz424534.htm |title = 神州大地：红岩革命纪念馆新馆开馆 |author =  |date = 2001-06-30 |publisher = 东方网 |language =  |accessdate =  |quote =  }}</ref>。新馆占地面积1780平方米，建筑面积5300平方米，展厅面积4900平方米<ref name=sohu/><ref name=hylxgu/>。是半山坡上的一座倒梯形建筑，外墙采用一万多块红色[[花岗岩|花岗岩]]干挂而成，整栋建筑自下而上层层向外向上倾斜，象征着以[[周恩来|周恩来]]为首的[[中国共产党|中国共产党]]人在此培育的红岩精神坚如磐石<ref name=hylxgu/>。
