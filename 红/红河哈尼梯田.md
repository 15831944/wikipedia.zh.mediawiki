'''红河哈尼梯田'''是在[[云南省|云南省]][[红河哈尼族彝族自治州|红河哈尼族彝族自治州]]的[[梯田|梯田]]。云南多山，亦多梯田，[[哀牢山|哀牢山]]哈尼梯田为云南梯田的代表作，被誉为“中国最美的山岭雕刻”。

2013年6月22日，在柬埔寨[[金边|金边]]举行的第37届世界遗产大会上，哈尼梯田通过投票被列入联合国教科文组织《[[世界遗产名录|世界遗产名录]]》，成为[[世界文化遗产|世界文化遗产]]。哈尼梯田也是继[[新疆天山|新疆天山]]成功入选[[世界自然遗产|世界自然遗产]]后，2013年中国成功“入遗”的第二个景点。<ref name="列入">{{cite web |url=http://www.zaobao.com/news/china/story20130623-219483 |language= |title=云南哈尼梯田列入世界遗产名录 |publisher=联合早报 |date=2013年6月23日 |accessdate= }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 简介 ==
哈尼梯田至今有1200多年历史，规模宏大，分布于云南南部[[红河哈尼族彝族自治州|红河州]][[元阳县|元阳县]]、[[红河县|红河县]]、[[金平苗族瑶族傣族自治县|金平縣]]、[[绿春县|绿春县]]四县，总面积约100万亩，其中[[元阳县|元阳县]]是哈尼梯田的核心区，当地的梯田修筑在[[山坡|山坡]]上，最高可达3000级，景观壮丽。<ref name="列入"/>

== 登录基准 ==
红河哈尼梯田因為符合以下標準，而被登錄為世界文化遺產：
*（iii）獨一無二或至少是非常特別地代表了一種文化傳統或是一種現存或已經滅絕的文明。
*（v）可作为传统的人类居住地或使用地的杰出范例，代表一种（或几种）文化，尤其在不可逆转之变化的影响下变得易于损坏。

此次申报的遗产区面积为16,603公顷，缓冲区面积29,501公顷，包括了最具代表性的集中连片分布的水稻梯田及其所依存的水源林、灌溉系统、民族村寨。<ref>[http://news.xinhuanet.com/2013-06/22/c_116249670.htm 哈尼梯田申遗成功]，新华网，2013-06-22</ref>
