'''複都制'''，是指一个[[国家|国家]]设置两个以上的[[首都|首都]]的制度。拥有广阔的领土的国家很多采用複都制，拥有两个首都的常被称为「'''两都制'''」或「'''两京制'''」。（相对的，只设一个都城的叫做「[[单都制|单都制]]」）。另外，也包括把首都功能分别发放到多个城市的制度。关于现代意义上的複都制，请参考[[首都#特殊情況|首都#特殊情況]]。

在[[中央集权|中央集权]]国家的複都制，皇帝常驻的[[京|京]]或都常被称为「上京」、「上都」、「皇都」、「京师」、「正都」等，其他的都叫「陪都」。但是，「陪都」的概念在[[西方文化|西方文化]]中并不存在，无对应的西方词汇，此为[[东方|东方]]特有的概念（[[陪都制|陪都制]]）。[[日本历史|日本历史]]上所说的難波遷都等，正确的解释是[[天皇|天皇]]的陪都巡守或是皇都升格，与迁都不同<ref>744年（天平16年）、聖武天皇が恭仁京を離れて難波に行幸し、一時難波を皇都と定めたが、これも陪都から皇都への昇格で都を移したのではなかった。翌年の平城京への還幸によって再び陪都に戻った。法制史学者の瀧川政次郎は、「従来の日本史では、遷都ということが多すぎる」と指摘している。（『京制並に都城制の研究』）</ref>。

当皇帝在[[行在|行在]]而不在正都时，国政由其他人代理，在权限限制的基础上由[[太子|太子]]代理被称为「太子[[监国|监国]]制」，由信赖的大臣代理被称为「[[留守|留守]]官制」。

== 罗马帝国的複都制 ==
公元3世紀後半叶，[[罗马帝国|罗马帝国]]由[[正帝|正帝]]和[[副帝|副帝]]分治，[[戴克里先|戴克里先]]皇帝的時代，帝国分为東西两部分，两人为正帝、两人为副帝的[[四帝共治制|四帝共治制]]因此出现。这四个首都是[[尼科美迪亚|尼科美迪亚]]（[[伊兹密特|伊兹密特]]）、[[塞尔曼|塞尔曼]]（{{lang|en|Sirmium}}）、[[梅蒂奥拉努|梅蒂奥拉努]]（{{lang|en|Mediolanum}}，今[[米兰|米兰]]）、[[奥古斯塔·温德利科伦|奥古斯塔·温德利科伦]]（{{lang|en|Augusta Vindelicorum}}），[[罗马|罗马]]因此失去了它的首都機能，只是名义上的帝国的首都。此後，西方正帝[[君士坦丁一世_(罗马帝国)|君士坦丁一世]]将帝国統一，在東都建立新的首都[[君士坦丁堡|君士坦丁堡]]。

395年，東帝[[狄奧多西一世|狄奧多西一世]]（西帝为形式上的次子[[霍诺里乌斯|霍诺里乌斯]]）死後，長子[[阿卡狄奧斯|阿卡狄奧斯]]为東帝，他将都城设置在东方的君士坦丁堡与西方的梅蒂奥拉努（後迁都至[[拉文納|拉文納]]）。公元4世纪，罗马帝国东西分治的现象已经较为明显，当时常使用「西边的皇帝」、「西边的领土」这样的称呼，而不使用「西罗马帝国」。

== 中国的複都制 ==
{{seealso|陪都}}
中國由[[秦朝|秦朝]]至[[唐朝|唐朝]]在[[中原|中原]]地方建立的大一統的皇朝近一千二百多年間，[[長安|長安]]與[[洛陽|洛陽]]長期交互成為首都，洛陽更隨著朝代更替，多次成為首都或東都，直至[[北宋|北宋]]滅亡為止。[[南宋|南宋]]定[[杭州|杭州]]為[[行在|行在]]，[[建康|建康]]改為[[留都|留都]]。

[[明朝|明朝]]在[[永樂帝|永樂帝]]后，將[[順天府|順天府]]（[[北京|北京]]）及[[应天府_(明朝)|應天府]]（[[南京|南京]]）皆立為首都，其中南京在[[明太祖|明太祖]]時期為[[京師|京師]]，[[永樂遷都|永樂遷都]]後改以北京為京師，但南京仍做為留都，維持首都的地位。

== 現代的複都制 ==
現代符合複都制的國家主要有[[南非|南非]] (行政首都[[比勒陀利亞|比勒陀利亞]]、立法首都[[開普敦|開普敦]]、司法首都[[布隆方丹|布隆方丹]])、[[馬來西亞|馬來西亞]](國會及皇宮所在地[[吉隆坡|吉隆坡]]、行政中心及司法機構所在地[[布城|布城]])等。
