'''晋冀鲁豫边区'''，是[[中国抗日战争|中国抗日战争]]中1941年在河北[[邯郸|邯郸]]成立的[[中國共產黨|中國共產黨]][[根据地|根据地]]。至[[第二次国共内战|第二次国共内战]]期间，演变为[[晋冀鲁豫解放区|晋冀鲁豫解放区]]。

== 历史 ==

[[中共晋冀鲁豫中央局|中共晋冀鲁豫中央局]]和[[晋冀鲁豫军区|晋冀鲁豫军区]]，与抗战时期成立的晋冀鲁豫边区政府同驻[[河北省|河北省]][[涉县|涉县]]，后迁至[[邯郸市|邯郸市]]。包括[[山西省|山西省]]、河北省、[[山东省|山东省]]、[[河南省|河南省]]各一部，面积为十五万平方公里，177个县，2500万人口。首府在河北省[[武安市|武安市]][[冶陶镇|冶陶镇]]。边区政府主席为[[杨秀峰|杨秀峰]]，副主席为[[薄一波|薄一波]]、[[戎伍胜|戎伍胜]]，参议会议长为[[申伯纯|申伯纯]]。

1929年，[[楊虎城|楊虎城]]投效[[蔣介石|蔣介石]]，在蔣唐戰爭中擊敗[[唐生智|唐生智]]。1930年[[中原大戰|中原大戰]]之後西歸升格為[[國民革命軍|國民革命軍]]第17路軍，6萬多人，被稱為「小西北軍」。原[[西北軍|西北軍]]是[[宋哲元|宋哲元]]、[[馮玉祥|馮玉祥]]。

1936年於[[西安事變|西安事變]]之後縮編為國民革命軍第38軍，由[[孫蔚如|孫蔚如]]率領。抗日戰爭期間國民革命軍第38軍擴編為國民革命軍第31軍團，由孫蔚如任軍團長；下轄38軍（軍長[[趙壽山|趙壽山]]）和96軍（軍長[[李興中|李興中]]），在[[中條山|中條山]]等地與[[日軍|日軍]]作戰。1946年，[[孔从洲|孔从洲]]率國民革命軍第38軍投奔晉冀魯豫軍區，被改編為西北民主聯軍38軍，歸陳謝兵團指揮，成為[[中國人民解放軍|中國人民解放軍]]的一部分。

在军事上，划分为晋区[[晋冀豫军区|晋冀豫军区]]和鲁区[[冀鲁豫军区|冀鲁豫军区]]两大块。晋冀豫军区（司令员[[倪志亮|倪志亮]]，政委[[黄镇|黄镇]]）由八路军[[129师|129师]]建立于1938年4月，包括[[太行军区|太行军区]]、[[太岳军区|太岳军区]]和[[冀南军区|冀南军区]]三个战略区。冀鲁豫军区（司令员[[黄克诚|黄克诚]]，政委[[崔田民|崔田民]]）由八路军[[115师|115师]]建立于1940年4月，包括豫北、直南和鲁西南地区。

== 1945年后 ==

1945年8月20日，在抗日战争胜利前的大反攻中，正式成立中共晋冀鲁豫中央局（书记[[邓小平|邓小平]]）和晋冀鲁豫军区（司令员[[刘伯承|刘伯承]]），下辖太行、太岳、冀南和冀鲁豫四块根据地。1947年，刘邓率晋冀鲁豫区主力部队南下[[大别山|大别山]]后，[[徐向前|徐向前]]主持晋冀鲁豫区工作。1948年，晋冀鲁豫军区与晋察冀军区合并，组成[[华北军区|华北军区]]。

== 晋冀鲁豫边区政府 ==
1948年6月12日，晋冀鲁豫边区政府与[[晋察冀边区行政委员会|晋察冀边区行政委员会]]正式[[合署办公|合署办公]]，改称[[华北联合行政委员会|华北联合行政委员会]]，但该委员会未正式办公。9月26日成立[[华北人民政府|华北人民政府]]。27日，正式撤消晋冀鲁豫边区政府及晋察冀边区行政委员会。
