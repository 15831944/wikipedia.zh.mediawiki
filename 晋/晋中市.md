'''晋中市'''是[[中华人民共和国|中华人民共和国]][[山西省|山西省]]下辖的[[地级市|地级市]]，位于山西省中部，[[太行山|太行山]]西，[[汾河|汾河]]以东，是山西省的铁路、公路枢纽。毗邻省会[[太原市|太原市]]，市政府驻[[榆次区|榆次区]]。山西大学城所在地。

== 历史沿革 ==
[[春秋时期|春秋]]时期，境内设置祁、邬、涂水、马首4县，属[[晋国|晋国]]；东部有白狄族[[肥国|肥国]]。[[战国|战国]]分属[[韩国_(战国)|韩]]、[[赵国|赵]]、[[魏国_(战国)|魏]]三国。[[秦代|秦代]]属[[太原郡|太原郡]]和[[上党郡|上党郡]]。[[东汉|东汉]]时，在今昔阳置[[乐平郡|乐平郡]]。[[三国|三国]]时属[[曹魏|魏国]][[并州|并州]]，分属太原、上党、西河、乐平4郡。

[[南北朝|南北朝]][[北魏|北魏]]统一北方后，分属并州太原郡、武乡郡、乐平郡、上党郡和汾州西河郡。[[隋|隋]]初废郡设州，改属并、介、韩、辽、吕5州。后废州复郡，属太原、西河2郡，后又改属太原、介休、霍山3郡。[[唐代|唐代]]建置变化频繁，先后属并、介、吕、韩、太、辽、榆、受、箕、汾、仪等州及太原府，至[[中和_(唐朝)|中和]]三年（883年）为太原府和汾、辽2州统领。[[北宋|北宋]]时期，属河东路太原府、汾州、辽州、平定军。宋末为金国所占。[[元代|元代]]分属冀宁路和晋宁路。明清两代，分属于太原、汾州、平阳3府和辽、平定、霍州3个直隶州。
 
[[民国|民国]]初期，全境分属冀宁道和河东道。民国16年（1927年），废除道制，直属山西省督军府，后为山西省政府、山西绥靖公署。[[抗日战争|抗日战争]]时期分属晋察冀、晋冀鲁豫、晋绥边区政府管辖。1948年7月，[[解放军|解放军]]进驻晋中。1949年9月，设立榆次区行政督察专员公署，称[[榆次专区|榆次专区]]。1950年，改称榆次区专员公署。1955年，更名山西省榆次专员公署。1958年，改名晋中专员公署，简称[[晋中专区|晋中专区]]。1968年，成立[[晋中地区|晋中地区]][[革命委员会_(文革)|革命委员会]]。1978年，改设晋中地区行政公署。1983年，榆次市县合并，成立新的县级榆次市。1992年，介休县改为县级介休市。

2000年10月，撤销晋中地区，设立地级晋中市，原县级榆次市改为榆次区。

== 自然地理 ==
市境处于[[黄土高原|黄土高原]]东部边缘，地势东高西低，山地、丘陵、平川呈阶梯状分布，大部分地区处于海拔1000米以上。

河流以[[太行山|太行山]]、[[太岳山|太岳山]]中脊为界，分属[[黄河|黄河]]流域和[[海河|海河]]流域。东部主要有松溪河、清漳河、浊漳河；西部主要有潇河、乌马河、昌源河、惠济河、龙凤河、静升河。[[汾河|汾河]]为流经境内第一大河。

全市属[[温带季风气候|暖温带大陆性季风气候]]，季节变化明显。全年平均日照时数为2530.8小时。年平均气温9.4℃。年平均无霜期151天。降水主要集中在夏季6至8月份，年平均降水量479.6毫米。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为3249425人<ref>{{cite book |author=晋中市统计局|title=《晋中市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，共增加232889人，增长7.72%，年平均增长率为0.75%。其中，男性人口为1694946人，占52.16%；女性人口为1554479人，占47.84%。总人口性别比（以女性为100）为109.04。0－14岁人口为555567人，占17.10%；15－64岁人口为2425230人，占74.64%；65岁及以上人口为268628人，占8.27%。

== 文化 ==
=== 方言 ===
晋中市境内的[[汉语方言|汉语方言]]主要属于[[晋语|晋语]][[并州片|并州片]]（西部平川的榆次、太谷、祁县、平遥、介休，以及东部山区的寿阳、榆社）、[[吕梁片|吕梁片]]（灵石）以及[[大包片|大包片]]（东部山区的昔阳、和顺、左权）。

== 经济 ==
晋中是全省[[粮食|粮食]]、[[蔬菜|蔬菜]]、畜产品、干鲜果的主要产区之一，蔬菜和畜禽产品综合产量连续多年位居全省第一。

[[工业|工业]]以[[机械|机械]]、[[煤炭|煤炭]]、[[冶金|冶金]]、[[化学工程|化工]]等产业为主，煤焦、纺机、玻璃器皿、玛钢、医药等产品在全省比重较高。
