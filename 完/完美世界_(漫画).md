《'''完美世界'''》（{{lang-ja|パーフェクトワールド}}，''Perfect World''）是[[日本|日本]][[漫畫家|漫畫家]]{{Link-ja|有賀理惠|有賀リエ}}的[[少女漫畫|少女漫畫]]作品，描繪了女主角和由於遭遇事故而殘疾的男性的戀愛故事<ref>{{Cite web|title=車椅子生活になった初恋の人との恋描くマンガ「パーフェクトワールド」実写化|url=https://web.archive.org/web/20170710150614/https://natalie.mu/eiga/news/234387|work=映画ナタリー|date=2017-07-10|accessdate=2019-02-19|author=|publisher=|language=ja}}</ref>。2014年5月開始在《[[Kiss_(雜誌)|Kiss]]》雜誌上連載。

題為《'''完美世界 與你在一起的奇跡'''》（{{lang-ja|パーフェクトワールド 君といる奇跡}}）的真人版電影於2018年公開。<ref name=":0">{{Cite web|title=三代目ＪＳＢ岩田剛典＆杉咲花、来年公開「パーフェクトワールド」でＷ主演！|url=https://web.archive.org/web/20171227122458/http://www.sanspo.com/geino/news/20170610/geo17061005030006-n1.html|work=SANSPO.COM|date=2017-12-27|accessdate=2019-02-19|author=|publisher=}}</ref>

2019年4月，由[[富士電視台|富士電視台]]翻拍同名電視劇。<ref name=":1">{{Cite web|title=松坂桃李が車いすの建築士に 山本美月と初共演のラブストーリー『パーフェクトワールド』4月放送|url=https://realsound.jp/movie/2019/01/post-312268.html|work=Real Sound｜リアルサウンド 映画部|accessdate=2019-02-19|language=ja}}</ref>

== 故事簡介 ==
在室內設計公司工作的'''川奈愛未'''在與公司同事的飲酒會上，與高中時代的同學'''鮎川樹'''再會。而鮎川在大學時代由於遭遇事故以致半身癱瘓，現在必須要依靠輪椅生活。在心動之後兩人卻要面臨來自家庭的壓力、疾病與傷痛、生活上的障礙等等。

== 登場人物 ==

; {{nihongo|川奈愛未|川奈 つぐみ（かわな つぐみ）}}
: 本部作品的主人公。
: 十分擅長繪畫，高中時的夢想是考入美術大學，後來放棄了這個夢想。高中畢業之後在東京的室內設計公司「Cranberries」工作。之後由於父親患病，辭去了工作回到了故鄉[[长野县|長野縣]][[松本市|松本市]]。
:川奈在高中時代便喜歡著鮎川，但由於當時的鮎川已有戀人，於是直到畢業也沒有向他告白。在東京和他久別重逢之後開始了交往。
:'''※關於譯名'''：由於川奈つぐみ沒有漢字寫法，所以有多種不同中文譯名。台灣[[東立出版社|東立出版社]]引進漫畫、香港引進真人版電影、以及台灣[[KKTV|KKTV]]引進真人版電視劇時均翻譯爲川奈愛未<ref>{{Cite web|title=完美世界 1書籍信息|url=http://www.eslite.com/Search_BW.aspx?query=%E5%AE%8C%E7%BE%8E%E4%B8%96%E7%95%8C%201|work=誠品網路書店|author=|date=|publisher=}}</ref><ref>{{Cite web|title=《Perfect World～與他一起的奇蹟～》 預告宣傳|url=https://www.facebook.com/NFinfinity/posts/2206922469593800|work=INFi∞ITY．新映搜畫官方FACEBOOK|author=|date=|publisher=}}</ref>。[[汉化组|民間翻譯]]亦有川奈亞實、川奈亞美等不同寫法。
; {{nihongo|鮎川樹|鮎川 樹（あゆかわ いつき）}}
: 本部作品的男主角，是在設計事務所工作的一級建築士。
:在大學三年級由於遭遇事故成為了[[身心障礙|身體障害者]]，從那之後一直依靠輪椅生活，成爲建築師后致力於[[無障礙環境|無障礙住宅]]的設計。
:在高中時代屬於籃球部，成爲身障者之後加入了[[輪椅籃球|輪椅籃球隊]]。
; {{nihongo|是枝洋貴|是枝 洋貴（これえだ ひろたか）}}
: 現在在東京作爲[[软件工程师|系統工程師]]工作的男性，是川奈和鮎川的高中同學。
: 在高中時代喜歡著川奈，但是直至畢業都沒有告白。和川奈重逢后再一次為她心動，在川奈和鮎川分手之後向她告白，并開始交往。
; {{nihongo|長澤|長沢（ながさわ）}}
: 樹的[[護理人員|護理師]]。
:本身是在復建中心工作的護理師，在樹入院之後和他相識，陪伴樹度過了他剛剛成爲身障者時十分絕望的階段。雖然她已婚，但仍然對樹保有愛意，所以也很嫉妒愛未。
; {{nihongo|晴人|晴人（はると）}}
: 東京的高中生。曾經是籃球名校的王牌隊員，之後成爲了身障者。
:父母在把家裏重新改裝成無障礙住宅的時候，認識了樹和愛未，在樹的邀請之下也加入了輪椅籃球隊。
; {{nihongo|舞花|舞花（まいか）}}
: 晴人的青梅竹馬。通過晴人認識了樹和愛未，愛未時常與她傾訴心事。

和原作不同的是，川奈愛未在電影中的設定是鮎川的後輩。
