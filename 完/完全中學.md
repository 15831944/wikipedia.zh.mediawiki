'''完全中學'''是一種學校制度，為一所同时设立[[初級中學|初級中學]]部及[[高級中學|高級中學]]部的高級中學，或同時大學[[預科|預科]]課程的[[中學|中學]]。

== {{CHNMLD}} ==
[[中國大陸|中國大陸]]許多中学同时设立[[初中|初中]]部及[[高中|高中]]部，这些中学就是完全中学。由于历史的原因，一些曾经开设初中但后来取消了初中部或者不直接开设的，仍然被称为完全中学。

== {{HKG}} ==
[[File:QualiEd_College_2011.jpg|thumb]]是一所完全中學]]
在[[香港|香港]]，完全中學屬於主流的中學類型，即大部分中學都開辦中一至中六課程。在舊學制下，這類中學一般會開辦中一至中七課程；當中，中一至中五不會正式地被分為「初中」和「高中」，但中六至中七可稱為「[[預科|預科]]」（Advanced-Level）。

在新學制下，這類中學則一般會開設中一至中六的課程；當中，中一至中三為「初中」，中四至中六為「高中」。近年來，一些專門以初中離校生為招生對象的高中學校陸續出現，如[[中華基督教會公理高中書院|中華基督教會公理高中書院]]、[[明愛華德中書院|明愛華德中書院]]等。

此外，在舊學制中一些學校專門提供[[預科|預科]]課程，並保送學生參加[[香港高級程度會考|香港高級程度會考]]以進入大學，較著名的有[[恒生商學書院|恒生商學書院]]和[[保良局莊啟程預科書院|保良局莊啟程預科書院]]等。因應[[三三四高中教育改革|三三四高中教育改革]]的實施，[[香港預科課程|香港預科課程]]於2012年被取締，這些學校亦要另覓出路，如轉型為高中學校或高等教育機構等。

== {{TWN}} ==
[[臺灣|臺灣]]在1968年實施[[臺灣九年國民義務教育|九年國民義務教育]]以前，中學分為[[初級中學|初級中學]]與[[高級中學|高級中學]]。當時省立中學許多都是完全中學，同時設立高中部與初中部，僅有少數是純高級中學。縣市立中學則多是初級中學為主，少數幾間則有高中部。後來為降低國民學校學生參加[[初級中學|初級中學]]的升學壓力，於五十一學年度起開始試辦[[省辦高中，縣市辦初中|省辦高中，縣市辦初中]]措施。1968年實施九年國教時，確立[[省辦高中，縣市辦初中|省辦高中，縣市辦國中]]原則，因此高級中學與國民中學多是分別設立，只有部分私立學校和公立實驗中學為完全中學。1979年高級中學法通過之後，明訂高中允許設立國民中學部或職業類科，因此各私立中學改制為高級中學，開始以[[高中|高中]]附設[[國中|國中]]部的完全中學形式。

1990年代以後，因高中需求量增加，政府政策有所調整，各縣市為擴大高中學生招生數額，出現許多原本以國中為基礎，增設高中部的完全中學，初期由於高中僅能由省或直轄市設立，因此只能稱為「中學」（High School），後來1999年[[高級中學法|高級中學法]]賦予縣市政府設立高中權限後，各縣市立中學紛紛更名為縣市立高級中學（Senior High School）。

=== 政府成立 ===
少數完全中學，為單獨成立，由市府支出，並非由[[中華民國教育部|教育部]]掌管。而其他國立高中，財政狀況大多不理想，使得教育經費日益困窘，須透過[[中華民國教育部|教育部]]補助或申請高中優質化、均質化等專案經費挹注。

==== 桃園市 ====
因高中部及國中部性質差異過大<ref>[http://news.ltn.com.tw/news/life/paper/550024 完全中學省過頭 學生挨垃圾桶坐]</ref><ref>[http://www.coolloud.org.tw/node/65675 缺資源 社區化難實現 全教會呼籲改善完全中學環境]</ref>，[[桃園市|桃園市]]公立的完全中學多已各自分立。2002年[[桃園市立平鎮高級中等學校|平鎮高中]]國中部獨立設校，復名為[[桃園市立平鎮國民中學|桃園縣立平鎮國民中學]]。2008年[[桃園縣立壽山國民中學|壽山國中]]因應改制[[桃園市立壽山高級中等學校|桃園縣立壽山高級中學]]，國中部停止招生，原國中部學區新生轉至龜山國中、幸福國中就讀，高中部及國中部僅共存一學年度。2008年[[桃園縣立大漢國民中學|大漢國中]]因應改制[[桃園市立大溪高級中等學校|桃園縣立大溪高級中學]]，國中部停止招生，原國中部學區新生轉至大溪國中就讀，高中部及國中部僅共存一學年度。2011年[[桃園市立南崁高級中等學校|南崁高中]]國中部獨立設校，復名為[[桃園市立南崁國民中學|桃園縣立南崁國民中學]]。2016年[[桃園市立永豐高級中等學校|永豐高中]]國中部與高中部開始進行國、高中各自獨立運作的籌備作業，於茄苳地區文中用地新設[[桃園市立永豐國民中學|桃園市立永豐國民中學]]<ref>{{Cite web|url=http://www.tycg.gov.tw/ch/home.jsp?id=9&parentpath=0,1&mcustomize=news_view.jsp&dataserno=201610240003&aplistdn=ou=news,ou=chinese,ou=ap_root,o=tycg,c=tw&toolsflag=Y|title=10月24日市政總質詢：呂淑真議員-桃園市政府|accessdate=2016-12-31|date=2016-10-24|last=桃園市政府|work=www.tycg.gov.tw}}</ref>。2017年5月桃園市政府教育局欲將[[桃園市立建國國民中學|桃園市立建國國民中學]]發展為市立[[技術型高級中等學校|技術型高級中等學校]]兼完全中學、增設商業職科，因校園空間不足等因素遭受一百多名校內教師連署反對<ref>[http://news.ltn.com.tw/news/local/paper/1100525 〈北部〉變完全中學？ 建國國中逾百師反對]</ref>。同年6月桃園市市長鄭文燦在議會備詢時，明確表示建國完全中學計畫不可行，市府除了實驗教育外（仁美華德福高中），不打算再做完全中學<ref>[http://news.ltn.com.tw/news/politics/breakingnews/2091941 建國國中改完全中學？ 鄭文燦打臉教育局長：不辦了]</ref><ref>[http://www.chinatimes.com/newspapers/20170620000553-260107 華德福實驗教育 催生高中]</ref>。公立完全中學僅存位於偏鄉的[[桃園市立觀音高級中等學校|觀音高中]]、[[桃園市立新屋高級中等學校|新屋高中]]及籌備中的[[桃園市立原住民族高級中等學校|原住民高中]]，以及計畫中的[[桃園市立大崗國民中學|大崗高中]]。

== {{JPN}} ==
在[[日本|日本]]，完全中學稱為「中等教育學校」或「中高一貫校」，其中「中」指中學校（初中）、「高」指高等學校（高中）；部分私立學校甚至會設置小學部，稱為「小中高一貫校」。實施完全中學學制的私立學校多以「學園」為名。
