'''納尼亞年表'''，是[[英國|英國]][[奇幻小說|奇幻小說]]系列《[[納尼亞傳奇|納尼亞傳奇]]》裡故事情節的大事年表。

《納尼亞》系列的作者[[C·S·路易斯|C·S·路易斯]]提供了一份納尼亞時間表來描繪這本小說終發生或提到的事件，他以手稿形式向[[沃爾特·霍伯|沃爾特·霍伯]]發布一篇「納尼亞歷史綱要」（Outline of Narnian History），並將其納入散文〈Past Watchful Dragons: The Fairy Tales of C. S. Lewis〉內<ref>Hooper, Walter (1979), Past Watchful Dragons: The Narnian Chronicles of C. S. Lewis, New York: Macmillan Pub Co, ISBN 0-02-051970-2</ref><ref>{{cite news|url=https://www.narniaweb.com/books/timeline/|title=Narnia Timeline|publisher=NarniaWeb|accessdate=2018-06-23}}</ref>。由於《納尼亞》系列中並未明確指出各事件發生的確切年分，因此這類訊息的唯一來源便是路易斯的這份納尼亞時間表。

納尼亞世界的時間流逝與人類世界往往是不連貫的，有時在納尼亞過去了幾百年時間、而人類世界只過了一年。

== 納尼亞時間表 ==

=== 第一個千年 ===

; 1年
: 納尼亞創世，[[陸上大帝|陸上大帝]]之子{{le|亞斯藍|Aslan}}賦予動物說話的能力，{{le|白女巫|White Witch}}隨著{{le|狄哥里‧寇克|Digory Kirke|狄哥里}}等人進入納尼亞，之後逃到北方極地。來自[[倫敦|倫敦]]的[[馬車夫|馬車夫]]{{le|法蘭克及海倫|Frank and Helen}}成為納尼亞首任[[國王|國王]]和王后<ref name="禤浩榮">{{Cite book|url=https://books.google.com.tw/books?id=J4ZbCgAAQBAJ&pg=PT152&lpg=PT152&dq=%E8%A1%A3%E6%AB%A5+%E7%B4%8D%E5%B0%BC%E4%BA%9E+%E9%9D%88%E6%84%9F&source=bl&ots=RhQ0G7VH1J&sig=g7znDB65pxBwxhwPfrpsdWkxhvM&hl=zh-TW&sa=X&ved=0ahUKEwjv3qus5svaAhXDTbwKHfX4AUc4ChDoAQhCMAU#v=onepage&q=%E8%A1%A3%E6%AB%A5%20%E7%B4%8D%E5%B0%BC%E4%BA%9E%20%E9%9D%88%E6%84%9F&f=false|title=魔幻王國的心靈世界|last=禤浩榮|publisher=天道書樓|year=2006|isbn=9622086721|location=|archiveurl=https://web.archive.org/web/20180423003749/https://books.google.com.tw/books?id=J4ZbCgAAQBAJ&pg=PT152&lpg=PT152&dq=%E8%A1%A3%E6%AB%A5+%E7%B4%8D%E5%B0%BC%E4%BA%9E+%E9%9D%88%E6%84%9F&source=bl&ots=RhQ0G7VH1J&sig=g7znDB65pxBwxhwPfrpsdWkxhvM&hl=zh-TW&sa=X&ved=0ahUKEwjv3qus5svaAhXDTbwKHfX4AUc4ChDoAQhCMAU#v=onepage&q=%E8%A1%A3%E6%AB%A5%20%E7%B4%8D%E5%B0%BC%E4%BA%9E%20%E9%9D%88%E6%84%9F&f=false|archivedate=2018-04-23|deadurl=no}}</ref>。

; 180年
: 法蘭克五世的小兒子科爾親王和其追隨者進入無人居住的{{le|亞成地|Archenland}}，成為那裡的首任國王<ref name="禤浩榮" />。

; 204年
: 一些亞成地的不良分子越過南方的大[[沙漠|沙漠]]，建立了{{le|卡羅門|Calormen}}國家<ref name="禤浩榮" />。

; 300年
: 一群卡羅門人離開自己的國家，在納尼亞西部的[[坦摩|坦摩]]地區定居<ref name="禤浩榮" />。

; 302年
: 在坦摩的卡羅門人變得越加惡劣，亞斯藍把他們變成不會說話的動物。同年，納尼亞國王Gale殺死肆虐寂島的惡龍，因而兼併了寂島皇帝的稱號<ref name="禤浩榮" />。

; 407年
: 亞成地國王奧爾文殺死雙頭[[巨人|巨人]]皮爾<ref name="禤浩榮" />。

; 460年
: 幾名來自[[人類|人類]]世界的[[海盜|海盜]]透過一條神奇的裂縫進入納尼亞的坦摩<ref name="禤浩榮" />，即為後來的坦摩人。

; 898年
: 白女巫賈迪絲重返納尼亞，使用一根具有強大法力的魔杖作為武器，可以將除了亞斯藍以外的所有人變成石頭。

; 900年
: 白女巫成功征服整個納尼亞，自封為納尼亞女王。她下了一道[[詛咒|詛咒]]，使整個納尼亞永遠維持在[[冬季|冬季]]。

=== 第二個千年 ===

; 1000年
: 當[[百年寒冬|百年寒冬]]滿第一千年時，來自英國的佩文西四兄妹透過一個[[衣櫥_(納尼亞)|舊衣櫥]]進入納尼亞，愛德蒙背叛，亞斯藍自我犧牲，在{{le|石桌 (納尼亞)|Stone Table|石桌}}上被白女巫所殺，次日早晨又在蘇珊和露西面前復活。[[第一次貝路納之役|貝路納之役]]爆發，在亞斯藍的援助下，白女巫身亡，她的黨羽潰散。彼得、蘇珊、愛德蒙和露西成為納尼亞[[凱爾帕拉瓦宮|凱爾帕拉瓦宮]]的四位君主<ref name="禤浩榮" />。

; 1014年
: 彼得大帝領軍擊敗了北方的巨人。另一方面，蘇珊女王、愛德蒙國王和[[人羊吐納思|人羊吐納思]]等人造訪南方的卡羅門，卡羅門的{{le|羅八達|Rabadash}}王子領軍侵略納尼亞和亞成地，終被擊敗。亞成地的半月國王發現男孩{{le|沙斯塔|Shasta (Narnia)}}就是他失蹤的兒子柯爾。

; 1015年
: 彼得大帝、蘇珊女王、愛德蒙國王和露西女王在追逐白雄鹿期間意外返回人類世界<ref name="禤浩榮" />，發現時間只過去幾秒鐘，他們自己都還是孩子。

; 1998年
: 坦摩人征服了納尼亞，征服者賈思潘一世成為納尼亞國王。

=== 第三個千年 ===

; 2290年
: 賈思潘九世之子{{le|賈思潘王子 (角色)|Prince Caspian (character)|賈思潘王子}}出生。賈思潘九世遭兄弟{{le|米拉茲|Miraz}}[[謀殺|謀殺]]並奪位<ref name="禤浩榮" />。

; 2303年
: 賈思潘王子逃出叔叔米拉茲的掌控，引發納尼亞內戰，由於賈思潘吹響了蘇珊女王遺留下來的號角，而陸續將佩文西四兄妹朝喚回納尼亞。米拉茲被自己的手下所殺，在{{le|第二次貝路納之役|Second Battle of Beruna}}當中，坦摩軍隊潰敗。賈思潘王子[[加冕|加冕]]成為賈思潘十世。

; 2304年
: 賈思潘十世擊敗北方巨人。

; 2306年
: 賈思潘十世同{{le|垂尼安勳爵|Lord Drinian}}、[[老鼠|老鼠]][[老脾氣|老脾氣]]等人出航至大東洋，尋找之前失蹤的七位勳爵，並航行至接近世界盡頭之處。

; 2310年
: 賈思潘十世與[[拉曼杜的女兒|拉曼杜的女兒]]成婚<ref name="禤浩榮" />。

; 2325年
: {{le|瑞里安|Rilian}}王子出生<ref name="禤浩榮" />。

; 2345年
: 賈思潘十世的王后被女巫「{{le|綠衣女士|Lady of the Green Kirtle}}」化身的大蛇殺死。瑞里安王子在尋找殺母仇人的過程中也失蹤了。

; 2356年
: {{le|尤斯提·史瓜|Eustace Scrubb}}和{{le|姬兒·波爾|Jill Pole}}進入納尼亞，他們跟[[沼澤|沼澤]]人{{Le|泥桿兒|Puddleglum}}一同從綠衣女士手中救出瑞里安王子。賈思潘十世在與兒子重聚後便去世了。

; 2555年
: 狡猾的[[猿猴|猿猴]]{{le|席夫特|Shift (Narnia)}}利用{{link-nl|迷糊|Puzzel (Narnia)|驢子迷糊}}偽裝成亞斯藍，欺騙了所有納尼亞人，納尼亞末代國王{{le|逖里安|Tirian}}為尤斯提和姬兒所救。納尼亞被卡羅門軍隊征服，戰役爆發。狄哥里、波莉、彼得、愛德蒙、露西在死去後皆重返納尼亞，納尼亞被黑暗壟罩，亞斯藍引導眾人進入{{le|亞斯藍的國度|Aslan's Country}}。

== 英格蘭 ==

* '''1888年'''：{{le|狄哥里‧寇克|Digory Kirke}}出生。
* '''1889年'''：{{le|波莉·普朗摩|Polly Plummer}}出生。
* '''1900年'''：狄哥里‧寇克與母親搬至[[倫敦|倫敦]]。狄哥里與[[鄰居|鄰居]]波莉意外透過[[安德魯·凱特里|安德魯舅舅]]實驗室裡的魔法戒指到達一個名叫{{le|查恩|Charn}}的世界，在那裡狄哥里無意中喚醒白女巫賈迪絲，並把她一同帶回來。白女巫在倫敦街頭製造了一陣騷亂，後來除了[[警察|警察]]之外的涉案人員，包括狄哥里、波莉、安德魯舅舅、馬車夫法蘭克、白女巫等人皆被魔戒帶到納尼亞世界<ref name="禤浩榮" />。
* '''1927年'''：{{le|彼得·佩文西|Peter Pevensie}}出生。
* '''1928年'''：{{le|蘇珊·佩文西|Susan Pevensie}}出生。
* '''1930年'''： {{le|愛德蒙·佩文西|Edmund Pevensie}}出生。
* '''1932年'''：{{le|露西·佩文西|Lucy Pevensie}}出生。
* '''1933年'''：{{le|尤斯提·史瓜|Eustace Scrubb}}和{{le|姬兒‧波爾|Jill Pole}}出生。
* '''1940年'''：由於[[倫敦大轟炸|倫敦遭受空襲]]，佩文西四兄妹被迫往鄉間撤離，他們暫時安住於狄哥里‧寇克[[教授|教授]]的家中。露西·佩文西最先發現了屋內的魔法衣櫥並透過其進入納尼亞，之後她的兄姊也陸續進入，在納尼亞世界待了十五年，但當他們返回到教授家中時發現在我們的世界僅過去了幾秒鐘而已<ref name="禤浩榮" />。
* '''1941年'''夏季：佩文西四兄妹在一個鄉間[[火車站|火車站]]準備坐火車去學校，他們受到招喚再度回到納尼亞<ref name="禤浩榮" />。
* '''1942年'''夏季：愛德蒙和露西·佩文西在表親尤斯提·史瓜家中度過[[暑假|暑假]]，並透過一幅畫穿越至納尼亞<ref name="禤浩榮" />。
* '''1942年'''秋季：在一所實驗學校，尤斯提·史瓜和同學姬兒‧波爾透過一扇舊門進入納尼亞。
* '''1949年'''：狄哥里、波莉、彼得、愛德蒙、露西、尤斯提和姬兒在一次聚會時見到納尼亞國王逖里安的幻象，他們一致決定要設法回到納尼亞，彼得和愛德蒙挖出了狄哥里‧寇克之前使用的魔法戒指，其他人則乘坐[[火車|火車]]要與之碰頭。不料這時卻發生一場嚴重的火車事故，所有人都在當中身亡了；並被送到納尼亞。
