'''納豆'''是[[日本|日本]]常見的傳統[[發酵食品|發酵食品]]，由[[黃豆|黃豆]]通過''Bacillus subtilis natto''（[[枯草桿菌|枯草桿菌]]亞種之一）藉由菌種發酵所釋放的酵素轉化蒸煮過黃豆的營養成分所製成的[[豆製品|豆製品]]<ref> 施英隆,范宜琮.2001. 納豆-神奇之保健食品. 生物資源生物技術 4(1):37-45 </ref>，气味浓烈，有黏性，不僅保有黃豆的營養價值、富含[[維生素K2|維生素K2]]、提高[[蛋白質|蛋白質]]的消化吸收率，更重要的是發酵過程產生了多種生理活性物質，具有溶解體內[[纖維蛋白|纖維蛋白]]及其他調節生理機能的保健作用。根據[[藤原明衡|藤原明衡]]所著的[[新猿樂紀|新猿樂紀]]推算，日本人食用納豆至今已有超過一千年的歷史。

== 起源 ==
雖然日本近代古書《[[和漢三才圖會|和漢三才圖會]]》中曾記載：『納豆自中國[[秦|秦]][[漢|漢]]以來開始製作。』，然《[[和漢三才圖會|和漢三才圖會]]》是1712年才出版，同時內容也被認為含有空想荒誕不經的成分。在可信的學術文獻中，「納豆」一詞最早出現於平安時代11世紀中期，由藤原明衡所著的『新猿樂紀』當中所記載的；『腐水葱香疾大根舂塩辛納豆』。日本的納豆粗略可分為兩種，一種是日本獨有的拉絲納豆(糸引き納豆)，另一種則是納豆一詞起源的腌咸納豆(塩辛納豆)，現在日本一般認知為納豆而食用的，是以拉絲納豆為主的產品，但是納豆原本的意思是指腌咸納豆。

日本獨有的拉絲納豆，其來源無法確定，一個來源說認為是自繩文時代由外傳入大豆與稻子之後，在漫長的過程中，偶然出現的一種發酵食品，另一個來源說則認為是由聖德太子所發現。<ref>[http://finedays.org/natto/nattohistory.html]，妄說 納豆之歷史。</ref>。而腌咸納豆則始於中國的[[豆豉|豆豉]]，日本也曾稱納豆為「豉」<!-- 但是否和中国作为咸菜的豆豉一样不得而知 -->，[[平城京|平城京]]出土的木簡中也有「豉」宇。由於豆豉在僧家寺院的[[納所|納所]]製造後放入甕桶貯藏，後由禪僧從中國傳播到日本寺廟，所以納豆首先在寺廟得到發展，是以亦稱「唐納豆」或「鹹納豆」。隨後依當地環境發展出納豆，如日本不用豆豉而用大醬，或用醬油不用豉汁。在日本更基於各地區特色，傳承出諸如大龍寺納豆、大德寺納豆、[[一休|一休]]納豆、大福寺的濱名納豆、悟真寺的八橋納豆等等地方特產。

== 製作方式 ==
[[File:Natto_wrapped_in__straw.jpg|right]]
[[File:Natto_boxed.jpg|thumb]]
傳統製法將蒸熟的[[黃豆|黃豆]]用[[稻草|稻草]]包裹起來，稻草浸泡在100度的沸水中以殺菌消毒，並保持在[[攝氏|攝氏]]40度放置一日，稻草上常見的枯草桿菌（{{link-ja|納豆菌}}）因可产生[[芽孢|芽孢]]而耐熱度高，殺菌過程不受破壞，高溫培養速度快也能抑制其他菌種，並使黃豆發酵後產生黏稠的絲狀物，這種黏稠外觀主要來自成分中的[[穀氨酸|穀氨酸]]，被認為是納豆美味的來源。有[[麴|麴]]的[[味噌|味噌]]製作時須維持低溫，納豆的發酵則需要較高的溫度。<ref>{{Cite news|url=https://www.xuehua.us/2018/04/20/%E6%B6%A8%E7%9F%A5%E8%AF%86%EF%BC%81%E5%A4%A7%E7%B1%B3%E5%92%8C%E7%BA%B3%E8%B1%86%E8%BF%98%E6%9C%89%E5%A6%82%E6%AD%A4%E5%AF%86%E4%B8%8D%E5%8F%AF%E5%88%86%E7%9A%84%E5%85%B3%E7%B3%BB%EF%BC%81/|title=涨知识！大米和纳豆还有如此密不可分的关系！ - 雪花新闻|author=|date=2018-04-20|work=|newspaper=雪花新闻|accessdate=2018-08-16|language=zh-CN|quote=製作的時候，如果維持低溫，大豆發酵後就會成為含有麹的味噌，但如果溫度過高，大豆就會發酵成【納豆】江戶時代會把味噌和【發酵失敗所產生的納豆】一起煮成湯飲用。}}</ref><ref>{{Cite web|url=http://www.books.com.tw/products/0010554118|title=你不可不知的日本飲食史|accessdate=2018-08-16|author=|date=|work=博客來|publisher=|pages=53}}</ref>

二十世紀後期高品質稻草取得不易，多已改用[[保麗龍|保麗龍]]或紙製容器盛裝販售。因此現代的製作方式，是利用蒸過的黃豆加上人工培養出的納豆菌混合後，直接放在容器中使之發酵，黃豆以外之某些食品亦能製成納豆。

而製作[[清酒|清酒]]時，原料的[[米|米]]中若不慎受納豆菌沾染，以其快速的繁殖速度，將抑制釀酒所需的酵母菌。釀清酒處所不可出現納豆，其他發酵食品可能也有類似情況，製程需多加留意。

納豆菌屬於弱[[酸性|酸性]]，會阻礙[[乳酸菌|乳酸菌]]製造的乳酸。在技術上已經開發出氣味較弱的納豆，但由於使用了活發性較低的納豆菌種，容易造成其他細菌的增殖空間。此外，納豆菌的天敵是會寄生在細菌上的[[噬菌體|噬菌體]][[病毒|病毒]]，在噬菌體開始作用後會降低納豆菌的活動力，並可能造成其他細菌開始繁殖，因此應避免食用超過保存期限的納豆。

== 食用方式 ==
傳統上先將納豆加上[[醬油|醬油]]或[[芥末|日式芥末]]（一般在百貨所賣的盒裝納豆，使用的是日式淡醬油還有黃芥末），攪拌至絲狀物出現，置於[[飯|白飯]]上食用，是為納豆飯。也有人將納豆和生[[雞蛋|雞蛋]]、[[蔥|蔥]]、[[茗荷|茗荷]]、蘿蔔、[[柴魚|柴魚]]等各種食材一起混合。在[[北海道|北海道]]、[[東北地方|東北地方]]有時也將納豆和[[砂糖|砂糖]]混合。亦有加上[[美乃滋|美乃滋]]的創意吃法。若未經攪拌便加入醬料會使水分過多，粘性也會有所消減，蔥和芥末則能抑制「納豆氨」刺鼻的氣味。不習慣吃納豆者則以為是腐壞的水煮黃豆。

== 纳豆的营养 ==
纳豆含有黄豆全部营养，再加发酵后增加的特殊养份，含有皂素，异黄酮，不饱和脂肪酸，卵磷脂，叶酸，食用纤维，钙，铁，钾，维生素及多种氨基酸、矿物质及100多种以上的种类，维生素K2，二吡啶酸等重要养分，适合长期食用维护健康。

经日本的医学家、生理学家研究得知，大豆的蛋白质具有不溶解性，而做成纳豆后，变得可溶并产生氨基酸，而且原料中不存在的各种酵素会由于纳豆菌及关联细菌产生，帮助肠胃消化吸收。纳豆的成分是：水分61.8%、粗蛋白19.26%、粗脂肪8.17%、碳水化合物6.09%、粗纤维2.2%、灰分1.86%，作为植物性食品，它的粗蛋白、脂肪最丰富。纳豆系高蛋白滋养食品，纳豆中含有的酵素，食用后可排除体内部分胆固醇、分解体内酸化型脂质，使异常血压恢复正常<ref>《身体科学》杂志1994年1月号，日本评论社出版，第4页</ref>。
