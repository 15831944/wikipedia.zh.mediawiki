'''納智捷U5 SUV'''（'''Luxgen U5'''）是一款由[[納智捷汽車|納智捷汽車]]于2017年推出的5人跨界休旅車系列。此車採 [[納智捷_S3|納智捷 S3]] 底盤為基礎進行改良。有 1.6L汽油引擎 和電動引擎(預計發售)兩種款。發售至目前已陸續提供雙色車頂、毛小孩、烈焰/風暴、天窗、等客製可供選(加)購。售價依車型不同為台幣NT$65.9萬到NT$78.9萬之間。<ref>{{Cite web |url = https://www.luxgen-motor.com.tw 
|title =納智捷官網 }}</ref>

本車為納智捷 [[510計劃|510計劃]] 中的首發車款，採用由[[華擎機械|華擎機械]]與[[水野和敏|水野和敏]]共同調教之1.6升直列四缸自然進氣引擎，可提供120匹最大馬力，較為吸睛的有，外觀上有從大燈延伸至車側葉子板的LED方向燈條，中央控制區使用了12吋觸控螢幕作為主要資訊顯示與操作，透過鏡頭與車機電腦演算的3D環景與自動警示鏡頭影像，並搭載可協助新手駕駛的APA自動停車輔助系統(需APA以上車型)。<ref>{{Cite web |url =  https://turnnewsapp.com/king/10123.html
|title = FUN玩雙色任我行 LUXGEN U5 |author = 翻爆-翻報 }}</ref><ref>{{Cite web |url =https://xw.qq.com/auto/20170914042365/AUT2017091404236500
|title = 納智捷510計劃 |author = 騰訊網|date = 2017年9月14日 }}</ref>

該車搭載智慧型電腦部分，初始由[[宏達電|宏達電]]協同開發的[[車載資訊系統|車載資訊系統]]和[[電子穩定程序|電子穩定程序]]，稱為THINK+ 4.0系統，特色為聲控功能、衛星導航、全車最多8顆CCD，其中包含夜視攝影機，並有行動上網功能可以更新系統和下載旅遊、商店、即時交通資訊，後續程式的合作據傳由鴻海子公司[[安泰電業|安泰電業]]接手。
<ref>{{Cite web |url =http://www.nextmag.com.tw/realtimenews/news/39314140
|title = 納智捷揮別htc　改攜手鴻海 |author = 壹周刊|date = 2016年5月15日 }}</ref>

在U5上所搭載的智慧系統也有著令人注目的項目，其一為聲控功能，可讓駕駛在行車中無需觸碰中央控制台的按鍵做調整，更加注意前方動態，大大降低駕駛分心的機會。
