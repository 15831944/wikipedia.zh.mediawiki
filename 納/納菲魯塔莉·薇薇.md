'''納菲魯塔莉·薇薇'''為日本漫畫家[[尾田榮一郎|尾田榮一郎]]所創作的日本少年漫畫《[[ONE_PIECE|ONE PIECE]]》中的一位角色。為[[阿拉巴斯坦王國|阿拉巴斯坦王國]]的公主，在秘密犯罪組織[[巴洛克華克|巴洛克華克]]卧底时期代號為「Miss 星期三」。後來因為[[ONE_PIECE海賊列表#草帽海賊團|草帽海賊團]]的相救下，薇薇和他們一起踏上了短暂的冒險之旅。

== 設計 ==
名字取自真实人物[[古埃及|古埃及]]法老[[拉美西斯二世|拉美西斯二世]]的大王后'''[[妮菲塔莉|妮菲塔莉]]'''（Nefertari）。而薇薇的姓氏「納菲鲁塔利」在埃及也有「美麗之人」的意思。<small><ref>《ONE PIECE》漫畫第17卷SBS詢問專欄。</ref></small>

== 人物 ==
特徵是過腰的淺藍色長髮（動畫版的髮色則是水藍色）。她有别於故事中公主温室花朵的形象，身负捍卫自己国家的重责大任。从小好奇心就相当旺盛，爱好和平，讨厌作不必要的纷争，个性坚强也很有责任感，但有时会做一些傻瓜的行为。使用武器为外型有如[[孔雀|孔雀]]羽毛般的机动刀「孔雀鋸子」。

根據時任[[王下七武海|王下七武海]]兼[[多雷斯羅薩|多雷斯羅薩]]王國的國王[[唐吉訶德·多佛朗明哥|唐吉訶德·多佛朗明哥]]表示，8百年前，20個國家的國王聯合成立世界政府後，紛紛攜帶其家眷遷移至聖地馬力喬亞，唯獨當時的娜菲魯塔利一族堅持留在阿拉巴斯坦，統治自己的王國，間接透露薇薇正是當年促成世界政府的君王的後裔<ref>漫畫第722話</ref>。

薇薇沒有惡魔果實能力，也沒有其它特殊能力，她主要的武器是一把名為「孔雀鋸子」的機動刀，由尾指指環繫著幼線操控，外型有如孔雀尾部羽毛般。平日藏於衣袖內。

== 劇中表現 ==
薇薇出身於[[阿拉巴斯坦王國|阿拉巴斯坦王國]]，小時候就了解國家的責任，非常懂事。5歲時跟寇沙第一次見面就打架。在一次决鬥時，遇到刺客，寇沙誓死保護薇薇，之後就成了朋友。

故事6年前，10岁的她曾以公主身份陪同父王娜菲魯塔莉·寇布拉前往聖地馬力喬亞出席「世界會議」。席間，寇布拉公開斥責磁鼓國國王瓦爾波，令他惱羞成怒，後竟公然一巴掌打倒薇薇報復，並出言侮辱。薇薇忍辱負重，反過來冷靜地攔住護衛尹卡萊姆，並大方地向瓦爾波道歉，一場王國之間的糾紛亦由此避免，讓時任磁鼓國護衛隊長的多魯頓深深佩服。

=== 第一部 ===
她在2年前為了得知[[克洛克達爾|沙·克洛克達爾]]的秘密計畫，和阿拉巴斯坦王國護衛隊長尹卡萊姆，一起潛入巴洛克華克這個神祕組織做[[臥底|臥底]]，但還是不幸被揭穿。後來因為[[ONE_PIECE海賊列表#草帽海賊團|草帽海賊團]]的相救下，薇薇和他們一起踏上了短暂的冒險之旅。

於小花園中和[[娜美|娜美]]聯手擊敗了Miss情人節，之後她在磁鼓王國篇從報紙上讀到阿拉巴斯坦王國發生內亂的消息，並委託[[魯夫|魯夫]]等人盡速趕到阿拉巴斯坦；但在得知娜美生病的消息後，決定讓魯夫等人優先為生病的娜美找醫生，跟著一行人來到磁鼓島，當地居民卻因為一年前被黑鬍子海賊團攻擊而不讓同樣身為海賊的一行人登陸，還不小心開槍打傷薇薇，目睹一切的魯夫因憤怒而打算出手時被薇薇阻止，薇薇在出面向居民求情的同時，更訓斥魯夫如果什麼事都想用暴力來解決，不但沒資格當一個領導人，還會危害到娜美的生命安全，此舉讓魯夫決定放下尊嚴跟居民磕頭道歉和求情，才終於獲得眾人的諒解。

當叛亂軍和王國軍隊發生激烈戰爭時，薇薇趁著Mr.7和Miss父親節企圖從鐘塔發射砲彈的時候，趕到鐘塔並成功擊敗Mr.7雙人組。後來草帽海賊團離開阿拉巴斯坦王國時，因不想讓薇薇添麻煩，背對著薇薇和跑得快舉起了畫在左手臂上的「X」記號（代表伙伴間的象徵），就此道别。不管是否離隊，薇薇和跑得快仍被認定是草帽海賊團的夥伴。

司法島事件後草帽海賊團全團被懸賞一事傳到阿拉巴斯坦，國王等人擔心薇薇會因知道妮可·羅賓加入草帽海賊團而傷心，但薇薇卻表示早已知曉。

=== 第二部 ===
「頂點戰爭」結束兩年後，薇薇變得更加成熟，並且在18歲生日當天拿著一份報紙，帶著睫毛與跑得快接受阿拉巴斯坦民眾的盛大歡迎。即將代替臥病在床的父親寇布拉參加「世界會議」。多雷斯羅薩事件落幕後，薇薇獲知草帽海賊團懸賞額提升的消息，跑得快還幫忙剪下攸關草帽海賊團的剪報做為記錄和留念<small> <ref name="ONEPIECE漫畫_815">《ONE PIECE》漫畫第815話扉頁連載。</ref></small>。

前往參加4年一次的世界會議的途中，閱報獲知魯夫等人因為在圓蛋糕島策畫暗殺「BIG MOM」未遂（實為火戰車海賊團策畫暗殺計畫，魯夫等人和火戰車海賊團與太陽海賊團合作解救賓什莫克家族）的消息感到震驚。

世界會議期間，薇薇和先後受魯夫等人幫助的櫻花王國國王多爾頓、Dr.古蕾娃、[[魚人島|魚人島]]龍宮王國公主[[白星|白星]]、[[多雷斯羅薩|多雷斯羅薩]]利克·多爾德三世的皇孫女蕾貝卡等人交流甚歡，獲多爾頓和Dr.古蕾娃協助解圍，擺脫昔日於世界會議惡意欺負她的邪惡黑組織磁鼓王國國王瓦波爾和後者的王妃金德瑞拉的當眾挑釁刁難。途中曾於兩年前的「天龍人事件」和魯夫等人結怨的查爾羅斯聖準備綁架白星作為自己的寵物，薇薇、蕾貝卡、多雷斯羅薩皇族護衛暨草帽大船團旗下成員雷奧、花之國八寶水軍現任領袖雜菜、龍宮王國國王尼普頓試圖為白星解危，但薇薇和蕾貝卡礙於對天龍人出手形同對國家造成威脅而束手無策，最終因為10年前曾被龍宮王國乙姬王妃所救的天龍人'''唐吉訶德·穆斯加魯德聖'''出面教訓查爾羅斯聖，順利為白星和尼普頓解危始平息這場風波。

== 能力 ==
* 孔雀鋸子：薇薇最主要的武器。由尾指指環繫著幼線操控的機動刀，外型有如孔雀尾部羽毛般。平日藏於衣袖內，方便隨時使用。
