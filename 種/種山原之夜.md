《'''種山原之夜'''》（{{lang-ja|種山ヶ原の夜}}）是2006年由日本[[吉卜力工作室|吉卜力工作室]]的[[男鹿和雄|男鹿和雄]]執導、改編自作家[[宮澤賢治|宮澤賢治]]同名著作的動畫。

== 概要 ==
作品的創作意圖為男鹿和雄在參與1997年吉卜力劇院動畫《[[魔法公主|魔法公主]]》完成後所想出，<ref name="interview">{{Cite web |url=http://www.ghibli.jp/taneyamagahara/35interview/ |script-title=ja:男鹿和雄インタビュー |publisher=[[吉卜力工作室|吉卜力工作室]] |date=2006-06 |accessdate=2012-09-13 |language=ja}}</ref>並為他個人首次擔任動畫導演的作品。<ref name="Kakunodate">{{Cite web |url=http://www.ghibli.jp/taneyamagahara/00info/000832.html#more |script-title=ja:角館イベント・ミニレポート |publisher=吉卜力工作室 |date=2006-07-28 |accessdate=2012-09-13 |language=ja}}</ref>

男鹿和雄在畫面上以貼近故事[[繪本|繪本]]的風格畫作，並選用日本民間文化紙上戲劇「[[紙芝居|紙芝居]]」來作為動畫中講解劇情的方式。<ref name="interview" />在台詞方面則以[[岩手縣|岩手縣]]的[[方言|方言]]表現，並請男演員[[山谷初男|山谷初男]]擔任台詞的朗讀。<ref name="interview" />

《種山原之夜》在成品製作完畢後，除在男鹿和雄故鄉[[秋田縣|秋田縣]]當地劇團、宮澤賢治故鄉[[岩手縣|岩手縣]]當地記念館及[[吉卜力美術館|吉卜力美術館]]外，<ref name="Kakunodate"/><ref>{{Cite web |url=http://www.ghibli.jp/taneyamagahara/00info/000880.html |script-title=ja:宮沢賢治イーハトーブ館にて |publisher=吉卜力工作室 |date=2006-09-28 |accessdate=2012-09-13 |language=ja}}</ref><ref>{{Cite web |url=http://www.ghibli.jp/taneyamagahara/00info/000860.html |script-title=ja:ジブリ美術館の夜 |publisher=吉卜力工作室 |date=2006-08-11 |accessdate=2012-09-13 |language=ja}}</ref>
未在其它地方電影院上公開。

== 劇情簡介 ==
在日本岩手縣[[種山原|種山原]]裡有幾名貧困的農夫，在該處一帶的高原處從早到晚忙著割草。

當他們晚上圍著營火休息時，其中一名叫做伊藤的農夫青年夢到了他與林野署的官員、以及住在山林裡的樹靈，為了是否需要砍伐山林樹木作為木炭一事爭論的奇妙的夢。
