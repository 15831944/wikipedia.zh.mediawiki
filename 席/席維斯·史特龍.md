== 生平 ==
父親是[[義大利人|義大利人]]，母親是[[猶太人|猶太人]]歌舞團演員。他的弟弟[[小法蘭克·史特龍|小法蘭克·史特龍]]（Frank P. Stallone, Jr. ）是一名演员、歌手兼[[吉他手|吉他手]]。

史泰龍在1970年成為電影演員，起初只是擔當小角色，直到1976年史泰龍根据他的現實生活編寫了劇本《[[洛奇|洛奇]]》並參與演出後造成舉世轟動，從此史特龍知名度大增。其後史特龍在之後數集《[[洛奇|洛奇]]》電影中持續自编自演自導，並聲名日隆。1982年他又自編自演《[[第一滴血|第一滴血]]》及其主角[[約翰·藍波|約翰·藍波]]，這齣電影再度備受注目，成為當年美國十大巨片。

1990年代，史泰龍開始跳脫硬派動作影星的路線，並且塑造出另一類型的英雄來吸引觀眾。當中《[[絕嶺雄風|-{zh-hans:巅峰战士;zh-hk:絕嶺雄風;zh-tw:巔峰戰士;}-]]》、《[[越空狂龍|-{zh-hans:超级战警;zh-hk:越空狂龍;zh-tw:超級戰警;}-]]》、《[[特警判官|-{zh-hans:超时空战警;zh-hk:特警判官;zh-tw:超時空戰警;}-]]》、《[[龍出生天|-{zh-hans:十万火急;zh-hk:龍出生天;zh-tw:十萬火急;}-]]》等都是史特龍近代成功的電影。

2006年，史特龍演出《[[洛基(電影)|洛基]]》第六集，《洛基：勇者無懼》，這是十五年來再度主演《[[洛基(電影)|洛基]]》電影系列。

2007年，相隔20年的《[[第一滴血|第一滴血]]》系列推出第四集，由席維斯·史特龍自導兼自演，由本片可看出60歲的史特龍依舊活力十足。

2010年，史特龍自编自導自演《[[浴血任務|浴血任務]]》其主角[[巴尼·羅斯|巴尼·羅斯]]，取得電影商業票房上的成功，並開始推行系列續集電影。

2013年，史泰龙与阿诺联袂主演越狱题材动作片《[[金蝉脱壳|金蝉脱壳]]》并和罗伯特·德尼罗二度搭档主演《[[旗鼓相当|旗鼓相当]]》。

== 家庭 ==
史泰龍經歷了三段婚姻，1974年與Sasha Czack結婚，育有兩名兒子：Sage及曾患自閉症的Seargeoh，1985年離婚。同年和丹麥模特兒兼演員碧姬妮遜（Brigitte Nielsen）結婚，1987年離婚。1997年與模特兒Jennifer Flavin結婚，婚姻維持至今，並育有三名女兒：Sophia、Sistine 及 Scarlet。

== 政治立場 ==
席维斯·史特龍與曾任加州州長的[[阿諾·史瓦辛格|阿諾·史瓦辛格]]一樣支持[[共和黨_(美國)|共和黨]]，他也是[[喬治·沃克·布希|喬治·沃克·布希]]最欣賞的演員之一。巧合的是，他們是同年同月同日出生。史特龍曾參加小布希的就職典禮，亦曾獲邀到白宮作客。2008年，他支持共和黨總統候選人[[約翰·麥凱恩|約翰·麥凱恩]]。
