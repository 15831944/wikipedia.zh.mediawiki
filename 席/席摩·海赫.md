'''席摩·海赫'''（{{lang-fi|Simo Häyhä}}，{{bd|1905年|12月17日|2002年|4月1日|catIdx=H}}）是一位已故[[芬蘭陸軍|芬蘭陸軍]][[狙擊手|狙擊手]]，被[[蘇聯|蘇聯]]部隊稱呼為「白色死神」（{{lang-ru|Белая Смерть}}；{{lang-fi|Valkoinen kuolema}}）。原本是農民兼獵人的海赫於1925年開始服兵役，接著加入芬蘭志願民兵{{link-en|白衛隊|White Guard (Finland)}}。[[第二次世界大戰|第二次世界大戰]]期間，海赫於1939年至1940年在[[冬季戰爭|冬季戰爭]]的[[柯拉戰役|柯拉戰役]]中對抗[[蘇聯紅軍|蘇聯紅軍]]，用{{link-en|SAKO|SAKO}}M/28-30步槍和[[索米M1931衝鋒槍|KP-31索米冲锋槍]]創下505名至542名經確認的擊殺記錄。1940年3月6日，海赫臉部中彈重傷昏迷，等他清醒時柯拉戰役已經[[莫斯科和平协定|結束]]。戰後他改當[[駝鹿|駝鹿]]獵人與育犬員，並在芬蘭小鎮[[魯奧科拉赫蒂|魯奧科拉赫蒂]]度過晚年。2002年4月1日，海赫於[[哈米纳|哈米纳]]一所退伍軍人護理之家逝世，享年96歲。

== 早年生涯 ==

席摩·海赫於1905年12月17日出生在[[芬蘭大公國|芬蘭大公國]][[維堡省|維堡省]]靠近[[俄羅斯|俄羅斯]]邊境的[[勞特耶爾維|勞特耶爾維]]（今屬芬蘭[[南卡累利阿區|南卡累利阿區]]）<ref name="Geni"/>。他生在一個傳統[[信義宗|信義宗]]農村家庭，是家中第七個孩子<ref name="Geni"/>，綽號「Simuna」。海赫是一位農民兼獵人，長期待在野外讓他打好了射擊底子<ref name="Mosin-Nagant.net"/><ref name="Historyextra"/><ref name="Heiskanen"/>。他在17歲時加入芬蘭志願民兵{{link-en|白衛隊|White Guard (Finland)}}，鍛鍊了射擊能力<ref name="Heiskanen"/><ref name="Yle"/>。1925年，海赫加入芬蘭軍隊服一年的兵役，服完兵役時海赫已升上[[下士|下士]]<ref name="Mosin-Nagant.net"/>。海赫在維堡省的射擊比賽中屢獲佳績。據報導，他家中擺滿了射擊比賽的獎盃<ref name="Gilbert"/>{{rp|88}}。

== 軍中生涯 ==

[[第二次世界大戰|第二次世界大戰]]期間，芬蘭與[[蘇聯|蘇聯]]於1939年至1940年間爆發[[冬季戰爭|冬季戰爭]]，海赫在其中的[[柯拉戰役|柯拉戰役]]擔任[[芬蘭陸軍|芬蘭陸軍]]第34團第6連的[[狙擊手|狙擊手]]，對抗[[蘇聯紅軍|蘇聯紅軍]]。海赫全身穿著白色迷彩服，在介於{{convert|-40|°C}}至{{convert|-20|°C}}間的低溫作戰。由於時任[[蘇聯最高領導人列表|蘇聯最高領導人]][[約瑟夫·維薩里奧諾維奇·史達林|約瑟夫·維薩里奧諾維奇·史達林]]在1930年代對軍事人才進行[[大清洗|大整肅]]，紅軍的秩序混亂，且在大多數戰役中，蘇聯軍隊都沒有分發到白色迷彩服，使得芬蘭方的狙擊手能輕易發現他們<ref name="Trotter"/>{{rp|145-146}}。

海赫使用的步槍是白衛隊發放的芬蘭國產步槍{{link-en|SAKO|SAKO}}M/28-30。該步槍是[[莫辛-納甘步槍|莫辛-納甘步槍]]的變型，身高偏矮、高160公分的海赫與之搭配得很好<ref name="Stirling2012"/>{{rp|79-80}}<ref name="ETtoday新聞雲"/>。他只使用槍身上的鐵製[[照門|照門]]與[[准星|準星]]。因為不使用[[瞄准镜|瞄準鏡]]，所以他不需要像有使用的狙擊手一樣抬頭對準瞄準鏡，從而縮小自身輪廓，大幅降低暴露及中彈的機率。不使用瞄準鏡還讓他從不用擔心日間作戰時會被陽光暴露自己的位置，因為瞄準鏡會反射光源，很容易暴露狙擊手的位置。海赫不熟悉其他步槍，所以不願意換用蘇聯的步槍（如m/91-30 PE或PEM）<ref name="Stirling2012"/>{{rp|79-80}}。海赫經常在他的狙擊點前方堆起雪堆來隱蔽身體，同時減少步槍開火時揚起的雪<ref name="Stirling2012"/>{{rp|79-80}}。他在狙擊時還會將雪含在口中，避免呼出的熱氣暴露行蹤<ref name="Stirling2012"/>{{rp|79-80}}。據傳，海赫的狙擊距離最遠達450公尺<ref name="Helsingin sanomat"/>。

[[File:Simo_hayha_honorary_rifle.jpg|thumb]]

在不到100天內，海赫平均每天擊殺5名蘇聯士兵<ref name="Jowett2006"/>{{rp|44-45}}<ref name="Pegler2006"/>{{rp|167}}<ref name="FareySpicer2009"/>{{rp|117-118}}。1939年12月21日，海赫創下他的單日最高擊殺數，共殺了25人<ref name="Saarelainen"/>。當時狙擊手的擊殺數需要狙擊手自己計算，戰友也要予以確認才算<ref name="Ilta-Sanomat / Kotimaa"/>。據書籍《白色狙擊手：席摩·海赫》（{{lang|en|''The White Sniper: Simo Häyhä''}}）的資料，海赫用M/28-30狙殺了259名蘇聯士兵<ref name="Saarelainen"/>。然而，由於被狙殺者都在蘇聯方，導致海赫的實際狙殺數無從考證<ref name="MarjomaaHäyhä"/>。1940年2月17日，海赫獲所屬單位的指揮官-{zh-tw:A·史雲遜;zh-cn:A·斯文松;}-（{{lang|sv|A. Svensson}}）頒發「榮譽步槍」<ref name="Saarelainen"/>。另外，海赫也是[[索米M1931衝鋒槍|KP-31索米冲锋槍]]高手，且他在冬季戰爭期間的擊殺數大多來自這把槍<ref name="Mosin-Nagant.net"/><ref name="Yle2"/>。被問到他是如何成為如此厲害的射手時，海赫回答道「多練習」<ref name="ETtoday新聞雲"/>。

[[File:Simo_hayha_second_lieutenant_1940.png|thumb]]

海赫令蘇聯軍聞風喪膽，蘇聯軍稱他為「白色死神」（{{lang-ru|Белая Смерть}}；{{lang-fi|Valkoinen kuolema}}）<ref name="Ilta-Sanomat / Kotimaa"/><ref name="ETtoday新聞雲2"/>。蘇聯軍使用{{link-en|反狙擊手戰術|Counter-sniper tactics}}和炮擊等，努力想殺掉海赫<ref name="Saarelainen"/>。一次，有一枚砲彈落在海赫的狙擊點附近，讓他受了點輕傷<ref name="Historyextra"/>。1940年3月6日，海赫在一場近距離突擊戰鬥中被紅軍的一枚{{link-en|爆破子彈|High-explosive incendiary/armor-piercing ammunition}}擊中左下顎，重傷昏迷。他到3月13日才逐漸清醒過來，剛好與[[莫斯科和平协定|柯拉戰役結束]]同一天<ref name="Saarelainen"/><ref name="Heiskanen"/><ref name="Brantberg"/><ref name="NenyeMunterWirtanenBirks"/>。海赫的傷口共開了26刀<ref name="Historyextra"/>。多年後，海赫表示，他在被擊中的當下仍拾起他的步槍擊殺了那名射中他的蘇聯士兵<ref name="Mosin-Nagant.net"/>。戰役結束後不久，芬蘭[[陸軍元帥|陸軍元帥]][[卡爾·古斯塔夫·埃米爾·曼納海姆|卡爾·古斯塔夫·埃米爾·曼納海姆]]元帥將他從[[下士|下士]]直接晉升為[[少尉|少尉]]<ref name="Feist2012"/>，是芬蘭軍事史中晉升最快的軍人<ref name="Yle"/>。他還獲得自由十字勳章、自由勳章和柯拉戰役十字勳章<ref name="HS"/><ref name="Helsingin sanomat"/>。據統計，海赫在冬季戰爭共擊殺了505名至542名敵軍<ref name="NenyeMunterWirtanenBirks"/><ref name="The Daily Telegraph"/><ref name="Forces Network"/>，是第二次世界大戰期間擊殺數最多的狙擊手<ref name="Sniper Central"/>。

== 戰後生涯與去世 ==

[[File:Simo_Häyhä's_grave.jpg|thumb]][[南卡累利阿區|南卡累利阿區]][[魯奧科拉赫蒂|魯奧科拉赫蒂]]教堂墓園的墓碑]]

中彈後的海赫失去了下[[顎|顎]]大部分的功能，且左臉頰也沒了。不過他傷勢痊癒的狀況甚佳，[[第二次世界大戰|第二次世界大戰]]結束後他成為成功的[[駝鹿|駝鹿]]獵人與育犬專家，還曾與前[[芬蘭總統|芬蘭總統]][[烏爾霍·吉科寧|烏爾霍·吉科寧]]一同打獵一天<ref name="Stirling2012"/>{{rp|79-80}}<ref name="E-Karjala maakuntaportaali"/><ref name="Daily Mail"/>。海赫故鄉的鄰居送給他一座農場，其位在芬蘭東南方靠近俄羅斯邊界的小鎮[[魯奧科拉赫蒂|魯奧科拉赫蒂]]。海赫在農場裡蓋了一棟小房子，度過晚年<ref name="Yle3"/>。2000年，海赫住進[[哈米纳|哈-{米}-納]]的一所退伍軍人護理之家，並於隔年加入屈米殘疾退伍軍人協會<ref name="Historyextra"/><ref name="Yle3"/>。《[[赫爾辛基日報|赫爾辛基日報]]》採訪海赫時問他是否對於那些槍下亡魂感到內疚，海赫回答：「我不過只是盡責地做到交代我做的事」<ref name="Helsingin sanomat"/>。他也曾說過，「打仗不是一次愉快的經驗，但我們不上戰場的話，那誰來保護這片土地？」<ref name="Historyextra"/>2002年4月1日，海赫於退伍軍人護理之家逝世，享年96歲<ref name="Geni"/><ref name="Feist2012"/><ref name="Dead"/>，遗體葬於芬蘭[[南卡累利阿區|南卡累利阿區]]的[[魯奧科拉赫蒂|魯奧科拉赫蒂]]<ref name="Geni"/>。海赫是芬蘭人眼中的戰爭英雄<ref name="Mosin-Nagant.net"/><ref name="ETtoday新聞雲2"/>，在芬蘭還有一座席摩·海赫博物館<ref name="Yle3"/><ref name="Yle4"/>。

== 相關 ==

有不少人努力想找到海赫的槍。據推測，海赫中彈後將步槍遺落在戰場上。至今，海赫的槍仍未尋獲<ref name="Yle2"/>。2018年3月，海赫傷後撰寫的回憶錄首度被公开<ref name="Yle3"/><ref name="Yle4"/>。他的生平事蹟曾改拍成電影《白色死神》（{{lang|en|''The White Death''}}），由大衛·麥克艾洛伊（{{lang|en|David McElroy}}）執導<ref name="Film"/>。海赫也是瑞典[[力量金屬|力量金屬]]樂團[[薩巴頓|薩巴頓]]收錄在專輯《{{link-en|紋章 (專輯)|Coat of Arms (album)|紋章}}》的一首歌曲《白色死神》（{{lang|en|''White Death''}}）的靈感來源<ref name="Song"/>。
