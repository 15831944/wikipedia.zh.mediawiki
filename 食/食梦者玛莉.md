== 故事简介 ==
擁有探视颜色并判断夢境能力的高中生'''藤原夢路'''，在暑假的某一天遇見了從「幻界（夢）」闖入「現界（現實）」的少女'''瑪莉'''。因此夢路被捲進在「幻界」和「現界」两个截然不同的境界裡，为了帮助瑪莉，而展開與多數夢魔之間的戰鬥。

=== 橘家住户 ===
; {{nihongo|藤原夢路|藤原 夢路（ふじわら ゆめじ）||聲：[[岡本信彦|岡本信彦]]、幼年：[[慶長佑香|慶長佑香]]}}
:生日:7月7日 身高:168cm 體重:53kg（巨蟹座）
: 本作[[男主角|男主角]]。高中2年级。家庭與橘家的關係很好，現因父母在外工作的关系寄宿在橘家。特点是头上的[[呆毛|呆毛]]。不会[[游泳|游泳]]。喜欢[[特摄|特摄]]的英雄，连自身的作品中也能隐约地受此影响。
: 可探测他人夢境的能力，但準確率不高（勇魚表示甚至不到30%），但唯獨代表惡夢的黑色能夠百分之百命中。
: 被約翰找上作為容器的人選，連續做了好幾場噩夢。而在被約翰追擊時被巧遇的瑪莉所救，逃過了一劫。後將昏倒的瑪莉帶回家中，在了解她的情況後，提出要協助她的提案。後在市松事件後正式定下協力契約。
: 原本沒有被夢魔選為容器的人類是無法進入白日夢中，但因為自身所擁有的不明力量而可以進入白日夢中。在白日夢中，能使用「武裝明晰夢」複製並且使用曾經看過的夢魔的能力，也能把夢魔的白日夢強制顯現。另外，身體在幻界所受到的傷害恢復也異常快速，似乎都是那不明力量的所帶來的影響，被認為與夢魔有所關連。而隨著使用的次數漸漸增加，右手也似乎漸漸發出疼痛。目前這夢魔力量的來源與真正的力量依舊是個謎，但據葛麗歐妮所言，這股力量的「聲紋」與海格力斯完全一樣。
: 本人對“幼年時没有遵守與某人的承諾”因而後悔，因此有了一旦訂下承諾就絕對要遵守的信念。右手上的傷疤有時會痛，傷疤的位置也和瑪莉一模一樣。而勇魚見到夢路帶著瑪莉回來時，驚訝之餘，卻也有著兩人在一起是理所當然的感覺。由此的很多跡象，暗示了兩人可能在小時候就見過面了。

; {{nihongo|玛莉・夢魔|メリー・ナイトメア、Merry Nightmare||聲：[[佐倉綾音|佐倉綾音]]}} 
:生日:不明 身高:147cm 三圍:B73 W54 H77
: 本作女主角，迷失在現世而無法回去「幻界」的夢魔少女，別名'''食夢者'''（{{lang|ja|夢喰い}}），真名为'''门卫'''（{{lang|ja|門番（シープ、Sheep）}}）。瞳形为细长的四角形。喜歡吃甜甜圈（本人表示：甜甜圈乃是現界的至宝），不喜歡碳酸飲料。似乎不受烏鴉等鳥類的喜愛。
: 从外表来看像是从小学高年级到中学生程度左右。一頭齊肩的短髮，而尖耳朵被头发和帽子遮盖着，帽子和衣服上都有類似閉眼模樣的圖案。右臂戴着护套以隐藏过去的伤疤，不過瑪莉本人似乎並不在意被人看見手上的傷疤。
: 不知為何出現在現世，也沒有到達現世以前的記憶，只有渴望回去的感覺。為找尋回去幻界的辦法而尋找其他夢魔，碰巧解救了被約翰追擊的夢路。后与梦路互相合作结为「契约」关系，以藉由梦路的能力協助搜尋夢魔，並找到回去的線索。对契约者梦路拥有好感且非常信任，常常会因为[[傲娇|傲娇]]而表达不了真实的心情。現在與夢路一樣借住在橘家，平時會在橘父親開的咖啡店「STO」當任服務生。
: 普通战斗以肉搏战为主。外表雖是纖細，但力氣卻超乎常人的大。受到震撼而發揮出真正實力時，手上會出現類似鑰匙串的武器，會發揮比平常更驚人的力量，其力量強大到甚至能破壞對方幻界中的所有物品，致使对手无力化，也就是破壞對手的DAY DREAM（白日夢）（不過正確來講，是被「遣返」回幻界）。但是當使出力量時，玛莉本人是處於腦袋空白的狀態，回過神來時，並不清楚發生什麼事。後來在奴娃之战過後已能控制。
: 認為沒有夢路的世界是不需要存在的，曾因為夢路被海格力斯重創臨死而暴走，展現出現極為可怕的力量，不只是連自己的身體都無法承受，還差點把「門」也一同破壞掉。後來被傷口恢復的夢路阻止。
: 碰到悲觀的情況時，常把「真是沒夢、沒希望」或者是「夢和希望都沒了」亦或者是「沒夢、沒希望了……」等句掛在嘴邊。
: 由於真正的別名「門衛」的關係，而有著把夢魔遣返回去的力量，曾被蘭保夫的話語迷失方向，而無法發揮原本的實力。
: 曾表示自己在現界看過10次夏季拍賣的掛旗，夢路推測她從10年前就流落到現界，並為了查出返回幻界的方法而不斷地尋找夢魔。似乎小時候就認識夢路，但已經忘記了。
: 自奴娃之战时，回想起了自己本是在“幻界”和“现界”分界中掌管“界境之门”的管理者，但除了身分以外的事還是想不起來。<br />她是能送回幻界里的梦魔才产生的特别梦魔。也可以用“锁上大门”的方式来阻止特定梦魔再度进入现界。固定台词是“前方禁止通行喔”。
: 设计灵感出自羊。名字源于美国童谣《[[Mary_Had_a_Little_Lamb|Mary Had a Little Lamb]]》（{{lang-zh|玛丽有只小羊羔}}）。

=== 夢魔 ===
; {{nihongo|約翰・杜|ジョン・ドゥ、John Doe||聲：[[中田讓治|中田讓治]]}} 
: 劇情-{zh-hans:开始;zh-hk:序盤;zh-tw:序盤;}-讓夢路每晚都做惡夢的主使者，別名'''追跡者'''（{{lang|ja|チェイサー}}、Chaser），身高191cm，外表穿着黑色緊身衣及紅色長袍，戴着面具盖住自己的面孔。目的是為了獲得前往「現界」所需的「容器」，但卻在親自出馬的那天遭到意外介入的瑪莉阻撓，使得计划没有成功。然而又对梦路的执念和玛莉的出身感兴趣，治疗了梦路的伤痕，并予以协助。稱自己是追隨真相的那一方。其實很在意夢路等人。

== 用語 ==
;{{nihongo|夢魔|夢魔（むま）}}
: 幻界(亦即所谓夢世界)的居民。各自擁有自己的幻界。
: 由于本身不能从幻界出去，只能附体于人类進入现界(瑪莉屬於極為少數的例外)。

;{{nihongo|幻界（梦）|幻界（ゆめ）}}
: 以梦魔为核心形成，拥有独自法则的多样空间集合体。每個夢魔都有各自的幻界庭院，庭院的景象反映著夢魔本身的能力與個性。

;{{nihongo|現界（现实）|現界（うつつ）}}
: 人类生活的现实世界，夢魔之间的称呼。

;{{nihongo|容器|器}}
: 侵入現界的梦魔依靠附身的对象，不限定於人類，只要是會做夢的動物皆可。
: 依據夢魔的性格，梦魔與容器的關係分為強行霸佔與和平共處兩類。前者會囚禁容器的人格使其沉睡，隨心所欲的使用容器的身體。後者會與容器達成協議，建立友好的共生關係。
: 一旦依附容器的夢魔死亡，容器的夢想也會消滅，會完全失去人生的目標與為此付出的上進心。

;{{nihongo|[[白日梦|白日梦]]|白昼夢（デイドリーム、Day Dream）}}
: 由夢魔自己展开的幻界，侵蚀周围夢魔容器的入侵現象。

;{{nihongo|地狱滑板人·格力卓|フルヘルボーダーグリッチョ||聲：[[川田紳司|川田紳司]]}}
: 夢路热衷收看的特摄英雄节目。
