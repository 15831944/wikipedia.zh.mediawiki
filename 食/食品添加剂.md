'''食品添加剂'''是为了保持味道或增强口感、改善外观添加到食物中的物质。
一些添加剂已经使用了几个世纪；例如，（用[[醋|醋]]）[[醃|腌制]]、[[食盐|盐]]腌来保存食物（如[[醃肉|醃肉]]），[[糖果|糖果]]的保存以及用[[二氧化硫|二氧化硫]]来保存[[葡萄酒|葡萄酒]]。随着二十世纪下半叶加工食品的出现，引入了越来越多的天然和人工合成的添加剂。

== 产业状况 ==
* 据统计，全球食品业每年花在食品添加剂商的[[投资|投资]]高达200亿美元。
* 平均每人每年要吃进6至7公斤的食品添加剂。

== 管理 ==
在20世纪50年代以前，食品添加剂採用負面表列的方式，凡是不在明令禁止的名单上，就可以添加到食品中。现在大多数国家都规定，只有获得许可的添加剂才能在食品中使用。根據食品藥物管理局發布之[http://consumer.fda.gov.tw/Files/food/law/%E9%99%84%E8%A1%A8%E4%B8%80%20%20%E9%A3%9F%E5%93%81%E6%B7%BB%E5%8A%A0%E7%89%A9%E4%BD%BF%E7%94%A8%E7%AF%84%E5%9C%8D%E5%8F%8A%E9%99%90%E9%87%8F_1020531.pdf 食品添加物使用範圍及限量]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}，其中規定了各類食品添加物在食品中所佔比例，例如[[己二烯酸|己二烯酸]]（Sorbic Acid），則逐一列出可使用於魚肉煉製品、肉製品、海膽、魚子醬、花生醬、醬菜類、水分含量25％以上（含25％）之蘿蔔乾、醃漬蔬菜、豆皮豆乾類及乾酪；並且也將用量數據化，例如"以Sorbic Acid計為2.0g/kg以下"。

== 争议 ==
=== 作用 ===
* 贊成方認為：使用食品添加剂是为了防止[[食物|食物]]變質，保護[[消费者|消费者]]免遭[[食物中毒|食物中毒]]。
* 反对方認為：用于阻止食物变质的添加剂只占食品添加剂总量的1%。大约90%的食品添加剂是装饰性的，这其中有染色的，[[甜味剂|甜味剂]]，及[[食用香料|食用香料]]。此外还有[[乳化剂|乳化剂]]，稳定剂之类的食品加工助剂。

=== 安全 ===
* 在允许使用的添加剂中，有很多人有异议。如在[[欧盟|欧盟]]，有人认为欧盟对食品[[香料|香料]]的管理不如对其它食品添加剂那样严。目前多数国家因在食品中香料加入非常少，所以对食用香料不采取逐个检测安全性的做法。这引起了一些人的不满。
* 对于国家卫生部门的检测方法也有人批评，他们认为几乎所有的检测都是在[[老鼠|老鼠]]身上进行实验，但实验结果是否同样适用于人还是个未知数。
* 另外还有人对大多数相關研究是由生产食品添加剂的公司进行的，这一状况不满，怀疑研究结果的公正性。

=== 趨勢 ===
2010年間自英國通路商發起自主推廣clean label，少添加甚至無添加的產品標誌蔚為風潮；然而多數消費者並不具備判斷標示屬實與否的能力，因此而起的食品詐欺儼然是食品衛生安全管理的重要議題，國際間也陸續成立國際或是區域層級的潔淨飲食推廣組織，其中以亞太無添加餐飲食品發展協會(Anti Additive Association)最具規模。

== 食品添加剂的種類 ==
{{main|{{link-en|List_of_food_additives|食品添加劑列表}}}}
食品添加劑的目的主要為：延長保存期限用、視覺調整、味覺調整、改善食品品質、提高營養價值及方便製造等。其按照功能性，可大致分為十七種：<ref>{{cite web |title=食品添加物使用範圍及限量暨規格標準 |url=http://www.fda.gov.tw/people_laws.aspx?peoplelawssn=1128&keyword=&classifysn=62 |date=2011-05-30 |work=[[行政院|行政院]][[衛生署|衛生署]][[食品藥物管理局|食品藥物管理局]] |accessdate=2011-06-18}}</ref><ref>{{cite book |editor=劉麗雲 |title=《食品衛生與安全》 |year=2010年12月 |location=臺北市 |publisher=秀威資訊科技 |series=實踐大學數位出版合作系列 |ISBN=978-986-221-614-9 |url=http://books.google.com.tw/books?id=qbB6QORqK74C&printsec=frontcover&dq=%E9%A3%9F%E5%93%81%E8%A1%9B%E7%94%9F%E8%88%87%E5%AE%89%E5%85%A8&hl=zh-TW&ei=EXj8TfDTGYbyvwO6wNyRAw&sa=X&oi=book_result&ct=result&resnum=1&ved=0CDcQ6AEwAA#v=onepage&q&f=false |accessdate=2011-06-18}}</ref>
* [[防腐劑|防腐劑]]（在部分情況會稱為'''品質改良劑'''，常被添加在即食類食品，例如[[便當|便當]]、[[麵包|麵包]]、[[麵條|麵條]]）
:例如：[[亚硝酸钠|亚硝酸钠]]、[[亞硝酸|亞硝酸]]、[[山梨酸鉀|己二烯酸鉀]]、[[己二烯酸|己二烯酸]]、[[丙酸钠|丙酸钠]]、[[对羟基苯甲酸酯|对羟基苯甲酸酯]]（尼泊金酯）
* 殺菌劑
* [[抗氧化劑|抗氧化劑]]
* [[漂白劑|漂白劑]]

* 著色劑

* [[香料|香料]]

:* 酸味劑、[[酸度调节剂|酸度调节剂]]（PH值調整劑）

下表列出食品添加剂，并介绍它们的功能分类、中文名称（英文名）、化学名/分子式、天然与否、典型食品、历史/价位/备注等。

===方便食品===
[[方便食品|方便食品]]由于保质期和商家促销的需要，普遍会使用食品添加剂，比如[[方便面|方便面]]和[[火腿肠|火腿肠]]中广泛使用防腐剂、色素等，是人民群众最容易接触到的食品添加剂。

===防腐剂===

* [[亚硝酸钠|亚硝酸钠]]（Sodium_nitrite），别名250号添加剂，人工合成，应用广泛，历史悠久， NaNO<sub>2</sub>。
* [[山梨酸钾|山梨酸钾]]（Potassium_sorbate），又名2,4-己二烯酸钾，人工合成，应用广泛，价位较高， C<sub>6</sub>H<sub>7</sub>O<sub>2</sub>K。

===着色剂（色素）===

* [[红曲米|红曲米]]，又名红粬，天然，《本草纲目》记载，常用于[[火腿肠|火腿肠]]、肉类酒类。
* [[辣椒红素|辣椒红素]]，天然色素。
* [[核黄素|核黄素]]，又称维生素B2，天然色素，常用于[[方便面|方便面]]。
* [[柠檬黄|柠檬黄]]，人工合成的偶氮类酸性染料，诞生于1884年。

===甜味剂===

* [[安赛蜜|安赛蜜]]（Acesulfame potassium），人工合成，化学名：乙酰磺胺酸钾，C4H4KNO4S。常见于袋装[[榨菜|榨菜]]。
* [[甜蜜素|甜蜜素]]（Sodium cyclamate），又称甜精，人工合成，化学名：环己基氨基磺酸钠。

===增味剂===

* [[谷氨酸钠|谷氨酸钠]]也称为MSG或味精，半天然增味剂，1909年人工制备。应用广泛。

===增稠剂===
* [[卡拉胶|卡拉胶]]（Carrageenan），又称鹿角菜胶，是从海洋红藻提取的多糖的统称，是多种物质的混合物。应用广泛，常见于果酱/肉酱。
* [[黄原胶|黄原胶]]（Xanthan gum），俗称玉米糖胶，是一种经由野油菜黄单孢菌发酵产生的复合多糖体。在美国，它通常是经由玉米淀粉所制造。常见于[[方便面|方便面]]。
* [[瓜尔胶|瓜尔胶]]（guargum），是从植物瓜尔豆中提取的一种高纯化天然多糖。常见于[[方便面|方便面]]。

== 檢測 ==
许多食品添加剂吸收光谱紫外区或可见光区域的辐射。使用外部校准可以将此吸光度用于测定样品中添加剂的浓度。但是，添加剂可能会一起出现，而一种添加剂的吸光度会干扰另一种添加剂的吸光度。预先的分离步骤是有必要的，添加剂们会首先用高压液相色谱法（HPLC）分开，然后由紫外线或可见光检测器在线监测。
