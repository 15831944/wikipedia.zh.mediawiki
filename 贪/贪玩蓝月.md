《'''贪玩蓝月'''》是一款由浙江盛和网络科技有限公司所研发的[[偽三維|2.5D]][[大型多人在线角色扮演游戏|大型多人在线角色扮演]]类[[网页游戏|网页游戏]]（原名《'''蓝月传奇'''》）<ref>{{Cite web|url=http://www.sh-game.com/|title=盛和游戏|accessdate=2018-02-02|work=www.sh-game.com}}</ref>，由江西贪玩信息技术有限公司于2016年4月23日正式开始运营<ref name=":0" />。

== 网络爆红 ==
2017年，由香港著名演员[[古天樂|古天乐]]、[[張家輝|张家辉]]、[[陳小春|陈小春]]等人所代言的游戏广告在网络上爆红，成为一个[[網路爆紅事物|网络迷因]]。广告中几乎每一句话都受到了大量的[[恶搞|恶搞]]，如“张家辉”恶搞为“渣渣辉”、“陈小春”搞为“沉笑蠢”、“古天乐”恶搞为“股添咯”（均为[[广式普通话|香港普通话]]谐音）和“古天乐绿了”（本意为游戏内[[游戏装备|装备]]颜色，之后有被戴上了[[绿帽子|绿帽子]]的意味）等。[[网民|网民]]利用广告素材进行了一系列[[音MAD|鬼畜]]视频与[[表情包|表情包]]的制作。

在游戏爆红之后，在游戏社区[[TapTap_(游戏社区)|TapTap]]上的《贪玩蓝月》手游被网民提交大量恶意好评，评论区也出现众多搞笑段子，直接导致游戏在TapTap上的评分超过8.3，随后TapTap官方发布公告并删除了高星评价。<ref>{{Citenews|url=https://www.taptap.com/topic/2214295|title=[公告] 关于恶意评价5星的情况处理 - TapTap|newspaper=TapTap|accessdate=2018-02-02}}</ref>

== 文化部责罚 ==
2018年1月25日，[[中华人民共和国文化部|中华人民共和国文化部]]加强对网络游戏的监管力度并披露《贪玩蓝月》的广告存在禁止内容，包括女性裸露身体的画面、低俗声音、文字以及动作等。<ref>{{Cite web|url=http://news.syd.com.cn/system/2018/01/24/011569080.shtml|title=文化部严查网游违法违规公布贪玩蓝月等6起典型案件_天天315_新闻中心_沈阳网|accessdate=2018-02-02|work=news.syd.com.cn}}</ref>文化部随后要求江西贪玩信息技术有限公司进行整改并罚款。<ref>{{Cite web|url=http://www.mcprc.gov.cn/whzx/whyw/201801/t20180125_831029.htm|title=文化部加强对网络游戏违规内容、经营活动监管执法力度|accessdate=2018-02-02|work=中华人民共和国文化部}}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 直播涉嫌虚假宣传 ==
2018年1月底，贪玩游戏的[[Bilibili|哔哩哔哩]]账号发布视频，宣布将于31日进行[[現場直播|直播]]，届时将有古天乐直播游玩《贪玩蓝月》<ref>{{Cite web|url=https://www.bilibili.com/video/av18886562/|title=大家好!我是古天乐,1月31日我来直播玩《贪玩蓝月》_哔哩哔哩（现已删除）|accessdate=2018-02-02|work=www.bilibili.com}}</ref>。{{or|视频发布之后}}即引发网络和媒体的大量讨论。直播当日早晨，数百万网友进入直播间观看直播，同时邀请了[[虎牙直播|虎牙直播]]的局长解说主持并称是双平台直播，然而直播主角古天乐12点40分左右才抵达直播现场。古天乐直播中也仅仅是在现场拍摄广告照、签名照片，并未有贪玩官方之前所宣布的互动与游玩。<ref>{{Cite web|url=http://pc.265g.com/news/1801/19218.html|title=B站《贪玩蓝月》直播两小时古天乐还没来 主持人尴尬疯了_单机新闻资讯_265G单机频道|accessdate=2018-02-02|work=pc.265g.com}}</ref>且直播过程中，摄像机疑似非正常拍摄。在哔哩哔哩直播间人气达到顶峰的时候，突然被掐断直播流，变成了虎牙直播平台独播。随后直播人员拿出虎牙平台相关物品、主持人称虎牙直播最有排面等挑衅言论和拿出古天乐签名照在私人直播间抽奖等行为引起哔哩哔哩观众不满，大量哔哩哔哩用户举报贪玩游戏直播间。<ref>{{Cite web|url=http://www.sohu.com/a/220273435_99957087|title=贪玩蓝月直播都坑了谁，为何会让B站永久封停？|accessdate=2018-02-02|date=2018-02-01|work=www.sohu.com}}</ref>当天，哔哩哔哩在[[知乎|知乎]]上的官方账号宣布永久封停贪玩游戏的账号、投稿和直播间。<ref>{{Cite web|url=https://www.zhihu.com/question/266366509/answer/308339819|title=如何看待1月31日古天乐在哔哩哔哩直播《贪玩蓝月》？ - 知乎|accessdate=2018-02-02|work=www.zhihu.com|language=zh}}</ref>
