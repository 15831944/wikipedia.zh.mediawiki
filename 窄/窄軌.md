'''窄軌'''，又稱'''狹軌'''，是指[[轨距|轨距]]小於1435mm[[標準鐵軌|標準軌距]]的軌道。大部分的窄軌距軌道的寬度是1067公厘或762公厘等。

== 窄軌距的優缺點 ==
===優點===
*鋪設成本少
*曲率半徑小
*車廂重量較輕，[[鐵路機車|機車]]動力不需太大
*可用於較大的路線坡度

===缺點===
*車廂空間較小
*行駛速度受限，最快僅能到160km/h

== 主要窄軌軌距 ==
*381mm（15英寸）
*508mm（20英寸）
*597mm
*600mm「德科維爾軌距」
*610mm（24英寸=2英尺）
*750mm
*762mm（30英寸=2英尺6英寸）「二六」、「五分」
*800mm
*914mm（3英尺）
*1000mm「米軌」
*1067mm（3英尺6英寸）「三六軌距」
*1372mm「馬車軌距」
*1432mm（[[倫敦地鐵|倫敦地鐵]]原有標準）

== 世界上的窄軌 ==
===中華人民共和國===
中华人民共和国的窄轨铁路主要有国有铁路、地方铁路、[[森林铁路|森林铁路]]、工矿铁路、游览铁路等。中国最早之[[淞滬鐵路|淞滬鐵路]]初筑时即为762mm。
成立初期主要采用[[蒸汽机车|蒸汽机车]]，如28吨C2型、42吨C4型；现多采用[[内燃机车|内燃机车]]、[[电力机车|电力机车]]，分别如JMY200、JMY380型；ZL20、ZL40型。
[[File:Different_gauges_in_China_Railway_Museum.jpg|thumb]]]]
==== 国有铁路 ====
[[中华人民共和国|中华人民共和国]]成立之时，铁道部即有窄轨铁路，如[[同蒲线|同蒲线]]、云南窄轨铁路等。后大部分改为标准轨。现仅余云南[[昆河线|昆河线]]。
1903年至1910年法国人在云南修建的轨距为1000mm的铁路，从[[昆明北站|昆明]]出发蜿蜒至中[[越南|越]]边境口岸[[河口|河口]]，并延长至越南港口城市[[海防|海防]]，其中云南段全长468km。从此它成为“[[云南十八怪|云南十八怪]]”之奇景：“火车没有汽车快”，“火车不通国内通国外”。2003年昆明至河口的米轨火车客运停运，2008年6月15日米轨火车客运重新运行。路线：西至[[石咀站|石咀站]]（[[车家碧|车家碧]]附近），南至[[呈贡县|呈贡县]]的{{站|王家营}}。此外还曾有{{站|鸡街}}到{{站|石屏}}的600mm铁路，长176km。后相继被改为1000mm轨距。而鸡街到个旧段未能改造，1990年废止。

==== 地方铁路 ====
现主要有[[河南地方铁路|河南地方铁路]][[禹郸铁路|禹郸铁路]]、[[朝杞铁路|朝杞铁路]]、[[竖开铁路|竖开铁路]]等。

采用762mm轨距的还有：湖南省的[[醴浏铁路|醴浏铁路]]、[[益灰铁路|益灰铁路]]、[[郴嘉铁路|郴嘉铁路]]；江西省的[[上犹铁路|上犹铁路]]、[[景涌铁路|景涌铁路]];广东省的[[梅隆铁路|梅隆铁路]]、[[乳阳铁路|乳阳铁路]]，雲浮[[六都鎮|六都鐵路]]；四川省的[[彭白铁路|彭白铁路]]、[[嘉阳铁路|嘉阳铁路]]）；云南省的[[鸡个铁路|鸡个铁路]]等。

==== 森林铁路 ====
多分布在东北、内蒙等林产区。曾有苇河森铁<ref>[http://www.civilization.com.cn/wm/wmjs/t20060111_123539.htm 森铁火车的命运绝唱] {{webarchive|url=https://web.archive.org/web/20060625000256/http://www.civilization.com.cn/wm/wmjs/t20060111_123539.htm |date=2006-06-25 }}</ref>（[[滨绥铁路|滨绥铁路]]支线）、江西[[赣南森铁|赣南森铁]]等。一部分现被改建为旅游路线，或作它用。中华人民共和国建立后多采用762mm。
==== 工矿铁路 ====
中華人民共和國有许多工矿企业采用了窄轨铁路，如北京[[首都钢铁集团|首都钢铁集团]]大灰厂专用线（废止）、[[芭石铁路|芭石铁路]]等。多采用762mm。

==== 游览铁路 ====
游览线路的规模不一，来由各异。小的如20世纪50年代建造的[[哈尔滨儿童铁路|哈尔滨儿童铁路]]<ref>[http://www.harbinnews.com/200706/20070618/20070618528720384ECDFEEA1B91746B6F30A2C2.html 新中国第一条儿童铁路]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>系公园内游览线路，且具有培养少年儿童之功能；又如郑州世纪欢乐园小火车为近年建造，采用新郑铁路退役之内燃机车。大的如芭石铁路，在沿线部分地区煤炭基本采完后，首先方便沿线居民出行，之后在便民基础上具有游览铁路的功能。

===香港===

原屬地鐵之港鐵[[觀塘線|觀塘線]]、[[港島線|港島線]]、[[荃灣線|荃灣線]]、[[將軍澳線|將軍澳線]]、[[東涌線|東涌線]]、[[機場快線|機場快線]]及[[迪士尼線|迪士尼線]]的軌距是1432毫米（按當時[[倫敦地鐵|倫敦地鐵]]標準建造；可與標準軌相通，因二者軌距相差僅3毫米，屬港鐵安全誤差範圍內；而部分市區線道岔，如[[荔景站|荔景站]]，屬標準軌距），[[香港電車|香港電車]]則是1067毫米。[[九廣鐵路|九廣鐵路]]英段（今港鐵[[東鐵線|東鐵線]]）建築時也使用窄軌（2英尺（610毫米）），通車後改用標準軌，原有軌道移往[[沙頭角支線|沙頭角支線]]。

===日本===
除了[[新幹線|新幹線]]和少部份[[在來線|在來線]]使用1435mm外，其他的鐵路大多是用窄軌，當中以1067mm為最普遍。也有1372mm和762mm的「輕便鐵道」。

===臺灣===
除了[[臺灣高速铁路|臺灣高速铁路]]、[[臺北捷運|臺北捷運]]、[[高雄捷運|高雄捷運]]、[[臺中捷運|臺中捷運]]、[[桃園機場捷運|桃園機場捷運]]之外，其他的[[台湾铁路管理局|台湾铁路管理局]]铁路、产业轻便铁路都是窄轨。台铁[[西部干线|西部干线]]轨距是1067mm；[[东部干线|东部干线]]轨距原本是762mm，在与西部干线连接后，于1982年完成拓宽工程，拓宽为1067mm。其他的产业轻便铁路，如[[台糖铁路|台糖铁路]]、[[阿里山森林鐵路|阿里山森林鐵路]]，轨距以762mm为主。

=== 非洲 ===
非洲各国之铁路大多为殖民时期建设的简易窄轨铁路，独立后由于经济基础薄弱，大多数线路仍沿用窄轨。此外，为了与既有路网衔接，后续建设的新线路也多为窄轨，如[[中华人民共和国|中华人民共和国]]援建的[[坦赞铁路|坦赞铁路]]等。
