'''郝明珠'''（{{bd|1916年|6月24日|2006年|4月|}}）女，[[陕西省|陕西省]][[安定县_(元朝)|安定县]]（现[[子长县|子长县]]）[[瓦窑堡|瓦窑堡]]人，[[习仲勳|习仲勳]]原配。

==生平==
在关中分区妇联工作，担任主任职务。1935年和当时的陕甘边苏维埃主席[[习仲勳|习仲勳]]结婚<ref>{{Cite book|title=延安女大 纪念延安中国女子大学建校五十周年 1939-1989|last=纪念延安女大五十周年筹委会|first=纪念延安女大五十周年筹委会|publisher=|year=1989年5月|isbn=|location=|pages=70-72}}</ref>。1936年6月至9月习仲勳任[[环县|环县]]县委书记期间，郝明珠曾在环县妇女工作部工作<ref>{{Cite web |url = http://gs.cnr.cn/gsxw/jzgc/201306/t20130609_512787216.shtml |title =环县第一任县委书记习仲勳  |author =  |date = 2013-06-09 |publisher =中国广播网  |language =  |accessdate =  |quote =  }}</ref>。二人先后育有五个孩子，其中两个孩子夭折（第5個孩子於1944年因難產早夭）<ref>《習乾平：瓦窑堡的一枝黑牡丹—习仲勳前夫人郝明珠的故事》 “就在1944年难产后，一个孩子死了，她大出血差点丧命，死里逃生后的她在心中默默地想着自己怎样才能减轻自己痛苦的办法。……直到有一天，丈夫对她出言不逊，向她提出分手的要求，她才感到丈夫的话如五雷轰顶。单纯、年轻的她，如坠烟雾，不知所措。……于1944年初冬，她毅然决然地与他离了婚，分道扬镳。……1943年至1944年10月期间，习仲勋担任中共绥德地委书记兼绥德警备司令部政委，妻子没有与他同去绥德，而是留在延安地区，如此分离一年多。他对妻子不与他同去绥德很不满意，认为妻子有了外心。他们之间书信往来闹意见，互相伤害。 妻子则因为一次又一次怀孩子，自己太受苦，想相对安静一段时间，休养一下身体。但她从未向丈夫讲述过她的想法，只是不与他同行绥德而分居。”</ref>。<ref>{{cite web|url=https://www.powerapple.com/news/articles/1239279|title=习仲勋二妻三子四女今何在|publisher=凤凰网|date=2013-10-14|accessdate=2017-08-16}}</ref>根据其次女[[习干平|习干平]]的回忆文章《瓦窑堡的一枝黑牡丹——习仲勳前夫人郝明珠的故事》记载，1944年初冬郝明珠与[[习仲勳|习仲勳]]离婚<ref>{{cite web|url=http://ir.lib.ncu.edu.tw:88/thesis/view_etd.asp?URN=100155005&fileName=GC100155005.pdf|title=习仲勳与中国共产革命运动之研究（1928—1949）|publisher=杨潓璿(Hui-Hsuan Yang)|date=2013-06-24|accessdate=2017-08-16}}</ref><ref>{{Cite book|title=习仲勳传 下|last=《习仲勳传》编委会编|first=|publisher=北京：中央文献出版社|year=2013年8月|isbn=978-7-5073-3817-1|location=|pages=361}}</ref>。

[[中华人民共和国|中华人民共和国]]时期，郝明珠曾任陕西省第五届人大常委会委员<ref>{{Cite web |url = http://www.npc.gov.cn/npc/zt/qt/dfrd30year/2011-03/31/content_1649889.htm |title = 陕西省五至十一届人大常委会组成人员名单 |author =  |date = 2011-03-31 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>。

==家庭==
*丈夫：[[习仲勋|习仲勋]]。
** 长女：[[习和平|习和平]]（已在文革中不堪迫害自杀去世，留下孩子由母亲郝明珠抚养成人）。
** 次女：[[习干平|习干平]]，曾用名郝平（1939年－）。
***次女婿：[[马宝善|马宝善]]，1941年生于山西省太原古交市，为《法制日报》编辑。
**长子：[[习正宁|习正宁]]（1941年－1998年11月27日），又名习富平，[[习仲勋|习仲勋]]长子，曾任陕西省委组织部副部长兼青干处处长，海南省政法书记兼司法厅长，1998年去世。
*姨父：[[吴岱峰|吴岱峰]]。
