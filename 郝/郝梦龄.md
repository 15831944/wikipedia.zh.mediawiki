'''郝梦龄'''（{{bd|1892年|2月18日|1937年|10月16日|catIdx=H}}），字锡九。抗战初期牺牲的第一位[[军长|军长]]。

== 生平 ==
生于[[河北省|河北省]][[藁城市|藁城县]]庄合村。[[保定军校|保定军校]]第六期毕业。

1921年起，郝梦龄在[[魏益三|魏益三]]部任营长、团长。

1926年跟随魏益三归属[[冯玉祥|冯玉祥]]的[[国民军|国民军]]，任第四军第二十六旅上校旅长。

在[[北伐战争|北伐战争]]中，由于他作战英勇，升任第四军第二师少將师长。打下[[郑州|郑州]]後，部队整编，任[[国民革命军|国民革命军]]第五十四师少將师长。

1930年[[中原大战|中原大战]]后，兼任郑州警备司令。后升为第九军中將副军长。

山西[[忻口战役|忻口战役]]、雁門關、娘子關，时任[[国民革命军第九军|国民革命军第九军]]中將军长，并于是役殉国（1937年10月16日），追赠陆军上将。



== 功勋 ==
郝梦龄将军是抗战初期牺牲在抗日疆场上的第一位[[军长|军长]]<ref>[http://news.xinhuanet.com/politics/2009-04/24/content_11250760.htm 新闻] {{webarchive|url=https://web.archive.org/web/20090429220557/http://news.xinhuanet.com/politics/2009-04/24/content_11250760.htm |date=2009-04-29 }}，新华网。</ref>，1938年的[[小学|小学]][[国语|国语]]课本中就有“中国第一个军长”的题目描写他。<ref>[http://dangshi.people.com.cn/BIG5/85038/9188385.html 郝夢齡：抗戰殉國第一將]，中国共产党新闻网。</ref>

== 纪念 ==
2005年拍攝的抗戰主題電影《[[太行山上|太行山上]]》，郝夢齡的角色由[[台湾|台湾]]演員[[劉德凱|劉德凱]]飾演。

为纪念郝梦龄的功勋，抗战胜利后，国民政府将汉口北小路改名为郝梦龄路。1985年，在抗日战争胜利40周年之际，武汉市人民政府为缅怀抗日英烈，重新恢复了以[[张自忠|张自忠]]、[[郝梦龄|郝梦龄]]、[[刘家麒|刘家麒]]、[[陈怀民|陈怀民]]4位抗日将士的名字命名的4条路名，并举行了隆重的命名仪式。

郝梦龄烈士墓园位于武汉市伏虎山西北侧的山腰间，列为市级文物保护单位。1966年郝梦龄墓被铲平，后重修为烈士墓园。
