'''圖雷·尼曼'''（1886年5月18日-1969年10月7日），是[[瑞典|瑞典]]的[[社会主义|社会主义者]]。除了政治活動之外，尼曼还會写诗寫歌。   

圖雷‧尼曼還有一對双胞胎弟弟，艺术家{{Link-en|Einar Nerman|Einar Nerman}}和考古学家{{Link-en|Birger Nerman|Birger Nerman}}。

== 早年 ==
尼曼在[[北雪平|北雪平]]的[[工人階級|工人]][[中產階級|中产]]家庭长大。尼曼的父亲拥有一家书店，并与一位年轻多岁的员工结婚。該名員工成为图雷和他的两个弟弟的母亲。尼曼在童年時期，在父親的書店閱讀許多書籍。 

1903年，尼曼自北雪平的一間[[文理中学|文理中學]]毕业。毕业當天，他拿起学校的[[聖經|圣经]]，把它扔在[[穆塔拉河|穆塔拉河]]河里。在自传中，尼曼将此描述为他第一次的革命行动。數年之后，[[瑞典社会民主工人党|瑞典社会民主工人党]]领导人[[卡尔·亚尔马·布兰廷|亞爾馬‧布蘭廷]]询问他是什么使他成为社会主义者时，尼曼回答说是对宗教的质疑。 

毕业后，他搬到[[乌普萨拉|乌普萨拉]] ，在[[乌普萨拉大学|乌普萨拉大学]]念書 。 

== 政治觉醒 ==
1905年，尼曼目睹[[1905年俄国革命|俄罗斯革命]]，和[[斯堪的纳维亚|斯堪的纳维亚]]的革命性发展──[[挪威|挪威]]宣布自[[瑞典|瑞典]]獨立。圖雷‧尼曼受到影響，在政治上向左派激进化。他开始阅读[[奥古斯特·斯特林堡|奧古斯特‧斯特林堡]]、[[列夫·托爾斯泰|列夫‧托爾斯泰]]和{{Link-en|愛倫‧凱|Ellen Key}} 。尼曼不久就接觸[[卡尔·马克思|马克思]]，开始在[[乌普萨拉|乌普萨拉]]参加社会主义青年会议。 

1907年，尼曼受徵招入伍。根据他自己的要求，尼曼接受[[军医|軍醫訓練]] 。此間，尼曼形成了{{Link-en|反军国主义|Antimilitarism}}立场。1908年，尼曼被秘密警察发现非法散佈反军国主义传单。尼曼最初被判入狱，但後被改判300[[瑞典克朗|克朗]]的罚金。尼曼父亲给他支付罰金的钱，但尼曼將其挪去充作[[巴黎|巴黎]]旅行的資金。在巴黎，尼曼待了几个星期。回到瑞典后，尼曼的父亲再度寄钱支付罚款，以避免尼曼入狱。 

1909年，尼曼搬到了北部城市[[松茲瓦爾|松茲瓦爾]]，担任[[瑞典社会民主工人党|社会民主工人党]]機關報《新社会》（''Nya Samhället''）的作家。尼曼也出版了几本诗集，其中许多是针对教会、瑞典国王和[[資產階級|资产阶级]]具激进，挑衅性的诗歌。 

圖雷‧尼曼加入[[瑞典社会民主工人党|瑞典社会民主工人党]]，很快与{{Link-en|霍格伦|Zeth Höglund}}一起採取左翼立場。尼曼成为该党反对[[改良主义|改良主義]]者[[卡尔·亚尔马·布兰廷|亞爾瑪‧布蘭廷]]的左翼反对派領导者之一。 

1912年元旦和接下来的几周里，尼曼與其友人在[[德国|德国]]選舉期間，前往德國跟随[[卡尔·李卜克内西|卡爾‧李卜克內西]]。 

== 第一次世界大战和齊美爾瓦爾德 ==
1912年11月，因[[巴尔干战争|巴尔干战争]]爆发，尼曼前往在[[瑞士|瑞士]][[巴塞尔|巴塞尔]]召開的[[第二国际|社会主义国际]]特别紧急会议。大会上，所有欧洲社会主義政党的领导人同意在国际上團結一起，防止任何未来的战争，主張在一个统一的国际工人阶级之下，就不再有更多的战争。 

也因此，1914年[[第一次世界大战|第一次世界大战]]的爆发，以及社会主义国际的崩溃，震惊图雷·尼曼。几乎所有欧洲社会主義政党领导人都突然支持本國资产阶级政府參與战争，并反对他们原先的社会主义盟友。工人階級在战场上杀死工人階級。 

但也有例外。[[卡尔·李卜克内西|卡尔·李卜克内西]]，在[[柏林|柏林]]的[[德意志帝国议会|帝國議會]]，与党内成员投票反对德国战争信債。 

圖雷‧尼曼和{{Link-en|霍格伦|Zeth Höglund}}代表[[齊美爾瓦爾德會議|齊美爾瓦爾德会议]]的瑞典─挪威成员，统一其它的国际社会主义反战运动，包含俄罗斯的[[弗拉基米尔·伊里奇·列宁|列宁]]、[[格里戈里·叶夫谢耶维奇·季诺维也夫|季诺维也夫]]和[[列夫·达维多维奇·托洛茨基|托洛茨基]]，瑞士的[[罗伯特·格林_(瑞士)|罗伯特·格林]]，以及德国的[[羅莎·盧森堡|罗莎·卢森堡]]、[[卡尔·李卜克内西|卡尔·李卜克内西]]。 德國代表盧森堡與李卜克內西因故不能参加會議，但发出了问候和支持。卢森堡因反战宣传而入狱，而德国军队则动员李卜克内西在前线挖掘战壕。雖然瑞典没有参加世界大战，但{{Link-en|霍格伦|Zeth Höglund}}回到瑞典后，也因參與国际反战运动而入狱。

== 在美国 ==
1915年年初，齊美爾瓦爾德會議之前，尼曼在[[美国|美国]]旅行了几个月。为了獲得旅行所需的資金，尼曼为几家瑞典报纸撰写文章。尼曼乘船抵达[[纽约|纽约]]，逗留了一小段时间，登上当时世界上最高的摩天大楼[[伍爾沃斯大樓|伍尔沃斯大楼]]的顶层。其後，乘坐特快列车穿越大陆前往[[旧金山|旧金山]]，参观了[[巴拿马万国博览会|1915年的世界博览会]] 。 

隨後，尼曼展開旅程與[[明尼阿波利斯|明尼阿波利斯]]和[[芝加哥|芝加哥]]的斯堪的纳维亚裔美国工人對談。 

此外，尼曼还花了一些时间去探访[[阿斯托利亞|俄勒冈州阿斯托里亚的]]一些亲戚。 

当圖雷‧尼曼在夏天返回瑞典时，{{Link-en|霍格伦|Zeth Höglund}}抱怨道，当瑞典社會民主工人黨內部左翼與右翼激烈的鬥爭之際，尼曼是否有需要前往美國。 

== 瑞典共产主义的诞生 ==
[[File:Lenin.Nerman.Lindhagen.Stockholm..jpg|右]]，圖雷‧尼曼和{{Link-en|卡爾·林德哈根|Carl Lindhagen}}，斯德哥尔摩，1917年4月 ]]
1917年初，[[瑞典社会民主工人党|社会民主工人党]]内部左翼与右翼之间的斗争导致分裂。{{Link-en|霍格伦|Zeth Höglund}}和尼曼，考虑離開黨，與其他被開除的激進主義者成立自己的共產主義政黨，包含Kata Dalström{{找中文}}、Fredrik Ström{{找中文}}和[[斯德哥尔摩|斯德哥尔摩]]市长{{Link-en|卡爾·林德哈根|Carl Lindhagen}} 。 

 {{Link-en|霍格伦|Zeth Höglund}}是党青年组织的领导者，在[[左翼党_(瑞典)|瑞典社會民主左翼黨]]的創立上成功地團結全体社会主义者。新的党成立于1917年5月，拥有约20,000名成员。左翼黨很快就会改名，成为瑞典第一个共产党。該黨機關報為{{Link-en|《政治報》|Folkets Dagblad Politiken}}，由其他国际共产党领导人撰写和出版文章。 

1917年4月，列宁從在瑞士的流亡返回到[[圣彼得堡|彼得格勒的]]途中，经过[[斯德哥尔摩|斯德哥尔摩]]。圖雷‧尼曼就是在斯德哥爾摩迎接列宁并照顾他的人之一。 

列宁返回俄罗斯，随后發動[[十月革命|十月革命]]。图雷·尼曼和社會民主左翼黨全力支持[[布尔什维克|布尔什维克，]]并企圖在瑞典發動类似的共产主义革命。 

作为瑞典共产党的国际代表，尼曼与来自世界各地的共产主义者有过接触：[[卡尔·李卜克内西|卡爾·李卜克內西]]；波蘭布爾什維克黨人[[卡尔·拉狄克|卡尔·拉狄克]]，曾在斯德哥尔摩住了一段时间；在斯德哥尔摩生活和工作的乌克兰─意大利社会主義者领袖{{Link-en|安格里卡·巴拉巴諾夫|Angelica Balabanoff}}；芬兰共产主義者{{Link-en|于尔约·西罗拉|Yrjö Sirola}}和[[奥托·威廉莫维奇·库西宁|奧托‧庫西寧]]多次访问斯德哥尔摩的圖雷‧尼曼。 

尼曼还与美国社会主义领导人[[尤金·V·德布斯|尤金·V·德布斯]]、[[約翰·里德_(記者)|約翰·里德]]以及美国共产主義领袖[[马克斯·伊斯特曼|馬克斯·伊斯特曼]]等人通信。 

== 在苏联 ==
[[File:Ture_nerman.jpg|左]]1918年夏末，圖雷‧尼曼与{{Link-en|安格里卡·巴拉巴諾夫|Angelica Balabanoff}}和'''Anton Nilson{{找中文}}'''一起前往[[布尔什维克|布尔什维克]][[俄罗斯|俄罗斯]]。  

在[[圣彼得堡|彼得格勒]] ，尼曼受邀访问[[彼得格勒苏维埃|彼得格勒苏维埃]]領導者[[格里戈里·叶夫谢耶维奇·季诺维也夫|季诺维也夫]]。兩人自齊美爾瓦爾德便彼此认识。季诺维也夫问尼曼：「你什么时候在瑞典进行革命？」尼曼谦虚地回答说还没有明确的日期。翌日，尼曼参加[[冬宫|冬宫]]外的集会，其中季诺维也夫和巴拉巴諾夫谈到了紅军士兵已參與[[俄国内战|內戰]]。 

彼得格勒的一些日子以後，尼曼前往[[莫斯科|莫斯科]]。在莫斯科，尼曼訪問[[列夫·鲍里索维奇·加米涅夫|加米涅夫]]夫婦。 

10月3日，尼曼参加了[[莫斯科大剧院|莫斯科大剧院]]的大会。演讲者包括[[雅科夫·米哈伊洛维奇·斯维尔德洛夫|斯維爾德洛夫]]，[[尼古拉·伊万诺维奇·布哈林|布哈林]]和主演人[[列夫·达维多维奇·托洛茨基|托洛茨基]] 。遭到枪击的列宁因傷无法出席。其間，尼曼与托洛茨基进行了简短的交谈。 

10月4日，尼曼与布哈林进行了长时间的采访。其中，布哈林表达了对[[世界革命|世界革命]]和[[社会主义|社会主义]]未来的乐观态度。同日，尼曼会见[[布尔什维克|布尔什维克]]女性领导人[[亚历山德拉·米哈伊洛芙娜·柯伦泰|亚历山德拉·科倫泰]]，未來的苏联驻瑞典大使。 

圖雷‧尼曼于10月底通过[[芬兰|芬兰]]群岛返回瑞典。其間，尼曼聽聞[[德国十一月革命|德国革命]]失败，以及卡尔·李卜克内西和罗莎·卢森堡遭到謀殺。 

1920年春天，圖雷‧尼曼与[[奧托·格日姆倫|奧托·格日姆倫]]一起，第二次访问俄罗斯，与列宁会面。 

夏天回到瑞典后，尼曼聽聞父亲的死讯。他的家人试图推迟葬礼，但尼曼仍然错过其父的喪禮。 

1927年，革命十周年，圖雷‧尼曼再次前往俄罗斯。列宁已死，事態开始发生变化。 

== 反对史達林主义 ==
圖雷‧尼曼谴责[[斯大林主义|史達林主义]]在俄罗斯的崛起，但当{{Link-en|霍格伦|Zeth Höglund}}於1924年和共产党分手时，尼曼選擇留下。该党的新领导階层由{{Link-en|卡尔·基尔博姆|Karl Kilbom}}和Nils Flyg{{找中文}}组成 。 

1929年， [[第三国际|共产国际]]确保瑞典共产党由{{Link-en|胡戈‧希倫|Hugo Sillén}}领导的少数史達林主义者控制 。该党的大多数成员，非史達林主义者，遭到驅除，包括圖雷‧尼曼、{{Link-en|卡尔·基尔博姆|Karl Kilbom}}和Nils Flyg。  少数派（史達林主义者）夺取了该党的总部及其档案，但多数派（非史達林主义者）设法控制了{{Link-en|《政治報》|Folkets Dagblad Politiken}} 。 

在{{Link-en|基尔博姆|Karl Kilbom}}的领导下，大多数人试图继续经营独立于莫斯科的瑞典共产主義政党。1934年，这个党取名为{{Link-en|社會黨|Socialist Party (Sweden, 1929)}}（socialistka partiet）。 

1931年，44岁的图雷·尼曼進入[[瑞典議會|议会]]，作为独立的共产主義者政党代表直到1937年。 

== 在佛朗哥的西班牙 ==
1937年，图雷·尼曼和他的朋友August Spångberg{{找中文}}前往[[西班牙|西班牙]]。此時，[[弗朗西斯科·佛朗哥|佛朗哥]]将军的[[國民軍_(西班牙內戰)|国民军]]掌权，而[[西班牙内战|内战]]正在蹂躏整个国家。他们经过[[納粹德國|纳粹德国，]]前往[[巴塞罗那|巴塞罗那]] 。 

5月3日，巴塞罗那爆发了不同的左翼团体之间的激烈战斗，尼曼等人下榻的旅館陷入持续三天的交火中。圖雷‧尼曼在此第一次遇到了[[维利·勃兰特|維利‧伯蘭特]]。 

5月7日，巴塞罗那战斗结束。500多人死亡，1,500多人受伤。 

之后，根据共产国际的命令，世界各地的史達林主义的媒体声称，瑞典的「托洛茨基─法西斯主义者圖雷‧尼曼和August Spångberg是巴塞罗那战鬥的主谋」。 

当尼曼自西班牙回到瑞典时，他在自己的{{Link-en|社會黨|Socialist Party (Sweden, 1929)}}遭遇混乱。 

Nils Flyg掌权，开除了前党主席{{Link-en|卡尔·基尔博姆|Karl Kilbom}}。Flyg希望圖雷‧尼曼留下来，但尼曼自愿离开。 

在接下来的两年里，圖雷‧尼曼没有参加任何政黨。他也在议会中失去了席位。1939年的[[五朔節|五朔節]]，圖雷‧尼曼重新加入[[瑞典社会民主工人党|瑞典社会民主工人党]] 。 

== 第二次世界大战 ==
1933年4月20日，希特勒在德国掌权后的几个月，尼曼在瑞典议会中要求瑞典应该为所有想来瑞典的德国[[犹太人|犹太人]]提供庇护。該議案遭到議會否決。 

当第二次世界大战在1939年爆发时，瑞典宣布[[中立|中立]]。奧圖‧尼曼在战争期间，以〈Trots allt！〉(縱使如此)一文的编辑者知名。 〈Trots allt！〉的反[[纳粹主义|纳粹]]立場引起德国驻斯德哥尔摩大使馆的愤怒。此外，〈Trots allt！〉也批評瑞典政府出於对[[阿道夫·希特勒|希特勒]]的恐惧，允许德国军队通过瑞典。 

〈Trots allt！〉 的標題來自於[[卡尔·李卜克内西|卡尔·李卜克内西]]一篇文章的標題 <ref><div> [http://www.marxists.org/deutsch/archiv/liebknechtk/1919/01/trotz.htm Marxists.org] </div></ref>。該文寫於1919年李卜克内西被谋杀前一天。 

最初，瑞典政府基於中立立場，不能容忍〈Trots allt！〉 。许多尼曼的反战與反纳粹作品都不允许通过[[审查制度|审查]]而出版。这篇文章非常受欢迎，并受到很多来自支持者的经济帮助。 

1940年，因为〈Trots Allt〉，圖雷‧尼曼被瑞典当局判处三个月监禁 。在該文中，尼曼声称希特勒下令在慕尼黑啤酒窖内进行1939年的轰炸和暗杀行动。 <ref><div> [https://cdnc.ucr.edu/cgi-bin/cdnc?a=d&d=VEST19400111.2.11&srpos=8&e=-------en--20--1--txt-txIN-%22ture+nerman%22-------1 https://cdnc.ucr.edu/cgi-bin/cdnc?a=d&d=VEST19400111.2.11&srpos=8&e=-------en--20--1--txt-txIN-%22ture+ nerman％22 ------- 1] </div></ref>   

当时，尼曼最亲密的合作者{{Link-en|以色列‧霍姆格倫|Israel Holmgren}} ，也被判入狱。   

1945年5月7日，纳粹德国终于投降了。5月16日，圖雷‧尼曼与流亡瑞典的挪威难民們一同前往解放後的挪威。〈Trots Allt〉曾经是挪威反对纳粹占领的自由斗争的伟大支柱，并且地下受到挪威人的喜愛。

尼曼成為瑞典加入[[北大西洋公约组织|北约]]的倡导者。在[[斯德哥爾摩群島|斯德哥尔摩群岛]]的{{Link-en|布勒|Blidö}} ，尼曼的住家外面懸掛著锤子和镰刀的[[苏联国旗|苏联国旗]] ，是他早期信仰的讽刺性参考。 

1969年，圖雷‧尼曼去世。 

[[斯德哥尔摩|斯德哥尔摩]]的[[国王岛|国王岛]]上，一條小巷以圖雷‧尼曼的名字命名 。[[挪威|挪威]][[卑爾根|卑尔根]]的一条街道亦以尼曼的名字命名。
