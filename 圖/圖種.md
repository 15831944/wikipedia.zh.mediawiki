'''圖種'''是一種利用[[數位影像|圖片檔]]來傳遞[[壓縮檔|壓縮檔]]的技巧。
圖種以[[圖片檢視器|圖片檢視器]]開啟時，就是一張單純的圖片。
但可以以[[壓縮軟體|壓縮軟體]]對其解壓縮，取得''附加''在圖片檔後的壓縮檔內容。

雖然圖種攜帶的是壓縮檔，而壓縮內可以是任意的檔案。
但檔案增大，圖種的大小也會跟著增大；引人懷疑。
所以通常圖種中夾帶的壓縮檔內容僅為一[[種子文件|種子文件]]，
供人[[下載|下載]]某其它檔案。固稱為圖種。


==原理==

大部份[[圖形文件格式比較|圖片格式]]檔案的結束點會有一標記，標記'''資料結束'''。
而大部份[[壓縮軟體比較|壓縮格式]]檔的一開頭也會有一個標記，標記'''資料開始'''。
將一壓縮檔'''附加'''在一圖片檔後；
即將一圖片檔的資料結束標記後緊接著壓縮檔的資料開始標記。

如此一來，圖片檢視器只會讀取到圖片檔的資料結束標記為止；
而壓縮軟體則會從壓縮檔的資料開始標記開始讀取。
也就是以圖片或壓縮檔開啟都可以。


==實現==

===UN*X===

在[[類UNIX|類UNIX]]系統中：

<source lang="sh">
$ cat 壓縮檔.zip >> 圖片檔.jpg 
    # 法一：
    # 將 壓縮檔.zip 的內容寫入到 圖片檔.jpg 後。

$ cat 圖片檔.jpg 壓縮檔.zip > 圖種.jpg
    # 法二：
    # 依序將 圖片檔.jpg , 壓縮檔.zip 的內容寫入到 圖種.jpg 內。
</source>


===[[Windows|Windows]]===

====[[dos|dos]]或[[命令提示字元|命令提示字元]]====

<source lang="batch">
copy /b 圖片檔.jpg + 壓縮檔.rar 圖種.jpg
</source>

====[[圖形介面|圖形介面]]====

# 將圖片的[[副檔名|副檔名]]改為[[rar|rar]]。此時雙擊開啟改變檔名的圖片檔會出現錯誤。
# 將壓縮檔'''拖曵'''到上步驟改變副檔名的圖片檔上。
## 按住壓縮檔。
## 將壓縮檔'''拉'''到圖片檔檔上。
## 放開壓縮檔。
# 改名後的圖片檔成為圖種。此時雙擊圖種可以開啟，看到的內容為原來的壓縮檔。
# 將圖種的副檔名更改為[[jpg|jpg]]。（或圖片原來的副檔名。）此時雙擊圖種可以開啟，看到的內容為原來的圖片。
