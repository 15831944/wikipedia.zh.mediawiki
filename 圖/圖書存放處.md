'''圖書存放處'''（{{lang-en|'''Book Depository'''}}，以前名為The Book Depository）是一間[[英國|英國]]網絡書店，擁有大規模的圖書目錄，以及免費郵寄至超過160個國家。商店是由前[[亞馬遜公司|亞馬遜公司]]僱員創立。2011年，亞馬遜購得該店。

== 歷史 ==
該公司由{{link-en|安德魯·克勞佛|Andrew Crawford (entrepreneur)}}和現任首席運營官Stuart Felton於2004年創立。其座右銘是"[[長尾|All Books Available to All]]"。

2011年7月4日，[[亞馬遜公司|亞馬遜公司]]達成協議收購圖書存放處。<ref>{{Cite web |url = https://www.theguardian.com/books/2011/jul/05/amazon-the-book-depository |title = Booksellers fear 'stranglehold' as Amazon snaps up British rival The Book Depository |author = Zoe Wood |date = 2011-07-05 |publisher = The Guardian |language = en}}</ref>
