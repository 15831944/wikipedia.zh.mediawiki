'''图波列夫TB-3'''（{{Lang-ru|Тяжёлый Бомбардировщик}}Тяжёлый Бомбардировщик，[[俄文羅馬化|轉寫]]： Tyazholy Bombardirovschik，中譯為重型轰炸机'''；民用型編號ANT-6'''）是西元1930年代[[蘇聯空軍|苏联空军]]列裝的重型轟炸機，并被使用於[[第二次世界大战|第二次世界大战]]。它是世界上第一个[[悬臂|悬翼]]四引擎的重型轰炸机。 西元1939年，TB-3因过时而正式退役，但TB-3仍在整個二次世界大戰进行轰炸和运输工作。 TB-3也以[[Zveno計畫|Zveno計畫]]戰鬥機母機和輕型戰車運輸機的角色參加戰鬥。

== 研發 ==
西元1925年， [[蘇聯空軍|苏联空军]]向[[中央空气流体动力学研究院|中央空氣流體動力學研究院]]（TsAGI）提出一個重型轟炸機的設計要求:擁有總馬力2000匹（1970 [[馬力|hp]]）和固定或可收起的兩輪起落架。[[图波列夫设计局|图波列夫设计局]]在西元1926年开始设计並於西元1929年定案。 TB-3是以圖波列夫設計局的[[圖波列夫_TB-1_重型轟炸機|TB-1]]作為設計基礎，起初打算使用平均輸出600匹馬力(590hp)的{{tsl|en|Curtiss V-1570|柯蒂斯 V-1570|Curtiss V-1570}}"征服者"发动机， 後來在生產時改為{{tsl|en|Mikulin M-17|米庫林 M-17}}（修改過的{{tsl|en|BMW VI|BMW VI}}引擎）。 西元1930年三月21日，TB-3的試作模型受到核可，而首個原型機在西元1930年10月31日完成。 原型機在西元1930年12月22日由{{tsl|en|Mikhail Gromov (military)|米哈伊爾·格羅莫夫（軍人）|米哈伊爾·格羅莫夫}}試飛。由于振动導致油門关闭，使得飛機差點墜毀，總體而言试飞算是成功。西元1931年4月20日，苏联空军批准使用M-17引擎的ANT-6大量生產。

TsAGI設計的TB-3原型機改用可輸出730匹馬力(720hp)的BMW VIz 500引擎，有较大的散热器和木制固定節距螺旋桨。单轮起落架被认为太過脆弱，取而代之的是 {{Convert|1,350|×|300|mm|lk=out|abbr=on}} 的[[轉向架|轉向架]]加上兩個輪子的起落架。 第一批預先生產的TB'''-3-4M-17'''在西元1932年一月4日由A. B. Yumashev和I. F. Petrov進行首次試飛。意料之外的是，随后的大规模生產，實際生產出來的機型，重量高過原型的10-12%，这大大影響了飛機的性能。主因是原料過高的誤差容忍度使飛機所使用的鋼板、管線比起精心製作的原型機還要更重、更厚、更厚。飞机也粗魯的涂上了一层厚厚的迷彩。工厂依照建議，提供工人自機體上每減少1公斤(2.2英磅)就給予100盧布的報酬。在设计局各方面的努力下，使得飛機减轻1000多公斤(2200英磅))。尽管如此，生产出來的飞机實際重量可能彼此不同，差異可能到几百公斤都有。

1933年，一架TB-3-4M-17藉由移除[[炮台|炮台]]和炸彈掛架、封住开口、安装路輪罩，來使得飛機更加流線。这僅增加最高空速4.5％和些微的最大作戰範圍。图波列夫的结论是，流線化對大型和緩慢的飛機來說，流線化的效益微乎其微。為了研究具波纹的飛機表層，在西元1935年一月二月的一架TB-3-4AM-34R在機身上增加了波纹復盖物。这增加了5.5%最快空速和27.5%的最高飛行高度。這架飞机明確的證明，安裝四片螺旋槳葉使[[爬升率|爬升率]]有顯著的增加。

== 设计 ==
[[File:Tupolev_TB-3_SA-kuva_7361.jpg|缩略图]]
TB-3的結構是全金屬建造而成, 飛機主體是使用三角槽型結構，外層以具波紋狀的非應力金屬覆蓋，厚度0.3 mm（0.012英寸）到 0.8 mm（0.031英寸） 不等。波紋深13 mm（0.51英寸）彼此間距 50 mm（1.97英寸）。[[機翼|機翼]]結構是由內四角異型管組成。西元1934年，因為[[高速鋼|高速鋼]]的發展，TB-3的[[翼展|翼展]]從39.5公尺增加到41.85公尺(129.6至137.3英尺）。人員可著軟底鞋在飛機表面任何地方行走，且不會使飛機受損，此外，將機翼前緣向下擺動，可以形成一條維修引擎的通道。控制系統包含以電纜驅動的可變傾角尾翼和當某邊引擎失效時啟用的平衡補償系統。被固定住的主[[起落架|起落架]]並沒有配置[[煞車|煞車]]，燃料庫沒有防火或防漏機制。不過，引擎有內部滅火系統。TB-3安裝的M-17引擎理論上在沒有[[火星塞|火星塞]]或[[化油器|化油器]]污垢時，能夠提供最大3250[[公里|公里]](1750海浬；2020英里)的航程。TB-3裝有[[輕機槍|輕機槍]]作為自衛武器，分別被安裝在五個[[砲塔|砲塔]]—一個在[[機鼻|機鼻]]、兩個在機體上方中間位置，以及在兩側引擎中間各有一個可伸縮的"垃圾桶"，裡面各有一挺機槍。在後期型，機體上方的其中一挺機槍被移到機尾。

== 服役紀錄 ==
[[File:TB-3_paratroops.jpg|右]]
[[File:T-27_tb-3.jpg|右]]
[[File:ANT-6A_Glavsevmorput.jpg|缩略图]]
1937年10月，苏联以144万美元价格对华军售并立即交付6架TB-3。这笔交易后来列入了1938年3月1日在莫斯科签署的《苏维埃社会主义联邦共和国政府与中华民国政府间关于实施伍仟万美金信用借款条约》，即抗战时期苏联第一次对华军事贷款，年息3%，从1938年10月31日起五年内还清。<ref>{{cite web|url=https://max.book118.com/html/2017/0717/122763182.shtm|author=孟德智|title=《抗日战争时期苏联对华援助研究》|publisher=黑龙江大学硕士学位论文|date=2015-03}}</ref>并列入1938年3月3日签订的苏联对华军售总价30321164美元的第一份合同之中。<ref>{{cite book|title=《苏中经贸关系史 1917-1974年》|author=斯拉德科夫斯基|location=莫斯科|version=1977|page=p.12}}</ref>TB-3由[[苏联志愿航空队|苏联志愿航空队]]驾驶于1937年10月27日飞抵[[迪化|迪化]]，10月31日飞抵[[兰州|兰州]]，由中国空军第8大队第10中队接收。11月30日5架由苏中混合机组驾驶经汉口飞抵南昌。<ref>{{cite web|url=http://mil.news.sina.com.cn/history/2017-02-09/doc-ifyameqr7367935.shtml|title=《苏联曾援华四发重型轰炸机》|date=2017-02-09|publisher=新浪军事网站}}</ref>12月13日从南昌机场试图起飞转场[[吉安市|吉安]]机场前夕，被日机炸毁2架、炸伤2架。12月25日修复后的2架与完好的1架返回兰州，当运输机使用。其中，1938年3月16日，第8大队第10中队分队长郭家彦中尉（26岁，空军军官学校第二期航空班）与张君泽少尉（24岁，中央航空学校第五期第二班(洛阳空军军校)航空班）驾驶的1602号TB3重型轰炸机在兰州拱星墩机场起飞去武汉，因单发失效，从[[平凉|平凉]]上空返航，在在兰州皋兰山营盘岭的山谷坠毁，机上运送的彼得·伊万诺维奇·阿列尼柯夫大尉以下共计20名苏联援华航空队人员<ref>{{cite news|url=http://lzcb.gansudaily.com.cn/system/2013/12/11/014804726.shtml|author=王文元|title=《俄罗斯驻华大使馆传来空难烈士名单》|publisher=兰州晨报|date=2013-12-11}}</ref>仅有坐在机尾部的2人生还，机组以及机上国民政府航空委员会高级翻译李仲武、南京无线电台台长张培泽、航空机械士许振球<ref>{{cite web|url=http://epaper.cz001.com.cn/site1/czrb/html/2014-08/19/content_769302.htm|title=《76年前兰州空难中的常州烈士》|author=许振球|publisher=常州日报|date=2014-08-19}}发表于2014年8月19日《常州日报》B2版</ref>等人全部丧生。<ref>{{cite web|url=http://news.ifeng.com/gundong/detail_2013_09/14/29605915_0.shtml|title=《抗战初期，兰州发生的惊天空难》|author=袁志学（中共兰州市委党史办公室副主任）|date=2013-09-14|publisher=兰州晨报}}</ref>所有遇难者葬于兰州机场南山烈士陵园。1944年以中国空军驻兰州的第四路军司令[[罗机|罗机]]署名的《空军第四军烈士墓记》碑文：“二十七年三月，东岗镇口机张君泽、李仲武等三十余烈士”。

TB-3被使用在對[[日本帝國|日本]]的1938年8月6日[[哈桑湖战斗|哈桑湖战斗]]，43架轰炸了反复争夺的几个高地；1939年[[諾門罕戰役|諾門罕戰役]]，出动了500架次以上用于夜间轰炸，还大量用于军事空运。對[[芬蘭|芬蘭]]的[[冬季戰爭|冬季戰爭]]中。雖然它在西元1939年被正式退役。然而，1941年6月22日—[[德蘇戰爭|德蘇戰爭]]開始時，[[蘇聯空軍|蘇聯空軍]]依舊擁有516架可運作的TB-3，另有25架屬於[[蘇聯海軍航空兵|蘇聯海軍航空兵]]。由於駐紮地遠離蘇聯西部邊界，TB-3從首波德國空襲中避免了災難性的損失，在這之後TB-3在該年6月23日起編入第三重型轟炸機團（TBAP）擔任夜間轟炸。但由於前线对地支援飞机与戰斗機大量损失，TB-3必須在沒有護航的情況下在昼間執行轟炸任務，缓慢的航速导致易于被地方高射炮击中；虽然有环形防御火力，但在现代戰鬥機面前等于无。1941年8月時，TB-3佔了蘇聯轟炸機部隊的25%。但TB-3很适合作为夜间轰炸机，低航速使得命中精度高；机队中积累了大批精英空勤人員，一夜可執行三架次出击任務。TB-3幾乎參加了一直到1943年蘇聯所有主要的戰鬥，包括[[斯摩棱斯克戰役_(1941年)|第一次斯摩稜斯克戰役]]、[[莫斯科戰役|莫斯科戰役]]、[[史達林格勒戰役|史達林格勒戰役]]、[[列寧格勒圍城戰|列寧格勒圍城戰]]和[[庫爾斯克會戰|庫爾斯克會戰]]中。1941年，苏联各种原因损失了超过40架TB-3。 在西元1945年一月，第18航空軍仍然擁有十架可運作的TB-3。TB-3廣泛被使用在貨物和傘兵運輸，最多可攜帶35士兵。在戰爭前五個月，TB-3運送了2,797噸的貨物和2,300名人員。TB-3也被用在幾個特殊的項目，例如:[[Zveno計畫|Zveno計畫]]的戰鬥機母艦和空運、降輕型{{tsl|en|T-27|T-27}}、[[T-37A_水陸兩棲輕型戰車|T-37]]和{{tsl|en|T-38 tank|T-38}}坦克。在西元1941年八月1日，兩架屬於Zveno計畫戰鬥團的TB-3，各攜帶了兩架各攜帶了一對250公斤(550磅)炸彈的[[伊-16|伊-16]]，以自身無損失的狀況下，摧毀了[[羅馬尼亞|羅馬尼亞]][[康斯坦察|康斯坦察]]港口中的油庫。西元1941年八月11日和13日，Zveno計畫戰鬥團成功地破壞了在羅馬尼亞境內[[多瑙河|多瑙河]]上的{{tsl|en|Anghel Saligny Bridge|卡羅一世國王橋}}。Zveno計畫在西元1942年因為母艦過於脆弱而結束。


為了表彰TB-3在戰爭中的貢獻，西元1945年六月18日的首次戰後航空遊行，有三架TB-3出現在其中。

== 变种 ==
[[File:Paratroopers_jumping_from_Tupolev_TB-3.jpg|缩略图]]
<small>资料来源：Shavrov</small>



'''TB-3-4M-17'''<br />第一個生產版本，大約一半的TB-3都是這個型號。<br /><br />'''TB-3-4M-34'''<br />使用修改過散熱器的米庫林 AM-34 發動機，增加了油冷卻器，約十架左右建成。<br /><br />'''TB-3-4M-34R'''<br />使用減少齒輪箱數量的米庫林 AM-34R引擎，大大改進了性能，在尾翼安裝了額外的砲塔、尾輪用液壓制動，以空氣動力學改進的機翼、機身，加入散熱器、可伸縮的風力發電機。<br /><br />'''TB-3-4-34RD'''<br />精簡機身和煞車的長程演示飛機。一些飛機使用直徑兩公尺(6.6英尺)的單一主機輪和三葉片金屬螺旋槳。在西元1933年到1934年用於前往華沙、巴黎、羅馬的航班。<br /><br />'''TB-3-4-34RN'''<br />使用AM-34RN引擎的高海拔版本，四片螺旋槳在另外兩片螺旋槳外側，使用直徑2公尺(6.6英尺)的機輪，使用ShKAS機槍，在4200公尺(13800英尺)飛行時，最高時速 288公里每小時（156節；179英里每小時） ，巡航高度 7,740公尺（25,390英尺）。西元1935年八月進行測試，但沒有因為TB-3變得過時而沒有進入生產。<br /><br />'''TB-3-4-34FRN/FRNV'''   <br />使用AM-34FRN/FRNV發動機，增加了馬力輸出並使用四螺旋槳葉片、空氣動力學的改進，包括精簡炮台，機輪安裝了煞車，最高時速超過 300公里每小時（160節；190英里每小時）。<br /><br />'''TB-3D''' <br />預計使用750匹馬力(740匹)的Charomsky AN-1柴油引擎，原本預估作戰範圍可達4,280 公里，因為其他特性不如TB-3-7-34RN而未投產。<br /><br />'''G-2'''<br />使用M-17和M-34發動機的退役TB-3，被[[俄羅斯航空|俄羅斯航空]]轉用於貨運業務。<br /><br />'''ANT-6-4M-34R"Aviaarktika"'''<br />封閉機體的TB-3，被用於西元1937年的北極遠征，使用直徑2公尺（6.6英尺） 的機輪、三葉片金屬螺旋槳。

== 事故和事件 ==

; 西元1938年三月17日
: 一架Polyarnaya Aviatsiya G-2(CCCP-N210)在降落Bukhta Teplits時坠毁；地面的大雾迫使機組員[[重飛|重飛]]。因為差勁的[[機員資源管理|CRM]] ，機組員將飛機再次飛入濃霧中。飛機高度持續降低，直到起落架卡在雪中，且剥去了起落架，造成飞机坠毁；機上七名成員皆倖存，但这架飞机後來被注销報廢。
; 西元1941年三月14日
: 一架俄罗斯航空的G-2(CCCP-L1496)因為失速而在[[烏茲別克|烏茲別克]]的Begovat附近坠毁。當時飛機受到亂流影響，導致飛機高度下降，機組員試圖爬升，但卻導致飛機失速。該意外導致基上六名機組員全數罹難。當時該機正在執行Tashkent到Fergana的货运服务。
; 西元1941年八月27日
