《'''稻香'''》是[[周杰伦|周杰伦]]专辑[[魔杰座|魔杰座]]的主打歌，由周杰伦填词作曲。这首歌曲内容上的灵感很多来源于周杰倫自小的生活经验，歌曲的创作来源是有感于近来在电视上看见有关天灾的报导，加上身边不少朋友常陷在困难的负面情绪裡<ref>{{cite web|url= http://m2.21cn.com/newdisc/mv/2008/09/25/5246771.shtml|title= 周杰伦《稻香》MV|accessdate= 2009-07-06|work= 21cn|date= 2008-09-25|deadurl= yes|archiveurl= https://web.archive.org/web/20090122144159/http://m2.21cn.com/newdisc/mv/2008/09/25/5246771.shtml|archivedate= 2009-01-22}}</ref>。

== 改编 ==
在魔杰座发行后，这首歌便遭恶搞，网友改编歌词变为《肉香》，以讽刺食堂饭菜质量不高，透露出对学校食堂饭菜的无奈<ref>{{cite web |url= http://news.163.com/08/1105/11/4Q00CARP000128OQ.html|title= 周杰伦新歌《稻香》遭恶搞变《肉香》|accessdate=2009-07-06 |work= [[网易|网易]]|date= 2008-11-05}}</ref>。

另外，由一人大女生以《超阳光可爱的小美女手语版稻香》为题在0.00.0.0.0.0网上发布的[[手语|手语]]版《稻香》则在网络走红<ref>{{cite web|url=http://www.sx.xinhuanet.com/rdsp/2008-11/30/content_15052407.htm|title=人大女生演绎手语版《稻香》走红网络|accessdate=2009-07-06|work=[[新华网|新华网]]|date=2008-11-30-|deadurl=yes|archiveurl=https://web.archive.org/web/20091023113902/http://www.sx.xinhuanet.com/rdsp/2008-11/30/content_15052407.htm|archivedate=2009-10-23}}</ref>。

此外，在中國知名彈幕網站[[嗶哩嗶哩|嗶哩嗶哩]]上，也有網友製作[[三國演義_(電視劇)|三國演義 (電視劇)]]中的[[諸葛亮|諸葛亮]]([[唐國強|唐國強]]飾)與[[王朗|王朗]]([[董驥|董驥]]飾)[[音MAD|音MAD]]聲源合成的翻唱版本。<ref>[https://www.bilibili.com/video/av2565074/?from=search&seid=15196787359075501822 【司徒主唱】~稻香~（来自朗朗的正能量）_人力VOCALOID_鬼畜_bilibili_哔哩哔哩]</ref>

[[厦门六中|厦门六中]]初中部室内合唱团[[阿卡贝拉|无人声伴奏]]改编了《稻香》和《[[青花瓷_(歌曲)|青花瓷]]》两首歌曲<ref>{{cite web |title=14名初中生用杯子“敲出”《稻香》：排练一个月，反响不错 |url=http://www.sohu.com/a/225402276_260616 |website=搜狐 |accessdate=2018-06-13 |date=2018-03-12}}</ref>，其中[[周杰伦|周杰伦]]本人曾与合唱团合唱《稻香》一曲<ref>{{cite web |title=厦门六中合唱团和周杰伦同台演出 周董：唱得比我好 |url=http://fj.sina.com.cn/news/s/2018-04-01/detail-ifysuufh4222997.shtml |website=新浪网 |accessdate=2018-06-13 |date=2018-04-01}}</ref>。
