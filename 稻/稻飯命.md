'''稻飯命'''（{{jpn|j = 稲飯命|hg = いない<small>（或いないい）</small> の みこと/いいなひめ|rm = Inapi no mikoto|kyu = 稻飯命|hgt = いなひ<small>（或いないひ）</small> の みこと/いいなひめ}}，稻飯姬命）為《[[日本書紀|日本書紀]]》裡的記述，《[[古事記|古事記]]》則記作稻冰命或彥稻冰命，是[[神武天皇|神武天皇]]（日本第1代[[天皇|天皇]]）的皇兄，也是[[新羅|新良國]]國主。稻飯命，是[[新羅|新羅]]國王之祖也。

== 事績 ==
《日本書紀》則記載'''稻飯命'''隨神武天皇[[神武東征|東征]]，至熊野（今[[和歌山縣|和歌山縣]]、[[三重縣|三重縣]]南部）外海突遇[[暴雨|暴風雨]]襲擊，祂感嘆：「嗟乎，吾祖則[[天神_(日本神話)|天神]]，母則[[海神|海神]]。如何厄我於陸，復厄我於海乎？」言畢乃拔劍入海，化為[[鋤持神|鋤持神]]（{{jpn|j = 鋤持神|hg = さいもち の かみ|rm = Tsapimoti no kami|kyu = |hgt = さひもち の かみ}}）。

=== 父母 ===
* 父：[[彥波瀲武鸕鶿草葺不合尊|彥波瀲武鸕鶿草葺不合尊]]（彥波瀲尊、彥澰尊） - [[彥火火出見尊|彥火火出見尊]]之子。[[天照大神|天照大神]]之四世孫。
* 母：[[玉依姬|玉依姬]] - [[海童|海童]]之小女。彥波瀲武鸕鶿草葺不合尊之[[姨|姨]]・[[乳母|乳母]]。

稻飯命，是彥波瀲武鸕鶿草葺不合尊的第二子也。

=== 同母兄弟姉妹 ===
* 兄：[[彥五瀨命|彥五瀨命]]（五瀨命） - 中賊矢薨于軍。
* 弟：[[三毛入野命|三毛入野命]]（三毛野命） - 渡于[[常世鄕|常世鄕]]。
* 弟：[[稚三毛野尊|稚三毛野尊]]（神日本磐余彥尊、神倭天皇） - 神武天皇（日本第1代[[天皇|天皇]]）。生母[[豐玉姬|豐玉姬]]（入[[海|海]]而歸[[龍宮|龍宮]]）。
