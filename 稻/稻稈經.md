'''稻稈經'''（梵語：śālistamba sūtra），為敘述[[緣起|緣起]]之[[大乘|大乘]]佛經，收於《[[大正藏|大正藏]]》經集部。以[[佛陀|佛陀]]見[[稻稈|稻稈]]之生長，告諸比丘：「若見因緣，彼即見法；若見於法，即能見佛。」為楔子，繼由[[舍利弗|舍利弗]]就此說，請[[彌勒菩薩|彌勒菩薩]]敷演其義<ref name="香光">{{cite web|title=佛說大乘稻芉經|url=http://www.gaya.org.tw/library/readers/guide-dgj.htm|website=佛書解題（香光版）}}</ref>。

== 題解 ==
從[[稻稈|稻稈]]生長的[[緣起|緣起]]，由種生芽，由芽生葉等，喻[[十二因緣|十二因緣]]生起的次第<ref name="香光"/>。

== 傳譯 ==
《稻稈經》的[[梵語|梵語]]本佚失不傳，不過有根據大乘論書所引用的文句加以整理而成的梵文本，西藏大藏經收有藏譯本<ref name="香光"/>。

本經有五種漢譯本<ref name="香光"/>：
*吳．[[支謙|支謙]]譯《了本生死經》
*東晉．闕譯《稻稈經》
*唐．[[不空|不空]]譯《慈氏菩薩所說大乘緣生稻𦼮喻經》
*宋．[[施護|施護]]譯《大乘舍黎娑擔摩經》
*失譯《大乘稻芉經》（敦煌本，江味農推測可能是唐．[[法成|法成]]譯）

== 內容 ==
本經將[[緣起|緣起]]法分為「內緣起」與「外緣起」，內、外緣起又各皆從因與緣而得生；即於內、外緣起之下復分因與緣之聯繫，此中，「內緣起」，約[[眾生|眾生]]生死的起滅說；「外緣起」，則約身外的一切而說，即[[無情|無情]]器界<ref name="釋惠敏">{{cite book|author1=釋惠敏|title=生命緣起觀: 梵本《淨明句論・第二十六品觀十二支分》初探|date=1996|publisher=法鼓|pages=17|url=https://books.google.com.tw/books?id=FNLk_gyWFhsC&lpg=PA17&ots=-nb2Qr4rhr&hl=zh-TW&pg=PA17#v=onepage&q&f=false}}</ref>。

又，凡愚眾生執眼等[[六根|內六處]]為「[[我_(佛教)|我]]」（人），於色等[[六塵|外六處]]執為「[[我所執|我所]]」（法）；故此經說：二種緣起同一無我（人、法二[[無我|無我]]），因緣所生故<ref name="釋惠敏"/>。

[[月稱|月稱]]論師所著《[[入中論|入中論]]》對[[空性|空性]]的論證，來自於龍樹的《[[中論|中論]]》再加上對《稻稈經》譬喻的引用<ref>{{cite journal|author1=劉婉俐|title=珍貴與神聖的身體： 藏傳佛教「道次第」與身體觀|journal=輔仁宗教研究|date=2013|volume=27|page=93 - 126|url=http://www.rsd.fju.edu.tw/Upload/27-7%E8%97%8F%E5%82%B3%E4%BD%9B%E6%95%99%E3%80%8C%E9%81%93%E6%AC%A1%E7%AC%AC%E3%80%8D%E8%88%87%E8%BA%AB%E9%AB%94%E8%A7%80-%E5%8A%89%E5%A9%89%E4%BF%90.pdf}}</ref>。

== 注解 ==
西藏大藏經現存有[[蓮華戒|蓮華戒]]《稻芉經廣釋》和記載作者名為龍樹的《大乘稻芉經頌釋》二種注釋書；大南龍昇（1967）推定：後者雖題名「龍樹」，但可能是後人假託之作<ref name="香光"/>。

中文則有唐[[法成|法成]]集《大乘稻芉經隨聽疏》，但依芳村修基（1956）之推定，以為《法成疏》乃依蓮華戒之《廣釋》而成。近代[[太虛大師|太虛大師]]有《佛說大乘稻稈經講記》<ref name="香光"/>。
