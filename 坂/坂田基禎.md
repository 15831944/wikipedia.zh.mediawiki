'''坂田基禎'''（{{lang-ja|さかた もとのり}}，{{lang-en|Motonori Sakata}}；{{bd|1954年}}），[[日本|日本]][[京都市|京都市]]出身的男性建築師。

== 簡歷 ==
坂田基禎從小在[[京都府|京都府]]京都市長大，1973年畢業於[[京都府立山城高等學校|京都府立山城高等學校]]，1979年則修畢[[福井工業大學|福井工業大學]]建築系建設工學科課程，取得了學士學位{{noteTag |name = MS |見於坂田基禎建築研究所官方網頁「PROFILE」。}}；同年4月加入 AZ Institute 環境計劃研究所工作，期間成功考獲一級建築師資格，至1985年6月便選擇離職，開設坂田基禎建築研究所，並於1990年代奪取多個獎項{{noteTag |name = MS}}。

2009年起，坂田基禎曾多次參與[[朝日放送|朝日放送]]電視節目《[[全能住宅改造王|全能住宅改造王]]》，因擅長將京都建築風格融入家居設計而受到注目<ref>{{Cite news |language = zh |author = 二姑娘家 |url = http://www.sohu.com/a/254589835_651183 |title = 住了半世纪的百年老危房，这两家人终于被解救了 |publisher = [[搜狐|搜狐]] |date = 2018-09-18 |accessdate = 2018-09-19}}</ref><ref>{{Cite news |language = ja |url = http://book.kyotoliving.co.jp/living/170218c/html5.html#page=3 |title = 3月11日（土）、「匠によるリノベーション セミナー」が開かれます |publisher = 《Living京都》中央版 第1792號（[[Sankei_Living新聞社|株式會社京都 Living 新聞社]]） |pages = 3 |date = 2017-02-18 |accessdate = 2017-02-19}}</ref>，又透過其他訪問分享個人對建築的見解與翻新住宅或設施的要訣<ref>{{Cite journal |author = 北川靖 |url = https://www.kyoto.med.or.jp/member/report/pv/ishoku003/_SWF_Window.html |title = 気鋭建築家坂田基禎が思い描く人と住まいの幸福の関係 |journal = 《京の医食住》（[[日本醫師會|京都府醫師會]]） |year = 2016 |volume = 3 |pages = 2 - 7}}</ref>；2017至2018年亦曾擔任[[京都|京都]]早晨[[扶輪社|扶輪社]]（日文原名：{{lang|ja|京都モーニングロータリークラブ}}）社長<ref>{{Cite news |language = ja |url = http://archive.fo/0626B |title = 会長あいさつ – 京都モーニングロータリークラブ（2017 - 2018年度） |publisher = [[國際扶輪|國際扶輪]]2650地區 |accessdate = 2019-04-10}}</ref>。
