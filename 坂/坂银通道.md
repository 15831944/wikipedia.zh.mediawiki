'''坂银通道'''是[[深圳市|深圳市]]兴建中的一条市政道路，通道南起[[北环大道|北环大道]]黄木岗立交北侧，依次下穿金湖调蓄库、鸡公山、[[厦深铁路|厦深铁路]]，北接[[坂雪岗大道|坂雪岗大道]]，线路整体呈“S”形，为双向六车道设计，全长10.7公里。全线共设坂银-泥岗、坂银-北环、坂银-南坪、坂银-环城四座互通立交，设鸡公山特长隧道一座，全长约4.5公里{{notetag|左线长约4540.8米，右线长约4625米}}{{refTag|1={{cite web|url=http://www.sz.gov.cn/cn/xxgk/szgg/tzgg/201304/P020130402467608723267.pdf|title=坂银通道规划设计初步方案|publisher=深圳市交通运输委员会|date=2013-04|format=pdf|deadurl=yes|archiveurl=https://web.archive.org/web/20141020051336/http://www.sz.gov.cn/cn/xxgk/szgg/tzgg/201304/P020130402467608723267.pdf|archivedate=2014-10-20}}}}。坂银通道已于2014年12月开工建设，预计于2017年12月通车。{{refTag|name=kg1|1=[http://www.sztb.gov.cn/jtzx/gzdt/gldt/201410/t20141020_45541.htm 坂银通道将于今年底动工（图）],深圳市交通运输委员会，2014-10-20}}
== 历史 ==
*随着深圳发展和特区内外城市一体化的推进，以及小汽车拥有量的快速增加，深圳二线关的交通特别是中部轴线上需求持续快速增长，而当时二线关中轴线主要道路仅有[[皇岗路|皇岗路]]，交通压力与日俱增。{{refTag|name=kg1}}
*2010年，为解决中部轴线龙华与福田往来道路于梅林检查站之间的交通拥堵，[[深圳市|深圳市]]计划新修一条通道，连接现有的[[彩田路|彩田路]]以及[[新区大道|新区大道]]，以实现分流[[皇岗路|皇岗路]]交通出行的压力。{{refTag|name=xc|1={{cite web|url=http://www.sztb.gov.cn/jtzx/gzdt/gldt/201004/t20100419_9521.htm|title=市交通运输委公示中部干线路网一体化改善规划|publisher=深圳市交通运输委员会（深圳市港务管理局）|accessdate=2014-07-30|deadurl=yes|archiveurl=https://web.archive.org/web/20140808041842/http://www.sztb.gov.cn/jtzx/gzdt/gldt/201004/t20100419_9521.htm|archivedate=2014-08-08}}}}，意在直接解决福田中心区、上步中心区与龙华新城之间的机动车通行效率。同年，工程的重要组成之一新彩隧道动工，2012年底皇岗-彩田立交开始改造{{refTag|1={{cite web |url=http://www.sznews.com/news/content/2012-11/15/content_7394852_3.htm |title=新彩通道隧道开挖 皇岗彩田立交开4匝道 |publisher=深圳新闻网 |accessdate=2014-07-30 |archive-url=https://web.archive.org/web/20150924112916/http://www.sznews.com/news/content/2012-11/15/content_7394852_3.htm |archive-date=2015-09-24 |dead-url=yes }}}}。2014年7月，新彩通道提前竣工并启用。日后的调查显示，新彩通道的建设成功实现了梅林关的分流，效果显著。{{refTag|1={{cite web|url=http://www.sztb.gov.cn/jtzx/tpxw/201407/t20140730_43331.htm|title=新彩通道通车了！（图）|publisher=深圳市交通运输委员会|date=2014-07-30|language=zh-hans|accessdate=2015-11-16|deadurl=yes|archiveurl=https://web.archive.org/web/20151117202448/http://www.sztb.gov.cn/jtzx/tpxw/201407/t20140730_43331.htm|archivedate=2015-11-17}}}}。
*在新彩通道提出兴建的同时，为缩短坂田片区与[[罗湖区|罗湖]]、[[福田区|福田]]片区的出现，深圳市政府还规划了坂银通道，初步方案为拟起于[[北环大道|北环大道]]，止于坂雪岗大道，全长6.6公里。但由于涉及到[[深圳地铁16号线|深圳地铁16号线]]的规划{{notetag|现在为[[深圳地铁10号线|深圳地铁10号线]]，并且深圳市政府已经决定提前建设，线路不会影响到坂雪岗大道}}，当时还处于方案讨论阶段。
*2013年10月，深圳市交通运输委官方网站发布《坂银通道工程环境影响报告书》，方案显示坂银通道为双向六车道设计，其中最外侧车道为社会车道，中间为高峰期普通公交车道，最内侧为快速公交车道，这是[[深圳市|深圳市]]继新彩通道采用快速公交道后第二个采用快速公交道的市政道路。{{refTag|1=[http://www.sztb.gov.cn/jtzx/gzdt/gldt/201310/t20131016_23562.htm 市交委启动坂银通道项目第二次环评公示]深圳市交通运输委员会,2013-10-16}}
*2014年12月31日，坂银通道正式开工建设。{{refTag|1=[http://www.sznews.com/news/content/2014-12/31/content_10961335_2.htm 坂银通道开工建设 3年后坂田到北环只需12分钟(图)],深圳新闻网，2014-12-31}}
*2014年9月，鸡公山特长隧道南北两端的洞口已经形成，即将进入隧道施工阶段。{{refTag|1=[http://news.sina.com.cn/c/2015-09-16/doc-ifxhxzxp4352622.shtml 坂银通道将进入隧道施工阶段]南方日报，2015-09-16}}

==== 泥岗立交 ====
{{ multiple image | align = right | direction = vertical | width = 240

坂银-泥岗立交为复合通道高架桥，为双向5车道设计，南行共2条车道，1条连接[[华富路|华富路]]（泥岗西路），另一条连接上步路；北行2条车道连接华富路，1条连接上步路。北环大道-泥岗立交节点处为双向六车道，与两端道路双向八车道不符，在泥岗立交兴建时将同步对该节点进行扩宽，北侧道路在现状桥墩之间增设1条车道，南侧道路在现有桥梁的基础上扩宽1条车道。由于泥岗立交[[上步路|上步路]]匝道平面与现状跨泥岗西路桥梁重合，需要拆除重建；同时在华富路设体育馆西一座快速公交站点，届时华富路将由双向六车道扩宽至双向十车道,同时将新建一座天桥（设有垂直电梯）。{{refTag|name=kg1}}
==== 北环立交 ====
{{ multiple image | align = right | direction = vertical | width = 240
|image1=Banyin-jigongshan-a.JPG|caption1=鸡公山隧道口|image2=|caption2=
}}
坂银-北环立交共设6条匝道，其中南线有3条，在北往南出隧道后设置北往西匝道、北往东匝道以及北往南匝道（泥岗立交复合通道）。其中北往东匝道通过接入新建拓宽的[[北环大道|北环大道]]西侧辅路，通过现状泥岗上步立交，可以满足华富、上步及泥岗东的转向交通需求，该路段通过北环西辅路内侧车道与坂银北环段高架双向五车道形成坂银双向六车道的复合通道，完成高峰期公交与社会车辆提前分离，可避免公交与社会车辆的交织，实现公交优先的规划意图。北线共有2条匝道，设有南往北匝道、西往北匝道；除此之外还将对银湖跨线桥进行改造，在北环大道东行增设一条跨线桥连接金湖路。受主线上跨净高以及纵坡影响，需要对金湖路整体抬高3.5米，同时金湖路将被扩宽至双向5车道{{refTag|name=kg1}}。2013年4月2日，深圳市交通运输委员会公布了坂银通道的初步设计方案，根据设计，北环立交隧道口恰好位于金湖调蓄库内，需要临时抽干湖内的水才能保证施工的正常进行，但在方案公布初期北环立交有两种方案，一是以上跨的方式跨越金湖调蓄库，二是以闭合框架下沉式隧道的方式下穿金湖调蓄库，不论哪一种方式施工难度都非常大{{refTag|1=[http://www.s1979.com/dianshi/first/20130403/829867.shtml 坂银通道 方案如何定 市民有话说]深圳广电集团,2013年4月3日 21:15}}。最终坂银通道采用市民强烈支持的闭合框架下沉式隧道的方案，但是闭合框架占用了金湖调蓄库部分容量，影响调蓄库的防洪，施工方将视情况将金湖调蓄库的堤坝加固以及提高，同时施工方还将面临堤坝的稳定性、闭合框架的防渗性等诸多难题。
