'''坂井泉水'''（本名：'''蒲池幸子'''，{{bd|1967年|2月6日|2007年|5月27日|catIdx=Sakai Izumi}}）是[[日本|日本]]女性[[歌手|歌手]]、[[作詞家|作詞家]]，她是日本流行歌曲團體[[ZARD|ZARD]]的主唱。

== 人物背景 ==
出生於[[神奈川縣|神奈川縣]][[平塚市|平塚市]]，家庭成員為父母、弟妹的五人家族，乃五人家庭的長女，下有一妹一弟，據說和妹妹長相相似，曾有歌迷誤認。就學過[[神奈川縣|神奈川縣]][[平塚市|平塚市]]、[[秦野市|秦野市]]成長(小學4年)，[[秦野市立西小學|秦野市立西小學]]，[[秦野市立西中學|秦野市立西中學]]，[[神奈川縣立伊志田高中|神奈川縣立伊志田高中]]畢業。學生社團活動曾參與：初中時代田徑隊，高中時代硬式網球隊。在[[松荫女子短期大学|松蔭女子短期大學]](也就是今天[[松蔭大學|松蔭大學]]的前身)畢業後，在新宿地區的大型企業"第一不動產"工作約兩年。

== 出道 ==
1988年，仍為不動產公司接待小姐的坂井泉水，被星探發掘，在Stardust Promotion事務所旗下，以本名蒲池幸子進行模特兒活動。

1989年起，東映卡拉OK皇后選拔，從三千位參賽者中脫穎而出獲勝。除了巡迴日本推銷東映的伴唱機，也拍攝共五首音樂MV：[[Wink|Wink]]「One Night In Heaven 〜真夜中のエンジェル〜」、[[中森明菜|中森明菜]]「CARIBBEAN」、[[永井真理子|永井真理子]]「23才」、[[藥師丸博子|藥師丸博子]]の「手をつないでいて」、[[家鋪隆仁|家鋪隆仁]]「未練〜STILL〜」。這份工作一直到以ZARD出道後不久才結束。

在電視劇（1989年「春燈」、1990年「{{lang|ja|渋谷・青春坂・ラブホテル}}」），及綜藝節目上數次出演。

1990年擔任[[日清|日清]]杯麵摩托車隊的[[賽車皇后|賽車皇后]]。

1990年拍攝寫真影片「Body Works」、「SEXY SHOOTING」，及寫真集「NOCTURNE」。另也拍攝雜誌照片。

== ZARD主唱時代 ==
{{main|ZARD}}
1990年8月，坂井泉水參加B.B.QUEENS合音甄選。雖然最終落選，實力卻被[[Being,Inc.|Being,Inc.]]社長[[長戶大幸|長戶大幸]]看中，決定以坂井泉水為中心，組成樂團[[ZARD|ZARD]]。從屬[[Being,Inc.|Being,Inc.]]關聯公司，「泉水」擔任該公司的副社長。1991年2月10日，ZARD以『Good-bye My Loneliness』初次亮相，由坂井泉水擔任[[ZARD|ZARD]]主唱，唱紅許多歌曲，成為90年代日本具有代表性的女歌手<ref>[http://www.oricon.co.jp/news/rankmusic/45018/ ZARD、6作品がランキングを急上昇 ニュース-ORICON STYLE]</ref>。初期在電視上曾表演過數次，1993年單曲「別認輸」之後就完全沒在媒體前曝光，徹底進行專心致志音樂製作。

== 特質與理念 ==
坂井泉水透露她不喜歡吃生魚片、穿高跟鞋和紅色衣服。特別鍾愛蓝色和白色，喜欢看电影和画画，她自己说过喜欢的乐器依次有1鼓2貝斯3吉他，最喜歡[[Bass_drum|Bass drum]]的聲音。她非常喜歡有古典氣息的加入弦樂和交響樂的風格。比如《永遠》就符合這樣的感覺。

她是一个细腻敏感的女性，工作時間大部分都在黃昏到凌晨這段時間，而入睡的時間大部分為早上，所以除了工作人員之外，很少有其他的朋友。

她並不擅長面對鏡頭，每當攝影師把鏡頭對準她，或者是在確定拍攝場景後，需要根據攝影師的想法去調整姿勢時，她總是顯得不自在。於是身為“坂井泉水”所擁有的魅力與細微的表情就會難以發揮出來。因此ZARD的CD封面，以坂井泉水的側面為封面。而不直視鏡頭的畫面佔大多數，因為若刻意去擺姿勢拍照的話，往往就拍不出理想的效果。事實上，如果不讓攝影師而是讓經紀人和工作人員潛入她的錄音室內，從側面的角度來捕捉她真實的工作情景，反而拍出來的表情會特別好。經紀人曾經嘗試這麼做，她也不會去介意鏡頭的存在，有時甚至沒有意識到自己正在被拍照。而這些時刻拍下來的照片，表情都顯得相當自然，完全展現了“ZARD”的魅力。

坂井泉水極為重視歌詞，在2004年演唱會上，曾經明白向歌迷傳達這樣的意念：“我總是非常珍視語言和歌詞，希望這一點能夠通過音樂來傳達給你們”。

除了歌詞，坂井泉水對於自己的演唱要求亦非常嚴格。她這種對細節不懈追求的性格很讓人敬佩。一聊起日常生活中的事情她總是非常開朗，讓人感到這是一個可愛隨和的女性。她極為關心身旁的人，比如對工作人員的照顧往往比工作人員還周到。在她留下的光輝成績背後，我們可以看到她把人生的大部份時間都用在了歌曲創作上。

ZARD給人的感覺就是“總是在錄製歌曲”，每週的二、四、六一直定期地進行，也就是說每個星期二、四、六她肯定是在唱歌的。比如說晚上初步錄完，工作人員會簡單做一下歌曲的合成，她會把半成品帶回家，檢查當天唱出的效果，然後下次在录的時候她自己就會注意在演唱時把那些上次不滿意的地方進行改善。她撰寫歌詞的時候一直到開始錄製之前的最後一刻都會仔細揣摩什麽樣的歌詞最能打動人們的內心。從那佈滿線條痕跡、不知改動了多少次的歌詞紙上不難看到她對“最深入人心的語言”所不懈追求的身影。

== 和病魔奮戰 ==
從2000年起，坂井泉水的身體就不斷出狀況，包含[[子宮肌瘤|子宮肌瘤]]、[[卵巢囊腫|卵巢囊腫]]、[[子宮內膜異位|子宮內膜異位]]症等，導致身體日漸虛弱。也因此在2001年休養，未發行新專輯。坂井泉水一直到2003年才全面恢復工作狀態。然而在2006年，拍完「點燃心火」的MV後當晚，身體突然不適入院，經檢查後證實罹患[[子宮頸癌|子宮頸癌]]。6月接受了摘除[[子宮|子宮]]的手術，後繼續休養。但2007年4月發現[[癌細胞|癌細胞]]擴散至[[肺部|肺部]]，再度入院治療。由於抗癌劑的治療良好，坂井泉水還準備於[[秋天|秋天]]復出歌壇。

== 醫院意外 ==
[[File:Entertainment_agency_after_Izumi_Sakai's_death.jpg|right]]
* 2007年5月26日清晨約5點40分，在慶應義塾大學病院接受癌症治療的坂井泉水，在散步後準備返回病房時，意外從醫院一二樓連接間斜坡摔落約3米，倒臥在醫院通往停車場的入口。雖然被路過民眾發覺後即時進行急救，但頭部受到嚴重撞擊，並於5月27日下午3點10分逝世。
* 2007年5月27日，坂井因'''脑挫裂伤'''（{{lang-en|{{tsl|en|Cerebral contusion|Cerebral contusion}}}}）逝世，享年40歲<ref>日歌手坂井泉水 墜樓梯身亡，[[民視|民視]]新聞</ref>。
* 2007年5月28日所屬事務所於官方網站wezard.net發出希望「[[ZARD|ZARD]]的音樂能一直在人們心中傳唱」的聲明。由於抱著不讓歌迷擔心的想法，坂井泉水相當低調，歌迷是透過這紙聲明，才得知已罹癌一年的消息。
* 2007年5月30日，在[[東京都|東京都]][[町田市|町田市]]的齋壇中，舉辦家族的[[葬禮|葬禮]]，戒名為『澄響幸輝信女』。
* 同時東京及大阪的經紀公司設置的獻花台湧進上萬名歌迷弔念她。
* 2007年6月26日，在東京青山喪葬儀所舉行的追思會，和坂井泉水有關的音樂人出席。如：[[織田哲郎|織田哲郎]]、[[大黑摩季|大黑摩季]]、[[池森秀一|池森秀一]]、[[小松未步|小松未步]]、[[中村由利|中村由利]]、[[淺岡雄也|淺岡雄也]]等。當日，[[B'z|B'z]]成員（[[松本孝弘|松本孝弘]]和[[稻葉浩志|稻葉浩志]]）、[[倉木麻衣|倉木麻衣]]和[[讀賣巨人隊|讀賣巨人隊]]終身名譽監督[[長嶋茂雄|長嶋茂雄]]發出了追悼評語。
* 2007年6月27日，在東京青山喪葬儀所開放給一般歌迷參加的音樂葬，約有40100位歌迷到場和坂井泉水道別，人數之多是戰後自[[hide|hide]]（約5萬人）、[[美空雲雀|美空雲雀]]（約4萬2000人）、[[忌野清志郎|忌野清志郎]]（約4萬3000人）後第四多的。 
* 2007年12月31日，生涯首度出現在[[紅白歌合戰|紅白歌合戰]]，卻是以錄影方式登場，以供追憶。
* 2006年10月發表的精選輯『Golden Best ～15th Anniversary～』27日以第6名再回[[Oricon|Oricon]]大碟公信榜，週榜也因此從300名外前進至第3名，這是[[Oricon|Oricon]]歷史上從未有過的現象，Le Portfolio 1991-2006(DVD)週榜也前進至DVD音樂部門第1名。

== 其他 ==
*依照蒲池家的心願，坂井泉水的長眠之處不公開。
*坂井泉水身故後，家人在其房間中找到近300份歌詞。
*除了作為歌手，坂井泉水也出版過四本詩詞集。此外也有學習油畫，其中數幅在逝世後的紀念展上展出。
