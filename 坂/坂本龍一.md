'''坂本龍一'''（{{lang-ja|坂本 龍一}}，{{bd|1952年|1月17日|||Sakamoto, Ryuuichi}}），是一位在[[西方國家|西方國家]]有影響力的[[日本|日本]][[作曲家|作曲家]]、[[演员|演员]]，一生獲獎無數；所創作的音樂曲風空靈脫俗、融合東西古今，是日本當代繼[[喜多郎|喜多郎]]之後，可稱為世界級音樂大師的代表人物，被譽為新音樂教父。

1999年後積極參與環保、和平活動，亦發起森林再造保育計畫“More Tree”；2006年成立全新音樂品牌“commmons”，2009年2月於日本出版第一部口述自傳《音樂使人自由》<ref>[http://ryukyushimpo.jp/news/prentry-142381.html 星（スター）を背負った人生　『音楽は自由にする』　坂本龍一著]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 生涯與事業 ==
坂本龍一出生于[[東京都|東京都]][[中野區|中野區]]，自小就學習[[古典音樂|古典音樂]]，畢業於[[東京藝術大學|東京藝術大學]]研究所，主修[[電子音樂|電子音樂]]及[[民族音樂|民族音樂]]，渐露头角始于他与[[细野晴臣|细野晴臣]]以及[[高桥幸宏|高桥幸宏]]组建的合成器摇滚乐队[[黃色魔術交響樂團|黃色魔術交響樂團]]（YMO）；该乐队在1980年代晚期和1990年代早期对[[acid_house|acid house]]和[[techno|techno]]运动有着开创式的影响，另類搞怪風格也令人側目卻廣受歡迎（例如：早年1970年代末[[黑膠唱片|黑膠唱片]]封面三人著[[中國共產黨|中國共產黨]]服[[紅星|紅星]]帽黑布鞋，與一名金髮美國[[比基尼|比基尼]]泳裝美女，同桌翹、縮腳在長板凳打[[麻將|麻將]]）。他和[[大卫·西尔维安|大卫·西尔维安]]（David Sylvian）合作了一系列[[单曲|单曲]]和大部分西尔维安的专辑。

1992年，坂本與紐約音樂怪客DJ Spooky为超过十亿观众通过电视收看的[[1992年夏季奧林匹克運動會|巴塞罗那奥运会]][[开幕仪式|开幕仪式]]谱曲。

1998年為[[三共製藥|三共製藥]]EB錠所製作的廣告配樂《Energy flow》，於日本銷售達百萬張，療效遠勝於EB錠；坂本說：這首歌獻給承受社會壓力的人。

2014年7月坂本龍一在其官方網站說明醫師診斷[[喉癌|喉癌]]，需要休養，決定暫時將工作放下，好好遵照醫師囑咐治療，並保證在痊癒後會再全心投入音樂工作。

== 電影配樂 ==
1983年他在[[大岛渚|大岛渚]]執導的電影《[[俘虜_(電影)|俘虜]]》（戦場のメリークリスマス／Merry Christmas, Mr. Lawrence）中，演出一位在印尼[[戰俘營|戰俘營]]的跋扈日軍營長，參與演出此片的[[演员|演员]]還有英国摇滚歌手[[大卫·鲍伊|大卫·鲍伊]]與[[北野武|北野武]]等。他还为此片写了电影音乐，其中由「Japan」樂團主唱[[大衛·西爾維安|大衛·西爾維安]]所演唱的主题曲《[[禁色|禁色]]》（''Forbidden Colours''）引起了一定的轰动；此片在世界獲得極大成功，大受各界影評好評，且榮登[[坎城影展|坎城影展]]評審團激賞大獎殊榮，並再次將上述三位演員演藝生涯再次捧上高峰。

1987年，他为[[柏納多·貝托魯奇|柏納多·貝托魯奇]]的影片《[[末代皇帝_(电影)|末代皇帝]]》（''The Last Emperor''）所作的音乐获得了[[奥斯卡奖|奥斯卡奖]]，並再度飾演監控[[溥儀|溥儀]]的日本軍官[[甘粕正彦|甘粕正彦]]。他还为[[佩德羅·阿爾莫多瓦爾|佩德羅·阿爾莫多瓦爾]]的影片《[[高跟鞋_(電影)|高跟鞋]]》（''High Heels''）以及[[奥利佛·史東|奥利佛·史東]]的《[[野棕榈|野棕榈]]》（''Wild Palms''）作曲，此外，[[基努·里维斯|基努·李維]]主演的《[[小活佛|小活佛]]》（''Little Buddha''）及大島渚的閉門之作《[[御法度|御法度]]》之電影配樂，亦出自坂本之手。

== 個人生活 ==
坂本曾娶日本钢琴家兼歌手[[矢野显子|矢野显子]]（現已離婚）为妻，并与她合作了一些她的专辑，女兒坂本美雨也在樂壇發展；[[中谷美紀|中谷美紀]]為其得意弟子，與其合作作品有《[[砂之果實|砂之果實]]》。

== 社會活動 ==
坂本從學生時代起即積極投入社會運動，大學時代曾在[[武滿徹|武滿徹]]的表演現場發放反武滿徹的小冊子，後來與武滿徹深談之後卻成為忘年之交，並於武滿徹過世後，於BTTB中以一曲Opus紀念他。他也因他对[[版权法|版权法]]的批评而出名，他认为在[[信息化|信息化]]时代那玩意已是非常落伍。

2001年因受到蘇格蘭人Chris Moon所演出的紀錄影片「{{link-en|I Shouldn't Be Alive|I_Shouldn't_Be_Alive}}」<ref>（關於[[莫桑比克|莫桑比克]]16年内戰遺留下的大量[[地雷|地雷]]，2005年11月25日播出）</ref>啟發，而發起反[[地雷|地雷]]的音樂計劃-{{link-ja|zero landmine|ZERO LANDMINE}}，並集結了為數眾多的世界音樂人共譜反地雷大作。2006年2月，因反對日本政府通過的《{{link-ja|家電用品安全法|電気用品安全法}}》（PSE法），自發性的募集7萬多名的反對法案簽名聯署；並於其個人[[部落格|部落格]]內，反對[[青森核電廠|青森核電廠]]<ref>（參見[[青森縣|青森縣]][[六所村|六所村]]）</ref>的建制。
