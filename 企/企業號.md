'''企業號'''（'''Enterprise'''或'''Enterprize'''）是一個在西方歷史上經常出現的軍事或非軍事武器命名。「Enterprise」這個字原本就擁有多種解釋，例如[[商業|商業]]上的一種事業體組合稱為「企業」，但也可代表一種美德（武德），是「進取」、「勇往直前」之意。依據《美國海軍字典》（''Dictionary of American Naval Fighting Ships''），船艦稱「Enterprise」取其「勇敢、活躍、及在實際事務上之創意」之意（Boldness, energy, and invention in practical affairs）{{R|USShip_1799}}。然因Enterprise一字之多重意義，在中文中此名稱的翻譯方式有多種不同的版本，通常译为「企業-{}-號」，除此之外也常見到「奮進-{}-號」、「進取-{}-號」、「勇往-{}-號」這樣的稱呼方式。

由於「企業號」這名稱多次出現在[[法國海軍|法國]]、[[英國皇家海軍|英國]]與[[美國海軍|美國]]的海軍艦艇上並且廣受歡迎，因此同樣的命名也被經常性地使用在虛構的事物上，例如電影或電視影集中的各式戰艦、飛船等。

== 法國海軍船艦 ==
法國海軍於1671年至1846年間，至少有23艘帆船命名為'''企業號'''（{{lang-fr|l'Entreprenant}}），如同上述理由，{{lang|fr|Entreprenant}}為法文中的「進取」之意。其中知名者為：
* 企業號快速帆船（{{lang|fr|l'Entreprise}}），實際上就是英國海軍第一艘以企業號為名的軍艦。1705年下服役後不久就遭英軍俘獲，重新命名之後於英國海軍服役。

== 英國皇家海軍船艦 ==
[[英國皇家海軍|英國皇家海軍]]有十艘命名為'''企業號'''的船艦，英文名為'''HMS Enterprise'''或'''HMS Enterprize'''，HMS意指「陛下的船艦」（Her/His Majesty's Ship）。

* {{link-en|企業號快速帆船 (1705年)|HMS Enterprise (1705)}}（HMS Enterprise），英國海軍第一艘以'''企業號'''命名的船隻，原先是法國海軍旗下的快速帆船企業號（L'Entreprise），於1705年從法國人手中擄獲。配有24門砲的企業號在英國海軍艦隊中被分類為第六級帆船（sixth-rate）。
* [[企業號風帆戰艦_(1709年)|企業號風帆戰艦 (1709年)]]（HMS Enterprise），英國海軍第二艘以「Enterprise」命名的船隻，是一艘配有40門砲雙甲板的第五級帆船；首航年份為1709年{{R|UKShip_H88}}。
* [[利物浦號快速帆船_(1741年)|利物浦號快速帆船 (1741年)]]（HMS Liverpool），原本預計成為英國海軍第三艘「Enterprise」，但在1741年下水前五個月改名為利物浦號{{R|UKShip_H88}}。
* {{link-en|諾里奇號風帆戰艦 (1693年)|HMS Norwich (1693)}}（HMS Norwich），一艘1693年成軍、配備有50門砲的四級風帆戰艦。在1744年改裝重造之後降級為五級戰艦，並同時改名為企業號（HMS Enterprise），成為英國海軍第四艘同名軍艦{{R|UKShip_H88}}。再服役後的企業號大部分的時間都是在加勒比海地區活動，直到1764年時除役，並於1771年報廢拆除{{R|UKShip_H88}}。
* {{link-en|企業號風帆戰艦 (1743年)|HMS Enterprize (1743)}}（HMS Enterprize）是一艘8門砲小型風帆戰船，原是一艘西班牙長船（<font size="-1">[[西班牙文|西班牙文]]</font>：barca longa），於1743年在地中海遭英軍擄獲後編入英軍艦隊，分級為單桅縱帆戰船（sloop）並改名為企業號。但與先前英軍的幾艘企業號不同的是，該艦艦名採用的是拼法有點不同的「Enterprize」。該艦主要擔任通信船與補給船用途，直到1748年在梅諾卡島售出。
* [[皇家海軍企業號_(1774年)|皇家海軍企業號 (1774年)]]：英國海軍第五艘以'''企業號'''命名的船隻，是一群27艘[[第六級帆船|第六級]][[快速帆船|快速帆船]]的領航船。於1774年首航，服役至1784年，之後作為招待船。
* [[皇家海軍企業號_(1848年)|皇家海軍企業號 (1848年)]]：英國海軍第六艘以'''企業號'''命名的船隻，原先是一艘商船，於1848年購入，作為[[北極|北極]]探險用。
* [[皇家海軍企業號_(1864年)|皇家海軍企業號 (1864年)]]：英國海軍第七艘以'''企業號'''命名的船隻，是一艘[[單桅縱帆船|單桅縱帆戰船]]。於1864年首航，並於1884年售出。
* [[皇家海軍企業號_(D52)|皇家海軍企業號 (D52)]]：英國海軍第八艘以'''企業號'''命名的船隻，是一艘[[綠寶石級巡洋艦|綠寶石級]][[輕型巡洋艦|輕型巡洋艦]]。於1919年首航，並於[[第二次世界大戰|第二次世界大戰]]中服役。
* [[皇家海軍企業號_(A71)|皇家海軍企業號 (A71)]]：英國海軍第九艘以'''企業號'''命名的船隻，是一艘近海[[勘察船|勘察船]]。服役年份從1959年至1985年。
* [[皇家海軍企業號_(H88)|皇家海軍企業號 (H88)]]：英國海軍第十艘以'''企業號'''命名的船隻，是一艘多功能勘察船([[水道學|水道學]]與[[海洋學|海洋學]]用途)，首航於2002年，並於2003年服役。

四艘皇家海軍船艦也命名為'''企業號'''，但並非授銜戰艦，因此英文名字前頭沒有"HMS"字首。
* 企業號，一艘[[英國東印度公司|英國東印度公司]]的[[明輪艇|明輪艇]]，從1839年至1840年於[[鴉片戰爭|鴉片戰爭]]中與艦隊並肩作戰，並於1852年[[二次英緬戰爭|二次英緬戰爭]]中作戰。
* 企業號，一艘未授銜的[[拖船|拖船]]，從1899年至1919年於[[樸次茅斯造船廠|樸次茅斯造船廠]]服役，於1919年重命名為'''冒險號'''('''Emprise''')。(她持續服役直到1947年。)
* 企業號，一艘未授銜的輔助性巡邏，隸屬[[哈威奇|哈威奇]]地方部隊[[反潛水艇|反潛網]]的[[漂網漁船|漂網漁船]]，服役年份從1914年至1918年。
* 企業號貳，一艘未授銜的[[漂網漁船|漂網漁船]]，基地原先在[[拉恩|拉恩]]，但後來於1915年11月轉移到[[義大利|義大利]]海域。於1916年3月，她撞到[[布林迪西|布林迪西]]外圍的[[水雷|水雷]]而沉沒，有八人罹難。

== 美國海軍船艦 ==
歷史上，迄今為止共存在九艘以'''-{zh-hans:进取号（或企业号）; zh-hant:企業號;}-'''為名的美國海軍船艦（United States Ship Enterprise，縮寫為'''USS Enterprise'''），分別為：

* {{Tsl|en|USS_Enterprise_(1775)|企業號單桅縱帆船 (1775年)}}：美國海軍第一艘以'''企業號'''命名的船隻。原先是未服役的[[英國|英國]][[單桅縱帆船|單桅縱帆船]]（sloop）「喬治號」（George），在[[美國獨立戰爭|美國獨立戰爭]]期間，於1775年被美軍俘獲。重新命名後，她以[[戰鬥用單桅縱帆船|戰鬥用單桅縱帆船]]（sloop-of-war）身份在獨立戰爭服役。
* [[企業號多桅縱帆船_(1776年)|企業號多桅縱帆船 (1776年)]]：美國海軍第二艘以'''企業號'''命名的船隻。這艘[[多桅縱帆船|多桅縱帆船]]（Schooner）在由[[美國大陸海軍|美國大陸海軍]]收購改用於軍事用途之前，曾經是一艘[[武裝民船|武裝民船]]（Privateer，又稱為「私掠船」）。由約瑟夫·坎貝爾上尉（Captain Joseph Campbell）所指揮的企業號主要是在[[契沙比克灣|契沙比克灣]]（Chesapeake Bay）地區擔任運輸、偵察與區域巡防任務，與其相關的文獻紀錄並不豐富。
* {{Tsl|en|USS_Enterprise_(1799)|企業號多桅縱帆船 (1799年)}}：第三艘以'''企業號'''命名的船隻。這艘[[多桅縱帆船|多桅縱帆船]]由[[亨利·史賓塞|亨利·史賓塞]]於1799年在[[馬里蘭州|馬里蘭州]][[巴爾的摩|巴爾的摩]]建造。曾在[[巴爾巴利戰爭|巴爾巴利戰爭]]中作出第一擊，對抗[[的黎波里人|的黎波里人]]所操作的[[的黎波里號|的黎波里號]]。
* [[企業號多桅縱帆船_(1831年)|企業號多桅縱帆船 (1831年)]]：第四艘以'''企業號'''命名的船隻。也是艘多桅縱帆船，於1831年服役。
* [[企業號螺槳動力帆船_(1874年)|企業號螺槳動力帆船 (1874年)]]：第五艘以'''企業號'''命名的船隻。是一艘[[螺槳動力帆船|螺槳動力帆船]]（screw sloop），於1877年服役。
* [[企業號區域巡邏艇_(SP-790)|企業號區域巡邏艇 (SP-790)]]：第六艘以'''企業號'''命名的船隻，是一艘[[汽艇|汽艇]]（motorboat）。[[第一次世界大戰|第一次世界大戰]]期間處於未役狀態。
* [[企業號航空母艦_(CV-6)|-{zh-hans:企业号航空母舰(CV-6); zh-hant:企業號航空母艦 (CV-6);}-]]：第七艘以'''企-{}-業號'''命名的船隻，也是美國海軍第一艘使用此名的[[航空母艦|航空母艦]]。
* [[企業號航空母艦_(CVN-65)|-{zh-hans:企业号航空母舰(CVN-65); zh-hant:企業號航空母艦 (CVN-65);}-]]：第八艘以'''企-{}-業號'''命名的船隻，是世界第一艘核動力航空母艦。
* [[企業號航空母艦_(CVN-80)|企業號航空母艦 (CVN-80)]]：美國海軍第九艘以'''企業號'''命名的船隻，是福特級核子動力航空母艦的三號艦。

== 星艦奇航記 ==
{{main|星艦企業號}}

在科幻影視作品《[[星艦奇航記|星艦奇航記]]》（''Star Trek''）中，前後共出現過九艘命名為'''企業號'''的[[星艦|星艦]]：

=== 地球政府時期 ===
* [[地球星艦企業號_(NX-01)|地球星艦企業號 (NX-01)]]：《星艦奇航記》第五代影集《[[星艦前傳|星艦前傳]]》的主角船。

=== 星際聯邦時期 ===
[[星際聯邦|星際聯邦]][[星際艦隊|星際艦隊]]轄下的'''聯邦星艦企業號'''（United StarShip Enterprise，縮寫也叫'''USS Enterprise'''）包括有：
* [[聯邦星艦企業號_(NCC-1701)|聯邦星艦企業號 (NCC-1701)]]：出現於《星艦奇航記》試映劇集兩集與原始影集《[[星際爭霸戰|星際爭霸戰]]》中。改拍成電影版時船艦改裝，以新造型出現於第一部電影《[[星艦迷航記|星艦迷航記]]》、第二部電影《[[星戰大怒吼|星戰大怒吼]]》與第三部電影《[[石破天驚|石破天驚]]》中，於《石破天驚》對抗[[克林貢人|克林貢人]]的過程中設定自毀，隨後墜落於行星。
* [[聯邦星艦企業號_(NCC-1701-A)|聯邦星艦企業號 (NCC-1701-A)]]：出現於《星艦奇航記》第四部電影《[[搶救未來|搶救未來]]》、第五部電影《[[終極先鋒|終極先鋒]]》與第六部電影《[[邁入未來|邁入未來]]》中，於《邁入未來》中提到即將除役。
* [[聯邦星艦企業號_(NCC-1701-B)|聯邦星艦企業號 (NCC-1701-B)]]：出現在《星艦奇航記》第七部電影《[[日換星移|日換星移]]》，於片中進行處女航。
* [[聯邦星艦企業號_(NCC-1701-C)|聯邦星艦企業號 (NCC-1701-C)]]：出現在《星艦奇航記》第二代影集《[[銀河飛龍|銀河飛龍]]》第三季劇集「[[昨日的企業號_(銀河飛龍)|昨日的企業號]]」中。
* [[聯邦星艦企業號_(NCC-1701-D)|聯邦星艦企業號 (NCC-1701-D)]]：《星艦奇航記》第二代影集《[[銀河飛龍|銀河飛龍]]》的主角船。亦出現在第七部電影《[[日換星移|日換星移]]》中，於該片中墜毀。
* [[聯邦星艦企業號_(NCC-1701-E)|聯邦星艦企業號 (NCC-1701-E)]]：出現於《星艦奇航記》第八部電影《[[戰鬥巡航|戰鬥巡航]]》、第九部電影《[[星際叛變|星際叛變]]》與第十部電影《[[星戰啟示錄|星戰啟示錄]]》中。
* [[聯邦星艦企業號_(NCC-1701-J)|聯邦星艦企業號 (NCC-1701-J)]]：出現在《星艦奇航記》第五代影集《[[星艦前傳|星艦前傳]]》第三季劇集「[[阿扎堤主星_(星艦前傳)|阿扎堤主星]]」中的未來時間線。

== 美國航天飞机 ==
受到《[[星际旅行：初代|星际旅行：初代]]》（《星艦奇航記》第一代影集）對於太空探險的鼓舞，當美國推出[[太空梭|太空梭]]這類太空飛行器時，《星艦奇航記》廣大的影迷寫信給[[美國航太總署|美國航太總署]]要求將之命名為「企業號」，爾後其中一艘太空梭果真命名為[[企業號太空梭|企業號]]，但這架太空梭僅用於自由滑降測試用途，未曾真正進入外太空[[軌道|軌道]]。

== 維京公司航太器具 ==

[[維京集團|維京集團]]之一的[[維京銀河航線公司|維京銀河航線公司]]（Virgin Galactic）計畫推出的次軌道太空飛行之旅，其載具稱為[[維京太空船|維京太空船]]（Virgin SpaceShip，VSS），而第一艘即命名為[[企業號維京太空船|企業號維京太空船]]（VSS Enterprise），用以紀念《星艦奇航記》中的企業號。
