'''湯與海'''是由[[高雄|高雄]][[獨立樂團|獨立樂團]][[淺堤|淺堤]]於2017年8月發行的實體[[單曲|單曲]][[唱片|唱片]]、同年11月上架於[[數位音樂下載|數位唱片]]市場，是繼2016年手工 EP《Demo. 1<ref>{{Cite web |url = https://blow.streetvoice.com/20591-淺堤釋出新歌%E3%80%88多崎作〉手工-demo-巡迴首賣/ |title = 淺堤釋出新歌〈多崎作〉手工 demo 巡迴首賣 |author = JohnnyWen |date = 2016年5月11日 |publisher = 吹音樂 |language = zh-tw |accessdate = 2017年10月28日 |quote = 新生代樂團淺堤 shallow levée 來自高雄，日前釋出的新歌〈怪手〉用台語描述台灣史上最大規模的「紅毛港遷村案」，目前點閱次數已破萬；昨日上傳了新歌〈多崎作〉，歌名取自日本作家村上春樹作品《沒有色彩的多崎作與他的巡禮之年》中的主角之名。}}</ref>》後的新作品，收錄4首單曲，並在發行後的9月隨即展開唱片宣傳巡迴演出<ref>{{Cite web |url = https://zh-tw.facebook.com/twindiemusicrecord/posts/1854474451260072 |title = 淺堤 ＠《湯與海》2017 台灣巡迴 |author = 台灣搖滾映像誌 |date = 2017年10月8日 |publisher = Facebook |language = zh-tw |accessdate = 2017年10月28日 |quote = 若在市場辦演出，會是怎麼樣的光景？來自高雄的淺堤就在故鄉的菜市場舉辦發片巡迴！淺堤是一支木吉他入歌的搖滾樂隊，集結著新生代的重量級樂手陣容，由女主唱兼吉他手蔡依玲領軍，成員包括孩子王吉他手周致宏，大象體操鼓手涂嘉欽，以及來自少年白的貝斯手方博聖。 }}</ref>。
