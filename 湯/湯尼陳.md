'''湯尼陳'''（{{bd|1961年|10月1日}}，[[英文|英文]]名：'''Tony Chen'''），本名'''陳志鵬'''，是[[台灣|台灣]][[男藝人|男藝人]]、英語[[補習|補習]]教師、主持人、網路電視台主播。

== 經歷 ==

2016年5月10日，湯尼陳與好娜姬、[[慧慈|慧慈]]、[[林雅惠|如花]]合組演藝團體「蘆薈雞湯」<ref name="蘆薈湯">{{cite web |url=http://star.ettoday.net/news/694775 |title=夢幻團體成形！ 如花＋慧慈＋湯尼陳組「蘆薈湯」出道|author=林孝庭|publisher=[[ETtoday新聞雲|ETtoday新聞雲]] |date=2016-05-09|accessdate=2016-09-17}}</ref><ref>{{cite news|url=http://www.appledaily.com.tw/realtimenews/article/entertainment/20160510/857557|title=【有片】如花慧慈合體湯尼陳 酸法拉利姊太貴|newspaper=|pages=|publisher=[[台灣蘋果日報|台灣蘋果日報]]蘋果即時 |author=蔡維歆、宇若霏 |date=2016-05-10 |accessdate=2017-05-20|quote=}}</ref>。團體於2018年2月5日解散。

據湯尼陳自述，[[九二一大地震|九二一大地震]]一周年時，[[民主進步黨|民主進步黨]]政治人物[[呂秀蓮|呂秀蓮]]到[[台中市|台中市]]視察[[組合屋|組合屋]]居民生活狀況，與他見面並在自己的著作上簽字留念，讓他留下印象。2001年，陳公開向已上任[[中華民國副總統|中華民國副總統]]的呂示愛，並在新聞媒體前親吻呂之照片，此舉引起話題。2002年2月9日上午，湯尼陳花費[[新台幣|新台幣]]100萬元（全包加一年租金）租下[[中山高速公路|中山高速公路]]彰化段[[交流道|交流道]]西側一座「T霸」（T-bar）[[廣告|廣告]]看板，藉[[西洋情人節|西洋情人節]]樹立示愛廣告：「副總統呂秀蓮我愛妳，英語名師湯尼陳真情告白，Happy Valentine's Day.」示愛廣告掛出後第三天就被拆除<ref name="秀蓮">{{cite web |url=http://www.epochtimes.com/b5/2/2/8/n169494.htm |title=痴戀呂秀蓮 湯尼陳百萬打廣告|publisher=大紀元 |date=2002-02-08|accessdate=2016-09-17}}</ref>。
