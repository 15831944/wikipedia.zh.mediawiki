'''湯瑪斯·史卡佛內克'''（'''Tomasz Schafernaker'''，{{bd|1979年|1月8日}}）是一名[[BBC天氣|BBC天氣]]工作的[[英國|英國]]-[[波蘭|波蘭]][[氣象學家|氣象學家]]。

== 早年生活 ==
史卡佛內克出生於[[波蘭人民共和國|波蘭]]的[[格但斯克|格但斯克]]，在他的家鄉波蘭和英國上學。他在[[樸茨茅夫|樸茨茅夫]][[南海城|南海城]]的獨立學校{{link-en|聖約翰學院 (樸茨茅夫)|St John's College, Portsmouth|聖約翰學院}}裡接受教育，他在學院的數學、物理和藝術方面取得了A級成績，然後在[[雷丁大學|雷丁大學]]獲得了氣象學[[理學士|理學士]]學位。

== 生涯 ==
加入[[英國氣象局|英國氣象局]]後，他被培訓成為氣象學家。他於2000年加入[[BBC天氣|BBC天氣]]中心，擔任廣播助理，並於同年成為報導員；這樣使他成為BBC天氣中最年輕的報導員<ref name="CarolineWestbrook">{{Cite web |url = https://metro.co.uk/2017/05/30/tomasz-schafernaker-has-been-named-britains-favourite-weather-presenter-6671228/ |title = Tomasz Schafernaker has been named Britain's favourite weather presenter |author = Caroline Westbrook |date = 2017-05-30 |publisher = Metro |language = en}}</ref>。他付出了一些時間在遍及英國的BBC地區天氣預報，並出現在2003年冬季的[[英國廣播公司世界新聞頻道|英國廣播公司世界新聞頻道]]。

由2004年到2005年，史卡佛內克在[[德雲郡|德雲郡]][[埃克塞特|埃克塞特]]的英國氣象局學院學習，以及培訓成[[航空|航空]]預報員。他也曾經在[[獨立電視台|獨立電視台]]（ITV）和倫敦氣象中心工作過一段時間。

史卡佛內克於2006年重新加入了BBC。他曾因為在以為關了鏡頭的情況下被BBC新聞報導員{{link-en|西門·麥考伊|Simon McCoy}}善意取笑後「[[豎中指|豎中指]]」而變得聲名狼藉<ref name="Metro2017">{{Cite web |url = https://metro.co.uk/2010/10/13/rob-mcelwee-tomasz-schafernaker-and-philip-avery-axed-from-bbc-546277/ |title = Weathermen axed from BBC to cut costs |date = 2010-10-13 |publisher = Metro |language = en}}</ref>。他的震驚和恐懼表情的片段被放到[[YouTube|YouTube]]上，獲得超過550萬次點擊。在採取了一系列削減成本的措施並選擇一個較小問題的全國天氣報導員後，他在2010年被BBC撤職<ref name="Metro2017" />。他後來在2010年的{{link-en|英國電視和廣播業俱樂部|Television and Radio Industries Club|英國電視和廣播業俱樂部獎}}上被名為最佳電視天氣報導員。

史卡佛內克仍然在[[英國氣象局|英國氣象局]]工作，他曾出現在各種英國電視網上的各種廣播項目，包括{{link-en|天氣頻道|The Weather Channel}}、[[探索頻道|探索頻道]]和[[第四台_(英國)|第四台]]。2012年5月29日，他加入[[BBC東南|BBC東南]]擔任天氣報導員。史卡佛內克於2013年1月19日回到BBC全國廣播公司。他通常在[[英國廣播公司新聞頻道|英國廣播公司新聞頻道]]、[[BBC廣播四台|BBC廣播四台]]或{{link-en|BBC廣播五直播台|BBC Radio 5 Live}}預報天氣。2017年，他在一次公投中被選為英國最受歡迎的天氣報導員。<ref name="CarolineWestbrook" /><ref>{{Cite web |url = https://www.radiotimes.com/news/2017-05-30/tomasz-schafernaker-and-windy-wilson-voted-the-uks-favourite-weather-presenters/ |title = Tomasz Schafernaker and Windy Wilson voted the UK's favourite weather presenters |author = Sarah Doran |date = 2017-05-30 |publisher = Radio Times |language = en}}</ref>

== 雜誌模特兒 ==
史卡佛內克在2010年1月為雜誌《''Attitude Active''》的封面擺姿勢，只穿內褲。<ref>{{Cite web |url = https://www.theguardian.com/media/shortcuts/2017/may/30/tomasz-schafernaker-weather-man-winner |title = From ‘Nowheresville’ to on-air puking: why Tomasz Schafernaker is the UK's fave forecaster |date = 2017-05-30 |publisher = The Guardian |language = en}}</ref>
