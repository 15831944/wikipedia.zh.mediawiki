'''湯姆·謝立克'''（{{lang-en|Tom Selleck}}，{{bd|1945年|1月29日|||catIdx=Selleck}}），1980年代美國[[好萊塢|好萊塢]]知名一線[[動作片演員|動作片演員]]；出生於[[密西根州|密西根州]][[底特律|底特律]]，政治上是美國前總統[[雷根|雷根]]好友及支持者，活躍於影、視、歌三棲長青樹。

== 出身 ==
* 成長於[[洛杉磯|洛杉磯]]。
* 以[[棒球|棒球]]員獎學金求學，大學畢業於[[南加大|南加大]][[企業管理系|企業管理系]]

== 電影 ==
* 1969年演出第一部電影。
* [[三個奶爸一個娃娃|三個奶爸一個娃娃]] [[親情|親情]][[喜劇片|喜劇片]]，擴闊戲路之作
* [[忽然囉囉攣|忽然囉囉攣]]中扮演一名[[同性戀|同性戀]]記者，更與[[凱文·克萊|凱文·克萊]]有接吻戲
* [[諾曼第大風暴|諾曼第大風-{}-暴]]（[[:en:Ike:_Countdown_To_D-Day|''Ike: Countdown To D-Day'']]，2004年），飾演[[艾森豪|艾森豪]]

== 電視 ==
[[File:Tom_Selleck_1988.jpg|thumb]]
* 近年轉型靜態[[TV_show|TV show]]主持人
* 《[[夏威夷神探|夏威夷之虎]]》（[[:en:Magnum_P.I.|Magnum P.I.]]），[[CBS|CBS]]電視劇1980年—1988年主演，飾演一名[[夏威夷|夏威夷]][[私家偵探|私家偵探]]，風流倜儻、著花花涼衫、開[[敞篷跑車|敞篷跑車]]，為其小螢幕代表作。

== 獎項 ==
* 1986年美國[[時人|時人]]雜誌(People)膺選[全世界最美麗50人]之一
* 1983年美國[[金球獎|金球獎]]最佳男主角獎
