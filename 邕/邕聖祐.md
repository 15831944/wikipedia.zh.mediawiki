'''邕聖祐'''{{NoteTag|1=2017-06-20 所屬社Fantagio娛樂確認 <big>'''邕聖祐'''</big> 為正名、由蘋果日報公開。　 [http://www.appledaily.com.tw/realtimenews/article/new/20170620/1144034/ 相關新聞報導]{{zh}}}}（{{Kor|k='''옹성우'''|rm=Ong Seong Wu}}，{{bd|1995年|08月25日}}），[[韓國|韓國]][[男歌手|男歌手]]、[[演員|演員]]、[[主持|主持]]及[[模特兒|模特兒]]，曾被譯名為'''雍成宇'''。2017年參與電視選秀節目《[[PRODUCE_101_第二季|PRODUCE 101 第二季]]》出線，並以限定組合[[Wanna_One|Wanna One]]成員身分活動過一年半，現為[[Fantagio|Fantagio]]娛樂旗下藝人。

== 歷程 ==
=== 成長背景 ===
邕聖祐家鄉為[[仁川廣域市|仁川廣域市]]，曾就讀{{link-ko|仁川九月中學|구월중학교}}、[[翰林演藝藝術高等學校|翰林演藝藝術高等學校]]實用舞蹈科，並以戲劇藝術學士畢業於東首爾大學。家庭成員有父母及一位胞姐{{NoteTag|1=2017-06-16 邕聖祐在《PRODUCE 101 第二季》第11集發表出道感言，提及「…還有我的家人，<big>'''爸爸、媽媽、姐姐'''</big>…」}}。

=== 出道前：練習生涯 ===
2007年、小學六年級時，邕與朋友組成樂隊，個人擔任鼓手一職{{NoteTag|1=2008-04-18 邕聖祐在網絡留言板上發表的日誌，內容節錄「 <big>'''去年9月組了樂隊'''</big>，我們在學校食堂表演的同時，還帶著樂隊參加了 <big>'''青少年歌謠大賽'''</big>…想到自己已經是 <big>'''六年級'''</big> 了，就這樣畢業真的好可惜…」　[http://i.imgur.com/DrpQyWF.jpg 相關日誌存檔]{{ko}}}}。樂隊在一次青少年歌謠大賽上得獎，當時邕被一所公司的星探發現，參與且通過邀約面試，成為旗下練習生{{NoteTag|1=2008-05-23 邕聖祐在網絡留言板上發表的日誌，內容節錄「…由於我們的樂隊 <big>'''在比賽上得獎'''</big>，歌手Younha所屬的公司聯繋我們，希望我們去 <big>'''參加他們的面試'''</big>…我真的很開心，現在要 <big>'''更努力地練好鼓'''</big>，說不定將來就可以成為跳唱歌手…我一定能做到！」　[http://i.imgur.com/YrL4uf2.jpg 相關日誌存檔]{{ko}}}}{{NoteTag|1=2008-09-22 邕聖祐在網絡留言板上發表的日誌，內容節錄「我 <big>'''通過了面試'''</big>！ 我現在是一名練習生了！」　[http://i.imgur.com/0jqxx7k.jpg 相關日誌存檔]{{ko}}}}。經過幾年訓練後，公司以年紀小為由，拒絕讓邕出道，更於及後倒閉。受合約所限，約滿前邕不能加入其他企劃社<ref>{{cite news|title=名單公開 EP182 - 現在就走花路吧♥ '邕聖祐' 的打工故事|url=https://www.youtube.com/watch?v=r_3sptA89CI|publisher=tvN|date=2017-09-04|language=ko}}</ref>。

公司倒閉後，邕繼續堅持在舞蹈學校練習，另外也加入地下舞團參與多場比賽，特長為[[Popping|Popping]]、[[Locking|Locking]]等<ref>{{Cite web |url = https://www.youtube.com/watch?v=0U8dwVUzO7I |title = 邕聖祐 20th Century BBoys 「Breaking、Popping」 舞蹈比賽片段 |publisher = YouTube|date=2017-03-12}}</ref><ref>{{Cite web |url = https://www.youtube.com/watch?v=YAmei_OLnR8 |title = 邕聖祐 Rhythm Presto Move Vol.4 「Popping」 舞蹈比賽片段 |publisher = YouTube|date=2017-05-14}}</ref><ref>{{Cite web |url = https://www.youtube.com/watch?v=Dm3MG7_FyE0 |title = 邕聖祐 Winner Dance School 「Locking」 舞蹈比賽片段 |publisher = YouTube|date=2017-04-18}}</ref>。在朋友協助下，邕接受歌唱訓練，並成為教會聖歌隊的主音<ref>{{cite news|title='PD101 2' 邕聖祐，教會聖歌隊的ACE 牧師兒子真實出現了|url=http://entertain.naver.com/read?oid=005&aid=0001001323|publisher=국민일보|date=2017-06-19|language=ko}}</ref>。邕高中修讀實用舞蹈系，與BTOB的[[陸星材|陸星材]]和VICTON的許燦是同班同學<ref>{{cite news|title=他們居然是同學！Wanna One邕聖祐和BTOB陸星材高中合照曝光|url=https://www.koreastardaily.com/tc/news/102433|publisher=KSD 韓星網|date=2018-02-05|language=zh}}</ref>，大學時又以話劇、音樂劇成員的身份活躍<ref>{{Cite web |url = https://www.youtube.com/watch?v=7QyCUG_OgdQ |title = 邕聖祐 東首爾大學「뜨거운 여름 2016」演出片段 |publisher = YouTube|date=2017-03-16}}</ref><ref>{{Cite web |url = https://www.youtube.com/watch?v=OQiBUiLB0Vo |title = 邕聖祐 東首爾大學「Whitesome Reading」拍攝花絮 |publisher = YouTube|date=2017-03-15}}</ref>。同時邕聖祐也有接一些模特工作，是不少商戶的模特兒<ref>{{cite news|title='PD101 2' 邕聖祐，打工王是我啊我！|url=http://www.sporbiz.co.kr/news/articleView.html?idxno=108752|publisher=한국스포츠경제|date=2017-05-24|language=ko}}</ref><ref>{{cite news|title='眾所周知… 打工王' 邕聖祐的髮型模特兒身份|url=http://www.dispatch.co.kr/748293|publisher=Dispatch|date=2017-05-16|language=ko}}</ref>。

2016年，邕通過了[[Fantagio|Fantagio]]的面試，再度成為一名練習生<ref>{{cite news|title=邕聖祐 2016年4月 在Fantagio面試片段|url=https://www.youtube.com/watch?v=gYUYzC_PHgk|publisher=YouTube|date=2017-06-28}}</ref>。
