《'''毀滅戰士：滅絕'''》（{{lang-en|'''''Doom: Annihilation'''''}}）是一部2019年[[美國|美國]][[科幻片|科幻]][[動作片|動作]][[恐怖片|恐怖片]]，由[[東尼·吉格里歐|東尼·吉格里歐]]自編自導，改編自[[id_Software|id Software]]開發的電子遊戲《[[毀滅戰士系列|毀滅戰士]]》。該遊戲曾於2005年推出過[[毀滅戰士_(電影)|改編電影]]，但反響不佳。其主演包括{{le|艾米·曼森|Amy Manson}}、{{le|路易斯·曼迪魯|Louis Mandylor}}、{{le|多明尼克·馬法姆|Dominic Mafham}}、{{le|妮娜·褒曼|Nina Bergman}}和{{le|路克·艾倫-蓋爾|Luke Allen-Gale}}。

== 劇情 ==
故事敘述一群太空海軍陸戰隊員在回應來自火星月球基地的遇險呼救時，卻發現基地早已被惡魔般的怪物佔領，且怪物們還企圖將地球變成地獄。

== 製作 ==
2018年4月21日，演員兼歌手的{{le|妮娜·褒曼|Nina Bergman}}在[[Twitter|Twitter]]上證實，[[環球影業|環球影業]]繼[[毀滅戰士_(電影)|2005年改編電影]]的失敗後，將再次製作電子遊戲《[[毀滅戰士系列|毀滅戰士]]》的改編電影；該片將於[[保加利亞|保加利亞]]拍攝，而褒曼也將參與演出<ref name="Variety(2018-04-21)">{{cite news|url=http://variety.com/2018/gaming/news/new-doom-movie-1202776280/|title=Universal Is Making a New ‘Doom’ Movie|work=Variety|date=2018-04-21|accessdate=2018-04-21}}</ref><ref name="Geek-Base(2018-04-21)">{{cite news|url=https://geek-base.toy-people.com/?p=3608|title=撿起你的BFG！演員證實《毀滅戰士》電影新作即將展開製作！|work=Geek-Base - 玩具人|date=2018-04-21|accessdate=2018-06-08}}</ref>。2018年6月5日，重啟版《毀滅戰士》由[[東尼·吉格里歐|東尼·吉格里歐]]執導，主演包括{{le|艾米·曼森|Amy Manson}}、{{le|路易斯·曼迪魯|Louis Mandylor}}、{{le|多明尼克·馬法姆|Dominic Mafham}}、褒曼、{{le|路克·艾倫-蓋爾|Luke Allen-Gale}}等，且已於4月起在保加利亞進行前製<ref name="Jovem Nerd News(2018-06-05)">{{cite news|url=https://jovemnerd.com.br/nerdbunker/elenco-do-filme-live-action-de-doom-divulga-imagens-e-videos-dos-bastidores/|title=Elenco do filme live-action de Doom divulga imagens e vídeos dos bastidores|work=Jovem Nerd News|date=2018-06-05|accessdate=2018-06-08}}</ref>；此外，導演吉格里歐也在Twitter上透露，重啟電影將忠於遊戲中的血腥風格<ref name="Screen Rant(2018-06-05)">{{cite news|url=https://screenrant.com/doom-movie-reboot-photos-blood/|title=Doom Movie Reboot Set Photos: There Will Definitely Be Blood|work=Screen Rant|date=2018-06-05|accessdate=2018-06-08}}</ref>。該片的製作預算並不高，且將以[[DVD首映|DVD首映]]的方式發行<ref name="Screen Rant(2018-06-05)" />。主要拍攝已於2018年6月底開始進行。其他拍攝於2019年初開始，以改善某些場景和視覺效果<ref name="delay">{{cite web|url=https://www.dreadcentral.com/news/288270/universal-pushes-back-new-doom-adaptation-but-for-positive-reasons/|title=Universal Pushes Back New Doom Adaptation But Fear Not! This is a Good Thing!|first=Jonathan|last=Barkan|work=Dread Central|date=2019-01-25|accessdate=2019-03-09|archivedate=2019-03-09|archiveurl=https://web.archive.org/web/20190309155648/https://www.dreadcentral.com/news/288270/universal-pushes-back-new-doom-adaptation-but-for-positive-reasons/}}</ref>。

2019年3月，在官方宣布了該片的正式片名《毀滅戰士：滅絕》，並在同月釋出了首支預告片。但該預告片獲得了粉絲及id Software的負面反饋，id更在事後證實他們並未參與該片的製作<ref>{{cite web|url=https://www.ign.com/articles/2019/03/12/doom-studio-makes-very-clear-it-has-nothing-to-do-with-new-doom-movie|title=Doom Studio Makes Very Clear It Has Nothing to Do With New Doom Movie|first=Adam|last=Bankhurst|work=IGN|date=2019-03-12|accessdate=2019-03-12|archivedate=2019-03-12|archiveurl=https://web.archive.org/web/20190313010232/https://www.ign.com/articles/2019/03/12/doom-studio-makes-very-clear-it-has-nothing-to-do-with-new-doom-movie}}</ref>。
