'''洲崎綾'''（{{lang-ja|洲崎綾}}、{{bd|1986年|12月25日|catIdx=Suzaki Aya}}<ref name="tower">{{Cite web |url=http://tower.jp/artist/2094285/洲崎綾|title=アーティスト詳細 洲崎綾|publisher=[[淘兒唱片|淘兒唱片]]|accessdate=2015-04-08|language=ja}}</ref>），[[日本|日本]][[女性|女性]][[聲優|聲優]]。隷屬於[[I'm_Enterprise|I'm Enterprise]]。出生於[[石川縣|石川縣]][[金澤市|金澤市]]<ref name="mro">{{ja}}[http://www.ota-suke.jp/news/132665 洲崎綾が地元・MRO北陸放送の石川のマンガ・アニメ特集に出演] おた☆スケ 2014年12月16日、同23日閲覧。</ref>。[[血型|血型]][[ABO血型系统|O型]]<ref name="profile">{{Cite web|url=http://yaplog.jp/clown-happy/profile/ |title=洲崎 綾のプロフィール |accessdate=2015-04-08 |archiveurl=https://web.archive.org/web/20130702125847/http://yaplog.jp/clown-happy/profile/ |archivedate=2013-07-02 |language=ja |deadurl=yes }}</ref>。

== 經歷 ==
* [[東京學藝大學|東京學藝大學]]<ref>{{ja}}[http://recochoku.jp/artist/2000221410/album/ 洲崎綾 アルバム一覧] [[RecoChoku|RecoChoku]] 2014年12月24日</ref>在校期間進入[[日本播音演技研究所|日本播音演技研究所]]<ref>{{Cite web |url=http://nichinare.com/about/native.html |title=主な出身者 |publisher=日本ナレーション演技研究所 |accessdate=2015-04-08|language=ja}}</ref>。畢業之後在一般公司擔任[[辦公室女職員|OL]]<ref name="tower"/>，直到2010年在《[[爆漫王。|爆漫王。]]》擔任角色配音正式出道。擔任作品中女主角亞豆美保所組成的聲優團體成員天音美紗代的配音。與飾演亞豆美保的[[早見沙織|早見沙織]]及飾演松田桃子的{{tsl|ja|春宮茉由}}為同事及好友，也在作品中出現的聖視覺女學院高等部合唱部擔任歌唱<ref name="davinci">{{ja}}[http://ddnavi.com/news/187105/ 洲崎綾さんに50の質問「声優の輪」【第16回】] - ダ・ヴィンチニュース</ref>。

* 在電視動畫《[[玉子市場|玉子市場]]》的試鏡會上，下意識地改變聲音，自然地將自己最大限度的可愛聲音呈現出來，因而第一次獲選為主角。這部作品同時也是她第一次作為正式成員演出的作品<ref>{{Cite journal|author=アニメイトニュース編集部|authorlink=アニメイト|date=2013-02-12|title=人気声優9人 撮り下ろし写真&独占インタビュー!!|journal=声優きゃらびぃ|issue=Vol.9|page=9|accessdate=2013-02-12}}</ref>。

* 2015年，獲得第9回聲優獎的新人女優賞<ref>{{Cite web|publisher=[[声優アワード|声優アワード]]|url=http://seiyuawards.jp/201503/06170044.php|title=第九回声優アワード受賞者一覧|accessdate=2015-04-08|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20150325034617/http://seiyuawards.jp/201503/06170044.php|archivedate=2015-03-25}}</ref>。

== 人物 ==
* 在家排行老大。有個小兩歲的弟弟<ref>『玉子市場 もちもちラジオ』第4回。</ref>。
* 中學與高中的時候都參加[[軟式網球|軟式網球]]社。中學時曾擔任社團副社長<ref>『玉子市場 もちもちラジオ』第3回。</ref>。
* 中學的時候看了《[[名偵探柯南|名偵探柯南]]》，是她成為聲優的契機。
* 崇拜的聲優是[[林原惠|林原惠]]，曾經在2003年9月打電話進[[日本放送協會|NHK]]節目「小孩子夢想提問箱（{{lang|ja|子供夢質問箱}}）」節目中向林原請教對於聲優的職涯規劃<ref>[http://www.nicovideo.jp/watch/sm21680223 土曜わくわくラジオ・子供夢質問箱2003年9月　(林原めぐみ　×　洲崎綾)]</ref>，另外在2014年4月12日的《[[林原惠的東京不羈夜|林原惠的東京不羈夜]]》上擔任嘉賓<ref>[http://www.nicovideo.jp/watch/sm23316769 林原めぐみのTokyo Boogie Night 洲崎綾 出演部分（2014.04.12）]</ref>。
* 喜歡的運動是[[軟式網球|軟式網球]]與[[水肺潛水|水肺潛水]]；擅長唱歌，特技是[[小號|小號]]及[[花道|花道]]，並擁有英檢3級、珠算初段，及[[幼稚園|幼稚園]]、[[小學|小學]]、[[特殊學校|特殊學校]]的教育職員證書。
