'''蜘蛛俠'''（{{lang-en|Spider-Man}}）是[[漫威漫画|漫威漫画]]的[[超級英雄|超級英雄]]，本名為'''彼得·班傑明·帕克'''（Peter Benjamin Parker）。他是由作家／編輯[[史丹·李|史丹·李]]及作家／畫家[[史蒂夫·迪特科|史蒂夫·迪特科]]所創造。他的第一次出現是在《Amazing Fantasy》第15期（1962年8月）。2011年5月，在[[IGN|IGN]]的前100名的漫畫書英雄中排行第三<ref>{{cite web | 
url=http://www.ign.com/top/comic-book-heroes/3 | title=Spider-Man  – Top 100 Comic Book Heroes | publisher=IGN Entertainment | accessdate=2011-05-27}}</ref>。成为漫威旗下最受欢迎的超级英雄<ref>{{cite news|title=Why Spider-Man is popular.|url=http://abcnews.go.com/Entertainment/story?id=101230&page=1|accessdate=2010-11-18}}</ref>（第一和第二为DC旗下的超人和蝙蝠侠）。

== 人物 ==
: 彼得·班傑明·帕克（Peter Benjamin Parker）在一次課外活動中意外的被一隻受過[[放射性|放射性]]感染的[[蜘蛛|蜘蛛]]咬傷後，獲得具有蜘蛛一般的特殊能力。改變彼得·帕克的一生，決定成為蜘蛛人的关键，就在於一場搶劫事故中皇后區的[[班·帕克|班叔]]（Uncle Ben）遭人殺害，彼得認為這是他的錯誤，因此選擇成為蜘蛛人來拯救世界上更多的人。李和迪特科將這個角色構想成一個被他的[[梅嬸嬸|梅嬸嬸]]和班叔撫養長大的孤兒；並且，因為身為一個青少年，他除了面對那些超級英雄該肩負的責任，同時也必須應付一些青春期時普遍會遇到的麻煩。蜘蛛侠的創造者們賦予了他超強的力量與敏捷性、能黏附在大部分表面上的能力、可以對危險快速做出反應的蜘蛛感應（Spider-Sense）以及讓他藉著使用自己發明並命名為「蛛網發射器」的裝置射出蜘蛛絲，使他得以對抗反派們。

=== 歷史 ===
* 蜘蛛人初次登場是在1962年8月，在雜誌《[[奇妙幻想|奇妙幻想]]》（Amazing Fantasy）第15期，後來十分受歡迎才有了以他為主題的漫畫。在蜘蛛人誕生的1960年代早期，出現在超級英雄漫畫裡的青少年角色通常只能扮演主角們的副手。然而蜘蛛俠系列藉著讓彼得．帕克，一個「有些自戀卻又自我排斥、缺乏信心且孤單」的高中生擔綱主角打破了這個局面，並成功引起了年輕讀者們的共鳴。與過往的青少年超級英雄不同－像是[[巴奇_(漫威漫畫)|巴基]]（Bucky）和[[羅賓_(漫畫)|羅賓]]，蜘蛛俠並非[[美國隊長|美國隊長]]或[[蝙蝠俠|蝙蝠俠]]這些成年英雄的門生，自然也無法在擁有一個成熟導師庇護與指引的環境中成長。因此，他必須自己去體認「能力越大，責任越大」這句在蜘蛛人第一個故事的最後一格出現的著名台詞。而在隨後的劇情裡，這句箴言被追溯為出自於他已故的監護人班叔之口。

* 現在漫威旗下有數部以蜘蛛人做為主角的系列漫畫，其中最早問世同時也是連載最久的一部是《The Amazing Spider-Man》。隨著時間的演進，彼得.帕克在這個故事裡也從原本害羞的高中生成長為一個苦惱但外向的大學生，接著結婚並且成為自己高中母校的老師。當時間來到2000年代後期，他的身份轉變為單身的自由攝影師，這也是他在進入社會後最具代表性的身份。蜘蛛人常常被暱稱為「Spidey」、「Web-Slinger」、「Wall-Crawler」或是「Web-Head」。

* 在被八爪博士交換記憶與人格後，八爪博士以彼得的身分考上了博士學位，甚至還創立了帕克工業。在八爪博士刪除自己的記憶人格後，彼得繼承了八爪博士的遺澤，成為了帕克工業的CEO，甚至被媒體稱為是「窮一點的[[鋼鐵人|東尼·史塔克]]」。彼得隨後買下了[[驚奇4超人|驚奇4超人]]的巴氏大樓作為帕克工業在紐約的總部（實為要保留巴氏大樓的所有權），並聘請了好友哈利·奧斯本、過去的老對手震動人等等，讓他們有正經工作可以做。

== 能力 ==
* 超級力量，力量級數4,200磅—25噸
: 在對付-{普通人}-時力量會有所收斂，除非對手也是超人類。

* 超級耐力，對傷害具有減輕的效果

* 身手非常敏捷和灵活，並有著超人的平衡感

* 超級的反射神經，反應速度是-{普通人}-的四十倍
: 蜘蛛人的戰鬥風格就是活用各種與地平面不同角度的各方平面，以自身的平衡、敏捷和反應並活用蛛絲所構成。

* 蜘蛛感應（Spider-sense）
: 擁有非常靈敏的超級感官和有預知危險的能力，結合反射神經可以讓他規避絕大部分的攻擊與偷襲。除此之外在看不到的環境下也能依此行動自如。在獲得能力前從未接受過任何武術訓練的蜘蛛人就是仗此能力躲避和反擊對手。
: 不過此能力只能感應到有危險以及該危險的程度與大致的方向，無法確切表明危險是怎麼發生的，而且特定角色能不引起該感應的注意，也可以透過特殊裝置干擾、屏蔽，甚至是讓蜘蛛人感到痛苦。

* 控制手脚产生静电从而拥有吸附的能力，能夠像蜘蛛一樣在牆上行走，並藉此吸附在絕大多數平面位置上
: 運用在攻擊上甚至可以配合本身的超級力量來撕裂對手，但因為太兇殘，所以只有在危急時彼得才會使用。除了用來在各種平面上移動外，也能用此能力讓面罩不會輕易脫落而暴露真實身分。

* 再生因子
: 雖然比不上金鋼狼和死侍或浩克的程度，但骨折或大面積組織損傷的話只要幾天就能復原。

* 蜘蛛絲
: 實為蜘蛛人所調配的特殊化學配方製成，平時是液體狀態，接觸到空氣就會變成類似蛛網的物質。能壓縮儲存於膠囊當中，透過手掌的「蛛網發射器」的控制裝置射出蜘蛛絲。蛛絲大約在兩個小時後就會自動分解消失，所以不會造成環境上的困擾。韌性極強，甚至能在短時間內束縛住浩克。至少可以承受一個一般成年人的重量，甚至是卡車的重量也能承受一定的時間。
: 用途廣泛，蜘蛛人平常是用來當成交通工具在大樓間擺盪遊走，戰鬥時可依情況利用蛛網發射器調整成球狀體作遠距離打擊、封住對手眼睛、槍口等來制止攻擊，也可用來當成護盾、陷阱等。
: 有時因應劇情需要，蜘蛛人也會研發有絕緣、防酸、耐火效果的蛛絲配方。
: 電影方面，2002年至2007年版本為了強調蜘蛛的特性，所以蜘蛛絲改由蜘蛛人的身體製造，只要擺出特殊的手勢（食指、拇指、小指伸出）即可發射；而在2012年的版本中，導演強調要忠於原著，所以還原為以控制器發射。

* 蛛網發射器（Web Shooters）
: 彼得自己所開發、可說是蜘蛛人最重要的裝備。彼得發現自己被蜘蛛咬了而有了蜘蛛一樣的超能力後，為了讓自己更加名副其實，而用自己的天才頭腦調配出了蜘蛛絲液體和製造這款用來發射液體使之變成蛛網的發射器。
: 發射開關位於掌心，需要將中指與無名指彎曲、快速雙擊開關才能發動，這樣的設定是為了防止蜘蛛人在握拳時觸碰到開關導致不必要的誤射。
: 蜘蛛人會在左右手各裝備一個發射器，每個發射器都有獨立的蛛絲液體儲藏盒。
: 早期只能用來發射蛛絲，隨著時代與劇情的演進，性能也隨之變化、增加。

* [[梅·帕克|梅·帕克]]（May Parker）
: 梅嬸，在彼得帕克的父母過世後，原先由班叔與梅嬸共同照顧彼得，而班叔過世後，梅嬸就是彼得的唯一親人。

* [[關·史黛西|關·史黛西]]（Gwen Stacy）
: 彼得的初戀女友。她生性善良但有點驕縱，聰明，美麗。-{zh-hans:格温; zh-hant:關;}-的父親是警察局長喬治‧史黛西（George Stacy）。彼得與-{zh-hans:格温; zh-hant:關;}-一開始是好朋友，後來漸漸地戀情增長，之後兩人正式交往，一直到-{zh-hans:格温; zh-hant:關;}-過世。-{zh-hans:格温; zh-hant:關;}-被綠惡魔從橋上丟下去造成死亡。在其他版本的漫畫{{tsl|en|House of M}}中，-{zh-hans:格温; zh-hant:關;}-依然活著並跟彼得結婚。在其他版本的漫畫{{tsl|en|Spider-Gwen}}中，-{zh-hans:格温; zh-hant:關;}-在意外中被一隻蜘蛛咬傷後，獲得具有蜘蛛一般的特殊能力，成為女蜘蛛人，但彼得成為了蜥蜴人並於意外中死亡。

* [[瑪莉珍·華生|瑪莉珍·華生]]（Mary Jane Watson）
: 另一位與彼得帕克友好的女孩，曾經與-{zh-hans:格温; zh-hant:關;}-一同競爭蜘蛛人。最後在-{zh-hans:格温; zh-hant:關;}-過世後與蜘蛛人正式交往。

* [[哈利·奧斯朋|哈利·奧斯朋]]（Harry Osborn）
: 彼得的好友，變過猛毒，是綠惡魔二世。

* [[閃電·湯普森|閃電·湯普森]]（Flash Thompson）
: 本名尤金·「閃電」·湯普森（英语：Eugene "Flash" Thompson）。閃電是蜘蛛人彼得·帕克的高中同學，後來在中東戰爭失去雙腿的他接受了美國政府的一項實驗，使他成功和猛毒共生體結合，成為一名超級英雄「猛毒特工」（Agent Venom）為美國政府工作。

* [[J·喬納·詹姆森|J·喬納·詹姆森]]（J. Jonah Jameson）
: 某間報章社雜誌的總編輯，愛發牢騷的老闆，認為蜘蛛人是個罪犯。他聘請彼得拍攝蜘蛛人的照片。

== 代表敵人 ==
*[[邪惡六人組|邪惡六人組]]（Sinister Six）：由八爪博士集合禿鷹、電光人、獵人克雷文、神秘客、沙人來對抗蜘蛛人的超級罪犯團體。
**[[八爪博士|八爪博士]]（Doctor Octopus）：核能科學家，因實驗意外使四隻工作用的機械臂溶接在他背上，並影響其腦部使他成為罪犯。
**[[禿鷹_(漫威漫畫)|禿鷹]]（Vulture）：因生意失利而穿上自己開發的飛行裝備變成反派。
**[[電光人|電光人]]（Electro）：本為電力工人，因意外得到能超能力。可從指尖放出高壓電、沿電流移動、及用電力強化體能，另外可控制任何電子儀器。
**[[獵人克拉文|獵人克拉文]]（Kraven the Hunter）：一個熱衷狩獵遊戲的瘋狂獵人，把擊敗蜘蛛人當成是可以成為世界上最偉大的獵人的證明。
**[[沙人_(漫威漫畫)|沙人]]（Sandman）︰本來只是小罪犯，因輻射意外，能隨意變成沙。
**[[神秘法師|神秘客]]（Mysterio）：認為擊敗蜘蛛人就能名揚天下的前電影特效工作者，精通幻術和科技。
*[[綠惡魔|綠惡魔]]（Green Goblin）：蜘蛛人的最大敵人，性格狂妄好戰，曾邀請過和自己有相近力量的蜘蛛人共同行動，但被拒絕，綠惡魔便開始將前者作為宿敵。
*[[變色龍_(漫威漫畫)|變色龍]]（Chameleon）：擁有改變外形成任何人的能力。於《Spider-man meets president》一作中假冒成新任[[美國|美國]]總統[[奧巴馬|奧巴馬]]。
*[[蜥蜴人|蜥蜴人]]（Lizard）：和蜘蛛人同樣能飛簷走壁，力量很強，再生能力很高。
*[[蠍子人|蠍子人]]（Scorpion）：本是為了找出蜘蛛人真面目的私家偵探，後加入實驗而擁有超人體能並穿著特製衣服，可替換長尾除了近身攻擊外，也能射出電擊器、酸液等。
*[[犀牛人|犀牛人]]（Rhino）：本來是黑社會打手，藉科學實驗得到超人體能及一件如犀牛皮堅韌的外衣。力大無窮，但頭腦簡單。
*[[震動人|震動人]]（Shocker）：專精破解保險箱的盜匪，以自己開發的震動手套和裝備進行搶劫。
*[[金霸王_(漫威漫畫)|金霸王]]（Kingpin）：同為[[夜魔俠|夜魔俠]]的對手，黑道老大。
*[[水人|水人]] （Hydro-Man）：有把身體變成水的能力。本身為水手，一次意外被不明物質影響擁有化成液體的能力，近似沙人。動畫中，曾利用他的細胞製作蜘蛛俠複製人。
*[[惡鬼_(漫威漫畫)|惡鬼]]（Hobgoblin）：發現了綠惡魔一個祕密基地的時裝設計師Roderick Kingsley因為自己的野心而使用了其中的裝備和藥劑開始作惡。
*[[猛毒|猛毒]]（Venom）：外星共生生命體，曾共生在蜘蛛人身上而複製了其能力，附在他人身上後有著比蜘蛛人相同且更強的能力。
*[[反猛毒|反猛毒]]（Anti-Venom）：由第一任猛毒因癌症治療後，共生體（猛毒）與其白血球結合轉變而成。
*[[血蜘蛛|血蜘蛛]]（Carnage）：由猛毒分裂出的部分共生體附於一名連環殺手身上產生的新共生體。
*[[毒素_(漫威漫畫)|毒素]]（Toxin）：由血蜘蛛分裂出的寄生體，擁有猛毒、血蜘蛛、蜘蛛人的能力，但不會和宿主的意識同化。
*豹人（Puma）：能變成半人半豹，變身後有著不輸給蜘蛛人的敏捷身手的反射神經。
*蜘蛛殺手（Spider-Slayer）：認為蜘蛛人是個威脅的Spencer Smythe教授製造出的機器人，具有追蹤蜘蛛人的功能。
*魔倫（Morlun）：以吞噬動物圖騰果腹的生命，擊敗過蜘蛛人多次，為了不讓蜘蛛人逃走甚至還會傷及無辜，也曾和黑豹作對過。

== 其他作品 ==
蜘蛛俠是最廣為人知而且在商業化上取得極大成功的[[超級英雄|超級英雄]]之一。身為漫威的紅牌角色與公司的象徵性人物，蜘蛛俠的漫畫亦一直暢銷，其中一些單元故事，更是大受歡迎。蜘蛛俠頻繁地出現在各種其他的媒體上包含一些動畫及真人的電視節目、同時刊載在多家報紙上的連環漫畫，以及一系列成功的翻拍電影。在已經發行的前三部電影裡是由[[陶比·麥奎爾|陶比·麥奎爾]]（Tobey Maguire）詮釋這個就像「大家的好鄰居」的超級英雄，而重啟蜘蛛俠電影則找來[[安德魯·加菲|安德魯·加菲]]（Andrew Garfield）接下這個任務，雖然這是一部重拍電影，但票房方面並沒有什麽影響，證明蜘蛛俠的魅力依然驚人。另一方面，在新的蜘蛛俠改編音樂劇《{{le|蜘蛛俠：消滅黑暗|Spider-Man: Turn Off the Dark}}》中蜘蛛俠是由[[李維·卡尼|李維·卡尼]]所扮演。 加入[[漫威電影宇宙|漫威電影宇宙]]後，蜘蛛人的角色由[[汤姆·赫兰德|汤姆·赫兰德]]飾演。於《[[美國隊長3：英雄內戰|美國隊長3：英雄內戰]]》中初登場，個人獨立電影《[[蜘蛛人：返校日|蜘蛛人：返校日]]》于2017年7月7日在美國首映。往後也加入《[[復仇者聯盟3：無限之戰|復仇者聯盟3：無限之戰]]》與《[[復仇者聯盟：終局之戰|復仇者聯盟：終局之戰]]》，《返校日》也將有兩部續集作品，包括《[[蜘蛛人：離家日|蜘蛛人：離家日]]》。
