'''淋文化'''是中文互联网上对台灣歌手[[蔡依林|蔡依林]][[惡搞|惡搞]]的一种[[迷因|迷因]]，其惡搞方式包括使用视频、[[P图|P图]]，以及拆分蔡依林的舞蹈動作、使用“淋語”等，起源於2009年，常見於[[百度贴吧|百度贴吧]]和[[新浪微博|新浪微博]]。淋文化在中國大陸同志[[亞文化|亞文化]]中佔有重要的地位，蔡依林本人亦回应过淋文化。

== 歷史 ==
淋文化常見於[[百度贴吧|百度贴吧]]和[[新浪微博|新浪微博]]，<ref name="南方"/>起源於2009年。因當時蔡依林在專輯《[[花蝴蝶|花蝴蝶]]》中的造型被指類似日本歌手[[濱崎步|濱崎步]]，兩家粉丝掀起骂战。2010年，蔡依林粉丝在贴吧和滨崎步粉丝对骂。期间滨崎步粉丝编造蔡依林得[[淋病|淋病]]，利用蔡依林在演唱会中表演劈叉一字马等动作制作了许多恶意侮辱女性下体的动图，并創造出了“淋逼”等词汇，並戏称蔡依林為“宇宙级别天后淋淋”。而蔡依林的一些歌词、语气词也在对骂的过程中成为代表性的词汇。<ref name="人民">{{cite web|author1=王文超|title=文本制造者：从粉圈用语看粉丝心理及粉圈文化|url=http://media.people.com.cn/n1/2018/0124/c416772-29784673-2.html|website=人民網|accessdate=2018-05-23}}</ref><ref>{{cite web|author1=王击凡|title=华语歌手“花名册”大揭秘|url=http://ngzb.gxrb.com.cn/html/2014-11/26/content_1057355.htm|website=南国早报|date=2014-11-26|accessdate=2017-10-21|deadurl=no|archiveurl=https://web.archive.org/web/20171021091716/http://ngzb.gxrb.com.cn/html/2014-11/26/content_1057355.htm|archivedate=2017-10-21}}</ref>黑粉们甚至开发出了恶搞蔡依林的游戏《降淋》，将蔡依林头像移花接木，变成怪兽、千手观音与性感美女等，引起了蔡依林粉丝的不满。<ref>{{cite web|title=网友曝萧亚轩粉丝制作游戏 恶搞蔡依林|url=http://ent.cntv.cn/20110811/100917.shtml|website=中国网络电视台|publisher=网易娱乐|date=2011-08-11|accessdate=2017-10-21|deadurl=no|archiveurl=https://web.archive.org/web/20120523151015/http://ent.cntv.cn/20110811/100917.shtml|archivedate=2012-05-23}}</ref>後來，網友們建立了貼吧“三木木吧”用以專門黑酸蔡依林，該貼吧是2015年百度貼吧增速第2大的黑粉吧，當年人氣增長達16萬。<ref>{{cite web|title=百度贴吧揭晓2015互联网八大神秘组织|url=http://tech.huanqiu.com/internet/2015-12/8210721.html|website=環球網|publisher=中國網|date=2015-12-18|accessdate=2017-11-05|deadurl=no|archiveurl=https://web.archive.org/web/20171105080231/http://tech.huanqiu.com/internet/2015-12/8210721.html|archivedate=2017-11-05}}</ref>然而，該貼吧現處於被封禁狀態。{{efn|貼吧搜尋“三木木”可得到“抱歉，根据相关法律法规和政策，本吧暂不开放。”之提示}}此外，随着韩流文化的席卷和中國国内流量小生的崛起，“淋语”在粉圈的使用情况逐渐减少。<ref name="人民"/>

== 文化特徵 ==
[[File:Lin_Culture.jpg|thumb]]為淋文化惡搞經典素材之一，取自該演唱會的本截圖常用作表情包。]]
淋文化的惡搞方式包括使用视频、P图，以及拆分蔡依林的舞蹈動作、使用“淋語”等。蔡依林的黑粉自稱“本質騎士”，而蔡依林本人則被黑粉們稱為“淋淋”。<ref name="南方">{{cite web|author1=杨蓓蓓|author2=蒋梦瑶|author3=吴笑丛|title=揭娱乐圈黑势力：有水准的黑比脑残粉更要技术|url=http://ent.qq.com/a/20130527/009340.htm|website=騰訊娛樂|publisher=南方人物週刊|accessdate=2017-10-21|deadurl=no|archiveurl=https://web.archive.org/web/20171021091716/http://ent.qq.com/a/20130527/009340.htm|archivedate=2017-10-21}}</ref>但久而久之，“被黑”也成了一股风潮，不少蔡依林粉丝也会把“淋淋”用作对偶像的爱称。<ref>{{cite web|title=何超仪叫“何超”，真的不是打漏了个字？杨千嬅叫“千fa”，真的就是图方便……|url=http://ent.ifeng.com/a/20141121/40372031_0.shtml|website=凤凰娱乐|publisher=南方都市报|accessdate=2018-03-30}}</ref>

淋语，又稱淋文，為網友根據蔡依林的说话方式发明出的一種交流方式，<ref name="南方"/>被戲稱為“淋王星的官方語言”，屬於“黑粉文化”的一種。<ref name=安徽新聞論壇/>所谓“淋语”就是在对话中反复使用几个语气助词，再加上一些变调的[[台灣國語|台式國語]]以及英文，辅以少量的专属名词。<ref name=才智/><ref name="坎坷">{{cite web|title=蔡依林15年进化论：情路坎坷 工作拼命|url=http://ent.sina.com.cn/y/2014-10-27/14434230855.shtml|website=新浪音樂|publisher=信息時報|accessdate=2017-10-20|deadurl=no|archiveurl=https://web.archive.org/web/20171021091714/http://ent.sina.com.cn/y/2014-10-27/14434230855.shtml|archivedate=2017-10-21}}</ref><ref name="南方"/>“噜”、“惹”、“泥”、“轰”、“奴”為淋語中常用的語氣助詞，<ref name="南方"/>其中“惹”带有[[傲嬌|傲嬌]]语气。淋語中還經常使用的“啾咪”表示装可爱，而“嘻嘻”表示威胁时的冷笑，“靴靴”则表示“谢谢”，“吸进淋X”（亦称“Flop到淋X”）则用于表示偶像十分过气。2013年，微博上即有大量网友使用淋语。<ref name="现代">{{cite web|title=惹、噜、啾咪|url=http://dz.xdkb.net/html/2013-12/17/content_312679.htm|website=现代快报|date=2013-12-17|accessdate=2017-10-20|deadurl=no|archiveurl=https://web.archive.org/web/20140224011403/http://dz.xdkb.net/html/2013-12/17/content_312679.htm|archivedate=2014-02-24}}</ref><ref>{{cite web|title=一周语文|url=http://www.bjnews.com.cn/ent/2018/01/22/473420.html|website=新京报|accessdate=2018-05-12}}</ref><ref>{{cite web|title=涨姿势周刊 粉丝“黑话”词典：流水的爱豆 熟悉的套路|url=http://ent.sina.com.cn/original/zishiweekly/09/?t=1477099203427|website=新浪娱乐|accessdate=2018-05-12}}</ref>淋语已经成为中文互联网一种全新的语态，有人甚至已经在网上开始对淋语的语言学研究。<ref name="呸">{{cite web|title=蔡依林：呸！恶搞适可而止|url=http://ent.163.com/15/0625/06/ASUFIG6A00034PF4.html|website=网易|accessdate=2017-10-20|deadurl=no|archiveurl=https://web.archive.org/web/20150717221444/http://ent.163.com/15/0625/06/ASUFIG6A00034PF4.html|archivedate=2015-07-17}}</ref>

淋文化的惡搞不局限於蔡依林本人。一位來自中國廣西[[南寧|南寧]][[大沙田街道|大沙田]]的蔡依林粉絲周英庭，因在百度貼吧中贬低其他歌手，力捧蔡依林，而遭到了網友[[人肉搜索|人肉搜索]]，並得綽號“周陰婷”“周陰中”“南寧仙子”等（“陰中”來源於其最初使用的網名中有“英中”二字，導致網友認為其真名叫周英中）。關於他的惡搞內容亦被視作淋文化的一部分，網友們甚至為他錄製了歌曲《找不到从良的理由》。<ref>{{cite web|url=http://www.shenchuang.com/gdrd/20170725/482098.shtml|title=南宁仙子周阴婷什么梗 周阴婷事件始末介绍|publisher=深窗资讯|accessdate=2017-11-03|deadurl=no|archiveurl=https://web.archive.org/web/20171105080231/http://www.shenchuang.com/gdrd/20170725/482098.shtml|archivedate=2017-11-05}}</ref>{{rs}}此外，由於網路紅人[[罗玉凤|罗玉凤]]被网友認為長相像[[蔡依林|蔡依林]]，<ref>{{cite web |title=凤姐新造型出炉 网友盛赞像蔡依林 |url=http://ent.sina.com.cn/s/m/2010-04-05/04412919027.shtml |website=新浪网 |accessdate=2018-08-06}}</ref>淋文化亦對羅玉鳳進行惡搞。淋淋、周陰婷、羅玉鳳三人并称为“淋系艺人”。{{來源請求}}

== 影响力 ==
淋文化在中国大陆[[同志_(LGBT)|同志]][[亞文化|亞文化]]中佔有重要的地位。“淋语”和蔡依林相关表情包几乎成为了同志的一种身份认证。<ref>{{cite web|title=有一种偶像叫做同志偶像|url=http://ent.qq.com/a/20151026/060192.htm|website=腾讯网|accessdate=2018-05-12|language=zh-CN}}</ref>由於蔡依林多次力挺同志，被視為[[同志偶像|同志偶像]]，因此許多中文圈的同志都會使用淋語。由於網絡空間的匿名性與寬鬆性，使得同志能夠藉由在網上使用“淋語”而構建出一種個體和集體的性身份認同感。另外，由於淋語中的“嚕”“惹”“熏”等字眼或多或少都能與性活動、性習慣等聯繫起來，因此同志-{}-喜好使用淋語，一方面是為了將自己的文化與主流文化區別開，以彰顯其獨特性以及文化上的“優越性”；另一方面則是一種無形的示威。並且，在主流社會中工作和生活的同志群體大多迫切需要找到一個可以互相慰藉的同類，但由於受到主流社會的道德束縛而使得他們不能公開追尋，因此他們不得不使用“淋語”這種獨特的語言來完成這一情感追求。<ref name=安徽新聞論壇>{{cite journal |author=沈祯 |title=边缘人的声音:同性恋亚文化的交往——以“淋语”文化现象为例 |url= |journal=安徽省第七届新闻传播学科研究生论坛论文集 |volume= |issue=|date=2015-12-12|pages= |doi=}}</ref>

蔡依林本人亦回应过淋文化。2014年，《南都娛樂週刊》在採訪蔡依林時曾要求蔡依林用淋语跟該刊读者問好，蔡依林使用了淋語“啾咪”，並稱使用淋語時“觉得自己好少女”。<ref>{{cite web|author1=云会|title=蹦跳太累？蔡依林：麦当娜都行没啥不可以|url=http://ent.sina.com.cn/z/y/2014-10-22/16084228509.shtml|website=新浪|publisher=南都娛樂週刊|accessdate=2017-11-05}}</ref>同年接受搜狐娛樂採訪時，蔡依林表示對於惡搞的圖片，自己也会用那些圖片来表达自己的情绪。而關於專門黑自己的貼吧，蔡依林則稱“我觉得应该有，但是我没有去查，我觉得每一个受关注的应该都有支持跟不支持的，都会有。但是我不会拿这些事情来让自己花这些精神去跟他们做辩论。”<ref name="搜狐娱乐">{{cite web|title=蔡依林：说我不好的人，我翻Ta个白眼|url=http://yule.sohu.com/s2014/ttrwcyl/|website=搜狐娱乐|accessdate=2018-03-18}}</ref>2015年，蔡依林在接受[[网易云音乐|网易云音乐]]的訪談時，表示網路上的惡搞應該“适可而止，不要去伤害到别人的自尊心的那种。因为我觉得有一些透过，像我很喜欢看一些动画类的电影，我觉得它有一些很贱的东西，对话或是很贱的人物，在你现实生活中不会出现的，你就会很放松很有趣这样子，当然也不是鼓励大家要很负面的对别人做攻击，我觉得就是用一种比较好玩的方式。”<ref name="呸"/>2018年底蔡依林新单曲《[[怪美的|怪美的]]》的MV中，亲自扮演网络疯传的[[地才|地才]]GIF表情包。
