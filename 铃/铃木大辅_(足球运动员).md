'''铃木大辅'''（{{bd|1990年|1月29日|||Suzuki, Daisuke}}），[[日本|日本]][[足球|足球]]运动员，現效力[[日职|日职]]球會[[浦和紅鑽|浦和紅鑽]]，司职[[后卫_(足球)|中后卫]]，前[[日本国家足球队|日本国家足球队]]成员。

==俱乐部生涯==
2007年，铃木大辅以特別指定選手的身份加盟[[日本职业足球联赛|日本职业足球联赛]]球队[[新潟天鹅|新潟天鹅]]，2008年正式加盟球队。2009年，他在天皇杯第二轮，新潟天鹅和[[奈良俱乐部|FC奈良]]的比赛中上演职业生涯的首秀。<ref>{{cite web |url=http://www.jsgoal.jp/news/_prog/view.php?press_code=00090845 |title=Match report: Nara Club vs Albirex Niigata |publisher=J's Goal |accessdate=10 February 2012 |deadurl=yes |archiveurl=https://archive.is/20120802182644/http://www.jsgoal.jp/news/_prog/view.php?press_code=00090845 |archivedate=2 八月 2012 }}</ref> 2010年5月1日，他在2010赛季第9轮2比1击败[[神户胜利船|神户胜利船]]的比赛中替补登场，首次亮相职业联赛。<ref>{{cite web |url=http://www.j-league.or.jp/data/view.php?c=niigata&t=p_record&g=j1_1&y=2010 |title=2010 prayer statistics |publisher=J. League |accessdate=10 February 2012 |deadurl=yes |archiveurl=https://archive.is/20120802212521/http://www.j-league.or.jp/data/view.php?c=niigata&t=p_record&g=j1_1&y=2010 |archivedate=2 八月 2012 }}</ref> 2011赛季，他终于成为球队主力，出场24次，并在5月28日和[[浦和红鑽|浦和红鑽]]的比赛中射进职业生涯的首个进球。<ref>{{cite web |url=http://www.sponichi.co.jp/soccer/news/2011/05/28/kiji/K20110528000910420.html |title=Suzuki scored his first goal |publisher=日本体育 |accessdate=10 February 2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20160128032935/http://www.sponichi.co.jp/soccer/news/2011/05/28/kiji/K20110528000910420.html |archivedate=2016年1月28日 }}</ref>

2013年1月，铃木大辅转会加盟[[柏雷素尔|柏雷素尔]]。

2016年2月16日，铃木大辅加盟[[西乙|西乙]]球會[[塔拉戈納體操|塔拉戈納體操]]。
