'''珠雞'''（'''''Numididae'''''）('''''guineafowl''''')，又稱'''嘉鸡'''、'''山雞'''、'''[[几内亚|畿內亞]]鳥'''，是[[雞形目|雞形目]]下的其中一[[科_(生物)|科]]。

== 特徵 ==
羽毛很特別，非比一般尋常，聲音很吵雜。

== 常有地區 ==
珠雞多出現在在[[撒哈拉以南非洲|撒哈拉以南非洲]]、英美、東非和印度。

== 食用和營養價值 ==
珠雞肉比雞肉乾韌，脂肪含量比它少一半, 每克珠雞肉含較少卡路里，較有野生肉味，比雞和[[火雞|火雞]]含更多蛋白質。<ref>USDA handbook #8 and circular #549, leclercq 1985</ref>而珠雞蛋比雞蛋營養更豐富。
