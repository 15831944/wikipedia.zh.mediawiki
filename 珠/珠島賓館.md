'''珠島賓館'''是位於[[广东省|廣東]][[广州市|廣州]]的一間四星級[[酒店|酒店]]，於1950年代初建立，位置在[[越秀区|越秀區]][[沿江路_(广州)|沿江東路]]463號。其為具備嶺南園林特色的别墅式賓館，是廣東省比較出名的政務接待賓館之一，有「廣東釣魚台」之稱。

== 建築 ==
* 高度：12m
* 佔地面積：120,000km²
* 建築面積：1,000,000km²
* 樓層：3層樓
* 內有16座商務樓與多間綜合性大樓、別墅式小樓。

== 歷史 ==
酒店於1950年代初建起，初期名為「廣東省委小島招待所」<ref>[http://www.hybsl.cn/beijingcankao/beijingfenxi/2013-06-03/37605.html 卸甲一书生：再谈毛泽东1949年以后的住所] 胡耀邦史料信息网</ref>。1970年，總理[[周恩来|周恩來]]將其改名做「珠島賓館」還讓董必武幫主席寫過賓館名。之後一直由廣東省委同省政府打理，為中華人民共和國黨同國家領導人來廣東的主要接待基地。1980年5月，[[习仲勋|習仲勛]]、[[叶剑英|葉劍英]]、許世友、[[杨尚昆|楊尚昆]]來過這裡入住與考察<ref>[http://news.ifeng.com/history/zhongguoxiandaishi/detail_2012_11/18/19272505_0.shtml 霍英东建白天鹅宾馆被命楼顶架高炮 叶帅亲自解围] 鳳凰網 （2012-11-18）</ref>；1983年，時中共中央總書記[[胡耀邦|胡耀邦]]來這裡入住過<ref>[http://www.gd-info.gov.cn/shtml/guangdong/sqyj/shgc/2015/06/12/128360.shtml 胡耀邦与广东改革开放] 广东省情网 （2015-06-12）</ref>；葉劍英也分別於1992年、1993年來過這裡入住<ref>[http://lrb.dayoo.com/html/2013-05/15/content_2247689.htm 王震关心广东改革开放]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }} 老人报 （2013-05-15 A39版）</ref>。2008年8月底，賓館南邊建起二沙二橋，直到2009年6月2號蓋好<ref>[http://gzdaily.dayoo.com/html/2009-06/02/content_587662.htm 二沙二桥珠岛小桥下月开通]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }} 廣州日報 （2009-06-02 A21版）</ref>。2008年10月搞過大規模裝修。

== 酒店設施 ==
珠島賓館內有301間客房、9間餐廳、18間多功能宴會廳，還有不少中大型會議廳。大樓設計風格具有嶺南園林風格特色，建築外面庭院地區有不少綠化，如同公園一般（緊挨東山湖公園）。
