{{vgname|仰望青空的龍少女|ja=空を仰ぎて雲たかく}}是一款由[[迪波之巢製作所|迪波之巢製作所]]於2010年7月30日發售的發售的[[日本成人遊戲|18禁]][[戀愛冒險遊戲|戀愛冒險遊戲]]。遊戲的目的是培養一個與人類有著相同外觀的龍族女孩成長。

2013年3月7日由{{tsl|ja|サイバーフロント|CyberFront}}於[[PlayStation_Portable|PlayStation Portable]]發行《仰望青空的龍少女 Portable》。2016年10月27日於[[Steam|Steam]]上發行了以PSP版為基底修改推出的電腦下載版本，並於2017年5月9日更新了簡體中文版開放下載<ref>{{Cite web |url = http://store.steampowered.com/app/456760/A_dragon_girl_looks_up_at_the_endless_sky/ |title = A dragon girl looks up at the endless sky|author = Steam |date = 2016-10-27 |publisher = [[Steam|Steam]] |accessdate = 2017-10-10}}</ref>。

== 遊戲簡介 ==
這是一款描述與龍族女孩交往並成長的電腦遊戲。本作遊戲內容共分成兩個部分，分別為使用劇情方式來描述培育龍族女孩敏特成長過程的「故事部分」、以及敘述培育過程中去狩獵其他龍族和攻擊怪物以增加自身能力的「RPG部分」。

根據玩家在「故事部分」中對待敏特的方式以及態度不同，將會產生出4種不同的性格。每種性格都會有屬於自己不同的故事分歧以及最終結局的差異。此外對應至每條路線的不同，故事中都有必須達成的「任務」，也有會改變目前狀態的事件<ref name="4Gamer">{{cite web|url=http://www.4gamer.net/games/191/G019116/20121101064/|title=PSP用美少女RPG「空を仰ぎて雲たかく Portable」が2013年3月7日発売。山本和枝氏描き下ろしのイベントCGほか複数の追加要素あり|date=2012-11-01|accessdate=2014-06-21|publisher=[[4Gamer.net|4Gamer.net]]}} </ref>。

「RPG部分」是指玩家透過操縱敏特打敗敵人，其玩法包含了橫向捲軸遊戲的元素<ref name="Game-style">{{cite web|url=http://game-style.jp/soft/201006/21/01aog_de.php|title=手塩にかけたドラゴン少女と世界を旅する、ドラゴン少女育成RPG！『空を仰ぎて雲たかく』7月30日発売！|accessdate=2014-06-21|date=2010年6月21日}}</ref>。

== 故事介紹 ==
亞杜尼斯從幾年前開始，為了成為馴龍師而在前馴龍師「達南」（{{lang|ja|ダナン}}）身邊進行著修行。某一天，他在森林中進行訓練的時候與龍族少女敏特相遇了。這位少女直接稱亞杜尼斯為「爸爸」，亞杜尼斯想著這肯定是有什麼誤會所以向少女問了很多問題，可是少女除了自己的名字以外什麼都不記得了。回到了師傅家裡詢問過後，從師傅口中得知在森林中與她的相遇，就是亞杜尼斯成為馴龍師的命運邂逅。於是與失去記憶的龍少女訂下契約的亞杜尼斯，為了尋找少女的記憶，開始踏上了在世界各地周遊的冒險旅程。
