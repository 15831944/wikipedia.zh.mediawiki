'''仰望星空派'''（Stargazy pie）是[[英國|英國]]的一道傳統菜餚，主要盛行在[[康沃爾郡|康沃爾郡]]，材料包括[[沙丁魚|沙丁魚]]、[[雞蛋|雞蛋]]、[[馬鈴薯|馬鈴薯]]。由於派中烤過的魚的魚頭突出，看起來在仰望天空，因此得名。但這樣做的本來目的是使烹飪過程中的放出的油回流到派中。2007年，Mark Hix憑藉這道菜的變體贏得[[BBC|BBC]]「英國菜單」節目冠軍<ref name="hix">{{cite news|last=Stacey|first=Caroline|title=Mark Hix serves up a seasonal feast|url=http://www.timesonline.co.uk/tol/life_and_style/food_and_drink/article4919532.ece|accessdate=7 January 2011|newspaper=[[The_Times|The Times]]|date=11 October 2008}}</ref>。

在一些介绍文章中，由于配图中的鱼头都睁眼垂直“望着”天空，因此中国大陆网友嘲讽为“死不瞑目”。<ref>{{Cite web |url=https://www.honglingjin.co.uk/21500.html |title=让人不能直视的“仰望星空”派（附食谱） — 英国红领巾 | 你必备的英国指南 – RED SCARF |accessdate=2018-12-03 |work=www.honglingjin.co.uk |language=zh-CN}}</ref><ref>{{Cite web |url=https://baijiahao.baidu.com/s?id=1587920916108606091 |title=仰望星空派不算啥，这些英国黑暗料理挑战你的三观！ |accessdate=2018-12-03 |date=2018-12-03 |work=baijiahao.baidu.com}}</ref><ref>{{Cite web |url=https://travel.ifeng.com/a/20160318/41564543_0.shtml |title=英国黑暗料理代表作 一只派也要“仰望星空” |accessdate=2018-12-03 |work=travel.ifeng.com}}</ref>
