'''蛇岛'''（{{lang-uk|'''Острів Зміїний''',''Ostriv Zmiinyi''}}; {{lang-ro|'''Insula şerpilor'''}}）是一座位于[[黑海|黑海]][[多瑙河三角洲|多瑙河三角洲]]附近的一座[[岛屿|岛屿]]，现属[[乌克兰|乌克兰]][[敖德萨州|敖德萨州]]管辖。蛇岛一名是意译自[[罗马尼亚语|罗马尼亚语]]，[[乌克兰|乌克兰]]称之为'''兹梅伊内岛'''。

== 地理 ==
[[File:Insula_Serpilor_map.png|200px]]<br />
蛇岛位于[[多瑙河|多瑙河]]河口以东，距[[海岸线|海岸线]]35公里处的[[黑海|黑海]]之中。周围富油气资源。
== 历史 ==
该岛最初由[[古希腊|古希腊]]人命名为 Λευκός ，意为“白色的岛屿”，因为该岛上有丰富的白色[[大理石|大理石]]而得名。岛上有许多古希腊文化遗迹。在[[奥斯曼帝国|奥斯曼帝国]]时期该岛被命名为 Fidonisi （[[希腊语|希腊语]]“蛇岛”之意）。后该岛曾一度被[[俄罗斯帝国|俄罗斯帝国]]占领。[[罗马尼亚|罗马尼亚]]独立后该岛属罗马尼亚所有。1948年，[[苏联|苏联]]强迫罗马尼亚将蛇岛移交给当时的苏联[[加盟共和国|加盟共和国]][[乌克兰苏维埃社会主义共和国|乌克兰]]。苏联解体后，乌克兰继续控制该岛，罗马尼亚一直表示抗议。1995年，罗马尼亚向乌克兰提出包括蛇岛在内的一些领土归属问题，要求重划两国边界，遭乌克兰方面反对。两国关系一度陷入紧张状态。直到1997年，乌克兰和罗马尼亚两国外长在乌克兰首都[[基辅|基辅]]签署了《乌克兰和罗马尼亚友好、睦邻、合作条约》，解除了两国关系的紧张状态，并承诺双方不向对方提出领土要求，乌克兰则保证不在蛇岛上部署进攻性武器，同意通过谈判解决海上边界问题，在问题解决之前任何一方都不开发争议海区的资源。后来，罗马尼亚为了进入[[北约|北约]]（要求加入北约的候选国不能与其邻国有任何领土争端）于2003年6月17日与乌克兰签署条约，正式承认蛇岛归乌克兰所有。

== 海洋划界 ==
[[File:ICJ_Romania_v_Ukraine.jpg|300px]]
2004年9月，罗马尼亚在与乌克兰上方就黑海[[大陆架|大陆架]]及[[专属经济区|专属经济区]]划分的问题进行数轮谈判未果后，向[[国际法院|国际法院]]提起诉讼，要求国际法院对两国海上边界进行划分。2009年2月3日，国际法院在[[荷兰|荷兰]][[海牙|海牙]]一致裁决，划分了罗马尼亚与乌克兰两国在黑海海域的边界，存在争议的区域约有８０％被判归罗马尼亚所有。
