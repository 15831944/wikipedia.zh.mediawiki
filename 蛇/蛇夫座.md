'''蛇夫座'''从[[地球|地球]]看位于[[武仙座|武仙座]]以南，[[天蝎座|天蝎座]]和[[人马座|人马座]]以北，[[银河|银河]]的西侧。蛇夫座是星座中惟一与其他星座-[[巨蛇座|巨蛇座]]直接連在一起，同时蛇夫座也是唯一同時橫跨[[天球赤道|天球赤道]]、[[銀道座標系|银道]]和[[黄道|黄道]]的星座。蛇夫座既大又宽，形状长方，天球赤道正好斜穿过这个长方形。尽管蛇夫座跨越的银河很短，但银河系中心方向就在离蛇夫座不远的[[人马座|人马座]]内。银河在这里有一块突出的部分，形成了银河最宽的一个区域。

== 特點 ==
[[File:OphiuchusCC.jpg|thumb]]]
在天文學上，每年11月30日[[太阳|太阳]]會从蛇夫座穿越，於1928年[[國際天文學聯合會|國際天文學聯合會]]（International Astronomical Union，縮寫為IAU）的國際天文學會議中，被認定為黃道上的第十三個星座。雖然[[地軸|地軸]]在數千年間有所變動，但地球繞行太陽的黃道面幾乎沒有變化，因此現在的黃道星座與占星學發明時的只有時間上需平移一段時間，除此之外並無分別。之所以傳統[[占星學|占星學]]認為黃道通過十二個星座，只不過是因為當時的星座分界與現在不同。意即以現今的星座分界來畫分，占星學發明時，太陽已有穿越蛇夫座，只是時間比現在早約50天。

== 神话与历史 ==
在[[希腊神话|希腊神话]]中，[[阿斯克勒庇俄斯|阿斯克勒庇俄斯]]是[[人马|人马]][[喀戎|喀戎]]的学生。喀戎将自己掌握的[[医学|医学]]知识毫无保留的传授给他。这激怒了[[冥王|冥王]][[黑帝斯|黑帝斯]]，于是在阿斯克勒庇俄斯使[[俄里翁|俄里翁]]复苏时（参见[[猎户座|猎户座]]神话），让[[宙斯|宙斯]]用雷劈死了阿斯克勒庇俄斯和俄里翁。由於[[阿斯克勒庇俄斯|阿斯克勒庇俄斯]]是著名的醫者，手中擁有[[阿斯克勒庇俄斯之杖|阿斯克勒庇俄斯之杖]]，杖上捲有一條靈蛇，以其蛻皮重生象徵醫學的治療概念，因此[[阿斯克勒庇俄斯|阿斯克勒庇俄斯]]死後宙斯将其变为蛇夫座，靈蛇則變成[[巨蛇座|巨蛇座]]。

蛇夫座还是[[特洛伊|特洛伊]][[祭司|祭司]][[拉奥孔|拉奥孔]]的化身，他曾警告特洛伊人别让木马进城。后来杀死[[拉奥孔|拉奥孔]]的其中一只海蛇则是与蛇夫座相连的[[巨蛇座|巨蛇座]]。<ref>{{cite book|author1=[英]萨拉·巴特利特|title=符号中的历史: 浓缩人类文明的100个象征符号|date=2016年8月|publisher=北京联合出版公司|isbn=9787550278998|pages=第20页|edition=第1版}}</ref>

古代蛇夫座属于[[天蝎座|天蝎座]]，由於天蝎尾部亮星过于偏南，使得蛇夫的脚无处可放，因此將这一部分星空以及与天蝎体形无关的部分连同一段[[黄道带|黄道带]]从[[天蝎座|天蝎座]]分出畫为蛇夫座。

== 有趣的星 ==
* [[巴纳德星|巴纳德星]]：从[[地球|地球]]看这颗星位于[[蛇夫座β|蛇夫座β]]东方，它是1916年被[[美国|美国]][[科学家|科学家]][[巴纳德|巴纳德]]首先发现的。它是仅次于[[南门二|南门二]]的[[太阳|太阳]]第二近邻，它距离太阳5.87[[光年|光年]]。同时它又是从地球看全天运动最快的星，自行一年为10.3秒，大约每189年其就会在[[天球|天球]]上运动一个[[月球|月球]]月面的距离。运动方向朝向太阳系，因此，再过数千年，其就会变为距离地球最近的[[恆星|恆星]]，可能有[[行星|行星]]围绕其旋转。
* [[蛇夫座70|蛇夫座70]]：这是一个[[双星|双星]]，互相环绕一周的时间是88年，两子星[[星等|星等]]分别为+4和+6，相距3.7[[弧秒|弧秒]]，距离正在扩大。

== [[深空天体|深空天体]] ==
[[File:Kepler_Drawing_of_SN_1604.png|thumb]]的繪圖描繪的“克卜勒超新星(Supernova 1604)的位置，腳下的蛇夫座”。]]
* [[M9|M9]]：一个结构松散的[[球状星团|球状星团]]，很小，最亮的星也在+14等左右，需要使用10英寸（25厘米）的[[望远镜|望远镜]]才能看见一部分，星等为+7.9。
* [[M10|M10]]和[[M12|M12]]：这两个球状星团相距3.4度，接近蛇夫座中心，结构都比较松散，如果在没有[[光污染|光污染]]的情况下，使用4英寸（10厘米）的望远镜即可容易的看见它们。它们的星等为+6.6，M10距离地球14000光年，M12距离地球18000光年。
* [[M14|M14]]：这是一个球状星团，使用小型望远镜只能看见模糊的轮廓，使用10英寸（25厘米）的望远镜可以分辨出亮星。它的星等为+7.5。
* [[M62|M62]]：这是一个明亮，外形相当圆的球状星团，位于恆星紧密的天区，星等为+6.6。
* [[M107|M107]]：这是[[梅西耶天体|梅西耶天体]]中最暗的球状星团，星等+8.1。
* [[NGC6633|NGC6633]]和[[IC4665|IC4665]]：这是可以使用双筒望远镜观测的大而松散的[[星团|星团]]，使用[[双筒望远镜|双筒望远镜]]可以看见其中的亮星。
