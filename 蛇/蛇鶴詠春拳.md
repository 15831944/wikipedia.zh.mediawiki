'''蛇鶴詠春拳'''，又稱'''蛇鶴詠春門'''，源於[[詠春拳|詠春拳]]<ref>[http://news.hkheadline.com/dailynews/headline_news_detail_columnist.asp?id=148064&section_name=wtt&kw=137 黑帶論壇 —— 永春白鶴拳（二）之來龍去脈]《頭條日報》2011-05-13</ref>，別名「'''盲拳'''」或「'''歸家拳'''」<ref>[http://www.intellects.org/forum/article.php?doc_id=671 蛇鶴詠春拳簡介]世界功夫武術段位制總會</ref>，[[南拳|南拳]]的一種。

== 歷史 ==
蛇鶴詠春始創於[[福建少林寺|福建少林寺]]（南少林）永春殿，當時在殿上有一對對聯：「永行忠義堪歌詠 言出九鼎志更春」，「詠春」這名字的來源就是取其最後一字，由[[至善禪師|至善禪師]]及[[五枚大師|五梅師太]]及一眾弟子所創，至善禪師精於蛇形拳，而五梅師太則精於鶴形拳，並觀察其蛇鶴之生活動態演變而成，所以這套拳法多以蛇手及鶴手為招式， 故名「蛇鶴詠春」，而這對聯亦成為此支派的門訓。

後來清廷火燒南少林，至善禪師跟五梅師太及眾僧侶分別四處逃命，至使這拳法正好由至善禪師逃難至南方一帶，並在[[嶺南|嶺南]]發揚光大。而蛇鶴詠春正是至善禪師於[[紅船|紅船]]避難時傳下來的，其中一位受惠者便是武生[[大花面|大花面]][[申錦|申錦]]，及後大多戲班中人練習這拳法。<ref>[http://www.scwc.com.hk/SCWC/Articles/000002.pdf 蛇鶴詠春門公開露面]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}《新武俠》武術雜誌</ref>

== 師祖與師承 ==
過往在傳承方面，以父傳子形式為主，及以個別小組形式低調地向外教授，但從不讓小組間師兄弟們認識。蛇鶴詠春為至善禪師及五梅師太所創，於嶺南一帶由至善禪師發揚為一傳，於紅船武生大花面申錦為二傳，晚年傳羅悌雲為三傳。

師祖羅梯雲，是晚清年代一名貢生，自小喜愛武技。在[[孫中山|孫中山]]先生[[辛亥革命|辛亥革命]]起義時，爲[[李褔林|李褔林]]作[[書記|書記]]。於[[民國|民國]]初年，棄官從醫，並退回祖居[[佛山|佛山]][[西樵|西樵]]，懸壺濟世。及後傳技於其子羅定周，羅定周亦在[[國民政府|國民政府]]時期為官，未解放前，移居[[香港|香港]]，投身警界，其後轉投文化事業。此技羅定周再傳其子羅昭榮至今，一脈相承，仍然保持其低調作風。

至善（一傳）、申錦（二傳）、羅梯雲（三傳）、羅定周（四傳）、羅昭榮（五傳）

== 練習方法 ==
蛇鶴詠春還有多個別名「盲拳」或「歸家拳」之稱號，因在練習黏手時可朦著眼睛練習，故曰盲拳，還有在練習時不能在公眾地方練習，當時多留在家中練習，故亦稱歸家拳。手法有擒拿手及穴位攻打，快速狠毒，所以不便輕傳及顯露於人前。

蛇鶴詠春有三套拳法－[[小念頭|小念頭]]、[[尋橋|尋橋]]和[[標指|標指]]，都有[[口訣|口訣]]附助；碌手、黏手、盤手等對練亦有口訣；而去對練的攻防，有8-9種空樁手法，亦是各附有口訣。除了套拳外，亦有心法及功法，心法有口訣，功法須埋樁鍛煉，手法開合，外形與泰國拳類似，但着重後發先至。技法方面 ，有鶴爪、有擒拿手法 ，可以扣打穴位，手法狠辣。兵器方面，有[[六點半棍|六點半棍]]和[[二字箝羊奪命刀|二字箝羊奪命刀]]，以斬剌爲主。
