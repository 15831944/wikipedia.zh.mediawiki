==薛楷莉事件==
2002年9月19日，時任TVBS主播的薛楷莉被旅居日本的畫家[[姚旭燈|姚旭燈]]指控，在一小時內刷了日本企業家[[德原榮玉|德原榮玉]][[信用卡|信用卡]]高達[[新台幣|新台幣]]184萬元，還拿了現款10萬[[日圓|日圓]]。德原認為，此舉已形同[[敲詐|敲詐]]。各媒體大幅報導此事件<ref>{{Cite web |url = http://old.ltn.com.tw/2002/new/sep/20/today-show2.htm |title = 兩分鐘刷爆兩百萬？ 薛楷莉疑涉「削凱子」 |publisher = 自由時報  |language = zh-tw |accessdate = Feb 18, 2015 }}</ref>。2002年10月24日，TVBS正式批准薛楷莉辭呈，原本雙方要到年底才會到期的合約關係提前結束<ref>{{Cite web |url = http://www.feja.org.tw/modules/wordpress/?p=52 |title = 薛楷莉事件(2002) |publisher = 卓越新聞獎基金會 |language = zh-tw |accessdate = Feb 18, 2015 |archiveurl = https://web.archive.org/web/20150217202711/http://www.feja.org.tw/modules/wordpress/?p=52 |archivedate = 2015-02-17 |deadurl = yes }}</ref>。

2003年6月10日，《[[台灣蘋果日報|台灣蘋果日報]]》報導，[[環球電視_(台灣)|環球電視]]新聞部大幅[[挖角|挖角]]，外傳薛楷莉與[[中天新聞台|中天新聞台]]主播[[吳中純|吳中純]]要跳槽環球電視，時任環球電視當家主播的前藝人[[王筱嬋|王筱嬋]]於本月9日揚言「不論誰來，我都要當環球電視一姐」，但薛楷莉的男朋友兼發言人[[顏冠得|顏冠得]]否認薛楷莉將跳槽環球電視，吳中純則證實已向[[中天電視|中天電視]]提出辭呈、但否認將跳槽環球電視<ref>{{cite news|author=林佳樺|publisher=台灣蘋果日報|url=http://ent.appledaily.com.tw/enews/article/entertainment/20030610/98213|title=王筱嬋放話當定一姐|date=2003-06-10|accessdate=2016-07-09}}</ref>。

2004年，薛楷莉與顏冠得結婚。2013年5月，薛楷莉與顏冠得協議離婚，但薛楷莉指控顏冠得對女兒妮妮[[家暴|家暴]]且有[[婚外情|婚外情]]。2015年1月21日，[[士林地方法院|士林地方法院]]裁定薛楷莉獲得女兒監護權。
