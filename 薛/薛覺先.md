== 生平 ==
薛覺先出身於粵劇世家，十兄弟中排行第五，二姊[[薛覺芳|薛覺芳]]，三姊[[薛覺非|薛覺非]]及九弟[[薛覺明|薛覺明]]三人，也先後作了粵劇演員。

13歲就讀[[香港|香港]][[聖保羅書院|聖保羅書院]]，學名薛鑾梅。<ref name="takung170514">{{cite web |url=http://www.takungpao.com.hk/culture/text/2017/0514/81321.html |title=﻿薛覺先—「書院仔」到粵劇泰斗|date=2017-05-14 |publisher=大公報 |author=張茅 }}</ref> 15歲中途輟學，17歲在皮革鋪做學徒，18歲拜姐夫[[新少華|新少華]]為師。1923年入人壽年班。因天資聰穎，20歲已做正印，能與[[千里駒|千里駒]]同台演出。1925年，在[[上海|上海]]成立[[非非影片公司|非非影片公司]]拍攝[[默片|默片]]《浪蝶》，結識了片中的女主角[[唐雪卿|唐雪卿]]（[[唐滌生|唐滌生]]的堂姐），並結為夫婦。1933年自編、自演的粵語片《白金龍》公映，大獲好評，轟動粵、港、滬三地。<ref name="takung170514" /> 在[[上海|上海]]的期間，[[昆劇|昆劇]]及[[京劇|京劇]]對他起了很大的啟發。他看見粵劇中亂穿服裝，他覺得很不合理，下定決心吸收京劇場面和體制，引進[[粵劇|粵劇]]，改革了音樂、鑼鼓等的技巧。

1927年回到香港加入「天外天」、「新景象」、「大江東」等戲班。當時男角在當時是沒有化妝的，最多是劃黑少許眼眉毛。他卻參照[[梅蘭芳|梅蘭芳]]的，所以在化妝方面更見色彩艷麗。1929年，自組[[覺先聲劇團|覺先聲劇團]]。在他旗下走紅的名伶有很多，包括[[新馬師曾|新馬師曾]]、[[陳錦棠|陳錦棠]]、[[麥炳榮|麥炳榮]]、[[呂玉郎|呂玉郎]]、[[羅品超|羅品超]]、[[黃超武|黃超武]]、[[陸飛鴻|陸飛鴻]]及[[林家聲|林家聲]]等。

<ref>[http://www.job853.com/macaunews/news_list_show_macao.aspx/controls/MacauNews/news_list_show_macao.aspx?type=4&id=423929&page=%20&y=0&m=0&d=0 薛覺先收的門人]</ref>[[陳皮梅_(粵劇演員)|陳皮梅]]演出的幾乎全是薛的名劇，稱譽為「女薛覺先」，粵劇造詣得到薛的肯定，<ref>薛覺先的舞台生涯-薛覺明遺作</ref>並大排筵席、正式收為徒。

可是大受歡迎的文唱武打名劇《胡不歸》，薛與[[上海妹|上海妹]]及[[半日安|半日安]]合演時被觀眾稱為「原裝胡不歸」的，能夠繼續延長壽命直到現在，主力卻是文武兼擅的兩人、後輩新馬師曾和入室弟子林家聲。2009年回到香港後大開中門，上門請益後輩多不勝數，林在病逝前卻特地傳授此劇給自己最後一個學生[[藍天佑|藍天佑]]及其團隊。

活躍於[[上海|上海]]之前，跟從[[朱次伯|朱次伯]]學習及走少年郎路線，常代替失場的朱演出《紅樓夢》的[[賈寶玉|賈寶玉]]，一個薛在晚年很少演的角色。

1954年，薛覺先離開香港定居[[廣州|廣州]]，出任廣州粵劇工作團藝術委員會主任及中國戲劇家協會廣州分會副主席，後來任全國政協委員。<ref name="takung170514" /> 薛覺先技藝全面，能演丑角、文武生、旦角、小生等等。52歲，在大陸演完「花染狀元紅」翌日便因[[腦溢血|腦溢血]]逝世，葬於[[三元里|三元里]]走馬崗[[八和會館|八和]]墓園。<ref>{{cite web|url=http://gcontent.oeeee.com/a/3f/a3fb4fbf9a6f9cf0/Blog/a7c/8b754f.html|title=梨园义冢藏闹市 三代守墓为琼花|accessdate=2012-09-05|date=2012-04-06|publisher=南方都市報|deadurl=yes|archiveurl=https://web.archive.org/web/20150603001633/http://gcontent.oeeee.com/a/3f/a3fb4fbf9a6f9cf0/Blog/a7c/8b754f.html|archivedate=2015-06-03}}</ref><ref>{{cite web|url=http://gcontent.oeeee.com/a/3f/a3fb4fbf9a6f9cf0/Blog/a84/c693fa.html|title=数百梨园后人 组团拜祭先辈|accessdate=2012-09-05|date=2012-04-05|publisher=南方都市報|deadurl=yes|archiveurl=https://web.archive.org/web/20150603004150/http://gcontent.oeeee.com/a/3f/a3fb4fbf9a6f9cf0/Blog/a84/c693fa.html|archivedate=2015-06-03}}</ref>

==粵劇改革==
薛覺先銳意改革粵劇。他取消“師約制”，打破紅船陋規，演出時制止小販叫賣，嚴禁人員隨意出入，以保良好秩序，廢棄“提綱戲”，杜絕“爆肚”，要求排演忠於劇本，唱做念打不得更動。腔調方面，他借鑒京劇，吸收江浙劇小曲和時代流行曲輕快流暢的長處，與小提琴家[[尹自重|尹自重]]合作設計出一種字正腔圓，節奏感強，善於表達情意的新腔。因為新腔突破曲調原來的板眼及句格，所以人稱「薛腔」。樂器方面，他加進了小提琴、吐林必、瑟士風、吉他等西洋樂器及京劇鑼鼓，組成西樂部，與中樂部互相輝映。表演方面，薛覺先找來了[[袁小田|袁小田]]教授北派功夫，自此粵劇便融合了京劇的武功，開創了北派的開打場面。他更是第一個把粵劇搬上銀幕。

由於他的大膽革新，吸收京劇及電影等的化裝、布景及伴奏樂技巧，並且他對問字求腔及對劇本之追求，使粵劇在各方面的成長極大幫助。他終於贏得了“粵劇伶王”、“萬能泰斗”等美譽，並被[[英國|英國]][[倫敦|倫敦]]“國際哲學科學藝術學會”聘為會員。

==家庭==
薛覺先育有一名兒子[[薛鴻楷|薛鴻楷]]。

== 故居 ==
===被強拆===
2013年6月10日深夜，香港[[麗豐控股|麗豐控股]]附屬公司廣州翠樺地產置業有限公司不理會當局發出的「緩拆令」，偷偷使出兩台挖土機，將位於廣州市[[越秀區|越秀區]][[觀綠路|觀綠路]]和[[詩書路|詩書路]]交界的金陵台和妙高台強行拆毀，事件引起全國嘩然。這兩棟建築物是中華民國時期的建築。薛覺先曾於二十世紀五十年代初在妙高台四號居住了兩三年。時任[[香港旅遊發展局|香港旅遊發展局]]主席[[林建岳|林建岳]]乃該公司的法定代表人。

事实上早于2012年5月,翠桦公司曾强拆妙高台一半,引起轩然大波。广州市规划局曾发出《关于暂缓拆除越秀区诗书路两处建筑的函》，但这份限期不明的“缓拆令”如今并无法阻止两幢建筑被毁。2012年的清拆亦有遭到过市文广新局的制止，其聘请专家在现场勘查后认定,金陵台和妙高台这两座建筑物
{{cquote|为民国时期兴建的建筑物,具有较好的'''历史、科学和艺术价值'''，建议作为'''历史建筑'''予以保留,并采取必要的措施加以保护。}}
就在当月,市国土房管局通知翠桦公司暂缓对金陵台2、4号和妙高台1、3号等房屋的拆除,并于同年8月对翠桦公司拆迁延期手续作退案处理,并建议该公司向市规划局申请明确规划红线范围后,再申请拆迁延期<ref name="凤凰">[http://news.ifeng.com/a/20140802/41412046_0.shtml 妙高台3号被认定非文物 三业主索赔480万败诉]，凤凰网（转载新快报） 2014-08-02（2014-08-03查阅）。{{zh-cn}}</ref>。
===改口不当历史建筑===
第二次拆楼后妙高台业主入禀越秀区法院控告清拆方翠桦公司侵犯其所有权，索偿481万元，2014年8月1日法院裁定不予支持原告，其中判词指政府在出让土地使用权时均未提出涉案房屋属于历史建筑不能拆除，又指案件审讯期间，文物管理部门有明确表示涉案房屋不属于文物保护单位,也不属于尚未核定为文物保护单位的不可移动文物，规划部门也表示涉案房屋不属于广州市第一批历史建筑<ref name="凤凰" />。根据行政部门意见法院就认定被毁故居不属于应予保留的历史建筑，恐怕法理基础不足，难彰公义。
