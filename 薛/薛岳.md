'''薛岳'''（{{bd|1896年|12月27日|1998年|5月3日|薛岳}}），原名'''仰岳'''，字'''伯陵'''，[[广东省|广东省]][[韶关市|韶关市]][[乐昌县|乐昌县]]（今广东省乐昌市）人，[[客家人|客家人]]，[[中華民國陸軍|中華民國陸軍]][[一級上將|一級上將]]，[[八年抗戰|八年抗戰]]名將之一，曾在抗日战争中取得多次对日作战胜利，被认为是“抗战中歼灭日军最多的中国将领”。<ref>{{cite web|url=http://news.china.com.cn/rollnews/2010-02/21/content_664334_2.htm|title=3次指挥长沙会战 薛岳:歼灭日军最多的中国将领|publisher=中国网|date= 2010-02-21 |accessdate=2014-02-20|language=zh-cn}}</ref><ref>{{cite web|url=http://news.ifeng.com/history/minjianshuoshi/chenqin/detail_2013_11/15/31293374_0.shtml|title=抗战时期国共两党全部将领哪位歼灭日军人数最多？|publisher=凤凰网|author=陈钦|date=2013-11-15|accessdate=2014-02-19|language=zh-cn}}</ref>

原名薛仰岳，因出生时在中日[[甲午战争|甲午战争]]中[[清朝|清朝]]刚战败，其父起名“仰岳”以求能效法[[岳飞|岳飞]]成为民族[[英雄|英雄]]。薛仰岳后改名薛岳，以示不仅仰慕岳飞，更将身体力行。{{citation needed|祖籍湖南郴州汝城县，据《薛岳自传》，薛岳的爷爷从湖南的客家聚居地─汝城县迁移到乐昌坪石, 而坪石是一个距离湖南只有10公里的小镇。}}

== 早年 ==
[[File:第十四次大會主席薛岳.jpg|thumb]]
1909年加入[[中国同盟会|中国同盟会]]，1912年春考入[[黄埔|黄埔]][[陆军小学|陆军小学]]，1917年初进入[[保定陸軍軍官學校|保定陸軍軍官學校]]第六期。1918年7月，在保定军校尚未毕业，便南下广东，参加孙中山新建立的援闽粤军，任司令部上尉参谋。後曾任[[孙中山|孙中山]]警卫团第一营营长。

1922年[[陈炯明|陈炯明]]与孙中山闹翻双方陷于战事，薛岳曾保护[[宋庆龄|宋庆龄]]脱险。[[北伐|北伐]]初期任[[国民革命军第一軍|国民革命军第一軍]]第一師師長。1927年[[四一二事变|四一二事变]]之前，他曾建议[[中共中央|中共中央]]：“把[[蒋介石|蒋介石]]当反革命抓起来！”之後多次参與的反蔣戰爭，屬於粵系軍[[張發奎|張發奎]]麾下。

== 第一次國共內戰 ==
1927年蔣介石[[四一二事变|清黨]]，薛岳因“不反对逮捕共产党，但是反对枪杀他们”被迫离开蒋介石第一军，回到广东改投[[李濟深|李濟深]][[国民革命军第四軍|国民革命军第四軍]]，出任新编第二师师长。1927年11月17日凌晨爆发[[广州张黄事变|广州张黄事变]]，薛岳投向了[[张发奎|张发奎]]。

[[广州起义_(1927年)|广州起义]]发生后，粤军第四军的主力在[[东江|东江]]一时回不来，只好把[[西江|西江]]方向的薛岳师调入城，彻底镇压了广州城内的共产党。

1927年下半年这一系列事件导致粤军第四军损失惨重，接受蒋介石改编参加二期北伐。第四军内部发生纷争赶走军长[[缪培南|缪培南]]，迎回张发奎任军长，薛岳又顶上去做张的副手。

中原大战时，粤军第四军倒向桂系李宗仁、白崇禧一起反蒋。粤（张发奎）桂联军进入衡阳的时候，粤军的蒋光鼐、蔡廷锴部尾随而至，断了张发奎后路。随之衡阳决战，结果张、桂联军惨败。张发奎下手，第四军被迫归顺蒋介石，实行中央化，薛岳当上了第四军军长。

[[国共战争|国共战争]]前期，为蒋介石重新重用追剿[[中国工农红军|中国工农红军]]，曾经多次重创红军。

中央红军[[二万五千里长征|长征]]，率领国民党中央军8个师穷追不舍，有「紅軍一路跑，薛岳一路追」之說，被當時紅軍視為「[[長征|長征]]頭號敵人」。红军入湘后，蒋介石任命[[何健|何健]]为追剿军总司令，薛岳为前敌总指挥、第二路司令，辖第59、90、92、93师。1934年11月「[[湘江戰役|湘江戰役]]」，薛部追上紅軍大戰六天，紅軍自原先8萬6千兵力剩下殘軍3萬潰敗於薛岳。1935年1月25日,蒋介石任命薛岳为贵阳绥靖主任，掌握了[[贵阳市|贵阳市]]与省政。1935年2月2日，成立湘鄂川黔边区剿匪军第二路军，总司令[[龙云|龙云]]，共十二个师、四个旅，薛岳为前敌总指挥，对长征路过云南的红军第一方面军主力进行防堵追剿，该路军下辖：
* 第一纵队：[[吴奇伟|吴奇伟]]部辖第59、90、92、93师，共4个师；
* 第二纵队：[[周浑元|周浑元]]部辖第5、13、96、99师，共4个师；
* 第三纵队：[[滇军|滇军]][[孙渡|孙渡]]部第2、5、7、预备旅共4个旅；
* 第四纵队：[[黔军|黔军]][[王家烈|王家烈]]部第1、2、3、8、教导师，共5个师。

== 抗日戰爭 ==
1937年6月2日，貴州省政府代主席薛岳宣誓就職<ref name="民">{{cite book |editors=[[李新_(1918年)|李新]]總主編，中國社會科學院近代史研究所中華民國史研究室，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=北京 |publisher=[[中華書局|中華書局]] |year=2011 |isbn=9787101079982}}</ref>{{rp|5437}}。[[抗日战争|抗日战争]]期间，参加[[淞沪会战|淞沪会战]]，指挥了[[武汉会战|武汉会战]]、[[徐州会战|徐州会战]]、[[长沙会战|长沙会战]]等著名会战；分別任[[第一戰區|第一戰區]]第一兵團司令，[[第九戰區|第九戰區]]第一兵團司令，第九戰區副总司令。1944年3月5日，[[湖南省政府|湖南省政府]]主席、第九戰區代司令長官薛岳在國民政府行政院會議發言中指出：「湖南省戰時對國家貢獻居全國之冠！」與會者全體起立鼓掌。<ref name="降書">{{cite news|title=〈國軍戰機「押送」日軍 湖南機場遞降書〉|publisher=《明報》|date=2015-08-30|page=新聞專題A10版}}</ref>薛岳曾经指挥[[万家岭战役|万家岭战役]]等多次战役，以其著名的“天爐戰法”，消灭了大量日军。1945年6月26日，[[國民政府|國民政府]]任命[[吳奇偉|吳奇偉]]為湖南省政府主席，原任薛岳免職<ref name=民/>{{rp|7746}}。

1946年10月10日，荣膺[[美国总统|美国总统]][[杜鲁门|杜鲁门]]所授[[总统自由勋章|总统自由勋章]]。

== 第二次國共內戰 ==
抗日战争结束后，薛岳指挥国军百万野战部队进攻[[华东|华东]][[解放区|解放区]]。1947年1月10日，國軍佔領蘇北[[沭陽|沭陽]]，蔣指示[[徐州|徐州]][[綏靖公署|綏靖公署]]主任薛岳，繼續進擊，求於10日內結束蘇北軍事<ref name=民 />{{rp|8263}}。

1947年1月[[鲁南战役|鲁南战役]]失败，薛岳把蒋介石最重视的机械化部队第一快速纵队给损失了，薛岳失去军事指挥权。参谋总长[[陈诚|陈诚]]接任华东国军指挥权，部署筹划了[[莱芜战役|莱芜战役]]，損失[[李仙洲|李仙洲]]部队6万余人。薛岳被当作替罪羊，1947年3月3日蒋介石以“指挥无力，名声低落”的罪名撤了薛岳的职。1947年5月10日，國民政府特任薛岳為[[中華民國總統府侍衛長|國民政府參軍長]]，原委[[商震|商震]]應免本職<ref name=民 />{{rp|8351}}。1948年5月，轉任[[总统府|总统府]]参军长。

== 撤離中國大陸 ==
1949年初，全家族从上海乘专列撤回广东[[乐昌|乐昌]]老家。1949年1月21日，代总统[[李宗仁|李宗仁]]任命薛岳為[[广东省政府|广东省政府]]主席兼广东省保安司令<ref>[[秦孝儀|秦孝儀]]總編纂：《總統蔣公大事長編初稿》卷七（下）</ref>{{rp|239}}。2月2日，廣東省政府主席薛岳對[[美聯社|美聯社]]記者聲稱，建立粵、桂、閩、湘四省聯盟<ref name=民 />{{rp|8801}}。2月4日，薛岳於廣州宣布施政三大原則：一、政治自由；二、經濟民主；三、軍事自衛<ref name=民 />{{rp|8803}}。中國共產黨占领大陆后，撤退至当时属[[广东|广东]]的[[海南岛|海南岛]]。1949年10月解放军发动[[广东战役|广东战役]]。

1949年12月28日，特派薛岳為[[海南特别行政区|海南特區]]行政长官公署副長官<ref>{{cite news|title=《總統府公報》第貳肆伍號|editors=總統府第五局|date=1949-12-31}}</ref>、海南防卫总司令官。薛岳把原来广东省的6个保安师运到海南，编成了[[國民革命軍第四軍|第四军]]、[[國民革命軍第六十三軍|第六十三军]]和[[國民革命軍第六十四軍|第六十四军]]，每军各2个师，又利用军舰和飞机在海南构筑了一道“伯陵防线”。把海南建设为反共基地，从这里“反攻大陆”。

== 台灣時期 ==
1950年4月，[[海南島戰役|海南島戰役]]失敗，薛岳撤出海南，隨後回台。之后升任[[一級上將|一級上將]]。在台曾任[[總統府|總統府]][[戰略|戰略]]顧問，[[行政院|行政院]][[政務委員_(中華民國)|政務委員]]，1966年出任[[光復大陸設計委員會|光復大陸設計委員會]]主任委員，1991年因光復會裁撤而自然卸任，擔任此職長達25年。

後隱居於台灣[[嘉義縣|嘉義縣]][[竹崎鄉|竹崎鄉]][[農村|農村]]大宅（忠恕邨），深居簡出；1980年代因為新出道歌手[[薛岳_(歌手)|薛岳]]同名同姓，將軍[[秘書|秘書]]去函建議歌手更改藝名遂又上新聞版面。

1998年5月3日逝世，享壽101歲。[[李登輝|李登輝]]令褒揚，全文如下：

{{cquote|總統府戰略顧問陸軍一級上將薛岳，剛健篤實，曉暢軍事。早歲追隨國父，獻身革命，警衛樞府，夙著忠勤。嗣參與東征、北伐、靖亂諸役，出入鋒鏑，迭建奇功；尤其剿共戰役，轉戰十數省，追奔二萬里，斬關奪寨，卓著勳勞，積功晉任陸軍上將。抗戰軍興，任第十九集團軍總司令等職，參與淞滬會戰，耀師蘭封及崑崙山諸戰場，揆文奮武，力挫強敵。旋任第九戰區司令長官，兼湖南省政府主席，民國二十八年、三十年及三十一年，日軍三犯長沙，將軍指揮若定，三殲頑敵，造成長沙三次大捷，戰功彪炳，聲震寰宇。勝利後，出任徐州綏靖公署主任、總統府參軍長、廣東省政府主席、兼海南防衛總司令等職，輔弼樞府，弘濟時艱。來臺後晉升陸軍一級上將，並於第一屆國民大會代表、總統府戰略顧問，暨光復大陸設計研究委員會主任委員等職任內，讜論閎謨，啟世牖民，忠藎精誠，良足矜式。綜其生平，緯武經文，公忠體國，勳績崇隆，名垂史乘，震古鑠今。遽聞溘逝，良深軫悼，應予明令褒揚，以示政府篤念勳耆之至意。}}
