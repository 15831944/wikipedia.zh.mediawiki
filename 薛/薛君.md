'''薛君'''（{{bd|1962年|||}}）[[山东省|山东省]][[临沂市|临沂市]][[费县|费县]]人，[[中国人民解放军少将|中国人民解放军少将]]<ref name=caixin>{{Cite web |url = http://china.caixin.com/2014-09-28/100734040.html |title = 薛君升任第20集团军政委 |author =  |date = 2014-09-28 |publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref><ref name=ifeng>{{Cite web |url = http://sd.ifeng.com/a/20170827/5945483_0.shtml |title = 薛君少将履新南海舰队副政委 山东人 |author =  |date = 2017-08-27 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
薛君1979年参军，先后在[[中国人民解放军军械技术学院|中国人民解放军军械技术学院]]、[[中共中央党校|中共中央党校]]、[[北京大学|北京大学]]、[[中国人民解放军国防大学|中国人民解放军国防大学]]学习，获工学学士 、管理学硕士学位，后为[[中国科学院|中国科学院]]博士，[[中国人民解放军南京政治学院|中国人民解放军南京政治学院]]客座教授。薛君历任干事、指导员、[[中国人民解放军总政治部办公厅|中国人民解放军总政治部办公厅]]研究员，2010年调入[[济南军区|济南军区]][[中国人民解放军陆军第二十集团军|陆军第二十集团军]]，后成为该集团军政治部主任。截至2014年曾三次立三等功<ref name=caixin/><ref name=ifeng/>。

2014年，薛君升任陆军第二十集团军政治委员，跻身正军级军官之列<ref name=caixin/><ref name=ifeng/>。2017年，陆军第二十集团军撤编，薛君担任[[中国人民解放军南部战区海军|中国人民解放军南部战区海军]]副政治委员<ref name=ifeng/>。

2011年12月晋升少将军衔<ref name=caixin/>。

==著作==
薛君在军内外报刊发表学术论文、研究文章二百多篇，并且多次获奖，出版有<ref name=ifeng/>：
*《欲立集》
*《解惑集》
*《知行集》<ref name=ifeng/>
