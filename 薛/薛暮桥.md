'''薛暮桥'''（{{bd|1904年|10月25日|2005年|7月22日|catIdx=Xue薛}}）原名'''薛雨林'''，[[江苏|江苏]][[无锡|无锡]]人，[[中华人民共和国|中华人民共和国]]经济学家，[[中国科学院|中国科学院]]哲学社会科学学部委员<ref name=shi/><ref name=rmw>{{Cite web |url = http://www.people.com.cn/item/newlsq/zz/lrw/r075.html |title = 薛暮桥(1904——) |author =  |date =  |publisher =人民网  |language =  |accessdate = 2017-02-15 |quote =  }}</ref><ref name=zgjjw>{{Cite web |url = http://www.ce.cn/economy/main/rwsy/200507/28/t20050728_4290476.shtml |title = 老一辈著名经济学家薛暮桥辞世 |author =  |date =2005-07-28  |publisher = 中国经济网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
薛暮桥生于无锡县礼社镇一个破落地主家庭。15岁时，薛暮桥正在江苏省立第三师范学校读初二，其父在50岁生日前几天因害怕债主逼债而[[上吊自杀|上吊自杀]]身亡。半年后，薛暮桥辍学来到[[杭州站|杭州站]]当练习生，学习[[会计|会计]]。22岁时，薛暮桥已成为[[笕桥站|笕桥站]]站长<ref>{{cite web|title=薛暮桥:从旧中国"牢监大学"走出来的著名经济学家|url=http://finance1.ce.cn/macro/jjbg/grxsbg/200606/15/t20060615_7361551.shtml|publisher=中国经济网|date=2006-06-15}}</ref>，是当时[[沪杭铁路|沪杭铁路]]各车站中最年轻的站长<ref name=laojian>{{Cite web |url = http://news.sohu.com/20050730/n226504432.shtml |title = 牢监大学毕业 |author =  |date = 2005-07-30 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。1927年，薛暮桥加入[[中国共产党|中国共产党]]。曾任[[沪杭甬铁路|沪杭甬铁路]]总工会常委、组织部长<ref name=shi/><ref name=rmw/><ref name=zgjjw/>。不久，在[[中国国民党清党|中国国民党清党]]中，薛暮桥在[[杭州|杭州]]被捕入狱，和[[中共浙江省委|中共浙江省委]]书记[[张秋人|张秋人]]都被关在甲监5号。在3年的狱中生活中，薛暮桥学习[[世界语|世界语]]、世界通史，以及[[苏联|苏联]]和西方学者的[[政治经济学|政治经济学]]著作<ref name=laojian/>。

出狱后不久，薛暮桥遇到了经济学家[[陈翰笙|陈翰笙]]。陈翰笙注重田野调查<ref name=wujia/>。1930年代初，薛暮桥参加了中国农村经济问题调查<ref name=zgjjw/>。经一个月调查，薛暮桥发表了首篇经济学调查报告《江南农村衰落的一个缩影》，以家乡礼社镇的薛姓家族经济状况演变为线索，揭示了农村封建经济破产的必然性；不久，此文被翻译到[[日本|日本]]<ref name=wujia>{{Cite web |url = http://news.sohu.com/20050730/n226504444.shtml |title = 稳定根据地物价 |author =  |date =  2005-07-30|publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

1931年之后，薛暮桥历任[[南京|南京]]民众教育馆《民众周报》编辑、中国农村经济研究会理事、《中国农村》月刊第一任主编<ref name=shi/><ref name=rmw/><ref name=zgjjw/>。《中国农村》月刊刊登了大量调查报告及论文，批评了农村改良主义，论证了土地制度变革的必要性<ref name=wujia/>。1935年5月5日，薛暮桥和[[罗琼|罗琼]]在[[上海|上海]]结婚。1935年9月，参加筹办[[三联书店|新知书店]]。1935年秋冬之际，根据[[中国共产党|中国共产党]]的《[[八一宣言|八一宣言]]》、《[[瓦窑堡会议|十二月决议]]》精神，在《中国农村》月刊上发文号召全中国的乡村工作者共赴国难。1935年12月，参与发起成立[[上海文化界救国会|上海文化界救国会]]。[[中国抗日战争|中国抗日战争]]时期，1938年10月后，历任[[新四军|新四军]]军部教导总队训练处处长、[[中国人民抗日军政大学|中国人民抗日军政大学]]华中总分校训练部部长，[[山东省参议会|山东省参议会]]参议员，山东省抗日政府工商局局长、山东省抗日政府秘书长兼实业厅厅长。1948年，任[[华北人民政府|华北人民政府]]财经办事处副主任、[[中共中央财政经济部|中共中央财政经济部]]秘书长<ref name=shi/><ref name=rmw/><ref name=zgjjw/>。

1949年后，薛暮桥任[[政务院财政经济委员会|政务院财政经济委员会]]秘书长兼私营企业局局长，[[中纪委|中纪委]]委员。1952年后，任[[中华人民共和国国家计划委员会|国家计划委员会]]委员、副主任，[[中华人民共和国国家统计局|国家统计局]]局长（1952年8月至1958年11月），[[中华人民共和国国家经济委员会|国家经济委员会]]副主任，全国物价委员会主任。1975年后，任国家计委顾问兼经济研究所所长，[[国家经济体制改革委员会|国家经济体制改革委员会]]顾问，国务院经济研究中心总干事，国务院经济技术社会发展研究中心名誉总干事，[[国务院发展研究中心|国务院发展研究中心]]名誉主任，中国计划学会会长，中国统计学会会长、名誉会长，中国物价学会名誉会长<ref name=shi/><ref name=rmw/><ref name=zgjjw/>。他是[[中国科学院|中国科学院]]哲学社会科学学部委员。是第一、二、三届全国人大代表，第六届全国人大常委会委员，第六届全国人大财政经济委员会委员；中国人民政治协商会议第五届全国委员会委员<ref name=shi/><ref name=shishi/><ref name=rmw/>。

1979年，薛暮桥发表了系统总结中华人民共和国历史经验的专著《中国社会主义经济问题研究》，成为中国经济体制改革的启蒙教材。针对城乡大批失业和半失业人员的问题，薛暮桥提出社会主义劳动力仍归劳动者自己所有的观点。1980年6月，薛暮桥率先提出，从改革流通制度入手，比从改革分配制度入手更重要。1980年代末，薛暮桥提出中国各地区发展战略应当以全国战略规划为基础，沿海外向型地区的发展战略应与腹地发展战略配合起来研究。1991年，薛暮桥指出应当逐步改用[[利率|利率]]政策，尽快学会使用经济手段，运用财政、税收、银行信贷等经济杠杆加强[[宏观调控|宏观调控]]，加快建立和完善间接调控体系<ref name=zgjjw/><ref name=shi>{{Cite web |url = http://money.163.com/12/1202/18/8HO78LL300252G50.html |title = 新中国经济学人史——薛暮桥 |author =  |date = 2012-12-02 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。2005年3月，薛暮桥获第一届中国经济学杰出贡献奖<ref>{{Cite web|url =http://news.xinhuanet.com/newmedia/2006-04/28/content_4487191.htm|title =大师与中国经济同行——追思世纪老人薛暮桥|author =|date =2006-04-28|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170216055449/http://news.xinhuanet.com/newmedia/2006-04/28/content_4487191.htm|archivedate =2017-02-16}}</ref>

2005年7月22日，薛暮桥在[[北京|北京]]病逝，享年101岁<ref name=shishi>{{Cite web |url =http://news.sohu.com/20050730/n226504325.shtml  |title =经济学界泰斗薛暮桥逝世  |author =  |date =2005-07-30  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
