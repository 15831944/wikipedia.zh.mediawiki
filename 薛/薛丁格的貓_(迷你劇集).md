《'''薛丁格的貓'''》（{{lang-en|''Schrodinger's Cat''}}）是一部於2018年推出的[[中華民國|臺灣]][[迷你影集|迷你劇集]]，為[[民視無線台|民視無線台]]於週日十點時段推出的「金鐘幸福好戲」，全劇共2集，分別於2018年4月22日及29日首播，由[[沈丹桂|沈丹桂]]執導、[[張耿銘|張耿銘]]執筆、[[天馬傳播事業有限公司|天馬傳播事業有限公司]]製作，[[方馨|方馨]]、[[夏于喬|夏于喬]]、[[柯叔元|柯叔元]]與[[鄭人碩|鄭人碩]]領銜主演。<ref name="FTV20180419-2"/>

== 製作 ==
本劇顧名思義是取自[[奧地利|奧地利]][[物理學家|物理學家]][[埃爾溫·薛定諤|埃爾溫·薛丁格]]於1935年提出的[[量子力學|量子力學]][[思想實驗|思想實驗]]「[[薛定諤貓|薛丁格的貓]]」，透過存在[[現實|現實]]與[[幻覺|幻覺]]、[[記憶|記憶]]與[[失憶症|失憶]]、[[意識|意識]]與[[催眠|催眠]]等矛盾狀態[[態疊加原理|疊加]]的[[心理疾患|心理錯亂]]角色，揭露一間神秘病房的[[謀殺|謀殺]]案件，藉以挑戰觀眾的[[推理|推理]][[邏輯|邏輯]]與尺度，描繪出人性的黑暗面。<ref name="FTV20180226"/><ref name="FTV20180419-2"/><ref name="ETtoday20180226"/><ref name="FTV20180315-1"/>本劇為[[民間全民電視公司|民視]]首部推理[[驚悚|驚悚]]劇，<ref name="FTV20180419-2"/>於2018年2月25日開鏡，<ref name="FTV20180226"/><ref name="ETtoday20180226"/>4月22日以及29日於晚間10點播映至11點45分。<ref name="Schedule"/>

== 演員與角色 ==
* [[方馨|方馨]] 飾演 莊媛伶
: 45歲，出身自醫學世家的[[精神科醫師|精神科醫師]]，以[[催眠|催眠]]治療見長，總能讓病患放鬆，並安心地受她引導找到傾訴的出口。
* [[夏于喬|夏于喬]] 飾演 陳曉蝶
: 32歲的[[護理人員|護理師]]，由於父親[[外遇|外遇]]造成家庭的破裂，使她扛著沉重的陰影成長，自此成為心中的一個缺口，也因長期處於壓力之下，而疑似患有[[思覺失調症|思覺失調症]]，遊走於真實與虛假的混亂之中。
* [[柯叔元|柯叔元]] 飾演 鍾孟國
: 48歲，[[神經外科|腦神經外科]]醫師，出身平凡的他，對事業極具野心，雖然不愛妻子，但妻子的背景卻能讓他上位，面對感情，他只能空虛地遊走在花花世界之中。
* [[鄭人碩|鄭人碩]] 飾演 周德鑫
: 35歲的[[科技|科技]][[工程師|工程師]]，是個腳踏實地又肯努力的[[青壯年|青壯年]]，但是耕耘多年的感情，卻似乎未能得到預期中的結果。
* [[蘇達|蘇達]] 飾演 [[魔幻對決#主要演員|江政達]]
: 偵查隊小隊長，以破案作為人生目標，有著不惜一切也要追求真相的執著。
* [[洪于晴|洪于晴]] 飾演 蔡佩玲
: 26歲的新進護理師，是個勇於追求所渴望之人事物的年輕人。
* [[邱德洋|邱德洋]] 飾演 [[魔幻對決#其他演員|吳清祥]]
: 資深[[警察|警官]]，比起真相，只求破案求生。

=== 選角 ===
曉蝶經常進入[[解離症|解離]]的狀態，因此精神容易緊繃，亦存在[[通姦|不倫戀]]的關係，還成為[[殺人|殺人]][[犯罪嫌疑人|嫌疑犯]]，這與夏于喬過往較常飾演的清純角色有很大的差異。<ref name="FTV20180315-1"/><ref name="FTV20180419"/>由於角色常處在恐怖的氛圍下，故夏于喬必須自我揣摩[[恐懼|恐懼]]的程度，以詮釋角色的心魔。<ref name="FTV20180315-1"/><ref name="FTV20180315-2"/><ref name="FTV20180419-2"/>以往較常演出剛硬、個性鮮明等角色形象的鄭人碩，此次飾演待女友相當溫柔的暖男，<ref name="FTV20180226"/><ref name="ChinaTimes20180226"/>但亦有詮釋精神分裂症的橋段，<ref name="FTV20180315-2"/>並為了配合劇情的緊湊氛圍，刻意讓自己的容貌顯得疲憊，以符合劇中的狀態。<ref name="FTV20180315-1"/>

方馨是繼2003年的《[[明星情人夢|明星情人夢]]》後，時隔15年接演[[中華民國國語|國語]]劇集，長年演出[[臺灣話|臺語]]戲劇的她，坦言因跳脫舒適圈而感到些許的不適應。方馨飾演的精神科醫師，台詞充斥專業術語，亦得以冷靜的語調引導病患，甚至還需和病患鬥智，她表示自己另外還觀賞了電影《[[你好，瘋子！|你好，瘋子！]]》，來為角色做準備。<ref name="FTV20180226"/><ref name="ChinaTimes20180226"/><ref name="NextMag"/>第一次演出驚悚戲劇的柯叔元，坦言自己為了詮釋角色，花了不少精力研究，<ref name="FTV20180419-2"/>且有突破性的表演。<ref name="FTV20180226"/><ref name="ChinaTimes20180226"/>

方馨與柯叔元在拍攝期間，仍共同參與著[[民間全民電視公司戲劇節目列表#八點檔|民視八點檔]]《[[幸福來了|幸福來了]]》的演出，並且在本劇同樣會有著情感上的糾葛。<ref name="FTV20180226"/>方馨與夏于喬時隔十幾年再度合作，她認為夏于喬的演出具有層次，也受到激發而改變自己的演出方式。談及首次合作的鄭人碩，方馨則透露自己因習慣拍攝八點檔而總是照著劇本演出，但鄭人碩的演出較為即興且具爆發力。<ref name="FTVGBA"/>

由於劇中角色的情緒層次豐富，故演員群事前均接受了[[郎祖筠|郎祖筠]]的表演課訓練。<ref name="FTV20180226"/><ref name="ChinaTimes20180226"/>導演[[沈丹桂|沈丹桂]]因欣賞鄭人碩具有演繹亦正亦邪之角色的能力，而邀請他參演，同時也認為夏于喬詮釋一名精神狀態不穩定的角色，是具突破性的演出，<ref name="FTV20180427"/>夏于喬亦特地為戲接髮。<ref name="FTV20180419-2"/>
