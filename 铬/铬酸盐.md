'''铬酸盐'''是[[铬酸|铬酸]]形成的非多聚[[阴离子|阴离子]]的盐，含有'''铬酸根'''离子——{{chem|CrO|4|2-}}，呈现特征性的黄色；'''重鉻酸根'''化學式則為 Cr<sub>2</sub>O<sub>7</sub><sup>2−</sup>，在水性溶液中呈橙色。铬酸盐中的[[铬|铬]]为 VI [[氧化态|氧化态]]，具氧化性、有毒且具有[[致癌性|致癌性]]，被[[国际癌症研究机构|国际癌症研究机构]]（IARC）划分为第一类致癌物质。

常见的铬酸盐有[[铬酸钠|铬酸钠]]、[[铬酸钾|铬酸钾]]、[[铬酸铅|铬酸铅]]和[[铬酸铵|铬酸铵]]。更多请参见[[:Category:铬酸盐|列表]]。

== 性质 ==
*铬酸盐具[[氧化性|氧化性]]，还原Cr<sup>VI</sup>常得到蓝绿色的Cr<sup>III</sup>。
*[[水溶液|水溶液]]中，铬酸根离子（黃色）与重铬酸根离子（橙色）处于[[化学平衡|平衡]]。加酸促进重铬酸根离子的生成，使溶液呈红色；加碱则使平衡左移，溶液呈黄色：

::2 CrO<sub>4</sub><sup>2−</sup> + 2 H<sup>+</sup> ⇌ Cr<sub>2</sub>O<sub>7</sub><sup>2−</sup> + H<sub>2</sub>O

*铬酸盐可用于测定水体中的[[化学需氧量|化学需氧量]]（COD）。
*[[重金属|重金属]]、[[镧系元素|镧系元素]]和[[碱土金属|碱土金属]]的铬酸盐大多微溶或难溶于水，用途很少。[[碱金属|碱金属]]和[[铵根离子|铵根离子]](NH<sub>4</sub><sup>+</sup>)铬酸盐的溶解度相对较大。当可溶性铬酸盐溶液加入一些离子时，会产生沉淀：
::Ba<sup>2+</sup> + CrO<sub>4</sub><sup>2-</sup> → BaCrO<sub>4</sub>↓（柠檬黄）
::Pb<sup>2+</sup> + CrO<sub>4</sub><sup>2-</sup> → PbCrO<sub>4</sub>↓（铬黄）
::2 Ag<sup>+</sup> + CrO<sub>4</sub><sup>2-</sup> → Ag<sub>2</sub>CrO<sub>4</sub>↓（砖红）<ref>《无机化学》第四版.高等教育出版社.P<sub>387</sub></ref>
*[[铬酸盐转化膜|铬酸盐转化膜]]可用于防止金属腐蚀及增强涂料的粘合力。
*[[氯铬酸吡啶盐|氯铬酸吡啶盐]]在[[有机合成|有机合成]]用于氧化[[醇|醇]]为[[醛|醛]]。
