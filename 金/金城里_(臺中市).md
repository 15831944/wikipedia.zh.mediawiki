'''金城里'''，位於[[臺中市|臺中市]][[大里區|大里區]]南側。

== 里名由來 ==
以「塗城」之「城」字，另加雅字「金」，定名為「金城」。

== 地理 ==
金城里面積0.7008平方公里，劃分為45鄰。本里東側及南側為[[瑞城里|瑞城-{里}-]]瓖繞，西隔中興路鄰[[東湖里_(臺中市)|東湖-{里}-]]，北為[[仁德里_(臺中市大里區)|仁德里]]。

== 行政區劃沿革 ==
金城里於[[清朝|清代]]時隸屬[[藍興堡|藍興堡]][[土城庄|土城莊]]；[[明治|明治]]34年（1901年）時，改隸[[臺中廳|臺中廳]]太平區塗城庄，至大正9年（1920年）則隨行政區調整，隸屬大屯郡大里庄塗城大字。[[第二次世界大战|二次大戰]]結束後，[[國民政府|國民政府]]接收臺灣，以原塗城大字範圍置'''[[塗城里|塗城村]]'''，隸屬[[臺中縣|臺中縣]][[大里區|大里鄉]]。[[中華民國|民國]]65年（1976年）析塗城村置'''金城村'''。民國82年（1993年）大里鄉升格為縣轄市'''[[大里區|大里市]]'''，隨之改制為'''金城-{里}-'''。
