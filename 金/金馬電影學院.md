'''金馬電影學院'''（{{lang-en|Golden Horse Film Academy}}）是2009年由[[臺北金馬影展執行委員會|臺北金馬影展執行委員會]]主席[[侯孝賢|侯孝賢]]創辦<ref>{{Cite web |url = http://www.ettoday.net/news/20151121/600937.htm?feature=Auto&tab_id=150 |title = 金馬史上第一次！侯孝賢2度拿傑出電影工作者 |author = 林映妤 |date = 2015-11-21 |publisher = 東森新聞 |language = zh-hant |accessdate = 2015-09-24}}</ref>。

== 歷史沿革 ==
首屆金馬電影學院僅由活動統籌彭雁筠負責承辦，並邀張鳳美擔任製片<ref>{{Cite web |url = https://star.ettoday.net/news/1300176 |title = 金馬超進化／侯孝賢10年前任性1句話　華語導演名次正在洗牌 |author = 洪文 |date = 2018-11-15 |publisher = 東森新聞 |language = zh-hant |accessdate = 2018-11-20}}</ref>。2015年[[侯孝賢|侯孝賢]]續任院長，並將原有二組名額增加一名學員，共錄取了14位<ref>{{Cite web |url = http://udn.com/news/story/7262/1217928 |title = 為金馬電影學院養人才 侯孝賢續任院長 |author = 項貽斐 |date = 2015-09-29 |publisher = 聯合報 |language = zh-hant |accessdate = 2015-09-24}}</ref><ref>{{Cite web |url = http://www.nownews.com/n/2015/09/29/1828862 |title = 金馬電影學院變「精英練功堂」　14學員再提升 |author = 翁新涵 |date = 2015-09-29 |publisher = 今日新聞 |language = zh-hant |accessdate = 2015-09-24}}</ref>。
