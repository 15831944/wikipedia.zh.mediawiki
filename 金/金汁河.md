'''金汁河'''，是流经[[中华人民共和国|中国]][[云南省|云南省]][[昆明市|昆明市]]主城区北部和东部的一条人工河道，是昆明古六河之一。

金汁河原是[[盘龙江_(滇池)|盘龙江]]的一条分支，最早开挖于[[南诏|南诏]]时期，原名“金棱河”。[[大理国|大理国]][[段素兴|段素兴]]在位时期，在河上筑 “春登堤”，种迎春柳，“黄花入河，如金汁然”，故名“金汁河”。

现金汁河起于昆明市[[盘龙区|盘龙区]][[龙泉街道_(昆明市)|龙泉街道]]中坝社区[[松华坝水库|松华坝水库]]大坝，蜿蜒向南流经上坝社区，下穿瀑布公园，经宝云社区、盘江社区、[[金辰街道|金辰街道]]任旗营社区、云波社区、天祥社区、[[联盟街道_(昆明市)|联盟街道]]金菊路社区、[[东华街道_(昆明市)|东华街道]]白云社区、新迎社区、[[青云街道_(昆明市)|青云街道]]佳园社区、西林社区、昙华社区等地，于王大桥流入[[官渡区|官渡区]]，南流过官渡区[[金马街道|金马街道]]大树营社区后转西流，经金马社区、曙光社区、[[吴井街道|吴井街道]]东郊路社区和董家湾南段社区后，过环城东路后复入盘龙区，环流[[拓东体育馆|拓东体育馆]]北侧后南流，过拓东路后再次进入官渡区，继续向南流经吴井街道石井社区、铁路新村社区、吴井新村社区、[[太和街道_(昆明市)|太和街道]]佴家湾社区、[[关上街道|关上街道]]双桥社区、福德社区、福发社区、日新社区、小街社区、双凤社区，最后在[[小板桥街道|小板桥街道]]中闸社区官南路与双凤路交汇处附近汇入枧槽河（清水河）。河道全长约28公里。<ref>{{cite web |title=昆明官渡金汁河补水再现水清岸绿景观 |url=http://news.163.com/15/1116/23/B8J3FG0700014AEE.html |website=网易 |accessdate=2018-10-04}}</ref>

金汁河历史上具有灌溉和防洪的功效，<ref>{{cite web |title=官渡区完成金汁河阶段性治理任务千年古河焕新颜|url=http://news.ifeng.com/gundong/detail_2012_11/19/19291127_0.shtml |website=凤凰网 |accessdate=2018-10-04}}</ref>随着城市发展，现在金汁河已成为一条城区排污沟渠和雨汛期行洪河道，两岸多开发为住宅小区和绿地公园。

金汁河在盘龙区金辰街道任旗营社区老李山附近还承接了第五水质净化厂每天18.5万方尾水。<ref>{{cite web |title=金汁河近3公里河道除险加固 第五水质净化厂每天18.5万方尾水排入金汁河(图)
