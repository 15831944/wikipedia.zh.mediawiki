'''金版'''，或称'''爰金'''、'''印子金'''、'''楚金钣'''、'''楚金版'''等，中国古代货币，[[春秋时期|春秋]]晚期到[[战国|战国]]末期的[[黄金|黄金]][[称量货币|称量货币]]，铸行于[[楚国|楚国]]。金版的形状多为版形，亦有饼形，多带有钤刻文字。

金币上带有的文字有“郢爰”、“陳爰”、“專爰”、“[[䣓爰|䣓爰]]”、“鑪金”、“垂丘”、“{{缺字|suo3|1}}”等印记，其中“[[郢|郢]]”、“[[陈国|陈]]”、“专”等均为地名，“爰”或释读为“爯”，为古代一种重量单位或货币单位，说法不一。

金版在[[湖北|湖北]]、[[湖南|湖南]]、[[安徽|安徽]]、[[陕西|陕西]]、[[河南|河南]]、[[山东|山东]]和[[江苏|江苏]]等省都有出土。出土的金版最多的是“郢爰”。已发现的整版“郢爰”印有戳记多在五十个左右，每件约重250至260克。爰金含金量非常高，一般在93%－97%之间，只有少量的会在85%左右。

== 名称 ==
[[File:Ying_yuan,_Guangdong_Museum.JPG|thumb]]所藏的郢爰]]
楚国的金版早在[[东晋|东晋]]时已见著录，《[[宋书|宋书]]·符瑞志》记载，东晋永和元年（345年），在舂谷城曾有出土。<ref>{{Citeweb |title=宋书·志第十九·符瑞下 |url=http://www.xysa.net/a200/h350/06songshu/s-029.htm |quote=晋穆帝永和元年二月，舂谷民得金胜一枚，长五寸，状如织胜。明年，桓温平蜀。永和元年三月，庐江太守路永上言，于舂谷城北，见水岸边有紫赤光，取得金状如印，遣主簿李迈表送。}}</ref>这是楚印记金钣最早的出土记载。<ref>张宏明. [http://cpfd.cnki.com.cn/Article/CPFDTOTAL-AHQB199309001024.htm 论楚贝币及铸贝钱范的特色].《钱币文论特辑（第二辑）》1993年</ref>北宋[[沈括|沈括]]的《[[梦溪笔谈|梦溪笔谈]]·异事》中记载：“寿州八公山侧土中及溪涧之间，往往得小金饼，上有篆文‘刘主’字，世传‘淮南王药金’也。得之者至多，天下谓之‘印子金’是也。”<ref>{{Citeweb|title=梦溪笔谈 卷二十一　异事异疾附 |url=http://gj.zdic.net/archive.php?aid=14200 |publisher=汉典古籍 |quote=寿州八公山侧土中及溪涧之间，往往得小金饼，上有篆文“刘主”字，世传“淮南王药金”也。得之者至多，天下谓之“印子金”是也。然止于一印，重者不过半两而已，鲜有大者。余尝于寿春渔人处得一饼，言得于淮水中，凡重七两余，面有二十余印，背有五指及掌痕，纹理分明。传者以谓埿之所化，手痕正如握埿之迹。}}</ref>清嘉庆年间所纂的《凤台县志》记述了当地出土“郢爰”金币，仍称其为“药金”；清末[[方濬益|方濬益]]据《[[尔雅|尔雅]]·释器》中“饼金谓之钣”和《[[周礼|周礼]]·职金》中“旅于上帝则共其金钣”的记载，考释其形制为饼金、金钣，并将其著作《缀遗斋彝器考释》中所载的“郢爰”、“陈爰”定名为“郢金钣”和“陈金钣”；其后有“郢爰黄金币”、“郢爰金饼”、“郢金”、“金饼”、“金版”、“爰金”、“楚金钣”等称法，安志敏在《金版与金饼》一文中称楚国金币为“金版”，而称汉代麟趾金、马蹄金为“金饼”。简柱华在《印子金正名》一文中认为方濬益所称的“金版”最为合适。<ref name=jian>简柱华. 印子金正名. 中国钱币.1989年01期</ref>

== 黄金来源 ==
楚兴起于[[汉江_(中国)|汉江流域]]的[[淅川|淅川]]一带，楚人来源尚无定论。[[西周|西周]]初年[[周成王|周成王]]时，[[熊绎|熊绎]]被封为[[子爵|子爵]]，[[楚国|楚国]]成立。建国后，逐步吞并周边的[[方国|方国]]、[[部落|部落]]成为雄踞[[汉江_(中国)|汉江]]和[[长江|江]][[淮河|淮]]流域的强国。<ref name="huah">胡世平;楚爰金的诞生和消亡[J];安徽钱币;2007年04期</ref>
楚国盛产黄金，当时就有“黄金珠玑犀象出于楚”{{注|语出《战国策·卷十六·楚三》：张子曰：“王无求晋国乎？”王曰：“黄金珠玑犀象出于楚，寡人无求于晋国。”}}的说法，《[[管子_(書籍)|管子]]》中曾提到“楚有汝汉之金”，可知楚国境内的[[汝水_(古河流)|汝]][[汉江_(中国)|汉]]流域在当时已是盛产黄金之地；而根据[[徐中舒|徐中舒]]《试论岷山庄王与滇王庄蹻的关系》的观点，四川[[丽水市_(中国)|丽水]]地区也是当时楚国黄金的重要产地<ref>徐中舒.试论岷山庄王与滇王庄蹻的关系.《思想战线》.1977年04期</ref>。通过各地出土金版分析，楚金币并非纯金，且含金量不一致，不具地域特点，而是与楚国其他金器一样具有[[自然金|自然金]]的特点。<ref>黄维.楚国黄金货币的成色与成分[J].中国钱币,2011,(3):16-21.</ref>

== 钱文 ==
楚国金版虽多钤有文字，但并不是全部，也有无印金版，1979年安徽寿县曾出土8件，整版重258.05-268.5克<ref name=dacid/>{{rp|23}}。

金版上的很多钱文的释读还没有定论。由于早期出土金版中整版的较少，曾有对整版金版上有几个戳记的讨论，但随着整版金版的大量出土，戳记已被证明仅是一种标记，并无定数。

== 铸造年代 ==
楚金版的铸造年代是一个复杂的问题。金版在安徽[[寿县|寿县]]大量出土当与[[楚考烈王|楚考烈王]]二十二年（前241年）楚国国都东迁到寿春并命名为“郢”有关，但由于“郢”是[[东周|东周]]时期楚国国都的代称，郢爰的铸造并不是始于前241年。自[[楚昭王|楚昭王]]迁都至[[鄀国|鄀]]时就称其为“鄢郢”，今湖北江陵被《汉志》称为“故楚郢都”，战国[[楚顷襄王|楚顷襄王]]时楚都被秦攻破，楚王逃至位于今河南[[淮阳|淮阳]]的陈城，也称陈为“郢陈”，再到[[楚考烈王|楚考烈王]]迁都寿春，还是称“郢”。湖北江陵、河南扶沟等地郢爰的出土也证实了郢爰在迁都寿春之前便有铸行。“陈爰”当为前275年迁都陈后铸造，陈本来是一个小国，曾被楚多次灭国，在当时已是楚国城邑。“专锊”的“专”也是地名，在鲁国东部，该金版当为楚考烈王八年灭鲁后所铸。因此，金版的铸行时间应始于不早于公元前三世纪过多的时间。多地曾出土西汉时期泥郢爰、陶郢爰，说明至西汉，金版仍有使用。

== 银版、铅版和金银饼 ==
除了金版，在已出土的同时期楚国文物中还发现了与金版相近的银版、铅版和金饼。银版为战国时期楚国白银称量货币，郑家相《中国古代货币发展史》中收录有一枚传说出土于安徽寿县的郢爰银版，呈长方曲版状。金饼为春秋战国时期的楚国黄金称量货币，多为圆饼形，也有近似于方形的，有的表面还带有刻文，发现的多已在入土之前就贝切成了碎块。银饼为战国时期的，呈圆饼状。
