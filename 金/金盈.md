'''金盈'''（{{lang-en|Liz Kam}}，{{bd|1990年|6月13日}}）是[[香港|香港]][[無綫電視|無綫電視]]的新聞從業員，現任[[無綫新聞台|無綫新聞台]]主播。<ref>{{cite web |title=新主播女神 揭金盈出鏡晒$20萬潮表 - 娛樂放題 - 娛樂追擊 |url=https://eastweek.my-magazine.me/main/84993 |website=東周網【東周刊官方網站】Eastweek.com.hk |accessdate=2019-03-23 |language=zh-tw}}</ref><ref>{{cite web |title=女神主播戴名錶報新聞 飛豪門預備備？ - 自由娛樂 |url=https://ent.ltn.com.tw/m/news/breakingnews/2690760 |website=ent.ltn.com.tw |accessdate=2019-03-23}}</ref>

==簡歷==
金盈2012年是[[香港珠海學院|香港珠海學院]]新聞及傳播學系畢業<ref>[https://www.chuhai.edu.hk/sites/default/files/FOASS2017.pdf 香港珠海學院文學與社會科學院課程簡介]</ref>。她初期於[[香港寬頻|香港寬頻]][[香港新聞台|香港新聞台]]當主播，2014年4月香港新聞台停播後，過檔[[鳳凰衛視香港台|鳳凰衛視香港台]]當主播，2015年12月再過檔[[無綫新聞|無綫新聞]]。
