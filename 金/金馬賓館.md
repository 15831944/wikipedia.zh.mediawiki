'''金馬賓館'''原是為前往金門、馬祖前線的[[中華民國國軍|中華民國國軍]]親屬送別、探親而建於[[臺灣|臺灣]]的賓館。

位於[[高雄市|高雄市]][[鼓山區|鼓山區]]鼓山一路111號[[壽山_(高雄市)|壽山]]山腳下的金馬賓館於1959年開始籌備、1967年由[[中華民國軍人之友社|中華民國軍人之友社]]興建完工後捐贈[[中華民國國防部|中華民國國防部]]，後於1998年轉由[[交通部鐵路改建工程局|交通部鐵路改建工程局]]管理使用；2012年，交通部鐵路改建工程局搬遷，金馬賓館一度廢棄閒置<ref>{{cite web |author1=蘇福男 |title=當兵感傷回憶地 金馬賓館變身軍事生態青年旅店 |url=http://news.ltn.com.tw/news/life/breakingnews/1689073 |website=自由電子報 |accessdate=2018-08-25}}</ref>；2018年，[[高雄市政府都市發展局|高雄市政府都市發展局]]委由御盟集團進行金馬賓館再造活化，蛻變為當代藝術館，並於2018年11月29日正式開幕。<ref>{{cite web |author1=顏瑞田 |title=高雄金馬賓館 變身文創旅館 |url=http://www.chinatimes.com/newspapers/20160425000109-260204 |website=中時電子報 |accessdate=2018-08-25}}</ref><ref>{{cite news |language = zh-hant |author = 王榮祥 |url = http://news.ltn.com.tw/news/life/breakingnews/1954080 |title = 高雄金馬賓館變身文創重地 預定今年冬天開張 |publisher = 自由時報 |date = 2017-01-20 |accessdate = 2018/08/01}}</ref><ref>{{cite news |language = zh-hant |author = 柯宗緯 |url = http://www.chinatimes.com/realtimenews/20170120005275-260405 |title = 全台唯一金馬賓館活化 變身文創藝廊 |publisher = 中國時報 |date = 2017-01-20 |accessdate = 2018/08/01}}</ref>。

位於[[基隆市|基隆市]]的金馬賓館已拆除<ref>[http://news.ltn.com.tw/news/local/paper/975407 〈南部〉全台僅存金馬賓館「第二春」 年中招商]，王榮祥，自由時報，2016年4月3日</ref>。
