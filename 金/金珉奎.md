'''金珉奎'''（{{kor|k='''김민규'''|rm='''Kim Min Gyu'''}}，{{bd|1997年|4月6日}}），藝名'''珉奎'''（{{kor|k='''민규'''|rm='''Min Gyu'''}}，{{jp|hg='''ミンギュ'''|rm='''Min Gyu'''|lead=yes|help=no}}），[[大韩民国|韓國]]男[[艺人|藝人]]、[[饒舌|饒舌]]歌手。 [[Pledis娛樂|Pledis娛樂]]旗下的13人男子流行音樂團體[[Seventeen_(組合)|SEVENTEEN]]成員，同時也是分隊Hip-hop Team的成員。

== 個人生涯 ==
1997年4月6日出生於[[大韩民国|韓國]][[京畿道|京畿道]][[安養市|安養市]]，家庭成員有父母及一位小四歲的妹妹。
珉奎從小學開始到-{zh-cn:初;zh-tw:國}-二當練習生前一直都是足球校隊的選手並擔任[[前鋒_(足球)|前鋒]]，曾於國小得過足球全國大賽第二名。也曾在塑膠飛機飛行大賽等比賽上得獎。{{notetag|於2017年《17 JAPAN CON Say the name SEVENTEEN》演唱會場刊中提及。|name=JCON}}

當上練習生的契機是在-{zh-cn:初;zh-tw:國}-二放學回家時被星探搭話，在面試中唱的歌是[[愛國歌_(大韓民國)|韓國國歌]]。合格並得到父母同意後，開始成為[[Pledis娛樂|Pledis娛樂]]練習生進行訓練。{{notetag|name=JCON}}

2016年2月3日畢業於{{Link-ko|首爾放送高等學校|서울방송고등학교}}資料處理科。

== 演藝生涯 ==
=== 2012－2015年：練習生時期 ===
[[File:Kim_Min-gyu_performing_at_Lotte_World_in_September_2013.jpg|thumb]]表演。]]

2012年，金珉奎與[[崔勝哲|S.coups]]、[[權順榮|Hoshi]]、[[全圓佑|圓佑]]和[[李知勳_(SEVENTEEN)|WOOZI]]出演[[NU'EST|NU'EST]]出道曲〈FACE〉音樂錄影帶；<ref name="FACE">{{cite video |language = ko |author = NU'EST |coauthors = 1theK |url = https://www.youtube.com/watch?v=gyXy0m-4bvE |title = [MV] NU'EST _ FACE |publisher = [[Youtube|Youtube]] |date = 2012-03-14 |accessdate = 2018-03-30}}</ref>又與Hoshi、圓佑、WOOZI出演[[Hello_Venus|Hello Venus]]出道曲〈Venus〉的音樂錄影帶。<ref name="Venus">{{Citation|last=HELLOVENUS 헬로비너스|title=HELLOVENUS - Venus M/V|date=2012-05-08|url=https://www.youtube.com/watch?v=zU3Ug5pw7sw}}</ref>

=== 2015年－至今：SEVENTEEN活動 ===
{{main|SEVENTEEN}}
2015年5月26日，珉奎以男子流行音樂組合[[Seventeen_(組合)|SEVENTEEN]]成員出道，同時也是三個分隊中{{lang|en|Hip-hop Team}}的成員。主要負責[[饒舌|Rap]]作詞，另外也參與作曲。
