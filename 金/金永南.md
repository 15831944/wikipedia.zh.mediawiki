'''金永南'''（{{kor|김영남|'''金永南'''|Kim Yong-Nam}}，{{bd|1928年|2月4日|||Kim, Yong-nam}}），本名'''金明参'''<ref name="electionsmeter">{{cite paper|title=Kim Yong-nam|url=http://electionsmeter.com/polls/kim-yong-nam?title=NorthKorea}}</ref>。[[朝鲜民主主义人民共和国|朝鲜]]前[[最高人民会议常任委员会委员长|最高人民会议常任委员会委员长]]和[[朝鲜劳动党中央政治局常委|朝鲜劳动党中央政治局常委]]（位列第二，仅次于[[金正恩|金正恩]]），自1998年到2009年<ref>{{cite paper|title=조선민주주의인민공화국 사회주의헌법|url=https://ko.wikisource.org/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98%ED%97%8C%EB%B2%95}}</ref><ref>{{cite paper|title=조선민주주의인민공화국 사회주의헌법을 수정보충함에 대한 보고|url=http://www.uriminzokkiri.com/index.php?ptype=igisa2&no=113290}}</ref>担任[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]在[[虚位元首|名义上]]的[[朝鲜民主主义人民共和国国家元首列表|国家元首]]。出生于[[中华民国|中华民国]][[奉天省|奉天省]]（今中華人民共和國[[辽宁省|辽宁省]][[宽甸县|宽甸县]][[振江镇|振江镇]]西江村）<ref name="electionsmeter">{{cite paper|title=Kim Yong-nam|url=http://electionsmeter.com/polls/kim-yong-nam?title=NorthKorea}}</ref>，毕业于[[金日成综合大学|金日成综合大学]]，曾在[[蘇聯|蘇聯]][[莫斯科国立大学|莫斯科国立大学]]留學。[[外交官|外交官]]出身，曾經出任政務院副总理兼[[朝鮮民主主義人民共和國外務省|外务省]][[外交部长|外务相]]<ref>[http://www.apdnews.com/news/30936.html 金永南抵京引猜测]，亚太日报，2013年8月2日</ref>。

他曾获[[金日成勋章|金日成勋章]]、劳动英雄称号、一级国旗勋章等。<ref name = j />

==經歷==
* 1925年<ref>https://prabook.com/web/yong-nam.kim/1411888</ref><ref name=s/>或1928年2月4日<ref name = j>沈志远 《金永南》//《世界首脑大典》吴锡山主编 第30页</ref>出生于[[咸镜北道|咸镜北道]][[明川郡|明川郡]]<ref name=s>《世界政治家大辞典 上册》作者: 杨元恪 陈有进等 第503页</ref>或[[平壤市|平壤市]]<ref name=j/>或[[中华民国|中华民国]][[奉天省|奉天省]]（今中国[[辽宁省|辽宁省]][[宽甸县|宽甸县]][[振江镇|振江镇]]西江村）<ref name="electionsmeter" />。
* 解放后就读于[[金日成大学|金日成大学]]<ref name=s />。
* 1953年在[[莫斯科大学|莫斯科大学]]留学<ref name=s /><ref name=m>《马克思主义人物辞典》何承艰 王德树 胡尔湖 第404页</ref><ref name = j />。
* 1956年任朝鲜劳动党中央委员会国际部秘书长<ref name=s /><ref name=m /><ref name = j />。
* 1960年8月-1971年4月任对外联络委员会副委员长<ref name=s /><ref name=m /><ref name = j />。
* 1961年任朝鲜劳动党中央委员会国际部副部长<ref name=s /><ref name=m /><ref name = j />。
* 1963年9月改任[[朝鮮民主主義人民共和國外務省|外务省]]副相（副部长）<ref name=s /><ref name=m /><ref name = j />。
* 1966年10月劳动党代表会议后任朝鲜劳动党中央委员会国际部副部长<ref name=s /><ref name = j />。
* 1967年兼任朝鲜职业同盟中央委员会副委员长<ref name=s />。
* 1968年兼任朝鲜拥护和平全国民族委员会副委员长<ref name=s /><ref name = j />。
* 20世纪60年代初从事对苏和东欧、中东的外交<ref name=s />。
* 1970年11月当选[[朝鲜劳动党中央委员会|朝鲜劳动党中央委员会]]委员<ref name=m /><ref name = j />。
* 1972年4月任朝鲜劳动党中央委员会国际部第一副部长<ref name=m /><ref name = j />。
* 1972年12月任朝鲜劳动党中央委员会国际部部长<ref name=m /><ref name = j />。
* 1972年12月当选为朝鲜第五届最高人民会议常设议会议员<ref name=m />。
* 1974年11月当选为[[朝鲜劳动党中央委员会政治局|劳动党中央政治委员会]]候补委员<ref name=m /><ref name = j />。
* 1975年被选为朝鲜劳动党中央委员会书记局书记<ref name=s />。
* 1977年12月当选为朝鲜第六届最高人民会议常设议会议员<ref name=s /><ref name=m />。
* 1978年8月升任[[朝鲜劳动党中央委员会政治局|劳动党中央政治委员会]]委員<ref name=s /><ref name=m /><ref name = j />，成为朝鲜领导人之一<ref name=y>金永南其人其事 《往事杂忆》延静 第38-39页</ref>仍任朝鲜劳动党中央委员会书记局书记、朝鲜劳动党中央委员会国际部部长<ref name=s />。
* 1980年10月当选劳动党六届一中全会上当选为朝鲜劳动党中央委员会政治局委员、朝鲜劳动党中央委员会书记局书记<ref name=s /><ref name=m /><ref name = j />仍任朝鲜劳动党中央委员会国际部部长<ref name = j />。
* 1982年2月当选为朝鲜第七届最高人民会议常设议会议员<ref name=j />。
* 1983年被免去朝鲜劳动党中央委员会书记局书记职务。<ref name=s />
* 1983年12月改任[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]副总理兼外务相（外交部长）<ref name=s /><ref name=m /><ref name = j />。
* 1986年11月当选为朝鲜第八届最高人民会议常设议会议员<ref name=j />。
*1989年9月任[[祖国和平統一委员会|祖国和平統一委员会]]副委員長。
* 1986年11月当选为朝鲜第八届最高人民会议常设议会议员<ref name=j />。
* 1990年4月当选为朝鲜第九届最高人民会议常设议会议员<ref name=j />。
* 1990年5月再次任政务院副总理兼外务相（外交部长）<ref name = j />。
* 在1994年金日成逝世的中央追悼大会上，金永南致悼词。
*1998年9月当选为朝鮮民主主義人民共和國第十届[[最高人民会议常任委员会委员长|最高人民会议常任委员会委员长]]。<ref name = j />
*2003年9月当选为朝鮮民主主義人民共和國第十一届[[最高人民会议常任委员会委員長|最高人民会议常任委员会委員長]]。
*2009年4月，在第12屆[[朝鲜最高人民会议|最高人民会议]]第1次會議上，再度續任常任委員會委員長。
*2010年9月，在[[2010年朝鲜劳动党代表会议|朝鲜劳动党代表会议]]上当选[[朝鲜劳动党中央政治局常务委员会|中央政治局常务委员会]]委员。
* 在2011年[[金正日|金正日]]逝世的中央追悼大会上，金永南致悼词。
* 2014年，第13屆最高人民会议，当选最高人民会议常任委员会委員長。
* 2019年，年届91岁的金永南卸任中央政治局常委和最高人民会议常任委员长，正式退休。

==外事活动==
* 20世纪70年代，[[金日成|金日成]]来中国每年一至两次访问，多为不公开行动，金永南每次来访都随行<ref name=y />。
* 1975年，[[金日成|金日成]]正式访华，金永南随同出席<ref name=y />。
* 1984年，金永南访问中国，除了在北京活动之外，还特意访问了[[深圳|深圳]]<ref name=y />。
* 1988年，金永南访问中国，在双方外长会谈中，[[钱其琛|钱其琛]]表示中国准备松动与韩国的关系<ref name=y />。
* 1992年中韩建交。7月钱其琛访朝，金永南在平壤顺安机场出面迎接，并陪同乘直升机前往[[延丰湖|延丰湖]]面见金日成<ref name=y />。
*1999年6月4日，金永南率朝鲜国家代表团访问中国，会见中共中央政治局常委、国务院总理[[朱镕基|朱镕基]]。<ref>[http://cpc.people.com.cn/GB/64107/66150/66152/66158/66207/4476523.html?ol4f 朱镕基会见来访的朝鲜最高人民会议常任委员会委员长金永南]</ref>
*2005年10月29日，金永南在平壤百花园国宾馆会见中共中央总书记、国家主席[[胡锦涛|胡锦涛]]。<ref>[http://www.gov.cn/ldhd/2005-10/30/content_86637.htm 胡锦涛会见朝鲜最高人民会议常任委员会委员长]</ref>
*2007年6月7日，金永南在平壤会见了中共中央委员、甘肃省委书记[[陆浩|陆浩]]率领的[[中国共产党|中国共产党]]友好代表团。<ref>[http://www.mfa.gov.cn/ce/cekp/chn/zcgx/gchf/sbgx/t327510.htm 金永南会见中共友好代表团]</ref>
*2008年6月17日，金永南在平壤会见中共中央政治局常委、国家副主席[[习近平|习近平]]<ref>[http://www.npc.gov.cn/npc/xinwen/szyw/zhbd/2008-06/18/content_1433833.htm 习近平分别会见朝鲜最高人民会议常任委员会委员长金永南和副委员长杨亨燮]</ref>
*2008年8月，赴北京参加[[第29届北京奥运会|第29届北京奥运会]]开幕式及相关活动。
*2009年10月5日，金永南在平壤会见中共中央政治局常委、国务院总理[[温家宝|温家宝]]。<ref>[http://www.mfa.gov.cn/ce/cekp/chn/zcgx/gchf/09sbgx/t618333.htm 温家宝会见朝鲜最高人民会议常任委员会委员长]</ref>
*2011年2月14日，金永南在平壤会见中国国务委员、公安部部长[[孟建柱|孟建柱]]。<ref>[http://news.sina.com.cn/c/2011-02-14/205521953044.shtml 孟建柱会见朝鲜最高人民会议常任委员会委员长]</ref>
*2012年5月11日，金永南访问[[新加坡|新加坡]]会见总统[[陈庆炎|陈庆炎]]，双方就[[朝鲜半岛|朝鲜半岛]]局势及新朝双边关系等共同关心的问题交换了意见。<ref>[http://news.ifeng.com/world/detail_2012_05/12/14484111_0.shtml 新加坡总统会见金永南 为金正恩后朝最高官员出访]</ref>
*2013年4月15日，金永南致电[[尼古拉斯·马杜罗|尼古拉斯·马杜罗]]，祝贺他当选[[委内瑞拉总统|委内瑞拉总统]]。<ref>[http://www.chinadaily.com.cn/hqzx/2013-04/17/content_16414480.htm 朝鲜最高人民会议委员长电贺马杜罗当选委内瑞拉总统]</ref>
*2014年，对非洲多国展开访问。2014年10月28日，金永南在[[刚果（布）|刚果（布）]]黑角市会见[[刚果共和国总统|刚果共和国总统]]、[[刚果劳动党|刚果劳动党]]中央委员会主席[[德尼·萨苏-恩格索|德尼·萨苏-恩格索]]。双方通报各自国家的情况，就当前国际形势交换看法，并就进一步发展朝刚两国友好关系、加强[[不结盟运动|不结盟运动]]等共同关心的问题进行了探讨。<ref>[http://www.xinhuanet.com/photo/2014-10/31/c_127162152.htm 金永南会见刚果（布）总统]</ref>
*2014年，赴俄罗斯[[索契|索契]]参加[[第22届冬奥会|第22届冬奥会]]开幕式，期间与俄罗斯总统[[普京|普京]]、联合国秘书长[[潘基文|潘基文]]等人会面。
* 2015年4月，参加了在[[印度尼西亚|印度尼西亚]]举办的[[万隆会议|万隆会议]]，会见了[[伊朗|伊朗]]、[[印度尼西亚|印度尼西亚]]、[[津巴布韦|津巴布韦]]、[[越南|越南]]等国的首脑。
*2015年5月8日，金永南访问[[俄罗斯|俄罗斯]]，在[[克里姆林宫|克里姆林宫]]会见总统[[普京|普京]]，并向普京转交了最高领导人[[金正恩|金正恩]]的亲笔信。<ref>[http://news.sina.com.cn/w/p/2015-05-11/210231819007.shtml 朝鲜高官金永南向普京转交金正恩亲笔信]</ref>5月9日，代替金正恩出席俄罗斯卫国战争胜利日大阅兵。

*2015年10月10日，金永南在平壤会见中共中央政治局常委、中央书记处书记[[刘云山|刘云山]]。<ref>[http://www.gov.cn/xinwen/2015-10/10/content_2944889.htm 刘云山会见朝鲜最高人民会议委员长金永南]</ref>
*2018年2月9日，金永南率团在[[2018年冬季奥林匹克运动会|平昌冬季奥林匹克运动会]]期间访问[[韩国|韩国]]。韩国总统府[[青瓦台|青瓦台]]表示欢迎，认为这显示朝鲜改善[[朝韩关系|朝韩关系]]的决心与诚意。<ref>[http://dz.xdkb.net/html/2018-02/06/content_481411.htm  朝鲜宣布金永南率团访韩]</ref>他此行身份为朝鲜高级别代表团团长，目的是出席[[2018年冬季奧林匹克運動會開幕式|第23届冬奥会开幕式]]。<ref>[http://www.chinatimes.com/cn/realtimenews/20180205001621-260408 北韩名义元首金永南 9日率高层代表团访南韩 [[中时电子报|中时电子报]]]</ref>

== 国家元首身份争议 ==

金永南作为最高人民会议常任委员会委员长，对外代表国家<ref>{{cite paper|title=조선민주주의인민공화국 사회주의헌법 (제10호)|url=https://ko.wikisource.org/wiki/조선민주주의인민공화국_사회주의헌법_(제10호)}}</ref>。

根据1998年9月5日实行的朝鲜民主主义人民共和国社会主义宪法（第7号）<ref>{{cite paper|title=조선민주주의인민공화국 사회주의헌법 (제7호)|url=https://ko.wikisource.org/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98%ED%97%8C%EB%B2%95_(%EC%A0%9C7%ED%98%B8)}}</ref>，第100条的规定：“国防委员会是国家主权的最高军事指导机关，同时也是完全承担国防管理职能的机关。”第111条规定：“最高人民会议常务委员会委员长负责常任委员会的事业和组织指导。最高人民会议常任委员会委员长是国家的代表，负责接受外国使节的国书和传召。”。2009年4月9日，朝鲜进行了修宪，即朝鲜民主主义人民共和国社会主义宪法（第8号）<ref>{{cite paper|title=조선민주주의인민공화국 사회주의헌법 (제8호)|url=https://ko.wikisource.org/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98%ED%97%8C%EB%B2%95_(%EC%A0%9C8%ED%98%B8)}}</ref>，第117条照搬了原宪法111条的叙述，而第100条有关国防委员会的内容，更改为：“朝鲜民主主义人民共和国国防委员会委员长是朝鲜民主主义人民共和国的最高领导者。”。2010年4月9日，颁布施行朝鲜民主主义人民共和国社会主义宪法（第9号）<ref>{{cite paper|title=조선민주주의인민공화국 사회주의헌법 (제9호)|url=https://ko.wikisource.org/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98%ED%97%8C%EB%B2%95_(%EC%A0%9C9%ED%98%B8)}}</ref>，有关内容沿用了第8号宪法的描述。金正日于2011年12月17日[[金正日之死|去世]]。而后在2012年4月13日颁布施行朝鲜民主主义人民共和国社会主义宪法（第10号）<ref>{{cite paper|title=조선민주주의인민공화국 사회주의헌법 (제10호)|url=https://ko.wikisource.org/wiki/%EC%A1%B0%EC%84%A0%EB%AF%BC%EC%A3%BC%EC%A3%BC%EC%9D%98%EC%9D%B8%EB%AF%BC%EA%B3%B5%ED%99%94%EA%B5%AD_%EC%82%AC%ED%9A%8C%EC%A3%BC%EC%9D%98%ED%97%8C%EB%B2%95_(%EC%A0%9C10%ED%98%B8)}}</ref>。新宪法第100条更改为：“朝鲜民主主义人民共和国国防委员会第一委员长是朝鲜民主主义人民共和国的最高领导者。”。2016年6月30日，朝鲜官方网络媒体[[我们民族之间|我们民族之间]]，引述了党报[[劳动新闻|劳动新闻]]有关宪法补充和修正的报告。<ref>{{cite paper|title=조선민주주의인민공화국 사회주의헌법을 수정보충함에 대한 보고|url=http://www.uriminzokkiri.com/index.php?ptype=igisa2&no=113290}}</ref>根据修改后的宪法，国防委员会改为国务委员会，第100条规定：“朝鲜民主主义人民共和国国务委员会委员长是朝鲜民主主义人民共和国的最高领导者。”目前，朝鲜的法定[[朝鲜民主主义人民共和国国家元首列表|最高领导者]]是[[金正恩|金正恩]]。

==中韩关系==
与[[大韩民国|大韩民国]]的交往和交涉，在朝鲜的对外政策中当然也是至关重要的。作为对外的国家代表，金永南在这方面也承担着重大职责和任务。在国际场合，他往往就是代表朝鲜发声的一位重要人物。

==={{PRC}}===
金永南先後多次訪華，包括鞏固[[中朝關係|中朝關係]]、尋求中國協助等，是與中國關係密切領導層。2008年[[2008年夏季奧林匹克運動會|北京奧運]]時，他亦代表北韓到中國出席开幕式，並在到訪期間與時任[[中共中央總書記|中共中央總書記]]兼[[中國國家主席|中國國家主席]][[胡锦涛|胡锦涛]]和[[中华人民共和国副主席|副主席]][[习近平|习近平]]等[[中国领导人|中国领导人]]进行了会见。2013年8月1日，在出席[[伊朗|伊朗]]候任总统[[鲁哈尼|鲁哈尼]]的就职典礼前秘密到訪北京，但行程沒有被官方公佈。

==={{ROK}}===
金永南在[[卢武铉|卢武铉]]在任[[大韓民國總統|總統]]時對[[大韓民國|大韓民國]]採取友善的態度。2007年時任韩国总统卢武铉访问北韓，金永南在卢武铉與朝鮮勞動黨總書記[[金正日|金正日]]會面前设宴招待。不过，自[[李明博|李明博]]政府上台后，[[朝韓關係|朝韓關係]]空前紧张，金永南對南韓的態度有所改變。2008年4月14日，金永南曾在纪念[[金日成|金日成]]诞辰96周年的中央报告大会上表示，实现“[[朝鮮半島統一|祖國統一]]”是[[朝鲜民族|朝鲜民族]]最重要的任务，也是不能再继续拖延的紧迫任务。他同时表示，朝鲜将沿着2000年朝韩首脑发表的《[[北南共同宣言|北南共同宣言]]》所开辟的道路，高举“民族共助”的旗帜，坚决粉碎韩国保守执政势力的反朝、对抗阴谋，开创自主统一、和平繁荣的新时代。在2008年8月出席[[2008年夏季奧林匹克運動會开幕式|北京奧運會开幕式]]之前，[[胡锦涛|胡锦涛]]设午宴招待訪华元首。李明博事先让助理为其精心设计了与金永南打招呼的问候用语、面部表情以及握手寒暄方式，但据说“金永南拒绝与李明博同席”，虽然最后金永南和李明博依然同桌进宴，但据《朝鲜日报》报道，“两人一言不发”。金永南会见[[德國|德國]]议员时指责韩国破坏了10年来的和解努力。
