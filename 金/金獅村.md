'''金獅村'''，是[[中華民國|中華民國]][[嘉義縣|嘉義縣]][[竹崎鄉|竹崎鄉]]轄下的行政區。位於嘉義縣東北方。面積約7.879764平方公里,行政區包含12個鄰、190戶、人口數487人。

==地理==
嘉義縣竹崎鄉金獅村主要地形由山地構成加上少數丘陵，海拔200~1200公尺，位於牛稠溪上游。

早期生產孟宗竹、桂竹、李子、竹造紙為主；現今生產檳榔、高山茶、甜柿等。以閩南人為主，主要從事務農業，信仰中心主要為信奉北極玄天上帝<ref name="竹崎鄉金獅村">{{cite web |author1=嘉義縣竹崎鄉公所 |title=金獅鄉 |url=https://chuchi.cyhg.gov.tw/cp.aspx?n=D541E9280B5F1692&s=731990F806254F77 |accessdate=2018-10-21}}</ref>。

==聚落歷史==
金獅村大多由各姓祖先於清朝時期從中國廣東省移民而來。由不同姓氏(朱、陳、林、劉、汪等等為主)所構成不同區域，集合成現今的金獅村<ref name="竹崎鄉金獅村歷史">{{cite web |author1=地名資訊服務網 |title=金獅村歷史 |url=http://gn.moi.gov.tw/geonames/GNMap/map_Admin/map/MapMain.aspx# |accessdate=2018-10-21}}</ref>。

==教育==
金獅村所屬學區為竹崎鄉龍山國小；所屬學區國中為竹崎高中國中部。

西元1950年設立龍山國小分班，西元1955年正式成立學校，所處環境自然秀麗<ref name="竹崎鄉金獅村教育">{{cite web |author1=龍山國小 |title=金獅村教育 |url=http://www2.lsps.cyc.edu.tw/modules/tad_book3/index.php?op=list_docs&tbsn=1 |accessdate=2018-10-21}}</ref>。竹崎高中國中部創立於西元1957年，原名為嘉義縣立竹崎初級中學，後改為嘉義縣立竹崎國民中學，西元2005年設立高中部<ref name="竹崎縣金獅國中教育">{{cite web |author1=竹崎高中 |title=金獅村教育 |url=http://www.ccjh.cyc.edu.tw/bin/home.php |accessdate=2018-10-21}}</ref>。

==文化遺產==
金獅村廣福宮位於台灣嘉義縣竹崎鄉，創立於1948年，始於康熙間，由中國廣東省之先民渡台時所攜帶的玄天上帝香火。道光時期，信徒增多，故建造小屋供奉。日治時期屢次改建，直到1979年重建至今<ref name="竹崎鄉金獅村文化">{{cite web |author1=地理資訊科學研究中心 |title=金獅村文化資源 |url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/juchi/1014019-GFG |accessdate=2018-10-21}}</ref>。

==景點==
擁有海拔800公尺的茶園景觀，或是遠眺文峰村的觀音瀑布。出水坑步道，早期由青石階砌成古汗道相疊，沿途孟宗竹和桂竹茂盛<ref>{{cite web |author1=阿里山國家公園 |title=金獅村景點 |url=https://www.ali-nsa.net/user/Article.aspx?Lang=1&SNo=04003717 |accessdate=2018-10-21 |ref=竹崎鄉金獅村景點}}</ref>。

==其他特色==
居民現主要種植十月龍眼、檳榔、高山茶、甜柿、苦茶樹、石蓮花、蘭花，農產類豐富多元。第二級產業生產龍眼乾<ref name="竹崎鄉金獅村" />。
