'''金華公園'''，又稱'''大安69號公園'''，位於[[台灣|台灣]][[台北市|台北市]][[大安區_(臺北市)|大安區]][[龍安里|龍安里]]，鄰近[[臺北市大安區新生國民小學|新生國小]]、[[臺北市立金華國民中學|金華國中]]，為鄰里社區公園，除基本公園設施外，另有[[鳳凰木|鳳凰木]]、[[小葉欖仁|小葉欖仁]]等多種植物可供教育及欣賞，是鄰近之社區居民遊憩的場所，並附有地下停車場。

== 範圍 ==
* 東面：[[青田街|青田街]]
* 西面：金華街164巷
* 南面：金華街172巷
* 北面：[[金華街|金華街]]

== 沿革 ==
金華公園於1998年完工，1999年7月由[[台北市政府工務局公園路燈工程管理處|台北市政府工務局公園路燈工程管理處]]移撥大安區公所接管，目前的認養單位為[[台北城市大學|台北城市大學]]與龍安里里長。

== 主要設施 ==
* 涼亭
* 廣場
* 兒童遊樂設施
* 花架
* 步道
* 地下停車場
