'''金巴利'''（{{lang-it|Campari}}）是一款[[力嬌酒|力嬌酒]]，起源於[[意大利|意大利]]，根据出售国家不同，起酒精浓度有多个版本，包括20.5%、21%、25%及28%。金巴利使用多种草药和水果酿制，包括{{tsl|en|Citrus myrtifolia|厚叶橙|厚叶橙}}、{{tsl|en|Cascarilla|灌木巴豆|灌木巴豆}}，其鲜红的颜色是它独特的标志，味道很苦。

== 历史 ==
金巴利的历史可以追溯到1860年，意大利人，[[诺瓦拉|诺瓦拉]]的Gaspare Campari发明了这种酒。最初的金巴利靓丽的红色来自于[[胭脂红|胭脂红]]——一种利用[[胭脂虫|胭脂虫]]制作的天然染色剂。

1904年，金巴利的第一家生产工厂开在[[米兰|米兰]]附近的[[塞斯托-聖喬凡尼|塞斯托-聖喬凡尼]]。当时，公司要求出售金巴利的酒吧都推广金巴利品牌。在Davide Campari——Gaspare的儿子的引领下，公司开始出口饮料，首先是[[法国|法国]][[尼斯|尼斯]]，继而畅销海外。金巴利品牌现在远销全球190多个国家。在2006年，金巴利集团停止使用胭脂红着色。<ref>[http://www.raysahelian.com/carmine.html Carmine coloring risk]</ref>

== 金巴利苏打水 ==
[[Image:Campari_Soda.jpg|thumb]]
在意大利市场，还有一种金巴利混合苏打水（10％的酒精度数）的饮料出售，称作金巴利苏打水。金巴利苏打水采用一种独特的瓶装，是由{{tsl|en|Fortunato Depero|福尔图纳托·德佩罗|福尔图纳托·德佩罗}}在1932年设计的。据说，金巴利一直是许多其他苦甜类饮料，如[[马耳他|马耳他]]1952年的Kinnie发明的灵感之源。

== 饮用 ==
[[Image:Negroni_served_in_Vancouver_BC.jpg|thumb]]
金巴利最初作为[[餐前酒和餐后酒|餐前酒和餐后酒]]饮用，但是鉴于其鲜亮的色彩，现在越来越多的被用于调制多种鸡尾酒。从简单的与苏打水、葡萄酒或橙汁混合饮用，到调制大名鼎鼎的Negroni鸡尾酒、Americano鸡尾酒和Garibaldi鸡尾酒。
