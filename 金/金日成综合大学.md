'''金日成綜合大學'''（{{lang-ko|'''김일성종합대학'''}}），位於[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]首都[[平壤市|平壤市]]東北部[[大城區|大城區]]的[[龍南山|龍南山]]山麓，1946年10月1日建校，是[[朝鮮半島|朝鮮半島]]北部的第一所正規[[大學|大學]]，以「幹部培養基地」、「[[主體思想|主體思想]]教育与[[科學|科學]]的最高學府」自居。金日成綜合大學改變了[[朝鮮半島|朝鮮半島]]北部沒有大學的[[歷史|歷史]]（[[朝鮮半島|朝鮮半島]]上最早的正規大學是於[[朝鮮日佔時期|日本殖民時期]]在今[[韩国|韩国]][[首爾|首爾]]設立的[[京城帝國大學|京城帝國大學]]，現為[[首爾大學|首爾大學]]的一部分）。

2010年，该校擁有7個[[專科大學|專科大學]]（[[學院|學院]]）、15個[[學系|學系]]、約50多個專業學科、約2000多名教職員和約10000多名[[學生|學生]]（全日制、夜間或远程教学），設置了基礎科學中心、[[社會科學|社會科學]]中心、信息中心、[[催化劑|催化劑]][[研究所|研究所]]、[[原子能|原子能]]研究所、科學實驗器具研究所、分析研究所、[[電子材料|電子材料]]研究所等10多個研究中心和研究所。

2019年起，該校校長是[[崔相建|崔相建]]。

金正日多次前往此学校，是金正日访问次数最多的大學。

== 歷史 ==
* 1945年11月，[[金日成|金日成]]提議建立幹部培養基地，發起組織建立綜合大學籌備委員會，並積極推行這項工作。
* 1946年7月，[[北朝鮮人民委員會|朝鮮人民臨時委員會]]以金日成之名，將綜合大學命名為「金日成綜合大學」。
* 1946年10月1日，金日成綜合大學宣告成立，初期只設有7個學系、60多位[[教師|教師]]及1,500位[[學生|學生]]。 
* 1948年，從金日成綜合大學分出了平壤工業大學（現為[[金策工業綜合大學|金策工業綜合大學]]）、[[平壤醫科大學|平壤醫科大學]]和[[元山農業大學|元山農業大學]]。
* 2010年，新建了財政專科大學，其後又合併了[[平壤醫科大學|平壤醫科大學]]、[[平壤農業大學|平壤農業大學]]及[[桂應泰農業大學|桂應泰農業大學]]。

== 校园设施 ==
以豎立在龍南山頂峰的金日成主席銅像為中心，有三個專科大學、13個系、10多個研究中心和研究所、研究院及[[博士|博士]]院。[[教師|教師]]、[[研究員|研究員]]中很多人獲得[[學位|學位]]。大學有主樓和1、2號教學樓，有200萬部圖書的[[科學|科學]][[圖書館|圖書館]]、[[自然|自然]][[博物館|博物館]]、[[出版社|出版社]]、印刷廠、實習工廠、大學副業[[農場|農場]]、[[體育館|體育館]]、[[宿舍|宿舍]]、[[海洋|海洋]]實習所、大學[[醫院|醫院]]、靜養所、綜合服務設施等。該校共有200個教室、180多個實驗室、200多個教研室和研究室。

== 著名教職員 ==
電子材料研究所所長：[[金相玉|金相玉]]教授，勞動英雄、人民科學家、院士、博士。他曾研發出[[壓電瓷|壓電瓷]]材料。<ref>[http://www.setn.com/News.aspx?NewsID=25333 成功研發壓電瓷材料，金所長成為國家推崇的人民科學家]</ref>

物理系系主任：[[徐相國|徐相國]]教授，[[朝鮮國防委員會|朝鮮國防委員會]]「極密委員」（負責朝鮮極密工作的專職委員）、勞動英雄、博士，曾獲「[[金日成獎|金日成獎]]」。他是朝鮮的「[[核彈|核彈]]之父」，專注[[核武|核武]]研究。

校長：[[黃長燁|黃長燁]]（[[主體思想|主體思想]]理論系統化人物，1997年變節前往韓國）
