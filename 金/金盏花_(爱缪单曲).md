《'''金盏花'''》（或译'''万寿菊'''，{{jpn|j=マリーゴールド|rm=Marīgōrudo}}，{{lang-en|Marigold}}）是[[日本|日本]][[创作歌手|创作歌手]][[爱缪|爱缪]]的第5张[[单曲|单曲]]，2018年8月8日发售。

==简介==
距离前作《{{link-ja|假如在滿月之夜|満月の夜なら}}》发售相隔不到4个月，本作《金盏花》是描述在多变的夏季风光中男女恋爱的场景。B面《为了你》则是从女性视角对男性一心爱慕的情歌<ref name="q">{{cite web|script-title=ja:あいみょん早くも新作「マリーゴールド」発売、Zepp含むワンマンツアー開催（動画あり） |url=https://natalie.mu/music/news/286270 |website=音楽ナタリー |accessdate=2019-01-09 |language=ja}}</ref>。

在6月12日宣布新单曲的同时公开了约50秒的副歌试听版本<ref name="q"/>。

本作的[[MV|MV]]由{{Link-ja|山田智和|山田智和}}导演，全程在[[上海|上海]]拍摄。MV的场景包括踩着[[滑板|滑板]]在街头滑行、以及独自一人在房间内唱歌的画面，意图表现“悲伤与爱”这一主题。这是爱缪第一次在国外拍摄MV<ref>{{cite web|script-title=ja:あいみょん新MVは上海で“切なさと愛しさ”表現、ツアー追加公演はBLITZで（動画あり / コメントあり） |url=https://natalie.mu/music/news/291528 |website=音楽ナタリー |accessdate=2019-01-09 |language=ja}}</ref>。

歌曲发售后爱缪在《{{link-ja|SUKKIRI|スッキリ (テレビ番組)}}》<ref>{{cite web |script-title=ja:あいみょん「スッキリ」に生出演、新曲「マリーゴールド」披露 |url=https://natalie.mu/music/news/293931 |website=音楽ナタリー |accessdate=2019-01-09}}</ref>、《[[CDTV|CDTV]]》<ref>{{cite web |script-title=ja:次回「CDTV」にPerfume、サンボマスター、あいみょん登場 |url=https://natalie.mu/music/news/294740 |website=音楽ナタリー |accessdate=2019-01-09 |language=ja}}</ref>、《[[MUSIC_STATION|MUSIC STATION]]》<ref>{{cite web |script-title=ja:次週「Mステ」にあいみょん、家入レオ、欅坂46、ジャニーズWEST、Perfume、ゆず |url=https://natalie.mu/music/news/294966 |website=音楽ナタリー |accessdate=2019-01-09 |language=ja}}</ref>等电视节目中演唱了这首歌。而在年末的《[[第69回NHK紅白歌合戰|NHK紅白歌合戰]]》中本作成为爱缪首次出场所表演的歌曲<ref>{{cite web |script-title=ja:『第69回紅白歌合戦』曲目発表 嵐はスペシャルメドレー 福山雅治&Perfumeは中継出演【全曲掲載】 |url=https://www.oricon.co.jp/news/2126032/full/ |website=ORICON NEWS |accessdate=2019-01-09}}</ref>。

==反响==
《金盏花》在配信开始后首周占据[[Oricon公信榜|Oricon公信榜]]单曲下载周榜第12位，第4周随着实体CD发售，以及在电视节目中演出一举上升到第5位<ref name="w">{{cite web |script-title=ja:あいみょん“お茶の間への浸透”で第2のステージへ　新曲「マリーゴールド」が自己最高初動を記録 |url=https://www.oricon.co.jp/confidence/special/51612/ |website=ORICON NEWS |accessdate=2019-01-09}}</ref>。同时Oricon单曲周榜也拿到自身最高初动3536张<ref name="w" />。

随着爱缪在年末电视节目《[[MUSIC_STATION_SUPER_LIVE|MUSIC STATION SUPER LIVE]]》和《[[NHK红白歌合战|NHK红白歌合战]]》中演唱了《金盏花》之后，本作的销量及排名再度急速上升。在2019年1月14日付的Billboard JAPAN Hot 100周榜中跃至第3位。在Oricon单曲下载周榜中也上升至第3位，流媒体周榜更超越此前连夺三冠的《[[今夜就这样|今夜就这样]]》，拿到首个周冠<ref>{{cite web |script-title=ja:あいみょん、紅白披露曲が初の1位 4週連続1位2位独占&4作同時TOP10入り |url=https://www.oricon.co.jp/news/2127064/full/ |website=ORICON NEWS |accessdate=2019-01-09}}</ref>。
