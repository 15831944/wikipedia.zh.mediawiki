'''金滅遼之戰'''發生於1120年至1125年間，始於[[金朝|金朝]]攻擊黃龍府，終於1125年金佔領遼的[[燕雲|燕雲]]之地，為之後南侵[[北宋|北宋]]打下基礎。

== 起因 ==
[[金朝|金朝]]建國之前，[[女真人|女真人]]一直為[[遼朝|遼朝]]契丹人所壓迫。後來[[完顏阿骨打|完顏阿骨打]]成功於1115年統一[[女真|女真]]各部，建立能與遼抗衡的[[金朝|金朝]]。金朝屢屢戰勝遼朝，因此與南方的宋朝聯盟，聯合[[宋朝|宋朝]]滅掉[[遼朝|遼朝]]。[[宋徽宗|宋徽宗]]其好大喜功的心理，在急功近利下，立即同意聯金滅遼。

== 經過 ==
金朝先攻下遼朝的[[黃龍府|黃龍府]]，於1120年再攻打[[臨潢府|臨潢府]]，於1121年攻佔[[遼陽府|遼陽府]]，於1122年再攻下[[大定府_(辽朝)|大定府]]，於1124年攻陷[[析津府|析津府]]及[[大同府|大同府]]，成功消滅遼國<ref>地球出版社編輯部. 中國文明史 宋遼金時期. 地球出版社. [[ISBN|ISBN]] [[957-714-048-3|957-714-048-3]]</ref>。

== 影響 ==
{{main|靖康之禍}}
金朝在攻打遼朝時，知道宋朝皇室及軍隊的軟弱及腐敗無能，便於翌年進攻宋朝，攻佔宋朝首都[[汴京|汴京]]，侵佔宋朝黃河以北的領土，徽欽二帝被俘，及後宋徽宗九子趙構在[[南京應天府|南京應天府]]稱帝，是為[[宋高宗|宋高宗]]，不久[[建炎南渡|建炎南渡]]，宋朝進入[[南宋|南宋]]時期。
