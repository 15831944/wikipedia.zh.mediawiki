
金門雄峙閩海、汪洋四環，舊名浯洲，又有仙洲、浯江、浯島諸稱<ref>[https://www.kinmen.gov.tw/cp.aspx?n=22F39676C32D22F8#a1 認識金門]金門縣政府© 2016</ref>。[[金門縣|金門]]自古屬[[福建|福建]][[泉州府|泉州府]][[同安縣|同安縣]]所轄，民國四年始創縣治<ref>[http://www.kinmen.gov.tw/Layout/main_ch/ArtHtml_Show.aspx?ID=4facbade-55b0-4af8-a41c-b41d35d9e436&path=10931 縣治沿革], 金門縣政府</ref>。金門一地在同安縣的歷史有著重要地位，有「無金不成銀」、「無金不成銅」{{efn|1=兩句俗諺中的「金」指金門，「銀」、「銅」都是指[[同安縣|同安縣]]，銀、銅字典故都與同安縣城有關。<ref>[http://www.xmnn.cn/tszk/wyb/200707/t20070730_271873.htm 颜立水论金：厦富金贵“两门”对开才合璧 ] {{webarchive|url=https://web.archive.org/web/20071123133756/http://www.xmnn.cn/tszk/wyb/200707/t20070730_271873.htm |date=2007-11-23 }}，廈門商報（2007-07-30）</ref>}}的俗諺。金門原名浯州，因其地形格外的重要，顯得「固若金湯，雄鎮海門」，金門這個地名由此而來。晉朝[[五胡亂華|五胡亂華]]時因百姓不甘受外族統治紛紛南遷，成為最早到金門的人民。唐朝時金門便有牧馬區當時跟隨馬監來金門的有將佐衛傑、李俊、王、錢，民國四年，金門設縣，民國26年至34年[[抗日戰爭|抗日戰爭]]期間被日本攻佔，民國38年[[中華民國政府|中華民國政府]]退守台灣，以金門作為反共的第一要塞。

== 晉代至唐代時期 ==
[[漢族人|漢族人]]對金門的開發依據可考的史料始於[[晉代|晉代]]。根據[[清代|清代]]《金門志》的記載，晉代共有蘇、陳、吳、蔡、呂、顏六姓家族因躲避戰禍移居金門。

貞元十九年（803年），[[唐代|唐代]]朝廷在[[泉州|泉州]]設置五個牧牛場，'''浯洲'''為其中之一，[[陳淵|陳淵]]任牧馬監， 閩觀察使[[柳冕|柳冕]]，在烈嶼島上[[紅石山|紅石山]]上設有[[牧馬寨|牧馬寨]]做為[[牧馬|牧馬]]之用，此為金門島上設置行政機構之始。[[蔡姓|蔡]]、[[許姓|許]]、[[翁姓|翁]]、[[李姓|李]]、[[張姓|張]]、[[黃姓|黃]]、[[王姓|王]]、[[呂姓|呂]]、[[劉姓|劉]]、[[洪姓|洪]]、[[林姓|林]]、[[蕭姓|蕭]]十二姓隨陳淵入島開墾，陳淵因此而被尊為「開浯恩主」。

== 宋代至明代初時期 ==
金門（舊稱浯洲）因產鹽， 在宋元兩代烈嶼島上就已設[[鹽場|鹽場]]。歷經[[五代|五代]]閩王[[王審知|王審知]]及元明清等朝開發後，造就金門東半島上之金沙灣周圍鹽埕林立，[[元朝|元朝]]統治時（1343年-1368年），中央為求實質統治，遂於浯洲鳳翔里十七都後學村（今沙美），設置[[浯洲鹽場司|浯洲鹽場司]]官職從七品官，在今金沙國中至東埔及榮光新村一帶）及[[浯洲書院|浯洲書院]]（今沙美菜市場），沙美因處[[金沙灣|金沙灣]]與[[汶水溪|汶水溪]]及[[金沙溪|金沙溪]]交匯處，在元代，係為金門地區最高行政機關浯洲鹽場司與浯洲書院之舊址（元朝浯洲鹽場司馬闕司令興建）。過往的金沙地區更是金門地區居住人口與[[風獅爺|風獅爺]]最為稠密的地方（金門全島共64尊風獅爺，金沙鎮則高達39尊、沙美有3尊）。元朝統治時，中國沿海各地與國外交流密集，當時的沙美（後學村），因位處金沙灣、浯洲鹽場司、浯洲書院（明代為金山書院）、官鎮埕、永安埕、浦頭埕、沙美埕等鹽埕邊緣，除了島民、鹽工與書生眾多及船運便利之外，區域內更是政商雲集。[[明朝|明朝]]統治時金門鹽產業到達顛峰，因為鹽場多集中於後學村（今[[沙美|沙美]]）附近，使後學街（今[[沙美老街|沙美老街]]）成為當時各宗族間及沙美與金門、中國、南洋等商業、產業、農漁產品買賣集散地。

== 明代末至清代時期 ==
洪武二十年（1387年），明太祖[[朱元璋|朱元璋]]令[[江夏|江夏]]侯[[周德興|周德興]][[經略|經略]][[福建|福建]]沿海，共設五衛十二所。[[金門守禦千戶所城|金門守禦千戶所]]為十二所之一，明[[兵部|兵部]]稱呼金門是「金門所」，下轄[[峰上巡檢司城|峰上]]、[[官澳巡檢司城|官澳]]、[[田浦巡檢司城|田浦]]、[[陳坑巡檢司城|陳坑]]四個[[巡檢司|巡檢司]]，後又增設[[烈嶼巡檢司城|烈嶼巡檢司]]。因金門固守福建東南海口，取「固若金湯，雄鎮海門」之意而得名金門。

明朝末年，[[鄭芝龍|鄭芝龍]]稱雄[[閩|閩]][[臺|臺]]海域，以金門、廈門為基地，樹旗招兵<ref>中央研究院，《漢籍電子文獻》，《漳州府志選錄》，第10頁</ref>。1645年（[[清|清]][[順治|順治]]2年）鄭芝龍降清後，先派其弟[[鄭鴻逵|鄭鴻逵]]屯金門。不久，金、廈為擁護[[魯王|魯王]][[朱以海|朱以海]]的鄭聯、鄭彩兄弟所據。 [[南明|南明]][[鄭成功|鄭成功]]於1646年在烈嶼島上[[吳山|吳山]]會文武群臣，曾派[[林習山|林習山]]駐守烈嶼，以此島做為'''「[[反清復明|反清復明]]」'''根據地之一。1650年（順治7年；南明[[永曆_(南明)|永曆]]4年），鄭成功驅逐鄭氏兄弟，金廈成為其抗清基地之一。 不久明鄭又重返金、廈兩島，至康熙平臺後，金門航運地位逐漸為廈門所奪。清代中後期，因生活困難，金門人相率下[[南洋|南洋]]或東渡[[日本|日本]]，成為海外僑民<ref>《金門縣志：卷首》，第154－156頁</ref>。 。1651年8月，南明魯王朱以海從[[舟山|舟山]]逃往金門，並於1662年11月病故於金門，死後安葬在金門城東之青山岩，為[[魯王墓|魯王墓]]。 1663年（[[康熙|康熙]]2年；永曆17年）11月，[[耿繼茂|耿繼茂]]、[[李率泰|李率泰]]督清軍及明鄭降兵攻取金門和廈門，明鄭[[周全斌|周全斌]]部不敵而退至[[東山島|東山]]。清軍登陸兩島後，男女被擄掠一空，最後「墮其城、焚其屋，棄其地而回。」<ref>夏琳，《閩海紀要》，臺灣銀行經濟研究室編，卷之上，第33頁</ref> 永曆十八年（1664年）清軍攻佔金門後，曾採取[[遷界|遷界]]措施，強制居民遷至海岸綫30[[華里|華-{里}-]]外，島上人煙無存。永曆二十八年（1674年）至三十三年（1679年），鄭氏復佔金門，並以此作爲對大陸進行軍事行動的前進基地。永曆三十四年（1680年）清軍二度攻佔金門，明鄭投降，金、廈遂為清朝所有。 永曆三十七年（1683年）清軍攻佔臺灣後實施復界，因遷界離開的居民陸續返回原籍。清代時為防[[倭寇|倭寇]]，立塞置[[汛|汛]]。

== 中華民國大陸時期 ==
民國元年（1912年）4月18日廈門軍政府參事會派代表向福建都督府請願獲准，廈門與金門一同從同安縣獨立為「思明縣」，但該年9月18日思明縣便升格為「思明府」，直到民國2年（1913年）3月30日又廢府改回「思明縣」{{r|郭}}。民國3年（1914年）福建巡按使[[許世英|許世英]]咨陳內政部，派[[左樹燮|左樹燮]]為金門籌辦設治委員，於同年7月奉准後將金門地區（大、小金門與大、小嶝等島）從思明縣獨立出來設置'''金門縣'''（歸廈門道管轄，乃二等縣治）{{r|郭}}。金門縣於民國4年（1915年）4月9日奉批令成立縣治，設縣知事，改縣丞署為縣公署{{r|郭}}。金門縣轄境包括大、小金門，大、小嶝島及周邊島嶼；分為六都、十保、一百六十六鄉。<ref>此時鄉約相當於今日之'''村'''的行政層級。</ref>民國22年（1933年）[[閩變|閩變]]發生後一度由[[中華共和國|中華共和國]]所據，劃為泉海省。1935年改為4區32聯保，民國25年（1936年）重新整編為3區12聯保。民國26年（1937年）[[抗日戰爭|抗日戰爭]]時期日軍佔領金門，金門縣政府[[大嶝金門縣政府舊址|遷至大嶝鄉上租借民房辦公]]。民國34年（1945年）[[第二次世界大戰|第二次世界大戰]]結束後，中華民國收復金門，設二鎮四鄉；民國35年（1946年）變更為二鎮二鄉。

== 中華民國軍管時期 ==
民國37年（1948年）12月10日，中華民國政府發佈全國戒嚴令，從此金門縣進入[[戒嚴|戒嚴]]。民國38年（1949年）年初，改為金城、沙美、烈嶼、大嶝四區區公所，同年10月，解放軍進佔福建沿海，中華民國國軍自[[大嶝街道|嶝島群島]]（大嶝、小嶝）撤退，[[古寧頭戰役|古寧頭戰役]]爆發，金門全境進入「'''軍管時期'''」，同年11月[[古寧頭戰役|古寧頭戰役]]結束後，金門縣政府建制被撤銷，中華民國國軍將金門全境劃分為金東、金西、烈嶼三個軍管區，各設民政處管轄地方行政，下轄[[城廂區|城廂區]]、[[金城區|金城區]]、[[金盤區|金盤區]]、[[滄湖區|滄湖區]]、[[碧湖區|碧湖區]]、[[金沙區|金沙區]]、[[烈嶼區|烈嶼區]]、[[古寧區|古寧區]]、[[瓊浦區|瓊浦區]]等九個區公所。民國39年（1950年）3月，金東、金西、烈嶼三個軍管區的民政處合併為「'''金門軍管區行政公署'''」。民國40年（1951年）7月分設金城、金寧、金湖、金沙、烈嶼五個行政區。同年12月增設金山行政區。

民國42年（1953年）2月結束軍管制度，改實施軍政一元化的'''戰地政務體制'''，金門縣政府恢復建制，原金門軍管區行政公署各行政區改為鄉鎮，成立三鎮三鄉，但地方行政的決策與指揮監督由金門防衛司令部下設「政務委員會」負責，以軍領政。民國43年（1954年）6月[[莆田縣|莆田縣]][[烏坵鄉|烏坵]]地區設鄉，指定暫由金門縣代管，此時金門縣共轄有[[金城鎮_(金門縣)|金城鎮]]、[[金沙鎮_(金門縣)|金沙鎮]]、[[金寧鄉|金寧鄉]]、[[金湖鎮|金湖鄉]]、[[金山鄉_(金門縣)|金山鄉]]、[[烈嶼鄉|烈嶼鄉]]、[[烏坵鄉|烏坵鄉]]等二鎮五鄉。民國48年（1959年）增設[[金瓊鄉|金瓊鄉]]，並改金湖為鎮。民國54年（1965年）9月撤置金山鄉及金瓊鄉。

民國47年（1958年）8月23日－10月5日，中華人民共和國聲稱「聲援中東人民的反侵略鬥爭」，於金門爆發[[金門砲戰|八七八砲戰]]。民國68年（1979年），[[美國|美國]]與[[中華人民共和國|中華人民共和國]]建交，中國發表《停止砲擊大、小金門等島嶼的聲明》，歷時21年的金門砲戰正式劃上句號。

中華民國國軍自1949年10月撤出金門縣'''大嶝郷'''後，中華人民共和國政府曾於1955年在大嶝郷設立[[金門縣人民政府|金門縣人民政府]],1970年12月併入[[同安縣|同安縣]]（今[[廈門市|廈門市]][[翔安區|翔安區]][[大嶝街道|大嶝街道]]），金門縣人民政府撤銷。

民國76年（1987年）3月7日至8日，[[烈嶼鄉|烈嶼鄉]]東崗漁蚵港外灘岸發生[[濫殺|濫殺]][[越南難民|越南難民]]的[[三七事件|三七事件]]（[[東崗慘案|東崗慘案]]）。<ref>{{cite journal | title=國軍屠殺越南難民的三七事件| author=管仁健 | url=http://mypaper.pchome.com.tw/kuan0416/post/1304293327 |work=《你不知道的台灣》|date=2008-03-07}}</ref>

民國80年（1991年）5月1日，《動員戡亂時期臨時條款》被廢除，同一天，金門縣因為仍屬戰地，進入臨時戒嚴。

== 中華民國正常化時期 ==
民國81年（1992年）11月7日，隨著[[兩岸|兩岸]]關係趨向和緩，加上各界力促金馬解嚴，[[中華民國國防部|國防部]]宣佈金門縣解除戒嚴，正式結束長達四十三年之久的戒嚴與實施三十六年的戰地政務實驗。

民國84年（1995年）10月18日，成立「[[金門國家公園|金門國家公園]]」，是中華民國第六九座[[國家公園|國家公園]]，是首座位於金馬的國家公園，也是首座以維護歷史文化資產、戰役紀念為主，並兼具自然資源的國家公園。

民國90年（2001年）2月3日，啟動金門-廈門航海線，開始「[[三小通|三小通]]」。

民國102年（2013年）5月23日，金門縣政府委託美國Aecom公司與日月辰工程顧問公司規劃「金門概念性總體規劃案」，預計花費近兩年時間完成<ref>{{cite news|url=http://www.cna.com.tw/News/aLOC/201305230308-1.aspx|title=總體規劃 金門成國際自由市|publisher=[[中央社|中央社]]|date=2013-05-23|language=zh-hant}}</ref>。

==風獅爺 ==
{{Main|胖獅爺}}
風獅爺自明、清兩代以來具有三、四百年以上的歷史是金門最具特色的人文景觀。由於金門地處孤懸海中，在明鄭大肆伐木造船，使得金門從綠野扶疏變為寸草不生的「風沙島」，以往金門冬天風害最為嚴重，生活極為艱苦，島民為求安居，乃在村落路口處，豎立胖獅爺，用以鎮風制煞，阻風沙護衛鄉土。設立胖獅爺之時，必先由乩童作法選定方位，宰殺白公雞，並將雞和血一起埋入士內，再將胖獅爺豎立其上，吸取白公雞之血氣，才能產生靈氣，以保護金門不受風沙之苦。而金門人民為感謝胖獅爺使他們可以免於風沙之苦，定每年的農曆八月十五日為胖獅爺的生日，村民在這一天會為胖獅爺披下紅巾，並將糕果、湯圓等貢品塞在其嘴裡，稱為「塞虎口」。
