'''金忠家族墓葬群'''是[[明代|明]][[永乐_(明朝)|永乐]]年间[[兵部尚书|兵部尚书]][[金忠_(明朝尚書)|金忠]]及其亲属的墓葬，位于[[中國|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[鄞州区|鄞州区]][[东钱湖镇|东钱湖镇]]境内，茶亭村后百步山东坡。被列入文物保护单位的墓葬包含金忠墓、白云公墓、赵太君墓和金宪一墓四座墓葬。金忠家族墓葬群于2010年被评选为宁波市[[第三次全国文物普查|第三次全国文物普查]]“百大新发现”<ref>{{Cite web|url=http://news.cnnb.com.cn/system/2010/10/22/006716948.shtml|title=宁波文物"百大新发现" 严康楙慈善建筑群上榜|accessdate=2018-03-25|author=梅子满、王玉琦、龚国荣|date=2010-10-22|last=|work=|publisher=中国宁波网}}</ref>，2011年列入第六批[[浙江省文物保护单位|浙江省文物保护单位]]<ref name="省保">{{Cite web|url=http://www.zj.gov.cn/art/2013/1/4/art_12880_71859.html|title=浙江省人民政府关于公布第六批省级文物保护单位和与现有省级文物保护单位合并项目及更名省级文物保护单位的通知|accessdate=2018-03-25|author=|date=2011-01-21|last=|work=|publisher=浙江省人民政府办公厅}}</ref>，2013年列入[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]<ref name="nbwb-info">{{cite web|url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=764&type=2|title=东钱湖墓葬群|accessdate=2018-03-25|author=东钱湖管委会|date=2006-02-10|publisher=宁波市文物保护管理所}}</ref>。

== 墓主生平 ==
{{main|金忠 (明朝尚書)}}
金忠生于[[浙江行省|浙江行省]][[宁波府|宁波府]][[鄞县|鄞县]]韩岭村（今属宁波市鄞州区东钱湖镇）<ref name="nbwb-info" />，曾替兄长补戍通州，因家贫，路费由[[袁珙|袁珙]]资助，后因擅长使用《[[易经|易经]]》算卦，由[[姚廣孝|姚廣孝]]推荐给燕王担任右长史。朱棣即帝位后，金忠先后升任[[工部右侍郎|工部右侍郎]]、[[东宫|东宫]]辅佐，后担任兵部尚书，卒于永乐十三年（1415年）。[[洪熙|洪熙]]元年，[[追贈|追贈]][[榮祿大夫|榮祿大夫]][[少師|少師]]，谥忠襄<ref>《[[明史|明史]]》（卷150）：金忠，鄞人。少讀書，善《易》卜。兄戍通州亡，忠補戍。貧不能行，相者袁珙資之……僧道衍稱於成祖……遂拜右長史，贊戎務，為謀臣矣。成祖稱帝，論佐命功，擢工部右侍郎，贊世子守北京。尋召還，進兵部尚書……明年四月卒……洪熙元年，追贈榮祿大夫少師，謚忠襄。</ref>。

金忠有一兄长名华，因金忠守通州有功，明成祖欲推恩授其官职，但金华推辞不接受，欲赠送其金綺，同样不接受，成祖称其为迂叟，乡人称其为“白云先生”<ref>《明史》（卷150）：忠有兄華，負誌節。忠守通州有功，欲推恩官之，辭不就。嘗召賜金綺，亦不受。成祖目為迂叟，放還……里中稱為「白雲先生」。</ref>。金忠之父为金宪一，因金忠而追赠[[榮祿大夫|榮祿大夫]]兵部尚书。金忠继配夫人为赵太君，曾为诰封太夫人，育有独子金达，曾任[[檢討|翰林检讨]]、[[給事中|給事中]]等职，以刚直敢言著称<ref name="nbwb-info" />。

== 墓葬形制 ==
[[File:Locations_of_Tombs_of_Jin_Zhong_Family.svg|thumb]]
金忠家族墓葬群位于百步山上，靠近钱湖十景中的“百步耸翠”<ref name="cnnb-discovery">{{Cite web|url=http://www.nbwb.net/info.aspx?Id=12168|title=明代兵部尚书家族墓葬群现身东钱湖畔|accessdate=2018-03-28|author=|date=2009-07-04|work=|publisher=宁波日报}}</ref>。墓葬群中的金忠墓、白云公墓、赵太君墓和金宪一墓形制较为相近，均由神道、平台、牌坊和墓室组成，墓前有牌坊，碑文两侧有牡丹、祥云等明代典型风格的装饰<ref name="nbwb-info" />。

金忠墓依山而建，面朝东北，长33.25米，宽11.3米。墓前牌坊已无坊，仅存立柱。墓碑刻有“明兵部尚书追赠荣禄忠襄金忠之墓”，碑文左右有勾藤图案，碑两侧有莲望柱。墓冢圆形，周边由条石砌筑，顶部压顶石已散落。距墓道北30米处有“明故政大夫兵部尚书兼詹事府詹事金公墓志碑”一通，碑文由[[翰林院|翰林学士]][[楊士奇|楊士奇]]撰写，赞颂金忠的功绩，碑额为篆书，碑文为正楷<ref name="nbwb-info" />，当地人称“尚书坟头”<ref>{{Cite web|url=http://www.zjww.gov.cn/news/2010-08-26/238620984.shtml|title=东钱湖墓葬群有望成为宁波的新“国保”|accessdate=2018-04-03|author=徐炯明、吴旭涛|date=2010-08-26|publisher=现代金报}}</ref>。

白云公墓距金忠墓100米，为金忠兄长金华的墓葬。墓葬长20米，宽7.5米，墓碑刻有“明赐白云迂叟墓”字样，两侧有莲望柱。墓冢圆形，周边为乱石砌，压顶石已散落<ref name="cnnb-discovery" />。

赵太君墓为金忠继配夫人赵太君与金忠独子金达的合葬墓，为宁波地区为数不多的母子合葬墓<ref name="cnnb-discovery" />。墓葬位于金忠墓北80米，长13.9米，宽10.2米，墓前有残缺牌坊立柱两根，墓碑刻“明诰封太夫人赵太君暨男大中大夫都转运盐使谦谦齐金公墓”。墓冢圆形，周围为乱石砌，压顶石已散落<ref name="nbwb-info" />。

金宪一墓位于赵太君墓之上，为长方形，长9.8米，宽6.85米。墓前牌坊仅存二柱。墓碑刻有“明诰赠荣禄大夫兵部尚书宪一金公墓”字样，正中有双狮捧球浮雕，墓碑两侧有莲望柱。墓圈为条石砌成，压顶石尚存<ref name="cnnb-discovery" />。

白云公墓正前方分布有两排17座小型墓葬，民间传说为金忠其他家族成员的墓葬，但墓碑已残损，无法分辨<ref name="cnnb-discovery" />。

== 保护 ==
{{external media|float=|width=250px|image1=[http://www.nbwb.net/UploadFile/images/2018012216134674.jpg 金忠家族墓葬群保护区划图]，原始内容[https://web.archive.org/web/20180328155052/http://www.nbwb.net/UploadFile/images/2018012216134674.jpg 存档]于2018年3月28日}}
金忠家族墓葬群对金忠一生及其家族史的研究具有参考价值，也是明初“简葬”之风的体现<ref>{{Cite web|url=http://www.nbwb.net/info.aspx?Id=10512|title=金忠家族墓葬群|accessdate=2018-03-28|author=|date=2010-11-23|work=|publisher=宁波市文物保护管理所}}</ref>，其中的金忠墓于1994年11月以“金忠墓及墓志碑”的名义被列入[[鄞州区文物保护单位|鄞县文物保护单位]]<ref>{{Cite web|url=http://www.nbwh.gov.cn/art/2008/9/20/art_3450_69368.html|title=宁波市文物事业发展蓝皮书（2008年）|accessdate=2017-07-10|author=|date=2008-09-20|publisher=宁波市文化广电新闻出版局|archiveurl=https://web.archive.org/web/20170814215130/http://www.nbwh.gov.cn/art/2008/9/20/art_3450_69368.html|archivedate=2017-08-14}}</ref>。2009年，在第三次全国文物普查中，金宪一等金忠家族成员墓地也被发现<ref name="cnnb-discovery" />。2011年，金忠家族墓葬群中保存较好的金忠墓、白云公墓、赵太君墓和金宪一墓以“东钱湖墓群”的名义被列入[[浙江省文物保护单位|浙江省文物保护单位]]<ref name="省保" />，2013年被列入[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]，并入[[第五批全国重点文物保护单位|第五批全国重点文物保护单位]]“东钱湖石刻”<ref name="nbwb-info" />。墓葬群西百步山山顶、东侧水稻田、南侧白虎山山脊、北侧青龙山山脊合围的区域为建设控制地带<ref>{{Cite web|url=http://www.nbwb.net/info_bhqh.aspx?id=41668|title=东钱湖墓葬群—金忠家族墓葬群|accessdate=2018-03-27|author=|date=2018-01-22|work=|publisher=宁波市文物保护管理所}}</ref>。当地政府对墓葬群实施24小时电子监控以防止盗窃和破坏文物<ref>{{Cite web|url=http://nb8185.cnnb.com.cn/html/1854/2017/0401/824396.html|title=东钱湖余有丁墓道石刻任其破败，原因是……|accessdate=2018-03-27|author=邱韵|date=2017-04-01|work=|publisher=中国宁波网}}</ref>。
