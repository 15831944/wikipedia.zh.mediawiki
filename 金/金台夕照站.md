'''金台夕照站'''是中国[[北京地铁|北京地铁]][[北京地铁10号线|10号线]]的一个车站，位于[[中国|中国]][[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[北京三环路|东三环中路]]与[[光华路_(北京)|光华路]]相交路口北侧过街天桥下。

== 命名 ==
[[File:金坛夕照石碑_Jintaixizhao_stone_tablet.jpg|thumb]]
“金台夕照”一名，源于[[燕京八景|燕京八景]]之一。2002年，[[京广大厦|京广大厦]]下出土[[乾隆帝|乾隆帝]][[御笔|御笔]]题金台夕照碑，清理后立在大厦之下。这一站在2008年10号线一期开通时借用这个典故命名。

== 结构 ==
金台夕照站为地下车站，[[分离式岛式站台|分离式岛式站台]]设计。车站总长169.2米。<ref>{{cite journal|author=张进联|coauthors=于德水，范京玲，杨枫，蒋雪梅|title=北京地铁10号线金台夕照站中洞逆作施工技术|journal=铁道标准设计|year=2008|volume=12|pages=217|ISSN=1004-2954|url=}}</ref>

==周边==
金台夕照站位于[[北京商务中心区|北京商务中心区]]内，周边高楼大厦密集。[[中央电视台总部大楼|中央电视台总部大楼]]等建筑在此站附近。

== 出入口 ==
金台夕照站有4个出入口，A、D口在东三环西侧，B、C口在东三环东侧。
