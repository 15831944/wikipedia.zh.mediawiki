'''金钱槭'''（[[学名|学名]]：{{lang|la|''Dipteronia sinensis''}}）别名'''双轮果'''（甘肃），是[[无患子科|无患子科]][[金钱槭属|金钱槭属]]的植物，是中国的特有植物。

== 形态 ==
落叶[[乔木|乔木]]，高达15米。叶子对生，为奇数羽状复叶；初夏开白色花，雌雄同株，圆锥花序；果实分为两个小坚果，各于周围有广翅，外形如钱。

== 分布 ==
分布于[[中国大陆|中国大陆]]的[[陕西|陕西]]、[[贵州|贵州]]、[[湖北|湖北]]、[[四川|四川]]、[[甘肃|甘肃]]、[[河南|河南]]等地，生长于海拔1,000米至2,000米的地区，常生长在林边以及疏林中，目前尚未由人工引种栽培。

== 异名 ==
