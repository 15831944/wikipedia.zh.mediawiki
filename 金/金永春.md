'''金永春'''（{{kor|諺='''김영춘'''|羅='''Kim Yong Chun'''|漢=金永春}}，{{bd|1936年|3月4日|2018年|8月16日|catIdx=Kim}}），[[朝鲜人民军元帅|朝鲜人民军元帅]]，曾任[[朝鲜人民军总参谋部|朝鲜人民軍總參謀長]]，[[朝鲜人民武装力量部|朝鲜人民武装力量部部长]]。

== 生平 ==
出生于[[江原道_(日本統治時代)|江原道]]<ref>[http://www.gotoread.com/vo/5750/page606794.html 1932年江原道] {{webarchive|url=https://web.archive.org/web/20150703055300/http://www.gotoread.com/vo/5750/page606794.html |date=2015-07-03 }}</ref>，[[万景台革命学院|万景台革命学院]]毕业。1986年当选[[朝鲜劳动党|朝鲜劳动党]]中央委員。1992年4月晋升人民軍大将。1993年10月任軍需動員總局局長。1994年3月任人民軍第6軍团長，1995年10月晋升人民軍[[朝鲜次帅列表|次帥]]。并出任人民軍总参謀長。2007年4月11日，在朝鮮第十一屆最高人民會議第五次會議上，當選為國防委員會副委員長。2009年2月11日，任[[朝鲜人民武力部|朝鲜人民武力部]]部长，是朝鮮軍方重要人物。2010年9月28日在[[朝鲜劳动党中央委员会2010年9月全体会议|朝鲜劳动党中央委员会2010年9月全体会议]]上当选为[[朝鲜劳动党中央军事委员会|朝鲜劳动党中央军事委员会]]委员。

2011年朝鲜最高领导人[[金正日之死|金正日]]的告別儀式，他是七位扶靈高官之一。2012年4月被免去人民武力部部长职务，转任朝鲜劳动党中央委员会总务部部长<ref>[http://www.ntdtv.com/xtr/gb/2013/12/14/a1023909.html   七个〝顾命大臣〞金正恩已干掉五个]</ref>。2016年4月14日，金永春晋升朝鲜人民军元帅。<ref name="cxsj">{{cite news|title=朝鲜授金永春和玄哲海元帅称号 李明秀次帅称号|url=http://news.sina.com.cn/o/2016-04-15/doc-ifxriqqx2492365.shtml|accessdate=2016-04-15|agency=中国新闻网|date=2016-04-15}}</ref> 

有传闻称，“金正日次子[[金正哲|金正哲]]和三子[[金正恩|金正恩]]的生母[[高英姬|高英姬]]（2004年去世）在世时，金永春属于‘高英姬派’。”韩国政府一位消息人士表示：“军队中拥护高英姬并称其为‘平壤母亲’的人就是金永春。”而金永春任朝鲜人民武力部部长也是为[[金正恩|金正恩]]顺利即位做铺垫。 <ref>[http://cnnews.chosun.com/site/data/html_dir/2009/02/13/20090213000007.html  金永春正跃升为北韩军方头号人物] {{webarchive|url=https://web.archive.org/web/20160305062022/http://cnnews.chosun.com/site/data/html_dir/2009/02/13/20090213000007.html |date=2016-03-05 }}</ref>

2018年8月16日，金永春因[[急性心肌梗塞|急性心肌梗塞]]離世，享年82岁。[[朝鮮中央通訊社|朝鮮中央通訊社]]報導了其死訊，並稱他是「忠誠的革命戰士」和「親近的革命戰友」<ref>[http://www.newspim.com/news/view/20180817000070 "'김정일의 오른팔' 김영춘 전 인민무력부장 사망...北, 국가장으로 장례] 뉴스핌 2018-08-17</ref>。8月20日，朝鲜劳动党中央委员、最高人民会议议员、人民武力省总顾问、人民军元帅金永春的国葬在[[平壤|平壤]]举行，出殡仪式在[[4·25文化會館|4·25文化會館]]吊唁场所举行，国家治丧委员会成员参加出殡仪式，随后告别仪式在[[愛國烈士陵園|新美里爱国烈士陵园]]举行，朝鲜最高领导人[[金正恩|金正恩]]与国家治丧委员会成员一起参加告别仪式，护送金永春灵柩登上烈士陵园，朝鲜人民军总政治局局长[[金秀吉|金秀吉]]陆军大将致悼词，金正恩向墓穴填土，朝鲜武力机关领导干部和故人家属向故人遗骸培土，在花圈敬献曲声中，以金正恩名义的花圈及[[朝鲜劳动党中央委员会|朝鲜劳动党中央委员会]]、[[朝鲜劳动党中央军事委员会|朝鲜劳动党中央军事委员会]]、[[最高人民会议|最高人民会议]]常任委员会、内阁名义的花圈敬献在坟前，金正恩同国家治丧委员会成员、故人家属、党武机关领导干部共同默哀悼念金永春<ref>{{Cite web |url = https://www.thepaper.cn/newsDetail_forward_2365328 |title =  朝鲜为人民军元帅金永春举行国葬，金正恩雨中默哀|author =  |date = 2018-08-21  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

== 简历 ==
* 1980年 - 朝鮮勞動党中央委員会候補委員 
* 1986年12月 - 人民軍總参謀部作戰局長 
* 1992年 - 人民軍大将 
* 1994年3月 - 人民軍第6軍団長 
* 1995年 - 人民軍次帥 
* 1995年10月 - 2007年4月 朝鲜人民軍總参謀長 
* 1995年10月 - 劳动党軍事委員会委员 
* 1998年9月 - 2007年4月 朝鲜国防委員会委员
* 2007年4月 - 国防委员会副委員長
* 2009年2月 - 人民武装力量部長
* 2013年4月 - 劳动党中央委员会部长
* 2016年6月-人民軍元帥
