{{漢傳密宗}}

'''金刚智''' （梵语：'''Vajrabodhi'''，669年－741年），音译'''跋日罗菩提'''，汉传[[密宗|密宗]]的祖师，[[开元三大士|开元三大士]]之一，主要弘传[[金刚界|金刚界]]一系。

== 在天竺 ==
金刚智是[[南印度|南印度]][[摩赖耶国|摩赖耶国]]人，[[婆罗门|婆罗门]][[种姓|种姓]]，从小就聪颖过人。十岁时在[[那烂陀寺|那烂陀寺]]出家 {{ref|monk}}，依止寂静智师学习[[声明|声明]]。十五岁时前往往[[西印度|西印度]]，用四年时间学习法称论师的著作，然后回到那烂陀寺。二十岁时受[[具足戒|具足戒]]。花六年时间学习了大小乘各种戒律。接着又学习了南宗《[[般若灯论|般若灯论]]》、《[[百论|百论]]》、《[[十二门论|十二门论]]》等著作。二十八岁时，在[[迦毗罗卫城|迦毗罗卫城]]，跟随[[胜贤|胜贤]]论师学习《[[瑜伽论|瑜伽论]]》、《[[唯识论|唯识论]]》、《[[辩中边论|辩中边论]]》，历时三年。三十一岁时前往[[南印度|南印度]]，当时[[龙树|龙树]][[菩萨|菩萨]]的弟子[[龙智|龙智]]年事已高，金刚智在而后的七年时间里，依止龙智学习了《[[金刚顶瑜伽经|金刚顶瑜伽经]]》、《[[毗卢遮那总持陀罗尼法门|毗卢遮那总持陀罗尼法门]]》等大乘经典，以及各种[[五明|五明]]论著，并受[[五部灌顶|五部灌顶]]，显密通达。辞别师父龙智后，金刚智回到中印度。由于南印度大旱，应其王邀请，金刚智前去请雨。而后国王为金刚智专门建造寺院。三年后，金刚智前往[[师子国|师子国]]瞻礼圣迹。回来后，准备前往东土。

== 在东土 ==
国王派人护送金刚智，并带上了许多的经典梵夾，以及许多珍宝。从海路经师子国、佛誓、裸人等二十余国，历经艰险，历时三年，719年 (开元七年)，终于到达[[广州|广州]]海面，节度使派数百船只前来迎接。次年初，到达东都[[洛阳|洛阳]]，而后在[[两京|两京]]{{ref|capital}}传教。先后在[[慈恩寺|慈恩寺]]、[[荐福寺|荐福寺]]、[[资圣寺|资圣寺]]、[[大荐福寺|大荐福寺]]等处，或建立坛场，或翻译经典，度化四众。741年 (开元二十九年)，金刚智在洛阳[[圆寂|圆寂]]。743年 ([[天宝|天宝]]二年)，在西龙门起塔{{ref|end}}。

== 译著 ==
* 723年 (开元十一年)，于资圣寺，[[东印度|东印度]][[婆罗门|婆罗门]]大首领直中书伊舍罗译语，嵩岳沙门温古笔受：
:《金刚顶瑜伽中略出念诵经》(即《瑜伽念诵法》)四卷，
:《佛说七俱胝佛母准提大明陀罗尼经》(即《七俱胝陀罗尼》)一卷，
* 730年 (开元十八年)，于大荐福寺，沙门智藏译语，一行笔受，删缀成文：
:《曼殊室利五字心陀罗尼》一卷
:《观自在瑜伽法要》一卷，
* 731年 (开元十九年)后译出：
:《金刚顶经瑜伽修习毗卢遮那三摩地法》一卷，
:《千手千眼观世音菩萨大身咒本》一卷
:《千手千眼观自在菩萨广大圆满无碍大悲心》一卷
:《陀罗尼咒本》一卷
:《不动使者陀罗尼祕密法》一卷
