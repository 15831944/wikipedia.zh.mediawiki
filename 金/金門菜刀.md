'''金門菜刀'''是[[金門縣|金門縣]]的特產之一，使用-{zh-tw:[[八二三炮戰|八二三炮戰]];zh-cn:[[金门炮战|金门炮战]]}-期間[[中国人民解放军|中国人民解放军]]砲擊金門的砲彈彈片、彈頭所製造，因菜刀鋒利耐用而著名，又稱'''砲彈鋼刀'''。<ref name="nownews">{{zh-tw}}[http://www.nownews.com/2006/07/14/11338-1965799.htm NHK好奇金門菜刀史：打台灣的砲彈作成刀　大陸人特愛]，今日新聞網，2006/07/14</ref>

== 歷史 ==
{{main|八二三炮戰}}
[[File:The_Making_of_Maestro_Wu's_Artillery_Shell_Knives.theora.ogv|200px]]
1958年至1979年之間，解放軍使用[[榴彈|榴彈]]和砲宣彈砲擊[[金門|金門]]；20年間有超過100萬枚[[砲彈|砲彈]]落在[[金門|金門]]。<ref name="nownews"/> 砲擊前期使用會爆炸的[[榴彈|榴彈]]；後期則使用不會爆炸的砲宣彈<ref name="ttvs">{{zh-tw}}[http://www.ttvs.cy.edu.tw/kcc/kinmen/knife.htm 金門之旅(九)]</ref>，這兩種炸彈是[[金門|金門]][[菜刀|菜刀]]的主要材料。另外，一顆砲彈可以製成40～60把[[菜刀|菜刀]]，因此這些砲彈至今尚未用完。

砲戰後，金合利鋼刀店老闆吳君乃大量搜購整座島內的廢棄彈殼，將其用於[[鍛造|鍛造]]刀具。由於彈頭的鋼鐵品質極佳，以之打製出來的菜刀也特別鋒利、耐用，鋼中帶韌的特性，即使切割硬物也不易變形，使金門菜刀在台灣遠近馳名。為防贗品混充，刀廠首創了開放現場指定[[砲彈|砲彈]]製刀。

目前製作金門菜刀的著名鋼刀店有『金合利鋼刀店』和『金永利鋼刀店』。<ref name="ttvs"/><ref>{{zh-tw}}[http://www.tw98.com/newsview.asp?id=422 金门菜刀]，2007-4-17</ref>

由於菜刀有紀念價值，早年到金門當兵的中華民國國軍返回台灣時，常會帶上幾把作爲紀念；在[[小三通|小三通]]後，金門菜刀亦成了[[中國大陸|中國大陸]]觀光客爭相搶購的紀念品。<ref name="nownews"/><ref name="ttvs"/>

== 金合利鋼刀 ==
1937年，吳朝熙創建「'''金合利製刀廠'''」，廠設於金門吳厝，其師承清光緒年間始祖吳宗山，始祖習藝於[[廈門|廈門]]，專營打鐵及農具鍛造。

== 金永利鋼刀 ==
1952年，林孫顯創立「'''金永利製刀廠'''」，廠設於金門金城鎮後埔街（古蹟邱良功貞節牌坊側），廣搜廢棄彈殼，專攻刀具焠煉及鍛燒工夫。<ref>{{Cite web|url=https://m.sohu.com/a/229451315_99981111/?pvid=000115_3w_a|title=与台湾的那场世纪炮战，竟意外造就了刀具史上最危险的经典之作-历史频道-手机搜狐|accessdate=2018-06-26|work=m.sohu.com|language=zh-cn}}</ref>
