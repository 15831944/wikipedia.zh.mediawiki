'''《金手指》'''是由[[王鍾_(香港)|王钟]]自编自导自演的[[粤语|粤语]][[驚悚片|惊悚片]]，由[[邵氏兄弟公司|邵氏兄弟公司]]制作并于1980年5月10日在[[香港|香港]]上映。<ref>{{cite web |title=《金手指》资料 |url=http://list.youku.com/show/id_zcc087156962411de83b1.html |publisher=[[优酷|优酷]] |accessdate=2018-08-10}}</ref> 这部电影是王钟的导演处女作，并创造200万[[港元|港元]]的香港票房。<ref>{{cite web |title=1980年香港票房 |url=http://www.boxofficecn.com/hkboxoffice1980 |publisher=中国电影票房排行榜 |accessdate=2018-08-10}}</ref>

== 剧情简介 ==
警方線人啞巴被殺害，於是警方聯繫啞巴好友，試圖說服他接替啞巴的[[臥底|臥底]]職責。好友為替啞巴報仇而承諾為警方臥底黑幫。在協助警方追查幕後黑手的時候，犯罪集團首腦卻另有他人。<ref>{{cite web |title=《金手指》劇情 |url=http://movie.mtime.com/34032/plots.html |publisher=[[時光網|時光網]] |accessdate=2018-08-10}}</ref>

== 评论反响 ==
《金手指》開創了香港電影里警匪題材寫實風的先河，并在劇情中大膽描寫普通市民成為警方臥底潛伏進犯罪集團協助警方查案。本片在一定程度上影響了[[李修賢|李修賢]]後期的電影作品《[[公僕_(電影)|公僕]]》和《皇家飯》。<ref name="douban" /> IMDb上，本片得分為6.8分（滿分10分）；<ref>{{cite web |title=Jin Shou Zhi |url=https://www.imdb.com/title/tt0445388/ |publisher=[[IMDb|IMDb]] |accessdate=2018-08-10}}</ref> 而在[[豆瓣|豆瓣]]上，本片的得分為6.1分。<ref name="douban">{{cite web |title=《金手指》 |url=https://movie.douban.com/subject/2349060/ |publisher=[[豆瓣|豆瓣]] |accessdate=2018-08-10}}</ref>
