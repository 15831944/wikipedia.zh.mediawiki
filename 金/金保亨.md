'''金保亨'''（{{lang-ko|'''김보형'''}}，{{lang-en|'''Kim Bo Hyung'''}}，{{bd|1989年|3月31日 |catIdx=Kim金}}）是[[大韓民國|大韓民國]]的歌手。2012年以[[B2M_Entertainment|B2M Entertainment]]推出的流行女子組合[[SPICA_(組合)|SPICA]]出道，為五位成員之一。

== 經歷 ==
金保亨於韓國出生，家庭成員有父母和與之相差五歲的哥哥，在2007年（19歲）那年參與CMB親親歌謠節演唱火曜飛的〈那樣的事〉獲得大獎初露頭角<ref>{{Cite web |url=http://serviceapi.nmv.naver.com/flash/linkPlayer.nhn?vid=E88AFC1733D51F6409975FAE62EEC49CF4F3&outKey=V1210a7569bde66424bcc0e798e7b382bb0e8e2840cc22a9a5d620e798e7b382bb0e8&service=facebook&title=%5BMV%5D+%EC%8A%A4%ED%94%BC%EC%B9%B4+%28SPICA%29+-+%EB%9F%AC%EC%8B%9C%EC%95%88%EB%A3%B0%EB%A0%9B+%28Russian+Roulette%29|title=천친가요제대상수상 김보형양|author=|date=2007-09-12|publisher=NAVER|language=ko|accessdate= 2012年10月2日}}</ref>。之後從2008年開始與[[2NE1|2NE1]]同屬在YG娛樂擔任練習生超過兩年以上的時間並作為2NE1的預備成員，卻因缺少和2NE1相符的特色而未能以該組合成員出道<ref name="ssd34">{{Cite web |url=http://news.naver.com/main/read.nhn?mode=LSD&mid=sec&sid1=106&oid=001&aid=0005496475|title=스피카 "산전수전 끝, 빛나는 별 될래요"|author=|date=2012-02-04  |publisher=hankooki|language=ko|accessdate= 2012年10月2日}}</ref>。對此她在2012年1月31日下午接受韓國體育的採訪中說道：
{{Cquote|我和2NE1四位成員一起做了練習，然後2NE1以四人出道後，我透過[[梁鉉錫|梁鉉錫]]社長的引薦來到現在的公司後成為[[SPICA_(組合)|SPICA]]的成員。<ref>{{Cite web |url=http://sports.hankooki.com/lpage/music/201202/sp2012020106024795510.htm|title='스피카' 김보형, 2NE1 연습생 출신|author=안진용기자|date=2012-02-01  |publisher=hankooki|language=ko|accessdate= 2012年10月2日}}</ref><ref name="ssd34"/>}}最終結束五年的練習生活在2012年2月9日以[[SPICA_(組合)|SPICA]]的成員站上[[Mnet|Mnet]]《[[M!_Countdown|M! Countdown]]》出道。
