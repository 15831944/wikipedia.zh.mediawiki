'''金肉人二世'''（'''{{lang|ja|キン肉マンII世}}''' Kinnikuman Nisei）是[[日本|日本]][[漫畫家|漫畫家]][[Yudetamago|蚵仔煎]]（{{lang|ja|ゆでたまご}}）的[[漫畫|漫畫]]作品、亦是原作與[[動畫|動畫]]作品及[[主角|主角]]的名字。

《[[金肉人|金肉人]]》連載完後多年，作者蚵仔煎於1998年在[[集英社|集英社]]的青年雜誌《週刊Playboy》上開始連載金肉人二世至今。

==劇情介紹（以動畫版為依據）==
===第一季===
; 海克力士學院～傳說的序章篇
:第一代金肉人就任王位之後28年，原本持續著沒有戰爭的和平日子，然而有一天、[[地球|地球]]突然遭到惡行超人們襲擊。金肉人等傳說超人們，為了阻止惡行超人決定挺身而出。但是完全上了年紀的傳說超人們，根本無力對抗惡行超人。
:為了培訓出能夠和惡行超人對抗的年輕正義超人，於是決定成立正義超人訓練學校—'''海克力士學院'''，並由傳說超人們擔任教練。金肉人的兒子金肉萬太郎及其他傳說超人的兒子們，全都陸續地被送到海克力士學院去。
; 魔鬼工廠篇
:萬太郎經過嚴厲的修煉，平安地從海克力士學院畢業。作為派駐日本的超人而前往地球，和“魔鬼工廠”的惡行超人對抗。
:相隔28年的正義超人 VS 惡行超人摔角決戰開始。在那裏有傳說超人羅賓假面的兒子—凱文假面的身影。
; 海克力士學院一期生・二期生替換戰篇
:魔鬼工廠毀滅之後，萬太郎忽略執行例行任務，正懶散的生活著，令委員長大怒。
:另外、第二屆的海克力士學院畢業生，以駐守日本的權利為賭注，跟第一屆的萬太郎等人挑戰。
:於是海克力士學院一期生、二期生之間的對決開始。
; 火場爆發力修行篇
:經過數場死鬥，身體變得狼狽不堪的萬太郎。身為金肉星的下任大王，為了學會真正的火場爆發力，要跟「目中無人者」三人組交戰。
; 最兇惡行超人軍篇
:動畫原創劇情。暌違三十四年的傳說慶典「超人世界盃」即將開幕，沒想到在開幕前夕的慶祝派對卻突然冒出六個邪惡萬太郎，挾持凜子、惠子、小環當做人質，還要求跟萬太郎一行人展開三回合的雙打對戰，他們的真面目到底是誰？跟萬太郎一行人之間又有什麼恩怨？超人世界盃就此被迫暫停，七位正義超人將與六位惡行超人展開戰鬥。

===第二季===
; 超人世界盃篇
:最兇惡行超人軍全軍覆沒後，傳說慶典「超人世界盃」正式開幕。萬太郎一行人將與來自世界各地的新世代超人們展開決鬥！

==主要人物==
===傳說超人===
*'''金肉人'''(Suguru Muscle)({{lang|ja|キン肉スグル}} Kinniku Suguru)
:金肉人一世，金肉萬太郎的[[爸爸|爸爸]]。
*'''陽光人'''(Sunshine)
:雷克斯人和將軍人的[[師父|師父]]。
*'''破裂一世'''(Brocken Jr.)
:傑德的師父。
*'''泰利人'''(Terryman)
:泰利小子的爸爸。
*'''羅賓假面'''(Robin Mask)
:凱文假面的爸爸。

===正義超人===
==== 海克力士學院(Hercules Factory) ====
===== 一期生 =====
*'''金肉萬太郎'''(Mantaro Kinniku)({{lang|ja|キン肉万太郎}} Kinniku Mantarō)
:本劇中心人物，金肉人二世，金肉人的兒子。必殺技為「金肉旋風」，在被刀疤臉破解之後，稍微加以改良，名為「迴旋金肉旋風」，最後還練成新必殺技，名為「金肉千禧年」。
*'''泰利小子'''(Terry the Kid)({{lang|ja|テリー・ザ・キッド}} Terī Za Kiddo)
:泰利人的兒子。必殺技為「小牛標誌」，「德州三葉草」。
*'''[[瞪羚|瞪羚]]人'''(Gazelleman)({{lang|ja|ガゼルマン}} Gazeruman)
:萬太郎與泰利小子的同學。
*'''[[海象|海象]]人'''(Seiuchin)({{lang|ja|セイウチン}} Seiuchin)
:萬太郎與泰利小子的同學。

===== 二期生 =====
*'''傑德'''(Jade)({{lang|ja|ジェイド}} Jeido)
:破裂一世的愛徒。特徵是頭上帶著一頂腳踏車帽，於一期生二期生替換戰後投靠萬太郎的陣營，時常待在萬太郎四人身邊，稱呼萬太郎四人為「學長」。胸前有一顆骷髏頭，可使出必殺技「柏林的紅雨」，曾將此招傳授給萬太郎。
*'''克利歐尼人'''(Clioneman)({{lang|ja|クリオネマン}} Kurioneman)
:特徵是果凍身體，手部能變成劍，胸前能變成[[凸透鏡|凸透鏡]]，也能變身成[[烏賊|烏賊]]和[[水母|水母]]等[[軟體動物|軟體動物]]。
*'''死亡信號'''(Dead Signal)({{lang|ja|デッド・シグナル}} Deddo Shigunaru)
:全身都是[[交通標誌|交通標誌]]，還可以旋轉成圓鋸。
*'''刀疤臉'''(Scarface／Mars)({{lang|ja|スカーフェイス}} Sukāfeisu／{{lang|ja|マルス}} Marusu)
:本名「馬爾斯」，真實身分為潛入「海克力士學院」的魔鬼工廠惡行超人餘黨，必殺技是「燕尾巨鏟」，實力相當強，能瓦解萬太郎的必殺技「金肉旋風」。最後被萬太郎的新必殺技「金肉千禧年」擊敗。

===金肉星===
*'''[[亞歷山大|亞歷山大]]‧米特'''(Alexandria Meat)({{lang|ja|アレキサンドリア・ミート}} Arekisandoria Mīto)
:萬太郎的搭檔，稱呼萬太郎為「二世」。
*'''委員長'''
:海克力士學院的管理委員長，也是一期生與二期生替換戰的主辦人。
*'''伊凱門‧馬斯爾'''
:委員長的兒子。
*'''賈桂琳'''
:金肉星出身的大美女，其美貌不只是地球上的所有男性與女性，就連萬太郎也都為之迷倒，甚至還令萬太郎前去搭訕。

===[[人類|人類]]===
*'''二階堂凜子'''({{lang|ja|二階堂凜子}} - {{lang|ja|にかいどうりんこ}} - Nikaidō Rinko)
:萬太郎的[[女友|女友]]。時常對萬太郎的花心感到不滿，內心有點暗戀傑德。
*'''惠子'''({{lang|ja|恵子}} - {{lang|ja|けいこ}} - Keiko)
:凜子的好友。
*'''前川環'''({{lang|ja|前川たまき}} - Maekawa Tamaki)
:凜子的好友。
*'''二階堂瑪莉'''({{lang|ja|二階堂瑪莉}} - Nikaidō Mari)
:凜子的媽媽。

===惡行超人===
====魔鬼工廠 (Demon Manufacturing Plant)====
惡行超人的訓練學校，就像正義超人有海克力士學院一樣。
*'''凱文假面'''(Kevin Mask)({{lang|ja|ケビン・マスク}} Kebin Masuku)
:正邪不明，但大多是以正義超人方出戰。羅賓假面的兒子，因受不了父親的英才教育而離家出走。最初為報復父親才加入魔鬼工廠。跟萬太郎是亦敵亦友的關係。必殺技為「鐘樓壓頂」。
*'''[[行動電話|行動電話]]人'''(Tel-Tel Boy)({{lang|ja|テルテル・ボーイ}} Teru Teru Bōi)
:外表像一隻手機，還可撥打號碼呼叫相關人物。
*'''極限人'''(MAXman)({{lang|ja|マックスマン}} Makkusu Man)
:可變身為多種武器，甚至也能變成球鞋。金肉人時期惡魔六騎士之一的スニゲータ人之孫，受到祖父嚴格訓練。弱點是潔癖，被萬太郎運用這點使用大便攻擊重創而敗。

=====魔鬼二人組（ナイトメアズ）=====
傳說超人—陽光人最優秀的兩名愛徒
*'''雷克斯人'''(Rex King)({{lang|ja|レックス・キング}} Rekkusu Kingu)
:肉食恐龍化身的惡行超人，絕招是「必殺恐龍手」，最後被泰利小子打敗。
*'''將軍人'''(Check Mate)({{lang|ja|チェックメイト}} Chekkumeito)
:[[西洋棋|西洋棋]]化身的惡行超人，萬太郎於魔鬼工廠之戰的最後一位對手。初登場態度頗為囂張，還對其師父陽光人直接拳打腳踢，令在場觀眾甚至連小孩子都看不下去。被萬太郎擊敗後，陽光人將他帶離治療，直到萬太郎與目中無人三人組的第一場對戰時再度出現，不過此時的他早已改邪歸正，變成正義超人，並時常待在萬太郎一行人身邊。與最兇惡行超人軍的第二戰時還中途干涉，幫忙瞪羚人與海象人對付彈力勇士與青蛙鬥士。
:能按下雙肩上的西洋棋按鈕變成馬匹及城堡；變成馬匹時能展開強大的踢擊力；變成城堡時擁有絕佳的防禦力，而且還能使身體分裂、壓制住敵人，必殺技是藉由同時按下雙肩上的西洋棋按鈕變成城堡人馬所使出的「駿馬榮譽攻擊」。

====目中無人三人組（NO RESPECT）====
動畫原創角色。
*'''霍克大巨人'''(Fork the Giant)({{lang|ja|フォーク・ザ・ジャイアント}})
:能變身成類似戰車的模樣。戰鬥結束後，受到觀眾群起毆打，但是被萬太郎伸出援手。
*'''半藏'''(Hanzō)({{lang|ja|ハンゾウ}})
:能用稻草人作為替身，萬太郎也藉由這招對付他。戰鬥結束後，被獄卒帶回牢裡去。
*'''冷骨戰士'''(Bone Cold)({{lang|ja|ボーン・コールド}})
:必殺技為「旋轉鑽頭」。爸爸名叫「金骨人」，與米特為舊識。

====最兇惡行超人軍(The Poison Six Pack)====
動畫原創角色，初登場於劇場版。
*'''達茲爾'''(Dazzle)({{lang|ja|ダズル}})
:泰利小子的對手，能用左肩上的眼睛使出幻術。一直用此招將泰利小子身陷其中，後來被萬太郎用釣竿鉤上大便破解。
*'''普羅鐵金剛'''(The Protector)({{lang|ja|ザ・プロテクター}})
:傑德的對手，身軀非常堅硬，連傑德的必殺技「柏林的紅雨」都傷不了。
*'''彈力勇士'''(Puri-Puri Man)({{lang|ja|プリプリマン}})
:瞪羚人的對手，特徵是屁股頭，瞪羚人經由萬太郎的提示，以「我打你屁股」對付他，讓他嚇得落荒而逃。雙手能變成雙腿，手腕上有長刺的盔甲，轉動時還會變成圓鋸，連岩石都能粉碎。本來想以此招給瞪羚人致命的一擊，可惜被將軍人破解。
*'''青蛙鬥士'''(El Kaerun)({{lang|ja|エル・カエルーン}})
:海象人的對手，舌頭能伸長，嘴巴能吐出綠色黏液。在決鬥中挾持海象人的媽媽與妹妹，並將她們拖入水裡。海象人去拯救她們時，還將他們一家三口陷入其境。
*'''犀暴愚'''(The Cyborg)({{lang|ja|ザ・犀暴愚}})
:凱文假面的對手，全身的盔甲都有長刺，必殺技是將身體捲起來的「葛雷德旋轉」，凱文假面對於這招只有閃躲的份，另外雙手還能像火箭一樣發射出去。雖然最後還是被打敗，但也使得凱文假面花費的時間超過20秒的限制，讓他被迫強制退場。
*'''巴隆極限超人'''(Baron Maximilian)({{lang|ja|バロン・マクシミリアン}})
:萬太郎的對手，平時外表看似[[魔術師|魔術師]]。是最兇惡行超人軍中的最強者，擁有隱形能力及超快移動速度，還可以二度進化，進化之後不僅實力同時跟著增強，就連體型也會跟著變大，一度把萬太郎打到昏迷不醒，必殺技是「巴隆龍捲風」。
