'''金町站'''（{{jpn|j=金町駅|hg=かなまちえき|rm=Kanamachi eki|f=y}}）是一個位於[[東京都|東京都]][[葛飾區|葛飾區]]{{link-ja|金町|金町}}六丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）、[[日本貨物鐵道|日本貨物鐵道]]（JR貨物）的[[鐵路車站|鐵路車站]]。現為葛飾區最北端的車站。

== 概要 ==
本站停靠[[常磐線|常磐線]]與[[總武本線|總武本線]]支線（通稱：{{Link-ja|新金貨物線|新金貨物線|新金線}}）。新金線是貨物列車專用路線，連結常磐線快速線。

客運業務僅行走於常磐線緩行線的[[常磐緩行線|常磐線各站停車]]停靠。[[車站編號|車站編號]]是'''JL21'''。

依據[[特定都區市內|特定都區市內]]制度，本站屬「東京都區內」，也是常磐線在該區內的最東端。另外，本站 - [[松戶站|松戶站]]間的營業里程為3.9公里，是{{Link-ja|東日本旅客鐵道東京支社|東日本旅客鉄道東京支社|東京支社}}管內常磐線（日暮里 - 取手間）的最長站距。

車站南口對向是[[京成電鐵|京成電鐵]][[京成金町線|金町線]][[京成金町站|京成金町站]]，兩站可轉乘。

本站也發售與京成電鐵的連絡定期券。

== 年表 ==
* 1897年（[[明治|明治]]30年）12月27日 - [[日本鐵道|日本鐵道]]車站開業。
* 1906年（明治39年）11月1日 - 日本鐵道國有化，為[[鐵道省|官設鐵道]]車站。
* 1909年（明治42年）10月12日 - 線路名稱制定，本站屬常磐線。
* 1936年（昭和11年）12月11日 - 日暮里 - 松戶間直流電化，金町全列車電車化（上野 - 松戶間）。往松戶以北使用機車牽引的全部列車不停靠本站。
* 1949年（昭和24年）
** 5月 - {{Link-ja|三菱製紙|三菱製紙}}中川工場專用線完工。
** 6月1日 - [[日本國有鐵道|日本國有鐵道]]成立。
* 1965年（昭和40年）2月 - 因複複線增線工程，高架站工程動工。
* 1967年（昭和42年）4月1日 - 北口開通<Ref>葛飾区編 | 『増補 葛飾区史 下巻』</Ref>。
* 1969年（昭和44年）3月23日 - 綾瀨 - 金町間切換至高架新線路（現在的緩行線）<Ref>葛飾区編 | 『増補 葛飾区史 年表』</Ref>。在緩行、快速分離以前由常磐線全列車使用。
* 1971年（昭和46年）3月1日 - 綾瀨 - 金町間複複線暫時啟用。國電與優等列車、貨物列車分離。
* 1971年（昭和46年）4月20日- 綾瀨 - 我孫子間複複線化完成，緩行線與快速線分離，成為帝都高速度交通營團千代田線直通的緩行線車站。取消往上野的列車。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]，為JR東日本、JR貨物車站。
* 2001年（[[平成|平成]]13年）11月18日 - [[IC卡|IC卡]]「[[Suica|Suica]]」啟用。
* 2003年（平成15年）3月23日 - 廢止定期貨物列車。

== 車站結構 ==
緩行線上設有[[島式月台|島式月台]]1面2線的[[高架車站|高架車站]]（土堤高架）。

閘口位於地下道近北口西側，對面的東側設有[[自動售票機|自動售票機]]與[[綠窗口|綠窗口]]（營業時間 7:00 - 20:00）。

[[大堂|大堂]]為南北各出入口的聯絡通道。由於車站周邊缺乏南北連絡道路，因此大堂成為實際上的生活道路。雖然車輛無法通行，但可牽自行車通過。北口有階梯，但設有無障礙斜坡供民眾使用。

月台與剪票口層之間設有一座電梯與雙向電扶梯。廁所在剪票口內正面，附設多功能廁所。

月台松戶側有候車室，龜有側有{{Link-ja|NEWDAYS|NEWDAYS|NEWDAYS MINI}}，中央樓梯有立食蕎麥麵店。

若要從本站往日暮里、上野方向，須在北千住站轉乘。

== 貨運業務 ==
2011年，本站僅剩臨時貨物業務。沒有定期貨物列車。

2003年以前，本站專用線有貨物業務。過去車站西北方有{{Link-ja|三菱製紙|三菱製紙}}中川工場，之間鋪有專用線。當時有輸送原料至該工廠的{{Link-ja|八戶臨海鐵道|八戸臨海鉄道}}{{Link-ja|北沼站|北沼駅}} - {{Link-ja|八戶貨物站|八戸貨物駅}} - 本站間的貨物列車，之後隨著工廠關閉，定期貨運於同年3月18日廢止。另外，以前三菱製紙中川工場北側{{Link-ja|三菱瓦斯化學|三菱ガス化学}}東京工場亦有專用線。

經由新金線的貨物列車會在側線待機。

== 車站周邊 ==
位居[[東京都區部|東京都區部]]東端，周邊1公里圈內皆是密集的住宅、建物。東方是[[千葉縣|千葉縣]]界的[[江戶川|江戶川]]，對岸是[[松戶市|松戶市]]郊外地域。

南口的41層複合型大廈「VENASIS金町」（ヴィナシス金町）於2009年6月底完工。2010年4月1日，[[國道6號_(日本)|國道6號]]與車站南側之間的連絡道路（區道）開通。

車站西北方的三菱製紙中川工場改建的[[東京理科大學|東京理科大學]]葛飾校區於2013年4月開學。
