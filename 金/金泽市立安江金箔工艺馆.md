'''金泽市立安江金箔工艺馆'''（{{lang-ja|1=金沢市立安江金箔工芸館}}）是一座位于[[日本|日本]][[石川县|石川县]][[金泽市|金泽市]]的[[金箔|金箔]][[博物馆|博物馆]]<ref>{{Cite web |url=http://www.visitkanazawa.net/popular-museums-in-kanazawa |title=Museums in Kanazawa |language=en |access-date=2018-11-10 |publisher=Visit Kanazawa |archive-url=https://web.archive.org/web/20171110225435/http://www.visitkanazawa.net/popular-museums-in-kanazawa# |archive-date=2017-11-10 |dead-url=no }}</ref>。

== 历史 ==
金泽市立安江金箔工艺馆的前身'''安江金箔博物馆'''建立于1974年，以建立者安江孝明的姓氏命名<ref>{{Cite web |url=http://www.kanazawa-museum.jp/kinpaku/pdf/english.pdf |format=PDF |language=en |access-date=2018-11-10 |title=Museum Leaf Gold Yasue |publisher=金泽市立安江金箔工艺馆 |archive-url=https://web.archive.org/web/20171215021117/http://www.kanazawa-museum.jp/kinpaku/pdf/english.pdf# |archive-date=2017-12-15 |dead-url=no }}</ref>。1985年，位於北安江的该博物馆及其所有文物于1985年捐赠给金泽市政府，并随后改名为'''金泽安江金箔博物馆'''。2010年，由于金泽市政府认为東山地区的金箔文化更浓厚，博物馆被市政府迁到了東山地区<ref name=off1>{{Cite web |url=http://www.kanazawa-museum.jp/kinpaku/english/index.htm |title=About the Museum |language=en |publisher=金泽市立安江金箔工艺馆 |access-date=2018-11-10 |archive-url=https://web.archive.org/web/20171110225011/http://www.kanazawa-museum.jp/kinpaku/english/index.htm# |archive-date=2017-11-10 |dead-url=no }}</ref>。

== 建筑 ==
金泽市立安江金箔工艺馆占地面积约为758平方米，建筑主体共3层。一楼由图书馆、多功能展览厅、办公室和信息中心组成，总面积为498平方米。二层为永久性展览区、休息区、临时展览区和多媒体区，总面积为444平方米。三层由研讨室和研究中心组成，总面积为438平方米。这座博物馆的建筑是按照金泽地区传统的仓库式样设计的，建筑内墙上镶嵌着[[金箔|金箔]]<ref name=off1 />。

== 展览 ==
该博物馆展出了大约300件金箔艺术品，同时展示了金泽金箔工艺的发展历史、金箔的制作工艺以及金箔的制作工具<ref>{{Cite web |url=http://www.kanazawa-tourism.com/eng/guide/guide1_4.php?no=5 |title=Kanazawa Yasue Gold-Leaf Museum |language=en |access-date=2018-11-10 |publisher=金泽市观光协会 |archive-url=https://web.archive.org/web/20171110225445/http://www.kanazawa-tourism.com/eng/guide/guide1_4.php?no=5# |archive-date=2017-11-10 |dead-url=no }}</ref>。展览品包括前江户时代、[[江户时代|江户时代]]和现代时期的作品，如[[屏风|屏风]]、[[漆器|漆器]]、金泽[[佛坛|佛坛]]、[[能剧|能剧]]服装等<ref>{{Cite web |url=http://www.en.visitkanazawa.jp/mustgoplace/touristspot/area4/5 |title=Must Go Place: Tourist Spot Area 4 |publisher=Visit Kanazawa |language=en |access-date=2018-11-10 |archive-url=https://web.archive.org/web/20171110225219/http://www.en.visitkanazawa.jp/mustgoplace/touristspot/area4/5# |archive-date=2017-11-10 |dead-url=no }}</ref>。
