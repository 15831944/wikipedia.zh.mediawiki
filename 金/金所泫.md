'''金所泫'''{{notetag|於個人官方微博帳戶名字及直播正名金所泫}} （{{lang-ko|'''김소현'''}}，{{lang-en|'''Kim So-hyun'''}}，{{bd|1999年|6月4日}}），[[韓國|韓國]]知名[[女演員|女演員]]<ref name = "tenasia" />，名字常音譯為'''金素賢'''或'''金所炫'''。2007年，客串出演[[韓國放送公社|KBS]]電視劇《幸福的女人》，展開其童星演藝生涯。2012年，連續出演電視劇《[[擁抱太陽的月亮|擁抱太陽的月亮]]》及《[[想你|想你]]》後獲得關注，演技得到大眾肯定。2015年，出演《[[Who_Are_You－學校2015|Who Are You－學校2015]]》一人分飾兩角大受好評，人氣旺盛。因外貌長得像韓國女演員[[韓佳人|韓佳人]]、[[孫藝真|孫藝真]]、[[尹恩惠|尹恩惠]]等多位公認氣質美女，所以擁有「小韓佳人」、「小孫藝真」及「小尹恩惠」等稱號。

== 早期 ==
金所泫於1999年6月4日在[[澳洲|澳洲]]出生。2003年時，金所泫的家庭移民回[[韓國|韓國]]<ref name="tenasia"/>。

==出道歷程==
===2007年－2011年：早期生涯===
金所泫出道時簽約[[SidusHQ|SidusHQ]]，並於2007年開始參與劇集演出。在多部劇集中，因她飾演女主角的童年而為人所熟識。金所泫曾在《{{link-en|被破壞的男子|Man of Vendetta}}》、《我們身邊的犯罪》和[[荊棘鳥_(電視劇)|荊棘鳥]]等電視劇和電影中客串。

===2012年－2014年：名氣漸升===
2012年，金所泫開始為人熟悉。她先於《[[擁抱太陽的月亮|擁抱太陽的月亮]]》飾演第二女主角的少年時期，再於《[[屋塔房王世子|屋塔房王世子]]》和《[[想你|想你]]》出演。金所泫在年尾的[[MBC演技大賞|MBC演技大賞]]中，獲得童星獎。

2013年，金所泫聯同[[SHINee|SHINee]]的[[珉豪|珉豪]]和[[盧弘喆|盧弘喆]]擔任《[[Show!_音樂中心|Show! 音樂中心]]》主持<ref>{{cite web |url=http://www.soompi.com/2013/04/11/the-new-music-core-mcs-announced/ |title=The New "Music Core" MCs Announced! |publisher=Soompi |date=2013-04-11 |accessdate=2016-12-01|language= en}}</ref>，於2015年離開該節目<ref>{{cite web|url=http://www.soompi.com/2015/04/18/block-bs-zico-and-kim-so-hyun-step-down-as-music-core-mcs/ |title=Block B’s Zico and Kim So Hyun Step Down as "Music Core" MCs |publisher=Soompi |date=2015-04-18 |accessdate=2016-12-01|language= en}}</ref>。

===2015年－現今：出演女主角===
2015年，金所泫出演《[[Who_Are_You－學校2015|Who Are You－學校2015]]》，分飾李恩菲（高恩菲）和高恩星兩角<ref>{{cite web|url=http://www.soompi.com/2015/03/31/btobs-sungjae-kim-so-hyun-and-nam-joo-hyuk-confirmed-for-school-2015/ |title=BTOB’s Sungjae, Kim So Hyun, and Nam Joo Hyuk Confirmed for “School 2015” |publisher=Soompi |date=2015-03-31 |accessdate=2016-12-01|language= en}}</ref>。她於第八屆[[韓國電視劇節|韓國電視劇節]]中，因此劇獲得「年度STAR賞」<ref>{{cite web|url=http://www.soompi.com/2015/10/09/kim-soo-hyun-kim-tae-hee-exos-chanyeol-and-more-win-at-2015-korea-drama-awards/ |title=Kim Soo Hyun, Kim Tae Hee, EXO's Chanyeol, and More Win at 2015 Korea Drama Awards |publisher=Soompi |date=2015-10-09 |accessdate=2016-12-01|language= en}}</ref>。

2016年，金所泫出演了電影《{{link-en|純情|Pure Love (film)}}》<ref>{{cite web |url=http://www.soompi.com/2016/01/07/exos-d-o-and-kim-so-hyun-experience-bittersweet-first-love-in-new-pure-love-posters/ |title=EXO's D.O and Kim So Hyun Experience Bittersweet First Love in New "Pure Love" Posters |publisher=Soompi |date=2016-01-07 |accessdate=2016-12-01|language= en}}</ref>，亦出演了《[[Page_Turner|Page Turner]]》和網絡劇《[[噩夢老師|噩夢老師]]》<ref>{{cite web|url=http://www.soompi.com/2015/09/22/btobs-minhyuk-confirmed-to-act-with-kim-so-hyun-in-upcoming-web-drama/ |title=BTOB's Minhyuk Confirmed to Act With Kim So Hyun in Upcoming Web Drama |publisher=Soompi |date=2015-09-22 |accessdate=2016-12-01|language= en}}</ref><ref>{{cite web |url=http://www.soompi.com/2015/11/16/kim-so-hyun-to-make-a-kbs-drama-comeback/ |title=Kim So Hyun to Make a KBS Drama Comeback |publisher=Soompi |date=2015-11-16 |accessdate=2016-12-01|language= en}}</ref>。在《[[打架吧鬼神|打架吧鬼神]]》中，她飾演了女主角金賢智<ref>{{cite web|url=http://www.soompi.com/2016/04/27/2pms-taecyeon-could-be-battling-ghosts-this-summer-in-drama-with-kim-so-hyun/ |title=2PM's Taecyeon Could Be Battling Ghosts This Summer In Drama With Kim So Hyun |publisher=Soompi |date=2016-04-27 |accessdate=2016-12-01|language= en}}</ref>。

2017年，金所泫出演電視劇《[[君主－假面的主人|君主－假面的主人]]》，同年與公司[[Sidus_HQ|Sidus HQ]]約滿不續約，離開公司。

2018年，金所泫出演電視劇《[[Radio_Romance|Radio Romance]]》，此劇亦是她成年後的第一部作品。
