'''金門縣公共自行車租賃系統'''（英文：'''K Bike'''）是[[金門縣政府|金門縣政府]]負責建置的[[公共自行車|公共自行車]]租賃系統，2017年4月16日由偉華科技實業負責正式營運。<ref>{{cite web |url=http://www.kinmen.gov.tw/KinmenWeb/wSite/page/1-235-81908.html|title=金門K Bike正式啟用 首月優惠免費騎|date=2017-04-17 |publisher= 金門日報社}}</ref>

== 發展情況 ==
為響應全球低碳環保潮流趨勢，金門縣政府率先設置離島地區首座自動化公共自行車租賃系統─「K Bike」，朝健康低碳島的目標邁進，讓[[金門縣|金門縣]]民「幸福指數」再度提升。有別於以往公共自行車租借，受限於傳統人工租借的服務方式，往往造成時間上的浪費，現在金門縣啟用的「K Bike」，採用24小時自動化租借系統，第一期設置有26處租借站點，29台自動租賃站，並提供500輛公共自行車，採甲租乙還24小時全程e化租借服務，且租借地點多，在大、小金門都有它的身影，連租借機台的設計都富有濃濃的金門風格。<ref>{{cite web |url=http://www.ydn.com.tw/News/232119|title=金門K BIKE試營運 低碳幸福飆升 |date=2017-04-17 |publisher=青年日報}}</ref>

== 租賃方式 ==
*會員: 縣民提出縣民卡，前四個小時免費，超過四個小時每半個小時收10元，最高收費150元。非縣民提出[[悠遊卡|悠遊卡]]或[[一卡通|一卡通]]，第一個小時免費，超過一小時每半個小時收10元，最高收費150元。

*非會員:無免費優惠，每半個小時收10元，最高收費250元。

*金門「K Bike」自動租賃服務機台載明租借對象分「會員」、「非會員」。還有增加了可提供[[銀聯卡|銀聯卡]]租借使用，全國首創。<ref>{{cite web |url=http://news.ltn.com.tw/news/life/breakingnews/2038298|title=金門「K Bike」啟動 離島第一名|date=2017-04-16 |publisher=自由時報}}</ref> <ref>{{cite web |url=https://udn.com/news/story/7327/2406576|title=金門K Bike很厲害 銀聯卡嘛通|date=2017-04-16 |publisher=聯合新聞網}}</ref>
