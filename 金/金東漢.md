}}{{Infobox Korean name|hangul=김동한|hanja=金東漢|rr=Kim Dong Han|title=金東漢}}'''金東漢'''（{{Kor|k=''김동한''|rm=''Kim Dong Han''}}，{{Lang-en|''Kim Dong Han''}}，{{Bd|1998年|7月3日}}），曾為大邱舞團The Face以及首爾舞團D.O.B成員之一，因為常現身街頭表演，所以在出道前已累積一定粉絲群，後因被經紀公司相中而退出舞團。現為OUI娛樂經紀公司練習生，練習時間為1年6個月，2017年參加[[Mnet媒體|Mnet]]節目[[PRODUCE_101_第二季|《PRODUCE 101 第二季》]]後，被網友掀出過去在舞團街頭表演的影片，其後因為有著高挑身材、出眾長相，年紀輕輕卻舞技精湛表現引起了關注。自2017年8月30日開始正式進行[[JBJ|JBJ]]團體相關的活動，並在同年10月18日發行迷你專輯出道。

== 個人背景 ==

=== 早期生活 ===
在出道前曾為舞團The Face以及D.O.B出身，會常現身街頭表演，過去挑戰2倍速《DOPE》的影片被翻出，動作敏捷跟上節奏音樂節奏一下，他立刻跟上2倍速的舞蹈，動作到位、強而有力，在網路上引起討論。但他後來被經紀公司相中而離開舞團，成为OUI娛樂經紀公司練習生，因參加韓國著名選秀節目《[[PRODUCE_101_第二季|PRODUCE 101 第二季]]》，憑著強勁的跳舞實力，被粉絲推選[[JBJ|JBJ]]組合成員之一。<ref>{{Cite news|url=https://star.ettoday.net/news/886223?t=%E3%80%8A101%E3%80%8B19%E6%AD%B2%E7%9A%84%E4%BB%96%E7%88%86%E7%B4%85%EF%BC%812%E5%80%8D%E9%80%9F%E9%98%B2%E5%BD%88%E3%80%8ADOPE%E3%80%8B%E8%AE%93%E7%B2%89%E7%B5%B2%E6%88%80%E6%84%9B%E4%BA%86|title=《101》19歲的他爆紅！2倍速防彈《DOPE》讓粉絲戀愛了 {{!}} ETNEWS星光雲 {{!}} ETNEWS新聞雲|newspaper=ETNEWS星光雲|accessdate=2017-10-26|language=zh-TW}}</ref>

==== JBJ时期 ====
8月30日開始正式加入活動期為7個月的限定團體[[JBJ|JBJ]]<ref>{{cite press release |language = zh-hant |url = http://www.koreastardaily.com/tc/news/97630 |title = JBJ正式宣佈出道！ 9月真人秀、10月發專輯，然而不是七人只有六人...... |publisher = KSD韓星網 |date = 2017-08-30  }}</ref>，於2017年10月18日出道並發表首張迷你專輯《[[Fantasy_(JBJ迷你專輯)|Fantasy]]》。

===2018年：Solo出道、初一位===
4月30日[[JBJ|JBJ]]活動正式結束後，所屬經紀公司 OUI Entertainment 於5月底發布金東漢將會在6月19日以Solo出道<ref>{{cite press release |language = ko |url = http://osen.mt.co.kr/article/G1110914767 |title =[JBJ 김동한 "6월19일 솔로데뷔, 멋있는 모습 위해 열심히 노력 중" |publisher = OSEN |date = 2018-05-31  }}</ref>，專輯將以迷你專輯的形式發行<ref>{{Cite news|url=http://www.nextdaily.co.kr/news/article.html?id=20180508800062|title=JBJ 김동한, 솔로 가수 데뷔 임박...6월 중순 목표로 준비중|newspaper=nextdaily|accessdate=2018-05-08|language=ko}}</ref>。6月8日公開應援色為#91011E和#130004雙色<ref>{{cite web|url=https://pbs.twimg.com/media/DfKCWSGU0AAlPV7.jpg:large|title=Kim Dong Han Official Color Open |publisher=|accessdate=2018-06-08}}</ref>。
