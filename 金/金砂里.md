'''金砂里'''位於[[臺南市|臺南市]][[西港區|西港區]]北部，是西港區最北的里，因過去存在的砂丘地形而得名{{r|縣|page=586}}{{r|西港常民文化|page=49}}{{r|地名}}。這些砂丘的形成與過去[[曾文溪|曾文溪]]（灣裡溪）舊河道流經此地有關，在1823年曾文溪改道後，舊溪床泥沙在東北季風吹襲下形成砂丘{{r|西港常民文化|page=49}}{{r|地名}}。另外也有傳說砂丘是一夕之間從麻豆區大山腳吹來的{{r|縣|page=585}}。過去砂丘分布範圍從新寮、砂凹仔、中社仔到施寮仔，但因為人為因素大多已被剷平{{NoteTag|除剷平砂丘以做為軍事用地和居民生活空間外，也有將砂土載到臺糖三股農場去填地{{r|縣|page=586}}{{r|西港常民文化|page=77}}。}}{{r|縣|page=586}}{{r|西港常民文化|page=77}}{{r|地名}}。

== 沿革 ==
該里在日治時期屬於[[西港庄|西港庄]][[大字|大字]]下宅子，二次大戰後將當地金黃色砂丘地形而命名為「金砂村」{{r|縣|page=585}}{{r|西港常民文化|page=49}}{{r|地名}}。2010年12月25日，因臺南縣市合併升格，改為金砂里。

== 聚落與地名 ==
金砂里境內有砂凹仔、中社仔、施寮仔、新寮、學甲寮仔、下宅仔等聚落{{r|縣|page=586}}{{r|西港常民文化|page=77}}{{r|地名}}。

== 教育 ==
在砂凹仔有後營國小金砂分校，成立於1964年，曾一度於1977年8月獨立成金砂國小{{r|縣|page=587}}。     

== 宗教信仰 ==
該里有砂凹仔金安宮、新寮新安宮、下宅仔進興宮等廟宇{{r|縣|page=587}}。
