'''金在奐'''<ref>{{Cite web |url = https://www.facebook.com/WMG.Taiwan/posts/10154900013006538 |title = 首張迷你專輯《Wanna One》 Teaser Movie #3 金在奐 |author = 華納音樂台灣韓語發行 |date = 2017-07-28 |publisher = [[Facebook|Facebook]] |language = zh-tw |accessdate = 2017-07-29 |quote =  }}</ref>（{{kor|'''김재환'''|rm='''Kim Jae-hwan'''}}，{{lang-en|'''Kim Jae-hwan'''}}；{{bd|1996年|5月27日||}}），韓國男歌手，2016年參加《[[Vocal_戰爭：神的聲音|Vocal 戰爭：神的聲音]]》以出眾的歌唱能力成為首位挑戰成功的參賽者。2017年參加韓國選秀節目《[[PRODUCE_101_第二季|PRODUCE 101 第二季]]》，最終以第4名成為限定男子團體[[Wanna_One|Wanna One]]的成員<ref>[http://www.allthatstar.com/tw/view.php?action=5207&postID=341870 《101》第二季昨晚落幕 「Wanna One」11位成員確定]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，在團內擔任主唱。
[[Wanna_One|Wanna One]]解散後，簽約SWING娛樂以SOLO歌手身分繼續演藝活動。<ref>[http://www.koreastardaily.com/tc/news/97047 金在煥終於結束個人練習生身份！ 簽約CJ E&M]</ref>

==應援色、粉絲名==
2019年1月19日公開應援色、粉絲名。<ref>{{Cite web|url=https://tw.news.yahoo.com/%E9%87%91%E5%9C%A8%E5%A5%90%E6%94%9C%E6%89%8Bswing%E7%B1%8C%E5%82%99solo%E5%87%BA%E9%81%93-%E7%B2%89%E7%B5%B2%E5%AE%98%E5%92%96%E9%A6%96%E6%97%A5%E6%9C%83%E5%93%A1%E8%B6%855%E8%90%AC-%E5%AE%98%E6%96%B9%E6%87%89%E6%8F%B4%E8%89%B2%E5%85%AC%E9%96%8B-000000860.html|title=金在奐攜手SWING籌備SOLO出道 粉絲官咖首日會員超5萬&官方應援色公開！ - Yahoo奇摩新聞|accessdate=2019-01-19|work=tw.news.yahoo.com|language=zh-Hant-TW}}</ref>
<br />金在奐的應援色，是由三種顏色組成的漸變色，左邊的'''[[彩通|彩通]]7702C{{Color sample|#48A9C5}}'''代表「不論何時都會用音樂為大家帶去心靈的安慰和幸福的、唱歌的金在奐」，右邊的'''[[彩通|彩通]]7464C{{Color sample|#A0D1CA}}'''代表「在金在奐身邊添加和聲、為幸福伴奏的粉絲」 ，而中間的'''[[彩通|彩通]]270{{Color sample|#B4B5DF}}'''則代表「連接金在奐和粉絲的，如月光一般的音樂」。
<br />2019年2月3日公開粉絲名：「WIN:D」<ref>[http://cafe.daum.net/KJHOfficial/mEpV/16 參考至金在奐官方FanCafe]{{ko}}</ref>（{{kor|'''윈드'''|rm='''Win Deu'''}}）。 <br />'''1.'''“WIND”是指風，風象徵無論何地，無論何時都會在的意義，而粉絲們就像風一樣，一直在一旁聆聽在奐的音樂，陪伴在奐身旁，一直為在奐應援。
<br />'''2.'''“WIND”前三個字“WIN”有很多的意思，在許多的意思之中有（特別是通過努力而贏得的）勝利這樣的意義，同時最後一個字“D”代表develop，指的是成長、發展。WIN+D 具有通過努力獲得好的成果並持續進步的意義。
<br />'''3.'''風是全世界範圍內無處不在的，希望在奂也可向著全世界前進，也象徵著粉絲們也會像風一樣支持著在奐。
<br />'''4.'''有風才會有聲音，為了讓在奐可以一直唱歌，粉絲們是像風一樣，必不可少的存在。
<br />'''5.'''“WIN:D”中的“:D”是 無論何時彼此看到都會露出笑臉的在奐和粉絲們的模樣。

== 經歷 ==
=== 參加《Korea's Got Talent 2》時期===
2012年，還是高中生的金在奐參加《Korea's Got Talent 2》，第一次進入大眾視野。同年進入Music Works經紀公司成為練習生，以樂團組合ShelRock主唱兼電吉他手準備了4年，但該組合最終解散。<ref>[https://www.youtube.com/watch?v=ZveNccF9pm8 Pretty boy's guitar "Jae-hwan Kim" - Korea's Got Talent 2, 김재환 - 코리아갓탤런트2]</ref><ref>[https://www.youtube.com/watch?v=GY8eGawKDg8&index=2&list=PLRXUaRzQoAGh-uy7QsJKmjhNpIGU8lwe4 Jae-hwan Kim - Korea's Got Talent 2, 김재환 - 코리아갓탤런트2]</ref><ref>[https://www.facebook.com/shelrockofficial/posts/312677418934282 ShelRock_셜록 NEWS]</ref>

=== 參加《Vocal 戰爭：神的聲音》時期===
2016年，金在奐參加《[[Vocal_戰爭：神的聲音|Vocal 戰爭：神的聲音]]》試播，而在節目錄製前4日被當時的經紀公司釋出。金在奐先於與「Rock神」[[尹度玹|尹度玹]]的對戰中取得勝利，成為節目中第一位挑戰成功的參賽者。期後於第二次挑戰時，選唱了[[朴正炫|朴正炫]]的〈[https://www.youtube.com/watch?v=e9NhmhazHgo&list=RDe9NhmhazHgo 對不起]〉，透過加入個人風格的編曲及吉他演奏，再次打動人心。<ref>[http://www.koreastardaily.com/tc/news/77934 從《神的聲音》到《Fantastic Duo》 韓國的素人都這麼會唱嗎？]</ref>

=== 參加《PRODUCE 101 第二季》時期 ===
{{For|《PRODUCE 101 第二季》之節目細節|PRODUCE 101 第二季}}
2017年初，金在奐以個人練習生身份參加大型男團生存實境節目《[[PRODUCE_101_(第二季)|PRODUCE 101 第二季]]》，是101位練習生中沒有隸屬任何[[經紀公司|經紀公司]]的5名練習生其中之一。在為期十週的節目中，金在奐與其他練習生接受一連串唱歌、舞蹈等嚴格訓練並競爭進入前11的出道名次，組成限定團體[[Wanna_One|Wanna One]]並以[[YMC娛樂|YMC娛樂]]旗下藝人身份進行為期1年6個月的活動。

2017年6月16日，金在奐在決賽分組競賽評價中擔任《[https://www.youtube.com/watch?v=wsP8LIHMNOI&t=6s Hands on Me]》的主唱。最終以1,051,735票排名第4，成功成爲[[Wanna_One|Wanna One]]的一員，並於2017年8月7日正式團體出道<ref>[https://www.koreastardaily.com/tc/news/96031 Wanna One敲定出道日期！ 8/7發佈首張迷你專輯+巨蛋開唱！]</ref>，是《[[PRODUCE_101|PRODUCE 101]]》兩季選拔中唯一一位沒有經紀公司背景而勝出的練習生。

金在奐至出道共經歷了4年2個月的練習生生活。

=== 以 SOLO歌手身分 活動 ===
2017年8月金在奐與[[Mnet媒體|CJ ENM]]簽訂合約，待限定團體[[Wanna_One|Wanna One]]解散後,將由[[Mnet媒體|CJ ENM]] 及SWING娛樂共同協助繼續進行演藝活動。<ref>[http://www.allthatstar.com/tw/view.php?action=5205&postID=353350 Wanna One金在奐簽約CJ E&M 計劃2018年正式加盟]</ref><ref>[https://www.koreastardaily.com/tc/news/112948 金在奐經紀約留給 Swing Entertainment　準備以個人歌手身份出道！]</ref>

== 個人生活 ==
金在奐是家中獨子，家庭經濟狀況一般。出道前與父母同住，三人共用一間房間，被問到會否感到擠迫時，金在奐表示：不會，睡在一起的感覺很好。雖然公寓單位空間十分有限，但另有一房間放滿金在奐的[[木吉他|木吉他]]及[[電吉他|電吉他]]，可見其對音樂的熱愛及父母對兒子的支持。<ref>{{Cite web|url=https://kimjaehwan.com/video/%EA%B0%80%EC%9D%84-%EC%9A%B0%EC%B2%B4%EA%B5%AD-%EC%95%9E%EC%97%90%EC%84%9C/|title=《神的聲音》到訪金在奐家中片段|accessdate=|author=|date=2016.02.10|publisher=}}</ref>

金在奐的宗教信仰為[[天主教|天主教]]，於食指上一直戴著一枚聖珠戒指，參加《[[PRODUCE_101_第二季|PRODUCE 101 第二季]]》期間有網民猜測是否與另一半的情侶戒指，但後期於節目播出的花絮中確認是在教會服務長達7年後獲得的聖珠戒指，並透露[[受洗|受洗]]名為「馬里奧」。<ref>{{Cite web|url=https://www.youtube.com/watch?v=Ee6ejnDQtCk|title=於《PRODUCE 101 第二季》中回答網民提問片段（3:02-3:08）|accessdate=|author=|date=2017.06.16|publisher=}}</ref>
