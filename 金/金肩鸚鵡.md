'''金肩鸚鵡'''（[[学名|学名]]：''{{lang|la|Psephotus chrysopterygius}}''）是[[澳洲|澳洲]][[昆士蘭|昆士蘭]][[約克角半島|約克角半島]]南部的稀有[[鸚鵡|鸚鵡]]。牠們細小及色彩鮮艷，是[[紅腰鸚鵡|紅腰鸚鵡]]的近親，曾被認為是[[異色金肩鸚鵡|異色金肩鸚鵡]]及[[天堂長尾鸚鵡|天堂長尾鸚鵡]]的[[超種|超種]]。

== 特徵 ==
金肩鸚鵡長23-28厘米及重54-56克。雄鳥主要呈藍色，肩上有明顯的黃色。冠黑色，前額有淡黃色的帶。下腹、大腿及下尾粉紅色，下背部呈灰褐色。雌鳥主要呈較沉的綠黃色，翼下有一奶白色闊帶。雛鳥與雌鳥相似。<ref>{{cite web| publisher =BirdLife International (2008) |url = http://www.birdlife.org/datazone/species/index.html?action=SpcHTMDetails.asp&sid=1473&m=0 |title = Species factsheet: ''Psephotus chrysopterygius'' | accessdate = 10-07-2013}}</ref>

== 棲息地 ==
金肩鸚鵡棲息在遼闊的[[森林|森林]]，以[[草|草]]的[[種子|種子]]為食物。[[白蟻|白蟻]]丘的存在影響著牠們選擇棲息地，因牠們會在丘中築巢。

== 繁殖 ==
金肩鸚鵡會在[[白蟻|白蟻]]丘中築巢，建築隧道。牠們每次會生3-6隻蛋，孵化期為20天。由於丘內的[[溫度|溫度]]受到控制，故牠們有時會丟下蛋不顧。

== 保育狀況 ==
金肩鸚鵡被列為[[瀕危|瀕危]]，且受到《[[瀕危野生動植物種國際貿易公約|瀕危野生動植物種國際貿易公約]]》附錄一的保護。牠們受到[[野貓|野貓]]的掠食、旅客騷擾及失去棲息地所影響。野生群落約有3000隻，在[[澳洲|澳洲]]正在飼養著1500隻。
