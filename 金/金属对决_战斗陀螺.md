《'''戰鬥陀螺 鋼鐵奇兵'''》（{{lang-ja|メタルファイト ベイブレード}}，英文：Metal Fight Beyblade/Beyblade: Metal Fusion）是[[TAKARA_TOMY|TAKARA TOMY]]玩具公司为推销新系列战斗陀螺玩具而制作的动画，其正篇于2009年4月5日在[[东京电视台|东京电视台]]播映。它和之前放送的战斗陀螺系列动画完全不同。日本在2010年4月4日接續播出第二部，以《戰鬥陀螺 鋼鐵奇兵 爆》（日文：{{lang|ja|メタルファイト ベイブレード 爆}}）命名播出。而第三部則於2011年4月3日播出，以《戰鬥陀螺 鋼鐵奇兵 4D》（日文：{{lang|ja|メタルファイト ベイブレード 4D}}）命名。

在香港，玩具由萬信有限公司代理。在台湾，这部动画被译作《戰鬥-{}-陀螺 鋼鐵奇兵》，玩具由丽婴国际代理。而在欧美地区，该片的播出也同样引起不少战斗陀螺爱好者关注。

2009年6月6日[[动画新闻网|动画新闻网]]宣布了[[加拿大|加拿大]]{{lang|en|[[:en:_Nelvana|NELVANA]]}}公司引进此片的消息，注册名为“'''Beyblade: Metal Fusion'''”，预计将于2010年在[[北美地区|北美地区]]播放<ref>参照[http://www.animenewsnetwork.com/news/2009-06-06/beyblade/metal-fusion-to-debut-in-u.s-in-fall-2010]与{{cite web |url=http://news.awn.com/index.php?ltype=top&newsitem_no=27947 |title=存档副本 |accessdate=2015-04-18 |0=2009-09-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20090903120241/http://news.awn.com/index.php?ltype=top |archivedate=2009-09-03 }}.</ref>。[[香港|香港]]於2010年1月7日起逢星期四及五17:15~17:45於[[翡翠台|翡翠台]]播出、2011年1月14日起逢星期四及五17:20~17:50於[[翡翠台|翡翠台]]播出第二期﹙爆），2012年1月20日起逢星期四及五17:20~17:50於[[翡翠台|翡翠台]]播出第三季﹙4D）。[[台灣|台灣]]於2010年5月7日起逢星期五及六18:00~18:30在[[卡通頻道|卡通頻道]]播放（同年8月9日改於只有星期六同時段播出）。

== 概要 ==
原作漫画由{{lang|ja|足立たかふみ}}在2008年10月号的月刊{{lang|ja|[[快乐龙|コロコロコミック]]}}上开始连载。由于人气不断的提升，在连载至第五话时决定游戏化，并于09年3月（漫画连载第六话）放出即将动画化的消息。

本作是在《战斗陀螺》系列作品完结后再度展开的新系列故事。前作中将[[四圣兽|四圣兽]]作为故事题材与战斗陀螺结合起来，而本作则是使用[[星座|星座]]或神話作为题材与战斗陀螺的故事相融。

漫画中的1～7话被分为“第一部”，第二部由第8话开始，小标题命名为“'''{{lang|ja|バトルブレーダーズ}}篇'''”（Battle Blader，[[台灣|台灣]][[青文出版社|青文出版社]]翻譯「戰鬥陀螺風雲會」）。

== 故事 ==
'''第一季'''
:* 第1集～第23集
:为了变得更强而正在旅行的少年钢银河，在偶然的机会下帮助并认识了同样热爱战斗陀螺的小学生汤宫健太。此后两人以战斗陀螺为缘由不断认识着更多的朋友与敌人，共同体验着陀螺对战带来的各种乐趣与磨练，并逐渐成长起来。（对应漫画：第1话～第7话）
:* 第24集～第51集　陀螺戰士全國大會篇
:由邪恶组织暗黑星云举办的「陀螺戰士全國大會」大会正如火如荼地进行中。银河以打败龍牙为自己的目标，在大会中不断地向各地的陀螺高手挑战。（对应漫画：第8集）
'''第二季（爆）'''
:銀河在遺跡解開了封印後，得到了銀河天馬，而之後正在學習如何掌握銀河天馬。後來在銀河面前出現了一個自稱是世界第一最強陀螺手的角谷正宗，初時他不斷向銀河挑戰。之後他便使出了真正實力將銀河天馬打敗。而其後銀河便與正宗、遊和翼代表日本前往世界各地參加世界盃大賽，挑戰來自世界各地的陀螺高手。
'''第三季（4D）'''
:宇宙中發生了一件不尋常事件。某一天，銀河們幫助著被神秘少年約翰尼斯襲擊的少年勇氣。勇氣是天才的天文學家「水星阿努比斯」陀螺持有者。勇氣通知銀河說聽到星的碎片傳出聲音。他說當晚目睹到天空落下了星的碎片(隕石)。有一道光指向著「水星阿努比斯」住所。據說他聽到星的碎片裡傳出聲音說：「即將有很大的災難………。」

== 登场战斗陀螺 ==
片中登场战斗陀螺，每个陀螺都有相對的[[星座|星座]]对称。其構造主要由纹章、輪盘、轴心、底盤四个部分组成，有些的轮盘分為兩部分，共五個部分。并分为攻击（Attack）、持久（Stamina）、防禦（Defense）和平衡（Balance）这四大类型。陀螺之间还能够根据需要来进行部件交换，从而改变其性能<ref>动画第十九话参照. </ref>。大部分的战斗陀螺都是右回转，然而有些则不然。主角们使用的陀螺也随着动画和原作的不同创作而有所变化。第四季，ZERO-G不再以星座為主題，而是改用了幻獸為主題，名稱組合上是以「職業+幻獸+底盤」，另外加上屬性。

== 用语 ==
* '''WBBA'''
:举办战斗陀螺大型比赛的組織，全称“World Beyblade Battle Association”。

* '''WBBA大会'''
:由WBBA定期举办的战斗陀螺大型比赛。

* '''Battle Bladers大会'''
:由暗黑星云举办的战斗陀螺大型比赛。

* '''战斗陀螺'''
:战斗陀螺是一种可供两人或多人对战用的玩具。详细介绍参照[[#登场战斗陀螺|登场战斗陀螺]]部分。

* '''积分对战'''
:积分对战是片中特有的一种对战方式。当两名选手进行陀螺对战后，输掉的人要给予对手10分，而这些都是强制的。积分通过一种叫做BeyPointer的装置获得。

* '''BeyPointer（{{lang|ja|ベイポインター}}）'''
:在陀螺对战中用来记录分数的小型仪器，颜色以黑色为主。它所记录的分数会根据使用者的对战结果增加或减少，負分時會加回0-120分和記GAME OVER一次。

* '''造分'''
:造分是战斗陀螺的作弊行為，战斗陀螺手使用其中一部BeyPointer來不斷给予其中一部BeyPointer 10分直至其中一部0分。

* '''Launcher Grip（{{lang|ja|ランチャーグリップ}}）'''
:用来安装战斗陀螺的装置，它们通常被佩戴在陀螺手的腰上。

* '''Challenge Blader（{{lang|ja|チャレンジブレーダー}}）'''
:Challenge Blader是动画中虚构的一个面向全国观众绝赞放映中的节目，由Merci主持，它要求参与该节目的陀螺手挑战陀螺闯关的游戏。游戏一共有十个关卡，而陀螺手在赢得对战后还可以获得精美的礼物<ref>动画第八话参照. </ref>。

* '''组合对战（{{lang|ja|タッグバトル}}）'''
:由WBBA举办的以挑战各种水上场地为主题的比赛，进行挑战的陀螺手必须要充分了解陀螺部件的功能并能够熟练地进行组合交替。

* '''紋章首狩者（{{lang|ja|フェイスハンターズ}}）'''
:是片中虚构的一个不良少年集团，约有100余人组成。在遇到盾神京谷以前由弁庆带领，平时喜欢通过一些不正规的比赛手段抢夺弱小的陀螺选手的积分，据点是工地上废弃的仓库。他们有一个共同的特征，就是大部分成员的腰上都佩戴着一个装饰着黄色字母“'''F'''”的Launcher Grip<ref name="multiple">动画第三话参照. </ref><ref name="multiple1">动画第四话参照.</ref>。

* '''狼谷（{{lang|ja|ウルフキャニオン}}）'''
:暗黑星云所属的自然公园，通称狼谷。它荒无人烟，可以将大自然真实的再现。

* '''古马村（{{lang|ja|コマむら}}）'''
:古马村是漫画中虚构的一个村庄，人称“战斗陀螺之乡”（{{lang|ja|ベイブレードのさと}}），也是银河及流星的故乡。禁断的战斗陀螺L Drago亦被封印于此。

* '''钢山（{{lang|ja|鋼山}}）'''
:钢山是片中虚构的一座山，与古马村相邻。在动画的设定中，禁断的陀螺Lightning L Drago就被封印在这里<ref>动画第十四话参照. </ref>。

* '''彷徨之山'''
:彷徨之山是动画中虚构的一座山，环境险峻道路复杂。在这里隐藏着通往古马村的必经之路。

* '''绿色地狱（{{lang|ja|緑の地獄}}）'''
:是古马村独有的以綠曜石为原材料构成的陀螺对战场地，通称“绿色的地狱”。它的表面如同[[玻璃|玻璃]]一般光滑。

* '''陀螺之森（{{lang|ja|ベイの森}}）'''
:陀螺之森是动画中虚构的一座森林，位于古马村附近。这里有很多练习战斗陀螺的绝佳场地，如“妖怪树墩”与“陀螺渡河”等。

* '''暗黑星雲'''
:故事中(活躍於第一季)一個虛構的邪惡組織,擁有先進的科技技術及設備以訓練陀螺手和改進陀螺,主要領導人為龍牙與大道寺,企圖利用爆旋陀螺(雷霆魔龍)潛藏的強大力量支配整個世界。不過實際操緃為HD研究院的捷古拉特博士,大道寺也是被其派往日本協助龍牙獲得並運用「雷霆魔龍」替其實現操緃世界的野心。換句話說,「暗黑星雲」只是用作獲取黑暗力量的空殼組織。

* '''HD研究院'''
:美國最大的陀螺研究研究院，培育美國代表破星者（碎星者），院長為捷古拉特博士。

* '''非法配件'''
:只有在战斗陀螺玩具中出現。是指非官方認可的配件，而是其他人把配件以非正常方式改造，多數來自中國內地，火花環是其中之一。另外也有些是陀螺手非法改造出來的，例如多重軸心。
