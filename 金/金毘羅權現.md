'''金毘羅權現'''（{{lang|ja|こんぴらごんげん}}）是[[琴平山_(香川縣)|琴平山]]（象頭山）的-{[[山岳信仰|山岳信仰]]}-和[[修驗道|修驗道]]融合的[[神佛習合|神佛習合]]之神，本地佛是[[十一面觀音|十一面觀音菩薩]]。神佛習合中是[[香川縣|香川縣]][[琴平町|琴平町]]象頭山鎮座的'''金毘羅大權現'''，[[神佛分離|神佛分離]]・[[廢佛毀釋|廢佛毀釋]]以前以[[讚岐國|讚岐國]]象頭山松尾寺金光院（現在香川縣琴平町的[[金刀比羅宮|金刀比羅宮]]）為總本宮，祭祀在日本全國的[[金毘羅宮|金毘羅宮]]、金毘羅權現社。

== 由來 ==
根據象頭山松尾寺<ref>[http://www.mandala88.com/88/10/16/16.htm 新四國曼荼羅靈場ホームページ - 香川部會 - 新四國曼荼羅靈場第十六番札所]</ref>的緣起，[[大宝_(日本)|大寶]]年間，[[修驗道|修驗道]]的[[役小角|役小角]]（神變大菩薩）登象頭山之際，因[[印度|天竺]][[靈鷲山_(印度)|靈鷲山]]（象頭山）的護法善神金毘羅（[[宮比羅|宮比羅]]）的神驗而開山，所以稱作'''象頭山金毘羅大權現'''。象頭山金毘羅大權現的本地佛是[[十一面觀音|十一面觀音菩薩]]。

[[摩伽羅|金毘羅（摩伽羅）]]本來是[[恆河|恆河]][[鱷|鰐]]神格化的水神，在日本是[[蛇|蛇]]型。金毘羅（摩伽羅）也是[[恆河女神|恆河女神]]的坐騎，所以，金毘羅權現作為海上交通的守護神受到信仰。在全國各地俯瞰大港的山上建有金毘羅宮、金毘羅權現社，祭祀金毘羅權現。

== 崇德天皇合祀 ==
[[長寬|長寬]]元年（1163年），[[崇德天皇|崇德天皇]]在象頭山松尾寺金光院参籠<ref>[http://www.konpira.or.jp/about/history/main/history.html 金刀比羅宮：ノート] {{webarchive|url=https://web.archive.org/web/20120625131107/http://www.konpira.or.jp/about/history/main/history.html |date=2012-06-25 }}</ref>，所以，受到[[修驗道|修驗道]][[御靈信仰|御靈信仰]]的影響，[[永萬|永萬]]元年（1165年），崇德天皇也合祀象頭山松尾寺金光院<ref>[http://www.konpira.or.jp/about/history/sutoku/sutoku_01.html 金刀比羅宮 - 崇德天皇] {{webarchive|url=https://web.archive.org/web/20120126113124/http://www.konpira.or.jp/about/history/sutoku/sutoku_01.html |date=2012-01-26 }}</ref>。

== 天狗信仰 ==
[[File:Tokaido12_Numazu.jpg|thumb]]『[[東海道五十三次|東海道五十三次]]・沼津』]]
修驗道盛行後，[[天狗_(日本)|天狗]]被視為是金毘羅權現的[[眷屬|眷屬]]。『[[和漢三才圖會|和漢三才圖會]]』記載「當山之天狗名為金比羅坊」。

[[江戶時代|江戶時代]]，背負天狗之面的白裝束的金毘羅道者（行人）在全國巡迴，使金毘羅信仰普及<ref> 『金毘羅信仰｜民衆宗教史叢書　第十九巻』, 守屋毅編, 雄山閣, ISBN 978-4639006657</ref>。也產生背負天狗之面前來参拜讚岐國象頭山松尾寺的習俗<ref>浮世絵『東海道五十三次・沼津』, 歌川廣重</ref><ref>『[[東海道中膝栗毛|東海道中膝栗毛]]』</ref>。

== 金毘羅参拜 ==
[[塩飽諸島|塩飽水軍]]信仰金毘羅權現，對全國寄港地的金毘羅信仰普及帶來貢献。

江戶時代後期，伴隨象頭山松尾寺金光院的金毘羅参拜盛行，[[四國|四國]]的[[丸龜市|丸龜]][[街道|街道]]、多度津街道、[[高松市|高松]]街道、[[阿波國|阿波]]街道、[[伊予國|伊予]]・[[土佐國|土佐]]街道等[[金毘羅街道|金毘羅街道]]受到整備。神佛分離以降，香川縣琴平町的[[金刀比羅宮|金刀比羅宮]]是全國約600座金刀比羅神社的總本宮。[[緣日|緣日]]是毎月10日。

== 神佛分離・廢佛毀釋 ==
[[明治|明治]]元年（1868年）3月，神佛分離令造成象頭山松尾寺金光院廢寺<ref>[http://www.saidaiji.jp/html/kannon/konpira.html 金毘羅大權現 - 西大寺觀音院（岡山縣西大寺）] {{webarchive|url=https://web.archive.org/web/20110209083610/http://www.saidaiji.jp/html/kannon/konpira.html |date=2011-02-09 }}</ref>，強制改組為[[國家神道|國家神道]]的琴平神社（同年7月改稱'''金刀比羅宮'''），主祭神定名為[[大物主神|大物主神]]，相殿奉祀崇德天皇。

全國的金毘羅宮・金毘羅權現社大多改成主祭神是大物主神的[[神道|神道]]的[[金刀比羅神社|金刀比羅神社]]・琴平神社・金比羅神社。
