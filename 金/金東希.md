'''金東希'''
（{{lang-ko|김동희}}，{{bd|1999年|6月13日}}），[[韓國|韓國]][[男演員|男演員]]，曾譯為「金東熙」。原為[[JYP娛樂|JYP娛樂]]旗下演員練習生，因戲劇處女作《[[A-TEEN_(電視劇)|A-TEEN]]》表現亮麗而簽訂正式演員契約。<ref>{{cite web |title= '에이틴' 김동희, JYP와 전속계약 체결 "성실한 태도+발전 가능성 높이 사"|url=http://stoo.asiae.co.kr/article.php?aid=49549855419 |website= 스포츠투데이 |accessdate=2018-9-21 |language=ko}}</ref>

== 生涯 ==
出生在[[慶尚北道|慶尚北道]][[安東市|安東市]]，血型AB型，最愛的顏色是藍色，喜歡披薩，能獨自吃完一整盤，對海鮮跟芒果過敏。家中獨子，因未與家人同住而常電話聯絡，以免父母擔心。

喜愛音樂與電影，在閒暇時常唱卡拉OK或看電影，國中時曾是樂團成員，嚮往成為歌手而就讀藝高。高中二年級時因為好奇而參加校內劇場表演才開始成為演員，進而成為[[JYP娛樂|JYP娛樂]]旗下練習生，因此也孕育了演員的夢想。

在還未累積許多演出經驗下通過熱門網路劇《[[A-TEEN_(電視劇)|A-TEEN]]》選角並以此劇出道，隨後演出高收視率電視劇《[[Sky_Castle|Sky Castle]]》，接連演出兩部當時的熱門戲劇作品而知名度漸開。享受音樂與表演，所以視男演員[[曹承佑|曹承佑]]為人生榜樣，想在音樂劇、戲劇和電影等領域全方位發展。<ref>{{cite web|title= 차교수를 아버님이라 부르고픈 사람들을 위한 서준 가이드.txt|url=https://1boon.kakao.com/newsade/kimdh |website=1boon |accessdate=2019-1-4 |language=ko}}</ref><ref>{{cite web |title= [인터뷰②] 김동희 ”'SKY캐슬' 결말? 남은 2회 충격+기대 함께할 것”|url=http://isplus.live.joins.com/news/article/article.asp?total_id=23322658&ctg=1502|website=joins |accessdate=2019-1-25|language=ko}}</ref>
