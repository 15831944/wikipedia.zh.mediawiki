'''金城湖'''，又名'''金城湖賞鳥區'''，原稱'''魔鬼潭'''、'''魔鬼湖'''，位於[[台灣|台灣]][[新竹市|新竹市]][[香山區|香山區]][[客雅溪|客雅溪]]出海口北岸的半天然湖，為[[新竹海埔新生地|新竹海埔新生地]]調節水位之用，現則是港南里農田主要排水出口、[[賞鳥|賞鳥]]區<ref>{{cite web |url=http://17km.hccg.gov.tw/ki13.htm |title=新竹市17公里海岸線主題網 金城湖賞鳥區 |publisher=17km.hccg.gov.tw |date= |accessdate=2014-08-10 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、[[香山濕地|香山濕地]]動物保護區<ref>{{cite web|url=http://conservation.forest.gov.tw/ct.asp?xItem=59988&ctNode=178&mp=10|title=新竹市濱海野生動物保護區|date=|accessdate=2014-08-10|deadurl=yes|archiveurl=https://web.archive.org/web/20140812213122/http://conservation.forest.gov.tw/ct.asp?xItem=59988&ctNode=178&mp=10|archivedate=2014-08-12}}</ref>，為[[十七公里海岸線|十七公里海岸線]]的景點之一。

== 名字 ==
[[File:王金城先生殉職紀念碑.JPG|300px]]
昔日有民眾以為水淺而涉水喪命，於是稱為魔鬼湖<ref name="金城湖賞鳥區—農田排水口成賞鳥勝地">{{cite web|url=http://www.walkingtaiwan.org/content/county/city_POI.asp?ids=485&jid=74&bid=7|title=金城湖賞鳥區—農田排水口成賞鳥勝地|publisher=Walkingtaiwan.org|date=|accessdate=2014-08-10}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。附近[[新竹海埔新生地|海埔地]]入口處有退輔會興建的金城先生紀念園，防潮堤則有榮工處興建的王金城先生殉職紀念碑，前者乃紀念[[非洲|非洲]]考察因交通事故喪生的[[金城_(農學家)|金城]]教授，後者為紀念在築堤時遭海水沖入海中的王金城。因兩處紀念皆有「金城」兩字，久而久之便將原稱改喚為金城湖<ref>{{cite web|url=http://okgo.tw/buty/02087.html |title=金城湖賞鳥區(魔鬼潭) |publisher=Okgo.tw |date=2014-04-03 |accessdate=2014-08-10}}</ref>。

== 生態 ==
湖面積約五公頃，含生態觀賞區則為三百五十公頃。湖位於整區的核心位置，中央的沙洲提供[[鷺科|鷺科]]、[[鷸科|鷸科]]及[[雁形目|雁形目]]的棲身場所，加上海埔新生地所形成的潮間帶海濱生物豐富，更成為鳥類的天然五星級餐廳，每逢漲、退潮，許多鳥類便飛來停歇覓食，達兩百多種，有[[蒼鷺|蒼鷺]]、[[小水鴨|小水鴨]]、[[赤足鷸|赤足鷸]]、[[黃尾鴝|黃尾鴝]]等，是為港南最重要的賞鳥據點<ref>{{cite web|url=http://people.chu.edu.tw/~b9017037/new_page_e8.htm |title=金城湖 |publisher=People.chu.edu.tw |date= |accessdate=2014-08-10}}</ref><ref>{{cite web|url=http://www.cc.chu.edu.tw/~b8917056/golden.htm |title=香山全記錄 金城湖 |publisher=Cc.chu.edu.tw |date= |accessdate=2014-08-10}}</ref><ref name="金城湖賞鳥區—農田排水口成賞鳥勝地"></ref>。
