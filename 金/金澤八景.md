'''金澤八景'''（{{jpn|j=金沢八景|hg=かなざわ はっけい|rm=Kanazawa Hakkei}}、古稱：{{jpn|hg=かなさわ はっけい|rm=Kanasawa Hakkei|link=no}}）是[[日本|日本]][[武藏國|武藏國]]倉城郡（現・[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]][[金澤區|金澤區]]）中選定的「[[八景|八景]]」。現在「金澤八景」作為地名使用。因為[[都市計畫|都市開發]]的影響已喪失昔日的面貌<ref name=Bunko>『金沢八景:歴史・景観・美術』神奈川県立金沢文庫編 1993年</ref>。

== 八景 ==
* '''小泉夜雨'''（{{jpn|link=no|hg=こずみ の やう|rm=Kozumi no yau}}） - 手子神社（小泉[[辯財天|辯財天]]）
* '''稱名晩鐘'''（{{jpn|link=no|hg=しょうみょう の ばんしょう|rm=Shōmyō no banshō}}） - [[稱名寺_(橫濱市)|稱名寺]]
* '''乙艫帰帆'''（{{jpn|link=no|hg=おっとも の きはん|rm=Ottomo no kihan}}） - [[海之公園|海之公園]]往内陸的寺前地區的舊海岸線
* '''洲崎晴嵐'''（{{jpn|link=no|hg=すさき の せいらん|rm=Susaki no seiran}}） - 洲崎神社
* '''瀨戶秋月'''（{{jpn|link=no|hg=せと の しゅうげつ|rm=Seto no Shōgetsu}}） - [[瀨戶神社|瀨戶神社]]
* '''平潟落雁'''（{{jpn|link=no|hg=ひらがた の らくがん|rm=Hiragata no rakugan}}） - [[平潟灣|平潟灣]]
* '''野島夕照'''（{{jpn|link=no|hg=のじま の せきしょう|rm=Nojima no sekishō}}） - [[野島_(神奈川縣)|野島]]夕照橋付近
* '''内川暮雪'''（{{jpn|link=no|hg=うちかわ の ぼせつ|rm=Uchikawa no Bosetsu}}） - 內川入江（能見堂的解釋）或瀨崎到九覽亭的平潟灣（金龍院的解釋）<ref name=Bunko/>

== 現在 ==
現在的金澤八景一帶是[[橫濱市立大學|橫濱市立大學]]、[[關東學院大學|關東學院大學]]等設施之所在地，是有名的學生街。又，[[京濱急行電鐵|京濱急行電鐵]][[金澤八景車站|金澤八景車站]]是[[京急本線|本線]]和[[京急逗子線|逗子線]]的分岐車站。
