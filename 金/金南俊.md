2017-12-28}}</ref>。}}）。韓國男藝人，為[[韓國|韓國]][[男子團體|男子團體]][[防彈少年團|防彈少年團]]的隊長，在團內擔任主Rapper<ref>[https://tw.news.yahoo.com/video/bts-%E9%98%B2%E5%BD%88%E5%B0%91%E5%B9%B4%E5%9C%98-%E8%87%AA%E6%88%91%E4%BB%8B%E7%B4%B9-rapmonster-091132750.html == 早年生涯 == 
:<small>— 整個段落大部分參考於多人傳記書籍《做嘻哈1：韓國，以及嘻哈》。</small>
1994年9月12日出生於韓國[[京畿道|京畿道]][[高陽市|高陽市一山]]，但是四歲之前在[[首爾|首爾]]生活，四歲時舉家搬至一山定居。2006年，接觸[[Epik_High|Epik High]]、{{link-ko|Tiger JK|타이거 JK}}、[[Eminem|阿姆]]等歌手，從此啟發對[[嘻哈|嘻哈]][[饒舌|饒舌]]的興趣。2007年國一時第一次嘗試寫歌詞。2008年國二開始在地下活動，進行許多公開表演，並且曾與[[Zico_(歌手)|Zico]]等人合作。同年，參加韓國UMF（Ultra Music Festival）落選，認知自己的表演技巧並不優秀。2009年參加Big Deal Records徵選初試合格，複試失敗，但是因此結識韓國嘻哈團體[[Untouchable_(南韓組合)|Untouchable]]的團員Sleepy。同年5月，在Sleepy和Big Hit娛樂製作人Pdogg的牽線之下，<ref name=hiphopplaya2>{{Cite web|url =https://hiphopplaya.com/magazine/16604|title =(HIPHOPPLAYA) Rap Monster: ‘even if it’s to say bad things, i really hope you would listen to the mixtape’|date =2015-03-24|language =en|deadurl =yes|archiveurl =https://web.archive.org/web/20151117164254/http://hiphopplaya.com/magazine/16604|archivedate =2015-11-17}}</ref>前往[[江南區_(首爾)|江南]]見到對他有興趣的Big Hit娛樂創辦人[[房時爀|房時爀]]。房時爀因為他而產生組成防彈少年團的基本構想<ref>{{Cite web |url =http://entertain.naver.com/read?oid=311&aid=0000674214|title =[XP인터뷰①] 방시혁 "방탄소년단, '기적'도 '운'도 아닙니다"|date =2016-11-18|language =ko}}</ref>，不過當時的構想是創立純嘻哈饒舌團體，而非偶像團體。{{r|hiphopplaya2}}由於當時剛好是得知自己模擬考試成績是全國前1%，可以上韓國[[首爾大學|首爾大學]]的時候，他猶豫是要繼續升學還是追逐夢想。但是聽了房時爀的構想和承諾，他馬上答應了，不過遭到父母反對。後來他說服了父母，進入Big Hit娛樂當練習生，成為防彈少年團第一位練習生，也是搬入宿舍的第一位練習生。練習生時期接觸到[[San_E|San E]]的〈Rap Genius〉，練習時因「Call me
rap monster cuz I rap non stop」這句歌詞而被公司開玩笑取了「Rap Monster」的綽號，之後討論出道藝名，遂沿用綽號取名為「Rap Monster」。

== 音樂生涯 ==
2015年3月20日，發行首張個人Mixtape《RM》<ref>{{Cite web |url =http://kpopn.niusnews.com/=K19obob7|title =Rap Mon《Do You》MV|date =2015-03-20|language =中文}}</ref>。此張Mixtape於年末被美國音樂媒體SPIN評選為「2015年最佳嘻哈專輯TOP 50」，使得他成為首位進入該評選的韓國歌手<ref>{{Cite web|url =http://www.allthatstar.com/tw/view.php?action=5212&postID=187667|title =Rap Monster入選美媒體評選2015最佳嘻哈專輯前50|date =2015-12-18|language =中文|access-date =2016-09-10|archive-url =https://web.archive.org/web/20160826001755/http://www.allthatstar.com/tw/view.php?action=5212&postID=187667#|archive-date =2016-08-26|dead-url =yes}}</ref>。

2018年10月23日，RM發行個人playlist《mono.》，登上全世界共90個地區的iTunes排行榜1位，成為韓國歌手最高紀錄。<ref>{{Cite web |url =https://www.billboard.com/articles/news/bts/8481015/rm-mono-mixtape-forever-rain-video|title =BTS' RM Releases 'Mono' Mixtape and 'Forever Rain' Music Video: Watch|date =2018-10-23|language=en}}</ref><ref>{{Cite web |url =http://www.newsen.com/news_view.php?uid=201810231225500410|title =방탄소년단 RM ‘mono.’ 아이튠즈 88개국 1위 ‘K팝 신기록’[뮤직와치]|date =2018-10-23|language=ko}}</ref>

2018年10月29日(美國時間)，個人playlist《mono.》登上[[Billboard_200|Billboard 200]]第26位、Artist 100排行第41名，刷新了韓國SOLO歌手的紀錄。<ref>{{Cite web |url =https://www.billboard.com/articles/columns/k-town/8482362/bts-rm-billboard-200-chart-mono|title =BTS' RM Enters Billboard 200 With 'Mono'|date =2018-10-29|language =en}}</ref>

2018年12月5日，[[告示牌_(雜誌)|美國告示牌]]公布年末榜單，RM憑藉個人playlist《mono.》分別登上World Album第9位及World Album Artist第7位。<ref>{{Cite web |url =https://www.billboard.com/articles/events/year-in-music-2018/8488875/year-in-k-pop-charts-bts-exo-blackpink-nct-127|title =The Year in K-Pop on the Charts: BTS Breaks Genre Barriers While the Scene Expands Its Reach|date =2018-12-06|language =en}}</ref>
