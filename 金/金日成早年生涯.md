'''金日成的早年生涯'''是指[[金日成|金日成]]（原名金成柱）1912年4月15日出生到1945年9月19日乘坐[[苏联|苏联]]军舰回到[[朝鲜|朝鲜]]这段时间的生活。

== 家世 ==
{{See also|金亨稷|康盘石}}
1910年，[[大日本帝国|大日本帝国]]吞并朝鲜。{{r|duck|page=1}}{{r|rat|page=30}}<ref name=" fish" >{{cite web |url=http://news.ifeng.com/history/zhongguojindaishi/detail_2011_12/25/11553657_0.shtml |title= 与金日成有特殊友谊的中国将领：死后金暗自流泪|accessdate=2011年12月25日 |publisher=凤凰网 |work=宋凤英 |language =zh-cn}}</ref>

1912年4月15日，金成柱出生在[[朝鮮日治時期|日治朝鮮]][[平安南道_(日本統治時代)|平安南道]][[平壤府|平壤府]][[古順和面|古順和面]]<ref>{{cite web |url=http://theme.archives.go.kr/viewer/common/archWebViewer.do?singleData=Y&archiveEventId=0049287462 |title=조선행정구획도 |date=1911-04 |website=國家記錄院 |publisher=朝鮮駐劄憲兵隊司令部 |access-date =2017-04-12 |language = ko}}</ref><ref>{{cite web |url=http://theme.archives.go.kr/viewer/common/archWebViewer.do?singleData=Y&archiveEventId=0049287614 |title=대동군 |date=1914 |website=國家記錄院 |publisher= |access-date =2017-04-12|language = ko}}</ref>（今[[平壤市|平壤市]]万景台）的一个农民家。{{r|duck|page=1}}{{r|rat|page=30}}<ref name=" fish" /> 金成柱的父亲叫“[[金亨稷|金亨稷]]”，为朝鲜爱国人士，曾经因反抗大日本帝国殖民统治而被逮捕羁押；金成柱的母亲叫“[[康盘石|康盘石]]”，为抗日斗士和[[妇女解放运动|妇女解放运动]]先驱。{{r|duck|page=1-2}}{{r|rat|page=30}}<ref name="man " >{{cite web |url=http://book.ifeng.com/shuhua/detail_2008_09/10/337578_0.shtml |title= 金日成的中国抗日生涯 建立朝鲜人民革命军|accessdate=2008年9月10日  |publisher= 凤凰网|work= |language =zh-cn}}</ref>

1917年，大日本帝国对日治朝鲜实行更加苛刻严酷的治理政策，诸多反日人士都被清理，部分人士幸存逃离国土来到[[中华民国|中华民国]]东北地区。{{r|duck|page=2-3}}{{r|rat|page=31}}

1920年，金亨稷带着家人搬迁至[[吉林省|吉林省]][[临江县|临江县]]。{{r|duck|page=3}}{{r|rat|page=31}}<ref name=" fish" /> 此时，金亨稷结识了吉林抚松县绅士张万程，张万程的儿子[[张蔚华|张蔚华]]{{noteTag|1=张蔚华，张万程的儿子，吉林抚松县人，曾出资资助金日成的革命事业，1937年10月27日在给金日成报信时牺牲，有大儿子张琅、二儿子张金泉、女儿张金禄。 }} 比金成柱小一岁，两人都在抚松县第一小学读书，并且结为至交。{{r|duck|page=4-10}}

== 读书 ==
1926年年初，金成柱到吉林[[毓文中学|毓文中学]]读书，后来加入[[朝鲜共产青年同盟|朝鲜共产青年同盟]]，{{r|rat|page=32}}<ref name=" fish" /> 同时，金成柱组织建立了秘密读书会，宣传[[马克思列宁主义|马克思列宁主义]]思想和革命武装抗日。{{r|rat|page=32}}6月5日，金亨稷因病逝世。{{r|rat|page=32}}金成柱继承父亲遗志，办起了刊物，创办了朝鲜第一份革命刊物《红日》。{{r|rat|page=33}}10月17日，金成柱在[[桦甸|桦甸]]建立了“打倒帝国主义同盟”。{{r|rat|page=33}}<ref name="man " />

1929年10月，金成柱因为刻印反日报刊而被[[奉系|奉系]][[军阀|军阀]][[张学良|张学良]]抓捕，1930年5月，金成柱经过张万程的积极营救被救出。{{r|duck|page=100}}<ref name="man " />

== 革命 ==
{{See also|张蔚华|杨靖宇|普天堡大捷}}
1930年6月30日，金成柱在吉林[[卡伦|卡伦]]举办的“朝鲜革命者会议”上和众多革命者制定了武装斗争的方针。{{r|rat|page=34}}1930年秋季，金成柱得到张万程的资助，购买了12支手枪。{{r|duck|page=102}}

1931年9月18日，大日本帝国制造了[[九一八事变|九一八事变]]后，在东北被扶持建立[[满洲国|满洲国]]，张万程再次给金成柱购买了40支步枪。{{r|duck|page=105}} 金成柱加入[[中国共产党|中国共产党]]，参加中国人民抗日斗争，金成柱主要活跃在吉东地区和[[长白山|长白山]]一带。{{r|rat|page=35}}<ref name=" fish" /><ref name="man " />

1932年，金成柱介绍张蔚华入中国共产党。{{r|duck|page=150}} 4月25日，金成柱在吉林[[安图县|安图县]]明月沟创建“[[安图县抗日游击队|安图县抗日游击队]]”。{{r|rat|page=35}}<ref name=" fish" /> 5月5日，金成柱带领游击队首次和日军交锋，大获全胜。{{r|rat|page=35}}<ref name=" fish" /> 11月，金成柱的安图县抗日游击队和中国共产党领导的“[[汪清抗日游击队|汪清抗日游击队]]”、“[[宁安抗日游击队|宁安抗日游击队]]”统合为“汪清抗日游击队”，金成柱担任政治委员。{{r|rat|page=35}}<ref name=" fish" /><ref name="man " /> 12月，[[杨靖宇|杨靖宇]]{{noteTag|1=杨靖宇（1905年2月26日-1940年2月23日）原名马尚德，字骥生，汉族，[[河南省|河南省]][[确山县|确山县]]人。 }} 将“磐石游击队”改编为“中国工农红军第三十二军南满游击队”。<ref name=" fish" />

1933年1月，“汪清抗日游击队”、“[[延吉抗日游击队|延吉抗日游击队]]”、“[[和龙抗日游击队|和龙抗日游击队]]”、“[[珲春抗日游击队|珲春抗日游击队]]”统合为“[[红军32军东满游击队|红军32军东满游击队]]”。金成柱带领游击队队员攻入日治朝鲜，并且召开了“王在山会议”。{{r|rat|page=36}} 9月18日，“红军32军东满游击队”整编为“东北人民革命军第二军独立师”、“南满游击队”改编为“东北人民革命军第一军第一独立师”，杨靖宇任师长兼政委，东北人民革命军第二军独立师下面有4个团，金成柱担任第三团政委。{{r|rat|page=37}}10月，杨靖宇带领部队夜晚渡过辉发江，进入东边道，开辟“江南游击区”，建造“[[金川河里根据地|金川河里根据地]]”。<ref name=" fish" />

1934年3月，金成柱将独立师第三团改为“[[朝鲜人民革命军|朝鲜人民革命军]]”，同时又叫“东北抗日联军”。{{r|rat|page=40}}<ref name="man " /> 金成柱联合中国救国军攻打东宁、罗子沟，后2次远征北满。金成柱后来与[[平南洋游击队|平南洋游击队]]、[[周保中|周保中]]领导的五军、[[李延禄|李延禄]]的四军联合作战，多次打败日军，因而被称为“金日成将军”，<ref name=" fish" /> 自此，金成柱便改名为“金日成”，并且名声逐步流传开来。11月7日，“东北人民革命军第一军”宣布成立，军长兼政委是杨靖宇。<ref name=" fish" /> 

1936年3月，中共南满党第二次代表大会召开，东北人民革命军第二军改编为“东北抗日联军第二军”，扩充为3个师，[[魏拯民|魏拯民]]出任政委和党委书记，金日成出任第三师师长。{{r|rat|page=41}}5月，金日成组建了朝鲜反日民族战线“[[祖国光复会|祖国光复会]]”，出任会长，并且在长白山建立抗日根据地。{{r|rat|page=41}}<ref name=" fish" /><ref name="man " /> 10月，魏拯民和金日成会师，两人协同作战。此时，金日成领导的朝鲜人民革命军和杨靖宇领导的[[东北抗日联军|东北抗日联军]]第一军为东南满地区的最大2个武装部队。10月，杨靖宇部下曹国安带了部队到长白县黑瞎子沟与金日成协同作战，两人结为至交，但是未谋面。{{r|rat|page=42}}

1937年6月3日，金日成带了150余人渡过[[鸭绿江|鸭绿江]]攻入日治朝鲜[[普天堡|普天堡]]，大获全胜，被誉为“[[普天堡大捷|普天堡大捷]]”。{{r|rat|page=43}}<ref name="man " /> （1937年11月18日，朝鲜《京城新闻》报道金日成被[[满洲国军|满洲国军]]击毙。满洲国军官方月刊《铁心》发表「金日成匪賊討伐詳報」。普天堡居民声称，金日成约为40岁左右。朝鲜总督府逮捕的共犯朴金喆和朴达也供述，金日成当时36岁。因此李命英、朴甲东等学者认为，此金日成并不是彼金日成。）10月27日，张蔚华为了给金成柱报信大日本帝国围剿革命军的消息，被日军抓捕杀害，留下遗属张琅、张金泉、张金禄。{{r|duck|page=250}}

1938年11月25日，金日成和杨靖宇在[[濛江县|濛江县]]首次见面，杨靖宇没有卸下行装而是先给朝鲜部队打招呼，赢得了金日成的尊敬。两人召开了“[[南牌子会议|南牌子会议]]”，批评了[[左倾冒险主义|左倾冒险主义]]。{{r|rat|page=44}}<ref name=" fish" />

1939年，大日本帝国增加对满洲国地区的兵力投入，金日成等抗日人士和军队的实力被大幅度削弱，不得不撤退，由进攻转为防御。4月，[[关东军|关东军]]司令官[[梅津美治郎|梅津美治郎]]{{noteTag|1=梅津美治郎（1882年1月4日-1949年1月8日）[[北九洲|北九洲]][[大分县|大分县]]人，毕业于日本陆军士官学校、陆军大学，[[大将|大将]]军衔，《[[何梅协定|何梅协定]]》推手，[[东京审判|东京审判]]的28个法西斯战犯之一。 }} 下达命令“从第一、第二独立守备队选拔人员编成挺进队，努力捕杀杨靖宇、金日成等匪首。”，满洲国政府悬赏1万元作为报酬。<ref name=" fish" /> 10月，关东军第二独立守备队司令官[[野副昌德|野副昌德]][[少将|少将]]{{noteTag|1=野副昌德，少将军衔，三省联合讨伐司令。 }} 带领7．5万人扫荡金日成及其部队、杨靖宇及其部队。<ref name=" fish" /> 5月，金日成带领部队在青峰、枕峰、茂山、新四洞、大红丹一带游击战，与日军周旋。<ref name=" fish" /> 

1940年1月，抗日联军第二、三军改编为支队，金日成的第一军也改编为多个小部队，以防止被日军全部剿灭。2月23日，杨靖宇在濛江县三道崴子被日军杀害。<ref name=" fish" /> 冬季，抗日联军被日军全部赶出满洲国，金日成等都撤退到苏联[[远东|远东]]地区。{{r|rat|page=46}}

1942年2月16日，金日成的儿子[[金正日|金正日]]出生在吉林长白山抗日游击队的营地。{{r|rat|page=47}}（苏联方面记录，1941年2月16日金正日出生于苏联俄罗斯苏维埃联邦社会主义共和国遠東區伯力附近维亚特斯科耶军营。）8月1日，东北抗日联军三路军改编为“苏联远东方面军第八十八独立步兵旅”，简称“抗联教导旅”。{{r|rat|page=48}}金日成出任一营营长，苏联授予其“[[大尉|大尉]]”军衔。{{r|rat|page=48}}

1945年8月，苏联出兵攻打东北日军，同月，朝鲜解放。{{r|rat|page=49}}<ref name=" fish" /><ref name="man " />  后来，[[大日本帝国|大日本帝国]]投降，中共东北委员会行改组，朝鲜同志组跟着朝鲜工作团回到朝鲜，中国同志组跟着苏联军队回到东北。<ref name=" fish" />  9月19日，金日成乘坐苏联军舰回到朝鲜，{{r|rat|page=50}}<ref name=" fish" /> 告别生活了25年的中国。
