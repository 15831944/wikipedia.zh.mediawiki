'''金文'''，亦称'''銘文'''或'''钟鼎文'''，乃铸或刻于[[青铜器|青铜器]]上的文字。初始于[[商朝|商朝]]中期，盛于[[西周|西周]]，記錄的内容与当时[[社会|社会]]，尤其是王公贵族的活动息息相关，多为祀典、赐命、征伐、[[圍獵|圍獵]]及[[契约|契约]]之事。大部分人以[[周宣王|周宣王]]在位时期铸造的[[毛公鼎|毛公鼎]]<ref>[http://www.npm.gov.tw/zh-tw/collection/selections_02.htm?docno=193&catno=19&pageno=2 西周晚期—毛公鼎]</ref>金文（又稱西周金文）为金文代表，毛公鼎铭文共32行，500字<ref name=毛公鼎铭文>[http://tech2.npm.gov.tw/dm2003/wcbi/b/object5/05.htm 院藏西周金文>冊命篇>毛公鼎] {{webarchive|url=https://web.archive.org/web/20121229014239/http://tech2.npm.gov.tw/DM2003/wcbi/b/object5/05.htm |date=2012-12-29 }}</ref>。

[[西周|西周]]以降，金文被普遍地使用，而用途則無甚改變。据统计，其文约有3,005字，可辨識者，計有1,804字，比[[甲骨文|甲骨文]]略多。由於[[商朝|商]][[周朝|周]]盛行[[青铜器|青铜器]]，而青铜礼器以「鼎」为代表，乐器以「[[鐘_(敲擊器)|鐘]]」为代表，因其刻於金器、大鐘上故稱之金文，亦因而得“钟鼎文”之名。

==金文的種類==
金文可略分為四種<ref name=金文>[http://www.wenyi.com/art/shufa/xianqing/jinwen.htm 先秦书法—金文]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，即[[殷|殷]]金文（前1300年左右～前1046年左右）、[[西周|西周]]金文<ref name=西周金文>[http://www.ccnt.com.cn/china/history/history/zhou/zhou30.htm 西周金文] {{webarchive|url=https://web.archive.org/web/20070609162335/http://ccnt.com.cn/china/history/history/zhou/zhou30.htm |date=2007-06-09 }}</ref>（前1046年左右至前771年）、[[東周|東周]]金文（前770年～前222年）和[[秦|秦]][[漢|漢]]金文 （前221年～219年）。

===商金文===
目前所发现的最早的金文，见于商代的[[二里岗文化|二里岗文化]]时期，例如河南中牟县出土的臣戈、陕西绥德墕头村出土的冬刃戈等，年代为[[商朝|商朝]]中期，早于[[殷墟|殷墟]][[甲骨文|甲骨文]]。<ref>[http://www.docin.com/p-1619505012.html 河南出土商代金文的初步整理与研究]</ref>绝大多数的商代金文，都产生在[[盤庚|盤庚]]遷[[殷|殷]]（今[[河南|河南]][[安阳|安阳]][[西北|西北]]）以後。商朝晚期的鑄有金文之青銅器日愈增多，然所述仍簡，多為鑄者或其先祖之名称。至商朝末期，方有铸刻的文章出現，如[[帝乙|帝乙]]时期的[[小子蒚卣|小子蒚卣]]，有器铭文4行44字、盖铭文3字。商朝金文中已发现的单字总共有1300多个。

===西周金文===
{{Original research|time=2012-08-15T06:18:56+00:00}}
及至[[周朝|周]]代[[商朝|商]]起，金文漸興，[[天子|天子]]之事，如[[周昭王|昭王]]南巡，[[周穆王|穆王]]西狩等，多有記述。
西周铭文[[青铜器|青铜器]]几乎涵括每一位周王，可以说西周周王年数的正确与否完全取决于金文，要是不懂青铜器铭文的所属是那一位周王，就是说没有完整的周王元年青铜器铭文的支持，谈论[[中国年表|中国年表]]西周部分毫无意义。西周周王青铜器的标准年月日的日期标注，以“唯王”开始，后面如果没有年数，默认的是该周王的元年，有年数的也可以换算成该王元年。有关周王的元年基本上是正月初吉，即现在的新年正月初一昭告天下，新年新气象取的是吉祥。也有周王在新年登基后赶赴外地诸侯处行政的例子。

===東周金文===
自[[周平王|平王]]東遷以降，鐵器漸見，[[鐘|鐘]]等青銅樂亦漸多，且亦能鑄文於青銅器外側，故金文所錄，已非如當初般，只為王公大臣之事，戰功、音階等，皆有鑄錄。此時金文被廣泛使用，堪稱全盛時期。

===秦漢金文===
[[秦始皇|秦始皇]]一統天下後，詔令書同文，並於四方立碑，所用之文字皆為[[小篆|小篆]]，且不再刻銘文於鐘鼎之上，由是金文漸衰。及至[[漢代|漢代]]，民間多鑄銘文於鐵器之上，青銅之器，不復使用，金文自是不見於史。

== 金文的製造過程 ==
殷周金文被鑄在青銅器的內側，但是怎樣在鑄模上刻印上金文仍然未能確定。根據在工場遺址所發現的大量模具所推斷，青銅器的製造方法如下：

# 利用粘土做一個與製成品大小相若的土胚（模型）。 
# 另外再用粘土包裹著模型，待乾透後切開外層的粘土，作為外模。
# 將模型削去外層，作為內模。
# 在內模刻上圖案文字。
# 組合起外模和內模，並在之間放入銅片作為間隔空隙以待注入銅液。
# 將已溶化的銅注入。
# 將模冷卻打破，取出青銅器。

但是，怎樣在內模上加上文字圖案等，仍然是一個謎。由於在青銅器內側的金文是凹入去的，因此在內模上的文字應該是凸出來的。加上這些凸出來文字的技法，有各種不同的假設：

*將溶成泥狀的粘土，逐漸貼上。<br />此為清朝[[金石學|金石學]]權威[[阮元|阮元]]提出的假設，但沒有實証實驗

*在內模貼上薄粘土，再削去多餘部份。<br />這是民國以前被提出的假設。工序中必然會在內模上造成痕跡，而青銅器上亦必然有之，然實際上並沒有。

*先在木片或龜甲上刻上文字，用粘土填滿後，再將粘土移印至內模上。<br />經實驗証明這個方法可行，但是沒有發現相關的物証，因此仍只可當作假設。

概论：除甲骨文外，在殷代的一种重要文字是金文。它的名称也是由书刻的原料而来。
