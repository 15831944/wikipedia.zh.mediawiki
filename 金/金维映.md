'''金维映'''（{{bd|1904年|8月16日|1941年|catIdx=J金}}）<ref>[http://dangshi.people.com.cn/n/2015/0921/c85037-27613962.html 《缅怀奶奶金维映烈士》“1904年8月16日，奶奶（金維映）出生在浙江舟山群岛第二大岛屿岱山的高亭镇（现属岱山县）。”]</ref>，清[[宁波府|宁波府]][[定海县|定海县]]（现属[[浙江省|浙江省]][[岱山县|岱山县]]）人。

== 早年 ==
1904年8月16日，生於浙江省岱山縣。<ref name="寒山碧">寒山碧原著，伊藤潔縮譯，唐建宇、李明翻譯：《鄧小平傳》，香港：東西文化事業公司，1993年1月</ref>{{rp|32}}1908年，由於天災，父親經營米店破產，後移居定海縣，其父在一招待所當賬房。<ref name="寒山碧"/>{{rp|32}}1913年，金維映進入縣立女子第一小學讀書。<ref name="寒山碧"/>{{rp|332}}少年時天資過人，好讀書，擅長演戲、演說，受到校長激進思想影響，從小嚮往革命。<ref name="寒山碧"/>{{rp|32}}

== 經歷 ==
1926年，加入[[中国共产党|中国共产党]]，为中共定海独立支部发起人之一。<ref name="寒山碧"/>{{rp|32}}当时宁波地委称其为“定海女将”。她是[[红一方面军|红一方面军]]参加[[长征|长征]]的30位女红军之一。1931年，和[[邓小平|邓小平]]相愛並結婚。<ref name="寒山碧"/>{{rp|33}}是邓小平第二任妻子。1933年，因[[“邓、毛、谢、古事件”|“邓、毛、谢、古事件”]]和個人理由，金維映与邓小平离婚，又與时任[[中共中央组织部|中共中央组织部]]部长[[李维汉|李维汉]]結婚。<ref name="寒山碧"/>{{rp|33}}令鄧小平痛苦萬分，在鄧小平生前，沒有人敢在鄧前提“金维映”三個字；鄧也只字未提金维映。金与李維漢在[[長征|長征]]途中育有一子名叫[[李铁映|李铁映]]。<ref name="寒山碧"/>{{rp|33}}

== 逝世及身後 ==
[[第二次世界大戰|第二次世界大戰]][[苏德战争|苏德战争]]爆发后，1941年在[[莫斯科|莫斯科]]的[[精神病院|精神病院]]里{{Fact|擔任志願[[護士|護士]]|}}的金维映被德軍[[轟炸機|轟炸機]][[空襲|空襲]]炸彈炸死，享年37歲。

出生地岱山縣[[高亭镇|高亭镇]]後街弄14号（黄家）仍保留其故居。
