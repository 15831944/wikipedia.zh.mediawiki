: 本作的主角，[[埼玉縣|埼玉縣]]人，兩眼視力皆為2.0，劇中年齡被固定為17歲。生日：昭和53年1978年8月5日（獅子座）；身高：{{height|m=1.67}}（第一部）→{{height|m=1.70}}（第二部）；體重：{{weight|kg=58}}。
: 就讀私立不動高中二年級，推理研究社社員。學業和體育成績都很差勁（事實上只是不喜歡讀書學習），即便到補習班上課也不見起色（見《絞首之學園殺人事件》、《獄門塾殺人事件》），經常遲到、蹺課，但卻是[[智商|智商]]（IQ）高達180（日本人平均的智商為100，達到180的機率僅有0.01%）的[[天才|天才]]，雖說功課跟體育方面的成績都很差勁，可是當初金田一一的入學成績，卻是不動高中創校以來的最高成績。（見《歌劇院殺人事件》）
: 女人緣極好，很喜歡美雪，卻不敢坦白表達。當朋友問到他跟美雪的關係時，常常只說對方是[[青梅竹馬|青梅竹馬]]，或用幼稚的行為避而不答，更經常偷窺美雪洗澡，或自行潛入美雪房間。
: 性格好色，但只要有帥哥接近美雪便會極力排斥貶低，尤其是對有才幹的帥哥態度更是差的令人不敢置信，最顯著的例子便是本作中的明智警視。平常的個性吊兒啷噹，一副不可靠的個性，常令美雪及老媽擔心。但是一遇到案件發生時，卻永遠是團體中最冷靜可靠的一位。繼承自爺爺的推理能力與正義感，引導他找出真相並解決事件。其性格可在幼稚園就讀「菊組」時的老師評語：「對沒興趣的事一點也不關心」中可見一斑（見《絞首之學園殺人事件》）。小學五年級時曾經參加幼童軍（《狐火漂流殺人事件》）。
: 在本作品裡，設定成[[名偵探|名偵探]][[金田一耕助|金田一耕助]]（作家[[橫溝正史|橫溝正史]]的推理小說系列所登場的偵探角色）的外孫，但實際上橫溝正史的原作小說中並未出現金田一一此人。
: 「推理研究社」社員之一，不過因為經常缺席，被冠上了「幽靈社員」的稱號。也曾因為七瀨美雪的邀約，在「演藝部社」兼職幫忙（見《歌劇院殺人事件》）。
: 常出現的名台詞：「{{lang|ja|ジッチャンの名にかけて！}}」（中譯：以爺爺的名義發誓）、「{{lang|ja|謎はすべて解けた！}}」（中譯：謎團已經全部解開了）。其解決事件的方法，是腳踏實地的收集證據，運用推理邏輯突破兇手設下的詭計，解開真相，找出兇手，偶爾也會設下陷阱讓兇手自投羅網；若使用讓兇手自述的方式讓兇手自曝其短的方式，常出現的台詞：「{{lang|ja|その言葉で完全に墓穴を掘ったぜ}}」（中譯：那句話等於是自掘墳墓）。
: 喜愛打電動玩具，雖說體育相當不行，但唯獨[[乒乓球|乒乓球]]卻是十分擅長（在電影《上海魚人傳說》中，甚至還會直排輪）（在《亡靈學校殺人事件》中，還可以一個人跟狸貓像對戰）。
: 雖然本身沒有[[懼高症|懼高症]]，但非常害怕刺激性的遊樂設施，往往在搭完該類遊樂設施時會感到腿軟無力（《塔羅山莊殺人事件》、《鏡迷宮殺人事件》）。此外，金田一一非常不擅長應對神經質的女性（見《絞首之學園殺人事件》、《薔薇十字館殺人事件》）。然而在劇中亦不乏異性當事人對其有好感。
: 向爺爺學習類似巧手或扒手的技能，能夠在當事者不自覺下拿走或更改物品（例如：拿走七瀨美雪的[[胸罩|胸罩]]、劍持警部的[[打火機|打火機]]、某些女性當事人的內褲、更改對方的[[手錶|手錶]]時間…等）。
: [[魔術|魔術]]也是金田一的特長， 而耍[[撲克牌|撲克牌]]的手法堪稱職業水準。（來源自《推理公式大破解》一書， 以及《魔術列車殺人事件》）。金田一一的另一強項是破解謎語，謎題， 多次被邀請參加解謎及尋寶活動。（見《秘寶島殺人事件》、《金田一少年的殺人》、《天草財寶傳說殺人事件》及《露西亞人偶殺人事件》）。
: 對摩斯密碼運用自如，更多次以各種密碼與明智警視作秘密通訊。（見漫畫《金田一少年的殺人》、《金田一少年敢死之行》、以及小說《殺戮的深藍》）。音樂方面不算是金田一一的強項， 但他也懂簡單的鋼琴演奏， 可彈奏〈小蜜蜂〉呢（來源自明智短篇《殺意四重奏》事件中）。
: 在《血溜之間殺人事件》中，金田一展現出兩種鮮為人知的秘技，其一是[[五子棋|五子棋]]處於從來沒有輸過的無敵狀態，另外一種特技是[[圍棋|圍棋]]，自稱從來沒有在爺爺金田一耕助手上得勝，可是卻在對開櫻學園的圍棋部長三石勳的對戰中輕鬆取勝。至於[[日本將棋|日本將棋]]方面， 金田一一能輕鬆的把擁有三段實力的劍持玩弄，可見其實力也有一定水準。
: 對於[[犯罪心理學|犯罪心理學]]有相當的認知，《悲戀湖傳說殺人事件》最後對著犯人說「[[卡涅阿德斯船板|卡涅阿德斯船板]]」的最後男人的內心煎熬；《亡靈學校殺人事件》中最後對犯人說：「誰かを殺すことで救われた人を、俺は一人も見たことがないんだ」（中譯：誰殺了誰就可以得到救贖，我一個也沒見過）。在該作品的一開始時，兇手多以自殺作為結束，到後期因為金田一一的規勸能力增強，讓兇手自首服刑的機率也相對提昇。但或許是不想承認人類有人性本惡或虛偽的一面，對於宿敵高遠遙一的心理及見解，完全不表認同。這顯示金田一一的性格中有其固執的一面。
: 認為即便案件當事人是犯下過錯的可憎之人，也不認同私下以牙還牙，在法律之外殺害當事人復仇的行為（《不動高中學園祭殺人事件》、《暗黑城殺人事件》）。很多時候會對犯案的當事人予以溫和的勸導，使其得以重新面對往後的人生。
: 不相信靈異鬼怪之說，會極力的搜集資料試圖解讀事件，儘管如此，偶爾仍會遇到其無法解釋的現象（見《蠟人形城殺人事件》、《亡靈學校殺人事件》、《金田一少年之惡夢》、《雪靈傳說殺人事件》）。 
: 喜愛的食物有金槍魚的眼睛， 不過就算不喜歡的食物依然能照吃不誤（來源自《推理公式大破解》一書）。喜愛的飲料是烏龍茶（見《金田一少年之惡夢》）， 但也常常陪劍持警部或美雪喝咖啡或下午茶。最高紀錄是吃掉10個漢堡包後，再把玲香為他預備的中華料理大餐秒殺掉（見《金田一少年敢死之行》）。
: 曾經是[[幼童軍|幼童軍]]，但因結夥去冒險失蹤兩日而讓帶隊的大人擔心。(《狐火漂流殺人事件》)
: 在第一部漫畫版最後劇情描述中，收到一封不明的信件因而在暑假展開四處旅行。但在2004年第二部的重新連載故事中，可得知該信件寄信人正是高遠遙一。
: 在第二部漫畫版中，金田一一跟宿敵高遠遙一交手的事件頻率變高，有時候就算高遠沒有直接於事件中現身，但整個殺人計劃也是由他為了針對金田一一跟明智警視而策劃。然而在某些案件裡，偶爾也會出現金田一一和高遠遙一攜手合作破案的情況（見《露西亞人偶殺人事件》、《薔薇十字館殺人事件》）。
: 《37歲事件簿》裡已成年（37歲），於公關公司「音羽布萊克公關公司」擔任職員，並對推理產生抗拒，但因《歌島度假村殺人事件》而再次重拾已遲緩的推理能力，並在歌島事件結束後前往監獄與將近二十年未見的宿敵-高遠遙一會面。

: 本作的女主角，11月24日生日[[人马宫|射手座]]（見「銀幕之殺人鬼」），三圍E88/W58/H89。《吸血鬼傳說殺人事件》裡，曾說過自己體重不滿47公斤。
: 七瀨美雪是金田一一的青梅竹馬，最佳夥伴同時也是互相暗戀/明戀的對象。
: 就讀私立不動高中二年級，樣貌漂亮，身材出眾，熱心課外活動之餘，成績也很優秀，做事中規中矩的優良個性，可說是不動校園的偶像。在學校相當受歡迎，經常收到男同學的告白，可是全都給美雪回絕。美雪在喝過酒後會發酒瘋，個性會變的相當開放，目前正學習著插花技巧，擅長快速背誦，在書道方面有三段的資格。喜歡的花是[[向日葵|向日葵]]，喜歡的藝術家是[[Mr.Children|Mr.Children]]。
: 美雪是現任不動高中的學生會會長，後來加入「推理研究社」，至中期接任社長職位。美雪她似乎仍在「演藝部社」兼職幫忙。初中時，美雪更曾加入「籃球社」（見有聲書《死神病院殺人事件》）。
: 由於其豐滿的上圍，故時爾在劇中成為玩笑的對象，在短篇漫畫《露營地的「怪」事件》更透露其中學時期，上圍已達到D罩杯的程度。
: 五歲時曾和金田一一起洗過澡，但本人卻是極力否認（但在《魔神遺蹟殺人事件》中向宗像五月發飆時大聲承認了）。對從小一起長大的金田一一，抱持著「朋友以上，戀人未滿」的感情。
: 不過在第二期漫畫版最後劇情中（即第一部完結時），美雪曾與金田一一接吻，對與金田一一成為一對戀人仍在努力發展中。事實上被人問及與金田一一之間的感情，卻會以青梅竹馬的藉口敷衍，讓人覺得不坦率。但若金田一一與其他同年齡層女性過好時，卻會極力阻撓，一開始只會大力踩金田一一的腳，至後期卻會扔石塊或拳頭攻擊。另外於《遊戲館殺人事件》中, 兩人已經很自然地拖手了，在《不動高中學園祭》兩人更親密地拍照，相信兩人的關係會一直轉變為更明朗化。
: 與一開始性格很好的設定感覺似乎有所轉變，又或者這才是七瀨美雪的真性情；就如《金田一兩日一夜小旅行》系列中表現出的「真我」的美雪或許才是美雪的真本性。
: 在短篇集「底片的不在場證明」中，美雪曾瞞著金田一赴草太的電影約會，且片名是「愛情盛開的季節」。
: 因為其亮麗的外型，故在劇中時常被邀請作為演員或擔任模特兒，在《香港九龍財寶殺人事件》更因為其容貌碰巧和模特兒楊蘭相仿，而被充當替身。
: 和金田一一共同經歷許多案件，也曾幾次陷入有生命危險的險境，美雪亦曾有被當成兇殺案中兇嫌的經驗。在《不動高中學園祭殺人事件》，曾對金田一一表示若案件當事人生前曾做出嚴重侵害他人的行為，那麼即便遭他人殺害，也是罪有應得，但金田一一卻對美雪的這番說法做出了反駁。其表兄橘川茂於《悲戀湖殺人事件》被兇手視為目標而遇害。
: 在許多事件中，因為美雪的細心跟觀察的敏銳，留意到的種種小線索，常常有助於金田一釐清事件真相（例如：《墓場島殺人事件》中，留意到被害者書寫的絕世之辭的方式…等）。
: 在眾多案件中，只有《雪影村殺人事件》、《殺人餐廳》、《明智警視之事件簿/證詞猜謎The Murder Train》、《死者的將軍棋》、《狐火漂流殺人事件》等作品沒有登場，在《速水玲香與不速之客》中，美雪也只在最後一格亮相。但在動畫版《殺人餐廳》及《明智警視之事件簿/證詞猜謎The Murder Train》劇情中，在逮捕犯人後美雪亦有登場。
:《37歲事件簿》裡已成年（37歲），於大型航空公司擔任主任乘務長，雖忙於工作但仍透過簡訊關心阿一。

* 代號：「地獄傀儡師」、「[[宙斯|宙斯]]」
* 殺人數：（原作）10人／（動畫）8人／（電視劇）4人
* 殺人未遂：（原作）3人／（動畫）2人／（電視劇）2人
* 教唆殺人致死：（原作）21人／（動畫）15人／（電視劇）12人
* 事件後：20年來被囚禁於監獄中的特別拘留區
* 登場案件： 《魔術列車殺人事件》・《速水玲香誘拐殺人事件》（電視劇除外）・《露西亞人偶殺人事件》・《金田一少年的敢死之行》・《獄門塾殺人事件》・《黑魔術殺人事件》・《劍持警部的殺人》・《香港九龍財寶殺人事件》（電視劇）・《薔薇十字館殺人事件》・《高遠少年的事件簿》・《亡靈校舍之殺人》・《蟻地獄壕殺人事件》・《金田一二三誘拐殺人事件》。
: 高遠遙一， 是本作最重要的犯罪角色之一， 以《魔術列車殺人事件》中的設定計算，年齡估計約23歲左右，體重50公斤（電視劇53公斤）。和明智警視同為私立秀央高中少見以全科滿分入學的高材生《高遠少年的事件簿》。與金田一一是偵探與犯人的宿敵關係，與明智警視卻是另一層面的敵對關係。
: 初登場在《魔術列車殺人事件》中，另一方面也有在其他事件使用[[教唆殺人|教唆殺人]]的身分登場。以「地獄傀儡師」的名義強調自己堅持的「犯罪藝術」，利用別人的動機為此人策劃殺人計劃，倘若執行者因執行被識破而招供，就會自己動手殺掉執行者，以求自己堅持的犯罪藝術完美。此外，高遠亦曾在《黑魔術殺人事件》（漫畫版）過後，設局毒殺被他雇用的替身演員（動畫版則是直接現身金田一面前之後逃走）。
: 自懂事以來，就和從事貿易工作的父親（後於《薔薇十字館殺人事件》證實為高遠遙一的養父）兩個人住在[[英國|英國]]。至高遠遙一7歲時，養父第一次帶他觀賞魔術表演，就此第一次見到母親'''近宮玲子'''(聲：[[小山茉美|小山茉美]])的魔術表演，使高遠遙一為之著迷，之後背著養父偷練魔術。10歲那年第二次見到母親，當時他在公園練習魔術，碰上近宮玲子。近宮玲子不但細心指導他，還承諾把最好的魔術點子送給他一人。17歲的時候養父去世，透過他的日記，得知近宮玲子就是自己的親生母親。在得知近宮玲子去世後不久，從意大利完成魔術學習歸來的高遠遙一， 收到母親答應「只送給他一人」， 記載著各項精華魔術點子的記事本。
: 幾個月後，高遠遙一前往日本，觀賞母親的徒弟們接手的「幻想魔術團」表演，發現表演者聲稱的獨創魔術，卻全是出自近宮玲子只傳給他一個人的記事本。高遠遙一認為母親的死因很可疑，於是以經理身份加入「幻想魔術團」進行調查，得知母親的確是被四個徒弟為了得到魔術點子記事本而害死的，因而在《魔術列車殺人事件》殺害其中三名徒弟，為母親報仇。
: 《魔術列車殺人事件》落幕後，高遠一度被警方逮捕送進監獄，本於案中被其作為目標卻躲過一劫的'''小丑左近寺'''，於表演魔術期間因為機關失靈，遭表演時放的火燒死。據明智事後轉述，左近寺其實是拿到近宮玲子刻意留下的魔術筆記本，沿用筆記記載的有缺憾的魔術手法進行表演，才會落入近宮玲子設下的陷阱導致自己慘死，高遠稱之為「近宮玲子的復仇」。
: 曾以成為與近宮玲子一樣的一流魔術師為目標， 並在意大利一位名魔術師下習魔術三年， 所以魔術技巧超一流。在《高遠少年的事件簿》透露其高中時期曾加入魔術社，連當時的魔術社指導老師'''姬野優未'''都對其才華感到驚嘆，但高遠曾向後者表示「天才都是孤獨的」。此外，當時的魔術社員'''霧島純平'''，亦提及少年時期的高遠其實潛藏著殺戮的本性。
: 擅長變裝技巧， 在《速水玲香誘拐殺人事件》及《金田一少年敢死之行》中可見識到他出神入化的變裝表演，也經常以變裝成神父或事件關係人現身事件之中。[[日文|日文]]、[[英文|英文]]及[[廣東話|廣東話]]也很流利，亦懂得些許[[德文|德文]]和花語（《薔薇十字館殺人事件》）。文學和音樂的造詣亦有相當水準，這方面的造詣完全不下於明智警視，因為居住英國時期照顧高遠的女傭係專修音樂學科，故在後者的指導下學會鋼琴（《高遠少年的事件簿》）。
: 在《魔術列車殺人事件》登場時給人的第一印象是個感覺木訥老實的好好先生，但被金田一識破其陰謀後露出其狡猾和冷靜的真面目，連說話語氣都和擔任經理時期的模樣截然不同。
: 在「露西亞人偶殺人事件」中，逃亡資金贊助者'''幽月來夢'''因故遭殺害，本決心殺害兇手為幽月復仇，卻因同情及體會其遭遇，因而阻止兇手自盡，就內心層面也有善良的一面。
: 在《高遠少年的事件簿》裡，揭露高遠實於高中時期便犯下首樁殺人案件。
: 常出現的名台詞為：「Good Luck」。以[[平行線|平行線]]或[[雙胞胎|雙胞胎]]比喻自己與金田一一的關係：「我和你是兩條不會相交的[[平行線|平行線]]……但是，平行線雖不會相交，却永不分離（或者说，正因为平行線不會相交，所以不會分離）」（原文：{{lang|ja|私と君は決して交わることのない平行線……だが、平行線は交わりこそしないがいつも隣にある}}）、「我和你就像黑暗和光明一般，是[[雙胞胎|雙胞胎]]一樣的存在」（原文：{{lang|ja|私とキミは闇と光の双子のような存在}}）。
: 在「金田一少年敢之死行」中，最後逃跑失敗而被逮捕。及後送交一封信件金田一一跟明智警視，內容記載有可能發生犯罪的地點與事件，被明智警視稱為「極度充滿惡意的『犯罪導覽地圖』」。
: 曾三次被金田一成功拘捕歸案，但高遠遙一每一次都能成功逃獄，被日本警方鎖定[[通緝|通緝]]。
: 變裝技術了得, 在《獄門塾殺人事件》中以英文教師「'''赤尾一葉'''」的身份出現（真人戲劇版尾聲更一度男扮女裝，假扮成外籍玩具攤販，瞞過當時和阿一及美雪同行的'''李白龍'''），被明智[[警視|警視]]用手槍射擊心臟部位，卻因為穿上了[[防彈背心|防彈背心]]而無大礙。在《黑魔術殺人事件》中則以黑魔術師「'''黑瓜鬼門'''」的姿態現身, 而在《金田一少年的敢死之行》以及《劍持警部的殺人》則用神父的身份登場，而在《薔薇十字館殺人事件》的高超變裝術更使金田一一跟警察都被玩弄得團團轉。
: 在第二部故事中，多件案件也與高遠遙一扯上關係，出場率增加。而在《薔薇十字館殺人事件》裡，高遠因收到匿名「薔薇十字」的對象寄送的藍色薔薇，和一封以其同父異母的妹妹性命作要脅的藍色薔薇發表會邀請函，為了保護妹妹的性命與確認生父的身分，遂向金田一提出「若能偵破此案，便自願向警方自首」的條件，換取對方的協助，化名插花師'''遠山遙治'''， 聯同金田一一和美雪共同行動查案。在事件末透過'''月讀吉賽兒'''的自白，繼而確認吉賽兒正是高遠的異母妹，並得知薔薇十字館正是親父留給自己的遺贈。
: 喜愛戴著面具，在公園或廣場表演傀儡木偶劇和魔術給附近的小孩子觀賞，也喜以人偶跟玫瑰作為自己的代號。劇情後期更自行架設個人網站「通往完美犯罪的大門」，引誘自願擔任其傀儡的目標上鉤。在《亡靈校舍之殺人》甚至比阿一等人搶先從黃金島的廢棄校舍找到黃金並據為己有。
: 對高遠重要的事物， 包括母親的魔法點子本、魔術、曾協助自己逃亡的朋友、以及完美的殺人藝術品。
: 在《37歲事件簿》中確認已被逮捕，20年來被囚禁於監獄中的特別拘留區中，另外他的頭髮已不再是過去的黑髮。以希臘神話的主神宙斯自居，培養了包括'''麻生早苗'''在內的十一名犯罪者。得知'''麻生早苗'''把有關奧林柏斯十二神的事告訴明智後，便透過書頁傳遞信息予位於監獄外的弟子把'''麻生早苗'''滅口。

: 當紅[[偶像_(日本)|偶像藝人]]、演員、歌手，真名'''「小城梓」'''。三圍B83/W58/H84。喜愛的食物是明太子，剛開始養的美國短毛貓，名叫波奇。5歲時進入劇團，14歲以「悲しき予感」一曲正式以歌手身份出道。
: 玲香很喜歡金田一，且對金田一相當專情，可說是本作第二女主角。
: 初登場在《雪夜叉傳說殺人事件》中，也由此與金田一一結識並產生好感，雖然是七瀨美雪感情上比較對象，但支持美雪發現自己的心意。小時候被綁架甚至認綁匪為父親（即[[斯德哥爾摩症候群|斯德哥爾摩症候群]]）等悲慘身世在《塔羅山莊殺人事件》被揭露（也因此經歷影響，以至於有人準備為她圍上圍巾時，玲香會當場面露懼色，並極力避免在頸部掛上任何飾物），又經歷「速水玲香誘拐殺人事件」等事件，讓個性外向好動的玲香，在快樂的表情下總是有著寂寞的面孔。唯一的親人哥哥小城拓也在《塔羅山莊殺人事件》為救她而死亡；在《速水玲香誘拐殺人事件》中，金田一一推測當紅女演員三田村圭子或許是速水玲香的親生母親，不過並未獲得證實。
: 在故事中常常陷入被懷疑是嫌犯的窘境。在其他多個事件開頭有登場客串，不過大多與該次事件劇情無關，是以描述和金田一之間的感情小故事為主。
: 在第二部漫畫中， 玲香先後在《速水玲香與不速之客》以及《鍊金術殺人事件》中再度登場，外型也有相當大的轉變，這是因為在第一部漫畫連載時期玲香的外型、服裝、髮型反映了日本90年代女性偶像歌手的樣貌，到第二部漫畫連載時已顯得過時，從而被修改成更符合時下偶像歌手的造型。{{-}}

: 金田一一的朋友及助手。「推理研究社」社員之一。哥哥龍太就讀不動高中一年級；動畫版及初代日劇版只有龍太。兄弟兩人血型皆AB型，星座[[雙子宮|雙子座]]。家庭成員父親（名 連太郎）、母親（名 良子），家中經營「佐木映像」的公司，[[攝影|攝影]]照相成為該家族最大的興趣及喜好。
: 攝影畫面總成為金田一一參考回顧案件的關鍵，其貢獻度僅次於七瀨美雪。漫畫版中龍太於《學園七不可思議殺人事件》登場，但在《異人館旅館殺人事件》中，因拍攝到兇手使用的手法而遭到殺害。其後由龍二為繼承其兄遺志，擔任金田一一的第二助手，於《金田一少年的殺人》中登場。
: 動畫版中，龍太在《異人館旅館殺人事件》僅遭到兇手襲擊並未死去。而在電視劇中則完全沒有被捲入。漫畫版中為了遵守「人死了不能復活」的規則，創作佐木龍二這角色，漫畫版背景中可以看到龍太的天使姿態模樣。兄弟兩人長相相似，但哥哥龍太個性較為陰沉，總是偷偷接近別人背後拍攝；相對弟弟龍二個性較為開朗活潑。除此之外，髮型也有些許不同。因此金田一一稱龍二為「佐木二號」。
: 在動畫版第55話《是誰殺了女神》，佐木龍太喜歡上「美術社」的神津沙也香，而在漫畫短篇集中，則是佐木龍二。在漫畫跟動畫故事中，二人同樣地自畫了一幅要送給對方，可惜卻是奇差無比，由此可知佐木龍太跟佐木龍二對於繪畫並不擅長。
: 電視劇版本中，佐木龍太被設定沒有戴眼鏡，且登場的案件極多。

== 音羽布萊克公關社 ==
; {{nihongo|'''葉山海'''|葉山 まりん （はやま まりん）}}
: 《金田一37歲之事件簿》中登場。公關公司「音羽布萊克公關公司」的營業部企劃課的女性職員，23歲。為擔任主任的[[金田一一|金田一一]]的部下，仰慕金田一一。

; 鹿爪剛
: 《金田一37歲之事件簿》中登場。公關公司「音羽布萊克公關公司」的銷售部部長，52歲。初登場時委託內神田指派底下的部屬參與歌島度假村的配對聯誼活動，間接促成歌島度假村殺人事件的開端。

: 《金田一37歲之事件簿》中登場。公關公司「音羽布萊克公關公司」的銷售部企劃部長，44歲。經常因為金田一一編理由請假和工作怠惰的表現，間接影響到自己的績效考核一事對其動怒。
