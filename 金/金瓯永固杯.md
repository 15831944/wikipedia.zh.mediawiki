'''金瓯永固杯'''，[[清朝皇帝|清代皇帝]]每年[[春节|元旦]]举行开笔仪式时专用的酒杯，由清宫[[造办处|造办处]]制造。金瓯永固杯共有四件，均为三足[[鼎|鼎]]形、夔龙耳、以象头为足、杯身[[点翠|点翠]]，[[浮雕|浮雕]]纹饰、铭文、款式。杯口一面铸“金瓯永固”，另一面铸“[[乾隆|乾隆]]年制”。

四件金瓯永固杯分别是乾隆四年的一件鎏金铜杯，乾隆五年的两件金杯和乾隆六十二年（即[[嘉庆|嘉庆]]二年{{efn|name=QL62}}）的一件金杯。[[英国|英国]][[伦敦|伦敦]]的[[华莱士典藏馆|华莱士典藏馆]]收藏有乾隆四年的鎏金铜杯和乾隆五年的一件金杯，[[台北|台北]][[国立故宫博物院|国立故宫博物院]]收藏有乾隆五年的另一件金杯，[[北京|北京]][[故宫博物院|故宫博物院]]收藏有乾隆六十二年的金杯<ref name="许晓东2012" />。

== 历史 ==
金瓯永固金杯是[[清朝皇帝|清朝皇帝]]于每年[[正月初一|正月初一]]举行元旦开笔仪式时的专用器。清朝元旦开笔仪式始于雍正年间，乾隆时期形成定制，每逢元旦在[[养心殿|养心殿]]东暖阁的明窗举行，金瓯永固杯是仪式中必有的御用酒杯之一，用以盛[[屠苏酒|屠苏酒]]。“金瓯”寓意国家政权，取名“金瓯永固”则反映了清朝统治者希冀永远巩固地保持政权的一种愿望<ref>{{cite web |title=明清时期酒器：金瓯永固金杯 |url=http://www.china.com.cn/aboutchina/zhuanti/zgjwh/2007-08/07/content_8640649.htm |website=中国网 |accessdate=2018-12-19 |date=2007-08-07}}</ref>。

清宫“内务府活计档”中对四件金瓯永固杯的制作过程有详细记载。乾隆四年十一月，第一件鎏金金瓯永固杯交造办处鎏金作制作，七品首领萨木哈、催总白世秀绘制图样后，呈乾隆皇帝御览，得到乾隆皇帝认可后照样制作，“其金杯足子做象鼻足子，镶珠宝点翠，金杯刻‘金瓯永固’、‘乾隆年制’之款”。乾隆五年正月，皇帝要求将金瓯永固杯“往细里做”，要求“耳子夔龙上各安大珠子一颗，两面，每面安珠子五颗，中间一颗安大些，花头要圆的”，照样再做两件。乾隆六十二年{{efn|name=QL62|乾隆六十二年，也即嘉庆二年。乾隆六十年，乾隆皇帝退居太上皇之位，次年改元嘉庆，但乾隆皇帝仍继续训政，皇宫内仍使用乾隆年号<ref>{{cite web |author1=金满楼 |title=乾隆纪年只用到六十年？太天真了，紫禁城里用到了乾隆六十三年 |url=https://new.qq.com/omn/20180629/20180629A0VARJ.html |website=腾讯 |access-date=2018-12-17 |archive-url=https://web.archive.org/web/20181217062914/https://new.qq.com/omn/20180629/20180629A0VARJ.html |archive-date=2018-12-17 |dead-url=no }}</ref>。}}十月二十六日，因[[乾清宫|乾清宫]]收贮金瓯永固杯有伤损，要求造办处再制作一件，为制作该杯从内库领用“九成[[金|金]]二十[[两|两]]、[[珍珠|正珠]]大小十一颗、[[红宝石|红宝石]]大小九块、[[蓝宝石|蓝宝石]]十二块、[[碧玺|碧牙𤥒]]四块”<ref name="张世芸1980">{{cite journal |author1=张世芸 |coauthors=故宫博物院院刊 |title=“金瓯永固”杯 |date=1980 |issue=2 |pages=80-81}}</ref>。

英国伦敦华莱士典藏馆的两件金瓯永固杯是{{le|理查德·华莱士|Sir Richard Wallace, 1st Baronet}}1872年在[[法国|法国]][[巴黎|巴黎]]的一次拍卖会上购得的，上一位收藏者名为阿莱格尔（Allègre）<ref name="许晓东2012">{{cite journal |author1=许晓东 |title=返绝久非藉 沧桑全亦奇——记四件“金瓯永固”杯及其他 |journal=紫禁城 |date=2012 |issue=12 |pages=88-99}}</ref>。据称两杯均为法军上校杜潘（Du Pin）于1860年从中国北京的[[圆明园|圆明园]]获得<ref name="刘阳">{{cite web |author1=刘阳 |title=圆明园里的文物：它们现在何处 |url=http://art.people.com.cn/n/2014/0221/c206244-24425468.html |website=人民网-河北青年报 |accessdate=2018-12-18 |date=2014-02-21}}</ref>。

== 对比 ==
=== 相同点 ===
四件金瓯永固杯均为三足鼎形，有两个夔龙耳和三个象首足。杯身以点翠为地，浮雕有纹饰、铭文和款式。杯口饰有一周回纹，回纹带内一面铸“金瓯永固”铭文，一面铸“乾隆年制”款式，均为[[篆体|篆体字]]。杯身布满缠枝花纹，花有大中小三类，除为夔龙尾遮压的两朵大花外，花心均嵌有珠宝。三个象首足头部均嵌有两个宝石<ref name="许晓东2012" />。
