'''金丸 雄介 '''(かなまる ゆうすけ， [[1979年|1979年]] [[9月14日|9月14日]] -)、 [[日本|日本]]男子[[柔道|柔道]]73kg級選手。段位是五段。身高170公分，已婚。

== 人物 ==
* [[石川縣|石川县]][[白山市_(日本)|白山市]] (石川郡舊鶴來町)出身。
* 石川縣立津幡高等學校畢業、[[筑波大学|筑波大学]]畢業。
* 目前是學校法人了德寺學園的職員。
* 有數次国际赛事的出場经验， 曾是[[2008年夏季奥林匹克运动会|北京奥运会]]的代表。
* 得意技是[[背负投|背負投]]。
* 2008年[[10月12日|10月12日]]，正式宣佈引退、退休 <ref>[http://sportsnavi.yahoo.co.jp/fight/other/headlines/etc/20081012-00000055-jij-spo.html 金丸雄介是退休了] {{webarchive|url=https://web.archive.org/web/20081015075434/http://sportsnavi.yahoo.co.jp/fight/other/headlines/etc/20081012-00000055-jij-spo.html |date=2008-10-15 }} 的2008年年10月12日查看</ref>。今后的工作是在了德寺學園對學生的辅导。
* 引退後，在[[富士電視台|富士电视]]台播放的柔道世界锦标赛中，擔任解說員的工作。
* 2012年，被任命為教练<ref>[http://www.judo-ch.jp/training/commission/ 日本加强件柔道]</ref>。
