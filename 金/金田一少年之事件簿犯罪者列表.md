'''金田一少年之事件簿的犯罪者'''是關於在《[[金田一少年之事件簿|金田一少年之事件簿]]》漫畫、[[金田一少年之事件簿_(動畫)|動畫]]及[[金田一少年之事件簿_(電視劇)|電視劇]]中出現的犯罪者的條目。

== 概述 ==
《金田一少年之事件簿》中出場的兇手多數以怪異的代名詞來自稱。一般源自兇手寄來的預告函或兇案現場留下的信息、兇案發生所在地流傳的怪物或幽靈等的名字、過去有過類似犯案手段的犯人的外號、與阿一相關的人目擊兇手喬裝時的樣子而得出的稱呼等等。

雖然某些案件存在兩個或以上的兇手，但多數都是一人犯案，很少有殺害其他共犯滅口或對等共犯關係的情況。阿一在揭露真相時只會指出一個真兇的情況比較多，複數兇手同時存活的情況也比較少。

* 代號：「死神魔術師」
* 登場案件： 《高遠少年之事件簿》
* 殺人數：2
* 殺人未遂：4（姬野優未，雙親，高遠遙一）
* 事件後：被高遠遙一刺殺
: 高遠就讀秀央高中時期的一年級生，為魔術社成員。初登場時以友善的態度主動邀請高遠加入魔術社，但隨著其犯人身分被揭穿，顯露其歹毒的真面目。霧島因為承載雙親的過分期待，加上雙親意圖粉飾太平，使其產生了扭曲的性格。
: 進入秀央高中並首度見到高遠時，霧島直覺其蘊藏著「殺手」的特性，為了不讓高遠潛藏的殺戮特性被埋沒和表現自己魔術般的犯罪手法，先刻意殺害魔術社成員'''藤枝樁'''，然後將自己佯裝成被兇手殺死的模樣，再把發現其凶手身分的'''黑江真砂也'''滅口，在準備將證物銷毀時，被高遠及時制止並揪出其犯人的身份。
: 真相被揭發後，霧島坦言自己本欲殺害高遠親近的時任音樂教師兼魔術社指導教師'''姬野優未'''，還盤算著殺害雙親以奪得遺產，但被高遠指謫其殺人動機過於薄弱，不過是自我滿足，持刀刺向高遠的腹部，孰料高遠早已料到霧島會企圖殺害他的舉動，轉而用魔術手法，將手裡的玫瑰插進霧島的胸膛，接著將玫瑰變成短刀。
: 霧島臨死前留下「你真應該在夜裡閃耀的」的遺言，得知自己是首位死於高遠手下的犧牲者，詢問高遠此時的感覺後，便因為傷重而斷氣身亡，遺體則被高遠運到原本該用於文化季的篝火堆裡加以焚化。
: 案發後，警方在篝火堆裡發現霧島和藤枝的遺體。當時已通過司法考試的明智返回母校探視師長，和姬野老師聊起這樁案件，並首次見到當時正在彈奏鋼琴的高遠。兩人亦因此案從此結下日後相會的契機。
