'''金炳昶'''（{{lang-ko|'''김병창'''}}，{{BD|1931年||2013年|1月5日|catIdx=J金}}）出生于[[北平|北平]]，师从[[杨海荃|杨海荃]]，是国家[[一级演员|一级演员]]、[[中国曲艺家协会|中国曲艺家协会]]会员，知名[[相声|相声]]表演艺术家。<ref name=" fish" >{{cite web |url=http://news.inewsweek.cn/news-42321-p-1.html |title=捧哏大师金炳昶去世 曾缔造沈阳相声辉煌时期 |accessdate=2013年1月6日 |publisher= 中国新闻周刊|archiveurl=http://news.inewsweek.cn/news-42321-p-1.html|archivedate=2013年1月6日 |work= |language=zh-cn}}</ref><ref name=" baech" >{{cite web |url=http://culture.ifeng.com/gundong/detail_2013_01/06/20844175_0.shtml |title= 82岁相声大师金炳昶离世 “捧哏”《假大空》曾风靡|accessdate=2013年1月6日 |publisher=凤凰网 |archiveurl= http://culture.ifeng.com/gundong/detail_2013_01/06/20844175_0.shtml|archivedate=2013年1月6日 |work= |language=zh-cn}}</ref>

== 生平 ==
1931年，金炳昶出生在北平，父亲是商人，儿时曾在西单启明茶社或天桥听相声，曾考入[[华北人民革命大学|华北人民革命大学]]，1950年，因为[[朝鲜战争|朝鲜战争]]爆发，他加入[[中国人民解放军|中国人民解放军]]，后在营口说唱团、沈阳曲艺团学习，1957年，金炳昶同[[马子恒|马子恒]]、[[王海江|王海江]]、[[杨金声|杨金声]]、[[周印金|周印金]]、[[杨振华|杨振华]]一起拜师[[杨海荃|杨海荃]]，1966年，[[文化大革命|文化大革命]]爆发，他下放到沈阳试验服装厂，文革结束之后，回到沈阳市曲艺团。2013年1月5日，他因为[[心脏衰竭|心脏衰竭]]在[[沈阳军区总医院|沈阳军区总医院]]病逝。<ref>{{cite web |url= http://news.hexun.com/2013-01-06/149805634.html|title=相声表演艺术家金炳昶去世 |accessdate=2013年1月6日 |publisher= 和讯网|archiveurl= http://news.hexun.com/2013-01-06/149805634.html|archivedate=2013年1月6日 |work= |language=zh-cn}}</ref><ref name=" fish" /><ref name=" baech" />

金炳昶的搭档是杨振华、[[于琪|于琪]]、[[常佩业|常佩业]]、[[贾承博|贾承博]]、[[王平|王平]]、[[孙伟|孙伟]]等，<ref name=" fish" /><ref name=" baech" />[[巩汉林|巩汉林]]的妻子金珠是金炳昶的徒弟。

== 感情生活 ==
金炳昶的妻子叫“富兰英”，也是梅海荃的徒弟，两人1957年结婚，1966年，富兰英因长期劳累患上[[精神分裂症|精神分裂症]]，2002年，富兰英因[[结肠癌|结肠癌]]逝世。
