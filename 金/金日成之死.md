[[朝鲜民主主义人民共和国领导人列表|朝鮮最高領導人]]、[[朝鮮勞動黨總書記|朝鮮勞動黨總書記]]、[[朝鮮國家主席|朝鮮國家主席]][[金日成|金日成]]於1994年7月8日因[[心臟病|心臟病]]病逝，享壽八十二歲。在金日成死後，[[朝鲜民主主义人民共和国|朝鮮]]官方並未即時公佈其死訊，而是延遲34個小時才對外宣佈<ref name = "NothingToEnvy">{{cite book | title = [[我們最幸福：北韓人民的真實生活|我們最幸福：北韓人民的真實生活]] | author = [[芭芭拉·德米克|芭芭拉·德米克]] | publisher = Spiegel & Grau | date = 2009-12-29 | isbn = 9780385523905 | language = en}}</ref>{{rp | page = 122}}。隨後，[[朝鮮政府|朝鮮政府]]又決定把7月10日至17日定為全國[[喪慟|哀悼]]期（及後再加兩日到7月19日）。在這10日[[國喪|國喪]]期間，婦女不許[[化妝|化妝]]或者整理頭髮，[[飲酒|飲酒]]、[[跳舞|跳舞]]和[[音樂|音樂]]活動一律被禁止，民眾也需要到金日成銅像前[[憑弔|憑弔]]<ref name = "NothingToEnvy" />{{rp | page = 132}}。

==背景==
[[File:Paying_Their_Respects.jpg|thumb]]
1990年代初，北韓被世界孤立，盟友[[苏联|苏联]]和[[中華人民共和國|中華人民共和國]]與[[大韓民國|大韓民國]]的關係日漸改善并建立外交关系，同時也暫停與北韓的貿易<ref name = "NothingToEnvy" />{{rp | page = 42}}。另一方面，隨著[[蘇聯解體|蘇聯解體]]，北韓失去了援助，經濟崩潰，糧食數量大減<ref>[[國際特赦組織|國際特赦組織]], supra note X, at 11</ref>。此外，金日成身邊的戰友也相繼離世，在這個環境下，金日成負荷增加。

1994年7月8日凌晨二時，金日成在[[平壤|平壤]]北部別墅突然心臟病發，其子[[金正日|金正日]]令[[直昇機|直昇機]]把金日成送到烽火醫院搶救。然而，這一晚正值大雨，能見度甚低，趕來的直昇機在半路中坠毀。最終，金日成在第二架直昇機趕到後才得以送院，但他還是回天乏術，心臟停止跳動而死<ref name = "朝鮮是個謎">{{cite book | title = 朝鮮是個謎 | author = 江迅}}</ref>{{rp | page = 285}}。

金日成死後34小時，北韓在7月9日中午12時官方正式公佈其死訊<ref name="mbc_kctv">{{cite news|url=http://imnews.imbc.com//20dbnews/history/1994/1933078_19434.html|title=북한 중앙방송과 평양방송, 김일성주석의 사망보도|publisher=[[文化廣播公司|文化廣播公司]]|date=1994-07-09|accessdate=2017-08-01|language=ko}}</ref><ref>{{cite news|url=http://imnews.imbc.com/20dbnews/history/1994/1933080_19434.html|title=김일성주석 사망보도, 내용과 과정 정리[정동영]|publisher=[[文化廣播公司|文化廣播公司]]|date=1994-07-09|accessdate=2017-08-01|language=ko}}</ref>，並由金正日主持為期一周的哀悼期。金正日又選了一張金日成在1986年[[西海水閘|西海水閘]]竣工儀式的照片作為哀悼期的全國統一領袖像。哀樂則是一首悼念當年[[抗日|抗日]]戰士用的曲子<ref name = "朝鮮是個謎" />{{rp | page = 286}}。

哀悼期間，全國共有近1000萬人到平壤弔唁金日成，佔北韓逾一半的人口<ref name = "朝鮮是個謎" />{{rp | page = 287}}；在全國各地追悼場地哀悼達2億人次<ref name = "朝鮮是個謎" />{{rp | page = 287}}。民眾又為金日成製作花圈，令平壤的鮮花一度全部售罄，野花也被採光，北韓當局只好從中國空運20萬[[人民幣|元]]的鮮花到平壤<ref name = "朝鮮是個謎" />{{rp | page = 287}}。

另外，由於金日成在事前並無任何病發的跡象，不少的民眾對於他的死也表現傷痛。據不少的[[脫北者|脫北者]]描述，當時人們在金日成像前長跪不起，放聲大哭，甚至哭得暈倒<ref name = "朝鮮是個謎" />{{rp | page = 287}}，還有的人因接受不了金日成的離世而[[自殺|自殺]]，为金日成[[殉死|殉死]]<ref name = "NothingToEnvy" />{{rp | page = 133}}。

7月17日是全國哀悼期的最後一天，金日成的遺體被永久保存於平壤的[[錦繡山議事堂|錦繡山議事堂]]。然而，民眾因金日成的死依未能平复，為了滿足人們悼念的需要，北韓政府決定順延哀悼期多兩天至7月19日<ref name = "朝鮮是個謎" />{{rp | page = 288}}。7月20日，平壤舉行追悼金日成的中央大會。隨後，北韓宣佈哀悼期正式結束，並呼籲民眾收拾心情，重新過回正常生活。

==反應==
===北韓===
[[朝鲜中央电视台|朝鲜中央电视台]]于1994年7月9日上午10时预告当天正午12时将进行“特别重大放送”<ref>{{cite web|url=http://www.dailynk.com/korean/read.php?cataId=nk09000&num=70089|title=북한 `중대방송' 허탈감 안겨줄 때도 많아|publisher=[[每日北韓|每日北韓]]|date=2009-04-09|accessdate=2017-08-02}}</ref>，后于12时整由播音员{{link-ko|田亨奎|전형규}}宣读《告全体党员和全国人民书》（{{Kor|名=北|諺=전체 당원들과 인민들에게 고함}}），首次播出金日成的死訊。<ref name="mbc_kctv"/>此後，北韓的媒體大幅度的報導此消息。電視新聞連續幾小時播放民眾痛哭的畫面，平壤電視播報員在節目中稱「我們的國家陷入朝鮮民族五千年來最深沉的悲痛之中」<ref name = "NothingToEnvy" />{{rp | page = 131}}。

北韓方面的宣传甚至稱金日成其實未死：「大元帥去世那一刻，數千隻祥鶴從天而降準備將他帶走。然而這些鳥兒看到北韓人民披頭散髮、痛哭流涕、捶胸頓足的樣子，又不忍心將他帶走了」<ref name = "NothingToEnvy" />{{rp | page = 132}}。還有的稱只要民眾哭得夠悲痛，金日成是會復活過來的<ref name = "NothingToEnvy" />{{rp | page = 132}}。

另一方面，為安定人心，北韓的媒體談起繼承人的事，電台廣播稱「只要我們親愛的金正日同志，偉大領袖的唯一繼承人，與我們在一起，我們的革命終將獲得勝利<ref name = "NothingToEnvy" />{{rp | page = 126}}。其他的宣傳中又強調「金正日將恪遵父親的遺志，為人民帶來不少的安慰」<ref>{{cite book | title = 北韓人眼中的北韓人 | author = 麥爾斯}}</ref>{{rp | page = 122}}。

==後續==
金日成死後，其子金正日繼承其位，成為[[朝鮮最高領導人|朝鮮最高領導人]]，並尊其父為「[[共和國永遠的主席|共和國永遠的主席]]」，[[守孝|守孝]]三年後在1997年繼承其父總書記的職務。掌握權力的金正日把其父之死歸咎於院方未有盡力救人，因而把相關的醫護人員和院長投進了監獄<ref name = "朝鮮是個謎" />{{rp | page = 285}}。

2011年12月17日，在位十七年的[[金正日之死|金正日逝世]]，他與其父的遺體一同保存於錦繡山議事堂（現名為[[錦繡山太陽宮|錦繡山太陽宮]]）供國民和遊客弔慰。兩年後，為特顯太陽宮的地位，北韓政府又通過《[[朝鮮民主主義人民共和國錦繡山太陽宮法|朝鮮民主主義人民共和國錦繡山太陽宮法]]》，規定錦繡山太陽宮是[[朝鮮民族|朝鮮民族]]的聖地，此處的供電、物資必須得到優先保障。

2014年7月10日，大韓民國傳媒引述消息人士指金日成遺體正在腐爛<ref>{{cite news |url = http://orientaldaily.on.cc/cnt/china_world/20140710/00180_005.html |title = 金日成遺體腐爛 「像乾鱈魚」 |publisher = [[東方日報_(香港)|東方日報]] |date = 2014-07-10 |language = zh-hant}}</ref><ref>{{cite news |url = http://hk.apple.nextmedia.com/realtime/international/20140710/52677215 |title = 金日成遺體疑正腐爛 「體型縮小了」 |publisher = [[蘋果日報_(香港)|蘋果日報]] |date = 2014-07-10 |language = zh-hant}}</ref>，頭部和身體萎縮，皮膚剝落及變色，身上出現很多黑斑，「像條脫水的乾鱈魚」。
