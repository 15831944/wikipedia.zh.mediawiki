'''金子有希'''（{{bd|1987年|1月19日|catIdx=Kaneko Yuuki}}<ref name="Nintama">{{Cite web|url=http://www.musical-nintama.com/cast#kuno|title=【ミュージカル忍たま乱太郎】第4弾『最恐計画を暴き出せ!!』 公式サイト » キャスト・スタッフ|accessdate=2012-12-07|deadlink=|archiveurl=https://web.archive.org/web/20120826161253/http://www.musical-nintama.com/cast#kuno|archivedate=2012年8月26日|deadurl=yes}}</ref><ref name="Aoni">{{Cite web |url=http://www.aoni.co.jp/junior/ka/kaneko-yuuki.html |title=青二プロダクション　金子有希 |accessdate=2012-12-07}}</ref><ref name="Aoni syousai"/>），出生於[[日本|日本]][[福岡縣|福岡縣]]<ref name="Aoni">{{Cite web |url=http://www.aoni.co.jp/junior/ka/kaneko-yuuki.html |title=青二プロダクション　金子有希 |accessdate=2012-12-07}}</ref><ref name="Aoni syousai"/><ref name="Blog profile" />，[[日本|日本]][[女性|女性]][[聲優|聲優]]、[[演員|舞台演員]]。隷屬於[[青二Production|青二Production]]（Junior）<ref name="Blog profile">{{Cite web |url=http://profile.ameba.jp/nekokanekoneko/ |title=金子有希のプロフィール |accessdate=2012-12-07}}</ref>，[[血型|血型]]B型<ref name="Blog profile"/>，身高162cm<ref name="Aoni syousai">{{Cite web |date= |url=http://www.aoni.co.jp/junior/ka/pdf/kaneko-yuuki.pdf |title=金子有希の詳細プロフィール |format=PDF |publisher=[[青二Production|青二Production]] |accessdate=2012-12-07}}</ref>。[[青二塾|青二塾]]大阪校25期生<ref name="Aoni"/><ref name="Aoni syousai"/>。

== 人物 ==
* 家中的獨生女。<ref>『[[玉子市場|玉子市場]] もちもちラジオ』第四回</ref>
* 小學時身高便已經來到159.8公分。<ref>『玉子市場 もちもちラジオ』第九回</ref>
* 喜歡起司。經常會在料理中放起司。<ref>{{Cite web |url=http://ameblo.jp/nekokanekoneko/entry-11479354267.html |title=魚缶焼き飯｜ねこねこかねこの気まぐれブログ。 |accessdate=2013-02-26}}</ref>
* 《[[玉子市場|玉子市場]]》是她第一次作為正式成員演出的電視動畫作品。其相關廣播節目、角色歌曲等周邊活動也都是她的第一次。<ref>{{Cite web |url=http://tamakomarket.com/diary/?p=167 |title=「ぺったんにいさん」｜『たまこまーこっと』制作日誌 |accessdate=2013-02-10}}</ref>
* 跟隸屬同一事務所、也參與過音樂劇《[[忍者亂太郎|忍者亂太郎]]》演出的[[津田美波|津田美波]]關係甚好。
