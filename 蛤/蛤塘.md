== 蛤塘村名字 ==
[[蛤|蛤]]，[[客家話|客家話]]可以有幾個意思；漢語言中「蛤」大致上有[[弎|弎]]個[[字義|字義]]，「[[蛤蜊|蛤蜊]]」（「[[蛤蠣|蛤蠣]]」）——生活在近海[[泥沙|泥沙]]之[[體外|體外]]有[[雙殼|雙殼]][[軟體動物|軟體動物]]（[[蛤蚌|蛤蚌]]）；「[[蛤蚧|蛤蚧]]」以及「[[蛤蟆|蛤蟆]]」（等於「[[蝦蟆|蝦蟆]]」）。
「蛤蜊」（「蛤蠣」、「蛤蚌」）、「蛤蚧」當中「[[蛤|蛤]]」讀音為[http://www.chineselanguage.org/dictionaries/ccdict/sounds/hakka/siyan/khap2.wav khap2/kap7]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}或者[http://www.chineselanguage.org/dictionaries/ccdict/sounds/hakka/siyan/kap2.wav kap2/gap7]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}。
「蛤蟆」（＝「[[青蛙|青蛙]]」、「[[蟾蜍|蟾蜍]]」）、「蝦蟆」當中「蛤」讀音為[http://www.chineselanguage.org/dictionaries/ccdict/sounds/hakka/siyan/ha11.wav hA2]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}。 

由於蛤塘村距離位於海唇之荔枝窩村尚且較遠，蛤蠣為海濱[[泥沙|泥沙]]中之小動物；因此蛤塘之蛤只有「蛤蟆」／「蛤蚧」（大[[壁虎|壁虎]]／[[大守宮|大守宮]]，[[熱帶|熱帶]]與[[亞熱帶|亞熱帶]]地區[[爬蟲類|爬蟲類]]動物）／「[[青蛙|青蛙]]」或者「[[蟾蜍|蟾蜍]]」其中一或兩個字義，「塘」之客家話與今日中文字義一樣，故而可推論「蛤塘」為青蛙（甚至[[牛蛙|牛蛙]]）出沒之[[山塘|山塘]]（池塘），村內亦相信有類似蛤蚧之爬蟲類出沒，因為蛤塘村背有[[客家風水林|客家風水林]]及[[原生山林|原生山林]]。

[[客家|客家]][[興寧|興寧]]人讀蛤塘之「蛤」為[http://www.chineselanguage.org/dictionaries/ccdict/sounds/hakka/siyan/ha11.wav hA2]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}（發音「[[哈|哈]]」）；佢兜人讀「蛤塘」為[http://www.chineselanguage.org/dictionaries/ccdict/sounds/hakka/siyan/ha11.wav hA2]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}[http://www.chineselanguage.org/dictionaries/ccdict/sounds/hakka/siyan/thong11.wav tong2]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}，「蛤嫲」為青蛙之意，連帶[[廣西省|廣西省]]客家話（當地人稱「[[新民話|新民話]]」）也以「蛤嫲」稱之。

== 地理位置 ==
蛤塘村村後有源自[[吊燈籠|吊燈籠]][[山頭|山頭]]（[[山坑|山坑]]）[[山澗|山澗]]，繞經蛤塘村口而注入[[荔枝窩|荔枝窩]]。

== 蛤塘客家鄉民生活點滴 ==
蛤塘村內之青磚樓房排列整齊<ref>[http://www.photo100fun.com/redirect.php?tid=6708&goto=lastpost 蛤塘－行山涉水－ photo100fun討論區－攝影路途並不孤獨，樂趣來自共同分享]</ref>，[[屋家|屋家]]內保存有較完整之[[木材|木材]]家具，可與[[西貢|西貢]][[北潭涌|北潭涌]][[上窰民俗文物館|上窰民俗文物館]]（[[博物館|博物館]]）之展示[[家具|家具]]相毗美，以及自五、六十年代以來見證[[香港工業|香港工業]]歷史變遷之手工精美製成品、[[印刷品|印刷品]]（包括六[[七十年代|七十年代]][[日曆|日曆]]、五[[六十年代|六十年代]]之[[火水油燈|火水油燈]]、1981年1月之[[香港地方行政白皮書|香港地方行政白皮書]]，以及皆為1981年之報紙、日曆，憑此可估計佢兜范氏客家鄉民於1981年遷移他方）。

蛤塘村由於沒有車輛可以駛入，因此村內沒有重大發展變化，如同[[邊境禁區|邊境禁區]](Frontier Closed Area)內之[[蓮麻坑|蓮麻坑]]（[[祖籍|祖籍]]：[[廣東|廣東]][[陸豐|陸豐]][[四縣腔|四縣腔]]）古老[[客家|客家]][[建築|建築]]群，得以更好地保存下來；例如村內傳統黃泥屋家、青磚民房（包括青磚瓦頂）、金字頂等等，予人進入[[清末|清末]][[民初|民初]]境地之感覺。

[[客家人|客家人]]較為重視[[家庭|家庭]][[倫理|倫理]]與[[氏族|氏族]]團結，所以其等於[[農曆新年|農曆新年]]期間，不計較蛤塘村之可達度以及偏僻程度，皆願意由（從）[[市區|市區]]、[[新界|新界]][[墟市|墟市]]、[[英國|英國]]、[[荷蘭|荷蘭]]等地之[[范氏|范氏]]客家[[原居民|原居民]]即會返到蛤塘村慶賀[[新春|新春]]佳節，然後才在假日過後返回[[佢丁人|佢丁人]]之移居地繼續事業；每逢節日蛤塘村都會有遷居他鄉或[[市區|市區]]之[[村民|村民]]回來[[掛紙|掛紙]]。2004年寒冬，曾有移居[[英倫|英倫]]之村民回來為進行整頓該山旮旯老村舊室、重立范氏[[宗祠|宗祠]]。
<ref>[https://web.archive.org/web/20080212164837/http://hk.news.yahoo.com/080211/12/2oo27.html 搶救禁區村莊歷史建築物 添香港品牌中西合璧色彩（《明報》二零零八年二月十二日禮拜二，年初六）]</ref>

== 對外遇上之阻隔 ==
由於蛤塘、[[梅子林|梅子林]]、[[荔枝窩|荔枝窩]]、[[鎖羅盤|鎖羅盤]]（鎖腦盤）、[[榕樹凹|榕樹凹]]、[[谷埔|谷埔]]、[[鳳坑|鳳坑]]（風坑）、[[三椏|三椏]]、[[三椏涌|三椏涌]]、[[犁頭石|犁頭石]]、上下[[苗田|苗田]]村（沙頭角[[船灣郊野公園|船灣郊野公園]]內之[[葉|葉]]氏[[客家|客家]][[原居民|原居民]]村落）、[[牛屎湖|牛屎湖]]、[[烏洲塘|烏洲塘]]村／[[戶洲塘|戶洲塘]]（沙頭角[[印洲塘|印洲塘]]以南海邊，客家荒廢[[村落|村落]]）、[[九擔租|九擔租]]（九担租）這一系列環[[船灣郊野公園|船灣郊野公園]]之客家原居民村落未有馬路前往，前往[[墟市|墟市]]、[[九龍|九龍]]、新界[[將軍澳|將軍澳]]、[[香港島|香港島]]極度不便。

蛤塘鄉民遇上緊急傷病痛之時難以前往[[公立醫院|公立醫院]]求診，[[政府飛行服務隊|政府飛行服務隊]][[直昇機|直昇機]]亦難以停靠於這系列[[慶春約|慶春約]]村落之空地，導致接送[[傷病者|傷病者]]相當費時費力。行山人士及郊遊人士、原居民在上述村落及相關郊野公園地帶難以透過[[手提電話|手提電話]]（俗稱[[手機|手機]]、手機欸）聯絡外界或者進行求救（只插有香港本地電話卡者），甚至出現[[訊號|訊號]]自動轉駁到內地電訊公司情況（使用中港兩地[[漫遊卡|漫遊卡]]者，或者插有只能在[[中國內地|中國內地]]及沿[[深圳河|深圳河]]、[[后海灣|后海灣]][[海濱|海濱]]（包括[[深圳灣口岸|深圳灣口岸]]）、[[藍塘海峽|藍塘海峽]]及[[擔桿列島|擔桿列島]]（擔竿列島，又名[[利瑪群島|利瑪群島]]）地帶使用之手提電話插卡），沿小徑亦未有[[全天候|全天候]][[999|999]][[緊急求助|緊急求助]][[通訊器|通訊器]]被安裝。

== 交通 ==
由[[大埔墟火車站|大埔墟火車站]]出發：乘搭約每小時一班[[小巴|小巴]]20R線到[[烏蛟騰|烏蛟騰]]村[[落車|落車]]，然後經過三條行山路線前往蛤塘村：
* 經過上、下[[苗田|苗田]]、[[三椏涌|三椏涌]][[營地|營地]]與[[三椏|三椏]]村（行下路，經過較多林蔭地帶）
* 經過[[犁頭石|犁頭石]]村與[[三椏|三椏]]村，（行上路，經過開揚多[[草|草]][[山徑|山徑]]，犁頭石村為古老客家村落，古[[屋家|屋家]]手工精緻，有高大[[樹木|樹木]]）
* 經過[[烏蛟騰|烏蛟騰]]田心村、[[亞媽笏|亞媽笏]]村（[[客家|客家]][[李|李]]氏），部分[[烏蛟騰郊遊徑|烏蛟騰郊遊徑]]路段、[[糞箕托|糞箕托]]（又名「芬箕托」；「糞箕」為[[客家話|客家話]]＝[[畚箕|畚箕]]）之西北及北邊，[[分水凹|分水凹]]前往之。
由[[粉嶺火車站|粉嶺火車站]]出發：可乘搭前往鹿頸（[[總站|總站]]為鹿頸路與新娘潭路交界）之56K[[小巴|小巴]]，於總站步行約兩又二份一小時至三個[[鐘頭|鐘頭]]步程攀山越嶺前往（可利用[[降下坳|降下坳]]燒烤場附近之[[烏蛟騰郊遊徑|烏蛟騰郊遊徑]]前往，經過[[荔枝窩|荔枝窩]]村）。<ref>{{Cite web |url=http://www.pfigaron.net/place4/p31_8.jpg |title=降下坳燒烤場 |accessdate=2008-02-14 |archiveurl=https://web.archive.org/web/20070107041351/http://www.pfigaron.net/place4/p31_8.jpg |archivedate=2007-01-07 |deadurl=yes }}</ref>（另外往返上水與沙頭角之九巴78K途經[[香港鐵路|香港鐵路]][[粉嶺火車站|粉嶺火車站]]，[[巴士站|巴士站]]位於[[粉嶺車站路|粉嶺車站路]]二號月台[[出入口|出入口]]旁邊、[[火車站|火車站]]往[[粉嶺名都|粉嶺名都]]天橋下面）<ref>[http://www.hkcities.com/forum/redirect.php?tid=9012&goto=lastpost 星期二去左吊燈籠行山，入黎睇相呀。] {{Wayback|url=http://www.hkcities.com/forum/redirect.php?tid=9012&goto=lastpost |date=20080320034102 }}</ref>
