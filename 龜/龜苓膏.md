'''龜苓膏'''是[[中國|中國]][[两广|两广]]特產，[[廣東|廣東]]、[[廣西|廣西]]一帶的傳統藥用食品。正統龜苓膏是以受保护动物[[三線閉殼龜|金錢龜]]、[[土茯苓|土茯苓]]、[[甘草|甘草]]等中藥材製成凝固的膏狀。

龜苓膏一般是黑色的膏狀，似[[凉粉|凉粉]]（[[仙草|仙草]]），不過有較濃的[[中藥|中藥]]苦味道，吃時可以加入少許鹽，中醫認為不應加糖。在[[夏天|夏天]]，龜苓膏冷吃會更美味。

現代的龜苓膏成品除了在[[涼茶|涼茶]]店、甜品店有售之外，在[[超級市場|超級市場]]也有[[罐頭|罐裝]]出售，不過，現在按食品衛生標準生產的龜苓膏，由於生產廠家及調製口味的不同，其藥物成分也與原始配方有別。現代的龜苓膏已經不會有金錢龜的成分了，連有沒有廉價草龜[[龜板|龜板]]成分也沒有保證，不過也很有效去對付不同的症狀。

== 配方 ==
800年前南宋末年，陆秀夫和御医商议，在凉粉草的基础上，将原来宫廷中一种清热去湿的养生食疗秘方加以改进，在鹰嘴龟、土伏苓、生地、蒲公英、金银花等原料中，加入新会崖山当地特有的“凉粉草”制成膏状药物，也是龟苓膏形成的雏形。

現在的龜苓膏配方如下：

*組方1： [[龜板|龜板]]、[[土茯苓|土茯苓]]、[[地黃|生地黃]]、[[金銀花|金銀花]]、[[茵陳|綿茵陳]]、[[火麻仁|火麻仁]]、[[甘草|甘草]]等。 
*組方2： 龜板、土茯苓、金銀花、[[生地|生地]]等。 
*組方3： 龜板、土茯苓、[[涼粉草|涼粉草]]、[[槐花|槐花]]、金銀花、[[蒲公英|蒲公英]]、[[菊花|菊花]]、[[苦瓜|苦瓜]]等。 
*組方4： 龜板、[[茯苓|茯苓]]、涼粉草、[[蜂蜜|蜂蜜]]、金銀花、蒲公英等。

== 功效作用 ==

龜苓膏的藥效主要來自[[龜板|龜板]]及[[土茯苓|土茯苓]]，龜板含有豐富的骨膠原，以金錢龜最有效，但因金錢龜為受保護動物，以其製成龜苓膏價錢昂貴且違法，因此現時大多數生產商的龜苓膏普遍要么以[[草龜|草龜]]及[[泥龜|泥龜]]代替，或是根本沒龜的成分。

*[[龜板|龜板]]，是[[龜|烏龜]]的腹板，性味甘、寒，有滋陰潛陽、補腎健骨、養血補心和止血的功效，可用於治療頭痛眩暈、盜汗遺精、手足震顫、腰膝酸軟、驚悸、失眠健忘、婦女崩漏及月經過多等。如果把龜板熬成固體膠塊，叫做龜板膠，其滋補力比龜板更強。由於龜板和龜板膠含有動物膠原、[[脂肪|脂肪]]、角蛋白、18種[[氨基酸|氨基酸]]，以及[[鈣|鈣]]、[[磷|磷]]、[[鋅|鋅]]等多種[[微量元素|微量元素]]，因此，它們能夠提高[[白血球|白細胞]]數量、增強免疫力，有補血、鎮靜、提高冠脈流量和提高耐缺氧能力等作用。

*[[土茯苓|土茯苓]]，性味甘、淡、平，有解毒、除濕、利關節的功效。從西醫角度來講，它還含有落新婦苷(Astilbin)、黃杞苷(Engeletin)、阿魏酸(ferulic acid)、[[澱粉|澱粉]]等，能夠起到利尿、鎮痛、抑菌、解[[汞毒|汞毒]]等作用。

當然，對於龜苓膏而言，僅僅是龜板和土茯苓還不夠，在這兩種主要原料的基礎上，加用[[金銀花|金銀花]]、[[蒲公英|蒲公英]]、[[菊花|菊花]]等清熱解毒、解暑、利濕的多味藥材才能更好地起到滋陰、解毒、解暑、解瘡毒等作用。

<sup>無藥理分析依據</sup>總括來說，{{fact|龜苓膏滋陰潤燥，降火除煩，清利濕熱，涼血解毒。用於正虛邪實、陰虛熱甚所致的虛煩、便秘、熱淋、白濁、癤腫、赤白帶下、皮膚搔癢、癤瘡紅腫等症。現代常用龜苓膏醫治多種口腔潰瘍、習慣性便秘、急慢性泌尿系感染、女性生殖系統炎症、某些神經功能障礙等症。}}

== 食用禁忌 ==

因爲龜苓膏可促進血液循環並屬于清涼解毒的食品，因此婦女於月經期間及孕婦不宜食用，體質虛弱者也不宜常食。
