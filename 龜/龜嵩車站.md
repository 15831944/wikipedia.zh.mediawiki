'''龜嵩站'''（{{jpn|j=亀嵩駅|hg=かめだけえき|rm=Kamedake eki|f=y}}）是[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）[[木次線|木次線]]的一座[[鐵路車站|鐵路車站]]，位於[[日本|日本]][[島根縣|島根縣]][[仁多郡|仁多郡]][[奧出雲町|奧出雲町]]。

== 概要 ==
開業時的所在地為仁多郡龜嵩村<ref>{{cite web|author={{lang|ja|三島幸子、今冨良介、牛島朗、佐々木英臣、中園眞人、細田智久}}|url=http://www.nakazono.lab.kde.yamaguchi-u.ac.jp/journal/shibu/2018/mishima.pdf|script-title=ja:町村合併と公共公益施設の立地動向との関連 -島根県仁多郡奥出雲町の事例研究 その3-|publisher={{lang|ja|日本建築学会中国支部研究報告集}}第41巻|date=2018-03|accessdate=2018/08/14}}</ref>。該站的站舍現今也是一座[[蕎麥麵|蕎麥麵]][[餐館|餐館]]，站長同時擔任該餐廳的店長<ref name=soba/>，乘客還可以事先預約該站的蕎麥麵當作[[鐵路便當|鐵路便當]]<ref name=asahitv/>。

站前有國道432號線經過，利用者主要是當地的高中生<ref>{{cite web|url=http://www.pref.shimane.lg.jp/unnan_kendo/nita/jigyou.data/kamedake.pdf|script-title=ja:一般国道432号 亀嵩工区 完成 仁多郡奥出雲町郡~亀嵩地内|publisher={{lang|ja|雲南県土整備事務所仁多土木事業所道路建設課}}|accessdate=2018/08/14}}</ref>。[[道之驛|道之驛]]酒藏奥出雲交流館亦位於本站附近<ref name=soba/>。

== 歷史 ==
* 1934年11月20日：木次線從出雲三成站延伸至{{link-ja|八川站|八川駅}}，本站開業<ref name=okuizumo>{{web cite|url=http://www.town.okuizumo.shimane.jp/files/pdf/kouhou/H28-09/p06-07.pdf|script-title=ja:木次線開業 地域とともに100年の歩み|publisher=奥出雲町|accessdate=2018/08/14}}</ref><ref>{{Cite web|url=http://kisuki-line.jp/history/year|script-title=ja:木次線利活用推進協議会年表|accessdate=2018-08-14|date=|publisher={{lang|ja|木次線利活用推進協議会}}|language=ja}}</ref>。
* 1973年：當時的站長在站內開設名為「扇屋」的蕎麥麵館<ref name=soba>{{cite news|url=http://www.asahi.com/travel/hitoekigatari/TKY201205140164.html|script-title=ja:駅長手打ち 出雲そばいかが 亀嵩駅（島根県、JR木次線）|publisher=[[朝日新聞|朝日新聞]]|accessdate=2018/08/14}}</ref>。
* 1987年4月1日：[[日本國鐵分割民營化|日本國鐵分割民營化]]，車站管理權由西日本旅客鐵道承繼<ref name=okuizumo/>。
* 2007年4月：為吸引遊客，JR西日本米子支社為木次站至{{link-ja|三井野原站|三井野原駅}}間各站指定來源於[[記紀|記紀]]兩書的愛稱<ref>{{cite news|url=http://www.asahi.com/kansai/travel/ensen/OSK200907040044.html|script-title=ja:山と海 鉄師が結んだ ＪＲ木次線|publisher=[[朝日新聞|朝日新聞]]|accessdate=2018/08/14}}</ref>，本站的愛稱為「[[少彦名命|少彦名命]]」<ref>{{cite web|url=http://www.hiikawa-summit.info/orochi/station/11.html|script-title=ja:停車駅のご案内 - 亀嵩駅|publisher={{lang|ja|出雲の國・斐伊川サミット事務局  ([[出雲市|出雲市]]総合政策部政策企画課)}}|accessdate=2018/08/14}}</ref>。

==與《砂之器》的關係==
該站在[[松本清張|松本清張]]的[[推理小說|推理小說]]《[[砂之器|砂之器]]》中有著重要的地位。因罹患[[漢生病|漢生病]]而要被送去[[漢生病療養院|療養院]]隔離的父親與兒子在該站的站台上告別，而以東北方言說出的本站站名也是最初的被害人留下的唯一證據。在1974年的同名電影播出後，更是吸引到許多松本作品的愛好者前來拜訪。但在這部電影中，作為龜嵩站站台的取景地則是附近的{{link-ja|出雲八代站|出雲八代駅}}。<ref>{{cite news|url=https://www.yomiuri.co.jp/life/travel/meigen/20160212-OYT8T50031.html|script-title=ja:宿命とは、この世に生まれて…映画「砂の器」|date=2016年02月15日|publisher=[[讀賣新聞|讀賣新聞]]|accessdate=2018/08/14}}</ref><ref>{{cite news|url=https://www.sankei.com/west/news/171228/wst1712280073-n1.html|script-title=ja:【鉄道アルバム・列車のある風景】JR木次線／映画・砂の器の舞台にて|date=2017.12.28|publisher=[[產經新聞|產經新聞]]|accessdate=2018/08/14}}</ref>

在車站附近的湯野神社前方有松本清張親筆題寫的「小說砂之器舞台之地」紀念碑<ref name=soba/>。

== 利用狀況 ==
民營化後，本站於1993年度的每日平均乘車人數曾達到87人，此後則呈下降趨勢<ref name=shimane/><ref>{{web cite|url=http://pref.shimane-toukei.jp/index.php?view=9471|script-title=ja:島根県統計書各年版|publisher={{lang|ja|しまね統計情報データベース}}|accessdate=2018/08/14}}</ref>。
