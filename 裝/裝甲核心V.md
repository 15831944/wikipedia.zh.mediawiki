{{Vgname|裝甲核心V|ja=アーマード・コア V|en=Armored Core V}}是[[機戰傭兵系列|機戰傭兵系列]]睽違4年的第五款正宗續作。世界觀翻新。[[廣告標語|廣告標語]]是「{{lang|ja|全てを焼き尽くす暴力}}」（能把一切燃燒殆盡的暴力）。是該系列首次有正體中文化。

== 玩法 ==

製作人鍋島俊文表示前作已試過把高速戰鬥推展到極限，地形幾乎失去意義，因此這次刻意另闢途徑<ref name="鍋島談起ACV延期">{{cite web|url=http://gigazine.net/news/20110825_nabeshima_interview/ |title=「アーマード・コアV」発売延期の真意を語る鍋島プロデューサーインタビュー |author=takamura |work=|publisher=gigazine.net |date=2011-08-25 |accessdate=2012-05-03}}</ref>，在多处做了变革。如機體尺寸由10[[公尺|公尺]]左右縮為7[[公尺|公尺]]好強化地形戰<ref name="鍋島談起ACV延期" />、實彈武器比照現實軍武分成[[:en:Kinetic_energy_penetrator|KE]](運動能量彈)與CE([[化學能|化學能]])<ref name="鍋島談起ACV延期" />、飛踢<ref name="鍋島談起ACV延期" />、只能靠三角跳攀升<ref name="PS3版封測報告">{{cite web|url=http://gigazine.net/news/20110728_armored_core_5_cbt/ |title=「アーマード・コアV」PS3版クローズドベータ終了、革新的なロボゲーの予感 |author=takamura |work=|publisher=gigazine.net |date=2011-07-28 |accessdate=2012-05-08}}</ref>、威力巨大的超限武器<ref name="ACV印象">{{cite web |url=http://www.4gamer.net/games/103/G010386/20120125038/ |title=これまでのACとは一味違う。フロム・ソフトウェア渾身の一作「ARMORED CORE V」のインプレッションを掲載 |author=Leyvan |date=2012-01-26 |work= |publisher=4gamer |accessdate=2012-05-08}}</ref>、關閉武器來節約能量並探索週遭的掃描模式<ref name="ACV印象" />等眾多新要素。日本版說明書因此厚達60頁<ref name="ACV的理由">{{cite web|url=http://www.4gamer.net/games/103/G010386/20120120009/index_3.html |title=フロム・ソフトウェアってどんな会社ですか。プロデューサー鍋島俊文氏を通して見えた「ARMORED CORE V」を作れた理由 |author=TAITAI |work=|publisher=4gamer |date=2012-01-25 |accessdate=2012-05-08}}</ref>。

=== 連線 ===

本作中连线系统的设计增强，除了教学任务与机体测试外的出击都可以连线合作。玩家在线上可加入别人的队伍壮大声势，并通过耳麦互相交谈；而無隊友配合時則透過「[[傭兵|傭兵]]」登錄機制配對。

導入領地戰。领地依大地图被分成8个区块，不同領地即為不同舞台。领地战时防守方与侵攻方的出发地点固定。任务完成后可累积团队点数，消耗掉特定點數(越高階隊伍就需要越多)可以发动决战任务，打赢了可以拿到舞台进行客制化再上传给其他队伍挑战，自己也能撩下去支援。最多5vs5，但這時雙方其中一人將負責戰情通信<ref>[http://www.gameinformer.com/games/armored_core_v/b/xbox360/archive/2011/02/02/armored-core-v-coming-stateside-in-2012.aspx Armored Core V Coming Stateside In 2012 - Armored Core 5 - Xbox 360]</ref>。成功退敌会增加领地评价，决定排名，该区第一名的队徽会出现在大家的大地图上；反过来说，持续被攻陷时耐久度会降低，降至0时得以领地为赌注开决战任务，赢了耐久度补满，输了就失去这块领地。棄之不顧會隨時間經過被沒收。

官方也會於特定時段(多半挑假期)釋出讓線上中高階隊伍合力挑戰的額外任務<ref name="ACV的理由" />，共五大種。另外，連線時購入的槍砲類武器可以選擇成長方向<ref name="ACV的理由" />。在在鼓勵玩家連線。

官方伺服器已2014年關閉。

== 世界觀 ==

《装甲核心V》一反過往的高科技，這次的ARMORED CORE都是從地下挖掘出來後再修復的古物。故事任務由第0章－第9章，共10關，屬於大型任務，有補給站與中繼點。剩下83個指令任務則是後日談，屬於小型任務，約5~10分內可過關<ref name="FAMI通訪談">{{cite web |url=
http://www.famitsu.com/sp/111215_armoredcore/ |title=AROMRED CORE V SPECIAL SCOOP |author= |date=2010-03-29 |work= |publisher=FAMI通 |accessdate=2012-08-06}}</ref>。

游戏主人公为傭兵，在剛出道時被企業雇用對付反抗組織，但備受冷遇。一年後改接受反抗組織的雇用。游戏还有都市、企业、候鳥、蜜蜂巢(Bee Hive)、MoH(Men of Honor)和[[黃道帶|索迪亞克]]等势力。

== 开发与发行 ==

[[From_Software|From Software]]最早在2010年初公佈[[機戰傭兵系列|機戰傭兵系列]]將推出新作<ref>[http://www.youtube.com/watch?v=BUbMDbwwRIg Armored Core 5 - Debut Trailer|youtube.com]</ref>，當時的[[標誌|標誌]]是紅黃交織的[[印度數字|印度數字]]５，簡稱AC5。但之後就了無音訊，只有持續地把舊作移植到[[掌上型遊樂器|掌上型遊樂器]]。直到時隔一年的2011年[[From_Software|From Software]]才打破沉默，除了公佈海外經銷交給[[萬代南夢宮|萬代南夢宮]]外，[[標誌|標誌]]也成了白色的[[羅馬數字|羅馬數字]]V<ref>{{cite web |url=http://gnn.gamer.com.tw/0/50780.html |title=《機戰傭兵 V》硬派機器人對戰遊戲年內登場 嶄新連線組隊對戰玩法|author=Sam |date=2011-02-08 |work= |publisher=gamer.com |accessdate=2012-05-03}}</ref>，甚至連網路宣傳轉播MIDNIGHT LIVE<ref>{{cite web|url=http://www.famitsu.com/news/201201/10008160.html |title= 『アーマード･コア V』のUSTREAM生放送番組“Midnight LIVE Presents 【ARMORED CORE V】～LAST BULLET～”の配信が決定|author= |date=2012-01-10 |work= |publisher=ファミ通.com |accessdate=2012-05-03}}</ref>最後都會讓三人擺出ACV三個字的姿勢。

起因是[[From_Software|From Software]]在2009年推出[[惡魔靈魂|惡魔靈魂]]，當初曾因為背離潮流而不被看好，後來卻傳出口碑叫好叫座<ref>{{cite web|url=http://bestof.ign.com/2009/ps3/best-role-playing-game.html |title= IGN：BEST OF 2009, Best Role-Playing Game |author= |date=|work= |publisher=IGN|accessdate=2012-05-03}}</ref>，從歐美紅回日本。跟[[From_Software|From Software]]有業務往來的日本玩具巨擎[[萬代南夢宮|萬代南夢宮]]因此表達了合作意願，可以提供資金與全球經銷－只要能做出全球都能賣的大作。

[[From_Software|From Software]]因此把即將完工的AC5收回去重作。大幅度補強了連線功能，時隔一年後完成的就是《機戰傭兵V》。但2011年先是發生311[[東日本大震災|東日本大震災]]，接著又是[[PSN|PSN]]遇駭<ref>{{cite web|url=http://jp.reuters.com/article/technologyNews/idJPJAPAN-20830920110427 |title= ソニーＰＳネットに不正侵入、7700万人の情報流出の可能性 |author= |date=2011-04-27 |work=|publisher=日本路透社 |accessdate=2012-05-03}}</ref>，最重要的[[連線|連線]]實測只得延到下半年<ref name="PS3版封測報告" />，再加上發布體驗版後修正用戶意見，最後遲至2012年才正式發售。

=== 英語版 ===

美版順利在[[ESRB|ESRB]]爭取到T級（相當於[[電腦及遊戲軟體分級辦法#分級|輔導級]]）。但有若干內容因[[政治正確|政治正確]]考量與日版有出入。要角RD在日版是年輕人，美版年長了許多。指令任務09、29、40、45、51、52、56、57、62、65、66、70、78的性別安排也有變化。中文版亦以此為準。
