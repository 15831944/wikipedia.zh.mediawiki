'''戈瑞'''（{{lang-en|Gray}}，缩写符号：{{lang|en|Gy}}，[[中国大陆|中国大陆]]译作“-{戈瑞}-”，[[台湾|台湾]]译作“-{戈雷}-”；亦有譯作“-{格雷}-”<ref name="HKO_edu">{{Cite web|url=http://www.hko.gov.hk/prtver/html/docs/education/edu02rga/radiation/radiation_09_uc.shtml|title=輻射術語詞彙及參考資料|publisher=[[香港天文台|香港天文台]]|author=趙孔儒|date=2011年6月|language=zh-hant}}</ref>），简称「戈」，是一个[[国际单位制导出单位|国际单位制导出单位]]，是物理量「[[电离辐射|电离辐射]]能量[[吸收剂量|吸收剂量]]」的标准单位。<ref>{{Cite web|url=http://www.aec.gov.tw/www/knowledge/index_03_2-3.php|title=輻射小掌故|publisher=[[行政院原子能委员会|行政院原子能委员会]] 原子能教室|author=翁寶山|accessdate=2012-11-09|language=zh-hant}}</ref>

== 定义 ==
戈瑞（符号：Gy或㏉）是用于衡量由[[电离辐射|电离辐射]]导致的「能量吸收剂量」（简称[[吸收剂量|吸收剂量]]）的物理单位，它描述了单位[[质量|质量]]物体吸收电离辐射能量的大小。一戈瑞（1 Gy）表示每公斤物質吸收了一焦耳的輻射能量<ref name="HKO_edu"/>。

除此之外，戈瑞也是物理量{{link-en|比释动能|Kerma (physics)}}的单位。

戈瑞的定义式是：
<math>\mathrm{1\, Gy = 1\,\frac{J}{kg} = 1\,\frac{m^2}{s^2}}</math>

即：1 戈瑞= 1 [[焦耳|焦耳]]/千克

其中「焦耳」是[[能量|能量]]的单位，「千克」是[[质量|质量]]的单位。用[[国际单位制基本单位|国际单位制基本单位]]表示为“[[米|米]]平方每秒平方”。

== 名称来源 ==
戈瑞之名来自[[英国|英国]][[物理学|物理学]]家、[[放射生物学|放射生物学]]之父[[路易斯·哈罗德·戈瑞|路易斯·哈罗德·戈瑞]]。

== 换算 ==
早前使用的吸收剂量单位是{{link-en|拉德_(单位)|Rad_(unit)|拉德}}（rad），它与戈瑞间的转换关系是：1Gy = 100 Rad。

1985年12月31日起，「拉德」在医学领域终止使用。

仅在描述[[X射线|X射线]]、[[伽马射线|伽马射线]]、[[β粒子|贝塔射线]]的辐射剂量时，戈瑞和另一个单位[[西弗|西弗]]（Sievert，中国大陆法定译作[[希沃特|希沃特]]）是等价的，因为这几种辐射的「辐射权重因数」都是1<ref>{{Cite web|url=http://www.guokr.com/article/12999/|title=【地震特辑】西弗到底是什么？|publisher=[[果壳网|果壳网]]|date=2011-03-15|language=zh-hans}}</ref>。二者单位相同，但戈瑞在实际应用中用于描述辐射[[吸收剂量|吸收剂量]]的大小，希沃特则描述[[等效剂量|当量剂量]]。

== 使用 ==
戈瑞主要应用在医学领域，描述[[放射线疗法|放射线疗法]]以及[[核医学|核医学]]中使用的[[辐射剂量|辐射剂量]]。
