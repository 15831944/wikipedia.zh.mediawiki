'''戈耳工'''（''Gorgon''，或者译作'''蛇发女妖'''，意为可怕的，另有人谓为高声咆哮的），是一種长有尖牙，头生毒蛇的女性怪物。戈爾貢最早出現在[[希腊神话|希腊神话]]中，指的是三姊妹[[斯忒諾|斯忒諾]]、[[歐律阿勒|歐律阿勒]]和[[美杜莎|美杜莎]]，據說她們是海神[[福耳庫斯|福耳庫斯]]與女神[[刻托|刻托]]之女<ref>[[赫西俄德|赫西俄德]]. 《[[神譜|神譜]]》第276頁</ref><ref>Pseudo-Apollodorus. 《[[書庫|書庫]]》；Book 1.2.6, 2.4.2''</ref><ref name=":0">[[品達|品達]].《Pythian Ode》，12.20</ref><ref name=":1">[[Nonnus|Nonnus]]《[[Dionysiaca|Dionysiaca]]》；Book 25.58</ref>。

== 字義 ==
[[古希臘語|古希臘語]]單數：Γοργώ，複數：Γοργόνες。字源來自於動詞：γόργω"恐怖、可怕"。如字義所述，戈耳工们是一群讓人感覺可怕又恐懼的傢伙。

== 历代传闻 ==
和現今大眾文化中流傳的戈爾貢形象（頭生毒蛇的女性怪物）不同，最早在希臘神話中登場的戈爾貢有著不一樣的面貌。有些传闻将祂们描绘成身长金色羽翼、黄铜利爪、野猪獠牙。在[[神话|神话]]中，看到戈耳工颜面的人会化为石头。[[荷马|荷马]]（''Homer''）在《[[伊利亚特|伊利亚特]]》（''ΙΛΙΑΣ'',''Iliad''）中只论及一个戈耳工，祂的头颅被嵌在[[宙斯|宙斯]]的神盾[[埃癸斯|埃癸斯]]（''Aegis''）之中。在《[[奥德赛|奥德赛]]》（''Odyssey''）中，这个戈耳工被描绘成一个地底妖怪。[[赫西俄德|赫西俄德]]（''Hesiod''，公元前8-7世纪）将戈耳工的数量增加到三个—[[斯忒諾|斯忒諾]]（''Stheno''，富有力量的）、[[歐律阿勒|歐律阿勒]]（''Euryale''，「遙遠的漫遊」或「廣闊的大海」）、[[美杜莎|美杜莎]]（''Medusa''，皇后），并使她们成为[[海神|海神]][[福耳库斯|福耳库斯]]（''Phorcys''）和[[刻托|刻托]]（''Ceto''）的女儿。据后来来自[[利比亚|利比亚]]的可靠资料称，她们的居所在[[西洋|西洋]]的辽远之处。在由[[欧里庇得斯|欧里庇得斯]]（''Euripides'', 480B.C. - 406B.C.）所重新上演的[[雅典|雅典]]传统剧目里戈耳工成了一个妖怪，祂由[[盖亚|盖亚]]（''Gaia''）所创，来辅助其几个儿子对付众神；为[[雅典娜|雅典娜]]（''Athena''，相当于[[罗马神话|罗马神话]]中[[密涅瓦|密涅瓦]]）所杀。

== 美杜莎 ==
根据[[奥维德|奥维德]]（''Ovid''）的《[[变形记|变形记]]》（''Metamorphoses''）所述，只有[[美杜莎|美杜莎]]的头发上有蛇，这全拜[[密涅瓦|密涅瓦]]（''Minerva'',[[雅典娜|雅典娜]]）的诅咒所赐。在[[密涅瓦|密涅瓦]]的神殿中，[[海神|海神]]（[[罗马神话|罗马神话]]作''Neptune''，[[希腊神话|希腊神话]]为[[波塞頓|波塞頓]]''Poseidon''）曾为[[美杜莎|美杜莎]]的一头金发所诱惑，并与之交合。因此[[密涅瓦|密涅瓦]]将这一头诱人的金发变成了条条毒蛇。[[埃斯库罗斯|埃斯库罗斯]]（''Aeschylus'', 525B.C.- 456B.C.）认为三个戈耳工只有一颗牙齿、一只眼睛，祂们只好轮换着用（又见[[格赖埃|格赖埃]]（''Graeae''））。

在三个戈耳工当中只有[[美杜莎|美杜莎]]是肉身，故[[珀尔修斯|珀尔修斯]]（''Perseus''）能够将其杀死。[[珀尔修斯|珀尔修斯]]从[[格赖埃|格赖埃]]手里得到的镜之盾，透过镜面看清[[美杜莎|美杜莎]]，才得以割下她的头颅。一些作家认为[[珀尔修斯|珀尔修斯]]所装备的是[[墨丘利|墨丘利]]（''Mercury''，希腊神话作[[赫耳墨斯|赫耳墨斯]]''Hermes''）给的长柄大镰和[[密涅瓦|密涅瓦]]给的镜子。不管是镜之盾还是长柄大镰，这些武器都使得他可以轻松打败[[美杜莎|美杜莎]]。她与[[波塞頓|波塞頓]]所生的两个儿子[[克律萨俄耳|克律萨俄耳]]（''Chrysaor''）和[[珀伽索斯|珀伽索斯]]（''Pegasus''）从她脖子上溅出的血里跳了出来。[[美杜莎|美杜莎]]的头颅有着将一切正视她的人变成石头的力量。[[珀尔修斯|珀尔修斯]]把头颅给了[[雅典娜|雅典娜]]而后者把它嵌在她的盾上；而据另一种说法，[[珀尔修斯|珀尔修斯]]将头颅埋在[[阿耳戈斯|阿耳戈斯]]（一古希腊城邦）的市集之中。

== 戈耳工头像 ==
[[女魔脸形饰|女魔脸形饰]]（''gorgoneion''，或名之曰石首，上刻有或绘有戈耳工脸孔：常常是舌头穿过尖牙，直吐于外，并有蛇在头上芜乱地凸现出来）经常摆设在门、墙、盾、护胸及墓碑之上，用以辟邪。就这方面来说，戈耳工头像和中国士兵盾牌上风格有时有些奇异的脸谱有些类似，那也被广泛地当作护身符用，它可以辟祛恶魔的眼睛。{{来源请求}}

== 戈耳工的魔力 ==
在[[希腊神话|希腊神话]]中，戈耳工的右侧身体的血有起死回生之妙，而左侧的则是迅速致命的毒药。[[雅典娜|雅典娜]]给了[[阿斯克勒庇俄斯|阿斯克勒庇俄斯]]一小瓶有回復作用的血液，这瓶血液却最终致他于死地。

据说[[赫剌克勒斯|赫剌克勒斯]]（''Heracles''）从[[雅典娜|雅典娜]]那裡得到过一绺[[美杜莎|美杜莎]]的头发（祂的头发拥有和头颅一样的威力），并把它给了[[格佛厄斯|格佛厄斯]]（''Cepheus''）的女儿[[丝黛罗普|丝黛罗普]]（''Sterope''），以保护[[帖该亚|帖该亚]]城（Tegea）免遭攻击。

== 石頭與青銅 ==
石頭（λίθος）似乎沒有比青銅（χαλκός）硬，又或者從物理觀點來看，人類和動物的骨骼歷經時間演化會石化成化石，而青銅這類的金屬則不會，所以[[美杜莎|美杜莎]]的石化功一遇到青銅物都無用武之地。書庫有明顯記載，[[雅典娜|雅典娜]]借給[[珀爾修斯|珀爾修斯]]（Περσεύς）青銅盾（ἀσπίδα χαλκῆν），盾面光滑如鏡，[[珀爾修斯|珀爾修斯]]借反射影像避免"直接"看到[[美杜莎|美杜莎]]的臉，所以青銅"間接"過濾掉石化的威力。又[[赫剌克勒斯|赫剌克勒斯]]借來蛇髮也將他存放在青銅水壺（ὑδρία χαλκῆ）裡。只有[[雅典娜|雅典娜]]的青銅盾能裝置[[美杜莎|美杜莎]]的頭。

== 后世评说 ==
后世的人们将[[美杜莎|美杜莎]]视为是一个美丽的女仆，后来头发被[[雅典娜|雅典娜]]变作了毒蛇。在艺术作品中，她的头颅为静寂的安眠所缠绕，被描绘得极其俊美。
