'''戈氏姬地鸠'''（[[学名|学名]]：{{lang|la|''Geopelia placida''}}）是[[鸠鸽科|鸠鸽科]][[姬地鸠属|姬地鸠属]]的一种，原生于[[澳大利亚|澳大利亚]]和[[新幾內亞|新几内亚]]。 本种与[[东南亚|东南亚]]的[[斑姬地鸠|斑姬地鸠]]和[[帝汶姬地鸠|帝汶姬地鸠]]的为近缘物种，三者曾被当成同一物种。

== 形态 ==
[[File:Peaceful_Dove_Cairns_RWD.jpg|左]]]]
本种是相对较小的鸽类，体长从从19到21厘米（7.6至8.4英寸）不等。胸部呈灰红色，[[翅膀|翅膀]]有灰褐色的花纹， 颈部至后背被黑色的细条纹所环绕。眼睛为浅灰色至白色，蓝灰色条纹从眼周延伸至[[喙|喙]]。[[亚成体|亚成体]]身上的斑纹暗淡，它们的眼眶颜色也较浅。近缘物种[[斑肩姬地鸠|斑肩姬地鸠]]的颈背[[羽毛|羽毛]]也有相似的条纹，但其喉咙部位并没有像本种一样的横纹。此外，本种颈背的羽毛偏灰褐色，而斑肩姬地鸠则为鲜艳的古铜色。

戈氏姬地鸠的叫声变化有高音“嘟啲嘟”、“咕咕咕”及响亮的“咕噜”等等。

== 习性 ==
[[File:Dove,_Peaceful_1.jpg|右]]中部[[阿翁卡湖|阿翁卡湖]]的戈氏姬地鸠]]
戈氏姬地鸠经常成对或小群出没、在地面上觅食，常见于公园及街道上。灌木丛、[[疏林|林地]]、[[流水|水道]]、铁轨间、[[雨林|热带雨林]]边缘地带均有其踪影。当不在地上觅食时，就会在树木上栖息。

本种[[飛行|飛行]]迅速、呈波状运动，翅膀同时会发出呼啸声。

== 筑巢 ==
10月到1月为本种在澳大利亚南部的繁殖季节，在澳大利亚北部则从3月至6月。鸠[[巢|巢]]一般都筑在树木的横干上，会下两个白色的蛋。繁殖期的戈氏姬地鸠会对其它鸟类具有攻击性。

== 分布 ==
本种的分布范围遍及整个澳洲，只有[[塔斯馬尼亞州|塔斯马尼亚]]、[[维多利亚州|维多利亚州]]南部和澳洲西南部除外。然而，由于受外来种[[珠颈斑鸠|珠颈斑鸠]]（{{lang|la|''Streptopelia chinensis''}}）竞争影响，本种的分布范围正在缩小。在新几内亚，它们主要分布于本岛的南部，北部几个地方和[[阿魯群島|阿鲁群岛]]也有分布。
