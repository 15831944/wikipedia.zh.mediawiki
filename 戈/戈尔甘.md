'''戈尔甘'''（{{lang-fa|'''گرگان'''}}、[[马赞德兰语|马赞德兰语]]：'''{{lang|mzn|Vərgan}}'''）是[[伊朗伊斯兰共和国|伊朗伊斯兰共和国]][[戈勒斯坦省|戈勒斯坦省]]的[[省会|省会]]和最大[[城市|城市]]，位于[[里海|里海]]沿岸。2005年戈尔甘估算人口241,177<ref>
[http://www.mongabay.com/igapo/2005_world_city_populations/Iran.html 伊朗人口]</ref>。戈尔甘以东150[[公里|公里]]处有[[戈勒斯坦国家公园|戈勒斯坦国家公园]]。

== 词源 ==
戈尔甘在[[古希腊|古希腊]]史料中被称为[[希尔卡尼亚|希尔卡尼亚]]，来源于[[中古波斯语|巴列维语]]“瓦尔卡纳”（{{lang|fa|Varkâna}}），意为“群狼之地”。尽管今日的戈尔甘只是一座城市和一个县，但古代希尔卡尼亚包括了[[里海|里海]]南岸的广大地区（包括今日[[戈勒斯坦省|戈勒斯坦省]]的全部、[[马赞德兰省|马赞德兰省]]的东部和[[土库曼斯坦共和国|土库曼斯坦共和国]]的南部）。近代以来直至1937年，戈尔甘被称为'''阿斯塔拉巴德'''（{{lang-fa|'''استرآباد'''}}）。

== 地理和气候 ==

辽阔的[[戈尔甘草原|戈尔甘草原]]位于戈尔甘北部，跨度为[[北纬|北纬]]37°00'－37°30'、[[东经|东经]]54°00'－54°30'，[[面积|面积]]约1,700[[平方公里|平方公里]]。

戈尔甘所在的戈勒斯坦省气候大体上温和湿润，被称为“温和的里海气候”，形成这种气候的因素有：[[阿尔布尔士山脉|阿尔布尔士山脉]]、山脉的走向、海拔高度、近海（[[里海|里海]]）、植被、当地的风和气候前线。由于上述因素，在这一地区形成了三种不同的气候：温和的草原气候、高山气候和半干旱气候。[[戈尔甘河|戈尔甘河]]流域属于半干旱气候。戈尔甘年平均[[气温|气温]]18.2[[摄氏度|摄氏度]]，年[[降水量|降水量]]556毫米。

== 历史 ==
{{Main|希尔卡尼亚}}
[[File:Gorgan_ceramic.JPG|thumb]]
希尔卡尼亚的都城是最大城市及王宫所在地“扎德拉卡塔”，很可能位于今天的戈尔甘城西边<ref>[http://www.iranica.com/newsite/index.isc?Article=http://www.iranica.com/newsite/articles/unicode/v11f2/v11f2034a5.html 《伊朗百科全书》“戈尔甘”词条]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在[[萨珊王朝|萨珊王朝]]时期，戈尔甘成为一座城市、一座省会和一个省的名字<ref name="Iranica-Bivar">Bivar, A.D.H.，《伊朗百科全书》“戈尔甘”词条[http://www.iranica.com 在线]</ref>。

[[阿契美尼德王朝|阿契美尼德王朝]]的[[居鲁士大帝|居鲁士大帝]]或[[冈比西斯二世|冈比西斯二世]]在位时期，希尔卡尼亚成为[[波斯帝国|波斯帝国]]的一部分。8世纪时这里被[[阿拉伯人|阿拉伯人]]征服，而13世纪时又被[[蒙古人|蒙古人]]征服，[[萨非王朝|萨非王朝]]以后复归[[波斯|波斯]]（[[伊朗|伊朗]]）。

戈尔甘原属[[马赞德兰省|马赞德兰省]]，1997年[[戈勒斯坦省|戈勒斯坦省]]从马赞德兰省中分出后成为戈勒斯坦省的[[省会|省会]]。

== 文化 ==
=== 手工艺 ===
戈尔甘和戈勒斯坦省[[土库曼人|土库曼人]]的[[波斯地毯|织毯业]]举世闻名，其模型来源于古代波斯都市[[布哈拉|布哈拉]]（今属[[中亚|中亚]][[乌兹别克斯坦共和国|乌兹别克斯坦共和国]]）。

== 交通 ==
戈尔甘的[[港口|外港]]是[[里海|里海]]边的[[托尔卡曼港|托尔卡曼港]]（旧称[[沙赫港|沙赫港]]），通有[[铁路|铁路]]，另有[[公路|公路]]西至[[马赞德兰|马赞德兰]]的[[萨里市|萨里市]]，东至[[呼罗珊|呼罗珊]]的[[马什哈德|马什哈德]]。

[[戈尔甘国际机场|戈尔甘国际机场]]于2005年9月建成使用，有航班飞往全国各大城市。
