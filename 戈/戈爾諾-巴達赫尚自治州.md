'''戈爾諾-巴達赫尚自治州'''（{{lang-tg|'''Вилояти Мухтори Кӯҳистони Бадахшон'''/Viloyati Mukhtori Kuhistoni Badakhshon/'''ولایت مختار کوهستان بدخشان'''}}；{{lang-ru|'''Горно-Бадахшанская''' (автономная область)}} / Gorno-Badakhshanskaya (a.o.)），又译作'''山地-巴達赫尚自治州'''，是[[塔吉克斯坦|塔吉克斯坦]]的一個多山地的省。面積大約佔塔吉克斯坦的40%。它的名稱源于[[俄語|俄語]]，意思是「多山地的[[巴达克山|巴達赫尚]]」{{efn|該州的俄語名稱是''{{lang|tg|Горный Бадахшан}}'' / ''Gornyy Badakhshan''直譯而來。這是現代俄語在地方命名上的特點。}}。[[中華民國政府|中華民國政府]]曾宣稱該地區大部分應該屬於[[中華民國疆域|中華民國版圖]]，屬[[新疆省|新疆省]]的一部分（参见[[原中華民國法理疆域行政區劃|原中華民國法理疆域行政區劃]]）。

== 歷史 ==
[[image:_Zunghar_Khanate_at_1750.jpg|thumb]]（粉紅色）南部與[[巴達赫尚|巴達赫尚]]（綠色）的邊界處。]]1895年以前，今天的戈爾諾-巴達赫尚自治州加上[[阿富汗|阿富汗]]東北部[[巴達赫尚省|巴達赫尚省]]所屬區域由若干個小型半自治政權構成，它們包括[[達爾瓦斯|達爾瓦斯]]、Shughnun-Rushan、[[瓦罕|瓦罕]]等。當時這塊區域同時被[[清朝|清朝]]、[[沙皇俄國|沙皇俄國]]和[[阿富汗酋長國|阿富汗酋長國]]聲稱為己方領土。清政府聲稱擁有整個[[帕米爾|帕米爾]]地區，但其軍隊僅僅控制了[[塔什庫爾干|塔什庫爾干]]城東面的關口。

''1890年以前，此地居民臣服於[[清朝|清朝]]，此地劃歸[[新疆省_(清朝)|新疆省]]管轄''<ref>[http://www.hebtu.edu.cn/col90/col157/col159/article.htm1?id=2036 中国近代史课程教案] {{webarchive|url=https://web.archive.org/web/20070806013127/http://www.hebtu.edu.cn/col90/col157/col159/article.htm1?id=2036 |date=2007-08-06 }}</ref>。19世紀後期[[大博弈|英俄在亞洲爭奪勢力範圍]]，1890年沙俄進駐並控制了包括此地在内的[[帕米爾高原|帕米爾高原]]北部地區，不久與清簽訂了[[中俄续勘喀什噶尔界约|中俄续勘喀什噶尔界约]]，與英國簽訂《關於帕米爾地區勢力範圍的協議》。從滿清到民國，中國政府始終堅持對此地擁有主權，不承認沙俄和[[蘇聯|蘇聯]]的佔領合法，但是卻無力進駐此地實行有效統治。 1990年代，[[中華人民共和國政府|中華人民共和國政府]]和[[塔吉克斯坦政府|塔吉克斯坦政府]]簽署邊界條約，正式承認此地屬於塔吉克斯坦所有，作為交換，塔方割讓了1100平方公里的土地，2011年邊界碑立下。<ref>[http://news.sohu.com/20050616/n225972452.shtml 中吉边界的现代“丝绸之路”]</ref>

蘇聯時代州被划成'''戈爾諾—巴達赫尚自治區'''（{{lang|ru|Горно-Бадахшанская автономная область}}），自治區建于1925年。1929年[[塔吉克蘇維埃社會主義共和國|塔吉克蘇維埃社會主義共和國]]成立后，該區成為共和國的一部分。1950年代，戈爾諾—巴達赫尚自治區的大量土著居民[[帕米爾人|帕米爾人]]被迫遷往塔吉克西南部。1991年塔吉克斯坦獨立后，該區改名為“戈爾諾－巴達赫尚自治州”，當1992年塔吉克斯坦爆發[[塔吉克內戰|内戰]]時，自治州政府宣佈獨立。在内戰期間，許多帕米爾人被塔吉克人等其他族群殺害，該州也同时成为[[塔吉克斯坦伊斯蘭復興黨|塔国伊斯兰政治反對派]]的大本營。後來自治州政府放棄了獨立要求。

== 地理 ==
[[帕米爾高原|帕米爾高原]]的最高處，以及[[蘇聯|蘇聯]][[中亞細亞|中亞細亞]]五座海拔7,000公尺以上的山峰其中三座都位於本州：[[列宁峰|獨立峰]]（前稱列寧峰，海拔 7,134公尺，在[[吉爾吉斯|吉爾吉斯]]邊界上），[[伊斯梅尔·索莫尼峰|伊斯梅爾·薩馬尼峰]]（前稱共產主義峰，海拔 7,495公尺）和[[卡季尼夫斯基峰|卡季尼夫斯基峰]]（海拔 7,105公尺）。

== 人口 ==
主要人口為[[帕米尔|帕米尔人]]，少數為[[吉尔吉斯人|吉尔吉斯人]]和[[塔吉克人|塔吉克人]]。最大城市為[[霍羅格|霍羅格]]（{{lang|tg|Хорӯғ}}）人口 22,000；第二大城市為[[末合_(塔吉克)|末合]]（{{lang|tg|Мурғоб}}），人口約 4,000。戈州是多種[[帕米爾語|帕米爾語]]的故鄉，包括[[舒格南語|舒格南語]]、[[瓦罕語|瓦罕語]]、[[色勒库尔語|色勒库尔語]]等等。除此之外，俄語和[[塔吉克語|塔吉克語]]都在當地通用，而其中末合地區有不少操[[吉爾吉斯語|吉爾吉斯語]]的居民。當地主要宗教為[[伊斯蘭教|伊斯蘭教]][[伊斯玛仪派|伊斯玛仪派]]，對[[阿迦汗|阿迦汗]]的尊崇十分流行。

== 交通 ==
對外聯繫有四條路線，分別是霍羅格—[[奧什|奧什]]、霍羅格—[[杜尚別|杜尚別]]（前兩者路況較佳，屬[[帕米爾公路|帕米爾公路]]一部份）、霍羅格—[[塔什庫爾干|塔什庫爾干]]（中經[[庫里瑪口岸|庫里瑪口岸]]）和霍羅格—[[瓦罕走廊|瓦罕走廊]]—[[阿富汗|阿富汗]]邊界。
