'''旺角東站'''（{{lang-en|'''Mong Kok East Station'''}}），前稱'''旺角車站'''、'''油麻地站'''，位於[[香港|香港]][[九龍|九龍]][[油尖旺區|油尖旺區]][[旺角|旺角]]東部[[聯運街|聯運街]]旁<ref>{{cite web|url=http://www.eac.gov.hk/pdf/distco/maps/dc2011e.pdf|title=二零一一年區議會選舉油尖旺區選區分界地圖|publisher=[[選舉管理委員會|選舉管理委員會]]|accessdate=2011年6月22日|language=zh-hant}}</ref>的[[港鐵|港鐵]][[東鐵綫|東鐵綫]][[車站|車站]]，於1910年建成。

旺角東站與[[荃灣綫|荃灣綫]]及[[觀塘綫|觀塘綫]]的[[太子站|太子站]]及[[旺角站|旺角站]]並非正式的轉車站。乘客若在兩站間轉乘，無論使用[[八達通|八達通]]還是單程車票，均會當作兩段車程處理，並繳納再入閘收費。乘客可途經有蓋[[行人天橋|行人天橋]]或經[[亞皆老街|亞皆老街]]步行約十分鐘到旺角站，以及經[[太子道西|太子道西]]步行約十分鐘到太子站，然後乘車往荃灣綫及觀塘綫沿途地方。

==車站結構==
雖然車站屬於架空車站，但是車站大部分被[[混凝土|混凝土]]圍封，使月台裏面的人以為身處地底，不過月台南盡頭附近並無混凝土圍封，乘客可以看到站外景色。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = MTR MKK (39).JPG | caption1 = 1、2號月台的頭等核准機
}}
旺角東站設有3個月台，以1個[[島式月台|島式月台]]及1個[[側式月台|側式月台]]排列，月台並沒有[[月台幕門|月台幕門]]。1號及2號月台共用同一個島式月台，主要處理東鐵綫往羅湖／落馬洲方向的列車，平日早上有两班由[[新界|新界]]開出的列車用1號月台為南行方向的終點站，之後分别折返往[[大埔墟站|大埔墟]]及返回東鐵綫的何東樓車廠。非繁忙時間有部份列車從紅磡貨場開出，在此站1號月台掉頭往紅磡站接載乘客。相反，部份列車從紅磡站清客後開往此站1號月台掉頭前往紅磡貨場停泊。此外，平日下午繁忙時間有一班列車從此站1號月台起載往羅湖。3號月台為側式月台，主要處理東鐵綫往紅磡方向的列車之用。

旺角東站是早年興建的車站，因此月台較窄，可供乘客等候列車的空間較少。加上車站的彎度較高，為了遷就不同闊度的列車駛經月台，故此列車與月台之間的空隙較大（最闊25厘米），是東鐵綫其中三個設有「空隙黑點」提示的車站之一（另外兩個是[[九龍塘站_(香港)|九龍塘站]]及[[大學站_(香港)|大學站]]）。

===出入口===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Grand Century Place Public Access to Mong Kok East station 2018.jpg | caption1 = 往[[亞皆老街|亞皆老街]]的行人通道由新世紀廣場管理
}}
在興建新世紀廣場前，A出入口是前往亞皆老街的主要通道。但在1997年興建新世紀廣場時，建造了連接B出入口至亞皆老街的天橋，並拆除舊有的通道。工程完成後，A出入口的地位被B出入口所代替，繼而被長時間封閉。直至2012年11月，港鐵重用該通道（通往旺角政府合署後巷）；同年12月，重新編上A出入口，其出入口指示為「政府合署停車場」，但平日甚少乘客使用。

===旺角貨場===
[[File:MTR_Mong_Kok_Freight_Terminal.JPG|thumb]]
在旺角東站1號月台旁曾經有一作為貨物分流的貨場，名為'''旺角貨場'''（{{lang-en|'''Mong Kok Freight Terminal'''}}），入口設於聯運街。後來因鐵路貨運服務萎縮而曾經空置，九廣鐵路公司遂以簽訂使用協議形式出租貨場，例如曾租予[[嘉士伯|嘉士伯]]（香港）有限公司作為物流中心，亦曾引起質疑違反批地條款<ref>{{cite news|title=九鐵變相租兩貨場地疑違例|url=http://the-sun.on.cc/channels/news/20021129/20021129015357_0001.html|work=太陽報|date=2002年11月29日|accessdate=2012年1月23日|language=zh-hant}}</ref>。兩鐵合併後，港鐵在大閘前的「旺角貨場」名稱前加上「港鐵」二字，貨場亦名'''嘉士伯（香港）物流中心'''（{{lang-en|'''Carlsberg (Hong Kong) Logistic Center'''}}）。現時嘉士伯（香港）已遷出，貨場交回港鐵營運，港鐵有意用作替代停泊因沙中綫工程而改建[[紅磡貨場|紅磡貨場]]的東鐵綫工程車。

==利用狀況==
{{original research|章節}}
[[File:Mong_Kok_East_Station_2017_03_part1.jpg|thumb]]的車站]]
[[File:Mong_Kok_East_Station_Public_Transport_Interchange_201601.jpg|thumb]]
由於旺角東站上蓋設有大型綜合商業物業發展項目──[[新世紀廣場|新世紀廣場]]的購物商場及[[寫字樓|寫字樓]]，而附近又有[[花墟|花墟]]、[[旺角大球場|旺角大球場]]、各家中小學校，再加上本站可以透過[[旺角行人天橋系|有蓋行人天橋系統]]接駁至[[彌敦道|彌敦道]]、[[西洋菜南街|西洋菜南街]]一帶的旺角和太子的核心區域和民居，乃至[[旺角站|旺角站]]和[[太子站|太子站]]，因此該站有很高的乘客量支持。即使如此，由於本站不設轉乘功能，本站相對旺角站和太子站而言甚少出現擠逼情況。而在油麻地、大角咀、旺角和太子站附近工作的乘客，多會前往本站上車，避免[[九龍塘站|九龍塘站]]須輪候轉車的情況。

雖然名稱及位置上與鄰近之旺角站十分相近，但往返各兩站之車程票價有一定差距，亦出現短程比長程貴之情況，而在本站下車較經九龍塘站轉車前往旺角站便宜。

本站因處於半密封設計的環境，通風不足長期受到詬病。

==接駁交通==
旺角東站平台建有[[旺角東站公共運輸交匯處|旺角東站公共運輸交匯處]]，方便乘客轉乘專綫小巴或巴士。鄰近[[太子道西|太子道西]]、[[旺角道|旺角道]]及[[亞皆老街|亞皆老街]]亦設有行人天橋及通道前往公共運輸交匯處。

===與太子站及旺角站接駁===
旺角東站與[[荃灣綫|荃灣綫]]和[[觀塘綫|觀塘綫]]的[[太子站|太子站]]及[[旺角站|旺角站]]並非正式的轉車站，乘客若在兩站間轉乘，無論使用[[八達通|八達通]]或是單程車票，均當作兩段車程處理，並且需要繳納再入閘費用，而旺角站和旺角東站有天橋連接。

===命名、啟用及第一次更名===
旺角東站最初稱[[油麻地站_(九廣鐵路英段)|'''油麻地站''']]<ref>[http://i627.photobucket.com/albums/tt355/fongcw_photo/klnmap19242.jpg 這張1942年地圖可見YAUMATI STATION的位置，即今天旺角東站的位置]</ref><ref>{{cite book|title=A History Kowloon-Canton Railway (British Section)|chapter=Chapter Four: The Route|pages=第61頁|year=1990|language=en}}</ref>，與現時旺角東站的位置相同<ref>{{cite web|url=http://hkclweb.hkpl.gov.hk/hkclr2/object?svc=objrtv&src=CM&itemid=PIVCUQY2NWVBBR3M&pid=1&mime=image/jpeg|title=旺角·大角咀·何文田街道詳圖|work=香港年鑑|publisher=[[華僑日報|華僑日報]]|year=1956|language=zh-hant|deadurl=yes|archiveurl=https://web.archive.org/web/20050914061401/http://hkclweb.hkpl.gov.hk/hkclr2/object?svc=objrtv|archivedate=2005-09-14}}</ref><ref>{{cite web|url=http://www.mtr.com.hk/jplanner/images/maps/mkk.pdf|title=旺角東站街道圖|publisher=港鐵公司|language=zh-hant}}</ref>，於1910年隨九廣鐵路（英段）通車啟用。1969年，站名由[[油麻地站_(九廣鐵路英段)|油麻地站]]更改為'''旺角車站'''（{{lang-en|'''Mong Kok Station'''}}）<ref>{{cite book|title=The Railways And Tramways of Hong Kong|author=Wilfrid F. Simms|year=1998|language=en}}</ref>。

===重建===
1980年代，為配合九廣鐵路電氣化，旺角東站（前稱旺角車站）原址重建，成為現時的樣貌，臨時月台設於原址的南方。

===翻新及擴建===
為配合[[新世紀廣場|新世紀廣場]]落成，旺角東站曾於1997年進行過一次翻新及擴建工程，將整個大堂及月台翻新。

===第二次更名===
2007年12月2日隨着[[兩鐵合併|兩鐵合併]]，為免與位於彌敦道的[[旺角站|旺角站]]產生混淆，於是更名為'''旺角東站'''，惟車站代號仍為MKK。

===翻新月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Mong Kok East Station 2017 03 part7.jpg | caption1 = 翻新後的月台
| image2 = Mong Kok East Station 2017 08 part1.jpg | caption2 = 所有月台現已完成地板翻新工程 
}}
兩鐵合併後，港鐵公司翻新東鐵綫各月台，以遮掩車站殘舊的觀感。工程包括重新為各車站配主題色，並加入花的代表，為月台橫樑及柱子翻上月台配色的油漆，貼上印有車站主題花的圖案、車站附近/主題建築物/街道、車站名書法字的牆紙，加設/更新直立站牌，路線圖，指示牌，告示等等。此站配色為深綠色，主題花為蓮花，街景為女人街。

===翻新大堂===
港鐵公司為了迎接東鐵綫（前稱九廣東鐵）營運百周年，決定為各個東鐵綫車站進行翻新工程，並且以融入大自然的[[設計|設計]]及[[建築|建築]][[概念|概念]]，翻新各座車站。旺角東站的大堂翻新工程獲得優先進行，工程包括大堂地板及牆身翻新，[[天花板|天花板]]採用特色環形通花吊飾設計，並且使用更多[[石材|石材]]，改為木色綫條設計，又將名為「蝌蚪機群」的[[藝術品|藝術品]]懸掛在天花下，9條蝌蚪有如行駛中的東鐵綫[[列車|列車]]般。與此同時，車站引入更多自然光，降低乘客對車站原有建築物料──[[金屬|金屬]]的感受。二樓改以[[藍色|藍色]][[玻璃|玻璃]]造出天花波浪，讓乘客彷彿置身於[[海洋|海洋]]。此外，亦有興建更多[[商店|商店]]，部份[[售票機|售票機]]改為入牆式、重置客務中心（包括降低櫃台，方便[[身心障礙|傷殘人士]]）及加建[[洗手間|洗手間]]，以騰出更多的大堂空間<ref>[http://hk.news.yahoo.com/旺角東站美化完工-074200336.html 旺角東站美化完工] 《星島日報》 2012年1月26日</ref>；B出入口亦有改建，整個工程於2012年1月26日竣工<ref>[http://hk.news.yahoo.com/旺角東站翻新-蝌蚪-高懸-220916949.html 旺角東站翻新「蝌蚪」高懸] 《星島日報》 2012年1月27日</ref>。
