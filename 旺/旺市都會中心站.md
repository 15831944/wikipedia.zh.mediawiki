'''旺市都會中心站'''（{{lang-en|Vaughan Metropolitan Centre Station}}）是一座[[多倫多地鐵|多倫多地鐵]][[央街－大學－士巴丹拿線|央街－大學－士巴丹拿線]]車站，坐落[[加拿大|加拿大]][[安大略省|安大略省]][[約克區|約克區]][[旺市|旺市]][[安大略7號省道|7號公路]]和珍街（{{lang|en|Jane Street}}）的交界處以西，於2017年12月17日聯同士巴丹拿地鐵延線項目其他車站一起開幕<ref name="open">{{cite news | last = Patton| first = Jessica| title = New Toronto-York Spadina Subway Extension officially launches| publisher = Global News| date = 2017-12-17| url = https://globalnews.ca/news/3921542/new-toronto-york-spadina-subway-extension-officially-launches/| accessdate =  2017-12-18}}</ref>，取代[[唐士維站|唐士維站]]成為該線的西北端終點站，以及整個多倫多地鐵系統中地理位置最北的車站。另外，此站啓用後亦成為多倫多市管轄範圍以外，以及約克區和旺市範圍以内兩個多倫多地鐵車站之一；另外一個則是[[407號公路站|407號公路站]]。

旺市市政府正計劃將車站一帶打造成該市的[[商業中心區|商業中心區]]，並將之取名為旺市都會中心，因此向TTC提出將車站命名為旺市都會中心站。然而，TTC内部認為該名稱過為冗長，因此於2010年建議將名稱縮短為旺市中心站（{{lang|en|Vaughan Centre Station}}）；此舉亦可避免車站名稱為附近的私人發展商項目作宣傳之用，並與其他車站名稱（如[[北約克中心站|北約克中心站]]和[[士嘉堡中心站|士嘉堡中心站]]）看齊<ref>{{cite web |url=http://www3.ttc.ca/About_the_TTC/Commission_reports_and_information/Commission_meetings/2010/Sept_30_2010/Supplementary_Reports/Toronto_York_Spadina.pdf |title=Toronto-York Spadina Subway Extension, Station Names |date=2010-09-30 |publisher=Toronto Transit Commission |accessdate=2013-02-02}}</ref>。TTC後來接納旺市市政府的建議，於2012年2月29日宣佈將此站名為旺市都會中心站<ref>{{cite web |url=https://www.vaughan.ca/news/Pages/City-Welcomes-Approval-of-Vaughan-Metropolitan-Centre-Subway-Station-Name-.aspx |title=City Welcomes Approval of Vaughan Metropolitan Centre Subway Station Name |publisher=旺市市政府 |accessdate=2013-02-02}}</ref>。
