'''旺角站'''（{{lang-en|Mong Kok Station}}）是[[港鐵|港鐵]]<ref name="merger_mtr_name">在[[兩鐵合併|兩鐵合併]]前，港鐵及[[香港鐵路有限公司|香港鐵路有限公司]]的前稱分別為'''地鐵'''（地下鐵路）及'''地鐵公司'''。</ref>車站，亦是主要[[跨月台轉車站|跨月台轉車站]]之一，位於[[油尖旺區|油尖旺區]][[旺角|旺角]][[彌敦道|彌敦道]]與[[亞皆老街|亞皆老街]]交界的地底，於1979年12月31日啟用，由金門/Kier/Lilley Construction聯營公司負責興建。

儘管位置相近並有[[旺角行人天橋系統|行人天橋網絡]]接駁，但旺角站與[[東鐵綫|東鐵綫]]<ref name="merger_eal_name">在[[兩鐵合併|兩鐵合併]]前，東鐵綫的前稱為'''東鐵'''或'''九廣東鐵'''，並由[[九廣鐵路公司|九廣鐵路公司]]營運。</ref>的[[旺角東站|旺角東站]]<ref name="merger_mkk_name">在[[兩鐵合併|兩鐵合併]]前，旺角東站的前稱為'''旺角站'''或'''旺角車站'''，並由[[九廣鐵路公司|九廣鐵路公司]]營運。</ref>並非為正式的轉車站。乘客若在兩站間轉乘，無論使用[[八達通|八達通]]還是單程車票，均需當作兩段車程處理，並繳納再入閘收費。

==結構==
旺角站總共有4層，最底2層為月台，月台之上為大堂。

===大堂=== 
2012年，港鐵在大堂中央位置裝設由中國藝術家[[廖一百|廖一百]]所創作的不鏽鋼及銅裝置雕塑作品「上善若水」。

====商店及自助服務====
旺角站內提供不同類型的商店供乘客購物，例如有[[便利店|便利店]]、[[麵包糕餅店|麵包糕餅店]]及[[速遞|速遞]]等服務<ref name="shop"/>。本站所有商店及自助服務設施皆設在非付費區（閘外）。

此外，旺角站也有不少自助服務設施供乘客使用，包括[[自動櫃員機|自動櫃員機]]、[[自動販賣機|自動售賣機]]、自動照相機等<ref name="shop"/>。車站內亦設有[[香港郵政|香港郵政]][[郵箱|郵箱]]方便乘客在車站內投寄郵件<ref>{{cite web|url=http://www.mtr.com.hk/chi/getting_around/post.html|title=鐵路四處遊＞郵箱服務|publisher=港鐵公司|work=港鐵網頁|accessdate=2009年3月2日|language=zh-hant}}</ref>、「iCentre」免費上網服務設施供乘客瀏覽[[萬維網|萬維網]]的內容<ref>{{cite web|url=http://www.mtr.com.hk/chi/getting_around/icentre.html|title=鐵路四處遊＞iCentre互聯網中心|publisher=港鐵公司|work=港鐵網頁|accessdate=2009年3月2日|language=zh-hant}}</ref>，以及「[[會員服務站|會員服務站]]」機供乘客登記為港鐵友禮會會員、查閱獎分及換領獎賞<ref>{{cite web|url=http://www.mtr.com.hk/ch/customer/services/csp.html|title=鐵路四處遊＞「會員服務站」|publisher=港鐵公司|work=港鐵網頁|accessdate=2016年9月26日|language=zh-hant}}</ref>。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Mong Kok Station 2017 03 part4.jpg |caption1 = 由月台至大堂的扶手樓梯通道
}}
旺角站是荃灣綫與觀塘綫的跨月台轉車站之一，乘客可以走到對面月台直接轉乘另一列列車前往目的地。車站共有4個月台，以2個[[島式月台|島式月台]]方式上下排列，所有月台均設有[[月台幕門|月台幕門]]，皆位于[[彌敦道|彌敦道]]地底。

另外，車站北面（L2層）和南面（L3層）均設有[[渡綫|交叉渡綫]]，可供列車調度又或讓列車來回觀塘綫和荃灣綫之間，當列車有車務調動時，列車可以在該站進行同方向的調度。由該站荃灣綫的1號月台或2號月台駛入該[[渡綫|渡綫]]，調度至觀塘綫，或由該站觀塘綫的3號月台或4號月台駛入該渡綫，調度至荃灣綫。因此，不載客列車間中使用該渡綫進行同方向的車務調動及調度。由於是同方向的調度，當此情況發生時，月台的列車將會稍為延誤。於荃灣線未啟用時，修正早期系統也是使用該渡綫，來往[[中環站|中環站]]及[[觀塘站|觀塘站]]。

旺角站共有15個出入口，與[[尖東站|尖東站]]並列為港鐵系統中設有最多出入口的車站（若只計算車站本身的出入口，則此站較尖東站擁有最多直接連接車站大堂的出入口）。通往E出入口的閣樓設有數間商店，C、D出入口則設有行人隧道橫過彌敦道地底<ref name="sta_info">{{cite book|author=港鐵公司|title=《旺角站車站指南》|publisher=港鐵公司|date=2008年9月}}</ref>，在通往B1和B2出入口的閣樓中間，有一個原通往工業貿易署大樓（旺角中心二期）地庫的出入口，自1989年起一直封閉<ref>[http://news.takungpao.com.hk/paper/q/2015/1027/3225200.html 工貿署啟德新大樓入夥 大公報 2015-10-27]</ref>，直到2018年6月16日重新對外開放，成為B4出口。此外，旺角站離東鐵綫<ref name="merger_eal_name" />旺角東站<ref name="merger_mkk_name" />約500米路程，自2003年[[旺角行人天橋系統|旺角行人天橋系統]]啟用以後，旺角東站被納入服務範圍，經B3出入口直接經行人天橋前往旺角東站，需時約15分鐘。各個出入口均設有指示牌顯示出入口周邊的建築物及設施列表。

旺角站位於[[香港|香港]][[九龍|九龍]][[油尖旺區|油尖旺區]]人流最暢旺的[[旺角市中心|旺角市中心]]，是香港其中一個大型及重要的商業、購物、娛樂和消閒地區。四周除設有[[住宅|住宅]]、[[辦公室|辦公室]]、[[診所|診所]]、[[銀行|銀行]]、[[酒店|酒店]]、[[補習社|補習社]]外，更有不少[[酒樓|酒樓]]、[[餐廳|餐廳]]、[[茶餐廳|茶餐廳]]、[[戲院|戲院]]、[[香港賽馬會|投注站]]、[[夜總會|夜總會]]、[[麻雀館|麻雀館]]、[[遊戲機中心|遊戲機中心]]、[[樓上書店|樓上書店]]、[[女僕咖啡店|女僕咖啡店]]等消遣及娛樂場所。亦鄰近不同類型、規模的購物點（大型商場如[[朗豪坊|朗豪坊]]；專門售賣各種潮流產品、時裝的[[信和中心|信和中心]]、[[旺角新之城|旺角新之城]]；著名售賣電腦產品的[[旺角電腦中心|旺角電腦中心]]；著名售賣手機的[[先達廣場|先達廣場]]；著名售賣攝影、音響器材的[[星際城市|星際城市]]；著名售賣玩具及[[比例模型|比例模型]]玩具的[[皆旺商場|皆旺商場]]；著名售賣球鞋的[[波鞋街|波鞋街]]；各種攤檔雲集的[[女人街|女人街]]；以及[[西洋菜南街|西洋菜南街]]），吸引香港居民及各地遊客前往。因此，本站與地面一樣，終日人流如卿，出入閘人流甚多。

此外，車站是港鐵[[觀塘綫|觀塘綫]]及[[荃灣綫|荃灣綫]]的重要轉車站，基本上由本站前往大部份港鐵重要車站皆無需或只需轉一次車，因此，本站的轉車客量與出入閘人流同樣甚多。

1989年8月[[觀塘綫|觀塘綫]][[東區海底隧道|東區海底隧道]]通車前，該站的繁忙時間轉乘人流極高，乘客往往要輪候數班車才能登車。1989年8月[[觀塘綫|觀塘綫]][[東區海底隧道|東區海底隧道]]通車後至2002年8月[[將軍澳綫|將軍澳綫]]通車前，部份乘客利用[[觀塘綫|觀塘綫]]過海，令該站轉車人流稍為回落。2002年8月[[將軍澳綫|將軍澳綫]]通車，[[觀塘綫|觀塘綫]]不能再直接過海，此不便使部份乘客重投此站轉乘[[荃灣綫|荃灣綫]]過海，該站轉車人流回升。2004年10月[[尖東站|尖東站]]通車，與[[尖沙咀站|尖沙咀站]]連接，促使部份[[東鐵綫|東鐵綫]]乘客改用這站轉乘[[荃灣綫|荃灣綫]]過海，該站轉車人流稍為回落。唯2009年8月[[九龍南綫|九龍南綫]]通車，[[東鐵綫|東鐵綫]]不能直達[[尖東|尖東]]，這不便促使更多[[東鐵綫|東鐵綫]]乘客重投[[九龍塘站|九龍塘站]]及本站轉乘[[荃灣綫|荃灣綫]]過海，本站轉車人流再度回升。而現時早上繁忙時間[[荃灣綫|荃灣綫]]往[[中環站|中環]]方向可能要輪候2至3班車才能登車，車頭位置極難登車，唯月台車尾明顯較易登車。換言之，乘客如果往返[[港島綫|港島綫]]、[[南港島綫|南港島綫]]及[[觀塘綫|觀塘綫]]，需要在本站及[[金鐘站|金鐘站]]或[[中環站|中環站]]轉車。[[觀塘綫延線|觀塘綫延線]]通車後下班繁忙時間[[觀塘綫|觀塘綫]]往[[調景嶺|調景嶺]]方向可能要輪候第2班車才能登車，車頭位置明顯較易登車，車尾位置因有大量入閘乘客而極難登車，尤以[[黃埔站|黃埔站]]開出列車最為明顯，因為該站每4分鐘左右才開出一班列車。

總括而言，由於本站位處[[旺角|旺角]]心臟地帶，加上屬重要轉車站，即使鄰近有四個重鐵車站（同樣屬荃灣綫及觀塘綫的轉車站的[[太子站|太子站]]和[[油麻地站|油麻地站]]、東涌綫的[[奧運站|奧運站]]、東鐵綫的[[旺角東站|旺角東站]]），但由於所屬路綫走向不盡相同，以及地理上較為偏離旺角市中心或近大角咀之緣故，因而絲毫不影響本站每日極高的人流，大堂及月台由早到晚依然十分擠逼，亦因此本站是[[港鐵|港鐵]]位於[[九龍|九龍]][[油尖旺區|油尖旺區]]的轉車站中人流最多及最繁忙的轉車站，更曾是整個前地鐵系統<ref name="merger_mtr_name" />第四繁忙的車站<ref>{{cite web |title = 1988年香港地鐵（主要旺角站）|url = http://www.youtube.com/watch?v=gYOOHQ5sZ_U&feature=related }}</ref><ref>{{cite news |title =  銅鑼灣站極擠迫 擬建地下街紓緩 |work = [[明報|明報]] |date = 2007年6月28日 }}</ref>。不過，兩層月台扶手電梯不足，對乘客造成不便。

當旺角站有列車故障或有列車臨時需要在該站作為終點站時，車站將會十分擠塞。

==歷史==
===修正早期系統===
1979年至1982年，[[香港地鐵|地鐵]]<ref name="merger_mtr_name" />荃灣綫未通車時，[[觀塘綫|修正早期系統]]列車曾經使用過旺角渡綫前往[[油麻地站|油麻地]]至[[中環站|中環]]各站。荃灣綫通車之前，修正早期系統只使用旺角站1號和4號月台，2號和3號月台並未開放，其中現4號月台在當時為2號月台<ref name="P4">[http://www.hkitalk.net/HKiTalk2/viewthread.php?tid=240306 [回到過去]80年代地鐵旺角站] hkitalk.net</ref>。

===命名===
旺角站啟用初期的英文名稱為'''Argyle'''，即所在的[[亞皆老街|亞皆老街]]之英文名稱，1985年5月31日改為旺角的英文'''Mong Kok'''。另外，九廣鐵路——英段（後的九廣東鐵，今[[東鐵綫|東鐵綫]]）<ref name="merger_eal_name" />早於1969年已設有同名車站，在[[兩鐵合併|兩鐵合併]]之後為了區別，後者名稱遂改為[[旺角東站|旺角東站]]<ref name="merger_mkk_name" />。

===E出口增設新行人通道===
由於旺角站的使用量長年高居不下，至2001年通往[[銀行中心|銀行中心]]（E出口）的唯一通道已極度飽和，地鐵公司不得不決定耗資一億五千多萬元進行一系列擴建工程，包括在E出口增設一條複製通道，設兩條往出口方向的單向扶手電梯，每小時人流量可達14,000多人次，可疏導人潮。工程也包括在快富街出口（A1出口）增設一部[[升降機|升降機]]及失明人士引導徑。<ref>[http://hk.apple.nextmedia.com/news/art/20020211/2430226 耗一億五千萬　解決人流樽頸 地鐵旺角九龍塘站添通道] [[蘋果日報_(香港)|蘋果日報 (香港)]] 2002年2月11日</ref>

===加建新出口===
由新鴻基地產耗資興建的[[旺角行人天橋系統|旺角道行人天橋系統]]，於2003年1月及8月分兩階段投入服務，根據施工條款，B3出入口屬於第二期配套，設有一條扶手電梯連接固有地鐵站通道，直通連接[[西洋菜南街|西洋菜南街]]、[[旺角道|旺角道]]、[[洗衣街|洗衣街]]及[[新世紀廣場|新世紀廣場]]的有蓋行人天橋網絡，與非正式轉車站[[旺角東站|旺角東站]]一脈相連，亦將[[新世紀廣場|新世紀廣場]]、[[旺角政府合署|旺角政府合署]]、[[帝京酒店|帝京酒店]]納入服務範圍。<ref>地鐵旺角站新出入口啟用 《星島日報》 2003年10月5日</ref>

===改建C3出入口===
朗豪坊施工期間，2003年4月1日，原設於砵蘭街的舊C3出口動工清拆並回填為行人路，地鐵另闢一條新的行人隧道直通商場地庫第二層，運輸署亦於亞皆老街與砵蘭街一帶進行交通改道，為期兩年。當新出口於2005年3月初啟用後，旺角站的出入口合共增至14個，冠絕全九龍。<ref>旺角地鐵站C3 出口清拆 《東方日報》 2003年3月22日</ref><ref>[http://hk.apple.nextmedia.com/news/art/20050214/4660663 旺角站增至14個　下月初通朗豪坊 地鐵7站擴建出口疏人潮] 《蘋果日報》 2005年2月14日</ref>

===加建洗手間===
港鐵於2012年3月19日宣佈，為轉車站加設[[洗手間|洗手間]]，旺角站為其中之一，洗手間於2015年1月24日落成啟用，設於大堂近A及B出入口的付費區內<ref>[http://hk.apple.nextmedia.com/news/art/20150125/19015698 港鐵旺角、太子站終有公廁] 《蘋果日報》 2015年1月25日</ref><ref>[http://orientaldaily.on.cc/cnt/news/20150125/00176_040.html 港鐵旺角站新增洗手間] 《東方日報》 2015年1月25日</ref>。

===合併付費區域及加建扶手電梯===
港鐵指為了令大堂人流更順暢，由2015年8月30日起，車站大堂介乎C及D出入口之間的閘外區域會併入已付費區域內，令站內兩端原本獨立的已付費區域合而為一，並增設兩部入閘機。往來C及D出口需使用大堂上方的走廊。

另針對下層月台只有兩條扶手電梯前往大堂，繁忙時間會出現擠擁。在新扶手電梯投入服務前，下班繁忙時間更常出現月台第二組列車到站後，扶手電梯人龍未清的擠擁情況，港鐵於車站加設3條扶手電梯，令來往大堂和月台層的人流更暢順<ref>{{cite web|author1=港鐵公司|title=提升港鐵車站設施的最新進展|url=http://www.legco.gov.hk/yr14-15/chinese/panels/tp/tp_rdp/papers/tp_rdp20150703cb4-1228-3-c.pdf|accessdate=2015-07-28}}</ref>，其中兩條新建扶手電梯已於2016年第四季啟用。

1989年，港府購入[[旺角中心|旺角中心]]二期作為[[前工業貿易署大樓|工業貿易署大樓]]，大樓連接旺角站的地庫出口及連接旺角中心第一期天橋亦同時封閉，有傳是因為政府資源重組，將通往地鐵站的通道改為辦事處而封閉。直至2016年，[[領展|領展]]計劃在大樓翻新後會重開此通道方便市民經此通道來往旺角中心和[[彌敦道700號|彌敦道700號]]，並會加設B4出入口，原預計2018年第一季完成。<ref>{{cite web|author1=信報|title=領展四億翻新前工貿大樓|url=http://www1.hkej.com/dailynews/article/id/1462075/%E9%A0%98%E5%B1%95%E5%9B%9B%E5%84%84%E7%BF%BB%E6%96%B0%E5%89%8D%E5%B7%A5%E8%B2%BF%E5%A4%A7%E6%A8%93|accessdate=2016-12-24}}</ref>最後延遲到同年6月16日對外開放，成為第15個出口。
