'''段应碧'''（{{bd|1940年|||}}）[[四川省|四川省]][[万县|万县]]五桥乡人。[[中华人民共和国|中华人民共和国]]官员<ref name=duan>{{Cite web |url = http://people.chinareform.org.cn/d/duanyingbi/resume/ |title = 段应碧 |author =  |date =  |publisher = 中国改革论坛网 |language =  |accessdate = 2017-05-15 |quote =  }}</ref>。

==生平==
1963年，自[[西南财经大学|四川财经学院]]农业经济系毕业，被分配至[[中国农业科学院|中国农业科学院]]农村经济研究所工作。1969年8月，调[[中华人民共和国农业部|中华人民共和国农业部]]业务组工作。1970年4月，调[[中华人民共和国农林部|中华人民共和国农林部]]政策研究室工作。1979年4月，调[[中华人民共和国国家农业委员会|中华人民共和国国家农业委员会]]政策研究室工作。1982年4月，调[[中共中央书记处|中共中央书记处]]农村政策研究室工作，负责农村金融、农产品流通、农业投资等政策研究。1990年10月，调任中华人民共和国农业部农村研究中心主任、党委书记<ref name=duan/>。

1993年4月，调任[[中央财经领导小组办公室|中央财经领导小组办公室]]农村组组长。1994年8月，任中央财经领导小组办公室副主任兼农村组组长，主要承担[[中央农村工作领导小组办公室|中央农村工作领导小组办公室]]的日常工作<ref name=duan/>。

2003年4月，任[[第十届全国政协委员|第十届全国政协委员]]，全国政协经济委员会副主任。2003年5月，任[[国务院西部地区开发领导小组|国务院西部地区开发领导小组]]办公室副主任<ref name=duan/>。

2006年起，任中国扶贫基金会会长。2003年至2010年，他还任中国农业经济学会会长<ref name=duan/><ref>{{Cite web|url =http://www.caas.net.cn/ysxw/yw/3717.shtml|title =中国农业经济学会召开常务理事会|author =|date =2010-03-01|publisher =中国农业科学院|language =|accessdate =|quote =}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>中国农业经济学会第七届代表大会暨“论城乡统筹发展”研讨会在广东成功举办，南方农村2003年06期</ref>。
