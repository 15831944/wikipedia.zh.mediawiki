'''段祺瑞'''（{{bd|1865年|3月6日|1936年|11月2日|catIdx=D段}}），原名'''启瑞'''，[[表字|字]]'''芝泉'''，[[中國|中國]][[安徽|安徽]][[合肥|合肥]]（今属[[肥西县|肥西县]]）人，為[[中華民國|中華民國]][[政治家|政治家]]，[[皖系军阀|皖系军阀]]首领，曾三次出任[[中华民国国务总理|国务总理]]，1916年至1920年為[[北洋政府|北洋政府]]的實際掌權者和領導人。1924年至1926年為[[中華民國臨時執政|中華民國臨時執政]]。

== 生平 ==
段祺瑞祖父及叔父皆為[[淮軍|淮軍]][[將領|將領]]，且與[[李鴻章|李鴻章]]同鄉。基於家學淵源與同鄉背景，段祺瑞在1885年考入由李鴻章創設的[[天津武備學堂|天津武備學堂]]，習[[炮兵|炮兵]]科。1889年畢業時，天津武備學堂擇選5位畢業生派赴[[德意志帝國|德意志帝國]]學習軍事，段祺瑞獲選為其中一員；段祺瑞在德國柏林陸軍學院留學期間繼續深造砲兵相關知識，隔年（1890年）回國。回國後，段祺瑞先短暫任職北洋軍械局委員，1891年改任威海隨營武備學堂教習兼炮兵統帶，曾參與[[威海衛之戰|威海衛之戰]]。

1894年[[甲午戰爭|甲午戰爭]]，[[北洋艦隊|北洋艦隊]]全滅後，1895年12月段祺瑞進入[[袁世凱|袁世凱]][[幕府|幕府]]，於[[新军|小站練兵]]，擔任砲兵隊統帶兼隨營學堂監督，成為袁的親信，其宦途也跟隨著袁世凱扶搖而上。

1899年，袁世凱編練的新軍開赴[[山東省|山東省]]鎮壓[[義和團|義和團]]；1901年，段祺瑞升任武衛右軍各學堂總辦，因袁世凱升官擔任[[直隸總督|直隸總督]]兼[[北洋大臣|北洋大臣]]，武衛右軍更名常備軍移駐保定；1902年6月，段祺瑞升任北洋軍政司參謀處總辦，負責北洋新軍所有編裝與訓練任務。

1903年10月，清朝政府決定設立練兵處，統轄新軍訓練，同年12月袁世凱兼任練兵處會辦大臣，因袁世凱推薦，段祺瑞升任練兵處軍令司正使、加[[副都統|副都統]]銜；此時段祺瑞与[[冯国璋|冯国璋]]、[[王士珍|王士珍]]並称为「北洋三杰」（亦被稱為[[龍|龍]]、[[虎|虎]]、[[狗|狗]]；王為龍，段為虎，馮為狗）。1904年，段祺瑞兼任常備軍第三鎮翼長署理；1905年2月正式升任新军[[陆军第四镇|陆军第四镇]]统制，驻军[[天津马厂|天津马厂]]，同年進行的河間秋操演習段祺瑞擔任北軍總統。1906年，段祺瑞改任[[陸軍第三鎮|陸軍第三鎮]]统制兼[[北洋武備學堂|北洋武備學堂]]督理，1906年3月被任命為[[福建省|福建省]]汀洲鎮[[總兵|總兵]]，段祺瑞雖南下赴任，但保留[[保定陸軍軍官學校|陸軍軍官學堂]]總辦職務；段祺瑞在北洋軍系影響力是基於他長期主掌軍隊訓練及養成，新軍逐漸編練成形的同時，段祺瑞也逐漸在軍隊中植入屬於他專屬的倫理性影響力。

1909年1月2日，袁世凯一度失勢，遭清政府下令“着即开缺，回籍养疴”，袁臨別前將北京私宅赠与段祺瑞。段祺瑞則在同年12月回任[[陆军第六镇|陆军第六镇]]统制。

1910年5月25日，清廷以段祺瑞督办北洋陆军学务有功，赏头品顶戴。12月18日加侍郎衔，外放任[[江北提督|江北提督]]，驻江苏淮安[[清江浦|清江浦]]，负责本地治安。

1911年[[武昌起义|武昌起义]]爆发后，清朝政府重新啟用袁世凱，袁世凱將段祺瑞召回北京，10月25日段升任清军第二军[[軍統|軍統]]；第二軍隨後開拔南下與第一軍（軍統馮國璋）一同清剿革命黨，清廷增封段祺瑞為[[湖廣總督|湖廣總督]]。由於袁世凱與革命黨間持續進行談判，南下清軍並未盡全力掃除武漢當地之敵；待11月13日袁世凱擔任[[內閣總理大臣|內閣總理大臣]]取得職權後，段祺瑞除湖廣總督之位外會辦撫剿事宜；由於[[馮國璋|馮國璋]]攻克武漢三鎮進度過順，11月28日袁世凱任命段祺瑞調任第一軍[[軍統|軍統]]，第二军軍統換由馮國璋擔任。12月5日，段祺瑞下令第一軍退出漢陽，開始進行[[南北議和|南北議和]]，革命軍[[黃興|黃興]]、[[孫文|孫文]]等同意，倘能促使[[宣統退位|宣統退位]]，可由[[袁世凱|袁世凱]]擔任[[中華民國大總統|中華民國大總統]]。

1912年1月26日，在袁世凱授意下，段祺瑞等[[北洋軍|北洋軍]]五十位[[將領|將領]]聯名發布[[徐樹錚|徐樹錚]]起草的《[[北洋五十將乞共和電|北洋五十將乞共和電]]》<ref name="徐">{{cite journal|1=|title=1925年12月30日，北洋军阀皖系将领徐树铮于廊坊车站遭冯玉祥仇杀|url=http://k.sina.com.cn/article_3579599314_d55c59d20010025u8.html}}</ref>，向[[隆裕太后|隆裕太后]]逼宮，不久，段祺瑞又發表《[[乞共和第二電|乞共和第二電]]》<ref name="徐" />，直接挑明「謹率全軍將士入京，與王公剖陳利害」，直接以[[武力|武力]][[恐嚇|恐嚇]]隆裕太后，隆裕于2月12日颁降懿旨，接受優待條件，[[溥儀|溥儀]]退位，中華民國正式成立。

[[中華民國|中華民國]]成立後，1912年3月10日袁世凱出任中華民國臨時大總統，段祺瑞同時出任陸軍總長。

1913年7月，段祺瑞代理[[中華民國國務總理|國務總理]]，組成[[段祺瑞臨時內閣|段祺瑞臨時內閣]]，調兵鎮壓[[二次革命|二次革命]]；12月陸軍總長一職由[[周自齊|周自齊]]接任。此後又署理[[湖北|湖北]][[都督|都督]]兼領[[河南|河南]]都督，鎮壓[[白朗_(民國)|白朗]]起義。

1914年2月，袁世凱段祺瑞召回北京述職，其湖北都督由[[段芝貴|段芝貴]]、河南都督在4月由[[田文烈|田文烈]]接任；1914年5月，袁世凱增設海陸軍大元帥統率辦事處，試圖收回由陸軍部主控的軍權，段祺瑞擔任陸軍總長；段祺瑞因不滿這個調任，在陸軍總長一職任內主要業務都由親信[[徐樹錚|徐樹錚]]定奪；1914年，[[日本軍隊|日本軍隊]]占领青岛，段主张对[[大日本帝國|日本]]决一死战。

1914年後，袁世凱與段祺瑞的不和日趨浮上檯面；1915年5月“[[二十一条|二十一条]]”签订后，段祺瑞告病辭職，赴西山养病；其陸軍總長一職由[[王士珍|王士珍]]接任。袁世凯在1915年底推行[[洪憲帝制|洪憲帝制]]時，段表示不拥护帝制。

1916年3月袁世凱被迫取消帝制，恢復[[共和制|共和制]]，邀请段出山任國務總理，最終段祺瑞回任[[參謀總長|參謀總長]]兼陸軍總長。6月袁世凱病逝，[[黎元洪|黎元洪]]接任大總統，段祺瑞擔任國務總理。之後段祺瑞成為北洋政府派系內的[[皖系|皖系]]領袖。

1917年，與大總統[[黎元洪|黎元洪]]之間因參與[[第一次世界大戰|第一次世界大戰]]問題引發[[府院之爭|府院之爭]]，引發由[[张勳|张勳]]领导的[[溥儀|溥儀]][[張勳復辟|復辟]]。復辟歷時十二日後即被段于[[天津馬廠|天津馬廠]]誓師镇压。在繼任的[[馮國璋|馮國璋]]總統任期内，就解决南北分裂，[[统一中国|统一中国]]的方式问题，与冯发生第二次[[府院之争|府院之争]]，1917年8月14日，宣佈对[[德意志帝國|德國]]及[[奧匈帝國|奧匈帝國]]宣戰，正式加入一战，並派華工到歐洲，及換取列強对中國如大國的待遇。于1917年11月22日辞职。

1918年3月22日复职。同年10月10日，馮國璋大总统的任期屆满，[[安福国会|安福国会]]选举[[徐世昌|徐世昌]]为[[中华民国大总统|大总统]]。段與馮約定共同[[辭職|下野]]{{NoteTag|段祺瑞建议推举[[徐世昌|徐世昌]]为下届总统，他自己则表示不做副总统，倘若冯国璋愿意退为[[中華民國副總統|副总统]]，他也可以同意，否则他愿意和冯国璋同时下野。段出此途，表面上说，是为了团结北洋派，因为徐世昌是北洋的元老，且和直系有很深渊源，徐出山，直系不会反对，段和冯同时下台，直系也可心平。<ref>丁中江《北洋军阀史话》</ref>}}。段祺瑞通过[[安福系|安福系]]继续在幕後操縱政權。1920年7月在[[直皖战争|直皖战争]]中失败，退隐天津。

1924年10月23日，[[冯玉祥|冯玉祥]]发动[[北京政变|北京政变]]，推翻大总统[[曹錕|曹錕]]，先邀请[[孙中山|孙中山]]北上，后与奉系妥协，请段祺瑞出山，任[[中華民國臨時政府_(段祺瑞)|中華民國臨時政府]]的[[臨時執政|臨時執政]]（國家元首）。

1925年4月24日，段正式下取消法統令，废除斷續運行12年的[[中华民国第一届国会|中华民国第一届国会]]，由[[中華民國|中華民國]][[臨時參政院|臨時參政院]]替代之。

1926年3月18日，发生北洋政府镇压北京学生运动的[[三·一八惨案|三·一八惨案]]，同年4月20日被[[冯玉祥|冯玉祥]]驱逐下台，退居[[天津日租界|天津日租界]]当寓公，潛心[[佛學|佛學]]，自号“正道居士”。

1931年[[九一八事變|九一八事變]]後，因日本軍隊佔領中國東北，拒絕與[[日本人|日本人]]往來，頗有民族操守。

1933年2月段祺瑞移居[[上海|上海]]。1935年被任命為[[国民政府委员|国民政府委员]]，但没有就职。1936年11月2日在上海病逝，葬於[[萬安公墓|萬安公墓]]，享年71歲。

段临终遗言有“八勿”，即：“勿因我见而轻起政争，勿尚空谈而不顾实践，勿兴不急之务而浪用民财，勿信过激言行之说而自摇邦本。讲外交者，勿忘巩固国防；司教育者，勿忘保存国粹；治家者，勿弃国有之礼教；求学者，勿鹜时尚之纷华。”

==歷代內閣==
=== 1916年段祺瑞第一次内阁 ===
{{Main|第一次段祺瑞内阁|第一次段祺瑞内阁改组内阁}}
1916年4月23日成立。国务卿段祺瑞，外交總長[[陸徵祥|陸徵祥]]（后由[[曹汝霖|曹汝霖]]兼署），內務總長[[王揖唐|王揖唐]]，財政總長[[孫寶琦|孫寶琦]]，陸軍總長段祺瑞兼，海军总长[[刘冠雄|刘冠雄]]，司法总长[[章宗祥|章宗祥]]，教育總長[[張國淦|張國淦]]，農商總長[[金邦平|金邦平]]，交通總長[[曹汝霖|曹汝霖]]（大半由梁士詒決定），參謀總長[[王士珍|王士珍]]，審計院長[[莊蘊寬|莊蘊寬]]。1916年6月30日改组，1917年5月23日因[[府院之争|府院之争]]结束。

国务总理段祺瑞，外交總長[[唐紹儀|唐紹儀]]（唐未到前由陳錦濤兼署，9月唐抵达，因[[督军团|督军团]]通电反对，旋即辞职，[[伍廷芳|伍廷芳]]接任），內務總長[[許世英|許世英]]，財政總長[[陳錦濤|陳錦濤]]，陸軍總長段祺瑞（兼任），海軍總長[[程璧光|程璧光]]，司法總長[[張耀曾|張耀曾]]（張耀曾未到前由張國淦兼署），教育總長[[孫洪伊|孫洪伊]]（次长[[吳闓生|吳闓生]]代理，[[范源濂|范源濂]]继任总长，孫洪伊改任內務總長，原內務總長[[許世英|許世英]]改任交通總長），農商總長[[張國淦|張國淦]]，交通總長[[汪大燮|汪大燮]]。

=== 1917年段祺瑞第二次内阁 ===
{{Main|第二次段祺瑞内阁}}
1917年7月17日成立，11月22日因第二次[[府院之争|府院之争]]结束。国务总理段祺瑞、外交總長[[汪大燮|汪大燮]]、內務總長[[湯化龍|湯化龍]]、財政總長[[梁啟超|梁啟超]]、陸軍總長段祺瑞兼、海軍總長[[劉冠雄|劉冠雄]]、司法總長[[林長民|林長民]]、教育總長[[范源濂|范源濂]]，農商總長[[張國淦|張國淦]]、交通總長[[曹汝霖|曹汝霖]]。

=== 1918年段祺瑞第三次内阁 ===
{{Main|第三次段祺瑞内阁}}
1918年3月29日成立，12月13日因新国会（[[安福国会|安福国会]]）成立结束。国务总理段祺瑞、外交總長[[陸徵祥|陸徵祥]]、內務總長[[錢能訓|錢能訓]]、財政總長[[曹汝霖|曹汝霖]]兼，陸軍總長[[段芝貴|段芝貴]]、海軍總長[[劉冠雄|劉冠雄]]、司法總長[[朱深|朱深]]、教育總長[[傅增湘|傅增湘]]、農商總長[[田文烈|田文烈]]、交通總長[[曹汝霖|曹汝霖]]、机密院院长曹泾沅。

== 西原借款 ==
[[西原借款|西原借款]]為1917年至1918年间段祺瑞政府和日本签订的一系列公开和秘密借款的总称。 

1917年7月，段祺瑞重任[[中华民国国务总理|中华民国国务总理]]后，为推行“武力统一”政策，镇压孙中山倡导的[[护法运动|护法运动]]，以中國權利為抵押品，向日本大量借款。1917-1918年，段祺瑞共向日本借款5亿日元。其中由西原龟三与段祺瑞政府的曹汝霖、陆宗舆、章宗祥商办议定的有吉会铁路、满蒙四铁路、吉林、黑龙江两省的森林和金矿、有线电信、参战、交通银行等八项借款，共计1.45亿日元。

通过这一借款，段祺瑞把中国山东和东北地区的铁路、矿产、森林等权益大量抵押给日本，但是抵押只是形式上的，而西原借款給中國的條件離苛刻很遠。

西原借款所得款項，日後用於財政性支出占65.22%，軍費占總支出占25.40%。

==中日协定==
{{main|中日共同防敵軍事協定}}
1918年5月16日，日本陆军少将斋藤季治郎与段祺瑞政府代表靳云鹏，在北京秘密签订《中日陆军共同防敌军事协定》。19日又签订《中日海军共同防敌军事协定》。

“协定”的主要内容是：中国与日本采取“共同防敌”的行动；日本在战争期间可以进驻中国境内；日军在中国境外作战时，中国应派兵声援；作战期间，两国互相供给军器和军需品。

通过“协定”，日本派出大批军队进入中国东北，日本迅速取代了沙俄在东三省北部的侵略地位，中国则面临沦为日本[[附属国|附属国]]的局面，不過當時許多中國人的判斷是日本會比蘇聯給中國更好的條件，因為[[八國聯軍|八國聯軍]]時日軍良好紀律及[[日俄戰爭|日俄戰爭]]前後中日友好的印象還深植於當時[[中國人|中國人]]的心中。

==軼事==
段祺瑞篤信[[佛教|佛教]]，平日多半[[素食|喫齋]]、[[念佛|念佛]]、朝夕[[燒香|燒香]]、[[禮佛|禮佛]]，能解《[[金剛經|金剛經]]》、《[[法華經|法華經]]》。

段祺瑞一生酷爱[[围棋|围棋]]，與其子段宏業、姪子[[段宏綱|段宏綱]]皆是圍棋高手。據說段祺瑞少年時曾與[[劉銘傳|劉銘傳]]下過圍棋，段祺瑞執政時资助过大批围棋好手（包括[[顧水如|顧水如]]、[[吳清源|吳清源]]等），被称作“中国围棋的大后台”，段祺瑞長子段宏業是國手級的大師，段祺瑞屢戰屢敗，故往往生氣不語，皆是由段宏綱勸解。

段祺瑞愛小小[[打麻雀|打麻雀]]，賭注不大，但亦是麻雀高手，往往一夜多勝。

段祺瑞之後統治中國的[[蔣介石|蔣介石]]，自稱在[[保定|保定]]讀軍校時，軍校總辦是段祺瑞，故蔣介石自認為「段祺瑞學生」，親自致贈禮金，加上同樣反共，故段祺瑞晚年與蔣介石關係不差。

== 評價 ==
后人评论段祺瑞者甚多，有支持的學者讚揚其品质高尚，爱国有心，组织才能非凡，善于利用政治手腕和军事手腕维护政權。雖為[[軍閥|軍閥]]，但[[人格|人格]]正面，為人严肃刻板，不苟言笑，生活朴素，清廉如水，无积蓄、無房产，不抽、不喝、不嫖、不赌、不贪、不占，人称「六不总理」，享譽於世。也有人骂其军阀习气，刚愎自用，迷信武力，倾心[[权术|权术]]，在军事上并无出色战功和理论，喜用私人為將領。

1924年3月，[[北京大學|北京大學]]紀念25周年活動中，大學生進行國內大人物票選，段祺瑞與[[胡適|胡適]]同居第四名，前三名為[[孫中山|孫中山]]、[[陳獨秀|陳獨秀]]和[[蔡元培|蔡元培]]。

段祺瑞因致电要求[[清帝退位|清帝退位]]、抵制[[洪宪帝制|洪宪帝制]]和讨伐[[張勳復辟|張勳復辟]]这三件事，有“三造共和”的美誉。[[梁启超|梁启超]]评价段祺瑞：“其人短处固所不免，然不顾一身利害，为国家勇于负责，举国中恐无人能比。”[[吴佩孚|吴佩孚]]：“天下无公，正未知几人称帝，几人称王，奠国著奇功，大好河山归再造；时局至此，皆误在今日不和，明日不战，忧民成痼疾，中流砥柱失元勋。”<ref>{{cite web |url = http://news.ifeng.com/history/zhongguojindaishi/detail_2012_03/26/13454198_2.shtml |title = 吴佩孚评价段祺瑞：天下无公 正未知几人称帝 |accessdate = 2012年 |work = |publisher = 凤凰网 |language = zh-cn}}</ref>

1924年，[[冯玉祥|冯玉祥]]发动[[北京政变|北京政变]]，段祺瑞被推为中華民國临时执政。1926年，中國國民黨與中國共產黨聯合發動學生抗議，北京軍警鎮壓请愿学生，造成“[[三·一八惨案|三·一八惨案]]”，旋被冯玉祥赶下台。“三一八”惨案发生后，有说法认为段祺瑞随即赶到现场，向死者长跪不起，并决定终身食素，至死都没有违背这一决定。根據當時新聞檔案，段祺瑞根本没有去“三一八”惨案现场，也没有出现在“悼念三一八惨案死难同胞大会”<ref>[http://view.news.qq.com/zt2012/duanqirui/index.htm 段祺瑞因枪杀学生而终身吃素忏悔?] {{webarchive|url=https://web.archive.org/web/20150402145300/http://view.news.qq.com/zt2012/duanqirui/index.htm |date=2015-04-02 }}</ref>，並且段政府称示威学生为“暴徒”<ref>{{cite book |author = 林本源 |title = 《三一八惨案始末记》 |year = 1926}}</ref>。有回忆者称段祺瑞在三一八之后在家中也吃荤，在其他场合[[吃素|吃素]]只是与信佛有关<ref>{{cite journal |author = 王楚卿 |title = 《段祺瑞公馆见闻》 |journal = 《文史资料选辑》第四十一辑}}</ref>。

[[國民政府|國民政府]]褒揚令︰前臨時執政段祺瑞，持躬廉介，謀國公忠。辛亥倡率各軍贊助共和，功在民國。及袁氏僭號，潔身引退，力維正義，節概凜然。嗣值復辟變作，誓師馬廠，迅遏逆氛，卒能重奠邦基，鞏固政體，殊功碩望，薄海同欽。茲聞在滬溘逝，老成凋謝，惋悼實深，應即予以國葬，並發給治喪費一萬元。生平事蹟，存備宣付史館。用示國家篤念耆勳之至意。此令！
