'''段姓'''是[[漢姓|漢姓]]之一，在《[[百家姓|百家姓]]》中排第218位。2006年[[中国|中国]]段姓人口排名第81，在[[韩国|韩国]]也有分布。

== 源出 ==
;漢族
#出自[[姬姓|姬姓]]，是[[春秋|春秋]]时[[郑武公|郑武公]]之子[[叔段|叔段]]的后代。
#据《[[史记|史记]]·老子列传》所载，[[老子|老子]]之子宗，[[春秋|春秋]]时为[[魏国|魏国]]将领，受封于[[段干|段干]]。後代改姓段。
#[[儒家|儒家]]學者[[段干木|段干木]]，后代有以单姓段为姓氏的。
#出自[[殷姓|殷姓]]，[[晉朝|晉朝]][[殷浩|殷浩]]後裔被奸人所害而[[族滅|族滅]]，為避禍而改姓段。

;漢族以外
#出自[[五胡十六国|五胡十六国]]時期[[段部鮮卑|段部鮮卑]]後裔。
#出自[[白族|白族]]，据《[[姓氏寻源|姓氏寻源]]》所载：“云南蛮段氏，魏末段延没蛮代为酋帅，裔孙凭入朝拜为[[云南|云南]][[刺史|刺史]]，本出[[武威|武威]]。”
#臺灣臺南市[[吉貝耍|吉貝耍]][[西拉雅族|西拉雅族]]，居民多數漢名為段姓。

== 古代名人 ==
*[[段熲|段熲]]：東漢名將，累官至太尉，號「涼州三明」
*[[段煨|段煨]]：漢末軍閥，官至大鴻臚、光祿大夫
*[[段巧笑|段巧笑]]：三国发明家
*[[段随|段随]]：五胡十六國時期西燕君王
*[[段部鮮卑|段部鮮卑]]諸位首領
*[[段韶|段韶]]：北齊重要軍事人物，官至太尉、相國
*[[段志玄|段志玄]]：唐朝军事人物
*[[段成式|段成式]]：唐朝作家
*[[大理國|大理]]皇族：
**[[段宝竜|段宝竜]]、[[段思平|段思平]]、[[段思英|段思英]]、[[段思良|段思良]]、[[段思聪|段思聪]]、[[段素顺|段素顺]]、[[段素英|段素英]]、[[段素廉|段素廉]]、[[段素隆|段素隆]]、[[段素真|段素真]]、[[段素兴|段素兴]]、[[段思廉|段思廉]]、[[段廉义|段廉义]]、[[段寿辉|段寿辉]]、[[段正明|段正明]]、[[段正淳|段正淳]]、[[段和誉|段和誉]]、[[段正兴|段正兴]]、[[段智兴|段智兴]]、[[段智廉|段智廉]]、[[段智祥|段智祥]]、[[段祥兴|段祥兴]]、[[段兴智|段兴智]]
*[[段玉裁|段玉裁]]：清朝语言学家
*[[段祺瑞|段祺瑞]]：清末、民初政治軍事人物

== 現代名人 ==
*[[段锡朋|段锡朋]]：民国政治人物
*[[段德昌|段德昌]]：中国军事人物
*[[段君毅|段君毅]]：中国政治人物，曾任中共中央顾问委员会常务委员、北京市委第一书记等职
*[[段学复|段学复]]：中国数学家
*[[段宜康|段宜康]]：台湾政治人物
*[[段永平|段永平]]：中国手机厂商[[OPPO|OPPO]]和[[Vivo|Vivo]]品牌创办人
*[[段偉倫|段偉倫]]：香港藝人，曾在亞洲電視擔任演員
*[[段奕宏|段奕宏]]：中国演員
*[[段宜恩|段宜恩]]：韓國偶像團體[[GOT7|GOT7]]成員，[[臺灣裔美國人|臺灣裔美國人]]
*[[段崇智|段崇智]]：國際知名生物醫學專家，[[香港中文大學|香港中文大學]]校長（2018年起）
*[[段林希|段林希]]：中國女歌手
*段世革（1924－2004）：台灣[[淡江大學|淡江大學]]國文系副教授，大理皇族後代一支遷徙至湖南，1949年至台灣，女兒段馨君。
*段馨君：國際著名劇場表演研究學者，[[國立交通大學|國立交通大學]]教授，大理皇族後代一支遷徙至湖南，父親段世革。
