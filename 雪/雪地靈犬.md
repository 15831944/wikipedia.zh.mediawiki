《'''雪地靈犬'''》（{{lang-en|''Balto''}}）是一部[[美國|美國]]真人[[動畫電影|動畫電影]]，這部電影採用真人與動畫來分別演出，於1995年12月23日於美國上映。《雪地靈犬》曾經獲得[[青年藝術家獎|青年藝術家獎]]提名。《雪地靈犬》的劇情以{{link-en|1925年血清運送行動|1925 serum run to Nome}}為底本，並進行大量改編，此電影是[[史蒂芬·史使匹柏|史蒂芬·史使匹柏]]的工作室{{link-en|安培林動畫部門|Amblimation}}的第三部也是最後一部動畫電影。此電影的真人演出部分於[[紐約|紐約]][[中央公園|中央公園]]拍攝，雖然《雪地靈犬》的票房在當年因為[[皮克斯動畫工作室|皮克斯動畫工作室]]的動畫[[玩具總動員|玩具總動員]]的亮眼表現而顯得黯然失色，但之後在{{link-en|家庭視頻|Home video}}的銷售上的成功，導致了以[[錄影帶首映|錄影帶首映]]形式出售的兩部續集系列：雪地靈犬2(2002)、雪地靈犬3(2004)。然而沒有一位配音員在續集回歸。
== 劇情簡介 ==
根據真實故事改編，在1925年，偏遠的阿拉斯加中諾姆小鎮中，突如其然爆發的嚴重傳染病-[[白喉|白喉]]，使城鎮中的孩童性命各個危在旦夕，在院方缺乏抗毒[[血清|血清]]的情況下，這個冰封小鎮，只能依靠[[狗拉雪橇|狗拉雪橇]]的方式從遠在600哩外的NENANA運送血清至諾姆，然而，運送血清的雪橇犬隊伍卻迷失在了暴風雪中，巴圖(Balto)是否能找到失聯的雪橇隊，及時將血清送往危在旦夕的諾姆孩童手中呢?
