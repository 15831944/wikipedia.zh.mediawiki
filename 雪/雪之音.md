《'''雪之音'''》（{{lang-ja|雪の音}}）是[[GReeeeN|GReeeeN]]的第18張單曲，於2012年12月19日由[[環球音樂_(日本)|NAYUTAWAVE RECORDS]]發行。

== 概要 ==
此單曲距離上一張單曲《[[OH!!!!困惑!!!!|OH!!!!困惑!!!!]]》（{{lang|ja|OH!!!!迷惑!!!!}}）發行約七個月，為[[GReeeeN|GReeeeN]]於2012年發行的第4張單曲。單曲的「初回限定盤」附送收錄了《雪之音》[[音樂影片|音樂影片]]的DVD。<ref name="natalie1127">{{cite web|url=http://natalie.mu/music/news/80636 |language=ja |title=<span lang="ja">GReeeeN、7カ月ぶりシングルはウインターラブソング</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2012-11-27 |accessdate=2013-02-16}}</ref>A面曲《雪之音》於同年12月5日起於[[iTunes_Store|iTunes Store]]、[[RecoChoku|RecoChoku]]等開放下載。<ref>{{cite web|url=http://natalie.mu/music/news/81093 |language=ja |title=<span lang="ja">GReeeeN、写真部を舞台にした恋模様を描く新曲PV公開</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2012-12-05 |accessdate=2013-02-16}}</ref>B面曲《在繁星落下的夜晚》則於同年12月19日開放下載。<ref>{{cite web|url=http://www.universal-music.co.jp/greeeen/news/2012/12/05/ |language=ja |title=<span lang="ja">GReeeeN ニューシングル「雪の音」ダウンロード・スタート！＆ PV解禁！</span> |publisher=UNIVERSAL MUSIC LLC |date=2012-12-05 |accessdate=2013-02-16}}</ref>

== 歌曲 ==
《雪之音》是GReeeeN第一首於單曲中發行，以[[冬天|冬天]]和[[戀愛|戀愛]]為主題的歌曲。<ref name="WHAT's IN? WEB">{{cite web|url=http://www.musicnet.co.jp/whatsin/news/43723 |language=ja |title=<span lang="ja">GReeeeN、甘酸っぱい恋愛模様描く「雪の音」MV公開！</span> |publisher=WHAT's IN? WEB |date=2012 |accessdate=2013-02-16}}</ref>根據GReeeeN於官方網站的介紹，他們將「發覺喜歡了你的那個瞬間」寫下來，「難過，心痛，卻又温暖」，「以冬天的氛圍，將世上最幸福的一瞬間放進歌裏」。<ref>{{cite web|url=http://www.universal-music.co.jp/greeeen/news/2012/12/19/ |language=ja |title=<span lang="ja">GReeeeN　ニューシングル「雪の音(ﾕｷﾉﾈ)」とうとうCD発売だー！！</span> |publisher=UNIVERSAL MUSIC LLC |date=2012-12-19 |accessdate=2013-02-16}}</ref>Billboard JAPAN的武川春奈評論指《雪之音》發行日期接近[[聖誕節|聖誕節]]，歌中亦有[[鈴|鈴]]的聲音，隨處散發着冬天的色彩。她又指這首帶有浪漫旋律的歌所描寫苦樂參半的戀愛心情，能讓很多十多歲至二十多歲的人有同感。她最後稱讚GReeeeN所寫的情歌「純度經常都很高」。<ref>{{cite web |url=http://www.billboard-japan.com/hs_discs/detail/585 |language=ja |title=<span lang="ja">GReeeeN “恋に気付く瞬間”を描いた冬のラブソング</span> |publisher=Billboard JAPAN |date=2012 |accessdate=2013-02-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20130314210916/http://www.billboard-japan.com/hs_discs/detail/585 |archivedate=2013年3月14日 |df= }}</ref>

《雪之音》被採用為[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）「JR SKISKI」的電視廣告歌曲。<ref name="natalie1127" />

== 音樂影片 ==
《雪之音》的[[音樂影片|音樂影片]]（PV）由[[小川直也_(導演)|直]]執導。<ref>{{cite web|url=http://www.spaceshowertv.com/search/detail.cgi?mu=0092936&ch=0 |language=ja |title=<span lang="ja">雪の音</span> |publisher=SPACE SHOWER TV |date= |accessdate=2013-02-16}}</ref>三名主角由[[中山繪梨奈|中山繪梨奈]]、[[竹内太郎|竹内太郎]]和{{lang|ja|[[大坪あきほ|大坪あきほ]]}}飾演。PV以一所大學的攝影部為舞台，描述「一心一意於攝影的遲鈍男生」、「傾慕這個男生的笨拙女生」和「率直表達感情的純真女孩」三人間的愛情故事。<ref name="WHAT's IN? WEB" />PV有大學校園和下雪街頭的場景。<ref name="WHAT's IN? WEB" />這個PV是GReeeeN的第14個PV，但他們仍未曾參與PV的演出。<ref name="WHAT's IN? WEB" />

== 排行榜成績 ==
《雪之音》於[[Oricon公信榜|Oricon公信榜]]單曲週榜取得第6名。<ref name="oricon" />此單曲於[[Japan_Hot_100|Billboard JAPAN Hot 100]]則取得第2名，屈居於[[乃木坂46|乃木坂46]]《[[制服模特兒|制服模特兒]]》之下。<ref name="hot100" />A面曲《雪之音》亦曾於iTunes Weekly Top Song Chart連續兩星期取得冠軍（2012年12月4日至12月9日、2012年12月10日至2012年12月16日）。<ref>{{cite web |url=http://listen.jp/store/musicnews_42420_all.htm |language=ja |title=<span lang="ja">GReeeeN、JR SKISKIのCFソングで話題のニューシングル「雪の音(ユキノネ)」発売</span> |publisher=Listen Japan |date=2012-12-19 |accessdate=2013-02-16 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
