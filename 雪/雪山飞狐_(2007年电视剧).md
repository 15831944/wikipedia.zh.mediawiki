'''《雪山飞狐》'''（{{lang-en|'''Fox Volant of the Snowy Mountain'''}}）2006年制作，导演[[王晶|王晶]]、[[钟少雄|钟少雄]]、[[谭友业|谭友业]]、[[江道海|江道海]]，主演聂远、朱茵、钟欣桐、安以轩、方中信、黄秋生、谭耀文、吴卓翰。2007年4月开始在各地电视台播出，同年5月14日在'''亚视本港台'''首播<ref>[http://yule.sohu.com/20070413/n249422377.shtml   广东台跟亚视争档期 新版《雪山飞狐》下周播出]</ref>。2018年5月14日MOD龍華戲劇台播出。OTT平台由 [[LiTV_線上影視|LiTV 線上影視]]全集播出。

==剧情==
{{See also|雪山飞狐}}
该剧根据[[金庸|金庸]]的武侠小说《[[雪山飞狐|雪山飞狐]]》改编而成。

== 花絮 ==
该剧耗资巨大，仅仅是“雪山对决”就耗资4000万人民币。<ref name="fish" >{{cite web |url=http://ent.sina.com.cn/v/m/2013-11-18/15574044859.shtml |title=《雪山飞狐》韩国开播 聂远掀经典武侠热 |accessdate=2013 |publisher=新浪网 |work= |language=zh-hans}}</ref>

2013年11月，该剧登陆[[韩国|韩国]][[中华TV|中华TV]]。<ref name="fish" />韩国《[[朝鲜日报|朝鲜日报]]》称：“中国电视剧凭借服装华丽、弘大精良的制作场景日益受到韩国观众热捧”是这部电视剧在韩国播放的重要原因。<ref>{{cite web|url=http://news.xinhuanet.com/ent/2013-11/18/c_125717708.htm|title=《雪山飞狐》今晚韩国开播 聂远再掀经典武侠热|accessdate=2013|publisher=新华网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20140408213031/http://news.xinhuanet.com/ent/2013-11/18/c_125717708.htm|archivedate=2014-04-08}}</ref><ref>{{cite web|url=http://ent.ynet.com/3.1/1311/18/8454022.html|title=《雪山飞狐》今晚韩国开播 聂远掀经典武侠热|accessdate=2013|publisher=北青网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160304205825/http://ent.ynet.com/3.1/1311/18/8454022.html|archivedate=2016-03-04}}</ref>

聂远夸赞导演王晶是“鬼才”。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2008-01-07/16451865368.shtml |title=《雪山飞狐》聂远谈王晶不愧是个“鬼才”导演 |accessdate=2008 |publisher=新浪网 |work= |language=zh-hans}}</ref>

该剧在中国大陆播出之后，收视率颇高，但是观众质疑该剧“太时尚”，胡斐的“辫子”都没有了。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2008-01-29/12071896213.shtml |title=聂远回应新版《雪山飞狐》争议 只想完成武侠梦 |accessdate=2008 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/v/m/2008-01-24/15561890877.shtml |title= 《雪山飞狐》被批太时尚 聂远回应这不是历史剧|accessdate= 2008|publisher=新浪网 |work= |language=zh-hans}}</ref>
