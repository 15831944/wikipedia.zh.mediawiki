《'''雪豹'''》，2010年上映于中国大陆的战争剧，讲述了[[中华民国|中华民国]]从“[[一二八事变|一二八事变]]”到[[淞沪抗战|淞沪抗战]]一直到[[抗日战争|抗日战争]]结束的历史。

== 剧情 ==
1932年初，大日本帝国在中华民国[[上海市|上海市]]寻事挑衅，随后向[[国民革命军第十九路军|国民革命军第十九路军]]发起攻击，“雪豹突击队”由此诞生，而后他们将面临[[大日本帝国陆军|大日本帝国陆军]]和国军的攻击，最后坚持到抗战胜利。

== 花絮 ==
该剧改编自网络小说《特战先驱》。<ref>{{cite web |url= http://amuse.nen.com.cn/amuse/483/3704483.shtml|title=文章《雪豹》首演军人 马伊琍恳求导演用替身 |accessdate=  2011-01-24 |publisher= 辽宁网|work= |language =zh-cn}}</ref>

文章作为主演，出演了一个精通通晓[[英文|英文]]、[[德文|德文]]、[[日文|日文]]的特战英雄。<ref>{{cite web |url=http://ent.qq.com/a/20100714/000297.htm |title= 《雪豹》将播 文章自曝戏中“客串”编剧导演|accessdate=2010年7月14日 |publisher= 腾讯网|work= |language =zh-cn}}</ref>

女主角陶飞霏则出演一个打入中国国民党内部的中共间谍，同时是交际花。<ref>{{cite web |url= http://ent.qq.com/a/20100226/000593.htm|title=《雪豹》上海杀青 陶飞霏变身性感交际花 |accessdate= 2010年2月26日|publisher=腾讯网 |work= |language =zh-cn}}</ref>
