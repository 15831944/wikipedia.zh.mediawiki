'''雪克来提·扎克尔'''（{{lang-ug|شۆھرەت زاكىر}} / {{lang|ug-Latn|Shöhret Zakir}}，{{bd|1953年|8月||}}），[[维吾尔族|维吾尔族]]，[[新疆维吾尔自治区|新疆]][[伊宁市|伊宁]]人；[[乌鲁木齐职业大学|乌鲁木齐职业大学]][[英语|英语]]专业毕业，[[中国社会科学院研究生院|中国社会科学院研究生院]][[工商管理|企业管理]]专业在职研究生，[[天津大学|天津大学]]高级管理人员工商管理硕士。1985年6月加入[[中国共产党|中国共产党]]。曾任[[第十一届全国人民代表大会|第十一届全国人大]][[全国人民代表大会民族委员会|民族委员会]]副主任委员，[[中华全国归国华侨联合会|中华全国归国华侨联合会]]副主席，[[新疆维吾尔自治区|新疆维吾尔自治区]][[人民代表大会制度|第十二届人大常委会]]主任。现任[[中国共产党新疆维吾尔自治区委员会|中共新疆维吾尔自治区党委]]副书记、[[新疆维吾尔自治区人民政府|自治区政府]]主席<ref name="xhw">{{Cite web|url=https://web.archive.org/web/20140811202648/http://news.xinhuanet.com/ziliao/2014-01/21/c_126035896.htm|title=雪克来提·扎克尔简历|accessdate=2018-12-22|date=2014-01-21|publisher=新华网|archiveurl=https://web.archive.org/web/20140811202648/http://news.xinhuanet.com/ziliao/2014-01/21/c_126035896.htm|archivedate=2014-08-11}}</ref>。

== 家庭背景 ==
雪克来提·扎克尔出身革命世家。祖父[[凯吾尔·扎克尔|凯吾尔·扎克尔]]是[[國民大革命|大革命时期]][[伊犁|伊犁地区]]的思想进步、活跃的知识分子，曾是“新疆天山（商会）社”的社长，后加入[[新疆民众反帝联合会|新疆民众反帝联合会]]；1943年9月，和中共党员[[陈潭秋|陈潭秋]]、[[毛泽民|毛泽民]]等被[[军阀|军阀]][[盛世才|盛世才]]秘密杀害。父亲[[阿不都拉·扎克洛夫|阿不都拉·扎克洛夫]]，早年赴[[苏联|苏联]]留学；1949年12月，成为中共在新疆地区发展的首批15名维吾尔族党员之一，是[[新疆省人民政府|新疆省人民政府]]首任秘书长；[[文化大革命|文化大革命]]前，官至[[中国共产党新疆维吾尔自治区委员会|中共新疆维吾尔自治区党委]]常委、[[新疆维吾尔自治区人民政府|自治区政府]]副主席。<ref>{{cite news|url=http://news.takungpao.com/mainland/focus/2015-01/2877351.html|title=“红色黑马”雪克来提|date=2015-01-04|work=大公网|accessdate=2015-01-13}}</ref>

和大多数维吾尔族不同的是，雪克来提·扎克尔拥有家族姓氏，即扎克尔。对他的称呼可以是“扎克尔先生”。

== 早年经历 ==
[[文化大革命|文化大革命]]期间，1970年12月至1972年2月，雪克来提·扎克尔先后在[[新疆维吾尔自治区|新疆]][[玛纳斯县|玛纳斯县]]、[[乌鲁木齐县|乌鲁木齐县]]接受再教育；而后，被分配到[[乌鲁木齐市|乌鲁木齐市]]第三十二小学，任代课教师；1974年2月，调[[乌鲁木齐县|乌鲁木齐县]]地窝堡学校任教。1978年3月起，在[[湖北省|湖北]][[长江大学|江汉石油学院]][[電子計算機|计算机]]专业学习；毕业后，担任新疆石油地球物理研究所技术员。1984年6月，扎克尔调[[新疆维吾尔自治区人民政府|新疆维吾尔自治区人民政府]]工作，开启从政之路。<ref name="xhw" />

== 仕途 ==
1984年6月，扎克尔调任[[新疆维吾尔自治区人民政府|新疆维吾尔自治区经济委员会]]工业处副主任科员、主任科员；1982年9月至1986年7月，在[[乌鲁木齐职业大学|乌鲁木齐职业大学]]英语专业学习；并在此期间，加入[[中国共产党|中国共产党]]。自1988年5月起，历任自治区经委副处长、处长，经贸委副秘书长；1997年7月，升任自治区经贸委副主任；其间，曾先后在[[中共中央党校|中共中央党校]]、[[中国社会科学院|中国社会科学院]]工业经济系进修。2000年12月，扎克尔出任[[中国共产党|中共]][[乌鲁木齐市|乌鲁木齐]][[市委書記|市委副书记]]、副市长、代理市长；次年3月，正式当选[[乌鲁木齐市|乌鲁木齐市]][[市长|市长]]；其间曾于2004年3月到2005年1月，在[[中共中央党校一年制中青年干部培训班学员列表|中共中央党校一年制中青年干部培训班]]学习。2005年12月，出任中共[[新疆生产建设兵团|新疆生产建设兵团]]党委常委、兵团副政委；于2007年，获[[天津大学|天津大学]]管理学院高级管理人员工商管理硕士学位。<ref name="xhw" />

在2008年3月召开的[[第十一届全国人民代表大会|第十一届全国人大]]第一次会议上，扎克尔当选[[全国人民代表大会民族委员会|全国人大民族委员会]]委员；2009年7月起，还兼任[[中华全国归国华侨联合会|中华全国归国华侨联合会]]副主席；2011年6月起，任全国人大民族委员会副主任委员、兼[[中华全国归国华侨联合会|全国侨联]]副主席，并被明确为副部级干部。2013年12月，出任中共[[新疆维吾尔自治区|新疆维吾尔自治区]][[人民代表大会制度|人大常委会]]党组书记；次年1月，在自治区第十二届人大第二次会议上，当选自治区人大常委会主任。<ref name="xhw" /><ref>{{cite news|url=http://renshi.people.com.cn/n/2013/0317/c139617-20815243.html|title=第十二届全国人民代表大会八个专门委员会主任委员、副主任委员、委员名单|date=2013-03-017|work=人民网|accessdate=2013-03-22}}</ref><ref>{{cite news|url=http://news.163.com/api/14/0120/20/9J2E5ML90001124J.html|title=雪克莱提·扎克尔当选新疆人大常委会主任|date=2014-01-20|publisher=网易|accessdate=2014-01-20}}</ref>2014年12月，接替调[[中华人民共和国国务院|国务院]]工作的[[努尔·白克力|努尔·白克力]]，出任[[中国共产党新疆维吾尔自治区委员会|中共新疆维吾尔自治区党委]]副书记、[[新疆维吾尔自治区人民政府|自治区政府]]代主席。2015年1月起任自治区政府主席。
