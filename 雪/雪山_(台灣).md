'''雪山'''，屬於[[雪山山脈|雪山山脈]]，為全[[台灣|台灣]]第二高峰，標高 3,886 公尺，僅次於[[玉山山脈|玉山山脈]]的[[玉山|玉山主峰]]（3,952 公尺）。'''雪山'''在日治時代被稱做次高山，是兩座超越富士山高度的台灣山脈之一。在[[台灣百岳|台灣百岳]]之中，雪山與[[玉山|玉山]]、[[秀姑巒山|秀姑巒山]]、[[南湖大山|南湖大山]]、[[北大武山|北大武山]]合稱「<nowiki>五岳</nowiki>」，為台灣最具代表性的五座高山之一。

==歷史==
雪山主峰地區是世居在此的[[泰雅族|泰雅族]]人的獵區。在志佳陽社（今環山村）的稱呼中，雪山名為「'''Babo Hagai'''」，意為「石頭山」，描述雪山頂到處堆積的砂岩塊。<ref>出自中國電視公司製作，「台灣五頂峰：雪山山脈｣節目，泰雅族人的訪問。</ref>；另有泰雅語稱為「'''Mahamayan'''」，譯作'''馬哈馬顏山'''。雪山在泰雅語中的第三個稱法，為「'''Sekoan'''」，意為「碎石與裂縫」。在《台灣府誌》中，描寫其積雪之景，稱為'''雪山'''、'''雪翁山'''，日人[[幣原坦|幣原坦]]認為「雪翁山」或「'''雪高翁山'''」為「Sekoan」的音譯。在《淡水廳誌》與《苗栗廳誌》中，稱此山為'''玉山'''，《苗栗廳誌》中，玉山霽雪為苗栗八景之一。然而如今所稱之玉山，係指台灣最高峰<ref>第二世代台灣百岳全集上冊，戶外生活圖書股份有限公司，陳遠見主編，2007年8月31日初版</ref>。

1867年，曾有英國軍艦Sylvia號行經台灣海域，據說船長以目視發現高聳的雪山，便將山以船命名為「''Sylvia''」，在早期的西方文獻中便以'''Mt.Sylvia'''稱呼雪山。<ref>{{cite book|title=臺灣今古談|date=1930-11-30|author=劉克明|publisher=新高堂書店}}</ref>1872年，[[馬偕|馬偕博士]]曾進入雪山山區活動；接著台灣進入[[台灣日治時期|日治時期]]，大正4年（1915年）7月10日，隸屬「蕃務本署」的測量員財津久平等人，經由埤亞南社（今南山村）越過埤亞南鞍部（今[[思源埡口|思源埡口]]），從雪山東峰稜線，順利登上雪山主峰，並量出高度為12,972英尺，比[[日本|日本]]本土最高峰[[富士山|富士山]]還高；1923年，裕仁親王（後來的[[昭和天皇|昭和天皇]]）訪問台灣，以「台灣第二高峰、僅次於新高山的山」命名為'''次高山'''；與新高山（玉山）、[[能高山|能高山]]並稱「台灣三高」<ref>http://www.sunriver.com.tw/takayama/sheishan.htm 台灣百岳全集試閱頁</ref>。山頂設有次高神社，包括水泥製[[神龕|神龕]]與[[鳥居|鳥居]]，現已不存。

戰後，雪山之名開始被使用；當時總統[[蔣中正|蔣中正]]在巡視[[中橫公路|中橫公路]]時，曾將雪山取名為'''興隆山'''，但此名未被長期廣泛地採用。

==地理==
[[File:HsuehMountain2.jpg|thumb]]

雪山主峰與副峰群分布在雪山山脈的中段，約在[[台中市|台中市]]與[[苗栗縣|苗栗縣]]的市縣界上，地處[[雪霸國家公園|雪霸國家公園]]境內。以雪山主峰為中心，周圍一帶地質特色、岩性皆相似的山塊，稱為'''雪山地壘'''。雪山地壘的基岩為[[始新世|始新世]]晚期或[[漸新世|漸新世]]早期之[[四稜砂岩|四稜砂岩]]，是一種[[石英|石英]]變質[[砂岩|砂岩]]，硬度非常高，耐風化侵蝕，因此造就了雪山地壘的高聳地勢。

以雪山主峰為頂點，共有六條稜脈匯聚，分別是：
*主稜：又稱西稜，為雪山山脈主稜，自雪山主峰向西延伸，經過翠池三叉山後北轉，在博爾可山再折向西南方。沿線包括[[火石山|火石山]]、[[頭鷹山|頭鷹山]]、[[大雪山_(台灣)|大雪山]]、[[中雪山|中雪山]]四座百岳山峰。
*南稜：自翠池三叉山向南轉折，沿線有雪山西南峰、[[大劍山|大劍山]]、油婆蘭山、[[佳陽山|佳陽山]]、[[劍山_(台灣)|劍山]]等高山，又稱雪劍稜脈、劍山稜脈或大小劍稜脈。
*北稜：自雪山主峰，向北經北稜角、凱蘭特崑山、[[雪山北峰|雪山北峰]]、穆特勒布山、素密達山、布秀蘭山、巴紗拉雲山到[[大霸尖山|大霸尖山]]、[[小霸尖山|小霸尖山]]的稜脈，即是[[聖稜線|聖稜線]]，亦稱雪霸線。
*北東稜：從布秀蘭山自北稜分出往東的支稜，沿線有[[品田山|品田山]]、[[池有山|池有山]]、[[桃山_(臺灣)|桃山]]與[[喀拉業山|喀拉業山]]，即[[武陵四秀|武陵四秀]]。
*東南稜：從雪山主峰南面，接上雪山南峰、[[志佳陽大山|志佳陽大山]]。
*東支稜：由主峰往東延伸至七家灣溪河谷，沿線經過甘木林山、[[雪山東峰|雪山東峰]]；也稱雪東線。

==水文==
雪山主峰主稜為[[大安溪|大安溪]]、[[大甲溪|大甲溪]]的分水嶺。主稜以北為大安溪集水區，以南則是大甲溪。發源於此的溪流包括雪山溪（匯入大安溪）、司界蘭溪、高山溪與桃山西溪（匯入大甲溪）。

在主峰西稜碎石坡下一公里處，為[[翠池|翠池]]，是大安溪源頭之一，也是台灣最高的高山湖泊。翠池之上、下各有上翠池、下翠池二池，上翠池為季節性小沼，乾季無水。

== 地質 ==
以雪山主峰為中心，共有數個'''冰斗地形'''構造環繞，其中規模最大的為一號冰斗，位於雪山主峰與北稜角下，開口朝向東方，長約1000公尺，最寬處約600公尺，不僅規模最大，形狀也完整。二號冰斗位於北稜角北側，規模稍小，但所在位置更高<ref name=tgru>[http://tgru.geog.ntu.edu.tw/GeogManager/magazine/NO_10/CH_2/SEL_1/10-3.htm 雪山的圈谷是崩谷、冰斗還是雪蝕斗?]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}，地景保育研究</ref>。三、四號冰斗位於主峰與北稜角間的埡口北面，為堆滿巨石塊的石流坡；五號、六號冰斗位於雪山主峰西北側，五號冰斗底的草窪地，積水形成上翠池<ref>中國電視公司監製，麥覺明製作，MIT台灣誌－雪劍線探勘</ref>。七號冰斗則位於雪山主峰西北坡下，為翠池的函水區。

除了雪山地區之外，在[[南湖大山|南湖大山]]、[[大霸尖山|大霸尖山]]、[[玉山|玉山]]、[[奇萊山|奇萊山]]、[[合歡山|合歡山]]等台灣山區亦可見類似的地形結構。最早關於台灣高山圈谷研究的文獻出自日籍學者[[早扳一郎|早扳一郎]]<ref name=tgru/>；其後博物學家[[鹿野忠雄|鹿野忠雄]]經多次考察，發表台灣高山圈谷為[[冰河|冰河]]地形遺跡的論文，引發關於圈谷成因的學術論戰。有論者認為，台灣高山地區是否曾有的冰河，證據並不充分，而所謂圈谷其實是水系發育、向源侵蝕與風化作用所造成的崩谷地形<ref>{{cite web |url=http://www.taroko.gov.tw/TarokoPortal/Files/Report/777958134332_A.pdf |title=存档副本 |accessdate=2008-06-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20071224002031/http://www.taroko.gov.tw/TarokoPortal/Files/Report/777958134332_A.pdf |archivedate=2007-12-24 }} 南湖大山圈谷群古冰河遺跡研究初步調查，2000年6月，[[王鑫_(台灣學者)|王鑫]]、楊建夫、郭彥超</ref>。其後由於一度缺乏實地研究，圈谷地型的確實成因陷入多方爭議而無定論。

1983年後，地理學家曾昭璇提出雪山圈谷為冰河遺跡的可能性；1998年，王鑫、崔之久、宋國城、楊建夫等學者在雪山的數個圈谷底部發現擦痕，並確認圈谷底部的基岩堤為冰坎，根據熱螢光定年成果，雪山地區的冰河作用可分為三期，最早為山莊冰期（44.25±3.72ka BP）、水源冰期（18.62±1.52ka BP）和最後的雪山冰期（10.68±0.84ka BP）三次主要冰期。並推算雪山山區在冰河最盛期時的雪線可能僅有3100公尺，因此雪線以上的山區都有可能長期積雪，進而發育出冰河，以及各種冰蝕的地形。雪山地區其他疑似冰河侵蝕的地形特徵，包括：
*[[冰坎|冰坎]]：受冰河推擠，而堆積在冰河底部的條狀基岩堤，可見於一號圈谷底部、翠池出水口下游。
*[[冰斗湖|冰斗湖]]：位於冰斗底部的冰蝕凹地，積水則成池。如翠池、上翠池等。
*[[冰蝕埡口|冰蝕埡口]]：冰河分流處受冰雪侵蝕所形成的缺口。北稜角與主峰間的鞍部可能為此地形。
*[[冰蝕擦痕|冰蝕擦痕]]：冰河搬運的岩塊與底部岩床摩擦，在裸岩上形成數條方向一致的細微擦痕。可見於一號冰斗出口的黑森林內、冰斗底部的岩石上。

由於台灣多降水，高海拔地區日夜溫差大，在嚴重的風化侵蝕作用下，除了一、二號冰斗之外，雪山山區的其它冰斗圈谷構造都已不鮮明，無明顯巨大下凹的碗狀外型。

== 景觀 ==
=== 雪山主峰 ===
'''雪山主峰'''標高3,886公尺，峰頂地形平緩，設有一等[[三角點|三角點]]與石碑。雪山的地質構造以變質砂岩為主，在主峰周邊各處均可見變質砂岩的交疊構造出露，以及散落的風化砂岩碎塊。主峰東面為雪山一號圈谷，北面經埡口接上北稜角，西面稜脈接上翠池三叉山，南面則為斷崖崩壁。

=== 北稜角 ===
'''雪山主峰北稜角'''為主峰北邊的山塊，海拔高度3,880公尺，與主峰相距甚近，高度相仿，因此普遍不以副峰視之。北稜角與主峰間隔著一處埡口，該埡口即為一號圈谷與四號圈谷的分壁，可能是冰河發育所造成的冰蝕構造。

=== 副峰群 ===
以雪山為名的群峰，尚有：
*[[雪山東峰|雪山東峰]]，百岳之一，位於雪東稜線上，海拔3,201公尺。
*雪山南峰，位於志佳陽稜線上，3,505公尺。
*雪山西南峰，位於雪劍稜線，3,471公尺。
*[[雪山北峰|雪山北峰]]，百岳之一，位於聖稜線上，3,703公尺。

==氣候==
雪山地區為典型的台灣'''高山寒帶重濕氣候'''，年降雨量三千[[公釐|公釐]]以上，年均溫[[攝氏|攝氏]]5度左右；降雨主要集中在冬季，雪季從11月到隔年5月；夏季則偶有颱風。受山脈地型開口影響，冬季的東北[[季風|季風]]常夾帶大量水氣自[[蘭陽溪|蘭陽溪]]谷湧入，造成大量降雪，因此雪山山區是台灣冬季降雪最多的地區之一。在雪山周遭的武陵、梨山、環山等谷地區域，氣溫稍高，屬高山溫帶氣候區。

==生態==
雪山地區自武陵、梨山溪谷的溫帶氣候，在短短數公里內因地勢拔高而演變至寒帶，因此林相多變，連帶的也有多種野生動物棲息在此地，包括[[台灣黑熊|台灣黑熊]]、[[山羌|山羌]]、[[金翼白眉|金翼白眉]]、[[帝雉|帝雉]]、[[櫻花鉤吻鮭|櫻花鉤吻鮭]]等[[台灣特有種列表|台灣特有種]]生物。在植物方面，雪山主峰基部以杉樹與闊葉樹混生林為主要林相，海拔高度提升後，轉為以[[台灣鐵杉|鐵杉]]、[[台灣冷杉|冷杉]]、[[玉山圓柏|玉山圓柏]]等多種針葉樹為主；在峰頂附近，則只有玉山圓柏、[[玉山杜鵑|玉山杜鵑]]、[[玉山小檗|玉山小檗]]等植物能夠生長。

雪山主峰東側有一全台海拔最高的冷杉純林，俗稱'''黑森林'''；在主峰西坡之下、翠池附近則有一處玉山圓柏純林，此區的因處於背風處，玉山圓柏生長得特別高大，型態有別於台灣其他地區；多數大型的柏樹年齡可能在兩三千年以上<ref name=libertytimes>http://www.libertytimes.com.tw/2008/new/apr/13/today-life2.htm 自由電子報：雪山千年圓柏 疑遭盜伐取暖</ref>。此外，主峰頂附近尚有玉山圓柏分布，但多已因火災而死亡，呈現白化的枯木狀態。

== 登山路線 ==
[[File:TheHolyRidgeline.jpg|thumb]]

雪山是台灣最熱門的登山景點之一，由於鄰近中橫公路，附近有[[梨山|梨山]]、環山、南山等高冷蔬菜園區，加上位於七家灣溪有[[武陵農場|武陵農場]]，開發甚早，交通基礎建設均完善。但在[[台8線|台8線]]封閉後，雪山山區的主要聯外公路只剩從宜蘭進入的[[台7線|台7線]]。

攀登雪山主要是順著稜線來進行，因此可分為六條路線，分別是：
*雪東線：自武陵農場登山口起，順雪山東稜上攀；沿線經過七卡山莊、雪山東峰、三六九山莊，設備最完善，路線最短，是最熱門的路線。
*聖稜線：沿北稜行進的路線。
*四秀線：自武陵農場起攀桃山，沿北東稜順武陵四秀接上聖稜線，最後至雪山主峰。
*大小劍線：自梨山沿南稜起登，沿途可順訪劍山、大劍山等，因此稱大小劍線，或稱雪劍線、劍山線。
*志佳陽線：自環山從志佳陽大山山腳處起登。在雪東線尚未建設之前，志佳陽線是最短的路線，又因自古即是環山一帶原住民登雪山的路線，因此歷史最為悠久。
*大雪線：自大雪山森林遊樂區的230林道的登山口進入，從中雪山起攀；一路經過大雪山、翠池再接上聖稜線最後抵達雪山，路線最長，水源不足，是難度相當高的路線。

== 主要山莊 ==
===七卡山莊===
位於雪東線，距離武陵農場登山口不到3公里；｢七卡｣的名稱由來，據說是因為此處距離以前的登山口約7公里（7K）；又說是此地古泰雅獵場的音譯<ref>見於七卡山莊前，雪霸國家公園解說牌。</ref>。

===三六九山莊===
位於主峰左近的甘木林山山坡。甘木林山舊測高度為3,690公尺，因此又稱3690峰，山莊因此得名。三六九山莊後方山坡即為黑森林，因林相茂密陽光易遮蔽而得名。野生動物眾多，登山路跡相較不明顯，早年山坡曾發生火災，燒死的冷杉成為一片白木林<ref name=libertytimes/>；此外亦分布著[[巒大花楸|巒大花楸]]。

===翠池山屋===
位於翠池旁，是[[大小劍線|大小劍線]]、[[大雪線|大雪線]]必經的山屋。

===瓢單山屋===
位於志佳陽線，瓢單池旁；隨著瓢單池乾涸，目前已廢棄。

==山難==
===馬來西亞籍鄭姓女登山客山難===
2015年1月30日，16:08時，一11人之登山隊行經雪山線雪山圈谷山域時，一鄭姓登山客 ( 馬來西亞籍，女性，42歲）在雪山線10.7K處不慎跌落山谷，劉姓領隊確認鄭姓登山客頭部破裂腦漿溢出，明顯死亡。<br />
臺中市政府消防局立即派遣4名梨山消防隊員及布農卡里布灣登山隊3人前往協助。
 
1月31日 06:54時，直升機抵達準備吊掛遺體，但現場氣流不穩定直升機無法進行吊掛，固將遺體搬運至三六九山莊附近空曠處。<br />
1月31日 14:25時，由直升機吊掛起遺體。<br />
1月31日 14:52時，降落至東勢河濱公園再轉送殯儀館等侯相驗。
