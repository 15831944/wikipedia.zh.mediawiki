《'''雪之进军'''》（{{lang-ja|雪の進軍}}）是一首旧[[日本军|日本军]]的[[軍歌|軍歌]]，由{{link-ja|永井建子}}作词、作曲。因其带有军歌中少有的反战、厌战情感而在[[太平洋战争|太平洋战争]]爆发后被禁止演唱。

== 背景 ==
这首歌是[[甲午战争|甲午战争]]时期，在[[大日本帝國陸軍|日本陆军]][[第2軍_(日本陸軍)|第二军]]司令部担任军乐队员的永井建子根据自己的亲身经历所作的歌曲<ref>{{Cite book|title=日本の軍歌 国民的音楽の歴史|last=辻田|first=真佐憲|publisher=幻冬舎|series=幻冬舎新書|year=2014-07-30|isbn=9784344983533|location=|pages=76-78|language=ja}}</ref>。在1895年1月进行的[[威海卫之战|威海卫之战]]中，在前线的日本士兵因缺少加热和保暖设备而常被冻伤，同时也缺乏食物、水和燃料等物资，不仅寒冷还要遭受饥饿<ref>{{Cite web|url=http://www.world-anthem.com/sp/march/yukino-shingun.html|title=雪の進軍　歌詞と解説 日本の軍歌・行進曲|accessdate=2019-01-16|work=世界の国歌・行進曲|language=ja|archive-url=https://web.archive.org/web/20170625120610/http://www.world-anthem.com/sp/march/yukino-shingun.html|archive-date=2017-06-25|dead-url=no}}</ref>。《雪之进军》正是反映了这样一种绝望的感受。它是一首反战歌曲，与当时其它的军歌十分不同。虽然《雪之进军》在很长的一段时间内被士兵们所喜爱并传唱。更有[[日本元帅|元帅]][[大山巖|大山巖]]陆军大将（即第二军的司令官）在临终前也听着这首歌的传闻。但它也被认为是“缺乏勇气”的象征。进入[[昭和|昭和]]时代后，歌词的最后一句“反正（上级）也没打算让我活着回去”（「{{lang|ja|どうせ生かして還さぬ積り}}」）被改成了“反正我也没打算活着回去”（「{{lang|ja|どうせ生きては還らぬ積り}}」）以减轻厌战的气氛，在[[太平洋战争|太平洋战争]]期间更是被禁止演唱。二战后，以[[八甲田雪中行軍遭難事件|八甲田雪中行軍遭難事件]]为题材改编的电影《{{link-ja|八甲田山 (电影)|八甲田山 (映画)|八甲田山}}》采用了这首歌作为插曲<ref>{{Cite web|url=https://filmarks.com/movies/4281/reviews/43600201|title=八甲田山のあまのかぐやのネタバレレビュー・内容・結末|date=2018-01-23|accessdate=2019-01-19|author=あまのかぐや|work=Filmarks|language=ja}}</ref>。

== 大众文化 ==
在电视动画《[[少女与战车|少女与战车]]》的第九集和第五部OVA中，秋山优花里與艾爾文无伴奏演唱了这首歌<ref>{{Cite web|url=http://www.lantis.jp/title/mahouritsu/index.html|title=ディープパンツァーCDです！ - TVアニメ『ガールズ＆パンツァー』 - 西住みほ( CV.渕上舞), 武部沙織( CV.茅野愛衣), 五十鈴華( CV.尾崎真実), 秋山優花里( CV.中上育実), 冷泉麻子( CV.井口裕香)|accessdate=2019-01-19|work=Lantis|language=ja}}</ref>（最后一句使用“反正我也没打算活着回去”（「{{lang|ja|どうせ生きては還らぬ積り}}」））。在剧场版中被作为知波单学园的主题曲。
