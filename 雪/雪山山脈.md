'''雪山山脈'''是[[台灣五大山脈|台灣五大山脈]]之一，位於[[中央山脈|中央山脈]]的西北方，東以[[蘭陽溪|蘭陽溪]]斷層谷與[[大甲溪|大甲溪]]上游縱谷（合稱[[匹亞南構造線|匹亞南構造線]]）與中央山脈分界。雪山山脈為台灣最北方的山脈，呈現東北至西南走向，北起新北市貢寮區的[[三貂角|三貂角]]，南迄[[南投縣|南投縣]][[名間鄉|名間鄉]][[濁水溪|濁水溪]]北岸的濁水山，若以此計算，整個山脈長約260公里，寬約28公里。因為位於冬季迎風面水氣較多，也是台灣降雪量最多的地方，故名，山上雪季一般在1月到2月初之間。[[台灣日治時期|日本時代]]稱為'''次高山脈'''，最高點3886米。

== 範圍 ==
雪山山脈北端的首座山峰係荖蘭山，其後經福隆山、隆隆山、灣坑頭山、三方向山、鶯子嶺而抵[[北宜公路|北宜公路]]，北宜公路以南，以[[鴻子山|鴻子山]]、[[三角崙|三角崙]]、[[烘爐地山|烘爐地山]]、[[大礁溪山|大礁溪山]]、[[阿玉山|阿玉山]]、[[紅柴山|紅柴山]]、[[中嶺山|中嶺山]]、[[拳頭母山|拳頭母山]]、[[棲蘭山|棲蘭山]]、[[巴博庫魯山|巴博庫魯山]]而抵[[北部橫貫公路|北部橫貫公路]]。其中[[鶯子嶺|鶯子嶺]]旁的[[鶯子頂山|鶯子頂山]]，是第一座超過1000公尺的山峰，而巴博庫魯山則是首座高於2000公尺的山峰。巴博庫魯山以南，經[[唐穗山|唐穗山]]、[[東保津寒山|東保津寒山]]、[[眉有岩山|眉有岩山]]、[[馬惱山|馬惱山]]、[[邊吉岩山|邊吉岩山]]而抵[[喀拉業山|喀拉業山]]，而喀拉業山是第一座超過3000公尺的山峰。
喀拉業山以南，進入[[雪霸國家公園|雪霸國家公園]]的範圍，高山連綿，依次為[[桃山|桃山]]、[[池有山|池有山]]、[[品田山|品田山]]、[[布秀蘭山|布秀蘭山]]、素[[密達山|密達山]]、[[雪山北峰|雪山北峰]]、[[雪山_(台灣)|雪山]]、[[大劍山|大劍山]]、[[佳陽山|佳陽山]]。
雪山山脈至此為[[大甲溪|大甲溪]]截斷，一般認為與[[劍山|劍山]]對望之[[白姑大山|白姑大山]]，實係雪山山脈之主脊，（另一說認為雪山西稜─[[博可爾山|博可爾山]]、[[火石山|火石山]]、[[頭鷹山|頭鷹山]]、[[大雪山|大雪山]]、[[中雪山|中雪山]]、[[小雪山|小雪山]]諸峰，才是主脊），[[白姑大山|白姑大山]]以南，[[守城大山|守城大山]]俯瞰[[埔里盆地|埔里盆地]]，而終於此。

經過鄉鎮市區為[[新北市|新北市]][[貢寮區|貢寮區]]、[[平溪區|平溪區]]、[[雙溪區|雙溪區]]、[[坪林區|坪林區]]、[[石碇區|石碇區]]、[[烏來區|烏來區]]、[[新店區|新店區]]、[[三峽區|三峽區]]，[[桃園市|桃園市]][[復興區_(台灣)|復興區]]，[[新竹縣|新竹縣]][[尖石鄉|尖石鄉]]、[[五峰鄉_(台灣)|五峰鄉]]，[[苗栗縣|苗栗縣]][[泰安鄉|泰安鄉]]、-{[[南庄鄉_(台灣)|南庄鄉]]}-，[[台中市|台中市]][[和平區_(台灣)|和平區]]、[[新社區|新社區]]、[[東勢區|東勢區]]，[[南投縣|南投縣]][[仁愛鄉|仁愛鄉]]、[[信義鄉_(臺灣)|信義鄉]]、[[國姓鄉|國姓鄉]]、[[埔里鎮|埔里鎮]]，[[宜蘭縣|宜蘭縣]][[礁溪鄉|礁溪鄉]]、[[員山鄉|員山鄉]]、[[大同鄉_(台灣)|大同鄉]]、[[南澳鄉|南澳鄉]]、[[頭城鎮|頭城鎮]]，[[花蓮縣|花蓮縣]][[秀林鄉|秀林鄉]]。

== 地質 ==
由於受到[[大漢溪|大漢溪]]、[[大安溪|大安溪]]及[[大甲溪|大甲溪]]的侵蝕，整個山脈被分割成北部的阿玉山階段山地、中部的雪山地壘以及南部的埔里陷落區三個地理區。[[雪山_(台灣)|雪山]]則恰位於雪山地壘的中心點，由此向外呈放射狀延伸，支脈綿亙北台灣。雪山山脈的地質由始新世至中新世之[[板岩|板岩]]及[[變質砂岩|變質砂岩]]為主之岩系構成，在年代上屬於比較古老的岩層，全山是由赤褐色[[頁岩|頁岩]]、[[砂岩|砂岩]]及[[板岩|板岩]]所構成。

== 古道 ==
雪山山脈也有許多古道，最北邊的隆嶺古道、龜媽坑古道、[[草嶺古道|草嶺古道]]、石坑古道、北宜古道、跑馬古道、桶後越嶺道、哈盆越嶺道、中嶺古道、司馬庫斯古道等，有許多是泰雅族原住民部落聯絡之用，如哈盆古道、中嶺古道等，其他位於雪山山脈的古道，尚有福巴越嶺道、角板山古道、鹿場連越嶺道、薩克亞金古道、北坑溪古道等。

== 百岳名峰 ==
在[[台灣百岳|台灣百岳]]名峰中，雪山山脈中屬於台灣百岳名峰的總計有19座，3000公尺以上的高峰則有54座。
