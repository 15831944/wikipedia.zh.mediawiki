'''雪橇犬'''是指藉著挽具在雪或冰上，拉無輪載具行駛的[[犬|犬隻]]。無法確定此種獨特的運輸方式起源地點，但可能是源於東[[西伯利亞|西伯利亞]]，當地的部落有悠久的冬季遷徙歷史。

== 品種 ==
雖然任何中等體型的[[犬種|犬種]]都可能用於拉[[雪橇|雪橇]]，但數個特定品種的犬通常被用作雪橇犬。純種雪橇犬種的範圍很廣，由廣為人知的[[西伯利亞雪橇犬|西伯利亞雪橇犬]]到極稀有的馬更些河哈士奇，然而馭犬者有長時間使用其他的犬種或混血種作為雪橇犬在掏金熱時期雜種犬隊主導，但也有[[獵狐犬|獵狐犬]]或[[獵鹿犬|獵鹿犬]]的隊伍
今日雪橇競速偏好沒有血統登記的混血阿拉斯加哈士奇，伴隨著多樣的雜種[[德國短毛波音達|德國短毛波音達]]通常被選為作為雜交的基礎。數年前一支標準[[貴賓犬|貴賓犬]]的隊伍參加了[[艾迪塔羅德狗拉雪橇比賽|艾迪塔羅德狗拉雪橇比賽]]。

== 体力要求 ==
雪橇犬被期待在牠們的工作中表現出兩種主要特質：

對於長途的旅程耐力是必須的，一日的距離可能自5到80[[英里|英里]]（8到130[[公里|公里]]）甚至更多。對於在合理的時間到達一定的距離速度也是必要的，趕路的雪橇犬可以達到平均每小時20英哩（30km/h）經過長達25英里（40km）的距離，在更長的距離牠們平均的拖行速度降至每小時10到14英里（16到22km/h），在惡劣的拖曳狀況下雪橇犬仍能維持平均每小時6或7英里（10或11km/h）的速度。

雪橇犬需有強而有力的四肢與健壯的體魄才能拉著各式不同的雪橇，從小至25磅（11[[kg|kg]]）短程競速橇到較大的塑膠底距離賽平底雪橇，雪橇犬也被用來拉貨運撬和毛皮獵人的高緣窄平底橇。

一隊的雪橇犬可能有三到兩打的犬隻。現代犬隊通常是用縱列相套來連結排列犬隻，成對的犬隻排成一列拉。毛皮獵人在深雪的狀況下會使用平底雪橇會將犬隻連結排成單一縱列，極地原住民的犬隊通常是扇狀連結，每隻犬隻有一支牠自己的挽具直接綁在雪橇上。

== 地區 ==
駕馭雪橇犬已經在[[北美|北美]]和[[歐洲|歐洲]]成為一項受歡迎的冬季娛樂和運動，雪橇犬現在甚至在不尋常的地方如[[澳大利亞|澳大利亞]]和[[巴塔哥尼亞|巴塔哥尼亞]]出現。
