'''[[雪霸國家公園|雪霸國家公園]]'''是[[臺灣|臺灣]]第五座[[國家公園|國家公園]]，可溯至[[台灣日治時期|日治時期]]成立之[[次高太魯閣國立公園|次高太魯閣國立公園]]，1991年3月1日公告計畫、1992年7月1日成立管理處，以[[雪山_(台灣)|雪山]]和[[大霸尖山|大霸尖山]]的「雪」、「霸」兩字命名，這兩座山也是該公園主要的保護目標。其涵蓋範圍從北邊的樂山到邊吉岩山之稜線為界，到南邊的宇羅尾山和[[大甲溪|大甲溪]]中游，總面積共有76,850[[公頃|公頃]]，橫跨[[新竹縣|新竹縣]]、[[苗栗縣|苗栗縣]]、[[臺中市|臺中市]]等3縣市。公園內高山林立，峻秀壯觀，並有稀有的[[台灣寬尾鳳蝶|台灣寬尾鳳蝶]]，及瀕臨絕種的[[櫻花鉤吻鮭|櫻花鉤吻鮭]]，是臺灣登山路線最多、最複雜的國家公園。

==歷史==
[[File:TheHolyRidgeline.jpg|thumb]]，從左方的[[雪山_(台灣)|雪山]]（北稜角），到遠處最右的[[大霸尖山|大霸尖山]]]]
臺灣第二高峰的[[雪山_(台灣)|雪山]]於[[台灣日治時期|日治時期]]被命名為次高山，與太魯閣地區早於1937年被劃設為「[[次高太魯閣國立公園|次高太魯閣國立公園]]」，公園總面積達27萬公頃<ref name="ch1">雪霸國家公園計畫（第二次通盤檢討），第一章</ref>。

1979年，[[行政院|行政院]]核定之臺灣地區綜合開發計畫將本區列為國家公園預定地之一，國家公園的規劃自1987年展開。1991年，雪山、[[大霸尖山|大霸尖山]]國家公園計畫通過，之後定名為雪霸國家公園。1992年7月1日，雪霸國家公園計畫公告生效並成立管理處，全區面積約76,850公頃，為臺灣第五個成立之國家公園。管理處最初設於[[臺中縣|臺中縣]][[東勢區|東勢鎮]]，至2001年正式進駐[[苗栗縣|苗栗縣]][[大湖鄉_(台灣)|大湖鄉]]汶水<ref>{{Cite web |url=http://www.spnp.gov.tw/Article.aspx?a=ytj8%2f%2f3cU%2b8%3d |title=汶水遊客中心 |publisher=雪霸國家公園管理處 |accessdate=2013-05-04}}</ref>。未來計畫將[[臺中市|臺中市]][[和平區_(台灣)|和平區]][[大甲溪|大甲溪]]上游溪源區域納入園區範圍，即園區東界向東沿臺中市與[[宜蘭縣|宜蘭縣]]交界，面積擴增兩千多公頃至79,609公頃，進而與[[太魯閣國家公園|太魯閣國家公園]]接壤<ref name="ch2">雪霸國家公園計畫（第二次通盤檢討），第二章</ref>。但此提案尚未獲得共識，於2012年第二次通盤檢討時未獲通過<ref>{{Cite web |url=http://cpabm.cpami.gov.tw/docsrc/DoclistDetailAction.do?seq=1000913D&iFunc=2 |title=101.8.17國家公園計畫委員會第100次委員會議審查結論 |publisher=內政部營建署 |accessdate=2013-05-04}}</ref>。

==範圍==
[[File:Shuepa.jpg|thumb]]
園區邊界自東北角的邊吉岩山起，向南經[[喀拉業山|喀拉業山]]、羅葉尾山、武佐野群山至[[大甲溪|大甲溪]]谷<ref name="ch1"/>。再沿大甲溪轉西而下，經[[德基水庫|德基水庫]]至志樂溪會合處後，沿稜線經宇羅尾山至三錐山。西界自三錐山起向北，經小雪山、南坑山、盡尾山、東洗水山、北坑山、樂山，再往東經觀霧、境界山、南馬洋山至東北角的邊吉岩山。

全區面積共約76,850公頃，依行政區域區分，園區範圍包括[[新竹縣|新竹縣]][[五峰鄉_(台灣)|五峰鄉]]1,328公頃、[[尖石鄉|尖石鄉]]5,416公頃、[[苗栗縣|苗栗縣]][[泰安鄉|泰安鄉]]39,536公頃以及[[臺中市|臺中市]][[和平區_(台灣)|和平區]]30,570公頃<ref name="ch2"/>。

==地形、水文==
[[File:HsuehMountain3.jpg|thumb]]主峰一號、二號圈谷]]
[[File:Hsuehshan_CueiChih_TheGreenJadePond.jpg|thumb]]]]
園區海拔高度自逾700公尺至全臺第二高峰的[[雪山_(台灣)|雪山]]3,886公尺<ref name="ch2"/><ref name="ch3">雪霸國家公園計畫（第二次通盤檢討），第三章</ref>。雪山遺留豐富的[[冰河|冰河]]地形，其中[[冰斗|冰斗]]密集分布在雪山主峰周圍，雪山主峰周圍九個圈谷中，八個是古冰斗遺跡。位在雪山北側坡面的一號圈谷是規模全台最大，二號圈谷則為全台海拔最高。但雪山圈谷是否為冰河遺跡，仍有爭議。[[雪山山脈|雪山山脈]]由傾斜甚緩的[[板岩|板岩]]及[[砂岩|砂岩]]所組成，園區內列名[[台灣百岳列表|臺灣百岳]]的山峰共19座。雪山地壘以雪山為中心，呈放射狀分歧，大致上可以分為[[雪山_(台灣)#地理|六道稜脈]]。

本區河流的主要分水嶺是由雪山主峰主稜和北稜（[[聖稜線|聖稜線]]）組成，即[[大霸尖山|大霸尖山]]往西南方經[[雪山北峰|雪山北峰]]、雪山，到[[大雪山_(台灣)|大雪山]]的稜脊，也大致為縣市行政區界<ref name="ch2"/>。西北為苗栗縣[[大安溪|大安溪]]流域，有全臺海拔最高的水池[[翠池|翠池]]；南部則為臺中市[[大甲溪|大甲溪]]的上游。大霸尖山東北方一帶，即北稜和北東稜間為新竹縣境內的[[淡水河|淡水河]]集水區，園區西北角另有新竹縣[[頭前溪|頭前溪]]的上游集水區。

大甲溪上游的[[七家灣溪|七家灣溪]]、無名溪、及武陵溪的河谷呈峽谷地形，在雪山與羅葉尾山之間的狹長谷地上，切出四階的階狀地，而七家灣溪臨羅葉尾山一側谷壁則呈陡峭狀。

==氣候==
依桑士偉氏氣候分類標準，寒帶重濕氣候區分布於園區之東部及中部，西部則屬溫帶重濕氣候區<ref name="ch2"/>。本區山區多雲霧，溫度較低，以觀霧測站（2,087公尺）為例，1月均溫約為6.8度，7月均溫約為16.6度。雨季大約是在每年的5月到10月，年雨量約在2,000公厘以上，局部地區可達4,000公厘。

==生態==
===植物景觀===
[[File:TheDarkForest_Hsuehshan.jpg|thumb]]]]
本區土地多為天然林，占地58,300公頃，約為國家公園面積的76%<ref name="ch5">雪霸國家公園計畫（第二次通盤檢討），第五章</ref>。人工林占全區面積近18%，其餘為草生地、裸露地、河川地及農墾區。

本區之造林地多為日治時期及戰後的產物，分布於林區海拔700至2,800公尺之間。造林樹種以針葉樹占絕大部分，主要有紅檜、台灣扁柏、台灣雲杉、台灣杉、台灣二葉松、台灣肖楠、柳杉、杉木等，闊葉樹主要有台灣赤楊、櫸樹、竹類等<ref name="ch3"/><ref name="ch5"/>。農墾區主要分布在武陵農場、松柏農場及台7甲線兩側，主要種植蘋果、梨、水蜜桃、梅、李等溫帶果樹及高麗菜等農作物蔬菜。

天然林包括低海拔之闊葉林、中海拔之針闊葉混合林以及高海拔之針葉林<ref name="ch3"/><ref name="ch5"/>。常綠闊葉林主要分布於2,500公尺以下，組成種類較為複雜，大致可依1,500公尺雲霧帶分為以上的櫟林帶（Quercus Zone）及以下的楠櫧林帶（Achilus-Casstanopsis Zone），結構通常有三至四層。落葉闊葉林分布於1,000至2,500公尺之間，多出現於崩坍地或伐木後的岩礫地，主要樹種有台灣赤楊、台灣紅榨槭、尖葉槭、栓皮櫟、[[台灣胡桃|台灣胡桃]]、[[台灣檫樹|台灣檫樹]]等。針葉樹種則有[[台灣鐵杉|台灣鐵杉]]、[[紅檜|紅檜]]、[[台灣扁柏|台灣扁柏]]、[[台灣杉|台灣杉]]、[[巒大杉|巒大杉]]、[[台灣黃杉|台灣黃杉]]、[[台灣華山松|台灣華山松]]、[[台灣二葉松|台灣二葉松]]等多種針葉樹。草本植物則常見[[五節芒|五節芒]]或[[高山芒|高山芒]]。

本區共記錄逾2,000種維管束植物，特有種佔505種，逾總數的四分之一<ref name="ch3"/>。其中稀有植物計有[[台灣曲軸蕨|台灣曲軸蕨]]、[[能高蟹甲草|能高蟹甲草]]、[[奇萊青木香|奇萊青木香]]、[[台灣山芥菜|台灣山芥菜]]、[[波葉櫟|波葉櫟]]、[[森氏毛茛|森氏毛茛]]、[[關山嶺柳|關山嶺柳]]等182種。

===動物生態===
====哺乳動物====
園區的哺乳動物共紀錄58種，大型的有[[台灣黑熊|台灣黑熊]]、[[台灣野山羊|台灣野山羊]]、[[台灣野豬|台灣野豬]]、[[台灣水鹿|台灣水鹿]]；中型的有[[台灣獼猴|台灣獼猴]]、[[山羌|山羌]]、[[豹貓|石虎]]、[[白鼻心|白鼻心]]；小型的則有[[穿山甲|穿山甲]]、[[食蟹獴|食蟹獴]]、[[鼬獾|鼬獾]]、[[黃喉貂|黃喉貂]]、各種鼠類及[[蝙蝠|蝙蝠]]<ref name="ch3"/>。

雪見地區大型動物資源豐富，尤其以[[大安溪|大安溪]]與雪山溪匯流口至北坑溪匯流口之間，尚可見大量而新鮮之台灣野山羊、台灣野豬、山羌、台灣獼猴等動物痕跡，亦可發現目前野外已少見之台灣水鹿足印。北坑溪古道沿線以台灣野豬、山羌、台灣獼猴等動物痕跡較多，但在春夏兩季亦有台灣黑熊在此活動<ref>{{Cite web |url=http://www.spnp.gov.tw/Article.aspx?a=MR6TqHt3Yoc%3d |title=雪霸國家公園大型哺乳動物族群與習性之研究（雪見地區） |author=李玲玲、朱賢斌 |publisher=雪霸國家公園管理處 |date=1995-08}}</ref>。其中，台灣黑熊為野生動物保育法所列瀕臨絕種[[臺灣保育物種列表|保育類動物]]，台灣水鹿在野外的族群量亦極低，台灣野山羊亦屬珍貴稀有保育類動物。

====鳥類====
[[File:Taiwan_Yuhina.jpg|thumb]]為園區內常見的特有種鳥類]]
全區已紀錄有154種鳥類，包括[[台灣山鷓鴣|台灣山鷓鴣]]、[[黑長尾雉|黑長尾雉]]、[[藍腹鷴|藍腹鷴]]、[[台灣藍鵲|台灣藍鵲]]、[[紋翼畫眉|紋翼畫眉]]、[[台灣噪眉|台灣噪眉]]、[[白耳畫眉|白耳畫眉]]、[[黃胸藪眉|黃胸藪眉]]、[[冠羽畫眉|冠羽畫眉]]、[[紫嘯鶇|紫嘯鶇]]、[[台灣叢樹鶯|台灣叢樹鶯]]、[[栗背林鴝|栗背林鴝]]、[[黃山雀|黃山雀]]及[[火冠戴菊鳥|火冠戴菊鳥]]等14種台灣特有種<ref name="ch3"/>。

雪見地區的前五種優勢鳥分別為[[冠羽畫眉|冠羽畫眉]]、[[繡眼畫眉|繡眼畫眉]]、[[山紅頭|山紅頭]]、[[藪鳥|藪鳥]]以及[[白耳畫眉|白耳畫眉]]<ref>{{Cite web |url= http://www.spnp.gov.tw/Article.aspx?a=MR6TqHt3Yoc%3d |title=雪霸國家公園鳥類監測模式之研究─以雪見地區為例 |author=李培芬、柯智仁等人 |publisher=雪霸國家公園管理處 |date=2004-12}}</ref>。鳥類豐富度以位於北坑溪古道的低海拔闊葉林型（29種）、雪見遊憩區所在的中海拔成熟林型（30種），以及海拔約2,000公尺的赤楊－二葉松林型（33種）最高。

====兩生爬蟲類====
已紀錄兩生類16種，以雪見地區分布13種最多，包括[[盤古蟾蜍|盤古蟾蜍]]、[[斯文豪氏赤蛙|斯文豪氏赤蛙]]、[[梭德氏赤蛙|梭德氏赤蛙]]、[[褐樹蛙|褐樹蛙]]、[[面天樹蛙|面天樹蛙]]、[[莫氏樹蛙|莫氏樹蛙]]、[[台北樹蛙|台北樹蛙]]等特有種，觀霧地區與武陵地區另有[[山椒魚|山椒魚]]分布紀錄<ref name="ch3"/><ref>{{Cite web |url= http://www.spnp.gov.tw/Article.aspx?a=MR6TqHt3Yoc%3d |title=雪霸國家公園雪見地區兩生爬蟲類調查及遊憩活動影響評估 |author=吳聲海 |publisher=雪霸國家公園管理處 |date=2010-12}}</ref>。已知爬蟲類39種，[[標蛇|標蛇]]、[[羽鳥氏帶紋赤蛇|羽鳥氏帶紋赤蛇]]、[[斯文豪氏遊蛇|斯文豪氏遊蛇]]、[[台灣鈍頭蛇|台灣鈍頭蛇]]、[[瑪家山龜殼花|瑪家山龜殼花]]、[[短肢攀蜥|短肢攀蜥]]、[[黃口攀蜥|黃口攀蜥]]、[[哈特氏蛇蜥|哈特氏蛇蜥]]與[[台灣蜓蜥|台灣蜓蜥]]等屬於台灣特有種。

====魚類====
[[File:Oncorhynchus_masou_formosanus.jpg|thumb]]]]
在[[大安溪|大安溪]]系、[[大甲溪|大甲溪]]系、[[蘭陽溪|蘭陽溪]]系、[[德基水庫|德基水庫]]等地共調查發現了17種魚類。大甲溪是台灣魚類分布的海拔最高點，一般溪魚分布的上限約為1,800公尺，分布於大甲溪上游的[[櫻花鉤吻鮭|櫻花鉤吻鮭]]則可達1,800公尺以上<ref name="ch3"/>。[[台灣石賓|台灣石賓]]、[[台灣間爬岩鰍|台灣間爬岩鰍]]、櫻花鉤吻鮭、[[台灣纓口鰍|台灣纓口鰍]]、[[明潭吻蝦虎魚|明潭吻蝦虎魚]]、[[短吻紅斑蝦虎魚|短吻紅斑蝦虎魚]]為已紀錄到的台灣特有種。

在雪見地區，[[北坑溪|北坑溪]]在台灣屬高山溪流，許多魚都無法適應這低溫的溪流環境，故魚類種類較少<ref>{{Cite web|url=http://np.cpami.gov.tw/campaign2009/filesys/dl/type1/49/shei-pa_111.pdf|title=園區魚類資源調查─雪見地區|author=陳正平、邵廣昭等人|publisher=雪霸國家公園管理處|date=2004-12|format=PDF}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。經調查北坑溪魚類計有[[短吻紅斑吻蝦虎|短吻紅斑吻蝦虎]]、[[台灣鏟頷魚|台灣鏟頷魚]]等兩種淡水魚類，及[[拉氏清溪蟹|拉氏清溪蟹]]等淡水蟹一種。北坑溪與[[大安溪|大安溪]]匯流口調查到[[台灣鏟頷魚|台灣鏟頷魚]]、[[台灣間爬岩鰍|台灣間爬岩鰍]]、[[台灣石賓|台灣石賓]]等三種淡水魚類。大安溪梅園段調查到台灣間爬岩鰍、台灣石賓、[[粗首鱲|粗首鱲]]等三種淡水魚類。

====昆蟲類====
武陵地區共記錄到111種蝶類，觀霧地區記錄到鱗翅目蝶類8科96種、蛾類17科121種、鞘翅目14科93種<ref name="ch3"/>。合計武陵、觀霧蝶類有170種，包括特有種[[台灣寬尾鳳蝶|台灣寬尾鳳蝶]]。

在雪見地區，發現16目之昆蟲種類，佔全世界昆蟲32目的一半，可見本區內蟲相之複雜度<ref>{{Cite web|url=http://np.cpami.gov.tw/campaign2009/filesys/dl/type1/49/shei-pa_165.pdf|title=雪霸國家公園昆蟲相之調查研究─雪見地區|author=唐立正等人|publisher=雪霸國家公園管理處|date=2002-12|format=PDF|deadurl=yes|archiveurl=https://web.archive.org/web/20160305035113/http://np.cpami.gov.tw/campaign2009/filesys/dl/type1/49/shei-pa_165.pdf|archivedate=2016-03-05}}</ref>。季節性昆蟲種類豐富度調查結果，以夏天昆蟲種類最多，其次為秋季及春季，再其次為冬季。而本區內之保育種僅發現鞘翅目的[[擬食蝸步行蟲|擬食蝸步行蟲]]、[[長角大鍬形蟲|長角大鍬形蟲]]兩種。雖然本區內保育種類不多，但其他之昆蟲稗種類及複雜度相當高，包括腐生肉食性、草食性、寄生性及捕食性種類，在生態階層上甚為完整。

==人文資源==
[[File:Mt_Dabajian-winter_view.jpg|thumb]]與[[小霸尖山|小霸尖山]]]的雪景]]
雪見鄰近地區具有十分豐富薈萃的人文史蹟資源<ref>{{Cite web |url=http://www.spnp.gov.tw/Article.aspx?lang=1&a=mxm97EgBoQM%3D |title=雪見遊憩區人文資源介紹 |publisher=雪霸國家公園管理處 |accessdate=2013-05-04}}</ref>。其原住民文化主要為[[泰雅族|泰雅族]]之北勢群，分布於[[大安溪|大安溪]]中游兩岸。雪見地區瀰漫著濃厚的文化特色，其中以北坑溪古道最具盛名，原為[[大甲溪|大甲溪]]泰雅族原住民遷移遊獵的山徑，古道中泰雅族原住民及[[台灣日治時期|日治時期]]所遺留下來的歷史事件與遺址，是瞭解日治時期竹苗地區[[樟腦|樟腦]]採集與理蕃政策的最佳地點。園區亦有[[賽夏族|賽夏族]]原住民活動區域，位於[[新竹縣|新竹縣]][[五峰鄉_(台灣)|五峰鄉]]<ref>{{Cite web |url=http://www.spnp.gov.tw/Article.aspx?a=oNd0WLhbXko%3d |title=賽夏族概述 |publisher=雪霸國家公園管理處 |accessdate=2013-05-04}}</ref>。

七家灣溪北側山麓緩坡河階上，則存有[[七家灣遺址|七家灣遺址]]，含上、下二個史前時期的文化層，下文化層約始於4,000至4,200年前，結束於2,600至2,800年前，屬於[[新石器時代|新石器時代]]中期到晚期之間<ref name="ch7">雪霸國家公園計畫（第二次通盤檢討），第七章</ref>。上文化層的年代大約距今1,200年開始，可能在距今500年左右結束。志樂溪上游則發現有石屋群遺址，為目前已發現海拔高度最高之遺址（約2,950公尺），較清晰可見結構的石屋共有21座。

==遊憩景點==
===武陵遊憩區===
[[File:武陵四秀.jpg|thumb]]]]
國家公園東南處。由[[中橫公路|中橫公路]]宜蘭支線（[[台7線|台7甲線]]）可達臺中武陵地區入口處<ref name="ch4"/>。[[武陵農場|武陵農場]]南谷46公頃土地為武陵遊憩區所在地。

===觀霧遊憩區===
位於國家公園西北端，面積29公頃。可經新竹122縣道再由大鹿林道進入。早期為[[行政院農業委員會林務局|林務局]]造林、伐林工作站<ref name="ch4">雪霸國家公園計畫（第二次通盤檢討），第四章</ref>。

===雪見遊憩區===
雪見遊憩區位於苗栗縣泰安鄉，居國家公園的西側，唯一聯外道路為司馬限林道<ref name="ch4"/>。該區位於稜線下，地勢平坦，可觀賞[[大霸尖山|大霸尖山]]到[[雪山_(台灣)|雪山]]十餘公里長的[[聖稜線|聖稜線]]雪景。
