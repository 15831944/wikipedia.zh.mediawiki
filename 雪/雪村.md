'''雪村'''（{{bd|1969年|4月23日|catIdx=X雪村}}），本名'''韩健'''，本籍[[吉林|吉林]][[辽源|辽源]]，北京大学德语系毕业，是一位[[中国大陆|中国大陆]]歌手、作曲人、作词人、电影人。

== 互联网造就的歌手 ==

2001年夏天，由于其在[[互联网|互联网]]上的[[Flash|Flash]] [[MV|MV]]《[[东北人都是活雷锋|东北人都是活雷锋]]》一炮走红，并参加了2002年[[中国中央电视台春节联欢晚会|中国中央电视台春节联欢晚会]]。其开创了新独特艺术形式“[[音乐评书|音乐评书]]”，并成为[[中国内地|中国内地]]第一位通过互联网及Flash而非[[传统媒体|传统媒体]]走红的[[艺人|艺人]]。

*《[[美女财福到你家|美女财福到你家]]》（与妻子俞晴合出），2009年3月3日

== 电影 ==

*2004年，参加电影《[[一石二鸟|一石二鸟]]》的演出，饰演片中男主角之一[[卜得辽|卜得辽]]。该片演员还有[[吴宗宪|吴宗宪]]、[[吴孟达|吴孟达]]、[[曾志伟|曾志伟]]、[[林心如|林心如]]等。导演为[[朱延平|朱延平]]。
*2006年，与[[黄渤|黄渤]]、[[范霞|范霞]]、[[曾志伟|曾志伟]]、[[李滨|李滨]]、[[李琦_(演员)|李琦]]等推出电影《[[新街口_(电影)|新街口]]》。雪村自任导演，主演片中男主角[[刘建军|刘建军]]。
*2014年，电影《卧龙岗》。雪村自任导演，並出演片中角色侯睦。

== 家庭 ==

*[[父亲|父亲]][[韩静霆|韩静霆]]，军旅作家，[[文职|文职]][[空军|空军]][[少将|少将]][[军衔|军衔]]。
*母亲[[王作勤|王作勤]]，军旅作家，资深媒体人。
*妻子[[俞晴|俞晴]]，又名莫尼古丽，回族，曾有“中国梦露”之称。2008年结婚。
