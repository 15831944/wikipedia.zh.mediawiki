'''雪'''是[[降水|降水]]形式的一种，是从[[云|云]]中降落的[[晶体|结晶状]][[固体|固体]][[冰|冰]]，常以[[雪花|雪花]]的形式存在。雪是由小的冰颗粒物构成，是一种{{link-en|颗粒材料|granular material}}，它的结构开放，因此显得柔软。因为气温和湿度不同，形成的[[雪花|雪花]]有多种的形状和大小。如果在降落过程中，雪融化后又重新冻结會形成球状降雪，此类降雪有[[霙|霙]]、[[霰|霰]]、[[冰雹|冰雹]]。

== 形成 ==
{{unreferenced-section|time=2015-03-07}}
雪是從大氣中的水蒸氣直接凝結而成。雲中的低溫使得水蒸氣結成冰晶，當氣溫夠低時，冰晶落到地面仍是雪花時，就是下雪了。雪在融化時會吸熱，所以融雪時地面氣溫會比下雪時低。

雪形成的條件是，大氣中需含冷的冰晶核，充分的水氣，以及氣溫在0℃（冰點）以下。
===降雪===
雪降落的过程被称为'''下雪'''或'''降雪'''。雲中溫度低於0℃的許多小雲滴在冰晶上互相碰撞凝結形成雪珠，小雪珠是由許多細白的冰粒聚集而成的。當冷空氣逐漸向前推移，上升氣流減弱，雲中水氣直接在冰晶上凝結成較大的形態，此即我們所見到的雪花。如果溫度接近冰點，則會落下溼雪，形成較大的雪花，特別是無風的時候。大型的星形雪花直徑可達5到7公分。多數的雪花在落下地面的途中會融化成[[雨|雨]]，只有當接近地面的空氣夠冷，才能讓雪花落到地面成雪。

=== 能降雪的地方 ===
降雪一般形成于[[溫帶氣旋|溫帶氣旋]]周围、空气向上运动的区域中。雪可能会伴随着[[鋒_(氣象)#暖鋒|暖锋]]的天气系统里向[[极地|极地]]方向降落。[[雷打雪|雷打雪]]可能发生在气旋的逗号头里和大湖效应的降水带里。在山区，上坡气流在上升过程中在山坡的{{link-en|顺风和逆风|Windward and leeward|迎风}}面达到臨界點，如果这时空气够冷的话，也可能有降雪发生。

[[氣候區|氣候區]]屬中緯度至高緯度（即大約於南回歸線以南／北回歸線以北地區）的地方就會有降雪的機會，如果於低緯度地方中有些地勢高於海拔2000米的高山或高原也有同樣的機會。

海洋氣流也能間接影響該區下雪的機會率，如果在高緯度地區一帶有較多[[暖流|暖流]]支配，會減低該區下雪的機會（例如[[日本|日本]][[本州|本州]]至[[九州|九州]]一帶）。
===大湖效应===
在有著相对溫暖的水体存在的地方（比如湖泊），[[大湖效應|大湖效应降雪]]顯得重要。大湖效應的降雪一般发生在暖湖的下风處、温带气旋后的寒冷的气旋气流中，可以在局部區域造成大量降雪。

===雪花结构===

雪花是在[[雲|雲]]內由微小的[[冰|冰]]晶互撞黏在一起後形成豐富多樣的形狀。沒有兩個雪花是完全相同的，但雪花仍然謹守著最初的冰晶基本的[[六角形|六角形]]對稱標準結構。透過顯微鏡可以看見雪花錯綜複雜的構造大多都是六角形的，而雪花的中心一定呈現出[[六方晶系|對稱的六角形]]<ref>[http://news.xinhuanet.com/forum/2005-01/30/content_2519823.htm 显微镜下的雪花真漂亮] {{webarchive|url=https://web.archive.org/web/20161220074642/http://news.xinhuanet.com/forum/2005-01/30/content_2519823.htm |date=2016-12-20 }}</ref>，它之所以有這樣的形狀，是因為它要在平面上以最有效率的方式佈置，它是[[結晶學|結晶學]]的研究對象之一。

===乾雪===
[[天氣|天氣]]非常寒冷時，雪呈細粉狀，非常散落，稍有風就會被吹走，落在衣服上也不留濕痕。這種雪沒有黏性，因為它們全是由冰晶構成的，裡面沒有水，所以是“乾”的。

===[[雨夹雪|雨夹雪]]===
而在冬末春初的降雪，由于气温较高，这种雪里面含有水滴，会形成“雨夹雪”。

== 雪的融化 ==
当有能量输入的时候，积雪会由[[固态|固态]]转变为其他形态。使积雪融化的[[能量|能量]]可能来自下列途径：[[辐射|辐射]]（太阳发出的短波辐射或者长波热辐射）、[[热|热]][[传导|传导]]（当气温高于0℃时），或者通过落入积雪的，温度超过0℃的雨滴。积雪的融化速度，不仅取决于所输入的能量的多少，也取决于当时的[[气温|气温]]和[[湿度|湿度]]。具体说来，空气越干燥，积雪融化的速度越慢，因为此时雪更容易[[升华|升华]]——即固体不经由液体，直接转变为气体的过程——而升华需要较多的能量，这使得周围的积雪被冷却，从而减缓了融化的速度。

通过“[[湿球温度|湿球温度]]”和“[[露点|露点]]温度”，我们可以界定积雪融化过程的三个状态。“湿球温度”指的是从[[干湿计|干湿计]]的湿球温度计上读出的温度，该温度始终低于大气温度——即使大气[[相对湿度|相对湿度]]为100%的时候也不例外。而“露点温度”指的是，空气中所含的气态水达到[[饱和|饱和]]，从而凝结成液态水所需要的温度。露点温度又总是低于湿球温度。

*当湿球温度低于0℃的时候，积雪升华。这个过程十分缓慢，而此时积雪也保持干燥。在相对湿度小于20%的时候，升华甚至可以在7℃的气温下进行。
*当湿球温度高于0℃，而露点温度低于0℃的时候，积雪[[熔解|熔解]]。这时，固态的雪既转化为气态，同时又转化为液态。
*当露点温度也高于0℃的时候，积雪会[[融解|融解]]，也就是说，积雪只从固态转变为液态。这时积雪融化的速度最快。

举例来说，在相对湿度为50%的时候，当气温低于3.5℃时，积雪升华；当气温介于3.5－10℃之间时，积雪熔解；当气温高于10℃，则积雪融解。

==雪对人类社会的影响==
===农业===
降雪有可能会对农业有利，比如中国就有“瑞雪兆丰年”之说。积雪可以成为[[隔热|隔热层]]，保持土壤的热量，使得[[农作物|农作物]]不被低于[[冰点|冰点]]的天气伤害。<ref name=kepu1>{{cite web |title=瑞雪兆丰年 |url=http://www.kepu.net.cn/gb/earth/icesnow/snow/snw205.html |work=冰雪馆：奇妙的固态降水──雪 |publisher=中国科普博览 |accessdate=2013-12-02}}</ref><ref name=kepu2>{{cite web |title=雪的保温作用 |url=http://www.kepu.net.cn/gb/earth/icesnow/snow/snw206.html |work=冰雪馆：奇妙的固态降水──雪 |publisher=中国科普博览 |accessdate=2013-12-02}}</ref><ref>{{cite web |author=M. Baldwin |url=http://www.newton.dep.anl.gov/askasci/gen01/gen01243.htm |title=How Cold Can Water Get? |publisher=[[Argonne_National_Laboratory|Argonne National Laboratory]] |date=2002-09-08 |accessdate=2009-04-16}}</ref>雪融下去的水留在土壤里，给庄稼积蓄了很多水，对春耕播种以及庄稼的生长发育都很有利。雪中含有很多氮化物，在融雪时被融雪水带到土壤中，成为最好的[[肥料|肥料]]。雪还能消灭害虫，减少虫害的发生。用雪水浇灌作物可以增加产量，提高品质。<ref name=weathercn>{{cite web |title=瑞雪兆丰年的科学依据：降雪的益处 |url=http://www.weather.com.cn/static/html/article/20081117/17934.shtml |publisher=中国天气网 |date=2008-11-17 |accessdate=2013-12-02}}</ref>

==地外行星的雪==
已知在[[火星|火星]]的高纬度地区有很少量的雪。<ref>{{cite web |url=http://news.nationalgeographic.com/news/2009/07/090702-snow-mars-phoenix.html |title="Diamond Dust" Snow Falls Nightly on Mars |author=Anne Minard|date=2009-07-02 |publisher=[[National_Geographic_News|National Geographic News]]}}</ref>在[[土星|土星]]的卫星[[土卫六|土卫六]]上，有可能存在一种由[[碳氫化合物|碳氫化合物]]构成的“雪”。<ref>{{cite web |url=http://www.nasa.gov/mission_pages/cassini/media/cassini-20061212.html |title=Massive Mountain Range Imaged on Saturn's Moon Titan |publisher=[[NASA|NASA]] |author=Carolina Martinez |date=2006-12-12}}</ref>

虽然在[[金星|金星]]上几乎没有水，但也存在一种与雪非常类似的自然现象。[[麥哲倫號金星探測器|麥哲倫號探測器]]在金星最高的山峰上拍到了高反射的物质，它和地球上的雪非常类似。这种物质的形成过程可能与雪的形成类似，但要求的温度很高。因为其高挥发性，它不能在地表凝结，因此成为气体上升到较冷的高地，然后在那里凝结并降落。这种物质的成分还不能确定，有可能是元素[[碲|碲]]或硫化铅（[[方铅矿|方铅矿]]）。<ref>{{cite web
