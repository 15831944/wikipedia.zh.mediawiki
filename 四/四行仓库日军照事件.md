'''四行仓库日军照事件'''发生2017年8月间，四名中国青年身穿二战[[日军|日军]]军服于8月3日晚在上海抗战遗址[[四行仓库|四行仓库]]前合影留念，照片在[[QQ空间|QQ空间]]公布后，引发中国民众舆论反对。[[上海市公安局|上海市公安局]]于23日在[[新浪微博|新浪微博]]公布涉事五名青年的处理结果<ref name="a1"/>。

== 四行仓库 ==

位于[[上海市|上海市]][[静安区|静安区]]的[[四行仓库|四行仓库]]由大陆银行仓库、北四行仓库两座建筑组成。1937年[[淞沪会战|淞沪会战]]中，中国军队接连受挫，而[[谢晋元|谢晋元]]率领国军在[[四行仓库|四行仓库]]进行的[[四行仓库保卫战|四行仓库保卫战]]有利地振奋了国人士气。此处建筑在1985年以“四行仓库抗日纪念地”之名列为[[上海市文物保护单位|上海市文物保护单位]]，是上海重要的抗战遗址。

== 事件经过 ==
{{参见|精神日本人}}

李某、高某、项某、胡某、刘某等人于8月1日前往上海拍照聚会。8月3日晚，其中四人身穿二战日军军服在四行仓库抗战纪念广场（弹孔墙）拍摄照片。数日后，李某在[[QQ空间|QQ空间]]发表照片并配发文字<ref name="a1"/>。参与拍摄的“利冯兹·维森”（是否是李某不详）称拍摄照片为“夜袭四行”，照片内容描绘为“战争末期曾经参与过上海事变的将校故地重游。”第二次上海事变是日本方面对[[淞沪会战|淞沪会战]]的称谓，同时自认为“[[汉奸|汉奸]]”<ref name="a2"/>。

8月7日，有网友在[[新浪微博|新浪微博]]披露此事，转发两张截图，一张为四人合影照片，另一张是“利冯兹·维森”的网文图片<ref name="a2"/>。次日上午，网友表示新浪微博已将披露此事的微博删除<ref>{{Cite web |url=http://www.sohu.com/a/165974987_425345 |title=从美国和德国对纳粹的不同态度看国内的“精日”现象 |accessdate=2017-08-24 |author= |date=2017-08-20 |work= |publisher=搜狐网，来源：察网|language=Zh-hans }}</ref>。

8月8日，四行仓库抗战纪念馆相关负责人马先生接受采访时，表示“作为一个国家级的抗战历史纪念馆，我们对于这一行为表示严厉的谴责和愤慨！<ref name="a2">{{Cite web |url=http://news.163.com/17/0808/13/CRAP8P990001899O.html |title=网传4名男子穿二战日军制服在上海四行仓库拍照|accessdate=2017-08-24 |author= |date=2017-08-08 |work= |publisher=网易，来源：新民网|language=Zh-hans }}</ref>”同日，上海市公安局指示静安分局处理此事。此后，李某五人投案<ref name="a1"/>。

8月23日，静安分局对五人中的李某、高某、项某处以[[行政拘留|行政拘留]]处罚，未成年的胡某、刘某予以教育训诫<ref name="a1"/><ref>{{Cite web |url=http://shanghai.xinmin.cn/xmsq/2017/08/23/31229735.html |title=四行仓库前穿仿制二战日本军服拍照 上海警方行拘3人训诫2人|accessdate=2017-08-24 |author= 记者：戴天骄、潘高峰，编辑：周春晟 |date=2017-08-23 |work= |publisher=新民网|language=Zh-hans }}</ref>。同日，上海市公安局在新浪微博帐号——@警民直通车-上海对此进行了通报<ref name="a1">{{Cite web |url=http://www.guancha.cn/society/2017_08_23_424118.shtml |title=四行仓库前穿仿制二战日本军服拍照 上海警方拘留3名涉案者|accessdate=2017-08-24 |author=  |date=2017-08-23 |work= |publisher=观察者网|language=Zh-hans }}</ref>。
