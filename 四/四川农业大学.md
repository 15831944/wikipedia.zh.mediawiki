'''四川农业大学'''（简称：'''川农'''，英文：{{lang|en|'''Sichuan Agricultural University'''}} 英文缩写：'''SICAU'''）,校区位于[[中国|中国]][[四川省|四川省]][[成都市|成都市]]和[[雅安市|雅安市]]两地，是首批“[[211工程|211工程]]”高校、“[[双一流|双一流]]”高校，入选“[[111计划|111计划]]”、“[[千人计划|千人计划]]”、卓越农林人才教育培养计划、援疆学科建设计划、国家建设高水平大学公派研究生项目、四川“[[2011计划|2011计划]]”、四川省卓越工程师教育培养计划；全国首批高等学校新农村发展研究院试点单位、[[中国乡村振兴战略研究院|中国乡村振兴战略研究院]]联盟发起单位；四川省首批人才优先发展试验区2所试点高校之一、省首批新型职业农民试点3所高校之一；是以生物科技为特色、农业科技为优势，农、理、工、经、管、医、文、教、法、艺多学科协调发展的[[重点大学|重点大学]]；具备[[自主招生|自主招生]]资格。

学校最早起源于1906年在成都创办的四川通省农业学堂，前身为原[[国立四川大学|国立四川大学]]农学院。1956年，四川大学农学院迁往雅安独立办学，定名四川农学院；1985年，学校更名为四川农业大学。现学校发展成为有[[雅安|雅安]]、[[成都|成都]]温江和[[都江堰|都江堰]]3个校区，占地总面积约4500亩，馆藏文献资源625万册（件）。有教职工3400余人，其中教授326人、副教授578人；博士生导师246人、硕士生导师647人；设有学院26个，研究所（中心）15个。有博士后科研流动站7个，博士学位授权一级学科10个，硕士学位授权一级学科17个，本科专业88个；全日制在校生4万余人，其中本科生3.7万余人，研究生近4000人。学校农业科学、植物学与动物学进入[[ESI|ESI]]国际学科排名前1%。2013年，学校国家科技奖获奖总数列全国高校13名，NPI列7，科研机构列14。

===四川通省农业学堂时期（1906年－1912年）===
清末年间，在全国兴学强国的潮流推动下，[[四川布政使|四川布政使]][[许涵度|许涵度]]上奏[[清廷|清廷]]，希望在[[四川|四川]]设立中等农业学堂，1906年（清[[光绪|光绪]]三十二年）四川通省农业学堂在[[成都|成都]]厚载门内宝川局右侧旧仓库内成立，这是四川成立的第一所农业[[学堂|学堂]]，也是四川农业大学的历史源头。

学堂于1906年6月1日先行开学，9月12日正式举行开学典礼，[[四川总督|四川总督]][[锡良|锡良]]在典礼上颁示训词：“蜀中沃野千里，古称陆海。夙以蚕丝之利与世相竞，徒以墨守故法，利寖外溢，故不能不提倡[[农学|农学]]，以为振兴农业之预备”。四川通省农业学堂初设[[农业|农业]]、[[蚕业|蚕业]]、[[林业|林业]]的预科和[[本科|本科]]。

1910年4月，学堂迁往成都东门外[[望江楼|望江楼]]附近的农业试验场修建新校舍。

===四川高等农业学校、四川公立农业专门学校时期（1912年－1927年）===
1912年[[辛亥革命|辛亥革命]]后，经[[四川省军政府|四川省军政府]]批准改名为四川高等农业学校，设大专部和中专部。

1914年，改名为四川公立农业专门学校，增设农本科、蚕本科。

===公立四川大学、四川省立农学院时期（1927年－1935年）===
1927年8月，四川公立农业专门学校与公立法政专门学校、公立工业专门学校、公立外国语专门学校、公立国学专门学校等五所专门学校合并组成[[公立四川大学|公立四川大学]]，公立农专成为公立四川大学农科学院。

1931年，[[四川省政府|四川省政府]]决定，将原公立四川大学的工科、农科改办独立学院，农科学院定名为四川省立农学院。

===四川大学农学院时期（1935年－1956年）===
1935年7月，四川省立农学院和[[重庆大学|重庆大学]]农学院并入[[国立四川大学|国立四川大学]]，组建农学院。此后学院有较大发展。

1952年，全国高等学校[[院系调整|院系调整]]，在部分系调出的同时，先后将[[西南农学院|西南农学院]]、[[云南大学|云南大学]]农学院、川北大学和西昌技艺专科学校的农艺、林业、畜牧、兽医专业调入四川大学农学院，为学院发展进一步奠定了基础。

===四川农学院时期（1956年－1985年）===
1956年，经[[国务院|国务院]]批准，四川大学农学院从成都迁往[[雅安|雅安]]独立建校，命名为四川农学院。1956年9月5日四川农学院在雅安举行成立大会，在校学生1134 人，教职工515人，其中专任教师205人。

1959年，开始招收[[水稻|水稻]]栽培、[[兽医|兽医]]产科、玉米育种、家畜饲养和家禽育种专业三年制硕士研究生。后因“[[文革|文革]]”中断，至1978年恢复。

===四川农业大学时期（1985年－）===
1985年，经[[四川省人民政府|四川省人民政府]]批准，学校更名为四川农业大学。同年开始招收[[博士研究生|博士研究生]]。时任中共中央总书记的[[胡耀邦|胡耀邦]]同志亲笔题写校名。

1991年，批准设立[[博士后|博士后]]科研流动站。

1996年和1998年四川农业大学先后通过国家“[[211工程|211工程]]”建设部门预审和立项审核，学校正式跨入国家重点建设的高校行列。

2001年，进入新世纪，在雅安扩建新校区。

2001年4月，原四川省林业学校整体并入四川农业大学，成为四川农业大学[[都江堰|都江堰]]校区。

2005年，学校在成都温江区开始建设研究生院和科学研究院，学校已有雅安、都江堰和温江三个校区。

2010年10月8日，四川农业大学成都校区正式启用并举行了开学典礼。至此，四川农业大学“一校三区”办学格局形成。

2016年9月，[[成都市|成都市]]人民政府与四川农业大学签署《深入推进全面创新改革共建世界一流农业大学战略合作框架协议》。双方将共同把四川农业大学建设成为世界一流农业大学。

2017年9月，[[教育部|教育部]]、[[财政部|财政部]]、[[国家发展改革委|国家发展改革委]]印发《关于公布世界一流大学和一流学科建设高校及建设学科名单的通知》，公布世界一流大学和一流学科（简称“[[双一流|双一流]]”）建设高校及建设学科名单。四川农业大学入选世界一流学科建设高校。

===迁雅争议===
五十年代，当局出于发展边远地区考虑，不顾大量师生反对，把[[四川大学|四川大学]]农学院从成都迁往川藏交界区的[[雅安|雅安]]独立建院。该地区经济落后，条件艰苦，但老一辈川农人在雅安也取得了一定成绩，对地方经济建设和农业技术的发展做出了一定的贡献。

改革开放后，由于地域局限矛盾越发显著，学校师资招聘和人才培养瓶颈凸显，招生工作也难以展开，校方开始研究迁回成都以获求更多的发展机会。

=== 重返成都 ===
该校先前在成都市温江区建有水稻研究所，1989年曾经提出合并同在[[温江|温江]]的温江农业学校（现成都农业科技职业学院）未果，2005年遂在温江水稻研究所的基础上征地600余亩新建了研究生院和科学研究院。2010年7月中国共产党四川农业大学第九次党代会上，学校做出了在此基础上建设四川农业大学教学科研高端平台－－四川农业大学成都校区的决定。

该校于2011年率先将农学院和风景园林学院迁至成都校区，随后又陆续将资源环境学院、林学院、动物医学院、经济管理学院等核心院系迁至该校区，目前学校优势学科和核心研究院所均已位于成都校区。

成都校区的建设，使阔别成都数十载的四川农业大学回到了建校故地，也使其获得了更宽广的发展舞台。重心搬迁至成都校区后，四川农业大学在师资队伍、招生工作、论文发表、实验室建设等各方面都取得了显著的成果，开启了高速发展的新篇章。

== 基本概况 ==
===三大校区===

四川农业大学拥有三个校区，分别是位于[[雅安市|雅安市]]的雅安校区（校本部）、位于[[成都市|成都市]][[温江区|温江区]]的成都校区和位于成都市[[都江堰市|都江堰市]]的都江堰校区，占地总面积约4500亩。

==== 雅安校区（校本部） ====
地址：雅安市雨城区新康路46号

邮编：625014

概况：四川农业大学雅安校区（校本部）与雅安市区共生，占地面积几乎相当于三分之一个雅安市区。雅安校区由濆江分隔老校区和新校区。雅安校区设有四川农业大学新农村发展研究院和10个二级学院。

雅安校区环境优美，老校区有上百棵历史悠久的法国梧桐树，号称“百万大道”，新校区有一里长的“桂花大道”以及安静优雅的“老板山读书公园”和富有特色的“农牧场”。雅安校区拥有四川乃至全国先进的动物疾病检测仪器、动物疾病研究控制室、动物医院。

==== 成都校区 ====
地址：成都市温江区惠民路211号

邮编：611130

概况：成都校区位于成都市温江区，占地约600亩，该校核心院系和科研机构均位于成都校区。目前设置有1个研究中心、13个研究所和10个学院。

成都校区前身是1988年设立于雅安的水稻研究所。1990年12月，水稻研究所从雅安校区迁至成都。2005年，以水稻研究所为基础，又将都江堰校区的小麦研究所、雅安的校区的玉米研究所、动物营养研究所及园林研究所迁至成都，成立了四川大学成都科学研究院。2008年“5.12”[[汶川大地震|汶川大地震]]后都江堰校区受灾严重，学校根据中央部委批准的灾后重建规划，将都江堰校区重建项目纳入成都校区扩建工程实行异地重建。2010年7月，四川农业大学成都科学研究院撤销，改为四川农业大学成都校区。2010年10月，成都校区正式启用，随后陆续迁入大量核心院所，集中了学校最具优势的学科和研究力量。

==== 都江堰校区 ====
地址：成都市都江堰市建设路288号

邮编：611830

四川农业大学都江堰校区位于世界旅游文化城市[[都江堰|都江堰]]，占地面积1660余亩，校舍建筑面积12余万平方米。其前身是1953年成立的四川省林业学校， 2001年4月整体并入四川农业大学设立都江堰分校，2009年11月更名为四川农业大学都江堰校区（2012年2月原小麦研究所地块整体置入都江堰校区）。

校区现设有4个学院。

===科研教学实力===

==== 学科建设 ====
学校现设有学院24个，研究所(中心)15个。有[[博士后科研流动站|博士后科研流动站]]7个，[[博士|博士]]学位授权一级学科10个、二级学科40个，[[硕士|硕士]]学位授权一级学科15个、二级学科67个，专业学位授予类别（领域）21个，本科专业86个；[[国家重点学科|国家重点学科]]和[[重点培育学科|重点培育学科]]4个，部省重点学科19个。建有教育部农业部重点实验室6个，农业部科学观测实验站3个，教育部农业部工程(技术)研究中心3个，科技部条件平台1个，教育部科技部农业部创新团队9个，以及四川省重点(高校)实验室24个、工程(技术)研究中心7个、社科研究基地4个、国际科技合作基地3个、协同创新中心4个。

2005年学校被确定为全国援疆学科建设计划40所重点高校之一。2012年在[[中华人民共和国教育部|中华人民共和国教育部]]全国高校一级学科评估中，四川农业大学共有6个学科参评，其中，5个学科排名前五，1个学科排名前十，[[风景园林学|风景园林学]]、[[作物学|作物学]]和[[畜牧学|畜牧学]]均排名第四。

==== 师资力量 ====
现有教职工3000余人，其中：教授323人、副教授578人；博士生导师234人、硕士生导师617人；[[中国工程院院士|中国工程院院士]]1人，国家杰出高级专家5人，[[长江学者|长江学者]]特聘教授1人，国家有突出贡献的中青年专家9人，国家杰出青年科学基金获得者1人，国家级教学名师1人，国家百千万人才工程人选11人，国家万人计划青年拔尖人才1人，国家[[973计划|973计划]]首席青年科学家1人，新世纪优秀人才支持计划人选16人，享受[[国务院政府特殊津贴|国务院政府特殊津贴]]专家130人；四川省学术和技术带头人90人，四川省有突出贡献的优秀专家51人，四川省[[千人计划|千人计划]]人选8人，四川省教学名师13人；[[何梁何利基金|何梁何利基金]]获得者4人，[[霍英东教育基金|霍英东教育基金]]获得者13人，四川省科技杰出贡献奖获得者4人。学校是全省首批人才优先发展试验区2所试点高校之一。

==== 科研成果 ====
截至2016年5月，学校获部省级以上科技成果奖励500余项，其中：[[国家技术发明一等奖|国家技术发明一等奖]]2项、二等奖3项，[[国家自然科学二等奖|国家自然科学二等奖]]1项，[[国家科技进步二等奖|国家科技进步二等奖]]16项，四川省科技进步特等奖3项、一等奖53项。70%左右的获奖成果得到推广转化，累计创社会经济效益1000多亿元。

“七五”共获奖82项（次），其中部省级以上奖励69项（次）；“八五”共获奖87项（次），其中部省级以上奖励58项（次）；“九五”共获奖67项（次），其中部省级以上奖励48项（次）；“十五”共获奖55项（次），其中部省级以上奖励42项（次）。78年以来，全校共取得科研成果534项，获奖441项（次），国家和部省级奖励351项（次）。其中国家发明一等奖2项、二等奖3项、三等奖1项；国家自然科学奖二等奖1项；国家科技进步二等奖13项、三等奖5项；全国科学大会奖5项；部省级特等奖4项、一等奖36项（次）、二等奖71项（次）、三等奖152项（次）。经过32年努力，四川农业大学“育种攻关课题组”培育了一大批水稻、小麦、玉米和薯类等作物的创新性品种及其配套栽培技术，使四川省的小麦、玉米和水稻的平均产量比攻关课题开始前的1979年分别增产了98%、82%和55%。

自1999年开展全国优秀博士学位论文评选以来，截至2013年该校共有5篇博士学位论文入选全国优秀博士学位论文，5篇获得提名。该校入选全国优秀博士学位论文的总数在全国农林类高校排位第5名，四川排名第三，仅次于[[四川大学|四川大学]]、[[西南交通大学|西南交通大学]]。

====校徽====
四川农业大学校徽由中英文校名、中心花状主体图案、1906、现第一行政楼及书本环绕的穗状图案构成。

四川农业大学校名由[[江泽民|江泽民]]同志亲笔题写；中心花状主体图案由花蕊和叶片构成，花蕊由古象形字“羊”变形而成，表示动物；围绕花蕊的叶片表示植物。“1906”表明学校的前身是成立于1906年的四川通省农业学堂，中心下方的现第一行政楼为原[[西康省|西康省]]人民政府办公大楼、学校1956年迁雅安独立建校后的代表性建筑。整个校徽图案主题鲜明、内涵丰富、协调美观，体现了学校悠久的办学历史、鲜明的办学特色，象征学校和谐、进取与勃勃生机。

截至2015年7月，学校与美国、英国、法国、德国、加拿大、俄罗斯、日本、韩国和香港、台湾等20多个国家和地区的30余所科研院所建立了交流与合作关系。与日本[[东京大学|东京大学]]、[[北海道大学|北海道大学]]、[[广岛大学|广岛大学]]，英国[[诺丁汉大学|诺丁汉大学]]，美国[[普渡大学|普渡大学]]、[[路易斯安那州立大学|路易斯安那州立大学]]、[[伊利诺伊大学|伊利诺伊大学]]等高校有着广泛的科研合作。学校是全国首批有条件接受留学生的200所高校之一，还是国家建设高水平大学公派研究生项目签约高校。
