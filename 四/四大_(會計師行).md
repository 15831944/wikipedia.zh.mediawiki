'''四大'''（{{lang-en|Big Four}}），又譯為'''四大會計師事務所'''、'''四大審計公司'''，合稱'''EDPK'''，是指4家最大的[[跨國公司|國際]]{{link-en|專業服務|Professional services}}網路，由[[戰後|二戰後]]的八大會計師事務所（{{lang|en|Big Eight}}）多次[[合併|合併]]而成。他們原本都是[[會計師事務所|會計師事務所]]，但因業務擴及{{link-en|鑑證服務|Assurance services}}、[[稅|稅務]]、[[管理顧問|管理顧問]]、諮詢、[[精算學|精算]]、企業財務、法律服務等整合性的{{link-en|企業服務|Corporate services|企業管理服務}}，因而以「專業服務」稱之。有別於一般的[[跨國公司|跨國公司]]，其於世界各地以類似[[加盟|加盟]]的形式設立據點，各地據點在經營上擁有一定的自主權。

四大為許多國際知名的[[上市公司|上市公司]]與[[私人公司|私人公司]]處理絕大多數的財務[[審計|審計]]，在全球專業服務業界形成[[寡佔|寡佔]]。報導指出99%的[[富時100指數|富時100指數]]企業、以及96%的[[富時250指數|富時250指數]]企業的審計業務均由四大負責<ref name="FTSE">{{cite news|url=http://online.wsj.com/article/SB10001424052748703806304576232231353594682.html |title=U.K. Auditors Criticized on Bank Crisis |date=2011-03-30 |author=Mario Christodoulou |publisher=Wall Street Journal }}</ref>。

== 六大（1989-1998） ==
隨著競爭激烈，Arthur '''Young'''與'''Ernst''' & Whinney合併而成Ernst & Young（[[安永會計師事務所|安永會計師事務所]]）；Deloitte、Haskins & Sells及Touche Ross合併成Deloitte Touche Tohmatsu（[[德勤會計師行|德勤會計師行]]），自此大幅拉開與二線（2nd Tier）會計師行的距離：

# 安達信會計師事務所
# 安永會計師事務所
# 容永道會計師事務所
# 德勤會計師行
# 畢馬威會計師事務所
# 普華會計師事務所

== 五大（1998-2002） ==
1998年7月容永道會計師事務所及普華會計師事務所所合併成[[普華永道會計師事務所|普華永道會計師事務所]]：

# 安達信會計師事務所
# 安永會計師事務所
# 德勤會計師行
# 畢馬威會計師事務所
# 普華永道聯合會計師事務所

== 四大（2002-迄今） ==
2001年底，[[安達信會計師事務所|安達信會計師事務所]]因[[安隆|安隆]]案而崩潰，業務分別被其餘大所購併（例如美國部份為安永所收購），但安達信管理顧問業務因在崩潰前分拆出Andersen Consulting（現稱[[埃森哲|埃森哲]]）而得以保留，成為商業顧問龍頭之一。

# 普華永道聯合會計師事務所
# 德勤聯合會計師事務所
# 安永聯合會計師事務所
# 畢馬威聯合會計師事務所
