'''四號突擊炮'''（'''Sturmgeschütz IV'''）是[[德國|德國]]在[[第二次世界大戰|第二次世界大戰]]所研制的裝甲戰鬥車輛。它以[[四號坦克|四號坦克]]的底盤為基礎而制造。德國最初打算以此作為向步兵和坦克提供近距離火力支援的武器，后被作為[[驅逐戰車|驅逐戰車]]使用。

== 發展 ==
四號突擊炮是為了滿足德國軍方對突擊炮的需求而由[[克虜伯|克虜伯]]公司所研制的。由於[[三號坦克|三號坦克]]並非由該公司生產，克虜伯公司便以自己生產的[[四號坦克|四號坦克]]的底盤為基礎，再加上作出輕微修改過的[[三號突擊炮|三號突擊炮]]的炮座，以此發展出四號突擊炮。克虜伯公司從1943年12月至1945年5月一共生產1139輛四號突擊炮。（其中31輛是由戰損的四號戰車改裝的）在實戰中絕大多數四號突擊炮都被部署到步兵師。

與同樣衍生自四號戰車底盤的[[四號驅逐戰車|四號驅逐戰車]]相比，四號突擊炮的火力雖然相等，但其防護力較差。但是由於其前方裝甲的面積與主砲搭載位置的關係，相較於頭重腳輕的四號驅逐戰車重量平衡度好得多，操控性與機動力也較佳。

四號突擊砲使用了四號戰車的車身，全長6.7m，重25.9吨。由於使用相同的上半車身結構，外觀上並不太容易分辨三號突擊砲與四號突擊砲，一眼就可以看出的主要差別在於路輪數。(四號：每邊八個；三號：每邊六個)

== 戰後 ==
目前已知基本保持完整的四號突擊炮一共有三輛，两辆位于波蘭境內，一辆在拉脱维亚。


波蘭境內的兩輛，一輛由im博物館所保存，是以三號突擊炮和四號戰車零件復原的，另一輛完好的四號突擊炮正在波茲南軍事博物館進行修復工作，处于可用状态<ref>{{cite web |url=http://moto.onet.pl/1569257,1,300-litrow-na-100-kilometrow,artykul.html?node=2 |title=存档副本 |accessdate=2014-08-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20120226034845/http://moto.onet.pl/1569257,1,300-litrow-na-100-kilometrow,artykul.html?node=2 |archivedate=2012-02-26 }}</ref>。


2011年10月，拉脱维亚境內的一輛没有发动机和传动系統的四號突擊炮在[[互联网|互联网]]上开价425000欧元出售。
