'''四大恶霸地主'''或'''四大地主'''，是[[中华人民共和国|中华人民共和国]]广为人知的四个[[阶级成份|地主]]反面典型，[[刘文彩|刘文彩]]、黄世仁、南霸天和[[周扒皮|周扒皮]]。四人中，除刘文彩外，三人均是[[文学形象|文学形象]]。

== 背景 ==
[[File:1950-08-黄世仁被批斗.png|thumb]]》中[[批斗|批斗]]黄世仁的情节]]
1945年4月[[中国共产党|中国共产党]]政权控制的[[延安|延安]]首演歌剧《[[白毛女|白毛女]]》，剧中黄世仁一角成为四大地主中最早出现的人物形象。当时[[抗日战争|抗日战争]]即将结束。中共“[[减租减息|减租减息]]”和“团结地主”的政策被“土地改革”和“打倒地主阶级”所取代<ref>{{cite web|url=http://news.qq.com/a/20110217/000729_2.htm|title=白毛女：革命话语压倒启蒙话语的典范 |publisher=腾讯网，来源：《随笔》|author=景凯旋|date=2011年2月17日 |accessdate=2012-11-11 |language=Zh-hans}}</ref>

== 刘文彩 ==
{{main|刘文彩|收租院}}
刘文彩是四川[[军阀|军阀]]、[[中华人民共和国|中华人民共和国]]林业部部长[[刘文辉|刘文辉]]之兄，其形象经文学改造后在在中华人民共和国的文化大革命中被宣传为“无恶不作”的大地主。1965年6月，创作的泥塑《[[收租院|收租院]]》以刘文彩庄园收租院为题材，描绘了刘文彩盘剥农民的情况。

== 黄世仁 ==
{{main|白毛女}}
1945年[[延安|延安]][[魯迅藝術學院|鲁迅艺术学院]][[集体创作|集体创作]]的歌剧《[[白毛女|白毛女]]》中的地主。

== 南霸天 ==
{{main|红色娘子军}}

1961年电影《红色娘子军》中的恶霸地主。

== 周扒皮 ==
{{main|周扒皮}}

周扒皮是小说《[[高玉宝_(小说)|高玉宝]]》中一个剥削、虐待雇农的周姓地主的[[绰号|绰号]]。

== 评论 ==
* 有人认为，这些文学形象在现实中都有原型，而原型主人公没有一个是恶人，这些文学形象创作出来都是为政治所服务的。刘文彩“恶霸地主”的事迹都是捏造的，为的是把中国农村的地主阶层丑恶化，以达到消灭中国农民传统精英阶层的目的。{{fact}}
