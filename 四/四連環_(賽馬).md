'''四連環'''是一个[[賽馬|賽馬]]術語，是[[賽馬|賽馬]][[博彩|博彩]]的其中一個投注種類，香港稱為「四連環」而新加坡稱為「四連」，澳門賽馬會暫不接受此投注種類。投注者需選中一場賽事中的前四名參賽馬，毋須順序，便獲派彩。

== 投注金額 ==
'''香港'''

每注最低投注金額為港幣$10。若每票投注總額達港幣$100或以上，可以靈活玩方式投注，而每注最低投注金額為港幣$1。

'''新加坡'''

新加坡最少每注2新加坡元 
== 投注方法 ==
{|class="wikitable" width="70%"
|- style="background:#C0C0C0;" align=center
|width=10%|'''投-{注}-種類'''||width=60%|'''投-{注}-方法'''
|-
|align=center|「複式互串」投-{注}-||假設您的心水馬為1號、2號、3號、8號、 9號，複式互串會將所選馬匹組成所有可能的組合，即分別為1+2+3+8、1+2+3+9、1+2+8+9 、1+3+8+9 及 2+3+8+9，總共五注，好處是提高命中機會。

|-

|align=center|「胆拖」投-{注}-||當您對一匹馬有很大信心能跑入前四名，您可選擇以該匹馬作「馬胆」，其他選擇則作為「配腳」。
假設重心馬為1號，以1號胆拖2號、3號、8號、9號，即分別為1+2+3+8、1+2+3+9、1+2+8+9 及 1+3+8+9 ，總共四注，只要其中一個馬匹組合同時跑入前四名，便為中彩，相比起「複式互串」好處是更能節省注項。
|}

== 中獎方法 ==
選中一場賽事中的前四名參賽馬，毋須順序，便獲派彩。派彩將會按照該馬匹組合的最後四連環賠率計算。
例如最後四連環賠率為180倍，以每注投注金額$10計算，派彩為$10 x 180倍=$1,800

== 有關香港賽馬四連環的派彩紀錄 == 
最冷紀錄於2008年產生，首四名馬匹分別為第一名正本仁心、第二名玩具神話、第三名魄力之城及第四名喜富龍, 派$541,411。<ref>東網派彩紀錄 http://racing.on.cc/racing/sta/current/rjstai0001x0.html</ref>

最熱紀錄於2012年產生，首四名馬匹分別為第一名崇山寶、第二名奪目商人、第三名赤膽及第四名川河快駒, 派$16。<ref>東網派彩紀錄 http://racing.on.cc/racing/sta/current/rjstai0001x0.html</ref>
