'''四子王旗'''（{{Mong|m=ᠳᠥᠷᠪᠡᠳ <br />ᠬᠣᠰᠢᠭᠤ|k=Дөрвөд хошуу|Ppp=Dörbed qosiɣu}}）位于[[中華人民共和國|中華人民共和國]][[内蒙古自治區|内蒙古自治區]]中部[[乌兰察布市|乌兰察布市]]的西北部。其西北與[[蒙古国|蒙古国]]接壤，其东北是[[锡林郭勒盟|锡林郭勒盟]]，东南是乌兰察布市的[[察右后旗|察右后旗]]、[[察右后旗|察右后旗]]、[[卓资县|卓资县]]，正南是内蒙古自治区的省会[[呼和浩特市|呼和浩特市]]，西南属[[包头市|包头市]]。

蒙语名字为杜尔伯特旗，得名于[[诺颜|诺颜]][[泰奥特根|泰奥特根]]的四个儿子。

== 行政區劃 ==
四子王旗轄下有1个 [[牧场|牧场]]、5個[[鎮|鎮]]、2個[[鄉|鄉]]、4個[[蘇木_(行政區劃)|蘇木]]。<ref>{{PRC admin/ref sgc|15/09/29/000/000}}</ref>
* 牧场：乌兰牧场；
* 鎮：烏蘭花鎮、庫倫圖鎮、吉生太镇、供济堂镇、白音朝克图镇；
* 鄉：東八號鄉、忽雞圖鄉；
* 蘇木：查干补力格苏木、脑木更苏木、红格尔苏木、江岸苏木。

== 航天器着陆场 ==
四子王旗是目前为止所有十一艘[[神舟飞船|神舟飞船]]返回艙的著陸處，此外还有实践十号[[中国返回式卫星|返回式卫星]]以及[[探月工程三期再入返回飞行试验器|探月工程三期再入返回飞行试验器]]（嫦娥五号探路星）在本着陆场着陆。
