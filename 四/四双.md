'''四双'''是[[篮球|篮球]]的术语，指一场比赛中球员的个人表现在以下五项中有四項达到两位数：[[得分|得分]]、[[篮板球|篮板]]、[[助攻|助攻]]、[[抢断|抢断]]和[[盖帽|盖帽]]。在台湾则以麻将术语称之为“大四喜”。也有以[[失誤_(籃球)|失誤]]等负面数据或者非正式数据来作为四项指标之一的四双，但一般来说此种另类四双不会被正式记载。NBA联赛中，得到正式认证的四双共有4次。

== NBA联赛中的四双成绩 ==

*[[内特·瑟蒙德|内特·瑟蒙德]]，1974年10月18日，[[芝加哥公牛|芝加哥公牛]]对[[亚特兰大鹰|亚特兰大鹰]]：22分，10篮板，13助攻，12盖帽。 
*[[埃尔文·罗伯特森|埃尔文·罗伯特森]]，1986年2月18日，[[圣安东尼奥马刺|圣安东尼奥马刺]]对[[菲尼克斯太阳|菲尼克斯太阳]]：20分，11篮板，10助攻，10抢断。 
*[[阿基姆·奥拉朱旺|阿基姆·奥拉朱旺]]：1990年3月29日，[[休斯敦火箭|休斯敦火箭]]对[[密尔沃基雄鹿|密尔沃基雄鹿]]：18分，16篮板，10助攻，11盖帽。 
*[[大卫·罗宾逊|大卫·罗宾逊]]：1994年2月17日，[[圣安东尼奥马刺|圣安东尼奥马刺]]对[[底特律活塞|底特律活塞]]：34分，10篮板，10助攻，10盖帽。

== CBA联赛中的四双成绩 ==
* [[胡雪峰|胡雪峰]]，2004年12月9日，[[江苏南钢|江苏南钢]]主场对[[云南红河奔牛篮球俱乐部|云南红河]]：16分、10篮板、12助攻，10抢断。比赛结果：131：102。
* 克里斯·威廉姆森，2009年12月25日，[[青岛双星|青岛双星]]主场对[[东莞烈豹篮球俱乐部|东莞马可波罗]]：15分、11篮板、11助攻、11抢断。比赛结果：122：103。

== NBA联赛中比较知名的接近四双的成绩 == 
 

*[[拉里·伯德|拉里·伯德]]，1985年2月18日，[[波士顿凯尔特人|波士顿凯尔特人]]对[[犹他爵士|犹他爵士]]：30分，12篮板，10助攻，9抢断。值得一提的是，这场比赛伯德仅打了3节便达到准四双数据，球队也早早领先20分以上，在场边记录员和主教练[[K.C·琼斯|K.C·琼斯]]的提醒甚至是催促中，伯德毅然拒绝了达成NBA史上第二次四双的机会下场休息，此举被认为是球员尊重对手和体育精神的体现。
*[[阿基姆·奥拉朱旺|阿基姆·奥拉朱旺]]，1990年3月3日，[[休斯敦火箭|休斯敦火箭]]对[[金州勇士|金州勇士]]：29分，18篮板，9助攻，11盖帽。 
*[[克莱德·德雷克斯勒|克莱德·德雷克斯勒]]，1996年11月2日，[[休斯敦火箭|休斯敦火箭]]对[[萨克拉门托国王|萨克拉门托国王]]：25分，10篮板，9助攻，10抢断。
*[[沙奎尔·奥尼尔|沙奎尔·奥尼尔]]，2001年6月8日，[[洛杉矶湖人|洛杉矶湖人]]对[[费城76人|费城76人]]（2001年总决赛第2场）：28分，20个篮板，9助攻，8盖帽。
*[[蒂姆·邓肯|蒂姆·邓肯]]，2003年6月15日，[[圣安东尼奥马刺|圣安东尼奥马刺]]对[[新泽西网|新泽西网]]（2003年总决赛第6场）：21分，20篮板，10助攻，8盖帽。
*[[麥可·卡特-威廉斯|麥可·卡特-威廉斯]]，2013年10月30日，[[費城七六人|費城七六人]]对[[邁阿密熱火|邁阿密熱火]]（2013年NBA 開幕戰）：22分，7篮板，12助攻，9抢断。(此場比賽為他生涯首戰)
*[[安東尼·戴維斯|安東尼·戴維斯]]，2015年3月16日，[[紐奧良鵜鶘|紐奧良鵜鶘]]对[[丹佛金塊|丹佛金塊]]：36分，14篮板，7助攻，9盖帽。
*[[里奇·盧比奧|里奇·盧比奧]]，2015年12月17日，[[紐約尼克|紐約尼克]]对[[明尼蘇達灰狼|明尼蘇達灰狼]]：9分，10篮板，12助攻，8抢断。
*[[德雷蒙德·格林|德雷蒙德·格林]]，2017年2月10日，[[金州勇士|金州勇士]]对[[曼菲斯灰熊|曼菲斯灰熊]]：4分，12籃板，10助攻，10抄截，其另有5阻攻的優異表現。


<br />
因此类准四双数据标准不一，而且场次不在少数，在此仅列举几次较为经典和著名的场次。

== 可能的四双成绩 ==  
需要指出的是NBA直到1973-74赛季才开始有抢断和盖帽这两项数据的统计，所以从1973年以后才有了四双的说法。有人猜测[[奥斯卡·罗伯特森|奥斯卡·罗伯特森]]、[[比尔·拉塞尔|比尔·拉塞尔]]和[[威尔特·张伯伦|威尔特·张伯伦]]也曾经拿到过四双甚至是五双，因为他们在盖帽或者抢断上都非常出众。但是因为没有足够的影音资料，所以统计也无从谈起。
