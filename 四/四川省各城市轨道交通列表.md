'''四川省各城市轨道交通列表'''，列明[[四川省|四川省]]各城市正在运营/建设/规划中的轨道交通系统。所有数据截止2017年7月。

== 成都市 ==

=== 成都市區 ===

==== 地鐵 ====
{{Main|成都地铁}}
成都市已经建成1、2、3、4、7和10号线。至2020年，成都市轨道交通里程将达到600千米。

==== 有轨电车 ====
{{Main|成都有轨电车}}
成都市规划了有轨电车蓉1线、蓉2线两条线路。其中，有轨电车蓉2线已进入施工阶段，将于2018年建成通车。
==== 市域铁路 ====
{{Main|成都市域铁路}}
成都市现有[[成灌铁路|成灌铁路]]、成灌铁路彭州支线、成灌铁路离堆支线三条市域铁路。未来，成都市将开通[[成蒲铁路|成蒲铁路]]并对成都市环线铁路进行公交化改造。

=== 新津 ===
==== 有轨电车 ====
新津现代有轨电车R1线已建成通车，R2线正在进行施工前期准备工作。

=== 都江堰市 ===
==== 市域铁路 ====
{{Main|成灌铁路}}
成灌鐵路[[离堆公园站|離堆公園]]支線因全線處於地下，且“站房布局似地铁　有3-4个出入口”，因此新聞媒體報導是，時常將此線稱作“都江堰地鐵1號線”，實際上此線並不屬於[[城市軌道交通|城市軌道交通]]。<ref>{{Cite news|url=http://sc.sina.com.cn/city/xwgz/gdxw/2013-07-23/09459438_2.html|title=都江堰“地铁1号线”今日开通(2)|author=|date=|work=|accessdate=}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>
==== 有轨电车 ====
都江堰市将与[[万达集团|万达集团]]合作建设M-TR旅游客运专线，该线路为现代有轨电车，以地面线路为主，全长二十公里左右。<ref>{{Cite news|url=http://cd.bendibao.com/news/2016422/84343.shtm|title=都江堰M-TR旅游客运专线2016年8月开工|author=|date=|work=|accessdate=}}</ref>

== 自貢市 ==
[[自贡市|自贡市]]以工代赈办公室主任、市政协委员彭吉泽在自貢市“[[两会|兩會]]”中提出3份提案，其中两份重点关注自贡城市交通状况。首次提出在自贡城区规划建设高架路网，缓解城市交通拥堵，并顺应形势，充分到重庆调研后大胆规划城市轻轨系统。<ref>{{Cite web|url=http://news.163.com/16/0218/05/BG37MQM300014AED.html|title=顺应丘陵城市气质 自贡建两条轻轨(图)_网易新闻中心|accessdate=2017-07-31|last=网易|work=news.163.com}}</ref>

== 瀘州市 ==

泸州市规划建设5条轻轨线路，其中4号线为现代有轨电车线路，将于2017年年内开工。

== 德陽市 ==
{{Main|德阳轨道交通}}

== 綿陽市 ==
{{Main|绵阳轨道交通}}
绵阳市提出，将在2019年开工2条轻轨线路，远期将形成5条线路组成的轨道交通网络。

== 广元市 ==
[[广元市|广元市]]提出预留用地下廊道空间，准备在[[十三五|十三五]]期间建设两条轻轨线路。

== 內江市 ==
內江市軌道交通线网规划方案已通过专家组评审，基本上确定了内江城市轨道交通的线网（1條環線3條放射線）的分布格局。<ref>{{Cite web|url=http://scnews.newssc.org/system/20150122/000531385.html|title=内江拟建4条轨道交通 1条环线 3条从中心城区发散 四川新闻网_四川新闻|accessdate=2017-07-31|work=scnews.newssc.org}}</ref>

== 樂山市 ==
[[乐山市|樂山市]]政府在2015年與中國鐵路通信信號股份有限公司簽訂《四川省樂山市現代有軌電車項目合作框架》，擬在樂山市建設3條有軌電車線路。<ref name=":0" />
{{Main|乐山有轨电车}}
乐山市2017年提出了修建从乐山大佛景区到峨眉山景区的悬挂式[[单轨|单轨]]旅游客运专线的计划。

== 眉山市 ==
据四川发布微信报道，3月30日上午，四川省交投集团与[[眉山市|眉山市]]政府签订战略合作协议。记者从签约仪式现场获悉，眉山市将加快推进成都轨道交通11号线东坡支线、[[成都地铁5号线|成都轨道交通5号线]]眉山延伸线、仁寿至黑龙滩轨道交通快线等3条轨道交通建设。此外，近期还将重点实施成乐高速扩容改造以及中法农业科技园彭山园区建设。<ref>{{Cite news|url=http://cd.bendibao.com/news/2017331/90329.shtm|title=成都地铁11号线、5号线将直通眉山|author=|date=|work=|accessdate=}}</ref>
