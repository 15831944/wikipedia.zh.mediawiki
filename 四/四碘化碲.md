'''四碘化碲'''（化学式：TeI<sub>4</sub>）是一种[[无机化合物|无机化合物]]，具有和[[四氯化碲|TeCl<sub>4</sub>]]和[[四溴化碲|TeBr<sub>4</sub>]]的四聚体固体形式不同的四聚体结构。<ref name = "Greenwood">{{Greenwood&Earnshaw}}</ref>在TeI<sub>4</sub>中，Te原子的配位结构为八面体，八面体的顶点被共用。<ref name = "Greenwood"/>

TeI<sub>4</sub>可以由单质制备，或者通过[[碲|碲]]和[[碘甲烷|碘甲烷]]的反应得到。<ref name = "Greenwood"/>在气态TeI<sub>4</sub>中会发生解离：<ref name = "Wiberg">''Inorganic Chemistry'',Egon Wiberg, Arnold Frederick Holleman Elsevier 2001 ISBN 0-12-352651-5</ref>

:TeI<sub>4</sub> → TeI<sub>2</sub> + I<sub>2</sub>

TeI<sub>4</sub>熔化时是一个[[导体|导体]]，这是由于解离出的TeI<sub>3</sub><sup>+</sup>和I<sup>−</sup>。有[[配位|配位]]能力的溶剂，如[[乙腈|乙腈]]，在溶解该物质时，溶液具有导电能力：<ref name = "Wiberg"/>
:TeI<sub>4</sub> + 2 CH<sub>3</sub>CN → (CH<sub>3</sub>CN)<sub>2</sub>TeI<sub>3</sub><sup>+</sup> + I<sup>−</sup>
