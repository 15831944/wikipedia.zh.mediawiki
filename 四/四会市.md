'''四会市'''（官方音译：-{'''Sihui'''}-，传统外文：-{'''Szewui'''}-），位于[[中华人民共和国|中华人民共和国]][[廣東省|广东省]]西部，是[[肇庆市|肇庆市]]代管的一个[[县级市|县级市]]，位于[[珠江|珠江]]支流[[綏江|绥江]]下游。

四会是廣東著名僑鄉，屬[[珠三角经济圈|珠三角经济圈]]，有「鱼米之鄉」、「柑桔之鄉」的美稱。
四會的河流及空氣汙染嚴重，環境衞生不佳。在2017，市委市政府決定將四會建為「全國文明城市」，改善以上問題，並取得成效。

== 历史<ref>{{Cite book|title=《人文四会》|last=陈|first=以良|publisher=广东旅游出版社|year=|isbn=978-7-80766-343-0|location=|pages=20页}}</ref> ==
拥有2000余年历史的四会古邑，其行政地域虽几经变更，但在岭南地区的政治与经济重镇地位是不容忽视的。

四会是[[岭南|岭南]]境内最早设置的县份之一。公元前219年，秦始皇派50万大军南征岭南，秦始皇三十三年（公元前214年）平定岭南，设置桂林、南海、象郡。四会属桂林郡。

汉高祖四年（公元前203年），赵佗据岭南立南越国，四会县为其属地。西汉元鼎六年（公元前111年），汉武帝灭南越国，在岭南地复置南海、苍梧、郁林、合浦、交趾、九真、日南、朱崖、儋耳九郡。

三国时期，四会县为吴国属地，{{cn|time=2018-03-11|黄武五年（226年）将四会县东南部（即今三水、鹤山、新会、江门、开平、台山、斗门等县市部分地区）划出置[[平夷县|平夷县]]（后改为[[新夷县|新夷县]]、新会郡、新会县）。}}
注： 據《 新會鄕土誌》壹書，鶴山，新會，江門，開平，臺山等地在秦，漢時期屬於番禺縣）. 

隋开皇九年（589年），撤销新招、乐成、化成三县并入四会县。隋大业三年(607年)，复置南海郡。四会县与化蒙县同属于南海郡。

唐武德五年（622年），以四会、化蒙等县地置南绥州。并复置新招、化注（原称“乐成”）、

化穆（原称“化成”）三县，同隶于州。唐龙朔元年（661年），佛教禅宗六祖惠能来到四会，在四会扶卢山（大南山北麓）中隐匿15年。

北宋熙宁六年（1073年），四会割属端州（后改肇庆府）。

明朝嘉靖三十八年（1559年），王钫及巡按御使徐辑采纳了肇庆知府卢磷的建议，割四会县的太平、橄榄、大圃、永义四都，设广宁县。四会县隶属肇庆府。

民国九年（1920年）秋，取消县以下的都、铺行政区划制，改用区、乡（镇）建制。四会隶属粤海道、西北绥靖区，第三专员公署。因其水陆交通发达，三水、高要、广宁、怀集、清远、等周边地区商人往来频繁、商贸活跃，四会成为有名的商埠，有“小广州”之称。

1952年5月，广宁县与四会县合并，称“广四县”。1954年6月。两县分开，将原来属广宁县管辖的黄田、石狗、江谷、江林等乡划归四会管辖。1958年10月，广宁、四会两县再次合并，仍称“广四县”。至1961年4月，两县再次分开。

1993年11月25日，经国务院批准四会撤县设市。

== 地理 ==
四会市境内河流众多，地势西高东低。中西部为[[山地|山地]]和[[丘陵|丘陵]]地形，南部和东部则属绥江、北江的冲积平原。全市年平均气温21℃，降水量1821毫米。

== 交通 ==
[[File:四会汽车站.JPG|thumb]]

广茂铁路穿越东南部的大沙镇。[[广贺高速公路|广贺高速公路]]是首条途经四会的高速公路，于2010年12月10日通车，[[江肇高速公路|江肇高速公路]]亦于2010年12月31日通车。编号为“G94”的[[肇花高速公路|肇花高速公路]]于2016底通车。早期的交通建设分别有连接三水和四会的[[北江马房大桥|北江马房大桥]]（[[321国道|321国道]]一部分），在1983年竣工通车，以及城区内最早修建的绥江大桥，目前是在四会其中一条最繁忙的跨江桥梁。近年来大桥因为高负荷交通，而产生了结构问题。四会市公路局决定在2009年限行大桥（20吨重，2.4米宽）。并在2012年开工建设新桥，2016年完工。

市内主要的陆路交通是客运汽车，设有四会粤运汽车站。另外设有大约10条市区公交线路，亦有线路前往各乡镇。正在建设的[[广佛肇城际轨道交通|广佛肇城际轨道交通]]在四会市[[大沙镇|大沙镇]]设[[大沙站|大沙站]]。水路有位于马房村附近的[[四会港|四会港]]。

== 经济 ==
农业主产[[水稻|水稻]]、[[蔬菜|蔬菜]]、[[花生|花生]]等，盛产[[柑橘|柑橘]]和[[橙子|橙子]]。工业则以[[轻工业|轻工业]]为主，有[[食品|食品]]、[[饮料|饮料]]、[[玉|玉]]器、[[摩托车|摩托车]]、[[纸|纸]]制造等行业。境内还有储量较大的[[石膏|石膏]]矿区。

[[南江工业园|南江工业园]]座落于四会市的南端，位于[[北江|北江]]、[[绥江|绥江]]和[[西江|西江]]三江汇合处，地理位置十分优越。东与三水接壤，距[[广州|广州]]、[[佛山|佛山]]只有30多公里；西距国家级历史旅游城市——[[肇庆市|肇庆市]]40公里；离'''四会市'''城区和[[贞山|贞山]]旅游区10公里。工业园所处地势高而平展，不受洪涝自然灾害影响。

近幾年，四會市委、市政府實施"工業立市"戰略，走工業園模式發展，創辦市级[[工業園|工業園]]13個。

==特产==
四會種植[[柑桔|柑桔]]已有四百多年，曾被列為貢品，享譽京華。四會仁稔、銀稔、人面果，2001年四會市被[[国家林业局|国家林业局]]評為“中国柑桔之乡”，四會[[沙糖桔|沙糖桔]]被評為中國名優果品。

雖然四會並不出產玉石，但卻是廣東其中一個最大的玉器集散地{{Better source|reason=此處原本引用360doc為來源，為不可靠來源|date=2019年4月}}，因為四會雲集了各地的玉器技師，成為了廣東的玉器加工中心<ref>[http://www.suncrystal88.com/show.aspx?id=487 四會玉器批發市場、天光墟遊記（上）]</ref>。
