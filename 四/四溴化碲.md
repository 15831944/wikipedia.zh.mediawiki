'''四溴化碲'''（化学式：TeBr<sub>4</sub>）是一种[[无机化合物|无机化合物]]，它和[[四氯化碲|TeCl<sub>4</sub>]]有着相似结构。<ref name = "Wiberg">''Inorganic Chemistry'',Egon Wiberg, Arnold Frederick Holleman Elsevier 2001 ISBN 0-12-352651-5</ref>它可以通过[[溴|溴]]和[[碲|碲]]的反应制备。<ref>{{Greenwood&Earnshaw}}</ref>在气态TeBr<sub>4</sub>中，按下式解离：<ref name = "Wiberg"/>
:TeBr<sub>4</sub> <nowiki>→</nowiki> TeBr<sub>2</sub> + Br<sub>2</sub>
TeBr<sub>4</sub>熔化时是一个导体，这是因为它解离出TeBr<sub>3</sub><sup>+</sup>和Br<sup><nowiki>−</nowiki></sup>两种离子。
它的[[苯|苯]]或[[甲苯|甲苯]]溶液不导电，且TeBr<sub>4</sub>以四聚体（Te<sub>4</sub>Br<sub>16</sub>）的形式呈现出来<ref name = "Wiberg"/>。在有[[配位|配位]]能力的溶剂，如[[乙腈|乙腈]]中，可以形成导电的溶液：
:TeBr<sub>4</sub> + 2CH<sub>3</sub>CN <nowiki>→</nowiki> (CH<sub>3</sub>CN)<sub>2</sub>TeBr<sub>3</sub><sup>+</sup> + Br<sup>−</sup>
