'''四季青站'''是[[杭州地铁7号线|杭州地铁7号线]]及[[杭州地铁9号线|杭州地铁9号线]]的地下同台换乘站，位于解放东路与秋涛路交叉口以东，秋石高架桥东侧。<ref>{{cite web |title=定了！杭州地铁5号线二期、9号线一期站点公布！快看哪站离你家近 |url=http://n.cztv.com/news/12674668.html |website=新蓝网}}</ref>

== 结构 ==
四季青站沿解放东路东西向布置。为地下两层平行双岛式四线车站，五柱六跨（局部四柱五跨）钢筋混凝土箱型框架结构，有效站台宽度14米、长140米，9号线与7号线双岛四线同台换乘，9号线在内，7号线在外，两个站台宽均为14米。9号线站前设单渡线、站后设交叉渡线，折返线段长250.7m，宽度为10.7m（局部加宽）。车站主体外包总长为242.4m（不包括9号线折返线段），站台结构总宽度为45.1m。车站总建筑面积34354平方米。<ref>{{cite web |title=杭州地铁9号线一期工程土建施工SG9-1标段土建施工SG9-1标段（四季青站、四季青站～中央公园站区间、中央公园站）招标公告 |url=http://www.gonglutielu.com/zbgg/57694.jhtml |website=中国铁公基信息网}}</ref>
