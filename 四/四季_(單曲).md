《'''四季（Four Seasons）'''》是[[韓國|韓國]][[女歌手|女歌手]][[金泰耎|太妍]]（[[韓國|韓國]][[女子組合|女子組合]][[少女時代|少女時代]]）的數位單曲。由[[SM娛樂|SM娛樂]]製作，{{link-ko|IRIVER|아이리버}}發行。2019年3月22日下午六點單曲同名主打歌MV先行公開，3月24日下午六點各大數位音樂網站公開線上音源。

== 背景和發行 ==
[[SM娛樂|SM娛樂]]在2019年3月18日在太妍個人官方網站公佈單曲預告照，同日公佈單曲主打歌《'''四季（Four Seasons）'''》及另一首收錄曲《'''Blue'''》。<ref>{{cite web | language=ko |url=http://www.osen.co.kr/article/G1111101547 |title=소녀시대 태연, 24일 새 싱글 '사계' 발표..귀호강 예고[공식입장] |publisher=osen |date=2019-03-18}}</ref><ref>{{cite web | language=中文 |url=https://www.koreastardaily.com/tc/news/114932 |title=聆聽太妍的〈四季〉景致　新單曲週日發行！ |publisher=韓星網 |date=2019-03-18}}</ref>3月19日起將依次公佈四支《四季（Four Seasons）》MV預告片。3月20日[[SM娛樂|SM娛樂]]表示該數位單曲的首唱舞台將在安可場《's...one TAEYEON CONCERT》演唱會中公開。<ref>{{cite web | language=中文 |url=https://www.koreastardaily.com/tc/news/115009 |title=太妍新單曲〈四季〉預告連發　華麗配樂＋感性畫面構成迷人氛圍|publisher=韓星網 |date=2019-03-20}}</ref>

== 成績 ==
3月24日同名主打歌《'''四季（Four Seasons）'''》在晚間11點於韓國Melon、Genie、Bugs、Mnet、Naver、Soribada、Flo、ollehmusic等八大音源網站獲得一位並達成「Real Time All-Kill」優異成績。韓國音源網站Melon首日{{NoteTag|name=SP1|1=首日意思為發行日24日下午六點至25日下午六點。}}收聽數為102萬1019人突破個人歌曲《I》收聽人數紀錄{{NoteTag|name=SP2|1=《I》Melon首日收聽人數為96萬3980人}}，亦是SM娛樂旗下藝人第一首收聽數破百萬的歌曲和藝人。<ref>{{cite web | language=中文 |url=https://www.koreastardaily.com/tc/news/115145 |title=不愧是太妍！新單曲甫發行便橫掃排行榜首 |publisher=韓星網 |date=2019-03-24}}</ref><ref>{{cite web | language=ko |url=http://www.osen.co.kr/article/G1111106675 |title=태연 '사계', 8개 음원차트 1위 올킬...'명불허전 음원퀸' [공식입장]|publisher=osen |date=2019-03-25}}</ref>收錄曲《'''Blue'''》在全球iTunes單曲榜獲得11國家/地區一位。<ref>{{cite web | language=ko |url=http://www.xportsnews.com/?ac=article_view&entry_id=1092885 |title=태연, 3일째 음원차트 올킬→中차트·아이튠즈 11개 지역 1위|publisher=xportsnews |date=2019-03-26}}</ref>
