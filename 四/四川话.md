'''四川话'''（[[四川话拼音|四川话拼音]]：{{lang|la|Si<sup>4</sup>chuan<sup>1</sup>hua<sup>4</sup>}}；[[国际音标|国际音标]]：{{lang|la|{{IPA|[sɨ˨˩˧tsʰwan˥˥xwa˨˩˧]}}}}），是流行于[[中華人民共和國|中国]][[四川省|四川省]]、[[重庆市|重庆市]]（[[巴蜀|巴蜀地区]]）及周邊省份臨近地區的主要汉语言<ref name="《四川方言与巴蜀文化》">{{cite book |author= 崔荣昌 |title=《四川方言与巴蜀文化》 |year=1996 |publisher=四川大学出版社 |chapter =第三章：四川的官话 |isbn=7561412967}}</ref>，包括[[汉语|汉语]][[西南官话|西南官话]]中源自[[古巴蜀语|古巴蜀语]]的[[成渝片|成渝片]]及[[灌赤片|灌赤片]]<ref name="《雪球——汉民族的人类学分析》">{{cite book |author= 徐杰舜 |title=《雪球——汉民族的人类学分析》 |year=1999 |publisher=上海人民出版社 |chapter =第八编第四章：西南汉族的方言 |isbn=9787208030596}}</ref>。四川话目前约有1亿2千万的使用者，数量次于[[日语|日语]]，多于[[德语|德语]]<ref name="ethnologue1">{{cite web | url=http://www.ethnologue.com/statistics/size | title=Ethnologue | publisher=SIL International}}</ref><ref name=Encarta>{{cite web |url=http://uk.encarta.msn.com/encnet/features/dictionary/dictionaryhome.aspx |title=Encarta Dictionary |accessdate=2009-11-16 |publisher=Microsoft Encarta 2006 |deadurl=yes |archiveurl=https://web.archive.org/web/20091028112912/http://uk.encarta.msn.com/encnet/features/dictionary/DictionaryHome.aspx |archivedate=2009-10-28 }}</ref>。现今四川话形成于元末明初“[[湖广填四川|湖广填四川]]”的大移民运动时期，是由[[明朝|明]]之前流行于四川地区的[[巴蜀语|巴蜀语]]和来自[[湖广|湖广]]、[[广东|广东]]、[[江西|江西]]等地的各地移民方言逐渐演变融合而形成的。四川话内部[[互通度|互通度]]较高，各方言区交流并无障碍<ref> 《巴蜀方言浅说》</ref>。

语言学家认为，四川话并不等同于“四川方言”<ref name="《四川方言与巴蜀文化》"/>，因为除四川话外，四川省境内较为常见的汉语言还有[[客家话|土广东话]]（属客家话）和[[湘语|老湖广话]]（属湘语）等，其都以[[方言岛|方言岛]]的形式分布于四川各地<ref name="《移民与四川人“打乡谈”》">{{Cite journal|author=严奇岩 |title=《移民与四川人“打乡谈”》|journal=成都大学学报（社科版）|date=2007年第4期}}</ref>。同时，四川省境内还拥有大量[[藏语|藏语]]、[[彝语|彝语]]、[[羌语|羌语]]的使用者。但由于四川话在四川、重庆具有强势地位，是事实上的[[通用语|通用语]]，一般而言，土广东话与老湖广话的使用者同时也是四川话的使用者<ref name="《移民与四川人“打乡谈”》"/>，而部分[[藏族|藏族]]、[[彝族|彝族]]、[[羌族|羌族]]地区，特别是[[西昌|西昌]]、[[康定市|康定]]、[[雅江|雅江]]、[[昭觉|昭觉]]、[[马尔康|马尔康]]、[[松潘|松潘]]、[[丹巴|丹巴]]等州府、县府所在地，也通用口音接近成渝片的四川话<ref name="《四川方言与巴蜀文化》"/><ref>田畅（2009年7月29日），《在四川灾区支教的300多个日夜》，鞍山日报</ref>。

== 名称 ==
四川话，学术上为与[[客家话|土广东话]]、[[湘语|老湖广话]]等其它四川方言区别，也称作“四川官话”。在四川民间各地旧因“[[湖广填四川|湖广填四川]]”的历史事件，而通称其为“湖广话”，但现已不常用。民间亦简称四川话为“川语”<ref>徐如宜（2009年3月8日），《在蜀相戀 與妻獨處川語傳情》，聯合報</ref><ref>{{cite web |url=http://ent.cctv.com/20081223/106244.shtml |title=《川语无敌》即将麻辣上市 就是要你攒劲笑过年 |accessdate=2010-02-21|work=王燕 |publisher=央视网}}</ref>、“川话”<ref>{{cite web |url=http://news.163.com/09/0407/11/569UFSIK000120GU.html |title=川话韩剧网上走红 |accessdate=2010-02-21|work=四川在线 |publisher=网易新闻}}</ref><ref>乔雪阳（2009年9月9日），《罗小刚开讲“川话脱口秀” 观众来了七八十 喉咙都快说哑了》，成都商报</ref>。四川的[[客家人|客家人]]由于四川话的通用语地位和其处于被四川话包围的土广东话方言岛的缘故，而称四川话为“街腔”（[[威远|威远]]、[[隆昌县|隆昌]]）、“四外话”（[[西昌|西昌]]）、“四邻话”或“四里话”（[[仪陇|仪陇]]）。四川的[[湘语|老湖广话]]使用者则称四川话为“贵州腔”（[[乐至|乐至]]）、“四平话”、“平话”或“四平腔”（[[达县|达县]]）。而四川的[[羌族|羌族]]称四川话为“汉话”<ref name="《华夏边缘：历史记忆与族群认同》"/>。除此之外四川各地方志中还曾以“蜀语”、“蜀方言”、“普通话”等来称呼四川话<ref name="《四川方言与巴蜀文化》"/>。而学术界往往称明之前流行于四川地区的语言为“蜀语”或“巴蜀语”，以区别于明清后由巴蜀语和各地移民方言融合而形成的四川话<ref>汪启明（2009年第1期），《“蜀语”名义阐微》，云南师范大学学报（哲学社会科学版）</ref>。

== 分布与分区 ==
四川话的使用人口主要分布于[[四川盆地|四川盆地]]一带，覆盖了除部分非汉族聚居区外的整个[[四川省|四川省]]和[[重庆市|重庆市]]，以及[[陕西|陕西]]、[[湖北|湖北]]、[[湖南|湖南]]、[[云南|云南]]、[[贵州|贵州]]等省与川渝毗邻的部分县市，使用人口约1.2亿，是使用人口最多的汉语分支之一<ref name="《雪球——汉民族的人类学分析》"/>。四川话内部互通度较高，词汇、语法、声韵等方面都较为一致，一般根据古入声的今读情况分为如下图所示的4个方言区：入声归阳平的[[成渝片|成渝片]]、入声保留的[[灌赤片|灌赤片]][[岷江小片|岷江小片]]、入声归去声的[[灌赤片|灌赤片]][[仁富小片|仁富小片]]和入声归阴平的[[灌赤片|灌赤片]][[雅棉小片|雅棉小片]]<ref name="《四川方言与巴蜀文化》"/><ref name="《汉语方言学》">{{cite book |author= 翟时雨 |title=《汉语方言学》 |year=2003 |publisher=西南师范大学出版社 |chapter =中篇第四节：四川话的分区 |isbn=9787562129424}}</ref>。
 
从保持中古时期巴蜀语特点的多少来看，[[川北|川北]]、[[川东|川东]]的成渝片由于地理区位等原因，是明清以来[[湖广|湖广]]等地移民的主要聚居区，受外来语言影响较大，保持古音较少，是语言的发展区，为新派四川话；而[[川西|川西]]、[[川南|川南]]的灌赤片由于四川土著居民存留较多、接受外来移民相对较少，保持古音较多，是语言的稳定区，为老派四川话<ref name="《汉语方言学》"/><ref name="《西南历史文化地理》"/>。灌赤片大部分区域保留了入声，同时从古代巴蜀地区“入声似去”的特点来看，古入声归去声仍可视为保留本地区中古音的一个表现；灌赤片同时还保留了较多的中古时期的巴蜀语词汇<ref name="《西南历史文化地理》">{{cite book |author= 蓝勇 |title=《西南历史文化地理》 |year=1997 |publisher=西南师范大学出版社 |chapter =第十三章：历史时期西南综合文化区的划分 |isbn=7-5621-1603-2}}</ref>。

== 历史 ==

四川话发源于上古时期非[[汉族|汉族]]语言的[[古蜀语|古蜀语]]和[[古巴语|古巴语]]，迄今四川话的原始层中仍然保留了“坝”（平地）、“姐”（母亲）、“养”（您）等来自上古时期古蜀语和古巴语的词汇<ref name="《四川方言与巴蜀文化》第一章第三节">{{cite book |author= 崔荣昌 |title=《四川方言与巴蜀文化》 |year=1996 |publisher=四川大学出版社 |chapter =第一章第三节：巴蜀的语言 |isbn=7561412967}}</ref>。之后四川话便随巴蜀地区的历史进程和移民更替而不断地发展变化，先是[[秦朝|秦]]灭[[巴国|巴]][[古蜀|蜀]]后，巴蜀地区逐步形成属于汉语族但独具特色的[[巴蜀语|巴蜀语]]。其后在明清时期，由于大量来自[[湖广|湖广]]等地的移民进入四川，巴蜀语同各地移民方言演变融合而最终形成了现今的四川话<ref name="《雪球——汉民族的人类学分析》"/>。

=== 上古时期 ===
上古时期，四川盆地中存在[[蜀族|蜀族]]与[[巴族|巴族]]两个非华夏族的民族以及[[古蜀|蜀国]]与[[巴国|巴国]]两个独立的国家，其不仅拥有各自独立的语言：古蜀语和古巴语，还拥有被概称为“[[巴蜀图语|巴蜀图语]]”的独立的文字系统<ref name="《巴族与蜀族文字考辨》">{{cite book |author= 钱玉趾 |title=《三星堆与巴蜀文化》 |year=1993 |publisher=巴蜀书社 |chapter =巴族与蜀族文字考辨 |isbn=9787805235264}}</ref>。古巴蜀语与当时的华夏语截然不同，与现今[[羌语|羌语]]、[[嘉戎语|嘉戎语]]、[[彝语|彝语]]、[[纳西语|纳西语]]和[[土家语|土家语]]等语言有着密切的关系<ref name="《四川方言与巴蜀文化》第一章第三节"/><ref name="《四川通史》第一册">{{cite book |author= 陈世权等 |title=《四川通史》第一册 |year=1996 |publisher=四川大学出版社 |isbn=7561409737}}</ref>。前316年，[[秦国|秦国]]相继灭掉巴蜀两国，逐步将中原华夏族的制度、政令推行到[[巴蜀|巴蜀地区]]，并开始大量的向巴蜀地区移民，巴蜀地区出现了古巴蜀语和华夏语并存并用、相互渗透的局面。之后随着秦汉时期大量的中原汉人迁徙入川，与四川地区原有的蜀族、巴族融合形成了具有巴蜀特色的[[巴蜀民系|汉族族群]]，并且在[[西汉|西汉]]末年形成了具有较为统一特色的[[巴蜀语|巴蜀语]]。《文选》卷四载[[左思|左思]]《蜀都赋》刘逵注引《地理志》中记载：“蜀人始通中国，言语颇与华同”，同时根据[[扬雄|扬雄]]《[[輶軒使者絕代語釋別國方言|方言]]》中的记载，当时梁益地区（即巴蜀地区）的方言与秦晋方言已经较为接近<ref name="《西南历史文化地理》第三章">{{cite book |author= 蓝勇 |title=《西南历史文化地理》 |year=1997 |publisher=西南师范大学出版社 |chapter =第三章：历史时期西南汉族移民与文化氛围的嬗变 |isbn=7-5621-1603-2}}</ref>，表明此时的巴蜀语已经属于[[汉语|汉语]]的一个分支。

西汉末形成的巴蜀语作为[[上古汉语|上古时期汉语族]]的一个独立分支，其特点主要体现在声调与词汇两方面。在声调方面，[[陆法言|陆法言]]《切韵序》有“秦陇则去声为入，梁益则平声似去”的记载。同时[[黄鉴|黄鉴]]《杨文公谈苑》中称：“今之姓胥、姓雍者，皆平声。春秋胥臣、汉雍齿旨是也。蜀中作上声、去声呼之，盖蜀人率以平为去。”这说明此时巴蜀语声调具有自身特色。而词汇方面，此时巴蜀语的一个显著特征是吸收了来自非汉语的古巴蜀语的词汇。扬雄《方言》以及其他一些历史文献中都记载了大量四川地区的特殊词汇，来自古巴蜀语的词汇包括“坝”（平地）、“姐”（母亲）、“不律”（笔）、“养”（您）、“曲鲙”（蚯蚓）、“阿婸”（我）等，其中“坝”、“姐”、“养”、“曲鲙”至今仍保存于四川话之中<ref name="《四川方言与巴蜀文化》第一章第三节"/>。

中古时期，巴蜀地区经济文化发展达到鼎盛，作为一个独立的语言区，巴蜀语继续得到发展<ref name="《方言与中国文化》">{{cite book |author= 周振鹤，游汝杰 |title=《方言与中国文化》 |year=1986 |publisher=上海人民出版社 |isbn=9787208062078}}</ref>，此时的巴蜀语独立性很强，与四川地区以外的语言较难沟通。[[宋朝|宋]][[范成大|范成大]]旅居蜀地时在《石湖诗集》卷十七《丙申元日安福寺礼塔》诗注中有如下记载：“蜀人乡音极难解，其为京洛音，輒谓之‘虏语’。或是僭伪时以中国自居，循习至今不改也，既又讳之，改作‘鲁语’。”从中可以看出巴蜀与中原语音完全两异。以[[宋代|宋代]]巴蜀语为例，首先巴蜀语韵部与[[宋代音系|宋代通语]]比较，韵部的分野或归字不同，如阳声韵寒先部的“言”字读人真文部、药铎部的“祈”读与屋烛部的“秃”相同等。其次，巴蜀语介音有合口化倾向，在一些字音上三个阳声韵尾相混。同时，巴蜀语声纽保留了诸如“[[古无舌上|古无舌上]]”、“[[照二归精|照二归精]]”等的古音遗迹。此外，声调方面，巴蜀语在平声字与上去声字归派与通语也有较大不同，如通语音归平声的“青雍句”在巴蜀语中“青”归入上声，“雍句”归去声<ref>刘晓南（2008年第2期），《从历史文献看宋代四川方言》，四川大学学报（哲学社会科学版）</ref>。

中古时期，巴蜀语也拥有大量特有词汇，如“波”（老人）、“偏涷雨”（夏日暴雨）、“百丈”（牵船绳）、“溉”（江边道路）、“块”（坟墓）、“秃”（砍）等<ref name="《西南历史文化地理》"/>，其中部分仍然存留于今天的四川话中（如下表所示）。将文献中记录的上古、中古时期巴蜀语特有词在现今四川话中的存留情况进行统计，结果如右表所示<ref name="《四川方言中的古巴蜀土著语研究》">向学春（2008年第5期），《四川方言中的古巴蜀土著语研究》，重庆三峡学院学报</ref>。从中可以看出，上古文献中收录的巴蜀语特有词汇约有一成保留于今四川话中，同时中古文献中收录的巴蜀语特有词汇有较为可观的三成得到保留。这表明虽然在近古时期四川地区人口构成发生[[湖广填四川|剧变]]，但现今四川话仍然与上古及中古时期的巴蜀语有一定程度的传承关系，中上古巴蜀语是现今四川话形成和发展的重要基础<ref name="《四川方言中的古巴蜀土著语研究》"/>。

=== 近古时期 ===
宋末[[元朝|元]]初，长达52年的[[四川抗元战争|四川抗元战争]]使自[[唐朝|唐]]以来经济文化高度发达的巴蜀地区遭受重创，人口锐减，经济凋敝。[[南宋|南宋]][[嘉定_(年号)|嘉定]]十六年（1223年），四川地区有259万户（约1200万人，占南宋全国的23.2%）；而到元政权占领四川之后的[[至元_(忽必烈)|至元]]十九年（1281年）便只余12万户（约60万人，占元全国的0.7%），减少了约95%<ref name="《宋代四川经济述论》">{{cite book |author= 贾大泉 |title=《宋代四川经济述论》 |year=1985 |publisher=四川省社会科学院出版社 |OCLC=567922027}}</ref>。元末[[明朝|明]]初，在这种低人口背景下，来自[[湖广|湖广]]、[[广东|广东]]、[[江西|江西]]等地的移民开始陆续进入四川，形成了四川历史上的第一次“[[湖广填四川|湖广填四川]]”大移民运动。中古巴蜀语和各地移民方言融合演变，现今四川话由此开始逐步形成。明代四川土著居民人数较移民仍占优势，四川话也仍以宋元巴蜀语为基础继续发展<ref name="《四川方音在宋代以后的发展》">彭金祥（2006年第3期），《四川方音在宋代以后的发展》，乐山师范学院学报</ref>，但已经与现今[[川西|川西]]、[[川南|川南]]一带的老派四川话（灌赤片）较为接近<ref name="《西南历史文化地理》"/>。

根据明末清初成书的《蜀语》以及其它文献的记载，明代四川话音韵上的主要特征包括：一、[[入声|入声]]仍然保留，独立成调，但[[川东|川东]]、[[川北|川北]]地区的入声尾（喉塞音）已经式微。二、平声未分阴阳，因而明代四川话中仅有[[声调|平上去入]]四个声调<ref name="《四川方音在宋代以后的发展》"/>。三、[-m]韵尾已经消失，并入[-n]韵尾<ref name="《〈蜀语〉所反映的明代四川方音的两个特征》"/>。这三个特点与此时的[[官话|官话]]迥异，首先，依据《中原音韵》记载，北方官话早在元代就已经入声消失，平分阴阳；同时北方官话中[-m]韵尾并未消失，因此四川话与北方官话并没有同步发展<ref name="《〈蜀语〉所反映的明代四川方音的两个特征》">黄尚军（1995年第4期），《〈蜀语〉所反映的明代四川方音的两个特征》，方言</ref>。另外从词汇上来看，明代四川话词汇以单音节词居多，表现出与南方诸汉语相类似的特点，这表明四川话在形成期更多的受到了南方汉语而非北方官话的影响<ref name="《〈蜀语〉所反映的明代四川方言的结构特征》">向学春、徐薇（2007年第5期），《〈蜀语〉所反映的明代四川方言的结构特征》，重庆三峡学院学报</ref>。

明末[[清朝|清]]初，四川地区再次陷入战乱，人口锐减，之后的第二次“[[湖广填四川|湖广填四川]]”大移民运动使四川人口构成发生[[屠蜀|剧变]]。就全川而言，移民运动之后四川土著居民仅占四川总人口的约30%<ref name="《西南历史文化地理》第三章"/><ref group="注">具体数值尚有争议，有学者指出约为10%至20%（如李世平著《四川人口史》），但并未考虑土著人口的自然增长；有学者依据方志与现存族谱指出约为50%（如胡昭曦著《张献忠屠蜀考辨》）。此处取中间数值。</ref>。但是，明末清初战乱中，四川各地受到的破坏程度相差很大，川东、川北地区受战乱影响严重，而川南地区却受影响十分轻微<ref name="《西南历史文化地理》第三章"/>。当时还有大量川东、川北的四川土著居民，前往川南地区以及与川南毗邻的滇黔北部地区躲避战乱<ref name="《张献忠屠蜀考辨》">{{cite book |author= 胡昭曦 |title=《张献忠屠蜀考辨》 |year=1980 |publisher=四川人民出版社 |OCLC=232009917}}</ref>。因而，川南地区的存留的四川土著居民人数可能较移民仍占优势，其中又以[[峨眉山市|峨眉]]、[[乐山|乐山]]、[[犍为|犍为]]一带为最<ref name="《张献忠屠蜀论》">任乃强，《张献忠屠蜀论》，社会科学研究丛刊——张献忠在四川</ref>，从而使以宋元巴蜀语为基础的明代四川话在川南地区仍然得到存留<ref name="《四川官话入声现象的历史文化透视》">杨波（1997年第5期），《四川官话入声现象的历史文化透视》，西南师范大学学报（哲学社会科学版）</ref>。与此同时，大量来自湖广、江西、广东等地移民进入川东、川北地区，从而使四川话内部产生了新派（川东、川北）与老派（川西、川南）的分化<ref name="《西南历史文化地理》第三章"/>，奠定了现今四川话内部新老两派并存的格局。

=== 声调 ===
四川话的调类总共有[[阴平|阴平]]、[[四聲|阳平]]、[[上声|上声]]、[[去声|去声]]、[[入声|入声]]五类，部分地区四川话中入声已经消失，但入声字全部整齐地归入某一个调类（如[[成都话|成都话]]入派阳平、[[自贡话|自贡话]]入派去声、[[雅安话|雅安话]]入派阴平），与入声杂乱派入四个调类的[[北方官话|北方官话]]截然不同。依据《四川方言音系》统计，四川话总计150个方言点中，拥有5个调类（即入声保留的）的有48点，约占方言点总数的三分之一；而拥有4个调类的有102点，约占方言点总数的三分之二<ref name="《四川方言音系》">甄尚灵等（1960年第3期），《四川方言音系》，四川大学学报（社会科学版）</ref>。

=== 声母 ===
四川话中包括零声母在内总计有25个声母，如下表所示（例字以[[成都话|成都话]]为准）。依据《四川方言音系》统计，四川话总计150个方言点中，[[巫溪|巫溪]]、[[奉节|奉节]]2点拥有19个声母，无[tʂ]、[tʂʰ]、[ʂ]、[ʐ]、[nʲ]、[f]；[[重庆|重庆]]等45点拥有20个声母，无[tʂ]、[tʂʰ]、[ʂ]、[ʐ]、[nʲ]；[[成都|成都]]等77点拥有21个声母，无[tʂ]、[tʂʰ]、[ʂ]、[ʐ]；[[盐边|盐边]]等8点拥有23个声母，无[z]、[nʲ]；[[自贡|自贡]]等11点拥有24个声母，无[z]；[[都江堰|都江堰]]等7点拥有全部25个声母<ref name="《四川方言音系》"/><ref group="注">《四川方言音系》并未将[v]声母纳入统计，此处为修正后的数值</ref>。

绝大多数地区的四川话没有tʂ组声母（[[卷舌音|卷舌声母]]）。四川话总计150个方言点中，有tʂ组声母仅为26点，且都江堰等7点只出现在[[入声|入声]]字中，并只与韵母[ə]（[[西充|西充]]为[ɿ]）相拼，只有自贡等19点（主要为入声归去声区）tʂ组声母可以和较多韵母相拼。同时，四川话还拥有部分[[普通话|普通话]]没有的声母，如舌尖浊擦音[z]（例字<ref name="例字" group="注">例字选用以成都话为准</ref>：人[zən<sup>2</sup>]<ref group="注">音标后数字表示四川话的声调，1为阳平、2为阴平、3为上声、4为去声、5为入声</ref>、{{fact|如[zu<sup>2</sup>]）、唇齿浊擦音[v]（例字：五[vu<sup>3</sup>]、乌[vu<sup>1</sup>]）、舌根浊鼻音[ŋ]（例字：我[ŋo<sup>3</sup>]、欧[ŋəu<sup>1</sup>]）、舌面前浊鼻音[nʲ]（例字：你[nʲi<sup>3</sup>]、宜[nʲi<sup>2</sup>]）等。}}

==== 入声紧元音 ====
约有三分之一地区的四川话入声独立成调，但其中入声塞音韵尾基本全部消失。四川话中保留有一套相对独立的仅用于入声字的韵母，如[iæ]、[uæ]、[ʊ]、[ɘ]、[ɐ]、[iɐ]、[uɐ]、[c]、[yʊ]、[yɵ]等，这些韵母有紧喉作用，在发音时[[喉头|喉头]]肌肉和[[口腔|口腔]]肌肉紧张，从而使整个音节表现出一种粗硬紧促的状态<ref name="《四川官话雅棉小片入声归阴平研究》">余江（2004年5月），《四川官话雅棉小片入声归阴平研究》，汕头大学</ref>。由此可见，在入声独立的四川话中，入声并不仅仅单纯以其相对音高区别于其他调类的，而是发展出了[[元音松紧对立|入声紧元音]]来取代入声塞音韵尾。因此古入声字在四川话入声独立区和入声归入舒声区拥有一定的读音差异，即主要元音的松紧对立，如下表所示<ref name="《四川官话雅棉小片入声归阴平研究》"/>。

=== 文白异读 ===
四川话由于受[[通用语|共同语]]（古代的[[雅音|雅音]]及近现代的[[北方官话|北方官话]]）的长期影响而产生了[[文白异读|文白异读]]系统。白读音通常是四川话所固有的读音，是对自身古代语音的继承；文读音则通常与北方官话较为接近。一般而言，白读音主要出现在高频日常生活用语中，而文读音主要出现在书面语、新词汇中。四川话的文白异读系统也在不断地发展变化中，但近几十年来，由于中国政府的[[推广普通话|单一语言政策]]，主要的趋势是文读音越来越占优势，部分字词白读音已趋于消失，固定为文读音<ref name="《普通话对成都话语音的影响》">崔荣昌、宋伶俐（1999年06期），《普通话对成都话语音的影响》，语文建设</ref>。四川话中部分常见的文白异读字如下表所示（例字以[[成都话|成都话]]为准）<ref name="《成都语音的初步研究》"/><ref>此处以1958年《成都语音的初步研究》为准，不包括近年来由普通话引起的四川话的音变</ref>。

=== 变音 ===

==== 变调 ====
[[变调|连读变调]]现象在四川话口语中十分常见，但各地略有差异，以成渝片为例，大致来说四川话中的变调可以分为4类<ref name="《四川（平昌）方言的连读变调》">谭伦华（1998年07期），《四川（平昌）方言的连读变调》，西南民族学院学报（哲学社会科学版）</ref>。一是重叠词中的变调，一般而言如果组成该重叠词的字声调为阳平或去声，则第二个字变调为阴平（例词：爸爸[{{IPA|pa}}<sup>2</sup>{{IPA|pa}}<sup>1</sup>]、婆婆[{{IPA|pʰo}}<sup>2</sup>{{IPA|pʰo}}<sup>1</sup>]、舅舅[{{IPA|tɕiəu}}<sup>4</sup>{{IPA|tɕiəu}}<sup>1</sup>]、帕帕[{{IPA|pʰa}}<sup>4</sup>{{IPA|pʰa}}<sup>1</sup>]）；同时，如果组成该重叠词的字声调为上声，第二个字变调为阳平（例词：姐姐[{{IPA|tɕiai}}<sup>3</sup>{{IPA|tɕiai}}<sup>2</sup>]、板板[{{IPA|pan}}<sup>3</sup>{{IPA|pan}}<sup>2</sup>]）。二是儿尾及儿化词中的变调，如果儿尾名词为阳平，儿尾变调为阴平（例词：娃儿[{{IPA|ua}}<sup>2</sup>{{IPA|ɚ}}<sup>1</sup>]、瓢儿[{{IPA|pʰiau}}<sup>2</sup>{{IPA|ɚ}}<sup>1</sup>]）；如果儿化词的最后一个字为阳平，该字变调为阴平（例词：葡萄<sub>儿</sub>[{{IPA|pʰu}}<sup>2</sup>{{IPA|tʰɚ}}<sup>1</sup>]、麻雀<sub>儿</sub>[{{IPA|ma}}<sup>2</sup>{{IPA|tɕʰyɚ}}<sup>1</sup>]）。三是特定字的变调，如“去”、“头”、“面”、“上”等虚词在多数情况下都变读为阴平（例词：下去[{{IPA|saŋ}}<sup>4</sup>{{IPA|tɕʰe}}<sup>1</sup>]、屋头[{{IPA|vu}}<sup>2</sup>{{IPA|tʰəu}}<sup>1</sup>]、外面[{{IPA|uai}}<sup>4</sup>{{IPA|mian}}<sup>1</sup>]、身上[{{IPA|sen}}<sup>1</sup>{{IPA|saŋ}}<sup>1</sup>]）。四是二字词及三字词的第二个字在很多情况下变读为阴平（例词：姑娘[{{IPA|ku}}<sup>2</sup>{{IPA|niaŋ}}<sup>1</sup>]、经佑[{{IPA|tɕin}}<sup>1</sup>{{IPA|iəu}}<sup>1</sup>]、院坝[{{IPA|yuən}}<sup>4</sup>{{IPA|pa}}<sup>1</sup>]、龙门阵[{{IPA|noŋ}}<sup>2</sup>{{IPA|mən}}<sup>1</sup>{{IPA|tsən}}<sup>4</sup>]、明年子[{{IPA|min}}<sup>2</sup>{{IPA|nʲian}}<sup>1</sup>{{IPA|tsɿ}}<sup>3</sup>]）<ref name="《四川（平昌）方言的连读变调》"/>。

==== 儿化 ====
[[儿化|儿化]]是四川话（除岷江小片小部分地区）中的一种常见的音变现象，四川话的儿化内部较为一致，但与[[北方官话|北方官话]]的儿化存在很大差异，特别是在儿化韵的数目上，四川话中的儿化韵仅有[ɚ]、[iɚ]、[uɚ]、[yɚ]4个，而北方官话有26个<ref name="《北京话和成都话、重庆话的儿化比较》">郑有仪（1987年02期），《北京话和成都话、重庆话的儿化比较》，重庆师范大学学报（哲学社会科学版）</ref><ref name="《四川官话雅棉小片入声归阴平研究》"/><ref name="《自贡方言音系》">萧玲玲（1995年04期），《自贡方言音系》，四川大学学报（哲学社会科学版）</ref>。以成渝片为例，成渝片中的37个韵母，仅[ɚ]、[iai]、[ue]、[yoŋ]、[yu]5个不能儿化，其余32个韵母中，所有撮口呼韵母儿化为[yɚ]（例词：汤圆<sub>儿</sub>[{{IPA|tʰaŋ}}<sup>1</sup>{{IPA|yɚ}}<sup>2</sup>]、麻雀<sub>儿</sub>[{{IPA|ma}}<sup>2</sup>{{IPA|tɕʰyɚ}}<sup>1</sup>]）；所有齐齿呼韵母儿化为[iɚ]（例词：妖艳<sub>儿</sub><ref group="注">妖艳<sub>儿</sub>，四川话中意为爱打扮，爱炫耀或得意洋洋</ref>[{{IPA|iau}}<sup>1</sup>{{IPA|iɚ}}<sup>2</sup>]、背心<sub>儿</sub>[{{IPA|pei}}<sup>4</sup>{{IPA|ɕiɚ}}<sup>1</sup>]）；所有合口呼韵母儿化为[uɚ]（例词：颤花<sub>儿</sub><ref group="注">颤花<sub>儿</sub>，四川话中意为爱表现、爱出风头的人</ref>[{{IPA|tsan}}<sup>4</sup>{{IPA|xuɚ}}<sup>1</sup>]、街娃<sub>儿</sub><ref group="注">街娃<sub>儿</sub>，四川话中意为不良少年</ref>[{{IPA|kai}}<sup>1</sup>{{IPA|uɚ}}<sup>2</sup>]）；除[o]、[oŋ]外的所有开口呼韵母全部儿化为[ɚ]（例词：齁包<sub>儿</sub><ref group="注">齁包<sub>儿</sub>，四川话中意为患有慢性气管炎或肺心病的人</ref>[{{IPA|xəu}}<sup>1</sup>{{IPA|pɚ}}<sup>1</sup>]、舌头<sub>儿</sub>[{{IPA|se}}<sup>2</sup>{{IPA|tʰɚ}}<sup>1</sup>]），而[o]、[oŋ]仅在同唇音声母相拼时儿化为[ɚ]（例词：棚棚<sub>儿</sub>[{{IPA|pʰoŋ}}<sup>2</sup>{{IPA|pʰɚ}}<sup>1</sup>]、钵钵<sub>儿</sub>[{{IPA|po}}<sup>2</sup>{{IPA|pɚ}}<sup>1</sup>]），其它情况下儿化为[uɚ]（例词：窝窝<sub>儿</sub>[{{IPA|o}}<sup>1</sup>{{IPA|uɚ}}<sup>1</sup>]、火锅<sub>儿</sub>[{{IPA|xo}}<sup>3</sup>{{IPA|kuɚ}}<sup>1</sup>]）<ref name="《北京话和成都话、重庆话的儿化比较》"/>。

== 词汇 ==
四川话的词汇主要由承自中上古时期[[巴蜀语|巴蜀语]]的词汇、由明清时期移民带来的移民语言的词汇、承自古代汉语通语的词汇三部分组成。近年来也有大量的来自[[普通话|普通话]]、[[英语|英语]]等语言的外来语词汇进入四川话，同时也有诸如“雄起”<ref group="注">雄起，四川话中意为加油，也有用激烈的言辞或行为回敬对方之意</ref>、“勾兑”<ref group="注">勾兑，四川话中意为通过一定的物质手段进行人际沟通</ref>、“假打”<ref group="注">假打，四川话中意为说一些夸张虚假的话而不去落实</ref>、“洗白”<ref group="注">洗白，四川话中相当于“完蛋”，是对无可挽回的损失的一种形象化描述</ref>、“冒泡<sub>儿</sub>”<ref group="注">冒泡<sub>儿</sub>，四川话中意为偶尔的出一下风头</ref>等四川话新词汇在成都、重庆等大城市产生，并迅速通行全川<ref name="《当代成都方言新词汇例释》">杨文全、鲁科颖（2005年05期），《当代成都方言新词汇例释——兼论其造词心理与民间文化意蕴》，西华师范大学学报（哲学社会科学版）</ref><ref name="《重庆言子儿的文化透视》">沈荭（2008年02期），《重庆言子儿的文化透视》，重庆大学学报（社会科学版）</ref>。四川话的词汇具有鲜明的地域特色，反映了巴蜀地区独特的民间文化和风俗习惯。

四川话的词汇与其它汉语的相比具有十分巨大的差异，与四川话关系较为密切的[[云南话|云南话]]也仅有58.3%的词汇与四川话相同<ref name="《四川方言与巴蜀文化》"/>，同时由于四川话在形成期更多的受到了由移民带来的[[湘语|湘语]]、[[赣语|赣语]]等南方汉语的影响<ref name="《〈蜀语〉所反映的明代四川方言的结构特征》"/>，从而使四川话的词汇与北方官话拥有很大差异。因此，虽然四川话通常被归为[[官话|官话]]的一支，但[[北方官话|北方官话]]中与四川话相同的词汇仅有47.8%，远不及新派湘语的54.9%和赣语的49.4%<ref name="《四川方言与巴蜀文化》"/>。

=== 特征 ===
==== 词形 ====
四川话词汇和普通话在词形上存在显著差异，以下从[[音节|音节]]和[[词素|词素]]两个方面进行比较。首先，在音节方面，四川话中大量词汇与普通话在音节数目上存在差异，例如有些四川话词汇的单音节词对应在普通话词汇中则是多音节词；与之相反，有些词汇在四川话中为多音节词，但对应在普通话词汇中则是单音节词；同时有些词汇虽然在四川话与普通话中同为多音节词，但音节数并不相同<ref name="《渠县方言词语研究》">罗家国（2002年5月），《渠县方言词语研究》，厦门大学</ref><ref name="《自贡方言词语研究》">刘孝利（2007年4月），《自贡方言词语研究》，四川大学</ref>。关于音节比较的部分例词如下表所示：

==== 词义 ====
四川话中部分词汇词形与普通话一致，但意义却截然不同，例如“造孽”在四川话中意为可怜，而在普通话中是[[佛教|佛教]]用语，意为做坏事；而“饮”在四川话中意为浇灌植物，在普通话中则为喝水或给牲畜水喝的意思；再如“不好”在四川话中意为生病，在普通话中则为“好”的反义词。四川话中还有部分词汇词形与普通话一致，但词义范围不同，例如“鼻子”在四川话中除了指[[鼻|人体器官]]外，还可指[[鼻涕|鼻涕]]；再如“新鲜”在四川话中还有“清醒过来”的含义，同时“醒”在四川话里还可以指食品变质<ref name="《自贡方言词语研究》" />。

=== 禁忌语 ===
四川话拥有丰富且自成体系的[[禁忌语|禁忌语]]系统，早在[[西汉|西汉]][[扬雄|扬雄]]所著《[[方言|方言]]》中便有对巴蜀语中禁忌语的记载，四川话的禁忌语中体现了较多的[[巴蜀|巴蜀地区]]的风俗、语言特点。首先，四川话中的忌讳说凶恶[[动物|动物]]，例如[[蛇|蛇]]在四川话中的讳称有“梭老二”、“梭梭”、“干黄鳝”、“长虫”等，[[老虎|老虎]]在四川话中的讳称有“猫猫”、“大猫”、“大头猫”、“扁担花”等，[[狐狸|狐狸]]在四川话中的讳称为“毛狗”，[[老鼠|老鼠]]在四川话中的讳称有“耗子”、“老水子”、“高客”、“喜马”等。其次，四川话中忌讳说不吉或不雅的词，如与“散”同音的“伞”往往被称作“撑花<sub>儿</sub>”或“撑子”，而死亡在四川话中通常被称为“不在”，并有“莫搞了”、“莫脉了”、“戳火了”、“撬杆<sub>儿</sub>了”、“翻翘了”等说法，再如生病在四川话中被称为“不好”或“装狗（狗）”、“变狗（狗）”。另外四川话中仅有“[[屄|屄]]”一个字音为[{{IPA|pʰi}}<sup>1</sup>]（意为女性生殖器，至今仍在使用），在民国前为交际中非常忌讳亦非常难以说出口的虚词，其它所有原本与之同音的字如“批”、“披”等一律转读为[{{IPA|pʰei}}<sup>1</sup>]<ref name="《四川人的禁忌语》">黄尚军（2006年10期），《四川人的禁忌语》，西华大学学报（哲学社会科学版）</ref>，{{fact|仅在20世纪60年代后}}，才同下江官话的类似词汇一样，为泄愤或骂人时使用。

== 语法 ==

=== 词法 ===

==== 词缀 ====
四川话的[[词缀|词缀]]与普通话相比有较大差异。首先，四川话中拥有部分普通话中没有的特有词缀，例如四川话中常见的动词词缀“倒”可以组合为“估倒”（逼迫）、“谙倒”（猜测）、“马倒”（欺压）、“默倒”（以为）、“审倒”（小心试探）、“阴倒”（不张扬）和“幽倒”（注意）等常用语。再如“头”在四川话中也是一个极常见的动词词缀，可以组成如“学头”、“看头”、“搞头”、“吃头”、“做头”、“讨论头”、“复习头”等常用语，如“钓鱼城有没得耍头”在四川话中意为“钓鱼城好不好玩”<ref name="《成都方言语法研究》第二章">{{cite book |author= 张一舟等 |title=《成都方言语法研究》 |year=2001 |publisher=巴蜀书社 |chapter =第二章：构词法 |isbn=7806592261}}</ref>。

第二，四川话中部分词缀虽然普通话中也有，但使用范围和附加含义却有很大差异，如四川话中名词词缀“子”可以构成“蜂子”（蜜蜂）、“耳子”（木耳）、“砣子”（拳头）、“羊子”（羊）、“烟子”（烟）、“今年子”（今年）、“明年子”（明年）等普通话中没有的词汇，同时四川话中还有“分分子”、“角角子”、“两两子”、“斤斤子”这类由[[量词|量词]]重叠后加词缀“子”，用来表数量少的特殊用法，例如“你咋个净给我些角角子”在四川话中意为“你怎么只给我角票”。另外，[[北京话|北京话]]中没有儿尾词，其词缀“儿”都以儿化的形式出现，但四川话中有“裤儿”、“帽儿”、“娃儿”、“刀儿”、“偷儿”、“样儿”、“锅儿”等大量儿尾词。同时名词词缀“家”在四川话中一方面可以用来表示人群之间的对立，如“姑娘家”与“儿娃子家”、“婆娘家”与“男人家”、“娃儿家”与“大人家”（这种情况下“家”词缀还可以重叠表轻视，如“儿娃子家家的，还啬得很”在[[成都话|成都话]]中意为“男孩子怎么这么吝啬”）；另一方面“家”词缀还可以表时间，如“春天家”、“白天家”、“往回家”等<ref name="《成都方言语法研究》第二章"/>。

=== 句法 ===
在四川话中，有很多特有的并不见于普通话的句式结构，仅与“得”字相关的常用句式结构就有“得V/不得V”、“V得/V不得”、“V得C/V得不C”、“V得C（O）”、“V得有（O）”（V为动词、C为补语、O为宾语）等5种，沉积了汉语各个历史时期“得”字的用法<ref name="《成都方言语法研究》第十三章">{{cite book |author= 张一舟等 |title=《成都方言语法研究》 |year=2001 |publisher=巴蜀书社 |chapter =第十三章：跟“得”字有关的句法结构 |isbn=7806592261}}</ref>。除此之外，四川话中的特有句式还包括：“V都V了”、“V起来/V起去/V起XY”、“到X不X（Y）”以及诸如“X不（没）XY”、“X都XY了”、“X都要XY”、“X就XY”一类的半拷贝结构等<ref name="《成都方言语法研究》第十四章">{{cite book |author= 张一舟等 |title=《成都方言语法研究》 |year=2001 |publisher=巴蜀书社 |chapter =第十四章：“V起来，V起去”与“V起XY”|isbn=7806592261}}</ref><ref name="《成都方言语法研究》第十五章">{{cite book |author= 张一舟等 |title=《成都方言语法研究》 |year=2001 |publisher=巴蜀书社 |chapter =第十五章：半拷贝结构 |isbn=7806592261}}</ref>。部分句式的例句如下表所示：

== 四川方言字 ==
{{主條目|四川方言字}}
由于四川话的词汇与作为中国通用语的北方官话具有很大差异，通用的汉字并不能满足巴蜀地区内部交流的需要，因此在四川话的基础上产生了四川方言字，来弥补通用汉字的不足<ref name="《汉字的结构及其流变》"/>。四川方言字通常采用形声、会意等手段来纪录四川话中不见于[[现代标准汉语|现代标准汉语]]的特有词汇。例如“㧯”（四川话拼音：nao<sup>3</sup>；抓取之意）、“𢭃”（四川话拼音：dou<sup>4</sup>；对碰、凑、接之意）、“𤆵”（四川话拼音：pa<sup>1</sup>；软、烂之意）等为采用形声创造的四川方言字；而“𨈓”（四川话拼音：nang<sup>1</sup>；瘦小之意）、“𡘧”（四川话拼音：qia<sup>2</sup>或ka<sup>2</sup>；跨、大步走之意）、“嫑”（四川话拼音：biao<sup>1</sup>；不要之意）等为采用会意创造的四川方言字<ref name="《汉字的结构及其流变》">{{cite book |author= 梁东汉 |chapter=第一章第四节：在方言基础上产生的方言文字 |title=《汉字的结构及其流变》 |year=1959 |publisher=上海教育出版社}}</ref><ref name="《汉字学概论》">{{cite book |author= 张玉金、夏中华 |chapter=第三章第七节：汉字的地域变异 |title=《汉字学概论》 |year=2001 |publisher=广西教育出版社|isbn=7543531836}}</ref>。其中部分四川方言字随着相关词汇也进入了现代标准汉语，例如“搞”字原本就是四川方言字<ref name="《汉字的结构及其流变》"/>。

== 和其它语言的关系 ==

=== 和其它汉语方言的关系 ===
将[[成都话|成都话]]作为四川话的代表与其它汉语方言进行词汇的比较，大致可以得出四川话与其它汉语方言的亲疏关系<ref name="《四川方言与巴蜀文化》"/>（如下表所示），与四川话词汇相同比例越高的，与四川话的关系则越亲密；反之，则越疏远。四川话与同属汉语西南官话的[[云南话|云贵官话]]关系最为紧密，但由于四川地区与云贵地区在人口构成上有一定区别，因而词汇上已经出现较大差异。

西南官话区外，[[湘语|湘语]]与四川话联系最为紧密，两者拥有大量的独具特色的共有词汇，同时值得一提的是，湘语与川渝地区以外云、贵、鄂等地的[[西南官话|西南官话]]关系却比较疏远，这主要是由于清前期的“[[湖广填四川|湖广填四川]]”移民运动中，有大量来自湘语区的移民进入四川，湘语在现今四川话的形成过程中扮演了重要的角色，从而带来了大量来自湘语的词汇<ref name="《湘方言词汇研究》">{{cite book |author= 罗昕如 |title=《湘方言词汇研究》 |year=2006 |publisher=湖南师范大学出版社 |chapter =第九章第三节：从词汇看湘语与西南官话的关系 |isbn=9787810816168}}</ref>。[[扬雄|扬雄]]《[[方言|方言]]》中归为古楚语的“謱謰”（四川话俗作“裸连”，啰嗦之意）、“革”（四川话称“老革革”，老之意）、“崽”（四川话称“崽儿”或“崽崽”，儿子之意）等词汇便进入了四川话<ref name="《湘方言词汇研究》"/>，同时四川话中“蚌壳”（蚌）、“跍”（蹲）、“侧边”（旁边）、“酽”（浓稠）、“灶屋”（厨房）、“堂屋”（正房）等词汇也很可能来自湘语<ref name="《四川境内的湘方言》">{{cite book |author= 崔荣昌 |title=《四川境内的湘方言》 |year=1996 |publisher=中华书局（台湾） |isbn=9789576713972}}</ref>。{{fact|此外，四川话在词汇上与[[赣语|赣语]]的相似性也超过了与四川话同属[[官话|官话]]的[[北方官话|北方官话]]，这也与清初大量来自[[江西|江西]]的移民进入四川有关。}}

建立[[古蜀|古蜀国]]的[[蜀族|蜀族]]便是由[[岷江|岷江]]上游的今[[羌族|羌族]]聚居地迁徙至[[成都平原|成都平原]]的古氐羌系民族的一支<ref>萧易（2008年第5期），《古蜀国 故乡还是他乡？》，先锋国家历史</ref>，作为四川话源头之一的非华夏语的[[古蜀语|古蜀语]]与[[羌语|羌语]]便具有密切的联系。同时由于羌族与巴蜀汉族交往密切，加之羌语各方言间难以沟通，使“汉话”（羌族对四川话的称呼<ref name="《华夏边缘：历史记忆与族群认同》">{{cite book |author= 王明珂 |title=《华夏边缘：历史记忆与族群认同》 |year=2006 |publisher=社会科学文献出版社 |isbn=9787802300439}}</ref>）成为了不同地区羌族之间用以交流的通用语<ref>孫宏開（1988年第4期），《論羌族雙語制——兼談漢語對羌語的影響》，民族語文</ref>，四川话（主要是羌族聚居区周边方言）对羌语造成了巨大的影响。

首先，南部羌语受四川话的影响而产生了声调<ref name="《四川方言与巴蜀文化》第六章">{{cite book |author= 崔荣昌 |title=《四川方言与巴蜀文化》 |year=1996 |publisher=四川大学出版社 |chapter =第六章：四川的闽语及其它 |isbn=7561412967}}</ref>。以羌语[[桃坪话|桃坪话]]为例，其拥有6个声调，其中5个都与四川话借词有关。这5个声调与羌族聚居区周边四川话不仅调类相同，且调值也几乎完全一样<ref name="《四川方言与巴蜀文化》第六章"/>（如右表所示）。其次，羌语中拥有大量的来自四川话的[[外来语|借词]]，这些借词以[[名词|名词]]、[[动词|动词]]为主，有少量[[量词|量词]]、[[副词|副词]]、[[连词|连词]]，涉及方面十分广泛。同时由于羌语区各地受汉族影响程度不同，因而汉语借词比例也不相同，总的来说四川话借词占羌语总词汇的10%到30%<ref name="《四川方言与巴蜀文化》第六章"/>。羌语中的汉语借词发音与羌族聚居区周边四川话发音基本相同，且同样具有保持古音较多的特征。另外羌语在四川话的影响下，增加了辅音音位[f]、单元音音位[ʅ]、二合元音、三合元音以及鼻尾韵母等语音要素<ref name="《四川方言与巴蜀文化》第六章"/>。

== 生存危机 ==
虽然四川省及重庆市通常被认为是方言较为强势的地区<ref>{{cite web |url=http://news.sina.com.cn/c/2005-05-04/17586559832.shtml |title=四川经验：剽悍方言无需要保卫 |accessdate=2010-05-15 |publisher=新浪网新民周刊专题}}</ref>，但事实上由于[[中国|中国]]政府的“[[推广普通话|推广普通话]]”政策，目前四川话的生存环境正受到普通话的强烈冲击，有学者认为在中国的政治经济背景下，四川话长远甚至有完全被普通话取代的可能<ref name="《近四十年成都话语音变动现象考论》">夏中易（2002年04期），《近四十年成都话语音变动现象考论》，成都大学学报（社科版）</ref>。一方面四川话在[[广播|广播]]、[[电视|电视]]等媒体以及部分公共场合的使用受到限制；另一方面学校尽量使用[[普通话|普通话]]教学，且没有教授四川话的课程，从而导致川渝地区80年代后出生的人使用母语的能力较弱，所使用的四川话受到了普通话的严重影响：其特有的语音成份有些正逐渐趋同于普通话，但另一方面普通话的语音成分又很难进入四川话，因而音韵系统正在简化<ref name="《20世纪成都话音变研究》">周及徐（2001年04期），《20世纪成都话音变研究》，四川师范大学学报（社会科学版）</ref>。

同时，普通话的推广还使川渝民众对自己母语产生了较为矛盾的心理：一方面仍然对四川话拥有很高的忠诚度，另一方面却认为四川话地位层次较普通话为低<ref name="《西部开发与语言规划——重庆地区语言态度调查研究》">肖肃（2003年01期），《西部开发与语言规划——重庆地区语言态度调查研究》，四川外语学院学报</ref>，虽然事实上各种语言之间并没有贵贱之分，但这种心理也一定程度上加快了四川话式微的过程。基于相同的心理，在四川话内部，入声保留的[[岷江小片|岷江小片]]，由于受到相对强势的成渝片和普通话的双重影响，处境更加危急<ref name="《张口闭口成都腔》">{{cite web |url=http://news.163.com/09/1124/10/5OSKLLUF000120GR.html |date= 2009-11-24 |title=张口闭口成都腔 谁还能说土掉渣的“岷江话” |accessdate=2010-05-15 |work=网易 |publisher=成都商报}}</ref><ref name="《谈谈乐山方言语音的偏移》">赖先刚（2004年第6期），《谈谈乐山方言语音的偏移》，天府新论</ref>。

目前，由于缺乏母语教育，年轻一代已经无法熟练掌握四川话丰富的词汇、短语及句式；同时也无法掌握四川话中大量字词的读音，音韵机械模仿普通话<ref name="《近四十年成都话语音变动现象考论》"/>，例如[v]声母在年轻一代中丢失严重，[ŋ]声母、[yo]韵母也有一定的丢失现象，同时[k]声母误读为[tɕ]、[s]声母误读为[tsʰ]、[x]声母误读为[ɕ]，[ən]韵母误读为[uən]、[ue]韵母误读为[o]等现象也很突出<ref name="《近四十年成都话语音变动现象考论》"/>。不过有学者认为四川话的部分固有的音系特征，如特有的调值，一些难以被普通话取代的韵母、声母，在一定时间内仍会继续得到保持<ref name="《20世纪成都话音变研究》"/>。年轻人四川话中常见的音变如下表所示（以[[成都话|成都话]]为例）：

四川话的式微势必会使以四川话为根基的[[巴蜀文化|巴蜀文化]]，特别是以四川话为载体的巴蜀传统艺术（如[[川剧|川剧]]、[[四川清音|清音]]、[[四川相书|相书]]、[[竹琴|竹琴]]、[[金钱板|金钱板]]等）遭受到严重的冲击，甚至也随之消亡，因而近年来中国学术界也出现了一定保护方言的呼声<ref name="《汉语方言文化遗产保护的意义与对策》">吴永焕（2008年04期），《汉语方言文化遗产保护的意义与对策》，中国人民大学学报</ref><ref name="《从中国文化的兴亡论方言的捍卫》">李巧兰（2006年04期），《从中国文化的兴亡论方言的捍卫》，社会科学家</ref>，而部分川渝民众对四川话不自信的态度也有所转变，例如[[成都地铁|成都地铁]]只使用普通话和英语报站，却不使用四川话报站，就引起了网络上的激烈争论<ref name="《民间版成都地铁报站词走红，用不用四川话惹议》">{{cite web |url=http://cd.qq.com/a/20100419/000926.htm|author=魏晖、张强 |date= 2010-04-19 |title=民间版成都地铁报站词走红，用不用四川话惹议 |accessdate=2010-05-15 |work=腾讯·大成网 |publisher=成都商报}}</ref>。而中国广电总局宣布的对四川话方言剧的禁令，也引发了很大争议<ref name="《广电总局方言禁令》">{{cite web|url=http://news.china.com/zh_cn/domestic/945/20090827/15616464.html|date=2009-08-27|title=广电总局“方言令”惹争议 东北话放生四川话受伤|accessdate=2010-05-15|work=中华网|publisher=重庆晚报|deadurl=yes|archiveurl=https://web.archive.org/web/20090830093441/http://news.china.com/zh_cn/domestic/945/20090827/15616464.html|archivedate=2009年8月30日|df=}}</ref>。
