'''四川省江油市第一中学'''简称“一中”为四川省[[绵阳市|绵阳市]][[江油市|江油市]]的公办普通高中，位于江油市中坝镇诗仙路东段168号（太白公园旁），创立于1914年，原名四川省立二中，首任校长[[李宗吾|李宗吾]]。江油一中于2000年创办成四川省级重点中学，2002年被确定为四川省省级示范高中，现为四川省二级示范性普通高中。<ref>{{Cite web|url=http://www.scjyyz.com/Article/b85c4a26-7823-4e32-9976-9a0bf4af5542.shtml|title=四川省江油市第一中学简介|accessdate=2017-08-02|author=陈小波|date=2016-04-20|publisher=江油一中官网}}</ref>

== 历史沿革 ==
* 1914年，四川省省立第二中学在江油成立，为四年制学校。<ref>{{Cite web|url=https://wenku.baidu.com/view/d3cf9f1be45c3b3567ec8bf4.html|title=江油县志|accessdate=2017-08-02|author=|date=2014-06-12|publisher=百度文库}}</ref>
* 1922年，省立第二中学改为六年制中学，提供初中与高中教育。
* 1935年，作为江油县立初中迁往省立二中校址。
* 1944年，江油县立初中迁回[[武都镇|武都镇]]。
* 1950年，设立川北省立江油中学。
* 1952年，高中部停办，更名为江油县立初级中学。
* 1970年，受[[文化大革命|文化大革命]]冲击，迁往双河口。
* 1973年，学校迁回省立二中原址。
* 1983年，成为高级完全中学，更名为四川省江油县第一中学。
* 1987年，四川省江油县第一中学随着江油撤县建市而更名为四川省江油市第一中学。
* 2000年，四川省江油市第一中学创办为四川省省级重点高中。
* 2002年，四川省江油市第一中学创办为四川省省级示范性高中。
* 2013年，四川省江油市第一中学接受复核并被确认为四川省二级示范性普通高中。<ref>{{Cite web|url=http://www.scjyyz.com/Article/1518ad60-45b6-4e02-a458-2a6268714ecf.shtml|title=百年校庆|accessdate=2017-08-02|author=|date=2015-05-04|publisher=江油一中官网}}</ref>

== 现状 ==
江油一中现占地面积100余亩，拥有教职工两百余人，学生近四千人， 近六十个教学班。在一些机构发布的中学排行榜中，江油一中排名在绵阳及四川前列。<ref>{{Cite web|url=http://mp.weixin.qq.com/s?src=3&timestamp=1501651041&ver=1&signature=AtrXI4VhZQeXixSXjTmVTSIgfU5g9SNVMCaxnIQJ6hfEYxPsy*216zHiaXdyTAHTNMg9nvSf4*7IEAmTmyh4reFjUHdmVaj1UnAJWf-nteHsE2FjuIa8EfMR8FyoOOBBx1AkvI*UNi7puPp*jw*w-LxPVo4yvHyCPxV7bomKxBg=|title=联合学校热闻 {{!}} 江油一中列入全国顶尖中学排行榜！|accessdate=2017-08-02|author=|date=2017-06-27|publisher=统一教育智慧校园公众号}}</ref>

== 特色 ==

=== 红色校园文化 ===
江油一中以“塑造气质，陶冶性情，惠泽心智”为出发点，形成了“红色校园文化”。

=== 优秀传统文化 ===
江油一中注重宣传中国优秀传统文化，目标为全力打造“具有丰厚人文内涵的书香校园”。

=== 李白文化 ===
按照江油一中官网的陈述，四川省江油市第一中学已经“形成了‘挖掘学校积蕴，传承国学精粹，凸显[[李白|李白]]文化’的校园文化主题与特色”。<ref>{{Cite web|url=http://www.scjyyz.com/Article/8c340cd1-1a2c-48f8-8b98-2df1837b7927.shtml|title=百年名校 江油一中|accessdate=2017-08-02|author=|date=2016-04-09|publisher=江油一中官网}}</ref>

== 争议 ==
包括江油一中在内的绵阳市众多高中被指责过度[[应试教育|应试教育]]，缺乏对学生综合素质的培养。<ref>{{Cite web|url=http://cd.qq.com/a/20160831/013730.htm|title=2016四川“十大超级高中”排行榜 没成都“四七九”|accessdate=2017-08-02|author=|date=2016-08-31|publisher=腾讯大成网-教育频道}}</ref>

2017年秋季学期，原江油市太白中学校长王国金任江油一中校长，依照原太白中学管理措施推出一系列更加严格的管理规定。同时，江油一中也要求学生全年穿全套校服并推出冬季校服让学生购买。这一系列措施被学校师生指责为“形式主义”。
