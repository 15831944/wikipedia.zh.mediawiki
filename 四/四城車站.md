'''四城車站'''位於[[台灣|台灣]][[宜蘭縣|宜蘭縣]][[礁溪鄉|礁溪鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[宜蘭線|宜蘭線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[島式月台|島式月台]]兩座。

== 利用狀況 ==
*目前為乙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]。
*僅停靠區間車。
*本站設有「鐵路之旅-小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1921年2月25日：設「四結乘降場」。
*1964年11月17日：改為「四城」。
*2004年2月1日：由[[三等站|三等站]]降為[[甲種簡易站|甲種簡易站]]，並指定由[[宜蘭車站|宜蘭站]]管理。
*2015年6月30日：啟用多卡通刷卡機。
*2017年9月16日：由[[甲種簡易站|甲種簡易站]]調整為乙種[[簡易站|簡易站]]。
