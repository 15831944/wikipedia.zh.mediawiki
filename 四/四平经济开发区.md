'''四平经济开发区'''是[[吉林省|吉林省]][[四平市|四平市]][[铁东区_(四平市)|铁东区]]于1996年创建的[[经济开发区|经济开发区]]，原名'''四平市铁东经济开发区'''。1998年12月，[[吉林省人民政府|吉林省人民政府]]将之列入吉林省经济技术开发区，更名为四平经济开发区<ref name="a1">{{Cite web |url=http://jkq.siping.gov.cn/jgzn/jkgk/201705/t20170518_98313.html |title=四平经济开发区基本情况 |accessdate=2018-05-30 |author= |date=|work= |publisher=四平经济开发区网站|language=Zh-hans }}</ref>。其辖区亦是铁东区的[[类似乡级单位|类似乡级单位]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|22/03/82/401/000}}}}</ref>。

== 行政区划 ==

下辖3个行政村（{{PRC admin/children|22/03/03/170/000}}<ref>{{PRC admin/ref sgc|22/03/03/170/000}}</ref>），17个自然屯<ref name="a1"/>。
