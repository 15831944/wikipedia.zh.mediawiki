'''四風宮'''（{{lang-pl|Pałac Pod Czterema Wiatrami}}），又稱特佩宮（'''Tepper Palace'''），是位於[[波蘭|波蘭]]首都[[華沙|華沙]]的一座[[洛可可|洛可可]]式宮殿建築。這座建築大約修建於1680年，很有可能是由Tylman van Gameren設計。1944年，德國人在鎮壓華沙起義後故意燒毀了這座宮殿<ref>{{cite web  |author =  |url = http://warszawa.naszemiasto.pl/kontakty_adresy/57657.html |title = Pałac pod Czterema Wiatrami |work = naszemiasto.pl |publisher =  |pages =  |page =  |date =  |accessdate = 2008-02-18|language=pl}}</ref><ref>{{cite web|author =Jerzy Baranowski|url =http://83.12.97.162/gallery4/main.php?g2_itemId=27755|title =Serwis fotograficzny XX wieku|work =fotohistoria.pl|publisher =|pages =|page =|date =|accessdate =2008-02-18}}{{dead link|date=January 2018 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web  |author =  |url = http://www.warszawa1939.pl/index.php?r1=dluga_40&r3=0 |title = Pałac pod Czterema Wiatrami |work = warszawa1939.pl |publisher =  |pages =  |page =  |date =  |accessdate = 2008-02-18|language=pl}}</ref>。
