'''四維國小站'''位於[[臺中市|臺中市]][[北屯區|北屯區]]，為[[臺中捷運|臺中捷運]][[臺中捷運綠線|綠線]]（[[烏日文心北屯線|烏日文心北屯線]]）之[[捷運車站|捷運車站]]<ref>[http://news.ltn.com.tw/news/local/paper/291933 新增3站…臺中捷運綠線確定設18站]，自由時報，2009-04-01</ref>。車站所在地古地名為'''[[二分埔|二分埔]]'''<ref name="市政新聞 站名">{{Cite web |url=https://www.taichung.gov.tw/9962/1030128/post |title=台中捷運綠線18個車站站名出爐 搭乘友善度為命名首要原則 |publisher=[[臺中市政府|臺中市政府]] |work=[[臺中市政府交通局|臺中市政府交通局]] |date=2018-06-30 |archiveurl=https://archive.today/20180701070333/https://www.taichung.gov.tw/8868/8872/1032778/post |archivedate=2018-07-01 |access-date=2018-08-23 |dead-url=no }}</ref>。

== 車站概要 ==
車站位於[[文心路|文心路]]四段與天津路四段路口附近（文心路四段上）；車站代碼為'''G5'''<ref>[http://www.dorts.gov.taipei/ct.asp?xItem=660998&CtNode=20285&mp=115001 臺北市政府捷運工程局 >捷運興建中路網 >臺中捷運烏日文心北屯線 >路線說明]，臺北市政府捷運工程局</ref>。

== 車站構造 ==
高架車站，兩座[[側式月台|側式月台]]，一個出入口。

=== 車站出口 ===
於文心路與天津路交叉路口東南角設有單側出入口，以土地開發方式辦理<ref>[https://www.youtube.com/watch?v=ME65y73Vnbc 土地開發場站G5動畫]，臺中市政府捷運工程處</ref>。另預留未來增設對側出入口之銜接口。
