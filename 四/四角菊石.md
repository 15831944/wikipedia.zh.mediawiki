'''四角菊石'''（[[學名|學名]]：{{lang|la|'''''Tetragonites'''''}}）是生存在[[白堊紀|白堊紀]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]。為[[四角菊石科|四角菊石科]]的模式屬。在弛菊石類中屬於比較先進的類群，於白堊紀的後半部完全取代了傳統的弛菊石類，持續繁衍至白堊紀末。<ref name="Tetragonites">{{Cite web |url = http://liusir.pixnet.net/blog/post/248633027-tetragonites-%E5%9B%9B%E8%A7%92%E8%8F%8A%E7%9F%B3-%28%E6%97%A5%E6%9C%AC-%E5%8C%97%E6%B5%B7%E9%81%93%29 |title = Tetragonites 四角菊石 (日本 北海道) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-10 }}</ref>牠們的[[化石|化石]]分布於[[日本|日本]]、[[北美|北美]]、[[歐洲|歐洲]]和[[南極洲|南極洲]]等地。<ref name="Fossilworks" />
