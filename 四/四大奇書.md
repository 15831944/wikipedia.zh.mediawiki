'''四大奇書'''有數種說法。

== 中國四大奇書（章回小說） ==
中國[[章回小說|章回小說]]《[[水滸傳|水滸傳]]》、《[[三國演義|三國演義]]》、《[[金瓶梅|金瓶梅]]》、《[[西遊記|西遊記]]》的合稱，[[李渔|李渔]]曰：“[[馮夢龍|馮夢龍]]亦有四大奇書之目，曰三國也，水滸也，西游與金瓶梅也。”據[[李渔|李渔]]所說，是由[[明朝|明]]末著名文學家[[馮夢龍|馮夢龍]]所定的。[[清|清]]代前期書店繼續流傳此說法<ref>『「四大奇書」の研究』（小松謙、汲古書院、2010年、ISBN 9784762928857）序。</ref>、[[清|清]]中期[[乾隆|乾隆]]年間（1736年 - 1795年）逐漸確立<ref>『明代小説四大奇書』（浦安迪(Andrew.H.Plaks)、中国和平出版社、1993年）第一章「文人小説的歷史背景」。</ref>[[民國時期|民國時期]]的知名作家[[鲁迅|鲁迅]]，赞同[[馮夢龍|馮夢龍]]的看法。

* 《[[水滸傳|水滸傳]]》（又名《[[江湖豪客水滸傳|江湖豪客水滸傳]]》），[[施耐庵|施耐庵]]著——中國第一部章回小說，以梁山聚义為主題，用[[白話文|白話文]]作為寫作語言。原有一百二十回本，清初[[金聖歎|金聖歎]]刪改成七十回本。
* 《[[三國演義|三國演義]]》（全名《[[三國誌通俗演義|三國誌通俗演義]]》），[[羅貫中|羅貫中]]著——中國第一部長篇[[章回小說|章回小說]]。“四大奇書”之中，以《三國演義》最能深入民心，流傳最廣。
* 《[[西遊記|西遊記]]》（又名《[[西遊釋厄傳|西遊釋厄傳]]》），[[吳承恩|吳承恩]]著——中國古典神魔小說，以[[唐代|唐代]][[玄奘|玄奘]][[法師|法師]]去[[天竺|天竺]]取佛經為故事的源頭。
* 《[[金瓶梅|金瓶梅]]》，（全名《[[金瓶梅詞話|金瓶梅詞話]]》)，[[蘭陵笑笑生|蘭陵笑笑生]]——[[中國|中國]]第一部文人獨立創作的長篇[[白話|白話]][[章回小說|章回小說]]，情節源於[[水滸傳|水滸傳]]。書名取書中人物[[潘金蓮|潘「金」蓮]]、李「瓶」兒、龐春「梅」三人名字各一字而來，清初[[張竹坡|張竹坡]]稱為“第一奇書”。

後人把《金瓶梅》替換為《紅樓夢》，改稱「[[四大小說|四大小說]]」。亦有人把「四大奇書”，加上[[清朝|清朝]]的《[[紅樓夢|紅樓夢]]》、《[[儒林外史|儒林外史]]》合稱為「六大奇書」。

== 日本推理四大奇書 ==
<ref>台湾版『匣の中の失楽』帯参照{{cite web |url=http://www.books.com.tw/exep/prod/booksfile.php?item=0010394047& |title=存档副本 |accessdate=2013年11月13日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090924005814/http://www.books.com.tw/exep/prod/booksfile.php?item=0010394047 |archivedate=2009年9月24日 |df= }}</ref>
<ref>台湾版『匣の中の失楽』に付された島崎博による解説は「四大推理奇書之四・《匣中的失樂》」と題されている。</ref>

[[日本|日本]]的[[推理小說|推理小說]]有三大奇書之說，包含[[夢野久作|夢野久作]]的《[[腦髓地獄|腦髓地獄]]》、[[小栗虫太郎|小栗虫太郎]]的《[[黑死館殺人事件|黑死館殺人事件]]》和[[中井英夫|中井英夫]]的《[[獻給虛無的供物|獻給虛無的供物]]》被稱為「日本三大推理奇書」<ref name="Futagami">日本の推理小説界で「三大奇書」および「四大奇書」という言葉が使われた古い例としては、[[二上洋一|二上洋一]]「日本探偵小説界の三大奇書と言えば、小栗虫太郎創る所の「黒死館殺人事件」、更には、夢野久作これを描く「ドグラ・マグラ」、次いで、塔晶夫こと中井英夫が生み出した「虚無への供物」を指し、更に又、「幻影城」主、[[島崎博|島崎博]]編集長の言葉によれば、若きノヴェリスト竹本健治の「匣の中の失楽」を入れて、日本探偵小説四大奇書と呼ぶのだそうな。」（竹本健治『匣の中の失楽』（双葉文庫、2002年10月）p.663より引用、初出：『幻影城』1978年10月号、初出時は金田一郎名義）</ref><ref>韓国版『虚無への供物』によせた出版社の紹介文参照[http://www.aladdin.co.kr/shop/wproduct.aspx?ISBN=8949705419&]</ref>、「日本異端文學史三大偉業」、「日本偵探小説三大高峰」<ref>[[笠井潔|笠井潔]]「竹本健治の『匣の中の失楽』を読むと、ごく自然に、日本探偵小説の三大巨峰をなす『黒死館殺人事件』、『ドグラ・マグラ』、『虚無への供物』といった作品のことを連想してしまう。」（『週刊読書人』1984年2月6日号より）</ref>、「日本本格偵探小説三大高峰」<ref>韓国版『ドグラ・マグラ』によせた出版社の紹介文参照[http://www.aladdin.co.kr/shop/wproduct.aspx?ISBN=8992723385]</ref><ref>笠井潔「完全犯罪としての作品」（初出：『幻想文学』9号（1984年12月））より（『物語のウロボロス』（筑摩書房、1988年）に所収）</ref>。

*《[[腦髓地獄|腦髓地獄]]》（1935年）
*《[[黑死館殺人事件|黑死館殺人事件]]》（1934年）
*《{{link-ja|獻給虛無的供物|虚無への供物}}》（1964年）

三大奇書後來加上[[竹本健治|竹本健治]]的《[[匣中失樂|匣中失樂]]》，便成為日本推理四大奇書<ref name="kishokanketsu">外部リンク「四大奇書完結--匣中的失樂」参照</ref>。

* 《[[匣中失樂|匣中失樂]]》（1977年）
