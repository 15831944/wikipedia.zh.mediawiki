《'''四处游走'''》<ref>{{cite web|title=魚韻 (sakanaction) >四處遊走(aruku around)|url=https://www.kkbox.com/tw/tc/album/dacZtQRWHs7z0ab0FZV6008l-index.html|website=[[KKBOX|KKBOX]]|publisher=KKBOX Inc.|accessdate=2016-07-26}}</ref>（{{lang-ja|アルクアラウンド}}，[[平文式罗马字|平文式罗马字]]：''Aruku Araundo''，是“Walk Around”的日式发音），又名《'''A Look Around'''》，是[[日本|日本]][[摇滚|摇滚]]乐团[[鱼韵|鱼韵]]2010年的一首歌曲，由主唱[[山口一郎|山口一郎]][[作词|作词]]、作曲。2010年1月6日，《四处游走》一曲首是先小范围地在[[数字音乐下载|数字音乐下载]]服务商平台推出数字格式专辑后，再于1月13日以CD形式广泛发售，成为鱼韵乐队的第二张单曲唱片。该曲于2010年3月17日收录在专辑唱片《[[汽空域|汽空域]]》中。本曲的[[音乐影片|MV]]于夜间在千叶的[[幕张展览馆|幕张展览馆]]录制，正如曲名那样四处游走，以山口一郎进出展览馆的行走为影评中心，以他边跑跳行走边唱歌中推动场面变化，同时MV中将一些歌词具体化成物体，行走带来的镜头角度的转变并利用错视将每个歌词从拆散的部分组合成整体，通过此来呈现歌词，MV中也利用了初期动画的形式展现歌词和动态的走动。

《四处游走》对不同时代舞曲的糅合、作词上体现出的内省和本身的诗意获得了乐评界的广泛好评。特别的是，该歌曲的MV也得到了称赞，在2010年第14届[[文化厅媒体艺术祭|文化厅媒体艺术祭]]娱乐板块上赢得了优秀奖，以及在2010年度SPACE SHOWER MUSIC AWARDS上获得年度最佳音乐影片奖，并获本唱片协会的金唱片认定。本曲对鱼韵而言也是商业上的一个成功，一度在日本[[Oricon公信榜|Oricon公信榜]]上排行前五之内。

==创作背景与发展==
成立于[[北海道|北海道]][[札幌市|札幌市]]的鱼韵乐队，札幌成为了他们最初的创作中心，并在那里发行了两张专辑。考虑到进一步激发乐队的音乐潜能，他们在2009年春来到了[[东京|东京]]，在当年的1月29日，他们在东京发行了首张专辑《[[新白|新白]]》（《Shin-shiro》），也就是在这录制《新白》时，乐队团体们寻得了一个新的创作方式，山口一郎要求乐队的其他成员应独立地创作，而不要像团队那样合作进行。以《Sen to Rei》和《Native Dancer》两首主打歌引领下，乐队在商业上获得了成功，在Oricon周榜上曾排到第8位。鱼韵并因此在当年的多个日本夏季音乐节上登台表演，包括了[[ROCK_IN_JAPAN_FESTIVAL|ROCK IN JAPAN FESTIVAL]]和[[RISING_SUN_ROCK_FESTIVAL|RISING SUN ROCK FESTIVAL]]<ref>{{cite web |url=http://sakanaction.jp/live/ |script-title=ja:Live |publisher=Hip Land Music Corporation |accessdate=2016年2月27日 |archiveurl=https://web.archive.org/web/20090417010147/http://sakanaction.jp/live/ |archivedate=2009年4月17日 |deadurl=yes }}</ref><ref>{{cite web |url=http://natalie.mu/music/news/16315 |script-title=ja:ライジングサン第2弾でユニコーン、ユアソンら4組追加 |publisher=ナターシャ |work=ナタリー |date=May 11, 2009 | accessdate=2016年2月27日}}</ref>。

《新白》发行后的2009年2月14日，鱼韵开启了帶有纪念性质的日本全国巡回演出：“Sakanaquarium 2009: Shinshiro”（{{lang|ja|『SAKANAQUARIUM 2009“シンシロ”』}}），在结束全国巡回演出后山口一郎便立刻开始了《四处游走》的创作<ref name="MusicShelfAruku">{{cite web | name="SSRtour">{{cite web | url=http://www.barks.jp/news/?id=1000046659 |script-title=ja:ブレイクに期待、オリコン初登場8位のサカナクション |publisher=Japan Music Network Inc.|date=2009年1月29日|accessdate=2016年2月27日}}</ref>。山口一郎曾表示也他对这《新白》最终销量止于三万这一数字而失望.<ref name="RockinOnKikuRetro">{{cite web | url=http://ro69.jp/feat/sakanaction_201109?p=4 | script-title=ja:特集　サカナクション | archivedate=2015年4月16日 }}</ref>，为此山口一郎意识到鱼韵需要一首具有识别性、有鲜明特征的歌曲，并因此而改变听众们对他们乐队的看法<ref name="MusicShelfAruku"/><ref name="HotExpress2"/>。山口一郎也感受到能令听众觉得能代表一个乐队的歌曲一定要是纪实类型的，于是山口一郎在结束《新白》的演出后将自己的感情融入了创作。也是在此时他才觉得东京才是他现在的家。乐队为了扩大受众面，在乐曲中将更强的现代流行音乐要素和80年代[[舞曲|舞曲]]元素和60年代的[[民俗音乐|民俗音乐]]要素融合其中，并兼有了电子（Electro）和摇滚乐元素<ref name="HotExpress2">{{cite web |url=http://www.hotexpress.co.jp/interview/100113_sakanaction/page2.html |script-title=ja:サカナクション インタビュー |author=平賀哲雄 |publisher=2011Plantech co.,ltd |date=2010年1月13日 }}</ref><ref>{{cite web|title=魚韻 (sakanaction) > 汽空域(kikUUiki)|url=https://www.kkbox.com/tw/tc/album/NYHRycT1u9Fbhv-0FZV5008l-index.html|website=[[KKBOX|KKBOX]]|publisher=KKBOX Inc.|accessdate=2016-07-26}}</ref>。将电子舞曲与摇滚相结合的这一做法也是山口一郎与也是乐队的电子琴师岡崎英美为乐队构创立一个基本约定<ref name="OfficialAruku">{{cite web | url=http://www.jvcmusic.co.jp/chvi/sakanaction2/ |script-title=ja:特集｜注目アーティスト：サカナクション |author=森朋之|publisher=Victor Entertainment |year=2010年 | accessdate=2016年7月27日}}</ref>。随后他们在各大夏季音乐节上使用被当作实验品的这首歌来参演，来测试一下听人群的反应，并根据听众们的反馈在原歌的基础上做一些修改<ref name="ExciteAruku">{{cite web | url=http://www.excite.co.jp/music/close_up/interview/1001_sakanaction/ |script-title=ja:サカナクション   インタビュー|publisher=Excite Japan |date=2010年1月15日 | ==专辑封面==
专辑封面由Hatos团队设计，前专辑《Night Fishing》也是由他们设计的。山口一郎希望Hatos这次的封面设计带有“日本文学的迷幻（Psychedelia）感”<ref name="HotExpress3">{{cite web | url=http://www.hotexpress.co.jp/interview/100113_sakanaction/page3.html |script-title=ja:サカナクション インタビュー |publisher=Hot Express |date=2010-01-13 | accessdate=2015-04-21}}</ref>。封面呈现的是乐队贝斯手草刈爱美<ref name="SakanarchiveBooklet">{{cite AV media notes |script-title=ja:Sakanarchive 2007—2011: Sakanaction Music Video Collection (Limited Edition) |others=鱼韵乐队 |year=2011年 |language=日语 |publisher=Victor Entertainment |location=日本东京}}</ref>的头部、仿佛是1980年代风格鲜艳的色调、印有《四处游走》歌词的蓝色（{{Colorbox|#16b0f8}}）覆盖了该女性的面容<ref name="CoverAnnounce">{{cite web | url=http://www.barks.jp/news/?id=1000055602 |script-title=ja:サカナクション、シングル初回盤にツアー先行予約シリアルナンバー封入|publisher=Barks |date=2009-11-21 | accessdate=2017-07-01}}</ref>。

在专辑封面设计上附有歌词是山口一郎不曾想到的，因著作权的问题，专辑封面不能发布在杂志与网络上，所以又乐队不得不发布了没有歌词并附带因缺少歌词而致歉的特绘封面<ref name="CoverAnnounce"/>。

==音乐影片==
[[File:Makuhari_Messe,_North_hall_1.jpg|thumb]]
《四处游走》的音乐影片由关和亮执导，影片和人物的艺术设计全由北泽“momo”寿志负责完成。音乐影片是夜间使用[[佳能_EOS_5D_Mark_II|佳能 EOS 5D Mark II]]在千叶的[[幕张展览馆|幕张展览馆]]拍摄的<ref name="SekiInterview">{{cite web |url=http://white-screen.jp/?p=3325 |script-title=ja:サカナクションからPerfumeまで。映像作家、関和亮の演出術 |publisher=WhiteScreen |date=2010-03-31 |accessdate=2017-07-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20161130054951/http://white-screen.jp/?p=3325 |archivedate=2016-11-30 }}</ref><ref name="FestivalWin">{{cite web |url=http://plaza.bunka.go.jp/festival/2010/entertainment/arukuaround/ |script-title=ja:“2010年 文化庁メディア芸術祭 エンターテインメント部門 優秀賞 アルクアラウンド／サカナクション 文化庁メディア芸術プラザ |archiveurl=https://web.archive.org/web/20110706124054/http://plaza.bunka.go.jp/festival/2010/entertainment/arukuaround/ |publisher=Japan Media Arts Plaza |archivedate=2011-07-06 |accessdate=2017-07-01 |deadurl=yes }}</ref>。影片中山口一郎一边行走或跑跳一边唱歌的方式进出于展览馆。MV中将一些歌词印刷体化、具体化。行走带来的镜头角度的转变通过利用错视将每个歌词从拆散的部分组合成成整体，通过此来呈现歌词<ref name="SekiInterview" /><ref name="FestivalWin" /><ref name="CDJM">{{cite web | url=http://www.cdjournal.com/main/news/sakanaction/28290 |script-title=ja:サカナクション「アルクアラウンド」ミュージック・ビデオ公開！ |publisher=CDJournal |date=2009-11-28 | accessdate=2017-07-01}}</ref>。在MV中乐队的其他成员也有登场，他们通过[[手翻书|快速翻页]]、电脑荧屏、鱼缸和传送带等多种手段呈现出歌词<ref name="FestivalWin"/>。

听后关和亮想制作一部音乐影片来更好展现歌词，并注意到歌词都完全是日语。于是在一次与乐队访谈中他专门谈到将歌曲的歌词视觉化，并提出使用他曾了解到的一镜到底拍摄手法。除开山口一郎的要求展现夜与暗两个元素外，乐队完全放手让关和亮去做。<ref name="SpaceShowerInterview">{{cite web | url=http://mva.jp/interview_bvideo.php | script-title=ja:INTERVIEW - サカナクション |archiveurl=https://web.archive.org/web/20110415083617/http://mva.jp/interview_bvideo.php |publisher=Space Shower |archivedate=2011-04-15| accessdate=2017-07-01}}</ref>。

== 评价 ==
===音乐评论家的评价===
《四处游走》获得了日本国内多数音乐评论家的好评。音乐杂志《CD Journal》给了这首歌一颗五角星，认为这是一首“刹拉间的快乐感、深厚的思虑和足以'沉醉倒下'的曲调共存的”乐曲。
