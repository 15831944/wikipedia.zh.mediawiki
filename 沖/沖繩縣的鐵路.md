'''沖繩縣的鐵路'''現在由[[沖繩縣|沖繩縣]]營運，和過去的鐵路相同。

== 概要 ==
沖繩縣的軌道運輸系統[[沖繩都市單軌電車線|沖繩都市單軌電車線]]於2003年在[[那霸市|那霸市]]展開營運，是一座[[單軌鐵路|單軌鐵路]]。（1945至2003年是日本唯一沒有鐵路的行政區）這是目前沖繩縣唯一的鐵道運輸，但在二戰前的[[沖繩島|沖繩島]]，便已經有[[輕便鐵路|輕便鐵路]]、[[有軌電車|有軌電車]]和{{link-ja|馬車鐵路|馬車鉄道}}存在。此外，為了搬運[[甘蔗|甘蔗]]，[[南大東島|南大東島]]的各地也存在產業鐵道。
== 歷史 ==
=== 戰後 ===
戰後失去了鐵路運輸，道路的建設加速推進，但是到了1970年代經濟活動活躍，道路日益堵塞。

== 現在的鐵路路線 ==

=== 沖繩都市單軌電車線 ===
戰後失去了鐵路運輸，道路的建設加速推進，但是到了1970年代經濟活動活躍，道路日益堵塞。作為對策，2003年8月[[沖繩都市單軌電車線|沖繩都市單軌電車線]]開始營運。並計畫2020年開建[[沖繩鐵路|沖繩鐵路]]。

== 未來 ==
