'''亞爾諾'''（''Arnor''），或譯'''雅諾'''，是[[英國|英國]]作家[[約翰·羅納德·瑞爾·托爾金|約翰·羅納德·瑞爾·托爾金]]的《[[魔戒|魔戒]]》登史詩式奇幻小說裡，位於[[中土大陸|中土大陸]]北方[[伊利雅德|伊利雅德]]地區的[[登丹人|登丹人]]王國。“亞爾諾”的意思是「君王的领土」（''Land of the King''），由[[辛達林語|辛達林語]]的''Ara-''（高等''high,kingly''）和''dor''（土地''land''）結合而成。
[[File:Enedwaith.jpg|right]]

== 歷史 ==

亞爾諾和[[剛鐸|剛鐸]]於[[第二紀元|第二紀元]]3320年由[[伊蘭迪爾|伊蘭迪爾]]和他的兩個兒子[[埃西鐸|埃西鐸]]及[[安那瑞安|安那瑞安]]共同建立的。二個王國歷史交錯，為人所知兩國都是登丹人的流亡國度。

在亞爾諾建立之前，已經有規模可觀的[[努曼諾爾|努曼諾爾]]殖民人口居住在那裡，他們是在[[努曼諾爾帝國皇帝|努曼諾爾帝國皇帝]][[塔爾-曼奈德|塔爾-曼奈德]]和[[塔爾-阿達瑞安|塔爾-阿達瑞安]]在位時期開始移民到達亞爾諾。但登丹人到達之時，與登丹人同一祖先的當地居民[[伊甸人|伊甸人]]後裔也在，他們並且與早期殖民結合，很快混和出土產人口。

亞爾諾最初傾向在南部的地區和剛鐸邊界附近建立勢力，因為[[諾多族|諾多族]]最高君王[[吉爾加拉德|吉爾加拉德]]居住在[[隆恩河|隆恩河]]''Lhûn''。但是，[[索倫|索倫]]的陰影安頓在南部，這導致了人口往北遷移，亞爾諾人不同於剛鐸人，他們仍然是精靈的主要朋友，更多的知識被保存了。

亞爾諾的第二任國王埃西鐸（並且同時是剛鐸國王）於[[第三紀元|第三紀元]]2年被[[半獸人|半獸人]]殺害，當埃西鐸到[[格拉頓平原|格拉頓平原]]，被半獸人伏擊，在危急之際埃西鐸戴上[[至尊魔戒|至尊魔戒]]隱形，但至尊魔戒背叛了他，至尊魔戒變大滑落了他的手指，埃西鐸因此被發現而亂箭射死，他的三個兒子也被杀死，留在[[瑞文戴爾|瑞文戴爾]]的幼子[[瓦蘭迪爾|瓦蘭迪爾]]繼承了亞爾諾和登丹人最高君王的王位。但由於瓦蘭迪爾的疏忽，他和其後的承繼人一直沒有要求奪回剛鐸王位，南北王國因此分裂，但亞爾諾的統治者保留了登丹人最高君王名銜，雖然南方王國的實力遠比亞爾諾強大。

== 衰落 ==

亞爾諾原先的首都是[[伊凡丁湖|伊凡丁湖]]''Lake Evendim''旁的[[安努米那斯|安努米那斯]]，但在[[第三紀元|第三紀元]]861年遷都於[[佛諾斯特|佛諾斯特]]，安努米那斯人口漸漸減少。

在亞爾諾的第十任國王[[艾蘭多|艾蘭多]]在第三紀元861年駕崩以後，他的三個兒子之間互相爭位，引發內戰。長子[[艾姆拉斯|艾姆拉斯]]要求亞爾諾王位，但最後-{只}-能統治[[雅西頓|雅西頓]]地區作為他的王國，另外兩個兄弟則建立了[[卡多蘭|卡多蘭]]和[[魯道爾|魯道爾]]王國。

重新稱呼自己為亞爾諾國王的是雅西頓國王[[亞瑞吉來布一世|亞瑞吉來布一世]]，卡多蘭向雅西頓稱臣。但是，雅西頓最終還是被毀壞了，成了廢墟。加上國土曾遭到東北安格馬邪惡勢力的入侵，亞爾諾的人民漸漸消亡，不過王國名義上的附庸[[哈比人|哈比人]]並不受影響，依然繼續在[[夏爾|夏爾]]過著與世無爭的悠閒生活。而另一些人類，則改為生活在如[[布理|布理]]的村莊中，亞爾諾僅餘的少量的登丹人移居到瑞文戴爾南部，成為行蹤飄忽的[[北方遊俠|北方遊俠]]。

== 重聯王國 ==

登丹人酋長[[亞拉岡二世|亞拉岡二世]]，即剛鐸的[[伊力薩|伊力薩]]王，統一了剛鐸及亞爾諾，成為[[重聯王國|重聯王國]]國王。他重新下令阿努米那斯成為名義上的首都，不過伊力薩王自己依然居住在剛鐸，亞爾諾人口從此再度增長，不過領土由於夏爾的獨立而有所減少。
