'''雅安市'''是[[中华人民共和国|中华人民共和国]][[四川省|四川省]]下辖的[[地级市|地级市]]，位于四川省中部偏西。市境东邻[[成都市|成都市]]、[[眉山市|眉山市]]、[[乐山市|乐山市]]，南界[[凉山州|凉山州]]，西接[[甘孜州|甘孜州]]，北临[[阿坝州|阿坝州]]。地处[[四川盆地|四川盆地]]西南边缘[[大相岭|大相岭]]区，[[青衣江|青衣江]]横贯中部并流经城区，南部有[[大渡河|大渡河]]。总面积1.5万平方公里，人口154.68万。雅安因其地多雨，素有“'''雨城'''”之称，亦是四川省主要的[[大熊猫|大熊猫]]栖息地。雅安曾是[[西康省|西康省]]省会，位处四川盆地与[[青藏高原|青藏高原]]的结合部，境内有[[藏族|藏族]]、[[彝族|彝族]]等[[中国少数民族|少数民族]]聚居，有“川西咽喉”、“西藏门户”及“民族走廊”之称。

== 历史 ==
[[File:Gao_Yi_Que5.jpg|thumb]]

古代为[[青衣羌国|青衣羌国]]，[[秦|秦]]置[[严道县|严道县]]（含今[[荥经县|荥经县]]、[[雨城区|雨城区]]）属[[蜀郡|蜀郡]]。[[西汉|西汉]][[元鼎|元鼎]]六年（前111年）置增置[[徙县|徙县]]（今[[天全县|天全县]]）、[[旄牛县|旄牛县]]（今[[汉源县|汉源县]][[九襄镇|九襄镇]]）、[[青衣县|青衣县]]（今[[名山区|名山区]]、[[芦山县|芦山县]]境），先后属[[沈黎郡|沈黎郡]]、[[蜀郡|蜀郡]]。[[东汉|东汉]][[延光|延光]]元年（122年）置[[蜀郡属国|蜀郡属国]]，治青衣县，后青衣羌请求内附。[[阳嘉|阳嘉]]二年（133年），改青衣县为[[汉嘉县|汉嘉县]]。[[三国|三国]][[蜀汉|蜀汉]][[章武|章武]]元年（221年）改蜀郡属国置[[汉嘉郡|汉嘉郡]]，治汉嘉县（今芦山县），属益州。[[西晋|西晋]][[永嘉|永嘉]]六年（312年），汉嘉郡分为[[沈黎郡|沈黎郡]]、[[汉原郡|汉原郡]]。永嘉后废严道县。[[西魏|西魏]]废帝二年（553年）置[[始阳县|始阳县]]（治今雨城区西），属[[蒙山郡|蒙山郡]]；又置[[蒙山县|蒙山县]]（治今名山区），隶蒙山郡。

[[隋朝|隋]][[开皇|开皇]]初废蒙山郡。开皇十三年（593年）改故蒙山郡置[[蒙山县|蒙山县]]；改故蒙山县为[[名山县|名山县]]。[[仁寿_(隋)|仁寿]]三年（603年）置[[卢山县|卢山县]]（治今芦山县）。仁寿四年（604年）于蒙山县置[[雅州|雅州]]（治今雨城区）。[[大业|大业]]元年（605年）置[[汉源县|汉源县]]（治今汉源县[[清溪镇|清溪镇]]），属雅州。[[大业|大业]]三年（607年）改雅州为[[临邛郡|临邛郡]]，改蒙山县为[[严道县|严道县]]，并于今故严道县城（今荥经县）设荥经水口戍。

[[唐朝|唐]][[武德|武德]]元年（618年）改临邛郡为雅州，改汉源县置[[阳山县|阳山县]]。武德三年（620年）于荥经水口戍置[[荥经县|荥经县]]，属雅州。[[贞观|贞观]]八年（634年）析名山县置[[百丈县|百丈县]]，属雅州。[[仪凤|仪凤]]二年（677年）析阳山县置[[大渡县|大渡县]]、[[飞越县|飞越县]]。[[长安_(年号)|长安]]二年（702年）省大渡县入飞越县。[[天宝|天宝]]元年（742年）改阳山县为[[通望县|通望县]]，属[[黎州|黎州]]。[[五代|五代]][[前蜀|前蜀]]省飞越县入汉源县。

[[北宋|北宋]][[庆历|庆历]]六年（1046年）省通望县入汉源县。[[元朝|元]]置[[碉门安抚司|碉门安抚司]]（治今[[天全县|天全县]]），后又分置[[天全招讨司|天全招讨司]]、[[六番招讨司|六番招讨司]]。[[至元|至元]]二十年（1283年）卢山县改名[[泸山县|泸山县]]。元末，[[明玉珍|明玉珍]][[大夏|大夏]]政权省荥经县入严道县。

[[明朝|明]][[洪武|洪武]]四年（1371年）省严道县入雅州。洪武六年（1373年）泸山县更名[[芦山县|芦山县]]，隶雅州；合天全、六番招讨司置[[天全六番招讨司|天全六番招讨司]]。洪武八年（1375年）省汉源县置[[黎州长官司|黎州长官司]]。洪武十年（1377年）省名山县入雅州。洪武十一年（1378年）升黎州长官司为[[黎州安抚司|黎州安抚司]]。洪武十三年（1380年）复置名山县，并省百丈县入名山县；复置荥经县，属雅州。[[万历|万历]]二十四年（1596年）降黎州安抚司为千户所。

[[清朝|清]][[雍正|雍正]]七年（1729年）升雅州置[[雅州府|雅州府]]，置[[雅安县|雅安县]]为雅州府治；天全六番招讨司“改土归流”置[[天全州|天全州]]，属雅州府。雍正八年（1730年）黎州千户所“改土归流”置[[清溪县|清溪县]]，属雅州府。

[[民国|民国]]二年（1913年）废府州改县，雅安、名山、荥经、芦山、清溪、天全等县属[[上川南道|上川南道]]（次年更名[[建昌道|建昌道]]）。1914年清溪县复名[[汉源县|汉源县]]。1928年析天全县北部穆坪土司地置[[穆坪设治局|穆坪设治局]]。民国十九年（1930年）改穆坪设治局置[[宝兴县|宝兴县]]，县名取“宝藏兴焉”之意；同年废建昌道。民国二十四年（1935年）各县属四川省第十七行政督察区。民国二十八年（1939年）设立[[西康省|西康省]]，除名山县改属四川省第四区外，其馀各县划入西康省第二行政督察区。

[[中华人民共和国|中华人民共和国]]成立后仍置西康省，雅安县为西康省省会，并置[[雅安专区|雅安专区]]。1950年汉源县迁治[[九襄镇|九襄镇]]。1951年析雅安县置雅安市；析[[冕宁|冕宁]]、[[越西|越西]]两县北部及汉源县西部置[[石棉县|石棉县]]。1952年汉源县迁治[[富林镇|富林镇]]。1954年[[温江专区|温江专区]]的名山县划入雅安专区。1955年西康省撤销，雅安专区改属四川省。1959年撤雅安市入雅安县。1968年雅安专区改称[[雅安地区|雅安地区]]。1983年撤销雅安县，复置雅安市。2000年6月，撤销雅安地区和县级雅安市，改设地级[[雅安市|雅安市]]；原县级雅安市改置雨城区。2012年9月，撤销名山县，设立[[名山区|名山区]]。

== 地理 ==
雅安市位于[[四川盆地|四川盆地]]西部边缘，[[长江|长江]]上游，域跨北纬28°51′10″—30°56′40″，东经101°56′26″—103°23′28″之间，东北邻[[成都市|成都市]]、东邻[[眉山市|眉山市]]、东南邻[[乐山市|乐山市]]、西连[[甘孜藏族自治州|甘孜藏族自治州]]、南界[[凉山彝族自治州|凉山彝族自治州]]、北接[[阿坝藏族羌族自治州|阿坝藏族羌族自治州]]，是[[青藏高原|青藏高原]]向[[成都平原|成都平原]]的过渡地带。城市环境特别好，青山环绕，四条河流穿过市区，使空气清新。

=== 气候 ===
雅安自古多雨，有西蜀天漏之说，俗语“雅无三日晴”，号称雨城。

雅安市属于[[亚热带季风气候|亚热带季风气候]]区。总的气候特点是：冬无严寒、夏无酷热、四季分明、雨量充沛、雨热同步、无霜期长、热量充足。春季回暖早、夏季气温较高、秋季多绵雨、冬季霜雪少，全年夜雨多、终年雾日少<ref>[http://orientaldaily.on.cc/cnt/news/20130421/00176_002.html 雅安三絕：雅雨 雅魚 雅女] 《東方日報》 2013年4月21日</ref>。境内地质地貌复杂，地势高差悬殊，气候的区域性、过渡性和复杂性特征突出。气象要素的垂直差异大，立体气候明显，部份山体从山脚到山脊先后可见亚热带、暖温带、中温带、寒温带、亚寒带、和寒带的植物。[[大相岭|大相岭]]（即泥巴山）横贯于本市中部，使得南北部的气候差异大。北部各县、区，雨日多、晴天少，年平均气温在15℃左右，年总雨量在1000～1750毫米之间；南部的[[汉源|汉源]]和[[石棉县|石棉县]]受中、低空下沉气流影响形成比较独特的干暖气候，云量少，晴天多，日照时间较长，气温较高，年均温为17～18℃，干湿季节分明，降水量较少，年总雨量仅750毫米左右。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为1507264人<ref>{{cite book |author=雅安市统计局|title=《雅安市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共减少15581人，减少1.02%。年平均减少0.1%。其中，男性人口为766605人，占50.86%；女性人口为740659人，占49.14%。总人口性别比（以女性为100）为103.50。0－14岁人口为243947人，占16.18%；15－64岁人口为1099336人，占72.94%；65岁及以上人口为163981人，占10.88%。

== 交通 ==
公路方面，[[318国道|318国道]]、[[108国道|108国道]]穿境而过，另有[[成雅高速公路|成雅高速公路]]、[[雅西高速公路|雅西高速公路]]、[[雅康高速公路|雅康高速公路]]、[[成渝地區環線高速公路|成渝地區環線高速公路]]樂雅段分别通向[[成都市|成都市]]、[[西昌市|西昌市]]、[[瀘定縣|瀘定縣]]、[[樂山市|樂山市]]。距成都140公里，到[[成都双流国际机场|成都双流国际机场]]仅需1小时40分。

铁路方面，在建[[川藏铁路|川藏铁路]]成雅段预计2018年底通车。

== 教育 ==
高校有[[四川农业大学|四川农业大学]]。中学以[[雅安中学|雅安中学]]著称，该校从清朝开始建立，是一所省重点名校。

== 人文 ==
[[File:YaanBridgeOldBridge2.jpg|thumb]]
雅安[[两汉|两汉]]文化历史底蕴丰厚，素有雅雨、雅鱼、雅女“雅安三绝”美誉。雅安是“中国优秀旅游城市”、2006年的“中国十佳魅力城市”，是全省乡村度假旅游和自驾车旅游的热点地区。雅安是世界上第一只[[大熊猫|大熊猫]]的发现地和模式标本产地，2006年，以雅安为主要核心区的四川大熊猫栖息地列入世界自然遗产名录；雅安是世界茶文化、茶栽培的发源地，是全球人工栽培[[茶树|茶树]]最早的地区。
