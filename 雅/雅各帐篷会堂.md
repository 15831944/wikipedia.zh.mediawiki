'''雅各帐篷会堂'''（[[希伯来语|希伯来语]]: Ohel Jakob）是[[德国|德国]][[慕尼黑|慕尼黑]]的[[犹太会堂|犹太会堂]]，建于2004-2006年。为慕尼黑犹太社区新的主要会堂，位于圣雅各广场（Sankt-Jakobs-Platz）。  犹太会堂于2006年11月9日在[[水晶之夜|水晶之夜]]68周年纪念日开幕。<ref>{{cite web |url=http://www.dw.de/new-munich-synagogue-opens-on-nazi-persecution-anniversary/a-2230576-1 |title=New Munich Synagogue Opens on Nazi Persecution Anniversary |work=Die Welt |date=2006-11-09 |accessdate=2014-01-31}}</ref>它是新的犹太中心的一部分，包括犹太会堂、[[慕尼黑犹太博物馆|慕尼黑犹太博物馆]]和一个社区中心。

== 建筑 ==
2001年7月6日建筑设计大赛后，建筑师Rena Wandel-Hoefer 和沃尔夫冈Lorch获得了雅各的帐篷合同。<ref>{{cite web |url=http://www.juedischeszentrumjakobsplatz.de/wettbewerbe.html |title=Wettbewerbe |work=Jüdisches Zentrum Jakobsplatz München |accessdate=2014-01-31|language=de}}</ref> 建筑师以前完成了[[德累斯顿|德累斯顿]][[新犹太会堂_(德累斯顿)|新犹太会堂]]。隆重仪式于2005年10月25日举行。开幕式的主角是德国犹太人中央委员会主席和慕尼黑正统犹太社区的负责人夏洛特 Knobloch。 

该建筑物是一个立方体混凝土建筑，下部为[[石灰华|石灰华]]石头贴面，顶部为玻璃立方体。玻璃屋顶代表帐篷，象征[[摩西|摩西]]穿越沙漠40年的旅程。大门是在[[布达佩斯|布达佩斯]]制作的，上面有[[希伯来字母|希伯来字母]]的[[十诫|十诫]]。内墙上镶着松柏装饰着金色的诗篇。

犹太教会堂可以容纳550信徒。它的造价约5700万欧元 (约7200万美元) ，资金来源包括慕尼黑市、[[巴伐利亚州|巴伐利亚州]]、慕尼黑的犹太社区以及私人捐赠。<ref name="HopeWP">{{cite web |url=https://www.washingtonpost.com/wp-dyn/content/article/2006/11/09/AR2006110900937.html |title=New Munich Synagogue Symbolizes Hope |work=[[Washington_Post|Washington Post]] |date=9 November 2006 |accessdate=31 January 2014}}</ref>

慕尼黑原来的主要犹太会堂于1938年6月被摧毁，位于离新犹太会堂不远的几个街区，原址现在上是一个停车坡道。
