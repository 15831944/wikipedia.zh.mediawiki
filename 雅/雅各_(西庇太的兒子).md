'''西庇太的儿子雅各'''（{{link-ja|聖經希伯來語|Biblical Hebrew}}：{{rtl-lang|he|יַעֲקֹב}} ''<sup><small>Yaʿqob</small></sup>''，[[通用希臘語|通用希臘語]]：{{lang|el|Ἰάκωβος}}，{{lang-cop|ⲓⲁⲕⲱⲃⲟⲥ}}）是[[耶稣十二门徒|耶稣十二门徒]]之一，[[天主教|天主教]]称'''载伯德的儿子雅各伯'''、'''圣雅各伯'''或'''聖長雅各伯宗徒'''，以区别于[[亚勒腓的儿子雅各|亚勒腓的儿子雅各]]。

是[[西班牙|西班牙]]、士兵、朝圣者、骑手的[[主保圣人|主保圣人]]，相传其遗骨葬于[[圣地亚哥-德孔波斯特拉|圣地亚哥-德孔波斯特拉]]，通往該城的[[聖雅各之路|聖雅各之路]]是[[朝圣|朝圣]]热点。

== 蒙召的经过 ==

当[[彼得|彼得]]（伯多禄）把船借给耶稣用，并顺服主的指示下网，圈住许多[[鱼|鱼]]时，雅各被招呼前去帮助，把鱼装满了两条[[船|船]]，在惊讶之时，听见主呼召的应许：“要得人如得鱼”，他便与彼得他们一同撇下所有的，跟从主耶稣，作得人的渔夫（[[路加福音|路加福音]]5章：4-11）。

== 一生的事迹 ==

# 个性的改变：雅各本来的个性是相当猛烈如暴躁的，如有一次因[[撒玛利亚|撒玛利亚]](撒玛黎亚)的一个村庄的人不肯接待耶稣，他就向主建议，求主从天上降火下来，烧灭这些对神的儿子侮慢的人，如古时[[以利亚|以利亚]]求神用火烧死那轻慢先知的人一样（路9：51-54），所以耶稣为他取个外号，叫半尼其（Boanerges），就是雷子的意思（[[马可福音|马可福音]]3：17）；但后来因着耶稣的训诲和[[圣灵|圣灵]]内住之改变，成为一个能接纳，宽容人的使徒。
# 最先殉道：[[五旬节|五旬节]]圣灵降临之后，使徒们勇敢为耶稣作见证，神迹显明，信徒相爱，使很多人归信耶稣，但更引起很多人的怨恨，而逼迫，阻挡门徒。聖經記載希律王（傳統上認為是[[希律亚基帕一世|希律亚基帕一世]]）用刀處決了雅各（[[使徒行传|使徒行传]]12：1-2） 。雅各成为第一位[[殉道|殉道]]的使徒。

== 身份 ==

母亲[[撒罗米|撒罗米]]（撒罗默）是[[耶稣|耶稣]]的母亲[[马利亚|马利亚]]（玛利亚）的妹妹，传说她在丈夫[[西庇太|西庇太]]死后也跟随了[[耶稣|耶稣]]。雅各、[[门徒约翰|约翰]]两兄弟在跟随[[耶稣|耶稣]]之前都曾经是[[施洗约翰|施洗约翰]]的门徒，因脾气暴躁，两兄弟皆被耶稣称为雷霆之子。他们对于「[[以色列|以色列]]复兴」、「挣脱[[罗马帝国|罗马帝国]]的殖民统治」以及「恢复[[大卫|大卫]]（达味）王朝的光荣」有着高度的热忱。这也是他们跟随[[施洗约翰|施洗约翰]]或[[耶稣|耶稣]]的原始动机。直到他们目睹复活的[[基督|基督]]以后，才激发他们宣扬福音的意识与使命。 <ref>董芳苑，《耶稣和十二使徒》，（台北：长青文化事业股份有限公司，2001），33-40。 </ref>

== 个人标志 ==

圣雅各的贝壳（[[扇贝|扇贝]]）：朝圣者把它装饰在头发或者衣服上，是长途旅行的标志和见证。
