'''雅叶高速公路'''，简称'''雅叶高速'''是[[国家高速公路网|国家高速公路网]]东西横线中的一条，属于远期展望线，由[[四川|四川]][[雅安|雅安]]至[[新疆|新疆]][[叶城|叶城]]，线路主要经过以下地区：[[雅安|雅安]]－[[天全|天全]]－[[泸定|泸定]]－[[康定|康定]]－[[理塘|理塘]]－[[芒康|芒康]]－[[八宿|八宿]]－[[林芝|林芝]]－[[拉萨|拉萨]]－[[日喀则|日喀则]]－[[噶尔|噶尔]]－[[叶城|叶城]]。<ref> {{Cite web |url=http://news.dichan.sina.com.cn/2013/07/23/804732.html |title=国家发布今后17年公路规划 新疆境内将新增9条国道5条高速 |author=  |date= 2013年7月23日|publisher= 新浪网 |language= 中文 |accessdate=  |quote= }}</ref>

== 雅安—康定段 ==
'''雅康高速'''是位于[[四川|四川]]西部[[雅安|雅安]]到[[康定|康定]]的一条[[高速公路|高速公路]]，是[[国家高速公路网|国家高速公路网]]雅安至[[叶城|叶城]]联络线中的一段，也是[[四川高速公路|四川省高速公路网]]“16·5·5”规划中5条东西横线之一康定至[[泸州|泸州]]的重要组成部分。路线起于[[雅安市|雅安市]][[草坝镇_(雅安市)|草壩鎮]]，接已建的{{China Expwy Name|G93}}[[樂山|樂山]]ー[[雅安|雅安]]段，在[[对岩镇|對岩鎮]]與[[京昆高速公路|京昆高速公路]]互通<ref>{{Cite web|url=https://sichuan.scol.com.cn/ggxw/201712/56053613.html|title=四川藏区首条高速来了！成都2.5小时到泸定 雅康高速雅安至泸定段31日试通车运行|accessdate=2018-10-03|author=|date=2017-12-28|publisher=四川在線}}</ref>，经[[天全县|天全县]]、[[泸定县|泸定县]]，止于康定县菜园子。路线全长约133公里，其中[[甘孜州|甘孜州]]境内约55公里，全程设计速度80公里/小时，路基宽24.5m，投资估算约220亿元。2017年12月31日，雅康高速泸定段正式试通行。<ref>{{Cite web|url=http://sc.sina.com.cn/news/m/2018-01-01/detail-ifyqcwaq6495688.shtml|title=雅康高速泸定段试通车 目前仅允许蓝牌照和小型新能源车通行|accessdate=2018-06-09|author=|date=2018-01-01|publisher=成都商报}}</ref>2018年12月31日，雅康高速公路全线建成并试通车运营<ref>[http://www.xinhuanet.com/fortune/2018-12/31/c_1123931063.htm 川藏“新干线”雅康高速公路全线建成并试通车]</ref>。

线路全长133km，设计时速80km。[[二郎山隧道_(雅康高速)|二郎山隧道]]为本段线路的控制工程，全长13.4km，于2012年12月开工。<ref>{{Cite web|url=http://news.beiww.com/2013/0916/article_126843.html|title=雅康高速 总投资235.9亿元 2017年建成|author=|date=2013年9月16日|publisher=北纬网|language=中文|accessdate=|quote=}}</ref>

== 未来发展 ==
{{SC Future infrastructure}}
康定至新都桥段預計於2018年年底開工，工期5年。線路全長78.8千米，設計時速80千米/時。其中63座橋樑總長6.9千米，佔線路全線長度8.9%。線路亦設有11座隧道，全長43千米，佔線路總長度之55.1%。線路初步規劃3處[[交流道|互通立交]]。工程預計投資137.7億圓[[人民币|人民幣]]，平均1千米1.7億圓。<ref>{{Cite web|url=https://sichuan.scol.com.cn/dwzw/201807/56375013.html|title=康定至新都桥高速正式招标 今年底可能开工_天府要闻_四川新闻_四川在线|accessdate=2018-10-18|work=sichuan.scol.com.cn}}</ref>線路其余路线仅为远期展望。
