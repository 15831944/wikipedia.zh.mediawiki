'''雅克·布弗莱斯'''（{{lang-fr|Jacques Bouveresse}}，1940年－），法国维也纳学派、语言哲学和分析哲学专家，[[法兰西学院|法兰西学院]]语言和知识讲席教授。理性主义的捍卫者，[[日内瓦大学|日内瓦大学]]杰出人物。研究兴趣广泛（[[维特根斯坦|维特根斯坦]]、[[卡爾·克勞斯|卡爾·克勞斯]]、[[皮耶·布迪厄|皮耶·布迪厄]]、[[羅伯特·穆齊爾|羅伯特·穆齊爾]]等），著述颇丰，很多论文集由法国Agone出版社出版。

2016年新著《反福柯的尼采：论真理、知识和权力》（Agone）即将问世。<ref>[http://autourdejacquesbouveresse.blogspot.fr 新书宣传页面]</ref>
2010年退休后，他的弟子{{link-en|克勞丁·提爾西林|Claudine Tiercelin}}担任[[法兰西学院|法兰西学院]][[形而上学|形而上学]]和[[知识哲学|知识哲学]]教授席位。

== 履历 ==
* 1961-1966年 : 就读于巴黎高师
* 1965年 : 取得法国中学、大学哲学教师资格
* 1966-1969年 : 巴黎索邦大学哲学系助理，教授逻辑
* 1969-1971年 : 巴黎大学哲学系助教
* 1971-1975年 : 法国国家科学研究中心研究员
* 1975年 : 获得法国国家博士学位 (université Paris I Panthéon-Sorbonne)
* 1975-1979年 : 巴黎大学副教授
* 1979-1983年 : 先后担任日内瓦大学例外教授、普通教授，教授分析哲学
* 1983-1995年 : 巴黎大学教授，兼职日内瓦大学教授
* 1992-1995年 : 法国国家科学研究中心科学史和科学哲学研究中心负责人
* 1995年 : 法兰西学院教授 (语言和知识讲席)
* 2010年起: 法兰西学院荣誉教授

== 评价 ==
另一位著名法国哲学家{{link-en|皮耶·哈多特|Pierre Hadot}}（法国最早关注维特根斯坦的人之一）在2001年的一篇访谈（“什么是伦理学？”）里，这么高度评价布弗莱斯的维特根斯坦研究：“在布弗莱斯谈论维特根斯坦之后，人们再也不敢谈论维特根斯坦了。”<ref>http://www.cairn.info/article.php?ID_ARTICLE=CITE_005_0127&DocId=106382&hits=15+14+8+7+6+4+3+2+1+</ref>
