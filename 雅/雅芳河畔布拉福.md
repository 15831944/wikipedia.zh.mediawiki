'''雅芳河畔布拉福'''（{{lang-en|Bradford on Avon 或 Bradford-on-Avon}}）是一個位於[[英格蘭|英格蘭]][[威爾特郡|威爾特郡]]西部的市鎮，人口為9326人{{r|g_census}}。

該市鎮自[[不列顛尼亞_(羅馬行省)|羅馬時代]]起便已經有人居住，而17世紀英國[[紡織業|紡織業]]崛起帶動了該市鎮發展。現時該市鎮保留很多歷史建築，為區內著名旅遊景點。

== 位置及名稱 ==
雅芳河畔布拉福位於[[科茨沃爾德|科茨沃爾德]]的邊陲{{r|t_info}}，距離[[薩默塞特郡|薩默塞特郡]]的[[巴斯_(薩默塞特郡)|巴斯市]]8英里{{r|v_info_b}}。威爾特郡的郡治[[特羅布里奇|特羅橋]]亦在該市鎮附近{{r|v_info_t}}。

市鎮名稱中的'''布拉福'''，意指「寬闊的[[渡口|渡口]]」{{r|o_history}}{{r|t_bridge}}；而'''雅芳河畔'''則指該市鎮建基於[[雅芳河_(布里斯托)|布里斯托雅芳河]]兩岸土地之上{{r|v_info_b}}。

== 歷史 ==
距今2500年前，一個[[鐵器時代|鐵器時代]]的部落抵達今英國西海岸地區，雅芳河畔布拉福一帶開始有人居住{{r|o_history}}。羅馬人曾在雅芳河畔布拉福之北的山上建造了一座堡壘，該堡壘遺址在2000年代於當地一所準備擴建的學校被發現。在其後的考古發掘中，人們又發現了一座建於3世紀中葉的羅馬莊園。該莊園的[[鑲嵌畫|鑲嵌畫]]仍保存完整，而其中一個房間被認為是早期基督教的教堂或[[洗禮|洗禮]]處{{r|t_roman}}。

[[File:St_Laurence_Bradford_01.JPG|thumb]]
雅芳河畔布拉福在[[盎格魯-撒克遜英格蘭|盎格魯-撒克遜]]時期由[[撒克遜人|撒克遜人]]佔據。他們居住在雅芳河之南、接近現在的聖馬加利山的地方{{r|o_history}}。現存的{{link-en|雅芳河畔布拉福聖羅倫斯教堂|St Laurence's Church, Bradford-on-Avon|聖羅倫斯教堂}}曾被認為是建造於這個時期：有資料指[[埃德赫姆_(英格蘭)|埃德赫姆]]在705年創立該教堂{{r|g_sl_church}}。由於現存的教堂帶有[[諾曼征服英格蘭|諾曼時期]]的建築風格，因此現在的人多認為它的建造時期應該遲一些{{r|taylor}}，例如是[[愛塞烈德二世|愛塞烈德二世]]撥地予{{link-en|沙夫茲貝利修道院|Shaftesbury Abbey}}後建成的。該教堂可能一度被用來安放[[殉教者愛德華|殉教者愛德華]]的遺骸{{r|t_saxon}}。

在13世紀，人們在雅芳河畔布拉福的渡口上建造了一座石橋。此橋至今仍然存在，並成為了市鎮的地標{{r|o_history}}{{r|t_bridge}}。

[[工業革命|工業革命]]使雅芳河畔布拉福在17世紀迅速發展：很多早期的[[紡織工業|紡織工業]]選擇在那裏建造[[水力磨坊|水力磨坊]]以取得生產紡織品所需的能源{{r|o_history}}。[[丹尼爾·笛福|丹尼爾·笛福]]曾於18世紀造訪雅芳河畔布拉福，並引述居民的談話，指當時的[[布販|布販]]平均財產值一萬至四萬[[英鎊|英鎊]]（如比較不同時代的物價或購買力，有關財產在2000年代約值數百萬英鎊）{{r|gbl_df}}。英格蘭北部出現新的工業城市後，雅芳河畔布拉福的地位漸漸被取代。其後[[橡膠|橡膠]]工業進駐雅芳河畔布拉福──{{link-en|雅方橡膠|Avon Rubber}}在20世紀中期曾經以雅芳河畔布拉福為主要生產基地{{r|ar_history}}。很多在工業時期建立的紡織廠和磨坊仍保留至今，並成為旅遊景點{{r|v_info_b}}{{r|t_wtd}}。

座落在雅芳河畔布拉福的{{link-en|威爾特郡音樂中心|Wiltshire Music Centre}}於1998年成立{{r|g_sl_school}}；另外雅芳河畔布拉福於2003年10月8日獲授予[[公平貿易城鎮|公平貿易城鎮]]的稱號{{r|f_list}}。

== 建築物 ==
[[File:Tithe_Barn_at_Bradford_on_Avon.JPG|thumb]]的倉庫]]
雅芳河畔布拉福的歷史建築物多被列為一級[[登录建築|登录建築]]。除了上文提及的石橋{{r|nh_bridge}}和聖羅倫斯教堂{{r|nh_sl}}外，鎮內的巴頓農場{{link-en|什一奉獻倉庫|Tithe barn}}{{r|nh_tb}}、聖三一座堂{{r|nh_ht}}等建築亦被納入一級受保護建築之列。

另外一些被列入較低級別的受保護建築則包括：
* [[聖公會|聖公會]]基督座堂{{r|nh_cc}}
* 前[[循道會|循道會]]教堂{{r|nh_mc}}
* 曾經為[[地方政府大樓|地方政府大樓]]、現時用以紀念[[托马斯·莫尔|托马斯·莫尔]]的天主教座堂{{r|nh_tm}}

== 地方政治 ==
雅芳河畔布拉福本身是一個[[民政教區|民政教區]]，擁有一個12人規模的市議會，該議會的主席被稱為市長。和英格蘭其他市議會一樣，雅芳河畔布拉福市議會屬於諮詢機構，而市內主要政府服務（如學校、道路、社會服務、緊急服務、文娛康樂、城市規劃、廢物處理等）主要由{{link-en|威爾特郡議會|Wiltshire Council}}負責{{r|c_councillors}}。

自2010年起，雅芳河畔布拉福歸屬重新設立的[[奇彭勒姆_(英國國會選區)|奇彭勒姆]]國會選區{{r|tg_chippenham}}。現時代表該選區的國會議員為[[自由民主黨_(英國)|自由民主黨]]的{{link-en|鄧肯·哈姆斯|Duncan Hames}}{{r|c_councillors}}。

== 交通 ==
[[File:2010_at_Bradford-on-Avon_station_-_main_building.jpg|thumb]]
途經雅芳河畔布拉福的主幹路為A363公路，大致呈南北走向，途經市內的地標石橋。其他途經該市鎮的道路較為次要，多為連接其他居民點。

來往[[布里斯托爾|布里斯托爾]]與[[樸茨茅夫|樸茨茅夫]]的鐵路以東西向橫越雅芳河畔布拉福，並設有{{link-en|雅芳河畔布拉福火車站|Bradford-on-Avon railway station}}，平日約每半小時有一班列車停靠{{r|fgw_tt}}。該鐵路取代了在更早時間建造、連接雅芳河與[[泰晤士河|泰晤士河]]的{{link-en|肯尼特與雅芳運河|Kennet and Avon Canal}}，但運河後來得到復修，並作為旅遊景點繼續開放{{r|t_canal}}。

布里斯托機場是距離雅芳河畔布拉福最近的機場，駕車來往兩地之間需時約50分鐘{{r|t_info}}。
