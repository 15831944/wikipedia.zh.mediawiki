'''雅尔塔会议'''（{{lang-en|'''Yalta Conference'''}}；{{lang-ru|'''Ялтинская конференция'''}}），又称'''雅爾達密約'''，是[[美国|美国]]、[[英国|英国]]和[[苏联|苏联]]三国領袖——[[美国总统|美国总统]][[富兰克林·德拉诺·罗斯福|富兰克林·德拉诺·罗斯福]]、[[英国首相|英国首相]][[温斯顿·丘吉尔|温斯顿·丘吉尔]]和[[苏联人民委员会主席|苏联人民委员会主席]][[约瑟夫·斯大林|约瑟夫·斯大林]]於1945年2月4日至2月11日期间，在[[苏联|苏联]][[克里米亚|克里米亚]][[雅尔塔|雅尔塔]][[里瓦幾亞宫|里瓦幾亞宫]]内举行之一次首脑会议。这次会议，制定了第二次世界大战战后的世界新秩序和列强利益分配方針，形成了「[[雅尔塔体系|雅尔塔体系]]」，对[[第二次世界大战|第二次世界大战]]后的世界局勢产生了深远的影响。

这次会议是继1943年[[德黑兰会议|德黑兰会议]]后，第二次[[同盟國|同盟國]]首脑会议，惟結論在1945年7至8月之[[波茨坦会议|波茨坦会议]]就有争议。許多人批评此次會議，使蘇聯及各國[[共产党|共产党]]得以控制[[中欧|中欧]]、[[东欧|东欧]]以及[[亞洲|亞洲]]許多[[國家|國家]]，主要是會中[[羅斯福|羅斯福]]以及[[邱吉爾|邱吉爾]]都未尊重战时被佔領国家之期望，將被苏联“解放”之国家先交由[[聯合國|聯合國]]代管（联合国1945年10月才成立，甚至连联合国宪章于雅尔塔会议结束2个月后才签署。且战后盟军没有任何一方将位于欧、亚的曾被占领国交由联合国代管；联合国曾托管的11个地区，其中7個在[[非洲|非洲]]、4個在[[大洋洲|大洋洲]]）。此外，為爭取蘇聯對日本宣戰，包括帮助中国从[[日本|日本]]关东军手中夺取满洲国，會中部份內容“侵犯”[[中國|中國]]權益甚大。會前其他国家並不知情，故有“雅爾達密約”之稱。

1944年6月，随着[[诺曼底登陆|诺曼底登陆]]，盟军在[[德国|德国]]西线展开大规模反攻，第三帝国已无力回天。1945年2月，美英认为有必要和苏联商讨德国战败后对欧的权力分配问题，并商讨下一步对日作战。对欧洲大陆，希望苏军加强对德攻势，基本原则以谁攻克的地区战后由谁控制，辅以对重要地区相互交换。在东南亚，美军在[[菲律宾|菲律宾]]开展一系列登陆战，开始把对日作战的重点从海战转向陆战；同期自1944年6月起对日本本土的B29大规模无差别轰炸效果有限，1945年2月4日[[柯蒂斯·李梅|李梅]]刚展开烧夷弹战术试验；在东亚大陆上中国军队尚与日军僵持，日军占领大片中国土地；而[[满洲|满洲]]地区则透過[[滿洲國|滿洲國]]拥有大片土地，重工业设施，70万[[关东军|关东军]]尚未直接参战。因此美英希望苏联尽快转入对[[东北亚|东北亚]]日军的进攻，打消日本依托东亚、东北亚，与盟国长期僵持并得以谈判的念头。

== 背景 ==
1944年7月19日，美国总统罗斯福致函苏联人民委员会主席斯大林，希望再次举行美、英、苏三国首脑会议。斯大林回信表示，他要亲自指挥苏军对德作战，不可出席。1944年10月，英国首相丘吉尔赴[[莫斯科|莫斯科]]会晤斯大林，英苏单独讨论[[欧洲|欧洲]]和[[巴尔干|巴尔干]]问题。美国驻苏大使[[W·埃夫里尔·哈里曼|哈里曼]]以观察员身份列席丘吉尔与斯大林之会谈（史称第四次莫斯科会议）。1944年12月，哈里曼奉命拜会斯大林，询问苏联有关参加对日作战及参战条件等事项。1944年底，三国一致同意三国首脑再次会晤<ref>{{cite web|author=侯成德|title=《苏美英首脑雅尔塔会议》|url=http://worldhistory.cass.cn/index/topwenzhang/jnffxs_txt/20050905016.htm|publisher=中国世界史研究网|accessdate=2010年2月|deadurl=yes|archiveurl=https://web.archive.org/web/20081017013732/http://worldhistory.cass.cn/index/topwenzhang/jnffxs_txt/20050905016.htm|archivedate=2008-10-17}}</ref>。

在1945年2月4日至11日，[[三巨头|三巨头]]——[[富兰克林·德拉诺·罗斯福|富兰克林·德拉诺·罗斯福]]、[[温斯顿·丘吉尔|温斯顿·丘吉尔]]和[[斯大林|斯大林]]聚集在[[克里米亚半岛|克里米亚半岛]]的[[雅尔塔|雅尔塔]]（前沙皇[[尼古拉二世_(俄罗斯)|尼古拉二世]]的之行宫-{里}-瓦幾亞宮）举行会议。這是二戰期間，繼1943年的[[德黑兰会议|德黑兰会议]]之後，盟國領袖的第二次重要會議。罗斯福去世後，再舉行''[[波茨坦会议|波茨坦会议]]''。

這次會議在蘇聯境內進行，是因為斯大林拒絕到[[黑海|黑海]]雅爾塔行宮以外開會。結果，丘吉爾和羅斯福必須再度遠涉重洋。

会前未邀請盟軍中國戰區最高統帥[[蒋介石|蒋介石]]参加。一方面，斯大林向来讨厌蒋介石{{FACT}}；另一方面，也有学者认为罗斯福此时对蒋介石失去信心<ref>{{cite web |author=林思云 |url = http://www.china-week.com/html/1712.htm |title=《蒋介石的眼泪》|publisher=中国报道周刊|date=2003年5月18日}}</ref>。

== 概观 ==
三巨頭各自带来自己之議程。羅斯福希望遊說蘇聯終止與日本不作戰的協定、對日宣戰，並且支持盟國在[[太平洋|太平洋]]作戰；邱吉爾希望斯大林在[[東歐|東歐]]戰後能放弃部分控制权，“自由”選出“民主”政府；斯大林則认为战争中苏联付出巨大代价，希望控制東歐，因為對蘇聯的戰略布局極重要。

還有，三方領袖尚須建立章程以管理戰後的德國。1943年，[[威廉·克里斯林·布利特|威廉·克里斯林·布利特]]之論點預告“紅色阿米巴變形蟲”正進入歐洲。1943年12月底時，戰線仍在蘇聯境內；到了1944年8月，[[蘇聯紅軍|蘇聯紅軍]]已經在波蘭及[[羅馬尼亞|羅馬尼亞]]境內，且繼續向西前進<ref>Traktuyev, Michael Ivanovich, ''The Red Army's Drive into Poland'' in ''Purnell's History of the Second World War'', editor Sir Basil Liddell Hart, Hatfield, UK, 1981, vol.18, p.1920-1929</ref>。會議進行時，[[格奧爾吉·朱可夫|格奧爾吉·朱可夫]][[元帥|元帥]]之部隊已經到達[[柏林|柏林]]40[[英哩|英哩]]外地方，史達林自信可在會議上採取強勢以爭取條件，而羅斯福則希望史達林承諾蘇聯加入[[聯合國|聯合國]]。

<blockquote>
“對於俄國人來說，波蘭問題不僅是榮譽問題，而且是安全問題。在整個[[歷史|歷史]]上，波蘭一直是敵人入侵俄國的走廊，波蘭是關係到俄國生死存亡之問題。”
</blockquote>

因此，斯大林清楚表達他對於波蘭的所求是沒有談判空間：蘇聯將從波蘭東部獲得領土，而波蘭將擴張西面領土作為補償。在這個要求下，波蘭西面數以百萬計的德國人將要離開。儘管建立了一個共產黨政府，斯大林答應波蘭會有自由選舉。1947年1月波蘭舉行選舉，結果在1949年，波蘭就成為[[社會主義|社會主義]]國家。

有人認為羅斯福要求蘇聯對日宣戰之焦慮是多余；事實上，斯大林很緊張能否一雪昔年[[日俄戰爭|日俄戰爭]]之失敗；他也想得回昔年損失之領土，並擴張蘇聯的影響力及於東亞。

不過，對於羅斯福是否有意讓紅軍登陸日本本土，就存在“爭議”。1945年，美军在2月雅尔塔会议后即在西太平洋开展2月硫磺岛战役、4月冲绳战役等一系列夺岛战，遭遇日军强烈抵抗，伤亡惨重，预计进攻日本本岛，需付出伤亡美军百万，英军五十万的代价。罗斯福本人希望苏联亦出兵日本本岛，分担一部分代价，并焦慮蘇聯是否肯對日宣戰。且罗斯福同意了由苏联进攻柏林，以避免英美军队伤亡。1945年4月12日，在苏联进攻柏林的前4天，罗斯福去世。

1945年8月6日和9日，美国在日本投下两颗原子弹，8月8日苏联全线出兵百万对日宣战；与夺岛战遇到的下层日军强烈抵抗截然相反，日本上层在双重压力下于8月10日6时通过中立国瑞典、瑞士向盟国转达，在保证天皇地位的前提下有条件投降；后在未获答复的情况下于8月15日由昭和天皇广播宣布无条件投降。投降之快出乎盟国意外。

罗斯福的幕僚和继任者杜鲁门副总统，对羅斯福是否有意讓紅軍登陸日本本土，就存在“爭議”。原本有美国将领希望能以无差别战略轰炸迫使日本投降，从而避免登陆战的巨大伤亡；繼羅斯福之位的[[杜魯門|杜魯門]]，更认为使用[[原子彈|原子彈]]即可結束戰爭；均希望避免紅軍攻進日本本土，使蘇聯無法如在德國般地瓜分日本的利益。

实际李梅于2月提出的烧夷弹战术，在日本则取得比在欧洲使用的高空轰炸更好的效果，造成的傷亡比後來的原子彈攻擊還要多，可用的原子弹也仅2枚。日本本土非但没有因B-29轰炸造成的伤亡退缩，甚至上下宣扬“举国玉碎”。同时各国民间对原子弹缺乏认识，没有辐射后效概念，原子弹造成的直接伤亡有限，直观上只是一个大炸弹。日本籍希望苏联中立，决心以满洲为依托，僵持、拒绝投降且最终迫使盟国与其谈判，使盟国無法如在德國般地瓜分日本的利益。
[[File:Livadiya_Conference.JPG|thumb]]

作为一个附带影响，俄國人並沒有因此解決當初日本南千岛群岛的主權問題至今。

<blockquote>
“重建歐洲的秩序和國民经济的大前提是：徹底消滅[[納粹主義|納粹主義]]和[[法西斯主義|法西斯主義]]，和由被解放的人民建立他們自己選擇的民主体制。”

== 會議重點 ==
{{德国国界}}
{{波兰国界}}

* 协定最重要是[[纳粹德国|纳粹德国]][[無條件投降|無條件投降]]，這樣保證盟國團結，盟國任何一方都不可能和納粹有任何談判空間。战后德国和柏林将被分成四个占领区。在德国统一前，柏林将由四国占领。 [[德国|德国]]被分裂，美英苏认为这是“今后和平與安全的必要条件”。
* 斯大林同意[[法国|法国]]將在德国和奥地利拥有第四个占领区，但必須成立於美國和英國區以外；並同意法国有资格进入同盟国对德国控制委员会。
* 德國将会进行非军事化和[[去纳粹化|去纳粹化]]。
* 德国战争赔款部份以强迫劳动形式來支付。
* 建立一個位於蘇聯之賠償理事會。
* [[波兰|波兰]]地位問題。盟国同意重組當時之波兰政府，加入其他組織，例如波兰[[临时政府|临时政府]]進行[[民主选举|民主选举]]，但排除了1939年所成立的[[波兰流亡政府|波兰流亡政府]]參與其中。
* 波兰东部边界将按照[[寇松线|寇松线]]，同时波兰將从德国得到西部領土补偿，而精确的边界将晚些决定。
* 斯大林承諾允許在波蘭自由選舉。
* 無視苏联和[[南斯拉夫|南斯拉夫]][[战俘|战俘]]意願，他們將會被送回國。
* 斯大林同意在击败德国以后三個月内，加入对日本作战。苏联战胜日本后，将可收復[[库页岛|库页岛]]南部、獲得[[千岛群岛|千岛群岛]]，並保障其在[[大連港|大連港]]、[[中東鐵路|中東鐵路]]、[[南滿鐵路|南滿鐵路]]的利益，以及恢復[[俄羅斯海軍|俄羅斯海軍]]在[[旅顺口|旅顺口]]的租賃作为报酬。
* 羅斯福得到斯大林承諾，蘇聯將加入联合国，同意[[联合国安全理事会|联合国安全理事会]]五個[[聯合國安全理事會常任理事國|常任理事國]]均將享有[[聯合國安全理事會否決權|否決權]]。
* 納粹戰犯都會懲處，並繩之以法。
* 將成立一個「决定分裂德国之委員會」。委員會將决定德国是否要分成六个国家，如果要分割，几个新德意志国家之间的边界和相互关系又将如何确定。
* 成立新国际组织（[[联合国|联合国]]），取代失敗的[[国际联盟|国际联盟]]。

== 會議協議 ==
* 所有被[[解放|解放]]之欧洲[[国家|国家]]内应该举行[[民主|民主]][[选举|选举]]。
* 4月在[[旧金山|旧金山]]进行就联合国成立之会议。基本确定联合国组织方式，采纳联合国安全理事会之主意。美国和英国同意当时属於苏联之[[乌克兰苏维埃社会主义共和国|乌克兰苏维埃社会主义共和国]]和[[白俄罗斯苏维埃社会主义共和国|白俄罗斯苏维埃社会主义共和国]]为联合国独立成员。
* 分裂德国，解散[[納粹德军|納粹德军]]，德国不准再拥有军队。美英苏认为这是“今后和平和安全之必要条件”。
* 德国应该为“她对同盟国在战争中造成的之损失”负战争赔款。战争赔款可以以德国国家资源（机器、船只、企业所有等）、一段时间内应该支付的之偿款或劳动力的之方式赔偿。美国和苏联达成协议偿款总额约220亿美元。英国认为在当时偿款总额还无法估计。
* 暂时搁置[[战争罪|战争罪]]问题。
* 一个“广泛民主临时政府”，应在波兰“尽快进行自由和不受他国控制、全民和秘密之选举”。
* 应在南斯拉夫建立一个保皇党和共产党联合政府。
* 暂时搁置关于[[意大利|意大利]]—南斯拉夫、意大利—[[奥地利|奥地利]]、南斯拉夫—[[保加利亚|保加利亚]]、罗马尼亚、[[伊朗|伊朗]]以及[[土耳其|土耳其]]管理之黑海与[[地中海|地中海]]间之海峡使用之问题。
* 不论被俘苏联公民愿不愿意，一律遣返回苏联。

就未来德国问题之处理，雅尔塔协议相当含糊，用词也為後來東西德分裂詮釋造就許多想像空間，亦有人認為這次會議為戰後[[冷战|冷战]]之濫觴。

這次會議的結果，就事後來看，永遠的改變了許多國家的命運（東歐諸國成为蘇聯[[衛星国|衛星国]]、南北韓分裂、[[外蒙古獨立|外蒙古獨立]]和[[中國國共内戰|中國國共内戰]]等），也懸留許多問题至今尚未解決。

== 后果 ==
[[File:Yalta1945.gif|thumb]]

雅尔塔会议是二战欧洲战事结束前及罗斯福总统去世前，最后一次的重要会议。東歐大部分地區的納粹軍隊已被紅軍消滅了，所以斯大林有條件和美英談判，結果蘇聯在東歐得到一個夢寐以求之緩衝區。過程中，一些小國利益因為要保持盟軍內部穩定而被犧牲，蘇聯繼續統治[[波罗的海国家|波罗的海国家]]：[[拉脱维亚|拉脱维亚]]、[[立陶宛|立陶宛]]和[[爱沙尼亚|爱沙尼亚]]。

雅尔塔会议促使苏联对满洲关东军发动进攻，而苏联曾承认满洲国，将满洲作为独立的敌对国看待，随后[[中華民國_(大陸時期)|中國]]与苏联签订《[[中苏友好同盟条约|中苏友好同盟条约]]》，以苏联击败关东军后退出满洲为条件，允诺[[外蒙古|外蒙古]]在战后进行公民投票以决定其前途。最后在苏联和中华民国监视下，公民投票赞成独立。1946年1月，中華民國政府通知[[烏蘭巴托|库伦]]方面，同意其独立；1947年7月28日，中華民國駐聯合國代表[[徐淑希|徐淑希]]在联合国安理會发表演說，指責蒙古人民共和国軍隊入侵中國新疆，反對外蒙古加入聯合國<ref>{{cite news |url = http://contentdm.lib.nccu.edu.tw/cdm/singleitem/collection/38clip/id/4256/rec/6

</ref>。1955年12月13日，中华民国代表在安理會否決蒙古人民共和国加入联合国，理由是全蒙古是中国的一部分。<ref>{{cite web |url = http://www.un.org/zh/sc/meetings/veto/pdf/a58_47_p2.pdf |publisher=聯合國 |title=因常任理事国投反对票而未获通过的决议草案或修正案各段}}</ref><ref>{{cite web |url = http://news.bbc.co.uk/2/hi/middle_east/2828985.stm|title=The veto and how to use it|publisher=BBC News Online}}{{en}}</ref><ref>{{cite web |url = http://www.globalpolicy.org/component/content/article/102/32810.html |title=Changing Pattern in the Use of Veto in the Security Council|publisher=Global Policy Forum}}{{en}}</ref><ref>{{cite web |url = http://www.globalpolicy.org/component/content/article/102/40069.html |title=Subjects of UN Security Council Vetoes|publisher=Global Policy Forum}}{{en}}</ref>。1961年在美国和苏联的压力下，中華民國最終不得不在[[聯合國安理會166號決議|缺席]]的情況下讓外蒙入聯。

另一方面，'''雅爾達會議更造成了今日的領土、領海問題'''。邱吉爾、羅斯福為了要讓蘇聯參戰，同意了史達林的要求：'''「南[[薩哈林島|庫頁半島]]、[[北方四島|北方四島]]和部分的[[阿留申群島|阿留申群島]]歸[[蘇聯|蘇聯]]所有，不得異議，[[蘇聯|蘇聯]]便參戰。」'''在苏联参战之后又有部分反悔，造成了今日的[[北方四島|北方四島]]、[[釣魚台列島|釣魚台列島]]和靠近[[美國|美國]][[阿拉斯加州|阿拉斯加州]]的幾個離島的經濟海域、[[荷蘭港|荷蘭港]]外圍的海底資源探勘等地的諸多國際問題。

經此會議後。蘇聯即对日宣戰促成五日后[[日本投降|日本投降]]（一次性出动89個師包括1,500,000人、3,704輛[[坦克|坦克]]、1,852具[[自走炮|自走炮]]、85,819部車輛及3,721架飛機，徹底地压垮日军已打算“举国玉碎”的顽抗意志），並出兵進攻[[日本|日本]][[關東軍|關東軍]]侵占的中國東北的[[嫩江省|嫩江省]]、[[黑龍江省|黑龍江省]]和[[察哈爾省|察哈爾省]]等地。在戰後亦不退還給[[中国|中国]][[国民政府|国民政府]]，而是交給扶植的[[中國共產黨|中國共產黨]]。当时东北没有国民政府所属的军队，跟从[[蔣介石|蔣介石]]的军队如东北军早已溃败入关，跟从[[汪精衛|汪精衛]]的军队，则在[[汪精衛政府|汪精衛政府]]承认[[满洲国|满洲国]]的背景下，转化为滿洲國军队，只有[[东北抗日联军|东北抗日联军]]在东北坚持抗战14年，其间在未受蔣介石援助又遭日、滿联军清剿的情况下数次几近灭绝。而後又在[[中国共产党|中国共产党]]和[[苏联|苏联]]的扶植下几度壮大。1946年蒋介石要求苏联暂缓从滿洲撤军，苏联大使3月22日通知[[中国|中国]][[国民政府|国民政府]]，4月底以前苏军将全部撤离滿洲，但国民政府的军队仍没有足够的兵力进入滿洲。
