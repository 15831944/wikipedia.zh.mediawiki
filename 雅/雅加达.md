'''雅加達'''（{{lang-id|Jakarta}}），全稱'''雅加達特別首都地域'''（{{lang-id|Daerah Khusus Ibukota Jakarta}}），是[[印度尼西亚|印度尼西亞]][[首都|首都]]和最大城市，位於[[爪哇岛|爪哇島]]的西北海岸。雅加達是印尼的經濟、文化和政治中心，現有人口10,187,595（截至2011年11月）<ref>{{cite web |url=http://www.waspada.co.id/index.php?option=com_content&view=article&id=243533:foke-lebih-yakin-lembaga-survei-asing&catid=17:politik&Itemid=30 |title=Foke lebih yakin lembaga survei asing |date=24 April 2012}}</ref>，而周圍的大雅加達地區居住超過3000萬人，是世界第二大都市圈。

雅加達在大約15世紀時建立，最早是一個在[[芝利翁河|芝利翁河]]口的小漁村，當時的中國文書譯為'''噶喇吧'''（Kelapa，馬來語：椰子；或作'''咬𠺕叭'''、'''咖𠺕吧'''）。[[葡萄牙|葡萄牙]]人是第一批到訪的[[歐洲|歐洲]]人，16世紀初，一位[[印度|印度]]國王允許葡萄牙人在此建立一座要塞。現在這個港口仍然被稱為[[巽他卡拉巴|巽他卡拉巴]]（Sunda Kelapa）。

1527年，這座城市被印度教[[万丹_(印尼)|万丹]]王國的法塔西拉（Fatahillah）所征服。1527年6月22日，法塔西拉將這座城市重新命名為查雅加達（Jayakarta，勝利之城），這個日子被訂為雅加達正式建城的日子。

1619年，[[荷屬東印度公司|荷屬東印度公司]]征服了這座城市，並重命名為'''巴達維亞'''（Batavia，[[荷蘭|荷蘭]]的羅馬名），巴塔維亞被定為[[荷屬東印度|荷屬東印度]]的首都。

[[英國|英國]]在1811年時佔領了雅加達，直到1816年還給荷蘭殖民政府。在1920到1940年之間，這座城市擴張而且變得現代化。

[[日本帝国|日本帝国]]在1942年佔領了這座城市，並重新命名為現名雅加達。1945年[[日本帝国|日本帝国]]戰敗後，荷蘭殖民政府重新回到這裡，並鎮壓了1945年8月17日獨立的印度尼西亞政府。直到1949年，荷蘭才將統治權移交給印度尼西亞共和國。

== 經濟 ==

作為印尼首都和最大城市，雅加達經濟主要以金融居多，佔該市生產總值28.7%，並擁有國內最大的金融和主要工商业机构。

雅加達早於十五世紀已是重要商港，殖民時代曾是[[荷屬東印度公司|荷屬東印度公司]]總部所在，貿易遍及[[亞洲|亞]]、[[歐洲|歐]]、[[非洲|非]]三大陸。今雅加達仍有多處大型綜合市場和專業商場，每年在此舉辦雅加達交易會。

工業集中於市郊，主要項目為煉油和化工。

2007年雅加達經濟增長率，由5.95%跳升至6.44%。增長來源主要是通訊(15.25%)、基礎建設(7.81%)和酒店業(6.88%)。

2007年的新法律禁止人们给乞丐、街头卖艺人和小商贩钱财，禁止在河岸和高速公路旁擅自修建房屋居住，禁止在公共交通工具上吐痰和抽烟。未经授权就清洗轿车挡风玻璃和在十字路口问路也会受到处罚。有人批评这样的法律很难执行，也将会忽视许多贫民的权益。

2014年根據房地產市調機構萊坊（Knight Frank）研究指出，雅加達房價自2010年飆升，近兩年精華區每年平均漲幅37%，高居全球三十大都會之冠，近五年被譽為「第五塊金磚」的印尼，在QE〈量化寬鬆〉減碼後，步入「脆弱五國」命運。<ref>[印尼是猛虎還是病貓] 天下雜誌 2014-03-29</ref>

== 行政區劃 ==
'''雅加达首都特区'''（[[印尼语|印尼语]]：Daerah Khusus Ibukota Jakarta，简称DKI Jakarta，直译为雅加达首都特别地域）是印度尼西亚34个[[印度尼西亚省份列表|省级行政区]]之一，中文常称之为'''雅加达省'''、'''雅京省'''，特区首长也常称作“省长”。

雅加達下轄五个行政市（Kota administrasi）<ref group="注">行政市与印尼其他地区的“市”Kota不同，印尼目前仅雅加达特区辖有5个此类政区。此类“行政市”并非真正意义上的自治城市</ref>：
* 東雅加達
* 西雅加達
* 中雅加達
* 北雅加達
* 南雅加達
 [[File:Jakarta_subdistrict.png|400px]]
此外另轄有一行政县（Kabupaten administrasi）<ref group="注">行政县为雅加达特区独有政区，目前仅千岛群岛一例。与行政市一样，行政县并非自治地区，不是真正意义上的县</ref>：千岛群岛。
