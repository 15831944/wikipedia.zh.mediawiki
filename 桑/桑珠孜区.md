== 历史 ==
[[File:Downtown_Shigatse.JPG|thumb]]

相传八世纪时，[[印度|印度]]高僧[[莲花生|莲花生]]路经此处，看到山川形胜，便停留下来修行讲经，并预言这里将会成仅次于[[拉萨|拉萨]]的城市。14世纪初，[[帕竹王朝|帕竹王朝]]在此建城，名“'''溪卡桑珠孜'''”，简称“'''溪卡孜'''”，[[藏语|藏语]]意为“最美的庄园”，[[汉语|汉语]]译音'''日喀则'''。[[藏巴汗|噶玛王朝]]时为[[都城|都城]]，城市获得了极大的发展。1960年设立[[日喀则县|日喀则县]]，1986年被中华人民共和国国务院授予[[国家历史文化名城|国家历史文化名城]]称号，同年撤县设市，成立[[日喀则市_(县级市)|县级日喀则市]]。2014年6月26日日喀则地區升格為地級日喀則市，原縣級日喀則市易名桑珠孜區。

== 地理 ==
桑珠孜地处西藏南部、[[喜马拉雅山|喜马拉雅山]]北麓、[[雅鲁藏布江|雅鲁藏布江]]和年楚河的交汇之处。[[海拔|海拔]]3840米，是世界上最高的城市之一。全市年平均气温6℃，降水量420毫米。人口中[[藏族|藏族]]居多数。农业主要有[[青稞|青稞]]、[[小麦|小麦]]种植和[[牦牛|牦牛]]牧养等，手工业比较发达。

== 扎什伦布寺 ==
[[File:Tashilhunpo_Monastery,_Shigatse.JPG|thumb]]]]
桑珠孜是历代[[班禅喇嘛|班禅喇嘛]]的驻锡之地。班禅喇嘛，又称班禅额尔德尼，是[[康熙|康熙]]賜封[[藏传佛教|藏传佛教]]中地位與[[达赖喇嘛|达赖喇嘛]]相等的另一位[[格鲁派|格鲁派]]领袖，與歷代[[达赖喇嘛|达赖喇嘛]]是互為師徒的關係。班禅的庙宇[[扎什伦布寺|扎什伦布寺]]是黄教的四大寺院之一，位于城南，依山而建、规模宏大，兼具[[宗教|宗教]]和[[政治|政治]]功能，是日喀则乃至後藏的行政核心。城东南的[[夏鲁寺|夏鲁寺]]则是藏传佛教[[夏鲁派|夏鲁派]]的起源地，建于十一世纪，并保存有诸多精美的[[壁画|壁画]]。城内外还有那当寺、俄尔寺、安贡寺等多处古老[[寺院|寺院]]和民居，大都仍在使用。
