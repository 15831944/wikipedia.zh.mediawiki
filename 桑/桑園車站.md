'''桑園車站'''（{{jpn|j=桑園駅|hg=そうえんえき|rm=Sōen eki|f=y}}）是一由[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）所經營的[[鐵路車站|鐵路車站]]，位於[[日本|日本]][[北海道|北海道]][[札幌市|札幌市]][[中央區_(札幌市)|中央區]]，是JR北海道[[函館本線|函館本線]]與[[札沼線|札沼線]]（學園都市線）的交會車站，屬於[[北海道旅客鐵道本社|JR北海道本社]]（札幌總部）的直轄範圍，由JR北海道直接派駐人員營運。車站編號為'''S04'''。電報略號為'''{{lang|ja|ソウ}}'''。

== 簡介 ==
桑園車站是函館本線的沿線車站與札沼線的起點車站，但行駛於札沼線上的列車實際上都會通過本站之後進入函館本線，駛抵[[札幌車站|札幌車站]]之後再行折返。

1988年改建的桑園採[[高架車站|高架車站]]結構，配置有兩座[[島式月台|島式月台]]共四條乘車線。站內各種相關設施包括[[綠窗口|綠窗口]]、[[自動售票機|自動售票機]]、[[自動補票機|自動補票機]]與[[Kiosk便利商店|Kiosk便利商店]]皆一應俱全。桑園車站是距離JR北海道本社大樓最近的車站，而除了JR北海道之外，[[JR貨物|JR貨物]]的北海道支設也將辦公室設於同一棟大樓之內。

本站的設置最早的起源，可以回溯到1907時在附近設立的[[札幌賽馬場|札幌賽馬場]]。1908年為了配合競馬場的營業，因此在馬場附近設置了臨時車站'''北五條假乘降場'''（{{lang|ja|北五条仮乗降場}}）。1911年時，改成每年都會設置的臨時車站'''競馬場前假乘降場'''（{{lang|ja|競馬場前仮乗降場}}）。但直到1924年時才開始正式開辦客運專用、沒有貨運服務的桑園車站（{{lang|ja|桑園駅}}）。

由於是競馬場的主要交通玄關，近年來車站周邊地區也陸續建造了許多住宅大樓，使得車站的旅運量逐年增加。其2010年的每日平均旅運量約為9,000人，名列JR北海道所有車站中的第六位。
== 歷史 ==
=== JR北海道 ===
* 1908年（明治41年）
** 7月30日 - [[鐵道省|國有鐵道]][[琴似車站_(JR北海道)|琴似站]]至[[札幌站|札幌站]]開設'''北五條假乘降場'''。只於8月5、6、15、16日營業。
** 8月17日 - 北五條假乘降場廢除。
* 1909年（明治42年）
** 8月25日 - 國有鐵道[[錢函車站|錢函車站]]至札幌站路段[[複線鐵路|複線化]]。
** 10月12日 - 按照國有鐵道制定鐵路名稱，[[函館車站|函館車站]]至[[旭川車站|旭川車站]]路段稱為[[函館本線|函館本線]]。
* 1911年（明治44年）之後 - 逢賽馬日設置賽馬場前簡易停車場。
* 1913年（大正2年）7月19日 - 函館本線琴似站至札幌站之間開設'''賽馬場前假乘降場'''。
* 1924年（大正13年）6月1日 - 函館本線琴似站至札幌站之間開設'''桑園站'''。處理乘客及行李服。同時廢除賽馬場前假乘降場。
* 1934年（昭和9年）11月20日 - '''札沼南線'''本車站至[[石狩當別車站|石狩當別車站]]路段開始營運。
* 1935年（昭和10年）10月3日 - 石狩當別站至[[浦臼車站|浦臼車站]]路段開始營運，札沼南線與'''札沼北線'''合併並改稱為'''札沼線'''<ref name=school>{{cite web|url=http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |language=ja |title=札沼線（学園都市線）の電化について |publisher=JR北海道 | date=2009-09-09 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20091006234745/http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |archive-date=2009-10-06 }}</ref>。
* 1940年（昭和15年）10月20日 - 開始貨物（盛載煤炭的貨車）處理業務。
* 1942年（昭和17年）6月 - 設置緊急用儲存木炭路軌。舖設[[北海道大學|北海道帝國大學]]區域內煤炭用路軌。
* 1949年（昭和24年）6月1日 - 本路線由[[日本國有鐵道|日本國有鐵道]]（國鐵）繼承。
* 1950年（昭和25年）10月 - 桑園倉庫企業組合（後來的桑園倉庫株式會社）專用側線投入使用。
* 1956年（昭和31年）2月10日 - 札幌用品庫（材料中心）由札幌站前搬遷至本車站背後東側。設置專用線。
* 1958年（昭和33年） - 札幌拓殖倉庫株式會社專用側線投入使用。此外[[日本通運|日本通運]]、札幌運通等倉庫亦於昭和30年代設置專用側線。
* 1959年（昭和34年）12月15日 - 函館本線（貨物支線）本車站至{{Link-ja|札幌市場駅|札幌市場車站}}路段開始營運。
* 1962年（昭和37年）6月27日 - 站房改建。同時遷移[[架空道路|跨線橋]]。
* 1968年（昭和43年）8月28日 - 函館本線[[小樽站|小樽站]]至[[瀧川車站|瀧川車站]]正式電氣化（[[交流電|交流電]]20,000[[伏特|V]]、50[[赫兹|Hz]]）。
* 1978年（昭和53年）10月2日 - 停止貨物處理。函館本線（貨物支線）本車站至札幌市場站路段廢除。
* 1985年（昭和60年）3月14日 - 停止行李處理。
* 1987年（昭和62年）4月1日 - 由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理。
* 1988年（昭和63年）11月3日 - 函館本線琴似站至札幌站，以及札沼線本車站至[[八軒車站|八軒車站]]完成[[交通立體化|高架化]]，同時本車站成為[[高架車站|高架車站]]。
* 1991年（平成3年）3月16日 - 札沼線的暱稱設定為「'''學園都市線'''」<ref name=school/><ref name=school2>{{cite web |url=http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |language=ja |title=札沼線（学園都市線）の電化開業時期について |publisher=JR北海道 |date=2011-10-03 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20111205160350/http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |archive-date=2011-12-05 |deadurl=yes }}</ref>。
* 1994年（平成6年）11月1日 - 函館本線本車站至札幌站之間增設札沼線（學園都市線）專用路軌，此路段三線化<ref name=school2/>。
* 1995年（平成7年）10月1日 - 新設立東側入口，並設置北海道首部傾倒型[[自動售票機|自動售票機]]，同時設置[[升降機|升降機]]。
* 1996年（平成8年）6月23日 - 八軒站成為高架車站，札沼線（學園都市線）本車站至八軒站路段高架化。
* 1998年（平成10年）10月24日 - 設置[[驗票閘門|自動閘機]]。
* 2000年（平成12年） - 札沼線（學園都市線）本車站至[[石狩月形車站|石狩月形車站]]路段引入自動進路制御裝置（PRC）。
* 2007年（平成19年）10月1日 - 設定車站編號（'''S02'''）<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |language=ja |title=駅番号表示（駅ナンバリング）を実施します |publisher=JR北海道 |date=2007-09-12 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20070930015220/http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |archive-date=2007-09-30 |deadurl=yes }}</ref>。
* 2008年（平成20年）10月25日 - 開始使用[[儲值車票|智能車票]]「[[Kitaca|Kitaca]]」<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |language=ja |title=Kitacaサービス開始日決定について |publisher=JR北海道 |date=2008-09-10 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20080913103748/http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |archive-date=2008-09-13 |deadurl=yes }}</ref>。
* 2012年（平成24年）6月1日 - 札沼線（學園都市線）本車站至北海道醫療大學站正式電氣化（交流電20,000V、50Hz）<ref>{{cite web|url=http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |language=ja |title=学園都市線電化開業に伴う電車の投入（第一次）について |publisher=JR北海道 | date=2012-03-14 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20120925203437/http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |archive-date=2012-09-25 }}</ref><ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |language=ja |title=平成24年10月ダイヤ改正について |publisher=JR北海道 |date=2012-08-03 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20120925203427/http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |archive-date=2012-09-25 |deadurl=yes }}</ref>。
* 2016年（平成28年）3月31日 - 旅遊中心停止營業<ref>{{cite web |url=https://www.jrhokkaido.co.jp/press/2016/160219-2.pdf |language=ja |title=ツインクルプラザ（JR 旅行センター）6店舗の営業終了について |publisher=JR北海道 |date=2016-02-19 |accessdate=2017-08-31 |archive-url=https://web.archive.org/web/20160220233912/http://www.jrhokkaido.co.jp/press/2016/160219-2.pdf |archive-date=2016-02-20 |deadurl=yes }}</ref>。

== 車站結構 ==
2面4線月台的高架車站。1、2號月台為函館本線；3、4號月台為札沼線。社員配置站，設有[[綠窗口|綠窗口]]（營業時間5時30分 - 24時00分）、自動售票機、自動閘機、詢問處。JR北海道本社最近的車站，亦是同社首次試驗引入自動閘機的車站。

1994年11月1日，當本車站至札幌站之間增設札沼線專用路軌時，路軌由過去按方向分配改為按路線分配。除此之外，雖然當時札沼線仍未電氣化，但仍於其3號線安裝[[架空電纜|架空電纜]]，而此電纜直至札沼線於2012年6月1日正式電氣化後投入使用。此外，由於鐵路訊號機的設置，函館本線的下行線列車能夠駛入札沼線專用單線，而相反情況亦可以進行。

過去本車站只有西側入口，1995年由於市立札幌醫院遷移至當時車站附近的國鐵札幌材料中心遺址，由此新設置東側入口、[[升降機|升降機]]及[[電動扶梯|電動扶梯]]。

== 車站周邊 ==
車站周邊過去主要是倉庫，1995年10月11日當市立札幌醫院由中央區北1條西9丁目遷移至本車站附近的中央區北11條西13丁目後，開始有大規模商業設施及高級公寓在附近發展，而市中心亦持續發展中。

亦是札幌賽馬場的鄰近車站，於賽馬日及場外投注日會於車站前提供免費接駁巴士。此外，車站旁為JR北海道的總部，而[[日本貨物鐵道|日本貨物鐵道]]（JR貨物）北海道分社亦在此建築物內。
