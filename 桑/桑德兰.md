'''桑德蘭'''（'''Sunderland'''，{{IPAc-en|audio=en-uk-Sunderland.ogg|ˈ|s|ʌ|n|d|ər|l|ən|d}}, {{IPAc-en|local|ˈ|s|ʊ|n|ᵈ|l|ən}}），又譯-{zh-cn:"巽德蘭"(台);zh-tw:「桑德蘭」(大陸);zh-hk:「巽德蘭」(台)}-和-{zh-cn:"新特蘭"(港);zh-tw:「新特蘭」(港);zh-hk:「桑德蘭」(大陸)}-，是個位於[[英格蘭|英格蘭]][[泰恩-威爾郡|泰恩-威爾郡]]的大鎮。屬於[[桑德蘭市|桑德蘭市]]。

土生土長的桑德蘭人被稱作''[[Mackem|Mackem]]''。<ref>{{cite web |url=http://www.bbc.co.uk/wear/content/articles/2005/06/21/wordhunt_feature.shtml|title=The Mackem Wordhunt|accessdate=2008-04-03 |author=BBC|year=2005}}</ref><ref>{{cite web |url=http://www.phrases.org.uk/bulletin_board/18/messages/814.html Mackems|title=The word Mackem origins|accessdate=2008-04-03 |author=Phrases.org website|year=2005}}</ref>

== 歷史 ==
=== 20世紀到今天 ===
從前在本市發展蓬勃的重工業逐站被淘汰，新的輕工業工廠進駐了本市，例如[[日產汽車|日產汽車]]工廠。

在1990年後，[[威爾河|威爾河]]的兩岸重新發展，在原建有船廠的位置興建了新興住宅、零售中心及商業區。在重建的時候也興建了國家玻璃中心（National Glass Centre）及[[桑德蘭大學|桑德蘭大學]]。

在[[第二次世界大戰|第二次世界大戰]]中，新特蘭是英格蘭被[[納粹德軍|納粹德軍]]轟炸的最嚴重的地方之一。幸好不少有歷史價值的舊建築不受影響，得以保留下來。

== 地理 ==
本市大多數部分位於與海岸平行的山脈上，位於海拔約80米。新特蘭被[[威爾河|威爾河]]在市中心分隔。 

== 交通 ==
* [[A19公路|A19公路]]途經本市。
* [[泰恩威爾地鐵|泰恩威爾地鐵]]在市內設有很多車站。
* [[英國國鐵|英國國鐵]]及泰恩威爾地鐵的[[新特蘭站|新特蘭站]]位於本市。

== 體育 ==
本市唯一的職業運動隊是現時在[[英超|英超]]作賽的[[桑德兰足球俱乐部|新特蘭足球會]]。他們的主場是位於市內的[[光明球場|光明球場]]，為前[[曼聯|曼聯]]球員[[堅尼|堅尼]]首支執教球隊。他們是英格蘭老牌球隊。新特蘭和同在泰恩-威爾郡的[[紐卡素足球會|紐卡素]]是宿敵，兩支球隊的交手稱為泰恩-威爾打比。
