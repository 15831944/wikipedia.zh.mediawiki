'''栽培品種'''（英語：'''Cultivar'''）是指經由人工栽培之後產生的[[植物|植物]]種類，也稱為'''品種'''或'''變種'''，不過後兩個字有其他不同意義。栽培品種根據特徵，有一個專有的命名，以和其他同種植物有所區隔。而命名方式，則是由《{{le|國際栽培植物命名法規|International Code of Nomenclature for Cultivated Plants}}》（''ICNCP''）來規範。

「栽培品種」一詞是來自英語中的「栽培」（'''culti'''vated）與「變種」（'''var'''iety），不過栽培品種的意義，與[[植物學|植物學]]或[[生物學|生物學]]上所說的變種並不相等<ref>{{cite web |url=http://www.hort.purdue.edu/hort/courses/HORT217/Nomenclature/description.htm |title=存档副本 |accessdate=2011-06-20 |deadurl=yes |archiveurl=https://archive.is/20121211140110/http://www.hort.purdue.edu/hort/courses/hort217/Nomenclature/description.htm |archivedate=2012-12-11 }}</ref>。在1959年1月1日以前，栽培品種通常有一個拉丁化的名稱，但是如此一來很容易和其種名混淆，因此後來改用各地語言來做命名，舉例如下
:''Chamaecyparis lawsoniana'' 'Aureomarginata'（[[拉丁化|拉丁化]]）
:''Chamaecyparis lawsoniana'' 'Golden Wonder'（來自[[英語|英語]]）
:''Pinus densiflora'' 'Akebono'（來自[[日語|日語]]）
