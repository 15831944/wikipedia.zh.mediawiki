'''貝利亞斯菊石'''（[[學名|學名]]：{{lang|la|'''''Berriasella'''''}}），又名'''貝里阿斯菊石'''，是生存在[[侏羅紀|侏羅紀]]末[[提通期|提通期]]至[[白堊紀|白堊紀]]初[[貝里亞期|貝里亞期]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]，在該段時期的地層中有廣泛的分布<ref name=FWBerriasella>[http://paleodb.org/cgi-bin/bridge.pl?action=basicTaxonInfo&taxon_no=14617 Paleobiology Database - ''Berriasella''] at [[Fossilworks|Fossilworks]].org</ref>，為此兩[[紀_(地質學)|紀]]間過渡階段的代表物種之一，其名稱即源自於貝里亞期。牠們是繁盛於侏羅紀的旋菊石類之後裔。<ref name="Berriasella">{{Cite web |url = http://liusir.pixnet.net/blog/post/248636438-berriasella-%E8%B2%9D%E9%87%8C%E9%98%BF%E6%96%AF%E8%8F%8A%E7%9F%B3-%28%E6%B3%95%E5%9C%8B%29 |title = Berriasella 貝里阿斯菊石 (法國) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-06-06 }}</ref>其[[化石|化石]]被發現於[[歐洲|歐洲]]、[[中美|中]][[南美|南美]]和[[西亞|西]][[南亞|南亞]]等地。<ref name="Fossilworks" />

== 特徵 ==
貝利亞斯菊石的殼形較為扁平，殼的側面有密集排列的彎曲[[Y|Y]]形肋條，於腹側中央不相連，由一淺溝隔開。<ref name="Berriasella" />
