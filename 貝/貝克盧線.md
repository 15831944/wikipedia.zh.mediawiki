{{倫敦地鐵RDT/貝克盧線}}
'''貝克盧線'''（{{lang-en|'''Bakerloo line'''}}）為[[倫敦地鐵|倫敦地鐵]]的一條路線，在地鐵圖上以棕色表示，線名取自線上兩個站「[[貝克街站|'''貝克'''街]]」和「[[滑鐵盧地鐵站|滑鐵'''盧''']]」的併稱。女王公園以北部份位於地面上，其餘皆於深層隧道中行駛。该线从象堡站作起點，从东南侧进入内伦敦，穿越伦敦西区，最后到达伦敦西北郊的哈罗和威爾斯東。该线有25个车站，其中15个是地下站。贝克卢线是伦敦地铁网络中第九繁忙的线路，年均客流量超过1.11亿人次。在女王公园站以北（也就是该线的地面段），贝克卢线和[[沃特福德直流電铁线|沃特福德直流電铁线]]共轨，也与英国铁路系统中的[[西海岸主线|西海岸主线]]平行。

== 歷史 ==
貝克盧線起初的名稱為'''貝克街－滑鐵盧鐵路'''，由[[倫敦地下電氣鐵路公司|倫敦地下電氣鐵路公司]]負責建造，1906年3月10日開始營運，路線為[[貝克街站|貝克街]]至[[蘭貝斯北站|蘭貝斯北]]（當時稱為「肯寧頓路站」）。同年8月5日延長至[[象堡站|象堡]]。1906年7月將簡稱「貝克盧線」正式化。

1907年3月27日延長至[[馬里波恩站|馬里波恩]]（當時稱為「大中央車站」）。同年6月15日再延長至[[埃奇韋爾路站_(貝克盧線)|埃奇韋爾路]]。1913年12月1日再延長至[[帕丁頓車站#帕丁頓地鐵站|帕丁頓]]。1915年1月31日延長至基爾伯恩公園，同年2月11日延長至女王公園。6月6日麥達維爾站啟用。

===沃特福德支線===
[[File:Paddington_Bakerloo.JPG|thumb]]貝克盧線南行月台]]
1915年5月10日沿[[沃特福德直流電鐵線|沃特福德直流電鐵線]]延長至[[威爾斯登交匯站|威爾斯登交匯站]]。1917年4月16日延長至[[沃特福德交匯站|沃特福德交匯站]]。同年8月1日[[石橋公園站|石橋公園站]]啟用。1933年7月3日[[南肯頓站|南肯頓站]]啟用。

於1982年縮短至石橋公園。而後於1984年重新延長至[[哈羅及威爾德斯通站|哈羅及威爾德斯通站]]，然而假日仍然只提供有限度服務。1989年起回復全線正常服務。

===史丹摩支線===
由於大都會線[[貝克街站|貝克街站]]至[[芬奇利路站|芬奇利路站]]一段長期嚴重擠塞，貝克盧線於1939年11月20日取代[[大都會線|大都會線]]史丹摩支線的地位。直至貝克盧線同樣遇上擠塞問題，於是此段支線於1979年5月1日改由新設之[[銀禧線|銀禧線]]取代。

===延伸計畫===
自1931年起，倫敦交通局便有意延伸貝克盧線，而新的南端終點站則有[[丹麦山站|丹麦山]]、[[劉易舍姆站|劉易舍姆]]和{{Link-en|坎伯韦尔站|Camberwell_railway_station_(London)|坎伯韦尔}}等站。

== 服務 ==
貝克盧線離峰時段班次如下：
* 每小時6班來往哈羅及威爾德斯通至象堡
* 每小時3班來往石橋公園至象堡
* 每小時11班來往女王公園至象堡

== 車廠 ==
[[File:Bakerloo_line_depot_at_elephant.jpg|thumb]]
貝克盧線目前由三個車廠負責維修，主車廠位於[[石橋公園站|石橋公園]]{{ref|map 26|map 26}}於1978年4月9日啟用，原址為[[英國鐵路|英國鐵路]]發電廠。原車廠於{{link-en|倫敦路 (南華克)|London Road, Southwark|倫敦路}}{{ref|map 27|map 27}}（介乎象堡至蘭貝斯北）及一個[[女王公園站|女王公園]]北側的車廠，{{ref|map 28|map 28}}建於1915年。女王公園車廠是倫敦地鐵網絡中一個獨特的車廠，乘客列車直接通過車廠。

貝克盧線仍有列車開往沃特福德時曾有一個額外的車廠，位於[[克諾斯萊綠地|克諾斯萊綠地]]，這個車廠隨着貝克盧線服務的縮短於1985年11月關閉。

貝克盧線仍有史丹摩支線時，位於史丹摩支線的尼斯登車廠為主車廠，然而自從銀禧線取代史丹摩支線後，貝克盧線新建了石橋公園車廠。

倫敦路車廠較為特別，雖然車廠位於地面而軌道位於地底，兩者以短而陡的支線隧道連接。
