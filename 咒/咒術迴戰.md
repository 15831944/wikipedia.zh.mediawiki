《'''咒術迴戰'''》（{{lang-ja|呪術廻戦}}）是[[日本|日本]][[漫畫家|漫畫家]]{{link-ja|芥見下下|芥見下々}}繪畫的[[日本漫畫|漫畫]]作品，[[集英社|集英社]]出版，於漫畫雜誌《[[週刊少年Jump|週刊少年Jump]]》2018年14號開始連載<ref>{{cite web|script-title=ja:ジャンプ編集部満場一致で連載決定のダークファンタジー「呪術廻戦」1巻|url=https://natalie.mu/comic/news/289750|website=[[Natalie_(網站)|Natalie]]|accessdate=2018年12月4日|language=ja|date=2018年5月30日}}</ref>。與作者另一作品《東京都立咒術高等專門學校》同一世界觀。

== 故事簡介 ==
虎杖悠仁是一位體育萬能的高中生，因為某種理由，想要每天17點之前回家的虎杖悠仁來到了不強制出席的靈異現象研究會，享受著悠閒的活動。

這樣的某一天，來學校尋找被封印的詛咒之物的青年伏黒恵出現在了虎杖的面前。他為了從「咒物」危機中解救學長姊，而吞下了詛咒的手指，讓「宿儺」這種詛咒跟自己合而為一。
之後他加入了專門培養咒術師的學校「咒術高專」，並遇到了伏黑惠與釘崎野薔薇這兩位同學。

某日，突然出現「特級咒物」，他們三人就奉命到現場支援。
為了實現爺爺要他「助人」的遺言，虎杖將會繼續與「詛咒」奮鬥下去！

=== 主要角色 ===
; {{nihongo|虎杖悠仁|虎杖 悠仁（いたどり ゆうじ）}}
:本作主角，身高173cm，體重80kg，體脂率極低，有極強的運動神經。
:由於要探望住院的爺爺的關係，每天都在五點前離開學校。爺爺過世後，在醫院遇見前來尋找宿儺手指的伏黑。之後意外吞下了宿儺的手指，成為了最強詛咒宿儺的附身者。
: 在被高專帶走並審判後，成為了東京咒術高專一年級男學生。五條評價他日後會成為與他並肩的術師。

; {{nihongo|五條悟|五条 悟（ごじょう さとる）}}
: 東京咒術高專一年級男班主任。28歲，身高超過190cm。大多時間為刺蝟頭(抓出來的)，並以黑布遮蓋住眼睛作為眼罩用的造型示人，偶爾會將頭髮放下。四名特級咒術師之一。因需要用動腦想很多事情來處理，所以嗜吃甜點，是甜食控。
:眼罩下藏著一對特別的眼睛，透過眼睛能夠展開領域「無量空處」，強制指定對象接收無止盡的情報。

=== 咒靈 ===
; {{nihongo|兩面宿儺|両面宿儺（りょうめんすくな)}}
: 「詛咒之王」，擁有四隻手兩張臉的虛構鬼神，是實際存在的人。
: 當時咒術處在全盛時代，咒術師糾集總立向他挑戰卻失敗了，後來冠上『兩面宿儺』的名字。死後化身成特級咒物被切掉二十個手指封印並分落在日本各處，經過一千年後連屍蠟銷毀不了。性格殘忍冷酷，身上帶著奇特刺青紋路，喜好殺戮，特別是女人或小孩。因主角虎杖攸仁吃掉宿儺手指附身在他身上，現今待在主角體內，伺機蠢蠢欲動支配肉體來毀滅世界。

== 用語解說 ==
; {{nihongo|東京都立咒術高等專門學校|東京都立呪術高等専門学校}}
: 該校為日本僅有的兩間咒術教育機關之一（對外形象是私立的宗教系學校）。
: 許多咒術師在畢業後仍會以此為起點展開活動。
: 校舍因建在郊外容易讓人懷疑這裡是不是東京。

; {{nihongo|京都友誼校交流會|京都姉妹校交流会}}
: 和京都的另一所高專開的交流會，以二三年級為主的活動。兩間學校的校長各提交一種比賽方法，分兩天進行。第一天為團體戰，依據拔除詛咒的多寡來決出勝負的咒靈討伐競賽，第二天為個人戰，除殺人外甚麼事情都能做的咒術合戰。今年的交流會由於憂太在國外，秤停學，因此悠人等一年級特例參加。
