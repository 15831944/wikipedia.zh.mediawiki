'''萼菊石'''（[[學名|學名]]：{{lang|la|'''''Calycoceras'''''}}）是生存在[[白堊紀|白堊紀]][[海洋|海洋]]中的一屬菊石，由於其粗壯的肋條和圓胖的殼形，萼菊石和其近親可能不擅長於水平游動，而較擅長垂直地在深淺海域之間往返遷移，其粗大的體管或許有助於讓牠們能更快速、大輻度的調節氣室浮力。<ref name="Calycoceras">{{Cite web |url = http://liusir.pixnet.net/blog/post/248632709-calycoceras-%E8%90%BC%E8%8F%8A%E7%9F%B3-%28%E7%BE%8E%E5%9C%8B%29 |title = Calycoceras 萼菊石 (美國) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-14 }}</ref>其[[化石|化石]]分布於[[北美|北美]]、[[英國|英國]]、[[保加利亞|保加利亞]]、[[法國|法國]]、[[巴布亞紐幾內亞|巴布亞紐幾內亞]]和[[南極洲|南極洲]]等地。<ref name="Fossilworks">{{Cite web |url = http://fossilworks.org/bridge.pl?a=taxonInfo&taxon_no=14666 |title = Fossilworks: Calycoceras |language = en |accessdate = 2018-08-14 }}</ref>

== 特徵 ==
萼菊石的外殼有著粗壯的肋條，腹側具有明顯的5排結節。和其他許多刺菊石類一樣，萼菊石殼體內部的體管相當粗大。<ref name="Calycoceras" />
