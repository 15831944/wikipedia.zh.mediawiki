'''则拉县'''（{{vie|v='''Huyện Chợ Lách'''}}）是[[越南|越南]][[槟椥省|槟椥省]]下辖的一个县。

== 行政区划 ==
则拉县下辖1市镇10社。
*则拉市镇（Thị trấn Chợ Lách）
*和义社（Xã Hòa Nghĩa）
*兴庆中B社（Xã Hưng Khánh Trung B）
*隆泰社（Xã Long Thới）
*富奉社（Xã Phú Phụng）
*富山社（Xã Phú Sơn）
*山定社（Xã Sơn Định）
*新禅社（Xã Tân Thiềng）
*永平社（Xã Vĩnh Bình）
*永和社（Xã Vĩnh Hòa）
*永成社（Xã Vĩnh Thành）

== 经济 ==
以农业为主，主要产[[稻米|稻米]]，[[榴莲|榴莲]]，[[山竹|山竹]]，[[红毛丹|红毛丹]]等作物。

== 交通 ==
则拉县以公路和水路为主，到[[周城县_(槟椥省)|周城县]]，[[前江省|前江省]][[该礼县|该礼县]]只能靠渡轮来往，57号国道贯通南北。
