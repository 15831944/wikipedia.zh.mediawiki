《'''魔獸世界'''》（{{lang-en|''World of Warcraft''}}，[[縮寫|縮寫]]作 '''{{lang|en|WoW}}'''），簡稱'''魔獸'''，是由[[暴雪娛樂|暴雪娛樂]]製作的一款[[大型多人在線角色扮演遊戲|大型多人在線角色扮演遊戲]]。魔獸世界的劇情開端點是在《魔獸爭霸3：寒冰霸權》結束之後的故事。

暴雪於2001年9月2日宣布製作魔獸世界<ref>{{cite web |url = http://www.games-fusion.net/press/content/blizzard_entertainment_announc.php |archiveurl = https://web.archive.org/web/20071103122615/http://www.games-fusion.net/press/content/blizzard_entertainment_announc.php |archivedate = 2007-11-03 |title = Blizzard Entertainment announces World of Warcraft}}</ref>。魔獸世界於2004年年中在北美公開測試，同年11月23日，魔獸[[跨媒體製作|跨媒體製作]]系列10周年之際，魔獸世界在[[北美|北美]]與[[大洋洲|大洋洲]]正式發行。

遊戲的第一部資料片，《[[魔獸世界：燃燒的遠征|燃燒的遠征]]》於2007年1月16日發布；

第二部《[[魔獸世界：巫妖王之怒|巫妖王之怒]]》於2008年11月13日發布；

第三部《[[魔獸世界：浩劫與重生|-{zh-hans:大地的裂变;zh-hant:浩劫與重生}-]]》於2010年12月7日發布；

第四部《[[魔兽世界：熊猫人之谜|-{zh-hans:熊猫人之谜;zh-hant:潘達利亞之謎}-]]》於2012年9月25日發布；

第五部資料片《[[魔獸世界：德拉諾之霸|-{zh-hans:德拉诺之王;zh-hant:德拉諾之霸}-]]》於2014年11月13日發布<ref>{{cite web | url = http://www.ign.com/articles/2013/11/08/world-of-warcraft-warlords-of-draenor-announced | title = World of Warcraft: Warlords of Draenor Announced |publisher=[[IGN|IGN]] |date=2013-11-08 | last=Dyer |first=Mitch}}</ref>；

第六部資料片《[[魔獸世界：軍臨天下|-{zh-hans:军团再临;zh-hant:軍臨天下}-]]》於2015年8月6日公布計劃，並在2016年9月1日正式發行<ref>{{Cite web|url = http://digi.163.com/15/0807/06/B0D718FO00162OUT.html|title = 《魔獸世界》7.0資料片公布 軍臨天下|accessdate = 2015-08-07|author = CNBETA|date = 2015-08-07|publisher = 网易}}</ref>；

第七部資料片《[[魔兽世界：争霸艾泽拉斯|-{zh-hans:争霸艾泽拉斯;zh-hant:決戰艾澤拉斯}-]]》於2017年11月4日發布消息，在2018年8月14日上線<ref>{{cite web|author1=GNN 記者 Edward|title=【BZ 17】《魔獸世界》公布全新資料片《魔獸世界：決戰艾澤拉斯》|url=https://gnn.gamer.com.tw/7/154817.html|website=巴哈姆特|publisher=巴哈姆特|accessdate=2017-11-08}}</ref>。

截至2008年底，魔獸世界於全球已有超過1150萬的用戶<ref>[https://web.archive.org/web/20081224162904/http://eu.blizzard.com/en/press/081223.html Blizzard Entertainment - Press Release]</ref>，並被收錄至2008年的《[[健力士世界紀錄大全|吉尼斯世界纪录]]》中<ref>{{cite web |url = http://gamers.guinnessworldrecords.com/records/pc_gaming.aspx |title = Guinness World Records Gamer's Edition - Records - PC Gaming |deadurl = yes |archiveurl = https://web.archive.org/web/20080405020338/http://gamers.guinnessworldrecords.com/records/pc_gaming.aspx |archivedate = 2008-04-05 }}</ref>。

== 遊戲内容 ==
魔獸世界的背景故事缘起于《[[魔獸爭霸III：寒冰霸權|冰封王座]]》四年後的[[艾澤拉斯|艾澤拉斯]]大陸。大陸上联盟与部落两大势力形成了对立状态，而新的威胁正在不断产生。

《魔獸世界》現有的遊戲内容基於本傳以及其後資料片的修改。
=== 角色、種族與職業 ===
[[File:Wowdwarf.jpg|thumb]]
{{Main|魔獸世界種族列表}}

通過註冊帳號可以在《魔獸世界》中建立角色。目前人物角色的最高等級為120級。

根據遊戲內載入介面的“小提示”（Tips）的內容，一個帳號在同一個服务器中能建立的角色個數上限為11個（第六個資料片【軍團再臨】前夕開放後，上限改為12個），而不同服务器總共相加至多能建立50個角色。

《魔獸世界》共有13個種族可供玩家選擇，這些種族被劃分在兩個對立的陣營中：[[聯盟_(魔獸世界)|聯盟]]和[[部落_(魔獸世界)|部落]]，在遊戲裡，對立陣營因為語言不通，所以無法直接溝通，每個種族的角色都可以選擇性别、髮型、臉型和特徵。

* 聯盟陣營（Alliance）由[[魔獸世界種族列表#人類|人類]]（{{lang|en|Human}}）、[[魔獸世界種族列表#矮人|矮人]]（{{lang|en|Dwarf}}）、[[魔獸世界種族列表#夜精靈|-{zh-hans:暗夜精灵;zh-hant:夜精靈}-]]（{{lang|en|Night Elf}}）、[[魔獸世界種族列表#地精|-{zh-hans:侏儒;zh-hant:地精}-]]（{{lang|en|Gnome}}）、[[魔獸世界種族列表#德莱尼|德莱尼]]（{{lang|en|Draenei}}）以及[[魔獸世界種族列表#狼人|狼人]]（{{lang|en|Worgen}}）組成。

* 部落陣營（Horde）由[[魔獸世界種族列表#獸人|獸人]]（{{lang|en|Orc}}）、[[魔獸世界種族列表#牛頭人|牛頭人]]（{{lang|en|Tauren}}）、[[魔獸世界種族列表#食人妖|-{zh-hans:巨魔;zh-hant:食人妖}-]]（{{lang|en|Troll}}）、[[魔獸世界種族列表#不死族（被遺忘者）|亡靈/被遺忘者]]（{{lang|en|Undead/Forsaken}}）、[[魔獸世界種族列表#血精靈|血精靈]]（{{lang|en|Blood Elf}}）、[[魔獸世界種族列表#哥布林|-{zh-hans:地精;zh-hant:哥布林}-]]（{{lang|en|Goblins}}）組成。

* 中立陣營（Alliance/Horde）：熊貓人（{{lang|en|Pandaren}}）。

* 玩家在遊戲中也會碰到其他種族的[[非玩家角色|非玩家角色]]，如-{zh-hans:食人魔;zh-hant:巨魔}-（{{lang|en|Ogres}}）、[[魔獸世界種族列表#魚人|魚人]]（{{lang|en|Murlocs}}）、納迦（{{lang|en|Nagas}}）等。

《魔獸世界》中一共有12種職業：戰士（{{lang|en|Warrior}}）、盜賊（{{lang|en|Rogue}}）、[[獵人_(魔獸世界)|獵人]]（{{lang|en|Hunter}}）、法師（{{lang|en|Mage}}）、術士（{{lang|en|Warlock}}）、牧師（{{lang|en|Priest}}）、德魯伊（{{lang|en|Druid}}）、聖騎士（{{lang|en|Paladin}}）、薩滿（{{lang|en|Shaman}}）、死亡騎士（{{lang|en|Death Knight}}）（巫妖王之怒版本新增）、武僧（{{lang|en|Monk}}）（熊猫人之谜版本新增）以及-{zh-hans:恶魔猎手;zh-hant:惡魔獵人}-（{{lang|en|Demon Hunter}}）（军团再临版本新增）。並非任意種族都能選擇任意職業，根據遊戲的背景設定，不同的種族可選的職業也各有不同的特色。

在8.0资料片《争霸艾泽拉斯》中，联盟和部落分别增加四个新的同盟种族，玩家需要达到相应阵营的声望并完成一系列同盟种族解锁任务后才可以创建这些种族的角色，且与主要种族出生等级为1级不同，同盟种族新建角色即为20级。

*聯盟陣營：虚空精灵、光铸德莱尼、黑铁矮人、库尔提拉斯人类。

* 部落陣營：至高岭牛头人、夜之子、玛格汉兽人、赞达拉巨魔。

=== 角色成长 ===
遊戲中的角色設有四項基本屬性（力量、敏捷、耐力、智力）以及其它强化属性(迴避、招架；早期版本還有命中，但已被移除)。屬性影響著角色的各種能力，不同職業的角色需要注重的屬性有所不同；基本屬性的強弱以數字表示，數字越大則越強。

===專業技能===
《魔獸世界》總共有11種專業技能可供玩家選擇，分别是：採礦、剝皮、草藥學、裁縫、附魔、鍛造、工程學、珠寶設計、製皮、鍊金術以及銘文學，這些專業技能分為採集型和製造型兩種而玩家可以從中選擇兩種。玩家在遊戲中通過專業獲得的資源和成品可以放到城市中的拍賣場與其他玩家進行交易。很多玩家選擇兩個互相搭配的商業技能，使其可以在採集到需要的資源後馬上進行製造。除此之外，遊戲中還有五種輔助技能：騎術、急救、烹飪、釣魚和考古學。
在8.0《決戰艾澤拉斯》已將輔助專業技能-急救移除，並將原本屬於急救技能的內容移至裁縫。

=== 副本（Raid） ===
'''副本'''在《魔獸世界》中，概念類似其他同類型遊戲的「地下城」，是要跨過傳送門而進入的特殊區域。為了避免不同隊伍間的相互干擾，針對同一個副本，系統會為每一支進入副本的隊伍產生獨立的地下城，並以地圖資料作為區分。因此在副本裡，不會看到隊伍成員以外的其他玩家，同樣的，進行副本任務時，隊伍外的玩家也無法對該隊伍做出支援。

《魔獸世界》擁有數量龐大的地下城（副本），擁有大量擊殺難度超過普通怪物的精英級怪物甚至首領級怪物（在玩家達到最高等级後等級仍顯示為骷髏的怪物），而任務獎勵的和怪物及首領（{{lang|en|Boss}}）掉落的裝備也會比普通任務的獎勵擁有更好的品質。每一個副本中，最多容許一個隊伍進行探險。根據地下城的規模不同一般將副本分為分為'''-{zh-hans:地下城;zh-hant:地城}-'''（最多允許5人進入）、'''團隊副本'''（由一個固定上限為10、20、25、40人或一個10-30人的隊伍進入）和'''-{zh-hans:场景战役;zh-hant:事件}-'''（一種特殊的副本形式，只允許一個3人小隊加入），而團隊副本的難度和強度都顯著高於-{zh-hans:地下城;zh-hant:地城}-以及-{zh-hans:场景战役;zh-hant:事件}-。

團隊副本通常都有冷卻時間，讓團隊無法靠連續完成同樣的副本來快速累積裝備。

=== 玩家對戰（PVP） ===
達到指定目標時取得勝利的玩家對戰可以賺取特定貨幣和陣營聲望，從而獲得面對玩家時更強的裝備或其他物品。

*'''戰場'''：陣營雙方各由10-40名玩家，於特定規則的戰場地圖進行對戰，略分為搶旗、領土佔領、資源爭奪及複合模式。

*'''地圖戰場'''：陣營雙方於特定地圖(地獄火半島、哈剌、冬臥湖)佔領重要據點以取得資源(BUFF、副本、榮譽獎章)。

*'''競技場'''：兩隊各由2人，3人或5人組成的隊伍進行生死鬥。

*'''決鬥'''：於野外地區插旗進行1對1生死鬥(無法擊殺玩家)。

野外阵营对战根据情况亦提供奖励。

=== 資料片 ===
{{Main|魔兽世界：燃烧的远征|魔兽世界：巫妖王之怒|魔獸世界：大地的裂变|魔兽世界：熊猫人之谜|魔兽世界：德拉诺之王|魔兽世界：军团再临|魔獸世界：決戰艾澤拉斯}}

2005年10月28日，暴雪公佈了《魔獸世界》的第一款資料片：《魔獸世界：燃燒的遠征》。游戏内容包括新大陸（外域）、角色等級上限提升到70級、重新設計各職業天賦的定位、飛行座騎、新的專業技能（珠寶加工）、新的副本和戰場，新種族：德萊尼（聯盟），血精靈（部落），各大主城開設理髮廳供玩家角色變換新造型。此资料片已於2007年正式上市。

《魔獸世界:巫妖王之怒》是魔獸世界系列遊戲的第二款資料片，此資料片於2007年8月在Blizzcon中正式公佈其開發資訊，並於2008年11月18日上市。游戏内容包括新英雄職業死亡骑士，开放了拥有众多新副本和新装备的北裂境，新專業銘文學，并将人物角色等级上限提升至80级，成就系統。

暴雪娛樂於[[BlizzCon|BlizzCon]] 2009中宣佈第三部資料片命名為《魔獸世界：浩劫與重生》(台服译法)/《魔兽世界：大地的裂变》（国服译法）並於2010年12月7日正式上線，遊戲舞台將回歸被死亡之翼破壞的艾澤拉斯大陸，同時亦有兩個新種族分別加入聯盟和部落。联盟方加入新种族：狼人；部落方加入新种族：哥布林。游戏内容将在改頭換面的艾泽拉斯大陆进行，重製了大部分艾泽拉斯地图與副本，增加了深岩之洲、海加尔、瓦斯琪尔、奥丹姆和暮光高地新区域，其中瓦斯琪尔是首次开放的水下地图。並且開放飛行座騎在艾澤拉斯飛行，增加新的第二专业技能：考古学，并将人物等级上限提升至85级。

2011年暴雪娛樂於[[BlizzCon|BlizzCon]]宣佈第四部資料片為《魔獸世界：潘達利亞之謎》。在《魔獸世界：潘達利亞之謎》中，玩家可以探索全新的大陸（潘達利亞），邁向全新的90級上限。另外，冒險者可以使用魔獸世界的第一個中立種族（熊貓人）來選擇投靠部落或聯盟，並使用全新的職業[[武僧|武僧]]來鑽研熊猫人的古拳法。重新設計職業定位（專精）與新天賦系統，帳號共享成就、坐騎與小寵物，新加入的小遊戲：戰寵系統，提供玩家在小寵物的收集與對戰的樂趣。

2013年暴雪于[[暴雪嘉年华|暴雪嘉年华]]中宣布第五部资料片是《魔兽世界：德拉诺之王》。在《魔兽世界：德拉诺之王》中，玩家可以探索新的大陆[[德拉诺|德拉诺]]，等级上限提升到了100级。

格林威治时间2015年8月6日下午6时，2015年德国科隆游戏展上，暴雪正式公布第六部资料片《魔兽世界：军团再临》，并展示了游戏细节。另外玩家可以探索新的大陆[[破碎群島|破碎群島]]，等级上限提升到了110级。新增英雄職業惡魔獵人且將專業技能等級上限提升至800。

2017年暴雪嘉年華公佈了新資料片，《魔獸世界：決戰艾澤拉斯》，等級上限提高到120級，部落將前往贊達拉說服食人妖海軍勢力加入戰局；而聯盟則將揚帆前往海上霸權王國庫爾提拉斯，抵達珍娜·普勞德摩爾的家園。

新開放的同盟種族能在任務後取得高嶺牛頭人、虛無精靈、黑鐵矮人等種族的信任，就能建立這些種族的新角色。

另外也公佈了《魔獸世界：經典伺服器》研發中的消息。

=== 正面评价 ===
* 2011年8月1日，中國大陸官方媒體[[CCTV|CCTV]]讚揚魔獸世界很有創意，一並稱赞美國[[暴雪|暴雪]]公司。同时也指責、批判國内的眾多網路遊戲缺乏創造力，良莠不齐<ref>[http://tech.hexun.com/2011-08-01/131965023.html] CCTV首次肯定《魔獸世界》，讚其有創意</ref>。

=== 负面评价 ===
* 美籍华人[[陶宏开|陶宏开]]在[[CCTV|CCTV]]表示，将《魔兽世界》与[[毒品|毒品]]相比，批评魔兽使青少年[[网络成瘾|网络成瘾]]。通过此次节目陶宏开获得了较大的知名度，借此其开办的“网瘾治疗基地”年获利千万余元。其本人近来代言了国产网络游戏《亮剑》。
* 獲得IGF設計創新獎者的程式設計師[[喬納森·布羅|喬納森·布羅]](Jonathan Blow)稱「在你的頭腦當中，《魔獸世界》製造的慾望揮之不去，而且它們又來得如此湊巧。」布羅說，「就像是廣告和對品牌的認同一樣，廠商透過各種推銷活動來製造人們對其的慾望與認同。就遊戲來說也是一樣，因為人是由自身的起源和環境所構成的產物，而我們一直在替他們製造這種環境，並幫他們決定何去何從。」「作為一名設計師，我希望能看到我們利用這一權力來轉變<ref>{{cite web |url = http://news.duowan.com/0711/60258297164.html |title=國外著名遊戲設計師稱魔獸世界就像毒品 |accessdate=2011-11-25}}</ref><ref>{{cite web |url = http://www.gamasutra.com/php-bin/news_index.php?story=16392 |title=MIGS 2007: Jonathan Blow On The 'WoW Drug', Meaningful Games |accessdate=2011-11-25}}</ref>。」

=== 媒介 ===
* 《魔兽世界》曾經在美國動畫连续剧[[衰仔樂園|-{zh-hk:衰仔樂園;zh-hans:南方公园; zh-tw:南方四賤客;}-]]第十季第八集《[[要愛，不要魔獸世界|要愛，不要魔獸世界]]》（{{lang-en|Make Love, Not Warcraft}}）中被谐仿，劇中的主角均進入了遊戲世界，而暴雪公司也参与了该集的制作并在剧中出现。剧集播出后受到好评<ref>{{en}}{{cite web |url = http://tv.ign.com/articles/737/737420p1.html |title=《要爱，不要魔兽世界》剧评 |accessdate=2009-03-06 |author=Dan Iverson |date=2006-10-05 }}</ref>。

=== 玩魔兽世界的明星 ===
* [[姚明|姚明]]玩《魔兽世界》為許多人所知，据传游戏角色为联盟猎人。曾有记者向在养伤期间的姚明问及游戏情况，姚笑答被一高等级牛头人战士杀死，并称对方“太没人品”，“要强烈谴责”<ref>{{en}}http://www.youtube.com/watch?v=4M9c28Go6r8</ref>。
* 曾保护[[北京奥运会|北京奥运会]]火炬的残疾运动员[[金晶|金晶]]也是《魔兽世界》的爱好者。游戏角色所在服为大陆一区的洛萨服务器，自称是一名“骨灰级玩家”<ref>{{zh-cn}}{{cite web|url=http://www.wowchina.com/news/media/90785.shtml|title=新浪专访金晶|accessdate=2009-03-13|author=新浪游戏|date=2008-04-11|deadurl=yes|archiveurl=https://web.archive.org/web/20090321100457/http://www.wowchina.com/news/media/90785.shtml|archivedate=2009-03-21}}</ref>。
* [[第81届奥斯卡金像奖|第81届奥斯卡金像奖]]以[[返老还童|返老还童]]荣获最佳视觉效果的获奖者之一Steve Preeg被披露是一位《魔兽世界》爱好者，并且作为公会会长带领会员击败了《巫妖王之怒》地下城几乎所有首领<ref>{{zh-cn}}{{cite web |url = http://games.sina.com.cn/o/g/2009-02-25/10026229.shtml |title=魔兽公会会长获得奥斯卡小金人 |accessdate=2009-03-16 |author=新浪游戏 |date=2009-02-25 }}</ref>。
* [[谢霆锋|谢霆锋]]一次在拍戏间隙被记者拍到玩魔兽世界的画面，职业盗贼<ref>{{zh-cn}}{{cite web |url = http://www.youtube.com/watch?v=VejHQKUqfzA&feature=share |title=Eric Kwok於Swing演唱會向葉佩雯求婚，並早就準備律師同時簽紙舉行婚禮！  |accessdate=2009-03-13 |date=2008-04-11 }}</ref><ref>{{zh-cn}}{{cite web |url = http://bbs.tianya.cn/post-funinfo-7523788-1.shtml#16 |title=说说我做五年酒店遇到的各种奇葩事（禁止转载）}}</ref>。
* 2009年8月3日香港作曲人[[Eric_Kwok|Eric Kwok]]於叱吒903電台節目早霸王，自稱長期沉迷於本遊戲，並說謝霆峰告訴他等級70只是遊戲的開始<ref>{{zh-cn}}{{cite web |url = http://www.youtube.com/watch?v=VejHQKUqfzA&feature=share |title=Eric Kwok於Swing演唱會向葉佩雯求婚，並早就準備律師同時簽紙舉行婚禮！ |accessdate=2009-03-13 |date=2008-04-11 }}</ref>。
* 2012年伦敦奥运会男子双杠冠军、男子团体冠军[[冯喆|冯喆]]也是一名魔兽玩家<ref>{{zh-cn}}{{cite web |url = http://w.163.com/12/0731/15/87OI171U00314C3U.html |title=交任务得金牌：奥运体操冠军冯喆自爆为魔兽玩家}}</ref>。
* 演员[[万茜|万茜]]在湖南卫视节目[[声临其境|声临其境]]为一段魔兽世界CG配音后，在微博自述曾是一位魔兽世界玩家。<ref>{{zh-cn}}{{cite web |url = http://m.weibo.cn/128476111/4341850019795723 |title=万茜：本人当年的确是一名魔兽玩家}}</ref>
* 韓國藝人團體Super junior團員金希澈也曾經在Twitter上說過：「我的WoW的牧師職業是很強的」。<ref>[https://games.sina.com.cn/o/z/wow/xszt.shtml]</ref>
