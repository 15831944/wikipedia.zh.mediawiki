《'''魔女戀愛日記'''》（{{lang-ja|魔女こいにっき}}）是Qoobrand在2014年5月30日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web |title=Qoobrand 1st product『魔女こいにっき』がマスターアップ ―2014年5月30日発売 |url=http://moedigi.com/game/2014/05/qoobrand-1st-project-master-up/ |website=moedigi |accessdate=2019-04-03 |language=ja}}</ref>，2015年12月24日發售[[PlayStation_Vita|PlayStation Vita]]版《魔女戀愛日記 Dragon×Caravan》。<ref>{{cite web |last1=Inc |title=恋愛ADV「魔女こいにっき Dragon×Caravan」PS Vita移植版が12月24日に発売。新規エピソードやキャラクターなどの追加要素も |url=https://www.4gamer.net/games/315/G031531/20150831013/ |website=4Gamer.net |language=ja |date=2015/08/31}}</ref>

==故事==
南乃愛麗絲在碧方學園的鐘塔找到一本奇妙的日記本，日記中寫者櫻井拓海與愛麗絲身邊認識女性的事情。然而現實中的她們並不認識拓海，為此愛麗絲便決定要開始調查真相。
