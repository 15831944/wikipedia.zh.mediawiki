在[[J·K·羅琳|J·K·羅琳]]的系列小說《[[哈利波特|哈利波特]]》中，'''魔杖'''（{{lang-en|Wand}}）是[[巫師_(哈利波特)|巫師]]用來施展[[魔法|魔法]]的一種工具，也是戰鬥時必備的武器。

== 材質 ==
每根魔杖是由一種特殊木材作成的。[[哈利·波特_(角色)|哈利·波特]]的魔杖材質是[[冬青|冬青]]木，而[[佛地魔|佛地魔]]魔杖的材質則是{{le|歐洲紫杉|yew|紫杉木}}；作者羅琳指出，在[[歐洲|歐洲]]的傳統裡冬青樹象徵著驅除邪惡，而紫杉樹則象徵著死亡與重生<ref>{{Cite web|url=http://www.jkrowling.com/textonly/en/extrastuff_view.cfm?id=18|title=Section: Extra Stuff - WANDS|accessdate=2018-10-15|author=J.K.Rowling|publisher=J.K.Rowling Official Site|archiveurl=https://web.archive.org/web/20090221005614/http://www.jkrowling.com/textonly/en/extrastuff_view.cfm?id=18|archivedate=2009-02-21}}</ref>。

具有魔法的杖芯（Wand Cores）則是魔杖的另一個重要組成部分，龍的心弦、[[鳳凰|鳳凰]][[羽毛|羽毛]]、[[獨角獸|獨角獸]]尾毛和迷拉的頭髮皆可用於杖芯<ref>{{Cite web|url=https://www.pottermore.com/writing-by-jk-rowling/wand-cores|title=Wand Cores|accessdate=2018-10-15|author=J.K. Rowling|publisher=Pottermore|language=en}}</ref>。哈利·波特與佛地魔兩人的魔杖杖芯皆是鳳凰[[佛客使|佛客使]]的羽毛<ref>{{Cite web|url=http://ent.sina.com.cn/m/f/2011-07-20/ba3364577.shtml|title=《哈利-波特》十周年回忆录之十大特殊配角(6)|accessdate=2018-10-15|date=2011-07-20|publisher=新浪网}}</ref>。

== 使用 ==
[[File:HP_Wand_lumos.gif|缩略图]]
幾乎所有的[[哈利波特中的魔法符咒|魔法符咒]]都是透過魔杖來施展的<ref>{{Cite web|url=http://www.quick-quote-quill.org/articles/2001/0301-comicrelief-staff.htm|title=Comic Relief live chat transcript, March 2001|accessdate=2018-10-15|publisher=Accio Quote!|archiveurl=https://web.archive.org/web/20080827220619/http://www.quick-quote-quill.org/articles/2001/0301-comicrelief-staff.htm|archivedate=2008-08-27}}</ref>。只有少數人能夠在不使用魔杖的情況下精準操控魔法。

在每個巫師購賣魔杖時，魔杖會自動選擇它的主人。第一集中[[奧利凡德|奧利凡德]]先生對[[哈利·波特_(角色)|哈利·波特]]說，如果使用了屬於別人的魔杖，效果就會大打折扣<ref name=":0">{{Cite book|title=哈利波特——神秘的魔法石|last=J·K·羅琳|publisher=皇冠文化|year=2001年3月|isbn=957-33-1724-9|location=臺北}}</ref>。

== 魔杖製造師 ==
[[英國|英國]]最著名的魔杖製造師為奧利凡德先生，他的商店位於[[斜角巷|斜角巷]]內<ref name=":0" />，佛地魔、哈利·波特等不少巫師的魔杖都是出自他之手。

在歐洲最著名的魔杖製造師為葛果羅威，他製造了[[維克多·喀浪|維克多·喀浪]]的魔杖。

== 魔法世界的標誌 ==
由J·K·羅琳和[[華納兄弟影業|華納兄弟影業]]共同擁有的[[跨媒體製作|跨媒體製作]]「{{Le|魔法世界 (跨媒體製作)|Wizarding World|魔法世界}}」（Wizarding World）的標誌由九支魔杖組成，九支魔杖的擁有者分別為佛地魔、[[榮恩·衛斯理|榮恩·衛斯理]]、[[妙麗·格蘭傑|妙麗·格蘭傑]]、哈利·波特、[[阿不思·鄧不利多|阿不思·鄧不利多]]、[[紐特·斯卡曼德|紐特·斯卡曼德]]、波本蒂娜·金坦、奎妮·金坦和[[蓋勒·葛林戴華德|蓋勒·葛林戴華德]]，皆為[[哈利波特系列電影|哈利波特系列電影]]和{{link-de|怪獸系列電影|Phantastische Tierwesen (Filmreihe)}}的主角；這個標誌在2018年3月發布的《[[怪獸與葛林戴華德的罪行|怪獸與葛林戴華德的罪行]]》電影預告中首度出現<ref>{{Cite news|url=https://www.pottermore.com/news/wizarding-world-brand-logo-launch|title=New logo marks an exciting year ahead for the Wizarding World|date=2018-03-12|newspaper=Pottermore|accessdate=2018-10-15|language=en}}</ref><ref>{{Cite news|url=https://www.hk01.com/%E9%9B%BB%E5%BD%B1/168187/%E6%80%AA%E7%8D%B8%E8%88%87%E7%89%A0%E5%80%91%E7%9A%84%E7%94%A2%E5%9C%B0-%E7%BA%8C%E9%9B%86%E9%A0%90%E5%91%8A%E5%90%91%E5%93%88%E5%88%A9%E8%87%B4%E6%95%AC-%E7%B4%90%E7%89%B9%E5%85%84%E5%BC%9F%E5%A4%A7%E6%88%B0%E8%91%9B%E6%9E%97%E8%8F%AF%E5%BE%B7|title=【怪獸與牠們的產地】續集預告向哈利致敬 紐特兄弟大戰葛林華德|author=林穎嵐|date=2018-03-14|newspaper=香港01|accessdate=2018-10-15|language=zh-HK}}</ref>。

羅琳創立的慈善機構「[[路摸思|路摸思]]」將於2018年10月18日至11月12日期間在[[倫敦|倫敦]]豎立9個巨型魔杖裝置，作為倫敦城市藝術計劃的一部分<ref>{{Cite news|url=https://www.syfy.com/syfywire/giant-harry-potter-wand-installation-sheds-light-on-london-charity|title=Giant Harry Potter wand installation sheds light on London charity|last=Oller|first=Jacob|date=2018-09-24|newspaper=Syfy|accessdate=2018-10-15|language=en}}</ref><ref>{{Cite news|url=https://www.breakingnews.ie/showbiz/giant-harry-potter-wands-to-light-up-london-for-jk-rowling-charity-870354.html|title=Giant Harry Potter wands to light up London for JK Rowling charity|date=2018-09-20|newspaper=Breaking News|accessdate=2018-10-15}}</ref>。

== 文化影響 ==
在《哈利波特》系列小說及後來的電影推出後，魔杖成為周邊商品之一，世界各地的一些哈利波特相關景點皆有販售魔杖<ref>{{Cite web|url=http://www.chinanews.com/gj/gj-xxsh/news/2010/02-07/2112915.shtml|title=哈利波特乐园将开放 卖魔杖魁地奇球让哈迷疯狂|accessdate=2018-10-15|date=2010-02-07|publisher=中新网}}</ref><ref>{{Cite web|url=http://japan.people.com.cn/n1/2016/0413/c368549-28273313.html|title=日本环球影城哈利波特园区将新增魔杖游乐项目|accessdate=2018-10-15|date=2016-04-13|publisher=人民网}}</ref>。也有書迷動手自製魔杖<ref>{{Cite web|url=http://cnews.chinadaily.com.cn/2016-12/16/content_27691545.htm|title=你要制杖吗？英国兄妹自制超还原魔杖|accessdate=2018-10-15|date=2016-12-16|publisher=中国日报网}}</ref>。
