《'''魔球'''》是[[日本|日本]]作家[[東野圭吾|東野圭吾]]的[[長篇小說|長篇]][[推理小說|推理小說]]。是東野圭吾早期的青春校園推理小說，講述開陽高中棒球社接連發生的命案。於1988年由[[講談社|講談社]]發行單行本。1991年講談社發行文庫版。

== 故事簡介 ==
開陽高中的棒球部主將北岡明與愛犬一同遇害。當警方正在追查兇手時，另一名被稱為天才[[投手|投手]]的主將須田武志也同樣遇害，屍體的右手臂憑空消失。這兩件兇殺案的關連於「{{link-ja|魔球 (棒球)|魔球|魔球}}」，隨著謎團逐漸解開，不為人知的陰暗面也慢慢浮現……

== 登場人物 ==
; {{nihongo|須田 武志|{{ruby|須田|すだ}} {{ruby|武志|たけし}}|Suda Takeshi}}
: 開陽高中的三年級學生。棒球部的投手。被喻為天才投手，以進入職棒生涯為目標。

; {{nihongo|北岡 明|{{ruby|北岡|きたおか}} {{ruby|明|あきら}}|Kitaoka Akira}}
: 被害者。開陽高中的三年級學生。擔任棒球部的[[捕手|捕手]]，也是棒球部的主將。

; {{nihongo|須田 勇樹|{{ruby|須田|すだ}} {{ruby|勇樹|ゆうき}}|Suda Yuuki}}
: 須田武志的弟弟。開陽高中的二年級學生。與哥哥不同，是個頭腦靈光的學生。

; {{nihongo|森川|{{ruby|森川|もりかわ}}|Morikawa}}
: 開陽高中的物理老師，棒球隊的教練。

; {{nihongo|手塚 麻衣子|{{ruby|手塚|てづか}} {{ruby|麻衣子|まいこ}}|Tezuka Maiko}}
: 開陽高中的國語老師。

; {{nihongo|蘆原 誠一|{{ruby|蘆原|あしはら}} {{ruby|誠一|せいいち}}|Ashihara Seiichi}}
: 東西電機的離職員工。因為工業事故右腳受傷而離職，曾在東西電機打過職業棒球。
