{{日本作品|魔術快斗|まじっく快斗}}，是日本一部未完結的都市奇幻題材[[漫畫|漫畫]]，作者是[[青山剛昌|青山剛昌]]。於《[[名偵探柯南|名偵探柯南]]》中有大量客串，而《名偵探柯南》中的角色也有於本劇客串。歷史雖早於連載數十冊的《名偵探柯南》，但因無固定連載，目前只出版5冊單行本。動畫版初時主要依附於相關作品《名偵探柯南》演出，2010年開始該劇組也著手獨立製作《神偷怪盜》的獨立動畫<ref>{{cite web |title=まじっく快斗 |publisher=Animax网站 |url=http://www.animax.co.jp/program/NN10000960 |language=ja |accessdate=2014-09-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20100414005616/http://www.animax.co.jp/feature/index.php?code=NN10000960 |archivedate=2010-04-14 }}</ref>。《[[周刊少年Sunday|周刊少年Sunday]]》2014年35号公布制作电视动画新作的消息，制作公司为[[A-1_Pictures|A-1 Pictures]]<ref name="ANN140725">{{cite news |title=Magic Kaito Manga Gets TV Anime by A-1 Pictures |publisher=[[动画新闻网|动画新闻网]] |url=https://www.animenewsnetwork.com/news/2014-07-25/magic-kaito-manga-gets-tv-anime-by-a-1-pictures/.77015 |language=en |date=2014-07-25 |accessdate=2014-07-25 }}</ref>，同年10月4日起播出。

== 故事簡介 ==
[[File:legendarythief.jpg|thumb]]
喜歡魔術的高中生[[怪盗基德|黑羽快斗]]，偶然發現了藏在自己房間裡的密室。這個密室是他的父親──世界知名的[[魔術師|魔術師]][[怪盗基德|黑羽盜一]]，在8年前一次事故中神秘死去前所使用的。
密室中存放著8年前消失的神秘大盜怪盜基德的裝備和服裝。黑羽快斗穿上怪盜基德的服裝，並與父親多年前的助手，如今正在假扮怪盜基德的寺井黃之助相遇，才知曉自己的父親當年是死於謀殺。為了揭開父親的死因，黑羽快斗決定繼承父親怪盜基德的身份，並且調查父親的死因。

在調查的時候，黑羽快斗偶然和一支稱他為「黑羽盜一」的神秘組織相遇。原來這個組織為了尋找一種可以帶來長生不老的神秘寶石——命運之石「潘朵拉」，而將妨礙到他們的黑羽盜一殺死。快斗因此開始了與神秘組織的交鋒，試圖阻止他們的野心。

== 登場角色 ==
=== 主要人物 ===
* {{nihongo|'''[[怪盗基德|黑羽快斗]]／怪盜基德'''|黒羽 快斗（くろば かいと）／怪盜キッド（かいとうキッド）}}
: 聲優：[[山口勝平|山口勝平]]<ref name='AnimeNewsNet2014-09-01'>{{cite news |title=Kappei Yamaguchi, M.A.O Star in Magic Kaito TV Anime Series |publisher=[[动画新闻网|动画新闻网]] |language=en |url=https://www.animenewsnetwork.com/news/2014-09-01/kappei-yamaguchi-m.a.o-star-in-magic-kaito-tv-anime-series/.78249 |date=2014-09-01 |accessdate=2014-09-01 }}</ref>（日本）；[[魏伯勤|魏伯勤]]<ref>《名偵探柯南》TV 76、515～628集、《魔術快斗》TV 1～6集</ref>→[[官志宏|官志宏]]<ref>《名偵探柯南》TV 132～470集；M 3～14部</ref>→[[曹冀鲁|曹冀鲁]]
: 本作的男主角。一名熱愛[[魔術|魔術]]的男高中生，為了尋找父親的死因，而成為第二代「怪盜基德」，其國際代號1412，又有「平成的亞森‧羅蘋」、「世紀末的魔術師」和「月光下的魔術師」之稱，主要道具是撲克牌槍。劇中快斗與青子對彼此的喜歡之意顯而易見，雖未正式表白。此外，最大弱點是怕魚。
: 在《[[名偵探柯南|名偵探柯南]]》中以怪盜基德身份登場，與主角[[江戶川柯南|江戶川柯南]]是對手。

* {{nihongo|'''中森青子'''|中森 青子（なかもり あおこ）}}
: 聲優：[[岩居由希子|岩居由希子]]→[[高山南|高山南]]／[[藤村步|藤村步]]／[[市道真央|M・A・O]]<ref name='AnimeNewsNet2014-09-01'/>（日本）；[[陶敏嫻|陶敏嫻]]／[[楊凱凱|楊凱凱]]（台灣）
: 本作女主角。善良活潑的女高中生，和黑羽快斗是同學兼[[青梅竹馬|青梅竹馬]]，中森銀三的獨生女。因為怪盜基德總是把自己的父親耍得團團轉，所以非常討厭怪盜基德，卻喜歡著黑羽快斗（劇中尚未正式告白）。

* {{nihongo|'''中森銀三'''|中森 銀三（なかもり ぎんぞう）}}
: 聲優：[[石塚運升|石塚運升]]（日本）；[[孫中台|孫中台]]→[[劉傑_(配音員)|劉傑]]
: 警視廳搜查二課的警部<ref>{{cite web |title=中森銀三 |url=http://www008.upp.so-net.ne.jp/conanmuseum/Kaito/jinbutu/keibu.html |publisher=魔術快斗資料館 |language=ja |accessdate=2014-09-10 }}</ref>，為中森青子的父親。從黑羽盜一的時代就在追捕怪盜基德，為了抓住怪盜基德甚至可以不管殺人案件<ref>第一話 甦醒的怪盜（{{lang|ja|「蘇る怪盗」}}） ISBN 978-4-09-122081-3（日文版） ISBN 978-7-80-664799-8（中文版）</ref>。有權力調動大批員警和裝甲車、直升機，抓捕時總是大張旗鼓，在怪盜基德面前總是棋差一著，每次都是狼狽而歸。
: 在《[[名偵探柯南|名偵探柯南]]》中也有客串出場，出場率僅次於怪盗基德。

=== 黑羽家相關者 ===
* {{nihongo|'''黑羽盗一'''|黒羽 盗一（くろば とういち）}}
: 聲優：[[池田秀一|池田秀一]]（日本）；[[徐健春|徐健春]]／[[吳文民|吳文民]]
: 黑羽快斗的父親，為巡迴世界各地表演的世界級[[魔術師|魔術師]]。在八年前表演魔術時被人暗殺並偽裝成意外，至今生死未卜。
: 在《[[名偵探柯南|名偵探柯南]]》中也有客串出場，工藤優作多次協助警方緝捕未果，「KID」的稱號來自工藤優作。此外，莎朗·溫亞德（[[黑暗组织#Vermouth|Vermouth]]）、[[工藤有希子|工藤有希子]]都曾向他拜師學魔術跟易容術。

* {{nihongo|'''黑羽千影'''|黒羽 千影（くろば ちかげ）}}
: 聲優：[[富澤美智惠|富澤美智惠]]（日本）；[[蔣篤慧|蔣篤慧]]（台灣）
: 黑羽快斗的母親，家庭主婦。其真實身份為昭和年代女二十面相的大盜──怪盜淑女「Phantom Lady」<ref>[http://www.youtube.com/watch?v=sAonX1b7N-Q&feature=relmfu Magic Kaito Special Chapter (The Phantom Lady, Part One!) ]</ref>。

* {{nihongo|'''寺井黃之助'''|寺井 黄之助（じい こうのすけ）}}
: 聲優：[[肝付兼太|肝付兼太]]／[[矢田耕司|矢田耕司]]／{{tsl|ja|羽佐間道夫}}（日本）；[[徐健春|徐健春]]／[[吳文民|吳文民]]
: 黑羽家過去的管家以及怪盜基德的助手，也曾經假扮過怪盜基德。負責準備快斗所需的道具，而道具似乎為其朋友博士(應為[[阿笠博士|阿笠博士]])製作。

=== 江古田高中 ===
* {{nihongo|'''白馬探'''|白馬 探（はくば さぐる）}}
: 聲優：[[石田彰|石田彰]]／同左／[[宫野真守|宫野真守]]<ref name='AnimeNewsNet2014-09-01'/>（日本）；[[官志宏|官志宏]]→[[劉傑_(配音員)|劉傑]]／[[吳文民|吳文民]]
: 警視廳白馬警視總監的兒子，在英國倫敦橋高中留學<ref>{{cite book|title=魔术快斗|volume=3|url=https://www.shogakukan.co.jp/books/09122083|publisher=小学馆|isbn=4091220835|date=1994-09-17|accessdate=2018-07-09}}</ref>，后来回国轉學到黑羽快斗的學校，發現黑羽快斗的真實身份（快斗本人從不承認）。喜歡隨身攜帶一支[[懷錶|懷錶]]，對時間的掌握精確到秒，身邊有一隻名叫[[華生|華生]]的[[老鷹|老鷹]]。此外，家中有個如親人般的「婆婆」。
: 在《[[名偵探柯南|名偵探柯南]]》中也有客串出場，是其他偵探們的對手，曾經說過怪盜基德是「唯一能擾亂我的思考的人」。

* {{nihongo|'''小泉紅子'''|小泉 紅子（こいずみ あかこ）}}
: 聲優：[[林原惠|林原惠]]／[[澤城美雪|澤城美雪]]／[[喜多村英梨|喜多村英梨]]<ref name='AnimeNewsNet2014-09-01'/>（日本）；[[魏晶琦|魏晶琦]]（台灣）
: 黑羽快斗的同班同學。真實身份是繼承紅魔法的[[魔女|魔女]]，可以使用各種[[巫術|巫術]]。可以運用巫術俘虜世界上所有男人的心，卻無法俘虜怪盜基德和黑羽快斗，因此知曉二者為同一人，並迷上了黑羽快斗。

* {{nihongo|'''桃井惠子'''|桃井 恵子（ももい けいこ）}}
: 聲優：[[岩居由希子|岩居由希子]]／[[茅野愛衣|茅野愛衣]]／[[种崎敦美|种崎敦美]]（日本）；[[方雪莉|方雪莉]]／[[蔣篤慧|蔣篤慧]]
: 黑羽快斗的同班同學，與青子相當要好，總是吐槽青子與快斗之間的事。

== 原型 ==
; [[玩世流犯|玩世流犯]] 
* 青山刚昌早期的短篇漫画作品，被认为是《魔术快斗》的原型漫画，仅收录在文库版《四号三垒·青山刚昌短篇集》中。男主角為擅長偷盜、魔術及易容的'''流犯快斗'''（读音：るぱん かいと，Rupan Kaito）；女主角為刑警之女，同時也是學校偵探的'''宝睦葵子'''（读音：ほむつ あおこ，Homutsu Aoko）。
兩人是2年級B班的學生，也是青梅竹馬。

== [[名偵探柯南|名偵探柯南]]客串 ==
=== 電視系列 ===
怪盜基德第一次在[[名偵探柯南|名偵探柯南]]中登場是在單行本第16卷（動畫版76話）《柯南vs怪盜基德》中。此後魔術快斗就變成名偵探柯南故事的構成要素之一。

該作主角[[江戶川柯南|柯南]]一再涉入並加以阻止怪盜基德的行竊行動，多以平手或解決第三者犯罪為結局。基德多次易容成柯南身邊的人物（如[[毛利蘭|毛利蘭]]、[[高木涉|高木警官]]），甚至[[工藤新一|工藤新一]]本人。基德在劇場版「世紀末的魔術師」中透過竊聽知道了主角的真實身份，此事不影響到該作品原作的發展和基本設定。本作其他人物[[中森銀三|中森銀三]]與[[白馬探|白馬探]]亦因為案件而現身於該作品，但其餘角色則極少登場機會並沒有交流。有人{{who}}認為殺死初代基德黑羽盜一的神秘組織就是名偵探柯南裡面的[[黑暗組織|黑暗組織]]，目前尚未證實<ref>{{cite web |title=人物相関図 |url=http://www008.upp.so-net.ne.jp/conanmuseum/Kaito/soukan.html |publisher=魔術快斗資料館 |language=ja |accessdate=2014-09-10 }}</ref>，青山刚昌则表示两个作品的结局不太可能合并<ref>{{cite book |title=名侦探柯南SDB10+ |publisher=长春出版社 |date=2006年4月 |ISBN=9787806648988 |quote=Q10：会和快斗的结尾合并吗？ A：这个不太可能。 }}</ref>。

本作有數部作品以名偵探柯南動畫名義被改編，分別是「工藤新一vs怪盗基德」（《名侦探柯南》电视動畫第219話前半部分）與「水晶之母」（同名小學館OVA，和前文《柯南vs基德vsYAIBA寶刀爭奪大作戰！》一樣，是加入原作沒有的柯南主角群）。因此雖然名偵探柯南的原作並沒有提及怪盜的真實身分，但動畫版中公開地把黑羽快斗的存在描寫出來。

== [[YAIBA|YAIBA]]客串 ==
1989年，[[週刊少年Sunday|週刊少年Sunday]]的30週年紀念增刊，曾經發表了神偷怪盜和YAIBA的番外篇《YAIBA鐵刃寶刀爭奪戰》，描述怪盜基德到峰家偷盜寶刀，並與鐵刃對決的故事。這個番外篇收錄在魔術快斗單行本第3卷中。其後，又將這部漫畫的情節加入[[名偵探柯南|名偵探柯南]]的人物，改編成名偵探柯南OVA《柯南vs基德vsYAIBA寶刀爭奪大作戰！》，但被設定成[[江戶川柯南|柯南]]的夢。
