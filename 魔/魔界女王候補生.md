《'''魔女的考驗'''》（{{lang-ja|シュガシュガルーン}}，{{lang-en|Sugar Sugar Rune}}）是由[[日本漫畫家|日本漫畫家]][[安野夢洋子|安野夢洋子]]所畫的[[少女漫畫|少女漫畫]]作品，於《[[Nakayoshi|Nakayoshi]]》雜誌上連載，曾獲第29回[[講談社漫畫賞|講談社漫畫賞]]<ref>{{cite web|url=http://corp.kodansha.co.jp/award/archive/manga.html|title=講談社漫画賞 (過去の受賞者一覧)|publisher=[[講談社|講談社]]|accessdate=2014年3月26日|language=Ja|deadurl=yes|archiveurl=https://web.archive.org/web/20130819020612/http://corp.kodansha.co.jp/award/archive/manga.html|archivedate=2013年8月19日}}</ref>。這部作品亦改編成[[電視動畫|電視動畫]]，在日本、台灣、香港等地播放。

== 故事簡介 ==
居住在魔界的兩個少女—巧克莉及香草，不僅是好朋友，同時也是下一任魔界女王的候選人。她們以競爭對手的身分到了人間，比賽誰能夠收集最多人類的愛心，也就是將人們對她們的各種心情，化為心的結晶。蒐集最高價值結晶的人，將成為下一任魔界女王。

原本在魔界大受歡迎的巧克莉，以及在魔界毫不起眼的香草，到了人類世界之後，受歡迎程度卻完全逆轉，使巧克莉陷入了困境。同時巧克莉對來歷不明的少年皮耶爾一見鍾情，而她魔女的心被他吸引。據說魔女的心，每個魔女都只擁有一顆。一旦被奪走，就會永遠失去靈魂，陷入無止盡的黑暗。巧克莉是否能夠守護她的心？也是否能通過艱巨的女王測驗，完成自己從小的夢想？同時，魔界反派種族—歐葛爾打算佔領整個魔界，並向兩位女王候選人下手，巧克莉及香草能否對抗歐葛爾？巧克莉能否和皮耶爾在一起？魔界女王將由誰來擔任呢？

== 用語 ==
=== 心 ===
魔界的魔法師只有一顆心，當失去心時將會死去，而人類的心則會不斷再生，當人類的心產生感情時，巧克莉與香草就會趁此收集，以獲得的心累積點數多寡決定女王繼承權。「心」的顏色不同，點數也不同。「心」可以彼此交換，而「心」的點數可以用來在魔界購物，就像人類世界的[[貨幣|貨幣]]一樣功能，而「心」的結晶會被送到「心工場」轉化為魔界的能量。

下-{面}-介紹每一種「心」的定義與其價值。

;水晶之心
: 於動畫版最後一集時，獨角獸把牠的角給巧克莉時的心。是通過[[獨角獸|獨角獸]]考驗才能獲得的心，極為珍貴。價值50,000亞古努。（動畫才有的心）
:獨角獸只會出現在擁有純潔心靈的人面前，他的考驗包括:「看清真相的眼」、「面對困難仍然勇敢向前的翼」、「真心全意去愛的心」。如果具備3個條件，就會觸動獨角獸的心，自然可以得到獨角獸的角(即牠的心)。
;白色的心
: 又稱「菲爾特雷」（「菲蓮度里」港譯），這是把諾亞爾之心淨化後才有的，價值35,000亞古努。
;黑色的心
: 代表「絕望、嫉妒、憎恨、貪婪」，除了法力高強的菲爾特雷，其餘魔女不能收取此類心，不然此人會逐漸失去法力而慢慢死亡，又稱作「諾亞爾」（動畫），價值30,000亞古努。因為歐葛爾一族受詛咒影響下，不能拿一般的心，所以只好去拿此禁物作為糧食，萌黃學園中學部的百合香她們幾個經常出現這種心。
;紅色的心
: 代表「熱情的愛」，價值5,000亞古努，這是最珍貴的心，當魔界的人結婚時會把此心互相交換，代表將自己託付給唯一心愛的人，付起保護對方生命的責任（如果心變成紅色，那你就一定會雙手奉上自己的心給你所愛的人），萬一被對方背叛就會死，永遠徘徊於黑暗中。
;藍色的心
:代表「尊敬」，價值3,000亞古努。巧克莉和香草都沒有拿過，在瓦芙爾使用女王王冠時才有出現（動畫才有的心）。
;紫色的心
: 代表「慾望與情慾」，價值2,500亞古努，這是一種僅次於紅色心的愛心，但不容易見到。（漫畫）
: 而在動畫中也曾出現過（第一話），是巧克莉的班主任和兩三個老師所生成的。
;粉紅色的心
: 代表「甜蜜的愛」，價值1,000亞古努。這一種心是因喜歡或有點好感出現的。
;彩虹之心
: 是開心的時候，只有同性才有的心，價值500亞古努。這是一種很稀有的心，不能收集（動畫才有的心）。
;綠色的心
: 代表「友情」，價值400亞古努。綠色的心與橙色的心相較之下，還比較有價值。
;橙色的心
: 代表「有些好感」，價值300亞古努。故事中，香草時常拿到這一種心，而這種心也是最常見的一種。
;灰色的心
: 代表「討厭」，價值250亞古努。
;黃色的心
: 代表「驚嚇」，價值5亞古努。這種心被羅賓戲稱為「尿素色」（piss），是巧克莉第一次拿到的心。漫畫中安布爾稱之為「PiPi」。
