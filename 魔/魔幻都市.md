《'''魔幻都市'''》（{{lang-en|'''''Magic City'''''}}）是一部[[美国|美国]]电视剧，于2012年3月30日起在美国[[Starz_(電視頻道)|Starz]]播出。本片由{{link-en|米奇·格雷澤|Mitch Glazer}}创作。首集在2012年3月30日試播，2012年4月6日首播。Starz在2012年3月20日續訂共8集的第二季<ref>{{Cite web |url = https://www.hollywoodreporter.com/news/starz-magic-city-renewal-jeffrey-dean-morgan-301921 |title = Starz Renews 'Magic City' Ahead of Period Drama's April Premiere |author = Lacey Rose |date = 2012-03-20 |publisher = The Hollywood Reporter |language = en}}</ref>。2013年8月5日，在第二季後取消本劇<ref>{{Cite web |url = https://deadline.com/2013/08/starz-magic-city-canceled-season-2-finale-557354/ |title = Starz's ‘Magic City’ To End Run After Two Seasons; Series Finale Airs Friday |author = Nellie Andreeva |date = 2013-08-05 |publisher = Deadline Hollywood |language = en}}</ref>。

== 简介 ==
本劇設定在1959年[[佛羅里達州|佛羅里達州]]的[[迈阿密|迈阿密]]，讲述主人公艾克·埃文斯（Ike Evans）在当地开设一家新的豪华饭店，而艾克和当地的黑帮组织有着见不得人的秘密。
