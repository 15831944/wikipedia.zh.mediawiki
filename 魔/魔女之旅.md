{{日本作品|魔女之旅|魔女の旅々}}是一部由白石定規創作的小說，插畫由{{lang|ja|あずーる}}負責。小說由{{tsl|ja|SBクリエイティブ|SB Creative}}出版。

2014年本作在Amazon Kindle首次公開，及後在2016年修正後單行本化。

== 故事簡介 ==
魔女伊蕾娜在滿師後便開始周遊列國。

身為旅人的她，在漫長的旅途中，總會與形形色色的人們邂逅，接觸他們的日常生活，又或是被捲入某些事件。

魔女日復一日的旅程，編織出相逢與離別的故事。

== 登場人物 ==
; {{nihongo|伊蕾娜|イレイナ|聲：[[本渡楓|本渡楓]]}}
: 本作主角，18歲的天才少女。
: 在15歲就成為了魔法使的最高位「魔女」，別名為「灰之魔女」。其後展開周遊列國的旅程。

; {{nihongo|芙蘭|フラン|聲：[[花澤香菜|花澤香菜]]}}
: 伊蕾娜的老師。別名為「星塵魔女」。
: 在伊蕾娜滿師後前往某個國家擔任教師。

; {{nihongo|沙耶|サヤ|聲：[[黑澤朋世|黑澤朋世]]}}
: 魔女見習生，於魔法師之國的旅館內工作。第二卷後成為隸屬魔法統合協會的「炭之魔女」。
: 出身自東國，為了成為魔女見習生而與妹妹一同前往魔法師之國，其後妹妹通過試驗而先行回鄉，而沙耶則留在魔法師之國準備下次的試驗。期間遇到旅行中的伊蕾娜，並向她請教。最終憑藉自己的努力不懈，成功當上魔女見習生。

; {{nihongo|艾姆妮西亞|アムネシア|聲：[[小原好美|小原好美]]}}
: 睡著便會喪失記憶的少女，正在旅行前往信仰之都「伊斯特」。

; {{nihongo|艾維莉亞|アヴィリア|聲：[[岡咲美保|岡咲美保]]}}
: 維護信仰之都「伊斯特」的治安，正統騎士團的幹部。
