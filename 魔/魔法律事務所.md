《'''魔法律事務所'''》（{{lang-ja|ムヒョとロージーの魔法律相談事務所}}）是[[日本漫畫家|日本漫畫家]][[西義之|西義之]]所繪的[[日本漫畫|日本漫畫]]作品。在[[週刊少年Jump|週刊少年Jump]]從2004年53號至2008年14號期間連載的黑暗幻想漫畫作品。2008年6月4日已發行最終單行本第18集。2018年3月在[[少年Jump+|少年Jump+]]上開始本作續篇「魔法律顧問事務所 魔屬魔具師篇」的新連載。

電視動畫已於2018年8月初開始播放中，於日本的[[SKY_PerfecTV!|SKY PerfecTV!]]、[[Animax|Animax]]頻道首播<ref>{{Cite news|publisher=株式會社Natasha|work=[[Natalie_(網站)|Comic Natalie]]|url=https://natalie.mu/comic/news/274068|title=「ムヒョとロージーの魔法律相談事務所」TVアニメ化！続編の連載もスタート|date=2018-03-19|accessdate=2018-03-19|language=ja}}</ref>。

==情節==

==登場人物==
===六冰魔法律事務所===
'''六冰 / 六冰透'''（{{lang|ja|ムヒョ / 六氷透}}，聲：[[村瀨步|村瀨步]]（日本）；[[鍾少庭|鍾少庭]]（台灣））
: 位居魔法律家最高等級「執行人」的年輕天才精英。平常沉著冷靜，雖然多半表現得冷漠無情，不過其實正義感強烈，也有善良的一面。使用魔法律之後體力會嚴重耗損，因此經常在睡覺。對身陷在禁魔法律中的好友圓宙其實很在乎。面對能力不足的助手郎次，雖然表面上很嚴厲，但其實非常信任郎次並能感覺郎次身上有未發揮出來的潛力。
:身高128cm，1月23日出生。喜歡的東西：[[週刊少年Jump|Jump]]，不喜歡的事物：妨礙睡眠的事物（特別是郎次）。

'''郎次 / 草野次郎'''（{{lang|ja|ロージー / 草野次郎}}，聲：[[林勇|林勇]]（日本）；[[鈕凱暘|鈕凱暘]]（台灣））
: 六冰的助手。從等級最低的「二級書記官」稍稍晉升，目前是臨時一級。個性軟弱善良，因此在面對惡靈時總是會發抖。為了要能夠獨當一面，每天都努力地在修煉魔法律。喜歡的東西是紅茶和點心。討厭的東西是幽靈和恐怖的六冰。有時六冰對待自己的感覺，常讓他感到疑惑，直到後期才明瞭自己當初是多麼無知，更下定決心要保護六冰和大家。過於善良及樂於幫助別人的個性常讓他惹上麻煩。
:身長170cm，9月29日出生。特技：超級[[紙糊|紙糊]]技巧、料理。喜歡的東西：[[紅茶|紅茶]]、點心。
:不喜歡的事物：幽靈、可怕的六冰。

===魔法律家===
'''洋一 / 火向洋一'''（{{lang|ja|ヨイチ / 火向洋一}}），聲:[[柿原徹也|柿原徹也]]（日本）；[[何志威|何志威]]（台灣））
:六冰的老同學，法官。精通除了執行以外的所有魔法律的專家，生性好色。在魔法界非常知名，一直希望能夠成為六冰的助手。
:身高173cm，8月5日出生。喜歡的東西：三明治。

'''孫子 / 我孫子優'''（{{lang|ja|ビコ / 我孫子優}}，聲: [[河村理惠|河村理惠]]（日本）；（台灣））
:六冰的老同學。魔具師，擅長製作符咒、筆、魔法律書等等的魔具，不會使用魔法律。除製作魔具以外的專長是烘培。
:身高126cm，2月28日出生。

'''今井玲子'''（{{lang|ja|今井玲子}}，聲: [[齋賀光希|齋賀光希]]（日本）；（台灣））
:洋一的前輩。法官，在魔監獄裡和六冰他們共同與「剝臉皮的蘇菲」作戰的勇敢女法官，除擅長魔法律外也是劍道五段的高手，為少數精通體術的法官。
:身高164cm，2月10日出生。

'''班治．克勞斯'''（{{lang|ja|ペイジ・クラウス}}，聲: [[羽多野涉|羽多野涉]]（日本）；（台灣））
:執行人。年老的男性。魔法律協會調査本部長中[[#専門用語|四賢人]]的一人。洋一的上司、六冰與圓宙的老師。
:曾對六冰為何選擇郎次而感到疑惑。
:直到在和郎次的對談中明白了，六冰原來要尋找的就是（純真的善心），郎次剛好就是不二人選。
:為了拯救落入禁魔法律界的愛徒，奮力對抗禁魔法界，並且相信圓宙還是保有善良的心。
:自稱詩人。
:身高190cm，5月25日出生。

'''五嶺 / 五嶺陀羅尼丸'''（{{lang|ja|ゴリョー / 五嶺陀羅尼丸}}）
:與六冰敵對的執行人。敗給禁魔法律家，被六冰他們所救。是龐大的「五嶺集團」負責人。
:在遇上六冰跟郎次前並不理解夥伴的真諦，對於助手法官非常冷酷，為達目的不擇手段。被拯救後理解與夥伴並肩作戰的真諦，並幫助六冰擊敗禁魔法家。
:身高178cm，12月20日出生。喜歡的東西：[[羊羹|羊羹]]。

'''惠比壽 / 惠比壽花夫'''（{{lang|ja|エビス / 恵比寿花夫}}）
:五嶺手下的法官。在幽靈社區中與六冰對決時犯錯而被開除，不過現在再次重回五嶺手下。
:因被五嶺所救，所以對於五嶺非常忠心，在五嶺被禁魔法律家擄走時奮不顧身的去拯救他。
:身高130cm，12月21日出生。

'''左近'''（{{lang|ja|左近}}）
:法官助理。

'''莉莉・馬西亞斯 & 馬利爾・馬西亞斯'''（{{lang|ja|マリル・マシアス & リリー・マシアス}}）
:二級書記官的雙胞胎兄妹。以研究魔法律聞名的魔法律博士。
:和郎次一起參加魔法律特別強化集訓。發現六冰及郎次有一特別的連結，兩人的命運互相牽引著對方。
:身高 兄143cm、妹142cm，10月13日出生。特長：[[妄想|妄想]]、[[分析|分析]]、[[研究|研究]]。

'''笹之葉梅吉'''（{{lang|ja|笹ノ葉梅吉}}）
:在毒島魔法律事務所擔任助手的一級書記官。非常崇拜六冰。
:實際上不是人，本體是毒島召喚的使者，正式名稱是「雲龍鼠」。有煉是因為毒島給予一半的靈魂。憧憬人類。
:身高123cm

'''毒島春美'''（{{lang|ja|毒島春美}}）
:女性執行人，與今井玲子同期。被助手的梅吉稱為「老大」。
:魔法界夢幻遠距離魔法的執行人，但因事務所生意不好因此也常常打零工。
:身高165cm，7月25日出生。喜歡的東西：[[酒|酒]]、[[香煙|香煙]]、[[開車|開車]]。

'''素柿家銀二''' ({{lang|ja|素柿家銀二}})
:魔法律學校高等部五年級生，委員長。法官助理。對圓宙的想法很不滿。

{{lang|ja|'''大河内 奏成'''（おおこうち そうせい）}}
:老色鬼，當初對理緒百般刁難。而後離奇死亡，因為被理緒報復。

{{lang|ja|'''莉莉‧艾蓮娜''' （リリー・エレナ）}}
:女性執行人，老婦人。理緒前來求救時，不理睬。而後離奇死亡，因為被理緒報復。

===一般人===
;竹乃內菜菜（聲：[[野水伊織|野水伊織]]（日本）；[[林美秀_(配音員)|林美秀]]（台灣））
:建二的鄰居，想成為攝影師，因委託六冰執行魔法律而認識。和六冰還算合得來，因某次好奇偷偷跟著六冰及郎次闖入魔法總部，喜歡郎次的紅茶，現在為班治‧克勞斯底下的靈力攝影工讀生，參與之後對抗禁魔法律界的旅程。
;佐籐建二（聲：[[天崎滉平|天崎滉平]]（日本）；（台灣））
:因為「這裡是騙人的地方」的這個理由，破壞六冰事務所招牌的惡作劇少年。自從在遭到惡靈攻擊時被六冰他們所救之後，就經常找理由到事務所來玩。

=== 禁魔法律家 ===
;圓宙 / 圓宙繼（聲：[[神谷浩史|神谷浩史]]（日本）；[[劉傑_(臺灣)|劉傑]]（台灣））
:六冰、洋一、孫子的魔法律學校同學。為了讓相依為命的媽媽過上好生活，在學校拼命學習，但卻總是追在天才六冰身後，因嫉妒種下後來踏入禁魔法律界的原因，原本前途光明的他，因在執行人考試時母親病危無法成為執行人而憎恨六冰，步上叛徒之路，在六冰及大家的幫助下明白母親的死是提基的陷阱，而改邪歸正
:現在被關進魔監獄中服刑。
;提基
:八百年來，持續遭魔法律協會鎖定的叛徒。利用不少人幹了很多壞事的惡徒，認為六冰他們是棘手的人物。
;理緒 / 黑鳥理緒
:孫子的師父，教主級魔具師。因為協會對她的母親見死不救而在絕望下成為叛徒。後來才明白一切都是提基的佈局，
:原本認為沒有臉回去大家身邊，後來在孫子的話語中，決定回去和大夥一起對付提基。
;三色菫
:暗戀郎次的女孩，被提基變成禁魔法律家。後來在六冰的幫助下淨佛了。
;湯瑪斯
:五嶺在MLS時代的導師，有著怪異的審美觀。成為方舟的人員時，把五嶺變成[[半靈|半靈]]，利用五嶺和六冰一夥人戰鬥。
:但是，在六冰和郎次的努力下，將其打敗，把五嶺救回。
;米克 / 麥可．高爾多羅
:善於使劍，被六冰和毒島合力打敗後借著普布的咒具再度復活，但又被洋一打敗。
;艾碧．高爾多羅
:米克的姐姐，靈使者，先後召喚強力的惡靈對付六冰等人，似乎跟五嶺有種關係，之後被木偶普布所殺。
;基特．高爾多羅
:米克和艾碧的弟弟，是普布的木偶，最後被洋一、今井和銀二打敗。
;木偶普布
:善用多種咒具及木偶來攻擊，本身的攻擊力也很驚人。是一位連同伴都可以殺害、或伸出魔掌的恐怖之人。
;富利歐尼爾
:研究禁魔法律的第一人，因被提基逼迫，與赤魔女訂定契約，後因提基被打敗，而消失。
