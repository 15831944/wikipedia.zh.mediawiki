{{vgname|魔界戰記|ja=魔界戦記ディスガイア|en=Disgaea: Hour of Darkness|diff=yes|aka=魔界戰記 Disgaea、魔界戰記迪斯凱亞}}是由[[日本一軟體|日本一軟體]]於2003年1月30日在[[PlayStation_2|PlayStation 2]]上推出的[[戰略角色扮演遊戲|戰略角色扮演遊戲]]，《{{link-ja|魔界戰記系列|魔界戦記ディスガイアシリーズ}}》的首作。衍生出漫畫、小說、動畫等多媒體作品，並在多種遊戲機平台上發行重製版。續作為《[[魔界戰記2|魔界戰記2]]》。

== 概要 ==

以「史上最兇{{lang|ja|やり込み}}<ref group="注">指与完成游戏主线无关，通过游戏中的某些反复操作让玩家产生成就感的图鉴收集或是角色培育等要素</ref>SRPG」為賣點，擁有遠多於一般[[SRPG|SRPG]]的「{{lang|ja|やり込み}}」要素。一般遊戲只到100左右的等級上限在本作飆高到9999，能培育出[[HP|HP]]超過4千萬，能力值到達2千萬的角色。

遊戲設計上包含道具和技能的收集、高自由度的道具強化、提高能力值成長率後再降回最低等級重新培育的轉生系統…等，號稱要完成全部的壓力摳米要素最少需要300小時以上的遊戲時間。甚至隱藏關卡裡出現的怪物等級在1000以上，每個道具內都有被稱為「道具界」的隨機迷宮，名副其實的「史上最兇{{lang|ja|やり込み}}SRPG」。

受到比起重視故事性的《{{link-ja|人形公主|マール王国の人形姫}}》系列，遊戲性較強的《{{link-ja|光之聖女傳說諸神黃昏|ラ・ピュセル 光の聖女伝説}}》受到較高的評價的影響，日本一軟體以遊戲性為重點進行本作的開發<ref>{{cite web|author=照井一功|date=2007年12月10日|url=http://trendy.nikkeibp.co.jp/article/column/20071207/1005122/?ST=ent|title=史上最凶のやりこみシミュレーションRPG『魔界戦記ディスガイア』シリーズの人気の秘密に迫る Part.1|work=日経トレンディネット|publisher=日経BP社|accessdate=2008-01-18}}{{ja}}</ref>，首次創下了賣出超過10萬套的紀錄<ref>{{cite web|date=2005年6月29日|url=http://www.famitsu.com/sp/eternalkingdom/news/050629.html|title=UMD『トリプルオモチャバコ』，介紹雙方公司對收錄作品的發言！ |work=ガスト・日本一ソフトウェア合同企画|publisher=[[ファミ通|ファミ通.com]]|accessdate=2008-01-18}}{{ja}}</ref>。

2004年11月3日，廉價板『魔界戰記迪斯凱亞 PlayStation 2 the Best』發售。2006年11月30日，追加艾特娜編和新的EX關卡的[[PSP|PSP]]版『魔界戰記迪斯凱亞 PORTABLE』發售。2007年11月29日，再追加連線對戰機能的PSP廉價版『魔界戰記迪斯凱亞 PORTABLE 開始連線對戰了。』（{{lang-ja|魔界戦記ディスガイア PORTABLE 通信対戦はじめました。}}）發售。2008年6月26日，[[NDS|NDS]]版『魔界戰記迪斯凱亞 魔界的王子與紅月』（{{lang-ja|魔界戦記ディスガイア　魔界の王子と赤い月}}）發售。2018年7月26日在[[任天堂Switch|任天堂Switch]]和[[PlayStation_4|PlayStation 4]]平台发行高清重制版《魔界战记Disgaea Refine》。<ref>{{Cite web|url=https://www.famitsu.com/news/201805/02156236.html|title=『魔界戦記ディスガイア Refine』本タイトルならではの要素や戦闘システム、汎用キャラクターなどを公開 - ファミ通.com|accessdate=2018-07-08|work=ファミ通.com|language=ja}}</ref>

本作於2003年被改編成漫畫與輕小說，2006年被改編成動畫。

== 故事 ==

魔王克-{里}-切夫斯寇伊去世後，魔王支配下的魔物們開始爭奪魔王寶座，魔界處於群雄分割的亂世中。

兩年之後，魔王的兒子拉哈爾從長眠中醒來，展開了奪回王位的故事。

== 登場人物 ==

=== 魔界 ===

; {{nihongo|拉哈爾|ラハール（Laharl） || 聲優：[[水橋香織|水橋香織]]}}
: 本作品的主角。魔王克-{里}-切夫斯寇伊與身為魔女見習生而來到魔界學習的人類母親之間所生的半惡魔。在父親死後，以繼承魔王為目標。年齡為1313歲，是主要角色的三人中最年輕的。在遊戲剛開始時的個性十分的自我中心、任性妄為、冷血，擁有與魔王兒子的身份十分相配的個性。非常討厭愛、溫柔、光明與積極之類的話語、以及大胸脯的女性。之後，遊戲中提到拉哈爾如此的否定愛的原因是出至於母親的死。由於與芙蓉的相遇，而開始展現出身為王者的度量及和善。擁有著非常強大的力量，能以一人之力將侵入魔界、超過200萬人的宇宙大艦隊殲滅（而且是在手下留情的狀態下），最後把魔王的權利交給艾特娜，跟母親一樣犧牲了自己把芙蓉救活也變成普林尼的一員。本人一直希望艾特娜跟芙蓉能一直待在身邊(沒有實際說出口)。三人在該公司的相關作品中一直有串場出現。

; {{nihongo|艾特娜|エトナ（Etona、另外PSP版的片尾動畫顯示為Etna） || 聲優：[[半場友惠|半場友惠]]}}
: 拉哈爾的家臣。與芙蓉相同，都是屬於本作品的女主角。但是和芙蓉相較之下，艾特娜比較像是吉祥物之類的角色。年齡為1470歲。非常有野心。比任何人都愛好自由，絕對不饒恕妨礙她自己的人。如果提及胸圍之類的話題，艾特娜會被激怒。至今仍然對逝世的前任魔王克-{里}-切夫斯寇伊展現出絕對的忠誠心。現在時時為了鍛鍊拉哈爾更有切夫斯寇伊的樣子努力中,在隱藏的房間內可以查閱到描述艾特娜的過去及心境的日記。有時會跟拉哈爾因為小事而吵架,2代還因此離家出走。是個騷味滿溢的貧乳小惡魔。

; {{nihongo|普林尼隊|プリニー隊（Prinny） || 聲優：[[間島淳司|間島淳司]]等人}}
: 被艾多娜以金錢雇來的隨從。經常被艾特娜和拉哈爾使喚著。外表穿著像是弄壞的企鵝般的服裝，裡面則是裝著罪人的靈魂。在人間犯下的罪必須經由工作來償還，魔王城的普林尼工作時間為一天20個小時左右才能得以升天。在語尾上會加上「～斯（{{lang|ja|〜ッス}}）」。佩帶在腰上的袋子裡裝有武器，動畫版的武器為魚。在魔界的其他地方其實也有普林尼的存在，受人雇用或者是受不了而跑走集成一個部落。

雖然大部分普林尼都很弱但身前實力很強大的變成普林尼後,也仍然會很厲害。

; {{nihongo|大姐姐普林尼|姉御肌のプリニー（Big Sis Prinny） || 聲優：[[齋藤千和|齋藤千和]]}}
: 有著與普通普林尼不同的紅色外表，在語尾也不會加上「～斯（{{lang|ja|〜ッス}}）」。和普通普林尼的個性不同，還曾經給芙蓉藥物。其實是拉哈爾的媽媽，因為犯了自殺罪而變成普林尼。

; {{nihongo|艾拉米斯|アラミス}}
: 艾特娜為了拉哈爾的工作而帶來的少年。

; {{nihongo|安德華爾|アントワール || 聲優：[[水橋香織|水橋香織]]}}
: 只在艾特娜篇登場的夜魔族的貴族惡魔。立場上似乎是瑪德拉斯的部下。

; {{nihongo|中魔王|中ボス（Mid Boss） || 聲優：[[鈴木千尋|鈴木千尋]]}}
: 最近急劇的擴展了勢力。自稱「美麗男爵拜亞斯」，卻被拉哈爾們稱為中魔王。之後，在遊戲中段到結局中間，因為常被拉哈爾們稱為中魔王，而忘了自己的名字「拜亞斯」，同時也不再以「美麗男爵」來稱呼自己。視拉哈爾為勁敵，但有時也會因不明原因而幫助拉哈爾。其實和大姐姐普林尼一樣，是與拉哈爾有著很深厚的關係之人物。

; {{nihongo|杰尼斯基|ゼニスキー || 聲優：[[西村知道|西村知道]]}}
: 愛好金錢的惡魔。原本是魔王克-{里}-切夫斯寇伊的家臣，但是在克-{里}-切夫斯寇伊死後將魔王城內的寶物據為己有。拉哈爾為了要支付普林尼隊的薪資，而曾經強襲杰尼斯基所在的成金城。有個為父親著想的兒子。

; {{nihongo|瑪德拉斯|マデラス（Maderas） || 聲優：[[宮田浩德|宮田浩德]]}}
: 吸血鬼種族的惡魔。原本侍奉著魔王克-{里}-切夫斯寇伊，卻因為偷吃了克-{里}-切夫斯寇伊愛吃的黑暗饅頭，而被放逐。

; {{nihongo|克-{里}-切夫斯寇伊|クリチェフスコイ（Krichevskoy） || 聲優：[[西村知道|西村知道]]}}
: 已經去世的前任魔王、拉哈爾的父親。持有被稱為大魔王的力量。據說在拉哈爾出生時，因為過於高興而將湖凍結了。廣為流傳的死因為「被黑暗饅頭給噎死了」，實際上為了封印超魔王巴爾導致魔力耗盡。因為是逝者，所以在作品中並只出現在回想裡，並無登場。十分受到艾特娜及魔王城的部下的敬愛。非常紳士與充滿王的威嚴。

; {{nihongo|普莉葉|聲優：[[中原麻衣|中原麻衣]]}}

=== 天界 ===

; {{nihongo|芙蓉|フロン（Flonne） || 聲優：[[笹本優子|笹本優子]]}}
: 本作品的女主角。奉大天使拉明頓的命令，而前來魔界暗殺魔王的天使見習生。年齡為1509歲，是主要角色的三人之中年紀最大的一位。是個時常將愛掛在口中的愛之狂熱者。有著溫和的個性、[[天然呆|天然呆]]、常會發出讓周遭的人們不解的發言。其實很喜歡戰隊、變身英雄、及[[超級機器人|超級機器人]]等物。主要為使用魔法。後來變成墮天使，待在拉哈爾的身邊，個性變得有點腹黑。

; {{nihongo|拉明頓|ラミントン（Lamington） || 聲優：[[間島淳司|間島淳司]]}}
: 身為天界指導者的大天使。喜愛著花與詩。曾經說過連天界的敵人－惡魔也擁有愛，是個充滿愛與溫柔的天使。似乎在與神秘人物策劃些什麼…。

; {{nihongo|布魯卡諾|ブルカノ（Vulcanus） || 聲優：[[宮田浩德|宮田浩德]]}}
: 拉明頓的直屬部下、天使長。有著一付如壞人般冷酷的面孔。雖然身為天使，卻擁有為了達到目的而不擇手段的個性。

=== 人間 ===

; {{nihongo|高登|ゴードン（Gordon） || 聲優：[[飛田展男|飛田展男]]}}
: 第37代的地球勇者。個性爽朗。雖然正義感很強，不過力量比珍妮佛還弱。

; {{nihongo|珍妮佛|ジェニファー（Jennifer） || 聲優：[[齋藤千和|齋藤千和]]}}
: 高登的助手。在十歲時便取得博士稱號的天才科學家，同時也是星期四的製作者。有著一具極好的身材，穿著著類似於比基尼的服裝。

; {{nihongo|星期四|サーズデイ（Thursday） || 聲優：[[落合祐里香|落合祐里香]]}}
: 珍妮佛在五歲時製作的萬能型機器人。可以變形為巨大的發射器。身上搭載著做為最終手段的自爆裝置。

; {{nihongo|卡奇斯|カーチス（Kurtis） || 聲優：[[鳥海浩輔|鳥海浩輔]]}}
: 另一位地球勇者。本來地球勇者只有一人，但是卡奇斯擅自自稱為地球勇者。卡奇斯常用卑鄙的手段奪取高登他們的工作，不過因為確實擁有的實力，在某種程度上好像被周圍的人們認同。
