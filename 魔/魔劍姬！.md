《'''魔劍姬！'''》（{{lang-ja|マケン姫っ！}}），是[[日本|日本]][[漫画家|漫画家]][[武田弘光|武田弘光]]的作品，並改編成動畫。

== 概要 ==
*2007年6月在《[[Dragon_Age_Pure|Dragon Age Pure]]》連載，2009年5月因《Dragon Age Pure》已於同年2月廢刊而轉至《[[月刊Dragon_Age|月刊Dragon Age]]》。
*2010年12月1日宣布動畫化企劃進行中<ref>{{cite web|url=http://www.animenewsnetwork.com/news/2010-12-01/maken-ki-manga-has-anime-project-in-the-works|title=Maken-Ki! Manga Has Anime Project in the Works|publisher =[[動畫新聞網|動畫新聞網]]|date=2010年12月1日報導|accessdate=2013年11月4日|language=en}}</ref>；2011年6月7日確定[[電視動畫|電視動畫]]。2012年9月7日再度宣布由武田弘光完全監修的新動畫化消息；2013年5月8日新動畫官網 {{nihongo|《魔劍姬！通》|マケン姫っ！{{ruby|通|つう}}}}<ref>日文{{nihongo|「通」|つう}}音同英文「two」。</ref>開設。
*以主角為中心被女性包圍的校園格鬥青春劇，話数使用[[中文數字|漢字大寫]]壹、貳、-{叁}-、肆等。

== 故事簡介 ==
原[[女校|女校]]改[[男女同校|男女同校]]的「天日学園」，'''大山武'''與[[青梅竹馬|青梅竹馬]]'''天谷春恋'''入学就讀。這學校是依「MAKEN」適性檢查結果的「元素特性」進行「MAKEN」的鍛鍊，然而武卻面臨一場場的戰鬥及[[殺必死|殺必死]]畫面……。

== 用語 ==
; {{nihongo|天日学園|天日学園}}
:以國家立法，擁有自治權的特別區域，獨立特區「天日」，來自全國少男少女就讀的學校，施以使用MAKEN、操控元素的訓練。
:學園宗旨是「心、美、體」，允許戀愛但用MAKEN決鬥，需雙方同意、有見證人才行。
:原本針對只要加入神狩或被修改記憶的能力者，創校學園長不顧神狩的反對，多出留在天日的選擇。
:創校初期三年曾是男女合校，因發生男教師與男生對女生施暴事件而改為女校。當事人都被操作記憶過。
:{{nihongo|靈峰天之原|霊峰アマノハラ}}為其管轄私有地，傳說荒神[[八歧大蛇|八歧大蛇]]被封印在[[天岩戶|天之岩戶]]。

; {{nihongo|魔導檢警機構|魔導検警機構}}
: 通稱{{nihongo|「魔檢機」|マケンキ}}，解決學園內麻煩的社團組織，由學生自治組織「統生會」以及輔佐組織「檢警社」、「魔導執行社」組成。 

;{{nihongo|MAKEN|マケン}}
:以人稱「元素」操縱、傳說日本眾神的道具，有八個MAKEN的「原型」。學園經「元素特性」判斷後發給學生MAKEN的仿造品，訓練如何使用。如不在20歲前訓練起來，隨著年齡增長會變得不好記且無法控制力量。
:概分外藏型與內藏型兩種，內藏型相當注重相性，雖少數出生體內即有MAKEN之人，如相性不好則無法使用，所以這種人稀少；不過，武的情形要對方處於發動狀態才能使用。
:「MAKEN」型態套用日文同音漢字，如魔劍、魔檢、魔研、魔拳、魔權、魔建、魔腱、魔鍵、魔圈等等，但採外來語意譯[[訓讀|訓讀]]，例如一樣是「魔劍」就有「maken」、「sword」跟「blade」三種不同唸法。
:目前八MAKEN中，出現的有魔劍「叢雲」、魔拳「神度」、魔鍵「八尺瓊」、魔劍 「羽羽斬」、魔見 「羽羽矢」、魔建 「死返」。

;{{nihongo|元素|エレメント}}
:魔力、靈力等各式各樣能力的總稱，依{{nihongo|「元素特性」|<ruby><rb>素因特性</rb><rp>（</rp><rt>エレメントとくせい</rt><rp>）</rp></ruby>}}給予適合MAKEN的訓練。

; {{nihongo|神狩|カミガリ}}
: 遠古眾神統治時代，由大和武尊創立、不想受其支配而創立對抗神族的戰鬥集團，至今為暗中使用MAKEN和元素力量支持國家的組織。
: 過去強拉年輕力量加入，修改或消除反抗者的記憶，結果大多失去能力，於是將人才集中與培養的工作交給天日學園。

;VENUS
:神狩以留學生名義派到學園的傭兵部隊，表面上調查要讓八歧大蛇復活的不明份子，實執行綁架可解除其封印的木靈。

; 神具
: VENUS所使用傳說外國諸神的道具，足可匹敵原型MAKEN的寶物。

;夜政
:也被稱作地下政府，委託能力者集團進行地下工作的機密組織。
