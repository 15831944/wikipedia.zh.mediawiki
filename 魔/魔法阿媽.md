《'''魔法阿媽'''》<ref group="註">片頭將「魔」的注音符號刻意標示為「ㄇㄨㄛˊ」，而非原有的「ㄇㄛˊ」。</ref>是由[[王小棣|王小棣]]所執導/監製，漫畫家[[麥仁杰|麥仁-{杰}-]]負責劇中造型設計與分鏡，[[史擷詠|史擷詠]]擔任配樂，[[文英|文英]]替動畫人物配音的1998年所上映[[台灣動畫|台灣動畫]][[電影|電影]]。<ref>{{cite web|url=http://toolkit.culture.tw/filminfo_137_146.html?themeId=1 |title=魔法-{阿}-媽 |work=[[中華民國文化部|中華民國文化部]] |language=zh-tw |accessdate=2017年10月8日}}</ref>

作品預算高達4000萬，且仿效迪士尼製作方式以真人表演來進行配音後，再繪製動畫畫面。本片內容以祖孫互動與[[台灣民間信仰|台灣民間信仰]]為主題，上映後在台灣社會取得迴響。<ref>{{Cite web|url=http://www.storm.mg/lifestyle/159054 |title=近20年的經典！被金馬獎背叛的《魔法-{阿}-媽》，演活了台灣最美的5種人情味 |work=[[風傳媒|風傳媒]] |author=陳伯璿 |language=zh-tw |date=2016年8月26日 |accessdate=2017年10月8日}}</ref>

== 故事大綱 ==
有一天，小男孩豆豆因為媽媽必須出國照顧在國外工作受傷的爸爸緣故，被留在位於[[基隆|基隆]]鄉下的阿媽家居住。豆豆一開始沒有辦法跟阿媽好好相處，並且阿媽都做著一些他無法理解的迷信行爲。

之後豆豆在好奇心之下，亂闖入阿媽曾告戒過不可進去，一間專門儲放被封印惡靈的儲藏室後，意外放出了一條專做壞事的惡靈。為了不想再被阿媽抓住好繼續留在世間裡作惡，惡靈伺機想利用豆豆，告訴了他可以把「阿媽賣掉」回到原來家裡的方法。

在惡靈的聳恿之下，豆豆一開始想照著對方的主意來進行，但在經歷到各種的奇妙事物後，開始對阿媽之間的情感產生了改變。

== 製作 ==
=== 過程 ===
{{quote box|quoted=true|width=20%|salign=center|quote=初聽時，我腦子裏立刻-{流佈}-了蛋黃、奶泡、薄荷綠……一些帶有冰淇淋味道的色彩。 |source= ── 導演[[王小棣|王小棣]]在初次聽見[[黃黎明|黃黎明]]構想的此部影片劇本時，心中所浮現的印象。<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈沉到人跡罕至的生命深處〉|page=103頁}}</ref>}}
於1996年夏季某日，此部影片編劇[[黃黎明|黃黎明]]在辦公時接到她母親打來的電話，電話內容是她母親抱怨在照顧孫子<ref group="註">對方是黃黎明弟弟[[黃仲崑|黃仲崑]]的兒子，為現今演藝界男演員[[黃遠|黃遠]]。</ref><ref>{{cite web |url=https://star.ettoday.net/news/1137217 |title=《魔法阿媽》獻給鄭彩霞阿嬤＆豆豆是黃仲崑媽和黃遠 |work=[[東森新聞|東森新聞]] |author=林映妤 |language=zh-tw |date=2018年4月3日 |accessdate=2018年4月15日}}</ref>時遇到的麻煩狀況。<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈從寫實到幻想的編劇樂趣〉|page=142至143頁}}</ref>接完電話後，黃黎明腦海裡浮現了一個故事靈感，劇情是「孫子不肯接受阿媽，而想把阿媽賣掉。」的內容。<ref name="Page144">{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈從寫實到幻想的編劇樂趣〉|page=144頁}}</ref>其中考慮到「要把阿媽賣掉」的橋段以現實影劇的手法不好表現，讓黃黎明想將此劇本以動畫片形式呈現出。<ref name="Page144"/>

在黃黎明向王小棣介紹完她所想出的新劇本並決定進行拍攝後，向[[行政院新聞局|行政院新聞局]]申請了輔助金案件，並得依據案件條件需在一年內將作品給提交出。<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈縱身入海話製片〉|page=125頁}}</ref>

起初王小棣想起她過去指導的學生-{[[石昌杰|石昌杰]]}-是擅長拍攝人偶動畫的創作者，在與-{石昌杰}-會面討論後，一度想與對方合作並將《魔法阿媽》以人偶動畫方式拍攝。之後因考量到經費以及拍攝時間會無法趕上所申請輔助金需要在一年內交案的條件等因素，讓王小棣改選擇用傳統的手繪動畫方式製作。<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈沉到人跡罕至的生命深處〉|page=106至108頁}}</ref><ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈熱情幽默的百變魔術師〉|page=165頁}}</ref>

後續王小棣透過學生王耿瑜的介紹下，讓發表過《零代傳說》、《[[天才超人頑皮鬼|天才超人頑皮鬼]]》等作品的漫畫家[[麥仁杰|麥仁-{杰}-]]進入製作團隊，讓他負責了劇中角色造型與場景的設計。<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈縱身入海話製片〉|page=133至134頁}}</ref>在配樂以及插入曲的部份則找來[[史擷詠|史擷詠]]擔任創作，<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈沉到人跡罕至的生命深處〉|page=116頁}}</ref>並請女歌手[[許景淳|許景淳]]為插入曲《會飛的寶貝》演唱。<ref>{{cite web |url=https://star.ettoday.net/news/1143108 |title=原聲帶沒收錄的隱藏神曲！哭爆兩岸三地+韓國 |work=東森新聞 |author=洪文 |language=zh-tw |date=2018年4月4日 |accessdate=2018年4月15日}}</ref>

影片完成後是在[[台北電影節|台北電影節]]上首映，於1998年10月5日當日在台北[[中山堂_(臺北市)|中山堂]]公開。<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈沉到人跡罕至的生命深處〉|page=123頁}}</ref>

=== 畫面 ===
作品構圖最初在人力不足的情形下，只由麥仁-{杰}-個人來獨自承擔繪製，之後透過有通過[[華特迪士尼影業|迪士尼]]動畫室測試的郭景州、王登鈺、陳偉松三名人士的協助下解決。<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈沉到人跡罕至的生命深處〉|page=121頁}}</ref>

因製作此影片的稻田電影工作室並非動畫公司，便有前往中國大陸以及韓國找尋可接洽的動作工作室，最後品質方面考量下選擇了韓國的Plus One Animation。<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈縱身入海話製片〉|page=136至137頁}}</ref>當時雖然有預算不足以支付對方費用的情況，但因為Plus One Animation的負責人李春滿肯定此影片劇本內容，便願意改透過用購買版權方式來解決王小棣一行人的預算問題，來承包此影片的動畫生產。<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈沉到人跡罕至的生命深處〉|page=121至122頁}}</ref>

=== 配音 ===
因考慮到講話嘴型與面部表情之間的配合，《魔法阿媽》是在作畫之前先進行配音流程，再讓畫面配合錄製好的聲音段落來繪製。<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈另一國度的魔法〉|page=168至169頁}}</ref>

劇中阿媽的配音由資深藝人[[文英|文英]]擔任，與她長期合作的[[許傑輝|許傑輝]]則擔任被邪靈附身時的黑貓酷羅，她們當時是在動畫畫面尚未繪製的情況下，靠著王小棣在旁指導來想像出劇中情節畫面來配音。<ref>{{cite web|url=http://ent.ltn.com.tw/news/breakingnews/1809748 |title=他回憶和文英配音《魔法-{阿}-嬤》 網瘋讚「經典神片」！ |work=[[自由時報|自由時報]] |language=zh-tw |date=2016年8月29日 |accessdate=2017年10月8日}}</ref>另麥仁-{杰}-也在劇中客串豆豆的爸爸一角。<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=關於《魔法阿-{媽}-》|page=17頁}}</ref>

為了要讓角色小男孩豆豆的聲音聽起來自然不做作，王小棣想以實際的兒童來配音，並請從事教導兒童表演的友人協助挑選，之後找來合適人選是當時唸小學六年級的莊博文。在莊博文做配音測試時，他五歲的弟弟莊建昱剛好有在一旁陪伴，當時莊建昱的聲音讓負責錄音的杜篤之引起了興趣，讓他一度曾向王小隷提議改讓莊建昱作為人選。最後結果是豆豆的配音仍由莊博文擔任，而莊建昱被挑選來演出劇中被壓扁的小蛇小扁。<ref>{{cite book |title=《魔法-{阿}-媽 我把阿媽變卡通了》 |publisher=麥田出版 |author=王小棣等人 |location=台灣 |isbn=957-708-744-2 |language=zh-tw |date=1999年4月1日 |chapter=〈沉到人跡罕至的生命深處〉|page=117至118頁}}</ref>

== 獎項入圍與爭議 ==
影片曾入圍[[第35屆金馬獎|第35屆金馬獎]]最佳動畫片作品名單，但之後被批評內容有宣揚迷信與牽涉怪力亂神等因素而未獲獎。<ref>{{cite book |title=《當代中國電影1998》 |publisher=[[時報文化|時報文化]] |author=黃寤蘭 |date=1999年3月1日 |location=台灣 |isbn=9789571328461 |language=zh-tw}}</ref>後續在《電影檔案：第35屆金馬影展得獎影片》一書上，有透露出當時評審之一黃木村認為此片沒有使用電腦特效而有手法過於古老粗糙的問題，另幾名評審劉立行、[[貢敏|貢敏]]、王清華等人批評內容有鼓吹迷信情況，並將此與宗教人士[[宋七力|宋七力]]過去引發過的爭議事件相提並論，之後遊說其他評審跟進而讓作品在金馬獎落選。
