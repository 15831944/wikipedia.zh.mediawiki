『'''魔界學園'''』（まかいがくえん）是[[日本|日本]][[漫畫|漫畫]]作品，原作是[[菊地秀行|菊地秀行]]、作畫是[[細馬信一|細馬信一]]，在「[[週刊少年Champion|週刊少年Champion]]」連載。原型是菊地秀行的小說『[[魔人學園|魔人學園]]』。

== 用語 ==
; 學生連合
: 通稱、學連。由學生構成的擁有巨大支配力的組織。東西各一個。
; 學校
: 主要登場的高校。
; 護衛團（ごえいだん）
: 各學校學生形成的自警團組織。負責防衛外部的攻撃或内部治安維持。最高位者稱作番長或總番、第二位是副番。
; 轉校生
: 正體不明的超人的存在。有魔戒十條的規則。
; 都立帝高校（とりつみかどこうこう）
: 物語開始時主人公轉學的高校。護衛團之名是烈鬼（れっき）。有[[夜間部|夜間部]]。
; 番外番（ばんがいばん）
: 不為人知的外番，各高護衛團的秘密兵器。

==地理==
; 日本
: 「東」和「西」完全分斷，不往來。
; [[淡路島|淡路島]]
: 大阪灣的島。
; 歐洲
: 日本西方有歐洲島，有歐洲諸國。
; 台灣
: 第1卷提及。「台灣有國際番長會議」的話題。
; 美洲
: 謎之大陸。
