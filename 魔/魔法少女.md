'''魔法少女'''（{{jpn|j=魔法少女|hg=まほうしょうじょ|rm=Mahou Shoujo|f=h}}），是一種[[日本|日本]]次文化的[[定型角色|定型角色]]。運用神奇的魔力，協助解決事件，或引發一場風波。

迄20世紀末葉，魔法少女動畫以《[[美少女戰士|美少女戰士]]》、《[[小魔女DoReMi|小魔女DoReMi]]》、《[[庫洛魔法使|庫洛魔法使]]》、《[[光之美少女系列|光之美少女]]》系列最廣為人知。2011年「反魔法少女」敘事的《[[魔法少女小圓|魔法少女小圓]]》帶來了[[後設小說|後設]]風潮的變革<ref>[https://www.youtube.com/watch?v=g-dMpko9ibo ③ 〔REVIEW〕 Mahou Shoujo Madoka Magica まどか☆マギカ レビュー Anime 〔SPOILER FREE〕 - YouTube]</ref>。近年來魔法少女動畫收視觀眾也有從幼女向轉移到男性向的現象。

==概要==
魔法少女的基本要素，是女角色會使用不為人知的神奇力量。依據力量的來源，大致分成「與生俱來」、「被賦予」兩種類型。視情形，力量也會有所升級<ref name="sm10">『The Sailor Moon Role-Playing Game and Resource Book』p.10</ref>。

此外，魔法少女的力量，不一定是字面上的魔法和魔力，也包括使用[[忍術|忍術]]或[[賽博格|賽博格]]力量的「變身少女」範疇<ref>[http://ranking.goo.ne.jp/ranking/026/anime_magicgirl/ gooランキング 思い出に残る女の子の変身少女キャラクターランキング] {{webarchive|url=https://web.archive.org/web/20140308000553/http://ranking.goo.ne.jp/ranking/026/anime_magicgirl |date=2014-03-08 }}</ref>。另外也存在《[[怪盜St._Tail|怪盜St. Tail]]》這種「毫無特殊力量」的魔法少女，以及如《[[網路安琪兒|網路安琪兒]]》在[[電腦網路|電腦網路]]虛擬空間活躍的少女<ref>『The Sailor Moon Role-Playing Game and Resource Book』p.8</ref>。

魔法少女的年齡大多是10歲至14歲之間，也就是正在出現[[第二性徵|第二性徵]]的時期。魔法少女的「變身」是女孩成熟的象徵<ref name="sm10" />。這是女孩憧憬長大成人的反映<ref name="sm10" />。魔法少女往往經由變身，讓身體變成大人。這些作品通常著重描寫：變身前後的認同衝突、雙重生活的困難（如保密、時間分配）。若有[[戀愛|戀愛]]要素則複雜度更將大增。通常會發展成「女主角為了心上人變身」，許多批評認為，這是宣傳混亂的價值觀<ref name="sm11">『The Sailor Moon Role-Playing Game and Resource Book』p.11</ref>。2011年的《[[魔法少女小圓|魔法少女小圓]]》藉由主角之一[[美樹沙耶香|美樹沙耶香]]的悽慘下場，對此一「傳統」極盡嘲諷。

魔法少女通常是天真無邪的，最常用的道具是魔法棒或[[水晶|水晶]]製品，但這通常只是引出力量的媒介。而力量良窳的真正關鍵在女主角本身的善良、堅定的意志與「愛」<ref name="sm11" />。

== 歷史 ==
一般認為[[橫山光輝|橫山光輝]]於1966年開始發表、改編自美國電視劇《{{link-en|Bewitched|Bewitched}}》的《莎莉變變變》（{{lang|ja|魔法使いサリー}}）是第一套魔法少女動畫，而第一套魔法少女漫畫(1962年發表)《[[甜蜜小天使|小魔鏡]]》就在《莎莉》結束後才動畫化，但那時是稱呼作「{{lang|ja|魔法使い}}」或「{{lang|ja|魔女っ子}}」；直至1980年的《[[魔法少女拉拉貝爾|小仙女麗寶]]》，日文才出現「魔法少女」這一名詞（原標題為「{{lang|ja|魔法少女ララベル}}」）。

1982年的《{{link-ja|魔法小仙子|魔法のプリンセス ミンキーモモ}}》確立了魔法少女必有「吉祥物」作伴的不成文鐵則。相對於魔法少女的情緒性，吉祥物的行動皆基於理性，並兼具教導、指正魔法少女的工作。雖然彼此間時常吵嘴，但總會互相諒解，並學著實踐吉祥物的正確指示。必須注意的是，吉祥物大多是雄性。他們彼此間的關係是一種秘密，既像是密友也像是夫妻，在互相理解下建構一段良好關係<ref name="sm11" />。

1992年的《[[美少女戰士|美少女戰士]]》替「戰鬥系魔法少女」派別立下了里程碑，然而此時魔法少女的技能仍多以單純的魔法為主，隨後2004年起強調拳腳的《[[光之美少女|光之美少女]]》（儘管只有初期特別強調），及加入軍事武裝要素的《[[魔法少女奈葉|魔法少女奈葉]]》則又是另一個轉折。

== 魔法少女作品 ==
魔法少女主要分成以下不同類型:
=== 日常系 ===
主角突然得到魔法力量，或主角由異世界來到人間界見習，故事是以圍繞日常生活為主，並使用魔法力量去幫助別人。通常都會與西方[[童話|童話]]中的皇室，以及[[小仙子|小仙子]](Fairy)文化有所關連。她們有時帶有傳統魔女的造型(即是高帽子、騎掃帚)。
==== 小魔女系 ====
大部分魔法都可以使用的日常系魔法少女，通稱做魔女っ子。但故事通常會有以下特徵：
* 除了主角之外，其他人都不知道她們是魔法少女(但有時候會設定為眾人皆知)；
* 魔法需要修行，甚至要進行考核才能得到某一種能力；
* 會有限定數量的道具才可以使用魔法；
* 不能利用魔法控制別人的思想；
* 使用禁斷魔法會得到相對的報應；
* 在2000年代後期開始出現以'''懂魔法的動物們'''作為主角的作品，2010年代的小魔女系作品主要在日本以外地區出品。

==== 變身系 ====
一種是主角得到一種獨特的變身力量，令主角可以利用道具而變身成心中想變成的人。早期主要變身成不同職業的大人，有些動畫更會變身成偶像歌手。2010年代前的作品唯一的規範就是變身時間是有時間限制，過了時間就可能永遠不能變回自己。在2010年代開始的偶像系不再局限於歌手，添加了舞蹈明星行列，並多數與街機卡片相關。

=== 魔法英雄系 ===
這種形式首見於七十年代的[[甜心戰士|甜心戰士]]和八十年代的[[She-Ra|She-Ra]]，由1990年代開始，直至今日仍然使用最多的主題。魔法英雄系也可以稱「美少女戰隊」系 ，即是主角或她的朋友（戰隊通常會有兩人或以上）突然得到變身力量，並利用變身後的力量(可能是被迫或半被迫地)與惡勢力對戰以保護[[地球|地球]]（或者主角居住的地方或星球）的故事。雖然相比起前面一種類型來說，這種的歷史最短，但亦是最廣為人知的類型，甚至已經成為了'''大多數人心目中對魔法少女最狹義的印象'''，並成為[[超級英雄|超級英雄]]的一個新類型。

=== 黑暗奇幻系 ===
到目前為此，只有2002年的《[[彩夢芭蕾|彩夢芭蕾]]》和2011年的《[[魔法少女小圓|魔法少女小圓]]》才有黑暗奇幻的故事架構，而且都被古典文學、童話的黑暗面影響深遠。如果不計《[[叛逆的物語|叛逆的物語]]》，這一系不論故事中段多麼殘酷痛苦，結局最終都以有點苦甜參半、開放式的Happy Ending結束。

2012年由佐藤健太郎創作的《[[末日的魔法少女|末日的魔法少女]]》又再一次推翻魔法少女的各種刻板印象，主角群不再由天真無憂的少女們擔任，故事也充滿無盡的殺戮和陰謀，將肌肉、鬼娃娃、殭屍、人偶等等以往悖離魔法少女的要素運用在故事中。甚至在《[[魔法少女網站|魔法少女網站]]》裡的主角們皆過著充滿黑暗和悲慘的人生。魔法少女並非代表愛、光明、希望等美好事物，而是死亡、恐怖、絕望、仇恨、殺戮、破壞之象徵。

== 日本國外 ==
現時，除了日本，海外也有魔法少女動畫的創作，當中以意大利的《[[魔法俏佳人|魔法俏佳人]]》、迪士尼的《[[魔力W.i.t.c.h.|魔力W.i.t.c.h.]]》，以及中國的《[[巴啦啦小魔仙|巴啦啦小魔仙]]》最為知名。現在總共有7個國家製作過此類題材作品，分別是：日本、意大利、美国、中國、韩国、法國以及德国。
