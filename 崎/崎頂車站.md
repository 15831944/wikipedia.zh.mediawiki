'''崎頂車站'''位於[[台灣|台灣]][[苗栗縣|苗栗縣]][[竹南鎮|竹南鎮]][[崎頂里_(苗栗縣)|崎頂里]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(北段)|縱貫線]]的[[鐵路車站|鐵路車站]]。崎頂車站因興建新站房，原木造站房遭拆除。目前的崎頂車站並沒有車站主建築，但設有公廁與採光罩屋頂的簡易候車區，兩座[[側式月台|側式月台]]之間是以人行天橋作為跨越鐵路的動線。

== 車站構造 ==
*[[側式月台|側式月台]]兩座。
*跨線[[天橋|天橋]]連接月臺。
*[[公共廁所|公共廁所]]。
*簡易式的採光罩結構候車區。
*西側北上月台，亦有入口方便民眾進出站。

==  公車資訊 ==
== 歷史 ==
*1928年3月12日：設「崎頂信號場」，當時路線還有連結目前已經廢棄的崎頂一、二號隧道。
*1929年：因為路線坡度過陡，拆除原本經過隧道的舊路線。
*1931年10月1日：改為「崎頂驛」，並增建一座木造站房。
*1981年5月1日：由[[三等站_(台鐵)|三等站]]降為甲種[[簡易站|簡易站]]，並指定由[[竹南車站|竹南站]]管理。
*1993年3月15日：由甲種[[簡易站|簡易站]]調整為[[簡易站|簡易站]]，同年5月3日起由[[簡易站|簡易站]]降為[[招呼站_(台鐵)|招呼站]]。
*1993年至1999年：於每年夏季崎頂海水浴場開放期間，由管理站[[竹南車站|竹南車站]]輪流派站員前往售票外，還會臨時增停部份[[復興號列車|復興號]]以及[[莒光號列車|莒光號]]列車，以方便前往海水浴場之旅客。
*1996年：木造站房拆除。
*2013年5月30日：啟用多卡通刷卡機。
