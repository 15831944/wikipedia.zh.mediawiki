'''圣安东尼奥'''（{{lang-es|San Antonio}}）位于[[美国|美国]][[得克萨斯州|得克萨斯州]]中南部，是[[美国|美国]][[得克萨斯州|得克萨斯州]]人口第二多的城市（排在[[休斯敦|休斯敦]]之后），同时也是美国人口第七多的城市。据2005年[[美國人口普查局|美國人口普查局]]估计，圣安东尼奥市人口超过120万人。它所在的由八个[[县_(美国)|县]]组成的都会区则有超过180万的人口，在美国排名第二十九。圣安东尼奥是[[比爾縣_(德克薩斯州)|比爾縣]]的郡治所在，覆盖南德克萨斯北边与Texas Hill Country东南超过400平方英里的土地。

1691年[[西班牙|西班牙]]探险者在到达此地区之日正值[[葡萄牙|葡萄牙]]的[[圣徒|圣徒]][[帕多瓦的圣安多尼|帕多瓦的圣安多尼]]的[[圣日|圣日]]，圣安东尼奥因此而得名。此城有强大的驻军，如：{{Link-en|Sam Houston堡|Fort Sam Houston}}、[[拉克兰空军基地|拉克兰空军基地]]、[[兰多夫空军基地|兰多夫空军基地]]和{{Link-en|Brooks市驻军基地|Brooks Air Force Base}}。圣安东尼奥也是南得克萨斯地区最大的、也是唯一一所提供医学研究和护理的医疗机构：[[南得克萨斯医疗中心|南得克萨斯医疗中心]]的所在。

拥有著名的河滨人行道(River Walk)、The Alamo、[[西班牙裔|西班牙裔]]文化、墨西哥文化的市集廣場(Market Square)，还有SeaWorld and Six Flags Fiesta Texas[[主题公园|主题公园]]，圣安东尼奥每年迎来2000万游人。它亦拥有德州第一座现代艺术[[博物馆|博物馆]]——the Marion Koogler McNay Art Museum。

根据2000年人口普查，圣安东尼奥共有人口1,144,646人，其中32%为非西班牙裔[[白人|白人]]、7%为[[非裔美国人|非裔美国人]]、2%为[[亚洲人|亚洲人]]、1%为[[美国原住民|美国原住民]]，其余58%为拥有西班牙血统的人及其它。

美国最大的固定电话和移动电话服务商[[美国电报电话公司|美国电报电话公司]]总部於1992至2008年位於聖安東尼奧。2008年搬至[[達拉斯|達拉斯]]市中心的AT&T一號廣場。

圣安东尼奥的AT&T中心球馆是[[NBA|NBA]]联赛[[圣安东尼奥马刺队|圣安东尼奥马刺队]]、[[WNBA|WNBA]]联赛[[圣安东尼奥銀星|圣安东尼奥銀星]]和[[美国冰球联盟|美国冰球联盟]][[圣安东尼奥暴牛|圣安东尼奥暴牛]]的主场。

==地理与气候==
圣安东尼奥属[[副热带湿润气候|副热带湿润气候]]，冬无严寒，降水充足而集中在5、6、和10月。冬季温和，时而偏向温暖，日最高气温低于或等于{{convert|5|C}}的平均日数为2.7天，日最低气温低于或等于{{convert|0|C}}的平均日数为18天，低于或等于{{convert|-5|C}}的平均日数为2.3天；夏季酷暑漫长，日高温达{{convert|30|C}}的日数年均有155天，达{{convert|100|F|disp=flip}}的有12天；日低温达{{convert|25|C}}的年均日数为26天。<ref name = "NWS New Braunfels, TX"/>最冷月（1月）均温{{convert|51.8|F|1|disp=flip}}，极端最低气温{{convert|0|F|0|disp=flip}}（1949年1月31日）。<ref name = "NWS New Braunfels, TX"/>最热月（8月）均温{{convert|85.3|F|1|disp=flip}}，极端最高气温{{convert|111|F|0|disp=flip}}（2000年9月5日）。<ref name = "NWS New Braunfels, TX"/>无霜期平均为271天（3月1日至11月26日）。<ref name = "NWS New Braunfels, TX"/>年均降水量约{{convert|32.3|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|10.11|in|disp=flip}}（1917年），{{convert|52.28|in|disp=flip}}（1973年）。<ref name = "NWS New Braunfels, TX"/>年均降雪量为{{convert|0.7|in|cm|disp=flip}}；许多冬季的积累降雪量微少以致难以测量或许无雪，1984–85年的降雪量最多，积累降雪量为{{convert|15.9|in|cm|disp=flip}}。<ref name = "NWS New Braunfels, TX"/>

=== 航空 ===
[[聖安東尼奧國際機場|聖安東尼奧國際機場]]（SAT）位於住宅區的中央聖安東尼奧，距離北部市區約八公里。它有兩個航廈，由21家航空公司提供44個目的地，其中包括[[墨西哥|墨西哥]]的六個城市。<br>
[[史汀森市立機場|史汀森市立機場]]（SSF）是一座後緩機場位於主城區南部6英里（10公里），聖安東尼奧。機場有兩條跑道，它也是德克薩斯州航空博物館的家。
