'''聖文森特和格林納丁斯'''（{{lang-en|Saint Vincent and the Grenadines}}），簡稱'''聖文森特'''，是[[北美洲|北美洲]]的一個國家，位于[[加勒比海|加勒比海]]的[[小安的列斯群岛|小安的列斯群岛]]中的[[向风群岛|向风群岛]]南部，[[聖露西亞|聖露西亞]]及[[格瑞那達|格瑞那達]]之間。居民主要是黑人和黑白混血种人。

== 歷史 ==
{{main|聖文森特和格林納丁斯歷史}}
圣文森特和格林纳丁斯原为[[印第安人|印第安]]加勒比部落居住地。
* 1492年，[[哥伦布|哥伦布]]到达[[圣文森特岛|圣文森特岛]]时，加勒比族[[印第安人|印第安人]]已在岛上居住。
* 1627年，[[英国|英国]]占领该岛。后[[法国|法国]]声称对此岛拥有主权，两国为争夺该岛进行多次战争。
* 1783年，《[[凡尔赛条约|凡尔赛条约]]》确认英国对该岛的统治。
* 1833年起，圣文森特从属向风群岛领地。
* 1958年1月，圣文森特岛与[[格林纳丁斯群島|格林纳丁斯群島]]一起加入“[[西印度联邦|西印度联邦]]”，直至1962年5月西印度联邦解散。
* 1969年10月，圣文森特和格林纳丁斯实行“内部自治”，为英国联系邦，但外交、防务仍由英国掌管。
* 1979年10月27日，圣文森特和格林纳丁斯宣布独立，仍是[[英联邦|英联邦]]成员国。

== 地理 ==
{{main|聖文森特和格林納丁斯地理}}
圣文森特和格林納丁斯由圣文森特本島及格林納丁斯群島組成，面積共389平方公里。聖文森特本島面積346平方公里，格林納丁斯群島則由三十二個小島組成，面積合共43平方公里。

主岛为一火山岛，多火山，最高峰苏弗里耶尔火山，海拔1234米，经常喷发，例如1970年曾喷发20多次。

虽然地处热带，但有海风调节，气候温和，年降水量2540毫米，夏秋多飓风。

== 行政區劃 ==
{{main|聖文森特和格林納丁斯行政區劃}}

聖文森特和格林納丁斯全國可分為6個教區（parishes）：

* [[格瑞那丁區|格瑞那丁區]]（Grenadines Parish）
* [[聖安德魯區_(聖文森及格瑞那丁)|聖安德魯]]（Saint Andrew Parish）
* [[聖大衛區|聖大衛]]（Saint David Parish）
* [[聖喬治區_(聖文森及格瑞那丁)|聖喬治]]（Saint George Parish）
* [[聖派屈克區_(聖文森及格瑞那丁)|聖派屈克]]（Saint Patrick Parish）

== 經濟 ==
{{main|聖文森特和格林納丁斯經濟}}
聖文森特和格林納丁斯最主要的產業為農業和國際金融業，主要作物為[[香蕉|香蕉]]。[[金融业|國際金融業]]也是聖文森特和格林納丁斯經濟重要收入來源。

[[旅遊業|旅遊業]]也是聖文森及格林納丁的經濟重要收入來源。

== 聖文森特和格林納丁斯證券交易所(SVGEX) ==
{{main|聖文森特和格林納丁斯經濟}}
[[聖文森特和格林納丁斯證券交易所|聖文森特和格林納丁斯证券交易所有限公司]] [https://www.svgex.com.vc (Saint Vincent and the Grenadines Securities Exchange)]是東加勒比海國家首三位交易量排名最高的[[證券交易所|證券交易所]]，並由[[東加勒比國家資本市場協會|東加勒比國家資本市場協會]][https://www.cmaec.org (Capital Market Association of the Eastern Caribbean)] 監督與管轄，並接受國際企業包括東南亞與亞太區公司申請[[上市公司|上市]]掛牌向全球投資者融資。

== 軍事 ==
圣文森特和格林纳丁斯為[[大英國協|大英國協]]的一員，但事實上本國並未擁有軍隊，島上僅擁有圣文森特和格林纳丁斯皇家警察（''Royal Saint Vincent and the Grenadines Police Force''）與海岸警備隊（''Coast Guard''）。也基於國家安全保障之故，為[[地區安全體系|地區安全體系]]（''Regional Security System''）創始時的加盟國。
