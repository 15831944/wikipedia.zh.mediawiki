'''圣训堂'''位于[[江苏省|江苏省]][[南京市|南京市]][[建邺区|建邺区]]，是[[南京市|南京市]]最大的[[基督教新教|基督教新教]]教堂。

== 历史 ==
2007年1月，南京市政协社会法制民族宗教委员会提交了名为《关于在河西新城区择址建设基督教堂的建议》的提案，南京市政府经过讨论，最后决定兴建教堂，以适应城市建设国际化的要求，并使其成为南京市一道新的文化景观。南京市基督教协会邀请[[东南大学|东南大学]]建筑研究设计院、[[南京大学|南京大学]]建筑设计院、北京中华建筑设计院和美国LDX公司参与设计，经过评审，东南大学建筑研究设计院方案最终入选。
2007年11月，圣训堂举行奠基仪式。<ref>{{Cite web|url = http://newtown.nanjing.gov.cn/xcyx/pzxc/wtpt/201703/t20170321_4403950.html|title = 基督教圣训堂介绍|accessdate =2018-03-08 |author = |date =2017-03-21 |publisher =南京市河西新区开发建设管委会 }}</ref>

2009年7月，教堂钢结构主体竣工。<ref>{{Cite web|url = http://www.jidujiao.com/xinwen/2240/|title = 南京市基督教举行“圣训堂钢结构主体竣工仪式”|accessdate =2018-03-08 |author = |date =2009-07-27 |publisher =基督教中文网 }}</ref>2013年3月31日，[[复活节|复活节]]当天，圣训堂举行献堂典礼和落成典礼，正式投入使用。<ref>{{Cite web|url = http://www.gospeltimes.cn/index.php/portal/article/index/id/17834|title = 南京市最大教堂圣训堂将于复活节献堂|accessdate =2018-03-08 |author = |date =2013-03-25 |publisher =福音时报 }}</ref>2014年，圣训堂作为[[2014年夏季青年奥林匹克运动会|青奥会]]指定接待宗教活动场所，接待外宾来访，举行宗教活动。<ref>{{Cite web|url = http://www.gospeltimes.cn/index.php/portal/article/index/id/24243|title = 青奥会指定接待宗教活动场所之一：圣训堂|accessdate =2018-03-08 |author = |date =2014-08-01 |publisher =福音时报 }}</ref>

== 建筑 ==
教堂由东南大学建筑研究设计院下设的UAL城市建筑工作室负责设计，采用玻璃幕墙的现代化风格。<ref>{{Cite web|url = http://www.gospeltimes.cn/index.php/portal/article/index/id/1344|title = 南京圣训堂建设促进城市建设国际化|accessdate =2018-03-08 |author = 马克|date =2007-12-26 |publisher =福音时报 }}</ref>结合教堂临水而立的场地条件，在设计中赋予教堂主堂的造型“[[诺亚方舟|诺亚方舟]]”的象征意义。<ref>{{Cite web|url = http://www.ualstudio.com/pro/detail.asp?ID=77|title = 南京基督教河西国际礼拜堂|accessdate = 2018-03-08|author = |date = |publisher = UAL城市建筑工作室}}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

教堂占地面积为20.72亩，建筑面积为13500平方米（其中主堂建筑面积7700平方米，附属房屋及地下停车场5800平方米），主堂可以容纳5000人聚会，另有两个辅堂。<ref>{{Cite web |url =http://www.njmzzj.gov.cn/33609/33628/njjdjh/201710/t20171013_5054799.html |title =圣训堂 |accessdate =2018-03-08 |author = |date =2017-10-13 |publisher =南京市市民族宗教事务局 }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
