'''圣言会会馆旧址'''是位于中国[[山东省|山东省]][[青岛市|青岛市]][[市南区|市南区]][[浙江路_(青岛)|浙江路]]15号甲、[[曲阜路_(青岛)|曲阜路]]1号的一座天主教会建筑群旧址，建于1899-1902年，初为[[天主教|天主教]][[圣言会|圣言会]]在青岛的传教中心。建筑群大部分保存至今<ref Name=qing>{{cite book |author=袁宾久 |title=《青岛德式建筑》|year=2009 |publisher=中国建筑工业出版社 |location=青岛 |isbn=9787112109050}}</ref><ref Name=de>{{cite book |author=华纳 |title=《德国建筑艺术在中国：建筑文化移植》|year=1994 |publisher=Ernst & Sohn |location=柏林 |isbn=3433024294}}</ref><ref Name=ming>{{cite book|author=王栋|title=《青岛影像 1898-1928：明信片中的城市记忆》|year=2017|publisher=中国海洋大学出版社|location=青岛|isbn=978-7-5670-1113-7}}</ref>。

==历史==
[[File:Bundesarchiv_Bild_137-032801,_P._Franz_Bartels_S.V.D..jpg|160px]]
[[File:Bundesarchiv_Bild_137-004088,_China,_Tsingtau,_Katholische_Mission.jpg|210px]]
[[File:青岛圣言会会馆1914年损毁状况.jpg|210px]]
[[File:青岛圣言会附属教堂旧址.jpg|210px]]
天主教[[圣言会|圣言会]]（SVD，又称斯泰尔修会，德语：Steyler Mission）自1882年起进入山东传教，并于1885年开辟[[天主教兖州教区|鲁南代牧区]]<ref Name=si>{{cite web |url=http://blog.sina.com.cn/s/blog_488832200100q0y4.html|title=斯泰尔修会的青岛路线（上）}}</ref>。1897年德国占领青岛后，[[天主教兖州教区|鲁南代牧区]]的宗座代牧[[安治泰|安治泰]]（Johann Baptist von Anzer）于1898年秋派遣圣言会神父[[白明德|白明德]]（Franz Bartels）至青岛开辟传教站<ref Name=bai>{{cite web |url=http://blog.sina.com.cn/s/blog_488832200100j27e.html|title=传教士白明德}}</ref>。最初白明德在[[青岛天后宫|天后宫]]紧后侧用木板建成了一座[[广西路_(青岛)#天后宫及圣言会临时教堂|临时天主教堂]]。

1898年9月，[[胶澳总督府|胶澳总督府]]公布了第一版城市规划图，并于10月开始了土地拍卖。起初，由于总督[[罗绅达|罗绅达]]（Carl Rosenthal）、继任总督[[叶世克|叶世克]]（Paul Jaeschke）乃至其顶头上司海军部国务秘书[[阿尔弗雷德·冯·提尔皮茨|阿尔弗雷德·冯·提尔皮茨]]（Alfred von Tirpitz）对传教会采取的不信任态度，总督府不肯依照圣言会希望的方式和价格提供他们看好的土地。城市规划图也因为将天主教堂规划在离主城区较远的西北方而受到了天主教会的抨击。总督府的一系列行为令安治泰大为恼火，他认为德军占领青岛的机会是[[曹州教案|用两位圣言会教士的生命换来的]]<ref Name=bai/><ref Name=si/>。

最终在安治泰与白明德的努力下，圣言会在青岛市区购买了至少7处土地，其中一处是德租时期路易特波尔德街（Luitpoldstraße，今[[浙江路_(青岛)|浙江路]]）、柏林街（Berliner Straße，今[[曲阜路_(青岛)|曲阜路]]）、阿尔贝特街（Albertstraße，今[[安徽路_(青岛)|安徽路]]）与霍亨洛厄路（Hohenloheweg，今[[德县路|德县路]]）的合围地块，这片土地位于一个山坡的顶部，风景秀丽，同时紧邻[[大鲍岛|大鲍岛]]华人区，便于传教工作的开展<ref Name=bai/>。1899-1902年，圣言会在该地块上建设了这座传教站建筑群，设计者是总督府建设局建筑师[[彼得·贝尔纳茨|彼得·贝尔纳茨]]（Peter Bernatz，此人是来自慕尼黑的天主教徒），建筑群容纳了圣言会的教会机构、传教士住宅、小教堂和印刷厂<ref Name=qing/>。在传教站与[[圣心修道院|圣心修道院]]之间是为圣言会建设天主教堂预留的位置。圣言会小教堂自1902年启用，一直到1934年[[圣弥爱尔大教堂|圣弥额尔主教座堂]]在传教站邻近的预留空地上建成，这里一直是青岛市区天主教徒的礼拜堂。同年，白明德在传教站院落内开设了一所男童学校<ref Name=xia>{{cite web |url=http://blog.sina.com.cn/s/blog_488832200100q0yb.html|title=斯泰尔修会的青岛路线（下）}}</ref>。约1906年，圣言会又在临德县路的位置建设了一栋两层楼。

1914年8月，[[青岛战役|青岛战役]]爆发。11月6日，圣言会会馆被日军的航空炸弹击中。根据白明德的回忆，11月7日德军投降前的20分钟内，传教站一共中了九发炮弹，所有的窗玻璃都被震碎，门也被炸毁，传教士们躲在地下室里，无人伤亡<ref Name=bai/><ref Name=xia/>。战后，会馆建筑根据原图纸修复。1923年3月，圣言会将原男子学校改组为明德小学<ref Name=xia/>。该建筑至五十年代一直是圣言会的地产<ref Name=xia/>。1950年代，明德小学改为[[青岛德县路小学|德县路小学]]。六十年代左右建筑曾遭破坏。约八十年代，德县路小学拆除了建于1906年的两层楼和建筑的另一小部分以建设教学楼。2008年，建筑西翼经历了大修。现西翼部分为天主教会所用，部分对外出租做为酒吧使用。其东侧的原小教堂现为市南交警大队中山路支队使用，其他房间有的现为商业用途<ref Name=xia/>，部分为德县路小学使用。

==建筑特色==
[[File:青岛圣言会旧址西翼老照片.jpg|210px]]
[[File:青岛圣言会旧址西翼.jpg|210px]]
建筑师贝尔纳茨在设计该建筑时可能参考了欧洲本土的[[修道院|修道院]]建筑的特点，整个建筑群分为不同单元而互相融为一体，包括了[[传教士|传教士]]住宅、办公室、[[小圣堂|小教堂]]、[[印刷厂|印刷厂]]等部分。建筑由[[花岗岩|花岗岩]]石块砌墙基，屋顶覆以[[仰合瓦|中式灰瓦]]。除西翼建筑和临德县路后建成的独立建筑为两层加阁楼及[[地下室|半地下室]]外，东侧建筑因资金不足均建为单层加阁楼及半地下室。单层的建筑外墙均为中式灰砖，双层建筑一楼外墙为灰砖，二楼为白墙，墙面和窗户以红砖、花岗岩及铁艺[[百合花饰|鸢尾花]]装饰，西翼南北两端均有塔楼，其南端街角的塔楼为该建筑的标志性特点，檐口中央有一精美的花岗岩砌山花，整个建筑具有德租初期的复古风格<ref>{{cite web |url=http://www.thatsqingdao.com/qingdao-societas-verbi-divini-svd/|title=Societas Verbi Divini (SVD)}}</ref><ref Name=de/>，其中式建筑材料的使用也体现出其应急性<ref Name=qing/>。

该建筑的部分屋顶后来改铺矩形红瓦。约上世纪六十至七十年代，两座塔楼的顶部和南翼主入口（面向曲阜路，即原教堂入口）屋顶曾经有一个[[十字架|十字架]]的[[老虎窗|老虎窗]]被拆除，西翼街角的拱窗也曾被改为大门。约80年代被拆除建设新的教学楼。2000年代时该建筑原灰砖被涂上蓝涂料，二楼的白墙改为黄墙，原教堂入口因交警使用被涂上警用建筑的蓝白两色涂装。2008年，年面向[[圣弥爱尔大教堂|主教座堂]]南侧广场的西翼被修缮，灰砖墙的蓝涂料连同历史包浆一并被清除，二楼墙面恢复为白色，被开作大门的窗户被修复，南北两端的塔楼顶部模仿原貌重建，但原木结构铺灰砖塔顶被建为绿色铁皮塔顶，北侧塔楼还加上了一个原本不存在的金色十字架。其东侧部分仍未修复。另外朝西的窗户也加上了木制[[百叶窗|百叶窗]]，后来又被拆卸下来。
