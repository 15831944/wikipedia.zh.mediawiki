'''圣职人员'''，又称'''神职人员'''，指在[[宗教|宗教]]机构中，担任'''宗教性职务'''的人员。大多數宗教都有圣职人员，虽然名称各异。而在某些宗教的某些教派，更對指定一定職級以上需要[[出家|出家]]。

== 道教 ==
以从事[[道教|道教]]活动为[[职业|职业]]的人员称为[[道士|道士]]。男性道士为[[乾道|乾道]]，也称道人、羽士、羽客、黄冠等，又尊称为'''道长'''。女性道士则为“'''坤道'''”，可称'''女冠'''，俗称'''道姑'''。

== 佛教 ==
[[佛教|佛教]]中的“圣职人员”（Buddhist clergy），中文一般通称为[[僧侣|僧侣]]、和尚、[[上人|上人]]、[[法師_(佛教)|法師]]、[[師父|師父]]，专指[[出家|出家]]众、[[僧伽|僧伽]]，包括[[比丘|比丘]]、[[比丘尼|比丘尼]]、式叉摩尼、沙弥和沙弥尼等，在家的信徒稱[[居士|居士]]僅為平信徒。近現代的教派和法令的變化，正式的僧侶也有不少結婚的，或居士一樣可以擔任聖職。

=== 天主教 ===
[[天主教|天主教]]中，圣职人员是因着[[圣秩圣事|圣秩圣事]]得到祝圣的[[主教|主教]]、[[神父|神父]]和[[执事|执事]]。神父僅修士可以擔當，主教則必須具備神父身分，在佔主流的拉丁禮傳統中，原則上兩者均需終身[[單身|不婚]]。

[[修士|修士]]和[[修女|修女]]没有领受[[圣秩圣事|圣秩圣事]]，因此不属于圣职人员。

=== 东正教 ===
[[东正教|东正教]]神职人员分为独身(黑神品)、结婚(白神品)两个系统。结婚在家者可成为[[诵经士|诵经士]]、[[助祭|助祭]]、[[辅祭|辅祭]]、[[大辅祭|大辅祭]]、[[司祭|司祭]]、[[大祭司_(东正教)|大祭司]]。出家修行者，可成为[[修士|修士]]、[[修士辅祭|修士辅祭]]、[[修士大辅祭|修士大辅祭]]、[[修士司祭|修士司祭]]、[[修士大司祭|修士大司祭]]、[[主教|主教]]、[[大主教|大主教]]、[[督主教|督主教]]、[[都主教|都主教]]、[[牧首|牧首]]。

== 伊斯兰教 ==
中国用'''[[阿訇|阿訇]]'''一词称呼[[伊斯兰教|伊斯兰教]]宗教职业者，一般为男性。他们的工作性质为讲经传道，领导宗教仪式。[[维吾尔族|维吾尔族]]等[[突厥语|突厥语]]族穆斯林则称其为[[毛拉|毛拉]]，其中德高望重者称“大毛拉”。

阿訇的妻子则被称为[[师娘|师娘]]。在中国特有的[[清真女寺|清真女寺]]中，她们还会担任管理者的角色。

== [[神道教|神道教]] ==

傳統男性稱'''神主'''或'''[[神官|神官]]'''，女性叫[[巫女|巫女]]。巫女要獨身但不出家的，如果巫女要結婚在過去要辭職，現在仍然當神社工作但改轉神官(神主)。

== [[印度教|印度教]] ==

一般稱[[上師|上師]]，但這個詞語在其他印度系宗教，如[[佛教|佛教]]也有使用的。

== 巴哈伊信仰 ==
[[巴哈伊信仰|巴哈伊信仰]]中，没有圣职人员和[[教堂|教堂]]。这是它与其他宗教最大的不同。
