'''聖人'''，指被大众认为具有特别[[美德|美德]]和[[神圣|神圣]]的人。在[[河洛故国|河洛故国]]，古代聖明的[[君主|君主]][[帝王|帝王]]，及後世道德高尚[[儒学|儒学]]造詣高深者，稱聖人。圣人的介定經常出現在[[诸子百家|诸子百家]]书籍。

有的[[宗教|宗教]]专门通过一定的仪式加封聖人，但也有聖人直接被大众尊奉为聖人，比方一些[[主要宗教創始人列表|宗教創始人]]或[[神職人員|神職人員]]。[[基督宗教|基督宗教]]（包括[[羅馬天主教會|羅馬天主教會]]、[[聖公宗|聖公宗]]、[[東正教會|東正教會]]）、[[印度教|印度教]]、[[佛教|佛教]]（猶如[[藏傳佛教|藏傳佛教]][[活佛|活佛]]和[[南傳佛教|南傳佛教]]的[[羅漢|羅漢]]）、[[伊斯兰教|伊斯兰教]]、[[古巴|古巴]]的[[桑特里亚教|桑特里亚教]]中都有聖人。一些[[新纪元运动|新纪元运动]]的人也将聖人的概念结合入他们的[[信仰|信仰]]。

如果沒有被宗教封聖的被公眾認可的人物，或者從當事人所不信仰的其他宗教的角度界定，可能會被稱為[[偉人|偉人]]或[[英雄|英雄]]。

在很多神話中聖人都被說成有[[超能力|超能力]]的，對於當代的奇幻或超級英雄影甚深。

== 中國的聖人 ==
=== 儒家和[[儒教_(宗教)|儒教]] ===
{{see also|君子}}
[[儒家|儒家]]主要強調[[堯|堯]]、[[舜|舜]]、[[禹|禹]]、[[商湯|商湯]]、[[周文王|周文王]]、[[周武王|周武王]]、[[周公|周公]]、[[孔子|孔子]]為傳續儒門[[道統|道統]]的聖人之代表。而[[顏子|顏子]]、[[孟子|孟子]]、[[子思|子思]]和[[曾子|曾子]]，還有一些先賢大儒，只能被視為境界接近聖人的人（雖然他們有時也尊稱為聖人，但並非絕對意義的聖人。）

《[[漢書|漢書]]·[[古今人表|古今人表]]》將古人大分類成上中下三級，又細分為九等，上上、上中、上下、中上、中中、中下、下上、下中、下下，其中，上上為聖人，「古今人表」中第一位聖人是「[[太昊|太昊帝宓羲氏]]」，最後一位是「仲尼」、也就是[[孔子|孔子]]。

《[[孔子家語|孔子家語]]-五儀解》孔子曰：「所謂聖人者，德合於天地，變通無方，窮萬事之終始，協庶品之自然，敷其大道，而遂成情性，明並日月，化行若神，下民不知其德，覩者不識其鄰，此謂聖人也。」孔子自省此論述標準，並不自認為聖人，謹慎的以次一等的「君子」自居，後世卻妄加違封為「聖人」、「至聖」...等，錦上添花。

=== 道家和道教 ===
[[道教|道教]]認為聖人無常心，以百姓心為心。《[[辩问|辩问]]》中說：“俗所谓圣人者，皆治世之圣人，非得[[道|道]]之圣人，得道之圣人，则黄老是也。治世之圣人，则周孔是也。”道教形成初期，人們视[[黄帝|黄帝]]与[[老子|老子]]等道家始祖为道教聖人。
{{cquote|天地不仁，以萬物為芻狗；'''聖人'''不仁，以百姓為芻狗。|《[[老子|老子]]》第五章}}

後來道教信徒供奉的後天得道[[仙人|仙人]]或[[真人|真人]]亦可稱為道教的聖人。

=== 其他 ===
在[[先秦|先秦]][[百家爭鳴|百家爭鳴]]的時期，[[墨子|墨子]]等哲學家也被其弟子尊為聖人。

《[[黄帝内经|黄帝内经]]·上古天真论篇第一》: 有圣人者，处天地之和，从八风之理，适嗜欲于世俗之间，无恚嗔之心，行不欲离于世，被服章，举不欲观于俗，外不劳形于事，内无思想之患，以恬愉为务，以自得为功，形体不敝，精神不散。

== 佛教 ==
{{Main|聖人 (佛教)}}
{{seealso|佛|菩薩|獨覺|聲聞}}

在佛教中，“聖人”就相當於“大成就者”，是證得[[聖果|聖果]]的修行人。其中最尊者爲[[佛陀|佛陀]]，佛是智慧的意思、是覺悟的意思，有智慧，大徹大悟，這稱之為佛。中國的聖是明本，聖人是明白人，對於宇宙人生的道理事相通達明瞭，這樣的人就稱為聖人。

此外，依照佛陀的指示修行者，可以得到[[阿羅漢|阿羅漢]]等[[四向四果|四向四果]]，以及獨覺[[辟支佛|辟支佛]]，這是聲聞緣覺和獨覺乘這[[二乘|二乘]]的聖果，是南傳[[巴利語系佛教|巴利語系佛教]]的主要聖果。而發願成佛的人，并逐步修行[[五十二位|證果]]的聖人，稱爲[[菩薩|菩薩]]，大菩薩修行圓滿將來必定[[成佛|成佛]]，這是[[北傳佛教|北傳梵語系佛教]]的主要聖果。另外，金剛乘裡還可證得[[法身佛|法身佛]]。

本來佛、菩薩、阿羅漢都是具體的歷史名人，跟神話中的[[神祇|神祇]]風馬牛不相關，但因為長期把聖人[[神化|神化]]，並在抽像的[[報身佛|報身佛]]也使用本來對於真人的稱謂，所以逐漸地變成了和古代神話相似的描述。<ref name="淨土大經解演義第八十五集">http://new.jingzong.org/Category_450/index.aspx?4772{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }} 《淨土大經解演義》 第二冊 第八十五集 2010/7/20 [[香港|香港]][[佛陀教育|佛陀教育]]基金會 02-039-0085</ref>

佛教通常把除了佛陀外的普通聖人分三品，即上品的[[菩薩|菩薩]]，中品的[[獨覺|獨覺]]，下品的[[聲聞|聲聞]]，稱為[[三乘|三乘]]。但實際上三者是互相滲透和融匯的，因為大乘佛教需要爲僧徒們樹立修[[菩提道|菩提道]]而成佛的信心，故分列三乘，突出菩薩乘最大，但也明確提出“三乘俱[[一佛乘|一佛乘]]”。

== 基督教聖人 ==
{{main|基督教聖人}}
=== 天主教 ===
{{Main|天主教聖人}}

按照[[羅馬天主教|羅馬天主教]]教義，聖人是指一些已經死去、如今與基督一同在天上的人；他們在世度聖潔的生活，有高尚的德行，死後被[[教廷|教廷]]册封，尊奉為聖人。[[特倫多大公會議|特倫多大公會議]]信經表示，信徒該求聖人為他們轉求[[天主|天主]]，也該尊崇聖人遺物和[[聖像|聖像]]{{來源請求}}。其他宗教也主張向聖人祈求。有些宗教聲稱，屬下的全體成員都是無罪的聖人。聖經多次提到聖人或聖徒。聖經把基督手下144000位受[[聖靈|聖靈]][[膏立|膏立]]的門徒稱為聖徒。

[[基督宗教|基督宗教]][[聖人|聖人]]是指在[[基督教|基督教]]中被[[教会|教会]]认可为聖的人物。雖然聖人的設立出現在基督宗教中，但不同教會對聖人的理解和認可卻不是一致的，有的更完全不認同。目前聖人的設立只出現於[[羅馬天主教會|羅馬天主教會]]、[[聖公宗|聖公宗]]、[[东正教|东正教]]等；而大多數[[基督新教|基督新教]]的宗派都不認為有在地上對人[[封聖|封聖]]的必要，因為他們認為不合聖經的教訓。一般來說，基督宗教的圣人被認為能直接与[[上帝|上帝]]相连，可以带領別人到上帝的道路。

天主教徒稱聖人為[[聖品|聖品]]。

===東正教===

其有獨特的聖人觀，其中最奇特的是恭敬[[聖愚|聖愚]]。

=== 向圣人祷告 ===
基督宗教各教派都诵念的[[宗徒信经|宗徒信经]]（[[使徒信经|使徒信经]]）里有“我信諸聖相通功/我信聖徒相通”一句，指出可以通过圣人代祷，而天主教教友除了直接向天主祈祷外，也会向圣人请求代祷，并认为这是天主无限慈爱的表示，无减天主的权威。但部分[[基督新教|基督新教]]教派的信徒認為聖經已經明示任何人均可直接向主祈禱，擺脫任何有形無形的關係束縛。

== 伊斯兰教聖人 ==
伊斯兰教的学者承认聖人，对他们的[[墓|墓]]和神祠的崇拜在伊斯兰教中是非常普遍的。此崇拜存在於[[伊斯兰世界|伊斯兰世界]]所有地区都有，包括非常教条化的[[阿拉伯半岛|阿拉伯半岛]]。伊斯兰教相信聖人有直接与真神[[阿拉|阿拉]]溝通的能力，因此聖人可以施行[[奇迹|奇迹]]和[[祝福|祝福]]信徒。在大多数[[穆斯林国家|穆斯林国家]]中，都存在与聖人有关的[[节日|节日]]。在节日中，信徒會崇拜当地的聖人以获得聖人的祝福。不過，聖人的祝福與[[先知|先知]]的祝福是不同的。

在民间的伊斯兰教中，聖人是一个非常重要的组成部分。尤其在伊斯兰神秘派的[[蘇菲派|苏菲主义]]中聖人占重要地位。苏菲主义中，有不同的专门给追随聖人的学者的教训。与基督教的聖人崇拜不同，苏菲主义中没有对崇拜聖人的物件和奉献给聖人的仪式。

== 桑特里亚教聖人 ==
[[天主教|天主教]]中对聖人的崇拜是古巴桑特里亚教的基础。在桑特里亚教中，圣人与[[约鲁巴神话|约鲁巴神话]]中的[[神|神]]是一体，在[[教堂|教堂]]中他们受到圣人的崇拜。在桑特里亚教的节日上，他们被崇拜为神。然此種說法，就天主教有聖人傳統遠早於天主教傳入古巴，而普遍不被相信。

==印度教==

主要是表現在對於信史時代前的神話的英雄，即[[毗濕奴|毗濕奴]]的化身，如兩大史詩[[羅摩衍那|羅摩衍那]]和[[摩訶婆羅多|摩訶婆羅多]]的英雄[[羅摩|羅摩]]和[[黑天|黑天]]等。另一聖人的類型是指較出世的[[仙人_(印度神話)|仙人 (印度神話)]]。在信史時代則推崇著名的哲學家和英雄，如[[商羯羅|商羯羅]]和聖雄[[甘地|甘地]]等。

==[[神道|神道]]==

主要是供奉歷代[[天皇|天皇]]名君如[[神武天皇|神武天皇]]或[[明治天皇|明治天皇]]，'''名將'''如[[織田信長|織田信長]]、[[豐臣秀吉|豐臣秀吉]]或[[德川家康|德川家康]]，'''賢臣'''如[[聖德太子|聖德太子]]、[[菅原道真|菅原道真]]等， 各地都建有其[[神社|神社]]，也有些是地方豪族的傳說的始祖和神話的英雄，最著名有[[日本武尊|日本武尊]]。
