'''聖露西亞'''（{{lang-en|Saint Lucia}}）是東[[加勒比海|加勒比海]]鄰近[[大西洋|大西洋]]的[[島國|島國]]。它在[[聖文森及格瑞那丁|聖文森及格瑞那丁]]之北和[[法屬馬丁尼克|法屬馬丁尼克]]之南，是[[小安地列斯群島|小安地列斯群島]]的一部份。首都为[[卡斯特里|卡斯特里]]。

== 历史 ==
{{main|聖露西亞历史}}
* 1639年，[[英国|英国]]人入侵该岛。
* 1651年，[[法国|法国]]人占领。
* 1814年，根据《[[1814年巴黎條約|1814年巴黎條約]]》正式将该岛划为英国[[殖民地|殖民地]]。
* 1979年2月22日，宣布独立，为[[英联邦|-{zh-hans:英联邦;zh-hk:英聯邦;zh-tw:大英國協;}-]]成员国。

== 政治 ==
{{main|聖露西亞政治}}
康普頓曾經創立[[聯合工人黨|聯合工人黨]]並領導聖露西亞邁向獨立。[[聖露西亞議會|聖露西亞議會]]為兩院制。為該國的立法機關。

== 地理 ==
{{main|圣卢西亚地理}}
圣卢西亚的火山島比許多其他的加勒比海的島有更多山，最高峰峻峭的山為[[吉米山|吉米山]]（Mount Gimie），海拔950米。富矿泉和地热资源。热带气候，年降水量沿海1400毫米，内部高地3500毫米以上。

== 經濟 ==
{{main|聖露西亞經濟}}
拉丁美洲香蕉的在歐盟進口政策改變和其他競爭增加使聖露西亞經濟多元化發展成為逐漸重要的議題。聖露西亞已經能夠吸引外國生意和投資，尤其銀行業和觀光業是主要收入來源。製造業的部門是在東加勒比海的區域中最不同的，而政府正在嘗試恢復繁榮。
