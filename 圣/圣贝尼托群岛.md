'''圣贝尼托群岛'''位于[[墨西哥|墨西哥]][[下加利福尼亚州|下加利福尼亚州]]西海岸附近的[[太平洋|太平洋]]上，距离[[塞德羅斯島|塞德罗斯岛]]25公里。 它们归属于[[恩森那达_(墨西哥自治区)|恩森那达自治区]]中的[[塞德罗斯自治区|塞德罗斯自治区]]。

该群岛由三个贫瘠的岛屿组成，总面积为3.899平方公里，四周环绕着岩石和海藻。2001年的一次[[人口普查|人口普查]]记录了西贝尼托岛的两名人口，其他岛屿则无人居住。<ref name=INEGI/>

== 地理和生态 ==
'''''西贝尼托岛'''''，面积为2.6平方公里。是该群岛最西部和最大的岛屿。 在它的中心附近有一个高600英尺（约202米）的土堆，看上去就像一个高地。岛上有一个4米高的[[灯塔|<font color="#0645ad">灯塔</font>]]，位于岛的南部。另一座高度为17米的灯塔，矗立在该岛的西北端附近。 两个陡峭的岩石则位于西贝尼托岛以西1.6千米处。<ref name = nga2004/>

'''''中贝尼托岛'''''（面积为0.4平方公里，位于{{coord|28|18|30|N|115|33|54|W|scale:100000}}）和'''''东贝尼托岛'''''（面积为0.9平方公里，位于{{coord|28|18|03|N|115|32|25|W|scale:100000}}）在西贝尼托岛东部。两岛被一条深200英尺（约65米）的水道分隔。<ref name = nga2004 /> 东贝尼托岛的特点是岛上有四个界线分明的山丘，最高达421英尺（约140米）。中贝尼托岛地势平坦，在其东端有一座高82英尺（约25米）的小山。

岛上的植被主要由低矮的[[灌木|灌木]]和[[草本植物|草本植物]]，以及一些大型[[仙人掌|仙人掌]](比如''Cylindropuntia prolifera''、''C. ramosissima''<ref name = cmicd2007/>).几乎没有陆地动物，也没有本土的哺乳动物。由于岛上环境未被破坏，鸟类种类非常丰富，尤其是在繁殖期的时候，Cassin's auklet （鸟类名）的数量会猛增。

=== 特有物种 ===
由于圣贝尼托群岛位置较为偏远，在当地[[进化|进化]]出了一些[[特有种|特有种]]，以下是它们的[[分類單元|分类]]：

'''动物:'''<ref name=Thayer/>
* 圣贝尼托侧斑鬣蜥，学名''Uta stansburiana stellata''
* 圣贝尼托[[家朱雀|家朱雀]]，学名''Carpodacus mexicanus mcgregori'''''（已[[灭绝|灭绝]]）'''
灭绝于20世纪40年代，主要是由于过度捕猎与标本收藏。
* 圣贝尼托麻雀，学名''Passerculus rostratus sanctorum''（或''Passerculus sandwichensis sanctorum''）
'''植物:'''<ref name = cmicd2007/>
* ''Cryptantha patula'' – 只存在于西贝尼托岛
* ''Dudleya linearis'' – 只存在于西贝尼托岛
* ''Hemizonia streetsii'' – 只存在于东贝尼托岛
* ''Lavatera venosa'' – 圣贝尼托群岛所有岛屿
* ''Mammillaria neopalmeri'' – 只存在于东贝尼托岛
* ''Senecio benedictus'' – 只存在于西贝尼托岛
