《'''圣天门口'''》，2011年上映于[[中国大陆|中国大陆]]的战争剧，讲述了1920年代初到1950年代末天门口镇两个家族与战争的故事，导演张黎、刘淼淼，主演段奕宏、袁文康、宋佳、黄志忠、练束梅等。2011年9月13日湖北电视台新闻综合频道首播<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2011-09-13/00453414096.shtml |title=《圣天门口》首播 段奕宏宋佳演绎另类革命情感|accessdate=2013年6月11日 |publisher=新浪娱乐  |work= |language =zh-cn}}</ref>, 2012年10月22日安徽卫视上星<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2012-10-22/09353768658.shtml |title=《圣天门口》风起云涌 安徽卫视今晚首播|accessdate=2012年10月22日 |publisher= 新浪娱乐  |work= |language =zh-cn}}</ref>。

== 剧情 ==
[[大别山|大别山]]区天门口镇的雪、杭两个家族世代不和，雪家崇尚文化教育，杭家崇尚武力健身，但是到了1920年代，杭姓家族的杭九枫却和雪姓家族的儿媳妇阿彩是情人，两人经常偷偷约会。<ref name="man " >{{cite web |url=http://ent.sina.com.cn/v/2011-06-08/17373328716.shtml |title= 资料：电视剧《圣天门口》故事梗概|accessdate=2011年6月8日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

[[大日本帝国|大日本帝国]]的入侵打破了天门口镇的平静日子，许多世代居住在此的人们开始为了逃避战乱而奔走，雪家和杭家因为根基深固、家族庞大无法逃走，只得坚守下来，一直败落。随后，[[国共内战|国共内战]]又席卷此地，原本繁华的天门口镇变得冷清和贫破，雪家和杭家相互扶持才度过这战乱岁月。<ref name="man " />

中华人民共和国建国后，阿彩去了省城，而杭九枫留在镇上，大饥荒的到来，使得天门口镇人饥饿不已，眼见乡亲们无法生存，杭九枫开仓放粮，拯救了饱受饥饿的人民。<ref name="man " />

== 花絮 ==
电视剧耗资5000万[[人民币|人民币]]。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2012-10-24/18003771298.shtml |title=《圣天门口》首秀获赞 “金牌导演”再推巨制 |accessdate=2012年10月24日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

刘醒龙原著主要是[[魔幻现实主义|魔幻现实主义]]元素，而电视剧是诗意[[浪漫主义|浪漫主义]]元素。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2012-10-23/23103770386.shtml |title=独家策划：《圣天门口》革命终极追问 |accessdate=2012年10月23日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

电视剧最大的特征是[[意识流|意识流]]电视语言的大量使用。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2011-06-08/10453328205.shtml |title=电视剧《圣天门口》：诗意重述中国革命史 |accessdate= 2011年6月8日|publisher= 新浪网|work= |language =zh-cn}}</ref>
