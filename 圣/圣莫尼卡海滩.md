'''圣莫尼卡海滩'''（Santa Monica State Beach）位于[[加州|加州]][[洛杉矶县|洛杉矶县]]，由[[圣莫尼卡|圣莫尼卡]]市管理<ref name=CSP>{{cite web|title=California State Parks: Santa Monica State Beach|url=http://www.parks.ca.gov/?page_id=624}}</ref>。

圣莫尼卡海滩沿着太平洋海岸公路，两英里长，有野餐区，商店，和码头，<ref name=CSP/> 以及救生员站，休闲餐馆，设备租赁点，健身活动设施，自行车道，木质小径，供在温暖的日子前往沙滩。<ref name=SM>{{cite web|title=Santa Monica Beach|url=http://www.santamonica.com/index.php/beach.html}}</ref>游客活动包括排球，篮球和跑道<ref name=CSP/>。

圣塔莫尼卡码头建于1909年，位于科罗拉多大道，有著名的拱门。这里有一处国家历史地标 -– 1922年的[[:en:Santa_Monica_Looff_Hippodrome|圣莫尼卡赛马场]]<ref name=SM/>。

[[:en:Muscle_Beach|肌肉海滩]]紧靠排球场以南，有双杠，吊环等运动设施<ref name=SM/>。

码头排球场以南几步是一个国际象棋公园。人行道上绘制公共棋桌和一个棋盘。<ref name=SM/>

Palisades公园位于圣莫尼卡著名的砂岩峭壁之上，提供了一个制高点，可以看到圣莫尼卡海滩和太平洋。<ref>{{cite web|title=LAMountains.com: Santa Monica State Beach|url=http://www.lamountains.com/parks.asp?parkid=124|deadurl=yes|archiveurl=https://web.archive.org/web/20080405145103/http://www.lamountains.com/parks.asp?parkid=124|archivedate=2008-04-05}}</ref>

圣莫尼卡海滩是[[猫头鹰之城|猫头鹰之城]]的专辑伞海滩（Umbrella Beach）的封面。
