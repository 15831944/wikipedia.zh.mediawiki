{{vgname|圣域3|en=Sacred 3}}是一款于2014年发行的[[砍杀游戏|砍杀游戏]]型的[[清版动作游戏|清版动作游戏]]。在2010年的[[Gamescom|科隆游戏展]]上，[[Deep_Silver|Deep Silver]]宣布从[[Ascaron|Ascaron]]处获得了''圣域3''的授权，且游戏将在2011-2012年时推出<ref>{{Cite web |url = http://www.neoseeker.com/news/14681-deep-silver-announces-risen-2-sacred-3/ |title = Deep Silver announces Risen 2, Sacred 3 |author = Sean Ridgeley |date = 2010-08-19 |publisher = neoseeker |language = en |accessdate = 2018-02-17 }}</ref>。最终本作在2014年8月才正式推出。由于本作舍弃了[[圣域系列|圣域系列]]原有的游戏风格而招致差评<ref>{{Cite web |url = http://play.163.com/17/0803/17/CQUAM7BD00318P1K.html |title = 游戏史上的今天：爽快有余内涵不足《圣域3》 |author =  |date = 2017-08-03 |publisher = 网易 |language = zh-cn |accessdate = 2018-02-17 }}</ref>，一位参与了本作开发的程序员表示，是Deep Silver的市场部门导致了本作的风格与[[圣域系列|圣域系列]]相异。<ref>{{Cite web |url = http://www.3dmgame.com/news/201408/3353361.html |title = 新作实在不咋地 前《圣域3》开发成员为游戏致歉 |author =  |date = 2014-08-19 |publisher = 3DM |language = zh-cn |accessdate = 2018-02-17 }}</ref>

==游戏系统==
单人模式中，玩家要从战士（男）、弓箭手（男）、天使圣骑士（女）、长枪兵（女）中选择一个作为自己操控的[[玩家角色|角色]]。

本作的线上模式支持最多4人同时进行游玩，每人都有自己独特的主/被动技能。
