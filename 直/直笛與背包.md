《'''直笛與背包'''》（{{lang|ja|リコーダーとランドセル}}），又譯'''竖笛与双肩书包'''、'''直笛與雙背帶書包'''，是[[日本|日本]][[漫畫家|漫畫家]]{{link-ja|東屋めめ|東屋めめ}}的[[四格漫畫|四格漫畫]]作品。

== 概要 ==
在『[[まんがくらぶオリジナル|まんがくらぶオリジナル]]』（[[竹書房|竹書房]]）2007年10月号掲載後，2008年2月号開始連載中。還有在『[[まんがライフオリジナル|まんがライフオリジナル]]』（同）2010年8月号掲載後，2011年5月号開始連載中。至2012年12月，[[單行本|單行本]]已發行5卷。
以身高不像小學生的敦志為中心的搞笑四格漫畫。

并且，在同一社的其他杂志『[[まんがライフMOMO|まんがライフMOMO]]』中也时常会揭载有以姐姐的宫川敦美视角描写的外传『リコーダーとランドセル あつみ編』。

在『まんがライフオリジナル』2011年10月号發表[[電視動畫|電視動畫]]化計劃後，於2012年1月到3月播放第1季『直笛與背包 DO♪』，同年4月到6月播放第2季『直笛與背包 RE♪』。另於2013年7月起播放第三季『直笛與背包 MI☆』<ref name=mantan-web>{{Cite web|publisher=MANTANWEB（まんたんウェブ）|url=http://mantan-web.jp/2013/03/30/20130329dog00m200025000c.html|script-title=ja:リコーダーとランドセル ： アニメ第3期「ミ☆」7月スタート|accessdate=2013-03-30|language=ja}}</ref>。

== 簡介 ==

宮川敦是一個身高 180 公分、完全不像小孩子的小學五年級生；而他的姊姊宮川敦美，則是一個身高只有 137 公分、看起來就是兒童的高中二年級生，作品圍繞著這對姐弟，描繪在他們生活周遭發生的各種趣事。

== 電視動畫 ==
約3分的短編動畫，於2012年1月以『'''リコーダーとランドセル ド♪'''』的標題經[[埼玉電視台|埼玉電視台]]、[[京都放送|KBS京都]]和[[niconico動画|niconico動画]]播放，完結後便立即在2012年4月播放第2季『'''リコーダーとランドセル レ♪'''』。第3季『'''リコーダーとランドセル ミ☆'''』在2013年7月起開始播放<ref name=mantan-web/>。第3期旁白為金谷英孝。第3期初回放送由いまざきいつき與置鮎龍太郎主持特別節目。
接續之前播放『[[沉默的森田同學|無口的森田同學]]』時的狀況，每集完結後都會播放大約30秒長，宣傳用的『{{link-ja|柴丸子小姐|しばいぬ子さん}}情報Corner』（由[[佐々木未来|佐々木未来]]聲演）。
{{TransH}}
第11話では[[アニたま|アニたま]]が乗っ取られた仕返しにたまタン（声 - [[橋本まい|橋本まい]]）が登場している。第3期はAT-Xのみ放送終了後にミニ番組『いまざきいつきvs置鮎龍太郎のOne Minute Challenge!!（ワンミニッツチャレンジ）』を放送。
{{TransF}}

前2季是[[竹書房|竹書房]]完全自社製作的，但第3季則改為用[[製作委員會方式|製作委員會方式]]製作。另外也有一些工作人員被更換。
