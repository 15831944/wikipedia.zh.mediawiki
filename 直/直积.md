在[[數學|數學]]中，經常定義已知對象的'''直積'''（direct product）來給出新對象。例子有集合的乘積（參見[[笛卡爾積|笛卡爾積]]），群的乘積（下面描述），[[環的乘積|環的乘積]]和其他[[代數結構|代數結構]]的乘積。[[積空間|拓撲空間的乘積]]是另一個例子。

== 例子 ==

* 如果我們認 <math>\mathbb{R}</math> 為實數的[[集合|集合]]，則直積 <math>\mathbb{R}\times \mathbb{R}</math> 完全就是[[笛卡爾積|笛卡爾積]] <math>\{ (x,y) | x,y \in \mathbb{R} \}</math>。

* 如果我們認 <math>\mathbb{R}</math> 為實數集在加法下的[[群|群]]，則直積 <math>\mathbb{R}\times \mathbb{R}</math> 仍構成自 <math>\{ (x,y) | x,y \in \mathbb{R} \}</math>。和上個例子的不同是 <math>\mathbb{R}\times \mathbb{R}</math> 現在是群。我們必須定義如何做它們的元素的加法。這個定義為 <math>(a,b) + (c,d) = (a+c, b+d) \, </math>。

* 如果我們認 <math>\mathbb{R}</math> 為實數集的[[环_(代数)|環]]，則直積 <math>\mathbb{R}\times \mathbb{R}</math> 仍構成自 <math>\{ (x,y) | x,y \in \mathbb{R} \}</math>。要使它成為環，我們必須定義它們的元素的運算，加法定義為 <math>(a,b) + (c,d) = (a+c, b+d) \, </math>，而乘法定義為 <math>(a,b) (c,d) = (ac, bd) \, </math>。

* 但是如果我們認 <math>\mathbb{R}</math> 為實數集的[[域_(數學)|域]]，則直積 <math>\mathbb{R}\times \mathbb{R}</math> 不存在！以類似上例的方式定義 <math>\{ (x,y) | x,y \in \mathbb{R} \}</math> 的结果不是一個域，因為元素 <math>(1,0)</math> 不存在乘法逆元。

以類似的方式，我們可以談論多於兩個對象的乘積，比如 <math>\mathbb{R} \times \mathbb{R} \times \mathbb{R} \times \mathbb{R}</math>。我們甚至可以談論無限多個對象的乘積比如 <math>\mathbb{R} \times \mathbb{R} \times \mathbb{R} \times \dotsb</math>。

== 群直積 ==

在[[群|群論]]中可以定義兩個群 (''G'', *) 和 (''H'', o) 的直積，指示為 ''G'' × ''H''。對於寫為加法的[[阿貝爾群|阿貝爾群]]，它也可以叫做兩個群的[[群的直和|直和]]，指示為 <math>G \oplus H</math>。

它定義為如下:
* 新群的元素的[[集合|集合]]是 ''G'' 和 ''H'' 的元素的集合的[[笛卡爾積|笛卡爾積]]，即 {(''g'', ''h''): ''g'' ∈ ''G'', ''h'' ∈ ''H''}； 
* 逐元素定義在這些元素上的運算: 
:(''g'', ''h'') × (''g' '', ''h' '') = (''g'' * ''g' '', ''h'' o ''h' '')</center> 
(注意運算 * 可以同於 o。)

這個構造給出了新群。它有[[同構|同構]]於 ''G'' (構成自形如 (''g'', 1) 的元素)的一個[[正規子群|正規子群]]，和同構於 ''H'' (構成自元素 (1, ''h''))的一個正規子群。

逆命題也成立，有下列識別定理: 如果群 ''K'' 包含兩個正規子群 ''G'' 和 ''H''，使得 ''K''= ''GH'' 并且 ''G'' 和 ''H'' 的交集只包含單位元，則 ''K'' 同构于 ''G'' × ''H''。将其中一个正规子群条件弱化为一般子群则給出[[半直積|半直積]]。

作為一個例子，選取 ''G'' 和 ''H'' 是唯一(不別同構之異) 2 階群 ''C''<sub>2</sub> 的兩個復本: 即 {1, ''a''} 和 {1, ''b''}。則 ''C''<sub>2</sub>×''C''<sub>2</sub> = {(1,1), (1,''b''), (''a'',1), (''a'',''b'')}，帶有逐元素運算。例如，(1,''b'')*(''a'',1) = (1*''a'', ''b''*1) = (''a'',''b'')，而 (1,''b'')*(1,''b'') = (1,''b''<sup>2</sup>) = (1,1)。

通過直積，我們得到一些自然[[群同態|群同態]]: 投影映射 
:<math>\pi_1 \colon G \times H \to G\quad \mathrm{by} \quad \pi_1(g, h) = g</math>，
:<math>\pi_2 \colon G \times H \to H\quad \mathrm{by} \quad \pi_2(g, h) = h</math>
叫做'''坐標函數'''。

還有，在直積上的所有同態 ''f'' 都完全決定自它的分量(component)函數 <math>f_i = \pi_i \circ f</math>。

對于任何群 (''G'', *)，和任何整數 ''n'' ≥ 0，多次應用直積得到所有 ''n''-[[元組|元組]]的群  ''G''<sup>''n''</sup> (''n''=0 時是平凡群)。例如:
*'''Z'''<sup>''n''</sup>。
*'''R'''<sup>''n''</sup> (帶有額外的[[向量空間|向量空間]]結構就叫做[[歐幾里得空間|歐幾里得空間]]，見后描述)。

== 模的直積 ==

[[模|模]]的直積(不要混淆於[[張量積|張量積]])非常類似於上述群直積的定義，使用[[笛卡爾積|笛卡爾積]]帶有逐分量的加法運算，和只分布在所有分量上的標量乘法運算。開始於 '''R''' 我們得到[[歐幾里得空間|歐幾里得空間]] '''R'''<sup>''n''</sup>，它是實 ''n''-維向量空間的原型例子。'''R'''<sup>''m''</sup> 和 '''R'''<sup>''n''</sup> 的直積是 '''R'''<sup>''m'' + ''n''</sup>。

注意有限索引 <math>\prod_{i=1}^n X_i </math> 的直積同一於[[模的直和|直和]] <math>\bigoplus_{i=1}^n X_i </math>。直和與直積只對無限索引有區別，這里直和的元素對於除了對于有限多個之外所有的項目是零。它們是對偶的: 直和是[[上積|上積]]，而直積是乘積。

例如，考慮 <math>X=\prod_{i=1}^\infty \mathcal{R} </math> 和 <math>Y=\bigoplus_{i=1}^\infty \mathcal{R}</math>，實數的無限直積和直和。在 ''Y'' 中只有有著有限多個非零元素的序列。例如，''(1,0,0,0,...)'' 在 ''Y'' 中但 ''(1,1,1,1,...)'' 不在。這兩種序列都在直積 ''X'' 中；事實上，''Y'' 是 ''X'' 的真子集(也就是 ''Y''⊂''X'')。

== 拓撲空間直積 ==

[[拓撲空間|拓撲空間]]的搜集 ''X<sub>i</sub>'' 即對於 ''i'' 在 ''I'' 中的某個索引集合的直積，再次利用了笛卡爾積

:<math>\prod_{i \in I} X_i </math>

定義[[拓撲|拓撲]]是有些技巧的。對于有限多個因子這是明顯和自然的事情: 簡單的選取開集構成的[[基_(拓撲學)|基]]為來自每個因子的開子集的所有笛卡爾積的搜集:

:<math>\mathcal B = \{ U_1 \times \cdots \times U_n\ |\ U_i\ \mathrm{open\ in}\ X_i \}</math>

這個拓撲叫做[[積空間|乘積拓撲]]。例如，直接通過 '''R''' 的開集們(開區間的不交并)定義在 '''R'''<sup>2</sup> 上的乘積拓撲，這個拓撲的基由在平面上的開矩形的所有不交并構成(明顯的它一致於平常的[[度量空間|度量]]拓撲)。

無限乘積的拓撲就有些曲折了，要能夠確使所有投影映射連續，并確使所有到乘積中的函數連續當且僅當所有它的分量函數是連續的(就是滿足乘積范疇定義: 這里的態射是連續函數): 我們同上面一樣的選取的開集構成的基圍來自每個因子的開子集的所有笛卡爾積的搜集，但帶有除了有限多個開子集之外所有都是整個因子的限制條件:

:<math>\mathcal B = \left\{ \prod_{i \in I} U_i\ |\ (\exists j_1,\ldots,j_n)(U_{j_i}\ \mathrm{open\ in}\ X_{j_i})\ \mathrm{and}\ (\forall i \neq j_1,\ldots,j_n)(U_i = X_i) \right\}</math>

在這種情況下更自然可靠的拓撲將是如上那樣選取無限多個開子集的乘積，而這產生了有些意思的拓撲，即[[盒拓撲|盒拓撲]]，但是不難找到其乘積函數不是連續的連續分量函數叢(例子請參見[[盒拓撲|盒拓撲]]的條目)。使這種曲折成為必須的問題最終根源於在拓撲定義中開集的交集對無限多集合不保證是開集的事實。

乘積(帶有乘積拓撲)關於保持它們因子的性質是良好的；例如，豪斯多夫空間的乘積是豪斯多夫空間；連通空間的乘積是連通空間，而緊緻空間的乘積是緊緻空間。最后一個也叫做[[吉洪诺夫定理|吉洪诺夫定理]]，它是[[選擇公理|選擇公理]]的另一個等價形式。

更多的形式和等價公式請參見單獨條目[[積空間|乘積拓撲]]。

== 二元關係的直積 ==

在帶有[[二元關係|二元關係]] ''R'' 和 ''S'' 的兩個集合上的笛卡爾積上，定義 (''a'', ''b'') T (''c'', ''d'') 為 ''a'' ''R'' ''c'' 并且 ''b'' ''S'' ''d''。如果 ''R'' 和 ''S'' 都是[[自反關係|自反]]的、[[反自反關係|反自反]]的、[[傳遞關係|傳遞]]的、[[對稱關係|對稱]]的或[[反對稱關係|反對稱]]的，則 ''T'' 有同樣性質。<ref>[http://cr.yp.to/2005-261/bender1/EO.pdf Equivalence and Order]</ref> 組合各性質，可得出這還適用於作為[[預序|預序]]和作為[[等價關係|等價關係]]情況。但是如果 ''R'' 和 ''S'' 是[[完全關係|完全關係]]，''T'' 一般不是。
