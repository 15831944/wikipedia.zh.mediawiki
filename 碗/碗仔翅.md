'''碗仔翅'''是一種[[香港|香港]]街頭仿[[魚翅|魚翅]][[羹|羹]]小吃，其材料以[[粉絲|粉絲]]為主，以澱粉將湯煮至濃稠，並加入老抽和生抽成棕色，冬菇絲和雞肉絲常被加入一起烹調，使湯羹的口感更為充實。進食碗仔翅時，可按個人喜好加入[[麻油|麻油]]、[[浙醋|浙醋]]、白[[胡椒粉|胡椒粉]]或[[辣椒油|辣椒油]]調味<ref>[http://www.christinesrecipes.com/2008/07/blog-post.html 碗仔翅 - 香港街頭小食]，簡易食譜，2008-7-9</ref>。

== 起源 ==
約[[香港1950年代|1950]]至[[香港1960年代|1960年代]]起，香港街頭出現了一批專門出售這類平民仿翅的小販<ref>[http://travel.cnn.com/hong-kong/eat/imitation-sharks-fin-149553 老闆娘的心機碗仔翅]，CNN 旅遊，2011-5-31</ref>，部分小販還會同時售賣[[生菜魚肉|生菜魚肉]]，不少人買碗仔翅時，喜歡加入魚肉同吃。碗仔翅本身沒有魚翅成份，但其外觀與香港[[酒樓|酒樓]]或酒家飲宴時供應的魚翅羹相似，並以小碗盛載，因而稱為「碗仔翅」。過往一些小販亦會出售碗仔翅拌[[通心粉|通心粉]]，能增加裹腹感，可作為簡單的午餐，但現在已較為少見。

碗仔翅的湯底主要由清水、[[醬油|醬油]]、[[味精|味精]]和[[澱粉|澱粉]]調配而成。由於小販和熟食檔出售的碗仔翅常會加入味精調味，因此即使碗仔翅是受歡迎的小吃，但仍被視為不健康的食物，一般家庭也很少會在家中烹調碗仔翅作為家常湯水，只會從食肆購買作為小吃。1990年代後期，香港有一些[[酒樓|酒樓]]開始出售加入香菇絲、蛋花、以至[[金華火腿|金華火腿]]、[[豬皮|豬皮]]、[[魚肚|魚肚]]的碗仔翅<ref>[http://news.stheadline.com/dailynews/headline_news_detail_columnist.asp?id=186928&section_name=wtt&kw=257 升呢花膠碗仔翅]，頭條日報，2012-4-14</ref>，也有部分[[港式快餐|快餐店]]把碗仔翅作為[[下午茶|下午茶]]時段供應的食品。

== 做法 ==
將100克雞胸肉蒸熟，用手把肉撕開；把1公升上湯煮開，加入雞肉絲、30克已浸軟粉絲、20克已浸軟[[冬菇|冬菇]]條，最後加老抽、澱粉、油攪勻。吃時伴以淅醋、麻油、白胡椒粉、辣椒油等<ref>[http://www.tfysw.com/2011/0930/29318.html 碗仔翅]，美食城，2011-12-10</ref>。

== 名稱 ==
碗仔翅是香港常見的小吃，大部分居民也理解「碗仔翅」只是用廉價的粉絲代替昂貴的魚翅，並不會有魚翅成分<ref>[http://www.cgcc.org.hk/gb/chamber/bulletin/files/Bulletin_1315369739.17075_P.40-47.pdf 地道美食吃出香港情懷]，商薈，2011年9月</ref>。從2000年代起，香港社會開始關注保護[[海洋|海洋]][[生態|生態]]，避免濫捕[[鯊魚|鯊魚]]做魚翅，提倡減少甚至停止食用魚翅，所以碗仔翅成為大眾感受魚翅湯味道的代替品。
