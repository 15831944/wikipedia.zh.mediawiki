'''菲利普斯学院'''（Phillips Academy）位于[[美国|美国]][[马萨诸塞州|马萨诸塞州]]埃塞克斯县[[安多佛|安多佛]]（又译“安多福”或“安道夫”）。该学院是一所私立高中，是[[十校联盟|十校联盟]]成员之一。該校现任董事长为[[華裔|華裔]]美籍[[投资|投资]]家[[唐騮千|唐騮千]]，他毕业于該校，后來進入[[耶鲁大学|耶鲁大学]]。<ref>{{Cite web|url=http://www.andover.edu|title=Big. Blue. Family.   Phillips Academy|accessdate=2018-10-13|author=|date=2018-10-13|last=Academy|first=Phillips|work=Phillips Academy|publisher=|language=en-us}}</ref>傳統上，菲利普斯学院的学生會進入[[耶鲁大学|耶鲁大学]]，而她的競爭對手[[菲利普斯埃克塞特学院|菲利普斯埃克塞特学院]]則會把學生送到[[哈佛大學|哈佛大學]]。

== 历史 ==
[[File:Samuel_Phillips_Hall.jpg|thumb]]

学校在1778年由小[[塞缪尔·菲利普斯|塞缪尔·菲利普斯]]建立，是[[美国|美国]]最早建立且现今还在运营的私立寄宿高中。美国第一任总统[[乔治·华盛顿|乔治·华盛顿]]曾经在18世纪末参观过学校，华盛顿后将自己的侄子送到菲利普斯学院学习。美国革命先驱、银匠[[保罗·列维尔|保罗·列维尔]]为学校设计了校徽。

小塞缪尔之叔于1781年在新罕布什尔州的[[埃克塞特_(新罕布什尔州)|埃克塞特]]建立了[[菲利普斯埃克塞特学院|菲利普斯埃克塞特学院]]。'''菲利普斯学院'''与[[菲利普斯埃克塞特学院|菲利普斯埃克塞特学院]]并称为美国最知名的两所私立高中，两所学校间的激烈竞争已经有两百多年的历史。

1973年，学院与毗邻的艾波特学院（Abbot Academy）合并后，学校变为男女合校。艾波特学院始建于1829年，是[[新英格兰|新英格兰]]地区第一所正式的女子学校。两校合并后，新学校仍保留菲利普斯学院的传统。

== 资金 ==

菲利普斯学院是全美最富有的私人教育机构之一。根据08年5月1日公布的数据，学校共计接收各类捐款8.2亿美元[http://www.nytimes.com/2008/01/26/business/26prep.html?_r=1]。

08-09学年度，根据董事会的决定，学院决定开始执行资金需求盲性录取政策（need-blind admissions），从而与[[菲利普斯埃克塞特学院|菲利普斯埃克塞特学院]]一道成为美国唯一两所奉行这种政策的学校。09-10学年度，在学校面临巨大财政压力的情况下，董事会仍决定采用此录取方式，从而成为美国唯一一所采用这种录取政策的高中。这种政策允许招生办把资金申请材料和普通入学申请材料分开审阅，面试官和材料审阅人不查看申请人的资金请求。因此，申请资金援助的学生不会因为资金上的困难遭到学校的拒绝。目前，42%的在校学生拥有资金援助，09年援助总额为1460万美元[http://www.andover.edu]。

学校现任董事长为[[華裔|華裔]]美籍[[投资|投资]]家[[唐騮千|唐騮千]]。[http://www.andover.edu]

== 学术 ==
[[File:Graves_Phillips_Academy.JPG|thumb]]

学院2009年招生录取率仅为16%，是世界上录取率最低的几所高中之一。2008年，学院共收到完整的申请材料2386份，总计录取457人，其中亚太地区共19人，中国大陆地区最终入学4人。2009年，学校中国大陆地区仅录取了三名非中国籍的学生。

学生平均[[SAT|SAT]]分数为2076分，是全美平均成绩最高的学校之一。2008-2009学年度，截至2009年1月5日为止，已有266人中的128人被大学提前录取，占总人数的48%。[http://www.andover.edu]自1840年来，学校的传统是把学生送往[[耶鲁大学|耶鲁大学]]。近年来，随着学生数量的日益增多，学生流向开始出现多元化。[[哈佛大学|哈佛大学]]、[[普林斯顿大学|普林斯顿大学]]、[[宾州大学|宾州大学]]、[[斯坦福大学|斯坦福大学]]、[[哥伦比亚大学|哥伦比亚大学]]成为学院主要的学生去向。

学校18个院系提供300门课程，共包括两部分：第一部分是核心课程，如数学100/150/210/220/310/320、美术225A/B/C、音乐-225/235、英语100/200/300、历史100/200/300等；第二部分是AP和大学课程、自主研究项目等延伸类课程，如数学-650、化学-550、音乐-300等。学校教学设施先进，占地两平方公里的校园包括美国艾迪生美术馆，罗伯特考古博物馆，奥利弗温德霍姆斯图书馆，全校三个图书馆共计藏书17万册。

== 体育 ==

体育是学校的传统强项。其足球队、摔跤队、[[橄榄球|橄榄球]]队、游泳队和长跑队在[[新英格兰|新英格兰]]地区一直名列前茅。其与[[菲利普斯埃克塞特学院|菲利普斯埃克塞特学院]]与1878年起便开始举行一年一度的橄榄球争霸赛，至今已举行了128次，是美国顶尖高中间的最古老的对抗赛事。

== 入学要求 ==
[[File:Phillips_Academy,_Andover,_MA_-_Samuel_Phillips_Hall.JPG|thumb]]

该校1105名学生来自美国的五十个州或海外领地和全世界的34个国家。2008-2009学年度，学校国际学生比例为7%，主要包括[[印度|印度]]、[[台湾|台湾]]、[[韩国|韩国]]和[[中国|中国]]学生，其中中国大陆地区12人，香港地区8人。

'''菲利普斯学院'''在中国一直保持着很低的录取率。每年一、二月份都有在北京、上海、香港等地召开的面试。材料递交截至日期是二月一日，录取通知发送日期是三月十日。
