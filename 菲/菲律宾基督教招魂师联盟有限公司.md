'''菲律宾基督教招魂师联盟有限公司'''<ref>{{cite web |url=http://unionespiritista.weebly.com/ |title=Union Espiritista Cristiana de Filipinas, Inc}}</ref>是菲律宾的一个宗教组织，其下拥有上千个本地中心（教会），<ref>{{cite web |url=http://unionespiritista.weebly.com/region-1.html |title=Centers}}</ref>也被认为是菲律宾最大的唯灵论者（巫师、招魂师）组织。<ref>{{cite web|url=http://www.metamind.net/licaucoart.html|title=American Traces History}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>在[[加利福尼亚|加利福尼亚]]、[[加拿大|加拿大]]、[[香港|香港]]、[[新加坡|新加坡]]、[[希腊|希腊]]、[[迪拜|迪拜]]、[[夏威夷|夏威夷]]、[[德国|德国]]、[[意大利|意大利]]和[[俄罗斯|俄罗斯]]等地的菲律宾社区中设有下属本地中心。被称为“基督教（联盟）招魂师”的联盟成员也与{{le|国际灵师联合会|International Spiritualist Federation}}有关。<ref>{{cite web |url=http://www.theisf.com/ |title=International Spiritualist Federation}}</ref>该组织始于1905年2月19日，[[邦阿西楠省|邦阿西楠省]]的一伙巫师加入了[[马尼拉|马尼拉]]的另一伙打算组织一个集会以联合菲律宾所有招魂师。1909年2月19日，创始人执行了章程并将宗教协会命名为“菲律宾基督教招魂师联盟有限公司”。
