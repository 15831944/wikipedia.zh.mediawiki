'''菲尼斯顿起重机'''（Finnieston Crane）[[苏格兰|苏格兰]][[格拉斯哥|格拉斯哥]]市中心的一个废弃的[[起重机|巨型悬臂起重机]]，高{{convert|174|ft}}<ref name=cwh/>它的起重能力为175吨，在三分半钟内完成一个完整的旋转。<ref name=cwh/><ref name=tgs>{{cite web |url=http://www.theglasgowstory.com/image.php?inum=TGSE01353 |title=Finnieston Crane |publisher=TheGlasgowStory |accessdate=28 March 2014}}</ref>该起重机曾经用于将货物，特别是[[蒸汽机车|蒸汽机车]]，装载到在世界各地出口的船舶上。随着船坞在1969年关闭.<ref name=qd>{{cite web |url=http://canmore.rcahms.gov.uk/en/site/44157/details/glasgow+stobcross+queen+s+dock/ |title=Glasgow, Stobcross, Queen's Dock |publisher=CANMORE |accessdate=30 March 2014}}</ref>，它已不再使用，而保留作为城市的工程遗产的象征<ref name=p198>{{harvnb|Crawford|2013|p=198}}</ref>。

它是在[[克莱德河|克莱德河]]上的四个这样的起重机之一，第五个已在2007年拆除。全世界仅剩下十一架这样的巨型悬臂起重机<ref name=et>{{cite web |url=http://www.engineering-timelines.com/scripts/engineeringItem.asp?id=1219 |title=Titan Crane, Clydebank |publisher=engineering-timelines.com |accessdate=22 March 2014}}</ref><ref name=attwf/>。起重机可以在BBC太平洋码头的新闻广播背景下看到<ref name=attwf>{{cite web |url=http://gbarr.info/2011/06/01/and-the-there-were-four-titan-cranes-of-the-clyde/ |title=And then there were four: Titan Cranes of the Clyde |publisher=gbarr.info |accessdate=23 April 2014}}</ref><ref>{{cite news |url=http://www.bbc.co.uk/news/uk-scotland-25882564 |title=Reporting Scotland studio gets a new look |date=27 January 2014 |publisher=BBC News |accessdate=23 April 2014}}</ref>。

==历史==
皇后码头建立于1877年8月，占地{{convert|61|acre}}，用于从格拉斯哥中心出口货物。<ref name=qd/>现在的菲尼斯顿起重机是克莱德河上最后一个巨型悬臂起重机。<ref name=cwh>{{cite web |url=http://www.clydewaterfrontheritage.com/finniestoncrane.aspx |title=Finnieston Crane |publisher=Clyde Waterfront |accessdate=30 March 2014 |deadurl=yes |archiveurl=https://web.archive.org/web/20140426201654/http://www.clydewaterfrontheritage.com/finniestoncrane.aspx |archivedate=26 四月 2014 }}</ref>它建于1928年-1931年，1932年投入使用<ref name="p261">{{harvnb|Riddell|1979|p=261}}</ref>。
