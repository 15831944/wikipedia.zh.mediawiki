'''菲律賓眼鏡蛇'''（學名：''Naja philippinensis''，[[英語|英語]]：''Philippine Cobra''），是[[脊索動物門|脊索動物門]][[爬蟲綱|爬蟲綱]][[有鱗目|有鱗目]][[蛇亞目|蛇亞目]][[眼鏡蛇科|眼鏡蛇科]][[眼鏡蛇|眼鏡蛇屬]]的生物。具毒性，主要分布於[[菲律賓|菲律賓]][[呂宋|呂宋]]、[[民都洛|民都洛]]、[[卡坦端內斯省|卡坦端內斯省]]及[[馬斯巴特|馬斯巴特]]等[[島嶼|島嶼]]。在[[他加祿語|他加祿語]]中，菲律賓眼鏡蛇又稱「ulupong」。

== 特徵與習性 ==
菲律賓眼鏡蛇整體比例較其他蛇種短胖，約長1.7[[米_(單位)|米]]，而大部分身處民都洛島更有兩米之長。身體顏色會由年輕時的暗[[棕色|棕色]]，逐漸蛻為成熟期的淺棕色，喉頸位置有時會有一條黑色的長紋。[[頸部|頸部]]附近有二十數環的蛇鱗，軀體中部有21環蛇鱗，[[腹部|腹部]]有182至193環蛇鱗，尾部位置有36至49環蛇鱗。鱗紋分布有時並不平均。

== 分布及棲身地 ==
菲律賓眼鏡蛇主要分布於菲律賓，除了呂宋諸島之外，亦散見於其它群島；而卡拉綿群島及巴拉望島的蛇蹤紀錄，則尚待進一步的確認。菲律賓眼鏡蛇的棲身地大致為[[耕地|耕地]]、民眾聚居地及[[森林|密林]]之中。

== 飲食及繁殖 ==
菲律賓眼鏡蛇主要進食[[鼠|鼠]]類、[[蛙|蛙]]類及小型的[[哺乳類|哺乳類]]動物。雌蛇每次生產10至20枚蛇卵，並會作出60至70天的孵育。

== 毒性 ==
菲律賓眼鏡蛇最著名的是其劇毒。牠的毒素是所有[[眼鏡蛇|眼鏡蛇]]種中最強力的，其毒性主要影響中毒者的[[心臟|心臟]]機能及[[呼吸系統|呼吸系統]]，令其在1小時內迅速死亡。

菲律賓眼鏡蛇的蛇毒主要是[[神經|神經]]毒素，這些毒素嚴重影響神經訊號的輸送，造成神經與[[肌肉|肌肉]]之間聯繫的障礙，中毒表徵有頭痛、嘔吐、腹痛、腹瀉、暈眩等。兒童中毒後，更會直接出現昏厥或痙攣的反應，這些反應將會是判斷其所中蛇毒種類的重要現象。而被菲律賓眼鏡蛇毒的影響之下，亦會出現[[低血壓|低血壓]]的常見徵狀，可是[[高血壓|高血壓]]甚至心跳過快及過慢等各類現象亦很常見，因此中毒後的實際反常反應可能有極大變化。最終，中毒者會因為[[呼吸系統|呼吸系統]]癱瘓而死亡。
