'''Fitness First'''中譯'''菲力斯第一'''<ref>{{cite web|url=http://www.hktdc.com/sourcing/hk_company_directory.htm?companyid=1X030KJT&locale=zh_TW|title=菲力斯第一(香港)有限公司 - 服務行業公司 - 香港貿易發展局|work=www.hktdc.com}}</ref>，俗稱'''FF'''，1993年於[[英國|英國]]南部[[伯恩茅斯|伯恩茅斯]]開設首間會所<ref>[http://www.fitnessfirst.com/group/about-us/history.aspx History] {{Wayback|url=http://www.fitnessfirst.com/group/about-us/history.aspx |date=20121203071451 }}，Fitness First網頁</ref>，隨後不斷擴充，現為全球最大由私人擁有的連鎖健身中心，迄今已有超過400間分店，遍佈15個國家，會員人數超過140萬<ref>{{cite web |url=http://www.fitnessfirst.com/group/other/club-and-members-by-country.aspx |title=存档副本 |accessdate=2011-06-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20110603130956/http://www.fitnessfirst.com/group/other/club-and-members-by-country.aspx |archivedate=2011-06-03 }}</ref><ref>http://www.altassets.net/private-equity-news/article/nz18840.html{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。於英國本土，分店數目達160所，會員人數達405,000<ref>{{cite web |url=http://www.fitnessfirst.co.uk/press-office/fitness-first-data-and-facts.aspx |title=存档副本 |accessdate=2011-06-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20110220235229/http://www.fitnessfirst.co.uk/Press-Office/Fitness-First-Data-And-Facts.aspx |archivedate=2011-02-20 }}</ref>。

==簡史==
*1993年，首間會所開設於英國小鎮伯恩茅斯。
*1999年，進駐[[比利時|比利時]]。
*2000年，進軍[[亞洲|亞洲]]並進駐[[西班牙|西班牙]]。
*2001年，進入[[馬來西亞|馬來西亞]]市場，開展[[法國|法國]]、[[荷蘭|荷蘭]]及[[意大利|意大利]]業務。年內全球新增77間會所。
*2002年，快速擴張，全年開設110間分店。
*2003年，{{link-en|Cinven|Cinven}}基金用了5億歐元收購Fitness First，並將它從[[倫敦交易所|倫敦交易所]]的AIM版中除牌。
*2005年，BC Partners私人股本基金從Cinven手中收購Fitness First，當時估值為10.2億歐元<ref>{{cite web |url=http://www.bcpartners.com/bcp/news/2005/2005-09-22/ |title=存档副本 |accessdate=2011-09-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20110830161131/http://www.bcpartners.com/bcp/news/2005/2005-09-22 |archivedate=2011-08-30 }}</ref>。
*2005-2006年，在[[澳洲|澳洲]]從Living Well及Space Health手中收購了多間分店，當時一度備受當地競爭及消費委員會（[[:en:Australian_Competition_and_Consumer_Commission|Australian Competition and Consumer Commission]]）關注<ref>{{cite web |url=http://www.accc.gov.au/content/index.phtml/itemId/764453/fromItemId/751043 |title=存档副本 |accessdate=2011-09-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20120228190040/http://www.accc.gov.au/content/index.phtml/itemId/764453/fromItemId/751043 |archivedate=2012-02-28 }}</ref><ref>{{cite web |url=http://www.accc.gov.au/content/index.phtml/itemId/712484/fromItemId/751043 |title=存档副本 |accessdate=2011-09-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20120228192319/http://www.accc.gov.au/content/index.phtml/itemId/712484/fromItemId/751043 |archivedate=2012-02-28 }}</ref>。
*2008年，打入[[印度|印度]]市場。
*2010年，打算在亞洲上市，希望集資14.6億歐元<ref>{{cite web|url=http://www.ft.com/cms/s/0/daac4964-81e0-11df-938f-00144feabdc0.html|title=Subscribe to read|work=Financial Times}}</ref>。
*2014年，目標5年內在亞洲新增逾42間全新會所<ref name="蘋果日報20140929"/>，總投資超過8億港元<ref name="蘋果日報20140520"/>。

==銷售手法爭議==
*在澳洲，於2009年被評為銷售手法最不友善的健身中心<ref>{{cite web|url=http://www.smh.com.au/national/gym-chain-slammed-for-putting-itself-first-20090502-aqsw.html|title=Gym chain slammed for putting itself first|date=2009-05-03|publisher=|via=The Sydney Morning Herald}}</ref>。
   
*有會員於2008年10月中到Fitness First健身中心尖沙咀店繳付一個月月費及行政費共998元，正式入會使用健身服務。翌日他到該中心做運動時，一名銷售員向他表示可提供新優惠，只要他支付2,988元預繳半年服務費，半年後可獲豁免三個月月費。他即時支付現金及獲發一張手寫收據，收據上印有該公司名稱及印章。12月初，會員發現信用卡賬戶被扣除健身服務月費，遂向該門市經理查詢，竟獲覆他持有的收據為舊版本，該公司已停用，並將有關員工解僱。健身中心回覆表示，已調查個案，並交警方處理。會員所持的收據乃涉案員工擅自偽造，也非舊收據式樣。該公司承諾會履行涉案員工給予胡先生的優惠以表歉意。<ref>[http://hk.apple.nextmedia.com/news/art/20081222/11996969 送「教練堂」其實已繳費服務收據竟是偽造保障權益：健身中心銷售員呃客]，[[香港蘋果日報|香港蘋果日報]]，2008年12月22日</ref>
