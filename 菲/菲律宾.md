'''菲律賓共和國'''（{{lang-fil|Republika ng Pilipinas}}；{{lang-en|Republic of the Philippines}}），通稱'''菲律賓'''，是位於[[東南亞|東南亞]]的一個[[群島|群島]][[國家|國家]]。其[[地理位置|地理位置]]處於西[[太平洋|太平洋]]，北隔[[呂宋海峽|呂宋海峽]]與[[臺灣|臺灣]]相望，雙方群島最近的距離為99公里<ref name = "2％華人掌握6成經濟！關於菲律賓 台灣人的10大盲點"> {{ cite web | title = 2％華人掌握6成經濟！關於菲律賓 台灣人的10大盲點 | url = http://www.chinatimes.com/realtimenews/20180219000019-260410 | publisher = 中時電子報 | author = 徐慈薇 | language = zh-tw | date = 2018-02-19 | quote = "菲律賓是一個群島國，領土最北端的巴丹群島中的Mavudis Island，距離蘭嶼南方小蘭嶼只有99公里" }} </ref>，南隔[[西里伯斯海|西里伯斯海]]與[[印度尼西亚|印度尼西亚]]相望，西隔[[南中国海|南中国海]]與[[越南|越南]]相望，東邊則為[[菲律宾海|菲律宾海]]。作為一個座落於[[环太平洋火山带|環太平洋地震帶]]上的[[熱帶|熱帶]]國家，菲律賓常年飽受[[地震|地震]]與[[颱風|颱風]]侵襲，然而其[[氣候|氣候環境]]也造就了豐富的[[自然資源|自然資源]]與[[生物多樣性|生物多樣性]]。菲律賓群島由7,641個島嶼組成<ref>{{cite web|url=https://www.gov.ph/web/guest/about-the-philippines|title=|language=en|script-title=en:About The Philippines|access-date=2018年6月22日}}</ref><ref>{{cite web|url=http://cnnphilippines.com/videos/2016/02/20/More-islands-more-fun-in-PH.html|title=|date=2016年2月20日|language=en|script-title=en:More islands, more fun in PH|access-date=2018年6月22日}}</ref>，可分為[[呂宋|呂宋島]]、[[米沙鄢群島|-{zh:米沙鄢群岛;zh-tw:維薩亞斯群島;zh-hk:維薩亞斯群島;zh-sg:米沙鄢群島;zh-cn:米沙鄢群岛;}-]]和[[棉兰老岛|-{棉兰老岛]]三大島群。其國內人口約9,300萬，加上約1,100萬[[海外菲律賓人|海外菲律賓人]]，2015年時總人口破億（[[國家人口列表|世界第12名]]），成為東南亞第二個人口破億的國家。菲律賓群島上的[[人種|種族]]與[[文化|文化]]為數眾多，史前的[[尼格利陀人|尼格利陀人]]可能是該地區最早的居民，隨後[[南岛民族|南島民族]]的遷徙帶來了[[马来族|馬來文化]]，隨著[[宗教|宗教]]與[[貿易|貿易]]的發展，各地分別受到了[[印度文化|印度文化]]、[[中華文化|中華文化]]與[[伊斯蘭文化|伊斯蘭文化]]的影響。

1521年[[麥哲倫|麥哲倫]]探險隊航海抵達此地，隨後[[西班牙人|西班牙人]]於1565年至1571年期間開始陸續佔領菲律賓群島，展開長達300多年的統治。19世紀末期，菲律賓經歷了[[菲律宾革命|對西班牙革命]]、[[美西战争|美西战争]]及[[美菲战争|美菲战争]]之後，成為[[美國領土擴張|美國殖民地]]，於[[第二次世界大战|第二次世界大战]]期間被[[大日本帝國|日本]]佔領，並在戰後獨立。美國在當地留下了[[英语|英文]]的主導地位以及對[[西方文化|西方文化]]的認同。獨立至今，菲律賓經歷過數次的經濟快速成長，然而，政局動盪、貪污問題及社會不安成為了阻礙其發展的一大因素。此外，其國內還有嚴重的貧富差距。但是，近年來菲律賓的經濟有所改善，海外打工移民陸續歸國，特別是北部各島嶼、[[马尼拉|马尼拉]]的建設達到新興國家水平，該國並榮獲了[[亞洲四小虎|亞洲四小虎]]的称号。

== 历史 ==
{{main|菲律宾历史}}
===古代===
數十萬年前，在[[旧石器时代|旧石器时代]]菲律賓已經有早期的人类活動。約一萬年前，現代菲律賓居民的祖先才定居菲律賓。在菲律賓當地，[[考古|考古]]發掘甚至發現了3世紀的[[中国|中国]]瓷器。

据中国史料记载，226年（東吳[[黃武|黃武]]五年），東吳官員宣化從事朱應、中郎[[康泰|康泰]]浮海巡撫，也曾到過今天菲律賓境內的臣延、耽蘭和杜薄{{notetag|1=康泰著有《[[吴时外国传|吴时外国传]]》——《扶南傳》。裴松之注《[[三國志|三國志]]》與《[[晉書|晉書]]》中也有提及。}}。[[唐朝|唐]]、[[宋朝|宋]]時期，[[漢人|漢人]]與菲律賓各地有貿易往來。

===苏禄苏丹国===
1380年，伊斯蘭傳教士阿爾·馬卡敦首先把伊斯蘭教傳入菲律賓的蘇祿<ref name = CNKI_2012> {{ cite web | title = 试论伊斯兰教在东南亚岛屿地区的传播 | url = http://sea.cssn.cn/webpic/web/sea/upload/2012/02/d20120217112602000.pdf | author = 李一平教授 | language = zh-cn | date = 2012-02-15 }} </ref>。1390年，来自[[蘇門答臘|苏门答腊]]的[[米南佳保人|米南加保人]]在菲律宾建立了[[蘇祿蘇丹國|苏禄苏丹国]]。1405年（明[[永乐_(明朝)|永乐]]三年），[[郑和下西洋|郑和下西洋]]，巡莅[[菲律賓群島|菲律賓群島]]。[[鄭和|郑和]]奉[[明成祖|永樂帝]][[詔書|詔書]]封旅菲侨领[[許柴佬|許柴佬]]爲[[呂宋|呂宋]]總督，統攬該地區的財、軍、文大權二十年之久。1417年（明[[永乐_(明朝)|永乐]]十五年），[[苏禄|苏禄]]王親自率使臣到中國拜見明成祖，回程路上病死于山东[[德州_(山东)|德州]]。另外，中国史籍中的提及的[[民都洛|麻逸]]、古麻剌朗等國皆於今日菲律賓國內。

=== 殖民地时期 ===

==== 伊斯蘭政權 ====
1450年-1520年間有多位阿拉伯人到菲律賓南部各地傳教，並建立多個蘇丹政權；1450年，[[阿拉伯人|阿拉伯]]商人謝里夫·阿布·巴卡爾（Abu Bakar）從[[印尼|印尼]][[蘇門答臘|蘇門答臘]]到菲律賓南部[[蘇祿群島|蘇祿群島]]的[[霍洛島|霍洛島]]與沿海地區宣傳穆斯林教義，後自稱阿布·巴卡爾蘇丹並建立[[伊斯兰|伊斯兰]][[政权|政权]]：[[蘇祿蘇丹國#伊斯蘭化及建國|環蘇祿海伊斯蘭蘇丹王國]]；1475年，阿拉伯人謝里夫·卡本蘇旺（Kabungsuwan）從[[馬來西亞|馬來西亞]][[柔佛|柔佛]]到菲律賓南部[[民答那峨島|民答那峨島]]的[[哥打巴托市|哥打巴托市]]傳教，宣達穆斯林教義到民答那峨島內陸地區並建立蘇丹政權。穆斯林到菲律賓南部宣傳教義也為原本部落帶來穆斯林的文化與法律制度，菲律賓南部的部分部落酋長稱謂也改稱蘇丹<ref name = CNKI_2012 /><ref name="陳鴻瑜2016">{{cite book|author=陳鴻瑜|title=菲律賓史: 東西文明交會的島國|url=https://books.google.com/books?id=90TmtAEACAAJ|year=2016|publisher=三民|isbn=978-957-14-6199-1}}</ref>。

==== 西班牙殖民時期 ====
1521年，[[麦哲伦|麦哲伦]]率领西班牙探险队於[[地理大發現|地理大發現]]首次環球航海時抵達此地，麦哲伦亦是在此地被土著砍死。1542年，西班牙人以王子[[腓力二世_(西班牙)|菲力普]]之名，將此群島命名為「{{lang|es|''Las Filipinas''}}」，成為「菲律賓」名稱的由來。<ref>{{Cite web |url = https://global.udn.com/global_vision/story/8662/2519399 |title = 菲律賓國慶的這一天，國會議員提案：「解殖」改國名 |author = 轉角24小時 |date = 2017-06-12 |language = 中文 }}</ref>

1565年，[[宿霧島|宿雾岛]]被[[西班牙|西班牙]]人所占领。1571年，西班牙人侵占吕宋島，建马尼拉城。1594年，西班牙人公告馬尼拉成為菲律宾群島的[[首都|首都]]。此即西班牙统治菲律宾的開始。

==== 菲律賓第一共和國 ====
1896年8月，[[安德烈·滂尼发秀|安德烈·滂尼发秀]]领导的[[卡的普兰|卡的普兰]]（Katipunan）在[[马尼拉|马尼拉]]近郊的[[马林塔瓦克山|马林塔瓦克山]]（卽巴林塔瓦克山）发动起义，各地响应，进攻西班牙殖民军。次年阵营内部发生分裂，[[埃米利奧·阿奎納多|埃米利奧·阿奎納多]]（Emilio Aguinaldo）篡夺领导权，杀害[[安德烈·滂尼发秀|滂尼发秀]]，并与西班牙殖民者妥协。但起义者坚持斗争，后起义者几乎占领全国。[[埃米利奧·阿奎納多|埃米利奧·阿奎納多]]於5月17日乘美艦麥克科洛济（Mc culloch）號返菲活動。1898年6月12日菲律賓宣告脫離[[西班牙|西班牙]]独立，建立[[菲律宾第一共和国|第一共和]]。

==== 美國殖民時期 ====
1898年（菲律賓宣告独立的同一年），爆發[[美西战争|美西戰爭]]，12月10日，[[西班牙|西班牙]]被[[美国|美国]]打敗，簽署「[[巴黎條約_(1898年)|巴黎和約]]」。[[美國聯邦政府|美國政府]]給西班牙2,000萬[[美元|美元]]購買菲律賓[[主權|主權]]，{{en-link|菲律賓美國統治时期|History of the Philippines (1898–1946)}}开始，菲律賓成為[[美國|美國]]的殖民地，以及美國在亞太補給的軍事基地。

==== 菲律賓國協 ====
1916年8月29日美國國會通過{{tsl|en|Jones Law (Philippines)|菲律賓自治法案}}，該法案成為菲律賓1934年立法以前的憲法基礎，1916年10月3日菲律賓舉行選舉，建立了{{tsl|en|Senate of the Philippines|菲律賓參議院}}。1919年2月23日菲律賓獨立代表團赴美國華盛頓首府遊說請願、另赴國會備忘錄表明菲律賓已經做好獨立準備。1920年美國總統[[伍德羅·威爾遜|伍德羅·威爾遜]]在美國國會進行告別演說，呼籲國會通過菲律賓獨立法案但未獲得美國國會同意。

1934年3月{{tsl|en|President of the Senate of the Philippines|菲律賓參議院議長}}[[曼努埃爾·奎松|曼努埃爾·奎松]]組團赴美國國會進行遊說，美國國會通過{{tsl|en|Tydings–McDuffie Act|泰丁斯·麥克杜菲法案}}規範菲律賓獨立需要進行的程序，1934年3月24日美國總統[[富蘭克林·德拉諾·羅斯福|富蘭克林·德拉諾·羅斯福]]簽署{{tsl|en|Tydings–McDuffie Act|泰丁斯·麥克杜菲法案}}成為正式法律，1934年5月1日菲律賓參議院通過{{tsl|en|Tydings–McDuffie Act|泰丁斯·麥克杜菲法案}}。1934年7月10日菲律賓選舉出制憲會議代表。1934年7月30日菲律賓制憲會議起草菲律賓憲法。1935年2月8日，菲律賓制憲會議通過憲法草案。

1935年3月23日，美國總統[[富蘭克林·德拉諾·羅斯福|富蘭克林·德拉諾·羅斯福]]簽署菲律賓憲法草案。1935年5月14日菲律賓舉行公民投票通過憲法草案。1935年9月17日菲律賓舉行正副總統大選，[[曼努埃爾·奎松|曼努埃爾·奎松]]當選菲律賓國協首任總統。1935年11月15日[[菲律賓國協|菲律賓國協]]正式建立。<ref name = "台灣自治白皮書_2016"> {{ cite web | title = 台灣自治白皮書 | url = https://books.google.com.tw/books?id=oy9QCwAAQBAJ&pg=PA73&lpg=PA73&dq=1919+%E8%8F%B2%E5%BE%8B%E8%B3%93+%E5%A5%8E%E6%9D%BE+%E7%BE%8E%E5%9C%8B+%E5%A8%81%E7%88%BE%E9%81%9C%E7%B8%BD%E7%B5%B1&source=bl&ots=HjjZKf40O1&sig=-xDUb-ygDeA5_E5uY50qNleWeiU&hl=zh-TW&sa=X&ved=2ahUKEwixqsSRz97cAhVHZt4KHTwqDCQQ6AEwAHoECAAQAQ#v=onepage&q=1919%20%E8%8F%B2%E5%BE%8B%E8%B3%93%20%E5%A5%8E%E6%9D%BE%20%E7%BE%8E%E5%9C%8B%20%E5%A8%81%E7%88%BE%E9%81%9C%E7%B8%BD%E7%B5%B1&f=false | publisher = 台灣語文出版館, 2016 | author = 吳崑松 | language = zh-tw | date = 2016-01-01 }} </ref>

==== 日本殖民時期 ====
1941年12月7日，日本攻擊美國[[珍珠港|珍珠港]]，1941年12月8日日本攻擊菲律賓國協，1942年1月3日日本佔領馬尼拉頒布戒嚴法，實施軍事管理。
1942年－1945年，[[第二次世界大战|第二次世界大战]]期间[[大日本帝国|日本]]佔領菲律賓，1943年6月日本要求菲律賓召開新的制憲會議，研擬殖民政府的新憲法草案。<ref name = "台灣自治白皮書_2016" />

==== 菲律賓第二共和國 ====
1943年10月建立[[菲律賓第二共和國|菲律賓第二共和國]]（[[傀儡政權|傀儡政權]]），日文漢字譯為「比律賓」。<ref name = "台灣自治白皮書_2016" />

=== 独立时期 ===
1946年7月4日，菲律宾共和国（第三共和）完全独立惟[[美国|美国]]仍在菲律宾保留[[军事基地|军事基地]]。1965年，[[费迪南德·马科斯|费迪南德·马科斯]]当选[[總統|總統]]。1969年馬可仕在被指選舉舞弊下成功连任，1970年1~3月發生首季風暴學生運動，1972年取消總統任期限制，宣佈[[戒严|戒严]]開始[[独裁|独裁]]統治。1981年馬可仕結束戒嚴，但仍然維持獨裁統治。1986年2月25日，[[柯拉蓉·艾奎諾|柯拉蓉·艾奎諾]]宣布竞选胜利后，費迪南德·馬可仕遭[[人民力量革命_(1986年)|人民力量革命]]推翻，流亡[[美国|美国]][[夏威夷州|夏威夷]]，直至[[1989年|1989年]]病逝。科拉松上台後，雖然仍歷經多次軍事政變，加上馬可仕下台後瀕臨崩潰的經濟，但最終仍完成任期，1992年6月30日卸任，由[[菲德尔·瓦尔德斯·拉莫斯|菲德尔·瓦尔德斯·拉莫斯]]將軍接任。菲律賓首次[[和平|和平]]轉移[[政权|政權]]，實現政黨輪替。1998年5月，[[約瑟夫·艾斯特拉達|約瑟夫·艾斯特拉達]]当选[[总统|总统]]。2001年1月20日，約瑟夫·艾斯特拉達因[[賄賂|受贿]][[醜聞|醜聞]]，而於[[人民力量革命_(2001年)|第二次人民力量革命]]中被迫下臺，[[副总统|副总统]][[格洛丽亚·马卡帕加尔-阿罗约|阿罗约夫人]]继任。2004年，阿罗约夫人当选[[总统|总统]]，成功連任。[[2006年菲律賓政變|2006年菲律賓政變]]，五千多人示威要求總統阿罗约夫人下臺，總統阿罗约夫人其後宣佈全國進入[[紧急状态|紧急状态]]。2010年5月10日，柯拉蓉之子[[貝尼格諾·阿基諾三世|諾諾·艾奎諾]]擊敗[[约瑟夫·埃斯特拉达|约瑟夫·埃斯特拉达]]，成功當選總統。6月30日，[[貝尼格諾·阿基諾三世|艾奎諾]]正式就任[[總統|总统]]，任期六年。

2016年达沃市市長[[罗德里戈·杜特尔特|罗德里戈·杜特尔特]]以大比數勝出總統選舉，成为菲律賓第一位來自-{zh:棉兰老岛;zh-tw:民答那峨島;zh-hk:棉蘭老島;zh-sg:棉兰老岛;}-的總統。另外代表自由黨競選的[[萊妮·羅布雷多|萊妮·羅布雷多]]勝出成为副總統。

菲律宾由7,107個島嶼組成，面積約30萬平方公里，海岸線總長36,289公里，居世界[[各国海岸线长度列表|第五位]]<ref name=About/><ref>Central Intelligence Agency.（2009）. == 政治 ==
{{main|菲律宾政治}}

[[File:Malacañang_Palace_(Cropped).jpg|thumb]]]]

菲律宾是[[东南亚国家联盟|东南亚国家联盟]]主要成员国，也是[[亚洲太平洋经济合作组织|亚洲太平洋经济合作组织]]21個成员国之一；1945年10月24日[[联合国|联合国]]建立之时，菲律宾也是其中参与国家。

菲律宾实行[[權力分立|三權分立]]的[[总统制|总统制]]。[[总统|总统]]是[[国家元首|国家元首]]、政府首脑兼[[武装部队|武装部队]]总司令，任期六年，但不能連任。由於總統選舉和副總統選舉是分別選出，因此會出現總統和副總統屬不同政黨。當總統無法履行職務時，副總統將會代行職務，當總統去世或辭職，而無法完成任期，副總統將會繼任總統。未能完成任期的總統如果擔任的年期沒有超過一半，可在下一個任期再次競選。

菲律宾實行[[兩院制|兩院制]]，其中參議院24人，眾議院287人，其中229人分區直選，58人按政黨得票比例分配。參議員任期六年，眾議院任期三年。參議院每三年改選一半議席。

=== 军事 ===
[[File:DF-ST-85-01374.jpg|缩略图]]
菲律宾不实行征兵制，军人全部是雇佣兵（志愿兵）。現役軍人12萬，預備役軍人13萬。1999年军费高达14亿美元，占国内生产总值（GDP）的2.2 %。[[菲律宾武装部队|菲律宾軍队]]由陸海空三軍所組成。菲律宾海军拥有114艘舰艇，但大部份舰艇是一些小型近海巡逻艇。
菲律賓的戰機目前為止只有幾架，大部分為外國軍援贈送。

2018年8月9日，菲律賓成功試射購買自以色列的艦艇用「增程型長釘飛彈」（Spike-ER）飛彈系統，這將使用在海軍於菲律賓沿海打擊恐怖份子與執行法律上。<ref name = "菲律賓海軍也有飛彈了 首套飛彈系統試射成功 - 國際 - 自由時報電子報"> {{ cite web | title = 菲律賓海軍也有飛彈了 首套飛彈系統試射成功 - 國際 - 自由時報電子報 | url = http://news.ltn.com.tw/news/world/breakingnews/2514809 | publisher = 自由時報 | language = zh-tw | date = 2018-08-09 }} </ref><ref name = "目標南海！菲海軍首次成功試射以色列製近程導彈 - 中時電子報"> {{ cite web | title = 目標南海！菲海軍首次成功試射以色列製近程導彈 - 中時電子報 | url = http://www.chinatimes.com/realtimenews/20180809004688-260417 | publisher = 中時電子報 | author = 盧伯華 | language = zh-tw | date = 2018-08-09 }} </ref>

== 行政区划 ==
{{main|菲律宾行政区划}}
菲律宾分为[[呂宋|吕宋岛]]、[[米沙鄢群島|-{zh:米沙鄢群岛;zh-tw:維薩亞斯群島;zh-hk:維薩亞斯群島;zh-sg:米沙鄢群島;zh-cn:米沙鄢群岛;}-]]和[[棉兰老岛|-{棉兰老岛]]三个岛组，劃分為17個大區（包括国家首都区、科迪勒拉行政区和[[棉兰老穆斯林自治区|棉兰老穆斯林自治区]]在内），下设76个省，最大島為吕宋岛，首都為[[马尼拉|马尼拉]]。

==== 菲律宾华人 ====
{{main|菲律宾华人}}
菲律賓華人，几乎遍佈全菲各地。據統計，目前在[[菲律賓人口|菲律賓人口]]的華人約有百萬，其中先祖來自[[福建|福建]][[閩南|閩南]]者（[[泉州|泉州]]、[[漳州|漳州]]、[[廈門|廈門]]）十之八九，當中又以泉州為最。

根據[[人類學|人類學]]家所作的這項研究，遍處東南亞地區的華人，與當地族群通婚最多者，首推[[泰國|泰國]]，其次為菲律賓，在菲律賓民族中，華人血統約佔20%；過去由於[[漢學|漢學]]教育不發達，大部份具華人血統的，鮮有機會接受[[華文|華文]]教育，也由於信奉[[天主教|天主教]]，結果[[同化|同化]]成純粹的菲律賓人。其中只有四分之一含有中國血統的，因為接受[[中文|華語教育]]而未徹底同化。

如果依照氣質和文化的觀點來說，目前凡是自認為華人、身處華人社會、保持華人傳統生活習俗、加入華人社團、在家裏和華人社群場所說華語或[[汉语方言|漢語方言]]、有意讓子女接受華文教育、以華人為子女選擇婚配主要對象、保有其成為華人的條件和傾向的華人，在菲律賓大約有六十萬至八十萬人；其中聚居在[[馬尼拉|馬尼拉]]大都會（Metro Manila）者，約佔百分之六十，散居在各地的，約佔百分之四十。如果加上跟這許多所認定華人標準比較接近，或是有間接關係含有中國血統者，放寬認定的標準，目前菲律賓華人的總數，當於百萬以上。

在所有菲律賓華人之中，不論其已否菲化，百分之八、九十以上屬[[闽南民系|閩南裔]]；其餘約百分之十，以[[广东省|廣東]]籍為多，其它省籍為數無幾。[[閩南語|閩南語]]（當地稱[[棉蘭福建話|福建話]]）是菲華社區的通行語。

福建籍華人中，以原屬[[泉州府|泉州府]]的[[泉州三邑|三邑]]（[[晉江縣|晉江縣]]、[[惠安縣|惠安縣]]、[[南安縣|南安縣]]）等縣最多，大部份居住在[[呂宋|呂宋島]]，尤其是大馬尼拉地區，所經營的工商業和文教事業，也以在大馬尼拉地區的最為發達。

原屬[[泉州府|泉州府]][[同安縣|同安縣]]（今[[廈門市|廈門市]]和[[金門縣|金門縣]]）、[[漳州府|漳州府]][[龙溪县|龍溪]]、[[海澄|海澄]]（今合併為[[龙海市|龍海市]]）等，大多數在中部[[維薩亞斯|維薩亞斯]]地區的[[宿霧島|宿霧]]等地，和[[民答那峨島|民答那峨島]]各處。

[[臺灣人|臺灣人]]因對菲律賓有投資事業，投資人眷屬及應聘前來之經營管理或製造技術人員，早年來菲人數不少，但近年來已逐漸減少，陸續被[[中国大陆|中國大陆]]與[[越南|越南]]等其他國家和地区的產業所取代。

=== 语言 ===

{{main|菲律賓語言}}
依據不同分類方法，菲律賓有120到175種語言和方言<ref>{{Cite journal |last=McFarland |first=C. D. |month= |title=Subgrouping and Number of Philippine Languages |journal=Philippine journal of linguistics |issn=0048-3796 |year=1994 |volume = 25 |issue=1-2 |pages=75-84  |postscript=<!--None-->}}</ref> 。幾乎全屬於[[馬來-波利尼西亞語族|馬來-波利尼西亞語族]]，僅有零星例外，如[[查瓦卡諾語|查瓦卡諾語]]是一種來自[[羅曼語族|羅曼語族]]的[[克里奧爾語|克里奧爾語]]。菲律賓有兩種官方語言（[[英語|英語]]和[[菲律賓語|菲律賓語]]）。而截至2017年，有19種官方輔助語言（地區語言）<ref name="7 mother tounges">[http://www.gmanetwork.com/news/story/317280/news/nation/deped-adds-7-languages-to-mother-tongue-based-education-for-kinder-to-grade-3 DepEd adds 7 languages to mother tongue-based education for Kinder to Grade 3]. GMA News. 7/13/2013.</ref>。若包括第二語言的使用者，使用菲律賓語的人口還是多於英語<ref>{{cite web|url=http://www.ethnologue.com/country/ph/languages/|title=Philippines|publisher=[[Ethnologue|Ethnologue]]}}</ref>。

現今菲律賓本土文字（如[[庫利坦字母|庫利坦字母]]、{{link-en|塔格巴努瓦字母|Tagbanwa}}等）的使用很少; 主要由於[[西班牙|西班牙]]和[[美國|美國]]的殖民地經驗，菲律賓語言今日大多使用[[拉丁字母|拉丁字母]]。然而，最著名的菲律賓本土文字{{link-en|貝貝因字母|Baybayin}}仍存在一些官方應用，用來書寫菲律賓鈔票上“Pilipino”一詞。另外，菲律賓南部一些穆斯林地區也使用[[阿拉伯字母|阿拉伯字母]]。

=== 宗教 ===
{{main|菲律賓宗教|菲律賓天主教}}
* [[天主教|天主教]]：菲律賓和[[东帝汶|東帝汶]]是亞洲僅有的兩個天主教國家。天主教徒佔菲律賓人口的83%；從西班牙傳入五百年來一直對菲律賓的[[政治|政治]]和[[社會|社會]]產生巨大影響。1986年費迪南德·馬可仕和2001年約瑟夫·艾斯特拉達被迫下臺，[[天主教會|天主教會]]都發揮了重要作用。全國城鄉各處遍佈大小[[教堂|教堂]]和售賣天主教用品的[[店鋪|店鋪]]。菲律賓有四座[[巴洛克|巴洛克]]風格教堂已列為[[世界文化遺產|世界文化遺產]]，分別坐落在[[马尼拉|馬尼拉]]、[[圣玛利亚_(邦阿西楠省)|聖瑪麗亞]]、[[帕瓦伊|帕瓦伊]]和[[米亞高|米亞高]]。菲律賓還擁有世界上最長的聖誕季，從12月16日開始。
* [[基督新教|基督新教]]：信徒佔人口的9%，乃是菲律賓信徒人口成長率最高的宗教。
* [[伊斯蘭教|伊斯蘭教]]：信徒佔人口的5%，集中在南部的民答那峨島。

此外，華人中仍有一部份信奉着[[佛教|佛教]]；[[土著民族|土著民族]]信仰著[[原始宗教|原始宗教]]。

== 文化 ==
菲國在歷史上一直沒有特有而一致的「菲律賓文化」。主要原因之一乃由於菲律賓是個「七千島國」，各地有太多不同的[[語言|語言]]，彼此不一定能充分交流，[[文化|文化]]發展和特色都非常局部性；現有80餘種[[語言|語言]]在境內使用。自古以來有許多移民遷居於菲律賓，各種東西方文化在此交匯，包括[[马来西亚|馬來西亞]]、[[印度尼西亚|印度尼西亞]]、[[西班牙|西班牙]]、[[荷兰|荷蘭]]、[[中國|中國]]、[[墨西哥|墨西哥]]和[[美國|美國]]。

菲律宾很早就有了丰富的[[口头文学|口头文学]]，包括[[戏剧|戏剧]]、[[史诗|史诗]]、抒情诗、[[神话|神话]]、[[謎語|谜语]]、[[谚语|谚语]]等多种样式。如古代[[伊富高人|伊富高人]]的叙事诗《阿丽古容》、《邦都地区的狩猎歌》、《孤儿之歌》，民答那峨島[[玛拉瑙人|玛拉瑙人]]的史诗《达兰岗》等。16世纪西班牙殖民者入侵后，禁毁菲律宾民族文学，早期的书面文学几乎凋零殆尽。[[弗兰西斯科·巴尔塔萨尔|弗兰西斯科·巴尔塔萨尔]]是近代菲律宾文学的开路人，他的《弗罗兰第和罗拉》（1838）批判殖民统治，歌颂爱情与自由。爱国者[[黎刹|黎刹]]创作了控诉小说《不许犯我》（另譯《社會毒瘤》，1887）和《起义者》（1891），揭露神权统治的虚伪，对独立运动深有影响。<ref>马燕冰、黄莺（2007），菲律宾.  社会科学文献出版社。页308 - 309. ISBN 7-80230-563-2</ref>

== 經濟 ==
{{main|菲律賓經濟}}

[[File:Makati_skyline_j_0_n.jpg|thumb]]]]

菲律賓在[[第二次世界大战|二戰]]後的1950年代至1970年代之間，與[[日本|日本]]、[[緬甸|緬甸]]同屬亞洲最富國之一。2006年，菲律賓被[[世界銀行|世界銀行]]依[[購買力平價|購買力平價]]列為第36大[[經濟體|經濟體]]。目前，菲律賓經濟規模居[[各国国内生产总值列表_(国际汇率)|世界第39位]]，2015年時的國內生產總值（國際匯率）為2,924.51億美元<ref name=imf2 />，是新興[[工業|工業]]國家及世界的新興市場之一。{{fact}}首都馬尼拉為國際第24大購買力城市。菲律賓的法定貨幣為[[菲律賓披索|菲律賓披索]]（₱ or PHP）。

菲律賓的產經結構以[[農業|農業]]及工業為主，尤其著重於[[食品|食品]]加工、[[紡織|紡織]][[成衣|成衣]]以及[[电子|電子]]、[[汽車|汽車]]組件等[[輕工業|輕工業]]。大部分的工業集中於[[馬尼拉大都會|馬尼拉大都會]]的市郊。此外，[[宿霧大都會|宿霧大都會]]近來也成為吸引外國及本地[[投资|投資]]的另一個地點。菲律賓的礦業有很大的潛力，擁有大量儲備的鉻鐵礦、[[鎳|鎳]]及銅。近來在[[巴拉望|巴拉望]]外島發現的[[天然氣|天然氣]]，也是菲律賓豐富的[[地熱|地熱]]、[[水力|水力]]及[[煤炭|煤炭]]等[[能源|能源]]儲備的一部分。

菲律宾主要出口品包括[[半導體|半導體]]、電子產品、運輸設備、成衣、[[銅|銅]]製品、[[石油|石油]]製品、[[椰子油|椰子油]]及[[水果|水果]]<ref name=CIAfactbook />。主要貿易伙伴為[[美國|美國]]、[[日本|日本]]、[[中國|中國]]、[[新加坡|新加坡]]、[[南韓|南韓]]、[[荷蘭|荷蘭]]、[[香港|香港]]、[[德國|德國]]、[[台灣|台灣]]及[[泰國|泰國]]<ref name=CIAfactbook />。菲律賓人均所得大約為1455[[美元|美元]]（2007年），為[[發展中國家|發展中國家]]，且貧富差距很大。[[貪污|貪污]]問題根深柢固，亦冠絕全[[東南亞|東南亞]]，經濟因而滑落，貧窮隨之而來，全國三分一人口處於貧窮線下，每日生活費不足一美元<ref>[http://the-sun.on.cc/cnt/china_world/20100510/00425_001.html 菲律賓改朝換代　貪污貧窮剷不走]《太陽報》2010年05月10日</ref>。

自從2010年[[貝尼格諾·阿基諾三世|艾奎諾]]就任總統後，政治全面改革，經濟逐步改善、外資投資大量注入，2014年經濟成長率高達6.1%，且人口紅利極高，平均人口年齡僅23歲，更擁有超過3,000萬的中產階級人數，獲專家譽為還有35年蓬勃發展光景之國家。

且菲律賓目前總人口數已破一億，每年約有1,100萬的海外勞工將從世界各地打拼所得資產帶回菲國，金額高達250億美元，造就菲律賓當地經濟蓬勃發展，薪資收入高於平均中產階級水平的人數也逐年增加，並且把所有國債完全還清，並有額外收入。

這使當地房產市場的消費族群不僅有外來的長駐者及投資者，同時內需買盤強力支撐，國內買家占消費者總比的五成，根據全球房地產指南（Global Property Guide）指出，2014年第一季菲律賓房地產的漲幅度驚人，排名世界第五。

=== 交通 ===

[[File:NAIA_from_the_Air.JPG|thumb]]]]

菲律賓的运输基本设施相對不發達。這是由於菲律賓的多山岭地勢及零散的岛屿所導致，又由於多屆政府對於菲律賓运输基础设施方面的投资不足。在2013年，菲律賓基础设施投资相当于該國3%的GDP，比鄰近國家低。菲律賓總共有213,151公里的道路，但只有25.56%是鋪有混凝土。

[[貝尼格諾·阿基諾三世|阿基诺三世]]政府提出多個项目去改善國家的基础设施和运输系统。現時菲律賓的主要城市及城镇廣泛使用巴士、吉普车、的士及机动三轮车。2007年，菲律賓有553万已註冊机动车辆，平均年增長為4.55%。

現時菲律宾85個國立机场管理及安全規管由[[菲律宾民航局|菲律宾民航局]]負責。[[尼诺伊·阿基诺国际机场|尼诺伊·阿基诺国际机场]]及[[克拉克国际机场|克拉克国际机场]]服务大马尼拉地区。亞洲最古老的商业航空公司[[菲律賓航空|菲律賓航空]]，及當地知名的廉價航空[[宿霧太平洋航空|宿霧太平洋航空]]，是菲律宾主要的航空公司，提供國內及國際航空服务。

各島之間的交通以航空、航海為主，故菲國航空業相當發達。國際線方面，[[尼诺伊·阿基诺国际机场|尼诺伊·阿基诺国际机场]]是主要的機場[[菲律賓航空|菲律賓航空公司]]是主要的[[航空公司|航空公司]]，往返於马尼拉及亚洲各国首都之间。[[宿霧太平洋航空|宿霧太平洋航空]]為第二大航空公司，以低價促銷的方式吸引顧客。

在國內線則有[[菲航快運|菲鷹航空]]（菲律賓航空公司的子公司）、[[亞航飛龍航空|亞航飛龍航空]]和{{le|South East Asian Airline|South East Asian Airline}}。

=== 華人产业 ===
華人在菲律賓百年來的經濟發展史上，一直扮演著先鋒的角色，尤其是引進國外的[[技術|技術]]、[[資本|資本]]與生產設備，開創初期的農耕與手工業。近年來台灣、香港及東南亞華商[[制造业|製造業]]之發展，並累積資金來菲投資發展，對提升菲國產業層次，成功的協助產品推向國際市場，再度扮演經濟先鋒的角色。而當地華商企業長才，多能順應潮流，掌握經濟脈動，發展製造業、[[房地產|房地產]]、[[金融|金融業]]，以及透過證券市場操作、收購企業、結成大集團，擴大對菲經濟的影響力與貢獻。據[[香港|香港]]《資本家》雜誌1992年9月份指出，菲律賓[[股票|股票]]公開上市的工商控股公司45家，華商為大股東的約有20家。此外，華商即使不透過控股公司組織，亦多進行綜合性投資，[[關係企業|關係企業]]涵蓋面極廣。據統計全菲500家最大公司中，華商約佔1/3，並在紡織及成衣、漂染、[[钢|鋼鐵]]、[[五金件|五金]]、制糖、塑料、木材加工、建築材料、[[百貨|百貨]]及金融等行業佔優勢。現時菲律賓首富為菲律宾华人[[施至成_(企業家)|施至成]]，2013年美國福布斯全球華人富豪榜第六位，財產達132億美元。

=== 旅遊业 ===
[[旅遊業|旅遊業]]是菲律賓的重要產業之一。國內有大量自然風光，加上曾受殖民統治，興建大量建築，故旅遊資源豐富
* [[图巴塔哈群礁|图巴塔哈群礁]]海洋公园（[[世界遗产|世界自然遗产]]）
* 菲律宾的巴洛克教堂：[[聖奧古斯丁教堂_(馬尼拉)|聖奧古斯丁教堂 (馬尼拉)]]（世界文化遗产）
* [[菲律賓科迪勒拉山水稻梯田|菲律賓科迪勒拉山水稻梯田]]（世界文化遗产，{{tsl|en|Banaue|巴拿威鎮}}）：2000多年前开辟，因地處偏遠擁有未受殖民政府毀壞的原始景觀，世界最大[[梯田|梯田]]與世界最大人造[[灌溉|灌溉]]系统。
* [[美岸|美岸（维甘）古城]]（世界文化遗产）：亚洲保存得最完好的西班牙[[殖民主义|殖民]]城市。
* [[普林塞薩港|公主港]]市北方的[[公主港地下河國家公園|公主港地下河國家公園]]（或稱：[[普林塞萨港地下河国家公园|普林塞萨港地下河国家公园]]）（世界自然遗产）
* [[宿霧省|宿霧省]]：[[麦哲伦|麦哲伦]]十字架、圣佩多要塞、圣婴教堂、[[哥伦布|哥伦]]路、巧克力山、麦克坦岛、薄荷岛沙滩和罗博儿童合唱团和Malapascua島<ref>丁楓峻（2010），東南亞自助潛水趣.  華都文化。頁140. ISBN 978-986-6860-93-5</ref>。
* [[马尼拉|馬尼拉]]：[[黎刹公园|黎刹公园]]、西班牙古城（[[圣·奥古斯丁教堂_(马尼拉)|圣·奥古斯丁教堂 (马尼拉)]]、马尼拉大教堂、圣地亚哥古堡）、[[唐人街|唐人街]]、[[马拉坎南宫|马拉坎南宫]]（总统府）、美军纪念公墓、塔尔湖
* 百胜滩（瀑布）。
* [[馬榮火山|马荣火山]]：吕宋岛东南端，「世界最完美的山錐」。
* [[碧瑶市|碧瑶]]：花都，避暑盛地，盛產草莓。
* [[长滩岛|長灘島]]：位於[[米沙鄢群島|-{zh:米沙鄢群岛;zh-tw:維薩亞斯群島;zh-hk:維薩亞斯群島;zh-sg:米沙鄢群島;zh-cn:米沙鄢群岛;}-]]的[[阿克兰省|阿克蘭省]]。

===渔业===
漁業是菲律宾重要的产业。2013年，菲律宾渔业产值占农业产值的16.27%。<ref>中国—东盟博览会官方网站{{cite web |url=http://www.caexpo.org/gb/cafta/t20130710_108416.html |title=存档副本 |accessdate=2014-12-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20141209042329/http://www.caexpo.org/gb/cafta/t20130710_108416.html |archivedate=2014-12-09 }}</ref>菲律宾与台湾因专属经济海域与重叠问题，曾发生多次渔业纠纷。2013年，[[菲律宾海岸防卫队|菲律宾海岸防卫队]]曾射杀台湾渔民。

== 海外菲律賓人 ==
{{main|海外菲律賓人}}

由於國內經濟環境差劣，貪污情況嚴重<ref>2011年研究指出，以1至10的排名（10為貪污問題最嚴重），菲律賓的分數為8.9。[http://www.abs-cbnnews.com/business/03/29/11/philippines-corruption-score-worsens Philippines' corruption score worsens]</ref>，菲律賓女性出外到鄰國謀生者眾，她們大多以[[僕人|家庭傭工]]為最主要職業，當[[性工作者|性工作者]]次之<ref>[http://www.atimes.com/atimes/Southeast_Asia/JC15Ae01.html Philippines exporting labor and sex]《Asia Times》Mar 15, 2008</ref>。是以菲律賓曾被戲稱為「傭人之國」<ref>[http://news.hkheadline.com/dailynews/content_hk/2009/04/01/75271.asp 撰文辱菲「傭人國」陶傑被禁制入境]《頭條日報》2009年4月1日</ref>，部份菲律賓人感到不滿並上街示威，撰文的香港作家[[陶杰|陶杰]]亦因此被當局禁止入境。於美軍基地附近酒吧打工的菲律賓女郎，隨退役美軍嫁到海外的，亦為數不少。

==社会==
=== 治安問題 ===
