'''菲利普三世（'''或译为'''好人菲利普）（'''英语: '''Philip the Good'''; 法语: '''Philippe le Bon'''; 荷兰语: '''Filips de Goede'''; {{bd|1396年|7月31日|1467年|6月15日}}）[[瓦卢瓦王朝|瓦卢瓦王朝]]的第三代[[勃艮第|勃艮第]]公爵（1419年—1467年在位），[[百年战争|百年战争]]末期[[欧洲|欧洲]]重要的政治人物之一。

[[File:Philippe_le_Bon.JPG|thumb]]

== 家世 ==
菲利普三世为勃艮第公爵[[无畏的约翰|无畏的约翰]]之子，生于[[第戎|第戎]]。他的母亲是[[巴伐利亚|巴伐利亚]]公主玛格丽特。在父亲于1419年与[[阿马尼亚克派|阿马尼亚克派]]谈判时遇刺后，菲利普三世继承了[[勃艮第公国|勃艮第公国]]。

== 在百年战争中的角色 ==
法国王太子查理（即后来的国王[[查理七世_(法兰西)|查理七世]]）曾經與他接触，但菲利普三世在位初期仍奉行与[[英格兰|英格兰]]结盟反对[[法国|法国]]瓦卢瓦王室的政策。他可能希望以此来实现削弱法国王室并将勃艮第公国发展为囊括众多[[伯国|伯国]]和[[骑士|骑士]]领地的欧洲强国。菲利普三世指控王太子查理参与了杀害其父亲的阴谋，并发誓要巩固与英格兰国王[[亨利五世_(英格兰)|亨利五世]]的同盟。1423年，菲利普三世安排了其妹与[[贝德福公爵|贝德福公爵]][[兰卡斯特的约翰|兰卡斯特的约翰]]（英军的主要将领，也是亨利五世暴卒后英国在法国的[[摄政|摄政]]）的婚姻。

菲利普三世是造成[[圣女贞德|圣女贞德]]惨死的罪魁。1430年，菲利普三世参加了对[[贡比涅|贡比涅]]的围困。勃艮第军队在这次战斗中俘获了贞德，并將她售予英國，被判火刑而死。

然而，菲利普三世最终与法国王室实现了和解。1435年，由于查理七世承诺将[[皮卡第|皮卡第]]割让给勃艮第公国，菲利普三世承认了查理七世的宗主权，并放弃了与英国人的同盟关系（1435年[[阿拉斯条约|阿拉斯条约]]）。

== 其他政绩 ==
菲利普三世在其在位时期利用联姻、战争等手段大大扩张了勃艮第公国的势力，特别是整个吞并了[[低地国家|低地国家]]（尼德兰）。从1428年到1443年，他相继吞并了属于低地国家的[[埃诺|埃诺]]、[[荷兰|荷兰]]、[[布拉班特公国|布拉班特]]、[[林堡|林堡]]、[[卢森堡|卢森堡]]等伯国；低地国家成为勃艮第的经济支柱。1463年，菲利普三世第一次在尼德兰召开了[[三级会议|三级会议]]。从菲利普三世时代起，勃艮第公国的政治中心实际已转移至尼德兰。

菲利普三世还是他那个时代重要的文艺庇护者。包括[[吉约姆·迪费|吉约姆·迪费]]在内的一大批才子受到过他的庇护。1430年，菲利普三世创立了勃艮第自己的[[骑士|骑士]]勋位：[[金羊毛勋章|金羊毛骑士团]]。

== 婚姻 ==

* 第一个妻子：法兰西的米歇尔，1409年结婚。
* 第二个妻子：阿图瓦的博内，1424年结婚。
* 第三个妻子：葡萄牙的伊莎贝拉，1430年结婚。
