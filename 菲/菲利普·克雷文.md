菲利普·克雷文<span style="font-size:smaller;">，[[MBE|MBE]]</span>（Sir '''Philip Craven'''，{{bd|1950年|7月4日|catIdx=Craven, Philip}}），出生於[[英國|英國]][[英格蘭|英格蘭]][[大曼徹斯特郡|大曼徹斯特郡]][[博爾頓|博爾頓]]，退役[[輪椅籃球|輪椅籃球]]運動員，前[[國際殘疾人奧委會|國際殘疾人奧委會]]主席（第2任）。

== 個人生活 ==
菲利普·克雷文於16歲時因攀爬意外致残，導致失去行走能力<ref>[http://blog.sina.com.cn/s/blog_470bf2570100a9mt.html~type=v5_one&label=rela_nextarticle 专访国际残奥委会主席菲利普·克雷文]《三聯生活周刊》2008年第34期</ref>，他稱意外之前喜歡踢足球，腳法雖有欠靈活，但眼手协调能力比较突出，成為日後投身輪椅籃球運動的原因<ref>[http://www.beijing2008.cn/paralympic/news/news/1year/s214153770/n214154013.shtml 克雷文谈亲身经历 勉励残疾人朋友找回积极人生态度] {{webarchive|url=https://web.archive.org/web/20090317164913/http://www.beijing2008.cn/paralympic/news/news/1year/s214153770/n214154013.shtml |date=2009-03-17 }}北京奧組委，2007年9月11日</ref>。1972年畢業於[[曼徹斯特大學|曼徹斯特大學]]地理學系。已婚，並育有兩子。

他曾於1986年至1991年間擔任英國煤炭公司行政主管。

== 運動員生涯 ==
克雷文自[[1972年夏季殘奧會|1972年夏季殘奧會]]至[[1988年夏季殘奧會|1988年夏季殘奧會]]五度代表[[英國|英國]]參加[[輪椅籃球|輪椅籃球]]項目，他更於[[1972年夏季殘奧會|1972年夏季殘奧會]]參加田徑及游泳項目<ref>{{cite web |title=Athlete Search Results |year=2008 |publisher=International Paralympic Committee|url=http://www.paralympic.org/release/Main_Sections_Menu/Sports/Results/paralympics_search.html?sport=all&games=all&medal=all&npc=GBR&name=craven&fname=phil*&gender=m |accessdate=2008-10-13}}</ref>。

他曾在1973年及1975年舉行的世界輪椅籃球錦標賽中，奪得金牌及銅牌；在1971年及1974年的歐洲錦標賽兩度獲得金牌，並在1993年獲得銀牌。他並獲得1994年歐洲冠軍盃及1970年[[英聯邦運動會|英聯邦運動會]]奪得金牌。

== 體育界領袖 ==
克雷文曾三度擔任英國輪椅籃球協會主席（1977年至1980年；1984年至1987年；1989年至1994年），並先後擔任國際斯托克曼德維爾輪椅運動聯合會籃球小組委員會主席（1984年至1988年）、國際輪椅籃球協會主席（1988年至2002年）及執行長（1994年至1998年）、英國男子輪椅籃球隊執行總監（1988年至2002年）等公職。2001年至2017年擔任[[國際殘疾人奧委會|國際殘疾人奧委會]]主席。

克雷文於2003年獲委任為[[國際奧林匹克委員會|國際奧林匹克委員會]]委員，並先後出任國際奧委會體育及環境委員會成員（2002年至2004年）、[[北京奧運會|北京奧運會]]協調委員會成員（2002年至2008年），並於2005年起擔任文化及奧運教育委員會成員<ref>[http://www.olympic.org/uk/organisation/ioc/members/bio_uk.asp?id=877 "Sir Philip CRAVEN, MBE"], official website of the Olympic Movement</ref>。
