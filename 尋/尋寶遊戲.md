《'''尋寶遊戲'''》 是[[中國|中國]]歌手[[許嵩|許嵩]]個人第七張[[錄音室專輯|錄音室專輯]]，数字版於2018年7月12日正式發行，实体版则是于2018年7月14日发行<ref>{{cite web |title=许嵩新专《寻宝游戏》正式上线 |url=http://ent.163.com/18/0712/13/DMH35JBH000396AR.html#from=relevant |website=网易 |accessdate=2018-08-09}}</ref>。该专辑一共收錄九首全創作歌曲，另有4首歌曲附带有MV。该专辑在发布1天后便登上[[酷狗音乐|酷狗音乐]]飙升榜榜首<ref>{{cite web |title=许嵩登星乐坊邀粉丝寻宝互动 揭秘低调却仍人气火热的真相 |url=http://ent.huanqiu.com/yuleyaowen/2018-08/12647369.html |website=环球网 |accessdate=2018-08-09}}</ref><ref>{{cite web |title=许嵩出席《寻宝游戏》发布会 续约太合音乐集团 |url=http://ent.sina.com.cn/y/yneidi/2018-07-25/doc-ihftenia1970900.shtml |website=新浪 |accessdate=2018-08-09}}</ref>。
