'''胶东半岛'''，简称'''胶东'''，古为[[东夷|东夷]]地区。指[[中國|中國]][[山东省|山东省]]境内[[胶莱河|胶莱河]]以东地区，包括[[烟台|烟台]]、[[威海|威海]]全部和[[青岛|青岛]]的东部，是[[山东半岛|山东半岛]]的主要部分，北面与[[辽东半岛|辽东半岛]]隔[[渤海湾|渤海湾]]相望。

远古时代，大连位于海底。大约于7万年前，海平面下降50多米，[[大连|大连]]、[[朝鲜半岛|朝鲜半岛]]、[[山东半岛|山东半岛]]露出海面，形成新大陆“胶辽古陆”。大约于3万年前，海平面再次下降140多米，[[日本列岛|日本列岛]]同[[胶辽古陆|胶辽古陆]]相连。

== 历史 ==
胶东地区古为[[东夷|东夷]]，
胶东地区的先民把鸟类和太阳视为其氏族的图腾而加以顶礼膜拜，这在[[长岛|长岛]][[北庄遗址|北庄遗址]]出土的陶器及其纹饰等发现均有体现。
蚩尤战败，[[东夷|东夷]]遗民在亡国后，经过今天辽东半岛和山东半岛中间的路桥迁移至[[辽宁|辽宁]]和[[朝鲜半岛|朝鲜半岛]]，建立[[箕子朝鲜|箕子朝鲜]][[高句丽|高句丽]][[百济|百济]]。留下来的东夷人叫[[莱夷|莱夷]]后被齐国征服，在[[齐桓公|齊桓公]]治理區域下，堪稱春秋五霸之首<ref>吳曉波，歷代經濟變革得失，華品文創出版股份有限公司，2014年，15頁</ref>。
[[金|金]][[元|元]]时期，山东大部分地区历经战乱人口骤减，经济文化遭到严重破坏。唯有当时胶东地区损失较小，保留了山东的人种和文化。在明朝初年一部分[[登州|登州]]府和[[莱州|莱州]]府的居民迁移至山东西部。
[[清朝|清朝]]末年至[[民国|民国]]时期山东掀起了巨大的[[闯关东|闯关东]]热潮，胶东地区因为方便走海路去关东，有大量人民移民[[中国东北地区|东北]]。对东北的人口构成和地域文化造成了重大影响。

== 经济 ==
[[File:Reefer_containers_in_Qingdao,_China_(6955067990).jpg|缩略图]]
对于以黃河、長江為中心的人而言，胶东地区乃是天之尽头，经济长期落后于中原；然而正因為其徧隅一方，造就其在西元前八世紀春秋時代，齊桓公治下，[[管仲|管仲]]變法時期的強盛，更被現代經濟學家吳曉波稱之為中國史上第一場經濟變革<ref>吳曉波，歷代經濟變革得失，華品文創出版股份有限公司，2014年，15頁</ref>。因此在經濟理念和大胆創新方面，都可算是中西历史上的先驅，吳曉波甚至把這個經濟改革定位為"觀念的優先往往比資源的優先更為重要"<ref>吳曉波，歷代經濟變革得失，華品文創出版股份有限公司，2014年，16頁</ref>。然而正因為其地處海濱，胶东半岛的漁業和煮鹽業，在春秋時代已具備與其他諸侯國出口的貿易規模<ref>吳曉波，歷代經濟變革得失，華品文創出版股份有限公司，2014年，20頁</ref>。民国以来，胶东半岛更成為中國東北的海運重地，以其沿海港口众多，有利对外贸易，经济发展率领先山东其他地区。近年来，更通过对韩國的密切贸易，及利用侨资发展乡镇企业，人民生活有较大提高。

== 语言 ==
胶东地区使用的[[方言|方言]]为[[胶辽官话|胶辽官话]]（以烟台话、威海话为代表），与[[山东|山东]]其他地区方言有较大差别，而与[[辽东半岛|辽东半岛]]的[[大连|大连]]方言相近。正因如此，胶东地区（尤其是烟台地区）民众相互间有较强的身份认同，往往对外称自己为胶东人，而非山东人，或烟台人，威海人。

== 饮食 ==
胶东菜以[[福山|福山]]菜为代表，是[[鲁菜|鲁菜]]的一个分支，擅长烹饪海鲜，煲制鲜汤。

== 资源 ==
*烟台[[招远市|招远市]]拥有占全国储量1/3的金矿。近年又在[[莱州市|莱州市]]发现了巨大金矿。
*烟台[[栖霞市|栖霞市]]在国内率先引种了日本的富士苹果，并发展成著名的[[红富士|红富士]]品种。
*烟台[[莱阳市|莱阳市]]有特产[[莱阳梨|莱阳梨]]。
