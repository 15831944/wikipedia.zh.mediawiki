'''胶济铁路济南车站旧址'''，位于[[山东省|山东省]][[济南市|济南市]][[天桥区|天桥区]]车站街东侧，欧式建筑。站房大楼于1904年动工，1915年建成并投入使用。

== 历史 ==

1898年，[[德意志帝国|德国]]强迫清政府签订《[[胶澳租借条约|胶澳租借条约]]》，其中允许德国在山东建造由[[青岛|青岛]]经[[潍县|潍县]]、[[博山|博山]]和经[[沂州|沂州]]、[[莱芜|莱芜]]到[[济南|济南]]的两条铁路<ref>在《胶澳租借条约》第二端（铁路矿务等事）第一款规定：中国国家允准德国在山东盖造铁路二道：其一由胶澳经过潍县、青州、博山、淄川、邹平等处往济南及山东界；其二由胶澳往沂州及由此处经过莱芜县至济南府。</ref>。1904年，[[胶济铁路|胶济铁路]]自东向西铺到济南，不久由英、德两国出借款项建造的[[津浦铁路|津浦铁路]]也铺到济南。因统辖权不同，两座颇具规模的车站相距数百米独立设置，均为欧式风格<ref>{{cite web |url=http://news.sina.com.cn/o/2006-09-15/010910020308s.shtml|title=《济南：失去的还在怀念》|accessdate=2009-09-09 |work=赵晓风|publisher=人民网-华东新闻}}</ref>。
[[File:Old_Jinan_Station_05.jpg|thumb]]

20世纪30年代以前，两座火车站曾经同时使用。直到1937年底[[侵华日军|侵华日军]]占领济南时，为了配合分区管理，将两座车站进行改造和扩建，第二年将胶济铁路济南车站并入津浦铁路济南车站<ref>{{cite web|url=http://edu.beelink.com.cn/20061128/2188171.shtml|title=《消失的钟声 记济南老站》|accessdate=2009-09-09|work=济南乎博客|publisher=百灵网}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1939年两条铁路接轨，胶济铁路济南站被改为济南铁路局机关，前为停车场。后作为济南铁路分局，今为济南铁路办事处<ref>{{cite web |url=http://blog.sina.com.cn/s/blog_4af749460100bxnt.html|title=《济南胶济车站旧址》|accessdate=2009-09-09 |work=和谐之韵|publisher=新浪博客}}</ref>。1995年12月被列入济南市文物保护单位，2006年12月被列入[[山东省文物保护单位|山东省文物保护单位]]。

== 建筑 ==

该建筑为一字形平面，主入口位于中间偏东部分并稍向南突出，通往候车大厅。门洞二层为石柱廊，有六根高大粗壮的爱奥尼石柱<ref>{{cite web|url=http://www.qlwb.com.cn/display.asp?id=284051|title=《老济南光影之胶济铁路济南火车站大楼》|accessdate=2009-09-09|work=赵晓林|publisher=齐鲁晚报}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
