'''袁惟仁'''{{BD|1968年|6月24日}}，[[臺灣|臺灣]][[作詞人|作詞人]]、[[作曲人|作曲人]]。母親為[[臺東縣|臺東縣]][[卑南族|卑南族]][[南王部落|南王部落]]工藝家陳鳳榮。前妻是演員[[陸元琪|陸元琪]]<ref>[http://news.tom.com/2016-12-25/OKV9/32631858.html 袁惟仁签字离婚 妻子陆元琪个人资料].TOM新闻.2016-12-25</ref>。2018年10月17日，袁惟仁在上海录制意外跌倒，头部受伤引发[[脑溢血|脑溢血]]，被送往[[上海瑞金医院|上海瑞金医院]]治疗<ref>{{cite web |title=【上海獨家直擊】袁惟仁腦溢血女友通報　瑞金醫院：有可能半身不遂 |url=https://tw.appledaily.com/new/realtime/20181018/1449614/ |website=苹果日报 (台湾) |accessdate=2018-10-18 |date=2018-10-18}}</ref>。

== 音乐历程 ==
* 1991年12月31日，“凡人”推出第一張專輯《杜鵑鳥的黃昏》，袁惟仁發表作品〈執迷不悔〉[[粵語|粵語]]版，由[[王菲|王菲]]演唱，隔年王菲以這首歌擔任專輯主打歌進軍臺灣。
* 1993年，“凡人”推出第三张专辑《大伙听我唱支歌》，获重唱团体金曲奖。
* 1994年1月，“凡人”推出第四张专辑《心甘情愿》获重唱团体金曲奖。
* 1995年，“凡人”推出第六张专辑《难兄难弟》（这也是两人合作的最后一张专辑）。
* 1996年，袁惟仁进入[[上华唱片|上华唱片]]制作部担任制作人。
* 1996年-2004年，担任[[S.H.E|S.H.E]]、[[动力火车|动力火车]]、[[齐秦|齐秦]]、[[迪克牛仔|迪克牛仔]]、[[熊天平|熊天平]]、[[许美静|许美静]]等歌手的专辑制作人。
* 1998年，为[[那英|那英]]写下《征服》、《梦醒了》、《梦一场》等歌曲, 其中《征服》入围金曲奖最佳作词最佳作曲。
* 1998年，与多位创作人一起推出《亚洲创作人原音大碟》，袁惟仁演唱个人作品《征服》，并获当月音乐人交流协会选为最佳单曲。
* 2000年1月，推出第一张个人专辑《袁惟仁同名专辑》。
* 2002年，加盟[[华研唱片|华研唱片]]担任制作部经理。
* 2007年-2013年，擔任臺灣選秀節目《[[超級星光大道|超級星光大道]]》的評審。
* 2007年，[[SHE|SHE]]专辑《PLAY》中推出为袁惟仁制作的歌曲《听袁惟仁弹吉他》。
* 2012年8月11日，擔任滬江網第二屆《非唱不可》網路外文歌曲大賽評委。
* 2014年，擔任臺灣歌唱節目《[[我要當歌手|我要當歌手3]]》的評審。
