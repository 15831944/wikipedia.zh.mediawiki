'''袁华智'''（{{bd|1961年|10月|||}}），[[湖北|湖北]][[仙桃市|仙桃]]人，[[中国人民解放军海军少将|中国人民解放军海军少将]]。

1978年参加[[中国人民解放军|中国人民解放军]]，长期在[[中国人民解放军海军|海军]]政治工作部门服役，曾任[[中国人民解放军海军西沙水警区|西沙水警区]]政委、[[中国人民解放军海军政治工作部|海军政治部]]宣传部新闻处处长、海军陆战旅政委、《[[人民海军报|人民海军报]]》社社长、海军政治部秘书长、[[南海舰队|南海舰队]]政治部副主任。2015年12月，任[[中国人民解放军海军装备研究院|海军装备研究院]]政委<ref>{{Cite web |url = http://news.sohu.com/20160422/n445569482.shtml |title = 袁华智少将任海军装备研究院政委 |author = |date =2016-04-22 |website = 搜狐 |language = |accessdate =2019-01-23 |quote =  }}</ref>。2017年3月，任[[中国人民解放军海军陆战队|中国人民解放军海军陆战队]]政治委员<ref>{{cite news|title=海军陆战队新领导层亮相：孔军、袁华智分任司令员、政委|url=https://www.thepaper.cn/newsDetail_forward_1695787|agency=澎湃新闻|date=2016-05-27}}</ref>。2018年12月，调入[[中国人民解放军空军|空军]]任职，升任[[中国人民解放军东部战区空军|东部战区空军]]政治委员<ref>{{cite web |title=海军陆战队首任政委袁华智已任东部战区空军政委 |url=https://news.sina.cn/2019-01-15/detail-ihqfskcn7333108.d.html?cre=wappage&mod=r&loc=3&r=9&doct=0&rfunc=23&tj=none |website=新浪 |accessdate=2019-01-23 |date=2019-01-15}}</ref>。2019年3月，重返海军任副政委。

2015年7月，晋升少将军衔<ref>{{cite web |author1=王俊 |title=海军91181部队司令员陈岳琪等8名军官晋升少将军衔 |url=https://www.thepaper.cn/newsDetail_forward_1354935 |website=澎湃新闻 |accessdate=2019-01-23 |date=2015-07-21}}</ref>。
