'''袁克桓'''（{{bd|1898年||1956年}}），[[河南|河南]][[项城|项城]]人，袁克桓是[[袁世凯|袁世凯]]的第六子。是一名实业家，曾执掌多家中国民族企业。1949年后，袁克桓选择留在中国大陆。1958年，袁克桓病逝于天津。

==生平==
袁克桓年少时曾在天津的新学书院就读，1913年到[[英国|英国]]留学，后辍学回国。他曾因父亲[[袁世凯|袁世凯]]病逝而推测婚期。因母亲杨氏曾劝告袁克桓不要从政，因此他开始从事实业，从1926年开始实业家生涯，曾执掌[[开滦煤矿|开滦煤矿]]和[[启新洋灰公司|启新洋灰公司]]等企业。1949年后，他曾被提名为[[天津市|天津市]]副市长，但他本人拒绝了这一提名。1956年，袁克桓在天津去世，多名政要参加了他的葬礼<ref>{{cite web |title=袁世凯去世第六子袁克桓推迟婚期(图) |url=http://www.chinanews.com/cul/2013/10-24/5418088.shtml |website=中国新闻网 |accessdate=2019-02-20}}</ref><ref name="周岩1991">{{cite book|author=周岩|title=袁世凯家族|url=https://books.google.com/books?id=sjwf2DnNymkC|year=1991|publisher=中國青年出版社|pages=223–229}}</ref><ref>{{cite book|title=民国八大总统家世|url=https://books.google.com/books?id=EKpPAAAAIAAJ|year=1999|publisher=中国世界语出版社|isbn=978-7-5052-0405-8|pages=323–328}}</ref><ref name=Zhang2014>{{cite web |title=袁世凯孙女婿口述：袁家三代旧事 |url=http://history.people.com.cn/n/2014/1009/c372328-25793960-2.html |website=人民网}}</ref>。

==家族==
袁克桓是[[袁世凯|袁世凯]]的第六子。袁克桓的妻子是江苏[[巡抚|巡抚]][[陈启泰_(同治进士)|陈启泰]]之女陈徵，儿子有袁家宸、袁家卫，女儿有袁家英、袁家仙、袁家蕖、袁家菽、袁家芯<ref name=Zhang2014/>。
