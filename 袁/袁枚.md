'''袁枚'''（{{bd|1716年||1797年|catIdx=Y袁枚}}），[[清代|清代]]诗人、[[文學|文學]]家、[[散文|散文]]家、[[美食家|美食家]]。[[表字|字]]'''子才'''，[[号|号]]'''简斋'''，别号'''随园老人'''，時稱'''隨園先生'''，[[浙江|浙江]][[錢塘縣|錢塘縣]]（今浙江[[杭州|杭州]]）人，[[祖籍|祖籍]]浙江[[慈谿县|慈谿]]<ref name="袁枚">{{cite web|url=http://www.ziyexing.cn/shici/yuanmei/yuanmei_about.htm |title=清代诗人袁枚 |publisher= ·子夜星网站 |author= |date= |accessdate=2013年1月20日}}</ref><ref>清‧袁枚《小倉山房尺牘》〈与钱竹初书〉：「枚祖籍慈溪，為兄部民，因生長杭州，數典忘祖。」 </ref>，年廿四中進士，曾官[[溧水|溧水]]、[[江浦縣|江浦]]、[[沭陽|沭陽]]、[[江寧縣|江寧]]等地[[知縣|知縣]]，不到卅八歲即辭官還鄉，致仕之後因投資地產有道，家財萬貫。

袁枚擅長[[詩|詩]]、[[賦|賦]]、[[制藝|制藝]]，能寫[[駢文|駢文]]、[[小品文|小品文]]、[[筆記|筆記]]，乾隆時期為诗坛盟主，又為“[[清代駢文八大家|清代駢文八大家]]”、“[[江右三大家|江右三大家]]”之一，文筆亦與[[大學士|大學士]][[直隸|直隸]][[紀昀|紀昀]]齊名，時稱「南袁北紀」。其喜好廣泛，甚至編寫[[食譜|食譜]]、[[志怪小說|志怪小說]]，著有《[[小仓山房文集|小仓山房文集]]》、《[[随园诗话|随园诗话]]》、《[[子不语|子不语]]》、《[[s:祭妹文|祭妹文]]》等。書信亦有名，其《小倉山房尺牘》與[[許葭村|許葭村]]《秋水軒尺牘》、[[龔未齋|龔未齋]]《雪鴻軒尺牘》，人稱「清代三大尺牘」。

袁枚生平喜稱人善、獎掖士類，也提倡[[女性文學|女性文學]]，廣收女弟子。不喜[[理學|理學]]、[[漢學|漢學]]，追求自由，反對統一思想，他說「物之不齊，物之情也，天亦不能做主，而況於人乎？」，故被當時的許多文人嚴厲批判，袁枚依然悠哉度日，在文壇享有盛名。

== 生平 ==
先世世居[[宁波府|宁波府]][[慈谿县|慈谿县]]，六世祖[[袁茂英|袁茂英]]，明[[万历|万历]]进士，官至[[布政使|布政使]]。袁枚家貧，父親袁濱，四處擔任[[幕賓|幕賓]]為生，仰賴母親章氏針線[[女紅|女紅]]持家，袁枚發憤苦讀，十二歲（1728年）中[[秀才|秀才]]，進入縣學，鄉里視為[[神童|神童]]，「少聰穎，年十二能文，嘗作《[[高帝|高帝]]》、《[[郭巨|郭巨]]》二論，莫不異之」<ref>[[錢泳|錢泳]]《履園叢話》</ref>。十五歲補增廣生，十八歲經浙江總督[[程元章|程元章]]推薦，進入[[萬松書院|萬松書院]]學習。十九歲得浙江學政[[帥念祖|帥念祖]]賞識，拔為優等。

[[乾隆|乾隆]]元年（1736年）因長相俊秀，文筆優美，受[[廣西巡撫|廣西巡撫]][[金鉷|金鉷]]推薦應[[博學鴻儒|博學鴻儒]]科考試，「乾隆元年，先生游廣西，省其叔父於巡撫金公幕，金公奇其狀貌，命為詩，下筆千言，遂大為讚歎。」年僅二十歲，是當時年紀最小的，故未錄取。乾隆三年（1738年）中式[[舉人|舉人]]，乾隆四年（1739年）联捷[[進士|進士]]第二甲第五名，選[[翰林院|翰林院]][[庶吉士|庶吉士]]，年方廿四。[[大學士|大學士]][[史貽直|史貽直]]翻閱其文，辭采豐美，論調凌厲，讚為「當世之[[賈誼|賈誼]]」。

袁枚的文名極佳，[[翰林|翰林]][[散館|散館]]，卻不知為何未受館內青睞（传说不耐满文），被外派為[[知縣|知縣]]，曾[[縣令|令]][[溧水|溧水]]、[[江浦|江浦]]、[[沭阳|沭阳]]等县<ref>光绪六年《江宁府志》卷二十二、光绪九年《溧水县志》卷五</ref>。由於年輕任官，其父袁濱怕袁枚施政不得民意，曾私下探訪[[溧水|溧水]]縣民對袁枚的看法，結果每一個人都說「吾邑有少年袁知縣，乃大好官也。」乾隆十年（1745年）任[[江宁|江宁]]（今[[江苏|江苏]][[南京|南京]]）知縣时<ref>《小仓山房文集》卷四</ref>，推行法制，不避权贵，颇有政绩。袁枚治理地方頗有一套，他四處探訪民情，並在縣內布有大量耳目，暗中將[[土豪|土豪]][[惡霸|惡霸]]、劣紳奸人名冊紀錄，每當有案件發生，他就一一核對名冊，並與當地的地方人士合作破案，他處罰壞人，用刑謹慎，但卻會將他們的敗行劣跡寫成文章，公告在縣衙，改過三年之後，方才撕毀布告，故惡人互相警戒，不敢犯法。[[两江总督|两江总督]][[尹继善|尹继善]]稱讚他：“可谓宰相必用读书人矣！”<ref>牛应之《雨窗消意录》卷三</ref>

袁枚三十八歲即厭惡官場生活，休官養親，不复为官，于[[金陵|金陵]][[小仓山|小仓山]]筑“[[随园|随园]]”恬淡自居，蒐集书籍，创作诗文，悠闲度过五十年。他在给友人[[程晋芳|程晋芳]]的信中说：“我辈身逢盛世，非有大怪癖、大妄诞，当不受文人之厄。”游历南方诸名山，与诗友交往。

八年任官，頗儲俸祿，加上文名在外，時常為各地[[富豪|富豪]]、[[世族|世族]]寫跋撰序、作[[墓誌銘|墓誌銘]]，潤筆所得甚多，致仕後藉以投資，理財有道，盛置[[田產|田產]]，招募大量[[佃農|佃農]]，蓄積數萬[[兩銀|兩銀]]，以財自娛，但為人仗義疏財，在好友[[程晉芳|程晉芳]]死時，焚毀程晉芳所欠白銀五千兩的借據，不但不向其親人索討，還拿了一筆錢餽贈程晉芳的遺孤<ref>《清史稿·文苑传二·袁枚》：“举借券五千金焚之，且卹其孤焉”</ref>。晚年過世時，留下了“田產萬金，餘銀兩萬”，交付經營，以餘息供家族後人讀書營生。

== 文學 ==
[[File:致西崖书札.jpg|缩略图]]
袁枚以[[诗|诗]]名闻当世，创作讲求性情个性，提倡“[[性灵说|性灵说]]”，反对清初以来[[拟古|拟古]]和形式主义的流弊，使诗坛风气为之一新，与[[蒋士铨|蒋士铨]]、[[赵翼|赵翼]]并称“[[江右三大家|江右三大家]]”，文筆又與[[纪晓岚|纪晓岚]]齊名，時稱「南袁北紀」。袁枚[[駢文|駢]][[散文|散]]皆通，為[[清代駢文八大家|清代駢文八大家]]之首，文章通發性情，兼取[[六朝|六朝]]駢儷，「[[駢文|駢]][[散文|散]]合一」，較[[桐城派|桐城派]]通達，亦提倡[[女性文學|女性文學]]。

下列其[[古詩|古詩]]《所见》。

<blockquote>
牧童骑黄牛，<br>
歌声振林樾。<br>
意欲捕鸣蝉，<br>
忽然闭口立。<br>
</blockquote>

1912年11月24日[[德語文學|德語文學]]大師[[卡夫卡|卡夫卡]]在寫信給當時的未婚妻菲莉斯.鮑威爾時，引用了袁枚的《寒夜》<ref>卡夫卡全集8.83-87頁</ref>：

<blockquote>
寒夜读书忘却眠，<br>
锦衾香尽炉无烟。<br>
美人含怒夺灯去，<br>
问郎知是几更天。
</blockquote>

袁枚《小倉山房尺牘》、[[許葭村|許葭村]]《秋水軒尺牘》、[[龔未齋|龔未齋]]《雪鴻軒尺牘》，人稱「清代三大[[尺牘|尺牘]]」。

袁枚收錄了許多[[鬼故事|鬼故事]]，以簡潔明暸的筆法，寫成[[筆記小說|筆記小說]]《[[子不語|子不語]]》，與[[紀曉嵐|紀曉嵐]]《[[閱微草堂筆記|閱微草堂筆記]]》齊名，兩人也曾在各自的著作中提起對方。

袁枚也是一位[[美食家|美食家]]，写有著名的《[[随园食单|随园食单]]》，是清朝一部系统地论述[[烹饪|烹饪]]技术和南北菜点的重要著作。该书出版于乾隆五十七年〈1792年〉。

== 家族 ==
袁枚之妹[[袁機|袁機]]亦甚具學識，《[[如皋縣誌|如皋縣誌]]》、《[[杭州府志|杭州府志]]》、《[[清史稿|清史稿]]·列女傳》皆有傳。袁機於乾隆二十四年（1759年）卒，袁枚八年後寫有〈祭妹文〉<ref>〈[[s:祭妹文|祭妹文]]〉首言：「乾隆丁亥冬，葬三妹素文於上元之羊山而奠以文」。</ref>。

正室王氏無子，[[過繼|收繼]]袁枚姪子[[袁通|袁通]]為子。

[[側室|側室]]陶姬，生一女。

袁枚六十餘歲時，[[側室|側室]]鍾氏生子，袁枚賦詩曰：「六十生兒太覺遲，即將遲字喚吾兒。」，取名[[袁遲|袁遲]]。'''袁遲'''（1779－1828），字'''真来'''，知名畫家，著有《画林新咏》等，袁遲生有三子。

==墓葬==
[[袁枚|袁枚]]在[[小仓山|小仓山]]筑[[随园|随园]]，题其室曰“小仓山房”，著《[[小仓山房文集|小仓山房文集]]》<ref name="小仓山房文集">{{cite wikisource |title=小仓山房文集 |wslink=小仓山房文集 |wslanguage=zh |author1=袁枚|authorlink=袁枚 |location=南京 |quote=逮乾隆癸酉馆金陵，谒先生于随园之小仓山房。……金陵自北门桥西行二里，得小仓山。山自清凉胚胎，分两岭而下，尽桥而止。……登小仓山，诸景隆然上浮。凡江湖之大，云烟之变，非山之所有者，皆山之所有也。}}</ref>。其墓亦坐落于此，碑文“皇清诰授奉政大夫显考袁简斋之墓”为[[姚鼐|姚鼐]]所撰<ref name="石城山志">{{cite wikisource |title=石城山志 |wslink=石城山志/卷1 |wslanguage=zh |author1=陈诒绂|authorlink=陈诒绂|year=1917 |location=南京 |quote=而北上，为小仓山，袁简斋太史枚之墓在焉。墓碑为姚惜抱先生所撰书也。}}</ref>，民国时尚存<ref>{{cite book|author1=朱偰|title=金陵古迹图考|year=1936|publisher=商务印书馆|accessdate=2017-03-19|quote=旧事迹荡然，仅有清袁随园先生墓碑，当干河沿南山坡上，袁子才先生祠，破椽一楹，当干河沿北；今又筑为马路，随园旧迹，扫地尽矣。}}</ref><ref>{{cite book|author1=黄裳|title=金陵五记|year=1983|publisher=金陵书画社}}</ref>，但在[[文化大革命|文化大革命]]时被毁坏，仅残留碑址。1974年3月18日修建[[五台山体育馆|五台山体育馆]]时，袁枚墓被全面清理。

== 逸事 ==
据[[葛建初|葛建初]]《折狱奇闻》记载，袁枚任江宁知县期间，曾巧断一起“裁缝之女奸杀案”。有一名裁縫之女兒被姦殺，地上有一截被咬斷的舌肉，[[衙役|衙役]]捕到一名舌傷的[[貨郎|貨郎]]，並要[[斬首|斬首]]示眾。袁枚認為，兇手如在[[舌吻|舌吻]]過程被咬斷舌頭，必然疼痛難忍，不可能再繼續強姦下去。所以袁枚謊稱死者[[託夢|託夢]]，即將透過[[纏足|纏足]]布顯靈，他要公開「審判那塊布」，吸引大批民眾圍觀。此時，袁枚逼迫每一個圍觀的鄉民都必須去觸碰那塊布，有兩個男子惶然不安，袁枚立刻加以拘捕，一問之下才得知，這兩名男子知貨郎因強吻少女，被咬斷舌頭之後，就跑去姦殺那名少女，心想可以嫁禍給那名貨郎，聽說「審布」奇聞，趕來觀看，誰知袁[[縣令|縣令]]智謀特高，只好甘心認罪<ref>[[葛建初|葛建初]]《折狱奇闻》</ref>。

袁枚與[[桐城派|桐城派]][[姚鼐|姚鼐]]因創作理論互相牴觸，所以被姚鼐攻擊為“詩家之惡派”，但其實袁與姚私下是好友。袁枚過世後，姚鼐深感惋惜，還不顧眾人反對，寫下《袁隨園君墓誌銘并序》，成為文壇佳話。

袁枚喜愛聲色，夜夜[[青樓|青樓]]，[[劉成禺|劉成禺]]《[[世載堂雜憶|世載堂雜憶]]》記載，袁枚曾被嶺南詩家[[黎二樵|黎二樵]]批評為「大嫖客」而不見，劉成禺稱：「故《隨園詩話》中，無及二樵者。」袁枚的好友[[趙翼|趙翼]]曾揶揄袁枚「占人間之艷福」、「引誘良家子女」等，還消遣他，說如有來生，袁枚會投胎成蜜蜂、蝴蝶，可以日日採花。袁枚對喜歡風月一事無所忌憚，還推崇古代名妓[[蘇小小|蘇小小]]，刻了一枚印章，寫著「錢塘蘇小小是鄉親」，四處蓋印。

袁枚為人放蕩不羈，又公開反對[[儒家|儒家]]、[[理學|理學]]許多封閉的思想，當時盛傳[[刘墉|刘墉]]擔任[[江寧府|江寧府]][[知府|知府]]時，認為袁枚败坏风俗，一度蒐集證據，要將他法辦，甚至[[斬首|斬首]]，但當時[[朱竹均|朱竹均]]出来講情，刘墉才沒有下手。也有人說刘墉並非要將袁枚殺死，只是要驱逐他，但是袁枚之恩师[[尹继善|尹继善]]勸阻劉墉，劉墉才停止。而後袁枚稱頌刘墉“月无芒角星先避，树有包容鸟亦知”，刘墉一生都能背誦。不過，當事人袁枚澄清，劉墉並未與他結怨，此事子虛烏有。
