'''袁一琦'''（{{bd|2000年|3月19日|catIdx=Yuan, Yiqi}}），中國偶像團體[[SNH48|SNH48]]的成員，所屬隊伍Team HII。於2016年9月15日的中秋特別公演中出道，成為SNH48的7期生<ref name="snh481">{{Cite web |url = http://www.snh48.com/html/allnews/zixun/2016/0919/1941.html |title = SNH48七期生首批成员正式出道 “小鲜花”首度登台亮相 |publisher = SNH48 }}</ref>。2016年10月15日，於《[[Team_B_4th_Stage「偶像的黎明」#SNH48_Team_HII_3rd_Stage「偶像的黎明」公演|偶像的黎明]]》公演進行首演。於2019年1月19日成為全新7人小組合的成員。

== 經歷 ==
2016年9月15日，在中秋特別公演中出道，成為SNH48的7期生。10月15日，於《[[Team_B_4th_Stage「偶像的黎明」#SNH48_Team_HII_3rd_Stage「偶像的黎明」公演|偶像的黎明]]》公演首演。

2017年1月7日，參與「[[SNH48第三屆年度金曲大賞BEST_50|SNH48第三屆年度金曲大賞BEST 50]]」，演出Team HII 隊歌《下課鈴聲》。7月29日，於SNH48第四屆總選舉，最終獲得17,418票，排名第56名<ref>[http://www.snh48.com/html/allnews/zixun/2017/0730/2456.html SNH48 GROUP第四届总决选落幕 鞠婧祎问鼎年度人气王]</ref>。

2018年2月3日，參與「[[SNH48_Group第四屆年度金曲大賞BEST_50|SNH48 Group第四屆年度金曲大賞BEST 50]]」，演出Team HII 隊歌《花與火》以及unit曲《再愛一回》<ref>[http://www.snh48.com/html/allnews/zixun/2018/0203/2883.html SNH48 首次大重组各队成员名单公布 分层级打造彰显新格局]</ref>。4月7日，參與SNH48首屆偶像運動會，獲得跳高銀牌和接力折返跑金牌<ref name="snh482">{{Cite web |url = http://www.snh48.com/html/allnews/zixun/2018/0407/2987.html |title = SNH48首届偶像运动会落幕 热血竞技展青春 |publisher = SNH48 }}</ref>。

2019年1月19日，於「[[SNH48_Group第五屆年度金曲大賞BEST_50|SNH48 Group第五屆年度金曲大賞BEST 50]]」中宣布以第五名的成績成為全新7人小組合的成員。4月12日，參與SNH48 GROUP 第二屆偶像運動會，獲得跳高金牌<ref name="snh483">{{Cite web |url = http://www.snh48.com/html/allnews/zixun/2019/0414/3875.html |title = 《加油女孩》 SNH48 GROUP第二届偶像运动会荣耀时刻！ |publisher = SNH48 }}</ref>。

== 人物 ==
* 喜歡：聲樂、滑板、電子競技<ref name="snh48official" />。
* 個人特長：唱歌、頭髮特長、電子競技<ref name="snh48official" />。
* 喜歡玩《[[守望先鋒|守望先鋒]]》，曾取得超過4000分，因此被粉絲稱為4000分少女。
* 曾與成員參與《[[穿越火線|穿越火線]]》沙海生存之夜活動，於活動中取得個人賽、雙人賽以及團隊賽三料冠軍<ref name="吃雞">{{Cite web |url = http://k.sina.com.cn/article_6312994078_17848a51e0010037c7.html |title = 想不到SNH48的袁一琦竟然是CF吃鸡老司机，一口气拿三个冠军！ |publisher = 新浪看點 }}</ref>。
* 關係親近的成員有[[許楊玉琢|許楊玉琢]]、張昕、徐晗、[[孫珍妮|孫珍妮]]、[[洪珮雲|洪珮雲]]、祁靜、陳倩楠等
