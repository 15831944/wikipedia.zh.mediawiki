'''袁彌明'''（{{lang-en|Erica Yuen Mi-ming}}；{{bd|1980年|9月28日|catIdx=Yuan袁彌明}}）是[[袁彌明生活百貨|袁彌明生活百貨]]（[[彌明生活百貨|彌明生活百貨]]前身）創辦人，曾參與電影及電視幕前演出。曾經擔任任[[香港政黨|香港政黨]][[人民力量|人民力量]]主席、人民力量執委會委員、[[電訊盈科|電訊盈科]][[練習生|見習]][[經理|經理]]、[[時事評論員|時事評論員]]、[[節目主持|節目主持]]、[[作家|作家]]、[[藝員|藝員]]、[[模特|模特]]等，早年於[[香港|香港]][[赤柱|赤柱]][[聖士提反書院附屬小學|聖士提反書院附屬小學]]（1992年畢業）以及[[聖士提反書院|聖士提反書院]]畢業（1997年中五），之後到美國[[麻薩諸塞州|麻薩諸塞州]]Northfield Mount Hermon School就讀預科（1999年畢業），並入讀[[美國|美國]][[波士頓|波士頓]][[塔夫茨大學|塔夫茨大學]][[經濟學|經濟]]系，於2003年畢業，祖籍[[浙江|浙江]][[寧波|寧波]][[鄞縣|鄞縣]]，是[[鄞县袁氏|西門袁氏]][[家族|家族]]的後人，祖父是[[袁勃|袁勃]]，父親是[[袁弓夷|袁弓夷]]<ref name="jobmarket">{{cite web |title=轉型闖入零售戰團 袁彌明 遇強愈強|publisher=JobMarket 求職廣場|url=http://www.jobmarket.hk/Resources/industry_trend_details.jsp?article_id=3698
|language= zh-hant|date=2010-04-09}}</ref><ref name="xpweekly">{{cite journal|url=http://www.xpweekly.com/share/xpweekly/0427/ent/20061026ent0050/content.htm |journal=[[快周刊|快周刊]] |issue=427 |date=2006-10-26 |accessdate=2016-02-11 |language=zh-hant |title=What's The BIG Deal? 袁彌明 }}</ref>。新民黨政策總裁[[袁彌昌|袁彌昌]]是她的胞兄，新民黨常務副主席[[容海恩|容海恩]]則是她的嫂子。

在2005年參加[[香港小姐|香港小姐]]競選，參選期間被視為三甲大熱門<ref>[http://the-sun.on.cc/channels/ent/20050820/20050819234150_0003.html 港姐今晚決戰 整容傳聞纏身 克勤踩多腳：唔認得(8)葉翠翠]</ref>，於準決賽前獲得「旅遊大使獎」<ref>[http://ent.sina.com.cn/s/h/2005-07-11/1012776203.html 組圖：港姐首個獎項產生 袁彌明獲旅遊大使獎]</ref>，並於決賽晉身五強但大熱倒灶三甲不入<ref>[https://hk.entertainment.appledaily.com/entertainment/daily/article/20050904/5195120 港姐生電王是非不遜葉翠翠 袁彌明父欠債3億兩度清盤]</ref>。2007年離開[[無綫電視|無綫電視]]，其後為報社和雜誌社寫專欄和錄[[YouTube|YouTube]]短片教美容，迅速創立護膚品和健康食品專門店「袁彌明生活百貨」([http://www.mimingmart.com/ 網站])<ref>{{youtube|UMH2Vd9PnEM|袁彌明生活百貨開張}}，上傳日期：2009年8月3日。</ref>。2010年因不滿[[民主黨_(香港)|民主黨]]支持政改方案而成立政治組織[[選民力量|選民力量]]並成為[[人民力量|人民力量]]執行委員。2012年與[[陳志全|陳志全]]（慢必）合組名單代表人民力量參選[[香港立法会|立法會]]（[[新界東|新界東]]）地區直選，獲得38,042票，令拍檔陳志全成功當選。

2016年9月10日，袁彌明宣佈退任人民力量主席，原因過去在社區解釋議會拉布，以及推動全民退保的工作做得不理想<ref>[https://hk.news.yahoo.com/%E8%A2%81%E5%BD%8C%E6%98%8E%E8%BE%AD%E4%BB%BB%E4%BA%BA%E6%B0%91%E5%8A%9B%E9%87%8F%E4%B8%BB%E5%B8%AD-%E9%99%B3%E5%BF%97%E5%85%A8%E6%9A%AB%E4%BB%A3%E8%81%B7%E5%8B%99-113300475.html 袁彌明辭任人民力量主席　陳志全暫代職務][[星島日報|星島日報]] 2016年9月10日</ref><ref>[https://www.thestandnews.com/politics/%E4%BA%BA%E6%B0%91%E5%8A%9B%E9%87%8F%E6%98%8E%E5%AE%A3%E4%BD%88%E4%BA%BA%E4%BA%8B%E8%AE%8A%E5%8B%95-%E5%82%B3%E8%A2%81%E5%BD%8C%E6%98%8E%E8%BE%AD%E4%BB%BB%E4%B8%BB%E5%B8%AD/ 人民力量明宣佈人事變動　傳袁彌明辭任主席][[立場新聞|立場新聞]] 2016年9月9日</ref>。

2018年，於網上撰寫文章直言女人追求虛榮是一場災難。並表示她們容易被表面風光支配自我價值<ref>[https://www.mings-fashion.com/%E8%A2%81%E5%BD%8C%E6%98%8E-%E7%B5%90%E5%A9%9A-%E6%84%9B%E6%83%85-7043/ 虛榮女的命運！袁彌明：追求虛榮的人，是我最看不起的。]</ref>。

亦不時分享個人護膚心得<ref>[https://www.mings-fashion.com/%E8%A2%81%E5%BD%8C%E6%98%8E-%E7%9A%AE%E8%86%9A%E6%95%8F%E6%84%9F-%E6%BF%95%E7%96%B9-11773/ 當時裝編輯遇上袁彌明　敏感肌膚出trip急救法]</ref>。

== 簡歷 ==
袁彌明在學時曾兼任專業[[模特兒|模特兒]]，1997年她參與電影《[[初戀無限Touch|初戀無限Touch]]》的演出，飾演[[陳曉東|陳曉東]]的妹妹、[[許冠英|許冠英]]的女兒，是個有同性戀傾向的女中學生。1998年，她參演《PR Girls青春援助交際》<ref>[http://hkmdb.com/db/movies/view.mhtml?id=8393&display_set=big5 PR Girls青春援助交際 (1998)] 香港影庫</ref>。

2005年，由前歌手好友[[陳苑淇|陳苑淇]]（前稱陳逸璇）提名參選[[香港小姐|香港小姐]]選舉，並進入最後五強，最終獲得「旅遊大使獎」<ref>{{Cite web |url=http://tvcity.tvb.com/special/misshk2005/profile/18.html |title=2005年度香港小姐競選候選佳麗個人檔案 |accessdate=2006-10-23 |archiveurl=https://web.archive.org/web/20061105092948/http://tvcity.tvb.com/special/misshk2005/profile/18.html |archivedate=2006-11-05 |deadurl=yes }}</ref>。

2007年離開TVB後，在《[[蘋果日報|蘋果日報]]》和《[[忽然一周|忽然一周]]》撰寫專欄。其後又自拍[[化妝|化妝]]美容短片上網，大收宣傳之效，於是在2009年8月創立護膚品和健康食品專門店「袁彌明生活百貨」<ref>[http://www.mimingmart.com/ 袁彌明生活百貨]</ref>。

2008年6月，推出親自執筆的第一本著作《反．串》。同時，夥拍[[蕭定一|蕭定一]]及[[愛明|愛明]]於[[MyRadio|MyRadio]]主持娛樂新聞評論節目《[[大娛樂家|大娛樂家]]》，亦曾經主持[[MyRadio|MyRadio]]另一個女性節目《[[Lady_First|Lady First]]》，並且在[[亞洲電視|亞洲電視]]客席主持節目《[[生活著數台|生活著數台]]》。另外她及她的兄長[[袁彌昌|袁彌昌]]亦有時以時事評論員身份擔任[[香港電台|香港電台]]時事評論節目《[[左右紅藍綠|左右紅藍綠]]》針砭時弊<ref>[http://www.rthk.org.hk/rthk/tv/pentaprismII/20081208.html 短加長減，賠了溢價又折客]</ref><ref>[http://www.rthk.org.hk/rthk/tv/pentaprismII/20081209.html 評港府處理滯泰港人事件]</ref>。

2010年9月起，為[[香港人網|香港人網]]主持時事評論節目《[[網想最大黨|網想最大黨]]》。同時，她與[[陳志全|陳志全]]（慢必）、[[林雨陽|林雨陽]]、[[劉嘉鴻|劉嘉鴻]]（劉嗡）、[[歐陽英傑|歐陽英傑]]（星屑醫生）組織非政黨社運組織——[[選民力量|選民力量]]，準備在2011年[[香港區議會|區議會選舉]]中狙擊[[民主黨_(香港)|民主黨]]。<ref>[http://powervoters.wordpress.com/ 選民力量]</ref>但她在提名期前因私人理由棄選。

2012年5月，袁彌明在香港人網宣佈放棄[[美國國籍|美國國籍]]以便參選[[立法會|立法會]]。7月15日在「人民力量陳志全袁彌明見證大會」上宣布和[[陳志全|陳志全]](慢必)合組名單參選[[2012年香港立法會選舉|2012年香港立法會選舉]]的新界東地區直選。<ref>[http://www.pp2012.hk/PW/nte.php 新界東競選網站 ] {{webarchive|url=https://web.archive.org/web/20120813020818/http://www.pp2012.hk/PW/nte.php |date=2012-08-13 }}</ref>該參選宣言一星期有接近十萬收看<ref>[http://www.youtube.com/watch?v=EkDNBataCp4&feature=related 袁彌明立法會（新界東）參選宣言 ]</ref><ref>[http://www.pp2012.hk/PW/nte.php 新界東競選網站] {{webarchive|url=https://web.archive.org/web/20120813020818/http://www.pp2012.hk/PW/nte.php |date=2012-08-13 }}</ref>。該名單於在同選舉中取得一席，成功把名單首位的陳志全送入立法會。

2013年7月17日，袁彌明當選[[人民力量|人民力量]]主席，接替[[劉嘉鴻|劉嘉鴻]]。

袁彌明已婚，丈夫為[[選民力量|選民力量]]創黨主席、[[謎米香港|謎米香港]]前行政總裁[[林雨陽|林雨陽]]。

=== 與TVB關係 ===
2007年10月，袁彌明公開被[[電視廣播有限公司|無綫電視]]（TVB）解約<ref>{{cite news|title=袁彌明寧賠錢誓與無線解約|url=http://paper.wenweipo.com/2007/10/23/EN0710230007.htm|publisher=文匯報|language=zh-hant|deadurl=yes|archiveurl=https://web.archive.org/web/20071124074117/http://paper.wenweipo.com/2007/10/23/EN0710230007.htm|archivedate=2007-11-24}}</ref>。她曾接受了電台訪問透露被香港電視廣播有限公司製作資源部總監[[樂易玲|樂易玲]]強迫簽約，因拒絕簽約而與TVB關係變得惡劣，有指合約內容儼如不平等條約，極度苛刻，內容是簽一份3年合約，每年騷錢只加100元，沒有底薪，TVB更是她的全球獨家經理人。她本想拿合約返家與家人商量，豈料有人跟她說「這份合約不能影印，不能拿出房門口」和「你一定要簽這份合約，否則不能走出這個房間」等恐嚇性字句，她自覺猶如被威嚇，所以她不要這種經理人。後來與家人商量後拒絕簽約，發生很多不愉快事件。由於TVB指與袁彌明的合約到2009才屆滿，袁向無綫發出律師信問題才獲解決。{{fact|time=2012-05-28}}

袁彌明經常在公開場合頂撞無綫，例如在2012年5月，政府正準備發出3個新的免費電視牌照，惟在該時候無綫曾建議政府重新審視發牌的必要性，並質疑香港[[廣告|廣告]]市場是否足夠容納3家新的免費電視台。5月27日，袁與慢必公開在[[旺角|旺角]]街頭高舉紙牌，促請政府盡快發牌，讓市民享有更多免費電視頻道節目。2人公開力斥無綫獨攬市場，迫市民看TVB製作的「垃圾劇」，指劇集質素差。<ref name="ent.sina">{{cite news|title=袁彌明狂轟TVB|url=http://ent.sina.com.hk/news/2812/4/1/812013/1.html|publisher=明報|date=2012-05-28}}</ref>袁彌明與無綫鬧翻多年，因此一直被無綫列入黑名單，不過在2017年她接受了[[無綫財經台|無綫財經台]]專訪，談論網上營商之道。<ref name="ent.sina"/>

2013年，袁彌明批評無綫在6時半新聞抽起「[[陳凱琳|陳凱琳]]握17萬選票撐[[普選|普選]]促特首聽[[民意|民意]]」的訪問是[[自我審查|自我審查]]。<ref name="大紀元">[http://www.epochtimes.com/b5/13/9/4/n3956082.htm%E3%80%8C%E6%99%AE%E9%81%B8%E6%B8%AF%E5%A7%90%E3%80%8D17%E8%90%AC%E7%A5%A8vs%E6%A2%81%E6%8C%AF%E8%8B%B1689%E7%A5%A8.html 「普選港姐」17萬票vs梁振英689票], [[大紀元|大紀元]], 2013年9月4日</ref>

=== 社會責任 ===
袁彌明披露，當年參選[[港姐|港姐]]時，無線多番提醒她們不講政治，以免惹起爭議。但她強調港姐不是花瓶，應該肩負社會責任，亦希望港姐要充份利用這個機會做些有益的事情。「港姐要履行的社會責任是很大的，不是只是說一個漂漂亮亮的女子做了冠軍，然後去拍戲，但對社會的貢獻是不夠。她選出來都有很多傳媒關注，我想她的言論會影響很多人，我想不要浪費這個機會了。」<ref>{{youtube|HlEuDeJR5P4|袁彌明港姐志願}}：渴望成為企業家，賺來的利潤可以用來貢獻社會。</ref>

=== 參選立法會 ===
2012年5月，袁彌明在香港人網宣佈放棄美國國籍。7月15日在「人民力量陳志全袁彌明見證大會」上和陳志全（慢必）宣布參選2012年立法會選舉。在立法會選舉前，袁彌明製作了幾集「政治BB班」影片，簡單介紹香港政治狀況。第一集一星期亦有接近十萬收看。

==感情生活==

2010年3月，袁彌明和男友聯名在灣仔買樓，並透露男友未夠30歲。<ref>{{cite news|url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20100301/13772600 |title=袁彌明聯名買樓試婚|publisher=蘋果日報 |date=2010-03-01}}</ref>

2011年3月，袁彌明和圈外從事金融界的男友拍拖三年後，因性格不合分手。<ref>{{cite news|url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20110324/15102853 |title=袁彌明與金融才俊分手|publisher=蘋果日報 |date=2011-03-24}}</ref>

2014年12月27日與拍拖一年多，曾任職「謎網」行政總裁，公民黨創黨成員的男友[[林雨陽|林雨陽]]舉行婚禮。
