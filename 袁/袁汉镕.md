'''袁汉镕'''{{BD|1933年|11月|||catIdx=Y}}[[广东人|广东人]]，[[华沙大学|华沙大学]]数学物理系硕士，中国大陆物理学家。<ref>{{cite web |url=http://ent.sina.com.cn/2004-11-10/1503561369.html |title=看，文坛这匹害群之马？ |accessdate=2004 |publisher=新浪网 |work= |language=zh-hans}}</ref>

==生平==
1933年，袁汉镕出生在[[广东省|广东省]][[潮州市|潮州市]]。

袁汉镕在[[华沙大学|华沙大学]]数学物理学得到[[硕士|硕士]]学位。

袁汉镕曾担任[[中国原子能科学研究室|中国原子能科学研究室]]研究员。

==家庭==
袁汉镕的妻子[[易丽君|易丽君]]是[[波兰文学|波兰文学]]翻译家。<ref>{{cite web |url=http://ent.sina.com.cn/2004-11-10/0637560683.html |title=翻译家夫妇为《费尔迪杜凯》吵架 |accessdate=2004 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://news.sina.com.cn/o/2010-04-12/020717355581s.shtml |title=“波兰人民很坚强” |accessdate=2010 |publisher=新浪网 |work= |language=zh-hans}}</ref>
