'''袁誉柏'''（{{bd|1956年|5月}})，[[湖北省|湖北省]][[公安縣|公安縣]]人。[[中国人民解放军海军中将|中国人民解放军海军中将]]。[[中国共产党第十九届中央委员会委员列表|中共第十九届中央委员]]。

== 略歷 ==
1968年9月至1972年6月，就讀於湖北省公安縣第二中學初、高中部。1974年至1976年2月，參加湖北省公安縣基本路線教育工作隊任專職隊員。

1976年2月，參加中國人民解放軍。3月入學[[青島潛艇學院|青島潛艇學院]]，進行基礎訓練、接受7個月専門訓練。1977年1月，學院畢業後，配屬[[青島市|青島]]潛艇基地第11艇員隊。1978年至1981年，青島潛艇學院學習。1981年，任青島潛艇基地艇員隊魚雷長。1982年，任青島潛艇基地司令部魚雷業務長。同年、同基地繼續擔任實習副艇長、就任某[[潛艇|潛艇]]副艇長。1990年2月，任青島潛艇基地某潜艇艇長。

2003年12月，任[[中國人民解放軍海軍潛艇第1基地|中國人民解放軍海軍潛艇第1基地]]（[[核動力潛艇|核動力潛艇]]基地，[[青島市|青島市]][[崂山区|崂山区]]）參謀長。2007年6月，任同基地司令員。2010年10月，任[[北海艦隊|北海艦隊]]參謀長、北海艦隊黨委常委會委員。2013年7月，任"112編隊"任務部隊指揮員、北海艦隊副司令員。2014年7月，任[[濟南軍區|濟南軍區]]副司令員兼北海艦隊第15任司令員<ref>{{cite news |title = 北海舰队半年两换帅 袁誉柏履新司令_财新网 |url = http://china.caixin.com/2014-07-10/100702062.html |accessdate = 2014年12月9日}}</ref>。2016年1月，任新建立的[[中国人民解放军北部战区|中国人民解放军北部战区]]副司令员兼首任[[中国人民解放军北部战区海军|北部战区海军]]司令员<ref>{{cite news |title=北海舰队司令员袁誉柏任北部战区副司令员兼战区海军司令员 |url = http://www.thepaper.cn/newsDetail_forward_1446278 |accessdate=2016-03-20 |publisher = 澎湃新闻}}</ref>。

2017年1月，任[[中国人民解放军南部战区|中国人民解放军南部战区]]司令员，為首位担任战区司令员的海军将领<ref>{{cite news|title=袁誉柏升任南部战区司令员，系首位担任战区司令员的海军将领|url=http://www.thepaper.cn/newsDetail_forward_1606030|accessdate=2017-01-25|newspaper=澎湃新闻}}</ref>。

2008年7月，晋升为[[海軍少將|海軍少將]]。2015年7月，晋升为[[海军中将|海军中将]]<ref>{{cite web |url = http://news.qq.com/a/20150721/041303.htm |title = 北海舰队两任司令员晋升中将 |accessdate=2015-07-21 |publisher=腾讯网}}</ref>。
