'''袁潔儀'''（英文名：'''Kiwi Kit Yee Yuen'''，{{bd|1970年|10月24日||}}<ref name="artiste.tvb.com">[http://artiste.tvb.com/main/artistedetail/yuenkityee 袁潔儀 Yuen Kit Yee - TVB藝人資料 - tvb.com]</ref>），現為[[香港|香港]][[無綫電視|無綫電視]]基本藝人合約女藝員及前[[亞洲電視|亞洲電視]]女藝員，參加1989年亞洲小姐後加入已終止廣播的[[亞洲電視|亞洲電視]]。曾在2003年短暫效力[[無綫電視|無綫電視]]，並於2016年5月重返至今，暱稱'''潔儀姐姐'''。

== 履歷 ==
1989年，袁潔儀在[[無綫電視|無綫電視]]特輯客串[[許冠傑|許冠傑]]《最喜歡你》部份的模特兒。同年參選“[[亞洲小姐競選|亞洲小姐競選]]”，同屆包括有亞軍[[萬綺雯|萬綺雯]]、季軍[[趙靜儀|趙靜儀]]、冠軍[[翁虹|翁虹]]和第五名[[伍詠薇|伍詠薇]]，獲得第四名，賽后加入[[亞洲電視|亞洲電視]]。最初，她兼任演員及主持兩棲發展。

1991年，亞視着力培養新人，點名力捧“[[十二星道|十二星道]]”，袁潔儀也是其中一員。因此，雖出道不久，便主演過《[[藍月亮|藍月亮]]》、《[[赤子雄風|赤子雄風]]》等劇。同時，她主持的兒童節目《[[精靈一族|精靈一族]]》，還有她參與的《[[歷史也瘋狂|歷史也瘋狂]]》，都受到家庭觀眾歡迎。

1990年代后期，因亞視自製劇減少，袁潔儀將工作重點放在主持《[[今日睇真D|今日睇真D]]》上，偶爾參演《[[我來自潮州|我來自潮州]]》、《[[我和殭屍有個約會|我和殭屍有個約會]]》等劇。

2003年后，袁潔儀轉投無綫電視，在此短暫停留后至2007年離開影視圈，近年在[[香港|香港]]和[[中國大陸|中國大陸]]間做買賣生意，亦有在[[酒吧|酒吧]]等[[公共場所|公共場所]]表演或主持工作。<ref>[http://www.xmnn.cn/dzbk/xmsb/20070902/200709/t20070902_303383.htm 主持人袁潔儀昨日來廈獻藝]</ref>

2013年，接受亞視節目《[[亞姐百人|亞姐百人]]》訪問，透露幾年前她嫁予從事有機肥料製造的丈夫而移居[[廣州|廣州]]，現在是全職家庭主婦。<ref>[http://v.youku.com/v_show/id_XNTQ4MzU3ODg0.html 【亞姐百人】袁潔儀]</ref>

2014年4月，宣佈復出主持亞洲電視全新婦女節目《[[我愛下午茶|我愛下午茶]]》，擔任主持之一<ref>[https://hk.news.yahoo.com/%E8%A2%81%E6%BD%94%E5%84%80-%E5%8F%AB%E8%80%81%E5%85%AC%E6%90%B5%E5%B0%8F%E5%9B%9B-223000024.html 袁潔儀 叫老公搵小四] 星島日報，2014年4月30日</ref>。

2016年2月6日，在[[亞洲電視欠薪及不獲續牌事件|亞視欠薪事件]]下，由於亞視未有在法定限期前發放前兩個月薪金，袁潔儀聯同另外四名亞視藝員引用《[[僱傭條例|僱傭條例]]》第10A條，終止與亞視的藝人僱傭合約關係<ref>{{cite web|title=亞視拖糧員工彈期票　蔡國威袁潔儀終止合約|url=http://hk.on.cc/hk/bkn/cnt/news/20160206/bkn-20160206173734629-0206_00822_001.html}}</ref>。而且亞視在4月2日終止廣播，因此她再次離開亞視。並於同年5月1日再次回巢無綫電視簽約一年<ref>{{cite web|title=曾反叛今回報　袁潔儀賺到錢請媽咪飲茶|url=http://hk.on.cc/hk/bkn/cnt/entertainment/20160507/bkn-20160507233800549-0507_00862_001.html?eventsection=hk_entertainment&eventid=402883474ddd0ef9014dfbab919f3b6a}}</ref>，另一位同期加盟的亞視藝人是[[蔡國威|蔡國威]]。

2018年，於無綫電視重頭劇《[[宮心計2深宮計|宮心計2深宮計]]》內飾演吳翠盈一角與[[陳煒|陳煒]]所演的太平公主一角的對手戲被觀眾視為其代表作之一。因演的是盲人，全程沒有張開眼演，純靠聲線肢體演出，雖然只是一集戲份，但獲一致好評。<ref>{{cite web |url=https://hk.ulifestyle.com.hk/topic/detail/205278/%E6%B7%B1%E5%AE%AE%E8%A8%88-%E9%99%B3%E7%85%92%E8%A2%81%E6%BD%94%E5%84%80%E5%B0%8D%E6%89%8B%E6%88%B2%E5%85%85%E6%BB%BF%E5%A3%93%E5%A0%B4%E6%84%9F-%E4%BA%9E%E8%A6%96%E5%B9%AB%E6%92%90%E8%B5%B7%E5%8D%8A%E9%82%8A%E5%A4%A9/1 |title=《深宮計》陳煒袁潔儀對手戲充滿壓場感　亞視幫撐起半邊天 |date=2018-05-30 |work=港生活}}</ref>此後接拍《[[C9特工|C9特工]]》，首次於無綫電視劇集擔任女主角之一。

== 個人 ==
袁潔儀圈中好友為亞視藝人[[寶珮如|寶珮如]]，曾出席好友新書宣傳活動。<ref>[http://www.xpweekly.com/share/xpweekly/0333/ent/20050104ent0046/content.htm 苦過樂透 寶珮如]</ref>
