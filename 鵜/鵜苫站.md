'''鵜苫站'''（{{jpn|j=鵜苫駅|hg=うとまえき|rm=Utoma eki|f=y}}）位於[[北海道|北海道]][[樣似郡|樣似郡]][[樣似町|樣似町]]字鵜苫，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|ウト}}'''。

名稱源自[[阿伊努語|阿伊努語]]的「u-tumam-pet」，意思為抱在一起的河流<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/1120P.pdf|title=アイヌ語地名リスト|accessdate=2017-08-16|author=北海道庁|date=|publisher=北海道庁}}</ref>。

== 車站構造 ==
* [[側式月台|側式月台]]1面1線的地面站。
* 站房是由[[守車|守車]]改造而來。

== 車站周邊 ==
* {{Link-ja|國道336號|国道336号}}
* 鵜苫郵局
* 日高中央漁業協同組合様似支所鵜苫事業所
* 様似町立鵜苫小學
* 鵜苫漁港
* 鵜苫河口
* JR北海道巴士「鵜苫站前」站牌

== 歷史 ==
* 1937年8月10日 - 設立。
* 1960年4月1日 - 業務委託化。
* 1977年2月1日 - 廢止貨物和行李處理的業務。無人化。
* 1987年4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
