'''麦地那'''（{{lang-ar|المدينة المنورة}}，意謂「光芒四射的城市」）原名'''雅特里布'''，伊斯蘭教第二大聖城，後又稱聖城（{{rtl-lang|ar|مَـديـنـة الـنّـبي}} ''Madinat al-Nabi''），該城位于[[沙烏地阿拉伯|沙烏地阿拉伯]]西部，為內陸高原城市。与[[麦加|麦加]]，[[耶路撒冷|耶路撒冷]]一起被称为[[伊斯兰教|伊斯兰教]]三大圣地。

非穆斯林被沙特中央政府禁止進入麥地那的神聖核心及市中心。

== 地理 ==
位於沙烏地阿拉伯王國[[漢志|漢志]]，即北緯24.28°、東經39.36°，四面環山，海拔約六百至八百公尺，屬於山區高原城市，西距[[紅海|紅海]]約一百六十公里，南距[[麥加|麥加]]約四百公里。

== 社會經濟 ==
*該地區的財經税收多仰賴每年伊斯蘭教教徒朝聖人潮與觀光事業。
*著名的麥地那綠洲地下泉水之蓄積極為豐富，其產物除[[椰棗|椰棗]]之外，尚有水果、蔬菜及穀物等。
*麥地那是宗教的作為城市發展之中心。

== 重要地標 ==
*最著名的有[[穆罕默德|穆罕默德]]親自督建的[[先知清真寺|先知清真寺]]，寺內有穆罕默德等陵墓，為伊斯蘭世界的第二大聖寺，是信徒主要的朝拜地点之一。
*阿拉伯帝國的[[阿布·伯克爾|阿布·伯克爾]]與[[歐麥爾·本·赫塔卜|歐麥爾·本·赫塔卜]]前兩位[[哈里發|哈里發]]之墓地亦在此。<ref>[http://ap6.pccu.edu.tw/Encyclopedia/data.asp?id=9801  中華百科全書典藏版-麥地那簡介]</ref>
*附近還有穆罕默德於622年9月遷徙麥地那途中修建的[[庫巴清真寺|庫巴清真寺]]、[[雙朝向清真寺|雙朝向清真寺]]。
*聞名的[[武侯德山|伍侯德]]戰役陣亡烈士陵園。<ref>[http://www.islam.org.hk/index.php?action-viewnews-itemid-4857 伊斯蘭之光 麥地那簡介]</ref>

== 歷史 ==
[[File:المدينة_المنورة.PNG|thumb]]

* 早在公元前，猶太人已定居在麥地那綠洲，而其古名稱為雅特里布（或亞斯利普）是一個重要的交通樞紐。到公元[[6世紀|6世紀]]，麥地那仍然是阿、猶混居之狀態。
* 公元622年，因穆罕默德在麥加傳教受威脅與迫害，而逃來此地，並改名為「麥地那——納比」，意為「先知之城」，簡稱「麥地那」， 麥地那成為繼[[希吉拉|希吉拉]]（即出走麦加）之后穆罕默德活動和壯大的場所，在此屢次擊敗了來自[[麥加|麥加]]城的阿拉伯古萊氏族人之攻擊，並成立最早的伊斯蘭教政權，所以成為[[大食|大食]]帝國第一個首都。到661年，始遷至[[大馬士革|大馬士革]]。
* 1919至1924年，麥地那為[[漢志王國|漢志王國]]所統治。
* 1924年，麥地那被[[阿卜杜勒-阿齐兹·本·阿卜杜-拉赫曼·本·费萨尔·阿勒沙特|伊本·沙特]]所佔領，他后來成為[[沙特阿拉伯|沙特阿拉伯]]的第一位國王。
