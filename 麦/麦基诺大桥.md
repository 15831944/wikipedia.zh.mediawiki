'''麦基诺大桥'''（{{lang-en|Mackinac Bridge}}）是横跨[[麦基诺水道|麦基诺水道]]，连接[[美國|美國]][[密歇根州|密歇根州]][[密西根上半島|上半岛]]和[[密西根下半島|下半岛]]的一座[[悬索桥|悬索桥]]。这座跨度为{{convert|26372|ft|km|adj=on}} 的大桥（人们通常称作 "'''Big Mac'''" 和 "'''Mighty Mac'''"）是世界上 [[世界悬索桥列表|跨度第十九长]]，西半球最长的悬索桥<ref name="mba-about">{{cite web |author = Mackinac Bridge Authority |url = http://mackinacbridge.org/about-the-bridge-8/ |title = About the Bridge |publisher = Mackinac Bridge Authority |year = 2010 |access-date = January 30, 2010 |deadurl = yes |archiveurl = https://web.archive.org/web/20091222184021/http://www.mackinacbridge.org/about-the-bridge-8/ |archivedate = 十二月 22, 2009 }}</ref>。麦基诺大桥是[[75號州際公路|75號州際公路]]和绕密歇根湖和休伦湖的[[大湖环线|大湖环线]]的一部分，也是[[北部国家风景步道|北部国家风景步道]]的一部分。大桥北侧连接[[圣伊尼亚斯_(密歇根州)|圣伊尼亚斯]]，南侧连接[[麦基诺城|麦基诺城]]。
