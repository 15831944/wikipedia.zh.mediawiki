'''麦贤得'''（{{bd|1945年|||}}）[[广东省|广东省]][[饶平县|饶平县]]人，[[中国人民解放军海军|中国人民解放军海军]]战斗英雄<ref name=zgwmw>[http://www.wenming.cn/syjj/dfcz/gd/201510/t20151019_2915460.shtml 饶平媳妇、麦贤得妻子获评全国道德模范，中国文明网，2015-10-19]</ref><ref name=nfw/>。

== 生平 ==
1945年，麦贤得出生在一个船民家庭。1964年3月，参加[[中国人民解放军|中国人民解放军]]。1965年8月，加入[[中国共产党|中国共产党]]。参军后来到[[广东|广东]][[虎门|虎门]]沙角海军联合学校学习。毕业后分配到[[中国人民解放军海军|中国人民解放军海军]]护卫艇第41大队4中队611号护卫艇当机电兵<ref name=zgwmw/><ref name=nfw/>。

1965年8月5日晨，[[台湾|台湾]]方面的[[中华民国海军|中华民国海军]]巡防第2舰队大型[[猎潜舰|猎潜舰]]“剑门”号、小型猎潜舰“章江”号自台湾[[左营|左营]]港出航，傍晚驶到[[福建省|福建省]][[东山岛|东山岛]]东南的兄弟屿海域。[[中国人民解放军海军南海舰队|中国人民解放军海军南海舰队]]令汕头水警区准备战斗。当日23时13分，汕头水警区4艘护卫艇、6艘鱼雷艇奉命出击。8月6日1时42分，护卫艇群发现目标。2时51分，开始集中攻击“章江”号，3时33分将该舰击沉。在战斗中，麦贤得所在的611号护卫艇机舱中弹，机电兵均负伤。弹片击中麦贤得的前额，导致脑组织外露。麦贤得苏醒后，已无法说话，这时艇舱进水，情况危急。麦贤得顽强回到机器旁坚守岗位3小时，直到战斗胜利，他才陷入昏迷。这次战斗即[[八六海战|八六海战]]<ref name=zgwmw/><ref name=nfw/>。
[[File:1968-02_1967年_海军革命派红联总与麦贤得.jpg|thumb]]
战后，麦贤得立一等功。1966年2月4日，[[中国共产主义青年团中央委员会|中国共产主义青年团中央委员会]]授予其“模范共青团员”称号。1966年2月23日，[[中华人民共和国国防部|中华人民共和国国防部]]授予其“战斗英雄”称号。麦贤得由此成为全国知名的英雄模范。1967年12月3日，海军学习毛主席著作积极分子代表大会在北京[[人民大会堂|人民大会堂]]召开，会前[[毛泽东|毛泽东]]等[[党和国家领导人|党和国家领导人]]单独接见战斗英雄麦贤得、[[舒积成|舒积成]]等几位代表。麦贤得先后出席1987年、2007年全军英雄模范代表会议。曾任[[中国人民解放军海军汕头水警区|中国人民解放军海军汕头水警区]]副司令员，大校军衔<ref name=zgwmw/><ref name=nfw/>。2007年，从[[中国人民解放军海军广州水警区|中国人民解放军海军广州保障基地]]副司令员岗位上退休<ref>{{Cite web |url =  http://news.china.com/history/all/11025807/20150930/20492671_all.html|title = 八六海战英雄麦贤得:苏醒不识爹娘只认主席像 |author =  |date = 2015-09-30 |publisher = 中华网 |language =  |accessdate =  |quote =  }}</ref>。
[[File:1968-02_1968年_毛泽东_林彪接见麦贤得.jpg|thumb]]、林彪接见麦贤得]]
由于脑部受重伤，麦贤得的语言表达能力、记忆力几乎全部丧失，并患有严重的外伤性癫痫。1971年[[九一三事件|九一三事件]]发生后，由于毛泽东接见麦贤得的照片上有[[林彪|林彪]]在旁边，有人便说麦贤得是林彪树立的假英雄，麦贤得在汕头水警区一度遭到攻击。1972年，李玉枝和麦贤得结婚。婚后李玉枝悉心照顾麦贤得，麦贤得的癫痫从1990年代中期起便没有复发，麦贤得还在李玉枝的帮助下爱上了练习书画。麦贤得的另一大爱好是养花。麦贤得和李玉枝育有一子一女，后来都考入海军院校学习<ref name=zgwmw/><ref>[http://news.sohu.com/15/78/news146077815.shtml 丹心依然照碧海―麦贤得女儿谈父母，搜狐，2001-08-01]</ref><ref name=nfw>[http://cz.southcn.com/content/2015-07/26/content_129235550.htm 43年贤妻为“钢铁战士”麦贤得筑起爱的港湾，南方网，2015-07-26]</ref>。

2017年7月，中央军委主席[[习近平|习近平]]签署命令，授予麦贤得等10位同志“[[八一勋章_(2017年)|八一勋章]]”<ref>{{Cite web |url = http://news.ifeng.com/a/20170728/51522244_0.shtml |title = 景海鹏、程开甲、王刚等10人被授予“八一勋章” |author =  |date = 2017-07-28 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。2017年7月28日，中央军委颁授“八一勋章”和授予荣誉称号仪式在北京[[八一大楼|八一大楼]]举行，中共中央总书记、国家主席、中央军委主席习近平向麦贤得等10位“八一勋章”获得者颁授勋章和证书。这是“八一勋章”首次颁授。麦贤得同时获授“意志坚强、不怕牺牲的钢铁战士”荣誉称号<ref>{{Cite web |url = http://news.ifeng.com/a/20170728/51524452_0.shtml |title = 习近平向“八一勋章”获得者颁授勋章和证书 |author =  |date =2017-07-28   |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
