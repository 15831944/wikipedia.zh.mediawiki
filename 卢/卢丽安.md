'''卢丽安'''（{{bd|1968年|||}}），生於[[台湾|台湾]][[高雄|高雄]]，為[[复旦大学|复旦大学]][[教授|教授]]、[[英国文学|英国文学]]研究专家，上海市台湾同胞联谊会会长<ref name=ifeng>{{Cite web |url = http://news.ifeng.com/a/20171006/52382692_0.shtml |title = 这位十九大代表是土生土长的台湾高雄人 |author =  |date =2017-10-06  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref name=fudan>{{Cite web |url =  http://jpkc.fudan.edu.cn/s/292/t/607/45/a2/info17826.htm  |title =主讲教师：卢丽安  |author =  |date =  |publisher = 复旦大学 |language =  |accessdate =2017-10-06  |quote =  }}</ref>。

==生平==
卢丽安出生於台湾高雄[[旗山區|旗山]]，畢業於[[高雄女中|高雄女中]]。1986年9月，考入[[国立政治大学|国立政治大学]]，1990年6月自国立政治大学西洋语文学系（后更名为英国语文学系）毕业。1991年9月，留学[[英国|英国]][[爱丁堡大学|爱丁堡大学]]英国文学系攻读文学硕士，1992年12月毕业。1993年9月，进入英国[[格拉斯哥大学|格拉斯哥大学]]英国文学系攻读文学哲学博士学位，1999年12月毕业。在英国学习期间，结识了在爱丁堡大学攻读博士学位一位男同学，两人交往后“[[裸婚_(結婚方式)|裸婚]]”。卢丽安的丈夫是“[[台中|台中]]籍台胞”<ref name=ifeng/><ref name=fudan/>。

1997年，卢丽安夫妇接受[[中华人民共和国人事部|中华人民共和国人事部]]安排，以“返国定居专家”身份到上海[[复旦大学|复旦大学]]任教，定居上海。卢丽安在复旦大学外文学院任教，1997年至2001年2月为讲师；2001年3月至2013年12月为副教授；2013年12月被评为教授。研究方向为英国文学、二十世纪英国小说、性别研究、[[女性主义|女性主义]]、文学社会学、生命书写、[[安徒生|安徒生]]研究。2009年9月至2010年6月，按照中美[[富布赖特学者|富布赖特访问学者]]计划，在[[美国|美国]][[密歇根大学#安娜堡分校|密歇根大学安娜堡分校]]妇女与性别研究中心与英文系进修。曾任[[上海市政协|上海市政协]]委员。自2013年起，卢丽安任上海市台湾同胞联谊会会长，同时继续担任复旦大学外文学院英文系教授、博士生导师、副院长。2015年加入[[中国共产党|中国共产党]]。

2017年6月19日至21日，在[[中国共产党全国台湾省籍党员代表会议|中国共产党全国台湾省籍党员代表会议]]上，卢丽安等10位[[台湾省_(中华人民共和国)|台湾省]]籍人士当选台湾省出席[[中共十九大|中共十九大]]代表，卢丽安是其中唯一出生在台湾的代表。10月，卢丽安作为大会代表参加了中共十九大，并接受媒体采访，期間更以[[台語|台語]]喊出「我們愛台灣，當然也可以愛大陸」<ref>{{Cite web |url =https://udn.com/news/story/7331/2742228 |title = 盧麗安獲選19大代表 張小月：她已入大陸籍 |author = |date = 2016-10-06 |publisher = 聯合報 |language =  |accessdate =  |quote =  }}</ref><ref name=ifeng/><ref name=fudan/><ref>{{Cite web |url = http://news.takungpao.com/mainland/focus/2017-10/3505384.html |title = 台籍代表卢丽安：入党因信服党坚定信念和执政成绩 |author =  |date = 2017-10-20 |publisher =大公网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.crntt.com/doc/1048/5/1/3/104851312.html?coluid=1&kindid=0&docid=104851312&mdate=1021003214 |title =会担心受台湾打压？卢丽安：这要问张小月 |author =  |date = 2017-10-21 |website = 中评网  |language =  |accessdate =  |quote =  }}</ref>。10月5日，[[中华民国|中华民国]][[行政院大陆委员会|行政院大陆委员会]]主委[[张小月|张小月]]表示，卢丽安确實在台湾出生，但已获陆籍，陆委会对此表示尊重。10月26日，陆委会副主委[[林正义|林正义]]在[[立法院|立法院]]表示，卢丽安夫妇的台湾户籍已被取消。盧麗安的家人痛批「政府炒作」<ref name=ltn>{{Cite web |url = http://m.ltn.com.tw/news/politics/breakingnews/2234271 |title = 盧麗安夫婦戶籍被取消 家人錯愕痛批「政府炒作」 |author =  |date =2017-10-26  |publisher = 自由時報 |language =  |accessdate =  |quote =  }}</ref>。10月28日，前[[中国国民党主席|中国国民党主席]][[洪秀柱|洪秀柱]]表示，政府不应用这种激烈手段制造对立，“这对于两岸的未来不是很好”<ref>{{Cite web |url =http://news.ifeng.com/a/20171028/52834922_0.shtml  |title =  台湾撤销19大代表卢丽安户籍 洪秀柱批“制造对立”|author =  |date = 2017-10-28 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。卢丽安被取消台湾户籍消息传至网路后引起讨论，[[PTT|PTT]]上的部分[[網友|網友]]表示支持<ref>[http://news.ltn.com.tw/news/politics/breakingnews/2234378 卢丽安夫妇户籍被砍乡民叫好：叛国还讲那么多].[[自由时报|自由时报]]. 2017-10-26.</ref>。

==家庭==
* 祖父：盧老枝，高雄旗山人，曾留學[[日本|日本]]，先後於奈良縣立郡山中學校（今郡山高等學校）、舊制大阪高等學校、[[京都大學|京都大學]]農學部畢業。
:根據《台灣人士鑑》記載，盧老枝在就讀奈良縣立郡山中學校四年級時，校內發生火災，大家都忙著逃命，只有盧老枝奮不顧身，衝入火場搶救出一幅天皇玉照，到達安全地帶後獲得皇室讚揚。
:[[二戰|二戰]]期間[[東京|東京]]街頭戰車、軍車大遊行，一支[[日本國旗|日本國旗]]掉進車底下，盧老枝攔住車輛，爬進車底下救出旗子，現場一陣掌聲，被媒體記者登在隔天日本各大報，並編進日本小學課本內。
:盧老枝返台後曾任[[旗山街|旗山街]]協議會員、[[旗山區農會|旗山信用組合]]理事、[[國立旗山高級農工職業學校|旗山農民學校]]組合議員，是[[台灣日治時期|日治時期]]很重要的[[臺灣人|台籍]][[仕紳|仕紳]]。
:[[大正|大正]]12年（1923年）4月16日，日本皇太子殿下（後來的[[裕仁天皇|裕仁天皇]]，當時的監國）來台行啟時，盧老枝因搶救天皇玉照的事蹟受「恩賜財團台灣濟美會」會長[[內田嘉吉|內田嘉吉]]（後來的[[台灣總督|台灣總督]]）贈金一封表彰。
:[[臺灣戰後時期|戰後]][[國民政府|國民政府]]來台，盧老枝加入[[國民黨|國民黨]]，成了忠黨愛國的中學校長，曾任[[高雄市立旗山國民中學|旗山國中]]校長，但因向學生強捐款項20餘萬，被以瀆職罪嫌起訴<ref name=fumu>{{Cite web|url =http://www.uzixun.com/guonei/201710/3518.html|title =卢丽安从政 父母支持：希望两岸人民都过得好|author =|date =2017-10-20|website =东森新闻云|language =|accessdate =|quote =}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
* 父：盧建志，醫生，曾任[[衛生福利部|衛生福利部]][[衛生福利部屏東醫院|屏東醫院]]院长<ref name=fumu/>。
* 母：蒋美樱，曾任旗山国中教师。後曾任旗美合唱团团长，旗山蕉城美术协会理事长<ref name=fumu/>。
