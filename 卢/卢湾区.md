'''卢湾区'''为[[中华人民共和国|中国]][[上海市|上海市]]已撤销的[[市辖区|市辖区]]，2011年7月26日，[[黄浦区_(2000–2011)|原黄浦区]]与卢湾区合併设立新的[[黄浦区|黄浦区]]<ref>《[[上海市人民代表大会常务委员会|上海市人民代表大会常务委员会]]关于撤销黄浦区和卢湾区、设立新的黄浦区若干问题的决定》（2011年7月26日上海市十三届人民代表大会常务委员会第二十八次会议通过）</ref>。撤销前，卢湾区辖域面积8.03平方公里，其中水域面积0.49平方公里。常住人口248,779人（2010年人口普查）<ref>2010年人口普查数据来源于《上海市[[第六次全国人口普查|第六次全国人口普查]]主要数据公报》</ref>，户籍人口305,144人（2010年人口普查）<ref>2010年户籍人口数据来源于《上海市[[第六次全国人口普查|第六次全国人口普查]]主要数据公报》</ref>。2010年地方财政收入136.61亿元，约占上海市地方财政收入的4.75%<ref>财政收入数据来源于[http://www.xzqh.org/html/2008/0421/4860.html 行政区划网] {{webarchive|url=https://web.archive.org/web/20110808094949/http://www.xzqh.org/html/2008/0421/4860.html |date=2011-08-08 }}，2010年上海市地方财政收入2873.6亿元（上海市人民政府2011年政府工作报告）</ref>。区人民政府驻[[重庆南路_(上海)|重庆南路]]，下辖4个[[街道办事处|街道办事处]]。

卢湾区地处上海市中心，区内商业、饮食业、服务业繁荣发达，世界知名的[[淮海路_(上海)|淮海路]]商业街区上有一批著名的百货店、特色店、饭店、宾馆、影剧院及娱乐、服务等场所。卢湾得名于区内社区[[卢家湾|卢家湾]]，区内有[[上海中山故居|上海中山故居]]等各类人文景观150多处。从[[孙中山|孙中山]]、[[蒋中正|蒋中正]]、[[毛泽东|毛泽东]]、[[周恩来|周恩来]]到[[郭沫若|郭沫若]]、[[梅兰芳|梅兰芳]]、[[张大千|张大千]]，中国近代史上诸多的著名的政治家、文人画家等，都在此留下过业绩和身影。区内交通便捷，[[地铁|地铁]]、地面交通、高架公路构成立体的交通网络。区内文化、教育、卫生、体育、社区等公益事业设施众多。2011年6月8日，上海市人民政府召开会议，宣布卢湾、[[黄浦|黄浦]]两区行政区划调整方案已获[[中华人民共和国国务院|国务院]]正式批复，撤销黄浦区、卢湾区建制，设立新的黄浦区。<ref>[http://roll.sohu.com/20110608/n309637327.shtml 搜狐：上海宣布黄浦区和卢湾区建制撤销 将设新黄浦区]</ref>

== 历史沿革 ==
1945年二战胜利后，重庆南路和鲁班路以西设第六区（卢家湾区），鲁班路以东设第五区（泰山区）。1947年泰山区改称嵩山区，1950年“卢家湾区”改称“'''卢湾区'''”。1956年撤销嵩山区，其大部并入卢湾区。1959年原邑庙区部分并入；2011年7月26日撤销并入黄浦区。

== 地理位置 ==
卢湾区位于[[上海市|上海市]]中心，全区面积8.03平方公里，其中陆地面积7.54平方公里，水域面积0.49平方公里。区界北至[[延安中路_(上海)|延安中路]]、[[金陵西路|金陵西路]]，与[[黄浦区|黄浦区]]、[[静安区|静安区]]交界；东至[[西藏南路|西藏南路]]、[[肇周路|肇周路]]、[[制造局路|制造局路]]、[[高雄路|高雄路]]、[[江边路|江边路]]，与黄浦区（原[[南市区_(上海市)|南市区]]）接壤；西至[[陕西南路|陕西南路]]、[[瑞金南路|瑞金南路]]，与静安区、[[徐汇区|徐汇区]]为邻；南至[[黄浦江|黄浦江]]河道中心线，与[[浦东新区|浦东新区]]相望。

== 人口 ==
2009年底，全区户籍人口30.74万人，常住人口26.94万，人均期望寿命为82.37岁。

* [[五里桥街道|五里桥街道]]

== 交通 ==
[[上海轨道交通|轨道交通]]有[[上海轨道交通一号线|一号线]]、[[上海轨道交通四号线|四号线]]、[[上海轨道交通八号线|八号线]]、[[上海轨道交通九号线|九号线]]及[[上海轨道交通十号线|十号线]]，公路有[[南北高架|南北高架]]、[[上海内环线|内环线高架]]、[[卢浦大桥|卢浦大桥]]、[[打浦路隧道|打浦路隧道]]以及四通八达的公路网，构成了卢湾立体快捷的交通网络。
