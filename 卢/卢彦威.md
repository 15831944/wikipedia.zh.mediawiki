'''卢彦威'''（{{bd|9世纪||10世纪|catIdx=L}}），[[晚唐|晚唐]][[军阀|军阀]]，[[光启|光启]]元年（885年）—[[乾宁|乾宁]]五年（898年）控制[[义昌军|义昌军]]，其间绝大部分时候是以其[[节度使|节度使]]的身份。

== 背景及夺取义昌 ==
卢彦威家世不详，官修史书《[[旧唐书|旧唐书]]》《[[新唐书|新唐书]]》均无他的传。《[[资治通鉴|资治通鉴]]》在光启元年七月首次提及他：当时，义昌军部[[沧州|沧州]]兵变，节度使[[杨全玫|杨全玫]]被逐，逃往邻镇[[卢龙|卢龙]]。士兵推时任牙将的兵变头领卢彦威为留后。<ref>[[:s:新唐书/卷9|《新唐书》卷九]]</ref>在位的[[唐僖宗|唐僖宗]]并未任卢彦威为节度使，而是任保銮都将[[曹诚|曹诚]]为节度使，<ref>[[:s:旧唐书/卷19下|《旧唐书》卷一十九下]]</ref>而任卢彦威为义昌下辖三州之一[[德州|德州]]的刺史。<ref name=ZZTJ256>[[:s:資治通鑑/卷256|《资治通鉴》卷二百五十六]]：沧州军乱，逐节度使杨全玫，立牙将卢彦威为留后，全玫奔幽州。以保銮都将曹诚为义昌节度使，以彦威为德州刺史。</ref>

但卢彦威拒绝让曹诚来义昌赴任，仍要求获任节度使；朝廷则一直回绝。[[大顺_(唐朝)|大顺]]元年（890年）六月，僖宗弟[[唐昭宗|唐昭宗]]在位时，兴兵讨伐大军阀[[河东节度使|河东节度使]][[李克用|李克用]]，卢彦威邻镇[[成德节度使|成德节度使]][[王镕|王镕]]和[[魏博节度使|魏博节度使]][[罗弘信|罗弘信]]再度上表朝廷请求任卢彦威为节度使。朝廷铁心要讨伐李克用，于是任时为权知沧州兵马留后的卢彦威为[[检校|检校]]尚书右仆射，兼沧州刺史、[[御史大夫|御史大夫]]，充义昌军节度、沧德观察处置等使。<ref name=BT20-1>[[:s:旧唐书/卷20上|《旧唐书》卷二十上]]</ref><ref name=ZZTJ258>[[:s:資治通鑑/卷258|《资治通鉴》卷二百五十八]]：[[光启|光启]]末，德州刺史卢彦威逐义昌节度使杨全玫，自称留后，求旌节，朝廷未许。至是，王镕、罗弘信因[[张濬|张濬]]用兵，为之请，乃以彦威为义昌节度使。</ref>

== 任节度使 ==
乾宁元年（894年）十二月，李克用攻占卢龙。[[卢龙节度使|卢龙节度使]][[李匡筹|李匡筹]]奔义昌，想逃奔京师[[长安|长安]]。卢彦威不但不保护李匡筹，还嫉妒李匡筹从卢龙带来的巨额辎重，攻杀李匡筹于[[景城|景城]]，<ref name=NBT10>[[:s:新唐书/卷10|《新唐书》卷十]]</ref>夺取了他的辎重、部众、僮仆、姬妾。<ref name=BT20-1/><ref>[[:s:旧唐书/卷180|《旧唐书》卷一百八十]]</ref><ref name=NBT212>[[:s:新唐书/卷212|《新唐书》卷二百一十二]]</ref><ref>[[:s:旧五代史/卷26|《旧五代史》卷二十六]]</ref><ref name=ZZTJ259>[[:s:資治通鑑/卷259|《资治通鉴》卷二百五十九]]：壬子，匡筹复发兵出[[居庸关|居庸关]]，克用使精骑当其前以疲之，遣步将[[李存审|李存审]]自他道出其背夹击之，幽州兵大败，杀获万计。甲寅，李匡筹挈其族奔沧州，义昌节度使卢彦威利其辎重、妓妾，遣兵攻之于景城，杀之，尽俘其众。</ref>

数年后，卢彦威残虐，不礼邻镇。五年（898年）三月，他和李克用扶植的卢龙节度使[[刘仁恭|刘仁恭]]争夺盐利。刘仁恭仗着先前刚打败李克用，有吞并河朔之心，遣子[[刘守文|刘守文]]攻沧州。卢彦威抵挡不住，率家眷弃城奔[[魏博|魏博]]，刘守文遂据沧州、[[景州|景州]]、德州，<ref>[[:s:旧五代史/卷90|《旧五代史》卷九十]]</ref><ref>[[:s:新五代史/卷39|《新五代史》卷三十九]]</ref><ref>[[:s:宋史/卷259|《宋史》卷二百五十九]]</ref>自称留后。<ref name=BT20-1/><ref name=NBT212/><ref>[[:s:旧五代史/卷135|《旧五代史》卷一百三十五]]</ref>罗弘信不接纳卢彦威，卢彦威逃到更远的[[宣武军|宣武军]]军部[[汴州|汴州]]。<ref name=NBT10/><ref name=ZZTJ261>[[:s:資治通鑑/卷261|《资治通鉴》卷二百六十一]]：义昌节度使卢彦威，性残虐，又不礼于邻道。与卢龙节度使刘仁恭争盐利，仁恭遣其子守文将兵袭沧州，彦威弃城，挈家奔魏州。罗弘信不纳，乃奔汴州。</ref>

[[天祐_(唐朝)|天祐]]二年（905年），卢彦威在左金吾上将军任上被改任为左威卫上将军。<ref>[[:s:旧唐书/卷20下|《旧唐书》卷二十下]]</ref>后事不详。

镇守沧州期间，卢彦威曾署[[李愚|李愚]]为安陵簿。<ref>[[:s:旧五代史/卷67|《旧五代史》卷六十七]]</ref>但李愚因丁母忧去职。<ref>[[:s:新五代史/卷54|《新五代史》卷五十四]]</ref>

== 轶闻 ==
[[后晋|后晋]][[开运|开运]]二年（945年）正月，都城[[汴州|汴州]]封丘门外壕水东北隅水上有图案，是大树花叶发散香气的样子，相连数十株，仿佛图画，全汴州的人都去看。有认得图案的人说：“唐朝[[景福|景福]]年间，卢彦威治下的浮阳壕水也有这样的图文，当时就有高深的尼姑辞别郡人说：'此地当有兵难。'到了[[光化|光化]]年间，其郡果然为燕帅刘仁恭所陷。”<ref>[[:s:旧五代史/卷141|《旧五代史》卷一百四十一]]</ref>四年（947年），[[契丹|契丹]]灭晋。
