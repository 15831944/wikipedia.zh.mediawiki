== 历史 ==
[[File:卢沟桥狮子_特写.JPG|thumb]]

早在[[战国|战国]]时期，卢沟河渡口便是[[燕|燕]][[蓟|蓟]]的交通要地。原先仅有浮桥。[[金朝|金朝]][[贞元|贞元]]元年（1153年）金朝定都燕京并改为[[金中都|中都]]之后，该浮桥成为南方各省入京的必经之路，逐渐不敷需要。[[金世宗|金世宗]]乃诏命建桥<ref name=xhw>{{Cite web |url = http://news.xinhuanet.com/politics/2010-09/02/c_12510649.htm |title =  卢沟桥|author =  |date =2010-09-02  |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。

卢沟桥是在[[金朝|金朝]][[金世宗|金世宗]][[大定|大定]]二十八年（1188年）五月决定修建的。但尚未开工，金世宗便在大定二十九年（1189年）初病逝。大定二十九年（1189年）六月，[[金章宗|金章宗]]开工兴建卢沟桥，[[明昌|明昌]]三年（1192年）三月竣工。初名“广利桥”。后因此桥跨越芦沟（今永定河），故人称“芦沟桥”。“卢沟晓月”从金章宗年间便成为“[[燕京八景|燕京八景]]”之一<ref name=xhw/>。

13世纪，[[意大利|意大利]]人[[马可·波罗|马可·波罗]]来到[[元大都|元大都]]。《[[马可·波罗游记|马可·波罗游记]]》里写道：“自从[[元大都|汗八里城]]发足以后,骑行十里，抵一极大河流名称“普里桑乾”, 此河流入海洋。商人利用河流运输商货者甚伙。河上有一美丽石桥，各处桥梁之美鲜有及之者。桥长三百步，宽逾八步，十骑可并行于上。下有桥拱二十四，桥脚二十四，建置甚佳，纯用极美之大理石为之。桥两旁皆有大理石栏，又有柱，狮腰承之。柱顶别有一狮。此种石狮甚巨丽，雕刻甚精。每隔一步有一石柱，其状皆同。两柱之间，建灰色大理石栏，俾行人不致落水，桥两面皆如此，颇壮观也。”<ref name="Polo2006">{{cite book|author=Marco Polo|title=The Travels of Marco Polo|url=http://books.google.com/books?id=FWCgc-rEpVcC&pg=171|date=2006|publisher=Cosimo, Inc.|isbn=978-1-60206-024-1|pages=171–172}}{{en}}</ref>西方人因此稱卢沟桥為「馬可·波羅橋」。

[[明朝|明朝]]自[[永乐|永乐]]十年（1412年）到[[嘉靖|嘉靖]]三十四年（1555年）共修桥6次，这6次均没有大的工程。其中明朝[[正统_(年号)|正统]]九年（1444年）重修卢沟桥是比较重要的一次。[[清朝|清朝]]自[[康熙|康熙]]元年（1662年）至[[光绪|光绪]]年间共修桥7次，其中5次工程不大，两次工程稍大。卢沟桥在康熙时毁于洪水，康熙三十七年（1698年）重建。[[康熙帝|康熙帝]]命在卢沟桥西端立碑，记述重修卢沟桥的经过。桥东端立有[[乾隆帝|乾隆帝]]为“燕京八景”题写的“卢沟晓月”碑。1908年，[[光绪帝|光绪帝]]逝世后，灵柩运往今[[河北省|河北省]][[易县|易县]][[清西陵|清西陵]]安葬，途中须经卢沟桥，因桥面窄而将桥边石栏拆除，添搭木桥，事后将石栏按照原样恢复<ref name=xhw/>。

1937年7月7日，在卢沟桥及桥东头的[[宛平县城|宛平县城]]发生“[[七七卢沟桥事变|七七卢沟桥事变]]”，成为[[抗日战争|抗日战争]]全面爆发的起点。在事变中，[[国民革命军|国民革命军]]第二十九军司令部“命令前线官兵坚决抵抗，卢沟桥即尔等之坟墓，应与桥共存亡，不得后退。”[[中华人民共和国|中华人民共和国]]成立后，在桥面上加铺柏油。此外还修缮了石栏、望柱（包括柱头石狮）及碑亭。1961年卢沟桥及[[宛平县城|宛平县城]]被公布为[[第一批全国重点文物保护单位|第一批全国重点文物保护单位]]<ref name=xhw/>。

[[File:1965-9_1965_卢沟桥.jpg|thumb]]
1967年8月，[[中央文革小组|中央文革小组]]命[[北京市革命委员会|北京市革命委员会]]解决卢沟桥交通不畅的问题。北京市革命委员会乃为卢沟桥加宽400多米步道、建立59道混凝土挑梁。该工程正值[[文革|文革]]期间，原工区主任以及老工程师都在“接受工人阶级再教育”（即参加劳动），施工队队长由一位青年工人担任，施工队中有位[[贫农|贫农]]出身的老石工，自己做主更换了望柱91根、栏板9块，在新的91根望柱上共雕出164只石狮，这比原有的91根望柱上的石狮总数多出5只，从而使卢沟桥望柱上的狮子总数达到491只<ref name=xhw/>。

1971年，为减少卢沟桥的运输量而兴建的[[卢沟新桥|卢沟新桥]]完工，卢沟新桥位于卢沟桥以南约1公里远处。卢沟新桥完工后，卢沟桥仍继续承担交通运输任务<ref name=xhw/><ref>{{Cite web |url =http://news.163.com/09/0529/15/5AG8I0NA0001124J.html  |title =卢沟新桥今通车 桥宽是过去两倍  |author =  |date = 2009-05-29 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。后来在经过调查后，有政协委员提出另建新桥以解决交通需求，将卢沟桥作为文物保护。交通部门拒绝了该建议，一再强调桥就是为交通而设。古建筑保护专家[[郑孝燮|郑孝燮]]听到后非常气愤：“饭碗本来是吃饭用的，但是故宫里展览的那些古代瓷器，如宋代玛瑙轮花碗、珐琅彩雉鸡牡丹纹碗等，就不是用来吃饭的，是供人欣赏的。卢沟桥这座古桥，距今有800多年的历史了，为什么还让它走车呢？现在它就是用来参观的。为了节约资金，少修一座桥，这不是保护的概念。”<ref name=caixin>{{Cite web |url = http://culture.caixin.com/2017-01-26/101049169.html |title = 城市规划专家郑孝燮去世 享年100岁 |author =  |date =  2017-01-26|publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref>经过多方努力，1985年又在卢沟新桥南边兴建了一座新桥，即后来的[[京石高速公路|京石高速公路]]大桥。卢沟桥停止机动车通行，机动车通行移至卢沟桥以南的卢沟新桥以及[[京石高速公路|京石高速公路]]大桥<ref name=xhw/>。

1986年，北京市政府成立了卢沟桥历史文物修复委员会，在修缮工程中清除了1949年后在桥面铺设的柏油和1967年加宽的步道和混凝土挑梁，并全面整修了桥券、桥墩，对全部栏板、望柱、地袱、桥面、华表、石碑等实施部分风化及防渗漏保护措施，桥面中间空出印心，完全恢复了卢沟桥的原貌；同时还修缮了[[宛平县城|宛平县城]]，复建了城楼。1988年9月3日，晴天一个霹雷将卢沟桥北侧东起第68根栏杆望柱击坏，该望柱的1只狮子和2只小狮子损坏。1997年，对部分被雷电击坏的石狮和望柱进行修补，上述北侧东起第68根栏杆望柱的3只被雷电击坏的石狮获得修复<ref name=xhw/>。

==建筑==
卢沟桥在北京市中心西南方约15公里处的丰台区永定河上，因跨越卢沟河（今永定河）而得名。[[中华人民共和国|中华人民共和国]]时期，卢沟桥下的永定河长年无水，但[[2008年北京奥运会|2008年北京奥运会]]前后曾重新给水。桥东为[[宛平县城|宛平县城]]，[[明朝|明朝]][[崇祯|崇祯]]十一年（1638年）建<ref name=xhw/>。

卢沟桥是北京市现存最古的石造联拱桥，也是[[华北|华北]]最长的古代石桥。卢沟桥全长266.5米，宽7.5米，最宽处9.3米。卢沟桥有十座桥墩，共11个桥孔，桥身全部为石体结构，关键部位由银锭铁榫连接。桥墩下面呈船形，迎水面砌为分水尖<ref name=xhw/>。

卢沟桥两侧石护栏共有281根[[望柱|望柱]]，每根望柱高1.4米，柱头刻莲座，座下为荷叶墩，柱顶刻有许多石狮。卢沟桥总计有501只石狮<ref name=xhw/><ref name=yuwen>{{Cite web |url =  http://news.sina.com.cn/s/2003-11-03/04221044260s.shtml|title =  语文课本501只《十万个为什么》485只 卢沟桥到底多少狮子|author =  |date = 2003-11-03 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。民间有「卢沟桥的狮子，数不清」的说法。据记载原来有627个<ref name=xhw/>（《從海記》中說：“橋柱刻獅凡六百二十有七”<ref>{{Cite web |url = http://news.sina.com.cn/c/2007-08-20/141413703032.shtml |title = 宛平的前世今生：1937～2007 |author =  |date = 2007-08-20 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>）。石狮多为明清时期雕刻，也有少量为金元时期雕刻<ref name=xhw/>。明朝[[蔣一葵|蔣一葵]]在《[[長安客話|長安客話]]》中說，这些石獅“凡一百狀，數之輒隱其一”。明朝《[[帝京景物略|帝京景物略]]》记载卢沟桥的石狮“数之辄不尽”。1962年，據北京市文物工作隊採用編號的方式調查，卢沟桥上的大小獅子共有485只。在1979年的复查中，又发现了16只，石狮总数达到501只<ref name=xhw/>。1988年遭受雷击后损失3只石狮，剩下498只；1997年修复了这3只石狮。同时根据卢沟桥文物保护所的统计，这一数字沿用至今<ref name=yuwen/><ref name=xhw/>。

[[人民教育出版社|人民教育出版社]]1996年10月版小学语文教科书首次收入了《卢沟桥的狮子》作为课文，其中说卢沟桥有望柱241根，狮子498只。1998年卢沟桥改造后，3只之前被雷击毁灭的狮子被修复，至此狮子数量由498只变成501只（见上文）。人民教育出版社咨询了卢沟桥文物保护所，将《卢沟桥的狮子》课文中的内容修改为281根望柱、501只狮子。人民教育出版社2001年版第七册小学语文教科书第一篇课文就是《卢沟桥的狮子》，记录了这一数据<ref name=yuwen/>。

建筑学家[[罗哲文|罗哲文]]《名闻中外的卢沟桥》一文中描绘了这些石狮子：“……有的昂首挺胸，仰望云天；有的双目凝神，注视桥面；有的侧身转首，两两相对，好像在交谈；有的在抚育狮儿，好像在轻轻呼唤；桥南边东部有一只石狮，高竖起一只耳朵，好似在倾听着桥下潺潺的流水和过往行人的说话……真是千姿百态，神情活现。”<ref name=xhw/>

卢沟桥的石狮除了望柱上的大小石狮外，桥东端还有顶着栏杆的石狮左右各1只。桥两端有华表各1对，华表顶部石兽各1对（其中东面是一对狮子，西面是一对大象）。两畔各有石碑一座，其中一座碑是清朝康熙二十七年重修卢沟桥的经过，另一座碑是乾隆帝题写的“卢沟晓月”碑，碑高4.52米，宽1.27米，厚84厘米<ref name=xhw/>。
