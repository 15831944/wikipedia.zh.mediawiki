'''卢少平'''（{{bd|1961年|10月||}}）[[广东|广东]][[揭阳|揭阳]]人<ref>{{Cite web |url = http://www.xinhuanet.com/mil/2016-03/06/c_128776921_2.htm|title = 卢少平少将：敢说“硬话” 敢于担当 |author =  |date = 2016-03-06 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
1978年12月入伍，曾历任班长、区队长、指导员、教导员、团政治处主任、团政委，师政治部主任、师副政委、[[广州军区|广州军区]]政治部宣传部代副部长、西藏日喀则和林芝军分区代副政委、师政委，[[第42集团军|第42集团军]]政治部主任。2012年12月任第42集团军副政委，2015年4月调任[[第16集团军|第16集团军]]政委。<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1322012|title = 42集团军原副政委卢少平跨区履新沈阳军区，任驻吉部队领导 |author =  |date = 2015-04-17 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>2017年，出任新组建的[[中国人民解放军陆军第八十三集团军|中国人民解放军陆军第八十三集团军]]政治委员。<ref>{{Cite web |url = http://m.china.caixin.com/m/2017-07-22/101114293.html|title = 多个集团军主官亮相 卢少平任83军政委 |author =  |date = 2017-07-22 |publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref>2018年12月，任[[东部战区|东部战区]]政治工作部副主任<ref>{{cite web |author1=林韵诗 |title=东部战区领导层调整 兰政、卢少平履新 |url=http://china.caixin.com/2019-01-13/101369532.html |website=财新网 |accessdate=2019-02-02 |date=2019-01-13}}</ref>。先后参加汕头抗台风救灾、98长江抗洪、中印边境武装巡逻、南方抗击雨雪冰冻灾害、支援第十六届亚运会等重大非战争军事行动，获评全军优秀政治教员、全军优秀基层干部、全军优秀指挥军官。

2012年12月晋升[[中国人民解放军少将|中国人民解放军少将]]军衔。
