'''卢本伟'''（{{bd|1993年|8月11日}}），前[[斗鱼网|斗鱼TV]]主播；游戏《[[英雄联盟|英雄联盟]]》前职业选手，曾为《[[英雄联盟|英雄联盟]]》战队[[皇族电子竞技俱乐部|皇族]]担任中单，在队伍获得[[英雄聯盟S3賽季全球總決賽|S3全球决赛]]亚军后退役<ref>{{cite web|url=http://www.laonanren.com/news/2015-08/102276.htm|title=皇族white富二代身份解密 卢本伟资料背景女友照片曝光_卢本伟个人资料_著名男人_名人|author=|date=|website=www.laonanren.com}}</ref>。

== 名称由来 ==
在英雄联盟2013赛季全球总决赛前夕，有记者询问他关于和世界第一中单[[Faker|Faker]]（韩国电竞选手，中文译名为李相赫）的比赛中的胜算问题，卢本伟认为这个问题很难回答，于是声称他在训练赛时与Faker“五五开”——意为两者不相上下。赛后皇族以0-3负于[[SK_Telecom_T1|SKT]]，Faker的表现亦远远好于卢本伟<ref>{{Cite web |url = http://www.91danji.com/news/87837.html |title = 英雄联盟扫盲那些传说中的梗之white 打谁都是55开 |author = 贝伦卡斯泰露 |date = 2014年10月24日 |publisher = 91单机游戏网 |language = zh-cn }}</ref>。后来卢本伟在直播中澄清了这个外号真正的来历，是当时他的女友赵梦玥特意让他在采访时以“五五开”来增加赛前信心。

== 争议 ==
* 2015年4月，卢本伟指责[[皇族俱乐部|皇族俱乐部]]拖欠三十万的比赛资金未发，称战队篡改合同日期<ref>[http://wangyou.pcgames.com.cn/505/5054003.html 55开：皇族已不再 对我的算计不想再提]</ref><ref>{{cite web|url=http://news.52pk.com/shtml/20150422/6395234.shtml|title=55开White微博声讨皇族俱乐部欠奖金事件惹争议_52pk新闻中心|first=|last=52pk|date=|website=news.52pk.com}}</ref>。
* 2015年10月，主播[[古手羽|古手羽]]在直播中提起卢本伟删除他微博好友后反应剧烈，其后粉丝在卢本伟的微博上进行语言攻击，其后卢本伟称其借过10万元。后双方互相回击。<ref>{{cite web|url=http://www.72g.com/news/110020.html|title=White55开古手羽撕逼大战究竟为何 两人互回长微博诉苦 - 72G.com手机游戏门户|author=|date=|website=www.72g.com}}</ref><ref>{{cite web|url=http://www.chinaz.com/game/lol/2015/0930/452863.shtml?id2|title=LOL主播55开回击古手羽：10W就当被偷了，不想把你逼死！|author=|date=2015-09-30|publisher=}}</ref>
* 2015年11月17日，卢本伟在直播中进行英雄联盟双人排位，而该场比赛为同队赏金猎人（女枪）升最强王者前的最后一场游戏，但卢本伟在队友提醒的情况下与对手[[德莱文|德莱文]]进行单挑导致拖延时间，虽然最终取得胜利，但是游戏结束时已到北京时间凌晨12时02分（在12时进行结算），导致女枪未取得最强王者框。而卢本伟其后在直播发表“我开心就好，我玩游戏就是开心，我完全没义务帮她”的言论，结果遭到网友的抨击。11月18日，卢本伟发表[[微博|微博]]道歉<ref>[http://www.ocn.com.cn/life/201511/lbxdn18121846.shtml LOL主播55开直播玩托遭群愤 女友UU帮其微博道歉]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web|url=http://games.sina.com.cn/e/n/2015-11-18/fxkszhk0346106.shtml|title=撕逼事件再起：55开玩心重致队友未上王者遭喷_电子竞技-国内_新浪游戏_新浪网|author=|date=|website=games.sina.com.cn}}</ref>。
* 2017年10月，卢本伟指责[[孙亚龙|孙亚龙]]在他的直播间刷礼物，并声称其一直在暗中攻击他。孙亚龙也宣布与卢本伟决裂。<ref>{{cite web|url=http://www.mnw.cn/keji/internet/1867903.html|title=五五开德云色笑笑怎么了发生什么事 卢本伟笑笑事件始末介绍-闽南网|author=|date=|website=www.mnw.cn}}</ref>
* 2017年11月25日，卢本伟在直播《[[絕地求生|绝地求生：大逃杀]]》时由于多个非常规操作，被认为有使用[[游戏外挂|外挂]]的嫌疑，并由此引发[[卢本伟外挂纷争|相关纠纷]]。<ref name=eastday>{{Cite web|url=http://sports.eastday.com/a/171201102825478000000.html|title=真相是什么？义薄云天卢本伟29杀吃鸡 粉丝不买账}}</ref>
* 2018年2月12日，卢本伟因教唆粉丝辱骂质疑者，被[[中国中央电视台|央视]]《[[焦点访谈|焦点访谈]]》点名批评，被实施跨平台封禁。<ref>{{Cite web|url=http://www.sohu.com/a/222456271_130887|title=监管整顿直播：焦点访谈点名 主播天佑、55开遭跨平台封禁|accessdate=2018-02-13|date=2018-02-12|work=www.sohu.com}}</ref><ref>{{Cite web|url=http://tv.cctv.com/2018/02/12/VIDEpxM0pCg3ZHA03ZCTHZyZ180212.shtml|title=重拳打击网络乱象|accessdate=2018-02-12|publisher=CCTV|work=tv.cctv.com}}</ref>在中国大陆的视频网站上（如优酷土豆和bilibili），“卢本伟”“五五开”等关键词已成为敏感词。然而，在卢本伟的直播间被封禁之后，用户仍可正常发送弹幕、刷礼物。据[[澎湃新闻|澎湃新闻]]报道，2018年9月14日10:40分，用户在卢本伟直播间送礼时会显示“主播分数回复后才能继续赠送礼物”，但仍可正常发送弹幕<ref>{{Cite web|url=https://www.thepaper.cn/newsDetail_forward_2439863|title=陈一发儿礼物系统被关闭，但斗鱼上被封禁还能获打赏的不止她|accessdate=2018-09-15|author=|date=2018-09-14|publisher=澎湃新闻 }}</ref>。
