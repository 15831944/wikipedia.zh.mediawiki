'''卢肯伯爵案'''是一起发生在[[英国|英国]][[伦敦|伦敦]][[貝爾格萊維亞|貝爾格萊維亞]]，震动英国的刑事案件。主角理查德·约翰·宾厄姆是第七代卢肯伯爵，喜愛赌博。1974年11月7日晚，他家的保姆桑德拉·里维特被发现于地下室被钝器击打致死，卢肯则于事后离开。卢肯妻子忆述当晚她曾遭到丈夫的殴打，而卢肯的母亲于晚上亦两度接到伯爵的电话。卢肯在半夜去了朋友伊思·马克威尔-斯科特家里，其妻子苏珊接待了他，之后卢肯不知去向，两天后他借的汽车发现被人弃置在英伦海峡港。卢肯一案經七个月的審理，最後陪审团在[[缺席审判|缺席审判]]的情况下裁决卢肯為凶手。警方认为他本来想谋杀婚姻关系破裂的妻子，却误杀了保姆。<ref>{{cite web |url=http://www.my285.com/zt/ztxg/fysy/027.htm |title=辛普逊-->法医生涯四十年-->28．卢肯伯爵案 |format= |work= |accessdate=}}</ref>

卢肯长期被列入[[国际通缉令|国际通缉令]]，1999年他在法律上被[[宣告死亡|宣告死亡]]，但多年来出现多个目击报告，有的来自澳大利亚、南非。一些人认为当年他已跳海自杀，而一名侦探则出书声称他逃到印度隐居丛林。<ref>{{cite web |url=http://news.sina.com.cn/w/2004-10-17/18433948861s.shtml |title=伦敦警方咬着“伯爵”不放――英国将重新调查30年前疑案(图)_新闻中心_新浪网 |format= |work= |accessdate=}}</ref><ref>{{cite web|url=http://www.dayoo.com/gb/content/2003-09/09/content_1214686.htm|title=英伦秘密:杀人贵族逃到印度当野人|format=|work=|accessdate=|deadurl=yes|archiveurl=https://web.archive.org/web/20160310230848/http://www.dayoo.com/gb/content/2003-09/09/content_1214686.htm|archivedate=2016-03-10}}</ref>
