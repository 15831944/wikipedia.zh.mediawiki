'''卢奇亚诺·帕瓦罗蒂'''（{{lang-it|Luciano Pavarotti}}，{{bd|1935年|10月12日|2007年|9月6日|catIdx=P}}），[[義大利|義大利]][[男高音|男高音]][[歌唱家|歌唱家]]，也是20世紀後半葉的世界三大男高音之一，別號「高音C之王」。

== 出身經歷 ==
帕瓦罗蒂生於[[意大利|意大利]]的[[摩德納|摩德納]]。他父親是當地頗有名氣的業餘男高音。少年時代，他曾參與摩德納城市合唱團。而他父親雖同樣一副好嗓子，但由於天生怯場，只能在[[墨索里尼|墨索里尼]]的[[軍隊|軍隊]]裡烤[[麵包|麵包]]，曾開設糕餅店。母親則在香菸廠工作。帕瓦罗蒂在從事[[歌劇|歌劇]]演出之前曾經想當職業足球員，其後當過[[保險|保險員]]和[[教师|小學老師]]。

帕瓦羅蒂在5歲便擁有了自己的吉他玩具，來伴奏唱一些[[民俗音樂|民歌]]。這些民歌都是他憑著聽了父親播放的唱片後自己學會的。他喜歡在午飯後唱，由於嗓音很高，所以被打擾了午睡的鄰居常常向他抗議。<ref>[http://news.sina.com.tw/ents/sinacn/cn/2007-09-06/130435171557.shtml 魯契亞諾-帕瓦羅蒂Luciano Pavarotti個人資料]</ref>

== 成名之路 ==

帕瓦罗蒂的首次[[歌剧|歌剧]]演出于1961年4月29日(26歲)，在北義里吉奥·艾米利亚-罗马涅区[[歌剧院|歌剧院]]担纲主演的歌劇是[[普契尼|普契尼]]的《[[波希米亚人_(歌剧)|波希米亚人]]》，饰演男主角鲁道夫。

1965年，帕瓦罗蒂隨女高音[[瓊·蘇瑟蘭|瓊·蘇瑟蘭]]的演出團隊赴美遊學。當年2月在[[邁阿密|邁阿密]]時，蘇瑟蘭的原定搭檔因病無法獻唱，蘇瑟蘭讓帕瓦罗蒂頂替，成就他的美國首演，獲得好評。同年4月8日帕瓦罗蒂在[[斯卡拉大劇院|斯卡拉大劇院]]首演，主演了其首本名劇《[[波希米亚人_(歌剧)|波希米亚人]]》在此演出一場歌劇的酬勞為4億[[里拉_(貨幣)|里拉]]，大約相當於400[[美金|美金]]。同年6月2日在[[倫敦皇家歌劇院|倫敦皇家歌劇院]]，帕瓦罗蒂首次主唱了日後令他名成利就的大作——[[董尼才第|董尼才第]]的《[[聯隊之花|聯隊之花]]（团队的女儿）》。

帕瓦罗蒂在1969年11月20日事業有所突破，在[[斯卡拉大劇院|斯卡拉大劇院]]與當紅[[女高音|女高音]][[雷纳塔·史科朵|雷纳塔·史科朵]]共同主演[[威爾第|威爾第]]的《[[倫巴底人_(歌劇)|倫巴底人]]》，演出被錄音製成專輯，廣為流傳。此後帕瓦罗蒂開始了自己的錄音生涯，早期的錄音包括一套董尼才第／[[威爾第|威爾第]]的詠嘆調集和與蘇瑟蘭共同主唱董尼才第的《[[愛情靈藥|愛情靈藥]]》。

他的歌唱事业的最大突破发生于1972年2月17日，當他在[[纽约|纽约]]大都會歌劇院再次演唱董尼才第的《聯隊之花》期間，輕鬆地唱出了九个高音C。帕瓦罗蒂自此逐步登上歌唱事業的高峰，更在全球掀起「帕瓦罗蒂热潮」，舉重若輕的唱功，令觀眾沉醉在那妙韻，也因此得到「高音C之王」的称号。亦因如此，在演出结束后，帕瓦罗蒂獲邀出场谢幕17次，久久不願散去。<ref>{{Cite news|url=http://www.epochtimes.com/gb/7/9/6/n1825416.htm|title=世界著名男高音帕瓦罗蒂病逝 - 大纪元|date=2007-09-06|newspaper=大纪元 www.epochtimes.com|accessdate=2018-01-25|language=zh-CN}}</ref>直至今日，这依旧是世界紀錄。

== 事業頂峰 ==

帕瓦罗蒂在西方社会可谓是家喻戶曉。原因是他開了歌剧劇現成直播節目的先例，而且定期上電視節目做現場演唱。因此他吸引了一大批的歌迷。他多次獲得[[葛莱美奖|葛莱美奖]]，還有多張白金唱片和金唱片。

1980年代初，帕瓦罗蒂決定舉辦帕瓦罗蒂國際聲樂大賽，並在1982年舉行了首屆。帕瓦罗蒂國際聲樂大賽至今舉辦了五屆，最近一屆在1997年。

1990年，帕瓦罗蒂演唱普契尼《[[杜蘭朵公主|杜蘭朵公主]]（杜蘭朵）》中的《[[公主徹夜未眠|公主徹夜未眠]]（今夜无人入睡）》詠嘆調成为BBC報導[[1990年國際足協世界盃|1990年世界盃]]電視片的主題曲，令帕瓦罗蒂更為全球所知。而在當年足球[[國際足協世界盃|世界杯]]期间，帕瓦罗蒂更偕同前对手[[多明戈|多明戈]]和[[卡雷拉斯|卡雷拉斯]]在[[羅馬|羅馬]]卡拉卡拉浴場前地举办了首場「[[三大男高音演唱會|世界三大男高音演唱会]]」。而他的首次個人露天演唱会在[[伦敦|伦敦]][[海德公园|海德公园]]举行，观众人数有约15万人。1993年7月，帕瓦罗蒂在[[纽约|纽约]]中央公园的演唱会更是有破纪录的50万听众在现场欣赏，而且全球各地的歌迷透过电视直播收看这次表演。1998年，在[[法国|法国]][[巴黎|巴黎]][[艾菲尔铁塔|艾菲尔铁塔]]前地举办的“世界三大男高音演唱会”，則吸引了30万的现场听众。在此前的1994年的[[洛杉磯|洛杉磯]]和此後2002年的[[橫濱|橫濱]]都有類似的演唱會。而帕瓦罗蒂的露天演唱会的唱片和录像带的销量，超越了[[猫王|猫王]]和[[滚石乐队|滚石乐队]]的唱片销量。

帕瓦罗蒂是[[甘迺迪表演藝術中心|肯尼迪中心]]荣誉的持有人，同時還持有兩個[[吉尼斯世界记录|吉尼斯世界记录]]——最多謝幕次數（165次）和最佳銷量古典唱片（1990羅馬「三高」演唱會）。1998年他獲得了[[葛萊美獎|-{zh-hans:格莱美;zh-hk:格林美;zh-tw:葛萊美;}-傳奇藝人獎]]。

帕瓦罗蒂是[[黛安娜王妃|黛安娜王妃]]的非常要好的朋友。他们曾经一起发动过为未成年矿工筹款的全球活动。当王妃不幸车祸逝世时，帕瓦罗蒂拒绝了在王妃丧礼上唱挽歌的邀请，并表示他的「喉咙悲伤得无法献唱」。

但帕瓦罗蒂也因為常常無法獻唱，還有與各大歌劇院惡劣的關係，因而獲得「取消大王」（{{lang|en|The King of Cancellations}}）的稱號。被歌劇界裡認為最極端的情況是發生在1989年。當年帕瓦罗蒂取消了40場在[[芝加哥歌劇院|芝加哥歌劇院]]裡的26場演出，開罪身為劇院經理的多年好友，阿迪斯·卡拉尼克（{{lang|en|Ardis Krainik}}）。卡拉尼克因此決定帕華洛帝終身不得再在芝加哥歌劇院演出。{{ref|1}}

2004年，帕瓦罗蒂在纽约[[大都会歌剧院|大都会歌剧院]]，最后一次主演歌剧《[[托斯卡|托斯卡]]》全剧，并宣布以后不会主演任何歌剧全剧。2005年，年届69岁的帕瓦罗蒂举办了一连串全球巡回告别演唱会，包括[[香港|香港]]（2005年12月2日）、[[上海|上海]]（2005年12月6日）、[[北京|北京]]（2005年12月10日）和[[台中|台中]]（2005年12月14日）。在完成全球巡回演唱後，帕瓦罗蒂將正式地退休。但他的健康狀況使他的告別演唱會斷斷續續，最終因他的去世而不能完成。

== 晚年 ==
2005年，帕瓦罗蒂曾接受頸部手術，修復兩節脊椎。同年3月，帕華洛帝因喉嚨發炎被迫缺席[[墨西哥|墨西哥]]的三大男高音演唱會。

2006年初，帕華洛帝接受背部手術後併發症感染，被迫取消英、美、[[加拿大|加]]三地的演唱會。{{ref|2}}

同年7月，帕瓦罗蒂被診斷出患[[胰腺癌|胰腺癌]]，需立即接受開刀除去腫瘤。手術過後，很快傳出帕華洛帝「復康進展良好」的報導。但他在2006年餘下的活動全部取消，根據帕華洛帝經理人的說法，帕瓦罗蒂將在2007年初，恢復環球告別演唱會。{{ref|3}}

2007年8月中旬時，因發高燒，曾在義大利摩德納醫院住上10天，在8月25日出院回家，9月4日，意大利傳媒傳出他病情突然惡化、不醒人事，及在家中接受醫師照料的消息。9月6日凌晨，因患[[胰腺癌|胰腺癌]]在摩德纳家中去世，终年71岁。<ref>杨爱国，[http://news.xinhuanet.com/newscenter/2007-09/06/content_6675974.htm 详讯：意大利著名男高音歌唱家帕瓦罗蒂去世]，新华网</ref>

== 逝世 ==
帕瓦罗蒂的經紀人[[特里·羅伯森|特里·羅伯森]]以電子郵件告知[[美聯社|美聯社]]帕瓦罗蒂的死訊，信件內寫著：「這位藝術大師與胰腺癌作了長期艱苦的抗爭，但病魔最終還是結束了他的生命。一如他對待生活和工作的態度一樣，他在生命的最後階段仍保持樂觀。」同日，[[維也納國家歌劇院|維也納國家歌劇院]]升起黑旗以示悼念。

去世兩日後，[[摩德纳大教堂|摩德纳大教堂]]舉行帕瓦罗蒂的[[追思彌撒|追思彌撒]]，教堂外設有8部大型電視供教堂外的歌迷觀看追思彌撒。追思彌撒開始前由巴伐洛堤的[[保加利亞|保加利亞]][[女高音|女高音]]好友高歌一曲；追思彌撒在當地時間15:20分正式開始，隨即進行宗教儀式。在祝福禮後，教堂內外播放早年由帕瓦罗蒂與父親同台演唱一段[[聖詩|聖詩]]，其後掌聲達三分鐘。在追思彌撒過後，當帕瓦罗蒂的棺木移離教堂時，[[意大利空軍|意大利空軍]]在教堂上面飛過，還噴出有[[意大利國旗|意大利國旗]]象徵的綠、白、紅三色，以代表意大利政府向帕瓦罗蒂的敬意。<ref>[http://agostinho.exblog.jp/d2007-09-08 Agostinho之思彥新聞日記]</ref>

[[法國|法國]]總統[[尼古拉·萨科齐|-{zh-hans:萨科齐;zh-hk:薩爾科齊;zh-tw:薩柯奇;}-]]在帕瓦罗蒂去世後讚美他是在[[恩里科·卡魯索|恩里科·卡魯索]]之後最偉大的古典歌唱家；[[維也納国家歌劇院|維也納国家歌劇院]]掛上黑旗，以及[[美国|美国]][[纽约|纽约]][[大都会歌剧院|大都会歌剧院]]也降半旗致哀；[[聯合國|聯合國]]秘書長[[潘基文|潘基文]]表示，聯合國失去了一位偉大的扶貧信使；美國總統[[乔治·沃克·布什|-{zh-hans:布什;zh-hk:布殊;zh-tw:布希;}-]]說，這世界失去了一位「偉大的藝術家和慈善家」；歌唱家[[莫華倫|莫華倫]]對帕瓦罗蒂病逝，感到可惜和震驚；形容他的聲音、技巧和表演也是完美，地位無人可以代替；又讚揚帕瓦罗蒂為人沒架子。

帕瓦罗蒂在追思彌撒後被安排[[火葬|火葬]]，之後在其家族墓園[[安葬|安葬]]。

== 家庭 ==
帕瓦罗蒂与他的助手结婚，并诞下一对[[双胞胎|双胞胎]]，但只有其中一女兒存活。而帕瓦罗蒂和前妻阿杜瓦育有3名女儿。

== 人道工作 ==
多年來，帕瓦罗蒂為慈善不遺餘力，多次舉辦慈善演唱會。1996年，他與他的朋友舉辦了「帕瓦罗蒂與朋友－為波斯尼亞兒童」慈善演唱會。1998年，他們再次合作舉辦「帕瓦罗蒂與朋友－為利比利亞兒童」舉辦慈善演唱會。<ref>[http://www.amazon.com/Together-Children-Bosnia-Nenad-Bach/dp/B000004297 Together for the Children of Bosnia],Amazon</ref>1999年6月1日，帕瓦罗蒂與朋友在自己的家鄉－[[摩德納|摩德納]]舉辦「帕瓦罗蒂與朋友－為[[危地馬拉|危地馬拉]]和[[科索沃|科索沃]]兒童」慈善演唱會。2000年，他們再次舉行慈善演唱會－「帕瓦罗蒂與朋友－為[[柬埔寨|柬埔寨]]和[[西藏|西藏]]」。<ref>[http://www.imdb.com/title/tt0301711/ IMDB]</ref>

1998年，他獲得[[聯合國|聯合國]]任命為「和平使者」<ref>[http://www.un.org/chinese/aboutun/faq/peace.htm 聯合國和平使者]</ref>，來協助喚醒世界對[[千年發展目標|千年發展目標]]、[[愛滋病|愛滋病]]、[[兒童權益|兒童權益]]、[[貧民窟|貧民窟]]及[[貧窮|貧窮]]等問題。與他合作的歌手來自四方八面，都是為戰火的兒童籌款。受惠的人士包括了[[巴基斯坦|巴基斯坦]]內的[[阿富汗|阿富汗]]難民、[[贊比亞|贊比亞]]內的[[安格拉|安格拉]]難民、[[伊朗|伊朗]]境內的[[伊拉克|伊拉克]]難民等等。<ref>[http://www.un.org/News/Press/docs/2005/org1441.doc.htm LUCIANO PAVAROTTI TO PROMOTE UN CAUSES DURING SERIES OF CONCERTS, 2005 - 2006]</ref> 2001年，他獲得[[聯合國|聯合國]]的難民署授予「[[南森獎章|南森獎章]]」來表彰他對援助難民所作出的重大貢獻。<ref>[http://www5.chinesenewsnet.com/MainNews/Topics/zxs_2007_1020760.shtml 聯合國悼念“和平使者”帕瓦羅蒂 贊其杰出貢獻]</ref> 另外，他在戰後的[[波斯尼亞|波斯尼亞]]南部城市[[莫斯塔爾|莫斯塔爾]]自資興建[[帕瓦罗蒂音樂中心|帕瓦罗蒂音樂中心]]，讓波斯尼亞的藝術家有機會磨練自己的技術。因此，他在2006年成了[[薩拉熱窩|薩拉熱窩]]榮譽市民。<ref>[http://findarticles.com/p/articles/mi_qn4188/is_20060222/ai_n16170550 Sarajevo authorities name Pavarotti honorary citizen]; retrieved on 2007-09-06</ref>

== 軼事 ==
義大利具影響力的《[[晚郵報|晚郵報]]》曾披露被譽為「世界第一男高音」的帕瓦罗蒂承認自己不識樂譜。  
  
在義大利中部[[翁布里亞|翁布里亞]]的[[卡斯泰洛城|卡斯泰洛城]]舉行的一次活動上，帕瓦罗蒂和另一名知名義大利演唱家[[加斯曼|加斯曼]]合作表演二重唱。加斯曼於綵排時發現帕瓦罗蒂無需樂譜，便意識到帕瓦罗蒂不懂樂譜，從此帕瓦罗蒂不懂樂譜的傳聞便傳開了。   
   
帕瓦罗蒂其後接受《晚郵報》採訪後表示：「這是真的，我不懂樂譜。」坦承後引起樂迷廣為討論，褒貶不一。   

在中央电视台《东方时空》中，主持人、记者水均益采访了帕瓦罗蒂并涉及此事，帕瓦罗蒂澄清，他只是不懂交响乐的乐谱。帕瓦罗蒂曾经教过音乐，必须得识谱。帕瓦罗蒂对此也并不在意，並感谢《东方时空》和水均益为他提供一个澄清的机会。
   
除此之外，帕瓦罗蒂的飲食習慣往往令人不敢恭維，其在[[斯卡拉劇院|斯卡拉劇院]]的第一筆薪水便拿來購買金屬搾汁機。成名之後，行李中常攜帶精選的肉類。一次入住[[邁阿密|邁阿密]]的酒店，竟要求為自己一套專用的肉類切片機，經理深怕得罪而照辦。且帕瓦罗蒂無論入住哪一家酒店都有將被單和枕頭弄髒的習慣。但是帕瓦罗蒂也不只一次的在公眾拿自己的食慾來開玩笑，深受樂迷歡迎。

關于帕瓦罗蒂金嗓子的秘密，可要歸功于他獨特的發音技巧。除了一般傳統的“面罩唱法”，還有“意大利式美聲唱法”。面罩唱法使用共鳴的原理發聲，可是人到達老年的時候，聲帶疲軟無法再開聲，歌唱技巧就開始走下坡。

而-{zh-hans:意; zh-hk:意; zh-tw:義;}-式唱法改用其他部位發音，所以帕瓦罗蒂到去世為止，世人都能夠聽見他美妙的歌聲。
