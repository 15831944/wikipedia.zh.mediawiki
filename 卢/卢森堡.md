'''卢森堡大公国'''（{{lang-lb|Groussherzogtum Lëtzebuerg}}{{NoteTag|发音：{{IPA|ˈgʀəʊ̯sˌhɛχt͡soːktuːm ˈlət͡səbuɐ̯ɕ}}}}；{{lang-fr|Grand-Duché de Luxembourg}}； {{lang-de|Großherzogtum Luxemburg}}{{NoteTag|{{IPA-de|ˈgʁoːsˌhɛʁt͡soːktuːm ˈlʊksəmˌbʊʁk|pron}}}}），通称'''卢森堡'''（{{lang-lb|Lëtzebuerg}}；{{lang-fr|Luxembourg}}；{{lang-de|Luxemburg}}），被邻国[[法国|法国]]、[[德国|德国]]和[[比利时|比利时]]包围，是一個位于[[欧洲|欧洲]]的[[内陸國家|内陸國家]]，也是現今[[歐洲大陸|歐洲大陸]]僅存的[[大公國|大公國]]，[[首都|首都]][[盧森堡市|盧森堡市]]。盧森堡是[[欧盟|欧盟]]成员国，因境内有[[欧洲法院|欧洲法院]]、[[欧洲审计院|欧洲审计院]]、[[欧洲投资银行|欧洲投资银行]]等多个欧盟机构被称为继[[布鲁塞尔|布鲁塞尔]]和[[斯特拉斯堡|斯特拉斯堡]]之后的欧盟“第三首都”。

== 历史 ==
{{main|盧森堡歷史}}
卢森堡历史由在[[中世纪|中世纪]]时卢森堡城堡的修建开始。城堡建成之后，附近开始逐渐发展，成为城镇。卢森堡位于战略地点，[[法国|法国]]、[[德国|德国]]和[[荷兰|荷兰]]都非常重视，卢森堡经常转手不同国家。19世纪，卢森堡开始成为独立自主的国家。

可是，虽然卢森堡声明中立原则，但在20世纪期间被[[德国|德国]]数次入侵。第二次世界大战期间，拥有数十个步骑兵的卢森堡遭到德国侵略，被迫作为同盟国参战，并且在德国的进攻下瞬间投降，成为二战中唯一一个零伤亡的参战国。[[第二次世界大战|第二次世界大战]]之后，卢森堡不再維持[[中立|中立]]，成为[[北約|北約]]和[[欧盟|欧盟]]的創始會員國之一。

== 政治 ==
[[File:Chamber_of_Deputies_of_Luxembourg.JPG|thumb]]
{{main|盧森堡政治|卢森堡政黨列表}}
盧森堡實行[[君主立憲制|君主立憲制]]。國家元首為盧森堡大公，也是目前歐洲唯一的一個[[大公國|大公國]]。而行政權則由內閣行使。國會共有60個席位，議員任期為5年。

== 地理 ==
[[File:Kutter_Luxembourg.jpg|缩略图]]
{{main|盧森堡地理}}
盧森堡位於西歐內陸，地勢北高南低，東鄰德國，南接法國，北部和西部同比利時接壤。北部為-{zh-hans:阿登高原; zh-hant:亞爾丁高地;}-，森林茂密，南部為丘陵。氣候溫和，屬溫帶海洋性氣候，風景優美。首都盧森堡城有“花都”之稱。鐵礦豐富。這裡也是中世紀的要塞。最高點為[[布爾格普拉茲峰|布爾格普拉茲峰]]，海拔約550-{zh-hans:米; zh-hant:公尺;}-。

== 经济 ==
{{main|盧森堡經濟}}
卢森堡擁有全球最高的人均GDP。自1999年以來，卢森堡一直是[[歐元區|歐元區]]的一部分。盧森堡的經濟過去以工業為主，現在盧森堡則是全球最大的金融中心之一。盧森堡是[[歐元區|歐元區]]內最重要的私人銀行中心及全球第二大的投資[[信托|信托]]中心（僅次於[[美國|美國]]）。

=== 民族 ===
[[卢森堡人|卢森堡人]]约占全国人口的76%。卢森堡人属于[[欧罗巴人|欧罗巴人]]和大西洋-波罗的海类型。

=== 语言 ===
{{main|盧森堡语言}}
[[File:Am_Centre_Mercure_zu_Esch.jpg|thumb]]
[[File:Galerie_Beaumont-101.jpg|缩略图]]
[[File:Mullerthal.jpg|thumb]]
卢森堡的[[官方语言|官方语言]]是[[法语|法语]]、[[德语|德语]]、[[卢森堡语|卢森堡语]]。

[[File:LUX_Luxembourg_Clausen_006_2016.jpg|thumb]]
“十字路口国家”的多语言现象都是由于不同民族杂居，而卢森堡却不然，传统上这个国家的主体民族只有一个———卢森堡族，这个民族血缘上和法国相近，但历史上却长期是德语系的“[[神圣罗马帝国|神圣罗马帝国]]”一部分，而欧洲宫廷上层又普遍有用法语交流的传统习惯，这就造成德法两种语言都成为书面语言的现象，而长期未曾书面化的卢森堡语，则作为[[口语|口语]]长期沿用。

卢森堡语是[[弗兰康阶|弗兰康阶]]语的一支，使用人口主要分布在[[莫泽尔|莫泽尔]]区以及接壤的德、法城市。卢森堡语虽属于[[高地德语|高地德语]]的中西语族，但有超过5000个法语语源词汇。作为母语，卢语亦卢森堡人看做是“心之语言”，用于日常口语而非书面语。

教育体制中卢、德、法三语循序渐进，并行不悖。小学低年级用卢森堡语授课，高年级开始用德语讲习，中学开始再转化成法语。熟练掌握这三门语言是当地中学毕业的必要条件，这就导致了半数的中学生都拿不到毕业文凭。在这种体制之下，[[移民|移民]]（尤其是非欧洲移民）子女尤其吃亏。

卢森堡的外国侨民特别多，占全国人口的三成以上，最大的移民团体是[[葡萄牙人|葡萄牙人]]和[[意大利人|意大利人]]。他们也同时带来了自己的语言。不过，葡萄牙语和意大利语基本只限于移民团体内部交流，在大范围内运用并不广泛。

=== 宗教 ===
多數信奉[[天主教|天主教]]，亦有部分信奉其他宗教（包含[[基督新教|基督新教]]和[[犹太教|犹太教]]）。

== 文化 ==
盧森堡城的老城區及其防御工事為[[世界文化遺產|世界文化遺產]]。卢森堡的许多菜肴都受到德国、法国、比利时和其他邻近的国家的影响。其中kach keis是一种涂抹奶酪，Gromper keeschelche是一种油炸馬鈴薯饼。

== 教育 ==
=== 教育制度 ===
[[File:Bettendorf,_Luxembourg.JPG|thumb]]
在盧森堡，國民[[義務教育|義務教育]]是11年：由2年幼儿园、6年基础教育和3年進階教育組成。盧森堡因此有一個三層的教育制度（幼稚園、小學、高中）。{{Fact|學生就學率達到100%|time=2017-11-27T00:10:20+00:00}}。{{Or|盧森堡的教育制度有許多問題，例如花費許多時間在德語、法語和英語的語言課程上，以及特別是從德語課程不成功、講羅曼語族語言的國家來的外國學生佔了相當高的比例|time=2017-11-27T00:10:20+00:00}}。

自從1999年以後，在2年的幼儿园之前設有1年的學齡前兒童預備班；這使得學童在例如語言、社會、學校方面，有可能獲得一個較好較完整的[[社會化|社會化]]。在6年的基礎教育之後學生可以在兩種中級的教育路途之間選擇：

# 科技高級中學（67%的學生），為期至少3年，並且到畢業（[[CATP|CATP]]）或能進大學之前總共至少6至7年。
# 普通高級中學（33%的學生）。7年後畢業拿到證書可以進大學就讀。

{{Fact|這個教育體制包含佔國家經費第二高的社會預算的11.3%。但其15歲學生在[[国际学生能力评估计划|国际学生能力评估计划]]中的成績一般。|time=2017-11-27T00:10:20+00:00}}

=== 高等教育 ===
{{main|卢森堡大学列表}}
自從2003年以後盧森堡有一間獨立的大學，其從2001開始建造。在醫學領域、自然科學、科技、法律、經濟、社會學和教育學，學生在這分類成7個的學院裡接受教育。然而盧森堡並未具備完整的學年，在外國修業更多學期的課程是必要的。對此[[比利時|比利時]]、法國和德國的大學特別扮演一個重要的角色。盧森堡的學生目前大部分仍然是在外國獲得學術的畢業文憑，其在外國就讀的學生超過6000人。大約19.6%的學生在德國大學註冊（特別是在科技的專門領域或自然科學），而22.6%的學生是在比利時和法國的大學就讀。

建造了盧森堡大學之後，盧森堡應該加強學術研究地位，並且穩固盧森堡的經濟基礎。此外應該擴大招收外國學生來盧森堡就讀。然而同時盧森堡在承認外國學校的畢業文憑又有困難，特別是對於德國的專科大學畢業生，例如在德國[[EG|EG]]建築準則要求4年的工作實習經驗，在盧森堡卻得不到證明。對此盧森堡有一間自己的專科大學Institut Supérieur de Technologie。

=== 職業教育 ===
科技進階教育由三年額外的、特別的學年組成，下列是三種教育方式：

# 3年工藝的或是工商業的職業訓練，不僅在企業和法國公立科技中學（Lycée Technique）之間的二元系統下進行，也實現在一個純粹的學校基礎上。畢業時有科技的和職業的能力認證考試[[CATP|CATP]]。這種職業教育的方式在缺乏學徒的許多職業中是顯著的。
# 4年的科技教育，獲得科技文憑。
# 在科技中心4年的教育，可獲得一個科技進階教育的畢業證書。

==軍事==
{{main|盧森堡軍事}}
盧森堡有盧森堡陸軍和盧森堡空軍，由於盧森堡是一個內陸國家，盧森堡不擁有海軍。
