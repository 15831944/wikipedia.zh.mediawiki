'''卢霖'''（又名'''卢林'''，{{bd|1947年|||}}），[[四川省|四川省]][[蓬安县|蓬安县]]人，擅长[[水墨画|水墨画]][[肖像|肖像]]作品。风格融合[[东方艺术|东]][[西方艺术|西方艺术]]，作品曾多次在博物馆展出，并被海内外[[收藏家|收藏家]]收藏<ref name="o5000">{{cite web |url=http://old.zhsh5000.com/grcompany.asp?id=3387 |title=艺术家卢霖简介 |last= |first= |date= |website=艺术官方网站 |publisher= |access-date= |quote=}}</ref>。

== 生平 ==
* 1947年，出生于[[重庆市|重庆市]]一个普通的工人家庭<ref name="发展中国">{{cite web |url=http://www.cdf.bj.cn/LiterArt/details.aspx?id=8103 |title=名家观察｜卢林艺术作品 |last= |first= |date=2018-09-17 |website=发展中国 |publisher= |access-date=2018-11-18 |quote=}}</ref>。
* 1960年，被[[四川美术学院|四川美术学院]]附中录取，后师从[[沈福文|沈福文]]、[[肖建初|肖建初]]、[[李有行|李有行]]、[[冯建吴|冯建吴]]<ref>{{cite web |url=http://news.163.com/14/0307/06/9MNDF20000014AED.html |title=画家卢霖 隐身小城的大家(组图) |last= |first= |date=2014-03-07 |website=网易 |publisher=四川在线-华西都市报(成都) |access-date=2018-12-26 |quote=}}</ref>。
* 2010年，作品于美国[[华盛顿州|华盛顿州]]近郊的P&S画廊中展出<ref name="o5000" />。
* 2015年，3幅[[水墨画|水墨画]]作品《[[诗琳通公主肖像|诗琳通公主肖像]]》、《古镇码头》和《濂溪祠·爱莲池》于“庆祝中泰建交40周年暨诗琳通公主殿下60寿辰中国书画慈善展”展出<ref>{{cite web |url=http://www.xinhuanet.com/overseas/2015-09/22/c_128254081.htm |title=中泰建交40年 中国艺术家献礼泰国公主60寿辰 |last= |first= |date=2015-09-22 |website=新华网 |publisher=华西都市报 |access-date=2018-11-18 |quote=}}</ref><ref>{{cite web |url=http://www.sohu.com/a/78304018_119824 |title=[人物]蓬安七旬翁 画作赠给泰国公主 |last= |first= |date=2016-05-29 |website=搜狐 |publisher=南充新闻网 |access-date=2018-11-18 |quote=}}</ref>。
