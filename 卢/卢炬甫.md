'''卢炬甫'''（{{bd|1947年|8月|2017年|11月3日}}），出生于[[湖北|湖北]][[大治乡|大治]]<ref name=":1">{{Cite web|url=http://www.hmyzg.com/2006-xmzg/news/2004/0407-0409month/040705-zg.htm|title=探天者的人生轨迹——记中国科技大学教授卢炬甫|accessdate=2017年11月14日|author=|date=|publisher=}}</ref>，中国[[天体物理学|天体物理学]]家，研究领域为黑洞[[吸积盘|吸积盘]]理论及其[[高能天体物理|高能天体物理]]应用，包括[[活动星系核|活动星系核]]、[[X射线双星|X射线双星]]、[[伽玛射线暴|伽玛射线暴]]等。第九届全国政协委员、第十届全国政协委员、[[第十一届全国政协委员|第十一届全国政协委员]]。

== 学术生涯 ==
1970年本科毕业于[[中国科学技术大学|中国科学技术大学]]化学专业。<ref name=":1" />

1985年在意大利国际高等研究生院获博士学位，导师为{{link-pl|马瑞克·阿图尔·阿布拉莫维奇|Marek Abramowicz}}。

1992年起在中国科学技术大学任教授。​1993年起享受[[国务院政府特殊津贴|国务院政府特殊津贴]]。

2000年起在[[厦门大学|厦门大学]]任教授。主持国家自然科学基金项目6项、教育部高校博士点基金项目项、中国科学院院长基金1项、参加国家科技部攀登计划项目2项、中国科学院知识创新工程项目1项。<ref name=":0">{{Cite web|url=http://astro.xmu.edu.cn/astro/lujufu|title=卢炬甫教授的个人主页|accessdate=2017年11月14日|author=|date=|publisher=|archive-url=https://web.archive.org/web/20171114202244/http://astro.xmu.edu.cn/astro/lujufu|archive-date=2017-11-14|dead-url=yes}}</ref>

曾担任[[中国天文学会|中国天文学会]]恒星专业委员会副主任、中国引力相对论学会理事、[[国际天文学联合会|国际天文学联合会]]会员，国家自然科学基金委员会数理科学部专家咨询委员会委员，国家科技部973计划咨询专家。<ref name=":1" /><ref name=":0" />　

== 社会活动 ==
加入[[中国致公党|中国致公党]]，担任致公党中央委员、福建省委副主委。

1998年，当选[[第九届全国政协委员|第九届全国政协委员]]，代表教育界。

2003年，当选[[第十届全国政协委员|第十届全国政协委员]]，代表教育界。

2008年，当选[[第十一届全国政协委员|第十一届全国政协委员]]<ref>{{cite news|title=第十一届全国政协委员名单|url=http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|accessdate=2013-04-09|newspaper=新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过)|date=2008年1月25日|deadurl=yes|archiveurl=https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|archivedate=2013年9月28日}}</ref>，代表教育界，分入第四十二组。<ref>{{cite news|title=全国政协信息-卢炬甫|url=http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0013790&wytype=wjwy|accessdate=2013-04-11|newspaper=全国政协委员会}}</ref>

== 荣誉 ==
[[中国青年科学家奖|第一届中国青年科学家奖]]获奖者。

关于黑洞的研究成果被国外学者称为"卢和Abramowicz的唯一性定理"。
