'''卢奇'''{{BD|1953年|8月21日|||catIdx=L}}，出生于[[四川省|四川省]][[凉山彝族自治州|凉山彝族自治州]][[甘洛县|甘洛县]]，毕业于部队[[文工团|文工团]]，是中国大陆影视演员，以出演“[[邓小平|邓小平]]”、“[[孙中山|孙中山]]”而知名。<ref name=" cat" >{{cite web |url=http://news.xinhuanet.com/yzyd/local/20121210/c_113973409.htm |title=卢奇 我与领袖在一起 |accessdate=2012 |publisher=新华网 |archiveurl=http://news.xinhuanet.com/yzyd/local/20121210/c_113973409.htm |archivedate=2013年1月2日 |work= |language=zh-cn}}</ref>

==生平==
1953年，卢奇出生在[[四川省|四川省]][[凉山彝族自治州|凉山彝族自治州]][[甘洛县|甘洛县]]，曾当过兵，在部队里的[[文工团|文工团]]学习过演员，后来在[[四川省人民艺术剧院|四川省人民艺术剧院]]当演员，最后转入[[八一电影制片厂|八一电影制片厂]]工作。<ref name=" cat" />

卢奇第一次出演邓小平是在1979年电影《山城雪》。<ref name=" cat" />

2012年，卢奇收下第一名弟子[[宜宾市|宜宾市]][[江安县|江安县]]人缪孝全，缪孝全爱好表演，喜欢模仿邓小平。<ref>{{cite web |url=http://www.ybxww.com/content/2012-11/30/20121130155245.316128.htm |title= 特型演员卢奇在川南收下首名弟子|accessdate=2012-11-30 |publisher= 宜宾新闻网|archiveurl= http://www.ybxww.com/content/2012-11/30/20121130155245.316128.htm|archivedate=2013年1月2日 |work= |language=zh-cn}}</ref>
