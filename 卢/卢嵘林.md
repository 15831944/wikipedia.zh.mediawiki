'''卢嵘林'''（{{bd|1954年|||}}）[[河北|河北]][[涞源|涞源]]人，[[中华人民共和国|中华人民共和国]]官员。

==生平==
1969年，卢嵘林参加工作。1971年加入[[中国共产党|中国共产党]]。参加工作后，卢嵘林历任[[中国人民解放军空军第九航空学校|中国人民解放军空军第九航空学校]]二团场站油料股化验员、技术员、股长，副政委、副团长，[[中国人民解放军空军钟祥场站|中国人民解放军空军钟祥场站]]站长、党委副书记、常委，[[中国人民解放军空军武昌场站|中国人民解放军空军武昌场站]]站长、党委副书记，[[中国人民解放军广州军区空军|中国人民解放军广州军区空军]]航空兵训练基地副司令员（1996年4月到1997年12月兼任中国人民解放军广州军区空军航空兵训练基地参谋长）。此后转业到地方，历任[[中共湖北省纪委|中共湖北省纪委]]派驻湖北省药品监督管理局纪检组副组长、[[湖北省监察厅|湖北省监察厅]]派驻湖北省药品监督管理局监察室主任，湖北省药品监督管理局人事教育处处长，[[湖北省食品药品监督管理局|湖北省食品药品监督管理局]]人事教育处处长，[[湖北省工商行政管理局|湖北省工商行政管理局]]纪检组组长、[[党组|党组]]成员<ref>{{Cite web|url = http://www.egs.gov.cn/info/iList.jsp?tm_id=312|title = 卢嵘林|author = |date = |publisher = 湖北省工商行政管理局|language = |accessdate = 2017-07-13|quote = }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web|url = http://www.egs.gov.cn/wzlm/zdgd/khfzg/wjfa/16580.htm|title = 省工商局开展深入学习实践科学发展观活动实施方案|author = |date = 2008-10-14|publisher = 湖北省工商行政管理局|language = |accessdate = |quote = }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url =http://www.hubfda.gov.cn/zxbw/gztz/jt/1696.htm   |title =关于印发《省食品药品监督管理局关于机关作风整顿的实施方案》的通知[鄂食药监文〔2007〕41号]  |author =  |date = 2007-06-02 |publisher = 湖北省食品药品监督管理局 |language =  |accessdate =  |quote =  }}</ref>。
