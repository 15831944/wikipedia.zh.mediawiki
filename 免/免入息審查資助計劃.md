== 資助金額及發放 ==
資助計劃於2017/18學年生效，每名學生每學年可獲定額的資助金，亦包括已入讀合資格課程的舊生，資助金在合資格課程的正常修業期內發放，學生只需支付扣除資助金額後的學費<ref>{{cite news |title=學費「減磅」平過八大 考生紛投自資學位 |url=http://std.stheadline.com/daily/article/detail/1631388-%E6%B8%AF%E8%81%9E-%E5%AD%B8%E8%B2%BB%E3%80%8C%E6%B8%9B%E7%A3%85%E3%80%8D%E5%B9%B3%E9%81%8E%E5%85%AB%E5%A4%A7+%E8%80%83%E7%94%9F%E7%B4%9B%E6%8A%95%E8%87%AA%E8%B3%87%E5%AD%B8%E4%BD%8D |work=星島日報 |date=2017-07-13}}</ref>，資助金將按實際入讀相關課程的合資格學生人數直接發放予參與院校，而有需要的學生仍可就其實際應繳交的學費申請學生資助。資助計劃的資助金額將跟隨綜合消費物價指數而調整<ref>{{cite news |title=自資學士資助額增至3.08萬 |url=http://news.wenweipo.com/2018/04/26/IN1804260050.htm |work=文匯報 |date=2018-04-26}}</ref>，計劃於2017/18學年開始時的資助金額為每名學生30,000港元，於2018/19學年的資助額增至30,800港元<ref>{{cite news |title=自資課程資助額增至$30800　新舊生均適用 |url=https://hk.on.cc/hk/bkn/cnt/news/20180427/bkn-20180427003054534-0427_00822_001.html |work=東方日報 |date=2018-04-27}}</ref>。

== 合資格院校 ==

== 受計劃資助的學生資格 ==
資助計劃提供免入息審查資助予下列合資格的香港學生<ref>{{cite web |title=免入息審查資助計劃 - 常見問題 |url=https://www.cspe.edu.hk/resources/pdf/tc/NSS/NSS_FAQ_TC.pdf}}</ref>：
#如報讀合資格的自資首年學士學位課程：須在[[香港中學文憑考試|香港中學文憑考試]]取得「3322」的成績（中國語文科及英國語文科取得第3級成績，數學科必修部分、通識教育科取得第2級成績）。
#如報讀合資格的自資銜接學位課程：須具有副學位資歷（即完成經本地評審的副學士學位或高級文憑課程及取得相關學歷）。

計劃資助的學生不設年齡限制，但受資助的課程均為全日制課程，如有關課程接納合併不同年度的香港中學文憑考試成績亦符合資格。資助計劃不接納以國際預科文憑、普通教育文憑高級程度考試等代替香港中學文憑考試，英國語文科的成績亦不接納以其他英國語文資歷代替。如報讀合資格的自資銜接學位課程必須取得本地認可的副學位學歷，即使取得專業文憑或其他類型以至更高程度的學位亦不符合資格。在香港中學文憑考試沒有「3322」成績的學生，如報讀自資首年學士學位課程並不符合資格，但仍可在取得本地認可的副學位學歷後，報讀合資格的自資銜接學位課程而得到計劃的資助。
