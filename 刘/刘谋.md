'''刘谋'''（1991年4月17日-，綽號'''pdd'''、'''骚猪'''、'''嫖老师'''、'''香油猪皮''' 等等），中国大陆《[[英雄联盟|英雄联盟]]》玩家，前英雄联盟职业战队iG上单选手兼队长。原[[战旗tv|战旗tv]]主播，2016年6月25日开始在[[熊猫TV|熊猫TV]]进行直播。<ref>[http://www.18touch.com/news/show-9-546-1.html PDD熊猫直播时间及直播间公布 25日输一盘送一万现金]</ref>2017年12月后长期停播至2019年3月25日，开始在【斗鱼】直播。

==早年经历==

刘谋于1991年4月17日出生于[[四川|四川]][[成都|成都]]，从小热爱电子游戏，并在《[[红色警戒|红色警戒]]》、《[[反恐精英|CS]]》、《[[帝国时代|帝国时代]]》等游戏中展示了出色的表现。到后来又转玩《[[DOTA|DOTA]]》，并自称“路人王”。<ref name="PDD法拉利">{{Cite web |url = http://games.qq.com/a/20160720/044060.htm |title = PDD：曾与女友吃不起肯德基 现送她法拉利 |date = 2016年7月15日 |publisher = 腾讯游戏 |language = zh-cn }}</ref>

2006年，刘谋加入了《[[澄海3C|澄海3C]]》，在此之后，由于他的水平很高，使得他在网吧里出了名。随后刘谋又转战[[浩方电竞平台|浩方电竞平台]]，在此期间，他的技术不断提高，他的名声也传开了，并在浩方里成为了名人。到后来，一个名叫“龙哥”的人在成都市组织了一个澄海3C大赛，并邀请PDD参赛，于是，PDD组织了一个名叫“5C”的战队并参加了比赛，在随后的比赛中，PDD所在的5C战队一路打进决赛，但在总决赛中，由于PDD的一个失误，使得5C战队仅获得亚军。<ref>{{Cite web |url = http://lol.shandian.biz/614.html |title = PDD刘谋人秘史，揭秘胖弟弟追初恋彭丹丹当网管等趣事 |date = 2015年12月14日 |publisher =  LOL闪电站 |language = zh-cn }}</ref>

在《[[英雄联盟|英雄联盟]]》刚进入中国大陆时，他加入《英雄联盟》，并主玩ADC位置。<ref name="PDD法拉利"/>

==职业生涯==

2010年，刘谋独自从四川来到上海，并成为一名电子竞技职业选手。2011年6月，他加入NGG战队，并担任ADC位置。随后他参加了2011年TGA大奖赛，但在半决赛上，NGG负于[[Team_WE|WE]]，仅获得季军。<ref>[http://lol.178.com/201406/t_196908076280.html iG.PDD职业生涯回顾]</ref>2011年8月，他加入{{link-ru|EHOME|EHOME}}战队。在此之后，他还帮助EHOME战队夺得EHOME广州站第四名，<ref>{{Cite web |url = http://games.qq.com/a/20160615/007116.htm#p=1 |title = 网瘾少年变身超级偶像 PDD自食其力赚千万买法拉利 |date = 2016年6月15日 |publisher = 腾讯游戏 |language = zh-cn }}</ref>但在2011年11月的TGC2011英雄联盟决赛上，EHOME战队最终不敌WE战队，仅夺得亚军。<ref>[http://news.pcgames.com.cn/tgc/1111/2378408.html TGC2011 英雄联盟决赛 WE vs EHOME]</ref>2012年，EHOME战队在IEM6全球挑战赛上以1胜4负的成绩退场，在此之后PDD选择暂时退役。<ref>{{Cite web |url = http://wangyou.pcgames.com.cn/zhuanti/lol/news/guonei/1203/2469831.html |title = 人生若只如初见 EHOME灵魂PDD宣布退役 |date = 2012年3月11日  |publisher = 太平洋游戏网 |language = zh-cn }}</ref>

在[[英雄联盟2013全明星赛|英雄联盟2013全明星赛]]上，刘谋操控的扎克被shy操控的杰斯对线，在杰斯杀掉带有被动的扎克（当扎克生命值为0后，触发被动变成四块粘液，只要有一块粘液存活就可复活）后，杰斯1炮轰死这四块粘液，后来 shy 因此获得不少知名度。<ref>[http://xin.tgbus.com/lol/144443.shtml LOL职业选手心中永远的痛！1炮轰死4个PDD]</ref>

北京时间2014年6月15日20时，PDD通过微博宣布正式退役。<ref>{{Cite web |url = http://game.zol.com.cn/460/4609855.html |title = 英雄联盟iG战队上单PDD正式宣布退役 |author = 王晓帆 |date = 2014年6月17日 |publisher = 中关村在线 |language = zh-cn }}</ref>
