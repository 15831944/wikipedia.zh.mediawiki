'''刘亚苏'''（{{bd|？|||}}）籍贯[[安徽省|安徽省]][[宿县|宿县]]，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
刘亚苏早年任[[中国人民解放军兰州军区|中国人民解放军兰州军区]][[中国人民解放军陆军第四十七集团军|陆军第四十七集团军]]第一三九师第四一七团（人称“夜老虎团”）团长，在[[两山轮战|两山轮战]]时亲自指挥了“黑豹”行动，其中以1987年1月7日的“1.7”战斗最为著名。当时第四一七团在老山前线发动了一次出击拔点167高地的作战，参战主力仅为第四一七团第四连，但作战十分勇猛。1987年1月8日晚，[[河内|河内]]广播电台报道称：“1月7日，中国军队向我边境地区发动了两个团规模的进攻。”1987年1月9日，[[美国之音|美国之音]]报道称：“1月7日，在中越边境爆发了自1979年以来规模最大的一次战斗。”<ref>{{Cite web |url = http://news.ifeng.com/a/20160805/49725925_0.shtml |title =首次公开：刘亚洲兄弟五人，两个将军一个师长  |author =  |date =2016-08-05  |publisher =  凤凰资讯|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.sohu.com/a/127395553_600466|title =情系老山的“红二代”上将刘亚洲向社会捐款数百万元，这些钱是从哪儿来的？  |author =  |date = 2017-02-27 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>后来，刘亚苏进入[[中国人民解放军总参谋部|中国人民解放军总参谋部]]工作，并晋升为少将军衔，2015年时正在担任[[中国人民解放军总参谋部情报部|中国人民解放军总参谋部情报部]]副军职专职情报干部<ref>{{Cite web|url =http://www.gulang.gov.cn/Html/glnews/201920E1K5E6B3C6KCBGB30JI5.html|title =我县隆重举行纪念红军西路军古浪战役79周年暨红军西路军古浪战役纪念馆开馆仪式|author =|date =2015-11-28|publisher =甘肃古浪|language =|accessdate =|quote =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 家庭 ==
*父：刘建德，曾参加[[抗美援朝战争|抗美援朝战争]]<ref>[http://www.chinamil.com.cn/site1/jszjzpd/2009-03/13/content_1688353.htm 刘亚洲：丧父日记] {{webarchive|url=https://web.archive.org/web/20130527103546/http://www.chinamil.com.cn/site1/jszjzpd/2009-03/13/content_1688353.htm |date=2013-05-27 }}</ref>。
*母：陈于湘，[[浙江省|浙江省]][[温州市|温州市]][[苍南县|苍南县]]宜山农村人<ref name=shouci>{{cite news|title=首次公开：刘亚洲兄弟五人 两个将军一个师长|url=http://news.sohu.com/20160805/n462806744.shtml|date=2016-08-05|newspaper=长安街知事}}</ref>。
