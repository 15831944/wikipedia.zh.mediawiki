'''刘虎''' {{BD|1975年|9月7日|||catIdx=刘}}，[[重庆|重庆]][[渝北区|渝北区]]人，中国大陆记者，曾就职于《[[新快报|新快报]]》。

==生平==
2013年年中，刘虎实名举报[[中华人民共和国国家工商行政管理总局|中华人民共和国国家工商行政管理总局]]副局长[[马正其|马正其]]等贪腐官员，指出“马正其在重庆期间在处理[[国企|国企]]改制事宜上涉嫌严重渎职，致数千万元国资被侵占”，8月23日，刘虎被以“[[谣言|造谣]]”罪名被[[北京市|北京市]]警察在[[重庆市|重庆市]][[渝北区|渝北区]]的家里[[刑事拘留|刑事拘留]]，警察带走他的[[笔记本电脑|笔记本电脑]]和[[电脑|台式电脑]]，批捕罪名是“[[诽谤|诽谤]]”罪。之后，刘虎被羁押在北京市一级看守所。<ref>{{cite web |url=http://news.qq.com/a/20131010/014828.htm |title=记者刘虎涉诽谤罪被批捕 此前因涉传谣被刑拘 |accessdate= 2013|publisher=腾讯网 |work= |language=zh-hans}}</ref><ref>{{cite web |url= http://news.163.com/13/0823/14/96VJ15K300011229.html|title= 《新快报》记者刘虎在重庆家中被北京警方带走|accessdate=2013 |publisher=网易 |work= |language=zh-hans}}</ref>刘虎的辩护律师称刘虎不构成犯罪。<ref>{{cite web|url=http://www.morningpost.com.cn/szb/html/2013-10/11/content_250465.htm|title=新快报记者刘虎被批捕|accessdate=2013|publisher=北京晨报|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20131015092630/http://www.morningpost.com.cn/szb/html/2013-10/11/content_250465.htm|archivedate=2013-10-15}}</ref>记者去重庆市采访时，原万州区蔬菜饮食服务公司工会主席赖东对记者说，“曾有百余名职工举报国有资产流失”：“这么多年来，举报一直没间断，但一直没结果。”赖东说，2008年，职工们掌握了马正其签字的文件，据此认为他有渎职行为，但举报无果。<ref>{{cite web |url=http://news.sina.com.cn/m/2013-07-30/111727809143.shtml |title=记者刘虎微博举报工商总局副局长马正其渎职 |accessdate=2013 |publisher=新浪网 |work= |language=zh-hans}}</ref>

2014年8月3日，被以取保候审名义释放。<ref>{{cite web |url=http://www.bbc.co.uk/zhongwen/simp/china/2014/08/140804_china_reporter_liuhu_release.shtml |title=中国记者刘虎涉诽谤羁押一年取保候审 |accessdate=2014 |publisher=BBC |work= |language=zh-hans}}</ref>
