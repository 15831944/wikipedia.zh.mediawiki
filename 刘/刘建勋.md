'''刘建勋'''（{{bd|1913年||1983年|4月23日|catIdx=}}），[[河北|河北]][[沧县|沧县]]人，中华人民共和国政治人物。

==生平==
刘建勋早年参加地下运动，[[抗日战争|抗日战争]]期间，担任[[中共晋中特委|中共晋中特委]]副书记、[[武乡县|武乡县]]委书记等职。[[第二次国共内战|第二次国共内战]]期间，担任中共太行第三地委书记兼军分区政治委员<ref>{{cite book |author=[[宋任穷|宋任穷]]|title=《宋任穷回忆录 》|year=1994年|location=解放军出版社 |page = 244 }}</ref>、中共晋冀鲁豫中央局组织部干部科科长。中华人民共和国成立后，担任[[中共湖北省委|中共湖北省委]]副书记、第二书记，[[中共中央中南局|中共中央中南局]]常委、秘书长；[[中共中央书记处|中共中央书记处]]第二办公室副主任。

1957年，担任中共[[广西省|广西省]]委<ref>{{cite book |author=中国社会科学院新闻研究所 |title=《中国新闻年鉴》|year=1999年|location=中国社会科学出版社 |page = 462}}</ref>、[[广西壮族自治区党委|广西壮族自治区党委]]第一书记兼[[广西军区|广西军区]]政委<ref>{{cite book|author=广西壮族自治区地方志编纂委员会 |title=《广西通志: 公安志》|year=2002年 |location=广西人民出版社|pages=436页}}</ref>。

1961年[[信阳事件|信阳事件]]后，临危受命于7月22日正式担任[[中共河南省委|中共河南省委]]第一书记兼[[河南省军区|河南省军区]]第一政委，着手解决河南饥荒<ref>{{cite web |title=刘建勋信阳事件后挂帅救灾：唯有吃饭高 |url = http://news.qq.com/a/20100728/001417.htm|publisher=中国新闻周刊 |accessdate=2011-12-24}}</ref>。刘建勋晚年在《自述》中回忆：“我到河南后，先抓救灾，保证中央给的钱、粮食确实让群众收到。我从新疆[[张仲翰|张仲翰]]那儿弄土豆，他是我的河北老乡，是[[献县|献县]]人，和纪晓岚是一个村子的，当时是[[新疆建设兵团|新疆建设兵团]]副司令员……”刘建勋要求全省铁路的大大小小火车站，对无钱购票的逃荒灾民，一律不得阻拦或扣留，统统给予放行。

1966年，担任[[中共中央华北局|中共中央华北局]]书记处书记<ref>{{cite book |authors=邵文杰, 河南省地方史志编纂委员会 |title=《河南省志: 共产党志》（卷17）|year=1997年 |location=河南人民出版社 |page =147}}</ref>。1968年1月至1978年10月，任[[河南省革命委员会|河南省革命委员会]]主任<ref>{{cite book |author=《中国共产党编年史》编委会 |title=《中国共产党编年史: 1966-1977》|year=2002年 |publisher=山西人民出版社 |ISBN = 9787203038818|pages=2475页}}</ref><ref>{{cite web |title=刘建勋：河南省委第一书记 |url= http://www.czmingren.net/Article/ShowArticle.asp?ArticleID=1505 |publisher=沧州名人网 |accessdate=2011-12-24}}</ref>；1971年3月至1978年10月任中共河南省委第一书记。1978年10月，中共中央政治局决定调整河南省委领导班子。中央领导同志的指示是：“刘建勋同志在河南十七年，做了大量工作，有很大成绩。他是个很老的同志，不是‘四人帮’的人。但是，建勋同志在河南工作期间是有错误的……（中央）决定将刘建勋同志调离河南，治好病后另行分配工作。”<ref>[刘立强：《红星照耀的家庭：共和国开创者家事追忆》 ]</ref>

刘建勋身患晚期[[骨癌|骨癌]]和严重[[肺心病|肺心病]]，1983年病逝。刘建勋的骨灰覆盖着[[中国共产党党旗|中国共产党党旗]]，被安放在了[[八宝山革命公墓|八宝山革命公墓]]副一室。2006年6月24日，经组织批准，同时也是根据刘建勋夫妇的遗愿，二人的部分骨灰合葬于[[太行革命根据地|太行革命根据地]]——河北省[[涉县|涉县]]的[[将军岭|将军岭]]上。
