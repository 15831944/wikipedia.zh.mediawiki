'''刘建'''（{{bd|？|||}}）籍贯不详，[[中国人民解放军中将|中国人民解放军中将]]。

==生平==
曾任[[北京军区|北京军区]][[中国人民解放军第27集团军|陆军第二十七集团军]]政治部主任。2010年，任[[中国人民解放军第65集团军|陆军第六十五集团军]]政委。2015年7月，任[[沈阳军区空军|沈阳军区空军]]政治委员<ref>{{cite web|accessdate=2016-10-04|url=http://china.caixin.com/2015-09-13/100849315.html|title=65集团军政委刘建升任沈阳军区空军政委|publisher=财新网}}</ref>。2016年，任[[北部战区|北部战区]]副政委兼政治工作部主任<ref name = "Liujian">{{cite news |title = 北部战区副政委兼政工部主任刘建：决不允许搞非组织行为 |url = http://www.thepaper.cn/newsDetail_forward_1540598 |accessdate = 2016-10-09 |publisher = 澎湃新闻}}</ref>。2018年6月，调任[[北京军区|北京军区]]善后办政委<ref>{{cite web |title=北部战区副政委兼政工部主任刘建转任北京军区善后办政委 |url=https://www.thepaper.cn/newsDetail_forward_2187898 |website=澎湃新闻 |accessdate=2018-06-25}}</ref>。

2016年8月，晋升[[中国人民解放军中将|中将]]军衔<ref>{{cite web|accessdate=2016-10-04|url=http://www.thepaper.cn/newsDetail_forward_1538539|title=解放军新晋27名中将，七成以上在军事指挥岗位上担任过主官|publisher=澎湃新闻}}</ref>。
