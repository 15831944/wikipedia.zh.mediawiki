'''刘白羽'''（{{bd|1916年|9月2日|2005年|8月24日|catIdx=Liu}}），原名'''刘玉赞'''，生于[[北京|北京]]<ref>{{cite web |url=http://china.cnr.cn/news/200511/t20051111_504126496.shtml |title=刘白羽同志追思会昨天举行 |accessdate=2005 |publisher=中国广播网 |work= |language=zh-hans}}</ref>[[中国|中国]]现代著名的[[文学家|文学家]]与文学政治家。2005年逝世于[[北京|北京]]。

== 获奖作品 ==
*1950年 影片《中国人民的胜利》： [[斯大林文艺奖|斯大林文艺奖]]
*1989年《芳草集》： [[优秀散文奖|优秀散文奖]]
*1991年《第二个太阳》：[[茅盾文学奖|茅盾文学奖]]，第三届
*1990年至1994年 《心灵的历程》：中国优秀传记文学作品奖
