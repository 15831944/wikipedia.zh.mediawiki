'''刘泽金'''（{{bd|1963年|10月||}}）[[山东省|山东省]][[无棣县|无棣县]]人，[[中国人民解放军少将|中国人民解放军少将]]，激光技术专家<ref name=pp/><ref name=xiaoyou>{{Cite web |url = http://xy.nudt.edu.cn/contents/80/543.html |title =刘泽金  |author =  |date =2015-03-02  |publisher =  哈军工 · 国防科大校友网|language =  |accessdate =  |quote =  }}</ref>。

==生平==
1983年7月入伍，1987年4月加入[[中国共产党|中国共产党]]。1986年9月进入[[中国人民解放军国防科学技术大学|中国人民解放军国防科学技术大学]]应用物理系光学工程专业硕士学习。1993年9月进入中国人民解放军国防科学技术大学应用物理系军用光学专业博士学习<ref name=xiaoyou/>。刘泽金的研究方向是高能激光技术领域，长期在中国人民解放军国防科学技术大学任教，历任该校光电科学与工程学院院长、科研部部长等职务。2017年升任该校副校长<ref name=pp/>。

刘泽金还是[[国务院学位委员会|国务院学位委员会]]学科评议组成员、[[中华人民共和国教育部|中华人民共和国教育部]]创新团队带头人、军队领军科技人才培养对象。2010年，因在科学技术研究中取得显著成绩，获[[中央军事委员会|中央军委]]记一等功。2017年4月，[[中国工程院|中国工程院]]网站公布《中国工程院2017年院士增选有效候选人名单》，刘泽金进入中国工程院2017年院士增选候选人名单，属于信息与电子工程学部<ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1734939 |title = 国防科技大学科研部部长刘泽金教授已升任副校长 |author =  |date =2017-07-18  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url =http://www.cae.cn/cae/html/main/col66/2017-04/21/20170421173226284953483_1.html|title =中国工程院2017年院士增选有效候选人名单|author =|date =2017-04-21|publisher =中国工程院|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170722101930/http://www.cae.cn/cae/html/main/col66/2017-04/21/20170421173226284953483_1.html|archivedate =2017-07-22}}</ref>。
