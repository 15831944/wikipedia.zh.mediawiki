'''刘鼎'''（{{bd|1902年|1月8日|1986年|7月25日}}），原名'''阚思俊'''，字'''尊民'''，曾用名'''阚泽民'''、'''甘作明'''、'''戴良'''等，[[四川|四川]][[南溪区|南溪]]人，[[中华人民共和国|中华人民共和国]]政治人物，军工与机械工业专家，[[中北大学|中北大学]]首任校长。

== 生平 ==
=== 第一次国共内战时期 ===
刘鼎出生于一个小知识分子的家庭，自幼以图画、手工、算术、理工知识为爱好。中学时代，就和同学[[郭伯和|郭伯和]]、[[李硕勋|李硕勋]]、[[阳翰笙|阳翰笙]]等带头组织学生会，开展进步活动。1920年，考入[[浙江省立高等工业学校|浙江省立高等工业学校]]（今属[[浙江大学|浙江大学]]）电机科学习<ref>{{cite book |author1=吴殿尧 |title=刘鼎传 |date=2012 |publisher=中央文献出版社 |location=北京 |isbn=978-7-5073-3621-4 |page=16}}</ref>。1923年，他进入[[上海大学|上海大学]]学习，加入[[中国社会主义青年团|中国社会主义青年团]]。1924年，赴德国勤工俭学，在德国[[哥廷根大学|哥廷根大学]]和[[柏林洪堡大学|柏林大学]]学习。期间，经[[朱德|朱德]]、[[孙炳文|孙炳文]]介绍，加入中国共产党。1926年，转赴苏联[[莫斯科东方大学|莫斯科东方大学]]和空军机械学校学习，并兼任教官<ref>{{cite book |author1=蔡寅生 |title=光辉岁月 共和国陆军装备研发史话 |date=2015 |publisher=中国财政经济出版社 |location=北京 |isbn=978-7-5095-6198-0 |page=261}}</ref>。

1929年底，刘鼎从莫斯科回到上海，到[[中央特科|中央特科]]工作，在由[[陈赓|陈赓]]领导的中央特科第二科任副科长。在中央特科工作时期，刘鼎负责联系[[杨登瀛|杨登瀛]]，进行情报搜集工作，还参与营救被捕的[[关向应|关向应]]<ref>{{cite book |author1=傅涯主编 |title=陈赓大将图传 |date=2013 |publisher=解放军出版社 |location=北京 |isbn=978-7-5065-6536-3 |page=51}}</ref>。1931年4月，特科负责人[[顾顺章|顾顺章]]投降国民政府后，中共中央特科进行改组，刘鼎仍留上海从事情报工作。10月10日，在一次接头中被逮捕入狱。因未暴露身份，在妻子[[吴先清|吴先清]]的多方周旋下，1932年9月，刘鼎经[[康泽|康泽]]等出面作保获释出狱<ref>{{cite web |author1=王旭烽 |title=神秘的红色女间谍——吴先清·紫罗兰 |url=http://view.news.qq.com/a/20131001/003977.htm |website=腾讯 |accessdate=2018-08-04 |date=2013-09-29 |archive-url=https://web.archive.org/web/20131012140656/http://view.news.qq.com/a/20131001/003977.htm |archive-date=2013-10-12 |dead-url=no }}</ref>。

1933年，刘鼎离开上海，前往[[中央苏区|中央苏区]]。途中，因国民政府封锁，被留在赣东北苏区。其后，担任闽浙赣军区组织部长、红军第五分校政委<ref>{{cite book |author1=苏士甲 |title=闪亮的红星：中国工农红军院校及其办校人 |date=2007 |publisher=新华出版社 |location=北京 |isbn=978-7-5011-8054-7 |page=36}}</ref>。后应[[方志敏|方志敏]]要求，出任[[洋源|洋源]]兵工厂政委。他亲自动手研制并组织生产了红军第一门炮及炮弹，开创了中国工农红军早期兵工事业。1935年1月，方志敏率领的[[红十军团|红十军团]]在[[怀玉山战役|怀玉山战役]]中失败；6月，刘鼎也在国民革命军搜山时被俘。同年秋，从俘虏营中脱逃，前往上海。1936年，到西安任中共驻[[东北军|东北军]]代表，对[[张学良|张学良]]开展统战工作。12月12日，[[西安事变|西安事变]]爆发。此前张学良担心发电会泄密，所以在行动开始后才通报。12日凌晨5时，刘鼎第一时间向中共中央报告事变情况，为中共应对事变提供了依据<ref>{{cite book|title=風雨四十年 第一部|author=童小鵬|location=北京|publisher=中央文獻出版社|year=1994|isbn=7-5073-0229-6|page=60}}</ref>。毛泽东曾多次指出：“西安事变，刘鼎同志是有功的”<ref>{{cite web |author1=刘文山 |title=毛泽东：西安事变，刘鼎是有功的 |url=http://www.crt.com.cn/news2007/News/jryw/151215144635D4C52H9G71I5FGB10J72.html |website=中国红色旅游网 |accessdate=2018-08-04 |date=2015-12-15 |archive-url=https://web.archive.org/web/20160503204016/http://crt.com.cn/news2007/News/jryw/151215144635D4C52H9G71I5FGB10J72.html |archive-date=2016-05-03 |dead-url=no }}</ref>。

=== 抗日战争与第二次国共内战时期 ===
1937年3月，刘鼎回到延安，任[[抗日军政大学|抗日军政大学]]特科大队队长，负责培训[[八路军|八路军]]的通讯技术人才。1940年5月，任八路军总部军工部部长，开创了[[八一式步马枪|八一式步马枪]]、单响短枪、[[五零掷弹筒|五零掷弹筒]]、炮弹、枪弹发射药和烈性炸药的批量生产，保证了部队作战的需要<ref>{{cite book |author1=中国抗日战争军事史料丛书编审委员会 |title=八路军 回忆史料 4 |date=2015 |publisher=解放军出版社 |location=北京 |isbn=978-7-5065-7124-1 |page=113-131}}</ref><ref>{{cite web |author1=刘文山 |author2=唐雷 |title=刘鼎与我军第一支制式化步枪的诞生 |url=http://www.xinhuanet.com/mil/2017-03/12/c_129507735.htm |website=新华网 |accessdate=2018-08-04 |date=2017-03-12 |archive-url=https://web.archive.org/web/20180804142929/http://www.xinhuanet.com/mil/2017-03/12/c_129507735.htm |archive-date=2018-08-04 |dead-url=no }}</ref>。1941年5月，成立[[太行工业学校|太行工业学校]]，刘鼎担任第一任校长，该校即是[[中北大学|中北大学]]的前身。1944年1月奉调回延安，任[[陕甘宁晋绥联防军|陕甘宁晋绥联防军]]军工局副局长。[[第二次国共内战|第二次国共内战]]期间，先后担任晋察冀军工局副局长、[[华北人民政府|华北人民政府]]公营企业部副部长等职务，继续领导军工生产工作<ref>{{cite book |author1=四川百科全书编纂委员会 |title=四川百科全书 |date=1997 |publisher=四川辞书出版社 |location=成都 |isbn=7-80543-629-0 |page=557}}</ref>。

=== 中华人民共和国成立后 ===
[[中华人民共和国|中华人民共和国]]成立后，刘鼎担任[[中央人民政府重工业部|中央人民政府重工业部]]副部长，组织兵工生产支援[[朝鲜战争|朝鲜战争]]。1954年9月，担任[[第一机械工业部|第一机械工业部]]副部长。1957年，又担任[[第二机械工业部|第二机械工业部]]副部长。1960年9月，再担任新组建的[[第三机械工业部|第三机械工业部]]副部长，后任常务副部长兼航空研究院院长。在此期间，他组织推广了电解加工、机夹刀具、组合夹具、金刚石工具、喷丸强化、滚压加工、高能成型等一大批新技术和新工艺项目<ref>{{cite web |author1=白薇 |title=追忆刘鼎同志：兵工泰斗 统战功臣 |url=http://scitech.people.com.cn/GB/25509/56813/228408/228410/15405449.html |website=人民网 |accessdate=2018-08-04 |date=2011-08-12 |archive-url=https://web.archive.org/web/20120911172902/http://scitech.people.com.cn/GB/25509/56813/228408/228410/15405449.html |archive-date=2012-09-11 |dead-url=no }}</ref>。[[文化大革命|文化大革命]]中，受到迫害，被关押七年之久。后担任航空工业部正部级顾问等职务<ref>{{cite book |author1=中国航空工业史编修办公室编 |title=中国航空工业人物传 领导篇 1 |date=2011 |publisher=航空工业出版社 |location=北京 |isbn=978-7-80243-723-4 |page=24}}</ref>。

刘鼎是第一届全国政协委员，第五、第六届全国政协常委，[[中国机械工程学会|中国机械工程学会]]理事长、名誉理事长和[[中国兵工学会|中国兵工学会]]顾问。1986年7月25日，在北京逝世<ref>{{cite book |author1=胡华主编 |author2=中共党史人物研究会编 |title=中共党史人物传 第43卷 |date=1990 |publisher=陕西人民出版社 |location=西安 |isbn=7-224-01026-X |page=325}}</ref>。
