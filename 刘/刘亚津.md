'''刘亚津'''{{BD|1959年|1月2日|||catIdx= L刘}}出生于[[天津市|天津市]]，[[天津市曲艺团|天津市曲艺团]]和[[解放军空政文工团|解放军空政文工团]]出身，为著名的小品演员、电影演员和电视演员。<ref name="duck " >{{cite web |url=http://eladies.sina.com.cn/qg/2010/0830/10521013957.shtml |title=刘亚津老婆：要不是娶了我 你能上春晚 |accessdate=2010年8月30日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

== 简介 ==
1959年，刘亚津出生于[[天津市|天津市]]。<ref name="duck " />
1978年，刘亚津到[[天津曲艺团|天津曲艺团]]学习相声，4年后学习[[京韵大鼓|京韵大鼓]]。<ref name="duck " />
1995年，刘亚津调入[[解放军空政文工团|解放军空政文工团]]。<ref name="duck " />

== 个人生活 ==
1986年，刘亚津和妻子[[张国丰|张国丰]]结婚。<ref name="duck " />
刘亚津的妻子张国丰是天津曲艺团京韵大鼓演员，两人居住于天津郊区的[[万新村|万新村]]。<ref name="duck " />
1988年6月，两人的女儿刘畅诞生。<ref name="duck " />
1990年，全家搬迁到[[北京市|北京市]]。<ref name="duck " />
