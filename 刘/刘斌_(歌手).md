'''刘斌'''（{{bd|1958年|3月||}}）祖籍[[山东|山东]]，生于[[辽宁|辽宁]][[锦州|锦州]]。[[中华人民共和国|中华人民共和国]]男高音歌唱家、作曲家，国家一级演员<ref name=wyyyy>{{Cite web |url = http://music.163.com/#/artist/desc?id=3937 |title =刘斌  |author =  |date =  |publisher = 网易云音乐 |language =  |accessdate =2017-08-27  |quote =  }}</ref>。

==生平==
刘斌早年是[[京剧|京剧]][[老生|老生]]演员。1974年到1984年，刘斌在长春市京剧团任[[老生|老生]]演员。其间曾师从[[余叔岩|余派]]传人[[万小浦|万小浦]]学习余派老生，代表剧目有《[[失空斩|失、空、斩]]》饰[[孔明|孔明]]，《碰碑》饰[[杨继业|杨继业]]等；后又师从[[马连良|马派]]传人[[丁英奇|丁英奇]]学习马派老生，代表剧目有《[[借东风|借东风]]》饰孔明，《[[四郎探母|四郎探母]]》饰[[杨延辉_(杨家将)|杨延辉]]等。自出演新编历史剧《凤凰二乔》饰[[乔玄|乔玄]]后，刘斌便正式拜高派创始人[[高庆奎|高庆奎]]之子[[高世寿|高世寿]]为师，学习高派老生，代表剧目有《[[逍遥津|逍遥津]]》饰[[汉献帝|汉献帝]]，《斩皇袍》饰[[赵匡胤|赵匡胤]]等。在十年的京剧表演生涯中，刘斌曾多次获省、市文艺奖项<ref name=wyyyy/>。

1984年，刘斌来到[[北京|北京]]，考入[[中国人民解放军陆军政治工作部文工团|北京军区战友歌舞团]]。先后师从[[中国音乐学院|中国音乐学院]][[金铁霖|金铁霖]]、[[上海音乐学院|上海音乐学院]][[高芝兰|高芝兰]]、[[中央音乐学院|中央音乐学院]][[王秉锐|王秉锐]]教授。1985年，创唱《南腔北调大汇唱》，以一曲“穷人的孩子早当家”（源自京剧《[[红灯记|红灯记]]》）及“今日痛饮庆功酒”（源自京剧《[[智取威虎山|智取威虎山]]》）走红。1986年，在[[1986年中国中央电视台春节联欢晚会|1986年中国中央电视台春节联欢晚会]]上演唱<ref name=wyyyy/>。

1986年，刘斌参加中央电视台举办的第二届全国“五洲杯”声乐大奖赛民族唱法专业组比赛，获银屏奖。1988年参加第三届全国“五洲杯”声东大奖赛民族唱法专业组比赛，获优秀奖。1990年参加第四届全国“五洲杯”声乐大奖赛民族唱法专业组比赛，获第二名。1992年参加第五届全国“五洲杯”声乐大奖赛民族唱法专业组比赛，获第一名。1992年8月获全军文艺汇演一等奖。1992年11月到[[香港|香港]]参加国际青年艺术节，获金奖。1992年，在[[日本|日本]][[明仁天皇|明仁天皇]]及夫人访华时，主演大型歌剧《歌仙——小野小町》饰皇太子。1984年到1995年，多次参加中央电视台举办的春节联欢晚会及大型文艺晚会，还有各省、市电视台举办的文艺晚会及演出活动<ref name=wyyyy/>。

1994年11月，刘斌应邀到[[日本|日本]]参加国际青年艺术节，在[[东京|东京]]、[[福冈|福冈]]演出五场音乐会，并参加世界上第一次由5万人合唱、12万人参与的[[贝多芬第九交响曲|贝多芬第九交响曲]]《[[欢乐颂|欢乐颂]]》，任领唱之一<ref name=wyyyy/>。

2006年11月28日，刘斌出任[[中国人民解放军陆军政治工作部文工团|北京军区战友文工团]]团长<ref>{{Cite web |url =http://chzt.sbsm.gov.cn/article/qt/spryqyqyys/jlfy/201111/20111100094703.shtml  |title =  战友文工团团长刘斌在演唱间隙的即席发言|author =  |date =2011-11-29  |publisher = 国家测绘地理信息局 |language =  |accessdate =  |quote =  }}</ref>。任内他除了领导全团工作外，还坚持演出。全团演出活动频繁，先后推出《红歌献给党·长征组歌交响合唱音乐会》等<ref>{{Cite web |url = http://news.163.com/11/0612/12/76BNN1P000014AED.html |title =  刘斌领衔红歌会《长征组歌》第二代指挥家蒋燮斌重回接棒地|author =  |date = 2011-06-12 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。他还推动了北京军区战友文工团管弦乐团的改革发展<ref>{{Cite web |url =http://www.chinadaily.com.cn/hqjs/zggf/2012-04-05/content_5615065.html  |title = 高清组图：战友乐团演出“战友之声”公益交响音乐会 |author =  |date =2012-04-05  |publisher =中国日报  |language =  |accessdate =  |quote =  }}</ref>。2014年卸任团长职务。

==作品==
刘斌首唱的歌曲有100多首，其中《当兵的人》、《依旧辉煌》、《华北大平原》、《不老的爱》、《这话听着亲》、《爱是同渡的船》及电视连续剧《[[唐明皇_(电视剧)|唐明皇]]》主题歌《倚天拨剑观苍海》等十多部电视剧主题歌是他的代表作<ref name=wyyyy/>。

刘斌还是一位作曲家。1995年8月1日，歌曲《[[当兵的人|当兵的人]]》（晓岭作词，云飞、刘斌作曲）获[[中国人民解放军总政治部|中国人民解放军总政治部]]颁发的第四届文艺大奖、90年代战士最喜爱的军旅歌曲特别特别奖、1995年全军优秀队列歌曲奖。1996年7月，歌曲《当兵的人》在中央电视台举办的全国第一届“军神杯”军旅歌曲大赛专业组比赛中获作品金奖。1996年9月，《中华儿女的心愿》获第二届“花生曲”歌曲创作征集特等大奖。1997年，创作歌曲《当兵的人》获90年代观众最喜爱的电视歌曲作曲奖<ref name=wyyyy/>。
