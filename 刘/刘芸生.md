'''刘芸生'''（{{bd|1910年|6月|2016年|8月10日}}）女，生于[[天津|天津]]，[[傅作义|傅作义]]夫人，[[全国人大代表|全国人大代表]]<ref name=huainian>{{Cite web |url = http://lgbzc.mwr.gov.cn/czgc/201609/t20160901_761757.html  |title = 默默耕耘持夫业 勤劳节俭度平生——怀念刘芸生老人 |author = 吴頔、李攀 |date = 2016-09-01 |publisher =  水利部离退休干部局|language =  |accessdate =  |quote =  }}</ref>。

==生平==
刘芸生的父亲叫刘竹岩，是[[北京|北京]]人，母亲是[[安徽|安徽]][[桐城|桐城]]人，当时他们住在[[天津法租界|天津法租界]]一座小楼。后来[[海河|海河]]发生洪灾，天津大部分地方被淹，他们全家才迁居北京<ref name=shiji>{{Cite web |url = http://lgbzc.mwr.gov.cn/czgc/201609/t20160901_761759.html |title = 世纪老人刘芸生的博大情怀 |author = 凌先有 |date = 2016-09-01 |publisher =  水利部离退休干部局 |language =  |accessdate =  |quote =  }}</ref>。1929年，刘芸生与傅作义结婚，当时傅作义是天津警备司令。婚后不久，傅作义任[[绥远省政府|绥远省政府]]主席，夫妇一同来到[[绥远省|绥远省]]<ref name=huainian/><ref name=shiji/>。

1931年[[九一八事变|九一八事变]]爆发。为了储备抗日力量，傅作义学习[[中国共产党|中国共产党]]的经验，开办乡训所，以加强军事训练。刘芸生除料理家务外，也帮乡训所开展图书阅览活动，并举办文化娱乐活动<ref name=huainian/>。

1933年，[[日军|日军]]占领[[热河省|热河省]]，分路进攻[[长城|长城]]各口，傅作义提出“宁做战死鬼，不做亡国奴”，率[[国民革命军|国民革命军]]第五十九军到[[怀柔|怀柔]]，负责独石口防御战。在[[长城抗战|长城抗战]]“最后有力的光荣一战”中，傅作义率部同友军成功阻击了日军十多次进攻，刘芸生则率女子战地慰问救护团到前线慰问和救护<ref name=huainian/>。

1936年，日军进攻绥远时，傅作义在绥远东部的[[红格尔图镇|红格尔图]][[百灵庙|百灵庙]]阻击。为了宣传[[绥远抗战|绥远抗战]]，刘芸生在红格尔图的玫瑰营举办舞会，招待中外记者。[[抗日战争|抗日战争]]全面爆发后，傅作义先后参加[[忻口会战|忻口会战]]、[[包头会战|包头会战]]、[[绥西战役|绥西战役]]、[[五原战役|五原战役]]，刘芸生率女子战地救护团向战士宣传战争动员法令、散发《动员报》、教唱抗战歌曲《[[大刀进行曲|大刀进行曲]]》、《[[义勇军进行曲|义勇军进行曲]]》和《[[黄河大合唱|黄河大合唱]]》等。她还与女子战地慰问救护团救护伤员<ref name=huainian/><ref name=shiji/>。

抗日战争胜利后，不久即爆发[[第二次国共内战|第二次国共内战]]。傅作义担任[[华北剿总|华北剿总]]司令后期，[[蒋介石|蒋介石]]希望傅作义率部队南撤，傅作义以固守华北的名义拒绝执行，这时刘芸生带着几个子女乘飞机到[[重庆|重庆]]，住在[[张伯苓|张伯苓]]家，以消除蒋介石的戒心。在重庆生活一段时间后，1949年初[[北平和平解放|北平和平解放]]协议签字后，傅作义请张伯苓送她和子女们来[[北平|北平]]。刘芸生携子女登上回北平的飞机，但没起飞就被国民党当局扣留，随后被软禁在重庆一个小旅馆中一个月，刘芸生一直鼓励子女要坚持。后经中共地下党组织帮助，刘芸生和子女才得以飞往[[香港|香港]]，从海上辗转抵达天津，与傅作义团聚<ref name=huainian/>。

[[中华人民共和国|中华人民共和国]]成立后，刘芸生参加了由原国民党知名人士的夫人组成的夫人支部，在[[宋庆龄|宋庆龄]]、[[何香凝|何香凝]]的指导下学习中央文件及时事政治。在[[抗美援朝运动|抗美援朝运动]]中，她和夫人支部的成员们赶制绢花义卖，捐钱捐物，编手套并写慰问信送给前方的[[中国人民志愿军|中国人民志愿军]]战士。她还协助人民政府开展[[中华人民共和国扫盲教育|扫盲运动]]、辅导夜校、帮[[街道办事处|街道]]工作。她还经常提建议，涉及争取妇女权利、婚姻自由，取缔娼妓并帮助娼妓改造和谋出路，收容流浪儿童，改造青少年罪犯等方面，受到[[全国妇联|全国妇联]]采纳及表扬<ref name=huainian/>。

1965年7月下旬，原[[中华民国总统|中华民国代总统]][[李宗仁|李宗仁]]自海外回国，刘芸生于8月中旬在家中设宴招待李宗仁、[[郭德洁|郭德洁]]夫妇，并请周恩来、[[陈毅|陈毅]]、[[彭真|彭真]]作陪。席间，傅作义、刘芸生夫妇共同送给李宗仁夫妇一对火狐标本，以“狐死首丘”的寓意，感佩李宗仁夫妇冒生命危险回国。李宗仁夫妇很感动，表示要与傅作义夫妇共同为祖国统一而努力<ref name=shiji/>。

1966年，傅作义同刘芸生商量，想将家中存款和部分财产上交国家，以支援国家建设，刘芸生积极支持傅作义的做法，并要子女积极配合。于是，傅作义数次致函[[周恩来|周恩来]]，最终这笔财产被收进国库<ref name=huainian/>。

1949年到1972年，傅作义担任水利部部长23年（初为[[中央人民政府水利部|中央人民政府水利部]]部长，后为[[中华人民共和国水利部|中华人民共和国水利部]]部长），每年花四分之一以上时间视察各地，走遍了[[长江|长江]]、[[黄河|黄河]]、[[黑龙江|黑龙江]]、[[珠江|珠江]]、[[淮河|淮河]]、[[海河|海河]]等处水利工程。傅作义每次出差，刘芸生都为他备好衣物。她还省吃俭用，自己买毛线编织了红色的帽子、围巾等，亲自拿到水库工地送给水利一线职工。她还曾到[[葛洲坝水利枢纽|葛洲坝水利枢纽]]参观<ref name=huainian/>。

1957年4月，傅作义到黄河[[三门峡水利枢纽|三门峡水利枢纽]]参加开工典礼突发[[心脏病|心脏病]]，[[周恩来|周恩来]]安排心脏病专家黄宛及时救治，使傅作义转危为安。刘芸生同机前往，照顾傅作义。1962年初，中央考虑到傅作义的心脏病较严重，安排傅作义一家到[[广东|广东]]休养。但傅作义表示，只有让他做些工作，他才去广东。中央和[[中共广东省委|中共广东省委]]只好同意他的要求。到广东后，傅作义先后视察了花县水库、新丰江水电站以及[[佛山|佛山]]、[[新会|新会]]、[[高要|高要]]等地的水利工程。每次刘芸生都为他提前备好[[硝酸甘油|硝酸甘油]]药片，然后傅作义才去视察<ref name=huainian/>。

1975年1月起，刘芸生历任第四、五、六、七届[[全国人大|全国人大]]代表<ref name=huainian/>。

刘芸生始终对子女的教育严格要求，她有六个子女考入名校，其中四人考入[[北京大学|北京大学]]，两人考入[[清华大学|清华大学]]。作为部长夫人，刘芸生拥有很高的待遇，但她在生活中保持勤俭。[[国家机关事务管理局|国家机关事务管理局]]每年都安排刘芸生到[[北京协和医院|北京协和医院]]体检，但她不愿麻烦别人，所以经常是几年才去一次。晚年，刘芸生的视力仍很好，每天都定时收看《[[新闻联播|新闻联播]]》。她出门总是自己坐[[公共汽车|公共汽车]]。100岁之前，生活几乎全部自理，甚至自己做家务<ref name=huainian/>。

在刘芸生生命的最后十多年里，水利部领导每年[[春节|春节]]前都去看望她。过了百岁后，改为每年看望两次，除春节慰问外，还为她生日祝寿<ref name=shiji/><ref>{{Cite web |url = http://news.qq.com/a/20160603/053698.htm |title = 水利部为傅作义夫人刘芸生祝贺106岁寿辰 |author =  |date = 2016-06-03 |publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref>。

2016年8月10日，刘芸生在北京病逝，享年106岁。2016年8月18日，遗体在[[北京市八宝山殡仪馆|北京市八宝山殡仪馆]]火化<ref name=huainian/>。
