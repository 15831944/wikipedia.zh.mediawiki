'''刘小枫'''（{{bd|1956年|catIdx=L劉}}），[[中国|中国]]当代[[学者|学者]]，[[中国人民大学|中国人民大学]]文学院[[教授|教授]]，[[博士生导师|博士生导师]]。[[中山大学|中山大学]]“逸仙”讲座教授，博士生导师。

== 生平 ==
1956年5月生于中国[[重庆市|重庆市]]。

1978年就读于[[四川外语学院|四川外语学院]]，获文学学士学位，专业日耳曼语言文学；1982年就读于[[北京大学|北京大学]]，获哲学硕士，美学方向；1985年在[[深圳大学|深圳大学]]中文系任教；1989年就读于瑞士[[巴塞尔大学|巴塞尔大学]]，获神学博士。

1993年起任[[香港中文大学|香港中文大学]]中国文化研究所研究员、[[北京大学|北京大学]]兼任教授、香港[[汉语基督教文化研究所|汉语基督教文化研究所]]学术总监、[[中山大学|中山大学]]哲学系教授，[[比较宗教|比较宗教]]研究所所长，香港中文大学中国文化研究所名誉研究员、[[复旦大学|复旦大学]]哲学系兼任教授、[[西南政法大学|西南政法大学]]名誉教授。

现为中国人民大学文学院教授，中国人民大学文学院古典文明研究中心主任。

== 著作 ==
刘小枫的主要学术领域为中西方古典思想、基督教思想史、德国近现代思想史、政治哲学；目前主要研究和教学方向为：古希腊哲学、先秦及两汉思想、古希腊语文学、德国近现代思想、宗教－政治哲学。

刘小枫被称为[[文化基督徒|文化基督徒]]的代表人物<ref>[http://www.methodfirst.cn/cl_in.html?cl_id=149 刘小枫:“文化”基督徒现象的社会学评注] {{webarchive|url=https://web.archive.org/web/20070927225703/http://www.methodfirst.cn/cl_in.html?cl_id=149 |date=2007-09-27 }}</ref>。

== 争议 ==
=== 学术 ===
近年，因为大量介绍[[美国|美国]][[保守主义|保守主义]]学者[[列奥·施特劳斯|列奥·施特劳斯]]的作品，令学术界有些学者以为，他将施特劳斯的研究方法作为解释古典著作的唯一方法，由此而引起一些争议。

=== 称毛泽东「國父」与文化大革命「人民民主」 ===
2013年5月15日，「鳳凰網讀書會」以「共和，中國的百年之累」為題，邀請中國政法大學教授[[王人博|王人博]]和[[中國人民大學|中國人民大學]]教授劉小楓坐而論道，評論近百年來中國爭取共和憲政的歷程。劉小楓提出：「憲政最大的難題就是如何評價[[毛澤東|毛澤東]]」，他明確表示：「誰是中國現代的[[國父|國父]]呢？從[[孫中山|孫中山]]、[[蔣介石|蔣介石]]、毛澤東中考慮，蔣介石已經被開出去了，那我們說是孫中山接著毛澤東，孫中山比起毛澤東差十萬八千里。毛澤東的功績大家都要承認。」<ref>[http://book.ifeng.com/dushuhui/special/salon133/ 讀書會NO.133：百年共和之累_讀書頻道_鳳凰網]</ref>，引發了中國知識界的許多爭議<ref>[http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20130518-%E8%91%97%E5%90%8D%E5%AD%A6%E8%80%85%E7%A7%B0%E6%AF%9B%E6%B3%BD%E4%B8%9C%E4%B8%BA%E2%80%9C%E5%9B%BD%E7%88%B6%E2%80%9D%E4%B8%BA%E6%96%87%E9%9D%A9%E6%8B%9B%E9%AD%82%E5%BC%95%E4%BA%89%E8%AE%AE 著名學者稱毛澤東為“國父”為文革招魂引爭議 - RFI]</ref>。

刘小枫还认为“[[文化大革命|文化大革命]]”是“[[人民民主專政|人民民主]]”，不赞成“十年动乱”的定论。<ref>{{cite web |url=http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20130520-%E5%8F%AF%E5%90%A6%E5%B0%8A%E6%AF%9B%E4%B8%BA%E2%80%9C%E5%9B%BD%E7%88%B6%E2%80%9D%EF%BC%9F |title=可否尊毛为“国父”？|accessdate= 2013年5月25日|publisher=法广中文网 |work= |language=zh-cn}}</ref>
