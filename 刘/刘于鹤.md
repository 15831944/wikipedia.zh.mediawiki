'''刘于鹤'''（{{bd|1937年|11月||}}）生于[[湖北|湖北]][[武汉|武汉]]，[[中华人民共和国|中华人民共和国]]林学家，[[中华人民共和国林业部|中华人民共和国林业部]]副部长，[[中国林业科学研究院|中国林业科学研究院]]院长<ref name=jy>{{Cite web |url = http://www.forestry.gov.cn/Zhuanti/content_201307qzlxjy/623611.html |title =原林业部副部长刘于鹤应邀来林科院进行传统教育  |author =  |date = 2013-08-22 |publisher =国家林业局政府网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1961年自[[苏联|苏联]][[列宁格勒|列宁格勒]]森林工程学院毕业。1961年到1982年，任[[中华人民共和国林业部|林业部]]规划设计局和[[中华人民共和国农林部|农林部]]林业局技术员、副处长、处长。1982年任林业部资源司副司长。1983年任[[西北林学院|西北林学院]]院长、党委副书记。1986年到1992年，任[[中国林业科学研究院|中国林业科学研究院]]院长、院分党组书记、[[中国林学会|中国林学会]]理事长。1992年起，任林业部科技司司长、林业部党组成员、副部长；[[中国花卉协会|中国花卉协会]]副会长。后曾任国务院稽查特派员，[[国有重点大型企业监事会|国有重点大型企业监事会]]主席。他是[[第九届全国政协委员|第九届全国政协委员]]、[[全国政协经济委员会|全国政协经济委员会]]委员，[[第十届全国政协委员|第十届全国政协委员]]、[[全国政协人口资源环境委员会|全国政协人口资源环境委员会]]委员，[[中共十三大|中共十三大]]、[[中共十四大|中共十四大]]代表。晚年任中国老科技工作者协会副会长、中国老科技工作者协会林业分会会长、中国老教授协会顾问<ref name=jy/><ref name=liuyh>{{Cite web |url = http://www.caf.ac.cn/news/yqjj/200909/2009-09-24-10-28.html |title = 刘于鹤(1986—1992年任院长、院分党组书记) |author =  |date = 2009-09-24 |publisher = 中国林业科学研究院 |language =  |accessdate =  |quote =  }}</ref>。

==学术贡献==
刘于鹤作为主要完成人完成了《森林资源连续清查技术体系的研建》项目，获1989年度国家科技进步二等奖、林业部科技进步一等奖。该技术体系的研建花费了25年，自1964年抽样技术引入中国森林调查工作起，经1960年代的试验和生产，“[[四五计划|四五]]”期间全国森林资源清查，“[[五五计划|五五]]”期间初步建立全国森林资源连续清查体系，到1988年“[[六五计划|六五]]”完成全国森林资源清查的复查。该体系的建成使中国森林资源连续清查工作步入世界先进行列<ref name=jy/>。
