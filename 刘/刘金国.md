'''刘金国'''（{{bd|1955年|4月||catIdx=L刘}}），[[河北|河北]][[昌黎|昌黎]]人；[[中共中央党校|中共中央党校]]领导干部函授本科班经济管理专业毕业。1975年加入[[中国共产党|中国共产党]]；是中共第十八届、十九届[[中央纪律检查委员会|中央纪律检查委员会]]委员、常委、副书记，第十九届中央委员会委员。曾任[[中华人民共和国公安部|公安部]]党委副书记、公安部常务副部长、纪委书记、督察长，[[中共河北省委|中共河北省委]]常委、省政法委书记等职。现任[[中国共产党第十九届中央纪律检查委员会|中共第十九届中央纪律检查委员会]]副书记。

== 早年经历 ==
刘金国出生在[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[昌黎县|昌黎县]]槐李庄乡小港村的一个普通农民家庭，在家里六个孩子中，排行老二。1974年，高中毕业之后，回村务农；曾并担任民兵连副连长。1975年加入中国共产党后，历任村党支部书记，公社党委副书记、书记。1983年9月，获推荐，进入[[中共河北省委|中共河北省委党校]]进修。两年学习结束后，回到[[昌黎县|昌黎县]]，任刘台庄工委副书记，并由此步入仕途。

== 仕途 ==
刘金国由基层党工干部做起，逐级升迁。曾历任中共昌黎县刘台庄工委副书记、书记，县委常委、组织部部长，县委副书记；中共[[秦皇岛市|秦皇岛市]][[山海关区|山海关区]]委员会副书记、区长；中共秦皇岛市委秘书长。

1992年5月，刘金国出任秦皇岛市公安局局长，开始在公安政法界任职。1994年11月，升任中共秦皇岛市委常委、副市长，兼公安局长；1995年8月，调任河北省公安厅副厅长；2002年1月，出任[[中共河北省委|中共河北省委]]政法委书记；当年12月，升任中共河北省委常委、兼政法委书记。在河北工作期间，刘金国获评全国公安系统一级英雄模范、河北省优秀人民公仆、优秀共产党员、廉政公仆等荣誉称号。

2005年3月，50岁的刘金国调中央工作，担任[[中华人民共和国公安部|中华人民共和国公安部]]副部长、党委委员，主管经济犯罪调查；并在2005年4月21日，被授予[[副总警监|副总警监]]警衔。没有政治背景和靠山的刘金国，在公安部受到[[周永康|周永康]]的排挤；2009年8月起，转而兼任公安部纪委书记、督察长<ref>[http://192.237.216.218/MIB/magazine/news.aspx?ID=M000000633 周永康排挤的刘金国晋升正部级]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}原载《明鏡月刊》49期</ref>。2008年[[汶川地震|汶川地震]]发生后，刘金国是公安部的前线总指挥，负责调集、指挥全国2万多名公安专业救援力量。[[2010年大连新港溢油事故|2010年大连新港溢油事故]]发生后，刘又奉命负责现场指挥救援工作，并因其突出表现被[[中华人民共和国国务院|国务院]]记个人[[一等功|一等功]]，并被[[中国中央电视台|中国中央电视台]]评为“2011年度[[感动中国|感动中国]]人物”<ref>{{cite news |title = 公安部副部长刘金国：感动中国 |url = http://news.qq.com/a/20120204/000219.htm |accessdate = 2012-06-10 |publisher = 腾讯网}}</ref>。2012年10月，在[[中共十八大|中共十八大]]上，当选[[中国共产党第十八届中央纪律检查委员会|中纪委]]委员。

在[[李东生|李东生]]因涉贪腐问题下台后，2014年2月，58岁的刘金国出任公安部常务副部长（正部长级）、党委副书记，兼[[中央防范和处理邪教问题领导小组|中央防范和处理邪教问题领导小组]]副组长、[[610办公室|办公室]]主任；还继续兼任公安部纪委书记、督察长。在当年10月召开的第十八届中纪委第四次全体会议上，被增选为中纪委常委、副书记。<ref>[http://news.ifeng.com/a/20141025/42296234_0.shtml 刘金国被选举为中纪委副书记]</ref>2017年10月，当选第十九届中共中央委员会委员。

== 事迹 ==
在刘金国调任公安部副部长之前一直在河北省任职。据媒体报道，拥有农转非审批权的刘金国，没有帮助过一个亲属；其老伴在退休前仍然是个临时工。记者问道，有人说刘金国是在装，刘金国回答说：“如果说我能装，那么我们共产党人都能装，装到死，不就是真的了么？”<ref>{{cite news |title = 公安部副部长刘金国冒死指挥大连火灾扑救 |url = http://news.sina.com.cn/c/2012-02-05/123223884185.shtml |accessdate = 2012-06-10 |newspaper = 新浪网}}</ref>
