'''刘国梁事件'''或称'''国乒兵变'''<ref name="a1">{{Cite web |url=https://www.hk01.com/體育/100317/-國乒兵變-中國乒協表示震驚痛心-退賽三將齊發道歉信|title=【國乒兵變】中國乒協表示震驚痛心　退賽三將齊發道歉信 |accessdate=2017-07-05 |author=張楊帆 |date=2017-06-25|work= |publisher=香港01网站|language=Zh-hant }}</ref>，由2017年6月间[[中国国家乒乓球队|中国国家乒乓球队]]总教练[[刘国梁|刘国梁]][[下課_(流行語)|下课]]所引发。23日，[[中国国家乒乓球队|中国国家乒乓球队]]男队队员、教练——[[许昕|许昕]]、[[马龙_(乒乓球运动员)|马龙]]、[[马琳|马琳]]、[[秦志戬|秦志戬]]、[[樊振东|樊振东]]、[[王皓_(乒乓球运动员)|王皓]]<ref name="shw1"/>等人在[[新浪微博|新浪微博]]公开声援刘国梁。同日，许昕、马龙、樊振东三名乒乓球员退出2017[[中国乒乓球公开赛|中国乒乓球公开赛]]男子单打16进8比赛<ref name="a1"/>。此后引爆中国大陆舆论，涉及[[中国体育|中国体育]]最高领导者——[[国家体育总局|国家体育总局]]局长[[苟仲文|苟仲文]]。舆论对运动员、教练表示支持，指责国家体育总局官员滥用权利，使国乒运动员、教练沦为权力斗争的牺牲品<ref name="a2"/>，更有网友批评苟仲文“不懂体育、瞎指挥”，甚至有呼声要求苟应引咎辞职<ref>{{Cite web |url=http://www.thepaper.cn/newsDetail_forward_1750478|title=中国新闻周刊：在争议中重启的中国体育改革 |accessdate=2017-08-02 |author= |date=2017-08-02|work= |publisher=微信公众号“中国新闻周刊杂志”|language=Zh-hans }}</ref>。最终，在2018年9月间，刘国梁“复出”，年底出任[[中国乒乓球协会|中国乒乓球协会]]主席。

== 背景 ==
乒乓球运动在中国有“[[国球|国球]]”之称，[[中国国家乒乓球队|中国国家乒乓球队]]历来战绩辉煌，夺得冠军的次数位居中国所有体育国家队之最。

== 事件经过 ==

=== 刘国梁下课 ===

2017年3月间，国乒举行教练员竞聘会<ref name="a3"/>。4月，公布刘国梁留任国乒总教练，[[秦志戬|秦志戬]]任男队主教练，[[孔令辉|孔令辉]]留任女队主教练。[[马琳|马琳]]、[[王皓_(乒乓球运动员)|王皓]]等任男一队教练<ref>{{Cite web |url=http://www.guancha.cn/sports/2017_04_07_402517.shtml|title=国乒竞聘结果出炉 王皓马琳将出任男队教练 |accessdate=2017-07-05 |author= |date=2017-04-07|work= |publisher=观察者网|language=Zh-hans }}</ref>。

5月底，女队主教练孔令辉因涉及赌博被中国乒乓球协会暂停职务。6月20日，中国乒乓球协会宣布撤销国乒总教练、男女两队主教练职位<ref name="a3">{{Cite web |url=http://www.sohu.com/a/152148456_498692|title=刘国梁事件回顾：国乒历史上的一次“惊天变”！ |accessdate=2017-07-05 |author= |date=2017-06-26|work= |publisher=搜狐网|language=Zh-hans }}</ref>，男女队各自设立教练组，实行扁平化管理<ref name="a2">{{Cite web |url=http://www.sohu.com/a/151742089_611480|title=国乒与股民适当性:是谁在傲慢? |accessdate=2017-07-05 |author= |date=2017-06-24|work= |publisher=搜狐网|language=Zh-hans }}</ref>。总教练刘国梁成为中国乒乓球协会副主席，此前中国乒乓球协会已有18位副主席，另有4位名誉副主席和4位特邀副主席<ref name="bjsj"/>。

有人认为刘国梁下课及后续事件，是[[国家体育总局|国家体育总局]]局长[[苟仲文|苟仲文]]与副局长[[蔡振华|蔡振华]]的权力斗争<ref>{{Cite web |url=http://www.sohu.com/a/153127183_488304|title=关于刘国梁事件，你所不知道的事实... |accessdate=2017-07-05 |author= |date=2017-06-29|work= |publisher=搜狐网|language=Zh-hans }}</ref>。有报道，进一步指出是苟仲文为与蔡振华争夺即将在[[中共十九大|中共十九大]]选举产生的[[中国共产党第十九届中央委员会委员|中央委员]]之位<ref name="a4">{{Cite web |url=http://china.dwnews.com/news/2017-06-25/59821839.html|title=中国乒坛兵变 或影响蔡振华入十九大 |accessdate=2017-07-05 |author=综编：朱建华 |date=2017-06-25|work= |publisher=多维新闻|language=Zh-hans }}</ref>。此前苟仲文虽在[[中共北京市委|中共北京市委]]任职，但未进入[[中共中央|中共中央]]高层。他的前任[[刘鹏|刘鹏]]曾任[[中国共产党第十六届中央委员会候补委员|中共十六届中央候补委员]]，[[中国共产党第十七届中央委员会委员|十七届]]、[[中国共产党第十八届中央委员会委员|十八届中央委员]]。而蔡振华曾任[[中国共产党第十七届中央委员会候补委员|中共十七届]]、[[中国共产党第十八届中央委员会候补委员|十八届中央候补委员]]。

=== 罢赛 ===

6月22日2017中国乒乓球公开赛进入正式比赛阶段。下午男子单打首轮比赛中，张继科（时世界排名第四）因伤退出与日本球员[[吉田雅己|吉田雅己]]比赛<ref>{{Cite web |url=http://news.163.com/17/0623/04/CNJBM7T8000187VI.html|title=中乒赛国手蓉城亮相 张继科男单首轮因伤退赛 |accessdate=2017-07-10 |author=责任编辑：王晓易_NE0011 |date=2017-06-23 |work= |publisher=网易，来源：成都晚报(成都)，四川新闻网-成都晚报 |language=Zh-hans }}</ref>。同日，已退役的乒乓球运动员[[王楠|王楠]]的丈夫郭斌在微博表示，国乒男女队员被集体禁言、禁止使用微博<ref name="bjsj"/>。

6月23日，中国乒乓球公开赛成都站举行开幕式。开幕式上，记者询问蔡振华对国乒人事变动的看法，蔡回应“具体落实的细节，你应该问现在的[[国家体育总局乒乓球羽毛球运动管理中心|乒羽中心]]。”<ref name="bjsj">{{Cite web |url=http://item.btime.com/36s13pntvmt8is9g28garfb4oe7|title=刘国梁事件回顾:任乒协副主席 国乒喊同一句话 |accessdate=2017-07-05 |author= |date=2017-06-23 |work= |publisher=北京时间|language=Zh-hans }}</ref>将矛盾公开化。

同日19:00点，国乒男队队员、教练——[[许昕|许昕]]（世界排名第三）、[[马龙_(乒乓球运动员)|马龙]]（世界排名第一）、[[马琳|马琳]]、[[秦志戬|秦志戬]]、[[樊振东|樊振东]]（世界排名第二）、[[王皓_(乒乓球运动员)|王皓]]<ref name="shw1">{{Cite web |url=http://www.sohu.com/a/151664459_505562|title=仨国手先后退赛，成都风暴席卷国乒……|accessdate=2017-07-05 |author=沈阳晚报体育部 |date=2017-06-24 |work= |publisher=搜狐网|language=Zh-hans }}</ref>、[[陈玘|陈玘]]<ref name="shw1"/>（前国乒队员）在[[新浪微博|新浪微博]]以统一格式发文：这一刻我们无心恋战……只因想念你“刘国梁”。随后，[[国际乒联|国际乒联]]陆续确认马龙、樊振东、许昕退出中国乒乓球公开赛。在成都赛场，观众齐声高呼“刘国梁”以示支持<ref name="a3"/>。

=== 事后处理 ===

6月24日，[[中国乒乓球协会|中国乒乓球协会]]在官网发表声明，表示将就事件反思<ref name="a1"/>。6月25日凌晨，马龙、许昕、樊振东、马琳等人转发中国乒乓球协会道歉信，表示歉意<ref name="a4"/>。

在中国乒乓球协会发表声明及中国国家乒乓队发表道歉信后，国际乒联对于中国球员退赛，要求中国乒乓球协会就此事提供详细报告，同时要求在场的国际乒联相关官员努力查清球员未能按时参赛的具体原因<ref name="b1"/>。

6月27日，刘国梁通过其个人微博公开道歉<ref>{{cite web|title=刘国梁就“退赛”事件道歉：管理上有不可推卸责任|url=https://www.chinanews.com/ty/2017/06-28/8263031.shtml|website=北京青年报|accessdate=2017-06-28|date=2017-06-28}}</ref>。6月29日，国际乒联发布7月2日[[澳大利亚乒乓球公开赛|澳大利亚乒乓球公开赛]]参赛名单，中国国家乒乓球队男队队员全部退出比赛。国际乒联表示，这是中国乒乓球协会的决定。比赛报名是成都赛事之前，退赛球员共有六名，分别是张继科、许昕、樊振东、[[闫安|闫安]]、[[林高远|林高远]]、[[梁靖崑|梁靖崑]]。女队[[朱雨玲|朱雨玲]]、[[陈梦|陈梦]]、[[陈幸同|陈幸同]]、[[顾玉婷|顾玉婷]]、[[张蔷_(乒乓球运动员)|张蔷]]、[[王曼昱|王曼昱]]六人仍参加此次比赛<ref>{{Cite web |url=http://sports.sohu.com/20170629/n499117092.shtml|title=国乒男队集体退出澳大利亚公开赛 女队正常参赛 |accessdate=2017-07-10 |author= |date=2017-06-29 |work= |publisher=搜狐网，来源：澎湃新闻|language=Zh-hans }}</ref>。随后，中国国家乒乓球队在中国乒乓球协会官网发表公告称：国乒男队退赛是因为连续参赛带来的身体过度疲劳，加之伤病困扰，不具备参赛的竞技状态。马龙、许昕和樊振龙三人亦在微博做出解释，马龙则是未报名参赛<ref>{{Cite web|url=http://www.sohu.com/a/153215859_486006|title=【乒乓球】中国乒乓球男队退出澳大利亚公开赛，球队官方回应：过度疲劳、伤病困扰|accessdate=2017-07-10|author=|date=2017-06-30|work=|publisher=搜狐网|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20170803131845/http://www.sohu.com/a/153215859_486006|archivedate=2017年8月3日|df=}}</ref>。

7月7日<ref name="b1"/>，国际乒联发布最新的一期全球排名，马龙、许昕、樊振东因为之前的罢赛各被扣除25个积分，但他们世界前三的名次没有变化<ref>[http://slide.news.sina.com.cn/slide_1_86058_182480.html/d/1#p=1 国际乒联公布最新排名 马龙被扣25积分].新浪体育.2017-07-07.[2017-07-10].</ref>。2017中国乒乓球公开赛冠军的德国球员[[迪米特里·奥恰洛夫|奥恰洛夫]]积分排名超过张继科升至第四位，亚军德国球员[[蒂莫·波尔|波尔]]升至第六位<ref name="b1">{{Cite web |url=http://www.cqcb.com/headline/2017-07-07/393249_pc.html|title=国际乒联重罚马龙樊振东许昕 三人均被罚25分 |accessdate=2017-07-10 |author= |date=2017-07-07|work=责任编辑：黄硕 US042 |publisher=头条—重庆晨报，来源:人民日报客户端|language=Zh-hans }}</ref>。

7月8日，公布选举产生的国家机关的中共十九大代表，苟仲文当选，蔡振华未入选<ref>{{Cite web |url=http://china.dwnews.com/news/2017-07-08/59824335.html |title=中共国家机关十九大代表出炉 蔡振华落选|accessdate=2017-07-23 |author=编辑：南希 |date=2017-07-08 |work= |publisher=星岛环球网|language=Zh-hans }}</ref>。
