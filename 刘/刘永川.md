'''刘永川'''（{{lang-en|[[:en:_Alex_Liu|Alex Liu]]}}）是一位“研究方法和[[数据科学|数据科学]]”的知名学者，被广泛称为“大数据分析与机器学习应用于商业与社会研究”的顶尖专家。刘永川也是[[中国|中国]]1980年代至90年代的留学生代表人物。

==职业简历==
刘永川1982年获得中国[[西北工业大学|西北工业大学]]的航空机械学士、1986年获得[[北京大学|北京大学]]的社会学硕士、1993年获得美国[[斯坦福大学|斯坦福大学]]的统计计算硕士和社会学博士。

刘博士1993年开始在[[斯坦福大学|斯坦福大学]]亚太研究中心从事医疗服务的比较研究、之后在美国[[IBM研究院|IBM研究院]]研究因果分析的方法和算法、再到[[圣地牙哥加利福尼亚大学|圣地牙哥加利福尼亚大学]]瑞帝管理学院研究落后地区创新创业的社会环境、并担任研究员。2002年至2011年,刘永川曾在[[南加州大学|南加州大学]]马歇尔商学院和[[尔湾加州大学|尔湾加州大学]]保罗梅拉吉商学院教授博士研究生的高级社会研究方法，并曾到南亚的斯里兰卡、中亚的哈萨克斯坦与吉尔吉斯斯坦和非洲的肯尼亚从事扶贫项目评估方法和创业发展援助的项目。刘博士2009年创建“研究方法与数据科学”全球协会并担任负责人，自2010年起曾担任多家跨国公司如 TRG 和 RS 的首席数据科学家，自2013年起担任 IBM 大数据分析的首席数据科学家, 于2018年获得[[IBM|IBM]]和开源行业协会的"数据科学"思想领袖证书。

==成长经历==
刘永川出生于中国、江西、[[井冈山|井冈山]]<ref>{{Cite web |title=中文博客|url = http://www.researchmethods.org/Chometown.htm|accessdate=2018-11-5}}</ref>，在1977年的高考中考入西北工业大学，是在中国引入数量社会科学的先驱之一<ref>{{Cite web |title=国外社会科学杂志|url = http://mall.cnki.net/magazine/article/GWSH198510023.htm|accessdate=2018-11-5}}</ref> <ref>{{Cite web |title=哲学研究|url = http://www.cnki.com.cn/Article/CJFDTotal-ZXYJ198106012.htm|accessdate=2018-11-5}}</ref>。他于1986年赴美国留学，1987年担任斯坦福大学[[中国学生学者联合会|中国学生学者联合会]]会长，1988年担任加州中国学生学者联合会主席，1989年7月经当时在美国的4万多中国留学生的代表选举担任[[全美中国学生学者自治联合会|全美中国学生学者自治联合会]]首届主席<ref>{{Cite web |title=CND|url = http://hx.cnd.org/2009/08/03/重聚芝加哥──全美学自联成立二十周年纪念集会/|accessdate=2018-11-7}}</ref>，于1990年8月经选举担任[[全球中国学联|全球中国学联]]首届总召集人。1995年至1997年，刘博士曾在美国硅谷主办 “华裔美国人创业高峰会议”，和“中美兵兵外交25周年纪念”<ref>{{Cite web |title=Pacific Rim Business Journal|url = http://www.researchmethods.org/diploma25.htm |accessdate=2018-11-7}}</ref>等。刘博士于2000年加入[[美国国籍|美国国籍]]，2004年成为[[基督徒|基督徒]]<ref>{{Cite web |title=生命与信仰|url = https://www.cclifefl.org/View/Article/1816|accessdate=2018-11-5}}</ref> <ref>{{Cite web |title=CND|url = http://hx.cnd.org/2004/12/21/大陆新移民向上帝靠拢/|accessdate=2018-11-5}}</ref>。2006年至2011年，刘博士提出[[精神资本|精神资本]]主导的四资本发展理论和相关的整体方法论<ref>{{Cite web |title=BOOK Spiritual Capital|url = https://www.taylorfrancis.com/books/e/9781351548021/chapters/10.4324%2F9781315087740-4/|accessdate=2018-11-5}}</ref>，近年专注于研究方法的中西比较分析。
