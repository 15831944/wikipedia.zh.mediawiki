'''刘炽平'''（{{bd|1973年}}），[[腾讯公司|腾讯公司]]总裁、执行董事，出生于[[北京|北京]]。<ref name=":0">{{Cite web|url = http://tech.hexun.com/2007-07-10/100148433.html|title = 腾讯公司总裁刘炽平：我没有活在马化腾的阴影下|accessdate = |author = 陈亮|date = 2007年7月10日|publisher = 和讯网}}</ref>

== 早年生涯 ==

刘炽平拥有美国[[密歇根大学|密歇根大学]]电子工程学士銜，[[斯坦福大学|斯坦福大学]]电子工程硕士銜以及[[西北大学_(伊利诺伊州)|西北大学]][[凯洛格管理学院|凯洛格管理学院]]工商管理硕士銜。<ref name=":0" /><ref name=":1">{{Cite web|url = http://news.pcgames.com.cn/chinajoy/2009/cgdc/people/0907/1508221.html|title = 高峰论坛演讲人刘炽平 腾迅公司总裁|accessdate = |author = |date = 2009-07-14|publisher = 太平洋游戏网}}</ref>在进入腾讯公司之前刘炽平曾担任[[高盛|高盛]]亚洲投资银行部的执行董事及电信、媒体与科技行业组的首席运营官。在此之前刘炽平在[[麦肯锡|麦肯锡]]公司从事管理咨询工作，在股票发行、兼并与收购以及管理咨询等领域工作长达9年。<ref name=":0" /><ref name=":1" />

== 任職腾讯 ==

2003年9月刘炽平曾和腾讯公司创始人马化腾有过接触。2005年2月刘炽平加盟腾讯公司，被任命为腾讯首席战略投资官。2006年2月15日，刘炽平出任腾讯公司总裁，协助马化腾负责公司的日常管理和运营。2007年3月，刘炽平被任命为腾讯公司执行董事。<ref name=":0" /><ref name=":1" />

2011年7月28日，[[金山软件|金山软件]]非执行董事[[张旋龙|张旋龙]]辞任，由刘炽平接任<ref>{{Cite web|url = http://tech.sina.com.cn/i/2011-07-29/11345855281.shtml|title = 张旋龙辞任金山非执行董事 刘炽平接任|accessdate = |author = |date = 2011年7月29日|publisher = 新浪}}</ref>。2011年9月19日，刘炽平辞去[[英利绿色能源控股有限公司|英利绿色能源控股有限公司]]独立董事兼薪酬委员会委员职务<ref>{{Cite web|url = http://finance.ifeng.com/usstock/realtime/20110919/4617642.shtml|title = 英利原独立董事刘炽平辞职 Richard Xue接任|accessdate = |author = |date = 2011年9月19日|publisher = 凤凰网}}</ref>。
