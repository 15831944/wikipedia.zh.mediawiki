'''刘青山张子善案'''是[[中华人民共和国|中华人民共和国]]建立初期，知名的[[贪污|贪污]]案件。1951年底案发，至1952年2月，[[刘青山_(河北省)|刘青山]]、[[张子善|张子善]]二人被[[枪决|枪决]]。

== 案情 ==
1950年至1951年[[刘青山_(河北省)|劉青山]]、[[张子善|张子善]]兩人在擔任[[天津专区|天津专区]]領導的期間，盜竊地方糧款289151萬元{{NoteTag|人民幣舊幣，舊幣1萬元折合新幣1元。<ref name="aa"/>}}、防汛水利專款30億元、救災糧款4億元、幹部家屬救濟糧款14000萬元，剋扣修理機場民工供應補助糧款54330萬元，賺取治河民工供應糧款37473萬元，倒賣治河民工食糧從中漁利22億元；此外還以修建為名騙取銀行貸款60億元，從事非法經營。有说总计1716272万元（[[第一套人民币|第一套人民币]]面值），可购近一吨黄金<ref name="a1">{{Cite web|url=http://history.voc.com.cn/article/201502/201502101009379513.html|title=历史上的今天:公审刘青山张子善 两人到底贪污多少?|accessdate=2016-05-27|author=责编：蒋俊|date=2015-02-10|work=|publisher=华声在线网站|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20150630224315/http://history.voc.com.cn/article/201502/201502101009379513.html|archivedate=2015-06-30}}</ref>。

== 案发 ==
中共天津地委委员、行署副专员[[李克才|李克才]]曾向上级举报刘青山、张子善二人，但无效果。1951年10月，李克才再度向[[中共河北省委|中共河北省委]]举报刘张二人。河北省委组织部部长、省纪检委副书记[[薛迅|薛迅]]负责接待。10月下旬，由薛迅带队的河北省委纪律调查组进入天津地委机关<ref name="a3">{{Cite web |url=http://www.xnwbw.com/html/2012-04/27/content_45308.htm |title=震惊世界的枪声——回眸新中国肃贪首案 |accessdate=2016-05-27 |author= |date=2012-04-27 |work= |publisher=西宁晚报电子版 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160617001204/http://www.xnwbw.com/html/2012-04/27/content_45308.htm |archivedate=2016-06-17 }}</ref>。

1951年11月12日至12月1日，在河北省保定举行中共河北省委第三次代表会议。李克用于11月21日，在此次会议上公开揭露刘青山、张子善的贪污问题。[[中共河北省委|中共河北省委]]当即成立“刘青山、张子善大贪污案调查处理委员会”。[[河北省人民政府|河北省人民政府]]主席[[杨秀峰|杨秀峰]]担任主任，[[河北省人民检察署|河北省人民检察署]]检察长<ref name="a2"/>[[孙光瑞|孙光瑞]]担任办案组组长。

11月29日，河北省公安厅在保定某家饭店逮捕张子善<ref name="a4">{{Cite web |url=http://barb.sznews.com/html/2013-01/30/content_2368226.htm |title=刘青山：以我做个“典型”吧 |accessdate=2016-05-27 |author= |date=2013-01-30 |work= |publisher=宝安日报多媒体数字版 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160813112826/http://barb.sznews.com/html/2013-01/30/content_2368226.htm |archivedate=2016-08-13 }}</ref>。同日，[[中共中央华北局|中共中央华北局]]将河北省委的请示作了讨论，并向中共中央、[[毛泽东|毛泽东]]作出书面报告。此前刘青山已作为中国青年友好代表团成员，出席[[奥地利|奥地利]][[维也纳|维也纳]]举行的世界和平友好理事大会。当时身处苏联的刘青山被电召回国。12月2日，刘青山随团乘火车抵达天津<ref name="a4"/>，在马场道18号“刘公馆”被控制<ref name="a3"/>，与张子善分别关押于保定的[[河北省公安厅|河北省公安厅]]内<ref name="a3"/><ref name="a4"/>。4日，经华北局批准，河北省委<ref name="aa">{{Cite web |url=http://cpc.people.com.cn/GB/64156/64157/4512322.html|title=刘青山、张子善案件 |accessdate=2016-05-27 |author= |date= |work= |publisher=人民网，新华网|language=Zh-hans }}</ref>开除刘青山、张子善[[党籍|党籍]]<ref name="a3"/>。

12月14日，河北省委向华北局提交处理意见，提议“处以[[死刑|死刑]]”。20日，华北局向中共中央提交处理意见，处理意义增加“或[[死刑缓期两年执行|缓期二年执行]]”<ref name="a2"/>。

1952年1月18日，孙光瑞签署河北省人民检察署对刘张案的公诉书<ref name="a1"/>。2月10日，在保定市体育场举行刘青山、张子善贪污案[[公审大会|公审大会]]。公审大会后，两人被押赴保定东关大校场<ref name="a4"/>刑场枪决<ref name="a2">{{Cite web |url=http://www.xtrb.cn/epaper/xzb/html/2011-09/25/content_283325.htm |title=刘青山、张子善最后的“晚餐”  |accessdate=2016-05-27 |author= |date=2011-09-25 |work= |publisher=邢台民生网站，来源：邢台日报-邢周报|language=Zh-hans }}</ref>。
