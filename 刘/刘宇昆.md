'''刘宇昆'''（{{lang-en|Ken Liu}}，{{bd|1976年|||}}）是一名[[美籍华裔|美籍华裔]][[科幻|科幻]]和[[奇幻作品|奇幻]]作家，也是翻譯員、律師和電腦程序員。

== 生平 ==
劉宇昆在1976年出生於中國[[甘肅省|甘肅省]]的[[蘭州市|蘭州市]]，8岁<ref name="bjd1" />（一说11岁<ref name="bjd1" /><ref name="雅虎">{{cite web| language =zh-cn| publisher =雅虎| title =华裔作家斩获科幻界诺贝尔奖 科幻也能写真情| url =http://cul.cn.yahoo.com/ypen/20120905/1293286.html| date =2012-09-05| accessdate =2011-09-07| deadurl =yes| archiveurl =https://web.archive.org/web/20121029171843/http://cul.cn.yahoo.com/ypen/20120905/1293286.html| archivedate =2012-10-29}}</ref>）时随父母从兰州移民[[美国|美国]]。毕业于[[哈佛大学|哈佛大学]]，主修[[英国文学|英国文学]]和[[法学|法学]]，业余副修计算机。毕业后成为[[软件工程师|软件工程师]]。后又进入[[哈佛法学院|哈佛法学院]]读研究生<ref name="bjd1">{{cite web| language =zh-cn| publisher =京报网，来源：北京晚报| title =刘宇昆：他把《三体》推向世界| url =http://bjwb.bjd.com.cn/html/2015-08/28/content_307661.htm| date =2015-08-28| accessdate =2015-09-19| author =须叔| deadurl =yes| archiveurl =https://web.archive.org/web/20160617060155/http://bjwb.bjd.com.cn/html/2015-08/28/content_307661.htm| archivedate =2016-06-17}}</ref>。目前在[[波士顿|波士顿]]从事[[律师|律师]]的工作。业余写作科幻小说。

2002年随着《迦太基的玫瑰》一文问世，刘宇昆逐渐受到美国文学界瞩目<ref name="bjd1"/>。2012年，他凭借短篇小说《[[手中纸，心中爱|手中纸，心中爱]]》（The Paper Menagerie）分别夺得[[星云奖|星云奖]]最佳短篇故事奖和[[雨果奖|雨果奖]]最佳短篇故事奖，<ref>{{cite web | language = zh-cn | publisher = 搜狐 | title =华裔作家刘宇昆获雨果奖 小说充满东方文化色彩| url = http://cul.sohu.com/20120905/n352357199.shtml| date = 2012-09-05 | accessdate = 2012-09-07 }}</ref>成为第一位获得星云奖和雨果奖两项世界科幻小说大奖的华裔作家。<ref name="雅虎"/>2013年9月，刘宇昆凭借作品《''Mono no Aware''》再次荣获雨果奖最佳短篇小说奖。

刘宇昆同时致力于中美文化交流，曾将[[刘慈欣|刘慈欣]]、[[郝景芳|郝景芳]]、[[陈楸帆|陈楸帆]]、[[马伯庸|马伯庸]]、[[夏笳|夏笳]]等[[中国|中国]]科幻作家的作品译成[[英文|英文]]在国外发表。<ref>{{cite web | language = zh-cn | publisher = 搜狐 | title =起到了沟通中西科幻的桥梁作用的华裔作者刘宇昆| url = http://roll.sohu.com/20120905/n352393527.shtml| date = 2012-09-05 | accessdate = 2012-09-07 }}</ref>他在2012年11月7日签约担任《[[三体_(小说)|三体]]》的译者。<ref>{{cite web | language = zh-cn | publisher = 凤凰网 | title =《三体》今起开译英文版| url = http://news.ifeng.com/gundong/detail_2012_11/07/18906164_0.shtml| date = 2012-11-07 | accessdate = 2013-07-01 }}</ref>2015年8月，《三体》获得雨果奖最佳长篇小说奖。2016年8月，他翻译的[[郝景芳|郝景芳]]的小说《[[北京折叠|北京折叠]]》获雨果奖最佳中短篇小说奖。2017年，刘宇昆的《折纸和其他故事（The Paper Menagerie and Other Stories）》获得[[軌跡獎|軌跡獎]]最佳短篇奖<ref>{{Cite web |url = http://www.locusmag.com/News/2017/06/do-not-touch-2017-locus-awards-winners/ |title = 2017 Locus Awards Winners |author =  |date = 2017-06-24 |publisher =  |language = en |accessdate = 2017-06-25 }}</ref>，日文譯本則由[[早川書房|早川書房]]於2017年出版，書名為「母の記憶に」<ref>{{Cite web |url = http://www.hayakawa-online.co.jp/shopdetail/000000013515/ |title =母の記憶に  |author =  |date = |publisher =  |language = ja |accessdate = 2017-10-25 }}</ref>。
