'''刘聘三故居'''，位于[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[镇海区_(宁波市)|镇海区]][[贵驷街道|贵驷街道]]贵驷社区景德路1号，建于民国时期，是[[中华劝工银行|中华劝工银行]]总经理、[[宁波旅沪同乡会|宁波旅沪同乡会]]副会长[[刘聘三|刘聘三]]的故居。<ref name="auto">{{cite web |url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=22538&type=2 |title=刘聘三故居 |author=<!--Staff writer(s); no by-line.--> |date=2018-01-08 |website=宁波文化遗产保护网|publisher=宁波市文物保护管理所 |access-date=2018-07-19 |quote=|deadurl=no|archiveurl=https://web.archive.org/web/20180719073146/http://www.nbwb.net/pd_wwbh/info.aspx?Id=22538&type=2|archivedate=2018-07-19}}</ref><ref>{{cite web |url=http://202.107.212.146:838/nbwshj/index.php?option=com_content&view=article&id=121:2011-09-08-05-57-38&catid=38:2011-09-07-00-48-02&Itemid=57 |title=宁波旅沪同乡会(前期)大事记 |author=董心琴 |date= |website=宁波文史资料汇集 |publisher=宁波市政协文史资料委员会 |access-date=2018-07-19 |quote=该会现任会长方椒伯、副会长刘聘三，负责人也都是宁波旅沪同乡会的负责人|deadurl=no|archiveurl=https://web.archive.org/web/20160318022606/http://202.107.212.146:838/nbwshj/index.php?option=com_content&view=article&id=121:2011-09-08-05-57-38&catid=38:2011-09-07-00-48-02&Itemid=57|archivedate=2016-03-18}}</ref>
2012年3月2日，刘聘三故居被公布为[[镇海区文物保护单位|镇海区文物保护单位]]。因自2015年9月起[[贵驷街道|贵驷街道]]由[[宁波高新技术产业开发区|宁波高新技术产业开发区]]实际管辖<ref>{{cite web|url=http://gjgxq.ningbo.gov.cn/doc/zwgk/gxdt/2015_9_1/232673.shtml|title=贵驷街道委托管理仪式举行|publisher=宁波国家高新技术产业开发区管理委员会|date=2015-09-02|accessdate=2017-10-22|deadurl=yes|archiveurl=https://web.archive.org/web/20171023012344/http://gjgxq.ningbo.gov.cn/doc/zwgk/gxdt/2015_9_1/232673.shtml|archivedate=2017-10-23}}</ref>，故2016年4月刘聘三故居由镇海区移交至[[宁波高新技术产业开发区|宁波高新技术产业开发区]]管辖。

== 建筑特色 ==
刘聘三故居坐北朝南，占地606平方米，为中西合璧风格的[[硬山顶|硬山顶]]砖木结构建筑。刘聘三故居由前后正屋和前后、东西四厢房组成。前后正屋均面阔五间两层，进深两柱七檩前廊。其中前正屋一层为厅堂，有四根圆形水泥挑[[柱#檐柱|檐柱]]，二层有方形水泥立柱，两层均铺设木制方条地板与小木条天花板，后正屋二楼有方桁梁，亦铺设有木制方条地板与小木条天花板，屋顶安装水泥葫芦瓶围栏。前东西厢房面阔两间两层，进深两柱七檩前廊，二楼于墙边安装壁炉。后东西厢房各一间一弄，进深两柱四檩。刘聘三故居对于宁波帮名人的研究起到一定价值。<ref>{{cite web |url=http://www.nbwb.net/info.aspx?Id=10499 |title=刘聘三故居 |author=<!--Staff writer(s); no by-line.--> |date=2010-11-23  |website=宁波文化遗产保护网 |publisher=宁波市文物保护管理所 |access-date=2018-07-19 |quote=|deadurl=no|archiveurl=https://web.archive.org/web/20180719073511/http://www.nbwb.net/info.aspx?Id=10499|archivedate=2018-07-19}}</ref><ref name="auto"/>
