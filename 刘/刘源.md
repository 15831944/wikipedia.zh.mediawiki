'''刘源'''（{{bd|1951年|2月22日|catIdx=L刘}}），[[湖南省|湖南省]][[宁乡市|宁乡市]]人，出生于[[北京|北京]]，是前[[中共中央副主席|中共中央副主席]]、首任[[全国人大常委会委员长|全国人大常委会委员长]]、第二任[[中华人民共和国主席|中华人民共和国主席]][[刘少奇|刘少奇]]与[[王光美|王光美]]之子。曾就读于[[北京市第四中学|北京市第四中学]]。刘源先从政，后从军。从政期间，历任[[河南省|河南省]][[郑州市|郑州市]]副市长，河南省副省长。进入军界后，历任[[武警部队|武警部队]]副政委，[[中国人民解放军军事科学院|中国人民解放军军事科学院]]政委、[[中国人民解放军总后勤部|中国人民解放军总后勤部]]政治委员等职。是[[中国共产党|中国共产党]]第十七届、十八届中央委员，[[中国人民解放军上将|上将]][[中国人民解放军军衔|军衔]]。现任[[全国人大财政经济委员会|全国人大财政经济委员会]]副主任委员。

== 简历 ==
=== 早年经历 ===
1951年生于北京；1964年7月，从[[北京第二实验小学|北京实验二小]]毕业，在[[中央警卫团|中央警卫团]][[军训|军训]]一个暑假，在[[中南海|中南海]]站门岗；1964年至1965年，[[北京四中|北京四中]]读初一；1965年7至8月，去位于北京郊区的[[北京卫戍区|北京卫戍区]]骑兵连学骑马；1965年至1966年，北京四中读初二；把北京四中高三（5）班的两位学长写的一封公开信转交给时任国家主席[[刘少奇|刘少奇]]。这封公开信要造资产阶级教育制度的反，称当时的教育考试制度特别是高考制度让搞白专的留下来了，把闹革命的工农兵都挡在学校外面。这封信刊载在1966年6月18日的《[[人民日报|人民日报]]》上。1966年6月[[高考|高考]]制度正式取消。1966年7月，在[[北京卫戍区|北京卫戍区]]警卫3师9团某连军训；8月5日，中共中央主席[[毛泽东|毛泽东]]发表针对国家主席[[刘少奇|刘少奇]]的《[[炮打司令部——我的一张大字报|炮打司令部——我的一张大字报]]》，刘源被部队退回。在北京四中参加了中学生[[红卫兵|红卫兵]]组织——[[西纠|西城区纠察队]]。1967年初到4月间的某一段时间，与西纠的战友们蹲了[[中华人民共和国公安部|公安部]]的监狱。

1967年9月13日，刘源和[[刘平平|刘平平]]、[[刘亭亭|刘亭亭]]两姊妹，连同简单的行李卷，被抛上一辆卡车，赶出了[[中南海|中南海]]。刘源无家可归住进了北京四中。1968年主动申请[[上山下乡|上山下乡]]，随大批北京知青一起分到[[山西省|山西省]][[雁北地区|雁北地区]][[山阴县|山阴县]]白坊大队插队七年。同年10月，父亲刘少奇被罢免一切职务开除出党，翌年被迫害致死。当时专案组冒用刘源的名义填写了刘少奇的屍體火化單。1975年秋，作为白坊大队最后一名北京知青抽调回城，到[[北京起重机厂|北京起重机厂]]当铆工。（从1970年开始所有的上山下乡北京知青都分在北京郊县插队，在山西已经没有新插队的北京知青；别的省市也大都推行了类似的城市知青就近插队落户的政策。）1977年参加恢复后的首届高考，被北京师范学院（今[[首都師範大學|首都師範大學]]）历史系录取，成为七七级大学本科生。

=== 从政河南 ===
1982年春大学毕业时加入中国共产党。1982年后，任[[河南省|河南省]][[新乡县|新乡县]][[七里营镇|七里营人民公社]]党委副书记、[[七里营乡|七里营乡]]副乡长，人称“[[人民公社|公社]]第十七把手”，分管[[社队企业|社队企业]]，奋斗一年抓出了成效。1983年4月，经新乡县人大常委会全票选举为[[新乡县|新乡县]]副县长，负责抓全县的多种经营和工业生产。1984年又全票当选为新乡县县长。1985年初，当选为[[郑州市|郑州市]]副市长，分管城市建设和对外经济贸易工作的，主持完成了郑州市天然气进城工程，并领导了城市建设的一系列工程，获得百姓好评。

1987年底至1988年初，在河南省第七届人民代表大会上，许多代表对河南省省级干部的候选人名单不满意，因为可供差额的比例太小，另外候选人的年龄太大。所以河南代表联名举荐（不是由中共党组织提名）刘源为河南省副省长候选人，[[中共河南省委|中共河南省委]]向[[中共中央|中共中央]]请示，中央表示尊重[[人民代表|人民代表]]的意愿。最终刘源以高票当选河南省副省长。任副省长期间，刘源主管工业、交通和安全工作<ref name="lyhn">{{cite web |url = http://history.chinaiiss.com/html/20136/3/a5f180_4.html |title = 揭秘刘源的将星之路 河南人评他一个字“好” |publisher = 战略网 |date = 2013-06-03 |language = zh-hans }}</ref>。参与领导了全省大型工业项目建设工作，如全国最大的50万伏直流超高压变电站、全国投资最大的中原制药厂（后因[[六四事件|六四事件]]后的国际形势对中国的不利影响，该厂最终倒闭）、拥有世界最先进的三套设备之一的中原化肥厂相继建成投产<ref>{{Cite book|title=湖南党建90年 下|last=雷国珍主编；刘华清，龚永爱副主编|first=|publisher=长沙：湖南人民出版社|year=2011.07|isbn=978-7-5438-7629-3|location=|pages=193}}</ref>。

1989年[[1989年中国民主运动|北京学生运动]]期间，刘源在[[天安门广场|天安门广场]]和火车站等地夜以继日的劝学生保持理性，并成功劝回大批到北京声援的河南学生。在国务院总理[[李鹏|李鹏]]宣布[[戒严|戒严]]时，刘源反对镇压。[[中共河南省委|中共河南省委]]、[[河南省人民政府|河南省人民政府]]亦是当时少数几个反对镇压的省份。[[六四事件|六四事件]]最终以镇压收场、包括总书记[[赵紫阳|赵紫阳]]在内的中共高层易人，刘源及河南省委在六四事件中的态度不仅使刘源本人的仕途受挫，连整个河南省也遭一股势力大肆攻击。在六四事件后的20年间，河南省几乎没有中央任何利好政策，经济、教育遭打压。<ref>{{cite book |author = |title =《明鏡月刊》 28期 |publisher = 明镜出版社 |year = 2012年 |ISBN = 9781630321758 |page = 第77页}}</ref>

=== 迁职军界 ===
1992年5月，刘源任[[武警水电指挥部|武警水电指挥部]]第二政委兼水电指挥部副主任，率部参加[[三峡水利枢纽工程|三峡水利枢纽工程]]建设<ref>{{Cite book|title=石壁立西江 中国三峡工程决策建设实录|last=张立先著|first=|publisher=武汉：长江出版社|year=2010.05|isbn=978-7-80708-867-7|location=|pages=108}}</ref>，被授予[[中国人民解放军少将|武警少将]][[警衔|警衔]]<ref>{{cite website |title = 刘源查办谷俊山挖背后两大老虎 三句话震巨奸 |url = http://news.163.com/15/1224/00/BBIES9SR0001124J.html |accessdate = 2015-12-24 |website = 网易 |date=2015-12-24}}</ref>。离开河南省那天，刘源没有声张，只是去了新乡七里营乡刘庄村告别，许多人还是跑到[[郑州站|郑州火车站]]为他送行。<ref name="lyhn" />

1998年任[[中国人民武装警察部队|中国人民武装警察部队]]副政委。2000年晋升武警中将警衔。2003年任[[中国人民解放军总后勤部|中国人民解放军总后勤部]]副政委，被授予[[中国人民解放军中将|解放军中将]]军衔。2005年任[[中国人民解放军军事科学院|中国人民解放军军事科学院]]政治委员。2009年晋升上将軍銜。2010年12月，出任[[中国人民解放军总后勤部|中国人民解放军总后勤部]]政治委员。

在担任[[中国人民解放军总后勤部|中国人民解放军总后勤部]]政治委员期间，刘源和总后勤部部长[[廖锡龙|廖锡龙]]等人以总后勤部党委的名义举报总后勤部副部长[[谷俊山|谷俊山]]的贪腐问题。谷俊山声称自己后面有人（[[徐才厚|徐才厚]]）。徐才厚也向刘源表示：“刘源你告谷俊山，还没准谷俊山把你整倒了呢？”<ref>{{cite web |title=刘源：徐才厚找我谈过，你告谷俊山，还没准谷俊山把你整倒了呢 |url=http://news.ifeng.com/a/20180819/59887304_0.shtml |website=凤凰网 |accessdate=2018-08-20}}</ref>刘源、[[廖锡龙|廖锡龙]]等人未受胁迫，坚持再次上报谷俊山的问题。2011年12月25日至28日，[[中国共产党中央军事委员会|中国共产党中央军事委员会]]在北京举行军委扩大会议，内容之一就是中共十八大的军队代表的人选问题。与会者达近百人，包括中央军委全体成员，以及各总部、各兵种、各军种、各大军区、和各省军区的主要负责人。到了讨论阶段，刘源上将则向与会者展示一张军方高级军官在北京为自己建造“将军府”的照片，将[[谷俊山|谷俊山]]问题直接当众公布。随后他称“贪污军产、倒卖军火、卖官鬻爵、侵吞公款、私用公款、乱吃回扣等”问题普遍存在于军方，并以开国大将[[陈赓|陈赓]]之子[[陈知建|陈知建]]少将被索贿之事为例，公然把矛头指向徐才厚、[[郭伯雄|郭伯雄]]和[[梁光烈|梁光烈]]三人<ref>{{cite web|title =上将刘源扳倒徐才厚内幕 军委扩大会上直接开炮|url =http://www.823u.com/html/4-49/49445.htm|accessdate =2015-07-28|website =红潮网}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

在[[中共中央|中共中央]]的支持下，2012年2月，谷俊山不再担任总后勤部副部长。然而军方内部仍然出现四次针对刘源的暗杀行动<ref>{{cite web |title = 徐才厚涉嫌受贿犯罪案终结 揭秘徐才厚四次暗杀刘源细节 |url = http://war.chinairn.com/news/20141029/09002788-2.shtml |accessdate = 2015-07-28 |website = 中研网|archiveurl=https://web.archive.org/web/20150802000412/http://war.chinairn.com/news/20141029/09002788-2.shtml |archivedate=2015-08-02}}</ref>，分别发生于2012年3月刘源座车在京津高速公路油缸燃烧，司机死亡，警卫为保护刘源烧伤。同年7月，刘源在青岛度假，房间被扔燃烧弹。9月刘源在西安检查工作被打冷枪，凶手被警卫击毙。9月下旬，在济南检查工作，所住军区招待所402房的浴室煤气爆炸<ref>{{cite web |title = 徐才厚死啦 徐才厚生前曾密谋四次暗杀刘源均失败 |url = http://www.xilu.com/news/xucaihouanshaliuyuan456.html |accessdate = 2015-07-28 |website = 西陆网}}</ref>。2012年11月[[中共十八大|中共十八大]]以后，[[习近平|习近平]]接任中共中央总书记和中央军委主席主政军政，刘源力谏进行军队反腐，他也是[[中共十八大以来的反腐败工作|习近平反腐]]的主要军界支持人之一<ref>{{cite web |title = 刘源：抓徐才厚、谷俊山是习近平决定、督办的 |url = http://military.china.com/important/11132797/20150313/19377576.html |accessdate = 2015-07-28 |website = 中华网}}</ref>。2014年3月，谷俊山涉嫌贪污、受贿、挪用公款、滥用职权犯罪案，由[[中国人民解放军军事检察院|中国人民解放军军事检察院]]于[[中国人民解放军军事法院|中国人民解放军军事法院]]提起公诉<ref>{{cite web|title =罗援：刘源和总后勤部党委一班人举报谷俊山贪腐|url =http://money.163.com/14/1110/10/AAMDCHJ900253B0H.html|accessdate =2015-07-28|website =网易|deadurl =yes|archiveurl =https://web.archive.org/web/20150309054923/http://money.163.com/14/1110/10/AAMDCHJ900253B0H.html|archivedate =2015-03-09}}</ref>。2014年10月27日，军事检察院对徐才厚涉嫌受贿犯罪案侦查终结，移送审查起诉。

2015年12月31日下午，国防部新闻发言人[[杨宇军|杨宇军]]表示，刘源任正大军区职期满，目前已免职，从领导岗位退下<ref>{{cite web |title = 国防部："打虎干将"刘源任职期满 目前已免职 |url = http://finance.sina.com.cn/roll/2015-12-31/doc-ifxneept3464219.shtml |accessdate = 2015-12-31 |website = 新浪}}</ref>。

=== 退居二线 ===
2016年2月，任[[第十二届全国人民代表大会各专门委员会|第十二届全国人大财政经济委员会]]副主任委员<ref>{{cite news |title = 原总后勤部政委、刘少奇之子刘源任全国人大财经委副主任委员 |url = http://www.thepaper.cn/newsDetail_forward_1436461 |accessdate = 2016-02-26 |publisher = 澎湃新闻}}</ref>。2018年3月，继续担任[[第十三届全国人民代表大会各专门委员会|第十三届全国人大财政经济委员会]]副主任委员。在第十三届全国人大一次会议选举[[国家监察委员会|国家监察委员会]]主任时，有两名全国人大代表选举刘源<ref>{{cite web |author1=游润恬 |title=两名人大代表选举刘源担任国家监察委主任 |url=http://www.zaobao.com/realtime/china/story20180318-843663 |website=联合早报 |accessdate=2018-06-19 |date=2018-03-18}}</ref>。
