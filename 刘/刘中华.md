'''刘中华'''（{{bd|1917年|1月|2018年|1月16日}}）[[山东|山东]][[文登|文登]]人，[[中国人民解放军开国少将|中国人民解放军开国少将]]。

==生平==
1917年1月生于文登天福山西字城村（今属文登营镇），1932年加入[[中国共产党|中国共产党]]。1935年，任[[中国共产主义青年团|共青团]]文登县委书记，不久又被挑选成为[[共青团中央|共青团中央]]驻[[青岛|青岛]]的地下交通员。1935年5月，因叛徒告密而被捕。直到[[抗日战争|抗日战争]]全面爆发后的1937年10月，[[第二次国共合作|第二次国共合作]]后，刘中华才获释<ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1953148 |title =将星陨落：百岁开国少将、原海军高级专科学校校长刘中华逝世  |author =  |date = 2018-01-16 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。

1937年12月24日，刘中华参加天福山起义，任[[山东人民抗日救国军第三军|山东人民抗日救国军第三军]]第一大队第三中队指导员。在起义第五天，刘中华被[[中国国民党|中国国民党]]方面逮捕，在狱中他既不接受改编，也不怕杀头，国民党方面迫于压力只好将他释放<ref name=pp/>。

1942年，[[日军|日军]]在山东发动大扫荡，[[八路军|八路军]][[115师|一一五师]]代理师长[[陈光|陈光]]、政委[[罗荣桓|罗荣桓]]率部在[[沂蒙山|沂蒙山]]区遭日军“围剿”，刘中华率加强营突破沂水河、沐水河封锁线，进入鲁中区牵制敌人，使一一五师师部和[[中共中央山东分局|中共中央山东分局]]顺利转移。[[第二次国共内战|第二次国共内战]]期间，刘中华率部打出山东，一直进至江南<ref name=pp/>。

1949年，刘中华开始在[[中国人民解放军海军|中国人民解放军海军]]工作，并于1950年被任命为第六舰队副政委。[[朝鲜战争|朝鲜战争]]爆发后，刘中华任舰队司令员兼政委。1952年，被派往[[苏联|苏联]][[伏罗希洛夫海军学院|伏罗希洛夫海军学院]]学习。1957年回国后，任[[中国人民解放军海军高级专科学校|中国人民解放军海军高级专科学校]]第一任校长。经过几年发展，该校开设了舰艇长系、舰艇政治系、导弹艇长系等6个系，培养出不少中国人民解放军海军指挥员<ref name=pp/>。

1955年被授予少将军衔<ref name=pp/><ref>{{cite news|title=揭秘：中共首批开国将军的级别是怎样划分的？|url=http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml|accessdate=2012-06-15|newspaper=凤凰网（转自摘自《党史博览》2010年第5期）|date=2010-06-13|deadurl=yes|archiveurl=https://web.archive.org/web/20120620091504/http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml|archivedate=2012-06-20}}</ref>，1988年被授予二级[[红星功勋荣誉章|红星功勋荣誉章]]<ref name=pp/>。

2018年1月16日，刘中华在[[青岛|青岛]]病逝，享年101岁<ref name=pp/>。
