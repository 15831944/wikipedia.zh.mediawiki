== 简介 ==
1983年，刘闻雯出生于[[湖北省|湖北省]][[武汉市|武汉市]]，在[[武汉科技大学|武汉科技大学]]读完大学课程。

2006年，刘闻雯参加[[世界杯|世界杯]]CCTV足球宝贝，排名前十。

2011年5月份，刘闻雯出版了自己的专著《变男》、《十八层的猫》。<ref>{{cite web |url=http://book.sina.com.cn/news/b/2011-07-04/1417288100.shtml |title="中国第一足球宝贝"刘闻雯武汉签售《变男》 |accessdate=  2011年7月4日 |publisher=新浪网 |work= |language =zh-cn}}</ref> 2011年12月，刘闻雯加入[[湖北省作家家协会|湖北省作家家协会]]。

2012年5月11日，刘闻雯出席电视节目时在答题环节回答[[李白|李白]]《[[将进酒|将进酒]]》“君不见黄河之水天上来”，她略加思索回答“恰似一江春水向东流”，主持人随后对其提示，但是其仍然没有回答出正确答案，节目在观众的叹息和质疑中结束。<ref>{{cite web |url=http://culture.ifeng.com/8/detail_2012_05/13/14491098_0.shtml?_from_ralated |title=80后女作家对错李白名篇遭轰 作协：不会因此开除 |accessdate=2012年5月13日 |publisher=凤凰网 |work= |language =zh-cn}}</ref> 随后，网友质疑[[中国作家协会|中国作家协会]]“现在凭一张脸就能进作协了？”<ref>{{cite web |url=http://yue.ifeng.com/news/detail_2012_05/13/14492664_0.shtml |title=武汉美女作家刘闻雯 作协会员身份遭网友质疑 |accessdate=2012年5月13日  |publisher=凤凰网 |work= |language =zh-cn}}</ref> 并认为刘闻雯是“冒牌作家”、“花瓶作家”、“伪作家”，<ref name="duck " >{{cite web |url=http://ws.chinadaily.com.cn/2012/0514/54577.shtml |title=女作家刘闻雯代笔门水落石出 张一一公布调查结果 |accessdate=2012年5月14日 |publisher=中国日报 |work= |language =zh-cn}}</ref> 面对质疑，刘闻雯痛骂道：“放你全家的屁。”<ref>{{cite web |url=http://culture.ifeng.com/8/detail_2012_05/14/14506754_0.shtml |title=美女作家不会李白名句遭疑 面对质疑“爆粗口” |accessdate=2012年5月14日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>
