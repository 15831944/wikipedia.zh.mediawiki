'''刘北宪'''（{{bd|1955年|1月||}}）[[湖南|湖南]][[安乡|安乡]]人，[[中华人民共和国|中华人民共和国]]官员<ref name=ifeng/>。

== 生平 ==
1972年9月至1974年6月，在[[昌平|昌平]]南邵公社[[插队落户|插队]]。1974年6月至1979年1月，担任北京新华印刷厂工人。1979年1月至1983年3月，就读于[[北京大学|北京大学]]一分校中文系。毕业后进入[[中国新闻社|中国新闻社]]，历任新闻部编辑，总编室、新闻部副主任、主任。1992年6月加入[[中国共产党|中国共产党]]<ref>{{Cite web|url=http://news.cngold.com.cn/20170817d1903n170265792.html|title=刘北宪涉严重违纪 曾任中国新闻社党委书记、社长|date=2017-08-17|publisher=中金网}}</ref><ref name=ifeng/>。1994年6月至1996年12月，中国新闻社副总编辑。1997年1月至2000年2月，中国新闻社香港分社社长。2000年2月至2004年9月，升任中国新闻社副社长兼副总编辑。2004年9月至2007年2月，担任中国新闻社常务副社长兼副总编辑、社委会委员<ref>{{Cite web|url=http://china.qianlong.com/2017/0817/1951018.shtml|title=中国新闻社原党委书记、社长刘北宪接受组织审查|date=2017-08-17|publisher=千龙网·中国首都网}}</ref><ref name=ifeng/>。

2007年2月至2009年2月，担任中国新闻社总编辑、常务副社长、社委会委员。2009年2月至2015年2月，[[中国新闻社|中国新闻社]]社长、党委书记、社委会委员<ref>{{Cite web|url=http://news.163.com/10/0517/13/66SUJ0L3000146BD.html|title=中德媒体论坛嘉宾：中国新闻社社长刘北宪|date=2010-05-17|publisher=网易}}</ref><ref name=ifeng/>。2015年2月，退休<ref>{{Cite web|url=http://ent.sina.com.cn/s/p/2010-12-21/23033184564.shtml|title=图文：BQ2010红人榜现场-刘北宪|date=2010-12-21|publisher=新浪网}}</ref><ref name=ifeng/>。

2017年8月16日，[[中央纪委|中央纪委]]驻中央统战部纪检组发布消息说，中国新闻社原党委书记、社长刘北宪涉嫌严重违纪，目前正接受组织审查<ref>{{Cite web |url =https://news.qq.com/a/20170816/042715.htm|title = 中国新闻社原党委书记、社长刘北宪接受组织审查 |author =  |date = 2017-08-16 |publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref>。2017年11月13日，据驻中央统战部纪检组消息，日前中央纪委驻中央统战部纪检组对中国新闻社原党委书记、社长刘北宪进行了立案审查，“经查，刘北宪违反政治纪律，对抗组织审查；违反[[中央八项规定|中央八项规定]]精神，多次违规出差乘坐飞机头等舱、收取礼金等；严重违反组织纪律，滥用职权，为[[民营企业|民营企业]]谋取巨额利益，造成重大国有资产流失；严重违反廉洁纪律，收受他人钱款、贵重物品。”“依据《中国共产党纪律处分条例》等有关规定，给予刘北宪开除党籍处分；依据《事业单位工作人员处分暂行规定》等有关规定，取消其退休待遇；将其涉嫌犯罪问题、线索移送司法机关依法办理。”<ref name=ifeng>{{Cite web |url =http://news.ifeng.com/a/20171113/53210799_0.shtml  |title =  中国新闻社原党委书记、社长刘北宪被开除党籍|author =  |date =2017-11-13  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>
