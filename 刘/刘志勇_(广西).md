'''刘志勇'''{{BD|1955年|10月}}，[[黑龙江|黑龙江]][[桦川|桦川]]人，[[满族|满族]]，中华人民共和国官员。

== 经历 ==
1971年12月参加工作，1976年12月加入中国共产党。[[广西大学|广西大学]]机械系机械工艺及设备专业毕业，大学工学学士，高级工程师。

历任广西东南矿机电厂修理工人、广西桂林冶金机械厂工人、广西桂林冶金机械总厂厂长兼桂林林业机械厂厂长、[[桂林市|桂林市]]人民政府副秘书长，市经贸委主任、党组副书记、[[柳州地区|柳州地区]]行署副专员、广西壮族自治区经济贸易委员会副主任、党组成员兼广西壮族自治区安全生产监督管理局（自治区煤矿安全监察局、自治区安全生产委员会办公室）局长、中共[[百色市|百色市]]委副书记、市长、[[广西农垦局|广西农垦局]]工委书记、广西壮族自治区农垦局局长，广西农垦集团有限责任公司董事长、总经理。2009年1月任中共[[梧州市|梧州市]]委书记，同年2月兼任梧州市人大常委会主任。2014年1月任广西壮族自治区政协副主席、党组成员<ref>{{cite web |url=http://gbzl.people.com.cn/grzy.php?id=131100133 |title=中国党政领导干部资料库：刘志勇 |publisher=中国共产党新闻网 |date= |accessdate=2016年1月30日 |archive-url=https://web.archive.org/web/20150929120822/http://gbzl.people.com.cn/grzy.php?id=131100133 |archive-date=2015-09-29 |dead-url=yes }}</ref>。

2016年1月29日，中纪委通报其严重违纪，受到撤销党内职务处分，降为正厅级非领导职务<ref name="重大职务调整">{{cite web |url = http://www.ccdi.gov.cn/xwtt/201601/t20160129_73665.html |title = 中央纪委2015年给予10名中管干部党纪重处分并作出重大职务调整 |accessdate = 2016年1月29日 |publisher = 中央纪委监察部网站}}</ref>。
