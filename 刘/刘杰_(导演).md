'''刘杰'''（{{bd|1968年|2月18日||}}），[[天津|天津]]人，中国[[独立电影|独立电影]]导演、编剧、摄影师、制片人。

== 生平 ==
1968年2月18日，刘杰在天津出生，父母均为农业大学教授。幼年曾随父母下放农村劳动。1975年在[[河北省|河北省]][[保定市|保定市]]上小学和中学，热爱绘画。1986年高中毕业报考美术学院时，偶然看了电影《[[黄土地_(电影)|黄土地]]》，遂放弃美术理想，1987年就读于[[北京电影学院|北京电影学院]]摄影系，1991年凭[[文学学士|文学学士]]学位毕业。1992年，担任剧情片《[[冬春的日子|冬春的日子]]》制片人、摄影师。1999年，担任《[[梦幻田园|梦幻田园]]》摄影指导。2001年，凭摄影作品《[[十七岁的单车|十七岁的单车]]》获得[[第51屆柏林影展|第51屆柏林影展]]评审团大奖和[[第38届金马奖|第38届金马奖]][[金马奖最佳摄影|最佳摄影奖]]提名<ref>{{cite journal |author1=刘杰 |title=请为艺术片留一个影厅 |journal=电影 |volume=2011年07期}}</ref>。

2002年7月10日，摄影作品《[[王首先的夏天|王首先的夏天]]》上映。2006年9月，长片处女作《[[马背上的法庭|马背上的法庭]]》获[[第63届威尼斯国际电影节|第63届威尼斯国际电影节]]地平线单元最佳影片奖<ref>{{cite web |title=《马背上的法庭》导演刘杰上台领奖 |url=http://ent.163.com/06/0910/11/2QLFCIO5000320TE.html |website=网易 |accessdate=2018-09-30 |date=2006-09-10}}</ref><ref>{{cite web |title=第13届上海国际电影节昨晚闭幕 刘杰获最佳导演 |url=http://yule.sohu.com/20100621/n272950858.shtml |website=搜狐 |accessdate=2018-09-30 |date=2010-06-21}}</ref>。2003年9月5日，担任执行制片人的[[王小帅|王小帅]]导演作品《[[二弟|二弟]]》上映。2007年8月担任[[第64届威尼斯国际电影节|第64届威尼斯国际电影节]]“未来金狮”单元评委<ref>{{cite web |title=威尼斯调整评委阵容 刘杰转入“未来金狮”单元 |url=http://news.mtime.com/2007/08/23/523576.html |website=Mtime时光网 |accessdate=2018-09-30 |date=2007-08-23}}</ref>。2010年6月20日，凭导演作品《[[碧罗雪山_(电影)|碧罗雪山]]》获第13届[[上海国际电影节|上海国际电影节]]最佳导演奖和评委会大奖<ref>{{cite web |title=上海电影节颁奖 《碧罗雪山》刘杰获最佳导演 |url=http://ent.163.com/10/0621/00/69LNKPF100032DGD_all.html |website=网易 |accessdate=2018-09-30 |date=2010-06-21}}</ref>。同年11月20日，凭与高山合作的电影《[[透析_(电影)|透析]]》获[[第47届金马奖|第47届金马奖]][[金马奖最佳原著剧本|最佳原著剧本奖]]<ref>{{cite web |title=图文：金马奖后台-刘杰高山获最佳原著剧本奖 |url=http://ent.sina.com.cn/m/p/2010-11-20/22153152765.shtml |website=新浪 |accessdate=2018-09-30 |date=2010-11-20}}</ref>。2012年执导青春电影《[[青春派|青春派]]》，并凭此获第16届上海国际电影节最佳影片<ref>{{cite web |title=传媒大奖下双黄蛋 《青春派》《小时代》获最佳影片 |url=http://ent.ifeng.com/movie/special/16thsiff/tvnew/detail_2013_06/23/26703497_0.shtml |website=凤凰网 |accessdate=2018-09-30 |date=2013-06-23}}</ref>。2015年执导电影《[[德兰_(电影)|德兰]]》，该片获第19届上海国际电影节金爵奖最佳影片奖<ref>{{cite web |title=第19届上海国际电影节闭幕 《德兰》获最佳影片 刘烨凭《追凶者也》加冕影帝 |url=http://news.mtime.com/2016/06/19/1556487.html |website=Mtime时光网 |accessdate=2018-09-30 |date=2016-06-19}}</ref>。同年11月4日，执导作品《[[捉迷藏_(2016年电影)|捉迷藏]]》上映<ref>{{cite web |title=《捉迷藏》今日上映 导演刘杰“告白”观众 |url=http://ent.ifeng.com/a/20161104/42713481_0.shtml |website=凤凰网 |accessdate=2018-09-30 |date=2016-11-04}}</ref>。

2018年10月19日，执导的[[失孤|失孤]]剧情片《[[宝贝儿|宝贝儿]]》上映<ref>{{cite web |title=杨幂扮丑首演文艺片《宝贝儿》定档10.19 侯孝贤监制 关注缺陷弃儿的"不完美生命" |url=http://news.mtime.com/2018/08/13/1583267.html |website=Mtime时光网 |accessdate=2018-09-30 |date=2018-08-13}}</ref>。该片由[[杨幂|杨幂]]主演<ref>{{cite web |title=杨幂郭京飞联袂主演 刘杰微博透露《宝贝》阵容 |url=http://ent.sina.com.cn/zz/2018-03-07/doc-ifxshkpn0657920.shtml |website=新浪 |accessdate=2018-09-30 |date=2018-03-07}}</ref>，入围[[2018年多倫多國際影展|第43届多伦多国际电影节]]主竞赛单元<ref>{{cite web |title=圣塞巴斯蒂安电影节入围片单揭晓 |url=http://www.artsbj.com/show-21-584500-1.html |website=北京文艺网 |accessdate=2018-09-30 |date=2018-08-18}}</ref>。
