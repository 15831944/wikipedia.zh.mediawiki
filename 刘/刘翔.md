'''刘翔'''{{BD|1983年|7月13日||}}，出生於[[中国|中国]][[上海|上海]]<ref>https://web.archive.org/web/20080810122643/http://results.beijing2008.cn/WRM/CHI/BIO/Athlete/1/236181.shtml 运动员资料：刘翔</ref>，祖籍[[江苏省|江苏]][[盐城市|盐城]][[大丰市|大丰]][[草堰镇|草堰镇]]西河村<ref>http://data.ent.163.com/ent/star/one/id=00030BOeBEUQ.html 网易娱乐：刘翔</ref>，就读于[[华东师范大学|华东师范大学]]，著名[[田径|田径]][[运动员|运动员]]。曾获一枚[[奥运会|奥运会]]金牌、六枚[[世界田径锦标赛|世锦赛]]奖牌和三枚[[亚运会|亚运会]]金牌，是两次世界冠军，并曾打破世界纪录，是[[110米栏|110米栏]]史上第一位同时集奥运会冠军、世锦赛冠军、世界纪录於一身的选手，同时也是中国[[全运会|全运会]]史上第一个三连冠田径选手，被称為「跨栏王」。政治上，刘翔是第十一届、十二届[[全国政协委员|全国政协委员]]、[[中国共产党|中国共产党]]党员及上海市体育局团委副书记<ref>{{cite news|title=曝刘翔已和[[葛天|葛天]]离婚 难道真的要和[[冬日娜|冬日娜]]在一起|url=http://www.chinadaily.com.cn/hqcj/xfly/2015-05-22/content_13732906.html|publisher=中国日报|date=2015年5月22日}}</ref>。

2015年4月7日，刘翔通过多个媒体渠道宣布退役。

== 业餘生涯 ==
刘翔七岁被所就讀的上海市管弄小学的校田径队教练[[仲锁贵|仲锁贵]]发掘，选进校田径队。1993年，刘翔被顾宝刚选入上海市普陀区少体校，主练跳高、辅练100米短跑等，开始了职业运动生涯。1995年，在一次市内的一百米短跑赛中，他被上海体育运动技术学院田径队二线队教练[[方水泉|方水泉]]教练看中，调他转去学习一百米跨栏。然而他的成绩却不如理想。按照惯例，表现不理想的学生，要回到普通学校学习。1997年，劉翔轉入上海市宜川中學。1998年2月，刘翔一家搬到了上海市普陀区海棠苑46号502室居住。1999年，在[[孙海平|孙海平]]的全力争取下，刘翔才回到了上海体育运动技术学院练习跨栏，并进一线队伍，这才开始刘翔的职业生涯。<ref>{{cite web |url=http://liuxiang.sports.cn/meet/career/2005-04-12/534901.html |title=存档副本 |accessdate=2009-10-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20090916101659/http://liuxiang.sports.cn/meet/career/2005-04-12/534901.html |archivedate=2009-09-16 }}</ref><ref>[http://sports.sina.com.cn/o/2015-04-08/09237567913.shtml 刘翔的上海足迹：童年旧居海棠苑 宜川中学结缘体育].新浪網.</ref>

== 职业生涯 ==
===奥运冠军===
2002年，刘翔在国际田联大奖赛中以13.12秒打破了110米栏[[世界青年纪录|世界青年纪录]]及亚洲纪录。同年， [[2002年亚洲运动会|釜山亚运会]]中他以13.27秒的成绩夺得[[金牌|金牌]]。次年，他在[[世界田径锦标赛|世界田径锦标赛]]男子110米栏中获得[[铜牌|铜牌]]。

2004年8月27日，在[[2004年夏季奥林匹克运动会|雅典奥运会]][[男子110米栏|男子110米栏]]项目的决赛中，刘翔完美地表现了自己的技术，以接近三米的优势率先冲过终点，最终以12.91秒的成绩夺得金牌，打破了奥运会记录，并追平了[[科林·杰克逊|科林·杰克逊]]在1993年创造并保持11年的世界纪录，为中国夺得了首面男子田径奥运会金牌。<ref>http://2004.sina.com.cn/cn/at/2004-08-28/0247104394.html 刘翔平世界纪录加冕跨栏王</ref>
赛后，[[2004年夏季奥林匹克运动会中国代表团|中国体育代表团]]团长[[袁伟民|袁伟民]]评价这块金牌为中国代表团在雅典奥运会上夺得「含金量最重」的金牌，刘翔也因此成為了该届奥运会闭幕典礼中国的持旗手。

在雅典奥运会上，刘翔是唯一一位能够在田径赛道上击败[[美洲|美洲]]和[[欧洲|欧洲]]选手的亚洲男子运动员。

2005年，刘翔成為[[十运会|全国运动会]]上海队的持旗手、[[2005年东亚运动会|东亚运动会]]中国队的持旗手。2006年，刘翔夺得[[中国中央电视台|中国中央电视台]]主办的“CCTV体坛风云人物”评选活动颁奖盛典的年度最佳运动员奖<ref>http://sports.people.com.cn/BIG5/22155/22168/23137/44534/80251/5579119.html CCTV体坛风云人物颁奖盛典 刘翔荣膺两项大奖</ref>，并在中国民众心中成为与[[姚明|姚明]]齐名的明星运动员。

===打破世界纪录===
2006年7月11日，刘翔在[[国际田联超级大奖赛|国际田联超级大奖赛]][[洛桑|洛桑]]站男子110米栏决赛中，以12秒88打破了已封尘13年之久的[[世界纪录|世界纪录]]。比赛中，刘翔起跑一开始就已经一马当先，直至撞线的最后一步一直领先。刘翔赛后表示自己没想过会把世界纪录打破。<ref>http://news.xinhuanet.com/sports/2006-07/12/content_4819540.htm 刘翔打破世界纪录</ref>[[美国|美国]]名将[[阿诺德|阿诺德]]亦在该次比赛中跑出12秒90破原有世界纪录的成绩。而古巴选手[[戴龙·罗伯斯|戴龙·罗伯斯]]则在这次比赛中跑出13秒04的成绩，是他当时最佳的成绩。<ref>http://www.henau.edu.cn/news/Article_Show.asp?ArticleID=2790</ref>

2007年8月31日，刘翔在[[大阪|大阪]][[世界田径锦标赛|世界田径锦标赛]]男子110米栏决赛上克服了被分在第9跑道的不利因素，凭借强劲的实力外加个人良好的心理素质最终以12秒95的成绩获得冠军，成为将该项目的奥运冠军、世界冠军和世界纪录保持者集一身的大满贯得主。<ref>http://news.hexun.com/2007-09-01/100411663.html 刘翔12秒95夺冠大阪世锦赛 圆110米栏大满贯</ref>

2008年3月9日，在[[西班牙|西班牙]][[瓦伦西亚|瓦伦西亚]]举行的第12届[[世界室内田径锦标赛|世界室内田径锦标赛]]60米栏决赛中，刘翔以7秒46夺得冠军。这是刘翔的第一个个人室内赛世界冠军头衔，也是中国室内田径世锦赛历史上的首枚男子金牌。<ref>http://sports.qq.com/a/20080309/000070.htm 刘翔首夺室内世界冠军 </ref>

===北京奥运会因伤退赛===
{{Wikinews|刘翔预赛因伤退赛 告别北京奥运会}}
2008年[[北京奥运会|北京奥运会]]男子110米栏第一轮比赛於2008年8月18日上午进行，刘翔被分配在第六组第二道出发。但由于伤势复发，刘翔最后临场退出比赛，卫冕失败。该项目的金牌最终由戴龙·罗伯斯夺得。

刘翔曾表示：「如果我不是万不得已，我不会做出退出的决定。」
刘翔在接受中央电视台採访时表示：「我是不会轻易言败的，这是我的性格。我知道我有实力，只要我的脚好了，因為我现在觉得各方面都很好，就是这个脚不行，也不说什麼吧，我觉得总有一天我会好起来的。我觉得太多人為我操心，有这麼多人来关心我，来支持我。我感觉我非常抱歉，因為我也是实在没办法了，做準备活动的时候，我就觉得我的脚不行了。但是今天，我也不知道怎麼搞的，怎麼会这样，我也不想这样，实在是受不了。如果挺下来的话，说不定我的跟腱……真的挺不下来了！我也想顶下来，所以说，那个时候真的是无法形容。」<ref>{{cite web |url=http://www.takungpao.com/news/08/08/20/WF-949245.htm |title=存档副本 |accessdate=2008-08-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20080823110922/http://www.takungpao.com/news/08/08/20/WF-949245.htm |archivedate=2008-08-23 }} 刘翔不言弃: 要证明我还是行</ref>

=== 复出 ===
教练[[孙海平|孙海平]]在2008年9月19日表示，有两种治疗刘翔伤势的方案：一个是保守治疗，另一个是动[[手术|手术]]。现在倾向选择採用保守治疗的办法，目前刘翔的伤已经康复了七八成，跑到13秒内都没有问题，预计刘翔可於2009年6月复出<ref>[http://news.wenweipo.com/2008/09/20/IN0809200002.htm 孙海平：刘翔明年6月定能复出]</ref>。其后，刘翔於2009年9月10日在国家田径队内的测试赛跑出13秒70的成绩，反映出刘翔的伤患已大致恢复，同日，官方宣佈刘翔会於9月20日举行的[[上海国际田径黄金大奖赛|上海国际田径黄金大奖赛]]正式复出<ref>[http://sports.sohu.com/20090911/n266664266.shtml 08后首次“正式比赛” 刘翔13秒70宣告即将复出]</ref><ref>[http://sports.sohu.com/20090912/n266675380.shtml 官方宣布刘翔上海赛复出 翔飞人首战PK特拉梅尔]</ref>。比赛的结果是刘翔与[[特拉梅尔|特拉梅尔]]均跑出了13秒15，后来通过[[电子仪器|电子仪器]]辨别，显示为特拉梅尔有效部位先过终点，至此刘翔获得[[亚军|亚军]]。<ref>[http://sports.sina.com.cn/o/2009-09-20/22064597429.shtml 上海大奖赛刘翔完美复出 13秒15同成绩屈居亚军]</ref>

2009年10月24日晚，在[[十一运会|十一运会]]110米栏预赛上，刘翔以13秒75小组第二晋级；10月25日晚的决赛中，刘翔以13秒34夺冠，也是他自伤癒复出后的首个冠军。这也使他成为首个[[全运会|全运会]]三连冠的选手<ref>[http://sports.sohu.com/20091025/n267720821.shtml 三连冠创全运历史 刘翔13秒34险胜摘金]</ref>。其后，他於[[广州|广州]]举行的[[亚洲田径锦标赛|亚洲田径锦标赛]]中以13秒51完成赛事，继全运会后再度成為三连冠<ref>[http://hk.sports.yahoo.com/091112/65/3kyu2.html 刘翔勇创亚锦赛三冠王]</ref>。在12月11日香港[[2009年东亚运动会田径比赛|东亚运动会]]的男子[[跨栏|110米跨栏]]决赛中以13秒66的成绩获得冠军。

2011年8月29日晚，在[[第13届世界田径锦标赛|第13届世界田径锦标赛]]110米栏决赛中，刘翔因为对手[[戴龙·罗伯斯|戴龙·罗伯斯]]的干扰，仅以13秒27的成绩获得铜牌，戴龙·罗伯斯以13秒14获得冠军，亚军是美国的[[杰森·理查德森|杰森·理查德森]]，成绩为13秒16。不过随后中国代表团进行申诉，国际田联取消了戴龙·罗伯斯的成绩，杰森·理查德森因此获得冠军，刘翔的铜牌也变为银牌，原来的第四名英国的[[安德鲁·特纳|安德鲁·特纳]]则递补为季军，成绩是13秒44。

2012年5月19日晚，在国际田联钻石联赛上海站110米栏决赛中，刘翔以12秒97获得冠军，这是他2008年之后第一次跑进13秒，同时创造了当时的当年世界最好成绩。

2012年6月3日，在国际田联钻石联赛尤金站110米栏决赛中，刘翔以12秒87夺冠军，因风速超过标准，记录不被承认。

===2012伦敦奥运比赛===
2012年8月7日，[[2012年夏季奥林匹克运动会|2012年夏季奥林匹克运动会]]男子田径比赛[[2012年夏季奥林匹克运动会田径比赛－男子110公尺跨栏|110公尺跨栏]]预赛第六组，刘翔在第一栏意外摔倒，未能进入準决赛。赛后被诊断為[[跟腱|跟腱]]断裂。

刘翔在比赛前一直受到伤患问题的困扰，情况反复。在进驻[[奥运村|奥运村]]后，刘翔曾接受中国代表团医生的会诊和治疗，并放弃了原来计划的训练，改以休息和治疗<ref>http://2008.sohu.com/20080818/n258973170.shtml 刘翔因伤退赛早有先兆</ref> 。

=== 伤患困扰 ===
教练孙海平表示，刘翔主要有两处的伤患，一是在[[大腿|大腿]]，二是在[[脚跟|脚跟]]。大腿是2008年上半年受伤的，经过治疗后，已经康复。另一问题在脚跟处，是旧患，在雅典奥运会前，就已经伤了，情况一直反复。每当进行高强度训练后，他就会感到疼痛。<ref>http://www.chinareviewnews.com/doc/1007/2/5/3/100725373.html?coluid=7&kindid=0&docid=100725373&mdate=0818142710 刘翔為何退出比赛　冯树勇孙海平解答实录</ref>核磁共振结果显示，他的伤实际上是附著在跟骨及跟腱的[[末梢|末梢]]有问题。刘翔两个[[脚跟骨|脚跟骨]]比一般人突出，容易磨损，形成一个泡，挑破以后，又形成硬茧。
<ref>http://2008.sohu.com/20080818/n258973170.shtml 刘翔因伤退赛早有先兆 脚跟腱旧伤复发梦断奥运</ref>

在2005年底，刘翔曾不慎扭伤了脚。经过几个月的休养后，就在2006年7月打破了世界纪录，创下了12秒88的好成绩。<ref>http://news.xinhuanet.com/olympics/2008-08/16/content_9379697.htm 揭秘刘翔脚伤成最大担心</ref>
2008年5月，刘翔再次弄伤跟腱。可是，这次受伤并不严重，但出於安全的原因，所以孙海平决定让刘翔退出2008年[[纽约大奖赛|纽约大奖赛]]。而几天后的[[尤金大奖赛|尤金大奖赛]]中，刘翔则因偷步被罚出场，之后就没有参加过任何正式的比赛。
<ref>http://www.chinareviewnews.com/doc/1007/2/4/9/100724901.html?coluid=119&kindid=0&docid=100724901&mdate=0818083518 今日看点：刘翔出战</ref>刘翔回到北京，接受治疗。经过2个月的休养，刘翔的状况越来越好。<ref>http://2008.sohu.com/20080818/n258973170.shtml 脚跟腱旧伤复发梦断奥运</ref>但在北京奥运会前，刘翔於8月16日的训练中，伤势突然加重。每当刘翔训练用力后，便有强烈的酸痛，最终导致其再一次因伤退出。

===商业生涯===
2003年崭露头角的他开始积极参与各类商业活动，首年年收入即达到160万元人民币<ref>http://sports.sohu.com/20150406/n410859828.shtml</ref>。其后在2004年夺得雅典奥运会冠军之后，其收入一路攀升，在2007年顶峰时，签了14个代言，年收入达到了惊人的1.6亿元人民币，并藉此跻身[[福布斯中国名人榜|福布斯中国名人榜]]三甲<ref>http://sports.sohu.com/20150406/n410859828.shtml</ref>。有人统计<ref>http://news.now.com/home/international/player?newsId=131860</ref>，在刘翔出道11年生涯中，在商业活动上总共收入逾5亿元人民币。
