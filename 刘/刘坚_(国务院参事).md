'''刘坚'''（{{bd|1944年|10月||}}）[[江苏|江苏]][[南通|南通]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中华人民共和国农业部|中华人民共和国农业部]]副部长、[[国务院扶贫开发领导小组办公室|国务院扶贫开发领导小组办公室]]主任。

==生平==
[[中国共产党|中国共产党]]党员，高级农艺师。1966年自[[苏北农学院|苏北农学院]]农学系农学专业毕业。同年参加工作，历任江苏省[[如皋县|如皋县]]农业局技术员、农业组长，江苏省如皋县农水办公室技术员，江苏省农林厅办公室秘书科办事员、副科长，江苏省农业技术推广中心主任，江苏省农林厅副厅长、党组成员，江苏省计经委副主任、党组成员，[[江苏省人民政府|江苏省人民政府]]副秘书长兼江苏省计经委副主任、党组成员，江苏省人民政府副秘书长、秘书长、党组成员，江苏省人民政府副省长，[[中共江苏省委|中共江苏省委]]委员。1999年1月任[[中华人民共和国农业部|中华人民共和国农业部]]副部长、党组成员。2003年“[[非典|非典]]”爆发时，兼任国务院“非典”防治领导小组农村组组长。2003年3月任[[国务院扶贫开发领导小组|国务院扶贫开发领导小组]]副组长、[[国务院扶贫开发领导小组办公室|国务院扶贫开发领导小组办公室]]主任、党组书记。2009年1月任[[国务院参事|国务院参事]]<ref>{{Cite web |url = http://wlb.yzu.edu.cn/art/2013/3/18/art_32567_448992.html |title =  共和国农业部长的铿锵人生——记刘坚校友|author =  |date = 2013-03-18 |publisher = 扬州大学校友会   |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.cs.com.cn/xwzx/03/201010/t20101013_2623043.html  |title =走近国务院参事：10个工作小组 多为经济决策咨询  |author =  |date = 2010-10-13 |publisher = 中证网 |language =  |accessdate =  |quote =  }}</ref>。2015年卸任国务院参事<ref>{{Cite web |url = http://www.gywb.cn/content/2015-05/28/content_3184578.htm |title =国务院参事室副主任方宁一行赴齐齐哈尔市调研  |author =  |date =2015-05-28 |publisher = 贵阳网 |language =  |accessdate =  |quote =  }}</ref>。
