'''刘烨'''（{{bd|1978年|3月23日|catIdx=L刘}}），中國大陸男[[演員|演員]]，出生于[[吉林省|吉林]][[长春市|长春]]，毕业于[[中央戏剧学院|中央戏剧学院]]。因主演《[[蓝宇|蓝宇]]》《[[美人草|美人草]]》獲得金马奖、金鸡奖最佳男主角。2015年帶兒女[[刘诺一|刘诺一]]和[[刘霓娜|刘霓娜]]參加親子節目《[[爸爸去哪兒第三季|爸爸去哪兒第三季]]》。2016年凭《[[追凶者也|追凶者也]]》获得[[上海电影节|上海电影节]]最佳男演员奖，繼[[鄧超|鄧超]]之后憑[[曹保平|曹保平]]電影獲[[金爵獎|金爵獎]]。

刘烨是[[中国国家话剧院|中国国家话剧院]]演员、[[中国电影家协会|中国电影家协会]]会员、[[吉林省青年联合会|吉林省青年联合会]]副主席，签约公司为北京星美千易演艺文化发展有限公司。2013年北京法国领事馆授予[[法蘭西藝術與文學勳章|法蘭西藝術與文學騎士勳章]]。2014年获以色列旅游部长乌齐·兰多授予以色列旅游亲善大使。

== 个人生活 ==
刘烨和[[谢娜|谢娜]]曾经相恋六年，2006年年头两人分手。根据谢娜撰写的《娜写年华》一书所述，两人分手是因彼此都忙于工作，聚少离多造成了误会和隔阂<ref>{{cite web |url=http://ent.sina.com.cn/s/m/2008-12-23/07202308458.shtml |title=谢娜：与刘烨分手还是朋友 想对张杰说谢谢你 |publisher=新浪网，来源：四川在线-华西都市报 |date=2008年12月23日 |accessdate=2012-05-10|work= |language =zh-cn}}</ref>。双方均表示和平分手，谢娜结婚时，刘烨送出祝福。<ref>{{cite web |url=http://news.cntv.cn/20110510/107549.shtml |title= 刘烨祝旧爱谢娜领证结婚 拒让妻子探班激吻戏(图)|publisher=中国新闻网 |accessdate=2011年5月10日 |work= |language =zh-cn}}</ref>

2006年底，刘烨与[[法国|法国]]籍[[犹太人|犹太人]]安娜伊思·马田（Anais Martane）相识。安娜伊思是一位在中国生活多年的[[摄影师|摄影师]]，同时是法国《[[自由报|自由报]]》驻北京市记者<ref>{{cite web |url=http://news.fengone.com/c/20101016/130531.html|title=刘烨法国妻子安娜 资料背景照片档案曝光|publisher=凤网，来源：新华网|date=2010-10-16 |accessdate=2012-05-10|work= |language =zh-cn}}</ref>。2009年7月，两人在[[北京|北京]]结婚，2010年10月生下儿子“[[刘诺一|刘诺一]]”，2012年生了一个女儿，叫“刘霓娜”。<ref>{{cite web |url= http://news.cntv.cn/20120216/112237.shtml|title=刘烨女儿取名“刘霓娜” 前女友谢娜送祝福 |publisher=中国网络电视台 |accessdate= 2012年2月16日|work= |language =zh-cn}}</ref> 两人感情生活甜蜜，刘烨直言：「是安娜改变了我，给了我很多力量。」<ref>{{cite web |url=http://ent.qq.com/a/20120912/000728.htm |title= 封面人物》刘烨：媳妇躺在身边 才能睡着 |publisher=腾讯娱乐[微博] |accessdate=2012年9月12日 |work= |language =zh-cn}}</ref>
