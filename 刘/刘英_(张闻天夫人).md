'''刘英'''（{{bd|1905年|10月|2002年|8月26日}}）女，原名'''郑杰'''，[[湖南|湖南]][[长沙|长沙]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中央纪委|中央纪委]]委员，第五届全国政协常委，[[张闻天|张闻天]]夫人<ref name=shishi>{{Cite web |url =  http://www.people.com.cn/GB/paper39/7157/691757.html|title =刘英同志逝世  |author =  |date =  |publisher = 人民日报海外版2002年9月4日，第四版 要闻·社会 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
=== 民国时期 ===
1905年10月生于湖南长沙。1924年进入[[长沙女子师范学校|长沙女子师范学校]]学习，1925年在校期间加入[[中国共产党|中国共产党]]，任该校党支部书记。1927年她任[[中共湖南省委|中共湖南省委]]候补委员兼妇女部部长，与[[滕代远|滕代远]]在长沙近郊组织“灰日暴动”，但因泄密而失败，[[王一飞|王一飞]]身亡<ref>{{Cite book|title=《中华之魂》星辰谱 母亲的求舍|last=《中华之魂》编委会编|first=|publisher=北京：知识产权出版社|year=2007.06|isbn=978-7-80198-460-9|location=|pages=150}}</ref>。在[[白色恐怖|白色恐怖]]中她到[[上海|上海]]向[[中共中央|中共中央]]汇报，行前一周与将赴[[醴陵县|醴陵县]]任中共醴陵县委书记的[[林蔚|林蔚]]结婚。1928年3月刘英回长沙再赴上海时，得知林蔚已在醴陵牺牲。刘英在上海的中共湖南临时省委“住机关”一年多。1929年春，被[[中共中央|中共中央]]派到[[苏联|苏联]][[莫斯科东方大学|莫斯科共产主义劳动大学]]学习。1930年该大学停办，她于1930年底转入国际无线电学校学习。1932年底由[[共产国际|共产国际]]派回中国，在[[江西|江西]][[瑞金|瑞金]][[中央苏区|中央苏区]]任[[共青团中央|少共中央]]组织部秘书，中央苏区少共福建省委书记、福建省委常委，少共中央局宣传部部长，后来改任组织部部长。1934年10月，参加[[中国工农红军|中国工农红军]][[二万五千里长征|二万五千里长征]]。1935年4月，经[[毛泽东|毛泽东]]提议，由[[李富春|李富春]]经办，将刘英调到中央纵队接替[[邓小平|邓小平]]担任秘书长。当时毛泽东及其他中央领导有意撮合刘英与张闻天。1935年11月，刘英任陕北少共中央局宣传部长、组织部部长<ref name=shishi/><ref name=bainian/>。中共中央机关于1935年11月进驻[[瓦窑堡|瓦窑堡]]后，刘英与张闻天结婚<ref>{{Cite web |url = http://dangshi.people.com.cn/n/2013/0205/c85037-20440466-2.html |title =张闻天刘英结婚 毛泽东赋诗“闹”洞房  |author =  |date = 2013-02-05 |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。

1937年[[抗日战争|抗日战争]]全面爆发后，刘英调中共中央机关工作，历任[[延安|延安]]中央秘书处处长兼[[张闻天|张闻天]]机要、政治秘书，高级干部学习组分组长、[[中共中央办公厅|中共中央办公厅]]行政处学委会副主任。1945年10月，随张闻天到[[中国东北|中国东北]]开拓中共根据地，任中共[[宁安县|宁安县]]委副书记。1946年5月，任[[中共合江省委|中共合江省委]]常委、组织部部长。1948年5月，任中共[[哈尔滨市|哈尔滨市]]委常委、组织部长、妇女委员会书记。1949年5月，任[[中共辽东省委|中共辽东省委]]常委、组织部部长<ref name=shishi/>。

=== 共和国时期===
[[中华人民共和国|中华人民共和国]]成立后，1950年2月，刘英随张闻天离开中国东北抵达首都[[北京|北京]]，参加[[中国常驻联合国代表团|中国常驻联合国代表团]]的筹备，任[[经济社会理事会|经济社会理事会]]副代表。1951年4月到1954年11月，在中国驻苏联大使馆任使馆参赞、特委副书记、党总支书记。1954年12月到1959年，历任[[中华人民共和国外交部|中华人民共和国外交部]]部长助理，机关党委委员、常委，监委书记，人事司司长等职。1956年8月，任[[中共中央国家机关工作委员会|中央国家机关党委]]委员、常委<ref name=shishi/>。

1959年在[[反右倾运动|“反右倾”运动]]中，刘英受丈夫张闻天牵连而被撤销党内外职务。“[[文化大革命|文化大革命]]”期间，刘英和丈夫张闻天遭到迫害。1978年后，刘英先后当选为[[第五届全国政协委员|第五届全国政协委员]]、常委，第十一届、十二届[[中央纪委|中央纪委]]委员。她是[[中共七大|中共七大]]、[[中共八大|八大]]、[[中共十四大|十四大]]、[[中共十五大|十五大]]代表，[[第二届全国人大代表|第二届全国人大代表]]<ref name=shishi/>。

2002年8月26日，刘英在北京病逝，享年97岁<ref name=shishi/>。
