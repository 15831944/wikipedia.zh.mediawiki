'''刘劲'''{{BD|1963年|12月7日|||L}}，饰演[[周恩来|周恩来]]的[[特型演员|特型演员]]。现为[[中国人民解放军总政治部话剧团|中国人民解放军总政治部话剧团]]演员，[[一级演员|一级演员]]。

== 生平 ==

刘劲出生于[[四川省|四川省]][[阿坝藏族羌族自治州|阿坝藏族羌族自治州]]，毕业于[[中国人民解放军艺术学院|中国人民解放军艺术学院]]。 

他的成名作品是电视剧《[[周恩来在上海|周恩来在上海]]》，曾在多部影视剧饰演[[周恩来|周恩来]]。在电视剧《[[周恩来在重庆|周恩来在重庆]]》中，是他第34次扮演周恩来<ref>http://ent.sina.com.cn/v/m/2008-03-12/14151945505.shtml</ref>。

有报道称<ref>{{cite web |url=http://sztqb.sznews.com/html/2009-03/18/content_552790.htm |title=存档副本 |accessdate=2009-11-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20100108181717/http://sztqb.sznews.com/html/2009-03/18/content_552790.htm |archivedate=2010-01-08 }}</ref>刘劲曾因电影《[[宋家皇朝|宋家皇朝]]》中在饰演[[张学良|张学良]]，而获[[香港电影金像奖|香港电影金像奖]]。实际上是在《宋家皇朝》中饰演[[宋查理|宋查理]]的[[姜文|姜文]]，获1998年[[香港电影金像奖|香港电影金像奖]]最佳男配角奖<ref>http://yule.sohu.com/20090414/n263385092.shtml</ref>。

== 获奖 ==
*获中国首届戏剧小品电视比赛一等奖，作品《芙蓉树下》
*[[中国中央电视台|中国中央电视台]]2004年度“十佳”男演员奖
*第25届[[中国电视金鹰奖|中国电视金鹰奖]]“最受观众喜爱的男演员”奖
*第5届[[中国电视金鹰艺术节|中国电视金鹰艺术节]]“最具人气的男演员”奖
*电影《长征》获得中国电影[[百花奖|百花奖]]“最佳男配角”奖提名
*2006年被评为中宣部第二批“四个一批人才”
