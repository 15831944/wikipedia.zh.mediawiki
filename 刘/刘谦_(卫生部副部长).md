'''刘谦'''（{{bd|1956年|2月||}}）生于[[北京|北京]]，籍贯[[云南|云南]][[昆明|昆明]]，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中华人民共和国卫生部|中华人民共和国卫生部]]副部长、[[中华人民共和国国家卫生和计划生育委员会|中华人民共和国国家卫生和计划生育委员会]]副主任<ref name=elys>{{Cite web |url =  http://www.moh.gov.cn/liuqian/pgrjlb/201305/5e112cb3ab3e4491a98721d390e46a1d.shtml|title =刘谦  |author =  |date =2013-05-02   |publisher = 中华人民共和国国家卫生和计划生育委员会 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1981年4月加入[[中国共产党|中国共产党]]，1974年1月参加工作，大学本科学历，研究员。1978年9月到1983年8月，在[[山西医学院|山西医学院]]医疗系学习。1984年4月到1995年8月，历任卫生部科技教育司科员、主任科员、副处长、处长。其间，1993年2月到5月在[[美国|美国]][[波士顿大学|波士顿大学]]医学院学习、1993年5月到1994年5月，在美国[[麻省理工学院|麻省理工学院]]生物技术工程中心做访问学者。1995年9月到1999年2月，历任[[中华人民共和国科学技术部|科技部]]（原为[[国家科委|国家科委]]）中国生物工程开发中心主任助理、副主任，1999年3月到2001年7月任主任（正局级）。在该中心任职期间，还曾兼任国际遗传工程和生物技术中心（ICEGB）中国分中心主任、中国—[[欧盟|欧盟]]生物技术中心（CEBC）中方主任，国家[[863计划|863计划]]生物技术领域办公室主任、专家委员会委员<ref name=elys/>。

2001年7月到2007年9月，任[[中国医学科学院|中国医学科学院]]、[[中国协和医科大学|中国协和医科大学]]党委书记、常务副院（校）长，2004年11月起兼任[[北京协和医院|北京协和医院]]院长。2007年9月，任卫生部副部长、党组成员。其间，2009年2月到3月在美国[[哈佛大学|哈佛大学]][[肯尼迪政府学院|肯尼迪政府学院]]（SEF班）学习。2013年2月起，兼任[[中央保健委员会|中央保健委员会]]委员、中央保健委员会办公室主任。2013年4月任国家卫生和计划生育委员会副主任、党组成员<ref name=elys/>。2018年当选[[第十三届全国人大代表|第十三届全国人大代表]]、[[全国人大教育科学文化卫生委员会|全国人大教育科学文化卫生委员会]]副主任委员<ref>{{Cite web |url =  http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=131402|title =代表信息：刘谦  |author =  |date =  |publisher = 中国人大网 |language =  |accessdate = 2018-03-20 |quote =  }}</ref>。
