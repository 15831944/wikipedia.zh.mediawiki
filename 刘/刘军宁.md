'''刘军宁'''（{{bd|1961年|11月|||L刘}}），[[安徽|安徽]][[舒城|舒城]]人，政治学家，中国年轻一代政治学者领军人物<ref name="刘军宁—腾讯 · 大家">{{cite web | url=http://dajia.qq.com/user/liujunning2011 | title=刘军宁—腾讯 · 大家 | accessdate=2013-11-10}}</ref>。1978年于[[舒城中学|舒城中学]]高中毕业<ref>{{Cite web|url = http://www.ahsczx.com/main/model/childcatalog/ChildCatalog.do?subfcode=0010804|title = 百年舒中-校友之窗|accessdate = 2014-10-07|deadurl = yes|archiveurl = https://web.archive.org/web/20141009205100/http://www.ahsczx.com/main/model/childcatalog/ChildCatalog.do?subfcode=0010804|archivedate = 2014-10-09}}</ref>，曾为中国社科院政治学所研究员，[[哈佛大学|哈佛大学]]费正清研究中心访问学者，2002年，因被举报在北大演讲中宣扬自由派思想，被该所开除<ref name="三名中国顶尖知识分子遭最高层点名警告">{{cite web | url=http://www.chinainperspective.com/ArtShow.aspx?AID=1481 | title=三名中国顶尖知识分子遭最高层点名警告 | accessdate=2009-05-21}}</ref>。现为文化部中国文化研究所研究员。于1993年获得[[北京大学|北京大学]][[政治学|政治学]]博士学位<ref name="三名中国顶尖知识分子遭最高层点名警告">{{cite web | url=http://www.chinainperspective.com/ArtShow.aspx?AID=1481 | title=三名中国顶尖知识分子遭最高层点名警告 | accessdate=2009-05-21}}</ref>。著有《共和·民主·宪政：自由主义思想研究》、《保守主义》《民主教程》等。编有《公共论丛》、《民主译丛》、《公共译丛》、《政治思潮丛书》系列，是[[中国大陆|中国大陆]]具有代表性的自由派，影响了一大批后来的自由派，比如[[王怡|王怡]]、[[陈永苗|陈永苗]]等。2006年有通俗著作新著《民主教程》。

刘军宁是2009年中共点名批判的三位「[[自由派|自由派]]」之一，另两人是国家一级编剧[[沙叶新|沙叶新]]、中国社科院哲学所研究员[[徐友渔|徐友渔]]<ref name="三名中国顶尖知识分子遭最高层点名警告">{{cite web | url=http://www.chinainperspective.com/ArtShow.aspx?AID=1481 | title=三名中国顶尖知识分子遭最高层点名警告 | accessdate=2009-05-21}}</ref>。原因在于他们参与签署[[08宪章|08宪章]]。

== 注释 ==
