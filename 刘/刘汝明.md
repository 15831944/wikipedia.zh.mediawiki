'''劉汝明'''（{{bd|1895年|5月12日|1975年|4月28日|catIdx=Liu刘}}），[[表字|字]]'''子亮'''，[[直隶省|直隶省]][[獻縣|獻縣]] (今[[河北省|河北省]][[献县|献县]])人，[[中華民國陸軍|中華民國陸軍]][[二級上將|二級上將]]。[[馮玉祥|馮玉祥]]手下的「十三太保」之一。<ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>

== 早年經歷 ==
劉汝明早年入[[馮玉祥|馮玉祥]]领导的第16混成旅。[[国民軍|国民軍]]成立后，劉汝明就任警備第1旅旅長。後来升任[[西北軍|西北軍]]暫編陸軍第10師師長。

== 中年經歷 ==
[[北伐|北伐]]後的1929年（[[民国紀元|民国]]18年），任[[国民革命軍|国民革命軍]]第五路軍总指揮兼[[國民革命軍第十二軍|國民革命軍第十二軍]]軍長。1930年[[中原大战|中原大战]]之后，被任命为[[宋哲元|宋哲元]]领导的[[國民革命軍第二十九軍|國民革命軍第二十九軍]]副軍長。<ref name=xu/><ref name=liu>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

1933年，劉汝明参加[[長城抗战|長城抗战]]，任第一四三師師長。1936年，兼任[[察哈爾省|察哈爾省]]政府主席。[[抗日战争|抗日战争]]爆发后的1937年，就任[[國民革命軍第六十八軍|國民革命軍第六十八軍]]軍長。不久被剥夺军指挥权。<ref name=xu/><ref name=liu/>

[[抗日战争|抗日战争]]结束後，劉汝明出任第四綏靖区司令官。1948年，被任命为第八兵团司令官。<ref name=xu/><ref name=liu/>在[[第二次国共内战|第二次国共内战]]中，劉汝明的軍队参加[[徐蚌会战|徐蚌会战]]（[[淮海战役|淮海战役]]），其间受到[[中国共产党|中国共产党]]方面策反，但被劉汝明拒绝，刘汝明还扣押前来策反的[[中国共产党|中国共产党]]方面的[[周鎬_(民国)|周鎬]]等人，押往南京。[[蒋介石|蒋介石]]也极力拉拢劉汝明。1949年，[[中国人民解放军|中国人民解放军]]执行[[渡江战役|渡江战役]]时，劉汝明自行决定率其第五十五军和第六十八军从[[安徽|安徽]][[贵池|贵池]]撤至[[福建|福建]][[漳州|漳州]]。<ref>[http://www.fjsq.gov.cn/showtext.asp?ToBook=161&index=65 第五章　国民革命军，福建省情资料库，于2013-04-08查阅]</ref> <ref name=xu/><ref name=bai/>

1949年8月，奉蒋介石、[[陈诚|陈诚]]之命，驻漳州的刘汝明第八兵团率第五十五军接防[[厦门|厦门]]，厦门原来的守军、蒋介石嫡系的[[李良荣|李良荣]]第二十二兵团第二十五军和第五军的第二〇〇师则奉命调往大小[[金门|金门]]。中国人民解放军于1949年10月17日攻占厦门时，刘汝明即率残部乘船逃往[[台湾|台湾]][[高雄港|高雄港]]。由于陈诚规定，凡来台湾部队，首先必须放下武器，然后按指令登陆、行军、宿营。刘汝明残部抵达高雄时，不肯缴械。陈诚遂下令，限期徒手登陆，否则便将船击沉。刘汝明残部乃被陈诚收编，刘汝明兵团彻底瓦解。刘汝明下船时，有[[宪兵|宪兵]]前来“陪同”。[[汤恩伯|汤恩伯]]称：“厦门失守，是部队执行命令未能彻底。”[[台北|台北]]《[[中央日报|中央日报]]》也发文责备厦门守军执行上级命令不力，矛头直指刘汝明。刘汝明由此一蹶不振。 <ref name=xu/><ref name=bai>张茂勋、丛乐天、邢志远：〈金门失利原因何在？〉，《百年潮》2003年第1期</ref>

== 晚年經歷 ==
1975年（民国64年）4月28日，劉汝明在台湾逝世。享年81岁。<ref name=xu/>

子[[劉鐵錚|劉鐵錚]]，法學者，於1985年獲[[蔣經國|蔣經國]]總統提名為中華民國[[司法院大法官|司法院大法官]]，直至2003年[[民主進步黨|民主進步黨]]執政期間退任。

== 評價 ==
劉汝明善結兵心，臨危不亂。<ref name="評傳，第67頁">王禹廷著：《胡璉評傳》，台北：傳記文學出版社，1987年6月15日，第67頁</ref>他一直把部隊完完整整帶到福建海岸，倘若船舶充足，能夠把他們全部接運到台灣，重新整編，仍是國軍有力之部隊。<ref name="評傳，第67頁"/>
