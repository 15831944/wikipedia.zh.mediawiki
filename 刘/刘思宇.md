'''刘思宇'''（{{bd|1985年|4月5日|catIdx=L}}）生于[[黑龙江省|黑龙江省]][[哈尔滨市|哈尔滨市]]，毕业于[[奥地利|奥地利]][[安东布鲁克纳音乐表演与舞蹈艺术大学|安东布鲁克纳音乐表演与舞蹈艺术大学]]，钢琴表演硕士学位。安东布鲁克纳音乐表演与舞蹈艺术大学钢琴艺术指导教师，奥地利Waldhausen国际大师班授课专家

== 求学历程 ==
刘思宇三岁开始学习钢琴，5岁起师从[[哈尔滨师范大学|哈尔滨师范大学]]钢琴教授周柱铨，6岁获得第一座钢琴比赛奖杯。与此同时获得钢琴家[[石叔诚|石叔诚]]的赏识并受其邀请录制少儿钢琴教学视频。18岁时考入奥地利安东布鲁克纳音乐表演与舞蹈艺术大学师从奥地利钢琴家[[Gottfried_Hemetsberger|Gottfried Hemetsberger]] 教授和俄罗斯钢琴家[[Oleg_Marchev|Oleg Marchev]]。2011年刘思宇受邀与Oleg Marchev同台为李斯特诞辰200周年音乐会演奏。
2012刘思宇获得钢琴演奏硕士学位。此前她于2010年以杰出的成绩毕业于[[萨尔茨堡莫扎特音乐大学|萨尔茨堡莫扎特音乐大学]]国际暑期学院，师从[[肖邦国际钢琴比赛|肖邦国际钢琴比赛]]评委会主席[[安德烈·雅辛斯基|安德烈·雅辛斯基]]（Andrzej Jasinski）。除此之外她还参加如[[Robert_Levin|Robert Levin]]，[[Jura_Margulis|Jura Margulis]]，[[Rolf_Plagge|Rolf Plagge]]和[[Filippo_Faes|Filippo Faes]] 等钢琴家们所开设大师班。
==艺术经历==
2013年9月她正式成为安东布鲁克纳音乐表演与舞蹈艺术大学声乐歌剧学院的钢琴艺术指导教师并获得了终身聘任合同。她是该校历史上首位中国籍钢琴艺术指导教师。<ref name="欧洲时报">{{Cite web |url=http://www.oushinet.com/epaper/pdf_zdo/20170707/FLASH/index.html |title=旅奥青年钢琴家刘思宇奏响弗赖施塔特 |accessdate=2017-08-19 |author=理查丁 |date=2017-07-07}}欧洲时报</ref>
2013年刘思宇受邀以艺术指导身份参加布鲁克纳国际音乐节歌剧项目——[[莫扎特|莫扎特]]歌剧[[女人心|女人心]]的排演工作，该剧导演是德国[[德累斯顿国家歌剧院|德累斯顿国家歌剧院]]剧院总监[[Hans_Joachim_Frey|Hans Joachim Frey]]。 2014，2015年，在奥地利[[巴特莱昂费尔登|巴特莱昂费尔登]]音乐剧周创始人，艺术总监，著名指挥家[[Thomas_Kerbl|Thomas Kerbl]]的邀请下，刘思宇以钢琴艺术指导的身份参加[[百老汇|百老汇]]音乐剧[[歌剧魅影|歌剧魅影]]和[[窈窕淑女|窈窕淑女]]的排演工作<ref name="新浪黑龙江">{{Cite web |url=http://hlj.sina.com.cn/news/b/2017-07-22/detail-ifyihrmf3152844.shtml?from=hlj_cnxh |title=被音乐大师点赞上《欧洲时报》的钢琴天才是冰城姑娘 |accessdate=2017-08-19 |date=2017-07-22}}新浪黑龙江</ref>。与此同时，奥地利指挥家，2001年[[世界合唱奥林匹克|世界合唱奥林匹克]]创始人[[Fritz_Hinterdorfer|Fritz Hinterdorfer]]教授邀请刘思宇以钢琴独奏者的身份为他50周年指挥生涯纪念演出登台演奏。之后，刘思宇又与奥地利[[VHS-Chor室内合唱团|VHS-Chor室内合唱团]]于[[弗赖施塔特|弗赖施塔特]]市共同演出圣诞音乐会。2015年刘思宇又于该市成功举办钢琴音乐会。音乐会受到了该市市长Christian Jachs和奥地利社民党党首Edi Anger等政要的大力支持，市长先生为该场音乐会亲自撰写了贺词<ref name="Freies Radio Freistadt">{{Cite web |url=https://cba.fro.at/286721 |title=Konzert für zwei Klaviere im Salzhof Freistadt mit Beni Liu |accessdate=2017-08-19 |author=Eva Schermann |date=2015-05-07}}Freies Radio Freistadt</ref>。
除此之外，刘思宇于2009年在家乡哈尔滨举办个人钢琴音乐会。2013年和2015年，她与[[大都会歌剧院|大都会歌剧院]]女高音歌唱家[[Barbara_Bonny|Barbara Bonny]]女士，[[维也纳国家歌剧院|维也纳国家歌剧院]]首席男低音歌唱家[[Robert_Holzer|Robert Holzer]]先生以及[[维也纳音乐与表演艺术大学|维也纳音乐与表演艺术大学]]教授Peter Edelmann先生共同主持奥地利[[Waldhausen国际歌剧大师班|Waldhausen国际歌剧大师班]]。<ref name="Anton Bruckner Privatuniversität">{{Cite web |url=https://www.bruckneruni.at/uploads/gtncachedevents/files/person_3F45C49E03D5EC23.pdf |title=Siyu Liu，Lebnslauf |accessdate=2017-08-19 |author=Anton Bruckner Privatuniversität}}Anton Bruckner Privatuniversität Lehrende Lebenslauf</ref>

== 媒体报道 ==
2009年8月2日，贝妮个人钢琴音乐会于[[哈尔滨|哈尔滨]]举行。她演奏了包括[[德彪西|德彪西]]，[[拉威尔|拉威尔]]，[[肖邦|肖邦]]和[[拉赫玛尼诺夫|拉赫玛尼诺夫]]等世界名家的作品。[[北方音乐|北方音乐]]杂志对音乐会给出了这样的评论：“贝妮以细腻的指法和收放自如的表演透过大众耳熟能详的旋律，将古典钢琴演奏的精致与高贵演奏出来，旋律时而低吟回荡，时而热情奔放，引领听者进入古典音乐神秘变幻的意境中。”<ref> {{Cite book | author =  | title = 北方音乐 | location = 黑龙江 | publisher = 《北方音乐》编辑部 | date = 2009年10月 | pages = 63 | ISSN = 1002-767X | accessdate = 2017-08-19 | url = | language = 中文 | quote =  }} </ref>

2017年4月16日，刘思宇携手韩国女高音歌唱家[[Margaret_Jung_Kim|Margaret Jung Kim]]，奥地利[[VHS-Singkreis室内合唱团|VHS-Singkreis室内合唱团]]以及捷克[[布杰约维采爱乐乐团|布杰约维采爱乐乐团]]（Budweiser Philharmonie）于奥地利[[弗赖施塔特|弗赖施塔特]]市演出约瑟夫[[海顿|海顿]]的[[Sancti_Joannis_de_Deo|Sancti Joannis de Deo]]降B大调弥撒。[[欧洲时报|欧洲时报]]以《旅奥青年钢琴家刘思宇奏响弗赖施塔特-华人演绎交响乐之父传世著作》为题进行了报道。
