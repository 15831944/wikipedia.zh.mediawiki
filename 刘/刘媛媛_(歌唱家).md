'''刘媛媛'''，[[苗族|苗族]]，出生于[[云南省|云南省]][[大理市|大理市]]，毕业于[[中央民族大学|中央民族大学]]、[[中央音乐学院|中央音乐学院]]，是[[中央民族歌舞团|中央民族歌舞团]]成员，[[一级演员|一级演员]]，关爱[[慈善|慈善]]事业，因演唱《[[五星红旗_(歌曲)|五星红旗]]》而知名。<ref name=" fish" >{{cite web |url= http://www.tianshannet.com.cn/news/content/2012-09/29/content_7293921.htm|title=歌唱家刘媛媛：“只要新疆召唤我就回来” |accessdate=  2012年9月29日 |publisher=天山网 |archiveurl= http://www.tianshannet.com.cn/news/content/2012-09/29/content_7293921.htm|archivedate=2012年12月17日 |work= |language=zh-cn}}</ref>

== 生平 ==

刘媛媛出生在[[云南省|云南省]][[大理市|大理市]]，她是苗族，孩提时生活在[[新疆维吾尔自治区|新疆维吾尔自治区]]，<ref name=" fish" />曾入读于[[中央民族大学|中央民族大学]]、[[中央音乐学院|中央音乐学院]]，师从[[郭淑珍|郭淑珍]]、[[金铁霖|金铁霖]]，刘媛媛擅长演唱[[中国民族音乐|中国民族音乐]]。<ref>{{cite web |url=http://www.northnews.cn/2012/1025/955497.shtml |title=刘媛媛亮相“民歌”盛典 从小与民歌情结浓厚 |accessdate=2012年10月25日 |publisher=正北方网 |archiveurl=https://web.archive.org/web/20160306050928/http://www.northnews.cn/2012/1025/955497.shtml |archivedate=2016年3月6日 |work= |language=zh-cn |dead-url=yes }}</ref>
