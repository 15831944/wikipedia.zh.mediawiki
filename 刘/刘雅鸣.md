'''刘雅鸣'''（{{bd|1957年|6月|}}），[[江苏|江苏]][[建湖|建湖]]人。女，[[汉族|汉族]]。[[水利部|水利部]]副部长、长江防总常务副总指挥。

== 简历 ==
*1975年7月参加工作。
*1981年6月加入[[中国共产党|中国共产党]]。[[武汉水利电力学院|武汉水利电力学院]]治河专业（今[[武汉大学水利水电学院|武汉大学水利水电学院]]）毕业。
*1982年从[[武汉水利电力大学|武汉水利电力大学]]（[[武汉大学|武汉大学]]）河流力学及治河工程专业毕业后，被分配到[[长江水利委员会|长江水利委员会]]水文局南京河床实验站，先后在技术科、科研室工作。1989年，任南京河床试验站主任、党委书记。1993年10月，任长江水利委员会水文局副局长。1995年2月，任[[水利部|水利部]]水文司副司长。1997年2月，任水利部直属机关党委副书记。1998年9月，任水利部政策法规司司长。1999年8月起，在水利部水文局（水利信息中心）历任副局长（副主任）、局长（主任）兼党委书记。2005年6月，任水利部人事劳动教育司司长。2012年5月，任水利部党组成员、人事司司长。2012年9月，任长江防总常务副总指挥，[[水利部长江水利委员会|水利部长江水利委员会]]主任、党组书记。2016年7月，任水利部副部长<ref>{{cite news||title=刘雅鸣(女)、陆桂华任水利部副部长 矫勇不再担任|url=http://district.ce.cn/newarea/sddy/201607/19/t20160719_13956655.shtml|accessdate=2016-07-20|newspaper=中国经济网}}</ref>、[[长江防总|长江防总]]常务副总指挥<ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1498854|title=长江水利委员会一把手调整：魏山忠接替刘雅鸣担任主任职务|publisher=澎湃新闻|accessdate=2016-07-15}}</ref>。

2016年12月，任[[中国气象局|中国气象局]]局长<ref>{{cite web |title = 杨宇栋任交通运输部副部长，刘雅鸣任中国气象局局长 |url = http://www.thepaper.cn/newsDetail_forward_1581274 |accessdate=2016-12-15 |website=澎湃新闻}}</ref>。

2013年，当选[[第十二届全国人大代表|第十二届全国人大代表]]<ref>{{cite news|title=全国人大代表信息-刘雅鸣|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=121562|accessdate=2013-04-05|newspaper=全国人大网}}</ref>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。
