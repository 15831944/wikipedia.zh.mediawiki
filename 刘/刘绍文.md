'''刘绍文'''（{{bd|1912年||1981年|6月9日}}）[[江西|江西]][[吉安|吉安]]人，[[中国人民解放军开国少将|中国人民解放军开国少将]]。

==生平==
1930年8月参加[[中国工农红军|中国工农红军]]。1931年2月加入[[中国共产主义青年团|中国共产主义青年团]]，1932年转入[[中国共产党|中国共产党]]。历任[[红十二军|红十二军]]三十五师一〇三团副排长、[[红十五军|红十五军]]四十四师一三〇团干事、红十五军政治部青年科科长、四十四师团政治处总支书记、十三师三十八团政治处俱乐部主任、[[红四方面军|第四方面军]]红军大学俱乐部主任。参加了[[中央苏区|中央苏区]]反“围剿”战争以及[[二万五千里长征|二万五千里长征]]<ref name=jxwmw>{{Cite web|url = http://heroes.jxwmw.cn/system/2007/10/31/010010001.shtml|title = 开国少将，刘绍文，吉安人氏|author = |date = 2007-10-31|publisher = 江西文明网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20131225095542/http://heroes.jxwmw.cn/system/2007/10/31/010010001.shtml|archive-date = 2013-12-25|dead-url = yes}}</ref><ref name=shishi/>。

长征途中，红四方面军在康北地区建立了各级波巴政府。[[波巴人民共和国|波巴人民共和国]]中央政府于1936年5月5日正式成立，驻[[甘孜县|甘孜县]]孔撒土司官寨，主席多德（德格），副主席达吉（甘孜）、孔撒（甘孜）。下设有民政、农业、畜牧、司法、军事、民族、宗教、财政等部。[[德格土司|德格土司]]所属的大头人军事“涅巴”[[夏格刀登|夏格刀登]]担任军事部部长，[[康区|康区]]富商[[邦达多吉|邦达多吉]]担任财政部部长。[[中华苏维埃共和国西北联邦|中华苏维埃共和国西北联邦]]政府派有中央政治检查处，[[邵式平|邵式平]]任党代表、刘绍文任顾问，参加波巴政府的工作<ref name=jxwmw/>。

1937年，刘绍文进入[[延安|延安]][[中国人民抗日军政大学|中国人民抗日军政大学]]学习。后任中央军委叁局政治处主任、[[陕甘宁晋绥联防军|陕甘宁晋绥联防军]]教导旅二团政委、[[西北野战军|西北野战军]]教导旅副政委。先后参加了[[延安保卫战|延安保卫战]]及[[青化砭战役|青化砭战役]]、[[羊马河战役|羊马河战役]]、[[沙家店战役|沙家店战役]]等战役<ref name=jxwmw/>。

[[中华人民共和国|中华人民共和国]]成立后，历任中央军委[[中国人民解放军总干部部|总干部部]]组织统计部（总干部部三部）副部长、[[山西省军区|山西军区]]副政委、[[北京卫戍区|北京卫戍区]]副政委。1966年5月，升任北京卫戍区政委，直到1987年10月离任（1968年12月15日—1978年7月，担任中国人民解放军空军第六军）。[[文革|文革]]期间，兼任[[北京市革命委员会|北京市革命委员会]]副主任。1971年至1979年，任[[中共北京市委|中共北京市委]]书记。在[[吴德|吴德]]主政北京市时期，北京卫戍区的[[杨俊生|杨俊生]]、[[吴忠|吴忠]]、[[黄作珍|黄作珍]]、刘绍文四位将军发挥了重要作用<ref name=jxwmw/><ref name=shishi/>。

1955年被授予少将军衔<ref>{{cite news|title=揭秘：中共首批开国将军的级别是怎样划分的？|url=http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml|accessdate=2012-06-15|newspaper=凤凰网（转自摘自《党史博览》2010年第5期）|date=2010-06-13|deadurl=yes|archiveurl=https://web.archive.org/web/20120620091504/http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml|archivedate=2012-06-20}}</ref>。曾获二级八一勋章、二级独立自由勋章、一级解放勋章<ref name=jxwmw/>。他是第四届、第五届全国人大代表，北京市政协副主席<ref name=shishi/>。

1981年6月9日，刘绍文在北京病逝，享年69岁。6月18日，刘绍文同志追悼会在[[八宝山革命公墓|八宝山革命公墓]]礼堂举行<ref name=shishi>刘绍文同志逝世，北京日报1981年6月19日，第一版</ref>。
