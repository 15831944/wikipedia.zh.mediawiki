'''刘家国'''（{{bd|1958年|11月||}}），[[四川|四川]][[达县|达县]]人，[[中国人民解放军中将|中国人民解放军中将]]。

==生平==
曾任[[中国人民解放军陆军第十三集团军|中国人民解放军陆军第十三集团军]]政治部主任等职务。2014年，任[[四川省军区|四川省军区]]政委。2015年3月，接替[[李亚洲|李亚洲]]任[[中共四川省委|中共四川省委]]常委<ref>{{cite news|title=四川省军区政委刘家国任四川省委常委，李亚洲不再担任|url=http://www.thepaper.cn/newsDetail_forward_1314400|accessdate=2017-02-05|work=澎湃新闻}}</ref>。2016年12月，任[[中国人民解放军陆军政治工作部|中国人民解放军陆军政治工作部]]主任<ref>{{cite news|title=四川省军区政委刘家国少将升任陆军政治工作部主任|url=http://www.thepaper.cn/newsDetail_forward_1604547|date=2017-01-19|work=澎湃新闻}}</ref>。2017年任[[中央军委国防动员部|中央军委国防动员部]]政委<ref name=jinsheng>{{Cite web |url = http://china.caixin.com/2017-10-22/101159594.html |title = 军委纪委原副书记宋丹中将转岗军委政法委 |author =  |date = 2017-10-22 |publisher =  财新网 |language =  |accessdate =  |quote =  }}</ref>。

是[[中共十九大|中共十九大]]代表<ref name=jinsheng/>。2018年7月，晋升中将军衔<ref>2018年7月27日CCTV7《军事报道》节目播报全国征兵工作电视电话会议，刘家国佩戴中将军衔出席。</ref><ref>{{cite web |title=中央军委机关3位新晋中将亮相 他成为最年轻中将 |url=http://news.163.com/18/0820/13/DPLIG0390001875N.html?baike |website=网易 |accessdate=2018-08-20 |date=2018-08-20}}</ref>。
