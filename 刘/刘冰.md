'''刘冰'''（{{bd|1921年|12月|2017年|7月24日|catIdx=}}），原名'''姚发光'''，[[河南省|河南省]][[伊川县|伊川县]][[吕店镇|吕店镇]]姚沟村人，[[中华人民共和国|中华人民共和国]]教育家、政治人物，[[刘冰上书事件|刘冰上书事件]]的当事人<ref name=tsinghua>{{Cite web |url = http://news.tsinghua.edu.cn/publish/thunews/9668/2017/20170725161016089473951/20170725161016089473951_.html |title = 深切缅怀清华老领导刘冰 |author =  |date = 2017-07-25 |publisher = 清华大学新闻网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
刘冰幼年丧母，少年丧父<ref name=texie/>。1938年，17岁的刘冰加入[[中国共产党|中国共产党]]。同年，刘冰进入[[延安|延安]]的[[中国人民抗日军政大学|中国人民抗日军政大学]]（简称“抗大”）学习。结业后，历任[[八路军|八路军]][[129师|一二九师]]随营干部学校青年工作股股长，抗大六分校青年干事，中共[[沁水县|沁水县]]工作委员会委员兼青年联合救国会主席，太岳岳南地区青年救国联合会主席，中共[[渑池县|渑池县]]委副书记、中共[[宜阳县|宜阳县]]委书记，中共豫西区党委青年工作科科长，豫西解放区青年联合会主任等职务<ref name=tsinghua/><ref name=texie/>。

1949年[[中华人民共和国|中华人民共和国]]成立后，刘冰历任[[中共河南省委|中共河南省委]]青年工作委员会书记，[[中国新民主主义青年团|中国新民主主义青年团]]河南省委书记。1953年，调任中国新民主主义青年团中央委员会办公室主任（1954年改任办公厅副主任，任至1956年）、农村青年工作部部长。1955年到[[中共中央党校|中共中央高级党校]]学习<ref name=tsinghua/><ref name=texie>{{cite news|title=特写：金秋时节访刘冰|url=http://news.ifeng.com/society/news/detail_2011_09/14/9181564_0.shtml|date=2011-09-14|newspaper=凤凰网}}</ref>。

1956年，刘冰调到[[清华大学|清华大学]]工作。1956年5月，在中国共产党清华大学第一次代表大会上，当选清华大学党委副书记。自此直到1966年“[[文化大革命|文化大革命]]”爆发，刘冰一直担任清华大学党委第一副书记，协助校长兼党委书记[[蒋南翔|蒋南翔]]，主持党委日常工作，任内主持制定了党的工作及学生工作若干条例。1959年2月，在中国共产党清华大学第二次代表大会上，刘冰代表清华大学第一届党委作工作报告。1962年5月5日，中国共产党清华大学第三次代表大会召开，选举产生中国共产党北京市第三次代表大会代表，根据[[中共北京市委|中共北京市委]]分配的名额，大会选出清华大学出席中国共产党北京市第三次代表大会的正式代表11人、候补代表2人，刘冰得票263票，票数最高。1962年10月，中国共产党清华大学第四次代表大会召开，刘冰再次代表清华大学党委作工作报告<ref name=tsinghua/>。

“文化大革命”爆发后，刘冰被打倒，遭受残酷迫害，多次被揪斗和殴打<ref name=tsinghua/><ref name=texie/>。1970年，才被“结合”进入清华大学领导班子，任清华大学党委副书记、革命委员会副主任<ref name=tsinghua/>。

刘冰在任内坚持原则，对于有的尚未平反的老干部的子女，刘冰在招生中一视同仁，不搞歧视。例如[[习仲勋|习仲勋]]之子[[习近平|习近平]]2000年接受《中华儿女》杂志访谈时回忆起刘冰在任期间特批他上清华大学的往事：“那时候报大学，清华有两个名额在延安地区，全分给了[[延川县|延川县]]。我三个志愿都填清华，你让我上就上，不让我上就拉倒。县里将我报到地区，县教育局领导仗义执言为我力争：清华来招生的人不敢做主，请示学校。这又是一个机遇。1975年7、8、9三个月，正是所谓的‘右倾翻案风’的时候。[[迟群|迟群]]、[[谢静宜|谢静宜]]都不在家，刘冰掌权，他说，可以来嘛。”于是清华大学招生办公室让习仲勋的下放单位洛阳耐火材料厂开了个“土证明”：“习仲勋同志属于人民内部矛盾，不影响子女升学就业”，习近平这才得以进入清华大学学习<ref name=tsinghua/>。2002年，福建省省外大学校友会等机构编写《福建博士风采》丛书时，习近平在第一卷中发表了《自述》一文，详细讲述了自己进入清华大学的曲折历程<ref name=qi/>。

1975年8月和10月，刘冰先后两次与清华大学党委的[[惠宪钧|惠宪钧]]、[[柳一安|柳一安]]、[[吕方正|吕方正]]等人联名写信给[[中共中央副主席|中共中央副主席]][[邓小平|邓小平]]转呈[[中共中央主席|中共中央主席]][[毛泽东|毛泽东]]，揭发清华大学党委书记迟群、谢静宜等人的严重问题。此即[[刘冰上书事件|刘冰上书事件]]。但却被指为“诬告信”，受到批判，并由此引发毛泽东在全国发动“[[批邓、反击右倾翻案风|批邓、反击右倾翻案风]]”运动，邓小平等人受到批判，影响了中国的整个政治局势<ref>{{cite news|title=刘冰上书毛泽东之后|url=http://news.xinhuanet.com/newmedia/2005-06/10/content_3068893.htm|date=2005-06-10|newspaper=新华网}}</ref><ref name=tsinghua/>。

1978年，刘冰获得平反，随后于同年调任[[兰州大学|兰州大学]]党委书记兼校长<ref name=tsinghua/>。当时“文革”结束不久，兰州大学和全国其他大学一样百废待兴，刘冰到任后立即着手改善学习及生活条件，很快取得明显成效<ref name=texie/>。刘冰任兰州大学党委书记兼校长时，清华大学老校友、原政治辅导员[[胡锦涛|胡锦涛]]正好在[[甘肃省建设委员会|甘肃省建设委员会]]工作，所以经常去兰州大学看望过去清华大学的老领导刘冰<ref name=zou/>。1979年，刘冰任[[甘肃省人民政府|甘肃省人民政府]]副省长。1981年起，历任[[中共甘肃省委|中共甘肃省委]]副书记兼秘书长、常务副书记。1986年，任[[甘肃省人大常委会|甘肃省人大常委会]]主任。1988年后，刘冰任[[第七届全国人大代表|第七届全国人大代表]]，[[全国人民代表大会教育科学文化卫生委员会|全国人民代表大会教育科学文化卫生委员会]]副主任委员、顾问等职。他先后当选为[[中共十二大|中共十二大]]、[[中共十三大|中共十三大]]代表<ref name=tsinghua/>。

离休后，刘冰曾多次回到清华大学，参加了《蒋南翔传》首发式等活动<ref name=tsinghua/>。2011年6月，在清华大学百年校庆和建党90周年之际，学校设立“刘冰奖”，表扬优秀的党建和学生工作者<ref>{{cite news|title=清华召开“刘冰奖基金”捐赠设立座谈会|url=http://www.tsinghua.org.cn/alumni/infoSingleArticle.do?articleId=10066866|accessdate=2012-04-29|newspaper=清华校友网|deadurl=yes|archiveurl=https://web.archive.org/web/20111027091707/http://www.tsinghua.org.cn/alumni/infoSingleArticle.do?articleId=10066866|archivedate=2011-10-27}}</ref>。

刘冰晚年著有《风雨岁月》，承认文革中自己犯过的错误，并“表示诚挚的道歉”<ref>{{Cite web|url=http://www.tsinghua.edu.cn/publish/thunews/9668/2011/20110225232113328927332/20110225232113328927332_.html|title=满天风雨，英雄本色——读刘冰同志《风雨岁月》札记|date=2011-02-25|work=清华大学}}</ref>。

2017年7月24日，刘冰在[[北京医院|北京医院]]病逝，享年96岁<ref name=zou>{{Cite web |url = http://news.ifeng.com/a/20170725/51502939_0.shtml |title =  96岁清华老人走了，他是国家领导人师长|author =  |date =2017-07-25  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。2017年7月30日，刘冰遗体送别仪式在[[北京市八宝山殡仪馆|北京市八宝山殡仪馆]]举行，习近平、[[李克强|李克强]]、[[张德江|张德江]]、[[俞正声|俞正声]]、[[刘云山|刘云山]]、[[王岐山|王岐山]]、[[张高丽|张高丽]]7位[[中共中央政治局常委|中共中央政治局常委]]，[[马凯_(官员)|马凯]]、[[刘延东|刘延东]]、[[张春贤|张春贤]]、[[赵乐际|赵乐际]]4位[[中共中央政治局委员|中共中央政治局委员]]，胡锦涛、[[李鹏|李鹏]]、[[朱镕基|朱镕基]]、[[温家宝|温家宝]]、[[宋平|宋平]]（偕夫人[[陈舜瑶|陈舜瑶]]）、[[李岚清|李岚清]]、[[曾庆红|曾庆红]]、[[吴官正|吴官正]]、[[李长春|李长春]]等已退休的原中共中央政治局常委，以及其他各级领导均赠送花圈<ref name=qi>{{Cite web |url = http://news.ifeng.com/a/20170731/51537997_0.shtml |title = 七常委哀悼这位老人 |author =  |date =2017-07-31  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
