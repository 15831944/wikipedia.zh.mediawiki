'''刘宇宁'''（{{bd|1987年|1月8日{{notetag|name=birthyear}}||}}），艺名'''小宁'''，中国内地著名[[男歌手|男歌手]]及[[演员|演员]]。早年曾参加《[[我型我秀|我型我秀]]》、《[[快乐男声|快乐男声]]》等选秀节目，之后组建乐队“摩登兄弟”并担任主唱，通过[[虎牙直播|YY直播]]、[[抖音短视频|抖音短视频]]平台演唱流行歌曲而走红。2018年中旬，刘宇宁开始活跃于综艺节目。<ref name="刘宇宁甲" >{{cite web |title=刘宇宁携摩登兄弟线下首唱 “希望有一天大家觉得我不是网红！” |url=http://epaper.oeeee.com/epaper/C/html/2018-08/20/content_44691.htm |website=南方都市报|accessdate=2018-09-18}}</ref><ref>{{cite web |title=刘宇宁 梦想登上“春晚”的舞台 |url=http://www.xinhuanet.com/ent/2018-09/11/c_1123411476.htm |website=新华网|accessdate=2018-09-18}}</ref>

== 演艺经历 ==
2007年，刘宇宁参加[[东方卫视|东方卫视]]选秀节目《我型我秀》。

2010年，参加[[湖南卫视|湖南卫视]]选秀节目《[[快乐男声|快乐男声]]》，在海选阶段被淘汰，未能晋级。<ref>{{cite web |title=摩登兄弟刘宇宁曾参加《快乐男声》被淘汰，想感谢评委的打击 |url=http://mini.eastday.com/a/180721003652796.html# |website=东方网 |accessdate=2018-10-17}}</ref>

2014年，刘宇宁与阿卓、大飞共同组建摩登兄弟乐队，并于2015年3月入驻YY直播平台。

2016年，出演电影《[[绑架女主播|绑架女主播]]》。<ref name="刘宇宁乙">{{cite web |title=网络大电影《绑架女主播》开机 玄机背后有秘密 |url=http://ent.qq.com/a/20160918/040539.htm |website=腾讯|accessdate=2018-09-18}}</ref>2017年，出演电影《[[秦贼有道|秦贼有道]]》，在剧中饰演阿龙。<ref>{{cite web |title=《秦贼有道》即将上映 娱加主播跨界演戏 |url=http://ent.ifeng.com/a/20170721/42958053_0.shtml |website=凤凰网|accessdate=2018-09-18}}</ref>

2018年7月，参加湖南卫视综艺节目《[[天天向上|天天向上]]》。<ref>{{cite web |title=好听！青春少年刘宇宁送你《阳光彩虹小白马》 |url=https://www.mgtv.com/l/100021506/4484537.html |website=芒果TV|accessdate=2018-09-18}}</ref>8月，先后参加了CCTV3《向幸福出发》、江苏卫视《金曲捞之挑战主打歌》、浙江卫视“中国蓝”十周年特别企划节目《嗨！蓝朋友》<ref>{{cite web |title=《金曲捞之挑战主打歌》：最撕心裂肺的情歌唱给你听|url=http://news.jstv.com/a/20180819/1534669725406.shtml?from=timeline |website=江苏卫视|accessdate=2018-09-19}}</ref><ref>{{cite web |title=凌潇肃蓝盈莹现身《嗨！蓝朋友》 摩登兄弟刘宇宁惊喜live实力圈粉 |url=http://www.zjstv.com/zcloud/4794843.html |website=浙江卫视|accessdate=2018-09-19}}</ref>；除此之外，刘宇宁还在8月17日携摩登兄弟队友在广州中央车站展演中心举办首次线下歌友会。<ref name="刘宇宁甲" />9月12日，为贾樟柯电影《[[江湖儿女|江湖儿女]]》翻唱宣传曲、[[迪克牛仔|迪克牛仔]]作品《有多少爱可以重来》<ref>{{cite web |title=《江湖儿女》贾樟柯导演玩转流行金曲 《有多少爱可以重来》经典旋律演绎扎心虐恋 |url=http://news.mtime.com/2018/09/12/1584201.html |website=Mtime时光网 |accessdate=2018-09-19 |date=2018-09-12}}</ref>；17日，发表首支个人单曲《想象》。<ref>{{cite web |title=摩登兄弟刘宇宁首单《想象》首发 |url=http://ent.163.com/18/0917/15/DRTRHO4I0003803U.html |website=网易|accessdate=2018-09-19}}</ref>10月19日，和摩登兄弟队友一同参加《2018浙江卫视秋季盛典》。<ref>{{cite web |title=浙江卫视美好奇妙夜全明星阵容公布 群星集结 |url=http://ent.people.com.cn/n1/2018/1009/c1012-30330399.html |website=人民网 |accessdate=2018-10-19}}</ref>11月，参加江苏卫视《[[蒙面唱将猜猜猜_(第三季)|蒙面唱将猜猜猜 (第三季)]]》。12月31日，参加《江苏卫视2019跨年演唱会》。

2019年1月，参加湖南卫视《[[歌手2019|歌手2019]]》以及《2019湖南卫视春节联欢晚会》。<ref>{{cite news|title=刘宇宁以超过146万票胜出|url=http://epaper.sxrb.com/shtml/sxwb/20190107/871982.shtml|accessdate=2019-01-07|work=山西晚报|date=2019-01-07|archive-url=https://web.archive.org/web/20190107181132/http://epaper.sxrb.com/shtml/sxwb/20190107/871982.shtml|archive-date=2019-01-07|dead-url=yes}}</ref>
