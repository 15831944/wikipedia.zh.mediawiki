《'''謀財陷阱'''》（{{lang-en|'''''Kind Lady'''''}}）是一部1951年的電影，由[[約翰·司圖加|約翰·司圖加]]執導。主演是[[埃塞爾·巴里摩爾|埃塞爾·巴里摩爾]]、{{link-en|莫里斯·埃文斯 (演員)|Maurice Evans (actor)|莫里斯·埃文斯}}、{{link-en|基南·懷恩|Keenan Wynn}}和[[安吉拉·蘭斯伯里|安吉拉·蘭斯伯里]]。<ref>{{Cite web |url = http://www.tcm.com/tcmdb/title/14729/Kind-Lady/ |title = KIND LADY(1951) |publisher = TURNER CLASSIC MOVIES |language = en}}</ref>

本片是1935年英文同名電影《[[人情鬼蜮|人情鬼蜮]]》的重拍版，原版由{{link-en|艾琳·麥克馬洪|Aline MacMahon}}飾演主角。<ref>{{Cite web |url = http://www.classicfilmguide.com/index125c.html/ |title = Kind Lady (1951) |author = Rob |date = 2014-08-29 |publisher = Classic Film Guide |language = en}}</ref>
