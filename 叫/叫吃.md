'''叫吃'''（或'''打吃'''）是[[圍棋術語|圍棋術語]]，指一方的棋子被對手圍住只剩一口氣的狀態<ref>{{cite web|url=http://www.nihonkiin.or.jp/juniorclub/01school/school004.htm|title=アタリ|accessdate=2011-03-28|publisher=日本棋院|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20120908123347/http://www.nihonkiin.or.jp/juniorclub/01school/school004.htm|archivedate=2012-09-08}}</ref>，如被圍的棋子下一步不逃將被提子。

==概要==
{{Goban 5x5|=
   a b c d e 
 1| | | | | |=
 2| | |b| | |=
 3| |b|w|A| |=
 4| | |B| | |=
 5| | | | | |=
   a b c d e|=
|24}}
在此情形之下，黑下a或b點的時候，如白棋未作出回應黑棋再落一子即可提白子。也就是說黑棋於a或b落子為叫吃的狀態。

下一步黑棋下在a點可提取中央的白子。白子逃出的唯一途徑是於a點連起。

==雙叫吃== 
在兩處同時叫吃的狀態為雙叫吃。

{{Goban 5x5|=
   a  b c d e 
 1| |  | | | |=
 2| |w1|b|w| |=
 3| | b|w| | |=
 4| | w| | | |=
 5| |  | | | |=
   a  b c d e|=
|24}}

在上圖中，白1可同時叫吃兩黑子，黑棋只能逃出其中一子。
