'''《叫魂：1768年中国妖术大恐慌》'''是[[美国|美国]][[汉学家|汉学家]][[孔飞力|孔飞力]]1990年出版的一本学术著作，作者通过对[[1768年|1768年]][[中国|中国]]一起妖术事件的研究，揭示了当时国家的运行机制。<ref>{{cite web |author1=刘擎 |title=叫魂，群体性疯狂如何可能 |url=https://cn.nytimes.com/culture/20120919/cc19liuqing/ |website=纽约时报中文网 |accessdate=2018-06-30}}</ref>

本书的中文译本于1999年由[[上海三联书店|上海三联书店]]在中国出版。<ref>{{cite web |author1=KIKI ZHAO |title=孔飞力已逝，当今中国“叫魂”术未断绝 |url=https://cn.nytimes.com/china/20160302/c02chinakuhn/ |website=纽约时报中文网 |accessdate=2018-06-30}}</ref>

== 主要内容 ==
1768年春，中国[[江西|江西]][[德清|德清]]出现了“[[叫魂|叫魂]]”案件，据传施法者可以通过被害人的姓名、衣物等对其做法，斩断他们的辫子，继而吸取其灵魂精气。流言迅速传播，并引起各地百姓的恐慌，民间出现自发的对妖术的惩罚行为。妖术谣言6月传播至长江上游，初秋传播到华北，10月份传播至陕西境内。时任统治者[[乾隆|乾隆]]皇帝下令在全国境内严查妖术，经过3个月调查，[[军机处|军机处]]认定妖术事件为谣言。<ref>{{cite web |author1=刘耿 |title=叫魂：一个谣言如何引发国家级事件 |url=http://history.sina.com.cn/bk/mqs/2015-09-07/1416125307.shtml |website=新华社-瞭望东方周刊 |accessdate=2018-06-30}}</ref>

孔飞力认为，[[乾隆盛世|乾隆盛世]]表面上经济繁荣，实则危机重重，人口过度增长，人均资源恶化，妖术事件实为平民阶层的一种权力补偿。<ref>{{cite web |author1=胡子华 |title=叫魂：没有什么能够伫立其间，以阻挡这种疯狂 |url=http://cul.qq.com/a/20160223/008429.htm |website=腾讯文化 |accessdate=2018-06-30}}</ref>
