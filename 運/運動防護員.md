'''運動防護員'''（[[英語|英語]]：'''Athletic Trainer'''，簡稱'''AT'''），又稱'''運動訓練員'''，是一種需考取專業證照之[[運動醫學|運動醫學]]專業人員，專門從事運動防護以及運動訓練等等工作。其中，運動防護是指各種體育運動之預防保健及安全維護。

在[[臺灣|臺灣]]，運動防護員屬[[中華民國教育部|教育部]][[教育部體育署|體育署]]所管轄，並由台灣運動傷害防護學會辦理「運動防護員資格檢定考試」。運動防護員常與[[物理治療師|物理治療師]]（英文簡稱 PT）合作，提供[[運動員|運動員]]和一般大眾完善的運動防護以及訓練方針。

== 執業資格 ==
=== 臺灣 ===
在[[臺灣|臺灣]]，從事運動防護工作多數會考取運動防護員資格考，不過健全的運動傷害防護師證照制度實在有待建立與充實。

* 教育部核准立案或符合本部採認規定之國內外專科以上學校運動醫學、運動保健、運動健康相關科、系、所畢業，領有畢業證書；其屬中華民國一百零六年一月一日以後畢業（包括應屆畢業，以下同）者，應經運動防護實習期滿，成績及格。
* 教育部核准立案或符合本部採認規定之國內外專科以上學校畢業，領有畢業證書，且修習運動防護員檢定規定課程，取得學分證明文件；其屬一百零六年一月一日以後畢業者，應經運動防護實習期滿，成績及格。
* 本辦法中華民國一百零三年十二月十八日修正施行前已擔任大專校院運動防護專業課程教師，並持有服務學校開具之證明。

=== 美國 ===
在[[美國|美國]]，若要考取運動防護員資格，必須先取得相關被認可的專業碩士學位或學程，並且通過由認證委員會（Board of Certification，BOC）辦理之檢定考試。各[[州|州]]針對運動防護所需之執業條件有不同的管理方式。目前，大部分的[[州|州]]（43州）需要考取防護員執照始能執業，有4州（[[科羅拉多州|科羅拉多州]]、[[俄勒岡州|俄勒岡州]]、[[西弗吉尼亞州|西弗吉尼亞州]]、[[夏威夷州|夏威夷州]]）需註冊成為防護員即可執業，有2州（[[紐約州|紐約州]]、[[南卡羅來納州|南卡羅來納州]]）需取得相關認證之證書始能執業，而[[加州|加州]]並沒有規範運動防護相關之執業規定。<ref>{{cite web |author1=Board of Certification (BOC), USA |title=State Regulation of Athletic Trainer|url=http://www.bocatc.org/athletic-trainers#state-regulation}}</ref>

== 業務範圍 ==
根據[[臺灣|臺灣]]《運動防護員資格檢定辦法》第三條第二項規定訂定：<ref>{{cite web |author1=教育部體育署 |title=運動防護員業務範圍之細項 |url=https://www.sa.gov.tw/wSite/ct?xItem=15427&ctNode=2346&mp=11}}</ref>
{{col-begin}}
{{col-2}}
* 運動傷害之預防：
# 以運動前體能檢測方式來判斷從事體育運動者之身體狀態。
# 參與及督導從事體育運動者體能訓練及調整訓練計畫（例如：關節活動度與柔軟度改善、肌力訓練與調整、神經肌肉控制與協調訓練等）。
# 檢視比賽或訓練時之環境狀態，並提供運動安全相關之建議。
# 定期檢查運動器具、設備及場地之安全。
# 協助使用合適之護具、評估護具之使用及維護狀況。
# 施予合適之貼紮或包紮。
# 提供運動傷害相關預防資訊。
# 熱身、整理操、伸展指導。

* 從事體育運動者之健康管理：
# 運動防護紀錄之建立、填寫、管理及維護。
# 運動場地、設備之安全維護及管理。
# 運動傷害預防與緊急處理之衛教及指導。
# 擬訂服務對象所處運動訓練場所之緊急事故管理及工作計畫。

{{col-2}}
* 運動傷害之辨別與評估及送醫前之緊急處理：
# 評估形成傷害與身體不適之機轉及程度。
# 評估與檢視受傷範圍及狀況。
# 使用適當之急救技術，並提供到院前之緊急救護處理。
# 協助送醫。

* 運動傷害後之防護及保健：
# 依照醫師指示，調整並協助復健流程。
# 受傷後各項體能調整與功能訓練之規劃與執行。

* 其他經主管機關認可之運動防護事務。
