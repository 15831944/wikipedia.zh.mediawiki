'''運輸署'''（[[英文|英文]]：'''Transport Department'''，[[縮寫|縮寫]]：'''TD'''）是[[香港特別行政區政府|香港特別行政區政府]][[運輸及房屋局|運輸及房屋局]]轄下的[[香港特別行政區政府部門|部門]]，於1968年成立，專門負責規管[[香港交通|香港交通]]事宜。

== 歷史 ==
1968年前，'''交通事務處'''隸屬於[[布政司署|布政司署]]；1968年12月1日成為獨立部門。1975年更改名稱為'''運輸署'''。1982年重組，負責管理運輸事宜。

=== 隸屬 ===
*[[布政司署|布政司署]]運輸科（1982年－1997年6月30日）
*[[運輸局|運輸局]]（1997年7月1日－2002年6月30日）
*[[環境運輸及工務局|環境運輸及工務局]]（2002年7月1日－2007年6月30日）
*[[運輸及房屋局|運輸及房屋局]]（2007年7月1日－）

== 職能 ==
# 管理[[香港道路|香港道路]]交通
# 進行各種車輛測試以判定其是否適合在馬路上行走，如[[傾斜測試|傾斜測試]]
# 委任獲授權人員監管香港[[隧道|隧道]]、[[公路|公路]]、[[橋樑|橋樑]]、[[香港幹線編號系統|幹線]]
# 監管香港[[公共交通|公共交通]]機構，包括[[香港巴士|巴士]]、[[香港小巴|小巴]]公司、[[香港的士|的士]]和渡輪，規劃和管理公共交通網絡
# 舉行駕駛考試、簽發[[香港駕駛執照|香港駕駛執照]]及[[車輛牌照|車輛牌照]]事務
# 提高道路安全意識
# 制訂香港長遠交通規劃
# 為[[長者及合資格殘疾人士公共交通票價優惠計劃|長者及合資格殘疾人士公共交通票價優惠計劃]]作出核數、管理、發還予有關公共交通機構等

該署亦設有[[緊急事故交通協調中心|緊急事故交通協調中心]]，於因為突發事件（如[[天災|天災]]、大型[[交通意外|交通意外]]）或特定時期（如開課日、新交通系統啟用）而影響交通時運作，負責協調各種[[交通工具|交通工具]]的服務。
