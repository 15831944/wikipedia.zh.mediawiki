'''运-10'''（Y-10）是由[[上海飞机制造厂|上海飞机制造厂]]制造的四发喷气式客机，是[[中国|中国]]首次自主研制的喷气式客机計畫，於1980年首飛成功，但与英美苏第一代喷气客机一样，因受当时技术条件的限制，运-10亦存在技术上的缺陷，仅製成兩架半样机便結束，除了當時的測試航班人員，未能實際載客。随后[[上海飞机制造厂|上海飞机制造厂]]转为与麦道公司合作，合资生产麦道80/90。设计團隊在均未投入大批量生产的运-10自研和麦道80/90合作生产项目上积累经验，其间又因[[上海大众|上海大众]]项目的成功有大批技术人员流动，至[[ARJ-21|ARJ-21]]项目终获成功。

首架組裝用作靜力測試。第二架配備的國產發動機為[[WS-8|WS-8]]，是[[普惠|普惠]][[普惠JT3D|JT3D-7]]的中国仿制品，使用的[[發動機|發動機]]及飛航電子俱為[[中國民航|中國民航]]機隊中707的後備組件。机身和[[机翼|机翼]]在位于[[西北工业大学|西北工业大学]]校园内的西安航空馆内。運-10總設計師是[[馬鳳山|馬鳳山]]，他也是轟-6轟炸機的總設計師，<ref>{{cite book|url = http://www.airforceworld.com/pla/h-6-bomber-china-4.htm|title = 運-10總設計師馬鳳山 |accessdate = 2014-01-16 |date =2009-10-01 |language =zh-hans }}</ref>副總設計師為[[程不時|程不時]]。<ref>http://www.comac.cc/xw/zhxx/201302/23/t20130223_641370.shtml</ref>

==研制、测试过程==
有观点认为，运-10的主要機體設計是經由[[波音707|波音707]]的[[逆向工程|逆向工程]]而來<ref>{{cite book|url = http://article.yeeyan.org/view/freeyiyu/4604|title = 十亿消费者－第五章：内外交困（1） |accessdate = 2012-06-18 |date =2005-10-11 |language =zh-hans |publisher =Simon & Schuster |author = James McGregor}}</ref>，其中該工程代號708，還被譏為上海波音707，但运十的研制者否定了这一说法。<ref>{{cite web |url = http://military.china.com/zh_cn/critical3/27/20070309/13980297_2.html |title = ARJ21---中国航空工业的希望兼谈运十下马 |accessdate = 2012-06-18 |date = 2007-03-09 |language = zh-hans |publisher = 中华网 |author =  |deadurl = yes |archiveurl = https://web.archive.org/web/20091221163221/http://military.china.com/zh_cn/critical3/27/20070309/13980297_2.html |archivedate = 2009-12-21 }}</ref> <ref>http://news.163.com/07/0226/14/388VCBOE00011232.html</ref>运-10机体结构是先采用苏联设计规范后参照美国适航条例全面自行研制的成果，引擎与航电由中国民航总局以波音707备件方式出资购买了数套。

運-10計劃配套動力為渦扇8，代號915甲，根據當時規劃可能為轟6放大遠程轟炸機方案動力。該發動機在波音707上進行過試驗，在運-10和遠程轟炸機兩個項目下馬後也停止研制。<ref>{{cite book|url = http://www.airforceworld.com/blog/chinese-turbofan-engine-family/|title = 運-10與中國渦扇發動機家族簡史 |accessdate = 2014-01-16 |date =2013-09-10 |language =zh-hans }}</ref>

*1970年7月，航空工业领导小组申请在[[上海|上海]]和[[广州|广州]]制造飞机，当局原则批准申请并将任务命名为'''708工程'''。
*1973年6月，[[国务院|国务院]]、[[中央军委|中央军委]]正式批准在上海研制大型首長專机。
*1975年6月，完成全部设计图纸。
*1976年9月，制出了供静力试验用的飞机。
*1978年11月，在飞机结构强度研究所进行全机静力试验，证明飞机强度符合要求。与此同时，开展了影响安全可靠的全机四大系统地面模拟试验。
*1980年9月26日，飞机由机长[[王金大|王金大]]首飞成功。
*从首飞成功到1984年共飞行了130多个起落、170多个飞行小时。先后飞抵[[北京|北京]]、[[哈尔滨|哈尔滨]]、[[乌鲁木齐|乌鲁木齐]]、[[郑州|郑州]]、[[合肥|合肥]]、[[广州|广州]]、[[昆明|昆明]]、[[成都|成都]]、[[拉萨|拉萨]]等国内主要城市，并七次沿“死亡航线”飞抵高原城市[[拉萨|拉萨]]，成为首架飞往拉萨市的自产飞机。
*1986年，[[中华人民共和国财政部|财政部]]否决了3000万元人民币研制费用预算，研制计划终止。

==取消及現況==
运-10工程的最终取消原因很多，市场不足和技术落后是最主要的原因，订单不足导致单价过高，民航市场不足导致运营成本极高<ref>{{cite news|title=短片：國產客機C919非國產客機第一胎 1980年「運10」首飛成功 因資金不足下馬|url=https://news.mingpao.com/ins/instantnews/web_tc/article/20170505/s00004/1493980026240|agency=明報|date=2017-05-05|language=zh-hant}}</ref>。政治因素被人认为是原因之一。文革后期，海外培养的科学技术人才凋零，受教育和政治影响，科技人员断層严重、质素不高。由于许多国产研發项目纷纷失败、或者大幅落后国外产品，当时中国科研经费剧减一半，从1.3%降到0.65%，低效科研平台被放弃，许多科研队伍解散。同时[[民航总局|民航总局]]一直对此项目持抵制也是一个原因。运-10工程开始于[[文化大革命|文化大革命]]中的1970年，主要支持者“[[四人帮|四人帮]]”成员[[王洪文|王洪文]]、[[张春桥|张春桥]]在[[文革|文革]]结束后遭到打击，也被部分人认为是运十取消的原因。現時僅存一架的运-10，尾號B-0002，自棄用後一直停放在[[上海大场机场|上海大场机场]]內中國商飛公司的草坪上。<ref>http://lt.cjdby.net/thread-2009485-1-1.html</ref>

技术上，原采用苏联设计规范，设计出来的零件可达到高抗疲劳强度，但整机将较波音707严重超重；后参照美国适航条例FAR-25标准研制，削薄减轻零件，因当时国内配套材料水平不足，制造出的整机仍较波音707重，抗疲劳强度却大降，经百余小时试飞后机体隔框出现裂纹，安全性较低，难以达到民航机实用标准。在尝试将定位从客机改为货运机及军用机未果后放弃。
