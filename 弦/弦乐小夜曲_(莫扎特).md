'''G大调弦乐小夜曲'''（{{lang-de|Serenade für Streicher in G-Dur}}，常稱{{lang|de|Eine Kleine Nachtmusik}}），又称“第13号小夜曲”，作品編號K.525，是[[奧地利|奧地利]][[音樂家|音樂家]][[莫扎特|莫扎特]]的著名作品，写于1787年。莫扎特一生共写有十三首[[小夜曲|小夜曲]]，这是其中最能表現明快、優雅及富有魅力等莫扎特的特點，具有代表性的一首作品，通常称作《弦乐小夜曲》。

作品共分四个[[乐章|乐章]]，好像一部小型[[交响曲|交响曲]]。

== 第一乐章：快板 ==

[[快板|快板]]，[[奏鸣曲|奏鸣曲]]结构。这一乐章具有[[进行曲|进行曲]]风格，开始是四[[小节|小节]]号角特征的引子乐句，[[G大调|G大调]]、4/4拍子。[[呈示部|呈示部]]主部主题清新欢畅、情绪饱满，连续出现[[顫音_(音樂)|顫音]]，更显得生机勃勃。副部主题由两部分构成，第一部分[[节奏|节奏]]鲜明，[[旋律|旋律]]流畅优美，以[[连奏|连奏]]和[[断奏|断奏]]的音型交替出现，4/4拍子。[[副部|副部]]的第二部分是连续反复的颤音和断奏，蕴涵着一种自得其乐的情绪。展开部非常短小，主要是本乐章引子乐句、副部主题第二部分颤音和断奏音型的展开。再现部按奏鸣曲式结构原则，副部主题的调性由属调转为主调，其它都基本是呈示部的再现。最后在短小精干的尾声中结束这一乐章。

== 第二乐章：浪漫曲（行板） ==

[[浪漫曲|浪漫曲]]，十分优美抒情，其结构具有[[回旋曲|回旋曲]]式的特征。主部主题淳朴、抒情，C大调、4/4拍子，曲调宽广。第一插部旋律富于流动性。第二插部跳弓与连奏形成鲜明对比。第三插部是全曲中唯一小调性的主题，它运用一个回音音型连续不断地出现于高音部和低音部，相互对比，加强乐曲的戏剧性，c小调、4/4拍子。

== 第三乐章：小步舞曲和中間樂段（稍快板） ==

短小、精致的[[小步舞曲|小步舞曲]]，[[複三部曲式|複三部曲式]]结构。第一部分由明快有力和抒情流畅两种富于对比性的旋律构成，G大调、3/4拍子。中间部柔和甜美，象是愉快的歌声，它也是由两种富于对比的旋律构成，D大调、3/4拍子。

== 第四乐章：迴旋曲（快板） ==

生动活泼的快板，回旋[[奏鸣曲|奏鸣曲]]式结构。[[呈示部|呈示部]]包含有两个主题，它们与第一乐章的单调有着紧密联系。主部主题是一首威尼斯流行歌曲，它洋溢着青春活力和生命的光辉，G大调、4/4拍子。副部主题优美生动而玲珑纤巧，它与第一乐章副部主题的第二部分类似。[[展开部|展开部]]是以主部主题为基础的调性变化与发展。[[再现部|再现部]]的主部和副部是倒置出现的，即副部主题在前，主部主题在后，因此加强了回旋曲所特有的热烈气氛。整个《弦乐小夜曲》在兴高采烈的欢快情绪中结束。

== 流行文化 ==
這首樂曲後改編為[[香港|香港]][[無綫電視|無綫電視]]處境喜劇《[[畢打自己人|畢打自己人]]》之主題曲（片頭曲）。另外，[[亞洲電視|亞洲電視]]節目《[[香港有飯開|香港有飯開]]》也以此樂曲作為主題音樂。
