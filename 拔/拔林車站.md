'''拔林車站'''位於[[台灣|台灣]][[臺南市|臺南市]][[官田區|官田區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*本站位於[[市道171號|市道171]]高架橋之下，有兩座[[岸式月台|岸式月台]]。
*為配合[[曾文溪橋改建工程|曾文溪橋改建工程]]，南岸之鐵路將高架化，並消除三處平交道，但本站並不會改建為高架車站，僅將站體往山側移動。

== 利用狀況 ==
*臺南市唯一之[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，僅停靠[[臺鐵區間車|區間車]]。
*拔林車站的[[閩南語|閩南語]]站名廣播依拔林的古名「拔仔林」發音（台羅：Pua̍t-á-nâ，「拔仔」即芭樂，也就是[[番石榴|番石榴]]之意）。
* 本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[Icash|icash2.0]]以及[[有錢卡|HappyCash]]進出站。

== 歷史 ==
*1901年12月16日：設「拔仔林乘降場」。
*1902年6月1日：改稱「拔仔林停車場」。
*1905年5月15日：廢止，同日恢復蕃子田停車場（今隆田車站）。
*1972年11月3日：復設此站，稱「拔林車站」並沿用至今，為招呼站，並指定由[[隆田車站|隆田站]]管理。
*1989年：改為簡易站，固定時間派員售票。
*2008年12月25日：再度降為招呼站。
*2013年5月30日：啟用多卡通刷卡機。
