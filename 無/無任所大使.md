'''無任所大使'''又稱'''巡迴大使'''，由總統或首相指派、授權代表國家的[[外交官|高级外交官]]或部長、大臣，不同於一般派駐海外固定國家、單一[[使館|使館]]的[[大使|大使]]。無任所大使通常沒有固定駐所；通常是為了外交的特定計劃或任務；通常經營有邦交或無邦交的多個國家、區域與[[聯合國|聯合國]]、[[歐盟|歐盟]]等國際組織的外交事宜。但是也有被派駐到特定國家，專門協助友邦特定事務的無任所大使。無任所大使也稱為特別[[外交使節|外交使節]]，但並不同於「特使」。<ref name = "【蘋果報你知】什麼是無任所大使_2017"> {{ cite web | title = 【蘋果報你知】什麼是無任所大使 | url = https://tw.appledaily.com/new/realtime/20170705/1154249/ | publisher = 蘋果日報 | author =  政治中心 | language = zh-tw | date = 2017-07-05 }} </ref><ref name = 外交部組織法> {{ cite web | title = 外交部組織法 | url = https://www.president.gov.tw/PORTALS/0/BULLETINS/PAPER/PDF/7002-1.PDF | accessdate = 2018-07-27 }} </ref><ref name = " 外交部主管法規共用系統-法規內容-外交部組織法 _2018"> {{ cite web | title =  外交部主管法規共用系統-法規內容-外交部組織法  | url = http://law.mofa.gov.tw/law_out/LawContent.aspx?id=FL005107 | language = zh-tw | accessdate = 2018-07-27 }} </ref><ref name = 立法院公報_第100卷_第33期_委員會紀錄> {{ cite web | title = 立法院公報_第100卷_第33期_委員會紀錄 | url = https://lis.ly.gov.tw/lgcgi/lypdftxt?10003301;0102;0177 | accessdate = 2018-07-27 }} </ref>

== 中華民國 ==
中華民國「無任所大使」制度自1998年起實施，為榮譽性質的無給職，主要借重民間力量爭取邦誼及國際空間；[[李登輝政府|李登輝政府]]與[[陳水扁政府|陳水扁政府]]都有聘任；[[馬英九政府|馬英九政府]]沒有聘任；[[蔡英文政府|蔡英文政府]]時又再度聘任。[[中華民國外交部|中華民國外交部]]依據《外交部組織法》，委由總統遴選無任所大使。
