'''无铰纲'''（[[學名|學名]]：{{lang|la|'''Inarticulata'''}}），又名[[腹茎纲|腹茎纲]]（[[學名|學名]]：{{lang|la|'''Gastrocaulia'''}}）<ref name=bioon>{{cite web |url=http://www.bioon.com/biology/Class422/11458.shtml |title=腕足动物门——无铰纲 |language=zh-hans |date=2004-04-16 |accessdate=2013-02-27 |author=南京大学地球科学数字博物馆 |archive-url=https://web.archive.org/web/20180102191521/http://www.bioon.com/biology/Class422/11458.shtml |archive-date=2018-01-02 |dead-url=yes }}</ref>，是[[腕足动物门|腕足动物门]]的一个已廢止的[[纲|纲]]級分類。

無鉸綱與[[有鉸綱|有鉸綱]]相對，是原始的腕足動物，在[[寒武紀|寒武紀]]時期十分普遍，直至[[奧陶紀|奧陶紀]]開始衰退並由有鉸綱腕足動物取代。其兩殼間並沒有[[鉸齒|鉸齒]]和鉸槽，兩殼依靠一套較為複雜的[[肌肉|肌肉]]系統來令兩邊保持對齊。殼主要成分是[[幾丁質|幾丁質]]或[[幾丁磷灰質|幾丁磷灰質]]<ref name=bioon>{{cite web |url=http://www.bioon.com/biology/Class422/11458.shtml |title=腕足动物门——无铰纲 |language=zh-hans |date=2004-04-16 |accessdate=2013-02-27 |author=南京大学地球科学数字博物馆 |archive-url=https://web.archive.org/web/20180102191521/http://www.bioon.com/biology/Class422/11458.shtml |archive-date=2018-01-02 |dead-url=yes }}</ref>、外殼主要由[[磷酸鈣|磷酸鈣]]組成，有[[肛門|肛門]]<ref name=bioon />。無鉸綱並非所有物種皆有[[肉莖|肉莖]]，有部份的外殼主要由較為堅硬的[[碳酸鈣|碳酸鈣]]組成，其肉莖有可能已退化。

== 分類 ==
无铰纲原來有两个目：
*[[无孔目|无孔目]]（Atremata）：例如[[海豆芽|海豆芽]]、Glottidia。
*[[新孔目|新孔目]]（Neotremata）：例如[[盘形腕足虫|盘形腕足虫]]（Discina）、Gramia。

後來被劃分為以下分類：
* †[[頂孔貝目|頂孔貝目]]（[[:en:Acrotretida|Acrotretida]]，又名[[乳房贝目|乳房贝目]]）
* [[舌形貝目|舌形貝目]]（[[:en:Linguilida|Linguilida]]）
* †[[小圆货贝目|小圆货贝目]]（[[:en:Obolellida|Obolellida]]）
** †[[管洞貝亞目|管洞貝亞目]]
** †[[小圆货贝亞目|小圆货贝亞目]]
* †[[神父贝目|神父贝目]]（[[:en:Paterinida|Paterinida]]）

現時無鉸綱的物種主要被分到[[舌形貝綱|舌形貝綱]]。
