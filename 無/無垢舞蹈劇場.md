'''無垢舞蹈劇場'''（[[英文|英文]]：Legend Lin Dance Theatre），是台灣一現代舞蹈表演團體，於1995年創立於[[台北|台北]]，藝術總監為舞蹈家[[林麗珍|林麗珍]]，團長為設計家[[陳念舟|陳念舟]]。

無垢舞蹈劇場秉持著林麗珍對藝術的理想與堅持，重質不重量，以「十年磨一劍」的精神欲提煉出歷久彌新的經典作品。林麗珍於無垢創立、實踐、並深化專屬於無垢舞蹈劇場的身體語言，屢屢受邀於國際重要的藝術節。

== 作品 ==
* 1995 - 《醮》，首演於台北[[國家戲劇院|國家戲劇院]]。曾受邀演出於法國．馬恩河谷國際雙年舞蹈節、法國．[[亞維儂藝術節|亞維儂藝術節]]、西班牙．馬德里夏季城市藝術節、中國．[[北京國家大劇院|北京國家大劇院]]、義大利．東西方藝術節開幕、法國．[[里昂國際雙年舞蹈節|里昂國際雙年舞蹈節]]、西班牙．庫雅劇院、西班牙．[[畢爾包|畢爾包]]亞力亞加歌劇院、德國・杜塞朵多舞蹈博覽會演出〈遙想〉。
* 2000 - 《花神祭》，首演於台北[[國家戲劇院|國家戲劇院]]。曾受邀演出於法國．[[里昂國際雙年舞蹈節|里昂國際雙年舞蹈節]]、台灣．亞太藝術論壇、西班牙．庫雅劇院、馬德里秋季藝術節、法國．[[里昂舞蹈之家|里昂舞蹈之家]]、法國．ARTE舞蹈盛宴演出〈春芽〉、 義大利．新喬凡尼劇院、法國．馬恩河谷國際雙年舞蹈節、德國．沃夫堡國際舞動舞蹈藝術節、西班牙．維多利亞劇院、奧地利．布萊根春季藝術節、奧地利．茵斯布魯克夏季舞蹈節、德國．[[德勒斯登|德勒斯登]]．[[塞姆佩爾歌劇院|塞姆佩爾歌劇院]]「芭蕾慶典」演出〈春芽〉。
* 2009 - 《觀》，兩廳院年度旗艦製作，首演於台北[[國家戲劇院|國家戲劇院]]。曾受邀演出於高雄至德堂文化中心、香港．世界文化藝術節、法國．亞眠文化中心、巴黎[[夏佑宮|夏佑宮]]國家劇院、里昂舞蹈之家、巴西．聖保羅艾爾法劇院、俄羅斯．契訶夫國際劇場藝術節、墨西哥・塞萬提斯國際藝術節（[[Festival_Internacional_Cervantino|Festival Internacional Cervantino]]）、德國・杜塞朵多舞蹈博覽會演出〈觸身有情〉
* 2014 - 《觸身・實境》，首演於台北兩廳院實驗劇場。

== 無垢的身體風格 ==
無垢從台灣本土的素材出發卻不受限於本土，其前衛現代的視覺風格與沉緩細緻的獨特美學是其重要的特色。無垢獨特的身體美學有「靜、定、鬆、沉、緩、勁」的六字訣，並以主張「中心點、中心圓、中心軸」為核心的身體，開創舞蹈嶄新語言。

== 紀錄片 ==
電影導演[[陳芯宜|陳芯宜]]於2005年開始為無垢拍攝紀錄片，共有《落花春泥——林麗珍》（文化容顏——國家文藝獎得獎者紀錄片）、《觀——製作全紀錄》紀錄片（獲第三屆「國家出版獎」佳作）、《行者》（獲台灣[[國際紀錄片影展|國際紀錄片影展]]「觀眾票選獎」、入圍「國際競賽」與「台灣競賽」、入圍[[金馬獎|金馬獎]]「最佳紀錄片」、入圍[[台北電影獎|台北電影獎]]）
