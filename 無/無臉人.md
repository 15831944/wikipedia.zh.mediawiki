《'''無臉人'''》為香港歌手[[何韻詩|何韻詩]]在2012年12月6日發行的一張國語單曲，是她的第二张个人国语专辑，主要内容是对社会现象的反应，主要讲述现代人对自我价值观的怀疑。何韻詩在接受采访时说这张专辑的同名歌曲《无脸人》更像是在说她自己<ref>{{cite web |title=何韵诗"出柜"被疑要给密友名分 回应：揣测而已 |url=http://www.chinanews.com/yl/2012/11-19/4339351.shtml |website=中国新闻网 |accessdate=2019-02-08}}</ref><ref>{{cite web |title=何韵诗黑白装扮拍摄新歌MV 忆伤痛泪盈眶 |url=http://ent.sina.com.cn/y/2012-12-11/08433808535.shtml |website=SINA |accessdate=2019-02-08}}</ref><ref>{{cite web |author1=TG |title=何韵诗《无脸人》MV首播 化身黑色剪影 |url=http://ent.sina.com.cn/y/2012-12-07/15093806455.shtml |website=SINA |accessdate=2019-02-08}}</ref>。
