《'''永世之戰'''》（{{lang-en|'''''The Forever War'''''}}）是美國作家[[喬·海德曼|喬·海德曼]]於1974年發表的[[軍事科幻|軍事科幻]]小說，故事講述人類與金牛星人（Taurans）之間進行[[宇宙戰爭|宇宙戰爭]]的沉思故事。本書在1975年獲得[[星云奖|星云奖]]，以及在1976年獲得[[雨果獎|雨果獎]]和[[軌跡獎|軌跡獎]]。《''{{link-en|Forever Free (novel)|Forever Free (novel)|Forever Free}}''》（1999年）和《''{{link-en|Forever Peace|Forever Peace}}''》（1997年）則是有各自主題的續集小說。中篇小說《''A Separate War''》（1999年）算是另一個續集，事件同時發生在《永世之戰》的最後部分。小說非正式地組成「{{link-en|永世之戰系列|The Forever War series}}」，小說還啟發了{{link-en|永世之戰 (漫畫)|The Forever War (comics)|漫畫}}和棋盤遊戲。<ref>{{Cite web |url = https://www.boardgamegeek.com/boardgame/5043/forever-war |title = The Forever War (1983) |publisher = BoardGameGeek |language = en}}</ref>

《永世之戰》是{{link-en|SF Masterworks|SF Masterworks}}系列的第一個標題。

== 說明 ==
這部小說廣泛被認為是作者在[[越戰|越戰]]期間服兵役時的描寫，並被看作透過[[宇宙史詩|宇宙史詩]]來寫出他的戰爭經歷。<ref>{{Cite web |url = http://web.mit.edu/m-I-t/science_fiction/profiles/haldeman_index.html |title = Joe Haldeman (1943- ) |publisher = Media in Transition |language = en}}</ref>本作品的自傳性的其他提示是主角的姓氏Mandella，這是作者姓氏的近似[[易位構詞遊戲|易位構詞]]，Mandella是一名物理學生，像海德曼一樣，還有主要女性角色的名字Marygay Potter，這與海德曼妻子的婚前姓名幾乎相同。

海德曼收到一封[[羅伯特·海萊因|羅伯特·海萊因]]祝賀他獲得星雲獎的信，其「意義比獎項本身大」。<ref>''Requiem'',  Yoji Kondo, editor, p. 274</ref> 根據作者{{link-en|斯彼得·羅賓遜|Spider Robinson}}所說，海萊因在頒獎宴會上找海德曼，並說這本書「可能是我讀過的最好的未來戰爭故事！」<ref>''Requiem'',  Yoji Kondo, editor, p. 315</ref>
