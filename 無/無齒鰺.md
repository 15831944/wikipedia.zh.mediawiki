'''無齒鰺'''，又稱'''黃鸝無齒鰺'''，俗名為虎斑瓜，為[[輻鰭魚綱|輻鰭魚綱]][[鱸形目|鱸形目]][[鱸亞目|鱸亞目]][[鰺科|鰺科]]的其中一個[[種|種]]。

== 分布 ==
本魚分布於[[印度|印度]][[太平洋|太平洋]]區，包括[[紅海|紅海]]、[[東非|東非]]、[[馬爾地夫|馬爾地夫]]、[[斯里蘭卡|斯里蘭卡]]、[[印度|印度]]、[[孟加拉灣|孟加拉灣]]、[[泰國|泰國]]、[[馬來西亞|馬來西亞]]、[[印尼|印尼]]、[[日本|日本]]、[[台灣|台灣]]、[[中國|中國]]、[[菲律賓|菲律賓]]、[[澳洲|澳洲]]、[[新幾內亞|新幾內亞]]、[[所羅門群島|所羅門群島]]、[[密克羅尼西亞|密克羅尼西亞]]、[[馬紹爾群島|馬紹爾群島]]、[[馬里亞納群島|馬里亞納群島]]、[[斐濟群島|斐濟群島]]、[[諾魯|諾魯]]、[[吐瓦魯|吐瓦魯]]、[[夏威夷群島|夏威夷群島]]、[[薩摩亞群島|薩摩亞群島]]、[[吉里巴斯|吉里巴斯]]、[[法屬波里尼西亞|法屬波里尼西亞]]、[[美國|美國]][[加利福尼亞州|加利福尼亞州]]、[[加利福尼亞灣|加利福尼亞灣]]、[[加拉巴哥群島|加拉巴哥群島]]、[[厄瓜多|厄瓜多]]、[[巴拿馬|巴拿馬]]沿岸海域。

== 深度 ==
水深0至20公尺。

== 特徵 ==
本魚下頜、鋤骨均無齒，僅舌面粗糙。體黃色，兩側有9至11條寬窄不一的黑色橫帶。體長可達110公分。

== 生態 ==
本魚主要棲息於珊瑚礁區，偶而也會進入岩石海岸及港口防波堤內。以小型無脊椎動物為食。

== 經濟利用 ==
本魚色澤鮮艷而且攻擊力不強，在水族箱裡又很溫馴，容易飼養。另外也可作為食用魚，適合各種烹飪方法。
