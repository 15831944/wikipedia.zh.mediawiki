'''無尾港水鳥保護區'''位於[[中華民國|中華民國]][[宜蘭縣|宜蘭縣]][[蘇澳鎮_(臺灣)|蘇澳鎮]]，[[蘭陽平原|蘭陽平原]]南端，東臨[[太平洋|太平洋]]、北至新城溪口，為依據《[[野生動物保育法|野生動物保育法]]》公告之野生動物保護區及野生動物重要棲息環境，是臺灣主要的雁鴨度冬區之一<ref>{{Cite web |url=http://conservation.forest.gov.tw/ct.asp?xItem=59987&ctNode=178&mp=10 |title=無尾港水鳥保護區 |accessdate=2013-01-09 |publisher=農委會林務局 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304085241/http://conservation.forest.gov.tw/ct.asp?xItem=59987&ctNode=178&mp=10 |archivedate=2016-03-04 }}</ref><ref>{{Cite web |url=http://conservation.forest.gov.tw/ct.asp?xItem=3356&ctNode=176&mp=10 |title=宜蘭縣無尾港野生動物重要棲息環境 |accessdate=2013-01-09 |publisher=農委會林務局 |deadurl=yes |archiveurl=https://web.archive.org/web/20121031231141/http://conservation.forest.gov.tw/ct.asp?xItem=3356&ctNode=176&mp=10 |archivedate=2012-10-31 }}</ref>。

==歷史==
無尾港原為《噶瑪蘭志略》中所記載的「馬賽港」，[[台灣日治時期|日治時期]]因海防之需引河水從頂寮附近出海，原出海口水量驟減，之後又因洪水、農地重劃，逐漸淤積成沒有出海口的河港，故被稱作無尾港<ref>{{Cite web |url=http://www.wuweiriver.org.tw/html/link2.asp |title=話說無尾港 |publisher=宜蘭縣無尾港文教促進會 |accessdate=2013-01-09}}</ref>。

1993年，因[[台灣電力公司|台灣電力公司]]計劃於當地興建「蘇澳火力發電廠」，當地民眾組織「港邊里反火力自救會」反對興建電廠<ref>{{Cite web |url=http://www.commonhealth.com.tw/article/article.action?id=5018768 |title=到無尾港尋一片忘憂地 |author=林宜昭 |publisher=康健雜誌 |date=2000-04 |accessdate=2013-01-09}}</ref>。後經生態調查發現，當地是鳥種豐富的重要溼地。宜蘭縣政府據此以水鳥公園的計畫案送至[[行政院|行政院]]，經核定後成為臺灣本島第一個水鳥保護區<ref>{{Cite web|url=http://www.rti.org.tw/big5/taiwanart/c-16.aspx|title=無尾港水鳥保護區|publisher=中央廣播電台|accessdate=2013-01-09}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2003年，自發性的民間組織「無尾港文教促進會」申請讓當地成為戶外行動式的地方文化館，由促進會負責管理，[[中華民國交通部|交通部]]之後也將保護區納入[[東北角暨宜蘭海岸國家風景區|東北角暨宜蘭海岸國家風景區]]。保護區內以[[小水鴨|小水鴨]]數量最多，其次是[[花嘴鴨|花嘴鴨]]和[[琵嘴鴨|琵嘴鴨]]<ref>{{Cite web |url=http://www.necoast-nsa.gov.tw/user/Article.aspx?Lang=1&SNo=04000559 |title=無尾港水鳥保護區 |publisher=交通部觀光局東北角暨宜蘭海岸國家風景區管理處 |accessdate=2013-01-09}}</ref>。2009年以來，[[尖尾鴨|尖尾鴨]]數量也觀察到有增加趨勢<ref>{{Cite news |url=http://www.libertytimes.com.tw/2013/new/jan/9/today-north16.htm?Slots=TPhoto |title=《提供食源見效》無尾港尖尾鴨 增逾10倍 |author=楊宜敏 |publisher=自由時報 |date=2013-01-09 |accessdate=2013-01-09}}</ref>。

==周邊聚落==
無尾港水鳥保護區屬於蘇澳鎮港邊里，周邊有港口、嶺腳與岳明新村三個聚落；前兩者是閩南人聚居的地方，而岳明新村則是國府為安置榮民而設立的眷村。今日比鄰無尾港水鳥保護區的岳明國小，雖然名稱稱為岳明，但是位置是在嶺腳聚落中。

==周邊景點==
無尾港除了是個賞鳥勝地之外，還有豐富的自然與人文景觀。

===自然景觀===

*水鳥
*沙丘
*湧泉
*海灘
