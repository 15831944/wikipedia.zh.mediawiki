'''無著庵'''，原稱'''無著地'''，是[[廣東省|廣東省]][[廣州市|廣州市]]的一個尼庵，亦是[[廣州市|廣州市]]唯一一個。位置在[[德政路|德政中路]]麗水坊四十八號。其是明末高僧[[天然和尚|天然和尚]]的胞妹[[今再比丘尼|今再比丘尼]]所创建。
== 名稱 ==
其稱無著地，有含兩個意思，其一是天然和尚與今再大師在俗是兄妹，出家則為法侶，如同[[印度佛教|印度佛教史]]上的[[無著|無著]]與[[世親|世親]]倆兄弟。其二寓意「清淨無染著」<ref>{{cite book|title=廣州市志·社會·宗教志}}</ref>。
== 歷史 ==
明代時庵址原是某尚書家園。清康熙六年（1667）開始改建，落成於十七年，工程共耗銀三萬五千八百三十餘兩。建成佛殿、祖殿、禪堂、觀音閣、齋堂等，還有住房三十餘間，放生池一口，總面積八畝多<ref>{{cite book|title=广州市区街镇大全|author=程慧}}</ref>。[[道光帝|道光]]年（1821～1850）後，逐步發展成八房子孫，輪流管理庵內常住事務。

民國13年（1924），[[廣州市政府|廣州市政廳]]以籌餉為名，將無著庵拍賣充公，各房集資大洋五千贖回大殿、僧房等部分，留存規模壓縮近一半。

共和國初期，部分殿堂房廳被租予民政工廠作生產場地。

[[文革|文革]]時庵遭搗毀，殿堂房舍分別為'''廣州市民政工廠'''、'''東山區彈簧廠'''、'''東山區房管局'''和'''廣州市房管局'''擴大使用或佔用，並將大殿前放生塘填平建起兩座住宅樓房。

1986年[[广州市人民政府|廣州市人民政府]]批准恢復開放無著庵，轉交廣州市佛教協會管理。廣州佛教協會從揚州請回寬敬法師主持庵務，並撥出十萬元作重建基金。

== 建築 ==
現時進入山門，經過建築依次為天王殿、大雄寶殿、教學樓。左右兩側是僧舍和學生樓，而大雄寶殿左後方就有觀音殿。復建庵舍占地3,000平方米，建築面積超過5,000平方米。
