'''閉花赤箭'''（[[學名|學名]]：''Gastrodia clausa''）是一種[[蘭科|蘭科]][[赤箭屬|赤箭屬]]的[[菌異營|菌異營]]植物，全株皆無[[葉綠素|葉綠素]]。閉花赤箭最初發現時被認為是[[烏來赤箭|烏來赤箭]]或[[春赤箭|春赤箭]]尚未開花的個體，但其花部解剖構造與前述兩者完全不同，且花冠筒從開始發育到結果脫落為止都不綻放。其[[種小名|種小名]]''clausa''意為閉合，即形容花冠筒之形態。

== 形態 ==
根稀，地下根莖肥短，披黃褐色根毛樣毛與輪生鱗片。花序長約2-4cm，開深褐色花。結果時花梗會伸長至14-43cm。[[花被片|花被片]]膜質並合生成一六裂的花筒。

== 習性 ==
棲息於海拔200-700m的低海拔山區林下，每年三月初開花，三至四月結果。平時以地下部生活，僅開花時會有突出地表的花軸，並於結果後枯萎消失。

== 分佈 ==
族群首次發現於[[台北|台北]]近郊，並於[[恆春半島|恆春半島]]、[[蘭嶼|蘭嶼]]、[[琉球群島|琉球群島]]亦有族群分佈。
