'''閉後圓唇元音'''是[[元音|元音]]的一種。用於部份口說語言當中。[[國際音標|國際音標]]用以表示此音的符號為{{IPA|{{尖括号|u}}}}；而[[X-SAMPA|X-SAMPA]]則以 <code>u</code>代表此音。

普通话的韵母u就是此音。

在多數有此音的語言中，此音以唇內('endolabial')的方式發音；然而在少部份的狀況下，此音以唇外('exolabial')的方式發音。

== 閉後突出元音 ==
'''閉後突出元音'''是閉後圓唇元音最常見的變體。它通常在IPA中被轉錄為{{angle bracket|{{IPA|u}}}}，本文中亦依照此慣例。由於沒有專門用於表示「突出」（{{lang-en|protruded}}）的IPA變音符號，因此可以使用表示唇化作用的舊式符號{{angle bracket|{{IPA|  ̫}}}}，即{{angle bracket|{{IPA|u̫}}}}。另一種可能的寫法是{{angle bracket|{{IPA|uʷ}}}} 或 {{angle bracket|{{IPA|ɯʷ}}}}（被唇內化的閉後元音），然而這有可能被理解為一個[[雙元音|雙元音]]。

== 閉後不突出元音 ==
{{Infobox IPA
|above=閉後不突出元音
|ipa symbol = u͍
|ipa symbol2 = ɯᵝ
}}
在[[瑞典語|瑞典語]]、[[日語|日語]]等語言中，其閉後元音使用了一種特別的[[元音圓唇度|圓唇方式]]，這樣的情況被稱作「不突出」（compressed）或唇外音（exolabial）。{{sfnp|Ladefoged|Maddieson|1996|p=295}}已知的語言中僅有[[上海話|上海話]]擁有與突出元音互相對比的的閉後元音。但事實上，兩個元音的高度都可以從可以在[[閉元音|閉]]到[[半閉元音|半閉]]之間自由變化。{{sfnp|Chen|Gussenhoven|2015|pp=328–329}}
