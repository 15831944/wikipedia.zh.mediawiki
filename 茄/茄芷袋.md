'''台客袋'''，原名'''加志'''（台羅：'''ka-tsì'''，音似「ㄍㄚ ㄐㄧ」）<ref name=appledaily>{{cite news |url=http://www.appledaily.com.tw/appledaily/article/finance/20090727/31814852/ |title=台客袋翻身 變潮牌 傳統網袋改良 演繹台灣魅力 |publisher=[[蘋果日報_(台灣)|蘋果日報]]|date=2009-07-27 |accessdate=2014-01-15|language=zh-hant}}</ref><ref>{{Cite web|url=http://twblg.dict.edu.tw/holodict_new/result_detail.jsp?n_no=1629&source=9&level1=11&level2=46&level3=0&curpage=0&sample=0&radiobutton=0&querytarget=0&limit=1&pagenum=0&rowcount=0&cattype=1|title=《教育部臺灣閩南語常用詞辭典》：加薦仔|accessdate=2017-09-16|author=|date=|publisher=中華民國教育部}}</ref>，又稱'''加薦仔'''、'''茭薦仔'''、'''茄芷袋'''<ref name=web66>{{cite web |url=http://066621043.web66.com.tw/web/S?postId=872827 |publisher=[[台灣黃頁|台灣黃頁]] |title=裕發塑膠公司：塑膠茄芷袋、金馬牌台客袋、跑馬牌pp塑膠繩、pe袋根|accessdate=2014-01-15|language=zh-hant}}</ref>、'''復古袋'''<ref>{{cite web |url=http://www.pcstore.com.tw/monga/M09070857.htm |title=艋舺復古袋（台客袋.茄芷袋） |accessdate=2014-01-15 |language=zh-hant}}</ref>，是台灣的一種有傳統風味的[[手提包|手提包]]<ref name=appledaily />，上邊比下邊略闊，可作書包、文件包或手挽包使用<ref name=appledaily />。名稱源於日語「かぎ編み (kagiami)」，意指其編織的手法。

茄芷袋相傳起源於[[臺南市|臺南市]][[後壁區|後壁區]]<ref name=web66 />[[菁寮里|菁寮里]]<ref name=camel51 />，最初由[[茳芏|茳芏]]（三角藺草）手工編織而成<ref name=camel51>{{cite web |url=http://mypaper.pchome.com.tw/camel51/post/1320658745 |title=介紹茄芷袋 |date=2010-02-21 |accessdate=2014-01-15 |language=zh-hant}}</ref>，被稱為草袋仔，隨着台灣的工業化，後來改為使用[[尼龍網布|尼龍網布]]製作<ref>{{cite web |url=http://tw.bysources.com/sample/show.php?supno=130685&recno=30447 |title=台客袋 尼龍網布材質,易攜帶,易清洗,重複使用,環保方便 產品 塔塔加企業社 |publisher=[[百索商情網|百索商情網]] |accessdate=2014-01-15 |language=zh-hant}}</ref>的半透明<ref>{{cite web |url=http://iamjulia.pixnet.net/blog/post/25515783-%EF%BC%88%E4%B8%89%E5%B3%BD%EF%BC%89%E5%8F%B0%E5%AE%A2%E8%A2%8B |title=（三峽）台客袋 |date=2009-12-14 |accessdate=2014-01-15|language=zh-hant}}</ref>手挽袋。
