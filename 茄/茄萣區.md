'''茄萣區'''是[[台灣|台灣]][[高雄市|高雄市]]的一個行政區，位於市區西北沿海，東鄰[[湖內區|湖內區]]、[[路竹區|路竹區]]，西濱[[臺灣海峽|臺灣海峽]]，南接[[永安區|永安區]]，北鄰[[臺南市|臺南市]][[南區_(臺南市)|南區]]。

本區與臺南市僅隔[[二仁溪|二仁溪]]，和該市[[灣裡_(臺南市)|灣裡]]鬧區更是隔溪對望。為[[臺南都會區|臺南都會區]]之一部分，居民的工作和求學與臺南市關係密切。

== 地名 ==
關於本區地名之緣起，至少有數種說法：

*[[平埔族|平埔族]]社名轉譯：此地古昔原為[[西拉雅|西拉雅]][[平埔族|平埔族]][[馬卡道|馬卡道]]支族的[[嘉定社|嘉定社]]（[[傑顛社|傑顛社]]）居住地之一，「茄萣」地名係由平埔族語：“Cattia”或“Cattea”音譯而來，這個字在平埔族語中是「十」字與「魚」字的結合，意謂「多魚之地」；後來漢人以此發音與沙洲上茂生的茄萣樹相類似，因此譯為「茄萣社」或「茄藤社」。另外，本地原應是一個濱外沙洲的孤島，後來漸漸地與陸地相連形成砂嘴形半島，後方的海水漸成[[海灣|海灣]]、[[潟湖|潟湖]]。[[荷蘭|荷蘭]]人測繪地圖中，有「漁夫角」、「漁夫灣」之名，顯見當時已有漁人搭寮居住於此。

*由於[[植物|植物]]「茄萣」而取名：《[[台灣通史|台灣通史]]》虞衡志：「茄萣：生海濱，本可為薪。皮色赭，以染網。安邑有茄萣莊。」茄萣樹原稱茄藤仔，亦稱海茄苳或海茄冬，為[[紅樹林|紅樹林]]（Mangrove）之一種。紅樹林為一種[[被子植物|被子植物]]，常分布於[[熱帶|熱帶]]或[[亞熱帶|亞熱帶]]地區風浪平靜之[[海岸|海岸]]，而[[鹹水|鹹水]]、[[淡水|淡水]]交會且泥砂淤積之處尤為多見。台灣北岸、西岸的河口附近及[[潟湖|潟湖]]海岸多有生長。台灣的紅樹林植物計有[[水筆仔|水筆仔]]、[[海茄苳|海茄苳]]、[[紅茄苳|紅茄苳]]、[[五跤梨|五跤梨]]、[[細蕊紅樹|細蕊紅樹]]、[[欖李|欖李]]等六種，本鄉沿海的紅樹林多為海茄苳。茄藤仔亦即海茄苳，學名為Avicennia marina (Forsk) Vierh，屬被子植物[[馬鞭草科|馬鞭草科]]（Verbenaceae）。為常綠[[喬木|喬木]]，葉對生、革質、橢圓形，長四至六公分。蒴果呈扁球形，脫離母株後，一面隨波飄流，一面自行發芽成長，待飄至有泥土之水濱，很快落地生根，長成新株。

「茄萣」之[[臺灣話|臺灣話]]讀作「Ka-tiann-á」（茄萣仔），但其外語發音有兩派讀法。早期[[中華民國內政部|內政部]]採用「茄萣」的外語讀音為「Jīadìng」（ㄐㄧㄚ　ㄉㄧㄥˋ），因此將該地名之拼音依[[注音符號第二式|注音符號第二式]]定為「Jiading」，區內路牌、聯外道路與[[高雄市茄萣區茄萣國民小學|茄萣國小]]、[[高雄市立茄萣國民中學|茄萣國中]]等學校譯名即採用「Jiading」。然而當地人外語幾乎一致讀「ㄑㄧㄝˊㄉㄧㄥˋ」，例如[[臺南市公車|臺南市公車]]、[[高雄客運|高雄客運]]以該讀音依照當時中央政府所採用的譯音方案－－通用拼音定為「Chieding」。
2011年6月，正式以當地居民閩南語發音以臺灣話羅馬字拼音定為「Katiann」。

== 歷史 ==
清代本區為[[鳳山縣_(臺灣)|鳳山縣]]與[[臺灣縣|臺灣縣]]交界之處，因二層行溪常更改入海處，導致縣界變更頻仍。

1920年本鄉轄區劃歸於[[高雄州|高雄州]][[岡山郡|岡山郡]][[湖內庄|湖內-{庄}-]]，1945年改制為高雄縣岡山區湖內鄉，人口達三萬餘人，屬一級鄉鎮。

1948年白砂崙、頂茄萣、下茄萣與[[崎漏|崎漏]]四聚落，自湖內鄉分出，獨立設置茄萣鄉。
2010年12月25日，高雄縣市合併後，改稱高雄市茄萣區。

== 地理 ==
位於[[嘉南平原|嘉南平原]]西南部；[[高雄市|高雄市]]的最西北，隔二仁溪和[[臺南市|臺南市]]相望，茄萣南北狹長約六公里，由北往南分為[[白砂崙|白砂崙]]、[[頂茄萣|頂茄萣]]、[[下茄萣|下茄萣]]、[[崎漏|崎漏]]（塭寮）四大聚落，境內幾乎每處都與海洋及潟湖內海近在咫尺，由潟湖興築起來的興達港是優良漁港，與永安區的全國最大火力發電廠-興達火力發電廠隔港嘴深情相望，茄萣外海是絕佳魚場，故有居民倚以近海捕魚維生，區內東側又是海、淡水養殖魚塭重鎮，所以人人幾乎是「海的子民」，驗證在地名的源起上，無論是「多魚之地」，或是海邊紅樹林樹種之一的茄萣仔（茄藤仔、海茄苳），或是「漁夫角」，都充分說明此事實。

在漁村現代化後，區內整體居屋結構以販厝透天厝為主。頂茄萣與下茄萣聚落貫連一氣，形成本區商業服務業活動最繁盛，人口最多的地區。隨著人口緩慢增加，近年新建案的推出，及新道路的開闢拓寬，狹長型的茄萣整體發展方向仍維持著填平魚塭，繼續往東挺進（閩南語：過港仔。），由大排水溝的仁愛路往莒光路推進。舊時位於電信局後的廣大舊墳區在撿骨及遷移先人後，開發成茄萣運動公園，已於2010年5月4日落成。

== 出身名人 ==
*[[鄭正煜|鄭正煜]]:前南社社長、教育台灣化聯盟社長
*[[王玉雲|王玉雲]]:中國國民黨中常委，前高雄市長。
*[[楊金海|楊金海]]:白色恐怖受害者-陳政府國策顧問。
*[[鄭龍水|鄭龍水]]:前新黨全國不分區立法委員。
*[[林仙保|林仙保]]：中國國民黨[[高雄|高雄]][[台灣地方派系|紅派]]大老，曾任茄萣農會總幹事、[[高雄縣|高雄縣]][[議員|議員]]，後連任四屆[[臺灣省議會|臺灣省議會]][[議員|議員]]，退休後將政治版圖交予其[[庶子|庶子]][[林益世|林益世]]。
*[[林益世|林益世]]：中國國民黨副主席兼前立委，曾因馬家軍身分獲馬英九任用為[[行政院|行政院]][[秘書長|秘書長]]。2012年6月29日因向廠商索賄事發而辭職。
*[[李進文|李進文]]：詩人。著有詩集《一枚西班牙錢幣的自助旅行》、《不可能；可能》、散文集《蘋果香的眼睛》、編有《Dear Epoch─創世紀詩選1994∼2004》、蔡志忠《莊子說》經典系列等多媒體書系、編撰《科技遊俠──溫世仁》等多媒體書。創作曾多次獲全國學生文學獎、中國時報文學獎、聯合報文學獎、中央日報文學獎、台北文學獎、台灣文學獎等詩獎。
*[[林天瑞|林天瑞]]：畫家。1927年出生於高雄縣茄萣鄉一個漁村，自幼即以愛畫、能畫名聞鄉里。早年追隨顏水龍、李石樵等前輩大畫家，奠定堅實的繒畫基礎，並力求建立自我風貌，進而於當時重要展覽全省美展、台陽展、全省教員展等屢獲大獎。此外，先生一生致力繒畫六十年如一日，聯展無數，國內外個展二十八次，成績斐然。除上述獎項外，近年更榮獲高雄縣美術獎、高雄市文藝獎、文建會文馨獎、教育部文化獎章等殊榮。
*[[薛平南|薛平南]]：書法名家，知名篆刻家。字盤石、定之，別署心玉主。師承王壯為教授、李普同教授。曾獲中興文藝獎、中山文藝獎、國家文藝獎。
*[[薛松村|薛松村]]：薛平南啟蒙，入大學師事周肇煒先生。現任教於國立高雄師範大學附屬高級中學。
*[[薛志揚|薛志揚]]：字叔眉，號白沙，1957年生，國立藝專美術科畢，曾獲省美展篆刻永久免審查，1995年度中山文藝獎。其藝術由家兄薛平南先生啟蒙，後師事王壯為教授、吳平教授，現專事書印創作。
* [[郭同全|郭同全]]：茄萣國小百年校慶10大名人之一，34歲當選高雄縣第八-九屆議員，當時媒體封"敢言八壯士"稱號，高雄議會議員十八羅漢結誼成員，下茄萣金鑾宮重建副主委，七十一年建醮主委逢西仕颱風該科王船祭活動令人拍案稱奇，世界郭氏宗親會創會30年理事，創建路最難，從無變有，其中的困難、阻礙與奔波，非外人能想像，總是責無旁貸，一肩挑起組織團結鄉里無私奉獻，使命感。待制度建立後，他更積極培養傳承，交棒給年輕一代，而不戀棧。如茄萣榴陽王十二始祖/金鑾宮社團法人基金會。民國71年蔡明耀競選縣長之輔選幕僚.共同創造全台最年輕縣市首長紀錄。王金平院長第一次選立委，郭信良議長第一次市選議員都曾出面助講為大家推薦好人才。[[薛茂竹|薛茂竹]]：高雄縣市合併第二屆官派首長。

== 名產 ==
===白香餅===
[[File:茄萣白香餅.jpg|thumb]]
只有茄萣才有的圓餅，薄皮，約莫一個掌大，有點像太陽餅，就是與椪餅相同，差別是外型較為扁平。口感也較為紮實好吃，早期只有白色，近年來開始有不同口味出現，如黑糖、黑糖龍眼、原味龍眼及特別的脆香餅口味。脆香餅有別於一般牛舌餅，更有嚼勁及濃郁香度，是近年離鄉在外的遊子回家，常外帶的伴手之一。這幾年臺南安平商家也把本地的名產帶到該地區去發展，常被觀光客誤認為這也是安平的特產。也算是另類的一種發揚光大。

== 文藝、宗教 ==

=== 王船祭 ===
[[File:Jiading_Township.JPG|茄萣區濱臨[[臺灣海峽]]；圖片左側為北方，隔[[二仁溪|二仁溪]]即為臺南市。|300px|right|thumb]]
茄萣區最重要的宗教祭典活動為建醮、進香出境；所謂「醮」是指地方為還願酬神而舉辦的大規模祭典，而在建王醮中的燒王船以蔚為本區民俗中最引人注目的一項活動，建醮又依日數不同區分為一朝、二朝、三朝、四朝、五朝醮，而在1996年10月茄萣區所舉辦的朝醮，規模算是空前。

一般來說建醮情事均依神明旨意而行，需在建醮日期敲定後，主辦的廟宇須在廟前搭建王船廠，待王船廠搭建好時，再擇日安王船廠，安王船廠時，廠前須豎燈篙、立廠燈、坑內放三寶（豆、釘、稻），再前往神明旨意的地點，取王船龍骨；而龍骨的尺寸長度，及造船的師父為誰，都須依神明降意指示。在建醮活動中各項儀式均各具意義，其過程為：

[[搭壇|搭壇]]：在所主辦的廟宇前搭內、外壇，內壇在建醮期間嚴密封禁，除道士及建醮工作人員外，其他人均不得進入，其目的在保持聖地的清潔；外壇則是開放的。

[[豎燈篇|豎燈篇]]：豎燈篙是對天神、地祇、鬼等「召告」此地舉行建醮祭儀，並歡迎共襄盛舉之意；在建醮活動的初期，信眾已陸續在大街小巷豎燈篙，豎燈篙係以長長的竹竿上懸光明燈，豎了多少個燈，就代表該戶有幾個男丁，燈坑則放置豆、釘、稻等物。

[[請水|請水]]：旨在敬請水神賜予聖水，用以清淨道場壇宇。

[[繞境|繞境]]：在燒王船前後，主辦的廟社也得動員信眾徒步繞境，燒王船之前係由全體信眾自主辦的廟前出發，長列隊伍浩浩蕩蕩行至外地，是為遊外境；燒王船後則遊內境繞經茄萣大街小巷，沿途由各藝陣表演，有宋江陣、旗陣、跳鼓、督陣、十二婆姐陣、五虎將、南管、北管陣等六、七十個陣頭，其目的在祈求神明保祐合境平安。

[[燒王船|燒王船]]：整個建醮活動的高潮即是燒王船，典故起因為古時常瘟疫，在醫學不發達時，信徒求籤，問卦造王船載眾將遊天府，以驅除病魔。由於在建醮日期敲定時必須恭請眾神蒞臨是為「請王」，活動結束後當然也得恭送眾神稱為「送王」，在送王的當天早上，在主辦的廟社前就已擠滿了各地擁入的香客，時間一到，王船由數百名志願為王爺服務的狀丁，手拉大繩索牽拉王船，緩緩地經過茄萣路，在沿濱海公路抵達海邊，王船行駛的前方由消防水車灑水，先行「淨路」象徵為王爺開水路，之後由各陣頭及民眾、香客前呼後擁的「護駕」前行，所經之處，街道為之雍塞，幾近水洩不通，王船拖抵定位後，信徒分工合作將添載的金紙、柴、米、鹽包、糖包及船上的貴重物品，推置在船上及船邊，隨即在神轎及諸多陣頭賣力操演下，王船起火燃燒，恭送王爺出航，同時也祈求王爺保祐漁民在海上能平安、豐收。

[[踏火|踏火]]：主要是在提高神明的神力，同時有辟邪作用。

[[普渡|普渡]]：此為建醮最後一個祭禮，家家戶戶擺出祭品，普渡天神、地祇、孤魂野鬼，對於前來之乞丐，亦散發錢財予他們，整個建醮活動亦圓滿結束。

=== 媽祖婆信仰 ===
位於漁港的茄萣因為地方靠海,居民大多捕魚維生，所以漁民與[[媽祖|媽祖婆]]的關係非常密切。因為[[媽祖|媽祖婆]]是漁民的守護神,所以茄萣[[台灣媽祖信仰|媽祖婆信仰]]是非常普遍的。逢年過節、初一十五、[[媽祖生|媽祖婆聖誕]]都會祭祀媽祖婆祈求平安。在茄萣區分為白沙崙、頂茄萣、下茄萣、崎漏四個地區，其中頂茄萣的[[賜福宮|賜福宮]]、下茄萣的[[金鑾宮|金鑾宮]]為當地著名的大廟，也是茄萣媽祖婆信仰的所在地。兩間媽祖廟的共同特色就是軟身的媽祖婆和醮期不定期。媽祖婆在茄萣又有一個別的稱呼，那就是“婆啊”，這種親切的稱呼彷彿像在稱呼家中的長輩一樣。自古以來，茄萣居民對媽祖婆仍抱持虔敬的心。

=== 威烈侯(陳湯銘)信仰 ===
本區白雲里有大陳眷村名「南田新村」，由[[大陳義胞|大陳義胞]]攜入此信仰建立「威武廟」，主祀[[威烈侯|威烈侯]](祖廟神諱「威武陳元帥」)。民國44年政府東遷來台，曾安奉神尊於[[路竹區|路竹區]]之路竹國民小學，民國45年遷至現址建廟。<ref>[http://crgis.rchss.sinica.edu.tw/temples/KaohsiungCity/chieding/1215003-WWM 威武廟]</ref>此信仰盛行在大陳移民之間，本區威武廟於2005年10月赴[[馬祖|馬祖]][[西莒島|西莒島]][[威武陳元帥廟|威武陳元帥廟]]謁祖進香<ref>[http://www.matsu-news.gov.tw/2010web/news_detail_101.php?CMD=open&UID=52507&keyword=%B0%AA%B6%AF%ADX%A9w%AB%C2%AFP%ABJ%BCq%BA%DE%B7|%A6%E8%B2%F7%B6i%AD%BB%B4M%AE%DA 馬祖日報：高雄茄定威烈侯廟管會西莒進香尋根]</ref>，以及威烈侯聖壽時，旅居外縣市、甚至國外的大陳鄉親常返鄉共襄盛舉，顯現信仰之誠。<ref>[http://www.matsu.idv.tw/topicdetail.php?f=176&t=12335 白犬島陳將軍，南台灣揚威名／馬祖通訊1997-5-3出刊]</ref>。
