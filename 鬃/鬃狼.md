'''鬃狼'''（学名：''{{lang|la|Chrysocyon brachyurus}}''）是[[南美洲|南美洲]]最大的[[犬科|犬科]]动物，在[[葡萄牙語|葡萄牙語]]被稱為''lobo-guará''（Guará 狼），[[瓜拉尼語|瓜拉尼語]]稱為''aguara guazu''（大狐），在[[西班牙语|西班牙语]]中被称为''zorro rojizo''（紅狐），主要分布在[[巴西|巴西]]南部、[[巴拉圭|巴拉圭]]以及[[安第斯山脉|安第斯山脉]]东侧的[[玻利维亚|玻利维亚]]境内。鬃狼现在濒临灭绝，而它一度生活在包括[[乌拉圭|乌拉圭]]、[[阿根廷|阿根廷]]北部的地区，虽然IUCN（[[世界自然保護聯盟|世界自然保護聯盟]]）的资料中将其列为“近危”级，但是鬃狼是[[鬃狼屬|鬃狼屬]]仅存的唯一一个物种。

== 特徵 ==
[[File:Chrysocyon.brachyurus.jpg|thumb]]
肩高约1米，重达20-25千克。在体侧皮毛是红褐色，而在背部和腿部为黑色，在尾巴尖部、喉咙处为白色。与众不同的鬃毛可以竖起，当遇到危险或准备进攻时，鬃狼会扩大身体轮廓，以显示自己。

== 活動週期 ==
觀察鬃狼的每日活動，結果發現牠們的活動頗為規律。

鬃狼每天出巢狩獵若干次。一般牠們都會在午夜過後，大約從凌晨3時半開始出外活動，直到清晨6時左右，鬃狼的活動達到最高潮。過了7時半，太陽升起，鬃狼就不再活動，返回巢穴休息，或是在小山岡上睡約4個小時。中午12時過後，鬃狼又開始活動，不過，並不像清晨時那麼劇烈。在午後大約活動6-7個小時，但並不急著停止。到夜間的11-12時左右，是一天中最後的活動期，但是，午夜的活動由於人們觀察困難，詳情無法知曉。

鬃狼所捕食的動物，主要是野鼠，而野鼠每天的活動週期恰與鬃狼的活動週期吻合。也就是說，鬃狼總是在野鼠最為活躍的刻時出來獵食。鬃狼每天有3個最活潑的活動時段，而且這些活動期多在日出之前或日落之後，因此，鬃狼具有相當強烈的[[夜行性|夜行性]]習性。

== 習性 ==
[[File:Maned_Wolf_Pup_at_White_Oak.jpg|thumb]]
=== 社會結構 ===
[[File:Chrysocyon_brachyurus2.jpg|thumb]]

与其他大型的犬科动物（如[[狼|狼]]、[[非洲猎犬|非洲猎犬]]、[[豺|豺]]等）不同的是鬃狼并不成群行动而且不会捕杀大型的动物。他们很可爱并且很少袭击人类。鬃狼夫妇会防禦一块共享的领地，雖然鬃狼很少相遇，除了[[繁殖季节|繁殖季节]]之外。數隻成狼可能同時聚集在一個食物充足的地方；舉例來說，一塊火耕過的草地，上面可能有一些曝露在外的小型[[脊椎動物|脊椎動物]]可供狼群搜捕獵食。一般[[妊娠期|妊娠期]]长至少约67天, 一胎会生下最多6个幼仔。

=== 食物 ===
鬃狼会在夜晚捕食[[老鼠|老鼠]]、野[[兔|兔]]、[[鸟|鸟]]等；其中最有趣的是，根據一些研究，[[水果|水果]]和植物在占其飲食的很大部分，超過50%。特別是一种类似[[番茄|番茄]]的水果“狼果”(Wolf Apple、Wolf Fruit或Lobeira )。過去飼養的鬃狼都被傳統地餵食大量肉類，這常導致腎結石的發生；動物園最近發現這種現象，開始餵食蔬果和狗食。

=== 與其他物種的關係 ===
鬃狼與牠們所吃的植物有[[共生|共生]]的关系，牠們攜帶传播許多不同的种子，並排便在[[切叶蚁|切叶蚁]]的蟻窩附近。這些螞蟻會用這些糞來為牠們的真菌農場施肥。鬃狼很容易[[感染|感染]]一种[[肾脏|肾脏]][[线虫|线虫]]，一種潛在致命的寄生蟲。鬃狼並不算是其他掠食者普遍的獵物，不過他們可能被兇猛的家[[犬|犬]]攻擊或殺死。[[美洲獅|美洲獅]]是一個潛在的競爭對手。(O. Courtenay, 1994; [https://web.archive.org/web/20040301171019/http://www.canids.org/PUBLICAT/CNDNEWS2/manedwf2.htm]).

== 分類 ==
根据最新的研究显示鬃狼与其他[[犬科|犬科]]动物并没有很紧密的关系，是[[更新世|更新世]]时期南美洲哺乳动物的生还者。

註釋：19世紀的參考文獻將其列為''Canis jubatus''，jubatus是[[拉丁文|拉丁文]]鬃毛的意思。鬃狼的現代分類是''Chrysocyon brachyurus''。
