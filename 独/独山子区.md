'''独山子区'''（{{lang-ug|مايتاغ رايونى}}）是[[中国|中国]][[新疆维吾尔自治区|新疆维吾尔自治区]][[克拉玛依市|克拉玛依市]]所辖的一个[[市辖区|市辖区]]。<ref>{{Cite web|url = http://map.mapbar.com/a_kelamayi_dushanzi_map|title = 图吧地图|accessdate = 2016年2月8日|author = |date = |publisher = }}</ref>总面积为400平方公里，独山子地名来源于区境内的独山。独山呈东西走向“一”字形，因不与其它山体相边，独立于戈壁中而得名。在维吾尔语和哈萨克语中，称独山子为“玛依塔克”和“玛依套”，意思是“油山”。2014年独山子区人口达到7万。独山子区已经建成每年產1000万吨炼油及120万吨乙烯的国家级石化基地。

== 发展历史 ==
汉宣帝神爵二年（公元前60年），汉朝设西域都护府，独山子归其领属。自此以来至清朝晚期，独山子一直都是游牧民族的牧场。

清朝、民国时期，独山子属乌苏县管辖。

清光绪二十三年（公元1897年），独山子石油开始开采。

民国二十五年（公元1936年），新疆省政府与苏联政府联合成立独山子炼油厂。开始引进苏联技术装备，在独山子（山）北坡形成石油工人聚居的矿区，即独山子矿区。

1955年2月22日，建立独山子矿区地方行政工作委员会。

1956年后，改为“新疆维吾尔自治区独山子镇人民委员会”（县级建制），由自治区人民委员会直接领导。

1958年，克拉玛依市成立，独山子划归为克拉玛依市的市辖区（县级建制）。

1970年，成立政企合一的独山子矿区革命委员会。

1982年6月，设立独山子区。

1984年，新疆维吾尔自治区决定克拉玛依市为直辖不设区的市，撤销克拉玛依市独山子区，成立克拉玛依市独山子镇（县级建制）。

1990年，新疆维吾尔自治区恢复克拉玛依市为设区的市，重新成立独山子区。<ref>{{Cite web|url = http://www.dsz.gov.cn/pages/xxgk/PublicDetail.aspx?id=100000000000158617|title = 独山子区 - 独山子区人民政府网|accessdate = 2016年2月8号|author = |date = |publisher = }}</ref><ref>{{Cite web|url = http://zt.iyaxin.com/content/2013-04/02/content_3920327.htm|title = 独山子历史沿革 - 亚心网|accessdate = 2016年2月8号|author = |date = 2013年4月2日|publisher = 亚心网}}</ref>

== 风景名胜 ==

=== 泥火山 ===
泥火山位于独山子北部的一个山峰上，主要有两个火山口。两者相距约100余米，较大的一个火山口直径约60厘米，高出地面1米余。喷口内有粘稠的泥浆，泥浆呈灰绿色，略带油气味，不停的向外翻泥泡，汇集的泥浆沿西南侧的开口外溢。涌出的泥流在地表形成锥型丘，底部直径在10米以上，表面覆盖有干裂的泥浆块。

=== 独山子大峡谷 ===
独山子大峡谷位于独山子区西南方，距城区约30公里。奎屯河流经独山子大峡谷之中峡谷近南北走向，峡谷古币悬崖陡峭，谷底流水时分时合，谷肩地面是荒漠草原。从独山子到大峡谷有公路或便道可抵达峡谷底或谷肩。

=== 鹿角湾 ===
从独山子出发，卡子左转往前进牧场方向，经乌拉斯台过安集海村至鹿角湾雷达山庄路口，约83公里。
