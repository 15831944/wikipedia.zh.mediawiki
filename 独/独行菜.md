'''独行菜'''（[[学名|学名]]：{{lang|la|''Lepidium apetalum''}}）是[[十字花科|十字花科]][[独行菜属|独行菜属]]的植物。

== 形态 ==
一年生或二年生[[草本|草本]]。[[茎|茎]]有乳头状短毛；狭匙形基生[[叶|叶]]，羽状分裂，上部条形叶，边缘有疏齿或全缘；春夏间开[[花|花]]，总状[[花序|花序]]顶生，果时伸长，花极小，[[萼片|萼片]]早落，丝状[[花瓣|花瓣]]；短角果近圆形或椭圆形，扁平，先端微缺，上部具窄翅；棕红色椭圆形[[种子|种子]]。

== 分布 ==
分布于[[亚洲|亚洲]]、[[欧洲|欧洲]]以及[[中国大陆|中国大陆]]的[[中国东北地区|东北]]、[[浙江|浙江]]、[[华北|华北]]、[[西南|西南]]、[[西北|西北]]、[[安徽|安徽]]、[[江苏|江苏]]等地，生长于海拔400米至2,000米的地区，多生长在山坡、路旁、山沟或村庄，目前尚未由人工引种栽培。

== 用途 ==
种子可榨油，入药有利尿、止咳化痰之效。

== 别名 ==
腺独行菜（东北植物检索表） 腺茎独行菜（秦岭植物志）
