'''独松树梁子隧道'''是[[中华人民共和国|中华人民共和国]][[四川省|四川省]][[攀枝花市|攀枝花市]]炳仁路的一段穿山[[隧道|隧道]]，为南北走向。隧道穿越区界，北端位于[[东区_(攀枝花市)|东区]][[炳草岗街道|炳]]四区南部，南端位于[[仁和区|仁和区]][[前进镇_(攀枝花市)|前进镇]]渡口村上方。独松树梁子隧道是炳仁路在[[炳草岗街道|炳草岗]]一端的起点，炳仁路在隧道北端洞口外与机场路通过[[环形交叉|环岛]]相连。

== 隧道形制 ==
独松树梁子隧道为双洞单向行车隧道，双向共四车道。两洞分别长585米和525米。单洞净宽10.5米，净高5米，设计行车速度40公里/小时<ref name="隧道贯通20070917">{{cite web|url=http://www.pzh.gov.cn/Article/zdxw/200709/27790.html|title=炳仁路后段独松树梁子隧道工程实现贯通|publisher=攀枝花日报|date=2007-09-17|5=|deadurl=yes|archiveurl=https://web.archive.org/web/20160409224344/http://www.pzh.gov.cn/Article/zdxw/200709/27790.html|archivedate=2016-04-09}}</ref>。

== 建设历史 ==
炳仁路或称炳仁线，是城市主干道。攀枝花市政府早在1990年代或更早即已筹划兴建，甚至当时的地图上就已标绘有炳仁线。炳仁路的勘察、设计等前期工作始于1997年<ref name="新区动力">{{cite web|url=http://pzhrb.newssc.org/html/2011-04/29/content_1259040.htm|title=构筑城市发展“动脉” 增添新区开发“动力”|publisher=攀枝花日报|date=2011-04-29|5=|deadurl=yes|archiveurl=https://web.archive.org/web/20160305090606/http://pzhrb.newssc.org/html/2011-04/29/content_1259040.htm|archivedate=2016-03-05}}</ref>，但工程直到2002年才正式开工。2002年因应[[攀枝花机场|攀枝花机场]]的建设而开工建设机场路，其中从紫荆山至今日独松树梁子隧道北口外长约6.5公里的路段兼作炳仁线的一部分，是攀枝花市第一条双向6车道的道路（隧道内为双向4车道），于2003年10月25日建成通车。

剩余的炳仁线在工程阶段被称为炳仁路后段，后被正式命名为炳仁路。攀枝花市政府于2002年初将利用[[世界银行|世界银行]]贷款建设炳仁路后段的设想上报四川省[[发改委|发改委]]；2006年10月，该项目初步设计才正式通过四川省发改委批复立项，[[中华人民共和国财政部|中国财政部]]与世界银行正式签订贷款协议，该项目在世行正式立项<ref name = "新区动力"/>。2006年12月1日，炳仁路后段独松树梁子隧道工程作为可追溯性贷款项目开工建设<ref>{{cite web|url=http://www.panzhihua.gov.cn/zfxxgk/cxjs/38019.shtml|title=炳仁路后段独松树梁子隧道工程实现贯通|publisher=www.panzhihua.gov.cn|date=2007-10-04|5=|accessdate=2013-03-11|archiveurl=https://web.archive.org/web/20120618022911/http://www.panzhihua.gov.cn/zfxxgk/cxjs/38019.shtml|archivedate=2012-06-18|deadurl=yes}}</ref>。该隧道由[[中國中鐵|中铁]]二十二局承建<ref>{{cite web|url=http://cr2204.cn/News_View.aspx?id=238|title=独松树梁子隧道B洞|publisher=中铁二十二局集团第四工程有限公司|date=2012-05-17|5=}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，合同总金额3847.0832万元<ref name = "隧道贯通20070917"/>。2007年6月3日B洞贯通，2007年9月16日A洞贯通<ref name = "隧道贯通20070917"/><ref>{{cite web|url=http://www.panzhihua.gov.cn/zfxxgk/cxjs/38051.shtml|title=独松树梁子隧道全线贯通|publisher=www.panzhihua.gov.cn|date=2007-10-04|5=}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，2011年4月28日正式通车<ref>{{cite web|url=http://pzhrb.newssc.org/html/2011-04/29/content_1259010.htm|title=炳仁路后段四十九以东通车|publisher=攀枝花日报|date=2011-04-29|5=|deadurl=yes|archiveurl=https://web.archive.org/web/20160305053359/http://pzhrb.newssc.org/html/2011-04/29/content_1259010.htm|archivedate=2016-03-05}}</ref><ref>{{cite web|url=http://news.163.com/11/0429/09/72Q2UOP800014AED.html|title=穿山越岭”4年多 攀枝花新干道通车(图)|publisher=华西都市报|date=2011-04-29|}}</ref>。
