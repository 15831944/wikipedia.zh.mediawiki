'''-{剛田武}-'''（{{lang-ja|ごうだ  たけし}}，{{lang-en|Goda Takeshi}}，6月15日—），通稱'''-{胖虎}-'''，是[[藤子·F·不二雄|藤子·F·不二雄]]的漫畫和動畫作品《[[哆啦A夢|哆啦A夢]]》中的重要角色之一。

剛田武為本名，其綽號為日版對'''巨人'''（{{lang|ja|ジャイアン}}，英語：Giant的[[片假名|片假名]]读法）<ref>罗马音寫法：Gian，讀法：Jai-an</ref>，舊譯'''-{技安}-'''、'''武-{胖虎}-'''、'''武-{技安}-'''，又稱'''剛田-{技安}-'''（台湾）、'''剛田-{胖虎}-'''（台湾&中国大陆）<ref>台灣青文版 第8卷〈吃下糖果變歌手〉P.43、第11卷〈Y賂棒作戰〉p.118</ref>、'''剛田武志'''（台湾）<ref>台版 大長篇《[[大雄的太陽王傳說|大雄的太陽王傳說]]》P.85</ref>、'''大熊'''（中国大陆）<ref>1989年的[[广东电视台|广东电视台]]版翻译，當時[[野比大雄|野比大雄]]則被譯為大宝；也是1991年的[[CCTV-2|CCTV-2]]&[[CCTV-1|CCTV-1]]版動畫的翻譯，當時[[野比大雄|野比大雄]]則被譯為康夫。</ref>、'''大雄'''（中国大陆）<ref>2007年的[[CCTV-14|CCTV-14]]版動畫的翻譯，當時[[野比大雄|野比大雄]]則仍被譯為康夫。</ref>，英/美國版本稱為'''Big G'''，中文版的早期譯名「技安」即出自这个绰号的音譯（当时港台地区常用该翻译，而中国大陆的一些盗版漫画和动画也使用过该翻译），而現今中文代理使用的「胖虎」則為字面上的相近。
厭惡別人奉承他(例如他煮的料理很難吃,但是大雄和小夫卻說很好吃)

== 人物 ==
設定於6月15日出生。在作品中是所有孩子們裡最胖最高的。肚臍是凸的，胖虎對此有些在意。在日文版中哆啦A梦和男孩子们一般都用绰号而非本名称呼他，而[[源靜香|靜香]]和母親等长辈们則以本名“武”（たけし）稱之而不用綽號(唯老師和出木杉有時候也會稱呼其本名，但也不是每一次這樣)，不過中文版卻完全忽略此特點。

眼睛在漫畫中多半被作者畫成一眼大一眼小的大小眼（不過比較大的是左眼還是右眼並不一定），但動畫版長期忽略這一點，直到2005年日本方面在[[聲優|聲優]]替換的同時採用更接近原著的人物造型後，眼睛的差異才得到統一。

家裡是開雜貨店(剛田雜貨店)的，因此經常被他母親要求看店或外出送貨。雖和富有的[[小夫|小夫]]形成明顯對比，但並未影響兩人友情。

經常與小夫被大雄用[[哆啦A夢_(角色)|哆啦A夢]]的道具作弄，而這經常是因為他們[[霸凌|作弄、毆打或取笑]]大雄，因此受到大雄報復。其後又會聯手搶道具，但有時兩人也會自食惡果。

=== 優點 ===
* 擅長運動，最喜歡棒球。
* 疼愛自己親妹妹和家人（包括他的媽媽,經常幫她去買東西），不放過欺負的人。
* 偶爾會展現愛心、勇氣和正義感的一面（與小夫形成對比）；而在欺負人的同時也以社區孩子們的老大自居，會在大家遇上壞人或緊急情形時挺身而出。在大長篇中也常常自告奮勇前去抗敵。
* 雖然經常欺負大雄，但有時看見大雄的勇氣和堅持亦會欣賞他而停止，甚至會轉為從旁協助(〈哆啦A夢休假〉)、(〈打倒胖虎的棒球隊〉)。在大長篇〈天方夜譚〉中更一路背著中暑的大雄橫越沙漠。
* 重視友情。大雄有一次被打劫，胖虎立即跑出來為大雄解圍，卻因被大雄使用道具而無法使用暴力，此時寧願自己被欺負都不願大雄被打。(〈胖虎有什麼優點？〉)

=== 缺點 ===
* 個性粗暴、蠻不講理、欺善怕惡、心胸狹小、絕對鐵腕獨裁
** 經常仗著自己長得高大與力量欺負同齡以及比他年紀小的孩子，[[野比大雄|大雄]]更因本性懦弱，成了他「最常欺負」的對象之一。雖然經常揍大雄，但有時就連跟他最要好的小夫也會揍。
** 對於含靜香在內的所有女性以及身心障礙者，雖極少毆打，但同樣會戏弄、恐嚇。從每次舉辦演唱會時女生也得出席，以及〈脫皮的故事〉可見一斑。
** 時常「借」了別人的東西卻不歸還（被「借」得最多的是[[小夫|小夫]]的[[漫畫|漫畫]]和模型玩具），然後到最後那些東西也「自動變成」他自己的。如果有人聲明歸還（即使態度已極度「懇求」、「卑微」、「低聲下氣」）仍會被他「自動視為」觸犯底線（講得一副我霸占你東西的樣子）而挨打。甚至還講出這種令人咋舌的離譜說辭：我什麼時候沒還了！？我只是「借永久」的！<ref>第8卷〈透明人眼藥水〉P.79</ref>如果用道具將其奪回他的反應皆是「我的（其實是別人的）東西怎麼被奪走了！？」，然後想盡辦法拿回來；日本的哆啦迷與[[ACG|ACG]]愛好者還將-{胖虎}-的名言「你的東西就是我的東西，我的東西還是我的東西」<ref>日文原文：{{lang|ja|お前のものは俺のもの、俺のものも俺のもの}}</ref><ref>美版譯文：{{lang|en|What’s mine is mine．What’s yours is mine！}}</ref>稱為「剛田哲學法/剛田主義（ジャイアニズム/Gianism）」，台灣則有「技安/胖虎名言」等不成文稱呼。
** 會對自己飼養的狗「酷哥」提出無理要求，例如倒立、扮貓叫、用肚皮褒水等，甚至因牠不能完成這些要求而遺棄牠。
** 時常看到別人很厲害的事件產生效法之心，隔天就隨自己高興決定執行以「為大家好，增強各位能力」（如反應速度）為目的的訓練（大家都看得出來根本是「藉機揍人的幌子」，因最後都會演變成所有人皆逃不過他拳頭而挨打），強迫別人一同參與，如表反對或委婉婉拒（通常為大雄）的意願，皆會被他「自動視為」抗命（不聽我的就挨打）而被迫接受。
** 總是跟小夫一同以「今天人手不夠，特別讓你加入」為理由「強迫」大雄加入棒球隊打球賽，如果拒絕或就算已說明有重大事務（如必須要寫功課或幫父母買東西）在身也會以「太神氣了」為由「視為不接受兩人好意」而強行拉他入隊或狠狠揍一頓。
** 有時會向別人分享自己學到的知識（如：磁鐵相吸相斥、披薩很好吃），當有人（如：大雄）說明事實（大家都已知道，甚至是基本常識）時，反應幾乎總是以「這是刻意讓我丟臉的奇恥大辱」為由要揍那個人<ref>第2巻第12話〈Ｎ．Ｓ徽章〉（{{lang|ja|「N・Sワッペン」}}）</ref>，甚至時常把別人的話「自行曲解」成針對自己（或其親屬）不利（某集大雄媽媽朋友鞋子不見，哆啦A夢和大雄用道具以另一隻鞋子為餌不小心釣到他，放他下來時儘管大雄說明兇手是「狗」，但他仍「認為」那是誣賴（暗指）酷哥，但結果兇手真是牠）。
* 不聰明，學業成績屬於班級中後段。在主要角色中成績僅優於大雄，當大雄考的比自己高分時甚至會生氣揍人。
* 唱歌能力極差，可以說是五音不全，卻喜歡唱歌，還嚮往當歌手。常常在空地開演唱會，並要求眾人來聽。其聲音恐怖到可以把玻璃震破、機器毀壞，甚至聽完後三天吃不下飯。在大長篇中有時還被用以擊退敵人。(詳見下面介紹)
* 料理能力也很差，卻也喜歡料理，但是自己卻不敢吃。曾做出一道料理名叫「胖虎湯」，當然大家都不敢吃。

=== 興趣 ===
; 歌唱
: 雖然最愛唱歌，志願是成為歌手，但歌聲卻是出了名的難聽，有如殺豬般，被評價為與[[核彈|核彈]]並列、「公害的一種」、「歌聲一旦進到房子裡，就再也不能住人了」。尤其哆啦A夢有次利用如果電話亭變出沒有聲音的世界，但當-{胖虎}-在紙上寫出歌詞後，全場人均感作嘔<ref>第16巻〈無聲世界〉</ref>，不但如此，他每次都「邀請」[[野比大雄|大雄]]、[[哆啦A夢|哆啦A夢]]、[[靜香|靜香]]、[[小夫|小夫]]等人出席他的[[演唱會|演唱會]]，如果大家不拍手叫好又會生氣，也曾要求[[野比大雄|大雄]]和[[哆啦A夢|哆啦A夢]]替他辦歌迷會。
: 最喜歡唱的曲子：我是胖虎，此首歌也是他的招牌歌。
: 台灣和中国大陆的上海配音版大山版動畫歌詞開頭為「我是-{胖虎}-，我是孩子王，我天下無敵…」；「要打架，要比賽運動，全都來吧！」等等，香港版「哆啦A夢大戰吸血鬼」的全版歌詞為：
:*'''''我叫做胖虎 我係小霸王 大力又強壯 係個男子漢 歌聲最響亮 覆蓋太平洋 我一開口唱 掌聲響全場'''''
:*'''''熱心幫你手 係知心好友 做人夠義氣 乜都肯幫忙 品性又純良 個個都讚賞'''''；
: 而中国大陆在上映《[[大雄的南极冰冰凉大冒险|大雄的南极冰冰凉大冒险]]》（国语版）时，胖虎演唱的歌曲则为《[[一闪一闪亮晶晶|一闪一闪亮晶晶]]》的国语版本。
: 雖然有時說有新歌，但主要也是唱這首，香港版有時也會唱哆啦A夢的主題曲。他也為自己每一次的演唱會創作其他不同的歌曲（詞曲一手包辦），但他音樂創作才華的好壞仍未知。
: 根據統計，-{胖虎}-在一次強逼哆啦A夢為他的演唱做全國廣播時出現誤傳，有35人曾經因為在深夜節目裡聽到-{胖虎}-的歌聲而氣絕，包括他的班級導師亦因此昏倒而請病假<ref>第30卷〈-{胖虎}-上電視〉（{{lang|ja|『ジャイアンテレビにでる!』}}）</ref>。亦由於歌聲的「威力」，經常被朋友基於其他目的而誘他去唱歌，例如：有一集被邀請到一家屋裡去唱歌，但其實是為屋主驅蟲，結果他的歌聲使444（谐音：死死死）隻[[蟑螂|蟑螂]]死亡<ref>第17卷〈狂音波發振機〉</ref>；在《[[大雄的魔界大冒險|魔界大冒險]]》，當眾人經過迷惑海洋時，-{胖虎}-聽到眾人說他的歌聲不及人魚，結果他跳出來高歌一曲，把人魚與海怪都嚇跑了；也曾在《技安殺人事件》中出現音樂學校的教師要錄他的歌聲，被小夫錯誤推理為慢性殺人，事實上那個人只是要把他的歌聲用在學校廣告作前後對比。

; 棒球
: 另外一個興趣是[[棒球|棒球]]，並對[[王貞治|王貞治]]很崇拜，所以，他創立的棒球隊亦參照[[讀賣巨人|讀賣巨人]]而稱為「（小）巨人棒球隊（台湾与大陆譯胖虎棒球隊）」（自己理所當然的做了隊長）並參加比賽，對於犯錯的隊員（特別是[[野比大雄|大雄]]）常常責打或強抓去特訓。而一旦自己犯了錯，就會找別人來出氣。
; 武術
: 對[[柔道|柔道]]、[[空手道|空手道]]、[[劍道|劍道]]等[[武術|武術]]都很有興趣，而[[野比大雄|大雄]]通常會成為練習對象。小夫有時也會是練習對象。
; 看漫畫
: 時常強制向別人借（或搶）別人漫畫而不還（前面已述），也曾借過1年份書籍<ref>[https://www.youtube.com/watch?v=hLQ-G89Esgc 大山版 坐雲霄飛車上學去]</ref>。
; 烹飪
: 對研發新的料理有興趣，在短篇《-{胖虎}-的料理》中，煮了一鍋「'''-{胖虎}-湯'''」，混和了如[[果醬|果醬]]、魚乾等物，散發的氣味讓在場所有人作嘔，大雄撒上道具「味素之王」才得以下嚥。
; 扮家家酒
: 在短篇《不請自來電話》中，[[哆啦A夢_(角色)|哆啦A夢]]撞見-{胖虎}-在玩[[扮家家酒|扮家家酒]]，-{胖虎}-請哆啦A夢吃[[銅鑼燒|銅鑼燒]]要他別說出去。

=== 後代 ===
* 兒子：剛田優史（{{lang|ja|ヤサシ}}），暱稱小胖虎（{{lang|ja|ジャイチビ}}），胖虎的長子，個性類似大雄同樣懦弱溫和且不擅運動，經常遭到朋友[[野比雄助|野比雄助]]欺負，本名出現於《[[哆啦美與迷你哆啦SOS|哆啦美與迷你哆啦SOS]]》。其母親是胖虎中學時期的同學後藤夏美。後藤夏美曾經在劍道對決中戰勝胖虎

== 未來 ==
在『大雄的結婚前夜』裏（短篇第25卷和以同故事為原型的電影）裏可以看到青年胖虎。婚禮前一天晚上在家跟大雄、小夫和出木杉痛喝一晚酒還祝福大雄跟靜香。

雖然想成為[[歌手|歌手]]，但根據併映電影《[[哆啦美與迷你哆啦SOS|哆啦美與迷你哆啦SOS]]》描述，繼承了家中雜貨店並改建成大型超市，已婚並生下長子優史（{{lang|ja|ジャイチビ}}）。

== 反應 ==
=== [[校園霸凌|校園霸凌]] ===
剛田武經常仗着自己的力量欺負同齡以及比他年紀小的孩子，現實世界中不時以他作為校園欺凌者的代表人物。台灣彰化縣曾以他作為欺凌者的形象舉辦反霸凌活動，並以「我們不當胖虎，不當小夫，不讓同學變成大雄」作口號<ref>{{cite news|title=彰縣反霸凌 有請哆啦Ａ夢|url=http://mag.udn.com/mag/edu/storypage.jsp?f_ART_ID=302012|newspaper=聯合報|date=2011-02-17}}{{dead link}}</ref>。

=== [[美國|美國]]霸權主義/[[日本|日本]]政治生態 ===
《為什麼靜香一定要嫁給大雄：從哆啦A夢發覺日常生活不思議》（源静香は野比のび太と結婚するしかなかったのか『ドラえもん』の現実）作者－中川右介，在第2章〈胖虎和小夫，其實就是日本民主黨政權──孩子王和小少爺的政治學〉中，指出胖虎和小夫其實就相當於國際社會中的美日關係，胖虎－美國在日本建立軍事基地並於二戰後限縮其國防，國際上則以「維護世界安全」為由強制介入他國內政，小夫－日本負擔美軍基地維護費用，國際上仰賴其保護及經濟優惠才有辦法對弱國作威作福；日本政治中由[[鳩山友紀夫|鳩山友紀夫]]（財閥/小夫）與[[菅直人|菅直人]]（有力者/胖虎）同盟領導的[[民主黨_(1996年)|民主黨]]。而事實上[[2014年|2014年]]美國引入水田版動畫時，社會上對於其相關無理「情節改動」反應即是——美國本就是個跟胖虎一樣的霸道國家。
