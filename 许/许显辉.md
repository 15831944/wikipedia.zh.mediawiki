'''许显辉'''（{{bd|1966年|2月||catIdx=X许}}），[[河南|河南]][[叶县|叶县]]人。1987年7月参加工作，1986年6月加入[[中国共产党|中国共产党]]。[[内蒙古大学|内蒙古大学]]法律专业毕业，大学学历，法学学士。

历任[[内蒙古自治区人民政府|内蒙古自治区人民政府]]法制局经济法规处副处长、处长；中共[[张家界市|张家界市]][[武陵源区|武陵源区]]委副书记、代区长、区长、区委书记<ref>http://www.zjjwly.gov.cn/zjjwly/zwgk/rsrm/20070104005752.html{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>等职。2005年8月，任[[湖南省人民政府法制办公室|湖南省人民政府法制办公室]]副主任<ref>{{cite web |url=http://www.hnsfzb.gov.cn/Item/898.aspx |title=存档副本 |accessdate=2012-02-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20140714202134/http://www.hnsfzb.gov.cn/Item/898.aspx |archivedate=2014-07-14 }}</ref>，2008年7月扶正<ref>{{cite web |url=http://www.chinalaw.gov.cn/article/dfxx/zffzdt/200806/20080600014454.shtml |title=存档副本 |accessdate=2012-02-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20140715070740/http://www.chinalaw.gov.cn/article/dfxx/zffzdt/200806/20080600014454.shtml |archivedate=2014-07-15 }}</ref>。2013年5月，当选张家界市长<ref>{{cite web|accessdate=2014-06-10|url=http://district.ce.cn/newarea/sddy/201305/27/t20130527_24422055.shtml|title=许显辉当选张家界市长|publisher=中国经济网}}</ref><ref>{{cite web|accessdate=2014-07-16|url=http://district.ce.cn/zt/zlk/bg/201402/08/t20140208_2260465.shtml|title=张家界市政府工作报告 (2013年12月25日 许显辉)|publisher=中国经济网}}</ref>。2015年6月，任中共[[益阳市|益阳市]]委副书记、市长<ref>{{cite web|accessdate=2016-12-05|url=http://district.ce.cn/newarea/sddy/201506/10/t20150610_5602045.shtml|title=胡忠雄接替魏旋君任益阳市委书记 许显辉提名市长|publisher=中国经济网}}</ref>。

2016年11月，当选中共湖南省纪委副书记<ref>{{cite news |title = 许显辉、李宗文、程纪龙当选湖南省纪委副书记|url = http://www.thepaper.cn/newsDetail_forward_1564453 |accessdate=2016-11-19 |publisher = 澎湃新闻}}</ref>。2016年12月，兼任[[湖南省监察厅|湖南省监察厅]]厅长<ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1588113|title=谢建辉、杨光荣任湖南省副省长，许显辉任省监察厅厅长|publisher=澎湃新闻|accessdate=2016-12-26}}</ref>。2017年1月，任省政府党组成员。2017年7月，任[[湖南省公安厅|湖南省公安厅]]厅长。<ref >{{cite web|url=http://district.ce.cn/newarea/sddy/201707/26/t20170726_24459656_1.shtml | language= | title=许显辉提名湖南省公安厅厅长 黄关春不再担任(简历) |publisher=中国经济网 |date=2017年7月26日 |accessdate= }}</ref>2018年1月当选湖南省副省长<ref>{{Cite web |url=http://media.china.com.cn/cmyw/2018-01-28/1216044.html# |title=许达哲当选湖南省省长　杜家毫当选湖南人大常委会主任 |access-date=2018-01-31 |archive-url=https://web.archive.org/web/20180131200944/http://media.china.com.cn/cmyw/2018-01-28/1216044.html# |archive-date=2018-01-31 |dead-url=yes }}</ref>。
