'''许士平'''（{{bd|1956年|4月||}}）[[山东|山东]][[聊城|聊城]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员。

==生平==
1981年12月，自[[山东大学|山东大学]]生物学系毕业，1975年4月加入中国共产党。历任[[国家档案局|国家档案局]]档案科学技术研究所办公室副主任、主任、副所长、所长，[[中央档案馆|中央档案馆]]国家档案局办公室主任（其间挂职担任[[四川省|四川省]][[自贡市|自贡市]]副市长）。2004年6月起，历任[[中共中央直属机关事务管理局|中共中央直属机关事务管理局]]副局长、巡视员、常务副局长、机关党委专职书记、局长。2014年5月，任中央档案馆副馆长、国家档案局副局长<ref>{{Cite web |url = http://renshi.people.com.cn/n/2014/0605/c139617-25108831-2.html |title =  许士平、胡旺林任国家档案局副局长|author =  |date = 2014-06-05 |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。2016年2月，免去中央档案馆副馆长、国家档案局副局长职务<ref>{{Cite web |url =  http://news.china.com/domestic/945/20160305/21692619.html|title =许士平被免去国家档案局副局长职务  |author =  |date =  2016-03-05|publisher =中华网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.gov.cn/gongbao/content/2016/content_5054761.htm |title = 中华人民共和国国务院任免人员 |author =  |date =  |publisher =  中华人民共和国国务院公报2016年第8号|language =  |accessdate =  |quote =  }}</ref>。
