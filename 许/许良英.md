'''许良英'''{{BD|1920年|5月3日|2013年|1月28日|catIdx=X许}}，生于[[浙江省|浙江省]][[临海市|临海市]]，中国科學史家、思想家、社会活动家，毕业于[[浙江大学|浙江大学]]，長期專於[[愛因斯坦|愛因斯坦]]的著作翻譯與研究工作。<ref name="fish" /><ref name="tiger" />

== 生平 ==
许良英生於今[[浙江省|浙江省]][[临海市|临海市]][[括苍镇|括苍镇]]张家渡村，1940年到达[[贵州省|贵州省]][[遵义县|遵义县]]，1942年，他从[[浙江大学|浙江大学]]物理系毕业，许良英是[[束星北|束星北]]和[[王淦昌|王淦昌]]的学生，主要研究[[相对论|相对论]]和[[量子力学|量子力学]]，1945年回浙大物理系任教，翌年加入[[中国共产党|中国共产党]]。<ref name="fish">{{cite web |url=http://news.qq.com/zt2011/master/master_61.htm |title = 许良英：从科学史家到思想启蒙者 |accessdate = 2011 |publisher = 腾讯网|archiveurl=http://news.qq.com/zt2011/master/master_61.htm |archivedate = 2012年11月12日 |work= |language =zh-cn}}</ref><ref name=" tiger" /><ref name=ifeng>{{cite web |url=http://news.ifeng.com/mainland/detail_2013_01/29/21695320_0.shtml|title=科学史家许良英去世 曾编译《爱因斯坦文集》 |date=2013-01-29|accessdate= 2013-01-29|publisher= 新京報 |work= |language =zh-cn}}</ref>

中共建政後，许良英於1952年從浙大被調到中國科學院，在《[[科学通报|科学通报]]》编辑室主任。1955年因为“[[胡风反革命集团案|反胡风]]和[[肃反运动|肃反运动]]”，被批判和停职审查一年。<ref name="fish" /><ref name=" tiger" /><ref name="ifeng" />，1957年[[中國科學院哲學研究所|中國科學院哲學研究所]][[自然辯證法|自然辯證法]]研究室創建時，與[[于光遠|于光遠]]一同擔任負責人。

1957年[[反右运动|反右运动]]期間，许良英被划为[[右派|右派]]，因抗拒強迫勞改的處置而被开除公職與黨藉，并遣送到老家接受[[劳动改造|勞改]]<ref>{{cite web |url = http://hb.qq.com/a/20111026/001677.htm |title = 人民日报1957年：许良英的灵魂浸透了右派思想 |accessdate = 2011年10月26日 |publisher=腾讯网 |archiveurl = http://hb.qq.com/a/20111026/001677.htm |archivedate=2012年11月12日 |work= |language =zh-cn}}</ref><ref name="tiger" />，1962年編譯《愛因斯坦文集》，至1970年出版。

1978年5月，中共中央发布[1978]55号文件，即《贯彻中央关于全部摘掉右派分子帽子决定的实施方案》，许良英恢复了党籍並回到北京市，被安排担任[[中国科学院自然科学史研究所|中国科学院自然科学史研究所]][[研究员|研究员]]<ref name="fish" /><ref name="tiger" /><ref name="ifeng" />。

1985年3月，许良英和[[方励之|方励之]]联合发起纪念[[爱因斯坦|爱因斯坦]]逝世三十周年，在[[浙江大学|浙江大学]]举办学术讨论会。期间，[[浙江大学|浙江大学]]学生和研究生邀请许良英演讲“开放改革”，他推荐[[方励之|方励之]]讲。由于方的演讲影响很大，被当局认为是宣扬“[[资产阶级自由化|资产阶级自由化]]”，[[中共中央纪律检查委员会|中共中央纪律检查委员会]]决定要第二次开除方的党籍，只是由于时任总书记[[胡耀邦|胡耀邦]]的作用，才拖延了一年多<ref name="Xu">{{cite web |url = http://beijingspring.com/bj2/2010/240/2012422193004.htm|title=许良英：《痛悼方励之》|accessdate= 2012年4月 |publisher = 北京之春|work= |language =zh-cn}}</ref><ref name="Fang">{{cite web |url = http://fang-lizhi.hxwk.org/tag/%E8%AE%B8%E8%89%AF%E8%8B%B1/ |title = 方励之：《许良英先生九秩贺》 |date=2010年7月 |publisher= 方励之纪念文集|work= |language =  zh-cn}}</ref>。 不过也因此误传，许良英因演讲被开除了党籍<ref name="fish " /><ref name="tiger" />。

1986年夏天，许良英與[[方勵之|方勵之]]、[[劉賓雁|劉賓雁]]共同發起次年舉辦《紀念反右運動三十周年學術討論會》，但因有被邀請者舉報而遭當局製止<ref name="Xu" /><ref name="Fang" />。1987年6月8日，[[陆铿|陆铿]]在[[英属香港|英属香港]]举办政治民主化研讨会，许良英寄去一篇文章。1989年，许良英写出政治民主化的联署信，42人签名，包括王淦昌、施雅風等。<ref name="fish" /><ref name="tiger" />

1992年，许良英发表《没有政治民主，改革不可能成功》，呼吁政治体制改革；1994年他與妻子王來棣及[[丁子霖|丁子霖]]等發起《為改善我國人權狀況呼籲》公開信，並刊登在《[[纽约时报|纽约时报]]》头版。翌年发起《迎接[[联合国|联合国]]宽容年，呼唤实现国内宽容》公開信，全世界上千著名科學家、學者，包括國內數十名院士和著名知識分子、國際上十名諾貝爾獎得主聯署。<ref name="fish" /><ref name="tiger" />

2006年，美国《纽约时报》发表专文《北京的爱因斯坦传人：一个为真理奋斗的叛逆者》<ref>{{cite web |url =  http://cn.nytimes.com/article/china/2013/01/29/c29xu-archive/ |title = 许良英，为真理奋斗的爱因斯坦传人 |accessdate= 2006年8月22日 |publisher=紐約時報 |language =zh-cn}}</ref>讲述许良英的生平故事。<ref name="fish" /><ref name="tiger" />2010年，他同学者[[邵建|邵建]]进行争论，提出应“走出复古迷津，重新回归‘[[五四运动|五四]]’”<ref name="ifeng" />。

2013年元旦前夕，其妻[[王來棣|王來棣]]去世，許良英亦因肺部感染住进[[北京海淀醫院|北京海淀醫院]]11樓[[深切治療部|重症監護室]]，病情一直反覆，約一月後的2013年1月28日13點25分因腦溢血去世，終年93歲<ref name="ifeng" />；夫婦遺體一同捐獻給[[北京大學醫學部|北京大學醫學部]]，遺體告別式於同月30日下午3時在北大醫學部西門遺體捐獻中心舉行。
