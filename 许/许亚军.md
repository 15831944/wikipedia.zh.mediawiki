'''许亚军'''（{{bd|1964年|2月4日||}}），生于[[河北|河北]][[涉县|涉县]]，[[中华人民共和国|中华人民共和国]]演员<ref name=mnw/>。

==生平==
小学在[[北京|北京]]一所重点文艺学校就读。8岁出演电视剧。12岁进入[[中央戏剧学院|中央戏剧学院]]表演系学习。1980年，16岁自中央戏剧学院表演系毕业，被分配到[[中国儿童艺术剧院|中国儿童艺术剧院]]当演员。1986年凭电视剧《[[寻找回来的世界|寻找回来的世界]]》中饰演忧郁迷人的不良少年“伯爵”获得“[[大众电视金鹰奖|大众电视金鹰奖]]”最佳男演员奖，吸引了许多女性观众。他也因此成为第一位登上《[[大众电影|大众电影]]》杂志封面的男演员。此后，1992年出演电视剧《[[风雨丽人|风雨丽人]]》，该剧获得第13届[[中国电视剧飞天奖|中国电视剧飞天奖]]长篇电视连续剧奖和第11届“大众电视金鹰奖”。2002年出演电视剧《[[空镜子|空镜子]]》，该剧收视掀起高潮。2017年在电视剧《[[人民的名义|人民的名义]]》中饰演省公安厅厅长祁同伟，其表演再获观众称赞<ref>{{Cite web |url = http://www.1905.com/mdb/star/708/life/ |title = 许亚军的星路历程 |author =  |date =  |publisher =  电影网|language =  |accessdate =  2017-04-30|quote =  }}</ref><ref name=mnw>{{Cite web |url =http://www.mnw.cn/news/ent/1673692-2.html  |title = 《人民的名义》许亚军饰演祁同伟 许亚军个人资料四任妻子 |author =  |date =2017-04-18   |publisher =闽南网 |language =  |accessdate =  |quote =  }}</ref>。

==家庭==
*第一任妻子：黄建萍，和许亚军同为中国儿童艺术剧院演员。后逝世<ref name=hun/>。
*第二任妻子：张晞，生于艺术家庭。父亲为演员，母亲为舞蹈家。5岁开始学[[钢琴|钢琴]]，后在[[天津音乐学院|天津音乐学院]]学习。曾任中国儿童艺术剧院青年轻音乐队键盘演奏员。1989年起出演影视剧，处女作为电影《豆蔻年华》的女主角。随后在电视剧《情系浏阳河》、《蹲班大将》中饰女主角<ref name=hun/>。
*第三任妻子：[[何晴|何晴]]，演员。1995年在和许亚军共同出演电视剧《风荷怨》时恋爱，后许亚军与张晞离婚，与何晴结婚。2001年和许亚军育有一子。2003年二人离婚<ref name=hun/>。
*第四任妻子：张澍。2004年在和许亚军共同出演一部戏时恋爱，2008年结婚。2010年和许亚军育有一子<ref name=hun>{{Cite web |url = http://yule.sohu.com/20130512/n375626276.shtml |title =揭秘许亚军4段婚姻 为小三把妻子逼出心脏病  |author =  |date = 2013-05-12 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
