'''许文广'''（{{bd|1960年|||}}）[[河南|河南]][[洛阳|洛阳]]人，[[中华人民共和国|中华人民共和国]]演员、制片人。

==生平==
许文广出生在[[洛阳市|洛阳市]]。少年时期先后在劳动路小学、洛阳市第二十六中学学习。中学毕业后入伍，从此离开家乡。入伍后，因相貌俊朗且声音好，被安排到宣传队工作。其间，许文广学会了主持技能，并学习了[[笛子|笛子]]、[[钢琴|钢琴]]演奏。后来，许文广考入[[吉林艺术学院|吉林艺术学院]]戏剧表演本科班，同班同学中有[[张凯丽|张凯丽]]。1985年毕业时，许文广收到十多个单位的调令，希望许文广去工作，最终许文广选择到[[中国煤矿文工团|中国煤矿文工团]]工作。此后，许文广出演了很多话剧、电视剧、电影。2011年，导演[[高希希|高希希]]请许文广在电视剧《[[楚汉传奇|楚汉传奇]]》中出演太监[[赵高|赵高]]，该角色引发观众热烈反响。后来导演[[郑晓龙|郑晓龙]]又请许文广在电视剧《[[芈月传|芈月传]]》中出演穆监<ref name=luo>{{Cite web |url = http://henan.163.com/16/0504/10/BM7E5JGJ022719NA.html |title =洛阳籍著名演员许文广近日回洛  |author =  |date = 2016-05-04 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。2016年，许文广凭借电视剧《芈月传》中的“穆监”一角获第三届“文荣奖”最佳男配角奖<ref name=nanpeijue>{{Cite web |url = http://ent.163.com/16/1031/11/C4MVJ67K000380EO.html |title =许文广凭借《芈月传》获“文荣奖”最佳男配角  |author =  |date = 2016-10-31 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

除演员外，许文广还有其他许多身份。许文广曾作为制片人参与制作《中国机长》、《三国》、《辛亥革命》等影视作品。2011年，[[中华人民共和国文化部|中华人民共和国文化部]]将其调到[[中国国家话剧院|中国国家话剧院]]工作，负责重新组建影视公司。许文广任中国国家话剧院国话影视文化传媒有限公司董事长<ref name=luo/>。他还是中国广播电影电视社会组织联合会演员委员会副会长<ref>{{Cite web |url =  http://www.zhongguoyanyuan.com/wyhjj/huichanglishijieshao/index.html|title = 会长与理事 |author =  |date =  |publisher = 中国广播电影电视社会组织联合会演员委员会 |language =  |accessdate = 2017-04-30 |quote =  }}</ref>。
