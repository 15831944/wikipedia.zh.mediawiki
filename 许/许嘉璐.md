'''許嘉璐'''（{{bd|1937年|6月3日|catIdx=X許}}）<ref>[http://www.mj.org.cn/xwll/mjxw/mjzyyw/201206/t20120604_139911.htm?authkey=kw0vr 许嘉璐：带着民进的影子前行【1937年出生的民进中央原主席许嘉璐，6月3日迎来了自己的第75个生日。】]</ref>，中国当代[[训诂学|训诂学]]家、[[语言学家|语言学家]]，出生于[[北京|北京]]，祖籍[[江蘇|江蘇]][[淮安|淮安]]；[[北京师范大学|北京师范大学]]中文系毕业，教授。1987年加入[[中国民主促进会|中国民主促进会]]，即被增补为第七届中央常委；随后，连任第八至十一届民进中央常委。曾任第九届民进中央副主席，第十、十一届民进中央主席；第九、第十届[[全国人大常委会|全国人大常委会]]副委员长。现任[[世界汉语大会|世界汉语教学学会]]会长、北京師範大學漢語文化學院院長、[[北京师范大学珠海分校|北京师范大学珠海分校]][[校监|校监]]及[[北京师范大学-香港浸会大学_联合国际学院|北京师范大学-香港浸会大学 联合国际学院]]校监。

== 简历 ==
許教授1959年於北京師範大學中文系畢業，並一直於母校任教。历任北京师范大学中文系助教、讲师、副教授、教授，教研室主任、中文系主任，北京师范大学副校长。他还曾先后担任中国国务院古籍整理出版规划小组成员，国家“八五”哲学社会科学规划语言学评审组组长，中国训诂学研究会会长等职。<ref>[http://news.xinhuanet.com/ziliao/2002-01/23/content_250857.htm 新华网：许嘉璐] {{webarchive|url=https://web.archive.org/web/20041117025533/http://news.xinhuanet.com/ziliao/2002-01/23/content_250857.htm |date=2004-11-17 }}</ref>

1993年曾應邀前往[[香港樹仁學院|香港樹仁學院]]中文系任教，其後轉往[[香港城市大學|香港城市大學]]中文、翻譯及語言學系出任客座教授。之後返回內地，1994年至1997年任中国民主促进会中央副主席、[[国家语言文字工作委员会|国家语言文字工作委员会]]主任。1996年4月19日，許教授再次回到樹仁學院，為學院新成立的普通話測試中心主持揭幕儀式。

1997年12月，許嘉璐当选[[民进|民进]]中央主席；并以此身份，在次年3月的九届人大一次会议上，当选[[全国人大常委会|全国人大常委会]]副委员长。此后，許嘉璐分别在2002年、2003年连任了这两项职务。2003年12月2日，許嘉璐獲[[香港浸會大學|香港浸會大學]]頒授榮譽文學博士，以表揚他對遠古中國著作和文化的貢獻<ref>{{cite web |url=http://www.hkbu.edu.hk/~ours/enews2003/20031203.htm |title=存档副本 |accessdate=2007-08-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20070930211006/http://www.hkbu.edu.hk/~ours/enews2003/20031203.htm |archivedate=2007-09-30 }} 香港浸会大学</ref>。

2005年，以许嘉璐为领导的民进中央，向[[中共中央|中共中央]]提交了“关于促进民族地区九年义务教育发展”、“推进民族地区双语教学”等四份建议。这些建议后来得到了中共总书记[[胡锦涛|胡锦涛]]的批示。2007年9月，《〈[[江澤民文選|江澤民文選]]〉成語典故》出版，收錄有百多個成語，許嘉璐為這本書作序<ref>[http://www.cns.hk:89/gn/news/2007/09-30/1040423.shtml 中國新聞網：《〈江澤民文選〉成語典故》將出版 許嘉璐作序] {{webarchive|url=https://web.archive.org/web/20080102112758/http://www.cns.hk:89/gn/news/2007/09-30/1040423.shtml |date=2008-01-02 }}</ref>。

2007年、2008年，許嘉璐分别卸任了民进中央主席和全国人大常委会副委员长的职务。

==学术活动及爭議==
許嘉璐主要从事[[古代汉语|古代汉语]]、[[训诂学|训诂学]]、[[音韵学|音韵学]]、《[[說文解字|说文]]》学、古代文化学、[[应用语言学|应用语言学]]等方面的教学和研究工作<ref>[http://www.pinghesy.com/data/2008/0312/article_2178.htm  平和书院：许嘉璐简介]</ref>。

有港独人士说，許嘉璐曾于1996年在[[香港大學|香港大學]]演說时，主張消滅中國的方言，並認為「方言是封建社會的產物」<ref>彭志銘，(2007年，〈香港母語教育的語言政治〉，《[[明報|明報]]》，2007年12月17日。</ref>，论证普通話教育就是愛國教育<ref>陳雲，2008年，《中文解毒》，第112頁，香港：天窗出版</ref>。此外，許嘉璐还被指包庇學術造假，容許多位從政府往學術界「退休」的前任高官，透過各種非法途徑剽竊其他人的學術論文、違反[[學術誠信|學術誠信]]<ref>《[[爭鳴|爭鳴]]》2006年10月號</ref>。

2009年3月，許嘉璐以世界汉语教学学会会长、[[叶圣陶研究会|叶圣陶研究会]]名誉会长的身份率团访问[[台湾|台湾]]，并出席了“第三届[[海峡两岸客家高峰论坛|海峡两岸客家高峰论坛]]”。因为卸任全国人大常委会副委员长的职务刚一年，有媒体评论，許的访问是两岸交流以来，“大陆赴台退休层级最高的国家领导人”<ref>[http://news.sina.com.cn/c/2009-03-22/025717455355.shtml 新浪网：全国人大常委会原副委员长许嘉璐抵台参会] 原载《[[新京报|新京报]]》2009年3月21日</ref>。
