'''许世铨'''（{{bd|？|||}}）籍贯不详，[[中华人民共和国|中华人民共和国]][[台湾问题|台湾问题]]专家。

==生平==
1964年，自[[国际关系学院|国际关系学院]]英语系国际新闻专业。1967年到1996年，历任[[新华社|新华社]]、[[光明日报|光明日报]]、[[人民日报|人民日报]]驻[[坦桑尼亚|坦桑尼亚]]、[[黎巴嫩|黎巴嫩]]、[[英国|英国]]、[[联合国|联合国]]记者，人民日报国际部主任。1996年11月，出任[[中国社会科学院台湾研究所|中国社会科学院台湾研究所]]所长。后来担任[[全国台湾研究会|全国台湾研究会]]常务副会长<ref>{{Cite web |url = http://www.taiwan.cn/plzhx/zhjzhl/tyzhj/xshq/200601/t20060116_226865.htm |title = 许世铨 |author =  |date = 2006-01-16 |publisher = 中国台湾网 |language =  |accessdate =  |quote =  }}</ref>。他是[[第十届全国政协委员|第十届全国政协委员]]<ref>{{Cite web |url =http://news.qq.com/a/20050303/000661.htm  |title = 两会特写：许世铨为台湾水果做广告:不比美国差 |author =  |date = 2005-03-03 |publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref>。
