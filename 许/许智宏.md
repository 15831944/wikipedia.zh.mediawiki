'''许智宏'''（{{bd|1942年|10月14日|catIdx=X许}}），[[江苏|江苏]][[无锡|无锡]]人，[[生物学家|生物学家]]、[[中国科学院院士|中国科学院院士]]<ref name=pku />。目前为北京大学现代农学院院长、[[北京大学生命科学学院|北京大学生命科学学院]]教授、[[中国科学院上海植物生理研究所|中国科学院上海植物生理研究所]][[研究员|研究员]]，兼任植物分子遗传国家重点实验室学术委员会主任<ref name=xdnxy /><ref>{{Cite web |url=http://www.sippe.ac.cn/cp1-6.asp |title=专家委员会 |publisher=中国科学院上海生命科学院植物生理生态研究所 |language=zh-hans |accessdate=2018-07-10 |deadurl=no |archiveurl=https://web.archive.org/web/20171115165603/http://www.sippe.ac.cn/cp1-6.asp |archivedate=2017-11-15 |df= }}</ref><ref name=jyw />。曾任[[中国科学院|中国科学院]]副院长、[[中国细胞生物学会|中国细胞生物学会]]理事长、[[中国植物学会|中国植物学会]]副理事长、[[中国生物工程学会|中国生物工程学会]]副理事长<ref name=jyw>{{Cite web |url=http://www.agrogene.cn/guandian/viewpoint-107.shtml |title=许智宏 |publisher=基因农业网 |language=zh-hans |accessdate=2018-07-08 |deadurl=no |archiveurl=https://web.archive.org/web/20150523015103/http://www.agrogene.cn/guandian/viewpoint-107.shtml |archivedate=2015-05-23 |df= }}</ref>。

许智宏教授长期从事植物[[发育生物学|发育生物学]]、植物[[细胞培养|细胞培养]]及其遗传操作、[[植物生物工程|植物生物工程]]的研究。在植物发育、组织和细胞培养以及[[生物工程|生物工程]]领域，已发表论文、综述、专著共200多篇（册）。

== 生平 ==
1965年，许智宏毕业于[[北京大学生物系|北京大学生物系]][[植物学|植物学]]专业。随后考上了[[中国科学院上海植物生理研究所|中国科学院上海植物生理研究所]]研究生，毕业后留在该所长期工作<ref name=zgjs>{{Cite news|url=http://jsnews.jschina.com.cn/jsyw/201705/t20170523_542518.shtml|title=北大原校长许智宏在无锡与母校师生交流 把喜欢的事做到极致就是成功|publisher=中国江苏网|author=马薇|language=zh-hans|accessdate=2018-07-08|date=2017-05-23|deadurl=no|archiveurl=https://web.archive.org/web/20180708112259/http://jsnews.jschina.com.cn/jsyw/201705/t20170523_542518.shtml|archivedate=2018-07-08|df=}}</ref>。1979年至1981年，先后在英国[[约翰依奈斯研究所|约翰依奈斯研究所]]和[[诺丁汉大学|诺丁汉大学]]从事研究工作。从1983年起，开始担任上海植物生理研究所副所长、所长兼[[植物分子遗传国家重点实验室|植物分子遗传国家重点实验室]]主任的职务。1992年10月至2003年2月，任[[中国科学院|中国科学院]]副院长一职。1994年，获[[香港大学|香港大学]]荣誉教授称号<ref name=pku>{{Cite web |url=http://www.bio.pku.edu.cn/teacher_dis_oa.php?cid=146&teaid=88 |title=许智宏 |publisher=[[北京大学生命科学学院|北京大学生命科学学院]] |language=zh-hans |accessdate=2018-07-08 |deadurl=no |archiveurl=https://web.archive.org/web/20180708112259/http://www.bio.pku.edu.cn/teacher_dis_oa.php?cid=146&teaid=88 |archivedate=2018-07-08 |df= }}</ref>。1995年和1997年10月分别当选为[[第三世界科学院|第三世界科学院]]院士和[[中国科学院院士|中国科学院院士]]。1999年12月至2008年11月任[[北京大学|北京大学]]校长<ref>{{Cite web|url=http://www.pku.edu.cn/about/lrld/lrxz/index.htm|title=北大概况 - 历任校长|publisher=[[北京大学|北京大学]]|place=[[北京|北京]]|accessdate=2018-07-08|language=zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180708170924/http://www.pku.edu.cn/about/lrld/lrxz/index.htm|archivedate=2018-07-08|df=}}</ref>。在任校长九年期间，创建北京大学为世界一流大学的计划被正式确定为国家战略，原北京大学与原[[北京医科大学|北京医科大学]]合并，被认为迎来了建校以来最好的发展时期<ref>{{Cite web |url=http://www.xdf.cn/zhuanti/xuzhihong.html |title=教育名人坊之许智宏 |accessdate=2018-07-10 |publisher=[[新东方学校|新东方]] |language=zh-hans |deadurl=no |archiveurl=https://web.archive.org/web/20161010001334/http://www.xdf.cn/zhuanti/xuzhihong.html |archivedate=2016-10-10 |df= }}</ref>。2006年，曾任第十一届国际组织培养与生物技术大会主席<ref>{{Cite news |url=http://genetics.cas.cn/xwzx/zhxw/200908/t20090821_2431157.html |title=第十一届国际组织培养与生物技术大会在京召开 |publisher=中国科学院遗传与发育生物学研究所 |language=zh-hans |accessdate=2018-07-10 |date=2009-08-21 |deadurl=no |archiveurl=https://web.archive.org/web/20180710144330/http://genetics.cas.cn/xwzx/zhxw/200908/t20090821_2431157.html |archivedate=2018-07-10 |df= }}</ref>。2012年6月，任中国科学院学部第五届科学道德建设委员会主席<ref>{{Cite web |url=http://www.casad.ac.cn/chnl/300/index.html |title=学部科学道德建设委员会 |publisher=[[中国科学院|中国科学院]]学部 |language=zh-hans |accessdate=2018-07-10 |deadurl=no |archiveurl=https://web.archive.org/web/20180710144327/http://www.casad.ac.cn/chnl/300/index.html |archivedate=2018-07-10 |df= }}</ref>。

2018年，北京大学现代农学院成立。[[北京大学|北京大学]]经研究决定，聘任许智宏为现代农学院院长<ref name=xdnxy>{{Cite news |url=https://www.thepaper.cn/newsDetail_forward_1974105 |title=中科院院士许智宏任北京大学现代农学院首任院长 |author=北京大学现代农学院 |date=2018-01-29 |accessdate=2018-07-08 |language=zh-hans |publisher=澎湃新闻 |deadurl=no |archiveurl=https://web.archive.org/web/20180316001544/http://www.thepaper.cn/newsDetail_forward_1974105 |archivedate=2018-03-16 |df= }}</ref>。

== 生活 ==
在许智宏年少时，就对大自然产生了浓厚的兴趣，经常与小伙伴们出门玩耍，例如爬树、抓虫子、钓金鱼等。因此，在许智宏回到母校无锡市辅仁高级中学时回忆道如何考上[[北京大学|北京大学]]时，戏称“考北大生物系完全是（小时候）‘玩出来’的”<ref>{{Cite news |url=http://www.sohu.com/a/142588321_507487 |title=中科院院士、北大原校长许智宏回母校辅仁高中与学子交流 |date=2017-05-22 |accessdate=2018-07-10 |language=zh-hans |author=无锡教育 |publisher=[[搜狐网|搜狐新闻]] |deadurl=no |archiveurl=https://web.archive.org/web/20180710110337/http://www.sohu.com/a/142588321_507487 |archivedate=2018-07-10 |df= }}</ref>。在许智宏开始从事教学工作后，他不断地向学生们强调对兴趣的专注在学习研究中十分重要。哪怕是有[[北京大学|北京大学]]的毕业生选择成为一名猪倌，许智宏也会支持他们的选择。许智宏认为，只要是真心喜欢某件事情，并专心地把它做到极致，一样可以获得巨大的成功<ref name=zgjs />。许智宏还认为，[[生物多样性|生物多样性]]的研究不分国界，要注重[[东南亚|东南亚]]青年人才的培养，不断强化科研力量<ref>{{Cite news |url=http://www.xtbg.cas.cn/xwzx/zhxw/201802/t20180205_4944067.html |title=东南亚中心2018年学术年会暨中缅生物多样性保护研究合作研讨会闭幕 |date=2018-02-05 |publisher=[[中国科学院|中国科学院]]西双版纳热带植物园 |accessdate=2018-07-08 |language=zh-hans |deadurl=no |archiveurl=https://web.archive.org/web/20180708112842/http://www.xtbg.cas.cn/xwzx/zhxw/201802/t20180205_4944067.html |archivedate=2018-07-08 |df= }}</ref>。
