'''许昌文峰塔'''，又称'''文明塔'''，'''文明寺塔'''，是位于[[河南省|河南省]][[许昌市|许昌市]]的[[文峰塔|文峰塔]]。2006年5月，文峰塔作为[[明代|明代]]古建筑，被[[国务院|国务院]]列入第六批[[全国重点文物保护单位|全国重点文物保护单位]]名单中<ref name="简介">{{cite web |url=http://xuchang.lotour.com/xcwenfengta/lvyoujianjie/ |language= |title=许昌文峰塔简介 |publisher=乐途网 |date=2009-11-16 |accessdate= }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 简介 ==
文峰塔于明[[万历|万历]]42年（公元1614年）由许州[[太守|太守]]郑振光建造。13层高52米，由基台、塔基、塔身、塔刹四部分组成。呈平面八角形，造型精美、结构谨严，为河南省200座明代'''砖塔'''之冠，并有“中国风水第一塔”的美誉<ref>{{cite web|url=http://cathay.ce.cn/pieces/200811/28/t20081128_17525082.shtml | language= | title="中国风水第一塔"许昌文峰塔开始纠偏(图) |publisher=华夏文明 |date=2008年11月28日 |accessdate= }}</ref>。<ref name= "简介"/>塔前立有三座[[石碑|石碑]]，分别为：明万历四十三年《文明塔赋》，清[[康熙|康熙]]五十八年《重修文明塔寺记》和清[[嘉庆|嘉庆]]十四年《重修文明寺释迦文佛殿鳌头观音殿碑记》。<ref>{{cite web |url=http://www.xcswhj.gov.cn/html/2007-6/200763163722.htm |language= |title=文峰塔 |publisher=许昌市文化新闻出版局 |date= |accessdate=2012年10月 |deadurl=yes |archiveurl=https://web.archive.org/web/20130406210453/http://www.xcswhj.gov.cn/html/2007-6/200763163722.htm |archivedate=2013-04-06 }}</ref>

目前它位于[[许昌市博物馆|许昌市博物馆]]东院，"'''文峰耸秀'''"为许昌十景之一 。<ref name= "简介"/>

近年来文峰塔出现了倾斜加速的现象。<ref>{{cite web |url=http://cul.shangdu.com/fanggu/20090225-18034/index.shtml |language= |title=许昌文峰塔 |publisher=大河网 |date=2009-02-25 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20130512172026/http://cul.shangdu.com/fanggu/20090225-18034/index.shtml |archivedate=2013-05-12 }}</ref>
