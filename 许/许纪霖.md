'''许纪霖'''{{BD|1957年||||catIdx=X许}}出生于[[上海市|上海市]]，毕业于[[华东师范大学|华东师范大学]]政治教育系，为华东师范大学教授、[[上海历史学会|上海历史学会]]成员、[[公共知识分子|公共知识分子]]。<ref name=" lio" >{{cite web |url= http://news.qq.com/a/20090828/001690.htm|title=许纪霖简介 |accessdate=	2009年8月28日 |publisher=腾讯网 |archiveurl= http://news.qq.com/a/20090828/001690.htm|archivedate= 2012年11月6日|work= |language =zh-cn}}</ref>

== 生平 ==
1957年，许纪霖出生在上海市，[[文革|文革]]爆发后中断学习，到上海[[南汇县|南汇县]]东海农场任图书管理员。恢复高考制度之后考入[[华东师范大学|华东师范大学]]政治教育系，毕业后留校任教。<ref name=" lio" />曾先后担任过[[香港中文大学|香港中文大学]]中国文化研究所、[[澳大利亚国立大学|澳大利亚国立大学]]亚洲太平洋研究院、[[新加坡国立大学|新加坡国立大学]]东亚研究所、[[哈佛大学|哈佛大学]][[哈佛燕京学社|哈佛燕京学社]]、台湾[[中央研究院|中央研究院]]人文社会科学研究中心、加拿大[[不列颠哥伦比亚大学|不列颠哥伦比亚大学]]亚洲研究所的访问研究员。
