'''淮安涟水机场'''（{{airport codes|HIA|ZSSH}}）是一座位于[[江苏省|江苏省]][[淮安市|淮安市]]的民航支线机场。

== 概况 ==
涟水机场位于[[淮安市|淮安市]]北郊，距市中心直线距离22公里。是国家“[[十一五|十一五]]”规划建设的重点支线机场。涟水机场是按国内民航支线机场一次规划、分期建设，飞行区近期等级为4C，兼顾D类。涟水机场于2010年9月26日投入使用。

== 航站楼 ==
涟水机场航站楼共2层，总面积7200平方米。分国内候机楼和港澳台及国际候机楼两部分。国内候机楼于2010年9月26日投入使用，港澳台及国际候机楼于2012年12月23日投入使用。

== 其他 ==
2011年4月29日，涟水机场成为[[中国飞龙专业航空|中国飞龙专业航空]]飞行员培训基地。

== 周边交通 ==
每天有定期机场专线巴士往返与涟水机场和淮安市区之间。
机场连接线从机场航站楼广场出口，向东南方向连接[[长深高速|长深高速]]公路、[[京沪高速|京沪高速]]公路。
