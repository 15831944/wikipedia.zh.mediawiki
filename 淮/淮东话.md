'''淮东话'''是[[淮语|淮语]][[洪巢片|洪巢片]]中的一小片，即《江苏省志·方言志》中所说的'''淮扬方言'''，大致包括[[江苏省|江苏省]][[淮安市|淮安市]]、[[扬州市|扬州市]]、[[盐城市|盐城市]]、[[镇江市|镇江市]]，相当于江苏省内除了[[南京|南京]]之外的[[洪巢片|洪巢片]]地区加上[[安徽省|安徽省]][[天长市|天长市]]（古属[[淮扬|淮扬]]地区），以[[扬州话|扬州话]]为代表方言。

== 语音特点 ==
*不分平翘舌音，其中[[扬州|扬州]]、[[淮安|淮安]]、[[盐城|盐城]]、[[镇江|镇江]]都读平舌音，[[海州|海州]]都读翘舌音；
*不分n、l，除[[盐城|盐城]]市区和[[建湖县|建湖县]]及[[扬州|扬州]]市郊外；
*中古止摄端组合口字（如退、对)；
*中古咸、山摄字在按主要元音分成三个部分，如关≠官、简≠剪等，与《[[中原音韵|中原音韵]]》寒山、桓欢、先天三韵的分法相同；
*中古假摄麻韵开口三等字，如爹、借、些等，在大部分地区，韵母为iI，与蟹摄见组二等字不同（如街、蟹等）；
*全區[[入声|入声]]韵均与阳声韵相配，与《[[广韵|广韵]]》的原则相同。入聲不分陰陽；
*相对于普通话而言，'''淮东话'''比[[淮西话|淮西话]]要难懂。

== 内部分区 ==
按语音及词汇特点，内部又可分为[[淮扬片|淮扬片]]、[[海州|海州]]片两个小片。

== 地方戏 ==
流行于此区域的地方戏，包括[[淮剧|淮剧]]、[[扬剧|扬剧]]等。[[淮剧|淮剧]]流行于[[淮安|淮安]]和[[盐城|盐城]]，并在上海市的苏北人中有相当影响，[[扬剧|扬剧]]流行于[[扬州|扬州]]。[[海州方言|海州方言]]为基础的[[淮海戏|淮海戏]]流行于淮河以北的[[海州|海州]]，[[泗阳|泗阳]]地区。
