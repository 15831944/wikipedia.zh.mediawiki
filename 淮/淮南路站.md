'''淮南路站'''，为[[中国|中国]][[安徽省|安徽省]][[合肥市|合肥市]]的一座[[地鐵車站|地铁车站]]，位于老淮南铁路与[[临泉路|临泉路]]交叉口，为[[合肥轨道交通3号线|合肥轨道交通3号线]]站点。<ref>{{cite web |title="图书馆"易误解 合肥市民为地铁3号线站名“挑刺”|url=http://www.ahwang.cn/hefei/20190125/1848031.shtml |publisher=安徽网 |accessdate=2019-01-25}}</ref>

== 车站结构 ==
[[淮南路站|淮南路站]]位于老淮南铁路与临泉路交叉口，沿临泉路东西向布置，为地下两层单柱双跨岛式站台。共设4个出入口,设2组风亭。<ref>{{cite web |title=合肥地铁3号线沿线站点公布！开通运营时间定在 |url=http://wemedia.ifeng.com/44596356/wemedia.shtml |publisher=凤凰网安徽 |accessdate=2019-02-01}}</ref>
