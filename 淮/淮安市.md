'''淮安市'''是[[中华人民共和国|中华人民共和国]][[江苏省|江苏省]]下辖的[[地级市|地级市]]，位于江苏省中部，[[南京都市圈|南京都市圈]]紧密圈层城市，[[淮河|淮河]]与[[京杭大运河|京杭大运河]]交点，属于[[江淮|江淮]][[淮扬|淮扬]]文化，自古为水运及[[漕运|漕运]]枢纽、南北要冲，明清时期与[[苏州|苏州]]、[[杭州|杭州]]、[[扬州|扬州]]并称为[[大运河|大运河]]沿线四大都市，有“'''运河之都'''”“九省通衢”之称<ref>[http://wenhua.eco.gov.cn/3/1/2/3/2009/0703/134780.html 千年漕运：留给江苏的丰厚积淀.]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是[[国家历史文化名城|国家历史文化名城]]。

== 地理 ==
淮安市境内绝大部分为辽阔的江淮平原（[[里下河平原|里下河平原]]），地势极为平坦。唯有[[盱眙县|盱眙县]]有若干[[丘陵|丘陵]]地形分布，铁山寺森林公园的海拔最高点接近200米。

淮安市河流湖泊众多，主要河流有[[京杭大运河|京杭大运河]]、[[淮河|淮河]]。西部濒临中国第四大淡水湖[[洪泽湖|洪泽湖]]，南部则有[[高邮湖|高邮湖]]、白马湖等。

由于地跨淮河两岸，淮安民风刚柔并济，人才文武兼备。除了淮阴区，涟水县其余大部分地区位于淮河以南（淮扬文化）。

2008年，淮安在市区北部的[[淮河|淮河]]河道上建筑“''中国南北地理分界线标志物''”。

== 历史 ==
{{Main|淮安历史}}

===秦汉===
相傳[[大禹|大禹]]曾在境內治水，「使淮水永安」。

公元前486年，吴王[[夫差|夫差]]为北上进取[[中原|中原]]开凿了邗城（今扬州）至末口（今淮安河下）的[[邗沟|邗沟]]，沟通淮河和长江，造就了秦汉时淮安的第一次繁荣。

淮安[[秦|秦]]時置縣，至今2200多年歷史。该地区那时设立了[[淮阴县|淮阴县]](今[[淮阴区|淮阴区]][[码头镇|码头镇]])， [[盱眙县|盱眙县]]和[[东阳县_(秦朝)|东阳县]](今安徽省[[天长市|天长市]]西北到盱眙县[[马坝镇_(盱眙县)|马坝]])。[[西汉|西汉]]又置[[淮浦县|淮浦县]]（今[[涟水县|涟水县]]）等县。
淮阴城正对泗水入淮口，东面的末口则是连接淮河与长江运河的起点.具有重要的交通位置。

秦汉时期，这里出过不少名人:辅佐[[刘邦|刘邦]]建立汉朝的大将[[韩信|韩信]]；[[枚乘|枚乘]]，枚皋和陈琳([[建安七子|建安七子]]之一)。

=== 魏晋南北朝===

淮安处于南北对峙前线，长期战争带来“千里长淮，赤地千里”的景象。

晋代末年，[[祖逖|祖逖]]筑山阳城，即今日所谓老城，四面各1600米。

南齐永明7年（公元489年），“割直渎、破釜以东，淮阴镇下流杂100户建[[淮安|淮安]]县。

这一时期，最优秀的诗人[[鲍照|鲍照]]是涟水人。

===隋唐===
[[File:文通塔.JPG|thumb]]
[[隋朝|隋朝]][[开皇|开皇]]元年（581年）置[[楚州_(隋朝)|楚州]]，治寿张县（后改[[淮阴县|淮阴县]]，今江苏'''淮安市'''[[淮阴区|淮阴区]]西南），十二年移治[[山阳县_(江苏省)|山阳县]]（今江苏'''淮安市'''[[淮安区|淮安区]]）。大业初废。

唐武德八年（625年）又改东楚州置。仍治山阳县。辖境相当今江苏省淮河以南，[[盱眙县|盱眙县]]以东，[[宝应县|宝应县]]、[[盐城市|盐城市]]以北地区。天宝、至德时曾改为[[淮阴郡|淮阴郡]]。

北宋属[[淮南东路|淮南东路]]。辖境略有缩小。

南宋绍定元年（1228年）改为[[淮安军|淮安军]]。

隋初，[[大运河|大运河]]开凿。

唐初，淮北盐场的开滩，[[涟水|涟水]]成为全国四大盐场之一。

楚州（治山阳，今淮安市淮安区）、泗州（治临淮，在今洪泽湖底）成为运河线上两座名城，其中楚州更是东南仅次于扬州的都会，被诗人[[白居易|白居易]]赞为“淮水东南第一州”

唐朝时，日本船可由[[涟水|涟水]]（今江苏[[淮安|淮安]]）出淮河口沿[[黄海|黄海]]北上，东渡黄海经[[朝鲜半岛|朝鲜半岛]]到[[日本|日本]]。

=== 宋元 ===
淮水是南宋与金国分界线。

[[南宋|南宋]]绍兴年间，[[韩世忠|韩世忠]]与夫人[[梁红玉|梁红玉]]在楚州抗金，前后十年，兵仅三万，而金人不敢犯。

南宋，金兵南征，淮安频遭战祸，人烟凋敝，遂改楚州为淮安州，元代再改为淮安路，治所仍然是山阳。

===明清===
明清时期，[[淮安|淮安]]是河、漕、盐东南三大政的枢纽重地，朝廷在此设立[[南河总督|南河]]、[[漕运总督|漕运]]两位总督，城市繁荣达于极盛，府城、[[河下|河下]]镇、板闸镇和[[清江浦|清江浦]]组成绵延数十里的巨大带形城市；與[[揚州|揚州]]、[[蘇州|蘇州]]、[[杭州|杭州]]並稱為[[京杭大运河|京杭大运河]]沿線的「四大都市」。

明代[[淮安府|淮安府]]是今江苏长江以北仅有的2个府之一，辖区范围基本上相当于今日淮安、盐城、宿迁、连云港4市以及徐州市东部。清代淮安府分出海州直隶州，又划出一部分给新设的徐州府，范围有所收缩，下辖6县：山阳（今淮安区）、清河（今淮阴区）、安东（今[[涟水县|涟水]]）、桃源（今泗阳）、[[盐城|盐城]]、[[阜宁|阜宁]]（分山阳县东部新设）。

[[File:淮安府大堂.JPG|thumb]]

1761年，因清河县城毁于洪水，将山阳县的[[清江浦镇|清江浦镇]]划给清河作为新县治。

===中华民国===
1914年，这一地区的几个县都因为有重名县而改名：山阳县改为淮安县，清河县改为淮阴县，安东县改为[[涟水县|涟水县]]，桃源县改为泗阳县。

民国建立后，撤销府级行政机构，保留了[[道|道]]。[[清江浦|清江浦]]是[[淮扬道|淮扬道]]的治所，从此正式取代淮安府城成为行政中心。淮扬道包括原来的淮安府和[[扬州府|扬州府]]，共辖13个县：淮阴县、淮安县（今淮安区）、涟水县、盐城县、阜宁县、泗阳县、[[江都县|江都县]]、[[东台县|东台县]]、[[高邮县|高邮县]]、[[仪征|仪征]]县、[[宝应县|宝应县]]、[[泰县|泰县]]、[[兴化县|兴化县]]。

1937年，[[抗日战争|抗日战争]]爆发，[[江苏省政府|江苏省政府]]从[[镇江|镇江]]迁来，直到1939年3月被日军攻占。

=== 中华人民共和国===

====清江市====
*1948年12月，中共军队在[[淮海战役|淮海战役]]末期占领淮阴和淮安。次年4月21日建立属于[[苏北行政公署|苏北行政公署]]管辖的[[淮阴专区|淮阴专区]]，包括10个县:[[淮阴县|淮阴]]、[[淮宝县|淮宝]]、[[泗阳县|泗阳]]、[[沭阳县|沭阳]]、[[灌云县|灌云]]、[[宿迁县|宿迁]]、[[睢宁县|睢宁]]、[[新安县_(解放区)|新安]]、[[邳睢县|邳睢]]和[[涟水县|涟水]]。地区公署设淮阴县。
*1950年12月18日，分淮阴县城区设立[[清江市|清江市]]。地区公署设清江市。
*1950年代淮阴专区行政上的变化：
#睢宁，邳睢和新沂3县划归[[徐州专区|徐州专区]]；
#从[[盐城专区|盐城专区]]划来[[淮安县|淮安县]]；
#从[[安徽省|安徽省]][[宿县地区|宿县地区]]划来[[泗洪县|泗洪县]]，从安徽省[[滁县地区|滁县地区]]划来[[盱眙县|盱眙县]]；
#1956年分淮阴，泗洪和盱眙三县设[[洪泽县|洪泽县]]，治[[高良涧镇|高良涧镇]]； 
#1957年又分灌云和涟水县设[[灌南县|灌南县]]；
#1958年，[[淮阴县|淮阴县]]和清江市一度合并为淮阴市。1964年，又恢复原状。

*1966年，淮阴地区管辖1个市（清江市）和12个县：灌云，灌南、沭阳、宿迁、泗阳、涟水、淮阴、淮安、洪泽、泗洪、盱眙和金湖。
*1975年，淮阴县迁治王营镇。

====淮阴市====
1983年，清江市再次更名为淮阴市，升为[[地级市|地级市]]，管辖[[清河|清河]]、[[清浦|清浦]]2个区和11个县：灌南、沭阳、宿迁、泗阳、涟水、淮阴、泗洪、淮安、洪泽、盱眙和金湖，灌云县划归连云港市。

1987年，宿迁和淮安2县改设县级市。

1996年，[[宿迁市|宿迁市]]升为地级市，管辖沭阳、泗阳和泗洪；灌南县划归[[连云港市|连云港市]]。淮阴市还有5个县：淮阴、涟水、金湖、洪泽和盱眙，淮安市，以及清河和清浦2个区。

====淮安市====
2001年2月10日，淮阴市更名为淮安市，仍驻清河区；原县级淮安市改设[[楚州区|楚州区]]，驻淮城镇；原淮阴县改设[[淮阴区|淮阴区]]，驻王营镇。

2012年1月31日，楚州区更名为淮安市[[淮安区|淮安区]]。

2016年6月8日，撤销淮安市[[清河区|清河区]]、[[清浦区|清浦区]]，设立淮安市[[清江浦区|清江浦区]]；撤销[[洪泽县|洪泽县]]，设立淮安市[[洪泽区|洪泽区]]。

== 经济 ==
2000年以后，韩国资本[[韩泰轮胎|韩泰轮胎]]进驻淮安。2007年以后，台资[[富士康|富士康]]、达方电子、宏盛箱包等企业陆续来此投资建厂，使得该市成为台资新的聚集区<ref>[http://news.xinhuanet.com/newscenter/2008-01/02/content_7354103.htm 江苏淮安成为台资新的聚集区]</ref>。

2011年，全市完成地区生产总值1690亿元；人均地区生产总值35181元(按常住人口)，按汇率折算为5447美元，略高于全国平均水平。城镇居民人均可支配收入20260元，农民人均纯收入8645元，经济总量和人均经济水平在全国及[[苏北|苏北]]地区居于中上等水平。

== 文化 ==
* [[中国历史文化名城|中国历史文化名城]]
淮安是运河之都，长期是州、郡、府、路的治所，是淮扬菜系的发源地，人文荟萃，名家辈出，历史上几度繁华，因此列为第二批[[中国历史文化名城|中国历史文化名城]]。

淮安古城构造独特，[[东晋|东晋]][[祖逖|祖逖]]所筑之老城，四面各1600米；北宋时又在老城以北1000米处筑新城，四面各长1000米；明代为防[[倭寇|倭寇]]，又在两城间筑联城（夹城），只有东西两面城墙，各长700米，老城、新城、联城三城并列，气势恢弘，在中国古代城池中独树一帜。

老城是三城核心，城墙高十米，有四座城门楼，三座角楼和三座水门。城内街巷纵横，市坊密布，城中央为镇淮楼。
*[[运河|运河]]之都
[[京杭运河|京杭运河]]淮安段，在春秋时最早开凿，在历史上发挥着五大中心作用，即漕运指挥中心、漕船制造中心、粮食储备中心、淮北食盐集散中心、河道治理中心，因此称[[运河|运河]]之都。

* [[淮扬菜|淮扬菜]]之乡
由于在历史上的重要地位，淮安成为淮扬菜的主要发源地之一。

明万历年间，淮安府志载：淮安饮食华奢，制度精巧，市肆百品，夸视江表。

清康熙年间，淮安府志载：涉江以北，宴会珍错之盛，淮安为最。民间或宴贵客，陈设方丈，伎乐杂陈，珍氏百味，一宴费数金。

著名菜点有：平桥豆腐、文楼汤包、[[蟹黄汤包|蟹黄汤包]]、钦工肉圆等；淮安特别以制作长鱼（[[黄鳝|黄鳝]]）见长，有所谓全鳝席，包括软兜长鱼、生炒蝴蝶片、[[炝虎尾|炝虎尾]]、大烧马鞍桥、[[煨脐门|煨脐门]]等菜肴。

著名特产有：涟水萝卜干、涟水鸡糕、高沟捆蹄、淮安茶馓、洪泽湖大闸蟹、盱眙龙虾等。

* [[淮剧|淮剧]]
淮剧是源自中国江苏省江淮地区的地方戏剧，主要流行于盐城、淮安、泰州、扬州等地，在上海、安徽等省市，中国以外如台湾也有一定的影响。唱腔朴实。淮剧的方言是以江淮官话洪巢片建鹽方言中建湖话音系为基础。

淮剧已经形成如下20个韵部：

四声韵部14个，即爬沙、婆娑、图书、愁收、乔梢、开怀、齐西、谈山、田仙、辰生、琴心、垂灰、常商、蓬松。实际运用中，通常将“辰生”和“琴心”混合使用。14个四声韵中，阴平、阳平字因在唱词中专用于下句，故称为下韵；上声、去声字专用于上句，则称之为上韵。
入声韵部6个，即霍托、活泼、六足、黑特、邋遢、锡铁。入声不分上、下，习惯称之为“一字韵”。
目前有16个公有淮剧团，分布在江苏省盐城、淮安、泰州、扬州、建湖、阜宁、射阳、滨海、大丰、涟水、兴化、泰兴、宝应等市、县。以及上海市。江苏省淮剧团设于盐城市。

===宗教===
在淮安市的宗教场所和可统计的宗教信徒中，[[基督新教|基督新教]]占有绝对的优势（800处，人数18万）；其次为[[佛教|佛教]]（40处、4万）和[[伊斯兰教|伊斯兰教]]（5处、6000人），而天主教信徒只剩下38人。清末至民国年间，淮安曾是[[美南长老会差会|美南长老会差会]]重要的传教基地，著名传教士包括[[赛珍珠|赛珍珠]]之父[[赛兆祥|赛兆祥]]和[[葛培理|葛培理]]的岳父[[钟爱华|钟爱华]]。1990年代以后，市内陆续兴建多座可容纳数千人的大型教堂，包括建筑面积3685平方米的[[淮安市基督教堂|淮安市基督教堂]]和建筑面积10100平方米的[[神恩堂|神恩堂]]，后者据称是目前亚洲最大的基督教教堂<ref>{{cite web |url=http://jsnews.jschina.com.cn/system/2011/10/23/011921596.shtml|title=清浦区基督教神恩堂|accessdate=2011-10-23|work=|publisher=淮安新闻网}}</ref>。淮安开放的著名佛教寺院有[[慈云寺_(淮安)|慈云寺]]、闻思寺、能仁寺(涟水)等。

===方言===
境内方言属[[江淮官话|江淮官话]]洪巢片，淮安方言为[[江苏省|江苏省]]淮（安）扬（州）方言的代表点。

== 旅游 ==
[[File:漕运总督衙门旧址.JPG|thumb]]
* [[清晏园|清晏园]]
清晏园是位于淮安市清浦区人民南路西侧、环城西路北侧的一座古典园林。曾用名淮园、瞻园、留园、城南公园，是中国漕运史上唯一存留至今的官宦园林。可以说是淮安风景最优美的园林。现为淮安市文物保护单位。

清晏园原是明代户部分司公署旧址，清康熙十七年（1678年），河道总督靳辅驻节清江浦，在此凿池种树，设立行馆。雍正七年（1729年），正式建立江南河道总督署，经过扩建，具备园林规模。

民国六年（1917年），李兰轩集资修园，名城南公园。1991年，恢复清晏园古名。2003年3月26日，被列入第二批淮安市文物保护单位名单并公布。[1]

其中，荷芳书院，建于清乾隆十五年（1750年），系南河总督高斌为迎接乾隆帝南巡，在园内荷池北侧兴建，是园内唯一幸存的古建筑。周围的碑亭和碑廊，反映当年的治水历史。其他如湛亭、淮香堂、蕉吟馆、今来雨轩、曲廊、水榭、船楼、曲桥和谦豫斋、关帝庙、叶园都属于仿古建筑。另有数座大型假山。

== 教育 ==
淮安拥有数所高等学校。其中淮阴工学院、淮安信息职业技术学院、江苏食品职业技术学院、江苏财经职业技术学院等均位于城南的高教园区（当地俗称“大学城”）；淮阴师范学院位于城北淮阴区境内。

淮安拥有7所国家级示范高中，包括[[江苏省淮阴中学|江苏省淮阴中学]]（前身是江北大学堂，创建于1902年）、[[江苏省清江中学|江苏省清江中学]]、[[江苏省淮州中学|江苏省淮州中学]]、江苏省淮安中学、江苏省盱眙中学、江苏省金湖中学、江苏省涟水中学；江苏省重点中学包括淮安市清浦中学（创建于1952年，曾用名城南民中、东方红中学）、淮阴师范学院附属中学、洪泽县中学、洪泽县第二中学、楚州中学、淮海中学、盱眙县马坝中学、金湖县第二中学、涟水县郑梁梅高级中学等。

淮安的重点小学包括淮安市实验小学（原长征小学，创办于1908年）、淮阴师范学院第一附属小学（原淮阴师范学院附属小学，创办于1959年）、淮安市人民小学（原人和小学，创办于1940年）等。

2009年，全市共有各类学校986所，在校生97.4万人，教职工6万人。其中：各类成规模幼儿园271所，在园幼儿15.2万人；小学455所，在校生30.6万人；普通中学204所，在校生32.1万人；中等职业学校43所，在校学生11.3万人；普通本科院校2所，高等职业技术学院4所，在校生7.9万人；特殊教育学校7所，在校生0.2万人。
