'''海山神社'''位於今臺灣[[新北市|新北市]][[中和區|中和區]]員山公園內，創設時屬於臺北州海山郡中和-{庄}-外員山，社格是無格社，1944年升為鄉社，主祭明治天皇、大己貴命與北白川宮能久親王，維持經費主要來自官有地的收入{{r|海}}。其殘蹟於2005年9月13日與附近的防空壕、積穗配水池一起以「[[海山神社殘蹟|海山神社殘蹟]]」之名被指定為古蹟{{r|文}}。

== 沿革 ==
海山神社建於1938年，原定興建於海山郡役所所在的[[板橋街|板橋街]]，但因板橋街內無合適地點，故將地點改為鄰近的中和庄圓山墓地，與海山公園的闢建工程一起進行{{r|海}}，於昭和十三年（[[1938年臺灣|1938年]]）5月13日舉行鎮座式{{r|社}}。二次大戰後，神社即遭到廢棄，約在1950年代便被破壞，現在僅存階梯與第三鳥居柱礎等殘蹟{{r|海}}。

== 建築 ==
海山神社座南朝北，與一般的神社方向不同，主要是要用以體現庇護北邊的中和、板橋居民而設計{{r|海}}。在進入第一鳥居後，為參道與神橋，步上一道階梯後為第二鳥居，之後在參道旁有狛犬、手水舍、社務所、祭器庫，再步上一道階梯（現存者）後即第三鳥居，後面是拜殿與主殿{{r|海}}。

而在建築樣式上，海山神社的主殿與拜殿採用神明造，祭器庫是校倉造，社務所與[[桃園神社|桃園神社]]相似，入口為唐破風，鳥居則使用神明鳥居的型式，狛犬為唐獅子{{r|海}}。

== 祭典 ==
海山神社最主要的祭典是在11月3日的新穀感謝祭，會由當地警風團抬神轎遊街，此外海山郡役所也會舉辦如摔跤、劍道比賽等活動，而除此之外在元旦及其他節慶時也會舉行慶典{{r|海}}。
