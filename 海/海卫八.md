'''海卫八'''（'''S/1989 N 1'''，'''Proteus'''）是环绕[[海王星|海王星]]运行的第二大的[[卫星|卫星]]。 它的大小已经接近不规则星体的极限，也是海王星的最大[[內衛星|內衛星]]。其[[英文|英文]]名字「Proteus」（[[普罗透斯|普罗透斯]]）源于[[希腊神话|希腊神话]]中会变形的海神。

== 发现 ==
海卫八是由[[旅行者2号|旅行者2号]]在1989年飞经海王星所摄取的图像而发现的。 它当时被暂命名为'''S/1989 N 1'''。和于1989年7月7日的[[IAUC|IAUC]] 4806会议上宣布了海卫八的发现。根据公布说明的“21天中获取的17幅图像”来推算，海卫八的发现日期是6月16日略前。海卫八在1991年9月16日被正式命名。<ref>{{cite web
|title = IAU Circular No. 5347
|date = September 16, 1991
|url = http://www.cfa.harvard.edu/iauc/05300/05347.html
|accessdate = 2007-04-10 }}</ref>

== 表面特徵 ==
[[File:Proteus_map_(Stooke).jpg|thumb]]
海卫八的直径超过400千米，比[[海卫二|海卫二]]大。但是因为海卫八非常靠近行星，容易被其反射太阳光掩盖，所以它未能被地基的[[望远镜|望远镜]]所发现。

海卫八是已知[[太阳系|太阳系]]内最暗的天体之一，它像[[煤煙|煤煙]]那麼黑。和[[土卫九|土卫九]]相似，它们都只反射被照射的太阳光的6%。海卫八的陨石坑众多，陨石坑也没有被地质活动改变的迹象；目前只有一個最大的撞擊坑被命名，該撞擊坑內還包含著其他較小的撞擊坑。海卫八不规则的形状让科学家们相信它已经到了类似密度天体在尚未被自身[[引力|引力]]拉至球状的最大极限。作为对比，[[土卫一|土卫一]]拥有更小的质量，但形状远较海卫八规则。
