'''海因茨-克里斯狄安·史特拉赫'''（{{lang-de|Heinz-Christian Strache}}，{{bd|1969年|6月12日}}），[[奧地利自由黨|奧地利自由黨]]政客。2004年3月起擔任[[奧地利自由黨|自由黨]][[維也納|維也納]]市黨部主席，2005年4月任[[奧地利自由黨|自由黨]]主席，2017年12月18日任奥地利副总理。

== 早年背景 ==
史特拉赫生於[[維也納|維也納]]，於Weiss職業中學學習一年，後進入4年制高等職業學校進修「假牙技師」專業，以合格成績畢業。1993年史特拉赫創立自己的企業。

1999年將企業轉讓，以便全力投入政界。

== 政治生涯 ==
1991年，年僅21歲的史特拉赫當選維也納市議會議員，創造了[[奧地利|奧地利]]當選議員最小年齡紀錄，至今仍無人打破，1996年離職。

兩年後(1998年)，接任[[維也納|維也納]]Wien-Landstraße區黨部主席，此外，史特拉赫還兼任[[奧地利自由黨|自由黨]]青年組織——自由青年連線(RFJ)[[維也納|維也納]]地區負責人。2001年成為[[維也納|維也納]]市議會[[奧地利自由黨|自由黨]]黨團代理召集人，2004年接任自由黨[[維也納|維也納]]市黨部主席(前任為希瑪爾·卡巴斯(Hilmar Kabas))。

史特拉赫一直被認為是[[约尔格·海德尔|约尔格·海德尔]](Jörg Haider)的接班人。史特拉赫接任[[維也納|維也納]]市黨部主席與海德爾在1990年代的經歷極為相似，但2005年初他卻與海德爾分道揚鑣。史特拉赫經常公開批評讓[[土耳其|土耳其]]加入[[歐盟|歐盟]]的提議。自由黨在地方選舉中連遭重挫後，史特拉赫取代時任自由黨主席——海德爾的姐姐[[烏爾蘇拉·哈布納|烏爾蘇拉·哈布納]](Ursula Haubner)並作為自由黨提名候選人的傳言甚囂塵上，但史特拉赫對此不置可否。

然而史特拉赫與海德爾之間的公開權力鬥爭並沒有發生。2005年4月海德爾組建新政黨——[[奧地利未來同盟|奧地利未來同盟]](Bündnis Zukunft Österreich(BZÖ))，新成立的[[奧地利未來同盟|未來同盟]]斷絕同自由黨在聯合政府中的聯盟關係。2005年4月23日史特拉赫在[[奧地利自由黨|自由黨]]代表大會上被選舉為黨主席。

2005年10月23日，由史特拉赫領軍的[[奧地利自由黨|自由黨]]在[[維也納|維也納]]州議會選舉中獲得14.88%的選票，遠遠落後於領先的[[奧地利社會民主黨|社會民主黨]](SPÖ)和居第二位的[[奧地利人民黨|人民黨]](ÖVP)。在選戰中，他提出的被認為涉及[[種族主義|種族主義]]的競選主張，經常被人們所詬病，自由黨的競選海報被毀壞事件多有發生。

2006年10月1日，史特拉赫帶領自由黨參加[[奧地利國民議會|奧地利國民議會]]選舉，得票率為11.21%,獲得總共183個席位中的21席，成為[[奧地利國民議會|奧地利國民議會]]第三大黨(社民黨35.71%；人民黨34.22%；[[奧地利綠黨|綠黨]]10.49%；[[奧地利未來同盟|未來同盟]]4.20%)。值得注意的是自由黨在首都[[維也納|維也納]]的得票率由2002年的7.97%飆升到14.36%。自由黨極端排外的政治主張會對日後的[[奧地利|奧地利]]政治產生怎樣的影響，還有待進一步觀察。選後社民黨與人民黨組建聯合政府。

== 政治立場 ==
史特拉赫代表[[右派民粹主義|右派民粹主義]]的政治勢力，其理念為:奧地利從屬於[[日爾曼|日爾曼]]民族但也應同時強調「[[奧地利|奧地利]][[愛國主義|愛國主義]]」。其主要訴求為:反對外來[[移民|移民]]、重視[[族群|族群]]議題以及改善被「濫用[[政治庇護|政治庇護]]」和「被強迫賣淫的非洲妓女」破壞的公共安全。近年他領導的政黨降低語調，轉為批評[[穆斯林|穆斯林]][[移民|移民]]及反對[[歐盟|歐盟]]。

2004年[[維也納|維也納]]州高等法院認定，史特拉赫的理念與[[國家社會主義|國家社會主義]]([[新納粹|新納粹]])近似。

2005年4月，時任[[維也納|維也納]]市黨部主席的他，因其創意的「[[維也納|維也納]]不能成為[[伊斯坦堡|伊斯坦堡]](Wien darf nicht Istanbul werden!)」競選海報引起廣泛爭議，被[[非政府組織|非政府組織]]以及政治對手稱為「仇視外國人」的政客。同年州議會選舉中，他領導[[奧地利自由黨|自由黨]]再次挑動外國[[移民|移民]]議題。9月，新文宣為「用[[德語|德語]]取代聽不懂的玩意兒(Deutsch statt 'nix verstehen')」「君子在自家居住(Herr im eigenen Haus bleiben)」以及「純正[[維也納|維也納]]人取代[[穆斯林|穆斯林]]人(Pummerin statt Muezzin)」。

2006年的國民議會選舉，史特拉赫以「[[維也納|維也納]]原住民取代[[伊斯蘭|伊斯蘭]]人(Daham statt Islam)」為選舉口號。他還因為另一個宣傳口號「每兩個[[難民|難民]]申請者中定有一個為罪犯(Jeder zweite Asylwerber wird kriminell)」引來很多社會組織的聲討。選舉後，自由黨成為最大反對黨，位居國會第三大黨。

不過，因為[[穆斯林|穆斯林]][[移民|移民]]增加及反對[[歐盟|歐盟]]的訴求，使自由黨的支持度不斷增加。

== 個人生活 ==
1999年結婚，2005年離婚，有兩個子女。兩個子女由前妻撫養。
