'''海庫階梯'''（{{lang-en|Haiku Stairs}}）是一条美国夏威夷[[瓦胡岛|瓦胡岛]]的陡峭[[健行|健行]][[小徑|步道]]<ref name = "intro"/>，由於部分路段幾乎垂直地面，又被稱為「通往天堂的阶梯」<ref name = "BIintro"/>。這條步道通往{{tsl|en|Koʻolau Range|科奧勞山}}山頂，從山頂可以俯瞰[[珍珠港|珍珠港]]以及整個瓦胡島東面<ref name = "intro"/>。[[第二次世界大戰|二战时期]][[美國海軍|美國海軍]]為了在科奧勞山山頂設立無線電天線而建立了這條步道<ref name = "BIintro"/><ref name = "radio"/>。1950年代，該無線電收發站退役後轉由[[美國海岸防衛隊|美國海岸防衛隊]]管理並且整修了步道<ref name = "radio"/>。1980年代因為有數個影劇作品在海庫階梯取景，使得它的知名度大幅上升<ref name = "BIintro"/>，但觀光客的增加也帶來維護上的困難<ref name = "reno"/>。海岸防衛隊在1987年因維護不易而以安全理由停止對民眾開放海庫階梯<ref name = "reno"/>。當地政府在2005年重新整修步道，但並未重新對外開放。目前海庫階梯由[[檀香山|檀香山]]政府與擁有附近土地的供水公司管理，供水公司在入口處設有警衛<ref name = "BIintro"/>。雖然觀光客仍然熱衷挑戰這條路線<ref name = "BIintro"/>，但海庫階梯並不正式對外開放，非法進入可能被罰款或逮捕<ref name = "arr"/>。步道的名稱來自步道的入口{{tsl|en|Haiku Valley|海庫谷}}。雖然海庫的發音雖然與[[俳句|俳句]]的日語發音相似<ref name = "JPhaiku"/>，但其來源並不是[[日語|日語]]，而是當地語言對{{tsl|en|Grevillea banksii|紅花銀樺}}的稱呼<ref name = "dict"/>。

==建立與歷史==
[[File:Haiku_Sairway.jpg|thumb]]
在[[珍珠港事件|珍珠港事件]]之後，[[美國海軍|美國海軍]]認為需要增加在太平洋的無線電通訊設備數量<ref name="navcom"/>，因此派遣海軍上尉黑德里·莫里斯（{{lang-en|Hedley Morris}}）到瓦胡島尋找新設無線電收發站的地點。莫里斯上尉選擇了{{tsl|en|Haiku Valley|海庫谷地}}，[[美國印太司令部|美國印太司令部]]很快便同意。美軍於1942年開始在此秘密興建海庫無線電收發站<ref name="navcom"/>。由於天線需要設在高處，海軍決定讓天線橫跨環繞海庫谷地的科奧勞山區，連接距離2.4公里的兩座山峰<ref name="history"/>。為了在山脊上建立天線，海軍派出攀岩隊進入科奧勞山區尋找適合登山的路線，並在1943年興建了一條上山單程需要超過三小時的木製階梯<ref name = "history"/>。海庫無線電收發站在1943年啟用，它配備了高功率的{{tsl|en|Alexanderson alternator|亞歷山大遜交流機}}來發送訊號給太平洋上的海軍艦艇。<ref name = "history"/>

[[第二次世界大戰|第二次世界大戰]]結束後，海庫無線電收發站在1950年代退役，[[美國海岸防衛隊|海岸防衛隊]]將它改裝成{{tsl|en|Omega (navigation system)|全球導航無線電站}}並整修了海庫階梯，把原先的木製階梯換成3,922級的金屬階梯<ref name = "BIintro"/>。1975年，海岸防衛隊開放遊客進入海庫階梯，海庫階梯成為當地熱門熱門的健行路線之一<ref name="BWSintro"/>。1980年代，[[夏威夷神探|夏威夷神探]]等影劇作品前來海庫階梯取景，因此它的知名度更進一步上升，並帶來了大量的觀光客<ref name="historyarchive"/>。1981年的{{tsl|en|Honolulu Star-Bulletin|檀香山星報}}以《上頭越來越擠啦！（{{lang|en|"It's getting crowded on the top!"}}）》為題，報導當時海岸防衛隊在週末動輒收到二百份攀登申請的盛況<ref name="historyarchive"/>。隨著知名度的提升，遊客留下的垃圾與對附近居民的干擾開始形成問題，在遊客數量大增的狀況下，不時出現的破壞行為帶來的安全隱憂也越益嚴重，在破壞行為日漸嚴重的狀況下，海岸防衛隊在1987年一次嚴重的破壞後決定關閉海庫階梯<ref name="historyarchive"/><ref name = "BIintro"/>。

海庫階梯關閉後不久，數位當地居民組成海庫階梯之友，致力於推動海庫階梯的重新開放<ref name="historyarchive"/>。另一方面，擁有海庫階梯附近土地產權的當地供水公司則主張應該以安全理由將海庫階梯拆除<ref name="hisfond"/>。2005年，當地政府與供水公司共同修復了海庫階梯，但仍然沒有對外開放<ref name ="BWSintro"/>。為了減少非法入侵與可能的意外責任問題，供水公司聘用警衛在攀登入口勸阻遊客<ref name ="BWSintro"/>，並要求堅持要進入的遊客必須投保理賠額在一百萬美元以上的保險，並簽署切結書放棄因事故向供水公司求償的權利<ref name = "huff"/>。

==現況==
雖然政府、產權人以及民眾對海庫階梯的未來沒有共識，許多遊客還是無視於海庫階梯不對外開放的規定持續造訪，2014年當地警方共逮捕了六名遊客，並開出一百三十五張違法入侵罰單<ref name = "arr"/>。2015年2月，海庫階梯在大雨後的山崩受損，供水公司希望將海庫階梯拆除或由其他單位接管<ref name="2015damage"/><ref name = "fine"/>，而支持海庫階梯觀光的市政府表示將負責修復海庫階梯<ref name = "citygov"/>。2017年，供水公司調查居民對海庫階梯去留的意見時有大量支持海庫階梯保留的意見在截止日前湧入，據該公司發言人說法，該公司收到的意見約有70 %希望能夠保留海庫階梯<ref name="BWSpoll"/>。對於調查的結果，海庫階梯之友會長表示滿意並呼籲該公司應尊重民意<ref name="BWSpoll"/>。雖然在2017年的意見調查中，多數表態民眾支持保留海庫階梯並開放觀光，但民眾對於管理上的意見並不一致，部分民眾提出應該限制攀登的人數，也有民眾認為應該對每個遊客收取50到100美元的入山費，以支付維護經費並幫助社區發展<ref name="BWSpoll"/>。
