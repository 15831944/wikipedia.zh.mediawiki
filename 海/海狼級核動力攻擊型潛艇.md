'''海狼级核动力攻击潜艇''' （'''Seawolf class'''）是[[洛杉磯級潛艇|洛杉磯級核動力攻擊潛艦]]的继任者，在[[冷战|冷战]]末期(1989年)开始建造。海狼级潜艇造价高昂，預計單价达到30亿美金，同时也被认为是最安静的核潜艇。最初[[美国海军|美国海军]]打算在10年间以每年3艘的速度，建造29艘海狼级潜艇，其後減至12艘。由于[[苏联解体|苏联解体]]冷战结束、删减国防预算和部分的技术问题，造价过于高昂的海狼级建造计划被取消。海狼级第三艘[[吉米·卡特号|吉米·卡特号]]造价高達35亿美元，高成本导致在1995年海狼级潜艇的订单被全面取消，[[美国海军|美国海军]]只批准建造了三艘海狼级潜艇，其部分任务改由较小的百夫长级，也是后来的[[维吉尼亚级潜艇|維珍尼亞級]]所取代 。<ref>http://www.csic.org.cn/7_dzkw/dzkw_5/77914.htm</ref>

==特徵==
海狼级潜艇比旧洛杉矶级潜艇更安静、更大且更快，共有8具660毫米口径鱼雷管发射管，这8具鱼雷管发射管都较过去的口径大，以便未来可安装新服役的武器，如当时在洛杉矶级潜舰孟菲斯號号（SSN-691 USS Memphis）所装设的实验性762毫米发射管。海狼级潜艇没有[[艦載垂直發射系統|艦載垂直發射系統]]，这也-{zh-hans:体现; zh-tw:呈現}-了它海战优势的设计思想。本潜艇设计深海作战优势，为了捕杀当时[[苏联|苏联]]海军先进[[弹道飞弹|弹道飞弹]]潜艇，如[[台风级|台风级]]，以及[[阿库拉级核潜艇|阿库拉级核潜艇]]。但是他们也有广泛的设备用作浅水作战，其中包括一个浮筒仓能够部署八个作战蛙人及其装备。该船还可以携带多达50个[[战斧巡航导弹|战斧巡航导弹]]攻击陆地和船舶的目标。海狼级潜舰设计的长宽比颇低，且具有六片舰艉平衡翼，有利其操控性的提升。而舰艏平衡翼可收缩至舰体，以利冰下操控。海狼级潜艇能够用极为安静的方式在水下以20节的速度航行，除了使海狼级潜艇更难被侦测到外，也不会受到潜舰本身的噪音影响搜寻，海狼级以HY-100/HY-130高张力钢板组合建造，性能超过HY-80钢，最大潜深可达610米。<ref>{{cite book|last = Polmar|first = Norman|title = The Naval Institute guide to the ships and aircraft of the U.S. fleet|publisher = [[Naval_Institute_Press|Naval Institute Press]]|year = 2004|location = Annapolis, Maryland|pages = 81–82|edition = 18|url = http://books.google.com/books?id=8MwyTX-iA2wC&pg=PA81|isbn = 978-1-59114-685-8|accessdate = 2011-07-01}}</ref><ref>{{cite book|last = Zimmerman|first = Stan|title = Submarine Technology for the 21st Century|publisher = [[Trafford_Publishing|Trafford Publishing]]|year = 2000|location = Victoria, British Columbia|page = 81|url = http://books.google.com/books?id=SjfgOfV8Am0C&pg=PA81|isbn = 978-1-55212-330-0|accessdate = 2011-07-01}}</ref> 

海狼级潜艇使用更先进的AN/BSY-2作战系统，包含宽孔径阵列（Wide Aperture Array，WAA）和潜舰主动侦测系统（Submarine Active Detection Systeam，SASD），其中包括一个新的更大球形高频（HF）及中频（MF）舰艏[[声纳|声纳]]系统，以及一个新的拖曳阵列声纳。每艘舰体是由一个专门为海狼级设计的S6W核反应堆所推动，提供45000马力（34-{zh-hans:兆瓦; zh-tw:百萬瓦}-）驱动一个低噪声喷水推进器。

==占美·卡特號==
占美·卡特號（Jimmy Carter）比其余两艘海狼级潜艇长大约100英尺（30米），因为它要安装一节多任务平台（MMP），它能够派出和回收的[[遥控载具|遥控载具]]及[[海豹部队|海豹部队]]。该多任务平台也可被用来作为一个水下剪接室窃听海底光纤电缆。这个角色的前身是由退役[[美国海军|美国海军]]潛艇帕爾謝魚號（SSN-683 USS Parche）所担任的，占美·卡特號目前驻扎在[[華盛頓州|華盛頓州]]Bangor基地。2006年，美国海军宣布全部海狼级潜艇都将会驻扎在這個基地。
