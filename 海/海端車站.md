'''海端車站'''為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。該站雖有[[海端|海端]]之名，但其位置並不在[[海端鄉|海端鄉]]，而是位於鄰近的[[關山鎮_(台灣)|關山鎮]]。本站為[[南橫公路|南橫公路]]唯二車站（另一座為[[台南車站|台南車站]]）。本站配合「花東新車站運動」計畫改建完工。

== 車站構造 == 
*[[側式月台|側式月台]]一座。
*南端設有[[安全側線|安全側線]]，防止列車對撞；有交會列車功能。
**雖有二股可交會，但只有待避股可供旅客上下車，因此在池上=關山雙軌化之前，不可能同時讓二列區間車在本站交會。
*本站無派駐站員，亦無發售車票之業務。只能以電子票證上下車。

== 利用狀況 ==
*本站為[[海端鄉|海端鄉]]主要門戶，目前為乙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]。
*停靠狀態：
**僅停靠[[臺鐵區間車|區間車]]。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1924年8月27日：設「新武呂驛」，以車站旁[[新武呂溪|新武呂溪]]命名。
*1945年：站名和地名改取此地原住民地名「Haitutuan」之音翻譯改稱為「海端」。「Haitutuan」源自布農語，據說意指此地地形「三面環山、一面敞開」之意。
*2003年8月11日：配合車站北端改線工程完工，完成路線切換，啟用新月台。
*2008年3月27日：由三等站降為乙種簡易站，並指定由[[關山車站_(臺灣)|關山站]]管理，仍具有[[列車交會|交會列車]]的功能。
*2016年6月28日：啟用多卡通刷卡機。
