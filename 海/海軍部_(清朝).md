'''海軍部'''是[[清朝|清朝]]於[[宣統|宣統]]二年（1910年）成立的政府部門，其前身可追溯至[[光緒|光緒]]十一年（1885年）[[中法戰爭|中法戰爭]]之後所設的[[海軍衙門|海軍衙門]]{{r|姜}}。其首長為海軍大臣，副首長是海軍副大臣，底下原有軍制、軍政、軍學、軍防司、軍醫、軍樞、軍儲、軍法八司，但軍防司於宣統三年（1911年）裁撤{{r|姜}}。

== 沿革 ==
清朝在1875年開始建設[[清朝新式海軍|新式海軍]]。光緒十一年（1885年）中法戰爭之後，清朝設立了海軍衙門，後來於光緒十四年（1888年）成立了[[北洋水師|北洋水師]]，但於[[甲午戰爭|甲午戰爭]]中全軍覆沒{{r|姜}}。之後於光緒三十三年（1907年）有恢復之議，遂設「海軍處」，暫時隸屬於[[陸軍部_(清朝)|陸軍部]]，置正使、副使各一人，內分機要、船政、運籌三司{{NoteTag|1=置司長、承發官各三人，及科長，一、二、三等科員，一、二、三等考工官，藝師、藝士、錄事等員額{{r|姜}}。}}，隔年（1908年）內部改為海政、船政、籌備、儲蓄、醫務、法務六司，不久又設主計處{{NoteTag|1=置計長、副長各一人{{r|姜}}。}}{{r|姜}}。

宣統元年（1909年），命[[載洵|載洵]]、[[薩鎮冰|薩鎮冰]]為籌辦海軍大臣，設參贊廳，內分秘書、庶務兩司，置一、二、三等參謀官，並暫設一、二、三、四共四司，之後又增設醫務司，各司置有司長{{r|姜}}。

宣統二年（1910年）制定〈海軍部暫行官制〉，將第一司改為「軍制司」，將第二司改為「軍政司」，將第三司改為「軍學司」，將第四司改為「軍防司」，而醫務司改為「軍醫司」，秘書司改為「軍樞司」，庶務司改為「軍儲司」，此外又增設了「軍法司」，共八司，並省參贊廳各職，不久便將籌辦海軍處改為海軍部，海軍部至此正式成立{{r|姜}}。宣統三年（1911年），裁軍防司，其職併入軍制、軍學二司{{r|姜}}。

== 編制職掌 ==
[[File:Imperial_Chinese_Navy_Secretary's_Flag_(1909-1911).svg|thumb]]
*海軍大臣，一人，部門首長，掌主海軍，稽覈水師及司令部，以固海疆{{r|姜}}{{r|清}}。
*副大臣，一人，部門副首長，協助海軍大臣{{r|姜}}{{r|清}}。
*參謀官六人（一等二人，二等四人，海軍學生充），掌參訂改革{{r|姜}}{{r|清}}。
*參事官二人，掌法律章制{{r|姜}}{{r|清}}。
*秘書官六人，掌機密文移{{r|姜}}{{r|清}}。

底下分七司，各司置司長一人（協都統、正參領充）。司下設科，各科置科長一人（共21人），並有一、二、三等科員（共60人）{{r|姜}}{{r|清}}。軍法司有一等司法官二人，二、三等司法官，學習司法官八人，由學律軍官充。
