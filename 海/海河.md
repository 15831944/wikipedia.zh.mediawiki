'''海河'''又称'''沽河'''，是[[中国|中国]]华北地区最大的河流，总流域面积达31.8万[[平方公里|平方公里]]，涵盖了[[天津|天津]]、[[北京|北京]]全部，[[河北|河北]]绝大部分，以及[[河南|河南]]、[[山东|山东]]、[[山西|山西]]、[[内蒙|内蒙]]等[[省|省]][[自治区|区]]，海河流域的总人口将近1.24亿。

== 名称 ==
海河由于地势不高，每当海中涨潮时，河水倒流，落潮时河水顺流，和海水的潮汐相同，因此取名为“海”河。海河支流的[[永定河|永定河]]原名“无定河”，就是因为河道经常改变，只是皇帝为了祈愿它不再泛滥才人为改名为永定河。

== 流域概况 ==
海河流域各河系大致分为两种类型：一种是发源于太行山与燕山的背风坡，这种河系源远流长，山区汇水面积大，水流集中，泥沙含量相对较多。另一种是发源于太行山与燕山的迎风坡，这种河系支流分散，源短流急，洪峰高、历时短、突发性强、泥沙含量较低。这两种类型河流在华北地区呈相间分布，清浊分明。<ref>{{cite web |url = http://www.hwcc.gov.cn/pub/hwcc/static/lygk/lyzs.htm |language = zh-cn |title = 流域综述 |publisher = 海河水利委员会网站 |accessdate = 2011-03-12 |deadurl = yes |archiveurl = https://web.archive.org/web/20110917081920/http://www.hwcc.gov.cn/pub/hwcc/static/lygk/lyzs.htm |archivedate = 2011-09-17 }}</ref>

=== 干流 ===
海河干流全位于天津市中部，全长73公里，从子牙河与北运河交汇的天津市内金刚桥[[三岔河口|三岔河口]]算起，到[[塘沽|塘沽]]区大沽口入海，实际直线距离只有不到50公里。三岔河口为天津市[[红桥区|红桥区]]、[[河北区|河北区]]和[[南开区|南开区]]的交点，流经和平区、河东区、东丽区直到滨海新区汇入渤海。

=== 主要支流 ===
海河包括了五大支流：[[北运河|北运河]]、[[南运河|南运河]]、[[大清河|大清河]]、[[子牙河|子牙河]]和[[永定河|永定河]]，以及许多更小的支流，如果从最长的支流浊漳河算起，全长为1329公里。
此外海河支流还有：[[潮白河|潮白河]]、[[洋河|洋河]]、[[桑干河|桑干河]]、[[拒马河|拒马河]]、[[府河|府河]]、[[唐河|唐河]]、[[滹沱河|滹沱河]]、[[滏阳河|滏阳河]]、[[卫河|卫河]]、[[清漳河|清漳河]]、[[浊漳河|浊漳河]]等。

== 水文特征 ==
海河流域地形没有过渡带，从[[太行山|太行山]]区直降到华北平原，这个地区雨季集中在每年的7、8月份，尤其太行山麓常发生暴雨，洪水冲刷泥沙，到平原后水流突然变缓，泥沙淤积，所以大部分河道宽浅，河道经常变迁改道，留下许多积水洼地。华北地区年雨量分布也不平衡，据史料记载，从1368年[[明代|明代]]立国，燕王迁都[[北京|北京]]到1949年[[中华人民共和国|中华人民共和国]]成立，海河流域共发生387次严重水灾，407次严重旱灾。

=== 历史 ===
[[File:Tianjin,_with_Western_Ships_in_Hai_River_and_Grand,_Colonnaded_Western_Building_on_the_River_Bank._China,_1874_WDL2109.png|thumb]]

1929年4月21日，国民政府为疏浚海河曾发行短期公债<ref>《疏濬河北省海河工程短期公債條例》</ref>。
1939年水灾冲毁津浦铁路，淹没天津市，市中心和平区比较低洼地带水深几达2米，街道行舟将近两个月。

由于海水倒灌不时淹死稻田，1959年在河口修建了拦河大坝，涨潮时关闸防止海水倒流。1963年太行山麓连下7天暴雨，[[衡水|衡水]]一带一片汪洋，为保护天津市的工业，只好在上游炸堤放水，[[京广铁路|京广铁路]]和[[京沪铁路|京沪铁路]]全部停运，从[[上海|上海]]到[[北京|北京]]乘火车需绕行[[兰州|兰州]]。[[毛泽东|毛泽东]]做出“一定要根治海河”的指示，于是下游开始挖入海减河，上游修建水库，此后，海河流域再也没有发生过水灾，却开始了干旱。1970年代后，由于海河上游城市发展，用水量迅速加大，建起了一系列大中小型水库，大型水库就有为北京供水的[[密云水库|密云水库]]、[[官厅水库|官厅水库]]、[[十三陵水库|十三陵水库]]，为[[保定|保定]]供水的[[西大洋水库|西大洋水库]]、[[王快水库|王快水库]]，为[[石家庄|石家庄]]供水的[[黄壁庄水库|黄壁庄水库]]、[[岗南水库|岗南水库]]，为[[邢台|邢台]]供水的[[朱庄水库|朱庄水库]]，为[[邯郸|邯郸]]供水的[[岳城水库|岳城水库]]，为[[林县|林县]]供水的[[红旗渠|红旗渠]]等，海河下游河道基本处于干涸，华北平原农业用水只得开采地下水，造成地下水位下降。位于“九河下梢”的天津面临严重缺水的问题。

[[File:天津冬季的海河冰上垂钓.jpg|thumb]]
1981年，为供津冀用水而修建的[[密云水库|密云水库]]为保障严重缺水的北京供水，而停止向津冀供水，直接导致津冀两地用水困难<ref>[http://www.bjqx.org.cn/qxweb/n24112c215.aspx 密云水库今天五十岁]，北青网，2010年9月23日查阅</ref>。1981年，为解决干流城市天津的用水，国务院决定兴修[[引滦入津工程|引滦入津工程]]，引华北平原第二大河[[滦河|滦河]]的水到海河下游的天津，1983年竣工，每年引水近10亿立方米，解决了天津缺水问题，为此天津市在三岔河口树立了[[引滦入津纪念碑|引滦入津纪念碑]]，并对上游地区进行长期经济援助。但从此海河口闸永闭，海河干流实际成为一个大水库。海河和滦河河口基本不再向[[渤海|渤海]]排放淡水，造成生态系统的变化，受影响最大的是[[洄游|洄游]]性鱼类。原来海河盛产优良水产，三叉河口的银鱼品质最好，和[[太湖|太湖]]的不同，是“金眼”银鱼，天津郊区[[胜芳镇|胜芳]]（现属河北）的螃蟹可以和[[江苏|江苏]]阳澄湖的大闸蟹媲美，现在这些品种都已经绝迹。

== 流域物产 ==
[[明代|明代]]时[[徐光启|徐光启]]将水[[稻|稻]]引种到中国北方，首先在海河下游落脚，由于北方水稻生长期长，稻米品质优良，闻名中外的“小站稻”就产于海河下游。但是海水倒灌不时淹死稻田，1959年在河口修建了拦河大坝，涨潮时关闸防止海水倒流。
