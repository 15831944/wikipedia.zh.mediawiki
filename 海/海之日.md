'''海之日'''（{{jpn|j=海の日|hg=うみのひ|rm=Uminohi}}）是[[日本|日本]]的[[日本節日|假日]]之一。日期是7月的第3個[[禮拜一|禮拜一]]。

==概要==
海之日是1995年（[[平成|平成]]7年）制定，1996年（平成8年）開始施行的[[日本|日本]][[日本節日|假日]]之一。制定當初是7月20日。2003年（平成15年）祝日法改正（[[Happy_Monday制度|Happy Monday制度]]）後，改成7月的第3個[[禮拜一|禮拜一]]。

「[[關於國民祝日的法律|關於國民祝日的法律]]」（祝日法、昭和23年7月20日法律第178號）第2條，主旨是「感謝[[海|海]]的恩惠，同時也祈願[[海洋|海洋]]國家日本的繁榮」。根據[[國土交通省|國土交通省]]的文書記述，「日本是世界各國之中唯一將『海之日』作為假日的國家」<ref>[http://www.mlit.go.jp/kisha/kisha02/10/100425_2_.html 平成14年｢海の旬間｣実施要領]</ref>。

因[[東京都|東京都]]主辦[[2020年夏季奧林匹克運動會|2020年夏季奧運會]]及[[2020年夏季帕拉林匹克運動會|帕運]]，為避免開閉幕式前後參賽選手、工作人員和觀眾的移動，以及與會各國政要的維安造成[[交通壅塞|交通壅塞]]，影響一般民眾生活作息，日本參議院於2018年6月13日通過《東京奧運暨帕運特別措施法》，調整2020年當年的部份假日，其中海之日調整至奧運開幕典禮前一天7月23日（星期四）放假。<ref>[https://udn.com/news/story/6809/3196398 為放「奧運連假」 日本立特別法通過]</ref>

== 海之記念日 ==
祝日化之前是稱作'''海之記念日'''的[[記念日|記念日]]。海之記念日是1876年（[[明治|明治]]9年），[[明治天皇|明治天皇]]的[[東北地方|東北地方]][[行幸|巡幸]]之際，回程從[[青森|青森]]搭乘汽船「[[明治丸|明治丸]]」，7月20日在[[橫濱|橫濱]]入港，1941年（[[昭和|昭和]]16年），因[[遞信大臣|遞信大臣]][[村田省藏|村田省藏]]的提倡而制定。

明治丸在之後作為東京商船學校（現[[東京海洋大學|東京海洋大學]]）的[[練習船|練習船]]使用，現在保存在東京海洋大學越中島校區。

== 內陸縣的對應 ==
* 沒有海岸線的[[奈良縣|奈良縣]]將7月的第3個禮拜一（海之日）定為「奈良縣山之日・川之日」<ref>[http://www.pref.nara.jp/kasen/yamanohi_kawanohi/yamakawajourei.html 奈良県山の日・川の日条例] {{Wayback|url=http://www.pref.nara.jp/kasen/yamanohi_kawanohi/yamakawajourei.html |date=20090904142012 }} - 奈良県公式ホームページ</ref>。
