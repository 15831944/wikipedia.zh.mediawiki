'''海沧街道'''是[[中华人民共和国|中国]][[福建省|福建省]][[厦门市|厦门市]][[海沧区|海沧区]]下辖的一个[[街道|街道]]。

== 历史沿革 ==
2006年，[[海沧镇|海沧镇]]撤消并成立海沧街道和[[新阳街道_(厦门市)|新阳街道]]<ref>[http://news.sohu.com/20060218/n241892603.shtml 海沧镇建制将撤消]</ref><ref>{{cite web|author=福建省人民政府|title=福建省人民政府关于同意厦门市海沧区撤销海沧镇建制设立海沧、新阳两个街道办事处的批复|url=http://www.fujian.gov.cn/zfxxgkl/xxgkml/jgzz/mzcjbzzc/200806/t20080619_75829.htm|accessdate=2011-11-23|date=2006-01-25|year=2006|deadurl=yes|archiveurl=https://web.archive.org/web/20140202231002/http://www.fujian.gov.cn/zfxxgkl/xxgkml/jgzz/mzcjbzzc/200806/t20080619_75829.htm|archivedate=2014-02-02}}</ref>。

== 行政区划 ==

海沧街道共辖6个社区和10个[[行政村|行政村]]，分别是：

海沧社区、海发社区、吴冠社区、钟山社区、温厝社区、海达社区、石塘村、东屿村、渐美村、囷瑶村、海沧村、青礁村、后井村、锦里村、贞庵村、古楼村、未来海岸。
