== 經歷 ==
海利勒出生於阿拉伯半島東南的亞茲德人部落，後來移居到巴士拉成長。<ref name="abit" />

他的逝世年份有786年和791年兩種記載，傳說他在專心思考時撞到了清真寺的柱子上摔倒，而產生了致命的跌傷。<ref name=kar3/>
== 作品 ==
