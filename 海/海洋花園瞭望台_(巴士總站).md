'''海洋花園瞭望台'''（[[葡萄牙文|葡萄牙文]]：'''TERRAÇO PANORÂMICO DOS JARDINS DO OCEANO'''、{{lang-en|'''Ocean Garden Panoramic Terrace'''<ref>新福利之報站譯作「Ocean Garden Observatory」</ref>}}），是位於[[澳門|澳門]][[氹仔|氹仔]]花園街的一個終點站，[[澳門新福利公共汽車有限公司|新福利]]'''[[澳門巴士15路線|15]]'''和'''[[澳門巴士34路線|34]]'''路線以該站為終點站；[[澳巴|澳巴]]'''[[澳門巴士35路線|35]]'''路線途經此站。

== 沿革 ==


== 歷史 ==
*2008年12月7日，澳門特區政府交通事務局進行第一階段[[巴士路線調整|巴士路線調整]]，其中[[新福利34路線|新福利34路線]]行程縮短，由本站開出之終點站由[[媽閣總站|媽閣總站]]改為[[菜園涌邊街總站|菜園涌邊街總站]]。<ref>[http://www.dsat.gov.mo/show_news.asp?id=633635823115 交通事務局新聞  - 首階段巴士路線重整下月7日實施]</ref>
*2009年10月18日，澳門特區政府交通事務局進行第三階段[[巴士路線調整|巴士路線調整]]，其中新增'''[[澳門巴士36路線|36]]'''路線，以本站為終點站。<ref>[http://www.gcs.gov.mo/showNews.php?PageLang=C&DataUcn=40414&Member=0 新聞局 - 第三階段巴士路線調整18日實施]</ref>
*2012年6月1日，[[維澳蓮運|維澳蓮運]]之[[澳門巴士35路線|35]]和[[澳門巴士36路線|36]]路線，起點站由'''海洋花園瞭望台總站'''遷至蘇利安圓形地，現時只剩下[[澳門新福利公共汽車有限公司|澳門新福利]]的[[澳門巴士15路線|15]]和[[澳門巴士34路線|34]]路線以本站為終點站；[[澳門巴士35路線|35]]路線往駕考中心仍會停靠本站作中途站。

== 設施 ==
*兩個候車亭。
*兩個[[澳門電訊|澳門電訊]]電話亭。
*新福利站長室。
*本站設有站長。
