{{地区用词4|start=[[学名|学名]]：{{lang|la|'''''Placuna placenta'''''}}|cn=海月|tw=雲母蛤|as=称}}，是[[莺蛤目|莺蛤目]][[云母蛤科|云母蛤科]][[云母蛤属|云母蛤属]]的一种。

== 特征 ==
海月贝壳近似圆形，扁平，薄而透明；放射肋和同心生长线极细密；壳面为白色，微紫色壳顶，壳内面白色，有[[云母|云母]]光泽。

== 分布 ==
主要产于暖海中，分布于[[新加坡|新加坡]]、[[马来西亚|马来西亚]]、[[印度尼西亚|印度尼西亚]]、[[中国大陆|中国大陆]]、[[台湾|台湾]]，常栖息在潮间带中、下区及潮线下浅水域。<ref>{{cite web|url=http://shell.sinica.edu.tw/chinese/shellbase_detail.php?science_no=2369|title=国际贝库：海月|publisher=台湾贝类资料库|accessdate=2009-08-11}}</ref>

== 用途 ==
中国古代建筑用贝壳镶嵌在门窗或屋顶上，可以透光，称“窗贝”或“明瓦”。<ref>郭璞《江赋》：“玉珧海月，土肉石华。”；李善注引《临海水土物志》：“海月大如镜，白色正圆。”</ref>
