'''海蒂·拉玛'''（{{lang-en|Hedy Lamarr}}，{{bd|1914年|11月9日|2000年|1月19日|catIdx=Lamarr, Hedy}}），出生於[[奧地利|奧地利]]，[[猶太人|猶太人]]後裔，[[美國|美國]][[演員|演員]]。[[展频|展频]]（{{lang|en|Spread Spectrum}}）技术的共同发明人，為現代許多[[無線通訊系統|無線通訊系統]]包括[[無線區域網路|無線區域網路]]與[[行動電話|行動電話]]的基礎關鍵。被后世尊为“藍牙（Bluetooth）之母”。于2014年被选入美国发明家名人堂。

於1933年上映的捷克电影中展示[[裸体|裸体]]，為世界上最早的裸体电影之一<ref>[http://www.findagrave.com/cgi-bin/fg.cgi?page=gr&GRid=20841 Hedy Lamarr]</ref>。2000年在美國[[佛羅里達州|佛羅里達州]]逝世。

== 早期 ==
海蒂·拉玛出生于奥地利维也纳，原名海德维希·爱娃·玛丽亚·基斯勒（Hedwig Eva Maria Kiesler）她的父亲是一位當地知名的猶太[[银行家|银行家]]，母親是[[布達佩斯|布達佩斯]]的鋼琴家。青少年时期的她被电影院吸引住了，因此她放棄當時學習的通信，退学成為一名演员。学习了[[钢琴|钢琴]]与[[芭蕾|芭蕾]]，后来被导演{{link-en|马克斯·莱因哈特|Max Reinhardt}}发现。莱因哈特將她带到了自己位于[[柏林|柏林]]的表演学校，而拉玛在回到维也纳后，以一名场记员為開端，进入了电影行业。

== 歐洲演員生涯 ==
1930年，年仅17岁的拉玛出演了一部德国电影《街上的钱》（{{lang|de|''Geld auf der Straße''}}），这也是她的第一部影片。在接下来的几年中她又拍摄了几部影片，其中1933年时她拍摄了[[捷克斯洛伐克|捷克斯洛伐克]]电影《神魂颠倒》（''Ecstasy''）(在影片中她拍摄过裸泳镜头，还赤身裸体离开发怒的丈夫，然后奔入森林在小屋中同一个男青年[[做爱|做爱]]<ref name="ga">[http://www.hollywoodsgoldenage.com/actors/hedy_lamarr.html Hedy Lamarr (1914 - 2000)]</ref>)。

== 婚姻 ==
在拍摄完这部影片后不久，她遇上了第一任丈夫{{tsl|de|Fritz Mandl (Industrieller)|弗里茨·曼德尔}}，此人为腰缠万贯的奥地利军火商。他曾与[[希特勒|希特勒]]以及[[墨索里尼|墨索里尼]]等法西斯份子打过交道，在招待丈夫的客户时，拉玛也时常会出席宴会，因此能够听到丈夫與客戶之间的一些交谈，她能擁有豐富的武器相關知識，一部分也要歸功於她與曼德尔的婚姻。

虽然生活在[[萨尔斯堡|萨尔斯堡]]一个城堡内的拉玛非常富裕，但由於被丈夫禁止涉足電影，加上拉瑪反對丈夫支持納粹，所以她并不快樂，在结婚后4年就尝试着逃跑。儘管逃跑失败了，却留下了某些传说。由于丈夫追得太紧，她钻入一家[[妓院|妓院]]躲进了某个房间，而妓院的顾客到来后，她只好同顾客发生性关系以避免被丈夫发现。不过也有其它说法指出，她只是躲进了在播放色情电影的俱乐部。当她再次试图逃跑时，她向丈夫派来监视她的[[女佣|女佣]]下了药，然后穿上了佣人的衣服，从下人出入的后门離開，也逃離了她和曼德爾的婚姻。

== 入主好莱坞 ==
逃到伦敦之後，拉瑪在那里表演过一些戏剧。並且遇到了[[米高梅|米高梅]]三位创始人之一的[[路易·B·梅耶|路易·B·梅耶]]，后者在法国的客轮上同她签订了一份为期7年的合同。因为《神魂颠倒》臭名远扬，梅耶让她改名，她的新名字来源于默片时代的米高梅演员芭芭拉·拉玛（Barbara La Marr）<ref name="lost">[http://www.lostmag.com/issue9/hedylamarr.php Hedy Lamarr]</ref>。

1938年海蒂·拉玛第一次出现在了好莱坞的影片《阿尔及尔》之中，1939年她为米高梅拍摄了第一部影片。她拒绝了主演1942年的《[[北非谍影|北非谍影]]》以及1944年的《[[煤气灯下|煤气灯下]]》，要不然还可以获得更大名声，而[[英格丽·褒曼|英格丽·褒曼]]则因为主演这两部影片获得了很大的声望。不过海蒂·拉玛依旧出演过许多颇受欢迎电影，同她合作过的男演员包括[[克拉克·盖博|克拉克·盖博]]、[[史賓塞·屈賽|史賓塞·屈賽]]等人<ref>[http://www.tcm.com/tcmdb/person/107525%7C68054/Hedy-Lamarr/biography.html HEDY LAMARR]</ref>。

== 成為發明家 ==
=== 靈感 ===
1940年夏天，拉玛在一个聚会上认识了从欧洲德國移民来的钢琴家[[乔治·安塞爾|乔治·安塞爾]]<ref name=":0">{{Cite web|url = http://technews.tw/2014/08/24/hedy-lamarr/|title = 好萊塢美麗影星與秘密通訊技術的發明者－－海蒂拉瑪|accessdate = 2015-11-13|author = liu milo|date = 2014-08-24|publisher = 科技新報 Technews}}</ref>，当时拉玛正同第二任丈夫分居。拉玛曾把安塞爾邀请到家中，詢問通過調節內分泌進行塑身的問題。后来他们讨论了更为严肃的话题，拉玛想起自己的第一任军火商丈夫，某次晚宴上她曾听到纳粹官员谈起过如何操控鱼雷的内容<ref>[https://web.archive.org/web/20120603163030/http://www.nytimes.com/2008/06/09/arts/music/09ball.html?pagewanted=all When George Antheil Met Hedy Lamarr (A True Story)]</ref><ref name="lost" />。如果是直接使用遥控则很容易被相同频率的信号干扰从而使鱼雷偏离目标，某个下午安塞爾漫不经心地弹着钢琴，拉玛想到如果改变钢琴按键就能改变声音，那么改变无线电信号频率同样能改变信号。如果不停随机地改变信号频率，敌人的干扰影响就会减小很多。而安塞爾则想出了具体的实施方法，他曾经在1926年的《机械芭蕾》（Ballet Mecanique）中使用了16架[[自动演奏钢琴|自动演奏钢琴]]，这些钢琴由滚筒驱动。当在[[鱼雷|鱼雷]]的接收器和舰船的发射器内安装上相同编码的滚筒，让两者同时运转时，藉此調整頻率，就可以完成这种[[跳频扩频|跳频扩频]]，使敵人難以掌控。<ref name=":0" />

=== 秘密通訊系統 ===
1942年8月11日，海蒂·拉玛与乔治·安塞爾从[[美国专利局|美国专利局]]获得了名为秘密通信系统（{{lang|en|Secret Communications System}}）的专利，编号为2,292,387<ref>{{US patent|2,292,387}}</ref>（後稱此技術為展頻技術 Spread Spectrum）<ref name=":0" />。事实上在他们的专利中一共使用了88种频率，这个数字和钢琴的按键相同。获得专利后他们没有开发相关的商业用途而是直接交给了政府，他们还需要为此担负相关的专利维护费<ref>[http://people.seas.harvard.edu/~jones/cscie129/nu_lectures/lecture7/hedy/lemarr.htm The birth of spread spectrum]</ref>。然而那时候还未发明[[晶体管|晶体管]]以及[[集成电路|集成电路]]，想要将庞大的[[电子管|电子管]]设备塞入鱼雷之中困难重重，因而海军拒绝了这一想法。虽然拉玛和安塞爾没有继续他们的发明，但是实际上日后扩频技术在被应用到众多无线电通信技术中，[[CDMA|CDMA]]（Code Division Multiple Acces，分碼多工存取）、[[WLAN|無線區域網路]]（WLAN）、[[Wi-Fi|Wi-Fi]]與 [[Bluetooth|Bluetooth]]都是奠基於展頻技術而來。海蒂拉瑪與喬治安瑟終其一生，未因此專利獲得任何一分錢，1997 年，這項專利發明了 56 年後，海蒂拉瑪與喬治安瑟的成就才終於獲得[[电子前哨基金会|電子前線基金會]]（Electronic Frontier Foundation；EFF）榮譽技術獎章的肯定。<ref name=":0" /><ref>{{Cite web|url = http://www.women-inventors.com/Hedy-Lammar.asp|title = Hedy Lamarr - Invention of Spread Spectrum Technology|accessdate = 2015-11-13|author = women inventors|date = |publisher = women inventors}}</ref><ref>{{Cite web|url = http://www.today-events.com/entertainment/what-you-need-to-know-about-hedy-lamarr-actress-by-day-inventor-by-night/|title = What you need to know about Hedy Lamarr: actress by day, inventor by night|accessdate = 2015-11-13|author = |date = 2015-11-09|publisher = |deadurl = yes|archiveurl = https://web.archive.org/web/20151113061811/http://www.today-events.com/entertainment/what-you-need-to-know-about-hedy-lamarr-actress-by-day-inventor-by-night/|archivedate = 2015-11-13}}</ref>

== 晚年生活 ==
二战时拉玛还通过贩售[[战争债券|战争债券]]的方式帮助抵抗法西斯，其中在一个晚上她就售出了7百万美元的债券，平均下来每晚卖掉了2万5千美元的债券<ref name="lost" />。

50年代以后，随着海蒂·拉玛美貌不再，加上新一代的电影演员迅速崛起，她渐渐地淡出了银幕。1951年之后她就很少拍电影了，1953年4月10日，加入美籍。1958年她在自己44岁那年正式退休。1967年后她出版了自传，然后定居到了佛罗里达州，1991年时已经78岁的拉玛因为在商店[[店舖盜竊|顺手牵羊]]被逮捕再次登上新闻<ref>[http://articles.orlandosentinel.com/1991-08-02/news/9108020984_1_hedy-lamarr-vargas-altamonte-springs Hedy Lamarr Charged In Shoplifting Case]</ref>。实际上在1966年时已经发生过类似的事情，那时12名陪审员中有10人认为拉玛无罪。2000年1月19日，85岁的拉玛死于佛罗里达的家中<ref name="lost" />。

海蒂·拉玛一共经历过6段婚姻，最后一段婚姻结束于1965年，在那之后她再也没有结过婚，并于不久后移居到了佛罗里达。所有的这些婚姻都没能长久，最长的第5段婚姻也不过只持续了7年，这一任的丈夫是一位德克萨斯的石油商人。她一共生下了两个孩子并领养过一个男孩<ref name="ga"/>。
