'''海港清真寺'''，位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[海港区|海港区]]清真街，是[[伊斯兰教|伊斯兰教]][[清真寺|清真寺]]。

==简介==
秦皇岛市海港清真寺是河北省著名清真寺之一<ref>{{Cite web |url =http://www.he.xinhuanet.com/zhengwu/hbgk/population/03.html  |title =宗教  |author =  |date =  |publisher =新华网 |language =  |accessdate =  2018-02-03|quote =  }}</ref>。1986年，[[科威特|科威特]]国际伊斯兰慈善基金会主席哈杰捐赠20万[[美元|美元]]，帮助秦皇岛市伊斯兰教协会翻建海港清真寺。1988年5月23日至24日，应秦皇岛市伊斯兰教协会会长李文清[[阿訇|阿訇]]、副会长刘辅仁阿訇邀请，哈杰专程到秦皇岛市访问。访问期间，哈杰参加了中国、[[科威特|科威特]]、[[突尼斯|突尼斯]]合资兴建磷铵厂的奠基仪式，参观了海港清真寺、[[山海关清真寺|山海关清真寺]]以及[[山海关|山海关]]，并欣赏了秦皇岛市海港[[穆斯林|穆斯林]]装饰艺术开发部饰有《[[古兰经|古兰经]]》经文的[[景泰蓝|景泰蓝]]、挂毯、瓷器、字画等民族艺术产品。哈杰途经北京时，还会见了[[中国伊斯兰教协会|中国伊斯兰教协会]]副会长[[马贤|马贤]]<ref>曹景合、刘伟，哈杰先生应邀访问秦皇岛，中国穆斯林1988(4):47</ref>。

2010年12月，河北省秦皇岛市海港区清真寺被[[中共中央统战部|中共中央统战部]]、[[国家宗教事务局|国家宗教事务局]]表彰为首届全国创建和谐寺观教堂先进集体<ref>{{Cite web |url = http://www.fjnet.com/jjdt/jjdtnr/201101/t20110110_174944.htm |title =中央统战部、国家宗教事务局关于表彰首届全国创建和谐寺观教堂先进集体和先进个人的决定  |author =  |date =2011-01-10  |publisher = 佛教在线 |language =  |accessdate =  |quote =  }}</ref>。2015年5月13日，秦皇岛市海港区清真寺被[[河北省人民政府|河北省人民政府]]表彰为全省民族团结进步模范集体<ref>{{Cite web|url = http://info.hebei.gov.cn/hbszfxxgk/329975/329982/6434555/index.html|title = 河北省人民政府 关于表彰全省民族团结进步模范集体和 模范个人的决定|author = |date = 2015-05-13|publisher = 河北省人民政府|language = |accessdate = |quote = }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
