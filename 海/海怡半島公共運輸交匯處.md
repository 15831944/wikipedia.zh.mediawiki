'''海怡半島公共運輸交匯處'''<ref>[http://www.budget.gov.hk/2005/chi/c-708.pdf#page=14 2005年財政預算案]中註明海怡半島為公共運輸交匯處</ref>（{{lang-en|South Horizons Public Transport Interchange}}）位於[[鴨脷洲|鴨脷洲]][[海怡半島|海怡半島]]第4期（御庭園）的地面，公共運輸交匯處的出口設於[[怡南路|怡南路]]。公共運輸交匯處附設巴士總站及專線小巴總站，該巴士總站隨路線不斷增加而不敷應用，有些還不能進站,要用鴨脷洲邨巴士總站/街站。

==歷史==
*1992年3月1日，因海怡半島第一期落成，中巴590線、[[城巴592線|中巴592線]]及[[新巴595線|中巴595線]]遷入海怡半島，而當時的巴士總站設於現海韻閣（第4座）站。
*1995年2月17日，第4座的巴士總站遷至現怡南路的公共運輸交匯處。

==中巴挽回聲望==
中巴590線、592線、595線，也是全空調服務，分別搶回給予[[城巴|城巴]]的90線、92和96線及98線。見中巴開始有成效之時的1995年，連中巴的592線也給予城巴，搶回的乘客也給回城巴。而595和590則於1998年交予新巴

==2005年鴨脷洲巴士路線重組==
於2005年的巴士路線重組，影響海怡半島的巴士路線：
*[[城巴90系路線|城巴90線]]：改為海怡半島 → 香港仔隧道 → 中環 → 薄扶林 → 海怡半島
*[[城巴90B線|城巴90B線]]：改為海怡半島 → 薄扶林 → 中環 → 香港仔隧道 → 海怡半島
此計劃由於[[鴨脷洲邨|鴨脷洲邨]]居民和前市政局議員[[甘乃威|甘乃威]]反對而擱置。

==總站路線資料（巴士）==
===[[城巴90B線|城巴90B線]]===
*{{HK Bus Route HKIsland Show:90B|format=inline}}（經[[薄扶林道|薄扶林道]]）
提供海怡半島、鴨脷洲、香港仔來往薄扶林道、中上環及金鐘的巴士服務，回程並途經上環及西營盤，於1994年12月19日起投入服務，是城巴首條服務薄扶林道的巴士路線。因城巴搶[[新巴91線|新巴91線]]的客源，最初總站設於美康閣（第19座）。

===[[城巴99線|城巴99線]]===
*{{HK Bus Route HKIsland Show:99|format=inline}}
於1994年11月28日投入服務，初因中巴搶[[城巴92線|城巴92線]]的客源，由海怡半島來往[[北角碼頭|北角碼頭]]。1995年城巴接手後，改由海怡半島至[[西灣河碼頭|西灣河碼頭]]，1997年4月14日改為海怡半島至筲箕灣，並繞經耀東邨。

===[[城巴99X線|城巴99X線]]===
*{{HK Bus Route HKIsland Show:99X|format=inline}}
於2012年8月20日投入服務，為[[城巴99線|城巴99線]]的輔助路線，提供海怡半島單向往東區的巴士服務。

===[[過海隧道巴士171線|過海隧道巴士171線]]===
*{{HK Bus Route Cross Harbour Show:171|format=inline}}
提供海怡半島、鴨脷洲邨、利東及黃竹坑來往九龍區的巴士服務，於1994年6月6日投入服務，由九巴及城巴合營。於2007年12月30日，總站由[[長沙灣|長沙灣]]延長至[[荔枝角|荔枝角]]。每當171P線提供服務時，此線往荔枝角方向會暫停服務。

===[[過海隧道巴士171P線|過海隧道巴士171P線]]===
*{{HK Bus Route Cross Harbour Show:171P|format=inline}}
171的特別班次，提供鴨脷洲(海怡半島、鴨脷洲邨)往西九龍區的巴士服務，中途不停利東、黃竹坑及灣仔堅拿道，袛於平日早上行走。

===[[新巴590線|新巴590線]]===
*{{HK Bus Route HKIsland Show:590|format=inline}}
提供海怡半島、鴨脷洲邨及黃竹坑來往灣仔及中環的巴士服務。

===[[城巴592線|城巴592線]]===
*{{HK Bus Route HKIsland Show:592|format=inline}}
於1992年3月1日投入服務，初因中巴搶[[城巴96線|城巴96線]]的客源而經[[利東邨|利東邨]]。於1995年交由城巴接手後，不再繞經利東邨。

===[[新巴595線|新巴595線]]===
*{{HK Bus Route HKIsland Show:595|format=inline}}
提供來往海怡半島、鴨脷洲邨及香港仔的穿梭巴士服務，於1992年5月26日起配合海怡半島第1期入伙而投入服務。於1998年給予新巴接手。

===[[城巴N90線|城巴N90線]]===
*{{HK Bus Route HKIsland Show:N90|format=inline}}
提供來往灣仔、金鐘及中環的通宵巴士服務。

===[[過海隧道巴士X970線|過海隧道巴士X970線]]===
*{{HK Bus Route Cross Harbour Show:X970|format=inline}}

==特別總站路線資料（巴士）==
===通宵巴士590S線===
*'''海怡半島''' ↔ [[中環|中環]]（[[交易廣場|交易廣場]]）
提供海怡半島、鴨脷洲邨及黃竹坑來往灣仔、金鐘及中環的節日通宵巴士服務，於2014年12月25日（聖誕節）凌晨起投入服務，2004年起改為祇在農曆年初一及聖誕節凌晨提供服務。

===通宵巴士592S線===
*'''海怡半島''' ↔ [[銅鑼灣（摩頓台）巴士總站|銅鑼灣（摩頓台）]]
提供海怡半島及鴨脷洲邨來往銅鑼灣的節日巴士服務，於1996年2月19日（農曆年初一）凌晨起投入服務，於農曆年初一、中秋節翌日、聖誕節及元旦凌晨提供服務。

===通宵巴士N595線===
*'''海怡半島''' ↺ [[香港仔|香港仔]]
提供來往海怡半島、鴨脷洲邨及香港仔的節日巴士服務，於1997年12月25日（聖誕節）凌晨起投入服務，於特別節日中秋節翌日、聖誕節、元旦日及農曆年初一凌晨提供服務
