'''海关总署驻上海特派员办事处'''成立于2002年7月12日，是[[中华人民共和国海关总署|中华人民共和国海关总署]]设立在[[上海市|上海市]]的派出机构，该办事处直属海关总署领导<ref name="shtpb">{{Cite web |url=http://shanghai_sup.customs.gov.cn/publish/portal93/tab4886/ |title=本办概况，海关总署驻上海特派员办事处，于2012-9-10查阅 |access-date=2012-09-10 |archive-url=https://web.archive.org/web/20140307095344/http://shanghai_sup.customs.gov.cn/publish/portal93/tab4886/ |archive-date=2014-03-07 |dead-url=yes }}</ref>。

== 主要职责 ==
海关总署驻上海特派员办事处的主要职责为负责管辖区域内直属海关的执法督查、干部考察、廉政督察和对署管干部进行诫勉谈话等工作。<ref>[http://www.people.com.cn/GB/shizheng/3586/20020710/773165.html 海关管理体制的重大改革 海关总署驻天津特派员办事处成立，人民网，2002年7月10日]</ref>

== 管辖范围 ==
海关总署驻上海特派员办事处的管辖范围包括：[[上海海关|上海海关]]、[[南京海关|南京海关]]、[[杭州海关|杭州海关]]、[[宁波海关|宁波海关]]、[[合肥海关|合肥海关]]、[[福州海关|福州海关]]、[[厦门海关|厦门海关]]、[[南昌海关|南昌海关]]、[[青岛海关|青岛海关]]、[[上海海关高等专科学校|上海海关高等专科学校]]。<ref name=shtpb/>
