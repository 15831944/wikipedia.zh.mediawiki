'''海港大樓'''是位於[[台灣|台灣]][[基隆市|基隆市]][[仁愛區|仁愛區]]的[[官署|官署]][[辦公大樓|辦公大樓]]，興建於[[台灣日治時期|日治時代]]，原名「'''基隆港合同廳舍'''」，是由時任[[台灣總督府|台灣總督府]]交通局技師的[[鈴置良一|鈴置良一]]所設計，興建目的是要將當時散置於[[基隆港|基隆港]]各處的11個港務機構及中起來以便利港務運作{{r|史}}{{r|基}}。該建築於2003年1月20日由[[基隆市政府|基隆市政府]]公告為[[基隆市歷史建築|基隆市歷史建築]]{{r|文}}，曾與鄰近的[[陽明海洋文化藝術館|陽明海運舊廈]]獲選為[[中華民國各縣市歷史建築十景列表#基隆市|基隆市歷史建築十景]]之一、以及[[臺灣歷史建築百景列表|臺灣歷史建築百景]]的第80名。

== 沿革 ==
基隆港合同廳舍動工於[[昭和|昭和]]五年（1930年）10月11日，於昭和九年（1934年）7月15日完工，所在地當時是明治町一丁目，其工程經費則來自1930年基隆築港工事費中的海岸設備費{{r|基}}。完工後遷入大樓內的機構有基隆稅關（[[財政部關務署基隆關|財政部關務署基隆關]]前身）、[[臺北州|臺北州]]港務部（[[臺灣港務公司|臺灣港務公司]][[基隆港務分公司|基隆港務分公司]]前身）、交通局基隆築港出張所、交通局基隆海事出張所、基隆郵便局波止場出張所、臺灣總督府植物檢查所基隆分所、臺灣總督府鳳梨罐詰基隆檢查所、臺灣總督府米穀檢查所基隆出張所、臺灣總督府肥料檢查所基隆分所、專賣局基隆出張所、基隆警察署水上派出所等{{r|基}}。

戰後，於1947年改稱「海港大樓」，目前仍有許多基隆港相關港務機構在此辦公{{r|基}}。

== 建築特色 ==
海港大樓的主體為四層樓高（原為三層樓高，戰後加蓋四樓），塔樓樓高五層，旁邊的附屬建築則為三層樓高，採用鋼筋混凝土架構{{r|史}}。其外觀在轉角處做成圓弧形，並且減少陽台的配置，而其外觀上一些構件除其機能性之外，還使建築帶有了船舶的意象{{r|基}}。
