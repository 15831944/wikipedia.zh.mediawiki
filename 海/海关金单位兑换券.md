'''海关金单位兑换券'''，简称“'''关金券'''”，是[[中华民国|中华民国]][[国民政府|国民政府]]的[[海关|海关]]用以计算税收的[[金本位|金本位]]单位。后来[[中央銀行_(中華民國)|中央银行]]发行关金券，用以收纳[[关税|关税]]，渐渐成为[[通货|通货]]的一种。

== 背景 ==
[[中国海关|中国海关]]本来以[[白银|白银]]征税，但是由于1929年[[世界|世界]]银价大跌，金贵银贱，影响了海关的收入。政府于1930年1月改用海关金单位收税，以0.601866克纯金（合0.4美元）为单位作标准计算，称“海关金单位”，价值为0.40[[美元|美元]]。

== 發行 ==
1931年5月[[中央銀行_(中華民國)|中央银行]]发行关金兑换券，供缴纳关税。初时的关金券流通量有限。1942年2月，关金提高含金量至0.88861克纯金，即一关金等于一美元。同年4月，定关金为一元折合法币二十元，与[[法币|法币]]自由兑换，并且同时在市面流通。之后[[法币|法币]]快速贬值，政府亦发行大额关金券，成为比法币面额更高的[[货币|货币]]。

== 新關金券 ==
1947年1月16日，中央銀行發行新關金券，分250元及500元兩種，規定對法幣比例為1：20。<ref name="民國史大事記">李新總主編，中國社會科學院近代史研究所中華民國史研究室編，韓信夫、姜克夫主編，《中華民國史大事記》（全十二卷，共十二册），北京：中華書局，2011年7月</ref>{{rp|8268}}新關金券實係5000元及10000元法幣大鈔之替代品。」<ref name="民國史大事記"/>{{rp|8268}}同日，財政部公佈實施攜帶國幣出口限制辦法。<ref name="民國史大事記"/>{{rp|8268}}在發行新關金券刺激下，漢口市場黃金[[飾金|飾金]]每兩陡增4萬元，其他物價亦隨之趨漲。<ref name="民國史大事記"/>{{rp|8269}}南京、上海[[金價|金價]]亦漲，各地[[美鈔|美鈔]]、港票亦被金價拉高。<ref name="民國史大事記"/>{{rp|8269}}中央銀行在[[上海|上海]]拋售黃金，但仍求過於供，金價上漲不已。<ref name="民國史大事記"/>{{rp|8269}}

1948年[[金圆券|金圆券]]发行前，'''最高面额'''的关金券面值为关金25万圆，相等于500万法币。1948年11月20日[[金圆券|金圆券]]发行后，关金与法币同时停止流通。关金券总共流通17年半，期间正式发行的关金券种类有47种，未发行的有4种。
