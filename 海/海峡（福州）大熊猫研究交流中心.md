'''海峡（福州）大熊猫研究交流中心'''，俗称'''海峡（福州）熊猫世界'''，位于[[福建省|福建省]][[福州市|福州市]][[鼓楼区_(福州市)|鼓楼区]]梦山路88号，为直属福州市建委的全民[[事业单位|事业单位]]<ref name=hxrc>{{Cite web |url =http://www.hxrc.com/JobDetail_c0a3bdd3-c091-4530-8ed0-2b311e6dc1a1.html  |title =  海峡（福州）大熊猫研究中心|author =  |date =  |publisher = 中国海峡人才网 |language =  |accessdate =  2017-09-15|quote =  }}</ref>。

==简介==
1970年代末，毕业于[[福建省农学院|福建省农学院]]畜牧兽医系的[[陈玉村|陈玉村]]调任[[福州动物园|福州动物园]]园长后，向[[中华人民共和国林业部|中华人民共和国林业部]]提出申请驯养大熊猫，获得林业部批准。1977年，陈玉村首次从[[甘肃省|甘肃省]]引进2只[[大熊猫|大熊猫]]到福州动物园，1986年繁殖第一次成功。1992年，成立'''福州大熊猫研究中心'''，陈玉村任主任。为响应中央对海西建设先行先试的号召，为[[海峡两岸|海峡两岸]]携手保护野生动物、关爱大熊猫、和谐大自然作出新贡献，根据港区[[全国政协委员|全国政协委员]][[林铭森|林铭森]]的提案，经[[国务院台湾事务办公室|国务院台湾事务办公室]]、[[国家林业局|国家林业局]]批准，2011年5月29日该中心更名为'''海峡（福州）大熊猫研究交流中心'''<ref name=gengming>{{Cite web |url = http://news.163.com/11/0530/08/759QA8D000014JB5.html |title = 福州大熊猫研究中心更名为海峡(福州)大熊猫研究交流中心 |author =  |date = 2011-05-30 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref name=dnw>{{Cite web |url = http://www.fjsen.com/hxlw/2014-08/28/content_14777643_all.htm |title =福州3只小熊猫东渡海峡移居宝岛  |author =  |date = 2014-08-28 |publisher =  东南网|language =  |accessdate =  |quote =  }}</ref>。

[[1990年亚运会|1990年亚运会]]吉祥物“[[盼盼|盼盼]]”的原型就是该中心饲养的大熊猫[[巴斯_(大熊猫)|巴斯]]<ref name=dnw/>。2011年5月更名时，该中心正在饲养8只大熊猫（其中包括赠台大熊猫“[[圆圆_(熊猫)|圆圆]]”的母亲“雷雷”），25只[[小熊猫|小熊猫]]（浣熊科）以及13只与大熊猫分类地位研究有关的其他熊类<ref name=gengming/>。2014年6月6日，该中心赠送的3只自繁小熊猫“欢欢”、“美可”、“丫丫”入住[[台北市立动物园|台北市立动物园]]<ref name=dnw/>。
