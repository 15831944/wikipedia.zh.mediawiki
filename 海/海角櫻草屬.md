'''海角櫻草属'''（[[学名|学名]]：''Streptocarpus''），又名'''菫蘭属'''或'''旋果苣苔属'''，為[[苦苣苔科|苦苣苔科]]下的[[雙子葉|雙子葉]][[開花植物|開花植物]]。其原生於[[非洲|非洲]]中部、東部、與南部的山區，亦分佈於[[馬達加斯加|馬達加斯加]]與[[葛摩群島|葛摩群島]]<ref name = Bellstedt2013>{{cite web|last=Bellstedt|first=Dirk U.|title=''Streptocarpus'': Geographical Distribution and Ecology|url=http://www.gesneriads.ca/Articles/Bellstedt%20-%20Streptocarpus/geographical_distribution_and_ecology.htm |publisher=The Gesneriad Reference Web |accessdate=24 July 2013}}</ref>。此屬共有176種並分為兩個亞屬Streptocarpus與Streptocarpella，其成員包含園藝常見的[[非洲堇|非洲堇]]、菫蘭等<ref name = Nishii>{{citation |author1=Nishii, K. |author2=Hughes, M. |author3=Briggs, M. |author4=Haston, E. |author5=Christie, F. |author6=DeVilliers, M.J. |author7=Hanekom, T. |author8=Roos, W.G. |author9=Bellstedt, D.U. |author10=Möller, M. |year=2015 |title=Streptocarpus redefined to include all Afro-Malagasy Gesneriaceae: Molecular phylogenies prove congruent with geographical distribution and basic chromosome numbers and uncover remarkable morphological homoplasies |journal=Taxon |volume=64 |issue=6 |pages=1243–1274 |url=http://www.ingentaconnect.com/content/iapt/tax/2015/00000064/00000006/art00009
|doi=10.12705/646.8}}</ref>。在其原生地非洲，這些植物通常生長在有陰影遮蔽的山坡或石壁上。而作為一種常見園藝植物，現在已有許多園藝雜交種的出現<ref>{{cite web|title="Streptocarpus-info"|url=http://www.streptocarpus-info.com}}</ref>。

在型态特徵上，此属植物的花均为五瓣，花冠為高腳碟狀，近似[[兰花|兰花]]的型态自[[花序|花序]]垂挂而下。而其名海角樱草又显示着植物的叶部型态与[[报春花科|报春花科]]的[[樱草|樱草]]相似（英文俗名：Cape Primrose 即 好望角报春花）。此属原被定义的分类特徵为「具有螺旋型果荚」（属名源自[[拉丁文|拉丁文]]''strepto'' = 旋型，''carpus'' = 果实），但后来发现此特徵广泛分佈于其他[[旧世界|旧世界]]的苦苣苔科物种上。

[[DNA|DNA]]分子親緣關係研究显示，虽然不具有螺旋型的果荚，但园艺植物[[非洲堇|非洲堇]]可能演化自[[坦尚尼亚|坦尚尼亚]]的''Streptocarpella''亚属，而主張將[[非洲堇屬|非洲堇屬]]降級為''Streptocarpella''亞屬的一個組──非洲堇組（sect. ''Saintpaulia''）。<ref name=Nishii/><ref>{{citation |url=http://www.amjbot.org/content/84/7/956.abstract |author1=Möller, M. |author2=Cronk, Q.C.B.  |lastauthoramp=yes |year=1997 |title=Origin and Relationships of ''Saintpaulia'' (Gesneriaceae) based on ribosomal DNA internal transcribed spacer (ITS) sequences |journal=American Journal of Botany |volume=84 |issue=7 |pages=956–965 |doi=10.2307/2446286}}</ref>
==名称==
[[File:Streptocarpus_fruit_1.jpg|缩略图]]
[[File:Streptocarpus_fruit_3.jpg|缩略图]]
旋果苣苔的名字来源于她长长的，螺旋状的果莢。Streptocarpus源自希腊语στρεπτός（streptos）是旋转的意思，而καρπός（karpos）是[[果实|果实]]的意思。

==特殊生長型態==
[[File:Streptocarpus_seedling_showing_main_and_rudimentary_seed_leaves.JPG|thumb]]
除園藝使用上，海角櫻草屬的植物亦因其花的多樣性與特殊的生長型態而被廣泛研究<ref name=Nishii/><ref name = Burtt1970>{{citation |author1=Burtt, BL. |title=Studies in the Gesneriaceae of the Old World XXXI: Some aspects of functional evolution |journal=Notes from the Botanic Garden Edinburgh |volume=30 |pages=1–10}}</ref><ref name = Jong1970>{{citation |last=Jong|first=K.|title=Developmental aspects of vegetative morphology in Streptocarpus|publisher=PhD dissertation. University of Edinburgh |accessdate=1970}}</ref><ref name = Jonh1975>{{citation |author1=Jong, K. |author2=Burtt, BL. |title=The evolution of morphological novelity exemplified in the growth patterns of some Gesneriaceae |journal=New Phytologist |volume=78 |pages=297–311}}</ref><ref name = Moeller2001>{{citation |author1=Möller, M. |author2=Cronk, Q. |title=Evolution of morphological novelity: A phylogenetic analysis of growth pattern in Streptocarpus (Gesneriaceae) |journal=Evolution |volume=55 |pages=918–929}}</ref><ref name = Nishii2004>{{citation |author1=Nishii, K. |author2=Kuwabara, A. |author3=Nagata, T. |title=Characterization of anisocotylous leaf formation in Streptocarpus wendlandii (Gesneriaceae): significance of plant growth regulators |journal=Annals of Botany |volume=94 |pages=457–467}}</ref><ref name = Harrison2005>{{citation |author1=Harrison, J. |author2=Möller, M. |author3=Langdale, J. |author4=Cronk, Q. |author5=Hudson, A. |title=The role of KNOX genes in the evolution of morphological novelity in Streptocarpus |journal=The Plant Cell |volume=17 |pages=430–443}}</ref><ref name = Nishii2007>{{citation |author1=Nishii, K.  |author2=Nagata, T. |title=Development analyses of the phyllomorph formation in the rosulate species Streptocarpus rexii (Gesneriaceae) |journal=Plant systematics and Evolution |volume=265 |pages=135–145}}</ref>，其特殊性包含二片子葉的不等大發育、無主莖的生長型、以及離層線的產生。

===子葉不等大現象===
雖然是雙子葉植物，但海角櫻草屬的植物在萌芽後，兩片[[子葉|子葉]]會不等速的生長，形成子葉不等大的現象（{{lang|en|Anisocotyly}}）<ref name = Burtt1970/><ref name = Jong1970/><ref name = Jonh1975/><ref name = Nishii2004/><ref name = Nishii2007/>。其中一片會持續延伸膨大而外觀與真葉相似；另一片則會停止生長至最後老化脫落。此特徵普遍存在於[[舊世界|舊世界]]的苦苣苔物種中。

==生長型各異==
依其[[葉|葉]]片數量與[[莖|莖]]部發育，海角櫻草屬植物可被略分為三種生長型 （{{lang|en|Growth form}}）<ref name=Nishii/><ref name = Burtt1970/><ref name = Jong1970/><ref name = Jonh1975/><ref name = Moeller2001/><ref name = Nishii2004/><ref name = Harrison2005/><ref name = Nishii2007/>：

===無主莖多葉型===
無主莖多葉型（{{lang|en|Acaulescent, rosulate}}），此生長型不具有明顯的莖，而新生葉片是自膨大後的子葉基部長出，形成多葉環狀外觀（Rosette）。

===無主莖單葉型===
無主莖單葉型 （{{lang|en|Acaulescent, unifoliate}}），此生長型不具明顯的莖，亦不生長新的葉片，唯一的葉片就是膨大的大子葉。在某些物種中（例：''S. grandis''）此子葉甚至可以生長至80公分長。

===有主莖型===
有主莖型 （{{lang|en|Caulescent}}），與其他植物相同，具有明顯的[[根|根]]、[[莖|莖]]、[[葉|葉]]構造。

===[[離層|離層]]線===
在某些[[多年生|多年生]]的單葉品種（及部分的多葉品種）中可看到此類植物獨有的「離層線」（{{lang|en|Abscission line}}）<ref name = Burtt1970/>
現象。冬天時，這些海角櫻草會在葉片中間形成一條「離層線」，離層線以外的區域（{{lang|en|abscission zone}}）會老化乾枯，但離層線以內的葉片會維持長綠。此與大多數植物中，葉片是直接自葉梗基部斷裂有所區別。
