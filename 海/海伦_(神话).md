'''海伦'''（{{lang-grc|Ἑλένη, Helénē}}，{{lang-en|Helen of Troy}}）是希腊神话中[[宙斯|宙斯]]与[[勒达|勒达]]之女，被称为“世上最美的女人”，她和[[特洛伊|特洛伊]]王子[[帕里斯|帕里斯]]私奔，引发了特洛伊战争。

== 家庭 ==

*父親：[[宙斯|宙斯]]（Zeus）
*养父：[[廷达瑞俄斯|廷达瑞俄斯]]（Tyndareus）
*母親：[[涅梅西絲|涅梅西絲]]（Leda）
*哥哥：[[卡斯托耳|卡斯托耳]]（Castor）/[[波吕丢刻斯|波吕丢刻斯]]
*姐姐：[[克吕泰涅斯特拉|克吕泰涅斯特拉]]（Clytemnestra）
*丈夫：[[莫奈劳斯|莫奈劳斯]]

== 身世 ==

希腊神话中，天神[[宙斯|宙斯]]勾引斯巴达王[[廷达瑞俄斯|廷达瑞俄斯]]之妻[[勒达|勒达]]，宙斯变身天鹅与勒达调情，勒达生下两枚鹅蛋。一个鹅蛋裂开，里面出来海伦的哥哥[[卡斯托耳|卡斯托耳]]与[[波吕丢刻斯|波吕丢刻斯]]，另一个鹅蛋裂开出来海伦和[[克吕泰涅斯特拉|克吕泰涅斯特拉]]。

== 忒修斯劫夺事件 ==

据说著名英雄[[忒修斯|忒修斯]]是第一个对海伦出手之人。忒修斯虽然已经年老，但仍出外抢劫妇女。其时海伦尚年幼，住在生母[[勒达|勒达]]之夫[[廷达瑞俄斯|廷达瑞俄斯]]的宫廷之中。忒修斯在前往[[斯巴达|斯巴达]]抢劫妇女的途中看见年幼的海伦在[[阿耳忒弥斯|阿耳忒弥斯]]神殿内起舞，惊为天人，遂将她劫往[[阿耳卡狄亚|阿耳卡狄亚]]的[[特格阿|特格阿]]。

此后忒修斯起意劫夺冥王[[哈得斯|哈得斯]]之妻，失败，遭囚禁。这期间海伦的兄长[[卡斯托耳|卡斯托耳]]和[[波吕丢刻斯|波吕丢刻斯]]攻破城池，救出海伦。

=== 关于海伦的年龄 ===

关于海伦被劫夺时的年龄，比较难以估计。忒修斯劫夺海伦后，被囚禁在冥府一段时间，将他救出的是[[赫拉克勒斯|赫拉克勒斯]]。当时赫拉克勒斯正在完成[[欧律斯透斯|欧律斯透斯]]所交给的第12项任务，正当壮年。此后他又经历过多年-{zh-cn:历;zh-tw:歷;}-险，并曾攻破[[特洛伊|特洛伊]]城，当时[[普里阿摩斯|普里阿摩斯]]仍是少年。赫拉克勒斯、忒修斯皆死亡、他们的子女成年并成为特洛伊英雄、普里阿摩斯年老之后，[[特洛伊战争|特洛伊战争]]方才发生，海伦大约在这时才被帕里斯拐走。如果这样计算，海伦的年龄很可能比大多数特洛伊英雄要老上一辈。

== 婚姻与求婚者的盟誓 ==

海伦成年后，求婚者不绝如缕，几乎涵盖了[[古希腊|古希腊]]所有王国的年轻才俊。养父[[廷达瑞俄斯|廷达瑞俄斯]]无法决定最后的人选，因为不论选择哪位英雄，他都难逃其他竞争者的群起围攻。此时[[奥德修斯|奥德修斯]]提议[[抽签|抽签]]，“由[[宙斯|宙斯]]决定海伦的归宿”；并且事先由在场的所有英雄举行盟誓，约定一旦中签者因为海伦而陷入麻烦，在场之人一律有义务发兵援助。

抽签的结果，是[[迈锡尼|迈锡尼]]王子[[莫奈劳斯|莫奈劳斯]]中签。由于莫奈劳斯的长兄[[阿伽门农|阿伽门农]]即位成为迈锡尼王，莫奈劳斯遂[[入赘|入赘]][[斯巴达|斯巴达]]，并在廷达瑞俄斯死后成为斯巴达王。

== [[帕里斯|帕里斯]] ==

当初，[[赫拉克勒斯|赫拉克勒斯]]攻陷[[特洛伊|特洛伊]]城，杀尽国王[[拉俄墨冬|拉俄墨冬]]全家，只留下公主[[赫西俄涅|赫西俄涅]]，将其嫁予好友——[[萨拉弥斯|萨拉弥斯]]国王[[忒拉蒙|忒拉蒙]]为妻。此二人便是[[大埃阿斯|大埃阿斯]]的父母。在赫西俄涅的请求下，另一位王族少年作为奴隶之身得以偷生，此人在赎身后回到特洛伊，便是特洛伊王[[普里阿摩斯|普里阿摩斯]]。

[[帕里斯|帕里斯]]是普里阿摩斯的一个儿子。由于普里阿摩斯想念仍旧身在[[撒拉弥斯|撒拉弥斯]]的姐姐[[赫西俄涅|赫西俄涅]]，帕里斯便组织一支使团（一说军队）前去要人。途径[[斯巴达|斯巴达]]，恰逢斯巴达王[[莫奈劳斯|莫奈劳斯]]外出，由王后[[海伦|海伦]]负责招待帕里斯一行。

[[帕里斯|帕里斯]]在[[金苹果事件|金苹果事件]]中，曾得爱神[[阿芙蘿黛蒂|阿芙蘿黛蒂]]的许诺，可以得到世上最美的女人的爱情。借助爱神的力量，海伦爱上了帕里斯，帕里斯携海伦私奔，并将王城扫荡一空。

== 特洛伊战争 ==

{{main|特洛伊战争}}

海伦是特洛伊战争中唯一重要女性角色。

据[[伊利亚特|伊利亚特]]第三卷记载，战争的第十年，当[[帕里斯|帕里斯]]与[[莫奈劳斯|莫奈劳斯]]商定在城下以一场单挑解决恩怨时，海伦闻讯后在心中涌起对丈夫和故乡的“甜蜜思念”，流着泪走上城头并目睹了这场战斗。当时，海伦坐在老王[[普里阿摩斯|普里阿摩斯]]身边，[[荷马|荷马]]借海伦之口向读者们介绍了[[古希腊|希腊]]人中的几位强者。

[[伊利亚特|伊利亚特]]第二十四卷最后几行描写，海伦在[[赫克托尔|赫克托尔]]的葬礼上痛哭失声，缅怀后者的善良心地与优雅谈吐。

赫克托耳与帕里斯的兄弟[[得伊福玻斯|得伊福玻斯]]在帕里斯阵亡后迎娶海伦。[[特洛伊|特洛伊]]城破后，海伦回到[[莫奈劳斯|莫奈劳斯]]身边。她站在船头，是如此美丽，以至于希腊联军的将士都被她的美丽震撼，认为即使打了十年的战争，死了无数的人，但为了海伦，这一切都是值得的。

== 战后生涯 ==

[[荷马史诗|荷马史诗]]《[[奥德赛|奥德赛]]》描写，[[莫奈劳斯|莫奈劳斯]]是少数得以平安返回祖国的英雄之一。战争结束后数年，[[俄底修斯|俄底修斯]]仍音讯全无，其子[[忒勒马科斯|忒勒马科斯]]周游列国探听父亲的音讯。他在斯巴达的宫廷见到王后海伦。作者描写，“她仍然如同一位女神”。

== [[荷马|荷马]]笔下的海伦 ==

[[荷马|荷马]]在史诗中对海伦的固定修饰语，包括“女人中闪光的佼佼者”、“长裙飘舞的”、“美髮的”等几个，而最常用的则是“[[阿耳戈斯|阿耳戈斯]]的”。此处的“阿耳戈斯”是全希腊的代称，海伦的故乡是[[斯巴达|斯巴达]]。

[[希臘神話|希臘神話]]中的美女'''海倫'''（{{lang|en|'''Hallen'''}}，'''海琳'''為海倫的異體），是[[宙斯|宙斯]]的女兒，從鵝蛋中生出來，16歲時被迫嫁給[[斯巴達|斯巴達]]王[[莫內勞斯|莫內勞斯]]（Menelaos），為斯巴達王后。後來[[特洛伊|特洛伊]]王子[[赫克托耳|赫克托耳]]和二王子[[帕里斯|帕里斯]]（Paris）到訪斯巴達，帕里斯把她拐去，帶回特洛伊。斯巴達王大怒，他向哥哥邁錫尼國王[[阿加门农|阿加门农]]求助，希臘聯軍組織一千艘的船艦出征，而引起著名的[[特洛伊戰爭|特洛伊戰爭]]。後世有人形容這場戰爭為「使千艘戰艦齊發的容貌，海倫的美引發了特洛伊戰爭。」海倫為自己引起的戰爭內疚不已，她走向特洛伊城上，俯望兩軍戰士，當數萬戰士望著她時，全看傻了，海倫美到兩軍戰士連戰都不想打，於是雙方休兵一天。十年後特洛伊陷落，希臘大軍從巨型木馬內部殺出，男的被屠盡，女人、小孩全變奴隸，莫內勞斯要取海倫性命，舉劍的手卻不忍下手，海倫重回了莫內勞斯的身邊。而特洛伊城則永遠的消失。[[希羅多德|希羅多德]]稱她是希臘世界最美麗的女人，古希臘盲詩人[[荷馬|荷馬]]在《伊利亞特》（''Iliad''）中，描述海倫“白皙的手臂”、“飄飄的長袍”、“閃閃發光的面紗”。
