'''海阳市'''是中国[[山东省|山东省]]一座[[县级市|县级市]]，由[[烟台市|烟台市]]代管。海阳市总面积1886.84平方公里，总人口69万，地处[[黄海|黄海]]之滨、[[胶东半岛|胶东半岛]]南翼，东经120°50至121°29，北纬36°16至37°10之间，位于[[青岛|青岛]]、[[威海|威海]]和[[烟台|烟台]]的中心地带，东与[[日本|日本]]、[[韩国|韩国]]隔海相望。

== 历史沿革 ==

海阳市二千多年前已有人类居住，上古时代为东夷地，[[夏朝|夏]]、[[商朝|商]]和[[西周|西周]]时期为[[青州_(古代)|青州]]地，[[春秋战国|春秋战国]]时期为[[齐国|齐国]]地，[[秦朝|秦]][[漢朝|汉]]属[[胶东郡|胶东郡]]，[[唐朝|唐]]、[[宋朝|宋]]、[[元朝|元]]属山东[[莱州府|莱州府]]。[[明朝|明]]属山东[[登州府|登州府]]。[[洪武|洪武]]三十一年（1398年），[[明廷|明廷]]析登州府地置[[大嵩卫|大嵩卫]]。[[清朝|清]][[雍正|雍正]]十二年（1734年），[[清廷|清廷]]裁大嵩卫置[[海阳县_(清朝)|海阳县]]，据《登州府志》：“以其地在海之阳，故名。”1996年中央政府裁县置海阳市。

== 範圍與交通 ==

海阳市总面积1886.84平方公里，总人口69万，东临[[威海|威海]]，西接[[青岛|青岛]]，北连[[烟台|烟台]]，[[青威高速公路|青威高速公路]]、[[荣乌高速公路|荣乌高速公路]]横贯东西，[[烟海高速公路|烟海高速公路]]、[[海凤铁路支线|海凤铁路支线]]纵贯南北，[[青烟荣城际轨道交通|青烟荣城际轨道交通]]从北部穿越。

海阳市作为胶东半岛海滨城市，海路交通发达，从海阳港出发的货运航班连接[[大连|大连]]、[[青岛|青岛]]、[[宁波|宁波]]、[[福州|福州]]、[[汕头|汕头]]、[[广州|广州]]，海阳市城区距胶东三大国际機場均不到60分钟车程，滨海大道西侧经由海即跨海大桥接入青岛市区。

== 气候植被 ==

海阳市北依胶东半岛[[崂山|崂山]]山系和[[昆嵛山|昆嵛山]]系，南面西太平洋，气候宜人。暖[[温带季风气候|温带季风气候]]，四季分明，降水充沛，冬无严寒，夏无酷暑，年无霜期长达200多天。1970年代以来年平均气温12.0℃，年平均降水量694.5mm。

海阳市四季气候特征如下：春季平均气温10.6℃，降水量119.6mm。由于受海洋气候调节，气温回升缓慢，大部年份无晚霜冻危害，有利于各种农作物及林果的种植、生长和发育。夏季平均气温23.4℃，降水量417.4mm。盛夏盛行东南季风，气候凉爽，是盛夏避暑胜地。秋季平均气温14.2℃，降水量126.6mm。由于北方冷空气的南下，多受大陆变性冷高压控制，秋高气爽，天高云淡。冬季平均气温-0.4℃，降水量30.9mm。虽受大陆气团西北季风控制，但无寒冷干燥之感觉，在山东半岛东西向山脉的阻挡下，冷流降雪少。受黄海海洋气候影响，气温降幅度小，无寒冷之感。

== 城市特点 ==

海阳市是中国首批沿海开放城市之一，有“东方[[夏威夷|夏威夷]]”之美誉，为中国优秀旅游城市，中国最佳文化生态城市，亚洲沙滩运动会承办城市。碧海金滩，茂山俊岭，名刹古寺，北国竹林为这个滨海小城的旅游名片。

海阳市地理位置得天独厚，位居胶东半岛三大著名海滨城市的节点，为中国稀有的背山面海坐北朝南的滨海小城，山俊海瀚，自然秀美，南向绵延30余公里的中国最大的天然海滩沙细、浪稳、坡缓、水清，是著名的避暑胜地。“度假的天堂，休闲的家园”是海阳市的城市品牌。

海阳市为亚洲四大体育赛事之一的[[亚洲沙滩运动会|亚洲沙滩运动会]]主办地，是中国最大的国际沙雕艺术节所在地，也是中国沙滩排球、沙滩足球国家队训练基地。海阳市拥有亚洲独有的苏格兰林克斯风格海滨[[高尔夫球|高尔夫球]]场，被美国高尔夫杂志评为中国十大顶级高尔夫球场之一。

海阳境内著名景点有海阳万米海滩浴场、招虎山[[国家森林公园|国家森林公园]]、云顶竹林自然风景区、丛麻禅院、海阳旭宝国际[[高尔夫|高尔夫]]球场、中国海阳国际[[沙雕|沙雕]]艺术公园、中国沙滩[[排球|排球]]运动基地、凤城水上运动中心、凤凰滩国家[[湿地公园|湿地公园]]、千里岩岛海洋生态系统保护区。

== 亚洲沙滩运动会 ==
海阳市是继印度尼西亚[[巴厘岛|巴厘岛]]和阿曼[[马斯喀特|马斯喀特]]之后的2012年亚洲沙滩运动会举办地。作为亚洲四大综合体育赛事之一，海阳市是唯一获得举办资格的中国海滨城市。

2006年12月2日[[亚奥理事会|亚奥理事会]]在[[卡塔尔|卡塔尔]][[多哈|多哈]]宣布，中国海阳市承办[[2012年亚洲沙滩运动会|2012年亚洲沙滩运动会]]。2008年4月，亚奥理事会与中国国家体育总局签署了2012亚洲沙滩运动会承办合同。
