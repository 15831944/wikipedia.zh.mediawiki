'''海丰县'''（传统外文：{{lang|pinyin|'''Hoifung'''}}）取义“臨海物丰”<ref name="cctv" />，是[[中国|中国]][[广东省|广东省]][[汕尾|汕尾]]市下辖的一个县，位于[[广东省|广东省]]东南海滨，处在[[惠州市|惠州市]][[惠东县|惠东县]]和[[汕尾市|汕尾市]][[陆丰市|陆丰市]]之间。

== 歷史 ==
=== 古代 ===
海丰何时置县，史存二说。一说汉，依据[[唐代|唐代]][[杜佑|杜佑]]《[[通典|通典]]》和[[宋代|宋代]]乐史《太平寰宇记》皆记录海丰为“汉旧县”。又有一说[[東晉|東晉]][[咸和_(东晋)|咸和]]六年（331年）始建海豐縣。

西汉元鼎六年（前111年）汉闽越王郢出兵攻南越，南越请求内附。南越平定后，建为汉十三州之一的交州，又增设了两个县揭阳县（今潮汕梅榕一带）、中宿县（今清远一带）。在汉代，海丰全境属交州南海郡傅罗县。东晋咸和元年（326年）析南海郡东部置东官郡。咸和六年（331）年，从博罗县析出宝安县、海丰县、安怀县（今东莞市）；同年，原揭阳县拆为海阳县、潮阳县、绥安县（今福建省云霄县）、海宁县等四县。海丰县，隶属东官郡，辖地包括现海丰全境及惠来、普宁、揭西之部分地区。

[[隋文帝|隋文帝]]开皇十一年（591），并梁化郡、东官郡和南海郡一部分置循州（郡治今惠州），海丰改属循州。隋炀帝大业三年（607）改州为郡，循州改为龙川郡，海丰属龙川郡。

[[唐高祖|唐高祖]]武德五年（622），龙川郡复为循州，海丰划东部置安陆县（今陆丰县）。太宗贞观元年（627），安陆县复归海丰县，隶属岭南道循州。
[[武则天|武则天]]天授元年（690）改循州为雷乡州。玄宗天宝元年（742）改雷乡州为海丰郡（郡治在归善）。肃宗乾元元年（758）复改为循州，属循州。

[[五代十国|五代十国]]时期，岭南一带为南汉所辖。大宝元年（958），改循州为祯州，属祯州。

[[北宋|北宋]]真宗天禧五年（1021）因避太子趟祯之讳改祯州为惠州，海丰县属广南东路惠州。

[[元代|元代]]海丰县属江西中书省广东道惠州路。
[[明代|明代]]海丰属广东布政司惠州府。[[嘉靖|嘉靖]]三年（1524）海丰划出龙溪都合置[[惠来县|惠来县]]。

[[清代|清代]]属[[广东省|广东省]][[惠州府|惠州府]]。[[雍正|雍正]]九年（1731）海丰划出石帆、吉康、坊廓3都置[[陆丰县|陆丰县]]。海丰存兴贤、石塘、杨安、金锡4都。旧志称：纵横不过270里，即东40里至白沙，东北50里至伯公凹，东南70里至大德港，皆接陆丰；西130里至鹅埠分水塘，西北80里至赤石大安峒，西南90里至小漠旺官圩，南50里至汕尾海，北40里至栅仔岭，皆抵归善（今惠阳）。

=== 近代 ===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = 1963-05 1963年 海丰县.jpg | caption1 = 1963年的海丰县
}}
[[中华民国|中华民国]]二年（1913年）属广东[[潮循道|潮循道]]；民国9年撤潮循道，改属东江绥靖委员公署；民国19年（1930年）属第十区行政视察专员公署（包括6县），后改属第四区行政督察专员公署。
1927年11月中国第一个[[苏维埃|苏维埃]]政权成立，称「海陆丰苏维埃」。

[[中华人民共和国|中华人民共和国]]成立后，县区域废都约，实行乡镇建制。1952年春，原属惠阳县的小漠、元澳2个建制乡，划为海丰县属。1958年冬原惠阳县的高潭区又划为海丰县属，至1962年7月高潭区复归惠阳县（今惠东县）。1988年1月，经国务院批准，在原海丰、陆丰两县的行政区域上设置地级[[汕尾市|汕尾市]]，并析海丰县南部沿海的汕尾、[[红草镇|红草]]、[[马宫街道|马宫]]、[[东涌镇_(汕尾市)|东涌]]、[[田墘街道|田墘]]、[[捷胜镇|捷胜]]、[[遮浪街道|遮浪]]7镇建置城区；设陆丰县北部山区的[[河田镇_(陆河县)|河田]]、[[河口镇_(陆河县)|河口]]、[[新田镇_(陆河县)|新田]]、[[螺溪镇_(陆河县)|螺溪]]、[[水唇镇|水唇]]、[[上护镇|上护]]、[[南万镇|南万]]、[[东坑镇_(陆河县)|东坑]]8个镇设置陆河县。汕尾市管辖[[城区_(汕尾市)|城区]]、海丰县、陆丰县、陆河县。

1992年底，市城区设出田墘镇、遮浪镇，新建红海湾经济开发试验区。1995年，设原陆丰所辖华侨农场为华侨管理区。以上两区属市政府派出机构。1995年，经国务院批准，陆丰撤县建市（县级市），由省政府直辖，委托汕尾市人民政府代管。2011年5月21日，深汕特别合作区授牌仪式在广州举行。海丰县划出[[鲘门镇|鮜门]]、[[赤石镇|赤石]]、[[鹅埠镇|鹅埠]]、[[小漠镇|小漠]]四镇总面积463平方公里，规划控制面积约200平方公里的“深汕特别合作区”的功能区。2013年底，汕尾市城区、[[红海湾|红海湾]]经济开发区、[[梅陇|梅陇]]农场、鲘门镇四地组合成汕尾红海湾新区。

== 文化 ==
=== 語言 ===
海豐縣大部分地區通行[[海丰话|海丰话]]，属于[[闽南语|闽南语]][[泉漳片|泉漳片]]，接近[[漳州话|漳州话]]，在當地稱之為學佬話、福佬話。由於海豐話與[[陸豐話|陸豐話]]存在很高地相似性，被合稱為[[海陸豐話|海陸豐話]]。

北部部分地區通行[[客家语|客家语]]，屬[[海陆片|海陆片]]，与臺灣[[海陸腔|海陸腔]]音系非常接近。
=== 戏曲 ===
盛行戲曲劇種為海陸豐[[白字戲|白字戲]]（序號217編號IV-74）和[[西秦戲|西秦戲]]（序號191編號IV-47），亦有[[正字戲|正字戲]]（正音戲，序號159編號IV-15，主要分布在[[陸豐|陸豐]]市），都已列進[[中國|中國]][[國務院|國務院]]2006年發表的第1批[[國家級非物質文化遺產|國家級非物質文化遺產]]名錄。

== 地理 ==
海豐縣南臨[[南中國海|南中國海]]，沿海多丘陵，西北部群峰起伏，中部則為平原。

== 行政區域 ==
海豐縣下辖16个镇，分别为：[[海城镇_(海丰县)|海城]]、[[附城镇_(海丰县)|附城]]、[[城东镇_(海丰县)|城東]]、[[黄羌镇|黄羌]]、[[公平镇_(海丰县)|公平]]、[[平东镇_(海丰县)|平東]]、[[可塘镇|可塘]]、[[大湖镇_(海丰县)|大湖]]、[[赤坑镇_(海丰县)|赤坑]]、[[陶河镇|陶河]]、[[联安镇|聯安]]、[[梅陇镇_(海丰县)|梅隴]]、[[小漠镇|小漠]]、[[鹅埠镇|鵝埠]]、[[赤石镇|赤石]]及[[鲘门镇|鮜門]]。其中[[深汕特別合作区|深汕特別合作区]]由鮜门、赤石、鹅埠、小漠四镇于2011年5月21日成立，为享有地级市一级管理权限的功能区<ref name="深汕特别合作区概况">{{cite web |title=了解深汕|url=http://www.shenshan.gov.cn/home/about#survey}}</ref>。

== 氣候 ==
一月份均溫為14℃，七月均溫28℃，年均降水量2,382毫米，[[夏季|夏季]]常有[[颱風|颱風]]，為[[廣東省|廣東省]]暴雨中心之一。

== 名人 ==
=== 政治 ===
* [[彭湃|彭湃]]：[[中國共產黨|中國共產黨]]早期領導人之一，中国现代农民运动领袖和先驱。
* [[陈炯明|陈炯明]]：[[辛亥革命|辛亥革命]]元勋，曾任广东省总督、[[中国致公党|中国致公党]]首任[[总理|总理]]。
* [[陳演生|陳演生]]：中國致公黨創黨祕書長。
* [[陳其尤|陳其尤]]：中国致公党第3屆副主席，第4、第5、第6届中央主席。
* [[黃鼎臣|黃鼎臣]]：1927年參加中國共產黨、1946年加入中國致公黨，中國致公黨第7、第8屆中央主席。
* [[林劍虹|林劍虹]]：早期中國同盟會會員，1925年9月29日就任海豐縣縣長
* [[曾广顺|曾广顺]]：1984年5月至1993年任[[中華民國僑務委員會|中華民國僑務委員會]]委员长、[[行政院大陸委員會|行政院大陸委員會]]委员。1988年7月当选[[中國國民黨|中國國民黨]]第十三届中央委员，并任中央常委委员。
*[[黃洋達|黃洋達]]：香港獨立主義政黨[[熱血公民|熱血公民]]領導。

=== 科技专业 ===
* [[彭士祿|彭士祿]]：[[中国工程院|中国工程院]]首批院士、核动力科学家、“中国核潜艇之父”。
* [[彭實戈|彭實戈]]：[[中国科学院院士|中国科学院院士]]、著名数学家、中国金融数学的奠基人。
*[[黄旭华|黄旭华]]：中国第一代核潜艇研发人员、总工程师。

=== 文学 ===
* [[钟敬文|钟敬文]]：中国民间文学和[[民俗学|民俗学]]的开拓者之一，“中国民俗学之父”。

* [[吳大江|吳大江]]：民族音樂家、作曲家、指揮家。
* [[敖嘉年|敖嘉年]]：香港男艺人，电视演員。
* [[林保怡|林保怡]]：香港演員。

* [[徐克|徐克]]：香港導演。
* [[黎耀祥|黎耀祥]]：香港演員。
