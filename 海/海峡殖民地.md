'''海峽殖民地'''（{{lang-en|'''Straits Settlements'''}}），是[[大英帝國|大英帝國]]於1826年至1946年對於馬六甲海峽週邊及鄰近地區各[[殖民地|殖民地]]的管理建制。最初由[[新加坡|新加坡]]（石叻埠）、[[檳城|檳城]]（槟榔屿）和[[馬六甲|馬六甲]]（吗六呷）三個重要英屬港口組成，因此被当时當地[[華人|華人]]稱為'''三州府'''或者'''叻嶼呷'''{{NoteTag|「叻」是「石叻」或「叻埠」（新加坡）、「嶼」是檳榔嶼、「呷」是馬六甲。}}，之後海峽殖民地涵蓋範圍及於[[曼绒县|天定]]及[[納閩|納閩]]。

== 英屬東印度公司時期 ==
1786年，[[英属东印度公司|英属东印度公司]]与[[吉打|吉打]]苏丹签下《[[1786年英吉条约|英吉条约]]》。条约中写明英属东印度公司在吉打受到敌人攻击时需帮忙，还有不能袒护吉打的敌人。并每年给与吉打苏丹30000西班牙元，以换取槟岛拥有权。

1824年，[[荷蘭|荷蘭]]為了獲得整個[[蘇門答臘島|蘇門答臘島]]的統治權，與英國簽訂了《[[1824年英荷條約|英荷條約]]》。在條約中，荷蘭將[[馬來半島|馬來半島]]上的[[馬六甲|馬六甲]]割與英國，以換取英國放棄蘇門答臘上的[[明古連|明古連]]地區。

1826年，英屬東印度公司決定整合在[[馬來亞|馬來亞]]的三個港口屬地[[新加坡|新加坡]]、[[檳城|檳城]]和[[馬六甲|馬六甲]]組成'''海峽殖民地'''。首府初時設於檳城，1832年遷往新加坡。

== 英殖民地部直轄時期 ==
[[File:Flag_of_the_British_Straits_Settlements_(1925–1946).svg|thumb]]
1867年4月1日，英国政府[[殖民地部|殖民地部]]正式接管海峽殖民地，並將其劃入[[皇家殖民地|皇家殖民地]]（Crown Colony）之列，總督駐守於新加坡。與此同時，[[印度洋|印度洋]]上的[[聖誕島|聖誕島]]和[[科科斯群島|科科斯群島]]劃歸海峽殖民地管轄。在此同時，[[海峽殖民地總督|海峽殖民地總督]]兼任英國駐[[馬來亞|馬來亞]]和[[汶萊|汶萊]]這兩個英保護國的高級專員。1874年，隨著《[[邦咯條約|邦咯條約]]》的簽訂，[[曼絨縣|天定]]和附近島嶼被納入殖民地的範圍。1906年10月30日，[[北婆羅洲|北婆羅洲]][[沙巴|沙巴]]的[[納閩|納閩]]港也劃歸海峽殖民地。

在[[太平洋战争|太平洋战争]]期間，海峽殖民地，除了科科斯群島外，都被日軍佔領。而科科斯群島則劃歸[[錫蘭|錫蘭]]代管。二戰後，海峽殖民地各個港口重歸英國統治。在戰後初期，英國對各港口施行軍法統治。

那時候使用的货币为[[叻币|叻币]]，至1939年起改为[[馬來亞元|馬來亞元]]。

== 解散 ==
1946年4月1日，英國宣佈解散海峽殖民地，[[新加坡|新加坡]]成為獨立的一個[[英国海外领土|直辖殖民地]]，並管轄[[科科斯群島|科科斯群島]]和[[聖誕島|聖誕島]]，直至兩島分別於1955年和1957年劃歸[[澳洲|澳洲]]。而[[檳城|檳城]]與[[馬六甲|馬六甲]]便和[[馬來聯邦|馬來聯邦]]四州及另五个[[马来属邦|马来属邦]]合組為[[馬來亞聯邦|馬來亞聯邦]]。而納閩則於同年7月15日劃歸[[英屬北婆羅洲|英屬北婆羅洲]]，成為後來[[馬來西亞联邦直辖区|馬來西亞联邦直辖区]]的一部分。
