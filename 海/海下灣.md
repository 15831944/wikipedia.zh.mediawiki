'''海下灣'''（{{lang-en|'''Hoi Ha Wan'''}}、'''{{lang|en|Jone's Cove}}'''）位於[[香港|香港]][[西貢半島|西貢半島]]北岸，[[地理|地理]]上屬於[[西貢西郊野公園|西貢西郊野公園]]，[[香港行政區劃|行政區劃]]上屬於[[大埔區|大埔區]][[西貢北|西貢北]]，是香港第二個[[香港海岸公園|海岸公園]] - [[海下灣海岸公園|海下灣海岸公園]]的所在地，[[面積|面積]]達2.6[[平方公里|平方公里]]，範圍以[[響螺角|響螺角]]、[[銀洲_(大埔區)|銀洲]]、[[磨洲|磨洲]]至[[灣仔_(西貢)|灣仔]][[棺材角|棺材角]]的尖端連起為界。海下灣西南角有一稱為'''海下'''（{{lang-en|'''Hoi Ha'''}}）的村落，顧名思義，是一個近海的地方（「下」為接近的意思）。

海下灣是一個生態價值高的地區，有高度的[[生物多樣性|生物多樣性]]，這是因為海下灣有一個天然屏障，阻擋海水雜質的湧入，使到海水比較潔淨，從而提高一個良好的海洋環境，是很多海洋有機生物的居所。在海中，大部分的[[珊瑚|珊瑚]]品種可以確定。以前海下灣是一處[[潛水|潛水]]熱點，可是現在需要申請許可。[[香港政府|香港政府]]為了維持海下灣的[[生態系統|生態系統]]，於1996年7月5日將其列為[[香港海岸公園|香港海岸公園]]之一<ref>[http://paper.wenweipo.com/2004/05/16/HK0405160059.htm 地理日記：西貢海下灣——孕育珊瑚之地]《文匯報》2004年5月16日</ref>，並且立例禁止任何影響生態系統的人為活動，例如[[釣魚|釣魚]]、[[捕魚拖網|捕魚拖網]]、使用[[硝化甘油炸藥|硝化甘油炸藥]]或[[氰化物|氰化物]]捕魚、把海產或[[珊瑚|珊瑚]]攜離等。

== 歷史 ==
海下灣於1800年至1939年間是盛行出產[[石灰|石灰]]的。現時在海下灣共發現四個[[石灰窟|石灰窟]]的，只有兩個是比較完整的，它們設置於海下灣的東岸。客家村落[[白沙澳|白沙澳]]多間大屋之壁採用來自海下灣之石灰材料。[[北潭涌|北潭涌]][[上窰民俗文物館|上窰民俗文物館]]（[[區域市政局|區域市政局]][[博物館|博物館]]）所在之[[黃|黃]]氏亦在[[海濱|海濱]]（[[客家話|客家話]]：海唇）之[[灰窰|灰窰]]燒成生產石灰外賣做生理（客家話：生意）而致富興起。

在200年前原籍[[汕頭|汕頭]]的[[翁姓|翁氏]]四兄弟乘船到海下灣建立海下村，初時以漁農為業，並且設有[[石灰窟|石灰窟]]燒[[石灰|石灰]]和[[木炭|木炭]]，所產之木炭以船運載遠赴[[香港仔|香港仔]]出售。現時海下村大部分村屋已租予村外人居住<ref>{{cite web |language=zh-hk |url=http://www.rthk.org.hk/elearning/ecotour/tour03/route03_point01.htm |title=海下村 |publisher=[[香港電台|香港電台]] |date= |accessdate=2010-09-15}}</ref>。

2010年9月14日，根據《明報》報道<ref>{{web reference|URL=http://hk.news.yahoo.com/article/100913/4/k7bj.html|title=日商籌建別墅群 海下告急 購地20萬呎造50幢 目標售予日人|work=明報|year=2010年|date=9月15日}}</ref>，JHC公司於同年8月起在[[日本|日本]]舉辦預售海下灣村屋的[[樓花|樓花]]的邀約活動，銷售對象主要為[[日本人|日本人]]。根據官方網站顯示<ref>{{web reference|URL=http://www.jhccapitalgroup.com/saikon.html|title=香港 西貢 SEASIDE LUXURY RESIDENCE 分譲住宅販売|work=株式会社JHCアセットコンサルティング|year=2010年|date=9月15日}}</ref>，別墅預計於2015年中落成。

== 行政區劃==
海下灣並不劃入[[西貢區|西貢區]]的原因是之前[[西貢|西貢]]並未開通[[西沙公路|西沙公路]]往[[十四鄉|十四鄉]]和未有標準[[道路|道路]]開往[[北潭坳|北潭坳]]、[[黃石碼頭|黃石碼頭]]等地區。而且西貢地區的[[鄉村|鄉村]]如[[深涌|深涌]]、[[荔枝莊|荔枝莊]]、[[榕樹澳|榕樹澳]]，和[[島嶼|島嶼]][[赤洲|赤洲]]等都依賴[[街渡|街渡]]、渡輪往來[[馬料水|馬料水]]和[[大埔滘|大埔滘]]。

== 人口 ==
根據香港政府資料，海下區於2013年總人口為110人，總規劃人口預計約590人<ref>[http://orientaldaily.on.cc/cnt/news/20131015/00176_014.html 海下擬增建村屋 環團反對] 《東方日報》 2013年10月15日</ref>。

== 生態 ==
海下灣像[[米埔|米埔]]一樣，[[海灣|海灣]]內的[[紅樹林|紅樹林]]地區達5.3[[平方公里|平方公里]]，在海下村的紅樹林沙灘總共有7種「真紅樹」，為成長中的[[魚|魚]]、生活於鹹水帶的[[無脊椎生物|無脊椎生物]]的棲息地。海下灣海岸公園有很多[[珊瑚|珊瑚]]，在[[香港|香港]]發現88種[[珊瑚|珊瑚]]中的59種都可以在這裏找到。於1989年5月1日成為[[具特殊科學價值地點|具特殊科學價值地點]]。

== 交通 ==
長久以來，海下灣的居民一直以海路交通出入。20世紀70年代[[香港政府|香港政府]]為興建[[萬宜水庫|萬宜水庫]]興建了[[大網仔路|大網仔路]]、[[北潭路|北潭路]]等接駁道路，海下亦間接受惠，獲興建一條[[海下路|海下路]]連接該區與大灘附近的北潭路。

1982年1月3日，[[九巴|九巴]]開辦了假日特別線95R線（[[北潭涌|北潭涌]]↔海下），方便假日市區遊客郊遊，同年2月亦開辦了[[新界區專線小巴7線|新界區專線小巴7線]]（[[麥理浩夫人度假村|麥理浩夫人度假村]]↔海下），唯95R線於1983年10月30日改為北潭涌↔[[黃石碼頭|黃石碼頭]]，不經海下灣。同日起7號小巴延長至北潭涌總站。2003年7月1日起7號小巴更延長至西貢小巴總站，以方便自海下灣水域劃為海岸公園後的遊客前往海下，同時亦方便海下居民到[[西貢市|西貢市]]購物。
