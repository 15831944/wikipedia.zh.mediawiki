'''海曙区'''，是[[浙江省|浙江省]][[宁波市|宁波市]]的一个市辖区，区内有一座建于唐代的鼓楼[[海曙楼|海曙楼]]，故名。地处宁波市中部，奉化江以西。东接[[鄞州区|鄞州区]]，南靠[[奉化区|奉化区]]，西邻[[余姚市|余姚市]]，北邻[[江北区_(宁波市)|江北区]]，东北部紧邻[[三江口_(宁波)|三江口]]，是宁波中心城区的一部分。距省会[[杭州市|杭州]]141千米，首都北京1210千米。全区面积595.03平方千米<ref>{{Cite web|url=http://www.nbch.gov.cn/nbplanch/news/201803/n74564.html|title=宁波市第一次地理国情普查公报|accessdate=2018-03-17|author=宁波市规划局|date=2018-03-14|publisher=|archive-url=https://web.archive.org/web/20180317035556/http://www.nbch.gov.cn/nbplanch/news/201803/n74564.html|archive-date=2018-03-17|dead-url=yes}}</ref>。

2017年末，全区常住人口90.9万，户籍人口62.6万<ref>{{Cite web|url=http://gtoc.ningbo.gov.cn/col/col140/index.html|title=宁波市人民政府 公民站 人口状况|accessdate=2018-03-17|work=gtoc.ningbo.gov.cn}}</ref>。下辖9个街道、7个镇和1个乡，共有103个社区居民委员会和167个行政村<ref>{{Cite web|url=http://gtoc.ningbo.gov.cn/col/col138/index.html|title=宁波市人民政府 公民站 行政区划|accessdate=2018-03-17|work=gtoc.ningbo.gov.cn}}</ref>。

== 自然地理 ==
海曙区位于宁波市中心，东隔[[奉化江|奉化江]]与[[鄞州区|鄞州区]]相连，北临[[姚江|姚江]]与[[江北区_(宁波市)|江北区]]相接，西与[[余姚市|余姚市]]接壤，南与[[奉化区|奉化区]]相连。海曙区属[[亚热带季风气候|亚热带季风气候]]区，年均气温16.7[[摄氏度|摄氏度]]，无霜期230天至240天，多年平均降水1480毫米。

== 历史沿革 ==
海曙区有人居住的历史可以追溯到西汉。[[东晋|东晋]]时，[[刘牢之|刘牢之]]为防范[[孙恩|孙恩]]起义军，曾修建筱墙，目前尚存筱墙巷的地名。[[刘裕|刘裕]]也因此将[[句章县|句章县]]县治搬迁到小溪，即区境内的鄞江镇，海曙区所在区域正式成为宁波的行政中心<ref name="大事记-晋">{{Cite web |url=http://fw.nbwh.gov.cn/art/2008/5/24/art_784_7655.html |title=宁波大事记·晋 |accessdate=2010-09-07 |work= |publisher=宁波文化网 |date=2008-05-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20110128201214/http://fw.nbwh.gov.cn/art/2008/5/24/art_784_7655.html |archivedate=2011-01-28 }}</ref>。此后，区境改属鄮县，后改鄞县。[[唐代|唐]][[长庆|长庆]]元年（821年），明州刺史韩察将明州州治迁到区境内的三江口西岸，三江口从此成为此后1200余年宁波城市中心的所在地<ref name="大事记-唐">{{Cite web |url=http://fw.nbwh.gov.cn/art/2008/5/24/art_784_7652.html |title=宁波大事记·唐 |accessdate=2010-09-07 |work= |publisher=宁波文化网 |date=2008-05-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20110128201209/http://fw.nbwh.gov.cn/art/2008/5/24/art_784_7652.html |archivedate=2011-01-28 }}</ref>。[[清代|清代]]，海曙区为[[宁绍台道|宁绍台道]]治所。1949年，宁波市区从鄞县分立，海曙、[[镇明区|镇明]]两区在三江口成立。1984年，镇明区并入海曙区。2002年鄞县撤县设区，成为鄞州区。2016年，原鄞州区奉化江以西并入海曙区，形成现在的区划。

== 人口 ==
1984年，海曙区与镇明区合并，成立新的海曙区。是年，全区共有70778户，212373人。随着改革开放带来的社会经济发展，企事业兴起，吸引各类人才和外来务工人员等前来落户，户数和户籍人口不断增加，至2008年达到顶峰，总计113522户，306556人。而后，户数基本稳定在11.3万户左右，人口则逐年微降，至2015年，户数为113560户，人口减少至297544人。

== 经济 ==
历史上，海曙区曾成为宁波最繁华的地区。[[宁波话|宁波方言]]中有“走遍天下，弗（不）如宁波江厦”，其中“江厦”，指的就是海曙的[[江厦街|江厦街]]一带地区。目前，海曙区仍然是宁波服务业最为发达的地区。服务业的支柱是金融保险、批发零售、住宿餐饮和房地产业。海曙区商业发达，[[天一广场|天一广场]]是宁波最受欢迎的假日消费场所，被誉为宁波的商业航母。[[和义大道|和义大道]]是宁波新兴的高档消费场所。

== 文化 ==
{{seealso|海曙区各级文物保护单位列表}}
[[File:Ningbo-DrumTower.JPG|200px]]宁波是历史文化名城。长期作为宁波的中心，海曙区拥有大量的文化遗产。[[月湖_(宁波)|月湖]]位于海曙区东部，原为唐代开掘的蓄水池，经过历代经营，成为宁波的文化重地，目前为宁波市级历史文化保护区。[[天一阁|天一阁]]是中国保存最完整的私人藏书楼。[[天封塔|天封塔]]为宁波旧城的标志。[[钱业会馆_(宁波)|钱业会馆]]为宁波钱庄进行[[票据|票据]]交易的场所。目前，海曙区拥有7处[[全国重点文物保护单位|全国重点文物保护单位]]：[[天一阁|天一阁]]（包括秦氏支祠）、[[它山堰|它山堰]]、[[永丰库遗址|永丰库遗址]]、[[宁波天宁寺|天宁寺]]、[[白云庄|白云庄]]、[[钱业会馆_(宁波)|钱业会馆]]和[[紫金街林宅|林宅]]，为宁波所辖区县中最多。此外，还拥有大量省级和市级文物保护单位。

== 交通 ==
海曙区交通便捷。宁波已建成的许多主干道以海曙区为中心，通向其他区县。[[杭甬高速公路|杭甬高速公路]]通过海曙区，[[甬金高速公路|甬金高速公路]]以区境西南部为起点，[[宁波绕城高速公路|宁波绕城高速公路]]穿过区境西部。铁路[[宁波站|宁波站]]位于海曙区，通过[[杭甬客运专线|杭甬客运专线]]接入全国高速铁路网络，并连接[[萧甬铁路|萧甬铁路]]和[[甬台温铁路|甬台温铁路]]。[[宁波轨道交通1号线|宁波轨道交通1号线]]穿过海曙区，为宁波第一条地铁线路。2020年线网规划中另有2号线、4号线、5号线和6号线通过海曙区。[[宁波轨道交通1号线|1号线]]和[[宁波轨道交通2号线|2号线]]的换乘站（[[鼓楼站_(宁波市)|鼓楼站]]）位于海曙区。
