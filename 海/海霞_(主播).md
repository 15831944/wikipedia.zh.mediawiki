'''海霞'''{{BD|1972年|3月9日||}}，[[河南|河南]][[郑州|郑州]]人，中国女新闻播音员、[[中国共产党纪律检查委员会|中国共产党纪律检查委员会]]特邀监察员，现在是[[中国中央电视台|中国中央电视台]]《[[新闻联播|新闻联播]]》的[[主播|主播]]之一。

== 生平 ==
海霞在[[河南省实验幼儿园|河南省实验幼儿园]]读书，小学在[[郑州市纬五路一小|郑州市纬五路一小]]度过，中学就读于[[河南省实验中学|河南省实验中学]]、[[郑州市回民中学|郑州市回民中学]]。<ref name="cat " />海霞在1993年毕业于[[北京广播学院|北京广播学院]]播音系。之后进入中央电视台新闻部播音组工作，曾任中央电视台一套《早间新闻》、《[[晚间新闻|晚间新闻]]》等資訊類節目主播。而晚間節目《[[现在播报|现在播报]]》的创建，使她的知名度和受歡迎程度大大提高。<ref>[http://www.cctv.com/anchor/compere/0047/profile.shtml 海霞個人簡歷]</ref>

2007年，中央電視臺《新闻联播》节目進行人員調整，補充新播音員。12月6日，海霞首次在該節目中擔任主播。后据[[搜狐|搜狐]]官方网站針對新任播音員所做的受歡迎程度调查，海霞獲得47%的支持率，在她與[[李梓萌|李梓萌]]、[[康辉|康辉]]、[[郭志坚|郭志坚]]這四位新人中排名第一。<ref>[http://news.sohu.com/20071211/n253962914.shtml 新闻联播四天连换新面孔]</ref>

2008年，当选第十一届全国政协委员<ref>{{cite news|title=第十一届全国政协委员名单|url=http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|accessdate=2013-04-09|newspaper=新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过)|date=2008年1月25日|deadurl=yes|archiveurl=https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|archivedate=2013年9月28日}}</ref>，代表新闻出版界，分入第四十四组。并担任民族和宗教委员会专委。<ref>{{cite news|title=全国政协信息-海霞|url=http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0015240&wytype=wjwy|accessdate=2013-04-09|newspaper=全国政协委员会}}</ref>

2010年12月，受聘[[中华人民共和国监察部|中华人民共和国监察部]]特邀监察员，并于2013年11月连任<ref name=xjb>{{cite news|title=央视女主播海霞在中纪委的特殊身份|url=http://news.ifeng.com/a/20180124/55430688_0.shtml|accessdate=2018-01-26|newspaper=政事儿综合 }}</ref>。

== 个人生活 ==
海霞的丈夫是罗永章，留美博士，[[清华大学|清华大学]]教授。<ref name="shark " >{{cite web |url= http://ent.163.com/special/00032H0C/cctv1229.html|title= 央视主持人的另一半|publisher=网易 |accessdate=2011年 |work= |language =zh-cn}}</ref>

== 获奖 ==
2007年获得中国播音主持界的最高奖项[[金话筒奖|金话筒奖]]。2018年1月，获得“全国德艺双馨电视艺术工作者”称号<ref name=xjb/>。
