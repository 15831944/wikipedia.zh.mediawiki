'''海牙荷蘭車站'''（{{lang-nl|Den Haag Hollands Spoor}}，簡稱為{{lang|nl|Den Haag HS}}）是[[荷蘭|荷蘭]][[海牙|海牙]]市內最古老的火車站，落在[[阿姆斯特丹－鹿特丹鐵路|阿姆斯特丹－哈勒姆－鹿特丹鐵路]]路線之上。它與[[海牙中央車站|海牙中央車站]]是海牙市的兩個主要車站。

從阿姆斯特丹開往鹿特丹及更南方（[[布魯塞爾|布魯塞爾]]）的火車班次，一般停靠海牙荷蘭車站；而開往[[烏特勒支|烏特勒支]]或更東或更東北方向的火車（也包括往{{link-en|萊頓中央車站|Leiden Centraal railway station|萊頓}}、[[阿姆斯特丹史基浦機場|阿姆斯特丹史基浦機場]]、阿姆斯特丹方向的火車）通常停靠海牙中央車站。部分往南方向行駛的火車兩站皆有停靠。

== 歷史 ==
[[File:Station_Hollands_Spoor_te_’s_Gravenhage,_1843.jpg|thumb]]
1843年12月6日，隨著荷蘭最古老的鐵路[[阿姆斯特丹－鹿特丹鐵路|阿姆斯特丹－哈勒姆鐵路]]向南延伸，海牙荷蘭車站揭幕使用。該鐵路路線在1847年更進一步南拓至[[鹿特丹|鹿特丹]]。在當時，鐵路南延腹地還是一片草原，屬於[[雷斯威克|雷斯威克]]市的轄區。由於缺乏在車站周邊地區執法的人力，雷斯威克便將土地劃讓予海牙市。車站根據它的營運單位{{link-nl|荷蘭鐵路公司|Hollandsche IJzeren Spoorweg-Maatschappij}}命名為「荷蘭車站」<ref name=idb>{{Cite web |last=Hoogland |first=Joyce |date=27 August 2017 |url=https://indebuurt.nl/denhaag/mysteries/waarom-zijn-er-in-den-haag-twee-grote-treinstations~10789/ |title=Waarom zijn er in Den Haag twee grote treinstations? |website=indebuurt Den Haag |language=nl |accessdate=2018-06-26}}</ref>。車站最初的建築體是由{{link-nl|弗萊德利克·威廉·康拉德|Frederik Willem Conrad}}所設計的；直到1891年，為了興建新的、由{{link-nl|德克·馬赫高丹特|Dirk Margadant}}所設計的[[新文藝復興建築|新文藝復興式]]車站建築，才將舊建築拆除<ref>{{Cite web |url=http://www.stationsweb.nl/station.asp?station=denhaaghs |title=station Den Haag HS |website=Stationsweb |language=nl |accessdate=2018-06-25}}</ref><ref>{{Cite web |url=http://www.stationsweb.nl/gebouw.asp?rec=302 |title=stationsgebouw II Den Haag HS |website=Stationsweb |language=nl |accessdate= 2018-06-26}}</ref>。

1870年，荷蘭鐵路公司的競爭對手{{link-nl|荷蘭萊茵鐵路公司|Nederlandsche Rhijnspoorweg-Maatschappij}}在海牙開設了第二個主要車站－海牙萊茵車站（{{lang-nl|Den Haag Rhijnspoor}}），在新建的{{link-en|高達－海牙鐵路|Gouda–Den Haag railway}}路線上。連接兩個車站的鐵路建設在1871年完工。海牙萊茵車站在1973年被拆除，並在原址新建了[[海牙中央車站|海牙中央車站]]。海牙荷蘭車站和海牙中央車站成為海牙市的兩個主要車站<ref name=idb/>。

* 跨市列車：行經海牙－鹿特丹－多德雷赫特－布雷達－恩荷芬，2 班／小時
