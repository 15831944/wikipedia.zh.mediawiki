在[[苏维埃俄国|苏俄]]与[[苏联|苏联]]的历史上，曾经两次存在过海军人民委员部，第一次是1918年—1923年，名称为“Народный комиссариат по морским делам”，第二次是1937年—1946年，名称为“Народный комиссариат по Военно Морского Флота”。

==第一次==
1917年11月8日，[[十月革命|十月革命]]爆发后的第二天，陆海军委员会（{{lang-ru|Комитет по военным и морским делам}}）成立，11月9日，陆海军委员会改称陆海军人民委员会（{{lang-ru|Совет народных комиссаров по военным и морским делам}}），1917年12月6日，该委员会被改组为军事人民委员部（{{lang-ru|Народный комиссариат по военным делам }}）。1918年初，军事人民委员部被分解为[[陆军人民委员部|陆军人民委员部]]（{{lang-ru|Народный комиссариат по военным делам }}）<ref>“Народный комиссариат по военным делам”之所以在前面翻译为“军事人民委员部”而现在翻译为“陆军人民委员部”是根据其率领的军种不同所致。</ref>与海军人民委员部<ref name="A01">[http://warstudy.com/theory/latterday_west/grand_depth/001.xml 图哈切夫斯基与大纵深作战理论]</ref>。[[帕维尔·叶菲莫维奇·德边科|帕维尔·叶菲莫维奇·德边科]]任海军人民委员。1923年11月12日<ref name="A01"></ref>，[[陆军人民委员部|陆军人民委员部]]和海军人民委员部被合并为[[陆海军人民委员部|陆海军人民委员部]]，海军人民委员部随即撤销。

==第二次==
1937年底<ref>[http://news.ifeng.com/mil/history/detail_2010_08/10/1920843_0.shtml 苏联海军司令库兹涅佐夫力主建大海军 被强令退役]</ref>，[[苏联共产党|苏联共产党]]决定将[[苏联海军|红海军]]的相关指挥机构从[[国防人民委员部|国防人民委员部]]中分出来，再次成立海军人民委员部，[[彼得·亚历山德诺维奇·斯米尔诺夫|彼得·亚历山德诺维奇·斯米尔诺夫]]担任第1任海军人民委员。1938年[[米哈伊尔·彼得诺维奇·弗里诺夫斯基|米哈伊尔·彼得诺维奇·弗里诺夫斯基]]担任第2任海军人民委员，1939年4月28日，[[尼古拉·格拉西莫维奇·库兹涅佐夫|尼古拉·格拉西莫维奇·库兹涅佐夫]]担任第3任海军人民委员。[[第二次世界大战|第二次世界大战]]期间，海军人民委员部领导[[苏联海军|苏联红海军]]进行了卓有成效的抵抗入侵战斗。

1946年2月25日，[[苏联最高苏维埃|苏联第二届最高苏维埃第一次会议]]决定海军人民委员部与[[国防人民委员部|国防人民委员部]]合并为[[苏联国防部|武装力量人民委员部]]，海军人民委员部从此撤销。
