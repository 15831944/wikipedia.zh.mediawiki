'''海因茨·诺克'''（{{lang-de|Heinz Knoke}}，{{bd|1921年|3月24日|1993年|5月18日}}）是[[第二次世界大战|第二次世界大战]]中的[[納粹德國空軍|纳粹德国空军]]王牌飞行员。他出击了2000多次。他的击坠数是33架，所有的击坠都是在西欧战场取得的，同时他还有19个未经证实的击坠。 他的击坠数包括19架[[美国陆军航空兵|美国陆军航空兵]]的重型轰炸机 。<ref>Knoke, 1997, p. 191</ref>

== 早年生活 ==
海因茨·诺克1921年3月21日出生于德国[[哈默尔恩|哈默尔恩]]。 1938年7月6日，在参观一次航展时，海因茨·诺克进行了他的第一次飞行，搭乘一架古老的运输机飞行十五分钟。同时他还为进入[[德国空军|德国空军]]接受了初步审查。 1939年11月15日，海因茨·诺克在第11号飞行训练团进行训练。在1940年八月他加入了 JG1，作为库尔中士的下属。库尔是一位目击了入侵波兰和法国的军事行动的有经验的飞行员。

== 1941年至1942年的战争生涯 ==
在1941年初海因茨·诺克收到了他的第一次战斗命令，使他加入 [[JG_52|JG52]]。他的战友们包括未来的空军王牌 [[格爾哈德·巴克霍隆|格尔哈德·巴克霍隆]], [[京特·拉尔|京特·拉尔]] 和[[沃尔特·克鲁平斯基。|沃尔特·克鲁平斯基]]。在1941年6月他短暂参与了入侵苏联的军事行动。1941年7月，海因茨·诺克被转移到 [[JG1|JG1]]。1941年8月28日海因茨·诺克与伊丽莎白·马克夫斯基在[[謝拉茲|谢拉兹]]结婚。

1942年2月，在3./JG1联队中，海因茨·诺克参加了德国战列舰[[沙恩霍斯特级战列巡洋舰|沙恩霍斯特号]]、[[格奈森瑙号|格奈森瑙号]]和重型巡洋舰[[欧根亲王号|欧根亲王号]]的[[海峡冲刺行动|海峡冲刺行动]]。

1942年2月14日海因茨·诺克脱离JG1，他被要求在挪威海岸保护这些船只。3月他回到JG1。 在3月5日，他与别人共享击落一架[[英国皇家空军|英国皇家空军]](RAF)的[[噴火戰鬥機|喷火]]侦察机型号(PRU)的战绩。 它的飞行员ADM Gunn中尉被俘虏。 在1942年10月海因茨·诺克成为JG1的第2'' Staffle'' 的指挥官。他声称他的第一个独自击坠发生在10月31日：一架英国皇家空军的[[布伦海姆轰炸机|布伦海姆轰炸机]]。

== 1943年至1944与美国陆军航空兵的战斗 ==
1943年，美国陆军航空兵的日间轰炸机强度逐渐增加，所以，JG1和JG11的防御行动数量也逐渐增加。在他的第164次行动中，海因茨·诺克摧毁了他的第一个重轰炸机目标，属于第44轰炸机大队的[[B-24|B-24]]轰炸机“''Maisie''”号，这是他在1943年2月26日于[[巴特茨维申安|巴特茨维申安]]击落的，有两名机组成员生还。<ref>'Battles with the Luftwaffe'; Boiten & Bowman, 2001, page 29</ref>而"The Writing 69th"的成员，第一次参与军事行动，也是最后一次参与军事行动的记者[[罗伯特·珀斯特|罗伯特·珀斯特]]也是在这次行动中牺牲的成员之一。<ref name="book">{{Cite book|url=http://www.greenharbor.com/wr69/Knoke.html|title=I Flew for the Fuhrer|last=Knoke, Heinz|publisher=Greenhill Books|accessdate=2014-12-25|year=1997|isbn=1853672637|pages=208}}</ref>

如何有效的进攻拥有重火力的轰炸机是1943年初[[德国空军|德国空军]]面临的一道难题。海因茨·诺克和他的朋友[[迪特尔·格哈特|迪特尔·格哈特]]（1943年3月18日在拦截[[B-24|B-24]]的行动中丧生），开发了空中轰炸的方法，打破了紧密的[[箱型编队|箱型编队]]，从而有力的攻击了强大的[[美国陆军航空兵|美国陆军航空兵]]轰炸机编队。

3月18日，海因茨·诺克获得了他的第五次空战战绩，在[[黑尔戈兰岛|黑尔戈兰岛]]上空击坠一架第93轰炸机大队的B-24。 3月22日，海因茨·诺克使用一枚250kg炸弹成功击落第91轰炸机大队的一架 [[B-17轟炸機|B-17飞行堡垒]]“''Liberty Bell''”号，这架飞机在攻击[[威廉港|威廉港]]后返航途中被击落。B-17在[[黑尔戈兰岛|黑尔戈兰岛]]以西30千米坠入[[北海_(大西洋)|北海]]。因此，他成为第一个用[[炸弹|炸弹]]消灭敌人飞机的战斗机飞行员。 所有的机组成员被杀害。<ref>Weal, 1999, p. 53.</ref> 然而，这种做法很快就受到限制，因为悬挂炸弹严重影响了[[Bf_109戰鬥機|梅塞施密特Bf-109G]]的高空性能，也使得这些飞机容易被护航的战斗机击落。

在1943年4月I/JG1成为新组建的[[JG11|JG11]]联队，海因茨·诺克从属的第2 ''Staffel'' 成为5./JG11.

1943年，海因茨·诺克宣称获得17架击坠，大多数是[[B-17|B-17]]和[[B-24|B-24]]。6月，他又击落第95轰炸机大队的另一架[[B-17|B-17]]。 几天之后，6月25日，海因茨·诺克的手被轰炸机的自卫火力击伤，造成他的拇指的一部分截肢。

1943年8月17日，在拦截攻击[[雷根斯堡|雷根斯堡]]的轰炸机时他再次受伤，这一次他被子弹的碎片击中，他的飞机也被轰炸机自卫火力击伤。 诺克在附近的[[波恩|波恩]]迫降，他的Bf-109G-6报废。

1943年9月27日诺克使用[[Werfer-granate21|Werfer-Granate21 无制导火箭]]击落一架第94轰炸航空团的[[B-17|B-17飞行堡垒]]”Elusive Elcy”号。这是他首次遇到护航的战斗机，他还摧毁了一架第56战斗机大队的[[P-47戰鬥機|P-47雷电]]，飞行中尉H.P杜加斯被杀害。  诺克随后被其他P-47击落并跳伞。

海因茨·诺克在同年10月4日再次被击落。 在正面攻击并击落一架第392轰炸机大队的[[B-24|B-24]]之后，他被机背机枪手击中，诺克弃机，跳进寒冷的北海。 在他战友的掩护下，诺克爬进一个由[[福克沃尔夫Fw_58|福克沃尔夫Fw 58]]投下的充气筏。 两小时后他被一个救生艇救出。

1943年10月10日，诺克获得了他的第18个击坠：一架B-17。然而他的[[Bf-109|Bf-109G]]被P-47击中，机体的75%损坏，迫使他降落在荷兰 [[Enschede_Airport_Twente|特文特]] 机场。

1944年2月4日，诺克再次被击落。 在2月10日联队长[[Günther_Specht|京特·施佩希特]]受伤，诺克成为了II./JG11的代理指挥官。

在3月4日诺克开始领导II./JG11，当时他曾参与对第363战斗机大队的攻击。在一次约60架[[P-51戰鬥機|P-51野马]]空袭[[汉堡|汉堡]]的行动中，美国陆军航空兵失去了12架P-51，诺克宣称自己击落了一架。

1944年4月15日至20日，诺克加入了[[列非希德|列非希德]]实验站，在那里他第一次飞行[[Me_262戰鬥機|梅塞施米特Me262]]喷气式战斗机。

1944年4月28日，诺克被晋升为联队长，因为"勇敢的面对敌人"，并且建立了II./ JG11。当时他23岁，是当时德国空军年龄最小的联队长。

4月29日，诺克在攻击第354战斗机大队的[[P-47|P-47]]的行动中被詹姆斯·卡农上尉击落，直到8月他因为严重的脑震荡一直在住院。在跳伞之前，诺克成功的击落了超前的[[P-47|P-47]]，飞行员詹姆斯·卡农上尉被俘虏。

== 1944年在诺曼底 ==
1944年8月13日，当他仍在恢复时，诺克被转为指挥在诺曼底前线的III./ JG1。8月14日诺克宣称于[[拉内|拉内]]上空，[[阿尔让唐|阿尔让唐]]东南击落一架属于第358战斗机大队的[[P-47|P-47]]( 飞行员S.A.Giamalva中尉 死亡)，次日他又宣称击落另一架。 8月16日，他宣称在[[埃唐普|埃唐普]]附近击落一架喷火式战斗机。8月17日，他击落一架从属于第31航空照相大队的[[P-38|P-38闪电]](飞行员 L.T.伍德中尉 死亡)，同一天他还有一个未确认的[[B-26|B-26掠夺者]]击坠。8月18日，他宣称击落两架[[P-51|P-51]].

8月25日他宣称击落了第354战斗机大队的一架[[P-51|P-51]]，但诺克在交战中被击落。 他跳伞到前线，险些被法国抵抗军队抓获。最终诺克成功回到德军战线并安全返回他的部队。

1944年8月底，在西线的III./ JG1几乎被消灭；诺克被命令转移部队到[[瓦格拉姆山麓费尔斯|瓦格拉姆山麓费尔斯]]，之后转移回德国以补充兵员和重新装备飞机。

在接受命令并转移III./JG1到维也纳后，1944年9月，在一次前往布拉格的汽车旅行中，海因茨·诺克的腿被一个党派埋放的地雷炸成重伤。

在1945年3月，虽然他仍然拄着拐杖，诺克成了[[耶费尔|耶菲尔]]空军基地的指挥官。 他还负责监督威廉港周围防御工事的工作。

1945年4月27日，他被授予 [[Knight's_Cross_of_the_Iron_Cross|骑士铁十字勋章]] (''Ritterkreuz des Eisernen Kreuzes'')。

1945年9月，海因茨·诺克复员。

== 政治生涯 ==
在1951年海因茨·诺克作为[[社会帝国党|社会帝国党]]的一名成员被选举到 [[Landtag_of_Lower_Saxony|下萨克森州立法机关]] ，然而在1952年德意志联邦共和国最高法院宣布该党非法。从1954年起，诺克仍然在政治上作为 [[Gemeinde_Schortens|Gemeinde Schortens]] 教区理事会的成员。

在几年时间内，他还担任Jevel啤酒的比尔森酿酒厂的经理。 

他后来成为一名[[自由民主党|自由民主党]]成员 (FDP)，并在1956年10月当选为社区议会/教区议会成员，在1961年3月、1964年9月和1968年9月的选举中他获得连任。

海因茨·诺克退休于1972年10月，1980年代中期进入[[奥斯纳布吕克大学|奥斯纳布吕克大学]]学习文学和哲学。

== 关于战争时期的着作 ==
在1950年代期间诺克写了一本关于他的战争时期的职业生涯的书《我为元首飞行》，C.Boesendahl在1952年出版本书(英文版本出版于1953年)。 这本书成为经典的航空著作。
