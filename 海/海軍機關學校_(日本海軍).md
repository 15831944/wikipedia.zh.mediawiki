{{日本海軍}}

'''海军机关学校'''（{{jpn|j=海軍機関学校 |hg=かいぐんきかんがっこう |rm=Kaigun Kikan Gakkō |f=h}}）是[[大日本帝國海軍|旧日本海軍]]内专门培养轮机科（{{link-ja|機関科}}）军官及士官的一所学校。海军机关学校的首次创办是在1881年（明治14年），1887年（明治20年）曾一度合并入[[海軍兵學校|海軍兵學校]]内。1893年（明治26年）海军机关学校再度创立，这次一直延续到1945年（昭和20年）[[太平洋战争|太平洋战争]]结束废校为止。

海军机关学校与[[海軍兵學校|海軍兵學校]]（兵科）、{{link-ja|海軍經理學校 (日本海軍)|海軍経理学校|海軍經理學校}}（主计科）并称“旧海军三校”。

== 历史 ==
[[file:Naval_museum_Japan01s3200.jpg|thumb]]
明治初年创立[[海軍兵學校|海軍兵學寮]]时，学员都在同一所学校学习相关知识。1874年（明治7年）4月24日，日本海军根据外聘的英国教员的建议，为了让轮机学员能在横须贺造船所进行实地学习，而在横须贺村四番地征得一栋房舍，并在向山接收一栋官舍，就此开始办学，而学校始称'''兵学分校'''<ref>海軍機関学校，[[#海軍機関学校一覧|#海軍機関学校一覧]]，4页</ref>。1878年（明治11年）11年1月兵学分校搬迁到造船所，6月4日改称'''兵学校附属机关学校'''<ref>海軍機関学校，[[#海軍機関学校一覧|#海軍機関学校一覧]]，5页</ref>。1881年（明治14年）7月28日改成海军机关学校<ref>海軍機関学校，[[#海軍機関学校一覧|#海軍機関学校一覧]]，6页</ref>。1887年（明治20年）7月，机关学校由于机构改革而并入兵学校，原机关学校学生转移到筑地海军兵学校；因此机关学校第4期生也编入兵学校内，成为兵学校第16期生。而海军机关学校的名称由机关师和机关手养成所沿用<ref>海軍機関学校，[[#海軍機関学校一覧|#海軍機関学校一覧]]，7页</ref>。

由于日本海军体制的原因（如轮机系统的海军军官最多只能晋升至海军轮机中将，也没有可能指挥军舰，遑论执掌舰队或者[[鎮守府_(日本海軍)|鎮守府]]等要职），导致兵科和轮机科合并办学时期，轮机科招生数量不足。为了解决这个问题，1890年（明治23年）10月18日，在兵学校内再次把将校科（兵科）和轮机科分设，对两者采取不同的培养方式<ref name=一覧8>海軍機関学校，[[#海軍機関学校一覧|#海軍機関学校一覧]]，8页</ref>。

1893年（明治26年）11月29日海军再次在横须贺设立海军机关学校，重新独立办学<ref name=一覧8/>。日本海军在[[甲午战争|甲午战争]]后制定了第一期、第二期海军扩张计划，急需大量扩充轮机人员数量。为此校方在横须贺白滨征收民用土地，建设新校舍<ref>海軍機関学校，[[#海軍機関学校一覧|#海軍機関学校一覧]]，12页</ref>。1901年（明治34年）9月1日白滨新校舍落成，总建造费用26万余日元<ref>海軍機関学校，[[#海軍機関学校一覧|#海軍機関学校一覧]]，15页</ref>。

1923年（大正12年）[[關東大地震|關東大地震]]，机关学校的校舍全部烧毁，学校只能临时迁往江田岛海军兵学校内，与兵学校一起授课。1925年（大正14年）在[[京都府|京都府]][[舞鶴市|中舞鹤]]，在原舞鹤[[海兵团_(日本海军)|海兵团]]旧址重新建设校舍和设施。新校址学校建筑占地约{{convert|40000|tsubo|m2|0|kl}}，另有{{convert|20000|tsubo|m2|0}}的练兵场（1944年的小册子则提到校舍占地20万平方米，训练场5万平方米<ref>旺文社，[[#海軍への道|#海軍への道]]，124页</ref>）。学校拥有一艘[[巡洋舰|巡洋舰]]、一艘[[潜艇|潜艇]]和一队[[驱逐舰|驱逐舰]]作为练习舰，还有一艘1000吨左右的附属舰艇也可以作为练习舰。<ref>海軍省海軍軍事普及部，[[#海軍兵学校・機関学校・経理学校現状|#海軍兵学校・機関学校・経理学校現状]]，27页</ref>

[[File:Japanese_navy_Engineering_College.jpg|thumb]]
1942年11月，日本海军终于推行争论多年的{{link-ja|海军轮机科问题|海軍機関科問題|兵机一体化}}政策，将一直以来兵科与轮机科军官分别管理的情况加以改变，而轮机科军官也得以与兵科军官合并成为“军官”。1944年10月，机关学校改为'''海军兵学校舞鹤分校'''。而“海军机关学校”一名，则由原{{link-ja|海軍工機學校 (日本海軍)|海軍工機学校|海軍工機學校}}继承。

战后的1945年11月30日，舞鹤分校闭校。

== 课程 ==
海军机关学校的目标是为日本海军培养轮机方面的人才。轮机人员在海上时指挥轮机科员、负责轮机设备运作；在陆地上则是研究、设计、制造轮机设备、飞机；对轮机科员进行教育；从事燃料、材料等的生产、供给、研究、调查等等工作<ref>海軍省海軍軍事普及部，[[#海軍兵学校・機関学校・経理学校現状|#海軍兵学校・機関学校・経理学校現状]]，26页</ref>。

机关学校的主要科目包括：军用轮机（军用[[蒸汽机|蒸汽机]]、[[内燃机|内燃机]]、[[电动机|电动机]]以及其他等），轮机设计和制图（设计、作图、素描），军制及兵学（军制、兵术、炮术、航海术、运用术（日本海军内指如何使用各种舰艇设备）、水雷术、通信术）、精神科学、数学、基础物理及基础化学、应用力学、热力学、电学、航空理论、船体、外语（主修英语，选修德语或者法语；1944年的一本小册子提到还有汉语<ref>旺文社，[[#海軍への道|#海軍への道]]，128页</ref>）、乘舰实习。<ref>海軍省海軍軍事普及部，[[#海軍兵学校・機関学校・経理学校現状|#海軍兵学校・機関学校・経理学校現状]]，28页</ref>

学校为了锻炼学生的体魄和精神，还利用学校附近的山地，开展冬季滑雪运动，以加强学生在寒冷地区的适应能力。此外也举行[[橄欖球類運動|橄榄球赛]]。<ref>海軍省海軍軍事普及部，[[#海軍兵学校・機関学校・経理学校現状|#海軍兵学校・機関学校・経理学校現状]]，31页</ref>

== 毕业去向 ==
学生从机关学校毕业后即成为海军轮机少尉候补生（{{lang-ja|海軍機関少尉候補生}}），进行一次远洋航行。这次远洋航行结束后，又要在舰队生活大约半年，才能正式成为海军轮机少尉<ref name=宝典84>本間晴，[[#海軍軍人志願者宝典|#海軍軍人志願者宝典]]，84页</ref>。

此后海军会挑选海军轮机中尉和海军轮机大尉前往{{link-ja|海軍工機學校 (日本海軍)|海軍工機学校|海軍工機學校}}学习高等科；有志于潜艇或者航空技术者，可以到各{{link-ja|海軍潜水學校 (日本海軍)|海軍潜水学校|潜水学校}}或者[[大日本帝國海軍航空隊列表|航空队]]接受高等专科教育。海军轮机大尉还能以轮机学生的身份到到[[海軍大學校_(日本海軍)|海軍大學校]]学习。此外，可以到[[东京外国语大学|东京外国语学校]]学习语言，或者到海军大学校、[[帝國大學|帝國大學]]研习专门的学科（如航空学、机械工程学、电气工程学、冶金、地质、采矿、化学、劳动关系、造船学、船用引擎等等）。也有部分人到海外进行留学。<ref>海軍省海軍軍事普及部，[[#海軍兵学校・機関学校・経理学校現状|#海軍兵学校・機関学校・経理学校現状]]，33页</ref>
