'''海上碑'''位于[[中国|中国]][[山东省|山东省]][[日照市|日照市]][[岚山区|岚山区]][[岚山头街道|岚山头街道]]岚阳路社区南、海州湾黄海岸边的天然岩石群上，北侧正对岚山头龙王庙，西北距原[[安东卫城墙|安东卫城]]约4公里，为一处明末清初的海上摩崖石刻。2009年12月被列入第一批[[日照市文物保护单位|日照市文物保护单位]]，2013年10月被列入第四批[[山东省文物保护单位|山东省文物保护单位]]。

== 历史 ==
海上碑始刻于清顺治二年（南明弘光元年、隆武元年，1645），正值明清交替之际，最初题字者为安东卫人、官至明监察御史的[[苏京_(崇祯进士)|苏京]]<ref>清·陈懋修、张庭诗纂，《光绪日照县志·卷六·选举》：“苏京，卫籍，（崇祯）丁丑，官至监察御史、建宁兵备道。”</ref>和河南孟津人、官至明礼部尚书的[[王鐸_(天啟進士)|王铎]]。二人均先为明臣，清军入关后曾往南京参与筹划复明事宜，复明无望后仕清。时苏京复明失败后一度返回故里隐居，王铎来此寻访，二人相继在海边同一块礁石上题字，苏京题“星河影动”、“撼雪喷云”，王铎题“万斛明珠”、“砥柱狂澜”<ref>{{cite web |url=http://paper.dzwww.com/qlwb/data/20090121/html/21/content_1.html|title =《观“海上碑”》|accessdate=2019-02-16 |author= 李学广|publisher=大众网}}</ref>，礁石右下尚留有空缺。清代康熙年间，山西榆次人阎毓秀在此任安东卫守备（康熙十年至十八年间，1671-1679）<ref>《光绪日照县志·卷五·秩官》：“安东卫守备阎毓秀，榆次，武进士，十年任，戆直廉静，不阿上官，不徇私情，任九载，地方安堵，有去思碑。”</ref>，于礁石空缺处补题“难为水”<ref>{{cite web |url=http://rizhao.dzwww.com/rzxw/201707/t20170706_16031533.html|title =《周平和：岚山“海上碑”，是万里海疆第一碑》|accessdate=2019-02-16 |author= 赵洪芝、侯彦平|publisher=大众网}}</ref>。

== 内容 ==
海上碑所在的岸边有一片东西长150米、南北宽30米的天然岩石群，题字即在最南侧的岩石上，坐南朝北，东西长4米，上下高4米，皆为楷书，其中右侧三碑文为横书，左侧二碑文为竖书（左右以北面为准），两侧碑文相距1.88米。北为面，可面朝大海观赏碑文，南为背，可抵御海潮冲击并避免贝类覆盖，从而保护碑文。涨潮时部分碑文没入水中，落潮时则露于浅滩之上<ref name="文物">{{cite web |url=http://www.sdww.gov.cn/hdjl/show-4734.html|title =《日照文化遗产寻访——涛声依旧海上碑》|accessdate=2019-02-16|author= 日照文物管理办公室 |publisher=山东省文化和旅游厅}}</ref>。

三人题字均与水有关，苏京与王铎的题字描述了不同时刻在此观海的情景和感受，阎毓秀的题字则出自《[[孟子|孟子]]·尽心上》“观于海者难为水”，也见于唐代[[元稹|元稹]]《离思五首·其四》“曾经沧海难为水”，一般理解为两层意思，其一为直观表达“难为水”，即临此沧海、别处的水再不足为顾，其二则暗喻面对此景，苏京、王铎的题字已无人可比<ref name="文物" />。

此外，据《日照县志》所载，在日照境内原有另一处海上碑，位于原石臼岛（今[[东港区|东港区]][[石臼街道|石臼街道]]海滨），为明代日照人、官至真定府同知的[[申安_(日照)|申安]]所书“天风海涛”、“万泒朝宗”（“泒”通“派”），今已不存<ref>《光绪日照县志·卷一·疆域·古迹》：“海上碑，石臼岛碑二，一天风海涛、一万泒朝宗，申安书。岚山头碑二，一星河影动，一撼雪喷云，苏京书。”另见《卷八·人物》：“申安，字仁宇，由举人官真定府同知。”</ref>。
