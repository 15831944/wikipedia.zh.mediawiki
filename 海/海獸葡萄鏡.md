'''海獸葡萄鏡'''，又稱海馬葡萄鑑<ref name="銅瑞獸葡萄紋鏡 附清乾隆《寧壽續鑑》鏡匣">{{Cite web |url=http://www.teldapbridge.org.tw/Search/Search_info?id=13177 |title=數位典藏經紀授權中心〈銅瑞獸葡萄紋鏡 附清乾隆《寧壽續鑑》鏡匣〉 |access-date=2017-10-24 |archive-url=https://web.archive.org/web/20171024205649/http://www.teldapbridge.org.tw/Search/Search_info?id=13177 |archive-date=2017-10-24 |dead-url=yes }}</ref>、狻猊葡萄鏡、瑞獸葡萄鏡，是[[唐代|唐代]]的一種銅鏡子。{{noteTag|見路迪民、王大業《中國古代冶金與金屬文物》（1998年由陝西科學技術出版社出版）〈第二章 古代青銅與青銅冶鑄技術〉第209頁。}}

== 歷史 ==
在[[隋代|隋代]]及[[唐代|唐代]]，以瑞獸和[[葡萄|葡萄]]構成主題的紋飾頗為流行。一般相信海獸葡萄鏡出現於[[唐高宗|唐高宗]]在位及[[武則天|武則天]]執政年間（649年至705年）<ref>{{web reference|URL=http://paper.people.com.cn/rmrb/html/2016-06/05/nw.D110000renmrb_20160605_3-12.htm|title=毛敏〈盛唐迷鏡：海獸葡萄紋銅鏡（國寶華光·海上絲路系列）〉|work=人民日報|year=2016年|date=6月5日}}</ref>。因為考古學家曾於1958年[[陝西|陝西]][[西安|西安]]洪慶村獨孤思貞（死於武則天在位時）的墓穴中出土過葡萄鏡{{noteTag|見楊泓《華燭帳前明: 從文物看古人的生活與戰爭》（2009年由香港城市大學出版社出版）第171頁。}}，反映該類紋飾在隋代及唐代年間最為流行。但獨孤思貞墓中的葡萄鏡並非最早出土的同類鏡子。最早出現的是在1970年代於陝西禮泉縣馬寨村[[鄭仁泰|鄭仁泰]]墓中出土、產於唐高宗麟德元年（664年）的瑞獸葡萄鏡。<ref name="隋唐五代紀年鏡集成">[http://linghua.web.fc2.com/data/001.html 菱花亭〈隋唐五代紀年鏡集成〉]</ref>事實上，自[[唐高宗|唐高宗]][[麟德|麟德]]二年（665年）至[[唐玄宗|唐玄宗]][[開元|開元]]十年（722年），皇室墓穴普遍都有陪葬品海獸葡萄鏡。海獸葡萄鏡在[[唐代|唐代]]鏡子的紋飾中，構圖最為繁縟，鑄造工藝最為複雜而精湛。葡萄鏡的構圖精巧表現了盛唐期間富麗堂皇的氣象，也反映唐代對外的文化交流。1980年，陝西[[漢中|漢中]]地區[[西鄉縣|西鄉縣]]出土一面直徑29釐米的海獸葡萄鏡，現為[[陝西|陝西]]歷史博物館館藏<ref name="海獸葡萄鏡">[http://www.sxhm.com/index.php?ac=article&at=read&did=10604 陝西歷史博物館〈海獸葡萄鏡〉]</ref>。

== 形狀紋飾 ==
海獸葡萄鏡為伏獸鈕，內區為三龍三獸，外區為瑞獸。鏡背有雀鳥相間環繞，[[葡萄|葡萄]]及葉蔓鋪地的圖案。而紋飾中的瑞獸為西域引進的獅子，即是[[海馬|海馬]]、[[海獸|海獸]]或[[狻猊|狻猊]]。葡萄鏡呈圓形。其直徑長17.7釐米，厚1.7釐米。重量計有1608克。<ref name="中國銅鏡藝術 唐代海獸葡萄鏡">[https://www.chiculture.net/20509/html/b32/20509b32.html 中國文代研究院 燦爛的中國文明〈中國銅鏡藝術 唐代海獸葡萄鏡〉]</ref>有些種類的海獸葡萄鏡以碩大的葡萄葉蔓作地紋，葡萄穗在邊緣區點綴。雕塑方面則有二隻奔逐的野豬、二隻振翅的鳳凰，一對雌雄梅花鹿，二隻長勁細腿翱翔翻飛的山鶴。外區紋飾除了[[鳳凰|鳳凰]]之外，其餘的紋飾在海獸葡萄鏡的類別中都極為罕見。{{noteTag|見收藏家雜誌社《銅器收藏鑑賞圖鑑》（2010年由中國輕工業出版社出版）。}}
