'''海水淡化'''也称'''海水化淡'''、'''海水脱盐'''，是指将[[海水|海水]]中的多余[[盐|盐]]分和[[矿物质|矿物质]]去除得到[[淡水|淡水]]的工序。

海水淡化主要是为了提供[[饮用水|饮用水]]和农业用水，有时[[食用盐|食用盐]]也会作为[[副产品|副产品]]被生产出来。海水淡化在極度缺乏[[降水|降水]]的[[中东地区|中东地区]]很流行，在某些[[岛屿|岛屿]]和船只上也被使用，一部份靠海的[[火力發電廠|火力發電廠]]及[[核能電廠|核能電廠]]也以海水淡化產出用於[[冷卻|冷卻]]爐心的水源。海水淡化需要大量[[能量|能量]]，所以在不富裕的国家經濟效益并不高。而淡化後的副產品「鹵水」（高鹽度[[海水|海水]]），也常有排放問題。

== 使用概況 ==
[[沙地阿拉伯|沙地阿拉伯]]的海水淡化廠佔全球海水淡化總量的24%。[[阿拉伯聯合酋長國|阿拉伯聯合酋長國]]的[[傑貝勒阿里|傑貝勒阿-{里}-]]海水淡化廠第二期是全球最大的海水淡化廠，每年可產生3億立方米[[淡水|淡水]]。

[[利比亚|利比亚]]开始考虑用[[核能|核能]]淡化海水。

使用淡化水的最大百分比的国家是[[以色列|以色列]]，制造海水淡化水占以色列国内用水量的40％。<ref>Pyper, Julia (February 7, 2014) [http://www.eenews.net/stories/1059994202 Israel is creating a water surplus using desalination]. ''EENews''</ref>

== 在自然界的海水淡化 ==
[[File:Saltcrystals_on_avicennia_marina_var_resinifera_leaves.JPG|thumb]]
在[[水循环|水循环]]的海洋水分蒸发是一个自然淡化过程。

[[海冰|海冰]]的形成也是海水淡化的过程。冻结时，[[盐|盐]]被从海水排出。尽管一些盐水被困，海冰的盐度总体比海水要低得多。

[[红树林|红树林]]生长在[[海水|海水]]中；它们通过部分的根捕获[[盐|盐]]，分泌[[盐|盐]]，然后由[[动物|动物]]（通常是[[螃蟹|螃蟹]]）食用。额外的[[盐|盐]]的去除是通过将其存储在[[叶子|叶子]]，然后脱落[[叶子|叶子]]完成。某些类型的红树林它们的叶子有类似海鸟海水淡化腺的方式的工作腺体。
