'''海鮮'''又稱海產[[食物|食物]]，是指利用[[海洋|海洋]][[動物|動物]]作成的料理，包括了[[魚類|魚類]]、[[甲殼類|甲殼類]]（[[蝦|蝦]]、[[蟹|蟹]]）、[[貝類|貝類]]、[[軟體動物|軟體動物]]與[[棘皮動物|棘皮動物]]。雖然[[海帶|海帶]]這類海洋生物也常是被料理成[[食物|食物]]，但是海鮮主要還是針對動物製成的料理為主。狹義上，只有新鮮的海產[[食物|食物]]才能稱為海鮮，經[[乾燥|乾燥]][[脫水|脫水]]處理的海產食物則稱為[[海味|海味]]。

== 歷史 ==
{{main|海鲜的历史}}
*早期人類的文化當中，海鮮是一個重要的食物來源，人類利用簍和籃這類的工具在[[河流|河流]]和[[湖|湖]]中[[捕魚|捕魚]]，[[古埃及|古埃及]]文明中，可見到以魚叉標記的計數方式。

*[[日本|日本]]古代[[繩文時代|繩文時代]]，貝類是他們的主食，[[考古學家|考古學家]]利用這些食用後被丟棄的[[貝類|貝類]]，計算該地當時的人口數量。

== 料理 ==
在世界各地沿海地區對於海鮮的料理各有特色，由[[日本|日本]][[生魚片|生魚片]]的飲食可見一斑，[[華人|華人]][[社會|社會]]中對於魚類的料理方式更是多樣。但也有不少地區雖然亦有海鮮，卻沒有常食海鮮的習慣風氣，如[[印度|印度]]、[[中東|中東]]地區等。

== 选购 ==
[[File:Colossoma_macropomum_ostergaard.jpg|220px]]攤位]]
海上鮮，不但味道鮮美，還含有豐富的營養成份。不過近十年來，世界各地的海域都受到嚴重的污染。若不小心進食到受污染的海產，就會食物中毒，嚴重者更可以導致死亡。 應在可靠的店鋪購買海鮮。選購急凍海產時，應留意店舖的冰凍設備、以及存放方法是否恰當。<ref>{{cite book|author1=代洪宝|title=河鲜海鲜128道|date=2008-01-01|publisher=中国轻工业出版社|isbn=7501962065|chapter=附录选购鱼类小窍门}}</ref>
===魚和蝦蟹===
*買海鮮盡量選購還活著會動的。尤其是買蟹，死蟹最好不要買來吃。
*新鮮的龍蝦是沒腥味，如果熟[[龍蝦|龍蝦]]有腥味，可能已经变质。
*冰凍的海鮮如有濃烈的腥味，可能已開始變質。
===貝殼類===
*蛤蜊、蠔、蚌、蜆之類海產，在烹調之前檢查一下殼是否仍緊閉。
*買新鮮[[鮑魚|鮑魚]]、蟶子或[[象拔蚌|象拔蚌]]等，可用手碰一碰，選取活的、會動的。

==食用==
進食海產類食物前，不妨先參考以下的防海鮮食物中毒的注意事項： 
*不要去吃龍蝦中綠色的「龍蝦肝」。雖然有些人覺得很好吃，但這器官（肝臟）會儲存著毒素，可能會引起食物中毒。
*有甲殼的海產，在烹調之前要用清水將其外殼刷洗乾淨。
*貝殼類海產烹煮前，在淡鹽水中浸約一小時，讓它自動吐出泥沙。浸泡時間不宜過長，否則原來新鮮的海產反會被其中部分腐死了的所污染。
