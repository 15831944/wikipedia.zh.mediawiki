'''海達邨'''（{{lang-en|Hoi Tat Estate}}），動工初期曾稱作「'''深旺邨'''」，是[[香港房屋委員會|香港房屋委員會]]轄下的一個興建中的[[香港公共屋邨|公共屋邨]]，佔地1.8公頃，是「西北九龍填海區六號地盤」的公營房屋建設項目。

該邨坐落於[[香港|香港]][[九龍|九龍]][[深水埗區|深水埗區]][[長沙灣|長沙灣]][[深旺道|深旺道]]之上，介乎[[連翔道|連翔道]]與[[興華街西|興華街西]]交界以東，東面為[[東京街|東京街]]西及[[港鐵|港鐵]][[東涌綫|東涌綫]]及[[西鐵綫|西鐵綫]]的[[南昌站_(香港)|南昌站]]，邨內部分下方為[[廣深港高速鐵路|廣深港高速鐵路]]香港段的隧道，海盛樓原址曾為[[深水埗（東京街西）巴士總站|深水埗（東京街）巴士總站]]。

本屋邨由房屋署總建築師（3）負責總體設計，並由李景勳、雷煥庭建築師有限公司（ALKF）負責詳細設計，[[有利建築|有利建築]]有限公司承建，整個海達邨項目由4座40層高的出租公屋大樓、商場、大型巴士總站、社會福利大樓、大型分區公共圖書館及室內體育館所組成，還設有長跨度行人吊橋橫跨鐵路及公路以連接對面的[[海盈邨|海盈邨]]及[[凱樂苑|凱樂苑]]，以及另一條天橋連接[[凱德苑|凱德苑]]。全邨預計於2020-2021年分三期落成。<ref>{{cite web |author1=ALKF+ |title=深旺邨公共房屋發展項目 |url=http://www.alkf.com/hk/projects/project-detail-tc/?pid=1170 |accessdate=2018-09-26}}</ref>

== 歷史 ==
[[File:Hoi_Tat_Estate_Colour_Test.jpg|240px]]
2009年6月，政府建議興建4幢樓高39至41層公共屋邨，提供約2400個住宅單位，徵詢深水埗區議會的意見。但區議員普遍認為樓宇密度太高，反對政府的建議。到2010年1月12日，政府修訂規劃，將住宅樓宇的數目由4幢減至3幢，單位數目由2400個減至2200個。但區議會拒絕政府建議，並動議當局擱置興建高密度公屋計劃。同年6月再修訂規劃，提出在通州街計劃興建政府綜合大樓。<ref>{{cite web |author1=立法會秘書處 |title=西北九龍填海區第六號地盤 |url=http://www.legco.gov.hk/yr10-11/chinese/sec/library/1011fs06-c.pdf |accessdate=2018-09-26}}</ref>

到2013年3月，政府提出興建5幢39層高公屋的方案，提供2,798個單位，可容納8,600多人，較2010年初建議增加近600個單位。<ref>{{cite news |title=西九龍填海區　擬建公屋居屋 |url=https://hk.news.appledaily.com/local/daily/article/20130305/18185048 |accessdate=2018-09-26 |work=蘋果日報 |date=2013-03-05}}</ref>

2018年3月，政府宣佈將[[綠表置居計劃|綠表置居計劃]]恆常化，海達邨（時稱深旺邨）亦被列入可考慮選址之一，但最終選址[[麗翠苑|麗翠苑]]作為第二個[[綠表置居計劃|綠表置居計劃]]屋苑，而海達邨暫時仍在往後期數備選名單上。<ref>[https://www.hk01.com/社會新聞/172708/綠置居-柴灣道宏照道公屋同納考慮選址-最快年底可申請 柴灣道宏照道公屋同納考慮選址　最快年底可申請]</ref>

== 屋邨資料 ==
屋邨提供3,310個公屋單位，居住9,600人。

== 交通 ==
