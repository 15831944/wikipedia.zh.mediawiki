'''海上自衛隊'''（{{jp|j=海上自衛隊|hg=かいじょうじえいたい||lead=yes|help=no}}，{{lang-en|Japan Maritime Self-Defense Force}}），簡稱'''海自'''（{{jp|j=海自|hg=かいじ||lead=yes|help=no}}<ref>{{Cite web
|url=http://www.mod.go.jp/msdf/kure/info/curry/
|title=呉海自カレー
|accessdate=2016-03-21
|publisher=海上自衛隊 呉地方隊
}}</ref><ref>{{Cite web
|url=https://www.mhi.co.jp/products/detail/uh-60j_rescue_helicopter.html
|title=UH-60J 救難ヘリコプタ（海自）
|accessdate=2016-03-21
|publisher=[[三菱重工業|三菱重工業]]
}}</ref>，{{lang|en|JMSDF}}），是[[日本|日本]][[自衛隊|自衛隊]]的[[海军|海上部隊]]，成立於[[1954年|1954年]][[7月1日|7月1日]]，全體武職人員稱「海上自衛官」。由於現行[[日本国宪法|日本国宪法]]的限制，因此編制上跟武器都是偏重防衛，原則上不配備具侵略性的艦種（如[[航空母艦|航空母艦]]、[[巡洋艦|巡洋艦]]和[[核子動力潛艇|核子動力潛艇]]等）以及兵種（如[[海軍陸戰隊|海軍陸戰隊]]）。其主要任務是防衛日本[[領海|領海]]，但近年來開始參與[[聯合國|聯合國]][[維和行動|維和行動]]。目前兵力約44000人左右，擁有各式艦艇152艘，[[軍港|主要基地]]5座（[[橫須賀基地|橫須賀]]、[[佐世保基地|佐世保]]、{{Link-ja|吳基地|呉基地|吳}}、{{Link-ja|舞鶴基地|舞鶴基地|舞鶴}}、{{Link-ja|大湊基地|大湊基地|大湊}}）。与[[陆上自卫队|陆上自卫队]]不同，海上自卫队是直接沿用了[[第二次世界大战|第二次世界大战]]前的[[大日本帝国海军|大日本帝国海军]]军旗。

由于吸收了[[太平洋战争|太平洋战争]]的經驗，海上自衛隊以對於[[反潛作戰|反潛]]與{{tsl|en|Minesweeping|掃雷作戰|掃雷}}的高度重視而聞名，訓練也集中在這兩項，這兩項是海上自衛隊的長處；其中扫雷排名世界第一，反潜能力仅次于[[美国海军|美国海军]]。海上自卫队拥有4个[[八八舰队|八八舰队]]，各配备1-2艘神盾舰，同时其余的反潜护卫舰均配备近中程防空导弹，弱點則是水面力量面對空中武力顯得脆弱，必須依賴[[航空自衛隊|航空自衛隊]]，而航空自衛隊的主要任務是防衛日本島嶼。

== 近期發展 ==
* 13,500噸[[日向級直昇機護衛艦|日向級直昇機護衛艦]]：首艦[[日向號護衞艦|日向號]]在2009年3月18日成軍服役，次艦[[伊勢號護衛艦|伊勢號]]亦在2011年3月16日開始服役。其機庫共可容納7架直昇機，加上甲板上可停放4架，共計11架。本艦取代榛名級。
* 19,500噸[[出雲級直升機護衛艦|出雲級直升機護衛艦]]：首艦[[出雲號護衞艦|出雲號]]在2015年3月25日成軍服役，次艦[[加賀號護衛艦|加賀號]]亦在2017年3月22日開始服役。其機庫共可容納14架直昇機，加上甲板上可停放5架，共計19架。本艦取代白根級。
* 5,100噸[[朝日級驅逐艦_(2016年)|朝日級驅逐艦]]：建造2艘，首艦{{link-ja|朝日號護衛艦|あきづき (護衛艦・2代)}}在2018年3月7日服役，次艦{{link-ja|不知火號護衛艦|しらぬい (護衛艦)}}於2019年2月27日服役。本級艦取代朝霧級
* 5,000噸[[秋月型護衛艦|秋月型護衛艦]]：建造4艘，首艦{{link-ja|秋月號護衛艦|あきづき (護衛艦・2代)}}在2012年3月服役，全數於2014年服役。本級艦取代朝霧級<ref>{{cite web| url=http://nippon.zaidan.info/seikabutsu/2003/00163/contents/0006.htm|title=海洋白書 2004| publisher=Nippon Foundation| accessdate=2008-02-11}}</ref>。
* 2,900噸級[[蒼龍級潛艇|蒼龍級潛艇]]：本艦為親潮級放大版，以增加作戰航程、潛航性能及靜肅性為設計主要考量。
* 570噸{{link-ja|平島級掃雷艦|ひらしま型掃海艇}}：首艦於2004年通過預算動工，全部於2008至2010年服役，編號為MSC-601~MSC-603。
* 2,950噸級海洋觀測艦：只有一艘{{link-ja|湘南號海洋觀測艦|しょうなん (海洋観測艦)}}編號為AGS-5106。與5,000噸級泛用驅逐艦同時建案，以海洋研究、水文紀錄為主要任務，另擔負情報蒐集、通信中繼、特種管制或救難協助等任務。
* 12,500噸{{link-ja|白瀨級破冰船|しらせ (砕氷艦・2代)}}：只有一艘[[白瀨號破冰船|白瀨號]]（編號AGB-5003）。本艦於2004年展開設計，2007年開始建造，2008年下水，2009年服役，主要支援南極觀測任務，亦可從事潛艦事故救難任務。

==流行文化==
日本海上自衛隊曾多次於動漫裡出現，例如[[名偵探柯南|名偵探柯南]]特別版就曾出現相關的間諜殺人事件；此外《[[次元艦隊|次元艦隊]]》和《[[沉默的艦隊|沉默的艦隊]]》兩套漫畫均是以海上自衛隊為主題。

另外海上自衛隊也與動漫《[[蒼藍鋼鐵戰艦|蒼藍鋼鐵戰艦]]》進行合作，其片頭曲「SAVIOR OF SONG」與片尾曲「Purest Blue」的正式[[音樂影片|音樂影片]]皆開放軍艦給予取景拍攝，還邀請[[淵上舞_(聲優)|淵上舞]]（伊歐娜聲優）、[[沼倉愛美|沼倉愛美]]（高雄聲優）及[[山村響|山村響]]（榛名聲優）參訪[[親潮級潛艇|親潮級潛艇]]、{{tsl|ja|きりしま (護衛艦)|金剛級霧島號護衛艦}}、{{tsl|ja|せとゆき (練習艦)|瀨戶雪號訓練艦}}及[[海上自衛隊吳史料館|海上自衛隊吳史料館]]等，並將畫面收錄於BD/DVD特典中<ref>[https://www.youtube.com/watch?v=vHyNVvw5_Hs 『蒼き鋼のアルペジオ -アルス・ノヴァ-』Blu-ray&DVD第３巻特典映像「渕上舞の海上自衛隊潜水艦部隊訪問記」ダイジェスト - YouTube]</ref><ref>[https://www.youtube.com/watch?v=fUKgu-vrCm4 「山村響の突撃！海上自衛隊」ダイジェスト映像 - YouTube]</ref>。此外海上自衛隊自行推出的產品入浴劑「海自乃湯」亦在2015年1月28日追加推出「[[榛名號戰艦|榛名]]」、「[[伊号第四百一潜舰|伊401]]」及「[[高雄號重巡洋艦|高雄]]」三款。<ref>[http://www.animen.com.tw/Page/News/NewsContent.aspx?ID=NS15010195 以海上自衛隊軍艦為主題！入浴劑「海自乃湯」公開第2彈宣傳影片！將與劇場版動畫《蒼藍鋼鐵戰艦-ARS NOVA-》展開合作 - 新聞區 - Animen] {{webarchive|url=https://web.archive.org/web/20150122140923/http://www.animen.com.tw/Page/News/NewsContent.aspx?ID=NS15010195 |date=2015-01-22 }}</ref>
