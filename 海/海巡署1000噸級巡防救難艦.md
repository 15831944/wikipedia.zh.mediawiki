'''1000噸級巡防救難艦'''，為[[中華民國|中華民國]][[行政院海岸巡防署|行政院海岸巡防署]]依據「強化海巡編裝發展方案」建造之巡防救難艦。<sub><ref name="苗栗艦下水">{{Cite news |url= http://www.appledaily.com.tw/realtimenews/article/new/20141104/500043/ |title= 「苗栗艦」下水　明年6月執勤<!--參照標題（必填）--> |author= 李菁豪 |date= 2014/11/04<!--原文發表日期--> |publisher= 蘋果日報<!--出版者--> |language= zh-tw<!--語言--> |accessdate= 2014/12/28<!--查閱日期--> |quote= <!--參照的相關原文--> }}</ref></sub>

==本級艦特色==
# 本艦設有夜視系統，提升夜間海上搜救能力。
# 本艦為具有直昇機起降平臺之巡防艦，可供5噸級直昇機起降，於五級海象時可持續有效作業，供直昇機緊急起降並裝置直昇機輔助降落系統，可於夜間執行起降救援任務。
# 本船之船體結構強度及完整穩度性能於九級海象時仍具存浮能力，上述相關性能要求，以電腦模擬分析方式驗證。
# 本船船殼設有非伸縮式穩定翼裝置，可兼顧高、低速時之橫搖穩定性能。
# 本船飲用淡水設計容量可供58人在海上持續執行30天任務。
# 本艦備有折臂式吊車,浮捲網及救難撈網，可安全撈起海上遇難人員。
# 本艦艉部拖曳設施，工作負荷30公噸，且拖曳速度5節以上。
# 本艦之海水及泡沫消防系統噴水槍共4支，可同時使用2支，水平射程120公尺以上。
# 本艦設有SUS 304不銹鋼製活動式冷凍櫃。
# 本艦武裝於艦艏裝有[[波佛斯40公釐高射砲|波佛斯40公釐L/70快炮]]1門，於艦艉01甲板後方設有<br />[[T-75_20毫米機砲|T-75 20毫米機砲]]1座，於左右兩舷船舯處設有<br />[[T75班用機槍|T75班用機槍]]2座。
#本艦左右兩舷有兩艘最高航速35節(64.82km/hr)之警備救難艇可從母艦放下執行救援及緝私任務。
