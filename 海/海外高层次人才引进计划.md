'''海外高层次人才引进计划'''是[[中国共产党中央委员会组织部|中国共产党中央委员会组织部]]主管、[[中央人才工作协调小组|中央人才工作协调小组]]组织实施的计划，计划引进2000名左右人才，俗称'''千人计划'''。<ref>
[http://www.lailook.net/kjrs/01/2010-06-28/4163.html 2010中国大陆“千人计划”名单全文]</ref>2018年底之后，该计划已不再公开对外提及<ref>[https://money.udn.com/money/story/5603/3405492 被美國盯上 傳中國引進人才不再提千人計畫]</ref><ref>{{cite news |title=分析：中国“千人计划” |url=http://www.ftchinese.com/story/001081093 |accessdate=2019-01-16 |work=FT中文网}}</ref>。

== 引进条件 ==
* 一般应在海外取得博士学位，原则上不超过55岁，引进后每年在国内工作一般不少于6个月，
* 并符合下列条件之一：
# 在国外著名高校、科研院所担任相当于教授职务的专家学者；
# 在国际知名企业和金融机构担任高级职务的专业技术人才和经营管理人才；
# 拥有自主知识产权或掌握核心技术，具有海外自主创业经验，熟悉相关产业领域和国际规则的创业人才；
# 国家急需紧缺的其它高层次创新创业人才。
