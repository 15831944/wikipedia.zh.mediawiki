== 背景 ==

1949年10月，中华人民共和国成立，[[两岸分治|两岸分治]]。为对抗新生的中华人民共和国政府，中华民国在[[中国领海|中国领海]]执行军事[[封锁|封锁]]——[[关闭政策|关闭政策]]。虽然中华人民共和国方面在其它海域逐步突破军事封锁，但在[[台湾海峡|台湾海峡]]，由于美国支持和中华民国军事实力强于中华人民共和国的缘故，中华民国得以长期维持关闭政策。[[中国人民解放军海军|解放军海军]]水面舰艇与民用船舶类似，在1974年前未通过台湾海峡。中国大陆南北调动小型舰艇——[[鱼雷艇|鱼雷艇]]时，用火车运输<ref name="a5">{{Cite web |url=https://finance.ifeng.com/a/20170614/15454521_0.shtml |title=西沙战事紧哪：1974年我海军战舰首次过海峡为何台军不拦截|accessdate=2019-04-06 |author= |date=2017-06-14 |work= |publisher=凤凰网|language=Zh-hans }}</ref>。大型舰艇则以通过台湾岛以东，绕行[[太平洋|太平洋]]的方式进行南北航行。1965年，[[八六海战|八六海战]]和[[崇武以东海战|崇武以东海战]]后，[[中華民國海軍|中华民国海军]]不再对解放军海军主动出击<ref>{{Cite web |url=https://www.guancha.cn/society/2017_03_29_401115.shtml |title=《西沙风云》电影开拍 会出现蒋介石“协助大陆”的剧情吗|accessdate=2019-04-02 |author= |date=2017-03-29 |work= |publisher=观察者网|language=Zh-hans }}</ref>。

== 首航历程 ==

=== 舰队编制 ===

南下舰艇编队即[[中国人民解放军海军东海舰队|中国人民解放军海军东海舰队]]驻浙江[[舟山|舟山]]的海军护卫舰第6支队第18大队，是当时的主力战斗部队。第18大队原由四艘仿苏联[[里加级护卫舰|里加级]]的[[6601型护卫舰|6601型护卫舰]]编成，分别是[[昆明舰|昆明舰]]（舷号505）、[[成都舰|成都舰]]（506）、[[贵阳舰|贵阳舰]]（507）、[[衡阳舰|衡阳舰]]（508）<ref name="a1"/><ref name="a2"/>。6601型护卫舰是小型近海护卫舰，满载排水量仅1320吨，航速28节。1970年代初，被海军改装为[[导弹护卫舰|导弹护卫舰]]，装备上游号舰对舰导弹发射架两座。四艘军舰中，贵阳舰因入厂中修，未参加编队。昆明舰为编队指挥舰，称一号舰，成都舰为二号舰，衡阳舰为三号舰<ref name="a1"/>。

=== 起航前准备 ===

1974年1月19日，[[西沙海战|西沙海战]]爆发。此前，[[中国人民解放军海军南海舰队|南海舰队]]司令员[[张元培|张元培]]即已向[[中共中央军委|中共中央军委]]、[[中国人民解放军海军|解放军海军]]请求[[中国人民解放军海军东海舰队|东海舰队]]支援<ref name="a2"/>。根据南海舰队兵力不足的现实，海军向中央军委请求，将东海舰队的第18大队调往南海舰队，加入其作战序列。经中央军委副主席[[叶剑英|叶剑英]]、中央军委委员[[邓小平|邓小平]]同意，再报总理[[周恩来|周恩来]]、主席[[毛泽东|毛泽东]]同意，中央军委批准了海军的请求，并要求[[福州军区|福州军区]]负责第18大队通过台湾海峡时的军事行动和掩护任务。此次行动的保密级别为[[机密级|机密级]]，除主要首长和作战部门外，其余参与者并不知情<ref name="a1"/>。

1月19日，海军下达预先号令，当日下午，第6支队进行了图上作业，制定各类航行方案<ref name="a2"/>。海军副司令员[[刘道生|刘道生]]负责编队南下行动的指挥，东海舰队司令员[[马龙_(解放军)|马龙]]在浙江舟山负责布署。由于保密需求，编队以[[春节|春节]]战备出海巡逻任务进行出航准备工作<ref name="a1"/>。

海军要求编队于1月20日<ref name="a2"/>（即[[腊月廿八|腊月廿八]]，1月22日为[[大年三十|大年三十]]）起航，巡逻海域为浙江南部海域。第18大队主官随队出海。同时，第6支队副队长[[严恩明|严恩明]]带领政治部保卫科长在内的10余名机关干部随同出海，他亦是编队指挥员<ref name="a1"/>。

=== 通过海峡前 ===

[[福州军区|福州军区]]方面，由军区司令员[[皮定均|皮定均]]负责指挥南下编队通过台湾海峡的行动。副司令员[[朱绍清|朱绍清]]等人则组成福州军区工作组，在编队抵达前赶赴福建[[沙埕港|沙埕港]]。得到中央军委批准后，皮定均下令：海军福建基地组织多支护卫艇、[[鱼雷艇|鱼雷艇]]编队进入沙埕、东冲、东山、崇武等8个港口内待命出击；福建海岸炮兵进入阵地，做好射击的准备；福建沿海海空军雷达站全线开机，掌握编队在台湾海峡的舰位和空情；海军福建基地派出有经验的航海人员，为编队领航；驻闽空军在福州、漳州机场的歼击机进入起飞待命状态，飞行员要坐在座舱里，等待命令起飞；福州军区技侦部队加强对台湾海空军部队调动出动情况的掌握；所有担负掩护任务的部队进入[[一级战斗准备|一级战斗准备]]，担负任务的领导干部一律坚守岗位。并要求上述部署在1月21日{{efn|name=aa1}}16时前到位<ref name="a1"/>。

1月20日{{efn|name=aa1}}6时许，编队从浙江舟山出发。中午11时许，在[[大陈岛|大陈岛]]抛锚，进行补给。编队首长向官兵传达东海舰队和海军的命令，告之此行真正的目的地。编队于当日{{efn|王昌太的原文<ref name="a1"/>是“于22日15时抵达福建沙埕锚地锚泊”，根据李兆新文章<ref name="b1"/>推论，当在20日。}}15时抵达福建沙埕港。抵达沙埕港后，朱绍清在昆明舰上会见了编队指挥员严恩明。福州军区领航组为编队制定了具体的通过台湾海峡的航线。同时，福州军区工作组要求编队在通过[[东引岛|东引岛]]至[[金門島|金门]]以东之间的海域时，保持[[无线电静默|无线电静默]]<ref name="a1"/>。

=== 通过海峡及后续 ===

1月21日{{efn|name=aa1}}17时30分，南下编队从沙埕港起航，19时在福建[[台山岛|台山岛]]以南海域进入台湾海峡。航行路线是此前领航组制定、福建军区批准的，在军区能提供火力掩护范围内的南下航线<ref name="a1"/>。自福建沙埕港起航后的具体航线是，在福建[[台山岛|台山岛]]进行第一次转向，[[东引岛|东引岛]]第二次转向，[[牛山岛|牛山岛]]第三次转向，[[崇武半岛|崇武]]以东海域第四次转向，龙海[[东锭岛|东锭岛]]第五次转向，广东[[南澎岛|南澎岛]]第六次转向，最终达到广东潮阳企望湾<ref name="a1"/>。

航行中，编队进入[[一级战斗准备|一级战斗准备]]，实行[[灯火管制|灯火管制]]<ref name="a1"/>，各舰前后主炮均炮弹上膛，随时进入战斗状态<ref name="a2"/>。海军福建观通部队，则每10分钟对南下编队进行一次[[雷达|雷达]]定位<ref name="a1"/>。定位以密码通报舰艇，编队依照陆地指挥航行<ref name="b1"/>。同时，定位层层上报至[[中国人民解放军总参谋部|解放军总参谋部]]作战值班室。22日{{efn|name=aa1}}凌晨3时，中央军委副主席[[叶剑英|叶剑英]]与福州军区作战指挥室通电话，他得知南下编队顺利通过乌丘后，表示“那好，你们辛苦啦。”<ref name="a1"/>

22日{{efn|name=aa1}}凌晨6时，南下编队抵达广东潮阳海面，进入南海舰队辖区。[[中国人民解放军海军汕头水警区|海军汕头水警区]]副司令员{{efn|王昌太的原文<ref name="a1"/>提及魏治国的职务是司令员，根据个人简历应是副司令员。}}[[魏治国|魏治国]]带领[[猎潜艇|猎潜艇]]第74大队接应南下编队<ref name="a1"/>。7点31分，汕头水警区的一艘猎潜艇向先导舰——昆明舰发出灯光信号：“海军[[刘道生|刘道生]]副司令员祝贺编队顺利通过台湾海峡<ref name="a2"/>。”随后，完成补给的编队继续南下，于23日17点30分抵达广东[[湛江港|湛江港]]，南海舰队司令员和政委出席欢迎仪式<ref name="a2"/>，后开赴[[西沙群岛|西沙群岛]]<ref name="a1"/>。

1974年年底，完成修整的[[贵阳舰|贵阳舰]]通过台湾海峡南下汇合，归入南海舰队序列<ref name="a1"/>。

== 中华民国（台湾）反应 ==
[[File:金馬前線綜合情況.jpg|缩略图]]

根据中华人民共和国方面的记述，[[中華民國國軍|中华民国国军]]在南下编队自沙埕港起航后，即已掌握、跟踪编队行程。在[[乌丘|乌丘]]以东海域巡航的[[中華民國海軍|中华民国海军]]阳字号驱逐舰编队，自南下编队进入台湾海峡后，转至[[台湾海峡中线|台湾海峡中线]]以东<ref name="a5"/>。国军的一架[[侦察机|侦察机]]亦一路跟踪编队，23日编队抵达广东潮阳企望湾后，国军飞机返回台湾<ref name="a1"/>。其余情况不详，中华民国方面未有公开说明。1980年代末，中国大陆开始流传时任[[中華民國總統|中华民国总统]]的[[蒋介石|蒋介石]]允许编队通过台湾海峡的伪史——[[西沙战事紧|西沙战事紧]]。

2010年代，中华民国[[國史館|国史馆]]的机密档案解密。其中，中华民国国军《金马前线综合情况（74.1.16-1.31）》文件提及当年“元月23日新发现汕头匪海军‘一二一特编部队’曾一度要求提供南海气象资料。判与匪加强南海备战措施有关。”此份文件中，国军将南下编队称为“一二一特编部队”，当是指其通过台湾海峡的时间（1月21日），并准确判断编队南下目的，但国军反应仅是“其后续动作注意中。<ref name="a4">{{Cite web |url=https://new.qq.com/omn/20190223/20190223A0HM97.html|title=蔡“省长”送出神助攻，又一历史谣言被揭穿 |accessdate=2019-04-03 |author= |date=2019-02-23 |work= |publisher=腾讯网|language=Zh-hans }}</ref>”
