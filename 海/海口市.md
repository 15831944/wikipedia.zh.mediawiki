'''海口市'''（[[海南白话字|海南白话字]]：{{sans-serif|'''Hái-kháu-sī'''}}，[[海口话|海口话]]：{{IPA|[hai˨˩˧ xau˨˩˧ si˧˧]}}，[[邮政式拼音|邮政式拼音]]：{{lang|la|'''Hoihow'''}}）是[[中华人民共和国|中华人民共和国]][[海南省|海南省]][[省会|省会]]，位於海南省东北部，[[南渡江|南渡江]]入海口，[[铁路轮渡|铁路轮渡]]入口，北隔[[瓊州海峽|瓊州海峽]]與[[廣東|廣東]][[雷州半岛|雷州半岛]]相对望。

== 历史 ==

[[汉朝|汉代]]时期，海口属[[珠崖郡|珠崖郡]][[玳瑁县|玳瑁县]]地，[[隋朝|隋代]]以后为[[琼山县|琼山县]]地直至[[民国|民国]]十五年建市。[[唐朝|唐朝]]设[[白沙津|白沙津]]，为[[琼州|琼州]]城（府城镇）外滩，设有古渡[[驿站|驿站]]，为当时琼州与[[雷州|雷州]]海渡和贸易的港口。[[南宋|南宋]]时[[海口浦|海口浦]]，范围包括今天的海甸街道与海口市区。之后“海口浦”便逐渐代替“白沙津”，成为海南渡海和物资进出口的主要[[港口|港口]]之一。1271年[[元朝|元朝]]设置番营；1368年[[明朝|明朝]]设置守御千户所，[[明朝|明代]][[洪武|洪武]]二十七年（1395年）开始筑城，按兵制“海口浦”改建制为所（称“海口所”或“海口区”）；相传今海口市的得胜沙路一带，古称“外沙”。[[道光|道光]]二十九年（1849年）遭外寇入犯，激烈战斗后，最后得胜外寇于外沙，得名“得胜沙”。[[清朝|清朝]][[咸丰|咸丰]]年间根据《[[天津条约|天津条约]]》规定成为对外通商口岸；清[[光绪|光绪]]元年（1875年）设置海口海关，对外贸易开始形成与发展。

[[民国|民国]]初期“海口所”改称“海口镇”；1926年12月自琼山县析建“海口市”。1950年4月23日[[中國人民解放軍|中國人民解放軍]]占領海口，[[中華民國國軍|中華民國國軍]]撤守[[臺灣|臺灣]]。中共遂建立了[[中华人民共和国|人民政府]]。1956年，[[中华人民共和国国务院|国务院]]将海口市划为广东省的地级直辖市。1988年4月13日，[[第七届全国人民代表大会|第七届全国人民代表大会]]通过关于设立[[海南省|海南省]]、建立[[海南经济特区|海南经济特区]]的决议，海南建省，海口市成为[[海南省|海南省]]省会。2002年10月16日，[[中华人民共和国国务院|国务院]]批复海口、[[琼山|琼山]]两市合并，成立新的'''海口市'''。

== 地理 ==
=== 地形 ===
海口地处海南岛北部，北濒[[琼州海峡|琼州海峡]]，隔18海里与[[广东省|广东省]][[海安镇|海安镇]]相望；东面与[[文昌市|文昌市]]相邻；南面与文昌市、[[定安县|定安县]]接壤，西面邻接[[澄迈县|澄迈县]]。海口市东起大致坡镇老村，西至西秀镇拨南村，两端相距60.6[[公里|公里]]；南起大坡镇五车上村，北至大海，两端相距62.5公里。全市土地面积2304.84[[平方公里|平方公里]]。

=== 氣候 ===
海口處海南島北部，屬[[熱帶季風氣候|熱帶季風氣候]]，隨著全球氣候不斷暖化已成定局，近年海口冬季溫度也有上升的趨勢，最冷月均溫高於18℃的情形越來越常見。夏季高溫多雨，夏秋季[[熱帶气旋|熱帶气旋]]頻繁。年平均氣溫24.5℃。1月平均氣溫18℃。極端最低氣溫2.8℃（1955年1月12日）。7月平均氣溫 28.8℃，極端最高氣溫39.6℃（2001年4月21日）。平均年降水量1686.4毫米，雨季一般集中在8～10月。

== 旅游 ==
[[File:Tomb_of_Hai_Rui_1.jpg|150px]]]]
[[File:Hainan_Museum_01.jpg|thumb]]大門]]
海口风光秀丽，名胜古迹多，加之近年新开发了不少旅游区和景点，游览内容十分丰富。比较著名的有[[五公祠|五公祠]]、[[海瑞墓|海瑞墓]]、[[丘浚墓|丘浚墓]]、[[李硕勋烈士纪念亭|李硕勋烈士纪念亭]]、[[秀英炮台|秀英炮台]]、[[海口崖州古城|海口崖州古城]]、[[海口人民公园|海口人民公园]]、[[海口儿童公园|海口儿童公园]]、[[万绿园|万绿园]]、[[假日海滩|假日海滩]]共等景点，还有[[金牛岭烈士陵园|金牛岭烈士陵园]]、[[滨海公园|滨海公园]]等景点，虽格局不太大，但也各具一定特色，且位于市内，交通方便，游人可参看《海口市旅游示意图》寻踪游览。并且海口、琼山地域相接，旅游景点参错一体，游人可参看两地旅游示意图，随趣选择多种旅游路线，以足游兴。位於海口市區的[[海南省博物馆|海南省博物館]]近年來亦成為了海口的旅遊熱點，吸引了眾多對海南歷史感興趣的中外旅客。

海口骑楼老街：海口骑楼主要分布于海口市得胜沙路、中山路、博爱路、新华路、解放路、长堤路等老街区，建筑群初步形成于1920年代至1940年代，其中最古老的建筑四牌楼建于南宋，距今有600多年历史<ref>{{Cite web|url=http://www.xinhuanet.com/photo/2018-02/12/c_1122410196.htm|title=海口：骑楼老街年味浓|accessdate=|author=|date=2018-02-12|publisher=新华网}}</ref>。2009年被文化部和国家文物局列为首批十大“中国历史文化名街”之一<ref>{{Cite web|url=http://www.gov.cn/jrzg/2013-01/21/content_2316969.htm|title=海口骑楼首批历史建筑对外开放|accessdate=|author=|date=2013-01-21|publisher=中央政府门户网}}</ref>。

海口的土特产品有咖啡、椰子糖、椰茸、天然椰子汁、椰子酱、香蕉、菠萝及菠萝蜜等热带水果。海口是海南著名的小吃海南粉的主要分布地区<ref>{{cite web |author1=海南香网站 |title=海南岛有名的粉 |url=http://hncx.anhees.com/HainanFood/dhndncsmfm |website=http://hncx.anhees.com/ |accessdate=2018-06-21}}</ref>。

近几年来海口市政府努力加强海口市建设，为打造国际旅游岛的省会投入众多资金，兴建大量工程。
