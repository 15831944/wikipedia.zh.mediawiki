'''海城市'''为[[中国|中国]][[辽宁省|辽宁省]]下辖[[县级市|县级市]]，由[[鞍山市|鞍山地级市]]代管。海城市经济较为发达，在[[县级行政区|县市]]中具备较强竞争力，其综合实力[[中国百强县#县域经济论坛评价|2010年位居辽宁省县（市）的第二位，全国百强县（市）列第十九位]]。全市辖5个管理区、1个开发区、23个镇，400个行政村。是[[辽宁|辽宁]]较大的县级市，全国百强县之一，是全国粮食、水果、畜牧业、[[柞蚕|柞蚕]]、乡镇企业生产基地之一，素有“渤海奥区”之誉。

== 地理 ==
海城市位于辽宁省南部，[[辽河|辽河]]下游东岸，辽东半岛北部。地处东经122°18′－123°08′，北纬40°29′－41°11′之间。行政区域东西长80公里，南北宽44公里。全境年平均气温10.4℃，降雨量691．3毫米，温带季风性气候。

== 历史 ==
海城有[[旧石器时代|旧石器时代]]小孤山洞穴，[[新石器时代|新石器时代]]析木石棚墓葬遗址。[[战国时期|战国时期]]，海城地区隶属于[[燕国|燕国]]的辽东郡。[[秦|秦]]统一中国后，仍属辽东郡。[[汉|汉]]代属辽东郡和玄菟郡，于境内设新昌、辽隧、安市3县，[[东汉|东汉]]一度改属乐浪郡。[[西晋|西晋]]沿袭汉制置新昌、安市2县。[[高句丽|高句丽]]占据辽东后，海城为高句丽腹地。自晋至[[隋|隋]]260余年间，境域为高句丽之卑奢城地。[[唐|唐]]高宗总章元年（668年），唐王朝收复辽东，海城为大唐安东都护府管辖。[[辽|辽]]代，隶属辽东道，于境内置海州、铜州、耀州，同设临溟、仙乡、析木、岩渊4县，并在海州置南海州，兼置临溟县为州治。海城之名源于海州至此始。[[金|金]]代，改海州为澄州，辖临溟、析木2县。[[元|元]]代，属[[辽阳|辽阳]]路，于境内置海州巡检司。[[明太祖|明太祖]]洪武九年（1376），在旧海州临溟县置海州卫（今牛庄），隶属辽东都指挥使司。[[清|清]]世祖[[顺治|顺治]]十年（1653），改海州为海城，属辽阳府管辖。顺治十四年（1657年）辽阳撤府后，改属[[奉天|奉天]]府（今沈阳）。[[中华民国|中华民国]]初期，划属奉天省辽沈道。

中华民国十八年（1929年），改奉天省为辽宁省，海城县隶属辽宁省。1931年[[日本|日本]]侵占东北后，[[满洲國|满洲國]]政權改辽宁为奉天省，海城县遂改属奉天省。1937年，满洲國撤销[[营口|营口]]县，以[[辽河|辽河]]为界，其东划归海城县，西划归盘山县。

1945年日本投降后，[[中国共产党|中国共产党]]从苏联手中接管，隶属于辽宁省辽南行政公署。1946年4月，[[中華民國|中華民國]]政府改东北为九省，海城县仍隶属辽宁省。 1947年10月[[中国人民解放军|中国人民解放军]]重占海城县。1948年2月，在县境地内另置牛庄县，县政府设在今牛庄镇；同年底，牛庄县撤销。1948年11月东北全境解放后，海城县划属辽东省管辖（省会在今丹东市）。1954年，辽东、辽西两省合并为[[辽宁省|辽宁省]]，海城县归辽宁省辽阳专署领导。1959年改属[[鞍山|鞍山]]市。1965年又转属辽南专属（专署驻盖州市）。1966年“[[文化大革命|文化大革命]]”期间，海城县人民委员会（即县政府）被撤销，于1967年3月成立海城县革命委员会，隶属于营口市。1973年又转属鞍山市。1980年7月，县革命委员会改称县人民政府。1985年1月，经中华人民共和国[[国务院|国务院]]批准，撤销海城县，成立海城市（县级市），隶属于辽宁省鞍山市至今。

海城市管辖有4个街道、21个镇。（不含已划归鞍山市千山区的甘泉镇、大屯镇）

街道：
*海州街道、兴海街道、响堂街道、东四街道

镇：
*[[孤山满族镇|孤山满族镇]]、岔沟镇、接文镇、析木镇、马风镇、牌楼镇、[[英落镇|英落镇]]、八里镇、毛祁镇、王石镇、南台镇、西柳镇、感王镇、中小镇、牛庄镇、腾鳌镇、耿庄镇、西四镇、高坨镇、望台镇、温香镇

== 物产 ==
有丰富的菱镁、滑石等矿藏资源。著名特产有[[南果梨|南果梨]]。

== 经济 ==
2008年，全市[[GDP|GDP]]410.4亿元，地方财政收入33亿元，其中80%以上来源于民营经济。市场经济比较发达，西柳服装市场、南台箱包市场、蛋禽市场在国内外都有一定影响。
