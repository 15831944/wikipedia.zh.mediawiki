'''海豚浏览器'''是一个支持[[Android|Android]]、[[蘋果公司|蘋果]][[iOS|iOS]]平台的[[移动浏览器|移动浏览器]]，由百纳信息（Mobotap）开发。它是Android平台上第一款支持[[多點觸控|多點觸控]]手势的浏览器。<ref name="cnet">{{cite web|url=http://download.cnet.com/Dolphin-Browser-for-Android/3000-2356_4-75005194.html|title=Dolphin Browser for Android|author=Jessica Dolcourt|work=CNET.com}}</ref><ref name="engadget">{{cite web|url=http://mobile.engadget.com/2009/11/29/dolphin-browser-for-android-adds-multitouch-support-on-droid/|title=Dolphin browser for Android adds multitouch support on Droid|author=Chris Ziegler|work=Engadget|publisher=AOL Inc.|access-date=2013-01-25|archive-url=https://web.archive.org/web/20110616204737/http://mobile.engadget.com/2009/11/29/dolphin-browser-for-android-adds-multitouch-support-on-droid/|archive-date=2011-06-16|dead-url=yes}}</ref>海豚浏览器使用[[Webkit|Webkit]]引擎和原生平台的渲染，它可以占用小的磁盘空间。<ref name="ars">{{cite web|url=http://arstechnica.com/gadgets/reviews/2011/01/hands-on-dolphin-hd-browser-for-android-is-swimmingly-good.ars|title=Hands on: Dolphin HD browser for Android is swimmingly good|author=Ryan Paul|work=Ars technica}}</ref>

== 特色功能 ==
* '''选项卡式浏览'''：用户可在屏幕上方或缩略图下方的按钮在多个选项卡之间切换。<ref name="cnet"/>
* '''海豚声纳'''用户可以在海豚浏览器里使用语音搜索，分享和导航，而不需要输入一个字。你可以说[[eBay|eBay]]的[[耐克|耐克]]鞋，海豚浏览器将会进入eBay网站，搜索并显示eBay上所有的耐克鞋<ref name="cnet"/>。
* '''手势浏览'''：用户可以创建个人网站的自定义手势，海豚浏览器还配备预先设置好的许多流行的网站的手势。例如，画一个“T”可访问[[Twitter|Twitter]]，画一个“F”可以访问[[Facebook|Facebook]]<ref name="cnet"/>。
* '''网络杂志'''：允许用户以杂志风格来显示网页内容，提供超过300个来源的话题，包括生活方式，音乐，技术和许多其他的选择内容。网络杂志也可缓存内容，脱机使用<ref name="cnet"/>。
* '''同步'''：允许使用Firefox和Chrome扩展跨设备同步。

==隐私问题==
在2011年10月，海豚浏览器曝出隐私问题，引起了人们的担忧，在所有安装海豚HD浏览器的设备上，海豚浏览器会将纯文本[[URL|URL]]转发到远程服务器<ref>{{cite web|last=McCullagh|first=Declan|title=Dolphin HD browser snared in security breach|url=http://news.cnet.com/8301-31921_3-20127195-281/dolphin-hd-browser-snared-in-security-breach/|publisher=[[CNET|CNET]]|accessdate=October 30, 2011}}</ref>，[[Ars_Technica|Ars Technica]]称其为“一个明确的违反隐私”的过程<ref name=atp>{{cite web|last=Paul|first=Ryan|title=Major privacy flaw found in Dolphin HD browser for Android|url=http://arstechnica.com/gadgets/news/2011/10/major-privacy-flaw-found-in-dolphin-hd-browser-for-android.ars|publisher=[[Ars_Technica|Ars Technica]]|accessdate=October 30, 2011}}</ref>。此问题已在之后的版本中修复。<ref>{{cite web|url=http://blog.dolphin-browser.com/2011/10/27/webzine-does-not-store-user-data|title=Webzine Does Not Store User Data|publisher=Dolphin Browser Blog|accessdate=November 29, 2011|deadurl=yes|archiveurl=https://web.archive.org/web/20111129211548/http://blog.dolphin-browser.com/2011/10/27/webzine-does-not-store-user-data/|archivedate=2011年11月29日}}</ref>
