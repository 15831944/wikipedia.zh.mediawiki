《'''海峽兩岸經濟合作架構協議'''》，是台湾與中国大陆（俗稱「[[海峽兩岸關係|兩岸]]」）的[[雙邊貿易|雙邊]][[自由貿易協定|經濟協議]]，由[[中華民國政府|中華民國政府]]於2009年提出，[[總統|總統]][[馬英九|馬英九]]視為加強[[臺灣經濟|臺灣經濟]]發展的[[政策|政策]]；後於2010年6月29日在[[重慶主城區|重慶]]簽訂第一次協議，後續貨品貿易、服務貿易、投資保障及爭端解決協議協商自此展開。該協議在簽訂前，曾一度稱為'''兩岸經濟協議'''。<ref>[http://www.apdnews.com/news/67960.html 年内完成ECFA后续协商值得期待] {{webarchive|url=https://web.archive.org/web/20140221232438/http://www.apdnews.com/news/67960.html |date=2014-02-21 }}，亚太日报，2014年2月12日</ref>

*英文：(Cross-Straits Economic Cooperation Framework Agreement)，又簡稱'''ECFA<sup>RT</sup>'''。<ref>[http://www.nownews.com/2009/02/27/10844-2414845.htm 尹建議英文名ECFA「A擱發」，馬定調兩岸經合架構協議]，[[NOWnews|NOWnews]]今日新聞網，2010年2月27日，於2010年7月7日查閱。</ref>

== 緣起 ==
2008年初，[[2008年中華民國總統選舉|第十二屆中華民國總統選舉]][[中國國民黨|中國國民黨]]籍候選人[[馬英九|馬英九]]於競選政綱中，即提到當選後將與中國大陆簽訂經貿協定。同年3月，[[中國國民黨|中國國民黨]]擊敗[[民主進步黨|民主進步黨]]再度[[馬英九政府|執政]]。

== 架構 ==
台灣官方強調 ECFA 的架構過程是模仿中國大陆與[[東協自由貿易區|東協自由貿易區]]的架構。可是在這當中卻存在著「系統相容」的問題，差別在於東協與中國大陸加入 WTO 是以開發中國家成員加入，而中華民國是選擇以已開發國家加入 WTO，所以，中華民國已放棄開發中國家的權利，以產品為主，貿易部分就只能引用與中國大陆不同的條款。

在 ECFA 裡強調「逐步到位」，所以雙方須在一定時間使商品貿易為零關稅，並非官方宣稱ECFA 將「僅先定架構及目標」、「像一本書的目錄」，至於「具體內容日後再協商」。

===早期收穫清單分類===
*因為WTO規定經濟合作協議「必須」達到九成雙方貿易產品的免關稅，該條約也達成分階段達成的默契；而其先期協議的產品內容，普稱為「早收清單」。
*在細項上，關稅免稅部份將包含了台灣輸往中國大陆的[[石化|石化]]、[[機械|機械]]、[[紡織品|紡織品]]及[[汽車零配件|汽車零配件]]業等經濟產業。

;部分半官方團體評估
*臺灣半官方組織[[中華經濟研究院|中華經濟研究院]]曾以國際通用[[GTAP|GTAP]]及[[TAIGEM|TAIGEM]]等經濟模型評估該條約。該評估認為：兩岸簽署ECFA對台灣經濟之影響，研究結果顯示簽署後對台灣 GDP、出進口、貿易條件、社會福利均呈現『暫時性正成長』，整體經濟成長率將增加1.65%-1.72%、總就業人數將增加25.7-26.3萬人，對總體經濟有明顯正面效益，全民皆可分享此經濟成長的果實。<ref>[ {{cite web |url=http://www.ecfa.org.tw/ShowNotice.aspx?id=50&catalogue=ECFA |title=存档副本 |accessdate=2010-07-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20100515104938/http://www.ecfa.org.tw/ShowNotice.aspx?id=50&catalogue=ECFA |archivedate=2010-05-15 }} 簽訂兩岸經濟合作架構協議之利益]</ref>
之後，國民黨皆以此評估作為簽訂條約的依據及說帖。而根據此評估，國民黨認為該條約的益處有「邁向全球化」，<ref>{{cite web |url=http://www.cna.com.tw/SearchNews/doDetail.aspx?id=201004250211 |title=存档副本 |accessdate=2010-07-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20140504171936/http://www.cna.com.tw/SearchNews/doDetail.aspx?id=201004250211 |archivedate=2014-05-04 }}</ref>「東亞戰略影響」<ref>[http://docs.google.com/View?id=dgv9n52c_6164pdj9ct 噗友接力聽打之「雙英辯論全文逐字稿」]</ref>。

== 爭議 ==
*[[民主進步黨|民進黨]]文宣稱海峽兩岸經濟合作架構協議A開放大量農產品，而實際上海峽兩岸經濟合作架構協議並無開放農產品，相較於扁政府時代因應開放936項農產品，此一爭議於海峽兩岸經濟合作架構協議辯論時交互詰問。
*前[[立法委員|立法委員]][[林濁水|林濁水]]表示，海峽兩岸經濟合作架構協議有中文版及英文版，但送交世界貿易組織審查的是英文版，英文版與中文版有差且更不利[[臺灣|臺灣]]的可能性很高，而且會是雙方各送一份截然不同的英文版條文。如果海峽兩岸經濟合作架構協議的送審版本接近中文及宣傳版本、可能被世界貿易組織否決；但若英文版大致符合世界貿易組織規範、其要求則是臺灣庶民所不可承受的。<ref>[http://tw.nextmedia.com/applenews/article/art_id/32644528/IssueID/20100708 ＜非典型論述＞馬政府在怕什麼（林濁水）]，[[林濁水|林濁水]]</ref> 對此，經濟部表示「鑑於WTO的官方語言為[[英文|英文]]、[[法文|法文]]及[[西班牙文|西班牙文]]，我國為履行WTO通知義務，協助其他會員瞭解ECFA，將主動提供ECFA約本的英文翻譯版，並會依據慣例，在英文翻譯版的卷首加註『本翻譯版本僅供會員參考，ECFA係以中文簽署生效，任何有關ECFA的理解，應唯一以[[中文|中文]]約本作準』等文字」。

*亦有學者與智庫{{who}}對[[英文|英文]]版之內容表示疑慮或質疑其在中華民國之法律效力。<ref>[http://www.libertytimes.com.tw/2010/new/jul/6/today-fo1.htm 學者：ECFA英文版本 將戳破馬政府謊言]</ref><ref>[http://www.libertytimes.com.tw/2010/new/jul/13/today-fo3.htm 通報WTO，不送立院／ECFA英文版 恐不具法律效力]</ref> 此後，中華民國經濟部表示「鑑於WTO的官方語言為英文、法文及西班牙文，我國為履行世界貿易組織通知義務，協助其他會員瞭解海峽兩岸經濟合作架構協議，將主動提供海峽兩岸經濟合作架構協議約本的英文翻譯版，並會依據慣例，在英文翻譯版的卷首加註『本翻譯版本僅供會員參考，海峽兩岸經濟合作架構協議係以中文簽署生效，任何有關ECFA的理解，應唯一以中文約本作準』等文字」<ref>[http://www.ecfa.org.tw/ShowNews.aspx?id=1071 有關自由時報錯誤報導「ECFA英文版本未經兩岸雙方認可簽署，未經立法院審議，恐不具法律效力」之說明]，2010/07/15</ref> 但由於在ECFA簽訂數月後，文本仍遲遲未送世界貿易組織，美國於2010年11月要求台灣與大陸將ECFA通報送交WTO。<ref>[http://news.rti.org.tw/index_newsContent.aspx?nid=270097 薄瑞光會施顏祥 談美牛、ECFA]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }} 中央社, 2010/11/30</ref> 

*前[[行政院|行政院]]副院長[[吳榮義|吳榮義]]亦於2011年3月批評馬政府「一延再延仍未通報WTO，大搞黑箱作業。」<ref>[http://www.libertytimes.com.tw/2011/new/mar/2/today-fo3.htm ＥＣＦＡ遲不通報ＷＴＯ 被批黑箱作業]</ref>
