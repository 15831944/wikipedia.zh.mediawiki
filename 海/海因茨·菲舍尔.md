'''海因茨·菲舍尔'''（{{lang-de|'''Heinz Fischer'''}}，{{bd|1938年|10月9日|catIdx=Fischer}}）[[奥地利|奥地利]]政治家，前任[[奥地利总统|奥地利总统]]（2004－2016）。

== 学业和早期生活 ==
出生于[[奥地利|奥地利]][[施蒂利亚州|施蒂利亚州]][[格拉茨|格拉茨]]，1956年菲舍尔中学毕业，此后在[[维也纳大学|维也纳大学]]读[[法学|法学]]，1961年获博士学位。

在校期間，開展政治生涯，1978年取得教授资格，1993年擔任[[因斯布鲁克大学|因斯布鲁克大学]][[政治学|政治学]][[教授|教授]]。

== 政治生涯 ==
[[File:Heinz_Fischer.jpg|thumb]]
菲舍尔大学毕业后就为[[奥地利社会民主党|奥地利社会民主党]]工作，任職议会团秘书。1971年选入议会，自此至2004年为止，除其中三年以外，一直是议会议员。1975年菲舍尔成为社民党议会团首领。1977年出任社民党副主席。1983年到1987年任联邦教育、科学和文化部长，此后又任社民党议会团首领。1990年选为國會議長，直到2002年为止。

== 總統 ==
[[File:Austrian_federal_president's_re-election_party,_Vienna_20100425.jpg|thumb]]
2004年1月，菲舍尔宣布参选，3月10日放弃所有社會民主党的党职。他當時的對手為的[[奥地利人民党|人民党]]的[[贝妮塔·费雷罗-瓦尔德纳|贝妮塔·费雷罗-瓦尔德纳]]。4月25日，菲舍尔取得52.39%选票当选。7月6日，原奥地利总统[[托马斯·克莱斯蒂尔|托马斯·克莱斯蒂尔]]突然逝世。7月8日，就职奥地利第八任联邦总统。

2010年，他在第一輪總統選舉投票中擊敗右翼的自由黨勝出，順利連任，任期至2016年。

== 個人生活 ==
菲舍尔是[[不可知論|不可知論]]者<ref>[http://www.bundespraesident.at/newsdetail/artikel/es-kann-auch-das-standesamt-sein-profil/ at www.bundespraesident.at]</ref>。1968年与玛吉特结婚，其两个孩子現已成年。

菲舍尔热愛[[登山|登山]]，长年任奥地利自然爱好者协会主席。
