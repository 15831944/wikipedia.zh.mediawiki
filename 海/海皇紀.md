《'''海皇紀'''》是一部已完結的日本漫畫作品，作者為川原正敏，描写文明崩溃后世界的作品。

== 概要 ==
连载于《[[月刊少年Magazine|月刊少年Magazine]]》（[[讲谈社|讲谈社]]）的1998年3月号至2010年8月号，因为作者的另一部作品《修羅之門》的外传：《[[修罗之刻|陆奥圆明留外传 修罗之刻]]》的连载，这部漫画被迫中止连载。至今共出了45卷单行本（第45卷于2010年9月17日发售）。本作合计发售数量超过1000万册（2007年3月）。

== 世界設定 ==
《海皇紀》中的世界曾经毁灭，人们用全新的技术回到从前。这个世界有着很多国家，他们互相争夺资源，“魔道之业”的势力快速扩张。这个时代没有石炭、石油，当时的交通工具是马和帆船。海的一族支配着海洋。

== 剧情简介 ==
索尔召回了自己的副官，并且解除了近卫队长的职务、囚禁了自己的哥哥，统治着海都。另一边，风离被任命为大海帥，前往布伊的首都基港，跟渥尔韩的国王卡萨尔·夏龙重逢。野心勃勃的卡萨尔吞并了很多国家，不断扩大自己的版图，他的下一个目标是加尔哈珊，亚路·雷欧尼斯·古尔拉在那里担任军师。他的目标是天才军师亚路，他率领军队向加尔哈珊的首都进发。加尔哈珊要怎么抵挡渥尔韩的大军呢？
