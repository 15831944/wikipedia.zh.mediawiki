'''海倫·亞當斯·凯勒（Helen Adams Keller）'''，（{{bd|1880年|6月27日|1968年|6月1日|catIdx=K}}）美國[[作家|作家]]，[[社會運動|社會運動]]家及[[講師|講師]]。

她一歲七個月時因急性[[腦炎|腦炎]]引致[[失明|失明]]及[[失聰|失聰]]，也使她無法說話。

在1887年，藉著她的導師[[安妮·蘇利文|安妮·蘇利文]]對她耐心的教導和關愛，並找到專家使她學會發音，讓她學會流暢的表達，才開始與其他人溝通並接受教育。

海倫·凯勒不但學會閱讀和說話，還以驚人的毅力完成了[[哈佛大學|哈佛大學]]的學業並於1904年畢業，成為有史以來第一個獲得文學學士學位的盲聾人士。

成年後，她繼續廣泛閱讀刻苦學習，成爲掌握了[[英語|英語]]、[[法語|法語]]、[[德語|德語]]、[[拉丁語|拉丁語]]和[[希臘語|希臘語]]的作家和教育家。

她致力於殘疾人事業，四處募捐以改善殘疾人的生活環境和受教育水平。她的事迹使她入選美國《[[時代周刊|時代周刊]]》“人類十大偶像之一”，被授予“[[總統自由勳章|總統自由獎章]]”。

== 生平 ==
[[File:Helen_Keller_with_Anne_Sullivan_in_July_1888.jpg|thumb]]海倫·凱勒於1880年6月27日出生在美國[[阿拉巴馬州|阿拉巴馬州]]的[[塔斯坎比亚_(阿拉巴马州)|塔斯坎比亚]] 。海倫·凱勒原為健康的嬰兒，但在19個月大的時候患了急性脑充血病，失去了聽覺和視覺。長大後運用自創的手語與家庭成員溝通。隨著年歲的增長，簡單的交流不能滿足她，脾氣變得暴躁。6歲時，她的父母在[[家庭醫生|家庭醫生]]的協助下，邀請[[柏金斯啟明學校|柏金斯啟明學校]]的[[安妮·蘇利文|安妮·蘇利文]]老師作為海倫·凱勒的[[啟蒙|啟蒙]]導師。

[[File:Helenkellerage7.jpg|thumb]]
蘇利文老師先瞭解海倫的脾氣躁動的原因：他的父母不忍心看她做錯事（打人、不守規矩、破壞東西等）被懲罰的樣子，於是在她做錯事時都給她糖果吃。她糾正父母的不正確行為，並且與海倫建立互信的關係，再耐心教導[[手語|手語]]，讓她能與別人溝通，再教導她一些生字，她第一個學會的單字是[[水|水]]（water）。其後再教導使用手指點字以及生活禮儀。10歲時她父母聘請了{{tsl|en|Horace Mann School|霍勒斯曼學校}}的[[莎拉·傅樂瓦|莎拉·傅樂瓦]]老師教導其說話，她把中指放在老師鼻子上、食指放在嘴唇上、大拇指放在咽喉上，而海倫亦學會說話。

[[File:HK-gradpic.jpg|thumb]]
1896年，海倫·凱勒進入了位於麻薩諸塞州的{{tsl|en|The Cambridge School of Weston|劍橋女子學校}}。1900年秋季進入哈佛大學拉德克利夫高研院（Radcliffe College）就讀。於1904年以優異成績取得文學學士學位，並成為首位畢業於高等院校的聾盲人。這些年來蘇利文一直留在海倫·凱勒身邊，並將教科書與上課內容寫在海倫·凱勒的手掌上。

[[File:Hellen_Keller_circa_1920.jpg|230px]]
海倫於1924年組成{{tsl|en|Helen Keller International|海倫·凱勒基金會}}，並加入美國盲人基金會，作為其全國和國際的關係顧問。其後她在[[國際獅子會|國際獅子會]]的年會上演說，她要求獅子會成為「協助失明人士戰勝黑暗的武士」。並說：「我為你們開啟機會的窗，我正敲著你的大門。」1946年任{{tsl|en|American Foundation for the Blind|美國全球盲人基金會}}國際關係顧問，共訪問35個國家。她爭取在世界各地興建盲人學校，並常去醫院探望病人，分享她的經歷。

1968年在家中去世，海倫·凱勒享壽87歲，先前經历多次中风。

== 評價 ==
有人曾如此評價她：「海倫·凱勒是人類的驕傲，是我們學習的榜樣，是人類善良的表現，相信她的事跡能成為後世的典範。」

海倫·凱勒雖以一個人道主義者廣為人知並極受推崇，但實際上她也是一個[[社會主義|社會主義]]者。1909年她加入了馬薩諸塞州的[[美國社會黨|美國社會黨]]，而後更成為「世界產業工人聯合會」的成員。她後期的社會主義及[[進步主義|進步主義]]主張，主張[[工人权益|工人权益]]及[[婦女選舉權|婦女選舉權]]，導致對他大加讚美的媒體反過來抨擊她，最典型的即是《布魯克林鷹報》（Brooklyn Eagle），他們評論她的「錯誤來自她在發展上受到明顯的限制」。<ref>[http://www.books.com.tw/web/sys_serialtext/?item=0010663654&page=4 博客來-老師的謊言：美國高中課本不教的歷史]</ref><ref>[http://history.people.com.cn/BIG5/198307/13387195.html 美國歷史教科書中的錯誤：海倫·凱勒其實很激進]</ref>

== 海倫·凱勒紀念日 ==
1971年，[[國際獅子會|國際獅子會]]的國際理事宣佈將每年6月1日定為「海倫·凱勒紀念日」。全球的獅子會在「海倫·凱勒紀念日」舉辦視力相關的服務活動。

== 軼事 ==
海倫·凱勒的故事被改編成舞臺劇《[[奇蹟之人|奇蹟之人]]》，在[[美內鈴惠|美內鈴惠]]的漫畫《玻璃面具》中也有此劇中劇。
