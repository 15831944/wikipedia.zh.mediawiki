'''海口美兰国际机场'''（{{airport codes|HAK|ZJHK}}），位于[[中國|中國]][[海南省|海南省]][[海口市|海口市]][[美兰区|美兰区]]灵山镇，距市区直线距离15公里。1993年12月开始立项建设，于1999年5月25日正式建成通航，占地面积583万平方米，飞行区等级为4E级，可满足波音747-400等大型飞机的起降，目前共开通国内外65个城市，220条航线，2003年进行扩建，扩建后，候机楼面积达到10.2万平米，设计能力达到年旅客吞吐量930万人次，年货邮吞吐量达到13.5万吨，飞行架次5.0万架次。2004年实际旅客接待量7,478,210人次，在中国民用机场中，居第8位，货邮吞吐量达66,582.7吨，居第17位，起降68,282架次，居第10位。

== 設施 ==
客運大樓面積60,200平方米，設有45個辦理登機手續櫃檯、11個保安檢查站。跑道长3,600米，宽60米，平行滑行道长3,600米，宽44米。

== 交通 ==
來往市区的交通如下：
* 铁路：[[海南东环铁路|海南东环铁路]][[美兰站|美兰站]]
* 的士：車程約30分鐘，費用約人民幣50元。
* 机场巴士：往返海口市区民航宾馆外(明珠广场对面)，票价¥20，30分钟一班。
* 公共汽车：'''21'''线往白沙门公园，分段收费，最高票价¥5。'''41'''线往丘海一横路口，分段收费，最高票价为¥6。

== 记录 ==
2012年1月27日，由于海口的大雾天气，导致2000名旅客滞留美兰机场。1月28日，天气好转之后美兰机场恢复通航，当日计划开航451架次，打破该机场通航以来的历史记录。<ref name=451架次打破纪录>{{cite news|author=周慧敏|title=海口美兰机场大雾逐渐消散 拟开航451架次破历史记录|url=http://news.xinhuanet.com/local/2012-01/28/c_111464892.htm|accessdate=2012-01-28|newspaper=新华网|date=2012-01-28}}</ref>
