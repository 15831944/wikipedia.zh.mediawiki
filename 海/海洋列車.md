'''海洋列車'''（{{lang-en|'''Ocean Express'''}}），是一個位於[[香港海洋公園|香港海洋公園]]園內的雙向登山[[纜索鐵路|纜索鐵路]]系統，[[列車|列車]]透過隧道連接海濱樂園（山下——[[黃竹坑|黃竹坑]]）與高峰樂園（山上——[[南朗山|南朗山]]）兩個區域，往返高峰站與海濱站。列車以探索[[潛水艇|潛水艇]]為設計藍本，透過車廂內的[[多媒體|多媒體]]設施，令遊客經歷一次彷彿穿越深海的旅程。全長1,300米，列車可載400人，車程5分鐘，[[軌距|軌距]]1435毫米（標準軌），每小時最高單向載客量為5,000人。海洋列車同時亦令[[海洋公園|海洋公園]]成為全球首個擁有三組[[索道|索道]]系統之[[主題公園|主題公園]]，即兩條登山[[纜車|纜車]]及海洋列車。

== 海洋列車隧道 ==
海洋列車的工程共開鑿出面積達71,300平方米的隧道，整條隧道全長1.3公里，長度相等於532輛車緊貼前後並排，而路軌長度足有29個足球場<ref name="tunnel">{{cite web | url=http://www.oceanpark.com.hk/html/tc/park-experience/attraction-show/ocean-express.html | title=海洋列車 | accessdate=2012-04-24}}</ref>。

== 車站 ==
=== 海濱站 ===
位於海洋公園海濱樂園（山下）的夢幻水都，於[[四川奇珍館|四川奇珍館]]旁，設一個[[港灣式月台|港灣式月台]]，海拔高度+28米。抵站的列車會先開啟一邊車門，等待乘客落車後關門，再開啟另一邊車門讓乘客上車。

=== 高峰站 ===
位於高峰樂園（山上）的熱帶雨林天地，於[[動感天地|動感天地]]之下，設一個[[港灣式月台|港灣式月台]]，海拔高度+134米。抵站的列車也是會先開啟一邊車門，等待乘客落車後關門，再開啟另一邊車門讓乘客上車。

=== 月台閘門 ===
所有車站月台均設有[[月台閘門|月台閘門]]。

== 列車 ==
海洋列車由[[瑞士|瑞士]]的Gangloff Switzerland製造，共有2列，每列各有2卡車廂，共4輛，[[軌距|軌距]]為1435毫米的標準軌，最高時速36公里，每列最多載客400人，設有[[空調|空調]]設備，當列車抵達車站時會進行充電，以供給列車所須的電源及維持空調設備運作。列車的外觀取材自19世紀探險家的深海[[潛水艇|潛水艇]]，車身呈古銅色，車廂內部同樣以潛水艇風格設計，天花中央設有多部緊接的[[液晶顯示器|液晶顯示器]]，以及[[音響|音響]]與燈光設備，在列車行駛時會播放[[多媒體|多媒體]]片段，模擬巨型八爪魚會在窗外施以突襲（經過會車處時），而優美的水母及魚群也會在窗外游過，令遊人仿如置身穿越深海的探秘旅程，上行和下行的播放次序相反<ref>[http://orientaldaily.on.cc/cnt/news/20090831/00176_040.html 潛艇設計十月變猛鬼列車]，東方日報，2009年8月31日</ref>。

== 事故 ==
2010年12月5日中午，一列列車載著逾百乘客上山途中，路軌煞車系統突啟動，導致列車緊急剎停，7名男女在踫撞時受傷，其中一名70歲老翁送院時危殆<ref name="hurt">{{cite web | url=http://gb.cri.cn/27824/2010/12/06/3365s3079638.htm | title=香港海洋公園"海洋列車"發生事故多人碰撞受傷 | date=2010-12-06 | accessdate=2012-04-24}}</ref>。

同年12月16日，停駛了11日的海洋列車重開，車上增設多項安全設施，車速減慢至每秒7米，車程時間增至3分50秒<ref>{{cite web | url=http://hk.apple.nextmedia.com/news/art/20101217/14775908 | title=增煞車保護罩 海洋列車重開  | date=2010-12-17}}</ref>。
