'''宪法级'''（Constitution class），是科幻系列《[[星际迷航|星际迷航]]》（Star Trek）中,隶属于[[星际联邦|星际联邦]][[星际舰队|星际舰队]]的一个[[星际旅行舰级列表|星舰级别]]；是《[[星际迷航：原初系列|星际迷航：原初系列]]》、《[[星际迷航：动画系列|星际迷航：动画系列]]》以及第一、二、三部[[正典|正典]]电影，《[[星际旅行：无限太空|无限太空]]》、《[[星际旅行：可汗怒吼|可汗怒吼]]》、《[[星际旅行：石破天惊|石破天惊]]》中主角舰[[联邦星舰进取号_(NCC-1701)|联邦星舰进取号 (NCC-1701)]]与第五、六部[[正典|正典]][[星际迷航电影列表|电影]]，《[[星际旅行：终极先锋|终极先锋]]》和《[[星际旅行：未来之战|未来之战]]》中主角舰[[联邦星舰进取号_(NCC-1701-A)|联邦星舰进取号 (NCC-1701-A)]]的舰级。此外，第十一部电影《[[星际迷航（电影）|星际迷航]]》中包括主角舰在内的所有该时代联邦星舰，都属于此舰级。此舰级亦出现于几乎所有星际迷航衍生作品中。

== 进取号 ==
{{main|联邦星舰进取号 (NCC-1701)}}
'''联邦星舰进取号 ({{lang|en|USS Enterprise}})'''，船籍編號'''NCC-1701'''，是星际舰队中第1艘以“进取号”（又译“企业号）（Enterprise）由[[詹姆斯·T·柯克|詹姆斯·T·柯克]][[船长|船长]]指挥。是《[[星际迷航：原初系列|星际迷航：原初系列]]》、《[[星际迷航：动画系列|星际迷航：动画系列]]》以及第一、二、三部[[正典|正典]]电影，《[[星际旅行：无限太空|无限太空]]》、《[[星际旅行：可汗怒吼|可汗怒吼]]》、《[[星际旅行：石破天惊|石破天惊]]》中的主角舰。此舰在《[[星际迷航：原初系列|星际迷航：原初系列]]》的试播集中，由克里斯多夫·派克（Christopher Pike）指挥。
在电影《[[星际旅行：石破天惊|石破天惊]]》中，自毁于创世星附近以防落入[[克林贡人|克林贡人]]手中。
== 进取号-A ==
{{main|联邦星舰进取号 (NCC-1701-A)}}
原约克郡号，属改装型。第一次出现于第四部电影《[[星际迷航：抢救未来|抢救未来]]》。是第五、六部[[正典|正典]][[星际迷航电影列表|电影]]，《[[星际旅行：终极先锋|终极先锋]]》和《[[星际旅行：未来之战|未来之战]]》中的主角舰。改名后由[[詹姆斯·T·柯克|詹姆斯·T·柯克]][[船长|船长]]指挥。
