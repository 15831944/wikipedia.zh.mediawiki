{{日本作品|由夢想與色彩組成的|夢と色でできている}}是[[feng|feng]]在2019年2月22日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=https://www.tgsmart.jp/article.aspx?a=24558|title=【本日発売＆メーカーコメント付き！】fengの最新作は残念なヒロインたちとの恋を描くADV！|website=TG Smart|date=2019-02-22|accessdate=2019-03-22|language=ja}}</ref>。

== 概要 ==
起初原定2015年發售。2014年在[[Comic_Market|Comic Market]]販賣各種商品。2015年冬季的秋葉原電氣外祭販售以飛鳥井姬色與黑羽鷗為封面的官方公式書<ref>{{cite web|url=https://www.tgsmart.jp/article.aspx?a=11998|title=karory氏が表紙を担当した"電気外祭り 2015 WINTER 公式ブックレット"が明日（12/5）発売！|website=TG Smart|date=2015-12-04|accessdate=2019-03-22|language=ja}}</ref>。

2018年5月5日公開官方網站<ref>{{cite web|url=https://twitter.com/ueeeeeee/status/992692343001116673|title=feng8th「夢と色でできている」公式サイト公開いたしました！|website=上樣＠feng推特|date=2018-05-05|accessdate=2019-03-22|language=ja}}</ref>，原定2019年1月25日發售，後來延期至2月22日<ref>{{cite web|url=https://twitter.com/ueeeeeee/status/1076137493617987584|title=お詫びとお知らせ…|website=上樣＠feng推特|date=2018-12-21|accessdate=2019-03-22|language=ja}}</ref>。

== 故事簡介 ==
乙原藍小時候和兒時玩伴組成「地球防衛軍」，但在飛鳥井姬色轉學之後，這個團體也隨之解散。而在藍高中二年級的冬天，姬色卻轉進了他所就讀的神門學園，並在另一位青梅竹馬黑羽鷗協助下成立了社團「神門防衛社」，成立了這個社團之後，姬色決定要找回地球防衛軍成員們的羈絆<ref>{{cite web|url=http://moedigi.com/game/2019/02/feng-yume-master-up/|title=feng『夢と色でできている』がマスターアップ。恋愛ができない残念な仲間たちの青春ADV|website=moedigi|date=2019-02-12|accessdate=2019-03-22|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|乙原藍|乙原 藍（おんばら あい）}}
: 作品男主角，神門學園二年級生。小時候在姬色轉學後主動解散地球防衛軍。
; {{nihongo|飛鳥井姬色|飛鳥井 姫色（あすかい ひいろ）|聲：{{lang|ja|桃山いおん}}}}
: 神門學園二年級生，藍的青梅竹馬。小時候前往海外留學，在高中二年級時的冬天轉到神門學園就讀。聽聞藍解散了地球防衛軍感到失望，後來在鷗的幫忙下成立社團「神門防衛社」並擔任社長。
; {{nihongo|黑羽鷗|黒羽 かもめ（くろばね かもめ）|聲：{{lang|ja|小波すず}}}}
: 神門學園二年級生，藍的青梅竹馬。有著毒舌的個性，但同時也擅長照顧人。喜歡[[角色扮演|角色扮演]]。
; {{nihongo|乙原戀|乙原 恋（おんばら れん）|聲：{{lang|ja|藤咲ウサ}}}}
: 神門學園二年級生，藍的妹妹。以前個性膽小，現今在學校是如同偶像的存在。由於母親前往海外工作而負責包辦家事。
; {{nihongo|刻乃雲|刻乃 雲（ときの くも）|聲：{{lang|ja|柚原みう}}}}
: 神門學園二年級生，藍的青梅竹馬。有著神祕氣質且沉默寡言的少女。家中經營神社。
; {{nihongo|七夕紫織|七夕 紫織（たなばた しおり）|聲：{{lang|ja|白月かなめ}}}}
: 神門學園三年級生，藍的青梅竹馬。從小在家裡的要求下學習格鬥技，平時個性溫柔優雅，但生氣時非常可怕。
