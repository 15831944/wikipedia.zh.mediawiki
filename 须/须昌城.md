'''须昌城'''，又称'''须城'''，是[[山东省|山东省]][[东平县|东平县]]西部一处古城邑。该城始建于[[春秋时期|春秋时期]]，曾为古[[须句国|须句国]][[都城|都城]]，是[[东平国|东平国]]、[[东平郡|东平郡]]、[[郓州|郓州]]治所。[[1000年|公元一千年]]（北宋咸平三年），因[[黄河|黄河]][[决口|决口]]而被淹没。至今其遗址仍在[[东平湖|东平湖]]水面下，为[[东平县县级文物保护单位|东平县县级文物保护单位]]<ref>东平政协文史资料委员会. 《东平名胜古迹》，56-57页-须昌故城</ref><ref>王龙，姜广智等. 《可爱的东平》，15页</ref>。

== 地理 ==
{{Quote box|width=15em|salign=right|quote=永望齊魯郊，白雲何悠悠。<br />傍沿鉅野澤，大水縱橫流。|source=——[[高適|高適]] 《[[:s:東平路中遇大水|東平路中遇大水]]》}}
须昌城遗址位于东平县西部的[[东平湖|东平湖]]中，金山东南，[[土山岛|土山岛]]、[[亭子村|亭子村]]旧址以东，埠子村旧址以西<ref>东平政协文史资料委员会. 《东平名胜古迹》，258页-东平重点名胜古迹分布图</ref>。此处湖面由[[老湖镇|老湖镇]]管辖。

须昌城尚未被淹没时，其位于[[济水|济水]]以东，为济水、[[大汶河|汶水]]的汇流处，航运便利。但其地地势十分低洼，多次遭遇洪水侵袭，如公元824年、931年、941年分别爆发大洪水，造成严重损失<ref name=":1">{{Cite web|url=http://www.sddp.net/dph/dphwy.asp?dph=126|title=被淹没的须昌古城 - 古州史话 - 东平湖文艺|accessdate=2018-07-07|author=展衍振|date=2006-8-30|work=www.sddp.net|publisher=}}</ref>。[[高适|高适]]的《東平路中遇大水》即描述了唐代須昌城遭到洪水侵袭的情况。

==历史==

根据历史记载，西周初，[[太昊|太昊]][[伏羲|伏羲氏]]的裔孙以子爵的身份受封，建立[[須句國|须句国]]（亦称须朐国），但亦有资料显示，作为“[[风姓|风姓四国]]”之一，須句國早在[[周朝|周朝]]之前就已經建立<ref name="风物志">东平县史志办公室，[http://lib.sdsqw.cn/bin/mse.exe?seachword=%20&K=c94&A=10&rec=28&run=13 《东平风物志》，第二编-第二章-第二节 古城址]</ref>。当时须句国最早的国都具体位置存在争议，有[[朐城|朐城]]、须昌城两种说法，但一般认为，须昌城曾作为须句国的首都。

公元前639年，[[邾国|邾国]]伐须句国，须句国战败灭亡。须句国民众四散，有的逃往邻近而同为风姓的[[宿国|宿国]]、[[任国|任国]]获得收留，而须句国君则逃往鲁国寻求帮助，因为[[鲁僖公|鲁僖公]]的母亲[[成风|成风]]是须句人<ref>《左传》:“邾人灭须句，须句子来奔，因成风也。成风言之于公曰：‘崇明祀，保小寡，周礼也；蛮夷猾夏，周祸也。若封须句，是崇皞、济而修祀，纾祸也’”</ref>。次年，[[魯僖公|鲁僖公]]出兵讨伐邾国，夺取须句并护送须句国君回国。当年八月八日，邾国再次攻打鲁国，鲁国轻视邾国，“不设备而御之”，结果大败，须句国再次灭亡<ref>《左传》：“邾人以须句故出师。公卑邾，不设备而御之。臧文仲曰：「国无小，不可易也。无备，虽众，不可恃也。《诗》曰：『战战兢兢，如临深渊，如履薄冰。』又曰：『敬之敬之！天惟显思，命不易哉！』先王之明德，犹无不难也，无不惧也，况我小国乎！君其无谓邾小，蜂虿有毒，而况国乎！」弗听。八月丁未，公及邾师战于升陉，我师败绩。邾人获公胄，县诸鱼门。”</ref>。公元前620年，[[魯文公|鲁文公]]报复邾国，三月十一日攻占须句。<ref>《春秋·文公七年》：“春，公伐邾，三月甲戌，取须句。”</ref>鲁文公封一个儿子于此，设置[[须句邑|须句邑]]<ref>{{Cite web|url=http://epaper.qlwb.com.cn/qlwb/content/20170212/ArticelA10002FM.htm|title=须句国:湖光山色中的孤独守望 - 齐鲁晚报数字报刊|accessdate=2018-07-07|work=epaper.qlwb.com.cn}}</ref>。

秦代，设置须昌县，属[[薛郡|薛郡]]<ref name="atlas">东平县地名委员会办公室. 《东平县地图册》</ref>{{rp|26}}。西汉须昌县属[[東郡|东郡]]{{r|atlas|page=27}}，[[东汉|东汉]]、曹魏属东平国{{r|atlas|page=28,29}}。西晋时期，东平国国治迁往须昌、随后改为东平郡，北魏仍属兖州东平郡。北齐时期须昌县县治迁移到宿城，东平郡废除。隋朝开皇十六年，须昌县迁回{{r|atlas|page=31}}。仁寿元年，在须昌城西门外建[[清水石桥|清水石桥]]。

贞观八年（公元634年），郓州治所迁往须昌{{r|atlas|page=32}}，自此须昌城成为东平政治经济文化中心，开启了一段繁荣时期，曾一度成为天平节度使，郓州，东平郡三级行政区划治所。公元753年，东平郡太守[[李祗|李祗]]讨伐[[安禄山|安禄山]]。777年，[[平盧節度使|平卢节度使]][[李正己|李正己]]自安史叛军手中收复多个州，自此将平卢节度使迁往郓州（即须昌）。781年，平卢节度使[[李纳|李纳]]叛唐，自立为齐王，819年叛乱被平<ref name=":1" />。五代时期，各路割据政权激烈争夺须昌。[[后唐|后唐]][[同光|同光]]元年（公元923年），须昌城因[[名讳|讳]]而改名须城。[[北宋|北宋]]时期，先后属[[京东路|京东路]]郓州、[[京東西路|京东西路]]郓州，为郓州治所{{r|风物志|atlas|page=33}}。

北宋咸平三年五月（公元1000年），黄河在[[王陵埽|王陵埽]]决口<ref>{{Cite web|url=http://www.my0538.com/2018/0305/410488.shtml|title=东平湖底"藏"古桥 已沉睡千余年_民生要闻_民生泰安_社会_中华泰山网|accessdate=2018-07-07|work=www.my0538.com|language=zh-CN}}</ref>，汹涌的洪水侵入须昌城，须昌城遂废，沦没陂泽之中<ref name="where">{{Cite web|url=http://history.sohu.com/a/198994103_199199|title=东平须昌故城源于何方？|accessdate=2018-07-07|date=2017-10-19|work=history.sohu.com}}</ref>。郓州[[知州|知州]][[姚鉉|姚鉉]]随后在汶阳乡[[王陵山|王陵山]]前五里处地势较高的地方建立新城<ref>于庆明、郑新道等，[http://lib.sdsqw.cn/bin/mse.exe?seachword=%20&K=f9&A=1&rec=3&run=13 《泰安五千年大事记》，宋代至明代]</ref><ref>姚铉，[[:s:迁移郓州谢表|《迁移郓州谢表》]]</ref><ref>《宋史·列传·卷二百》：“咸平三年，河决郓州王陵埽，东南注钜野，入淮、泗，城中积水坏庐舍，以铉知州事，徙州于汶阳乡之高原，委以营度，许便宜从事。”</ref>，即为今东平县州城街道所在地的[[州城_(城邑)|州城]]<ref name=":0" />。

须昌城被淹没以后，其邻近的埠子坡村因因地势较高，幸免水淹，成为须昌城的一处遗存遗址<ref name="find" />。根据[[蒋作锦|蒋作锦]]的《[[东原考古录|东原考古录]]》考证，埠子村西的东岳庙为须昌城东关，埠子村西南石刻“南门”二字为须昌城南关。埠子坡村作为村庄一直存在到新中国建立以后。二十世纪五十年代初，为了黄河防汛安全，政府决定建立[[东平湖|东平湖]]水库，埠子泊村自此也被淹没，须昌城遗址自此完全沉没于水中{{r|where}}。

== 建筑与考古 ==
{{主条目|清水石桥|溯源亭}}
由于资料缺乏，目前须昌城的城市构造尚不清楚。须昌城附近还有两个值得注意的文物，分别为[[清水石桥|清水石桥]]和[[溯源亭|溯源亭]]。清水石桥在须昌城西侧<ref>{{Cite web|url=http://news.163.com/10/0512/14/66G93UF2000146BD.html|title=堪与赵州桥相媲美 探寻山东东平湖底的隋代石 桥_网易新闻|accessdate=2018-07-07|last=网易|work=news.163.com}}</ref><ref>{{Cite web|url=http://www.sddpw.net/news/show-34377.html|title=我国内陆水下考古第一站“潜水”东平湖 须昌古城和清水石桥将现真容 - 中国东平网|accessdate=2018-07-07|work=www.sddpw.net|deadurl=no|archiveurl=https://web.archive.org/web/20171216155103/http://www.sddpw.net/news/show-34377.html|archivedate=2017-12-16|df=}}</ref>，“石作华巧、与赵州石桥相埒”，比赵州桥要早五到十年，也比[[赵州桥|赵州桥]]长60米，与须昌城同为县级文物保护单位<ref>东平政协文史资料委员会. 《东平名胜古迹》，254-257页-东平县文物保护单位一览表</ref>。洄源亭（或溯源亭）在须昌城西南12里，亭子村旧址处<ref>{{Cite web|url=http://www.qlwb.com.cn/2018/0306/1214218.shtml|title=风华齐鲁-平静的东平湖下，深埋着一座千年古城－山东新闻－齐鲁晚报网|accessdate=2018-07-07|author=|date=|last=白丽|work=www.qlwb.com.cn|publisher=}}</ref>，为唐代东平太守[[苏源明|苏源明]]所作，如今溯源亭遗址附近仍有露出水面的断壁残垣以及巨石，当地渔民时常在此休息<ref>东平政协文史资料委员会. 《东平名胜古迹》，157-158页-清水石桥</ref>。须昌故城附近还可能有类似于[[宿城汉墓群|宿城汉墓群]]的大型[[墓葬|墓葬群]]。除大型文物古迹外，当地渔民亦在须昌故城附近的东平湖水域中打捞到过[[铜钱|铜钱]]等文物{{r|find}}。

2015年春，中国[[国家文物局|国家文物局]]水下遗产保护中心与山东省有关部门对东平湖进行水下科考工作，科考对象包括须昌故城及其附属古迹如清水石桥等，作为中国内陆水下考古首站<ref>{{Cite web|url=http://www.sssc.cn/a/20150804/1438675656150513.shtml|title=内陆水下考古首站——东平湖水下考古项目启动 - 考古研究 - 资讯 - 盛世收藏网|accessdate=2018-07-07|work=www.sssc.cn|language=zh-cn}}</ref>。尽管东平湖很浅，但其淤泥层很深，给科考工作带来很大困难<ref name="find">{{Cite web|url=http://shizheng.xilu.com/20151216/1000150004591333.html|title=寻找梁山泊下的古城|accessdate=2018-07-07|author=|date=|work=shizheng.xilu.com|publisher=}}</ref>。

== 争议 ==
须昌城最大的争议在于其与须句国的关系。《[[汉书|汉书]]·地理志》认为须昌即古须句国，而须昌城与朐城无关<ref>《汉书·地理志·须昌》:“须昌，故须句国，风姓”</ref>。据《[[水经注|水经注]]》载，[[西晋|西晋]]地图学家[[京相璠|京相璠]]称须句是由须朐迁出新置的城国<ref>《水经注》:“济水又北过须昌县西。京相璠曰：须朐一国二名，盖迁都。须昌，朐是其本，秦以为县”</ref>。晚清地理学家杨守敬认为朐城为“未迁之朐”，须朐为“已迁之朐”。而现代学者李继生也认为，须句就在今东平县西北老湖镇须昌城，须朐与须句无关<ref name=":0">{{Cite web|url=http://www.sdhh.gov.cn/qlhh/hhzs/10/164013.shtml|title=被东平湖淹没的城市_黄河之声_山东黄河河务局|accessdate=2018-05-15|last=山东黄河网|work=www.sdhh.gov.cn|deadurl=no|archiveurl=https://web.archive.org/web/20121113124140/http://www.sdhh.gov.cn/qlhh/hhzs/10/164013.shtml|archivedate=2012-11-13|df=}}</ref>。

== 逸闻 ==
由于“宿”与“须”在[[东平方言|东平方言]]中发音相同（[[國際音標|IPA]]：[{{Ipa|ɕy˩˦}}]），因此人们时常将今[[东平街道|东平街道]]西部的古城邑宿城与须昌城混淆。自清代开始，[[宿城故城|宿城]]就被讹记“须城”。直到1988年，须城区才改回原名[[宿城镇|宿城区]]<ref>东平政协文史资料委员会. 《东平名胜古迹》，53-55页-宿城小考</ref><ref>{{Cite web|url=http://www.dpdmw.cn/newsinfo.asp?ArticleID=463|title=东平地名趣闻考——宿城|accessdate=2018-07-07|author=东平地名网|date=2010-08-16|work=www.dpdmw.cn|publisher=|deadurl=no|archiveurl=https://web.archive.org/web/20150815121140/http://dpdmw.cn/newsinfo.asp?ArticleID=463|archivedate=2015-08-15|df=}}</ref>。今东平街道西有宿昌路，但东平历史上并无“宿昌”之名，只有“宿城”和“须昌”。
