《'''奇諾之旅 -the Beautiful World-'''》（{{lang-ja|キノの旅 -the Beautiful World-}}）是[[日本|日本]][[作家|作家]][[時雨澤惠一|時雨澤惠一]]撰寫的[[輕小說|輕小說]]系列，插畫由[[黑星紅白|黑星紅白]]繪製。繁體中文版由台灣角川代理，譯者莊湘萍（第1-19卷）、[[譯者K.K.|K.K.]]（第20-21卷）。

曾於2003年與2017年二度[[電視動畫|電視動畫]]化，2003年版已於4月至7月播映，共13集。2017年版已於10月至12月播映，共12集。

兩部漫畫版分別於3月在《少年Magazine Edge》2017年4月號開始連載，由{{Tsl|ja|シオミヤイルカ|シオミヤイルカ}}繪製；以及於5月在《[[月刊Comic電擊大王|月刊Comic電擊大王]]》2017年7月號開始連載，由鄉繪製。

== 故事簡介 ==
[[旅行者|旅行者]]奇諾（Kino）騎著她那會說話的[[摩托車|摩托車]]漢密斯（Hermes）到各個[[國家|國家]][[旅行|旅行]]，在每個國家奇諾都只停留三天，在她體會了該國家人民的文化、風俗之後，三天內她便離開並前往另一個國家……

== 主要人物 ==
※擔當聲優配音順序為：電視動畫第1作／{{tsl|ja|多数決ドラマ|多數決廣播劇}}及動畫第2作／廣播劇CD及[[電擊文庫_FIGHTING_CLIMAX|電擊文庫 FIGHTING CLIMAX]]，無記載的代表此版本無配音，以外的聲優另外註記。

; {{nihongo|奇諾|キノ|聲：{{link-ja|前田愛 (藝人)|前田愛 (女優)|前田愛}}／[[悠木碧|悠木碧]]／[[久川綾|久川綾]]}}
: 本作[[主人公|主人公]]，外貌特徵是短髮、穿著風衣的中性打扮，有著大眼睛、端正​​精悍面孔的少女。旅人，與漢密斯一同旅行，執行著一個國家只停留三天的規則。本名不詳（應是某種花的名稱）。
: 具有「說服者（persuader）」{{noteTag|1={{zh-cn}}[[:天闻角川#.E5.85.A7.E5.AE.B9.E6.94.B9.E5.8A.A8|2012年3月开始出版的《奇诺之旅》系列将枪械名称“说服者（persuader）”改为“帕斯艾德”。]]}}{{noteTag|1=作者偶爾會不小心把persuader打成Pathada，Pathada是沒有意義的。}}槍械段位，對於旅行中所遇到的事物多採中立不關心的態度，武器主要是左輪「[[柯爾特1851海軍型轉輪手槍|卡農]]」、單發自動式手槍——「[[柯爾特護林者手槍|森之人]]」及附消音器的可拆卸自動[[狙擊步槍|狙擊步槍]]「[[九九式步槍|長笛]]」以及一把基於[[莫斯伯格500泵動式霰彈槍|莫斯伯格500泵動式霰彈槍]]（Mossberg 500 Persuader）的泵動式霰彈槍、不少於五把刀，其中一把刀能發射子彈。此外，廚藝似乎不佳。
: 原大人之國的居民，家裡經營旅館，由於不想變成「大人」（該國規定12歲以上為大人，必須做手術把腦中「小孩」的成份取出），在旅行者「奇諾」替她擋下刺向她的刀後，騎著「漢密斯」逃離大人之國，之後成為並繼承旅人的名字「奇諾」開始旅行。

; {{nihongo|漢密斯|エルメス|赫米斯{{noteTag|1=小說版譯名為「漢密斯」；[[ANIMAX|ANIMAX]]版譯名為「赫米斯」，名字與希臘神話無關。}}|聲：{{link-ja|相瀨龍史|相ヶ瀬龍史}}／[[齊藤壯馬|齊藤壯馬]]／[[野田順子|野田順子]]}}
: 奇諾的搭檔，為一台會說話的摩托車（Motoride），是由當時經過大人之國的旅行者「奇諾」修理而成。會與周遭對話或自言自語，常吐槽「奇諾」，但必要時會激勵「奇諾」。
: 原型為Brough Superior SS100。動畫版中的[[引擎|引擎]]聲是採用真正的引擎聲。

; {{nihongo|西茲|シズ|聲：{{link-ja|入江崇史}}／[[梅原裕一郎|梅原裕一郎]]}}
: 穿著綠色毛衣的青年，具有積極助人的個性，武器是一把[[武士刀|武士刀]]。開著四輪越野車，帶著會說話的狗「陸」還有「蒂」一起旅行，為了尋找可以移民的地方而旅行的。
: 前某國王子，因為父親謀殺篡位當時身為國王的祖父，而連同生死不明的兄弟姊妹一起被父親放逐，七年後回到因為暴政而荒廢的故鄉打算暗殺國王，卻被「奇諾」以流彈形式所發射的特製彈頭完成暗殺目標，之後離開該國並繼續流浪的旅行。
: 在單行本第十七集「過境的國家」中曾向該國申請移民成功，但是卻在觀察期內因為複雜的狀況下超速違規，導致最後被該國遣送出境。

; {{nihongo|陸|りく|聲：[[大塚芳忠|大塚芳忠]]／{{link-ja|松田健一郎}}}}
: 西茲的搭檔，自稱是西茲忠實的僕人，為一隻說話的大白[[狗|狗]]。有著一副看起來開心笑臉的可愛，雖然他並不是真的很開心。跟著西茲一起旅行，一開始稱呼西茲為「西茲王子」之後改為「西茲少爺」。遇到蒂之後常與蒂一起行動，並不是很喜歡被蒂抱著。
: 原形應是作者養的同名[[薩摩耶犬|薩摩耶犬]]。

; {{nihongo|蒂／蒂法娜|ティー／ティファナ|聲：[[能登麻美子|能登麻美子]]／[[佐倉綾音|佐倉綾音]]}}
: 西茲的搭檔。是西茲和陸在船之國認識的一位白髮綠瞳少女，非常纖瘦，她平時不愛說話，不太表現感情，說話時常用簡短的句子。被身為旅行者的父母拋棄在船之國，之後在很複雜的狀況下離開國家。現跟西茲他們一起行動，視陸為一個好夥伴，似乎很喜歡抱著陸。
: 旅行途中西茲經常教導她一些事情跟道理，雖然陸認為她從來沒聽進去，但偶爾會說出西茲曾教導過的事情跟道理，甚至會說一些讓西茲無法回應的話或是一些西茲跟陸沒想到的論點。
: 主要武器是[[Mk_2手榴彈|Mk 2手榴彈]]和西茲某次任務的報酬所拿到的[[榴彈發射器|榴彈發射器]]以及飛刀。

; {{nihongo|師父|師匠（ししょう）|聲：[[渡邊明乃|渡邊明乃]]（過去）、{{link-ja|翠準子}}（現今）／[[Lynn|Lynn]]（過去）、{{link-ja|澤田敏子|沢田敏子}}（現今）}}
: 奇諾的師父，擁有長髮的妙齡女性。使用說服者的能手，教導奇諾說服者跟旅行知識的女人，是曾在各國引起大騷動的問題人物。曾警告奇諾為自身安全不要透露任何跟她有關的事，只要有人問起就回答他「不認識」。

; {{nihongo|夥伴|相棒（あいぼう）|聲：[[千葉進步|千葉進步]]／[[興津和幸|興津和幸]]}}
: 師父的搭檔，身高略矮的俊俏男子，在「有首領的國度」與師父結為搭檔。雖然看來沒用，但有時會有出乎意料的舉動，擅長槍枝保養及維修技術。

; {{nihongo|芙特|フォト|聲：無／[[水瀨祈|水瀨祈]]}}
: 「在雲端前面─Eye-opener─」登場的少女。17歳。本名現已經不再使用，「蘇」曾經替她取過別的名字，但因為過於難以記憶以及工作的關係該國國民常以「芙特」稱呼，而「芙特」則源自於照片的英文發音。
: 來自一個奉行「無條件相信別人」教義的宗教體制國家，是個孤兒。因為該國無法付清行商的貨品費，因此被該國的教祖兼元首當成抵押品交給商人，被商人當成奴隸對待，但因為商人誤食毒草而死得以自由，且聽見被堆放在商品裡的折疊式機車「蘇」的呼喚並與蘇一起帶著商人的部分商品及卡車離開。
: 與蘇進入某個國家後，受到幫助並順利販售了大部份的商品後，僅留下在該國非常稀有的[[單眼相機|單眼相機]]。商品售出並扣除稅金後的所得，據蘇所說芙特的財產在該國中是相當富有的，但是因為過去的關係，習慣節儉及工作。在蘇的建議下，以興趣及工作為由在該國擔任攝影師，常與蘇一起出去拍照。

; {{nihongo|蘇|ソウ|聲：無／[[緒方惠美|緒方惠美]]}}
: 芙特的搭檔，為一台折疊式機車。原本是買下芙特的商人的貨品，與芙特相遇並成為夥伴，經常照顧過於天真的芙特。初次與陸相遇時稱呼其為「稀奇的東西」。
: 原型是[[HONDA|HONDA]]生產的迷你機車「Honda Motocompo」。

; {{nihongo|旅人奇諾|旅人キノ|聲：[[井上和彥|井上和彥]]／[[小野大輔|小野大輔]]}}
: 旅人，通稱「前奇諾」。在「大人之國」旅行時向一名少女說明了身為旅行者的思想後，但少女因為不想成為大人而不願進行改做手術，該國家要殺掉當時11歲的少女時，捨身擋下刺向少女的刀並當場死去，自此少女便以「奇諾」之名旅行。
