'''契瓦瓦州'''（{{lang-es|Chihuahua}}）位於[[墨西哥|墨西哥]]西北內陸，北靠[[美國|美國]][[新墨西哥州|新墨西哥州]]和[[德克薩斯州|德克薩斯州]]，是該國面積最大的[[墨西哥行政區劃|州]]，佔全國面的八分之一。

== 地理 ==
奇瓦瓦除了有廣大的[[沙漠|沙漠]]外，其[[森林|森林]]面積比墨西哥其它州都要多。在山坡地上有短草草原，是墨西哥主要的[[農業|農業]]區。

== 歷史 ==
州名源於州的首府。「契瓦瓦」這個名字的來源不詳，但在[[西班牙|西班牙]]人來到前已經存在，可能出自[[納瓦特爾語|納瓦特爾語]]的''Xicuahua''，意思是「乾燥、多沙的」。

该州在[[墨西哥革命|墨西哥革命]]時期發揮了重要作用。

== 經濟 ==
主要產品有[[蘋果|蘋果]]、[[堅果|堅果]]、[[木材|木材]]、[[牛|牛隻]]及[[奶製品|奶製品]]、[[綿羊|綿羊]]，[[金|金]]、[[銀|銀]]、[[鉛|鉛]]、[[鋅|鋅]]等各種金屬，[[水泥|水泥]]及[[陶瓷材料|陶瓷材料]]。工業方面以出口美國為主。[http://en.wikipedia.org/wiki/Mata_Ortiz Mata Ortiz]的新式陶瓷飲譽國際。[[第三產業|第三產業]]發達，以[[旅遊業|旅遊業]]、[[銀行|銀行業]]、高科技產業為主。然而由於鄰近美國，亦成為了[[毒販|毒販]]活躍的地方，大大影響了當地的治安及旅遊業。

== 民族 ==
[[麥士蒂索人|麥士蒂索人]]為主，[[白人|白人]]的比例全墨西哥第一（35%）。其它族群有居於山區的[[印地安人|印地安人]]，[[門諾派|門諾派]]（有[[德國|德國]]-[[荷蘭|荷蘭]]血統）、[[摩門教|摩門教]]（有[[英國|英]]美血統）徒等。
