'''奇葩朵朵'''，2018年中国大陆校园爱情喜剧电影。由[[张若昀|张若昀]]、[[马思纯|马思纯]]、[[李现|李现]]联袂主演，[[刘敏涛|刘敏涛]]、[[姜超|姜超]]、[[李霄云|李霄云]]共同出演，导演[[李欣|李欣]]执导<ref>{{Cite web |url=https://www.sohu.com/a/225422193_114733 |title=《奇葩朵朵》定档4.4 张若昀马思纯组CP火花对撞 |accessdate=2018-03-15 |author= |date=2018-03-13 |work=|publisher=搜狐|language=Zh-hans }}</ref>。影片于2015年4月1日在天津杀青<ref>{{Cite web |url=http://ent.qq.com/a/20150402/027031.htm |title=张若昀《奇葩朵朵》杀青 神似“小胖”恶搞不停 |accessdate=2018-03-15 |author= |date=2015-04-02 |work=chengchengjin|publisher=腾讯娱乐|language=Zh-hans }}</ref>，时隔三年于2018年4月4日上映。
