'''奇皮奥纳'''（{{lang-es|Chipiona}}），是[[西班牙|西班牙]][[安达卢西亚自治区|安达卢西亚自治区]][[加的斯省|加的斯省]]的一个[[市镇_(西班牙)|市镇]]。总面积33平方公里，总人口19104人（2017年）<ref name=po>{{Cite web |url=http://www.ine.es/jaxiT3/Tabla.htm?t=2864&L=1# |title=Cádiz: Population by municipality and sex. |accessdate=2018-10-12}}</ref>，人口密度580人/平方公里。该镇位于[[大西洋|大西洋]]沿岸，平均海拔高度6米。

奇皮奥纳的主要景观是[[奇皮奥纳灯塔|奇皮奥纳灯塔]]。它的高度为63米，是世界上排名第17高的古代[[灯塔|灯塔]]，也是西班牙最高的灯塔。<ref>{{Cite web |url=https://web.archive.org/web/20181012080643/http://www.ibiblio.org/lighthouse/tallest.htm |title=The Tallest Lighthouses |language=en |accessdate=2018-10-12 |author=Rowlett, Russ |publisher=The Lighthouse Directory}}</ref>其存在可以追溯到[[罗马时代|罗马时代]]。<ref>[[斯特拉波|Στράβων]], III, 1, 9</ref>如今看到的是经1867年重修后的灯塔。<ref>{{Cite web |url=https://web.archive.org/web/20181012080134/https://www.diariodejerez.es/provincia/ano-Faro-Chipiona_0_656934690.html |title=El año del Faro de Chipiona |accessdate=2018-10-12|language=es}}</ref>
