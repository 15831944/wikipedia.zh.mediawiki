《'''迷踪：第9鹰团'''》（{{lang-en|'''The Eagle'''}}）2011年上映于美国和英国的史诗片，主演是[[查尼·泰坦|查尼·泰坦]]、[[杰米·贝尔|杰米·贝尔]]、[[唐纳德·萨瑟兰|唐纳德·萨瑟兰]]、[[马克·史壮|马克·史壮]]，导演是[[凯文·麦克唐纳|凯文·麦克唐纳]]。

== 剧情 ==
公元前46年，罗马[[凯撒大帝|凯撒大帝]]在[[西班牙|西班牙]]征伐时建立“[[第九“西班牙”军团|第九军团]]”，公元前41年，[[屋大维|屋大维]]对第九军团进行改编，其独特的排兵布阵技巧让它名震[[欧洲大陆|欧洲大陆]]。<ref name=" fish" />

公元2世纪，[[罗马帝国|罗马帝国]]入侵[[英格兰|英格兰]]，第九军团神秘失踪。

公元140年，第九军团消失在英格兰北部[[加勒多尼亚|加勒多尼亚]]，马库斯·弗拉菲·乌亚居拉是一个年轻的[[百夫长|百夫长]]，他到达英格兰，被任命为一个要塞司令官。马库斯的父亲曾经在罗马军团服役，后来在英格兰时运不济而死亡。罗马军团消失在英格兰时，权杖跟着消失。马库斯来到英格兰叔叔家养伤的时候，听说权杖出现在英格兰，于是他便开始了寻找权杖和第九军团之路。<ref>{{cite web |url=http://ent.ifeng.com/movie/news/mainland/detail_2011_04/21/5889620_0.shtml |title=国内即将上映《迷踪：第九鹰团》中文版预告 |accessdate=2011年4月21日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

== 花絮 ==
该片改编自罗斯玛丽·苏特克里夫于1954年出版的小说罗斯玛丽·苏特克里夫《第9军团》，1977年曾被英国[[英国广播公司|英国广播公司]]（BBC）改编成电视剧。

该片源自真实的历史，公元120年罗马帝国第九军团神秘失踪。<ref>{{cite web |url=http://ent.163.com/11/0505/11/739O5CQ1000300B1.html |title=角斗士二代《迷踪：第九鹰团》 五月生猛上映 |accessdate= 2011-05-05|publisher=网易 |work= |language =zh-cn}}</ref>

该片展示了[[不列颠群岛|不列颠群岛]]上冷兵器战争的壮阔豪迈。<ref>{{cite web |url=http://news.mtime.com/2011/05/05/1458436.html |title= 角斗士征讨英伦 《第九鹰团》激发男性荷尔蒙|accessdate= 2011-05-05 |publisher=时光网 |work= |language =zh-cn}}</ref>

该片没有女人和爱情剧情戏份。<ref>{{cite web|url=http://www.hn.xinhuanet.com/ent/2011-04/29/content_22649272.htm|title=《迷踪：第九鹰团》今上映 没女人的兄弟情电影|accessdate=2011年4月29日|publisher=新华网|work=|language=zh-cn}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

导演麦克唐纳谈到该片进入中国大陆市场的时候说：“中国对于我来说是一个神秘而遥远的国家，拥有相当悠久的历史和文化传统。在那里也有很多没有解开的历史谜题，例如关于[[秦始皇|秦始皇]]的很多传说。我相信那里的观众对《迷踪：第九鹰团》这部电影会感兴趣，更何况除了历史之外，我们还有很好的演员以及出色的动作场面。”<ref name=" fish" >{{cite web |url= http://ent.sina.com.cn/m/f/2011-04-26/15233292598.shtml|title= 《迷踪：第九鹰团》抢占五一档 与中国有渊源|accessdate= 2011年4月26日 |publisher=新浪网 |archiveurl= http://ent.sina.com.cn/m/f/2011-04-26/15233292598.shtml|archivedate=2012年11月22日 |work= |language =zh-cn}}</ref>
