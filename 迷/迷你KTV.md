'''迷你KTV'''（又名'''电话亭KTV'''）是一种形状和大小类似于[[电话亭|电话亭]]且里面配套有[[卡拉OK|卡拉OK]]机器的隔间，流行于中国、[[韩国|韩国]]、[[越南|越南]]等[[东亚|东亚]]地区<ref>{{cite web |author1=Hakyung Kate Lee |title=Coin-operated karaoke machines are taking over South Korea |url=https://abcnews.go.com/International/coin-operated-karaoke-machines-taking-south-korea/story?id=53602844 |website=ABC News |date=2018-05-13 |access-date=2019-03-13 |archive-url=https://web.archive.org/web/20181128171629/https://abcnews.go.com/International/coin-operated-karaoke-machines-taking-south-korea/story?id=53602844 |archive-date=2018-11-28 |dead-url=no }}</ref><ref>{{cite web |author1=Chang Hsuan-che; Huang Chung-shan; Sherry Hsiao |title=Taichung owners call for karaoke booth regulations |url=http://www.taipeitimes.com/News/taiwan/archives/2018/07/03/2003696004 |website=Taipei Times |date=2018-07-03 |access-date=2019-03-13 |archive-url=https://web.archive.org/web/20180706090045/http://www.taipeitimes.com/News/taiwan/archives/2018/07/03/2003696004 |archive-date=2018-07-06 |dead-url=no }}</ref><ref>{{cite web |title=Lizks: Hệ sinh thái âm nhạc kiểu mới |url=https://nhipcaudautu.vn/doanh-nghiep/lizks-he-sinh-thai-am-nhac-kieu-moi-3320627/ |website=Tạp chí Nhịp Cầu Đầu Tư |date=2017-10-17}}</ref>。

== 构造与使用 ==
迷你KTV外观类似于电话亭，占地约2平方米，墙面由透明玻璃制成，部分悬挂有窗帘，其内部配备有一台卡拉OK点歌机器以及椅子、[[麦克风|麦克风]]和头戴式[[耳机|耳机]]。大多数迷你KTV设计为两人使用，其隔音效果也比较好。用户通过戴上耳机，使用点歌机进行检索点歌，然后使用麦克风进行演唱，期间可以通过点歌机上的控制面板来调解音效和音量以控制耳机的输出效果。不少迷你KTV也提供了录音功能，用户可以录制自己所唱的歌。与传统的卡拉OK店不同的是，迷你KTV使用自助式付费模式，没有工作人员进行收费，用户是通过投币或使用[[移动支付|移动支付]]进行付费使用。而迷你KTV也一般放置于商场、[[游戏厅|游戏厅]]等人流量大的地方。<ref>{{cite web |title=电话亭KTV来了 你还会去传统KTV吗 |url=http://www.xinhuanet.com//fortune/2017-03/08/c_1120590657.htm |website=新华网 |publisher=工人日报 |date=2017-03-08}}</ref>

== 发展 ==
日本是最早流行一人唱卡拉OK的地区，并有不少供单人使用的卡拉OK店，如ONE卡拉等<ref>{{cite web |title=日媒：迷你KTV风靡中国 日本也掀起“一人K歌”潮 |url=http://news.163.com/18/0217/00/DAQBQKOT00018AOQ_all.html |website=网易新闻 |publisher=参考消息网 |date=2018-02-17}}</ref>。2013年，在单人卡拉OK概念的影响下，中国大陆的厂商开始尝试研发和生产迷你KTV，但起初的迷你KTV占地面积较大。2014年，经过技术上的升级和改造，迷你KTV的占地面积减小到2平方米左右，便开始投入生产，主要的品牌有[[咪哒|咪哒]]、[[友唱|友唱]]等<ref>{{cite web |title=迷你KTV风靡街头 成熟商业模式待建立 |url=http://www.xinhuanet.com/info/2017-12/31/c_136862769.htm |website=新华网 |publisher=人民邮电报社 |date=2017-12-31 |access-date=2019-03-13 |archive-url=https://web.archive.org/web/20180222125934/http://www.xinhuanet.com/info/2017-12/31/c_136862769.htm |archive-date=2018-02-22 |dead-url=no }}</ref>。接下来的几年中，迷你KTV的产量逐渐发展，开始投入到中国大陆的各个大城市，后来又开始向小城市发展，随后也引入到台湾等地区<ref>{{cite web |title=影／中國迷你 KTV 商機延燒台灣　電話亭 KTV 現身京站 1 首 50 元 6 個月就能回本？ |url=https://www.limitlessiq.com/news/post/view/id/3138/ |website=LimitlessIQ |date=2018-01-15 |access-date=2019-03-13 |archive-url=https://web.archive.org/web/20190202220917/https://www.limitlessiq.com/news/post/view/id/3138/ |archive-date=2019-02-02 |dead-url=no }}</ref>。东亚其他国家和地区也有引进或自己研发的迷你KTV。

== 存在问题 ==
* 曲库不全：有用户表示，迷你KTV的曲库不是很全，有时候自己想唱的歌会检索不到。<ref name="耳机麦克风卫生监督问题引关注" />
* 卫生问题：由于迷你KTV采用自助式服务，所以有些用户卫生习惯不太好，有时工作人员没有对迷你KTV进行清洁或清洁力度不够。<ref name="耳机麦克风卫生监督问题引关注">{{cite web |title=迷你KTV逐渐流行 耳机麦克风卫生监督问题引关注 |url=https://www.chinanews.com/sh/2017/07-14/8277470.shtml |website=中国新闻网 |publisher=广州日报 |date=2017-07-14 |access-date=2019-03-13 |archive-url=https://web.archive.org/web/20181023030050/http://www.chinanews.com/sh/2017/07-14/8277470.shtml |archive-date=2018-10-23 |dead-url=no }}</ref>
* 版权问题：个别迷你KTV出现了使用了未经授权的歌曲的情况，侵犯了歌曲创作者的[[著作权|著作权]]。<ref>{{cite web |title=迷你KTV興起 背後卻隱藏黑暗面！ |url=https://tw.news.yahoo.com/%E8%BF%B7%E4%BD%A0ktv%E8%88%88%E8%B5%B7-%E8%83%8C%E5%BE%8C%E5%8D%BB%E9%9A%B1%E8%97%8F%E9%BB%91%E6%9A%97%E9%9D%A2-145512551.html |website=雅虎奇摩新闻 |publisher=三立新聞網 |date=2018-06-29}}</ref>
