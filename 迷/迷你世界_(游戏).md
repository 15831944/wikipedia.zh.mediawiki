'''《迷你世界》'''（{{lang-en|MiniWorld}}）是一款3D方块[[沙盒游戏|沙盒游戏]]<ref>{{Cite web|url=https://miniworld.gamepedia.com/Mini_World|title=Mini World - Mini World Wiki|accessdate=2018-05-12}}</ref>，由深圳市迷你玩科技有限公司开发。2017年12月8日，[[腾讯|腾讯]]与深圳迷你玩科技有限公司合作，在[[Wegame|Wegame]]上线了《迷你世界》页面；2017年12月14日，《迷你世界》正式上线Wegame<ref>{{Cite web|url=https://miniworld.gamepedia.com/Mini_World|title=《迷你世界》DAU超千万：我的世界出击 腾讯携手乐高备大招_手机网易网|accessdate=2018-06-10}}</ref>。

目前，该游戏的语言有[[简体中文|简体中文]]、[[泰语|泰语]]、[[越南语|越南语]]以及[[英语|英语]](国际版）{{来源请求}}。

== 法律纠纷 ==
2017年8月15日，[[网易|网易]]以涉嫌抄袭其旗下游戏《[[我的世界|我的世界]]》为由对迷你玩发出维权函，并要求进行删除下架处理。<ref>{{Cite web|url=http://games.ifeng.com/a/20170829/44670000_0.shtml|title=网易又发维权函：《迷你世界》侵权 赶紧删除|accessdate=2018-07-04|work=games.ifeng.com}}</ref>维权函发出后，游戏社区[[TapTap_(游戏社区)|TapTap]]暂时下架了该游戏，不久后重新上架。但因其声称有“版权问题”而无法下载。<ref>{{Cite web|url=https://www.taptap.com/app/7054|title=迷你世界 {{!}} TapTap发现好游戏|accessdate=2018-08-12|work=TapTap}}</ref>
2018年，网易起诉迷你世界运营方侵权，但因管辖权异议尚未有实际进展。<ref>{{cite web |title=网易在与迷你玩的官司中败诉？说这话的人是在说笑吗？ |url=https://fsight.qq.com/insight/excellentInfo/110051050c614e26a54ecbcf070e0fe34dad4df1.html |accessdate=2018-10-08}}</ref><ref>{{cite web |title=深圳市迷你玩科技有限公司、广州华多网络科技有限公司著作权权属、侵权纠纷、商业贿赂不正当竞争纠纷二审民事裁定书|url=http://wenshu.court.gov.cn/content/content?DocID=042639db-e79f-43cb-9b1e-a8fd009ea029|accessdate=2018-4-03}}</ref>2018年10月13日，网易撤回对迷你玩运营方的起诉。<ref name="中华人民共和国最高人民法院">{{cite web | title=广州网易计算机系统有限公司、上海网之易吾世界网络科技有限公司等与深圳市迷你玩科技有限公司等商业贿赂不正当竞争纠纷一审民事裁定书 | website=中华人民共和国最高人民法院 | url=http://wenshu.court.gov.cn/content/content?DocID=10d9ddba-9ca7-4d1b-a401-a9fb00a214e2 | language=zh-hans | date=2019-02-21| access-date=2019-04-04}}</ref><ref name="mini.eastday">{{cite web | title=网易为什么对迷你世界撤诉？究竟是迷你贿赂还是网易害怕认输？ | url=http://mini.eastday.com/bdmip/190402175134033.html | language=zh-hans | date=2019-04-02 |website=mini.eastday.com|access-date=2019-04-04}}</ref>
