'''現金分享計劃'''（{{lang-pt|'''Plano de Comparticipação Pecuniária no Desenvolvimento Económico'''}}，{{lang-en|'''Wealth Partaking Scheme'''}}）是2008年由時任[[澳門行政長官|澳門行政長官]][[何厚鏵|何厚鏵]]提出的，目的是分享經濟發展成果，抗擊通脹以及2008年金融危機的衝擊。首次發放為2008年，至今（2018年）已經發了11次，並且金額也有所提升。

== 受惠對象 ==
凡在限期時持有有效或可續期的[[澳門居民身份證|澳門居民身份證]]居民均是為此計劃之受惠對象。

== 發放方式 ==
澳門政府把[[社會工作局|社會工作局]]經濟援助金受益人、領取直接津貼的教職員、[[教育暨青年局|教育暨青年局]]學生福利基金以及大專助學金受惠學生、敬老金受益人、[[公務員|公務員]]、領取[[退休金|退休金]]的退休公務員及撫恤金受領人自動轉帳至其銀行帳戶；其他合乎資格人士透過郵寄方式把劃線[[支票|支票]]至受惠人士。

== 計劃詳情 ==
=== 2008年度 ===
2008年5月23日，[[經濟財政司|經濟財政司]]司長[[譚伯源|譚伯源]]聯同相關的局級官員在[[澳門政府總部|澳門政府總部]]公布現金分享計劃及有關資料；凡在2008年7月1日持有[[澳門永久性居民|澳門永久性居民]]獲發5000元[[澳門幣|澳門幣]]，而[[澳門非永久居民|澳門非永久居民]]則獲發3000元澳門幣<ref>[[華僑報|華僑報]]，[http://www.jornalvakio.com/index.php?tn=viewer&ncid=1&nid=131734&dt=20080524&lang=tw 政府公布有關「現金分享」計劃發放安排]，2008年5月24日，{{zh}}</ref>。自澳門政府公布現金分享計劃後，超過1萬1000人合乎資格領取澳門身份證人士均向[[身份證明局|身份證明局]]辦理證件，以冀望獲得澳門政府發放現金<ref>[[新華網|新華網]]，[http://news.xinhuanet.com/gangao/2008-05/28/content_8268279.htm 澳門身份證明局接獲居留權申請書數量暴增]，2008年05月28日北京時間11時14分45秒發出，{{zh}}</ref><ref>[[人民網|人民網]]，[http://hm.people.com.cn/BIG5/42275/7460814.html 逾千澳門人排隊辦身份証 加緊打印劃線支票] {{webarchive|url=https://web.archive.org/web/20100415022216/http://hm.people.com.cn/BIG5/42275/7460814.html |date=2010-04-15 }}，2008年07月02日北京時間18時30分發布，{{zh}}</ref>。

澳門政府分別在2008年7月4日、10日及16日以自動轉帳方式向有關受惠人士至其帳戶。另一方面，現金分享計劃第一批劃線支票於同月14日起以出生年份分階段發放至受惠人士；最後一批劃線支票則於該年9月12日全部寄出<ref>現金分享發放輔助中心，[http://www.planocp.gov.mo/schedule_fs_C.html 現金分享計劃發放時間安排表] {{webarchive|url=https://web.archive.org/web/20090309081029/http://www.planocp.gov.mo/schedule_fs_C.html |date=2009-03-09 }}，2009年4月1日查閱，{{zh}}</ref>

=== 2009年度 ===
[[澳門行政長官|澳門行政長官]][[何厚鏵|何厚鏵]]於2008年11月11日在[[澳門立法會|澳門立法會]]發表[[施政報告|施政報告]]時表示為了進一步向居民分享經濟成果以及對抗[[金融海嘯|金融海嘯]]而面臨帶來的壓力和衝擊；施政報告中表示澳門政府將於2009年上半年作另一次發放一次性現金分享計劃，而分享金額原則上不會少於第一次現金分享計劃發放的金額<ref>[[新華網|新華網]]，[http://news.xinhuanet.com/gangao/2008-11/12/content_10345939.htm 澳門政府再向每人派發5000元紅包應對金融風暴]，2008年11月12日[[北京時間|北京時間]]13時02分59秒發出，{{zh}}</ref>。2009年4月16日，何厚鏵在出席立法會答問大會時表示永久性居民可獲發6000澳門幣，而澳門非永久居民則獲發3600澳門幣，並預計是次分享計劃將於5月中開始發放<ref>[[澳門特別行政區政府|澳門特別行政區政府]]，[http://www.gov.mo/egi/Portal/rkw/public/view/showcomp.jsp?id=InfoShowTemp&docid=c373e920ae5070258f7f3e1851af9ac8 政府推出現金分享及醫療補貼計劃]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}，2009月4月16日18時00分06秒發出，{{zh-hant}}</ref>。是次計劃發放工作自2009年5月中旬展開至7月20日<ref>{{cite news| language = zh-mo| url = http://www.planocp.gov.mo/files/press_rel_20090724_c.html| title = 2009年度現金分享計劃新聞公佈| publisher = [[澳門特別行政區政府|澳門特別行政區政府]]| date = 2009-07-24| accessdate = 2009-08-19| deadurl = yes| archiveurl = https://web.archive.org/web/20100109074111/http://www.planocp.gov.mo/files/press_rel_20090724_c.html| archivedate = 2010-01-09}}</ref>。

===2010年度===
2010年3月16日，行政長官崔世安發表2010年施政報告，宣佈繼續現金分享計劃、醫療券等惠民政策，金額與去年相同，即向澳門居民發放6000元。

===2011年度===
2010年11月16日，行政長官崔世安發表施政報告，永久居民和非永久居民現金分享計劃金額，分別下調至四千元及二千四百元。

2011年4月20日，鑑於同年香港政府推行「$6,000計劃」，澳門特別行政區行政長官崔世安在立法會答問大會上宣布，特區政府將於下半年再推出一次性紓困措施，向每名永久性居民發放3000元，每名非永久居民1800元，預計總開支為17億元。 他又稱，現金分享只會對物價造成一次性影響，長期影響並不明顯，根據估算，每千元現金分享會推高通脹率不超過0.15%，預計此輪現金分享對通脹率的影響不超過0.45%。是次措施後來定名為「2011年度現金補助」。<ref>[[正報|正報]]，[http://www.chengpou.com.mo/news/2011/4/21/12105.html 再派三千抗通脹]，2011年4月21日，{{zh}}</ref>

===2012年度===
2011年11月15日，行政長官崔世安發表施政報告時表示向永久性居民和非永久性居民分別發放7,000及4,200元，較上一個財政年度高75%。<ref>{{cite news |title=本年度現金分享擬下季發放 |url=http://new.tdm.com.mo/c_news/radio_news.php?id=143473 |publisher=澳門電台 |date=2012-02-14 |accessdate=2012-10-07 |archive-url=https://web.archive.org/web/20160305122029/http://new.tdm.com.mo/c_news/radio_news.php?id=143473 |archive-date=2016-03-05 |dead-url=yes }}</ref>

===2013年度===
2012年11月13日，行政長官崔世安發表施政報告時表示向永久性居民和非永久性居民分別發放8,000及4,800元，較上一個財政年度高12.5%。<ref>{{cite news |title=明年度現金分享永久居民加至8000元 |url=http://new.tdm.com.mo/c_news/radio_news.php?id=165773 |publisher=澳門電台 |date=2012-11-13 |accessdate=2013-02-22 |archive-url=https://web.archive.org/web/20160903063823/http://new.tdm.com.mo/c_news/radio_news.php?id=165773 |archive-date=2016-09-03 |dead-url=yes }}</ref>

===2014年度===
2013年11月12日，行政長官崔世安發表施政報告時表示向永久性居民和非永久性居民分別發放9,000及5,400元.

===2015年度===
2014年11月11日，行政長官崔世安於施政報告建議繼續向永久性居民和非永久性居民分別發放9,000及5,400澳門元。

===2016年度===
2015年11月17日，行政長官崔世安於施政報告建議繼續向永久性居民和非永久性居民分別發放9,000及5,400澳門元。

===2017年度===
2016年11月15日，行政長官崔世安於施政報告建議繼續向永久性居民和非永久性居民分別發放9,000及5,400澳門元。

===2018年度===
2017年11月14日，行政長官崔世安於施政報告建議繼續向永久性居民和非永久性居民分別發放9,000及5,400澳門元。

===2019年度===
2018年11月15日，行政長官崔世安於施政報告建議繼續向永久性居民和非永久性居民分別發放10,000及6,000澳門元<ref name="2019Y">{{cite web |url= https://hk.news.appledaily.com/local/realtime/article/20181115/58914696 |title = 明年卸任 崔世安「加碼」派錢 永久居民每人1萬元
|work= [[蘋果日報|蘋果日報]] |date= 2018-11-15 |language= zh |archiveurl= http://web.archive.org/web/20190110191430/https://hk.news.appledaily.com/local/realtime/article/20181115/58914696 |archive-date = 2019-01-10 |deadurl= no |access-date  = 2019-01-10}}</ref>。

== 辦事處 ==
為協助現金分享計劃，澳門政府在[[塔石廣場|塔石廣場]]商業中心設立「現金分享發放輔助中心」，並於在2008年7月1日正式運作，以處理及協助有需要的居民解決收取款項問題及提供查詢等服務；基於現金分享計劃之支票兌現期為期三年，因此於同年11月3日起遷至南灣大馬路[[中華廣場|中華廣場]]二樓[[民政總署|民政總署]]綜合服務中心代為處理<ref>現金分享發放輔助中心，[http://www.planocp.gov.mo/Dispatch_20081031_fs_C.html 「2008年度現金分享計劃」新聞稿] {{webarchive|url=https://web.archive.org/web/20090116074726/http://www.planocp.gov.mo/Dispatch_20081031_fs_C.html |date=2009-01-16 }}，2008年10月31日發出，{{zh}}</ref>。
