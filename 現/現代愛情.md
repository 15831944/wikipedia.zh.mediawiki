《'''現代愛情'''》（{{lang-en|''Modern Love''}}）是一部[[獨立單元劇|獨立單元劇]]形式的[[美國|美國]][[浪漫喜劇|浪漫喜劇]][[網絡劇|網絡電視劇集]]，根據《[[紐約時報|紐約時報]]》每週發表的同名專欄文章改編，將在[[亞馬遜影片|亞馬遜影片]]首播。

== 概要 ==
《現代愛情》將探索「多種形式的愛——包括性愛、浪漫、家庭、[[柏拉圖式愛情|柏拉圖式愛情]]以及自愛<ref name="SeriesOrder">{{cite web |last1=Andreeva |first1=Nellie |title=Amazon Orders ‘Modern Love’ Anthology Comedy Series, Passes On ‘Making Friends’ |url=https://deadline.com/2018/06/amazon-orders-modern-love-anthology-comedy-pilot-passes-making-friends-1202407408/ |website=[[Deadline_Hollywood|Deadline Hollywood]] |date=2018-06-11|accessdate=2018-12-10|language=en-US}}</ref>。」

== 製作 ==
=== 開發 ===
2018年6月11日，[[亞馬遜影片|亞馬遜影片]]宣佈預訂《現代愛情》第一季，共8集。這部劇集由[[約翰·卡尼|約翰·卡尼]]編劇、執導並製作。製作公司包括Storied傳媒集團和《[[紐約時報|紐約時報]]》<ref name="SeriesOrder"/><ref>{{cite web |last1=Lawrence |first1=Derek |title=Amazon orders anthology series based on 'New York Times' 'Modern Love' column |url=https://ew.com/tv/2018/06/11/amazon-modern-love-anthology-series/ |website=[[娛樂周刊|娛樂周刊]] |date=2018-6-11|accessdate=2018-12-10|language=en-US}}</ref><ref>{{cite web |last1=Goldberg |first1=Lesley |title='Modern Love' Anthology Based on New York Times Column Set at Amazon |url=https://www.hollywoodreporter.com/live-feed/modern-love-anthology-based-new-york-times-column-set-at-amazon-1118929 |website=[[好萊塢報道|好萊塢報道]] |date=2018-6-11|accessdate=2018-12-10|language=en-US}}</ref><ref>{{cite web |last1=Amatulli |first1=Jenna |title=New York Times 'Modern Love' Column To Become Amazon Series |url=https://www.huffingtonpost.com/entry/new-york-times-modern-love-column-to-become-amazon-series_us_5b1e9144e4b09d7a3d755c07 |website=[[哈芬登郵報|哈芬登郵報]] |date=2018-6-11|accessdate=2018-12-10|language=en-US}}</ref>。11月26日，報道稱[[艾美·羅森|艾美·羅森]]、{{le|莎朗·霍根|Sharon Horgan}}和[[湯姆·霍爾|湯姆·霍爾]]將加入劇組擔任副導演。莎朗·霍根和湯姆·霍爾也會為他們參與執導的單集撰寫劇本，艾美·羅森所執導的單集劇本將由{{le|奧黛麗·威爾斯|Audrey Wells}}完成。此外，托德·霍夫曼、山姆·多尼克和{{le|科瑞·斯查|Choire Sicha}}將擔任執行製片人，特里斯·霍夫曼為製片人，丹尼爾·瓊斯為顧問製片人<ref name="DirectorsCast">{{cite web|url=https://deadline.com/2018/11/modern-love-anne-hathaway-tina-fey-john-slattery-dev-patel-cast-amazon-anthology-comedy-1202508813/|title=‘Modern Love’: Anne Hathaway, Tina Fey, John Slattery, Dev Patel Among Cast Set For Amazon’s Anthology Comedy Series|website=Deadline Hollywood|first=Denise|last=Petski|date=2018-11-26|accessdate=2018-12-10|language=en-US}}</ref>。

=== 選角 ===
2018年11月26日，亞馬遜宣佈[[安妮·海瑟薇|安妮·海瑟薇]]、[[蒂娜·菲|蒂娜·菲]]、[[戴夫·帕特尔|戴夫·帕特尔]]、[[約翰·斯拉特里|約翰·斯拉特里]]、{{le|布蘭登·維克多·迪克森|Brandon Victor Dixon}}、{{le|凱薩琳·基納|Catherine Keener}}、[[朱莉亞·加德納|朱莉亞·加德納]]、[[安迪·加西亞|安迪·加西亞]]、[[克莉絲汀·米利歐提|克莉絲汀·米利歐提]]、[[奥利维亚·库克|奥利维亚·库克]]、[[安德魯·史考特|安德魯·史考特]]、[[希亞·溫漢|希亞·溫漢]]、{{le|加裏·卡爾|Gary Carr (actor)}}、[[蘇菲亞·波提拉|蘇菲亞·波提拉]]以及[[小約翰·加拉赫|小約翰·加拉赫]]將出演本劇<ref name="DirectorsCast"/>。
