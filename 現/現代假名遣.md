'''現代假名遣'''（{{lang-ja|'''現代仮名遣い'''}}），通稱'''新假名'''（{{lang|ja|新仮名}}），是日本政府發佈的日語[[假名|假名]]的使用規範。現行的現代假名遣是[[日本政府|日本政府]]將1946年提出的内閣告示「現代假名遣」（{{lang|ja|現代かなづかい}}）修改後，在1986年7月1日以内閣告示第1號公布（日語正式名稱改寫為“現代仮名遣い”）。現代假名遣是當代用來記載[[日語|日語]]的方式之一。

[[假名遣|假名遣]]指的是假名使用方式的一組規範。「{{lang|ja|遣い}}」在這裡的意思是「使用」。

== 性質 ==
46年版現代假名遣（「{{lang|ja|現代かなづかい}}」）定性為「將現代語以假名來書寫的準則」，而86年版現代假名遣（「{{lang|ja|現代仮名遣い}}」）將其改為表現「為了書寫現代國語所使用假名的益處」，淡化具有限制色彩的內容。

46年版現代假名遣曾被設想為轉移至表音式假名遣之間的過渡工具。但是實際上想要將假名遣完全地表音化是不可能的，46年版現代假名遣就這樣定型了下來。86年版現代假名遣的不同表述可說是基於對這一事實的認識。

就具體内容來説，86年版現代假名遣的内容和46年版現代假名遣幾乎完全沒有變動。

== 原則 ==
86年版現代假名遣有以下兩個原則：

* 大致依照現代語的[[音韻|音韻]]來書寫。
* 遇到某些特定用語時，尊重記載的慣例。

因為現代假名遣是將[[歷史性假名遣|歷史性假名遣]]的記載法，基於表音主義依照現代語的[[音韻|音韻]]修改而成的假名遣，所以並不是完整的表音式記載法。

== 形式 ==
46年版現代假名遣（「{{lang|ja|現代かなづかい}}」）採取將[[歷史性假名遣|歷史性假名遣]]改寫的形式，而86年版現代假名遣（「{{lang|ja|現代仮名遣い}}」）首先說明將「把用語遵照現代語的音韻來書寫」之「原則」，再把「記載的慣例」作為「特例」補充在最後。

例如發音為「{{lang|ja|ワ}}（wa）」的單字通常記載成「{{lang|ja|わ}}」，不過助詞的「{{lang|ja|は}}」發音同「{{lang|ja|ワ}}」但記載成「{{lang|ja|は}}」。助詞「{{lang|ja|を}}」發音同「{{lang|ja|お}}（o）」一樣但記載為「{{lang|ja|を}}」。助詞「{{lang|ja|へ}}」也是發「{{lang|ja|え}}（e）」但記載成「{{lang|ja|へ}}」。

一個音的後面，跟著和同一段相同母音，而且該發音為「{{lang|ja|オ}}」「{{lang|ja|エ}}」的情況時，通常記載成「{{lang|ja|う}}」「{{lang|ja|い}}」。

例如：
* '''{{lang|ja|交換}}'''。發音為{{lang|ja|コーカン}}（kookan），記載成{{lang|ja|こうかん}}（koukan）。[[平文式罗马字|平文式]]為「kōkan」。
* '''{{lang|ja|永久}}'''。發音為{{lang|ja|エーキュー}}（eekyuu），記載成{{lang|ja|えいきゅう}}（eikyuu）。平文式為「eikyū」。

而歷史性假名遣在「{{lang|ja|ほ・を}}」發「{{lang|ja|オ}}」音的字詞，則將母音部分記載為「{{lang|ja|お}}」。

例如：'''狼'''。歷史性假名遣的記載法是{{lang|ja|おほかみ}}，發音為{{lang|ja|オーカミ}}。現代假名遣記載為{{lang|ja|おおかみ}}。

歷史性假名遣的「{{lang|ja|[[ゐ|ゐ]]}}」轉換成「{{lang|ja|い}}」、「{{lang|ja|[[ゑ|ゑ]]}}」轉為「{{lang|ja|え}}」。

== 看法 ==
反對現代假名遣的看法分成以下兩種意見。
# 假名遣沒有反映發音，破壞了語言本身
# 過於依賴漢字，不能算是表音

[[日語維基百科|日語維基百科]]原則上按照現代假名遣使用假名。

=== [[丸谷才一|丸谷才一]]的批評要點 ===
* 日語變得難讀了
* 使得無法瞭解字的語源
* 造成了許多詞彙沒有人知道該怎麼記載
* 破壞了和五十音圖關聯的動詞活用法則
* 消滅了同語異發音的緩和空間
* 將拗音、促音用小字記載的方式讓書寫變得繁瑣
