'''鹅唛县'''（{{lang-ms|Daerah Gombak}}），是[[马来西亚|马来西亚]][[雪兰莪州|雪兰莪州]]中部的一个县，也是州内第六大县。其面积为650.08平方公里，人口于2010年为682,226。该县北临[[乌鲁雪兰莪县|乌鲁雪兰莪县]]，东临[[彭亨州|彭亨州]][[文冬县|文冬县]]的[[云顶高原|云顶高原]]，东南临[[乌鲁冷岳县|乌鲁冷岳县]]，西临[[瓜拉雪兰莪县|瓜拉雪兰莪县]]，西南临[[八打灵县|八打灵县]]，西临[[吉隆坡|吉隆坡]]。鹅唛县的县府为[[士拉央新镇|士拉央新镇]]。其他地区包括[[鹅唛镇|鹅唛镇]]、{{link-ms|士拉央|Selayang}}、
{{link-ms|士拉央新镇|Bandar Baru Selayang}}、[[万挠|万挠]]、[[黑风洞镇|黑风洞镇]]、[[轰埠|轰埠]]、{{link-ms|根登|Kundang}}、{{link-ms|淡江 ( 雪兰莪)|Hulu Kelang|淡江}}等。

鹅唛县于1974年2月1日成立，当天也是[[吉隆坡|吉隆坡]]正式脱离[[雪兰莪州|雪兰莪州]]成为[[联邦直辖区|联邦直辖区]]。也因如此，当时的吉隆坡县、瓜拉雪兰莪县部分地区被划入此县，形成现今的GSR地区（Gombak-Selayang-Rawang，即鹅唛、士拉央和万挠三个主要城镇。），并由以下地区组成：

*乌鲁冷岳县的[[安邦|安邦]]区（Mukim Ampang）一部分。（该地区属于[[安邦市议会|安邦市议会]]（MPAJ）管辖）
*吉隆坡县的[[淡江区|淡江区]]（Mukim Ulu Klang）（该地区属于[[安邦市议会|安邦市议会]]（MPAJ）管辖）
*吉隆坡县的[[文良港|文良港]]区（Mukim Setapak）一部分 *吉隆坡县的[[峇都|峇都]]区（Mukim Batu）一部分
*乌雪县和瓜雪县的[[万挠区|万挠区]]（Mukim Rawang）

在1997年以前，[[万挠|万挠]]是鹅唛县的县府，但自从鹅唛县议会（Majlis Daerah Gombak）于1997年1月1日升级为现今的[[士拉央市议会|士拉央市议会]]（Majlis Perbandaran Selayang）后，该县县府便迁移至[[士拉央新镇|士拉央新镇]]（Bandar Baru Selayang）。

此县东部，北部和西部部分地区是山地，而中间为平原。鹅唛河、巴生河、峇都河和毛糯河（Buloh River）流经此县

== 行政区划 ==
鹅唛县下辖四个[[巫金|巫金]]（区级行政区），顺序依据大小排列。<ref>{{cite web|url=http://www2.selangor.gov.my/gombak.php/pages/view/18?mid=61|title=Portal Rasmi PDT Gombak Profil Gombak|work=www2.selangor.gov.my}}</ref>
*万挠（有时也被分为万挠1区和2区）
*峇都
*文良港
*淡江

==政府==
鹅唛县县府（政府所在地）为[[士拉央新镇|士拉央新镇]]。

县内下有两个[[大马地方政府|地方政府]]
* [[士拉央市议会|士拉央市议会]]（大部分区域）
* [[安邦市议会|安邦市议会]]（淡江）

==教育==
截至2014年，鹅唛县共有53间国民小学、八间华文小学、七间淡米尔小学、30间国民中学、两间宿舍学校、两间伊斯兰中学和两间技术职业中学（Vocational School）。<ref>{{cite web|url=http://www.mps.gov.my/ms/pelawat/pendidikan/sekolah/page/0/1|title=Sekolah|first=|last=primuscoreadmin|date=2015-11-12|publisher=}}</ref><ref>{{cite web |url=http://jpnselangor.moe.gov.my/v4/images/PDF/senarai_sekolah/gombak.pdf |title=存档副本 |accessdate=2017年6月6日 |deadurl=yes |archiveurl=https://web.archive.org/web/20170622184139/http://jpnselangor.moe.gov.my/v4/images/PDF/senarai_sekolah/gombak.pdf |archivedate=2017年6月22日 |df= }}</ref>
