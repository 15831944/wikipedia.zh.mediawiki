'''家蚕'''（[[学名|学名]]：''{{lang|la|Bombyx mori}}''）是[[鳞翅目|鳞翅目]]的[[昆虫|昆虫]]，[[丝绸|丝绸]]的主要原料来源，在人类[[经济|经济]]生活及文化歷史上占有重要地位。原产[[中国|中国]]，华南地区俗称之'''蚕宝宝'''或'''娘仔'''。

家蚕的英文名为“silkworm”（意为“丝虫”）是因为它用丝织[[茧|茧]]。茧是由一根300-900米长的丝织成的。

==特征==
幼虫有13个环节，胸腹部有8对足，依品種不同身体呈青白色或黑色，虫足極具黏性。

成虫有3对足，两对翅膀，身体上有白色的鳞毛。

===性別特徵===
蠶在四齡時，身體變大，易区分雄与雌。

從蠶的身體的尾部算起，如果在第二、三環節之間有一個叫做[[海洛爾特氏腺|海洛爾特氏腺]]的凸出的小圓點，则是雄性；如果在倒數第二、三環節的下面，有被稱為[[石渡氏腺|石渡氏腺]](Ishiwata's gland)的四個凹形的小圓點，则是雌性。

====從蠶繭分辨====
蠶繭尾端的下方，如果有個Ｘ形，则是雌的；如果沒有则是雄的，而且母的蠶繭較为肥胖，雄的蠶繭比较尖長。

====從成蟲分辨====
蠶蛾更容易分辨：雌蠶蛾身體較肥胖，交配後會產卵，雄蠶蛾身體則比较尖长。

==歷史==
{{main|絲綢歷史}}
从考古家在吳越發現的絲綢實物來看，在5000年前的江南地區就有[[桑|桑]]蠶業。<ref>（http://content.edu.tw/primary/nature/ks_gc/nctheme/ckl01/3-5-9.htm）。在{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}[[浙江|浙江]][[吳興|吳興]]的郊外，考古学家發現了具有五千年之久的人类的遗物，其中就有絲織品。</ref>

相傳早於西元前三千多年，[[黃帝|黃帝]]的妻子[[嫘祖|嫘祖]]就開始「養蠶取絲」。

中國[[絲綢|絲綢]]的秘密，後來隨移民傳播至[[朝鲜|朝鲜]]、[[印度|印度]]等地，傳說在公元六世紀[[拜占庭|拜占庭]]的[[基督教|基督教]]僧侶从中國偷運蠶蛹，養蠶技術才传到了歐洲。

蚕是對人類貢獻最久、最偉大的昆蟲之一。

== 习性 ==
[[File:CSIRO_ScienceImage_10746_An_adult_silkworm_moth.jpg|thumb]]
[[File:Silkmoth.jpg|thumb]]
[[File:Cocoon.jpg|thumb]]
[[File:Weighing_and_sorting_the_cocoons_(Sericulture_by_Liang_Kai,_1200s).jpg|thumb]]《蚕织图》，蚕茧]]

家蚕主食为[[桑树|桑葉]]，也可用其它一些樹葉（柞樹，柘樹亦可養蠶）。家蚕有很强的食慾，它们昼夜不停地吃[[桑|桑]]叶，因而长得很快，有成语'''蠶食鯨吞'''。此外蠶非常怕冷，溫度稍低就有可能冷死，必須注意室溫保暖，蠶同時也是螞蟻的美食，也需注意螞蟻對蠶的生命威脅。

当它们头部的颜色变黑的时候，即表明它们将要蜕皮。在完成四次蜕皮之后，它们的身体会变为浅黄色，皮肤也变得更紧，这表明它们将要用丝来包裹自己（茧），在茧中[[变态|变态]]成[[蛹|蛹]]。它们從小长到長大，可以長大9000倍。

如果蚕蛹变为[[成虫|成虫]]，它们会将茧溶解并钻出。由于它们破洞而出，故而蚕茧会被破坏，[[丝线|丝线]]将会变短，不能用于纺丝织绸，所以要在其尚未破茧以前，即将蚕茧放入沸水中，以杀死蚕蛹，并使茧易于拆解。

家蚕的蟲及蛹可以食用，並有[[食療|食療]]的功效。

成熟的蚕（“蚕蛾”）不能飞，它们的唯一目的只在于产卵，从而繁殖后代。蠶蛾交配完後，雄蛾會先死去，雌蛾則繼續產卵，最後也死去。

因其久远的历史和经济上的重要性，家蚕的[[基因|基因]]已成为现代科学的重要研究对象。
但由於近年家蠶疾病猖獗，原多用作教學上的目的，已漸漸式微，甚至在作為研究用途上也具一定的挑戰性。

== 疾病 ==
目前，以病毒引起的軟化症為主，其病體和糞便若無隔離處置，將導致該群體大量死亡。
並且飼養的合適溫度為23度為佳。

== 医用 ==
[[File:Silkworm_01.png|thumb]]
蚕是传统[[中药|中药]]“僵蚕”的来源。僵蚕是蚕的4-5期幼虫感染了一种名为[[白僵菌|白僵菌]]的[[真菌|真菌]]并死亡后的乾燥体，具有散风和化痰祛痰的功效。[[孙思邈|孙思邈]]的《备急千金要方》载：“蚕蛹性味咸温，主益精气”；《食疗本草》载：“蚕蛹富含养分，食用甚益人”。現代醫學證實蚕蛹油脂和蚕蛹蛋白具有降血糖、增强免疫、抗衰老等功效。

== 食用 ==
在中國北方，「炸蠶蛹」是一種传统的食品，[[北魏|北魏]]时期[[賈思勰|賈思勰]]的《[[齊民要術|齊民要術]]》中就有“以蠶蛹禦宴客”的記載。

在泰國有炸蠶料理。

== 教育 ==
