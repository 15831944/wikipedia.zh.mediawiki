'''霍建寧'''（{{lang-en|FOK Kin Ning, Canning}}，{{bd|1951年|8月22日|catIdx=H霍}}）<ref>{{cite news|url=http://www.zj.xinhuanet.com/magazine/2006-09/25/content_8123899.htm|title=“打工皇帝”霍建宁|publisher=新華網|date=2006-09-25|deadurl=yes|archiveurl=https://web.archive.org/web/20080713174749/http://www.zj.xinhuanet.com/magazine/2006-09/25/content_8123899.htm|archivedate=2008-07-13}}</ref>是[[長江集團|長江集團]]的高層管理人之一。霍建寧是[[香港|香港]]著名的超級「打工皇帝」之首，也是[[香港稅務局|香港稅務局]][[薪俸稅|薪俸稅]]的大戶之一。在2006至2007年，他的收入是1.3亿港元，是当年香港收入最高的人，有媒體指霍氏自1994-2014的收入達25.4亿港元。<ref>{{cite news|url=http://www.southcn.com/news/hktwma/gangao/mrb/200208191207.htm|title= 香港“打工皇帝”霍建宁年赚过亿|publisher=南方網|date=2002-08-19}}</ref> <ref>[http://hk.apple.nextmedia.com/financeestate/art/20150514/19146522  趣BLOGBLOG：霍建寧袋25億代價]</ref>
 
== 早年 ==
1960年代，霍建寧就讀於福榮街官立小學以及[[拔萃男書院|拔萃男書院]]。1970年代，霍建寧以[[文学士|文學學士]]於[[香港大學|香港大學]]畢業，然後留學[[美國|美國]]，其後亦於[[澳洲|澳洲]][[新英格蘭大學|新英格蘭大學]]取得財務管理文憑。
== 创业 ==
1979年返港加入[[長江集團|長江集團]]。晉升為會計主任後，再取得澳洲特許會計師協會會員資格，于1983年離開与友人合资开会计师行。可是，会计师行不久便告结业。
== 发掘 ==
翌年回巢，1985年出任長實董事，年僅33歲。<ref>http://hk.apple.nextmedia.com/financeestate/art/20150525/19159054</ref>1993年，霍建宁到日本公干，被李嘉诚的另一间公司和黄集团借调过去帮忙，结果和黄集团的董事总经理[[马世民|马世民]]意外发现此人是一个出色的谈判专家。由日本回到香港后，马很快就向李嘉诚推荐了霍。马世民离职后，霍建宁接替其走马上任，成为和黄新的“一把手”（董事总经理）。<ref>{{Cite press release |language = zh-cn |author = 凤凰周刊 |url = http://news.ifeng.com/shendu/fhzk/detail_2012_05/08/14393899_0.shtml |title = 首次披露了当年他是如何看中比他小12岁的霍建宁 |date = 2012-05-08 }}</ref>
== 軼聞 ==
2006年4月霍建寧以3.5億購買[[深水灣道|深水灣道]]37號的豪宅 <ref> [https://hk.news.appledaily.com/local/daily/article/20070223/6841423 福布斯全球調查霍建寧新居上榜亞洲5大豪宅香港佔4間]</ref>。
