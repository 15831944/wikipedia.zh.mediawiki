'''霍爾蘭六邊形'''（{{lang-en|Holland Hexagon/Holland Codes}})，又名'''霍爾蘭職業類型論'''，是由[[心理學家|心理學家]][[約翰·霍爾蘭|約翰·霍爾蘭]](又譯''霍蘭德'')發明，主要應用於[[生涯規劃|生涯規劃]]及職業輔導，藉以協助受測者了解自己人格特質，選擇能反映個人人格特質的職業
<ref>{{cite web|url=https://www2.palomar.edu/pages/directory/category/employees/|title=Employees – Directory|author=|date=|website=www2.palomar.edu|accessdate=2018-12-13}}</ref>
<ref>{{cite web |url=http://tiunet.tiu.edu/acadinfo/cas/socsci/psych/robinson/210/hollandcodes.htm |title=存档副本 |accessdate=2006-03-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20070928152908/http://tiunet.tiu.edu/acadinfo/cas/socsci/psych/robinson/210/hollandcodes.htm |archivedate=2007-09-28 }}</ref>
<ref>{{cite web |url=http://career.missouri.edu/students/explore/thecareerinterestsgame.php |title=存档副本 |accessdate=2006-03-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20060404064546/http://career.missouri.edu/students/explore/thecareerinterestsgame.php |archivedate=2006-04-04 }}</ref>
<ref>{{cite web |url=http://peercareer.appstate.edu/holland.html |title=存档副本 |accessdate=2006-03-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20060902114653/http://www.peercareer.appstate.edu/holland.html |archivedate=2006-09-02 }}</ref>。

受測者主要利用問卷調查來瞭解自己的性向，並根據分數而計算出個人對六種特質的偏好。這六種特質排列成一個[[六邊形|六邊形]]，分別代表六大類工作技能<ref>{{Cite web |url=http://www.careerkey.org/asp/your_personality/hollands_theory_of_career_choice.asp |title=(RIASEC) |access-date=2006-03-24 |archive-url=https://web.archive.org/web/20060320065400/http://www.careerkey.org/asp/your_personality/hollands_theory_of_career_choice.asp |archive-date=2006-03-20 |dead-url=no }}</ref>的特質:
[[Image:Seriousjobseeker.com_hexagon.gif|right]]
<li>實用型（Realistic） - 工具，機械，自然，物理</li>
<li>研究型（Investigative） - 分析，調查，觀察，邏輯</li>
<li>藝術型（Artistic） - 創作，想象，感性，直覺</li>
<li>社會型（Social） - 合作，協助，訓練，護理</li>
<li>企業型（Enterprising） - 領導，談判，企業，策略</li>
<li>事務型（Conventional） - 組織，系統，數字，精確</li>

霍爾蘭指出受測者通常會有多於兩項主要技能，所以通常在分析過後都不會指明受測者所屬的類型，而是會把受測者在各個組別的測試結果擺在一起。例如：RIASEC：18/40/14/21/8/14，以方便對受測者進行更詳細的指導。
