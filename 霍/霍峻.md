'''霍峻'''（{{bd|177年||216年|catIdx=H}}），字'''仲邈'''，[[荊州_(古代)|荊州]][[南郡|南郡]][[枝江|枝江]]人，[[三國|三國]]時期[[刘备|刘备]]部将。

== 生平 ==
其兄[[霍篤|霍篤]]曾在故鄉聚集起[[部曲|部曲]]數百人。後來霍篤逝世，[[劉表|劉表]]命令霍峻繼承部曲。208年，劉表病逝，霍峻便率部曲歸降[[劉備|劉備]]，並被任為[[中郎將|中郎將]]。 

213年，被邀入蜀的劉備與蜀主[[劉璋|劉璋]]決裂，劉備軍由[[葭萌|葭萌]]向南進攻劉璋，而留霍峻守葭萌城。漢中的[[張魯|張魯]]派[[楊帛|楊帛]]想誘惑霍峻獻城，但霍峻說：「小人頭可得，城不可得。」楊帛惟有回去。後來劉璋將領[[扶禁|扶禁]]、[[向存|向存]]等率萬多人由[[閬水|閬水]]北上攻打霍峻，並圍城達一年之久，但仍不能攻下。

葭萌城中只有數百人，不過霍峻以逸待勞，挑選精銳之兵出擊，大破敵軍，亦斬殺向存。214年，劉備成功入主[[益州|益州]]，霍峻有功而受到嘉許，又乃分[[廣漢|廣漢]]為[[梓潼|梓潼]]郡，任霍峻為梓潼太守、裨將軍。在任三年，四十歲逝世。葬於[[成都|成都]]，劉備十分哀悼和可惜，下詔[[諸葛亮|諸葛亮]]：「峻既佳士，加有功於國，欲行酹。」親率群僚弔祭，更留宿在墓上，當時非常榮耀。

== 特徵 ==
霍峻為人冷靜、善戰，率領數百人的自己面對有數萬人的敵軍也能冷靜分析形勢，更可大破敵軍，斬殺敵人。

== 三國演義 ==
在62回出場，由[[孟達|孟達]]推薦給劉備，和孟達一起去守葭萌關。65回遇到[[張魯|張魯]]派[[馬超|馬超]]，楊柏，[[馬岱|馬岱]]攻擊葭萌關，孔明派[[張飛|張飛]]迎戰，[[李恢|李恢]]說服馬超投降。70回遇到[[張郃|張郃]]攻打葭萌關，孔明派[[黃忠|黃忠]][[嚴顏|嚴顏]]迎戰。霍峻在三國演義中擔任葭萌關守將，沒有戰績。

== 評價 ==
[[三國志|三國志]]評曰:「霍峻孤城不傾，[[王連|王連]]固節不移，[[向朗|向朗]]好學不倦，[[張裔|張裔]]膚敏應機，[[楊洪|楊洪]]乃心忠公，[[費詩|費詩]]率意而言，皆有可紀焉。以先主之廣濟，諸葛之準繩，詩吐直言，猶用陵遲，況庸后乎哉！」 

[[劉備|劉備]]下詔[[諸葛亮|諸葛亮]]：「峻既佳士，加有功於國，欲行酹。」
