'''霍爾木茲海峽'''（{{lang-ar|'''مضيق هرمز'''}}，{{lang-fa|'''تنگه هرمز'''}}）是連接[[波斯灣|波斯灣]]和[[印度洋|印度洋]]的海峽，亦是唯一一個進入波斯湾的水道。海峽的北岸是[[伊朗|伊朗]]，有[[阿巴斯港|阿巴斯港]]。海峽的南岸是[[阿曼|阿曼]]的[[飞地|飞地]][[穆桑代姆省|穆桑代姆省]]，其南面是[[阿拉伯联合酋长国|阿拉伯联合酋长国]]。海峽中間偏近伊朗的一邊有一個大島叫做[[格什姆島|格什姆島]]，隸屬於伊朗。現時霍爾木茲海峽是全球最繁忙的水道之一。

== 名字的由來 ==
霍爾木茲海峽這個名字的由來，據說有4個說法：
# '''霍爾木茲'''在[[波斯語|波斯語]]的意思是「光明之神」。
# {{fact|time=2011-12-27|古希臘的[[馬其頓國王|馬其頓國王]][[亞歷山大|亞歷山大]]大帝派大將[[霍爾木茲雅|霍爾木茲雅]]率艦隊攻波斯，艦隊在海峽中的一個無名島嶼上停泊，後人紀念這位統帥，便將他的名字作為那海峽和島嶼的名字。「霍爾木茲雅」漸因音變轉化為阿拉伯腔的「霍爾木茲」。}}
# {{fact|time=2011-12-27|約在10世紀時由[[阿拉伯人|阿拉伯人]]在海峽中的霍爾木茲島上建立了一個叫做“[[忽里模子|忽里模子]]”的国家，海峽從此以此命名。}}
# {{fact|time=2011-12-27|16世紀[[葡萄牙人|葡萄牙人]]入侵，在該處建立基地，因為帶來繁華和財富，葡萄牙人稱為 '''Ormuch'''（意思為這裡多[[金|金]]），再由音譯改變成「霍爾木茲」。}}

== 海峽資料 ==
* 長度：約150公里
* 寬度：平均約50公里，最窄處僅得38.9公里
* 全年水溫範圍：24-32℃
* 海灣進口處的鹽度：千分之37-38

== 歷史 ==
霍爾木茲海峽既然是連接波斯灣和印度洋的唯一水道，故此與古時[[波斯|波斯]]地區貿易往來起了重要的關係。[[威尼斯共和国|威尼斯共和国]]大旅遊家[[馬可孛羅|馬可孛羅]]在其遊記上記載過该地，而在15世紀時[[鄭和|鄭和]]亦曾多次到訪過一個名為“[[忽魯謨斯|忽魯謨斯]]”的岛国，而「忽魯謨斯」可能是「霍爾木茲」的古音譯。[[忽魯謨斯|忽魯謨斯]]是當時使用波斯語強大的蒙古帖木耳帝國的一部份，當時帖木耳帝國可說是僅次於大明帝國的世界強國，雖然帖木耳帝國鼎盛時期很短，統治波斯只有三十多年的時間。故明朝初年與帖木耳帝國的交往是意義重大的，尤其永樂皇帝北征時。

== 戰略價值 ==
霍爾木茲海峽可以說是[[中東|中東]]地區的「油庫總[[閥門|閥門]]/[[咽喉|咽喉]]」。自從中東地區發現[[石油|石油]]後，這裡每天有數以百計的運油船經過，以水路將石油運出中東。每年有7.5億噸以上運往世界各地的中東石油須經過霍爾木茲海峽，佔海灣地區石油總出口量的90%、全球石油產量的五分之一及出口貿易量的三分之一。故此這個海峽的海上安全，對全球經濟的影響和發展極為重要。
