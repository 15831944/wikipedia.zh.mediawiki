'''霍尔瓦特票'''或'''霍尔瓦特卢布'''<ref>{{cite journal |author1=董昕 |title=奉系时期的东三省币制 |journal=中国钱币 |date=2009 |issue=4 |pages=16-20}}</ref>（{{lang-ru|хорватовский рубль}}{{efn|俄语中也称之为'''哈尔滨卢布'''（{{lang|ru|Харбинский рубль}}）或'''俄亚银行卢布'''（{{lang|ru|рубль Русско-Азиатского банка}}）。}}），又称'''华俄道胜银行中东铁路券'''<ref>{{cite web |title=1351 华俄道胜银行中东铁路券（1917年）50戈比2枚、1卢布1枚，合计共有3枚 |url=http://auction.artron.net/paimai-art5123521351/ |website=雅昌艺术网}}</ref>，是[[中东铁路|中东铁路]]管理局授意[[华俄道胜银行|华俄道胜银行]]发行的一种[[卢布|卢布]]纸币，有50戈比、1卢布、3卢布、10卢布和100卢布等5种面额，由{{le|美国钞票公司|ABCorp (American Banknote Corporation)}}印制，总发行量为2000万卢布。

== 历史 ==
{{seealso|羌帖}}
1918年12月，[[华俄道胜银行|华俄道胜银行]]哈尔滨分行奉[[中东铁路|中东铁路]]督办兼护路军总司令{{tsl|ru|Хорват, Дмитрий Леонидович|狄米特里·列奥尼德维奇·霍尔瓦特}}命令以市面缺乏小额货币为名发行卢布兑换券，被称为“霍尔瓦特票”。该币的发行受到了社会舆论的抵制，尽管哈尔滨《远东报》极力为其辩护、鼓吹，但始终未获得商民认同。1919年1月10日和15日，黑龙江督军[[鲍贵卿|鲍贵卿]]先后电告[[吉林省|吉林]]督军[[孟恩远|孟恩远]]、省长[[郭宗熙|郭宗熙]]和[[奉天省|奉天]]巡阅使[[张作霖|张作霖]]一致拒绝接受，并通令各商会一律拒用。鲍贵卿指出“霍尔瓦特现发行新羌帖，为数甚巨……假借找零为名，实则扩张票额。查中东铁路经过地方，本为我国领土，俄人于此行使纸币，已属碍我主权。自羌帖低折以来，商民损害不下万万。现霍氏又发新帖，万难听其行使。”由于俄方的全力推行，最终在哈尔滨[[道里区|埠头区]]及中东铁路沿线勉强行用。1920年3月13日，霍尔瓦特因中东铁路的中俄工人罢工被迫去职，中国政府接管中东铁路。此后，华俄道胜银行拒绝收兑霍尔瓦特票，中东铁路也不再使用，霍尔瓦特票成为废纸。<ref name="dbw">{{cite web |author1=张新知 |title=“羌帖”：老哈尔滨人的祸害 |url=https://heilongjiang.dbw.cn/system/2008/04/27/051233410.shtml |website=东北网－新晚报 |date=2008-04-27}}</ref><ref name="qiao">{{cite journal |author1=乔传义 |title=“霍尔瓦特票”发行始末 |journal=中国钱币 |date=2007 |issue=1 |pages=72-75}}</ref><ref name="wang">{{cite journal |author1=王学文 |coauthors=张新知 |title="羌帖":危害白山黑水五十载 |journal=俄罗斯研究 |date=2003 |issue=3 |pages=78-83}}</ref>

== 票样 ==
=== 描述 ===
该币正面有火车图案，背面均印有俄语告示，意为“本券在所有的中东铁路收款处和在哈尔滨、[[海拉尔|海拉尔]]、[[宽城子|宽城子]]的华俄道胜银行分行均可接受，并与1917年以前包括1917年流通的国家钞票通用，伪造者依法必究”，两面均有序列号、俄文银行名“{{lang|ru|РУССКО-АЗІАТСКІЙ БАНКЬ}}”（意为俄亚银行，其中文名为华俄道胜银行）、俄文地名“{{lang|ru|ХАРБИНЬ}}”（哈尔滨）、英文印钞厂名“American Bank Note Company”（美国钞票公司），以及俄语和阿拉伯数字的面额。50戈比票券正面和其他面额票券背面的下侧均印有行长、铁路局长及副行长签字。霍尔瓦特票的发行广告上有对票面上的具体签字情况的详细描绘：“该项兑换券在50戈比券上印道胜银行行长{{tsl|ru|Путилов, Алексей Иванович|阿列克谢·伊万诺维奇·普蒂洛夫|普吉罗夫}}及东铁局长霍尔瓦特之押；在1卢布、3卢布、10卢布券上除印普吉洛夫及霍尔瓦特之押外，并印有哈尔滨分行行长皮灭诺夫以检查员名目之押；在100卢布券上印有皮灭诺夫个人之押，或下列哈尔滨分行副行长之一，即秋列灭夫、保保夫、柏夫洛夫、吉米库斯、马切沃司或沃洛达夫之押”。<ref name="qiao"/>
