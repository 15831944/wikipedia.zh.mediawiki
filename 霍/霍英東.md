'''霍英東'''博士{{大紫荊}}（Dr. '''Henry Fok'''，{{bd|1923年|5月10日|2006年|10月28日|catIdx=H霍}}），原名'''霍官泰'''，結婚時按[[字輩|字輩]]取[[譜名|譜名]]「'''好釗'''」，生於[[香港|香港]]四大原住民系中[[蜑家|蜑家]]的[[水上人|水上人]]，原籍[[廣東|廣東]][[番禺|番禺]]练溪村<ref>http://www.chinanews.com/ga/news/2006/10-30/811896.shtml</ref>，[[香港|香港]]企業家，亦是著名的[[香港親中共人士|親中共人士]]，曾任[[全国政协副主席|全国政协副主席]]、[[全国人大常委|全国人大常委]]等政府要职，是成為首名躋身[[党和国家领导人|国家领导人]]級別的[[香港人|香港人]]，有「愛國商人」稱號。2006年[[福布斯|福布斯]]雜誌將霍英東列入全球富豪排行榜第181名、香港富豪榜第9名，資產37億美元。官方[[新華社|新華社]]將他形容為「[[中國共產黨|中國共產黨]]的親密朋友」；而前行政長官[[曾蔭權|曾蔭權]]稱讚他為「左派港人」<ref>[http://www.apdnews.com/news/41524.html 霍英东家族争产风波再掀波澜]，亚太日报，2013年10月30日</ref>。

== 生平 ==
[[File:HK_Causeway_Bay_Hysan_Avenue_2_Empire_Court.JPG|thumb]]]]
霍英東是[[蜑家|艇戶]]出身，在家中排行第四，祖父[[霍達潮|霍達潮]]擁有風帆來往[[港澳地區|港澳]]間運輸貨物，父親[[霍耀容|霍耀容]]繼承祖業。1930年一次風災中兩名哥哥翻艇身亡，其後父親患[[癌|癌]]身故，霍母帶同[[霍英東|霍英東]]及一姐一妹上岸生活，在[[灣仔|灣仔]][[石水渠街|石水渠街]]居住。[[霍英東|霍英東]]小時就讀於帆船同業義學，後轉往[[敦梅小學|敦梅小學]]，1936年考入[[皇仁書院|皇仁書院]]讀中學，為圖報國，取名「英東」即英武地屹立在世界的東方；[[二戰|二戰]][[香港淪陷|香港淪陷]]期間被迫停學並從事苦力等工作謀生，其後霍家湊足資本，在灣仔[[鵝頸橋|鵝頸橋]]開辦「有如雜貨鋪」。戰後先以買賣各種物資獲利。

1950年代初[[朝鲜战争|朝鲜战争]]爆發後，[[香港殖民地時期#香港政府|港英政府]]據[[聯合國|聯合國]]決議，對[[中國大陸|中國大陸]]實行[[禁運|禁運]]。中共聯絡在港商人[[抗美援朝|抗美援朝]]，擁有完整船隊的[[霍英東|霍英東]]爲中國打朝鲜战争提供支援。他收購戰爭及醫藥物資後，每晚親自指揮船隊運貨往中國大陸獲得利润。

他隨後轉而投資地產，1953年創辦[[霍興業堂置業有限公司|霍興業堂置業有限公司]]，再開設立[[信置業有限公司及有榮有限公司|信置業有限公司及有榮有限公司]]，在灣仔及[[銅鑼灣|銅鑼灣]]一帶發展住宅樓宇，包括[[蟾宮大廈|蟾宮大廈]]、[[香港大廈|香港大廈]]、[[海誠大廈|海誠大廈]]等，更首創[[樓花|樓花]]制度，容許買家在大廈未落成前預先訂購，當時引起市民猜疑，但此法令樓市活躍起來，地產商紛紛效法，影響至今，他亦身家暴漲。

1960年代與[[何鴻燊|何鴻燊]]、[[葉漢|葉漢]]等奪得澳門賭場經營權，建立「東方蒙地卡羅」王國，並獲得豐厚財富。1965年牽頭成立[[香港地產建設商會|香港地產建設商會]]，並獲推舉為首任會長。1967年5月[[六七暴動|六七暴動]]前夕離港逃難，至12月暴亂平息後才回港，之後開始淡出地產轉戰石油產品市場。

1970年代石油危機後，在香港經營[[東方石油公司|東方石油公司]]，售賣在中國出產的石油產品。1981年[[董浩雲|董浩雲]]逝世，[[東方海外|東方海外]]交予[[董建華|董建華]]打理。1985年，[[東方海外|東方海外]]瀕臨破產，[[霍英東|霍英東]]在1986年注資近九億四千萬元使其渡過難關。

1977年成立的[[香港霍英東基金有限公司|香港霍英東基金有限公司]]一直以捐獻與非牟利投資形式在香港策劃多個項目；1986年成立[[霍英東教育基金會|霍英東教育基金會]]獎勵國內對教育、科研、社會科學等領域有貢獻之學生及教師；2002年成立[[澳門霍英東基金會|澳門霍英東基金會]]致力於推動澳門教育、醫療、體育、文化及公共事業。晚年致力開發[[珠江|珠江]]西岸的[[南沙港|南沙港]]工程；霍英東亦愛好踢[[足球|足球]]，是[[香港足球總會|香港足球總會]]永遠名譽會長，[[東昇足球隊|東昇足球隊]]會長，亦參與各項體育事務。

== 涉足賭界 ==
[[File:Macau_Hotel_Casino_Lisboa_Macau_Mo707.JPG|thumb]]的旗艦[[葡京酒店|葡京酒店]]]]
1961年10月，[[霍英東|霍英東]]與[[何鴻燊|何鴻燊]]、[[葉德利|葉德利]]及[[葉漢|葉漢]]合組財團，以316.7萬元在[[澳門|澳門]]奪得賭牌，並在1962年[[元旦|元旦]]開設首間賭場[[新花園娛樂場|新花園娛樂場]]，同年[[澳門旅遊娛樂有限公司|澳門旅遊娛樂有限公司]]正式註冊，他同時成為澳娛最大股東，多年來雄霸澳門賭業<ref>賭壇老祖霍英東  香港蘋果日報  2006-09-03</ref>。

澳娛的股權分佈，外界所知不多，[[霍英東|霍英東]]亦曾承認公司股份的很複雜。他憶述澳娛成立時，股東包括[[何鴻燊|何鴻燊]]、[[葉漢|葉漢]]、[[鄧永祥|新馬師曾]]和[[葉德利|葉德利]]等，但他直指澳娛帳目無人說得清，過去亦有分紅不均的情形。其中澳娛每年淨賺10多億元，佔三分一股權的[[霍英東|霍英東]]卻只分得1億。2001年，澳娛另一名股東[[何婉琪|何婉琪]]傳出欠債事件，及後更因[[何婉琪|何婉琪]]欲將旗下澳娛股權轉到兒子[[麥舜銘|麥舜銘]]名下，引發與[[何鴻燊|何鴻燊]]就澳娛派息政策展開罵戰及法律<ref>[[霍英東|霍英東]]：澳娛帳目無人說得清 明報  2005-11-25</ref>。

在股東內閧之際，[[霍英東|霍英東]]2002年4月1日<ref>與[[何鴻燊|何鴻燊]]議價售股談不攏 [[霍英東|霍英東]]盡捐澳娛股份　明報　2002-04-02</ref>宣布把他在澳娛的27.7%股權（共22,654股）全數捐贈同年6月28日成立的[[澳門霍英東基金會|澳門霍英東基金會]]，另再捐贈2,000萬元[[澳門幣|澳門幣]]至其中，估計基金管有資產達60至100億元。但是，當時只有[[何鴻燊|何鴻燊]]表示有意收購他手上的股份，雙方仍未能就價錢問題作出結論。霍指出自己「四十年沒看過賭場盤數，二十年沒去過澳門」，創辦澳娛只想行善而非牟利。

在放棄股權時，[[霍英東|霍英東]]對[[何鴻燊|何鴻燊]]的批評亦引起外界猜測兩人不和。[[霍英東|霍英東]]在基金會成立時曾發表演說，批評[[何鴻燊|何鴻燊]]好勝：「1962年向他借40萬元，爭奪澳門賭權。」他又翻舊帳指1967年遭[[英國|英]][[美國|美]]政府封殺時，[[尖沙咀|尖沙咀]][[星光行|星光行]]被斷水、斷電及斷電話線，結果[[何鴻燊|何鴻燊]]把原本值30億元的大樓，以3,000萬元「賤售」。[[何鴻燊|何鴻燊]]後來說：「幾十年朋友，他開玩笑，講講故事而已，我晚晚都唔輸得（不可以輸）。」<ref>蝕讓[[星光行|星光行]]被指唔輸得 賭王「罰企」聽[[霍英東|霍英東]]炮轟　 明報  2002-06-29</ref>

== 與英美交惡 ==
[[File:HK_StarHouse.jpg|thumb]]]]
[[霍英東|霍英東]]雖然在[[英國殖民地|英國殖民地]]的香港出生，但與英國和盟友[[美國|美國]]都一直處於敵對關係。1950年代[[韓戰|韓戰]]爆發，英美根據[[聯合國|聯合國]]決議，對中國大陸實施[[禁運|禁運]]。“凡是一个士兵可以利用的东西不许运往共产党中国。”[[美国商务部|美国商务部]]的官员这样说道。而在[[日本|日本]]、[[台湾|台湾]]再到[[东南亚|东南亚]]的包围圈中，[[香港|香港]]和[[澳门|澳门]]是最后的缺口。香港当局严格执行着“全面禁运”的要求。当时的香港报纸上，经常可以看到[[英国海军|英国海军]]“开枪追击走私船只”的报道。

在接下来的近三年时间-{zh-hans:里;zh-hk:裏;zh-tw:裡;}-，霍英东不仅承担了在港澳和中國间运输军用物资的主要任务，他还组织了精密的侦察队伍，监控港英当局的缉私艇的动向；他的船队每天半夜都从[[英国皇家海军|英国皇家海军]]的[[军舰|军舰]]旁悄悄绕过，驶向[[公海|公海]]；而为了摆脱当局的监视，他甚至一天之内换了三个不同的地点，作为整个运输系统的“指挥部”。 [[霍英东|霍英东]]提供大量战略物资给[[中共|中共]]，帮助中共[[抗美援朝|抗美援朝]]，惹来英美政府的不满，并且因此一直受到港英政府的非难，但也因此受到了中共的信任，后来荣居[[全国政协副主席|全国政协副主席]]，被誉为“'''红顶巨贾'''”。

霍英東與[[何鴻燊|何鴻燊]]、[[何添|何添]]、[[關明|關明]]及[[鍾明輝|鍾明輝]]等人合組[[九龍置業公司|九龍置業公司]]於[[1962年|1962年]]開始在[[尖沙咀|尖沙咀]]興建[[星光行|星光行]]（當時稱為[[九龍商業大廈|九龍商業大廈]]），但[[星光行|星光行]]落成招租時，當時[[美國駐港領事|美國駐港領事]]將[[星光行|星光行]]列入「[[黑名單|黑名單]]」，明確宣布所有[[星光行|星光行]]的租客都不能買賣美國貨。就算[[星光行|星光行]]的租戶向當時由英資[[大東電報局|大東電報局]]控制的[[香港電話|香港電話]]公司申請電話線時，也遭到刁難，指「星光行的租戶申請電話線可能遙遙無期」。結果大部份租戶連按金都不要就走了。最後，霍在[[星光行|星光行]]承租乏人問津的情況中，被迫將[[星光行|星光行]]出售[[置地公司|置地公司]]。

另外，[[香港殖民地時期#香港政府|香港政府]]在1960年代末期，推出[[葵青貨櫃碼頭|葵涌一、二、三、四號四個貨櫃碼頭]]項目，[[霍英東|霍英東]]個人獨力投得最大的[[貨櫃碼頭一號碼頭|貨櫃碼頭一號碼頭]]，但港英政府卻无理要求霍擔保每年至少要有20萬個貨櫃在碼頭停泊，而當時全港的貨櫃箱總數都沒有20萬個，[[霍英東|霍英東]]找「船王」[[董浩雲|董浩雲]]（前任[[香港行政長官|香港行政長官]][[董建華|董建華]]父親）商量後，認為沒有把握，最後只好放棄。[[貨櫃碼頭一號碼頭|貨櫃碼頭一號碼頭]]最後由英資[[太古集團|太古集團]]的[[現代貨箱碼頭有限公司|現代貨箱碼頭有限公司]]承辦。

== 回中國大陸經商 ==
[[霍英東|霍英東]]會議上演講，自稱是「第一批響應返回祖國發展旅遊事業興建賓館」。1983年2月6日，考慮過[[北京|北京]]或[[上海|上海]]後，[[霍英東|霍英東]]的[[白天鵝賓館|白天鵝賓館]]正式在[[廣州|廣州]]開業，成為改革「樣板」，開業初造成轟動，連當時領導人[[鄧小平|鄧小平]]也被吸引。

有中國傳媒報道，[[鄧小平|鄧小平]]曾三度到訪白天鵝，1984年1月31日的行程包括參觀[[總統套房|總統套房]]，及到酒店內的[[絲綢之路|絲綢之路]]餐厅進餐；1985年的2月19日是農曆大[[除夕|除夕]]，[[鄧小平|鄧小平]]更在酒店內發表拜年講話；4日後（同年2月23日）鄧再到酒店，並自掏腰包買麵包，邊吃麵包邊說：等他回到上海時，就會宣佈再開放沿海14個城市。當時在場的[[霍英東|霍英東]]說，一個小小的麵包，就能使[[鄧小平|鄧小平]]有這麼大的啟發，促進中國[[改革開放|改革開放]]。

[[霍英東|霍英東]]更提出自行設計、施工及管理，當年投資額達5000萬美元，由霍的[[維昌發展有限公司|維昌發展有限公司]]與[[廣東省旅遊局|廣東省旅遊局]]簽訂協議，以中外合作的方式，經營期為15年，到97年再將協議延長至2003年。目前[[霍英東|霍英東]]集團涉足沙業、地產、科技園開發。

== 政界發展 ==
1980年8月25日政協第五屆全國委員會常務委員會中被選為[[全國政協委员|全國政協委员]]，1988年當選為第七屆[[全國人大常委會|全國人大常委會]]委員。

1989年[[六四事件|六四事件]]後第三天，他與另外19名港澳人士發表聯合公開聲明，譴責北京屠城、並哀悼死難者。

1993年3月27日政協第八屆全國委員會第一次會議當選為[[全國政協副主席|全國政協副主席]]，成為首位躋身[[黨和國家領導人|黨和國家領導人]]的香港人。

1996年及2000年兩次旗幟鮮明支持[[董建華|董建華]]任[[香港行政長官|香港行政長官]]，但隨著董建華下台，霍英東亦淡出政壇并因年紀問題，鮮有公開露面。

== 建造南沙港 ==
[[File:Humen_Bridge.JPG|thumb]]帶動[[南沙|南沙]]建設]]
[[霍英東|霍英東]]晚年大力投資故鄉[[番禺|番禺]][[南沙|南沙]]，昔日該處被譏為「番禺的[[西伯利亞|西伯利亞]]」，但[[霍英東|霍英東]]認為當地可連結西面[[江門|江門]]一帶，與中國政府商談合作，[[國務院|國務院]]1993年批准開發[[南沙經濟技術開發區|南沙經濟技術開發區]]，計劃耗資逾百億元。[[霍英東|霍英東]]當時與多個財團捐款建造[[虎門大橋|虎門大橋]]，使江門一帶往東的行程縮短近102公里，毋須繞道[[廣州|廣州]]，之後又興建客運[[碼頭|碼頭]]，使[[香港|香港]]至[[南沙|南沙]]的航程只需1小時15分鐘。

不過，[[南沙|南沙]]由荒蕪之地崛起，波折重重。當年「[[南沙68號|南沙68號]]」建成後，[[中華人民共和國|中華人民共和國]][[政府|政府]]拒絕發牌，並突然要求繳付逾百萬元的「堤圍費」，事件擾攘近一年，後來由他捐款1000萬元興建的[[洛溪大橋|洛溪大橋]]亦發生收費風波，當地政府聲稱無收過捐款，要向市民徵過路費償還銀行貸款，事件最後鬧大，番禺政府才肯讓步。

自1998年開始，[[霍英東|霍英東]]與[[香港科技大學|香港科技大學]]合作發展佔地250公頃的[[南沙資訊科技園|南沙資訊科技園]]，首期由[[霍英東|霍英東]]出資3億元發展，吸引多家大型公司如[[豐田|豐田]]車廠進駐。2005年[[霍英東|霍英東]]進一步向科大捐款8億元，當中5億元為配合[[南沙科技園|南沙科技園]]發展，科大已在[[南沙|南沙]]覓地開設教研院，取錄約600名研究生，加強科研工作<ref>[http://202.82.86.100/index.php?option=com_content&task=category&sectionid=5&id=93&Itemid=66| [[霍英東|霍英東]]開發[[南沙|南沙]]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }} 2004年8月23至25日亞洲時報</ref><ref>[http://www.nanfangdaily.com.cn/southnews/tszk/nfdsb/sd/200412210514.asp 霍英东　梁柏楠南沙恩怨再调查 2004年12月21日 南方都市報]</ref><ref>[http://mag.qq.com/mag/mag537_intro.shtml|霍英东遭受盘剥 2004年9月]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://news.sina.com.cn/c/2004-09-17/16504350324.shtml 第39期瞭望東方周刊]</ref>。

== 體育發展 ==
霍英東熱愛足球，早年組織「有榮小足球隊」及「卓然隊」，更親自落場在[[灣仔|灣仔]][[修頓球場|修頓球場]]與其他小型球隊比賽。及後更成立[[東昇足球隊|東昇足球隊]]加入[[香港甲組足球聯賽|香港甲組足球聯賽]]，亦曾長時間擔任香港左派球會[[愉園體育會|愉園體育會]]會長一職。1970年出任[[香港足球總會|香港足球總會]]會長，成為首位華人會長。霍英東更曾獲得兩屆全港網球雙打冠軍。
他也是[[香港象棋總會|香港象棋總會]]的榮譽會長。

1958年8月19日，[[中華人民共和國|中華人民共和國]]鑑於不少國際體育組織均視[[中華民國|中華民國]]為代表中國的合法政府，與[[國際奧委會|國際奧委會]]斷絕關係，同時北京方面也極力反對國際上將臺灣視爲獨立國家，繼而在其他國際單項體育組織失去會員席位。隨著中華人民共和國和其他西方國家的外交關係恢復正常化，霍英東在1970年代遊說中國大陸再踏國際體育的門檻，其後中華人民共和國取代中華民國並以中國的名義取得會席，並協助中國重返國際體壇。

1978年，他聯同其他亞洲和非洲國家成立世界羽聯，並以自己位於中環畢打街的公司作總部。世界羽聯和國際羽聯其後在日本談判，會前不眠不休地商討要點，霍英東亦參與談判。從1979年中國重返國際奧委會，至1984年由[[許海峰|許海峰]]擊落中國的[[奧運|奧運]]「第一金」，霍英東在1984年10月以一億港圓成立「霍英東體育基金會」，並以此名義捐贈金牌<ref>[http://news.xinhuanet.com/sports/2004-09/01/content_1935869.htm “霍英东体育基金会”千万港币重奖奥运健儿] {{webarchive|url=https://web.archive.org/web/20040904040836/http://news.xinhuanet.com/sports/2004-09/01/content_1935869.htm |date=2004-09-04 }}</ref>予中國奧運獲獎運動員、或興建不同的運動設施。其中最著名的是北京亞運村裡面的「英東游泳館」。1992年中國申辦[[2000年奧運会|2000年奧運会]]，但最後不敵悉尼，當時有人擔心霍英東會自殺。霍英东投资兴建的“[[英东游泳馆|英东游泳馆]]”在奥运会时是重要的比赛场馆，负责水球的全部比赛以及现代五项的游泳部分。

== 私人生活 ==
[[File:HK_StoneManor.jpg|thumb]]大宅「石頭莊園」]]
[[霍英東|霍英東]]有一妻（[[呂燕妮|呂燕妮]]）二妾（[[馮堅妮|馮堅妮]]) 三太太([[林淑端|林淑端]]），下有十子三女，其孫皆在[[英國|英國]]接受教育，[[霍英東|霍英東]]元配夫人為[[呂燕妮|呂燕妮]]，誕下三子[[霍震霆|霍震霆]]、[[霍震寰|霍震寰]]、[[霍震宇|霍震宇]]及三女[[霍麗萍|霍麗萍]]、[[霍麗娜|霍麗娜]]、[[霍麗勵|霍麗勵]]，其中霍震霆為[[港協暨奧委會|港協暨奧委會]]會長，次子[[霍震寰|霍震寰]]為[[香港中華總商會|香港中華總商會]]主席，幼子霍震宇主力打理南沙業務。[[霍震霆|霍震霆]]的太太為1977年[[香港小姐|香港小姐]]冠軍[[朱玲玲|朱玲玲]]，兩人現已離婚。

[[霍英東|霍英東]]的二太太[[馮堅妮|馮堅妮]]是他的初戀情人，經常陪伴霍老出席公開活動，二人育有[[霍文芳|霍文芳]]、[[霍文斌|霍文斌]]、[[霍文遜|霍文遜]]等三名兒子。霍、馮傳說是韓戰時的生意拍檔，1977年8月霍母病逝，[[霍英東|霍英東]]在報上的訃聞印有馮及其兒子們的名字，正式承認[[馮堅妮|馮堅妮]]的身分。[[霍文芳|霍文芳]]之前妻為[[新馬師曾|新馬師曾]]遺孀[[洪金梅|洪金梅]]之十一妹[[洪國華|洪國華]]。

[[霍英東|霍英東]]三太太[[林淑端|林淑端]]育有[[霍顯旋|霍顯旋]]、[[霍顯光|霍顯光]]、[[霍顯強|霍顯強]]、[[霍顯楊|霍顯楊]]四子，但為人低調，較少曝光，喜愛[[粵曲|粵曲]]，是[[紅線女|紅線女]]的歌迷。

霍家的生活往往成為傳媒追訪的新聞。其中[[霍震霆|霍震霆]]1978年9月迎娶港姐[[朱玲玲|朱玲玲]]成為當時轟動的新聞，但2000年以離異告終。[[霍英東|霍英東]]次子[[霍震寰|霍震寰]]太太則是藝人[[陳琪琪|陳琪琪]]，成為一時佳話。

2004年，[[霍啟剛|霍啟剛]]與[[郭晶晶|郭晶晶]]被傳媒拍下親密照；另一孫兒[[霍啟山|霍啟山]]亦在2004年被傳媒拍下在派對上與[[章子怡|章子怡]]跳舞。另一名孫兒[[霍啟中|霍啟中]]2004年因藏有[[大麻|大麻]]及吸食大麻用具，判罰款1500元及留案底，[[霍英東|霍英東]]當時指孫兒需要教訓，認為懲罰恰當。

== 患病 ==
早於1983年，霍英東曾證實患有[[淋巴瘤|淋巴癌]]，當時被估計只能撐3數年，因為這種癌症的特徵是每隔三、五、七年必定復發，但霍英東大病後十多年間，安然無恙，此後霍英東曾向[[壹週刊|壹週刊]]披露自創了一套叫「冰火浴」之稱的冷熱浸浴法，方法是在一個載滿冰塊，只有攝氏一、二度的冰水池中浸一分鐘，然後即刻轉到一個高溫[[桑拿|桑拿]]房焗一分鐘，又或者在熱水池浸一分鐘，如是者每天堅持來回冰水桶桑拿房，用以鍛鍊身體。這種獨門秘方，霍英東一用便持續了20年，可惜2003年癌病復發，在醫生指示下停止使用。

身兼[[全國政協副主席|全國政協副主席]]的霍英東，病逝前多次傳出重病消息。2006年7月21日，有傳媒<ref>霍英東癌復發 國醫搶救 東週刊 2006-07-26</ref>指霍英東癌症第二次復發，病情一度危殆，靠儀器維持生命；中央派出專機及醫療組來港，接他到[[北京301醫院|北京301醫院]]接受治療<ref>
[http://www.rthk.org.hk/rthk/news/expressnews/20061029/news_20061029_55_353208.html 霍英東病逝]香港電台即時新聞</ref><ref>[http://news.orisun.com/ncnews/hknews/ncbrka00_20061028235031.html 霍英東病逝]東方日報即時新聞</ref>。

但霍英東身為中共政要，其病为[[國家機密|國家機密]]，而且家人對病情三緘其口，因此外界一直所知不多；其後病情曾轉好，霍震寰更於8月15日表示病情無大礙，並快將回港。

== 逝世 ==
{{Wikinews|霍英東病逝 享年83歲}}
病逝前一週數度傳出[[彌留|彌留]]狀態；2006年10月28日19時30分於[[北京協和醫院|北京協和醫院]]病逝，10月31日，其子[[霍震霆|霍震霆]]护送遗体回[[香港|香港]]。11月7日，霍英东以[[党和国家领导人|国家领导人]]的最高规格举殡，是香港第二位榮受“[[國葬|國葬]]”儀式的人；遺體安葬在位於[[柴灣|柴灣]][[歌連臣角道|歌連臣角道]]的[[香港佛教聯合會|香港佛教聯合會]][[柴灣佛教墳場|柴灣佛教墳場]]。

新華社至10月29日下午5時許發表訃告，稱霍英東是「傑出的社會活動家，著名的愛國人士，香港知名實業家，[[中國共產黨|中國共產黨]]的親密朋友，[[中國人民政治協商會議|中國人民政治協商會議]]第八、九、十屆全國委員會副主席，[[香港中華總商會|香港中華總商會]]永遠名譽會長」。

== 身后评价 ==
大部分對霍英東的正面評價都是基於他親[[中共|中共]]的政治立場和在大陸[[改革開放|改革開放]]時期對中國的投資，例如：
*[[香港特别行政区行政长官|香港特别行政区行政长官]]曾荫权表示，霍英东在国家的经济改革开放过程中贡献宏大，又竭力促进香港的繁荣安定，他的逝世是国家及香港的重大损失。
*[[香港民政事务局|香港民政事务局]]局长何志平亦感到非常惋惜和十分悲痛，他指出，霍英东十分关注香港[[体育|体育]]运动的推广，并全力支持祖国的文教康体事业。何志平说，霍英东为人十分低调谦虚，生活简朴，十分值得尊敬。
*[[新华社|新华社]]发表通告称，霍英东是“杰出的社会活动家，著名的爱国人士，香港知名实业家，中国共产党的亲密朋友”。
*[[全国人大常委会委员|全国人大常委会委员]][[曾宪梓|曾宪梓]]表示，早上获悉病逝消息，感到非常悲痛。曾形容霍英东是自己的恩师亦是好友。并称赞霍是爱国商人，长期推动国家体育事业发展，为人诚恳，平易近人。
*[[香港中文大學|香港中文大學]]於2006年12月7日舉行的第63屆學位頒授典禮上，向霍英東追頒榮譽法學博士學位，而霍氏生前也表示願意接受榮譽學位。
*2007年2月25日，霍獲選[[中央電視台|中央電視台]]「感动中国2006年度人物」榮譽稱號。颁奖典礼在翌日晚上八點至十點在央視一台播出。
*在足球發展方面，霍英東在1980年代為了扶植左派球隊，打擊其他球隊的發展，特意推行「全華班」政策（即是球隊員只能夠有華人球員）。實施的三個賽季期間，入場人數應聲下挫，香港足球的發展亦從此一蹶不振。霍英東可謂香港足球發展的罪人。
