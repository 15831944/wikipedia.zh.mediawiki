'''霍亨索伦城堡'''（Burg Hohenzollern）是[[德国|德国]]南部最著名的两大[[城堡|城堡]]之一，与[[巴伐利亚州|巴伐利亚州]]的[[新天鹅堡|新天鹅堡]]齐名于世。

== 位置 ==
位于[[图宾根|图宾根]]南方20公里的一座丘陵上，所在的小城市叫[[黑欣根|黑欣根]]，每天都有附近城市开往这个小城市的班车。从图宾根坐区间车20多分钟就到了，大约一个小时一趟；然后再做专车去城堡，一天一个来回，一般分别为上午的十一点和下午四点来回，周末有稍微调整。要想从火车站坐免费班车到城堡需准确把握时间。

== 发源 ==
古堡是[[霍亨索伦家族|霍亨索伦家族]]的发源地（霍亨索伦家族是[[布蘭登堡-普魯士|布蘭登堡-普魯士]]1415～1918年及[[德意志帝国|德意志帝国]]的主要统治家族）。霍亨索伦堡建于11世纪，但是老城堡后来被毁坏。在1819年，当时的王子威廉（Friedrich Wilhelm）决定重建这座祖居城堡。1844年，已经继承了皇位的[[腓特烈·威廉四世|威廉四世]]（Friedrich Wilhelm IV）在一封信中提到：“1819年在这座城堡里时留下的记忆是如此美妙，尤其是每当日落的时候向外观望……宛如一场愉悦的美梦。如今这个年轻时的梦想已成为重新将城堡变成家园的愿望。”现今的霍亨索伦堡是在1850年至1867年，由普鲁士建筑家Von Prittwitz和Stuler共同改建的，与新天鹅堡的兴建大约是同一时期，但是与新天鹅堡宛如童话般的外貌相比，霍亨索伦堡就似充满英雄主义的阳刚之气，这也正代表普鲁士王朝的辉煌历史。<ref>[http://www.burg-hohenzollern.com/Welcome.html]，官网。</ref>

== 腓特烈大帝雕像 ==
在城堡的边缘可以看到霍亨索伦家族著名国王或皇帝的巨大雕像，尽显普鲁士的威严。
其中就有[[腓特烈大帝|腓特烈大帝]]的雕像，从雕刻角度讲，和其他帝王没有太大区别，但是这座雕像还是很好的展示了腓特烈大帝的风尚。经过他的改革，他治下的普鲁士跃居欧陆豪强之列，也让霍亨索伦家族有了对抗[[奥地利|奥地利]][[哈布斯堡家族|哈布斯堡家族]]的资本，为此后普鲁士一统德国奠定了基础。他成为在[[查理大帝|查理大帝]]和[[奥托大帝|奥托大帝]]之后又一个享有“大帝”称号的普魯士国王，只可惜虽然孜孜不倦地与[[特雷西亚女皇|特雷西亚女皇]]争夺霸权，但他只是在东部攻城掠地、始终没有当上[[德意志民族|德意志民族]]的[[神圣罗马帝国|神圣罗马帝国]]的皇帝。

== 现状 ==
1918年[[普魯士王國|普魯士王國]]和[[德意志第二帝国|德意志第二帝国]]灭亡后，[[威廉二世_(德國)|威廉二世]]逃往[[荷兰|荷兰]]避难。而其他大多数皇室成员继续留在德国生活。现在的城堡仍是普鲁士王朝末代皇帝/國王威廉二世的子孙留居此地，城堡的意义在于，它是普鲁士霍亨索伦家族聚会的地方，当然，它也是博物馆，城堡部分对外收费开放。另外，霍亨索伦城堡是德国[[Naldo交通体系|Naldo交通体系]]里最著名的旅游景点。
