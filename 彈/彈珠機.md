'''弹珠机'''是一種具[[娛樂|娛樂]]與[[賭博|賭博]]成份的機器，在[[日本|日本]]很常見，在[[台灣|台灣]]稱之為'''-{柏青哥}-'''，中國大陆称为'''扒金宫'''、'''爬金库'''（取日語「Pa-chin-ko」音譯）或'''小鋼珠'''。弹珠机本為純機械式操作，但現代的弹珠机加入投幣式自動賭博機（slot machine）元素。弹珠机店遍佈日本全國各地，受到不少人歡迎；但因為有[[非法|非法]]賭博成份，弹珠机難以避免地與[[組織犯罪|組織犯罪]]扯上關係。

[[File:Pachinko_Datiokai_in_Nagoya_2017.jpg|缩略图]]的弹珠机店]]
== 歷史 ==
弹珠机在[[20世纪|20世纪]]初期於[[名古屋市|名古屋市]]發明，起源於[[大正時代|大正時代]]有獎品的投幣式遊戲機，當時為供[[兒童|兒童]]遊玩的遊戲機。1942年曾被禁止，令弹珠机店需要關閉。[[第二次世界大戰|第二次世界大戰]]後，1946年獲得解禁。

== 玩法規則 ==
而在[[中国大陸|中国大陸]]由於除了極少數特區外不允許[[賭博|賭博]]，引入后则主要演化成了仅供[[儿童|儿童]]玩耍的儿童式弹珠机，玩者投币一元[[人民幣|人民幣]]，机器会出来五颗（可以自行调节）玻璃珠，把玻璃珠（一次可放1-10）颗放进投珠口中，位于正平面上的倍数灯（2、3、4、5）与钉子板底部上的12个通道灯就会开始闪耀；此时按下启动按钮，机器将随机确定倍数与通道灯，拉右边拉杆，把玻璃面板内的大塑料母球弹入通道后进入钉子板，若最后通过先前已亮着红灯的通道，即可获得其相应倍数的玻璃珠，中奖所得的玻璃珠可以重复使用。<ref name="Reed about">{{cite web |url= http://faculty.ccp.edu/faculty/dreed/Campingart/pachinko/about.htm |title=A Little About the Machines |author=Reed, D. J. |work=Dan's Pachinko Data Page |accessdate=9 November 2009}}</ref> 

[[File:Medal_Pachinko.jpg|缩略图]]
在[[臺灣|臺灣]]的柏青哥經營與[[日本|日本]]大致相同，唯臺灣機台並不像日本有專門機構（[[厚生勞動省|厚生勞動省]]）驗證，將原本三齒的彈珠彈射器改為四齒，故每分鐘使用鋼珠數比日本多了33%，加上多數商家會更改日本原始程式降低中獎機率，所以在臺灣柏青哥[[期望值|期望值]]過低變成非常具有「娛樂性」的活動，<ref name="vintagepachinko1">{{cite web|url=http://www.vintagepachinko.com/information/vintage-vs-modern-pachinko.htm |title=Modern vs Vintage Pachinko Machines |publisher=Vintagepachinko.com |accessdate=9 November 2009}}</ref>並不如日本賭博意謂濃厚。在臺灣由手動以[[彈簧|彈簧]]裝置發射的稱為'''[[彈珠台|彈珠台]]'''，由[[機械|機械]]發射的才稱為'''-{柏青哥}-'''，另有以鋼珠代替[[硬幣|硬幣]]投入的[[小瑪莉機台|小瑪莉機台]]。放在-{柏青哥}-遊樂場的[[角子機|吃角子老虎]]，則稱為'''-{柏青嫂}-'''。
