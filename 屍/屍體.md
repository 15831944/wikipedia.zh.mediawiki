'''屍體'''，是不同于[[生物|生物]]的概念，指生物体[[死亡|死亡]]後遺留的尚未完全[[腐烂|腐烂]]的躯体。至於完全腐壞、僅殘餘[[骨骼|骨骼]]組織的动物尸体则称为[[骸骨|骸骨]]，亦称白骨、骷髅、遗骨等，其中“骷髏”一詞可能帶有恐懼吓唬人的意味。

== 概要 ==
一般来说，「遗体」用于尊重死者人格的场合。在台灣，某些宗教人士與多數媒體亦將遺體稱作'''大體'''，然而'''大體'''一詞是由「[[大體解剖學|大體解剖學]]」一詞而來的[[簡稱|簡稱]]，這裡的大體是代表「巨觀的」、「總成的」，是相較於「[[微觀解剖學|微觀解剖學]]」、「[[神經解剖學|神經解剖學]]」等處理肉眼可看見的[[器官|器官]][[组织_(生物学)|組織]]的解剖學，上述將「遺體」稱作「大體」，把「[[屍皮|屍皮]]」稱為「大體皮膚」乃為誤用<ref>{{cite web |url=http://www.tsubasa.com.tw/mt/archives/2009/08/post-806.html |title=存档副本 |accessdate=2012-04-24 |0=2010-05-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20100502005054/http://www.tsubasa.com.tw/mt/archives/2009/08/post-806.html |archivedate=2010-05-02 }}</ref>。而「尸体」用于[[医学|医学]]、[[刑事案件偵查|刑事案件偵查]]等领域。在[[地质学|地质学]]和[[生态学|生态学]]中，通常使用“遗体”一词，如「生物遺体」、「動物遺体」、「植物遺体」等。尸体会发生一系列[[尸体现象|尸体现象]]，是刑事案件偵查時推測死亡時間的參考依據。

== 古代記載 ==
在古籍《酉陽雜俎·物異》《尚書故實》《南村輟耕錄·卷十四》中，「人臘」是指乾枯的人的屍體。
