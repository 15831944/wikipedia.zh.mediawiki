{{作品名稱|name=''Swiss Army Man''|hk=救你命3000|tw=屍控奇幻旅程|cn=瑞士军刀男}}是一部2016年美國[[喜劇片|喜劇]][[劇情片|劇情片]]，由丹尼爾·舒奈特與丹尼爾·關執導和編劇。電影主演包括[[保羅·迪諾|保羅·迪諾]]、[[丹尼爾·雷德克里夫|丹尼爾·雷德克里夫]]和[[瑪麗·伊莉莎白·文斯蒂德|瑪麗·伊莉莎白·文斯蒂德]]。《屍控奇幻旅程》最初於2016年1月22日在[[日舞影展|日舞影展]]上首映，並由[[A24_(公司)|A24]]負責發行，定於2016年6月24日在美國上映。

==劇情==
故事敘述被困在一座荒島上的漢克（[[保羅·迪諾|保羅·迪諾]] 飾）在打算自殺時，看到一具男性屍體（[[丹尼爾·雷德克里夫|丹尼爾·雷德克里夫]] 飾）被沖到了岸上；漢克很快地發現，屍體竟會像活人般活動和說話，並得知他的名字是曼尼。在經過相處下，漢克和曼尼發展出超現實的友誼，並讓漢克燃起求生意志。

==製作==
2015年6月29日，[[保羅·迪諾|保羅·迪諾]]、[[丹尼爾·雷德克里夫|丹尼爾·雷德克里夫]]和[[瑪麗·伊莉莎白·文斯蒂德|瑪麗·伊莉莎白·文斯蒂德]]加入電影擔任主演<ref name="deadline1"/>。正式拍攝於2015年7月14日開始進行，直至2015年8月7日結束<ref>{{cite web |url=http://www.ssninsider.com/on-the-set-for-72015-daniel-radcliffe-starts-shooting-swiss-army-men-meryl-streep-finishes-florence-foster-jenkins/ |title=On the Set for 7/20/15: Daniel Radcliffe Starts Shooting ‘Swiss Army Men’, Meryl Streep Finishes ‘Florence Foster Jenkins’ |publisher=SSN Insider |date= |accessdate=2015-10-26 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url=http://www.ssninsider.com/on-the-set-for-73115-chris-pratt-wraps-magnificent-seven-dwayne-johnson-completes-central-intelligence-chris-hemsworth-finishes-the-huntsman/ |title=On the Set for 7/31/15: Chris Pratt Wraps ‘Magnificent Seven’, Dwayne Johnson Completes ‘Central Intelligence’, Chris Hemsworth Finishes ‘The Huntsman’ |publisher=SSN Insider |date= |accessdate=2015-10-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20151002222617/http://www.ssninsider.com/on-the-set-for-73115-chris-pratt-wraps-magnificent-seven-dwayne-johnson-completes-central-intelligence-chris-hemsworth-finishes-the-huntsman/ |archivedate=2015-10-02 }}</ref>。

==發行==
電影於2016年1月22日在[[日舞影展|日舞影展]]上首映<ref name="thr-sam">{{cite web|url=http://www.hollywoodreporter.com/review/swiss-army-man-sundance-review-858537 |title='Swiss Army Man': Sundance Review |work=The Hollywood Reporter |date=January 22, 2016 |accessdate=2016-01-24}}</ref>。2016年1月30日，該片被[[A24_(公司)|A24]]收購獲得在美國的發行權<ref name="Brent Lang">{{cite web|url=http://variety.com/2016/film/news/a24-daniel-radcliffe-swiss-army-man-sundance-1201692377/ |title=A24 Lands Daniel Radcliffe Corpse Movie ‘Swiss Army Man’ (EXCLUSIVE) |work=Variety |date=January 29, 2016 |accessdate=2016-01-29}}</ref>。原定於2016年6月17日在美國上映<ref>{{cite web|author=Maane Khatchatourian |url=http://variety.com/2016/film/news/daniel-radcliffe-farting-corpse-swiss-army-man-trailer-1201744802/ |title=‘Swiss Army Man’ Trailer: Daniel Radcliffe Is a Farting Corpse |publisher=Variety |date=2016-04-04 |accessdate=2016-04-08}}</ref>，但一個星期後改至6月24日<ref>{{cite web|url=http://www.boxofficemojo.com/movies/?id=swissarmyman.htm|title=Swiss Army Man|website=[[Box_Office_Mojo|Box Office Mojo]]|accessdate=April 22, 2016}}</ref>。A24還收購了全球的發行權，並獲得了國外的合作電影發行商<ref>{{cite web|url=http://deadline.com/2016/04/swiss-army-man-worldwide-distribution-a24-sundance-daniel-radcliffe-1201732184/|title=A24 To Handle ‘Swiss Army Man’ Global Release In Company’s First World Rights Move; New U.S. Release Date – Update|website=[[Deadline.com|Deadline.com]]|first=Nancy|last=Tartaglione|date=April 5, 2016|accessdate=April 22, 2016}}</ref>。

==反響==
電影在[[日舞影展|日舞影展]]上首映時獲得褒貶不一的評價，甚至許多觀眾提前離席<ref name="variety-walkouts">{{cite web|title=Sundance: Daniel Radcliffe's Farting Corpse Movie Prompts Walk-Outs|url=http://variety.com/2016/film/news/daniel-radcliffe-farting-corpse-swiss-army-man-1201686756/|website=Variety|accessdate=25 January 2016}}</ref><ref>{{cite web|url=http://www.ew.com/article/2016/06/21/daniel-radcliffe-swiss-army-man-sundance-walkouts|title=Daniel Radcliffe was surprised by Swiss Army Man walkouts at Sundance|website=[[Entertainment_Weekly|Entertainment Weekly]]|first=Clark|last=Collis|date=January 21, 2016|accessdate=June 29, 2016}}</ref>。[[爛番茄|爛番茄]]新鮮度67%，平均得分6.5／10<ref>{{cite web |url=http://rottentomatoes.com/m/swiss_army_man/ |title=Swiss Army Man (2016) |publisher=[[Rotten_Tomatoes|Rotten Tomatoes]] |accessdate=2016-06-27}}</ref>。[[Metacritic|Metacritic]]得分64，正評居多<ref>{{cite web|url= http://www.metacritic.com/movie/swiss-army-man/|title=Swiss Army Man|publisher=[[Metacritic|Metacritic]]|accessdate=September 26, 2016}}</ref>。
