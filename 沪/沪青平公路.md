'''沪青平公路'''是[[上海市|上海]]经过青浦、浙江[[平湖市|平湖]]到[[吴江区|吴江]]平望的公路，是[[318国道|318国道]]（到西藏樟木）上海到吴江一段。

这条公路自上海市区人民广场至江苏省吴江县平望镇。全长84公里，是上海市通往苏、浙、皖三省的主要干线之一。路基宽12米左右，路面宽7米，全线桥梁53座，桥梁总长1617.56米<ref name="dfz" />。从民国21年(1932年)起，至1965年分四次筑成。

因解放前该公路只从上海至青浦，故习惯上也称青沪公路。老青沪公路虹桥机场新村—虹桥路段是[[闵行区|闵行区]]、[[长宁区|长宁区]]的边界线。

== 历史 ==
民国21年(1932年)，青浦县计划修筑上海至朱家角的沪朱公路，因财力不足，改筑青浦至上海虹桥的青沪路。民国25年筑成通车，为县内第一条公路<ref name="dfz">{{cite web|title=青浦县志 >> 第十二篇交通邮电 >> 公路|url=http://www.shtong.gov.cn/newsite/node2/node4/node2250/node4427/node5574/node10643/index.html|website=上海通网站|publisher=上海市地方志办公室|accessdate=2017-08-03}}</ref>。

翌年，抗日战争爆发。为防止日军西进，民国27年（1938年）8月20日，共产党游击队发动了“火烧青沪路桥”战役，桥梁有十一座被毁<ref>{{cite web|title=青浦县志 >> 第二十一篇军事 >> 战事|url=http://www.shtong.gov.cn/newsite/node2/node4/node2250/node4427/node5586/node11007/node11030/userobject1ai18747.html|website=上海通网站|publisher=上海市地方志办公室}}</ref>。由于战争因素，另有部分路基被毁，公路未能使用。中共建政后，人民政府首先修复青沪公路，后延伸至朱家角、淀峰和江苏平望镇<ref name="dfz" />。

与上海其他的公路一样，沪青平公路修筑的早年，是在水乡田埂土路的基础上连接起来的。沪青平公路上有许多座桥，从[[程家桥|程家桥]]的新泾港桥开始编号命名，直至青浦的西横头。

1987年青浦镇至徐泾段拓宽为四车道。1990年代上海市大规模修整沪青平公路，其成为标准化的水泥马路。

[[沪青平高速公路|沪青平高速公路]]位于沪青平公路的南侧，大致与其平行。
