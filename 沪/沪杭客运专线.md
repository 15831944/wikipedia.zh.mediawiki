'''沪杭客运专线'''，也称'''沪杭高速铁路'''，为[[中国|中国]]一條連接[[上海市|上海市]]与[[浙江省|浙江省]][[杭州市|杭州市]]的铁路[[客运专线|客运专线]]，是中国[[“四纵四横”铁路快速客运通道|“四纵四横”铁路快速客运通道]]网络中[[滬昆客運專線|滬昆客運專線]]的组成部分。全线设计时速为350公里。工程于2009年2月26日动工兴建，于2010年10月26日通车营运。

== 規劃及建設 ==
[[File:High-speed_rail_line_under_construction_in_Hangzhou_2010.jpg|thumb]]上拍摄建设中的沪杭高铁]]
根據[[铁道第四勘察设计院|铁道第四勘察设计院]]于2008年11月发布的资料<ref name=r1>{{cite web |url=http://www.crfsdi.com/hg/hh.htm |title=存档副本 |accessdate=2009-03-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20090107060821/http://www.crfsdi.com/hg/hh.htm |archivedate=2009-01-07 }} 《新建铁路沪杭甬客运专线上海至杭州段环境影响报告书（简本）》</ref>，沪杭客运专线由[[上海虹桥站|上海虹桥站]]至[[杭州东站|杭州东站]]，中途設[[松江南站|松江南]]、[[金山北站|金山北]]、[[嘉善南站|嘉善南]]、[[嘉兴南站|嘉兴南]]、[[桐乡站|桐乡]]、[[海宁西站|海宁西]]、[[余杭站|余杭]]等站，投资总额估算為255亿元[[人民幣|人民幣]]。

2009年2月19日，铁道部落实建设方案，全线设置[[虹桥站|虹桥站]]、[[松江南站|松江南站]]、[[金山北站|金山北站]]、[[嘉善南站|嘉善南站]]、[[嘉兴南站|嘉兴南站]]、[[桐乡站|桐乡站]]、[[海宁西站|海宁西站]]、[[余杭站|余杭站]]和[[杭州东站|杭州东站]]共9个站点，此外在[[春申三角区|春申三角区]]和[[杭州东站|杭州东站枢纽]]附近分别引出联络线至[[上海南站_(铁路)|上海南站]]和[[杭州站|杭州站]]<ref>[http://funds.hexun.com/2009-02-26/115003539.html 沪杭高铁今开工]</ref><ref>[http://politics.people.com.cn/BIG5/14562/8869155.html 滬杭城際客運專線明天開工]</ref>。

2009年2月26日在上海金山枫泾镇正式奠基<ref>[http://cpc.people.com.cn/GB/64093/117005/8882498.html 沪杭客运专线建设启动 俞正声出席 赵洪祝讲话]</ref>，根据报道，沪杭客运专线设计时速350公里，预计2010年[[上海世博会|上海世博会]]开幕前建成运营<ref>{{cite web | title = 沪杭高速铁路建设首战告捷 2010年底建成| publisher = 东方网 | date = 2009-07-24 | url = http://sh.eastday.com/qtmt/20090724/u1a605171.html | accessdate = 2009-07-24 | language = zh }}</ref>。预测运量至2020年為136对/日，2030年為182对/日<ref name=r1/>。

2009年11月18日，杭州市[[余杭区|余杭区]]附近在建的沪杭客运专线海杭特大桥桥墩模板突然倒塌，造成1人死亡，5人受伤。<ref>[http://www.chinanews.com.cn/sh/news/2009/11-19/1973605.shtml 沪杭客运专线在建桥墩模板发生倒塌 造成1死5伤]</ref>

2010年1月21日，沪杭客运专线嘉兴大桥段工地发生脚手架倒塌事故，造成1人死亡，2人受伤。<ref>[http://news.sina.com.cn/c/2010-01-22/080516974729s.shtml 沪杭高铁工地坍塌1人死]</ref>

2010年9月28日，[[CRH380A|CRH380A]]在沪杭客运专线进行高速试验，并先后两次刷新“中国铁路第一速”，并刷新了当今世界上正常营运编组列车最高试验速度。当日上午10时40分，一列由上海[[上海虹桥站|虹桥]]开往杭州方向的CRH380A列车的试运行时速达到413.7公里/小时<ref>{{Cite web |url=http://news.xinhuanet.com/politics/2010-09/28/c_12615068.htm |title=沪杭高铁试运行时速达到413.7公里 再次刷新世界纪录 |accessdate=2010-09-28 |publisher=新华网 |date=2010-09-28 }}</ref>。不到一小时后的11时37分，由杭州折返到上海虹桥试运行途中，最高时速达到416.6公里/小时<ref>{{Cite web |url=http://news.qq.com/a/20100928/001085.htm |title=沪杭高铁试运行 时速416.6公里全程仅40分钟 |accessdate=2010-09-28 |publisher=腾讯新闻网 |date=2010-09-28 }}</ref>。

===初期列車開行方案===
2010年10月21日，上海铁路局正式发布沪杭高铁列车开行方案。沪杭高铁共开行80对动车组列车，其中（G字頭）高速动车组列车50对，（D字頭）动车组30对。

50对高速动车组列车中，杭州至上海虹桥间40对，每整点开行直达列车，每日15对，用时45分，其余时刻为非直达列车，耗时从52分到60分不等；杭州上海间5对，其中直达列车一对，耗时70分，其余4对为停一站或两站，耗时从1小时17分到1小时26分不等；杭州南京间5对，无直达列车，耗时从2小时45分到3小时09分不等。此外，所有上海（南）站始发经沪杭铁路至杭州以远的原有或新开动车组列车全部移至[[上海虹桥站|上海虹桥站]]始发并经沪杭高铁跨线运行，由于里程较既有线短，因此票价有所降低。<ref>《[http://news.xinhuanet.com/politics/2010-10/21/c_12686494.htm 沪杭高铁开车方案及票价公布]》(新華網，2010年10月21日)</ref>

===首发仪式===
铁道部门在2010年10月26日上午在上海虹桥站及杭州站同步举行「沪杭高速铁路通车典礼」，并在上海虹桥站及杭州站对开列车，分别为：
* G5001次：上海虹桥开09:00，到杭州09:39；
* G5002次：杭州开09:00，到上海虹桥09:40； 
* G5003次：上海虹桥开09:20，到杭州10:05。

在首发列车开行期间，当地电视台安排一架直升飞机从嘉善前往杭州，与一辆从上海虹桥站开往杭州站的G5001次首趟高铁列车进行竞速，两者在9时15分在嘉善南站相遇；然而高铁列车迅速超越了直升机，高铁列车在9时39分到达杭州站，而直升机于10时05分抵达杭州站上空。

===票價===
按當局公佈，途經「沪杭客专」的（G字頭）高速动车组票價如下<ref>《[http://www.shrail.com/3576.htm 沪杭高铁开车方案及票价公布　上海虹桥至杭州最快45分钟 22日起开始售票]》(上海鐵路局，2010年10月21日)</ref>：
* 杭州↔上海虹桥：一等座123.5元，二等座77.5元；
* 杭州↔上海：一等座147.5元，二等座92.5元；
* 杭州东↔上海虹桥：一等座117元，二等座73元。

===降速运行===
受中国铁路第二次全面降速的影响，上海铁路局重新公布了沪杭高铁列车运行图，沪杭高铁自2011年8月28日进行的运行图第二阶段调整起，最高营运速度降低到300km/h。运行图调整后，开行83对动车组列车，其中33对时速300km本线动车组列车（杭州-上海虹桥），15对时速300km跨线动车组列车（杭州-上海4对，杭州-南京5对，杭州-北京南4对，杭州-济南西、杭州-天津西各一对），35对时速200km动车组列车。票价有所降低，G字头列车本线78元，跨线76元；D字头列车杭州站发49元，杭州南站发54元。

===最短运行时间===
目前{{站|杭州东}}-{{站|上海虹桥}}区间最短运行时间为45分钟，中途不停靠任何站点，车次为[[沪广高速动车组列车|G1306次]]、[[沪温高速动车组列车|G7536次列车]]（上行）；[[沪广高速动车组列车|G85次列车]]（下行）。

同时部分列车沪杭区间亦是一站直达，运行时间在45至50分钟之间，车次分别是[[沪长高速动车组列车|G1348次]]、[[沪昆高速动车组列车|G1372次]]、[[沪昆高速动车组列车|G1374次]]、[[沪贵高速动车组列车|G1330次]]、G1660次、[[沪广高速动车组列车|G86次]]、[[沪温高速动车组列车|G7536次]]（上行）；G1653次、[[沪昆高速动车组列车|G1373次]]和[[沪广高速动车组列车|G1305次列车]]（下行）。
