'''沪港關係'''，是指中國[[上海市|上海市]]与[[香港特别行政区|香港特别行政区]]之间的关系。香港和上海有著密切的關係，且相互影響。香港早期的經濟發展，很大部分是由南遷香港的上海商人帶動；到了今天，許多港商亦有在上海投資<ref>{{cite web |author = tse york ting |title=上海考察後觀感 |url = https://www.cb.cityu.edu.hk/ef/learning/Study_Tours/Shanghai%20Study%20Tour%2001-02/Remarks/Tse%20York%20Ting2.htm |website=成大商学院 |accessdate=2016-12-19 }}</ref>。1990年，[[朱鎔基|朱鎔基]]访问香港时曾说：“香港之所長，是上海之所短；上海之所長，是香港之所需。”滬港兩地制度不同，基礎不同，經濟腹地不同，運作方式也不一樣，各有特點和長處。香港是亚洲区域性的金融中心，而上海现时还是以国内市场为主的金融中心<ref>{{cite book |author=任慧文 |title=中南海跨世纪领导层 |year=1998 |publisher=太平洋世紀出版社 |page=第328页 |url = https://books.google.de/books?id=XOlfAAAAIAAJ&q |accessdate=2016-12-19 }}</ref>。

时任中共中央政治局委员、上海市委书记[[黄菊|黄菊]]在2001年接受[[香港文汇报|香港文汇报]]采访时指出，香港不用担心被上海取代；沪港两地的关系是“优势互补，互惠互利，共同繁荣”；在金融、航运、服务、管理，以及高层次专业人才等方面，香港都较上海更具优势<ref>{{cite news |title=黄菊：沪港优势互补 香港不用担心被上海取代 |url = http://look.people.com.cn/GB/channel426/427/1504/200109/20/107743.html |website=[[人民网|人民网]] |agency = [[中国新闻社|中国新闻社]] |accessdate=2016-12-19 |date=2001年9月20日 }}</ref>。

目前，香港特別行政區在上海設有駐上海經濟貿易辦事處，負責對上海和周邊省市的聯絡溝通等職務。<ref>https://www.sheto.gov.hk/tc/home/index.html</ref>

==经济==
{{Main|沪港通}}

沪港两地的金融业早在20世纪30年代就有很多往来，当时上海是远东最大的金融中心。1949年中华人民共和国成立后，先后关闭了上海各类金融市场，与此同时香港则承接了一批来自上海的资金、银行家及金融机构。中国改革开放后，香港工商界积极返回上海投资，促进了两地经济与金融业的往来。<ref>{{cite web |author = 谢国梁 |title=CEPA 框架下的沪港金融关系 |url = http://60.16.24.71/KCMS/detail/detail.aspx?filename=N2006090406000113&dbcode=CYFD&dbname=CYFD |website=中国知网 |publisher=港澳经济年鉴社 |accessdate = 2016-12-19 |year = 2005 }}</ref><ref>{{cite book |author1=林华生 |author2=黄枝连 |title = 剖析东亚经济: 中华经济协作系统第十届国际研讨会论文集 |publisher=World Scientific |page=345页 |url = https://books.google.de/books?id=tjNhpcrTdwYC&pg }}</ref>

2014年11月17日，[[沪港通|沪港通]]正式開通，使[[中國内地|中國内地]]和[[香港|香港]]投资者允许通过当地证券公司或者经纪商买卖规定范围内的对方交易所上市的股票。<ref name="沪港通介绍">[http://edu.sse.com.cn/col/shhkconnect/intro/intro/ 上交所：沪港通介绍]</ref>2018年4月11日，滬港通每日交易額度計劃將擴大4倍。<ref>{{Cite web|url=https://cn.wsj.com/articles/CT-MKT-20180411104542|title=滬港通、深港通南北向每日額度將擴大四倍|website=華爾街日報|access-date=2018-04-11}}</ref><ref>{{Cite web|url=https://www.881903.com/Page/ZH-TW/newsdetail.aspx?ItemId=1002395&csid=261_341|title=金管局歡迎擴大滬港通及深港通每日額度|website=商業電台網站|access-date=2018-04-11}}</ref>
