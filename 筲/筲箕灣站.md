'''筲箕灣站'''（{{lang-en|'''Shau Kei Wan Station'''}}）是[[港鐵|港鐵]][[港島綫|港島綫]]的一個[[鐵路車站|鐵路車站]]，位於[[香港島|香港島]][[東區_(香港)|東區]][[筲箕灣|筲箕灣]]近[[東區走廊|東區走廊]]的地底，並於1985年5月31日啟用。

==車站設計==
筲箕灣站是一個2層深的地底車站，大堂層為售票大堂、商舖及車站控制室；下層為2個月台、機房及員工設施；上方為東區走廊及峻峰花園；而大堂和月台也有結構為東區走廊及峻峰花園提供支承，而站體位於彎曲的道路一方，所以設計都是彎曲的。

===出入口===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Shau Kei Wan Station Underground Access 2018.jpg | caption1 = B及D出入口以24小時行人隧道形式，橫跨東喜道地底
| image2 = Shau Kei Wan Station 2018 05 part6.jpg | caption2 = 地面升降機及B1出入口
}}
B、D出入口以24小時行人隧道形式，橫跨東喜道地底，在車站服務時間以外連接大堂和24小時行人隧道的通道，會落下捲閘分隔。
使用輪椅的乘客可以使用B1出口的升降機，或經由連接大堂和D出口行人隧道的斜道，再經行人隧道各個斜道出口前往地面。

==利用狀況==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Shau Kei Wan Station 2014 04 part6.JPG | caption1 = 月台往大堂通道
}}
港島綫通車初期，因當時筲箕灣站不鄰近筲箕灣的主要地區，因此客量偏低；直至該站旁的新屋苑-{入伙}-，加上[[中華汽車有限公司|中巴]]以及[[專線小巴|小巴]]服務質素欠佳，因此筲箕灣居民逐漸改用[[香港地鐵|地鐵]]（現稱港鐵）進出市區，而且暑假有不少市民利用該站轉乘[[香港巴士|巴士]]或[[專線小巴|小巴]]前往[[石澳|石澳]]泳灘，客量一直保持平穩。直至2000年，愛秩序灣新填海區的新屋苑陸續-{入伙}-，加上[[東區區議會|東區區議會]]致力推廣[[東區旅遊徑|東區旅遊徑]]及[[香港海防博物館|香港海防博物館]]等旅遊景點，使車站的人流不斷增加。

==歷史==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Shau Kei Wan Station.jpg | caption1 = 兩鐵合併初期的月台（2008年）
}}
筲箕灣站於1985年5月31日隨著港島綫第一期啟用。

===車站外升降機加建工程===
筲箕灣站的B及D出口是連接行人隧道的出口，但行人隧道僅以斜道連接地面，使用上並不方便，故港鐵公司於2014年4月展開工程，於B1出口增設一部升降機，方便有需要的乘客以新的無障礙通道往返車站<ref>[https://www.mtr.com.hk/ch/corporate/projects/projects_improvement_lift_skw.html 鐵路網絡改善工程　車站外升降機加建工程]，[[港鐵公司|港鐵公司]]</ref>。

這項工程已於2016年完成，升降機並於同年12月21日開放予公眾使用<ref>[http://www.mtr.com.hk/archive/corporate/en/press_release/PR-16-130-C.pdf 港鐵筲箕灣站新站外升降機令出入更方便]，[[港鐵公司|港鐵公司]]，2016年12月21日
</ref>。
