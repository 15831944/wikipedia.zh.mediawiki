'''筲箕灣福德廟'''<ref name="華人廟宇委員會"/>是一座位於[[香港|香港]][[筲箕灣|筲箕灣]]愛秩序村4227號地段的廟宇群。廟宇群內分別建有福德廟、觀音廟、關帝廟、洪聖古廟、張飛廟和劉備廟<ref name="灼見名家@20180213">{{Cite web |url = http://www.master-insight.com/%E5%8A%89%E9%97%9C%E5%BC%B5%E7%B5%90%E7%BE%A9%E7%AD%B2%E7%AE%95%E7%81%A3/ |title = 劉關張結義筲箕灣 |author = 陳天權 |date = 2018-02-13 |publisher = 灼見名家傳媒有限公司 |language = zh-hk |accessdate = 2018-04-28 |deadurl = no |archiveurl = https://web.archive.org/web/20180429010344/http://www.master-insight.com/%E5%8A%89%E9%97%9C%E5%BC%B5%E7%B5%90%E7%BE%A9%E7%AD%B2%E7%AE%95%E7%81%A3/ |archivedate = 2018-04-29 }}</ref><ref name="HK01"/>，同時是全香港唯一同時供奉[[劉備|劉備]]、[[關羽|關羽]]及[[張飛|張飛]]三人的地方<ref name="HK01"/><ref name="on.cc"/><ref name="港自遊"/>。

== 歷史 ==
早於1887年，有居民在颱風過後於海邊拾得一尊福德公像，其後將該神像置於淺水碼頭村村口的一間細小棚屋內，被當地居民稱為南安坊福德祠<ref name="灼見名家@20180213"/><ref name="To Ti Temple@SKW">{{Cite book | author = 香港政府 | title = To Ti Temple S.K.W., 13.12.1945 - 09.08.1977. | location = 香港 | publisher = 香港政府檔案處 | accessdate = 2018-04-28 | language = zh-hk }}</ref>。1937年，殖民政府打算發展南安坊一帶，福德祠便從筲箕灣道街尾附近遷至柴灣道。1959年，政府指廟宇因丟空多時而失去原有的宗教功能，對廟宇所在的地段行使「放棄租住權契據（租約）/土地歸還契約（土地）」，將該地段還原為官地<ref name="To Ti Temple@SKW"/><ref name="香港史新編上">{{Cite book | author = 王賡武 | title = 香港史新編（上冊） | location = 香港 | publisher = 三聯書店（香港）有限公司 | date = 1998-01-01 | pages = 205-206 | ISBN = 9789620412554 | accessdate = 2018-04-28 | language = zh-hk}}</ref>。1965年，政府打算清拆福德廟並將福德公遷往筲箕灣東大街的福德祠（今[[筲箕灣城隍廟|筲箕灣城隍廟]]），但遭南安坊坊眾反對<ref name="灼見名家@20180213"/><ref name="To Ti Temple@SKW"/><ref name="香港史新編上"/>。1968年，南安坊坊眾會去信筲箕灣街坊福利會，代為向政府申請撥地重建福德祠，而政府亦撥出愛秩序村山坡予坊眾建廟。最終福德廟於1970年1月落成開光，並於同年11月向華人廟宇委員會註冊<ref name="To Ti Temple@SKW"/><ref name="香港史新編上"/>。

1971年，南安坊坊眾會在福德廟旁的山洞供奉觀音，並起名「觀音岩」<ref name="灼見名家@20180213"/>。1976年，筲箕灣街坊福利會將觀音岩擴建成廟，同時在毗鄰增建關帝廟<ref name="灼見名家@20180213"/>。另外，在政府清拆愛秩序村後，南安坊坊眾會在福德廟旁重建洪聖古廟，並於1980年落成<ref>{{Cite web |url = http://www.takungpao.com/fk/content/2012-09/28/content_1176134.htm |title = 筲箕灣的廟宇 |author = 潘高爽 |date = 2012-09-28 |publisher = 大公網 |accessdate = 2018-04-28 |language = zh-cn |deadurl = yes |archiveurl = https://web.archive.org/web/20180429010344/http://www.takungpao.com/fk/content/2012-09/28/content_1176134.htm |archivedate = 2018-04-29 }}</ref>。

1981年觀音誕，居民鄭興陪同患病的弟弟鄭寶義前往山坡上的觀音廟參拜，弟弟突然被神靈附體，他自喟是擁有「不同姓氏、不同性格、不同面孔」的三兄弟之一，並指只要他們設壇供奉，他便能讓鄭興的弟弟康復<ref name="HK01">{{Cite web |url = https://www.hk01.com/%E7%A4%BE%E5%8D%80%E5%B0%88%E9%A1%8C/40870/%E7%9B%82%E8%98%AD%E5%8B%9D%E6%9C%83-%E5%BE%9E%E5%9C%B0%E5%8D%80%E5%BB%9F%E5%AE%87%E5%87%BA%E7%99%BC-%E6%83%85%E6%B7%B1%E5%AE%88%E5%BB%9F%E4%BA%BA-%E5%85%B6%E5%AF%A6%E6%98%AF%E5%98%89%E5%B9%B4%E8%8F%AF |title = 【盂蘭勝會】從地區廟宇出發　情深守廟人：其實是嘉年華 |author = 陳穎然、何潔泓 |date = 2016-09-03 |publisher = 香港01 |accessdate = 2018-04-28 |deadurl = no |archiveurl = https://web.archive.org/web/20180429010347/https://www.hk01.com/%E7%A4%BE%E5%8D%80%E5%B0%88%E9%A1%8C/40870/%E7%9B%82%E8%98%AD%E5%8B%9D%E6%9C%83-%E5%BE%9E%E5%9C%B0%E5%8D%80%E5%BB%9F%E5%AE%87%E5%87%BA%E7%99%BC-%E6%83%85%E6%B7%B1%E5%AE%88%E5%BB%9F%E4%BA%BA-%E5%85%B6%E5%AF%A6%E6%98%AF%E5%98%89%E5%B9%B4%E8%8F%AF |archivedate = 2018-04-29 }}</ref><ref name="on.cc">{{Cite web |url = http://hk.on.cc/hk/bkn/cnt/news/20180426/bkn-20180426060003890-0426_00822_001.html |title = 劉關張穿越筲箕灣結義？香港人都拜張飛 |author = on.cc東網 |date = 2018-04-26 |publisher = 東方報業集團 |accessdate = 2018-04-28 |language = zh-hk |deadurl = no |archiveurl = https://web.archive.org/web/20180429010344/http://hk.on.cc/hk/bkn/cnt/news/20180426/bkn-20180426060003890-0426_00822_001.html |archivedate = 2018-04-29 }}</ref><ref name="港自遊">{{Cite book | author = 陳天權 | title = 港自遊 | location = 香港 | publisher = 文化葫蘆 | date = 2018-03 | pages = 41、65 | accessdate = 2018-04-28 | language = zh-hk }}</ref>。後來鄭興推斷是張飛降臨，漁民出身的他便把賴以為生的船隻賣掉，得款14萬元，最後在1982年於福德廟範圍興建了全香港唯一的張飛廟<ref name="HK01"/><ref name="港自遊"/>。此外，廟宇地上設有紀念碑描寫張飛與鄭寶義的故事<ref name="HK01"/>。

其後，有居民認為既然有關帝廟和張飛廟，倒不如再興建劉備廟，於是鄭興發動籌款，最終劉備廟於1993年建成<ref name="灼見名家@20180213"/>。

[[古物古蹟辦事處|古物古蹟辦事處]]曾將這六間廟宇納入歷史建築物評級名單，但最終於2010年2月被[[古物諮詢委員會|古物諮詢委員會]]確定不予評級<ref name="灼見名家@20180213"/><ref>{{Cite web |url = http://www.aab.gov.hk/form/AAB-SM-chi.pdf |title = List of the 1,444 Historic Buildings with Assessment Results (as at 22 March 2018) |author = 古物諮詢委員會 |date = 2018-03-27 |accessdate = 2018-04-29}}</ref>。

== 祭祀 ==
六間廟宇各有不同賀誕日子，包括農曆二月初二的福德誕、二月十三日的洪聖誕、二月十九日的觀音誕、六月廿四日的關帝誕、八月初八的劉備誕，以及十二月十九日的張飛誕<ref name="灼見名家@20180213"/>。

在張飛誕，醒獅隊會於[[筲箕灣譚公廟|筲箕灣譚公廟]]起步，經[[筲箕灣東大街|筲箕灣東大街]]及其他街道前往廟宇，抵達後逐一向各廟宇參拜，並在道堂經師帶領下在張飛廟前參拜誦經<ref name="灼見名家@20180213"/><ref name="港自遊"/>。而張飛誕更於2013年被納入香港首份非物質文化遺產清單<ref name="灼見名家@20180213"/><ref>{{Cite web|url =http://www.districtcouncils.gov.hk/north/doc/2012_2015/tc/committee_meetings_doc/cbcrac/3721/n_cbcrac_2013_028_ch.pdf|title =北區區議會 社區建設、文化及康樂事務委員會 全港性非物質文化遺產普查|author =康樂及文化事務署|date =2013-11-07|publisher =康樂及文化事務署|accessdate =2018-04-28|deadurl =no|archiveurl =https://web.archive.org/web/20180429010344/http://www.districtcouncils.gov.hk/north/doc/2012_2015/tc/committee_meetings_doc/cbcrac/3721/n_cbcrac_2013_028_ch.pdf|archivedate =2018-04-29}}</ref>。

另外，鄭興表示祭祀張飛有獨特的祭祀次序。由於福德廟為最先坐落在該地的廟宇，因此要先向福德廟上香，其次祭祀劉備及關羽，最後才祭祀張飛<ref name="on.cc"/>。

== 傳聞 ==
在鄭興的弟弟鄭寶義被張飛治好後，當區居民都慕名而前來排隊看病。有一次，張飛附上鄭寶義之身，寫下藥方，結果不出40日，該名居民病瘉<ref name="HK01"/>。

2012年11月，網上流傳一則於福德廟後山誤撞山妖的傳聞<ref>{{Cite web |url = http://archive.am730.com.hk/column-130705 |title = 轉角 - 筲箕灣山妖 |author = 陳雲 |date = 2012-11-20 |publisher = am730 |accessdate = 2018-04-28 |deadurl = no |archiveurl = https://web.archive.org/web/20180429010345/http://archive.am730.com.hk/column-130705 |archivedate = 2018-04-29 }}</ref>。內容大致是當事人為求身體健康，誤以為一些較少人參拜的廟宇特別靈驗，於是便前往山上的一些供奉諸神的廟宇參拜，並在途中向路邊的破爛神像上香，繼而發生怪事。在當事人前往廟宇酬謝神恩的途中取道偏僻岔路，結果走了大約四十多分鐘仍然看不見廟宇，最後被洪聖爺拯救。
