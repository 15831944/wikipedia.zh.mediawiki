==登場角色==
; {{nihongo|莊敦大|荘 敦大（かがり あつひろ）|聲：[[鈴木達央|鈴木達央]]<ref name="character">{{cite web|url=http://akkun-kanojo.jp/character/|title=CHARACTER|work=電視動畫「敦君與女朋友」官方網站|language=ja}}</ref>}}
:與片桐音是男女朋友關係，但在音面前會裝成傲嬌的樣子，私底下則是非常溺愛音，有時還會偷偷拍音的照片。

; {{nihongo|片桐音|片桐 のん（かたぎり のん）|聲：[[諏訪彩花|諏訪彩花]]<ref name="character"/>}}
:與莊敦大是男女朋友關係，知道敦大在她面前都會裝成傲嬌的樣子，敦大實際上是非常喜歡她的。但是不知道敦大經常偷拍她。十分單純。

; {{nihongo|松尾真砂|松尾 真砂（まつお まさご）|聲：{{link-ja|植田圭輔}}<ref name="character"/>}}
:莊敦大的好友，喜歡玩galgame，只喜歡galgame裡的女角色，但現實中卻意外受到國中女性們的仰慕。

; {{nihongo|莊千穗|荘 千穂（かがり ちほ）|聲：[[香里有佐|香里有佐]]<ref name="character"/>}}
:莊敦大的妹妹，喜歡松尾，害怕看鬼片和蟑螂。

; {{nihongo|窪村匠|窪村 匠（くぼむら たくみ）|聲：[[羽多野涉|羽多野涉]]<ref name="character"/>}}
:和敦大及音還有小凪的學校的老師。

; {{nihongo|入江小凪|入江 小凪（いりえ こなぎ）|聲：{{link-ja|嘉陽光}}<ref name="character"/>}}
:和敦大及音同個學校的學生，喜歡窪村匠老師。
; {{nihongo|莊圭太|聲：}}
:莊敦大和莊千穗的父親，還不知道自己的兒子已經有交往四年的女友這件事。

; {{nihongo|莊櫻子|聲：}}
:莊敦大和莊千穗的母親。

; {{nihongo|片桐正一郎|聲：}}
:片桐音的父親，與敦大一樣，在音的母親面面前都會裝成傲嬌的樣子。

; {{nihongo|片桐月實|聲：}}
:片桐音的母親。

; {{nihongo|日向野優衣|聲：}}
:千穗的好友兼同學，喜歡帥哥。

; {{nihongo|紗良|聲：}}
:千穗的好友兼同學。
