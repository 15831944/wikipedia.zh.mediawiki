《'''敦煌乐谱'''》，又称《'''敦煌曲谱'''》或《'''敦煌琵琶谱'''》，是于[[敦煌|敦煌]][[莫高窟|莫高窟]]藏经洞发现的一组[[琵琶|琵琶]]乐谱，属于[[工尺谱|工尺谱]]的一种，抄写日期为晚[[唐|唐]]至[[五代|五代]]。[[记谱法|记谱法]]使用的音乐符号有23个，类似汉字[[部首|部首]]、[[注音符号|注音符号]]或日语[[片假名|片假名]]，成为今人无法识别的“[[天书|天书]]”<ref name="sohu" /><ref>{{cite web |url=http://gs.ifeng.com/a/20180611/6645446_0.shtml |title=古乐谱复活记之从敦煌经卷走出的唐琵琶曲 |last= |first= |date=2018-06-11 |website=凤凰网 |publisher=王文元 |access-date=2019-03-20 |quote=}}</ref>。

1908年，法国人[[伯希和|伯希和]]从藏经洞带走的经卷中有其中三卷：《[[长兴四年中兴殿应圣节讲经文|长兴四年中兴殿应圣节讲经文]]》（[https://gallica.bnf.fr/ark:/12148/btv1b8303290v/f10.planchecontact P.3808]）、《[[尔雅|尔雅]]》（[https://gallica.bnf.fr/ark:/12148/btv1b8302820z/f5.planchecontact P.3719]）和《[[佛本行集经|佛本行集经]]尤波离品次》（[https://gallica.bnf.fr/ark:/12148/btv1b83006835/f2.planchecontact P.3539]），比较特别的是这三卷经卷的背面都写有乐谱，现均收藏在[[法国国家图书馆|法国国家图书馆]]<ref name="sohu">{{cite web |url=http://www.sohu.com/a/231944370_200821 |title=《敦煌曲谱》——千年谜音！  |last= |first= |date=2018-05-17 |website=搜狐 |publisher=上海市书法家协会 |access-date=2019-03-20 |quote=}}</ref>。

== 破解 ==
最早研究敦煌乐谱的破解是日本的学者{{link-ja|林谦三|林謙三}}，后来[[任二北|任二北]]、[[饶宗颐|饶宗颐]]等人也尝试过破解，但均未达成共识<ref>{{cite web |url=http://public.dha.ac.cn/Content.aspx?id=593665752339&Page=2&types=1 |title=60年潜心民乐“考古”拿下敦煌乐谱最终解释权 |last= |first= |date=2015-04-01 |website= |publisher=文汇报 |access-date=2019-03-21 |quote=}}</ref>。它和唐俗乐二十八调的破译，曾被认为是中国古乐研究领域的两个“[[哥德巴赫猜想|哥德巴赫猜想]]”式的问题<ref>{{cite journal |author1=宋瑞桥 |title=祖孝孙音阶与唐代雅、俗乐的宫调(下) |journal=乐府新声(沈阳音乐学院学报) |date=1996 |issue=3}}</ref>。

1992年10月10日，原甘肃敦煌艺术剧院院长[[席臻贯|席臻贯]]成功破译了敦煌乐谱上的音乐符号，收录破解后曲谱的《敦煌古乐》图书和磁带由敦煌文化出版社和甘肃音像出版社出版<ref>{{cite web |url=http://culture.gansudaily.com.cn/system/2012/03/29/012426754.shtml |title= 【陇原记忆】耗尽一生 他使千年古乐重现人间(图)|last= |first= |date=2012-03-29 |website=每日甘肃 |publisher= |access-date=2019-03-20 |quote=}}</ref>。

== Unicode ==
敦煌乐谱符号尚未在Unicode中收录，但已有相关编码方案提案，收录的编码范围为1D280-1D2DF，所属的区块名为“中国音乐符号”（Chinese Musical Symbols）<ref>{{cite web |url=https://www.unicode.org/L2/L2017/17311-n4848-lute.pdf |title= JTC1/SC2/WG2 N4848 L2/17-311|last= |first= |date=2017-09-07 |website=Unicode |publisher= |access-date=2019-03-20 |quote=}}</ref>。
