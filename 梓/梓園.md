'''梓园'''是一处中西结合的园林，位于中国[[上海市|上海市]][[黄浦区|黄浦区]][[上海老城厢|老城厢]]的东南部，乔家路113号。

== 历史<ref>{{cite book|author1=上海市地方志办公室|title=上海名建筑志|date=2005|publisher=上海社会科学院出版社|location=上海|isbn=7-80681-736-0|url=http://www.shtong.gov.cn/Newsite/node2/node71994/node81772/node81776/node81788/userobject1ai109092.html|accessdate=2018-04-05}}</ref> ==
梓园从1667年开始建造，最早的主人是[[周金然|周金然]]。到1907年，[[王一亭|王一亭]]买下此园林并且建造了一栋西式建筑，将园林改名为梓园。王一亭是一位商界名人并且也是上海著名的书画家。

梓园是一座中西结合的住宅楼。在门口的两侧各有两根爱奥尼亚柱式，这是十分令人惊叹的。梓园的建筑结构在中国佛教亭阁中很少见。哥特式的窗户与希腊风格的亭柱与这座法式经典主义建筑巧妙的结合在了一起。

虽然梓园是一处受政府保护的历史文化场所<ref>{{cite web |language =  |publisher = 上海市文化广播影视管理局 |title = 黄浦区县级文物保护单位、登记不可移动文物 |url = http://wgj.sh.gov.cn/wgj/node1257/node1264/u1a75610.html |author = 市文广影视局 |date = 2011-10-28 |accessdate = |archiveurl= https://web.archive.org/web/20120512174125/http://wgj.sh.gov.cn/wgj/node1257/node1264/u1a75610.html | archivedate=2012-05-12 }}</ref>，但仍然有许多家庭居住于此，周围的卫生环境不容乐观。
