'''蓝鹱'''（学名：''Halobaena caerulea''）是[[鹱科|鹱科]]的一种小型海鸟。也是[[蓝鹱属|蓝鹱属]]的唯一一种。

== 外观 ==

蓝鹱的翅膀颜色前部呈灰色，后部呈白色，从上面看上去形成了一个“M”字型。尾部尖端为白色。<ref>ZipCode Zoo (16 Jul 2009)</ref>

== 食性 ==

蓝鹱主要以磷虾为食，也会捕食其他甲壳类动物、鱼类和乌贼。他们潜水可以潜下6米深。<ref>ZipCode Zoo (16 Jul 2009)</ref>

== 繁殖 ==

和其他鹱形目的鸟类一样，蓝鹱的繁殖地也很广阔。他们在洞穴里筑巢，一次繁殖期只产一枚卵。双亲轮流孵卵，孵化期50天，幼鸟于55天大时开始长羽。贼鸥是幼鸟和蛋的主要威胁。<ref>ZipCode Zoo (16 Jul 2009)</ref>

== 分布和生境 ==

蓝鹱分布于南半球的海域，从非洲南部、澳大利亚和南美洲到南极地区均有生活。他们在亚南极地区的海岛上筑巢，诸如arion Island, the Crozet Islands, Kerguelen Islands, Macquarie Island, South Georgia, Prince Edward Island等。<ref>ZipCode Zoo (16 Jul 2009)</ref>
