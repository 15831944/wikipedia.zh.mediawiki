'''蓝天野'''（{{bd|1927年|5月3日|catIdx=W}}），原名'''王润森'''，又名'''王皇'''，出生于[[河北省|河北省]][[衡水市|衡水市]][[饶阳县|饶阳县]][[大官亭乡|大官亭乡]]。[[中国|中国]][[北京人民艺术剧院|北京人民艺术剧院]][[话剧|话剧]][[演员|演员]]、[[导演|导演]]。1944年，就成为话剧演员。中华人民共和国成立后，成为北京人民艺术剧院第一批主要演员。离休后，从事[[中国画|中国画]]的创作。话剧代表作品《[[茶馆_(话剧)|茶馆]]》、《[[蔡文姬|蔡文姬]]》，電視代表作品《[[渴望|渴望]]》的王子涛、《[[封神榜_(1990年)|封神榜]]》的[[姜子牙|姜子牙]]。<ref>{{cite web |url=http://news.fjsen.com/2014-03/16/content_13697202.htm |title=蓝天野87岁仍坐镇排练场 亲自拄杖为演员搭戏(图)  |accessdate=2014 |publisher= |work= |language=zh }}</ref><ref>{{cite web |url=http://ent.163.com/photoview/5B8A0003/522515.html#p=9LUUJEK15B8A0003 |title=87岁艺术家蓝天野再导话剧 |accessdate=2014 |publisher=163.com |work= |language=zh }}</ref><ref>{{cite web |url=http://money.163.com/14/0223/01/9LNVS0SA00253B0H.html |title=87岁蓝天野再出山不谈退休 |accessdate=2014 |publisher=163.com |work= |language=zh }}</ref><ref>{{cite web |url=http://yule.sohu.com/20140226/n395679875.shtml |title=人艺复排《吴王金戈越王剑》 蓝天野重执导筒 |accessdate= 2014|publisher=Sohu |work= |language=zh }}</ref>
