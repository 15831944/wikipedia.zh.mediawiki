'''蓝白井字旗'''是[[辛亥革命|辛亥革命]]时期[[中国同盟会|中国同盟会]]提出的[[国旗|国旗]]草案。该旗曾一度作为[[北洋政府|北洋政府]]的元帅旗和[[中国致公党|中国致公党]]的党旗使用。其寓意一是[[井田制|井田制]]，二是寓意[[九州_(中国)|天下九州]]。

== 革命的青天白日旗 ==

* [[光緒|光緒]]十九年（1893年），革命先烈[[陆皓东|陸皓東]]寄住於[[尢列|尢列]]（[[四大寇|四大寇]]之一）在[[广州市|廣州]]家中的「聽濤閣」時，設計一面「青地、中有白日」的[[青天白日旗|青天白日旗]]，預備作為革命的旗幟。由於陸已在首次廣州起義(第一次革命)殉難，旗之正確圖案，知者不多，「當時旗上所排列叉光，多寡不一，縫製者多莫名其妙，後經總理（中山先生）解釋，謂叉光即代表干支數目，故叉光應排作十二，以代十二時辰。自是旗上叉光之數，始確定不易。」<ref>孫鎮東著，國旗國歌國花史話，第8頁</ref><ref>革命逸史，第一冊第二十八頁</ref>
* 光緒二十一年（1895年），[[興中會|興中會]]總部在[[夏威夷州|夏威夷]]成立，孫文召集革命同志舉行其首次幹部會議。興中會決定發動[[乙未廣州起義|廣州起義]]，會員陸皓東設計之青天白日旗作為起義軍之旗，旗底為藍色，以示青天，旗中置一白日，有白光芒十二道<ref name="謎" >{{cite book |title=《1901～1949中華民國之謎》（修訂版） |authors=王曉華、張慶軍 |year=2016 |location=台北市 |publisher=靈活文化}}</ref>{{rp|8}}。此後多次起義皆以此旗為標幟號召，揭示「光明正照」、「自由平等」之義。
* 光緒二十六年（1900年），第二次革命時，鄭士良舉義於[[惠州市|惠州]]三洲田起義時，第一次打出青天白日旗為標幟<ref name=謎 />{{rp|8}}。

== 討論未果 ==
[[File:蓝白井字旗.jpg|thumb]]
[[File:Naval_Jack_of_the_Republic_of_China.svg|thumb]]]]

光緒三十二年冬（1906年），[[中国同盟会|中國革命同盟會]]討論《革命方略》和[[国旗|國旗]]之形式，[[孫中山|孫文]]主張沿用[[青天白日旗|青天白日旗]]，以紀念已犧牲之設計人陸皓東及其他[[兴中会|興中會]]先烈；[[廖仲恺|廖仲愷]]提議、並連同[[黄兴|黃興]]共同支持使用[[井字旗|井字旗]]有[[井田制|井田]]和[[九州_(中国)|天下九州]]的寓意；另有人提出用金瓜[[钺|钺斧]]旗，以發揚[[汉族|汉族]]精神；用[[十八星旗|十八星旗]]，以代表[[内地十八省|十八行省]]；用[[五色旗|五色旗]]，則可以順應[[中国历史|中国历史]]和[[中华文化|中华文化]]之習慣。孫文在青天白日旗上加紅地，改成紅、藍、白三色，以符合世界上如[[法國國旗|法國國旗]][[自由、平等、博愛|自由、平等、博愛]]之真意，想以此旗作為革命軍國旗圖樣式<ref name=謎 />{{rp|8}}。此想法也成為「[[青天白日滿地紅旗|青天白日滿地紅旗]]」的濫觴。[[黄兴|黄兴]]認為青天白日旗之形式不美觀，而且以太陽為標誌，是效法日本[[旭日旗|旭日旗]]，有[[日本|日本]]并华之嫌；於是設計井字旗，以表示[[平均地權|平均地權]]之意，但被孫否定<ref name=謎 />{{rp|8-9}}。

双方争执不下，[[孙中山|孙中山]]坚持己见，[[黄兴|黄兴]]力争不得，甚至大怒，誓言要脱离[[同盟会|同盟会]]籍。[[章太炎|章太炎]]、[[刘揆一|刘揆一]]则从中调解，暂时搁置议案。这是[[辛亥革命|辛亥革命]]以前孙、黄最大的一次分歧。<ref>[http://www.sohu.com/a/119746936_251661 用青天白日旗，还是用井字旗，黄兴孙中山曾闹掰过]</ref><ref>[http://www.sohu.com/a/128178835_557768 孙中山与黄兴因国旗样式险些闹翻]</ref>

當時同盟會南洋分會副會長張永福之妻陳淑字（一說淑宗）依孫文「加紅色的新構想」，縫製了四種國旗草案，其中的草案一即仿自[[美國國旗|美國星條旗]]的設計，以五條紅線作為[[青天白日旗|青天白日旗]]的襯底。然而該草案始終未被採用。孫文在民國元年（1912年）1月12日覆參議會函中，清楚定義：青天表示清清白白，白日象徵光明，代表光明正照，自由平等。紅色表示博愛。自丁末1907年[[潮州黃岡之役|潮州黃岡之役]]起，至辛亥1911年[[黃花崗起義|黃花崗起義]]止，中經8次起義（第3次至第10次），皆用[[青天白日滿地紅旗|青天白日滿地紅旗]]為革命標幟，黃興迭任主帥，亦從無反對表示<ref>孫鎮東著，國旗國歌國花史話，第9-10頁</ref>。

== 國旗多次更易 ==
[[武昌起義|武昌起義]]後，[[中華民國|中華民國]]於民國元年（1912年）1月1日在[[南京|南京]]肇建，1月2日起[[各省都督府代表聯合會|各省都督府代表聯合會]]代行[[中華民國臨時參議院|中華民國臨時參議院]]職權。1月10日，各省都督府代表聯合會決議以[[五色旗|五色旗]]作為中華民國國旗，取[[紅|紅]]、[[黃|黃]]、[[藍|藍]]、[[白|白]]、[[黑|黑]]五色，代表[[漢族|漢]]、[[满族|滿]]、[[蒙古族|蒙]]、[[回族|回]]、[[藏族|藏]]五族共和之意；定[[十八星旗|铁血十八星旗]]為陸軍旗，[[青天白日紅旗|青天白日紅旗]]為海軍旗，請孫頒行全國<ref name=謎 />{{rp|10}}<ref name="軍旗沿革">{{cite book |author=歐陽晧 |title=青天白日旗飄揚：中華民國軍旗沿革 |date=2015年10月 |publisher=銘閎實業 |isbn=978-986-92450-0-5 }}</ref>{{rp|16}}。在討論[[中華民國國旗|中華民國國旗]]的問題時，因各省革命軍使用的旗幟並不一致，引起各省代表的爭執。其中，[[湖北|湖北]]、[[湖南|湖南]]、[[江西|江西]]省跟隨[[共進會|共進會]]使用[[十八星旗|鐵血十八星旗]]，代表[[内地十八省|十八行省]]；[[江蘇省|江蘇省]]、[[浙江省|浙江省]]、[[安徽省|安徽省]]的[[中国同盟会|中国同盟会]]卻使用五色旗，以代表[[五族共和|五族共和]]；[[廣東|廣東]]、[[廣西|廣西]]、[[福建|福建]]、[[雲南|雲南]]、[[貴州|貴州]]省則沿襲同盟會傳統，使用[[青天白日旗|青天白日旗]]；[[陳炯明|陳炯明]]在[[惠州|惠州]]舉兵時曾採用井字旗，會師[[廣州|廣州]]後，未再使用。1月12日，孫認為五色表示五族取義不確，將五色上下排列亦不平等、仍有階級，未可遽付頒行，遂函覆各省都督府代表聯合會可俟諸民選國會成立後，再經由國民公開表決<ref name=謎 />{{rp|10}}<ref>（一）清國舊例，海軍以五色旗為一二品大官之旗，今黜滿清之國旗而用其官旗，未免失體。（二）其用意為五大民旗，然其分配代色，取義不確，如以黃代滿之類。（三）既言五族平等，而上下排列，仍有階級。[http://sunology.culture.tw/cgi-bin/gs32/gsweb.cgi?o=dalldb&s=id=%22OD0000006516%22.%20&searchmode=basic 孫文：咨覆參議會論國旗文，國父全集，第六冊,pp.002-003]</ref>。

1912年1月28日中華民國臨時參議院成立，2月15日，臨時參議院選舉[[袁世凱|袁世凱]]為[[中華民國大總統|臨時大總統]]，5月6日，參議院討論國旗統一案<ref name=謎 />{{rp|12}}。5月14日，臨時參議院議決以五色旗為國旗，將[[十八星旗|十八星旗]]改為[[十九星旗|十九星旗]]以置於五色旗左上角為陸軍旗，青天白日旗置於五色旗左上角為海軍旗，袁世凱認為後兩者設計不妥，於5月26日覆議，最後參議院臨時於6月5日通過十九星旗為陸軍旗，[[青天白日滿地紅旗|青天白日滿地紅旗]]為海軍旗。6月8日，此案由臨時大總統袁世凱公布施行：「以五色旗為國旗，商旗適用國旗，以十九星旗為陸軍旗，以青天白日旗{{NoteTag|依當時政府公報刊載圖，可知此處「青天白日旗」是指青天白日滿地紅旗。|name=海軍旗}}為[[海軍旗|海軍旗]]」<ref name="軍旗沿革" />{{rp|16}}。<ref>[http://www.globalflag.idv.tw/wg/roc.htm#gqjs 旗海图帜]</ref>

== 青天白日滿地紅旗 ==
[[File:Flag_of_the_Republic_of_China.svg|thumb]]

1910年底，孫中山希望於三藩市設立中華籌餉局籌餉，並打算把起義成功後將要正式使用的國旗印在籌餉的金幣券上，但過去起義的旗幟圖樣，因輾轉傳製很不統一，若要付印，需加以標準美化。經[[李鐵夫|李鐵夫]]修改設計的起義旗幟圖案，印在了金幣券上<ref>{{Cite web|url=https://www.itsfun.com.tw/%E6%9D%8E%E9%90%B5%E5%A4%AB/wiki-7512839-4794168|title=李鐵夫 -油畫家-華人百科|accessdate=2017-06-28|work=www.itsfun.com.tw|language=zh-TW}}</ref>。

1921年5月5日，[[孫中山|孫中山]]在[[廣州|廣州]]就任非常大總統時，宣佈廢止[[五色旗|五色旗]]及十八星旗，規定青天白日滿地紅旗為國旗和軍旗<ref name=謎 />{{rp|12}}。孫中山以五色旗乃前[[滿清|滿清]]官旗、五色代表五大民族之分配代色取義不確、及五色上下排列違背五族平等等理由反對五色旗，以[[青天白日紅旗|青天白日紅旗]]為國旗。但[[北洋政府|北洋政府]]仍為當時統治全國的政權，五色國旗仍然廣受認同。

1924年6月30日，[[中國國民黨|中國國民黨]]中央執行委員會決定以青天白日滿地紅旗為中華民國國旗。1925年7月，[[國民政府|國民政府]]於[[廣州|廣州]]成立，1926年進行[[國民革命軍北伐|北伐]]，所克之處，皆豎青天白日滿地紅旗。

1928年12月，[[國民革命軍北伐|國民革命軍北伐]]結束，[[中國東北|中國東北]]的[[奉系軍閥|奉系]]將領[[张学良|张学良]]在12月29日宣佈[[東北易幟|東北易幟]]，廢五色旗，立[[青天白日紅旗|青天白日紅旗]]。國民政府名義上統一[[中国大陆|中國大陸]]各[[省_(中華民國)|省]]，青天白日滿地紅旗亦隨之用於全[[中國|中國]]各地，國旗的認同逐步建立。

但也有許多名流不承認其正統性，如在1936年，國學大師[[章太炎|章太炎]]逝世的時候，只願以五色旗覆蓋，不承認[[青天白日紅旗|青天白日紅旗]]。<ref>《憶辛亥革命前後先父章太炎若干事》[http://www.nownews.com/2004/07/14/10844-1657896.htm 中國評論學術出版社] {{webarchive|url=https://web.archive.org/web/20091013142941/http://www.nownews.com/2004/07/14/10844-1657896.htm |date=2009-10-13 }}</ref>

1946年12月25日，[[制憲國民大會|制憲國民大會]]通過《[[中華民國憲法|中華民國憲法]]》，明文規定『中華民國國旗定為紅地，左上角青天白日』，將國旗入憲。

1949年10月1日，[[中華人民共和國|中華人民共和國]]於[[北京|北京]]成立後，採用[[中國人民政治協商會議|中國人民政治協商會議]]確定的[[五星紅旗|五星紅旗]]作為其國旗，退守[[台灣|台灣]]的[[中華民國政府|中華民國政府]]則繼續沿用青天白日滿地紅旗，此後青天白日滿地紅旗僅於[[台澎金馬|台澎金馬]]地區普遍使用至今。

少數情形會被稱作「青天白日旗」<ref name="軍旗沿革" />{{rp|16}}<ref>「參議院咨大總統公布國旗及陸海軍旗式樣」，中華民國政府公報第47號，1912年6月16日</ref><ref>「大總統令公布國旗陸軍旗海軍旗」，中華民國政府公報第43號，1912年6月12日</ref><ref>「國旗陸軍旗海軍旗旗式」，中華民國政府公報第45號，1912年6月14日</ref>。

== 中国致公党党旗（1925年10月—1950年4月） ==
[[File:历史上中国致公党党旗（1925年10月—1950年4月）.jpg|thumb]]

早期的中国致公党党旗源于井字旗的设计，在井字旗的基础上做了略微修改。

1932年“中国致公党香港支部初届职员会议”职员的集体照，在合影人员上端并排悬挂着两面旗帜，旗帜下垂，右面一幅是五色旗，是1912年中华民国成立时确定的国旗，左面一幅是井字旗，井字图像总体上清晰可辨。这就是文字记载中的致公党党旗，它一直延续到中华人民共和国成立。<ref>[http://www.shzgd.org/renda/zgd/node10954/node16110/u1a1771454.html 上海致公]</ref>
