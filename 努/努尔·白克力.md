'''努尔·白克力'''（{{lang-ug|نۇر بەكرى}} / {{lang|ug-Latn|Nur Bekri}}；{{bd|1961年|8月|catIdx=Bekri, Nur}}），[[维吾尔族|维吾尔族]]，[[新疆维吾尔自治区|新疆]][[博乐市|博乐]]人，[[中华人民共和国|中华人民共和国]][[省部级正职|正部长级]]官员，[[中国共产党|中国共产党]][[中国共产党第十七届中央委员会候补委员列表|第十七届中央候补委员]]、[[中国共产党第十八届中央委员会委员列表|第十八届中央委员]]。曾任[[乌鲁木齐市|乌鲁木齐市]][[乌鲁木齐市市长列表|市长]]、[[新疆维吾尔自治区人民政府|新疆维吾尔自治区人民政府]][[新疆行政长官列表|主席]]<ref name="jianli">{{cite web|url=http://district.ce.cn/newarea/sddy/201302/01/t20130201_24086633.shtml|title=新疆新一屆政府主席、副主席簡歷(主席努爾白克力)|accessdate=2014-12-14|date=2013-02-01|publisher=中国经济网}}</ref>、[[中华人民共和国国家发展和改革委员会|中华人民共和国国家发展和改革委员会]]副主任（正部长级）兼[[国家能源局|国家能源局]]局长。

== 从政经历 ==
1961年，努尔·白克力出生在[[新疆维吾尔自治区|新疆]][[博乐市|博乐县]]一个靠近[[哈萨克斯坦|哈萨克斯坦]]的边陲小镇，从小接受汉族式教育。1977年，以全乡第一名的成绩考入高中；因为成绩突出，1978年，只读了一年高中的白克力就参加了[[普通高等学校招生全国统一考试|高考]]，并被[[新疆大学|新疆大学]]政治系录取<ref name="taikunpao" />。1982年12月，在校读书期间，加入了[[中国共产党|中国共产党]]。毕业后，留校参加党群工作，历任政治系辅导员、团总支书记，新疆大学共青团委副书记、书记，校党委常委、宣传部部长；期间，曾先后在[[陕西师范大学|陕西师范大学]]高校干部培训班和[[中共中央党校|中共中央党校]]三年制研究生班进修<ref name="jianli" />。

1993年，白克力离开了曾经学习、工作过十五年的新疆大学，赴地方任职；历任新疆[[喀什地区|喀什地区]][[地区行政公署|行政公署]]专员助理、行署副专员（曾于1994年5月至11月，在[[山东省|山东省]]挂职锻炼，担任[[肥城市|肥城市]][[市长|副市长]]），[[新疆维吾尔自治区人民政府|新疆维吾尔自治区人民政府]]副秘书长，[[中国共产党乌鲁木齐市委员会|中共乌鲁木齐市委]]副书记，[[乌鲁木齐市人民政府|乌鲁木齐市人民政府]]副市长、代理市长、市长<ref name="jianli" />。据媒体报道，1998年的努尔·白克力是当时中国最年轻的省级行政区首府市长之一<ref name="xwzk" />。

自2000年12月，白克力开始在[[中国共产党新疆维吾尔自治区委员会|中共新疆维吾尔自治区党委]]和[[新疆维吾尔自治区人民政府|自治区人民政府]]任职，历任自治区党委常委、[[政法委员会|政法委]]副书记，自治区党委副书记；并于2007年11月召开的[[中国共产党第十七次全国代表大会|中共十七大]]上，当选[[中国共产党第十七届中央委员会候补委员列表|中央候补委员]]。2008年1月，白克力当选[[新疆维吾尔自治区人民政府|自治区人民政府主席]]<ref name="jianli" /><ref>{{cite web|url=http://news.xinhuanet.com/newscenter/2007-12/28/content_7332906.htm|title=努尔·白克力代理新疆维吾尔自治区主席|accessdate=2014-12-14|date=2007-12-28|publisher=新华网}}</ref><ref>{{cite web|url=http://news.sina.com.hk/cgi-bin/nw/show.cgi/9/1/1/658053/1.html|title=新疆新主席回應分裂分子對雙語教育攻擊|accessdate=2014-12-14|date=2008-03-04|publisher=新浪香港|archiveurl=https://web.archive.org/web/20080328060410/http://news.sina.com.hk/cgi-bin/nw/show.cgi/9/1/1/658053/1.html|archivedate=2008-03-28}}</ref>，力主推行“[[双语教学|双语教学]]”。通过这一政策，使全新疆学前班和中小学在2008年末的236万少数民族学生中共有60余万接受了双语教育<ref name="xwzk" />。2012年11月，在[[中国共产党第十八次全国代表大会|中共十八大]]上，当选[[中国共产党第十八届中央委员会委员列表|中央委员]]；并于2013年1月召开的新疆维吾尔自治区第十二届人民代表大会第一次会议上，连任自治区政府主席<ref>{{cite web|url=http://www.ce.cn/xwzx/gnsz/gdxw/201301/31/t20130131_24083195.shtml|title=努尔·白克力当选新疆维吾尔自治区主席|accessdate=2014-06-28|date=2013-01-31|publisher=[[中国经济网|中国经济网]]|archive-url=https://web.archive.org/web/20141216055824/http://www.ce.cn/xwzx/gnsz/gdxw/201301/31/t20130131_24083195.shtml|archive-date=2014-12-16|dead-url=yes}}</ref>。

2014年12月，白克力调中央工作，任[[中华人民共和国国家发展和改革委员会|中华人民共和国国家发展和改革委员会]]副主任兼[[国家能源局|国家能源局]]局长<ref>{{cite web|url=http://news.sina.com.cn/c/2014-12-31/135931348820.shtml|title=努尔-白克力任国家发改委副主任 接掌能源局|accessdate=2014-12-31|date=2014-12-31|publisher=新浪网}}</ref>。2015年1月9日，努尔·白克力正式上任<ref>{{cite journal |title=中华人民共和国国务院任免人员 |journal=国务院公报 |date=2015 |issue=5 |url=http://www.gov.cn/gongbao/content/2015/content_2818474.htm}}</ref>。

2018年9月21日，[[中国共产党中央纪律检查委员会|中国共产党中央纪律检查委员会]]网站发布消息，努尔·白克力涉嫌严重违纪违法，正接受中央纪委[[中华人民共和国国家监察委员会|国家监委]]纪律审查和监察调查<ref>{{cite web|url=https://app.peopleapp.com/Api/600/DetailApi/shareArticle?type=0&article_id=2462390|title=又一虎！国家发改委副主任，国家能源局局长努尔·白克力被查！|accessdate=2019-09-21|date=2019-09-21|publisher=[[人民日报|人民日报新闻客户端]]}}</ref>。被查消息公布前，白克力还于9月18日在莫斯科与[[中华人民共和国国务院副总理|副總理]][[韓正|韓正]]參加中俄投資合作委員會，9月19日乘坐飞机回国，抵达[[首都国际机场|首都国际机场]]便被帶走<ref>{{cite web|url=https://www.bbc.com/zhongwen/trad/chinese-news-45604118|title=維族高官白克力仕途一路看好 突然「違紀」落馬|accessdate=2019-09-21|date=2019-09-21|publisher=BBC中文网}}</ref>。10月2日，国务院免去努尔·白克力的国家发改委副主任兼国家能源局局长职务<ref>{{cite journal |title=中华人民共和国国务院任免人员 |journal=国务院公报 |date=2018 |issue=30 |url=http://www.gov.cn/gongbao/content/2018/content_5335390.htm}}</ref>。2019年3月16日，中共中央纪委、国家监察委宣布给予努尔·白克力开除党籍、开除公职处分<ref>{{cite web |title=国家发展和改革委员会原党组成员、副主任，国家能源局原党组书记、局长努尔·白克力严重违纪违法被开除党籍和公职 |url=http://www.ccdi.gov.cn/toutiao/201903/t20190316_190625.html |website=中央纪委国家监委网站 |accessdate=2019-03-16 |date=2019-03-16}}</ref>。4月2日，最高檢依法以涉嫌受賄罪對努爾．白克力作出逮捕決定，該案正進一步辦理中。

== 执政观点与公共形象 ==
{{See also|东突厥斯坦独立运动|三股势力}}

努尔·白克力不僅其母語維吾爾語流利，他在公众面前更说着一口流利的漢語普通话<ref name=taikunpao/>。对此他曾在接受[[凤凰卫视|凤凰卫视]]记者的采访时表示其作为中华人民共和国的公民，“学习掌握本国的通用语言天经地义”<ref>{{cite news|url=http://china.huanqiu.com/roll/2010-03/736730.html|title=努尔·白克力:作为中国的公民 学习汉语天经地义|date=2010-03-07|work=新华网|publisher=环球网|accessdate=2014-12-14}}</ref>。同僚则描述其“为人谦和、讲话充满激情”，有见解且善于脱稿演讲<ref name=xwzk/>。

2014年3月6日的第十二届全国人大第二次会议上，努尔·白克力称“总有一小部分人，天天都在做着把新疆从伟大祖国分裂出去的梦想。当然他们的目的不可能实现，也永远不会实现，对此我们充满信心”。白克力指出，“暴力恐怖分子代表不了任何一个民族，也代表不了任何一个宗教。他们干的是反社会反人类的罪恶暴行”<ref>{{cite news|url=http://gb.cri.cn/42071/2014/03/06/7211s4452370.htm|title=努尔·白克力:分裂势力后面总有境外势力作祟|date=2014-03-06|publisher=国际在线|accessdate=2014-12-14}}</ref>。

同年4月，白克力在《[[新疆日报|新疆日报]]》上发表了《凝心聚力 团结奋进 共创新疆各族人民美好未来》一文，称新疆在党中央领导下取得了良好发展，住房、就业、教育和社会保障方面都得到了显著改善。文中其视宗教极端思想为“社会毒瘤”，表示“此祸不除、新疆难安”<ref>{{cite web|url=http://www.guancha.cn/local/2014_04_07_220043.shtml|title=新疆主席努尔·白克力：坚决铲除宗教极端思想这个社会毒瘤（图）|accessdate=2014-12-14|date=2014-04-07|publisher=观察者}}</ref>。

== 相关事件与外界评价==
香港《[[大公报|大公报]]》评论指出努尔·白克力是学而优则仕的典范。他“赶上了一个极好的时代”，对[[丝绸之路|丝绸之路]]经济带的发展倾尽全力、贡献良多<ref name="taikunpao">{{cite web|url=http://news.takungpao.com/mainland/focus/2014-08/2681788.html|title=努尔·白克力：好时代里遇上大挑战|accessdate=2014-12-14|date=2014-08-19|publisher=大公网}}</ref>。《[[中国新闻周刊|中国新闻周刊]]》专访报道称，努尔·白克力作为大学教师出身，拥有文人治政的温文气质，不过在面对分裂势力时会更有铁腕的一面<ref name="xwzk">{{cite news|url=http://www.chinanews.com/gn/news/2009/03-16/1602768.shtml|title=新疆自治区主席努尔·白克力:铁腕文人"先发制敌"|author=王维博|date=2009-03-16|publisher=中国新闻网|accessdate=2014-12-14}}</ref>。

一度在[[中央民族大学|中央民族大学]]任职[[经济学|经济学]]副教授的维吾尔族学者[[伊力哈木·土赫提|伊力哈木·土赫提]]曾开办了[[维吾尔在线|维吾尔在线]]网站，2009年3月在訪問中批評努爾·白克力「無能」，又撰文批评努尔·白克力在[[乌鲁木齐七五事件|乌鲁木齐七五事件]]后偏袒汉族人，努爾·白克力則在事件後的電視講話中指責維吾爾在線網站「大肆進行煽動宣傳，傳播謠言」，伊力哈木·土赫提在2009年7月6日遭到了中国政府的逮捕。[[国际笔会|国际笔会]]狱中作家委员会4天后对中国政府的行为表示谴责并呼吁其尽快释放伊力哈木·土赫提，称此举违反了《[[公民权利和政治权利国际公约|公民权利和政治权利国际公约]]》（中国政府尚未提交全国人大批准）的第19条<ref>{{Cite news|url=http://www.rfa.org/mandarin/zhuanlan/zhongguotoushi/panel-10062014091939.html|title=伊力哈木案及其後果|date=2014-10-06|publisher=[[自由亚洲电台|自由亚洲电台]]|accessdate=2014-10-06}}</ref><ref name="rfa">{{cite news|url=http://www.rfa.org/mandarin/yataibaodao/wei-07122009172935.html|title=国际笔会狱中作家委员会呼吁中国政府释放维族学者|author=唐琪薇|date=2009-07-12|publisher=自由亚洲电台|accessdate=2014-12-14}}</ref>。

[[世界維吾爾代表大會|世界维吾尔代表大会]]内务部长伊利夏提曾撰文称努尔·白克力是“中共的傀儡”，“没有民族性”，“不能代表维吾尔人”<ref>{{cite web|url=http://www.boxun.com/news/gb/pubvp/2009/09/200909092107.shtml|title=伊利夏提：努尔白克力能代表维吾尔人吗？|accessdate=2014-12-14|date=2009-09-09|publisher=博讯}}</ref>。而该组织发言人迪里夏则表示伊力哈木·图赫提对白克力的批评完全属于“陈述事实”<ref name=rfa/>。
