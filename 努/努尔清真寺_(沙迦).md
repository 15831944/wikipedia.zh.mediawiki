'''阿尔诺尔清真寺'''（Al Noor Mosque）是[[沙迦|沙迦]]酋长国的一座[[清真寺|清真寺]]。它位于Buhaira Corniche的 Khaled 泻湖。<ref>{{cite web|url=http://www.arabianprofile.com/Al%20Noor%20Mosque.html |title=Al Noor Mosque |publisher=Arabian Profile |accessdate=2010-06-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20110707154626/http://www.arabianprofile.com/Al%20Noor%20Mosque.html |archivedate=2011-07-07}}</ref> 它属于[[奥斯曼建筑|奥斯曼建筑]]风格，受土耳其[[苏丹艾哈迈德清真寺|苏丹艾哈迈德清真寺]]的影响。<ref name="Khaleej"/><ref>{{Cite web |title=Mosque is a haven for prayer and serenity  |first=Afshan |last=Ahmed |work=The National |date=2011-08-10 |accessdate=2015-02-04 |url=http://www.thenational.ae/news/uae-news/mosque-is-a-haven-for-prayer-and-serenity }}</ref> 它是沙迦的600多座清真寺中，向公众开放的三个清真寺之一<ref name="Khaleej"/><ref>{{Cite web |title=Travel: Middle Eastern culture and allure in Sharjah |work=Western Daily Press |date=2014-10-11 |accessdate=2015-02-04 |url=http://www.westerndailypress.co.uk/Travel-Middle-Eastern-culture-allure-Sharjah/story-23098467-detail/story.html |archive-url=https://web.archive.org/web/20141222195444/http://www.westerndailypress.co.uk/Travel-Middle-Eastern-culture-allure-Sharjah/story-23098467-detail/story.html |archive-date=2014-12-22 |dead-url=yes |df= }}</ref><ref>{{Cite web |title=Sharjah opens mosque to non-Muslims as bridge between cultures |first=Afshan |last=Ahmed |work=The National |date=2011-09-04 |accessdate=2015-02-04 |url=http://www.thenational.ae/news/uae-news/sharjah-opens-mosque-to-non-muslims-as-bridge-between-cultures}}</ref>。

2014年，该清真寺为[[斋月|斋月]]捐款活动而设的“世界上最大的木制慈善框”被列入[[吉尼斯世界纪录|吉尼斯世界纪录]]<ref>{{Cite web|title=Guinness World Records register Sharjah's wooden charity box |work=Gulf Today|date=2014-07-27 |accessdate=2015-02-04 |url=http://gulftoday.ae/portal/8961f396-77f6-4089-ae65-b3f8fd12cf3f.aspx |deadurl=yes |archiveurl=https://web.archive.org/web/20150204184852/http://gulftoday.ae/portal/8961f396-77f6-4089-ae65-b3f8fd12cf3f.aspx |archivedate=2015-02-04 }}</ref><ref>{{Cite web |title=Sharjah charity box sets world record |first=Jumana |last=Khamis |work=Gulf News |date=2014-08-22 |accessdate=2015-02-04 |url=http://gulfnews.com/news/gulf/uae/society/sharjah-charity-box-sets-world-record-1.1375319 }}</ref>。
