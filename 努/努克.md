'''努克'''（'''Nuuk'''），[[格陵蘭|格陵蘭]]自治领地的最大[[城市|城市]]，1979年格陵蘭實行[[自治|自治]]後，成為格陵兰的[[首府|首府]]，兼[[瑟莫蘇克|瑟莫蘇克]]自治市政府所在地。

都市名稱在[[格陵蘭語|格陵蘭語]]（因纽特语的一种）中為“岬角”之意。[[丹麥人|丹麥人]]及[[挪威人|挪威人]]稱之為'''戈特霍布'''（[[丹麥語|丹麥語]]：<span lang="da">Godthåb</span>），意為“美好的希望”，與“[[好望角|好望角]]”的名稱由來相似，在丹麥出版的[[地圖|地圖]]上即標記為“戈特霍布”的名稱。

== 歷史 ==
[[File:Nuuk_main_road.JPG|thumb]]
據考古學家的研究，努克是古[[因纽特人|因纽特人]]的[[薩喀克文明|薩喀克文明]]影響的地區，[[公元前2000年|公元前2000年]]即有因纽特人土著居民居住。[[公元10世紀|公元10世紀]][[維京人|維京人]]入侵該地。16世紀，来自北欧的[[諾爾斯人|諾爾斯人]]在格陵兰岛东岸定居，但其与[[因纽特人|因纽特人]]之間幾無交流。據16世紀以來的人口流入及氣候變遷的記載，諾爾斯人移民在該地逐漸減少以至消失。

1721年，[[挪威|挪威]][[路德派|路德派]]傳教士[[漢斯·埃格德|漢斯·埃格德]]建立起這座城市，並命名為“戈特霍布”。埃格德藉由[[薩迦|薩迦]]傳說的魅力，從[[丹麥|丹麥]]與[[挪威|挪威]]的聯合國王[[弗雷德里克四世|弗雷德里克四世]]處獲得在格陵蘭傳教與殖民的許可。當時格陵蘭是[[丹麥-挪威聯合|丹麥-挪威聯合]]下的挪威[[直轄殖民地|直轄殖民地]]，與[[北歐|北歐]]的本土隔絕已久，先期抵達島上的[[丹麥人|丹麥人]]居住地已經破敗，於是埃格德覓得一處因紐特人部落。埃格德以該因紐特人部落為基礎至1724年的3年間，建立起今日努克市鎮的雛形，並用丹麦语命名為 Godthåb 。1733年由欧洲人传入的[[天花|天花]]流行，導致本地因紐特人人口銳減。

1979年，戈特霍布更名為努克。今日的努克是一個因紐特人土著居民與丹麥移民共存的都市，格陵蘭人口的約25%居住於此。

== 人口与宗教 ==

努克是格陵兰18个行政区中人口最多的一个，岛上四分之一以上的人口定居于此<ref>[https://www.cia.gov/library/publications/the-world-factbook/geos/gl.html [[中情局|中情局]][[世界概况|世界概况]]——格陵蘭]</ref>。据2009年1月的统计数据，努克计有人口15,438人，其中男性8,162人，女性7,276人<ref name="multiple">[http://www.stat.gl/LinkClick.aspx?link=BefolkStat%2fBev%e2%80%98gelser%2fGr%c3%b8nlands+befolkning+pr.+1.+jaunuar+2009.xls&tabid=86&mid=457&language=en-US 格陵兰统计局]</ref>。

大部分人口信奉[[基督新教|基督新教]][[路德宗|路德宗]]，还有部分人口信奉[[天主教|天主教]]和新教的其他派别。

== 地理 ==

努克位於格陵蘭西岸戈特霍布[[峡湾|峡湾]]口，約在[[北極圈|北極圈]]以南240公里，北緯64°10′，西經51°43′，面積約105,000 km²<ref name="multiple" />。

此外由于此处有[[暖流|暖流]]流经，海水终年不冻，适宜捕鱼。

== 经济 ==

努克的经济以[[渔业|渔业]]为主，1960年代，鱈魚業相當發達，現已沒落，目前主要出产[[螃蟹|螃蟹]]和[[比目鱼|大比目鱼]]。欧洲最大的水产公司——[[皇家格陵兰水产公司|皇家格陵兰水产公司]]的总部即设在这里。

当地以[[丹麦克朗|丹麦克朗]]为主要流通货币。

== 交通 ==

海运方面，努克是{{Link-en|北极Umiaq航线|Arctic Umiaq Line}}上的一座重要的[[港口|港口]]。

航空方面，[[努克机场|努克机场]]位于市中心东北4公里，是[[格陵兰航空|格陵兰航空]]的重点机场，提供前往格陵兰全国各地以及丹麦首都[[哥本哈根|哥本哈根]]的航线。

== 教育 ==

[[格陵兰大学|格陵兰大学]]设在努克，是格陵兰全国唯一的高等学府。
