'''努尔-苏丹'''（{{lang-kk|Нұр-Сұлтан}}），原名'''阿斯塔納'''（{{lang|kk|Астана}}）、'''阿克莫拉'''（{{lang|kk|Ақмола}}），是[[哈萨克斯坦|哈萨克斯坦]]的[[首都|首都]]和直轄市，该国第二大城市，位於中北部的[[阿克莫拉州|阿克莫拉州]]中，但為哈薩克的[[自治市|自治市]]。[[伊希姆河|伊希姆河]]穿城而过，生态环境良好，是哈萨克斯坦工农业的主要生产基地、全国铁路交通枢纽。

1997年，哈萨克斯坦将首都自[[阿拉木图|阿拉木图]]搬迁至阿克莫拉，并将新首都改名为“阿斯塔纳”，在哈萨克语中意为“首都”。<ref name=chinanews01>{{Cite news |url=http://www.chinanews.com/gj/2016/07-06/7930087.shtml |title=阿斯塔纳：“冷都”今年已十八 |publisher=中新網 |date=2016-07-06}}</ref>

2019年3月20日，[[哈萨克斯坦议会|哈萨克斯坦议会]]通过宪法修正案，为了纪念前总统[[努尔苏丹·纳扎尔巴耶夫|努尔苏丹·纳扎尔巴耶夫]]，将首都阿斯塔纳更名为努尔-苏丹。<ref name="改名2">{{Cite web |url = http://www.chinanews.com/gj/2019/03-20/8785617.shtml |title = 哈萨克斯坦通过宪法修正案 将首都更名为努尔苏丹|website=中国新闻网 |date = 2019-03-20 |language = zh |accessdate = 2019-03-20 }}</ref><ref name="change name">{{Cite web |url = https://tass.ru/mezhdunarodnaya-panorama/6238253 |title = Парламент Казахстана принял законопроект о переименовании Астаны в Нурсултан |date = 2019-03-20 |language = ru |accessdate = 2019-03-20 }}</ref><ref name="change name2">{{Cite web |url = https://www.svoboda.org/a/29832026.html |title = одобрил переименование Астаны в Нурсултан |date = 2019-03-20 |language = ru |accessdate = 2019-03-20 }}</ref>

==词源==
城市名称努尔-苏丹为[[哈萨克斯坦|哈萨克斯坦]]第一任[[哈萨克斯坦总统|总统]][[努尔苏丹·纳扎尔巴耶夫|努尔苏丹·纳扎尔巴耶夫]]的名字。2019年[[哈萨克斯坦议会|哈萨克斯坦议会]]通过总统[[卡瑟姆若马尔特·托卡耶夫|托卡耶夫]]的提议将原首都名阿斯塔纳改为努尔-苏丹。此举意在纪念努尔苏丹·纳扎尔巴耶夫。<ref name="改名2"/>

== 歷史 ==
[[File:Astana-embankment-construction-7801.jpg|thumb]]
1824年，一隊由俄羅斯[[鄂木斯克|鄂木斯克]]出發的[[西伯利亞|西伯利亞]][[哥薩克|哥薩克]]軍隊在[[伊希姆河|伊希姆河]]河岸建立了一個城堡，以後變成了阿克莫林斯克（Akmolinsk）。在20世紀初，阿克莫林斯克成為哈薩克的[[鐵路|鐵路]]樞紐，也為阿克莫林斯克帶來經濟的影響。在[[史達林|史達林]]時期，哈薩克有11個[[勞改營|勞改營]]（[[古拉格|古拉格]]），其中最惡名昭彰的坐落於努尔苏丹附近，專門用來關押史達林政壇敵人的配偶。1961年，這城市改名為[[切利诺格勒|切利诺格勒]]（Tselinograd）。這時的努尔苏丹成為墾荒運動的中心，目標是成為[[蘇聯|蘇聯]]第二大的[[糧食|糧食]]生產地。墾荒運動時期，有大量[[移民|移民]]移入阿斯塔納，當中大多數是俄國人。

當哈薩克斯坦在1991年獨立後，這城市改稱為阿克莫拉（Aqmola，白色墳墓之義）。在1995年，為了哈薩克化北部俄羅斯人佔多數的地區，哈薩克政府把阿克莫拉預備為新的首都。於1997年，哈薩克由[[阿拉木圖|阿拉木圖]]遷都往阿克莫拉，並改稱為阿斯塔納。<ref>[http://news.bbc.co.uk/1/hi/world/asia-pacific/country_profiles/1298395.stm] BBC Kazakhstan Timeline</ref>哈薩克政府認為[[阿拉木圖|阿拉木圖]]作為首都有許多麻煩，例如受[[地震|地震]]威脅、擴充土地不便，再加上阿拉木圖離[[吉爾吉斯|吉爾吉斯]]的邊界太靠近，哈薩克政府認為遷都有助解決以上問題，但當時遷都活動卻不受矚目。<ref>{{cite book

許多哈薩克人卻不同意遷都。有人批評阿斯塔纳在哈薩克草原上很孤立，來往交通不方便，而且阿斯塔纳冬天氣候嚴峻。<ref>{{cite news|last=Green|first=Peter|title=Kazakhs Raise Curtain on Remote New Capital, but Many Ask Why|publisher=International Herald-Tribune|date=11 June 1998|url=http://www.iht.com/articles/1998/06/11/kazak.t_0.php}}</ref>事實上，阿斯塔纳是全世界第二冷的首都，僅次於[[蒙古国|蒙古]]首都[[烏蘭巴托|烏蘭巴托]]。

2017年，阿斯塔纳舉行[[2017年世界博覽會|2017年世界博覽會]]，主題為「未來能源」。參加該次[[世界博覽會|世界博覽會]]的[[國家|國家]]和[[國際組織|國際組織]]分別達到115個與22個。<ref>{{cite web|title=Number of Countries Participating in EXPO-2017 Exceeded the Plan|url=https://expo2017astana.com/en/news/2016/sentyabr/number-of-countries-participating-in-expo-2017-exceeded-the-plan|website=expo2017astana.com|deadurl=yes|archiveurl=http://archive.wikiwix.com/cache/20170910055258/https://expo2017astana.com/en/news/2016/sentyabr/number-of-countries-participating-in-expo-2017-exceeded-the-plan|archivedate=2017-09-10}}</ref>

== 地理 ==
努尔-苏丹位於廣闊的哈薩克中北部半沙漠草原，<ref name= Xinhua >{{cite web
 | url = http://news.xinhuanet.com/ziliao/2002-06/01/content_418797.htm
 | title = 哈薩克斯坦首都阿斯塔納
 | author = 楊立杰
 | publisher = 新華網
 | accessdate = 2012-07-26}}</ref>座落[[伊希姆河|伊希姆河]]旁。努尔-苏丹之所以廣大是因為位於哈薩克北部和中部草原地帶，而且[[伊希姆河|伊希姆河]]流過其中，才會有會有這麼寬敞廣闊的景觀。伊希姆河北方是努尔-苏丹的舊區，南方是發展較新的地區。努尔-苏丹海拔347公尺（1,138呎）。面积300平方公里，座標是北緯51.1度、東經71.5度。

=== 氣候 ===
[[File:Astana,_Kazakhstan.JPG|thumb]]

努尔-苏丹的年均溫是攝氏1.8度。努尔-苏丹最冷的月份是1月，月均溫度攝氏-17.3度。而最熱的月份是7月，月均溫度達攝氏20.2度。

就[[氣候學|氣候學]]上來說，努尔-苏丹是全世界第二冷的首都，僅次於[[蒙古|蒙古]]的[[烏蘭巴托|烏蘭巴托]]。在努尔-苏丹當上首都前，全世界第二冷的首都是[[加拿大|加拿大]]的[[渥太華|渥太華]]。而且努尔-苏丹是全哈薩克最冷的地方，曾經錄得最低溫度攝氏-51.6度。在冬天時，努尔-苏丹的溫度在攝氏-35度到-40度之間。而且努尔-苏丹的均溫有半年是在零度以下。總體來說，努尔-苏丹屬於溫帶大陸性濕潤氣候，但瀕臨半乾旱氣候。努尔-苏丹也有寒冷的冬天和溫暖的夏天。

== 人口 ==
根據2017年11月1日的統計，努尔-苏丹的[[人口|人口]]有1,026,897人，而[[人口密度|人口密度]]是每[[平方公里|平方公里]]1,282人。努尔-苏丹的主要民族有：[[哈薩克人|哈薩克人]]、[[俄羅斯人|俄羅斯人]]、[[烏克蘭人|烏克蘭人]]、[[德意志人|德意志人]]及[[韃靼人|韃靼人]]，並各占65.2％、23.8％、2.9％、1.7％、1.5％的人口。在1999年時，努尔-苏丹的人口約281,000人，30％是哈薩克人，70%是俄羅斯、烏克蘭、德國人等外地人。而2007年時，人口比1999年統計時還多出了一倍，有約60萬人。在1990年代時，有大量的民工從哈薩克四面八方而來，同時努尔-苏丹也是哈薩克年輕人創業的好地方。因此努尔-苏丹的人口比例大大的改變，讓原本努尔-苏丹哈薩克人的比例從30％變成60％，2017年7月4日努尔-苏丹人口已經突破一百萬。

== 經濟 ==
[[政治|政治]]與[[政府|政府]]活動是努尔苏丹最主要的[[經濟活動|經濟活動]]，同時也使努尔-苏丹自立成為一個[[經濟特區|經濟特區]]。而在遷都後，大樓工程顯得重要。因此哈薩克在10年間，建了許多政府大樓、官邸、清真寺、公園和紀念碑。

== 建築 ==
努尔-苏丹在[[蘇聯歷史|蘇聯時代]]建造的建築現已拆除，被遷都時期努尔-苏丹所蓋的[[建築|建築]]取代。總統[[努爾蘇丹·納扎爾巴耶夫|努爾蘇丹·納扎爾巴耶夫]]對努尔-苏丹的建築很關注；最近才完成的建築物都是被國際[[設計師|設計師]]認可的。

=== 巴伊傑列克觀景塔 ===
[[巴伊傑列克觀景塔|巴伊傑列克觀景塔]]是努尔苏丹最著名的地標。巴伊傑列克塔的形狀象徵著一顆[[楊樹|楊樹]]。在巴伊傑列克塔的球形頂樓有[[哈薩克斯坦總統|哈薩克斯坦總統]][[努爾蘇丹·納扎爾巴耶夫|纳扎尔巴耶夫]]的手印。在2010年夏天，巴伊傑列克塔舉辦哈薩克有史以來最盛大的露天畫展。

== 交通 ==
日本設計師[[黑川纪章|黑川纪章]]設計了[[努尔苏丹·纳扎尔巴耶夫国际机场|努尔苏丹·纳扎尔巴耶夫国际机场]]，他同時也設計了努尔-苏丹的一些建築。位于努尔-苏丹市区北部的[[阿斯塔纳站|阿斯塔纳站]]是哈薩克斯坦北部的重要樞紐，市区东南部另设有2017年6月1日启用的{{link-en|阿斯塔纳光明大道站|Astana Nurly Zhol railway station}}。<ref>{{cite news|url=http://www.inform.kz/cn/6-1_a3019583|title=阿斯塔纳新火车站起名为“光明大道” 6月1日正式投入使用|publisher=哈萨克国际通讯社|date=2017-04-21|accessdate=2017-08-11}}</ref><ref>{{cite news|url=http://www.inform.kz/cn/article_a3031527|title=纳扎尔巴耶夫总统为"光明大道"火车站剪彩|publisher=哈萨克国际通讯社|date=2017-05-31|accessdate=2017-08-11}}</ref>從努尔-苏丹出發的火車都可以到達哈薩克斯坦大部分的大城市，尤其是到阿拉木圖的特快車。努尔苏丹的國際列車可以到[[俄羅斯|俄羅斯]]、[[烏克蘭|烏克蘭]]、[[烏茲別克斯坦|烏茲別克斯坦]]和[[吉爾吉斯斯坦|吉爾吉斯斯坦]]，但大部分的車一個星期只有一班。自2008年夏天後，努尔-苏丹每個星期都有一班車開到[[中國|中國]]的[[烏魯木齊|烏魯木齊]]。
