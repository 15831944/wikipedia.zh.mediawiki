'''濁水車站'''位於[[台灣|台灣]][[南投縣|南投縣]][[名間鄉|名間鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[集集線|集集線]]的[[鐵路車站|鐵路車站]]，原名「'''名間車站'''」。

== 車站構造 ==
*[[島式月台|島式月台]]一座。

== 利用狀況 ==
*目前為丙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]。
*行駛車輛為DR1000型[[柴油|柴油]]客車，等級為[[臺鐵區間車|區間車]]。
*本站[[埃德蒙森式鐵路車票|名片式車票]]已停止販售。
*是目前集集支線唯一辦理列車交會的車站。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1922年1月14日：設「名間」驛。
*1922年3月31日：改「濁水」驛。
*1967年6月9日：站房重新修建，1999年[[921大地震|九二一大地震]]時全毀。
*1991年9月1日：由三等站直接降為招呼站，並指定由[[二水車站|二水站]]管理。
*2002年：站房再次重新修建2003年11月23日完工啟用。
*2008年5月15日：由招呼站提升為丙種簡易站。
*2008年5月20日：開始販售名片式和薄紙式車票。
*2015年6月30日：啟用多卡通刷卡機。
