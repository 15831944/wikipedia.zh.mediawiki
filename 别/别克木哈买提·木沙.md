'''别克木哈买提·木沙'''（{{bd|1939年|12月|2017年|4月9日}}），[[新疆|新疆]][[特克斯县|特克斯县]]人。[[哈萨克族|哈萨克族]]<ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1663546 |title = 78岁新疆维吾尔自治区政协原副主席别克木哈买提·木沙逝世 |author =  |date =2017-04-16   |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref><ref name=tsw>{{Cite web |url =http://news.ts.cn/content/2017-04/16/content_12598124.htm  |title = 别克木哈买提·木沙同志逝世 |author =  |date = 2017-04-16  |publisher =  天山网|language =  |accessdate =  |quote =  }}</ref>。1965年8月参加工作，1971年4月加入中国共产党。

==生平==
1960年8月，进入[[北京师范大学|北京师范大学]]化学系化学专业学习。1965年8月，任[[伊犁地区|伊犁地区]]社教工作队副组长。1967年2月，到[[伊犁哈萨克自治州|伊犁哈萨克自治州]]工交局工作。1971年4月加入[[中国共产党|中国共产党]]。1974年4月，任伊犁哈萨克自治州工交局副局长。1975年9月，任中共[[奎屯市|奎屯市]]委副书记、奎屯市革委会主任。1978年12月，任[[新疆维吾尔自治区|新疆维吾尔自治区]]化工局副局长。1983年9月，任新疆维吾尔自治区化学工业总公司总经理。1987年6月，任新疆维吾尔自治区化工局党组副书记、局长。1993年5月，任中共伊犁哈萨克自治州党委副书记、州长。1998年1月，任[[新疆维吾尔自治区政协|新疆维吾尔自治区第八届政协]]副主席，2005年6月退休<ref name=pp/><ref name=tsw/>。

2017年4月9日，在[[乌鲁木齐|乌鲁木齐]]病逝，享年78岁<ref name=pp/><ref name=tsw/>。
