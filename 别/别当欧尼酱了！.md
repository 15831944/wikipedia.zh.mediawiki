{{日本作品|别当欧尼酱了！|お兄ちゃんはおしまい!}}是由[[日本漫画家|日本漫画家]]猫豆腐（{{lang|ja|ねことうふ}}）创作的[[同人|同人]]漫画。

作品最初在日本插画网站[[pixiv|pixiv]]连载，后niconico静画也同步更新，2018年1月27日作者猫豆腐自费出版本作的繁体中文电子版<ref>{{cite web|url=http://www.acgdoge.net/archives/24336|title=开拓市场转化盗版的范例，日本漫画家自费出版中文版漫画电子版大赚一笔|date=2018-02-03|publisher=ACGdoge}}</ref>，2018年6月27日起由[[一迅社|一迅社]]进行商业化出版发行。

== 简介 ==
{{Plot_style}}
喝了妹妹研制的怪药后醒来发现自己变成女儿身的绪山真寻，与天资聪慧直接跳级上大学的妹妹绪山美波里的男性女性化实验，她们之间会发生什么样的故事呢？

== 登场人物 ==
※声优均为广播剧声优<ref>{{cite web|url=http://grin.oops.jp/onimai|title=お兄ちゃんはおしまい！特設ページ|publisher=GRINP}}</ref>
; {{nihongo|绪山真寻|緒山 まひろ（おやま まひろ）|声：[[高野麻里佳|高野麻里佳]]}}
: 本作的主人公，是个[[自宅警备员|自宅警备员]]，平时喜欢玩[[日本成人游戏|成人游戏]]，原本为男性身体，在喝了自己的妹妹研制出的一种药后变成女儿身，之后便慢慢习惯了女性身体的生活。
; {{nihongo|绪山美波里|緒山 みはり（おやま みはり）|声：[[石原夏织|石原夏织]]}}
: 绪山真寻的妹妹，天资聪慧过人，直接跳级上大学，平时在家喜欢搞一些古怪的研究，研制出了能使男性变成女性的药并在自己的哥哥身上进行了实验，对哥哥有超越兄妹的感情，名字意为“监视”。
; {{nihongo|穗月枫|穂月 かえで|声：[[金元寿子|金元寿子]]}}
: 高中生，美波里的朋友，是学校里的[[辣妹|辣妹]]，擅长做家务，很羡慕美波里的姐（兄）妹之情。
; {{nihongo|穗月椛|穂月 もみじ}}
: 穗月枫的妹妹，初中生，刚开始被真寻当成了男孩子。
