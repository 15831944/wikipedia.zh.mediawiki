'''别技军'''（{{kor|k=별기군}}），正式名称为'''教练兵队'''，是[[朝鲜历史|朝鲜历史]]上第一支[[近代化|近代化]][[军队|军队]]，由[[日本|日本]]人训练。

== 创建 ==

1876年[[李氏朝鮮|朝鲜]]开港以后，统治朝鲜的[[明成皇后|闵妃]]集团开始实行改革，军事改革是最先进行的部分。1881年（[[朝鲜高宗|高宗]]十八年）四月二十三日，在[[日本|日本]]驻朝公使[[花房义质|花房义质]]和兵曹判书[[闵谦镐|闵谦镐]]的建议下，朝鲜王朝正式设立了别技军（“别技军”即精锐部队之意），教授新式武器及战法，由[[日本|日本]][[陆军|陆军]][[工兵|工兵]]少尉[[堀本礼造|堀本礼造]]担任教官，负责训练别技军。<ref>《高宗实录》卷18，十八年四月二十三日条：“殿讲入侍时，经理事[[闵谦镐|闵谦镐]]曰：‘间者日使花房义质，抵书于臣及礼判洪佑昌。而“简选兵士，操炼戎务，为今日急务。荐引陆军少尉堀本礼造以为训教之师。亦要择定敎场从事肄习”云矣。此盖出于厚谊，恐难辜负其意。各营兵士之健壮者，抄出点取，别定将领，而统率练习。教场可合处，亦为方便择定之意。臣与礼判商议于总理大臣及经理诸堂，有所讲确，故敢达矣。’[[朝鲜高宗|教]]曰：‘依为之。选兵操练等事，以节目磨炼；而将领之任，勿拘文荫武择差可也。’”</ref>

== 概况 ==

别技军兵源来自[[朝鲜|朝鲜]]京军五营（训练都监、龙虎营、禁卫营、御营厅、总戎厅），从中选拔80名士兵来组成别技军。别技军由[[闵妃|闵妃]]的侄子[[闵泳翊|闵泳翊]]掌管，任教练所堂上；正领官为[[韩圣根|韩圣根]]，左副领官为[[尹雄烈|尹雄烈]]，右副领官为[[金鲁莞|金鲁莞]]，参领官为[[禹范善|禹范善]]。

别技军以小铳和洋枪为主要武装，训练地点是[[汉城|汉城]]（今[[首尔|首尔]]）[[昭义门|西大门]]外的下都监。此后人数逐渐增加，到[[壬午兵变|壬午兵变]]前夕时达到400人。作为军事改革的另一部分，[[闵妃|闵妃]]集团还从京城[[两班|两班]]子弟中选出100多人，称为“士官生徒”，前往[[日本|日本]][[户山军校|户山军校]]进接受军事教育和训练，以作为别技军军官的后备力量。

== 波折 ==
{{main|壬午兵变}}

别技军作为一支新式军队，成为了[[闵妃|闵妃]]集团的宠儿，对其特别重视和优待。与此同时，旧式军队的地位却迅速下降，与别技军形成鲜明对比。别技军装备先进，衣装整齐，军饷丰厚，而旧式军队的待遇十分糟糕，竟10多个月欠发军饷。不仅如此，闵妃集团还决定扩充别技军，减少旧军队，将京军五营缩编为武卫、壮御两营，半数旧式军人被迫解甲。在这种情况下，别技军与旧式军队的矛盾日益尖锐。

不仅如此，别技军与[[朝鲜|朝鲜]]百姓的矛盾也不断激化。本来别技军由[[日本|日本]]人训练，许多朝鲜人对此非常不满，因此民间都俗称其为“'''倭别技'''”。而且别技军训练时“荷铳跳踢，尘粉蔽空”，附近居民生活深受其干扰，对其更加憎恨。<ref>黄玹.《梅泉野录》：[[国史编纂委员会|国史编纂委员会]]，1955年：56.</ref>因此别技军虽然为[[近代化|近代化]]的新式军队，却在朝鲜不得人心，处于非常孤立的地位。

到了1882年7月23日（[[朝鲜高宗|高宗]]十九年六月初九日），由于拖欠军饷13个月及反感别技军，旧式军人终于发动暴动，大量[[汉城|汉城]]市民参与进来，史称“[[壬午兵变|壬午兵变]]”。别技军作为壬午兵变的祸首之一而遭到攻击，旧式军队在兵变当天下午围攻别技军所在的下都监。闵妃集团企图依靠别技军来压制哗变军队，正巧这日别技军放假，400名别技军大部分回家了，只有20多名别技军士兵和[[堀本礼造|堀本礼造]]等负责人。哗变士兵和附近民众很快击破别技军，杀死了营官[[丁龙燮|丁龙燮]]，砸毁洋枪。堀本礼造趁乱逃跑，起义士兵和市民一路追赶，最终[[堀本礼造|堀本礼造]]被汉城市民[[孙顺吉|孙顺吉]]、[[孔致元|孔致元]]以及下都监的杂役[[尹顺龙|尹顺龙]]活活打死。<ref>权锡奉．《壬午军变》：[[国史编纂委员会|国史编纂委员会]]，1975年．</ref>

壬午兵变导致[[闵妃|闵妃]]集团倒台，[[朝鲜高宗|高宗]]生父[[兴宣大院君|兴宣大院君]]上台执政。大院君下令裁撤别技军。既而闵妃引来[[清朝|清朝]]军队平定兵变，别技军由得以恢复，但元气大伤。1883年（高宗二十年）八月一日，别技军被合并进御营厅，至此退出了历史舞台。<ref>《高宗实录》卷20，二十年八月初一日条。</ref>

== 意义 ==

别技军是[[朝鲜历史|朝鲜历史]]上最早的[[近代化|近代化]]新式軍隊，代表著處在新時代的潮流中原本保守鎖國的朝鮮王朝嘗試接受開化走向近代化的象徵，但施行的缺失卻也成了[[壬午兵变|壬午兵变]]發生的原因之一，雖然遭遇到了守舊派的反撲而嘎然而止，但對於朝鮮王朝來說是在[[甲午更張|甲午更張]]前一次別具意義的里程碑。
