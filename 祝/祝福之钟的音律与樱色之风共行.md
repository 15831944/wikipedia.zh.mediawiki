'''《祝福之钟的音律与樱色之风共行》'''（{{jpn|j=祝福の鐘の音は、桜色の風と共に。}}）是由{{lang|ja|{{Tsl|ja|すたじお緑茶}}}}于2012年10月26日发行的[[18禁|18禁]][[恋爱冒险游戏|恋爱冒险游戏]]。<ref>{{cite web|url=http://www.getchu.com/soft.phtml?id=745451|script-title=ja:祝福の鐘の音は、桜色の風と共に。|publisher=Getchu.com|accessdate=2014年3月20日|language=ja}}</ref>

== 故事简介 ==

主人公堀田秀治和音乐家父亲堀田祐平相依为命，过着贫苦的生活。有一天，秀治的外祖父鳳源三出现了。说是要秀治来继承家业，并和他的表親麻里亚许下婚约。<br />
婚约不是绝对的，但一年之内秀治必须找到合适的妻子，否则就要和麻里亚正式结婚。

== 登场人物 ==

; {{nihongo|堀田秀治|堀田　秀治（ほった　しゅうじ）||声：无|}}
: 故事主人公。因为长期的贫困生活，养成了省吃俭用的习惯。温柔大方，略喜欢开玩笑。
; {{nihongo|凤麻里亞|鳳　まりあ（おおとり　まりあ）||声：[[五行薺|五行薺]]<ref>{{cite web|url=http://www.studio-ryokucha.com/syukufuku/character.html|script-title=ja:鳳　まりあ|publisher={{lang|ja|{{Tsl|ja|すたじお緑茶}}}}|accessdate=2016年12月21日|language=ja}}</ref>|}}
: 凤家大小姐，是秀治的表姐妹。做事认真，以身为凤家之女为荣。因此很排斥突然成为未来家主的秀治。努力型的高才生，经常背着他人独自努力。
; {{nihongo|西九条奏音|西九条　奏音（にしくじょ　かのん）||声：{{lang|ja|[[西野みく|西野みく]]}}<ref>{{cite web|url=http://www.studio-ryokucha.com/syukufuku/character_02.html|script-title=ja:西九条　奏音|publisher={{lang|ja|すたじお緑茶}}|accessdate=2016年12月21日|language=ja}}</ref>|}}
: 双亲是著名音乐家。曾多次在各种大赛中优胜，钢琴拿手，前途无量。一年级生，常在音乐室練習钢琴。
; {{nihongo|北园纱夜|北園　紗夜（きたぞの　さや）||声：{{lang|ja|[[咲ゆたか|咲ゆたか]]}}<ref>{{cite web|url=http://www.studio-ryokucha.com/syukufuku/character_03.html|script-title=ja:北園　紗夜|publisher={{lang|ja|すたじお緑茶}}|accessdate=2016年12月17日|language=ja}}</ref>|}}
: 知名政治家的孙女，三年级學姐。身材矮小，满面稚气，贫乳。对“小学生”这个词语很敏感。非常路痴，校内广播经常出现关于她的寻人启事。北園家与凤家的爷爷之间水火不容。
; {{nihongo|东云麗|東雲　うらら（しののめ　うらら）||声：[[櫻川未央|櫻川未央]]<ref>{{cite web|url=http://www.studio-ryokucha.com/syukufuku/character_04.html|script-title=ja:東雲　うらら|publisher={{lang|ja|すたじお緑茶}}|accessdate=2016年12月21日|language=ja}}</ref>|}}
: 秀治的同班女生。喜欢大幅修饰校服而被教师惩罚。对秀治一见钟情并黏住秀治不走。父亲是近期迅速崛起的金融公司的经理。
; {{nihongo|四谷隼人|四谷　隼人（よつや　はやと）||声：{{lang|ja|[[佐藤タカ|佐藤タカ]]}}<ref>{{cite web|url=http://www.studio-ryokucha.com/syukufuku/character_05.html|script-title=ja:四谷　隼人|publisher={{lang|ja|すたじお緑茶}}|accessdate=2016年12月21日|language=ja}}</ref>|}}
: 秀治的同班同学。知名银行的公子。喜欢勾引女生，却实在无法让人讨厌他。
; {{nihongo|青野七歌|青野　七歌（あおの　ななか）||声：{{lang|ja|[[有栖川みや美|有栖川みや美]]}}<ref>{{cite web|url=http://www.studio-ryokucha.com/syukufuku/character_06.html|script-title=ja:青野　七歌|publisher={{lang|ja|すたじお緑茶}}|accessdate=2016年12月21日|language=ja}}</ref>|}}
: 凤家的女仆。为秀治服务。以女仆为荣，却经常出差错。性格十分活泼。
