'''焰色反應'''，或稱作'''焰色測試'''及'''焰色試驗'''、'''焱色試驗'''，是[[化學|化學]]上用來測試某種[[金屬|金屬]]是否存在在於化合物的方法，该反应为物理变化。

其原理是每種元素都有其個別的[[光譜|光譜]]。樣本通常是粉或小塊的形式。以一條清潔且對化學惰性的金屬線（例如[[鉑|鉑]]或[[鎳|鎳]][[鉻|鉻]]合金）盛載樣本，再放到[[本生燈|本生燈]]的無光焰（藍色火焰）中。

由於鈉的金黃色火焰容易蓋過其他金屬的焰色，所以常用{{le|鈷藍玻璃|Cobalt glass}}觀察其他金屬的焰色。
==历史==
中国[[南北朝|南北朝]]时期，著名的炼丹家和医药大师[[陶弘景|陶弘景]]在他的《[[本草经集注|本草经集注]]》中记载“以火烧之，紫青烟起，云是真硝石(硝酸钾)也”。说明人们很早就知道用焰色反应鉴别硝酸钾<ref>{{cite web|author1=岳晓明|title=焰色反应小史及其局限性|url=http://chem.cersp.com/JXZY/JXSC/200707/3714.html|website=中国化学课程网|accessdate=2016-09-06|date=2007-07-16|deadurl=yes|archiveurl=https://web.archive.org/web/20081025035523/http://chem.cersp.com/JXZY/JXSC/200707/3714.html|archivedate=2008-10-25}}</ref>。

==焰色試驗的步驟==
===鉀的焰色測試===
# 先準備一支[[鉑|鉑]]或[[鎳|鎳]][[鉻|鉻]]線，[[鈷藍玻璃|鈷藍玻璃]]及鹽或其溶液。
# 把線圈浸在濃[[氫氯酸|氫氯酸]]中以清除先前餘下的物質，再把線放在無光焰（藍色火焰）中直至沒有顏色的變化。
# 用[[蒸餾水|蒸餾水]]或[[去離子水|去離子水]]或純水沖洗線圈。
# 用線圈接觸鹽或溶液，通過無光焰（藍色火焰）中加熱。 
# 當鈉離子存在於所測試的溶液中，用[[鈷藍玻璃|鈷藍玻璃]]過濾鈉離子的焰色。
# 最後將觀察焰色。鉀的焰色為明亮的紫紅色火焰。

===銅的焰色測試===
# 先準備一支[[鉑|鉑]]或[[鎳|鎳]][[鉻|鉻]]線圈及[[硫酸銅|硫酸銅]]或其溶液。
# 把線圈浸在濃鹽酸（或其它強酸）中以清除先前餘下的物質，再把線圈通過無光焰（藍色火焰）中直至沒有顏色的變化。
# 用[[蒸餾水|蒸餾水]]或[[去離子水|去離子水]]或純水沖洗線圈。
# 用線圈蘸上[[硫酸銅|硫酸銅]]或其溶液，通過無光焰（藍色火焰）中加熱。 
# 本測試不必使用鈷藍玻璃過濾離子焰色。
# 最後將觀察焰色，銅的焰色為藍綠色。
