'''嶺腳車站'''位於[[臺灣|臺灣]][[新北市|新北市]][[平溪區|平溪區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[平溪線|平溪線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*岸式月台一座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]。
*行駛車輛為DR1000型[[柴油|柴油]]客車，等級為[[臺鐵區間車|區間車]]。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1929年10月1日：設立，初名'''嶺腳寮'''。
*1962年12月15日：與[[保安車站|保安]]、[[菁桐車站|菁桐]]、[[十分車站|十分]]、林榮（廢止）、[[南平車站_(臺灣)|南平]]、月眉(月美)等車站同時改稱今名。
*1991年10月15日：由簡易站降為招呼站，並指定由[[十分車站|十分站]]管理。
*2001年7月16日：管理站由[[十分車站|十分站]]改為[[瑞芳車站|瑞芳站]]。
*2013年9月30日：啟用多卡通刷卡機。
