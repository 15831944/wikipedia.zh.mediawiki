'''义乌市'''位于[[中国|中国]][[浙江省|浙江省]][[金华市|金华市]]中部。义乌为[[浙江省|浙江省]]综合实力第三大[[县级行政区|县市]]，也是中国经济发达县市之一，其综合竞争力[[中国百强县#县域经济论坛评价|2010年位居浙江省第三位，全国百强县（市）第八位]]。2005年，[[联合国|联合国]]与[[世界银行|世界银行]]、[[摩根士丹利|摩根士丹利]]公司等多家世界权威机构联合发布《震惊世界的中国数字》报告中，[[义乌国际商贸城|义乌市场]]被称为“全球最大的小商品批发市场”。[[义乌小商品博览会|义乌小商品博览会]]已成为[[广交会|广交会]]、[[中国华东进出口商品交易会|华交会]]之后国内第三大贸易类展会。2010年，义乌被设为国际贸易方面的[[国家综合配套改革试验区|国家综合配套改革试验区]]。<ref>[http://news.sina.com.cn/c/2011-03-10/054322085145.shtml 义乌获批国际贸易综合改革试点成为我国第十个经济新特区]</ref>

== 历史沿革 ==
* [[春秋|春秋]]时属[[越国|越国]]。前621年，越侯[[无壬|无壬]]把都城迁到[[嶕岘|嶕岘]]，据考其位于义乌稠城。越侯[[允常|允常]]在前538年迁出到[[勾嵊|勾嵊]]。
* [[秦|秦]]王[[嬴政|嬴政]]（秦始皇）二十五年（前222年）建县名乌伤，属[[会稽|会稽]]郡。传说秦时有个颜乌，事亲至孝，父死后负土筑坟，一群乌鸦衔土相助，结果乌鸦嘴喙皆伤，故称乌伤县。[[新莽|新莽]]时（9年）改县名乌孝。
* [[东汉|东汉]][[建武|建武]]初年复称乌伤。曾为[[会稽|会稽]]西部都尉治。初平三年（192年）分割西部辖境，设置长山县（即后之金华县）。
* [[三国|三国]][[吴|吴]][[赤乌|赤乌]]八年（245年）分南境，置永康县。宝鼎元年（266年），分会稽郡西部设东阳郡（郡治长山），乌伤县属东阳郡。
* [[隋|隋]][[开皇|开皇]]九年（589年），分割吴州置[[婺州|婺州]]。
* [[唐|唐]][[武德|武德]]四年（621年）于乌伤县置稠州，并分置乌孝、华川二县。[[武德|武德]]七年（624年）废稠州，合乌孝、华川为一县，改名义乌县。稠州以稠山（德胜岩）而得名。华川又名绣川，以[[绣湖|绣湖]]得名。义乌其义与乌伤、乌孝同。唐[[垂拱|垂拱]]二年（686年）析义乌县东境设东阳县。[[天宝|天宝]]十三年（754年），又分县境北部及兰溪、富阳各一部分，设浦阳县（今浦江县）。
* [[元|元]]代，义乌隶属婺州路总管府。[[至正|至正]]十八年（1358年），吳王[[朱元璋|朱元璋]]部攻取婺州，改婺州路为宁越府。[[至正|至正]]二十二年（1362年）又改名金华府。
* [[明|明]][[嘉靖|嘉靖]]三十八年（1559年）[[戚继光|戚继光]]的[[戚家军|戚家军]]成军于义乌、[[永康市|永康]]一帶，总兵力四千人，主力是义乌、永康的农民和矿工。自成军起，大小数百战未尝败绩。
* 明清仍旧，义乌隶属关系未变。
* [[辛亥革命|辛亥革命]]后，废府制代以道制，义乌属[[金华道|金华道]]。1927年废道制改为[[省县两级制|省县两级制]]，义乌直属[[浙江|浙江]]省。后设行政督察专员公署，义乌属金华专区或浙江省第四专区。
* 1949年5月8日义乌[[解放|解放]]。[[中华人民共和国|中华人民共和国]]成立后，义乌属金华专区。
* 1959年[[浦江县|浦江县]]并入义乌，1967年仍析出。
* 1988年撤销义乌县，设立义乌市（县级）。<ref>[http://www.yw.gov.cn/glb/ywgl/lsyg/200901/t20090107_166758.html 义乌历史沿革] {{webarchive|url=https://web.archive.org/web/20090216223011/http://yw.gov.cn/glb/ywgl/lsyg/200901/t20090107_166758.html |date=2009-02-16 }}</ref>

== 地理 ==
义乌地处[[金衢盆地|金衢盆地]]的东侧边缘，境内主要河流有[[义乌江|义乌江]]。

== 交通 ==
[[File:義烏機場.jpg|缩略图]]]]
重要公路有[[320国道|320国道]]、[[沪昆高速公路|沪昆高速公路]]。

过境铁路有[[沪昆铁路|沪昆铁路]]、[[沪昆客运专线|沪昆客运专线]]，在城北设有[[义乌站|义乌站]]，另有规划中[[甬金铁路|甬金铁路]]（宁波-金华）过境。2014年12月9日，第一趟连接义乌和欧洲的“义新欧”货运列车抵达[[西班牙|西班牙]][[马德里|马德里]]。义乌至欧洲的货运铁路服务宣告贯通，不仅让沿线的12个国家成为[[新丝绸之路经济带|新丝绸之路经济带]]建设的参与者和受益者，也将带来大量的贸易机遇。<ref>[http://www.guancha.cn/Project/2014_12_10_303024.shtml 1.3万公里横跨欧亚 首趟"义新欧"班列抵达马德里]</ref>

城北有[[义乌机场|义乌机场]]，是浙中地区唯一的航空港。

==人口==

2014年末全市户籍人口766604人。<ref name="2014统计公报"/>

== 语言 ==
义乌市通用[[方言|方言]]为[[义乌话|义乌话]]，属于[[吴语|吴语]][[金衢片|金衢片]]。由于义乌交通方便，与外地交往频繁，因此义乌话受邻近方言的影响较大，内部差异也较明显，故有“义乌十八腔，隔溪不一样”的说法。<ref>{{cite book |author= 方松熹 |location =  | publisher = 浙江省新聞出版局 |year= 2000 |title= 義烏方言研究 |pages=4 |isbn= }}</ref>

义乌在2008年中国综合实力[[百强县|百强县]]排名第八，2008年义乌城镇居民人居收入达到全国县级市第一。义乌城市规模与经济实力等多方面均已超过了金华市区，而且和周边的[[东阳|东阳]]、[[永康市|永康]]等县市联系密切。2010年，义乌被设为国际贸易方面的国家综合配套改革试验区。成立于1992年的[[义乌经济技术开发区|义乌经济技术开发区]]在2012年3月成为[[国家级经济技术开发区|国家级经济技术开发区]]。2014年全市完成地区生产总值968.6亿元，人均生产总值达126907元。<ref name="2014统计公报"/>

===农业===
义乌农业发达，主产[[水稻|水稻]]、[[甘蔗|甘蔗]]等，特产主要有[[南枣|南枣]]、[[蜜枣|蜜枣]]、[[红糖|红糖]]、[[火腿|火腿]]、[[豆腐皮|豆腐皮]]、[[腐竹|腐竹]]、[[白字酒|白字酒]]、[[顶陈酒|顶陈酒]]、[[山花梨|山花梨]]，也是[[金华火腿|金华火腿]]主要产地之一。

===工业===
依托于小商品市场的发展，义乌建立起了较多的制造业企业，所生产的产品大都通过小商品市场销往中国国内和海外，实现了“贸工联动”。2014年全市实现规模以上工业总产值847.1亿元，销售产值805.1亿元。<ref name="2014统计公报"/>

===第三产业===
义乌拥有闻名遐迩的小商品市场，目前已成为世界上最大的小商品集散地，其市场的规模和成交量均居中国各专业市场之首。<ref>{{cite web |url=https://money.udn.com/money/story/5635/2431558 |title=文交會 顯現義烏小商品之都的轉型升級 |publisher=聯合財經網 |date=2017年4月29日 }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 教育 ==
