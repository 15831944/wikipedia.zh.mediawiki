'''义和团纪念馆'''，位于[[河北省|河北省]][[邢台市|邢台市]][[威县|威县]]义和广场南端，是展示[[义和团|义和团]]历史的博物馆。

==简介==
威县是义和团运动的发源地，有[[义和拳议事厅旧址|义和拳议事厅旧址]]等多处史迹。在纪念[[义和团运动|义和团运动]]100周年之际，威县投资800多万元人民币兴建义和团纪念馆，2003年10月建成并对外开放，是县级革命纪念类纪念馆，也是中国唯一全面介绍义和团运动的大型纪念场馆。纪念馆位于威县县城义和广场南端，占地40亩，建筑面积5000平方米。馆外有[[中国史学会|中国史学会]]、中国义和团研究会立的“义和团运动一百周年纪念碑亭”和“义和团著名领袖赵三多纪念亭”。馆内陈列面积2000平方米，分为三个展厅。序厅陈列“民族之魂”大型群雕，第一、第二展厅用图片、实物、文字、景观等手段介绍了义和团运动的始末。馆藏有关义和团运动的兵器、乐器、图书、文字资料。其中图片500多张，文物260件。“重修张家庄保缮寨碑”是国家二级文物，“万宝符衣”是国家三级文物<ref>{{Cite web |url =  http://www.sohu.com/a/126301432_372971|title = 义和团纪念馆 |author =  |date =2018-03-06  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref><ref name=wx>{{Cite web |url = http://wxyht.com.cn/zgdt/ |title =展馆简介  |author =  |date =  |publisher = 威县义和团纪念馆网上展馆 |language =  |accessdate = 2018-03-06 |quote =  }}</ref>。

义和团纪念馆先后被[[中共河北省委|中共河北省委]]、[[河北省人民政府|河北省人民政府]]命名为“河北省爱国主义教育基地”、被河北省人民政府、[[河北省军区|河北省军区]]命名为“河北省国防教育基地”。2012年被评为“全国AA级旅游景区”<ref name=wx/>。
