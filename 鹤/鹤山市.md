'''鹤山'''（官方音譯：{{lang|en|Heshan}}，传统外文：{{lang|en|Hokshan}}）是[[中华人民共和国|中国]][[广东省|广东省]][[江門市|江門市]]下轄[[縣級市|縣級市]]，位于广东省南部[[珠江三角洲|珠江三角洲]]腹地，与[[佛山市|佛山市]][[南海区|南海区]]、[[顺德区|顺德区]]隔[[西江|西江]]相望，因市内有山形似仙鹤而得名。鹤山的[[地理坐标|地理坐标]]在[[北纬|北纬]]22.29°~22.52°，[[东经|东经]]112.28°~113.25°。

== 歷史 ==
[[清|清]][[雍正|雍正]]九年七月己丑（1731年8月30日）划新会，开平两地部分地区建立县制，因縣城北有山形似鶴而得名鶴山。1958年与[[高明区|高明]]合併為高鶴縣。1981年12月7日撤銷高鶴縣，恢復高明和鶴山兩縣。1993年撤縣建县级市。

== 語言 ==
鹤山市全境通行标准[[粵語|粵語]]和[[普通话|普通话]]，本地方言有鹤山话、[[四邑方言|四邑方言]]、[[客家話|客家話]]三种。从地域分布来看，北部龙口、桃源、沙坪、古劳一带通行鹤山话（沙坪话、古劳话），雅瑶一带通行四邑方言（雅瑶话），中部鹤城、共和等地通行客家话，

1980年以後外來人口大量增加，使得[[四邑方言|四邑方言]]使用人口比例下降。造成不少家庭一家多语。今日一個鹤山[[客家|客家]]移民家庭的語言情況很可能是：祖父母讲[[客家話|客家話]]，父母兩人用[[四邑方言|四邑方言]]對話，子女私下以[[標準粤语|標準粤语]]交談，子女在校用[[普通话|普通话]]溝通。

== 地理 ==
鶴山市地處珠江三角洲西側，中部較高，東南低平，東部[[西江|西江]]沿岸為[[沖積平原|沖積平原]]。

=== 氣候 ===
1月份均溫12.9℃，七月均溫28.3℃，年均降水量1,800毫米。属于[[熱帶季風氣候|熱帶季風氣候]]，境内具有[[海洋气候|海洋气候]]特征。

== 經濟 ==
鶴山市是廣東省商品糧生產基地。
*農產品：[[稻|稻]]、[[花生|花生]]、[[茶葉|茶葉]]、[[煙草|煙草]]、[[甘蔗|甘蔗]]、[[黃麻|黃麻]]、[[水果|水果]]、[[蠶桑|蠶桑]]等。
*工業：[[紡織|紡織]]、[[機械|機械]]、[[製衣|製衣]]、[[食品|食品]]、[[建材|建材]]、[[水泥|水泥]]、[[造紙|造紙]]、[[印刷|印刷]]、[[造船|造船]]、[[電子|電子]]、[[化工|化工]]等。
2005年鶴山市生產總值87億7千萬元[[人民幣|人民幣]]，人均生产总值19321人民币。
