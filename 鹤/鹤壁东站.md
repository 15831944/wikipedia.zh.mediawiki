'''鹤壁东站'''为[[京广高速铁路|京广高速铁路]]的一座车站，于2012年12月26日开通。

== 简介 ==
[[File:Platforms_1-2_of_Hebidong_Railway_Station_(20170607134435).jpg|thumb]]
东站站房长173米、宽43米，局部三层，有4个站台<ref>{{cite web |url=http://www.hebiw.com/News/116/20121122/158122 |language= |title=鹤壁东站站房建设接近尾声 年底前投用 |publisher=鹤壁网 |date=2012-11-22 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20121213172822/http://www.hebiw.com/News/116/20121122/158122 |archivedate=2012-12-13 }}</ref>。候车室内共有800多个座椅<ref>{{cite web |url=http://www.hebiw.com/News/117/20121213/158910 |language= |title=鹤壁东站候车室内的800多个座椅安装完毕 |publisher=鹤壁网 |date=2012-12-13 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20121221104536/http://www.hebiw.com/News/117/20121213/158910 |archivedate=2012-12-21 }}</ref>。站前广场占地10.5万平米，于2012年12月10日完成。<ref>{{cite web |url=http://www.hebiw.com/News/116/20121126/158247 |language= |title=鹤壁东站站前广场预计12月10日交付使用 |publisher=鹤壁网 |date=2012-11-26 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20121213172655/http://www.hebiw.com/News/116/20121126/158247 |archivedate=2012-12-13 }}</ref>
