'''鹤泊站'''（{{jpn|j=鶴泊駅|hg=つるどまりえき|rm=Tsurudomari eki|f=y}}）位于[[青森县|青森县]][[北津轻郡|北津轻郡]]鹤田町大字鹤泊字梅林，是[[东日本旅客铁道|东日本旅客铁道]]（JR東日本）管辖的[[五能线|五能线]]上的[[鐵路車站|鐵路車站]]<ref name="station"/>。

== 历史 ==
* 1918年9月25日 - 车站投入运营。开业时的地址为北津轻郡六乡村大字野中字梅林93番2号<ref>出处:『六郷村史』後編322～323頁 「第十一章官公署 第五節鶴泊驛」（五所川原市立图书馆阅览）。</ref>。
* 1971年10月1日 - 车站无人化改造。
* 1987年4月1日 - [[国铁分割民营化|国铁分割民营化]]后成为JR东日本车站。

== 车站结构 ==
[[侧式站台|侧式站台]]1台1线的[[地上车站|地上车站]]。

[[五所川原站|五所川原站]]管理的无人站<ref name="station"/>。

== 使用情况 ==
2014年日均乘车人数42人。
