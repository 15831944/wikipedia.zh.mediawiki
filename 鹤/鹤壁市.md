'''鹤壁市'''是[[中华人民共和国|中华人民共和国]][[河南省|河南省]]下辖的[[地级市|地级市]]，位于河南省北部。市境东、西、北三面皆被[[安阳市|安阳市]]包围，南面与[[新乡市|新乡市]]邻接。地处[[太行山脈|太行山脈]]东南之丘陵与平原区间，[[卫河|卫河]]及[[淇河|淇河]]为境内主要河流。鹤壁是河南省重要的[[煤炭|煤炭]]城市，因煤而兴。全市总面积2,140平方公里，人口160.6万。

== 历史 ==
=== 名称典故 ===
“鹤壁”一名最早见于《[[金史|金史]]》，相传古时有双鹤栖于南山之峭壁，逐名其山为“鹤山”，其村为“鹤壁”。亦有一说为其名源于道士在饭店[[影壁|影壁]]上所画白鹤可起舞的神话传说。

=== 建制沿革 ===
[[商代|商代]]最后一位君主[[帝辛|帝辛]]（被后人贬称[[商纣王|商纣王]]）定都于'''沫邑'''，后改为'''[[朝歌|朝歌]]'''，即今[[淇县|淇县]]。商末，帝辛与[[周武王|周武王]]决战于此。武王灭商后，封帝辛之子[[武庚|武庚]]为诸侯，统治朝歌一带，并派[[管叔|管叔]]、[[蔡叔|蔡叔]]、[[霍叔|霍叔]]监视其举动，称“[[三监|三监]]”。周武王封[[康叔|康叔]]为卫侯，[[春秋时期|春秋]]初为[[卫国|卫国]]属地，朝歌曾为卫国都城，后被[[晋国|晋]]所占。《[[左传|左传]]》记曰：[[鲁襄公|鲁襄公]]二十三年（前550年）[[齐国|齐]]“伐晋，取朝歌”。[[齐景公|齐景公]]三十八年（前510年）设'''中牟邑'''，即今鹤壁市山城区。[[三家分晋|三家分晋]]后为[[赵国|赵国]]地。[[赵献子|赵献子]]元年（前423年），赵国将都城迁至中牟邑。后被[[魏国_(战国)|魏]]所占。

[[汉代|汉代]]至[[三国|三国]]分属[[河內郡_(中國)|河内郡]][[荡阴县|荡阴县]]（今鹤壁市区）、[[朝歌县|朝歌县]]（今淇县）及[[魏郡|魏郡]][[黎阳县|黎阳县]]（今[[浚县|浚县]]）。[[东汉|东汉]][[建安|建安]]十七年（212年）荡阴县、朝歌县改属魏郡。[[西晋|西晋]][[泰始|泰始]]元年（265年）朝歌县属[[汲郡|汲郡]]。[[东晋|东晋]][[永和|永和]]七年（351年）于黎阳县置[[黎阳郡|黎阳郡]]。[[太元|太元]]十四年（389年）废黎阳郡复黎阳县入汲郡。

[[北魏|北魏]][[天兴|天兴]]四年（401年）析[[冀州_(古代)|冀州]]置[[相州|相州]]，治[[邺县|邺县]]（今[[河北省|河北省]][[临漳县|临漳县]]邺镇）。[[永安_(北魏)|永安]]元年（528年）分黎阳县置[[东黎县|东黎县]]。[[东魏|东魏]][[天平_(东魏)|天平]]元年（534年）改相州为[[司州|司州]]，废荡阴县入[[邺县|邺县]]，于黎阳县置[[黎州|黎州]]。天平二年（535年）置[[魏德县|魏德县]]。[[北齐|北齐]]年间废魏德县、东黎县。[[北周|北周]][[建德|建德]]六年（577年）司州复为相州。[[宣政|宣政]]元年（578年）分汲郡置[[卫州|卫州]]，治枋头城（今浚县西南）。[[大象_(北周)|大象]]二年（580年）相州移治安阳城（今[[安阳市|安阳市]]）。

[[隋朝|隋]][[开皇|开皇]]六年（586年）复置荡阴县。开皇十六年（596年）改荡阴县为[[荡源县|荡源县]]，属黎州。[[大业|大业]]二年（606年）改朝歌县为[[卫县|卫县]]，改卫州为汲郡，废黎州。[[唐朝|唐]][[武德|武德]]元年（618年）复置卫州，卫县属之，次年于黎阳县复置黎州。武德六年（623年）改荡源县为[[汤阴县|汤阴县]]。[[贞观|贞观]]元年（627年）卫州移治汲县（今[[卫辉市|卫辉市]]）。[[北宋|北宋]][[熙宁|熙宁]]六年（1073年）废卫县，[[元祐|元祐]]初复置。[[政和|政和]]五年（1115年）改黎州为[[濬州|濬州]]，仍治黎阳县。[[宣和|宣和]]初汤阴县改属濬州，后复属相州。[[金朝|金]][[明昌|明昌]]三年（1192年）升相州为[[彰德府|彰德府]]。

[[元朝|元]][[元宪宗|宪宗]]五年（1255年）于卫县西鹿台乡置[[淇州|淇州]]，属[[卫辉路|卫辉路]]，卫县废为集，并置[[临淇县|临淇县]]；濬州属[[大名路|大名路]]；汤阴县属[[彰德路|彰德路]]。[[至元|至元]]三年（1266年）废临淇县入淇州。[[明朝|明]][[洪武|洪武]]元年（1368年）改淇州为[[淇县|淇县]]，属[[卫辉府|卫辉府]]；改濬州为[[濬县|濬县]]，属[[大名府|大名府]]；汤阴县属[[彰德府|彰德府]]。[[清代|清代]]因之。[[民国|民国]]二年（1913年）废府存县，淇县、濬县、汤阴县均属[[河南省|河南省]][[河北道|河北道]]。1928年废道制。1932年，三县属河南省第三行政督察区。

[[中华人民共和国|中华人民共和国]]建立后，濬县、淇县、汤阴三县属[[平原省|平原省]][[安阳专区|安阳专区]]。1953年平原省撤销，安阳专区改属河南省。1954年，淇县并入汤阴县。1956年9月建鹤壁矿区。1957年3月，析汤阴县7个乡及3个村之地置鹤壁市，为[[省辖市|省辖市]]，委托安阳专区领导，后改为省直接领导。1958年12月，安阳专区撤销，鹤壁市归[[新乡专区|新乡专区]]领导。1960年10月，安阳专区的汤阴县划归鹤壁市。1961年11月，汤阴县又复归安阳专区；同年设[[山城区|山城]]、[[鹤山区|鹤山]]两个[[市辖区|市辖区]]。1961年12月恢复安阳专区，鹤壁市又改属之。1962年恢复淇县建制。1966年设立郊区。1986年1月，将[[安阳市|安阳市]]的浚县、淇县划归鹤壁市管辖。2001年12月，郊区更名为[[淇滨区|淇滨区]]。

=== 市区搬迁 ===
市区经历了三次搬迁。1957年，市区选在鹤壁集，随着二矿、三矿的建设，1957年12月市区南移到中山；后又随着五矿、六矿的建设，1959年，市区又从中山迁到现在的老区大湖。随着矿区的发展，市区的周围基本上都是塌陷区，城市的发展受到制约。1992年，鹤壁在东南浚县淇县交界处建立淇滨经济开发区，随着规模的扩大，1999年5月市政治中心迁到淇滨区，目前新区已成为鹤壁市新的政治、经济、文化中心<ref>{{Cite web|url=http://www.xzqh.org/html/show.php?contentid=12385|title=鹤壁市历史沿革|publisher=行政区划网|date=2009-12-06|accessdate=2011-08-14}}</ref>。

== 地理 ==
鹤壁位于河南北部，[[太行山|太行山]]东麓，与[[华北平原|华北平原]]接连。

== 资源 ==
矿产资源有[[煤|煤]]、[[硫铁矿|硫铁矿]]、[[陶土|陶土]]、[[铝矾土|铝矾土]]、[[白云岩|白云岩]]等。九十年代发现大量[[金属镁|金属镁]]资源。是中国重要煤炭基地之一。

== 经济 ==
*有[[煤炭|煤炭]]、[[机械|机械]]、[[化工|化工]]、[[纺织|纺织]]等工业。所产汽车配件质量在国内名列前茅。[[耐火砖|耐火砖]]、[[铸铁管|铸铁管]]、[[釉面砖|釉面砖]]等先后打入国际市场。
*农业盛产[[小麦|小麦]]、[[玉米|玉米]]、[[谷子|谷子]]、[[红薯|红薯]]、[[柿子|柿子]]、[[苹果|苹果]]。
*著名的淇河三珍：[[淇河鲫鱼|淇河鲫鱼]]、[[缠丝鸭蛋|缠丝鸭蛋]]、[[冬凌草|冬凌草]]。
*豫北黑[[山羊|山羊]]为优良畜种。

== 交通 ==
铁路：[[京广铁路|京广铁路]]从鹤壁市穿境而过，在鹤壁设[[鹤壁站|鹤壁站]]。[[京港客运专线|京港客运专线]]于2012年12月26日全线通车，在鹤壁设[[鹤壁东站|鹤壁东站]]。

高速公路：[[京港澳高速|京港澳高速]]和[[范辉高速|范辉高速]]。国道：[[107国道|107国道]]。

== 名胜古迹 ==
鹤壁的人文景观和自然风光各具特色，是河南省重要的旅游观光地区。境内[[大伾山|大伾山]]、[[浮丘山|浮丘山]]、[[云梦山|云梦山]]、[[五岩山|五岩山]]各具特色。华北地区唯一未被污染的[[淇河_(卫河支流)|淇河]]从境内蜿蜒流过，两岸景色秀丽，为《[[诗经|诗经]]》所咏唱。
*鹤壁集古瓷窑遗址
*五岩寺石窟
*玄天洞石塔

== 文化名人 ==
*[[鬼谷子|鬼谷子]]，又称王蝉，道号鬼谷子。曾隐居[[云梦山|云梦山]]（今[[淇县|淇县]]境内），培养了著名军事家[[孙膑|孙膑]]、[[庞涓|庞涓]]，纵横家[[张仪|张仪]]、[[苏秦|苏秦]]。
*[[箕子|箕子]]，殷商贵族，后出走[[朝鲜|朝鲜]]。
*[[荆轲|荆轲]]，战国末期[[卫国|卫国]]人(今[[淇县|淇县]])。生年不详，因刺杀秦王于公元前227年被车裂。
*孙思缈，一代药圣，相传曾定居大嶗山(今鹤山区境内五岩山)。
*[[赵洪璋|赵洪璋]]，小麦育种专家，[[中国科学院学部委员|中国科学院学部委员]]。
*[[赵玉芬|赵玉芬]]，当选年龄最年轻的中国科学院女院士。
*[[牛得草|牛得草]]，著名戏剧家。
*[[金不换|金不换]]，豫剧名丑。
*[[程相文|程相文]]，著名玉米育种专家。
