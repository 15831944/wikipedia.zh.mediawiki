'''鶴鴕'''（學名''Casuarius''），又稱為'''食火雞'''，是居住於[[新几內亞|新几內亞]]及東北[[澳洲|澳洲]]大型不能飛的[[鳥類|鳥類]]，統合在'''鶴鴕屬'''。在一些附近的島嶼上亦有少量的鶴鴕，但究竟是因自然而生或是從新畿內亞貿易而得就不能而知。牠們以[[果實|果實]]為食物，尤其是落下的果實或生在低樹枝的果實。牠們也會吃[[真菌|真菌]]、[[蝸牛|蝸牛]]、[[昆蟲|昆蟲]]、[[青蛙|青蛙]]、[[蛇|蛇]]及其他細小的[[動物|動物]]。亦有報告指牠們襲擊[[人類|人類]]，不過這多是因人類入侵牠們的領域或是令牠們感覺被威脅。

== 分類 ==
鶴鴕是屬於[[古顎總目|古顎總目]]，當中亦包括[[鴯鶓|鴯鶓]]、[[鴕鳥|鴕鳥]]、[[恐鳥|恐鳥]]及[[鷸鴕|鷸鴕]]等。現今已確認了三個種：

*'''[[南方鶴鴕|南方鶴鴕]]'''或'''雙垂鶴鴕'''（''C. casuarius''）居住於[[澳洲|澳洲]]及[[新畿內亞|新畿內亞]]。
*'''[[單垂鶴鴕|單垂鶴鴕]]'''（''C. unappendiculatus''）居住於新畿內亞。
*'''[[侏鶴鴕|侏鶴鴕]]'''（''C. bennetti''）居住於新畿內亞及[[新不列顛|新不列顛]]。

單垂鶴鴕及侏鶴鴕並不十分著名。所有鶴鴕一般都是羞怯的及鬼鬼祟祟的，生活在森林的深處及遠離人煙的地方。縱然較為人熟悉生活在[[昆士蘭|昆士蘭]][[雨林|雨林]]的南方鶴鴕，也不甚為人了解。

== 演化歷史 ==
鶴鴕的[[演化|演化]]歷史並不清楚。在[[澳洲|澳洲]]就曾有報告關於[[化石|化石]]的品種，但根據[[生物地理學|生物地理學]]的解釋這並不肯定是鶴鴕，或可能是屬於史前像鶴鴕的原始[[鸸鹋|鸸鹋]]''Emuarius''。

== 特徵 ==
[[File:Cassowary-head.jpg|thumb]]
[[南方鶴鴕|南方鶴鴕]]是[[澳洲|澳洲]]第二大的鳥，而在[[世界|世界]]上則是第三大，僅次於[[鴕鳥|鴕鳥]]及[[鸸鹋|鸸鹋]]。雌性的體型較大及更有光澤色彩。南方鶴鴕約1.5-1.8米高，有些雌性個體甚至可達2米高，體重為60公斤。牠們的頭上有一個骨盔，是用作擊打草叢，牠亦是唯一備有裝甲的[[鳥類|鳥類]]。

鶴鴕的腳有三趾，每趾都有鋒利的爪。最内侧趾的爪就更像匕首，長120毫米。這爪特別危險，因鶴鴕只要一踢就可以用它來殺死敵人或除去[[腸子|腸子]]。牠們可以以時速50公里穿越密林，利用頭上的骨盔推倒小樹及叢林。牠們可以跳躍達1.5米高，且是游泳能手。

2004年的[[健力士世界紀錄|健力士世界紀錄]]就將鶴鴕列為世界上最危險的[[鳥類|鳥類]]。一般來說，鶴鴕都是非常害羞的，但當受騷擾時則會以牠們強勁的腿猛掃。當[[第二次世界大戰|第二次世界大戰]][[美軍|美軍]]及[[澳軍|澳軍]]在[[新畿內亞|新畿內亞]]駐守時，就曾被警告遠離牠們。牠們可以造成致命的傷害。多數的攻擊是因挑撥而起。受傷或陷入絕境的鶴鴕特別危險。鶴鴕能靈巧地利用牠們身處的環境，來避開[[人類|人類]]的圍捕。由於[[動物園|動物園]]內員工的受傷率及嚴重性，鶴鴕被認為是動物園中最危險的[[動物|動物]]。

== 生活習性 ==
野生的鶴鴕喜歡獨來獨往，到繁殖季節才群集一起交配。雌性鶴鴕每次會生3-8隻蛋。牠們的蛋大小約為9-14厘米，呈淡藍綠色，只有[[鴕鳥|鴕鳥]]及[[鴯鶓|鴯鶓]]的蛋比牠們的大。雌性鶴鴕不會照顧牠們的蛋或幼鴕，雄性鶴鴕會孵化蛋兩個月，再照顧幼鴕九個月。

== 保護狀況 ==
[[南方鶴鴕|南方鶴鴕]]及[[單垂鶴鴕|單垂鶴鴕]]因失去棲息地而成為瀕危物種。據估計牠們的數量約為1,500-10,000頭左右。在[[澳洲|澳洲]]約有40頭鶴鴕在籠中受保護。失去棲息地令鶴鴕從[[雨林|雨林]]走到[[人類|人類]]社區，造成與人類（尤其是農民）的衝突。但是在一些地方如[[昆士蘭|昆士蘭]]的[[美聲海灘|美聲海灘]]，就出現了參觀鶴鴕的旅遊。
