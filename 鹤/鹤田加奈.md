'''鹤田加奈'''（{{lang-ja|鶴田 かな}}，{{bd|1992年|12月12日||}}），日本AV女優。她出身于[[神奈川县|神奈川县]]<ref name=A>{{cite web|url=http://actress.dmm.co.jp/-/detail/=/actress_id=1012352/ |title=公式プロフィール |work=AV女優名鑑 |publisher=[[DMM.com|DMM.R18]] |accessdate=2014-5-26}}</ref>，所属于{{link-ja|C-more Entertainment}}事务所。

== 简历 ==
鹤田加奈出身于[[日本|日本]][[神奈川县|神奈川县]]<ref name=A/>。

2011年推出两部{{link-ja|着衣色情|着エロ}}作品之后，于2012年2月以[[S1_(AV片商)|S1]]专属AV女优身份出道，还与[[绘里奈摩亚|绘里奈摩亚]]担任同月的S1封面女郎<ref>
{{cite web |url=http://www.s1s1s1.com/covergirl/1202.html |title=カバーガール 【2012年2月分】 |publisher=エスワン |accessdate=2012-06-18 |archive-url=https://web.archive.org/web/20130323012610/http://www.s1s1s1.com/covergirl/1202.html |archive-date=2013-03-23 |dead-url=yes }}
</ref>。

2014年暂时休业，2015年重返AV圈。

2017年10月13日，她在[[Twitter|Twitter]]上宣布将于同年12月31日正式引退<ref>{{cite tweet |user=turutakana|number=918671388767956993|author=鶴田かな|date=2017-10-13|accessdate=2017-12-23|title=ご報告}}</ref><ref>{{cite news|url=https://news.dmm.co.jp/article/997810/ |title=つるかなファイナル！AV女優として駆け抜けた6年の日々を振り返る！▼サバ責めって何！？食ザーで平気だったのは…▼怖かったTOHJIRO監督との初対面！▼明日12/14ドグマイベントに降臨！【鶴田かな引退惜別ロングインタビュー第１回】 |work=DMM  |date=2017-12-13 |accessdate=2019-02-26}}</ref>。
