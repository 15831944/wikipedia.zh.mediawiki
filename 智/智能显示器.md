'''智能显示器'''（{{lang-en|'''Smart Display'''}}，最初代号'''Mira'''）是[[微软公司|微软公司]]以配有[[觸控式螢幕|觸控式螢幕]]的便携式[[液晶显示器|LCD]][[顯示器|顯示器]]通过[[Wi-Fi|Wi-Fi]]连接作为[[IBM_PC兼容机|PC]][[瘦客户端|瘦客户端]]的一个倡议。

微软于2002年初宣布该计划，并于2003年初发布产品，但计划最终于2003年12月宣告终止，产品的[[市场渗透率|市场渗透率]]很低。<ref name="zdnet3">[http://www.zdnet.co.uk/news/emerging-tech/2004/01/06/microsoft-dumps-smart-display-39118897/ Microsoft dumps Smart Display] - ZDNet, 6 January 2004</ref>

== 技术 ==
智能显示器是采用[[电池|电池]]供电的配有触摸屏的10[[英寸|英寸]]或15英寸LCD显示器（尺寸和形状类似[[平板電腦|Tablet PC]]），它通过一个802.11b WiFi 网络连接到PC，通过Transcriber（类似[[Graffiti_英文手寫輸入法|Graffiti]]）或使用弹出式软键盘进行文本输入，内置{{le|计算机扬声器|Computer speaker|扬声器}}。部分型号有一个与有线PC、[[电脑键盘|电脑键盘]]和[[鼠标|鼠标]]连接器接合的坞。

此种显示器运行在基于[[Windows_Embedded_Compact|Windows CE]]和[[.NET框架|.NET框架]]的'''Smart Display OS'''或'''Microsoft [[Windows_Embedded_Compact|Windows CE]] for Smart Displays'''上。而其远程技术则基于[[远程桌面连接|Windows终端服务]]。Smart Display OS 1.0只能连接到一个[[Windows_XP版本列表|Windows XP Professional]]主机系统，不过也有些人报告称使用NetMeeting可控制任何版本的Windows。

ViewSonic是第一家将智能显示器推向市场的制造商，其在2003年初推出了airpanel V150。这款机型包括一个15英寸1024×768 LCD，一个400 MHz [[XScale|Intel XScale]]处理器，32MB ROM，64MB RAM，以及802.11b无线网和一个面向主机的USB无线集线器。

== 问题 ==
早在2002年初，分析家们就注意到了Mira计划的问题。
* Smart Display OS 1.0中，显示器使用时会锁定主机PC。微软将此归咎于授权问题（Windows XP Professional的每个副本授权给一名用户<ref>[http://comment.zdnet.co.uk/mattloney/0,39020679,2126784,00.htm Mira, Mira in the bin] (Matt Loney, ZDNet UK, 29 November 2002)</ref>）和资源管理问题。许可要求——不允许设备独立工作，不允许设备在PC主屏幕活动时连接到主机，且不允许多台智能显示器控制一台主机。这在媒体界引发了广泛嘲笑。
* 一台主机PC同一时间只能受到一台智能显示器的连接，阻碍了家庭中使用多台设备连接一台PC。<ref name="zdnet1">[http://www.zdnet.co.uk/news/desktop-hardware/2002/03/13/cebit-2002-mira-gets-rugged-and-magnetic-2106528/ Mira gets rugged and magnetic] - ZDNet, 13 March 2002</ref>
* 尽管这些设备的CPU和[[内存|内存]]配置以及[[操作系统|操作系统]]与稍大的[[Pocket_PC|Pocket PC]]相似，但重量却与[[笔记本电脑|笔记本电脑]]相当，电池续航时间也相似，且它们还没有独立功能，只能配合主机PC使用。分析师猜测<ref>[https://www.theregister.co.uk/2002/03/18/microsofts_mira_take_smart_display/ Microsoft's Mira - take smart display, maim, serve] (John Lettice, ''The Register'', 18 March 2002)</ref>Smart Display受到了削弱，以免有影响[[平板電腦|Tablet PC]]市场的风险。
* 不可能进行视频流连接速度不够快，且[[遠端桌面協定|遠端桌面協定]]不包含有助视频的增强功能。
* 微软曾打算将智能显示器的售价定位在约500[[美元|美元]]，但这些设备在市场上的最终售价为1000-1500美元，相较而言的笔记本电脑价格大约是600美元。<ref>[https://www.nytimes.com/2003/01/02/technology/circuits/02stat.html?pagewanted=3 And now, the portable desktop PC, up to a point] - [[纽约时报|New York Times]], 2 January 2003</ref>

== 历史 ==
[[比尔·盖茨|比尔·盖茨]]在2002年的[[消费电子展|消费电子展]]上宣布了Mira计划。

最早宣布的技术合作伙伴包括{{le|AboCom}}、[[富士通|富士通]], [[英特尔|英特尔]], [[LG電子|LG電子]], [[美國國家半導體|美國國家半導體]], [[日本電氣|NEC]]、{{le|Philips Consumer Electronics}}、[[大同公司|大同公司]]、[[三寶電腦|三寶電腦]]、[[优派|优派]]、[[纬创资通|纬创资通]]和{{le|Wyse}}。

Smart Display OS 1.0受到了多家显示器制造商的支持，包括[[三星集团|三星集团]]、LG、TriGem、Philips、ViewSonic、Fujitsu和NEC。产品包括Fujitsu FMSDP-101、the NEC PK-SD10、the Philips DesXcape、the Trigem Play@PAD和ViewSonic airpanel。<ref>[http://www.zdnet.co.uk/news/desktop-hardware/2002/03/13/mira-device-gains-new-supporters-2106457/ Mira device gains new supporters] {{webarchive|url=https://web.archive.org/web/20110721185058/http://www.zdnet.co.uk/news/desktop-hardware/2002/03/13/mira-device-gains-new-supporters-2106457/ |date=2011-07-21 }} - ZDNet, 13 March 2002</ref>

Smart Display OS 1.5原定于2003年年底推出，旨在使个人电脑能同时连接普通显示器与智能显示器。微软后来变更了时间表，将其归纳并于2004年后半年发布了Smart Display OS 2.0，其中增加了移动[[图像处理|图像处理]]等额外功能。

== 取消 ==
在与多家液晶显示器制造商签订合作协议，并且其中一些制造商（如LG）准备于2004年初发布智能显示器1.5产品的情况下，微软最终于2003年12月22日宣布取消该倡议。

LG表示，即便没有微软的支持也会继续坚持智能显示器，尽管不会有产品出现。<ref>[https://www.theregister.co.uk/2004/01/08/lg_will_persist_with_smart/ LG `will persist' with Smart Display, even after Microsoft drops it] (Guy Kewney, ''newswireless.net'', Thursday 8 January 2004)</ref><ref>{{cite web|title=微软停止智能显示器开发 LG与三星一守一退|url=http://tech.sina.com.cn/it/2003-12-31/0755275743.shtml|website=新浪科技|accessdate=2017-11-05|date=2003-12-31}}</ref>

{{who|有人}}提出，{{tsl|en|List of Microsoft codenames||代号}}Origami的[[超級移動電腦|Ultra-Mobile PC]]（UMPC）是智能显示器的未来方向。<ref>{{cite web |url=http://digital-lifestyles.info/display_page.asp?section=cm&id=3080 |title=存档副本 |accessdate=2006-09-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20061029060352/http://www.digital-lifestyles.info/display_page.asp?section=cm&id=3080 |archivedate=2006-10-29 }} Smart Display evolves</ref>
