'''泛政治化'''({{lang-en|Para-Political}})指在討論非[[政治學|政治學]]領域內容、超出政治學限度，在非必要情況下牽扯到政治相關因素。在[[政黨政治|政黨政治]]的國家體制內，泛政治化也指政黨之間，為互相競爭而不論對國家人民是否有利，抵制對方政黨的政策或主張。<ref name="鼎文書局編者：艾德2017">{{cite book|author1=鼎文書局|author2=編者：艾德|title=高普特考【政治學（含概要）】（內容精要試題精析．附階段讀書計劃表）: 10版|url=https://books.google.com/books?id=aPNoBgAAQBAJ&pg=PT22|date=11 August 2017|publisher=鼎文|isbn=978-957-454-750-0|pages=22–}}</ref>

== 政治學限度 ==
人類生活的時空範圍內並非所有領域皆與[[政治學|政治學]]相關，因此若在與公眾利益風險無關，或僅是個人隱私、私人情趣一般生活領域牽扯到政治因素，則超出政治學涵蓋的領域範圍，成為泛政治化。<ref name="鼎文書局編者：艾德2017" /><ref name="孫德喜2010">{{cite book|author=孫德喜|title=閱讀泰國?解讀北韓|url=https://books.google.com/books?id=ZjFib9g5TFEC&pg=PA245|date=1 December 2010|publisher=秀威出版|isbn=978-986-221-610-1|pages=245–}}</ref>

== 案例 ==
* [[台灣|台灣]]在[[民進黨|民進黨]]執政處理面對[[中國|中國]]議題時，不論對於兩岸三通、經貿交流、文教交流、甚至大學學歷的承認與贈與大熊貓都賦予太多政治意涵，而採用緊縮與拒絕開放阻礙兩岸發展，是泛政治化。台灣社會內部藍綠意識形態的對抗，也是泛政治化。<ref name="周志怀主编2017">{{cite book|author=周志怀主编|title=两岸关系和平发展的巩固与深化|url=https://books.google.com/books?id=BvU_DwAAQBAJ&pg=PA162|date=22 November 2017|publisher=九州社 崧博出版社|isbn=978-7-5108-2039-7|pages=162–}}</ref>在[[陳水扁|陳水扁]]總統執政時期，因朝小野大，[[國民黨|國民黨]]在立法院佔多數，因此如行政院的組織改造等需要立法法案基礎的改革與重大政策，被國民黨以泛政治化的政黨對立進行杯葛，拒絕通過對國家有益的法案，影響政府改革與運作效率。<ref name="林嘉誠2012">{{cite book|author=林嘉誠|title=全力以赴--林嘉誠政治經濟文集: 林嘉誠政治經濟文集|url=https://books.google.com/books?id=tO1YCR27abUC&pg=PA4|date=1 April 2012|publisher=秀威出版|isbn=978-957-41-8977-9|pages=4–}}</ref>

* [[北韓|北韓]]政治體制的個人崇拜與領袖迷信，包括要求每個國民在胸前佩戴領袖徽章，這種典型的個人崇拜與[[文化大革命|文化大革命]]時期的[[中國|中國]]相同；北韓境內充滿紀念領袖的永生塔，許多建築物掛著領袖畫像；也有許多地方掛著領袖語錄，這種無所不在的個人崇拜就是泛政治化。<ref name="孫德喜2010" />

* [[中原文化|中原文化]]的[[獨尊儒術|獨尊儒術]]與對於[[道家|道家]]、[[儒家|儒家]]、[[釋家|釋家]]三教合一，造成宗教首領受政府控制等政策即是泛政治化，由帝制思想為掌控政權而影響學術思想之案例。<ref name="邵雍2011">{{cite book|author=邵雍|title=近代會黨與民間信仰研究|url=https://books.google.com/books?id=WUs9AQAAQBAJ&pg=PA253|date=1 July 2011|publisher=獨立作家－秀威出版|isbn=978-986-221-759-7|pages=253–}}</ref>

* 中國[[湖南衛視|湖南衛視]]的娛樂節目''[[超級女聲|超級女聲]]''播出後，媒體提出節目會引發[[2005年超级女声#泛政治化|人民想要民主與直接選舉的政治議題]]，也是泛政治化。<ref name = 中国人民大学_2006>{{cite book|title=中国政治|url=https://books.google.com/books?id=GDI5AQAAIAAJ|year=2006|publisher=中国人民大学书报资料社}}</ref>

== 影響 ==
人類生活若受泛政治化影響，則會造成衝突不斷。原有生活時空範圍與政治因素無關者，加入泛政治化因素後，造成原有價值觀判斷與泛政治化之價值觀發生衝突，可能因泛政治化而限縮民眾原有生活言行舉止選擇，或造成民眾僅依照政治認同行事選擇的偏頗現象。<ref name="鼎文書局編者：艾德2017" />在學校教育系統中，如果教師採用泛政治化的意識形態教學，會影響教學效果導致教學品質下降，如此學生不但無法學好專業科目，也學不到正確的[[政治學|政治學]]內容，只有學到[[意識形態|意識形態]]。<ref name="倪文锦2015">{{cite book|author=倪文锦|title=挑战与应答：语文教学科学化、艺术化的追求: 新学堂数字版|url=https://books.google.com/books?id=8kI8DAAAQBAJ&pg=PT92|date=1 December 2015|publisher=XinXueTang|isbn=978-7-5328-7136-0|pages=92–}}</ref>泛政治化的社會也會影響人們在不理解問題本質或不經過討論思考的情況下，就做出非理性決定，如此反而會妨礙原本正常發展的政治制度，也可能造成大部分民眾不想關心政治議題，或是將政治無限上綱，認為只要採取某個意識形態就能解決一切問題，成為迷信權威與盲目崇拜現象。<ref name = 中国人民大学_2006 />
