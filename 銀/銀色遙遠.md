{{Vgname|銀色遙遠|ja=銀色、遥か}}是由[[Visual_Art's|Visual Art's]]旗下品牌tone work's製作並於2016年8月26日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref name="product">{{Cite web|date=|url=http://www.game-style.jp/special/201605/23/00gin_tw.php|title=銀色遙遠特集|website=Game-Style|accessdate=2017-07-18|language=ja}}</ref>。

== 故事簡介 ==
在國內最北的政令指定都市「幌路市」居住的男主角新見雪兔，在升上國中二年級的春天，與五名少女相遇了<ref name="product"/>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|新見雪兔|新見 雪兎（にいみ ゆきと）}}
: 本作男主角，名字可更改<ref>{{Cite web|url=http://toneworks.product.co.jp/ginharu/character.html|title=角色介紹|publisher=銀色遙遠官網|accessdate=2017-07-18|language=ja}}</ref>。
; {{nihongo|貝絲莉·羅斯·戴斯利|ベスリー・ローズ・ディズリー|聲：{{lang|ja|橘まお}}}}
: 從[[加拿大|加拿大]]轉到日本就讀的轉學生，有著一頭金色長髮。擅長繪畫。
; {{nihongo|蒼井雛多|蒼井 雛多（あおい ひなた）|聲：{{lang|ja|柳ひとみ}}}}
: 小雪兔一歲的學妹。個性開朗活潑，因此有很多朋友。
; {{nihongo|名白椛|名白 椛（なしろ もみじ）|聲：{{lang|ja|鈴谷まや}}}}
: 雪兔的同班同學，對演戲非常有興趣，放學後經常一個人留下來練習。
; {{nihongo|如月瑞羽|如月 瑞羽（きさらぎ みずは）|聲：[[風音|風音]]}}
: 雪兔從幼稚園開始就認識的青梅竹馬，是世界首屈一指的[[花式滑冰|花式滑冰]]選手。
; {{nihongo|新見雪月|新見 雪月（にいみ ゆづき）|聲：{{lang|ja|くすはらゆい}}}}
: 雪兔沒有血緣關係的妹妹，小雪兔一歲，個性內向。

== 評價 ==
銀色遙遠在2016年Getchu.com舉辦的美少女遊戲大賞中獲得綜合部門第6名<ref>{{Cite web|publisher=Getchu.com|url=http://www.getchu.com/pc/2016_g_ranking/|script-title=ja:美少女ゲーム大賞2016 －総合部門投票・結果発表－|accessdate=2017-07-18|language=ja}}</ref>、劇本部門第7名<ref>{{Cite web|publisher=Getchu.com|url=http://www.getchu.com/pc/2016_g_ranking/scenario.html|script-title=ja:美少女ゲーム大賞2016 －シナリオ部門投票・結果発表－|accessdate=2017-07-18|language=ja}}</ref>、繪圖部門第8名<ref>{{Cite web|publisher=Getchu.com|url=http://www.getchu.com/pc/2016_g_ranking/graphic.html|script-title=ja:美少女ゲーム大賞2016 －グラフィック部門投票・結果発表－|accessdate=2017-07-18|language=ja}}</ref>、音樂部門第2名<ref>{{Cite web|publisher=Getchu.com|url=http://www.getchu.com/pc/2016_g_ranking/music.html|script-title=ja:美少女ゲーム大賞2016 －ミュージック部門投票・結果発表－|accessdate=2017-07-18|language=ja}}</ref>、影片部門第8名<ref>{{Cite web|publisher=Getchu.com|url=http://www.getchu.com/pc/2016_g_ranking/movie.html|script-title=ja:美少女ゲーム大賞2016 －ムービー部門投票・結果発表－|accessdate=2017-07-18|language=ja}}</ref>、角色部門貝絲莉·羅斯·戴斯利第6名、如月瑞羽第14名<ref>{{Cite web|publisher=Getchu.com|url=http://www.getchu.com/pc/2016_g_ranking/chara.html|script-title=ja:美少女ゲーム大賞2016 －キャラクター部門投票・結果発表－|accessdate=2017-07-18|language=ja}}</ref>。於2016年度「[[萌系遊戲大賞|萌系遊戲大賞]]」獲得純愛系作品賞金賞<ref>{{cite web|url=http://moe-gameaward.com/prize/2016/pure.html#p01|script-title=ja:萌えゲーアワード2016年 純愛系作品賞・金賞作品|work=萌系遊戲大賞|accessdate=2017-07-18|language=ja}}</ref>，並於年間排名獲得第9名<ref>{{cite web|url=http://moe-gameaward.com/annual/2016.html|script-title=ja:萌えゲーアワード2016年間ランキング確定|work=萌系遊戲大賞|accessdate=2017-07-18|language=ja}}</ref>。
