'''银'''（{{lang-en|silver}}）是[[化学元素|化学元素]]，[[化学符号|化学符号]]'''Ag'''（来自{{lang-la|argentum}}）<ref>{{cite book | title = Encyclopedia of the Elements | author = Enghag | year = 2004 | publisher = Wiley-VCH | isbn = 3-527-30666-8 | page=144}}</ref>，[[原子序数|原子序数]]47，是柔软有白色光泽的[[过渡金属|过渡金属]]，在所有金属中[[导电率|导电率]]、[[导热率|导热率]]和[[反射率|反射率]]最高。銀在自然界中的存在方式有纯净的游离态单质（[[自然银|自然银]]），与[[金|金]]等其他金属的[[合金|合金]]，还有含银矿石（如[[辉银矿|辉银矿]]和[[角银矿|角银矿]]）。大部分银都是[[精炼|精炼]][[铜|铜]]、金、[[铅|铅]]和[[锌|锌]]的副产品。

银不易受化學藥品腐蝕，长久以来被视为[[贵金属|贵金属]]。银比金来源更丰富，在现代以前的货币体系中作为[[投资型硬币|硬币]]使用，有时甚至和[[金银复本位|金一道使用]]。除了[[货币|货币]]之外，银的用途还有[[太阳能电池板|太阳能电池板]]、[[净水器|净水器]]、[[珠宝|珠宝]]和装饰品、高价餐具和器皿（[[银器|银器]]），[[银币|银币]]和{{link-en|银条|bullion}}还可用于投资。银在工业上用于{{link-en|电接点|electrical contact}}和[[导体|导体]]、特制镜子、窗膜和化学反应的[[催化剂|催化剂]]。银的化合物用于[[胶片|胶片]]和[[X光|X光]]。稀[[硝酸银|硝酸银]]溶液等银化合物会产生{{link-en|微动力效应|oligodynamic effect}}，可以[[消毒|消毒]]和消灭[[微生物|微生物]]，用于[[绷带|绷带]]、伤口敷料、导管等[[医疗器械|医疗器械]]。

==性质==
纯白银颜色白，金属光泽，质软，掺有杂质后变硬，颜色呈灰、红色。纯白银比重为10.5，熔点960.5℃，导电性能佳，溶于硝酸、浓硫酸中。

===化学性质===
银是古代发现的金属之一。银在自然界中虽然也有单质存在，但绝大部分是以化合态的形式存在。

银具有很高的延展性，因此可以碾压成只有0.00003厘米厚的透明箔，1克重的银粒就可以拉成约两公里长的细丝。

银的导热性和导电性在金属中名列前茅。

银的特征氧化数为+1，其化学性质比铜差，常温下，甚至加热时也不与水和空气中的氧作用，但久置空气中能变黑，失去银白色的光泽，这是因为银和空气中的硫化氫(H<sub>2</sub>S)化合成黑色硫化銀(Ag<sub>2</sub>S)的缘故。其化学反应方程式为：
: 4Ag + 2H<sub>2</sub>S + O<sub>2</sub> = 2Ag<sub>2</sub>S + 2H<sub>2</sub>O

银不能与稀盐酸或稀硫酸反应放出氢气，但银能溶解在硝酸或热的浓硫酸中：
:2Ag + 2H<sub>2</sub>SO<sub>4</sub>（浓） —<u><sup>Δ</sup></u>→ Ag<sub>2</sub>SO<sub>4</sub> + SO<sub>2</sub>↑ + 2H<sub>2</sub>O

银在常温下与卤素反应很慢，在加热的条件下即可生成卤化物：
:2Ag + F<sub>2</sub> —<u><sup>473 K</sup></u>→ 2AgF暗棕色
:2Ag + Cl2 —<u><sup>Δ</sup></u>→ 2AgCl白色
:2Ag + Br2 —<u><sup>Δ</sup></u>→ 2AgBr黄色
:2Ag + I2 —<u><sup>Δ</sup></u>→ 2AgI橙色

银对硫有很强的亲合势，加热时可以与硫直接化合成Ag<sub>2</sub>S：
:2Ag + S =Δ= Ag<sub>2</sub>S
类似地，银和[[硒|硒]]、[[碲|碲]]的反应为：
: 2 Ag + Se → Ag<sub>2</sub>Se
: 2 Ag + Te → Ag<sub>2</sub>Te

== 同位素 ==
{{Main|銀的同位素}}
自然界存在的銀有两种[[稳定同位素|稳定同位素]]：<sup>107</sup>Ag和<sup>109</sup>Ag，其中前者的[[豐度|豐度]]略高（51.839%）。銀的两种[[同位素|同位素]]的豐度幾乎相同，這在元素周期表中十分罕見（[[溴|溴]]是另一個例子）。銀的[[原子量|原子量]]是107.8682 (2) [[摩尔质量|克/摩爾]]。<ref>{{cite web|accessdate = 2009-11-11|url = http://www.chem.qmul.ac.uk/iupac/AtWt/index.html|title = Atomic Weights of the Elements 2007 (IUPAC)}}</ref><ref>{{cite web|accessdate = 2009-11-11|url = http://physics.nist.gov/cgi-bin/Compositions/stand_alone.pl?ele=&ascii=html&isotype=some|title = Atomic Weights and Isotopic Compositions for All Elements (NIST)}}</ref>已确定銀的二十八個[[放射性同位素|放射性同位素]]的特性，其中最穩定的依次是<sup>105</sup>Ag（半衰期41.29天），<sup>111</sup>Ag（半衰期7.45天），<sup>112</sup>Ag（半衰期3.13小時）。銀有很多[[核同质异能素|亚稳态核素]]，其中最穩定的依次是<sup>108m</sup>Ag（半衰期418年），<sup>110m</sup>Ag（半衰期為249.79天），<sup>106m</sup>Ag（半衰期8.28天）。其餘的[[放射性|放射性]]同位素的半衰期皆短於一小時，大部分短於三分鐘。

銀的同位素原子量从92.950（<sup>94</sup>Ag）到129.950（<sup>130</sup>Ag）不等。<ref>{{cite web|accessdate = 2012-01-15|url = http://periodictable.com/Isotopes/047.94/index2.full.dm.html|title = Isotope data for Silver94 in the Periodic Table}}</ref><ref>{{cite web|accessdate = 2012-01-15|url = http://periodictable.com/Isotopes/047.130/index2.full.dm.html|title = Isotope data for Silver130 in the Periodic Table}}</ref>丰度最高的稳定同位素（<sup>107</sup>Ag）之前的同位素的[[放射性#衰变类型|衰变类型]]主要是[[電子捕獲|電子捕獲]]，生成[[钯|钯]]（46号元素）的同位素，而<sup>107</sup>Ag之后的同位素的衰变类型则主要是[[β衰變|β衰變]]，生成[[镉|镉]]（48号元素）的同位素。<ref>{{cite web|accessdate = 2012-01-15|url = http://periodictable.com/Isotopes/047.107/index2.full.dm.html|title = Isotope data for Silver107 in the Periodic Table}}</ref>

<sup>107</sup>Pd β衰變成<sup>107</sup>Ag的半衰期為650萬年。[[鐵隕石|鐵隕石]]是仅有的「鈀-銀比」高到可以測量<sup>107</sup>Ag富度變化的物体。由放射性产生的<sup>107</sup>Ag首次发现于1978年美國[[聖塔克拉拉|聖塔克拉拉]]的隕石。<ref>{{cite journal|doi = 10.1029/GL005i012p01079|title = Evidence for the existence of <sup>107</sup>Pd in the early solar system|year = 1978|author = Kelly, William R.|journal = Geophysical Research Letters|volume = 5|page = 1079|first2 = G. J.|last2 = Wasserburg|bibcode=1978GeoRL...5.1079K}}</ref>發現者提出，一些小型鐵核的行星與其異體，可能是在一千多萬年前的[[核合成|核合成]]事件中產生的。從這熔化過的星球本體中，觀察到的<sup>107</sup>Pd–<sup>107</sup>Ag比值，反映出早期[[太陽系|太陽系]]的[[吸積|吸積]]中應存在著不穩定的核種。<ref>{{cite journal|title = Origin of Short-Lived Radionuclides|first = Sara S.|last = Russell|coauthor = Gounelle, Matthieu; Hutchison, Robert|journal = Philosophical Transactions: Mathematical, Physical and Engineering Sciences|volume = 359|issue = 1787|year = 2001 |page = 1991–2004|doi = 10.1098/rsta.2001.0893|page = 1991|jstor=3066270|bibcode = 2001RSPTA.359.1991R }}</ref>

== 特點 ==
* 性質穩定，活躍性低
* 氧氣相對其他[[氣體|氣體]]能更容易溶解於銀。
* 导熱，導電率高
* 不易受化學藥品腐蝕（但仍然能被[[硫|硫]]、[[硒|硒]]、[[硫化物|硫化物]]、[[硝酸|硝酸]]、[[氫氟酸|氫氟酸]]、[[氢碘酸|氢碘酸]]、[[氯气|氯气]]等腐蚀）
* 质軟
* 富有延展性

== 應用 ==
* 银600-800美元每千克（工业应用必考虑成本，2013年春，相比较铜的价格在8~12美元每千克）。
* 製造高價值的物件如[[銀元|銀元]][[貨幣|貨幣]]、首飾，並用於製造[[勋章|勋章]]、獎座、盃、牌和種種裝飾。
* 與汞、錫等其他金屬在室溫混合成的混合物，被廣泛用於牙醫上。
* 製造控制棒來控制[[核連鎖反應|核連鎖反應]]。
* 用作[[催化劑|催化劑]]，是一種對工業非常重要的催化劑，化學實驗室中也會使用。
* 用作電線等導電體，常見於音響設備及鍵盤。
* 加入[[鎳|鎳]]、[[銅|銅]]以增加硬度。
* 在[[電子工業|電子工業]]上是重要的[[導電|導電]]材料。
* 制造[[合金|合金]]、[[硝酸银|硝酸银]]和其它银的化合物等。
* 用作製造[[鏡子|鏡子]]反光面。
* 飾品、精品、[[工藝品|工藝品]]皆有使用。較好的材質為925銀，即92.5%[[銀|銀]]加入7.5％的[[銅|銅]]，為 [[Tiffany_&_Co.|Tiffany & Co.]] 所開創的標準。
* 銀能對硫等元素反應，也對某些微生物有殺菌功效卻對人體無害，加上有美觀價值，因此常被做為高級餐具或食物容器。古代也曾有利用這種特性而出現「銀針探毒」的驗毒技術，但今日已證實銀僅對部分元素、化合物及微生物有反應，部分食物如[[雞蛋|雞蛋]]等因含硫即使無毒亦會有反應，驗毒功效並非百分之百。

== 名稱來源 ==

銀拉丁原名為''argentum''，是其化學符號的來源。

因為銀的活躍性低，其元素型態易被發現亦易提取，故此在古時的中國和西方分別已被認定為[[五金|五金]]和[[煉金術|煉金術]]七金之二，僅於'''[[金|金]]'''之後一名。

古代西方的[[煉金術|煉金術]]和[[占星術|占星術]]也有將金屬中的銀與七曜中的[[月亮|月]]連結，又為[[金|金]]和[[太陽|日]]之後一名。

==化合物==
===+1价态化合物===
银在化合物中主要以+1价的形式存在。

银溶于[[硝酸|硝酸]]（{{chem|HNO|3}}），生成[[硝酸银|硝酸银]]（{{化學式|銀|氮|氧|3}}）。硝酸银是一种透明晶体，有感光性，且易溶于水。硝酸银是合成许多其他银化合物的原料，也可作为[[防腐剂|防腐剂]]，还用于[[花窗玻璃|彩色玻璃]]中的黄色添加剂。银不易与硫酸反应，因此硫酸在珠宝制造中用于清洗[[硬焊|银焊]]及[[退火|退火]]后留下的氧化铜{{tsl|en|firescale|火痕}}。银易与[[硫|硫]]以及[[硫化氢|硫化氢]]（{{chem|H|2|S}}）反应生成黑色的[[硫化银|硫化银]]（{{chem|Ag|2|S}}），这在失去光泽的[[银币|银币]]或其他物品上很常见。当银制{{tsl|en|Electrical contacts|电气触点}}在富含硫化氢的环境下工作时，触点上的硫化银还会生成[[晶鬚|银晶须]]。
:4 Ag + O<sub>2</sub> + 2 H<sub>2</sub>S → 2 Ag<sub>2</sub>S + 2 H<sub>2</sub>O
[[File:Cessna_210_Hagelflieger_Detail.jpg|thumb]]装备了碘化银發生器]]

向硝酸银溶液中加入[[氯离子|氯离子]]会沉淀出[[氯化银|氯化银]]（{{chem|AgCl}}），同样地，加入[[溴化物|溴盐]]或[[碘化物|碘盐]]可以沉淀出用于制造{{tsl|en|photographic emulsion|感光乳剂}}的其他[[卤化银|卤化银]]。氯化银用于制造检测[[pH值|pH值]]和测量[[电位|电位]]的{{tsl|en|glass electrode|玻璃电极}}，以及用于玻璃的透明[[水泥|水泥]]。将[[碘化银|碘化银]] （{{chem|AgI}}）撒入云层以[[人工降雨|人工降雨]]。卤化银在水溶液中高度不溶（除了[[氟化银|氟化银]]），因而常用于[[分析化学#.E9.87.8D.E9.87.8F.E5.88.86.E6.9E.90|重量分析]]。

向硝酸银溶液加入[[碱|碱]]，沉淀得到[[氧化银|氧化银]] ({{chem|Ag|2|O}})。氧化银用作[[纽扣电池|纽扣电池]]的[[阴极|正极]]。向硝酸银溶液加入[[碳酸钠|碳酸钠]] ({{chem|Na|2|CO|3}})，沉淀得[[碳酸银|碳酸银]]({{chem|Ag|2|CO|3}})。<ref name=photo/>
:2 AgNO<sub>3</sub> + 2 OH<sup>−</sup> → Ag<sub>2</sub>O + H<sub>2</sub>O + 2 NO<sub>3</sub><sup>−</sup>
:2 AgNO<sub>3</sub> + Na<sub>2</sub>CO<sub>3</sub> → Ag<sub>2</sub>CO<sub>3</sub> + 2 NaNO<sub>3</sub>
[[雷酸银|雷酸银]]({{chem|AgONC}})是一种强烈的、对碰撞敏感的[[炸药|炸药]]，是银与硝酸在[[乙醇|乙醇]]({{chem|C|2|H|5|OH}})的存在下反应得到的，用于[[雷管|雷管]]。其他危险易爆的银化合物包括[[叠氮化银|叠氮化银]] ({{chem|AgN|3}})，由硝酸银与[[叠氮化钠|叠氮化钠]] ({{chem|NaN|3}})反应得到，<ref>{{cite book|url = http://books.google.com/?id=ATiYCfo1VcEC&pg=PA284&dq=%22silver++fulminate%22| page = 284| title = Explosives| author = Meyer, Rudolf; Köhler, Josef and Homburg, Axel publisher = Wiley–VCH| year = 2007| isbn = 3-527-31656-6}}</ref>还有[[乙炔银|乙炔银]]({{chem|Ag|2|C|2}})，由硝酸银或[[银氨溶液|银氨溶液]]与[[乙炔|乙炔]]({{chem|C|2|H|2}})反应得到。

卤化银晶体曝光后形成的{{tsl|en|Latent image|潜像}}经[[还原剂|还原剂]]，如[[氢醌|氢醌]]、[[米吐尔|米吐尔]](4-(甲氨基)苯酚硫酸氢盐)或[[抗坏血酸|抗坏血酸]]的[[碱性|碱性]]溶液显影处理后，曝光的卤化银被还原成金属银。硝酸银的碱性溶液（银氨溶液）可被[[还原糖|还原糖]]，如[[葡萄糖|葡萄糖]]等还原为金属银，这个反应用于制造银[[镜|镜]]，以及玻璃{{tsl|en|Christmas ornament|圣诞饰品}}的内表面。卤化银可溶于[[硫代硫酸钠|硫代硫酸钠]]({{chem|Na|2|S|2|O|3}})溶液，因此硫代硫酸钠可作为{{tsl|en|photographic fixer|定影剂}}，去除显影后感光乳剂上多余的卤化银。<ref name=photo>{{cite book| pages = 156–166| title=Silver-halide recording materials: for holography and their processing| author = Bjelkhagen, Hans I.| publisher= Springer| year =1995| isbn = 3-540-58619-9}}</ref>

在[[溴化钾|溴化钾]]({{chem|KBr}})的存在下，金属银可被强[[氧化剂|氧化剂]]如[[高锰酸钾|高锰酸钾]]({{chem|KMnO|4}})或[[重铬酸钾|重铬酸钾]]({{chem|K|2|Cr|2|O|7}})侵蚀；这些化合物在摄影中用于[[漂白|漂白]]可见影像，将其转化为卤化银，既可以被硫代硫酸钠去除，又可以重新显影以加强原始的影像。在过量的[[氰化物|氰根离子]](CN<sup>-</sup>)存在下，[[氰化银|氰化银]]({{chem|AgCN}})可以形成可溶于水的氰[[配合物|配合物]]({{chem|Ag(CN)|2}}<sup>-</sup>)。银的氰配合物溶液用于[[电镀|电镀]]银。<ref name=photo/>

===其它价态化合物===
银还能形成其它价态的化合物，如[[氟化亚银|氟化亚银]]（Ag<sub>2</sub>F）、[[二氟化银|二氟化银]]（AgF<sub>2</sub>）、[[一氧化银|一氧化银]]（AgO）等。

== 在生物中作用 ==

银的离子以及化合物对某些[[细菌|细菌]]、[[病毒|病毒]]、[[藻类|藻类]]以及[[真菌|真菌]]显现出毒性，但对人体却几乎是完全无害的。银的这种杀菌效应使得它在活体外就能够将生物杀死。然而，银制品的测试以及标准化却存在很大难度。

[[希波克拉底|希波克拉底]]曾经有描述银在治疗和防止疾病方面的功用。[[腓尼基人|腓尼基人]]曾经用银瓶子来盛放[[水|水]]、[[酒|酒]]和[[醋|醋]]，以此防止这些液体變壞。20世纪初期，人们也曾把银币放在[[牛奶|牛奶]]，以此来延长牛奶的保鲜期。银的杀菌机制长期以来一直为人们所争论探讨，但至此还没有确凿的定论。其中一个很好的例子是微动力效应，成功的解释了银离子对[[微生物|微生物]]的作用，但却不能解释其对病毒的作用。

[[凝胶|凝胶]]以及[[绷带|绷带]]大量使用銀。银的抗菌性来源于银离子。由于银离子可以和一些微生物用于呼吸的物质（比如一些含有[[氧|氧]]、[[硫|硫]]、[[氮|氮]][[元素|元素]]的[[分子|分子]]）形成强烈的[[结合键|结合键]]，以此使得这些物质不能为微生物所利用，从而使得微生物[[窒息|窒息]]而亡。

在[[抗生素|抗生素]]發明之前，银的[[銀化合物|相关化合物]]曾在[[第一次世界大战|第一次世界大战]]时用于防止感染。

银作为效用广泛的抗菌剂正在进行新的应用。其中一方面就是将[[硝酸银|硝酸银]]溶于[[海藻酸盐|海藻酸盐]]中，用于防止伤口的感染，尤其是烧伤伤口的感染。2007年，一个公司设计出一种表面镀上银的玻璃杯，这种杯子号称具有良好的抗菌性。除此之外，美国食品和药品管理协会（FDA）最近也审批通过了一种内层镀银的导气管的应用，因为研究表明这种导气管能够有效的降低[[导气管型肺炎|导气管型肺炎]]。

銀並不會對人的身體產生毒性，但長期接觸銀金屬和無毒銀化合物也會引致[[銀質沉著症|銀質沉著症]](Argyria)。因為身體[[色素|色素]]產生變化，[[皮膚|皮膚]]表面會顯出灰藍色，雖無毒性，但會影響外觀。
