'''銀池寺'''（{{lang-th| วัดตระพังเงิน}}，英語：Wat Traphang Ngoen）位於[[泰國|泰國]][[大城府|大城府]]的素可泰遺蹟公園（英語：Sukhothai Historical Park）。銀池寺興建於14世紀（[[素可泰王國|素可泰王朝]]時期），與[[瑪哈泰寺|瑪哈泰寺]]（Wat Mahathat）在同一時期建造。

素可泰古城內有四座大水池{{noteTag|name=note1}}，分別為東方金池、西方銀池、北方白池與空心菜池，由於銀池寺建於銀池池畔，故以銀池為名<ref>{{cite book|author1=謝明蓉|title=非常泰國: 曼谷vs.清邁酷遊筆記|date=2005-11-01|publisher=宏碩文化|isbn=9789867143044|page=59|url= https://books.google.com/books?id=xk2T5RU_WQoC&pg=PA59&lpg=PA59&dq=%E9%8A%80%E6%B1%A0%E5%AF%BA&source=bl&ots=jaXwtFcehX&sig=6J29GpDTuCp0bnQ0zT_ultWLsdI&hl=zh-TW&sa=X&ved=0ahUKEwjCvumYo8fXAhUG1GMKHYk2AgYQ6AEIOzAG#v=onepage&q=%E9%8A%80%E6%B1%A0%E5%AF%BA&f=false|accessdate=2017-11-24}}</ref>。

== 建築 ==
銀池寺沒有邊界墻，由主佛塔、正殿、偏殿組成。主佛塔受到[[錫蘭|錫蘭]]風格影響，塔頂呈現長橢圓形，佛塔上有四組蓮花花蕾狀的壁龕<ref>{{cite web|author1=馬小璇|title=彭世洛與素可泰|url=http://big5.taiwan.cn/tsh/zxyd/srtgdqhzl/201307/t20130717_4461303_2.htm|publisher=台灣網|accessdate=2017-11-24}}</ref>。
正殿位於東側池塘中央的小島上，在[[素可泰王國|素可泰王朝]]時期，「正殿」象徵完整性，一般會用池水將偏殿與其餘設施隔開，正殿現僅存基座石柱與部分磚頭<ref>{{cite book|author1=中村浩|title=ぶらりあるきチェンマイ・アユタヤの博物館|date=2016-12-23|publisher=芙蓉書房|isbn=9784829507018|page=93-94}}</ref>。

== 坐佛 ==
銀池寺裡供奉的坐姿[[釋迦牟尼佛|釋迦牟尼佛]]像，佛像的面部呈長橢圓型，表情肅穆沉靜，雙目半闔俯視群生，頭上顆粒狀的髮螺上繫著火焰狀的髮髻，造形相當獨特。佛像背倚著蓮花佛塔，在朝夕餘輝映照下，倒影會剛好映在銀池之中，這是當初設計建造者的巧思構想<ref>Dawn F. Rooney: ''Ancient Sukhothai, Thailand's Cultural Heritage'', S. 95</ref>。

== 行走佛 ==
[[行走佛|行走佛]]浮刻在一面磚牆上，是[[素可泰王國|素可泰王朝]]時期佛教藝術的特色，行走佛足踏蓮花，寓意佛陀慈悲憐憫眾生，於塵世中救度眾生而不染著於塵世。佛像面容飽滿圓潤，右手輕捻手印、左手自然垂下<!--，體現出自然生動，身姿纖柔之美-->。

== 白色佛像 ==
銀池寺前有一尊白色佛像，盤膝而坐，腰帶垂至肚臍，為[[素可泰王國|素可泰王朝]]時期佛像特徵。
