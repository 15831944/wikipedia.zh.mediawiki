'''巫山县'''位于[[重庆市|重庆市]]东部，东、南与[[湖北省|湖北省]]相邻，西接[[奉节县|奉节县]]，北依[[巫溪县|巫溪县]]。幅员面积2957平方千米，常住人口49.59万人。<ref name=pop/>

== 历史沿革 ==
* [[戰國|战国]]间，巫属[[楚国|楚国]]之巫郡。
* [[秦国|秦]][[秦昭襄王|昭襄王]]三十年（前277年）建巫县。
* [[东汉|东汉]][[建安_(东汉)|建安]]十五年（210年），[[刘备|刘备]]主[[荆州_(古代)|荆州]]，析巫县置北井县。
* [[蜀汉|蜀汉]][[章武|章武]]二年（222年），县境属吴，隶属[[宜都郡|宜都郡]]。
* [[東吳|吴]][[孫休|孙休]][[永安_(东吴)|永安]]三年（260年）分宜都郡置[[建平郡_(东吴)|建平郡]]，治巫县。
* [[西晋|西晋]][[泰始_(西晋)|泰始]]四年（268年）隶属[[巴东郡|巴东郡]]。
* [[隋朝|隋]][[开皇|开皇]]三年（583年）置建平郡，巫县更名为巫山县。
* 1952年，[[湖北省|湖北省]][[建始县|建始縣]]的銅鼓鄉劃歸四川省巫山縣<ref>1952年3月19日內務部【內民字第72號】批覆</ref>。
* 1997年6月28日，巫山县划归重庆直辖市。

== 自然地理 ==
* 巫山县位于重庆市最东端，三峡库区腹心，素有“渝东门户”之称，地跨[[长江|长江]][[巫峡|巫峡]]两岸，县城位于大宁河与长江交汇处。海拔相对高差2606米，跨越东经109°33′-110°11′，北纬30°45′-23°28′。巫山地形十分复杂，南北高中间低，峡谷幽深，岩溶发育，山地面积占96%，丘陵平坝占4%。巫山属亚热带季风性湿润气候，气候温和，雨量充沛，年均温度18.4℃，年平均降水量1041毫米。<ref>{{cite news|title=巫山县自然地理|url=http://wush.cq.gov.cn/E_ReadNews.asp?NewsID=18593|accessdate=2016-11-20|publisher=巫山县政府|date=2015-10-20}}</ref>

== 行政区划 ==
2011年8月，撤销龙井乡，并入巫峡镇。
辖2街道、11镇、11乡、2个民族乡，共567个村。
* 街道：高唐、龙门。
* 镇：巫峡、福田、龙溪、大昌、庙宇、双龙、官阳、骡坪、抱龙、官渡、铜鼓。
* 乡：两坪、建平、曲尺、大溪、金坪、平河、三溪、竹贤、笃坪、培石、当阳11个乡。
* 民族乡：邓家土家族乡、红椿土家族乡。

== 旅游资源 ==
* 境内有举世闻名、以峡谷、江河风光为代表的[[长江三峡|长江三峡]]
* 有幽深秀丽、以探险、休闲、漂流旅游为主要内容的[[小三峡|小三峡]]、[[小小三峡|小小三峡]]
* 有曲径探幽、以神女峰为代表的巫山“三台八景十二峰”景点群。
* 有座“四门可通话，一灯照全城”的“袖珍古城”之称的大昌古镇。
* 2003年三峡库区二期蓄水后，巫山景区峡谷相通、湖库相连，更加特色鲜明、令人向往。
* 1991年，长江三峡、巫山小三峡双双被评为中国旅游胜地四十佳，2000年被评为全国首批AAAA景区。巫山旅游日接待能力达8000人以上，年接待游客100万人次以上，为全国旅游大县。<ref>{{cite news|title=巫山概览|url=http://www.people.com.cn/GB/guandian/2196641.html|accessdate=2012-11-20|newspaper=人民网|date=2003-11-18}}</ref>
