《'''巫女、科學與鬼話連篇'''》（[[日語|日語]]：{{lang|ja|巫女と科学の嘘八百万}}），是[[颯田直斗|颯田直斗]]繪製的一系列[[日本|日本]][[漫画|漫画]]作品。

為發佈於網路漫畫發布網站[[FlexComix布拉德|Flex漫画]]<ref>{{cite web|url=https://news.mynavi.jp/article/20120123-a073/ |language=ja |title=FlexComixブラッドとフレア、人気作をニコニコ静画で配信
 |publisher=マイナビニュース | date=2012-01-23 |accessdate=2018-05-17 }}</ref>於2011年2月2日～2012年11月27日間<ref>連載日期為2012年9月26日。</ref>的連載漫畫系列。全部共15話、發行3本漫畫單行本。

== 登場人物 ==
; 圓．Ｍ．芙蘭朵
: 本作主角。自稱"天才科學家"的[[日裔美國人|日裔美国人]]。18歲。擁有金髮碧眼。為了躲債而來到外祖父在日本的神社，與舅舅誠司一同生活，並成為了神社的[[巫女|巫女]]。然而，圓行事開放、為達目的不擇手段的她，只想著要賺大錢還債，常常拋下巫女的工作，埋頭開發[[高科技|高科技]]機器。<ref>{{cite news|title=《巫女、科學與鬼話連篇》第2集大戰上市!|url=https://www.animen.com.tw/animen_web/web/news_inside?news_id=18329 |language=zh-tw |author= |publisher=Animen 動漫新聞 |date=2013-08-14 |accessdate=2018-05-17}}</ref>
; 糺 誠司
: 御御名神社的代表負責人，比姪女(本作主角)還要年輕的17歲。對於高科技產物[[过敏|过敏]]，而且放置在眼前的話會得到[[荨麻疹|荨麻疹]] 。看起來沒有就讀高中。
; 幸篤井 千蒔
: 12岁。暴力團團長的女朋友。由於失憶而接受了圓的幫助，而和圓成為了結拜義妹。目前待在神社中擔任巫女。
; 鈴掛 鈴花
: 鈴掛神社的巫女。16歲。除了有點愛妄想之外，可說是完美無缺的大和撫子。
; 卡洛斯
: 滑稽的[[墨西哥|墨西哥]]人，御御名神社的宮司。25歲。對於巫女有著異常的執著心，對於本作主角下了「巫女失格」的判定，認為她「完全不適合穿巫女服」，是個令人感到十分遺憾的巫女。
