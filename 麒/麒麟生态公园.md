'''麒麟生态公园'''位于[[中華人民共和國|中國]][[南京市|南京市]][[江宁区|江宁区]]，长3公里，宽500米，绿化景观总面积近140万平方米，被誉为城市“绿肺”。<ref name="2R">{{Cite web |url = http://qilinpark.nanjing.gov.cn/xwzx/ztbd/201707/t20170728_4885154.html |title = 打造高品质的科技、生态、宜居新城|website=南京市麒麟科技创新园（生态科技城）开发建设管理委员会 |date = 2017-05-19 |language = zh |accessdate = 2019-03-09 }}</ref>

==位置==
麒麟生态公园位于[[中华人民共和国|中华人民共和国]][[江苏省|江苏省]][[南京市|南京市]][[江宁区|江宁区]][[青龙山|青龙山]]和[[紫金山|紫金山]]之间500米宽的生态廊道上。<ref name="1R">{{Cite web |url = http://js.cri.cn/20180829/c5e2af60-19f5-6528-730b-8a06b2915fb8.html |title = 麒麟生态公园 初具规模|website=国际在线|date = 2018-08-29 |language = zh |accessdate = 2019-03-09 }}</ref>

==设施==
[[File:麒麟生态公园10.jpg|缩略图]]
生态公园内设有展览馆、文化馆、环湖商业及游客中心等设施。<ref name="1R"/>

==景观==
[[File:运粮河03.jpg|缩略图]]
生态公园拥有大面积的原生态草坪，[[明代|明代]]古河道运粮河、上坝河、百水河在此交汇，建有10公里长的滨水景观带和[[人工湖|人工湖泊]]。<ref name="2R"/>

==交通==
[[File:麒麟生态公园站.jpg|缩略图]]
生态公园交通方便，已建成[[麒麟有轨电车|麒麟有轨电车]]设有麒麟生态公园站，有轨电车可到达商业区马群。

[[南京公交|南京公交]]312路设有麒麟生态公园站。<ref>{{Cite web |url = https://m.8684.cn/nanjing_x_d79ed612 |title = 南京公交312路路线图 |website=8684公交网 |language = zh |accessdate = 2019-03-09 }}</ref>
