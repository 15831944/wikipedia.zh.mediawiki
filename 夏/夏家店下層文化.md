'''夏家店下層文化'''為[[东北亚|东北亚]]重要古文化之一。該文化遺址因在[[赤峰市|赤峰市]][[夏家店|夏家店]]發現，依照考古慣例，以發現處所為名。另外，此於1959年發現的遺跡，之所以命名為「下層文化」，乃是為了同樣在夏家店垂直出現，但年代稍晚的[[夏家店上層文化|夏家店上層文化]]相區隔。夏家店下層文化是[[燕山|燕山]][[海河|海河]]地区与中原[[黄河|黄河]]流域古文化相并立的本土文化，并受到了西部草原文化的强烈影响。

== 年代范围 ==
夏家店下層文化，是為公元前20世紀－西元前14世紀之[[青铜文化|青銅文化]]。该地区在新石器时代曾存有著名的[[红山文化|红山文化]]，但夏家店文化并非其直接继承者，红山文化的直接继承者[[小河沿文化|小河沿文化]]族群在向西向南迁徙，与[[蒙古高原|蒙古高原]]南部已掌握了简单青铜技术的[[马家窑文化|马家窑文化]]接触之后，促成了[[河套地区|河套地区]]的老虎沟文化，公元前22世纪青铜技术开始由西往东传入[[燕山|燕山]]南北，结合当地土著原有的[[新石器文化|新石器文化]]，就形成了夏家店下层文化。

== 分布地域 ==
夏家店下層文化分布地點以[[遼河|遼河]]上游一帶为中心，西起[[蒙古高原|蒙古高原]]，北至[[西拉木伦河|西拉木伦河]]，东至[[医巫闾山|医巫闾山]]，向南传播至辽西[[大凌河|大、小凌河]]一带，[[燕山|燕山]]地带与[[拒马河|拒马河]]以北低地至[[渤海|渤海]]，以及[[永定河|永定河]]上游一带。

== 内部类型 ==
夏家店下層文化可基本分为燕山南北两大类型，以及西部的壶流河类型，其中燕北辽西类型是核心，但晚期则逐渐向燕山南麓转移。壶流河类型与燕南型更为接近，也有小部分相邻的晋中南地区文化影响。燕南型又可称为“先燕文化”，包括大坨头文化、围坊文化、张家园文化等分支。

== 文化特征 ==
夏家店下層文化是该地区最早的[[青铜器|青铜器]]出现时期，以环首刀等兵器为主，并有明显地方特色的金器，以喇叭口状耳环、指环、金臂钏为主，是通过青铜草原之路由南西伯利亚[[印歐人|古印欧人]]的[[安德罗诺沃文化|安德罗诺沃文化]]传播而来。陶器以青灰色為主，手製泥條組成，其燒製火候較高。2005年11月，該地點還發現下層文化已有專門製造陶器的[[窯|窯]]。另外，也有该地区传统的[[玉器|玉器]]饰品。在建筑方面在燕山以北多有石构的城堡与民居，燕山以南建筑较为原始。

== 古国族属 ==
夏家店下層文化是该地区[[燕国|燕亳]]、[[孤竹|孤竹]]、[[令支|令支]]、[[蓟|蓟]]、[[肃慎|肃慎]]等古族共有的本土文化。[[殷商|殷商]]祖先的玄鸟氏部族便源于辽河上游，是小河沿文化后裔，南迁以后与西部的草原部族进行接触，在获得先进的青铜与军事技术后愈发壮大，形成了诸多部族，其中以有易氏最为强大。殷人于燕亳之地进入父系社会，拥有了文字，在[[上甲微|上甲微]]作为部族首领时期率族继续南迁，征服南方土著以后入主中原，建立了殷商王国，势力范围包括其祖源地的诸多部落氏族。
