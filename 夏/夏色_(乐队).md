== 樂團簡介 ==
夏色是2012年成立的樂團，由爵士歌手森川七月與北川加奈、山崎好詩未三名女性組成。樂團名稱的概念起源是來自於希望永遠一直都是夏季<ref name = natalie/><ref name = musicman/>。同年7月28日在{{lang|ja|Hills パン工場}}以{{lang|ja|「森川七月祭　～なついろお披露目～」}}名義舉辦首次演唱會<ref>[http://www.barks.jp/news/?id=1000081085 赋予夏天新意义的夏色、以《名侦探柯南》主题曲闪亮出道] ''[[BARKS|BARKS]]''</ref>、8月29日以單曲《[[對你的淚水如此愛戀|對你的淚水如此愛戀]]》出道。

2016年10月乐队在官网宣布活动休止，同时山崎从音乐活动中引退<ref>{{cite web |title=【山崎好美、引退のご報告】 |url=http://www.jma-model.com/blog/201609141176/ |website=モデル事務所 大阪｜ジャパンモデルエージェンシー |accessdate=2019-03-28 |language=ja |date=2016-09-14}}</ref>。
