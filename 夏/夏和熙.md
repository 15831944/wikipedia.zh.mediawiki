'''夏和熙'''（{{bd|1988年|6月19日||catIdx=F夏}}），本名'''蔡翰維'''，原藝名'''大野'''，[[台灣|台灣]]男藝人，透過中天談話性節目《[[大學生了沒|大學生了沒]]》出道。高中時曾是網球選手，他的姐夫為大聯盟馬林魚隊投手[[陳偉殷|陳偉殷]]。2014年12月推出個人專輯<ref>{{cite web |author1=陳慧貞 |title=夏和熙爆陳偉殷「姊夫不正經」 收粉紅丁字褲羞羞 |url=https://stars.udn.com/star/story/10091/3023817 |website=聯合報 |accessdate=2018-08-17}}</ref><ref>{{cite web |author1=盧薇淩 |title=跆拳腳毛弟節目上狠踢！　夏和熙翻臉「請2位離開」全笑歪 |url=https://star.ettoday.net/news/1224009 |website=ETtoday新聞雲 |accessdate=2018-08-17}}</ref>。
