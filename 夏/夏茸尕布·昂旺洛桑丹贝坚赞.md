'''夏茸尕布·昂旺洛桑丹贝坚赞'''（{{bo|z=ལ་མོ་ཞབས་དང་དཀར་པོ་ས་ཕེང་གསམ་པ་ངག་དབང་བོ་བཟང་བསན་པའླ་རལ་མཚན}}，{{bd|1660年||1728年|}}）[[藏族|藏族]]，今[[青海省|青海省]][[贵德县|贵德县]]人，第三世（不计追认）[[夏茸尕布活佛|夏茸尕布活佛]]<ref name=kao>考洛巴·陈列奥赛，夏茸尕布传，西藏研究1994年第4期</ref><ref name=xzzx>{{Cite web |url = http://www.tibetol.cn/html/2013/zdyg_0405/393.html |title =夏茸尕布活佛  |author =  |date = 2017-05-23 |publisher =  西藏在线|language =  |accessdate =  |quote =  }}</ref><ref name=cairang>才让扎西，拉茂德乾寺历史与现状研究，青海民族大学法学硕士学位论文，2016年</ref>。

==生平==
[[藏历|藏历]]十一饶迥铁鸡年（[[清朝|清朝]][[顺治|顺治]]十七年，1660年）生于[[贵德县|贵德县]]森波拉山下的羊桑曲家<ref name=kao/>。后由[[五世达赖|五世达赖]]认定为第二世夏茸尕布活佛的[[转世灵童|转世灵童]]，并师从丹希仓受戒出家。此后他曾先后师从[[嘉木样活佛|嘉木样协巴]]、[[敏珠尔活佛|敏珠尔法王]]、拉茂扎巴尖措、赤钦旦贝扎巴、洛者仓等50余位高僧。清朝[[雍正|雍正]]三年（1725年），他应[[西宁办事大臣|西宁办事大臣]]的邀请，按照[[蒙古|蒙古]]之例，将所辖僧俗部众编为四个“达参”（佐领）。他成为青海左右两翼蒙古部众的共同上师<ref name=cairang/><ref name=kao/>。
 
他一生五次进[[西藏|西藏]]。十二岁时，随老师智干巴到[[拉萨|拉萨]]，从五世达赖受[[沙弥戒|沙弥戒]]，在[[哲蚌寺|哲蚌寺]]郭莽扎仓拜洛哲坚措为师。翌年因病返回家乡。二十岁时，再赴拉萨，又入哲蚌寺郭莽扎仓学习。清朝[[康熙|康熙]]十九年（1680年），与智干巴、阿嘉昂索华旦尖措、华锐确隆华觉桑布一起，从五世达赖受[[近圆戒|近圆戒]]。不久，和智干巴一起返回家乡。从西藏回来后，以戴青帛全为施主，征收[[贵南县|贵南]]沙沟、[[茫拉乡|茫拉川]]、[[化隆回族自治县|化隆]][[巴燕镇|巴燕]]以西地区以及[[同德县|同德]]境内的[[巴曲|巴曲]]流域以东地区的僧税<ref name=cairang/>。

清朝康熙二十一年（1682年），建成了[[拉莫德钦寺|拉茂德乾寺]]，并且将[[古日寺_(尖扎县)|古日寺]]的显宗学院迁到这里<ref name=cairang/>。[[康熙帝|康熙帝]]为该寺赐名“格措林”（即“善汇寺”）<ref name=kao/>。清朝康熙三十八年（1699年），再度赴拉萨，为[[六世达赖|六世达赖]]和[[五世班禅|五世班禅]]以及各大寺院赐金屑斋，布施。清朝康熙四十四年（1705年），奉召到[[北京|北京]]受封，清朝朝廷颁发了“察罕诺门汗”诏书并封他为[[扎萨克喇嘛|扎萨克喇嘛]]<ref name=cairang/><ref name=kao/>。清朝康熙五十三年（1714年），[[七世达赖|七世达赖]]来[[塔尔寺|塔尔寺]]驻锡，他是七世达赖的受戒堪布和主要经师之一<ref name=xzzx/>。同年，七世达赖封他为“阿齐图诺门汗”。清朝雍正年间，在[[罗卜藏丹津|罗卜藏丹津]]之乱后，清朝在[[青海蒙古|青海蒙古]]设二十九旗，封他为察罕诺们汗旗一等台吉大喇嘛，由此他成为[[尖扎县|尖扎]]、[[泽库县|泽库]]、贵南、化隆西部、[[贵德县|贵德]]南部、同德西部、[[海北藏族自治州|海北]][[海晏县|海晏]]等地的[[政教合一|政教合一]]的统治者<ref name=cairang/><ref name=kao/>。

在五次进藏中，他从[[卫藏|卫藏]]等地迎请[[观世音菩萨|观世音菩萨]]像，[[第巴|第巴]]吉雪巴供奉的主尊[[宗喀巴|宗喀巴]]像（宗喀巴曾亲口说此像“酷似我”），供奉在[[布达拉宫|布达拉宫]]内的[[度母|度母]]像、[[弥勒|弥勒]]像，红[[檀香|檀香]]木制作的和人等高的观世音菩萨像等佛像300多尊，供奉在拉茂德乾寺，被赞誉为“拉茂德乾寺的大宝库”。这些文物后在[[文革|文革]]中被毁灭<ref name=kao/>。

清朝雍正六年（1728年）四月初八日兆现示寂，享年69岁。遗体火花后，眼、心、舌仍如生时，并产生诸多[[舍利|舍利]]。灵塔的主要施主是[[巴图尔珲台吉|巴图尔珲台吉]]，三尺高，用纯金铸造，供奉在[[拉莫德钦寺|拉茂德乾寺]]噶尔哇经堂中。1958年，灵塔和灵骨外壳在[[民主改革|民主改革]]中被上交给政府。但其眼、心、舌及舍利被信众收藏，如今供奉在拉茂德乾寺<ref name=cairang/><ref name=kao/>。
