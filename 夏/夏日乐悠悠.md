《'''夏日乐悠悠'''》是2011年9月30日于中国大陆上映的爱情电影，由[[彭于晏|彭于晏]]和[[Angelababy|Angelababy]]领衔主演。影片在中国大陆首周三天收获125万元<ref name="entgroup">{{cite web | title=《夏日乐悠悠》艺恩主页 | url=http://m.entgroup.cn/591755/ | publisher=艺恩网}}</ref>，至10月16日累计460万元<ref>{{cite web | title=中国电影报2011年10月19日新浪微博 | url=http://weibo.com/2304129841/xtsTJErM9 | date=2011-10-19}}</ref>，最终票房为520万元人民币<ref name="entgroup"/>。

== 剧情 ==
夏米的父母双双葬身大海，将最后一根救命木头留给了她。她被渔夫大叔救得，此后一直痛恨大海。夏米有读懂唇语的神奇技能，在上班时，凭借懂唇语而被一名叫JK的客户看中，被派往一个小岛调查JK的对手游乐乐的非法诈骗行为。在前往海岛途中，她遇上了游乐乐和一对新娘新郎的结婚船，当时游乐乐假扮成海賊持刀威逼新娘，要新郎说出结婚前的所作所为，这是新娘对新郎的考验。而夏米却不知道，于是扔鞋过去，打伤了游乐乐。游乐乐和夏米的关系自此进入紧张状态。一次，两人同时出海，遇到一个熟人，才知道两人的关系并非简单。这时候，一群奇怪的高层来到了海岛，游乐乐才知道自己也是被陷害，夏米向困境中的游乐乐伸出援手，两人的爱情也由此开始。游乐乐不为人知的秘密也一一被揭发。<ref name="webtx11 " >{{cite web |url=http://ent.qq.com/a/20120109/000415.htm |title=这个夏天特别长——港三区寰亚《夏日乐悠悠》 |accessdate=2012年1月9日 |work=|publisher= 腾讯网|language= zh-hans}}</ref>
