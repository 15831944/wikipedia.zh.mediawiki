==簡介==
'''[[垒球|垒球]]'''自[[1996年夏季奥林匹克运动会|1996年亚特兰大夏季奥运会]]起成为[[夏季奥运会|夏季奥运会]]比赛项目（只有女子），2005年7月和2009年7月[[国际奥委会|国际奥委会]]投票决定2012年和2016年暫停奥运会垒球比赛<ref name="IOC sports">{{cite web |url=http://www.olympic.org/uk/organisation/commissions/programme/full_story_uk.asp?id=1437 |title=Singapore 2005: 2012 Olympic Sport Vote |publisher=International Olympic Committee |accessdate=2007-03-18}}</ref>，2006年2月9日确认。<ref name="IOC sports2">{{cite web |url=http://www.olympic.org/uk/organisation/commissions/programme/full_story_uk.asp?id=1657 |title=Olympic programme for London 2012 confirmed |work=The Movement |publisher=International Olympic Committee |accessdate=2007-03-18}}</ref>

2010年7月，[[國際奧委會|國際奧委會]]決定將壘球列入2020年奧運會侯選項目。為此原國際壘球總會也在2013年3月和原國際棒球總會合併為[[世界棒壘球聯盟|世界棒壘球聯盟]]。此舉不但節省場地資源。也有益於壘球運動的推展。 

2013年9月8日，国际奥林匹克委员会表決2020年奧運會增列項目；棒壘球卻以24:49票敗給[[角力|角力]]和正式項目絕緣。但同年11月，IOC主席巴哈認為壘球可重返[[2020年奧運|2020年奧運]]。。此事出現轉機，隨即擬定奧運改革方案「2020計畫」目的在讓主辦城市在比賽項目選擇有自主權。該方案在2014年2月在俄羅斯索契的年會中提出；12月8日在摩納哥舉行的臨時年會中以98票贊成表決通過。「2020計畫」也刪除現行28大項的上限。一般認為壘球幾乎已確定重返2020東京奧運。
