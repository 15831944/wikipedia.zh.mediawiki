'''夏茸尕布·昂旺确珠丹贝坚赞'''（{{bo|z=ལ་མོ་ཞབས་དང་དཀར་པོ་ས་ཕེང་དག་པ་ངག་དབང་མཆོག་གབ་བསན་པའླ་རལ་མཚན}}，{{bd|1832年||1872年|}}）今[[青海省|青海省]][[湟中县|湟中县]]人，第六世（不计追认）[[夏茸尕布活佛|夏茸尕布活佛]]<ref name=kao>考洛巴·陈列奥赛，夏茸尕布传，西藏研究1994年第4期</ref><ref name=xzzx>{{Cite web |url = http://www.tibetol.cn/html/2013/zdyg_0405/393.html |title =夏茸尕布活佛  |author =  |date = 2017-05-23 |publisher =  西藏在线|language =  |accessdate =  |quote =  }}</ref><ref name=cairang>才让扎西，拉茂德乾寺历史与现状研究，青海民族大学法学硕士学位论文，2016年</ref>。

==生平==
[[藏历|藏历]]十四饶迥水龙年（[[清朝|清朝]][[道光|道光]]十二年，1832年）生于[[安多|安多]]贡本米纳部落拉顶村（位于今[[青海省|青海省]][[湟中县|湟中县]]境内），传说他是[[赛赤活佛|拉茂赛赤活佛]]的宗亲。生父名确迥扎西，生母名措科。由色康巴洛桑丹增嘉措等等上师喇嘛认定为第五世夏茸尕布活佛的[[转世灵童|转世灵童]]，在色康巴洛桑丹增嘉措膝下出家。藏历铁狗年四月的节日上，在[[嘉木样·罗桑图旦久美嘉措|第三世嘉木样活佛]]身前受[[具足戒|具足戒]]，取法名“洛桑楚成嘉措”<ref name=cairang/><ref name=kao/>。 

在[[拉莫德钦寺|拉茂德乾寺]]，他拜尼玛丹贝坚赞为其经师，学习显密经典。因为天性好动，所以不久便奉[[西宁办事大臣|西宁办事大臣]]之命，被送往[[塔尔寺|塔尔寺]]学习显密经典<ref name=cairang/><ref name=kao/>。 

自第五世夏茸尕布活佛时期乌尔格和达如玉部落遭到霍尔黑账部落抢劫后，夏茸尕布活佛（即察罕诺们汗）属下各部落人民无法安居。在他[[坐床|坐床]]初期，部落因为历年属民被抢劫和逃离而面临解体。清朝道光十八年（1838年）十二月，由[[黄河|河]]北两翼蒙古正副盟长郡王[[车凌端多布_(青海扎萨克郡王)|车凌端多布]]、郡王[[恭楚克吉默特|恭楚克吉默特]]等呈请：“因察罕诺们汗年甫七岁，无力管理旗务，旗民逃散，所剩人户只有原来的十分之四。请移过河北，由郡王车凌端多布代理旗务。”同年十二月二十七日，奉硃批准其所请，令车凌端多布留心稽查，不准滋事作贼之人混淆移过河北。翌年正月二十二日，察罕诺们汗旗宰桑等，率所部善良之人渡过[[黄河|黄河]]移居河北，至正月二十六日共移走271户，1122口人。此后虽仍有部分游牧属民移居河北，但许多游牧属民以及绝大部分农户仍留在黄河以南的原住地，由此形成了察罕诺们汗属民分居黄河南北的情况。河北属民在[[湟水|湟水]]西源以及达玉一带，即今[[青海省|青海省]][[海北藏族自治州|海北藏族自治州]][[海晏县|海晏县]]；河南属民在今[[黄南藏族自治州|黄南藏族自治州]][[尖扎县|尖扎县]]、[[海东市|海东市]][[化隆回族自治县|化隆回族自治县]]、[[海南藏族自治州|海南藏族自治州]][[贵德县|贵德县]]、[[贵南县|贵南县]]、[[同德县|同德县]]等地<ref name=cairang/>。

藏历十五饶迥水猴年（清朝[[同治|同治]]十一年，1872年）阴历十二月圆寂，享年41岁<ref name=cairang/><ref name=kao/>。
