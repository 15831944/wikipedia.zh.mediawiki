'''夏威夷盖饭'''（{{lang-haw|Poke}}，音“剖剋”，意为“切块”）是[[夏威夷|夏威夷]]人的一种料理生鱼的方法，将生鱼切块，然后用酱汁腌制后盖于米饭上<ref>{{cite web |author1=Fairy Shanghai |title=让你大“块”朵颐的“盖浇饭”，竟然能减肥！
|url=http://www.sohu.com/a/226818946_552335 |website=搜狐 |accessdate=2018年11月13日}}</ref>。

传统上，Poke在夏威夷作为一种凉菜，主要食材为小珊瑚魚和章鱼，以海盐腌制好后辅以[[石栗|石栗]]和[[海藻|海藻]]。

随着日本移民的到来，[[日本料理|日本料理]]对原本的Poke从用料到形式产生了极大的改变。除原有的食材外，[[金枪鱼|金枪鱼]]更为流行<ref>{{cite web |title=6 件關於夏威夷魚生飯 Poke 的重要事情 |url=https://guide.michelin.com/hk/hong-kong-macau/features/6-things-to-know-about-real-hawaiian-poke/news |website=米其林指南 |accessdate=2018-11-13}}</ref>，还可加有[[毛豆|毛豆]]、日式海藻，调味料则变为酱油、小葱以及香油和[[柚子醋|柚子醋]]，并以[[盖饭|盖饭]]的形式佐寿司姜作为主菜出售。

在传到[[美国本土|美国本土]]西岸后，主料则可选为[[三文鱼|三文鱼]]，当地流行的[[牛油果|牛油果]]、以及[[墨西哥辣椒|墨西哥辣椒]]和[[玉米|玉米]]也会被当做食材，而酱料则包括[[是拉差香甜辣椒醬|是拉差香甜辣椒醬]]甚至芥末蛋黄酱<ref>{{cite web |title=现在流行夏威夷菜饭 腌生鱼拌饭Poke Bowl |url=https://www.zaobao.com.sg/zlifestyle/food/story20170304-731743 |website=联合早报 |accessdate=2018-11-13}}</ref>
