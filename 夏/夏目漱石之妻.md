{{日本作品|夏目漱石之妻|夏目漱石の妻}}，2016年9月24日於[[日本|日本]][[NHK電視台|NHK電視台]]週六時段（21:00 - 22:13，[[日本標準時|JST]]）播出的[[電視連續劇|電視連續劇]]，共四集，由[[尾野真千子|尾野真千子]]主演<ref>{{Cite web |date=2016-09-07 |url=http://www.nikkansports.com/entertainment/news/1706374.html |title=尾野真千子「夏目漱石の妻」で長谷川博己と夫婦役 |publisher=[[日刊體育|日刊體育]] |accessdate=2016-10-31}}</ref>。

== 製作經緯 ==
2016年為日本知名作家[[夏目漱石|夏目漱石]]逝世100週年，其妻{{link-ja|夏目鏡子|夏目鏡子}}口述，出版了一部名為《漱石の思い出》的回憶錄，此劇即為根據此回憶錄為原案，製作改編而成的真人版連續劇<ref>{{Cite web |date=2016-09-23 |url=http://tvfan.kyodo.co.jp/feature-interview/1068649 |title=【夏目漱石の妻】長谷川博己「尾野さんはうらやましいぐらい自由」尾野真千子「好きな人だと顔が近くても苦にならない（笑）」 |work=エンタメOVO |publisher=共同通信社|accessdate=2016-10-31}}</ref>。以夏目漱石的妻子{{link-ja|夏目鏡子|夏目鏡子}}為視角、《[[我是貓|我是貓]]》、《[[少爺|少爺]]》、《[[心_(小說)|心]]》等著名的夏目漱石作品為基底，描述在動盪的[[明治|明治]]時代中夏目夫妻的生活<ref>{{Cite web |date=2016-03-30 |url=https://www.nhk.or.jp/dramatopics-blog/6000/241151.html |title=尾野真千子×長谷川博己『夏目漱石の妻』制作開始! |work=NHKドラマ |publisher=日本放送協会|accessdate=2017-10-02}}</ref>。

此劇獲選第72回日本[[文化廳|文化廳]]所舉辦的「{{link-ja|文化廳藝術祭|芸術祭 (文化庁)}}」電視劇部門優秀賞，並於2017年10月18日至10月21日重播<ref>{{Cite web |date=2017-09-28 |url=http://www.nhk.or.jp/dramatopics-blog/90000/280600.html |title=再放送情報「夏目漱石の妻」 |work=NHKドラマ |publisher=[[日本放送協会|日本放送協会]] |accessdate=2017-10-02}}</ref><ref>{{Cite web |date=2017-10-04 |url=http://www.nhk.or.jp/dramatopics-blog/20000/281146.html |title=平成29年度 文化庁芸術祭 参加作品決定! |work=NHKドラマ |publisher=NHK |accessdate=2017-10-08}}</ref>。

== 評價反響 ==
此劇播出後，其精良的製作和[[尾野真千子|尾野真千子]]、[[長谷川博己|長谷川博己]]的演技受到普遍好評，並指此劇為兩位演員的代表作之一<ref>[http://topics.smt.docomo.ne.jp/article/galac/entertainment/galac-20161216110813172 圧巻！尾野真千子の存在〜土曜ドラマ「夏目漱石の妻」]</ref><ref>[https://www.oricon.co.jp/news/2079939/full/ 『夏目漱石の妻』が好調 尾野真千子&長谷川博己の配役ズバリ]</ref><ref>[https://mainichi.jp/articles/20161007/dde/018/070/021000c 「夏目漱石の妻」　迫真の演技で夫婦の成長描く]</ref>，此劇亦獲得該屆多項電視劇獎。

== 故事大綱 ==
原本在富裕家庭中長大的鏡子（[[尾野真千子|尾野真千子]] 飾），於19歲時和金之助（夏目漱石，[[長谷川博己|長谷川博己]] 飾）[[相親|相親]]，兩人決定結婚。婚後，金之助自[[英國|英國]]留學歸國，罹患神經衰弱疾病，精神狀態極度不安定，加上家中資助許多有志創作的學生、亦有四個孩子需養育，夏目家的經濟捉襟見肘，夫妻倆漸生摩擦。原擔任官職的鏡子父親中根（{{link-ja|舘廣|舘ひろし}} 飾）此時於政界失勢，在外借貸並向鏡子要求請金之助當自己的保證人，鏡子忍痛拒絕父親。其後，金之助的養父塩原（[[竹中直人|竹中直人]] 飾）前來見從小被拋棄的金之助，表示金之助有義務資助自己晚年生活，並以當年的字據要脅金之助。已有胃病的金之助此時健康狀況加劇。於是鏡子將家中所有存款交給養父，設法要回了字據。

養父不再糾纏金之助後，金之助的胃病症狀卻不見減緩，甚至大量吐血.......。
