'''夏荊山'''（{{bd|1923年|8月1日}}），1923年生於[[山东省|山東省]][[潍坊市|濰坊市]]。字光樺，又名菩謙、楠竺、慧芝、林生、林長壽。書画藝術家，擅長工筆畫、佛畫、山水人物畫。

== 生平 ==
夏荊山師從當代畫家[[郭味渠|郭味蕖]]，1949年以[[军医|军医]]身份随[[中華民國政府|中華民國政府]]軍隊到[[臺灣|台灣]]，拜南亭法師為師，开始信奉佛教。夏荊山后来結識名人[[南懷瑾|南懷瑾]]，1964年隨[[南懷瑾|南懷瑾]]搬遷至[[台北|台北]][[建國南路_(臺北市)|建國南路]]，創立[[立康美術有限公司|立康美術有限公司]]。

從軍隊退役後搬遷至[[台北|台北]]創立[[立康美術有限公司|立康美術有限公司]]，多次捐助國際在台協會。1971年與家人移居[[美国|美國]][[加利福尼亚州|加州]]，進入美術學院研習，積極推廣[[中国画|中華傳統書畫藝術]]。

1988年，閉關十月，體會到[[佛像|佛像]]微妙之境，發願餘生只畫殊勝的[[佛像|佛像畫]]。1994年，回到[[中國|中國]]，於[[北京市|北京]]開辦「[[荊山書畫院|荊山書畫院]]」，免費招收學生。

夏荊山曾援助中國古寺修復工程<ref name=":0">{{Cite web|title=國畫大師夏荊山的佛畫藝術及公益行動|url=https://artouch.com/news/content-2028.html|work=典藏 ARTouch.com|accessdate=2017-10-13|language=zh-TW}}</ref>，包括[[陕西省|中國陝西省]][[法门寺|法門寺]]大佛殿、[[浙江省|中國浙江省]][[天台山國清寺|天台山國清寺]]、[[山西省|中國山西省]][[五台山|五台山]]等。

2014年，於[[臺灣|台灣]]創立[[財團法人夏荊山文化藝術基金會|財團法人夏荊山文化藝術基金會]]。

== 年表 ==

* 1923年，出生[[山东省|中國山東省]][[潍坊市|濰坊市]]。
* 1939年，師從[[郭味渠|郭味蕖]]，學習書畫。
* 1949年，隨[[中華民國政府|中華民國政府]]軍隊到[[臺灣|台灣]]，以醫官身份駐軍[[嘉義縣|嘉義]]。
* 1954年，拜南亭法師為師，[[皈依|皈依]][[佛教|佛門]]。
* 1958年，結緣[[南懷瑾|南懷瑾]]。
* 1964年，隨[[南懷瑾|南懷瑾]]搬遷至[[台北|台北]][[建國南路_(臺北市)|建國南路]]，創立[[立康美術有限公司|立康美術有限公司]]。
* 1971年，與家人移居[[美国|美國]][[加利福尼亚州|加州]]。
* <span class="s6"></span>1979年，資助古寺修復，包括[[陕西省|中國陝西省]][[法门寺|法門寺]]大佛殿、[[浙江省|中國浙江省]][[天台山國清寺|天台山國清寺]]、[[山西省|中國山西省]][[五台山|五台山]]等<ref name=":0" />。
* 1988年，閉關十月，一日作[[觀音|觀音]]，發願餘生只畫殊勝的[[佛像|佛像]]畫。
* 1994年，回[[中國|中國]]，於[[北京市|北京]]開辦「[[荊山書畫院|荊山書畫院]]」，免費招收學生。
* 2009年，於[[中國|中國]][[无锡市|江蘇省無錫市]]召開的「第二屆[[世界佛教論壇|世界佛教論壇]]」發表《[[佛像典藏|佛像典藏]]》畫冊，用[[工笔画|工筆]][[重彩|重彩]]繪制的5163幅[[佛像|佛像]][[繪畫|繪畫]]作品，全套9卷、每卷9冊，共有81集，被譽為「造像版的[[大藏经|大藏經]]」。
* 2010年，作品《[http://www.xjsacf.com/TW/News04/ugC_News_Detail.asp?hidNewsCatID=4&hidNewsID=12 自在觀音菩薩像]》受[[中國|中國]][[故宫博物院|北京故宮博物院]]典藏。
* <span class="s6"></span>2014年，於[[臺灣|台灣]]創立[[財團法人夏荊山文化藝術基金會|財團法人夏荊山文化藝術基金會]]。
* 2015年，發行《[[荊山美學|荊山美學]]》雜誌。
* 2015年，發行《[[夏荊山藝術論衡|夏荊山藝術論衡]]》學術期刊。
* 2017年，籌辦「[[荊山經典文創藝術獎|荊山經典文創藝術獎]]」，鼓勵年輕學子創意思考，結合書畫藝術精髓進行創作與設計。

== 展覽紀錄 ==

* 2009年，9月29日至10月30日，於[[陕西省|中國陝西省]][[西安市|西安市]]的[[法门寺|法門寺]]舉辦「佛畫大師夏荊山佛像藝術展」。 
* 2010年，3月中旬至6月中旬，於[[中國|中國]][[北京故宫博物院|北京故宮博物院]]舉辦「佛像典藏：夏荊山佛像藝術展」<ref>{{Cite web|title=夏荆山佛像艺术展在故宫博物院举行(组图)_艺术中国|url=http://art.china.cn/zixun/2010-03/17/content_3422600.htm|work=art.china.cn|accessdate=2010-03-17|author=|date=|publisher=}}</ref>。
* 2012年，1月，於「洛杉磯美術展：歷史與傳統」的中國特展展出多幅畫作。
* 2012年，4月，於[[澳門|澳門]]「佛頂骨舍利睹禮大會」展出百餘張畫作。
* 2016年，12月17日至12月31日，於[[臺灣|台灣]][[臺南市|台南]][[吳園藝文中心|吳園藝文中心]]舉辦「夏荊山故事館：經典藝術國際特展」。
* 2017年，4月，於[[臺灣|台灣]][[新北市|新北市]][[石碇區|石碇]][[無極天明宮|無極天明宮]]舉辦「春之頌：夏荊山書畫藝術裝置展」。

* 2017年，9月，於[[臺灣|台灣]][[臺南市|台南]][[南臺科技大學|南臺科技大學]]舉辦「荊山經典．聚南臺：夏荊山書畫藝術特展」。
* 2018年，4月，二度於[[臺灣|台灣]][[新北市|新北市]][[石碇區|石碇]][[無極天明宮|無極天明宮]]舉辦「櫻花季・春之頌：夏荊山書畫藝術裝置展」。
* 2018年，7月30日至8月14日，於[[中國國家博物館|中國國家博物館]]舉辦「夏荊山藝術展」。
* 2018年，12月1日至12月14日，於[[南京大学|南京大學博物館]]舉辦「荊山Style：夏荊山書畫藝術暨荊山經典文創藝術獎得獎作品展」。
* 2019年，3月，三度於[[臺灣|台灣]][[新北市|新北市]][[石碇區|石碇]][[無極天明宮|無極天明宮]]舉辦「櫻花季・春之頌：夏荊山書畫慈善藝術鑑賞會」，首度於[[臺灣|台灣]]展出巨幅真跡〈[[龍王朝觀音|龍王朝觀音]]〉畫作。
