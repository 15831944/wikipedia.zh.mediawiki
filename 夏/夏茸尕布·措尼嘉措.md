'''夏茸尕布·措尼嘉措'''（{{bo|z=ལ་མོ་ཞབས་དང་དཀར་པོ་ས་ཕེང་དང་པོ་ཚོགས་ག཈ླས་ར་མཚ}}，{{bd|？||？|}}）[[藏族|藏族]]，[[前藏|前藏]]拉茂仁青岗人，第一世（不计追认）[[夏茸尕布活佛|夏茸尕布活佛]]<ref name=kao>考洛巴·陈列奥赛，夏茸尕布传，西藏研究1994年第4期</ref><ref name=xzzx>{{Cite web |url = http://www.tibetol.cn/html/2013/zdyg_0405/393.html |title =夏茸尕布活佛  |author =  |date = 2017-05-23 |publisher =  西藏在线|language =  |accessdate =  |quote =  }}</ref><ref name=cairang>才让扎西，拉茂德乾寺历史与现状研究，青海民族大学法学硕士学位论文，2016年</ref>。

==生平==
夏茸尕布·措尼嘉措生于西藏卫地拉茂仁青岗（{{bo|z= ལ་མོ་རླན་ཆེན་སང}}），后来在策赛寺出家。先后进入[[哲蚌寺|哲蚌寺]]郭莽扎仓、[[下密院|下密院]]学习佛法，系统学习了《五部经典》以及显密二宗经典。不久便任下密院引经师，获得[[三世达赖|三世达赖]]重用。[[明朝|明朝]][[万历|万历]]五年（1577年），[[青海湖|青海湖]]左右二翼[[蒙古|蒙古]]四大部落亲王，蒙古[[俺答汗|俺答汗]]、二十四旗的王公及属民这三方同时邀三世达赖访问。三世达赖赴青海湖畔与俺答汗相会，措尼嘉措同行。三世达赖在与俺答汗临别时，应[[火落赤|火落赤]]的邀请，派措尼嘉措作为三世达赖的代表去二十四旗讲经。措尼嘉措在二十四旗调解旗内矛盾，弘传佛法，还任二十四旗的“翁拉”（{{bo|z=དབ་བ}}，至尊上师）。当时师从他的僧人达600余名。火落赤特将[[尖扎县|尖扎县]]阿噶香曲林寺奉献给他。此后不久，[[明朝|明朝]][[万历帝|万历帝]]封他为“阿噶格杰尕布”（{{bo|z=ར་སར་ས་སེས་དཀར་པོ}}）并敕赐金印。“格杰”藏语意为转世，“尕布”藏语意为白色，因此当地民众称他为“夏茸尕布”（又譯「夏容嘉布」，意为善业白色活佛）。此后，因弘法有功，他又获得“察汗[[诺门汗|诺门汗]]即夏茸尕布”的尊号。此即夏茸尕布名称之由来<ref name=kao/><ref name=xzzx/><ref name=cairang/>。

晚年，他在阿噶香曲林寺苦修坐定，获圆满彻悟。他在[[藏族|藏族]]和[[蒙古族|蒙古族]]人民之间架起了友好桥梁。蒙古族尊封他为第七世[[呼毕勒罕|呼毕勒罕]]。他圆寂后，灵塔建在今[[青海省|青海省]][[贵德县|贵德县]]麦力干扎仓，后在[[文革|文革]]中被毁<ref name=kao/><ref name=xzzx/><ref name=cairang/>。
