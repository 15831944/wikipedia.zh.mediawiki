'''夏茸尕布·洛追嘉措'''（{{bo|z=ལ་མོ་ཞབས་དང་དཀར་པོ་ས་ཕེང་ག཈ླས་པ་བོ་གོས་ར་མཚ}}，{{bd|1610年||1659年|}}）[[蒙古族|蒙古族]]，第二世（不计追认）[[夏茸尕布活佛|夏茸尕布活佛]]<ref name=kao>考洛巴·陈列奥赛，夏茸尕布传，西藏研究1994年第4期</ref><ref name=xzzx>{{Cite web |url = http://www.tibetol.cn/html/2013/zdyg_0405/393.html |title =夏茸尕布活佛  |author =  |date = 2017-05-23 |publisher =  西藏在线|language =  |accessdate =  |quote =  }}</ref><ref name=cairang>才让扎西，拉茂德乾寺历史与现状研究，青海民族大学法学硕士学位论文，2016年</ref>。

==生平==
生于[[藏历|藏历]]第十饶迥铁狗年（[[明朝|明朝]][[万历|万历]]三十八年，1610年），是[[蒙古|蒙古]][[土默特部|土默特部]]首领[[火落赤|火落赤]]之子。由[[四世班禅|四世班禅]]确认为第一世拉茂夏茸尕布活佛的转世化身。他继承了第一世夏茸尕布活佛的事业，先后拜四世班禅，[[隆务寺|隆务寺]]曲哇活佛，[[夏日仓·噶丹嘉措|夏日仓·噶丹嘉措]]等人为师，学习经典<ref name=kao/><ref name=xzzx/><ref name=cairang/><ref name=xzzx/>。

洛追嘉措早年常驻阿哇寺（即阿噶尔绎曲林寺）、[[洛多杰智合寺|洛多杰智合寺]]（{{bo|z=ལོ་རོ་རེ་བག}}）。[[清朝|清朝]][[顺治|顺治]]三年（1646年），在丹麻曲吉·奥色扎西（又译“丹麻曲结·鄂色扎西”）的协助下，开始建造古日殊胜寺，聚集了300余位僧人，委托奥色扎西任第一任[[堪布|堪布]]<ref name=cairang/>。他将原先只有一座建筑的[[古日寺_(尖扎县)|古日寺]]（意即“帐房”）扩建为经殿宽广宏大，可容纳300名僧众的古日殊胜寺，并且将祖籍属下的100家牧户作为“拉德”<ref name=kao/>。寺内主要供奉[[固始汗|固始汗]]（蒙古[[和硕特部|和硕特部]]首领）、[[巴图尔珲台吉|巴图尔珲台吉]]（蒙古[[准噶尔部|准噶尔部]]首领）的银质灵塔、[[拉萨|拉萨]][[释迦牟尼|释迦牟尼]]金像及[[克主杰|克主杰]]的僧帽、班禅亲赐的[[妙音天女|妙音天女]]画像等宝物。据说此妙音天女画像后被迎至[[拉莫德钦寺|拉茂德乾寺]]。而后因洛追嘉措具有慧因德缘，妙音天母画像又飞回古日殊胜寺，如今供奉在古日殊胜寺经堂东侧，成为该寺护法<ref name=cairang/>。

他所处的时代，民族纠纷频发。为缓和各类矛盾，他曾奉[[五世达赖|五世达赖]]之命，先后赴冲突中的[[喀尔喀蒙古|喀尔喀蒙古]]和[[卫拉特蒙古|卫拉特蒙古]]双方调停议和。为此，他本人获“察罕诺门汗”称号。而后他平息了[[化隆县|化隆县]]的卡力岗、[[巴燕镇|巴燕]]两个部落间的长期纷争，获得了固始汗、巴图尔珲台吉的赞赏<ref name=kao/><ref name=cairang/><ref name=xzzx/>。固始汗等人奉他为上师，献来很多属民，并且将化隆的觉扎、龙钦、格许兰敖等部落划给他管辖<ref name=cairang/>。他还解决了果哇诺门汗属下部落和察罕诺门汗属下百姓间的冲突。他曾获皇帝赐的“[[扎萨克|扎萨克]]印”。他加强了[[蒙古族|蒙古族]]和[[藏族|藏族]]人民间的友好往来，使[[藏传佛教|藏传佛教]]在蒙藏地区广泛传播<ref name=kao/>。

藏历十一饶迥土猪年（[[清朝|清朝]][[顺治|顺治]]十六年，1659年），夏茸尕布·洛追嘉措在古日殊胜寺圆寂，享年50岁。灵塔毁于[[文化大革命|文化大革命]]<ref name=kao/><ref name=xzzx/>。
