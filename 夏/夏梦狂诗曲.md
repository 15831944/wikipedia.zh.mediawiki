'''夏梦狂诗曲'''，2016年中国大陆都市音乐偶像剧。改编自[[君子以泽|君子以泽]]同名小说，由上海时悦影视文化有限公司出品，芒果TV、刚泰影视、上海张翰影视文化工作室联合出品<ref>{{Cite web |url=http://yule.sohu.com/20160328/n442573580.shtml |title=《夏梦狂诗曲》曝海报 暗喻华丽背后的灵魂之声|accessdate=2017-12-01 |author= |date=2016-03-28 |work= |publisher=搜狐娱乐|language=Zh-hans }}</ref>，[[杨玄|杨玄]]导演，[[张翰|张翰]]、[[高準熹|高準熹]]、[[朱一龙|朱一龙]]、[[宋轶|宋轶]]、[[朱梓骁|朱梓骁]]、[[郭子千|郭子千]]、[[陈奕|陈奕]]、[[刘长德|刘长德]]、[[迟宁宁|迟宁宁]]等主演。2016年3月于上海开机<ref>{{Cite web |url=http://yule.sohu.com/20160315/n440463912.shtml |title=《夏梦狂诗曲》上海开机 小花旦宋轶为爱疯魔|accessdate=2017-12-01 |author= |date=2016-03-15 |work=王媛 UK032 |publisher=搜狐娱乐|language=Zh-hans }}</ref>，同年7月于维也纳杀青<ref>{{Cite web |url=http://ent.sina.com.cn/v/m/2016-07-13/doc-ifxtwiht3676024.shtml |title=《夏梦狂诗曲》杀青 张翰高俊熙浪漫携手|accessdate=2017-12-01 |author= |date=2016-07-13 |work= |publisher=新浪娱乐|language=Zh-hans }}</ref>。2017年3月，因“[[禁韩令|禁韩令]]”，以韩国演员高俊熙主演的《夏梦狂诗曲》短期内播出无望<ref>{{Cite web |url=http://share.iclient.ifeng.com/news/shareNews?forward=1&aid=119594266#backhead |title=禁韩令一出，张翰多部电视剧播出无望，恐成血本无归的最大输家|accessdate=2017-12-13 |author= |date=2017-03-06 |work= |publisher=凤凰新闻|language=Zh-hans }}</ref>，截至2017年底仍未有定档的消息。
