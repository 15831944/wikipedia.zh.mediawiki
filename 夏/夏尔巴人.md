'''夏尔巴人'''（[[藏語|藏語]]：{{藏|'''ཤར་པ'''}}），或音譯為'''雪巴人'''、'''謝爾巴人'''，是一支散居在[[尼泊尔|尼泊尔]]、[[中華人民共和國|中国]]、[[印度|印度]]和[[不丹|不丹]]等国边境[[喜玛拉雅山脉|喜玛拉雅山脉]]两侧的民族。

== 分布 ==
人口约15万人，主要居住在尼泊尔境内，中国[[西藏|西藏]]境内有约1200人，主要生活在[[定结县|定结县]][[陈塘镇_(定结县)|陈塘镇]]和[[聂拉木县|聂拉木县]][[樟木镇_(日喀则市)|樟木镇]]。

== 族源 ==
“夏尔”在[[藏语|藏语]]中意为东，“巴”意为人，族名即表示“（来自）东方的人”，相传先祖来自中国[[甘孜|甘孜]]地区。最近有先祖為[[党項羌|党項羌]]（[[西夏|西夏]]王族）的說法<ref>唐榮堯，王朝湮滅：為西夏帝國叫魂。光明日报出版社，2006。</ref>。

== 語言 ==
[[藏语群|藏语群]][[卫藏方言|卫藏方言]]下的[[夏爾巴語|夏爾巴語]]，但无文字，书面使用[[藏文|藏文]]。

== 習俗 ==
夏尔巴人有姓无氏，同姓不婚，一般也不与外族通婚。宗教信仰为[[藏传佛教|藏传佛教]]，以[[萨迦派|萨迦派]]和[[噶举派|噶举派]]为主，保留有较多的原始信仰。夏尔巴人忌食[[鱼|鱼]]、[[狗|狗]]和小[[牛|牛]]，由于受[[印度教|印度教]]影响，虽不忌牛肉，但从不主动屠宰牛。主食以[[玉米|玉米]]为主，讲究使用[[茴香|茴香]]、[[辣椒|辣椒]]等[[调味料|调味料]]。

夏尔巴人由于常年生活在高山地带，是天生的登山向导，为各国登山队提供向导和后勤服务已成为夏尔巴人的主要经济来源之一。根據尼泊爾政府規定，一個夏尔巴人每一次服務只能背負20[[磅|磅]]，收費125[[美元|美元]]左右。但是年輕力壯的夏尔巴人通常會加倍負擔，好多賺一點錢<ref>閻紀宇，2014，世界之巔急先鋒 雪巴人：永遠的無名英雄</ref>。
