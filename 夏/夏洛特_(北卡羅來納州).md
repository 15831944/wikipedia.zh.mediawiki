'''夏洛特'''（{{lang-en|Charlotte}}）是[[美國|美國]][[北卡羅來納州|北卡羅來納州]]最大和全國第17大的城市。2004-2014期间，夏洛特共计新居民88.8万人，美国大城市排名第一。<ref>{{cite web|url=http://www.charlottestories.com/heres-charlotte-became-fastest-growing-city-country-past-decade/|title=Here’s Why Charlotte Became The Fastest Growing City in The Country Over The Past Decade|last=|first=|date=May 2017}}</ref>根据美国2005-2015普查数据，夏洛特是全国千禧年一代移居最多的大城市。<ref>{{cite web|url=https://www.bizjournals.com/charlotte/news/2016/11/16/millennial-magnet-charlotte-ranks-as-top-city-in.html|title=Millennial magnet: Charlotte ranks as top city in young-adult population growth|last=|first=|date=November 2016}}</ref>根據2011年估計有人口751,087人，<ref name="夏洛特商会">[http://charlottechamber.com/eco-dev/demographics-economic-profile/ 夏洛特商会] {{webarchive|url=https://web.archive.org/web/20120525212931/http://charlottechamber.com/eco-dev/demographics-economic-profile/ |date=2012-05-25 }}</ref>略小于[[旧金山|旧金山]]等城市。整個[[美国大都市圈|夏洛特都會區]]在2010年的估計裡有人口1,758,038人。而夏洛特-[[加斯頓尼亞_(北卡羅來納州)|加斯頓尼亞]]-[[索斯貝利_(北卡羅來納州)|索斯貝利]][[綜合統計區|綜合統計區]]（MSA）則在2010年估計裡共有人口2,402,623人。<ref name="CB2">[http://www.census.gov/popest/metro/tables/2008/CBSA-EST2008-02.csv 2008年綜合統計區人口估計 - 美國人口普查局（CSV）]</ref>

夏洛特也是[[梅克倫堡縣_(北卡羅萊納州)|梅克倫堡縣]]的縣府，位置在於北卡羅來納州中南部、靠近與[[南卡羅萊那州|南卡羅萊那州]]的邊界。夏洛特是以[[英國|英國]]國王[[喬治三世|喬治三世]]的妻子[[夏綠蒂王后|夏綠蒂王后]]而名，也因此得到女王之城（The Queen City）這個暱稱；[[俄亥俄州|俄亥俄州]][[辛辛那提|辛辛那提]]和[[纽约州|纽约州]][[水牛城_(纽约州)|水牛城]]同樣也有“女王之城”這個暱稱。[[美國獨立戰爭|美國獨立戰爭]]時，英軍將領[[康華利將軍|康華利將軍]]（General Cornwallis）在被夏洛特當地居民激烈抵抗後撤出城市，他後來稱夏洛特為「叛亂者的一個黃蜂窩」（a hornet's nest of rebellion）。夏洛特的居民一般被稱為''Charlottean''（[[國際音標|IPA]]: {{IPA|ˌʃaɹləˈtʰiːən}}）。

如今的夏洛特是一座高度现代化的工商业城市，是美国重要的金融、贸易和交通运输中心，也是美国东南部地区经济发展速度最快的城市。美国的12家银行，其中包括5家美国最大的银行，在夏洛特市经营银行金融业务。

==歷史==
早於幾千年前夏洛特已有原住民居住。而首個[[歐洲|歐洲]]人到來夏洛特的紀錄，是於1567年。城市成立於1768年。

==地理与气候==
夏洛特属典型的[[副热带湿润气候|副热带湿润气候]]，四季分明，气温变化相对和缓，全年降水分配均匀，代表美国东南部内陆地区的气候。冬季温和，微潮，时而偏向温暖，日最高气温低于{{convert|0|C}}的平均日数为1.5天，日最低气温低于{{convert|0|C}}的平均日数为59天，低于{{convert|-5|C}}的平均日数为16天；夏季相对炎热潮湿，日最高气温超过{{convert|30|C}}的日数年均有83天，超过{{convert|35|C}}的有12天。<ref name = "NWS Greer, SC"/>最冷月（1月）均温{{convert|40.1|F|1|disp=flip}}，极端最低气温{{convert|−5|F|0|disp=flip}}（上次出现于1985年1月21日）。<ref name = "NWS Greer, SC"/>最热月（7月）均温{{convert|78.5|F|1|disp=flip}}，极端最高气温{{convert|104|F|0|disp=flip}}（上次出现于2012年6月29日至7月1日）。<ref name = "NWS Greer, SC"/>无霜期平均为219天（3月31日至11月4日）。<ref name = "NWS Greer, SC"/>年均降水量约{{convert|41.6|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|26.23|in|disp=flip}}（2001年），最多为{{convert|68.84|in|disp=flip}}（1884年）。<ref name = "NWS Greer, SC"/>年均降雪量为{{convert|4.3|in|cm|disp=flip}}，<ref name = "NWS Greer, SC"/>但许多冬季的积累降雪量微少以致难以测量，1959–60年的降雪量最多，积累降雪量为{{convert|22.6|in|cm|disp=flip}}。<ref name = "NWS Greer, SC"/>

== 經濟 ==
夏洛特是美国第二大银行业中心，是世界第一大銀行[[美國銀行|美國銀行]]及其[[美林證券|美林證券]]的總部。[[富國銀行集團|富國銀行集團]]也在夏洛特成立[[富國銀行集團|富國銀行集團]]的東部總部。

== 教育 ==
該城市的著名高校有美國著名的公立研究型大學[[北卡羅萊納大學夏洛特分校|北卡羅萊納大學夏洛特分校]]。比較著名的學院是Belk商學院。
