'''夏明翰故居'''位于衡阳县[[洪市镇|洪市镇]]礼梓村，始建于清乾隆年间，2005年修葺后正式对外开放<ref name=":0">{{Cite news|url=http://www.china.com.cn/chinese/TR-c/960781.htm|title=“红色之旅”到衡阳 夏明翰故居重修后对外开放|author=李柏芳|date=2005年9月5日|work=衡阳日报|accessdate=}}</ref>。夏明翰故居是首批衡阳市爱国主义教育示范基地、衡阳市党员教育基地、全国爱国主义教育示范基地，衡阳市首个党性教育基地<ref>{{Cite news|url=http://news.163.com/17/1025/09/D1J85RHE000187VG.html|title=衡阳首个党性教育基地落户夏明翰故居|author=易建军|date=2017年10月25日|work=红网|accessdate=}}</ref>。

== 建筑 ==
夏明翰故居为典型的清代湘南民居建筑，始建于清乾隆年间（约1745年前后），夏家大院二进六厢，坐北朝南，土砖木结构，共有房屋45间，面积2316平方米<ref name=":1">{{Cite web|url=http://www.hydsb.cn/html/dsyz/hsly/2015/1130/134.html|title=夏明翰故居|accessdate=2018年7月19日|author=|date=2015年11月30日|publisher=中国衡阳党史网}}</ref>。

== 陈列馆 ==
维修后的夏明翰故居分为四个部分：故居管理所，[[夏明翰|夏明翰]]生平事迹陈列展览部分，明翰精神书法陈列展览部分，夏明翰旧居复原陈列展览部分<ref name=":1" />。

== 馆藏 ==
陈列馆共收集了近400件文物，[[郭沫若|郭沫若]]、[[何叔衡|何叔衡]]等知名人士为夏明翰题写的挽联和诗词<ref name=":0" />。

== 交通 ==
衡阳市城区和衡阳县[[西渡镇|西渡镇]]均有班车到达，S2120线洪市镇礼梓村下车即到<ref name=":1" />。
