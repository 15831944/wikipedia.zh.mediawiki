'''夏嫣'''（{{bd||1月5日}}），[[中國|中國]]新生代[[女演員|女演員]]，[[歌手|歌手]]。畢業於[[北京電影學院|北京電影學院]]。第五屆[[澳門國際電影節|澳門國際電影節]]金蓮花獎最佳影視新人獎。<ref>{{cite web |title=夏嫣《反贪风暴3》杀青，揭秘其演艺身份背景_网易娱乐 |url=http://ent.163.com/17/1128/10/D4AQG9LR000333J5.html |website=ent.163.com}}</ref>

== 演藝生涯 ==

2012年夏嫣獲[[黃百鳴|黃百鳴]]賞識，在一次電影選角中，被選中擔任第一女主角，與[[林峯|林峯]]搭檔主演電影《[[詭嬰吉咪|詭嬰吉咪]]》飾演天晴；<ref>{{cite web |title=《哪咤降妖记》夏嫣杀青 甜美女神打戏精彩引期待{{!}}哪咤降妖记{{!}}夏嫣{{!}}杀青_新浪娱乐_新浪网 |url=http://ent.sina.com.cn/v/m/2017-11-13/doc-ifynstfh7337683.shtml |website=ent.sina.com.cn |date=2017-11-13}}</ref>。年底，參演港式喜劇群星賀歲片《[[百星酒店|百星酒店]]》。

2013年4月，夏嫣與香港演員[[吳卓羲|吳卓羲]]搭檔情侶主演都市情感勵志偶像劇《[[待嫁十年|待嫁十年]]》；10月，熱愛賽車的夏嫣參加在[[上海國際賽車場|上海國際賽車場]]舉行的「2013壹基金慈善明星賽」，奪得女子組賽車冠軍；12月，夏嫣參加第五屆[[澳門國際電影節|澳門國際電影節]]，獲得[[金蓮花獎|金蓮花獎]]最佳影視新人獎。

2014年夏嫣搭檔著名香港演員[[古天樂|古天樂]]主演電影《[[反貪風暴|反貪風暴]]》，飾演[[ICAC|ICAC]]調查員Tammy。<ref>{{cite web |title=夏嫣古天乐再携手《反贪风暴3》杀青获张智霖红包{{!}}反贪风暴3{{!}}夏嫣{{!}}古天乐_新浪娱乐_新浪网 |url=http://ent.sina.com.cn/m/c/2017-11-20/doc-ifynwxum6861389.shtml |website=ent.sina.com.cn |date=2017-11-20}}</ref>及武術題材電影《[[精武青春|精武青春]]》。同年，參演《[[屌絲男|屌絲男]]士》第四季第七集中與大鵬搭檔情侶首演網絡喜劇。

2015年6月，主演金雞百花獻禮片《[[我們遇見松花湖|我們遇見松花湖]]》，與[[林家棟|林家棟]]在電影中展開一段傾城之戀 ，該片獲得民族特別貢獻獎；10月，參演的電影《[[王家欣|王家欣]]》在香港上映，飾演王安欣。同年，參加了[[浙江衛視|浙江衛視]]的真人秀綜藝節目《[[我不是明星|我不是明星]]》在節目中夏嫣是黃百鳴義女的身份曝光；並与[[李治廷|李治廷]]搭檔情侶拍攝微電影《[[酒店情緣|酒店情緣]]》；

2016年電影《[[反貪風暴2|反貪風暴2]]》中飾演譚美莉； 並主演電影《[[綁靈|綁靈]]》，飾演酷酷可愛的阿麗； 与[[泰迪罗宾|泰迪罗宾]]导演首次合作，与[[郑嘉颖|郑嘉颖]]搭档情侣拍摄电影《[[八部半喜怒哀乐|八部半喜怒哀乐]]》；電影《[[反貪風暴3|反貪風暴3]]》開拍，飾演ICAC譚美莉；電視劇《[[哪吒降妖記|哪吒降妖記]]》第一次以古裝扮相示人，飾演英姿颯爽、武力高強的魚姬。<ref>{{cite web |url=[http://ent.qq.com/a/20171110/026772.htm}}</ref>
