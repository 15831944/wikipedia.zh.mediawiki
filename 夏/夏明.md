'''夏明'''，生于1965年，1981年考入[[复旦大学|复旦大学]]国际政治系<ref>{{Cite web |url = http://www.open.com.hk/content.php?id=2779#.WgFS0Du_Pmh|title = 王滬寧和夏明：復旦精英的不同抉擇  |accessdate =2016-04-18|website = 開放雜誌網}}</ref>，1984年加入[[中国共产党|中国共产党]]，现任美国[[纽约城市大学|纽约城市大学]]研究生中心和斯德顿岛学院政治学教授<ref>{{Cite web |url =http://www.rfa.org/mandarin/pinglun/chenpokong/jc-07212015143650.html|title =与王沪宁分道扬镳 －读夏明《红太阳帝国》（陈破空）|accessdate =2015-07-21|website =自由亚洲电台}}</ref>，海外知名的自由派学者<ref>{{Cite web |url =http://www.guancha.cn/songluzheng/2013_07_21_159807.shtml|title =与美国夏明教授再谈中国未来|accessdate =2013-07-21|website =观察者网}}</ref>。

==与王沪宁的十年交往==
夏明与[[中国共产党第十九届中央委员会|第十九届]][[中国共产党中央政治局常务委员会|中央政治局常务委员会]]委员[[王沪宁|王沪宁]]同为复旦大学校友，曾为师生、同事，从1981到1991年，夏明与王沪宁交往相处十年。王沪宁在1982至1983年教授夏明的“西方政治思想史”和“现代西方政治思想”课程。1989年学生民主运动之后，夏明与王沪宁分道扬镳<ref>{{Cite web |url =https://theinitium.com/article/20171102-opinion-xiaming-wanghuning/|title =夏明：我和王沪宁的十年交往|accessdate =2017-11-02|website =端传媒}}</ref>。
