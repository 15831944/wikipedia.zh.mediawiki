'''普鲁士的夏洛特公主，萨克森-迈宁根公爵夫人'''（'''Prinzessin Charlotte von Preußen, Herzogin von Sachsen-Meiningen'''，全名'''维多利亚·伊丽莎白·奥古斯特·夏洛特'''，'''Viktoria Elisabeth Auguste Charlotte'''，{{bd|1860年|7月24日|1919年|10月1日|catIdx=C}}），德皇[[腓特烈三世_(德国)|腓特烈三世]]和[[腓特烈皇后|维多利亚长公主]]的长女，英女皇[[维多利亚女皇|维多利亚]]的第一名外孙女。

==早年生活==
=== 童年与家庭 ===
[[File:Frederick_III,_German_Emperor_with_his_wife_and_their_children.JPG|left]],[[腓特烈皇后|维多利亚皇太子妃]], [[腓特烈三世_(德国)|腓特烈皇储]]抱着[[玛格丽特公主_(普鲁士)|玛格丽特公主]], 威廉王子(后成为[[威廉二世_(德国)|威廉二世]])和夏洛特公主。坐者从左至右： {{link-en|维多利亚公主（普鲁士）|Princess Viktoria of Prussia|维多利亚公主}}, [[索菲_(普魯士公主)|索菲公主]]和[[瓦尔德马王子_(普鲁士)|瓦尔德马王子]]。该照片摄于1875年。]]
夏洛特公主于1860年7月24日出生于[[波茨坦|波茨坦]]的[[新宮_(波茨坦)|新宫]]，是时为皇储的[[腓特烈三世_(德国)|腓特烈王子]]及[[腓特烈皇后|维多利亚长公主]]的长女，也是他们的第二名子女。由于她是英国[[维多利亚女皇|维多利亚女王]]的第一位外孙女，女王有意让其使用她的名字，但普鲁士人民则希望公主能承袭沙皇尼古拉一世的皇后[[亚历山德拉·费奥多罗芙娜皇后_(尼古拉一世)|亚历山德拉·费奥多罗芙娜]]之名(其婚前名为普鲁士的夏洛特公主)。后来双方各自妥协，公主起名为维多利亚·伊丽莎白·奥古斯特·夏洛特（德语:Viktoria Elisabeth Auguste Charlotte）,家人称呼她为夏洛特。

夏洛特年幼时，神经过敏，情绪激动，经常咬自己的指甲，撕扯自己的衣服。[[维多利亚女皇|维多利亚女皇]]在写给自己的女儿[[腓特烈皇后|腓特烈皇妃]]（亦即夏洛特的母亲）的信中，写道：“告诉夏洛特，我得知她咬自己的东西后，十分震惊。外婆不喜欢顽皮的女孩”。夏洛特曾因此受罚，但她在受罚后，依然如故。除此之外，她也不喜欢读书，皇妃因此感到十分伤心。不过，夏洛特的祖父母普鲁士国王[[威廉一世_(德国)|威廉一世]]（后为德国皇帝）、普鲁士王后[[奥古斯特_(萨克森-魏玛-艾森纳赫)|奥古斯特]]（后为德国皇后）十分宠爱她。她是与长兄[[威廉二世_(德国)|威廉二世]]一起成长的。

== 婚姻 ==
1876年，夏洛特与第二代表兄[[贝恩哈德三世，萨克森-迈宁根公爵|萨克森-迈宁根亲王]]订婚，最终在1878年2月18日在[[柏林|柏林]]结婚。两人育有一女，[[费奥多拉_(萨克森-迈宁根)|费奥多拉]]，生于1879年5月12日，是维多利亚女皇的第一个曾孙。费奥多拉出生后，夏洛特回到柏林，过社交生活。费奥多拉则由保姆、仆人照顾。夏洛特认同长兄及其宰相，[[奥托·冯·俾斯麦|奥托·冯·俾斯麦]]的保守主义政治观点，使她与母亲之间的关系，进一步恶化。

== 晚年 ==
1914年，萨克森-迈宁根亲王继承父亲公爵爵位，是为贝恩哈德三世，萨克森-迈宁根公爵。夏洛特亦因此获得公爵夫人头衔，但持有这个头衔的时间不长，因为贝恩哈德在[[第一次世界大战|第一次世界大战]]完结时被迫退位。贝恩哈德退位时，夏洛特因为长期吸食香烟，健康已经十分恶劣。她最终在1919年10月1日病逝，得年59岁。

== 后世对她的医学研究 ==
最近，有医学研究发现，夏洛特及其独女费奥多拉，可能患有[[紫質症|紫質症]]。这种病症，大概遗传自夏洛特的外高祖英皇[[乔治三世_(英国)|乔治三世]]。
