'''夏萍'''（英文名：{{lang|en|Teresa Apple Lo}}，{{bd|1937年|10月6日|||夏萍}}），原名'''盧少萍'''，[[香港|香港]]著名[[女演員|女演員]]，[[香港|香港]][[甘草演員|甘草演員]]，前[[無綫電視|無綫電視]][[部頭合約|部頭合約]]女藝員。

== 簡介 ==
1937年生於香港，[[二次大戰|二次大戰]]時曾逃至內地居住，至戰後回港。1955年她投考了電影公司成為演員，其後她一直都在[[粵語長片|粵語長片]]裡亮相。1970年她加入[[邵氏|邵氏電影]]拍片13年，至1982年加入[[無綫電視|無綫電視]]至今。夏萍無論在古裝或者時裝都能將角色演至淋漓盡致，雖未能成為[[無綫電視|無綫]]的金牌甘草演員，但她的演藝精神是眾藝員的學習榜樣。故2005年，夏萍在[[2005年度萬千星輝賀台慶|萬千星輝頒獎典禮2005]]中獲頒萬千光輝演藝大獎，嘉許她在演藝圈的貢獻。至今她已演出過百部電視劇；而電影方面，則有逾200部。現為監製[[張乾文|張乾文]]的御用演員。近期她的作品包括《[[舌劍上的公堂|舌劍上的公堂]]》、《[[貓屎媽媽|貓屎媽媽]]》及《[[新抱喜相逢|新抱喜相逢]]》。

== 名字由來 ==
夏萍的名字，據她本人說，是六十年代參演電影時，由電影導演[[吳回|吳回]]幫她取的。

== 家庭 ==
1964年，曾與演員[[江一帆|江一帆]]閃電訂婚，然而在1965年3月因意見相左，協議解除婚約<ref>1965年4月18日（星期日），[[香港|香港]]《[[華僑日報|華僑日報]]》，第六張，第四頁。</ref>。1970年，夏萍與樂師[[張雲_(樂師)|張雲]]結婚，但丈夫於1991年過世，兩人並無子女，她有一[[乾兒子|乾兒子]]，是演員[[林家棟|林家棟]]。由於一直獨居，無人照顧，於2015年10月24日於家中跌倒逾10小時無人知曉，直至同日晚上8時許無綫同事致電閒話家常，才揭發意外發生。其後夏萍已被安排入住老人院舍。
