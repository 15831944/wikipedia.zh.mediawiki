'''夏亚斯河'''（{{lang-gl|Río Xallas}}<ref>{{cite web |title=El consistorio adaptará el nombre de 22 calles a la toponimia gallega |url=https://www.lavozdegalicia.es/noticia/coruna/coruna/2016/11/29/consistorio-adaptara-nombre-22-calles-toponimia-gallega/0003_201611H29C1995.htm |publisher=La Voz de Galicia |accessdate=2019-01-21|date=2016-11-29 |language=es}}</ref>；{{lang-es|Río Jallas o Ézaro}}，哈利亚斯河（埃萨罗河）<ref>{{cite web |author1=Ministerio de Fomento |title=BOE-A-1925-7999 |url=https://www.boe.es/datos/pdfs/BOE//1925/220/A00919-00920.pdf |publisher=[[西班牙国家官方公报|Gazeta de Madrid]] |archiveurl=https://web.archive.org/web/20190123025635/https://www.boe.es/datos/pdfs/BOE//1925/220/A00919-00920.pdf |archivedate=2019-01-23 |page=919 |date=1925-08-08}}</ref>）是西班牙[[加利西亚_(西班牙)|加利西亚]]自治区[[拉科鲁尼亚省|拉科鲁尼亚省]]的一条河流。它发源自[[科里斯坦科|科里斯坦科]]卡斯特罗山，最终由[[科尔库维翁湾|科尔库维翁湾]]注入[[大西洋|大西洋]]</s>。夏亚斯河是欧洲唯一一条以[[瀑布|瀑布]]形式入海的河流<ref>{{cite web |title=EL único río de Europa que desemboca en cascada |url=https://emigracion.xunta.gal/es/conociendo-galicia/galicia-multmedia/galicia-50-prodigios/unico-rio-europa-desemboca-cascada |publisher=Xunta de Galicia |accessdate=2019-01-19 |archiveurl=https://web.archive.org/web/20160311105113/http://emigracion.xunta.gal/es/conociendo-galicia/galicia-multmedia/galicia-50-prodigios/unico-rio-europa-desemboca-cascada |archivedate=2016-03-11 |language=es}}</ref>。
==生态==
夏亚斯河常见的垂钓鱼是[[鱒魚|鱒魚]]<ref>{{cite web |title=Un pescador de Negreira captura dos truchas que suman más de tres kilos |url=https://www.lavozdegalicia.es/noticia/santiago/2018/03/20/pescador-negreira-captura-dos-truchas-suman-tres-kilos/0003_201803S20C6994.htm |publisher=[[La_Voz_de_Galicia|La Voz de Galicia]] |accessdate=2019-01-21 |language=es |date=2018-03-20}}</ref>。该河的生态遭受过外来物种[[克氏原螯虾|美国螯虾]]的[[生物入侵|入侵]]破坏，这些螯虾最早出现在[[瓜达尔基维尔河|瓜达尔基维尔河]]，2000年伊始扩散到了加利西亚的[[死亡海岸|死亡海岸]]<ref>{{cite web |title=Los que llegaron para quedarse |url=https://www.lavozdegalicia.es/noticia/carballo/a-laracha/2016/11/24/llegaron-quedarse/0003_201611C24C2991.htm |publisher=La Voz de Galicia|accessdate=2019-01-21 |language=es |date=2016-11-24}}</ref>。
