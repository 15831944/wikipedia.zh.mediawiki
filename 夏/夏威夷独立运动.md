'''夏威夷独立运动'''（{{lang-en|'''Hawaiian Independence Movement '''}}）是[[夏威夷|夏威夷]]部份[[原住民|原住民]]发起的争取夏威夷从[[美国|美国]]分离，重建独立的夏威夷国家的运动。

自1898年美国吞并[[夏威夷王国|夏威夷王国]]以来，部分土著居民一直认为这是一次非法的吞并，并致力于恢复独立。1946年，新成立的[[联合国|联合国]]也将夏威夷列入了联合国非自治领土范围，根据该条例，美国负有责任引导其逐渐独立。但是，1959年，美国政府在夏威夷举行[[全民公投|全民公投]]之后，宣布夏威夷作为美国第五十个州加入联邦。[[分离主义|分离主义]]者认为在这次公决中大量美国军事相关人员也参与投票，不能完全代表土著居民的意愿，拒绝承认公决的法律效力。近年来，夏威夷独立运动采用和平手段，在美国首都[[华盛顿哥伦比亚特区|华盛顿]]以及[[波士顿|波士顿]]、[[洛杉矶|洛杉矶]]等大城市举行示威游行，发表自己的意愿。

==历史沿革==
1000年前，[[波利尼西亚|波利尼西亚]]人来到夏威夷定居。<ref name="sheep " >{{cite web |url= http://news.sina.com.cn/w/2008-05-02/073513821086s.shtml|title=夏威夷“独立”组织，宣称要“行使”主权 |publisher=新浪网 |accessdate=2008-05-02 |work= |language =zh-cn}}</ref>

[[File:"Aniheneho._L'un_des_Premiers_Officiers_de_Tahmahamah",_pen_and_ink_wash_over_graphite_by_Jacques_Arago,_1819,_Honolulu_Academy_of_Arts.jpg|thumb]]

1810年，原住民大酋长[[卡美哈梅哈一世|卡美哈梅哈一世]]统一了夏威夷地区的所有岛屿，建立了大统一的“[[夏威夷王国|夏威夷王国]]”。<ref name=" lion" >{{cite web |url= http://news.sohu.com/20080527/n257115340.shtml|title=“夏威夷独立”挑战美国双重标准 |publisher= 搜狐网|accessdate= 2008年5月27日|work= |language =zh-cn}}</ref><ref name="wolf " >{{cite web |url=http://news.sina.com.cn/o/2008-06-30/155414094691s.shtml |title= 夏威夷土著女王率原住民闹独立|publisher=网易 |accessdate=2008年6月30日 |work= |language =zh-cn}}</ref>

1842年12月，[[美国总统|美国总统]][[约翰·泰勒|约翰·泰勒]]在给[[美国国会|国会]]的文章中提出保留夏威夷的独立。<ref name=" lion" />

1843年11月，[[英国|英国]]和[[法国|法国]]签署了一项协定，共同保证夏威夷领土、主权完整和独立。<ref name=" lion" /><ref name="wolf " />

[[File:kingdavidkalakaua_dust.jpg|thumb]]

1875年，美国政府签署了《[[美夏互惠条约|美夏互惠条约]]》，美国武力占据了[[珍珠港|珍珠港]]。<ref name=" lion" /><ref name="wolf " />

1893年，[[美国海军陆战队|美国海军陆战队]]武装支援夏威夷的美国侨民发动政变。<ref name=" lion" />

1894年，美国人推翻[[利留卡拉尼女王|利留卡拉尼女王]]及其政府，随后软禁女王在伊奥拉尼皇宫，并且在1894年宣布成立“[[夏威夷共和国|夏威夷共和国]]”。<ref name=" lion" /><ref name="cat " >{{cite web |url= http://war.163.com/08/0508/14/4BE6NSCS00011MTO.html|title= 夏威夷原住民号召独立 要无限期占领原王国宫殿|publisher= 网易|accessdate=2008-05-08  |work= |language =zh-cn}}</ref><ref name="wolf " /><ref name="sheep " />

[[File:Lowering_the_Hawaiian_flag_at_Annexation_ceremony_(PPWD-8-3-006).jpg|thumb]]正式吞并[[夏威夷|夏威夷]]，在王宫门前降下夏威夷国旗。]]

1898年，[[美西战争|美西战争]]后，[[美国国会|美国国会]]的[[众议院|众议院]]和[[参议院|参议院]]联合通过了“兼并夏威夷”的决议。8月12日，夏威夷正式成为美国的[[海外领地|海外领地]]，美国人“夏威夷共和国总统”多尔被任命为第一任领地[[总督|总督]]。<ref name=" lion" /><ref name="wolf " />

1959年3月，夏威夷作为第50个州加入美国，成为美国国旗上的第50颗星。<ref name=" lion" /><ref name="wolf " /><ref name="sheep " />

[[20世纪70年代|20世纪70年代]]开始，夏威夷原住民中组成主权申索团体，拒绝承认1893年推翻夏威夷女王的政变和1898年的兼并，并且要求美国政府对这两个事件和1898年后长期的军事占领进行道歉和赔偿。<ref name=" lion" />

1978年，[[美国宪法|美国宪法]]规定在夏威夷州将[[夏威夷语|夏威夷语]]与[[英语|英语]]同为夏威夷州的官方语言。<ref name=" lion" />

[[File:USS_Boston_landing_force,_1893_(PP-36-3-002).jpg|thumb]]

1993年，美国国会众议院三分之二的票和参议院65票同意对34票反对，通过了“道歉法案”。11月15日，美国总统[[比尔·克林顿|比尔·克林顿]]签署同意该法案，并且宣布代表美国人民，给1893年美国政府推翻夏威夷女王的政变正式道歉。<ref name=" lion" /><ref name="wolf " />

[[夏威夷大学|夏威夷大学]]政治科学系学生[[大卫·基阿努·萨伊|大卫·基阿努·萨伊]]，自谓“夏威夷王国代理摄政委员会主席”、“夏威夷王国代理内政部长”。<ref name=" lion" /> 2000年萨伊用“夏威夷王国”的称号，在[[海牙国际仲裁法庭|海牙国际仲裁法庭]]上，请求夏威夷州独立，但是没有获得同意。<ref name=" lion" />

2008年4月30日，夏威夷王国政府组织占据了[[夏威夷州|夏威夷州]][[檀香山市|檀香山市]]市中心的夏威夷王国末代女王的住所博物馆，要求[[美国联邦政府|美国联邦政府]]赔偿原住民的损失、撤掉驻军和要求主权独立。<ref name=" lion" /><ref name="wolf " />
