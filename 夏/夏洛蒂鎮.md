'''夏洛特鎮'''或'''夏洛茨维尔'''（Charlottesville）是[[美国|美国]][[弗吉尼亚州|弗吉尼亚州]]的一座城市，[[詹姆斯河|詹姆斯河]]的支流rivanna河流经市内，[[藍嶺山|藍嶺山]]位于城市以西約20英里。

该市成立於1762年，以前的路被稱為「三道缺口」（今[[美國國道250|美國國道250]]）。其他通過夏律第的主要公路有[[美國國道29|美國國道29]]，從[[華盛頓|華盛頓]]到[[丹維爾_(維吉尼亞州)|丹維爾]]，和[[64號州際公路|64號州際公路]]、[[美國國道250|美國國道250]]。

[[托马斯·杰斐逊|托马斯·杰斐逊]]创建的美国第一所公立大学[[弗吉尼亚大学|弗吉尼亚大学]]位于该市。杰斐逊的故居位于夏洛特鎮的[[蒙蒂塞洛|蒙蒂塞洛]]，每年吸引数十万的游客到访。该市附近还有不少美國[[南北戰爭|南北戰爭]]时期的战场。由于这些原因， 這個城市是美國的文化和歷史縮影。

== 歷史 ==
於1762年，貿易商來到了這地，他們沿著三道缺口（現今的[[美國國道250|美國國道250]]）從[[里奇蒙|里奇蒙]]到[[丹維爾|丹維爾]]。它的命名是由[[喬治三世|喬治三世]]的妻子[[夏綠蒂王后|夏綠蒂王后]]來命名的。在這期間[[美國獨立戰爭|美國獨立戰爭]]，1779年到1781年之間，軍隊被監禁在夏律第鎮的[[阿爾伯馬爾營房|阿爾伯馬爾營房]]在。

自十九世紀末，此城的人口始見增長，由原來二千多人，增加至1910年的六千多人；而在1910-20年間，人口首次破萬，增長達58%。

== 體育 ==
夏律第鎮沒有職業體育隊，而是大學體育隊，[[維吉尼亞騎士|維吉尼亞騎士]]。維吉尼亞騎士體育領域從足球到籃球。維吉尼亞騎士的[[橄欖球|橄欖球]]季在學年期間吸引大量的觀眾，橄欖球賽在[[史考特體育場|史考特體育場]]。

== 教育 ==
[[File:2008-0830-Charlottesville-DowntownMall.jpg|left]]
夏律第鎮由夏律第鎮公立學校服務。學校系統有六所[[小學|小學]]、Buford[[中學|中學]]，和夏律第鎮高中。區域私立學校包括夏律第鎮天主教學校、夏律第鎮Waldorf 學校、契約學校、夏律第鎮領域學校,阿爾伯馬爾米勒學校,Montessori 社區學校，北部分支學校，皮博迪學校，新生學校，聖安妮-Belfield 學校，縱排朋友學校，村莊學校，維吉尼亞學院。阿爾伯馬爾縣公立學校系統在夏律第鎮市區範圍外面。

== 交通 ==
夏律第鎮由[[夏律第阿爾伯馬爾機場|夏律第阿爾伯馬爾機場]]和[[夏律第車站|夏律第車站]]駐地，和[[灰狗巴士|灰狗巴士]]城市間公共汽車總站服務。城市間的公共汽車服務是由星光快運提供。夏律第鎮由JAUNT提供區域公共汽車服務。通過夏律第鎮的高速公路為[[64號州際公路|64號州際公路]], 它是東西向路線，另有[[美國國道250|美國國道250]]，和南北向的[[美國國道29|美國國道29]]。
