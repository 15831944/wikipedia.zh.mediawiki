'''夏茸尕布·洛桑图丹格勒坚赞'''（{{bo|z=ལ་མོ་ཞབས་དང་དཀར་པོ་ས་ཕེང་བཞླ་བ་བོ་བཟང་ཐབ་བསན་དགེ་ལེགས་རལ་མཚན}}，{{bd|1729年||1796年|}}）今[[青海省|青海省]][[果洛藏族自治州|果洛藏族自治州]][[阿尼玛卿山|阿尼玛卿山]]附近人，第四世（不计追认）[[夏茸尕布活佛|夏茸尕布活佛]]<ref name=kao>考洛巴·陈列奥赛，夏茸尕布传，西藏研究1994年第4期</ref><ref name=xzzx>{{Cite web |url = http://www.tibetol.cn/html/2013/zdyg_0405/393.html |title =夏茸尕布活佛  |author =  |date = 2017-05-23 |publisher =  西藏在线|language =  |accessdate =  |quote =  }}</ref><ref name=cairang>才让扎西，拉茂德乾寺历史与现状研究，青海民族大学法学硕士学位论文，2016年</ref>。

==生平==
[[藏历|藏历]]饶迥土鸡年（[[清朝|清朝]][[雍正|雍正]]七年，1729年）生于今青海省果洛藏族自治州[[阿尼玛卿山|阿尼玛卿山]]附近的阿力克桑杰家族，父名拉尕次成尖措，母名扎西吉。由[[七世达赖|七世达赖]]、赤钦丹贝尼玛共同确认为第三世夏茸尕布活佛的[[转世灵童|转世灵童]]。他曾先后拜[[阿柔格西|阿柔格西]]钦毛（又译“阿力克格西钦茂”）、七世达赖、夏玛尔昂旺成列、赤钦嘉纳巴等人为师<ref name=xzzx/><ref name=kao/><ref name=xzzx/>。18岁时，在[[茫拉乡|茫拉]]修行时，[[大梵天|大梵天]]、[[阿尼玛卿山|玛沁山神]]、赤雪措曼女神都现身显灵。他还目睹了吉祥怙主。他在玛古南杰林寺建了一尊用各种宝物制成的[[释迦牟尼|释迦牟尼]]像，并且为此建了四层高的金铜顶佛殿<ref name=kao/>。他又据[[拉卜楞寺|拉卜楞寺]]的规章建成了[[时轮金刚|时轮金刚]]修供仪轨以及[[胜乐金刚|胜乐金刚]]、[[密集金刚|密集金刚]]、[[大威德金刚|大威德金刚]]三尊正规仪轨，后来发展为[[拉莫德钦寺|拉茂德乾寺]]的密宗学院<ref name=xzzx/><ref name=kao/>。

他一生曾两次进[[西藏|西藏]]，第一次进藏时，[[清朝|清朝]]朝廷授予他[[扎萨克|扎萨克]]爵，由此成为政教首领。[[嘉木样·久美旺波|第二世嘉木样协巴]]以及[[贡唐仓活佛|贡唐仓加样活佛]]叩拜过他。他曾任[[八世达赖|八世达赖]]的托钵师<ref name=xzzx/><ref name=xzzx/>。

据说，他用业轮统治大地，在[[茫拉乡|茫拉]]、沙沟河上下游以及秀龙等地陆续建立“乌尔格”（府邸），还在曲培扎萨克旗下内争时进行了调解，曲培在逝世前把[[扎萨克|扎萨克]]印信、旗民部落以及[[同德县|同德县]]的[[巴曲|巴曲]]上下游等地赠给他。七世达赖封他为青海左右翼蒙古所有部落的“至尊上师”<ref name=kao/>。

藏历十三饶迥火龙年（清朝[[嘉庆|嘉庆]]元年，1796年）圆寂，享年68岁。灵塔建在拉茂德乾寺大殿的内殿里，[[文革|文革]]中被毁<ref name=xzzx/><ref name=kao/><ref name=xzzx/>。
