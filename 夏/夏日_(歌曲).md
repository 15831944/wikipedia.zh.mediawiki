《'''夏日'''》（{{lang-en|''Summer''}}）是苏格兰唱片艺人[[凱文·哈里斯|凱文·哈里斯]]的一首歌曲，收录于他的第4张录音室专辑《[[音浪狂潮|音浪狂潮]]》中，歌曲以专辑的第2支单曲发行<ref>{{cite web |last=Corner |first=Lewis |url=http://www.digitalspy.co.uk/music/news/a557592/calvin-harris-premieres-new-single-summer-listen.html |title=Calvin Harris premieres new single 'Summer' – listen |publisher=[[Digital_Spy|Digital Spy]] |date=2014-03-14 |accessdate=2014-03-22}}</ref>。《夏日》与上一支单曲《[[Feel_So_Close|Feel So Close]]》一样，均是由哈里斯担任主唱。随行的音乐录影带由埃米爾·納瓦执导并于2014年4月6日首映。

歌曲于首周登上[[英國單曲排行榜|英國單曲排行榜]]榜首，成为哈里斯第6首英国冠军单曲<ref>{{cite web |last=Lane |first=Daniel |url=http://www.officialcharts.com/chart-news/calvin-harris-scores-his-sixth-uk-number-1-single-with-summer-2980/ |title=Calvin Harris scores his sixth UK Number 1 single with Summer |publisher=[[Official_Charts_Company|Official Charts Company]] |date=2014-05-04 |accessdate=2014-05-07}}</ref>。歌曲在[[告示牌百强单曲榜|《告示牌》百强单曲榜]]最高到達第7位，成为他第2首作为主艺人和总共第3首在榜单前十位的歌曲。截至2014年7月，歌曲已在美国售出超過百萬張<ref name="USsales">{{cite web |last=Grein |first=Paul |url=https://www.yahoo.com/music/bp/chart-watch-diss-or-not-iggy-azalea-is-still-1-224749026.html |title=Chart Watch: Diss Or Not, Iggy Azalea Is Still #1 |publisher=[[Yahoo!_Music|Yahoo! Music]] |date=2014-07-02 |accessdate=2014-12-26}}</ref>。《夏日》在[[2015年全英音樂獎|2015年全英音樂獎]]获得「英国单曲」和「年度英国艺人录影带」的提名<ref>{{cite web |last=Hann |first=Michael |url=https://www.theguardian.com/music/2015/jan/15/sam-smith-george-ezra-and-ed-sheeran-dominate-brits-nominations |title=Sam Smith, George Ezra and Ed Sheeran dominate Brits nominations |work=[[The_Guardian|The Guardian]] |date=2015-01-15 |accessdate=2015-01-18}}</ref>。

==音乐录影带==
《夏日》的音乐录影带由埃米爾·納瓦执导并于2014年4月6日首映<ref>{{cite web |last=Gottlieb |first=Steven |url=http://www.videostatic.com/watch-it/2014/04/07/calvin-harris-summer-emil-nava-dir |title=Calvin Harris "Summer" (Emil Nava, dir.) |publisher=VideoStatic |date=2014-04-07 |accessdate=2014-08-20}}</ref><ref>{{cite web |url=http://imvdb.com/video/calvin-harris/summer |title=Calvin Harris – Summer (2014) |publisher=IMVDb |accessdate=2014-04-12}}</ref>，英国演员[[傑森·史塔森|傑森·史塔森]]客串其中<ref>{{cite web |url=http://www.dailymail.co.uk/tvshowbiz/article-2598761/Jason-Statham-makes-cameo-appearance-Calvin-Harris-brand-new-video-Summer.html |title=It's fast and furious!
