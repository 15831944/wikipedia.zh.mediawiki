'''田氏角石'''（[[學名|學名]]：{{lang|la|'''''Tienoceras'''''}}）是一[[屬|屬]]已[[滅絕|滅絕]]的[[鸚鵡螺亞綱|鸚鵡螺類]]。生存在[[二疊紀|二疊紀]]的[[海洋|海洋]]中。<ref name="Fossilworks">{{Cite web |url = http://fossilworks.org/bridge.pl?a=taxonInfo&taxon_no=12616 |title = Fossilworks: Tienoceras |language = en |accessdate = 2018-05-15 }}</ref>牠們的[[化石|化石]]分布於[[中國|中國]][[湖南|湖南]]。其名稱是來自於對湖南地層及[[古生物|古生物]]供獻很多的[[田奇瑞|田奇瑞]]教授。<ref name="Hunan">{{Cite web |url = http://159.226.74.248:8000/pagelinks/7098.pdf |title = 湖南譚家山二疊紀的頭足類化石 - 中國科學院南京地質古生物研究所 |language = zh-hans |accessdate = 2018-05-15 }}</ref>

本屬在[[直角石目|直角石目]]中的地位尚未確定。
