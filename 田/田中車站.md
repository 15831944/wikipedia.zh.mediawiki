'''田中車站'''位於[[台灣|台灣]][[彰化縣|彰化縣]][[田中鎮|田中鎮]]，位於[[社頭車站|社頭]]至[[二水車站|二水]]間，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]的[[鐵路車站|鐵路車站]]。因[[彰化車站_(高鐵)|高鐵彰化站]]於2015年12月1日啟用，[[彰化縣政府|彰化縣政府]]與臺鐵研擬將永靖到田中改線，田中站移至與高鐵彰化站共構。<ref>[http://publicworks.chcg.gov.tw/07other/other01_con.asp?topsn=648&data_id=16163 「三横 三縱 三鐵 三橋」之交通建設發展構想]</ref>依據2017年3月[[行政院|行政院]]所公布的[[前瞻基礎建設計畫|前瞻基礎建設計畫]]，未來將會在此站興建一條支線連接[[高鐵彰化站|高鐵彰化站]]，並會把此站用作該條支線的起點，並做為[[集集線|集集線]]起訖站。<ref>[http://infrastructure.ey.gov.tw/cp.aspx?n=C8A462AE07F6F3C5 前瞻基礎建設計畫專區]</ref> <ref>[http://event.motc.gov.tw/home.jsp?id=1658&websiteid=201705040001 前瞻基礎建設軌道建設報告書下載區]</ref>

== 車站構造 ==
*[[島式月台|島式月台]]、[[岸式月台|岸式月台]]各一座。
*[[跨站式站房_(台鐵)|跨站式車站]]，購票及服務設施設在二樓大廳，與前後站均有連通。
*設有[[手扶梯|手扶梯]]及[[電梯|電梯]]。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#二等站|二等站]]，主要停靠[[臺鐵區間車|區間車]]、區間快車、莒光號為主，多數[[自強號列車|自強號]]也有停靠。(對號列車停靠率：67.5%；所有車種停靠率85.9%)
**[[自強號列車|自強號]]：停靠順行102、170、108、114、118、6126、134、144、150次、274次，逆行101、105、113、121、123、135、141、177、181次。
**[[普悠瑪列車|普悠瑪號]]：停靠順行280次，逆行283次，亦為西部幹線唯一停靠普悠瑪號之二等站。
**僅有1班海線自強號北上(順行)6160次為起點。
**東部幹線的自強號6275次南下(逆行)為終點。
*目前共有5個班次的[[集集線|集集線]]列車以田中站為到發站。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1905年3月26日：設「田中央驛」。
*1920年10月1日：改為「田中」。
*2003年1月23日：擴建跨站式站房落成。
*2015年6月30日：啟用多卡通刷卡機。
*2015年10月15日：改點後[[普悠瑪列車|普悠瑪號]]取代原來[[太魯閣列車|太魯閣號]],也開始停靠[[普悠瑪列車|普悠瑪號]]以始用280車次、283車次每日來回。
*2017年4月27日起：配合高鐵與台鐵合作「搭雙鐵，遊集集」旅遊方案，部分[[集集線|集集線]]車次改為田中站起訖。<ref>[https://www.thsrc.com.tw/tw/News/Detail/0c5c4c9d-7931-4fe0-b2ef-e4340a730c6c/1 搭雙鐵，遊集集]</ref>
*2018年3月26日：利用田中站開業日與日本[[信濃鐵道線|信濃鐵道線]]{{link-ja|田中站|田中駅|田中站}}締結為姐妹站。<ref>[https://www.railway.gov.tw/tw/news_Detail.aspx?SN=14691&NewsType=0&n=6807 臺鐵局與日本信濃鐵道友好協定暨田中站姊妹車站締結典禮]</ref>
