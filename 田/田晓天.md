田晓天（{{bd|1988年|2月27日}}）[[中国大陆|中国大陆]]知名[[演员|演员]]。

== 简介 ==
2009年，田晓天在[[上海|上海]]拍摄[[偶像剧|偶像剧]]时，被剧组冠之以“捣蛋鬼”可爱称号。<ref>{{cite web |url=http://music.yule.sohu.com/20090707/n265050366.shtml|title=“捣蛋鬼”田晓天 担任剧组“开心果” |accessdate=2009年7月7日|work=|publisher=搜狐网|language= zh-hans}}</ref>

2010年，田晓天被邀拍摄时尚大片，拍摄地点在[[北京市|北京市]]世贸天阶，演绎一个健康、阳光的[[九零后|九零后]]女孩。<ref>{{cite web |url=http://ent.sina.com.cn/s/m/2010-07-26/11323029319.shtml|title=田晓天演绎夏日娇娃 杂志侧拍显露鬼马个性 |accessdate=2010年7月26日|work=|publisher=新浪网|language= zh-hans}}</ref>
