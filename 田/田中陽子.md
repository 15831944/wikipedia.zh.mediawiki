'''田中陽子'''（{{bd|1993年|7月30日|catIdx=Tanaka Yoko}}）是一位[[日本|日本]]女子[[足球|足球]][[運動員|運動員]]，效力於[[INAC神戶雌獅|INAC神戶雌獅]]，位置[[中場|中場]]。[[日本|日本]][[山口縣|山口縣]][[山口市|山口市]]出身，[[日本足球協會|JFA]]福島學院畢業。暱稱ようこ、たなよう<ref>[http://inac-kobe.com/?player=%e7%94%b0%e4%b8%ad%e9%99%bd%e5%ad%90 INAC神戸レオネッサ公式サイト　田中陽子]</ref>。

== 經歷 ==
她是[[日本足球協會|日本足球協會]]所設立的足球菁英[[完全中學|完全中學]]JFA福島學院培養的一期生<ref>[http://www.zakzak.co.jp/sports/soccer/news/20120823/soc1208231121000-n1.htm “ヤングなでしこ”に新ヒロイン登場！田中陽子が大奮闘] zakzak 2012年8月23日</ref>。在[[紐西蘭|紐西蘭]]舉辦的[[2008年U17女子世界盃|2008年U17女子世界盃]]，與[[岩淵真奈|岩淵真奈]]同以15歲的年齡越級入選參加。2010年同樣代表日本參加[[2010年U17女子世界盃|2010年U17女子世界盃]]，全部6場比賽皆有出場，貢獻4個進球，最終日本獲得亞軍。

2012年從學院畢業後，加入[[INAC神戶雌獅|INAC神戶雌獅]]。3月15日，在[[日韓女子聯賽冠軍錦標賽|日韓女子聯賽冠軍錦標賽]]面對[[高陽大教Noonnoppi|高陽大教Noonnoppi]]首次在正式比賽上場。6月23日，聯賽盃對上[[福岡J・錨|福岡J・錨]]攻入首個正式比賽的進球。8月開始在[[日本|日本]]進行的[[2012年U20女子世界盃|2012年U20女子世界盃]]面對[[瑞士國家女子足球隊|瑞士]]的比賽中，她上下半場各以右腳及左腳主踢[[任意球|任意球]]破網，引起關注<ref>「陽子2発！右で左でFK弾」（「[[產經體育|產經體育]]」2012年8月27日）</ref>。

== 踢球風格 ==
身材嬌小，但盤球能力優異，是她最大的武器<ref>[http://sportiva.shueisha.co.jp/clm/jfootball/2012/06/24/18_1/index2.php 【なでしこ】ポスト澤候補の18歳、田中陽子。「U-20Ｗ杯で優勝したい」] sportiva 2012年06月24日</ref>。能夠以左右兩腳踢定位球，是可以使球隊踢法變化多端的司令官<ref>[http://sportsnavi.yahoo.co.jp/soccer/japan/2012/text/201208270002-spnavi_2.html ヤングなでしこ、田中陽子という驚き （２／２） Ｕ－２０スイス女子代表　０－４　Ｕ－２０日本女子代表] {{webarchive|url=https://web.archive.org/web/20120830002933/http://sportsnavi.yahoo.co.jp/soccer/japan/2012/text/201208270002-spnavi_2.html |date=2012-08-30 }} スポーツナビ 2012年8月27日</ref>，不論攻擊或防守[[中場|中場]]皆可以勝任<ref>[http://hochi.yomiuri.co.jp/soccer/japan/news/20120824-OHT1T00226.htm 猶本＆陽子、美少女Ｗボランチで決勝Ｔ進出決める！…Ｕ―２０女子Ｗ杯] {{webarchive|url=https://web.archive.org/web/20120830165013/http://hochi.yomiuri.co.jp/soccer/japan/news/20120824-OHT1T00226.htm |date=2012-08-30 }} スポーツ報知 2012年8月25日</ref>。
