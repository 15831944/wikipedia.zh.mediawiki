'''田子昌'''（{{bd|？||1949年|12月2日}}），佤名'''艾门'''，[[沧源佤族自治县|沧源]][[岩帅镇|岩帅]]人，佤族部落政治及军事人物，[[岩帅|岩帅]]头人[[田桑保|田桑保]]之子、[[田兴武|田兴武]]之侄。

[[班洪事件|班洪事件]]中曾率岩帅武装支援班洪抗英。1943年，被[[龙云|龙云]]封为“班洪抗日自卫队副指挥官”，参与[[沧源抗日斗争|抗日斗争]]。1949年4月，时任岩帅头人[[田兴武|田兴武]]、[[田兴文|田兴文]]接受[[中国共产党|中国共产党]]的领导，田子昌被任命为“迤南边区人民自卫军第一支队佧佤山守备大队”大队长，率[[田兴文|田兴文]]的第十大队和守备大队武装于5月6日攻下[[勐董镇|勐董]]的国民党[[沧源设治局|沧源设治局]]，后率部参与[[沧源保卫战|沧源保卫战]]。沧源解放后，第十大队和佧佤山守备大队合编为“中国人民解放军滇、桂、黔边区纵队第九支队澜、宁、源整训总队联合大队”。后经[[双江拉祜族佤族布朗族傣族自治县|双江]][[彭肇模|彭肇模]]、[[彭肇栋|彭肇栋]]游说，企图发动叛乱占据沧源，并协助彭肇模部南渡[[小黑江_(澜沧江右支流)|小黑江]]。情报被[[田兴武|田兴武]]、[[田兴文|田兴文]]和[[李培伦|李培伦]]获悉，于12月2日晚与其父一同被击毙。<ref>{{cite web |url=http://dangshi.people.com.cn/n/2013/1019/c85037-23259888.html |title=临沧抗战纪事 |author=谌桂绚 |date=2013-10-19 |website=dangshi.people.com.cn |publisher=[[云南日报|云南日报]] |access-date=2018-05-06 }}</ref><ref>{{cite web |url=http://ein.ynnu.edu.cn/wasearch/articleview.aspx?id=1305 |title=田子昌 |author1=罕贵娇 |author2=常数 |author3=杨宝康 |date=2013-01-01 |website=ein.ynnu.edu.cn |publisher=佤族大辞典 |access-date=2018-05-06 }}</ref><ref>{{cite web |url=http://www.lincang.gov.cn/Zzlc/Lcnj/200603/16203.html |title=临沧大事记（民国部分） |author=临沧市信息产业办公室 |website=www.lincang.gov.cn |publisher=临沧市人民政府 |access-date=2018-05-06 |quote=}}</ref>
