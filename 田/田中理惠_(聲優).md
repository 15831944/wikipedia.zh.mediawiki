'''田中理惠'''（{{lang-ja|田中 理恵}}、{{bd|1979年|1月3日|catIdx=Tanaka, Rie}}）是一位日本[[女性|女性]][[聲優|聲優]]。2019年3月起成為[[Office_Anemone|Office Anemone]]所屬<ref>{{cite web|url=https://ameblo.jp/rie-tanaka/entry-12443510538.html|script-title=ja:【ご報告です！】田中理恵オフィシャルブログ「RIE RIE WORLD」|work=田中理惠的官方個人部落格|date=2019-03-01|accessdate=2019-03-19|language=ja}}</ref>。官方網站上寫聲優、歌手兩棲。[[日本|日本]][[北海道|北海道]][[札幌市|札幌市]][[南區_(札幌市)|南區]]出身<ref>據其本人所說，在[[定山溪溫泉|定山溪溫泉]]附近。</ref>。

== 經歷 ==
*本來喜歡讀書跟畫畫，特別中意[[豬股睦美|豬股睦美]]的畫<ref>高中時期曾立志要以漫畫家的身份在漫畫甲子園中出場。</ref>。但在聽了從書店中看到豬股所畫的《CD THEATER [[勇者鬥惡龍|勇者鬥惡龍]]》而買下來的[[廣播劇|廣播劇]]CD之後，而立志要當一個[[聲優|聲優]]。
*高中畢業後進入[[代代木動畫學院|代代木動畫學院]]的聲優科，在學中獲了由[[MediaWorks|MediaWorks]]與SCITRON合辦的歌手試演會中的冠軍，而在《[[悠久幻想曲|悠久幻想曲]]》第二張專輯中唱了「永遠的好朋友」。也因為如此，歌手部份比起聲優還要早出道。
*出道動畫作是《[[平行世界物語|平行世界物語]]》。馬上就擔任了女主角真田三月的配音。這部動畫是雙女主角制的，與擔任另一女主角羅螺觀月配音的[[豐口惠|豐口惠]]共演的機會很多。
*擅長於《[[機動戰士GUNDAM_SEED|機動戰士GUNDAM SEED]]》、《[[機動戰士GUNDAM_SEED_DESTINY|機動戰士GUNDAM SEED DESTINY]]》中的[[拉克絲·克萊因|拉克絲·克萊茵]]、《[[Chobits|Chobits]]》的唧或《[[光之美少女|光之美少女 Max Heart]]》的九條光/夏妮露米納斯等等的帶有神秘感的角色。或是《[[花右京女僕隊|花右京女僕隊]]》的瑪莉耶爾與《[[旋風管家|旋風管家]]》的瑪麗亞這樣年輕母性型角色。另一方面也有像《[[校園漫畫大王|校園漫畫大王]]》的水原曆、《[[爆彈小新娘|爆彈小新娘]]》的八千草響子與《[[工作狂人|工作狂人]]》的松方弘子等現代型的角色，或是熱血系的角色如《[[熱拳本色|熱拳本色]]》的高嶺菊，敵役《[[薔薇少女|薔薇少女]]》的水銀燈與《[[舞-乙HiME|舞-乙HiME]]》巴．瑪格麗特等等的，擔任了各式各樣不同個性的角色。
*常常[[Cosplay|扮演]]自己所配的角色（如《[[薔薇少女|薔薇少女]]》的水銀燈）來參加活動，以及在節目中唱歌。也有著因為[[Cosplay|扮演]]了《[[鋼鐵天使|鋼鐵天使]]》中的莎奇而成為雜誌附錄的海報之類的事情。
*所屬的會社，當初是SME Visual WORKS（現在的[[Aniplex|Aniplex Inc.]]），2003年轉屬到Victor Entertainment Inc.。在當年秋天所釋出的「Chara de Rie」之後，從同社所出的就只停留在《[[機動戰士鋼彈SEED|機動戰士鋼彈SEED]]》系列作相關的樂曲的程度。近年以來，少有除了以角色名義的歌曲或是特別活動之外的歌手活動的機會。
*近年在《[[熱拳本色|熱拳本色]]》之後，出演[[東映動畫|東映動畫]]與Marvelous Entertainment Inc.所製作的動畫的機會很多。
*在電視動畫《[[玻璃艦隊|玻璃艦隊]]》、《[[交響情人夢|交響情人夢]]》、《[[sola|sola]]》、以及田中本人亦有出演的《[[TIGER×DRAGON！|TIGER×DRAGON！]]》、《[[CANAAN|CANAAN]]》裡，工作人員表記載[[田中理惠_(音响技师)|同名同姓的音響監督]]，不過是不同人。
*曾經在[[日本新力音樂|日本新力音樂]]所營運的寬頻網站《MORRICH》中，擔任[[Aniplex|Aniplex Inc.]]介紹節目的領航員。
* 2012年6月17日，與同是聲優的[[山寺宏一|山寺宏一]]結婚<ref>[http://ameblo.jp/rie-tanaka/entry-11280386177.html ご報告｜田中理恵オフィシャルブログ「RieRie world」Powered by Ameba]{{ja}}</ref>。其後在2018年8月宣佈離婚<ref>{{Cite web |url = https://www.oricon.co.jp/news/2116913/full/ |title = 山寺宏一、田中理恵との離婚報告「共に過ごした日々はかけがえのないもの」 |author = Oricon News |date = 2018-08-03 |language = ja |accessdate = 2018-08-03}}</ref>。

*在官方網站裡獨自一人使用「'''{{nihongo|wafu wafu～。|わふわふ～。}}'''」的方式來打招呼。
*喜歡的話是「'''不要畏懼不要逃避不要放棄'''」。
*說到要成為聲優所必備的物品時，舉出了蜂蜜味喉嚨噴霧器、鼻噴霧器、六色原子筆。
*在房間裡練習演技時，因為發出了太逼真的悲鳴聲，而使得隔壁的鄰居去向警察通報。
*本人自稱性格急躁。在搭乘電車時遭遇[[癡漢|癡漢]]時，抓住了癡漢的手大叫「'''要下車了喔！'''」然後拉著一起下車走到剪票口，在癡漢要逃的時候也大叫過「'''不要逃！'''」。<ref>聽到了這故事的[[甲斐田雪|甲斐田雪]]跟[[小西克幸|小西克幸]]發出了「好厲害啊！」或「妳真是太帥了！」之類的感嘆。她自己本人似乎也希望自己的個性能多像女孩子一點。（在《[[少年陰陽師|少年陰陽師]]・聽著那邊發出的聲音～簡略孫子廣播》節目中擔任特別來賓時講的。）在[[東京電玩展|東京電玩展]]的2006年活動中也公開講過她曾經被自己的經紀人講說「妳真是個男人。」。</ref>

=== 超次元游戏 海王星 袭击事件 ===
2013年6月23日举行的《超次元游戏 海王星》视动画第1-2话先行上映会，会场于中午1点40分左右遭到一名自稱來自東京都足立區的男子手持長約十幾公分的利刃袭击。事件导致田中本人在逃难中摔下楼梯，膝盖受轻伤，但是内心由于袭击者当场大喊「快把田中理恵交出來，我才是那个受害者」而受到重大打击<ref>[http://ccsx.tw/2013/06/24/nep-anime-event-accident/ 恐怖！暴漢持刀大鬧《超次元戰記 戰機少女》動畫首映會，導致「田中理恵」等聲優逃難受傷]</ref>。

事件发生后，田中所属的事务所也在她的博客《RieRie world》張貼一篇正式聲明，向所有參加此次活動的粉絲和特別來賓們致上最深的歉意，并感谢大家的关心与慰问。其后更导致田中本人宣布Twitter停止更新<ref>[http://www.sponichi.co.jp/entertainment/news/2013/06/25/kiji/K20130625006085480.html 声優・田中理恵、ツイッター休止へ…ナイフ男乱入 ケガは「回復」] - スポニチアネックス 2013年6月25日（2013年6月25日閲覧）</ref>。

有关事件的凶器，新闻中一般指为长约10公分的刀具，已经违反日本的《刀具管制法》；而在现场的粉丝则表示，凶器是铁撬一类的工具。

至于犯人的动机，结合犯人当场所言，有部分网友认为事件与一年前田中与山寺宏一结婚有关，犯人是田中的狂热粉丝（声优[[厨房_(网络用语)|厨]]或称声豚）。亦有人认为，犯人是山寺宏一的狂热粉丝，且是[[LGBT|LGBT]]人士。还有人认为，犯人是狂热的原作厨，因为不满本次电视动画改编而采取了行动。

== 交友關係 ==
*與女性的同業者能很快地熟稔，在自己所主持的廣播節目或是在別人所主持的節目中擔當特別來賓時，表現了異常高漲的情緒也是家常便飯。在這之中因為《[[天使的尾巴|天使的尾巴]]》而有所接觸的[[野川櫻|野川櫻]]與[[千葉紗子|千葉紗子]]兩者之間的交情相當不錯。在聚集在一起出演的特別活動中，可以看到互相在舞台上很親熱地聊天。在各自的廣播中、特別節目中的脫口秀裡或是官方網站中的日記裡也很熱衷於敘說對方的事情。

=== 與野川櫻的緣份 ===
*特別是與野川相同是從[[代代木動畫學院|代代木動畫學院]]出身，在田中以女主角的神代彌生的角色演出《[[破壞魔定光|破壞魔定光]]》時，野川則是以學級委員長的高杉光子第一次演出電視動畫的角色，到在其DVD特典中各自在不同的單元中出演都被認為是因緣之一（當時的交情還很淺）。
*在出演《極・聲優的京都》時走進了[[安利美特|Animate]]京都店時，正好看到野川的第一張專輯的宣傳影片在店裡放送中，於是離開店裡時也忠實地為這張專輯做了宣傳。
*在《有樂町 Anime Town》也聽到了野川所唱的曲子，相同的節目裡當野川以來賓的身份上節目時，田中表現了與平常所不能比近乎崩壞的高漲情緒<ref>在2006年11月19分放送的那次，野川說在兩人交換彼此手機郵件地址時可以看到田中表現出以往所看不到的「'''嬌羞'''」。之後田中馬上說了「'''真想被生成男生啊'''」。</ref>。
*因為所屬的公司不同的關係，有沒有共演作品的日子持續了好一陣子，但從《[[薔薇少女|薔薇少女]]》的共演之後又開始增加了。與上面所提到的[[千葉紗子|千葉]]在《[[舞-乙HiME|舞-乙HiME]]》中終於達成了三人久違的共演。

=== 與吉田尚記的孽緣 ===
*從2002年10月開始的日本放送《東京Character Show RADIO》（2005年4月以《有樂町 Anime Town》再開）等廣播節目開始共演的同局的廣播員。對於吉田尚記，有時吉田會拋著對方不管而不斷地談有關《[[鋼彈|鋼彈]]》的話題而進入暴走狀態的情況很多，一半感到愕然一半也以慎重的態度對待，有時可以看得出來田中處於正要生氣的狀態。
*最近毫不留情的吐嘈的價值慢慢顯現出來，吉田暴走的狀況比以前要多受到控制的感覺，現在也被評為「是少數可以控制吉比（吉田尚記的暱稱）的人之一」。而且，田中本人也認為與吉田彼此互相對話的能力也有提升。
*本人的說法是，與吉田是「哥哥與極力爭辯的妹妹」的關係。
*2007年春天，就像《有樂町 Anime Town》放送終了而替換成新節目一樣，田中在《MuComi》中以週四的助手的型式而繼續下去。

== 與遊戲有關的趣聞 ==
*聲優界公認的遊戲狂熱者，在自己的公式網頁裡的[[網誌|網誌]]中以電玩為話題的狀況也很常見，據說有著一到假日就埋頭在遊戲裡接近20個小時的事情（有工作的日子也會在空閒的時間打電動）。
*「[[女神轉生|女神轉生]]系列」的忠實玩家，在廣播與公式網站的網誌中也都提過「是從初代就開始玩的」。在日記中也經常記載玩自己有出演的《[[女神異聞錄3|女神異聞錄3]]》的事情等等，似乎相當入神的樣子。
*在《[[機動戰士鋼彈SEED|機動戰士鋼彈SEED]]》系列、《[[光之美少女_Max_Heart|光之美少女 Max Heart]]》或《[[The_Third_～蒼之瞳的少女～|The Third ～蒼之瞳的少女～]]》等等的收錄現在與共演的[[子安武人|子安武人]]討論遊戲相關的話題（主要以《[[神龍奇兵|神龍奇兵]]》為主），在《[[女神異聞錄3|女神異聞錄3]]》、《[[熱拳本色|熱拳本色]]》或《[[飛輪少年|飛輪少年]]》等作品的收錄現場中與共演的[[綠川光|綠川光]]熱烈地討論有關《[[女神異聞錄3|女神異聞錄3]]》的事情，此外與其他的共演者聊跟遊戲相關的事情也相當多的樣子。

=== [[網路遊戲|網路遊戲]]篇 ===

== 公式愛好者俱樂部「Cafe de Rie」 ==
: 「Cafe de Rie」是愛好者俱樂部的正式名稱。雖然在2003年7月才開始活動，但是募集卻是從4月開始。
: 除了向會員發行附有會員號碼的會員証以外，還向會員提供像是暑期問候等等的明信片、愛好者俱樂部的限定活動和活動門票的先行販賣、愛好者俱樂部的限定通信販賣、在專用網頁內的情報。
: FC名是CD、冊子（第3期以後）會報的名稱。CD內容以田中為店長、聽眾為客人構成。收錄會員寄來的留言、詩、畫作的介紹與自己本身的最新情報、原創歌曲、[[廣播劇|廣播劇]]等等。
: 冊子會報刊登著田中的最新情報、專欄、照片等等。另外，會員投稿的畫作、詩、感想等等也刊登在上面。

=== Cafe de Rie 第1期 ===
: 在2003年開始活動的公式愛好者俱樂部。會期從2003年7月到2004年6月為止，實驗性的舉行。
: CD方式的會報一年4回，每隔三個月發行一次（2003年8月時CD會報「Cafe de Rie Vol.1」發行）。毎回除了CD以外，也會附上照片與訊息卡片…等等，種種的贈品。
: 直到2003年秋天為止，在名古屋、大阪、東京租下咖啡廳，用來舉辦愛好者俱樂部會員限定的「Tea Party」（參加資格限定為一回）。
: 在公式網站上設置會員向的BBS，愛好者俱樂部會員可以利用它來交流。
: 年會費為6000日圓，第一年可獲得減少年會費1000日圓的優惠（第一年的年會費為5000日圓）。最終的推定會員數大約1200人。

=== Cafe de Rie 第2期 ===
: 2004年度後半開始的公式愛好者俱樂部。因為Cafe de Rie第1期被受好評而決定延長，會期從2004年7月開始到2005年6月結束。
: 除了「Cafe de Rie」第1期所發行的CD會報以外，活動也積極的進行著。
: 田中與愛好者俱樂部的會員直接通電話的「Tea Call」（2005年1月）以及在都內的咖啡廳的「Tea Party」（2005年1月），還有「Cafe de Rie Summer Special Live&Party 2004」（2004年8月）與初次登上舞臺表演的《壽!》（2005年4月）的門票先行販賣給愛好者俱樂部會員。
: 另外，愛好者俱樂部會員展開私下所策劃的計劃，在演唱會活動「Tea Party」上，替當天不能抵達會場的同好者約50人向田中贈送訊息與花束。但是這個計劃並不是愛好者俱樂部的公式活動。
: 《壽!》DVD的通販如果是經由FC而購入的，則可以獲得會員限定的特典（表演中的感光紙照片）。
: 事務局實施關於下個年度的愛好者俱樂部營運的調查，並且在次日向回答調查的會員寄送照片。

=== Cafe de Rie 第3期 ===
: 2005年度後半開始的公式愛好者俱樂部。會期從2005年7月開始到2006年6月結束。
: 第3期開始年會費變更成5000円，會報的形式變更為CD會報一年2回，紙會報一年2回。在2005年7月寄送暑期問候，2005年9月初次以冊子的形式發行會報。
: 「Tea Party」在大阪2005年11月27日、東京2005年12月4日舉辦。在大阪舉辦的「Tea Party」為今年首度舉辦。「Tea Call」進行方式和去年一樣。「Tea Party」原則上是為了讓應募的全員，在兩個會場中能夠參加其中一個（最後希望參加兩個會場活動的人也可以全部參加）。「Tea Call」也向應募者全員打電話。
: 12月時CD會報Vol.8發行。
: 2006年1月1日發送賀年卡。
: 第3期再次舉辦的第2回「Tea Party」於2006年5月28日在東京舉辦，隔週的6月5日則在大阪舉辦。
: 在7月底以前完成第4期更新手續的會員會收到暑期問候的訊息卡片與照片。

=== Cafe de Rie 第4期 ===
: 2006年度後半開始的公式愛好者俱樂部。會期從2006年7月到2007年6月結束。
: 基本的內容和第3期是一樣的。
: 10月時發行冊子形式的會報Vol.3。
: 12月3日在[[日本放送|日本放送]]本社內的「imagine STUDIO」裡舉辦會員限定的迷你演唱會，同月10日在[[大阪市|大阪市]]內的咖啡廳裡舉辦「Tea Party」。
: 2007年6月17日在[[東京鐵塔|東京鐵塔]]內舉辦「Tea Party」。

=== Cafe de Rie 第5期 ===
: 2007年度後半開始的公式愛好者俱樂部。會期從2007年7月開始到2008年6月結束。
: 基本的內容和第4期是一樣的。
: 2007年11月11日在[[東京鐵塔|東京鐵塔]]內以及11月25日在[[大阪市|大阪市]]內的咖啡廳裡舉辦「Tea Party」。 
: 2008年1月27日在都內舉辦約3年5個月內沒舉辦過的獨唱演唱會（先對會員先行發售，之後才對大眾一般發售）。
