'''田义祥'''（{{bd|？|||}}）籍贯不详，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
田义祥曾任[[中国人民解放军总参谋部作战部|中国人民解放军总参谋部作战部]]应急办公室主任。任内曾在2008年[[汶川大地震|汶川大地震]]期间兼任军队处置突发事件领导小组办公室负责人，在[[2008年北京奥运会|2008年北京奥运会]]期间担任北京奥运安保协调小组军队工作部部长<ref name=ifeng>{{Cite web |url =http://news.ifeng.com/a/20160427/48602439_0.shtml  |title =总参动员部原副部长田义祥出任军委办公厅副主任  |author =  |date = 2016-04-27 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://zqb.cyol.com/content/2008-05/09/content_2173811.htm |title =  解放军奥运安保部队严阵以待确保平安奥运——专访北京奥运安保协调小组军队工作部部长田义祥|author =  |date =2008-05-09   |publisher =  中国青年报|language =  |accessdate =  |quote =  }}</ref>。2011年任[[国家国防动员委员会综合办公室|国家国防动员委员会综合办公室]]专职副主任<ref>{{Cite web |url =http://news.163.com/11/0915/20/7E13NVA100014AEE.html  |title =国内城市规划专家齐聚廊坊 共话城市形象提升  |author =  |date = 2011-09-15 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。2013年任[[中国人民解放军总参谋部动员部|中国人民解放军总参谋部动员部]]副部长。2016年1月任[[中央军委办公厅|中央军委办公厅]]副主任<ref name=ifeng/>。2018年5月，出任[[中央军委审计署|中央军委审计署]]审计长<ref>{{cite web |title=田义祥少将接替郭春富 出任中央军委审计署审计长 |url=http://news.163.com/18/0614/13/DK90D81N000187VE.html |website=网易 |accessdate=2018-06-15}}</ref>。

2012年7月晋升少将军衔。
