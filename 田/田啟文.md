'''田啟文'''（{{lang-en|Tenky Tin Kai Man}}，{{bd|1961年|10月6日}}），[[綽號|綽號]]'''田雞'''，[[香港|香港]][[演員|演員]]及[[香港電影|電影]]人，曾是[[周星馳|周星馳]]所屬之[[星輝公司|星輝公司]]負責人之一，負責形象推廣、業務拓展、演藝經紀及至影片的監制策劃等，[[香港電影工作者總會|香港電影工作者總會]]會長。他代表香港電影製作行政人員協會任香港電影金像獎董事局董事<ref>{{cite web |url=https://topick.hket.com/article/2270717/%E3%80%8A%E6%96%B0%E5%96%9C%E5%8A%87%E4%B9%8B%E7%8E%8B%E3%80%8B%E7%94%B0%E5%95%9F%E6%96%87%E5%85%A5%E8%A1%8C%E8%BF%9140%E5%B9%B4%E8%A6%96%E6%98%9F%E7%88%BA%E5%8D%8A%E5%80%8B%E5%B8%AB%E5%82%85%E3%80%80%E7%94%B0%E9%9B%9E%EF%BC%9A%E6%85%B6%E5%B9%B8%E4%BA%BA%E7%94%9F%E9%81%87%E5%88%B0%E5%91%A8%E6%98%9F%E9%A6%B3 |title=《新喜劇之王》田啟文入行近40年視星爺半個師傅　田雞：慶幸人生遇到周星馳 |last= |first= |date=2019-2-14 |website= |publisher=香港01 |access-date= 2019-2-18|quote=}}</ref>，也是香港電影工作者總會第七屆執行委員會（2017年至2019年）會長<ref>{{cite web |url=https://www.singtaousa.com/%E6%97%A5%E5%A0%B1/%E5%A8%9B%E6%A8%82/1061188-%E7%94%B0%E5%95%9F%E6%96%87%E7%88%86%E6%9E%97%E6%B7%91%E8%B2%9E%E5%BF%83%E6%83%85%E5%BF%90%E5%BF%91/ |title=田啟文爆林淑貞心情忐忑 |last= |first= |date=2018-7-27 |website= |publisher=星島日報 |access-date= 2019-2-18|quote=}}</ref>與香港紅星[[鄭中基|鄭中基]]的經理人<ref>{{cite web |url=https://hk.on.cc/hk/bkn/cnt/entertainment/20150212/bkn-20150212151514966-0212_00862_001.html |title=田啟文爆鄭中基喜獲B仔湊「好」字 |last= |first= |date=2015-2-12 |website= |publisher=東網 |access-date= 2019-2-18|quote=}}</ref>。田啟文任職星輝時為周星馳最信賴的人和得力下屬。目前已自立門戶，創辦「腦力工作室有限公司」繼續發展電影事業。

==背景==
田啟文早年在[[九龍塘|九龍塘]]火車站旁邊的小屋居住和種植農作物，後來移居[[九龍灣|九龍灣]]木屋區（[[臨時房屋區|九龍灣臨時房屋區]]，[[MegaBox|MegaBox]]商場一帶）與[[秀茂坪|秀茂坪]]徙置區21座。他小學畢業後到昔日位於[[觀塘|觀塘]][[裕民坊|裕民坊]]銀都戲院附近的中學就讀初中。但讀至中二時因為家窮而輟學，並開始在酒茶居工作攢錢。同時在夜暑讀夜校完成學業。田啟文在茶居任職點心學徒兼侍應，跟隨姨丈學習做點心為生。他及後轉到一間酒樓繼續工作。當了三年點心學徒後升為腸粉師傅（腸粉部主管）{{noteTag|關於田啟文早年生活的資料出自香港電台第二台廣播節目《守下留情》於2018年2月26日播出的訪問「三師兄田雞田啟文訪問第一集 + 尋找女郵差叔叔的故事第一回，重溫！」。}}。田啟文於18歲時加入電視圈。當時他的一位點心學徒師弟邀請他到麗的電視當臨時演員，於是田啟文便在電視台展開其演員生涯。但還未正式加入電視圈。而他之所以立志在娛樂圈發展始於他看過的一齣電影《五毒天羅》（1976年由[[楚原|楚原]]導演）。直至他有一次在[[又一村|又一村]]會堂的酒樓認識了唱片騎師李學斌，並向對方毛遂自薦，希望可獲入行機會。李學斌勸喻田啟文思考清楚入行之後應如何發展才能站得住腳。結果，田啟文因入行心切而放棄腸粉師傅的正職。在其父親極力反對下，16歲的他仍然堅持當全職臨時演員且離家出走，在麗的電視台寄宿。為了多賺收入與參演機會，他主動爭取成為臨時演員公司的領班。不久更轉戰幕後，學懂了一些服裝管理、化妝、剪接、拍攝等技巧。初時，田啟文先當場記，再兼任劇務。其第一部負責幕後工作的劇集為1979年的《[[天蠶變_(電視劇)|天蠶變]]》。與此同時，田啟文在劇集中作臨時演員演出至1980年代初，[[邱德根|邱德根]]入主亞洲電視以後要求削減幕後人員的薪金。田啟文不想接受減薪安排，又認為已學習到拍攝作品的基本技巧。因此決定轉往電影界發展。{{noteTag|關於田啟文入行至轉戰電影界的經過的資料出自香港電台第二台廣播節目《守下留情》於2018年2月27日播出的訪問「三師兄田雞田啟文訪問第二集 + 好想食腸粉，重溫！」。}}

田啟文轉拍電影後初年，於友人的桑拿浴室寄宿。首部參演的電影為《[[忌廉溝鮮奶_(電影)|忌廉溝鮮奶]]》。他當年未完全投電影界前與[[香港電台|香港電台]]簽了一年合約，參演十多集劇集《左鄰右里》。另外，亦任職電影劇務、執行製片、副導演等崗位。田啟文本來不認識周星馳。直至於1992年拍攝《[[武狀元蘇乞兒|武狀元蘇乞兒]]》時，本為專門照顧周星馳的製片助理，即其胞妹周星霞轉工。沒有人肯負責這項工作。最後田啟文擔任該職位，逐漸與周星馳熟絡和得到對方信任。田啟文在1994年拍攝《[[九品芝麻官|九品芝麻官]]》後開始在[[周星馳|周星馳]]的電影中演出。及後在周星馳主演的電影中，田啟文多有亮相。在該段時期，田啟文屬永盛影視的員工。至1996年，獲周星驰邀請加盟他所創立的[[星輝海外有限公司|星輝電影公司]]。{{noteTag|關於田啟文轉戰電影界前後經過、與周星馳熟絡和加入星輝電影公司的資料出自香港電台第二台廣播節目《守下留情》於2018年2月28日、3月1日播出的訪問「三師兄田雞田啟文訪問第三集 + 蘿蔔糕戰車，重溫！」、「三師兄田雞田啟文訪問第四集 + 守下留情精選三重彩之元宵盃，重溫！」。}}除幕前演出外，田啟文亦負責行政工作，是周星馳班底的主要人物。而在電影以外，田啟文亦不時主持電視節目，如[[亞洲電視|亞洲電視]]的《[[老散窩輪周記|老散窩輪周記]]》。田與周二人合作至2009年結束賓主關係。田於同年創辦「腦力工作室有限公司」，主力從事電影製作與培育新人工作<ref>{{cite web |url=http://form.hktdc.com/UI_VisitorIntranet/Public/SpeakerDetailPublic.aspx?EVENTID=e839d1af-4ac9-468f-a80c-8827259461fa&SPEAKERID=d5614d6e-a017-408b-9fa7-08d7bed7d8c0&LANGID=2 |title=開幕CEO論壇及主題研討會 名人分享坊 講者資料 田啟文先生 |last= |first= |date= |website= |publisher=香港貿易發展局 |access-date= 2019-2-18|quote=}}</ref>。

田啟文於2016-2017年度微笑企業大獎頒獎典禮擔任微笑大使，頒獎予港澳地區多個於微笑服務表現優異的企業。2017年，田啟文接受訪問時憶述與[[周星馳|周星馳]]的合作，細數合作的起源，透露[[周星馳|周星馳]]曾經從與田啟文的對話中取材，變成後來電影《喜劇之王》中的對白，並對周星馳表示感激 <ref>[http://bka.mpweekly.com/interview/%E9%9B%86%E9%AB%94%E5%9B%9E%E6%86%B6/%E7%94%B0%E9%9B%9E-%E6%98%9F%E7%88%BA-%E5%91%A8%E6%98%9F%E9%A6%B3-%E9%9B%86%E9%AB%94%E5%9B%9E%E6%86%B61 【集體回憶】揭穿巨星的弱點　田雞星爺奇緣廿五年（一）]</ref>。其後憶述於拍攝電影[[食神|食神]]時，因發錯通告而要化解與萬梓良之間的誤會<ref>[http://bka.mpweekly.com/interview/%E9%9B%86%E9%AB%94%E5%9B%9E%E6%86%B6/%E7%94%B0%E9%9B%9E-%E5%91%A8%E6%98%9F%E9%A6%B3-%E5%9C%8B%E7%94%A2%E5%87%8C%E5%87%8C%E6%BC%86-%E9%9B%86%E9%AB%94%E5%9B%9E%E6%86%B6?cat=3 【集體回憶】化解萬子「六國大封相」 田雞星爺奇緣廿五年（三）]</ref>
