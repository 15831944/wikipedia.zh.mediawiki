'''田志希'''（'''{{lang-ko|전지희}}'''，{{bd|1992年|10月28日||}}），轉籍前原名'''田旻煒'''，是一名[[韓國|韓國]]的女子[[乒乓球|乒乓球]]運動員<ref>{{cite web|url=http://www.ittf.com/ittf_stats/stats_per_player.asp?player_id=118994|title=Stats Records - ITTF Statistics|deadurl=yes|archiveurl=https://web.archive.org/web/20140421114933/http://www.ittf.com/ittf_stats/stats_per_player.asp?player_id=118994|archivedate=2014-04-21}}</ref><ref>{{cite web|url=http://www.ittf.com/ittf_ranking/world_ranking_per_name.asp?Siniors=1&Player_ID=118994|title=World ranking Record for JEON Jihee (KOR)|deadurl=yes|archiveurl=https://web.archive.org/web/20140421114938/http://www.ittf.com/ittf_ranking/world_ranking_per_name.asp?Siniors=1&Player_ID=118994|archivedate=2014-04-21}}</ref><ref>{{cite web |url=http://www.ittfranking.com/cgi-bin/ttplayer.cgi?number=118994&lang=0&format=detailed|title=JEON Jihee (KOR) - ITTF Rankings}}</ref>。

出生於[[中國|中國]][[河北省|河北省]][[廊坊市|廊坊市]]。她的父亲是小学乒乓球教练，受此影响，七岁开始进行乒乓球训练。2008年，进入中国[[鲁能俱乐部|鲁能俱乐部]]。同年，转到韩国。2011年，成为父亲[[朝鲜族|朝鲜族]]朋友的养女，通过韩国[[归化考试|归化考试]]，1月加入[[韩国国籍|韩国国籍]]<ref name="a1"/>。

[[2014年仁川亚运会|2014年仁川亚运会]]前，田志希在韩国媒体的采访中表示“每次看到太极标志（韩国国旗）都会心情激动万分”，“如今在世界排名中列第四、第五的中国选手都是我曾一起在青年队中的队友。我是代表韩国参赛的韩国人。想着一定要打败她们”。此报道在中国引起热议，中国网友称之为“第二个[[小山智丽|小山智丽]]”<ref name="a1">{{Cite web |url=http://sports.qq.com/a/20140401/004245.htm |title=中国乒乓新星归化韩国 称要在亚运会击败国乒 |accessdate=2015-09-19 |author=麦媛 |date=2014-04-01  |work= |publisher=腾讯网|language=Zh-hans }}</ref>。而面对中国媒体的采访时，田志希则表示“我从来没有说过想击败中国，不知道为什么韩国媒体会这样写。”“而且以我现在的实力，确实和国乒主力选手还有差距，能在中国打上主力都是世界顶级选手。”<ref>{{Cite web |url=http://sports.qq.com/a/20140927/032024.htm |title=韩乒归化小将：没说想击败中国 比国内挣得多 |accessdate=2015-09-19 |author=记者：王怡薇 |date=2014-09-27 |work= |publisher=腾讯网|language=Zh-hans }}</ref>最终，在[[2014年亚洲运动会乒乓球比赛|2014年仁川亚运会乒乓球比赛]]中，她与[[金珉錫|金珉錫]]合作获得男女混合双打第三名（与日本队并列）。
