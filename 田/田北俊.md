'''田北俊'''{{GBSHK}}<span style="font-size:smaller;">，[[OBE|OBE]]</span>{{JP}}（{{lang-en|'''James Tien Pei-chun'''}}，{{bd|1947年|1月8日|catIdx=T田}}），人稱「'''田少'''」，祖籍[[安徽|安徽]]，生於[[上海|上海]]，[[田元灝家族|田元灝家族]]成員，[[香港|香港]]商人、政治人物。現為[[自由黨_(香港)|自由黨]]榮譽主席。曾任[[香港旅遊發展局|香港旅遊發展局]]主席、[[行政會議|行政會議]]成員、[[中國人民政治協商會議|中國人民政治協商會議]]第10至12屆全國委員會委員（於第12屆中途被撤職）、[[香港立法會|香港立法會]]議員和[[香港立法局|立法局]]議員。

==簡介==
田北俊1964年在[[拔萃男書院|拔萃男書院]]中學畢業，之後留學[[美國|美國]]，獲[[化學工程|化學工程]]碩士學位。他是[[萬泰製衣|萬泰集團]]主席。

田北俊1985年獲委任為[[葵青區議會|葵青區議會]]議員開始踏足政界，1988年被[[港督|港督]][[衛奕信|衛奕信]]爵士委任為[[香港立法局|立法局]]議員，任期至1991年，之後並沒有再被委任。

時任工業界（第一）[[功能界別|功能界別]]議員[[張鑑泉|張鑑泉]]在1993年度[[香港記者協會|香港記者協會]]晚宴後突然去世，田北俊臨危受命參與補選，並重返立法局，之後順利過渡，成為[[臨時立法會|臨時立法會]]議員，並於1998年[[1998年香港立法會選舉|參加立法會功能組別選舉]]，成功當選[[功能組別|功能組別]]（商界(一)）議員<ref name="田北俊">{{cite web|language=zh-hk|publisher=|title=www.jamestien.com|url=http://www.jamestien.com/Profile.htm|author=|date=|accessdate=2008-09-12}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2003年7月6日，時任[[自由黨_(香港)|自由黨]]主席田北俊在[[香港立法會|香港立法會]]就《國家安全條例草案》進行二讀辯論前，一向支持法案條文內容的他突然宣佈不支持於7月9日進行二讀，並即時辭去[[行政會議|行政會議]]職務；特區政府於是連夜召開行政會議特別會議。由於欠缺自由黨於立法會的支持，條例草案將無法通過，當時行政長官[[董建華|董建華]]於7月7日凌晨宣佈撤回草案<ref name="董建華聲明">{{cite web |language = zh-hk|publisher=政府新聞公報 |title=行政長官董建華發表聲明 |url= http://www.info.gov.hk/gia/general/200307/07/0707015.htm|author= |date=2003-07-07 |accessdate=2008-09-13 }}</ref>。董建華撤回草案後稱，政府並無立法時間表，政府在得到社會的支持時才展開立法程序<ref name="政府撤回23條草案">{{cite web|language=zh-hk|publisher=香港政府新聞網|title=政府撤回國家安全條例草案|url=http://www3.news.gov.hk/isd/ebulletin/tc/category/lawandorder/030905/html/030905tc08003.htm|author=|date=2003-09-05|accessdate=2008-09-13|deadurl=yes|archiveurl=https://web.archive.org/web/20060104021517/http://www3.news.gov.hk/ISD/ebulletin/tc/category/lawandorder/030905/html/030905tc08003.htm|archivedate=2006-01-04}}</ref>。董建華其後委任[[周梁淑怡|周梁淑怡]]接替田北俊出任[[行政會議|行政會議]]<ref name="行政長官委任周梁">{{cite web |language = zh-hk|publisher=政府新聞公報 |title=行政長官委任行政會議新成員 |url=http://www.info.gov.hk/gia/general/200309/22/0922143.htm |author= |date=2003-09-22 |accessdate=2008-09-12 }}</ref>。[[2004年香港立法會選舉|2004年香港立法會選舉]]，田北俊首次在地區直選[[新界東選區|新界東選區]]順利當選。

[[2008年香港立法會選舉|2008年香港立法會選舉]]，田北俊在地區直選[[新界東選區|新界東選區]]落敗，失去直選議席，第二度結束立法會生涯。

[[2012年香港立法會選舉|2012年香港立法會選舉]]，他代表[[自由黨_(香港)|自由黨]]參加立法會選舉出戰新界東地區直選，成功當選，重返立法會。

田北俊自[[梁振英|梁振英]]上任後一直批評梁振英。[[2012年香港行政長官選舉|2012年香港行政長官選舉]]，自由黨支持[[唐英年|唐英年]]，拒絕支持梁振英。

== 評價 ==
田北俊雖然是建制派人士，但其立場較溫和。雖然會批評泛民主派，但不會不留情面，對於建制派及政府施政亦都會毫不留情地批評，在建制派人士中民望屬較高的人。

== 積極參與賽馬活動 ==
田北俊、田黃玉華夫婦和兒子田啟文、女兒[[田啟安|田啟安]]（包括其夫婿鄭宇）是[[香港賽馬會|香港賽馬會]]會員，自1994-1995馬季開始擁有21匹以「武士」、「勇士」命名的馬匹，包括快樂武士、醒目武士、幸運武士、忠心武士、自由武士、新東武士、聰明武士、精靈武士、神速武士、現代武士、風雲武士、閃爍武士、影之武士、人民武士、威水武士、陽剛武士、盈威勇士、忠心勇士。（為表示公平，只列出田北俊家族個人名下馬匹。）

==爭議==
1994年6月29日，田北俊在立法局审议1994年立法局（選舉規定）（修訂）條例草案二读辩论发言：“我最近見到的英國朋友都說：「唉！如果我們這十幾廿年來有咁既一個政府管治，我們就不會如此「仆街」！（對不起，是經濟上這樣不濟。）”<ref>[http://www.legco.gov.hk/yr93-94/chinese/lc_sitg/hansard/h940629.pdf 立法局记录(页码3528)]</ref>这是香港立法局成立有记录以来首次有人使用这个词。

2004年8月17日，田北俊在選舉論壇上，回應[[社民連|社民連]][[梁國雄|梁國雄]]關於其質疑自由黨經常從政府中得到好處時。田北俊在事件中的說法引起公眾對他的說話輕視基層，感到不滿。

2004年12月23日，田北俊嚴厲批評當時政務司司長[[曾蔭權|曾蔭權]]主持的政策委員會是將[[西九龍文化區#2005年計劃修改|西九龍發展計劃]]繞過行政會議及立法會。事件中他的說話，惹來公務員團體的不滿。

2005年5月20日，田北俊回應發展商進行樓盤內部認購時沒有價目表的說法不可靠，事後他為失言而道歉。

== 住所 ==
