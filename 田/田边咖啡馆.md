'''田边咖啡馆'''<ref name="taiguo" />（{{lang-th|ร้านกาแฟชายทุ่ง}}，{{lang-en|Chai Thung Coffee}}）是一家位于[[泰国|泰国]][[巴吞他尼府|巴吞他尼府]]{{link-en|探耶武里县|Thanyaburi District}}以[[诗琳通公主|诗琳通公主]]为主题的[[咖啡馆|咖啡馆]]，由诗琳通公主所开设和亲自命名。该馆为单层[[别墅|别墅]]式建筑，店内[[装修|装潢]]有[[诗琳通公主|诗琳通公主]]的画像及照片<ref>{{cite web |url=https://workpointnews.com/2018/09/28/%E0%B8%AA%E0%B8%B1%E0%B8%A1%E0%B8%9C%E0%B8%B1%E0%B8%AA-%E0%B8%A3%E0%B9%89%E0%B8%B2%E0%B8%99%E0%B8%81%E0%B8%B2%E0%B9%81%E0%B8%9F%E0%B8%8A%E0%B8%B2%E0%B8%A2%E0%B8%97%E0%B8%B8%E0%B9%88%E0%B8%87/ |title=สัมผัส “ร้านกาแฟชายทุ่ง” ร้านกาแฟของสมเด็จพระเทพฯ |last= |first= |date=2018-09-28 |website=Workpoint News |publisher= |access-date=2018-09-29 |quote=}}</ref>，周围有绿树和池塘环绕，装修风格简单<ref>{{cite web |url=https://siamrath.co.th/n/47724 |title=พระเทพฯ จะเสด็จฯ เปิด “ร้านกาแฟชายทุ่ง” รังสิต-คลองสี่ เสาร์ที่ 29 ก.ย.นี้ |last= |first= |date=2018-09-26 |website=siamrath |publisher= |access-date= 2018-09-27|quote=}}</ref>。

== 开业 ==
