'''田佳良事件'''，又称'''洁洁良事件'''，是发生于2018年4月的一起网络事件。[[厦门大学|厦门大学]]研究生、中共党员田佳良在[[新浪微博|新浪微博]]使用“洁洁良”的账号发表一系列批評中国人素質極差，以及嚴重反感中国及中共的言论。期间使用一系列与[[支那|支那]]相关的词汇，以表达她对中国和中共的羞辱<ref name="a1"/>，被中国輿論認為是“[[精日|精日]]、[[反华|反华]]”分子，随后又被揭發有學術不端行為。

== 厦门大学与田佳良 ==
田佳良，女，[[辽宁|辽宁]][[兴城|兴城]]人，毕业于[[辽宁师范大学|辽宁师范大学]]城市与环境学院，期间加入[[中国共产党|中国共产党]]。[[厦门大学|厦门大学]]环境与生态学院在读研究生。她曾阐述：“在思想上，我积极向上，热爱祖国，热爱中国共产党，拥护中国共产党的领导......”<ref name="a1"/>。[[新浪微博|新浪微博]]账号“洁洁良”<ref name="hxcb">{{cite news|title=厦门大学:给予"精日"女生留党察看、留校察看处分|url=http://news.163.com/18/0424/00/DG4AF9LM0001875P.html|accessdate=2018-04-25|work=海西晨报|date=2018-04-24}}</ref>，事发后田佳良注销账号<ref name="a2"/>。

厦门大学是爱国华侨[[陈嘉庚|陈嘉庚]]在1921年所创办。[[抗日战争|抗日战争]]期间，陈嘉庚更是主张积极对日抗战，1938年在第二届[[国民参政会|国民参政会]]上提交议案：“日寇未退出我国土之前凡公务员对任何人谈和平条件概以[[汉奸|汉奸]]国贼论”（后世总结为“言和即漢奸論”），被视为“最伟大提案”<ref>{{Cite web |url=http://news.sohu.com/20070810/n251518379.shtml |title=陈嘉庚“最伟大提案”电报原稿在南京发现(图) |accessdate=2018-04-25 |author= |date=2007-08-10 |work= |publisher=搜狐网，来源：台海网|language=Zh-hans }}</ref>。厦门大学宣传时亦称党政领导将陈嘉庚先生的爱国精神视为厦门大学的优良传统，是厦门大学的“四种精神”之一<ref>{{Cite web |url=http://www.xujc.com/index.php?c=Article&a=view&id=17839 |title=厦门大学的“四种精神” |accessdate=2018-06-23 |author= |date=2013-12-04 |work= |publisher=厦门大学嘉庚学院网站|language=Zh-hans }}</ref>。事发前，田佳良提及通过厦门大学组织的党员参观活动，“熟知了福建省革命的发源及过程，也体会到了我校独有的爱国情怀与革命的光荣传统。<ref name="a1"/>”因此，田佳良厦门大学的学生身份亦被网友和厦门大学校友所愤慨<ref name="a2">{{Cite web |url=http://m.news.cctv.com/2018/04/24/ARTIDX9f3FF3DokL12d9XpJw180424.shtml |title=厦大女生发表辱华言论 校方处理结果出来了 |accessdate=2018-06-23 |author= |date=2018-04-24 |work= |publisher=央视网|language=Zh-hans | quote =“有网友和厦门大学校友发帖表示，厦门大学素有爱国传统，并贴出厦大创办人、爱国华侨陈嘉庚先生为抗日筹集资金，四处奔走的故事。而如今，这位抗日华侨创立的大学里，居然出现了这样一位，身为党员满口“支那”的人。”}}</ref>。

== 事件經過 ==
=== 借時事侮辱人群 ===
2018年4月19日，[[華特迪士尼影業集團|迪士尼影業]]為《[[復仇者聯盟3：無限之戰|復仇者聯盟3：無限之戰]]》在[[上海迪士尼乐园|上海迪士尼]]舉辦的宣傳活動。由於迪士尼公關安排失當，造成活動現場混亂，觀眾人身安全一度無法保證。19日活動結束後，觀眾被批評在現場遺留大量垃圾。微博用戶“潔潔良”對此發出“惡臭[[支那|你支]]”的評價{{noteTag|該詞語中的“恶臭”是“[[淫梦|淫梦]]小鬼”（即被中国[[仲夏夜之淫梦#淫梦民|淫梦民]]所不齿的滥用/误用淫梦用语的对象）特有用语，比通常中文字面上的“恶臭”更恶劣。<ref>{{cite web |author1=ACGdoge |title=恶臭税，日本学校班级设立淫梦语录使用罚款三周收入将近 1.9 万日元|url=http://www.acgdoge.net/archives/22339 |website=www.acgdoge.net |accessdate=2018-09-18 |language=zh-cn |date=2017-11-20}}</ref> }}。其後19日至20日期間，潔潔良進行評論時，繼續使用“[[小粉紅|粉紅豚]]”、“國家支意”等一系列“[[支那|支那]]”一类貶義詞{{noteTag|該詞語在中國清末民初時期開始，由原本一個中性稱號被加以扭曲，到二戰後被普遍中國人視為仇視中華人民共和國及中華民國的[[歧视语|歧视语]]；在日本，除少数地理名词（如[[南海|南海]]被称为“南支那海”（{{lang|ja|南シナ海}}））外，“支那”一词也被视为差别用语而被禁止在大众媒体使用，但民间使用并未受限。 }}。隨後有網友私信提醒，“支那”是侮辱中國的意思，但“潔潔良”表示“廢話，不侮辱，我說它幹嘛”。不但拒絕刪帖，而且變本加厲愈鬧愈烈。當被提及本科學校時，否認畢業於遼寧師範大學的同時，更稱之為“[[野雞大學|野雞大學]]”<ref name="a1"/>。

=== 中共幹部身份被公開 ===
随后“洁洁良”被网友人肉搜索，其身份也被曝光。迫于压力，田佳良删除了微博账号。田佳良是中共党员，曾获12项荣誉称号，现担任厦门大学2015级硕士生第三党支部书记，并已获得博士生保送资格。在这样的身份之下，她的言论引起舆论一片哗然。相当一部分的质疑转向了田佳良的中国共产党干部身份，指出了中共在入党与干部选拔上的功利主义问题。此外，网友人肉搜索出在微博上为田佳良辩护的用户“热心人士赵先生”实为田的男友赵郁鑫，此人也是厦大研究生，同样长期发表亲日反华言论。

=== 涉嫌学术不端 ===
之后有网友找出其本科期间发表过的论文《中国水市场的运作模型研究》，文章完全抄袭自2001年的一篇同名论文（包括标题、摘要），2014年发表在《北方经贸》上的《辽西港口建设对城市发展的影响分析》一文，与2009年发表论文《辽宁沿海区域“[[辽宁沿海经济带|五点一线]]”经济带中的锦州湾沿海经济区开发研究》文字复制比高达78.5%。由于田佳良凭借本科期间发表过的论文被保送厦大研究生，故也引发了对厦大及其本科学校——[[辽宁师范大学|辽宁师范大学]]的质疑，质疑集中在保研程序的合法性上<ref name="a1">{{cite news|author1=校园司令|title=厦大一党支部书记辱华上热搜，竟还涉嫌论文抄袭，厦大和紫光阁回应了|url=http://www.sohu.com/a/229060433_231683|accessdate=2018-04-25|publisher=搜狐}}</ref>。

== 处理结果 ==
2018年4月23日，厦门大学环境与生态学院发布《关于对田佳良同学处理情况的通报》，给予该生留党察看、留校察看的处分<ref name="hxcb" />。但“双留”决定在当时引起了部分人的不满，认为处罚过轻，甚至有网民转而向中纪委国家监委举报<ref name=dw01>{{cite web|url=http://news.dwnews.com/china/news/2018-04-25/60054265.html |title=习近平曾誓言清除两面人 港媒：厦大处理田佳良令人不满|publisher=多维新闻|date=2018-04-25|accessdate=2018-09-04}}</ref>。8月15日，鉴于田佳良在留党察看期间又被发现有学术不端行为，厦门大学环境与生态学院党委依纪给予田佳良开除党籍处分。同时，学院提出中止田佳良博士培养、给予退学处理的意见，学校研究同意田佳良退学，而田佳良也已经办理了退学离校手续。这一消息在厦门大学新闻网于9月1日发表的一篇报道中得到了证实，与其同时被整肃的还有厦大历史学系助理教授[[周运中|周运中]]<ref>{{cite news|author1=|title=厦门大学:“精日”研究生田佳良被开除党籍退学|url=http://news.sina.com.cn/c/2018-09-02/doc-ihinpmnr5966251.shtml|accessdate=2018-09-02|publisher=观察者网}}</ref>。据办公地点在北京的媒体[[多维新闻|多维新闻]]表示，开除田佳良可能来自于习近平试图清除党内[[两面人|两面人]]的指示精神<ref name=dw01/>。

==评论==
事件发生后，中国内地主流舆论对田佳良的言论纷纷表示谴责，并支持厦大对田氏开除学籍以及勒令退学的处理。

但在海外部分媒体中却有不同观点，如[[法国国际广播电台|法国国际广播电台]]评论就认为其“讲真话”“碎了强国玻璃心”，遭打击报复被开除党籍及勒令退学<ref>{{cite web|url=http://cn.rfi.fr/%E4%B8%AD%E5%9B%BD/20180903-%E5%8E%A6%E5%A4%A7%E5%8D%9A%E5%A3%AB%E7%94%9F%E6%95%99%E6%8E%88%E6%B6%89%E8%AE%B2%E7%9C%9F%E8%AF%9D%E7%A2%8E%E4%BA%86%E5%BC%BA%E5%9B%BD%E7%8E%BB%E7%92%83%E5%BF%83%E8%A2%AB%E5%BC%80%E9%99%A4%E5%85%9A%E7%B1%8D%E6%95%99%E8%81%8C |title=厦大博士生教授涉讲真话碎了强国玻璃心被开除党籍教职|publisher=法国国际广播电台|date=2018-09-03|accessdate=2018-09-04}}</ref>。

对于“田佳良事件”以及中国内地其它高校频现“精日”，香港《[[东方日报|东方日报]]》认为，内地高校腐败频发，部分校长及教授沉迷权钱美色，高校变成勾心斗角、尔虞我诈的生意场，成为“精日”泛滥的原因。该报也指出中共“两面人”的问题，表示不少人加入中共只为前途投机，甚至表示中共到底有多少“两面人”“只有神知道”。<ref name=dw01/>
