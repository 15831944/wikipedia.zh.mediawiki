《'''就是愛。'''》（{{lang-ja|愛でした。}}）是[[日本|日本]]男子組合[[關西傑尼斯8|關西傑尼斯8]]的第20張單曲，於2012年6月13日由[[Imperial_Records|Imperial Records]]發行。

== 發行及宣傳 == 
《就是愛。》跟上一張單曲《[[愛著你的一切|愛著你的一切]]》（{{lang|ja|ツブサニコイ}}）發行相隔約10個月，為[[關西傑尼斯8|關西傑尼斯8]]於2012年，即出道八週年發行的首張單曲。<ref name="CDJournal">{{cite web|url=http://artist.cdjournal.com/d/-/4112050824 |language=ja |title=<span lang="ja">関ジャニ∞(エイト)/愛でした。</span> |publisher=CDJournal |date=2012 |accessdate=2013-02-09}}</ref>單曲分為「初回限定盤」和「通常版」，共兩種形式。

為紀念出道八週年，關西傑尼斯8於2012年8月8日於[[大阪巨蛋|京瓷巨蛋大阪]]舉辦讓四萬名歌迷免費參加的特別活動。希望參加的人須擁有此單曲和下一張單曲《[[ER_(關西傑尼斯8單曲)|ER]]》{{noteTag|1=此消息公布時，下一張單曲《ER》的名稱和發行日期尚未公開。<ref name="oricon20120531" />}}附送的參加應徵劵，其中獲抽籤選中的人才能參加。<ref name="oricon20120531">{{cite web|url=http://www.oricon.co.jp/news/music/2012337/full/ |language=ja |title=<span lang="ja">関ジャニ∞、8月に8周年記念祭！　京セラドームに4万人招待</span> |publisher=[[Oricon|Oricon]] |date=2012-05-31 |accessdate=2013-02-09}}</ref>

關西傑尼斯8曾於2012年6月15日的《[[MUSIC_STATION|MUSIC STATION]]》演唱《就是愛。》。<ref>{{cite web|url=http://natalie.mu/music/news/70480 |language=ja |title=<span lang="ja">Mステにaiko、きゃりー、関ジャニ∞、柴咲、JUJU、BENI</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2012-06-04 |accessdate=2013-02-09}}</ref>

== 歌曲 ==
《就是愛。》是關西傑尼斯8成員[[錦戶亮|錦戶亮]]主演的[[TBS週四9時連續劇|TBS土九電視劇]]《[[爸爸偶像|爸爸偶像]]》的主題曲。<ref>{{cite web|url=http://natalie.mu/music/news/67983 |language=ja |title=<span lang="ja">関ジャニ∞新作シングルは錦戸ドラマ「パパドル！」主題歌</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2012-04-17 |accessdate=2013-02-09}}</ref>{{noteTag|1=關西傑尼斯8全體成員都有參與此劇演出，而且都是飾演本人。<ref>{{cite web|url=http://beauty.oricon.co.jp/news/2010237/full/ |language=ja |title=<span lang="ja">錦戸亮、優香と結婚!?　チャペルでドラマの製作会見</span> |publisher=[[Oricon|Oricon]] |date=2012-04-16 |accessdate=2013-02-09}}</ref>}}根據官方網站的描述，這是一首向在孤獨和不安圍繞的日子中給予自己曙光的人，表達不會動搖的「愛」的歌曲。<ref>{{cite web|url=http://www.teichiku.co.jp/artist/kan8/discography/TECI-832.html |language=ja |title=<span lang="ja">関ジャニ∞［愛でした。：TECI-831／TECI-832</span> |publisher=Imperial Records |date=2012 |accessdate=2013-02-09}}</ref>

== 排行榜成績 ==
此單曲發行首周的銷量約為26.2萬張，比上一張單曲《愛著你的一切》的約15.3萬張多了10.9張；也是關西傑尼斯8自其第15張單曲《[[LIFE～朝著眼前的方向～|LIFE～朝著眼前的方向～]]》（{{lang|ja|LIFE～目の前の向こうへ～}}）以來，相隔約1年10個月再次有單曲的首周銷量超過25萬張。<ref name="oricon" />此單曲取得該周[[Oricon公信榜|Oricon公信榜]]單曲周榜的首位。關西傑尼斯8憑此單曲，由第14張單曲《[[Wonderful_World!!|Wonderful World!!]]》開始連續7張單曲取得首位；出道至今則共有15張冠軍單曲。<ref name="oricon" />

此單曲於2012年的年終銷量約為32.0萬張，於Oricon公信榜單曲年榜排行第19名。<ref name="oricon yearly" />
