'''狭纹虎鲨'''（[[學名|學名]]''Heterodontus zebra''），又名'''斑紋異齒鮫'''、'''斑紋異齒鯊'''，俗名角鯊、虎沙，是[[軟骨魚綱|軟骨魚綱]][[虎鯊目|虎鯊目]][[虎鯊科|虎鯊科]][[虎鲨属|虎鲨属]]下的一个[[种|种]]。

== 分布 ==
本魚分布於西[[太平洋|太平洋]][[亞熱帶|亞熱帶]]介乎40°N至20°S的海域，包括中国大陆和[[台灣|台灣]]<ref>http://fishdb.sinica.edu.tw/chi/species.php?id=383115</ref>、[[日本|日本]]、[[韓國|韓國]]、[[印尼|印尼]]、[[菲律賓|菲律賓]]、[[越南|越南]]、[[泰國|泰國]]、[[馬來西亞|馬來西亞]]、[[澳洲|澳洲]][[昆士蘭|昆士蘭]]等。

== 深度 ==
水深50至200公尺。

== 特徵 ==
本魚頭大而鈍，眼大，口小，低的眶頂冠在[[眼睛|眼睛]]後逐漸傾斜，體側扁，略呈卵圓形。有[[背鰭|背鰭]]及[[臀鰭|臀鰭]]，淺淡色的身體上有窄而直的[[斑馬|斑馬]]狀斑汶。牠就像其他的[[虎鯊科|虎鯊科]]一樣，[[尾鰭|尾鰭]]有長的圓突，[[腹鰭|腹鰭]]較背鰭短，而[[脊椎|脊椎]]升起至尾鰭處。第一、二背鰭皆具硬棘，尾鰭具深缺刻。體長可達1.25公尺。

== 生態 ==
本魚主要生活於[[大陸棚|大陸棚]]海域，平時停棲於海底，游泳能力不佳，行動緩慢。屬肉食性，以小型底棲[[無脊椎動物|無脊椎動物]]為食。[[卵生|卵生]]的，卵大且具具螺旋形卵殼，四角另具短鬚狀突起。

== 經濟利用 ==
不常見，價值性不高，偶爾在水族館展示。其背鰭具硬棘。
大型魚種易經由食物鍊累積大量重金屬，導致神經系統病變，不宜食用。
