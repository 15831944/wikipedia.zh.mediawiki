'''枝江市'''位于[[中华人民共和国|中华人民共和国]][[湖北省|湖北省]]南部，是[[宜昌市|宜昌市]]代管的一个[[县級市|县級市]]。

枝江地处[[长江|长江]]中游北岸，[[江汉平原|江汉平原]]西缘，[[宜昌|宜昌]]与[[沙市|沙市]]之间，属[[长江三峡|三峡]]之末，[[荆江|荆江]]之首。西北部[[丘陵|丘陵]]、岗地占总面积的58.8%，东南部[[平原|平原]]占41.2%。

== 历史 ==
[[秦|秦]]时，因长江至此分枝而得名枝江，属[[南郡|南郡]]。汉置县，县治沮中，为南郡十八县之一。

1955年2月22日，并入宜都县。1962年10月20日，恢复枝江县，属湖北省宜昌地区。1992年3月，枝江属宜昌市。1996年7月30日，撤县设立枝江市。

== 区划 ==
枝江市下辖1个[[街道|街道]]、8个[[镇|镇]]、198个[[行政村|行政村]]。
*街道：马家店街道；
*镇：七星台镇、问安镇、董市镇、安福寺镇、白洋镇、百里洲镇、[[顾家店镇|顾家店镇]]、仙女镇。

== 交通 ==
枝江交通便利，[[长江|长江]]贯东而去，[[焦柳铁路|焦柳铁路]]穿市南下，汉宜高速铁路、[[枝江北站|枝江北站]]位于城区3公里外。
宜黄高速公路和[[318国道|318国道]]并行东西，[[三峡机场|三峡机场]]距市中心30公里。

== 工业与财政 ==
