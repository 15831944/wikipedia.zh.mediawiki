'''枝野幸男'''（{{jpn|j=枝野 幸男|hg=えだの ゆきお|rm=Edano Yukio|f=y}}，{{bd|1964年|5月31日|||Edano Yukio}}）是[[日本|日本]][[政治人物|政治人物]]，現任[[日本眾議院|日本眾議員]]（當選9屆）兼[[立憲民主黨_(日本)|立憲民主黨]]代表（黨魁）。日本[[東北大學_(日本)|東北大学]]法学部毕业。[[律師|律師]]出身，曾任[[內閣府特命擔當大臣（行政刷新擔當）|內閣府特命擔當大臣（行政刷新擔當）]]、[[民主黨_(1998年)|民主黨]]幹事長、民主黨政策調査會長、[[内阁官房长官|內閣官房長官]]兼內閣府特命擔當大臣（沖繩及北方對策擔當）、[[經濟產業大臣|經濟產業大臣]]等職。

== 個人簡歷 ==
枝野出身於[[宇都宮市|宇都宮市]]一個普通的[[上班族|受薪階級]]家庭<ref name="aera">AERA編集部 澤田晃宏「[https://dot.asahi.com/aera/2017102100006.html?page=1 女子高生も「エダノン」と叫ぶ　立憲民主・枝野幸男の変わらない政治の原点]」[[AERA|AERA]]2017年10月30日号 [[AERA_dot.|AERA dot.]]2017.10.21</ref>。祖父以日本「憲政之神」[[尾崎行雄|尾崎行雄]]為其命名，原本打算使用以[[平假名|平假名]]書寫的「{{lang|ja|ゆきお}}」，後來改為[[日語漢字|漢字]]書寫但讀音相同、而且「筆畫數會帶來好運」的「幸男」來命名<ref name="TheJournal2009">{{cite web
|author = 《THE JOURNAL》編集部
|date = 2009-12-12
|url = http://www.the-journal.jp/contents/politician/2009/12/19.html#more
|title = 政治家に訊く：枝野幸男
|publisher = THE JOURNAL
|accessdate = 2010-12-22
}}</ref><ref name="aera"/>。受到自己名字由來的影響，因而思考立下從政的志願<ref name="Hayano">{{cite book
|author = 早野透
|year = 2002
|title = 政治家の本棚
|publisher = 朝日新聞社
|pages = 439p
|isbn = 4-02-257746-0
}}</ref>。1982年考入[[東北大學_(日本)|東北大學]]法學部，同級生中有後來成為日本參議員的[[森雅子|森雅子]]。1988年通過司法考試，之後成為[[律師|律師]]<ref name="aera"/>。1993年代表[[日本新黨|日本新黨]]參選日本眾議員，於{{link-ja|埼玉縣第5區 (中選舉區)|埼玉県第5区 (中選挙区)|埼玉縣第5選區（舊制）}}當選，至此踏入政壇。

2011年1月14日，枝野就任[[內閣官房長官|內閣官房長官]]，以46歲7個月的年齡成為史上第二年輕的內閣官房長官。兩個月後，[[東日本大震災|東日本大震災]]發生，枝野因擔任內閣官房長官而需代表[[日本政府|日本政府]]發言、向媒體簡報救災狀況，曾連續105小時未曾闔眼，但仍能條理分明地進行發言工作，因此有日本網友於3月14日在[[Twitter|Twitter]]上寫「今晚8點半，我們大家一起懇請枝野先生去睡一覺吧」<ref>{{cite web|url=http://www.nfpeople.com/story_view.php?id=842|title=枝野的兩堂危機管理課|publisher=[[南方人物周刊|南方人物周刊]]|date=2011-04-18|accessdate=2013-11-16|archive-url=https://web.archive.org/web/20130826055817/http://www.nfpeople.com/story_view.php?id=842|archive-date=2013-08-26|dead-url=yes}}</ref>。

[[2015年|2015年]]1月19日，枝野應新當選的民主黨代表[[岡田克也|岡田克也]]邀請，出任民主黨幹事長。2016年，民主黨改組為[[民進黨_(日本)|民進黨]]，枝野續任為首任幹事長。

[[2017年|2017年]]10月2日，在民進黨決定於將到來的[[第48屆日本眾議院議員總選舉|第48屆日本眾議院議員總選舉]]與[[希望之黨|希望之黨]]合作的背景下，枝野宣布脫離民進黨成立[[立憲民主黨_(日本)|立憲民主黨]]，他表示成立新政黨的主要原因是與希望之黨理念及政策方向不同<ref>[http://www.cna.com.tw/news/firstnews/201710030260-1.aspx 日本立憲民主黨成立 眾院大選三分天下]</ref>；立憲民主黨之後在該場選舉拿下55席，成為最大的在野黨。
