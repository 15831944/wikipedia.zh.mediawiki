{{日本作品|數碼寶貝大冒險tri.|デジモンアドベンチャー tri.|Digimon Adventure tri.}} ，是由[[東映動畫|東映動畫]]企劃及制作的[[数码宝贝系列|数码宝贝系列]]OVA動畫，於2015年11月到2018年5月5日分為6章在剧场上映及网络播映。其中的“[[tri.|tri.]]”有「[[3|3]]」的意思，表示本作為電視動畫第一作《[[數碼寶貝大冒險|數碼寶貝大冒險]]》及第二作《[[數碼寶貝大冒險02|數碼寶貝大冒險02]]》的續篇第三作，同時亦為數碼寶貝大冒險15周年紀念計劃最重要的一部分。<ref>{{Cite Web|title=―デジモンアドベンチャー15th Anniversary Project― 8月1日お台場イベントで新ニュースを続々発表！ 「デジモンアドベンチャー新シリーズ」製作決定！！！|url=http://corp.toei-anim.co.jp/press/2014/08/15th_anniversary_project81.php|publisher=東映動畫|language=ja|date=2014年8月1日|accessdate=2015年5月22日|deadurl=yes|archiveurl=https://web.archive.org/web/20141009104904/http://corp.toei-anim.co.jp/press/2014/08/15th_anniversary_project81.php|archivedate=2014年10月9日}}</ref>

中国大陆[[爱奇艺|爱奇艺]]和香港[[Viu|Viu]]同步日本网络播映全6章，马来西亚由片商Lambaian Filem代理发行简体中文、英文、马来文字幕（日语原声）版本[[DVD|DVD]]和[[Blu-ray|Blu-ray]]。

曾經為本系列主唱眾多樂曲的歌手[[和田光司|和田光司]]先生，在本作第三章OVA製作期間，其官方事務所「SOLID VOX」於2016年4月8日發出聲明，指和田光司因[[鼻咽癌|鼻咽癌]]已於4月3日逝世，享年42歲，葬礼为近亲参加的密葬，本作第二章片尾曲「Seven～tri.Version～」 也成為生前最後的錄音作品。由於正值本作上映風潮期間，消息瞬間廣泛-{zh-cn:报道;zh-tw:報導;}-到世界各地，4月26日在惠比寿LIQUIDROOM举行了“和田光司 告别会”，官方[[東映動畫|東映動畫]]、本作[[聲優|聲優]]、音樂業界以及眾多和田光司的生前好友都致以哀悼，當中包括參與「Butter-Fly」的製作人千綿偉功及渡部チェル、太田美知彦、[[AiM|AiM]]、宮崎步、[[谷本貴義|谷本貴義]]等等。不少觀眾得悉後表現悲傷，但同時亦讚賞其奮鬥癌症13年的堅強意志力與生命力令人欽佩不已，以及感謝和田光司所帶來的童年回憶，將永遠緬懷他的音樂。而在《[[數碼寶貝大冒險|數碼寶貝大冒險]]》和《[[數碼寶貝大冒險02|數碼寶貝大冒險02]]》中配過[[武之內空|武之內空]]的聲優[[水谷優子|水谷優子]]女士和[[八神太一|八神太一]]的聲優[[藤田淑子|藤田淑子]]女士，先後皆因[[乳腺癌|乳腺癌]]而逝世。

== 故事簡介 ==
=== 第1章「再會」 ===
在被选召的孩子们首次接触数码世界的6年后、并且与贝利亚吸血魔兽战斗结束的3年后。2005年的一天，东京港区海面上的波浪在朝阳下摇曳。

17岁的高中生八神太一，如常地用手按停眼前的闹钟，踢开混乱的被铺，低声滴咕下伸展四肢。拉开了窗帘，光线照到桌上一幅“那一天”的集体照上。随后他骑上自行车出门，奔向那片初夏的青空。

不久，古加兽出现在人类世界，并开始进行破坏。亚古兽重新出现，并和伙伴们战胜了3只古加兽。原来数码世界的数码兽被感染，在人类世界寻找望月芽心和她的数码兽。

=== 第2章「決意」===
奧法獸事件過後，隨著時間流逝，御台場地區也漸漸復興著。

「回到日本之後，我就決定一定要去大大的澡堂泡澡！」

因為美美這句話，加上為了歡迎新加入的夥伴芽心和緬因貓獸，大家前往附近的溫泉主題公園。

雖然度過了一段開心的時光，但是這兒並沒有阿丈的身影。

面對大學聯考臨近的事實，他在聯考和「被選召的孩子們」的使命之間產生了煩惱。

「為什麼又必須是我們去面對啊？」
 
這個時候，在御台場再度出現了感染的数码兽、這是擁有著高傲靈魂的奧加獸。

而看著​​電視中奧加獸的暴走，美美和巴魯獸決定出擊。

「必須要讓人們知道，也有好的数码兽！」
 
然而，美美和巴魯獸無視光子郎而參與的戰鬥，卻造成了意想不到的結果。

各自怀揣煩惱的阿丈與美美。在月島綜合高中文化祭當天，一個男人出現在了他們面前。

=== 第3章「告白」 ===
缅因猫兽突然发生变异，杀害狮子兽，使得太一等人产生动摇。

缅因猫兽的感染，使得光子郎想寻找到原因。感染有什么征兆？是在什么时候呢？有什么对策吗？为了这一切的答案，光子郎努力的工作着。

望月芽心想要急切地知道原因，但是也没有什么有效的办法。芽心内心急切地希望缅因猫兽能够想起她，她相信追究缅因猫兽感染的原因是有必要的。但是谁也回答不了芽心的疑问。

为了防止其它数码兽感染，光子郎关上了自己办公室唯一的大门；但在其它地方，感染的迹象开始显现。同时，恆常性通过小光的声音把这个重大的秘密告诉了数码兽們。

之後，在與再度現身的緬因貓獸交戰之時，那個「時刻」到來了。

太一他们愤慨秘密被告知他人，在各自想法的交错中，他们有了自己的决定。

等待的话，总有一天人类会变成大人的。

=== 第4章「喪失」 ===
緬因貓獸的暴走，然後重啟(Reboot)的發生。

留下了苦惱著的芽心，太一等人向數碼世界前進了。

與他們再會的搭檔數碼獸們，失去了至今所有的記憶。

儘管如此，被選中的孩子們也要再建立起他們之間的羈絆，紛紛伸出了手，數碼獸們也相對作出回應。

然而，進化成成長期的比丘獸依然保持著警戒心…

「都說了以前的事情我都不知道啊！」

對凡事先人後己的空的善良反而不信任。

面對著受傷的空，太一与大和也無從以對安慰她。

到數碼世界後該怎麼辦一

一群人在討論當兒，緬因貓獸突然出現，過後就消失了。不知為何還保有原來的記憶，彷徨著含淚尋找著芽心一

看著那個身影，孩子們決定為拯救緬因貓獸而在數碼世界旅行。但是，他們的面前出現了帶領著黑暗四天王的那個男人。

這時，在現實世界的西島，接到了姬川失蹤的報告。

調查當兒，發現了姬川此前的行動，都隱藏了某個目的。

這與決定兩人命運的過去發生的事有關…

=== 第5章「共生」 ===
看到搭檔望月芽心被有着玄內樣貌的神秘男人所傷，緬因貓獸再度開始暴走。

「不能讓你誕生在世上…」

從現實世界消失的緬因貓獸，正是歪曲本身。是擁有過於巨大力量的，毀滅世界的鑰匙——

西島大吾以及出現在望月博士之處的黑客獸，逐漸解明真相。

保證世界調和的恆常性將現在變得過於強大的緬因貓獸視為危險，而要將其切除。

隨着暴走的緬因貓獸的出現，現實世界開始進入毀滅前的倒計時。

其所波及的地點接連發生異變。因歪曲而現身的數碼獸們，等待着那一時刻的來臨——

被選召的孩子們，受到異常的數碼世界排斥，返回現實世界，卻因為與搭檔數碼獸在一起而遭到人們的追趕。

眾人陷入孤立無援卻仍然拚命地尋求對策，與此同時芽心獨自一人持續煩惱着。

同伴和數碼獸的聲音，都傳達不到因背負過大的負擔而遭受打擊的她心中…

之後，殘酷的命運，甚至逼近了靈魂比任何人都率直纖細的八神光身邊——

=== 第6章「我們的未來」 ===
世界的崩溃开始了。

暴走的大缅因猫兽吸收了黑暗进化的神圣天女兽、

用强大的力量进化成了秩序兽。

根据世界树的想法，现实世界将被数码世界所吞噬。

在绝望迫近的时候，被选中的孩子们拼命地向前。

现在太一不在了，能代替他带队的只有大和。

「为了拯救世界，我们还有必须要做的事情！」

加布兽的话给大和带来了活力，而亚古兽则坚信太一定会回来。

另一方面，恒常性要处置失去控制的秩序兽、

并尝试施行干涉现实世界的最终计划。

为了阻止毁灭的到来，被选召的孩子们和他们的搭档拼命地战斗。

「所有的希望，都在缅因猫兽身上…」

绝望的迪路兽向大家说到。

孩子们做出最后决定的时候到了。

被选中的孩子们，自己将选择怎样的未来？

现在，冒险将再一次进化——

== 動畫製作 ==
=== 企劃 ===
: 本作企劃初期，雜誌《[[Fami通|Fami通]]》、《[[Animage|Animage]]》以「[[電視動畫|電視動畫]]」名義釋出有關情報，但其後修改為新系列字眼。本作曾經確定為全六章[[劇場版|劇場版]]，直到第四章上映前，官方最後定性為[[OVA|OVA]]作品、並分成數個版本模式上映：
* '''「網絡配信版」'''
: 為最早出現於網絡上發佈的版本，在日本本土網絡配信版則使用與劇場上映版相同版本，而日本以外地區的網絡配信版為每章剪輯成4-5話、以24分鐘為1話的TV版本，片頭、片尾畫面配上1分30秒的主題曲、結片尾曲，當主題曲播畢後，片中顯示「#01」、「#02」、「#03」、「#04」等若干標記，明確表示內容為分集第1-4話。
* '''「劇場上映版」'''
: 影院中播放的版本則沒有分集及刪去標記，以達到劇場版的流暢度；主題曲及片尾曲分別在開頭及結尾部分足本播放一次；片头片尾有「[[東映|東映]]」和「[[映倫|映倫]]」的标志。
* '''「BD/DVD」'''
: [[Blu-ray_Disc|Blu-ray Disc]]和[[DVD|DVD]]則使用日本本土網絡配信版以及劇場上映版相同版本。
: 若按照官方確定為全六章、每章分為4集長度推算的話，本作全篇原定極大可能相等於兩個季度的深夜電視動畫番組（約20-26話的半年番組長度），顯示製作單位臨時更改了發佈和製作形式，但官方則未曾為此等推論作任何解釋。然而日本全國只有10所院線上映的情況下，本作第1章票房收益達到2.3億日元的好成績，在小型影院票房中連續兩周排名第1名。由於反應熱烈、以及與劇場版同日發售的限定藍光光碟瞬間缺貨下，官方將原定剧场上映三周延長一周等加映措施。
: 製作組雖然以東映動畫名義邀請[[外包|外包]]公司「颱風Graphics」協力製作，但宏觀所有製作人員名單中、除企劃＆製作總監為東映動畫董事長＆社長外，絕大部分並非東映動畫內部員工，本作實質上是由東映動畫提供製作資源給外包公司「颱風Graphics」、並由動畫監督[[元永慶太郎|元永慶太郎]]邀請所相熟的業界人員主力製作，因此本作與人氣電視動畫《[[約會大作戰|約會大作戰]]》及劇場版《約會大作戰劇場版 萬由里審判》的製作人員極為相同<ref>{{Cite Web|title=約會大作戰製作組名單|url=http://date-a-live-anime.com/staff|publisher=}}</ref>。

=== 製作、設定 ===
==== 第1章「再會」 ====
: 本作使用全新的[[電腦繪圖作品|CG]]效果製作進化過程，相對歷代作的手繪作畫顯得現代化。但本作設定了數碼獸進化過程需要變回數碼蛋、由數碼蛋中變大及分裂後才能進化成功，而歷代第一作的內容中曾經強調數碼獸只有死亡後、或誕生前才能成為數碼蛋的設定條件。此後所有歷代系列作都有嚴格遵守這項設定，從未有主要角色的數碼獸進化期間透過數碼蛋進化成功的先例，因此本作的進化模式被指未有跟歷代系列作統一設定之一。

==== 第2章「決意」 ====
: 本篇章加入完全體及究極體進化版本，兩個進化版本中已沒有再出現數碼蛋元素，完全體改為由角色所屬的紋章穿過成熟期數碼寶貝的身體進化而成；究極體進化版本則直接使用普通光圈進化而成，較成熟期及完全體進化版本都欠缺特色。另外，製作數碼寶貝進化畫風由CG效果（成熟期）改回手繪作畫（完全體、究極體），令進化過程畫風未能統一之餘、亦令動畫幀數不足問題更加明顯。

==== 第4章「喪失」 ====
: 本作再次修改製作模式，東映動畫連同之前已播出的篇章一致修改作品字眼，由劇場版改為製作成本較低的[[OVA|OVA]]作品而並未有官方說明。合共本次作品字眼修改，本作已經是第三次修改製作模式（電視動畫→劇場版→OVA），綜合各種製作定位飄忽和外判製作導致作品質素低劣，可見東映動畫對本系列作為15周年紀念企劃並未受到應有的重視。絕大部分OVA作品只收錄於Blu-ray Disc當中，但本次字眼修改並未影響上映模式，第4章合共13所院線於日本上映；以及過往本作聲優＆歌手在特備節目、公開活動場合中多次稱呼本作為劇場版，所以坊間暫並未因此改動而轉變稱呼。

==== 第5章「共生」、第6章「我們的未來」 ====
: 印證上述前言的推算，本作全篇以26話、相當於兩季的半年番組長度完結，此製作決定導致全6章總共花了約兩年半的時間才播放完畢，原因除了動畫製作需時本身，亦要安排各日本院線的播放排期及上映日子，以及其他周邊商品如[[BD|BD]]及[[DVD|DVD]]光碟印製等，令到每個篇章需要相隔長達六個月或更長時間才能播出。

: 但充裕的製作時間則未能換來高水準的作品質素，除上述提及的作畫和美術品質外，角色刻劃、故事節奏亦相當敗筆，如第2章末端獅子獸被殺死後，在第3章開段，眾主角只顧調查緬因貓獸暴走原因、其角色數碼獸歡天喜地吃零食等冷血刻劃；第5章，現實世界危機將至，眾主角忽然在學校課室嬉戲說鬼故事等與主線無關的拖沓情節不斷在本作中發生，直到最後兩章才能清楚表達其主線的中心思想，因此大量伏筆到最後亦未能回收。

=== 音樂 ===
: 經歷歲月不饒人，本作在音樂方面，先後有兩位資深歌手及音樂創作人離世。[[和田光司|和田光司]]官方網站於2016年4月8日發出聲明，表示其因[[喉癌|喉癌]]已於4月3日病逝，其主題曲「Butter-Fly」成為和田光司出道作、也是最後的告別作；創作歷代背景音樂的作曲家[[有澤孝紀|有澤孝紀]]亦於2005年因[[膀胱癌|膀胱癌]]去世，後者已由創作《[[約會大作戰|約會大作戰]]》背景音樂的坂部剛代替。

==== 第1章「再會」 ====
: 本作將舊有的歌曲重新編曲，當中以主題曲「[[Butter-Fly|Butter-Fly]]」曲風變化最大，原因是[[和田光司|和田光司]]於2003年患上喉癌而失聲長達2年、繼而於2011年再次因為癌症復發全面停工。2013年復出後，聲帶及唱功水準只能以舒情曲風為主，因此本作的主題曲風格比舊有版本柔和，而當時已病重的和田光司從曲中副歌部分可見相當吃力，但仍聲嘶力竭地完成歌曲錄音更獲得觀眾高度評價。

: 本作亦有部分歷代背景音樂由坂部剛重新編曲，如收錄在1999年發售的OST《デジモンアドベンチャー 歌と音楽集 Ver.1》－曲目23「絕望」、曲目29「ハッピーエンド」、插曲「brave heart」的伴奏版等。

==== 第2章「決意」 ====
: 本章片尾曲改編自第一作插曲「Seven」，歌曲中的背景音樂雖然重新編曲，但當時和田光司再次因為癌症第三次復發而缺席歌曲錄製，新版本「Seven～tri.Version～」的人聲音源實際是抽取和田光司於2009年、出道10周年所發行的紀念版本「Seven ～10th MEMORIAL Version～」，並由宮崎步、AiM及編曲家渡部チェル加入和音元素作為新曲發售，因此「Seven～tri.Version～」並不是和田光司最後一首錄音作品，而是單曲發售作品，亦可見在這首歌當中，和田的聲線、歌唱技巧明顯比「Butter-Fly～tri. Version～」來得自然及響亮。

==== 第5章「共生」 ====
: 歷代作詞家山田ひろし、作曲／編曲家太田美知彦，再次為本系列創作歌曲「アイコトバ」。太田美知彦從第一作《[[數碼暴龍大冒險|數碼暴龍大冒險]]》開始直到第四作《[[數碼寶貝無限地帶|數碼寶貝無限地帶]]》包攬絕大多數主題曲及插曲、部分角色曲及片尾曲的作曲／編曲製作，歌曲旋律及風格於前四作一直處於高水準狀態，由他所負責的歌曲不但沒有因歲月流逝而被淡忘，反而更創下多首經典名曲，「brave heart」、「僕らのデジタルワールド」、「The Biggest Dreamer」、「3 Primary Colors」、「an Endless tale」等十多年來流傳至今，[[谷本貴義|谷本貴義]]的出道作「One Vision」亦是太田美知彦的作品。於《[[數碼暴龍拯救隊|數碼暴龍拯救隊]]》製作插入歌「Believer」後，近年則以音樂人及歌手身份活躍於各項現場音樂演出，因而淡出了本系列的音樂工作。

: 由早期被選召的孩子們持有的暴龍機。外型像個鬧鐘，事實上也有鬧鐘的功能，還能顯示附近同伴的方位。其主要作用是聯繫被選中的孩子和他們的搭檔，使他們心意相通，促使數碼寶貝從成長期進化為成熟期、超進化和究極進化。
; {{nihongo|D-3|Digimon Detect & Discover}}
: 由後期其中6名被選召的孩子們持有。除卻早期數碼佩章的功能，還能顯示附近同伴的方位。其可以將儲存在D-T中的數碼裝甲數據調出，併疊加在相應數碼寶貝上，進行不受邪惡力量箝制的武鬥進化。另外亦可通過兩台D-3，將兩隻相應數碼寶貝聯展進化為一隻數碼寶貝。
; {{nihongo|徽章|紋章|Emblems}}
: 被选召的孩子们心中特质以图案形式所表现出的东西。
; {{nihongo|数码终端器|ディーターミナル|D-Terminal}}
: 可以傳遞E－mail、儲存數碼元素（數碼裝甲）的機器，因為有這個機器的存在，一隻數碼寶貝才能進化成兩種不同的數碼寶貝。
; {{nihongo|笔记本型电脑|window pro}}
: 外型仿照iBook，八人中僅光子郎持有一台，被玄內改裝後，可和数码佩章連線讀取資料。
; {{nihongo|多功能护目镜|多機能式ゴーグル}}
: 光子郎发明的可以窥探现实世界失真的护目镜。
