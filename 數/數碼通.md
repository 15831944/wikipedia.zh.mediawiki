== 公司簡介 ==
'''數碼通電訊集團有限公司'''（{{sehk|0315}}）是一家在[[香港交易所|香港交易所]]上市的綜合企業公司，為香港四大電訊商之一。

載至2016年，在香港目前共有31間門市，其中8間位於香港島、11間位於九龍、12間位於新界。<ref>[http://www.smartone.com/tc/privileges_and_support/contact_us/store_location.jsp SmarTone> 聯絡我們> 分店地址 2016-07-26]</ref>

公司是[[新鴻基地產|新鴻基地產]]的附屬機構，在[[百慕大|百慕達]]註冊，[[總裁|總裁]]為[[葉安娜|葉安娜]]。2014/2015年度[[純利|純利]]為約9.35億港元<ref>{{Cite news|title=2014/2015年度之全年業績公佈|url=http://www.smartoneholdings.com/about/investor/major_announcements/2015/09/2015_09_01_437_chi.pdf|date=2015-09-01|publisher=數碼通}}</ref>。

== 歷史 ==
1992年由新鴻基地產及[[ABC佳訊|ABC佳訊]]組成，1993年3月開始提供[[GSM|GSM]]流動通訊服務，根據OFCA的「電訊里程碑」，為香港首間推出GSM制式的公共流動無線電話服務的通訊商<ref>[http://tel_archives.ofca.gov.hk/zh/telecom_fact/milestones/main.html 電訊管理局>電訊里程碑]</ref>，亦成為香港繼[[香港電訊CSL|香港電訊CSL]]、[[和記電訊|和記電訊]]及[[訊聯電信|訊聯電信]]後第四間流動通訊公司。1996年10月於香港聯合交易所有限公司上市。1998年3月收購[[八方通訊|八方通訊]]，以新品牌'''數-{}-碼通EXTRA'''推出[[PCS|PCS]]流動通訊服務；後合併為[[雙頻|雙頻]]網絡。

[[電訊|電訊]]服務為數-{}-碼通（SmarTone）現時的主要業務之一，為了爭奪[[3G|3G]]市場的佔有率，於2004年12月15日加入了的策略夥伴[[沃达丰|Vodafone]]，並由原來的名稱SmarTone 3G Limited 改為「SmarTone-Vodafone」提供市場獨有的流動通訊硬件。<ref>[https://www.smartone.com/about_us/media_centre/press_release/2004/12/2004_12_15_chi.pdf SmarTone 與 Vodafone 簽訂夥伴合作協議 ，2004年12月15日]</ref>直至2011年，數碼通決定不再與Vodafone延續市場營銷合作，且於同年12月恢復使用SmarTone 作為其商標。<ref>{{Cite news|title=數碼通及Vodafone雙方同意不再延續市場營銷合作|url=http://www.smartone.com/about_us/media_centre/press_release/2011/09/2011_09_19_337_chi.pdf|date=2011-09-19|publisher=數碼通}}</ref>

== 網絡發展和改變 ==

==== 2G ====
1993年3月，推出[[亞洲|亞洲]]首項[[GSM|GSM]]流動通訊服務

1993年12月，推出亞洲首個自動[[GSM|GSM]]漫遊服務

2002年12月，推出SmarTone iN!提供多媒體服務的入門網站

2012年8月28日，由於數碼通未能投到2600MHz[[無線電頻譜|無線電頻譜]]，故只能重整原先作為[[2G|2G]]用途的1800MHz[[無線電頻譜|無線電頻譜]]，改作[[4G|4G]][[LTE|LTE]]網絡，並在當日正式啟用。

2014年9月3日，數碼通在2013/2014年度之全年業績公佈時表示，已開始將[[2G|2G]]用途的900MHz[[無線電頻譜|無線電頻譜]]重整為[[4G|4G]][[LTE|LTE]]，以提供更佳的室內覆蓋，預計頻譜重整於同年年底完成。<ref>[http://www.smartone.com/tc/about_us/our_company/milestones/index.jsp SmarTone> 關於我們> 里程碑]</ref>

==== 3G ====
2001年9月，投得第三代流動通訊服務牌照<ref>[http://tel_archives.ofca.gov.hk/zh/3g-auction/progress/status.html  電訊管理局：第三代流動服務頻譜競投結果公佈]</ref>

2004年12月，以[[W-CDMA|W-CDMA]]制式，使用2100MHz[[無線電頻譜|無線電頻譜]]，推出[[3G|3G]]服務

2007年7月，成為全球首間公司提供[[3G|3G]]手機瀏覽網頁時串流收看[[Flash|Flash]]影片

2010年4月，推出28.8Mbps [[HSPA+|HSPA+]]網絡

2011年3月，投得850MHz[[無線電頻譜|無線電頻譜]]，並且加入[[3G|3G]]服務中<ref>[http://www.smartone.com/tc/about_us/our_company/milestones/index.jsp SmarTone> 關於我們> 里程碑]</ref>

==== 4G ====
2012年8月28日，數碼通正式啟動[[4G|4G]][[LTE|LTE]]服務，由網絡建基於4G LTE 1800MHz[[無線電頻譜|無線電頻譜]]，支援所有在香港出售的4G手機。於1800MHz[[無線電頻譜|無線電頻譜]]的無線電傳輸相比較高的頻譜提供更佳的室內覆蓋。<ref>[http://www.smartone.com/tc/about_us/our_company/milestones/index.jsp SmarTone> 關於我們> 里程碑]</ref>

9月12日，[[蘋果公司|蘋果]][[iPhone_5|iPhone 5]]發行，蘋果發佈會上點名數碼通成為香港第一間能支援iPhone 5以[[4G|4G]]上網的電訊商，數碼通也多次強調自己是「全港唯一可配合iPhone5提供4G網絡」。香港其他四大網絡供應商的4G網絡建基於多個頻譜（如1800MHz、2300MHz、2600MHz），以提供高速的室外覆蓋及較佳的室內滲透率。數碼通由於未能投到2600MHz頻譜，故只能清空原先作2G用途的1800MHz頻譜，改作4G LTE網絡，卻意外成為首家獲蘋果點名的香港4G上網服務電訊商。現時[[3香港|3香港]]、[[1010|1010]]、[[One2Free|One2Free]]及[[中國移動香港|中國移動香港]]的4G LTE已於較早時獲蘋果認證，能支援iPhone 5以其4G LTE網絡上網。

11月21日，數碼通於其[[Facebook|Facebook]]專頁宣佈4G LTE網絡已全面覆蓋[[港鐵|港鐵]][[港島綫|港島綫]]車站，並將分階段陸續擴展至其餘的港鐵站，預計2013年年初所有港鐵站及其連接隧道內將會有全面4G LTE覆蓋。<ref>[https://www.facebook.com/photo.php?fbid=10151232286603503&set=a.90706063502.84300.5950283502 數碼通 4G 網絡全面覆蓋港鐵港島綫各站]</ref>

2013年1月22日，數碼通完成在港鐵的[[香港地鐵公司|舊地鐵]]的七條綫路內（即港島綫、觀塘綫、荃灣綫、將軍澳綫、東涌綫、迪士尼綫以及機場快綫）鋪設4G LTE網絡。<ref>[http://chinese.engadget.com/2013/01/22/major-lte-provider-finish-mtr-coverage/ 香港五大LTE網絡宣佈打通港鐵主要路線，那非主要的呢？]，2013年1月22日</ref>同年3月20日成功競投港府新增的兩組4G 2.5/2.6GHz頻段的頻譜。透過新頻譜，數碼通將可擴大其數據容量及透過頻段之整合以改善最高數據速度。

11月8日，數碼通公佈其[[4G|4G]]網絡將覆蓋舊九鐵網絡（即現時的東鐵綫、西鐵綫及馬鞍山綫）及落馬洲支線。<ref>[http://chinese.engadget.com/2013/11/10/3hk-smartone-4g-lte-mtr-hk/ 3HK 與 SmarTone 也宣佈延伸其 4G LTE 網絡覆蓋了]，2013年11月10日</ref><ref>[http://www.smartone.com/about_us/media_centre/press_release/2013/11/2013_11_12_366_chi.pdf  SmarTone 進一步擴展港鐵內 4G LTE 網絡覆蓋]</ref>

2014年8月7日，數碼通推出[[VoLTE|VoLTE]]話音服務，但只支援部分電話型號。

9月3日，數碼通在2013/2014年度之全年業績公佈時表示，已開始將2G 900MHz頻譜重整為4G LTE，以提供更佳的室內覆蓋，預計頻譜重整於同年年底完成。

數碼通有計劃於未來透過結合基站加密(cell densification)、測試中的小覆蓋基站科技(small cell technologies)、[[LTE_Advanced|LTE Advanced]]、重整現有頻譜(refarming)及獲取額外頻譜等方式應付進一步的需求。

2016年10月23日，數碼通公布其4G覆蓋伸延至全新觀塘綫延綫的黃埔站及何文田站。<ref>[http://topick.hket.com/article/1525444/%E6%95%B8%E7%A2%BC%E9%80%9A%E7%8D%B2%E9%82%80%E7%AE%A1%E7%90%86%E6%B8%AF%E9%90%B5%E6%96%B0%E7%B7%9A%E7%B6%9C%E5%90%88%E7%84%A1%E7%B7%9A%E9%80%9A%E4%BF%A1%E7%B3%BB%E7%B5%B1 數碼通獲邀管理港鐵新線綜合無線通信系統]，2016年10月23日</ref>

12月28日，SmarTone 宣佈 4.5G 網絡覆蓋伸延至全新南港島綫，包括海洋公園站、黃竹坑站、利東站及海怡半島站及於港鐵新增 2100MHz 頻譜。<ref>[http://www.smartone.com/about_us/media_centre/press_release/2016/12/2016_12_28_390_chi.pdf]，2016年12月28日</ref> <ref>[http://www.mobilesetting.com.hk/2016/12/%E6%96%B0%E5%A2%9E-2100mhz-%E9%A0%BB%E8%AD%9C%EF%BC%81%E6%B8%AF%E9%90%B5%E5%8D%97%E6%B8%AF%E5%B3%B6-4-5g-%E5%85%A8%E8%A6%86%E8%93%8B.html]</ref>

== 網絡故障事件 ==
2012年4月9日清晨，數碼通網絡系統局部發生故障8個小時，使港九新界多個地區多達162萬用戶受影響，當日清晨6時至下午2時無法以手機通話及上網，數碼通被批評沒即時公佈。數碼通解釋，其中一座機樓電力供應故障，後備電源操作數小時後也故障，令通訊服務受阻。

事後數碼通向用戶致歉，表示正商討補償方案，有受影響的用戶聲稱，已經成功爭取免一天月費。<ref>[http://etnet.com.hk/www/tc/news/home_categorized_news_detail.php?newsid=ETN220410517 數碼通（３１５）機樓故障致網絡癱瘓，向用戶致歉並探討補償方案]</ref>前任數碼通總裁黎大鈞表示已經做好保護措施，又要求其他電訊商亦要效法數碼通。
