'''數理資優班'''，有時在媒體中簡稱'''數資班'''<ref>參考：{{Cite web|url=http://acad.tcfsh.tc.edu.tw/zh_tw/sped/MathematicalGifted/MG_past_exam|title=數資班歷屆考題|accessdate=2019-02-17|author=|date=|publisher=臺中市立第一高級中等學校|archive-url=https://web.archive.org/web/20160420060824/http://acad.tcfsh.tc.edu.tw/zh_tw/sped/MathematicalGifted/MG_past_exam|archive-date=2016-04-20|dead-url=no}}、{{Cite web|url=https://forum.babyhome.com.tw/topic/3923978|title=請問新竹科學園區實驗高中數資班的上課情況?? 謝謝|accessdate=2019-02-17|author=|date=|publisher=BabyHome 寶貝家庭親子論壇}}</ref>，是[[中華民國|中華民國]][[中華民國教育部|教育部]]開設的一種分組教學[[資優|資優教學]]計畫<ref>{{Cite web|url=http://excellent.set.edu.tw/index3-11.asp|title=資優教育資源網|accessdate=2017-07-16|author=|date=|work=excellent.set.edu.tw|publisher=中華民國教育部|archive-url=https://web.archive.org/web/20170719165024/http://excellent.set.edu.tw/index3-11.asp|archive-date=2017-07-19|dead-url=yes}}</ref>，是[[學術性向資賦優異班|學術性向資賦優異班級]]的一種。

法規上，數理資優班的設立、管理由[[教育部國民及學前教育署|國教署]]負責，但實際教學計畫大致上由各學校自由辦理。在臺灣，數理資優班的甄試季節為每年七月，每每引起媒體的大幅關注。即便如此，台灣政府在特殊教育方面著重於障礙教育，因此在學術資優班的著墨上其實較少<ref name=":0">{{Cite journal|title=臺灣資優教育四十年（一）：回首前塵|author=吳武典|url=|journal=資優教育季刊|issue=126|doi=10.6218/GEQ.2013.126.01-11|others=|year=2013|volume=|page=1-11|pmid=}}</ref>。

== 法源依據 ==
依據特殊教育法及身心障礙及資賦優異學生鑑定辦法 ，學術性向資賦優異是指在[[语文学|語文]]、[[数学|數學]]、[[社会科学|社會科學]]或[[自然科学|自然科學]]等學術領域，較同年齡者具有卓越潛能或傑出表現者。其表現並由下列四個方式判斷<ref>{{Cite web|url=http://edu.law.moe.gov.tw/LawContentDetails.aspx?id=FL009187|title=身心障礙及資賦優異學生鑑定辦法|accessdate=2017-07-16|author=|date=2013-09-02|work=edu.law.moe.gov.tw|publisher=|archive-url=https://web.archive.org/web/20190112155415/http://edu.law.moe.gov.tw/LawContentDetails.aspx?id=FL009187|archive-date=2019-01-12|dead-url=no}}</ref>：

# 在任一領域測驗得分在[[平均数|平均數]]+2[[標準差|標準差]]或[[百分位数|PR]]97以上，並經專家、老師推薦。
# 參加政府機關或學術研究機構舉辦之國際性或全國性（指中華民國）有關競賽或展覽表現優異，獲前三等獎項。
# 參與學術研究單位長期輔導之有關學科研習活動，成就優異，經主辦單位推薦。
# 獨立研究成果優異並刊載於學術刊物，經專家學者或指導教師推薦。

依照資優教育法，資優生可要求學習超前一年的課程。因此許多不須經由考試入學的高中生會選擇在高三時先選擇大學的課程。
