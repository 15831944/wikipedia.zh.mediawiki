'''數學科延伸部分'''，又称'''數學科延伸單元'''、'''數學科延伸课程'''、'''數延'''、'''M1M2'''，是[[香港中學文憑試|香港中學文憑試]]数学科的选修部分。數學科延伸部分不被視為獨立科目，地位低於普通選修科，一些理科大專課程會計算延伸部分之成績，醫學、法律等學系則不會獨立計算<ref>中大醫科不計算，港大醫科合併M1/M2與數學算作一科。{{Cite |url= http://www.beagazine.com.hk/hot-topics/人人都想讀醫-你估人人都入到醫咩/ |title= 人人都想讀醫 你估人人都入到醫咩？ |publisher= 遵理BEAGAZINE |date= 2017-11-11 }}</ref>。

數學延伸部份分為單元一（M1，微積分與統計）及單元二（M2，代數與微積分）。文憑試考生可以選擇不修讀任何延伸部分、只修讀單元一或只修讀單元二，不可以同時修讀單元一和單元二。

== 课程设计及内容 ==
在[[香港教育|香港教育]]制度，數學科是高中核心科目，分为「必修部分」、「單元一：微積分與統計」和「單元二：代數與微積分」<ref name="曾鈺成"/>。必修部分的設立原意是供將來不打算攻讀對數學水平有要求的學系、或是數學能力較低的高中學生修讀，程度屬基礎，強制所有高中學生學習和應考；單元一和單元二屬於延伸部分，設立原意是供需要掌握更多數學知識的學生修讀，程度比必修部分深奧<ref name="曾鈺成"/>。

單元一对应[[三三四高中教育改革|三三四高中教育改革]]前[[香港中學會考|香港中學會考]]的[[附加數學|附加數學]]，以及[[香港高級程度會考|香港高級程度會考]]的應用數學科（但不包括該科的主要部分[[牛頓力學|牛頓力學]]）、香港補充高級程度數學與統計學科<ref name="學界憂削弱港生數學能力"/>。这个单元適合將來進修或工作時需要更多數學知識的高中學生，課程分為「基礎知識」、「微積分」、「統計」三個領域，圍繞[[應用數學|應用數學]]，強調數學的應用性而非嚴謹性，偏重方法而非理論<ref name="曾鈺成"/><ref name="hkasme"/><ref name="curriculum"/>。具體課題有<ref name="curriculum"/>：

== 考评方式 ==
在香港中學文憑試，应考數學科延伸部分单元一或单元二的考生均需要完成一份试卷，作答八至十二道短題目、三至五道長題目（不设选答题），限时兩小時三十分鐘，考试时可以使用[[电子计算器|电子计算器]]<ref name="hkeaa"/>。數學科延伸部分并不是一个独立科目，但在文憑試会独立評級，考生的文憑試成績表上会同时印有數學科等級和延伸部分等級<ref name="學界憂削弱港生數學能力"/><ref name="葉建源"/>。

數學科延伸部分的评等方法如同其他科目，等级分为5**（最高等级）、5*、5、4、3、2、1（最低等级）、U（不予评级）。

== 修读情况 ==
2011/12学年，香港分别有81%和77%的中学为高中学生提供延伸部分单元一和单元二；2015/16学年，只有59%的中学提供单元一课程，70%的中学提供单元二课程<ref name="ashk"/>。有些中学顾虑到学习數學科延伸部分较为吃力，因此引导高中学生专注于必修部分，以节省资源与时间<ref name="ashk"/>。在首届文凭试（2012年），有22.9%的考生修读數學科延伸部分；到了2016年，只有13.9%的考生修读延伸部分，这个比率较不少已发展地区为低<ref name="ashk"/>。

數學科延伸部分的退修情況显著，比很多传统高中学科还要严重；在2015/16学年，近三千名学生在升上中五（高中二年级）后退修延伸部分，超过原先修读人数的一半，当中以单元一的退修人数较高<ref name="星島日報"/>。

== 与本科收生的关系 ==
數學科延伸部分是为计划在大专修讀工程、科學或商业相关學科的学生而设，选择单元一的学生通常有意攻读統計或商科，选择单元二的学生则通常有意攻读工程、[[精算|精算]]或理科<ref name="edb"/><ref name="星島日報"/>。

在首届文凭试（2012年），香港八所公立大學（简称“八大”）的200多个课程当中，只有6个课程明确把數學科延伸部分视为特定选修科、9个课程明确把延伸部分列为收生要求，亦有工程科系对修读延伸部分的考生给予加分或优先考虑，一些科系则含糊地表示会对有修读的考生“给予优势”，當中[[香港科技大學|香港科技大學]]最先視M1M2為平等選修科<ref name="新高中課程欠規劃 M1M2退修率7成"/>。到了2016年，八所公立大學当中，有四所大学在判定学生是否符合入學要求时会把數學科延伸部分视为選修科目，其他大學只在个别本科课程收生时承認數學科延伸部分<ref name="ashk"/>。此外，不同大专院校及科系考虑數學科延伸部分的办法各有不同，例如：
* 有些科系会建议（但不硬性要求）修读延伸部分；有的把修读延伸部分列为硬性的附带要求；有的优先考虑有修读的考生<ref name="部分大學課程對M1及M2取態"/>。
* 有些科系会在计算入学分数时提高數學科延伸部分的占分比重<ref name="ashk"/>。
* 有些大学会视乎考生在其他科目的成绩，如果一名考生在數學科延伸部分的成绩比他最佳的选修科目成绩还要高，就会把延伸部分纳入计算<ref name="fss"/>。
* [[香港大學李嘉誠醫學院|香港大學李嘉誠醫學院]]会视乎考生在数学必修部分的成绩，如果一名考生在延伸部分的成绩比必修部分更好，就会纳入计算<ref name="醫學院"/>。

香港以外的大学也会考虑香港学生有否修读數學科延伸部分，所以有些学生即使在文凭试考获优秀成绩，却可能因为没有修读延伸部分而不被海外知名大学取录<ref name="港生不讀延伸數學 海外名校或拒收"/>。有些海外大学的个别科系要求香港学生必须在延伸部分取得一定等级，才会考虑收生；有些大学则只把數學科必修部分视为半个科目的程度，即使考生在必修部分得到最高等级，亦会被换算成较低分数<ref name="港生不讀延伸數學 海外名校或拒收"/>。在[[英國大學及院校招生事務處|英國大學及院校招生事務處]]的{{link-en|英國大學及院校招生事務處分數對照制度|UCAS Tariff|分數對照制度}}（修订于2017年9月），數學科必修部分、延伸部分各被视为半个数学科，如果考生在數學科必修部分取得某一等级，所得分数-{只}-相当于其他科目同一等级所得分数的一半；因此，如果文凭试考生没有修读延伸部分，即使在必修部分取得最高等级，也只会被换算为其他科目最高等级的一半分数，比其他科目的第四等级还要低<ref name="UCAS"/>。

== 評論 ==
=== 認同===
[[香港中文大學|香港中文大學]]教授區國強把數學科必修与延伸部分的设计形容为“十分聰明的安排”，认为这样可以为普通学生提供基本的数学训练，同时也提供进深的数学课程；但是，他批评学生不能同时选修单元一与单元二，因此无法同时学习统计与代数，有欠均衡<ref name="hkasme"/>。

=== 反對地位降低 ===
數學科延伸部分在香港教育界備受爭議，[[香港科學院|香港科學院]]等批評者認為此科地位低於昔日之[[附加數學|附加數學]]，不吸引高中學生，课程安排也有不当，令修读人数甚少，影响大學新生之数学程度，從以影響科研人才培訓<ref>{{Cite |url= https://m.mingpao.com/ins/instantnews/web_tc/article/20180108/s00001/1515392145739 |title= DSE數學M1M2報考人數跌　港大教授倡院校調整比重 |publisher= 明報 |date= 2018-01-08 }}</ref>。香港科學院在2016年12月發表《科學、科技和數學教育與香港創新科技的發展》報告，批評香港高中學制輕視[[高等數學|高等數學]]，造成[[STEM教育|STEM教育]]斷層；報告提出，大學收生時大多不考慮數學科延伸部分，令高中學生缺乏誘因修讀，加上課程困難、課時不足，造成少人修讀的情況<ref name="ashk"/>。

[[教育評議會|教育評議會]]創會會員、科学教师黃冬柏认为，在新高中的数学教育情况下，高中物理科、经济科、资讯及通讯科技科等科目的教学会因学生数学水平下降而出现困难，而且很多高中毕业生的数学能力不足，日后在本科学习时面对巨大学习难题<ref name="master"/>。前任[[香港立法會|香港立法會]]主席、前数学教师[[曾鈺成|曾鈺成]]认为，就培养STEM（科學、技術、工程、數學）人才而言，香港高中学生選修数学与理科的情况令人担忧<ref name="曾鈺成"/>。香港中文大學工程學院高层指出，不修读數學科延伸部分的高中毕业生难以应付工程科系所需，导致工程學院要为约五分之一新生补救高等数学基础<ref name="兩成工程新生「補底」追「數」"/>。
