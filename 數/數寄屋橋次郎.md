'''數寄屋橋次郎'''（{{lang-ja|すきやばし次郎}}）是由日本[[壽司|壽司]]師傅[[小野二郎|小野二郎]]開設在[[東京都|東京都]][[銀座|銀座]]一帶的壽司店。

== 歷史 ==
店舖於1965年成立，位置是在大樓的地下一樓裡<ref>{{cite web |url=http://www.nikkansports.com/general/news/p-gn-tp3-20140424-1289971.html |script-title=ja:オバマ大統領「人生で一番おいしいすし」 |work=[[日刊體育|日刊體育]] |language=ja |date=2014年4月24日 |accessdate=2016年12月10日 |deadurl=yes |archiveurl=https://web.archive.org/web/20140902050049/http://www.nikkansports.com/general/news/p-gn-tp3-20140424-1289971.html |archivedate=2014年9月2日 }}</ref>。在1994年曾被《國際先驅論壇報》選為全球最佳餐廳第6名<ref>{{cite web |url=http://jiro-movie.com/cast |script-title=ja:キャスト・スタッフ |work=Transformer.Inc |language=ja |accessdate=2016年12月10日 |archive-url=https://web.archive.org/web/20160126193340/http://jiro-movie.com/cast |archive-date=2016年1月26日 |dead-url=yes }}</ref>，之後在2007年獲得《[[米其林指南|米其林指南]]》的三顆星評價後；創下連續10年拿下該評比的成績<ref>{{cite web |url=http://www.sankei.com/life/news/161129/lif1611290039-n1.html |script-title=ja:ラーメン店「鳴龍」が一つ星、「すきやばし次郎」など４店が１０年連続三つ星　ミシュランガイド東京２０１７発表 |work=[[產經新聞|產經新聞]] |language=ja |date=2016年11月29日 |accessdate=2016年12月10日}}</ref>。

在小野二郎年齡漸增後，已慢慢將店舖轉交給長子小野禎一經營<ref>{{cite web |url=http://style.nikkei.com/article/DGXZZO10066880Z21C16A1000000?n_cid=TPRN0016 |script-title=ja:「三十星」が４軒　ミシュランガイド東京、１０周年で |work=NIKKEI STYLE |author=渡邊智哉 |language=ja |date=2016年11月29日 |accessdate=2016年12月10日}}</ref>。

== 裝潢 ==
店鋪在門外有擺置著陶製的杯具作爲裝飾，其中某個為小野二郎親自捏造的作品<ref>{{cite book |title=《壽司之神全技法：小野二郎的壽司聖經》 |publisher=城邦文化 |author=小野二郎 |location=台灣 |isbn=978-986-568087-9  |date=2015年2月1日 |chapter=〈第四章 簡潔肅然的空間〉|page=137頁|language=zh-tw}}</ref>。在進入門內的左側為料理吧檯，其位置是為了配合小野二郎本人為[[左撇子|左撇子]]的因素<ref>{{cite book |title=《壽司之神全技法：小野二郎的壽司聖經》 |publisher=城邦文化 |author=小野二郎 |location=台灣 |isbn=978-986-568087-9  |date=2015年2月1日 |chapter=〈第四章 簡潔肅然的空間〉|page=131頁|language=zh-tw}}</ref>。

室內在牆上會有為了配合不同季節而間隔更換的畫作和掛簾，其中以[[藍染|藍染]]所製作的掛簾則是店舖成立時期便已使用<ref>{{cite book |title=《壽司之神全技法：小野二郎的壽司聖經》 |publisher=城邦文化 |author=小野二郎 |location=台灣 |isbn=978-986-568087-9  |date=2015年2月1日 |chapter=〈第四章 簡潔肅然的空間〉|page=135頁|language=zh-tw}}</ref><ref>{{cite book |title=《壽司之神全技法：小野二郎的壽司聖經》 |publisher=城邦文化 |author=小野二郎 |location=台灣 |isbn=978-986-568087-9  |date=2015年2月1日 |chapter=〈第四章 簡潔肅然的空間〉|page=138頁|language=zh-tw}}</ref>。

== 分店 ==
數寄屋橋次郎的分店在[[六本木新城|六本木新城]]裡開設<ref>{{cite web |url=http://www.roppongihills.com/shops_restaurants/restaurants/00138.html |script-title=ja:鮨 すきやばし 次郎 |work=[[六本木新城|六本木新城]] |language=ja |accessdate=2016年12月10日}}</ref>，由小野二郎的次子所經營，有獲得過《米其林指南》兩顆星的評價<ref>{{cite web |url=http://www.bs-tbs.co.jp/green/bn150.html |script-title=ja:寿司とゴルフに大事なこと　それは"握り"と"リズム" |work=[[TBS電視台|TBS電視台]] |language=ja |date=2013年3月2日 |accessdate=2016年12月10日}}</ref>。

== 電影場景 ==
2011年上映的紀錄片《[[壽司之神|壽司之神]]》以數寄屋橋次郎作為場景進行拍攝，由美國導演大衛·賈柏執導。該片講述全球最年長米其林三星餐廳數寄屋橋次郎大廚小野二郎的傳奇人生和成功的故事<ref>{{cite web |url=http://travel.163.com/12/1210/16/8ICIUUQ800063KE8.html |title=走近寿司之神小野二郎 |accessdate=2012 |publisher=网易 |work= |language=zh-hans}}</ref>。

== 其它 ==
2013年6月24日店內曾發生一名壽司師傅使用稻草燻[[鰹魚|鰹魚]]後，在將稻草放回儲藏室時不小心引發火災的意外，但之後並無人傷亡<ref>{{cite web |url=https://www.japantoday.com/category/national/view/fire-breaks-out-in-ginza-building-with-famous-sushi-restaurant |title=Fire breaks out in Ginza building with famous sushi restaurant |work=Japan Today |language=en |date=2013年6月24日 |accessdate=2016年12月10日}}</ref>。
