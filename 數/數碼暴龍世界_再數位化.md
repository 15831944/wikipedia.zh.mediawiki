== 概要 ==
{{模板:Video games}}
=== 迎来2012年数码兽系列15周年 ===
数码兽系列以1997年登场的液晶携带游戏为首，在电玩、集换卡片游戏、模型、漫画等多领域跨平台展开。1999年，动画《数码兽大冒险》开始放映，在海内外引起较大反响成长为人气系列。2012年，数码兽迎来了15周年。

=== 数码兽世界 ===
1999年，以育成RPG在PlayStation平台推出的初代《数码兽世界》是一款育成具有寿命数码兽的冒险游戏。本作品以其独特的育成系统让众多玩家入迷，并获得了很高的人气。加上动画放映的效果，创造了日本国内40万累计160万的销售记录。直到现在初代数码兽世界的游戏系统依然受到很高的评价。<br />
然后时隔13年，继承初代游戏的「数码兽世界」复活了！

=== 「前作+新作」游戏容量大幅提升的「复原 解码」 ===
作为新的冒险，追加了人气X抗体数码兽活跃的「恸哭的X抗体篇」、七大魔王暗中活跃的「谋划的魔王篇」两个剧本。当然前作「亚生命体缺陷篇」依然可以玩，而且支线任务大幅增加，游戏容量进化为前作两倍以上。<br />
追加大量人气数码兽，可育成数码兽达到160体，包含BOSS数码兽在内登场数码兽总数达200体以上。<br />
调整了游戏平衡，使新玩家更加容易上手。而且3DS的双屏幕使得界面焕然一新。搭载其他各种新系统，竞速通关要素充实。

== 剧情 ==
=== 亚生命体缺陷篇 ===
大型企业GIGO公司运营的大型游戏《数码怪兽》，这是一款孕育数码生命体「数码兽」并让他们战斗的网络游戏。<br />
主人公也是沉迷于游戏的一位玩家。某日，像平日里一样他骑着自行车回家，路过的风景就像司空见惯的现实世界的日常景象。<br />
在到达自家公寓乘坐电梯时，像往日一样启动手机游戏《数码怪兽》，检查亚古兽的育成情况。手机画面的另一头是数码兽们所生活的数码世界。<br />
与朋友们一起登陆游戏，收到了一封邮件。因将所记载的暗号解明，而误闯入异世界数码世界。在那里遇到的公公兽向主人公委托「让出门的数码兽返回创始村」，于是他一边协助一边寻找回到原来世界的线索。但是，在旅途中遭遇到被称为黑化现象的受控数码兽们，以及谜之生命体缺陷。不久，主人公就与伙伴们为了拯救世界而开始战斗了。

=== 恸哭的X抗体篇 ===
3DS「复原 解码」追加的剧本。<br />
因为人工生命体的失控，数码世界变得不安定。因此为了恢复世界的安定，数码世界的主计算机「世界树」活性化！随后世界树仿真被GIGO公司封印的无限山，为净化数码世界开始向数码世界散布强制删除异份子的「X程序」。因从无限山泄露的X病毒，镇上发生了数码兽消失事件。为了阻止突如其来的惨剧，主人公一行一边协助普罗特兽和多路兽，一边开始探索无限山。

=== 谋划的魔王篇 ===
3DS「复原 解码」追加的剧本。<br />
击破缺陷后迎来了片刻的休息，然而在现实世界看到这般景象的四之宫莉娜感觉异变仍未结束。这时，魔王巴尔巴兽制作的程序运行，莉娜被吸入了使用的终端，被带到了数码世界。然后巴尔巴兽在主人公一行面前出现，在数码之塔的最顶层宣布支配数码世界。被抓住的莉娜的存在也水落石出，少年们决心阻止巴尔巴兽的野心！

== 登場角色 ==
; {{nihongo|大河|タイガ|[[声優|声]]：[[白石涼子|白石涼子]]}}
: 默认名字与漫画版的名字叫「大河（{{lang|ja|タイガ}}，Taiga）」，在游戏中可更改。
: 本作的主人公，喜欢骑折叠式自行车的中学生。迷上了网络游戏「[[数码怪兽|数码怪兽]]」，是排名第三的驯兽师。特别喜爱数码兽，被一封邮件引入数码世界。
: 最初遇到的搭档数码兽是亚古兽，天真无邪和蔼可亲，在数码世界与主人公一起战斗。
; {{nihongo|尼可莱·耶科维奇·佩特洛夫|ニコライ·ヤーコヴィッチ·ペトロフ|声：[[日野聰|日野聰]]}}
: 主人公的挚友，与大河在现实世界的住所很近，交情很深。昵称「尼克」。单纯直率的性格，具有强烈正义感的少年，偶尔会不顾周围的情况失控。
: 在游戏「数码怪兽」中是排名第四的驯兽师，与主人公是对手玩家。父亲是俄罗斯人，在运营数码兽游戏的GIGO公司工作。尼克对父亲的事是发自内心仰慕的。
: 搭档数码兽是名为「莎夏」的加奥兽，沉默寡言，性格比较克制。与尼克在恰似友情的搭档关系中向强大的数码兽进化。
; {{nihongo|铃童明步|鈴童アキホ|声：[[佐藤聰美|佐藤聰美]]}}
: 主人公的好友。开朗好奇心旺盛，喜欢开心的事情，是个比任何人都相信与数码兽之间羁绊的少女。很会照顾人，也时常帮助在数码世界遇到困难的数码兽们。与数码兽们的相性很好，主人公问不出的情报她也能问到。
: 真是身份是网络游戏「数码怪兽」排名第一的「喵咪驯兽师」（{{lang|ja|にゃんこテイマー}}）。借由对数码兽的爱能无限引出拍档力量，天生的数码兽驯兽师。—到现在仍未能适应当下的现实生活，反而导致她更加投入「数码怪兽」。
: 搭档数码兽是名为「数码鸟琳（{{lang|ja|デジトリん}}）」的比丘兽，性格心地善良，与明步一起帮助有困难的数码兽。结合明步的爱情向强大的数码兽进化。 
; {{nihongo|久我裕也|久我ユウヤ|声：[[甲斐田幸|甲斐田幸]]}}
: 经常在「数码怪兽」游戏排行榜中保持上位，操纵强大数码兽的少年。被灌输精英主义，有着强烈及敏锐意识的数码兽驯兽师。经常言谈柔和，总是笑嘻嘻的表情，好像为了自身的某个目的而探索数码世界。对自己的能力、意见有着绝对的自信，是个能力主义明确的成熟少年。
: 搭档数码兽名为「小黑（Black）」，类型是谓之黑暗暴龙兽X抗体的特殊事物，犹如隐藏在裕也内心的凶暴性的实体化。这是只有使用GIGO的后门将各种稀有数据结合起来才可能的进化，稀有度非常高。所以裕也平常对战时不使用这个搭档，而是用别的账户进行游戏。
; {{nihongo|御神樂米蕾|御神楽ミレイ|声：[[澤城美雪|澤城美雪]]}}
: 圣与暗，带着相反属性数码兽的神秘少女。在数码世界与现实中自由阔步，精明的黑客/程序员。给人以冷静且冰冷的印象。有着强悍气质的决心。好像知道向数码世界逼近的危机的真相。将主人公等人召唤到数码世界也是她，其程序构建的能力凌驾于专业人士之上。
: 搭档数码兽是清廉洁白的天使型数码兽天女兽和高纯度的黑暗中的邪恶化身女恶魔兽。美玲进行特殊的编码操纵拥有了两体搭档。本来以这样的形式拥有搭档被认为是不可能的，但拥有高超编程设计能力的美玲将这个化为了可能。
; {{nihongo|四之宫莉娜|四ノ宫リナ|声：[[户松遥|户松遥]]}}
: 3DS版《数码宝贝世界 复原 解码》中追加的主要角色。平时很大方，经常调节周围气氛，不过内心却很紧张，是个温柔的少女。对扭曲的事情或虐待弱者等行为态度毅然。其实有点喜欢可爱的东西。隐藏着强烈的信念。
: 搭档数码兽是V仔兽（究极体是究极V龙兽）。

: 拍擋数码兽為鼻涕獸。
: 是[[铁拳系列|铁拳]]中客串演出的角色。

: 莉莉的執事。
: 拍擋主要是天使系的数码兽。

《数码宝贝世界 复原 编码》（{{lang|ja|デジモンワールド リ：デジタイズ エンコード}}、Digimon World Re:Digitize Encode），[[V_Jump|V Jump]] 2013年6月号（2013年4月23日发售）开始连载至2014年2月号（2013年12月21日发售），从2014年3月号（2014年1月21日）开始转移到Web网载。同年6月4日更新后停载，2015年6月号（2015年4月21日）转移到[[V_Jump|V Jump plus]]网载，2016年1月号（2015年11月21日）作者因身体抱恙，无期限休载。单行本3卷，正篇暂22话。漫画：藤野耕平，原案：[[本鄉昭由|本鄉昭由]]，监修：[[万代南梦宫游戏|万代南梦宫游戏]]（15话之后改为[[万代南梦宫娱乐|万代南梦宫娱乐]]）。

==== 剧情 ====
《[[数码怪兽|数码怪兽]]》是一款由[[万代|GIGO公司]]提供的，通过育成“[[数码兽|数码兽]]”参加竞技，来争夺排名的在线游戏。主人公就是其中一名热衷于这款游戏的玩家。某日，像往常一样登陆游戏的主人公大河和朋友们，收到了一封神秘的[[邮件|邮件]]。在解读了这封邮件的[[密码|密码]]（密码为“Re：Digitize”）后，主人公便踏上了通往异世界“[[数码世界|数码世界]]”的旅行...
