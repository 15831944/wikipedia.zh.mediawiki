'''奢侈品'''是在[[市场|市场]]上无论是质量，还是[[价格|价格]]都是最高檔次的[[商品|商品]]。不同的市场也有各自不同的奢侈区间，譬如[[房屋|房屋]]、[[土地|土地]]、[[超跑|超跑]]、[[轿车|轿车]]、[[重機|重機]]、[[服飾|服飾]]、[[鞋|鞋]][[襪|襪]]、[[電子產品|電子產品]]、[[高傳真|高傳真]]、[[葡萄酒|葡萄酒]]、[[香檳|香檳]]、[[茶|茶]]，甚至[[私人飛機|私人飛機]]、[[遊艇|遊艇]]、專用[[列車|列車]]、[[武器|武器]]、[[寵物|寵物]]、[[食品|食品]]、[[巧克力|巧克力]]、[[瓶裝水|瓶裝水]]。

因为其的高利润回扣和严格控制着的[[品牌|品牌]]，奢侈品市场被认为对于经济的起落有比较高的敏感度。

== 歷史 ==
在中國古代，飲食常是一種奢侈品，富戶常互競奢華。[[謝肇淛|謝肇淛]]在《五雜俎》中就指出“今之[[富豪|富家]]巨室，窮山之珍，竭水之錯，南方之[[牡蠣|蠣]]房，北方之[[熊掌|熊掌]]，東海之[[鰒|鰒]]炙，西域之馬[[嬭|嬭]]，真昔人所謂富有小四海者，一筵之費，竭中家之產，不能辦也。此以明得意，示豪舉，則可矣，習以為常，不惟開子孫驕溢之門，亦恐折此生有限之福。”又說“孫承佑一[[宴|宴]]，殺物千餘，[[李德裕|李德裕]]一[[羹|羹]]，費至二萬，[[蔡京|蔡京]]嗜[[鹌鹑|鵪子]]，日以千計，[[齊王|齊王]]好[[雞爪|雞跖]]，日進七十。江無畏日用[[鯽魚|鯽魚]]三百，[[王黼|王黼]]庫積[[雀|雀]]鮓三楹。口腹之慾，殘忍暴殄，至此極矣！今時王侯閹宦尚有此風。”[[顾公燮|顾公燮]]在《消夏闲记摘抄》中说：“即以吾苏郡而论，[[西洋|洋]][[货|货]]、[[皮|皮]]货、[[服飾|衣饰]]、[[金|金]][[玉|玉]]、[[珠宝|珠宝]]、[[人參|参]][[中藥|药]]诸铺，[[戏院|戏院]]、[[游船|游船]]、[[酒家|酒肆]]、[[茶室|茶座]]，如山如林，不知几千万人。有千万人之奢华，即有千万人之生理。若欲变千万人之奢华而返于淳，必将使千万人之生理亦几于绝。”

== 奢侈品寡头 ==
[[酩悅·軒尼詩－路易·威登集團|酩悅·軒尼詩－路易·威登集團]]是世界上最大的、拥有七十多个[[品牌|品牌]]的奢侈品供应商。2003年它的销售额达到120亿欧元，利润达到20亿欧元。其他的市场领导者包括[[历峰集团|历峰集团]]和[[开云集团|开云集团]]。

== 典型的奢侈品 ==
在大众文化或者公众印象中，一些商品已经成为了奢侈的代名词。包括[[鲟鱼|鲟鱼]][[鱼子酱|鱼子酱]]、黄金浴具、高級巧克力、[[劳斯莱斯|劳斯莱斯]](Rolls-Royce Motor)轿车、[[兰博基尼|兰博基尼]](Automobili Lamborghini)跑车、各大型時裝品牌，像是[[紀梵希|紀梵希]](Givenchy)、[[香奈兒|香奈兒]](Chanel)跟[[克里斯汀·迪奧_(品牌)|迪奧]](Dior)，[[路易威登|路易威登]](Louis Vuitton)手袋等等。这些都被认为是地位象征而被用来凸现购买者拥有显赫的财富。

== 地段 ==

== 在經濟學 ==
奢侈品是指[[需求的收入彈性|需求的收入彈性]]大於一的物品。當一個人的實質收入增加1%，那人購買奢侈品的增加量將大於1%。
