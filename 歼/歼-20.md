== 历史 == 
[[File:Chengdu_J20_Schematic.png|缩略图]][[罗伯特·盖茨|罗伯特·盖茨]]访华，被認為有政治意涵。]]
歼-20于2007年作为重点型号工程立项进入研制阶段。中国军方与机关支持有能力的单位承研子系统、分系统级机载设备，研制过程采取多家竞争方式，以降低项目风险。<ref name="任德平"/>2009年11月，中國人民解放軍空軍副司令[[何为荣|何為榮]][[將軍|將軍]]預期殲-20將于2017年投入服役。

==設計==
===特点===
歼-20採用全动[[鸭翼|鸭翼]]及[[垂尾|垂尾]]和[[DSI進氣道|DSI進氣道]]，拥有多种[[低可侦测性|低可侦测性]]设计：大致呈[[五边形|五边形]]的機頭及機身横截面、切尖外倾斜双垂尾、带锯齿边空中受油管收納艙、弹舱及起落架舱艙門、镀膜整体座舱盖<ref>{{cite news |title=“明星项目”背后的“王牌先锋”——记中航工业航材院科研一岗专家、博士生导师颜悦 |author= |url = http://www.cannews.com.cn/zghkb/html/2011-06/14/content_20179.htm |newspaper=中国航空报 |date=2011年6月14日 |accessdate=2013-07-25}}</ref>、隐形特种涂料<ref>{{citeweb |title=成飞攻克首架重点机型特种涂料最核心喷涂技术 |author= |url=http://www.chinanews.com/mil/2014/12-02/6836654.shtml |newspaper=中新网 |date=2014-12-02 |accessdate=2014-12-02 |archive-url=https://web.archive.org/web/20141203170503/http://www.chinanews.com/mil/2014/12-02/6836654.shtml |archive-date=2014-12-03 |dead-url=no }}</ref>等，此皆隐形战机主要特征。歼-20验证机[[复合材料|复合材料]]用量达27%，<ref>{{cite news |title=先进复合材料在航空领域的应用 |author=王平 |url=http://cpfd.cnki.com.cn/Article/CPFDTOTAL-BJSD201210009008.htm |journal=第17届全国复合材料学术会议（复合材料应用及产业化分论坛）论文集 |date=2012年 |accessdate=2014-03-02}}</ref>原型机[[钛|钛]]用量达20%、复合材料用量达29%，<ref>{{cite news |title=曹春晓：先进战斗机歼20和歼31是怎么炼成的 |url=http://www.thepaper.cn/newsDetail_forward_1795628 |accessdate=2017-09-15 |archive-url=https://web.archive.org/web/20170915111740/http://www.thepaper.cn/newsDetail_forward_1795628 |archive-date=2017-09-15 |dead-url=no }}</ref>相比中国[[第四代戰鬥機|第四代重型战斗机]][[歼-11B|歼-11B]]分别仅为15%和9%的水平有较大提高。<ref>{{cite news |title=勇于创新 敢为人先——访中国航空工业集团公司沈阳飞机设计研究所副总设计师章怡宁 |author=周娜 |url=http://www.cnki.com.cn/Article/CJFDTotal-ZKJL200902026.htm |journal=中国科技奖励 |date=2009年2月 |issue=2 |accessdate=2014-03-02 |archive-url=https://web.archive.org/web/20150112123832/http://www.cnki.com.cn/Article/CJFDTotal-ZKJL200902026.htm |archive-date=2015-01-12 |dead-url=no }}</ref>此外，歼-20采用了国际先进的液态感温技术、气动伺服控制技术取代铂电阻感温技术、电机作动技术，并采用部件原位检测技术。<ref name="任德平">{{cite web|title=挂帅型号总师 实现军品突破 |url = http://www.cannews.com.cn/epaper/zghkb/2015/11/26/A02/story/727179.shtml |publisher=中国航空报 |date=2015-11-26}}</ref>

同時是中国首型装备整体座舱盖的战机。该整体座舱透明件研制重大攻关于2010年开始，研制团队提出“渐次悬拉＋真空辅助”工艺实现整体座舱透明件成形、镶嵌式过渡边缘连接、柔性吸附定位高精度孔位加工以及复杂曲面均匀膜系镀制等系列技术，使中国成为继美国之后第二个具备研制该类透明件技术能力的国家。<ref>{{cite web |url=http://mil.cnr.cn/zgjd/201106/t20110614_508096001.html |title=专家：突破某型号整体座舱透明件关键技术 |publisher=中国航空报 |date=2011-06-14 }}</ref>

據估計，殲-20長約21米，翼展约13.8米，<ref>{{cite web |url = https://www.strategypage.com/dls/articles/The-J-20-Clarified-3-16-2011.asp |title = The J-20 Clarified |publisher = strategypage |date = 2011年3月16日 |language = en |access-date = 2014-03-02 |archive-url = https://web.archive.org/web/20140302074449/https://www.strategypage.com/dls/articles/The-J-20-Clarified-3-16-2011.asp |archive-date = 2014-03-02 |dead-url = no }}</ref>，推測可提供较大的空間布置內置彈艙和油艙。由於較大的燃料酬載，故較適合遠航。也由於其較大的機體，據推測應配合較大推力的發動機，或預備日後的升級空間。

歼-20在設計佈局上，没有采用美國的[[F-22|F-22]]和俄國[[T-50|T-50]]的常规布局，而是采用了雙激渦流升力体边条翼鴨式氣動佈局，類似颱風戰機，此布局使飞机拥有較优秀的超音速配平能力，良好的高攻角升力特性，較高的瞬時盤旋角速度與滾轉率。但鴨式佈局最大的缺點為最大攻角與巡航持續攻角升阻比的矛盾性，而雙激渦流可平衡這項缺陷。並且鸭翼偏转时产生强度较大的雷達回波，对飞机前向RCS影响比常规布局飞机大。该机外形具有較佳的低可探测設計，但尾喷口的隐形处理不及F-22。按其針對高速突入能力進行強化的思路，預計该机可能拥有反介入、奪取制空權的重點戰術目的。<ref>{{cite web |url=http://news.ifeng.com/mil/special/j20/ |title=深度分析中國四代隱形戰機J-20 |publisher=News.ifeng.com |date= |accessdate=2019-01-22 |archive-url=https://web.archive.org/web/20190102071541/http://news.ifeng.com/mil/special/j20/ |archive-date=2019-01-02 |dead-url=no }}</ref>根據2016年11月12日的官方報導，歼-20未來將裝有與霹靂-10格鬥短距導彈配合的頭盔瞄準器，能實行越肩向側向後攻擊。<ref>{{cite web|url=https://www.youtube.com/watch?v=pQkrpjZ-N-I |title=歼-20結束中美战机代差 |publisher=YouTube.com |date=2016-11-12 |accessdate=2019-01-22}}</ref>

2016年11月殲-20在珠海航展亮相，11月2日[[央視|央視]]專題節目宣称其隱身塗料具有高吸收雷達波性能，對比美國隱形戰機使用隱形塗料具有優勢<ref>{{cite web|url=https://www.youtube.com/watch?v=vphQpAs69wo |title=央視-歼-20首次公开亮相 中国空军将进入隐形战机时代 |publisher=YouTube.com |date=2016-11-01 |accessdate=2019-01-22}}</ref>，同時航電雷達系統受惠於較晚研發所以較高水準，相比F-22於九十年代末設計只能採用當時的電子科技去設計和量產，11月中旬央視專題節目称目前發動機為過渡型號，最終將有新的發動機研製完成換裝，與F-22同等的[[超音速巡航|超音速巡航]]能力是未來的目標。<ref>{{cite web |url=https://www.youtube.com/watch?v=Ffxo7Dxhr_Q |title=央視-殲20最終型態 |publisher=YouTube.com |date=2016-11-07 |accessdate=2019-01-22 |archive-url=https://web.archive.org/web/20161110180849/https://www.youtube.com/watch?v=Ffxo7Dxhr_Q |archive-date=2016-11-10 |dead-url=no }}</ref>

據聲紋分析该机的动力为應該屬於[[AL-31|AL-31]]發動機的某款改型，核心機聲紋特徵與AL-31完全吻合。殲-20的最終發動機将為[[涡扇-15|涡扇-15]]，但聲紋分析也發現殲-20動力已使用一型尚未被確認的發動機，與AL-31M2同時在試飛。<ref>{{cite web |author=Čeština |url=http://en.wikipedia.org/wiki/Chengdu_J-20#cite_note-wake-up-35 |title=Chengdu J-20 - Wikipedia |publisher=En.wikipedia.org |date= |accessdate=2019-01-22 |archive-url=https://web.archive.org/web/20181107193555/https://en.wikipedia.org/wiki/Chengdu_J-20#cite_note-wake-up-35#cite_note-wake-up-35 |archive-date=2018-11-07 |dead-url=no }}</ref>对于歼20发动机具有多方猜测，目前（2019年1月）裝備服役發動機為AL-31M2或WS-10C，但尚未裝備向量噴管。[[尹卓|尹卓]]将军曾在[[中国中央电视台|中央电视台]]表示歼20发动机为三代半国产发动机，最终使用涡扇-15发动机。2017年9月的央視節目正式公佈，配備國產發動機的全國產化殲-20A已經下線，新機型噴口配有鋸齒狀構造隱身，同時可進行有限度超音速巡航。美國《大眾科學》月刊同月做出相同報導，<ref>{{cite web |url=https://www.youtube.com/watch?v=EfTVGN-13oo |title=央視官方頻道- 歼-20升级 中国空军战斗力再提升 |publisher=YouTube.com |date=2017-09-14 |accessdate=2019-01-22 |archive-url=https://web.archive.org/web/20171212021224/https://www.youtube.com/watch?v=EfTVGN-13oo |archive-date=2017-12-12 |dead-url=no }}</ref>認為中國已經突破使用錸鎳超級合金製造單晶[[涡轮叶片|渦輪葉片]]的技術，從此高推力軍用發動機已經沒有障礙。

===武器挂载===
[[File:J-20_fighter_(44040541250)_(cropped).jpg|thumb]]上，一架歼-20打开弹仓展示武器挂载]]
2018年珠海航展上展示歼-20弹仓容量为4枚中距弹+2枚近距格斗弹，而格斗弹倉門是子母式獨創設計，可將飛彈伸出後再關閉80%艙門確保氣動外型平滑，當近距空戰中隱身性已經失去大部分優勢，氣動性能的機動為重要關鍵。另一方面首次展示了埋設於機身中的[[空中加油|空中加油]]管。<ref>{{cite web |url=http://tv.cctv.com/2018/11/13/VIDEN3wSfbIijFvWWWQntMEI181113.shtml |title=央視-利劍藍天 |publisher=Tv.cctv.com |date= |accessdate=2019-01-22 |archive-url=https://web.archive.org/web/20181115113233/http://tv.cctv.com/2018/11/13/VIDEN3wSfbIijFvWWWQntMEI181113.shtml |archive-date=2018-11-15 |dead-url=no }}</ref>
