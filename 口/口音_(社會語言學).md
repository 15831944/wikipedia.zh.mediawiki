'''口音'''（{{lang-en|'''Accent'''}}），又稱'''腔調'''。16岁以下口音可以透過从小习得，（12岁以下效果最好，16岁以下比12岁弱）<ref>{{Cite web|url=http://academic.brooklyn.cuny.edu/english/patkowski/PATKOWSKI-03.pdf|title=Mark patkowski AOA|accessdate=|author=|date=|publisher=}}</ref><ref>{{Cite web|url=http://www.jimflege.com/files/Flege_Yeni-Komshian_age_constraints_JML_1999.pdf|title=Age Constraints on Second-Language Acquisition|accessdate=|author=|date=|publisher=}}</ref>，因此從口音可以反映人的[[出生|出生]][[地区|地区]]或[[社會|社會]]背景。學習某一種口音會使某個社會階層產生認同感。[[演員|演員]]學習口音為了使[[角色|角色]]更傳神。16岁以上可以利用感知磁石效应和参照找对英语学习方法的第一本书<ref>{{Cite book|title=找对英语学习方法的第一本书|last=|first=|publisher=光明日报出版社|year=2012|isbn=9787511225870|location=|pages=3-13 23-27}}</ref><ref>{{Cite web|url=http://web.mit.edu/carrien/Public/speechlab/papers/kuhl1991.pdf|title=perceptual magnet effect - MIT|accessdate=|author=|date=|publisher=}}</ref><ref>{{Cite web|url=https://www.ncbi.nlm.nih.gov/pubmed/7860832|title=Mapping the perceptual magnet effect for speech using signal detection theory and multidimensional scaling.
|accessdate=|author=|date=|publisher=}}</ref>学习。

== 標準音與有口音 ==
* 標準音是根據某個地方的通用音來制定，所以標準音被認為是無口音。
* 有口音通常是指與標準音有一定分別的[[發音|發音]]。
