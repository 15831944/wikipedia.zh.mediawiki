《'''口不對心'''》是[[黎明|黎明]]的國語[[音樂專輯|音樂專輯]]<ref>{{cite web|url= http://www.pmdb.org.tw/?flag=advance&musicName=&kotokoer=&compositioner=&albumName=%E5%8F%A3%E4%B8%8D%E5%B0%8D%E5%BF%83&maker=&publishcomp=&singer=&gender=&language= |title=臺灣流行音樂資料庫|website= www.pmdb.org.tw }}</ref><ref>{{Cite book|title=黎明國語金曲系列|last=黎明|first=|publisher=廣東音像出版社|year=2001|isbn=7885121003}}</ref><ref>{{Cite book|title=百歌爭嗚 成名曲、流行曲快樂大放送|last=林玉鑫編|first=|publisher=北京：海潮出版社|year=2001.01|isbn=7-80151-283-9|location=|pages=371}}</ref>，於1997年推出，專輯收錄了十二首歌曲<ref>{{cite web|url= https://www.kkbox.com/hk/tc/album/RP6gD7j9Md2NtVr0F2uL008l-index.html |title=黎明(Leon Lai) - 口不對心專輯|website= KKbox}}</ref><ref>{{cite web|url= https://mojim.com/tw100171x23.htm |title=口不對心專輯歌曲-黎明(Leon)|website=Mojim.com 魔鏡歌詞網}}</ref>，包括主打歌曲《口不對心》、《情深說話未錯講》(國語版)及電影《[[甜蜜蜜_(電影)|甜蜜蜜]]》主題曲<ref>{{Cite web |url =http://www.merit-times.com.tw/NewsPage.aspx?unid=295831|title =金曲甜蜜蜜 原來是印尼歌|date = 2013年2月25日 |publisher = 人間福報|language = 中文}}</ref><ref>{{Cite web |url =http://news.singtao.ca/toronto/2013-02-27/entertainment1361944397d4368411.html|title =早期創作不涉及版權問題 華語金曲「甜蜜蜜」印尼民謠改編|date =2013年2月27日|publisher =星島日報|language =中文}}</ref>等。
