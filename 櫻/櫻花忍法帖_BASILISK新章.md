《'''櫻花忍法帖 BASILISK新章'''》（{{lang-ja|桜花忍法帖 バジリスク新章}}）是由日本作家[[山田正紀|山田正紀]]創作的小說。小說以作家[[山田風太郎|山田風太郎]]創作的[[甲賀忍法帖|甲賀忍法帖]]及漫畫家{{tsl|ja|せがわまさき|瀨川雅樹}}創作的[[甲賀忍法帖_(動畫)|同名漫畫]]為基礎，描寫甲賀忍法帖十年後所發生的故事<ref>{{Cite web |url = http://news.kodansha.co.jp/20170423_b01 |title = 超名作『甲賀忍法帖』続編！ 伊賀と甲賀が共闘する「忍法バトル」の最高峰 |author = 赤星秀一 |date = 2017-04-23 |publisher = 講談社BOOK倶楽部 |language = ja |accessdate = 2018-06-08}}</ref>。

2017年9月，製作委員會宣佈本作將被製成電視動畫，以「'''Basilisk ～櫻花忍法帖～'''」為標題，並定於2018年1月播放<ref>{{Cite web |url = https://natalie.mu/comic/news/241391 |title = 「バジリスク」新章、TVアニメ化決定！ヤンマガでコミカライズも始動 |author = natalie |date = 2017-07-19 |language = ja |accessdate = 2018-06-08}}</ref><ref name="natalie@171106">{{Cite web |url = https://natalie.mu/comic/news/255629 |title = 年明けにアニメ版も始動、10年後描く新章「バジリスク～桜花忍法帖～」1巻 |author = natalie |date = 2017-11-06 |language = ja |accessdate = 2018-06-08}}</ref>。

== 故事簡介 ==
在忍者集團甲賀與伊賀爆發戰爭並最終迎來雙方全滅後十年，寬永三年，天下變得太平。

在這個和平的環境下，忍者成為了無用之物，唯有隱藏起來。其後，甲賀與伊賀的族人希望透過雙方幼主進行聯婚，以他們的血脈及力量能鞏固族群的命脈。

甲賀五寶連的首領甲賀八郎，以及伊賀五花撰的首領伊賀響，雙方都繼承了祖先的力量而懂得使用瞳術，二人結下契約，然後發動了稱為「櫻」的未知力量。但可惜，在日子漸久後，他們才發現原來他們是親兄妹。<ref name="natalie@171106"/><ref name="story"/>
