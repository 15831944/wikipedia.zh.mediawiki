== 故事簡介 ==
剛升上高中二年級時，吹上慎司在班級裡看到了同班同學「遠野香奈」的幽靈，並在她的引導下知道殺死她的是披著人皮的怪物－「惡靈」。之後慎司受同學桐遠暮羽邀請，與她一起組成「櫻之杜·淨夢者」維護櫻之杜的和平。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|吹上慎司|吹上 慎司（ふきがみ しんじ）}}
: 本作男主角，擁有能看見惡靈及幽靈的能力。
; {{nihongo|秋津圓香|秋津 まどか（あきつ まどか）|聲：藤森ゆき奈}}
: 慎司的青梅竹馬與戀人，被慎司告白後遭人所殺，後來以幽靈的身分與慎司一同行動。興趣是騎[[公路自行車|公路自行車]]。
; {{nihongo|桐遠暮羽|桐遠 暮羽（きりとう くれは）|聲：仙地祐-{里}-衣}}
: 立誓要討伐「惡靈」的少女，在與慎司相識後兩人開始並肩作戰。
; {{nihongo|閑宮真幌|閑宮 真幌（しずみや まほろ）|聲：莓山文太郎}}
: 慎司與圓香的學姐。擁有把夢境跟現實連接起來的能力。
; {{nihongo|衿坂美冬|衿坂 美冬（えりさか みふゆ）|聲：榛名れん}}
: 美術社社員，經常身著白袍，喜歡描繪自然風景。以前與暮羽感情很好，目前則是交惡狀態。
; {{nihongo|吹上初音|吹上 初音（ふきがみ はつね）|聲：白月かなめ}}
: 慎司的表妹。雖然理解慎司的使命感，但並不希望慎司遇到危險。
