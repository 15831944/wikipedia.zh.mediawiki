《'''櫻花大戰系列登場機體'''》展示於遊戲《'''[[櫻花大戰系列|櫻花大戰系列]]'''》所登場的機體。

== 靈子甲冑 ==
* '''{{Nihongo|光武|光武}}'''
: 全名為『虎型靈子甲胄 光武』，由山崎真之介設計，神崎重工開發，配置給帝國華擊團・花組的初期機體。
: 該機體配置七架，於櫻花大戰第一話登場，後在第八話不敵初登場的降魔而嚴重損毀（大破），其後改由神武出擊。
: 登場系列作：『櫻花大戰』、『櫻花大戰 ～熾熱之血～』
* '''{{Nihongo|神武|神武}}'''
: 全名為『卯型靈子甲胄 神武』，在光武嚴重損毀後的後繼機體。
: 該機體同樣配置七架，第八話之後登場，後在對戰降魔王撒旦時損壞，但因修復困難而被中止。
: 登場系列作：『櫻花大戰』、『櫻花大戰 ～熾熱之血～』
* '''{{Nihongo|光武・改|光武・改}}'''
: 全名為『改良型虎型靈子甲胄 光武・改』，光武經過改良後的機體。
: 該機體配置九架（初期的七架+後期的兩架），於櫻花大戰2第一話登場，一度在第八話改駕駛天武，最後再次改回駕駛。
: 登場系列作：『櫻花大戰2 ～願君平安～』、『櫻花大戰3 ～巴黎在燃燒嗎～』
* '''{{Nihongo|天武|天武}}'''
: 全名為『辰型靈子甲胄 天武』，由李紅蘭與花屋敷分部工廠設計，神崎重工開發，並由高村樁從北海道分部暗中運輸而來的機體。
: 該機體同樣配置九架，第八話時登場，後在發現都市能源被敵人掌握的情況下放棄，改回駕駛光武・改。
: 登場系列作：『櫻花大戰2 ～願君平安～』
* '''{{Nihongo|光武F|光武F}}'''
: 法日合作開發，以光武・改的資料為基礎，配置給巴里華擊團・花組的初期機體。
: 該機體配置六架，於櫻花大戰3第一話登場
: 登場系列作：『櫻花大戰3 ～巴黎在燃燒嗎～』
* '''{{Nihongo|光武F2|光武F2}}'''
: 該機體同樣配置六架，第八話時登場
: 登場系列作：『櫻花大戰3 ～巴黎在燃燒嗎～』、『櫻花大戰4 ～戀愛吧少女～』
* '''{{Nihongo|光武二式|光武二式}}'''
: 全名為『改良型虎型靈子甲胄 光武二式』，藉由參考光武F跟光武F2的資料後，再次改良光武・改的機體。
: 該機體配置九架，於櫻花大戰4第一話登場
: 登場系列作：『櫻花大戰4 ～戀愛吧少女～』
* '''{{Nihongo|雙武|双武}}'''
: 全名為『巳型靈子甲胄 雙武』，高性能輸出的決戰用特殊機體。
: 該機體不同於其他光武系列，需要兩名駕駛員操縱（主駕駛為大神+副駕駛為與大神好感度高的五名成員之一），因此只有一架，於櫻花大戰最終話登場。
: 登場系列作：『櫻花大戰4 ～戀愛吧少女～』
* '''{{Nihongo|愛森克萊特|アイゼンクライト}}'''
: 登場系列作：『櫻花大戰2 ～願君平安～』、『櫻花大戰V ～再見吾愛～』
* '''{{Nihongo|STAR|STAR}}'''
: 登場系列作：『櫻花大戰V ～再見吾愛～』
* '''{{Nihongo|三式光武|三式光武}}'''
: 登場系列作：『新櫻花大戰』

== 靈子戰鬥機 ==
在降魔大戰後取代靈子甲冑的全新機種，同時因駕駛者的靈力需求降低的因素，而成為各國的基本配置。

* '''{{Nihongo|無限|無限}}'''
: 為神崎重工以三式光武為基礎，所開發的全新機體，比起靈子甲胄來說性能更加穩定。
: 登場系列作：『新櫻花大戰』
