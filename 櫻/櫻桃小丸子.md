《'''櫻桃小丸子'''》（{{lang-ja|'''ちびまる子ちゃん'''}}，英文：'''Chibi Maruko Chan'''），是由日本已故漫畫家[[櫻桃子|櫻桃子]]創作的日本漫畫作品，於1986年在少女漫畫雜誌《[[Ribon|Ribon]]》開始連載，單行本由[[集英社|集英社]]發售共17冊<ref>{{cite web |title=『ちびまる子ちゃん』最終巻発売へ 作者死去の名作、それぞれのその後 |url=https://news.nifty.com/article/entame/rl/12184-44871/ |website=ニフティニュース |language=ja |date=2018年12月23日}}</ref>，後來改編成動畫、遊戲、電視劇。故事描敘小丸子以及其家人和同學展開，有關於親情、友誼或是一些生活事情，令人回想起童年的稚氣。

==設定==
本作品是以作者[[櫻桃子|櫻桃子]]的童年（昭和49年，即1974年）生活為藍本的1970年代故事，主角的姓名也是與作者同名。故事背景是在日本[[静冈县|静冈县]][[清水市|清水市]]（从2005年起靜岡縣的靜岡市與清水市合併改制为静冈县[[静冈市|静冈市]][[清水区_(日本)|清水区]]）。

== 出版書籍 ==
[[漫畫|漫畫]]曾於1986年8月至1996年6月期間在《RIBON》雜誌連載，連載期間該雜誌創下最高銷量記錄，2002年1月至2016年1月再次不定期刊載，另外在2007年至2011年於《[[東京新聞|東京新聞]]》連載。單行本共發售17集，累計發售3200萬本以上。<ref>{{cite web |title=ちびまる子ちゃん、17巻で完結 集英社、25日発売 |url=https://www.asahi.com/articles/ASLDL5CYPLDLUCVL01N.html |website=朝日新聞デジタル |language=ja |date=2018年12月18日}}</ref>

== 電視動畫 ==
電視動畫是由[[日本動畫公司|日本動畫公司]]製作於1990年1月7日到1992年9月27日期間由[[富士電視台|富士電視台]]播放第一季共142集，並曾於1990年10月28日達到39.9%的收視率，成為自1977年9月26日的紀錄以來[[日本|日本]]史上最高收視的動畫。1995年1月8日開始播放第二季一直持續到現在，與[[SHIN-EI動畫|SHIN-EI動畫]]的《[[蠟筆小新|蠟筆小新]]》共同成為長壽動畫節目。2012年2月12日播出集數累計達成1000回，2015年1月推出播放25周年紀念特別篇。

[[香港|香港]]於1995年由[[香港有線電視兒童台|香港有線電視兒童台]]首播，[[台灣|台灣]]則於1994年有多家電視台播放過[[富國錄音|富國錄音]]配音版。因為本劇受歡迎所以主角小丸子配音的配音員（[[鄭麗麗|鄭麗麗]]、[[林佑俽|林佑俽]]）的代表作之一。

==電視劇==
===日本电视剧===
{{main|櫻桃小丸子 (日本電視劇)}}
為慶祝《櫻桃小丸子》卡通版播放15週年，富士電視台拍攝電視真人版連續劇，共3卷（小丸子愉快的生活／友情大集合／我最喜歡小玉），皆為動畫版的單元劇，在2006年4月18日晚上七時（日本時間）於該台播出，平均收視突破22%，最高更達27%，本輯DVD同年10月27日發行。2008年2月10日，[[香港|香港]][[翡翠台|翡翠台]]亦配合[[農曆新年|農曆新年]]，以《'''福星高照小丸子'''》之名播映。

稍後富士電視台為紀念該動畫製播750集，決定再開拍小丸子真人版，劇集加入豬太郎父母、櫻友藏（小丸子爺爺）朋友中野先生、小玉爸爸與濱崎媽媽，並於同年10月31日播出，也是3卷（小丸子煩惱寫作文／小丸子說謊／櫻家大危機）。本劇亦於2008年3月21日於[[香港|香港]][[翡翠台|翡翠台]]播映，並更名為《'''小丸子部落格'''》。

承接上述兩套真人版短篇連續劇，富士電視台在2007年4月19日起，逢星期四起晚上七至八點於該台播出新一系列小丸子連續劇。劇中櫻桃家除了爺爺角色（元冬樹）全都換上新演員（在下面篇幅敘述時稱為「第二代」，上者短劇則稱為「第一代」），另外小丸子班導師也沿用前劇之演員。小丸子角色則從800人中篩選，便由[[伊籐綺夏|伊籐綺夏]]當選演員，造型亦與第一代演員[[森迫永依|森迫永依]]十分相似。本劇亦於2008年7月6日於[[香港|香港]][[翡翠台|翡翠台]]播映，並更名為《'''小丸子上電視'''》。
