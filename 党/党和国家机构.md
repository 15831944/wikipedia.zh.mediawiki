'''党和国家机构'''，俗称'''四套班子'''或'''四大班子'''，是[[中华人民共和国|中华人民共和国]]的[[政治术语|政治术语]]，指从中央到地方各级政治机构的[[集体领导|领导集体]]，分别为[[党委|党委]]、[[人民代表大会|人大]]、[[中华人民共和国政府|政府]]、[[中国人民政治协商会议|政协]]。除奉行[[一国两制|一国两制]]的[[香港|香港]]、[[澳門|澳门]]外，中国大陆的31个[[省级行政区|省级行政区]]均有设置<ref>{{Cite web |url=http://politics.people.com.cn/n1/2018/0213/c1001-29821312.html|title=全国31个省区市“四大班子”全知道（1200余人全名单）|accessdate=2019-04-18 |author=  |date=2018-02-13 |work=责编：王吉全 |publisher=人民网|language=Zh-hans }}</ref>。

若加上[[纪委|纪委]]，则称'''五套班子'''或'''五大班子'''<ref>{{Cite web |url=http://politics.people.com.cn/n1/2017/0111/c1001-29016526.html|title=广州市新五套班子全部选出开始为市民服务|accessdate=2019-04-18 |author=记者：罗艾桦 |date=2017-01-11 |work=责编：王吉全 |publisher=人民网|language=Zh-hans }}</ref>。2018年，新成立的各省级行政区[[监察委员会|监察委员会]]全部组建完成，[[中华人民共和国国家监察机关|中华人民共和国国家监察机关]]宣告成立<ref>{{Cite web |url=https://news.qq.com/a/20161227/038368.htm|title=一府两院加“一委”：新设的监察委与纪委有何不同|accessdate=2019-04-19 |author=张轩婷 |date=2016-12-27 |work=责任编辑：josezhang |publisher=腾讯网，来源：澎湃新闻|language=Zh-hans }}</ref><ref>{{Cite web |url=http://news.ifeng.com/a/20180212/55985313_0.shtml|title=各省区市监察委员会全部组建完成|accessdate=2019-04-19 |author= |date=2018-02-12 |work= |publisher=凤凰网，来源：中国纪检监察报|language=Zh-hans }}</ref>，与纪委[[合署办公|合署办公]]。
