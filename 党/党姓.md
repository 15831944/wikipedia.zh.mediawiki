'''-{党}-姓'''為[[中文姓氏|中文姓氏]]之一，在《[[百家姓|百家姓]]》中排名第291位，另有罕见姓氏「-{黨}-」，明代有-{[[黨崇雅|黨崇雅]]}-。「-{黨}-」與「-{党}-」為二個不同姓氏，[[中國大陸|中國大陸]]將「-{黨}-」視為「-{党}-」的[[繁体字|繁体字]]廢除<ref>有不少字简化时被合并至另一字，具体有两类情况：一是用已经存在的简单的字代替复杂的字，如“-{涂}-”代替“-{塗}-”、“-{党}-”代替“-{黨}-”、“-{丑}-”代替“-{醜}-”、“-{冲}-”代替“-{衝}-”、“-{后}-”代替“-{後}-”、“-{斗}-”代替“-{鬥}-”等等；二是用一个新的简化字代替两个或多个字</ref>、「-{党}-」兩姓合为“-{党姓}-”，而台灣方面亦將「-{党}-」視為「-{黨}-」的簡體字，所以「-{黨}-」与「-{党}-」兩姓多以「-{黨}-」姓稱之。

== 起源 ==

* 源于[[党項|-{党}-项]][[羌人|羌人]]的姓氏，亦可能出自[[回族|回族]]中的-{党}-姓。

== 地域 ==
[[山东|山东]]、[[山西|山西]]等地多有以“-{党}-家”命名的地名。比如济南-{党}-家庄、-{党}-家镇等。

== 著名人物 ==
* [[党进|-{党}-進]] - 朔州马邑 (今[[山西|山西]][[朔|朔]]县)人，[[北宋|北宋]]初年名将。[[宋太祖|宋太祖]][[乾德|乾德]]五年，曾任[[彰信军|彰信军]]节度兼侍卫步军[[都指挥使|都指挥使]]。[[宋太宗|宋太宗]][[太平兴国|太平兴国]]二年，又任[[忠武军|忠武军]][[节度使|节度使]]。
* [[党怀英|-{党}-怀英]] - 祖籍馮翊（今陝西大荔），生於奉符（今山東泰安），金代文學家、書法家，字世傑、號竹溪、諡文獻。
* [[党崇雅|党崇雅]] - [[陝西省|陝西省]][[宝鸡县|宝鸡县]]贾村原蟠龙人。大明天启五年（1625年）进士出身。卒於大清康熙五年（1666年）。
* [[党蒙|-{党}-蒙]] - [[陝西省|陝西省]][[同州府|同州府]][[韓城縣|韓城縣]]人，[[清朝|清朝]]政治人物。大清[[光緒|光緒]]二年（1876年）進士出身。
