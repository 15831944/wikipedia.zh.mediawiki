'''{{ruby-py|峴|-{zh-cn:xiàn;zh-tw:ㄒㄧㄢˋ;zh-hk:jin<sup>6</sup>}-}}港市'''（{{vie|v=Thành phố Đà Nẵng|hn=城庯沱㶞}}），本名“'''沱㶞'''”，华文世界习作“'''峴港'''”或“'''蜆港'''”，位於[[越南|越南]]中部，古都[[順化|順化]]的附近，地理上屬[[南中部|南中部]]。峴港是越南第四大城市，僅次于[[胡志明市|胡志明市]]、[[河内市|河内市]]和[[海防市|海防市]]，是越南中部最大及重要的城市之一。峴港市瀕臨[[南中國海|南中國海]]、下轄六郡兩縣；距離北方的河內市764公里、南方的胡志明市964公里。

==譯名==
峴港本名為'''沱㶞'''。在越南漢文古籍和法屬以後的國語字資料中，峴港的正式名稱一直是沱㶞。法國人則將沱㶞的法文名稱改作“土倫港”。二戰結束後，[[香港|香港]]媒體曾將峴港譯作“大南”，而越南華人華僑將沱㶞譯作“峴港”<ref>《越南地圖集》（又名《越綿遼地圖集》），越南南圻中華總商會出版，1949年。</ref>和“蜆港”<ref>見於二戰時越南華文資料。</ref>。

==歷史==
“沱㶞”之名最早见于1555年的《[[烏州近錄|烏州近錄]]》，在该书的第五卷记载：“松江祠：祠在思榮縣思客海門，並在廣南陀㶞海門。”

16世紀時，作為[[廣南國|廣南阮主]]首都順化的外港會安，是一個小漁村，作為與周邊地區貿易的港口，曾與日本商人進行交易（南蠻貿易）。

1858年8月，法軍在[[拿破崙三世|拿破崙三世]]的命令下登陸並佔領，成為[[殖民地|殖民地]]，並改名為「'''土伦港'''」（{{lang|fr|Tourane}}），但越南语汉字仍写作“沱㶞”。

1888年10月，[[法屬印度支那|法屬印度支那]]總督府以土伦港為總督直轄地，是越南的第三大城市（僅次於西貢市和河內市）。

1945年，越盟發動八月革命。為紀念反法人士[[蔡璠|蔡璠]]，曾一度改名“蔡璠市”，後恢復沱㶞之名。

在[[越南共和國|越南共和國]]時代，峴港市主城区属于直辖市[[岘港市社|岘港市社]]，和荣县和城郊属于[[廣南省|廣南省]]。1976年，[[越南南方共和国|越南南方共和国]]将广南省、[[广沱省|广沱省]]和岘港市合并为[[廣南-峴港省|廣南-峴港省]]，岘港市为该省的省辖市。

1996年11月6日，廣南-峴港省分拆成[[廣南省|廣南省]]和峴港[[直辖市|直辖市]]<ref>[https://thuvienphapluat.vn/van-ban/Bo-may-hanh-chinh/Nghi-quyet-chia-va-dieu-chinh-dia-gioi-hanh-chinh-tinh-40091.aspx 国会决议：广南-岘港省恢复设立直辖市岘港市和广南省]</ref>。新设立的岘港直辖市包括原岘港市和[[和荣县|和荣县]]、[[黄沙县|黄沙县]]1市2县，并重新划分为海洲郡、莲沼郡、清溪郡、五行山郡、山茶郡、和荣县和黄沙县5郡2县。

2003年7月15日，岘港市被越南政府评定为第一级都市<ref>[https://thuvienphapluat.vn/van-ban/Bo-may-hanh-chinh/Quyet-dinh-145-2003-QD-TTg-cong-nhan-thanh-pho-Da-Nang-la-do-thi-loai-I/51143/noi-dung.aspx 145/2003/QĐ-TTg：评定岘港市为第一级都市]</ref>。

2005年8月5日，增设锦荔郡<ref>[https://thuvienphapluat.vn/van-ban/Bat-dong-san/Nghi-dinh-102-2005-ND-CP-thanh-lap-phuong-xa-thuoc-quan-thanh-khe-huyen-Hoa-Vang-thanh-lap-quan-Cam-Le-thanh-pho-Da-Nang-2725.aspx 102/2005/NĐ-CP：岘港市增设锦荔郡]</ref>。

==行政區劃==
峴港市分為6個郡和2個縣。
* [[海洲郡|海洲郡]]（Quận Hải Châu）
* [[清溪郡|清溪郡]]（Quận Thanh Khê）
* [[山茶郡|山茶郡]]（Quận Sơn Trà）
* [[五行山郡|五行山郡]]（Quận Ngũ Hành Sơn）
* [[蓮沼郡|蓮沼郡]]（Quận Liên Chiểu）
* [[錦荔郡|錦荔郡]]（Quận Cẩm Lệ）
* [[和榮縣|和榮縣]]（Huyện Hòa Vang）
* [[黃沙縣|黃沙縣]]（Huyện Hoàng Sa)，争议领土，無实际控制權，為越南所虛設的縣份之一。

== 文化遺蹟 ==
岘港市擁有眾多歷史文化遺蹟，如[[占婆|占婆]]博物館，[[高台教|高台教]]寺廟，等等。

== 經濟 ==
岘港市以工業為主，出口海鮮、藤製家具、家用品、衣物等。近期往高科技產業發展，故設立[[峴港高科技園區|峴港高科技園區]]。

== 交通 ==
峴港位於[[東西經濟走廊|東西經濟走廊]]的末端。

[[峴港國際機場|峴港國際機場]]位於市中心，是越南第三個[[國際機場|國際機場]]。

== 旅游 ==
岘港是越南中部重要交通枢纽，半径100平方公里之内范围内可直达锦南岛（Cẩm Nam）、[[占婆岛|占婆岛]]、{{link-vi|婆那山|Bà Nà}}（Bà Nà）、{{link-vi|五行山 (越南)|Ngũ Hành Sơn|五行山}}、山茶半岛、[[美山圣地|美山遗址]]、[[顺化皇城|顺化皇城]]、[[會安市|会安古镇]]等著名景点。
