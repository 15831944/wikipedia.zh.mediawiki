'''徐元白'''（1892-1957），號'''原泊'''，[[浙江|浙江]][[台州|台州]]人，是中國著名[[浙派古琴|浙派]][[古琴|古琴]]演奏家，亦通詩詞書畫。<ref name="古曲網">{{Cite web |url= http://info.guqu.net/guqindashi/20060729230859_561.html|title=“现代浙派琴家的代表”徐元白|accessdate=2014-06-25 |work= |publisher=中國古曲網|date=2006-07-29}}</ref>

== 生平 ==

徐元白於[[光緒|光緒]]十八年（1892年）出生於浙江台州海門（今屬[[椒江區|椒江區]]）百口井，父親是[[徐月秋|徐月秋]]。其父工琵琶和古琴，因此自小受其薰陶。1912年到[[蘇州|蘇州]][[天平山|天平山]]向浙派大師[[大休上人|大休上人]]學習古琴，後來又向[[王宾鲁|王宾鲁]]、[[李子昭|李子昭]]、[[杨宗稷|杨宗稷]]、[[张益昌|张益昌]]學習古琴，與[[查阜西|查阜西]]、[[吴韬|吴韬]]亦有來往。<ref name="藝苑"/><ref name="古曲網"/>

1913年經[[李濟深|李濟深]]介紹，前往[[廣州|廣州]]追隨[[孫中山|孫中山]]，曾擔任[[何應欽|何應欽]]的祕書，也曾參加[[北伐戰爭|北伐戰爭]]，在福建、江蘇、河南、浙江、四川的法政部門工作。然而徐元白無心於政治，後遂退出政壇。1936年協助[[査阜西|査阜西]]創辦[[今虞琴社|今虞琴社]]，亦曾在[[今虞琴刊|今虞琴刊]]上發表文章<ref name="國學"/>。[[抗日戰爭|抗日戰爭]]爆發之前曾由上海[[百代公司|百代公司]]錄製其古琴演奏的唱片。<ref name="古曲網"/> 他曾在南京、開封、重慶組織成立青溪琴社、中州琴社、天风琴社。徐元白亦嘗斫琴五十餘牀，發明了“徐氏式”琴<ref name="國學"/>。抗日戰爭結束後，1946年居[[杭州|杭州]]，與妻[[黄雪輝|黄雪輝]]重新修葺[[西湖|西湖]][[雷峯塔|雷峯塔]]下的“半角山房”<ref name="藝苑">{{cite journal|author = 鄭紅|title=故人惜此去 留琴明月前——记浙派古琴大师徐元白|journal=《藝苑》 |issue=2006年08期  |url=http://www.cnki.com.cn/Article/CJFDTotal-YYYB200608022.htm|accessdate=2014-06-27 }}</ref>，併同[[马一浮|马一浮]]、[[张宗祥|张宗祥]]、[[徐映璞|徐映璞]]等人成立“西湖月会”，玩賞琴棋書畫。<ref name="古曲網"/>

1953年蒐集以前的作品，集成《天鳳琴譜摘抄》<ref name="國學">{{Cite web |url= http://www.confucianism.com.cn/html/A00030007/11051859.html|title=重振浙派第一人 论古琴家徐元白的成就与贡献|accessdate=2014-06-25 |work= |publisher=中國國學網|date=2010-06-21}}</ref>。1956年參與査阜西組織的古琴錄音活動，次年因病在杭州逝世。
