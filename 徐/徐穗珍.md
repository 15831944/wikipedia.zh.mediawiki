'''徐穗珍'''（{{kor|k='''서수진'''|rm='''Seo Soo-jin'''}}，{{bd|1998年|3月9日||}}），出生于[[韩国|韩国]][[京畿道|京畿道]][[华城市|华城市]]<!-- {{Citation needed}} begin -->{{fix |link=Wikipedia:列明来源|text=来源请求|class=Template-Fact }}<!-- {{Citation needed}} end -->，韩国女歌手、舞者。现为[[Cube娱乐|Cube娱乐]]旗下女子组合'''[[(G)I-DLE|(G)I-DLE]]'''成员，担任主舞<ref name="showcase">{{cite web|url=https://www.youtube.com/watch?v=Uqhm7vq3u1Y |title=[풀영상] (G)I-DLE 'LATATA' Showcase ((여자)아이들, I am, 쇼케이스, MAZE, 소연, SOYEON) |website=Youtube TongTongTV |date=2018-05-02}}</ref>和副唱<!-- {{Citation needed}} begin -->{{fix |link=Wikipedia:列明来源|text=来源请求|class=Template-Fact }}<!-- {{Citation needed}} end -->。

== 演艺生涯 ==
曾在[[Modern_K实用音乐学院|Modern K实用音乐学院]]学习音乐<ref>{{cite web |title=ModernK Music Academy - 모던K출신 아티스트 |url=http://www.modernm.co.kr/news/activity/ |website=学校官网 |date=2018}}</ref>。

2015年，曾是DN娱乐旗下女团VIVIDIVA(비비디바)的预备成员，以艺名N.NA参与出道前活动<ref>{{cite web |title=[비비디바 VIVIDIVA] 나인뮤지스 - 드라마 창작안무 / 9muses - DRAMA Original Choreography / Choreography - J처리 |url=https://www.youtube.com/watch?v=CmVapptI-0Q |website=Youtube ent DN |date=2015-05-11}}</ref>，但她和成员KANA于出道前离开，该团现已解散<ref>{{cite web |title=Vividiva: Where are they now |url=https://www.allkpop.com/forum/threads/vividiva-where-are-they-now.176044/ |website=allkpop |date=2018-05-22}}</ref>。

高中三年级时参加[[Cube娱乐|Cube娱乐]]面试<ref name="introduction">{{cite web |title=(여자)아이들((G)I-DLE) - 나의 이름은。수진(SOOJIN) |url=https://www.youtube.com/watch?v=lJfzlFJq3Ts |website=Youtube United CUBE |date=2018-04-25}}</ref>，表演是唱[[寶兒|BoA]]《No.1》，跳[[Red_Velvet|Red Velvet]]《Dumb Dumb》舞蹈<ref>{{cite web |title=爱奇艺独家专访(G)I-DLE |url=https://www.iqiyi.com/v_19rqydjhvs.html |website=爱奇艺 |date=2018-07-31}}</ref>。2016年，进入Cube娱乐{{r|introduction}}。

2017年9月9日，作为Cube娱乐练习生首次公开<ref>{{cite web |title=#SSJ |url=https://www.instagram.com/p/BY0g8YDgRTf/ |website=Instagram addxcubetree |date=2017-09-09}}</ref>。在《Jelly》<ref>{{cite web |title=전소연(JEON SOYEON) - 'Jelly' Official Music Video |url=https://www.youtube.com/watch?v=azY7skjwg6Y |website=Youtube United CUBE |date=2017-11-05}}</ref>《Idle Song》<ref>{{cite web |title=전소연(JEON SOYEON) - '아이들 쏭 (Idle song)' Official Music Video |url=https://www.youtube.com/watch?v=2CdgxK7zmO0 |website=Youtube United CUBE |date=2018-02-28}}</ref>MV中饰演面具女孩。

2018年4月，参与综艺节目《首次亮相之前的女子组合》<ref>{{cite web |title=걸그룹 데뷔를 앞둔 한 소녀에게 어마무시한 일이 일어났다?! 데뷔 전부터 살벌한(?) 신인 걸그룹 '아이들' ! |url=https://www.youtube.com/watch?v=FoiDU23wjO4 |website=Youtube dingo |date=2018-04-26}}</ref>和进行舞蹈翻跳活动<ref>{{cite web |title=2018 아는만큼 보이는 아이돌 댄스! 이거 다 맞추면 춤잘알 인정 (여자)아이들 (G)I-DLE DANCE MEDLEY |url=https://www.youtube.com/watch?v=YkrS9kfFO_s |website=Youtube dingo music |date=2018-04-20}}</ref>为出道预热。

2018年5月2日，穗珍以女子组合[[(G)I-DLE|(G)I-DLE]]出道{{r|showcase}}。

== 个人生活 ==
中学就读于{{link-ko|韩国艺术高等学校|한국예술고등학교}}。曾学习跆拳道，获得4级证书{{r|introduction}}。

2018年8月3日，经纪公司声明穗珍与[[Hui|Hui]]已分手<ref>{{cite web |title=[이슈S] 큐브 측 "펜타곤 후이-(여자)아이들 수진, 이미 결별" |url=http://www.spotvnews.co.kr/?mod=news&act=articleView&idxno=229149 |website=SPOTVNEWS |date=2018-08-03}}</ref>。
