'''徐晨軒'''（{{lang-en|Jessie Hsu}}，{{bd|1996年|8月23日}}）是一位臺灣{{le|時裝模特兒|fashion model}}。曾於2014年獲得第31屆{{le|世界精英模特大賽|Elite Model Look}}大中華地區決賽冠軍，<ref>{{cite news |language = zh-tw |author = 楊慧玲 |coauthors =  |url = http://news.tvbs.com.tw/ttalk/blog_author_detail/1053 |title = 超模之路超艱辛！徐晨軒：放棄比較容易 所以珍惜 |publisher = TVBS |pages =  |date = 2016-06-21 |accessdate = 2017-07-06}}</ref><ref>{{cite news |language = zh-tw |author = 吳曉涵 |url = https://style.udn.com/style/story/8065/1034968 |title = 遇見Jessie Hsu徐晨軒 勇敢追夢的時尚小妞 |publisher = 聯合新聞網 |date = 2015-07-07 |accessdate = 2017-07-06}}</ref><ref>{{Cite web |url = http://asia.be.com/fashion/5-things-to-know/models/5-things-know-jessie-hsu-65133.html#item=1 |title = 5 Things to know about Jessie Hsu |author = Be Editorial Team |date = 2015-06-26 |publisher = Be Magazine |language = en |accessdate = 2017-07-06 }}</ref>她同時也是第一位登上{{lang|en|models.com}}的臺灣模特兒。

==私人生活==
徐晨軒出生於台灣，大部份時間在美國[[舊金山|舊金山]]度過，<ref name=Look/>她曾就讀於[[臺北市私立大同高級中學|臺北市私立大同高級中學]]。

她現在居住在[[美國|美國]][[紐約|紐約]]。

==職業生涯==
徐晨軒的模特兒生涯始於15歲，那時她受邀至[[實踐大學|實踐大學]]服裝設計系走秀。2014年，她參加{{le|世界精英模特大賽|Elite Model Look}}並贏得大中華地區決賽冠軍，隨後開始為[[中國|中國]]和[[臺灣|臺灣]]版的《[[時尚_(雜誌)|時尚]]》、《[[哈潑時尚|哈潑時尚]]》、《[[ELLE|ELLE]]》、《Numéro》以及《[[美麗佳人|美麗佳人]]》拍攝廣告。<ref name=Look/>

2016年，徐晨軒開始進軍國際伸展台，她於[[米蘭|米蘭]]參與第一場國際時裝秀，之後曾到[[倫敦|倫敦]]、[[紐約|紐約]]時裝周走秀。<ref>{{cite news |language = zh-tw |author = 廖育婉 |url = https://www.storm.mg/article/398777 |title = 登上紐約時裝周的台灣模特兒！「每個人都獨一無二」徐晨軒、許毓哲勇闖世界舞台 |publisher = 風傳媒 |date = 2018-02-12 |accessdate = 2019-03-30}}</ref><ref>{{Cite web |url = https://www.vogue.com.tw/vogueme/content-39537.html |title = #MeSpecial 19歲就成功拿下米蘭時裝秀！國際模特兒Jessie徐晨軒：「儘管出國工作收入不穩、睡眠不夠、常覺得寂寞，我還是喜歡挑戰自己。」 |author = Tiffany Leu |date = 2018-03-28 |publisher = [[Vogue|Vogue]] |language = zh-tw |accessdate = 2019-03-30 }}</ref>
