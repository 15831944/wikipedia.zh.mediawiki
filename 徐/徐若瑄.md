'''徐若瑄'''（[[泰雅語|泰雅語]]：{{lang|tay|'''Bidai Syulan'''}}，{{lang-en|'''Vivian Hsu'''}}，{{lang-ja|ビビアン・スー}}，{{lang-ko|비비안 수}}<ref name="bio">{{Cite web |url=http://www.vivianhsufanclub.com/traditional/profile.aspx |title=profile |accessdate=2011-06-19 |work=徐若瑄 Official Website |deadurl=yes |archiveurl=https://web.archive.org/web/20110612125557/http://www.vivianhsufanclub.com/traditional/profile.aspx |archivedate=2011-06-12 }}</ref>，{{bd|1975年|3月19日|catIdx=H徐}}），本名'''徐淑娟'''，[[中華民國|中華民國]][[台中縣|台中縣]][[豐原區|豐原市]]（今[[台中市|台中市]]豐原區）出生，[[亞太地區|亞太地區]]知名[[女歌手|女歌手]]、[[女演員|女演員]]、[[主持人|主持人]]。擅長詞曲創作並精通多國語言與樂器。其演藝事業在[[台灣地區|台灣]]、[[香港|香港]]、[[日本|日本]]、[[韓國|韓國]]、[[中国大陆|中国大陸]]以及[[亞洲|亞洲]]各地都有優異的成績<ref>{{cite web

== 經歷 ==
徐若瑄本名'''徐淑娟'''，是家中的二女兒<ref name=avex_profile>{{cite web| language = zh-tw| url = http://www.avex.com.tw/vivian/profile.asp
| title = 徐若瑄 profile | publisher  = 艾迴唱片 | accessdate = 2009年5月6日}}</ref>，父親為[[廣東|廣東]][[梅縣|梅縣]]人，母親為[[臺灣原住民|臺灣原住民]][[泰雅族|泰雅族]]<ref name=cctv_profile>{{cite web| language = zh-cn| url = http://space.tv.cctv.com/act/article.jsp?articleId=ARTI1225793428524182| title = 徐若瑄小档案| publisher = [[CCTV|CCTV]]| accessdate = 2009年5月7日| deadurl = yes| archiveurl = https://web.archive.org/web/20081219202624/http://space.tv.cctv.com/act/article.jsp?articleId=ARTI1225793428524182| archivedate = 2008年12月19日}}</ref>。

1979年四歲時和家人擔任[[林青霞|林青霞]]與[[秦漢_(演員)|秦漢]]所拍攝的電影《[[難忘的一天|難忘的一天]]》[[臨時演員|臨時演員]]。1990年參加[[台灣電視公司|華視]][[綜藝節目|綜藝節目]]《[[TV新秀爭霸站|TV新秀爭霸站]]》舉辦的「才藝美少女選拔」獲得第一名後開始其演藝生涯<ref name=avex_profile/>。並與主持人曹蘭之妹曹琳及吳佩瑜組成[[小貓隊|小貓隊]]作為助理主持，同時的助理主持還有小虎隊
比賽之後與[[開麗創意組合|開麗創意組合]]簽約，並與其他兩位參賽者[[吳珮瑜|吳珮瑜]]、[[王思涵(1977年)|王思涵]]組成偶像團體[[少女隊_(台灣)|少女隊]]，但並沒有大紅<ref name=cctv_profile/>。少女隊解散後，徐若瑄將重心轉為拍攝台灣電視劇和香港電影。1994年和[[林志穎|林志穎]]合作《[[笑林小子|笑林小子]]》後在香港及東南亞略具知名度，同年11月出演《[[終極獵殺|終極獵殺]]》為其第一部有裸露戲的影片，拍攝殺青後及披露將有[[全裸|全裸]][[寫真集|寫真集]]發市。

1990年代初期為[[香港|香港]]情色[[三級片|三級電影]]的盛放期，原本走清純偶像路線的徐若瑄也在此時被重新塑造，拍攝[[情色電影|情色電影]]《[[天使心_(台灣電影)|天使心]]》和同名的情色照片[[寫真集|寫真集]]後知名度大增<ref name=avex_profile/>。

在演出多部戲劇之後，徐若瑄在1995年前往[[日本|日本]]發展，以{{lang|ja|ビビアン}}（即Vivian）的藝名與南南見狂也（[[南原清隆|南原清隆]]），天山（[[天野博之|天野博之]]）及[[丁愷蒂|丁愷蒂]]（團體成名後加入）組成歌唱團體「[[黑色餅乾|黑色餅乾]]」，並在[[日本電視台|日本電視台]]《[[火焰大對抗|火焰大對抗]]》節目中與[[内村光良|内村光良]]為首的[[口袋餅乾|口袋餅乾]]表演多項對抗賽，逐漸成為廣為日本人所知的台灣明星<ref name=warner_profile/>。

1995年10月在日本初次發行日文單曲《くちびるの神話》(紅唇的神話)，1996年10月在韓國推出發行首張韓文專輯《[[天使美少女|天使美少女]]》，1997年黑色餅乾推出發行第一張單曲《[[鬥志_(單曲)|鬥志]]》及個人首張國語專輯《[[大麻煩|大麻煩]]》，並在1998年底上[[紅白歌合戰|紅白對抗]]及推出單曲《[[輕鬆|輕鬆]]》。之後陸續發行了《[[不敗的戀人|不敗的戀人]]》以及《[[假扮的天使|假扮的天使]]》兩張中文專輯，2003年暫別日本演藝圈返台，朝多方位發展。2010年是徐若瑄出道20周年，並宣佈重返日本演藝圈，復出單曲[[BEAUTIFUL_DAY|BEAUTIFUL DAY]]<ref name=avex_profile/>。

2015年中，徐若瑄首次主要投資《[[我來自紐約|我來自紐約]]》擔當監製一職<ref>[http://www.chinapress.com.my/20160226/%E5%BE%90%E8%8B%A5%E7%91%84%E7%82%BA%E3%80%8A%E6%88%91%E4%BE%86%E8%87%AA%E7%B4%90%E7%B4%84%E3%80%8B%E7%95%B6%E7%9B%A3%E8%A3%BD/ 徐若瑄為《我來自紐約》當監製]馬來西亞中國報 2016年2月26日</ref><REF>[http://www.wownews.tw/article.php?sn=35651 《我來自紐約》劇本幽默溫暖 徐若瑄首任監製盼電影感動全世界]wownews 2016年3月6日</REF>。

== 個人生活 ==
徐若瑄于2014年2月5日于一家酒店内举行了私人婚礼，嫁給[[新加坡|新加坡]]的[[马可波罗海业|马可波罗海业]]集团主席李云峰，之后双方亲友一起用餐。受邀出席婚礼的演艺圈人士包括台湾女星[[林心如|林心如]]、港星[[任达华|任达华]]等。

其实在1月份，徐若瑄就透过经纪人向《ETtoday东森新闻云》记者证实，25日和新加坡富商李云峰在台北完成订婚仪式。典礼当天只有邀请双方长辈跟兄弟姐妹，仪式是按传统中式礼仪进行，中间穿插原住民象徵“长长久久”的喝小米酒敬酒仪式。徐若瑄也表示婚后会多留一点时间陪家人和老公，绝对不会息影，对好剧本仍然继续自己的演艺之路。<ref>[http://eladies.sina.com.cn/news/2014/0207/07081239788.shtml 台媒：38岁徐若瑄结婚 嫁小2岁新加坡富商]</ref>

2015年3月20日，在Facebook宣布懷孕，稱孩子是「最美好的生日禮物」。孩子昵称为“李V宝”<ref>[http://ent.qq.com/a/20150321/001193.htm 徐若瑄10月将生天秤宝宝 经纪人称是自然受孕]</ref>。2015年8月13日，在[[新加坡|新加坡]]伊莉莎白醫院產下兒子Dalton<ref>{{cite news|author=封以恩|title=辛苦臥床142天 徐若瑄生了！|url=http://ent.ltn.com.tw/news/breakingnews/1415014|publisher=自由時報|date=2015-08-17}}</ref>。

2016年下半年起全球海運業長期不景氣的後果開始爆發，諸多國際大航商隱隱發生周轉問題<ref>[https://tw.news.yahoo.com/%E5%BE%90%E8%8B%A5%E7%91%84%E9%BB%98%E8%AA%8D%E8%80%81%E5%85%AC%E6%AC%A0%E5%82%B513%E5%84%84-222018978.html 女神老公欠債13億港幣]</ref>，韓國世界第七大的韓進海運破產，各國銀行開始對航運業銀根審查趨緊，9月份，彭博社報導马可波罗海业集团結算遭追討台幣50多億以上的欠債，<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20160921/952541/ 徐若瑄家庭財務危機]</ref>一年內要先還43億，而公司現金只有4億多，可能要變賣資產若不足償還就是破產。但經過兩年的奔走努力，2017年11月徐若瑄的先生李雲峰提出公司重組建議，並成功找到9個願意資助他的金主，獲得6千萬新加坡幣（約13.5億元台幣）資助，債權人及銀行也願意削減所有債務，令他的公司重組成功，一舉清償66億台幣債務。

剛跨入2018年元旦的凌晨，徐若瑄的父親因藥物造成抵抗力不佳，加上感冒引起併發症，跨入2018年的凌晨12點多於臺大醫院病逝，享壽64歲。徐若瑄曾於2009年為救肝硬化的父親，花費近千萬讓徐父到中國大陸換肝，之後徐父需要一直服用抗排斥藥物維持肝臟功能。
