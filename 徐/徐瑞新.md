'''徐瑞新'''（{{bd|1941年|||}}）[[河北省|河北省]][[乐亭县|乐亭县]]中堡王庄乡徐家店村人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中共中央办公厅|中共中央办公厅]]副主任，[[中华人民共和国民政部|中华人民共和国民政部]]副部长。

==生平==
徐瑞新是大专文化。1961年参加工作，1974年加入中国共产党。历任[[中共中央办公厅|中共中央办公厅]]副科长、科长、副处长、处长、副局长、局长、副主任。1995年任[[中华人民共和国民政部|中华人民共和国民政部]]副部长。2001年任[[中国社会工作协会|中国社会工作协会]]会长。曾任[[中南海业余大学|中南海业余大学]]第四任校长。是[[中共十三大|中共十三大]]代表<ref>{{Cite web |url = https://news.qq.com/a/20090812/002295.htm |title =  中国社会工作协会会长徐瑞新|author =  |date =2009-08-12  |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://cncasw.swchina.org/xhjj/ljld/2014021110187.shtml |title =中国社会工作协会第四届理事会  |author =  |date = 2014-02-11 |publisher = 中国社会工作联合会 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://cncasw.swchina.org/zzjg/wjzz/2013/1201/1970.shtml |title = 中国社会工作协会第三次会员代表大会 |author =  |date = 2013-12-01 |publisher =中国社会工作联合会  |language =  |accessdate =  |quote =  }}</ref>。
