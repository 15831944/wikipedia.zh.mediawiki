'''徐宏源'''（{{bd|1963年|5月||}}）[[湖北|湖北]][[石首|石首]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员<ref name=chsh>{{Cite web |url = http://www.changsha.gov.cn/csswyh/ldjs/201707/t20170721_1992973.htm |title =徐宏源  |author =  |date =  2017-07-21|publisher =中共长沙市委  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1985年7月参加工作，1988年1月加入中国共产党，大学学历，高级经济师。1981年9月至1985年7月，在[[中国人民大学|中国人民大学]]财政金融专业学习，获经济学学士学位。1985年7月至1987年2月，任[[国家计委|国家计委]]经济预测中心综合处干部。1987年2月至1992年1月，任[[国家信息中心|国家信息中心]]经济预测部干部。1992年1月至1995年9月，任国家信息中心经济预测部经济预测处副处长。1995年9月至1997年4月，任国家信息中心经济预测部经济调研室副处长。1997年4月至1998年9月，任国家信息中心经济预测部经济调研室处长。1998年9月至2002年5月，任国家信息中心经济预测部副主任（副司级）。2002年5月至2005年10月，任国家信息中心发展研究部副主任（副司级）。其间，2004年12月至2005年12月借调到[[中共中央办公厅调研室|中共中央办公厅调研室]]工作。2005年10月至2005年12月，任国家信息中心信息化研究部主任（正司级）。2005年12月至2009年7月，任中共中央办公厅调研室巡视员<ref name=chsh/>。

2009年7月至2015年7月，任[[中共湖南省委|中共湖南省委]]副秘书长。2015年7月起，任[[中共长沙市委|中共长沙市委]]副书记（正厅）、市委党校（长沙行政学院）校（院）长<ref name=chsh/><ref>{{Cite web |url = http://hn.people.com.cn/n/2015/0722/c195194-25674503.html |title = 湖南省委决定：徐宏源同志任长沙市委副书记|author =  |date =2015-07-22  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
