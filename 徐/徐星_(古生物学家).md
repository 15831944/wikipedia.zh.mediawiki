'''徐星'''（{{bd|1969年||catIdx=}}），[[新疆|新疆]][[伊犁|伊犁]]人，中國[[古生物學家|古生物學家]]，以不少知名的古生物學發現與研究而聞名，近年來亦致力於[[科學普及|科普]]寫作與傳播工作，現職於[[中國科學院|中國科學院]]{{tsl|en|Institute of Vertebrate Paleontology and Paleoanthropology|古脊椎動物與古人類研究所}}。

== 生平 ==
=== 早年生涯 ===
1969年出生於[[中華人民共和國|中華人民共和國]][[新疆|新疆]][[伊宁市|伊宁市]]的徐星祖籍[[江蘇|江蘇]]，[[文革|文革]]時期父母因故移居新疆<ref name = read01>{{ cite web | url = https://read01.com/6nzMDj.html | title = "徐星：穿越時光恐龍緣" | author = 張晶晶 | work = 中國科學報（壹讀轉載） | date = August 1, 2014（Original） }}</ref><ref name = nature>{{ cite web | url = http://www.nature.com/news/china-s-dinosaur-hunter-the-ground-breaker-1.11352 | title = "China's dinosaur hunter: The ground breaker" | author = Kerri Smith | work = [[自然_(期刊)|Nature]] | date = September 5, 2012}}</ref>。僅管生活並不富裕，但徐星的父母讓他從小就閱讀了不少圖書，使讀書成為他小時候了解來自外面訊息的主要管道，早在學生時代，他就對學術抱有極大的興趣，但他的現職[[古生物學家|古生物學家]]起初並不是他的志向。在他小學時，恰逢[[陳景潤|陳景潤]]因為提出[[陳氏定理|陳氏定理]]而聲名大噪，這股熱潮使徐星曾和當時不少青年一樣立志要當[[數學家|數學家]]；到了中學，又因為迷上[[物理|物理]]，轉而想成為[[天體物理學|天體物理學家]]<ref name = uua>{{ cite web | url = http://www.uua.cn/show-12-6758-1.html | title = "我与新疆的恐龙缘――专访著名古生物学家徐星" | author = | work = 化石網 | date = June 4, 2015}}</ref>，在此同時，他也下定決心要進入首屈一指的[[北京大學|北京大學]]就讀<ref name = read01/>。

但徐星[[普通高等學校招生全國統一考試|高考]]那年，北京大學的物理系在新疆沒有招生名額，其他的名額也大多是冷門的學科。為了進北大，他最後在志願卡上選擇服從分配，影響了他日後的發展<ref name = read01/>。

=== 大學時期 ===
1988年，徐星成功考上[[北京大學|北京大學]]<ref name = nature/>，當時錄取通知書上只寫著是分發到古生物學。徐星表示，當時全家人甚至學校的老師都不清楚這門學科是做什麼的，就連學科中最為人熟知的研究對象[[恐龍|恐龍]]他都不曾聽過，他是直到入學後才知道古生物學是[[地質學|地質學]]的一部分範疇，但這個學科此時仍不在他感興趣的範圍內（他也因此嘗試過轉系但沒有成功<ref name = read01/>），因此並沒有好好學習系上的課程<ref name = SciAm>{{ cite journal | author = 鄭靜琪 | title = "追尋飛上天的恐龍——徐星" | journal = [[科學人|科學人]]中文版 | volume = 172 | date = 2016年6月 | pages = 94–97 | issn = 1682-2811 }}</ref>，反而在就讀大學時看了許多諸如[[尚-保羅·沙特|尚-保羅·沙特]]、[[路德維希·維根斯坦|路德維希·維根斯坦]]、[[馬克思|馬克思]]等[[哲學家|哲學家]]與[[經濟學家|經濟學家]][[保羅·薩繆森|保羅·薩繆森]]等人的著作，還到[[人民大學|人民大學]]旁聽了幾門經濟學的課程，後來更因此曾想在[[中國研究生入學考試|考研]]時報考經濟學專業<ref name = read01/><ref name = uua/>。

依照規定，大學畢業後若不繼續升學就必須回家鄉，為了能繼續留在北京，徐星於1992年從北京大學地質學系畢業後，聽了當時在[[北京大學醫學部|北京醫科大學]]就讀[[博士|博士]]的同鄉建議，報考了本科系的[[中國科學院|中國科學院]]{{tsl|en|Institute of Vertebrate Paleontology and Paleoanthropology|古脊椎動物與古人類研究所}}（IVPP），但他仍不想從事古生物學研究，在此期間還曾自修電腦程式設計，但沒做出名堂，直到為了準備畢業論文，整理了手上的幾個恐龍標本，「才發現古生物學其實還挺有意思的」而開始對古生物學產生濃厚的興趣<ref name = uua/><ref name = SciAm/>。

=== 職業生涯 ===
徐星就讀研究所期間，師從古生物學家[[趙喜進|趙喜進]]。1995年，徐星從古脊椎動物與古人類研究所畢業並留所從事研究工作至今。
