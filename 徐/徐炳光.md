'''徐炳光'''[[博士|博士]]，（{{lang-en|'''Dr. TSUI Ping Kwong, Edward'''}}{{bd|||catIdx=Chan陳}}），[[廣東省|廣東省]][[惠州市|惠州市]]人，出生於[[香港|香港]]，為香港[[螺絲業|螺絲業]][[工業家|工業家]]，[[誠興集團|誠興集團]]<ref>{{Cite web |url = http://www.shing-hing.com.hk/aboutus/?type=detail&id=33 |title = 主席的話|publisher=[[誠興集團|''shing-hing.com.hk'']]}}</ref>創辦人兼總裁，同時身兼多項公職，現任[[香港螺絲業協會|香港螺絲業協會]]創會主席<ref>{{Cite web |url = http://www.hk-sfc.org.hk/Structure.asp |title = 協會架構 |publisher = [[香港螺絲業協會|''hk-sfc.org.hk'']] |deadurl = yes |archiveurl = https://web.archive.org/web/20160229191732/http://hk-sfc.org.hk/Structure.asp |archivedate = 2016-02-29 }}</ref>及[[香港中華廠商聯合會|香港中華廠商聯合會]]副會長<ref>{{Cite web |url = http://www.cma.org.hk/gc |title = 會董名錄 |3 =  |publisher = [[香港中華廠商聯合會|''cma.org.hk'']] |access-date = 2015-12-16 |archive-url = https://web.archive.org/web/20180624150433/http://www.cma.org.hk/gc |archive-date = 2018-06-24 |dead-url = yes }}</ref>等。

== 簡歷 ==
徐氏從事螺絲行業40餘年，由其創辦的誠興行實業有限公司於1977年在[[香港|香港]]成立，起初以[[貿易|貿易]]為主，至1993年開始打造自己品牌的螺絲，並提供從生產、檢測至出貨的壹站式服務。徐氏於2003年牽頭成立[[香港螺絲業協會|香港螺絲業協會]]，現為該會永遠名譽主席，其豐富的行業經驗，於業內及傳媒中，被稱為「螺絲大王」。

徐炳光現任創新及科技基金創新及科技支援計劃評審委員會委員、東區撲滅罪行委員會委員、[[香港中華廠商聯合會|香港中華廠商聯合會]]副會長，並在東莞市鳳崗外商投資企業協會、香港創新科技及製造業聯合商會、香港科技協進會等多個商社團中擔任要職。徐炳光自2011年擔任[[孔聖堂中學|孔聖堂中學]]校董<ref>{{Cite web |url = http://www.chss.edu.hk/?op=staff |title = 教職員|publisher=[[孔聖堂中學|''chss.edu.hk'']]}}</ref>，並於2015年獲正式委任為該校[[校監|校監]]，貢獻母校。
