'''徐承云'''（{{bd|1945年|||catIdx=}}）[[安徽|安徽]][[肥东|肥东]]人，[[中国人民解放军中将|中国人民解放军中将]]。

==生平==
1990年，任步兵第三十六师师长。1994年起，历任[[中国人民解放军陆军第十二集团军|陆军第十二集团军]]副军长、[[南京军区|南京军区]]副参谋长、陆军第十二集团军军长<ref>{{Cite web |url = http://news.xinhuanet.com/misc/2002-03/10/content_309713.htm |title =构筑新时代坚固长城:军队代表谈全球化与国家安全  |author =  |date =2002-03-10  |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref>。2002年1月，任南京军区装备部部长<ref>{{Cite web |url =http://news.sohu.com/20/78/news202377820.shtml  |title = 南京军区机关与驻宁部队举行歌咏大会 |author =  |date =2002-07-30  |publisher =  搜狐|language =  |accessdate =  |quote =  }}</ref>。2003年1月，任南京军区参谋长<ref>{{Cite web |url = http://news.china.com/zh_cn/focus/fxkh_2003/11011581/20030711/11504819.html |title = 南京军区出动16000多官兵投入淮河流域抗洪救灾  |author =  |date = 2003-07-11 |publisher = 中华网 |language =  |accessdate =  |quote =  }}</ref>。2004年，任南京军区副司令员<ref>{{Cite web |url =http://www.xinhuanet.com/chinanews/2004-11/28/content_3298671.htm  |title =江苏成功举行“钟山2004”反恐怖综合演练  |author =  |date = 2004-11-28 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。2008年12月退役。

退役后，徐承云担任了中国新四军研究会第一副会长等社会职务<ref>{{Cite web |url = http://news.qq.com/a/20160627/021229.htm |title = 今天，李先念女婿刘亚洲来湖北大悟，为啥 |author =  |date = 2016-06-27 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

1992年晋升少将军衔，2004年7月晋升中将军衔。是[[第九届全国人大代表|第九届全国人大代表]]，[[中共十六大|中共十六大]]代表。2008年当选第十一届全国政协委员<ref>{{cite news|title=第十一届全国政协委员名单|url=http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|accessdate=2013-04-09|newspaper=新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过)|date=2008年1月25日|deadurl=yes|archiveurl=https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|archivedate=2013年9月28日|df=}}</ref>，代表特别邀请人士，分入第五十六组。<ref>{{cite news|title=全国政协信息-徐承云|url=http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0021380&wytype=wjwy|accessdate=2013-04-09|newspaper=全国政协委员会}}</ref>
