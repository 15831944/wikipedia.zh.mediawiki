'''徐玠'''{{BD|868年||943年||catIdx=X}}<ref name=SGCQ21>《[[十國春秋|十國春秋]]》[[s:十國春秋/卷021|卷21]]</ref>，字'''蘊圭'''，封'''高平王'''，[[五代十国|五代十国]][[杨吴|杨吴]]、[[南唐|南唐]]大臣，在南唐烈祖[[李昪|李昪]]时代担任宰相（司徒、右丞相）。徐州彭城县（今[[江苏省|江苏省]][[徐州市|徐州市]]）人。

== 背景 ==
徐玠于[[唐懿宗|唐懿宗]]咸通九年（868年）生于彭城。徐玠敏干有辞办，作为奉國军节度使（治蔡州，今[[河南省|河南省]][[驻马店市|驻马店市]][[汝南县|汝南县]]）崔洪的军吏<ref name=SGCQ21/>。899年，蔡州军队将领崔景思等杀害了崔贤，劫持崔洪，背离[[宣武军节度使|宣武军节度使]]（治今河南省[[开封市|开封市]]）[[朱全忠|朱全忠]]，驱赶[[蔡州|蔡州]]的全部军队和百姓渡过淮水投奔[[淮南节度使|淮南节度使]]（治今[[江苏省|江苏省]][[扬州市|扬州市]]）[[杨行密|杨行密]]。<ref name=ZZTJ261>《[[資治通鑑|資治通鑑]]》[[s:資治通鑑/卷261|卷261]]</ref>崔洪先派徐玠去见杨行密，于是成为杨行密手下的军官。
<ref name=SGCQ21/>

906年，杨行密的儿子[[杨渥|杨渥]]主持淮南，派兵攻打鎮南军（治今[[江西省|江西省]][[南昌市|南昌市]]），当时[[鎮南军节度使|鎮南军节度使]][[钟传|钟传]]死后，他的儿子[[钟匡时|钟匡时]]和他的养子鍾延規争位，淮南趁机吞并鎮南<ref name=ZZTJ265>《資治通鑑》[[s:資治通鑑/卷265|卷265]]</ref>。徐玠参与这次战役，担任粮料使，胜利后，徐玠担任[[吉州|吉州]]刺史（今[[江西省|江西省]][[吉安市|吉安市]]）<ref name=SGCQ21/>。

== 吴国时代 ==
[[徐温|徐温]]和[[张颢|张颢]]杀害杨渥，立杨渥的弟弟[[杨隆演|杨隆演]]为弘农王。912年，杨行密的旧将鎮南军节度使劉威、[[歙州|歙州]]（治今[[安徽省|安徽省]][[黄山市|黄山市]]）刺史陶雅、[[宣州|宣州]]（治今[[安徽省|安徽省]][[宣城市|宣城市]]）刺史李遇、[[常州|常州]]（治今[[江苏省|江苏省]][[常州市|常州市]]）刺史李簡表示不满，特别不尊重徐温。李遇常说：“徐温是什么人，我不曾见过面，一日之间竟当政了！”館驛使徐玠出使[[吴越国|吴越国]]，路过宣州，徐温让徐玠劝说李遇到廣陵朝见新王杨隆演。李遇开始应允了；徐玠说：“您不这样，人家说您谋反。”李遇勃然大怒说：“徐君说我李遇谋反，杀死侍中的人不是谋反吗！”徐温派遣都指挥使[[柴再用|柴再用]]攻打宣州。李遇打开城门，请求归降，徐温派柴再用把他斩首，杀了他全族。<ref name=ZZTJ268>《資治通鑑》[[s:資治通鑑/卷268|卷268]]</ref>李遇的儿子娶了杨行密的女儿。由于是杨隆演的妹妹，于是免于处死。她后来嫁给徐玠为妻。<ref name=SGCQ21/>

918年，徐温的长子[[徐知训|徐知训]]被[[朱瑾|朱瑾]]杀死。徐温的其他亲生子都很年轻,于是由养子[[徐知诰|徐知诰]]辅政。<ref name=ZZTJ270>《資治通鑑》[[s:資治通鑑/卷270|卷270]].</ref>徐知诰认为徐玠治理地方贪污腐败，因此把他免职。然而，徐温认为徐玠善于事奉，<ref name=SGCQ21/>因此让徐玠担任副使。<ref name=ZZTJ276>《資治通鑑》[[s:資治通鑑/卷276|卷276]]</ref>徐玠成为徐温的亲信。徐玠憎恨徐知诰，他经常对徐温说：“辅政的地位重要，不能交给异姓。应该让嫡子[[徐知询|徐知询]]取代徐知诰。”<ref name=SGCQ21/>徐温正在考虑此事，就得病在927年去世。<ref name=ZZTJ276/>

徐温死后，杨隆演的弟弟新即位的吴国皇帝[[杨溥_(吴)|杨溥]]加徐知诰都督中外總軍事，任命徐知询为諸道副都統、鎮海（今江苏省[[镇江市|镇江市]]）寧國（今安徽省宣城市）节度使兼侍中。徐知诰留在吴国中央政府控制朝政，徐知询在金陵（在今代[[江苏省|江苏省]][[南京市|南京市]]）控制国家军队，二人互相争权<ref name=ZZTJ276/>。

徐知诰担心徐知询的军力，但徐知询傲慢，对待各个弟弟也很刻薄，他的亲弟弟们也怨恨他。徐玠诡谲多智，善揣摩，知道徐知询不可辅佐，掌握着他的短处以归附徐知诰。929年，徐知询叫徐知诰到[[金陵|金陵]]解除为父亲徐温治丧的丧服，徐知诰拒绝，声称皇帝杨溥不让他离开。与此同时，徐知询被徐知诰骗入朝中。徐知诰到达广陵，徐知诰留下徐知询做统军，兼领镇海节度使，不允许他回到金陵。派遣右雄武都指挥使柯厚去征调金陵的士卒返回江都，徐知诰从此开始独揽吴国政权。<ref name=ZZTJ276/>

由于徐玠效忠徐知诰，徐知诰和他变得亲密无间，他们之间的怨恨消失了。徐知诰镇守金陵，任命徐玠为行军司马。<ref name=SGCQ21/>934年，徐玠和节度副使[[李建勋|李建勋]]等人多次陈奏徐知诰的功业，希望杨溥禅让皇位给徐知诰<ref name=ZZTJ279>《資治通鑑》[[s:資治通鑑/卷279|卷279]]</ref>。徐知诰被任命为大元帅，936年建立大元帅府，于是徐知诰任命都统判官[[宋齐丘|宋齐丘]]为元帅府左司马，行军司马徐玠为元帅府右司马。<ref name=ZZTJ280>《資治通鑑》[[s:資治通鑑/卷280|卷280]]</ref>937年，徐知诰被封为齐王，于是委任左、右司马宋齐丘和徐玠为左、右丞相<ref name=ZZTJ281>《資治通鑑》[[s:資治通鑑/卷281|卷281]]</ref>。

== 南唐时代 ==
937年[[九月十七|九月十七]]日，杨溥禅让皇位于徐知诰，吴国灭亡。[[十月初五|十月初五]]，徐知诰即皇帝位，建立齐国（[[南唐|南唐]]）。[[十月初六|十月初六]]，徐知诰遣派右丞相徐玠奉送上尊号的册文去进诣杨溥，称言受禅老臣诰谨拜稽首上皇帝尊号为高尚思玄弘古让皇，宫室、乘舆、服御都照旧，宗庙、正朔、徽章、服色都仍按吴国制度。次年六月，群臣争着请求更改府寺州县的名称中有“吴”和“阳”字的，留守判官杨嗣请求改姓羊，徐玠说：“陛下应天顺人，不是逆取，谄邪之人以这些事更改讨好，不是当务之急，不要听从。”徐知诰认为很对。<ref name=ZZTJ281/>

徐玠被任命为寧國军节度使（治今[[安徽省|安徽省]][[宣城市|宣城市]]）。在宣州时，他的妻子杨氏愤恨而卒。之后，徐玠改任鎮南军节度使（治今江西省南昌市），兼[[中書令|中書令]]。之后召回金陵，担任[[司徒|司徒]]、[[右丞相|右丞相]]，但并不参与朝政。据说他年老时，更加贪婪，他的下属都以他为祸患。他还参与炼丹，往往买折扣的[[朱砂|朱砂]]练药，被人嘲笑。943年五月，徐玠卒，时年七十六岁，赠南平郡王<ref name=SGCQ21/>。
