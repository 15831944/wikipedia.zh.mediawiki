'''徐承恩'''（'''Eric S.Y. Tsui'''）{{BD|1978年|}}，字弼群，是一名[[香港|香港]]醫生、論政人、業餘歷史學者。早年畢業於[[皇仁書院|皇仁書院]]，後來入讀[[香港中文大學|香港中文大學]][[新亞書院|新亞書院]]醫學院並於2002年畢業，2004年開始做私家醫生。再入讀[[香港中文大學|香港中文大學]]研究院攻讀社會學文學碩士和哲學碩士，師從[[陳健民|陳健民]]。2009年到英國攻讀政治學博士，但在一個月後返回香港。<ref>{{ cite web |url=https://www.cw.com.tw/article/article.action?id=5082218 |title=從醫生到民間史家，徐承恩的香港歷史書寫 |date=2017-04-28 |publisher=天下雜誌 |author=林怡廷 }}</ref>

2012年，徐承恩和伍子豐、易汶健合著《菁英惡鬥：香港官商霸權興衰史》<ref>{{ cite book |url= |title=《精英惡鬥－－香港官商霸權興衰史》 |date=2012-07-01|publisher=東寶|author=徐承恩、伍子豐、易汶健 |location=香港 | pages=  |ISBN=9789881853783 }} </ref><ref>{{ cite web |url=https://sytsui.wordpress.com/2015/08/01/battles_of_hongkong_elites/ |title=《菁英惡鬥：香港官商霸權興衰史》電子版 |date= |publisher= |author= }}</ref>，研究香港政商界的社會網絡和影響力。2013年，徐承恩將閱讀十二本英文香港史著作的讀書筆記寫成《城邦舊事：十二本書看香港本土史》<ref>{{ cite book |url= |title=《城邦舊事：十二本書看香港本土史》 |date=2014-01-01|publisher=紅出版(青森文化)|author=徐承恩 |location=香港 | pages=  |ISBN=9789888270279 }} </ref>。之後再在《城邦舊事》之基礎上，寫出《香港，鬱躁的家邦：本土觀點的香港源流史》（香港版：《鬱躁的城邦：香港民族源流史》）這本本土派史觀的通史。

他以[[台灣獨立運動|台灣獨立運動]]推動者[[王育德|王育德]]為偶像，並且與台灣學界的[[吳叡人|吳叡人]]、[[吳介民|吳介民]]、[[吳豪人|吳豪人]]和[[何明修|何明修]]等人相熟。推崇的人物包括日本國學者[[本居宣長|本居宣長]]、推動朝鮮現代化改革的[[徐載弼|徐載弼]]及一生推動臺灣獨立的[[史明|史明]]，喜讀[[葛兆光|葛兆光]]、[[杜贊奇|杜贊奇]]，以及[[華南學派|華南學派]]和[[新清史|新清史]]之著述。他經常批評[[羅永生|羅永生]]和[[梁文道|梁文道]]對本土的看法，認為二人對本土派的批評既不公道、亦欠真誠<ref>[https://www.facebook.com/hknationalhistory/photos/pb.345348192341558.-2207520000.1512107120./655583484651359/?type=3&theater 港國述事：徐承恩本土歷史台 帖子]</ref>。
