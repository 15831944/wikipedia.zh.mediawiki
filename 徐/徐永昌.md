'''徐永昌'''（{{bd|1887年|12月15日|1959年|7月12日|catIdx=Xu徐}}），字'''次宸'''，[[山西省|山西省]]崞{{NoteTag|音「ㄍㄨㄛ」，“guo”}}县（[[原平市|原平市]]）人，<ref name="徐永昌">[http://museum.mnd.gov.tw/Publish.aspx?cnid=1482&p=12180 徐永昌（西元1889～1959）]</ref>[[中華民國陸軍|中華民國陸軍]][[一級上將|一級上將]]，[[第二次世界大戰|第二次世界大戰]]結束時，代表[[中華民國|中華民國]]參加[[盟軍|盟軍]]受降儀式，後出任[[中華民國國防部部長|中華民國國防部部長]]。

== 早年經歷 ==
1908年，入讀[[武衛左軍隨營總堂|武衛左軍隨營總堂]]，畢業後授[[副軍校|副軍校]]（[[中尉|中尉]]），任[[武衛左軍|武衛左軍]]路前營左哨副哨長。[[辛亥革命|辛亥革命]]爆發時，駐守[[北京|北京]][[寶泉局|寶泉局]]。1914年，考入[[陸軍大學|陸軍大學]]第四期。1916年，赴山東參與[[居正|居正]]、吳大洲之民軍，進行[[護國戰爭|倒袁運動]]<ref name="徐永昌"/>。

== 中年經歷 ==
1917年冬，任[[直隸軍官教育團|直隸軍官教育團]]教官。1924年10月，任[[國民軍|國民軍]]第三軍第一混成旅旅長。1925年1月，任國民軍第三軍第一混成旅[[旅長|旅長]]；8月，任國民軍第一師[[師長|師長]]兼[[陝西|陝西]]警備司令。1926年9月，馮玉祥於五原誓師，徐永昌以主張不合，乃與馮玉祥疏遠。1927年，率部入山西，歸入[[閻錫山|閻錫山]]屬下，參加[[國民革命軍北伐|國民革命軍北伐]]。1928年1月，任國民革命軍第十二路軍總指揮；6月，任[[河北省政府|河北省政府]]委員；10月，任[[綏遠省政府|綏遠省政府]]第一任主席。1929年8月，調任[[河北省政府|河北省政府]]主席。

1930年4月，[[中原大戰|中原大戰]]爆發後，任閻錫山之副司令兼前敵總司令；9月，被任為晉綏警備總司令。
[[中原大戰|中原大戰]]結束後，1930年11月5號至11月30號短暫停職一切事務，1930年12月1日，復職國民政府軍事委員會委員，1931年8月，[[國民政府|國民政府]]任命徐永昌為[[山西省政府|山西省政府]]主席<ref name=徐永昌 />。

1937年3月6日，國民政府特任陸軍二級上將徐永昌為軍事委員會辦公廳主任<ref name="民" >{{cite book |editors=[[李新_(1918年)|李新]]總主編，中國社會科學院近代史研究所中華民國史研究室，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=北京 |publisher=[[中華書局|中華書局]] |year=2011 |isbn=9787101079982}}</ref>{{rp|5379}}。3月18日，蔣指示徐永昌檢查各地國防工事<ref name=民 />{{rp|5387}}。4月2日，閻錫山在太原綏靖公署召開省防會議，討論防務、訓練等問題，徐永昌、傅作義及在綏遠各將領出席會議<ref name=民 />{{rp|5395}}。7月7日，[[蘆溝橋事變|蘆溝橋事變]]發生，任[[軍事委員會委員長|軍事委員會委員長]][[保定|保定]]行營主任，指揮[[第一戰區|第一戰區]]抗日軍事。1938年1月，軍事委員會改組，奉命出任[[國民政府軍令部|國民政府軍令部]][[部長|部長]]，期間運籌為幄，獻替良多，於1943年10月獲頒[[青天白日勳章|青天白日勳章]]<ref name=徐永昌 />。

1945年8月，日本宣佈無條件投降，國民政府特派徐永昌代表中華民國參與9月在[[日本|日本]][[東京灣|東京灣]]美艦[[密蘇里號|密蘇里號]]之受降典禮（同行人員尚有[[楊宣誠|楊宣誠]]中將、[[朱世明|朱世明]]少將、[[王之|王之]]少將、[[李樹正|李樹正]]少將、[[王丕承|王丕承]]少將）。1946年，被選為[[制憲國民大會|制憲國民大會]]代表<ref name=徐永昌 />。

1947年，當選第一屆[[國民大會代表|國民大會代表]]；12月30日，行政院會議決議，派徐永昌任[[陸軍大學|陸軍大學]]校長<ref name=民 />{{rp|8477}}。

== 晚年經歷 ==
1948年12月，徐永昌任[[中華民國國防部部長|中華民國國防部部長]]<ref name=徐永昌 />。1949年夏天，任[[行政院|行政院]]政務委員<ref name=徐永昌 />。8月22日下午，[[陸軍大學|陸軍大學]]校長徐永昌奉命去西北公幹，從廣州飛重慶，8月23日因為飛機在中途故障，致8月24日成行<ref>{{cite web |title=徐永昌赴西北公幹 |url=https://mmis.hkpl.gov.hk/coverpage/-/coverpage/view?_coverpage_WAR_mmisportalportlet_actual_q=%28%20%28%20allTermsMandatory%3A%28true%29%20OR+all_dc.title%3A%28%E8%A1%86%E5%88%A9%E8%BC%AA%E7%88%86%E7%82%B8%29%20OR+all_dc.creator%3A%28%E8%A1%86%E5%88%A9%E8%BC%AA%E7%88%86%E7%82%B8%29%20OR+all_dc.contributor%3A%28%E8%A1%86%E5%88%A9%E8%BC%AA%E7%88%86%E7%82%B8%29%20OR+all_dc.subject%3A%28%E8%A1%86%E5%88%A9%E8%BC%AA%E7%88%86%E7%82%B8%29%20OR+fulltext%3A%28%E8%A1%86%E5%88%A9%E8%BC%AA%E7%88%86%E7%82%B8%29%20OR+all_dc.description%3A%28%E8%A1%86%E5%88%A9%E8%BC%AA%E7%88%86%E7%82%B8%29%29%20AND+%28%20dc.category.category%3A%28Newspaper%29%29%29&_coverpage_WAR_mmisportalportlet_sort_field=score&p_r_p_-1078056564_c=QF757YsWv58JCjtBMMIqol7ZMNUu8xVa&_coverpage_WAR_mmisportalportlet_sort_order=desc&_coverpage_WAR_mmisportalportlet_o=0&_coverpage_WAR_mmisportalportlet_hsf=%E8%A1%86%E5%88%A9%E8%BC%AA%E7%88%86%E7%82%B8 |publisher=《[[華僑日報|華僑日報]]》 |date=1949-08-25 }}</ref>。9月1日，徐永昌自銀川飛赴南鄭，轉回重慶<ref name=民 />{{rp|9001}}。[[中華民國政府|政府]]遷台，率陸軍大學師生遷移[[台北|台北]]<ref name=徐永昌 />。

1952年，任[[中華民國總統府資政|總統府資政]]；10月，晉升[[中華民國陸軍|陸軍]][[一級上將|一級上將]]<ref name=徐永昌 />。1954年11月，兼任[[光復大陸設計研究委員會|光復大陸設計研究委員會]]副主任委員<ref name=徐永昌 />。

1959年7月12日，退役陸軍一級上將徐永昌在[[台北市|台北市]]積勞病故，享壽72歲<ref>《旌忠狀》，台北：[[總統府|總統府]]，1959年10月26日</ref>。

著有《徐永昌日記》（1989年出版）。
