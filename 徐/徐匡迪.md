'''徐匡迪'''{{BD|1937年|12月11日|catIdx=X}}，初名'''徐[[中国抗日战争|抗敌]]'''<ref>{{cite news|url=http://news.sina.com.cn/c/2002-04-09/0435539448.html|title=上海市前市长徐匡迪的故事：名字原来叫“抗敌”|publisher=劳动报|date=2002-04-09|accessdate=2018-04-15}}</ref>，[[浙江省|浙江省]][[崇德縣|崇德縣]]（今属[[桐乡市|桐乡市]]）人，是[[中華人民共和國|中国]][[钢铁|钢铁]][[冶金|冶金]]专家和政治人物。北京钢铁学院（现[[北京科技大学|北京科技大学]]）冶金系毕业。1983年6月加入[[中国共产党|中国共产党]]，是中共第14届[[中央候补委员|中央候补委员]]，第15、16届[[中央委员|中央委员]]。

历任[[上海工业大学|上海工业大学]]常务副校长、[[上海市教育局|上海市高等教育局]]局长，[[上海市发展和改革委员会|上海市计划委员会]]主任，[[上海市人民政府|上海市人民政府]][[上海市市长|市长]]（1995年至2001年），[[中国工程院|中国工程院]]党组书记、院长（2002年至2010年），第十届[[全国政协副主席|全国政协副主席]]（2003年至2008年，位列[[国家级副职|国家级副职]][[党和国家领导人|党和国家领导人]]）。

现任[[中国金属学会|中国金属学会]]理事长、[[中国工业经济联合会|中国工业经济联合会]]会长，[[京津冀协同发展|京津冀协同发展专家咨询委员会]]组长，[[上海大学|上海大学]][[教授|教授]]、[[博士生导师|博士生导师]]。1995年当选[[中国工程院院士|中国工程院院士]]，现为中国工程院[[主席团|主席团]]名誉主席。

== 治学 ==
1954年，从[[浙江省杭州高级中学|浙江省杭州高级中学]]毕业后，徐匡迪选择以当时国家急需的钢铁事业作为自己的专业，进入了北京钢铁工业学院（今[[北京科技大学|北京科技大学]]）学习，毕业后留校任教。1963年，被调往[[上海工业大学|上海工学院]]，在冶金工程系当助教，后升任炼钢教研室副主任。“[[文革|文革]]”开始后，1971年徐匡迪被下放到[[安徽省|安徽省]][[鳳陽縣|鳳陽縣]]“[[五七干校|五七干校]]”劳动；一年后回到讲台，先后在[[上海理工大学|上海机械学院]]和[[上海工业大学|上海工业大学]]冶金工程系任教。

自1980年起，徐匡迪历任[[上海工业大学|上海工业大学]]（今[[上海大学|上海大学]]）冶金工程系副主任、主任、副教授和教授，期间还曾到[[英国|英国]][[帝国理工学院|帝国理工学院]]做合作研究及访问学者。1984年至1985年间，他又曾赴[[瑞典|瑞典]]斯堪的纳维亚·兰塞尔公司工作，任副总工程师和技术经理。1986年回国后，被任命为上海工业大学的常务副校长，1989年出任[[上海市高等教育局|上海市高等教育局]]局长<ref>[http://news.xinhuanet.com/ziliao/2002-06/04/content_424148.htm 新华网：徐匡迪简历]</ref>。

== 从政 ==
对于从政，徐匡迪曾表示自己在瑞典[[兰赛尔公司|兰赛尔公司]]工作时，经常到国际上去招标與投标，而參與项目时更需要有一些融资的方案，慢慢地就对资金的运作比较清楚，也開始对经济感兴趣<ref>[http://www.usacn.com/bmx/bmx047/4724.htm 徐匡迪离开上海之后]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }} 作者杜平，《北美行》</ref>。1990年，徐匡迪陪同时任[[上海市市长|上海市市长]][[朱镕基|朱镕基]]出访[[欧洲|欧洲]]，受到赏识。翌年被任命为上海市计划委员会主任，以54岁的年龄进入政界<ref>[http://news.ifeng.com/mainland/detail_2010_06/07/1591384_0.shtml 中国现任工程院院长徐匡迪院士即将卸任]</ref>。

1991年8月，徐匡迪被提升为上海市副市长，三个月后当选中共上海市委常委，两年后又晋升中共上海市委副书记。1995年2月，接任上海市市长，同年又当选为[[中国工程院|中国工程院]]院士。这使他成为中国第一位拥有“院士”资格的[[市长|市长]]<ref>[http://www.people.com.cn/GB/paper2086/5676/577533.html 人民网：徐匡迪：从科学家到市长]</ref>。

2001年12月7日，徐匡迪突然辞去了上海市市长的职务。次年年初，被宣布出任[[中国工程院|中国工程院]]党组书记，并在当年5月当选[[中国工程院|中国工程院]]院长。2003年3月，在[[第十届全国政协|第十届全国政协]]第一次会议上，徐匡迪又当选为全国政协副主席，成为[[党和国家领导人|党和国家领导人]]，同年9月起又兼任[[中国工业经济联合会|中国工业经济联合会]]会长，2006年连任中国工程院院长。

2010年6月，已担任两届工程院领导职务的徐匡迪正式退休。

2014年，担任[[京津冀协同发展|京津冀协同发展专家咨询委员会]]组长。

== 学术成就 ==
徐匡迪教授长期从事电炉炼钢、喷射冶金、钢液二次精炼及熔融还原的研究，曾成功研制的SGDF型喷粉罐获得了广泛应用。其他的研究成果如高纯管线钢的真空循环脱气、喷粉（RH-IJ）技术、超低硫钢冶炼技术及铁浴法熔融还原不锈钢母液等等，都在国内外企业中得到应用。徐匡迪还提出了铁液脱硫的“拟一级不可逆反应处理法”及锰熔融还原三步反应模式等；发表学术论文50余篇，专著6本<ref>[http://www.cae.cn/experts/detail.jsp?id=131 院士信息：徐匡迪] {{webarchive|url=https://web.archive.org/web/20080308062649/http://www.cae.cn/experts/detail.jsp?id=131 |date=2008-03-08 }}</ref>。

== 学术荣誉 ==
徐匡迪曾先后获得过六项国家级、部委级和上海市重大科技成果奖、科技进步奖，还曾被授予“国家级有突出贡献的中青年科技专家”称号，徐同时还是「英国皇家土木工程师协会」荣誉会员和「[[英国|英国]][[皇家工程院|皇家工程院]]」荣誉院士。

*2003年5月，当选[[瑞典皇家工程科学院|瑞典皇家工程科学院]]外籍院士。
*2006年9月，获“瑞典皇家北极星大十字司令官”勋章。
*2007年8月，被「日本铁钢学会」授予荣誉会员称号。

== 评价 ==
徐匡迪是中国第一位“院士市长”<ref>[http://news.xinhuanet.com/st/2006-06/16/content_4707194_1.htm 徐匡迪：院士不言退休 我必须终生为之奋斗]</ref>，担任市长期间还带了几个[[博士|博士]]班，与学术界关系密切，而且[[英语|英语]]流利，曾经用英语接受[[BBC|BBC]]访谈节目《{{link-en|HARDtalk}}》专访。

徐的主要贡献是改善了上海的投资环境，继续了[[浦东开发开放|浦东开发开放]]的政策。因其任内出色的工作表现及他与[[朱镕基|朱镕基]]的密切关系，曾经一度传言有可能升任[[国务院副总理|国务院副总理]]。但在2001年12月7日，徐匡迪突然辞去上海市长职务，此事引起外界猜测，有媒体揣测徐匡迪与前中共上海市委书记[[黄菊|黄菊]]不合，但继任人选兼常务副市长的[[陈良宇|陈良宇]]曾解释说，徐匡迪辞职的原因是因为年龄到限，65岁是中共对于担任正部级官员年龄的最高规定<ref>[http://news.bbc.co.uk/chinese/simp/hi/newsid_1840000/newsid_1845100/1845113.stm 上海新市长谈徐匡迪辞职] BBC中文网</ref>。
