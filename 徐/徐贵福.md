'''徐贵福'''（{{bd|？|||}}）籍贯不详，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
徐贵福长期在[[中国人民解放军沈阳军区|中国人民解放军沈阳军区]]服役。曾任沈阳军区政治部干部部任免处处长<ref>{{Cite web |url = http://www.people.com.cn/GB/junshi/60/20010111/375962.html |title = 打得赢：军人提升学历层次的终极坐标——来自沈阳军区干部在职学习的讨论与思考 |author =  |date = 2001-01-11 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>，沈阳军区65116部队政委、副大队长，[[中国人民解放军陆军第十六集团军|中国人民解放军陆军第十六集团军]]装甲第四师政治委员等职。2008年调任[[中国人民解放军陆军第四十集团军|中国人民解放军陆军第四十集团军]]副政治委员。2014年3月升任政治委员<ref name=tkp>{{Cite web |url = http://news.takungpao.com/mainland/zgzq/2014-04/2441818.html |title = 徐贵福“扶正” 接替刘念光升任陆军40军政委 |author =  |date = 2014-04-25 |publisher = 大公网 |language =  |accessdate =  |quote =  }}</ref>。2017年，改任[[中国人民解放军中部战区陆军|中国人民解放军中部战区陆军]]副政治委员<ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1740065|title =原40集团军政委徐贵福改任中部战区陆军副政委  |author =  |date =2017-07-23  |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。
