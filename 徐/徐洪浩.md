'''徐洪浩'''（{{bd|1973年|10月28日|catIdx=X徐}}）,出生于[[安徽省|安徽省]][[合肥市|合肥市]]，毕业于[[上海戏剧学院|上海戏剧学院]]表演系，是一位中国大陆男演员。

==生平==
1996年，参演黄克敏执导的[[爱情|爱情剧]]《女人为什么哭泣》开启演员生涯。2004年，主演[[医疗|军医剧]]《[[追日|追日]]》，饰演马俊一角，并凭此获得第十七届“[[金星奖|金星奖]]”优秀演员奖<ref>{{cite web |url=http://news.sina.com.cn/o/2005-02-25/08325198401s.shtml |title=第17届全军电视剧金星奖评奖结束(附获奖名单) |date=2005-02-25 |accessdate=2017-03-06 |author=新华网 |publisher=新浪新闻}}</ref>。2008年起，先后出演由[[刘猛_(作家)|刘猛]]执导的《阻击生死线》、《[[我是特种兵|我是特种兵]]》、《我是特种兵之利刃出鞘》、《我是特种兵之火凤凰》、《[[特警力量|特警力量]]》等[[军旅|军旅剧]]。2017年2月，其主演的[[消防|消防剧]]《[[火线出击|火线出击]]》在网络平台上线播出<ref>{{cite web |url=http://news.163.com/17/0228/01/CEATE69J00018AOP.html |title=《火线出击》开播 《我是特种兵》 原班人马打造 |date=2017-02-28 |accessdate=2017-03-06 |author=新快报|publisher=网易新闻}}</ref>。
