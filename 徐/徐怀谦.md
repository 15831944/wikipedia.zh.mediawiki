'''徐怀谦'''{{BD|1968年||2012年|8月22日|catIdx=X徐}}出生于[[山东省|山东省]][[高密市|高密市]]，毕业于[[北京大学|北京大学]]中文系、[[中国社科研究生院|中国社科研究生院]]文学硕士，曾是[[虞城县|虞城县]]县委副书记，为《[[人民日报|人民日报]]》大地副刊主编。<ref name="fish " >{{cite web |url= http://www.chinawriter.com.cn/fwzj/writer/134.shtml|title=徐怀谦 |accessdate=2012年11月5日 |publisher=中国作家网 |archiveurl=http://www.chinawriter.com.cn/fwzj/writer/134.shtml |archivedate=2012年11月5日 |work= |language =zh-cn}}</ref>

== 生平 ==
1968年，徐怀谦出生在[[山东省|山东省]][[高密市|高密市]]，曾在[[北京大学|北京大学]]中文系读书，大学毕业后，分配到[[首钢集团|首钢集团]]工作，1年后，又调配到《人民日报》文艺部工作，1999年，徐怀谦出任[[河南省|河南省]][[商丘市|商丘市]][[虞城县|虞城县]]县委副书记，去职之后，获得[[中国社科研究生院|中国社科研究生院]]文学硕士学位，最后出任《人民日报》大地副刊主编，后来因为[[抑郁症|抑郁症]]而退休到北京安定医院治疗，2012年，徐怀谦因为抑郁症在下午14点跳楼自杀，北京朝阳医院抢救无效逝世，<ref>{{cite web |url=http://www.china.com.cn/photochina/2012-08/24/content_26326483.htm |title=人民日报副刊主编徐怀谦坠楼身亡 |accessdate=2012-08-24 |publisher=中国网 |archiveurl= http://www.china.com.cn/photochina/2012-08/24/content_26326483.htm|archivedate=2012年11月5日 |work= |language =zh-cn}}</ref><ref name=" man" >{{cite web |url=http://news.ifeng.com/mainland/detail_2012_08/23/17023183_0.shtml |title=《人民日报》大地副刊主编徐怀谦疑患抑郁症去世 |accessdate=2012年8月23日 |publisher=凤凰网 |archiveurl= http://news.ifeng.com/mainland/detail_2012_08/23/17023183_0.shtml|archivedate=2012年11月5日 |work= |language =zh-cn}}</ref><ref name="tiger " /> 自杀前，[[缅甸|缅甸]]政府宣布取消新闻审查制度。<ref name="bull ">{{cite web  |url=http://www.cenews.eu/?p=36761  |title=人民日报副刊主编徐怀谦自杀网民反响强烈  |accessdate=2012-08-25  |publisher=中欧新闻网  |archiveurl=https://web.archive.org/web/20130327025702/http://www.cenews.eu/?p=36761  |archivedate=2013-03-27  |work=  |language=zh-cn  |dead-url=yes  }}</ref>
