'''徐好'''（{{bd|1995年|10月30日}}），出生於[[江蘇|江蘇]][[鹽城|鹽城]]，中國大陸女演員，毕业于[[北京电影学院|北京电影学院]]。12岁起搬迁到北京生活，学习舞蹈、歌唱等艺术技巧，曾出演《[[顫抖吧，阿部！|顫抖吧，阿部！]]》等电视剧<ref>{{cite web |title=徐好《颤抖吧，阿部》完美表现让网友直呼不舍 |url=http://ent.163.com/17/0907/10/CTNM8FQN00038793.html |publisher=网易娱乐 |accessdate=2018-07-28}}</ref><ref>{{cite web |title=徐好献银幕初吻 敢爱敢恨苏点多 |url=http://ent.sina.com.cn/s/m/2017-08-30/doc-ifykkfat2568355.shtml |publisher=sina |accessdate=2018-07-28}}</ref>。
