'''徐柏園'''（{{bd|1902年|2月8日|1980年|12月2日|catIdx=X徐}}），[[浙江|浙江]][[蘭谿|蘭谿]]人。[[中華民國|中華民國]]著名財經家、金融家，前任[[中華民國財政部|財政部]]部長、[[中華民國中央銀行|中央銀行]]總裁。[[中國|中國]][[中央銀行制度|中央銀行制度]]的開創者。

== 生平 ==
1902年生於[[浙江|浙江]][[金華府|金華府]][[蘭谿縣|蘭谿縣]]。1921年[[杭州|杭州]][[之江大學|之江大學]]附中畢業後，考入[[國立東南大學|國立東南大學]]（後改為[[國立中央大學|國立中央大學]]、[[南京大學|南京大學]]），先在[[南京|南京]]念預科一年，翌年轉到東大分設[[上海商科大學|上海商科大學]]（現[[上海財經大學|上海財經大學]]），1926年畢業。

畢業後初任[[寧波工商|寧波工商]]主任教員。[[國民革命軍北伐|國民革命軍北伐]]時任杭州《[[民國日報|民國日報]]》主筆、[[中國國民黨|中國國民黨]]浙江省黨部書記長，不久到[[廣州|廣州]]任兩廣地質調查所暨交通部秘書。1930年赴[[美國|美國]][[芝加哥|芝加哥]][[北伊利諾大學|北伊利諾大學]]研究財經金融理論。1933年回國，歷任上海電氣公司副總經理、[[中華郵政|郵政儲金匯業總局]]副局長，[[交通銀行|交通銀行]][[北平|北平]]、[[天津|天津]]分行經理。[[中国抗日战争|抗戰]]起擔任第一屆[[國民參政會|國民參政會]]參政員。1938年任[[交通銀行|交通銀行]][[昆明|昆明]]分行經理。1939年擔任[[中華民國中央銀行|中央銀行]]、[[中國銀行|中國銀行]]、[[中國農民銀行|中國農民銀行]]、[[交通銀行|交通銀行]]四行管理處副秘書長。1946年擔任[[中華民國財政部|財政部]]政務次長及四行聯合管理處秘書長，其後出任中央銀行副總裁。1949年赴[[香港|香港]]接掌[[中國國際商業銀行|中國銀行]]董事長。

1950年到[[台灣|台灣]]，任[[台灣區生產事業管理委員會|台灣區生產事業管理委員會]]常務委員兼主任秘書，後任[[台灣銀行|台灣銀行]]董事長。1953年後任[[台灣省政府財政廳|台灣省政府財政廳]]廳長、[[行政院外匯貿易審議委員會|行政院外匯貿易審議委員會]]（外貿會）主任委員。1954年出任[[中華民國財政部|財政部]]部長。1960年出任中央銀行總裁。1962年後再度出任外貿會主委，並兼[[中國國民黨中央委員會|中國國民黨中央委員會]]財務委員會主任委員。1970年出任[[國際貨幣基金會|國際貨幣基金會]]執行董事，翌年出任[[中聯信託|中聯信託]]董事長、[[圓山聯誼會|圓山聯誼會]]理事主席。1980年病逝於[[台北市|台北市]]。

== 事略 ==
徐柏園在大學時代曾代表學校參加1925年[[五卅慘案|五卅慘案]]後的學聯，擔任最高決策的執行委員，處事果決。畢業後投身財經界。1937年抗戰爆發後，進入財經中樞，參與策劃戰時金融政策，草擬[[中國|中國]]第一個外匯管理辦法。1942年[[國民政府|國民政府]]獲[[美國|美國]]巨額貸款，徐柏園力主[[中央銀行|中央銀行]]統一發行貨幣，這一創舉使中央銀行從此成為銀行之銀行。

1950年到台後，負責實際主持[[台灣|台灣]]生產事業，參與貨幣制度改革與恢復重建台灣經濟。作為台灣財經系統骨幹人物，積極主持展開對外貿易，為以後台灣的對外經濟貿易奠定良好基礎；參與制定進口替代戰略，維持外匯收支平衡，穩定物價，整頓財稅，推進經濟發展，頗多建樹。1969年因為[[剝蕉案|剝蕉案]],辭中央銀行總裁、行政院政務委員。

綜觀徐柏園一生，才識魄力超人，往往在遭遇財經困難之時能夠指揮若定，妥為安排，為財經界的歷史性人物。

== 被整 ==
{{fact|徐柏園因為屬於夫人派（[[蔣宋美齡|蔣宋美齡]]）而遭到太子派（[[蔣經國|蔣經國]]）鬥爭，官方製造了「剝蕉案」冤獄來鬥爭，徐柏園屬下被牽連，徐柏園被迫離開權力核心。此事件牽連「香蕉大王」[[吳振瑞|吳振瑞]]，引發日本方面不滿，因此臺灣失去賺取大筆外匯的[[香蕉|香蕉]]外銷日本業務。}}

== 家庭 ==
徐柏園之妻[[陸寒波|陸寒波]]為知名的婦女權益運動者，兒子[[徐小波_(律师)|徐小波]]則活躍於台灣法律、創投界。

==故居==
徐柏園在央行總裁任內時，居住於台北市新生北路三段62巷24號的官邸，該官邸建物由企業家[[丁武始|丁武始]]興建，於1949年落成。1951年賣給中央銀行，供徐柏園居住，1961年轉由[[張導民|張導民]]居住，2001年歸還給中央銀行。

2006年由台北市文化局認定為歷史建築，並由該局發起的「老房子文化運動」計劃媒合進行整修，成為第一個透過該計劃完成整修的歷史建築，2017年5月，利用該官邸所設立的餐廳開業。<ref>[http://www.culture.gov.taipei/frontsite/oldhouse/aboutsubjectAction.do?method=viewContentDetail&subMenuId=8031&siteId=MTE3&contentId=NzUzNg==%20&iscancel=true 老房子文化運動]</ref><ref>[https://udn.com/news/story/7241/2477779 央行總裁老官邸賣便當 會是什麼滋味呢？]</ref>
