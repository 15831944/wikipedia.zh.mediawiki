'''徐平'''，现任[[应急管理部森林消防局|应急管理部森林消防局]]司令员<ref>{{cite web |title=森林防火 紧绷安全这根弦 |url=http://www.gov.cn/xinwen/2018-12/17/content_5349450.htm |website=中国政府网 |date= 2018-12-17 |quote= ……应急管理部森林消防局'''司令员'''徐平介绍}}</ref>，[[中华人民共和国消防救援衔|消防救援总监]]衔。

==经历==
徐平曾任中国人民武装警察部队北京总队副司令员<ref>{{cite web |title=9·3阅兵上的将军领队都有谁？ |url=http://military.people.com.cn/n/2015/0904/c1011-27544860.html |website=人民网-军事频道 |date= 2015-09-04}}</ref>。2015年7月，晋升武警少将警衔。2015年9月3日，在[[纪念中国人民抗日战争暨世界反法西斯战争胜利70周年大会|纪念抗战胜利70周年阅兵]]中，担任武警部队抗战英模方队领队。<ref>{{cite web |author1=澎湃新闻记者 王俊 |title=武警北京总队副司令员徐平等8名警官晋升武警少将警衔 |url=https://www.thepaper.cn/newsDetail_forward_1359597 |website=澎湃新闻 |date= 2015-08-02}}</ref>
2018年9月，徐平转任[[应急管理部森林消防局|应急管理部森林消防局]]司令员<ref>{{cite web |author1=国务院办公厅 |title=国务院办公厅关于调整成立国家森林草原防灭火指挥部的通知 国办发〔2018〕92号 |url=http://www.gov.cn/zhengce/content/2018-09/30/content_5327027.htm |website=中国政府网 |date= 2018-09-30}}</ref>。2018年11月，国务院授予其[[中华人民共和国消防救援衔|消防救援副总监]]衔。<ref>{{cite web|url=http://www.chinasafety.gov.cn/jb/201811/P020181115452145558916.pdf|title=应急管理部简报第1期（总第1期）|accessdate=2018-11-19|date=2018-11-15|publisher=中华人民共和国应急管理部|quote=根据《中华人民共和国消防救援衔条例》规定，国务院总理李克强日前签署命令，授予黄明、王玉普同志总监消防救援衔；授予'''徐平'''、戴建国同志副总监消防救援衔；授予琼色、詹寿旺、张福生、罗永强、魏捍东、周天、亓延军、王新建、闫鹏、彭小国、郭小平同志助理总监消防救援衔。}}</ref>
