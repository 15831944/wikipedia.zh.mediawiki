'''徐起零'''（{{bd|1962年|7月||}}），[[河南|河南]][[淮阳|淮阳]]人，[[中国人民解放军少将|中国人民解放军少将]]，[[第十三届全国人民代表大会代表|第十三届全国人民代表大会代表]]<ref>{{cite news|title=全国人民代表大会|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=200813293511|accessdate=2018-05-29}}</ref>。

==生平==
曾任[[中国人民解放军济南军区|中国人民解放军济南军区]]人民防空办公室副主任、中国人民解放军济南军区作战部副部长等职务。后任[[中国人民解放军陆军第五十四集团军|中国人民解放军陆军第五十四集团军]]参谋长<ref>{{cite news|author1=陶社兰|title=济南军区副参谋长石正露升任54集团军军长|url=http://mil.sohu.com/20150327/n410423357.shtml|accessdate=2017-01-15|agency=搜狐网中国新闻网}}</ref>。2016年，任[[中国人民解放军中部战区陆军|中国人民解放军中部战区陆军]]副司令员<ref>{{cite news|author1=王俊|title=陆军第54集团军原参谋长徐起零少将任中部战区陆军副司令员|url=http://www.thepaper.cn/newsDetail_forward_1505043|accessdate=2017-01-15|agency=澎湃新闻|date=2016-07-27}}</ref>。2017年3月，任[[中国人民解放军陆军第七十九集团军|中国人民解放军陆军第七十九集团军]]军长<ref>{{Cite web |url = http://china.caixin.com/2017-07-18/101114293.html |title = 多个集团军主官亮相 徐起零任79军军长 |author =  |date =  2017-07-18|publisher =  财新网|language =  |accessdate =  |quote =  }}</ref>。2018年12月，升任[[中国人民解放军东部战区陆军|东部战区陆军]]司令员<ref>{{cite web |title=接棒秦卫江 徐起零出任东部战区陆军司令 |url=http://news.sina.com.cn/c/2019-01-03/doc-ihqfskcn3708593.shtml |website=新浪 |accessdate=2019-01-03 |date=2019-01-03}}</ref>。
