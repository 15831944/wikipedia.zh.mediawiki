'''徐扬生'''，[[中国|中国]]机械及自动化专家，[[中国工程院|中国工程院]][[院士|院士]]。[[浙江省|浙江省]][[绍兴市|绍兴市]]人。

==生平==
徐扬生历任[[香港中文大学|香港中文大学]][[副教授|副教授]]、[[教授|教授]]、讲座教授。
* 1982年，获得[[浙江大学|浙江大学]][[学士|学士]][[学位|学位]]。
* 1984年，获得[[浙江大学|浙江大学]][[硕士|硕士]]学位。
* 1989年，获得[[美国|美国]][[宾夕法尼亚大学|宾夕法尼亚大学]][[博士|博士]]学位。
* 1989年至1997年，担任[[美国|美国]][[卡耐基梅隆大学|卡耐基梅隆大学]]计算机学院机器人研究所研究员，领域为空间机器人研究。
* 1997年，赴香港香港中文大学任教。
* 1997年至2004年，担任香港中文大学自动化系系主任。
* 1998年至2004年，徐扬生是香港研究资助局工程组成员。
* 2003年至2006年，出任香港生产力促进局董事。
* 2004年至2006年，担任中国国家高科技航天航空领域委员会顾问。
* 2006年至2008年，担任香港中文大学校长助理。
* 2006年至今，担任[[中国科学院|中国科学院]]香港中文大学[[深圳|深圳]]先进集成技术研究所所长
* 2006年至今，担任中国科学院深圳先进技术研究院副院长
* 2008年至2013年，担任香港中文大学协理副校长
* 2008年至今，出任深圳市决策咨询委员会顾问。
* 2008年3月17日，受聘担任[[西安交通大学|西安交通大学]]兼职教授。<ref name="徐扬生 西安交大">{{cite web |url=http://nanyang.xjtu.edu.cn/eec/new_ee/content.php?articleid=1543 |title=香港中文大学徐扬生院士受聘我校兼职教授 |publisher=[[西安交通大学|西安交通大学]] |date=2008-03-18 |accessdate=2010年4月7日 |language=zh }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
* 2010年5月28日，受聘担任[[东方电气集团公司中央研究院|东方电气集团公司中央研究院]]特聘专家。<ref name="徐扬生 东方电气">{{cite web |url=http://www.dongfang.com/dec/news/news_display.php?&NID=4598&Language=GB |title=中国工程院院士徐扬生教授受聘中央研究院特聘专家 |publisher=[[东方电气集团|东方电气集团]] |date=2010-06-01 |accessdate=2011年2月11日 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20110603082628/http://www.dongfang.com/dec/News/News_Display.php |archivedate=2011年6月3日 }}</ref>
* 2013年至今，出任[[香港中文大学（深圳）|香港中文大学（深圳）]]校长。<ref name="徐扬生 香港中文大学（深圳）">{{cite web |url=http://www.cuhk.edu.cn/simp/xuys.htm |title=存档副本 |accessdate=2014-04-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20140413144517/http://www.cuhk.edu.cn/simp/xuys.htm |archivedate=2014-04-13 }}香港中文大学（深圳）校长 徐扬生 教授</ref>

==学术荣誉及担当==
徐扬生是中国自动化学会理事（2002年当选至今），并是国际欧亚科学院院士（2001年当选），国际宇航科学院通讯院士（2007当选），并于2003年当选国际电子电器工程师协会会士（[[IEEE|IEEE]] Fellow）。2007年，徐扬生入选中国工程院院士(but there was an alleged fraud or fabrication, c.f. http://www.xys.org/xys/ebooks/others/science/dajia6/xuyangsheng.txt )。

徐扬生曾在1997年至2002年期间担任IEEE机器人与自动化学报副主编。
