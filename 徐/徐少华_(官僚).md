'''徐少华'''（{{bd|1958年|1月|||catIdx=X}}），[[广东|广东]][[广州|广州]]人。1974年12月参加工作，1984年3月加入中国共产党。中山大学历史系历史专业毕业，中山大学经济系政治经济学专业研究生学历。曾任[[中共广东省委|中共广东省委]]常委、常务副省长，广东省人大常委会副主任<ref name="fish" >{{cite web |url=http://www.gd.gov.cn/govpub/ld/201202/t20120229_157312.htm |title=广东省人民政府首页 > 政务公开 > 省政府领导：徐少华 |accessdate= 2013|publisher=广东省人民政府 |work= |language=zh-hans}}</ref><ref>{{cite web|url=http://politics.people.com.cn/n/2013/0529/c70731-21665214.html|title=广东省政府公布领导同志最新分工 李春生履新|accessdate=2013|publisher=人民网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160303232546/http://politics.people.com.cn/n/2013/0529/c70731-21665214.html|archivedate=2016年3月3日|df=}}</ref><ref name="bull" >{{cite web |url=http://cpc.people.com.cn/gbzl/html/121000466.html |title=徐少华 |accessdate=2013 |publisher=中国共产党新闻网 |work= |language=zh-hans}}</ref>。

==生平==
1974年12月，任[[广东省|广东省]][[肇庆地区|肇庆地区]]石牌大队知青。1976年12月，任肇庆矽钢片厂工人。1979年9月，在[[中山大学|中山大学]]历史系历史专业学习。1983年7月，任中山大学团委组织部长。1985年7月，任中山大学团委副书记（副科级，其间：1985年9月－1987年7月，在中山大学政治经济学专业研究生班学习）。1988年2月，任中山大学党委宣传部秘书（正科级）。1988年5月，任中山大学综合服务总公司副总经理（享受副处级待遇）。

1989年4月，任[[封开县|封开县]]副县长。1992年4月，任中共封开县委常委、副县长。1992年6月，任中共封开县委常委、副县长、代县长、县长。1994年10月，任中共封开县委书记。1996年3月，任[[肇庆市|肇庆市]]副市长。1998年11月，任中共[[湛江市|湛江市]]委常委、常务副市长。2001年9月，任中共湛江市委副书记、常务副市长。2002年3月，任中共湛江市委副书记、市长。2005年8月，任中共湛江市委书记。2008年4月，任[[中共广东省委|中共广东省委]]副秘书长、办公厅主任。2008年9月，任中共广东省委常委、秘书长、办公厅主任。2012年2月，任中共广东省委常委、[[广东省人民政府|广东省人民政府]]副省长。2012年5月，任中共广东省委常委、常务副省长<ref>{{cite web |url=http://news.southcn.com/g/2012-02/28/content_39068917.htm |title=省十一届人大常委会第三十二次会议召开 任命徐少华为省人民政府副省长 |accessdate=2012 |publisher=南方网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://www.gd.gov.cn/gdgk/gdyw/201207/t20120719_164322.htm |title=徐少华到东莞调研：攻坚克难 奋力推进城轨交通项目 |accessdate=2012 |publisher=广东省人民政府 |work= |language=zh-hans}}</ref>。2017年1月，辞去广东省副省长<ref>{{cite news|title=黄龙云辞任广东省人大常委会主任，徐少华辞任副省长|url=http://www.thepaper.cn/newsDetail_forward_1605727|accessdate=2017-02-06|work=澎湃新闻}}</ref>，当选[[广东省人大常委会|广东省人大常委会]]副主任<ref>{{cite news|title=马兴瑞当选广东省省长，李玉妹当选广东省人大常委会主任|url=http://www.thepaper.cn/newsDetail_forward_1606616|accessdate=2017-02-06|work=澎湃新闻}}</ref>。
