'''徐佳青'''{{BD|1967年|10月22日|catIdx=}}，[[中華民國|中華民國]][[臺北市|臺北市]]人，[[民主進步黨|民主進步黨]][[政治人物|政治人物]]，曾任兩屆民進黨中央常務委員會委員，前任[[臺北市議會|臺北市議會]][[大安區_(臺北市)|大安區]]、[[文山區|文山區]]選區議員。徐佳青是前民進黨發言人，現為民進黨副秘書長，內政部長[[徐國勇|徐國勇]]為其親叔叔。

== 學歷 ==
台灣[[中山女高|中山女高]]、[[台灣大學|台灣大學]]農藝系學士、[[德國|德國]][[馬爾堡大學|馬爾堡大學]]社會學研究所、[[陽明大學|陽明大學]]衛生福利政策研究所畢業。

== 經歷 ==
由於[[彭婉如|彭婉如]]事件踏入政壇，為臺北市議會第9到11屆議員。

2014年台北市議員選舉，徐佳青宣布不再連任，交棒助理[[簡舒培|簡舒培]]<ref>{{Cite news|url=https://tw.news.yahoo.com/%E8%BD%89%E6%8F%9B%E8%B7%91%E9%81%93-%E5%BE%90%E4%BD%B3%E9%9D%92-%E7%B0%A1%E4%BD%99%E6%99%8F%E4%B8%8D%E9%80%A3%E4%BB%BB-120050333.html|title=轉換跑道! 徐佳青、簡余晏不連任|language=zh-tw|publisher=民視|author=|date=2014-03-15|accessdate=2014-09-13}}</ref>。

== 家庭 ==
丈夫為美國籍公民Mr. Bo Tedards，出身於[[耶魯大學|耶魯大學]]政治學學士、[[塔夫斯大學|塔夫斯大學]]弗萊徹學院法律與外交的碩士，居住在台灣已有20年，過去一直在任職於[[台灣人權協會|台灣人權協會]]、國家政策研究機構、[[臺北時報|臺北時報]]、中華民國外交部研究和規劃委員會顧問。2004年到2013年，他曾在[[臺灣民主基金會|臺灣民主基金會]](TFD) 擔任亞洲民主化世界論壇（WFDA）的協調員。2007年至2010年，他還兼任TFD的國際合作部主任。目前在[[臺灣民主基金會|臺灣民主基金會]]工作<ref name="徐佳青老公">{{Cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20110930/33705022/|title= 徐佳青老公 請育嬰假遭降職 民主基金會涉性別歧視 罰5萬|language=zh-tw|publisher=《蘋果日報》|author=黃揚明|date=2011-09-30|accessdate=2014-09-13}}</ref>，同時也是是[[國際特赦組織台灣分會|國際特赦組織台灣分會]]的主任。
