== 生平 ==
徐仁壽祖籍[[廣東|廣東]][[五華縣|五華]]<ref>{{cite web |quote=徐公仁壽，聖名伯多祿，原籍廣東五華，道良之子，庚娣、肖娟之夫，梅英、瑞蘭、應祥、振祥、家祥、聲祥、萬祥、世祥、謙祥、平英、信祥、傑英、建祥、友祥、達祥、廣祥、婉貞、總祥、增英、念祥、溢祥、義祥、參祥、通祥、念伍、陸祥、多喇之父。|url=https://www.facebook.com/hongkongcemetery/photos/a.350436585519718/393345164562193/?type=3&theater |title=墳場研究 Cemetery Study 在長沙灣天主教聖辣法厄爾墳場 |website=www.facebook.com |date=2019-02-19 |accessdate=2019-02-20}}</ref>，為粘坑十二世[[徐道發|徐道發]]次子。祖父[[徐厚祝|徐厚祝]]是基督教會長老，父親道發為教師。徐仁壽幼年隨父到香港生活，入讀[[聖若瑟書院|聖若瑟書院]]，其後篤信[[天主教|天主教]]<ref name=wypsa>[http://www.wahyan-psa.org/memories/mem_history001.htm 華仁創辦人徐仁壽先生簡史]</ref>。畢業後到廣東[[梅縣|梅縣]]任教中學，四年後回港到英華書院及母校聖若瑟書院任教。1919年，徐仁壽獨自創立[[香港華仁書院|華仁書院]]<ref>{{cite web |quote=徐公仁壽，聖名伯多祿，原籍廣東五華，道良之子，庚娣、肖娟之夫，梅英、瑞蘭、應祥、振祥、家祥、聲祥、萬祥、世祥、謙祥、平英、信祥、傑英、建祥、友祥、達祥、廣祥、婉貞、總祥、增英、念祥、溢祥、義祥、參祥、通祥、念伍、陸祥、多喇之父。|url=https://www.facebook.com/hongkongcemetery/photos/a.350436585519718/393345164562193/?type=3&theater |title=墳場研究 Cemetery Study 在長沙灣天主教聖辣法厄爾墳場 |website=www.facebook.com |date=2019-02-19 |accessdate=2019-02-20}}</ref>，其後邀得舊同事[[林海瀾|林海瀾]]一同辦學；到1924年又到九龍開辦[[九龍華仁書院|分校]]。

徐仁壽在辦學事業巔峰的時候突然退休，把兩家華仁書院的辦學權，轉讓給天主教修會[[耶穌會|耶穌會]]。他宣告退休後，便轉往[[北婆羅洲|北婆羅洲]]經營[[橡膠|樹膠]]業。徐仁壽妣陳氐、關氐，生十九子八女：子為應祥、振祥、家祥、聲祥、萬祥、世祥、謙祥、友祥、義祥、建祥、達祥、廣祥、總祥、溢祥、森祥，通祥、陸祥及信祥。<ref>{{cite web |quote=徐公仁壽，聖名伯多祿，原籍廣東五華，道良之子，庚娣、肖娟之夫，梅英、瑞蘭、應祥、振祥、家祥、聲祥、萬祥、世祥、謙祥、平英、信祥、傑英、建祥、友祥、達祥、廣祥、婉貞、總祥、增英、念祥、溢祥、義祥、參祥、通祥、念伍、陸祥、多喇之父。|url=https://www.facebook.com/hongkongcemetery/photos/a.350436585519718/393345164562193/?type=3&theater |title=墳場研究 Cemetery Study 在長沙灣天主教聖辣法厄爾墳場 |website=www.facebook.com |date=2019-02-19 |accessdate=2019-02-20}}</ref>三子[[徐家祥|徐家祥]]是香港首位華人政府行政官。<ref>{{cite web |quote=徐公仁壽，聖名伯多祿，原籍廣東五華，道良之子，庚娣、肖娟之夫，梅英、瑞蘭、應祥、振祥、家祥、聲祥、萬祥、世祥、謙祥、平英、信祥、傑英、建祥、友祥、達祥、廣祥、婉貞、總祥、增英、念祥、溢祥、義祥、參祥、通祥、念伍、陸祥、多喇之父。|url=https://www.facebook.com/hongkongcemetery/photos/a.350436585519718/393345164562193/?type=3&theater |title=墳場研究 Cemetery Study 在長沙灣天主教聖辣法厄爾墳場 |website=www.facebook.com |date=2019-02-19 |accessdate=2019-02-20}}</ref><ref name=wypsa /> 徐仁壽姊清和（1886-1985），妻崇謙堂村彭氏樂三。崇謙堂村百年古宅「乾德門」乃彭樂三與友人凌啟蓮所建。

==崇謙堂村故居==
徐仁壽在[[粉嶺|粉嶺]][[崇謙堂村|崇謙堂村]]的故居、位於崇謙堂對面的「[[石廬|石廬]]」，已被[[香港特區政府|香港特區政府]]列為二級文物。
