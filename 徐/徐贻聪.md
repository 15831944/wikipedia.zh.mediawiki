'''徐贻聪'''（{{bd|1938年||||catIdx=徐}}），[[中华人民共和国|中华人民共和国]]政治人物、外交官。曾担任外交部美大司、拉美司副司长，新华社驻巴拿马分社记者和世界知识出版社社长、党委书记。<ref>{{cite news |language = zh-hans |author = 胡铭 |url = http://bjyouth.ynet.com/3.1/1211/17/7619715.html |title = 往事亲历 徐贻聪：我参与组建外交部拉美司 |publisher = 北京青年报 |pages = C1 |date = 2012-11-17 |accessdate = 2014-11-13 |deadurl = yes |archiveurl = https://web.archive.org/web/20141113060708/http://bjyouth.ynet.com/3.1/1211/17/7619715.html |archivedate = 2014-11-13 }}</ref>

1991年，接替[[王钢华|王钢华]]，担任[[中国驻厄瓜多尔大使列表|中华人民共和国驻厄瓜多尔大使]]。<ref>{{cite web| title = 驻厄瓜多尔共和国历任大使 |url = http://www.fmprc.gov.cn/mfa_chn/ziliao_611306/wjrw_611324/2167_611334/2174_611348/t9284.shtml |publisher = 中华人民共和国外交部 |accessdate = 2014-11-13}}</ref>1993年，接替[[陈久长|陈久长]]，担任[[中国驻古巴大使列表|中华人民共和国驻古巴大使]]。<ref>{{cite web| title = 驻古巴历任大使 |url = http://www.fmprc.gov.cn/mfa_chn/ziliao_611306/wjrw_611324/2167_611334/2174_611348/t9286.shtml |publisher = 中华人民共和国外交部 |accessdate = 2014-11-13}}</ref>1997年，接替[[汤永贵|汤永贵]]，担任[[中国驻阿根廷大使列表|中华人民共和国驻阿根廷大使]]。<ref>{{cite web| title = 驻阿根廷共和国历任大使 |url = http://www.fmprc.gov.cn/mfa_chn/ziliao_611306/wjrw_611324/2167_611334/2174_611348/t9278.shtml |publisher = 中华人民共和国外交部 |accessdate = 2014-11-13}}</ref>2000年，由[[张沙鹰|张沙鹰]]接任。
