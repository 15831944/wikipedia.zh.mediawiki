'''徐俐'''（{{Bd|1961年||||catIdx= X徐俐}}），[[湖南|湖南]][[临湘|临湘]]人，前[[中国中央电视台中文国际频道|中国中央电视台中文国际频道]]《[[中國新聞_(中國中央電視台節目)|中国新闻]]》及《今日关注》[[主持人|主持人]]。曾有过两段婚姻，现任丈夫为《[[北京青年报|北京青年报]]》编辑张天蔚。她和前夫育有一子，另著有多部书籍。

== 职业生涯 ==
1978年，徐俐考入[[长沙人民广播电台|长沙人民广播电台]]。<ref name="msa " /> 1986年，徐俐被调入[[长沙电视台|长沙电视台]]。<ref name="msa " >{{cite web |url= http://money.591hx.com/article/2010-12-07/0000135913s_2.shtml|title=从地方台跳槽央视后大红大紫的女主播 |accessdate= 2010年|work= |publisher=华讯 |language =zh-cn}}</ref>

1992年，徐俐从[[长沙电视台|长沙电视台]]应聘上[[中国中央电视台|中国中央电视台]]。<ref name=" hsa" >{{cite web |url=http://media.people.com.cn/GB/137684/11463287.html |title=知性女主播徐俐：31岁方知自己美丽 |accessdate= 2010年|work= |publisher=人民网 |language =zh-cn}}</ref>后任中国中央电视台中文国际频道新闻组组长。<ref name="msa " />

2017年4月2日，徐俐发表文章，宣布离开《中国新闻》主播岗位<ref>{{cite web|authorlink1=徐俐|title=徐俐：在《中国新闻》的最后一个播出日|url=http://mp.weixin.qq.com/s/fOPvKktuIxFg8Q1c5pQA7A|accessdate=2017-04-02|date=2017-04-02}}</ref>。

== 个人生活 ==
徐俐的丈夫名叫“张天蔚”，北京人，父母都是[[中国|中国]][[中央美术学院|中央美术学院]]教授、雕塑家。<ref name=" hsa" /> 两人日常夫妻生活甜蜜，张天蔚亲切地称呼徐俐为“小可爱”、“小东西”。<ref name=" hsa" />
