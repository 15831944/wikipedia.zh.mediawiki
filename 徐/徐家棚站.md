==车站==
徐家棚站规模较大，大致可以分成三个部分：①5号线车站位于和平大道与团结路交叉口，沿和平大道方向设置，为地下两层车站；②8号线车站位于和平大道原三三七铁路货场内，为地下三层侧式站台车站，车站长约575m；③7号线车站位于秦园路与和平大道交叉路口南侧，沿秦园路布设，为地下四层岛式站台车站，地下一层为物业开发，地下二层为公路隧道层，地下三层为地铁车站站厅及设备夹层，地下四层为地铁车站站台层。

徐家棚站是[[武汉地铁|武汉地铁]]网络中第3个三线换乘车站，亦是[[武昌|武昌]]第1个三线换乘车站，三线车站统一设计。由于7号线站台距离其他两线较远，约400至500米，故本站在付费区换乘通道设立了四部自动步行步道，使得本站成为武汉地铁网络中设立自动步道最多的车站。该站换乘模式较为复杂，8号线客流通过地下三层的两部楼梯上至5号线站台层付费区；5、7号线的换乘通过5号线的站台层接入5、7号线的付费区换乘通道实现付费区换乘；5号线通向大里程端的付费区换乘通道设置了四部自动步道及通向5号线站厅层的疏散楼梯间，通过闸机进入非付费区。<ref>魏重丽．武汉地铁徐家棚换乘站设计方案研究[J]. 铁道工程学报，2012年，(6)：109-113.</ref><ref>{{Cite web |url =http://news.cjn.cn/sywh/201712/t3114908.htm|title =把东湖绿道、芦苇荡放进地铁站，8号线有点小清新|author =张晟, 袁永华|date =2017-12-04|publisher =长江网-长江日报|language = 中文|accessdate =  |quote =  }}</ref>

值得注意的是，在轨道交通5号线开通之前，'''徐家棚站暂时无法实现轨道交通7号线和8号线之间的站内换乘'''。<ref>{{Cite web |url =http://news.cnhubei.com/xw/wuhan/201809/t4159946.shtml|title =武汉首座4层地铁车站亮相，系3条线换乘站，金秋芦苇作背景|author = 李永刚; 袁永华|date =2018-09-05|publisher = 武汉发布  }}</ref>欲在7号线和8号线之间换乘的乘客可选择出站换乘。
