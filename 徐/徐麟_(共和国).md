'''徐麟'''（{{bd|1963年|6月||}}），男，[[汉族|汉族]]，[[上海市|上海市]]人，1982年9月加入[[中国共产党|中国共产党]]。[[上海师范大学|上海师范大学]]毕业，在职研究生，工商管理硕士。现任[[中共中央宣传部|中共中央宣传部]]副部长，[[国务院新闻办公室|国务院新闻办公室]]主任。[[中国共产党第十九届中央委员会委员列表|中共第十九届中央委员]]。

== 生平 ==
历任上海市[[南汇区|南汇县]][[华东师范大学附属周浦中学|周浦中学]]教师。1990年，任中共上海市南汇县委常委、政策研究室主任。1992年，任南汇县副县长。1995年，任中共上海[[嘉定区|嘉定区]]委副书记，同年任[[西藏自治区|西藏自治区]][[日喀则市|日喀则]]地委副书记、上海市第一批援藏干部联络组组长。1997年，任上海市农工商（集团）总公司党委副书记、副董事长、总经理。2000年，任上海市农工商集团党委书记、董事长<ref>{{cite news|title=原农工商集团董事长徐麟调任中央网信办副主任|url=http://www.shnczq.org/detail.php?id=2016|accessdate=2015-08-05|agency=中央网信网|date=2015-08-05}}</ref>。2003年3月，任中共上海市民政局党委书记、局长。2007年2月，任上海市农业委员会主任。2008年2月，任中共[[浦东新区|浦东新区]]区委书记。2013年5月，任[[中国共产党上海市委员会|中共上海市委]]常委、宣传部部长。

2015年6月，任[[中央网络安全和信息化领导小组|中央网络安全和信息化领导小组]]办公室副主任<ref>{{cite news|title=董云虎任上海市委宣传部部长 看看前几任部长去哪儿了？|url=http://finance.chinanews.com/gn/2015/08-05/7448348.shtml|accessdate=2015-08-05|agency=中国新闻网|date=2015年8月5日}}</ref><ref>{{cite news|title=上海市委原常委徐麟调任中央网信办副主任|url=http://news.ifeng.com/a/20150619/44011914_0.shtml#_zbs_baidu_bk|accessdate=2015-08-05|agency=凤凰网|date=2015年6月19日}}</ref>。2016年6月，任[[中共中央宣传部|中共中央宣传部]]副部长、[[中央网络安全和信息化领导小组|中央网络安全和信息化领导小组]]办公室主任<ref>{{cite news|url=http://news.xinhuanet.com/politics/2016-06/29/c_1119135194.htm|title=江西省委等地方和部门主要负责同志职务调整|publisher=[[新华社|新华社]]|date=2016-06-29|accessdate=2016-06-29}}</ref>。2016年7月，兼任[[国务院新闻办公室|国家互联网信息办公室]]主任、[[国务院新闻办公室|国务院新闻办公室]]副主任<ref>{{cite news|title=国务院任免国家工作人员：刘慧任国家民族事务委员会副主任|url=http://www.thepaper.cn/newsDetail_forward_1500919|accessdate=2016-07-20|newspaper=澎湃新闻}}</ref>。

2018年7月31日，中央网信办召开干部大会，中央组织部副部长周祖翼同志宣布中央决定，中宣部副部长庄荣文同志任中央网络安全和信息化委员会办公室主任，徐麟同志不再担任中央网络安全和信息化委员会办公室主任职务，另有任用。<ref>{{cite web |author1=联合早报 |title=庄荣文任中央网信办主任 徐麟不再担任 |url=https://www.zaobao.com.sg/realtime/china/story20180801-879788}}</ref>

2018年8月21日，国务院任免国家工作人员：任命徐麟为国务院新闻办公室主任，同时免去其国家互联网信息办公室主任职务。<ref>{{cite web |author1=人力资源和社会保障部|title=国务院任免国家工作人员（2018年8月21日）|url=http://www.mohrss.gov.cn/SYrlzyhshbzb/sydqjdxs/SYguowuyuanrenmian/201808/t20180821_299601.html}}</ref>
