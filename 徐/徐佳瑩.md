'''徐佳瑩'''（{{bd|1984年|12月20日|catIdx=Hsu徐}}），生於臺灣[[花蓮縣|花蓮縣]]，成長於臺灣[[臺中市|臺中市]]，籍貫[[四川省_(中華民國)|四川省]][[簡陽市|簡陽縣]]<ref>{{Cite video|url=http://v.iqilu.com/sdws/sdcqyl/201708/13/4450036.html|title=超强音浪：主持人嘉宾即兴说唱大比拼 各地美食大联盟|medium=电视|publisher=[[山东广播电视台|山东广播电视台]]|date=2017-08-13|accessdate=2017-10-28|time=00:03:00}}</ref><ref>{{Cite news|url=http://news.ifeng.com/gundong/detail_2013_07/30/28049194_0.shtml|title=来蓉大谈“理想人生” 徐佳莹自曝祖籍简阳|website=凤凰网|publisher=成都晚报|author1=钟茜|date=2013-07-30|accessdate=2017-10-28}}</ref>，[[華語流行音樂|華語流行音樂]][[創作歌手|創作女歌手]]。

2008年，徐佳瑩參加[[中國電視公司|中視]][[超級星光大道_(第三屆)|第三屆《超級星光大道》]]歌唱選秀比賽，并於同年8月15日在總決賽中以自創詞曲作品奪得「總冠軍」而出道。2009年5月29日，徐佳瑩發行首張個人專輯《[[LaLa首張創作專輯|LaLa首張創作專輯]]》，迄今已正式發行5張個人專輯及2張演唱會專輯。

2010年，徐佳瑩憑藉自己的首張個人專輯於[[第21屆金曲獎|第21屆金曲獎]]榮獲[[最佳新人獎_(金曲獎)|最佳新人獎]]，成為第一位從《超級星光大道》比賽畢業後獲得金曲獎的星光幫歌手。2015年，徐佳瑩以專輯《[[尋人啟事_(專輯)|尋人啟事]]》入圍[[第26屆金曲獎|第26屆金曲獎]]六個獎項。

2016年1月，徐佳瑩參加[[湖南衛視|湖南衛視]][[我是歌手_(第四季)|《我是歌手》第四季]]，成為八位首發歌手中的一位，最終順利晉級總決賽并走完所有賽程，也使得她在中國大陸人氣大增，并於賽后獲邀為多部影視作品獻唱主題曲。

2018年，徐佳瑩憑藉專輯《[[心裡學|心裡學]]》於[[第29屆金曲獎|第29屆金曲獎]]榮獲[[最佳國語專輯獎_(金曲獎)|最佳國語專輯獎]]與[[最佳國語女歌手獎_(金曲獎)|最佳國語女歌手獎]]兩項大獎<ref name="gma29">{{Cite news|url=http://www.cna.com.tw/news/gpho/201806240002-1.aspx|title=金曲29 徐佳瑩先封歌后再奪最佳國語專輯|publisher=中央社|date=2018-06-23|accessdate=2018-06-23}}</ref>。

== 早年生活 ==
徐佳瑩的父親為退役[[中華民國空軍|中華民國空軍]][[地勤|地勤]]軍官，母親為[[幼稚園|-{zh-cn:幼儿园;zh-hk:幼稚園;zh-tw:幼稚園;zh-sg:幼稚园;}-]]老師，她在兄弟姊妹之中排行第三。

== 出道歷程 ==
徐佳瑩國中曾參加校內歌唱比賽獲得第二名，專科時期為管樂社打擊兼指揮，曾在中山醫學院附屬醫院實習，台中榮總工作。大學期間曾經參加「[[校園金曲獎|校園金曲獎]]」比賽，獲得「女生個人演唱組」演唱獎、「詞曲創作組」創作獎，曾參與四張校園金曲合輯<ref>{{cite web|url=http://www.my-music.com.tw/history/index-history.htm|title=校園金曲獎 活動回顧|deadurl=yes|archiveurl=https://web.archive.org/web/20080630210608/http://www.my-music.com.tw/history/index-history.htm|archivedate=2008-06-30}}</ref>。

2008年徐佳瑩參加[[中國電視公司|中視]][[超級星光大道_(第三屆)|《超級星光大道》第三屆]]比賽，表現優異<ref>{{cite web|publisher=中時電子報|url=http://news.yam.com/chinatimes/entertain/200803/20080317884900.html|title=星光３班選20強 美國區４名過關 徐佳瑩23分成資優生|date=2008-03-17}}</ref>，在一開始她就說拿到獎金要開星光雞排店，而且已經發明店歌，在第六集時，以自創曲「《[[我身騎白馬|我身騎白馬]]》抒情版」拿下25分滿分的成績，是超級星光大道開播以來，最快得到滿分的參賽者<ref>{{cite web|publisher=中時電子報|url=http://news.yam.com/chinatimes/entertain/200802/20080225458241.html|title=賴聖恩領軍台灣20強 徐佳瑩獲評審滿分激賞|date=2008-02-25}}</ref><ref>{{cite news|publisher=中央社|title=超級星光大道美國區參賽者 今晚加入戰局|date=2008-03-06}}</ref>。8月15日，徐佳瑩在總決賽中獲得[[超級星光大道_(第三屆)|《超級星光大道》]]總冠軍<ref>{{cite web|publisher=中國時報|title=《星光3》自創曲勝出 星光三徐佳瑩奪冠|url=http://showbiz.chinatimes.com/Chinatimes/ExteriorContent/Showbiz/Showbiz-Page/0,4434,content+110511-1+112008081600151,00.html|date=2008-08-16|deadurl=yes|archiveurl=https://web.archive.org/web/20080817120247/http://showbiz.chinatimes.com/Chinatimes/ExteriorContent/Showbiz/Showbiz-Page/0,4434,content+110511-1+112008081600151,00.html|archivedate=2008-08-17}}</ref><ref>{{cite web|publisher=聯合報|title=徐佳瑩摘三班星冠 抱回250萬元|url=http://udn.com/NEWS/ENTERTAINMENT/ENT1/4474755.shtml|date=2008-08-16}}</ref><ref>{{cite web|publisher=自由時報|title=星光三班 徐佳瑩奪冠|url=http://www.libertytimes.com.tw/2008/new/aug/16/today-show1.htm|date=2008-08-16|deadurl=yes|archiveurl=https://web.archive.org/web/20080818222157/http://www.libertytimes.com.tw/2008/new/aug/16/today-show1.htm|archivedate=2008-08-18}}</ref>，繼而獲得發片機會，成為歌手。

== 音樂風格 ==
徐佳瑩的歌詞創作，以細膩且獨特的方式敘事，在歌曲中融入台灣的歌仔戲元素，創造一種似曾相識，卻又耳目一新的感覺。靈活運用國語和閩南語文辭，將兩種不同語言文字邏輯蹦湊一起，自然而又順暢。曲式上，主歌節奏步調通常較快，副歌則多為明朗、抒情且較好記的旋律。在伴奏及和聲的部份，她試圖創造出獨有的和絃進行方式，轉調方式與現有流行音樂多有不同。

== 歌唱經歷 ==
徐佳瑩曾向[[蘇通達|蘇通達]]學習，有師徒之稱。《身騎白馬》是兩人合作的作品。

2009年5月29日，徐佳瑩發行個人首張創作專輯《[[LaLa首張創作專輯|LaLa首張創作專輯]]》。此專輯在[[第21屆金曲獎|第21屆金曲獎]]流行音樂類共有6項入圍，同屆入圍總數目僅次於[[張惠妹|張惠妹]]，入圍量在所有報名的台灣專輯中居亞，素質備受肯定。此專輯入圍獎項包括：最佳年度歌曲獎（身騎白馬）、最佳國語專輯獎（徐佳瑩 LALA創作專輯）、最佳作曲人獎（徐佳瑩、蘇通達╱身騎白馬）、最佳新人獎（徐佳瑩）、最佳編曲人獎（蘇通達╱身騎白馬）和最佳單曲製作人獎（蘇通達╱身騎白馬）<ref>{{Cite web|url=http://stars.udn.com/star/StarsContent/Content25434/|title=2010金曲獎入圍公佈}}</ref><ref>{{Cite web|url=http://stars.udn.com/star/StarsContent/Content25433/#27|title=2010第21屆金曲獎入圍名單}}</ref>。

2010年6月26日，徐佳瑩於第21屆金曲獎獲頒最佳新人獎<ref>{{Cite web|url=http://forums.chinatimes.com/showbiz/2010GMA/index.htm|title=2010第21屆金曲獎得獎名單}}</ref>，也是第一位從[[超級星光大道_(第三屆)|《超級星光大道》]]比賽畢業後獲得[[金曲獎|金曲獎]]的星光幫歌手。同年9月3日，徐佳瑩發行第二張國語專輯《[[極限_(專輯)|極限]]》，創作及演唱均更加純熟。

2012年6月6日，徐佳瑩發行第三張國語專輯《[[理想人生|理想人生]]》。發行後亦屢獲不少海內外獎項，如[[中華音樂人交流協會|中華音樂人交流協會]]2012年度十大優良專輯暨單曲雙榜。2013年，首度入圍[[第24屆金曲獎|第24屆金曲獎]]『[[最佳國語女歌手獎_(金曲獎)|最佳國語女歌手獎]]』的提名。

2014年6月13日，徐佳瑩發行第四張國語專輯《[[尋人啟事_(專輯)|尋人啟事]]》，並於2015年3月21日舉辦《日全蝕》[[臺北小巨蛋|臺北小巨蛋]]演唱會，是徐佳瑩的首場小巨蛋個唱。同年，徐佳瑩以第四張專輯《尋人啟事》入圍[[第26屆金曲獎|第26屆金曲獎]]最佳年度歌曲獎、最佳國語專輯獎、最佳國語女歌手獎、最佳作曲人獎、最佳作詞人獎以及最佳編曲人獎等六項大獎。

2016年1月，徐佳瑩參加[[湖南衛視|湖南衛視]][[我是歌手_(第四季)|《我是歌手》第四季]]，成為八位首發歌手中的一位。初登場即演唱其代表作《失落沙洲》打敗眾多強敵，獲得首集冠軍，且在參賽期間表現亮眼，獲得多位原唱者的肯定，演唱歌曲點閱率居參賽歌手之冠，最終順利晉級總決賽並走完所有賽程。賽後獲邀為多部影視作品獻唱主題曲，並於同年7月在[[北京市|北京]]、[[上海市|上海]]、[[深圳市|深圳]]等城市舉辦《日全蝕》大型巡演。

2017年4月22日，徐佳瑩於臺北小巨蛋舉辦[[是日救星巡迴演唱會|《是日救星》巡迴演唱會]]<ref>{{Cite web |url=https://www.kkbox.com/tw/tc/column/live_reviews-15-413-1.html|title=「讓我的音樂拯救你們」—徐佳瑩 是日救星演唱會|accessdate=2017-12-04|date=2017-04-27|website=KKBOX}}</ref>，並於同年5月起先後在[[高雄市|高雄]]、[[香港|香港]]、上海、[[馬來西亞|馬來西亞]]等地舉辦《是日救星》大型巡演。同年12月27日，徐佳瑩發行第五張國語專輯《[[心裡學|心裡學]]》，並以該專輯於2018年6月23日獲得[[第29屆金曲獎|第29屆金曲獎]]最佳國語女歌手獎與最佳國語專輯獎<ref name="gma29" />。
