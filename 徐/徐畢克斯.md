'''徐畢克斯'''（'''{{lang-de|Johann Baptist Ritter von Spix}}，'''1781年2月9日 –1826年3月13日）是[[日耳曼|日耳曼]][[自然历史|生物学家]]。他到巴西探險，帶回了大量植物、昆蟲、哺乳動物、鳥類、兩棲動物和魚類標本，成為今日慕尼黑邦立動物學蒐藏館的重要基礎。他的許多人類學藏品，如舞蹈面具等等，現在是慕尼黑[[五大洲博物馆|五大洲博物館]]收藏的一部分。

== 传记 ==
徐畢克斯出生在[[艾施河畔赫希施塔特|赫希施塔特]]，位於今日的[[中弗兰肯行政区|中法蘭克尼亞]]，在家中十一名孩童中排行第七。他的童年住所自2004年起成為徐畢克斯博物館，向大眾開放。徐畢克斯在[[班贝格|班貝格]]取得哲學博士學位，後來又到[[维尔茨堡|威茲堡]]學習神學，在參加年輕教授[[弗里德里希·谢林|謝林]]的講座後，他對自然產生興趣而中斷了神學訓練，開始學習醫學，並在1807年獲得了第二個博士學位。

徐畢克斯短暫地在班貝格擔任醫生一段時間後，在1808年被[[马克西米利安一世_(巴伐利亚国王)|巴伐利亞國王麥西米連一世]]任命，到位於慕尼黑的巴伐利亞科學院，擔任[[动物学|動物學]]研究助理，他也因此獲得獎學金，去巴黎接受[[喬治·居維葉|喬治·居維葉]]等人當代科學化的動物學訓練。 同行，他還首次遊覽了法國北部的[[诺曼底大区|諾曼地]]海岸。 之後，徐畢克斯又前往法國南部和義大利，為巴伐利亞科學院的動物學館藏收集動物標本，並調查海洋動物。

1810年，徐畢克斯回到慕尼黑對動物學收藏進行分類，並撰寫他第一本關於海星和其它海洋動物的出版物。1811年，徐畢克斯第一本關於[[动物分类学|動物分類學]]史的重要著作出版，隨後被任命為巴伐利亞科學院研究員。徐畢克斯也掌管巴伐利亞的動物學蒐藏，如同今日的館長職位，為今日慕尼黑邦立動物學蒐藏館奠立重要基礎。他還發表其它作品，其中最重要的，是1815年一部以拉丁文撰寫，搭配精彩石版畫插圖的著作《頭源學》(''Cephalogenesis'')。這本書比較了不同動物的頭顱型態，包括人類、猿、爬行動物、鳥類和其它動物。

== 探勘巴西 ==
[[File:Testudines_Tab._XI.jpg|缩略图]]
1817年，徐畢克斯和日耳曼植物學家[[卡尔·冯·马齐乌斯|馬荻士]]加入一群陪伴奧地利公主利奧菩婷妮(Maria Leopoldine Josepha Caroline von Österreich)前往[[巴西|巴西]]的奧地利博物學者團。他們先到了[[里約熱內盧|里約熱內盧]]，但很快地，徐畢克斯和馬荻士便離開奧地利隊伍，獨自在巴西行動。他們二人從里約熱內盧南部前往[[聖保羅_(巴西)|聖保羅]]北部，這段旅程還有奧地利畫家Thomas Ender同行。接著他們到巴西[[米纳斯吉拉斯州|米納斯吉拉斯州]]的[[欧鲁普雷图|歐魯普雷圖]]和[[迪亚曼蒂纳|迪亞曼蒂納]]，並描述了那邊開採鑽石的情形。之後，又更深入內陸，然後回到薩爾瓦多海岸。

他們越過巴西東北部乾燥的卡廷加林區(Caatinga)，飽受各種疾病之苦，幾次瀕臨渴死。在整個旅途中，他們收集並記錄了各種動植物，以及其它一切對科學研究而言重要的事物。他們也記錄原住民及其習慣，以及任何可能具有經濟價值的事物。 此外，他們調查了巨大的本德戈隕石(Bendegó meteorite)，還發現了桑塔納組化石出土地(Santana Formation)的魚化石。

巴西探險之旅最後到了[[亞馬遜河|亞馬遜河流域]]，在那裡徐畢克斯和馬荻士決定走不同路線。徐畢克斯抵達靠近秘魯的邊界[[塔巴廷加_(亞馬遜州)|塔巴廷加]]，再從[[马瑙斯|瑪瑙斯]]轉到[[尼格罗河|尼格羅河]]。馬荻士則搭船到優普拉河(Yupurá River)，他在那裏買下分別來自尤瑞族(Juri)與馬瑞哈族(Miranha)的兩位原住民孩童，帶回慕尼黑之後，受洗取名為Johannes和Isabella。1820年，徐畢克斯和馬荻士帶著數以千計的動植物與人類學物品回到慕尼黑，那些動物標本成為慕尼黑邦立動物學蒐藏館的基礎，人類學物品則成為慕尼黑民族學博物館（現為[[五大洲博物馆|五大洲博物館]]）的基礎。

== 出版物 ==
徐畢克斯回國後，巴伐利亞國王授予他騎士功勛勳章。他努力整理他的動物學資料，也著手和馬荻士共同撰寫描述他們的巴西之旅。此出版物共有三卷，分別在1823、1828和1831年出版，並翻譯成葡萄牙文，第一卷還有英文版。這份作品至今仍非常重要。徐畢克斯在準備出版第二卷期間過世，馬荻士藉由他們的筆記來完成出版。徐畢克斯描述了許多鳥類、猿類、蝙蝠和爬蟲動物，也為牠們命名。他總共記述了大約500到600種物種和亞種。此外，還有一些種動物以他的名字命名。1826年5月13日，徐畢克斯在慕尼黑過世，可能是[[雅司病|雅司病]]或[[查加斯病|恰加斯病]]等熱帶病造成的死因。

== 紀念 ==
為了紀念他，徐畢克斯的名字被用於三種南美洲爬蟲動物的學名之中：[[黑腹刺頸龜|黑腹刺頸龜]](''Acanthochelys spixii'')、徐畢克斯遊蛇(''Chironius carinatus spixi'') 、亞馬遜珊瑚蛇(''Micrurus spixii'')。亞馬遜珊瑚蛇是一種毒蛇。<ref>Beolens, Bo; Watkins, Michael; Grayson, Michael (2011). ''The Eponym Dictionary of Reptiles''. Baltimore: Johns Hopkins University Press. xiii + 296 pp. </ref> 德國班貝格大學<ref>[https://www.uni-bamberg.de/universitaet/profil/geschichte-und-tradition/persoenlichkeiten/bedeutende-persoenlichkeiten/spix/ Ein Bamberger Alumnus, der Internationalität lebte]{{de}}</ref>則以他的名字設立一個年度國際訪問學者席位。
