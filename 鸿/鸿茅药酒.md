'''鸿茅药酒'''，是一个产自[[内蒙古自治区|内蒙古自治区]][[凉城县|凉城县]][[鸿茅镇|鸿茅镇]]<ref name=thepaper1/>的[[中成药|中药]][[药酒|药酒]]，现由[[鸿茅国药|内蒙古鸿茅国药股份有限公司]]（鸿茅国药）{{efn|原名“内蒙古鸿茅药业有限责任公司”，2017年11月13日改为今名<ref>{{Cite web|url=http://www.hongmaoyaoye.com/news/detail/793.html|title=公司更名通知|accessdate=2018-04-17|author=|date=|work=|publisher=内蒙古鸿茅国药股份有限公司|language=Zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180418072323/http://www.hongmaoyaoye.com/news/detail/793.html|archivedate=2018-04-18}}</ref>。}}生产，是一种甲类[[非处方药|非处方药]]（OTC），批准文号为[[国药准字|国药准字]]Z15020795<ref>{{Cite web |url=http://finance.sina.com.cn/chanjing/gsnews/2018-04-13/doc-ifyuwqfa0662447.shtml |title=医生指鸿毛药酒是毒药被捕 是否构成损害商品声誉罪? |accessdate=2018-04-15 |author= |date=2018-04-13 |work= |publisher=新浪网|language=Zh-hans | archive-url=https://web.archive.org/web/20180416084412/http://finance.sina.com.cn/chanjing/gsnews/2018-04-13/doc-ifyuwqfa0662447.shtml|archive-date=2018-04-16|dead-url=no }}</ref>。

== 历史 == 
鸿茅药酒由移居[[凉城县|凉城县]]的山西籍名医王吉天于清朝[[乾隆|乾隆]]四年（1739年）创制。据史料记载称，鸿茅古镇在清代[[康熙|康]][[乾隆|-{乾}-]]时期位于厂汉营一带，是“[[走西口|走西口]]”的通道中，过“[[杀虎口|杀虎口]]”的第一个落脚点。另据《凉城县志》记载，1739年王吉天途经鸿茅古镇，发现当地居民多患风寒湿痹，关节疼痛，行动不便，于是以独门秘法入酒调制，创出可治风寒湿痹等多种病症，又兼具补益养生作用的药酒。王吉天以古镇名“鸿茅”为药酒名称，“鸿茅药酒”由此得名。道光十年（1830年），大同府官员将鸿茅药酒呈现朝廷，被定为贡酒<ref name=as>{{Cite book|title=内蒙古本土品牌成长与特色优势产业集群研究|last=黄小葵著|first=|publisher=北京：[[经济管理出版社|经济管理出版社]]|year=2015.07|isbn=978-7-5096-3674-9|location=|pages=315-316}}</ref><ref name=thepaper1>{{Cite web |url=https://www.thepaper.cn/newsDetail_forward_2079367 |title=鸿茅药酒所在地鸿茅镇：去年才从其他镇划出，设置还不满一年 |accessdate=2018-04-16 |author=岳怀让 |date=2018-04-16 |work= |publisher=澎湃新闻|language=Zh-hans }}</ref>。《[[绥远通志稿|绥远通志稿]]》称：清乙卯年（[[雍正|雍正]]十三年，1735年），晋地荒旱，榆次名医出关入塞。至鸿茅古镇，见沃川雁翔草异，遂安居。然冬寒春阴，寒症频，吉天悯人载物，举中药六十七味，融以洌酒，肇孕鸿茅药酒<ref>{{Cite web|url=http://www.hongmao1739.com/story.aspx|title=鸿茅药酒起源 ——从清乾隆四年开始|accessdate=2018-04-15|author=|date=|work=|publisher=内蒙古鸿茅健康大药房连锁有限责任公司、内蒙古鸿茅国药股份有限公司网站|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20180416073640/http://www.hongmao1739.com/story.aspx|archivedate=2018-04-16}}</ref>。

[[中华民国大陆时期|中华民国时期]]，鸿茅药酒有二十四两和十二两两种包装，售价分别为[[银圆|银圆]]一元二角和七角，行销山西、内蒙、外蒙各地。由于鸿茅药酒独特的功效，鸿茅药酒在蒙语中被称为“力敌希地泰”，在汉语中的意思是“神力之酒”<ref>{{Cite web |url=http://www.hongmaoyaoye.com/liquor/story/index.html|title=鸿茅药酒故事 |accessdate=2018-04-15 |author= |date=|work= |publisher=内蒙古鸿茅国药股份有限公司网站|language=Zh-hans }}</ref>。1930年代，传到王吉天后代王继林之手。当时，因经济、战乱影响，药酒产量逐渐减少。[[抗日战争|抗日战争]]全面爆发后，酒坊被迫关门，王氏传人逃离凉城县。日军搜查药酒配方和王氏传人无果。1948年3月，中国人民解放军攻占凉城县厂汉营。中共政权的工作队接管酒坊，但因缺少配方、熟练工人及王氏传人杳无音信，未能恢复生产。时任凉城县长的[[李耀厚|李耀厚]]多方寻找，于1956年3月18日，在凉城县三苏木乡（今[[岱海镇|岱海镇]]）乡民手中购得配方，收归国有。1962年，成立国营鸿茅酒厂，再度生产鸿茅药酒<ref>{{Cite web |url=http://news.163.com/14/0915/14/A66KVUDO00014AEE.html |title=凉城记忆：鸿茅药酒方子失而复得始末(图) |accessdate=2018-04-17 |author= |date=2014-09-15 |work= |publisher=网易，来源: 齐鲁网(济南)|language=Zh-hans}}</ref>。

另根据官网介绍，1945年抗日战争胜利前夕，[[中国共产党第七次全国代表大会|中国共产党第七次全国代表大会]]召开期间，时任绥南专属专员的[[郑天翔|郑天翔]]特派人送出一批鸿茅药酒至延安，不少中共中央领导喝过鸿茅药酒之后，称赞“南有茅台，北有鸿茅”<ref>{{Cite book|title=走进内蒙古旅游手册|last=欣苑编著|first=|publisher=呼和浩特：[[内蒙古人民出版社|内蒙古人民出版社]]|year=2002.08|isbn=7-204-05906-9|location=|pages=256-257}}</ref><ref>{{Cite book|title=中华风物谚谣|last=何令俭编著|first=|publisher=昆明：[[云南人民出版社|云南人民出版社]]|year=2000.01|isbn=7-222-01072-6|location=|pages=338}}</ref>；同年10月，[[贺龙|贺龙]]因[[胆囊炎|胆囊炎]]在凉城养病，当地百姓闻讯送来鸿茅药酒，贺龙喝了近半个月的鸿茅药酒后得以恢复健康；1973年9月，法国总统[[乔治·蓬皮杜|蓬皮杜]]访华，期间与时任[[国务院总理|国务院总理]][[周恩来|周恩来]]总理说起他父亲在山西和绥远传教的经历。当时他父亲患上关节炎，双膝疼痛难行，后来服用教友送来的鸿茅药酒，症状得以消失。来华之前，总统父亲特地嘱咐，希望能给他带回几瓶鸿茅药酒。周总理闻听后欣然允诺，指示层层下达，鸿茅药酒厂特制了一批鸿茅药酒，作为特殊国礼送给了蓬皮杜总统<ref>{{Cite book|title=内蒙古本土品牌成长与特色优势产业集群研究|last=黄小葵著|first=|publisher=北京：经济管理出版社|year=2015.07|isbn=978-7-5096-3674-9|location=|pages=316-317}}</ref>（根据{{tsl|en|Geni.com|Geni.com}}上的信息，蓬皮杜总统的父亲Léon Pompidou逝世于总统访华4年前的1969年2月4日<ref name=geni>{{Cite web |url=https://www.geni.com/tr8n/language/switch?language_action=switch_language&locale=zh-CN&source_url=https://www.geni.com/people/L%C3%A9on-Pompidou/6000000014697109842|title=Léon Pompidou (1887 - 1969) - Genealogy |accessdate= |author= |date=2015-02-19 |publisher=geni.com|language=Zh-hans }}</ref>）。然而这些说法引来了自媒体“春雨医生”的公开质疑<ref name=dw>{{Cite web|url=http://culture.dwnews.com/renwen/news/2018-03-23/60047744.html|title=在争议中把生意越做越大的鸿茅药酒到底有何神通|accessdate=2018-04-15|author=|date=2018-03-23|publisher=多维新闻|language=Zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180418072323/http://culture.dwnews.com/renwen/news/2018-03-23/60047744.html|archivedate=2018-04-18}}</ref>。

1992年，国营凉城县鸿茅酒厂进行股份制改革，成立凉城鸿茅酿酒有限公司。同年10月16日，鸿茅药酒经内蒙古自治区卫生厅批准注册，原批准文号为“内卫药准字（86）I－20－1355号”，到1998年换发批准文号（ZZ-4773-内卫药准字（1998）1773号）。2002年，根据原[[国家药品监督管理局|国家药品监督管理局]]要求，市场上销售的所有国产药品统一使用“国药准字”批准文号（地方监管部门发放的批准文号随之停用），鸿茅药酒完成了批准文号的变更，批准文号为Z15020795<ref name=nbd>{{Cite web |url=http://www.nbd.com.cn/articles/2018-04-15/1208400.html|title=“神药”鸿茅药酒的前世今生！2006年，其掌门与同伴花500多万吞超4000万资产 |accessdate=2018-04-16|author= |date=2018-04-15 |publisher=e公司官微|language=Zh-hans }}</ref><ref name=caijing>{{Cite web|url=http://yuanchuang.caijing.com.cn/2018/0416/4436968.shtml|title=鸿茅药酒是如何通过审批的？|accessdate=2018-04-16|author=|date=2018-04-16|publisher=财经网|language=Zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180416141620/http://yuanchuang.caijing.com.cn/2018/0416/4436968.shtml|archivedate=2018-04-16}}</ref><ref name=xhw>{{Cite web|url=http://www.xinhuanet.com/fortune/2018-04/16/c_1122691690.htm|title=鸿茅药酒是如何成为非处方药的？——国家药品监督管理局回应鸿茅药酒有关监管情况|accessdate=2018-04-16|author=|date=2018-04-16|publisher=新华网|language=Zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180418072323/http://www.xinhuanet.com/fortune/2018-04/16/c_1122691690.htm|archivedate=2018-04-18}}</ref>。2003年11月25日，原国家食品药品监督管理局印发《关于公布第六批非处方药药品目录的通知》（国食药监安（2003）323号），公布鸿茅药酒为甲类[[非处方药|非处方药]]<ref name=xhw/>。

1995年，鸿茅与内蒙古金火公司签订全国总经销协议<ref name=nbd/>。1997年[[香港回歸|香港回歸]]期间，鸿茅药酒被中國輕工業協會推薦為「97香港回歸八大特級珍藏品牌」，號稱「向香港同胞送去了草原人民的深情厚意」。此前，鸿茅药酒被包裝成出口產品，远销港澳海外<ref name=appledaily>{{Cite web |url=https://hk.news.appledaily.com/international/daily/article/20180416/20362776|title=鴻茅藥酒曾列「香港回歸特級珍藏品」  |accessdate=2018-04-16|author= |date=2018-04-16 |publisher=香港苹果日报|language=Zh-hant }}</ref>。由于鸿茅药酒因“过度夸大疗效”掠夺市场，在1990年代末，暂时离开消费者视线<ref name=thepaper/>。

2001年11月，上市公司[[金宇集团|金宇集团]]（现为生物股份）以77.1％的股份入主鸿茅，实现私有化；2004年，鸿茅酒厂年生产能力由原来的300吨增加到了3000吨，并形成药酒、白酒、冰酒、奶酒、保健酒等多元化、规模化生产格局<ref name=nbd/>。2006年，以杜海军、[[鲍洪升|鲍洪升]]为首的药圈行销巨头，经过多轮谈判全資收购鸿茅酒厂，并组建鸿茅实业公司，将公司办公地点移至北京市朝阳区<ref name=nbd/>，此后鸿茅药酒开始大量投放广告以增加銷量<ref name=appledaily/>。

2006年，鸿茅药酒配制技艺作为[[中医传统制剂方法|中医传统制剂方法]]下的分项列入第一批[[国家级非物质文化遗产|国家级非物质文化遗产]]<ref>{{Cite web |url=http://www.gov.cn/zhengce/content/2008-03/28/content_5917.htm|title=国务院关于公布第一批国家级非物质文化遗产名录的通知|accessdate=2018-04-15 |author= |date=|work= |publisher=中华人民共和国国务院|language=Zh-hans }}</ref>。2010年，鸿茅经[[中华人民共和国商务部|中华人民共和国商务部]]认定为[[中华老字号|中华老字号]]<ref name=rmw>{{Cite web |url=http://society.people.com.cn/n1/2018/0416/c1008-29927478.html|title=别把药当酒喝!鸿茅药酒广告违法2630次,谁是它的护身符|accessdate=2018-04-16 |author= |date=2018-04-16 |publisher=人民网-健康时报|language=Zh-hans }}</ref>。2011年，鸿茅药业请演员[[张铁林|张铁林]]代言，迎合市场需求。之后被邀请的意见领袖还有[[德德玛|德德玛]]、[[谢芳|谢芳]]、[[黄健翔|黄健翔]]等<ref name=as />，同年获得“[[中国驰名商标|中国驰名商标]]”称号，而鸿茅中医药酒文化再次以传统医药类别，入选内蒙古自治区《非物质文化遗产名录》；2014年，鸿茅药酒配制技艺入选国务院批准，文化部确定的《第四批非物质文化遗产代表性项目名录》<ref name=rmw/>。2016年，鸿茅药酒被[[国家质检总局|国家质检总局]]认定为[[中国地理标志产品|中国地理标志产品]]<ref>{{Cite web|url=http://www.cgi.gov.cn/Home/Detail/948/|title=质检总局关于受理洛隆糌粑等产品申报|accessdate=2018-04-16|author=|date=2016-08-29|publisher=质检总局科技司|language=Zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180418072324/http://www.cgi.gov.cn/Home/Detail/948/|archivedate=2018-04-18}}</ref>。

== 市场营销 ==
鸿茅药酒是中国大陆各大电视台的常客，根据央视市场研究发布的《2018年第一季度中国广告市场报告》，其为中国大陆地区[[電視廣告|电视广告]]的最大投放者，而在2018年第一季度其电视投放增速达3成以上。而除去鸿茅药酒，在大陆地区电视广告的主要金主前十名大也多是药品品牌。而外界对其宣传的疗效一直存疑。<ref name=":0">{{Cite news|url=https://www.zaobao.com/news/china/story20180516-859321|title=鸿茅药酒跨省案被抓医生 谭秦东突发精神病送院治疗|date=2018-05-16|newspaper=早报|accessdate=2018-05-16|language=en}}</ref>

鸿茅药酒曾邀请影视明星[[陈宝国|陈宝国]]、歌唱家[[德德玛|德德玛]]，表演艺术家[[谢芳|谢芳]]、[[雷恪生|雷恪生]]等人为鸿茅药酒代言，在吉林、四川、广州市场还邀请了文体界名人明星助阵。由于这一原因，鸿茅药酒销售量不断上升，并通过各级电视台的立体交叉投放，让鸿茅药酒的知名度不断增强。2008年，鸿茅药酒销售额突破1亿元人民币；到2016年时，鸿茅药酒2016年零售药店终端（包含实体药店和网络药店两大市场）销售额约为16.3亿元，在中成药市场上仅次于[[东阿阿胶|东阿阿胶]]。鸿茅药业还是当地的纳税大户，2016年1-10月份，鸿茅药业上缴国税税收2680万元，地税税收393万元。2017年、2018年，鸿茅药酒两度入选[[中国中央电视台|中国中央电视台]]“国家品牌计划”<ref>{{Cite web|url=http://news.163.com/18/0416/19/DFHP66LK0001899N.html|title=风波中的鸿茅药酒：纳税大户 1年广告投放额过百亿|accessdate=2018-04-17|author=|date=2018-04-16|publisher=新京报|language=Zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180418072323/http://news.163.com/18/0416/19/DFHP66LK0001899N.html|archivedate=2018-04-18}}</ref><ref>{{Cite web|url=http://www.infzm.com/content/134350|title=屡上黑榜，屡拿批文 鸿茅药酒：“我们是正规企业”|accessdate=2018-04-17|author=南方周末记者 马肃平 南方周末实习生 韩钰泽|date=2018-03-23|publisher=南方周末|language=Zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180418072324/http://www.infzm.com/content/134350|archivedate=2018-04-18}}</ref>。从2016年开始，鸿茅药酒将广告策略转向黄金档影视剧植入和卫视宣传，如《[[老爸当家|老爸当家]]》、《[[我的老爸是奇葩|我的老爸是奇葩]]》等电视剧中均出现了鸿茅药酒的身影。在中国大陆各大药店，鸿茅药酒促销装经常摆放于药店显著位置，通常以3瓶一盒，6瓶一箱的形式销售，有的药店工作人员建议顾客“最好一次买三箱，这样可喝一年<ref name="rmw" />”。

根据国家食药监总局统计，2011年至2018年，鸿茅药酒药品广告总数为1192条，为第二名[[阿胶|阿胶]]的2倍多<ref name=xlcj/>。另根据[[尼尔森|尼尔森]]网联AIS全媒体广告监测显示，2017年1月至11月，鸿茅药业替代[[宝洁|宝洁]]，位列中国大陆投放广告企业第一，投放总额同比增长55.9%，宝洁有限公司以同比下降24.2%的成绩位列第二，其老对手[[联合利华|联合利华]]在传统媒体投放上减少了27.4%，排名第十位。而根据央视市场研究媒介智讯（CTRMI）的数据，2016年鸿茅品牌在电视广告中的投放额为150亿元，位居第一位。而鸿茅药酒曾多次因广告内容违法被25个省市级食药监部门通报，违法次数共2630次，被暂停销售数十次<ref name=rmw/>。

而鸿茅董事长鲍洪升也在其认证微博上大肆宣传鸿茅药酒。2011年6月26日，鲍洪升在微博回答网友关于喝鸿茅药酒是否有年龄规定的提问时表示“鸿茅药酒没有年龄限制”，而鸿茅药酒药品说明书中则明确写明“儿童、孕妇禁用”；同年6月15日，鲍洪升再次回答网友关于每天喝多少鸿茅药酒合适的提问表示“每天喝三两正好，决对（绝对）不上火，放心吧”，而根据产品说明书规定，该药品每次服用15毫升，一日2次，即一日最多只能服用半两。2012年2月9日，鲍洪升转发了网友声称“老人坚持每天喝点鸿茅药酒非常好，喝了酒就不用吃药了”的评论，并跟评称“我们的良心药”<ref name=thepaper>{{Cite web|url=https://www.thepaper.cn/newsDetail_forward_2029110|title=鸿茅药酒的封神之路：推手鲍洪升和2630条违法广告|accessdate=2018-04-17|author=澎湃新闻记者 姚晓岚|date=2018-04-16|publisher=澎湃新闻|language=Zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180418072324/https://www.thepaper.cn/newsDetail_forward_2029110|archivedate=2018-04-18}}</ref>。

== 争议 ==
=== 配方问题 ===
鸿茅药酒因配方及产品功效一直受到部分人士的争议。2018年3月，由于鸿茅药酒配方中的“[[豹骨|豹骨]]”来源不明（豹骨自2006年1月1日起被中国全面禁止），并且没有按照[[国家林业局|国家林业局]]规定，在瓶身及包装上贴专用标识，得到众多动物保护工作者的转发和讨论。而鸿茅药业的说法则是“鸿茅药酒使用的豹骨是国家特批的，所以可以一直通过正规渠道购买豹骨，现在还会定时定量购买”<ref name=dw/>。著名作家[[方舟子|方舟子]]也发文指出，一般的药酒仅用几种、十几种药材，而鸿茅药酒却用了67种中药药材。方舟子也指出，对中药的毒性，目前缺乏系统、透彻的研究，绝大部分是不清楚的<ref name=caijing/>。

4月16日，鸿茅国药公司生产中心总经理王生旺和总经理助理韩军就鸿茅药酒近期讨论的质疑作出回应，称鸿茅药酒是[[国家中药保护品种|国家中药保护品种]]，其临床实验数据、毒理学实验数据可以不公开，而这些数据已经上报至内蒙古自治区食药监局和原国家食药监总局。王生旺还称“鸿茅药酒毒性很低”，一个人一天喝165斤鸿茅药酒才会中毒。另外，王生旺也就鸿茅药酒的广告违法问题作出回应，称该公司发布的广告“都是合法合规”的，每条广告均有药品广告批准文号，且自2008年起，原国家食药监总局网站发布的违法广告汇总、公告、通告，均未涉及鸿茅药酒。而公司只向央视和一些地方卫视电视台投放广告。王生旺还表示，被通报违法的广告是“当地经销商投放的，而非鸿茅药酒公司”。韩军最后表示“鸿茅药酒未收到一例不良反应的报告”。而谈到食药监总局通报中提及的不良反应，韩军表示“很多原因都可以导致这些症状，不一定是饮用鸿茅药酒所致<ref>{{Cite web|url=https://www.thepaper.cn/newsDetail_forward_2083936|title=鸿茅药酒回应四大质疑，称实验显示“一天喝165斤才中毒”|accessdate=2018-04-18|work=澎湃新闻}}</ref>。”

4月17日，内蒙古自治区人民检察院指令[[凉城县|凉城县]]人民检察院將本案退回公安局继续侦查，而已经关押超3个月的谭秦东最终得以取保候审。<ref name=":0" />

4月18日，曾多次购买鸿茅药酒的北京律师殷清利，分别向[[中华人民共和国商务部|商务部]]、[[中华人民共和国文化和旅游部|文化和旅游部]]提出申请。殷清利称，由于鸿茅药酒在广告上的多次严重违法、违规，认为其已不符合继续使用中华老字号的称号（根据《“中华老字号”认定规范（试行）》第四条，中华老字号认定的其中一个条件为“具有良好信誉，得到广泛的社会认同和赞誉”），因此向商务部提交履行职责申请，责令整改、暂停或取消其“中华老字号”称号。另外，殷清利还依据《政府信息公开条例》，向文化和旅游部请求将鸿茅药酒配制技艺入选“第四批国家级非物质文化遗产代表性项目名录扩展项目名录”的推荐材料、专家初评和审议、公示等政府信息内容公开<ref>{{Cite web|url=https://www.thepaper.cn/newsDetail_forward_2082561|title=一律师提申请：对鸿茅药酒配制技艺入选“非遗”政府信息公开|accessdate=2018-04-18|work=澎湃新闻}}</ref>。

4月21日，据腾讯财经报道，与鸿茅药酒相关的内容已经从“CCTV国家品牌计划”的首页上撤下<ref>{{Cite web|url=https://finance.qq.com/a/20180421/011364.htm|title=鸿茅药酒从国家品牌计划官网消失|accessdate=2018-04-21|work=腾讯财经}}</ref><ref>{{Cite web|url=http://cbp.cctv.com|title=CCTV国家品牌计划|accessdate=2018-04-21|work=央视网}}</ref>。此前于4月19日，该品牌已被[[湖南卫视|湖南卫视]]撤下公益冠名赞助商。2019年1月17日，[[国家市场监督管理总局|国家市场监督管理总局]]通报，对鸿茅药酒曾入选“CCTV国家品牌计划”广告用语涉嫌违反《中华人民共和国广告法》问题，约谈[[中央广播电视总台|中央广播电视总台]]，并责成北京市市场监督管理局依法立案调查<ref>{{Cite web|url=https://news.sina.com.cn/s/2019-01-17/doc-ihqhqcis7104663.shtml|title=国家品牌计划被查：鸿茅药酒曾入选 10企业砸42亿|accessdate=2019-01-17|work=央视网}}</ref>。

4月26日，鸿茅国药发布自查整改报告，称鸿茅药酒“按说明服用是安全的，研究和试验结论是安全有效的”，另外公司购买和使用豹骨的行为也符合法律规定。而企业此前所投放的广告现已全部停播，并派出检查小组开展自查自纠工作<ref>{{Cite web|url=http://finance.jrj.com.cn/2018/04/26152824456640.shtml|title=鸿茅药酒：公司从未宣称鸿茅药酒可包治百病 |accessdate=2018-04-26|work=金融界}}</ref>。

5月11日上午，当事人谭秦东突发精神疾病。治疗两天后，病情稍许稳定，但血压较高，偶尔还会说胡话。目前，正在广东省人民医院综合一科接受治疗。<ref>{{Cite web|url=https://3g.163.com/news/article/DHQBH5R00001875P.html#adaptation=pc|title=吐槽鸿茅药酒被抓医生因精神疾病入院 曾被关97天_手机网易网|accessdate=2018-05-16|date=2018-05-15|work=3g.163.com}}</ref>

5月17日，当事人谭秦东通过其妻子刘璇微博发布对鸿茅国药股份有限公司道歉声明。 随后鸿茅药酒回应谭秦东致歉声明：接受致歉，撤案撤诉。随后鸿茅药酒董事长鲍洪升在自己的社交平台点赞该微博，网友怀疑次道歉是可能被威胁不得已而为<ref>{{Cite web|url=http://m.baidu.com/sf?pd=realtime_article&openapi=1&dispName=iphone&from_sf=1&resource_id=4584&word=%E9%B2%8D%E6%B4%AA%E5%8D%87&title=%E6%90%9C%E7%8B%90&keysign=http%3A%2F%2Fbusiness.sohu.com%2F20180517%2Fn537968520.shtml&source=realtime_a&fks=a48049&lid=12247107221453485300&referlid=12247107221453485300&ms=1&frsrcid=19&frorder=3|title=停更微博半年多，鸿茅董事长鲍洪升点赞谭秦东道歉声明|accessdate=|author=|date=|publisher=搜狐新闻}}</ref> ​。

6月，上海市闵行区人民法院就鸿茅国药状告律师程远名誉侵权案作出一审判决，驳回了鸿茅国药的诉讼请求。法院审理后认为，涉案文章虽引用了“上海发布”（[[上海市人民政府新闻办公室|上海市人民政府新闻办公室]]）微信公众号的《这12件典型虚假广告涉嫌违法，已被依法查处》一文，但并未标明鸿茅药酒是该12件虚假广告之一，被告亦表明对鸿茅药酒广告的质疑系网友留言，应不致引起歧义和误导；其次涉案文章标题使用“广告史劣迹斑斑”的评论性表述，系源于互联网及其他媒体已披露的鸿茅药酒违法广告史，但不构成侮辱诽谤<ref>{{Cite web|url=http://www.xinhuanet.com/fortune/2018-06/22/c_1123019238.htm|title=状告某新媒体名誉侵权被驳回 鸿茅药酒“正名之门”暂被关闭  |accessdate=2018-06-22|work=法制日报}}</ref>。

10月，内蒙古自治区食药监局回答了消费者就鸿茅药酒舆情关注的一些焦点问题，这些问题以“政府信息公开告知书”的形式来回答。在政府信息公开告知书中指出，鸿茅药酒处方中所用的附子（制）、天南星（制）、半夏（制）全部为炮制加工品，不属于毒性中药品种，且有治疗作用。同时，鸿茅药酒所用的豹骨符合国家相关规定，飞行检查质量均为合格<ref>{{Cite web|url=http://www.xinhuanet.com/fortune/2018-10/26/c_129979944.htm|title=内蒙古官方回复鸿茅药酒焦点问题|accessdate=2018-10-28|work=经济参考报 }}</ref>。
