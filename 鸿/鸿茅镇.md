'''{{PAGENAMEBASE}}'''是{{PRC admin/fullname|15/09/25/208/000|3}}下辖的一个[[行政建制镇|镇]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|15/09/25/208/000}}}}</ref>，是凉城县人民政府的驻地所在（原为[[岱海镇|岱海镇]]），也是内蒙古[[鸿茅国药|鸿茅国药股份有限公司]]的所在地<ref name="岳怀让"></ref>。

==历史==

2012年1月20日，内蒙古自治区人民政府批复同意从[[曹碾满族乡|曹碾满族乡]]划出部分区域，设置[[厂汉营乡|厂汉营乡]]，辖脑包平、厂汉营、大圐圙、头号、十一号、保全庄、四号、二蛮沟、铁铺、王三顺、北水泉、省城窑、东厂汉营、中水泉、十七号、十九号、大洼17个村，乡人民政府驻厂汉营<ref>《内蒙古自治区人民政府关于调整乌兰察布市部分苏木乡镇行政区划的批复》（内政字〔2012〕36号）</ref>。

2017年，[[内蒙古自治区人民政府|内蒙古自治区人民政府]]发布内政字〔2017〕105号文件，宣布[[厂汉营乡|厂汉营乡]]撤乡设镇成立鸿茅镇<ref name="民政部">{{cite web |url=http://www.mca.gov.cn/article/sj/tjbz/a/2018/201803131442.html |title=2017年中华人民共和国县以下行政区划变更情况（12月份） |date=2018-03-13 |website=www.mca.gov.cn |publisher=[[中华人民共和国民政部|中华人民共和国民政部]] |access-date=2018-04-20 }}</ref>，而实际却是从[[岱海镇|岱海镇]]的部分区域划出<ref name="岳怀让">{{cite web |url=https://www.thepaper.cn/newsDetail_forward_2079367 |title=鸿茅药酒所在地鸿茅镇：去年才从其他镇划出，设置还不满一年 |author=岳怀让 |date=2018-04-16 |website=www.thepaper.cn |publisher=[[澎湃新闻|澎湃新闻]] |access-date=2018-04-20 }}</ref>，凉城县人民政府凉政发〔2017〕152号文件确认新设鸿茅镇<ref name="邢占国">{{cite web |url=http://inews.nmgnews.com.cn/system/2017/09/21/012398750.shtml |title=凉城县设鸿茅镇 传承鸿茅药酒文化 |author=邢占国 |date=2017-09-21 |website=inews.nmgnews.com.cn |publisher=《[[内蒙古晨报|内蒙古晨报]]》 |access-date=2018-04-21 }}</ref>，2017年9月16日举行建镇揭牌仪式<ref name="马先震">{{cite web |url=http://finance.ce.cn/rolling/201709/19/t20170919_26112873.shtml |title=鸿茅镇恢复古镇名 鸿茅药酒为凉城打造地区发展新名片 |author=马先震 |date=2017-09-19 |website=finance.ce.cn |publisher=中国经济网 |access-date=2018-04-20 }}</ref>。对比2002年审图<ref group="注">审图号：蒙S（2002）026号</ref>并在2004年由[[中国地图出版社|中国地图出版社]]出版的《内蒙古自治区地图册》<ref>{{cite book |author=訾冬梅 高秀静 主编 |title=《内蒙古自治区地图册》 |url=http://map.ps123.net/china/UploadFile/201402/2014021007343876.jpg |location=[[北京市|北京]] |publisher=[[中国地图出版社|中国地图出版社]] |page=27 |date=2018-04-20 |isbn=7-5031-2175-0 }}</ref>和现今凉城县人大网中的行政区划图，厂汉营乡地域实际上被并入[[曹碾满族乡|曹碾满族乡]]<ref name="人大地图">{{cite web |url=http://www.nmglcrd.gov.cn/images/map.png |title=乡镇地图 |website=www.nmglcrd.gov.cn |publisher=凉城县人民代表大会常务委员会 |access-date=2018-04-20 }}</ref>。

==地理==

鸿茅镇东邻[[岱海镇|岱海镇]]和[[岱海|岱海]]，南靠[[六苏木镇|六苏木镇]]，西南接[[永兴镇_(凉城县)|永兴镇]]，西为[[蛮汉镇|蛮汉镇]]，北部是[[卓资县|卓资县]]的[[大榆树乡|大榆树乡]]<ref name="人大地图"></ref>。鸿茅镇总面积368平方公里，共有13个村委会和10个社区，辖102个村民小组、56个居民小组，总户数36,766户，常住人口100,816人<ref name="岳怀让"></ref>。“从岱海镇分出”的鸿茅镇面积数据和人口总户数却与2017年初凉城县政府公布的岱海镇数据完全相同<ref>{{cite web |url=http://www.nmlc.gov.cn/Impression/Details/1039.html |title=岱海镇 |date=2017-01-09 |website=www.nmlc.gov.cn |publisher=凉城县人民政府 |access-date=2018-04-20 |quote=（第三段）全镇总户数36766户；（第四段）全镇总土地面积368平方公里 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==争议==

中国经济网和《内蒙古晨报》曾报道，有数百年历史的“鸿茅镇”恢复了古镇名，“《凉城县志》中记载”鸿茅古镇在[[清朝|清朝]][[康熙|康熙]][[乾隆|乾隆]]时期位于[[厂汉营乡|厂汉营]]一带，一位山西医生途径此地，调制出能治风寒湿痹等多种病症又兼具补益养生作用的药酒，并以古镇命名药酒为“[[鸿茅药酒|鸿茅药酒]]”<ref name="邢占国"></ref><ref name="马先震"></ref>。而1988年出版的《[[内蒙古自治区|内蒙古自治区]]地名志：[[乌兰察布市|乌兰察布盟]]分册》中记载，清朝乾隆年间，山西省[[原平市|崞县]]和[[阳高县|阳高县]]等地的人到此经商，定居开地，由两县县名中各取一字，命名为“崞阳庄”，并未提到该地有所谓“鸿茅古镇”的历史<ref>{{cite book |author=内蒙古自治区地名委员会 |title=《内蒙古自治区地名志：乌兰察布盟分册》 |location=[[呼和浩特市|呼和浩特]] |publisher=内蒙古自治区地名委员会 |page=414 |date=1988年 }}</ref>。1993年出版的《凉城县志》中，“大事记”、“建置沿革”、“乡村>>概况>>厂汉营乡”等编写侧重于地方史的章节段落同样没有提到“鸿茅古镇”，“食品、酿造工业”章节的鸿茅药酒段落也没有提到鸿茅药酒名称来源是“鸿茅古镇”<ref>{{cite book |author=凉城县志办《凉城县志》编纂委员会 |title=《凉城县志》 |url=http://116.113.96.251:8080/Search/SearchContent.aspx?RootId=20130717153259322&BookId=20150526161143806 |location=1993年 |publisher=[[內蒙古人民出版社|內蒙古人民出版社]]  }}</ref>。
