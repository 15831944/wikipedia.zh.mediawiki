'''莱州市'''，[[中國|中國]]的一個沿海城市。為[[山東|山東]]省縣級市，[[煙台市|煙台市]]代管。位於渤海[[萊州灣|萊州灣]]東岸。總面積1878平方公里，人口88萬。

歷史上曾为[[過國|過國]]、[[萊侯國|萊侯國]]、[[萊子國|萊子國]]、[[掖邑|掖邑]]地。西汉初年，即置[[掖縣|掖縣]]，属[[东莱郡|东莱郡]]、[[光州_(北魏)|光州]]、[[莱州_(古代)|萊州]]、[[莱州府|莱州府]]。1988年，掖县改县级莱州市。

== 行政區劃 ==
現轄11鎮、6個街道辦事處。市政府驻于 文昌街道。
* 街道辦事處：[[文昌街道|文昌街道]]、[[永安街道|永安街道]]、[[三山島街道|三山島街道]]、[[文峰街道|文峰街道]]、[[城港路街道|城港路街道]]、[[金仓街道|金仓街道]]。
* 鄉鎮：[[金城鎮_(萊州市)|金城鎮]]、[[平裡店鎮|平裡店鎮]]、[[朱橋鎮|朱橋鎮]]、[[程郭鎮|程郭鎮]]、[[驛道鎮|驛道鎮]]、[[郭家店鎮|郭家店鎮]]、[[柞村鎮|柞村鎮]]、[[夏邱鎮|夏邱鎮]]、[[虎頭崖镇|虎頭崖镇]]、[[沙河鎮|沙河鎮]]、[[土山鎮|土山鎮]]。

== 地理 ==
萊州市位於[[膠東半島|膠東半島]]的西北部，瀕臨[[萊州灣|萊州灣]]，東面分別和[[招遠|招遠]]市、[[萊西|萊西]]市接壤，東北與[[龍口|龍口]]市接壤，南面與[[平度|平度]]市為鄰，西南面隔膠萊河與[[昌邑|昌邑]]市隔河相望。

== 民族 ==
有[[漢族|漢]]、[[滿族|滿]]、[[回族|回]]、[[朝鮮族|朝鮮]]等10個民族，其中[[少數民族|少數民族]]不足萬分之一。

==資源==
* 礦藏資源主要有[[黃金|黃金]]、[[滑石|滑石]]、[[菱鎂石|菱鎂石]]、[[大理石|大理石]]、[[鐵礦石|鐵礦石]]、[[磷礦石|磷礦石]]、[[石灰石|石灰石]]、[[螢石|螢石]]、[[水晶石|水晶石]]、[[甲長石|甲長石]]、[[綠冬石|綠冬石]]、[[石英石|石英石]]、[[銀|銀]]、[[銅|銅]]、[[鉬|鉬]]、[[鹵水|鹵水]]。
* 水產資源主要有魚、蝦、貝、蟹、海参、藻類、海盐等。
* 土特產品有[[對蝦|對蝦]]、[[文蛤|文蛤]]、[[梭子蟹|梭子蟹]]、[[大竹蟶|大竹蟶]]四大名鮮，此外，还出产海肠、爬虾、扇贝、海参、大菱鲆养殖鱼。
* 麻渠大糖、面魚、羊湯等都是很有特色的小喫。
* 农业：莱州适合多种作物种植，粮食以[[小麦|小麦]]、[[玉米|玉米]]为主，历史上曾普遍种植过[[高粱|高粱]]。经济作物以苹果为主。

萊州資源類產業發展迅猛，但可持續發展類產業較弱。主要產業為糧食種植、蘋果種植、漁業、石材、黃金開採。萊州黃金儲量和石材資源豐富，但是淡水資源嚴重缺乏，沿海有人口居住地區的地下水普遍受海水倒灌影響，内地部分乡镇由于当地企业无节制排放污水而导致地下水污染，莱州淡水资源日益喪失飲用價值。

==名人==
*[[任克加|任克加]] ，原名任鸿禧，曾用名彭洛。山东省莱州市神堂镇彭家村人。1937年参加革命，同年9月加入中国共产党，与李伯民一起创办县委党报《民声》。后任胶东抗日游击第三支队中队政治指导员、大队政委，八路军山东纵队第五支队掖县锄奸团团    长，中共胶东区委军事部谍报科长、队列科长，山东省第三军区司令部五科长、直工科长，五支队政治部组织科长、后方政治处主任、支队卫生部和野战医院政委，东海军分区政治部副主任、武装部长，胶东军区武装部副部长等职。解放战争时期，任胶东东海军分区副政委，后被派到国民党军队起义的“重庆” 号军舰任军代表和政委。中华人民共和国成立后，曾任华东军区警备六旅、四旅副政委兼政治部主任，解放军步兵第一○一师副政委、副书记，志愿军铁道工程兵和铁道兵一○一师师长兼政委，铁道兵干部管理部副部长、部长，铁道兵科研部政委，铁道兵党委常委、政治部副主任、政治部党委副书记等职。1964年任铁道兵政治部副主任。1982年离休。
* [[徐勤先|徐勤先]]，1989年[[六四|六四]]拒率部隊入京鎮壓學生的38軍軍長。
* [[李登海|李登海]]，紧凑型玉米杂交之父，全国人大代表，创办登海种业，与袁隆平合称“南袁北李”。
* [[张宗昌|张宗昌]]，（1881～1932.9.3），字效坤。绰号“狗肉将军”、“混世魔王”、“长腿将军”、“三不知将军”、“五毒大将军”、“张三多等，奉系军阀头目之一。
* [[晏婴|晏婴]]，（前578－前500年），字仲，谥平，习惯上多称平仲，又常被称晏子。齐国莱地夷维人（今山东省莱州市平里店镇婴里），春秋后期外交家、思想家。
* [[张瑞敏|张瑞敏]]，(1949年1月5日～至今)，著名企业家，目前任海尔集团首席执行官。
* [[王沪宁|王沪宁]]，（1955年10月～至今），现任中共中央政治局常委，中央书记处书记，中央政策研究室主任。
