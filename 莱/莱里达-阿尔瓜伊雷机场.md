'''莱里达-阿尔瓜伊雷机场'''<ref>{{Cite book |title=《交通建筑设计》 |author=（西）卡尔斯·布鲁托|editor= 王今琪，王媛媛|language=zh |publisher=华中科技大学出版社 |year=2012 |isbn=9787560971377}}</ref>（{{lang-es|Aeropuerto de Lérida-Alguaire}}，{{lang-ca|Aeroport de Lleida-Alguaire}}；{{airport codes|ILD|LEDA}}<ref name="ICAO">{{cite web|title=Airport Movements: ILD|url=https://www.icao.int/safety/RunwaySafety/Lists/Airport%20Movements/DispForm.aspx?ID=3737|publisher=[[ICAO|ICAO]]|accessdate= 2016-02-10}}</ref><ref>{{cite web |url=http://www.iata.org/publications/Pages/code-search.aspx |title=Airline and Airport Code Search |publisher=International Air Transport Association}}</ref>）简称'''莱里达机场'''，按照加泰罗尼亚语简称'''列伊达机场'''<ref>[https://web.archive.org/web/20180926071800/http://sanzidaima.00cha.net/ild.html 阿尔瓜伊雷机场 三字代码ILD]{{zh}}</ref>，是位于西班牙东北部[[加泰罗尼亚|加泰罗尼亚]]自治区[[莱里达省|莱里达省]][[阿尔瓜伊雷|阿尔瓜伊雷]]市郊的一个小型国际机场，距离省会[[莱里达|莱里达]]市中心约15公里。

在外观上，该机场以多彩塔台和木质外观的航站楼而闻名。<ref>{{Cite web |url=https://web.archive.org/web/20181001022357/http://www.xinhuanet.com/world/2016-10/13/c_129319806.htm |title=世界最美的高颜值航站楼 飞机延误也不怕 |accessdate=2018-10-01 |date=2016-10-13 |publisher=新华社|language=zh}}</ref>
