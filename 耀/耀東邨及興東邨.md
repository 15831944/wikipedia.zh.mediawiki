{{engname|耀東邨|Yiu Tung Estate}}及{{engname|興東邨|Hing Tung Estate}}是[[香港|香港]]的[[香港公共屋邨|公共屋邨]]，分別位於[[香港島|香港島]][[東區_(香港)|東區]][[筲箕灣|筲箕灣]]和[[西灣河|西灣河]]半山，[[柏架山|柏架山]]北麓，分別於1994年及1996年開始入伙。

{{engname|東駿苑|Tung Chun Court}}及{{engname|東盛苑|Tung Shing Court}}是鄰近耀東邨的[[居者有其屋計劃|居者有其屋計劃]]屋苑；{{engname|東熹苑|Tung Hei Court}}、{{engname|東霖苑|Tung Lam Court}}及{{engname|東欣苑|Tung Yan Court}}是鄰近興東邨的居者有其屋計劃屋苑。

耀東邨及興東邨所在地前身為[[木屋區|木屋區]]<ref>{{Cite web |url=https://na.cx/i/JVHe8D3.jpg |title=約1980年代西灣河一帶航空照片}}</ref>，包括成安村、聖十字徑村、澳貝龍村等<ref name=hk-place>[http://www.hk-place.com/view.php?id=143 香港地方-消失了的地方]</ref>。在1989年，[[港府|港府]]收地並且興建了耀東邨及興東邨，兩邨由耀興道連接至筲箕灣及西灣河。耀東邨為港島區首個設有和諧三型及附翼大廈的公共屋邨，現由香港房屋委員會負責屋邨管理。

==簡介==
'''耀東邨'''位於[[筲箕灣|筲箕灣]]半山，於1994年至2000年間分5期入伙，項目編號為HK09NR，耀東邨第1期建有4幢[[和諧一型|和諧一型]]及2幢附翼大廈、第2期則為東駿苑、第3期建有2幢和諧三型大廈、第四期則建有5座和諧三型及1座附翼大廈。及後在2000年以剩餘地積比率加建第5期並劃為居屋'''東盛苑'''，讓耀東邨居民優先選購。

'''興東邨'''位於[[西灣河|西灣河]]半山，項目編號為HK10NR，門牌號碼為耀興道53號，為{{engname|興東邨第2期|Hing Tung Estate Phase 2}}，前身為橫坑村木屋區<ref name=hk-place>[http://www.hk-place.com/view.php?id=143 香港地方-消失了的地方]</ref>，房委會於1996年1月把興東邨第2期首次透過試驗性的「[[自選單位計劃|自選單位計劃]]」把單位編配給受[[柴灣邨|柴灣邨]]9-12座重建影響的租戶，讓他們自行揀選合適的居住單位，並於同年開始入伙。本屋邨設有3座公屋大廈，房委會於1997年把原本為{{engname|興東邨第3期|Hing Tung Estate Phase 3}}的興欣樓劃為居屋'''東霖苑'''，門牌號碼為耀興道63號，當中4至38樓單位用作出售，而地下至3樓單位則保留作弱智兒童學校暨宿舍及長者住屋，該部份屬於房委會物業，並由房委會負責管理。東霖苑亦為港島區首個設有1人單位的居屋屋苑，也是香港唯一一個設有長者住屋的居屋屋苑。1998年{{engname|興東邨第四期|Hing Tung Estate Phase 4}}的興真樓及興善樓也劃作居屋出售，成為居屋'''東欣苑'''，門牌編號為耀興道68號，分別由1幢35層高和諧一型及1幢26層高的和諧三型大廈所組成。由於東欣苑各座1-4樓部份單位作為多層[[停車場|停車場]]，該部份屬於房委會物業，並由房委會負責管理，其他地方則作為居屋用途，同時屋苑設有1座[[升降機|升降機]]大樓連接[[行人天橋|行人天橋]]通往東霖苑，由業主立案法團負責管理。現在之管理公司為富寧物業管理有限公司。東欣苑為港島區唯一一個設有和諧3型大廈的居屋屋苑。

'''東熹苑'''是位於興東邨西南的居屋屋苑，門牌號碼為耀興道38號，為{{engname|興東邨第1期|Hing Tung Estate Phase 1}}，前身為成安村、聖十字徑村和澳貝龍村[[木屋區|木屋區]]<ref name=hk-place>[http://www.hk-place.com/view.php?id=143 香港地方-消失了的地方]</ref>，於1995年開始入伙，本屋苑設有4座大廈，由業主立案法團負責管理。屋苑設有一個多層[[停車場|停車場]]，與升降機大樓連接，通往興東商場，其中多層停車場由[[領展|領展]]負責管理，升降機大樓則由興東邨負責管理。
