'''常熟－合肥高速公路溧阳支线'''，简称'''溧阳支线'''，又称'''常溧高速公路'''，江苏省省级高速公路网编号S85，是[[常合高速公路|常合高速公路]]连接[[江苏省|江苏省]][[溧阳市|溧阳市]]的一条省内支线[[高速公路|高速公路]]。<ref>[http://www.jiangsu.gov.cn/jsgov/tj/bgt/201311/t20131103_404921.html 省政府办公厅关于印发江苏省省道公路网规划（2011－2020年）的通知]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>该线路西起溧阳市境内的[[扬溧高速公路|扬溧高速公路]]前马枢纽，向东北至[[常州市|常州市]][[武进区|武进区]]与[[金坛区|金坛区]]交界处的金坛经济开发区枢纽接入[[常合高速公路|常合高速公路]]，沿途经过溧阳市[[南渡镇_(溧阳市)|南渡镇]]、[[昆仑街道|昆仑街道]]、[[竹箦镇|竹箦镇]]、[[埭头镇_(溧阳市)|埭头镇]]、[[上黄镇|上黄镇]]、常州市金坛区[[儒林镇_(常州市)|儒林镇]]、[[尧塘镇|尧塘镇]]、武进区[[嘉泽镇|嘉泽镇]]。线路全长38.42公里，总投资28.14亿元，道路设计为双向四车道，采取全封闭、全立交模式，设计车速为每小时120公里。

常溧高速公路于2010年12月奠基<ref>{{Cite web |url = http://www.czjt.gov.cn/publicfiles/business/htmlfiles/czsgz/tpxw/201012/28105.html |title = 常州至溧阳高速公路正式奠基 |author = 常州市高速公路建设指挥部 |date = 2010-12-27 |accessdate = 2014-12-25 |deadurl = yes |archiveurl = https://web.archive.org/web/20150927164240/http://www.czjt.gov.cn/publicfiles/business/htmlfiles/czsgz/tpxw/201012/28105.html |archivedate = 2015-09-27 }}</ref>，2011年9月开工<ref>{{Cite web |url = http://dz.xdkb.net/html/2011-09/10/content_122555.htm |title = 常溧高速开工 预计2014建成 |author = 现代快报 |date = 2011-09-10 |accessdate = 2014-12-25}}</ref>，2015年9月26日建成通车<ref>{{Cite web |url = http://www.zhong5.cn/article-397175-1.html |title = 常溧高速26日通车 提升常州交通枢纽城市地位 |author = 常州晚报</font> |date = 2015-09-25 |accessdate = 2015-09-25}}</ref>。
