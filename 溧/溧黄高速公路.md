'''溧阳—黄山高速公路'''简称'''溧黄高速'''，江苏段编号S51<ref>[http://www.jscd.gov.cn/art/2010/7/21/art_11028_461281.html 江苏省高速公路命名和编号对照表] {{webarchive|url=https://web.archive.org/web/20130531102246/http://www.jscd.gov.cn/art/2010/7/21/art_11028_461281.html |date=2013-05-31 }}</ref>，安徽段编号S01<ref>[http://www.ahglj.com/preview/lzxxw/gsmm/ 安徽省高速公路命名和编号] {{webarchive|url=https://web.archive.org/web/20140415143248/http://www.ahglj.com/preview/lzxxw/gsmm/ |date=2014-04-15 }}</ref>，是连接[[中華人民共和國|中華人民共和國]][[江苏省|江苏省]]西南部和[[安徽省|安徽省]]东南部的一条南北向省际[[高速公路|高速公路]]。该线路起自江苏省[[溧阳市|溧阳市]]新昌枢纽，北接[[扬溧高速|扬溧高速]]，途经安徽省[[郎溪县|郎溪县]]、[[广德县|广德县]]、[[宁国市|宁国市]]、[[绩溪县|绩溪县]]及[[歙县|歙县]]，在歙县呈村降接入[[杭瑞高速|杭瑞高速]]。

根据2013年6月20日公布的《国家公路网规划(2013年-2030年) 》<ref>http://zfxxgk.ndrc.gov.cn/PublicItemView.aspx?ItemID={93c7d13b-aa0d-4beb-955e-268adade8a8f} {{webarchive|url=https://web.archive.org/web/20161016051120/http://zfxxgk.ndrc.gov.cn/PublicItemView.aspx?ItemID=%7B93c7d13b-aa0d-4beb-955e-268adade8a8f%7D |date=2016-10-16 }} 国家公路网规划(2013年-2030年)</ref>，溧黄高速公路已纳入[[中国国家高速公路网|中国国家高速公路网]]，成为{{China Expwy Name|G4012}}的一部分。

==过境路段==
===江苏段===
溧阳（新昌枢纽）至苏皖省界段，又称溧广高速公路，未开工。

===安徽段===
苏皖省界至广德（誓节枢纽）段，又称溧广高速公路，2013年9月1日开工<ref>{{Cite web |url = http://news.newsxc.com/shizhixinwen/2013-09-04/127472.html |title = 溧广高速公路试验段同时在郎溪广德两地开工 |date = 2013-09-04 |publisher = 宣城新闻网 |language = zh-cn |accessdate = 2015-01-21 |archiveurl = https://web.archive.org/web/20150121192909/http://news.newsxc.com/shizhixinwen/2013-09-04/127472.html |archivedate = 2015-01-21 }}</ref>，未通车。

广德至宁国（誓节枢纽至八里枢纽）段，又称广宁高速公路，未开工。

八里枢纽至宁国枢纽段，与{{China Expwy Name|S05|P=Anhui}}共线，2009年9月28日开工，2013年9月8日通车。<ref>{{Cite web |url = http://ah.anhuinews.com/system/2009/09/29/002347875.shtml |title = 宁宣杭高速公路宣城至宁国段开工 |date = 2009-09-29 |publisher = 中安在线 |language = zh-cn |accessdate = 2015-01-21 |archive-url = https://web.archive.org/web/20150121184315/http://ah.anhuinews.com/system/2009/09/29/002347875.shtml |archive-date = 2015-01-21 |dead-url = yes }}</ref><ref>{{Cite web |url = http://ah.anhuinews.com/qmt/system/2013/09/09/006056075.shtml |title = 宣城至宁国高速9月8日通车试运营 |date = 2013-09-09 |publisher = 中安在线 |language = zh-cn |accessdate = 2015-01-21 }}</ref>

宁国至绩溪（宁国枢纽至绩溪互通）段，又称宁绩高速公路，2009年11月26日开工，2014年12月24日通车。<ref>{{Cite web |url = http://ah.anhuinews.com/qmt/system/2009/11/27/002448088.shtml |title = 扬绩高速公路宁绩段开工建设 |date = 2009-11-27 |publisher = 中安在线 |language = zh-cn |accessdate = 2015-01-21 }}</ref><ref>{{Cite web |url = http://ah.anhuinews.com/system/2014/12/24/006635312.shtml |title = 宁绩高速今日正式通车 两地车程缩短为40分钟 |date = 2014-12-24 |publisher = 中安在线 |language = zh-cn |accessdate = 2015-01-21 }}</ref>

绩溪至歙县（绩溪互通至呈村降枢纽）段，又称绩黄高速公路，2008年7月30日开工，2012年1月16日通车。<ref>{{Cite web |url = http://ah.anhuinews.com/system/2008/07/30/002078872.shtml |title = 宣黄高速一期今天正式动工 |date = 2008-07-30 |publisher = 中安在线 |language = zh-cn |accessdate = 2015-01-21 }}</ref><ref>{{Cite web |url = http://www.cnjx.gov.cn/content/news_view.php?id=16126 |title = 绩黄高速公路通车试运行 |date = 2012-01-19 |publisher = 绩溪县人民政府 |language = zh-cn |accessdate = 2015-01-21 |deadurl = yes |archiveurl = https://web.archive.org/web/20150121192736/http://www.cnjx.gov.cn/content/news_view.php?id=16126 |archivedate = 2015-01-21 }}</ref>
