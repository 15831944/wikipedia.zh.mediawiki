'''溧阳市博物馆'''是一座位于[[中华人民共和国|中国]][[江苏省|江苏省]][[溧阳市|溧阳市]]的综合性城市[[博物馆|博物馆]]，位於溧阳市区南部，燕湖公园西侧。

溧阳市博物馆开建於2017年2月，2018年4月10日建成启用，同年11月2日向公众开放。建筑面积1.8万平方米，展区面积1.1万平方米。<ref name="lb1702">{{cite web |author1=冯国智 |title=文化艺术中心 博物馆项目建设火热推进 |url=http://www.lysk.net/index.php?m=content&c=index&a=show&catid=283&id=16841036 |website=溧阳时空网 |publisher=溧阳广播电视台 |accessdate=2018-11-02 |date=2017-02-07}}</ref><ref name="lb1804">{{cite web |author1=张浩 |title=中外嘉宾参观溧阳博物馆和规划馆 |url=http://www.lysk.net/index.php?m=content&c=index&a=show&catid=283&id=16849537 |website=溧阳时空网 |publisher=溧阳广播电视台 |accessdate=2018-11-02 |date=2018-04-10}}</ref>

== 展厅 ==
* 博物馆，位于建筑第一、二层，主要分为中华曙猿、溧阳通史、非物质文化遗产三个主题区域。
* 规划展示馆，位于建筑第三层，展览溧阳城市规划情况。
