'''溧水区'''是[[中华人民共和国|中华人民共和国]][[江苏省|江苏省]][[南京市|南京市]][[市辖区|市辖区]]之一，位于南京市南部，距南京主城区42千米。[[隋朝|隋朝]][[开皇|开皇]]十一年（591年），置'''溧水县'''。2013年2月，撤县设区。

因境内古[[溧水|溧水]]（亦称濑水，今之[[胥河|胥河]]）得名。又古称'''中山'''。地处江南[[长江三角洲|长江三角洲]]西部，地形主要为为丘陵山区和平原圩区，山水交错，风景优美。地处[[苏南|江苏西南部]]，近年来经济发展迅速，经济发展在江苏省处于较好水平。全区总面积1067平方公里，辖8个镇、1个省级开发区、2个国有农林场圃。

== 历史 ==
[[隋朝|隋朝]][[开皇|开皇]]十一年（591年），析[[溧阳县|溧阳县]]西北境及故[[丹阳县|丹阳县]]（治今南京市[[江宁区|江宁区]]小丹阳）地东部置溧水县。[[元朝|元朝]]时，升为[[溧水州|溧水州]]。[[明朝|明朝]][[洪武|洪武]]二年（1369年）降为县，属于[[应天府_(明朝)|应天府]]。[[弘治_(明朝)|弘治]]四年（1491年），又划出高淳镇置[[高淳县|高淳县]]<ref>《[[明史|明史]]·志第十六·地理一》：“应天府………溧水府东。元溧水州。洪武二年降为县。东南有东庐山，秦淮水别源出焉。南有石臼湖，西连丹阳湖，注大江。高淳府南。弘治四年以溧水县高淳镇置………”</ref>。[[清朝|清朝]]时，属[[江宁府|江宁府]]。

1949年4月23日，[[中国人民解放军第35军|中国人民解放军第35军]]攻占[[南京城|南京城]]。25日，解放军追击国军进入溧水县境，宣告“溧水解放”。28日，[[中国人民解放军第34军|中国人民解放军第34军]]进入溧水县城。同日，中共任命的县长[[张君实|张君实]]抵达溧水。30日，中共溧水县委书记[[苏进程|苏进程]]等人抵达溧水，公布政府行政机构和工作人员名单。5月1日，宣布[[溧水县人民政府|溧水县人民政府]]成立<ref>{{Cite web |url = http://www.lishuicn.com/article-322-1.html |title = 新中国60年溧水县大事记 （一） |accessdate = 2015-01-11 |author =  |date = 2012-03-06 |work =  |publisher = 溧水信息港，来源：卞氏人家 友情提供--溧水县档案局 县志办编 2011年9月6日 |language = zh-hans |deadurl = yes |archiveurl = https://web.archive.org/web/20150210041141/http://www.lishuicn.com/article-322-1.html |archivedate = 2015年2月10日 }}</ref>。1949年后从[[南京|南京]]划出，属[[镇江专区|镇江专区]]，1958年属[[常州专区|常州专区]]，1959年9月，属[[镇江专区|镇江专区]]。1983年1月，划回南京市<ref>{{Cite web|url=http://njls.gov.cn/jbgk.htm|title=基本概况|accessdate=2015-01-11|author=|date=|work=|publisher=南京市溧水县人民政府网站，来源：县方志办　县国土局|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20130726054837/http://njls.gov.cn/jbgk.htm|archivedate=2013-07-26}}</ref>。2013年2月，撤县设区。
==地理==
溧水区东邻常州[[溧阳|溧阳]]市，南连南京市[[高淳区|高淳区]]，西与安徽省马鞍山市[[博望区|博望区]]毗邻，西北与南京市[[江宁区|江宁区]]交界，东北与镇江市句容市接壤。属于宁镇扬丘陵山区，地势东南高西北低，低山丘陵面积占总面积72.5%，最高海拔368.5米。区境内浮山、东庐山、回峰山、芳山、秋湖山、无想山拱据东南，连绵环列，西横山突兀西端，逶迤绵延。

==行政区划==
现区政府驻地为[[永阳街道|永阳街道]]（原名[[在城镇|在城镇]]），下辖8个镇分别为[[永阳街道|永阳街道]]、[[洪蓝街道|洪蓝街道]]、[[石湫街道|石湫街道]]、[[柘塘街道|柘塘街道]]、[[白马镇|白马镇]]、[[东屏街道_(南京市)|东屏街道]]、[[晶桥镇|晶桥镇]]和[[和凤镇|和凤镇]]。由于近年来经济的发展，原来老开发区的土地大都已经开发完毕，目前[[柘塘街道|柘塘街道]]的全部和[[东屏街道_(南京市)|东屏街道]]的部分地区已经并入[[溧水经济开发区|溧水经济开发区]]。

==方言==
溧水区位于[[吴方言|吴方言]]与[[北方方言|北方方言]]江淮次方言的分界线上，白马、晶桥、石湫、和凤、洪蓝五镇为吴方言区；柘塘和东屏的群力为江淮方言区；其中[[永阳镇_(南京市)|永阳镇]]新城区多讲吴方言，老城区多讲江淮方言，永阳镇的原城郊、东庐基本说吴方言；吴方言占总人口比重超过75%。

==旅游==
溧水风景秀丽，兼有山区和水乡两种风貌，植被保存完好，森林覆盖率高，旅游资源丰富，在环境改善方面先后获得国家生态示范县、国家卫生城市和国家园林城市等荣誉称号。溧水主要有天生桥（AAA）、傅家边(AAAA)大金山国防园(AAA)、无想山的无想寺天池、东庐山的观音寺等知名景点，还有已经开工建设的江苏石湫影视基地，一年一度的梅花节草莓节更是吸引了众多的游客前来观光旅游。而近年来由于生态环境的进一步改善，一些国家保护的野生动物也可以时常在溧水的乡间经常看到：在石臼湖，每年都有数千只天鹅前来越冬，被人们称为“天鹅湖”；在一些山区野猪野兔山鸡甚至已经“泛滥成灾”。溧水的特产也很丰富，有乾隆皇帝下江南时品尝并赐名的玉带糕，群力的老鹅，石湫的狗肉，明觉的酸菜，白马的黑莓，傅家边的梅子草莓等。

== 交通 ==
溧水是长三角重要交通节点，交通十分发达。方圆200公里范围内，分布着南京、上海、杭州、合肥、苏州、无锡、常州等40多个大中型城市。目前已逐步形成连接五大洲沟通全世界的水陆空并举的立体交通网。溧水区即將成為江苏省的四大“长三角最具投资价值的地区”（[[江阴|江阴]]、'''溧水'''、[[常熟|常熟]]、[[如皋|如皋]]）。

; 高速公路
宁杭、宁高、沿江、溧马(马鞍山)高速、溧镇(江)高速等数条高速公路穿区而过，是中国拥有高速公路最多的区市之一。
距南京只有0.5小时车程；距杭州、只有2小时车程；距上海、只有2.5小时车程。

; 空運
距南京[[禄口国际机场|禄口国际机场]]14公里。

; 水運
水路经[[秦淮河|秦淮河]]、[[石臼湖|石臼湖]]可直达[[长江|长江]]。距南京新生圩港59公里。

; [[中国铁路总公司|国铁]]
连接江浙两省省会的国家干线高速铁路[[宁杭城际铁路|宁杭城际铁路]]在溧水设站，已于2012年10月建成通车。距南京南站47公里。

; [[城市轨道交通|城市轨道交通]]
溧水区境内有两条属于[[南京地铁|南京地铁]]系统的轨道交通正在建设：[[宁高城际|宁高城际]]（S1号线）二期工程已全面开工，[[宁溧城际|宁溧城际]]（S7号线）试验段已开工。
