'''溧阳－芜湖高速公路'''，简称'''溧芜高速'''，江苏段编号S68<ref>{{cite web |url=http://www.jscd.gov.cn/art/2010/7/21/art_11028_461281.html |title=存档副本 |accessdate=2015-01-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20130531102246/http://www.jscd.gov.cn/art/2010/7/21/art_11028_461281.html |archivedate=2013-05-31 }} 江苏省高速公路命名和编号对照表</ref>，安徽段编号S28<ref>{{cite web |url=http://www.ahglj.com/preview/lzxxw/gsmm/ |title=存档副本 |accessdate=2014-04-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20140415143248/http://www.ahglj.com/preview/lzxxw/gsmm/ |archivedate=2014-04-15 }} 安徽省高速公路命名和编号</ref>，是连接[[中华人民共和国|中华人民共和国]][[江苏省|江苏省]]南部和[[安徽省|安徽省]]东部的一条东西向[[高速公路|高速公路]]。该线路起自[[江苏省|江苏省]][[溧阳市|溧阳市]][[溧黄高速|溧黄高速]]南渡南枢纽（规划），途经[[南京市|南京市]][[高淳区|高淳区]]、[[安徽省|安徽省]][[宣城市|宣城市]][[宣州区|宣州区]]雁翅，止于[[芜湖县|芜湖县]][[六郎镇|六郎镇]]，在六郎枢纽接入[[沪渝高速|沪渝高速]]。

==过境路段==
===江苏段===
{{China Expwy Name|S68|P=Jiangsu}}

溧阳南渡至高淳和凤段（简称'''溧高高速'''），全长34.6公里，溧阳段於2017年4月开工<ref>{{cite web |author1=汪璐 |title=常州新添一条通往皖南的高速，2小时到芜湖，1.5小时到高淳 |url=http://www.ourjiangsu.com/a/20171214/1513222828138.shtml |website=我苏网 |accessdate=2019-01-20 |language=zh-cn |date=2017-12-14}}</ref>，高淳段於2017年10月开工<ref>{{cite web |author1=王芳 |author2=宋端平 |title=溧高高速公路施工现场 |url=http://www.njgc.gov.cn/jrgc/tpxw/201805/t20180509_293152.html |website=南京市高淳区政府网 |accessdate=2019-01-20 |language=zh-cn |date=2018-05-09}}</ref>。

高淳和凤至苏皖省界段（简称'''高芜高速'''或'''淳芜高速'''），全长29.7公里，2009年12月29日开工<ref>http://js.xhby.net/system/2009/12/30/010656837.shtml 溧水至马鞍山、高淳至芜湖高速开工</ref> <ref> ，2015年2月3日通车http://ah.anhuinews.com/system/2015/02/04/006675596.shtml 芜湖至高淳高速公路通车</ref> 。

===安徽段===
{{China Expwy Name|S28|P=Anhui}}

苏皖省界至芜湖段（简称'''芜雁高速'''），全长16.9公里，2006年12月9日开工，2012年12月20日通车<ref>{{cite web |url=http://www.whsjtj.gov.cn/Content.aspx?pNewID=1573 |title=存档副本 |accessdate=2013-03-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20150204080652/http://www.whsjtj.gov.cn/Content.aspx?pNewID=1573 |archivedate=2015-02-04 }} 芜雁高速公路正式开通</ref>。
