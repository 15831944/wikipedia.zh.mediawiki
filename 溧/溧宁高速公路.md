'''溧阳－宁德高速公路'''（[[中国国家高速公路网|国家高速公路网]]编号'''G40<small>12</small>'''，简称'''溧宁高速'''），是[[沪陕高速公路|G40]]的联络线之一。该线路起自江苏省[[溧阳市|溧阳市]]新昌枢纽，北接[[扬溧高速|扬溧高速]]，途经安徽省[[宁国市|宁国市]]、[[绩溪县|绩溪县]]、[[歙县|歙县]]、浙江省[[建德市|建德市]]、[[龙游县|龙游县]]、[[云和县|云和县]]、[[泰顺县|泰顺县]]，终于福建省[[福安市|福安市]]坂中枢纽，南接[[甬莞高速公路|甬莞高速公路]]。<ref>[http://zfxxgk.ndrc.gov.cn/PublicItemView.aspx?ItemID={4f589ce2-c398-4770-858d-3036b98fa405} 国家高速公路网规划简介] {{webarchive|url=https://web.archive.org/web/20160306182347/http://zfxxgk.ndrc.gov.cn/publicitemview.aspx?itemid=%7B4f589ce2-c398-4770-858d-3036b98fa405%7D |date=2016-03-06 }}</ref><ref>[http://zfxxgk.ndrc.gov.cn/PublicItemView.aspx?ItemID={93c7d13b-aa0d-4beb-955e-268adade8a8f} 国家公路网规划(2013年-2030年)] {{webarchive|url=https://web.archive.org/web/20161016051120/http://zfxxgk.ndrc.gov.cn/PublicItemView.aspx?ItemID=%7B93c7d13b-aa0d-4beb-955e-268adade8a8f%7D |date=2016-10-16 }}</ref>。

== 路段 ==
===江苏段===
溧阳至苏皖省界（原规划为{{China Expwy Name|S51|P=Jiangsu}}，又称溧广高速公路江苏段），2018年12月18日开工<ref>{{cite news |author1=石小磊 |title=我省再开工两条高速公路 苏皖再增高速通道 |url=http://epaper.yzwb.net/html_t/2018-12/20/content_499345.htm |accessdate=2019-01-20 |work=扬子晚报 |date=2018-12-20 |page=A2 |language=zh-cn}}</ref>，未通车。

===安徽段===
苏皖省界至广德誓节段（原规划为{{China Expwy Name|S01|P=Anhui}}，又称溧广高速公路安徽段），2013年9月1日开工<ref>{{Cite web |url = http://news.newsxc.com/shizhixinwen/2013-09-04/127472.html |title = 溧广高速公路试验段同时在郎溪广德两地开工 |date = 2013-09-04 |publisher = 宣城新闻网 |language = zh-cn |accessdate = 2015-01-21 |archiveurl = https://web.archive.org/web/20150121192909/http://news.newsxc.com/shizhixinwen/2013-09-04/127472.html |archivedate = 2015-01-21 }}</ref>，2016年9月30日通车<ref>{{cite web|author1=陈欣然 |title=安徽省对接长三角再添一条快速通道 溧阳至广德高速安徽段通车试运营|url=http://ah.anhuinews.com/system/2016/09/30/007474682.shtml|website=中安在线|accessdate=2016-12-18|date=2016-10-01}}</ref>。

广德誓节至宁国河沥溪段（原规划为{{China Expwy Name|S01|P=Anhui}}，又称广宁高速公路），2016年6月15日开工<ref>{{cite web|author1= |title=广宁高速破土动工|url=http://www.ngcm.cn/articleview/2016-6-15/article_view_41653.htm|website=宁国传媒网|accessdate=2017-11-18|date=2016-06-15}}</ref>，2018年12月20日通车<ref>{{cite web |author1=胡旭 |author2=张敬波 |author3=金海礁 |title=广宁和滁淮高速公路通车运营 全省高速公路网通车里程增加163公里 |url=http://xc.anhuinews.com/system/2018/12/20/008037598.shtml |website=中安在线网 |publisher=安徽日报 |accessdate=2019-01-20 |language=zh-cn |date=2018-12-21}}</ref>。

歙县呈村降至皖浙省界（原规划为S48{{China Expwy Name|S48|[[建黄高速公路|建黄高速]]|P=Anhui}}），未开工。

丽水北埠至云和段（丽龙高速公路），与{{China Expwy Name|G25}}共线，2006年12月31日通车。

云和至景宁段（原属{{Zhejiang Expwy Name|S34}}，又称云景高速公路），2008年3月20日开工<ref>{{cite web|author1=丽水日报 |title=龙丽温（泰）高速公路云景段昨开建|url=http://www.lishui.gov.cn/zfzx/tpxx/200803/t20080321_834106.html|website=丽水市政府网|accessdate=2016-12-18|date=2008-03-21}}</ref>，2013年4月10日通车<ref>{{cite web|author1=夏丹|author2=张帆|author3=叶尚蓉|title=云景高速公路正式开通 畲乡融入“四小时经济圈”|url=http://zjnews.zjol.com.cn/05zjnews/system/2013/04/11/019271824.shtml|website=浙江在线|accessdate=2016-12-18|date=2013-04-11}}</ref>。

景宁至文成段（原规划为{{Zhejiang Expwy Name|S34}}景宁至文成段，又称景文高速公路），2018年12月14日开工<ref>{{cite web |title=景文高速开工建设 |url=http://www.zj.gov.cn/art/2018/12/24/art_13104_2299939.html |website=浙江省政府网 |accessdate=2019-01-20 |date=2018-12-24}}</ref>，未通车。

文成至浙闽省界（原规划为{{Zhejiang Expwy Name|S35}}，又称龙丽温高速公路文泰支线），2016年12月15日开工<ref>{{cite web|author1=浙江省交通运输厅|title=龙丽温高速文泰段于12月15日开工|url=http://www.zjt.gov.cn/art/2016/12/16/art_17_968201.html|website=浙江交通|accessdate=2016-12-18|date=2016-12-16|deadurl=yes|archiveurl=https://web.archive.org/web/20161220153225/http://www.zjt.gov.cn/art/2016/12/16/art_17_968201.html|archivedate=2016-12-20}}</ref>，未通车。
