'''朱少麟'''{{BD|1966年|10月2日||}}，本名朱蕙麟，出生於[[台灣|台灣]][[嘉義市|嘉義市]]，祖籍[[湖北|湖北]]，為台灣當代女性作家。

== 簡傳 ==

朱少麟，曾就讀[[曉明女中|曉明女中]]、[[輔仁大學|輔仁大學]]英國語文學系畢業，早先是在從事政治公關公司工作之餘，挑燈寫下第一本小說《[[傷心咖啡店之歌|傷心咖啡店之歌]]》，獲得廣大好評，被譽為「天生的作家」。第二部小說《[[燕子_(小說)|燕子]]》之後則專職寫作，五年多後方有第三部小說《[[地底三萬呎|地底三萬呎]]》出版。

媒體曾指出，作者寫第一本書時，是想要叛離生活，寫第二本時是想試試自己是否能夠持續寫作，而第三本《地底三萬呎》，則是為了想寫出一本自己心目中的最佳作品、想挑戰自己能夠超越的藝術成就。

[[File:吟陸咖啡的明信片_朱少麟.jpg|朱少麟留給吟陸咖啡的明信片（左上角二）]]


2000年，朱少麟獲得[[中國文藝協會|中國文藝協會]]頒發的[[中國文藝獎章|中國文藝獎章]]小說創作獎。


==著作==
目前出版了三本[[長篇小說|長篇小說]]，皆由[[九歌出版社|九歌出版社]]出版：
*《[[傷心咖啡店之歌|傷心咖啡店之歌]]》（1996年）<ref>{{cite web |url=http://www.kingstone.com.tw/Book/Book_Page.asp?id=2018573741488 |title=存档副本 |accessdate=2006-04-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20060306065033/http://www.kingstone.com.tw/Book/Book_Page.asp?id=2018573741488 |archivedate=2006-03-06 }}</ref>
*《[[燕子_(小說)|燕子]]》（1999年）
*《[[地底三萬呎|地底三萬呎]]》（2005年）。
* 其三本巨著的簡体字版亦陸續於中國正式發行。

== 關於傷心咖啡店 ==
關於傷心咖啡店，朱少麟是以素人演員[[陸弈靜|陸弈靜]]開設<ref>參考</ref>的[[臺北市|臺北市]][[景美|景美]][[景中街|景中街]]「[[吟陸商號|吟陸商號]]」（吟陸咖啡，舊名「芙陸咖啡」、「聯禾咖啡」）為藍圖，朱少麟也常在吟陸商號寫作《[[傷心咖啡店之歌|傷心咖啡店之歌]]》一書。對於此說，朱少麟沒有直接承認，但曾有報導指出他習慣在隱密的狀態動筆，應不會在咖啡館裡寫作。

吟陸商號於2006年四月[[轉讓|轉讓]]，不再由陸弈靜經營；據媒體訪談朱少麟內容得知，《傷心咖啡店之歌》一書所提及的人物，除了海安之外，均真有其人。
