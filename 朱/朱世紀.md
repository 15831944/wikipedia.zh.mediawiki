'''朱世紀'''（{{bd|1940年||2007年|2月9日}}）是[[中華民國|中華民國]]（[[臺灣|臺灣]]）政治人物，是[[左派台獨|左派]][[臺灣獨立運動|臺灣獨立運動]]主要領導人之一，主張「[[無產階級專政|無產階級專政]]」。其早年在[[日本|日本]]活動，先後成立「臺灣學生聯誼會」、「料理會」等組織，並創辦[[社會主義|社會主義]]雜誌《[[台灣文化_(雜誌)|臺灣文化]]》。在返回臺灣後，他先後投入「[[台灣建國運動組織|台灣建國運動組織]]」、「台灣衝組」、「臺灣民族革命黨」的組織活動，支援各種議題的示威遊行，後來成為[[建國黨|建國黨]]秘書長。其後則在[[臺北市|臺北市]][[光華商場|光華商場]]附近擺攤維生，並因心肌梗塞逝世。

== 早年生活 ==
1940年，朱世紀在[[台灣日治時期|日治臺灣]][[宜蘭縣|宜蘭]]鄉下出生{{sfnm|陳銘城|1992年|1loc=第78頁|簡笙簧等|2004年|2loc=第238頁}}。其父親朱阿輝是[[臺灣共產黨|臺灣共產黨]]成員，因為羅東紙廠罷工案被逮捕，後來因為遭到[[臺灣總督府警察|臺灣總督府警察]]刑求，出獄後不久即逝世{{sfnm|陳銘城|1992年|1loc=第78頁}}。由於父母親從小逝世，朱世紀居住在姑媽家，堂兄[[陳都|陳都]]後來成為[[世界台灣同鄉會聯合會|世界台灣同鄉會聯合會]]會長{{sfnm|陳銘城|1992年|1loc=第78頁}}。朱世紀先後就讀[[國立宜蘭高級中學|臺灣省立宜蘭中學]]高中部與[[國立臺灣大學|國立臺灣大學]]法律學系<ref name = "劉開元 01">{{cite news |language = Zh-hant |author = 劉開元和孔令琪 |coauthors =  |url =  |title = 涉多次參與群眾抗爭 旅日男子朱世紀被捕 |publisher = 《[[聯合晚報|聯合晚報]]》 |pages = 7版 |date = 1993年7月1日 |accessdate = 2019年4月18日 }}</ref>，[[林義雄|林義雄]]便是他的高中同學、以及大學晚一屆的學弟{{sfnm|陳銘城|1992年|1loc=第78頁}}。1964年，他前往[[日本|日本]][[早稻田大學|早稻田大學]]留學{{sfnm|陳銘城|1992年|1loc=第78頁|簡笙簧等|2004年|2loc=第238頁}}。在早稻田大學就讀1個學年後，轉至[[法政大學_(日本)|法政大學]]寄籍就讀6年，並利用學生身分取得「[[永久居留證|在留卡]]」，掩護自己從事學生運動{{sfnm|陳銘城|1992年|1loc=第78頁}}。

朱世紀長期對於「[[台灣青年社|台灣青年社]]」的作風有所批評，反對所有的成員經常參與街頭示威，認為這將會導致組織政治色彩過於濃厚{{sfnm|陳銘城|1992年|1loc=第78頁至第79頁}}。在這時期，他一方面打工，一方面從事留學學生運動，並且研究[[共產主義|共產主義]]思想<ref name = "盧福賓 01">{{cite news |language = Zh-hant |author = 盧福賓 |coauthors =  |url =  |title = 「台灣衝組」稱將建立武裝力量 |publisher = 《[[聯合報|聯合報]]》 |pages = 4版 |date = 1996年11月7日 |accessdate = 2019年4月18日 }}</ref>{{sfnm|陳銘城|1992年|1loc=第79頁}}。1956年，他組織「臺灣學生聯誼會」，成員約有400人至500人，當中約有80%的成員後來返回臺灣{{sfnm|陳銘城|1992年|1loc=第78頁至第79頁}}。1964年，他參加日本的「台灣民族解放陣線」組織<ref name = "劉開元 01"/><ref name = "朱復元 02">{{cite news |language = Zh-hant |author = 朱復元 |coauthors =  |url =  |title = 六二五鬧事傷警 朱世紀被收押 |publisher = 《[[聯合報|聯合報]]》 |pages = 7版 |date = 1993年7月2日 |accessdate = 2019年4月18日 }}</ref>。當時朱世紀的「台灣民族解放陣線」與[[史明|史明]]的「[[獨立臺灣會|獨立臺灣會]]」同時反對日本[[左派|左派]]人士與[[右派|右派]]的[[台灣獨立建國聯盟|台灣獨立建國聯盟]]<ref>{{Cite web |url = http://www.taiwannation.org.tw/wufi/wufijpn.htm |title = 日本的台獨運動 |author = [[黃昭堂|黃昭堂]] |date =  |publisher = 現代文化基金會 |language = Zh-hant |accessdate = 2019年4月8日 |quote =  }}</ref>。

1972年，隨著[[中華民國|中華民國]]與日本[[中華民國與日本關係|斷交]]，許多「臺灣學生聯誼會」幹部被[[在日台灣同鄉會|在日台灣同鄉會]]吸納{{sfnm|陳銘城|1992年|1loc=第78頁至第79頁}}。1973年，朱世紀在[[東京|東京]]創辦、發行中文版的[[社會主義|社會主義]]刊物《[[台灣文化_(雜誌)|臺灣文化]]》，提供給不能完全理解[[日語|日語]]的留學學生閱讀{{sfnm|陳銘城|1992年|1loc=第79頁、第241頁|簡笙簧等|2004年|2loc=第238頁}}。到了1990年開始，由於他認為在日台灣同鄉會應該和[[台灣獨立運動|台灣獨立運動]]分隔開來，另外舉辦針對學生的「料理會」{{sfnm|陳銘城|1992年|1loc=第78頁至第79頁}}。在這時期，他藉由經營分租宿舍維生，每個月約有100萬[[日圓|日圓]]的收入，這也成為其持續發行《台灣文化》雜誌及「料理會」的經費來源{{sfnm|陳銘城|1992年|1loc=第79頁}}。1992年8月底，他獲准返回臺灣{{sfnm|陳銘城|1992年|1loc=第79頁}}。

1992年12月25日，朱世紀獨自返回臺灣，妻子與兒子則留在日本<ref name = "朱復元 02"/>，爾後其經常往返日本與臺灣<ref name = "劉開元 與此同時，朱世紀還積極和「[[台灣建國運動組織|台灣建國運動組織]]」的成員建立關係<ref name = "朱復元 01"/>，並擔任該組織的總幹事，試圖重建當時逐漸式微的台灣建國運動組織{{sfnm|簡笙簧等|2004年|1loc=第238頁至第239頁}}。他與台灣建國運動組織的成員多次參與、支援不同團體的示威遊行<ref name = "劉開元 01"/>{{sfnm|簡笙簧等|2004年|1loc=第252頁}}。1996年，他與該組織的核心幹部[[廖宗環|廖宗環]]積極聯繫各個體制外的台灣獨立運動團體，號召共同展開「臺灣獨立建國」的行動{{sfnm|簡笙簧等|2004年|1loc=第239頁}}。同年10月至11月，在先後舉辦兩場「建國兄弟研習營」後，朱世紀正式對外公開自己的組織，並更名為「台灣衝組」<ref name = "彭威晶 01">{{cite news |language = Zh-hant |author = 彭威晶 |coauthors =  |url =  |title = 激進團體 罕見的理性 |publisher = 《[[聯合報|聯合報]]》 |pages = 3版 |date = 1997年5月5日 |accessdate = 2019年4月18日 }}</ref>{{sfnm|簡笙簧等|2004年|1loc=第236頁、第238頁至第239頁}}。「台灣衝組」主張建立武裝力量，並且不定期發行宣傳文件《臺灣衝組報》與《臺灣革命報》<ref name = "盧福賓 01"/>{{sfnm|簡笙簧等|2004年|1loc=第239頁}}。

1997年5月4日，朱世紀與「台灣衝組」成員參加「504為台灣而走」遊行，但在遊行前遭到主辦單位請離，隨後其宣傳車自行前往[[凱達格蘭大道|凱達格蘭大道]]演講<ref name = "彭威晶 01"/>。同年，由於「台灣建國運動組織」和「台灣衝組」成員並未形成一個「堅強的[[統一戰線|統一戰線]]」，他將「台灣衝組」改組為「[[台灣民族革命黨|台灣民族革命黨]]」{{sfnm|簡笙簧等|2004年|1loc=第236頁、第239頁至第240頁}}。而因為缺乏大眾傳播工具進行組織招募，他則傾向透過「運動人際網絡」中的面對面私下接觸，主動前往其他運動團體和具參與潛力的組織建立關係{{sfnm|簡笙簧等|2004年|1loc=第267頁至第268頁、第281頁至第282頁}}。他四處接觸包括[[全民計程車隊|全民計程車隊]]、[[台灣勞工運動|勞工運動]]與農民運動人士，試圖藉此補充新成員，因此經常在各個團體之間活動{{sfnm|簡笙簧等|2004年|1loc=第239頁至第240頁、第267頁至第268頁}}。

1998年，朱世紀以無黨籍的個人名義參與[[1998年中華民國立法委員選舉|立法委員選舉]]，為[[臺北市第二選舉區_(立法委員)|臺北市第二選舉區]]候選人<ref>{{cite news |language = Zh-hant |author = 林松青 |coauthors =  |url =  |title = 北縣侯選人間出現領先群 |publisher = 《[[聯合報|聯合報]]》 |pages = 8版 |date = 1998年11月25日 |accessdate = 2019年4月18日 }}</ref>，最終獲得333張選票<ref>{{Cite web |url = http://elections.olc.tw/candidates/view/54d9c46a-dcfc-4acd-a6b5-6b30acb5b862 |title = 1998-12 立法委員 臺北縣 第2選區區域 候選人 |author =  |date =  |publisher = 選舉黃頁 |language = Zh-hant |accessdate = 2019年4月8日 |quote =  }}</ref>。熱衷政治的他後來還擔任[[建國黨|建國黨]]秘書長<ref name = "姚岳宏 01">{{Cite web |url = https://news.ltn.com.tw/news/society/paper/118844 |title = 接到罰單 心臟病發猝死 |author = 姚岳宏和王述宏 |date = 2007年3月6日 |publisher = 《[[自由時報|自由時報]]》 |language = Zh-hant |accessdate = 2019年4月8日 |quote =  }}</ref>。1999年4月，朱世紀等10多人前往[[慈湖陵寢|慈湖陵寢]]抗議，揚言對蔣中正鞭屍<ref>{{cite news |language = Zh-hant |author = 林文義 |coauthors =  |url =  |title = 台灣衝組 赴慈湖抗議 |publisher = 《[[聯合報|聯合報]]》 |pages = 4版 |date = 1999年4月5日 |accessdate = 2019年4月18日 }}</ref>。後來他選擇在[[臺北市|臺北市]][[光華商場|光華商場]]附近擺攤，以販賣豬血糕維持生計，自己便獨自在附近租屋居住<ref name = "姚岳宏 01"/>。但由於是違法設攤，他遭到[[臺北市政府警察局中正第一分局|臺北市政府警察局中正第一分局]]忠孝東路派出所多次取締<ref name = "姚岳宏 01"/>。後來他成為臺北市攤販自救會會長，曾帶著攤販發起抗議行動<ref name = "姚岳宏 01"/><ref name = "鄭勝為 01">{{Cite web |url = https://news.tvbs.com.tw/entry/332407 |title = 〈獨家〉罰單氣死人！　攤商遭罰心肌梗塞亡 |author = 鄭勝為 |date = 2007年3月6日 |publisher = [[TVBS新聞台|TVBS新聞台]] |language = Zh-hant |accessdate = 2019年4月8日 |quote =  }}</ref>。2007年2月9日，在遭到警方取締告發後不久，他因為[[心肌梗塞|心肌梗塞]]猝死<ref name = "姚岳宏 01"/><ref name = "鄭勝為 01"/><ref>{{Cite web |url = https://www.thestandnews.com/politics/台灣/台灣主體性的建構與危機-從二二八到太陽花-三/ |title = 台灣主體性的建構與危機：從二二八到太陽花（三） |author = 陳峻毅 |date = 2016年4月25日 |publisher = [[立場新聞|立場新聞]] |language = Zh-hant |accessdate = 2019年4月8日 |quote =  }}</ref>。

== 思想主張 ==
{{Quote box| width = 300px | align = left | quoted = true | bgcolor = #FFFFF0 | salign = right | quote = ……只要社會上存在著階級的對立，只要有一部份人在榨取另一部份人，則這戰爭就成了不可避免的事，我們為了絕滅這榨取與剝削，除了戰鬥之外找不到任何可以解決的方法。 | source= 朱世紀{{sfnm|簡笙簧等|2004年|1loc=第251頁}} }}

在1960年代至1970年代，朱世紀已經在日本發展出自己的[[政治意識形態列表|政治意識形態]]，並自此長期主張台灣獨立運動與[[馬克思列寧主義|馬克思列寧主義]]{{sfnm|陳銘城|1992年|1loc=第78頁|簡笙簧等|2004年|2loc=第246頁、第263頁至第264頁}}。他強調以[[工人階級|工人階級]]與[[農民階級|農民階級]]作為[[工農聯盟|行動主體]]，進而推翻[[蔣中正|蔣中正]]政府{{sfnm|陳銘城|1992年|1loc=第78頁}}。朱世紀認為在社會主義[[世界革命|世界革命]]的總體利益中，台灣獨立運動是革命的其中一部分，即在推動臺灣獨立運動的同時，也必須藉由「[[無產階級專政|無產階級專政]]」、「被壓迫民族專政」等，朝向更為終極的目標——社會主義革命與世界革命{{sfnm|簡笙簧等|2004年|1loc=第246頁、第263頁}}。他因而主張無產階級專政是從[[資本主義|資本主義]]過渡到共產主義的必然階段{{sfnm|陳銘城|1992年|1loc=第78頁|簡笙簧等|2004年|2loc=第251頁至第252頁}}。

朱世紀認為臺灣社會仍然處在[[資產階級|資產階級]]專政的階段，[[中國國民黨|中國國民黨]]是資產階級和外國資本的統治工具，利用[[大漢族主義|大漢沙文主義]]來阻止臺灣[[民族自決|民族自決]]，亦會利用[[民主|民主]]和選舉的西方「資產階級民主」予以修飾，透過有形與無形的手段對無產階級進行「階級專政」{{sfnm|簡笙簧等|2004年|1loc=第246頁、第251頁至第252頁}}。他還強調在達到台灣獨立建國和無產階級專政的過程中，存在著「暴力、武力」手段的必然性，認為放棄無產階級革命便是放棄革命自身，將永遠無法從資本主義社會過渡到共產主義社會{{sfnm|簡笙簧等|2004年|1loc=第251頁至第252頁、第263頁}}。他認為台灣存在立即革命的條件，革命力量必須透過武力行動不斷累積，並且應當排除任何選舉或[[改良主義|改良主義]]的手段{{sfnm|簡笙簧等|2004年|1loc=第248頁至第249頁、第251頁}}。

朱世紀還將[[社會階級|社會階級]]的立場本質化，認為整個[[臺灣歷史|臺灣歷史]]的本質並非民族壓迫、而是社會階級的問題，民族的壓迫只是台灣歷史的現象{{sfnm|簡笙簧等|2004年|1loc=第246頁}}。因此他批評史明過於強調民族壓迫，不但沒有凸顯出[[階級衝突|階級矛盾]]，反而是將階級矛盾的問題加以掩蓋<ref>{{Cite web |url = https://www.thestandnews.com/politics/台灣主體性的建構與危機-從二二八到太陽花-二/ |title = 台灣主體性的建構與危機：從二二八到太陽花（二） |author = 陳峻毅 |date = 2016年4月22日 |publisher = [[立場新聞|立場新聞]] |language = Zh-hant |accessdate = 2019年4月8日 |quote =  }}</ref>{{sfnm|簡笙簧等|2004年|1loc=第246頁}}。在[[1994年中華民國省市長暨省市議員選舉|1994年省市長暨省市議員選舉]]期間，針對獨立臺灣會和[[新黨|新黨]]人士發生衝突對峙事件，當時認為組織還不夠健全的史明為了避免出現武力衝突，勸退獨立臺灣會的成員，朱世紀則批評史明的表現過於軟弱，而且「時空認識錯誤」{{sfnm|簡笙簧等|2004年|1loc=第248頁至第249頁}}。而由於自己參與的組織運作鬆散、力量薄弱，朱世紀還主張建立一個[[革命|革命]]性質的「前衛黨組織」{{sfnm|簡笙簧等|2004年|1loc=第252頁、第254頁}}。
