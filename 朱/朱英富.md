'''朱英富'''（{{bd|1941年|7月12日|}}），[[中国|中国]]著名舰艇设计及造船专家，[[中国海军|中国海军]]国防中多种[[现代化|现代化]]舰艇的总设计师，包括新近下水引起世界关注的[[辽宁号航空母舰|辽宁号航空母舰]]<ref name="朱英富 上海交通大学">{{cite web |url = http://news.sjtu.edu.cn/info/1002/128715.htm |title = 上海交大校友朱英富、张峥荣任我国第一艘航母总设计师和舰长[图] |publisher = [[上海交通大学|上海交通大学]]新闻网 |author = 责任编辑: snew |date = [发布时间]: 2012年09月26日 |accessdate = 2012年12月2日 |language = zh-hans |archive-url = https://web.archive.org/web/20121105134206/http://news.sjtu.edu.cn/info/1002/128715.htm |archive-date = 2012-11-05 |dead-url = yes }}</ref>。

== 生平 ==
籍贯[[浙江|浙江]][[宁波|宁波]]，1941年出生于[[上海|上海]]。父母都是宁波人，都是上海织造的工人，家庭贫困。家中有六个子女，朱英富排行老大。少年时本打算赴[[辽宁|辽宁]][[大连|大连]]学习船舶驾驶，但因家贫付不起路费，遂放弃。<ref name="朱英富 隐形舰船">{{cite web |url = http://news.163.com/11/1209/03/7KQ8521U00014AED.html |title = 朱英富：“隐形舰船”总设计师(图) |publisher = [[网易|网易]]新闻 |author = 来源：长江日报(武汉) |date = 2011-12-09 |accessdate = 2012年12月2日 |language = zh-hans}}</ref>

考入[[上海交通大学|上海交通大学]]，学习[[船舶工程|船舶工程]]。1966年，毕业于上海交通大学。被分配到[[武汉船舶设计研究所|武汉船舶设计研究所]]（[[中国船舶重工集团公司|中国船舶重工集团公司]]第701研究所），之后便一直任职于该所。现任701所所长。<ref name="朱英富 上海交通大学" /><ref name="朱英富 隐形舰船" />

1982年至1984年，赴[[美国|美国]]访问研究，是美国[[加州大学伯克利分校|加州大学伯克利分校]]船舶与海洋工程系的访问学者，从事船舶流体动力学方面的研究。<ref name="朱英富 中国工程院院士">{{cite web |url = http://www.csmoe.sjtu.edu.cn/display.jsp?name=xin_wen_1_86 |title = 朱英富校友当选中国工程院院士[图] |publisher = 上海交通大学 |author =  |date = 2011年12月8日 |accessdate = 2012年12月2日 |language = zh-hans}}</ref>

== 主持设计型号 ==
朱英富主持设计、建造了中国现代最先进的几种舰艇，包括：
* 1996年，担任新一代两型驱逐舰[[052C型驱逐舰|052C型驱逐舰]]总设计师。代表舰艇有：[[“武汉”号导弹驱逐舰|“武汉”号导弹驱逐舰]]和[[“海口”号导弹驱逐舰|“海口”号导弹驱逐舰]]。这款舰艇先后参加过中俄联合军演，中国人民解放军海军访问欧洲四国，首批赴[[索马里|索马里]]海域执行国际护航任务等。使中国成为世界上继[[美国|美国]]，[[苏联|苏联]]（今[[俄罗斯|俄罗斯]]）之后第三个能自主研制国产化的编队区域防空型驱逐舰的国家。
* 1990年，担任F25T型护卫舰工程总设计师，该型号出口[[泰国|泰国]]。使中国一跃成为世界新型护卫舰主要出口国之一。
* 中国海军“隐形舰船”总设计师<ref name="朱英富 隐形舰船" />。
* 中国首艘[[航空母舰|航空母舰]][[辽宁号航空母舰|辽宁号航空母舰]]总设计师<ref name="朱英富 上海交通大学" />。
