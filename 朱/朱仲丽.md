'''朱仲丽'''{{BD|1915年||2014年|2月8日|catIdx= Z朱}}，中共领导人[[王稼祥|王稼祥]]的夫人，早年追求丈夫王稼祥闹[[革命|革命]]，曾担任毛泽东保健医生，中后期遭到[[文化大革命|文化大革命]]的[[政治迫害|政治迫害]]，晚年潜心写作，成为作家。<ref name="bull" >{{cite web |url=http://news.ifeng.com/history/zhiqing/ziliao/detail_2013_03/07/22834016_0.shtml |title=文革王稼祥获罪殃及儿孙，其子被折磨投湖自尽 |accessdate=2013-03-07 |publisher=凤凰网 |work= |language=zh-hans}}</ref><ref name="fishwu" >{{cite web |url=http://www.crt.com.cn/news2007/News/tgjx/2009/1125/0911251855893JF0EH0FAC46KJ6K5C2_2.html |title=吴迪，王稼祥的故事 |accessdate=2009-11-25 |publisher=中红网 |work= |language=zh-hans}}</ref><ref name="bull" />

== 生平 ==
1915年，朱仲丽出生在一个书香家庭，其祖宗是[[明朝|明朝]]开国皇帝[[朱元璋|朱元璋]]，父亲[[朱剑凡|朱剑凡]]。<ref name="fishwu" /><ref name="bull" />

1932年11月13日，朱仲丽和姐姐[[朱仲芷|朱仲芷]]（[[蕭劲光|蕭劲光]]妻子）、[[李英|李英]]（[[邓中夏|邓中夏]]妻子）被[[上海|上海]][[中国国民党|中国国民党]]的[[特务|特务]]逮捕，由于没有暴露身份才得以释放。<ref name="fishwu" /><ref name="bull" />

1936年，朱仲丽考入上海东南医学院，1937年，朱仲丽来到[[延安|延安]]，在[[陕甘宁边区|陕甘宁边区]][[医院|医院]]当外科医生。<ref name="fishwu" /><ref name="bull" />

1974年，王稼祥在[[北京|北京]]去世，朱仲丽放弃医生职业，开始写作，写了《春霞润我》、《艳阳高照》、《王稼祥夫人：朱仲丽自传三部曲》等作品。<ref name="fishwu" /><ref name="bull" />

2014年2月8日因病醫治無效，於北京逝世，享年99歲。<ref>{{cite news|title=朱仲丽同志逝世|url=http://cpc.people.com.cn/n/2014/0213/c87393-24342880.html|work=人民网|date=2014-02-13}}</ref>

== 家庭 ==
在延安一次会议之后，[[毛泽东|毛泽东]]有意撮合王稼祥和朱仲丽，在对朱仲丽的了解深入之后，王稼祥对她产生了好感，而且王稼祥是朱仲丽姐夫[[蕭劲光|蕭劲光]]的救命恩人。蕭劲光在朱仲丽面前经常说起王稼祥的家世、学识、能力等，也使得朱仲丽对王稼祥有所了解。<ref name="fishwu" /><ref name="bull" />

1939年3月5日，王稼祥和朱仲丽在延安结婚。<ref name="fishwu" /><ref name="bull" />
