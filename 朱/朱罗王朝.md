'''朱羅王朝'''（泰米尔语：{{lang|ta|சோழர் குலம்}}, {{IPA2|'ʧoːɻə}}）），又名'''注輦'''，是1世紀至13世紀时[[印度半岛|印度半岛]]古国，其地在今[[泰米爾納德邦|泰米爾納德邦]]。朱羅王朝最早起源于[[高韦里河|高韦里河]]（Kaveri）流域，以欧赖宇尔（Urayur）为国都。[[朱羅王朝|朱羅王朝]]的統治者們曾經征服印度半島南部，吞併[[斯里蘭卡|斯里蘭卡]]，并佔領了[[馬爾代夫|馬爾代夫]]，甚至還成功地入侵馬來群島的王國。在公元1010年至1200年期間，領土南到馬爾代夫群島，最北到[[安得拉邦|安得拉邦]]的達戈達瓦里河沿岸。[[朱羅王朝|朱羅王朝]]在13世紀進入衰退，隨著[[潘地亞王朝|潘地亞王朝]]的崛起而最終滅亡。


== 早期历史 ==
早期朱羅的历史来自[[泰米爾|泰米爾]]參淦（Sangam）文獻，兩位國王喀里佧剌（Karikala）和克僧干楠（Kocengannan）較有名。

7世紀初，唐代高僧玄奘《[[大唐西域記|大唐西域記]]》称朱羅为[[珠利耶|珠利耶]]國：“周二千四五百里。國大都城周十餘里。土野空曠，藪澤荒蕪…城東南不遠，有[[窣堵波|窣堵波]]。[[無憂王|無憂王]]之所建也。[[如來|如來]]在昔曾於此處現大神通，說深妙法……城西不遠，有故伽藍。提婆菩薩與羅漢論議之處。”

== 中世纪 ==
9世纪中叶[[韦迦亚剌雅|韦迦亚剌雅]]王（Vijayalaya 在位848年－871年）奋起，夺取[[坦贾武尔|坦贾武尔]]（Thanjavur），设为国都，建立朱羅[[韦迦亚剌雅王朝|韦迦亚剌雅王朝]]。
985年[[罗茶罗乍|罗茶罗乍]]（{{lang|en|'''Raja Raja''' Chola I}}）登位，雄才大略，扩展朱羅疆域，南至錫蘭，北達[[羯陵迦|羯陵迦]]（Kalinga），統一南印度。罗茶罗乍还征服[[溜山|溜山]]國。

[[宋真宗|宋真宗]][[大中祥符|大中祥符]]八年(1015年)，注輦王[[罗茶罗乍|罗茶罗乍]]，遣使朝貢。
宋元时期，注輦国和中国有交通。[[宋代|宋代]]《[[嶺外代答|嶺外代答]]》、《[[諸蕃誌|諸蕃誌]]》各有专条。从中国前往注輦，可从[[泉州|泉州]]乘海舶，取道[[故臨|故臨]]，再轉搭小船可達注輦国；也可取道[[蒲甘|蒲甘]]前往。

注輦都城有七重，城高七尺，第一、二城為民居，有環城水濠隔離。第三、四城為官府，第五、六城為王府，第七重為王宮。注輦国好战，畜養戰象六萬，每頭戰象高七八尺，象背馱小屋，戰士用弓箭遠戰，和近距離用長矛格殺，战士英勇善戰，視死如歸，戰勝者得到獎旗表功。
在公元1048年左右，朱罗王朝直接统治了1480万个居民。

== 晚期歷史（公元1070-1279年） ==
公元1070年，[[毗罗·拉金德拉·朱罗|毗罗·拉金德拉·朱罗]]（[[Virarajendra_Chola|Virarajendra Chola]]）的兒子[[阿迭·拉金德拉·朱罗|阿迭·拉金德拉·朱罗]]（[[Athirajendra_Chola|Athirajendra Chola]]）在內亂中被暗殺，[[库罗通伽·朱罗|库罗通伽·朱罗]]（[[Kulothunga_Chola|Kulothunga Chola]]）的兒子[[罗闍罗闍·纳兰德拉|罗闍罗闍·纳兰德拉]]（[[Rajaraja_Narendra|Rajaraja Narendra]]）登上了王位，朱罗与[[东遮娄其王朝|东遮娄其王朝]]（[[Chalukya|Chalukya]]）合并，又称朱罗-遮娄其王国。朱羅王朝後期開始。

朱羅王朝的統治和領土完整，直到库罗通伽·朱罗（[[Kulothunga_Chola|Kulothunga Chola]]三世）之前都非常穩定。但到了1215年，在他的統治下朱羅王朝被1190年复国的[[潘地亚|潘地亚]]的[[摩罗伐摩·孙达罗|摩罗伐摩·孙达罗]]（[[Maravarman_Sundara_Pandiyan_II|Maravarman Sundara Pandiyan II]]）所擊敗。隨後，朱羅王朝也失去了[[斯里蘭卡|斯里蘭卡]]的控制，被復興的[[僧伽羅人|僧伽羅人]]趕走。

此後，南方的[[潘地亚|潘地亚]]實力逐漸增強，并最終導致朱羅王朝于公元1279年滅亡。当时的潘地亚国王是[[查太伐摩·孙达罗|查太伐摩·孙达罗]]（[[Jatavarman_Sundara|Jatavarman Sundara]]）。

朱罗王朝灭亡后，中国依然称呼印度南部为注辇，[[明朝|明朝]]史籍称为'''琐里'''<ref>《[[明史|明史]]》卷325：琐里，“近西洋琐里而差小”<br>《[[殊域周咨录|殊域周咨录]]》卷8：“琐里又曰西洋琐里……其国与伽蓝洲师子国相邻，或云南距柯枝，西濒海，自柯枝海行可三日至”。</ref>、'''西洋琐里'''<ref>《[[明会要|明会要]]》卷79<br>《[[东西洋考|东西洋考]]》卷2<br>《[[咸宾录|咸宾录]]》卷4<br>《[[续通典|续通典]]》卷149<br>《[[清一统志|清一统志]]》卷424<br>《皇舆考》卷12<br>《外国词》<br>《裔乘》卷7<br>《明四夷考》卷下<br>《象胥录》卷5</ref>。
