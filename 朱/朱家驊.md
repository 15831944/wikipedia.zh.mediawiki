'''朱家驊'''{{BD|1893年|5月30日|1963年|1月3日|catIdx=Zhu朱}}，字'''騮先'''，[[浙江省|浙江省]][[湖州府|湖州府]][[吳興縣_(中華民國)|吳興縣]]（今[[湖州市|湖州市]][[吴兴区|吴兴区]]）人，[[中华民国|中华民国]]地质学家、政治人物，中国近代地质学的奠基人。[[中国国民党|中国国民党]]内亲[[德国|德国]]派人士。<ref name=xu>{{Cite book|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref name=kmt>{{Cite book|author = 劉国銘主編|title = 中国国民党百年人物全書|year = 2005|publisher = 团結出版社|isbn = 7-80214-039-0}}</ref><ref name=zx>{{Cite book|author = 東亜問題調査会|title = 最新支那要人伝|year = 1941|publisher = [[朝日新聞社|朝日新聞社]]|isbn = }}</ref>屬[[中央俱樂部|CC派]]。

朱家驊曾任[[中華民國|中華民國]][[中央研究院|中央研究院]]總幹事、代理院長，还曾任[[中华民国|中华民国]][[行政院|行政院]]及[[考试院|考试院]]副院长、[[中華民國教育部|教育部]]部長、交通部部長、[[浙江省|浙江省]]政府主席等职务。他是1920年代至1940年代[[中德合作_(1911年-1941年)|中德合作]]中的重要人物。他还曾任[[中国国民党中央執行委員會調查統計局|中国国民党中央執行委員會調查統計局]]局长，被视为[[陳果夫|陳果夫]]、[[陳立夫|陳立夫]]兄弟的[[CC系|CC系]]的一員。<ref name=xu/><ref name=kmt/><ref name=zx/>

==詳細生平==
===早年求學生涯===
[[File:Zhu_Jiahua3.jpg|thumb]]》中的朱家驊照片|left|150px]]

1893年5月30日（[[清|清]][[光緒|光緒]]十九年四月十五），朱家驊生于大清浙江省[[湖州府|湖州府]]吳興縣。早年先后進入[[南潯正蒙学堂|南潯正蒙学堂]]、[[南潯公学|南潯公学]]等學堂就讀。1908年（[[光緒|光緒]]34年）秋天，入[[上海|上海]][[同济大学|同济德文医学校]]学习。在校期间，因知道[[辛亥革命|辛亥革命]]爆发後，朱家驊在上海参加革命派的中国敢死团。1912年（[[民国紀元|民国]]元年），回到已改组为[[同济大学|同济医工专門学校]]的母校復学，入電機系。<ref name=xu/><ref name=kmt/><ref name=zx/>

1914年（民国3年）大學毕业後，朱家驊獲得[[張静江|張静江]]的資助，前往[[德国|德国]]留学，入[[柏林工业大学|柏林工业大学]]学习冶金。后来因受[[第一次世界大战|第一次世界大战]]影響，朱家驊无法继续学业，乃于1917年（民国6年）歸國。歸國後，到[[北京大学|北京大学]]担任德語教员，不久再到[[瑞士|瑞士]]留学，入[[伯尔尼大学|伯尔尼大学]]。1920年，回到德国，入[[柏林洪堡大学|柏林大学]]学习[[地質学|地質学]]，1922年获哲学博士。<ref name=xu/><ref name=kmt/><ref name=zx/>
===教職生涯===
1924年（民国13年）春，朱家驊归国，任北京大学地質系教授兼德語教员官。1926年（民国15年）3月，北京学生在游行中遭到[[中华民国临时政府_(段祺瑞)|临时政府]]武力镇压，酿成[[三一八惨案|三一八惨案]]，朱家驊被指为学生游行的責任者，遭到北京临时政府的通缉。朱家驊遂逃回家乡，同年7月出任[[国立广東大学|国立广東大学]]地質系教授、系主任。10月，广東大学改组为[[国立中山大学|国立中山大学]]，朱家驊出任中山大学委員会代理委員長。<ref name=xu/><ref name=kmt/><ref name=zx/>
<br><br><br>
=== 投入政治===
1926年11月，朱家驊成为[[广州国民政府|广州国民政府]]的[[广東省_(中華民国)|广東省]]政府委員，从此开始政治家生涯。1927年5月，调任[[浙江省_(中華民国)|浙江省]]政務委員会委員兼浙江省農工厅厅長。6月，任中山大学副校長，两广地質調査所创办后，兼任所長。此外还任中国国民党广東省党部執行委員等要職。9月，改任中山大学教務長。10月，任浙江省政府委員兼民政厅厅長、中国国民党浙江省党部執行委員。1929年3月，当选[[中国国民党|中国国民党]]第三届中央執行委員（此後直到第六届一直连任中央執行委員）。<ref name=xu/><ref name=kmt/><ref name=zx/><ref name=liu>{{Cite book|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = 中華書局|isbn = 7-101-01320-1}}</ref>
=== 重返教職===
1930年9月，任中山大学校長。11月，改任[[国立中央大学|国立中央大学]]校長。1931年4月，任中英[[庚款|庚款]]委員会（1943年（民国32年）改组为中英文教基金董事会）委員長。<ref name=xu/><ref name=kmt/><ref name=zx/><ref name=liu/>
=== 重返政治===
1931年9月，任全国经济委員会秘書長。12月，任国民政府[[中華民国教育部|教育部]]部長。1932年10月，改任[[中華民国交通部|交通部]]部長。1935年（民国24年）12月，退任交通部部長，出任中央政治委員会代理秘書長。1936年春天，应国立[[中央研究院|中央研究院]]院長[[蔡元培|蔡元培]]招聘，出任中研院总幹事。同年12月至1937年（民国26年）11月，任浙江省政府主席。<ref name=xu/><ref name=kmt/><ref name=zx/><ref name=liu/>
=== 抗战時期===
[[抗日战争|抗日战争]]爆发后的1938年3月，朱家驊任[[国民政府軍事委員会|国民政府軍事委員会]]参事室主任。4月，任[[中国国民党中央執行委員会|中国国民党中央執行委員会]]秘書長兼党務委員会主任委員兼[[中央調査統計局|中央調査統計局]]局長，在党務方面担当枢要。因担任中央調査統計局局長，被视为陳果夫、陳立夫兄弟的[[CC系|CC系]]的一員。7月，任[[三民主義青年团|三民主義青年团]]幹事会常務幹事兼中央团部代理書記長。[[第二次世界大战|第二次世界大战]][[德国|德国]]发动闪击战后，朱家驊继续主张[[中德合作_(1911年-1941年)|中德合作]]，但後来[[三国同盟条约|德意日三国軍事同盟]]成立，中德合作未能继续。<ref name=xu/><ref name=kmt/><ref name=zx/>

1939年12月[[葉楚傖|葉楚傖]]接任朱家驊的中央執行委員會秘書長职务。

1940年（民国29年）3月，中研院院長[[蔡元培|蔡元培]]逝世，朱家驊代行中研院院長职务。同年9月，朱家驊再任中山大学校長，12月改任中大校長。12月，出任[[考試院|考試院]]副院長。1944年（民国33年）11月，接替[[陳立夫|陳立夫]]被任命为教育部長。<ref name=xu/><ref name=kmt/><ref name=liu/>
=== 二战後活動 ===
[[File:Zhu_Jiahua1.jpg|thumb]]
[[日本投降|日本投降]]後的1945年9月，朱家驊作为[[中華民國|中国]]首席代表出席了[[联合国|联合国]]教育文化会議。翌年11月，[[聯合國教育、科學及文化組織|聯合國教育、科學及文化組織]]正式成立，朱家驊临时担任中国代表团团長。

1947年4月12日，教育部長朱家驊主持[[聯合國教育、科學及文化組織|聯合國教育、科學及文化組織]]中國委員會首次籌備會議，定於6月在南京成立。<ref name="民國史大事記">{{cite book|editor=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編|title=《中華民國史大事記》|location=[[北京|北京]]|publisher=[[中華書局|中華書局]]|date=2011年7月}}</ref>{{rp|8333}}

1947年4月13日，聯合國文教組織決議調查中國戰爭損失。<ref name="民國史大事記"/>{{rp|8333}}

1947年4月17日，國民政府任命朱家驊兼任[[中華民國科技部|国防科学委員会]]副主任委員兼委員。<ref name="民國史大事記"/>{{rp|8336}}

1947年4月23日，國民政府准免教育部部長張厲生本職；特任朱家驊為行政院政務委員，兼教育部部長。<ref name="民國史大事記"/>{{rp|8340}}

1947年5月25日，教育部長朱家驊在[[國民參政會|國民參政會]]報告教育工作，參政員嚴厲質詢學潮；朱家驊謂學潮係因共黨分子在學校活動所致。<ref name="民國史大事記"/>{{rp|8361}}

1947年8月15日，北京大學教授生活困難，90餘位教授，當面向教育部長朱家驊要求合理改善待遇。<ref name="民國史大事記"/>{{rp|8397}}

1948年，榮獲中華民國[[中央研究院|中央研究院]]第1屆(數理科學組)院士<ref>[https://academicians.sinica.edu.tw/index.php?func=1-D 李四光 逝世院士一覽表 中央研究院]</ref>。
=== 臨危受命 ===
1948年5月31日，朱家驊被任命為行政院政務委員兼教育部部長。1949年6月12日，正值[[第二次国共内战|第二次国共内战]]末期，朱家驊卸下教育部部長之職，並撤離至[[广州|广州]]，任[[中華民國政府|中華民國政府]][[行政院|行政院]]副院長。其後，中國大陸關入[[鐵幕|鐵幕]]，朱家驊跟隨[[中華民國政府|中華民國政府]]撤離至[[台湾|台湾]]。1950年3月，朱家驊被任命為[[总統府資政|总統府資政]]。1950年7月，創辦《大陸雜誌》。同時也是中国国民党中央評議委員。<ref name=xu/><ref name=kmt/><ref name=liu/>

1963年1月3日，朱家驊病逝。享年71岁（满69歳）。<ref name=xu/><ref name=kmt/>
