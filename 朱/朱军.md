'''朱军'''（{{lang-en|Deon Zhu}}，{{bd|1964年|4月26日|}}）[[甘肃|甘肃]][[兰州|兰州]]人，祖籍[[河南|河南]][[孟津县|孟津县]][[平乐镇_(孟津县)|平乐乡]]朱仓村<ref>{{cite web|url=http://news.163.com/13/0328/09/8R1T1I1500014JB6_all.html | language= | title=朱军家世揭秘：父亲是“兰战”的创团元老 |publisher=人民网 |date=2013-03-28 |accessdate= }}</ref>，中国电视节目[[主持人|主持人]]、[[影视|影视]][[演员|演员]]以及[[喜剧小品|小品]]演员。著有《[[时刻准备着|时刻准备着]]》，曾主持[[中国中央电视台|中国中央电视台]]谈话节目《[[艺术人生|艺术人生]]》，他也是多届[[央视春晚|央视春晚]]主持人之一。

== 简介 ==
朱军1964年出生于[[兰州|兰州]]，1981年10月至1985年9月于[[兰州军区|兰州军区]]84703部队侦察连参军，1985年9月至1988年9月成为[[甘肃省|甘肃省]]歌舞团演员，1988年9月后加入兰州军区战斗歌舞团。1996年3月加入中央电视台，参与工作至今。此外，他自1997年开始主持[[中央电视台春节联欢晚会|中央电视台春节联欢晚会]]（直至2017年，连续主持21届），多次同[[冯巩|冯巩]]合作表演小品，同时，他曾是一名相声演员。1999年在全国金话筒节目主持人评选中获银奖第一名。曾主持谈话类节目《[[艺术人生|艺术人生]]》、选秀类节目《[[星光大道|星光大道]]》等。

== 家庭生活 ==
朱军的父亲于1922年生于河南洛阳市孟津县朱仓村，幼时成为[[孤儿|孤儿]]，一个远房的王姓舅爷收留其父，其父就生活在距离朱仓不远的送庄，并在那里与朱军母亲相识，由于舅爷家子女多且生活贫苦，朱军父亲又被送到了洛阳的[[孤儿院|孤儿院]]长大。1944年其父参军入伍，后来成为[[兰州军区|兰州军区]]政治部战斗歌舞团[[单簧管|单簧管]]演奏员，于是就把家安在了兰州。<ref>{{cite web|url=http://book.people.com.cn/n/2012/1128/c69360-19720587.html | language= | title=朱军家世揭秘：父亲是“兰战”的创团元老 |publisher=人民网-读书频道 |date=2012年11月28日 |accessdate= }}</ref>

朱军的妻子[[谭梅|谭梅]]是一个[[舞蹈|舞蹈]]演员。<ref name="lion " >{{cite web |url=http://roll.sohu.com/20120308/n337080196.shtml |title=央视主持人妻子大揭秘 |publisher=搜狐网 |accessdate=2012-03-08 |work= |language =zh-cn}}</ref><ref name="shark " >{{cite web |url=http://ent.163.com/special/00032H0C/cctv1229.html |title=央视主持人的另一半 |publisher=网易 |accessdate=2010年 |work= |language =zh-cn}}</ref><ref name="duck " >{{cite web |url=http://photo.gmw.cn/2012-03/09/content_3733396.htm |title=央视主持人妻子大揭秘 |publisher=光明网 |accessdate= 2012-03-09|work= |language =zh-cn}}</ref>两人于1994年结婚，育有一个儿子朱思谭。{{cn}}

朱军兄弟姐妹总共7人<ref>{{cite web |url=http://www.gscn.com.cn/military/system/2012/03/27/010026737.shtml |language= |title=朱军 父亲的慈爱与严厉是永远的记忆 |publisher=中国妇女报 |date=2012-03-27 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20161115132404/http://www.gscn.com.cn/military/system/2012/03/27/010026737.shtml |archivedate=2016-11-15 }}</ref>，他有一位哥哥叫[[朱志良|朱志良]]，2000年任甘肃省劳动和社会保障厅厅长、2008年任甘肃省人大常委会副主任。{{cn}}

== 風格 ==
朱軍進入2000年代後，他開始有機會在節目中採訪大量[[香港|香港]]，[[台灣|台灣]]著名藝人，但其煽情風格在訪問人中充滿爭議，甚至於反被被訪者調侃，比較著名的例子有2004年訪問香港著名歌手[[張學友|張學友]]<ref>{{cite news|url=http://ent.sina.com.cn/v/2004-11-02/1052552591.html|title=朱军与港台艺人难交流 张学友不懂“德艺双馨”|publisher=[[重庆晚报|重庆晚报]]|date=2004-11-02|accessdate=2017-10-22}}</ref>以及2009年訪問著名演員[[周潤發|周潤發]]<ref>{{cite news|author=李丽|url=http://www.ce.cn/xwzx/gnsz/gdxw/200912/04/t20091204_20559370.shtml|title=朱军在节目中给观众下跪 主持人朱军"被主持"|publisher=[[羊城晚报|羊城晚报]]|date=2009-12-04|accessdate=2017-10-22|archive-url=https://web.archive.org/web/20171022141352/http://www.ce.cn/xwzx/gnsz/gdxw/200912/04/t20091204_20559370.shtml|archive-date=2017-10-22|dead-url=yes}}</ref>。而{{查证请求|据传}}[[崔永元|崔永元]]在接受某周刊采访时更暗指朱军对其采访嘉宾不尊重，曾在节目后说出“这傻逼还挺配合的”，朱军在此后接受采访时否认此事<ref>{{cite web|language =zh-hans|url=http://yule.sohu.com/20101110/n277505903.shtml|title=崔永元炮轰央视名嘴 朱军：我不是那样的人|publisher=搜狐娱乐|date=2010年11月10日|accessdate=2010-12-23 }}</ref>。
