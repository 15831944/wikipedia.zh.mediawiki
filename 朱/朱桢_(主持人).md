'''朱楨'''（{{bd|1978年|11月4日|catIdx=Zhu,朱}}）是[[上海文廣新聞傳媒集團|SMG]]的當紅主持人，出生在[[上海|上海]]，籍贯[[江蘇|江蘇]][[蘇州|蘇州]]，畢業於[[上海戲劇學院|上海戲劇學院]]電視藝術系。因其同時主持八檔電視節目，最多時橫跨五大電視頻道，故而自稱“朱八檔”。

== 主持生涯 ==
1998年起以上戲在校生的身份在上海音樂頻道實習，與[[學虹|學虹]]共同主持《[[繽紛歌壇|繽紛歌壇]]》，2001年畢業後加盟，曾主持過《繽紛歌壇》、《[[勁爆榜|勁爆榜]]》、《[[第一琴房|第一琴房]]》、《[[繽紛音樂劇|繽紛音樂劇]]》、《[[周末大點播|周末大點播]]》等欄目。現任《[[東方新人|東方新人]]》、《[[音樂前線|音樂前線]]》、《[[天地英雄校園行|天地英雄校園行]]》、《閃電星感動》節目主持人。在東視文藝頻道播出的滬語情景劇《老娘舅|[[老娘舅和他的兒孫們|老娘舅和他的兒孫們]]》、《[[開心公寓|開心公寓]]》中分別扮演董斌斌和朱利安。

== 著作 ==
朱楨在[[2006年8月|2006年8月]]出版了自傳《[[玩美囑憶|玩美囑憶]]》。

== 評價 ==
朱楨以其活潑幽默的主持風格風靡滬上熒屏，主持的《東方新人》、《天地英雄校園行》等節目廣受年青觀眾的歡迎。在一些節目中，朱楨常常妙語連珠，逗得觀眾笑聲不斷。但也有人稱他的主持風格過於活潑，在一些節目中大開主持搭檔和嘉賓玩笑。
