'''朱政騏'''（{{bd|1978年|04月12日|catIdx=T段}}），又被称为「牛糞博士」，生於[[台灣|台灣]][[台北|台北]]，中華民國[[政治人物|政治人物]]，社運人士，[[台灣立報|台灣立報]]主筆。曾任[[民主進步黨|民主進步黨]]台北市黨部主委。目前積極活動於反禁菸組織，重視吸煙者人權，積極與反煙團體溝通，期盼取得平衡。<ref>{{cite news |language = 中文 |author = 陳鈞凱 |url = http://tobacco.hpa.gov.tw/Content.aspx?MenuId=556&CID=23189&year=0&month=0 |title = 以價制量？菸品健康捐擬漲20 擁菸團體批衛署禁菸法西斯 |publisher = NOWnews |date = 2012-08-17 |accessdate = 2017-04-29}}</ref><ref>{{cite news |language = 中文 |author = 新國際編輯部 |url = http://www.lihpao.com/?action-viewnews-itemid-113689 |title = 菸害防制法是與非 |publisher = 台灣立報 |date = 2011-12-15 |accessdate = 2017-04-29}}</ref>

==學歷==
朱政騏在成功高中畢業後，考取[[台北醫學大學|台北醫學大學]]保健營養系。在大學時期擔任系學會會長，相當關心公共議題，並積極投入社會運動。1999年，仍就讀大三的他，為了呼籲學生參與公共事務，開始在校園內自費發行並自辦"正崎報"。<ref>{{Cite web |url = http://mypaper.pchome.com.tw/missingray/post/1234745024 |title=一人報紙校園發聲 朱政騏樂此不疲|publisher=聯合報系|author=梁仲昇|location=台北|accessdate=2014-04-20|language=zh-tw}}</ref>。其後畢業於[[世新大學|世新大學]]社會發展研究所碩士班，[[台灣大學|台灣大學]]社會所博士班肄業。
