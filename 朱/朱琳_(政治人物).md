'''朱琳'''（{{bd|1930年|11月7日||}}）[[上海|上海]]人，[[中华人民共和国|中华人民共和国]]官员、高级经济师，[[李鹏|李鹏]]夫人。

==生平==
生于上海市一个女工的家庭。初中在[[重庆市兼善中学|重庆市兼善中学]]学习<ref>{{Cite web |url = http://news.ifeng.com/gundong/detail_2012_12/17/20228475_0.shtml |title =拓展优质教育资源 做大做强兼善教育  |author =  |date = 2012-12-17 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>，高中在[[重庆市巴蜀中学|重庆市巴蜀中学]]学习<ref>{{Cite web |url = http://news.163.com/17/0428/08/CJ3JGOJB00018AOP.html |title = 重庆市中学生标准化入团仪式在巴蜀中学举行 “最初的梦想”主题团日活动忒走心 |author =  |date = 2017-04-28 |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。1950年代初，在哈尔滨外语学院（现[[黑龙江大学|黑龙江大学]]）学习，专业为[[俄语|俄语]]。毕业后被分配到[[吉林市|吉林市]]化工厂担任俄文翻译。后来曾任华北电力局外事处负责人，[[广东|广东]][[大亚湾核电站|大亚湾核电站]]驻北京办事处主任，国务院特区办公室研究室主任。曾担任[[中国关心下一代工作委员会|中国关心下一代工作委员会]]副主任，[[中国妇女发展基金会|中国妇女发展基金会]]副会长，中国关心下一代工作委员会专家委员会名誉主任等社会兼职<ref name=hexun>{{Cite web |url = http://renwu.hexun.com/figure_8319.shtml |title = 朱琳 |author =  |date =  |publisher = 和讯 |language =  |accessdate = 2017-11-03 |quote =  }}</ref>。
