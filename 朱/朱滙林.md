'''朱滙林'''（'''William Chu'''，{{bd|1987年|9月1日}}），[[香港|香港]]男演員，現為香港[[無綫電視|無綫電視]]基本藝人合約藝員<ref>{{cite web|url=http://artiste.tvb.com/main/artistedetail/williamchu|title=TVB男藝人|}}</ref><ref>{{cite web | language = | publisher = 嗇色園官方網站| title = 電視廣播有限公司（TVB）藝員紅星到訪| url = http://www.siksikyuen.org.hk/public/contents/news?ha=&wc=0&hb=&hc=&revision_id=42801&item_id=42773 }}</ref><ref>{{cite web | language = | publisher = Qoos.com | title = 全澳小學生講故事大賽2014決賽暨頒奬禮圓滿結束
| url = http://news.qoos.com/%E2%80%9C%E5%85%A8%E6%BE%B3%E5%B0%8F%E5%AD%B8%E7%94%9F%E8%AC%9B%E6%95%85%E4%BA%8B%E5%A4%A7%E8%B3%BD2014%E6%B1%BA%E8%B3%BD%E6%9A%A8%E9%A0%92%E5%A5%AC%E7%A6%AE%E2%80%9D%E5%9C%93%E6%BB%BF%E7%B5%90%E6%9D%9F-1801143.html }}</ref>。

==背景==
朱滙林在香港高級程度會考考取2A2B佳績，畢業於[[聖若瑟書院|聖若瑟書院]]及[[香港大學|香港大學]]法律學院，曾擔任香港大學聖約翰學院男子排球隊隊長。2007年於無線電視舉辦的[[TVbeople|TVbeople]]選拔中脫穎而出<ref>{{cite web | language = | publisher = 蘋果日報| title = 八新鮮人出爐| url = http://hk.apple.nextmedia.com/entertainment/art/20070317/6909659 }}</ref><ref>{{cite web | language = | publisher = YouTube| title = TVB Tvbeople首晤傳媒界 眾師兄師姐挺身支持| url = https://www.youtube.com/watch?v=z3rpsyd_5Js }}</ref>，成為旗下藝員。出道初以主持為主，2012年他在處境劇《[[愛·回家_(第一輯)|愛·回家]]》飾演李志成（醒哥）<ref>{{cite web | language = | publisher = 明報周刊第2365期Scan圖檔| title = 【得寵】《朱滙林 一屋兩狗 左右逢源》| url = https://www.facebook.com/media/set/?set=a.802233173139469.1073742030.424816137547843&type=1}}</ref><ref>{{cite web | language = | publisher = 明報周刊第2365期官方付費網站| title = 【得寵】《朱滙林 一屋兩狗 左右逢源》| url = http://www.mpweekly.com/archive/archive.php?searchID=3&search=&Year=2014&page=5}}</ref>而人氣急升<ref>{{cite web | language = | publisher = TVB| title = 2014年7月8日無綫電視節目《今日VIP》專訪| url = http://programme.tvb.com/variety/greenroom/episode/20140708/}}</ref><ref>{{cite web | language = | publisher = HKChannel| title = 愛．回家 收視贏重頭劇 演員名利全收大滿貫| url = http://www.ihktv.com/mingpaoweekly-2321-comehomelove.html }}</ref><ref>{{cite web | language = | publisher = 香港數碼廣播有限公司| title = 【新春娛聞樂】朱滙林 送祝福！| url = http://www.dbc.hk/radio1/focus-detail/Id/42971/type/2/%E3%80%90%E6%96%B0%E6%98%A5%E5%A8%9B%E8%81%9E%E6%A8%82%E3%80%91%E6%9C%B1%E6%BB%99%E6%9E%97%20%E9%80%81%E7%A5%9D%E7%A6%8F%EF%BC%81 }}</ref>。除演藝工作外，亦有投資食肆生意<ref>{{cite web | language = | publisher = 東方日報| title = 朱滙林擲百萬 唔做律師開炒雞店| url = http://orientaldaily.on.cc/cnt/entertainment/20160531/00282_024.html}}</ref><ref>{{cite web| language =| publisher =東方日報| title =10項死慳死抵規條| url =http://orientaldaily.on.cc/cnt/entertainment/20160531/00282_025.html.html}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2015年9月, 包括朱滙林等《[[愛·回家_(第一輯)|愛·回家]]》的班底聯合成立了慈善義工團體[[全愛義工隊|全愛義工隊]], 成員們決定善用《[[愛·回家_(第一輯)|愛·回家]]》的名氣將愛心化為行動, 把愛送給有需要的人, 為社會盡一份心力<ref>{{cite web| language =| publisher =晴報| title =全愛義工隊成立了| url =http://www.skypost.hk/column/%E9%BB%8E%E8%8A%B7%E7%8F%8A/007002001013/%E5%85%A8%E6%84%9B%E7%BE%A9%E5%B7%A5%E9%9A%8A%E6%88%90%E7%AB%8B%E4%BA%86/189715| deadurl =yes| archiveurl =https://web.archive.org/web/20160305010959/http://www.skypost.hk/column/%E9%BB%8E%E8%8A%B7%E7%8F%8A/007002001013/%E5%85%A8%E6%84%9B%E7%BE%A9%E5%B7%A5%E9%9A%8A%E6%88%90%E7%AB%8B%E4%BA%86/189715| archivedate =2016-03-05}}</ref>。
