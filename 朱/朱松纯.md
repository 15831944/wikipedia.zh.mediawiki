'''朱松纯'''（{{lang-en|Song-Chun Zhu}}，{{bd|1968年}}），[[湖北|湖北]][[鄂州|鄂州]]人，美国[[加州大学洛杉矶分校|加州大学洛杉矶分校]]统计学系与计算机系教授，主要从事[[计算机视觉|计算机视觉]]、[[模式识别|模式识别]]等领域的研究。

== 生平 ==
朱松纯于1991年毕业于[[中国科学技术大学|中国科学技术大学]]计算机系，获学士学位。后赴美国留学，1996年获[[哈佛大学|哈佛大学]]计算机博士学位，师从著名数学家[[戴维·芒福德|戴维·芒福德]]。此后曾任教于[[斯坦福大学|斯坦福大学]]、[[俄亥俄州立大学|俄亥俄州立大学]]。2002年起任[[加州大学洛杉矶分校|加州大学洛杉矶分校]]副教授，2006年升任正教授。2007年受聘为中国教育部[[长江学者讲座教授|长江学者讲座教授]]。<ref>{{cite web|url=http://www.edu.cn/tepin_4798/20070919/t20070919_255654.shtml|title=“长江学者”教授风采：朱松纯|date=2007-09-19|website=中国教育和科研计算机网|access-date=2017-11-12|archive-url=https://web.archive.org/web/20180104192822/http://www.edu.cn/tepin_4798/20070919/t20070919_255654.shtml|archive-date=2018-01-04|dead-url=yes}}</ref>2010年入选[[中组部|中组部]][[千人计划|千人计划]]。同年当选为[[电气电子工程师学会|电气电子工程师学会]]（IEEE）会士。<ref name="qrzk">{{cite web|url=http://www.1000thinktank.com/gcczj/5528.jhtml|title=朱松纯|website=千人智库|date=2014-10-27}}</ref>

此外，朱松纯还于2005年在其故乡湖北鄂州创办了民营研究院——莲花山视觉与信息科学研究院。<ref name="qrzk" /><ref>{{cite web|url=https://news.qq.com/a/20051114/001120.htm|title=抢占计算机视觉科研制高点 一民营研究院落户鄂州|date=2005-11-14|website=腾讯新闻}}</ref>

朱松纯因其在计算机视觉与模式识别领域的贡献而获得过许多奖项，包括2003年{{le|国际计算机视觉大会|International Conference on Computer Vision}}{{le|马尔奖|Marr Prize}}、2008年{{le|国际模式识别协会|International Association for Pattern Recognition}}阿加沃尔奖（J.K. Aggarwal Prize）、2013年国际计算机视觉大会亥姆霍兹奖（Helmholtz Prize）等。<ref name="qrzk" /><ref>{{cite web|url=https://www.computer.org/portal/web/tcpami/Helmholtz-Prize|title=ICCV Helmholtz Prize|website=IEEE Computer Society}}</ref>

== 家庭 ==
朱松纯之女[[朱易|朱易]]是一名女子单人[[花样滑冰|花样滑冰]]运动员。
