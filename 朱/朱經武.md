'''朱經武'''{{JP}}<ref>{{cite web|title=根據《太平紳士條例》(第510章)第3(1)(b)條獲委任的人士|url=http://www.info.gov.hk/cml/miscell/index3e.htm|website=info.gov.hk|accessdate=2016-02-17}}</ref>（{{lang-en|'''Chu Ching-wu'''}}，{{bd|1941年|2月12日|||catIdx=Zhu朱經武}}），[[美籍华人|美籍华人]][[物理学家|物理学家]]，曾擔任[[香港科技大學|香港科技大學]][[校長|校長]]。2012年底以第一[[志工|志工]]之精神擔任[[臺灣綜合大學系統|臺灣綜合大學系統]]首任校長。<ref>{{cite web|url=http://news.cnyes.com/Content/20121106/KFNVCNNZNM6N2.shtml|title=朱經武出任台綜大首任總校長 冀躋身世界一流學府之林|publisher=鉅亨網|date=2012-11-06}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 簡歷 ==
朱經武出生於[[中華民國|中華民國]][[湖南|湖南]][[芷江|芷江]]，1948年隨父母遷移至[[臺灣省|臺灣省]][[台中縣|台中縣]][[清水區_(臺灣)|清水鎮]]（現[[台中市|台中市]]清水區）。

畢業於臺灣省立清水中學（現[[臺中市立清水高級中等學校|臺中市立清水高級中等學校]]）初中部及高中部(第七屆)，1962年在臺灣省立成功大學（現[[國立成功大學|國立成功大學]]）取得[[自然科學|理学]][[学士|学士]]学位，1965年取得[[紐約|紐約]][[福坦莫大學|福坦莫大學]]的理学[[硕士|硕士]]学位，1968年在[[加州大学圣地亚哥分校|加州大学圣地亚哥分校]]取得[[哲学博士|哲学博士]]学位（[[物理学|物理学]]）。

畢業後，於[[新澤西|新澤西]][[貝爾實驗室|貝爾實驗室]]從事工業研究工作。

1987年，[[休士頓大學|休士頓大學]][[德州超導中心|德州超導中心]]委任朱教授為首位主任。<ref>{{cite web
|url=http://www.chinanews.com.cn/zhonghuawenzhai/2000-12-01/new/%2821%29%201.html|title=与诺贝尔奖擦身而过--物理学家朱经武剪影|publisher=中國新聞網|date=2000-12-01}}</ref> 同年，朱经武與[[吳茂昆|吳茂昆]]等人首次宣布得到了90[[热力学温标|K]]以上电阻消失的[[超导体|超导体]]。<ref>{{cite web|url=https://www.nytimes.com/1987/08/16/magazine/in-the-trenches-of-science.html|title=IN THE TRENCHES OF SCIENCE|publisher=[[紐約時報|紐約時報]]|date=1987-08-16}}</ref><ref>{{cite web|url=http://sa.ylib.com/MagArticle.aspx?Unit=columns&id=3720|title=超導大師朱經武|publisher=[[科學人|科學人]]|date=2008-10}}</ref><ref>{{cite web|url=http://sa.ylib.com/MagArticle.aspx?Unit=featurearticles&id=3721|title=當自由的心靈遇到高溫超導|publisher=[[科學人|科學人]]|date=2005-09}}</ref><ref>{{cite web|url=https://journals.aps.org/prl/heating-up-of-superconductors|title=Heating up of Superconductors|publisher=[[物理評論快報|物理評論快報]]|date=2017}}</ref><ref>{{cite web|url=http://research.sinica.edu.tw/superconductor-wu-maw-kuen/|title=「超導體，我研究了一輩子！」專訪超導物理專家吳茂昆|agency=[[中央研究院|中央研究院]]|publisher=《研之有物》|date=2002-11-01}}</ref>

2001年至2009年，朱經武擔任[[香港科技大學|香港科技大學]]校長。

2011年，[[中華民國教育部|中華民國教育部]]核定[[國立成功大學|國立成功大學]]、[[國立中興大學|國立中興大學]]、[[國立中山大學|國立中山大學]]及[[國立中正大學|國立中正大學]]成立[[臺灣綜合大學系統|臺灣綜合大學系統]]。2012年11月5日上午舉行校長佈達典禮，宣布朱經武擔任首任系統校長<ref>[http://www.nownews.com/2012/11/05/91-2869943.htm 築巢引鳳！馬英九讚朱經武不支薪任台灣綜合大學系統校長]</ref>。

== 學術榮譽 ==
朱教授是[[俄羅斯工程學院|俄羅斯工程學院]]、[[美國科學院|美國科學院]]、[[美國人文及科學學院|美國人文及科學學院]]、[[中華民國中央研究院|中華民國中央研究院]]、[[中國科學院|中華人民共和國中國科學院]]及[[發展中世界科學院|發展中世界科學院]]六院院士。

此外，朱教授屢獲殊榮，包括美國科學家最高榮譽的[[美國國家科學獎章|國家科學獎]]及[[馬蒂亞斯獎|馬蒂亞斯獎]]。他又獲頒工程界最崇高的{{tsl|en|John Fritz Medal|約翰弗里茨獎}}，[[愛迪生|愛迪生]]、[[貝爾|貝爾]]及[[费米|费米]]等傑出科學家，也是這個獎的得主。

== 家庭 ==
朱的妻子陳璞是著名數學大師[[陳省身|陳省身]]的女兒。原籍[[廣東|廣東]][[台山|台山]]之朱經武會講流利[[粵語|粵語]]。

== 曾參於之電視節目／廣告／宣傳片 ==
2008年，朱經武於擔任[[香港科技大學|香港科技大學]]校長期間為鼓勵香港[[高科技|高科技]]發展，參與[[香港電台|香港電台]]節目《[[創新戰隊|創新戰隊]]》，除了提供科學上的專業意見，還戴上半邊面罩飾演劇中[[超級戰隊系列|戰隊]]領袖「石開武博士」。

由於劇集除了在電視上播放，亦有在巴士上的《[[RoadShow|RoadShow]]》頻道上播出精華片段，事件引起民間廣泛討論<ref>[http://www.youtube.com/view_play_list?p=B24023C253F44D6D youtube:創新戰隊]</ref>。之後，他又替[[電視廣播有限公司|TVB]]拍攝「香港力量」宣傳片（2009年）<ref>{{Youtube |id=mRdEXUKBSTg}}</ref>。
