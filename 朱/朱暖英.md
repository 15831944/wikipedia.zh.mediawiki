'''朱暖英'''（{{lang-en|Nuan-Ying Chu}}，{{bd|1965年}}），[[臺中|臺中]]市人，女性，屬[[中國國民黨籍|中國國民黨籍]]，現任第二屆直轄市[[臺中市議員|臺中市議員]]<ref>{{cite web |author1=陳世宗 |url=https://www.chinatimes.com/realtimenews/20180430001412-260407 |title=盧秀燕、朱暖英區里拜票馬不停蹄|website=中時電子報 |accessdate=2018-09-30}}</ref>。2017年，朱暖英曾為救助菜農而認購600公斤[[高麗菜|高麗菜]]<ref>{{cite web |author1=金武鳳 |title=搶救菜農　 朱暖英認購600公斤高麗菜送愛心 |url=https://www.nownews.com/news/20170119/2378551/ |website=今日新聞 |accessdate=2018-09-30}}</ref>。2018年，朱暖英之子舉行婚禮，多位政治人物均到場參加<ref>{{cite web |author1=陳世宗 |title=盧秀燕、朱暖英區里拜票馬不停蹄 |url=https://www.chinatimes.com/realtimenews/20180717003924-260405 |website=中時電子報 |accessdate=2018-09-30}}</ref>。

==言論==
2019年3月9日，朱暖英接受[[中國評論通訊社|中國評論通訊社]]專訪表示：「[[王金平|王金平]]談兩岸關係，卻沒有提到[[九二共識|九二共識]]，將大陸當成敵人，這樣的論述不是國民黨的主張，反而是綠營的主張」。<ref>{{cite news |title=朱暖英：王金平將大陸當敵人是綠營主張 |url=http://hk.crntt.com/doc/1053/6/0/1/105360147.html |agency=中評社 |date=2019-03-09}}</ref>
