'''朱活'''（{{bd|1921年||1997年|12月7日}}），[[号|号]]甦斋<ref name="崔明泉" />，出生于[[浙江|浙江]][[湖州|湖州]]，中国[[钱币学|钱币学家]]、先秦[[史学家|史学家]]，曾担任[[中国钱币学会|中国钱币学会]]常务理事、名誉理事、学术委员会委员，以及[[山东省博物馆|山东省博物馆]]研究员等职。其研究领域主要是先秦钱币，在[[語文學|文献学]]、[[古文字学|古文字学]]、[[考古学|考古学]]和[[历史学|历史学]]等方面也有大量著述。

== 生平 ==
1921年，朱活出生于浙江湖州。因幼时曾得重病，后来痊愈，因此起名为“活”<ref name="崔明泉">{{cite web |author1=崔明泉 口述 |author2=夏晓然 整理 |title=读点·口述：那时在济南，跟着朱活先生赏玩古钱币 |url=http://news.iqilu.com/shandong/kejiaoshehui/20170320/3456135.shtml |website=齐鲁网 |date=2017-03-20}}</ref>。1944年，于重庆[[国立中央大学|国立中央大学]]毕业。曾先后在[[江苏|江苏]][[徐州|徐州]]，[[山东|山东]][[济南|济南]]、[[青岛|青岛]]等地院校任教。曾在[[中国科学院|中国科学院]]山东分院历史研究所从事研究工作<ref>{{cite web |author1=李剑波 |title=认识“孔方兄” |url=http://www.iqilu.com/html/dianshibao/news/2015/1119/2608094.shtml |website=齐鲁网}}</ref>，1959年开始任该所古史组负责人。1971年调山东省博物馆工作，1983年任该馆研究员，直至[[退休|退休]]<ref name="coin001">{{Citeweb|title=朱活老师书法|url=http://www.coin001.com/museum.php?action=myshowdata_show&uid=8461&showdata_id=195 |publisher=殷国清博物馆}}</ref>。

[[文化大革命|文化大革命]]时期，朱活曾被到山东曲阜活动的[[北京师范大学|北京师范大学]][[红卫兵|红卫兵]]指为[[孔子|孔子]]的“孝子贤孙”、“反革命修正主义分子”，曾与曲阜孔子像一起游街，一同被游街的是山东省副省长[[余修|余修]]和中共曲阜县委书记[[李秀_(曲阜官员)|李秀]]<ref name="崔明泉"/>。

此外，曾担任[[中国钱币学会|中国钱币学会]]第一、二届常务理事，第三、四届名誉理事，学术委员会委员；中国考古学会理事等多项职务<ref name="coin001"/>。1986年3月5日，[[国家文物鉴定委员会|国家文物鉴定委员会]]在北京成立，朱活被聘为委员<ref>{{cite journal |title=国家文物鉴定委员会成立 |journal=中国钱币 |date=1986 |issue=2 |page=37}}</ref>。

1987年，朱活突发[[脑溢血|脑溢血]]，抢救后留下难以好转的后遗症<ref name="崔明泉" />。1997年12月7日，因病去世<ref>{{cite journal |author1=姚世铎 |title=哭朱活先生 |journal=西部金融 |date=1998 |issue=5 |page=63}}</ref>。

== 荣誉 ==
曾获得科研奖项“金泉奖”，曾被[[山东省人民政府|山东省政府]]授于“有突出贡献文化艺术工作者”称号，并获[[中国国务院|中国国务院]]颁发的“[[政府特殊津贴|政府特殊津贴]]”证书<ref name="coin001"/>。
