'''朱华荣'''（{{bd|1929年|1月26日|2018年|2月15日}}）[[四川省|四川省]][[璧山县|璧山县]]人，[[中华人民共和国|中华人民共和国]]刑法学的创始人之一。

==生平==
1948年秋，自[[四川大学|四川大学]]法律系毕业，获[[法学士|法学士]]学位。1951年8月被保送至[[中国人民大学|中国人民大学]]法律系学习。1953年2月被抽调至中国人民大学刑法教研室担任教员，研究[[苏联|苏联]]刑法，并负责筹建中国刑法教程，成为中华人民共和国刑法学的创始人之一。1958年被划为“[[反右运动|右派]]”，调到资料室工作。1963年被下放支援[[安徽|安徽]][[芜湖|芜湖]]，先后在中学、小学任教。直到1978年恢复重建[[华东政法学院|华东政法学院]]，他才被调到[[上海|上海]]任教。1982年参加了[[中华人民共和国司法部|中华人民共和国司法部]]领导的“[[林彪、江青反革命集团案|审判林彪、江青反革命集团]]”律师组工作，被[[中华人民共和国最高人民法院特别法庭|最高人民法院特别法庭]]指定为[[江青|江青]]的辩护律师。此后任华东政法学院教授、研究生导师。主要研究方向是中国刑法学、中国古代刑法史、比较刑法学、法律思想史<ref name=zhhr>{{Cite web |url = http://news.ifeng.com/a/20180225/56270156_0.shtml |title =  新中国刑法学创始人之一朱华荣病逝 曾为江青辩护|author =  |date = 2018-02-25 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://phtv.ifeng.com/program/zgjy/detail_2009_12/14/1062816_0.shtml |title = “两案”审判侧记 江青律师述说内幕 |author =  |date = 2009-12-14 |publisher = 凤凰网|language =  |accessdate =  |quote =  }}</ref>。

2018年2月15日，朱华荣在[[上海华山医院|上海华山医院]]病逝，享年89岁<ref name=zhhr/>。

==著作==
朱华荣发表学术论文100多篇。出版或参编《中华人民共和国刑法总则讲义》、《各个刑法比较研究》、《中国刑法通史》、《中国法律思想史》等著作<ref name=zhhr/>。
