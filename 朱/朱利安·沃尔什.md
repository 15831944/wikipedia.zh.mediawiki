'''朱利安·伊茹米·沃尔什'''（{{lang-en|Julian Jrummi Walsh}}，{{bd|1996年|9月18日||}}）是一名代表日本参赛的混血田径运动员，主攻400米赛跑。他的父亲是牙买加人，母亲是日本人<ref name="jamaicaobserver"/>。沃尔什3岁时移居日本，之后在那里成长。他在中学时由于所在学校田径部被废除，他只能加入篮球部。<ref name="higashimurayama">{{cite web|url=https://www.city.higashimurayama.tokyo.jp/shisei/koho/pickup/28pickup/pickup20170105.html |script-title=ja:平成29年新春対談「ウォルシュ ジュリアン選手（陸上競技）×渡部尚（東村山市長）」 |publisher={{lang|ja|[[東村山市|東村山市]]公式サイト}} |date=2017-01-05 |accessdate=2017-03-12 }}</ref>2012年，正就读高中1年级的他开始接受了正式的田径训练，当时他周围并没有任何参加过正式比赛的教练<ref name="RM1410">{{Cite journal|date = 2014-10|script-title=ja:第2回全国高校陸上選抜大会　大会REPROT|journal = {{lang|ja|陸上競技マガジン}}|issue = 16|volume = 64|publisher={{lang|ja|ベースボール・マガジン社}}|pages = 32-35}}</ref>。在升入高中2年级后，情况出现转机，他开始师从曾参加国民体育大会的武井智巳，在那之后，他的水平有所提高<ref name="sportiva">{{cite web|url=https://sportiva.shueisha.co.jp/clm/othersports/rikujo/2017/09/21/___split_46/index.php |script-title=ja:競技歴４年で五輪日本代表になった男、ウォルシュ・ジュリアンは何者か |publisher=[[Sports_Graphic_Number|Sports Graphic Number]] |date=2017-09-21 |accessdate=2017-10-11 }}</ref>。2013年7月，他在县西部地区国民体育大会400米项目中跑出50秒16的成绩<ref>{{cite web |url=http://eishin.ac/club/?p=71 |script-title=ja:ウォルシュ・ジュリアン・ジャミイ君、400mに初出場し優勝！ |publisher={{lang|ja|東野高等学校}} |date=2013-07-26 |accessdate=2015-04-21 |title=存档副本 |archive-url=https://web.archive.org/web/20160305045205/http://eishin.ac/club/?p=71 |archive-date=2016-03-05 |dead-url=yes }}</ref>。同年秋，他首次在该项目闯入49秒，高中3年级时，他在全国高等学校综合体育大会北关东预选赛上以46秒98的成绩获得第一名，成功取得正赛资格<ref>{{Cite journal|date = 2014-0|script-title=ja:インターハイ地区大会 ハイライト|journal = {{lang|ja|月刊陸上競技}}|issue = 9|volume = 48|publisher = {{lang|ja|[[讲谈社|讲谈社]]}}|pages = 72}}</ref>。后来，他被日本田径联盟选中，得以代表日本参加世界青年田径锦标赛男子4×400米接力赛事，最终他和队友获得该项目银牌<ref name="jamaicaobserver">{{Cite web|url=http://www.jamaicaobserver.com/sport/Julian-Walsh--Jamaican-born-Japanese-athlete-making-a-name-in-400m_17543296|title=Julian Walsh: Jamaican-born Japanese athlete making a name in 400m - Sport|last=Campbell|first=Howard|date=2014-09-14|publisher=Jamaica Observer|access-date=2016-08-15}}</ref>。2016年，他在日本田径锦标赛男子400米项目上以45秒35的成绩获得第一名，这一成绩达到了奥运参赛资格标准成绩（45秒40），当时仅有4年的田径参赛经历的他也因此成功获得里约奥运参赛资格<ref name="sportiva"/><ref name="sponichi">{{cite web|url=https://www.sponichi.co.jp/sports/news/2016/06/26/kiji/K20160626012851850.html|script-title=ja:新星１９歳ジュリアン圧巻Ｖ　ジャマイカとハーフ「ヤバイです」|publisher={{lang|ja|[[体育日本|スポーツニッポン]]}}|date=2016-06-26|accessdate=2018-09-25}}</ref>。
