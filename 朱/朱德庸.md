'''朱德庸'''{{BD|1960年|4月16日|catIdx=Z}}，[[籍貫|籍貫]][[江蘇|江蘇]][[太倉|太倉]]，[[台灣|台灣]][[漫畫家|漫畫家]]，[[世新大學|世界新聞專科學校]][[三專|三專]]制（三年制專科學校）電影編導科畢業。代表作為《[[醋溜族|醋溜族]]》、《[[雙響炮|雙響炮]]》、《[[澀女郎|澀女郎]]》。

「人」是他漫畫的主要因素；他曾經說：「在我自己的漫畫裏，其實最喜歡的是《醋溜族》。」

朱德庸於2016年获得[[文化部|文化部]]所举办第7届[[金漫奖|金漫奖]]评审团的“特别贡献奖”，推崇他在漫画文化的推广上的长足贡献。<ref>{{Cite news|url = http://www.epochtimes.com/gb/16/6/14/n7997314.htm|title = 金漫奖入围揭晓 朱德庸获特别贡献奖|last = |first = |date = 2016-06-15|work = |accessdate =|publisher = 大纪元}}</ref>

== 人物 ==
* 主要創作為[[四格漫畫|四格漫畫]]。1993年以[[林則徐|林則徐]]為藍本創造男性角色「[[徐則林|徐則林]]」，之後另外創造女性角色「徐則琳」，作為[[行政院衛生署|行政院衛生署]]、[[董氏基金會|董氏基金會]]拒菸公益廣告的雙代言人。2007年《甜心澀女郎》獲[[國立編譯館優良漫畫獎|國立編譯館優良漫畫獎]]短篇漫畫組第2名。
* 作品主題常是現代社會中的男女關係，最早成名的作品是《[[雙響炮|雙響炮]]》，內容由懼內的老公跟兇惡其貌不揚的老婆開始，延伸至他們的岳母、兒子、媳婦、親家母等，其中的妙言妙語令人會心一笑。另一作品《[[澀女郎|澀女郎]]》，內容是由四個不同個性的女子來看都會女子的生活、工作與愛情，曾被改編為[[連續劇|連續劇]]。除四格漫畫外，也曾創作非四格漫畫《[[什麼事都在發生|什麼事都在發生]]》。
