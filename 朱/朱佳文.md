'''朱佳文'''（英文名'''Namei'''，{{bd|1993年|5月4日 || }}），ID原为NaBhenmei,后改为namei。游戏《[[英雄联盟|英雄联盟]]》玩家，前职业选手，在职业选手期间担任ADC。

== 简介 ==

幼年时，朱佳文成长于单亲家庭，由于缺乏家庭的关怀，朱佳文在2000年左右接触到网络游戏。<ref name="demaxiya">{{Cite web|url = http://www.demaxiya.com/news/20150727712.html|title =意料之外的世界第一ADC：想赢 这是Namei的故事}}</ref>

2012年9月，朱佳文正式加入WE俱乐部并在队中开始入队的试训。在加入WE试训后不久，朱佳文以优异的成绩留队，并加入当时的WE二队i-Rocks（后战队改名为PE战队）。<ref name="demaxiya"></ref>2013年LPL夏季赛，PE战队取得LPL夏季赛冠军。<ref name="lol17173">{{Cite web|url = http://lol.17173.com/content/2013-11-24/20131124204100620.shtml|title =PE击败OMG获2013年LPL夏季赛总冠军宝座}}</ref>

2014年2月，朱佳文从PE战队转会到EDG战队。<ref name="qq">{{Cite web|url = http://games.qq.com/a/20140210/016539.htm|title =娜美将加入EDG 西卡放弃2014年LPL春季赛}}</ref>同年，EDG取得LPL春季赛冠军。<ref name="pcgames">{{Cite web|url = http://wangyou.pcgames.com.cn/390/3903741.html|title =2014LPL春季赛颁奖仪式：EDG加冕冠军}}</ref>

2014年S4世界总决赛，由于朱佳文因为吃牛肉面搞坏肚子，导致全队发挥失常，EDG战队以2：3的败于皇族，止步八强。<ref name="sina">{{Cite web|url = http://dj.sina.com.cn/article/fychhuq3369821.shtml|title =现实很残酷！世界第一ADC竟沦为三线主播}}</ref>

2015年3月9日，朱佳文从EDG战队转会到皇族战队。<ref name="loldjsina">{{Cite web|url = http://lol.dj.sina.com.cn/2015-03-10/1159607259.shtml|title = LOL娜美加入“复仇者联盟” 13日首批皇族战袍出征}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2015LPL春季赛，在皇族战队因战绩不佳降级至LSPL联赛，朱佳文随同除godlike以外的皇族战队成员转入King战队。<ref name="carry6">{{Cite web|url = http://www.carry6.com/lol/201508/4181551326.html|title =从世界亚军到降级 你是否还记得当年的皇族}}</ref>2015年LPL夏季赛，KING战队降级至LSPL联赛。<ref name="pcgames5396464">{{Cite web|url = http://wangyou.pcgames.com.cn/539/5396464.html|title =撑天巨树助2144D晋级 King遗憾降级LSPL}}</ref>

2015年12月24日，朱佳文从King战队转会到RNG战队。<ref name="carry62">{{Cite web|url = http://www.carry6.com/lol/201512/1204758505.html|title =娜美Insec转入RNG 皇族原队员全部留队}}</ref>

2017年1月5日，朱佳文宣布自己已离开RNG战队。<ref name="mnw">{{Cite web|url = http://www.mnw.cn/sports/dzjj/201701/1533835.html|title =Namei正式宣布离开RNG战队成自由人}}</ref>

== 评价 ==

2013年，[[拳头公司|拳头公司]]将朱佳文评为世界第一ADC。<ref name="demaxiya"></ref>
