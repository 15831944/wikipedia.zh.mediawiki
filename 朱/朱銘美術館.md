'''朱銘美術館'''（{{lang-en|Juming Museum}}）由藝術家[[朱銘|朱銘]] （{{lang|en|Ju Ming}}）於1999年9月19日正式成立<ref>{{Cite web |url=http://www.juming.org.tw/opencms/juming/know/affair/affair_0000.html |title=朱銘美術館開幕 |accessdate=2014-03-02|publisher=朱銘美術館}}</ref>，是[[台灣|台灣]]最大的戶外美術館<ref>{{Cite web|url=http://museum.moc.gov.tw/frontsite/museum/museumListAction.do?method=doMuseumDetail&museumId=37|title=財團法人朱銘文教基金會朱銘美術館|accessdate=2014-03-02|publisher=中華民國文化部|deadurl=yes|archiveurl=https://web.archive.org/web/20140302051909/http://museum.moc.gov.tw/frontsite/museum/museumListAction.do?method=doMuseumDetail&museumId=37|archivedate=2014-03-02}}</ref>，座落在[[新北市|新北市]][[金山區_(台灣)|金山區]]西勢湖2號<ref>{{Cite web |url=http://travel.network.com.tw/main/travel/point/showpage/912.html |title= 旅遊資訊王TravelKing 旅遊導覽 - 新北市 - 金山區 - 朱銘美術館|accessdate=2014-03-02|publisher=旅遊王 }}</ref>。原址本是朱銘用來放置大型[[雕塑|雕塑]]作品之地，而後作品漸多，朱銘乃萌生創建美術館的念頭。整座美術館的設計、建造共耗費12年<ref>{{Cite web |url=http://artmagazine.com.tw/ArtCritic/article658.html |title=以朱銘美術館做為台灣發展文創產業的典範  |accessdate=2014-03-02|publisher=全球華人藝術網 |date=2013-07-13 }}</ref>。為大台北地區重要的美術館之一。

== 園區簡介 ==
美術館佔地約11甲<ref>{{cite news|title=朱銘美術館 |url=http://www.taiwanfun.com/north/taipei/recreation/0312/0312JumingTW.htm |publisher=Taiwan Fun雜誌 |date=2003-12 |accessdate=2014-03-02}}</ref>，目前開放8甲左右<ref>{{cite news|title=朱銘美術館 金山的藝術花園|url=http://www.epochtimes.com.tw/014/1569.htm|publisher=台灣大紀元|accessdate=2014-03-02|deadurl=yes|archiveurl=https://web.archive.org/web/20130630150831/http://www.epochtimes.com.tw/014/1569.htm|archivedate=2013-06-30}}</ref>，大多為戶外展示空間。全館空間可分成服務中心、第一展覽室、第二展覽室、運動廣場、藝術表演區、朱雋區、兒童藝術中心、[[太極|太極]]廣場、美術館本館、人間廣場、慈母碑、藝術交流區、科學園區、藝術長廊等。園區依山而建，高度約在海拔165-185公尺間，園區內可遠眺金山海景。

== 典藏作品 ==
館內除了收藏朱銘本人的作品外，亦藏有其師[[李金川|李金川]]、[[楊英風|楊英風]]，其子[[朱雋_(雕塑家)|朱雋]]以及許多國內外重要藝術家之作品──國內部分，有[[郭柏川|郭柏川]]、[[廖繼春|廖繼春]]、[[李澤藩|李澤藩]]、[[楊三郎|楊三郎]]、[[張萬傳|張萬傳]]、[[洪瑞麟|洪瑞麟]]等人的作品。國外部份，有[[畢卡索|畢卡索]]、[[米羅|米羅]]、[[達利|達利]]、[[亨利·摩尔|亨利·摩尔]]、[[安迪沃荷|安迪沃荷]]等人之作。

== 獲獎記錄 ==
2000年，朱銘美術館以「透過與天空山海的對話，治癒現代人心靈的21世紀藝術創作空間」為由，榮獲日本第14屆[[東京創新大賞|東京創新大賞]]海外賞<ref>{{cite news|title=朱銘 「塑」造人間經典 |url=http://orientaldaily.on.cc/cnt/lifestyle/20140302/00322_001.html |publisher=東方日報 |accessdate=2014-03-02}}</ref>。並讚揚該館「超越現有的美術館、徜徉於大自然中『活生生的美術館』，是休憩的場所，亦是創作的地方。」
