'''朱江户'''（{{bd|1908年||1994年|8月1日|朱江户}}），又名'''朱刚夫'''，江西省莲花县琴亭镇花塘村人。国民党议员、江西教育厅厅长[[朱念祖|朱念祖]]之子，清末帝师、京师大学堂监督[[朱益藩|朱益藩]]之侄曾孙。[[柏林大学|柏林大学]]经济学博士，茶叶专家，中国[[林业经济学|林业经济学]]先驱<ref> {{Cite thesis |degree= 硕士|chapter=1.2.1 关于林业经济管理教育起源的研究 |title= 中国林业经济管理教育发展及其特征研究 |author= 王磊 |year= 2012 |publisher= 北京林业大学}}</ref>。先后于[[德国共产党语言小组|德国共产党语言小组]]、国民革命军七十军、[[江西省莲花中学|莲花县立初级中学]]、[[东南茶叶研究所|东南茶叶研究所]]、中央人民政府农业计划司、[[中南军政委员会|中南军政委员会]]、[[北京大学|北京大学]]、[[北京林学院|北京林学院]]、中国林业经济学会<ref>{{cite journal en|author = 中国林业经济学会秘书处 |year = 1985|title =  林业经济专题讲座在京举行 |journal = 《林业经济》 |issue = 05|pages = 59|publisher = 中国林业经济学会|location = 北京市|issn = 1673-338X|language = 中文|accessdate = 2016-03-06 }}</ref>等单位任职。为中国民主同盟北京市第五届委员会市委委员<ref>{{Cite web |url = http://www.bjmm.org.cn/outpart/managerarticle.do?method=page&articleId=749 |title = 中国民主同盟北京市第五届委员会 |date = 2012-07-10 |language = 中文 |accessdate = 2016-03-06 }}</ref>，中国民主同盟北京市第六届<ref>{{Cite web |url = http://www.bjmm.org.cn/outpart/managerarticle.do?method=page&articleId=750 |title = 中国民主同盟北京市第六届委员会 |date = 2012-07-10 |language = 中文 |accessdate = 2016-03-06 }}</ref>、第七届<ref>{{Cite web |url = http://www.bjmm.org.cn/outpart/managerarticle.do?method=page&articleId=751 |title = 中国民主同盟北京市第七届委员会 |date = 2012-07-10 |language = 中文 |accessdate = 2016-03-06 }}</ref>委员会顾问。

== 个人经历 ==

1908年，朱江户出生于日本江户。1927年朱江户高中毕业。1928年至1928年在日本{{tsl|jp|浪速高等学校 (旧制)|浪速高等学校 (旧制)|浪速高等学校}}留学，期间参加了反对帝国主义大同盟。1932年前往德国柏林大学学习。1935年在德国参加[[中国共产党|中国共产党]]担任德国共产党语言小组的组织委员，同时积极参加反帝活动<ref> {{Cite book | author = 罗银胜 | title = 乔冠华传：红色外交家的悲喜人生 | location = 北京 | publisher = 文化艺术出版社 | date = 2012年5月 | pages = 20 | ISBN = 978-7-5039-5015-5  | language = 中文  }} </ref>，参加了旅德华侨抗日救国联合会，负责宣传工作<ref>{{Cite web |url = http://www.gzzxws.gov.cn/gzws/gzws/ml/35/200809/t20080916_7925_1.htm |title = 爱国革命志士 英勇抗日将军——纪念张炎烈士殉难四十周年 |date = 2008-09-16 |language = 中文 |accessdate = 2016-03-06 }}</ref>。在德留学期间，对[[卡夫卡|卡夫卡]]的著作产生浓厚兴趣<ref>{{Cite web |url = http://www.zgchawang.com/culture/show-16881.html |title = 一个茶叶家族的浮沉往事，一段辛酸的奋斗史 |date = 2012-02-22 |language = 中文 |accessdate = 2016-02-06 }}</ref>，与同在德国留学的[[乔冠华|乔冠华]]关系颇好。一次“留德学生抗日联合会”召开大会，中国驻德使馆通知希特勒政府，派出大队警察，与[[黄琪翔|黄琪翔]]一同被捕，后经多方交涉，给予释放。<ref> {{Cite book | author = 王大鲁,刘清云 | title = 黄琪翔传 | location = 北京太平桥大街23号 | publisher = 中国文史出版社 | date = 1994年11月 | pages = 84 | ISBN = 7-5034-0728-X  | language = 中文  }} </ref>

1938年2月，由全欧华侨抗日救国联合会组织，与乔冠华、[[温朋九|温朋九]]、刘绛雯（温朋九之妻）一同乘坐法国游船“霞飞”号回国，与陈希周、许德瑗、游毓桢被中共长江局派遣参加[[云干班|云干班]]（国民革命第七十军战时步兵干部培训班）<ref>{{cite journal en|author = 杨秀春，宋耕春|year = 2007|title = 抗战时期中共对湖南国民党乙派的统战工作 |journal = 湖南工程学院学报(社会科学版) |issue = 1|pages = 105-108|publisher = 湖南工程学院|location = 湖南|issn = 1671-119X|language = 中文|accessdate = 2016-03-06 }}</ref>，教授“大众哲学”和“日语”，于国民革命军七十军担任参议和政治总教官，上校军衔，受陈希周委托与[[杨东莼|杨东莼]]分别在武汉和长沙招收青年学生，办事处设原日租界。<ref>{{cite journal en|author = 陈安贤,韩绍武,赵慨萍,文振亚|year = 1989|title = 我党在“云干班”的活动|journal = 《湘潮》|issue = 07|pages = 22-25|publisher = 《湘潮》杂志社|location = 长沙市韶山路1号|issn = 1003-949X|language = 中文|accessdate = 2016-03-03 }}</ref>1940年，陈希周被暗杀，国民革命第七十军军长[[李觉|李觉]]紧急安排他去桂林避难，是时其弟朱洁夫正于桂林黄琪翔处任职。1941年任福建省崇安东南茶叶研究所研究员。1942年3月，任莲花县立初级中学校长<ref> {{Cite book | author = 李炳恩,陈移新 | title = 江西莲花中学志 | location = 北京 | publisher = 人民日报出版社 | date = 2011 | pages = 426-427 | ISBN = 978-7-5115-0681-8  | language = 中文  }} </ref>，1945年6月应[[吴觉农|吴觉农]]邀请重返位于衢州万川的东南茶叶研究所工作。<ref>{{cite journal en|author = 陈 观 沧|year = 1989|title = 忆崇安茶叶研究所|journal = 《福建茶叶》|issue = 03|pages = 48-49|publisher = 福建省茶叶学会|location = 福建省祥福州市湖东路189号“凯捷大厦”6层西侧 |issn = 1005-2291|language = 中文|accessdate = 2016-03-03 }}</ref>1946年4月前往台湾省工作，任台北商品检验局局长。

1950年2月返回大陆，任中央人民政府农业计划司编纂处副处长。同年9月，因[[陈铭枢|陈铭枢]]请求，由[[周恩来|周恩来]]致电[[邓子恢|邓子恢]]，改任中南军政委员会农林部副部长（陈铭枢任部长）。1952年5月调任北京大学任教。同年10月，北京林学院成立，调任该校担任教授，加入[[中国民主同盟|中国民主同盟]]，参与学院创建、教学以及科研工作。<ref> {{Cite book | author = 李炳恩,陈移新 | title = 江西莲花中学志 | location = 北京 | publisher = 人民日报出版社 | date = 2011 | pages = 426-427 | ISBN = 978-7-5115-0681-8  | language = 中文  }} </ref>1953年，北京林学院于林学专业下开设了“林业政策与管理”课程，由朱江户教授负责，属于全国首创。<ref>{{Cite web |url = http://fzghc.bjfu.edu.cn/xwdt/113706.htm |title = 《林业经济管理》学科简介 |date = 2012-10-29 |language = 中文 |accessdate = 2016-02-06 }}</ref>1954年11月，北京林学院成立“林业经济教研组”，朱江户任该组主任。<ref>{{Cite web |url = http://em.bjfu.edu.cn/xygk_1/fzlc/174430.htm |title =     北京林业大学经济管理学院历史沿革 |date = 2014-09-15 |language = 中文 |accessdate = 2016-02-06 }} </ref>1957年5月，参加林业部组织的赴[[德意志民主共和国|德意志民主共和国]]和[[捷克斯洛伐克共和国|捷克斯洛伐克共和国]]林业考察团。<ref> {{Cite book | author = 北京林业大学校史编辑部 | title = 北京林业大学校史 1952-2002 | location = 北京市西城区刘海胡同7号 | publisher = 中国林业出版社 | date = 2002年10月 | pages = 336 | ISBN = 7-5038-3222-3  | language = 中文  }} </ref>

在北京林学院任教期间加入[[中国民主同盟|中国民主同盟]]，历任学院民盟小组副组长、组长（1979年）、主任委员（1985年）。<ref> {{Cite book | author = 北京林业大学校史编辑部 | title = 北京林业大学校史 1952-2002 | location = 北京市西城区刘海胡同7号 | publisher = 中国林业出版社 | date = 2002年10月 | pages = 415-416 | ISBN = 7-5038-3222-3  | language = 中文  }} </ref>文革期间受到冲击。1986年5月，重新加入中国共产党。

1987年，朱江户教授退休。1994年8月1日，朱江户教授去世，享年86岁。
