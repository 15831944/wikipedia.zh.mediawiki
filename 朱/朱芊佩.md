'''朱芊佩'''（{{lang-en|Chu Tsin Pui}}，{{bd|1988年|10月30日}}），香港[[網絡紅人|網絡紅人]]及運輸從業員。

== 簡歷 ==
朱芊佩有一位妹妹，父親是個經營物流公司的香港商人，母親則來自[[台灣|台灣]][[日月潭|日月潭]]高山族，一家居住在[[美孚新邨|美孚新邨]]<ref name=CCP>{{cite web |url=https://new.qq.com/omn/20180604/20180604A1EGT4.html |title=港版罗拉：像女人一样强 |publisher=[[騰訊網|騰訊網]] |date=2018-06-04 |accessdate=2019-01-30}}</ref>。她於1995年唸小學二年級時隨家人搬到台灣定居，2001年又因父親往內地發展，便在[[廈門|廈門]]接受了三年初中教育<ref name=CCP2>{{cite web |url=https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/190892/%E6%B8%AF%E7%89%88%E7%BE%85%E6%8B%89%E6%89%8B%E6%8C%87%E9%AA%A8%E8%A3%82%E6%9C%AA%E7%97%8A%E7%99%92%E5%8D%B3%E5%BE%A9%E5%B7%A5-%E5%A0%85%E5%BC%B7%E8%83%8C%E5%BE%8C%E5%85%85%E6%BB%BF%E6%B4%8B%E8%94%A5 |title=港版羅拉手指骨裂未痊癒即復工　堅強背後充滿洋蔥 |publisher=[[香港01|香港01]] |date=2018-05-22 |accessdate=2019-01-30}}</ref>；正值初中三年級，其父生意失敗且常為錢爭執，令她必須出外工作以幫補家計，未幾兩者關係日漸惡劣，遂於2004年獨自回香港投靠姑母<ref name=CCP0/>，插班繼續完成[[香港中學會考|會考]]課程。

2006年，朱芊佩中五畢業後曾當過表姐夫公司的文員，轉任酒店前堂接待員與保安、室内泳池救生員和兼職清潔工<ref name=CCP/>。直至2008年，她開始任職運輸從業員，主要在[[新蒲崗|新蒲崗]]工廠區跟車，分別向批發市場、超市運送水果及糧油雜貨，經歷過兩次工傷，導致左手一指不能正常屈曲，現時亦從事廢紙箱回收、建築工地廢料清理、室内裝修、[[搭棚|搭棚]]等副業<ref name=CCP/>；還在媒體訪問裏再三透露，想考取客貨車駕駛執照，以提升技能來增值<ref name=CCP/>。
