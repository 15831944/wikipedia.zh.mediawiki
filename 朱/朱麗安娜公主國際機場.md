'''茱莉安娜公主國際機場'''（'''{{lang|nl|Princess Juliana International Airport}}'''，{{Airport codes|SXM|TNCM}}）是一個位於[[加勒比海|加勒比海]][[聖馬丁島|聖馬丁島]]荷屬部分的國際機場，機場命名自[[荷兰|荷蘭]]當時的王位推斷繼承人[[朱丽安娜_(荷兰)|茱莉安娜公主]]（即後來的茱莉安娜女王）。茱莉安娜公主機場是[[背风群岛_(加勒比海)|背風群島]]地區重要的航空樞紐，其繁忙程度在-{zh-hans:东; zh-hant:東;}-加勒比地區僅次於[[聖胡安國際機場|聖胡安國際機場]]。在2005年，此機場共接待了逾166萬名旅客。<ref>[http://www.pjiae.com/business-movements.html 數據] {{webarchive|url=https://web.archive.org/web/20080517121040/http://www.pjiae.com/business-movements.html |date=2008-05-17 }}</ref>

== 歷史 ==
茱莉安娜公主機場於1942年開始運作，初期為軍事用途，1943年轉為民用。機場於1964年進行翻新，興建了一幢新的客運大樓及[[塔台|塔台]]，相關設施於1985年再度被更新。

== 翻新計劃 ==
由於航空客運量持續上升，此機場於1997年起，定下了大型翻新計劃，工程共分三期進行。<ref>[http://www.pjiae.com/masterplan.html 茱莉安娜公主國際機場整體翻新計劃] {{webarchive|url=https://web.archive.org/web/20070814032552/http://www.pjiae.com/masterplan.html |date=2007-08-14 }}</ref>

=== 第一期 ===
是一個短期計劃，主要項目是為提升現有設施及改善服務，其中包括：
* 擴闊、加固及翻新跑道
* 增加滑行道的負重量
* 建造新的停機坪，以增加飛機停泊空位
* 翻新客運大樓

第一期計劃已於2001年完成<ref>[http://www.pjiae.com/phase-one.html 茱莉安娜公主國際機場第一期翻新計劃] {{webarchive|url=https://web.archive.org/web/20070927100750/http://www.pjiae.com/phase-one.html |date=2007-09-27 }}</ref>

=== 第二期 ===
工程包括：
* 興建雷達設備及新的控制塔台
* 增建一幢面積更大（約27,000平方米），設備更完善的客運大樓，每年可處理約250萬旅客。新大樓樓高4層，共有42個辦理登機手續櫃檯、8個轉機櫃檯、11個登機閘口及15個辦理出入境的櫃檯（10個設於入境大堂，5個設於出境大堂）。
* 根據[[国际民航组织|國際民航組織（ICAO）]]的規定，於跑道兩端盡頭增設150米長的緩衝區

新塔台於2004年3月29日開始投入運作，而新客運大樓則於2006年11月10日正式啟用。<ref>[http://www.pjiae.com/phase-two.html 茱莉安娜公主國際機場第二期翻新計劃]</ref>

=== 第三期 ===
截至2008年8月為止，第三期工程仍未展開。倘若流量增長率達到預期水平，第三期工程便會展開。項目包括擴建全新的大樓、及興建與跑道平行的滑行道系統。<ref>[http://www.pjiae.com/phase-three.html 茱莉安娜公主國際機場第三期翻新計劃] {{webarchive|url=https://web.archive.org/web/20070927102625/http://www.pjiae.com/phase-three.html |date=2007-09-27 }}</ref>

== 跑道 ==

此機場的最大特色，就是其只有2,349米（7,708呎）長的跑道<ref>[http://www.pjiae.com/pdfs/company-profile.pdf 機場檔案] {{webarchive|url=https://web.archive.org/web/20050526154747/http://www.pjiae.com/pdfs/company-profile.pdf |date=2005-05-26 }} (2007)</ref>，但大型客機仍可安全降落。由于10号跑道距离海边太近，无法设置气流缓冲区。當飛機在到達機場附近的[[瑪侯海灘|瑪侯海灘]]（Maho Beach）時，離海灘高度只有約10-20米，故成為了不少飛機愛好者[[飞机摄影|拍攝飛機]]降落的勝地。同時瑪候海灘也豎立了警告牌，提示遊人注意飛機降落時所帶來的氣流。<ref>{{Cite web |url=http://travel.sina.com.cn/ent/2009-01-06/102151867.shtml |title=全球10大最惊险的飞机跑道——朱莉安娜公主国际机场(图)_新浪旅游_新浪网 |accessdate=2018-10-26 |work=travel.sina.com.cn}}</ref>

2008年,原09/27跑道更名为10/28跑道。

截至2007年初，[[荷蘭皇家航空|荷蘭皇家航空]]（KLM）停止了至茱莉安娜公主機場的定期[[波音747|波音747]]航班。但之后又重新使用波音747执行飞往该岛的航班，直至2016年。
