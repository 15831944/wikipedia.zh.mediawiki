'''朱丽安娜女王'''（'''Juliana Louise Emma Marie Wilhelmina'''，{{bd|1909年|4月30日|2004年|3月20日}}），全名'''朱丽安娜·路易丝·艾玛·玛丽·威廉明娜'''，荷兰女王[[贝娅特丽克丝_(荷兰)|贝娅特丽克丝]]的母亲。1948年9月[[登基|登基]]，任[[荷兰女王|荷兰女王]]，1980年4月[[內禪|內禪]]让位给女儿[[贝娅特丽克丝|贝娅特丽克丝]]。

朱丽安娜在位時見證荷蘭兩大殖民地—[[荷屬東印度|荷屬東印度]]和[[荷屬圭亞那|荷屬圭亞那]]先後獨立為[[印度尼西亞|印度尼西亞]]和[[蘇里南|蘇里南]]。在她駕崩時，她以94歲之齡成為世上擔任過君主中最年長的人。

== 生平 ==
朱丽安娜出生于[[荷兰|荷兰]][[海牙|海牙]]的[[努兒登堡宮|努兒登堡宮]]，是[[威廉明娜女王|威廉明娜女王]]和[[亨德里克亲王_(梅克伦堡-什未林)|亨德里克亲王]]的獨女，故甫出生已是荷蘭王位推斷繼承人。

二战期间[[德国|德国]]侵占荷兰后，荷兰王室於1940年5月到[[英国|英国]]和[[加拿大|加拿大]]避难，朱丽安娜与4个女儿中的3个在加拿大。戰爭結束後朱麗安娜等人於1945年回国，1948年9月威廉明娜女王因健康原因退位，39岁的朱丽安娜继承王位。
[[File:Koningin_Juliana_bezoekt_de_Ir_Jonkie_school_te_Hwijk,_Bestanddeelnr_910-8472.jpg|thumb]]
剛登基的朱丽安娜急需處理荷蘭最重要的殖民地—荷屬東印度獨立的事宜，她登基後一年，荷屬東印度的大部分便正式獨立成印尼；僅餘的[[荷屬新畿內亞|荷屬新畿內亞]]也於13年後（1962年）在爭議中被迫交出，並併入印尼。而這片前殖民地在獨立後仍帶予女王和國家麻煩—爭取脫離印尼獨立的[[南摩鹿加共和國|南摩鹿加激進份子]]曾嘗試綁架女王，以求迫使荷蘭承認南摩鹿加主權。綁架行動最後失敗，但仍導致國內發生多宗類似綁架事件。<ref>{{cite web|url=https://www.nytimes.com/1989/06/09/world/vught-journal-remember-the-moluccans-is-this-a-last-stand.html|title=Vught Journal; Remember the Moluccans? Is This a Last Stand?|author=Rule, Sheila|date=9 June 1989|publisher=''The New York Times''}}</ref>

此外，在她任內，荷蘭王國的政治架構於1954年出現重大變革：原本的殖民地[[荷屬安的列斯|荷屬安的列斯]]和[[蘇里南|蘇里南]]去殖民化，成為與本土平起平坐的[[構成國|構成國]]，並實現自治。惟後者於1975年完全獨立。

朱丽安娜在荷兰受到广泛的拥戴，被誉为“人民的女王”，每年4月30日她的生日是荷兰公共假日“女王日”。她的丈夫[[贝恩哈德亲王_(利珀-比斯特费尔德)|贝恩哈德亲王]]曾经是荷兰军队总司令，因1976年卷入[[洛克希德|洛克希德]]贿赂案而放弃所有的军衔和军职。她71岁生日时把王位传给長女碧翠丝。2004年，94岁的太上女王朱丽安娜因[[肺炎|肺炎]]於3月20日在[[莎士迪克宮|莎士迪克宮]]逝世。

== 子嗣 ==
[[File:De_prinsessen_Beatrix,_Irene,_Margriet_en_Christina_wandelen_in_het_park_van_het,_Bestanddeelnr_255-7522.jpg|thumb]]
朱丽安娜和丈夫[[貝恩哈德親王_(利珀-比斯特費爾德)|贝恩哈德亲王]]共有4个女儿：
# [[荷兰贝娅特丽克丝女王|贝娅特丽克丝女王]]－1938年1月31日生于蘇斯臺克宮。
# [[伊莲公主_(荷兰)|伊莲公主]]－1939年8月5日生。
# [[玛格丽特公主_(荷兰)|玛格丽特公主]]－1943年1月19日生于[[加拿大|加拿大]]首都[[渥太華|渥太華]]的渥太華醫院(Ottawa, Ontario)的[[国际|国际]][[租借地|租借地]]{{noteTag|加拿大當時是英國的領地，在加拿大誕生者會予以英國國籍，但是荷蘭並不承認雙重國籍。有人错误地认为当时加拿大國会给的是“荷兰”租借地，其实，荷兰是属人主义，加拿大（英国统治）是属地主义，只要公主出生在国际领土从而避免获得英国籍就可以了。因此為了讓瑪格莉特公主不要失去荷蘭的國籍和公主的身分，[[加拿大國會|加拿大國會]]特別通過特別法，讓朱麗安娜等人的病房成為法外治區（请注意，不是临时的“荷兰”租借地，而是临时的“国际”租借地），以保護瑪格莉特公主的權利。朱麗安娜對此深深地感謝，戰後贈送鬱金香的球根給渥太華。}}。
# [[克里斯蒂娜公主_(荷兰)|克里斯蒂娜公主]]－1947年2月18日生于蘇斯臺克宮。
