'''朱蓓薇'''（{{bd|1957年}}），女，[[江苏|江苏]][[宜兴|宜兴]]人，[[大连工业大学|大连工业大学]]食品学院院长、[[博士生导师|博士生导师]]，主要从事农、水产品精深加工基础理论和应用研究。

== 简历 ==
1982年取得[[大连轻工业学院|大连轻工业学院]]（今大连工业大学）制糖工程专业工学学士学位，1982年留校至今从事教学科研工作。2004年获[[日本|日本]][[冈山大学|冈山大学]]农学博士学位。2013年当选[[中国工程院|中国工程院]]院士。<ref >{{cite web|url=http://news.cnr.cn/native/city/201312/t20131219_514440738.html | language= | title=大连工业大学朱蓓薇教授增选为中国工程院院士 |publisher=中国广播网 |date=2013-12-19 |accessdate= }}</ref>

== 头衔 ==
*中国食品科学技术学会常务理事
*国家海洋食品工程技术研究中心主任
*国家高技术研究发展计划（863计划）海洋技术领域主题专家
*海洋食品教育部工程研究中心主任
*农业部“农产品加工技术研发贝类加工专业分中心”主任
*辽宁省食品科学技术学会理事长

== 荣誉 ==
主持完成国家、省、市攻关课题30多项，获17项各种科研成果奖励。获得40多项国际国内授权[[发明专利|发明专利]]；出版8部学术著作；发表160多篇学术论文。

获1项[[国家技术发明奖|国家技术发明奖]]二等奖、1项[[国家科技进步奖|国家科技进步奖]]二等奖、[[何梁何利基金|何梁何利基金]]“科学与技术创新奖”，以及大连市科学技术功勋奖。
