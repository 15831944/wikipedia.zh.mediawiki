'''朱光潜'''（1897年9月19日－1986年3月6日），字'''孟實'''，筆名'''孟實'''、'''孟石'''。[[安徽|安徽]]省[[桐城县|桐城县]]人，[[淸華大學|淸華大學]]、[[北京大學|北京大學]]、[[四川大學|四川大學]]、[[武汉大学|武汉大学]]等教師。

== 早年生活 ==
自幼接受父親朱子香嚴格的[[私塾|私塾]]教育，十五歲開始就讀於孔城高小（今[[孔城鎮|孔城鎮]]桐鄉書院內），熟讀[[古文|古文]]及[[唐|唐]]、[[宋詩|宋詩]]奠定其[[中國|中國]]古典文學的基礎。半年后考入[[桐城中學|桐城中學]]，棄時文而從古文，受國文教師潘季薰陶，對中國舊詩產生了濃厚的興趣。中學畢業后，朱氏進入[[国立武汉大学|武昌高等師範學校]]中文系（今[[武汉大学文学院|武汉大学文学院]]）學習。入學后很快考取[[北洋政府|北洋政府]]教育部派送生資格，於民國七年至民國十一年，就讀於[[香港大學|香港大學]]，學習了英國語言與文學、教育學、生物學、心理學等課程，奠定了其一生參與教育活動和學術活動的方向。期間曾受[[梁启超|梁启超]]与[[胡適|胡適]]之新文化運動影響，毅然放棄古文和文言文，改寫白話文，不久即用白話文發表了其美學研究的處女作《無言之美》。

香港大學畢業後，經同班好友[[高覺敷|高覺敷]]介紹結識了[[吳淞|吳淞]][[中國公學|中國公學]]的校長、[[清华大学|清华大学]]的原校长[[張東蓀|張東蓀]]，并於民國十一年夏到該校中學部教習英文課程，兼校刊《旬刊》的主編。江浙戰爭中吳淞中國公學關閉，隨即到[[浙江|浙江]]白馬湖[[春暉中學|春暉中學]]教書，因而結識[[朱自清|朱自清]]、[[匡互生|匡互生]]、[[夏丏尊|夏丏尊]]、[[豐子愷|豐子愷]]、[[葉聖陶|葉聖陶]]等好友。後來決定離開春暉中學到[[上海|上海]]剛成立[[立達學園|立達學園]]，提出[[教育|教育]]獨立[[自由|自由]]的宣言，並一方面籌辦開明書局（出版作品皆以中學生為對象的出版社）和雜誌《一般》。

== 留學英法 ==
1925年，考取公費留英，到[[愛丁堡大學|愛丁堡大學]]選修英國文學、哲學、心理學、歐洲古代史和藝術史等，畢業後至[[倫敦大學學院|倫敦大學學院]]聽莎士比亞課程，同時又在巴黎大學註冊，對研究西方[[哲學|哲學]]、[[藝術|藝術]]美學、[[心理學|心理學]]產生濃厚興趣。期間受巴黎大學文學院長德拉庫瓦教授所講授的[[藝術心理學|藝術心理學]]啓發，起念寫下了《文藝心理學》。后又就讀於法國[[斯特拉斯堡大學|斯特拉斯堡大學]]，幷於1933年獲得斯特拉斯堡大學[[博士|博士]]學位，同時完成了其著作《悲劇心理學》。在歐洲留學期間，朱氏經常由於公費不發，經濟拮据，只得靠稿費維持生活。朱氏為開明書局的《一般》撰稿，曾輯成《給青年的十二封信》,該書反應了當時一些青年的心理狀況，暢銷全國。接著《變態心理學派別》（開明書局）、《談美》、《變態心理學》（[[商務印書館|商務印書館]]）、《文藝心理學》、《[[詩論|詩論]]》，及譯著的《美學原理》陸續問世。朱氏還在[[羅素|羅素]]的影響下寫過一部講述符號邏輯派別的書（交於商務印書館，後在[[抗日戰爭|抗日戰爭]]中遭焚毀）。

==返國任教==
===國民政府時期===
1933年回國，聘為[[安徽师范大学|安徽大学]]（现[[安徽师范大学|安徽师范大学]]）{{NoteTag|原安徽大学为安徽师范大学前身。}}文学院院长后，經好友[[徐中舒|徐中舒]]介紹給當時的北京大學文學院院長胡適，委聘為北京大學西語系教授、主任，除教授西方名著選讀和文學批評史以外，還在北京大學中文系和[[清華大學|清華大學]]的中文系研究班講授《文藝心理學》和《詩論》，後又應留法結識的好友[[徐悲鴻|徐悲鴻]]邀請到中央藝術學院教授了一年的《文藝心理學》。當時正值[[京派|京派]]文藝和[[海派|海派]]文藝對壘。京派於[[新月|新月]]時期最盛，卻在詩人[[徐志摩|徐志摩]]死於飛機失事后日漸衰落。[[胡適|胡適]]、[[楊振聲_(教育家)|楊振聲]]等人想重振京派，便組織朱光潛、[[沈從文|沈從文]]、[[周作人|周作人]]、[[林徽因|林徽因]]等人組成編委會，籌辦《文學雜誌》。朱光潛時任主編，[[朱自清|朱自清]]、[[聞一多|聞一多]]、[[馮至|馮至]]、[[李廣田|李廣田]]、[[何其芳|何其芳]]、[[卞之琳|卞之琳]]等人常為雜誌撰稿。抗日戰爭期間，朱光潛经[[国立西南联合大学|国立长沙临时大学]]文学院教授，應[[四川大學|四川大學]]代理校長張頤之約，任文學院院長，后游於[[武漢大學|武漢大學]]任教。[[国民党|国民党]]六大召开前夕，列入[[朱家驊|朱家驊]]与[[陈立夫|陈立夫]]联名向[[蒋介石|蒋介石]]推荐的98名“最优秀教授党员”之一<ref>[http://news.ifeng.com/history/zhongguojindaishi/detail_2013_04/16/24258575_0.shtml 45年国民党“最优秀教授党员”：华罗庚陈寅恪冯友兰]</ref>。

===中華人民共和國時期===
1949年中共建政以后，朱氏留居中国大陆，他到1952年秋季以后，转入北京大学哲学系，此后一直居住于燕京大学撤销后的燕东园27号。朱氏雖擁護無産階級專政，可是其美學思想曾數度遭受批判，[[文化大革命|文化大革命]]時被指控為「反動學術權威」，遭到抄家、鬥爭，並被剝奪執譯者的權利。
1976年文革結束後，朱光潛被恢復教授職務。此後致力於学术研究和教育领域，積極翻译各類外文名著，撰写文稿，发表演讲及指导研究生。1983年3月，他应邀去香港中文大学讲学，一开始他就声明自己的身份：「我不是一个[[共产党|共产党]]员，但是一个[[马克思主义|马克思主义]]者。」以此作為他对自己后半生的思想定位。

===逝世===
1984年夏朱光潜由于多年工作过于疲累，罹患疲劳综合症，出现脑血栓，儘管健康狀況大為轉差，他仍舊勤於執筆寫作。在朱氏逝世的前3天，他趁自己神志仍有稍许清醒，乘家人不防，竟艰难地沿梯独自悄悄向楼上书房爬去，家人发现急来劝阻，他嗫嚅地说，要赶在死前把《新科学》的注释部分完成。1986年3月6日，朱光潜在北京病逝，终年89岁。

== 學術成就==
朱氏為中国现代[[美学|美学]]的开拓者和奠基者之一。並致力於文藝思想硏究和批評、美學研究、美學教學，治學態度認真，介紹、翻譯、論著美學的著述極為豐富，多達六百餘萬字，開拓了中國美學研究的新[[領域|領域]]。《西方美学史》是朱光潜最重要的一部著作之一，也是中国学者撰写的第一部美学史著作，具有开创性的学术价值，代表了中国研究西方美学思想的水平。对[[黑格尔|黑格尔]]《美学》的翻译，为他赢得了崇高声誉。他的数量众多的译著和译文为中国的美学研究和文艺理论研究铺平了前进的道路。早年，朱光潜是[[意大利|意大利]]美学家[[克罗齐|克罗齐]]的忠实信徒，并对近代美学的的真正开山祖[[伊曼努尔·康德|康德]]非常崇敬。
