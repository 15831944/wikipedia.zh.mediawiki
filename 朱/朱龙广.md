'''朱龙广'''（{{bd|1939年|9月||}}）生于[[陕西省|陕西省]][[西安市|西安市]]，[[中华人民共和国|中华人民共和国]]演员、导演。

==生平==
朱龙广高中毕业后，进入兰州艺术学院表演系，1962年毕业。在校期间，朱龙广曾被校医院误诊为[[肺结核|肺结核]]，休学回西安后，在七家医院检查确定是误诊。朱龙广休学在家时，宝鸡市话剧团正排话剧《降龙伏虎》，朱龙广经他人介绍成为该剧男一号，此角为朱龙广赢得全国青年演员一等奖。1961年，朱龙广调入[[中国人民解放军工程兵|军委工程兵]]文工团，历任演员队长、导演<ref name=gmw/><ref name=dyw/>。

1965年，朱龙广出演了电影《[[地道战_(电影)|地道战]]》。当时，工程兵文工团接到编写《地道战》剧本的任务后，开始演员选拔工作，导演让朱龙广饰演男主角高传宝。朱龙广随剧组赴[[河北省|河北省]][[清源县|清源县]][[冉庄地道战遗址|冉庄]]拍摄。《地道战》上演后，朱龙广塑造的民兵队长高传宝一角红遍中国<ref name=gmw/>。

[[文化大革命|文化大革命]]期间，1969年因“中毒较深、改造态度不好”，朱龙广被从部队复员至西安东风仪表厂当钳工。不久被调到该厂党委宣传部工作。一年以后，在一场军工企业文艺汇演中，朱龙广自编自演的《齿轮问题》获得陕西省相关部门领导注意，乃被调到[[陕西人民艺术剧院|陕西省人民艺术剧院]]负责招生授课。他担任考官招收了[[李琦_(演员)|李琦]]等学生<ref name=gmw/>。

1980年，朱龙广加入[[中国戏剧家协会|中国戏剧家协会]]。在1980年代拍摄的电视剧《[[西游记_(1986年电视剧)|西游记]]》中，朱龙广饰演的[[如来佛祖|如来佛祖]]广受观众欢迎，后来以朱龙广的这一形象印制的如来佛祖画像在[[中国大陆|中国大陆]]及海外广泛流传。1992年，朱龙广被聘为北京电影电视学院教筹处主任，东方电影电视学院高级顾问和副院长。退休后被聘为中国老龄委金秋艺术团团长<ref name=dyw>{{Cite web |url = http://www.1905.com/mdb/star/8868/ |title = 朱龙广 |author =  |date =  |publisher = 电影网 |language =  |accessdate = 2017-04-17 |quote =  }}</ref><ref>{{Cite web |url = http://mini.eastday.com/a/160630191045679.html |title = 77岁的如来佛祖.朱龙广近照 |author =  |date =2016-06-30   |publisher = 东方网 |language =  |accessdate =  |quote =  }}</ref>。退休后，朱龙广常到艺术学校教表演课，并继续在影视作品中饰演角色，还经常在家画[[中国画|中国画]]<ref name=gmw>{{Cite web |url = http://www.gmw.cn/01wzb/2005-09/08/content_301714.htm |title =“高传宝”细说舞台缘  |author =  |date = 2005-09-08 |publisher = 光明网|language =  |accessdate =  |quote =  }}</ref>。
