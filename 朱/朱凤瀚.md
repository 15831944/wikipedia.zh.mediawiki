'''朱凤瀚'''（{{bd|1947年|7月||}}）[[江苏|江苏]][[淮安|淮安]]人，生于[[北京|北京]]，[[中华人民共和国|中华人民共和国]]历史学家，[[中国历史博物馆|中国历史博物馆]]馆长，[[中国国家博物馆|中国国家博物馆]]常务副馆长、党委副书记<ref name=ellb>{{Cite web |url =http://auction.artron.net/20050622/n15959.html  |title =朱凤瀚 为国家博物馆2008全新登场  |author =  |date = 2005-06-22 |publisher =雅昌艺术网  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
朱凤瀚是[[中国共产党|中国共产党]]党员。1988年在[[南开大学|南开大学]]获历史学博士学位。1990年任南开大学历史系教授，1993年任[[博士生导师|博士生导师]]，1995年任系主任，同年任南开大学人文学院副院长。1998年7月任[[中国历史博物馆|中国历史博物馆]]副馆长，2000年7月任馆长。2003年2月任[[中国国家博物馆|中国国家博物馆]]常务副馆长、党委副书记。后兼任[[北京大学|北京大学]]及南开大学历史系教授、博士生导师，南开大学学术委员会委员、[[中国博物馆学会|中国博物馆学会]]副理事长、中国先秦史学会副会长、中国殷商文化学会副会长、北京大学中国考古学研究中心与中国古代史研究中心学术委员会委员、北京大学历史系学术委员会委员、[[中国社会科学院|中国社会科学院]]考古研究所《考古学报》、《考古》杂志编委等职。是[[中华人民共和国人事部|中华人民共和国人事部]]批准的国家级“有突出贡献的中青年专家”<ref name=ellb/>。

==著作==
朱凤瀚主要从事中国古代史·先秦史、古文字、[[青铜器|青铜器]]研究。主要著作有：
*《商周家族形态研究》
*《古代中国青铜器》
*《先秦史研究概要》<ref>{{Cite web|url = http://www.scuphilosophy.org/ScholarsLibrary_list.asp?userid=834|title = 朱凤瀚|author = |date = |publisher = 四川大学哲学研究所|language = |accessdate = 2018-01-04|quote = |archive-url = https://web.archive.org/web/20171029090618/http://www.scuphilosophy.org/ScholarsLibrary_list.asp?userid=834|archive-date = 2017-10-29|dead-url = yes}}</ref>
