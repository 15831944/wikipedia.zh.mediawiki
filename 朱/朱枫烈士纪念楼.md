'''朱枫烈士纪念楼'''，又名'''憩园'''，位于[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[镇海区_(宁波市)|镇海区]][[招宝山街道|招宝山街道]][[镇海中学|镇海中学]]内，是中国共产党情报人员[[朱枫|朱枫]]的故居。1996年11月，朱枫烈士纪念楼被公布为[[镇海区文物保护单位|镇海区文物保护单位]]。朱枫纪念楼现为国家安全教育基地、浙江省省级爱国主义教育基地、浙江省党史教育基地、宁波市十佳党史教育基地。<ref>{{cite web |url=http://www.zhxww.net/zhnews4073/mskd/mskd_news/201703/20170329070056.asp |title=镇中多种形式纪念朱枫 |author=杨东 |date=2017-03-29 |website=镇海新闻网 |publisher=镇海区新闻中心 |access-date=2018-07-30 |quote=位于镇海中学校园内的朱枫故居被列为国家安全教育基地、省级爱国主义教育基地、省党史教育基地、市十佳党史教育基地。}}</ref>

== 历史 ==
朱枫烈士纪念楼建于[[中华民国_(大陆时期)|民国时期]]，为[[镇海县|镇海]]、[[舟山市|舟山]]地区知名富商、朱枫之父朱云水晚年所建的朱家花园的一部分，1905年时朱枫在此出生。1992年，[[镇海中学|镇海中学]]扩建田径场时保留并拨款30余万元修缮了朱家花园内的憩园，并向社会征集朱枫生前的遗物。1995年时，朱枫纪念楼向社会开放，内陈列朱枫的生平事迹。<ref>{{cite web |url=http://news.cnnb.com.cn/system/2010/12/26/006792389.shtml|title=枫叶红于二月花——烈士朱枫的传奇人生 |author=<!--Staff writer(s); no by-line.--> |date=2010-12-26日 |website=中国宁波网 |publisher= |access-date=2018-07-30 |quote=|deadurl=no|archiveurl=https://web.archive.org/web/20171223220111/http://news.cnnb.com.cn/system/2010/12/26/006792389.shtml|archivedate=2017-12-23}}</ref><ref>{{cite web |url=http://dzb.hxnews.com/2014-01/23/content_164474.htm |title=朱枫纪念楼中 有吴石身影  |author=<!--Staff writer(s); no by-line.--> |date=2014-01-23 |website=海峡网 |publisher=海峡都市报 |access-date=2018-07-30 |quote=}}</ref>1996年11月，朱枫烈士纪念楼成为第二批镇海区文物保护单位。<ref>{{Cite web |url=http://www.bytravel.cn/Landscape/35/zhufenglieshijinianlou.html |title=朱枫烈士纪念楼 |language=zh-hans |accessdate=2018-07-30 |publisher=博雅文化旅游网 }}</ref>

== 建筑形制 ==
朱枫烈士纪念楼为江南园林风格的砖木结构建筑，占地217平方米，建筑大小五间两层，有曲折小巧的特点。外园月亮门上部为圆形门洞，下部两侧为饰有卷草、如意纹样的须弥座式基石，于门楣上刻有憩园二字，围墙饰有镂空围窗，天井内有花坛、石笋等装饰。<ref>{{cite web |url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=957&type=2|title=朱枫烈士纪念楼|author=<!--Staff writer(s); no by-line.--> |date=2018-01-08|website= 宁波文化遗产保护网|publisher=宁波市文物保护管理所|access-date=2018-07-30|quote=|deadurl=no|archiveurl=https://web.archive.org/web/20150328235433/http://www.nbwb.net/pd_wwbh/info.aspx?Id=957&type=2|archivedate=2015-03-28}}</ref>
