'''朱音唯'''（{{lang-ja|朱音ゆい}}，{{bd|1989年|8月8日||}}），[[日本|日本]]前[[AV女優|AV女優]]，[[東京都|東京都]]出身<ref name="PRESTIGE"/>。曾是「[[PRESTIGE|PRESTIGE]]」的專屬女優<ref name="PRESTIGE"/>，所屬於「{{link-ja|MUSE COMMUNICATION|ミューズコミュニケーション}}」事務所<ref>{{cite web|url=http://musefan.com/talent.html |title=MUSE COMMUNICATION TALENTS PROFILE |publisher={{link-ja|MUSE COMMUNICATION|ミューズコミュニケーション}} |accessdate=2012-06-26}}</ref>。

== 人物 ==
* 興趣是唱[[卡拉OK|卡拉OK]]、玩[[遊戲|遊戲]]<ref name="PRESTIGE"/>。

== 简歷 ==
2011年9月，在日本[[AV片商|AV片商]]「[[PRESTIGE|PRESTIGE]]」成為專屬女優[[AV|AV]][[出道|出道]]<ref>{{cite web|url=http://www.playno1.com/article-6428-1.html#.XBpdpFwzaUk |title=prestige新人朱音ゆい |publisher=AVNo.1-PLAYNO.1 |date=2011-09-03}}</ref>。

2012年12月，在作品總集篇《イキ過ぎた著エロBEST8時間》發布之後，宣布從AV界引退<ref>{{cite web|url=http://www.playno1.com/article-8457-1.html#.XBoByVwzaUl |title=朱音ゆい快玩完了? |publisher=AVNo.1-PLAYNO.1 |date=2012-07-23}}</ref>。
