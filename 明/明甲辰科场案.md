'''明甲辰科场案'''，也称'''嘉靖二十三年会试科场案'''，是指明朝[[嘉靖|嘉靖]]二十三年（1544年）的科举[[会试|会试]]舞弊案。

== 背景 ==
嘉靖年间，京师地区的考试舞弊现象已经存在。礼部官员[[陈裴|陈裴]]曾为考生冒籍事件向[[明世宗|明世宗]]作过详细报告，历述了顺天府科举考试的舞弊情况。之后有御史上书揭发朝廷官员的亲属冒充京籍中举。御史[[谢九仪|谢九仪]]也揭发郑梦纲等十人的冒籍行为。有官员揭露顺天乡试副主考[[浦应麒|浦应麒]]出卖考题<ref>{{Cite book|title=古代科举|last=熊庆年编著|first=|publisher=上海：[[东方出版中心|东方出版中心]]|year=2008.04|isbn=7-80627-148-1|location=|pages=215-216}}</ref>。

== 经过 ==
嘉靖二十三年（1544年），甲辰科会试主考官、[[明朝禮部尚書|礼部尚书]]兼[[翰林学士|翰林学士]][[張潮_(正德進士)|张潮]]在进入[[贡院|贡院]]后，三场笔试刚结束突然病逝。副主考官[[江汝璧|江汝璧]]主持试事，并在同考官[[茅瓒|茅瓒]]的协助下主持阅卷，录取了328人。期间江汝璧与考官[[沈坤|沈坤]]、[[彭鳳|彭凤]]、[[欧阳㬇|欧阳㬇]]、[[高節_(探花)|高节]]等人，徇私舞弊，录取了[[大学士|明朝大学士]][[翟銮|翟銮]]的儿子翟汝孝、翟汝俭。殿试之后，翟汝孝位列一甲第三名。事后，有人举报翟氏兄弟在嘉靖二十一年一同乡试中举，会试又一同登进士。舆论一时沸腾，并流传歌谣“一銮当道，双凤齐鸣”<ref>{{Cite book|title=三千年文祸|last=谢苍霖，万芳珍著|first=|publisher=南昌：[[江西高校出版社|江西高校出版社]]|year=2015.10|isbn=978-7-5493-3572-5|location=|pages=353}}</ref>；因此，翟銮提出复试的要求。但此提议遭到[[明世宗|明世宗]]拒绝，明世宗改派吏部、[[都察院|都察院]]调查处理<ref>{{Cite book|title=鲤鱼跃龙门科举文化丛书 科考轶闻知多少|last=蔺德生，赵萍著|first=|publisher=北京：[[语文出版社|语文出版社]]|year=2015.10|isbn=978-7-5187-0109-4|location=|pages=125-126}}</ref>。

后经审查，乡试时正副主考[[秦鳴夏|秦鸣夏]]、[[浦应麒|浦应麒]]存在讨好翟銮而录其二子。而会试时副主考江汝璧等又有意把翟氏兄弟、翟氏受业师崔奇勋，翟汝俭亲家焦清等四人的座位安排在一起（同号）<ref>{{Cite book|title=中国考试辞典|last=[[汤德用|汤德用]]等主编|first=|publisher=合肥：[[黄山书社|黄山书社]]|year=1998.10|isbn=7-80630-341-3|location=|pages=359}}</ref>。考官欧阳㬇作为翟氏兄弟老师佯装避嫌，却私下帮忙辨认字迹。同考官中，彭凤录取翟氏兄弟及崔奇勋，沈坤录取陆炜，[[高節_(探花)|高节]]录取江一中。此外，高节通过校尉张岳受贿五百两银，录取彭谦<ref>{{Cite book|title=中国科举文化通志 七史选举志校注|last=[[赵伯陶|赵伯陶]]校注|first=|publisher=武汉：武汉大学出版社|year=2015.12|isbn=978-7-307-17188-6|location=|pages=626}}</ref>。

明世宗将翟銮、翟汝孝、翟汝俭、崔奇勋、焦清、彭凤、欧阳㬇削职为民。江汝璧及秦鸣夏、浦应麒无受贿情节，各杖六十，革职闲住。沈坤、陆炜、江一中未见贿赂罪行，情节较轻，留用供职。高节、张岳二人有受贿罪行，发配充军，彭谦削职为民<ref>{{Cite book|title=古谣谚 一百卷|last=（清）[[杜文澜|杜文澜]]辑|first=|publisher=北京：[[中华书局|中华书局]]|year=1958.01|isbn=|location=|pages=957}}</ref>。负责会试纠察的监察御史[[王珩_(嘉靖進士)|王珩]]、[[沈越_(明朝進士)|沈越]]失职，各降一级，调任地方<ref>{{Cite book|title=中国考试制度史资料选编|last=[[杨学为|杨学为]]等主编|first=|publisher=合肥：黄山书社|year=1992.08|isbn=7-80535-293-3|location=|pages=298}}</ref><ref>{{Cite book|title=人文论丛 2015年第1辑（总第23卷）|last=教育部人文社会科学重点研究基地，武汉大学中国传统文化研究中心主办|first=|publisher=武汉：[[武汉大学出版社|武汉大学出版社]]|year=2015.09|isbn=978-7-307-16553-3|location=|pages=185}}</ref>。

== 其他 ==
