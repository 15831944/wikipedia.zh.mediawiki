'''明天'''{{BD|1995年|4月8日|||M}}，是一名中国[[足球|足球]]运动员，司职后卫。目前效力于[[中甲|中甲]]球队[[武汉卓尔|武汉卓尔]]。

==俱乐部生涯==
明天出道于[[武汉卓尔|武汉卓尔]]青训系统。[[中华人民共和国第十二届运动会|2013年全运会]]乙组比赛代表湖北队出场，表现受到好评。2014年1月与队友[[聂傲双|聂傲双]]一起被选派至法国[[索肖足球俱乐部|索肖足球俱乐部]]进行为期三个月的学习训练。<ref>{{cite web |url=http://sports.sina.com.cn/j/2013-11-28/13436907876.shtml |title=卓尔牵手法甲索肖俱乐部 两希望之星将赴法“镀金” |author= |date=2013年11月28日 |publisher=新浪体育  |accessdate=2017年1月18日}}</ref>2014年进入武汉卓尔一线队名单但并未在联赛中出场。同年入选[[中国国家青年足球队|中国国家U-19青年队]]并参加[[2014年亚足联U-19锦标赛|亚洲青年锦标赛]]。2016年开始进入球队主力阵容发挥出色并夺得当赛季中甲最佳新人奖。<ref>{{cite web |url=http://sports.sohu.com/20161111/n472922486.shtml |title=中甲最佳新秀明天：感谢武汉卓尔这个大家庭 |author= |date=2016年11月11日 |publisher=搜狐体育  |accessdate=2017年1月16日}}</ref>2017年1月12日武汉卓尔俱乐部官方宣布续约明天至2018年。<ref>{{cite web |url=http://sports.sohu.com/20170112/n478495737.shtml |title=拒绝挖角！武汉卓尔续约中甲最佳新人明天 |author= |date=2017年1月12日 |publisher=搜狐体育  |accessdate=2017年1月16日}}</ref>
