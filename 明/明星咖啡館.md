'''明星咖啡館'''是位於[[臺灣|臺灣]][[臺北市|臺北市]][[中正區_(臺北市)|中正區]]武昌街一段5號的[[西餐廳|西餐廳]]，同時販售[[麵包|麵包]]與[[糕點|糕點]]，成立於1949年。

== 歷史 ==
[[俄國共產革命|俄國共產革命]]後，流亡到[[中國|中國]]的[[俄羅斯|俄羅斯]]皇室軍官艾斯尼（George Elsner）<ref>{{cite web |url = https://www.mirrormedia.mg/story/20161021bus001/ |title = 《台灣老店》蛋糕裡的俄國貴族返鄉路　明星咖啡 |author = 謝祝芬 |date = 2016-10-25 |publisher = [[鏡週刊|鏡週刊]] |accessdate = 2018-07-10}}</ref>投靠在上海[[霞飛路|霞飛路]]經營「ASTORIA咖啡廳」的同鄉布爾林<ref name="文化部">[http://travel.culture.tw/template/StrokePage.aspx?Sid=153 濃醇的臺北文學地標──明星咖啡館] {{Wayback|url=http://travel.culture.tw/template/StrokePage.aspx?Sid=153 |date=20180113035500 }}，文化部臺灣深度之旅</ref>。兩人均為[[俄國內戰|俄國內戰]]後流徙海外的「[[白俄_(移民)|白俄]]」。

1949年，艾斯尼等人與[[簡錦錐|簡錦錐]]於台北市武昌街一段7號合作經營「明星西點麵包廠」製作俄式麵包與西點，並於麵包店二樓開設「明星咖啡館」販賣俄羅斯風味簡餐與咖啡。「明星」名稱是從其[[俄文|俄文]]店名「Astoria」而來<ref name="文化部" />。已故總統[[蔣經國|蔣經國]]夫人[[蔣方良|蔣方良]]即常來此品嚐家鄉味。蔣經國本人也曾多次與夫人在此品嘗俄式料理，並慶祝新年。<ref>[http://news.xinhuanet.com/book/2009-02/23/content_10873509.htm 蒋经国与俄罗斯夫人蒋方良的两三珍闻轶事]</ref>亦吸引中華民國政要、商人，畫家們（如[[郎靜山|郎靜山]]、[[陳景容|陳景容]]、[[楊三郎|楊三郎]]、[[顏水龍|顏水龍]]等）亦常會來此聚會。

1959年，詩人[[周夢蝶|周夢蝶]]在明星咖啡館樓下[[騎樓|騎樓]]擺起小書攤，引來愛好文藝者聚集，初露嶄角的年輕作家亦常於此寫稿，如[[三毛|三毛]]、[[黃春明|黃春明]]、[[林懷民|林懷民]]、[[白先勇|白先勇]]、[[季季|季季]]、[[陳若曦|陳若曦]]、[[楚戈|楚戈]]、[[方明|方明]]、[[劉大任|劉大任]]、[[王禎和|王禎和]]、[[陳映真|陳映真]]等人；《現代文學》、《[[創世紀詩刊|創世紀詩刊]]》、《文學季刊》也曾在此編輯討論。白先勇曾言：「臺灣六十年代的現代詩、現代小說，羼著明星咖啡館的濃香，就那樣，一朵朵靜靜地萌芽、開花。」<ref>白先勇（1984）。明星咖啡館：白先勇論文雜文集。皇冠出版社，台北，63-64。</ref>

1989年，明星咖啡店休業，二樓空間租與居仁堂素食館，但一樓的麵包廠依然營業。

2003年明星咖啡店[[火災|火災]]，簡錦錐的女兒簡靜惠自美返臺；2004年5月，明星咖啡館重新營業。<ref>[http://www.cna.com.tw/news/firstnews/201405015006-1.aspx 周夢蝶熟悉的明星咖啡屋，中央通訊社，2014/05/01]</ref>

2012年1月9日，[[臺北市政府文化局|臺北市政府文化局]]「北市文化二字第10130269900號」公告登錄武昌街一段5、7號明星咖啡館為臺北市歷史建築。<ref>[https://nchdb.boch.gov.tw/assets/advanceSearch/historicalBuilding/20120109000001  明星咖啡館].文化部文化資產局</ref>

2015年5月29日，明星咖啡館舉辦66周年暨新書《明星咖啡館》發表會，簡錦錐夫婦與來賓一起切蛋糕慶祝<ref>{{cite news|author=郭佳容|publisher=[[中國時報|中國時報]]|url=http://www.chinatimes.com/newspapers/20150530000400-260115|title=明星咖啡館66年 藝文人回家齊聚|date=2015-05-30|accessdate=2015-05-30}}</ref>。
