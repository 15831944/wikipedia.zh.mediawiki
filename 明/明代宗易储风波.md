'''明代宗易储风波'''，是指中国[[明朝|明朝]][[明代宗|明代宗]]接替[[土木之变|土木之变]]中被俘的[[明英宗|明英宗]]皇位后，欲废除明英宗太子[[朱见深|朱见深]]立自己孩子为太子的政治事件。明代宗在[[于谦|于谦]]等大臣的辅佐下，获得了[[北京保卫战|北京保卫战]]的胜利后，将太上皇明英宗迎回来，并囚禁于[[南宫|南宫]]，防止其[[复辟|复辟]]。与此同时，产生了将自己的儿子立为皇太子的念头，为了实施这一意图，他进行了一系列工作，不择手段，乃至贿赂大臣，不惜武力威慑{{参|name=参考1}}。皇帝行贿百官换太子算是中国历史上少有的事情{{参|傅贵，《景泰帝行贿换太子》，《决策》 , 1996 (4) :48-48。}}。

== 背景 ==
{{main|土木之变}}
正统十四年（1449年），明代宗的皇帝哥哥明英宗御驾亲征，留他为监国，结果发生[[土木堡之变|土木堡之变]]，明英宗被俘，[[瓦剌|瓦剌]]咄咄逼人，进攻北京，一时之间，朝野震惊，京城内外人人自危。八月二十八日，[[王文_(明朝)|王文]]上书明代宗，希望他能以江山社稷为重，承继大统。当时的明英宗的长子[[朱见深|朱见深]]年仅两岁{{参|《明英宗睿皇帝实录》卷之一百八十一 正统十四年八月 ○丁卯。司礼监太监金英传，奉皇太后圣旨，今立皇帝庶长子见深为皇太子。}}，为免主少国疑，于谦等大臣劝服[[孙太后|孙太后]]，最终在正统十四年（1449年）九月拥立明代宗为帝，改元景泰，遥尊明英宗为太上皇{{参|《明英宗睿皇帝实录》 卷之一百八十三 废帝郕戾王附录第一 正统十四年九月癸未，上在迤北，郕王即皇帝位，尊上为太上皇帝，诏天下曰：朕以皇考宣宗章皇帝仲子奉藩京师}}。同时孙太后表示大明江山依然是明英宗的，明代宗只是代理执政而已，并立明英宗之子[[朱见深|朱见深]]为太子。

明代宗在位期间，知人善任，启用于谦等正直之人，励精图治，选将练兵，击退了瓦剌的入侵，使得江山社稷转危为安，又对政治、经济、军事等方面进行了整顿和改革，使当时明朝社会由乱而治渐开中兴，可谓英明之主{{参|尹选波，《试论景泰帝朱祁钰》，明史研究, 1993。}}。随着帝位渐渐巩固，明代宗并不满足，他不仅自己要做皇帝，而且希望自己儿子[[朱见济|朱见济]]能够取代明英宗的太子朱见深成为皇位的合法继承人，于是他一手导演了贿赂朝臣的闹剧。

== 过程 ==
[[File:明宪宗像.jpg|180px]]
由于太子是皇太后立的，明代宗不得不慎重。他先试探颇有资历的太监[[金英|金英]]，说“七月初二日，是东宫太子的生日”。金英立刻回答，“东宫生日是十一月初二日。”前者是[[朱见济|朱见济]]的生日，后者是太子朱见深的生日{{参|明·陆容 《菽园杂记·卷一》：“景皇帝既即位，意欲易储。一日，语英曰：“七月初二日，东宫生日也。”英叩头云：“东宫生日是十一月初二日。”上为之默然。”。 }}{{参|name=参考1|牛创平，《景帝朱祁钰易立太子案》，《文化交流》2008年第10期。}}。明代宗听到金英这样说，感觉没有达到理想的效果。恨恨作罢。

明代宗隐忍了一段时间，,他仍旧不停地向周边人进行试探，一个叫[[兴安_(明朝)|兴安]]的太监为他出了一个主意，就是通过甚至贿赂朝臣，让他们在重建储君的问题上能站在自己这边。不久后，明代宗召集内阁成员包括首辅[[陈循|陈循]]、次辅[[高榖|高榖]]等六人开会，而明代宗只是跟他们随便聊了一些家常，谈一些无关紧要的公事。后宣布散会，搞得大臣们一头雾水，不过散会后，兴安太监分别找到了他们，奉上事前准备好的银子，具体数额是：首辅陈循、次辅高榖每人一百两，其余四位阁员每人五十两{{参|《明史·陈循传》中载：“帝〔明景帝〕欲易太子，内畏诸阁臣，先期赐循(文渊阁大学十陈循)及高谷(大学士)白金百两，江渊、王一宁、肖鎡半之。比下诏议，循等遂不敢诤(而遵旨票拟)”。}}。这六人都是精明之人，知道皇帝的意思，所以睁一只眼闭一只眼，以后默不作声。

明代宗搞定了内阁大臣，接下来就是群臣，虽然钱不多，只有区区百十两银子，但其中意思大臣们都心领神会。收了钱的大臣们一致同意改立太子的倡议{{参|{{cite web |author1=刘继兴 |title=奇闻：明景帝向群臣行贿 花百十两银子“买”太子 |url=http://news.ifeng.com/history/zhongguogudaishi/detail_2012_07/13/16012824_1.shtml |website=凤凰网 |publisher=《廉政瞭望》2010年03期 |accessdate=2018-06-10}}}}。经过明代宗的不惜努力终于换来宦官和朝臣的默认，可是他的[[汪皇后|汪皇后]]坚决不能同意丈夫的做法。她悍然争辩，却被废了皇后，打入冷宫。景泰三年（1452年），明代宗以“父有天下必传于子”为借口，废掉明英宗之子朱见深的皇太子之位，贬为沂王{{参|《明史卷·十三·本纪第十三》： 英宗留瓦剌，皇太后命立为皇太子。景泰三年，废为沂王。天顺元年，复立为皇太子，改名见深。}}，册立自己的儿子朱见济为皇太子，朱见济的生母杭氏母凭子贵立为[[杭皇后|杭皇后]]。可想不到，景泰五年（1454年）朱见济就夭折了。明代宗也因此在精神上受到了沉重的打击，皇储再次出现空缺。当初是明代宗亲自废了朱见深，如果再立，恐怕百年之后会遭到清算。在这些内外因素的作用下，明代宗对于复立太子一事，保持着高度警惕。此时朝中大臣希望他在重新确立朱见深的太子之位，被他给驳回了{{参|王嘉谅，《做忠臣不能胡来》，《传奇故事:百家讲坛中旬》 , 2013 (7) :12-12。}}。甚至他们被关押并受到残酷的对待，有几人被鞭笞致死{{参|{{cite web |author1=剑桥中国明代史 |title=北京的防御和新帝的即位 |url=http://www.guoxue123.com/other/jq/ming/024.htm |website=国学导航 |accessdate=2018-06-10}}}}。

明代宗保国有功，未能把皇位还给明英宗还说得过去，但他把明英宗之子的太子位也废了，朝臣上下认为朱祁钰私心过重，有失民心。贵州道监察御[[史钟同|史钟同]]曾言：“太子薨逝，足知天命有在。” {{参|《明史·卷一百六十二·列传第五十》：“父有天下，固当传之于子。乃者太子薨逝，足知天命有在。臣窃以为上皇之 子，即陛下之子。”。}}同时兼陈一切弊政。明代宗闻讯大怒，钟同被下狱杖死。

== 后续 ==
{{main|夺门之变}}
明代宗正当壮年，子嗣的问题在这个时候尚未显得突出。可是到了景泰八年（1457年）正月初，他突然得了重病，皇储的问题再次让众臣议论纷纷。而内宫传来消息，说明代宗病体康健了。于是众臣准备第二天上朝的时候再商议皇储问题。这一夜，却爆发了夺门之变。明英宗重新登上帝位，而明代宗却被赶下皇位，一个月后病死。

== 评价 ==
明代宗时期的易储风波，在纵向维度上贯穿于整个英宗时期，即正统末、景泰朝、天顺初三个时间段。在横向维度上涉及了明英宗时期的各种争执力量。从明代宗正式登基伊始，政治隐患已然埋下，而景泰三年的更易皇储事件直接导致世袭帝系的变化，后因新皇储朱见济的去世，再次引起朝中对复立朱见深的纷争。在这样的背景下，更储事件中朝中各方势力集团的此消彼长对更储进度的影响{{参|王景，《明英宗时期更储风波研究》， 黑龙江大学，2016年09月14日。}}。

景泰朝这一系列事件，反映出在新君与旧主并存的两难困境之下，君臣之间灵活运用社稷、宗法、君臣之义、父子之伦等话语来具体解决现实的帝位与储位问题的过程{{参|任建敏，《景泰年间的君臣关系与政治——围绕帝位与储位问题的考察》，中山大学 , 2011。}}。
