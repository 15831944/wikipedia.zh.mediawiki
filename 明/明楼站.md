'''明楼站'''是[[浙江省|浙江省]][[宁波市|宁波市]]规划中的一座地下轨道交通车站。明楼站是[[宁波轨道交通3号线|宁波轨道交通3号线]]和[[宁波轨道交通6号线|6号线]]的换乘站，3号线车站计划于2019年启用<ref name="main">{{cite web|url=http://daily.cnnb.com.cn/nbrb/images/2014-12/24/A2/nbrb20141224a02.pdf|title=宁波地铁3号线一期开工|author=冯瑄|publisher=宁波日报|date=2014-12-24|accessdate=2014-12-26}}</ref>。

== 位置 ==
明楼站位于[[鄞州区|鄞州区]]中兴路、通途路路口。3号线车站位于中兴路上，为南北方向。6号线车站位于通途路上，为东西方向<ref name="main" />。

== 车站形制 ==
明楼站3号线车站为地下二层岛式车站，长152米，宽21.7米，总建筑面积15776平方米。6号线为地下三层岛式车站，位于通途路。两线通过通道换乘<ref name="design">市发展改革委关于宁波市轨道交通3号线一期工程初步设计的复函, 甬发改审批〔2014〕653号</ref><ref name="main" />。

== 出口 ==
3号线明楼站规划设置3个出入口<ref name="design" />。
