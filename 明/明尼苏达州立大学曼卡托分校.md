'''明尼苏达州立大学曼卡托分校'''（Minnesota State University, Mankato）是位於美國[[明尼蘇達州|明尼蘇達州]][[曼卡托_(明尼蘇達州)|曼卡托]][[布盧厄斯河|布盧厄斯河]]河谷的一所公立大學，是[[明尼蘇達州立大學|明尼蘇達州立大學]]的第二位成員和旗艦校<ref name="The Mankato Free Press"/><ref>{{cite web |title=President Davenport responds to Free Press questions|publisher=Mankato Free Press |url=https://www.mnsu.edu/news/read/?id=old-1207229736&paper=topstories |accessdate=July 22, 2015}}</ref><ref>{{cite web |title=Making a monetary mark: economic impact |url=http://www.stcloudstate.edu/news/scsunow/default.asp?storyID=20073 |archive-url=https://web.archive.org/web/20140919191417/http://www.stcloudstate.edu/news/scsunow/default.asp?storyID=20073 |dead-url=yes |archive-date=September 19, 2014 |accessdate=July 22, 2015 }}</ref>，該州第二大公立大學。1858年作為師範學校成立<ref>{{cite book |title=Minnesota Laws First Legislature: An Act to Provide for the Establishment of State Normal Schools |date=August 2, 1858 |publisher=Minnesota State Legislative Reference Library |location=Saint Paul |pages=261–264 |url=https://www.revisor.mn.gov/laws/1858/0/General+Laws/Chapter/79/pdf/ |accessdate=7 August 2018}}</ref>，2019年《[[美国新闻与世界报道|美国新闻与世界报道]]》将其列在“中西部地區大學”中的第111位。<ref name="USNWR">{{cite web |url=https://www.usnews.com/best-colleges/minnesota-mankato-2360 |title=Minnesota State University, Mankato|accessdate=2018-10-23 |publisher=''U.S. News & World Report''}}</ref>
