'''明礬'''（{{lang-en|'''Potassium alum'''}}，[[化學式|化學式]]：'''KAl(SO<sub>4</sub>)<sub>2</sub>·12H<sub>2</sub>O'''），又稱'''鉀鋁礬'''、'''鋁明礬'''、'''鉀礬'''、'''白礬'''、'''生礬'''、'''羽涅'''<ref> {{cite book |author=[[孫星衍|孫星衍]] |coauthors=[[孫馮翼|孫馮翼]] |title=《[[神農本草經|神農本草經]]輯注》 |year=清[[嘉慶|嘉慶]]四年（1799年） |publisher= |location= |isbn= }}</ref>或'''雲母礬'''，可以用作[[水淨化|淨化水質]]、鞣製[[皮革|皮革]]、當作[[發粉|發粉]]或者代替[[硫酸鋁|硫酸鋁]]作[[凡拉明藍|凡拉明藍]][[防染劑|防染劑]]。明礬也可以添加在[[化妝品|化妝品]]中當作除臭劑，或為刮鬍子時造成的小傷口當作止血劑。<ref name="AlumAbout.com">{{cite web | url=http://chemistry.about.com/od/moleculescompounds/f/What-Is-Alum.htm | title=What is Alum? | publisher=About.com | accessdate=2013-04-19 | author=Helmenstine, Anne Marie}}</ref>

==性質==
[[File:Potassium_alum_octahedral_like_crystal.jpg|thumb]]明礬結晶。]]
[[File:Alum.jpg|thumb]]上的明礬晶體]]
明礬結晶為極易溶於水中的無色或白色的八面體晶體，其晶體結構屬異極性[[三方晶系|三方晶系]]。其水溶液可使[[石蕊試紙|石蕊試紙]]變紅、可用作[[收斂劑|收斂劑]]，加入[[碳酸鈉|碳酸鈉]]可得「中性明礬」。明礬受熱時會先溶解於其自身所含的結晶水中，繼續加熱則失去水分而成無水硫酸鋁鉀白色粉末，稱為「燒明礬」或稱「枯礬」（burnt alum）。

==礦物形态和生成==
明礬礦是含有[[硫化物礦物|硫化物礦物]]和含鉀礦物經過[[風化作用|風化作用]]和[[氧化還原反應|氧化還原反應]]自然生成的[[硫酸鹽礦物|硫酸鹽礦物]]，常常存在於礦區堅硬的岩石表面。在過去，明礬都是由生成於含硫的火山沉積物裡面的[[明礬石|明礬石]]提煉出來<ref>Bottomley (2010) p. 35</ref>，所以明礬石應該就是以前鋁和鉀的來源<ref name="WebMineral">{{cite web | url=http://webmineral.com/data/Alum-%28K%29.shtml | title=Alum-(K) Mineral Data | publisher=Mineralogy Database | accessdate=2013-04-19}}</ref><ref name="MinDat">{{cite web | url=http://www.mindat.org/show.php?id=3267 | title=Alum-(K) mineral data and informaton | publisher=MinDat | accessdate=2013-04-19}}</ref>。目前[[義大利|義大利]][[維蘇威火山|維蘇威火山]]、[[澳洲|澳洲]][[昆士蘭省|昆士蘭省]]的Springsure東部、[[美國|美國]][[田納西州|田納西州]]的Alum Cave、美國[[亞利桑那州|亞利桑那州]][[聖克魯斯縣|聖克魯斯縣]]的Alum Gulch及[[菲律賓|菲律賓]]的[[宿霧島|宿霧島]]都有產明礬石。[[纖維鉀明礬|纖維鉀明礬]]（kalinite）是一種與明礬石相似的礦物，它的分子式為KAl(SO<sub>4</sub>)<sub>2</sub>·11(H<sub>2</sub>O)（十一水合硫酸鋁鉀）<ref name="MinDat2">{{cite web | url=http://webmineral.com/data/Kalinite.shtml | title=Kalinite Mineral Data | publisher=MinDat | accessdate=2013-04-19}}</ref>。

==用途==
明礬常被拿來當作[[膨松剂|膨鬆劑]]（[[發粉|發粉]]），用于[[油条|油条]]和[[冬粉|冬粉]]的製作，以及用于造纸、製筆、製[[皮革|皮革]]和泡沫滅火器中。也用作將缺乏親和力的纖維染色的{{le|媒染剂|Mordant}}。<ref>{{Cite web |url=http://www2.cnsh.mlc.edu.tw/~eao/aveplan/99-2/Documents/ForStudents/99-2-s2.pdf |title=青出於藍 |accessdate=2013-05-26 |date=2011-03-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20160310061230/http://www2.cnsh.mlc.edu.tw/~eao/aveplan/99-2/Documents/ForStudents/99-2-s2.pdf |archivedate=2016年3月10日 |df= }}</ref>

在[[中國|中國]]古代，明矾的使用可追溯至[[宋朝|宋朝]]，宋朝对盐、茶、矾等都行使专卖<ref>《[[宋史|宋史]]·刘熙古传》：“建隆二年，受詔制置晋州榷矾，增课八十餘万緡。”</ref>，並建立有榷矾制度<ref>陈振《宋代社会政治论稿》</ref>。[[温州|温州]][[矾山镇|矾山镇]]在當時有“世界矾都”之称。中醫認為明礬可以解毒殺蟲，燥濕止癢，《得配本草》記載：“礬石即白明礬，甘草為之使。畏麻黃，惡牡蠣。酸、鹹，澀，入肝肺二經。”而印度的傳統醫學[[阿育吠陀|阿育吠陀]]中稱為「phitkari」或「saurashtri」的東西亦是指明礬。

[[照相定影劑|照相定影劑]]中常常添加明礬當作固相劑。

水质处理中，明矾用作净水剂。其原理为[[鋁|鋁]][[離子|離子]]Al<sup>3+</sup>在水中會[[水解|水解]]生成[[氫氧化鋁|氫氧化鋁]][[膠體|膠體]]，吸附水中的[[懸浮微粒|懸浮微粒]]。反應式為：

Al<sup>3+</sup> + 3H<sub>2</sub>O ↔ Al(OH)<sub>3</sub> + 3H<sup>+</sup>

{{Fact|明礬有殺菌及消毒的功效。|time=2018-12-16T03:55:17+00:00}}由於可以抑制細菌的生長，因此可以當作天然{{le|除臭劑|Deodorant}}，消除[[體臭|體臭]]；然而，這種用法並沒有止汗的功效。此外，明礬亦可在小傷口上止血，特別是在[[刮鬍子|刮鬍子]]時造成的輕微割傷。

==製備==
可由[[硫酸鋁|硫酸鋁]]與[[硫酸鉀|硫酸鉀]]溶液適量混合後結晶製得，或蒸發水分製得無水硫酸鋁鉀。其反應式為：
:<math>\rm \ Al_2(SO_4)_3 + K_2SO_4 + 24H_2O \rightarrow 2KAl(SO_4)_2\cdot 12H_2O</math>

==安全性==
含鉀明礬的除臭劑中可能會對皮膚有輕微的[[刺激性|刺激性]]。<ref>{{cite journal |author=Gallego H, Lewis EJ, Crutchfield CE 3rd |title=Crystal deodorant dermatitis: irritant matitis to alum-containing deodorant |journal=Cutis |volume=64 |issue=1 |pages=65–6 |pmid=10431678|date=July 1999}}</ref>
