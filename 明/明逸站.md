'''明逸站'''（{{kor|諺=명일역|漢=明逸驛|羅=Myeongil yeok|f=y}}）是一個位於[[首爾特別市|首爾特別市]][[江東區_(首爾)|江東區]]{{link-ko|明逸洞|명일동}}，屬於[[首爾地鐵5號線|首爾地鐵5號線]]的[[地鐵站|地鐵站]]。另外[[京畿道|京畿道]][[河南市|河南市]]市民也會使用[[首都圈電鐵|首都圈電鐵]]。
== 車站結構 ==
2面2線的[[對向式月台|對向式月台]]的地下車站。設有[[月台幕門|月台幕門]]。出口共4個。

== 车站周边 ==
车站位于首尔东部城区，周围有不少中高层[[公寓|住宅大厦]]。
== 历史 ==
*1995年11月15日 - 本站随着5号线通车而落成。
