'''明治'''是[[日本|日本]][[明治天皇|明治天皇]]在位期間使用的[[年號|年號]]，時間為1868年10月23日（[[天保历|旧历]]9月8日）至1912年7月30日。

== 年號更改 ==
*[[慶應|慶應]]4年9月8日（[[1868年|1868年]]10月28日） - [[明治天皇|明治天皇]]即位，年號更改。
*[[1912年|1912年]]（明治45年）7月30日（[[1873年|1873年]]開始實行西洋曆） - [[明治天皇|明治天皇]]去世，[[大正天皇|大正天皇]]即位，改年號為[[大正|大正]]。

== 名稱來源 ==
「明治」一詞出自《[[易經|易經]]·說卦傳》之「[[聖人|聖人]]南面而聽[[天下|天下]]，'''嚮明而治'''」。

== 年表 ==
*明治2年（1869年）：[[戊辰戰爭|戊辰戰爭]]結束、[[版籍奉還|版籍奉還]]
*明治4年（1871年）：[[廢藩置縣|廢藩置縣]]、[[日清修好條規|日清修好條規]]、[[新貨條例|新貨條例]]、[[岩倉使節團|岩倉使節團]]出使外國。
*明治5年（1872年）：[[格里曆|格里曆]]的採用（11月9日的改暦詔書。明治5年12月2日（舊曆）的翌日作為明治6年1月1日）。<!--旧暦の日付は漢数字で。 -->
*明治6年（1873年）：[[徵兵令|徵兵令]]、[[地租改正|地租改正]]、[[明治六年政變|明治六年政變]]（[[西鄉隆盛|西鄉隆盛]]・[[板垣退助|板垣退助]]等下野）
*明治7年（1874年）：[[民撰議院設立建白書|民撰議院設立建白書]]、[[牡丹社事件|台灣出兵]]
*明治8年（1875年）：[[樺太千島交換條約|樺太千島交換條約]]
*明治9年（1876年）：[[江华条约|日朝修好条規]]、[[廢刀令|廢刀令]]
*明治10年（1877年）：[[西南戰爭|西南戰爭]]（- 明治11年）
*明治11年（1878年）：[[地方三新法|地方三新法]]、[[紀尾井坂之變|大久保利通遇刺身亡]]、[[竹橋事件|竹橋事件]]
*明治12年（1879年）：8月31日、皇太子明宮嘉仁親王（[[大正天皇|大正天皇]]）誕生。[[沖繩縣|沖繩縣]]設置（[[琉球處分|琉球處分]]）
*明治13年（1880年）：[[国会期成同盟|国会期成同盟]]結成
*明治14年（1881年）：[[自由民權運動|明治十四年政變]]、[[國會|國會]]開設的詔書發佈。→[[大隈重信|大隈重信]]垮台後、大蔵卿[[松方正義|松方正義]]的[[松方通貨緊縮|松方通貨緊縮]]
*明治15年（1882年）：[[福島事件|福島事件]]、[[壬午事變|壬午事變]]
*明治17年（1884年）：[[秩父事件|秩父事件]]、[[甲申政变|甲申政变]]
*明治18年（1885年）：[[中日天津会议专条|天津條約]]、[[内閣制|内閣制]]開始實施
*明治22年（1889年）：《[[大日本帝國憲法|大日本帝國憲法]]》發布
*明治23年（1890年）：[[第1屆日本眾議院議員總選舉|第1屆日本眾議院議員總選舉]]、第1回[[帝國議會|帝國議會]]召集、『[[教育敕語|教育敕語]]』發布
*明治24年（1891年）：[[大津事件|大津事件]]、{{link-ja|下濑雅允|下瀬雅允}}研制出下濑火药
*明治26年（1893年）：长井长义自[[麻黄碱|麻黄碱]]合成出[[甲基苯丙胺|甲基苯丙胺]]
*明治27年（1894年）：[[甲午農民戰爭|甲午農民戰爭]]→[[日英通商航海條約|日英通商航海條約]]→[[日清戰爭|日清戰爭]]→（明治28年）
*明治28年（1895年）：[[馬關條約|馬關條約]]、[[乙未戰爭|乙未戰爭]]開始統治臺灣
*明治33年（1900年）：[[皇太子|皇太子]][[嘉仁|嘉仁親王]]（大正天皇）與[[貞明皇后|九條節子]]結婚。[[義和團事件|義和團事件]]。
*明治34年（1901年）：4月29日、[[皇太孫|皇太孫]]迪宮裕仁親王（[[昭和天皇|昭和天皇]]）誕生。[[足尾銅山礦毒事件|足尾銅山礦毒事件]]、國營[[八幡製鐵所|八幡製鐵所]]開始作業、[[高峰让吉|高峰让吉]]首次分离出[[肾上腺素|肾上腺素]]
*明治35年（1902年）：[[日英同盟|日英同盟]]
*明治37年（1904年）：[[日俄戰爭|日俄戰爭]]
*明治38年（1905年）：[[樸茨茅斯條約|樸茨茅斯條約]]、[[日比谷縱火事件|日比谷縱火事件]]
*明治39年（1906年）：[[南滿州鐵道|南滿州鐵道]]設立。
*明治40年（1907年）：[[海牙密使事件|海牙密使事件]]
*明治41年（1908年）：[[赤旗事件|赤旗事件]]
*明治42年（1909年）：[[伊藤博文|伊藤博文]]遇刺身亡
*明治43年（1910年）：[[日韓併合|日韓併合]]、[[大逆事件|大逆事件]]、[[铃木梅太郎|铃木梅太郎]]从米糠中提取出了抗脚气病酸，将它命名为[[硫胺|硫胺]]（维生素B1）
*明治44年（1911年）：[[關稅自主權|關稅自主權]]的回復、幕末以来的[[不平等條約|不平等條約]]完全廢除。
*明治45年（1912年）：7月30日、[[明治天皇|明治天皇]][[駕崩|駕崩]]、[[大正|大正]]改元

== 與西曆的對照表 ==
明治n年與[[公元|公元]]g年的關係是n = g - 1867 = g - 1900 + 33。也就是，公元1900年之後，年份最後兩個數字再加上33就是明治的年份。
