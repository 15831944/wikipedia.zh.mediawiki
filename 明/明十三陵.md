'''明十三陵'''，坐落於[[北京市|北京市]][[昌平区|昌平区]][[十三陵镇|十三陵镇]][[天壽山|天壽山]]下40平方公里的小[[盆地|盆地]]，距離北京市区約50公里，是[[中国|中国]][[明朝皇帝|明朝皇帝]]的墓葬建築群。自[[永乐_(明朝)|永樂]]七年五月（1409年）起用，直到安葬[[明思宗|崇禎帝]]後結束，歷時230多年，共葬有13位[[明朝皇帝|明朝皇帝]]，23位明朝皇后、2位明朝太子、30餘名[[妃嬪|妃嬪]]、1位[[太監|太監]]，是全球保存完整皇陵墓葬群之一。2003年7月3日被列为[[世界文化遗产|世界文化遗产]]。

== 历史 ==
[[明朝|明朝]]共有16位皇帝，但[[北京|北京]]的明十三陵只有13位皇帝，未入陵者因由各异。明朝开国皇帝[[朱元璋|朱元璋]]早年建都于[[南京|南京]]，死后葬于[[南京|南京]][[紫金山|紫金山]]的「[[明孝陵|孝陵]]」；继位的[[明太祖|明太祖]][[朱元璋|朱元璋]]长孙[[明惠帝|明惠帝]][[朱允炆|朱允炆]]，在其叔父燕王[[朱棣|朱棣]]（即后来的[[明成祖|明成祖]]）发起「[[靖难之变|靖难之役]]」攻破南京之后下落不明，加上永樂帝不承认其帝位，因此没有帝陵；第七帝[[明代宗|明代宗]][[朱祁钰|朱祁钰]]，在其兄[[明英宗|明英宗]]于[[土木堡之变|土木堡之变]]被[[瓦剌|瓦剌]]所俘之后登上帝位。后来[[明英宗|明英宗]][[复辟|复辟]]，[[朱祁钰|朱祁钰]]速死，[[明英宗|明英宗]]不承认其帝位，捣毁其在天寿山修建的寿陵。还有一说，寿陵被改建为在位僅29天的[[明光宗|明光宗]]的陵墓，改名为[[明庆陵|庆陵]]。[[明憲宗|明憲宗]]以[[亲王|亲王]]身分将他葬于[[北京|北京]]西郊[[玉泉山|金山]]的[[景泰陵|景泰陵]]。

十三陵的主陵是[[朱棣|朱棣]]于1409年至1413年最早兴建的[[明长陵|明长陵]]，他当时经「车驾临视」，钦定山名为「天寿山」，1424年他于北征[[鞑靼_(蒙古)|鞑靼]]途中[[驾崩|驾崩]]，后安葬于此，但[[明长陵|明长陵]]工程直至1427年始全部竣工。期后近20年的兴建，形成长达7公里多的完整建筑群。

13座陵墓中，[[永樂帝|永樂帝]]的[[明长陵|明长陵]]、[[嘉靖帝|嘉靖帝]]的[[明永陵|明永陵]]和[[万历帝|万历帝]]的[[明定陵|明定陵]]，均是生前所建，规模亦最大，其馀陵墓则是死后才动工，大约会用半年修建。[[崇祯帝|崇祯帝]]因为是亡国之君，并没有正式建陵，现时的陵墓是以其妃[[田秀英|田氏]]的墓穴改建。

1957年，[[北京市人民政府|北京市人民政府]]公布十三陵为北京市第一批重点古建文物保护单位。1961年，十三陵被公布为全国重点文物保护单位。1982年，[[中华人民共和国国务院|中华人民共和国国务院]]公布[[八达岭|八达岭]]——十三陵风景区为全国44个[[中國國家級風景名勝區|重点风景名胜保护区]]之一。1995年12月，「明十三陵博物馆」成立。2003年7月3日，明十三陵列入[[联合国|联合国]]《[[世界遗产名录|世界遗产名录]]》。

目前，长陵大殿、定陵地宫可供游客参观，其余陵均未开放。

== 風水佈局 ==
[[File:Ming_Tombs_1875-1908.jpg|thumb]]

十三陵屬於[[太行山脈|太行山脈]]，西通[[居庸關|居庸關]]，北通[[黃花鎮|黃花鎮]]，南向[[昌平|昌平]]州，成為十三陵及京師之北面屏障。太行山起澤州，蜿蜒綿亙北走千百里山脈不斷，至[[居庸關|居庸關]]。明末清初學者[[顧炎武|顧炎武]]曾指：「群山自南來，勢若蛟龍翔；東趾踞盧龍，西脊馳太行；後尻坐黃花（指黃花鎮），前面臨神京；中有萬年宅，名曰康家莊；可容百萬人，豁然開[[明堂|明堂]]。」明代視此為[[風水地|風水地]]，陵區以常綠的[[松|松]][[柏樹|柏樹]]為主。

明十三陵依山而建，沿襲南京孝陵的模式，即除神道共用外，各陵都是前為祭享區，後為墓冢區。陵墓規格相近，各據山頭，陵與陵之間相距500米至8000米不等。除[[思陵|思陵]]偏在西南一隅外，其餘均成扇面形分列於長陵左右。亦有陵仿孝陵之制，稱有一座叫「[[啞巴院|啞巴院]]」的建築，或更設一座[[琉璃|琉璃]][[照壁|照壁]]作為屏幕障。

每座陵墓的陵門設有[[碑|碑]][[亭|亭]]，碑文記載皇帝生前的業績，應由嗣皇帝來撰寫。但自從[[明仁宗|明仁宗]]為其父朱棣寫了一篇3500字的紀功碑文後，再也沒有嗣皇帝續寫。所以現在除了長陵碑外，其餘各陵都成了[[無字碑|無字碑]]。

以規模較大的定陵（萬曆帝墓）為例，其地面建築的總布局，呈前方後圓，象徵「[[天圓地方|天圓地方]]」。地面建築佔地18萬平方米，前有寬闊院落三進，後有高大寶城一座。陵墓有祠祭署、宰牲亭、定陵監、神宮監、神馬房等附屬等建築物300多間，往後是陵園最外面的圍牆－外羅城（圍牆外的圍牆）。

== 耗費 ==

明十三陵雖屬皇家工程，但總體耗費並無全面的精準紀錄，但部分陵寢的紀錄仍可作參考。1584年動工的[[萬曆|萬曆]]帝陵墓定陵，是十三陵中三大陵墓之一，1584年動工，歷時6年，當時共耗用800萬[[銀兩|銀兩]]。

安葬[[明穆宗|明穆宗]]朱載垕及三位皇后的[[明昭陵|昭陵]]，早年施工不周，完工後一年，建築便出現了[[地基|地基]]沉陷的問題。1574年，昭陵神宮監官[[陶金_(明朝)|陶金]]等上奏說：「六月以來，陰雨二日，本陵棱恩門裡外磚石沉陷。」1575年正月，[[明神宗|明神宗]]不得不委派[[工部侍郎|工部左侍郎]][[陳一松|陳一松]]等提督再修昭陵。第一次興建地面建築，共動用[[庫銀|庫銀]]390932兩，仍未計算當中木植、[[白城磚|白城磚]]、[[大石窩|大石窩]]等費用。後來又有戶、兵二部動用110119銀兩，總計501050銀兩。其中；[[營繕司|營繕司]]又用204422銀兩，虞衡司 13145兩，都水司118854兩，屯田司164628兩。

第二次修葺的費用，文獻中沒有明確記載，但據《[[明熹宗實錄|明熹宗實錄]]》記載，前後兩次修建共用銀150餘萬兩。這還不算[[嘉靖|嘉靖]]時營建玄宮的費用。如算上嘉靖年間營陵的費用，其總用度至少在200萬兩以上，當時[[隆慶|隆慶]]年間一年的財政總收入亦只有230餘萬兩。

由於營建昭陵需要龐大的錢糧開支，工部庫銀匱乏到了極點。萬曆二年（1574年）八月修繕涿州橋，工部拿不出銀兩，兵部派不出軍匠，最終輔臣[[張居正|張居正]]請求明神宗懇請太后解囊捐銀，僱工修建。

== 明朝皇帝陵列表 ==
[[明朝|明朝]]历任16位皇帝，当中13人葬於明十三陵，但[[朱元璋|朱元璋]]葬于[[南京|南京]]，此后追谥祖上[[朱百六|朱百六]]、[[朱四九|朱四九]]、[[朱初一|朱初一]]及[[朱世珍|朱世珍]]等。[[明惠宗|明惠宗]][[朱允炆|朱允炆]]失踪，无帝陵，而[[景泰帝|景泰帝]]则葬于[[北京|北京]]西郊金山下。相信是[[明英宗|明英宗]]1457年发动[[夺门之变|夺门之变]][[复辟|复辟]]之后，不承认景泰帝的帝位，以及景泰帝不肯歸还本屬他的皇位，但並无去其年号。详见[[土木堡之变|土木堡之变]]。

== 近代大事紀 ==

* 1929年：1月30日，[[河北省|河北省]]委員會第六十一次會議通過《[[河北省昌平縣明陵保護辦法|河北省昌平縣明陵保護辦法]]》。決定在長陵設護陵警察分駐所
* 1935年：1月，[[北平|北平]]市政府根據[[國民政府|國民政府]]保護古物的命令，派人勘測長陵，估算修繕經費。3月21日開工修葺，6月27日竣工。工程項目有大紅門、神功聖德碑亭、龍鳳門、長陵陵門、陵內碑亭、祾恩門、祾恩殿、內紅門、[[牌樓|牌樓]]門、明樓、皇牆、神帛爐等
* 1937年：思陵等先後被各股[[土匪|土匪]]盜發
* 1937至1948年：獻、景、裕、茂、泰、康、永、昭、慶、德等陵殘壞的祾恩門、祾恩殿，在戰亂中被逐漸拆毀，只餘殘垣斷壁。思陵享殿、碑亭、明樓被[[國民革命軍|國民革命軍]]拆除後，將磚運走修炮樓
* 1949－1950年：隸屬河北省通縣專署的[[昌平縣|昌平縣]]政府，設立護陵委員會駐景陵村
* 1952年：10月，河北省文化局指示，改護陵委員會為十三陵文物保管所
* 1955年：1月，[[北京市人民政府|北京市人民政府]]根據前政務院關於接管十三陵，修繕古建築，植樹造林，闢為公園的指示，派市建築工程局第一建築公司備料施工，修葺長、景、永三陵；9月，明十三陵由河北省昌平縣劃歸[[北京市園林局|北京市園林局]]。
* 1957年：北京市人民政府公布十三陵為北京市第一批[[重點古建文物保護單位|重點古建文物保護單位]]。5月，打開定陵地下玄宮，出土文物近3000件
* 1958年：[[中华人民共和国文化部|文化部]][[国家文物局|文物局]]批准成立定陵博物館。
* 1959年：10月，定陵正式對外開放（隸屬北京市文化局）
* 1961年：[[中华人民共和国国务院|國務院]]公布十三陵為[[第一批全國重點文物保護單位|第一批全國重點文物保護單位]]。
* 1961年至90年代：修復工作進行，其餘陵墓陸續開放
* 1966年8月24日：“[[文化大革命|文化大革命]]”時期，在“打倒地主階級頭子萬曆”的口號聲中，十幾個[[紅衛兵|紅衛兵]]衝進定陵地下博物館，把「[[明神宗|明神宗]]朱翊钧」、「[[孝端显皇后|孝端显皇后]]」王氏及「[[孝靖皇后|孝靖皇后]]」王氏的三具屍骨拉出來擺放在博物館大紅門前廣場上，進行批鬥，最終付之一炬。
* 1987年：4月，修缮昭陵工程开工；1992年竣工。
* 1995年：5月，修繕獻陵工程竣工；7月，神路二期工程竣工；12月「明十三陵博物館」成立。
* 2003年：7月3日，經聯合國教科文組織審議，明十三陵列入《[[世界遺產名錄|世界遺產名錄]]》。
*2001年起，[[昌平区|昌平区]]在已修缮部分陵寝的基础之上，投资上亿元开始了又一轮的大规模修缮，先后修缮了德陵、康陵、庆陵、泰陵、茂陵，到2013年初，最后一座残陵裕陵修缮完工。2013年1月1日起，[[十三陵特区办事处|十三陵特区办事处]]将永陵、康陵、茂陵、泰陵、德陵、庆陵6座陵寝对外开放，但不接待散客。<ref>[http://culture.ifeng.com/whrd/detail_2012_12/29/20655049_0.shtml 十三陵明年新开放6座陵寝 不接待散客遭质疑，凤凰网，2012年12月29日]</ref>
