'''明仁'''（{{jpn|j=明仁|hg=あきひと|rm=Akihito|f=y}} {{IPA-ja|akiçito|}}；{{bd|1933年|12月23日|catIdx=Akihito}}）是[[日本|日本]]第125代[[天皇|天皇]]，為現任之天皇（[[今上天皇|今上天皇]]）。1989年1月7日即位，[[年號|年號]]為[[平成|平成]]（{{jp|j=平成|hg=へいせい|rm=Heisei}}）。其為[[昭和天皇|昭和天皇]]與[[香淳皇后|香淳皇后]]的長子，出生時稱號'''繼宮'''（{{jp|j=継宮|hg=つぐのみや|rm=Tsugu no miya}}），[[御印|御印]]為「榮」（{{jp|j=榮|hg=えい|rm=Ei}}），「榮」是「[[毛泡桐|桐]]」（{{jp|j=桐|hg=きり|rm=Kiri}}）的別稱。

== 人物 ==
1933年（[[昭和|昭和]]八年）明仁于[[東京|東京]][[皇居|皇居]]內的產殿誕生。為[[昭和天皇|昭和天皇]]的長子，[[母親|母親]]是[[香淳皇后|香淳皇后]]，祖父是[[大正天皇|大正天皇]]。1952年([[昭和|昭和]]二十七年)18岁成年后被立为[[皇太子|皇太子]]。1959（[[昭和|昭和]]三十四年）年与[[皇后美智子|正田美智子]]结婚，后育有[[皇太子德仁|浩宫德仁亲王]]（現為皇太子）、[[秋篠宮文仁親王|礼宮文仁親王]]（現為秋篠宮文仁親王）、[[黑田清子|紀宮清子内親王]]（黑田清子）两男一女。

1989年1月7日（昭和64年）[[父親|父親]][[昭和天皇之死|昭和天皇崩御]]（[[驾崩|驾崩]]）享壽87歲，明仁於當日即位为第125代[[天皇|天皇]]，並於隔日改元為[[平成|平成]]；1990年（平成2年）11月12日在東京皇居舉行{{link-ja|天皇即位禮|即位の礼|即位大典}}，這也是天皇的[[登基|登基]]儀式首次離開[[京都|京都]]在東京舉行。

[[第二次世界大戰|第二次世界大戰]]後，依據[[象徵天皇制|象徵天皇制]]，日本天皇已經沒有太多實權；但現年{{age|1933|12|23}}岁的明仁至今仍频繁参与公务和宫中祭祀活动，在一年间要审阅约1,000件的资料并署名盖章、出席约200场各种活动（[[平成|平成]]19年的數字<ref>{{Cite web|author=宮内庁|url=http://www.kunaicho.go.jp/activity/activity/01/activity01.html|title=天皇皇后両陛下のご活動|language=ja|accessdate=6月16日}}</ref>）、参加近20件祭祀活动<ref>{{Cite web|author=宮内庁|url=http://www.kunaicho.go.jp/about/gokomu/kyuchu/saishi/saishi.html|title=宮中祭祀|language=ja}}</ref>，故被很多人认为是一位非常热情和勤勉的天皇。

1992年4月[[中共中央总书记|中共中央总书记]][[江泽民|江泽民]]访问日本，会见了明仁和皇后[[美智子|美智子]]，并邀请天皇伉俪访华。同年10月，明仁与皇后应[[中国国家主席|中国国家主席]][[杨尚昆|杨尚昆]]邀请访问中国，还会见了中共中央总书记[[江泽民|江泽民]]<ref>{{Cite web|author=人民日报|url=http://www.people.com.cn/GB/historic/1024/3545.html|title=1992年10月24日 江泽民会见明仁天皇和皇后|language=zh-hans|accessdate=10月24日}}</ref>，作为第一位访问中国的日本天皇打破了这种局面，也表明当时的[[中日关系|中日关系]]相当友好。

在日本历代天皇中，明仁对[[明治天皇|明治天皇]]、特别是[[昭和天皇|昭和天皇]]感情深厚，[[皇太子|皇太子]]时代也经常提及[[后水尾天皇|后水尾天皇]]。在即位十周年见面会上的演说，明仁说：「作为在战争中长大的人，对之前的[[第二次世界大戰|世界大战]]有强烈的感受，过去的6月23日（[[冲绳县|冲绳]][[慰灵日|慰灵日]]）、8月6日（[[广岛原爆|广岛原爆日]]）、8月9日（[[长崎原爆|长崎原爆日]]）和8月15日（[[第二次世界大戰對日戰爭勝利紀念日|终战日]]）是四个絕不可忘记的日子。」

明仁外祖母[[邦彥王妃俔子|久彌宮妃俔子]]是舊[[薩摩藩|薩摩藩]][[藩主|藩主]][[島津忠義|島津忠義]][[公爵|公爵]]的七女，而[[島津家|島津家]]曾经征服[[琉球|琉球]]；因此，具有島津家血统的明仁对[[冲绳|冲绳]]有很深的感情，也曾几度唱过[[琉歌|琉歌]]，是日本第一位唱琉歌的天皇。

明仁迷恋[[大提琴|大提琴]]，自1970年起师从[[大提琴家|大提琴家]][[清水勝雄|清水勝雄]]；又热衷于[[网球|网球]]运动（这也成了他和美智子认识的契机，即所谓「网球场之恋」），另外也在[[马术|马术]]、[[自行车|自行车]]等运动中表现优秀。明仁同時也是一位[[生物學|生物學]]者，專門研究[[魚類學|魚類學]]。

2009年11月12日，日本政府在[[日本国立剧场|國立劇場]]举行明仁{{link-ja|天皇陛下御在位二十年紀念典禮|天皇陛下御在位二十年記念式典|即位20周年纪念庆典}}，同日稍晚舉行由民間籌辦的{{link-ja|慶祝天皇陛下御即位二十年國民祭典|天皇陛下御即位二十年をお祝いする国民祭典|即位20周年慶祝活動}}。

=== 生前退位 ===

2016年7月，年事已高且健康稍有問題的明仁，初步表達有意於數年之內，[[內禪|內禪]]予[[皇太子|皇太子]][[德仁親王|德仁親王]]，並獲得[[日本皇后列表|皇后]][[美智子|美智子]]、皇太子[[德仁|德仁]]及二皇子[[文仁|文仁]]接受<ref>{{cite web |url = http://news.rthk.hk/rthk/ch/component/k2/1272478-20160713.htm?spTabChangeable=0 |title = 報道：日皇明仁有意生前退位 |publisher=[[香港電台|香港電台]] |date=2016年7月13日}}{{zh-hant}}</ref>。明仁在8月8日於日本全国发表電視演說，正式表达生前退位的意向<ref>{{Cite web |author=东方网 |url = http://news.eastday.com/w/20160808/u1a9606793.html |title=日本天皇发表讲话正式表明“生前退位”意向 明治维新来首次 |language=zh-hans |accessdate=2016-08-08}}</ref>。

2017年[[日本內閣|日本內閣]]依據“關於減輕天皇公務負擔等有識之士會議”之報告<ref>{{cite news|title=日本明仁天皇退位后擬被稱為“上皇”|url=http://japan.people.com.cn/BIG5/n1/2017/0422/c35421-29228977.html|accessdate=2017-04-24|agency=新華社|publisher=人民網|date=2017-04-22}}</ref>，在不修改《[[皇室典範|皇室典範]]》的情形下，提出適用於明仁生前退位的{{link-ja|天皇退位相關之皇室典範特例法|天皇の退位等に関する皇室典範特例法|特別法}}，並獲得[[众议院_(日本)|众议院]]及[[參議院_(日本)|參議院]]之通過<ref>{{cite news|title=日本众议院全体会议通过天皇退位法案|url=http://news.xinhuanet.com/world/2017-06/02/c_1121077136.htm|accessdate=2017-06-02|agency=新华社|date=2017-06-02}}</ref><ref>[https://www.thenewslens.com/article/70429 日本國會通過天皇退位特別法，但僅只一任、女性依然無繼承權]</ref>。同年12月1日的皇室会议商討後，決定明仁於2019年4月30日「[[明仁天皇退位|生前退位]]」（{{link-ja|譲位}}）、德仁親王於5月1日[[即位|即位]]並更改[[年號|年號]]為「[[令和|令和]]」。明仁會成為自1817年[[光格天皇|光格天皇]]以來，相隔202年首位生前退位的天皇<ref>{{cite news|title=天皇陛下譲位は平成３１年４月３０日　改元は５月１日|url=http://www.sankei.com/politics/news/171201/plt1712010015-n1.html|accessdate=2017-12-01|publisher=产经新闻|date=2017-12-01|archiveurl=https://web.archive.org/web/20171201025508/http://www.sankei.com/politics/news/171201/plt1712010015-n1.html|archivedate=2017-12-01|language=ja|deadurl=no}}</ref>。

1933年12月23日早上6時39分，明仁於宫城（現[[皇居|皇居]]）内的御产殿誕生。是[[裕仁天皇|裕仁天皇]]和[[香淳皇后|香淳皇后]]的第五个孩子，而且是第一个男孩，因而受到了国民盛大的祝福。称号「繼宮」和名字「明仁」虽然都是昭和天皇所起，但却是出自明治3年1月3日（1870年2月3日）明治天皇即位之际发布的诏书「…立極垂統、列皇相承、繼之述之… 宣明治教以宣揚惟神之大道也…」。

1936年3月29日，明仁满兩岁后离开双亲，开始是由[[迎賓館赤坂離宮|赤坂离宮]]内的東宮[[仮御所|仮御所]]東宮傅育官抚养。虽然最初的幾個周日可以回到宫中，但过了一个月之后，連周日也开始在东京仮御所中度过<ref>『良子皇太后』河原敏明</ref>。遵从惯例按女孩子的打扮抚养，在[[学習院初等科|学習院初等科]]入学之时，被自作主张剪了娃娃头上的头发后郁闷了好几天，之后竭力抗议道：「从此别再骗我做这种事了！」

1944年，二战战火扩大，明仁被疏散到枥木县[[日光市|日光市]]的[[日光田母沢御用邸記念公園|田母澤御用邸]]，后来又转移到[[奥日光|奥日光]]湯元的南間酒店，并在当地直至战争结束。另外根据[[皇族身位令|皇族身位令]]的规定，皇太子在滿10岁后要在陆海军擔任[[少尉|少尉]]军官，于是明仁被要求加入[[近衛師團|近衛師團]]。虽然軍部也有这种要求，但是根据昭和天皇的意思而没有去任职，故此明仁并没有军队履历。

1946年10月至1950年12月，昭和天皇遵从「学习西洋思想和习惯」的方针，任用[[美国|美国]]著名[[儿童文学|儿童文学]]家[[贵格派|贵格派]]教徒[[伊莉莎白·格雷·維寧|伊莉莎白·格雷·維寧]]为明仁的[[家庭教師|家庭教師]]。明仁受她薰陶，并通过她的介绍跟[[駐日盟軍總司令|駐日盟軍總司令]][[麦克阿瑟|麦克阿瑟]]见面。明仁婉拒了維寧开始时给他起的英文名字（据称为「Jimmy」）。另外明仁在[[学习院|学习院]]学习的时候，因为皮肤较黑，令人联想到[[蚊香|蚊香]]的[[香炉|香炉]]，因而被同学们称为「{{span-tt|チャブ|Chabu}}」。

=== 大天災鼓勵國民 ===
在1995年[[阪神大地震|阪神大地震]]後，明仁曾發書面聲明鼓勵日本民眾。

2011年3月11日發生[[東日本大震災|東日本大震災]]與[[福島第一核電廠事故|福島核危機]]後，明仁於3月16日罕有地發表全國電視演說，鼓勵國民勇敢活下去，及對日本人天災後的表現備受全球各國讚賞感到自豪。這次為歷史上天皇首次於大災難後發表電視演說（主要也是因為[[日本憲法|日本憲法]]限制天皇對外的任何發言都必須經過內閣的同意，所以天皇直接出面發言的機會不多）。在震後第19天，明仁天皇夫婦親訪設立在東京[[武道館|武道館]]的暫時安置所，探望失去家園和親人的民眾，明仁天皇和皇后[[美智子|美智子]]兩人跪坐在災民面前，關切的傾聽災民的心聲並給予溫暖的撫慰，兩人探望災民的時間還超過原定時間1小時。除了武道館，明仁夫婦災後還經常前往[[東北地方|東北地方]]災區探視。

== 遭遇暗殺 ==
1975年7月17日明仁夫婦以日本皇太子和太子妃身份首次訪問沖繩，當時前往沖繩「[[姬百合之塔|姬百合之塔]]」悼念沖繩戰役犧牲者。追求[[琉球獨立|琉球獨立]]的沖繩琉球人川野純治和另一名青年藏身壕溝一週，待明仁夫婦參拜活動當日刺殺明仁夫婦，當時他們向明仁夫婦投擲燃燒彈，燃燒瓶著火後讓明仁夫婦驚嚇不已，燃燒瓶投擲事件之後沒多久明仁就回東京。<ref>{{Cite news|url=https://c1.m.ifeng.com/shareNews?fromType=vampire&forward=1&sfrom=pc&aid=sub_53985877|title=日本最年輕的縣不認同日本皇室，1975年曾刺殺當時的皇太子明仁|author=|date=|work=|accessdate=}}</ref>
