'''明末三案'''，是[[中国|中国]][[明朝|明朝]]末期宫廷中发生的[[梃击案|梃击案]]、[[红丸案|红丸案]]、[[移宫案|移宫案]]的总称。这三起事件引發了滿朝[[士大夫|士大夫]]互相侵軋，故有“'''[[三大案|三大案]]'''”之称。事實上早在三大案之前，[[明神宗|明神宗]][[萬曆|萬曆]]年間還有[[楚宗之亂|二書二楚]]四案，即兩次[[妖書案|妖書案]]，兩次[[楚藩案|楚藩案]]，這些案件表面上的問題都是所謂的「[[國本之爭|國本之爭]]」，但實際上是激烈的[[東林黨爭|東林黨爭]]，直到[[南明|南明]]滅亡為止。

==前奏==
===二書二楚===
{{main|楚宗之亂|妖書案|楚藩案}}

萬曆廿六年（1598年），有人撰《[[憂危竑議|憂危竑議]]》，說[[鄭貴妃|鄭貴妃]]欲以其子[[明朝藩王列表_(福王系)|福王]][[朱常洵|朱常洵]]奪[[儲君|儲君]]之位。神宗懷疑此書是出自吏科[[給事中|給事中]][[戴士衡|戴士衡]]、[[全椒縣|全椒]]知縣[[樊玉衡|樊玉衡]]之手，士衡、玉衡俱貶，後來神宗又認為[[大学士|大学士]][[張位|張位]]是主謀．罷免了[[张位|张位]]，貶謫親近張位的[[礼部侍郎|礼部侍郎]][[刘楚先|刘楚先]]、右[[都御史|都御史]][[徐作|徐作]]、[[國子監祭酒|國子監祭酒]][[刘应秋|刘应秋]]、[[给事中|给事中]][[杨廷兰|杨廷兰]]、[[礼部|礼部]][[主事|主事]][[万建昆|万建昆]]等，是為「[[第一次妖書案|第一次妖書案]]」。

萬曆卅一年（1603年），又出現《[[續憂危竑議|續憂危竑議]]》一書，再次宣告鄭貴妃欲以福王奪儲位，辱及[[天子|天子]]、[[宰相|宰相]]，是為「[[第二次妖書案|第二次妖書案]]」。此書為[[大學士|大學士]][[朱賡|朱賡]]所得，朱賡面奏神宗，神宗怒命[[東廠|東廠]]與[[錦衣衛|錦衣衛]]偵辦。此時[[齐楚浙党|浙黨]][[沈一貫|沈一貫]]乘機構陷[[東林黨|東林黨]][[沈鯉|沈鯉]]、[[郭正域|郭正域]]等人，事態擴大，株連甚多，連高僧[[紫柏真可|紫柏真可]]禪師都因與沈鯉往來受波及，遭到錦衣衛打成重傷，隨即[[涅槃|涅槃]]。由於[[太子|太子]][[朱常洛|朱常洛]]向東廠[[陳萬化|陳萬化]]施壓，不牽連大臣，最後東廠歸罪於[[順天府|順天府]][[諸生|諸生]][[皦生光|皦生光]]所作，皦生光遭到[[凌遲|凌遲]]處死。

除了妖書案，兩次的[[楚藩案|楚藩案]]亦成為黨爭的導火線，與兩次「[[妖書案|妖書案]]」合稱為[[二書二楚|二書二楚]]，又稱[[楚宗之亂|楚宗之亂]]。

==三大案==
===梃击案===
{{main|梃擊案}}

万历四十三年（1615年）五月初四日傍晚，有一男子[[张差|张差]]，手持枣木棍，闯入太子[[朱常洛|朱常洛]]的[[慈庆宫|慈庆宫]]，打伤守门[[太监|太监]]，太子内侍[[韩本用|韩本用]]率眾將之逮捕，张差被捕后声称为[[郑贵妃_(明神宗)|郑贵妃]]手下太监[[庞保|庞保]]、[[刘成|刘成]]二人引进，主使者遙指鄭貴妃本人，此为梃击案<ref>{{cite web |author=阎崇年 |title=《明亡清兴六十年（上）·梃击案(1)》 |url = http://book.sina.com.cn/nzt/his/mingwangqingxing/48.shtml |website = 新浪网 |accessdate=2014-01-17 |language = zh }}</ref><ref>{{cite web |author=阎崇年|title=《明亡清兴六十年（上）·梃击案(2)》 |url = http://book.sina.com.cn/nzt/his/mingwangqingxing/49.shtml |website = 新浪网 |accessdate=2014-01-17 |language = zh }}</ref><ref>{{cite web |title=梃击案 |url = http://www.gs5000.com/today/14289.html |website = 中国历史故事网 |accessdate=2014-01-17 |language = zh |deadurl=yes |archiveurl = https://web.archive.org/web/20130815005717/http://gs5000.com/today/14289.html|archivedate=2013-08-15}}</ref> 。

===红丸案===
{{main|紅丸案}}

萬曆四十八年、[[泰昌|泰昌]]元年（1620年）八月，万历帝崩，朱常洛即位，是为[[明光宗|明光宗]]，鄭貴妃為了結好光宗，立即獻上了美女數名。光宗大喜，夜夜春宵，縱慾過度，因而生病，[[司礼监|司礼监]]秉笔太监[[崔文昇|崔文昇]]進上[[泻药|泻药]]，使皇帝病情加剧。[[鸿胪寺|鸿胪寺]]丞[[李可灼|李可灼]]进[[红丸|红丸]]，自称仙方，光宗服药后死去，时人怀疑为郑贵妃下毒，是为红丸案<ref>{{cite web|author=阎崇年|title=《明亡清兴六十年（上）·红丸案》 |url = http://book.sina.com.cn/nzt/his/mingwangqingxing/50.shtml|publisher=新浪网|accessdate=2014-01-17 |language = zh }}</ref> 。

===移宫案===
{{main|移宮案}}

泰昌元年（1620年），[[明熹宗|天啟帝]]即位，抚养皇帝的[[李康妃|李选侍]]和[[魏忠贤|魏忠贤]]，利用熹宗年幼強居[[乾清宫|乾清宮]]，把持朝政，東林黨人[[杨涟|楊漣]]、[[左光斗|左光-{斗}-]]等反对，不让李选侍与皇帝同住，迫使她移居他处，是为移宫案<ref>{{cite web |author=阎崇年 |title=《明亡清兴六十年（上）·移宫案(1)》|url=http://book.sina.com.cn/nzt/his/mingwangqingxing/51.shtml|publisher=新浪网 |accessdate=2014-01-17 |language = zh }}</ref> <ref>{{cite web |author=阎崇年 |title=《明亡清兴六十年（上）·移宫案(2)》|url=http://book.sina.com.cn/nzt/his/mingwangqingxing/52.shtml| website = 新浪网 |language = zh }}</ref> 。
