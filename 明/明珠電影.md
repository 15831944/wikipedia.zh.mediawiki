'''明珠電影'''（{{lang-en|PEARL MOVIE}}），前稱'''明珠930'''（{{lang|en|Studio 930}}）<ref>也指週一至五同時段的外語電視劇集</ref>，又称'''周末大电影'''（Weekend Blockbuster）、'''明珠大电影'''，是香港[[無綫電視|無綫電視]]（TVB）[[明珠台|明珠台]]所選播的[[電影|電影]]節目，部份電影設有[[麗音|麗音]]雙語廣播。

== 選播電影 ==
明珠电影當中以英語電影佔最多，皆標以[[香港電視節目分類制度|PG級別]]，但儿童不宜的内容会被删剪。但亦有例外，有些电影被要求全片不删剪，如《[[泰坦尼克号_(1997年电影)|铁达尼号]]》的女主角[[凱特·溫斯蕾|凱特·溫斯蕾]][[裸体|裸体]]片段就被保留<ref>{{Cite web|url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20070210/6802195|title=TVB重播《鐵達尼》 拒刪露點戲|work=[[香港蘋果日報|香港蘋果日報]]|author=林妃|date=2007-02-10|accessdate=2018-12-31|quote=無綫02年12月於明珠台播出《鐵達尼號》（Titanic），由於未有刪剪女主角琦溫絲莉（Kate Winslet）於片中的露點鏡頭而接獲八宗投訴，最後廣管局裁定投訴不成立。至04年，明珠台重播《鐵》片，繼續接獲22宗市民投訴，最後廣管局依然裁定投訴不成立。}}</ref>，而《[[魔戒|魔戒]]》、《[[魔盗王|魔盗王]]》等超長時間的電影也是足本。

目前明珠电影最高收视记录为2002年12月15日播映《[[泰坦尼克号_(1997年电影)|铁达尼号]]》，平均收视21点、约183万观众收看，最高收视24点、约160万观众收看<ref>{{Cite web|url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20021217/3007250|title=《鐵達尼號》破明珠收視紀錄|work=香港蘋果日報|author=|date=2002-12-17|accessdate=2018-01-08|quote=昨日兩台公布收視。明珠台本周日播映的電影《鐵達尼號》，平均收視二十一點、即一百三十八萬觀眾收看，最高收視達二十四點、即約有一百六十萬觀眾，成績理想。}}</ref>。

== 雙語廣播 ==
往時明珠台的电影多数是以[[立體聲|双声道]]播出（粵語配音及原聲），以吸引香港觀眾。2015年9月起，明珠台刪減粵語配音，包括首播時已配音的電影；但兒童電影及部分非TVB製作之粵語配音的電影仍會繼續提供粵語配音選擇。TVB指，过去五年提供的配音节目急增近六成，在庞大工作量下，为平衡不同节目需要，确保配音节目质素维持最佳水平，TVB必须作出取舍；明珠台英语节目方面，碍于中西[[文化差异|文化差异]]，粤语配音成效未如理想，没有達到善用资源，故有需要作出相當的调整。

== 播出時間 ==
明珠台的電影播出時間原是每星期五六日共三日。2015年3月6日，明珠台取消於星期五晚上播出電影，改於《[[無綫電視晚間新聞|晚間新聞]]》後播出「週五星級影院」，直到2015年9月因要播出[[香港賽馬會賽馬節目|香港賽馬會賽馬節目]]而終止播映。惟獨於[[香港節日與公眾假期|公眾假期]]，仍會於上午、下午或者晚上9時30分播出電影。

現時明珠台一般會於星期六及星期日晚上播出電影，每個月首星期日均推出一部新片。官方播映時間爲9點30分，惟實際加上廣告及《[[新聞提要|新聞提要]]》後要到約9點35分之後才開始，超過深夜12点則先播新闻、再接駁上餘下部分，往往要播到深夜1、2點。2005年之后，有時超过3小時以上（计上广告时间）的電影（如《[[鐵達尼號_(1997年電影)|鐵達尼號]]》、《[[魔戒|魔戒]]》、《[[奇幻逆緣|奇幻逆緣]]》）就会提前1小時於8點30分播出，并将新闻延至电影播完后，以求在深夜1点前结束。其它情况除了有特殊原因，如轉播[[奧運|奧運]]、[[网球|网球]]公开賽，才会暫停。

== 重播 ==
某些电影（不计华语电影）的重播频率很高，如《[[哈利波特|哈利波特]]》、《[[惊天核网|惊天核网]]》、《[[寶貝智多星|寶貝智多星]]》、《[[职业特工队|职业特工队]]》、《[[盗海豪情|盗海豪情]]》等的大片就经常重播，其餘的一般是每一两年便重播一次。特別是《鐵達尼號》经常是逢[[情人節|情人節]]均會重播一次，《[[盗海豪情|盗海豪情]]》、《[[惊天核网|惊天核网]]》曾经最多一年重播三次。然而某些电影也会被淘汰，例如2000年－2005年经常重播的《盗墓迷城》于2005年后没有再重播过。

有些大片如《[[蝙蝠俠|蝙蝠俠]]》、《[[蜘蛛俠|蜘蛛俠]]》、《鐵達尼號》、《[[魔戒三部曲|魔戒三部曲]]》、《[[職業特工隊|職業特工隊]]》，在[[翡翠台|翡翠台]]節目檔期較為寬鬆時重播。相反的是，[[香港數碼地面電視廣播|数码广播]]后，某些新购的电影先行于[[高清翡翠台|高清翡翠台]]與翡翠台播出，而后再于明珠台重播，但也是个别少数的例子。
