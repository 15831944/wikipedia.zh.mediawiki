'''明清官话'''<ref>葉寶奎（2001），《明清官話音系》，福建：廈門大學出版社。</REF><REF>丁鋒（2008），《日漢琉漢對音與明清官話音研究》，中華書局。</ref>，明清时期官僚通用语，因在官场通用而得名。[[明朝|明朝]]繼承[[元朝|元]]，即“壹以中原雅音为正”，但參照了下江官話，形成[[南京官话|南京官话]]为国家标准语音。到了清朝中期，[[北京官话|北京官话]]形成，影响逐渐扩大。清末卢戆章创制[[中國切音新字|中國切音新字]]，仍然倡议以南京话为“各省之正音”，可见南京官话在清末仍然有很大的影响。明朝官話與清代官話不同，明朝官話貼近舊韻，而清代官話則變化較大。到了[[民国时期|民国]]，北京官话被确定为[[中華民國國語|国语]]，以后在中国大陆改称[[現代標準漢語|普通话]]。

== 名稱：與「官话方言」的区别 ==
作为历史上标准语的官话（即本条目所指的“官话”），与作为现代汉语分支的[[官话|官话]]，是两个范畴的概念：前者是（古代）汉语的一个语种，它在同一个时期内的语言学特征（[[语音|语音]]、[[语法|语法]]、[[词汇|词汇]]等）是完全相同的，每个词都有其固定的发音；后者是(现代)汉语的包含众多语种的一个分支（基于立场的不同可称这个分支为方言或独立语言），它因地域的区别有着许多语言学特征不尽相同甚至差别很大的不同语种，同一个词，在不同的地域往往不同。

== 官話的演變史 ==
{{see also|官话#历史|l1=官話歷史|南京話#历史|l2=南京話歷史|北京话#歷史|l3=北京話歷史}}
中國的官方語言早期以[[中原|中原]][[雅音|雅音]]為正音，即[[上古汉语|上古汉语]]时期（主要是[[周朝|周朝]]至[[汉朝|汉朝]]）流行于[[中原|中原]]的全国通用的语言。

[[五胡亂華|五胡亂華]]、[[衣冠南渡|衣冠南渡]]以後，[[漢族|漢族]]中央政權第一次轉移到南方，使得中原雅音南移，漢語正音因此分為南北兩支。但因[[东晋|东晋]]遷都[[南京|南京]]，中原雅音為基礎正音，由於當時中原漢民帶來雅言並流行於上層社會和知識階層，稱為「士音」，雅言和吳語逐漸融合成為[[南京官話|南京官話]]，又稱金陵雅音。

[[南北朝|南北朝]]時期，[[南朝|南朝]]治下的[[江東|江東]]崛然而起，趕超中原地區，成為漢民族文化的正統地帶。因此以中原雅音為基礎的[[南京官話|南京官話]]以「正統身份」成為中國官方語言的正音，日本來華所學亦是[[吳音|吳音]]。

隋朝統一後以金陵雅音和洛陽雅音為基礎正音，進入[[中古汉语|中古汉语]]时期後與（主要是[[唐|唐]]、[[宋|宋]]）流行于[[中原|中原]]（时洛、汴一带）的中原雅音融合，以《切韻》《廣韻》為標準的唐宋官方語音成為全国通用的正音。

[[蒙古|蒙古]]征服[[南宋|南宋]]，位於北方的[[北京市|燕京]]首次成為全國的政治中心，[[黃河|黃河]]流域逐漸衰落。元朝依据的仍然是唐宋以来占据主导地位的中原音（[[汴京|汴]][[洛阳|洛]]音）做為共通語的正音，亦有受燕京音影響，參看《[[中原音韻|中原音韻]]》<ref>周德清 《中原音韻》1324年初版</ref><ref name= HAS>{{Cite book | author = 胡安順 | title = 音韻學通論 | location = 北京 | publisher = 中華書局 | date = 2003 | ISBN = 9787101032970}}</ref>。

[[明朝|明朝]]滅[[元|元]]定都南京，「一以中原雅音為正」，建立南京官話<sup>[[#注释|2]]</sup>，南京官話與現在南京話是否相同，學界仍有爭議<ref>麦耘，朱晓农〈南京方言不是明代官话的基礎〉，收入《语言科学》，徐州師範大學編，2012年4期</ref>，[[王力|王力]]等學者主張南京官話主要仍是受北方漢語影響。

明永樂年間遷都北京，從南京移民之北京四十萬人超過了北京人口，[[南京官话|南京官话]]受以北京語音和北方話的腔調，逐漸產生含有北京聲調的官話，作為國語正音的官話又再次分為南北兩支，此時的北京官話仍不成熟，通行範圍主要在北京及周圍地區的民間，地位較低，官場仍以[[南京官話|南京官話]]為國語正音。

[[明|明]][[清|清]]兩代來華的西方傳教士所流行的中國話，基本上是以[[南京官話|南京官話]]為標準的南方官話，周邊國家也是如此，比如日本從江戶時代到明治時期所教的中國語都是[[南京官話|南京官話]]。

清初仍使用[[南京官话|南京官话]]的腔調，經過時間，北方腔調官話的影響逐漸擴大。清中葉以後，[[北方官话|北方官话]]逐漸取代南京官話取得主流的地位。宣统元年（1909年）清政府正式設立「國語編審委員會」，此即清末的國語<ref>{{cite web |title=舌尖上的粵語 |author=鄭惠芳 |publisher=聯合早報 |url=http://lkyspp.nus.edu.sg/ips/wp-content/uploads/sites/2/2013/04/pa_WF_LHZB_She-Jian-Shang-De-Yue-Yu_020613.pdf |language=zh-sg |date=2013-06-02 |accessdate=2015-08-13 |archiveurl=https://web.archive.org/web/20150813035551/http://lkyspp.nus.edu.sg/ips/wp-content/uploads/sites/2/2013/04/pa_WF_LHZB_She-Jian-Shang-De-Yue-Yu_020613.pdf |archivedate=2015年8月13日 |deadurl=yes |df= }}</ref>。

[[民國|民國]]初年擬定[[現代標準漢語|國語]]以及[[京國之爭|京國之爭]]以後，以北京官話為標準的北方官話取代[[老國音|老國音]]，故今日的國語（普通話）以北京話為基礎。

概而言之，南方官話與北方官話都是由「[[西晉|西晉]]以前的中原官話」演化而來，兩者有差異，也有聯繫。比如明朝遷都北京時，北京城中南京移民超北京本地人口，此後的北京話也大量融入了南京話。相比較而言，以北京話為標準的北方官話語音發展變化比較快，以南京話為標準的南方官話則融入了一些吳語方言。
*南方官話
:作為中國官方語言的[[南方官話|南方官話]]，不同于廣義上的南方地區的官話，是以[[江淮官話|江淮官話]]為基礎方言、以[[南京音|南京音]]為標準，又稱[[南京話|南京官話]]。
*北方官話
:作為中國官方語言的[[北方官話|北方官話]]，不同于廣義上的北方地區的官話，是以[[華北官話|華北官話]]為基礎方言、以[[北京音|北京音]]為標準，又稱[[北京官話|北京官話]]。

== 音韵 ==
[[明朝官話|明朝官話]]大致屬於《切韵》體系，以《[[正韻|正韻]]》為代表作；而清代官话則脱胎于元代《中原音韵》为代表的音系，相对于以《切韵》为代表的中古音系，它有如下的变化：

* [[声母|声母]]：共21个。[[全浊声母|全浊声母]]消失，平声变成送气清音，仄声变成不送气清音。

* [[韵母|韵母]]：见系开口二等腭化韵与相应三、四等韵混同；閉口韻[-m]并入[-n]；入聲派入三聲之重，[-p、-t、-k]合为[-{{IPA|ʔ}}]，入声韵改配阴声韵。

* [[声调|声调]]：平声分为阴平和阳平，全浊上声字并入去声，入声逐渐消失，相应的字派入其他三声之中。

而到了明中後期，北京官话（即当时的北京音）的音系已与现代北京话相差无几，由萬曆年北京人徐孝所作之《重訂司馬溫公等韻圖經》所見，与明初的官话相比，产生了[[韵尾|韵尾]]消失、入聲消失，[[尖团合流|尖团合流]]等重大的变化。

==明清官话對後世的影響==
尽管明清官府并没有对官话正式命名（「官话」是社会上约定俗成的称呼），也没有将官话进行推广，但它对中国普通百姓语音的影响是巨大的。

明清时期各地汉语尽管大都与官话差别很大，但都或多或少的受到官话的影响。越是官话使用频率高的地方，这种影响就越剧烈。一个典型的例子是[[江西|江西]][[景德镇|景德镇]]，长期作为官方瓷器供应地，官话在城区盛行，导致当地的方言往官话靠拢，最终出现只有三个声调这种[[赣语|赣语]]乃至整个“东南方言”极其罕见的现象。{{ref|3tones}}

官话的另一个影响是产生了带方音的官话（下文简称为“方音官话”）。古代非官话所在地的人，要学习官话，往往会受[[母语|母语]]的极大影响，从而说出母语特征极浓的官话来，如[[藍青官話|藍青官話]]。方音官话与（标准）官话之间，只是[[口音|口音]]之间的区别，而非[[方言|方言]]之间的区别。

今日的[[江淮官话|江淮官话]]绝大多数方言及[[天津话|天津话]]和[[苏北|苏北]]、[[皖北|皖北]]一带的[[中原官话|中原官话]]是明代[[南京官话|南京官话]]的直接“后代”。1913年決議之[[老國音|老國音]]亦是兼通南北。

== 明清官話在现代的遗留 ==
官话在现代仍有少量保留，不過不是在“活的”語言裏{{来源请求|time=2014-12-04}}，而是在[[戏剧|戏剧]]中，而且只保留了某些特定时期的官话。如[[京剧|京剧]]的语音，仍保留了清末以前官话中的区分尖团的特点；[[越剧|越剧]]的语音则继承了绍兴的方音官话的音系。
