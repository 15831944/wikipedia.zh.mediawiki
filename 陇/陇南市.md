'''陇南市'''是[[中华人民共和国|中华人民共和国]][[甘肃省|甘肃省]]下辖的[[地级市|地级市]]。位于甘肃省东南部，[[岷山山脉|岷山山脉]]与[[黄土高原|黄土高原]]交汇处，东邻[[陕西|陕西]]，南接[[四川|四川]]，有“陇上江南”之称。

== 历史 ==
[[秦代|秦代]]置武都道。[[汉武帝|汉武帝]][[元鼎|元鼎]]六年（前111年）置[[武都郡|武都郡]]，属[[凉州刺史部|凉州刺史部]]。汉末，[[曹操|曹操]]分[[司州|司州]]、凉州置雍州，武都郡改属[[雍州|雍州]]。

[[三国|三国]]时期，武都郡是魏、蜀两国的边界。[[建安|建安]]二十四年（219年），[[刘备|刘备]]攻取了[[曹操|曹操]]军的[[汉中郡|汉中郡]]，阻断了武都与雍州的联系，于是曹操弃武都郡，迁治至[[右扶风|右扶风]]小槐里。此后，魏、蜀两国于武都展开多次大战，最著名的是蜀相[[诸葛亮|诸葛亮]]“[[六出祁山|六出祁山]]”伐魏，并于229年攻取武都、阴平二郡，自此武都郡为蜀所据，直至[[蜀国|蜀国]]被曹魏所灭。 

[[东晋|东晋]]、[[南北朝|南北朝]]时期，陇南境内先后建立[[仇池|仇池]]、[[宕昌国|宕昌]]、武都、[[武兴国|武兴]]、[[阴平国|阴平]]5个[[胡人|胡人]]政权，称为“陇南五国”。
*296年，[[氐族|氐族]]首领[[杨茂搜|杨茂搜]]建立仇池国，至371年为[[前秦|前秦]][[苻坚|苻坚]]所灭。族人[[杨定|杨定]]于历城再建仇池国，至442年被[[南朝|南朝]][[刘宋|刘宋]]所灭。
*424年，[[北魏|北魏]][[拓跋焘|拓跋焘]]封[[羌人|羌人]][[梁弥忽|梁弥忽]]为宕昌王，都城在今宕昌县城。566年，宕昌国被[[北周|北周]]所灭。
*443年，仇池国灭亡后，氐人[[杨文德|杨文德]]建立武都国。477年，武都国为[[北魏|北魏]]所灭。
*477年，武都王[[杨文度|杨文度]]被北魏军所杀后，[[南朝|南朝]][[刘宋|刘宋]]政权封[[杨文弘|杨文弘]]为武都王，退守武兴（今陕西[[略阳|略阳]]），后改武都王为武兴王，建立武兴国。552年，武兴国为[[西魏|西魏]]所灭。
*477年，北魏灭武都国时，[[杨文度|杨文度]]之叔[[杨广香|杨广香]]助魏军杀杨文度，受封为阴平公，后[[南齐|南齐]]承认杨广香为[[阴平国|阴平国]]之主，都阴平（今文县）。580年，阴平国为[[北周|北周]]所灭。

[[隋|隋]][[唐|唐]]时期，陇南地区政治经济相对穏定，唐代于陇南置文、武、成、迭、宕、岷等州。[[宝应|宝应]]元年（762年），[[吐蕃|吐蕃]]攻占陇南。[[咸通|咸通]]年间，唐收复[[武州|武州]]，更名阶州，咸通七年（866年）收复[[成州|成州]]。

[[北宋|北宋]][[熙宁|熙宁]]六年（1073年），大将[[王韶|王韶]]打败吐蕃，收复陇南，吐蕃据有陇南前后共311年。宋末陇南为宋、金两国边境，战争频繁。[[南宋|南宋]][[绍兴|绍兴]]三十年（1160年），金大举攻宋，破凤州、[[大散关|大散关]]，朝廷命大将[[吴璘|吴璘]]据守陇南。吴璘之子[[吴挺|吴挺]]与[[西夏|西夏]]缔盟，共同攻金。[[开禧|开禧]]二年（1206年），吴挺之子[[吴曦|吴曦]]叛宋，以阶、成、文、西和4州建立受金朝支持的政权，后宋军将士杀吴曦，4州归宋。

[[端平|端平]]三年（1236年）蒙古军占领陇南。[[元代|元代]]于礼县置礼店蒙古军元帅府，后更名为礼店文州番汉军民元帅府，在河州设吐蕃宣慰司。[[明|明]][[清|清]]时期，陇南经历明末[[李自成起义|李自成起义]]、[[白莲教之乱|白莲教之乱]]、[[西北回民起义|西北回民起义]]、[[太平天国|太平天国]]等事件。

1949年7月，成立武都分区行政督察专员公署，驻武都县，属[[陕甘宁边区|陕甘宁边区]]甘肃行政区。1950年5月，岷县划归[[武都专区|武都专区]]。1956年，礼县、西和县、成县划归[[天水专区|天水专区]]。1958年4月，撤销武都专区，辖县划归天水专区。1961年11月，恢复[[武都地区|武都地区]]。1963年10月，撤销[[临洮专区|临洮专区]]，岷县划归[[武都专区|武都专区]]。1985年5月，武都地区更名为[[陇南地区|陇南地区]]，岷县划归[[定西地区|定西地区]]。

2004年1月，撤销陇南地区，设立地级陇南市，原[[武都县|武都县]]改为武都区。

== 地理 ==
陇南市位于甘肃省南部，东经104°01′19″至106°35′20″，北纬32°35′45″至34°32′00″。北与[[天水市|天水市]]、[[武山县|武山县]]、[[甘谷县|甘谷县]]接壤；南与[[四川省|四川省]][[青川|青川]]、[[平武|平武]]、[[九寨沟县|九寨沟县]]毗连；西依[[迭部|迭部]]、[[舟曲|舟曲]]和[[岷县|岷县]]；东接[[陕西省|陕西省]][[宁强|宁强]]、[[略阳|略阳]]、[[勉县|勉县]]、[[凤县|凤县]]。地势大致上呈西北高，东南低，平均海拔1000米。全市东西长约237公里，南北宽约230.5公里。
 
==气候==
全市分为北亚热带、暖温带、中温带三大类。
*北亚热带：[[白龙江|白龙江]]、[[白水江|白水江]]、[[嘉陵江|嘉陵江]]河谷浅山地区。年平均气温12－14°C，降水量在600毫米左右。
*暖温带：中部、东部及南部的广大地区，海拔在1100－2000米之间，降雨量500－800毫米之间。
*中温带：北部和西部地区，海拔在2000米以上，年最低气温在-20°C以下。

== 汶川大地震 ==
2008年5月，四川發生[[汶川大地震|大地震]]，最終引致隴南發生強烈災難，造成330人死亡，5380人受傷。
