'''锗酸镤(IV)'''是一种[[无机化合物|无机化合物]]，化学式为PaGeO{{sub|4}}。

== 制备 ==
将[[二氧化镤|二氧化镤]]和[[二氧化锗|二氧化锗]]在真空烧结得到产物。
::<math>\mathsf{ PaO_2 + GeO_2 \ \xrightarrow{1100-1200^oC}\ PaGeO_4 }</math>

== 物理性质 ==
锗酸镤(IV)有以下两种结构：
* α-PaGeO{{sub|4}}，属于[[正交晶系|正交晶系]]，晶胞参数''a'' = 0,5106 nm, ''c'' = 1,138 nm, Z = 4，具有[[钨酸钙|CaWO{{sub]]结构；
* β-PaGeO{{sub|4}}，属于[[正交晶系|正交晶系]]，晶胞参数为''a'' = 0,7157 nm, ''c'' = 0,6509 nm, Z = 4，具有[[硅酸锆|ZrSiO{{sub]]结构。<ref>{{книга
