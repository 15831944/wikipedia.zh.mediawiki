'''拓北站'''（{{jpn|j=拓北駅|hg=たくほくえき|rm=Takuhoku eki|f=y}}）是位於[[北海道|北海道]][[札幌市|札幌市]][[北區_(札幌市)|北區]]拓北6條3丁目9番1號，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[札沼線|札沼線]]（學園都市線）的[[鐵路車站|鐵路車站]]。車站編號為'''G09'''。電報略號為'''{{lang|ja|タク}}'''。

== 歷史 ==
車站周圍主要是由泥碳地組成，難以開發農業的區域。北海道工人住宅生活協同組合（住宅生協）於1963年開始於當地發展「向日葵團地」。在聽取該團地居民的要求設置新車站後，決定設置本車站。
* 1967年（昭和42年）12月15日 - [[日本國有鐵道|日本國有鐵道]]（國鐵）[[札沼線|札沼線]]'''東篠路站'''開始營業。為處理旅客業務的業務委託車站，當時為[[側式月台|側式月台]]1面1線的車站。
* 1987年（昭和62年）4月1日 - 由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理。
* 1991年（平成3年）3月16日 - 札沼線的暱稱設定為「'''學園都市線'''」<ref name=school>{{cite web|url=http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |language=ja |title=札沼線（学園都市線）の電化について |publisher=JR北海道 | date=2009-09-09 |accessdate=2017-08-22 |archive-url=https://web.archive.org/web/20091006234745/http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |archive-date=2009-10-06 }}</ref><ref name=school2>{{cite web |url=http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |language=ja |title=札沼線（学園都市線）の電化開業時期について |publisher=JR北海道 |date=2011-10-13 |accessdate=2017-08-22 |archive-url=https://web.archive.org/web/20111205160350/http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |archive-date=2011-12-05 |deadurl=yes }}</ref>。
* 1995年（平成7年）3月16日 - 改稱為'''拓北站'''。同時車站[[複線鐵路|複線化]]，增設月台變成2面2線的車站，並設置[[行人天橋|跨線橋]]。站房同時改建。
* 1997年（平成9年）3月22日 - 札沼線（學園都市線）[[篠路車站|篠路車站]]至[[愛之里教育大車站|愛之里教育大車站]]（包含本車站）路段全面複線化。
* 2000年（平成12年） - 札沼線（學園都市線）[[桑園車站|桑園車站]]至[[石狩月形車站|石狩月形車站]]路段引入自動進路制御裝置（PRC）。
* 2006年（平成18年）1月 - 引入自動列車導向裝置。並設置按方向的導向用液晶顯示屏。
* 2007年（平成19年）10月1日 - 設定車站編號（'''G09'''）<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |language=ja |title=駅番号表示（駅ナンバリング）を実施します |publisher=JR北海道 |date=2007-09-12 |accessdate=2017-08-22 |archive-url=https://web.archive.org/web/20070930015220/http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |archive-date=2007-09-30 |deadurl=yes }}</ref>。
* 2008年（平成20年）10月25日 - 開始使用[[儲值車票|智能車票]]「[[Kitaca|Kitaca]]」<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |language=ja |title=Kitacaサービス開始日決定について |publisher=JR北海道 |date=2008-09-10 |accessdate=2017-08-22 |archive-url=https://web.archive.org/web/20080913103748/http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |archive-date=2008-09-13 |deadurl=yes }}</ref>。
* 2012年（平成24年）6月1日 - 札沼線桑園站至[[北海道醫療大學車站|北海道醫療大學車站]]正式[[電氣化鐵路|電氣化]]（[[交流電|交流電]]20,000[[伏特|V]]、50[[赫兹|Hz]]）<ref>{{cite web|url=http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |language=ja |title=学園都市線電化開業に伴う電車の投入（第一次）について |publisher=JR北海道 | date=2012-03-14 |accessdate=2017-08-22 |archive-url=https://web.archive.org/web/20120925203437/http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |archive-date=2012-09-25 }}</ref><ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |language=ja |title=平成24年10月ダイヤ改正について |publisher=JR北海道 |date=2012-08-03 |accessdate=2017-08-22 |archive-url=https://web.archive.org/web/20120925203427/http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |archive-date=2012-09-25 |deadurl=yes }}</ref>。
* 2017年（平成29年）
** 3月15日 - 南方入口進行搬遷。
** 3月30日 - 設置[[升降機|升降機]]。

== 車站構造 ==
現時為[[側式月台|側式月台]]2面2線的地面車站。站房位於車站北側。

由[[桑園車站|桑園車站]]管理的業務委託車站（北海道JR Servicenet），北方出口設有[[綠窗口|綠窗口]]（營業時間5時30分至0時30分）、[[自動售票機|自動售票機]]及[[驗票閘門|自動閘機]]（對應[[Kitaca|Kitaca]]及磁性票）。而南方入口為無人狀態，設有簡易自動售票機及簡易自動閘機（對應Kitaca及磁性票）。

== 車站周邊 ==
車站前並沒有形成商店街。周圍主要是住宅區，約500米外出現農田及牧場。住宅集中在與鄰近車站的連接的東側及西南側。
* 札幌拓北郵局
* [[北陸銀行|北陸銀行]]麻生分店東篠路出張所
* 北海道札幌英藍高等學校
* 札幌市立篠路中學
* 札幌市立拓北小學
* 札幌三育小學
* 北海道中央巴士「向日葵團地」站牌
