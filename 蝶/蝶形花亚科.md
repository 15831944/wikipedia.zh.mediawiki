'''蝶形花亞科'''（[[学名|学名]]：{{lang|la|Faboideae}}）是[[豆科|豆科]]植物六個亞科之一，476[[属|属]]，13,855[[种|种]]，广泛分布在全世界 <ref name="APW">[http://www.mobot.org/MOBOT/Research/APweb Angiosperm Phylogeny Website]，Stevens, P. F. (2001 onwards). Angiosperm Phylogeny Website. Version 9, June 2008 [and more or less continuously updated since].</ref>。中国有22族，129属，一千余种，其中包括了上百種的引進栽培種類 <ref name="FOC" />。

== 學名 ==

蝶形花亞科學名為 {{lang|la|Faboideae}}，來自[[蚕豆|蚕豆]]學名 '' Vicia faba'' ，''faba'' 之拉丁原意即"豆"，為合於國際植物命名法規的學名。此外異名{{lang|la|Papilionoideae}}也常見於學術文獻中<ref name="Vienna code">[http://ibot.sav.sk/icbn/frameset/0022Ch3Sec2a018.htm#18.5. 國際命名法規(Vienna code)]，法規18.5、法規18.6及法規19.7。</ref>。

== 形態 ==

蝶形花亞科植物的[[花冠|花冠]]為蝶形，即由五枚離生花瓣構成外形如蝴蝶的形狀，屬於两侧对称，花瓣由外至內分別為：最遠軸(看起來常為最上方)一瓣為旗瓣，二片翼瓣位於對稱軸兩側，二片龍骨瓣於最內側。龍骨瓣經常部份合生，使得不同種類具有不同的傳粉特性，有些需要其他動物(通常是昆蟲)傳粉，有些種類則因龍骨瓣合生而為自花授粉。

蝶形花亚科植物包括各种[[豆类|豆类]]，以及[[槐樹|槐]]、[[紫檀|紫檀]]等[[树|树]]木，共有一万多[[种|种]]，分布在全世界各地。大部分为[[草本|草本]]，也有[[木本|木本]]和[[藤本|藤本]]植物。

== 生态 ==

蝶形花亚科的花冠能保护生殖器官、又能显示美丽以吸引昆虫。蝶形花冠的旗瓣大而开展，色彩吸引昆虫；翼辧在左右两侧略伸展，可供昆虫停立；龙骨瓣背部边缘合生，将雌雄蕊包襄在内，防止雨水、害虫的侵袭。蝶形花的[[雄蕊|雄蕊]]常合生成管状，基部内面有蜜腺，只有[[蜜蜂|蜜蜂]]一类具长门吻的昆虫可以伸入；有时对着旗瓣的一枚雄蕊与合成管状的其他9枚离生，形成一缺口，便于蜜蜂深入到蜜腺基部；而蝶形花的雄蕊当昆虫来访时，借虫体重量压下龙骨瓣松开时，柱头才能伸出。
