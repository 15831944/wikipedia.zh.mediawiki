'''臨海線'''（{{jpn|j=りんかい線|hg=りんかいせん|rm=Rinkai sen|f=y}}）是一條連結[[東京都|東京都]][[江東區_(東京都)|江東區]]的[[新木場站|新木場站]]與[[品川區|品川區]]的[[大崎站|大崎站]]，屬於[[東京臨海高速鐵道|東京臨海高速鐵道]]（TWR）的鐵道路線。計劃、開業時的路線名是「臨海副都心線」（{{lang|ja|臨海副都心線}}），2000年起根據招募的名稱決定採用「臨海線」作為導覽<ref>2000年改稱後名稱定為「臨海線」，國土交通省監修的資料《{{link-ja|鐵道要覽|鉄道要覧}}》在平成9年度版採用的路線名稱「臨海副都心線」也在平成18年度版改為「臨海線」。舊名稱「臨海副都心線」使用的機會雖然減少了，但仍有一部分現存。</ref>。[[車站編號|車站編號]]使用的路線記號是'''R'''。

在大崎站與[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[埼京線|埼京線]]、[[川越線|川越線]]轉乘和[[直通運行|互相直通運行]]。大半的路段都在地下運行，國土交通省監修的統計資料<ref>例えば、

== 路線資料 ==
* 路線距離（{{link-ja|營業距離|営業キロ}}）：12.2公里
* [[軌距|軌距]]：1067毫米
* 站數：8個（包括起終點站）、1信號場
* 複線路段：全線
* 電氣化路段：全線（直流1500V、高架電纜）
* [[閉塞_(鐵路)|閉塞方式]]：（複線）自動閉塞式
* 保安裝置：[[自動列車停止裝置#ATS-P形（デジタル伝送パターン形）|ATS-P]]
* 最高速度：100公里/小時
* [[車輛基地|車輛基地]]：{{link-ja|東臨運輸區|東臨運輸区}}（八潮車輛基地）－位於為{{link-ja|東京貨物總站|東京貨物ターミナル駅}}側
* 地面路段：大崎－[[大井町站|大井町]]（東京綜合車輛中心附近為止）、[[國際展示場站|國際展示場]]（隧道出口為東側）－新木場
* 地底路段：大井町－國際展示場

== 沿革 ==
* 1996年（平成8年）3月30日：{{ST|新木場}}至{{ST|東京電訊}}之間的第1期區間通車。當時是以4-{zh-tw:節;zh-hk:卡;zh-cn:节;}-編組的[[東京臨海高速鐵道70-000系電力動車組|70-000系]]電聯車提供服務，路線名稱為「'''臨海副都心線'''」。
* 2000年（平成12年）9月1日：開始使用「'''臨海線'''」的暱稱。同時路線名由臨海副都心線改為臨海線。
* 2001年（平成13年）2月8日：八潮車輛基地啟用。
* 2001年（平成13年）3月31日：{{ST|東京電訊}}至{{ST|天王洲島|1}}之間、屬於第2期區間的部分路段通車。
* 2002年（平成14年）12月1日：伴隨第2期區間{{ST|天王洲島}}至{{ST|大崎}}之間的剩餘路線通車，臨海線全線開業，並同時開始與[[JR東日本|JR東日本]][[埼京線|埼京線]]的列車直通運行，相互駛入對方的路線。在直通運行業務開始初期，JR直通用列車是使用10-{zh-tw:節;zh-hk:卡;zh-cn:节;}-編組，而臨海線内的専用列車則為6-{zh-tw:節;zh-hk:卡;zh-cn:节;}-編組。
* 2004年（平成16年）10月16日：全部列車變更為10-{zh-tw:節;zh-hk:卡;zh-cn:节;}-編組，並開始出現部份只在臨海線內運行的JR車輛。

臨海線在大崎站連接JR東日本[[埼京線|埼京線]]，並與埼京線相互直通運行。而埼京線的北端亦與[[川越線|川越線]]（至[[川越站|川越站]]）相互直通運行。結果組成一個三線相互直通運行的系統。而在JR東日本的路線圖上，此運行系統顯示為「埼京線、川越線、直通東京臨海高速鐵道臨海線」（{{lang|ja|埼京-川越線、東京臨海高速鉄道りんかい線直通}}）。

一般來說，臨海線列車均以新木場站為起、終點站。但進入[[東臨運輸區|八潮車輛基地]]的列車會以東京電訊港站為起、終點站。

列車班次方面，平日每20分鐘開出一班往川越的快速列車，期間則有1-2班臨海線內列車。但不論列車類別，所有列車在臨海線內均停所有車站。雖然在全線開業前，曾編排了一個特別的時間表以應付非常繁忙的情況，但因客量不符合預期，這個時間表也一直沒有使用的機會。而臨海線在一些特定日子（如[[大除夕|大除夕]]、[[東京灣大花火祭|東京灣大花火祭]]、[[Comic_Market|Comic Market]]（Comiket））亦會加開特別班次疏導乘客。

列車運用方面，在全線開業初期曾使用6輛編組列車服務線內折返班次，但隨著列車統一為10輛編組，現時也有使用埼京線的E233系服務。

由於[[京葉線|京葉線]]在[[東京站|東京站]]並沒有連接其他路線，因此東端連接京葉線的臨海線便成為一條捷徑。不少自[[中央本線|中央線]]與[[東海道本線|東海道線]]出發前往[[東京迪士尼渡假區|東京迪士尼渡假區]]的[[臨時列車|團體專用列車]]，均會取道臨海線前往東京迪士尼渡假區所在的[[舞濱站|舞濱站]]。

== 女性専用車 ==
7時46分～9時41分於[[大崎站|大崎站]]發車往新木場方向行的全列車和22時34分以後由[[新木場站|新木場站]]發車往大崎方向行的全列車非假日設有女性専用車卡。
