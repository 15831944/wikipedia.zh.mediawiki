'''角板山祠'''，俗稱角板山神社，是[[臺灣日治時期|臺灣日治時期]]在[[新竹州|新竹州]][[大溪郡|大溪郡]]角板山地區（今[[桃園市|桃園市]][[復興區|復興區]]復興區中正圖書館旁）<ref>{{Cite web |url = https://www.google.com.tw/maps/@24.8180303,121.3512591,3a,37.5y,346.02h,78.22t/data=!3m6!1e1!3m4!1sYmfWoHTy7_f7BqBF-OFXlA!2e0!7i13312!8i6656?hl=zh-TW |title = google地圖上"角板山神社遺址"告示牌 |language=zh-TW }}</ref><ref>{{Cite web |url = http://blog.xuite.net/richard.tw/twblog/108265527 |title = 神社殘跡 角板山祠 |author = Hank |date = 2010-07-11 |language=zh-TW }}</ref><ref name="金"/>的[[神社|神社]]。

== 歷史 ==
角板山祠選址在[[新竹州|新竹州]][[大溪郡|大溪郡]]番地角板山社(今復興區中正圖書館旁{{r|金}})興建，於昭和13年（1938年）8月15日鎮座{{r|金}}。祭祀[[豐宇氣毘賣神|豐受大神]]、[[北白川宮能久親王|北白川宮能久親王]]{{r|金}}，例祭日11月15日{{r|金}}，1939年當時由時任縣社新竹神社社司宮島由多加管理<ref>{{Cite book | author = 臺灣總督府文教局社會課 | title =臺灣圖書館，日治時期圖書影像系統（需密碼才能看圖），[昭和十四年度] 臺灣總督府文教局社會課 臺灣ニ於ケル神社及宗敎 | location = 臺北市錦町 | publisher = 臺灣總督府文教局社會課 | date = 昭和15年1月19日 | pages = 原書第18頁 | url = http://hyerm.ntl.edu.tw:2135/cgi-bin/gs32/gsweb.cgi/ccd=b4CcL./record?r1=50&h1=0 | language = ja}}</ref>。二戰後建物陸續拆除，原本的本、拜殿一帶今成為空地，參道路徑尚存並成為巷道。附近尚留有部分[[石燈籠|石燈籠]]構件"竿"<ref>{{Cite web |url =https://www.google.com.tw/maps/@24.8183216,121.3509599,3a,15y,49.14h,72.04t/data=!3m6!1e1!3m4!1s4UIizqpX-tjIsm0CsR9D8Q!2e0!7i13312!8i6656?hl=zh-TW |title =google地圖街景中的原角板山祠時燈籠構件"竿" }}</ref><ref>{{Cite web |url = http://portal.tacp.gov.tw/onthisdate_archive_detail_for_subsite/17285 |title = 角板山神社石燈籠座2 |author = 台灣原住民族數位典藏知識入口網 |language=zh-TW  }}</ref>。

== [[神社建築|神社建築]] ==
角板山祠原配置有碎石參道、（應為混凝土造）[[鳥居|靖國鳥居]]2基、[[燈籠|燈籠]]數對、[[切妻造|切妻造]]手水舍、[[神明造|神明造]]拜殿與[[神明造|神明造]]本殿<ref>{{Cite web |url = http://nrch.culture.tw/view.aspx?keyword=%E8%A7%92%E6%9D%BF%E5%B1%B1&s=3389072&id=0000613808&proj=MOC_IMD_001# |title =國家文化資料庫中的角板山祠舊照}}</ref>。
