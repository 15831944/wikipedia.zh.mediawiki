在[[几何学|几何学]]中，'''角'''（[[拼音|拼音]]：jiǎo，[[注音符號|注音符號]]：ㄐㄧㄠˇ）是由两条有公共端点的[[射线|射线]]组成的几何对象。这两条射线叫做角的[[边|边]]，它们的公共端点叫做角的[[顶点|顶点]]。一般的角會假設在[[欧几里得平面|欧几里得平面]]上，但在[[非欧几里得几何|非欧几里得几何]]中也可以定義角，特別是在[[球面幾何學|球面幾何學]]中的{{link-en|球面角|spherical angle}}是用[[大圓|大圓]]的圓弧代替射线。角在[[几何学|几何学]]和[[三角学|三角学]]中有着广泛的应用。

几何之父[[欧几里得|欧几里得]]曾定义角为在[[平面|平面]]中两条不[[平行|平行]]的[[直线|直线]]的相对斜度。[[普罗克鲁斯|普罗克鲁斯]]認為角可能是一種特質、一種可量化的量、或是一種關係。{{link-en|歐德謨|Eudemus}}認為角是相對一[[直線|直線]]的偏差，{{link-en|安提阿的卡布斯|Carpus of Antioch}}認為角是二條相交直線之間的空間。欧几里得認為角是一種關係，不過他對直角、銳角或鈍角的定義都是量化的<ref>{{cite book |ref=harv|first=Johan Ludvig |last=Heiberg |year=1908 |title=Euclid |editor-first=T. L. |editor-last=Heath |editor-link=T. L. Heath |series=The thirteen books of Euclid's Elements |volume=1 |publisher=Cambridge University press|url=http://books.google.com/books?id=UhgPAAAAIAAJ|pages=177–178}}</ref>。

[[平面角|平面角]]的大小定义是以两[[射线|射线]]交点为圆心的[[圆|圆]]被射线所截的[[弧|弧长]]与[[半径|半径]]之比，单位包括[[弧度|弧度]]和[[角度|度]]、[[角分|分]]、[[角秒|秒]]等。<ref>{{Cite GB|id =3102.1 |title =空间和时间的量和单位 |year =1993 |previous =1986 |promulgate_date =1993-12-27 |effective_date =1994-07-01 }}</ref>

== 表示方法 ==
角通常用三个字母表示：两条边上的点的字母写在两旁，顶点上的字母写在中间。图中的角用∠AOB表示。但若在不會產生混淆的情形下，也會直接用顶点的字母表示，例如角∠O。

在數學式中，一般會用[[希臘字母|希臘字母]]（<math>\alpha,\beta,\gamma,\theta,\varphi,...</math>）表示角的大小。為避免混淆，符號[[Pi|π]]一般不用來表示角度。

== 角的測量 ==
=== 常見测量单位 ===
以角的端点为[[圆心|圆心]]做[[圆弧|圆弧]]。由于圆弧的[[半径|半径]]和弧长成[[正比|正比]]，而角是长度的[[比例|比例]]，所以圆的大小不会影响角的测量。
* '''弧度'''：用角在圆上所切出的[[弧|圆弧]]的长度除以圆的半径，一般记作rad。[[弧度|弧度]]是[[国际单位制|国际单位制]]中规定的角的度量，但却不是中国法定计量单位，[[角度|角度]]则是角在中国的法定计量单位。此外，弧度在[[数学|数学]]及[[三角学|三角学]]中有广泛的应用。
* '''角度'''：由角在圆上所切出的圆弧的长度除以圆的周长再乘以360的结果，一般用°来标记，读作“度”。一度可以继续分为60“分”或3600“秒”。[[角度|角度]]在[[天文学|天文学]]和[[全球定位系统|全球定位系统]]中有重要应用。
* '''[[梯度_(角)|梯度]]'''：是角在圆上所切出的圆弧的长度除以圆的周长再乘以400的结果。

=== 其他测量单位 ===
角度的量測可以視為弧長<var>s</var>和半徑<var>r</var>的比例，再依選用單位乘以一比例係數<math>\frac{2\pi}{n}</math>。

:<math> \theta = \frac{n}{2\pi} \frac{s}{r} </math>

例如以上的弧度、角度和梯度，其轉換係數<var><math>n</math></var>分別為<math>2\pi</math>、360和400。

以下是一些其他的测量单位，對應不同的<math>n</math>值。
*圈數或[[轉_(角)|轉數]]（<math>n=1</math>）：是指完整旋轉一圈，依應用的不同，會簡寫為''cyc''、''rev''或''rot''，不過在[[每分鐘轉速|每分鐘轉速]]（RPM）的單位中，只用一個字母r表示。
*[[直角|直角]]（<math>n=4</math>）：是<math>\frac{1}{4}</math>圈，是[[幾何原本|幾何原本]]中用的角度單位，直角<math>=90^\circ=\frac{\pi}{2}\mathrm{rad}=\frac{1}{4}\mathrm{turn}=100\mathrm{grad}</math>。在德文中曾用<math>\llcorner</math>表示直角。
*[[時角|時角]]（<math>n=24</math>）：）常用在[[天文學|天文學]]中，是<math>\frac{1}{24}</math>圈。此系統是用在一天一個週期的循環（例如星星的相對位置），其六十進制下的子單位稱為「時間分角」及「時間秒角」，這兩個單位和角度的[[角分|角分]]及[[角秒|角秒]]不同，前者大小為後者的十五倍。1時角<math>=15^\circ=\frac{\pi}{12}\mathrm{rad}=\frac{1}{6}\mathrm{quad}=\frac{1}{24}\mathrm{turn}\approx16.667\mathrm{grad}</math>。
*{{link-en|米位|Mil}}（<math>n=6000\sim 6400</math>）：此單位是指一個單位大約等於毫[[弧度|弧度]]的角度，有許多不同的定義，其數值從0.05625度到0.06度（3.375至3.6角分），而毫弧度約為0.05729578度（3.43775角分）。在[[北大西洋公約組織|北大西洋公約組織]]的國家中，米位定義為圓的<math>\frac{1}{6400}</math>。其數值大約等於一個角度的弧長為一公尺，其半徑為一公里的角度（<math>\frac{2\pi}{6400}=0.0009817...\doteqdot \frac{1}{1000}</math>）。
*[[角分|角分]]（<math>n=21600</math>）：定義為一度的<math>\frac{1}{60}</math>，是<math>\frac{1}{21600}</math>圈，會用 ′ 表示，例如3° 30′ 等於<math>3+\frac{30}{60}</math>度，也就是3.5度，有時也會出現小數，例如<math>3^\circ 5.72'=3+\frac{5.72}{60}</math>度。[[海里|海里]]曾定義為在地球的[[大圓|大圓]]上一角分的弧長。
*[[角秒|角秒]] （<math>n=1296000</math>）：定義為一角分的<math>\frac{1}{60}</math>，會用 ″ 表示，例如3° 7′ 30″等於<math>3+\frac{7}{60}+\frac{30}{3600}</math>度，或是3.125 度。

=== 正角和負角 ===
以上角的定義均未考慮數值為負的角。不過在一些應用時，會將角的數值加上正負號，以標明是相對參考物不同方向的旋轉。

在二維的[[笛卡兒坐標系|笛卡兒坐標系]]中，角一般是以x軸的正向為基準，若往y軸的正向旋轉，則其角為正角，若往y軸的負向旋轉，則其角為負角。若二維的笛卡兒坐標系也是x軸朝右，y軸朝上，則逆時針的旋轉對應正角，[[順時針|順時針]]的旋轉對應負角。

一般而言，<math>-\theta</math>角和一圈減去<math>\theta</math>所得的角等效。例如<math>-45^\circ</math>和<math>360^\circ -45^\circ (=315^\circ)</math>等效，但這只適用在用角表示相對位置，不是旋轉概念時。旋轉<math>-45^\circ</math>和旋轉315°是不同的。

在三維的幾何中，順時針及逆時針沒有絕對的定義，因此定義正角及負角時均需列出其參考的基準，一般會以一個通過角的頂點，和角所在平面垂直的[[向量|向量]]為基準。

在[[導航|導航]]時，[[導向|導向]]是以北方為基準，正向表示順時針，因此導向45°對應東北方。導向沒有負值，西北方對應的導向為315°。

===其他量測角大小的方法===
除了量測角本身的大小外．也有其他的方式，可以量測角的大小。

[[坡度|坡度]]等於一個角的正切值，常用百分比或千分比來表示。當一個角的坡度小於5%時，其坡度近似於角以弧度表示的數值。

在{{link-en|有理幾何學|rational geometry}}中，一個角的大小是以伸展度（spread）來表示，伸展度定義為角對應正弦的平方，而任一角正弦的平方和該角補角正弦的平方相等。因此任一角和其補角在有理幾何學中是等同的。

== 角的种类 ==
=== 個別的角 ===
; 零角
: 角度等於0°，或弧度為0的角。
; 锐角
: 角度大于0°且小于90°，或弧度大于0且小于<math>\frac{\pi}{2}</math>的角。
; 直角
: 角度等於90°，或弧度为<math>\frac{\pi}{2}</math>的角。
; 钝角
: 角度大于90°且小于180°，或弧度大于<math>\frac{\pi}{2}</math>且小于<math>\pi</math>的角。
; 平角
: 角度等於180°，或弧度为<math>\pi</math>的角。
; 優角或反角
: 角度大於180°且小於360°，或弧度大於<math>\pi</math>且小於<math>2\pi</math>的角。
; 周角
: 角度等於360°，或弧度為<math>2\pi</math>的角。

=== 角的組合 ===
有三種特殊角的組合，其度數和均為特殊的值：
*[[餘角|餘角]]：當兩個角的度數之和等於90°，即一個[[直角|直角]]，這兩個角便是餘角。若兩個相鄰的角互為餘角，兩個非共用邊會形成直角。在[[欧几里得几何|欧几里得几何]]中，非直角的兩角即互為餘角。
:若角''A''和''B''互為餘角，以下的數學式會成立：
:<math>\begin{align} \sin^2A + \sin^2B &= 1\\ \cos^2A + \cos^2B &= 1\\ \tan A &= \cot B \\ \sec A &= \csc B \end{align}.</math>
:（一角的[[正切|正切]]等於其餘角的[[餘切|餘切]]，一角的[[正割|正割]]等於其餘角的[[餘割|餘割]]）
*[[補角|補角]]：當兩個角的度數之和等於180°，即一個[[平角|平角]]，這兩個角便是互補角。若兩個相鄰的角互為餘角，兩個非共用邊會形成一直線。不過兩個不相鄰的角也可以是補角，例如平行四邊形中，任兩鄰角為互補角。[[圆内接四边形|圆内接四边形]]的對角也是互補角。
:若點P為圓O外的一點，而過點P作圓的切線，切點分別在點T和點Q，則∠TPQ和∠TOQ為互補角。
:兩互補角的正弦相等，其餘弦及正切（若有定義義）大小相等，但符號異號。
:在欧几里得几何中，三角形兩角的和為第三角的補角。

===同頂角===

<math>a+b+c=360^\circ</math>

===直線上的鄰角===

===與平行線有關的定理===
[[File:parr-angles.png|left]]

當<math>AE</math>平行於<math>BD</math>,
*<math>a=c</math>（[[同位角|同位角]]，<math>AE//BD</math>）
*<math>b=d</math>（[[內錯角|內錯角]]，<math>AE//BD</math>）
*<math>b+c=180^\circ</math>（[[同旁內角|同旁內角]]，<math>AE//BD</math>）

由角度的關係也可以推得兩直線平行
*當<math>a=c</math>，<math>AE</math>平行於<math>BD</math>（[[同位角|同位角]]，相等）
*當<math>b=d</math>，<math>AE</math>平行於<math>BD</math>（[[內錯角|內錯角]]，相等）
*當<math>b+c=180^\circ</math>，<math>AE</math>平行於<math>BD</math>（[[同旁內角|同旁內角]]，互補）

[[Image:Curve_angles.svg|thumb]]
曲線和直線的的夾角或是二曲線間的的夾角定義為二曲線在交點處[[切線|切線]]的夾角。
{{-}}
==點積及其拓展==
在[[欧几里得空间|欧几里得空间]]中，二個[[向量|向量]]'''u'''及'''v'''的角和其[[點積|點積]]及向量的長度有關：

:<math>\mathbf{u} \cdot \mathbf{v} = \cos(\theta)\ \|\mathbf{u}\|\ \|\mathbf{v}\|.</math>

依上式可以用二個平面（或曲面）的[[法向量|法向量]]，計算二者之間的夾角，也可以根據二[[歪斜線|歪斜線]]的向量計算其夾角。

==內積==
在一個抽象的實數[[内积空间|内积空间]]中，在定義角時可以用內積 <math>\langle\cdot,\cdot\rangle</math>取代欧几里得空间的點積( '''·''' )：

:<math>\langle\mathbf{u},\mathbf{v}\rangle = \cos(\theta)\ \|\mathbf{u}\|\ \|\mathbf{v}\|.</math>

在複數的内积空间中，為了使餘弦的數值仍維持實數，因此需修改為

:<math>\operatorname{Re}(\langle\mathbf{u},\mathbf{v}\rangle) = \cos(\theta)\ \|\mathbf{u}\|\ \|\mathbf{v}\|.</math>

或者使用絕對值的標示：
:<math> |\langle\mathbf{u},\mathbf{v}\rangle| = \cos(\theta)\ \|\mathbf{u}\|\ \|\mathbf{v}\|.</math>

後者不考慮向量的方向，因此是描述由向量<math>\mathbf{u}</math>及<math>\mathbf{v}</math>所生成的二個一維[[子空間|子空間]]<math>\operatorname{span}(\mathbf{u})</math>及<math>\operatorname{span}(\mathbf{v})</math>之間的夾角。

==黎曼几何中的角==
在[[黎曼几何|黎曼几何]]中，利用[[度量张量|度量张量]]來定義二條[[切線|切線]]之間的夾角，其中''U''及''V''是切線向量，''g''<sub>''ij''</sub> 是度量张量''G''的分量。

:<math>
\cos \theta = \frac{g_{ij}U^iV^j}
{\sqrt{ \left| g_{ij}U^iU^j \right| \left| g_{ij}V^iV^j \right|}}.
</math>

==地理學及天文學中的角==
以[[地理|地理]]的觀點，地球上任何一個位置都可以用[[地理座標系統|地理座標系統]]來表示，此系統標示位置的[[經度|經度]]及[[緯度|緯度]]，兩者都以此點連至地球球心連線的角度來表示，經度是以[[格林威治子午線|格林威治子午線]]為參考基準，而緯度是以[[赤道|赤道]]為參考基準。

在[[天文學|天文學]]中，[[天球|天球]]的一點可以用任何一種[[天球坐标系统|天球坐标系统]]來表示，不過其基準則因坐标系统不同而不同。天文學量測二顆星星的'''角距離'''時，會假想分別有二顆星星分別和[[地球|地球]]連成的直線，再量測這二條直線的夾角，即為角距離。

天文學家也會用[[角直徑|角直徑]]量測一物體的表觀大小。例如[[滿月|滿月]]的角直徑約為0.5°。{{link-en|小角公式|small-angle formula}}可以將上述的角測量轉換為距離和大小的比值。
