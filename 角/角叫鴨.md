'''角叫鴨'''（學名：''Anhima cornuta''）是屬於[[叫鴨科|叫鴨科]]的一種雀鳥，'''角叫鴨屬'''（學名：''Anhima''）的[[單型|單型]]種，即唯一一種。分佈於南美洲，數量正在下降<ref>Delany, S.; Scott, D. 2006. Waterbird population estimates. Wetlands International, Wageningen, The Netherlands.</ref>。

== 分佈 ==
玻利維亞、巴西、哥倫比亞、厄瓜多爾、法屬圭亞那、巴拉圭、秘魯、蘇里南和委內瑞拉。

== 形態 ==
前額長有約10厘米的硬質角，細長而彎曲，喙與雞類似，翅膀上長有兩個大小不同的距。腳趾長，蹼已退化一半，適合在濕地步行。
[[File:Anhima_cornuta_-_Flickr_-_Dick_Culbert.jpg|thumb]]

== 習性 ==
善於涉水和游泳，主要棲息於海拔800以下的平原水域，但有時也會上樹。主要吃[[水生植物|水生植物]]的草木種子，果實等，偶爾也捕食昆蟲。叫聲洪亮，遠處也能聽見，有時會在晚上聚集在起鳴叫。在[[沼澤|沼澤]]地中築巢，屬於浮巢。每次產卵4-7枚，孵化期43-45天。5年一代。
