'''[[角分|角分]]符号'''（{{lang|en|Prime}}, <span lang="en"> ′ </span>）、'''[[角秒|角秒]]符号'''（{{lang|en|Double Prime}}, <span lang="en"> ″ </span>）以及不常用的[[三上标号|3个撇的符号]]（{{lang|en|Triple Prime}}, <span lang="en"> {{Unicode|‴}} </span>）是一系列用于表示不同的度量单位的符號，在[[数学|数学]]、[[科学|科学]]、[[语言学|语言学]]、[[音乐|音乐]]等有不同的用途。

角分符号不应與[[撇号|撇号]]（ ' ）、标准的[[引号|单引号]]（-{ ‘’ }-）、[[尖音符|尖音符]]（ ´ ）或[[重音符|重音符]]（ ` ）；角秒符号不应混淆于[[引号|垂直双引号]]（ " ）、标准[[引号|双引号]]（-{ “” }-）<ref>{{cite book |first=Ron |last=Goldberg |chapter=Quotes |chapterurl=http://books.google.com/books?id=uo1j1buy2qYC&pg=PA67 |editors=Frank J. Romano |title=Digital Typography: Practical Advice for Getting the Type You Want When You Want It |publisher=Windsor Professional Information |location=San Diego |year=2000 |pages=67-69 |isbn=1-893190-05-6 |oclc=44619239 |accessdate=}}</ref>或[[同上符号|同上符号]]（ 〃 ）混淆。

==指代度量单位==
角分符号（ {{lang|la|′}} ）常用于表示[[英尺|英尺]]（ft）、[[角分|角分]]（am）与[[分钟|分钟]]（min）。

角秒符号（ {{lang|la|″}} ）表示[[英寸|英寸]]（in）、[[角秒|角秒]]（as）与[[秒钟|秒钟]]（s）。

因此，3{{lang|la|′}} 5{{lang|la|″}}代表3英尺5英寸，或者3分5秒的时间。度量角度时，3{{lang|la|°}} 5{{lang|la|′}} 30{{lang|la|″}}表示3[[角度|度]]5[[角分|角分]]30[[角秒|角秒]]。

==在数学、统计学中的使用==
在数学中，角分符号一般用来从已有的便利名字产生更多的相关变量名字，而不诉诸下角标 – {{lang|la|''x''′}} 一般意味着与 x 有关或者从 x 推导出来的变量。例如[[笛卡尔坐标系|笛卡尔坐标系]]中的点(x, y)，旋转、平移或者反射后，可以用({{lang|la|x′, y′}})表示。

通常，{{lang|la|''x''′}} 在第一次使用时给出定义。但有时其定义是事先约定的：
* [[导数|导数]]或导函数：{{lang|la|''f''′(''x'')}} 与 {{lang|la|''f''″(x)}} 是函数 {{lang|la|''f''(''x'')}} 关于 {{lang|la|''x''}} 的一阶与二阶导数。类似地，定义 {{lang|la|''y''}} = {{lang|la|''f''(''x'')}}，则 {{lang|la|''y''′}} 与 {{lang|la|''y''″}} 是 {{lang|la|''y''}} 关于 {{lang|la|''x''}} 的一阶与二阶导数。
* [[集合_(数学)|集合]]的[[补集|补]] {{lang|la|''A''′}} 是集合 {{lang|la|''A''}} 的补。
* [[概率论|概率论]]中[[事件_(概率论)|概率事件]]的否定：{{lang|la|Pr(''A''′)}} = {{lang|la|1 − Pr(''A'')}}
* 数学变换的结果：{{lang|la|''Tx''}} = {{lang|la|''x''′}}
* 矩阵的[[转置|转置]]。

==在物理学中的使用==
物理学中，角分符号表示一个变量在某个事件后的值。例如，''v''<sub>A</sub>′ 表示对象A在事件后的速度。在[[相对论|相对论]]中事件在[[惯性参考系|惯性参考系]]''S''中的坐标为(x, y, z, t)，在惯性参考系''S''′中的坐标为(x′, y′, z′, t′).

==在化学中的使用==
在化学中，角分符号用于区别一个分子中连接在同一原子上的不同官能团，例如R与R′, 表示一个有机化合物中连接在某个原子上的不同的烃基.

==在生物化学中的使用==
[[生物化学|生物化学]]、[[分子生物学|分子生物学]]，角分符号用于表示在[[脱氧核糖|脱氧核糖]]或[[核糖|核糖]]的环状结构上的碳原子位置。角分符号仅用于区分这两个化学结构的碳原子位置，而不是[[DNA|DNA]]或[[RNA|RNA]]其它部分的原子位置，如[[磷酸盐|磷酸盐]]基团或[[核酸|核酸]]。因此，当指明一个酶沿着DNA的链移动的方向时，分子生物学家说它从5′端移向3′端。角分符号还可用来指明分子内部结合其它基团的位置，例如“5′-单磷酸”。

==音乐中的使用==

[[File:Helmholtz_pitch_notation_c.svg|150px]]
在[[赫尔姆赫兹音调记号法|赫尔姆赫兹音调记号法]]中，小写字母与角分符号、秒分符号与‴相结合表示不同的八度音程中的音符，因此c表示比[[中央C|中央C]]低八度，c′表示中音C，c″ 比中央C高八度，c{{Unicode|‴}}表示比中央C高两个八度。大写字母与角标角分符号表示更低的音符。 C 表示比[[中央C|中央C]]低两个八度, C ͵表示比[[中央C|中央C]]低三个八度。

== 历史==
在英文环境，角分符号称作“prime”源于误读。因为在20世纪初期，''x''-{′}- 读作“x prime”并不是因为在 ''x'' 后跟着一个“prime symbol”，而是因为它是''x''-{′}-，''x''″ ("x second") 与{{Unicode|''x''‴}} ("x third") 中的第一个，这里prime是“最初的”意思。后来在1950年代与1960年代，术语“prime”开始用于[[撇号|撇号]]这一类的符号。现在''x''″与{{Unicode|''x''‴}}在英文中通常读作“x double prime”与“x triple prime”。

“modifier letter prime”与“modifier letter double prime”用于语言学目的，如在转写[[西里尔字母|西里尔字母]]文字时标识重音。

某些字符集，如[[ISO_8859-1|ISO 8859-1]]，不包含这些符号。一般用[[撇号|撇号]]或[[引号|垂直双引号]]代替。

[[LaTeX|LaTeX]]数学模式中，<code>f'</code> (f带一个撇号)被绘制为<math>f'\,\!</math>。此外，LaTeX提供一个大尺寸的角分符号，<code>\prime</code> 将显示为<math>\prime</math>，用于下标表示。例如，<code>f_\prime^\prime</code>绘制为<math>f_\prime^\prime</math>。
