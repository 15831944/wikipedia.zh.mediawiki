'''角落小夥伴'''（{{lang-ja|すみっコぐらし}}），又名'''角落生物'''，是由日本企業[[San-X|San-X]]所創造出的擬人角色，其設定為喜歡躲在角落的四周圍 <ref>{{cite web |url=https://news.gamme.com.tw/1418547 |title=可愛到犯規《角落生物》喜歡躲在牆角的邊緣吉祥物正夯～ |accessdate=2019-01-08 |date=2016-06-23}}</ref>。

== 登場角色 ==
=== 角落 ===
; {{nihongo|北極熊（白熊、白白）|しろくま}}
: 一隻白色、胖胖且害羞的熊，因為覺得北方真的太冷了，毅然決定逃往南方，喜歡喝熱熱的茶，常常帶著一個裹巾，那是他的行李。
; {{nihongo|企鵝？(香瓜)|ぺんぎん？}}
: 其實自己也不知道自己是什麼，只記得以前頭上好像有個蓋子，不知道自己是什麼東西，所以對自己很沒自信。喜歡閱讀和聽音樂。
; {{nihongo|炸豬排|とんかつ}}
: 吃完炸豬排的時候，盤子裡總有些外層肥肉，麵皮等被剩下，因為太油了不會吃光，這個角色便是這些剩餘麵皮與肥肉的組合物，全身共有99%的肥肉與麵皮，全身上下只有1%的瘦肉，也就是他的鼻子，因為自己大部分都是被剩下來的，所以對自己很沒有自信。
; {{nihongo|貓咪|ねこ}}
: 害羞的貓咪，喜歡躲在角落裡背對着大家抓牆壁。最喜歡的食物是罐頭~
; {{nihongo|恐龍(蜥蜴)|とかげ}}
: 假冒蜥蜴的恐龍，倖存的恐龍，因為怕被發現所以假冒蜥蜴。愛吃:魚，特長:游泳。

=== 迷你 ===
; {{nihongo|包袱巾（裹巾）|ふろしき}}
: 白熊的行李，占位時常用到，內容物不明，沒表情，被珍惜的時候會微笑。
; {{nihongo|炸蝦尾|えびふらいのしっぽ}}
: 炸蝦的尾巴，因為太硬所以被吃剩下來，和炸豬排是好朋友。
; {{nihongo|雜草|ざっそう}}
: 角落生物中唯一積極的角色，夢想是成為花，很積極地在過日子。用根部走路，永遠是一張笑臉，植物種類不明。
; {{nihongo|蛞蝓|にせつむり}}
: 假冒蝸牛的蛞蝓，和假冒蜥蜴的恐龍是好朋友，什麼都背，靠身體的黏性把外殼黏在身上，外殼是稀有的水玉點點，動不動就道歉。
; {{nihongo|粉圓|たぴおか}}
: 喝粉圓奶茶時，總會有幾顆粉圓到最後沒被吸到，它們便是這樣的存在，口頭禪是：「啊～做不下去了啦～」，是群懶散且消極的傢伙們，繽紛多彩。黑粉圓不常出現，比彩色粉圓更愛鬧彆扭。
; {{nihongo|灰塵|ほこり}}
: 會飛、無憂無慮的小灰塵，會大量出現，集結在一起會變大，認為角落就是自己的庭院。
; {{nihongo|麻雀|すずめ}}
: 只是一隻平凡的麻雀，喜歡吃炸豬排，所以會啄食炸豬排。
; {{nihongo|小山|やま}}
: 一座嚮往富士山的小山，所以出現在溫泉區假裝成富士山。
; {{nihongo|飯糰|おにぎり}}
炸蝦最喜歡吃的東西，常常被放在要去遠足的便當盒裡。
; {{nihongo|幽靈|おばけ}}
: 生活在閣樓的角落裡，他安靜又愛乾淨，很討厭嚇人，喜歡打掃和喝咖啡。
