'''角色客製化'''，是一種[[大型多人線上角色扮演遊戲|大型多人線上角色扮演遊戲]][[創建角色|創建角色]]系統和元素，玩家可以透過這種系統調整角色造型數值自主創建自己所想的玩家角色造型。

==功能==
玩家可以自主調整諸如性別、種族、體型等角色造型設定<ref name="《BLESS》釋出角色客製化系統介紹影片 展示多達180種調整細節">{{cite news |language = 中文 |author =  |coauthors =  |url = https://www.gamebase.com.tw/news_old/news_content.php?sno=98092951 |title = 《BLESS》釋出角色客製化系統介紹影片 展示多達180種調整細節 |publisher = 遊戲基地新聞 |pages =  |date = 2015-09-07 |accessdate = 2017-10-23}}</ref>，甚或設定獨特的能力與技能。<ref name="《正當防衛》前設計師參與 黑暗風格 MOBA 新作《死靈異種》首度公開">{{cite news |language = 中文 |author =  |coauthors =  |url = https://gnn.gamer.com.tw/7/91347.html |title = 《正當防衛》前設計師參與 黑暗風格 MOBA 新作《死靈異種》首度公開 |publisher = 巴哈姆特GNN新聞網 |pages =  |date = 2014-01-07 |accessdate = 2017-10-23}}</ref>與[[紙娃娃系統|紙娃娃系統]]不同，角色客製化是由[[遊戲開發者|遊戲開發者]]設置各造型數值的最大值和最小值，再由玩家在該範圍內自行調整造型數值，防止角色造型千篇一律或與其他玩家過份相似<ref name="《AION 永恆紀元》超強角色客製化 讓玩家不再成為複製人">{{cite news |language = 中文 |author =  |coauthors =  |url = http://www.qk123.com/2009/04/4201/ |title = 《AION 永恆紀元》超強角色客製化 讓玩家不再成為複製人 |publisher =  香港數碼消費網站遊戲新聞 |pages =  |date = 2009-09-04 |accessdate = 2017-10-23}}</ref>；而後者則僅提供不同樣式的造型部分，玩家通常不可調整數值。

==遊戲性==
自主化的角色創建系統可讓玩家體驗多樣性<ref name="《BLESS》27 日韓國公測　公開角色客製化與場景影片">{{cite news |language = 中文 |author =  |coauthors =  |url = https://gnn.gamer.com.tw/9/127049.html |title = 《BLESS》27 日韓國公測　公開角色客製化與場景影片 |publisher = 巴哈姆特GNN新聞網 |pages =  |date = 2016-01-26 |accessdate = 2017-10-23}}</ref>，更大幅度提高玩家的遊戲情緒和自我認同。<ref name="角色扮演遊戲中化身客製化與情緒、自我認同之差異研究">{{Cite thesis |degree= 碩士 |chapter= |title= 〈角色扮演遊戲中化身客製化與情緒、自我認同之差異研究〉 |url= http://ndltd.ncl.edu.tw/cgi-bin/gs32/gsweb.cgi?o=dnclcdr&s=id=%22096NCTU5376021%22.&searchmode=basic#XXX |author= 鍾舜安 |last= |first= |year= 2008年 |publisher= |accessdate= 2017-10-23 |docket= |oclc= }}</ref>

這種系統可以作為一種遊戲元素加入其它類型的遊戲，例如[[多人線上戰鬥競技場遊戲|多人線上戰鬥競技場遊戲]]。<ref name="《正當防衛》前設計師參與 黑暗風格 MOBA 新作《死靈異種》首度公開"></ref>
