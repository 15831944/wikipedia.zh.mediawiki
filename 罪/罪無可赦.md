{{作品名稱|name=''Only God Forgives''|hk=罪無可赦|cn=唯神能恕|equiv-tw=hk}}是一部2013年丹麥和法國合拍的犯罪電影，由[[尼可拉·溫丁·黑芬|尼可拉·溫丁·黑芬]]執導和編劇，主演是[[賴恩·高斯寧|賴恩·高斯寧]]、[[姬絲汀·史葛·湯瑪絲|姬絲汀·史葛·湯瑪絲]]和{{link-en|維塔亞·潘斯林加姆|Vithaya Pansringarm}}。本片在[[泰國|泰國]]的[[曼谷|曼谷]]拍攝，而且與《[[極速罪駕|極速罪駕]]》一樣是獻給智利導演[[亞歷山卓·尤杜洛斯基|亞歷山卓·尤杜洛斯基]]<ref>{{Cite web |url = https://www.theguardian.com/film/2013/jul/29/patterson-only-god-forgives |title = Only God Forgives this level of tedium |author = John Patterson |date = 2013-07-29 |publisher = The Guardian |language = en}}</ref>。本片在[[第66屆康城影展|2013年康城影展]]上競爭[[金棕櫚獎|金棕櫚獎]]。
