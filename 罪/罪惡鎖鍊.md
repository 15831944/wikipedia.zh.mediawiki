《'''罪惡鎖鍊'''》（{{lang-en|'''''Invisible Stripes'''''}}）是一部1939年的[[犯罪電影|犯罪電影]]，由{{link-en|洛埃·貝肯|Lloyd Bacon}}執導。主演是{{link-en|喬治·拉夫特|George Raft}}，飾演一名出獄回家後無法改邪歸正的黑幫，其他演員有{{link-en|珍·布萊恩|Jane Bryan}}、[[威廉·荷頓|威廉·荷頓]]和[[堪富利·保加|堪富利·保加]]。{{link-en|華倫·B·朵芙|Warren B. Duff}}編劇，改編自{{link-en|劉易斯·E·拉韋斯|Lewis E. Lawes}}的英文同名小說，而劉易斯·E·拉韋斯是一名為監獄改革工作的熱心運動參加者。<ref>{{Cite web |url = http://www.tcm.com/tcmdb/title/79374/Invisible-Stripes/ |title = INVISIBLE STRIPES(1939) |publisher = Turner Classic Movies |language = en}}</ref><ref>{{Cite web |url = https://www.bfi.org.uk/films-tv-people/4ce2b72eccd85 |title = Invisible Stripes (1939) |publisher = British Film Institute |language = en}}</ref>
