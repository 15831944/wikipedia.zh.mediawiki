《'''動漫影后'''》（[[希伯來語|希伯來語]]︰כנס העתידנים，{{lang-en|'''''The Congress'''''}}）是一部2013年的{{link-en|真人與動畫電影|Films with live action and animation|真人/動畫}}[[科幻電影|科幻電影]]，導演和編劇是{{link-en|阿黎·佛爾曼|Ari Folman}}，改編自[[史坦尼斯勞·萊姆|史坦尼斯勞·萊姆]]的小說《{{link-en|未來學大會 (小說)|The Futurological Congress|未來學大會}}》。本片於2013年5月15日在[[第66屆康城影展|2013年康城影展]]首映<ref>{{Cite web |url = http://www.hollywoodreporter.com/news/cannes-ari-folman-s-congress-442939 |title = Cannes: Ari Folman’s 'The Congress' to Open Directors' Fortnight |author = Rhonda Richford |date = 2013-04-19 |publisher = The Hollywood Reporter |language = en}}</ref>。

== 劇情 ==
女星羅冰·活麗在電影中扮演自己，曾在[[荷里活|荷里活]]風光無限，但隨年華老去戲約大減，加上為了照顧自己得到罕見疾病{{link-en|尤塞氏綜合症|Usher syndrome}}的兒子，與對藝術的個人堅持，讓她屢屢拒絕戲約，終致片廠的抵制。經紀人屬意讓她接下最後一份工作，而這份工作可以視為她演藝生涯的另一個開端，同時也是終點。電影公司決定使用數位技術，將她的影像掃描進電腦裡，保存下來；未來他們有權使用她的真人捕捉影像，放進任何一部片裡，不再需要她的真人飾演，而這樣的影像，也永遠保存了她的年歲。 

二十年後，在影迷心中青春不朽的羅冰，為了出席電影公司舉辦的「未來主義會議」，一腳踏進了奇幻的虛擬動畫世界，在這個世界，任何想像都可能發生，每個人都可以扮演著別人。意外遭到狙擊的她，碰上了反對未來大會的革命軍，大舉破壞電影公司基地。羅冰即使在動亂中活下來，但因身體對藥劑出現嚴重過敏反應，醫生束手無策，決定冷凍她的身體。多年後，她終於醒過來，決意回現實世界找回她的兒子。可惜她回到現實世界，眼前是一眾已吸入藥劑，沉醉自己構想的世界的人。她找回昔日的家庭醫生，發現兒子在現實世界苦候她十九年，最後他支撐不住，決定吸入藥劑。最後，她同樣選擇返回幻想世界，在幻想中跟兒子重逢。

== 關於史坦尼斯勞·萊姆的《未來學大會》 ==
根據導演{{link-en|阿黎·佛爾曼|Ari Folman}}所說，電影中有些元素的靈感來自[[史坦尼斯勞·萊姆|史坦尼斯勞·萊姆]]的科幻小說《{{link-en|未來學大會 (小說)|The Futurological Congress|未來學大會}}》內的角色{{link-en|Ijon Tichy|Ijon Tichy}}。在早期的訪問中佛爾曼說過︰<ref>{{Cite web |url = http://culture.pl/en/article/ari-folman-on-the-genius-of-stanislaw-lem |title = Ari Folman on The Genius of Stanisław Lem |date = 2011-03-31 |publisher = Culture.pl |language = en}}</ref><blockquote>「電影的第一部分當然沒有甚麼是基於萊姆的小說。第二部分則絕對不同，但我用了很多《未來學大會》的元素作為靈感來源，而不是改編。」</blockquote>

後來，在電影的官方網站上的一個訪問，佛爾曼說這個把萊姆的作品放進電影的想法於他在電影學校期間已經開始。他描述了他如何重新討論萊姆的[[共產主義國家|共產主義獨裁]]寓言放至一個更現實的設定，即是[[演藝界|娛樂業]]中的獨裁，並表示他相信儘管偏離原著，但他也保留了原著的精神。
