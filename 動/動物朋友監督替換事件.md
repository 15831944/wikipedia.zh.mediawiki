'''動物朋友監督替換事件'''（{{jpn|j=けものフレンズ監督降板騒動|hg=けものフレンズかんとくこうばんそうどう|rm=Kemono Furenzu kandoku kōban sōdō|f=y}}），是一件在2017年發生的[[日本動畫|日本動畫]]業界爭議事件，主因為[[電視動畫|電視動畫]]《[[動物朋友|動物朋友]]》第二期中替換[[監督|監督]][[達紀|達紀]]後引起粉絲不滿，加上[[KADOKAWA|角川]]本身的不當應對方式而引發的一連串風波。

==事件背景==
電視動畫《動物朋友》在動畫監督[[達紀|達紀]]的帶領之下，以大約十人的團隊在業界內不看好的情況下花費500餘天完成，且在2017年1月開始於[[東京電視網|東京電視網]]放送後成功引起大流行。其中在電視動畫第四集後開始在日本網路引發話題，且動畫內容如「好好玩─！」、「好厲─害！」等台詞登上[[Twitter|Twitter]]熱門關鍵字<ref>{{Cite web|url=http://game.ettoday.net/article/865854.htm|script-title=哇～好好玩～！新番《動物朋友》掀起日本智能低下熱潮|date=2017-02-13|website=東森新聞雲|accessdate=2017-02-13|language=zh-tw}}</ref>造成一股流行風潮，且帶動了相關周邊發展以及合作。<ref>{{cite web|url=http://www.asahi.com/articles/ASK4S675NK4SULZU01T.html |language=ja |title=「けものフレンズ」でサーバル人気　多摩動物公園 |publisher=朝日新聞 | date=2017-04-25 |accessdate=2017-04-28 }}</ref><ref>{{cite web|url=http://www.appledaily.com.tw/realtimenews/article/international/20170526/1126544/ |language=zh-tw |title=【2次元】老婆跟人跑　企鵝愛上動漫女主角 |publisher=蘋果日報 | date=2017-05-26 |accessdate=2017-05-27 }}</ref>

==事件經過==
2017年9月25日晚上，達紀在Twitter公告：「事發突然，離開了《動物朋友》動畫事務，大概是角川的指示吧。對不起，我也感到很遺憾」<ref>https://twitter.com/irodori7/status/912270635610472448</ref><ref>{{cite news|script-title=ja:たつき監督、「けものフレンズ」続編から降板か　Twitterで明らかに |url=http://www.itmedia.co.jp/news/spv/1709/25/news117.html |publisher=ITmedia|date=2017-09-25|accessdate=|language= ja}}</ref>引起大量[[日本|日本]]、[[台灣|台灣]]與[[歐美|歐美]]等地區的[[愛好者|愛好者]]關注，對角川的撻伐與不平<ref>{{cite news|script-title=Animen 動漫平台 - 大人氣動畫《動物朋友》監督「たつき」表示將無法繼續參與動畫製作，粉絲們紛紛表示無法接受|url=https://www.animen.com.tw/NewsArea/NewsItemDetail?NewsId=23606&categoryId=500|publisher=Animen新聞部|date=2017-09-26|accessdate=2017-10-19|language=zh-tw}}</ref>。當天晚上[[角川多玩國|角川多玩國]]股票下挫5.18%，但在隔天開盤後迅速恢復<ref>{{cite news|script-title=ja:カドカワ株価急落の理由は！？フレショック炎上をまとめてみた！|url=https://xn--xuu60njodv0u.biz/kadokawa-kabuka/|publisher=ITmedia|date=2017-09-26|accessdate=2017-10-19|language= ja}}</ref>，對於股價並無造成重大影響。

===角川及製作委員會方面===
角川多玩國社長[[川上量生|川上量生]]在事發當下似乎不知情，表示很擔心達紀被撤換，也肯定他的《動物朋友》讓[[Niconico動畫|Niconico動畫]]熱絡起來，無論[[角川|角川]]或[[多玩國|多玩國]]關於這件事沒有不能談的，至於結果如何，目前還未瞭解掌握狀況。<ref>{{cite news|script-title=ja:『けものフレンズ』たつき監督が外される騒動　川上「社内で議論になる」|url=http://news.nicovideo.jp/watch/nw2989683 |publisher={{lang|ja|ニコニコニュース}}|date=2017-09-26|accessdate=|language=}}</ref>[[製作委員會|製作委員會]]相關的[[武士道_(公司)|武士道]]社長{{tsl|ja|木谷高明|木谷高明}}，以[[職業摔角|職業摔角]]比喻抽換監督風波：「別讓人看到在場外亂鬥，這樣對支持的來客是很失禮的。」 <ref>{{cite web|author=木谷高明|script-title=ja:少し言わせてください。|url=https://twitter.com/kidanit/status/912463709439639552 |website= |via=[[Twitter|Twitter]] |date=2017-09-26|accessdate=|language= ja}}</ref>此話引起愛好者社群憤怒，認為不該對角川的作為做辯護。但後來川上量生與製作委員會立場一致，並套用木谷高明的「場外亂鬥說」，批評任何一方都不該將觀眾情感帶進大人的爭吵中。<ref>{{cite web|author=川上量生（かわんご@kawango）|script-title=ja:個人的には大人の喧嘩をユーザーの感情を利用した場外乱闘に持ち込むのはどっちの側もやってはいけないことだと思います。|url=https://friends.nico/@kawango/36706379|website=|via=friends.nico|date=2017-09-27|accessdate=|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20170929043949/https://friends.nico/@kawango/36706379|archivedate=2017-09-29}}</ref>

2017年9月27日凌晨，《動物朋友》官方網站以製作委員會名義發布聲明，稱相關各方希望結合動畫、舞台劇、遊戲等平台擴充動物朋友的世界觀，因此將使用新的體制進行後續製作與發展，達紀所屬的動畫製作公司{{nihongo|八百萬|ヤオヨロズ}}因無法接受而退出。<ref>{{Cite web |url = http://kemono-friends.jp/archives/category/news/  |title = 「けものフレンズ」の映像化プロジェクトに関するご報告 |author = けものフレンズプロジェクトA |date = 2017-09-27 |language = ja |accessdate = 2017-09-27  }}</ref>聲明中亦指責，八百萬未連絡告知製作委員會，即使用作品。

但有業界消息指稱原作者[[吉崎觀音|吉崎觀音]]跟達紀在對《動物朋友》的意見上出現一些分歧所導致，吉崎觀音甚至提出了替換達紀的要求；導致部分網友灌進吉崎觀音的推特頁面謾罵<ref>{{cite web|author={{lang|ja|松本淳（まつもとあつし）}}|script-title=ja:「けものフレンズ」たつき監督降板！？――コンテンツは誰のものなのか？|url=https://news.yahoo.co.jp/byline/matsumotoatsushi/20170926-00076202/ |website={{lang|ja|Yahoo!ニュース}} |via= |date=2017-09-26|accessdate=|language= ja}}</ref><ref>{{Cite web |url=http://dailynewsonline.jp/article/1359354/ |script-title=ja:けものフレンズ監督降板騒動「功労者をのけものにする」KADOKAWAの企業体質|website=Daily News Online |language = ja |date = 2017-09-28}}</ref>。

10月1日，川上量生於論壇中發文，標題為「關於動物朋友」，內文卻寫了「怎麼可能寫出來啊」，引起網友憤怒，從事件開始就持續關注並透露消息的多玩國集團[[執行董事|執行董事]]栗田穰崇也指責川上不應該開玩笑。此篇文章在放出後1小時遭刪除。

10月2日，角川執行長井上伸一郎在Twitter上發言，認同達紀的功績，並表示會與八百萬再談。<ref>https://twitter.com/HP0128/status/915071890648993792</ref>

10月10日，負責雕鴞助手配音的聲優[[上原明里|上原明里]]在推特發文，表明自己10月11日起從{{tsl|ja|キャトルステラ|Quatre Stella}}移籍{{tsl|ja|株式会社クレール|株式會社Clair}}，兩間公司為姊妹公司，聲優工作不會受到影響。但是Clair為10月6日所創立，且成員只有上原明里一人；被認為是由於事件發生後僅有上原明里做出聲援達紀的行為，而遭到打入冷宮。<ref>{{Cite web|url = http://xw.qq.com/cmsid/2017101603849800|title =業界之暗！唯一聲援《獸娘》監督的聲優被打入冷宮|author=騰訊網|date = 2017-10-16|language = zh-cn|accessdate=2017-10-19}}</ref> 上原明里要在2019年4月才再有機會為動畫聲演主要角色。

===角川的合作廠商===
在製作委員會指責八百萬在未告知下使用作品之後，跟《動物朋友》有廣告合作關係的[[日本中央競馬會|日本中央競馬會]]與[[日清食品|日清食品]]皆在官網上加註聲明，澄清所有合作案都是與角川或《動物朋友》製作委員會直接洽談並獲得授權<ref>{{Cite web |url = http://umabi.jp/kemono-friends/ |title = 日本中央競馬會廣告宣傳網站 |author = 日本中央競馬會 |date = 2017-09-27 |language = ja |accessdate = 2017-09-27 |deadurl = yes |archiveurl = https://web.archive.org/web/20171227211517/http://umabi.jp/kemono-friends/ |archivedate = 2017-12-27 }}</ref><ref>{{Cite web |url = http://www.donbei.jp/kemono-friends/ ||script-title=ja: 日清のどん兵衛 × けものフレンズ |author = 日清食品 |date = 2017-09-27|language = ja |accessdate = 2017-09-27 }}</ref>。

9月27日晚上於[[niconico生放送|niconico生放送]]節目{{nihongo|《動物朋友時間》|けものフレンズアワー}}中，負責[[浣熊|浣熊]]配音的[[聲優|聲優]][[小野早稀|小野早稀]]在節目一開始對該事件造成的騷動道歉，這種在愛好者團體中通稱「[[聲優之盾|聲優之盾]]」、讓無關人士當擋箭牌的行為引起了更大的爭議和不滿。<ref>{{cite news|title=是火上加油的朋友！角川派動物朋友聲優謝罪氣炸粉絲|url=https://game.ettoday.net/article/1020300.htm |language=zh-tw |author=羅仁豪 |publisher=ETNEWS遊戲雲 |date=2017-09-26 |accessdate=2017-09-28 }}</ref>

角川公關負責人針對公司因撤換監督一事而成為箭靶，表示並未個別通知監督，也否認由該公司解任的傳言。<ref>{{Cite news |url = https://news.careerconnection.jp/?p=41081 ||script-title=ja: 【けもフレ】たつき監督解任騒動、製作委員会の公式声明発表後も残る謎　結局だれが何をしたのが問題なのか？ |publisher={{lang|ja|キャリコネニュース}}| |date = 2017-09-27|language = ja |accessdate = 2017-09-27 }}</ref>

===監督支持者及八百萬方面===

日本方面已有網友發起連署，反對角川抽換《動物朋友》監督的決定 <ref>{{cite web|script-title=ja:Petition · けものフレンズ視聴者のみんなへたつき監督辞めないで！|url=https://www.change.org/p/けものフレンズ視聴者のみんなへたつき監督辞めないで |website= Change.org|date=|accessdate=|language=}}</ref>，在發起後三小時內就超過一萬兩千多人連署<ref>{{cite web|script-title=角川急撤《動物朋友》動畫導演網友暴怒　引萬人連署抗議|url=https://game.ettoday.net/article/1019162.htm|website= ETNEWS遊戲雲([[東森新聞|東森新聞]])|date=2017年9月26日|accessdate=2017年10月22日|language=zh-tw}}</ref>。

在有吉崎觀音要求抽換《動物朋友》監督的流言出現後，曾與吉崎觀音在《[[Keroro軍曹|Keroro軍曹]]》合作的動畫監督[[佐藤順一|佐藤順一]]在推特發文聲援吉崎觀音，表示「吉崎老師是很尊重監督意願的作者」<ref>{{cite web|author=佐藤順一|script-title=ja:西田さんがなんで怒ってるのかと思ったら怪しい記事が。|url=https://twitter.com/satojumichi/status/913386701124337664 |website= |via=[[Twitter|Twitter]] |date=2017-09-29|accessdate=|language= ja}}</ref>。

八百萬官網曾在事件發生後無法使用，但無其他方面做出聲明。10月2日，八百萬社長福原慶匡在Twitter上發文，表示將與角川方面迅速處理此次事件<ref>https://twitter.com/fukuhara_ystd/status/915074525783453696</ref>。

====「燃燒紙飛機」行動====

9月30日，由《動物朋友》台灣支持者所發起的行動「燃燒紙飛機」（{{lang|ja|燃える紙飛行機}}）在日本受到廣大迴響，亦有[[香港|香港]]、韓國及歐美地方的支持者響應，當天衝上Twitter的趨勢榜；但是大約於8點30分於趨勢榜上消失，有網友認為是角川施壓的結果，亦有網友認為因為Twitter系統機制的關係而被當成機器人洗推文而撤除。當日最後紀錄為10萬推。

10月20日，《動物朋友》支持者發起了第二次燃燒紙飛機行動，在推特上的標籤為「乘著希望的紙飛機」（{{lang|ja|紙飛行機に希望を乗せて}}）。

== 事件結果 ==
經過多次的開會協商，12月27日，福原慶匡於Twitter上發文，角川沒有回覆相關事件，默認達紀將不會是第二期的監督。<ref>https://twitter.com/fukuhara_ystd/status/945943082431758336</ref><ref>{{cite web |url=https://www.oricon.co.jp/news/2103178/amp |title=存档副本 |accessdate=2017年12月30日 |deadurl=yes |archiveurl=https://web.archive.org/web/20171227162910/https://www.oricon.co.jp/news/2103178/amp/ |archivedate=2017年12月27日 |df= }}</ref>

== 後續 ==

2018年9月2日，角川在Youtube釋出動物朋友2的PV，獲大量負評，截至該月16日，正評為5700左右，負評則為51000。<ref>{{cite web|title=TVアニメ『けものフレンズ2』PV　第一弾

|url=https://www.youtube.com/watch?v=Jb94vH0VAek}}</ref>同時公開選拔新組合聲優。

2018年9月3日，受害者在推特上發推，揭露選拔聲優的台本盜用別人的-{腳本}-。<ref>{{cite web|title=未開始先炎上，動畫《動物朋友2》聲優選拔課題內容被揭發盜用他人台詞本|url=https://hkacger.com/archives/61100/}}</ref>

2018年9月14日，達紀監督於Twitter上發布推文，發表對盜聲優劇本的看法，以及表明至今動畫劇本費、劇本版稅連1日元都沒有收到，並看見了角川對創作者的鄙視。<ref>{{cite web|title=たつき監督が「けもフレ」の台本無断転載に苦言「創作への蔑視」|url=http://news.livedoor.com/article/detail/15307709/}}</ref><ref>{{cite web|title=「全話劇本費和劇本版稅一日元都沒付」，たつき監督再爆料《動物朋友》製作委員會|url=https://hkacger.com/archives/62346/}}</ref>

《動物朋友2》最後於2019年1月7日於[[東京電視台|東京電視台]]系統開始播映，但很快因各種因素受到觀眾的惡評，至4月2日最后一话播出结束后，[[niconico|niconico]]网友中仅有2.6%给出了好评，为niconico史上评分最低的一集动画。

该事件也在[[中国大陆|中国大陆]]有强烈的反响。大部分中国大陆粉丝也纷纷批评角川的处理方式不当。而在[[bilibili|bilibili]]上，其系列原创短动画《欢迎来到加帕里公园》仅获得2.5分的评价，为bilibili史上番剧最低评分。動畫二期也僅獲得2.6的評價。
