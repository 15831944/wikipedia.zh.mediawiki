'''笕桥中央航校旧址'''位于[[杭州市|杭州市]][[江干区|江干区]][[笕桥街道|笕桥街道]]东北部的横塘村，是民国时期的中央航空学校，被誉为“[[中華民國空軍|中国空军]]的摇篮”。2006年成为[[全国重点文物保护单位|全国重点文物保护单位]]。原校現已遷往台灣成為[[中華民國空軍軍官學校|中華民國空軍軍官學校]]。<ref name="旧址">{{cite web |url = http://culture.zjol.com.cn/05culture/system/2006/08/04/007788509.shtml |language = |title = 笕桥中央航校旧址 |publisher = 浙江在线 |date = 2006年5月18日 |accessdate = }}</ref>

== 简介 ==
[[File:Curtiss_Hawk_II_in_Chinese_service.jpg|thumb]]於筧橋中央航校上空]]
1930年，中華民國國民政府軍事委員會委員長[[蒋中正|蒋中正]]决定择址杭州笕桥，在原中央军校航空班的基础上扩建成中央航空学校。1931年春用蚕校(浙江理工大学)校舍建成，航校东部为机场、弹油库、机修厂、飞机制造厂等军事设施区，西部为运动场、办公楼、学生宿舍、别墅群等教学生活区。学校设立机构、采购[[飞机|飞机]]、招生办学，并先后在[[洛阳|洛阳]]、[[广州|广州]]设立分校。至1937年抗战前共培训了500余名学员。抗战爆发后，航校先后迁往[[云南|云南]][[昆明|昆明]]、[[巴基斯坦|巴基斯坦]][[拉合尔|拉合尔]]，并改名为'''中央空军军官学校'''，而[[笕桥机场|笕桥机场]]作为[[中华民国空军|空军]]的主基地使用。航校的许多毕业生在[[抗日战争|抗日战争]]中英勇奋战、为保衛中華民國而牺牲。<ref name="旧址" />

抗战胜利后学校又迁回杭州笕桥。1948年冬迁往[[台湾|台湾]]。1949年以后航校由[[中国人民解放军空军|中国人民解放军空军]]部队接管，仍作为航空学校和[[军用机场|军用机场]]使用。1957年改成军民两用机场。1972年为使[[美国总统|美国总统]][[尼克松访华|尼克松访华]]所使用的[[波音707|波音707]][[空军一号|空军一号]]能降落在机场，机场在3个月内通过“118工程”将原2200米长的跑道扩建至3200米，同时修建了建筑面积达5870平方米的候机楼，连同34000平方米的停机坪<ref>{{cite web|title=笕桥速度：杭州总动员|url=http://www.qnsb.com/news/index.php?m=content&c=index&a=show&catid=81&id=37557|publisher=浙青网|date=2013-05-30}}</ref>。2000年，由于杭州[[萧山机场|萧山机场]]建成使用，民航撤出笕桥机场。<ref name="旧址" />

学校的部分建筑在抗战中被破坏，后得以修复。目前航校旧址建筑基本保存完好，其建筑的时代特色对研究民国时期的军校建筑具有重要意义。<ref name="旧址" />

==问题==
===限高===
[[File:201611_City_Landscape_of_Hangzhou.jpg|thumb]]到[[钱江新城|钱江新城]]（远处高楼）约13公里路程形成了高度分明的建筑群]]
随着杭州市的发展，笕桥机场已被城市包围。一方面杭州城区大部分的建筑高度因为机场净空区限高的因素而无法超过299米<ref>{{cite news|title=钱江新城的天际线 很美|url=http://www.hangzhou.com.cn/20090629/ca1751977.htm|publisher=杭州网|date=2008-10-15}}</ref>；而另一方面空军则抱怨机场周围密度极高的民居对飞行安全、军事机密造成了隐患和威胁，经常在战机降落过程中因前方突然出现气球、烟花等而不得不复飞<ref>{{cite web|title=【风云】今日笕桥机场：被钢筋水泥包围|url=http://www.shobserver.com/news/detail?id=3324|publisher=上观|date=2015-01-17}}</ref>。

===噪声===
由于笕桥机场的跑道西侧正对杭州城西一带，因此飞行部队训练时的航线将经过这些住宅区密集的区域，[[战斗机|战斗机]]起降产生的巨大噪音引发了部分居民的不满（尤其以夜间为甚）。<ref>{{cite web|title=杭州城西受解放军笕桥机场战机噪音干扰区域一览|url=https://www.19lou.com/forum-111-thread-40638693-1-1.html|website=19楼}}</ref>

==事件==
*1965年11月11日，解放军空军第8师22大队李显斌驾驶[[伊尔-28|伊尔-28]]轰炸机从[[杭州|杭州]]笕桥机场起飞，驾驶飞机在[[中華民國|中華民國]][[国军与解放军间的驾机叛逃事件|桃園軍机场降落]]，降落時李显斌未將鼻輪放出，導致機首座艙內领航员李才旺雙腿骨折，尾舱通信员[[廉保生|廉保生]]著陸後自杀身亡。<ref>[http://udn.com/news/story/4/1989760 聯合報：叛逃變烈士 廉保生的回鄉路走了51年]。</ref>。
*1994年1月29日，[[中国东方航空|中国东方航空]]2318号航班在地面遭受劫持。杭州机场随后在地面制服飞机上两名劫机歹徒，成功粉碎一起劫机事件<ref>{{cite web|title=浙江省改革开放30年大事记-1994年|url=http://lishi.zhuixue.net/2016/1202/51142.html|date=2016-12-02|publisher=历史追学网}}</ref>。
*2000年2月22日，[[中国民航|中航]][[浙江航空|浙江航空公司]]一架[[龐巴迪Dash_8|龐巴迪Dash 8]]担当杭州飞往温州的F65576次航班在起飞15分钟、爬升至3300米后左侧发动机内部滑油泄漏，引起发动机失效自动停车。飞机随后紧急返回笕桥机场并安全降落。机上其中的15名乘客被随后被安排乘坐B-3352航班飞往温州，另有部分乘客次日乘坐空中客车航班继续前往温州<ref>{{cite news|title=图文：发动机出现故障 杭温航线昨演惊险一幕|url=http://news.sina.com.cn/china/2000-2-23/64409.html|publisher=新浪网|date=2000-02-23}}</ref><ref>{{cite news|title=杭州迫降小型客机故障原因查清|url=http://news.sina.com.cn/china/2000-2-24/64776.html|publisher=新浪网|date=2000-02-24}}</ref>。
