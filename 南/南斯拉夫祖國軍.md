'''南斯拉夫祖國軍'''（Jugoslovenska vojska u otadžbini，JVUO），是[[二次世界大戰|二次世界大戰]]期間於[[南斯拉夫|南斯拉夫]]地區活動的抗德游擊部隊。他們另一個較為人熟知的名字是「切特尼克（Četnici）」，意為「軍事部隊（military company）」，源自於反抗[[鄂圖曼|鄂圖曼]]的[[塞爾維亞|塞爾維亞]]與[[馬其頓|馬其頓]]部隊。

== 緣起 ==
1941年4月軸心國軍隊進入[[南斯拉夫王國|南斯拉夫]]，[[彼得二世_(南斯拉夫)|彼得二世]]流亡希臘後不久，當時仍是陸軍上校的[[德拉查·米哈伊洛維奇|德拉查·米哈伊洛維奇]]，集合仍留在南斯拉夫境內各地抵抗德軍的散落部隊，組成南斯拉夫祖國軍<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突(1991-2006)〉，碩士論文，東海大學政治系，頁21。</ref>。

== 發展 ==
這支部隊主要由[[塞爾維亞人|塞爾維亞人]]與[[蒙特內哥羅人|蒙特內哥羅人]]組成，效忠於南斯拉夫皇室，在1943年以前受到[[英國|英國]]的支持，被視為是[[同盟國_(第二次世界大戰)|同盟國]]的作戰部隊。戰爭初期6個月，切特尼克部隊與[[狄托|狄托]]領導的[[南斯拉夫人民解放军和游击队|南斯拉夫人民解放军]]協同作戰，但是後來由於目標不同而分道揚鑣，進而相互攻擊<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突(1991-2006)〉，碩士論文，東海大學政治系，頁21-22。</ref>。

== 消滅 ==
由於米哈伊洛維奇受到指控，不但沒有對[[軸心國|軸心國]]作戰，反而攻擊狄托的部隊。1943年的[[德黑蘭會議|德黑蘭會議]]後，英國政府立場轉變，由支持米哈伊洛維奇轉為支持狄托。隔年6月16日狄托與皇室政府代表{{Link-en|伊万·苏巴西奇|Ivan Šubašić|伊万·苏巴西奇}}在[[維斯島|維斯島]]簽訂了[[維斯條約|維斯條約]]，認可南斯拉夫人民解放軍取代南斯拉夫祖國軍為南斯拉夫流亡政府的軍隊。8月彼得二世更進一步解除米哈伊洛維奇祖國軍統帥的身分，改將這個職務授與狄托<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突(1991-2006)〉，碩士論文，東海大學政治系，頁22。</ref>。
