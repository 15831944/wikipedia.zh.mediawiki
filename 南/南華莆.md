'''南華莆（Nam Wa Po）'''位於[[香港|香港]][[新界|新界]][[大埔區|大埔區]]北部，[[粉嶺公路|粉嶺公路]]和[[大窩西支路|大窩西支路]]以西，為原居民村落，原居民為林姓或鄭姓。

== 南華莆林氏 ==

林氏原籍[[福建|福建]][[莆田|莆田]]<ref>[http://www.phr2012.com/wenxue/sanwen/190.html 香港明朝客家村(坑下莆)祖先来自福建漳州平和县] {{webarchive|url=https://web.archive.org/web/20150924072707/http://www.phr2012.com/wenxue/sanwen/190.html |date=2015-09-24 }}</ref>，[[林永盛|林永盛]]於1607年由[[東莞|東莞]][[莆崗|莆崗]]移居[[林村|林村]][[坑下莆村|坑下莆]]。至坑下莆無地發展，再建[[較寮下村|較寮下村]]。1906年，後人[[林大穩|林大穩]]與二子於[[泰亨|泰亨]]附近向[[香港新界文氏|文氏族人]]買地建村，是為南華莆村<ref>{{cite web |url=http://www.somanhing.com/gotowalk/dist/taipo/lamchuen/lam.pdf|title=林氏宗祠}}</ref>。

== 景點 ==
村內特色建築，為崇真會[[南華莆崇真堂|南華莆崇真堂]]，原有堂舍於1953年建成<ref>[http://www.oocities.org/hk/tphistory/history_church02.htm 大埔今昔 南華莆崇真堂]</ref>，現有堂舍於2007年按原貌重建。
