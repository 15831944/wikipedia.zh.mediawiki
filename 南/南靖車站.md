'''南靖車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[水上鄉|水上鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線南段]]及[[臺灣糖業股份有限公司|臺灣糖業股份有限公司]]南靖線（廢止）的[[鐵路車站|鐵路車站]]，位置距離基隆起點301公里。本站為嘉義縣最南端的車站，車站南方為[[八掌溪|八掌溪]]。

== 車站構造 ==
本車站站體結構為[https://ja.wikipedia.org/wiki/%E5%A4%A7%E5%A3%81 「大壁式」]木構造，是一種仿磚石牆身的西方木構造形式。用木框架和編竹夾泥牆或鐵絲網鏝灰泥的方式，去仿造磚牆的牆量體。
由於本站重建時間位於二戰末期，物資缺乏，採用此構造可以短時間完成外觀較雄偉的建築物。東部幹線的關山舊站、第二代花蓮港站也是採用此種構法。

*[[島式月台|島式月台]]、[[岸式月台|岸式月台]]各一座。

== 利用狀況 ==
*目前為甲種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，主要停靠車種以[[臺鐵區間車|區間車]]為主。
*本站無任何對號班次停靠。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
* 本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[Icash|icash2.0]]以及[[有錢卡|HappyCash]]進出站。

== 歷史 ==
*1911年3月20日：配合南靖糖廠興建，乃將距基隆端304.4公里的'''水窟頭驛'''遷移至現址。<ref name=":0">{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>
*1920年：所在地地名由「水窟頭」改為「水上」，驛名亦隨之變更。<ref name=":0" />
*1941年12月17日：嘉南地區發生大地震，水上驛木造站房被震毀，隔年開始重建。<ref name=":0" />
*1943年3月：現存站房落成。
*1949年8月10日：於距基隆端304.4公里復設「水頭站」，10月改名為[[水上車站|水上站]]，同時將原「水上站」改名為「'''南靖'''」。
*1986年6月1日：由三等站降為甲種簡易站，並指定由[[嘉義車站|嘉義站]]管理。<ref name=":0" />
*2013年9月30日：啟用多卡通刷卡機。
