'''南非山兔'''（[[學名|學名]]''Bunolagus monticularis''），又名'''灌叢穴兔'''，是最為稀有及[[瀕危|瀕危]]的[[哺乳動物|哺乳動物]]，數量大約只200隻。牠們只分佈在[[南非|南非]][[開普省|開普省]]高原中部及南部。

== 特徵 ==
南非山兔由嘴角橫跨兩頰有一道黑色斑紋。[[腹部|腹部]]及[[喉嚨|喉嚨]]呈奶白色，後腳很闊。尾巴褐色，末端黑色。

== 棲息及分佈地 ==
南非山兔只分佈在[[南非|南非]]高原的一些地區，但這些地區都並非保護區。在[[弗雷澤堡|弗雷澤堡]]（Fraserburg）、[[薩瑟蘭|薩瑟蘭]]（Sutherland）及[[西維多利亞|西維多利亞]]（Victoria West）也有細小的群落。

== 行為 ==
南非山兔是夜間活動的，日間則會休息。牠們夜間的[[排泄|排泄]]物很硬，而日間的則柔軟。可見牠們會從消化道內的[[細菌|細菌]]吸收[[維他命B|維他命B]]，而[[鈣|鈣]]及[[磷|磷]]等[[礦物質|礦物質]]則會重用。
