'''南锣鼓巷站'''位于[[北京市|北京市]][[东城区_(北京市)|东城区]]，是[[北京地铁|北京地铁]][[北京地铁6号线|6号线]]和[[北京地铁8号线|8号线]]的一座[[换乘站|换乘站]]。为北京地铁首个[[侧式叠式站台|叠落式布置]]的车站<ref>{{cite news|url=http://www.bjnews.com.cn/news/2010/10/14/75505.html|title=地铁南锣鼓巷站将“叠落”运行|work=新京报|date=2010-10-14|accessdate=2014-01-09}}</ref>。车站以“[[南锣鼓巷|南锣鼓巷]]”{{noteTag|南锣鼓巷因保存完好的四合院民居成为著名的北京胡同文化风情体验区。}}命名。6号线车站随着6号线一期工程于2012年12月30日开通投入运营；而8号线车站随着8号线二期工程南段剩餘线路于2013年12月28日投入运营。

== 位置 ==
南锣鼓巷站位于[[地安门东大街|地安门东大街]]（[[平安大街|平安大街]]，东西向）和[[南锣鼓巷|南锣鼓巷]]（南北向）的交汇处。<ref>{{cite web|url=http://www.bjghw.gov.cn/web/static/articles/catalog_14300/2010-01-19/article_ff808081262b02de01262c2fa1a90005/ff808081262b02de01262c2fa1a90005.html|title=轨道交通6号线一期工程规划方案公告|publisher=[[北京市规划委员会|北京市规划委员会]]|date=2010-01-14|accessdate=2010-04-03|deadurl=yes|archiveurl=https://web.archive.org/web/20100408065020/http://www.bjghw.gov.cn/web/static/articles/catalog_14300/2010-01-19/article_ff808081262b02de01262c2fa1a90005/ff808081262b02de01262c2fa1a90005.html|archivedate=2010-04-08}}</ref><ref>{{cite web|url=http://www.bjghw.gov.cn/web/static/articles/catalog_14300/2009-10-19/article_ff80808123fa497b0124528b720f000a/ff80808123fa497b0124528b720f000a.html|title=地铁8号线二期规划方案公告|publisher=[[北京市规划委员会|北京市规划委员会]]|date=2009-10-14|accessdate=2010-04-03|deadurl=yes|archiveurl=https://web.archive.org/web/20100327013929/http://www.bjghw.gov.cn/web/static/articles/catalog_14300/2009-10-19/article_ff80808123fa497b0124528b720f000a/ff80808123fa497b0124528b720f000a.html|archivedate=2010-03-27}}</ref> 地处地安门东大街与[[北河沿大街|北河沿大街]]路口的西侧。两线车站均为东西走向，沿地安门东大街南北两侧，6号线车站偏南，8号线车站偏北，呈平行状布置{{noteTag|8号线在地安门路口由南北走向向东拐弯，与6号线并行，从西向东行至南锣鼓巷站，出站后两条线路分别沿北河沿大街、美术馆后街向南拐弯，因此，两条线路在南锣鼓巷站呈平行状态。}}。

== 结构 ==
受地理条件限制以及出于文物保护原因{{noteTag|南锣鼓巷站址受西端地安门东大街南侧的基督教宽街教堂和北侧的[[东不压桥|东不压桥]]遗址制约。}}，地安门东大街下方地下空间有限无法满足四线并行要求，故两线双向轨道、站台上下叠摞，左右线隧道在南锣鼓巷站呈叠落状，出站后左右线隧道逐渐分离，最终等高并行，站台亦采用[[侧式叠式站台|侧式叠式站台]]。为了避免施工对地安门东大街的影响，8号线车站位于道路北侧，6号线车站位于道路南侧，站位稍微相互错开布置。车站采用类似[[分离式站台|分离式站台]]的形式。

6号线为地下三层车站，总长210米，出于6号线将沿[[北河沿大街|北河沿大街]]转向南拐弯的原因（亦或为规避地上文物），轨道线路偏向东南方向，车站平面呈楔形，东端宽度近50米，西端宽度12米<ref>{{cite news|url=http://www.fawan.com/Article/bs/jd/2012/05/16/141817156035.html|title=南锣鼓巷将增地铁换乘站|work=法制晚报|date=2012-05-16|accessdate=2014-01-09|archive-url=https://web.archive.org/web/20131112153958/http://www.fawan.com/Article/bs/jd/2012/05/16/141817156035.html|archive-date=2013-11-12|dead-url=yes}}</ref>，6号线车站地下1层为站厅层，地下2、3层为“叠摞”布局的站台层，6号线往五路居的线路位于地下二层，往草房方向的线路位于地下三层；而8号线车站由于受到拆迁限制，建设为地下四层车站，8号线车站地下1层为风道和设备间，地下2至4层与6号线车站地下1至3层高度相同，功能也一致，8号线往朱辛庄的线路位于地下三层，往中国美术馆的线路位于地下四层。两线站厅层中部的两处扶梯直接连接到下层站台层，站厅层东、西两侧扶梯连接上层站台层，两层站台之间也有楼梯连通。两线之间上下层同向站台均以兩條长约百米的通道在6號綫站台東端和8号线站台西端相连，实现近似[[跨月台換乘|跨站台換乘]]。<ref>{{cite journal|author=任云|title=地铁8号线二期工程车站建筑功能分析|journal=现代城市轨道交通|year=2011|issue=1}}</ref><ref>{{cite journal|author=曹宗豪|title=北京地铁6号线换乘车站设计特色|journal=都市快轨交通|year=2013|volume=26|issue=1}}</ref><ref>{{cite news|url=http://roll.sohu.com/20120516/n343274687.shtml|title=6号线南锣鼓巷站主体完工 明年换8号线不必爬楼梯|date=2012-05-16|accessdate=2012-05-16}}</ref> 

乘客从6号线下行开往海淀五路居站方向站台层可直接换乘到8号线上行开往朱辛庄站方向站台层，6号线上行开往草房站方向站台层可直接换乘到8号线下行开往中国美术馆车站方向站台层，每層兩條换乘通道采取单向换乘，乘客若到反向站台层可以通过站台层之间互通的方式到达。<ref>{{cite news|url=http://www.takefoto.cn/?action-viewnews-itemid-57931|title=北京地铁8号线南锣鼓巷站首班车5点半|work=北京晚报|date=2013-12-26|accessdate=2014-01-10}}</ref>

== 出入口 ==
南锣鼓巷站有四个出口，临近6号线车站的A口、B口位于平安大街以南，临近8号线车站的E口、F口位于平安大街以北。南锣鼓巷站由E口出站既是南锣鼓巷南口。
由于南锣鼓巷属旅游点，高峰客流极大时，8号线南锣鼓巷站E口和F口将实行单进单出。

== 装饰 ==
由于车站位于北京市保存最完整的[[四合院|四合院]]区，因此站内设计风格突出展现了老北京民居特色与风俗文化。车站装饰仿照四合院的[[灰砖|灰砖]]、[[檩|檩条]]、[[砖雕|砖雕]]等元素。<ref>{{cite news|url=http://roll.sohu.com/20120511/n342903809.shtml|title=地铁6号线一期车站装修全面展开|date=2012-05-11|accessdate=2012-05-11}}</ref> 6号线车站内部装修是以灰色为主基调的北京传统民居胡同造型；8号线车站内部装修是以灰色、红色为主打色调的灰墙、红柱样式。

站厅、站台墙壁装饰有“城市记忆”、“时光绘印”、“南锣印象”、“北京·记忆”为题材的公共艺术设计作品。<ref>{{cite web|url=http://www.bjghw.gov.cn/web/dtggyshz/dtggys_list8.html|title=2012年度地铁公共艺术品建成展示--8号线|publisher=北京市规划委员会|accessdate=2014-01-09}}。</ref> 8号线车站装饰墙名为“北京·记忆”的公共艺术作品由4000多个6厘米见方琉璃块组成，琉璃块中内藏二维码用来收录向市民征集的承载对城市记忆的小物件。

== 其他 ==
6号线南锣鼓巷站施工区域内保留有三处未拆迁的居民院。为兼顾工期，由明挖施工改成了半暗挖施工，东段50米长的站体改为暗挖施工。<ref>{{cite news|url=http://www.people.com.cn/h/2012/0516/c25408-4154874862.html|title=“南锣”成首座地铁双向同台换乘站|author=涂露芳|work=北京日报|date=2012-05-16|accessdate=2014-01-09}}</ref>

6号线开通首日，有网友在微博发布消息称，南锣鼓巷站内的一处墙体倒塌，引起很多市民感到不安。经地铁公司通报澄清，在当日22时52分倒塌的是正在施工中的6号线与8号线换乘通道的泡沫砖围挡，只是施工中属临时搭建的隔墙并非站台结构的墙体。<ref>{{cite news|url=http://www.bjnews.com.cn/news/2013/01/01/242511.html|title=6号线开通首日临时围挡坍塌|work=新京报|date=2013-01-01|accessdate=2014-01-09}}</ref>

由于中国美术馆站施工缓慢，暂缓开通，故8号线二期南段开通后，南锣鼓巷站成为8号线南端的终点站，列车在南锣鼓巷站清客后空车行驶到中国美术馆站折返<ref>{{cite news|url=http://www.morningpost.com.cn/ttxw/2013-12-26/539732.shtml|title=8号线延长连上昌平线|date=2013-12-26|accessdate=2014-05-23|work=北京晨报|deadurl=yes|archiveurl=https://web.archive.org/web/20140523231037/http://www.morningpost.com.cn/ttxw/2013-12-26/539732.shtml|archivedate=2014-05-23}}</ref>。预计中国美术馆站将随8号线三期北段在2017年一同开通<ref>{{cite news|url=http://bjwb.bjd.com.cn/html/2013-12/25/content_137304.htm|title=8号线南北新线周六开通|date=2013-12-25|accessdate=2014-05-23|work=北京晚报|author=孟环|deadurl=yes|archiveurl=https://web.archive.org/web/20140523230705/http://bjwb.bjd.com.cn/html/2013-12/25/content_137304.htm|archivedate=2014-05-23}}</ref><ref>{{cite news|url=http://bjrb.bjd.com.cn/html/2013-11/23/content_127899.htm|title=8号线二期南段步行接驳优先|date=2013-11-23|accessdate=2014-05-25|work=北京日报|author=耿诺|deadurl=yes|archiveurl=https://web.archive.org/web/20140525213912/http://bjrb.bjd.com.cn/html/2013-11/23/content_127899.htm|archivedate=2014-05-25}}</ref>，届时南锣鼓巷站将不再是终点站。
