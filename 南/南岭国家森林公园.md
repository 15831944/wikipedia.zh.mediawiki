'''南岭国家森林公园'''位于中國廣東省[[韶關市|韶關市]][[乳源瑶族自治县|乳源瑶族自治县]]，近湖南[[莽山|莽山]]，地處[[南岭山脉|南岭山脉]]的核心地带，是广东省最大的自然保护区，總面積約273平方公里<ref>{{Cite web |url=http://www.sgta.gov.cn/2010/0114/1544.html |title=南岭国家森林公园 |date=2010-01-14 |publisher=韶关市旅游局 }}</ref>。是[[珠江|珠江]]支流[[北江|北江]]的发源地。海拔1902米的[[石坑崆|石坑崆]]是[[广东省|广东省]]最高峰。

2012年，獲評定為[[国家4A级旅游景区|国家4A级旅游景区]]。<ref>{{Cite web |url=http://www.gdta.gov.cn/gzdt/dsdt/7407.html |title=韶关市乳源丽宫国际温泉旅游区、南岭国家森林公园成功创建国家4A级景区 |date=2012-03-06 |publisher=广东省旅游局 |deadurl=yes |archiveurl=https://web.archive.org/web/20150623202622/http://www.gdta.gov.cn/gzdt/dsdt/7407.html |archivedate=2015-06-23 }}</ref>

==景區==
南岭国家森林公园內劃分了四個景區，分別為[[小黃山|小黃山]]、瀑布群、親水谷和[[石坑崆|石坑崆]]。<ref>{{Cite web |url=http://www.nlslgy.com/guide/spot-introduce.html |title=景区介绍-南岭国家森林公园}}</ref>
===小黃山===
小黃山海拔1600米，為世界上最大片廣東松原始森林。頂峰為乳峰，入口處種有迎客松，并建有乳峰閣。
===瀑布群===
瀑布群入口處為九重山，水源于發財嶺，共有八條主要瀑布，包括千米瀑、孔雀瀑、飞流瀑、惊心瀑、虎口瀑、清心瀑、音韵瀑和双飞瀑，全程2200米。
===親水谷===
親水谷的主要景點有飞花潭、珍珠潭、九曲潭、青松潭、通幽峡、仙女潭以及卧龙峡。裏面保存了[[壶穴|壶穴]]地貌。
===石坑崆===
{{main|石坑崆}}
石坑崆海拔1902米，為廣東最高山峰，景點包括金蟾望月和空军雷达站旧址。
