'''南昌地铁4号线'''（官方文件正式名称：南昌市轨道交通4号线）是南昌一条拟建的轨道交通线路，该线路规划从望城到城东。

== 简介 ==
4号线代表色为南昌轨道绿{{color|#{{南昌地铁识别色|4}}|■}}

== 历史 ==
* 2008年9月15日开始公示5条线路的规划中包括4号线<ref>[http://www.lsnc.cn/gs/gdbgs.html 南昌市城市快速轨道交通线网及建设规划公众征询意见] {{webarchive|url=https://web.archive.org/web/20130720103017/http://www.lsnc.cn/gs/gdbgs.html |date=2013-07-20 }}</ref><ref>[http://www.jxgdw.com/jxgd/xwzxztmb/2008jxx/knc/userobject1ai845634.html 南昌地铁远景规划征询民意 将有5条地铁线路] {{webarchive|url=https://web.archive.org/web/20160303222001/http://www.jxgdw.com/jxgd/xwzxztmb/2008jxx/knc/userobject1ai845634.html |date=2016-03-03 }}</ref><ref>[http://jiangxi.jxnews.com.cn/system/2008/10/16/002860601.shtml 南昌地铁远景规划征询民意(图)]</ref>。
* 2009年7月29日确认4号线作为远期规划将根据实际情况而建设<ref>{{Cite web |url=http://www.ncnews.com.cn/ncxw/bwyc/t20090729_469995.htm |title=南昌地铁线路站点公布 1号线2014年通车 |access-date=2013-01-02 |archive-url=https://web.archive.org/web/20140219125225/http://www.ncnews.com.cn/ncxw/bwyc/t20090729_469995.htm |archive-date=2014-02-19 |dead-url=yes }}</ref>。
* 2013年1月30日公示《南昌市城市快速轨道交通线网规划调整》中2号线规划了新的南延计划，原西延并入本线<ref>[http://ghj.nc.gov.cn/qtpqgg/3782.jhtml 南昌市城市快速轨道交通线网规划调整（九龙湖段）—2号线延长线选线方案]</ref>。
* 2013年4月市政府表示将4号线的建设计划提前<ref>[http://jiangxi.jxnews.com.cn/system/2013/04/11/012366266.shtml 南昌考虑再建五条地铁线 拟规划到十号线形成半小时经济圈]</ref><ref>[http://jiangxi.jxnews.com.cn/system/2013/04/17/012375524_02.shtml 南昌市市长陈俊卿：2015年五条地铁全部开建]</ref>。
* 2013年5月底政府规划第二轮地铁建设的三个方案，三个方案都将建设4号线<ref>[http://jndsb.jxnews.com.cn/system/2013/05/31/012446645.shtml 明年进入第二轮地铁建设期]</ref><ref>{{Cite web |url=http://www.ncnews.com.cn/ncxw/jrnc/t20130531_1024778.htm |title=南昌未来7年有望建成3至4条地铁 |access-date=2013-08-11 |archive-url=https://web.archive.org/web/20130909090344/http://www.ncnews.com.cn/ncxw/jrnc/t20130531_1024778.htm |archive-date=2013-09-09 |dead-url=yes }}</ref><ref>[http://jiangxi.jxnews.com.cn/system/2013/05/30/012446400.shtml 专家把脉南昌地铁第二轮建设规划 三套方案备选]</ref>。
* 2013年6月1日待审批的《南昌市城市快速轨道交通建设规划（2014-2020）》中明确了4号线一期的走向，西站到望城段也将纳入一期计划中，计划最快于2015年9月进入土建阶段，于2019年12月通车<ref>[http://jiangxi.jxnews.com.cn/system/2013/06/02/012449170.shtml 地铁3号线一期工程明年下半年或开工]</ref>。
* 2013年8月15日最新的规划确定4号线一期工程调整长度由35.2公里缩短为32.5公里，但起讫站点未变动<ref>{{Cite web |url=http://www.ncnews.com.cn/ncxw/jrnc/t20130815_1055647.htm |title=南昌地铁1-4号线最新规划出炉 4号线线路有调整 |access-date=2013-08-15 |archive-url=https://web.archive.org/web/20140203231539/http://www.ncnews.com.cn/ncxw/jrnc/t20130815_1055647.htm |archive-date=2014-02-03 |dead-url=yes }}</ref>。
* 2014年1月23日3、4号线优化调整规划批前公示，[[南昌地铁3号线|3号线]]和4号线进行较大调整，4号线在洪城路以北被分成三段做不同变动，原洪城路站以西段向北移动到桃苑大街、站前西路，4号线不经过洪城路站而改从[[绳金塔站|绳金塔站]]与3号线换乘，[[彭家桥站|彭家桥站]]、[[洪都中大道站|洪都中大道站]]等一段被取消，位于火炬大道部分成为3号线的一部分；在[[绳金塔站|绳金塔站]]换乘后改为在[[京九铁路|京九铁路]]西侧往北在[[丁公路南站|丁公路南站]]与2号线换乘，在[[丁公路北站|丁公路北站]]与1号线换乘，在二七北路站与3号线换乘后承接原3号线北段<ref>[http://jiangxi.jxnews.com.cn/system/2014/01/23/012919119.shtml 南昌地铁3、4号线规划调整 4号线调整较大 线路增加8公里]</ref><ref>[http://ghj.nc.gov.cn/qtpqgg/6540.jhtml 南昌市轨道交通3、4号线优化调整规划批前公示]</ref>。
* 2014年1月26日的2014-2020建设规划显示4号线将设年限从2015年到2020年<ref>[https://web.archive.org/web/20140205091417/http://www.ncmtr.com/newsshow.php?id=1778 南昌市城市快速轨道交通建设规划（2014-2020）社会稳定风险分析公众参与信息公告]</ref>。
* 2014年3月21日公布的“南昌市城市快速轨道交通建设规划（2014-2020）环评”显示4号线详细走向<ref>[https://web.archive.org/web/20140324131027/http://www.ncmtr.com/newsshow.php?id=1882 南昌市城市快速轨道交通建设规划（2014-2020）环评公众意见征询]</ref>。。

== 未来发展 ==
2008年在最初的规划中4号线计划东起艾溪湖西侧的火炬大街，西至[[南昌西站|南昌西站]]，规划长度为23km<ref>[http://jiangxi.jxnews.com.cn/system/2008/10/16/002860601.shtml 南昌地铁远景规划征询民意(图)]</ref>；当时还计划过江前在在生米大桥东南处延伸出一条支线继续南行到小兰工业区西侧，这条支线计划在后续规划中不在出现。
* 4号线目前的规划以及2号线交给的望城延伸段均被纳入一期工程中，尚无延长计划。
