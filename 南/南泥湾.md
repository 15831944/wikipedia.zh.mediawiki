'''南泥湾'''，位于[[陕西省|陕西]][[延安市|延安]]城东南45公里处，1939年2月针对[[國民政府|国民政府]]和[[日本帝国陆军|侵华日军]]的封锁，[[毛澤東|毛澤東]]提出了“自己动手，丰衣足食”的口号，动员根据地军民开展[[大生产运动|大生产运动]]。

1941年3月，[[八路军|八路军]][[国民革命军第十八集团军第120师|120师]]三五九旅在[[王震|王震]]率领下，在南泥湾进行[[军垦|军垦]]，以满足根据地的物资供给。

== 地理环境 ==
[[File:Nanniwan_today.jpg|thumb]]
'''南泥湾'''，位于[[陕西省|陕西]][[延安市|延安]]城东南45公里处，抗日战争时期归属[[中国共产党|中共]][[陕甘宁边区|陕甘宁边区]]管辖，根据1943年《南泥湾调查》报告：“南泥湾现在是延安县金盆区的一个乡，地面辽阔，西界鄜，甘二县境延水与洛水分水岭的大动脉，北靠延水与临镇川的分水岭，东与金盆湾本区接壤，南靠固临境内，黑蛇川与临镇川的分水岭，纵横各约八十里，计六千四百方里，中心地区由三道河川构成：南盘龙川自西向东月三十五里，九龙川自南而北约二十余里，在南泥湾回合称南阳府川，东北向约二十五里至金盆湾。”<ref name=":0">《南泥湾调查》，《中共中央西北局文件汇集》一九四三年（一），第268-269页，中央档案馆/陕西省档案馆，1994年11月出版</ref>农场管理处估计，可耕地约百万亩，川地一万五千亩，其余为广大森林区。<ref name=":0" />1940年人口仅2、30户<ref name=":0" />

== 大生产运动 ==
[[File:1943年毛泽东《南泥湾》题词.jpg|thumb]]1939年2月针对[[國民政府|国民政府]]和[[日本帝国陆军|侵华日军]]的封锁，[[毛澤東|毛澤東]]提出了“自己动手，丰衣足食”的口号，动员根据地军民开展[[大生产运动|大生产运动]]。

1940年6月，乐天宇率领森林考察团，考察森林情况时，发现南泥湾地区，同年末，中央财经处计划开发，将管辖由固临县拨给新中国大农场经营。<ref name=":0" />

1941年初，南泥湾地区仅有4个农场，至七月，增加到14个<ref name=":0" />，

1941年3月，[[八路军|八路军]][[国民革命军第十八集团军第120师|120师]]三五九旅在[[王震|王震]]率领下，在南泥湾进行[[军垦|军垦]]，以满足根据地的物资供给。五月，八团到金盆湾，七月，炮兵团到南盘龙川，“皆曾种了一些地”，十一月警卫营开到，1941年居民增至101户。<ref name=":1">《南泥湾调查》，《中共中央西北局文件汇集》一九四三年（一），第270页，中央档案馆/陕西省档案馆，1994年11月出版</ref> 三五九旅采用刀耕火种的原始烧荒方式，焚毁森林，烧木肥土，使南泥湾森林消失殆尽<ref name=":3" />

1942年，成立农场管理处，三月，特务团到驻马坊，六月，九团驻九龙川，八月，四支队来南泥湾，“皆曾种地，但中心在种[[特货|特产]]”<ref name=":1" />，1942年农场增加到30余个，居民增至243户。

== 南泥湾地区的鸦片种植 ==
{{See also|特货}}

自从[[陳永發|陈永发]]《红太阳下的罂粟花》出版后，舆论出现中共大生产运动中种植的作物包括粮食<ref>{{cite web|url=http://www.gs.xinhuanet.com/dfpd/2005-11/01/content_5484505.htm|title=民主革命时期的镇原县边区政府|publisher=新华网|date=2005-11-01|accessdate=2014-03-03|deadurl=yes|archiveurl=https://web.archive.org/web/20120829213641/http://www.gs.xinhuanet.com/dfpd/2005-11/01/content_5484505.htm|archivedate=2012-08-29}}</ref>及[[特货|特货]]（即[[鸦片|鸦片]]）<ref>{{cite journal|title=展望|journal=展望雜誌社|year=1972|issue=第65-78期|page=28页}}</ref><ref>{{cite book|author=吴殿尧, 宋霖|title=《朱理治传》|year=2007年|location=中共党史出版社|isbn=9787801997425|pages=366页}}</ref>的怀疑。种植[[罂粟|罂粟]]的原因，是因为当时中国鸦片被广泛当做[[硬通货|硬通货]]使用<ref>陕甘宁边区财政经济史编写组、陕西省档案馆编：《日战争期间陕甘宁边区财政经济史资料摘编》，陕西人民出版社1981年版，第四册，第438-439页</ref>，陕甘宁边区物产贫瘠，种植鸦片出口用于平衡外贸。国民政府方面也一直在指责中共贩毒<ref>{{cite web|title=南泥湾的“大生产”原来是在种鸦片|url=http://www.epochtimes.com/gb/12/8/17/n3661334.htm|website=大紀元|accessdate=2017-03-18}}</ref>。

根据大陆学者张耀杰的研究，[[毛泽东|毛泽东]]亲笔题词的[[张思德|张思德]]，并非死于烧木炭，而是在制鸦片的过程中意外身亡，虽未证实，并引起反驳<ref>{{Cite web|url=http://www.guokr.com/post/442951/|title=辟谣：张思德竟然是熬鸦片时死的？|accessdate=2013-02-27|author=|date=|publisher=}}</ref>，也印证存在南泥湾地区鸦片种植的争议。

南泥湾地区也有鸦片种植区域：
* 1942年特务团、九团、四支队来南泥湾“皆曾种地，但中心在种[[特货|特产]]”<ref name=":1" />。在1942年各单位中底面积调查表中，八团种植1000亩，收烟300两，特务团收烟100两，警卫团收烟800两，炮兵团种植六百亩只出八十亩，收烟几十两。<ref name=":2">《南泥湾调查》，《中共中央西北局文件汇集》一九四三年（一），第272页，中央档案馆/陕西省档案馆，1994年11月出版</ref>“第一，贪多；第二缺乏经验，未好好的采访老百姓的意见，如炮兵团下种时，老百姓说太早，种了不出来，我们不听，结果六百亩只出了八十亩。其他是施肥割烟皆不熟悉……”。<ref name=":2" />
* 时任三五九旅九团九连连长，后长期担任南泥湾生产大队长的刘宝斋所述，所在部队在南泥湾孟酒沟种大烟，用多余的粮食酿酒，烟酒贩到国统区卖钱，“为这事常同机关的打架。连队要挣钱，机关要收税。一次往河东贩卖烟土，货上了船，船帮上一圈端刺刀的战士，看谁敢挡……”。<ref name=":3">{{Cite book|title=延安一片月|last=高红十|first=|date=2012年1月|publisher=武汉大学出版社|year=|isbn=978-7-307-09237-2|location=|pages=第182页|chapter=哭刘老}}</ref><ref>{{Cite web |url=http://www.zgdsw.org.cn/GB/218989/14910072.html |title=党史动态--中国共产党历史网--人民网 |accessdate=2017-10-18 |last=407 |work=www.zgdsw.org.cn}}</ref>
* 1947年胡宗南进攻延安，从陇东佯攻，主力从宜川北上延安，走多年未走小路奇袭。南泥湾的金盆湾是其中路线之一，国军参谋人员化妆为鸦片走私商人进入边区，“沿路贩卖西药和布料，归途带鸦片回来”，确认了路线通畅<ref>{{Cite book|title=盛文先生口述历史|last=张朋园|first=|publisher=九州出版社|year=2013年|isbn=978-7-5108-1826-4|series=中研院近代史研究所口述历史系列|location=北京|pages=第66页|chapter=攻延安的部署}}</ref>
