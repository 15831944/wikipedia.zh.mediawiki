'''南高史地研究會'''，民國9年（1920）5月13日成立於[[南京高等師範學校|南京高等師範學校]]（後續更名[[國立東南大學|國立東南大學]]、[[國立中央大學|國立中央大學]]），由民國8年（1919）10月1日成立之南京高師地學研究會擴改而成。民國10年（1921）國立東南大學建立，改稱「南高東大史地研究會」；民國12年南高併入東大，改稱「東南大學史地研究會」。通常統稱為「南高史地研究會」，學界亦以此為學派稱為「'''南高史地學派'''」。《史地學報》是南高史地研究會創辦的著名學術刊物。

== 人物 ==
歷任總幹事：[[諸葛麒|諸葛麒]]、[[陳訓慈|陳訓慈]]、[[胡煥庸|胡煥庸]]、[[向達|向達]]。另，[[龔勵之|龔勵之]]為南高史地研究會前身地學研究會總幹事。

指導員：[[柳詒徵|柳詒徵]]、[[竺可楨|竺可楨]]、[[梁啟超|梁啟超]]、[[徐則陵|徐則陵]]、[[白眉初|白眉初]]、[[王毓湘|王毓湘]]、[[陳衡哲|陳衡哲]]、[[顧泰來|顧泰來]]、[[朱進之|朱進之]]、[[蕭純錦|蕭純錦]]、[[曾膺聯|曾膺聯]]、[[杜景輝|杜景輝]]等。

南高史地研究會，除指導員多學界翹楚外，其成員及弟子對中國歷史學、地理學等方面的發展貢獻極巨，湧現出了多位中國現代歷史、地理諸多學術領域的奠基者、開拓者，如張其昀（人文地理學，歷史地理學、政治地理學、國家戰略學等），胡煥庸（人文地理學與自然地理學，人口地理學等），鄭鶴聲（中國史學史、中國近代史、中西交通史、中外關係史等），王庸（歷史地圖學等），等等。

== 《史地學報》 ==
《史地學報》 於民國9年（1920）創刊，初為季刊，自二卷二期始改月刊（寒暑假無），民國14年（1925）一度中斷，至民國15年（1926）10月後停刊，共出版4卷21期。《史地學報》 列為南京高師叢書，由上海之商務出版社出版。

歷任總編輯/編輯主任：[[張其昀|張其昀]]、[[陳訓慈|陳訓慈]]、[[繆鳳林|繆鳳林]]、[[陸維釗|陸維釗]]。

《史地學報》為南高史地學會之機關刊物。除此之外，南高史地學人在不同時期還主辦過《史學與地學》、《地理學雜誌》、《方志月刊》、《史學雜誌》、《國風》、《史地雜誌》等刊物。南高史地學人也是《[[學衡|學衡]]》雜誌的重要作者群。

== 中國史地學會、中國史學會 ==
民國15年（1926），柳詒徵、竺可楨、繆鳳林、陳訓慈、張其昀、鄭鶴聲、劉掞藜、范希曾、向達、黃靜淵、胡煥庸、陸鴻圖等人在[[南京|南京]]發起成立中國史地學會，同年創刊《史學與地學》雜誌。

民國18年（1929）1月，組織成立南京中國史學會，同年3月創刊《史學雜誌》。南高學人陳訓慈、張其昀、繆鳳林等人先後推動在[[杭州|杭州]]成立了浙江中華史地學會、在[[上海|上海]]成立了吳越史地研究會。直到民國32年（1943），全國性的[[中國史學會|中國史學會]]才在南京[[國立中央圖書館|國立中央圖書館]]成立。在此之前，民國22年（1933），竺可楨、張其昀和[[翁文灝|翁文灝]]等人發起在南京成立了[[中國地理學會|中國地理學會]]。

== 南高史地學派 ==
學界以南高史地學會為學派，稱為「南高史地學派」，但有學者認為南高史地學會為學術組織，為非專持某種主張的派别。南高史地學派是南高學派的重要代表，南高學人的《史地學報》、《文哲學報》、《學衡》，以及其他雜誌如《民心週報》、《經世報》、《亞洲學術雜誌》、《湘君》等，是當時北大《新青年》新文化運動等的主要批評者。南高史地學會成員和[[學衡派|學衡派]]、[[國風社|國風社]]有密切關係。南高史地學人是[[學衡|學衡]]派的重要構成者，柳詒徵、繆鳳林等均為《學衡》雜誌的重要撰稿人。
