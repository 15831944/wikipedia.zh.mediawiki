'''南海爭端'''指[[南中國海|南中國海]]周边的幾個國家因為對於該海域的部分島嶼的主權归属、海域划分和相关海洋权利的聲索產生重疊從而發生的衝突。涉及該爭端的國家及地区有[[中華人民共和國|中華人民共和國]]、[[中華民國|中華民國]]、[[汶萊|汶萊]]、[[馬來西亞|馬來西亞]]、[[印尼|印尼]]、[[菲律賓|菲律賓]]和[[越南|越南]]。

南中國海領土爭端包括[[南沙群島|南沙群島]]（見[[南沙群島主權爭議|南沙群島主權爭議]]）、[[西沙群島|西沙群島]]和[[中沙群島|中沙群島]]的主權糾紛、[[北部灣|北部灣]]海上邊界問題以及其他地區的爭議。後來有關位於印尼和馬來西亞附近的[[納土納群島|納土納群島]]的主權歸屬問題也被歸類進該爭端範圍中<ref>{{cite news |last=Keck |first=Zachary |date=20 March 2014 |title=China’s Newest Maritime Dispute |url=http://thediplomat.com/2014/03/chinas-newest-maritime-dispute/ |newspaper=The Diplomat |access-date=12 February 2015 }}<br />{{cite news |last=Vaswani |first=Karishma |date=19 October 2014 |title=The sleepy island Indonesia is guarding from China |url = http://www.bbc.com/news/world-asia-29655874 |newspaper= |location= |access-date=19 October 2014 }}<br />{{cite news |last=R.C. Marshall |first=Andrew |date=25 August 2014 |title=Remote, gas-rich islands on Indonesia's South China Sea frontline |url = http://www.reuters.com/article/2014/08/25/us-southchinasea-indonesia-natuna-insigh-idUSKBN0GP1WA20140825 |newspaper=Reuters |access-date=12 February 2015 }}</ref>。各個國家出於自身利益，希望控制自己所主張擁有主權的島嶼的實際控制權，其中包括島嶼附近的專屬經濟區和捕撈地帶。由於懷疑南中國海的油氣蘊藏十分豐富，當事國也紛紛加入爭奪行列。其他一些爭奪該地區主權的理由有重要的航運通道，戰略控制等。

[[香格里拉對話|香格里拉對話]]是相關國家為解决該地區爭端和衝突的一個重要平臺<ref>{{cite web |url = https://www.iiss.org/en/events/shangri%20la%20dialogue/archive/shangri-la-dialogue-2015-862b/opening-remarks-and-keynote-address-6729/keynote-address-a51f |title=Keynote Address: Lee Hsien Loong |publisher=|accessdate=30 July 2015}}</ref>。{{Link-en|亞太安全合作理事會|Council for Security Cooperation in the Asia Pacific}}則是各國間為協調亞太地區的安全問題的另外一個重要平臺<ref>{{cite web |url = http://www.cscap.org/uploads/docs/CRSO/CRSO2014.pdf |title=Regional Security Outlook 2014 |work=CSCAP |accessdate=26 November 2015}}</ref><ref>{{cite web |url = http://www.cscap.org/uploads/docs/CRSO/CSCAPCRSO2015.pdf |title=Regional Security Outlook |work=CSCAP |accessdate=26 November 2015}}</ref>。

2013年，[[菲律賓|菲律賓]]提出[[南海仲裁案|南海仲裁案]]，控告[[中國|中國]]的[[九段线|九段线]]主张和海洋执法活动違反[[联合国海洋法公约|联合国海洋法公约]]。美國，日本等域外國家相繼介入，令南海爭議由地區提升至國際層面。

==中華人民共和國立場==
本質上中国大陆以[[国民政府|国民政府]]迁台前确立的[[十一段线|十一段线]]区域为依据表示繼承，在相关岛屿保持和维护了早期民国时期建立的纪念碑和标示物。 1958年9月4日，中華人民共和國發表關於[[領海|領海]]的聲明<ref>[http://news3.xinhuanet.com/ziliao/2003-01/24/content_705061.htm 1958年9月4日中華人民共和國政府關於領海的聲明] {{webarchive|url=https://web.archive.org/web/20160305013610/http://news3.xinhuanet.com/ziliao/2003-01/24/content_705061.htm |date=2016年3月5日 }}</ref>。同年9月14日，[[越南民主共和國|越南民主共和國]]（北越）政府主席[[范文同|范文同]]在給中華人民共和國總理[[周恩來|周恩來]]的[[照會|照會]]中表示：「越南民主共和國承認和贊同中華人民共和國政府一九五八年九月四日關於規定中國領海的聲明，越南民主共和國政府尊重這一決定」。<ref>[http://commons.wikimedia.org/wiki/Image:1958_diplomatic_note_from_phamvandong_to_zhouenlai.jpg 1958年9月14日越南總理範文同致周恩來總理的外交文書 ]</ref>

1974年，與[[越南共和國|越南共和國]]（南越）爆發[[西沙海戰|西沙海戰]]，控制了[[西沙群島|西沙群島]]，行政上劃歸广东省管轄。1988年，與越南（[[越南社会主义共和國|越南社会主义共和國]]）在[[赤瓜礁海戰|赤瓜礁海戰]]，控制了[[赤瓜礁|赤瓜礁]]等七個南沙島礁。1997年重申「南海」[[九段线|九段线]]（中华民国政府迁台前提出的“十一段线”）及其內之所有島礁之主權。

隨美國重返亞太戰略對於南海加強爭議的策略，2012年，國務院批准設立[[地级市|地级]]海南省[[三沙市|三沙市]]（意為西沙、中沙、南沙），行政机关驻扎在[[永兴岛|永兴岛]]，管轄南海各島嶼，但实际上仅实际控制其中部分岛屿。同時開始大規模填海造陸，<ref>[http://www.chinatimes.com/newspapers/20140901001572-260301 南海大造島]</ref>南薰礁、赤瓜礁、華陽礁、東門礁、永暑礁、安達礁等擴建島嶼形成機場。

中華人民共和國立場本質認為從歷史觀點看該地全部島嶼與大量水下沈船遺跡證明中國歷史上漫長的歲月在當地活動，所有島礁都已經被發現，廣東與[[海南島|海南島]]一代漁民從民間活動認知到歷史諸多典故，都與南海有密切關聯，1279年[[郭守敬|郭守敬]]提案「四海測驗」的詳細紀錄就有南海諸多島嶼，選定的27個緯度測量點之一就有黃岩島是鐵證事實，而其他國家民族的活動證據弱點在於沒有中國的詳細記載甚至不可考，[[明朝|明朝]]時期則将中沙群岛及南沙群岛统称为[[万里长沙|万里长沙]]，[[清朝|清朝]]时期称為[[千里石塘|千里石塘]]，[[潭门镇|潭门镇]][[渔民|渔民]]在该海域捕鱼历史悠久，有祖传的600年《[[更路薄|更路薄]]》作為證據<ref>[http://news.wenweipo.com/2016/07/15/IN1607150068.htm 文匯-南海漁民展示《更路簿》]</ref>記載著西沙和南沙絕大部分島礁的位置以及島礁之間的作業路線。

但南亞一代諸多原住民的海上活動從常理推論也有涉及南海，因此[[鄧小平|鄧小平]]時代就提出主權屬我、共同開發、擱置爭議的觀點，<ref>[http://data.tiexue.net/view/1416530 tiexue.net南海爭議專輯] {{webarchive|url=https://web.archive.org/web/20160229123551/http://data.tiexue.net/view/1416530 |date=2016-02-29 }}</ref>主張核心思想為中國既然能提出較高強度歷史證據，其他周邊國只有推論證據，那「歷史權力」無疑將全部或至少大部分以中方定義為主，且潛在觀點為兩場南海戰役的勝利也應該是戰勝者說了算。但本著不以武力霸權為第一選擇的態度，願意與諸國用雙邊協商方式談判解決爭議，主張中方歷史權力但不全盤否定周邊國也有部分歷史權力的理論，爭議解決前可以共同開發南海資源。

==中華民國立場==
1946年10月5日，法國Chevreud號侵入並登陸[[南沙群島|南沙群島]][[南威島|南威島]]和[[太平島|太平島]]，在島上建立石碑，當時中華民國政府提出抗議，並與法方決定於該月及翌年1月4日進行談判，但因越南戰爭緊張，法國自動放棄談判。1946年10月，[[中華民國|中國]][[國民政府|國民政府]]決定派出「中業號」、「永興號」、「太平號」、「中建號」等四艘軍艦，由指揮官[[林遵|林遵]]、[[姚汝鈺|姚汝鈺]]率領南下，並有內政及陸海空各部代表隨往視察，會同海軍在廣州出發，前往西沙、南沙進駐接收，當時美國並未表示意見，甚至派軍艦護航接收行動，之後長達30多年間美國民間販售的地圖上都將南海大量島嶼標註為中國領土。1949年中華民國政府遷至台灣。

2002年起的東協南海聲明，中華民國政府並未參與《[[南海各方行為宣言|南海各方行為宣言]]》，並拒絕承認《南海各方行為宣言》。

2015年後南海爭端較劇烈，國民黨執政的中華民國政府採取堅持十一段線與固有疆域立場，並對菲律賓南海仲裁案中將[[太平島|太平島]]定义為礁石的說法抗議，[[馬英九|馬英九]]總統於2016年登島宣示主權。

==東協國家立場==
2002年為南海周邊各方國家之主權聲明:
# 各方重申以《[[聯合國憲章|聯合國憲章]]》宗旨和原則、1982年《[[聯合國海洋法公約|聯合國海洋法公約]]》、《[[東南亞友好合作條約|東南亞友好合作條約]]》、[[和平共處五項原則|和平共處五項原則]]以及其它公認的國際法原則作為處理國家間關係的基本准則。
# 各方承諾根据上述原則，在平等和相互尊重的基礎上，探討建立信任的途徑。
# 各方重申尊重并承諾，包括1982年《[[聯合國海洋法公約|聯合國海洋法公約]]》在內的公認的國際法原則所規定的在南海的航行及飛越自由。
# 有關各方承諾根据公認的國際法原則，包括1982年《聯合國海洋法公約》，由直接有關的主權國家通過友好磋商和談判，以和平方式解決它們的領土和管轄權爭議，而不訴諸武力或以武力相威脅。
# 各方承諾保持自我克制，不采取使爭議复雜化、擴大化和影響和平与穩定的行動，包括不在現無人居住的島、礁、灘、沙或其它自然构造上采取居住的行動，并以建設性的方式處理它們的分歧。在和平解決它們的領土和管轄權爭議之前，有關各方承諾本著合作与諒解的精神，努力尋求各种途徑建立相互信任
# 有關各方愿通過各方同意的模式，就有關問題繼續進行磋商和對話，包括對遵守本宣言問題舉行定期磋商，以增進睦鄰友好關係和提高透明度，創造和諧、相互理解与合作，推動以和平方式解決彼此間爭議。
# 各方承諾尊重本宣言的條款并采取与宣言相一致的行動。
# 各方鼓勵其他國家尊重本宣言所包含的原則。
# 有關各方重申制定南海行為准則將進一步促進本地區和平与穩定，并同意在各方協商一致的基礎上，朝最終達成該目標而努力。

大致上東協諸國採取中美兩邊平衡獲利的做法，因為許多國家在南海其實沒有爭議領土也沒有太大利益，2016年2月16日美國與東盟領導人非正式會議結束，結論宣言並沒有如美國媒體的預期出現譴責中國的文字，甚至沒有提到任何南海字眼，表明多數國家認為與中國的商貿關係將在未來佔有巨大利益，沒有動機站在任一方，而實質上沒有表態則對北京立場有利，因為表示菲律賓和越南將單獨面對中國的財力與軍力在南海博弈屈居下風。

===馬來西亞===
馬來西亞只是到了1978年12月，才在其公佈的大陸架地圖上將南沙群島的部分島礁和海域標在馬來西亞境內。70年代以前沒有任何法律文件或領導人講話提及本國領土範圍包括南沙群島。

===菲律賓===
美國與西班牙1898年簽訂的巴黎條約和1900年簽訂的華盛頓條約曾明確規定了菲律賓的領土範圍，但並未包括南沙群島。 1953年菲律賓憲法、1951年菲美軍事同盟條約等也對此作了進一步確認。1955年在馬尼拉召開的國際民航組織太平洋地區航空會議通過的第24號決議，要求臺灣當局在南沙群島加強氣象觀測，會上沒有任何一個代表對此提出異議或保留。70年代以前沒有任何法律文件或領導人講話提及本國領土範圍包括南沙群島。

菲律賓為美國的聯盟關係，對於《南海各方行為宣言》表面依然採支持立場，但單方動作較頻繁與劇烈表示出抗衡中國大陸的立場，早在1956年3月1日，菲律賓民間創辦的{{tsl|en|PMI Colleges|菲律賓海事學院}}校長[[托馬斯·克洛馬|托馬斯·克洛馬]]為首帶領40人組成探險隊，從馬尼拉出發前往南沙群島海域探險，並登上了北子礁、南子礁、中業島、南鑰島、西月島、太平島、[[敦謙沙洲|敦謙沙洲]]、鴻庥島、南威島等9個主要島嶼。其聲索的當時這些島為無人島，自己首先發現的立場，在國際普遍不承認。但其地理位置讓菲律賓依然有南海相關國的身分能參與相關事務的發言與爭議。

2012年發生[[黄岩岛主权问题|黄岩岛主权问题]]，[[菲律賓總統|菲律賓總統]][[諾諾·艾奎諾|艾奎諾三世]]於9月12日頒布第29號行政命令，正式將南海命名為「西菲律賓海」。此行政命令已於2012年9月5日簽訂完成。<ref>{{Cite web |url=http://www.cna.com.tw/News/aCN/201209120280.aspx |title=菲頒命令 南海改名西菲律賓海 林行健/馬尼拉/中央社 2012-9-12 |access-date=2016-02-21 |archive-url=https://web.archive.org/web/20140515000641/http://www.cna.com.tw/News/aCN/201209120280.aspx |archive-date=2014-05-15 |dead-url=yes }}</ref>其立場與美國親近，支持美軍等域外國家參與南海爭端，避免自己孤軍奮戰的策略。然而杜特蒂總統上台後採取了全面相反的態度，認為與中國大陸交好是菲經濟改善的唯一希望，同時對西方指責其反毒政策的態度感到憤恨，將戰爭邊緣的南海問題突然降格成「漁業糾紛」。其訪華與習近平會商後，雙方態度大幅降溫，菲漁船也能回到黃岩島捕魚，菲律賓達成了不再挑戰中國主權的立場，一切態勢回歸爭議前原點。

===越南===
[[File:Collins_Reef_building_by_Vietnam.jpg|缩略图]]的越南設施]]
越南在1975年以前明確承認中國對南沙群島的領土主權。越南1960年、1972年出版的世界地圖及1974年出版的教科書都承認南沙群島是中國領土{{來源請求}}。但其後认为其以歷史的及大陸棚的理論而擁有南中国海全部及其內之島嶼，其歷史的理由是[[法國|法國]]於1930年曾表示擁有全部[[斯普拉特利群岛|斯普拉特利群岛]]（南沙群岛）的主權。越南目前控制了大部份南沙島礁，並宣稱對於在[[西沙海戰|1974年與中華人民共和国的海戰]]中所被中国占领的西沙群岛擁有主權。

越南媒體網站（vietnamnet.vn）2012年9月12日報導，越南教育部和胡志明共青團日前簽署「2012-2013學年度合作計劃備忘錄」，將加強提供學生南海（越稱東海）有關訊息。<ref>[http://www.cna.com.tw/News/aOPL/201209120208.aspx 認識南海 越加強教育宣傳 方沛清/河內/中央社 2012-9-12]</ref>

==日本立場==
1939年，日本侵占了南海諸島。第二次世界大戰期間，日本發動侵華戰爭，佔領了中國大部分地區，包括南沙群島。 《[[開羅宣言|開羅宣言]]》和《[[波茨坦公告|波茨坦公告]]》及其他國際文件明確規定把被日本竊取的中國領土歸還中國，包括了南沙群島。日本投降後，時任國民黨海軍海防第二艦隊司令的[[林遵|林遵]]率“太平”、“中業”兩艦接收南沙群島，將南沙主島以所乘軍艦艦名改為“[[太平島|太平島]]”，並在島上樹立刻有“太平島”字樣的紀念碑。隨後接收人員又到[[中業島|中業島]]、[[西月島|西月島]]、[[南威島|南威島]]，分別在島上舉行接收儀式，並立碑為證。1952年的《[[中日和約|中日和約]]》中，日本放棄西沙群島和南沙群島的全部主權。

日本1952年由[[外務大臣_(日本)|外務大臣]]岡崎勝男親筆推薦的《標準世界地圖集》、以及1962年由外務大臣[[大平正芳|大平正芳]]推薦出版的《世界新地圖集》以及1979年日本共同社出版的《世界年鑑》都承認南沙群島是中國領土。

但[[安倍晉三|安倍晉三]]上台後採行在東海與南海采取强硬的做法以抗衡[[中國崛起|中國崛起]]，但距離遙遠的域外國家身分，使其在南海沒有其他国家那么多發言權，大致上跟從美國的航行自由理論，認為南海事務對日本航運與石油通道有關聯，所以有權發表關注。

==声索国记录==
1946年[[法國|法國]]的入侵才開始引起主權爭議。中國與[[越南|越南]]兩方均聲稱擁有該群島完整的主權，[[馬來西亞|馬來西亞]]、[[菲律賓|菲律賓]]則宣稱擁有部分島嶼的主權，並且派軍隊進占相關島礁，[[文萊|文萊]]與[[印度尼西亞|印度尼西亞]]則是宣稱擁有此區海域的主權，進而間接包含了[[南沙群島|南沙群島]]部分區域。

中國古代政府對西沙群島的管理始於[[秦代|秦代]]。公元前221年，[[秦始皇|秦始皇]]統一六國後，分全國為42郡，其中[[南海郡|南海郡]]管轄包括[[西沙群島|西沙群島]]在內的整個南海諸島。

公元前111年，[[漢武帝|漢武帝]]平定[[南粵|南粵]]之亂後，在[[海南島|海南島]]設[[儋耳|儋耳]]、[[朱崖|朱崖]]兩郡，轄南海諸島，並派水師巡視西沙。[[東漢|東漢]][[楊孚|楊孚]]《[[異物誌|異物誌]]》有「漲海崎頭，水淺而多磁石」的記載。這裡的「漲海」是當時中國對南海的稱呼，{{Fact|「崎頭」則是當時對包括[[西沙群島|西沙群島]]和[[南沙群島|南沙群島]]在內的南海諸島的島、礁、沙、灘的稱呼|time=2017-02-22T01:01:04+00:00}}。

[[三國|三國]][[東吳|東吳]]將領康泰所著《[[扶南傳|扶南傳]]》不僅提到了南沙群島，而且對其形態描述道：「漲海中，到珊瑚洲，洲底有盤古，珊瑚生其上也。」另外[[三国|三国]]时的《南国异物志》有[[南沙群岛|南沙群岛]]的记载。

[[南朝|南朝]]時，朝廷仍派出舟師巡視南海。 《[[舊唐書|舊唐書]]•地理志》記載，[[振州|振州]]（今[[海南|海南]][[三亞|三亞]]）疆域「西南至大海千里」，[[西沙群島|西沙群島]]包括在內。

[[唐|唐]][[宋|宋]]年間，許多歷史地理著作將西沙和南沙群島相繼命名為「九乳螺洲」、「石塘」、「長沙」、「[[千里石塘|千里石塘]]」、「[[千里長沙|千里長沙]]」、「[[萬里石塘|萬里石塘]]」、「[[萬里長沙|萬里長沙]]」等。[[宋朝|宋]][[元朝|元]][[明|明]][[清|清]]四代，以「石塘」、「長沙」為名記述南海諸島的書籍多達上百種。

[[唐穆宗|唐穆宗]]時明令[[嶺南節度使|嶺南節度使]]管轄的範圍包括南海諸島。宋代[[趙汝適|趙汝適]]所著《[[諸蕃志|諸蕃志]]》記載,唐[[貞元|貞元]]二年（公元789年），當時被稱為「千里長沙」、「萬里石塘」的南海諸島即隸屬於海南四州軍。

《瓊州府志》記載，[[宋仁宗|宋仁宗]][[慶曆|慶曆]]年間，將[[西沙群島|西沙群島]]列為廣南水軍的巡海範圍之內。[[宋代|宋代]]官修兵書《[[武經總要|武經總要]]》記載，公元971年[[宋太祖|宋太祖]]建立巡海水師，對南海實施巡管，由此中國政府將南海海域納入到海防範圍。大約成書於1203至1208年的地方志書《瓊管志》記載南海諸島隸屬於[[廣南西路|廣南西路]][[瓊管安撫都監|瓊管安撫都監]][[吉陽軍|吉陽軍]]的管轄範圍，這標誌著中國政府最晚從唐宋時期起，已將南海諸島納入版圖，並明確了行政轄治。

[[元代|元代]]，對南海諸島地理位置的記載更為詳細。[[汪大淵|汪大淵]]所著《[[島夷志略|島夷志略]]》中有「萬里石塘，由潮州而生，迤邐如長蛇，橫亙海中......原其地脈。歷歷可考。一脈至爪哇，一脈至渤泥及古里地悶，一脈至西洋遐崑崙之地。」其中「萬里石塘」指包括今南沙在內的南海諸島。《[[元史|元史]]》地理志和《元代疆域圖敘》記載元代疆域包括了南沙群島。其中《元史》还记载了元朝海军「发泉州，...过七洲洋，万里石塘（南沙群岛），历交趾，占城界」巡辖了南沙群岛<ref>《元史》卷一百六十二列传第四十九史弼</ref>。《元史》記載，至元十六年（1279年），[[元世祖|元世祖]][[忽必烈|忽必烈]]委派同知太史院士[[郭守敬|郭守敬]]到南海進行測量，在繪製的《廣輿圖》中，把南海諸島的西沙群島標為「千里長沙」，南沙群島標為「萬里石塘」。

[[明代|明代]]《[[混一疆理歷代國都之圖|混一疆理歷代國都之圖]]》中標有石塘、長沙和石塘。從圖中標繪的位置看，後一個石塘是今南沙群島。明代《海南衛指揮僉事柴公墓誌鉻》記載：「廣東瀕大海，海外諸國皆內屬」，「公統兵萬餘，巨艦五十艘」，巡邏「海道幾萬里」。表明南沙群島屬於明代版圖，明代[[海南衛|海南衛]]巡轄了西沙、[[中沙|中沙]]和南沙群島。有關學者指出，明代《鄭和航海圖》標繪的「石塘」和「萬生石塘嶼」指的就是今天的西沙、南沙群島<ref>{{Cite web |url=http://book.ce.cn/read/army/nstm/201207/11/t20120711_23481349_51.shtml |title=《南沙探秘》 |access-date=2016-07-28 |archive-url=https://web.archive.org/web/20160817094204/http://book.ce.cn/read/army/nstm/201207/11/t20120711_23481349_51.shtml |archive-date=2016-08-17 |dead-url=yes }}</ref>。明代[[黃佐|黃佐]]所著《[[廣東通志|廣東通志]]》記載「[[七洲洋|七洲洋]]」即今西沙群島，屬於巡海備倭官巡海設防的範圍。 

[[清朝|清朝]]設立[[崖州|崖州]]協水師營，負責南海海疆巡視。清代《[[泉州府志|泉州府志]]》和《[[同安縣|同安縣]]志》記載了[[廣東水師|廣東水師]]副將[[吳升|吳升]]對南海海域巡視的史實。《瓊州府志》和《崖州志》則記載:「崖州協水師營分管洋面，東自[[萬州|萬州]][[東澳|東澳]]港起，西至[[昌化縣|昌化縣]][[四更沙|四更沙]]止，共巡洋面一千里，南面直接[[暹羅|暹羅]]、[[占城|占城]]夷洋」。這清晰地表明，清代水師巡視範圍涵蓋了包括西沙、南沙在內的整個南海海域。 1909年4月廣東總督[[張人駿|張人駿]]命廣東[[水師提督|水師提督]][[李準|李準]]前往西沙巡視。李准率軍地官員及測繪人員、化驗師、工程師、醫生、工人等共計170餘人，分乘伏波、琛航、廣金三艘兵船至西沙各島及海域遍為巡查，先後命名了西沙14座島嶼。

[[清代|清代]]《更路簿》記載了中國[[海南島|海南島]]漁民所習用的南沙群島各個島、礁、灘、洲的地名具體方位，其中南沙計73個地名。在[[清代|清代]]，政府将南沙群岛标绘在地圖上<ref>陆心贤等《地学史话》，上海科学技术出版社，1979，第145頁</ref>，对南沙群岛行使行政管辖。1724年的《清直省分圖》之《天下總輿圖》、1755年《皇清各直省分圖》之《天下總輿圖》、1767年《[[大清萬年一統天下全圖|大清萬年一統天下全圖]]》、1810年《[[大清萬年一統地量全圖|大清萬年一統地量全圖]]》、1817年《大清一統天下全圖》和1895年印行的《古今地輿全圖》等許多地圖均將南沙群島列入中國版圖。一些清朝的外交官員如駐英使節[[郭嵩燾|郭嵩燾]]亦指出西沙群島屬於中國<ref>楊作洲《南海風雲: 海域及相關問題的探討》正中書局，1993，第38頁</ref>。而越南方面则持否定态度，2012年7月越南国家博物馆公布一份[[清朝|清朝]]地图（原圖為《皇朝直省輿地全圖》，此圖僅包括部分清代直省地區,並非全圖），指地图未包括南海岛屿。

在清代官方文籍中，如1676年[[兩廣總督|兩廣總督]][[金光祖|金光祖]]纂修的《[[廣東通志|廣東通志]]》山川·萬州條，1725年[[經筵講官|經筵講官]]、[[戶部尚書|戶部尚書]][[蔣廷錫|蔣廷錫]]等校修、1726年[[雍正|雍正]]皇帝御序的《[[欽定古今圖書集成|欽定古今圖書集成]]》職方典·瓊州府山川考二·萬州條，1731年[[廣東總督|廣東總督]][[郝玉麟|郝玉麟]]等修纂的《廣東通志》山川·萬州條，1822年兩廣總督[[阮元|阮元]]總裁、[[廣東巡撫|廣東巡撫]][[李鴻賓|李鴻賓]]等監修的《廣東通志》山川略十三·瓊州府萬州條，1679年[[萬州|萬州]][[知州|知州]][[李炎|李炎]]等原著、1819年萬州知州汪長齡主修、1828年萬州知事胡端書續修的《[[萬州志|萬州志]]》川條， 1841年[[明誼修|明誼修]]、[[張岳崧|張岳崧]]編纂的《瓊州府志》萬州海防條等官方文籍，均把“千里長沙、萬里石塘”列入[[廣東省|廣東省]][[瓊州府|瓊州府]][[萬州|萬州]]轄治範圍內。

中國人民至遲明初就到南沙群島從事開發漁業生產。早在明代，有[[海口|海口]]港、[[舖|舖]]前港和清瀾港漁民及[[文昌縣|文昌縣]]漁民到南沙群島去捕撈海參等物。

1868年《中國海指南》記載了中國漁民在南沙群島活動情況，[[鄭和群礁|鄭和群礁]]有「海南漁民，以捕取海參，貝殼為活，各島都有其足跡，亦有久居礁間者，海南每歲有小船駛往島上。攜米糧及其他必需品，與漁民交換參貝。船於每年十二月或一月離海南，至第一次西南風起時返。」清末以來，中國[[海南島|海南島]]和[[雷州半島|雷州半島]]各地漁民都有人到南沙群島去捕魚，其中以文昌、[[瓊海|瓊海]]兩縣最多，每年僅從此二地去的漁船就各有十幾條到二十多條。

民國以來中國漁民開發經營南沙群島的史實，中外史料均有記載。[[日本|日本]]小倉卯之助《暴風之島》記載1918年他組織的探險隊到達[[北子島|北子島]]時發現三位文昌縣海口人。 1933年日本三好和松尾到南沙調查時看到北子島有中國人2名、[[南子島|南子島]]有中國人3名住在那裡。日本《新南群島概況》記載，[[中業島|中業島]]有漁民「栽種之甘藷」，「昔時有[[中華民國|中華民國]]漁民居住於此島，並種植椰子、木瓜、蕃薯和蔬菜等」。

自從大量勘探海底[[石油|石油]][[天然氣|天然氣]]資源以後，圍繞南海海域及島嶼的主權爭議，一直被視為[[亞洲|亞洲]]最具潛在危險性的衝突點之一。中國政府認為其對「南海」的主權是不容質疑的，其他国家也不愿放弃其对南海海域的主权要求。目前，[[中華民國|中華民國]]、汶萊、馬來西亞、菲律賓和越南、中華人民共和國等南海週遭[[國家|國家]]均宣稱對南海諸島或部分島嶼擁有主權。

為了防止敵對行動，[[東南亞國家聯盟|東南亞國家聯盟]]十国與中華人民共和國在2002年簽署了《[[南海各方行為宣言|南海各方行為宣言]]》。

下列為南海周邊各方國家之主權聲明:
# 各方重申以《[[聯合國憲章|聯合國憲章]]》宗旨和原則、1982年《[[聯合國海洋法公約|聯合國海洋法公約]]》、《[[東南亞友好合作條約|東南亞友好合作條約]]》、[[和平共處五項原則|和平共處五項原則]]以及其它公認的國際法原則作為處理國家間關係的基本准則。
# 各方承諾根据上述原則，在平等和相互尊重的基礎上，探討建立信任的途徑。
# 各方重申尊重并承諾，包括1982年《聯合國海洋法公約》在內的公認的國際法原則所規定的在南海的航行及飛越自由。
# 有關各方承諾根据公認的國際法原則，包括1982年《聯合國海洋法公約》，由直接有關的主權國家通過友好磋商和談判，以和平方式解決它們的領土和管轄權爭議，而不訴諸武力或以武力相威脅。
# 各方承諾保持自我克制，不采取使爭議复雜化、擴大化和影響和平与穩定的行動，包括不在現無人居住的島、礁、灘、沙或其它自然构造上采取居住的行動，并以建設性的方式處理它們的分歧。在和平解決它們的領土和管轄權爭議之前，有關各方承諾本著合作与諒解的精神，努力尋求各种途徑建立相互信任
# 有關各方愿通過各方同意的模式，就有關問題繼續進行磋商和對話，包括對遵守本宣言問題舉行定期磋商，以增進睦鄰友好關係和提高透明度，創造和諧、相互理解与合作，推動以和平方式解決彼此間爭議。
# 各方承諾尊重本宣言的條款并采取与宣言相一致的行動。
# 各方鼓勵其他國家尊重本宣言所包含的原則。
# 有關各方重申制定南海行為准則將進一步促進本地區和平与穩定，并同意在各方協商一致的基礎上，朝最終達成該目標而努力。
注：同時主張擁有南海主權的[[中華民國政府|中華民國政府]]並未參與《南海各方行為宣言》，並拒絕承認《南海各方行為宣言》。

* 1974年8月，71岁高龄的克罗马宣布辞去[[国家元首|国家元首]]的头衔，将其传给[[约翰·B·德·马里维勒斯|约翰·B·德·马里维勒斯]]亲王，其政府班子成员随后宣布将国名改为[[科洛尼亚王国|科洛尼亚王国]]（{{lang-en|Kingdom of Colonia}}）。当年11月，克罗马被菲律宾总统[[费迪南德·马科斯|费迪南德·马科斯]]下令逮捕，并交付军事法庭审判，原因是他曾经自称为[[菲律宾共和国|菲律宾共和国]]的“海军上将”，被指控招摇撞骗。克罗马被迫签下一份证书，以一[[菲律賓比索|菲律賓比索]]的价格将其王国出售给菲律宾，以换取自己的自由。科洛尼亚王国的国王、首相、议长等人面临菲律宾方面的通缉，逃往[[马来西亚|马来西亚]][[沙巴州|沙巴州]]的[[纳闽岛|纳闽岛]]，组建了[[流亡政府|流亡政府]]<ref>{{cite web | language = zh-hans | publisher = 南沙群岛在线>南沙群岛历史点滴 | title = 南海诸岛大事年表（1950年－1974年） | url = http://www.nansha.org.cn/history/3/nanhai_zhudao_chronology_1950_1974.html | author = 张涛 (zt) | date = 2006年4月24日 | accessdate = 2006年4月24日}}</ref>。
* 2011年，[[菲律賓政府|菲律賓政府]]通過將官方文件中所称的「南中国海」改稱為「西菲律賓海」。

越南在1975年以前明確承認中國對南沙群島的領土主權。越南1960年、1972年出版的世界地圖及1974年出版的教科書都承認南沙群島是中國領土。及後卻认为其以歷史的及大陸棚的理論而擁有南中国海全部及其內之島嶼，其歷史的理由是[[法國|法國]]於1930年曾表示擁有全部[[斯普拉特利群岛|斯普拉特利群岛]]（南沙群岛）的主權。越南目前控制了大部份島礁，並宣稱對於在[[西沙海戰|1974年與中華人民共和国的海戰]]中所被中国占领的西沙群岛擁有主權。
