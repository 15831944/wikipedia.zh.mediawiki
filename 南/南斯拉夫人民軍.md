'''南斯拉夫人民軍'''（塞爾維亞[[克羅埃西亞語|克羅埃西亞語]]：Jugoslovenska narodna armija／Jugoslavenska narodna armija；[[塞爾維亞語|塞爾維亞語]]及[[馬其頓語|馬其頓語]]，西里爾字母：Југословенска народна армија—ЈНА，拉丁字母：Jugoslovenska narodna armija；[[克羅埃西亞語|克羅埃西亞語]]及[[波士尼亞語|波士尼亞語]]：Jugoslavenska narodna armija—JNA；[[斯洛文尼亞語|斯洛文尼亞語]]：Jugoslovanska ljudska armada—JLA）一般簡稱為'''JNA'''，國際上有時會採用英語 Yugoslav People's Army簡稱YPA。

== 緣起 ==

南斯拉夫人民軍的前身為[[第二次世界大戰|第二次世界大戰]]時的「[[南斯拉夫人民解放軍和游击队|南斯拉夫人民解放軍]]」，其第一支正規部隊為1941年12月21日建立的「第一無產階級旅(Prva proleterska brigada)」。二戰結束後，1945年，南斯拉夫以戰時的人民解放軍為基礎建立國家軍隊「南斯拉夫軍」。之後1951年12月建軍十周年時更名為「南斯拉夫人民軍(JNA)」。<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突(1991-2006)〉，碩士論文，東海大學政治系，頁21-22。</ref>

== 組織 ==
南斯拉夫人民軍有完整的陸海空三軍編制，全盛時期曾經為世界第四大作戰部隊。為了機動性及管理方便，軍隊部屬分為四大軍區。
[[貝爾格萊德|貝爾格萊德]]軍區：負責[[克罗地亞|克罗地亞]]東部、包含[[伏伊伏丁那省|伏伊伏丁那省]]的[[塞爾維亞|塞爾維亞]]北部以及波斯尼亞－黑塞哥維納共和国。
[[薩格勒布|薩格勒布]]軍區：負責[[斯洛文尼亞共和国|斯洛文尼亞共和国]]及克罗地亞共和国北部。
[[斯科普里|斯科普里]]軍區：負責[[馬其頓共和国|馬其頓共和国]]、塞爾維亞共和国南部以及[[黑山共和国|黑山共和国]]。
[[斯普利特|斯普利特]]海軍區。
人民軍18萬人的部隊當中，有超過10萬人是屬於志願軍。另外還設立了「[[邊防軍_(南斯拉夫)|邊防軍]]」做為輔助部队。

== 輔助單位 ==
1968年8月20日，由於[[華沙条約组织|華沙条約组织]]中的苏联、东德、波兰、匈牙利、保加利亚五國集结30万军队[[布拉格之春|入侵]][[捷克斯洛伐克|捷克斯洛伐克]]，[[铁托|铁托]]在各共和國及自治省建立了「[[邊防軍_(南斯拉夫)|邊防軍]]」做為人民軍的輔助單位。 从1974年起，邊防軍部隊直接聽命於其所代表的共和國及自治省，並且與人民軍平等。

== 消逝 ==
1992年，[[波黑|波黑]]獨立之後，[[塞爾維亞|塞爾維亞]]與[[黑山|黑山]]在4月27日重組聯邦，稱為「[[南斯拉夫聯盟共和國|南斯拉夫聯盟共和國]]」。5月8日，南斯拉夫人民军重新規劃為「[[南斯拉夫聯盟共和國|南斯拉夫軍隊]](Vojska Jugoslavije, VJ)」，並且從波斯尼亞撤離大約14000人的部隊，留下大約8000人的部隊組成波斯尼亞「[[塞族共和國軍|塞族共和國軍]]」。<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突(1991-2006)〉，碩士論文，東海大學政治系，頁32-33。</ref> 

2003年，南斯拉夫聯盟共和國更名為「{{flag|塞爾維亞和黑山}}」，同時南斯拉夫军队也更名為「[[塞黑|塞爾維亞和黑山武裝力量]]」(Vojska Srbije i Crne Gore, VSCG)。並且在之後隨著[[黑山共和国|黑山共和国]]独立，再分離出[[黑山共和國|黑山共和國軍隊]]，剩餘的部分改組成[[塞爾維亞軍事|塞爾維亞軍隊]]。
