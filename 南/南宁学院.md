'''南宁学院'''（[[英文|英文]]：Nanning University），原名'''邕江大学'''，2012年11月经教育部批准升格为本科院校并更名为南宁学院。学校坐落于[[广西壮族自治区|广西壮族自治区]]首府[[南宁市|南宁市]]。学校设有[[工学|工学]]、[[管理学|管理学]]、[[文学|文学]]、[[法学|法学]]、[[农学|农学]]等5大学科门类。<ref name="简介">[http://www.nnxy.cn/?singleview-1 南宁学院学校简介]</ref>

== 历史发展 ==
*1985年，邕江大学由[[中国国民党革命委员会|中国国民党革命委员会]]广西壮族自治区委员会（简称民革广西区委）牵头创办。
*2009年7月起，学院由南宁威宁资产经营有限责任公司与民革广西区委合作共办。
*2012年11月23日，教育部下批文同意邕江大学升格为本科高校并更名为南宁学院。

== 校园概况 ==
学校主办方筹措资金近9.7亿元兴建新校园，2011年7月，学校从北湖校区到五象新校园的整体搬迁。学校位于[[广西|广西]][[南宁市|南宁市]]龙亭路8号，占地1296.28亩，其中建筑面积36.80万平方米。校园内有广西特色建筑风雨桥，校园环境优美。<ref name="校园风景">[http://www.nnxy.cn/?singleview-89 南宁学院校园风景]</ref>

== 院系设置 ==
学院设有6个[[二级学院|二级学院]]，39个专业，6个专业方向
*工学院
*信息工程学院
*管理学院
*人文学院
*交通学院
*开放教育学院
