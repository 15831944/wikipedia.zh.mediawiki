'''南京西路1418号郭氏兄弟楼'''是位于上海市[[静安区|静安区]][[南京西路|南京西路]]1400-1418号的两幢历史建筑，原为[[郭沛勳家族|郭沛勳家族]]的成员，[[永安百貨|永安百貨]]的創辦人，上海[[永安公司|永安公司]]老板[[郭乐|郭乐]]（排行第二，1874年-1956）和上海永安纺织公司经理[[郭顺|郭顺]]（排行第六）兄弟的住宅，建于1926年。现为上海市人民政府外事办公室。

郭氏兄弟楼位于[[上海公共租界|上海公共租界]]西区的主干道[[静安寺路|静安寺路]]（今[[南京西路|南京西路]]）1400-1418号，西侧毗邻[[哈同路|哈同路]]，静安寺路对面是英国籍犹太地产大亨[[哈同|哈同]]的豪宅，著名的[[爱俪园|爱俪园]]（哈同花园），西侧不远处是华人地产大王[[程霖生|程霖生]]的产业--[[南京西路1522弄住宅|南京西路1522弄住宅]]，以及静安寺路1550号的[[程霖生公馆|程霖生公馆]]。郭氏兄弟楼由[[公和洋行|公和洋行]]设计，陶桂记营造厂承建。采用法国文艺复兴风格，立面三层分别采用塔式干、爱奥尼和科林斯三种柱式，天花板饰有彩绘。花园占地广阔，设有雕塑喷泉。
 
1994年，该建筑列为第二批[[上海市优秀历史建筑|上海市优秀历史建筑]]<ref>{{cite web |language = zh-cn |publisher = 上海市规划和国土资源管理局 |title = 上海市第二批优秀历史建筑名单 |url = http://www.shgtj.gov.cn/bxzc/zsc/lsfm/lsjz/200905/t20090521_301677.htm |author = |date = |accessdate =  }}</ref>。2014年4月4日， 南京西路1400－1418号住宅被列为第八批[[上海市文物保护单位|上海市文物保护单位]]<ref>{{cite web |language =  |publisher = 上海市人民政府办公厅 |title = 上海市人民政府关于核定并公布上海市文物保护单位的通知 |url = http://www.shanghai.gov.cn/shanghai/download/gongkai/hff1429.pdf |author = 上海市人民政府 |date = 2014-04-15 |accessdate = }}</ref>。
