'''南夢宮'''（{{lang-ja|株式会社ナムコ}}，{{lang-en|'''NA'''kamura '''M'''anufacturing '''Co'''., Ltd}}，[[縮寫|縮寫]]“'''NAMCO'''”）於1971年正式啟用NAMCO商標，1977年正式更名為株式會社NAMCO<ref name="multiple">{{cite web |url=http://www.bandainamcogames.co.jp/corporate/namco/history50.php |title=存档副本 |accessdate=2008-08-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20090328162618/http://www.bandainamcogames.co.jp/corporate/namco/history50.php |archivedate=2009-03-28 }} NAMCO五十年的歷史</ref>，是[[日本|日本]]一家知名的[[电视游戏|电视游戏]]和[[街机|街机]]游戏生产商。南梦宫与[[万代|万代]]于2005年9月29日合并，產生新的公司名為[[万代南梦宫控股|万代南梦宫控股]]，將南梦宫轉為旗下公司。

== 歷史 ==
南夢宮在1955年6月1日由[[中村雅哉|中村雅哉]]在[[東京都|東京都]][[池上_(大田區)|池上]]創辦，当时叫做'''中村製作所'''（Nakamura Manufacturing），一開始是在[[百貨店|百貨店]]頂樓擺設2部電動木馬乘坐機。中村製作所在1971年正式更名为南夢宮（NAMCO），同年在[[香港|香港]]設立'''南港企業亞洲有限公司'''(Namco Enterprises Asia Ltd.)。<ref name="multiple">{{cite web |url=http://www.bandainamcogames.co.jp/corporate/namco/history50.php |title=存档副本 |accessdate=2008-08-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20090328162618/http://www.bandainamcogames.co.jp/corporate/namco/history50.php |archivedate=2009-03-28 }} NAMCO五十年的歷史</ref>2005年，南夢宮正式創設50周年<ref name="mutiple">{{cite web |url=http://www.bandainamcogames.co.jp/corporate/namco/history00.php |title=存档副本 |accessdate=2008-08-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20080424041252/http://www.bandainamcogames.co.jp/corporate/namco/history00.php |archivedate=2008-04-24 }} NAMCO五十年的歷史─2000年代</ref>。
