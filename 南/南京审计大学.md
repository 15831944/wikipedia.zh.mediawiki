'''南京审计大学'''，简称'''南审'''，是一所位于[[中華人民共和國|中国]][[江苏省|江苏省]][[南京市|南京市]]的财经类公立[[高等院校|高等院校]]。学校始建于1983年<ref name="概况"/>，是中国[[审计|审计]]高等教育发源地之一。
其以审计为品牌，[[经济学|经济学]]、[[管理学|管理学]]、[[法学|法学]]、[[文学|文学]]、[[理学|理学]]、[[工学|工学]]等学科相互支撑，为[[国家审计署|国家审计署]]、[[财政部|财政部]]、[[中華人民共和國教育部|教育部]]、[[江苏省人民政府|江苏省人民政府]]三部一省共建高校，拥有莫愁、浦口、仙林三个校区。

2015年11月17日，[[中华人民共和国教育部|中国教育部]]同意南京审计学院更名为南京审计大学<ref>{{cite news|title=南京审计学院更名为南京审计大学|url=http://news.sohu.com/20151118/n426788811.shtml|accessdate=2015-11-21|agency=搜狐|date=2015-11-18}}</ref><ref>{{cite web|title=南京审计学院正式更名为南京审计大学|url=http://iaep.nau.edu.cn/s/138/t/164/5f/3d/info24381.htm|website=南京审计大学|accessdate=2015-11-21}}</ref>。

== 历史 ==
学校前身是始建于1983年的'''南京财贸学院'''，1987年更名为'''南京審計學院'''，2015年更名为南京審計大學。

学校由[[国家审计署|国家审计署]]和南京市联办；1991年转制为[[国家审计署|审计署]]直属；1993年升格为全日制普通本科院校；1999年通过[[中華人民共和國教育部|教育部]]本科教学工作合格评价；2000年办学体制调整为[[江苏省|江苏省]]政府与国家审计署合作共建，以江苏省管理为主；2002年[[南京金融高等专科学校|南京金融高等专科学校]]并入，组建新的南京审计学院；2009年经国务院学位委员会批准，学校被列入新增硕士学位授权立项建设单位；2011年学校被国务院学位委员会批准成为全国首批审计硕士专业学位授权单位；2011年[[国家审计署|国家审计署]]、[[财政部|财政部]]、[[中華人民共和國教育部|教育部]]、[[江苏省人民政府|江苏省人民政府]]共建南京审计学院。

== 概况 ==
学校设有11个二级学院、3个教学部，以及1个二级民办学院、1个继续教育学院，拥有26个本科专业。其中，[[审计学|审计学]]、[[金融学|金融学]]、[[财政学|财政学]]为教育部特色专业建设点，财政学类、金融学类、管理科学与工程类、工商管理类、审计学为江苏省“十二五”重点专业，审计学、金融学为江苏省品牌专业，财政学、[[会计学|会计学]]、[[工商管理|工商管理]]、[[国际经济与贸易|国际经济与贸易]]、[[信息管理与信息系统|信息管理与信息系统]]为江苏省特色专业。学校现有全日制普通本科在校生近15237名。

== 传统 ==

=== 校训 ===
'''诚信求是，笃学致公'''

“诚信”——发己之诚，得人之信；

“求是”——去伪存真，探索创新；

“笃学”——敏而好学，慎思明辨；

“致公”——致力奉公，公平公正。
