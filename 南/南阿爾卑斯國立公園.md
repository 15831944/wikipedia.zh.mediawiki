'''南阿爾卑斯國立公園'''（假名：{{lang|ja|みなみあるぷすこくりつこうえん}}），是位於[[日本|日本]][[山梨縣|山梨縣]]、[[長野縣|長野縣]]、[[靜岡縣|靜岡縣]]的[[日本國立公園|國立公園]]。

== 年表 ==
* 1964年6月1日 - 被指定為'''南阿爾卑斯國立公園'''。

== 主要景點 ==
日本第二高峰・-{[[北岳|北岳]]}-等。

== 面積 ==
* 面積 - 35,752ha （陸域部分）
** 山梨縣 - 18,286ha
** 長野縣 - 14,079ha
** 靜岡縣 - 3,387ha

== 關連市町村 ==
* 山梨縣 - [[韮崎市|韮崎市]]、[[南阿爾卑斯市|南阿爾卑斯市]]、[[北杜市|北杜市]]、[[早川町|早川町]]
* 長野縣 - [[伊那市|伊那市]]、[[飯田市|飯田市]]、[[富士見町|富士見町]]、[[大鹿村|大鹿村]]
* 靜岡縣 - [[靜岡市|靜岡市]]、[[川根本町|川根本町]]
