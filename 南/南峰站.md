'''南峰站'''是[[杭临城际铁路|杭临城际铁路]]的一个高架站，位于[[杭州市|杭州市]][[余杭区|余杭区]][[中泰街道|中泰街道]]02省道舟南线路口北侧的空地内。

==结构== 
南峰站为高架二层岛式站台，地面为站厅层，二层为站台层，车站总长126m，总宽20.6m，站台宽12m，离地高度为8.35m，总建筑面积6450平方米。<ref>{{cite web |title=杭州至临安城际铁路工程余杭段首座车站顺利封顶 - 工程聚焦 - 人民铁道网 - 中国官方铁路门户 |url=http://www.peoplerail.com/rail/show-466-402493-1.html |website=人民铁道网}}</ref>

== 历史 ==
*2016年1月4日，车站随试验段一起开工。<ref>{{cite web |title=杭富城际铁路、杭临城际铁路下月正式开工 _ |url=http://www.hzzx.gov.cn/hzxw/content/2015-12/29/content_6027564.htm |website=杭州政协新闻网}}</ref>
*2019年3月25日，车站定名为南峰站。<ref>{{cite web |title=杭州市人民政府关于杭州至临安城际铁路工程车站站名的批复 _ |url=http://www.hangzhou.gov.cn/art/2019/3/28/art_1256295_31738047.html |website=中国杭州}}</ref>
*2019年4月1日，车站结构封顶。
