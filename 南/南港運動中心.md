南港運動中心是台北市第4座啟用市民運動中心，其樓地板面積17,540平方公尺，建物為地上8層、地下4層<ref>[http://tsh.tms.gov.tw/web/topicdemo2.php?AS_ID=6 台北市運動地圖資訊網] {{webarchive|url=https://web.archive.org/web/20100312092710/http://tsh.tms.gov.tw/web/topicdemo2.php?AS_ID=6 |date=2010-03-12 }}</ref>。

== 沿革 ==
*建築基地為南隆鐵工廠舊址，屬工業用地，由遠雄建設集團取得後，透過捐地方式將工業用地轉換為住宅用地，而南港運動中心也是由遠雄集團捐地後興建而成<ref>曾敬德.[https://tw.appledaily.com/forum/daily/20070120/23192799 南港運動中心旁 重大工程激勵房價 鐵路地下化利多 新大樓每坪衝上47萬].蘋果日報.2007-01-20</ref>。
*2006年12月1日正式啟用，承包經營單位是[[中國青年救國團|中國青年救國團]]<ref>[http://www.319.com.tw/store/show/3635 微笑台灣]</ref>。特色設施為射擊場、潛水池。
*南港運動中心將於2015年11底受託營運期滿，於10月份辦理招標，然招標過程遭受質疑，被懷疑有圖利之嫌<ref>梁珮綺.[http://news.ltn.com.tw/news/local/paper/922785 南港運動中心 救國團續約否受矚].自由時報.2015-10-11</ref>。
*中心於2015年(民國104年)11月30日受託營運期滿，後續將依促進民間參與公共建設法，續由中國青年救國團經營，並進行閉館整修及營運，預計整修期間自104年12月1日至105年4月28日
*運動中心於2016年4月初提前完工，並開始重新營運<ref>{{Cite web |url=http://udn.com/news/story/7323/1606881 |title=南港運動中心整容 嶄新出擊 |access-date=2016-07-08 |archive-url=https://web.archive.org/web/20160826083927/http://udn.com/news/story/7323/1606881 |archive-date=2016-08-26 |dead-url=yes }}</ref>。
*2018年游泳池旁的SPA池上方輕鋼架天花板崩塌，並砸傷泳客<ref>蘇怡璇.[https://www.setn.com/News.aspx?NewsID=403741 獨／這款品質！？南港運動中心泳池天花板掉落砸傷婦人！].三立新聞網.2018-07-14</ref>。

== 地理位置 ==
地址位在[[台北市|台北市]][[南港區|南港區]]玉成街69號，臨市民大道口，近[[台北捷運|台北捷運]][[後山埤站|後山埤站]]、[[台北捷運|台北捷運]][[松山車站_(台灣)|松山站]]和[[五分埔|五分埔]]。

== 設備 ==
*南港運動中心除地下停車場、行政空間之外，運動空間包括室內溫水游泳池、棋奕室、閱覽室、社區教室、健身中心、舞蹈教室、幼兒遊戲室、籃球場兼活動中心、羽球場、[[桌球|桌球]]場、[[高爾夫球|高爾夫球]]教室、射擊教室、頂樓射箭場等設施。頂樓射箭場提供日本[[弓道|弓道]]和[[箭術|西洋弓]]等教學。
*南港運動中心也是唯一設有潛水教室的室內運動中心，並設有專業潛水教練認證課程<ref>[http://2017.taipei/files/15-1000-945,c165-1.php 世大運 U運動 南港運動中心]</ref>。
