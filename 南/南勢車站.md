'''南勢車站'''位於[[台灣|台灣]][[苗栗縣|苗栗縣]][[苗栗市|苗栗市]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺中線|臺中線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]兩座。
*跨線[[天橋|天橋]]。
*西側北上月台，亦有入口方便民眾進出站。
*公共廁所
*公共廁所旁的柵欄旁邊有一座小[[遮雨棚|遮雨棚]]，可停放機車或自行車。

== 營運狀況 ==
*南勢車站是一指定由[[苗栗車站|苗栗車站]]管理的[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，臺鐵並未派人駐守。本站僅有[[臺鐵區間車|區間車]]停靠，是台鐵台中線（山線）沿線諸站之中停車班次最少的一座。因無人管理，包含售票與回收票根等相關的票務服務，全都是由停靠列車上的列車長負責處理。
*本站無任何对号列車停靠。
*已不再提供票務服務的南勢車站站房，主要是由臺鐵臺中工務段南勢道班作為辦公室使用。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1903年7月1日：「南勢信號場」開業。
*1914年2月27日：改為「南勢驛」，三等站，辦理客貨運業務。
*1977年：水泥站房完工。
*1999年8月15日：由三等站直接降為招呼站，並指定由[[苗栗車站|苗栗站]]管理。
*2015年6月30日：啟用多卡通刷卡機。
*2017年9月：本站月台加高工程已完工，增高至最終高度115cm。
