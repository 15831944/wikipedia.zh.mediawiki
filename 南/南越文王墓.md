'''南越文王墓'''，又称为'''南越王墓'''，是[[中国|中国]][[西汉|西汉]]时期[[南越国|南越国]]第二代王[[赵眜|赵眜]]（《[[史记|史记]]》称为[[赵胡|赵胡]]）的[[陵墓|陵墓]]。位于[[中国|中国]][[广东省|广东省]][[广州市|广州市]][[解放路_(广州市)|解放北路]]的[[象岗山|象岗山]]上，在[[中国大酒店|中国大酒店]]的北侧，[[越秀公园|越秀公园]]西侧。赵眜是南越国建国的第一代君王[[赵佗|赵佗]]的孙子，号称“南越文帝”，公元前137年至前122年在位。南越王墓被认为是[[岭南|岭南]]地区已发现的陵墓当中，规模最大，随葬品最多，墓主人身份最高的陵墓。该墓出土的陪葬品[[文帝行玺金印|文帝行玺金印]]是中国考古发掘出土的第一枚帝印。1983年在该地建楼宇时被发现，挖掘清理之后在原地建立起“[[西汉南越王博物馆|西汉南越王博物馆]]”进行保护，是[[全国重点文物保护单位|全国重点文物保护单位]]。

== 历史 ==
[[秦朝|秦朝]]末年，中原陷入了“[[楚汉战争|楚汉相争]]”的混乱状态。公元前203年，[[南海郡|南海郡]]尉赵佗起兵兼并[[桂林郡|桂林郡]]和[[象郡|象郡]]，以番禺为都城（今广州市）在岭南地区建立南越国。公元前196年，赵佗臣服于已建立[[西汉|西汉]]政权的[[汉高祖|汉高祖]][[刘邦|刘邦]]，成为汉朝的[[藩属国|藩属国]]。

公元前137年，赵佗去世，其位由次孙[[赵眜|赵眜]]继承，成为第二代南越王，号称“南越文帝”。赵眛在位12年后于公元前125年去世，其位传给[[赵婴齐|赵婴齐]]。赵眜死后陵墓建在南越国都城番禺内，即是现在的“南越王墓”。

===墓室===
南越王墓是凿山建造的，在[[广州市|广州市]]解放北路的象岗山上凿入20米深的地方，用750多块的[[红砂岩|红砂岩]]筑成。整个陵墓坐北朝南，成“士”字形平面，面积在100平方米左右，仿照“前堂后寝”的形制由七个墓室构成，前面三室分别为前室和东、西耳室，后面四室为主棺室、东西侧室和后藏室。

*墓室前部前室的四壁和顶上绘有朱、墨两色云纹图案，象征朝堂。
*东耳室放饮宴用器，西耳器放兵器车马和珍宝等。
*墓主居于后部的主棺室、以一棺一椁入殓，身着[[丝缕玉衣|丝缕玉衣]]，两侧有10把铁剑，9枚印鉴，其中一枚是"文帝行玺"的金印。
*东侧室是4个姬妾的殉葬室，西侧室为7个厨役的殉葬室。
*后藏室为食物储藏室。

===随葬品===
陵墓内共出土随葬品1000多件，包括金器、银器、铜器、铁器、陶器、玉器等，殉葬者15人，是岭南地区已发现的陵墓当中，规模最大，随葬品最多，墓主人身份最高的陵墓。其中丝缕玉衣和“文帝行玺”的金印最为珍贵，而殉葬者最为令人惊心。

*[[文帝行玺金印|文帝行玺金印]]：由黄金铸成，印钮为盘曲的游龙，印面刻有小篆阴文的“文帝行玺”四字，是中国考古发掘出土的第一枚帝印。
*[[丝缕玉衣|丝缕玉衣]]：共用2291块玉片用朱红色丝带穿系而成，是中国发掘出土的年代最早的丝缕玉衣，比[[河北|河北]][[中山靖王|中山靖王]][[刘胜|刘胜]]的[[金缕玉衣|金缕玉衣]]还要早10年。
*[[殉葬|殉葬]]者：共有15人，除了东侧室有4名姬妾、西侧室有7名厨役之外，还在前室发现1名“景巷令”、在东耳室发现1名乐伎、在墓道发现1名卫士和1名车夫。这些殉人都是[[赵眜|赵眜]]的宠妃和身边近待，多数是被击砸后脑致死，其状非常惨烈。

== 西汉南越王博物馆 ==
{{main|西汉南越王博物馆}}
[[西汉南越王博物馆|西汉南越王博物馆]]是南越王墓被发掘之后，为了保护南越王墓，而在原址建立起来的遗址博物馆，设计人[[莫伯治|莫伯治]]和[[何镜堂|何镜堂]]是岭南派建筑师的代表。

== 学术价值 ==
[[南越国|南越国]]由于地处[[西汉|西汉]]疆土的最南方，虽然是西汉的藩属国，但更象一个独立的王国。关于南越国的历史，《[[史记|史记]]》和《[[汉书|汉书]]》等史书都有记载，但内容非常简陋。南越王墓的发掘，对于了解南越国的历史和当时[[岭南|岭南]]地区的情况都提供了非常宝贵的资料。
