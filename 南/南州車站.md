'''南州車站'''位於[[屏東縣|屏東縣]][[南州鄉|南州鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[屏東線|屏東線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[島式月台|島式月台]]兩座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
*停靠狀態：
**[[自強號列車|自強號]]：停靠(逆行)313、317、323、373次、327次，(順行)302、304、306、372次、308、314次。
**[[莒光號列車|莒光號]]：停靠(逆行)701、751、705次，(順行)704、706、754、708、710次。
**[[復興號列車|復興號]]：791次、792次。
**[[臺鐵區間車|區間車]]、[[臺鐵區間快車|區間快車]]：皆停靠。（[[復興號列車|復興號]]車廂編組或是一至兩組[[臺鐵柴聯車|柴聯車]]編組）。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1923年10月11日：初名'''溪州停車場'''，與雙溪停車場（今[[雙溪車站|雙溪車站]]）同日開始運輸營業<ref>告示第百六十七號，府報第三千六十二號，臺灣總督府</ref>。
*1958年2月10日：因[[屏東縣|屏東縣]]「溪州鄉」與[[彰化縣|彰化縣]][[溪州鄉|溪州鄉]]同名，配合鄉名變更（因該地位於潮州之南，故更名為南州），與[[勝興車站|勝興]]、[[港嘴車站|港嘴]]（廢止）、[[內壢車站|內壢]]等車站同時改稱今名。
*2016年1月1日：啟用多卡通刷卡機。
