'''南方医科大学珠江医院'''，位于[[广东省|广东省]][[广州市|广州市]][[工业大道|工业大道中]]253号，是[[南方医科大学|南方医科大学]]的综合性教学医院，为综合性[[三级甲等医院|三级甲等医院]]<ref name="医院历史">{{Cite web |url=http://www.zjyy.com.cn/about/default.aspx?id=4 |title=医院历史 |accessdate=2012-06-14 |work= |publisher=南方医科大学珠江医院 |date= |archive-url=https://web.archive.org/web/20120504200031/http://www.zjyy.com.cn/about/Default.aspx?id=4 |archive-date=2012-05-04 |dead-url=yes }}</ref><ref name="医院简介">{{Cite web |url=http://www.zjyy.com.cn/about/default.aspx?id=1 |title=医院简介 |accessdate=2012-06-14 |work= |publisher=南方医科大学珠江医院 |date= |archive-url=https://web.archive.org/web/20120505235741/http://www.zjyy.com.cn/about/Default.aspx?id=1 |archive-date=2012-05-05 |dead-url=yes }}</ref><ref>{{Cite book|title=广州指南|last=广东省出版进出口公司，本社编辑部|first=|publisher=广州：广东科技出版社|year=1981年10月|isbn=|location=|pages=228}}</ref><ref>{{Cite book|title=迎亚运医学英语100句|last=黄炯烈主编|first=|publisher=广州：广州出版社 ,|year=2009年4月|isbn=978-7-80731-935-1|location=|pages=30}}</ref>。

== 沿革 ==
*1947年2月，在[[第二次国共内战|解放战争]]前线[[山东省|山东省]][[沂蒙山|沂蒙山]]区江裕镇组建'''中国人民解放军第十后方医院'''。1947年2月至1955年6月，医院先后转战山东省、[[河北省|河北省]]、[[江苏省|江苏省]]、[[安徽省|安徽省]]、[[江西省|江西省]]、[[福建省|福建省]]，参加了解放战争华中、华东战场各大战役的前线收治工作，收治伤病员34000多名<ref name = "医院历史" />。
*1947年10月，更名'''中国人民解放军第三后方医院'''<ref name = "医院历史" />。
*1948年12月，更名'''中国人民解放军济南市第一临时医院'''<ref name = "医院历史" />。
*1950年4月，更名'''中国人民解放军第五野战医院'''<ref name = "医院历史" />。
*1952年10月，更名'''中国人民解放军第一野战医院'''<ref name = "医院历史" />。
*1955年1月，更名'''中国人民解放军第一〇三医院'''。1955年6月，医院整编为第一〇三医院，进驻[[福建省|福建省]][[永安市|永安市]]，结束了野战医院任务，开始担负驻军医院任务。从1955年6月到1978年5月归建[[中国人民解放军第一军医大学|中国人民解放军第一军医大学]]的23年间，医院为驻地军民诊疗查体16万余人次，培训基层医务人员4000多人<ref name = "医院历史" />。
*1978年5月1日，根据[[中国人民解放军总参谋部|中国人民解放军总参谋部]]批复和[[中国人民解放军总后勤部|中国人民解放军总后勤部]]通知，“确定一〇三医院调给第一军医大学作教学医院”，医院迁至[[广东省|广东省]][[广州市|广州市]]，归属中国人民解放军第一军医大学（隶属总后勤部），对内称'''中国人民解放军第一军医大学第二附属医院'''，对外称'''珠江医院'''。1981年10月迁至广州市[[海珠区|海珠区]]院址，1983年1月在广州正式开诊<ref name = "医院历史" />。
*1994年，被评为军队首批[[三级甲等医院|三级甲等医院]]。
*2004年8月，根据国务院、中央军委命令，医院随第一军医大学整体移交广东省，更名'''南方医科大学第二临床医学院（珠江医院）'''<ref name = "医院历史" />。
*2007年11月，总建筑面积16万余平方米的新医疗区正式启用<ref name = "医院历史" />。
