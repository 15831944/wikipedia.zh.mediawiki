'''南极星软件公司'''是[[澳大利亚|澳大利亚]]的一家[[软件|软件]]公司。它最初为PC平台电脑开发南极星文书處理系统，分中文版、日文版和韩文版。其后當 Windows 系統普及，開始推出 NJStar Communicator。现时澳大利亚不少大學的電腦室及[[网吧|网吧]]都有安裝他們的軟體，以便亞裔學生與家鄉的親友通讯。
在90年代末中国电脑上也是受到热捧，还是十分著名的中文平台。和当时的《中文之星》、《四通利方》齐名。

南极星是比较早推出中文输入法的公司之一。在中文输入法软件历史中具有举足轻重的作用。

== 新版本 ==
NJStar Communicator现在版本支持Windows Vista，据测试，该软件可以在Windows 7下运行。

== 优点 ==
允许多国文字处理，支持多编码。中文输入法较多。

== 缺点 ==
输入法有很多用不上。

试用版只能用30天。

GBK和GB18030模式下不能输入“囧”、“镕”等GB2312以外的汉字，形同虚设。
