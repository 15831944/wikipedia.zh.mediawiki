'''南澳車站'''位於[[臺灣|臺灣]][[宜蘭縣|宜蘭縣]][[蘇澳鎮_(臺灣)|蘇澳鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[北迴線|北迴線]]的[[鐵路車站|鐵路車站]]。本站雖位於蘇澳鎮南強里範圍內，但越過[[台九線|台九線]]即為[[南澳鄉|南澳鄉]][[南澳村|南澳村]]，距離南澳主要的聚落較近，是該鄉的門戶。

== 車站構造 ==
*[[島式月台|島式月台]]兩座。
*辦理貨運。

== 歷史 ==
本站是在1980年2月1日，伴隨[[蘇澳新站|蘇澳新站]]至[[花蓮新站|花蓮新站]]之間的北迴線全線通車，所一同啟用的車站之一。
*1982年1月1日：由二等站降為三等站。
*2016年3月1日：啟用多卡通刷卡機。

== 營運狀況 ==
南澳是台鐵旗下的[[臺灣鐵路管理局車站等級#三等站|三等站]]之一，除每站皆停的[[臺鐵區間車|區間車]]班次全停之外，包括[[臺鐵區間車|區間快車]]、[[復興號列車|復興號]]、[[莒光號列車|莒光號]]與[[自強號列車|自強號]]等較高等級的列車，都是採部分班次停靠的方式營運。至於等級最高、車速最快的[[太魯閣列車|太魯閣號]]則僅有順行228、逆行229次停靠。[[普悠瑪列車|普悠瑪號]]則完全不停靠。

由於隔鄰的[[武塔車站|武塔]]是一個連站房都沒有設置的無人車站，因此南澳與武塔兩站的「鐵路之旅 - 小站巡禮紀念章」戳章，都是保管於南澳車站之內以供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。
*停靠狀態
**[[普悠瑪列車|普悠瑪號]]：全部班次皆不停靠。
**[[太魯閣列車|太魯閣號]]：停靠南下228次；北上229/6229次。
**[[自強號列車|自強號]]：停靠南下204/6208(週日行駛)、212、172、246次；北上407、175/6175、177/6177、239/6239、251/6251次。
**[[莒光號列車|莒光號]]：不停靠南下562次、[[團體列車|團體列車]]皆不停靠。
**[[復興號列車|復興號]]：停靠南下684、688/6688次；北上689次。
**[[臺鐵區間車|區間快車]]：皆停靠。
**[[臺鐵區間車|區間車]]：皆停靠。
**4240次[[臺鐵區間車|區間車]]為本站終點站 4111次(週一至週六行駛)、4325次(僅週日行駛)[[臺鐵區間車|區間車]]為本站始發站
