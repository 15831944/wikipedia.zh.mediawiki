'''南部斯拉夫民族'''（南斯拉夫語系语言，以[[西里爾字母|西里爾字母]]書寫：{{lang|cyl|Јужни Словени}}，以[[拉丁字母|拉丁字母]]書寫：{{lang|la|Južni Sloveni}}；{{lang-ru|Южные славяне}}）是[[斯拉夫民族|斯拉夫民族]]的一個分支，主要分布於[[巴爾幹半島|巴爾幹半島]]上的[[前南斯拉夫|前南斯拉夫]]地區及保加利亞一带。
 [[File:South_Slavic_Europe.svg|thumb]]

== 起源 ==

[[斯拉夫民族|斯拉夫民族]]的最初分布在[[喀爾巴阡山|喀爾巴阡山]]北麓，[[波蘭|波蘭]]的東南部，[[白俄羅斯|白俄羅斯]]西部和[[烏克蘭|烏克蘭]]西北部，[[維斯杜拉河|維斯杜拉河]]，[[涅斯特河|涅斯特河]]（Dniester）與[[南布格河|南布格河]]（South Bug）的發源地一帶。從西元一世紀的時候開始四處散佈，西元二世紀時，已有大批斯拉夫人遷移到[[多瑙河平原|多瑙河平原]]。依其居住地可分為東、西、南三支，西斯拉夫及南斯拉夫（South Slavs）之間原先並沒有區分。然而在第六～八世紀時，[[阿瓦人|阿瓦人]]（Avars）及[[馬札爾人|馬札爾人]]（Magyars）侵入中歐，南部斯拉夫人向巴爾幹半島的西北部逃遷，進入[[伊利里亞人|伊利里亞人]]居住的區域，與西斯拉夫斷了聯繫，形成了南斯拉夫族群。<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突（1991-2006）〉，碩士論文，東海大學政治系，頁11-12。</ref>

== 分支 ==
傳統上，南斯拉夫族群主要可以分為[[保加利亞人|保加利亞人]]（българи，bǎlgar）、[[克羅埃西亞人|克羅埃西亞人]]（Hrvati）、[[塞爾維亞族|塞爾維亞人]]（Срби，Srbi）與[[斯洛維尼亞人|斯洛維尼亞人]]（Slovenci）四支。 

然而，由於巴爾幹半島複雜的歷史背景及戰亂，今天主要可以分七支：信仰[[天主教|天主教]]的克羅埃西亞人、斯洛維尼亞人，信仰[[東正教|東正教]]的塞爾維亞人、[[蒙特內哥羅人|蒙特內哥羅人]]、[[馬其頓人|馬其頓人]]、[[保加利亞人|保加利亞人]]，以及受[[鄂圖曼土耳其帝國|鄂圖曼土耳其帝國]]影響信仰[[伊斯蘭教|伊斯蘭教]]的[[波士尼亞克人|波士尼亞人]]。

== 語言和文字 ==
南部斯拉夫民族均有自己的語言，克羅埃西亞人使用[[克羅埃西亞語|克羅埃西亞語]]、斯洛維尼亞人使用[[斯洛維尼亞語|斯洛維尼亞語]]、塞爾維亞人使用[[塞爾維亞語|塞爾維亞語]]、蒙特內哥羅人使用[[蒙特內哥羅語|蒙特內哥羅語]]、馬其頓人使用[[馬其頓語|馬其頓語]]、保加利亞人使用[[保加利亞語|保加利亞語]]、波士尼亞人使用[[波士尼亞語|波士尼亞語]]。

由於均屬於同一語族，因此彼此之間的語言都有一定的互通性。

文字使用方面，使用西里爾字母的有塞爾維亞人、蒙特內哥羅人、馬其頓人和保加利亞人，使用拉丁字母的有克羅埃西亞人、斯洛維尼亞人和波士尼亞人。
