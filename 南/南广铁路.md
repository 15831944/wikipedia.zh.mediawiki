'''南宁至广州铁路'''<ref>[http://www.ndrc.gov.cn/fzgggz/nyjt/zhdt/200803/t20080314_197592.html 新建南宁至广州铁路项目建议书通过批复]</ref>，简称'''南广铁路'''，是连接中国[[广西壮族自治区|广西壮族自治区]][[南宁市|南宁市]]与[[广东省|广东省]][[广州市|广州市]]的[[高速铁路|高速铁路]]，是以客运为主、客货共线的铁路干线，由[[南宁铁路局|南宁铁路局]]管轄。工程2008年11月9日起动工，2014年4月18日广西段竣工通车，同年12月26日全线通车。由[[中华人民共和国铁道部|铁道部]]与广西壮族自治区政府、广东省政府合资建设，总投资约410亿元。<ref>[http://news.sina.com.cn/c/2008-11-09/174716618528.shtml 南宁至广州铁路今日开工 投资总额410亿元]</ref>

== 路线 ==
铁路走向沿[[珠江|珠江]]水道的[[西江|西江]]方向，从[[南宁市|南宁市]]出发，与[[柳南城际铁路|柳南城际铁路]]并行至[[宾阳县|宾阳县]][[黎塘镇|黎塘镇]]后向东延伸，经[[贵港|贵港]]，跨[[郁江|郁江]]，过[[桂平|桂平]]、[[平南|平南]]、[[藤县|藤县]]至[[梧州|梧州]]进入[[广东|广东]]，经[[郁南|郁南]]、[[云浮|云浮]]至[[肇庆|肇庆]][[肇庆东站|肇庆东站]]后与[[贵广线|贵广线]]一起平行延伸至[[广州南站|广州南站]]。全长576公里，其中广西境内349公里，广东境内227公里。

南广铁路线路设站23个，分别为：[[南宁站|南宁]]、[[南宁东站|南宁东]]、五塘、[[宾阳站|宾阳]]、覃塘北、根竹、[[贵港站|贵港]]、厚禄、[[桂平站|桂平]]、[[平南南站|平南南]]、永康、[[藤县站|藤县]]、[[梧州南站|梧州南]]、[[郁南站|郁南]]、[[南江口站|南江口]]、[[云浮东站|云浮东]]、大湾、[[肇庆东站|肇庆东]]、[[三水南站|三水南]]、[[佛山西站|佛山西]]、奇槎、三眼桥、[[广州南站|广州南]]。其中五塘、覃塘北、根竹、厚禄、永康、大湾、奇槎和[[三眼桥|三眼桥]]不办理客运业务。

== 主要技术标准 ==
南广铁路按双线[[电气化|电气化]]标准建设，设计时速200公里，预留250公里的设计时速。

*[[铁路等级|铁路等级]]：国铁Ⅰ级；
*正线数目：双线；
*旅客列车设计行车速度：200km/h，预留提速至250km/h的条件；
*[[最小曲线半径|最小曲线半径]]：3500m；
*[[限制坡度|限制坡度]]：南宁至贵港12‰，贵港至广州南6‰；
*到发线有效长度：850m，只办理客车的车站650m； 
*[[牵引种类|牵引种类]]：电力牵引； 
*[[牵引质量|牵引质量]]：4000t；
*[[闭塞方式|闭塞方式]]：自动闭塞； 
*建筑限界：满足开行双层集装箱列车运输要求。

==历史背景==
[[广西壮族自治区|广西壮族自治区]]、[[广东省|广东省]]毗邻而居，两省区间的产业合作、经贸往来、人员交流由来已久，因此自古便有“共饮一江水、[[两广|两广]]一家亲”之说。经济发展，交通现行。然而2014年底前，两广[[省会|省会]]与[[首府|首府]]间仍未有一条直通的[[铁路|铁路]]线。由[[广西|广西]][[南宁|南宁]]出发，必须向南绕道[[贵港|贵港]]、[[玉林|玉林]]，再经[[茂名|茂名]]、[[肇庆|肇庆]]等地才能到达[[广州|广州]]，全程800多[[公里|公里]]，需要耗时13个多[[小时|小时]]。单线、绕行、旅途时间长等铁路因素，一定程度上制约了两省区间经济社会的进一步深度融合。南广铁路的开通运营，有效补充了铁路交通通道能力方面的不足，由[[南宁|南宁]]横向穿过[[贵港|贵港]]、[[梧州|梧州]]、[[肇庆|肇庆]]直达[[广州|广州]]，570公里的运输距离较过去缩短230公里，旅途时间压缩8小时，大大节约了人们出行的时间成本，同时为两广加快区域经济一体化建设提供了一条人流、物流、资金流快速交换通道。

== 建设进程 ==
*2008年11月9日动工。
*2009年10月10日，首座隧道贯通。
*2011年6月22日，南广铁路关键控制性工程——花培岭隧道提前贯通，成为南广铁路全线四大控制性工程中第一座提前贯通的隧道，是南广铁路建设的里程碑。<ref>{{Cite web |url=http://www.xinhuanet.com/chinanews/2011-06/23/content_23077537.htm |title=南广铁路关键控制性工程——花培岭隧道提前贯通 |date=2011年6月22日 }}</ref>
*2012年5月22日，南广铁路松根隧道顺利贯通，松根隧道位于广东省云浮市郁南县建城镇与宝珠镇交界处，全长3123米。
*2014年3月4日，南廣鐵路北嶺山隧道順利貫通，它是南廣鐵路線上最后一座貫通的隧道，北嶺山隧道位於廣東省肇慶境內，全長12438米，是南廣鐵路全線最長的一座隧道。<ref>{{Cite web |url=http://gx.people.com.cn/BIG5/n/2014/0304/c179430-20699541.html |title=南廣鐵路最后一座隧道勝利貫通 年底全線開通運行 |date=2014年3月4日 }}</ref>
*2014年4月18日，南广铁路广西段开通。南宁到梧州南仅需2小时半。<ref>{{cite web |url=http://www.gx.xinhuanet.com/newscenter/2014-04/18/c_1110303518.htm |title=存档副本 |accessdate=2015-10-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20150924024550/http://www.gx.xinhuanet.com/newscenter/2014-04/18/c_1110303518.htm |archivedate=2015-09-24 }}</ref>
*2014年7月7日，梧州至肇庆段铺接到肇庆东站，南广铁路实现全线贯通。<ref>[http://www.gx.xinhuanet.com/dtzx/wzswz/2014-07/10/c_1111554197.htm 南广铁路铺轨接入肇庆东站] {{webarchive|url=https://web.archive.org/web/20140725214800/http://www.gx.xinhuanet.com/dtzx/wzswz/2014-07/10/c_1111554197.htm |date=2014-07-25 }}</ref>
*2014年11月8日，南广铁路最后一段进入全线联调联试，为14年年底全线通车奠定基础。<ref>[http://news.163.com/14/1109/16/AAKDGPKU00014AED.html 南广铁路最后一段进入全线联调联试]</ref>
*2014年12月26日，南广铁路全线通车。<ref>[http://news.xinhuanet.com/photo/2014-12/26/c_127337917.htm 贵广、南广高铁正式开通]</ref>

== 现状 ==
南广铁路自2014年12月26日全线开通运营以来，客流持续处于旺盛的状态。铁路客流统计数据显示，南广铁路全线营运仅18天，已累计运送旅客69.5万人次，动车全程客座平均利用率达到92.5%，位居全国营运中各条高铁路线前列。2015年[[春运|春运]]期间，部分原先结伴骑[[摩托車|摩托車]]从广东返回广西的[[民工|民工]]改乘南广动车组列车返乡。<ref>{{cite news|url=http://www.chinanews.com/gn/2015/02-08/7044686.shtml|title=从“摩托大军”到高铁乘客：曾一天酿交通事故15起|publisher=[[工人日报|工人日报]]|date=2015-02-08|accessdate=2015-02-09}}</ref><ref>{{cite news|url=http://ndgd.oeeee.com/nis/201502/08/326124.html?c=sz|title=200名广西外来工免费坐南广高铁回家|publisher=[[南方都市报|南方都市报]]|date=2015-02-08|accessdate=2015-02-09|deadurl=yes|archiveurl=https://web.archive.org/web/20150209154433/http://ndgd.oeeee.com/nis/201502/08/326124.html?c=sz|archivedate=2015-02-09}}</ref>

== 長遠發展 ==
根據規劃，南廣鐵路將直接接入[[東盟|東盟]]的[[泛亚铁路_(中国)|泛亞鐵路]]網絡，屆時，從[[廣州|廣州]]坐火車出發，經[[廣西|廣西]]可前往[[老撾|老撾]]、[[泰國|泰國]]、[[新加坡|新加坡]]等國家。 

南廣鐵路是《中長期鐵路網規劃》的重點項目。這條鐵路能夠在[[中國西南|中國西南]]、[[華南|華南]]間形成一條最便捷、最快速的鐵路運輸通道，把[[北部灣經濟區|北部灣經濟區]]與[[珠三角地區|珠三角地區]]更加緊密地聯繫在一起。並將形成由中國東南沿海通往東盟國家的國際運輸大通道，加強中國與東盟國家間的經貿等各領域的交流與合作。南廣鐵路通車後，旅客可直接從[[廣州|廣州]]乘車至[[越南|越南]]，最終組成[[泛亞鐵路|泛亞鐵路]]東線方案，連接到[[新加坡|新加坡]]。<ref>{{Cite web |url=http://news.sina.com/ch/phoenixtv/102-101-101-102/2008-11-11/23533417407.html |title=南廣鐵路動工 廣州火車將直通新加坡 |date=2008年11月11日 }}</ref>
