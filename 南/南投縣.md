'''南投縣'''（{{lang-nan-tw|'''Lâm-tâu Kuān'''}}；[[國姓鄉|國姓鄉]][[四縣腔|四縣客語]]：'''Namˇ Teuˇ Ien'''），是[[中華民國|中華民國]][[臺灣省|臺灣省]]的[[縣_(中華民國)|縣]]，位處[[臺灣|臺灣]][[中臺灣|中部]]，坐落在本島正中央，是臺灣唯一的內陸縣，縣治為[[南投市|南投市]]。

縣境內[[台灣原住民|原住民]]有[[泰雅族|泰雅族]]、[[賽德克族|賽德克族]]、[[布農族|布農族]]、[[鄒族|鄒族]]及居住在日月潭畔之[[邵族|邵族]]。臺灣最高峰[[玉山|玉山]]、最大天然湖泊[[日月潭|日月潭]]、最長河流[[濁水溪|濁水溪]]的源頭與[[臺灣地理中心|臺灣地理中心]]皆位於該縣，而鄰近彰化與台中一帶因人口聚集，形成中部衛星都市的分佈型態。特產包括[[青梅|青梅]]、[[香蕉|香蕉]]、[[甘蔗|甘蔗]]、[[凍頂烏龍茶|凍頂烏龍茶]]、[[紹興酒|紹興酒]]、[[竹藝品|竹藝品]]、南投陶、[[花卉|花卉]]。

[[南投|南投]]地名由來，源自[[平埔族|平埔族]][[洪雅族|洪雅族]]支族'''[[阿立昆族|阿立昆族]]'''（Arikun）[[洪雅語|族語]]「'''Ramtau'''」，即以[[臺語|臺語]]音譯為「南投」<ref name="南投的地方起源與變遷">[http://www.taiwan123.com.tw/local/name03-8.htm 南投的地方起源與變遷] {{webarchive|url=https://web.archive.org/web/20120206023503/http://www.taiwan123.com.tw/local/name03-8.htm |date=2012-02-06 }}</ref>。古志日:「虎尾之北有大武郡山，東為南投山，內社二，溪南為南投，溪北為北投，故名「南投」。

南投在[[明鄭時期|明鄭時期]]屬[[天興縣|天興縣]]（註：今臺南市以北均屬之），後屬北路安撫司。1683年（清康熙二十二年）隸屬於[[福建省|福建省]]臺灣府[[諸羅縣|諸羅縣]]。1723年（雍正元年）增設[[彰化縣|彰化縣]]，該縣隸屬之。至1759年（乾隆二十四年）正式設縣丞，此為南投縣有政教之始。1887年（光緒十三年）福建、臺灣分治，臺灣改制為臺灣省，這時的南投行政區屬[[臺灣府|臺灣府]][[臺灣縣_(1887年)|台灣縣]]（縣治在今[[臺中市|台中市]]南區），縣丞駐'''[[南投堡|南投堡]]'''。同時[[雲林縣|雲林縣]]衙設在[[竹山鎮_(台灣)|竹山]]，管轄[[雲林縣_(清朝)|雲林縣]]。

1894年（光緒二十年）[[甲午戰爭|甲午戰爭]]清廷戰敗，遂於第二年（1895年）將台、澎割讓[[日本|日本]]。1895年劃入[[臺中縣_(日治時期)|臺中縣]]，1901年獨立為'''[[南投廳|南投廳]]'''，1920年改為[[臺中州|臺中州]]'''[[南投郡|南投郡]]、[[竹山郡|竹山郡]]、[[新高郡|新高郡]]、[[能高郡|能高郡]]'''。

1945年日本戰敗，戰後初期該縣仍屬-{台中}-縣轄。1950年10月，臺灣地區實施[[地方自治|地方自治]]，政府調整全省行政區，南投自-{臺中}-縣劃出單獨設'''南投縣'''，並於是年10月21日設立南投縣政府，此為南投縣之誕生。縣治設於南投鎮（今[[南投市|南投市]]）。1957年7月1日，[[臺灣省政府|臺灣省政府]]遷至南投縣南投市營盤口設置[[中興新村|中興新村]]至2018年7月1日省政府[[De_facto|實質上]]解散為止，南投成為當時臺灣省[[省會|省會]]所在地。

===南投縣行政區劃演變===
自清代設立南投縣丞起，先後屬彰化縣及臺灣縣下，處理南投地區行政。而後日治初期劃分的1895年「臺灣民政支部」及1896年「臺中縣」時期，南投直隸於支部、縣之下，於1897年後方恢復區域行政機能。<ref>[http://thcts.ascc.net/kernel_ch.htm 臺灣歷史文化地圖核心應用系統]</ref>

== 地理 ==
南投縣縣境絕大部分位於[[北回歸線|北回歸線]]以北（北回歸線通過其極南端），位於台灣島正中央，亦是唯一不靠海的縣份，但有唯一的島嶼：[[拉魯島|拉魯島]]，位於[[日月潭|日月潭]]。'''南投縣'''信義鄉、[[高雄市|高雄市]][[桃源區|桃源區]]及[[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]交界處的[[玉山|玉山]]，為[[臺灣|台灣島]]上最高峰。縣內眾多的高山擋下來自[[南太平洋|南太平洋]]與[[東太平洋|東太平洋]]的[[季風|季風]]水氣，提供[[台灣|台灣]]豐富的雨水來源，使[[台灣|台灣]]較同[[緯度|緯度]]的地區擁有更多的[[年雨量|年雨量]]；縣境內並孕育出臺灣中部四大水系的其中兩個（[[濁水溪|濁水溪]]水系、[[烏溪|烏溪]]水系）。

南投縣境內各地的原住民族有[[賽德克族|賽德克族]]、[[泰雅族|泰雅族]]、[[布農族|布農族]]、[[鄒族|鄒族]]及[[邵族|邵族]]。

=== 生活圈分區 ===
[[File:NantouLife.png|250px]]

南投縣以其境內生活圈可分為下列四大區塊<ref>[http://gisapsrv01.cpami.gov.tw/cpis/cprpts/NANTOU/INDEX.HTM 南投縣綜合發展計畫報告書] {{webarchive|url=https://web.archive.org/web/20100909012826/http://gisapsrv01.cpami.gov.tw/CPIS/cprpts/NANTOU/INDEX.HTM |date=2010-09-09 }}</ref>：
*南投區：位於南投縣西北，包含南投市、草屯鎮、名間鄉、中寮鄉，以南投市、草屯鎮兩地為主要市鎮，[[彰化縣|彰化縣]][[芬園鄉|芬園鄉]]的人文經濟活動與本區也相當密切；分區人口數共有約25.2萬多人。
*埔里區：位於南投縣東北，包含埔里鎮、國姓鄉、魚池鄉、仁愛鄉，以埔里鎮為核心市鎮，埔里鎮亦為通往日月潭、清境農場等旅遊景點之重要樞紐，[[921大地震|921大地震]]後，由於[[中橫公路|中橫公路]]主線毀損，[[梨山|梨山]]地區與本生活圈有更多來往；分區人口數共有約13.1萬多人。
*竹山區：位於南投縣西南，包含竹山鎮、鹿谷鄉，以竹山鎮為其主要市鎮，竹山鎮亦為通往[[太極峽谷|太極峽谷]]、[[溪頭|溪頭]]、[[杉林溪|杉林溪]]必經之地，本區與[[雲林縣|雲林縣]][[斗六市|斗六市]]往來密切，亦與雲林為共同就學區，且雲林縣地名來源即為鎮內之「雲林坪」；分區人口數共有約7.3萬多人。
*水-{里}-區：位於南投縣東南，包含水-{里}-鄉、集集鎮、信義鄉，以水-{里}-鄉為主要市鎮，本生活圈也包含鹿谷鄉清水溝地區；分區人口數共有約4.5萬多人。

== 區劃人口 ==
[[File:南投行政區劃.png|thumb]]
民國三十九年（1950年）調整行政區域，將[[臺中縣|臺中縣]]所屬[[南投區|南投區]]的2鎮2鄉、[[玉山區|玉山區]]的1鎮2鄉、[[能高區|能高區]]的1鎮2鄉及[[竹山區|竹山區]]的1鎮1鄉等12個鄉鎮合併獨立為南投縣，並自集集鎮分出水-{里}-鄉，縣治設於南投鎮，轄5鎮、8鄉。民國七十年（1981年）南投鎮升格為縣轄市，全縣轄有1[[縣轄市|市]]（南投市）、4[[鎮_(中華民國)|鎮]]（埔里鎮、草屯鎮、竹山鎮、集集鎮）、8[[鄉_(中華民國)|鄉]]（名間鄉、鹿谷鄉、中寮鄉、魚池鄉、國姓鄉、水-{里}-鄉，仁愛鄉、信義鄉為[[山地鄉|山地鄉]]），共有13個鄉鎮市。

== 政治 ==
{{main|南投縣政府|南投縣縣長|南投縣議會|南投地方法院|臺灣地方派系|台灣選舉|臺灣中部地區選舉地理#南投縣}}
=== 縣政組織 ===
[[南投縣政府|南投縣政府]]是南投縣的最高[[行政機關|行政機關]]，在中華民國政府架構中為縣自治的行政機關，同時負責執行中央機關委辦事項，南投縣的自治監督機關為行政院各部會（主要為[[中華民國內政部|內政部]]）。縣長由全體縣民直接選舉產生，任期為四年，連選可連任一次。南投縣政府並置縣政會議，為縣政最高決策機構，在縣長及副縣長之下，設有14個內部單位、7個所屬一級機關、20個所屬二級機關、174所各級學校。

南投縣縣長是[[南投縣政府|南投縣政府]]之行政首長，負責綜理縣政，並指揮、監督所屬職員及機構。現任南投縣長[[林明溱|林明溱]]為[[中國國民黨|中國國民黨]]籍第17屆南投縣縣長。

[[南投縣議會|南投縣議會]]是南投縣的最高[[民意機關|民意機關]]，代表南投縣全體縣民立法和監察縣政府之施政。縣議員由公民直選選出，任期為四年，可連選連任。共有37位縣議員，第一選區10席縣議員、第二選區8席縣議員、第三選區4席縣議員、第四選區5席縣議員、第五選區7席縣議員、第六選區1席縣議員、第七選區1席縣議員、第八選區1席縣議員{{notetag|南投縣議會第一選區為南投市、名間鄉。第二選區為草屯鎮、中寮鄉。第三選區為集集鎮、水里鄉、魚池鄉、信義鄉。第四選區為竹山鎮、鹿谷鄉。第五選區為埔里鎮、國姓鄉、仁愛鄉。第六選區為平地原住民選區。第七選區為山地原住民選區（信義鄉、南投市、竹山鎮、鹿谷鄉、名間鄉、集集鎮、水里鄉）。第八選區為山地原住民選區（仁愛鄉、埔里鎮、國姓鄉、魚池鄉、草屯鎮、中寮鄉）。}}，議長、副議長由37位縣議員互選產生。

=== 司法機關 ===
南投縣之民、刑訴訟和審理由[[臺灣南投地方法院|臺灣南投地方法院]]管轄，[[臺灣高等法院臺中分院|臺灣高等法院臺中分院]]為上訴法院，並以[[最高法院_(中華民國)|最高法院]]為終審法院。

檢察機關由[[臺灣南投地方檢察署|臺灣南投地方檢察署]]偵辦，上級單位為[[臺灣高等檢察署臺中檢察分署|臺灣高等檢察署臺中檢察分署]]。

===選民結構及政治勢力===
南投縣位於臺灣中部，乃全國唯一不臨海的縣，縣內[[中興新村|中興新村]]為[[臺灣省政府|臺灣省政府]]所在地。該縣人口結構，以[[台灣河洛人|河洛人]]為主，[[台灣原住民|原居民]]人口也較為集中，[[台灣外省人|外省人]]比例則較少<ref name="林清察"/>。過去該縣選民結構藍大於綠，泛藍領先泛綠五個百分點左右，泛藍在各項選戰中得票皆領先泛綠，但近期已呈現藍綠五五波的態勢<ref name="林清察">《台灣與大陸政治評論集》，林清察，2006年</ref>。當地政治生態主要由三大勢力主導，即[[泛藍|泛藍]]的舊派、[[無黨派|無黨派]]的新派、及[[泛綠|泛綠]]的山頭勢力<ref name="王建民">《台灣地方派系與權力結構》，王建民，2003年。</ref>。而在歷屆總統大選，泛藍得票較全國領先兩到三趴。就地域而言，新派及泛綠票源在平原較為集中。舊派的經營則較為扎實，在全縣均有穩定的票源分布，尤其在山區優勢更為明顯<ref name="王建民"/>。

該縣泛藍得票率較為穩定，泛綠表現則較不穩定<ref name="南投縣選民">[http://www.doc.ncnu.edu.tw/chisp/files/statistics/electorate-analysis.doc 南投縣選民結構分析]，南投縣政府主計處</ref>。由於泛綠長期面臨內部整合問題，選舉常呈現分裂的狀態。然而，該縣亦曾於1990年代末期起，連續八年綠色執政。而整體而言，該縣[[無黨籍|無黨籍]]人士得票亦占有相當比率，其中縣議員及鄉鎮市長選舉皆占五成以上，顯示縣民投票並非以政黨為主要前提<ref name="南投縣選民"/>。

===文學===
;文學史及相關論述
目前為止，還沒有一本關於南投區域文學史、南投地區古典文學，以及南投戰後文學發展之類的書籍出版，在市面上流通。至於碩博士論文，有張宏誠〈台灣解嚴後縣市文學獎研究(1987~2010)〉、劉麗卿〈清代[[台灣八景|台灣八景]]與[[八景詩|八景詩]]〉、蘇玉筑〈當代台灣「大台中」書寫─以中、彰、投四縣市作家作品為探討對象〉、張淑玲〈台灣南投地區[[傳統詩|傳統詩]]研究〉、黃宏介〈南投地區民間現存[[傳統詩社|傳統詩社]]研究〉等論文。總之，南投地區的傳統詩、傳統詩社、當作的作家作品等議題，已經有人研究，雖然數量不多，有很大的進步空間，但已經不是文學研究的空白區域。

;民間文學與古典文學
縣文化局在2006年出版《生事歸清恬：張達修詠讚台灣漢詩百首精選譯註》與《醉草園文集》；竹山鎮公所在2002年出版《竹山民間故事集》。詳細情況請見「南投縣文學資料館」網站的「文學年表」<ref>該網頁網址：http://literature.nthcc.gov.tw/05year/05year.asp{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>。

;基金會、文學團體
推廣及贊助文學活動的基金會，目前有「南投縣文化基金會」<ref>台灣文學館出版《2007 台灣文學年鑑》的「文學活動基金會」名錄。</ref>；地方性的文學社團有「南投縣文藝寫作協會」、「南投藍田書院詩學研究社」<ref>台灣文學館出版《2007 台灣文學年鑑》的「文學團體」名錄。</ref>。

;文學活動
南投縣文化局每年舉辦「[[玉山文學獎|玉山文學獎]]」徵文活動、頒獎典禮，並且將得獎作品的集結出版。其次，縣文化局已經建構「南投縣文學資料館」<ref>位在南投縣政府文化局圖書館4樓，地址：南投市中興路669號4樓。網址：{{cite web |url=http://literature.nthcc.gov.tw/00home/home.asp |title=存档副本 |accessdate=2012-03-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20120320215416/http://literature.nthcc.gov.tw/00home/home.asp |archivedate=2012-03-20 }} </ref>，並且在南投中興新村虎山藝術館的後山，設置「南投縣文學步道」步道上設19座文學導覽牌，將作家的文學佳句雕刻在玉石或琉璃上，供民眾閱覽。此外，縣文化局還陸續出版「南投縣文學家作品集」。詳細情況請見「南投縣文學資料館」網站的「文學年表」<ref>網頁網址：http://literature.nthcc.gov.tw/05year/05year.asp{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>。

;縣籍作家
在南投縣出生的作家、文學評論者有：[[張良澤|張良澤]]、[[白棟樑|白棟樑]]、[[天官賜|天官賜]](本名：孔維勤)、[[李文賢|李文賢]]、[[林柏維|林柏維]]、[[郁馥馨|郁馥馨]]、[[單德興|單德興]]、[[游喚|游喚]](本名：游志誠)、[[黃忠慎|黃忠慎]]、[[熊秉元|熊秉元]]、[[劉靜娟|劉靜娟]]、[[潘煊|潘煊]](本名：潘玉秀)、[[盧兆琦|盧兆琦]]、[[賴樹明|賴樹明]] <ref>台灣文學館，「2007台灣作家作品目錄」搜尋系統。</ref>、[[鄧相揚|鄧相揚]]、[[向陽_(詩人)|向陽 (詩人)]]、[[陳千武|陳千武]]、[[岩上|岩上]]、[[李威熊|李威熊]]、[[李瑞騰|李瑞騰]]、[[王灝|王灝]]、[[林廣|林廣]]、[[林世明|林世明]]等人。
