'''南星宮新大眾廟'''，是位於[[臺灣|臺灣]][[新竹市|新竹市]][[東區_(新竹市)|東區]]光鎮-{里}-的[[地藏庵|地藏庵]]，原為[[陰廟|陰廟]]，主祀[[地藏王菩薩|地藏王菩薩]]及[[十殿閻君|十殿閻君]]、五方五壇[[大眾爺|大眾爺]]。

== 緣起 ==
'''南星宮新大眾廟'''廟址原為亂葬崗，昭和13年（1938年）原[[南壇大眾廟|南壇大眾廟]]之齋心普化祠及納骨堂遷移至此處新建之納骨塔，1965年再由[[林山燕|林山燕]]及[[吳火獅|吳火獅]]等人捐款，拆除改建該納骨塔，號為'''南星宮新大眾廟'''，並新雕五方大眾爺，於1982年11月落成，2007年向內政部登記成立「財團法人佛教新大眾基金會」<ref name="江怡葳"/>。

== 祭典 ==
每年農曆7月12日，南星宮新大眾廟及南壇大眾廟會舉辦「牽-{zh-hant:{{缺字|zhuang4|1}}; zh-hans:{{缺字|zhuang4|2}};}- 」儀式，新大眾廟由兩位高功道士同時主持「雙壇普渡」 ，及「拜腳-{zh-hant:{{缺字|zhuang4|1}}; zh-hans:{{缺字|zhuang4|2}};}- 媽」的儀式，為本廟有別於其他廟宇普渡的特色<ref name="黃進仕"/>。

以前新竹若有車禍頻傳、[[自殺|自殺]]等不寧靜的區域，會請[[城隍爺|城隍爺]]與六將前去進行「祭煞」，抓到「煞」之後再送往頭前溪放水流，但反而造成溪邊意外，現在則是把抓到的「煞」，送到南星宮新大眾廟<ref name="江怡葳"/>。

== 日本人塚 ==
[[File:2018_New_Dazhong_Temple_日本人塚.jpg|thumb]]

昭和20年（1945年）[[日本二戰投降|日本二戰投降]]，[[國民政府接管臺灣|國民政府接管臺灣]]，日裔民眾遭遣返日本，[[日本禪宗|日本禪宗]][[曹洞宗|曹洞宗]]新竹禪寺住職[[佐久間尚孝|佐久間尚孝]]禪師最後方歸，1946年歸國前將寺內信徒寄託靈骨五百餘，親自移葬於新大眾廟後，並立碑誌之<ref name="余"/>。
