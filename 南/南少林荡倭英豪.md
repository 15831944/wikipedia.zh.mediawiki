== 剧情 ==
[[明朝|明朝]]中后期的[[嘉靖|嘉靖]]二十九年，来自[[扶桑|扶桑]]的[[倭寇|倭寇]]大举入侵江浙闽一带，石井[[镖局|镖局]]总镖头林钦一家在战争中遭遇不幸，家园毁坏，二儿子林展鹏被倭人掳走，大儿子林展翔和三儿子林展浩幸亏得到[[南少林|南少林]]佛门弟子的帮助才得以幸免于难。

倭寇头目三枝图田精通刀法，他将年幼的林展鹏收为义子，赐名“三枝乱步”。

林钦的大儿子林展翔不久之后爱上了少女陆燕怀，然而，陆燕怀在次年倭寇入侵中被杀。林钦的三儿子林展浩加入南少林，苦学武功，成为武学大师。

三枝乱步残忍地杀掉了义父三枝图田，自己当了倭寇头目，多年之后，他入侵明朝，被林展翔和林展浩打败，自杀。

== 花絮 ==
电影林楠表示，该剧为“武侠战争史诗传奇，儿女情、兄弟情、家国情将成为本剧的三大主线。”<ref>{{cite web |url=http://ent.ifeng.com/tv/news/mainland/detail_2011_11/09/10536319_0.shtml |title=《南少林荡倭英豪》主演曝光 打造武侠战争史诗传奇 |accessdate=2011年11月9日 |publisher= 凤凰网|work= |language =zh-cn}}</ref>

电视剧使用了大量的古代中国的[[兵法|兵法]][[阵法|阵法]]以及古代日本的兵法阵法，如[[真田幸村|真田幸村]]的“[[影武者搅乱阵法|影武者搅乱阵法]]”、“军神”[[上衫兼信|上衫兼信]]的“[[车悬之阵|车悬之阵]]”。<ref name=" fish" >{{cite web |url=http://yule.sohu.com/20110927/n320680263.shtml |title=《南少林荡倭英豪》开机 震撼概念片花曝光 |accessdate=2011年9月27日 |publisher=搜狐网 |work= |language =zh-cn}}</ref>

泉州少林寺方丈[[常定法师|常定法师]]为电视剧的特别顾问。<ref name=" fish" />

电视剧花费人民币5000万。<ref name=" fish" />

电视剧的打斗场面甚多，演员张帆说内心犯怵。<ref>{{cite web |url= http://news.cntv.cn/20111207/100505.shtml|title=《南少林荡倭英豪》打戏多 张帆坦言犯怵 |accessdate=2011年12月7日  |publisher=中国网络电视台 |work= |language =zh-cn}}</ref>

电影在沿海多地取景，包括[[福建省|福建省]][[泉州市|泉州市]]，[[漳州|漳州]][[火山岛|火山岛]]，[[浙江省|浙江省]][[横店影视城|横店影视城]]。<ref>{{cite web |url=http://roll.sohu.com/20120116/n332282305.shtml |title=《南少林荡倭英豪》来泉拍摄 |accessdate=2012年1月16日 |publisher=搜狐网 |work= |language =zh-cn}}</ref>
