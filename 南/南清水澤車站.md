'''南清水澤站'''（{{jpn|j=南清水沢駅|hg=みなみしみずさわえき|rm=Minami-Shimizusawa eki|f=y}}）是位於[[北海道|北海道]][[夕張市|夕張市]]南清水澤一丁目，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[石勝線|石勝線]]（夕張支線）已廢止的[[鐵路車站|鐵路車站]]。車站編號為'''Y22'''。電報略號為'''{{lang|ja|ミミ}}'''。

== 歷史 ==
* 1962年（昭和37年）12月25日 - [[日本國有鐵道|日本國有鐵道]]'''南清水澤站'''開始營運，處理旅客服務。
* 1967年（昭和42年）11月1日 - 車站業務委託化。
* 1984年（昭和59年）3月31日 - 車站簡易委託化。
* 1987年（昭和62年）4月1日 - 由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理。
* 2019年（平成31年）4月1日 - 隨者石勝線夕張支線廢線，本站廢止。

== 車站構造 ==
廢站時是[[側式月台|側式月台]]1面1線的地面車站。

由[[新夕張站|新夕張站]]管理的簡易委託站（窗口營業時間為5:45 - 17:00）。

== 載客量 ==
* 1998年度的每天載客量為128人。

== 車站周邊 ==
過去由於夕張南高校（現在的夕張高校）搬遷至鄰近的農地，周圍開始形成市區。車站前有若干的商店並列著。
* [[國道452號|國道452號]]
* 夕張南清水澤郵局
* 夕張市立夕張小學
* 夕張市立夕張中學
* 北海道夕張高等學校
* ACOOP夕張南清水澤
* HOMAC NICOT夕張店
* Seicomart夕張南清水澤店
* 夕張鐵道（夕鐵巴士）「南清水澤站前」站牌
