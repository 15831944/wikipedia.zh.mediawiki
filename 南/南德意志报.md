《'''南德意志报'''》<ref>《世界报刊、通讯社、电台译名手册》修订本，辛华（即[[新华通讯社|新华通讯社]]译名室），北京：[[商务印书馆|商务印书馆]]，1978</ref>（[[德语|德语]]：'''''{{lang|de|Süddeutsche Zeitung}}'''''，读作{{IPA-de|ˈzyːtˌdɔʏtʃə ˈtsaɪtʊŋ|}}，缩写：{{lang|de|''SZ''}}；又譯為《'''南德日報'''》）是[[德国|德国]]发行量最大的以订阅方式发行的全国性报纸，由[[南德意志出版社|南德意志出版社]]发行，报社总部位于[[慕尼黑|慕尼黑]]。

== 发行 ==
《南德意志报》通过订阅方式在德国全国发行，每周发行六天。报纸的日发行量超过44万份，读者人数超过110万。

== 历史 ==
1945年10月6日，德国投降5个月之后，《南德意志报》成为战后首份获得驻[[拜恩|拜恩]][[美军|美军]]签发的发行许可证的报纸，在获得许可证的当晚首期报纸即付印。当时报纸的售价为20[[德国马克|芬尼]]。

在报纸的发刊词中这样写到：

{{Cquote2|自黑暗的恐怖统治崩溃以来在[[慕尼黑|慕尼黑]]首次出现了一份由德国人主持的报纸。虽然它还受限于目前的政治必然性，但却不受检查制度的束缚，不被压制以违心行事。《南德意志报》不是[[政府|政府]]或某个[[党派|党派]]的喉舌，而是所有团结一致热爱[[和平|和平]]、憎恨[[极权主义|极权]]国家、厌恶一切[[纳粹主义|纳粹主义]]事物的德国人的扬声筒。}}

第一版的标题页会在这里找到。<ref>[http://www.sueddeutscher-verlag.de/assets/sv_geschichte/451006_sz_erstausgabe.pdf 第一期《南德意志报》头版（pdf档） {{de}}]</ref>

在新千禧年初期，广告收入的直线下降使该报社进入了财务寒冬，到2002年10月份该报社濒临破产。但随后，一家掌控地区报系的公司以1.5亿[[欧元|欧元]]的价格成为了该报社的控股方，并成立了南德意志报业媒体。在随后的三年中，报社为了控制开支，进行了大刀阔斧的改革，包括但不限于：把原来的425项职位裁剪到307项职位，关闭位于杜塞尔多夫的分社，以及将专门报道柏林地区新闻的专栏从报纸中去除。

在2004年春季，SZ开启了一份项目：50部著名的20世纪小说将会在每周以精装书的形式在某些报摊和书店发行，之后50部以DVD形式发行的著名电影也随之开始。在2004年后期该报社开始发行属于自己的流行科学杂志，而在2005年，多部著名儿童图书也相继开始以精装书的形式发行。

在2015年，该报社从一位匿名人士收到了总值2.6TB的材料，里面包含大量与未知会计机构有关的机密文件。南德意志报随后与[[国际调查记者同盟|国际调查记者同盟]]进行了为时一年的调查，并在2016年4月3日陆续公布了文件里的主要内容，史称[[巴拿马文件|巴拿马文件]]。

== 概况 ==
《南德意志报》在内政上持支持[[自由主义|自由主义]]立场，在经济政策上支持[[自由主义经济|自由主义经济]]。因此该报也被戏称为拜恩州的惟一反对党，因为拜恩州(Bayern)长期以来一直由[[保守主义|保守]]派立场的[[基社盟|基社盟]]执政。

全国发行的《南德意志报》包含四个部分：[[政治|政治]]、[[文化|文化]]、[[经济|经济]]、[[体育|体育]]。在慕尼黑当地发行的地方版还包括刊登地方新闻的副刊。

《南德意志报》在每周一都有一份选登《[[纽约时报|纽约时报]]》文章的增刊，每周五有《SZ杂志》副刊，每周六有周末副刊，每周四出版的慕尼黑地方版还有一份刊登各种即将举行的活动的增刊。

== 社会影响 ==
《南德意志报》第一版名为“高光”（''Streiflicht''）的短评专栏具有很高的知名度，自1946年起在每期报纸头版的左上角都会刊登一篇讽刺性短评（''Glosse''），而且每篇短评都固定有72行。所有短评虽然都是匿名发表，但实际上通常都是由知名作者撰写，因此它们经常能够排入当天德国媒体的报道日程。

此外，报纸第三版的长篇报道和背景文章，第四版的由该报知名作者撰写的社论也是《南德意志报》的特色。

2004年8月《南德意志报》、《[[明镜周刊|明镜周刊]]》以及[[阿克塞尔施普林格股份公司|阿克塞尔施普林格股份公司]]旗下报纸共同决定放弃使用新[[德语正寫法|德语正寫法]]恢复使用旧正寫法，同时《法兰克福汇报》也决定恢复使用旧正寫法。这些报刊的举动大大加剧了对[[1996年德语正写法改革|德语正寫法改革]]的争论。

2005年，在由[[新闻学|新闻学]]家Siegfried Weischenberg主持的一项有1536名德国新闻记者参与的调查研究中，经过记者们的选择，《南德意志报》（34.6％）超过《明镜周刊》（33.8％）被认为是德国的“重点媒体”（''Leitmedium''）。
