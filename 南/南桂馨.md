'''南桂馨'''（{{bd|1884年|3月23日|1966年|9月14日}}）<ref>生卒时间见北京档案史料2007（1-2），第242页</ref>字'''佩蘭'''，[[山西省|山西省]][[宁武县|宁武县]]城内人。中华民国、中华人民共和国政治人物。<ref name=xzrb>[http://www.xzrbw.com/info/1277/52094.htm 宁武南桂馨的辛亥革命，忻州日报网，2012-04-26] {{webarchive|url=https://web.archive.org/web/20131029184601/http://www.xzrbw.com/info/1277/52094.htm |date=2013-10-29 }}</ref><ref>{{Cite book|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref>{{Cite book|author = 劉国銘主編|title = 中国国民党百年人物全書|year = 2005|publisher = 团結出版社|isbn = 7-80214-039-0}}</ref>

==生平==
南桂馨自幼入[[私塾|私塾]]，后来中[[秀才|秀才]]。1902年，入[[山西大学堂|山西大学堂]]西斋学习。1906年，参加[[中国同盟会|中国同盟会]]。1907年，留学日本。<ref name=xzrb/>在東京警察学校学习。<ref>徐主編（2007）、1009頁及劉国銘主編（2005）、1708頁都有该描述，但实际上并没有叫“东京警察学校”的学校，故具体所指的是哪所学校不详。</ref>1908年归国。此后历任[[神池县|神池县]]自治传习所所长、太原禁烟局委员等职务，后担任[[新建陆军|新军]][[陆军第四十三混成协|第四十三混成协]]第八十五标军需官。<ref name=xzrb/>

[[辛亥革命|辛亥革命]]爆发后，1911年10月28日，南桂馨参加了发动[[太原起义|太原起义]]的决策会议。太原起义成功后，他奉[[山西军政府|山西军政府]]之命，赴南京求援。[[南北议和|南北议和]]期间，他敦请孙中山严拒袁世凯不承认山西省为起义省份的图谋。[[中华民国|中华民国]]成立后，南桂馨历任山西巡警道、代理晋北镇守使、河东筹饷局局长、粮服局长、同武将軍公署参謀長。1917年（[[民国紀元|民国]]6年）9月<ref>劉国銘主編（2005）、1708頁作此。徐主編（2007）、1009頁作1924年。</ref>，出任山西全省警務处处長兼省会警察厅厅長。<ref name=Xu>徐主編（2007）、1009頁。</ref><ref name=Liu>劉国銘主編（2005）、1708頁。</ref><ref name=xzrb/>

[[阎锡山|阎锡山]]投向[[国民政府|国民政府]]后，1928年2月，南桂馨任[[中国国民党|中国国民党]]中央政治会議太原分会委員。1928年3月，任山西省政府委員兼民政厅厅長。1928年6月，出任[[天津特别市政府|天津特别市政府]]市長，在任3个月。此後，历任中国国民党山西省党務指導委員、[[国民政府軍事委員会|国民政府軍事委員会]]委員。阎锡山在[[中原大战|中原大战]]中敗北下野後，1930年（民国19年）11月，南桂馨调任[[陝西省_(中華民国)|陝西省]]政府委員。1931年（民国20年）10月，被任命为[[中華民国立法院|立法院]]立法委員。1935年（民国24年）6月，调任[[河北省_(中華民国)|河北省]]政府委員。他还担任全国水利委員会委員。<ref name=Xu/><ref name=Liu/><ref name=xzrb/><ref>{{Cite book|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

[[抗日战争|抗日战争]]期间，他在天津闲居，拒不出任亲日政权职务。1946年（民国35年）11月，当选[[制憲国民大会代表|制憲国民大会代表]]。1947年（民国36年）3月，任憲政实施促進委員会常務委員。1948年3月，任[[第一届国民大会|第一届国民大会]]代表。同年任[[傅作义|傅作义]]部顾问等职务。<ref name=Xu/><ref name=Liu/><ref name=xzrb/>

[[中华人民共和国|中华人民共和国]]成立后，他担任了联合银行董事、[[山西省政协|山西省政协]]委员、[[北京市文史研究馆|北京市文史研究馆]]馆员等职。<ref name=xzrb/><ref name=bjda>北京档案史料2007（1-2），第242页</ref>

1966年9月14日，南桂馨在北京病逝，享年83岁。<ref name=xzrb/><ref name=bjda/>
