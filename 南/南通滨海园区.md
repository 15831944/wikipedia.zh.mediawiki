'''南通滨海园区'''，是[[南通市|南通市]]政府为开发沿海[[通州湾|通州湾]]地区成立的一个经济开发区。

目前实际区域包括[[通州区_(南通市)|通州区]][[三余镇|三余镇]]和[[如东县|如东县]][[大豫镇|大豫镇]]。

南通远期规划有市域轨道交通连接滨海园区和市区核心区。

==歷史==
2011年7月，临海公路南通滨海园区段開始動工<ref>{{cite web |url=http://js.people.com.cn/html/2013/08/10/248115.html|title=临海公路南通滨海园区段预计年底前建成通车 |date=2013-08-10 |agency=人民網 |access-date= 2018-04-24}}</ref>。

2012年11月14日，通州湾至通洋高速快速通道開始動工<ref>{{cite web |url=http://www.zgnt.net/content/2012-11/15/content_2119995.htm |title=通州湾快速通道开工 建成后市区至南通滨海园区仅半小时车程[图] |date=2012-11-15 |publisher=南通网 |access-date=2018-04-24 |archive-url=https://web.archive.org/web/20140902193216/http://www.zgnt.net/content/2012-11/15/content_2119995.htm |archive-date=2014-09-02 |dead-url=yes }}</ref>。

2014年8月28日，以外语旅游为特色的「[[江南外语城|江南外语城]]」项目在园区正式动工<ref>{{cite web |url=http://js.ifeng.com/nt/business/industry/detail_2014_08/29/2848091_0.shtml|title=“江南外语城”南通滨海园区开建 |date=2014-08-29 |agency=江海明珠网 |publisher=鳯凰江蘇 |access-date= 2018-04-24}}</ref>。

2015年1月7日，市人大視察園區<ref>{{cite web |url=http://www.zgnt.net/content/2015-01/08/content_2372630.htm|title=市人大视察南通滨海园区 通州湾旅游新品成亮点 |date=2015-01-08 |publisher=南通网 |access-date= 2018-04-24}}</ref>。

2016年4月26日，园区边防开始治理成品油走私的問題<ref>{{cite web |url=http://www.legaldaily.com.cn/police_and_frontier-defence/content/2016-04/26/content_6602126.htm?node=23301 |title=江苏南通滨海园区边防开展成品油走私综合治理工作 |date=2016-04-26 |publisher=法制网 |access-date= 2018-04-24}}</ref>。
