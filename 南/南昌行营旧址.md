'''南昌行营旧址'''是一座位于江西省南昌市中山路71号<ref name="a1"/>的五层欧式建筑<ref>{{Cite web |url=http://www.people.com.cn/GB/shenghuo/1094/2007445.html |title=民国南昌三大建筑之一行营：被城市遗忘的过去 |accessdate=2017-07-12 |author= 万庆博、汪钱萍、黄铭，责任编辑：郭亚飞|date=2003-08-08 |work= |publisher=人民网|language=Zh-hans }}</ref>，地处[[東湖_(南昌)|东湖]][[百花洲_(南昌)|百花洲]]之中。1928年10月建成时为[[江西省立图书馆|江西省立图书馆]]馆舍，以1933年至1935年间为[[国民政府军事委员会委员长南昌行营|国民政府军事委员会委员长南昌行营]]驻地知名。江西省立图书馆时期为民国时期南昌三大建筑之一。

南昌行营被撤销后，江西省立图书馆搬回此处。[[抗日战争|抗日战争]]期间，前楼三层建筑被日军炸毁，于1946年修复<ref name="a2"/>，一说在1947年按原样重建前楼<ref name="a1">{{Cite web|url=http://www.nc.gov.cn/ljnc/rwys/wbdw/jxswwbhdw/200809/t20080925_108121.htm|title=江西省立图书馆旧址|accessdate=2017-07-12|author=|date=2008-09-25|work=|publisher=中国南昌，中共南昌市委、南昌市人民政府主办网站，稿源：南昌市文化局|language=Zh-hans}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1995年，[[江西省图书馆|江西省图书馆]]搬至南昌市洪都北大道。此幢建筑闲置，后成[[危房|危房]]。2006年12月，[[江西省人民政府|江西省人民政府]]以“[[江西省立图书馆旧址|江西省立图书馆旧址]]”之名列为第五批[[江西省文物保护单位|江西省文物保护单位]]。2013年，列入[[涉台文物|涉台文物]]。2014年8月，<ref name="a2">{{Cite web |url=http://news.163.com/15/1215/10/BASAV6DL00014AEE.html|title=蒋介石南昌行营拟明年开工“复原”(组图)|accessdate=2017-07-12 |author= |date=2015-12-15 |work= |publisher=网易，来源：大江网|language=Zh-hans }}</ref>江西省图书馆馆长周建文称在2015年启动修缮工作，按原貌进行修复<ref>{{Cite web |url=http://jx.people.com.cn/n/2014/1227/c190260-23364785.html|title=蒋介石南昌行营旧址修复 将变身典籍博物馆|accessdate=2017-07-12 |author= |date=2014-12-27 |work= |publisher=人民网-江西频道 |language=Zh-hans }}</ref>，完成后在此设立江西省典籍博物馆，但2015年12月的报道称<ref name="a2"/>，修缮并未进行。

2017年1月<ref>{{Cite web |url=http://nc.jxnews.com.cn/system/2017/03/15/015875841.shtml|title=江西省立图书馆旧址保护维修工程启动|accessdate=2017-07-12 |author= |date=2017-01-04  |work= |publisher=网易，来源：南昌新闻网(南昌)|language=Zh-hans }}</ref>和3月<ref>{{Cite web |url=http://nc.jxnews.com.cn/system/2017/03/15/015875841.shtml|title=蒋介石南昌行营开始修缮|accessdate=2017-07-12 |author=作者：王祺，编辑：朱玉琴 |date=2017-03-15 |work= |publisher=中国江西网 |language=Zh-hans }}</ref>的报道称修缮工作已经开始，相关报道图片显示，建筑的上半部正在被拆除。是否是“按原貌修复”之故，不详。
