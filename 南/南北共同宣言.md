'''《南北共同宣言》'''又称'''《北南共同宣言》'''、'''《6·15共同宣言》'''是2000年6月15日朝韩双方经过多轮外交谈判后达成的联合宣言<ref>[http://news.eastday.com/epublish/gb/paper58/1/class005800006/hwz68470.htm 韩国舆论欢迎签署南北共同宣言]，东方新闻</ref>。《南北共同宣言》是朝韩分治后，双方领导人[[2000年朝韓首腦會晤|首次会晤]]后签署的历史性文件，是[[朝韩关系|朝韩关系]]发展中的里程碑。在《宣言》签订5周年之际，朝韩双方在[[平壤|平壤]]召开了“民族统一大会”。大会通过了《民族统一宣言》并将6月15日定为“民族之日”。<ref>[http://news.163.com/05/0616/11/1MC7VAUS0001121Q.html 朝鲜韩国发表民族统一宣言　共同呼吁拒绝核战]，网易</ref>

2007年10月2日至4日，[[卢武铉|卢武铉]]政府与北韩召开了第二次[[朝韩首脑会晤|首脑会晤]]并签订了《[[北南关系发展与和平繁荣宣言|北南关系发展与和平繁荣宣言]]》。《南北共同宣言》的宗旨得到延续。2008年2月，[[李明博|李明博]]出任[[韩国总统|韩国总统]]后，对《南北共同宣言》持否定态度<ref>[http://www.beijingreview.com.cn/2009news/guoji/guancha/2010-06/24/content_281095.htm 朝韩关系：寻求和解是正途]，北京周报</ref>，调整了前两届政府对北韩的[[阳光政策|阳光政策]]，双方关系开始由和解转为对峙<ref>[http://www.zjol.com.cn/05world/system/2008/11/25/015009373.shtml 朝宣布中断朝韩交流项目]，国际新闻</ref> 。

== 背景 ==
《南北共同宣言》是在[[苏联|苏联]]、[[东欧|东欧]]社会主义国家解体，中国[[改革开放|改革开放]]，冷战体制彻底崩溃的历史大背景下签订的。 

[[金大中|金大中]]在在野党时期就积极主张朝鲜半岛的和平统一，在其就职演说时就表明一有机会就举行[[朝韩首脑会晤|朝韩首脑会晤]]，并在外交场合强调开展南北会晤。[[金大中|金大中]]的提议得到了[[金正日|金正日]]的积极响应。朝鲜半岛的和解气氛最终导致第一次[[朝韩首脑会晤|朝韩首脑会晤]]的召开和《南北共同宣言》的签订。<ref>[http://www.hanguo.net.cn/news/xwzq/xy1.htm 南北韩共同宣言的产生背景]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}，韩国在线</ref>

== 影响 ==
在《南北共同宣言》签署后的数年中，双方交流频繁，朝韩关系发生了历史性的变化。[[卢武铉|卢武铉]]总统称“《6·15共同宣言》是改变朝鲜半岛命运的历史转折点。”<ref>[http://www.china.com.cn/chinese/HIAW/588605.htm 朝韩关系发生历史性变化]，中国网</ref> 朝鲜报纸称《宣言》是“民族共同的统一纲领”<ref>[http://chinese.yonhapnews.co.kr/allheadlines/2012/06/15/0200000000ACK20120615002600881.HTML 朝鲜称《6.15共同宣言》是民族共同的统一纲领]，韩联网</ref>。

朝韩首次会晤和《南北共同宣言》的签署使时任[[韩国总统|韩国总统]][[金大中|金大中]]获得[[诺贝尔和平奖|诺贝尔和平奖]]。

== 發展 ==
2008年2月，[[大國家黨|大國家黨]]的[[李明博|李明博]]就任[[韓國總統|韓國總統]]，開始否定[[陽光政策|陽光政策]]。

2008年7月11日，韓國53歲女遊客朴旺子在區內旅遊時，據稱因誤入相距不遠的朝方軍事區而被朝方哨兵射殺，韓國政府隨即宣布暫停韓國遊客前往[[金剛山觀光地區|金剛山觀光]]。

2010年3月26日，韓國軍艦[[天安號|天安號]]在[[白翎島|白翎島]]附近海域沉沒，即[[天安號沉沒事件|天安號沉沒事件]]。40多名韓國官兵被殺。

2010年11月23日，發生[[延坪島炮擊事件|延坪島炮擊事件]]。

2011年12月17日，[[金正日|金正日]]去世，[[金正恩|金正恩]]接任朝鮮最高領導人。

2013年2月25日，[[朴槿惠|朴槿惠]]正式宣誓就任第18任[[韓國總統|韓國總統]]，李明博卸任總統職務。

李明博時期因[[朝鮮核問題|朝鮮核問題]]，金剛山槍擊事件，天安號沉沒事件以及延坪島炮擊事件的發生，朝韓關係一直緊張。朴槿惠接任後，和李明博一樣，堅定要求朝鮮放棄發展核武，才會展開對話。
