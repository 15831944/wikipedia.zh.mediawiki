'''南京广播电视台'''（{{lang-en|'''N'''anjing '''B'''roadcasting '''S'''ystem，'''NBS'''}}，前称'''南京广播电视集团'''）是中国[[国家广电总局|国家广电总局]]批准的第一家全国副省级城市广电集团，于2003年9月29日正式成立<ref>{{Cite web |url=http://www.chinanjbc.com/nj_broadcasting_group/corpmemo.jsp?col_id=028&col_pa_id=028 |title=存档副本 |access-date=2013-02-24 |archive-url=https://web.archive.org/web/20130616084234/http://www.chinanjbc.com/nj_broadcasting_group/corpmemo.jsp?col_id=028&col_pa_id=028 |archive-date=2013-06-16 |dead-url=yes }}</ref>，由南京电视台、南京人民广播电台、南京广播电视报社和[[南京音像出版社|南京音像出版社]]等单位组建而成。2013年1月18日，改为现名并启用新台标。

== 频道列表 ==
=== 电视服务 ===

=== 广播服务 ===
南京广播电视台的广播服务源于1979年1月1日开播的'''南京人民广播电台'''，由南京市广播局实行[[一个机构两块牌子|一个机构两块牌子]]体制；后改为[[中共南京市委|中共南京市委]]直属单位，现有6个广播频率。<ref>{{cite news|title=《江苏省志·广播电视志》：第一章 中波、短波无线广播电台→第六节  地、市级广播电台|url=http://www.jssdfz.com/book/gbdsz/D1/D6J.html |author=《江苏省志·广播电视志》编纂委员会|publisher=江苏省地方志办公室 |date=2000年12月 }}</ref>

== 争议 ==
2013年1月18日12点起，南京广播电视台的新台标正式启用。根据官方解释，新台标来源于[[南京|南京]]“现代化国际性人文绿都”的城市定位<ref>[http://news.sina.com.cn/m/2013-01-18/101526066811.shtml 南京广播电视台启用新台标]</ref>。然而这引发了网友的质疑：有人指其像“[[妇炎洁|妇炎洁]]”的商标，并戏称：“妇炎洁到底是给了南京电视台多少广告费？”<ref>[http://news.house365.com/gbk/czestate/system/2013/01/21/021339995_03.html 南京电视台新台标启用 网友吐槽像两面针和妇炎洁标志]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>
