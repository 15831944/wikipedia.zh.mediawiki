'''南汉二陵'''位于[[中国|中国]][[广州市|广州市]][[番禺区|番禺区]]的[[小谷围|小谷围]]岛上，[[五代十国|五代十国]]时期[[南汉|南汉]]国两位[[国君|国君]]的[[陵墓|陵墓]]。

== 康陵 ==
{{main|康陵 (南漢)}}
康陵位于北亭村大香山南坡，南汉高祖[[刘龑|刘龑]]的陵墓，葬于942年。其依山坡地势南北向兴建，地面有陵园和陵台，地下筑玄宫。陵体呈圆丘形，四周绕以长方形围墙，大约东西宽57.3米，南北长96.8米。墓中出土的一块完整的[[哀册|哀册]]文石碑，为[[中国大陆|国内]]考古发掘中首次出土的完整的哀册全文，也是目前年代最早的一块哀册文石碑。

== 德陵 ==
{{main|德陵 (南汉)}}
德陵北亭村青岗北坡、大香山南坡，被考古工作者认定为南汉烈宗[[刘隐|刘隐]]的陵墓<ref>[http://luxury.qq.com/a/20070822/000619_3.htm 远古的沉睡 探帝王神秘陵寝]</ref>。
南距康陵800米，由[[墓道|墓道]]、封门、前室、后室四部分组成，坐南朝北。墓室已被盗，[[文革|文革]]时期当作[[防空洞|防空洞]]，但墓道中出土的青瓷罐和釉陶罐，为广州首次发现大量的五代[[瓷器|瓷器]]。

== 发现 ==
2003年2月在[[番禺区|番禺区]][[新造镇|新造镇]][[小谷围|小谷围]]岛上兴建[[广州大学城|广州大学城]]，随即对岛上[[文物|文物]]进行全面调查，期间发现二陵。入选[[全国十大考古新发现|2004中国考古十大新发现]]名列第八位，2006年5月25日被公布为第六批[[全国重点文物保护单位|全国重点文物保护单位]]。
