'''南平北站'''位于中国[[福建省|福建省]][[南平市|南平市]][[延平区|延平区]][[夏道镇|夏道镇]]安济村，距南平城区25公里，是[[合福客运专线|合福客运专线]]和建设中的[[南三龙铁路|南三龙铁路]]所共有的火车站，于2015年6月28日建成投入使用。<ref>[http://np.fjsen.com/2014-10/22/content_15080245.htm 合福高铁南平北站预计明年初竣工验收]</ref>

== 车站结构 ==
===站房===
{{multiple image|align=right|direction=vertical|width=220
|image1=20170716 Nanpingbei Railway Station 2.jpg|caption1=车站站房
}}
南平北站设于南平市延平区，车站站房线侧平式站房，车站的外观似在水面展翅而飞的白鹭。站房共设两层，两个检票口与两个候车区，设计高峰时每小时发送量为2300人次，最高聚集人数为2000人。一号检票口位于一层，设有4个闸机，可进入站侧站台，但由于该站台尚未完工，该检票口与配套候车区均不开放。二号检票口位于二层，设有4个闸机，可通过天桥进入1、2、3、4号站台。车站售票厅设于车站东侧，出站口设有两个，位于地面广场与车站地下。<ref>{{Cite book|url=http://www.docin.com/p-268088373.html|title=南平北站实施方案0801|last=|first=|publisher=中建（北京）国际设计顾问有限公司|year=2011|isbn=|location=|pages=|chapter=}}</ref><ref name=":0">{{Cite web|url=http://fjnews.fjsen.com/2015-04/24/content_15998931_all.htm|title=探秘福建合福高铁上的新车站 - 产经财经 - 东南网|accessdate=2017-07-17|work=fjnews.fjsen.com}}</ref>

南平北站采用“上进下出”的进出站模式，乘客可由地面广场进入车站，并由车站天桥进入对应站台。由于地面出站口不开放，出站的乘客只可由地下出站。为避免黑车司机闯入出站口拉客，出站口设有两道闸机，第一道闸机为检票闸机，在出站通道交叉口后各方向均设有第二道闸机，乘客应提前决定出站的方向。<ref>{{Cite web|url=http://np.fjsen.com/2017-05/02/content_19462253.htm|title=南平高铁北站地下通道明日启用 市民出行更加便捷 - 南平新闻 - 东南网|accessdate=2017-07-17|work=np.fjsen.com}}</ref>

=== 站台 ===

南平北站共设有2条正线，4条到发线。站台设有3个站台，5个台面，其中与站房连接的侧式站台未启用，其余两个岛式站台的台面由南往北以1、2、3、4编号命名。进站的旅客通过天桥进入站台，出站的旅客通过地下通道出站。南平北站目前已接入[[合福客运专线|合福客运专线]]，未来还再将接入[[南三龙铁路|南三龙铁路]]。<ref>{{Cite web|url=http://fj.qq.com/a/20131211/004587.htm|title=南三龙铁路全线开工 南平到龙岩仅1.5小时_大闽网_腾讯网|accessdate=2017-07-17|work=fj.qq.com|language=zh-CN}}</ref><ref name=":0" />

== 接驳交通 ==
{{multiple image|align=right|direction=vertical|width=220
|image1=20170716 Anji Coach Station.jpg|caption1=安济客运综合枢纽
}}
由南平市区进站的公交会停靠在站前广场道路上，乘客可在该处下车直接进站。

南平北站西侧设有安济客运综合枢纽，该枢纽站地上一层设有公交枢纽、客运汽车站地下一层设有出租车候客区，乘客可由南平北站2号西侧出站口通过地下通道实现与公共交通的无缝换乘。<ref>{{Cite web|url=http://news.sina.com.cn/c/2017-05-03/doc-ifyetxec7454237.shtml|title=南平安济综合客运枢纽正式投入运营|accessdate=2017-07-17|work=news.sina.com.cn}}</ref>

== 逸事 ==
该站虽然名为“南平北站”，但实际上，该站在南平市的南部远郊，比原有的[[峰福线|峰福线]]上的[[南平南站|南平南站]]更偏南，但仍被命名为“北站”。
