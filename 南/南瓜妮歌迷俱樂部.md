'''南瓜妮歌迷俱樂部'''（Pumpkinney Fan Club）是台灣[[樂團|搖滾樂團]]，由主唱柯家洋、吉他手何俊葦（小青蛙）、貝斯手陳弘禮（弘禮）、鼓手林一根（一根）組成。近年台灣獨立音樂圈知名地下樂團之一。作品：《微光》、《果陀》、專輯《他我》，以英式、迷幻、電子等不同曲風征戰各大校園與國際舞台。

== 成軍 ==
南瓜妮歌迷俱樂部成立於2009年夏末，組團後隨即在全台大專院校最具影響力的[[金旋獎|金旋獎]]創作音樂比賽中，共拿下樂團創作組第二名、樂團創作組最佳人氣獎、最佳吉他手及最佳貝斯手等四項大獎，受到音樂圈內的高度關注。

2012年，台灣最大的音樂創作發表平台StreetVoice街聲網站宣告「南⽠妮歌迷俱樂部」為年度十大新團，其後活躍于各大音樂節演出，2014年更獨力完成10所校園巡迴，其後數年間屢次受邀參加臺北、上海的Simple Life [[簡單生活節|簡單生活節]]<ref>{{Cite news|url=http://ent.qq.com/a/20150918/063732.htm|title=十一就别去看人海了 跟着简单音乐节一起嗨|author=kkliao|date=2015-09-21|work=腾讯娱乐|accessdate=2015-09-21}}</ref><ref>{{cite web |author1=南瓜妮歌迷俱樂部 |title=南瓜妮歌迷俱樂部簡介 |url=https://www.indievox.com/pumpkinneyfanclub/intro |website=https://www.indievox.com |publisher=南瓜妮歌迷俱樂部 |accessdate=2014}}</ref>。

發行過《微光》、《果陀》、《海王星》三張EP，南⽠妮歌迷俱樂部以迷幻且粗糙的聲響和內斂的表演⾵風格著稱。曲風多變的他們，時而迷幻墜落、時而溫柔內斂，無可救藥的悲傷卻也帶著愛情的悸動浪漫，受到英式、迷幻、電子等不同⾳樂影響的樂團成員，加上主唱柯家洋深厚的嗓音和抑鬱的旋律，藉由歌詞表達屬於他們獨特的世界觀<ref>{{cite web |author1=馬世芳 |title=耳朵借我：專訪南瓜妮歌迷俱樂部 |url=https://medium.com/%E6%88%91%E6%98%AF%E9%A6%AC%E4%B8%96%E8%8A%B3/pumpkinney-fan-club-b8f3afd137d |website=https://medium.com |publisher=馬世芳 |accessdate=07/23/2018}}</ref>。
