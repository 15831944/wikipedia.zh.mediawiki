'''南部商务区站'''是[[浙江省|浙江省]][[宁波市|宁波市]]规划中的一座地下轨道交通车站。南部商务区站是[[宁波轨道交通3号线|宁波轨道交通3号线]]和[[宁波轨道交通8号线|8号线]]的换乘站，3号线车站计划于2019年启用<ref name="main">{{cite web|url=http://daily.cnnb.com.cn/nbrb/images/2014-12/24/A2/nbrb20141224a02.pdf|title=宁波地铁3号线一期开工|author=冯瑄|publisher=宁波日报|date=2014-12-24|accessdate=2014-12-26|archive-url=https://web.archive.org/web/20141226155000/http://daily.cnnb.com.cn/nbrb/images/2014-12/24/A2/nbrb20141224a02.pdf|archive-date=2014-12-26|dead-url=no}}</ref>，8号线车站则为预留建设条件车站<ref>{{Cite web |url=http://nb8185.cnnb.com.cn/tiezi/minsheng_view.php?aid=910613 |title=8号线南部商务区站是同步建设的还是预留建设条件 |accessdate=2018-09-03 |author=宁波轨道交通 |date=2018-09-03 |publisher=宁波市网络民生服务平台|archive-url=https://web.archive.org/web/20180903093307/http://nb8185.cnnb.com.cn/tiezi/minsheng_view.php?aid=910613|archive-date=2018-09-03|dead-url=no}}</ref>。

== 位置 ==
南部商务区站位于[[鄞州区|鄞州区]]天童南路和泰康中路路口，沿天童南路南北向设置<ref name="main" />。

== 车站形制 ==
南部商务区站地下一层为站厅，二层为[[岛式车站|岛式车站]]。车站位于地下11米，长484.6米，宽19.7米，总建筑面积22502平方米<ref>市发展改革委关于宁波市轨道交通3号线一期工程初步设计的复函, 甬发改审批〔2014〕653号</ref>，初步设计设4个出口<ref name="main" />。
