'''南部中心'''（{{lang-en|'''South Central'''}}），也稱'''迦密山'''（{{lang|en|'''Mount Carmel'''}}），[[美國|美國]][[田納西州|田納西州]][[華盛頓縣_(田納西州)|華盛頓縣]]西南部一個[[非建制地區|非建制地區]]，其海拔約為445米（1460英尺）。<ref name="GNIS">{{Cite GNIS | 1315544 | 田納西州南部中心}}{{en}}</ref>當地位於圖斯庫盧姆（{{lang|en|Tusculum}}）市東南方和瓊斯堡鎮南部（{{lang|en|Jonesborough}}），並鄰近107號田納西州州道（{{lang|en|Tennessee State Route 107}}）以及353號田納西州州道（{{lang|en|Tennessee State Route 353}}）。南部中心本身並沒有任何郵政局與專屬的[[美國郵遞區號|美國郵遞區號]]，因此當地借用附近查基（{{lang|en|Chuckey}}）的37641作為其郵政編碼。<ref>{{cite web | title=Zip Code 37641, Chuckey, TN | url=https://www.zip-codes.com/zip-code/37641/zip-code-37641.asp | language=en | publisher=Zip-Codes.com | accessdate=2018年5月22日}}</ref>
