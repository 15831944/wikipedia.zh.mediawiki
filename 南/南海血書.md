《'''南海血書'''》，是一則[[中華民國|中華民國]]在[[中國國民黨|中國國民黨]][[威權|威權]]時代下的虛構故事。本作於1978年12月19日在《[[中央日報|中央日報]]》副刊發表的文章，作者[[朱桂_(作家)|朱桂]]。最初偽稱是「[[越南難民|越南難民]]阮天仇」的「絕筆信」，這篇故事成為[[臺灣|臺灣]]中小學的教材，政府機關也大量印送。

這篇故事，後來證實是[[中華民國政府|中華民國政府]]的[[反共|反共]]宣傳之一，為當時國民黨情報主管[[王昇|王昇]]上將為配合[[國民黨政府|國民黨政府]]的[[威權統治|威權統治]]所策劃的虛構故事，目的在於鼓動[[臺灣人|臺灣人]]反共情緒，並以[[越南|越南]]的淪亡歸咎於內部的不團結、藉此強調國民黨[[一黨獨大|一黨獨大]]之「必要性」來打壓[[黨外|黨外]]人士、[[異議分子|異議分子]]，以及發洩[[中華民國與美國關係|與美國斷交]]後的不平情緒。

2003年，作者朱桂坦承是一篇虛構的故事，由他本人創作。

== 內容提要 ==
《南海血書》於1978年12月19日在《[[中央日報|中央日報]]》副刊裏刊登，根據《中央日報》表示是本文是「譯者」[[朱桂_(作家)|朱桂]]（實為作者）的[[內弟|內弟]]在出海捕魚時，在[[南中國海|南中國海]]的一個[[荒島|荒島]]上所「發現」，是一個名叫「'''阮天仇'''」的越南[[難民|難民]]的絕筆。譯者朱桂為[[基隆|基隆]][[崇右企專|崇右企專]]的教師。書中表示，[[越南共和國|南越]]被[[越南民主共和國|北越]]所併吞與[[越戰|越戰]]的失敗，是「[[美利堅合眾國|偉大盟邦]]」與「[[民主|民主]]鬥士」的過錯。

文中提到家人是怎樣死於[[越南共產黨|越共]]的殘酷手法：大哥死在[[越戰|越戰]]的炮火中，姪兒在一場暴動中被[[子彈|流彈]]擊中，九十三歲的老祖母與七歲的姪女被越共政府「照顧」到餓死，絕口不提政治的父親，卻在鬥爭大會上被一棒一棒地打死，三哥（有版本作二哥）在[[集中營|集中營]]裡因為飢餓，忍不住吃了一個[[番薯|番薯]]，結果被[[槍斃|槍斃]]，大嫂無故被關，病死在[[監獄|監獄]]中，母親在逃亡時被共產黨幹部推下海淹死，妻子被[[海盜|海盜]]射殺，一家人只剩下阮天仇與兒子文星。船在怒濤上飄流，[[難民|難民]]們漂到一個[[珊瑚礁|珊瑚礁]]島上，但兒子文星在第十三天痛苦地死去，其遺體也被難民分食。吃了文星的難民後來陸續餓死，只剩阮天仇在島上等死，最後阮捱了四十幾天才死亡。

== 影響 ==
由於當時中華民國與[[美國|美國]]斷交，《南海血書》的內容強烈暗示越戰命運可能會降臨台灣，挑起台灣的敏感神經；《南海血書》不但成為當時國中小學生的必讀書籍，公務機關也大量發放，甚至銷售達二十萬冊，最後還拍成[[電影|電影]]。其中一句話：「'''今日不為自由鬥士，明天將為海上難民！'''」，更傳頌一時。

== 真相 ==
但是《南海血書》之內容有諸多矛盾、失實之處，引發台灣省議員[[何春木|何春木]]在總質詢時指責《南海血書》誇大渲染、荒誕離譜；[[林濁水|林濁水]]也撰寫《拙劣的越南預言——剖析「南海血書」的真相­》，除諷刺阮天仇為「神人」外（挨餓四十餘日後，仍有餘力用鮮[[血|血]]撰寫三千多字），這「神人」也不可能用螺尖沾血於襯衫上寫字等不合[[邏輯|邏輯]]處，同時指出「捏造該文的人士是以南越淪亡的例子打擊[[黨外|黨外]]人士」。而在[[臺灣|臺灣]]民主化之後，原著作者朱桂也於2003年承認「阮天仇」為虛構的人物、《南海血書》是虛構的情節。後經證實為當時國民黨情報頭子[[王昇|王昇]]上將為配合國民黨政府的威權統治所策劃的虛構故事，以假亂真，欺騙民眾。

==現況==
*中華民國政府事實上也曾屠殺越南船民，例如[[三七事件|三七事件]]。
*南海血書已經全面從教科書中刪去，官方也不再發行。
