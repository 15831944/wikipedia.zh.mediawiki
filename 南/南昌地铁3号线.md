'''南昌地铁3号线'''（官方文件正式名称：南昌市轨道交通3号线）是南昌一条拟建的轨道交通线路，该线路规划从京东大道到莲塘，将设22站，2015年开始土建预计2020年全线通车<ref>{{cite news|author=谢子玥|title=南昌地铁3号线明年开工2019年建成通车 设有21个车站|url=http://jiangxi.jxnews.com.cn/system/2014/12/20/013508962.shtml|accessdate=2013-08-11|newspaper=江南都市报|date=2014-12-20}}</ref>。

== 简介 ==
3号线代表色为南昌轨道蓝{{color|#{{南昌地铁识别色|3}}|■}}，按照南昌地铁建设规划这将是第一条通车到[[南昌县|南昌县]]实际管辖区域内，并达到县城莲塘的线路。

== 附属设施 ==
=== 车辆段 ===
一期将设两个车辆段，分别是莲塘定修段和高新停车场<ref>{{Cite web |url = http://www.jxnews.com.cn/jxrb/system/2014/04/11/013036297.shtml |title = 地铁3号线征迁摸底即将启动[图] |author =  |date = 2014-04-11 |publisher = 中国江西网-江西日报 |language = |accessdate = |archiveurl= |archivedate=}}</ref>。

=== 控制中心 ===
控制中心设一个在[[地铁大厦站|地铁大厦站]]附近<ref>[http://www.ncmtr.com/newsshow.php?id=2169 南昌市轨道交通3号线工程系统设计02标段（通信、信号系统设计）招标公告]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
=== 主变电所 ===
在[[青山路口站|青山路口站]]设35KV开关站并共享2号线的主变电站，在[[阳光路站|阳光路站]]近设置1座主变电站<ref>[http://www.ncmtr.com/newsshow.php?id=2170 南昌市轨道交通3号线工程系统设计03标段（主变电所设计）招标公告]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
=== 人防系统 ===
地下区间段的隧道和车站将具备人防系统<ref>[http://www.ncmtr.com/newsshow.php?id=2171 南昌市轨道交通3号线工程系统设计04标段（人防系统设计）招标公告]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 未来发展 ==
2008年在最初的规划中3号线计划东起瑶湖北，南至莲塘，包括远期将长达31km<ref name=":0" />；当时该线远景规划建设一条支线从英雄大桥东侧过江到扬子洲后二次过江到蒋巷镇，不过后续的规划所有线路的远景支线都不在出现。
* 东延：根据2012年底的山江湖综合开发战略确认3号线将以穿越瑶湖的方式延伸到瑶湖东侧<ref>{{Cite web|url=http://ghj.nc.gov.cn/qtgh/2935.jhtml|title=山江湖综合开发战略|accessdate=|author=南昌市城乡规划局|date=2015-08-17|publisher=南昌市城乡规划局}}</ref>；2018年9月30日公示的《艾溪湖隧道工程交通详细规划》确认车辆隧道下方有规划公共廊道<ref>{{Cite web|url=http://ghj.nc.gov.cn/qtpqgg/18575.jhtml|title=艾溪湖隧道工程交通详细规划|accessdate=2018-11-13|author=|date=|work=ghj.nc.gov.cn|publisher=}}</ref>，随后11月4日公示该项目由南昌轨道交通集团有限公司来负责建设<ref>{{Cite web|url=http://www.ncmtr.com/index.php?s=/Home/management/detail/id/3067.html|title=南昌市艾溪湖隧道工程项目社会稳定风险公示|accessdate=2018-11-13|author=|date=|work=www.ncmtr.com|publisher=|archiveurl=https://web.archive.org/web/20181113042828/http://www.ncmtr.com/index.php?s=%2FHome%2Fmanagement%2Fdetail%2Fid%2F3067.html|archivedate=2018-11-13|dead-url=no}}</ref>。
