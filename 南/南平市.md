'''南平市'''（{{sans-serif|{{lang|mnp|Nâng-běng-chī}}}}）是[[中华人民共和国|中华人民共和国]][[福建省|福建省]]下辖的[[地级市|地级市]]，位于福建中部偏北，地处[[闽江|闽江]]上游，別称'''[[闽北|閩北]]'''。全市总面积2.63万平方公里，人口264万。

南平市南连[[三明市|三明]]，东连[[宁德市|宁德]]，西部和北部分别与[[江西省|江西省]]和[[浙江省|浙江省]]接壤。“南平”一名来源于昔日的[[南平縣|南平县]]，即今日南平市的[[延平区|延平区]]。南平市政府驻地在[[建阳区|建阳区]]。

== 历史 ==
=== 古代 ===
==== 汉及以前 ====
早在4000多年前的[[新石器时代|新石器时代]]，就有土著人在此繁衍生息。[[夏朝|夏]][[商朝|商]]时期，天下分九州。南平属扬州。[[周朝|周]]时，属七闽地。[[周顯王|周显王]]三十五年（前334年），建立[[闽越国|闽越国]]，南平属之。秦始皇二十六年（前221年），南平属闽中郡。[[西汉|西汉]][[刘邦|高祖]]五年（前202年），南平属闽越国。汉[[建元_(西汉)|建元]]六年（前135年），[[馀善|馀善]]封为东越王，在闽北割踞，闽越改称[[东越|东越]]。[[余善|馀善]]分别在今[[浦城县|浦城县]]、[[武夷山市|武夷山市]]、[[建阳区|建阳市]]、[[邵武市|邵武市]]建成六座城堡拒汉。汉[[元封_(西汉)|元封]]元年（前110年），东越被汉王朝所灭，立冶县，南平属之。

南平建置区划始于[[东汉|东汉]][[建安_(东汉)|建安]]初年（约196年），福建境内设置5县，南平境内有南平、建安（今建瓯）、汉兴（今[[浦城县|浦城]]）3县，属会稽南部都尉，是最早的县级行政区划。汉建安八年（203年）移南部都尉于建安。建安十年（205年），析建安桐乡及今江西[[上饶市|上饶]]一部分地置建平县（今建阳）。

==== 三国 ====
吴永安三年（260年），以会稽南部地置建安郡，郡治建安（今建瓯）等10县（即今福建全境）。建安郡成为福建省最早的地区级建制。
==== 魏晋南北朝 ====
实行州、郡、县三级地方政区制。[[晋朝|晋]]太康三年（282年），建安郡属[[扬州|扬州]]。元康元年（291年），属江州（今江西[[九江市|九江]]）。南朝梁普通六年（525年），属[[东扬州|东扬州]]。陈永定初年（约557年），改属[[闽州|闽州]]（今[[福州市|福州]]）。光大二年（568年）属[[丰州|丰州]]（今江西[[抚州|抚州]]）。
==== 隋唐五代 ====
[[隋朝|隋]][[开皇|开皇]]九年（589年）废郡为县，将原州、郡、县三级政区制改为州、县二级政区制，废郡后的建安属泉州（今[[福州|福州]]）。大业初（约605年），属闽州（治福州）。[[大业_(年号)|大业]]三年（607年），闽州又改名建安郡。

[[唐朝|唐]]王朝实行道、州、县制度。[[武德|武德]]四年（621年），州治从福州移至建安（今建瓯），并改郡为建州，属泉州都督府。景云二年（711年），属闽州都督府。天宝元年（742年），改建州为建安郡，属江南东道采访使。乾元元年（758年），又改为建州，属福州都督府。肃宗上元元年（760年），属福州节度使。大历六年（771年），属福州都团练观察处置使。

[[五代十国|五代]][[闽_(十国)|闽]][[龙启|龙启]]二年（934年），[[王审知|王审知]]在福建建立闽国，建州属之。后晋天福八年（943年），王审知之子[[王延政|王延政]]在建州建国，国号“[[殷|殷]]”。[[南唐|南唐]][[保大_(南唐)|保大]]四年（946年），殷亡，改建州为[[永安军|永安军]]，不久又改为[[忠义军|忠义军]]。同年，析建州地置剑州，治剑浦（今南平）。时境内二州并立。
==== 宋元 ====
[[宋朝|宋]]代，实行路、府（州、军）、县三级政区制。[[开宝|开宝]]八年（975年）改忠义军为[[建州_(唐朝)|建州]]。[[太平兴国|太平兴国]]三年（978年），[[建州_(唐朝)|建州]]、[[剑州_(五代)|剑州]]同属[[两浙路|两浙路]]。太平兴国四年（979年），因四川亦有剑州，改南平剑州为南剑州。太平兴国五年（980年）增置邵武军，治邵武，境内2州1军并立。雍熙二年（985年），建州、南剑州、邵武军改属[[福建路|福建路]]。[[端拱|端拱]]元年（988年），升建州为建宁军节度，绍兴三十二年（1162年）再升为建宁府。

[[元朝|元]]代，[[至元_(忽必烈)|至元]]十三年（1276年）改邵武军为[[邵武路|邵武路]]。至元十五年（1278年），改建宁府为[[建宁路|建宁路]]，南剑州为[[南剑路|南剑路]]。大德六年（1302年），改南剑路为[[延平路|延平路]]，闽北3路同属[[福建|福建]][[行中书省|行中书省]]。

==== 明清 ====
[[明朝|明]]代，[[洪武|洪武]]元年（1368年），改建宁路为[[建寧府|建宁府]]，邵武路为[[邵武府|邵武府]]。洪武二年（1369年），改延平路为[[延平府|延平府]]。3府同属[[福建布政使司|福建布政使司]]。

[[清朝|清]]代，建制沿袭明制，境内设3府。<ref>{{cite web |url=http://www.np.gov.cn/npgl/npgk/jzyg/4980874.shtml |language=zh |title=清及其以前南平市建置 |publisher=南平市人民政府门户网站 |date=2013-05-14 |accessdate=2013-07-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20140405100525/http://www.np.gov.cn/npgl/npgk/jzyg/4980874.shtml |archivedate=2014-04-05 }}</ref>
=== 民国 ===
[[中華民國|民国]]元年（1912年），废府（州、厅）改道，实行省、道、县三级政区制。福建省设东、西、南、北4道，闽北为北路道，道所在地南平（辖16县）。民国3年6月，改为[[建安道|建安道]]。民国16年废道，实行省、县二级政区制，闽北各县直属福建省。

民国22年11月“[[福建事变|福建事变]]”，成立[[中华共和国人民革命政府|中华共和国人民革命政府]]，即福建人民政府，在闽北设延建省。民国23年1月，福建人民政府失败，2月成立福建省政府。

民国23年7月由于福建省属“剿匪”省份，为便于督察，实行行政督察专员制度，设置行政督察专员公署。全省划为10个行政督察区，闽北境内设第三（驻[[南平縣|南平县]]）、第九（驻[[邵武市|邵武县]]）、第十（驻[[浦城县|浦城县]]）3个行政督察区。

民国24年10月，福建省改划为1市7个行政督察区，闽北境内设第二（驻南平）、第三（驻浦城）2个行政督察区。民国27年8月，第三行政督察区专署由浦城移驻[[建阳县|建阳县]]。此后，两区建制沿续至民国38年5月南平地区大部分县解放。

=== 中华人民共和国 ===
民国38年5月，闽北大部分县解放，9月境内设两个行政督察专员公署，为福建省人民政府的派出机构，即第一行政督察专员公署，驻建瓯县；第二行政督察专员公署，驻南平县。

1950年3月，为便于与民国建制的识别，省人民政府通知行政督察专员公署改冠以驻县名。4月1日，第一行政督察专员公署改名建瓯行政督察专员公署，第二行政督察专员公署改名南平行政督察专员公署。9月27日，建瓯行政督察专员公署驻地迁至建阳，改称建阳行政督察专员公署。11月1日，建阳、南平两公署分别改称福建省建阳区专员公署、福建省南平区专员公署。1955年3月，又分别改称建阳专员公署、南平专员公署。1956年3月26日，撤销建阳专员公署，所属各县划归南平专员公署。

“[[文化大革命|文化大革命]]”中，1967年2月19日成立南平地区革命造反委员会。3月18日。南平地区革命造反委员会改称南平地区革命委员会。1968年5月15日，根据福建省军区的指示，改称为南平专区革命委员会。
1970年9月，[[南平专区|南平专区]]驻地迁驻建阳。1971年7月1日，正式挂牌称[[建阳地区|建阳地区]]革命委员会。1978年3月，撤销建阳地区革命委员会，成立福建省建阳地区行政公署。

1983年10月，邵武撤县建市。1988年10月24日，经国务院批准建阳地区驻地迁回南平，1989年1月1日，正式更名为南平地区行政公署。1989年8月，崇安撤县设市更名为武夷山市。1992年12月，建瓯撤县设市。1994年3月，建阳撤县设市。1994年9月，国务院批准撤销南平地区设立地级南平市，原县级南平市同时改设为延平区，南平地区建制至1995年1月18日终止。<ref>{{cite web |url=http://www.np.gov.cn/npgl/npgk/jzyg/4980876.shtml |language=zh |title=中华人民共和国成立后南平市建置 |publisher=南平市人民政府门户网站 |date=2013-05-14 |accessdate=2013-07-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20140405095847/http://www.np.gov.cn/npgl/npgk/jzyg/4980876.shtml |archivedate=2014-04-05 }}</ref>2014年5月，国务院批复同意撤销建阳市、设立南平市建阳区的行政区划调整方案，南平市政府由[[延平区|延平区]]迁至[[建阳区|建阳区]]。<ref>{{cite web|url=http://www.chinanews.com/df/2014/05-27/6218648.shtml|language=zh |title=福建省南平市行政区搬迁获国务院批准|publisher=中国新闻网| date=2014-05-27|accessdate=2014-05-27}}</ref><ref>[http://www.fj.xinhuanet.com/news/2014-05/27/c_1110885201.htm 国务院批复同意南平市行政区划调整方案] {{webarchive|url=https://web.archive.org/web/20150924013639/http://www.fj.xinhuanet.com/news/2014-05/27/c_1110885201.htm |date=2015-09-24 }}</ref><ref>[http://www.np.gov.cn/npyw/5019708.shtml 南平市政府迁址建阳获国务院批复] {{webarchive|url=https://web.archive.org/web/20140528095420/http://www.np.gov.cn/npyw/5019708.shtml |date=2014-05-28 }}</ref><ref>[http://fjnews.fjsen.com/2014-05/28/content_14163823.htm 南平市政府第三次“搬家” 将迁至新设建阳区]</ref>2015年3月18日，建阳区政府正式挂牌成立。<ref name="建阳撤市设区2015">{{cite web|url=http://fjrb.fjsen.com/fjrb/html/2015-03/19/content_818205.htm?div=-1|title=建阳撤市设区正式挂牌|accessdate=2015-03-18|author=|publisher=福建日报|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20150323000200/http://fjrb.fjsen.com/fjrb/html/2015-03/19/content_818205.htm?div=-1|archivedate=2015-03-23}}</ref>

== 地理 ==
=== 位置 ===
地处福建北部，故称闽北。介于东经117°00′至119°25′，北纬26°30′至28°20′之间。南靠闽江，北枕[[武夷山|武夷]]。与[[宁德市|宁德市]]、[[三明市|三明市]]、浙江省[[江山市|江山]]、[[龙泉市|龙泉]]、[[庆元县|庆元]]等县（市）相邻，江西省[[资溪县|资溪]]、[[铅山县|铅山]]、[[广丰区|广丰]]等县（市）接壤。是福建省面积最大的一个设区市政区域。

=== 地形 ===
境内地形地貌受构造运动的影响强烈、构造地貌特征相当明显，山脉多呈东北--西南走向。[[低山|低山]][[丘陵|丘陵]]分布广，河谷地形呈峡谷或“V”、“U”状镶嵌该市各地，山间盆谷地沿河交替分布，山地切割明显，高差悬殊，以断裂为主的断块山，山峰陡峭，断层崖、断裂谷等断层地貌分布广。中山集中分布于西北、东北和西南部，由[[武夷山|武夷山]]、杉岭、[[仙霞岭|仙霞岭]]、[[鷲峰山|鹫峰山]]4大山脉构成地形走势。中部和南部以低山丘陵盆地为主。土地总面积2.63万平方公里，其中海拔1000米以上的土地占总面积12.09%，海拔500-1000米的占44.03%，海拔300-500米的占21.38%，海拔300米以下的占22.50%。<ref>{{cite web |url=http://www.np.gov.cn/npgl/npgk/zrdl/4980869.shtml |language=zh |title=南平市地形地貌 |publisher=南平市人民政府门户网站 |date=2013-05-14 |accessdate=2013-07-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20140405095424/http://www.np.gov.cn/npgl/npgk/zrdl/4980869.shtml |archivedate=2014-04-05 }}</ref>

=== 物产 ===
野生植物有[[杉|杉]]、[[松|松]]、毛竹、黄杨、楠、樟、花梨等。
南平县是中国重点竹区，有竹林422万亩，占全国竹区总面积的10.5％。南平是1999年中国林业部命名为“竹子之乡”的10个县市之一。1995年竹业出口创汇1.03亿元。
矿产资源有铜、铁、钨、石英砂、石灰石、大理石、重晶石、瓷土、铌、钽等。
森林资源丰富，造纸业发达。

== 經濟 ==
南平市經濟在[[福建|福建]]處於中下游水平，碍于地理条件限制，经济难以赶超沿海地区。目前主要经济支柱为[[化学工业|化学工业]]及[[旅游业|旅游业]]，[[南孚电池|南孚电池]]即为南平地方品牌。南平具有丰富旅游资源，第三产业有较大发展空间。

==文化==
南平市的南平[[南词|南词]]、武夷山市的[[武夷岩茶|武夷岩茶]]制作技艺、邵武市的邵武[[傩舞|傩舞]]、建瓯市的建瓯桃幡及政和县的[[四平戏|四平戏]]均被列入[[国家级非物质文化遗产名录|国家级非物质文化遗产名录]]。

===方言===
南平市境内主要通行[[閩北語|閩北語]]，西部為帶有閩北語和[[贛語|贛語]]過渡性質的[[閩贛方言|閩贛方言]]。[[浦城县|浦城县]]北部地区讲[[吴语|吴语]][[处衢片|处衢片]]方言。[[延平区|延平区]]和[[西芹镇|西芹镇]]讲[[南平话|南平话]]，当地人称之为“土官话”，是[[官话|官话]]在[[闽语|闽语]]地区的一处[[方言岛|方言岛]]。

=== 航空 ===
[[File:Xiamen_Airlines_B737_new_paint_at_ZSAM.jpg|'''连接武夷山市与厦门市的[[厦门航空公司]][[波音737|波音737]][[客机|客机]]'''|300px|right|thumb]]
[[武夷山机场|武夷山机场]]是南平地区唯一的民用机场。武夷山市的山区地形，造成地面交通耗时较长，武夷山机场则成为福建北部地区极其重要的外联快速交通基地。扩建后的武夷山机场，已连接中国特大城市[[北京|北京]]、[[广州|广州]]、[[深圳|深圳]]以及[[香港|香港]]，年旅客流量达70万人次<ref>{{cite web|url=http://arch.fjsen.com/a/2009-09/03/content_1251583_2.htm|title=《闽北交通发展历程：小木船到"铁公机"并举》|accessdate=2010-04-22|publisher=福建日报东南网|language=zh-hans}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。此外，南平还使用邻近的[[福州长乐国际机场|福州长乐国际机场]]，市区有开通机场巴士通达。

武夷山机场航点：厦门（[[厦门航空公司|厦门航空公司]]、[[中国南方航空公司|中国南方航空公司]]）、深圳（[[深圳航空公司|深圳航空公司]]）、上海（[[中国东方航空公司|中国东方航空公司]]、厦门航空公司）、北京（厦门航空公司）、广州（厦门航空公司）、[[郑州|郑州]]（中国南方航空公司）。
