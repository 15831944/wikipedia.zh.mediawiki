'''南挪威'''（[[書面挪威語|書面挪威語]]、[[新挪威語|新挪威語]]：'''Sørlandet'''），[[挪威|挪威]]地理上的南部，[[斯卡格拉克海峽|斯卡格拉克海峽]]沿岸的兩個[[挪威行政區劃#郡|郡]]，包括[[東阿格德爾|東阿格德爾]]（{{lang-no|Aust-Agder}}）與[[西阿格德爾|西阿格德爾]]（{{lang-no|Vest-Agder}}），總面積達{{convert|16433|sqkm|sqmi|0}}<ref>{{cite web |url=http://www.statkart.no/?module=Files;action=File.getFile;ID=24831 |work=Statens kartverk |title=Arealstatistikk for 2006 |date=2006-01-01 |language=no |accessdate=2008-08-28 |archive-url=https://www.webcitation.org/68TtuyvAK?url=http://www.statkart.no/filestore/Profil/Areal_milj_og_plan_gml_dok/Arealstatistikk-2006-01-092.xls |archive-date=2012-06-17 |dead-url=yes }}</ref>，依2008年7月所公告統計結果<ref>{{cite web |url= http://www.ssb.no/folkendrkv/ |work=Statistisk Sentralbyrå |title=Kvartalsvise befolkningsendringer |language=no |accessdate=2008-08-28}}</ref>，人口共有273,818人。

== 起源 ==
[[File:Flekkefjord-Nirwana.jpg|thumb]]
南挪威大致上是歷史上[[阿格德爾|阿格德爾]]（{{lang-no|[[:no:Agder|Agder]]}}）王國的領地，1902年，由當地作家[[威爾海姆·克拉格|威爾海姆·克拉格]]（{{lang|no|[[:no:Vilhelm_Krag|Vilhelm Krag]]}}）提出南挪威這個名稱與觀念，在這之前，這個區域被視為是[[西挪威|西挪威]]的一部份。有時候，挪威人會把鄰近[[羅加蘭|羅加蘭]]與[[泰勒馬克|泰勒馬克]]兩個郡的部份地方，視為南挪威的一部份。

依[[文化地理學|文化地理學]]的觀點，南挪威可被定義為[[斯卡格拉克海峽|斯卡格拉克]]海岸帶，與北方山谷有所區別。

== 海岸地區 ==
[[File:Farsund.jpg|thumb]]
南挪威內八成人口分布在氣候溫和的沿岸地區，接近漁獲豐富的[[漁場|漁場]]，而且是斯卡格拉克海峽[[海上航線|海上航線]]所經之處，著名的城鎮從西到東有：

*[[弗萊克福德|弗萊克福德]]（{{lang|no|Flekkefjord}}）
*[[法爾松|法爾松]]（{{lang|no|Farsund}}）
*[[曼達爾|曼達爾]]（{{lang|no|Mandal}}）
*[[克里斯蒂安桑|克里斯蒂安桑]]（{{lang|no|Kristiansand}}）
*[[利勒桑|利勒桑]]（{{lang|no|Lillesand}}）
*[[格里姆斯塔|格里姆斯塔]]（{{lang|no|Grimstad}}）
*[[阿倫達爾|阿倫達爾]]（{{lang|no|Arendal}}）
*[[特維德斯特蘭|特維德斯特蘭]]（{{lang|no|Tvedestrand}}）
*[[雷塞爾|雷塞爾]]（{{lang|no|Risør}}）
