'''南京城'''是[[明代|明代]]修筑的[[都城|都城]]，延续至今即[[南京市|南京市]][[主城区|主城区]]所在，是[[南京历史|南京历史]]上修筑的等级最高的[[城池|城池]]之一。

南京有[[六朝古都|六朝古都]]之称，明代之前有数个政权定都于此，但至明朝才第一次成为全国性政权的都城。

== 明代 ==
[[元朝|元朝]]末年，[[朱元璋|朱元璋]]建立吴政权，初改[[集庆路|集庆路]]为[[应天府_(明朝)|应天府]]。原[[金陵城|金陵城]]东的[[燕雀湖|燕雀湖]]被填平，以建吴王新宫，即[[明故宫|明故宫]]。有说填湖泥土来自城南的[[三山|三山]]，故有“移三山，填燕雀”的说法<ref>{{cite web |url = http://news.sina.com.cn/o/2015-11-02/doc-ifxkfmhk6791842.shtml |title=李白诗中眺望的“三山” 并非朱元璋挖山填湖的三山 |accessdate=2017-11-13 |author= |date=2015-11-02 |work=新浪网，来源：扬子晚报 |language=zh-Hans }}</ref>。吴元年（1367年），吴王新宫建成。[[洪武|洪武]]元年（1368年）八月，建都，曰南京。洪武二年（1369年）九月，在[[金陵城|金陵城]]的基础上建南京城。洪武六年八月，城池建成。与中国城池的传统正方形或矩形平面布局不同，南京城布局为不规则形状。由四重城墙构建出不同的城池区域，由内而外，依次是[[宫城_(中国)|宫城]]、[[皇城|皇城]]、内城和外城。洪武十一年（1378年），曰京师。[[永乐_(明朝)|永乐]]元年（1403年），仍称南京。后[[明朝政府|明朝政府]][[永乐迁都|迁都]]北京。南京城逐为[[留都|留都]]。而应天府下辖的[[上元县|上元县]]、[[江宁县|江宁县]]亦为[[倚郭县|倚郭县]]。两县依旧沿袭以[[秦淮河|秦淮河]]为界，分南京城，同城而治的管辖状态。

== 清代 ==
[[清朝|清朝]][[顺治|顺治]]初年，[[清军|清军]]攻占南京，圈占[[明故宫|明故宫]]，以南京城东部置[[江宁满城|江宁满城]]。

[[咸丰_(年号)|咸丰]]三年（1853年），[[太平军|太平军]]攻克江宁城，定都于此，改名天京，筑[[天王府|天王府]]。[[同治|同治]]三年（1864年），[[湘军|湘军]]攻克天京，[[屠城|屠城]]并焚毁天王府。

== 民国时期 ==

民国十六年（1927年），北伐军攻占南京。其后，设立[[南京特别市|南京特别市]]，后成为[[中华民国首都|中华民国首都]]。1927年后，[[国民政府|国民政府]]颁布[[首都计划|首都计划]]，开启了南京城现代化之路。1928年，南京铺设了第一条柏油马路——中山大道。从1929年至1937年间，在南京进行了大规模的首都建设。但城区并未突破明代南京城墙范围。[[通济门|通济门]]、[[尧化门|尧化门]]、[[高桥门|高桥门]]等地仍存在大片农田，并以各类[[土特产|土特产]]知名<ref>{{Cite web |url=http://news.ifeng.com/gundong/detail_2013_09/05/29333622_0.shtml |title=民国南京特产，为何如今消失殆尽 |accessdate=2017-11-13 |author= |date= |work= |publisher=凤凰网，来源：金陵晚报|language=Zh-hans }}</ref>。
