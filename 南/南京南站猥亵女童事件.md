'''南京南站猥亵女童事件'''发生于2017年8月12日晚，青年男子段某某在[[南京南站|南京南站]]候车室公开猥亵同行女童，事件经[[新浪微博|新浪微博]]曝光后，引发舆论关注。8月14日<ref name="a1">{{Cite web |url=http://js.ifeng.com/a/20170818/5923159_0.shtml |title=南京猥亵女童案养父身份曝光 系河南滑县原公务员 |accessdate=2017-08-18 |author= |date=2017-08-18 |work=责任编辑：胥大伟 |publisher=凤凰网，来源：北京青年报|language=Zh-hans }}</ref>，南京铁路警方在河南[[滑县|滑县]]将段某某抓获。

== 背景 ==


== 事件经过 ==

8月12日17时左右，在[[南京南站|南京南站]]候车室B6检票口，段某某与父母、养妹一同出行，另有一位熟人在场。段某某公开数度猥亵其妹，其父母、熟人并未阻止。因引起他人关注后，段家人匆匆离去。期间，一位女性网友拍摄下段某某猥亵其妹的照片，向微博用户——作家陈岚投诉。当晚，陈岚在新浪微博将此事曝光。至13日，共有11万网友转发<ref name="b1">{{Cite web |url=http://news.ifeng.com/a/20170813/51623237_0.shtml |title=禽兽！南京南站一年轻男子当众猥亵小女孩，疑似女孩家人竟漠视！10万网友转发 |accessdate=2017-08-18 |author= |date=2017-08-13 |work=编辑：小桃子 |publisher=凤凰网，来源：扬子晚报|language=Zh-hans }}</ref>。

13日上午，南京铁路公安南京南所在官方微博回应陈岚，表示已展开调查。同日，南京警方表示事件正在调查中<ref name="b1"/>。14日，南京铁路警方在河南省滑县将段某某抓获。28日，段某某被批捕<ref>{{Cite web|url=http://news.sina.com.cn/zx/2017-08-29/doc-ifykiurx2652853.shtml|title=南京南站猥亵儿童案犯罪嫌疑人段某某被批捕|accessdate=2018-02-12|work=news.sina.com.cn}}</ref>。

== 后续 ==

=== 案件性质争议 ===

有律师认为，按照相关法律，猥亵8岁以上女童的，属于[[自诉|自诉]]案件。如段某父母不追究其责任，他将仅受到[[治安处罚|治安处罚]]而免于[[刑事处罚|刑事处罚]]<ref>{{Cite web |url=http://jiangsu.sina.com.cn/news/s/2017-08-15/detail-ifyixtym4468074.shtml |title=南京南站涉嫌猥亵女童的男子被抓 双方疑为兄妹 |accessdate=2017-08-18 |author=实习生：沈忱，记者：梅建明 |date=2017-08-15 |work= |publisher=新浪网，来源：扬子晚报网|language=Zh-hans }}</ref>。但这种说法，遭到了南京市公安局江宁分局的反驳，认为此案属[[公诉|公诉]]案件，段某父母是否追究段某责任与案件无关，公众不必担心<ref>{{Cite web |url=http://news.sina.com.cn/sf/news/fzrd/2017-08-15/doc-ifyixias1100288.shtml |title=南京南站猥亵女童案：警方解读四大法律疑问 |accessdate=2017-08-18 |author= |date=2017-08-15 |work= |publisher=|language=Zh-hans }}</ref>。

=== 曝光人被死亡威胁 ===

15日的报道称，事件曝光后，陈岚在新浪微博收到几千条[[死亡威胁|死亡威胁]]、辱骂<ref>{{Cite web |url=http://news.cnr.cn/dj/20170815/t20170815_523901654.shtml |title=“高铁站猥亵案”作家爆料人反遭死亡威胁：最担心家人 |accessdate=2017-08-18 |author=记者：王晶 |date=2017-08-15 |work= |publisher=央广网|language=Zh-hans }}</ref>，她的家庭地址、手机号码等具体訊息被公开。陈岚认为对她发出死亡威胁的人是由网路集结的[[恋童癖|恋童癖]]者<ref>{{Cite web |url=http://news.cnr.cn/dj/20170816/t20170816_523904201.shtml|title=“南京猥亵女童案”爆料人遭致命威胁：已报案 准备搬家 |accessdate=2017-08-18 |author=记者：王晶 |date=2017-08-16 |work= |publisher=央广网|language=Zh-hans }}</ref>。
