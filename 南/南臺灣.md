'''南臺灣'''，為'''[[臺灣|臺灣]][[南部|南部]]'''簡稱。關於南臺灣的範圍定義有許多種，但地理上通指臺灣西南部地區（位於台灣東南部的[[臺東縣|臺東縣]]不被視為南臺灣）。{{Fact|[[臺灣話|臺灣話]]諺語「[[頂港|頂港]]有名聲，下港有出名」中常稱的「'''下港'''」亦意指南部或南臺灣地區|time=2019-03-17T13:43:26+00:00}}。以'''[[高雄市|高雄市]]'''與'''[[臺南市|臺南市]]'''兩大都會區組成雙核心結構的區域核心，並形成臺灣第三大的[[高雄都會區|高雄都會區]]及第五大的[[臺南都會區|臺南都會區]]，並擁有以[[嘉義市|嘉義市]]為核心的[[嘉義都會區|嘉義都會區]]等，共計三個都會區。

== 範圍定義 ==
[[File:Southern_Taiwan_official_determined.svg|thumb]]
* 四分法：依據[[中華民國國家發展委員會|行政院國發會]]都市及住宅發展處所擬訂之「臺灣地區綜合開發計畫」，將[[臺灣|臺灣]]劃分為四個區域，其「南部區域」即為「南台灣」，包括[[嘉義縣|嘉義縣]]、[[嘉義市|嘉義市]]、[[臺南市|臺南市]]、[[高雄市|高雄市]]、[[屏東縣|屏東縣]]及[[澎湖縣|澎湖縣]]<ref>[http://www.cepd.gov.tw/m1.aspx?sNo=0000486&key=%20&ex=2&ic=0000153 《都市及區域發展統計彙編》內容簡介，行政院經濟建設委員會官方網站] {{webarchive|url=https://web.archive.org/web/20071014192628/http://www.cepd.gov.tw/m1.aspx?sNo=0000486&key=&ex=2&ic=0000153 |date=2007-10-14 }}</ref>，為[[中華民國|中華民國]]官方標準劃分方式；面積約10745.048平方公里。
* 四分法：[[交通部中央氣象局|交通部中央氣象局]]亦將臺灣劃分為四個區域，其「南部地區」即為「南臺灣」，包括臺南市、高雄市及屏東縣，與行政院國發會的劃分方式不同。<ref>[http://www.cwb.gov.tw/V7/life/ 生活氣象]，中央氣象局官方網站</ref>
**這與[[國防部陸軍司令部|國防部陸軍司令部]]直屬部隊[[陸軍八軍團|陸軍八軍團]]管轄範圍相符。
* 六分法：[[1972年立法委員選舉|1972年]]至[[1986年立法委員選舉|1986年增額立法委員選舉]]採用大選區制，將台灣劃分成六個選區，其中第五選區及高雄市選區即所謂「高屏澎」地區，可視之為狹義的「南台灣」，包括高雄市、屏東縣及澎湖縣<ref name="中選會">《中華民國選舉概況》，[[中央選舉委員會|中央選舉委員會]]，1984年</ref>。
* 二分法：[[1969年立法委員選舉|1969年立法委員增選]]採用大選區制，將台灣劃分成兩個選區，其中第二選區即為「南台灣」，包括[[彰化縣|彰化縣]]（含）以南的諸縣市及[[花蓮縣|花蓮縣]]、[[臺東縣|臺東縣]]、澎湖縣<ref name="中選會" />。
* 2018年[[嘉義縣政府|嘉義縣政府]]表明從氣候、地景、交通、意識，嘉義縣是南台灣的一部分。<ref name="中央社 CN18">{{cite web |author=黃國芳 |title=觀光局補助漏列阿里山 嘉縣府：難以理解 - 地方 |website=[[中央通訊社|中央通訊社]] CNA |date=2018-05-16 |url=http://www.cna.com.tw/news/aloc/201805160199-1.aspx |language=zh |access-date=2018-05-17|quote=縣府指出，嘉義縣不是「中南部」，從氣候、地景、交通、意識，是清清楚楚的正南方，交通部既然挑選補助指標性風景區，卻忽略最具經典意義的阿里山，甚至放棄全國唯一以濕地為特色的雲嘉南濱海國家風景區，有違期待。|archive-url= https://web.archive.org/web/20180517170217/http://www.cna.com.tw/news/aloc/201805160199-1.aspx|archive-date=2018-05-18}}</ref>

以上係以整個[[臺灣地區|臺灣地區]]作衡量，若僅考慮「台灣本島」時，上述分法均不計入澎湖縣。

另外，還有以[[濁水溪|濁水溪]]為界，以南為南台灣、以北為北台灣的劃分方法，但較不正式。<ref>如 [http://www.sunriver.com.tw/ 上河文化股份有限公司] 所出版之 《台灣人文地理全覽圖》-北島</ref>

== 產業情勢 ==
{{see also|重北輕南}}
南台灣現時主要以高雄市為中心。[[產業|產業]]包括[[重工業|重工業]]，其中高雄地區是台灣主要的[[石油化學工業|石油化學工業]]、[[造船業|造船業]]與[[鋼鐵|鋼鐵]]業基中地。其他尚有[[民生|民生]][[輕工業|輕工業]]，而高雄及台南地區亦有[[南部科學園區|南部科學園區]]、[[農業|農業]]、[[漁業|漁業]]。近年來逐漸轉往[[觀光業|觀光業]]和[[高科技產業|高科技產業]]發展。而嘉義地區主要以一級產業、二級產業及三級產業並重，形成嘉義地區的產業分佈，嘉義地區以[[工業|工業]]、[[農業|農業]]、[[漁業|漁業]]、[[服務業|服務業]]、[[觀光業|觀光業]]為重要產業。

==社會情勢==
研究顯示，南部地區的[[隔代教養家庭|隔代教養家庭]]為數眾多。

== 統計資料 ==
=== 人口 ===
南臺灣[[族群|族群]]分佈與全台灣一樣，以[[閩南民系|閩南民系]]佔多數，分布於城市、港埠與濱海平原。傳統上[[客家人|客家人]]分佈於高屏[[六堆|六堆]]及雲嘉部分[[鄉鎮|鄉鎮]]，[[臺灣原住民族|原住民]]則集中於嘉義縣、高雄市、屏東縣之山區地帶；而[[臺灣外省人|臺灣外省人]]的[[眷村|眷村]]主要分布居於各大[[都市|都市]]及[[城鎮|城鎮]]中。
