'''南北大道有限公司'''（{{lang-ms|PLUS Expressways Berhad}}，{{MYX|5052}}）是[[马来西亚|马来西亚]]国内最大的高速公路特许和[[BOT模式|BOT模式]]经营公司。该公司隶属于[[东南亚|东南亚]]最大、世界第八大的收费高速公路运营公司{{tsl|ms|Kumpulan UEM|马友乃德集团}}<ref>{{cite web|url=http://www.uem.com.my/pmb.aspx|title=UEM Group Berhad - PLUS Malaysia Berhad|website=www.uem.com.my}}</ref>。

== 历史 ==
公司1986年6月27日以高速公路特许经营有限公司（Highway Concessionnaires Berhad）之名成立，是{{tsl|ms|Kumpulan UEM|马友乃德集团|马友乃德工程有限公司}}的成员公司之一<ref name="hids6">{{cite web|url = http://hids.arkib.gov.my/readarticle.php?article_id=710&y=2007&m=4|title = UEM Tandatangani Perjanjian Dengan JKR Untuk Pembinaan Lebuhraya Utara-Selatan|author = <!--Staff writer(s); no by-line.-->|date = 1986-12-29|website = Hari Ini Dalam Sejarah|publisher = [[National_Archives_of_Malaysia|National Archives of Malaysia]]|access-date = 2015-05-31|quote = |deadurl = yes|archiveurl = https://web.archive.org/web/20150925114823/http://hids.arkib.gov.my/readarticle.php?article_id=710&y=2007&m=4|archivedate = 2015-09-25}}</ref>。1988年3月13日，该公司更名为「南北大道工程有限公司」（Projek Lebuhraya Utara Selatan Berhad，简称PLUS）<ref name="plus2">{{cite web |url = http://plus.com.my/index.php?option=com_content&view=article&id=4&Itemid=142&lang=en|title = Subsidiaries: Projek Lebuhraya Utara-Selatan Berhad|author = <!--Staff writer(s); no by-line.-->|date = |website = |publisher = [[PLUS_Expressways|PLUS Malaysia Berhad]]|access-date = 2015-09-19|quote = }}</ref>。2002年1月29日，公司再度易名为南北大道高速公路有限公司（PEB），在马来西亚股票交易局注册成为上市公司<ref>{{cite web|url=https://azreenandanis.weebly.com/plus-berhad.html|title=Plus Berhad|website=TOUCH 'N GO INFORMATION SYSTEM}}</ref>。八年后，南北大道公司正式于2010年11月29成为投资控股公司<ref name="auto1">{{cite web|url=http://www.plus.com.my/index.php?option=com_content&view=article&id=139&Itemid=140&lang=en|title=PLUS MALAYSIA BERHAD - Background|first=|last=Administrator|website=www.plus.com.my}}</ref>2011年11月29日，马来西亚南北大道有限公司对原有公司进行并购，并成为了南北大道公司、第二中环大道公司、第二通道公司、北海－居林大道财团公司、自强科技公司、南北大道直升机服务公司的控股公司，同时也是一触即通公司的大股东。南北大道有限公司也从马友乃德建筑买下了槟威大桥有限公司<ref name="auto1"/>。南北大道有限公司的并购工程于2012年1月12日完成，使该公司成为马来西亚最大、世界第八大的收费高速公路运营公司<ref name="auto1"/>。

'''南北大道总部'''（{{lang-ms|Persada PLUS}}）是马来西亚南北大道有限公司的总部，位于[[马来西亚|马来西亚]][[雪兰莪州|雪兰莪州]][[八打灵再也|八打灵再也]][[新巴生河流域大道|新巴生河流域大道]]梳邦交通枢纽的南北大道总部（Persada PLUS）<ref name="auto"/>总部曾是南北大道公司和大道管理公司（PROPEL）在新巴生河流域大道的办公室，而总部外也设有一所迷你足球场供南北大道公司旗下足球队{{tsl|ms|PLUS FC|南北大道足球俱乐部}}踢球<ref>{{cite web|url=http://us.soccerway.com/venues/malaysia/stadium-mini-persada-plus/v20439/|title=Stadium Mini Persada PLUS - Soccerway|website=us.soccerway.com}}</ref>。[[万达镇－巴生线|万达镇－巴生线]]此前有意在南北大道总部设立站点，惟之后在民众反馈下移至附近的商业区<ref>{{cite web|url=http://www.sinchew.com.my/node/1635963|title=LRT 3第21站绕庙而行．修改6站点路线|date=2017-04-21|publisher=}}</ref>。

==目前南北大道公司的产品==

===南北大道里程卡===
南北大道里程卡（{{lang-en|PLUSMiles}}）是由南北大道公司推出的收费回扣忠诚计划，于2008年12月17日推出。持卡者可享有收费回扣和获得参与活动的商家送出的商品。每张里程卡都具有一触即通卡的功能，可用来缴付过路费、停车费、[[巴生谷|巴生谷]]的公共交通车费等。<ref>https://plusmiles.com.my/about?id=88a370a3-3dda-4413-9c46-7ea0f96ceb0c</ref>

===PLUSTrack===
PLUSTrack优惠卡是一种先付形式的预付卡；为使用者提供3%大道通行费回扣等，还提供一系列奖品和福利<ref>{{cite web|url=http://www.touchngo.com.my/CMS/Business/Business-Products/PLUSTrack/About-PLUSTrack/|title=Touch 'n Go :: About PLUSTrack|first=Touch N|last=Go|website=www.touchngo.com.my}}</ref>。

==南北大道巡逻队==
南北大道巡逻队（{{lang-ms|PLUS Ronda}}）于1990年4月1日创立，在1998年10月7日获得[[马来西亚皇家警察|马来西亚皇家警察]]授予[[輔警|輔警]]资格<ref name="auto2">{{cite web|url=http://www.plus.com.my/index.php?option=com_content&view=article&id=86&Itemid=58&lang=en|title=PLUS MALAYSIA BERHAD - PLUSRonda|first=|last=Administrator|website=www.plus.com.my}}</ref>。

===工作范围<ref name="auto2"/>===
* 提供全天24小时协助抛锚或意外事件。
* 提供拖车至最靠近的安全地方服务
* 报告联通流量。Report traffic flow.
* 在紧急时协助有关当局
* 管理休息站和公路服务区的设施状况。
* 作为大道的辅警部门。
