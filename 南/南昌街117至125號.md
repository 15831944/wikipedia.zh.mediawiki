'''南昌街117至125號'''（{{lang-en|117-125 Nam Cheong Street}}），是[[香港|香港]]一連五座的戰前[[唐樓|唐樓]]群，位於香港[[九龍|九龍]][[深水埗區|深水埗區]][[南昌街_(香港)|南昌街]]117至125號。

此為深水埗區現存最大型的唐樓群<ref>{{Cite web|url=http://www.hk-place.com/view.php?id=263|title=建設及建築物 － 戰前唐樓 ( 二 ) 九龍|accessdate=2018-10-19|author=|date=|publisher=香港地方}}</ref>，其中117號為著名的「南昌押」，建築群1999年曾被列為[[香港二級歷史建築列表|二級歷史建築]]，後來於2010年被降評為[[香港三級歷史建築列表|三級]]<ref>{{Cite web|url=http://www.aab.gov.hk/form/AAB-SM-chi.pdf|title=1,444幢歷史建築物名單和評估結果（截至2018年9月6日）|accessdate=2018-10-19|author=古物諮詢委員會|date=2018-09-06|publisher=}}</ref>。

* 117號唐樓：建於1920年代，樓高5層，為典當舖「南昌押」

* 119及121號唐樓：建於1933年，樓高3層（加1層閣樓），之前曾經是「海麗賓館」，現在為「香港港麗商務酒店」<ref>{{Cite web|url=http://www.isletforum.com/forum.php?mod=viewthread&tid=11542|title=南昌街119-125號 列為三級歷史建築|accessdate=2018-10-19|author=Cucumber|date=2015-08-27|publisher=小島討論區}}</ref>

* 123及125號唐樓：建於1930年代，樓高4層，123號曾經是「德興茶樓」，125號地舖曾為「銀龍茶餐廳」<ref>{{Cite web|url=https://www.hk01.com/%E7%A4%BE%E5%8D%80%E5%B0%88%E9%A1%8C/109356/%E8%B7%9F%E7%9D%80%E6%B8%AC%E9%87%8F%E5%B8%AB%E9%80%9B%E6%B7%B1%E6%B0%B4%E5%9F%97-%E5%98%89%E9%A0%93-%E9%B4%A8%E8%A8%98%E6%87%89%E4%BF%9D%E8%82%B2-%E6%B4%BB%E5%8C%96%E8%A6%81%E9%A1%A7%E5%8F%8A%E5%85%A7%E9%83%A8%E7%B5%90%E6%A7%8B%E7%89%B9%E8%89%B2|title=跟着測量師逛深水埗：嘉頓、鴨記應保育　活化要顧及內部結構特色|accessdate=2018-10-19|author=吳映彤|date=2017-08-03|publisher=香港01}}</ref>，牆身仍留有招牌痕跡

2010年，南昌街119及121號以7380萬元易手，而原業主於2004年斥2000萬元入市，現分租地舖以及於閣樓和1至3樓經營商務酒店<ref>{{Cite news|url=https://hk.finance.appledaily.com/finance/daily/article/20120707/16491555|title=近朗豪坊酒店轉手翻一番|author=|date=2012-07-07|work=蘋果日報|accessdate=2018-10-19}}</ref>。

== 南昌押 ==
位於117號的整座唐樓為一當舖「南昌押」，原名「同安大押」，位於南昌街及[[汝州街|汝州街]]交界，業主為在香港開設不少典當鋪的賭業大亨 — [[高可寧|高可寧]]家族<ref>{{Cite web|url=https://hk.news.appledaily.com/local/daily/article/20150914/19294668|title=地段貴　高可寧家族大押難逃重建|accessdate=2018-10-19|author=鍾雅宜|date=2015-09-14|publisher=蘋果日報}}</ref>。

香港在1926年典當合法化後<ref>{{Cite web|url=https://www.discoverhongkong.com/tc/see-do/neighbourhoods/sham-shui-po/live.jsp|title=香港 ‧ 大城小區：深水埗 - 住|accessdate=2018-10-19|author=|date=|publisher=香港旅遊發展局}}</ref>，典當業隨之急速發展，同安大押（現在南昌押）亦在這個時候立足於深水埗，並至到1950年代改名並沿用至今<ref>{{Cite web|url=https://travel.ulifestyle.com.hk/DetailTour.php?id=ADkRZBEu|title=深水埗　活化建築之旅|accessdate=2018-10-20|author=|date=2014-08-29|publisher=U Travel}}</ref>。

樓宇的總樓面面積約800呎，每層均有「南昌押」的字樣及一圓型小氣窗<ref>{{Cite web|url=http://oldhkphoto.com/117-nam-cheong-st/|title=深水埗南昌押|accessdate=2018-10-19|author=Sharon Lam|date=2015-02-12|publisher=香港舊照片}}</ref>，而兩邊街道都分別掛上招牌及於晚上亮起[[霓虹燈|霓虹燈]]，使交界兩邊都能見到。

面向南昌街的[[騎樓|騎樓]]有兩條橫跨行人路的支柱，與旁邊的唐樓群形成一條迴廊。唐樓亦沿用傳統「上居下舖」的模式，分別地下作商舖，而樓上為居所。

== 清拆危機 ==
2018年1月，業主公開招標出售119丶121號及123號共三幢物業，意向價2.2億元，若重建商住大廈最多可以9倍地積比率發展<ref>{{Cite web|url=http://paper.wenweipo.com/2018/01/17/ME1801170010.htm|title=南昌街三級歷史建築2.2億元標售|accessdate=2018-10-19|author=|date=2018-01-17|publisher=文匯報}}</ref>。
