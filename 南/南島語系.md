'''南島語系'''（{{lang-en|Austronesian languages}}）是主要由[[南岛民族|南岛民族]]所使用的语言，是世界現今唯一主要分布在島嶼上的一個[[語系|語系]]，包括約1300種語言。其分布主要位于南太平洋群岛，包括[[臺灣|臺灣]]、[[海南|海南]]、[[越南|越南]]南部、[[菲律宾|菲律宾]]、[[马来群岛|马来群岛]]，東達[[南美洲|南美洲]]西方的[[復活節島|復活節島]]，西到東[[非洲|非洲]]外海的[[馬達加斯加島|馬達加斯加島]]，南抵[[紐西蘭|紐西蘭]]。[[臺灣|臺灣]]是南島語系內部分化的源頭，也是目前南島語系最北部的地區<ref>[http://udn.com/NEWS/NATIONAL/NAT5/4706748.shtml 紐西蘭研究 玻里尼西亞人來自臺灣]</ref>，居住在[[新北市|新北市]][[烏來區|烏來區]]的[[泰雅族|泰雅族]]聚落，目前是全世界南島語系居住地中，是人口分布上地理最北端的聚落；東西的延伸距離，超過地球圓周的一半，總人口數大約兩億五千萬之多，絕大多數居住在東南亞的菲律賓<ref>[http://udn.com/NEWS/NATIONAL/NAT5/4706751.shtml 巴丹人 也源自臺灣原住民]</ref><ref>[http://udn.com/NEWS/NATIONAL/NAT5/4706746.shtml 臺灣原民、南島語族 同條文化臍帶]</ref>、印尼、馬來西亞，至於[[新畿內亞|新畿內亞]]以東的太平洋島嶼有一百多萬人。

== 主要分佈區域 ==

所謂的「南島」（Austronesia），是1899年由學者Wilhelm Schmidt自[[拉丁文|拉丁文]]字的字根「auster」與[[希臘文|希臘文]]「nêsos」所合組而成<ref>[http://www.cip.ntu.edu.tw/main06.htm 國立臺灣大學原住民族研究中心－－南島文化] {{webarchive|url=https://web.archive.org/web/20081001002117/http://www.cip.ntu.edu.tw/main06.htm |date=2008-10-01 }}</ref>，前者為南風之意，後者意指島，所以日本語將「Austronesia」翻譯成「南島」。中文在近代沿用了這個日本語詞彙。

使用南島語系的區域包括[[臺灣|臺灣]]、[[海南|海南]]、[[菲律賓|菲律賓]]、[[馬來西亞|馬來西亞]]、[[新加坡|新加坡]]、[[印尼|印尼]]、[[馬達加斯加|馬達加斯加]]、[[新畿內亞|新畿內亞]]、[[紐西蘭|紐西蘭]]、[[夏威夷|夏威夷]]、[[密克羅尼西亞|密克羅尼西亞]]、[[美拉尼西亞|美拉尼西亞]]、[[玻里尼西亞|玻里尼西亞]]等各地島嶼的語言，外加[[馬來半島|馬來半島]]上的[[馬來語|馬來語]]、[[中南半島|中南半島]]上[[越南|越南]]與[[高棉|高棉]]的[[佔語|佔語]]和[[泰國|泰國]]的[[莫肯語|莫肯語]]（{{lang|en|Moken}}）及[[莫克倫語|莫克倫語]]（{{lang|en|Moklen}}）<ref name = "AS">中央研究院平埔文化資訊網 nd, a</ref>。北起臺灣，南抵紐西蘭，西至馬達加斯加，東至智利復活節島。

南島語系分佈的區域很廣，語言的數目也頗多，根據《[[民族語|民族語]]》這個世界語言資料庫的資料，其總數有1262種之多（Ethnologue: Languages of the World 2004）。
由於移民及近代帝國主義殖民，在劃分為南島語系的區域內，部分的島上也有使用非南島語族的語言。以臺灣為例，除了有隸屬於南島語族的[[台灣原住民|原住民族]]族語以外，由於17與18世紀時，來自大明帝國、大清帝國的移民而同時存在著不同的漢語，包括了[[閩南語|閩南語]]（[[臺灣話|臺灣話]]）和[[客家話|客家話]]（[[臺灣客家語|臺灣客家語]]）。台灣於19世紀末至20世紀中期的[[台灣日治時期|台灣日治時期]]則以[[日語|日語]]為官方語言；[[臺灣戰後時期|第二次世界大戰後時期]]，[[中華民國|中華民國]]政府統一使用[[現代標準漢語|現代標準漢語]]（[[中華民國國語|國語]]）。<ref name = "AS">中央研究院平埔文化資訊網 nd, a</ref>

其他如菲律賓相繼受到西班牙與美國殖民、夏威夷受到美國殖民、紐西蘭受英國殖民、印尼受到荷蘭殖民、馬來西亞受到英國殖民、越南與柬埔寨受到法國殖民等，其殖民母國的語言在其現今語言系統內仍造成不同影響。

== 分類 ==
[[File:Austronesian_languages.PNG|right]]被視為南島語系的主要源頭]]

根據《民族語》的分類，'''南島語系'''可以再分為兩大語族，第一是「'''[[台灣南島語|台灣南島語]]'''」，第二為「'''[[馬來-玻里尼西亞語|馬來-玻里尼西亞語]]'''」。前者有23種語言，可以再分成[[泰雅語群|泰雅語群]]、[[排灣語群|排灣語群]]、和[[鄒語群|鄒語群]]三大系統。後者則有1239種語言，可以再分成「[[中-東部馬來-玻里尼西亞語族|中-東部馬來-玻里尼西亞語族]]」（Central-Eastern Malayo-Polynesian）、「[[西部馬來-玻里尼西亞語族|西部馬來-玻里尼西亞語族]]」、以及兩種尚無法歸類的語言<ref>《民族語》''Ethnologue: Languages of the World 2004''</ref>。

關於「南島語」（Austronesian languages）和「[[馬來-玻里尼西亞語|馬來-玻里尼西亞語]]」的關係，還有其他的說法。其說法是「南島語」有十種次語系，其中有九種是在台灣，最後一種則是「馬來-玻里尼西亞語」<ref>Diamond(2000)</ref>。也就是說，雖然其分類方式和《民族語》所列舉的並不一致，但一樣是將「馬來-玻里尼西亞語」當成是「南島語系」的一個次語系。

台灣[[中央研究院|中央研究院]]（1996）明白陳述：「南島語系（Austronesian）又稱馬來-玻里尼西亞語系（Malay-Polynesian），是世界上唯一主要分布在島嶼上的一個大語系」。中央研究院平埔文化資訊網也說，「生活在台灣的土著民族，即平埔族和高山族，他們所使用的語言，不僅自己相互之間有很深的關係，而且跟太平洋、印度洋島嶼上的許多種語言都有親屬關係，形成學術界所稱的「南島語系」（Austronesian） 或 （Malayopolynesian）」<ref>{{Cite web |url=http://twstudy.sinica.edu.tw/twstudy/preHistory/part3.html |title=史前時代南島語系民族的大遷移 |access-date=2009-01-24 |archive-url=https://web.archive.org/web/20090114122539/http://twstudy.sinica.edu.tw/twstudy/preHistory/part3.html |archive-date=2009-01-14 |dead-url=yes }}</ref>。

某些語言學者相信[[泰語|泰語]]可能也可以算是擴大定義之「南島語系」的一部份，雖然多數學者似乎仍舊將其歸類為「[[壯侗語族|壯侗語族]]」。學者[[白保羅|白保羅]]認為「南島語系」和「[[南亞語系|南亞語系]]」以及「壯侗語族」可能有血緣關係，正研究[[南方大語系|南方大語系]]，由此成為語言學家的另一項研究專題。

== 特征 ==
南岛语系各语言都是[[黏著语|黏著语]]，語音上辅音有[[清音|清辅音]]和[[浊音|浊辅音]]；語法上有[[格_(语法)|格]]，但沒有[[性_(语法)|性]]的区别；广泛利用[[词缀|词缀]]和詞幹[[元音|元音]]音变来表达[[语法|语法]]意义；[[动词|动词]]有[[时态|时态]]、[[语态|语态]]和[[语体|语体]]的变化，[[主语|主语]]和动词在变化中互相呼应。南岛语系各语言的词都有[[重音|重音]]，另外除了長期受到周圍漢語、黎語等有聲調語言影響的[[回輝話|回輝話]]以外，其他南島諸語沒有聲調系統。

== 歷史研究 ==
===起源和擴散===

[[File:Polynesian_Migration.svg|right]]是南島語系[[玻里尼西亞|玻里尼西亞]]人的原鄉]]

關於南島語的起源問題，{{when|是學界上還算熱門的一個研究題目。}}南島語系的起源，與[[南島民族|南島民族]]的起源有緊密相關。随着分子人类学与基因组学的Y单倍型研究，[[汉-南岛同源祖语|汉-南岛同源祖语]]得到正视。祖语的语音形式以 CVCV 、CVCVC 两种为绝大多数。这些语言里面，南岛语、尤其是台湾和菲律宾的南岛语，最为接近这个「大O超语系」祖语的原貌。

臺灣[[原住民|原住民]]與[[菲律賓|菲律賓]]、[[印度尼西亞|印度尼西亞]]各島原住民皆屬於南島民族且有密切關係。這些臺灣史前移民的語言因而與上述其他地區具有共通性。比較早期的說法，包括該語言起源於[[麥克羅尼西亞|麥克羅尼西亞]]、[[華南|華南]]、[[中南半島|中南半島]]、西[[新畿內亞|新畿內亞]]等論點。<ref><!--關於這些說法的簡單回顧，可以參考-->{{Cite web|url=http://www.ianthro.tw/ <!--原www.sinica.edu.tw/~pingpu/-->|title=平埔文化專題<!--前中央研究院平埔文化資訊網-->|accessdate=|author=|last=|first=|authorlink=|coauthors=|date=|year=|month=|trans_title=|format=|work=中研院民族所數位典藏|publisher=中央研究院|pages=|language=|doi=|archiveurl=https://web.archive.org/web/20140819090346/http://www.ianthro.tw/|archivedate=2014-08-19|quote=|deadurl=yes}}</ref>

=== 與其他语系的关系 ===
关于南岛语系和其他语系的同源关系，学者议论纷纷，目前有五个不同假设：
# 德国传教士Wilhelm Schmidt 1899年提出南岛语系和[[南亚语系|南亚语系]]同源。
# 美国学者Dahl 1970年指出南岛语系和[[印欧语系|印欧语系]]同源。
# 法国漢学家[[沙加尔|沙加尔]]1990年提出南岛语系和[[汉藏语系|汉藏语系]]同源。
# 日本一些學者認為南島語系與[[阿爾泰語系|阿爾泰語系]]構成一種混合語言就是[[日語|日語]]。
# 美国人潘乃德提出泰语（[[壮侗語族|壮侗語族]]）就是南岛语系。
