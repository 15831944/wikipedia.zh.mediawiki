'''南阪奈道路'''（{{lang|ja|南阪奈道路／みなみはんなどうろ}}，MINAMIHANNA ROAD）是從[[大阪府|大阪府]][[堺市|堺市]][[美原區|美原區]]至[[奈良縣|奈良縣]][[葛城市|葛城市]]的[[快速道路|快速道路]]（營業路線名）。略稱為'''南阪奈道'''（{{lang|ja|南阪奈道／みなみはんなどうろ}}）。

另外，'''南阪奈道路'''及{{link-ja|大和高田繞道|大和高田バイパス}}（{{link-ja|國道165號|国道165号}}、{{link-ja|166號|国道166号}}、[[国道24号|24号]]）合併，被指定為[[地域高規格道路|地域高規格道路]]計畫路線上的'''南阪奈道路'''（法定路線名）<ref>{{cite web|date = |title = 地域高規格道路|url = https://www.kkr.mlit.go.jp/road/kinki_road/tiikikoukikaku.html|format = |page = |publisher = 国土交通省近畿地方整備局道路部|accessdate = 2012-6-12}}</ref>。

高速公路路線編號與{{link-ja|大和高田繞道|大和高田バイパス}}（弁之庄匝道 - 四条匝道間的高架路段）共同被編為 {{Ja Exp Route Sign|E91}} <ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/list/index.html|title=高速道路ナンバリング一覧|accessdate=2017年3月3日|publisher=国土交通省}}</ref><ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/map/index.html|title=高速道路ナンバリング路線図|accessdate=2017年3月3日|publisher=国土交通省}}</ref>。

== 概要 ==
[[File:Taishi_TB,_Minamihanna_Road.JPG|thumb]]附近]]
[[地域高規格道路|地域高規格道路]]計畫路線上的'''南阪奈道路'''包含{{link-ja|大和高田繞道|大和高田バイパス}}（{{link-ja|國道165號|国道165号}}、{{link-ja|166號|国道166号}}、[[国道24号|24号]]）。

營運單位總計有大阪府、[[日本道路公團|日本道路公團]]、[[國土交通省|國土交通省]]、奈良縣共4機關，收費制度也隨著路段有所不同，是條相當複雜的路線（詳細請參考[[#道路體系|道路體系]]）。

啟用1年間的一日平均交通量約為15,000台，雖然起了緩和[[西名阪自動車道|西名阪自動車道]]交通堵塞的作用，但仍遠遠不及原先大阪府道路公社所計畫的一日交通量27,000台的目標。

=== 規格 ===
*{{link-ja|道路構造令|道路構造令}} : 第1種第3級（普通道路）
* 車道數 : 完成4車道（羽曳野 - 新庄 [[2車道高速公路|暫定2車道]]<ref>部分路段有附加車道</ref>）
* 設計速度 : 80 km/h
;詳細請參考[[#車道與速限|車道與速限]]。
