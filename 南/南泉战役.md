'''南泉战役'''，也称重庆战役，是[[第二次国共内战|第二次国共内战]]中，中共领导的[[中国人民解放军|解放军]]南下部队与[[中華民國國軍|中华民国国军]]发生的最后一场攻打特大城市的[[战役|战役]]。

== 背景 ==

1949年10月，[[刘伯承|刘伯承]]，[[邓小平|邓小平]]率领的[[中国人民解放军第二野战军|中国人民解放军第二野战军]]为主力的西南部队攻入[[貴州省_(中華民國)|贵州]]，整个西南地区最重要的城市重庆便成为了國共战争中[[中華民國國軍|国军]]控制的最后一座特大型工业城市。守卫重庆的国军兵力有川陕甘边区绥靖公署主任[[胡宗南|胡宗南]]部从[[成都平原|成都平原]]劳师远征到达重庆的3个兵团，西南军政长官公署长官[[张群|张群]]部5个兵团等部，总计37个军约45万人。

== 战役进程 ==

===第一阶段===
*1949年11月7日，[[中国人民解放军第二野战军|中国人民解放军第二野战军]]3兵团全面进攻[[宋希濂|宋希濂]]部[[川东|川东]]防线，攻克川东南门户[[秀山县|秀山县]]。
*11月11日攻克[[酉阳县|酉阳县]]、[[建始县|建始县]]、[[恩施县|恩施县]]，并分路11日占领[[咸丰县|咸丰县]]、12日攻克[[黔江县|黔江县]]，11月21日进入[[彭水县|彭水县]]，在[[涪陵县|涪陵县]]重创[[宋希濂|宋希濂]]主力第2军9师，22日涪陵县被攻克。自此重庆已经危在旦夕。

===第二阶段===
*11月24日，解放军开始发起对[[重庆主城区|重庆主城区]]的攻击。在南温泉一线，解放军与[[胡宗南|胡宗南]]嫡系第一军展开残酷的拉锯战。解放军11军损失惨重，被迫停止攻击南温泉。
*11月26日，解放军转向重庆外围发起冲击。第二野战军第12军103团再次南泉激战三天，吸引了国军主力注意力，但国军顽强的抵抗依旧造成解放军3000人以上的伤亡。在解放军久攻不下再次撤离南泉战场的同时，遭受沉重打击的胡宗南部167师亦弹尽粮绝退出南泉。
*11月27日，第12军35师攻占[[綦江县|綦江县]]进入[[江津县|江津县]]，迂回重庆主城区。此时，解放军对重庆主城区已形成东、西、南三面合围之势。国军开始撤退，同日，重庆城中发生大火，全城大半被毁。{{fact|关押于[[中美合作所|中美合作所]]|time=2013-03-07}}[[渣滓洞|渣滓洞]]、[[白公馆|白公馆]]的政治犯被处决。

===第三阶段===
*11月28日，解放军第三兵团和[[中国人民解放军第47军|第47军]]将[[宋希濂|宋希濂]]部3万余人歼灭于[[南川县|南川县]]，并重庆外围歼灭胡宗南的国军第1军一部。
*11月29日，[[蒋介石|蒋介石]]布置对全重庆市兵工厂、发电厂、钢铁厂等许多个重要目标实施破坏，30日凌晨6时，在[[重庆白市驿机场|重庆白市驿机场]]乘机离开重庆。
*11月30日，解放军发起对渝中半岛的总攻。47军从[[朝天门|朝天门]]，11军从[[南纪门|南纪门]]、[[储奇门|储奇门]]、[[南坪|南坪]]攻入市区，攻占重庆。<ref name="军事科学">《中国人民解放军全史》军事历史研究部 编，军事科学出版社，2000年，ISBN 7-80137-315-4，卷“中国人民解放军战役战斗总览”。</ref>。

==总结==
南泉战役以解放军占领重庆告终，国军退出永久陪都重庆后，无论是军事上还是政治舆论上都已经宣布其在大陆的势力基本灭亡。

但解放军在南泉的重大伤亡，在战役结束的很多年后，自己亦是四川人（巧合的是刘的出生地[[开县|开县]]在数十年后亦划入新的重庆直辖市）的总指挥[[刘伯承|刘伯承]]元帅依然对此作出了检讨，认为是眼光局限于南泉一点造成了不必要的伤亡。此外，重庆主城被纵火，这在解放军攻打特大城市的历史上是唯一一次城市基本全部被毁的失败，这场火灾也使得重庆工业、商业瘫痪，在几十年时间内失去了和[[北京|北京]]、[[上海|上海]]、[[武汉|武汉]]、[[广州|广州]]一样的中心城市地位，影响深远。加之关押在中美合作所的政治犯没有及时被营救以至于大部分被处决，都被认为是这场战役解放军一方战斗指挥不完美之处。然而另一方面来讲，重庆地形复杂，渝中半岛三面临江，一面有佛图关天险，整体地形又天然呈碉堡状，自古易守难攻。炮兵、重步兵数量的优势几乎无从发挥，加之作战初期即制订目标，要求像解放[[北京|北京]]、[[上海|上海]]乃至[[武汉|武汉]]一样解放重庆，避免市区工商业设施被毁，导致处处进攻过度小心谨慎，造成解放军久攻不下。
