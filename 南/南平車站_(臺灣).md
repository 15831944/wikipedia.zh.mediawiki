'''南平車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[鳳林鎮_(臺灣)|鳳林鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]，本站配合「花東新車站運動」計畫進行改建，改建工程已於2014年7月正式完工。

== 車站構造 ==
*[[島式月台|島式月台]]一座。

== 利用狀況 ==
*目前為乙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]。
*停靠狀態：僅停靠[[臺鐵區間車|區間車]]。
*貨運為主，為[[中國鋼鐵公司|中國鋼鐵公司]][[煉鋼|煉鋼]]原料[[白雲石|白雲石]]運送站。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1915年9月16日：設立林田驛開始營業。
*1962年12月15日：改稱南平。
*2016年6月28日：啟用多卡通刷卡機。
*2017年9月16日：本站由[[三等站|三等站]]降為乙種[[簡易站|簡易站]]，並指定由[[鳳林車站|鳳林站]]管理。
