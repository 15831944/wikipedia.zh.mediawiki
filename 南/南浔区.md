'''南潯區'''是[[中國|中國]][[浙江省|浙江省]][[湖州市|湖州市]]下轄的一個區，於2003年1月經[[中華人民共和國國務院|國務院]]批准設立。南潯區北瀕[[太湖|太湖]]，東接[[江蘇省|江蘇省]][[蘇州市|蘇州市]]，地處[[長三角|長三角]]城市群的中心腹地，距離[[上海|上海]]、[[蘇州|蘇州]]、[[杭州|杭州]]等大城市均為100公里左右，[[318國道|318國道]]和[[湖鹽公路|湖鹽公路]]貫通全境，[[京杭運河|京杭運河]]和[[長湖申航道|長湖申航道]]、[[申蘇浙皖高速公路|申蘇浙皖]]和[[申嘉湖高速公路|申嘉湖高速公路]]穿境而過。區域面積716平方公里，轄9個鎮、1個省級經濟開發區，人口51.4萬。區人民政府駐[[南潯鎮|南潯鎮]]。

== 地理 ==
[[File:南浔洪济桥.jpg|thumb]]
南潯區位於[[湖州市|湖州市]]東北部，地處[[浙江|浙江]]北部[[杭嘉湖平原|杭嘉湖平原]]。東南鄰[[桐鄉市|桐鄉市]]，東北毗[[江蘇|江蘇]][[苏州市|苏州市]][[吳江区|吳江区]]，南連[[德清縣|德清縣]]，西北接[[吳興區|吳興區]]。總面積716平方公里。地勢低平，河網密布，屬典型的水網平原。

氣候屬北[[亞熱帶季風氣候|亞熱帶季風氣候]]區，濕潤溫和，四季分明，年平均氣溫在15．5~C～16~C之間，1月氣溫最低，平均2．8~C～3．8℃；7月最高，平均28~C～28．3~C。年平均雨日142～155天，平均降水量在1050—1850毫米左右，平均濕度為78％左右，年平均風速為3．2米／秒左右，無霜期224—246天。年日照為45％，氣候溫和宜人。

== 沿革 ==
{{無來源章節}}
據南潯區境內發現的洪城、花城、上石等古文化遺址證明，早在[[新石器時代|新石器時代]]，原始社會氏族部落就在此繁衍生息。歷史上，南潯隸屬關係變化很大。
*相傳[[夏|夏]][[禹|禹]]治水，劃天下為[[九州|九州]]，南潯轄域隸屬[[揚州|揚州]]。
*[[春秋|春秋]][[戰國|戰國]]時期先後屬[[吳|吳]]、[[越|越]]、[[楚|楚]]。
*公元前333年，楚以此為[[春申君|春申君]]黃歇之封邑，始建[[菰城縣|菰城縣]]，南潯轄域隸屬菰城縣。
*[[秦|秦]]滅六國後，分天下為三十六郡，置[[會稽郡|會稽郡]]，下設烏程、由拳等縣，今區境屬[[烏程縣|烏程縣]]。
*[[西晉|西晉]][[太康|太康]]三年（公元282年），分烏程縣東鄉置[[東遷縣|東遷縣]]，縣治在今[[舊館鎮|舊館鎮]]。
*[[南朝|南朝]]宋[[元徽|元徽]]四年（公元476年），東遷縣改名[[東安縣|東安縣]]，次年仍複名東遷縣。
*[[隋|隋]][[開皇|開皇]]九年（公元589年）東遷縣並入烏程縣。
*[[宋|宋]][[太平興國|太平興國]]7年，分烏程縣東南15鄉置[[歸安縣|歸安縣]]。
*[[民國|民國]]元年（1912年），烏程、歸安合並為[[吳興縣|吳興縣]]，今南潯轄域隸屬吳興縣。
*1949年5月2日，南潯被中共控制，今南潯轄區隸屬吳興縣。 
*1981年1月，撤消吳興縣，改稱[[湖州市|湖州市]]。
*1983年7月，撤消[[嘉興地區|嘉興地區]]，湖州市改為省轄市，建立城、郊兩區，南潯隸屬湖州市郊區。
*1988年11月，撤消湖州市城、郊區建制，實行市管縣又直接領導原城、郊區所屬鄉鎮、街道的體制，南潯轄區各鎮直屬湖州市。
*1993年9月，湖州市區下設[[城區|城區]]、南潯區、[[菱湖區|菱湖區]]3個區工委。今轄區內南潯鎮、練市鎮、雙林鎮、善璉鎮、舊館鎮屬南潯區；菱湖鎮、和孚鎮、千金鎮、石淙鎮屬菱湖區。
*2003年1月，湖州市撤消城區、南潯區、菱湖區三個區委、區管委會，設立[[吳興區|吳興區]]、南潯區兩個市轄區。南潯區區委、區政府駐南潯鎮，轄南潯、雙林、練市、善璉、舊館、菱湖、和孚、千金、石淙9個鎮和南潯經濟開發區。

==行政區劃==
南潯區現轄9個鎮，33個社區（其中15個村居合一社區），221個行政村（3954個村民小組）。

九大鎮有：[[南潯鎮|南潯鎮]]、[[練市鎮|練市鎮]]、[[雙林鎮|雙林鎮]]、[[善璉鎮|善璉鎮]]、[[舊館鎮|舊館鎮]]、[[菱湖鎮|菱湖鎮]]、[[和孚鎮|和孚鎮]]、[[千金鎮|千金鎮]]、[[石淙鎮|石淙鎮]]。

== 人口 ==
全區總人51.4萬人。其中南潯鎮11.74萬人，練市鎮8.54萬人，雙林鎮7.5萬人，善璉鎮3.05萬人，舊館鎮1.8萬人，菱湖鎮8.4萬人，和孚鎮5.4萬人，幹金鎮2.35萬人，石淙鎮1.49萬人。

== 歷史 ==
南潯鎮建鎮於宋代，與[[周莊|周莊]]、[[烏鎮|烏鎮]]、[[西塘|西塘]]、[[同里|同里]]、[[甪直|甪直]]並稱江南六大古鎮，位列浙江歷史文化名鎮之首，有「江浙雄鎮」之稱，2004年人口11.74萬。

==文化旅遊==
南潯自古以來文化昌盛，經濟發達，人才輩出，書香不絕。[[明代|明代]]時就有「九里三閣老，十里兩尚書」之諺。僅宋，明，清三代，南潯就出了進士41名。與南潯相距7里遠的輯里村是著名的輯里絲的產地，輯里絲曾獲1915年[[巴拿馬國際金獎|巴拿馬國際金獎]]。
