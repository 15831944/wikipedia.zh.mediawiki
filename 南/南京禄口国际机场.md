'''南京禄口国际机场'''（{{airport codes|NKG|ZSNJ}}），位于[[江苏省|江苏省]][[南京市|南京市]][[江宁区_(南京市)|江宁区]][[禄口街道|禄口街道]]，距市中心直线距离约36公里，有一条29公里的机场[[高速公路|高速公路]]通往市区，并和沪宁、宁杭、宁通、宁连等高速公路连接，地铁S1号线于此设站。禄口国际机场是中国主要干线机场之一，华东地区的主要客货运机场之一，与上海虹桥机场、浦东机场互为备降机场，位列全国千万级吞吐量的大型机场行列。南京禄口机场是[[中国东方航空|中国东方航空]]江苏公司的主基地和[[深圳航空|深圳航空]]在华东地区的重点机场。此外，[[中国南方航空|中国南方航空]]、[[厦门航空|厦门航空]]每日在此也有较多航班运营。货运方面，南京禄口机场是[[中国邮政航空|中国邮政航空]]的枢纽机场，每日凌晨有大量货运航班往来全国各主要城市，为[[中国邮政|中国邮政]]的信件和货物提供集中转运服务。

== 简介 ==
禄口机场始建于1995年2月28日，并于1997年7月1日正式通航。2018年禄口机场全年旅客吞吐量2858.1万人次，货邮吞吐量36.5万吨，起降22万架次。<ref name="JS1" >{{cite web|url = http://www.caac.gov.cn/XXGK/XXGK/TJSJ/201903/t20190305_194972.html |title = 2018年民航机场生产统计公报 |publisher =[[中国民用航空局|中国民用航空局]] |date = 2019-03-05 |accessdate = 2018-03-31}}</ref>2013年，机场出入境航班总量为11,515架次，年出入境旅客总数达到178.6万人次。<ref>{{cite news |author = 唐磊 |url = http://www.njdaily.cn/2013/1231/712971.shtml |title = 同比增三成 南京空港2013年出入境流量创新高 |publisher = 南报网 |date = 2013-12-31 |accessdate = 2014-05-29 |archive-url = https://web.archive.org/web/20140529124113/http://www.njdaily.cn/2013/1231/712971.shtml# |archive-date = 2014-05-29 |dead-url = yes }}</ref>

机场于2002年通过[[ISO|ISO]]9001质量体系认证，60多个国内主要城市、20多个国际和港澳台地区城市的130多条航线。<ref>{{cite news |author = 民航资源网 |url = http://data.carnoc.com/corp/airport/nkg.html |title = 南京禄口国际机场 |publisher = 民航资源网 |date = 2013-12-31 |accessdate = 2018-04-08}}</ref>2014年7月12日，随着T2航站楼和第二跑道投入使用，成为华东地区第四家双跑道运行机场<ref name="T2" >{{cite news |author = 民航资源网 |url =http://news.carnoc.com/list/287/287193.html |title = 南京机场12日启用T2 成华东第4家双跑道机场 |publisher = 民航资源网 |date = 2014-07-09 |accessdate = 2014-07-09}}</ref>。

== 历史 ==

南京禄口国际机场是[[江苏省|江苏省]]省会城市[[南京|南京]]的空中门户，由江苏省政府、南京市政府、国家民航总局联合投资，于1995 年2月28日正式开工建设，1997年7月1日正式通航；同年11月18日，经[[中华人民共和国国务院|中华人民共和国国务院]]批准，禄口机场对外国籍飞机开放。

禄口机场二期扩建工程于2008年12月29日开工，2011年4月1日正式开工建设，2014年7月12日正式投运，机场二期建设包括航站区工程（T2航站楼、交通中心、停车场）、第二跑道及滑行道工程、国内货运区工程等三大部分，T1、T2预计合计年3000万客流量、货邮吞吐量100万吨、飞行26万架次。同时，已超负荷运行多年的T1航站楼被暂时关闭整修。<ref>[http://www.njiairport.com/newsDetail.aspx?cid=G3&id=71711 中国民航网：南京禄口机场T1航站楼改扩建工程今日开工]</ref>

2009年机场以年旅客吞吐量1083.72万人次，首次突破1000万人次大关，跻身全国千万级大型机场行列。2013年，机场全年旅客吞吐量突破1500万人次。2016年，禄口机场晋级为4F级机场。<ref>[http://news.carnoc.com/list/348/348550.html 升级啦！南京禄口机场跻身“4F级俱乐部”]</ref>

2016年9月27日起至2017年1月5日，南京禄口国际机场06/24跑道（老跑道）进行了跑道及其配套设备大修，期间所有航班改为07/25跑道（新跑道）起降。<ref>{{cite news|title=南京机场即将跑道大修，你做好准备了吗？|url=http://news.carnoc.com/list/368/368647.html|publisher=民航资源网|date=2016-09-22}}</ref>。

T1航站楼建筑面积13.2万平方米，设有16个近机位，机场货运仓库库内面积6700平方米，库外面积4万平方米；机坪面积44.7万平方米，其中货机坪6.57万平方米。7月12日零时起，随着T2航站楼的正式启用，T1航站楼暂停使用进入维修扩建状态<ref name="T2" />，预计于2020年重新开放<ref>{{cite web|url=http://www.xdkb.net/index/article/2017-12/30/content_1079492.htm|title=南京机场T1航站楼改扩建工程开工，2020年T1、T2“双楼合璧”|publisher=现代快报|date=2017-12-3}}</ref>。

二期扩建工程新建的T2航站楼位于T1航站楼南侧，分为主楼和东西两侧指廊，主楼面宽315米，进深约120米；指廊长约1200米，宽38米，设有32座登机桥，总建筑面积约23.4万平方米，设计旅客吞吐量1800万人次<ref name="T2-c" >{{cite news|url=http://www.jssb.gov.cn/jssq/jjgk/gdtp1/201212/t20121219_150127.html|title=南京禄口国际机场二期扩建|date=2012-12-19|accessdate=2013-11-10}}</ref>。新建停机坪52万平方米，具有55个机位，其中T2航站楼近机位35个，T2航站楼东侧布置远机位20个<ref name="2nd runway" />。T2航站楼启用后，禄口机场所有国内出港航班旅客办理乘机手续截止时间统一调整为航班起飞前40分钟（国际航空公司有特殊规定的，按照特殊规定执行）<ref name="T2" />。

目前禄口机场有70个登机通道，停机坪可容纳80架飞机，可满足包括空客A380在内各类飞机起降。<ref>[http://www.njiairport.com/upload/2015/20151204001.docl 南京禄口国际机场简介]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

=== 跑道 ===
机场一期工程建有一条长3600米、宽60米的4E级跑道和一条长3600米宽45米的滑行道，满足世界上各类运输机全重起降。

二期扩建工程在航站楼南侧，现有跑道以南2000米处建有，与现有跑道平行，长3600米、宽60米的4F级第二跑道，同时还新建2条平行滑行道；在既有垂直联络道西侧，增建两条垂直联络道，道面各宽25米和23米，连接南、北两个飞行区<ref name="2nd runway">{{cite news|url=http://www.njiairport.com/list/de_d2_WCA1_detail__188fb2662faa4fff831888ab765cc982.aspx|title=禄口机场新建飞行区开始校飞|date=2014-04-01|accessdate=2014-05-17|deadurl=yes|archiveurl=https://web.archive.org/web/20140517123809/http://www.njiairport.com/list/de_d2_WCA1_detail__188fb2662faa4fff831888ab765cc982.aspx|archivedate=2014-05-17}}</ref>。
=== 交通中心 ===
二期扩建工程新建的交通中心包括宁高城际轨道交通S1号线车站、长途汽车站、配套餐饮和购物设施，以及一座有三百间客房的五星级帕尔曼酒店（Pullman Hotel）<ref> {{Cite book| title = 南京禄口机场二期主体工程完成近80%| date = 2012-11-26| accessdate = 2012-12-02| url = http://www.caacnews.com.cn/newsshow.aspx?idnews=236351| deadurl = yes| archiveurl = https://web.archive.org/web/20131203042511/http://www.caacnews.com.cn/newsshow.aspx?idnews=236351| archivedate = 2013-12-03}} </ref>。交通中心前的新停车场可供2400辆车同时停放。

===轨道交通===
[[宁高城际|宁高城际]]轨道交通一期（[[南京地铁S1号线|南京地铁S1号线]]）在T1航站楼与T2航站楼间设[[禄口机场站|禄口机场站]]连接南京禄口机场和[[南京南站|南京南站]]。该线长35.8千米，已于2014年7月1日开通运营。从南京南站到禄口机场全程票价7元，而南京地铁线网最高票价15元，亦即从禄口机场乘地铁到达南京各地铁站票价不超过15元。

===机场巴士===
市区－机场
* 南京南站－机场：5：40－21：00每30分钟一班。班车点设在南京南站的汽车客运站内，前往机场的旅客可至南站汽车客运站内购票乘车。由南京南站至机场约需35分钟。
* 南京火车站东广场－机场：4:30 -- 21:00，每30分钟一班（经停龙蟠中路451号南京机场宾馆）。<ref name>{{cite web|author1=南京禄口国际机场有限公司|title=公 告（机场大巴时刻调整）|url=http://www.njiairport.com/newsDetail.aspx?cid=G4&id=71754|website=njiairport.com|publisher=南京禄口国际机场|accessdate=2018-04-08}}</ref>
* 河西万达广场（万达希尔顿酒店北侧天幕东端）至机场，发车时间为4:30 -- 21:00，每30分钟发一班。
* 江北专线： 江北至机场（025-57671293）
六合（阿尔卡迪亚酒店）04:40、06:30、09:00、11:30、13:30、15:30、17:30、19:30（票价50元）；

===出租车===
机场到达大厅外可以便捷地乘坐出租车。出租车至南京市区大约需要100元人民币（约17美元）。

===高速公路===
机场由空港路连接南京机场高速公路（S88），机场高速与宁宣高速（S55）共线。 宁宣高速向北可连接S337公路往[[马鞍山市|马鞍山]]，[[芜湖|芜湖]]方向；连接G2501南京绕城高速至G42沪宁高速、G40合宁高速、G25宁连高速、G40宁通高速。宁宣高速向南可连接S38沿江高速、G25宁杭高速。
