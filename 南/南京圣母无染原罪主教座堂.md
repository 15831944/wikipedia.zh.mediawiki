'''石鼓路天主堂'''，即'''圣母无染原罪始胎堂'''，位于[[南京市|南京市]][[秦淮区|秦淮区]][[石鼓路|石鼓路]]112号，西邻[[螺蛳转弯|螺蛳转弯]]，是[[天主教南京教区|天主教南京教区]]的[[主教座堂|主教座堂]]。

石鼓路天主堂建成于[[清朝|清]][[同治|同治]]九年（1870年），[[北伐战争|北伐战争]]中该堂遭到严重破坏，1928年由[[国民政府|国民政府]]拨款重修。1930年代以后，这里一直是[[天主教|天主教]][[天主教南京教区|南京教区]]的[[主教座堂|主教座堂]]。1978年后，[[江苏省|江苏省]]及南京市[[中国天主教爱国会|天主教爱国会]]在此合署办公。1982年被列为[[江苏省文物保护单位|江苏省文物保护单位]]，后入选中国著名教堂。

== 前身 ==
石鼓路天主堂是由最早来中国传教的[[意大利|意大利]][[传教士|传教士]][[利玛窦|利玛窦]]创办。16世纪末，利马窦曾先后三次来南京，于1599年在城西罗寺湾购买私宅，略加改造，作为个人进行宗教活动的场所。1618年，由于[[南京教案|南京教案]]的原因，教堂被拆除。

== 重建 ==
1864年，[[法国|法国]]传教士[[雷居迪|雷居迪]]（Adrien de Carrère S.J.，[[史式徽|史式徽]]《[[江南传教史|江南传教史]]》一书中作雷遹駿）随法国的炮舰来南京复堂，在原佛教古刹寺的废墟上建堂，1870年（[[同治|同治]]九年）建成，定名为“圣母无染原罪始胎堂”，即现在的石鼓路天主教堂。[[北伐战争|北伐战争]]中该堂遭到严重破坏，曾一度改作[[马厩|马厩]]。1928年，国民政府拨款15万元重修，现在的教堂基本保存了重修后的风貌。在1966年开始的[[文化大革命|文化大革命]]期间，教堂设施遭受破坏，宗教活动中止。1980年12月25日重新开放。2011年下半年起聖堂進行大修，於2013年2月2日祝聖。

== 建筑特点 ==
外观及内部结构上模仿[[罗曼式建筑|罗曼式]]，无梁、十字形、[[拱顶|拱顶]]、砖木结构、铁皮盖并有钟楼。主要采用了西方古代教堂拱顶结构及光纤色彩的传统手法。由于技术和材料的限制，该堂的屋顶结构仍然采用中国传统的木屋架，下面用曲线的木构件和灰板条模仿罗马风的圆拱作为吊顶，虽是仿作，但几可乱真。
