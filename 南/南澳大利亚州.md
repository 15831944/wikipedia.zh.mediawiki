'''-{zh-hans:南澳大利亚州; zh-hant:南澳大利亞洲}-'''（{{lang-en|'''South Australia'''}}，缩写为{{lang|en|'''SA'''}}），簡稱-{zh-tw:'''南澳''';zh-hk:'''南澳洲'''、'''南澳''';zh-cn:'''南澳洲'''、'''南澳州'''、'''南澳'''}-，位於[[澳大利亚|澳大利亚]]中南部，与澳大利亚大陆的其他五州接壤，是[[澳大利亚联邦|澳大利亚联邦]]的一[[澳大利亞行政區劃|州]]，其下劃分為69個[[地方政府區域|地方政府區域]]。

南澳大利亚南濱[[印度洋|印度洋]]，轄境总面积约1,043,514平方公里，为澳大利亚大陆总面积的1/8、六州中的第三大州（不包括[[北領地|北領地]]），其中50%以上是[[牧地|牧地]]；包括[[坎加鲁岛|坎加鲁岛]]（袋鼠岛）和數个较小的岛屿。州內有6.5%的土地成立为[[国家公园|国家公园]]和[[野生动植物保护区|野生动植物保护区]]；人口约1,580,000人（2007年），每平方公-{里}-[[人口密度|人口密度]]约为1.5人，是唯一与[[澳大利亚|澳大利亚]]大陆上所有州都接壤的一[[州|州]]。

南澳主要生产[[初级产品|初级产品]]，如[[小麦|小麦]]、[[羊毛|羊毛]]、[[大麦|大麦]]、[[水果|水果]]、[[葡萄酒|葡萄酒]]、[[牛肉|牛肉]]、[[羊肉|羊肉]]等。

[[府城|府城]][[阿德莱德|阿德雷得]]市[[人口|人口]]116万人，为澳大利亚继[[悉尼|悉尼]]、[[墨尔本|墨尔本]]、[[布里斯本|布里斯本]]、[[珀斯|珀斯]]后的人口第五大[[城市|城市]]，气候为[[地中海型气候|地中海型气候]]，[[夏季|夏季]]干爽，[[冬季|冬季]]短暂多雨。

== 歷史 ==
[[英國|英國]]殖民澳大利亞之初，現在的南澳是[[新南威爾士|新南威爾士]]殖民地的一部分。1834年，[[英國國會|英國國會]]通過《1834年南澳大利亞法令》，允許設立“'''南澳大利亞省'''”。和之前建立的其他殖民地不同，南澳從建立起就沒有流放犯，而是完全由自由移民殖民，殖民計劃由南澳大利亞公司計劃營銷。1836年殖民地宣佈成立，1857年實行民主自治，原來由[[南澳州總督|總督]]任命的[[南澳州立法局|立法局]]改爲選舉產生、並設立[[南澳州眾議局|眾議局]]。1901年南澳大利亞省加入澳大利亞聯邦，成爲南澳大利亞州。

== 教育 ==
主管南澳教育的政府部門為[[教育及兒童培育署|教育及兒童培育署]]。[[高等教育|高等教育]]系统以[[澳大利亚八校联盟|澳大利亚八校联盟]]之一的[[阿德莱德大学|阿德雷得大學]]为主体，另有[[福林德斯大学|福林德斯大学]]及[[南澳大学|南澳大学]]。

== 州徽 ==
{{main|南澳大利亞州徽}}
右圖是澳大利亚君主陛下伊麗莎白二世女王於1984年4月19日御前授與南澳州的州徽。

在南澳州的州徽最上層是西澳特產（the Sturt's desert pea斯特爾特沙漠豌豆），是在南澳大利亞的沙漠中才能見到的花朵，也是代表南澳大利亞地方的州花。

在藍色的盾牌上，有一圈金黃色的光盤，那光盤是代表著在南澳大利亞州地方初升起的朝陽。

在金黃色光盤中的鳥，是代表南澳大利亞州地方的州鳥。有人說是[[伯勞鳥|伯勞鳥]]，也有人說是[[喜鵲|喜鵲]]，南澳州政府說是{{le|笛管伯勞|piping shrike}}，結果；馬上就跳出來一大堆學者反駁說，應該是[[澳洲喜鵲|澳洲喜鵲]]，而鳥專家們也說應該是[[白背鵲|白背鵲]]（Cracticus tibicen telonocua前Gymnorhina tibicen leuconota），所以；澳洲人至今仍很難確定在南澳大利亞州徽和州旗中的鳥，到底是隻什麼鳥。
