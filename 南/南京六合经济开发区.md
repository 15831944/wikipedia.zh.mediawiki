'''{{PAGENAMEBASE}}'''，是{{PRC admin/fullname|32/01/16/400/000|3}}的一个[[开发区|开发区]]。1993年11月，由[[江苏省人民政府|江苏省人民政府]]批准成立。开发区的核准面积10平方公里。2016年时，[[江北新区|江北新区]]规划修编，开发区规划面积扩大至25平方公里<ref>{{Cite web |url=http://www.jslhkfq.com/about.html |title=园区概况|accessdate=2018-09-27 |author= |date= |work= |publisher=南京六合经济开发区网站|language=Zh-hans }}</ref>。辖区以'''六合开发区'''之名列为六合区的一个[[类似乡级单位|类似乡级单位]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|32/01/16/400/000}}}}</ref>。

开发区管委会是由中共六合区委、六合区人民政府的派出机构，负责开发区的行政管理。下设八个部门、五个直属[[事业单位|事业]]（企业）单位<ref>{{Cite web |url=http://jslhkfq.com/jgsz.html |title=机构设置|accessdate=2018-09-27 |author= |date= |work= |publisher=南京六合经济开发区网站|language=Zh-hans }}</ref>。管委会与六合区人民政府办公室同处雄州南路268号六合大厦内<ref>{{Cite web |url=http://www.njlh.gov.cn/www/njlh/qzjd.htm |title=区政府机构设置|accessdate=2018-09-27 |author= |date= |work= |publisher=六合区人民政府网站|language=Zh-hans }}</ref>。
