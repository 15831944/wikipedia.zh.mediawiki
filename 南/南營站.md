'''南营站'''（{{kor|諺=남영역|漢=南營驛|羅=Namyeong-yeok|f=y}}）是位於[[韓國|韓國]][[首爾特別市|首爾特別市]][[龍山區_(首爾)|龍山區]]葛月洞，屬於[[京釜線|京釜線]]的[[電鐵站|電鐵站]]。站名來自[[南營洞|南營洞]]，但其位置卻不位於[[南營洞|南營洞]]，而是相鄰的葛月洞。另外，此站鄰近[[淑大入口站|淑大入口站]]。
== 車站結構 ==
1面2線[[島式月台|島式月台]]，設有[[月台幕門|月台幕門]]。左側6線有[[韓國高速鐵道|KTX]]、[[京釜線|京釜線]]一般列車的通過線。只有1個出口。

== 历史 ==
* 1974年4月19日：新建站舍開工。
* 1974年6月30日：新建站舍竣工。
* 1974年8月15日：[[首都圈電鐵1號線|首都圈電鐵1號線]]開通的同時開始營業。
* 1983年8月20日：升格為運轉簡易站。
* 2009年12月1日：鐵道乘車券終端機拆去。
* 2014年：完成設置[[月台幕門|月台幕門]]。

== 站名由來 ==
[[光復節_(韓國)|光復]]後制定洞名時，因為位於首都以南的軍營而命名為[[南營洞|南營洞]]，在此處興建的車站亦命名為南營站。
