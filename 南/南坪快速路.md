'''南坪快速路'''为[[深圳市|深圳市]]一条东西走向的道路，粤高速编号为'''S301'''<ref>{{cite web |url=http://www.gdcx.gov.cn/web/gov/JourneyResource/HighWayNumChange.aspx |title=存档副本 |accessdate=2013-03-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20130225101221/http://www.gdcx.gov.cn/web/gov/JourneyResource/HighWayNumChange.aspx |archivedate=2013-02-25 }} 广东省高速公路网命名与编号调整</ref>，设计时速主线80公里/小时，支线60公里/小时，双向6-8车道，公路技术等级为[[中华人民共和国公路等级|高速公路]]。

道路工程分三期。一期自布吉水径龙景立交至龙珠大道，二期从龙珠大道至南山前海，三期自[[水官高速|水官高速]]横坪出口至坪山。一，二期与三期工程之间靠水官高速相连。二期A段同乐立交至南头立交段处于缓慢施工中，B段则完全停工，“烂尾”至今。

==一期==
一期工程始於[[深圳市|深圳市]][[南山區|南山區]]塘朗，至[[龍崗區|龍崗區]][[布吉街道|布吉街道]]龍景立交，经[[水官高速|水官高速]]往[[龍崗區|龍崗區]][[龍翔大道|龍翔大道]]连接，全長19.05公里，於2006年6月30日通車。

==二期==
二期工程始於南坪快速路一期的塘朗立交橋，經沙河西路、南頭、[[前海深港现代服务业合作区|前海深港现代服务业合作区]]至[[興海大道|興海大道]]，全長15.4公里，當中由塘朗至南頭，全長11.178公里的南坪快速路（二期）A段工程（1－4标段）於2012年7月31日通車<ref>[http://news.sz.soufun.com/2012-07-28/8196582.htm 房天下網 南坪快速二期 塘朗立交至南光高速段31日通车]</ref>，二期A段南头立交至同乐立交及B段，截至2018年8月仍未建成通车，工期延误原因分别是由于A段5标段（中山园段）拆遷賠償无法达成一致，中山园段需改线沿[[平南鐵路|平南铁路]]正上方高架，按规定只能在没有列车通过的铁路天窗期内施工；B段则因[[前海深港现代服务业合作区|前海深港现代服务业合作区]]获批，前海区域规划变更，道路设计改为地下；A段6标段（南头立交）已经建成，因为5标段未建成而与其余路段无法联通，现处于闲置中。<ref>[http://www.sznews.com/news/content/2014-10/29/content_10607116.htm 快速路二期B段通车 兴海大道可直上广深沿江高速 南坪快速路二期B段通车 兴海大道可直上广深沿江高速]</ref>广东省人民政府办公厅回应查询时指2018年年底可通车。<ref>{{Cite web|url=http://bbs.szhome.com/30-10001-detail-176209512.html|title=南坪快速二期B段（中山园段）预计2018年底开通！_家在宝安 - 家在深圳|accessdate=2018-08-23|work=bbs.szhome.com}}</ref>

==三期==
三期工程由[[水官高速|水官高速]]橫坪立交向東伸延，由[[水官高速|水官高速]]橫坪立交起，經碧嶺工業區至坪山新區聚龍路（規劃外環高速公路田頭立交），全長22.2公里，其中從水官高速至碧嶺工業區為雙向六車道，長約8.5公里，部分路段利用橫坪公路改造，而碧嶺工業區至聚龍路為雙向八車道，長約13.7公里，為新建路段。全段共設置15座互通立交。水官高速横坪立交至碧嶺工業區段已随新横坪公路于2018年1月25日亮灯，碧岭至龙坪路立交段于2019年1月30日通车，全线预计2019年底完工通车。<ref>[http://wb.sznews.com/html/2014-01/21/content_2760426.htm 深圳晚報 2014年1月21日 南坪快速三期今年开工相接大亚湾] {{webarchive|url=https://web.archive.org/web/20141103132110/http://wb.sznews.com/html/2014-01/21/content_2760426.htm |date=2014年11月3日 }}</ref><ref>[http://www.sztb.gov.cn/jtzx/tpxw/201401/t20140121_37991.htm 深圳市交通運輸委員會 2014年1月21日 南坪快速路三期或年内开工] {{webarchive|url=https://web.archive.org/web/20140402204005/http://www.sztb.gov.cn/jtzx/tpxw/201401/t20140121_37991.htm |date=2014年4月2日 }}</ref>

== 设计问题 ==
南坪快速，尤其在西丽塘朗附近的道路平纵线形设计及数起事故，引起了批评和争议。一期通车后南坪快速西行常常在长岭皮水库大桥至塘朗隧道入口路段发生拥堵，有货车司机认为是因为长下坡车速较快，导致追尾事故易发和需要三四小时才能通过相关路段。连接塘朗隧道入口的高架桥是一个半径相当小的急弯，不仅由主线4车道减为2车道，限速更低至30km/h。此弯道已多次发生车辆倾覆甚至掉落桥下的事故，在部分驾驶者口中享有“死亡高架桥”的恶名。政府有关部门回应指塘朗隧道属于支线，计划拓宽至3车道，事后亦调整和增加沿途的减速带和标志。<ref>{{Cite news|url=http://news.sznews.com/content/2009-11/27/content_4213560_2.htm|title=南坪快速并不“快”司机很无奈|author=欧东勇|date=2009-11-27|work=|publisher=深圳新闻网|newspaper=晶报|accessdate=2018-08-24|editor-last=李|editor-first=晓蕾}}</ref><ref>{{Cite web|url=http://sz.bendibao.com/news/20091127/155205.htm|title=南坪快速为何不快?记者体验一下午5起事故- 深圳本地宝|accessdate=2018-08-23|author=|date=|work=sz.bendibao.com|publisher=本地宝}}</ref><ref>{{Cite news|url=http://www.sznews.com/news/content/2007-11/07/content_1632355.htm|title=大货车翻落桥底下 轮胎飞出百米外|author=李博|date=2007-11-06|work=|publisher=深圳新闻网|newspaper=深圳商报|accessdate=2018-08-24|editor-last=小月|editor-first=}}</ref>
