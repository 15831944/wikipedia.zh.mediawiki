'''南浦特別市'''（{{kor|諺=남포특별시|漢=南浦特別市|羅=Nampho Thŭkpyŏlsi|f=y}}）是[[朝鮮|朝鮮]]的一个[[特別市|特別市]]，是平壤重要的貿易港口、工業都市。人口366,815（2008年人口普查）。

== 地理環境 ==
'''南浦'''是一個港灣都市，距離[[大同江|大同江]]河口約39[[公里|公里]]。北面為[[平壤直轄市|平壤直轄市]]，南面為[[黃海北道|黃海北道]]。
[[File:125.39255E_38.73470N_Port_of_Nampho.png|thumb]]

== 歷史 ==
'''南浦'''在[[朝鮮王朝|朝鮮王朝]]時代只是一個小漁村，1894年的[[甲午戰爭|甲午戰爭]]成為[[日本|日本]]軍的後勤基地。1908年至1945年改名為鎮南浦、歸屬[[平安南道|平安南道]]。

== 工業 ==
從殖民地時代開始，南浦市已經是朝鮮北部最大的貿易港口，貿易總額僅次於[[釜山|釜山]]、[[仁川|仁川]]，居全國第三位。各種冶煉廠、玻璃廠、造船廠、化學工業等工業相繼發展起來，而[[漁業|漁業]]亦十分興旺。現在南浦市是全國[[玻璃|玻璃]]、[[有色金屬|有色金屬]]的冶煉中心。 

和[[韓國|韓國]]的合資的企業「[[平和自動車|平和自動車]]」有[[汽車|汽車]]生產。

== 交通 ==
作為[[平壤市|平壤市]]的海上門戶，南浦市的[[海運|海運]]、[[河運|河運]]、[[鐵路|鐵路]]、[[公路|公路]]十分發達，是朝鲜的對外貿易中心。

== 關於南浦直轄市 ==
* 1980年，[[平安南道|平安南道]]的南浦市、大安市、龍岡郡（當時龍岡郡的全境）合併為獨立於平安南道的南浦直轄市。
* 1991年，浦市面積大約為1000[[平方公里|平方公里]]，人口約807000。

由於朝鮮於2004年1月9日重編國家行政區劃，新增了「[[特級市|特級市]]」（隸屬於道），南浦市改制为特级市。2010年，南浦市升格为特別市建制，并将江西、大安、温泉、龙岗、千里马五郡劃入。合计1市5郡总人口为98万3660（2008年人口普查值）。
