'''南條SA'''(平假名：なんじょうサービスエリア）位於[[福井縣|福井縣]][[南條郡|南條郡]][[南越前町|南越前町]]的[[北陸自動車道|北陸自動車道]][[服务区_(公路)|服務區]]。由[[中日本高速公路|中日本高速公路]]管理。此服務區設有智能IC。

== 連接道路 ==
* [[北陸自動車道|北陸自動車道]]

== 歷史 ==
* 1977年12月8日 - [[北陸自動車道|北陸自動車道]][[敦賀IC|敦賀IC]]至[[武生IC|武生IC]]開通，此服務區也同時啟用。
* 2005年12月17日 - 試用智能IC。
* 2009年4月1日 - 正式啟用智能IC。
