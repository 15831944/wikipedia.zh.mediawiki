== 外觀 ==
有三根犄角，胸角向前，微彎成弧形；頭角末端向上，中間有片狀凸起，[[翅鞘|翅鞘]]是青銅色，有金屬光澤。雄蟲體長45~108mm，雌蟲體長41~64mm。

== 分布 ==
南洋大兜蟲分布於[[東南亞|東南亞]]的熱帶雨林中，包括[[印度尼西亚|印尼]]、[[菲律宾|菲律賓]]、[[印度|印度]]和[[中南半島|中南半島]]等。

== 習性 ==
棲襲於低海拔的原始雨林，幼蟲期10~16個月，成蟲期約2~4個月，性格暴躁兇猛。

== 亞種 ==

* ''Chalcosoma atlas atlas''  分布於[[苏拉威西岛|蘇拉威西]]
* ''Chalcosoma atlas butonensis''  分布於[[布顿岛|布頓島]]
* ''Chalcosoma atlas keyboh'' 分布於[[馬來半島|馬來半島]]、[[蘇門答臘|蘇門答臘]]和[[婆罗洲|婆羅洲]]等
* ''Chalcosoma atlas mantetsu'' 分布於[[緬甸|緬甸]]、[[泰國|泰國]]、[[寮國|寮國]]和[[印度|印度]]等
* ''Chalcosoma atlas simeuluensis''  分布於[[錫默盧島|錫默盧島]]
* ''Chalcosoma atlas sintae'' 分布於[[珀倫島|珀倫島]]
* '''亞特拉斯南洋大兜蟲''' ''Chalcosoma atlas hesperus'' 分布於[[菲律賓|菲律賓]]
