'''南新里'''是[[中華民國|中華民國]][[臺灣|臺灣]][[嘉義縣|嘉義縣]][[太保市|太保市]]轄下的行政區位在太保市的東邊，東邊是太保市[[竹村里|竹村里]]、北邊有[[民雄鄉|民雄鄉]]與[[新港鄉|新港鄉]]而西邊是太保市[[麻寮里|麻寮里]]、南鄰太保市[[埤鄉里|埤鄉里]]。面積約2.259889方公里，行政區包含17個鄰、699戶、數1755人。水牛公園與牛將軍廟為該地景點。

== 地理 ==
南新里位在太保市的東邊，東邊是竹村里、北邊有民雄鄉與新港鄉而西邊是麻寮里、南邊埤鄉里，地勢平坦，屬於鄉村都市化的村落。<ref name=":0">{{Cite web|url=http://gn.moi.gov.tw/geonames/index.aspx|title=地名資訊服務網|accessdate=2017-12-17|author=|date=|publisher=}}</ref>

== 聚落歷史 ==
根據資料調查南新與[[北新里|北新里]]出原本為北新庄後改名為[[水牛厝|水牛厝]]。日治時期，實行[[保甲制度|保甲制度]]水牛屬於[[水虞厝|水虞厝]]所屬保之內。國民政府時期，將水虞厝一、二保，分成南新、北新。而南新是因相對北新而取的。<ref>{{Cite book|title=太保市志|last=|first=|publisher=|year=|isbn=|location=|pages=}}</ref>

== 教育 ==
南新里有一座教育設施­­‑[[南新國小|南新國小]]，建立西元1914年一開始名為嘉義公學校水虞厝分教場，後來有多次更名，最後一次西元1967年改為南新國民小學，在埤鄉、麻寮有兩分班。此外，南新里、[[麻寮里|麻寮里]]、[[過溝里|過溝里]]、埤鄉里….等都在學區中。<ref name=":0" />

== 文化景點 ==
南新里有兩座宗教設施，[[恩主宮廟|恩主宮廟]]與[[水牛將軍廟|水牛將軍廟]]。恩主宮廟為[[道教|道教]]廟宇，神像原來供奉於玉泉時濟善堂，後來由一位許姓堂主在家祭拜，後來在西元1955年於後[[潭濟善堂|潭濟善堂]]籌建恩主宮廟。另一座牛將軍廟，有台灣唯一供奉的牛將軍，在總統[[蔣經國|蔣經國]]參訪後擴建，位於南新里水牛公園內。水牛公園位於中山路，特色是以牛車車輪做成欄杆的裝置藝術與水牛將軍廟。<ref name=":0" /><ref>{{Cite web|url=http://outdoor.cyc.edu.tw/outdoor/taibao/b1.htm|title=太保市牛將軍廟|accessdate=2017-12-17|author=|date=|publisher=}}</ref>
