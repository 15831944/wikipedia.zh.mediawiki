'''南崙村'''是[[新港鄉|新港鄉]]最北端崙仔村南邊的村落，是[[嘉義縣|嘉義縣]]新港鄉轄下的行政區。轄區面積約3.471096km2，行政區包含13鄰、計369戶、轄區人口計1115人，為具有觀光藝術特色的聚落。

== 地理 ==
南崙村位於新港北端，過去曾與[[北崙村|北崙村]]合稱為崙仔村。崙仔村地名的由來，是相傳[[北港溪|北港溪]]中的泥沙受到北風吹襲堆積成山丘稱為崙仔因而得名<ref>{{Cite web|url=http://gn.moi.gov.tw/geonames/index.aspx|title=地名資訊服務網|accessdate=2017-07-05|author=|date=|work=地名資訊服務網|publisher=}}</ref>。

== 聚落歷史 ==
南崙村舊地名包括「廍仔」與曾有過[[糖廍|糖廍]]製糖歷史有關<ref>{{Cite web|url=http://gn.moi.gov.tw/geonames/index.aspx|title=地名資訊服務網|accessdate=2017-07-05|author=|date=|work=地名資訊服務網|publisher=}}</ref>。

== 景點 ==
南崙彩繪村：在2014年11月至今由南崙村民自主彩繪創造的街區景觀<ref>{{Cite web|url=https://guide.easytravel.com.tw/scenic/80185|title=南崙彩繪村|accessdate=2017-07-05|author=|date=|work=台灣旅遊景點|publisher=}}</ref>。

== 教育 ==
由於南崙村並沒有設置學校，所以當地學生多到北崙復興國小就讀。復興國民小學創校於民國五十八年八月一日，前身是古民國民學校崙子分校，是新港鄉六所國小中，最晚成立的， 有四十多年的歷史<ref>{{Cite web|url=http://www.fsps.cyc.edu.tw/modules/tadnews/page.php?nsn=5|title=復興國小全球資訊網 - 沿革|accessdate=2017-07-05|author=|date=|last=|work=復興國小全球資訊網|publisher=|language=zh-TW}}</ref>。

== 文化資源 ==
崙仔代天宮：位於南崙村98號，創立於同治已丑四年，主祀神明為池府千歲，合祀玄天上帝、法主公<ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/shingang/1007021-DTG|title=崙仔代天宮 — 文化資源地理資訊系統|accessdate=2017-07-05|author=|date=|work=文化資源地理資訊系統|publisher=|language=zh-tw}}</ref>。
