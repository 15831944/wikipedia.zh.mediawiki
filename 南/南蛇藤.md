'''南蛇藤'''（[[学名|学名]]：{{lang|la|''Celastrus orbiculatus''}}）为[[卫矛科|卫矛科]][[南蛇藤属|南蛇藤属]]的植物。

== 形态 ==
藤状[[灌木|灌木]]；近圆形或宽椭圆形的叶子互生，有锯齿；夏季开花，腋生或顶生聚伞花序；近球形的[[蒴果|蒴果]]裂开；种子有橙红色假种皮。

== 分布 ==
分布在[[朝鲜|朝鲜]]、[[日本|日本]]以及[[中国大陆|中国大陆]]的[[山东|山东]]、[[江苏|江苏]]、[[四川|四川]]、[[内蒙古|内蒙古]]、[[湖北|湖北]]、[[辽宁|辽宁]]、[[安徽|安徽]]、[[浙江|浙江]]、[[河北|河北]]、[[陕西|陕西]]、[[山西|山西]]、[[江西|江西]]、[[甘肃|甘肃]]、[[黑龙江|黑龙江]]、[[吉林|吉林]]、[[河南|河南]]等地，生长于海拔450米至2,200米的地区，一般生长在山坡灌丛中，目前尚未由人工引种栽培。

== 别名 ==
蔓性落霜红（苏南） 南蛇风、大南蛇、香龙草（湖南长沙） 果山藤（湖南）

== 异名 ==
* Celastrus orbiculatus Thunb. var. jeholensis (Nakai) Kitag.
