'''南雅加达行政市'''（{{lang-id|Kota Administrasi Jakarta Selatan}}），当地华人常称之为'''雅加达南区'''或'''雅京南区'''，是[[印度尼西亚|印度尼西亚]][[雅加达|雅加达特区]]下属五个{{link-id|行政市 (印尼)|Kota administrasi|行政市}}之一，辖区位于雅加达特区的西南部，2010年统计共有2,057,080人，是雅加达人口第三多的行政市。行政中心位于{{le|新巴油兰|Kebayoran Baru|新巴油兰区}}。

南雅加达行政市东接[[东雅加达行政市|东雅加达行政市]]，南邻[[西爪哇省|西爪哇省]]的[[德波|德波市]]，西靠[[万丹省|万丹省]]的[[坦格朗|坦格朗市]]，北依雅京的[[西雅加达行政市|西雅加达行政市]]。<ref>{{cite map |publisher=DesignMap |title=Peta Administrasi Kota Jakarta Selatan Provinsi DKI Jakarta |trans_title=雅加达首都特区南雅加达市行政区划图 |map= |mapurl= |date= |year=2013 |url=https://petatematikindo.files.wordpress.com/2013/06/administrasi-jakarta-selatan.jpg |scale=1:65,000 |series= |version= |cartography= |page= |pages= |section= |inset= |edition= |location= |language=id |format= |isbn= |id= |ref= |accessdate= |archiveurl= |archivedate= }}</ref>

[[雅加达地铁|雅加达公共交通捷运系统]]南北线的起点位于南雅加达的勒巴克布鲁斯（Lebak Bulus），2017年8月开始安装轨道，预计2019年3月运行<ref>{{cite web |url=http://www.shangbaoindonesia.com/indonesia-berita/%E9%A2%84%E8%AE%A12019%E5%B9%B43%E6%9C%88%E8%BF%90%E8%A1%8C%E9%9B%85%E4%BA%AC%E5%9F%8E%E5%B8%82%E5%85%AC%E5%85%B1%E6%8D%B7%E8%BF%90%E4%BA%A4%E9%80%9A%E5%88%97%E8%BD%A6%E8%BD%A8%E9%81%93%E5%BC%80.html |title=预计2019年3月运行 雅京城市公共捷运交通列车轨道开始安装 |last= |first= |date=2017-08-06 |website=印度尼西亚商报 |publisher=雅加达点滴新闻网 |access-date=2017-11-09 |quote=}}</ref>。

2017年6月30日，“伊斯兰国”组织的两名追随者在南雅加达的一座清真寺内袭击两名警察，劝降无效后被击毙<ref>{{cite web |url=http://www.shangbaoindonesia.com/indonesia-berita/%E8%A2%AD%E5%87%BB2%E5%90%8D%E8%AD%A6%E5%91%98%E7%9A%84%E6%81%90%E6%80%96%E5%88%86%E5%AD%90%E5%AB%8C%E7%8A%AF%E5%B7%B2%E8%A2%AB%E8%AD%A6%E6%96%B9%E5%87%BB%E6%AF%99%E8%AD%A6%E6%96%B9%E7%A7%B0%E9%9B%85.html |title=袭击2名警员的恐怖分子嫌犯已被警方击毙 警方称雅京和棉兰的袭击警察事件有类似动机 |last= |first= |date=2017-07-03 |website=印度尼西亚商报 |publisher=雅加达安达拉新闻网 |access-date=2017-11-09 |quote=}}</ref>。2017年7月4日，一名不明身份男子把“[[伊斯兰国|伊斯兰国]]”（ISIS）的旗帜挂在南雅加达{{le|旧巴油兰|Kebayoran Lama|旧巴油兰区}}警察局外侧围栏上，并扬言要将雅加达变成像[[菲律宾|菲律宾]]南部城市[[马拉维_(菲律宾)|马拉维]]一样的战场<ref>{{cite web |url=http://www.shangbaoindonesia.com/indonesia-berita/%E5%8D%B0%E5%B0%BC%E9%9B%85%E5%8A%A0%E8%BE%BE%E8%AD%A6%E5%B1%80%E9%81%AD%E5%88%B0is%E6%81%90%E5%90%93%E8%A6%81%E5%B0%86%E5%8D%B0%E9%A6%96%E9%83%BD%E5%8F%98%E6%88%90%E6%88%98%E5%9C%BA.html |title=印尼雅加达警局遭到IS恐吓 要将印首都变成战场 |last= |first= |date=2017-0-04 |website=印度尼西亚商报 |publisher=中国新闻网 |access-date=2017-11-09 |quote=}}</ref>。
