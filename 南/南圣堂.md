'''南圣堂'''位于[[中国|中国]][[浙江省|浙江省]][[湖州市|湖州市]][[德清县|德清县]][[新市镇_(德清县)|新市镇]]南栅驾仙桥南、南昌街西侧，坐西向东，面向市河，为当地运河渔民祭祀鱼神、蚕神、农神的场所，为京杭大运河浙江段仅存的渔民祭祀建筑<ref>{{cite web |url=http://dqnews.zjol.com.cn/dqnews/system/2015/05/14/019328905.shtml|title=《航面廊谜语一样的老地名》|accessdate=2016-03-18|work=|publisher=德清新闻网}}</ref>。该建筑始建于清咸丰元年（1851），民国十三年（1924）重修<ref>{{cite web |url=http://dqnews.zjol.com.cn/dqnews/system/2009/01/14/000912185.shtml|title=《南圣堂 历史有点久》|accessdate=2016-03-29|work=朱宇飞|publisher=德清新闻网}}</ref>，现面阔一间，硬山顶，南北均与民居相邻。
