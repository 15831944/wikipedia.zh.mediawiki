'''南京市博物总馆'''，位于[[江苏省|江苏省]][[南京市|南京市]]，是综合性历史艺术类博物馆。

==简介==
2014年2月27日，南京文化遗产保护工作会议在南京市级机关大礼堂举行，会上[[杨卫泽|杨卫泽]]、[[缪瑞林|缪瑞林]]分别为南京市博物总馆、南京城墙保护管理中心授牌<ref>{{Cite web |url =  http://news.sina.com.cn/c/2014-02-28/064029584811.shtml|title =寸土不让守护文化积淀  |author =  |date =2014-02-28  |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.huaxia.com/zhwh/whbh/2014/02/3769320.html?ejnc5  |title = 南京将全面启动明城墙申遗 实现明城墙开放利用 |author =  |date = 2014-02-28  |publisher =华夏经纬网  |language =  |accessdate =  |quote =  }}</ref>。

南京市博物总馆隶属南京市文化广电新闻出版局，由[[南京市博物馆|南京市博物馆]]、[[太平天国历史博物馆|太平天国历史博物馆]]、[[中国共产党代表团梅园新村纪念馆|中国共产党代表团梅园新村纪念馆]]、[[南京市民俗博物馆|南京市民俗博物馆]]、[[渡江胜利纪念馆|渡江胜利纪念馆]]、[[江宁织造博物馆|江宁织造博物馆]]、[[六朝博物馆|六朝博物馆]]、南京市考古研究所、南京市文化遗产保护研究所共9家南京市文化广电新闻出版局所属的文博馆所组成<ref name=yiji>{{Cite web|url = http://www.jscnt.gov.cn/whzx/jdxw/201701/t20170124_45932.html|title = 南京市博物总馆晋级国家一级馆|author = |date = 2017-01-24|publisher = 江苏省文化厅|language = |accessdate = |quote = }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2017年，南京市博物总馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref><ref name=yiji/>。
