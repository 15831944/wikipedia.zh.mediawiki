'''南港展覽館站'''位於[[台灣|台灣]][[台北市|台北市]][[南港區|南港區]]，並極接近[[新北市|新北市]][[汐止區|汐止區]]界（僅250公尺），為[[臺北捷運|台北捷運]][[文湖線|文湖線]]與[[板南線|板南線]]僅二交會的[[鐵路車站|捷運車站]]（另一為[[忠孝復興站|忠孝復興站]]），也是兩條路線的端點站。兩線站體均設有[[全高式月台門|全高式月台門]]，另外，本站內湖線月台與西湖站月台為文湖線僅二的高架[[島式月台|島式月台]]，文湖線其餘高架車站皆為[[側式月台|側式月台]]。其中的出口1無縫連接[[南港展覽館|南港展覽館]]。

南港展覽館站目前是台北捷運地理位置最東的車站與最東的轉乘車站，同時也是台北捷運系統唯一雙路線端點車站。站址緊鄰[[新北市|新北市]][[汐止區|汐止區]]，[[內湖機廠|內湖機廠]]即位於本站內湖線站體之東北側。

2017年3月宣布的「[[基隆輕軌|基隆輕軌]]」新計畫，採用[[Tram-train|Tram-train]]（電車—列車系統），規劃本站為路線終點，新建平面車站，全線共19公里，預定於2022年通車營運<ref>[http://news.ltn.com.tw/news/politics/breakingnews/2007566. 獨家》蔡英文宣布 將建南港到基隆線輕軌]</ref><ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/2007893|title=基隆輕軌採用Tram-train系統 行駛路線圖曝光|last=鄭|first=瑋奇|date=2017-03-17|work=自由時報電子報|accessdate=2017-03-19}}</ref>。

== 車站概要 ==
*車站位於南港路兩側，與研究院路、[[經貿二路|經貿二路]]的交叉口以及[[經貿一路|經貿一路]]口間。
*車站代碼為'''BR24／BL23'''。
*車站名稱取自位於北邊的台北世界貿易中心南港展覽館。
*南港展覽館站於規劃時期稱為'''經貿南站'''，而此站北方的[[南港軟體園區站|南港軟體園區站]]之前則稱為經貿北站。
*本站為[[文湖線|文湖線]]高架之終點站，通車營運於2009年7月4日，營運單位為[[臺北捷運公司|臺北捷運公司]]。
*本站為[[板南線|板南線]]地下之終點站，通車營運於2011年2月27日，營運單位為[[臺北捷運公司|臺北捷運公司]]。
*本站為[[基隆輕軌|基隆輕軌]]規劃之終點站，預定設在平面，並於2022年通車營運，營運單位為[[臺灣鐵路管理局|臺灣鐵路管理局]]<ref>[http://news.ltn.com.tw/news/politics/breakingnews/2007566. 獨家》蔡英文宣布 將建南港到基隆線輕軌]</ref><ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/2007893|title=基隆輕軌採用Tram-train系統 行駛路線圖曝光|last=鄭|first=瑋奇|date=2017-03-17|work=自由時報電子報|accessdate=2017-03-19}}</ref>，[[基隆輕軌|基隆輕軌]]延伸至[[南港車站|南港站]]已經納入方案，[[基隆輕軌|基隆輕軌]]是否延伸到[[南港車站|南港站]]<ref>[https://tw.news.yahoo.com/%E5%9F%BA%E9%9A%86%E8%BC%95%E8%BB%8C%E6%8D%B7%E9%81%8B%E8%B7%AF%E7%B7%9A%E7%A4%BA%E6%84%8F%E5%9C%96%E8%8D%89%E6%A1%88-%E5%9C%96-082702798.html 基隆輕軌捷運路線示意圖草案 (圖)]。</ref><ref>[https://tw.appledaily.com/headline/daily/20170809/37742872  基隆輕軌延伸南港站 納入方案]。</ref>。

== 車站構造 ==
本站為[[台北捷運|台北捷運]]路網東側最重要的轉運站，因此本站為中運量設計之高架與高運量設計之地下交會車站，以穿越南港路的地下通道直接連結。[[臺鐵|臺鐵]]及[[臺灣高鐵|高鐵]]隧道與地下車站共構，但未設站。內湖線及南港線共有七個出入口。

== 利用狀況 ==
站外的南港路設有「捷運南港展覽館站」站牌，可轉乘前往市中心與[[汐止區|汐止]]、[[基隆市|基隆]]等地，因此成為台北市以東乘客進城的重要轉運站，在台北捷運路網東端是運量較高的車站。

由於本站為端點站，棕線及藍線的深夜末班車同時於24:00發出，此後到站的列車均無法轉乘另一線，目前已新增廣播(本列車抵達終點站南港展覽館站時，已無文湖線列車，往文湖線旅客請在忠孝復興站轉乘)。
本站在2015年8月7日前假日是不開放自行車進出，2015年8月8日起可於假日開放自行車進出本站（僅限板南線車站）。

本站於內湖線月台東端設有直達地下一樓的[[電扶梯|電扶梯]]以便利內湖線乘客前往南港線，然而僅有向下單向一座。

2009年之旅次統計日均人次以7月4日通車算起至年底計181天。

== 公共藝術 ==
[[File:TNEC_S0001.JPG|thumb]]
*「空中之河」為本站的裝置藝術創作品，以不鏽鋼及彩繪壓克力玻璃製作。位於月臺層中央天窗下。作者為[[賴純純|賴純純]]。

*「光隙」是指南港展覽館站西側廣場圓柱，作者是莊普。

*本站於動工時原稱為'''經貿南站'''（英文名Nangang Business Park South），後改為今名。<ref>[http://www2.dorts.gov.tw/news/newsletter/ns202/rp202.htm 《捷運報導》第202期] {{Wayback|url=http://www2.dorts.gov.tw/news/newsletter/ns202/rp202.htm |date=20110403015648 }} .臺北市政府捷運工程局 .2004年12月</ref>
*2009年2月22日：內湖線南港展覽館站實質完工。
*2009年7月4日：隨著內湖線正式通車而啟用，並同時啟用南港線南港站至本站間免費[[接駁公車|接駁公車]]（由[[指南客運|指南客運]]行駛），以期減低[[忠孝復興站|忠孝復興站]]的運量負擔。
*2011年2月20日：南港線營業列車全面駛入本站南港線月台調頭，但仍會在[[南港車站|南港站]]預先清車。
*2011年2月27日：下午兩點南港線月台隨著南港線「南港展覽館—[[南港車站|南港]]」段正式通車而啟用，並在下午三點停駛兩站之間免費接駁公車<ref name="Nangang Extension" />。
*2015年8月8日：即日起板南線南港展覽館站假日開放自行車進出，唯不得進入文湖線月台。
