'''南掸邦军'''是[[掸邦军|掸邦军]]（SSA）之一，英文简称SSA-S，由[[掸邦|掸邦]]政治组织掸邦复兴委员会（RCSS）领导，总部位于泰缅边境的累泰亮，后来这支部队和[[北掸邦军|北掸邦军]]合并，形成了[[掸邦军|掸邦军]]，致力于[[掸邦独立|掸邦独立]]建国事业。

== 历史沿革 ==
北掸邦军和南掸邦军是由原掸邦进步党领导的掸邦军分化演变而成的两支军队。

南掸邦军的前身是由坤沙领导的蒙泰军，1996年坤沙投降缅甸军政府时，其部下昭耀世拒绝投降后接到密旨，便带领2000余不能屈服的战士渡过萨尔温江来到掸邦中部成立南掸邦军。

2005年，南掸邦军和于1989年缅共人民军解体时从北掸邦军中分裂而自立门户的掸邦民族军和并，并对外宣布成立掸族联邦政府。

2013年11月掸邦政府修改宪法，将当时的从缅甸“独立”出去的路线改为“掸邦自治”。2014年5月21日，南掸邦军（掸邦复兴委员会）领袖召耀世在泰亮山总部举行“掸邦革命53周年纪念日”上正式宣告南掸邦军（掸邦复兴委员会）与北掸邦军（掸邦进步党）合并为“掸邦军”。

== 活动范围 ==
地盘最大。景栋以南，邦弄以东地区。

== 兵力 ==
有正规军50000余人，分编为756旅、757旅、758旅和759旅，共4个旅。民兵数万。有消息称超过十余万军队(兵力已超过佤联军)

== 和佤邦联合军的关系 ==
佤联军数次试图进入南掸邦军控制区，南掸邦军故此还击，发生了若干次战争。
