'''南達科他州州旗'''以[[蔚藍色|蔚藍色]]為底，中央為該州[[南達科他州州徽|州徽]]，其周圍環繞著由許多黃色三角形構成的太陽光芒圖案，上方有「南達科他」（South Dakota）字樣，下方則有該州暱稱「[[拉什莫爾山|拉什莫爾山]]之州」（The Mount Rushmore State）。<ref>{{cite web |url=http://www.state.sd.us/state/sdsym.htm |title=存档副本 |accessdate=2007-01-20 |0=2008-02-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20080220004837/http://www.state.sd.us/state/sdsym.htm |archivedate=2008-02-20 }}</ref>該州於1992年起採用了現今的設計。<ref>http://www.netstate.com/states/symb/flags/sd_flag.htm</ref>

2001年，在一項由[[北美旗幟協會|北美旗幟協會]]進行的調查中，南達科他州州旗被評價為全北美倒數第五差的州旗設計。<ref>http://www.nava.org/Flag%20Design/State&Provincial%20Survey%202001/surveyresults.htm {{webarchive|url=https://web.archive.org/web/20110606033347/http://www.nava.org/Flag%20Design/State%26Provincial%20Survey%202001/surveyresults.htm |date=2011-06-06 }}</ref>該協會認為旗幟是圖像性的而不應包含過多文字，而南達科他州州旗上卻非如此，光是州名「South Dakota」字樣就在此旗幟出現了兩次。<ref>{{cite web |url=http://www.nava.org/Flag%20Design/GFBF/gfbf-10.htm |title=存档副本 |accessdate=2010-09-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20100328064549/http://www.nava.org/Flag%20Design/GFBF/gfbf-10.htm |archivedate=2010-03-28 }}</ref>
