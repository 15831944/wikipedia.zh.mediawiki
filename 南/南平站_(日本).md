'''南平站'''（{{jpn|j=南平駅|hg=みなみだいらえき|rm=Minami-daira eki|f=y}}）是一個位於[[東京都|東京都]][[日野市|日野市]]南平六丁目，屬於[[京王電鐵|京王電鐵]][[京王線|京王線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KO30'''。

== 年表 ==
* 1926年4月28日，作為玉南電氣鐵道的車站啟用。
* 1926年12月27日，玉南電氣鐵道併入京王電氣軌道，南平站成為京王的車站。
* 1944年5月31日，京王與[[東京急行電鐵|東京急行電鐵]]合併後，南平站作為京王線的車站運行。
* 1948年6月1日，東急與京王帝都電鐵分拆，南平站成為京王的車站。
* 2013年2月22日，以'''KO30'''作為南平站的編號。

== 車站構造 ==
本站是[[相對式月台|相對式月台]]2面2線[[地面車站|地面車站]]。剪票口在[[平山城址公園站|平山城址公園]]側。

2010年開始進行車站改良工程。2011年完成跨站式站房，並新設電扶梯與電梯。

廁所位於下行月台。

== 站名由來 ==
過去此地明為'''平村'''，傳聞因是平資綱所有而得名<ref>『{{link-ja|角川日本地名大辭典|角川日本地名大辞典}}』 13 東京都』[[角川書店|角川書店]]、1991年9月再版、P446、P683。</ref>。 因郡內有同名的村（現[[八王子市|八王子市]]平町），以南北做區分而稱'''南平'''。
