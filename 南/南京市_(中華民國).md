'''南京市'''，通稱'''南京'''（[[漢語拼音|漢語拼音]]：Nanjing；[[郵政式拼音|郵政式拼音]]：Nanking），簡稱「'''京'''」，在[[民國紀年|民國]]三十八年（1949年）前為[[中華民國首都|中華民國首都]]<ref>{{cite web|title=國家符號：國都的選定|publisher=中華民國國史館|quote=國都必須因應事實需要不斷搬遷：元年3月，遷都北京；16年4月，奠都南京；26年12月，遷都重慶；35年5月，還都南京；38年4月，遷都廣州；38年10月，遷都重慶；38年12月，遷都臺北。|accessdate=2011-01-05|url=http://book.drnh.gov.tw/Content_Display.aspx?MenuKey=46|deadurl=no|archiveurl=https://web.archive.org/web/20120530122901/http://book.drnh.gov.tw/Content_Display.aspx?MenuKey=46|archivedate=2012-05-30}}</ref>，是[[中華民國|中華民國]]在[[民國紀年|民國]]十六年（1927年）[[國民革命軍北伐|北伐]]至[[民國紀年|民國]]三十八年（1949年）[[第二次國共內戰|政府遷台]]前所設置的12個[[直轄市_(中華民國)|直轄市]]之一，也是當時全國的政治與文化中心。以[[南京城|南京城]]为主体，辖区面积小于今南京市。

== 歷史沿革 ==
[[宣統|宣統]]三年（1911年）爆發[[辛亥革命|辛亥革命]]，動搖[[清朝|滿清]]的君主統治。同年12月2日，江浙聯軍攻破清廷東南重鎮江寧府，使東南數省連成一片，形成與北方清廷分庭抗禮的局面。12月29日，起義的17省代表在江寧府選舉[[孫中山|孫中山]]為[[臨時大總統|臨時大總統]]，國號為[[中華民國|中華民國]]。

[[File:Presidential_Palace_at_Nanjing_Sun's_Office.JPG|200px]]
民國元年（1912年）1月1日，中華民國臨時政府在南京成立，孫中山就任中華民國臨時大總統{{NoteTag|孫中山認為南京是適合建都的城市，他在《[[建國方略|建國方略]]》中這樣評價南京：「南京為中國古都，在北京之前，而其位置乃在一美善之地區。其地有高山，有深水，有平原，此三種天工，鐘毓一處，在世界中之大都市，誠難覓如此佳境也。而又恰居長江下游兩岸最豐富區域之中心，南京將來之發達，未可限量也。」}}，遂改[[江寧府|江寧府]]為'''[[南京府|南京府]]'''，定為中華民國首都。

民國元年（1912年）2月，[[溥儀|溥儀]]在[[顺天府|北京]]宣佈退位；孫中山在南京辭去臨時大總統一職，由[[袁世凱|袁世凱]]繼任。為防止袁世凱擴權，[[中華民國臨時參議院|臨時參議院]]制定《[[中華民國臨時約法|中華民國臨時約法]]》以為規範。不久袁世凱將民國首都遷往北京，南京府改制為[[江寧縣_(古代)|江寧縣]]。

[[北洋軍閥|北洋軍閥]]混戰期間，江寧縣（南京）曾為軍閥[[孫傳芳|孫傳芳]]的大本營，成為號令華東諸省的中心。民國十五年（1926年）冬，[[國民革命軍北伐|國民革命軍北伐]]擊敗孫傳芳部隊。民國十六年（1927年）3月24日，[[國民革命軍|北伐軍]]節節勝利，但進入南京後，發生影響深遠的[[南京事件|南京事件]]，成為[[國共分裂|國共分裂]]的重要前奏之一；不久後[[上海|上海]]就發生[[四一二事件|四一二事件]]。

民國十六年（1927年）4月18日，[[南京國民政府|國民政府]]在南京成立，定南京市為首都，成立'''南京市政廳'''。5月25日，國民政府任命[[劉紀文|劉紀文]]為南京特別市市長<ref>《國民政府公報》寧字第4號，民國16年6月1日，第5頁。</ref>。6月6日，國民政府公佈《[[南京特別市暫行條例|南京特別市暫行條例]]》，共6章、34條。因為國民政府所在地之特別行政區域，故定名為南京特別市<ref>《南京特別市暫行條例》第一條：「本市為中華民國國民政府所在地之特別行政區域，定名為南京特別市。」</ref>。

[[File:Border_of_Nanjing.jpg|thumb]]
民國十八年（1929年）6月1日安葬孫中山於[[中山陵|中山陵]]，置首都特別市。民國十九年（1930年）更改為院轄市（[[直轄市|直轄市]]）。民國二十年（1931年）6月1日公佈的《[[中華民國訓政時期約法|中華民國訓政時期約法]]》，第五條規定「中華民國國都定於南京」，確定南京在法律上做為中華民國首都的地位。

[[File:Nanking_Congress_building.jpg|thumb]]，建成於民國二十五年（1936年）]]
民國十八年（1929年）至二十六年（1937年）間，南京進行大規模的首都建設，奠定南京現代城市發展的良好基礎。民國十八年（1929年），國民政府為了迎接由北京至南京奉安的孫中山靈柩，趁勢在南京進行建設，包括在長江口修建新碼頭（中山碼頭），並將移靈經過的道路進行拓寬與鋪設柏油，形成今天的[[中山北路_(南京)|中山北路]]、[[中山路_(南京)|中山路]]、[[中山東路_(南京)|中山東路]]，並大量種植法國梧桐樹綠化市容。此後數年間，南京陸續建成中正路（中山南路）、太平路、白下路、漢中路、中華路、雨花路、山西路、國府路、熱河路、珠江路、[[莫愁路|莫愁路]]、[[昇州路|昇州路]]、上海路、廣州路及[[頤和路|頤和路]]一帶住宅區道路，形成南京的近代幹道系統。到了[[中國抗日戰爭|中國抗日戰爭]]前夕，南京市人口增加到100萬以上，是當時中國的六大城市之一{{NoteTag|當時中國六大城市為：[[上海市_(中華民國)|上海]]、[[北平|北平]]、天津、廣州、南京、漢口。}}。

民國二十六年（1937年）7月，抗日戰爭全面爆發。8月中旬，南京下游300公里的上海爆發[[淞滬會戰|淞滬會戰]]，南京開始遭到日軍空襲轟炸，市容與城市建設受到嚴重破壞。11月，上海淪陷，國民政府和在京的學校、工廠等陸續西遷。12月8日日軍[[南京保衛戰|進犯南京]]，由於指揮官[[唐生智|唐生智]]臨陣脫逃，國軍在南京的防禦潰散，12月12日南京淪陷。12月13日起，日軍對平民和戰俘展開長達6個星期的大規模屠殺，約30萬人遇難，史稱「[[南京大屠殺|南京大屠殺]]」。南京淪陷後，侵華日軍將南京做為扶植[[傀儡政權|傀儡政權]]的根據地。民國二十九年（1940年），[[汪兆銘|汪兆銘]]在南京成立「[[汪兆銘政權|中華民國國民政府]]」，替日本人統治。

民國三十四年（1945年）8月，日本戰敗投降。9月9日，中國政府代表[[何應欽|何應欽]]作為中國戰區代表在南京接受[[第二次世界大戰中國戰區受降儀式|侵華日軍的無條件投降]]。民國三十五年（1946年）5月5日，國民政府從[[重慶|重慶]]還都南京。由於不久之後國共內戰全面爆發，國府已無充足的財政在這一時期的南京大規模的建設活動，南京不復抗戰前的盛況榮景。

同年11月，[[制憲國民大會|制憲國民大會]]在[[國民大會堂|國民大會堂]]（今南京人民大會堂）召開，原本憲法草案定國都為南京，但審查會及第一讀會決定改為[[北平|北平]]。經大會主席團代表蔣中正出面說明，主張國都地點，不必定於憲法，始將第一讀會通過的「國都定於北平」一條予以刪除<ref>荊知仁：《中國立憲史》，聯經出版公司，1984年12月，461頁.</ref>。12月25日，[[中華民國憲法|中華民國憲法]]三讀通過，民國三十六年（1947年）1月1日公佈，同年12月25日正式實行。自此，南京失去中華民國法定國都的地位。

民國三十六年（1947年）11月21日至23日依憲法規定舉行[[1947年國民大會代表選舉|第一屆中華民國國民大會代表選舉]]，民國三十七年（1948年）1月21日至23日則舉行[[1948年中華民國立法委員選舉|第一屆中華民國立法委員選舉]]；同年3月29日於南京報到召開行憲後的[[國民大會|國民大會]]，4月在南京選舉出[[蔣中正|蔣中正]]為第一屆[[中華民國總統|中華民國總統]]，[[李宗仁|李宗仁]]為第一屆中華民國副總統，5月20日在[[南京總統府|南京總統府]]舉行就職典禮。1月間選出的立法委員則於5月8日自行集會於南京國民大會堂，並於5月18日正式開議。
[[File:President_election.jpg|300px]]
《[[中華民國憲法|中華民國憲法]]》[[行憲|施行]]後，國民政府改組為[[中華民國政府|中華民國政府]]，首都則續設於南京。民國三十七年（1948年）底，[[第二次國共內戰|第二次國共內戰]]情況對國民黨趨於不利的態勢，國軍、解放軍在[[徐州|徐州]]附近一帶進行的[[徐蚌會戰|徐蚌會戰]]，國軍損失將近60萬大軍，解放軍攻佔長江長江中、下游以北的廣大地區。國軍在華東、中原戰場上的主要兵力喪失殆盡，首都南京及金融中心上海處於解放軍的直接威脅之下。民國三十八年（1949年）1月國府[[行政院|行政院]]及各單位開始遷往廣州，僅總統府仍留守南京，4月21日，[[中國共產黨|中國共產黨]][[中国人民解放军|軍隊]]渡過長江，4月23日攻陷南京，中華民國政府南遷[[廣州|廣州]]，後又遷[[重慶|重慶]]、[[成都|成都]]，最後遷抵[[臺北|臺北]]{{NoteTag|[[中華民國政府|中華民國政府]]遷台後，仍長期在官方出版的地圖及教科書上將南京市記為首都<ref>中華民國中等學校歷史教科書. 1996年出版. 國立編譯館.</ref>。自2002年起，則不再將南京市標記為首都迄今<ref>《國民中學社會》第3冊，南一書局，2008年8月，第5頁附圖.</ref>。對中華民國首都的認定，詳見[[中華民國首都|中華民國首都]]。}}。

中國共產黨建立「[[中華人民共和國|中華人民共和國]]」後，南京成為由「[[中央人民政府|中央人民政府]]」直轄的「中央直轄市」。1952年11月15日，「中央人民政府」決定南京由「[[直轄市|直轄市]]」改為「[[省轄市|省轄市]]」，作為「江蘇省[[省會|省會]]」。

== 行政區劃 ==
<ref>參閱[http://njdfz.nje.cn/UpLoadFile/html/市志1-4册/第一册.html 南京地方志 第一卷 政區建置] {{webarchive|url=https://web.archive.org/web/20121102034007/http://njdfz.nje.cn/UpLoadFile/html/%E5%B8%82%E5%BF%971-4%E5%86%8C/%E7%AC%AC%E4%B8%80%E5%86%8C.html |date=2012-11-02 }}</ref>

民國元年（1912年），[[南京臨時政府|南京臨時政府]]廢[[上元縣|上元縣]]、[[江寧縣|江寧縣]]，置[[南京府|南京府]]。民國二年（1913年），撤南京府，以舊上元、江寧二縣復置江寧縣。

民國十六年（1927年，析江寧縣城區置南京市。民國十九年（1930年）國民政府制定新的《市組織法》，上所規定的設區條件，南京市設市有一定困難。民國二十年（1931年）3月，實行京市自治，全市劃分為21個自治區，分區籌辦，但進展遲緩。民國二十二年（1933年）1月，經南京國民政府同意，援照杭州市設區的變通辦法，依原有轄區界整併為8區，分別以序數區名<ref>《國民政府公報》第1040號，民國22年1月28日，第3頁.</ref>。以後陸續增加市轄區。民國三十八年（1949年）中華民國政府遷往廣州前，全市劃分為15市轄區：

1949年5月10日，南京市人民政府成立，為中央直轄市。下設13個區。8月增設1個郊區。10月，南京市的大黃洲、小黃洲劃歸安徽省的和縣管轄。12月，蘇南區的句容、江寧縣，皖南的當塗縣，蘇北的六合縣，皖北的江浦、和縣劃歸南京市領導。1950年1月，南京市由新成立的[[華東軍政委員會|華東軍政委員會]]領導。華東局決定將江寧、句容縣仍劃歸蘇南的鎮江專區，六合縣仍劃歸蘇北的揚州專區。當塗、江浦、和縣劃歸安徽省管轄。6月，南京市重新劃區，將13個市區和1個郊區劃為11個市區和1個陵園區。

== 管轄範圍 ==
全市轄境，大抵為今江蘇省南京市玄武、鼓樓、建鄴、秦淮4區全部及浦口、棲霞、雨花台、江寧4區各一小部。面積為559.28平方公里<ref>據《中華民國年鑑》，內政部方域司《中華民國行政區域簡表》（第11版）作465.75平方公里，不含江寧縣湯水、麒麟、東流、古泉4鄉鎮。</ref>。期間市區轄域屢次變更。

據民國十六年（1927年）4月南京建市之初，暫定以[[明南京城|明南京城]]外郭內及浦口地區為市區範圍，面積約157平方公里。6月國民政府公佈《[[南京特別市暫行條例|南京特別市暫行條例]]》第四條規定「本市區域暫以南京城廂內外及浦口之原有區域為南京特別市行政範圍。其區域之分別，由市政府呈請中央政府核定之。」<ref>《國民政府公報》寧字第5號，民國16年6月11日，第9頁。</ref>。但[[江蘇省政府|江蘇省政府]]反對市區擴大，爭持不讓。8月22日，國民政府修訂《暫行條例》第四條，規定：「本市區域暫以[[江寧縣|江寧縣]]原有區域行政範圍，以原有之江寧縣屬之。」<ref>《國民政府公報》寧字第12號，民國16年9月30日，第13頁。</ref>8月25日，市政府接收江寧縣。國民黨江寧縣黨部及各民眾團體聯合反對，隨後又作改變。10月17日，國民政府再次修訂《暫行條例》第四條，規定：「本市區域暫以南京城廂內外原有區域及八卦洲為南京特別市行政範圍。」<ref>《國民政府公報》第2期，民國16年10月，第7頁。</ref>此次變更後的市區，範圍局促，江寧縣政府也未能遷出城區，「於建設上及行政權限上滯礙殊多」。

民國十七年（1928年）3月15日、16日，南京國民政府各相關部局、江蘇省政府、南京特別市政府在南京共同商討相關問題，在有關行政區劃分方面，達成共識，以城廂內外原有區域及八卦洲為其區域，西南大勝關、江心洲，東北烏龍山一帶劃入南京特別市<ref>《國民政府公報》第45期，民國17年3月，第32頁。</ref>；省、市疆界應由省、市政府代表及內政部代表組成劃界委員會，依據所議標準勘定。3月31日，國民政府令飭有關機關遵照辦理。但江蘇省政府一再拖延，並在大勝關、江心洲及烏龍山等地的歸屬問題上激烈爭執，劃界不得不暫時停頓。9月，南京特別市政府曾以市區範圍過小，不利首都建設，向國民政府要求將浦口商埠劃歸市政府管轄，遭到國民政府否決<ref>《國民政府公報》第92期，民國17年9月，第22頁。</ref>。此後，國民黨中央執行委員會政治會議第164次會議決定廢除江寧縣，將江寧縣全境及浦口商埠全部劃入南京特別市<ref>《國民政府公報》第31號，民國17年11月30日，第2頁。</ref>。這一決定遭到江蘇省方面反對。南京國民政府、江蘇省政府、南京特別市政府三方再次協商，決定擴大南京特別市市區，將中山陵及其他有關首都建設的區劃劃入市區，重行由內政部督同省市2政府勘定區域。國民黨中央執行委員會政治會議第168次會議協商結果決定由內政部會同省市2政府勘定市區區域；江寧縣暫緩撤廢，待以後擴充市區時再決定撤廢日期<ref>《國民政府公報》第53號，民國17年12月27日，第6頁。</ref>。

民國二十三年（1934）9月，南京市區域擴大，將原屬江蘇省江寧縣轄地的孝陵衛地區劃歸南京市。市域之四界劃定為：東北起自長江南岸烏龍山，向南到堯化門，再沿明代外郭舊走土城根經仙鶴門、麒麟門，折西南經滄波門、高橋門抵達上坊門，為東界。再沿秦淮河向西，經鐵心橋、西善橋、格子橋，改沿運糧河到江邊，為南界，且於東、南兩邊陸界立界標82座，劃分南京市與江寧縣轄地。從大勝關外入江包括江心洲，以江心與江浦縣分界，折西北以江北浦口商埠原界，亦與江浦縣接壤，為市區西界。再入江包括八卦洲，以江心與六合縣分界，至烏龍山，為市區北界。民國二十四年（1935年）9月30日，省、市、縣三方代表複勘浦口邊界，南部九袱洲新圩及永生洲福字號、祿字號屬市，北部以二祥溝及復生洲一部分屬市。至此，南京市區得以確定，面積465.95平方公里。在此範圍內，以鍾山全部劃為中山陵園區，東、北兩面以環陵路為界，西面以中山門至太平門城垣為界，南面以京湯公路為界，面積30.58平方公里，直屬國民政府管轄。

民國三十四年（1945年）12月，根據行政院訓令，江蘇省江寧縣屬湯水鎮、麒麟鄉、東流鄉、古泉鄉劃屬南京市管轄，南京市轄區向東擴大。民國三十五年（1946年）3月14日，辦理交接。湯水鎮以南界線，於民國三十六年（1947年）2月28日，議定以青林鄉北部之炮兵學校射擊場公路為市、縣分界。

== 人口 ==
民國三十七年（1948年）國民政府主計處統計局公佈的統計資料，南京市人口為103萬0572人<ref>[http://www.xzqh.org/bbs/read.php?tid=1014&fpage=18 民國直轄市、省轄市一覽表（出自《行政區劃論壇》）]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

中華民國內政部戶政司於民國七十九年（1990年）[[臺閩地區|臺閩地區]]戶口及住宅普查報告，臺閩地區籍貫為南京市的人數為2萬5329人，佔非臺灣省籍269萬4917人口當中的0.93%<ref>行政院戶口普查處編. 中華民國79年臺閩地區戶口及住宅普查報告. 臺北市: 行政院戶口普查處. 1992</ref>。

== 政府體制 ==
=== 市行政機構 ===
{{See also|南京市政府 (中华民国)}}

民國十六年（1927年）成立南京特別市，市設市長1人，下轄財政、工務等局；6月，設立浦口商埠管理處。民國十七年（1928年）7月，根據新公佈的《特別市組織法》，改為土地、社會等局。民國十九年（1930年）6月27日，根據《市組織法》，改名為南京市，直隸於行政院<ref>《國民政府公報》第507號，民國19年6月28日，第3頁。</ref>。民國二十六年（1937年），日軍攻陷南京。民國三十四年（1945年）8月抗戰勝利後恢復其行政管轄區域。民國三十六年（1947年）1月13日，行政院公佈《[[南京市政府組織規程|南京市政府組織規程]]》<ref>《國民政府公報》第2725號，民國36年1月16日，第3頁。</ref>，設市長1人、參事2人、下設民政、財政等局。市政府於民國三十八年（1949年）4月下旬隨著國軍撤離而消亡。
