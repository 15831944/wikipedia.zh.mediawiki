'''南極睡鯊'''（學名：''Somniosus antarcticus''）為一種生活於[[深海|深海]]水深 400 ~ 1,100 公尺處<ref name = "IUCN"/><ref name = "FishBase">{{FishBase| genus = Somniosus | species = antarcticus |month = February | year = 2012}}</ref>的[[夢棘鮫科|夢棘鮫科]][[鯊魚|鯊魚]]，棲息於[[大西洋|大西洋]]、[[印度洋|印度洋]]與[[太平洋|太平洋]]南方<ref name = "IUCN"/>。體長可達{{convert|4.4|m|ft|abbr=on}}<ref name = "FishBase"/>，屬於{{link-en|伏擊型掠食者|ambush predator}}，以[[頭足綱|頭足綱]]（尤其是[[大王酸漿鱿|大王酸漿鱿]]）與[[魚類|魚類]]為主食，但其胃中偶爾也可發現[[海洋哺乳動物|海洋哺乳動物]]與[[鳥類|鳥類]]的殘骸<ref name = "IUCN"/>。南極睡鯊時常在漁業捕捉[[大西洋胸棘鯛|大西洋胸棘鯛]]和[[小鱗犬牙南極魚|小鱗犬牙南極魚]]時遭到{{link-en|誤捕|bycatch}}，目前尚不可知這樣的行為會對南極睡鯊造成多大的威脅<ref name = "IUCN"/>。

南極睡鯊在之前一直被認為是[[小頭睡鯊|小頭睡鯊]]（''Somniosus microcephalus''）或是[[太平洋睡鯊|太平洋睡鯊]]（''Somniosus pacifius''），一直到 2004 年才被確認為一新種<ref name = "Yano2004">{{cite journal |last1=Yano |first1=Kazunari |last2=Stevens |first2=John D. |last3=Compagno |first3=Leonard J. V. |title=A review of the systematics of the sleeper shark genus ''Somniosus'' with redescriptions of ''Somniosus'' (''Somniosus'') ''antarcticus'' and ''Somniosus'' (''Rhinoscymnus'') ''longus'' (Squaliformes: Somniosidae) |journal=Ichthyological Research |volume=51 |issue=4 |year=2004 |pages=360–73 |doi=10.1007/s10228-004-0244-4 }}</ref>。
