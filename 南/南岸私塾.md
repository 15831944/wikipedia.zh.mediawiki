'''南岸私塾'''，位于[[湖南省|湖南省]][[韶山市|韶山市]][[韶山乡|韶山乡]]土地冲，现为中国青少年励志教育基地、全国重点文物保护单位，始建于[[清朝|清朝]]晚期，毛泽东曾在此就读。

== 沿革 ==
清朝晚期，南岸私塾建立。

1902年（[[大清|大清]][[光绪|光绪]]二十八年），毛泽东在南岸私塾读书，读了2年，主要学儒家经典《[[三字经|三字经]]》、《[[百家姓|百家姓]]》、《[[论语|论语]]》、《[[孟子|孟子]]》和《[[诗经|诗经]]》，时任老师[[邹春培|邹春培]]。<ref>{{cite web |url=http://www.ssmzd.com/zhanlanzhendi/jiuzhijieshao/28562.html |title=南岸私塾 |accessdate=2010-12-29 |publisher=韶山毛泽东同志纪念馆 |work= |language=zh-cn |archive-url=https://web.archive.org/web/20160304232947/http://www.ssmzd.com/zhanlanzhendi/jiuzhijieshao/28562.html |archive-date=2016-03-04 |dead-url=yes }}</ref>

1968年4月17日，[[湖南省革命委员会|湖南省革命委员会]]常委会议决定开发南岸私塾给旅游者参观，韶山管理局负责监管，12月26日，南岸私塾正式开放参观。

1972年9月1日，湖南省革命委员会将南岸私塾评为省级重点文物保护单位。

1983年10月10日，湖南省人民政府将南岸私塾评为全国重点文物保护单位。

2002年10月，韶山管理局对南岸私塾进行翻修维护。

2010年10月，[[韶山毛泽东同志纪念馆|韶山毛泽东同志纪念馆]]对南岸私塾进行翻修。

== 建筑 ==
南岸私塾主要是清朝建筑，外面的墙是青砖，其他都是泥砖，屋顶是青瓦，墙面是白色，建筑面积394.52平方米，房间共计9间。
