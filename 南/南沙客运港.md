'''南沙客运港'''位于广州市南沙区[[虎门大桥|虎门大桥]]以南商贸大道二路2号，[[珠江|珠江]][[虎门水道|虎门水道]]西侧；于1992年启用，目前由番禺南沙港客运有限公司独家经营，年设计客运量为160万人次。<ref name=":0">{{Cite web |url=http://www.nskyg.com/about |title=公司简介 – 广州市南沙港客运码头 |accessdate=2018-10-05 |work=www.nskyg.com}}</ref>。南沙客运港是连接香港的中国内地港口之一，也是广州仅有的两个城际客运码头之一。（另一个为位于番禺莲花的[[莲花港客运码头|莲花港客运码头]]）<ref>{{Cite web |url=http://news.ycwb.com/2018-09/30/content_30103384.htm |title=“十一”黄金周 广州南沙客运港出入境高峰时段公布 |accessdate=2018-10-05 |work=news.ycwb.com |language=zh-cn}}</ref>。

== 历史 ==
20世纪80年代末至90年代初，当时还属于[[番禺区|番禺市]]的[[南沙区|南沙]]因离番禺市区遥远且交通不便，被戏称为番禺的“[[西伯利亚|西伯利亚]]”。祖籍南沙的香港著名富商[[霍英東|霍英东]]于那时开始了将其打造成现代化海滨新城的大计划。第一代南沙客运港作为这座新城通往港澳等地重要的交通配套工程，由[[霍英东家族|霍英东家族]]财团与当时的番禺市南沙开发区合资建设，于1992年2月2日成立番禺南沙港客运有限公司，同年码头建成并投入使用。至2003年为止，第一代南沙客运港已累计输送816万人次的旅客来往南沙与港澳之间。2000年10月，投资1.1亿元，位于现址的第二代南沙客运港开始建设，并于2004年7月取代第一代南沙客运港而开始运营。<ref>{{Cite web|title=南沙新客运港7月启用 乘船1小时可抵香港|url=http://www.chinahighway.com/news/2004/71562.php|work=www.chinahighway.com|accessdate=2019-04-02}}</ref>2010年06月29日开通往返[[澳门|澳门]][[外港客運碼頭|外港客运码头]]的班次<ref>{{Cite web|title=“广州南沙港-澳门”客运航线29日开通_网易新闻|url=http://news.163.com/10/0629/23/6ACP35U6000146BC.html|work=news.163.com|accessdate=2019-04-02}}</ref>，但于2011年1月4日起取消往返澳门平日船班<ref>{{Cite web|title=“南沙往返澳门航线”航班调整公告 – 广州市南沙港客运码头|url=http://www.nskyg.com/notice/u0026quotnansha-from-macau-routeu0026quot-air-adjustment-notice.html|work=www.nskyg.com|accessdate=2019-04-02}}</ref>，目前澳门航线已暂时停航。2017年7月19日，码头增设[[往来港澳通行证|港澳通行签证]]自助终端，方便旅客现场办理赴港手续。<ref>{{Cite web|title=今天南沙客运港的港澳通行签证自助机正式开通啦！ – 广州市南沙港客运码头|url=http://www.nskyg.com/notice/%E4%BB%8A%E5%A4%A9%E5%8D%97%E6%B2%99%E5%AE%A2%E8%BF%90%E6%B8%AF%E7%9A%84%E6%B8%AF%E6%BE%B3%E9%80%9A%E8%A1%8C%E7%AD%BE%E8%AF%81%E8%87%AA%E5%8A%A9%E6%9C%BA%E6%AD%A3%E5%BC%8F%E5%BC%80%E9%80%9A%E5%95%A6.html|work=www.nskyg.com|accessdate=2019-04-02}}</ref>

== 运营公司 ==
'''番禺南沙港客运有限公司'''（南沙港客运公司）是南沙客运港的运营公司，主要经营广州南沙往返香港的水上客运服务，负责南沙客运港及自有船只的经营维护工作，也负责码头接驳巴士的调度运营。现拥有四艘三舱（贵宾座、头等座、普通座）配置，载客量380人的高速双体客轮；另自有20辆巴士用于码头接驳专线。<ref name=":0" />

南沙港客运公司由[[霍英東家族|霍英东家族]]的番禺建设基金有限公司（番禺建设基金）与广州南沙经济技术开发总公司（南沙经开总公司，南沙区政府全资属下的[[国有企业|国有企业]]）于1992年合资设立；目前注册资本有1112.5万美元，其中番禺建设基金占有南沙港客运公司52%的股份，剩余股份由南沙经开总公司代[[南沙区|南沙区]]持有。
