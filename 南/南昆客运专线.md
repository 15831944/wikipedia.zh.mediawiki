'''南昆客运专线'''（原名云桂铁路<ref>{{Cite web|url=http://ngzb.gxnews.com.cn/html/2015-08/21/content_1167520.html|title=为满足大南宁发展需要申请铁路站点更名 坛洛站将更名为南宁西站|date=2015年8月21日}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>）是中国[[云南省|云南省]]与[[广西壮族自治区|广西壮族自治区]]境内的一条国铁一级电气化双线铁路，设计时速200-250km/h，起于广西[[南宁市|南宁市]]的[[南宁站|南宁站]]，止于云南省[[昆明市|昆明市]]的[[昆明南站|昆明南站]]，全长715.8公里。2009年12月27日，南昆客运专线在广西[[百色|百色]]正式开工建设，全线新建车站17个，改建车站3个，桥隧总长500公里，总投资近900亿元，工期6年。2015年12月11日，南宁至百色段开通运营；2016年12月28日全线开通，使得昆明南站至南宁站的运行时间大幅缩短至4小时30分。<ref>{{Cite web |url=http://news.sohu.com/20091227/n269238081.shtml |title=云桂铁路开工建设 |date=2009年12月27日 }}</ref>

== 线路概况 ==
南昆客运专线定位为一条以客运为主、客货兼顾的西部铁路新干线，是国家《中长期铁路网规划》中的干线铁路，是我国完善铁路网布局和西部大开发的重要基础设施，是[[云南省|云南省]]境内第一条时速200km/h（预留250km/h条件）以客运为主，兼顾集装箱、轻快货物运输的出海铁路通道。铁路东起南宁枢纽的南宁站，沿既有南昆铁路走廊前行后引入既有百色站。之后向西跨过右江进入云南境内，经过文山州的富宁县、广南县和丘北县；于弥勒县江边街附近跨南盘江，经红河州的弥勒县后进入昆明的石林县，在宜良南侧再次跨越南盘江，经过阳宗镇，于昆明市呈贡新区东侧设昆明南客站，之后引入昆明枢纽。途经广西[[隆安县|隆安县]]、[[平果县|平果县]]、[[田东县|田东县]]、[[田阳县|田阳县]]；云南[[文山州|文山州]][[富宁县|富宁县]]、[[广南县|广南县]]、[[丘北县|丘北县]]，[[红河州|红河州]][[弥勒县|弥勒县]]，[[昆明市|昆明市]][[石林县|石林县]]、[[宜良县|宜良县]]，[[玉溪市|玉溪市]][[澄江县|澄江县]]，正线全长709.518km，其中[[广西壮族自治区|广西壮族自治区]]境内274.83km，[[云南省|云南省]]境内431.183km。

=== 南宁-百色段 主要技术指标 ===
*铁路等级：Ⅰ级
*正线数目：双线
*限制坡度：12‰
*最高速度：250km/h
*最小曲线半径：一般5500m、困难4500m
*牵引种类；电力
*机车类型；动车组、SS7E
*牵引质量：4000t
*到发线有效长度：650m
*闭塞类型：[[自动闭塞|自动闭塞]]

=== 百色-昆明段 主要技术指标 ===
*铁路等级：Ⅰ级
*正线数目：双线
*限制坡度：百色至昆明9‰,加力坡18.5‰；石林西至昆明南局部23.5‰
*最高速度：200km/h（预留250km/h）
*最小曲线半径：一般5500m、困难4500m
*牵引种类；电力
*机车类型；动车组、SS7E、货机HXD（百色-石林西段）
*牵引质量：4000t
*到发线有效长度：百色至石林西880m，石林西至昆明南650m
*闭塞类型：[[自动闭塞|自动闭塞]]

全线设[[南宁站|南宁]]、[[南宁西站|南宁西]]、[[隆安东站|隆安东]]、[[平果站|平果]]、[[田东北站|田东北]]、[[田阳站|田阳]]、[[百色站|百色]]、[[阳圩站|阳圩]]、[[平安站|平安]]、[[富宁站|富宁]]、[[白腊寨站|白腊寨]]、[[广南站|广南]]、[[珠琳站|珠琳]]、[[普者黑站|普者黑]]、[[腻革龙站|腻革龙]]、[[新哨站|新哨]]、[[弥勒站_(云南省)|弥勒]]、[[石林西站|石林西]]、[[阳宗站|阳宗]]、[[昆明南站|昆明南]]20个车站，其中隶属南宁局的阳圩站为局间分界站。<ref>{{Citation|title = 铁总运函〔2016〕836号　中国铁路总公司关于公布2016年底调整列车运行图的通知|publisher =[[中国铁路总公司|中国铁路总公司]]|date=2016-10-29|pages=5}}</ref>

== 建设 ==
*2009年12月27日，南昆客专（雲桂鐵路）正式开工建设。
*2011年6月20日，南昆客专小团山隧道贯通，这是云桂铁路贯通的第三个隧道，也是目前云桂线顺利贯通的首个200米以上隧道。<ref>{{Cite web |url=http://yn.yunnan.cn/html/2011-06/20/content_1673669.htm |title=云桂铁路小团山隧道贯通 |date=2011年6月20日 |deadurl=yes |archiveurl=https://web.archive.org/web/20151222094216/http://yn.yunnan.cn/html/2011-06/20/content_1673669.htm |archivedate=2015年12月22日 }}</ref>
*2015年8月22日，由雲桂鐵路更名為南昆客运专线。
*2015年10月，南昆客专南宁到百色段开始联调联试，预计于12月开通运营<ref>{{Cite web |url=http://www.nanning.gov.cn/Government/jjgc/zxdt/201508/t20150821_495331.html |title=南昆客专10月1日起联调联试 南宁至百色运行时间将缩短至一个半小时 |date=2015年8月21日 }}</ref>
* 2015年12月11日，南宁至百色段开通运营，本线开通动车组，使用车辆为[[和谐号CRH2型电力动车组|CRH2A]]，<ref>{{cite news|url=http://news.gxnews.com.cn/staticpages/20151211/newgx5669fee0-14073930.shtml|title=南昆客专南百段开通 南宁-百色动车二等票82.5元|publisher=广西新闻网|date=2015-12-11|accessdate=2015-12-11}}</ref>而湛江-昆明K982/3 K984/1次、昆明-上海南[[T381/382次列车|T382/1次]]在南宁-百色区间改经南昆客专运行。<ref>《关于公布2015年底调整列车运行图的通知》</ref>
*2016年12月28日，百色至昆明段开通运营<ref>{{cite news|title=云南至广西铁路昆明至百色段将开通运营|url=http://www.chinanews.com/cj/2016/12-25/8104325.shtml|publisher=中国新闻网|date=2016-12-25}}</ref>，标志着南昆客运专线全线开通营运。
*2017年5月25日，昆明南至洛羊镇段开始联调联试。<ref>{{cite news|url=http://www.crcen.com/newsshow/3408.aspx|title=昆明南站至洛羊镇站进入联调联试阶段|publisher=中国铁建电气化局集团北方工程有限公司|date=2017-05-26|accessdate=2017-07-05}}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
*2017年6月18日，昆明南站至昆明站联络线开启联调联试。<ref>{{cite news|url=http://news.yntv.cn/content/15/201706/19/15_1519888.shtml|title=昆明站至昆明南站“试跑”动车 两站联络线开启联调联试|publisher=云南电视台|date=2017-06-19|accessdate=2017-07-05}}</ref>
*2017年9月4日，昆明南至昆明段开通运营。<ref>{{cite news|url=http://www.ynta.gov.cn/Item/33789.aspx|title=今起昆明站至昆明南站间开通动车|publisher=云南日报|date=2017-09-04|accessdate=2017-09-04}}</ref>
