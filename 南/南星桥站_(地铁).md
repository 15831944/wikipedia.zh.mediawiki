'''南星桥站'''是[[杭州地铁4号线|杭州地铁4号线]]西南段的一个站点，也是未来[[杭州地铁5号线|杭州地铁5号线]]的一个站点，位于钱江路与飞云江路交叉路口，为地铁4号线与5号线的换乘站。4号线于2018年1月9日建成通车。<ref>{{cite web|title=杭州地铁4号线南段开通试运营|url=http://society.people.com.cn/n1/2018/0109/c1008-29754942.html|website=人民网}}</ref>

== 结构 ==

南星桥站为4、5号线的换乘站。其中4号线车站沿钱江路呈东西向布置，为地下二层岛式车站，车站总长约200米，宽约25米，深约19米；5号线车站沿飞云江路呈南北向布置，为地下三层岛式车站。车站共设5个出入口，3组风亭。
===楼层分布===
南星桥站为地下三层换乘车站，4号线在上，5号线在下，呈T型布置。站台、站厅以抽象梧桐叶体现“凤栖梧桐”的风格。<ref>{{cite web|title=杭州地铁4号线南段近期开通 8个站中联庄站暂缓|url=http://hwyst.hangzhou.com.cn/xwfb/content/2018-01/04/content_6765358.htm|website=杭州网}}</ref>

== 事故 ==
2015年8月28日上午9时33分，南星桥站在地下连续墙施工中不慎挖破燃气管道（直径300mm）导致泄漏。事发后，施工单位通知消防、燃气公司。市地铁集团领导赶赴现场指挥抢险。9:39 ，消防到达现场疏散周边人员；9:55，燃气公司开始抢险，燃气管道阀门及时被关闭。16:20，受损燃气管道焊接作业已完成，16：50，相关燃气管道已恢复通气，此事件对沿线居民及其他用户供气的影响已全部消除。事件未造成施工人员伤亡。<ref>{{Cite web|url = http://weibo.com/1700720163/CxT1GeCKH?type=comment#_rnd1440731996339|title = 杭州地铁4号线南星桥站施工中挖漏燃气管道|accessdate = 2015-08-28|author = |date = |publisher = }}</ref>
