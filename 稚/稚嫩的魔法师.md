'''稚嫩的魔法师'''（{{bd|1992年}}），是[[中国大陆|中国大陆]]网络播客、游戏解说人<ref name="稚嫩的魔法师丁">{{cite web|author1=胡琴沁、王崇帝、牛玮琦|title=研究生客串游戏解说达人 “粉丝”为其建百科词条|url=http://sh.people.com.cn/n/2015/0819/c357188-26042418.html|website=人民网|accessdate=2017-08-29|location=北京|language=zh-cn}}</ref>。于2013年首次在[[哔哩哔哩|哔哩哔哩弹幕视频网]]发布游戏解说视频，成为视频播客<ref name="稚嫩的魔法师甲"/>。

稚嫩的魔法师本科和研究生阶段就读于[[武汉大学|武汉大学]][[历史学|历史学系]]<ref name="稚嫩的魔法师甲"/>，其视频作品以历史策略类游戏为主，特别是[[Paradox_Interactive|Paradox Interactive]]发行的系列游戏<ref name="稚嫩的魔法师丙"/>。不过近年来他也开始进行视频直播和实况解说<ref name="稚嫩的魔法师甲"/>。

== 个人经历 ==
2014年7月，正值研究生一年级的“稚嫩的魔法师”发布了自己的首个游戏解说视频。“稚嫩的魔法师”这个网名源于[[英国|英国]]作家[[菲利普·肖特|菲利普·肖特]]所著《[[毛泽东传|毛泽东传]]》中一个章节的名称<ref name="稚嫩的魔法师乙"/>。初期默默无闻，并未得到太大关注。他在回忆当时的经历时表示，他最初很担心自己做的视频没人看，评价也不高，“这样会让自己显得更傻<ref name="稚嫩的魔法师甲">{{cite web|author1=吴俊宇|title=“剪接历史”的魔法师|url=http://mcms.nbweekly.com/iphone/article.php?id=209386778&ref=iphoneshare&from=singlemessage&isappinstalled=0&url_type=39&object_type=webpage&pos=1|website=南方都市周刊|accessdate=2017-08-29|location=广州|language=zh-cn}}</ref>。”

但之后，他的视频开始逐步在网络上走红<ref name="稚嫩的魔法师甲"/><ref name="稚嫩的魔法师丙">{{cite web|author1=胡琴沁、王崇帝、牛玮琦|title=研究生客串游戏解说达人 “粉丝”给他建立百科词条|url=http://t.m.youth.cn/transfer/index/url/news.youth.cn/jsxw/201702/t20170207_9089736.htm|website=新华网（转引长江日报）|accessdate=2017-08-29|location=北京|language=zh-cn}}</ref>。稚嫩的魔法师的作品以长视频为主，并以历史策略战棋类游戏——《[[十字军之王II|王国风云II]]》、《[[维多利亚II|维多利亚II]]》、《[[钢铁雄心III|钢铁雄心III]]》作为创作主轴<ref name="稚嫩的魔法师甲"/>。稚嫩的魔法师的视频受众不局限于游戏玩家，而是将普通观众作为其视频的主要受众群体，并常向观众科普游戏背后的历史知识<ref name="稚嫩的魔法师丙"/>。他在接受专访时表示，“自己早已不会再用感性的目光去看待历史，只是喜欢把生活里的脑洞和段子融入游戏，给观众们带来乐趣<ref name="稚嫩的魔法师乙"/>。”

除进行游戏解说外，稚嫩的魔法师还曾通过视频介绍历史专业书籍，讲解《[[三国志|三国志]]》等历史学专著，被认为“以一己之力开辟B站读书区”<ref name="稚嫩的魔法师甲"/><ref name="稚嫩的魔法师丙"/>。他在接受《[[长江日报|长江日报]]》访问时说，自己就是“突发奇想脑洞大开，开了这么一个新坑，邀请大家和我一起体验读史书的乐趣”<ref name="稚嫩的魔法师甲"/><ref name="稚嫩的魔法师丙"/>。他还表示，“我就是想强调一下就是原始史料的重要性。我们现在有太多人撇开史料谈历史，很多人的观点都是从网络文章或是看电视剧得来的，以此在网络上争论，这毫无意义。<ref name="稚嫩的魔法师乙">{{cite web|author1=吴俊宇|title=我们跟武大历史系的博士生聊了聊，这款让他乐在其中的历史游戏|url=http://mp.weixin.qq.com/s/6vkM7WavAZ8XyNhfTxyj2g|website=南方都市周刊|accessdate=2017-08-29|location=广州|language=zh-cn}}</ref>”

== 评价 ==
视频播客“资深小狐狸”曾评价稚嫩的魔法师：“造梗能力一流<ref name="稚嫩的魔法师甲"/>。”

《[[南都周刊|南都周刊]]》认为，脑洞够大、剧情夸张、造梗能力强是他走红的重要原因<ref name="稚嫩的魔法师乙"/>。《南都周刊》评价说，稚嫩的魔法师善于在游戏解说中制造逻辑反转，从而使其视频出现戏剧冲突，这增强了整个视频的娱乐性，提升了整个作品的代入感，并形成了一种“文化认同”<ref name="稚嫩的魔法师甲"/>。

稚嫩的魔法师本人认为，“好的游戏会在历史考证和世界观的构建上有一整套自己的体系，会反映出游戏公司的功力，你能在其中学到很多东西<ref name="稚嫩的魔法师乙"/>。”同时，由于他精通[[膜蛤文化|膜蛤文化]]并在视频中加以运用，因而其亦被一些网友戏称为“稚嫩的膜法师”<ref name="百科甲">{{cite web|author1=萌娘百科编者|title=萌娘百科上关于稚嫩的魔法师外号的介绍|url=https://zh.moegirl.org/index.php?title=%E7%A8%9A%E5%AB%A9%E7%9A%84%E9%AD%94%E6%B3%95%E5%B8%88&mobileaction=toggle_view_desktop|website=萌娘百科|accessdate=2017-08-29|location=|language=zh-cn}}</ref>。
