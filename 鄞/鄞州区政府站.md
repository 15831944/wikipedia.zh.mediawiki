'''鄞州区政府站'''，规划站名'''鄞县大道站'''，是[[浙江省|浙江省]][[宁波市|宁波市]]规划中的一座地下轨道交通车站。鄞州区政府站是[[宁波轨道交通3号线|宁波轨道交通3号线]]和[[宁波轨道交通5号线|5号线]]的换乘站，3号线车站计划于2019年启用<ref name="main">{{cite web|url=http://daily.cnnb.com.cn/nbrb/images/2014-12/24/A2/nbrb20141224a02.pdf|title=宁波地铁3号线一期开工|author=冯瑄|publisher=宁波日报|date=2014-12-24|accessdate=2014-12-26}}</ref>，5号线车站计划于2020年启用。

== 位置 ==
鄞州区政府站位于[[鄞州区|鄞州区]]天童路与鄞县大道交叉口，3号线沿天童路南北向设置，5号线沿鄞县大道东西向设置<ref name="main" />。

== 车站形制 ==
鄞州区政府站3号线车站为地下二层岛式车站，长241.9米，宽22.9米，总建筑面积15987平方米。5号线为地下三层岛式车站，长165米，宽23.1米。车站总建筑面积14865平方米。两线为L换乘<ref name="design">市发展改革委关于宁波市轨道交通3号线一期工程初步设计的复函, 甬发改审批〔2014〕653号</ref><ref name="main" />。

== 出口 ==
鄞州区政府站规划设置5个出入口<ref name="design" />。
