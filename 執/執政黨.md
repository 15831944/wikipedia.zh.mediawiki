'''執政黨'''，是一個[[國家|國家]][[政府|政府]]中獲得[[政權|政權]]的[[政黨|政黨]]，一般是[[國家元首|國家元首]]、[[政府首腦|政府首腦]]或地方[[行政長官|行政長官]]所屬的政黨，或者是其他形式而掌握執政權的政黨。

标准的議會民主國家中，执政党是指实际控制立法行政权力的政党，政治下的执政权力一般限于国家治理，不是能真正掌握国家制度的权力，因为单一政党及其政治家无法代表绝对多数政治力量。[[民主|民主]]國家的執政黨政治人物在担任国家及地方政府重要职务时，擁有依法組織政府和制定政策的权力，同时执政党負起政治責任，以維護執政權，否則會在下次[[選舉|選舉]]中成為在野黨甚至丧失国家层面的政治活动地位。在[[政党轮替|政党轮替]]中與執政黨相反的是[[在野黨|在野黨]]，即在国家机关中拥有席位，有政治地位的政党，或[[反對黨|反對黨]]，即广义上反对执政党的政党。

== 總統制 ==
[[總統制|總統制]]的國家，則以[[總統|總統]]所屬的政黨作為執政黨，總統兼任國家元首和政府首腦。如[[美國|美國]]、[[墨西哥|墨西哥]]、[[巴西|巴西]]、[[阿根廷|阿根廷]]、[[智利|智利]]、[[哥倫比亞|哥倫比亞]]、[[菲律賓|菲律賓]]和[[南韓|南韓]]。執政黨在[[議會|議會]]未必是多數黨，或擁有佔多數的執政聯盟，擁有行政权力，属[[行政主導|行政主導]]。政治較為穩定，總統受到任期保障，除非犯下嚴重罪行，否則難以被趕下台，而民主國家的議會擁有[[彈劾|彈劾]]總統的權力。

== 半總統制 ==
[[半總統制|半總統制]]又稱[[雙首長制|雙首長制]]的國家，擁有行政权的[[總統|總統]]和领導內閣的[[總理|總理]]共同行使政治权力，總統与總理並非一定屬同一政党，但總理的权力一般是基于其政党在議會是否維持多数，如：[[中華民國|中華民國]]。而[[中華民國|中華民國]]設置[[中華民國總統|總統]]卻不設[[總理|總理]]，取而代之的是[[行政院院長|行政院院長]]，且[[中華民國總統|總統]]與[[行政院長|行政院長]]通常同屬一個政黨，另依據[[中華民國憲法|中華民國憲法]]第55條規定：行政院院長，由總統提名，經立法院同意任命之。後[[中華民國憲法增修條文|中華民國憲法增修條文]]第三條改由行政院院長由總統直接任命之。

== 內閣制 ==
[[內閣制|內閣制]]或[[君主立憲制|君主立憲制]]國家的政府需要經[[公民|公民]]直接選舉[[國會|國會]][[議員|議員]]，再由國會多數黨作為執政黨。現時行[[兩院制|兩院制]]的國家，多數會以[[下議院|下議院]]多數，作為執政基礎，政府亦需要獲下議院的多數才能繼續執政。

若有一黨贏得國會下議院過半數席次並自行組閣，該黨即為執政黨，如[[英國|英國]]和[[日本|日本]]；若無任一政黨席次過半，則各參與[[聯合政府|聯合政府]]的政黨皆為執政黨，如[[德國|德國]]和[[意大利|意大利]]；若数个政党组成联盟的国家内，赢得最多议席的政党占据了主导优势，如[[马来西亚|马来西亚]]。

== 混合制 ==
也因实行[[總統內閣制|總統內閣制]]，總統由议会選出，但握有实际行政权力，如[[緬甸|緬甸]]、[[南非|南非]]。

== 君主獨裁 ==
[[君主專制|君主專制]][[獨裁|獨裁]]國家，政權為[[君主|君主]]絕對掌控，如[[沙烏地阿拉伯|沙烏地阿拉伯]]和[[阿曼|阿曼]]。這些國家政權不會有政黨存在。

== 共和獨裁 ==
在[[一黨獨大|一黨獨大]]或[[一黨執政|一黨執政]]的國家，只有一個具支配地位的合法執政黨，反對黨長期處於在野地位且无机会執政，部分國家不允許具競爭性的政黨存在，如[[俄羅斯|俄羅斯]]、[[白俄羅斯|白俄羅斯]]、[[亞塞拜然|亞塞拜然]]、[[土耳其|土耳其]]、[[敘利亞|敘利亞]]、[[葉門|葉門]]、[[新加坡|新加坡]]、[[土庫曼斯坦|土庫曼斯坦]]、[[烏茲別克斯坦|烏茲別克斯坦]]、[[哈薩克斯坦|哈薩克斯坦]]、[[辛巴威|辛巴威]]、[[蘇丹|蘇丹]]、[[厄利垂亞|厄利垂亞]]、[[赤道幾內亞|赤道幾內亞]]、[[坦桑尼亞|坦桑尼亞]]、[[莫三比克|莫三比克]]等。

== 共产主义国家 ==
在[[共產黨|共產黨]]执政的国家，该国共产党常常是唯一合法的执政党，只不過有時也允许存在一些其他[[卫星党|卫星党]]參政，卻不能取得执政地位，屬[[政治花瓶|政治花瓶]]，如[[朝鮮|朝鮮]]、[[中華人民共和國|中華人民共和國]]、[[越南|越南]]、[[古巴|古巴]]。
