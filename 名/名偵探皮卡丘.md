{{vgname|名偵探皮卡丘|ja=名探偵ピカチュウ|en=Detective Pikachu|diff=yes}}是一款由[[Creatures|Creatures]]開發的[[冒險遊戲|冒險遊戲]]，由[[精靈寶可夢_(公司)|精靈寶可夢公司]]於[[任天堂3DS|任天堂3DS]]平台發行，同時也是《[[精靈寶可夢|精靈寶可夢]]》系列遊戲的衍生作。玩家能在遊戲中使用[[皮卡丘|皮卡丘]]來解決各種懸疑事件。

== 內容簡介 ==
遊戲的故事圍繞著一隻奇特的[[皮卡丘|皮卡丘]]展开。雖然沒有像其他的皮卡丘一样強大和敏捷，但它卻相當的聰明，自稱是名偉大的偵探。有一天，這隻皮卡丘遇見了一名叫作提姆·古德曼的男孩，他能夠聽得懂牠在說什麼。因此，兩人開始合作，解決城市中圍繞神奇寶貝发生的各種神秘事件。

== 發行 ==
前身為2016年2月3日在日本[[任天堂eShop|任天堂eShop]]發行的{{nihongo|《名偵探皮卡丘 ～新搭檔誕生～》|名探偵ピカチュウ 〜新コンビ誕生〜}}<ref name="Kotaku: trailer" />；2018年3月23日，实体游戏加强版《名偵探皮卡丘》在全球发售，該版本追加大量新要數，不仅加入了对[[Amiibo|Amiibo]]的支持，也支持繁简中文<ref name="WesternDate" /><ref name="GypNB" />。「新搭檔誕生」版本為加強版收錄至第三章內容，於加強版販售後移除，原先消費者在期限內從任天堂eShop購入加強版可享有折扣。根據[[4Gamer.net|4Gamer.net]]統計，本遊戲首週於日本通路的販售總量為42,013套。<ref>{{Cite web |title=週間販売ランキング＋ |url=http://www.4gamer.net/games/117/G011794/20180328082/ |date=2018-03-28 |access-date=2018-04-02 |publisher=4Gamer.net}}</ref>

== 改編電影 ==
{{main|POKÉMON 名偵探皮卡丘}}
2016年7月20日，在電影《[[Pokemon_the_movie_XY&Z_波爾凱尼恩與機關人偶_瑪機雅娜|波爾凱尼恩與機關人偶 瑪機雅娜]]》上映四天後，美國電影公司[[傳奇影業|傳奇影業]]宣布，他們已獲得了該遊戲的真人電影改編權，並預定於2017年開始製作<ref name="rP5oA" />。同年，據報導，{{le|艾力克斯·赫希|Alex Hirsch}}和{{le|妮可·帕爾曼|Nicole Perlman}}將會負責撰寫劇本<ref name="Collider" />。[[狄恩·伊斯拉利特|狄恩·伊斯拉利特]]、[[勞勃·羅里葛茲|勞勃·羅里葛茲]]和[[提姆·米勒|提姆·米勒]]都正在會談擔任導演。該片在日本由[[東寶|東寶]]發行，而[[環球影業|環球影業]]則負責日本以外的地區發行。2016年11月30日，據[[Deadline.com|Deadline]]報導，傳奇影業選擇了{{le|羅勃·萊特曼|Rob Letterman}}來執導該片<ref name="sVQ2W" />。2017年10月15日，宣布拍攝將於2018年1月開始<ref name="iTMSO" />。隔月，{{le|賈斯提斯·史密斯|Justice Smith}}將加入飾演人類的角色<ref name="j5e9y" />。{{le|凱瑟琳·紐頓|Kathryn Newton}}則在同月時被宣布將會擔任女主角<ref name="KtN13" />。2017年12月6日，宣布[[萊恩·雷諾斯|萊恩·雷諾斯]]將會飾演偵探皮卡丘（動作捕捉和配音）<ref name="uYpyv" />；此外，該角也曾考慮過[[巨石強森|巨石強森]]、[[馬克·華伯格|馬克·華伯格]]和[[休·傑克曼|休·傑克曼]]<ref name="HEJ7U" />。2018年1月19日，宣布[[渡邊謙|渡邊謙]]也將加入演出<ref name="PlMxz" />。

環球該片定於2019年5月10日在美國上映<ref name="GDP Movie 2019" />。後來發行權轉到[[華納兄弟|華納兄弟]]手上。
