'''名古屋高速11號小牧線'''（[[平假名|平假名]]：なごやこうそく11ごうこまきせん）是一條連接[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[北區_(名古屋市)|北區]][[楠JCT|楠JCT]]與同縣[[小牧市|小牧市]][[小牧IC|小牧IC]]/[[小牧北出入口|小牧北出入口]]的[[名古屋高速道路|名古屋高速道路]]路線。

== 概要 ==
* 起點 : 愛知縣名古屋市北區大我麻町
* 終點 : 愛知縣小牧市村中
* 路線長度 : 8.2km
* 出入口 : 10個（入口:5個、出口:5個）
* 分支 : 2個
* [[行車線|行車線]] : 4車線

== 历史 ==
* 2001年3月11日 : 楠JCT至小牧南出入口開通。
* 2001年10月19日 : 豐山北入口開通，小牧南出入口至小牧北出口開通，小牧IC可以連接東名高速道路與名神高速道路。
* 2002年4月24日 : 小牧北入口開通，全線開通。
* 2009年3月12日 : 堀之內出入口可接受ETC。
