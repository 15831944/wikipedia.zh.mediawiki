== 本作特色 ==
* 少年偵探團隊員．小哀未登場的最後作品。

== 劇情簡介 ==
與[[毛利小五郎|毛利小五郎]]有關係的人一個個都遭到不明兇手的襲擊。兇手襲擊後都會留下一張花色為黑桃的[[撲克牌|撲克牌]]，因而警方將兇手指向是曾任賭場荷官且被毛利逮捕的村上丈，但這只是兇手的圈套。兇手透過某些管道得知村上丈善於玩撲克牌，加上村上丈曾經被當時任職[[刑警|刑警]]的[[毛利小五郎|毛利小五郎]]逮捕，進入米花監獄服刑。凶手先殺害村上丈，並因兇手想要殺害的人名字中都有數字的緣故，遂使用撲克牌行兇嫁禍給村上丈。

後來[[柯南|柯南]]發現，村上丈是左撇子，而兇手卻是右撇子，而故在東京灣的行兇完全不是村上丈所為。透過當時被踢翻的飲料飛濺在兇手褲子上的痕跡、小山內奈奈遇害時塞進兇手口袋中塗過[[指甲油|指甲油]]的“軟木塞”以及兇手口袋中剩餘的黑桃A撲克牌，證明了兇手就是[[澤木公平|澤木公平]]。罪行被揭發後，[[澤木公平|澤木公平]]早以旭勝義秘書的名義，向外叫來了一部[[直昇機|直昇機]]，企圖逃亡。他用刀挾持[[毛利蘭|毛利蘭]]作為人質，並引爆了建築物底下的炸彈，使整座建築物傾倒漸漸沉入海中。所幸經由大家的危機應變，終於將[[澤木公平|澤木公平]]繩之以法。

小蘭一直無法瞭解多年前爸爸為什麼要在媽媽被意圖逃亡的村上丈挾持時開槍射擊她，最後得知當時[[毛利小五郎|毛利小五郎]]之所以讓[[妃英理|妃英理]]因此受了輕傷是要使[[人質|人質]]成為歹徒逃亡的負擔，以便救回人質並趁機將歹徒繩之以法，令小蘭解開了在心中的結，但是小五郎也在那次事件之後就不做警察了。
