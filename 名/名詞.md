'''名詞'''
是[[詞類|詞類]]的一種，也是[[实词|实词]]的一种，是指代人、物、事、時、地、情感、概念、方位的名詞等實體或抽象事物的詞。名詞可以独立成句。在短語或句子中通常可以用[[代詞|代詞]]來替代。

== 名詞的特徵 ==
1.一般能受數詞與量詞的修飾。<br />
例：一個人，一本書

2.一般不受副詞修飾（前面也用｢不｣修飾）。<br />
例：很智慧，不智慧
    
3.一般可以與介詞組成介賓結搆（介賓短語不能作主語）。<br />
例：在房間，在教室

4.句子中通常當作主語和賓語，也可作定語。<br />
例：爸爸，媽媽，老師，同學等

5.一個東西<br />
例：狗，貓，熊等

== 名詞的分類 ==
* 表示事物：例：树木，月饼
* 表示人物：例：作家，教师
* 表示时间：例：黑夜，春节
* 表示处所：例：学校，城市
* 表示方位：例：上方，东面
* 抽象名詞​​：例：事業，道德
* 概念名词：例：数学，物理，力，能量，群论，环

== 分类方法 ==
名词在不同的語言中可以分為“[[專有名詞|專有名詞]]”、“普通名詞”和“集合名詞”等等。另一种分类方法是把名词分為“具體名詞”和“抽象名詞”。前者指有實體的確定事物（如[[貓|貓]]、[[床|床]]、[[金字塔|金字塔]]等）；後者指情感、意見、概念等抽象的事物（如[[自由|自由]]、[[正義|正義]]等）。兩者之間的區別並不是非常明顯，互相滲透的情況很多。

在部份語言的句子中，名詞有會因為指稱物體的數量、類別或相對於動作的用途等而附加不同的[[屈折變化|屈折變化]]。名词最主要的屈折变化是[[格_(語法)|格]]、[[數_(語法)|數]]和性，此外还有较为少见的[[指小|指小]]屈折。名词在句子中发生屈折变化时，有時與名詞在意义上相关联的動詞或形容詞等，也會有相對應的屈折變化；而在用代詞指稱名詞時，也往往會因為所指稱的名詞的[[格_(語法)|格]]、[[數_(語法)|數]]與性等不同，而使用不同的代詞來指稱。在一些名词有屈折变化的语言中，在名词之前或之后还有[[冠词|冠词]]。

在句中，名詞一般以[[主語|主語]]或[[賓語|賓語]]的形式出現，也常常作[[定语|定语]]直接修饰另一名词，或是作[[同位语|同位语]]或[[呼语|呼语]]。在某些语言中，名词还可以作[[表语|表语]]和[[状语|状语]]。

===专有名词===
人名、地名等，以及除此之外的特指某一事物的名词。[[语法|语法]]上的一些固定搭配如谦词敬词。
例：“台灣”、“维基百科”、“殿下”、“令堂”等

== 各种语言中的名词 ==
[[德语|德语]]中的全部名词和名词化的其他词类都要大写首字母，这在现在全世界的主要语言中是一个有趣的特例。[[英語|英語]]等很多使用[[拉丁字母|拉丁字母]]、[[西里尔字母|西里尔字母]]等分大小写的[[拼音文字|拼音文字]]体系的語言仅用大寫標記專有名詞，如人名、地名、商標名等等。[[汉语|汉语]]沒有屈折變化，也沒有冠詞，所以普通名詞和專有名詞出現在書面中時，若不附加[[专名号|专名号]]等標記，就沒有任何形式上的分別。

另外，英語等語言中也有表達群體的“集合名詞”或不可數的“物質名詞”等，汉语裏則沒有這個區別。汉语要表達名詞數量，是通過附加[[量詞|量詞]]的方法。汉语也有一般情況下“不可數”的詞，如「正義」。但這通常是語義上的，而非語法上的限制，如有時也會出現「兩種正義」這類的詞句。
