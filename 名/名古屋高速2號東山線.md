'''名古屋高速2號東山線'''（[[平假名|平假名]]：なごやこうそく2ごうひがしやません）是一條連接[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[中村區|中村區]][[新洲崎JCT|新洲崎JCT]]與同縣同市[[東區_(名古屋市)|東區]][[高針JCT|高針JCT]]的[[名古屋高速道路|名古屋高速道路]]路線。

== 概要 ==
* 起點 : 愛知縣名古屋市中村區名站南
* 終點 : 愛知縣名古屋市東區高針
* 路線長度 : 10.3km
* 出入口 : 6個（入口:6個、出口:6個）
* 分支 : 3個
* [[行車線|行車線]] : 4車線

== 連接高速道路 ==
* [[名古屋高速5號萬場線|名古屋高速5號萬場線]]（於[[新洲崎JCT|新洲崎JCT]]連接）
* [[名古屋高速都心環狀線|名古屋高速都心環狀線]]（於[[丸田町JCT|丸田町JCT]]連接）
* [[名古屋第二環狀自動車道|名古屋第二環狀自動車道]]（於[[高針JCT|高針JCT]]連接）

== 历史 ==
* 1986年10月27日：新洲崎JCT至白川出入口開通。
* 1988年4月26日：白川出入口至吹上出入口（現時為吹上東出入口）開通，吹上連絡路啟用。
* 1995年9月19日：吹上連絡路停用。
* 2000年12月11日：吹上東出入口至四谷出入口之西行線開通，吹上出入口改名為吹上東出入口。
* 2001年6月1日：吹上東出入口至四谷出入口之東行線開通。
* 2003年3月11日：四谷出入口至高針JCT開通，同時春岡入口啟用，全線開通。
