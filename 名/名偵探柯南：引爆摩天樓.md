{{日本作品|名偵探柯南：引爆摩天樓|名探偵コナン 時計じかけの摩天楼|Detective Conan: The Time Bomb Skyscraper}}，是[[日本|日本]][[漫畫家|漫畫家]][[青山剛昌|青山剛昌]][[漫畫|漫畫]]系列《[[名偵探柯南|名偵探柯南]]》的第1部[[劇場版|劇場版]]，片長95分鐘，於1997年4月15日在[[日本|日本]]上映，在[[日本|日本]]有11億[[日元|日元]]的票房收益。

== 本作特色 ==
* 本作是《名偵探柯南》第一部劇場版動畫，在此之後固定於每年的[[黃金周_(日本)|黃金周]]上映一部劇場版。
* 片頭的兇殺案與劇中的事件沒有直接關係（但提到新一與事件的關聯及背景）。
* 片中的米花市政大樓的原型是[[東京都廳|東京都廳]]。

== 劇情簡介 ==
[[江戶川柯南|柯南]]用新一的身份在電話不自願地約會了[[毛利蘭|小蘭]]，陪她看5月3日的午夜場電影，以便請求小蘭替他參加森谷帝二教授舉辦的茶會。柯南為此十分煩惱，不知如何掩飾他變小的事實。目送小蘭離開後，柯南接到一個電話，是來歷不明的炸彈客挑戰柯南！

[[少年偵探團|少年偵探團]]收到的引爆遙控飛機、被安裝了炸彈並裝著小貓的寵物籃，還有東都環狀線的五顆炸彈陷阱，大家奮力推理，才及時預防了可能發生的意外。可是，最後被安裝了炸彈的地點其實是[[工藤新一|工藤新一]]和小蘭的約會地點－米花市政大樓！柯南該怎麼辦呢？

|'''森谷帝二'''||石田太郎||[[孫誠|孫誠]]||[[張炳強|張炳強]]||[[周永光|周永光]]||
: 47歲。東都大學建築系教授，也是全日本首屈一指的建築師，30歲以年輕建築師的身分崛起、還以設計東都環狀線橋梁而獲得日本建築協會最佳新人獎，理想是設計左右[[對稱|對稱]][[建築|建築]]的[[完美主義|完美主義]]。
: 他原本的名字是貞治，但為了使其名字的四個字均為[[對稱|對稱]]圖形才改名。他原先設計出的作品因為建築法或營建業者的要求，無法達到完美的[[對稱|對稱]]，與他的理想背道而馳，所以他決定炸毀，掩飾他對[[完美主義|完美主義]]無法達成的陰影。另一方面，由於森谷15年前還年輕時的想法與現今不一致，自然設計出來的[[建築|建築]]也有不同的結果，因此現今與當年的作品不同，便對他造成了一種羞辱。為了掩飾這一切，也決定炸毀，當作它不存在。
: 最後炸毀米花市政大樓的動機則是在於不接受政府財政困難的理由，由於當時[[日本|日本]]發生[[經濟泡沫|經濟泡沫]]，政府與企業財政困難，無法順著森谷的心意建造大樓，而使該大樓成為森谷一生中最失敗的建築作品。另外，由於當時的西多摩[[市長|市長]]因為過失撞死路人而下台（該案件由[[江戶川柯南|工藤新一]]偵破），進而造成森谷的[[新市鎮|新市鎮]]計畫也在新市長繼任之後完全停擺，因此森谷也決定報復[[江戶川柯南|工藤新一]]，並企圖在炸毀米花市政大樓時順便炸死[[江戶川柯南|工藤]]的女友[[毛利蘭|毛利蘭]]。為了達成自己的完美主義與報復[[工藤新一|工藤新一]]才做出了這件事。
: 有一位弟子'''風間英彥'''，詳見《[[名偵探柯南：往天國的倒數計時|往天國的倒數計時]]》。
: 其名'''森谷帝二'''（もりや ていじ，Moriya Teiji）是[[歇洛克·福爾摩斯|福尔摩斯]]探案集中著名反派角色[[詹姆斯·莫里亞蒂教授|莫里亚提]]（Moriaty）的谐音。
