'''名古屋高速6號清須線'''（[[平假名|平假名]]：なごやこうそく6ごうきよすせん）是一條連接[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[西區_(名古屋市)|西區]][[明道町JCT|明道町JCT]]與同縣[[清須市|清須市]][[清洲JCT|清洲JCT]]的[[名古屋高速道路|名古屋高速道路]]路線。

== 概要 ==
* 起點 : 愛知縣名古屋市西區
* 終點 : 愛知縣清須市朝日
* 路線長度 : 7.0km
* 出入口 : 4個（入口:4個、出口:4個）
* 分支 : 2個
* [[行車線|行車線]] : 4車線

== 連接高速道路 ==
* [[名古屋高速都心環狀線|名古屋高速都心環狀線]]（於[[明道町JCT|明道町JCT]]連接）
* [[名古屋第二環狀自動車道|名古屋第二環狀自動車道]]（於[[清洲JCT|清洲JCT]]連接）
* [[名古屋高速16號一宮線|名古屋高速16號一宮線]]（於[[清洲JCT|清洲JCT]]連接）

== 历史 ==
* 1996年 : 開始建造堀越至清洲JCT。
* 2001年 : 開始建造明道町至淺間町着工。
* 2004年 : 開始建造[[新名西橋|新名西橋]]（擬稱）。
* 2004年 : 開始建造淺間町-庄内通 - 堀越着工。
* 2006年3月 : 開始建造明道町JCT。
* 2006年7月 : 開始建造庄内通 - 堀越竣工。
* 2007年1月 : 新名西橋 - 清洲JCT建成。
* 2007年4月 : 明道町JCT - 明道町建成。
* 2007年6月 : 決定出入口與路線名稱。
* 2007年7月 : 明道町 - 淨心交差點、鳥見町 - 新名四橋建成。
* 2007年9月 : 淨心交差點 - 庄内通建成。
* 2007年11月21日 : 跨越[[庄内川|庄内川]]的橋樑命名為「紅Tonbo橋」。
* 2007年12月9日 : 全線開通。
