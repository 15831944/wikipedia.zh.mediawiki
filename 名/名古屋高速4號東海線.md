'''名古屋高速4號東海線'''（[[平假名|平假名]]：なごやこうそく4ごうとうかいせん）是一條連接[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[中川區|中川區]][[山王JCT|山王JCT]]與同縣[[東海市_(日本)|東海市]]新寶町[[東海JCT_(愛知縣)|東海JCT]]的[[名古屋高速道路|名古屋高速道路]]路線。在此路線中，只餘下六番北出入口與木場出入口通未開通，但已預計在2013年3月開通。

== 概要 ==
* 起點 : 愛知縣名古屋市中川區山王
* 終點 : 愛知縣東海市新寶町
* 路線長度 : 12.0km
* 出入口 : 7個（入口:7個、出口:7個）
* 分支 : 2個
* [[行車線|行車線]] : 4車線

== 連接高速道路 ==
* [[名古屋高速都心環狀線|名古屋高速都心環狀線]]（於[[山王JCT|山王JCT]]連接）
* [[伊勢灣岸自動車道|伊勢灣岸自動車道]]（於[[東海JCT_(愛知縣)|東海JCT]]連接）

== 历史 ==
* 2004年3月：Kiraku橋建成。
* 2005年2月：開始建設山王JCT至港明出入口之間路段。
* 2006年11月：開始建設木場出入口至船見出入口之間路段。
* 2007年1月：開始建設船見出入口至東海JCT之間路段。
* 2007年8月6日：山王彎位（現時：山王JCT）完成擴展。。
* 2009年2月6日：宣布不可以在2010年全線開通。
* 2010年9月4日：山王JCT至六番北出入口之間開通。
* 2011年11月19日：木場出入口至東海JCT之間開通，同時山王入口、尾頭橋出口啟用<ref name="kaituyotei">[http://www.nagoya-expressway.or.jp/kousin/topics/111014kaitu.html 名古屋高速道路（高速４号東海線 木場～東海JCT間）の開通のお知らせ] {{webarchive|url=https://web.archive.org/web/20120623193824/http://www.nagoya-expressway.or.jp/kousin/topics/111014kaitu.html |date=2012-06-23 }}</ref>。
* 2013年3月（預計）：六番北出入口至木場出入口之間開通<ref name="Nagoya expwy" />。
