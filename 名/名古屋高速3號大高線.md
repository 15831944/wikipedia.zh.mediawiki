'''名古屋高速3號大高線'''（{{jpn|j=名古屋高速3號大高線|hg=なごやこうそく3ごうおおだかせん|rm=Nagoya kōsoku gō odakasen}}）是一條連接[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[昭和區|昭和區]][[鶴舞南JCT|鶴舞南JCT]]與同縣同市[[綠區_(名古屋市)|綠區]][[名古屋南JCT|名古屋南JCT]]的[[名古屋高速道路|名古屋高速道路]]路線。

== 概要 ==
* 起點 : 愛知縣名古屋市昭和區鶴舞
* 終點 : 愛知縣名古屋市綠區大高
* 路線長度 : 12,1km
* 出入口 : 11個（入口:6個、出口:5個）
* 分支 : 3個
* [[行車線|行車線]] : 4車線

== 历史 ==
* 1979年7月25日：高辻出入口至大高出入口開通，可連接名四國道及知多半島道路。
* 1985年5月7日：鶴舞南JCT至高辻出入口開通，可連接都心環狀線。
* 1996年10月14日：大高出口啟用。
* 1997年3月27日：大高入口啟用。
* 2003年3月23日：大高出入口至名古屋南JCT開通，全線開通，可連接伊勢灣岸自動車道。
* 2004年12月1日：擴展星崎收費站。
* 2011年3月20日：在名古屋南JCT可連接名古屋第二環狀自動車道。
