'''名古屋高速16號一宮線'''（[[平假名|平假名]]：なごやこうそく16ごういちのみやせん）是一條連接[[愛知縣|愛知縣]][[清須市|清須市]][[清洲JCT|清洲JCT]]與同縣[[一宮市|一宮市]][[一宮中入口|一宮中入口]]的[[名古屋高速道路|名古屋高速道路]]路線。

== 概要 ==
* 起點 : 愛知縣清須市朝日
* 終點 : 愛知縣一宮市綠
* 路線長度 : 8.9km
* 出入口 : 10個（入口:5個、出口:5個）
* 分支 : 2個
* [[行車線|行車線]] : 4車線

== 連接高速道路 ==
* [[名神高速道路|名神高速道路]]（於[[一宮IC|一宮IC]]連接）
* [[名古屋第二環狀自動車道|名古屋第二環狀自動車道]]（於[[清洲JCT|清洲JCT]]連接）
* [[名古屋高速6號清須線|名古屋高速6號清須線]]（於[[清洲JCT|清洲JCT]]連接）
