'''名塚佳織'''{{BD|1985年|4月24日|||Nazuka, Kaori}}是[[日本|日本]][[東京都|東京都]]出身的女性[[聲優|聲優]]。血型B型，身高162cm。目前並沒有隸屬任何經紀公司。

主要演出作品有《[[.hack|.hack系列]]》（志乃）、《[[True_Tears|True Tears]]》（湯淺比呂美）、《[[交響詩篇艾蕾卡7|交響詩篇艾蕾卡7]]》（艾蕾卡）、《[[搞笑漫畫日和|搞笑漫畫日和]]》、《{{tsl|ja|『銀色のオリンシス|銀色的奧林西斯}}》（テア）、《[[反叛的魯路修|反叛的魯路修]]》（娜娜莉·蘭佩洛基）等<ref>{{Cite web|date=|url= http://blog.excite.co.jp/kamosan|title= 名塚佳織 公式ブログ 〜かもさんのらくがき帳〜|accessdate=2009-09-30}}</ref>。

==人物介紹==
*暱稱的「{{lang|ja|かもさん}}」由來：就讀幼稚園時，[[NHK|NHK]]教育台播放「鴨子的橋」這首歌的時候會跟著一起唱唱跳跳，因此被父親叫做「{{lang|ja|かもさん}}」。
*從小學生開始就以童星之姿在舞台上演出。也有如音樂劇『トラップ一家物語』等，擔任多個角色連續演出多場的作品。
*在1998年的音樂劇《{{lang|ja|フラワー}}》中，除飾演{{lang|ja|サイヤ}}以外，也擔任女主角{{lang|ja|メズーラ}}的代班演員。前來觀賞同台演出的[[齋藤彩夏|齋藤彩夏]]的動畫導演[[大地丙太郎|大地丙太郎]]因此留下了印象。在導演的另一部作品[[反斗小王子|反斗小王子]]中客串演出子石的角色，是為聲優出道作。
*2006年3月從[[桐朋学园艺术短期大学|桐朋學園短期大學]]演劇學科畢業。
*以前由母親擔任經紀人，現在則全部自己一手包辦。
*家庭成員有爸爸、媽媽、姊姊和一隻貓（MOKO）。
*有「就算連續三天喝到天亮也沒事」的好酒量。
*喜歡的音樂是以J-POP為主。
*曾學過七年的[[小提琴|小提琴]]，是一項才藝。
*雖然本人似乎沒有注意到，在廣播節目中常用「{{lang|ja|てなわけで}}」、「{{lang|ja|そんなわけで}}」。
*2011年2月21日，在其个人官方博客上公开结婚消息<ref>{{ja}}{{Cite web |url=http://blog.excite.co.jp/kamosan/14948323/|title=支えて下さる皆様へ　感謝の気持ちを込めて…(满怀感激的心情 向一直支持我的诸位宣布…)|accessdate=2012-01-01}}</ref>。同年12月31日，又于其个人官方博客上宣布已育有一女<ref>{{ja}}{{Cite web |url=http://blog.excite.co.jp/kamosan/17506142/|title=あけましておめでとうございます(恭贺新年快乐)|accessdate=2012-01-01}}</ref>。
