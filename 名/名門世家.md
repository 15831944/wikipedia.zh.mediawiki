'''名門世家'''（{{lang|en|Windsor Arch}}）是[[澳門|澳門]]一座興建中的豪宅樓盤，由新建業集團及威得利企業發展股份有限公司共同發展的地產物業項目，位於[[氹仔|氹仔]][[澳門賽馬會|賽馬會]]對面的背山地段，預計於2010年落成入伙。

名門世家項目背山而建，由10座樓高47層的住宅大廈組成，面向澳門賽馬會，背向[[葡京花園|葡京花園]]，共提供約1,500個住宅單位，而位於每座物業頂層的天際別墅共19間，並設有多層[[停車場|停車場]]、平臺花園、住客[[會所|會所]]等設施。<ref name="md-080520">[[澳門日報|澳門日報]] 2008年5月20日，《樓盤走出澳門全球巡迴推介》</ref>

== 設計概念 ==
名門世家是一個鑑賞級豪宅，以[[英國|英國]]皇室貴族曾入住的Brocket Hall為設計概念，整個屋苑充滿[[十五世紀|十五世紀]]英倫風味。

十座大樓頂層設有共19間的天際別墅，每幢「別墅」都有一個獨立名字，並設有私人[[泳池|泳池]]。而且屋苑設有雙會所，除20萬方呎的住戶會所，更在45層設立1.5萬方呎的世界華商會館，包括休閒度假、宴會餐飲、會議展覽及綜合商務中心等。<ref>[http://hk.realestate.yahoo.com/080202/299/2oevg.html 氹仔名門世家 馬場景成賣點] {{webarchive|url=https://web.archive.org/web/20080328022231/http://hk.realestate.yahoo.com/080202/299/2oevg.html |date=2008-03-28 }}</ref>

== 設施 ==
名門世家一共提供1629個居住單位以及52個商店鋪位；停車場提供約1406個私家車車位，及396個摩托車車位；而兩個會所分別稱為「名門會所」及「尊戶會所」。“名門會所”面積約20萬方呎；備有戶內戶外雙泳池；「尊戶會所」則位於五至七座「WindsorTower」的四十四樓；而其他設施包括服務泳池、健身房、豪華大堂、水療室、約500米共5層高的裙樓商場等。

== 宣傳 ==
2008年5月，發展商一連三日在[[英國|英國]][[倫敦|倫敦]]舉行了第一站全球推介會，吸引近200名來自世界各地包括[[香港|香港]]、澳門、[[台灣|台灣]]、[[倫敦|倫敦]]的買家、銀行業、地產業代表、傳媒以至多位海外政要出席。<ref name="md-080520">[[澳門日報|澳門日報]] 2008年5月20日，《樓盤走出澳門全球巡迴推介》</ref>

名門世家的發展商非常注重宣傳活動，由表演活動、頒獎禮以至全球宣傳，整個樓盤的宣傳費暫定花費2億元。<ref>[http://property.mpfinance.com/cfm/pa3.cfm?File=20080430/paa01/lan1.txt 澳名門世家 下月全球推廣]</ref>

== 樓高爭議 ==
「名門世家」由原本不高於16層，隨後獲政府批准增建至47層高，遮擋背後山腰「玫瑰山莊」及「葡京花園」的景觀，由[[林鳳娥|林鳳娥]]發起組織「反對名門世家興建超越合法高度居民委員會」，以「名門世家」違反《都市建築總規章》，向特首辦和[[澳門廉政公署|澳門廉政公署]]遞請願信，要求當局調查整個批給程序<ref>[http://property.hk.msn.com/app/msnHK/article/view.html?articleId=20941&seriesId=17 市民循法律阻名門世家超高] {{webarchive|url=https://web.archive.org/web/20080629064152/http://property.hk.msn.com/app/msnHK/article/view.html?articleId=20941&seriesId=17 |date=2008-06-29 }}</ref>。
