'''名古屋高速1號楠線'''（[[平假名|平假名]]：なごやこうそく1ごうくすのきせん）是一條連接[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[東區_(名古屋市)|東區]][[東片端JCT|東片端JCT]]與同縣同市[[北區_(名古屋市)|北區]][[楠JCT|楠JCT]]的[[名古屋高速道路|名古屋高速道路]]路線。

== 概要 ==
* 起點 : 愛知縣名古屋市東區[[東片端|東片端]]
* 終點 : 愛知縣名古屋市北區大我麻町
* 路線長度 : 5.6km
* 出入口 : 5個（入口:3個、出口:2個）
* 分支 : 2個
* [[行車線|行車線]] : 4車線

== 連接高速道路 ==
* [[名古屋高速都心環狀線|名古屋高速都心環狀線]]（於[[東片端JCT|東片端JCT]]連接）
* [[名古屋第二環狀自動車道|名古屋第二環狀自動車道]]（於[[楠JCT|楠JCT]]連接）
* [[名古屋高速11號小牧線|名古屋高速11號小牧線]]（於[[楠JCT|楠JCT]]連接）

== 历史 ==
* 1988年12月21日 : （臨時）楠出入口至萩野出入口開通。
* 1991年3月19日 : 東名阪自動車道（均一區間）（現：名古屋第二環狀自動車道）勝川IC至[[清洲東IC|清洲東IC]]開通，楠出入口至楠JCT也同時開通。
* 1994年11月16日 : 路線延長工程，全線關閉。
* 1995年9月19日 : 全線重開，並且新區間東片端JCT至萩野出入口開通，同日，萩野出入口停用。
* 1997年10月13日 : 黑川出入口啟用。
* 2001年3月11日 : 楠JCT可以連接11號小牧線。
