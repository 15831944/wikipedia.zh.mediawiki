'''名古屋高速5號萬場線'''（[[平假名|平假名]]：なごやこうそく5ごうまんばせん）是一條連接[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[中區_(名古屋市)|中區]][[新洲崎JCT|新洲崎JCT]]與同縣同市[[中川區|中川區]][[名古屋西JCT|名古屋西JCT]]的[[名古屋高速道路|名古屋高速道路]]路線。

== 概要 ==
* 起點 : 愛知縣名古屋市中區
* 終點 : 愛知縣名古屋市中川區
* 路線長度 : 6.8km
* 出入口 : 3個（入口:3個、出口:3個）
* 分支 : 2個
* [[行車線|行車線]] : 4車線

== 連接高速道路 ==
* [[名古屋高速都心環狀線|名古屋高速都心環狀線]]（於[[新洲崎JCT|新洲崎JCT]]連接）
* [[名古屋高速2號東山線|名古屋高速2號東山線]]（於[[新洲崎JCT|新洲崎JCT]]連接）
* [[名古屋第二環狀自動車道|名古屋第二環狀自動車道]]（於[[名古屋西JCT|名古屋西JCT]]連接）
* [[東名阪自動車道|東名阪自動車道]]（於[[名古屋西JCT|名古屋西JCT]]連接）

== 历史 ==
* 1986年10月27日：新洲崎JCT至名古屋西JCT開通，全線開通，當時在新洲崎JCT只可連接名高速2號線。
* 1988年4月26日：新洲崎JCT可以連接都心環狀線。
