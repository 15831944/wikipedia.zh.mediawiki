'''点对点网络托管'''为使用[[点对点网络|点对点网络]]来分发所需的[[网页|网页]]。<ref>{{cite web|url=http://www.activewebhosting.com/faq/legal-p2p.html|title=Peer-To-Peer File Sharing|accessdate=3 November 2013|publisher=Active Web Hosting}}</ref>此概念与使用专用[[网页服务器|网页服务器]]向最终用户客户端共享网页数据的[[主從式架構|主從式架構]]不同，對等網路代管的表现类似Dijjer和{{le|Coral Cache}}的[[内容分发网络|内容分发网络]]，允许用户保存单个网页的数据副本和将该缓存分发给其他用户，从而加快访问高峰时的速度和减少专用服务器的带宽需求。{{Fact|date=July 2014}}
