'''對高岳車站'''位於[[台灣|台灣]][[南投縣|南投縣]][[信義鄉|信義鄉]]，為[[行政院農業委員會林務局|林務局]][[阿里山森林鐵路|阿里山森林鐵路]]祝山線之[[鐵路車站|鐵路車站]]。

== 車站構造 ==
岸式簡易月台一座

== 利用狀況 ==

2017年7月開始，每週一的郵輪式列車停靠本站

== 車站週邊 ==
*對高岳觀日坪
*對高岳登山歩道

== 歷史 ==
*1986年：祝山線通車而啟用。
*據說於同年2月13日因出軌意外而停用，廢站時間與原因有待考證。
*2017年：成為郵輪式列車停靠站
