《'''对你的泪水如此爱恋'''》（{{lang-ja|'''君の涙にこんなに恋してる'''}}）是[[夏色_(乐队)|夏色]]的第一张单曲。2012年8月29日由[[GIZA_studio|GIZA studio]]／D-GO发售。

== 概要 ==
森川七月率领的音乐团体的出道单曲，以及D-GO发售的第一张单曲，为[[读卖电视台|ytv]]制作・[[日本电视台|日本电视台]]系《[[名侦探柯南_(动画)|名侦探柯南]]》片头曲<ref name = natalie>[http://natalie.mu/music/news/72396 3人組女性ユニット・なついろ、「コナン」主題歌でデビュー]</ref>。初回盘作为名侦探柯南盘，制作成为TV版本，同时主要角色的[[毛利兰|毛利兰]]在海岸用手指描画歌手名和主题曲<ref>[http://www.cdjournal.com/main/news/natsuiro/46252 なついろ、「名探偵コナン」オープニングテーマでデビュー] ''CDジャーナル''</ref><ref name = bill>[http://www.billboard-japan.com/d_news/detail/6182 『名探偵コナン』OPのコナン盤ジャケット写真解禁 ] ''Billboard Japan''</ref>。在由D-GO发售的作品中，此曲是第一次进入公信榜周间排位100位之内。
