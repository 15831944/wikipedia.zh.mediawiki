{{校对翻译}}
在[[经济学|经济学]]中，'''资源分配'''（resource allocation）是指将可用的[[资源|资源]]分配到各种用途。在整个经济背景下，资源可以通过各种方式分配，例如[[市场|市场]]或{{le|中央计划|Central planning}}。

在[[项目管理|项目管理]]中，资源分配或者{{le|资源管理|Resource management}}（resource management）是在考虑资源可用性和项目时间的情况下，安排活动所需的活动时间和资源<ref>{{cite web|author=Name: * |url=http://www.pmhut.com/pmo-and-project-management-dictionary |title=PMO and Project Management Dictionary |publisher=Pmhut.com |date=2007-08-20 |accessdate=2014-06-24}}</ref>。

== 经济学 ==
在[[经济学|经济学]]中，[[公共財政|公共財政]]涉及三个广泛的领域：[[宏观经济学|宏观经济稳定]]、[[收入分配|收入]]与财富分配，以及资源分配。

对资源分配的大部分研究致力于找出哪些特定的资源分配机制能导致[[帕累托最优|帕累托最优]]的出现。在这种条件下，不损害另一方的利益，就不能改善任何一方的状况。
== 战略规划 ==
在[[战略计划|战略计划]]中，资源分配是利用可用[[自然资源|资源]]的{{tsl|en|plan||计划}}。以[[人力资源|人力资源]]为例，常见目标是在短期内实现某个或某些未来目标。这是在各[[项目|项目]]与业务单位之间分配稀缺资源的一个过程。

解决资源分配问题有着多种方法，包括手动方法<ref>CEITON technologies [http://www.ceiton.de/CMS/EN/scheduling/tasks.html#Manual_Planning "Manual planning"] ''{{tsl|en|Ceiton technologies}}'' Retrieved on 3 June 2014</ref>、算法计算（见下文）<ref>Production planning [http://www.productionplanning.com/Production-Planning/Algorithmic-Sequencing.aspx "Algorithmic sequencing"] ''Production planning'' Retrieved on 3 June 2014</ref>或两者结合。<ref>Inteng [http://www.inteng.com.au/planning.htm "Planning manual and algorithm"] {{Wayback|url=http://www.inteng.com.au/planning.htm |date=20140314180315 }} ''Inteng'' Retrieved on 3 June 2014</ref>

项目中可能包含应急机制，诸如需要时排除出计划的优先顺序，以及有更多资源可用时应纳入计划的项目优先级<ref>{{cite web|author=Samuel90 |url=http://www.slideshare.net/kbgluciaja/resource-allocation-16175159 |title=Resource allocation |publisher=Slideshare.net |date=2013-01-25 |accessdate=2014-06-24}}</ref>。

== 算法 ==
{{TransF}}
特定领域可能会使用[[计算机程序|计算机程序]]来完成资源分配，并自动、动态分配资源给申请人。

这在[[路由|路由]]和[[通信|通信]]用途的[[电子设备|电子设备]]中尤为常见。例如，[[無線通訊|無線通訊]]中的信道分配可由[[基地收发机站|基地收发机站]]采用[[算法|算法]]决定。<ref>{{cite web|url=http://www.mit.edu/~modiano/papers/C67.pdf |title=Wireless Channel Allocation Using An Auction Algorithm |format=PDF |date= |accessdate=2014-06-24}}</ref>

还有某些资源是以申请人的经济实力争得最好的资源，例如[[网上拍卖|网上拍卖]]（另见{{tsl|en|auction theory|拍卖理论}}）。

在[[中央处理器|CPU]][[抢占式多任务处理|抢占式多任务处理]]分配的一篇论文中<ref>{{cite web|url=http://www.citeulike.org/user/zflavio/article/1043948 |title=Tycoon: A Distributed Market-based Resource Allocation System |publisher=Citeulike.org |date=2004-04-05 |accessdate=2014-06-24}}</ref>，拍卖算法与{{tsl|en|proportional share scheduling|按比例共享调度}}经受了比较。
