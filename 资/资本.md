在[[经济学|经济学]]，'''资本'''（[[法语|法语]]、[[英语|英语]]、[[西班牙语|西班牙语]]: Capital，[[德语|德语]]: Kapital）， 是一種[[生产要素|生产要素]]，為用于[[生产|生产]]的[[耐久財|耐久財]]，即资金、厂房、设备、材料等物质资源，用來生產其他商品或產生收入的累積物力與財務資源<ref>[http://db1x.sinica.edu.tw/caat/caat_rptcaatc.php?_op=?SUBJECT_ID:300138242 藝術與建築索引典—資本（經濟概念）] {{webarchive|url=https://web.archive.org/web/20111122163404/http://db1x.sinica.edu.tw/caat/caat_rptcaatc.php?_op=%3FSUBJECT_ID%3A300138242 |date=2011-11-22 }} 於2011 年3 月28 日查閱</ref>。在[[金融学|金融学]]和[[会计|会计]]领域，资本通常用来代表金融[[财富|财富]]，特别是用于[[经商|经商]]、兴办企业的金融资产。

广义上，资本也可作为[[人类|人类]]创造[[物质|物质]]和精神[[财富|财富]]的各种社会经济资源的总称。

簡單而言，资本可以分为三类：
# [[制度|制度]]或社会生产关系资本。它的提升或增值由社会政治思想等变革来实现；
# [[人力|人力]]资本；
# 物力资本，包括自然赋予的和人类创造的两种。

[[亚当·斯密|亚当·斯密]]在《[[国富论|国富论]]》認為，資本因節儉而增加，因浪費和錯誤運用而減少。因此資本具有儲備的特質，如果因為錯置而用來維持非生產性的項目（如豪奢的暴飲暴食、過度的軍費開支），那麼將資本引入歧途的結果，便是全國的生產價值遭到減損。亚当·斯密也认为，“大國不會因為私人的奢華和誤用資本而陷於貧窮，反而是政府的浪費和措施失當，時常會導致這種後果”。

==狹義和廣義的資本==
在[[古典經濟學|古典經濟學]]和[[新古典經濟學|新古典經濟學]]中，資本是[[生产要素|生产要素]]中的一項，此外的生产要素還包括[[土地_(經濟學)|土地]]及[[勞動|勞動]]。在古典經濟學中，所有其他的資本均稱為[[無形資產|無形資產]]，包括組織、企業家精神、知識、信誉及管理等，有些經濟學則將其分類為[[才能|才能]]、[[社会资本|社会资本]]或{{link-en|教学资本|instructional capital}}等。

具有以下特性的[[商品|商品]]即為资本：

*在製造其他商品時會使用到（因此是生产要素中的一項）。
*不會變為其他商品中的一部份（和原料或半成品不同），在製造過程中也不會將此商品耗盡（不過资本會有[[折舊|折舊]]．也是企業支出的一部份）。
*需要經過生產製造流程才能得到（和生产要素中的「土地」不同，後者是指自然存在的資產，例如礦產）。

也有一種用{{link-en|存量及流量|Stock and flow}}定義资本的方式，资本是存量，可以估計任何特定時間點下的數值和金額。[[投资#經濟或總體經濟方面|投资]]是要加到资本存量中的金額，會用一段時間（例如「每年」）中發生的數值來表示．因此是一種流量。

在[[马克思主义政治经济学|马克思主义政治经济学]]中，資本可以分為以下的形式：
* [[不變資本|不變資本]]，指转变为生产资料的那部分资本，由于其在生产过程中并不改变自己的价值量，因此被称为不变资本。
* [[可變資本|可變資本]]，是指转化为劳动力的那部分资本，其成本會依照[[工资|工资]]而變動，而且只有在雇主和勞工的雇用關係存在時才需支付。它再生产自身的等价物和一个超过这个等价物而形成的余额——[[剩余价值|剩余价值]]。这个剩余价值本身是可以变化的，是可大可小的。这部分资本从不变量不断转化为可变量。因此，这部分资本被称为可变资本。
* {{link-en|虚拟资本|Fictitious capital}}，是指[[无形资产|无形资产]]或是像[[股票|股票]]、[[債券|債券]]及其他[[證券|證券]]等可交易的{{link-en|账面财富|Paper wealth}}。

早期會將資本視為是有實體的物品，例如在生產製造過程中用到或所需的工具、車輛及建築物。不過1960年代以後經濟學家開始注意更廣義的資本。例如在技術提昇或教育上的投資會視為是提昇[[人力資本|人力資本]]或是[[知識資本|知識資本]]，在[[智慧財產|智慧財產]]上的投資則視為提昇[[智慧資本|智慧資本]]，不過這些更廣義的資本也帶來了一些疑問或是爭議。

==現代資本的分類==
許多理論及實務上都會將資本分類，大致可以分為以下幾類：
*{{link-en|金融資本|Financial capital}}，像[[股票|股票]]、[[債券|債券]]等，會以{{link-en|资本资产|Capital asset}}的形式出現，可以在金融市場中交易。其市場價值不是以其累計的投資金額為準，而是以其市場對於未來[[盈餘|盈餘]]及風險的預期值為準。
*{{link-en|自然資本|Natural capital}}，是指固有的自然[[生態|生態]]，提供生活所需及其他資源，例如河流可以提供農田水源。
*[[社会资本|社会资本]]，在私人企業中常會視為是[[商譽|商譽]]或[[品牌價值|品牌價值]]，不過社会资本是一較廣的概念．是指為實現工具性或情感性的目的，透過社會網路來動員的資源或能力的總和。
* {{link-en|教學資本|Instructional capital}}，一開始在學術上定義是指有關教學及將知識轉移到原來不具備相關知識的個體，像[[知識資本|知識資本]]及[[智慧資本|智慧資本]]也是描述類似的概念．但是其定義和上述學術定義不完全相同，在會計處理上也還沒有一定的共識。
*[[人力資本|人力資本]]，泛指社会资本、教學資本及個人才能的總和。在技术经济学（technical economic）中會用人力資本來定義平衡成長，也就是人力資本和金融資本的同步提昇。

== 与资产的关系 ==
亚当·斯密在《国富论》第二篇的开端，对资本和[[资产|资产]]有所描述。他认为，属于某人的财物，就是这个人的资产。出让使用权，以获取利润收益的资产，就是资本。资本的使用方法有两种，一种是买进[[原料|原料]]后加工，将[[制成品|制成品]]卖出去获利，或者是买进商品后转售获利，这样的资本被称作[[流动资本|流动资本]]。第二种是买进土地、机器设备等，这些不通过转手就能获利的资本，叫[[固定资本|固定资本]]。

== 資本禀赋 ==
禀赋（Endowment）是某一事務在處理前的自然狀態，生產製造過程讓禀赋變成為資本。資本可以區分為自然資本及其他，禀赋也可以區分為自然禀赋或是人口禀赋<ref>{{Cite web |url=http://www.econlib.org/cgi-bin/searchbooks.pl?searchtype=BookSearchPara&id=bbPTC&query=endowment |title=Search Books "endowment" |access-date=2013-01-26 |archive-url=https://web.archive.org/web/20140427084010/http://www.econlib.org/cgi-bin/searchbooks.pl?searchtype=BookSearchPara&id=bbPTC&query=endowment |archive-date=2014-04-27 |dead-url=yes }}</ref>。
