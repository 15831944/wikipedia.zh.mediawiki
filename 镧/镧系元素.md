'''镧系元素'''是第57号元素[[镧|镧]]到71号元素[[镥|镥]]15种[[元素|元素]]的统称。镧系元素的外层和次外层的[[电子构型|电子构型]]基本相同，电子逐一填充到4[[f轨道|f轨道]]上。镧系元素也属于[[过渡元素|过渡元素]]，只是镧系元素新增加的电子大都填入了从外侧数第三个[[电子层|电子层]]（即4f电子层）中，所以镧系元素又可以称为4f系。为了区别于[[元素周期表|元素周期表]]中的d区过渡元素，故又将镧系元素（及[[锕系元素|锕系元素]]）称为[[内过渡元素|内过渡元素]]。由于镧系元素都是[[金属|金属]]，所以又可以和[[锕系元素|锕系元素]]统称为f区金属。镧系元素用符号'''Ln'''表示。

所有镧系元素既能生成化学性质类似的三价化合物，个别镧系元素也能生成比较稳定或不很稳定的四价或二价化合物，所以15个元素的化学性质并不完全相似，在光学、电磁学等物理性质也有较大的差别。

镧系元素[[原子|原子]][[基态|基态]]的[[电子构型|电子构型]]是4f<sup>0~14</sup>5d<sup>0~1</sup>6s<sup>2</sup>。

== 争论 ==

[[镥|镥]]（Lu）是否属于镧系元素有争论。{{citation needed|最近的[[色谱分析|色谱分析]]显示，镥和[[铹|铹]]应属于IIIB族，而非镧系和锕系。}}但为了方便叙述，现今仍习惯将其与镧系合称。参见[[锕系元素|锕系元素]]中关于铹的地位的争论。

== 氧化态 ==

镧系元素在[[固态|固态]]、[[水|水]][[溶液|溶液]]中或其他[[溶剂|溶剂]]中的特征[[氧化态|氧化态]]是+3。由于镧系元素在[[气态|气态]]时，失去两个6s电子和一个5d电子或失去两个6s电子和一个4f电子所需的[[电离能|电离能]]比较低，所以一般能形成稳定的+3氧化态。除+3特征氧化态外，镧系元素还存在着一些不常见的氧化态。例如：[[铈|铈]]、[[镨|镨]]、[[钕|钕]]、[[铽|铽]]、[[镝|镝]]存在+4氧化态，原因是它们的4f层保持或接近全空、半满或全充满的状态比较稳定，但只有+4氧化态的[[铈|铈]]能存在于[[溶液|溶液]]中，它是很强的[[氧化剂|氧化剂]]。

同理，铈、钕、[[钐|钐]]、[[铕|铕]]、[[铥|铥]]、[[镱|镱]]还存在+2氧化态。

== 原子半径和离子半径（镧系收缩） ==

与同[[族_(化学)|族]]的[[钪|钪]]、[[钇|钇]]、[[镧|镧]][[原子|原子]]半径逐渐增大的规律恰恰相反，从[[铈|铈]]到[[镥|镥]]则是逐渐减小。这种镧系元素的原子半径和离子半径随[[原子序数|原子序数]]的增加而逐渐减小的现象称为'''[[镧系收缩|镧系收缩]]'''。

== 磁性 ==
镧系元素具有顺磁性（鑭的化合物除外），鑭的化合物呈反磁性。
