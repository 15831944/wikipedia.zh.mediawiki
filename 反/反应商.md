在[[化学|化学]]中，一个'''反应商'''（{{lang-en|'''Reaction quotient'''}}）Q<sub>r</sub>是一个[[活性度|活性度]]或反應內化学物[[浓度|浓度]]的[[函數|函數]]。在[[化学平衡|化学平衡]]這個特殊情况下，该反应的反应商等于[[平衡常数|平衡常数]]。

== 定義 ==
在一个一般的化学反应中，反应物A份量為α[[摩爾|摩爾]]，反应物B份量為β摩爾，给出產物S為σ摩爾及产物T為τ摩爾則化學方程可以写成
:αA + βB {{eqm}} σS + τT

雖然在许多情况下，反应物可能接近完全反应，但該方程仍被寫成一個平衡。混合A和B並允许反应时，反应商，''Q<sub>r</sub>''定义为<ref>{{cite book | author=Zumdahl, Steven; Zumdahl, Susan | title=Chemistry 6th Edition| publisher=Houghton Mifflin Company | year=2003 | isbn=0-618-22158-1}}</ref>:

:<math>Q_r = \frac{\left\{S_t\right\}^\sigma \left\{T_t\right\}^\tau }{\left\{A_t\right\}^\alpha \left\{B_t\right\}^\beta } </math>

当中{''X''<sub>t</sub>}表示''X''在時間''t''的''瞬时''[[活性度|活性度]]。<ref>在某些情况下（见[[化学平衡|化学平衡]]）某个项，例如{''A''}，可能以一个长期的浓度[''A'']取代，那么反应商与平衡常数都尽浓度商。</ref>一个較複雜的定义是：

:<math>Q_r=\frac{\prod_j a_j^{\nu_j}}{\prod_i a_i^{\nu_i}}</math> 

當中''П<sub>j</sub>''是在所有的含有下标''j''的变量的积，对''П<sub>i</sub>''亦同样，該分數的分子，是反应物的活性至次方为[[反应系数|反应系数]]''ν<sub>j</sub>''後相乘，而分母则为对产物的相似做法。所有的活性皆指时间''t''。

随着时间过去，假设没有[[能量障碍|能量障碍]]的情况下，反应与生成物发生反应，因此反应商数会改变。最后，化物的活性度成为常数，這個時候混合物便可以说成达至平衡。因此，[[平衡常数|平衡常数]]''K''则可以符号写成为
:'''K''' = '''Q<sub>r</sub>'''（t→∞）
实际上，在大部分反应中，混合物會在有限时间内达至化学平衡。

== 與平衡位置關係 ==
* 如果Q<sub>r</sub> < K :反應向右移，即是[[正反應|正反應]]速度高於[[逆反應|逆反應]]的。
* 如果Q<sub>r</sub> > K :反應向左移，即是[[逆反應|逆反應]]速度高於[[正反應|正反應]]的。
* 如果Q<sub>r</sub> = K :反應到達[[化學平衡|平衡]]狀態

根據范特霍夫等温公式推導，反應商Q<sub>r</sub>與[[吉布斯能|吉布斯能]]（ΔG）及標準形態的吉布斯能（ΔG<sup><s>0</s></sup>）的關係如下：
   
:ΔG = ΔG<sup>0</sup> + RT ln Q<sub>r</sub>
