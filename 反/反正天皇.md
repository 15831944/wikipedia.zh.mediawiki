'''反正天皇'''（{{jpn|j=反正天皇|hg=はんぜいてんのう|rm=Hanzei Tennō}}；？-410年1月23日）是[[日本|日本]]第18代[[天皇|天皇]]（在位406年1月2日-410年1月23日），名'''瑞齒別尊'''（{{jpn|j=瑞歯別尊|hg=みつはわけのみこと|rm=Mitsuhawake no mikoto}}）、'''多遲比瑞歯別尊'''（たじひのみずはわけのみこと），是[[古墳時代|古墳時代]]的天皇。

== 系譜 ==
[[仁德天皇|仁德天皇]]的第三皇子。母親是磐之媛命（{{lang|ja|いわのひめのみこと}}）。
[[履中天皇|履中天皇]]的同母弟，[[允恭天皇|允恭天皇]]的同母兄。以下是其-{后妃}-和子嗣列表：

* 皇夫人：津野媛（{{lang|ja|つのひめ}}）
** 香火姫皇女（{{lang|ja|かひひめのひめみこ}}）
** 円皇女（{{lang|ja|つぶらのひめみこ}}）
* 妃：弟媛（{{lang|ja|おとひめ}}）
** 財皇女（{{lang|ja|たからのひめみこ}}）
** 高部皇子（{{lang|ja|たかべのみこ}}）

== 略歴 ==
406年於履中天皇駕崩後即位。

在位5年後，於410年駕崩。《[[古事記|古事記]]》記載其時年60。

== 諡号・追号・異名 ==
別名瑞齒別天皇（{{lang|ja|みつはわけのすめらみこと}}）。

《古事記》以水齒為別名（みずはわけのみこと）。

中國《[[宋書|宋書]]》及《[[梁書|梁書]]》記載中的[[倭五王|倭五王]]，倭王讚為[[仁德天皇|仁德天皇]]或[[履中天皇|履中天皇]]，反正天皇為倭王珍的可能性甚高。

== 宮 ==
都在河內丹比柴籬宮（たじひのしばかきのみや、[[大阪府|大阪府]][[松原市_(日本)|松原市]]上田7丁目的柴籬神社）。

== 陵墓・靈廟 ==
葬於百舌鳥耳原北陵（もずのみみはらのきたのみささぎ）<br />
[[大阪府|大阪府]][[堺市|堺市]]北三國中丘町2丁目的田出井山古墳（前方後墳，全長148m）。
