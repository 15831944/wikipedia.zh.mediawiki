'''反中'''，又稱'''反中國'''、'''反中國因素'''、'''反陸'''、'''反共'''（'''共'''指[[中國共產黨|中國共產黨]]），是指對於[[中華人民共和國|中華人民共和國]]的厭惡和反感情緒，反對其[[中华人民共和国政府|政府]]的所作所為、[[中國特色社会主义|社会制度]]與對外影響力。對中國政府的反感，可能延伸为對於其[[大陆人|人民]]言論及行為的反感。

[[中華人民共和國政府|中華人民共和國政府]]與[[中華民國政府|中華民國政府]]都自稱是代表[[中國|中國]]的唯一合法政府，但現今[[中華民國|中華民國]]在國際上不被广泛承认，且不再堅持擁有[[中國大陸|中國大陸]]主權，只能代表[[台澎金马|台澎金马]]，所以“反中”一般是指反[[中華人民共和國|中華人民共和國]]。因为目前[[中華人民共和國政府|中華人民共和國政府]]由[[中國共產黨|中國共產黨]]执政，所以“反中”者一定是反共的，但“反共”者不一定就是“反中”的。

由于反中与[[反華|反華]]在英语中都是Anti-Chinese（意指反中國人或反華，但容易涉入[[種族歧視|種族歧視]]），加上外貌相似容易混淆，國際上并不对[[中国人|中国人]]與[[华人|华人]]加以区分，而[[中華人民共和國|中華人民共和國]]又將反對[[中华人民共和国政府|其]][[政權|政權]]或政策的反中論述或行为，劃為反華行為，所以反中与反华两个词常常被混為一談。但是對於中國的反對情緒，有时並不一定會延伸到成為對於華人或是[[中華文化|中華文化]]整體的反感情緒。

== 反中理由 ==
{{see also|中國威脅論|中國因素}}
反中人士有許多不同的理由，其中最主要的理由是，反对[[中國共產黨|中國共產黨]]在中國的[[一黨執政|一黨執政]]<ref>专访唐柏桥：中共何时垮？谁执政？[http://www.epochtimes.com/gb/5/6/7/n947752.htm]</ref><ref>叶孝刚：一党专政势必淘汰[http://beijingspring.com/bj2/2010/140/2015215191406.htm]</ref>及[[專制|專制]][[獨裁|獨裁]]統治，以致其國內的[[民主|民主]]與[[人權|人權]]長期受到打壓，[[異見人士|異見人士]]被拘捕以及人身安全受到威脅；反对中國對於內部[[分離主義|分離主義]]的強力壓制，[[西藏獨立運動|西藏獨立運動]]、[[新疆独立运动|新疆独立运动]]、[[香港独立运动|香港独立运动]]等人士，皆向國際進行訴求，希望獲得更大自治權力，对[[中國政府|中國政府]]在國際上的形象受损。

另外，部份反中人士“反中”的原因是基于反對[[共產主義|共產主義]]，出于對於[[中國共產黨|中國共產黨]]的反感，如[[日本右翼|日本右翼]][[石原慎太郎|石原慎太郎]]在2010年接受《[[南方人物週刊|南方人物週刊]]》訪問時，就宣稱他[[反中共|反對中國共產黨]]，但喜愛中國傳統文化，並不反華<ref>{{cite news|url=http://www.epochtimes.com/b5/10/10/19/n3058652.htm|title=南方人物專訪石原慎太郎：反共不反華|publisher=[[大紀元|大紀元]]|date= 2010-10-18|author=}}</ref>。

在中國經濟持续成長之下，中國對外力量大增，周圍國家對於中國以[[國家資本主義|國家資本主義]]進行对外扩张的憂慮增加<ref name = "反中"/>，又對外長期傾向[[霸權主義|霸權主義]]與[[新帝國主義|新帝國主義]]，其帶有侵略性的[[民族主義|民族主義]]情緒，經常以武力威胁周邊國家，與周邊國家有領土上的爭議，使各國內部產生[[中国威胁论|中国威胁论]]的論點。例如台灣[[太陽花學運|太陽花學運]]，反映台灣年輕人對於中國威脅的焦慮，特別是中國透過政治及經濟力量來影響台灣內部的民主發展<ref name="反中"> [http://buzzorange.com/2015/04/16/the-younger-people-see-about-china-and-taiwan/ 台灣年輕人普遍陷入焦慮，促使「反中護台」意識高漲]</ref>。2012年，台灣[[民主進步黨主席|民主進步黨主席]]、前總統候選人[[蔡英文|蔡英文]]與中國大陸流亡異議人士[[王丹|王丹]]進行對談時，蔡英文就曾提出民進黨不反中，但是反對中國霸權與不民主<ref>{{cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20120325/34113204/|title=蔡英文：民進黨不反中|publisher=[[蘋果日報|蘋果日報]]|date= 2012-03-24|author=}}</ref>。

中國遊客在觀光時，其粗魯及不文明行為，造成國際上對於中國人的觀感不佳。如香港的[[蝗蟲論|蝗蟲論]]，這也造成了反中的情緒增長。

== 对反中的界定与解释 ==
反對中華人民共和國，不必然等於反對或不認同[[華人|華人]]或[[中華文化|中華文化]]，但常常會被等同於[[反華|反華]]，因為[[中華人民共和國政府|中華人民共和國政府]]在國際上大多數國家認為是中國的唯一合法政府，是中國與全球華人的唯一代表，因此中華人民共和國政府常以「境外反華勢力」一詞，將反對其政權或政策的論述，擴大劃為對中國、中國人民、中國文化或中華文化的全盤反對行為。不認同中華人民共和國為中國或中華唯一代表的人士，也經常將反對中華人民共和國政府以及中國政府不支持的任何舉動，皆等同於反中、反華，例如支持[[西藏獨立運動|西藏獨立運動]]或[[台灣獨立運動|台灣獨立運動]]。在香港，部分支持及擁護中共政權的[[親中共|親共]]人士，設立一個名為「反泛民，救香港」網站，將[[泛民主派|泛民主派]]人士列為反華的「[[漢奸|漢奸]]人渣」、「反中亂港」<ref>{{cite news|url=http://hk.apple.nextmedia.com/news/art/20130309/18189484|title= 將不同意見視為反中亂港 極左網站打壓民主 |publisher=[[蘋果日報|蘋果日報]]|date= 2013-03-29|author=林俊謙}}</ref>，這是另一例子。

台灣[[中央研究院|中央研究院]]社會學所副研究員[[吳介民|吳介民]]出版的《第三種中國想像：中國因素與台灣民主》中，認為[[中國因素|中國因素]]威脅了台灣民主發展。但他主張將中國政府與中國人民分開對待，認為應在傾中與反中兩個方向中，建立[[第三條道路|第三條道路]]，希望台灣與大陸民主派份子及各公民社團積極交往，不要只著眼在中國政府及中國共產黨。民進黨前主席[[謝長廷|謝長廷]]於2014年1月14日接受《[[中國時報|中國時報]]》專訪，他認為，台灣雖然反對被中國政府統治，這種情緒不應該擴及中國人民，不應該對中國來台的配偶或留學生存有負面心態<ref>{{Cite web|url=http://www.chinatimes.com/newspapers/20140115000475-260102|author=林佩怡、朱真楷、楊舒媚|language=zh-tw|title=謝長廷：綠什麼都反對 但都阻止不了|publisher=中國時報|date=2014-01-14|accessdate=2014-01-15}}</ref>。林義雄主張台灣應該要與中國發展國家與國家之間的良性互動，不要進行國際政治上的對抗<ref>{{Cite web|url=|author=[[何榮幸|何榮幸]]|language=zh-tw|title=林義雄：推動公投 解決統獨問題|publisher=中國時報|date=2005-05-28|accessdate=2015-02-14}}</ref>，這個主張獲得名嘴[[董智森|董智森]]的支持。<ref>{{cite news|title=兩岸關係，聽我的|url=http://magazine.chinatimes.com/ctweekly/20150529005012-300106|author=董智森|newspaper=《[[時報周刊|時報周刊]]》第1945期|date=2015-05-29|accessdate=2015-09-13}}</ref>

== 反中情況 ==

=== {{CHNML}}===
{{See also|对中国共产党的评论|两面人}}
近年来，[[中华人民共和国公民|中华人民共和国公民]]表达对中华人民共和国的不滿並非罕见，而网络将此类情绪扩展开来。[[你国|你国]]由表达反共情绪的词汇演变成表达反中情绪的词汇，[[贵支|贵支]]亦是由[[支那|支那]]演绎而来的反中词汇。同时，反中人士可能在实际生活中掩盖真实的反中情绪。2018年4月发生的[[田佳良事件|田佳良事件]]中的当事人在网络上发表一系“反华、[[精日|精日]]”言论，日常生活中却表态“热爱祖国、热爱中国共产党”。

=== {{HKG}} ===

香港的親共分子發起[[六七暴動|六七暴動]]，被港英政府鎮壓後，香港人的反共意識萌芽。之後香港經歷過支持[[保釣|保釣]]、[[六四事件|六四事件]]等支援中國民主的社會運動。六四後，反對和恐懼[[中國共產黨|中國共產黨]]的心理一直揮之不去，包括那些移民外國的部分香港人。

1997年[[香港回歸|香港回歸]]後，由於[[CEPA|CEPA]]的簽訂、[[港澳個人遊|港澳個人遊]]的開通、愈來愈多來自內地的[[香港新移民|新移民]]移居香港，使到教育、醫療、房屋等社會資源和就業市場變得緊絀，不少香港人均要面對與內地人的直接競爭，部分新移民更有組織涉種票擴大[[建制派|建制派]]的支持盤，內地遊客大量湧進香港購物並排擠本地消費需求，以及被指佔用[[香港醫療|香港醫療]]資源，嚴重影響民生。此外由于回归之后香港不再是中国内地和西方经贸交流的唯一中转站，在面对深圳等内地城市的竞争中因为劳动力成本较高等原因不敌后者导致香港经济發展放緩，生活成本上涨，一些香港人将这归咎于中国大陆政府<ref>{{cite web|url=http://thinktank.qianlong.com/2016/0316/457220.shtml |title=普通人看香港问题：经济问题是根本 |publisher=千龙网 |date=2016-3-16 |accessdate=2018-03-3}}</ref><ref>{{cite web |url=http://news.cngold.com.cn/20160210d1903n63195490.html |title=李嘉诚再次自证英明 香港经济衰退大解析 |publisher=中金网 |date=2016-2-10 |accessdate=2018-03-3 |archive-url=https://web.archive.org/web/20180303105622/http://news.cngold.com.cn/20160210d1903n63195490.html |archive-date=2018-03-03 |dead-url=yes }}</ref>。兩者文化和经济的差距已影響香港人的日常生活，使部分香港人的反共心態，已經走向反對[[中国民族主义|中国民族主义]]、反[[大一統思想|大一統思想]]和拒絕[[中国中心主义|中国中心主义]]，以及擴大至反對[[基本法第二十三條|基本法第二十三條]]、「[[德育及國民教育科|德育及國民教育科]]」和所有試圖促進中港融合的政策。

在1949年後，因[[第二次国共内战|第二次国共内战]]战事失利，中國國民黨的[[中華民國政府|中華民國政府]]撤退至臺灣，宣稱仍擁有中國大陸的完整主權；在大陸成立的中華人民共和國，也自稱為中國唯一合法代表，宣稱擁有所有中國領土包括臺灣的主權，但至今仍未能夠統治臺灣。

在臺灣，現在對於中國大陸關係應如何發展，長期以來存在爭議。傾向[[臺灣獨立|臺灣獨立]]的[[民主進步黨|民進黨]]、[[時代力量|時代力量]]、[[臺灣團結聯盟|臺灣團結聯盟]]等被視為是臺灣主要的反中政黨，傾向和大陸合作的[[中國國民黨|國民黨]]、[[親民黨|親民黨]]、[[新黨|新黨]]等，則被認為是「[[親中|親中]]」及[[統派|統派]]政黨。但民進黨中也有人主張與大陸發展更和諧關係，以[[許信良|許信良]]、[[謝長廷|謝長廷]]等人為代表<ref>{{cite news |language=zh-tw |url=http://www.herald-today.com/content.php?sn=6404 |title=許信良：民進黨非反中而是恐中 |publisher=今日導報|date=2013年3月26日 |accessdate=2013-06-26}}</ref>；[[陳水扁|陳水扁]]、[[蘇貞昌|蘇貞昌]]等人則被視為是反中派<ref>{{cite news |language=zh-tw |url=http://www.ettoday.net/news/20130209/163527.htm|title=《新華社》批蘇貞昌：媚日、賣台、反中 |publisher=[[東森新聞雲|東森新聞雲]]|date=2013年02月29日 |accessdate=2013-06-26}}</ref>。

主張應該親中的人士，包括[[蔡衍明|蔡衍明]]、[[石之瑜|石之瑜]]等。蔡衍明主張親中，認為在臺灣的反中運動，受到蔣介石時代[[反中共|反中共]]的影響<ref>{{cite news|title=蔡衍明董事長 創富論壇致詞全文|url=http://www.chinatimes.com/newspapers/20150808001010-260301|author=本報訊|newspaper=旺報|date=2015-08-08|accessdate=2016-12-16}}</ref><ref>{{cite news|title=倡導親中 蔡衍明：反中都是老蔣的忠誠信徒|url=http://newtalk.tw/news/view/2015-08-07/63197|author=邱珮文|newspaper=新頭殼|date=2015-08-07|accessdate=2016-12-16}}</ref>，這些評論獲得歷史學者[[李功勤|李功勤]]與政治學者[[范世平|范世平]]的讚同<ref>{{cite news|title=蔡衍明闡述反中現象 學者讚坦然|url=http://www.chinatimes.com/newspapers/20150808001009-260301|author=陳君碩|newspaper=旺報|date=2015-08-08|accessdate=2016-12-16}}</ref>。

=== {{VNM}} ===
越南與中國之間有[[南海|南海]]領土的主權爭議。

2014年5月，中國在越南周邊的爭議海域著手開鑿油田的準備工作。5月2日開始，越南海巡船開始接近爭議海域中的中國鑽油平台。5月12日，中國和越南的海巡船在西沙群島的鑽油平台附近對峙，用水砲互相攻擊。5月13日，越南國內發動反中遊行抗議，在平陽省演變成暴動<ref>{{Cite news|url=http://www.cna.com.tw/news/aopl/201405110089-1.aspx |title=南海升溫 越民反陸示威|publisher=[[中央社|中央社]]|date=2014-05-11|accessdate=2014-05-14|language=zh-tw}}</ref>，連累多國華人。在中國媒體中，稱此次事件為[[2014年越南排華暴動|排華暴動]]，包括美國、日本與台灣等地媒體，則稱為反中暴動。

=== {{PHL}} ===
菲律賓與中國之間存在領土爭議，雙方皆主张擁有[[南海|南海]]島嶼，其中包括[[黃岩島|黃岩島]]的主權爭議。
