{{日本作品|反斗小王子|おじゃる丸}}，是[[NHK教育頻道|NHK教育頻道]]於1998年10月5日開始播放的[[電視動畫|電視動畫]]，自2007年4月2日起以[[高清晰度电视|HD畫質]]播放，目前已經播放至第20個系列。是一個圍繞一名五歳小孩邪留丸身邉的故事。

==角色==
※括弧內為擔當的系列數

; {{nihongo|坂上邪留丸|坂ノ上おじゃる丸||聲優：（日）[[小西寬子|小西寬子]]（1～3）→[[西村千奈美|西村千奈美]]（4～）、（港）[[林丹鳳|林丹鳳]]}}
:平安町的王子，喜歡吃[[布丁|布丁]]，身邊有電蟲服侍，戴着綠色烏紗帽，手上持着大魔王的魔王令。他喜歡乙女老師和小町。
; {{nihongo|田村和真|田村カズマ||聲優：（日）[[淵崎有里子|淵崎有-{里}-子]]、（港）[[區瑞華|區瑞華]]}}
:十分喜歡[[石頭|石頭]]，十分怕[[狗|狗]]。當邪留丸不想行時，便會要求和真背著邪留丸。
; {{nihongo|電蟲三十郎|電ボ三十郎||聲優：（日）[[岩坪理江|岩坪理江]]（1～7）→佐藤なる美（8～）、（港）[[林元春|林元春]]}}
:牠是服侍邪留丸的電書[[螢火蟲|螢火蟲]]，有空便會弄[[飯團|飯團]]。電蟲很想[[拍拖|拍拖]]，可惜並不持久。
; {{nihongo|青兵衛|アオベエ||聲優：（日）[[一條和矢|一條和矢]]、（港）[[梁偉德|梁偉德]]}}、{{nihongo|赤根|アカネ||聲優：（日）[[南央美|南央美]]、（港）[[梁少霞|梁少霞]]}}、{{nihongo|黃助|キスケ||聲優：（日）[[上田祐司|上田祐司]]、（港）[[袁淑珍|袁淑珍]]}}
:受大魔王的命令——搶回邪留丸手上的魔王令，而來到月光市。青兵衛的身體是藍色，赤根的身體是紅色，黃助的身體是黃色。他們要求邪留丸給回魔王令給他們時，會合體的。雖然他們天天都要求邪留丸給回魔王令給他們，但次次都不成功。他們是湯美的鄰居。
; {{nihongo|龜姬|オカメ姫||聲優：[[三石琴乃|三石琴乃]]、（港）[[鄭麗麗|鄭麗麗]]}}
:她十分喜歡邪留丸，所以毎次見到邪留丸，便會追着他。但可惜邪留丸不喜歡她，所以毎次見到龜姬，便會便會要求和真背他逃走。她身邊有電蟲子服侍。
; {{nihongo|心灰|うすい さちよ||聲優：[[仲尾梓|仲尾梓]]、（港）[[鄭麗麗|鄭麗麗]]}}
:她28歲，喜歡畫少女漫畫，她夢想是自己的少女漫畫成名。但她畫的畫大多是恐怖漫畫，其他人看到她畫的畫後，都會嚇得立即逃走。雖然如此，心灰亦堅持希望實現這個夢想，所以她不斷畫少女漫畫。
:此外，她經常把茶包重複使用很多次。
; {{nihongo|田村富美男（湯美）|田村富美男|聲優：[[上田敏也|上田敏也]]、（港）[[林保全|林保全]]}}
:他是和真的爺爺，他會製造木頭人偶，是三隻小妖(青兵衛、赤根、黃助)的鄰居。
; {{nihongo|坂田金太郎（金仔）|坂田金太郎|聲優：[[生駒治美|生駒治美]]（3以外）、[[小松里歌|小松里歌]]（3）、（港）[[雷碧娜|雷碧娜]]}}
:他是和真的好朋友，十分喜歡吃波板糖，他有空便會到月夜池塘找出奇。
; {{nihongo|怒犬、笑犬|オコリン坊、ニコリン坊||聲優：[[上田祐司|上田祐司]](怒犬)/[[南央美|南央美]](笑犬)、（港）[[李錦綸|李錦綸]](怒犬)/[[鄭麗麗|鄭麗麗]](笑犬)}}
:牠們是滿願神社的守護犬，也是一對兄妹。天天均會四出籌款，希望重建滿願神社，但由於[[貧窮神|貧窮神]]還留在[[神社|神社]]（貧窮神會帶來貧窮和無元氣），令牠們的願望不能實現。怒犬的身體是藍色和白色，性格[[傲嬌|傲嬌]]；笑犬的身體是粉紅色和白色，性格溫柔。此外，怒犬雖然是男生，卻意外的有喜歡蒐集可換裝娃娃的天然呆喜好。
; {{nihongo|冷徹齋星月|れいてつさい せいげつ||聲優：[[高橋廣司|高橋廣司]]、（港）[[李錦綸|李錦綸]]}}
:他是一名占卜師，天天都穿着黑色的衣服。他身邊有倉鼠公子幫他為人[[占卜|占卜]]。
; {{nihongo|小野小町|おの こまち||聲優：[[西村千奈美|西村千奈美]]、（港）[[陳凱婷|陳凱婷]]}}
:她是和真的同學，性格可愛。家裡開美容院。
