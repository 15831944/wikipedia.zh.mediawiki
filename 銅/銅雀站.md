'''銅雀站'''（{{kor|諺=동작역|漢=銅雀驛|羅=Dongjak yeok|f=y}}）是一個位於[[首爾特別市|首爾特別市]][[銅雀區|銅雀區]]{{link-ko|銅雀洞|동작동}}與[[瑞草區|瑞草區]]{{link-ko|盤浦洞|반포동|盤浦本洞}}邊界，屬於[[首爾地鐵4號線|首爾地鐵4號線]]與[[首爾地鐵9號線|首爾地鐵9號線]]的[[轉乘站|轉乘站]]。副站名是「顯忠院」（{{lang|ko|현충원}}）<ref>[[首爾交通公社|首爾交通公社]]與[[首爾地鐵9號線株式會社|首爾地鐵9號線株式會社]]所屬的所有車站括號內的部分都是首爾市告示的官方站名，與[[韓國鐵道公社|Korail]]的副站名制度不同。</ref>，鄰近[[國立首爾顯忠院|國立首爾顯忠院]]<ref>{{cite web |url=http://www.metro9.co.kr/common/download.do?fileIndex=2847&orderNum=1 |title=存档副本 |accessdate=2018年1月21日 |deadurl=yes |archiveurl=https://web.archive.org/web/20160305093704/http://www.metro9.co.kr/common/download.do?fileIndex=2847&orderNum=1 |archivedate=2016年3月5日 |df= }} 9호선 영업거리표</ref>。
== 歷史 ==
* 1983年9月13日：決定站名為'''銅雀站'''（{{lang|ko|동작역}}）<ref>[http://www2.seoul.go.kr/snews/data/CN_MST/1983-984.pdf#page=233&view=FitV&scrollbar=1&toolbar=1&statusbar=1&messages=1&navpanes=1 서울특별시고시 제475호], 1983년 9월 13일.</ref>。
* 1985年10月18日：[[首爾地鐵4號線|首爾地鐵4號線]][[漢城大入口站|漢城大入口]]－[[舍堂站|舍堂]]路段開通的同時開始營業<ref>[http://newslibrary.naver.com/viewer/index.nhn?articleId=1985101800209201010&editNo=2&printCount=1&publishDate=1985-10-18&officeId=00020&pageNo=1&printNo=19707&publishType=00020 地下鐵(지하철)3.4호선 완전開通(개통)], 동아일보, 1985년 10월 18일 작성.</ref>。
* 2008年9月18日：[[首爾地鐵9號線|首爾地鐵9號線]]站名決定為'''銅雀站'''（{{lang|ko|동작역}}）<ref>[http://www2.seoul.go.kr/snews/data/CN_MST/%C1%A62862%C8%A3.pdf#page=20&view=FitV&scrollbar=1&toolbar=1&statusbar=1&messages=1&navpanes=1 서울특별시고시 제2008-317호], 2008년 9월 18일.</ref>。
* 2009年5月28日：改名為'''銅雀（顯忠院）站'''（{{lang|ko|동작(현충원)역}}）<ref>[http://www2.seoul.go.kr/snews/data/CN_MST/%C1%A62910%C8%A3.pdf#page=169&view=FitV&scrollbar=1&toolbar=1&statusbar=1&messages=1&navpanes=1 서울특별시고시 제2009-211호], 2009년 5월 28일.</ref>。
* 2009年7月24日：[[首爾地鐵9號線|首爾地鐵9號線]][[開花站|開花]]－[[新論峴站|新論峴]]路段開通的同時成為[[轉乘站|轉乘站]]。

== 車站結構 ==
[[首都圈電鐵4號線|首都圈電鐵4號線]]是2面2線曲線[[對向式月台|對向式月台]]，[[首爾地鐵9號線|首爾地鐵9號線]]是2面4線的雙[[島式月台|島式月台]]，兩條路線都設有[[月台幕門|月台幕門]]。
