== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]
**[[觀光列車|環島觀光列車]]：皆不停靠。

*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1903年10月7日：設'''銅鑼灣驛'''，規畫階段擬定驛名為'''福興街'''。<ref>{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>
*1920年：改稱今名。
*1935年4月6日：原木造站房毀於[[1935年新竹台中地震|中部大地震]]，同時期如今留下的遺跡就是[[魚藤坪斷橋|魚藤坪斷橋]]。
*1935年10月：重建站房，於翌年4月竣工。
*1995年：[[銅鑼鄉_(台灣)|銅鑼鄉]]公所興建車站前廣場。 
*2015年6月30日：啟用多卡通刷卡機。

== 車站週邊 ==
[[File:Zhongguang_Clinic,_Tongluo,_Miaoli_(Taiwan).jpg|thumb]]
*{{Jct|country=TWN|CH|119}}
*銅鑼市區
*{{Jct|country=TWN|PH|13}}：俗稱[[台13線|尖豐公路]]，其位於山線鐵路以東之路段為銅鑼鄉中正路，是南北貫穿市中心鬧區的主要道路之一。
*苗栗故事館：中正路（台13線）往北方向之左側，是一間由民間熱心傳統文化人士所開設的懷舊故事館，內部並附設餐飲。
*銅鑼鄉公所
*銅鑼鄉農會
*苗栗縣政府警察局苗栗分局銅鑼分駐所
*銅鑼郵局
*站前圓環：位於銅鑼火車站前，木造古老店面「隨意亭飲食店」旁。
*廢棄防空洞：位於後車站內。
*[[重光診所|重光診所]]：苗栗縣定歷史建築，診所已停業多時，招牌也已拆除。重光診所是已過世之早期地方仕紳劉肇芳醫師（女作家[[朱天文|朱天文]]之[[祖父母|外祖父]]）所興建之傳統[[日式建築|日式建築]]，位在武聖街底。目前仍有劉肇芳之後代子女居住和保養此房屋，屬於私人民宅因此一般民眾或遊客無法入內參觀。(現在假日可供參觀，但是房內較陰暗潮濕)<!--而且早期這裡拍過電影，這戰早期時的模樣，及銅鑼早期時的模樣，通通都有入鏡喔，片名叫:冬冬的假期。還有一個電影也有，片名叫做:大將徐傍興-->
