《'''轉職必勝班'''》（{{lang-ja|'''エンゼルバンク-ドラゴン桜外伝-'''}}，直譯為「'''天使銀行-龍櫻外傳-'''」之意），是[[三田紀房|三田紀房]]所畫的[[日本漫畫|日本漫畫]]作品。自2007年開始在[[講談社|講談社]]的《[[週刊Morning|週刊Morning]]》第45號（2007年10月25日出刊）上開始連載。原文作品標題說明這是《[[東大特訓班|東大特訓班]]》的延續，在中文版的書名上則是以副標題「東大特訓班外傳」表示。在作品中仍有許多《東大特訓班》的人物出場。日文版由講談社發行單行本，共14冊完結，中文版與《東大特訓班》相同，由[[台灣東販|台灣東販]]發行，至[[2011年10月|2011年10月]]已出完全14冊。

改編的[[日劇|日劇]]已经在[[2010年1月|2010年1月]]上映。

== 概要 ==
原龍山高中的[[英文|英文]][[老師|老師]]井野真真子對於教學感到倦怠而決定轉職。透過櫻木所辦的「商務突破講座」，介紹了轉職業務員海老澤康生給井野。在海老澤與櫻木所說的「{{nihongo|人的價值是由市場決定|人の価値は相場で決まる}}」、「{{nihongo|30歲以後要用利息過生活|30過ぎたら利息で暮らせ}}」建議之下，井野接受了海老澤的邀請，進入海老澤所屬的LIFE PARTNERS公司，開始了轉職的事業。

== 日文書名標題的由來 ==
海老澤小學4年級時，利用零食公司的「金天使」與「銀天使」[[食玩|食玩]]進行交易而產生[[市場|市場]]的概念。海老澤自己將這個交易機制稱之為「天使銀行」，體認到「價值是由市場決定」，並因此發展出之後的價值觀。

== 出場人物 ==
; {{nihongo|海老澤康生|海老沢康生（えびさわ やすお）}}
: 轉職事業LIFE PARTNERS公司的轉職代理人。因擁有獨特的價值觀而在公司內被認為怪人。
; {{nihongo|井野真真子|井野真々子（いの ままこ）}}
: 原龍山高中的英文老師。轉值到海老澤的公司當轉職業務員。有時會跑到｢櫻木律師事務所｣找櫻木律師抬槓。
; {{nihongo|櫻木健二|桜木健二（さくらぎ けんじ）}}
: 《[[東大特訓班|東大特訓班]]》的男主角，原來是[[暴走族|暴走族]]，後來成為[[律師|律師]]，在龍山高中[[破產|破產]]前一刻依照「{{link-ja|民事再生法|民事再生法}}」提出改善計畫並獲得成功，並且也在之後成功救起數個學校，被稱為「學校法人的救世主」。說話毒舌(由其是針對井野)。
; {{nihongo|山口彬裕|山口彬裕（やまぐち あきひろ）}}
: 井野在轉職後第一個客人。
; {{nihongo|水野直美|水野直美（みずの なおみ）}}
: 《[[東大特訓班|東大特訓班]]》的人物，考上[[東京大學|東京大學]]。因為經歷過貧困的生活，所以夢想要有錢。其家境雖然還是卡拉OK店，但已好轉。
; {{nihongo|本田美智子|本田美智子（ほんだ みちこ）}}
: 《[[東大特訓班|東大特訓班]]》的人物，原矢島勇介的[[家庭教師|家庭教師]]。東大經濟系畢業，原本在一間投信業工作，然而厭倦跑去聽櫻木的轉職講座。
;矢島勇介
: 《[[東大特訓班|東大特訓班]]》的人物，後來也考上東大理I組，似乎對水野已放下競爭的心態，與水野變成好友。
