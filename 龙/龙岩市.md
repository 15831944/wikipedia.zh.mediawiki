'''龙岩市'''（[[閩南語|閩南語]][[白话字|白话字]]：{{lang|nan|Lêng-nâ-chhī}}；[[客語|客語]][[客語白話字|白話字]]：{{lang|hak|Liùng-ngâm-sṳ}}）是[[中华人民共和国|中华人民共和国]][[福建省|福建省]]下辖的[[地级市|地级市]]，位于福建省西南部，[[九龙江|九龙江]]上游，通称'''[[闽西|闽西]]'''。全市总面积1.9万平方公里，人口261万。

== 歷史 ==
[[晋朝|晋]][[太康_(司馬炎)|太康]]三年（282年）置新罗县。[[唐朝|唐朝]]开元二十四年（736年）置汀州，唐[[天宝|天宝]]元年（742年）汀州改为临汀郡，唐大历十二年（777年）龙岩县改属漳州。明[[洪武|洪武]]元年（1368）为汀州府。清[[雍正|雍正]]十二年（1734）升龙岩县为直隶州，辖漳平、宁洋2县，清末汀州府、龙岩州均属汀漳龙道。1912年废州、府，复龙岩为县。1949年8月至11月，[[中国人民解放军|中国人民解放军]]相继攻取各县，为第八行政督察区。1950年3月改称龙岩专区。1981年撤销龙岩县，设省辖县级龙岩市。1997年5月1日撤销龙岩地区并设立省辖地级龙岩市。

== 地理 ==
龙岩市位于北纬24°23'—26°02'，东经115°51'—117°45'，东与[[福建省|福建省]][[泉州|泉州]]、[[漳州|漳州]]两市交界，西与[[江西省|江西省]][[赣州市|赣州市]]接壤，南与[[广东省|广东省]][[梅州市|梅州市]]交界，北连[[福建省|福建省]][[三明市|三明市]]。距[[厦门|厦门]]142公里（高速公路里程，下同）、[[泉州|泉州]]216公里、[[福州|福州]]376公里。龙岩属[[亚热带海洋性季风气候|亚热带海洋性季风气候]]，日照时数1804小时~2060小时。全年气候温和湿润，雨量充沛，适宜亚热带作物和林木的生长。

=== 語言 ===
龙岩市与漳平市通行闽南语，属闽南语西片方言。以龙岩方言城关音为代表。漳平市通行闽南语龙岩方言漳平音。
另永定，武平两县大部通行客家方言梅州音，长汀县北部，上杭县为客家方言长汀音。长汀县南部，连城县两地方言为闽中方言为底层，融入部分闽西客家方言的一种混合性方言。

== 經濟 ==
龍岩市经济在福建属于中游水平，近年来发展速度较快。大型企业有[[紫金矿业|紫金矿业]]（{{hkex2|2899}}，{{sse2|601899}}）、[[中国龙工|中国龙工]]({{sehk2|3339}})、[[龙净环保|龙净环保]]({{sse2|600388}})、龙岩烟草工业有限责任公司等。

龙岩旅游资源丰富，有2个5A级景区，8个4A级景区以及5个3A级景区。

5A级景区:永定土楼、古田会议旧址

== 饮食 ==
龙岩市地处福建省西部，地势多为山地，气候潮湿温润，形成了具有当地特色的饮食文化，其中有龙岩人喜爱的清汤粉。

另外享有盛誉的还有闽西八大干（严格来说并不是所有都属于龙岩特产）：长汀豆腐干，连城地瓜干，宁化老鼠干，上杭萝卜干，清流笋干，武平猪胆干，明溪肉脯干，永定梅菜干。
