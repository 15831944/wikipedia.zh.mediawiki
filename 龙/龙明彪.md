'''龙明彪'''（{{bd|1962年|5月||}}）[[湖南|湖南]][[华容|华容]]人，[[中华人民共和国|中华人民共和国]]官员<ref name=cpc>{{Cite web |url =http://renshi.people.com.cn/n/2015/0413/c139617-26834090.html  |title =  龙明彪任国台办副主任|author =  |date = 2015-04-13 |publisher = 中国共产党新闻网 |language =  |accessdate = |quote =  }}</ref><ref name=lmb>{{Cite web |url =http://www.gwytb.gov.cn/gtb/longmingbiao/  |title = 龙明彪 |author =  |date =  |publisher =中共中央台办、国务院台办  |language =  |accessdate =2017-10-11  |quote =  }}</ref>。現任國台辦副主任兼海協會常務副會長。

==生平==
1982年6月加入[[中国共产党|中国共产党]]，1983年7月参加工作，大学本科学历。1983年7月至1987年12月，任[[中华人民共和国交通部|交通部]]人民警察学校教师、教研室副主任。其间，1985年7月至1986年7月，任[[中央讲师团|中央讲师团]]赴[[四川|四川]][[南充|南充]]支队党支部书记<ref name=cpc/><ref name=lmb/>。

1987年12月至1992年1月，任[[中华人民共和国监察部|监察部]]第四监察司干部。1992年1月至1993年1月，任监察部法规司副处级监察员。1993年1月至1996年2月，任[[中央纪委|中央纪委]]、监察部法规室副处长。1996年2月至1997年11月，任中央纪委、监察部法规室处长。其间，1996年3月至1997年2月，在[[福建省|福建省]][[厦门市|厦门市]][[思明区|思明区]]挂职任区委副书记<ref name=cpc/><ref name=lmb/>。

1997年11月至1998年3月，在[[中央台办|中央台办]]、[[国台办|国台办]]机关党委办公室工作。1998年3月至2000年6月，任中央台办、国台办机关党委办公室主任。2000年6月至2000年8月，任中央台办、国台办人事局人事处处长、机关纪委副书记。2000年8月至2003年9月，任中央台办、国台办人事局副局长兼人事处处长、机关纪委书记。2003年9月至2005年2月，任中央台办、国台办机关纪委书记（正局级）兼人事局副局长。2005年2月至2009年4月，任中央台办、国台办秘书局（人事局）局长。其间，2006年2月至2007年1月，参加[[中央党校中青班|中央党校中青班]]第22期学习<ref name=cpc/><ref name=lmb/>。

2009年4月至2009年6月，任中央台办、国台办主任助理、人事局局长。2009年6月至2015年3月，任中央台办、国台办主任助理。2015年3月起，任中央台办、国台办副主任<ref name=cpc/><ref name=lmb/>。2018年4月起，任[[海峡两岸关系协会|海峡两岸关系协会]]常务副会长。
