'''龍崗區'''是[[深圳|深圳]]的一個[[市轄區|市轄區]]，位於深圳東部，辖区总面积388.59平方公里（不含[[大鹏新区|大鹏新区]]），常住人口為 205.24萬，其中户籍人口47.72万。龙岗是2010年深圳市[[经济特区|经济特区]]扩大到全市范围之前，位于经济特区之外的两个的市辖区之一（另外一个为原[[宝安区|宝安区]]）。1993年宝安县撤县设区时，龙岗区的辖区范围还包括今日的[[坪山区|坪山区]]和[[大鹏新区|大鹏新区]]。目前龙岗区内已建立数所大学，成为深圳市的教育新区。

== 歷史 ==
秦始皇二十八年（西元前219年），[[秦朝|秦]]將屠睢率軍南下攻擊[[百越|百越]]發動戰爭，是為[[秦攻百越之戰|秦攻百越之戰]]。秦始皇三十三年（西元前214年），秦軍第二次南下，在[[任囂|任囂]]和[[趙佗|趙佗]]率領下占領嶺南，秦設[[桂林郡|南海]]，[[象郡|象郡]]，[[桂林郡|桂林]]三郡。龍崗一帶正式劃入秦嶺土。

宋末時期，龍崗地區已發展有荷坳等村落<ref>{{web reference|URL=http://szyuanzhumin.com/a/culture/villages/2011/1215/11358.html|title=尋找遠去的荷坳古村落|work=中深圳原住民網|year=2011年|date=11月15日}}</ref>。

明洪武二十七年（西元1394年），境內大鵬半島建有「大鵬守禦千戶所城」，即「[[大鵬所城|大鵬所城]]」。

明萬曆元年（西元1573年），新安縣由東莞縣分出。龍崗境內除東北部由歸善縣（今[[惠州市|惠州市]]），其餘地區由新安縣（今深圳與[[香港|香港]]）管轄。

清康熙年間，清廷頒布[[遷界令|遷界令]]，龍崗地區的廣府住民被迫內遷。康熙七年，[[廣東巡撫|廣東巡撫]][[王來任|王來任]]聯同[[兩廣總督|廣東廣西總督]][[周有德|周有德]]上疏朝廷請求復界。康熙八年（西元1669年）清廷准許復界。由於回遷原住民寥寥，官府實行招墾政策，龍崗地區遷入大量來自[[福建|福建]]，[[江西|江西]]及粵東山區的[[客家人|客家人]]<ref>{{web reference|URL=http://news.chinatimes.com/wantdaily/11052101/112011112600253.html|title=取消遷界令 客家人深圳墾荒|work=中國時報|year=2011年|date=11月26日}}</ref>。此後從嘉慶到民國初年都有客家移民不斷遷入龍崗。形成了龍崗與深圳其餘地方[[廣府民系|廣府人]]為主完全不同的客家人占主體的居民結構。

1908年，[[广九铁路华段|广九铁路华段]]建筑工程开展并于1911年10月建成开通，首批设立的车站中就包括现今位于龙岗境内的[[平湖站|平湖站]]、[[李朗站|李朗站]]和[[深圳东站|布吉站]]<ref>{{web reference|URL=http://www.kcrc.com/sc/about-kcrc/history.html|title=九广铁路公司历史|work=九广铁路公司|year=2017年|date=10月8日}}</ref>。

1958年11月，现深圳境内的龙岗、横岗、坪山、大鹏等地由惠阳县划归同属[[惠阳地区|惠阳地区]]管辖的[[宝安县|宝安县]]<ref>{{web reference|URL=http://www.huiyang.gov.cn/hygk/lsyg/index.shtml|title=惠阳概况 历史沿革|work=惠阳区人民政府|year=2017年|date=7月24日}}</ref>。

1979年3月，寶安縣升格為深圳市。1980年8月，深圳經濟特區成立，龍崗地区並未被劃入[[經濟特區|經濟特區]]範圍。1982年12月國務院批准復設寶安縣，隸屬深圳市，轄經濟特區以外地區。

1992年11月11日，國務院再次批准撤銷寶安縣，分設龍崗，寶安兩區；仍隸屬深圳。

1993年1月1日，龍崗區正式掛牌辦公。

2010年7月1日，龍崗區和寶安區一起被劃入經濟特區範圍。

2016年10月11日，坪山新区正式从龙岗区分离出来，成立[[坪山区|坪山区]]。

2016年12月26日，龙岗区管辖的街道由8个拆分为11个，其中布吉、横岗、龙岗、龙城4个街道办事处分设为布吉、吉华、横岗、园山、龙岗、龙城、宝龙7个街道办事处，平湖、坂田、南湾、坪地街道办事处不变<ref>{{Cite news|url=http://sz.leju.com/news/2016-12-27/10306219338475091841427.shtml|title=深圳宝安区街道“6变10” 龙岗区街道“8变11”|last=|first=|date=2016-12-27|work=南方日报|accessdate=2017-01-10}}</ref>。

== 行政區劃 ==
{{深圳行政区划地图}}
龙岗区下辖有14个[[街道办事处|街道办事处]]，其中11个街道为区政府直接管辖，3个街道由[[大鹏新区|大鹏新区]]管辖。区政府位于[[龙城街道_(深圳市)|龙城街道]]。
=== 行政区划演变 ===

1993年龙岗建区时，区政府位于龙岗镇。区内设有10个镇，即平湖镇，布吉镇，横岗镇，龙岗镇，坪地镇，坑梓镇，坪山镇，葵涌镇，大鹏镇，南澳镇。2004年1月2日，龙岗镇撤镇设街道办事处并一分为二，区政府所在的龙城中心城及其周边地区设为龙城街道办，其余地区设为龙岗街道办。同年8月26日，龙岗区剩余的9个镇全部撤镇改街道，辖区因此形成11个街道的格局。2006年4月，龙岗区宣布将布吉街道办调整为布吉，坂田，南湾三个街道。龙岗区下辖街道办数目由此增加到13个。另有一个类似乡级单位——[[深圳市大工业区|深圳市大工业区]]。

2009年，深圳市决定在龙岗区的坪山、坑梓街道和深圳市大工业区设置功能新区，坪山新区于6月30日挂牌成立。此后深圳市于2011年12月30日在位于大鹏半岛的葵涌，大鹏，南澳三个街道增设一个功能新区，即大鹏新区。这两个新区的设立均不涉及行政区划的调整。2016年10月11日，国务院批复广东省政府，同意设立深圳市龙华区和坪山区。坪山新区由此升格为深圳市的行政区，不再隶属于龙岗区，新设立的坪山区于2017年1月7日正式揭牌成立。2016年12月26日，龙岗区进行了新一轮的街道调整，将布吉街道一分为二，形成布吉、吉华街道；横岗街道一分为二，形成横岗、园山街道；龙岗、龙城街道合二分三，形成新的龙岗、龙城、宝龙街道；其他平湖、坂田、南湾、坪地4个街道维持现状不变，形成了目前的11个街道+大鹏新区的新格局<ref>{{Cite news|url=http://ilonggang.sznews.com/lgnews/content/2016-12/26/content_14668984.htm|title=正式揭牌！龙岗区街道由8个变为11个|date=2016-12-26|work=龙岗新闻网|accessdate=2017-09-30|archive-url=https://web.archive.org/web/20161227151501/http://ilonggang.sznews.com/lgnews/content/2016-12/26/content_14668984.htm|archive-date=2016-12-27|dead-url=yes}}</ref>。

== 地理 ==
=== 地貌 ===

龍崗屬典型的濱海丘陵地形，山脈走向為東北至西南。海岸線長度為133公里，其中大鵬半島東面為[[大亞灣|大亞灣]]，西面為[[大鵬灣|大鵬灣]]。境內最高點為大鵬半島的[[七娘山|七娘山]]，海拔867米。

== 人口 ==
根据2015年统计<ref>{{web reference|URL=http://www.sztj.gov.cn/xxgk/tjsj/tjnj/201701/W020170120506125327799.pdf|title=深圳统计年鉴2016|work=深圳統計|year=2017年|date=9月30日}}</ref>，龙岗区常住人口（不包括大鹏新区）為205.24万人，占深圳市总人口的18%；其中户籍人口为47.72万，占全区常住人口的23%。人口密度为5282人/平方公里，略低于全市平均水平（5697人/平方公里）。
=== 語言 ===
1980年設立[[深圳經濟特區|深圳經濟特區]]之前，龍崗除平湖，荷坳等地通行[[莞寶粵語|粵語莞寶片]]的[[圍頭話|圍頭話]]以及[[大鵬半島|大鵬半島]]通行的[[大鵬話|大鵬話]]，大部分居民為操[[客家話|客語]]的[[客家人|客家人]]。1980年之後大量移民的遷入以及[[推廣普通話|推普運動]]影響的使得漢語[[現代標準漢語|普通話]]成為常用語言之一,本地人仍多使用客家話交流。

==经济==
2016年，龙岗区（不含大鹏新区）实现地区生产总值3177.06亿元（人民币，下同），占全市的16.3%，居深圳各区第三（仅次于南山区，福田区），广东省第四（仅次于南山区，广州市[[天河区|天河区]]，福田区），与上年相比增长9.9%。第一产业增加值0.19亿元，下降2.6%；第二产业增加值2047.19亿元，增长10.2%，其中工业增加值 1935.01亿元，增长9.9%；建筑业增加值112.19亿元，下降13.9%；第三产业增加值1129.67亿元，增长9.3%。三次产业比例为0.01：64.43：35.56。<ref>{{Cite news|url=http://www.lg.gov.cn/szslgtjxxw/tzgg/201704/58dab3bf1642451bbae32c34ec4d772a.shtml|title=关于反馈2016年各街道生产总值数据的通知|last=|first=|date=2017-04-13|work=深圳市龙岗区统计局|accessdate=2017-09-30|archive-url=https://web.archive.org/web/20180310200857/http://www.lg.gov.cn/szslgtjxxw/tzgg/201704/58dab3bf1642451bbae32c34ec4d772a.shtml|archive-date=2018-03-10|dead-url=yes}}</ref>，区内有[[华为|华为]]和[[比亚迪|比亚迪]]等企业。

== 社区及文体娱乐 ==
[[深圳大运中心|深圳大运中心]]位于龙岗区龙城街道，该项目包括一主体育场，一主体育馆，以及一游泳馆，是深圳市为举办[[2011年世界大学生夏季运动会|2011年世界大学生夏季运动会]]而兴建的主要体育项目，总投资约45亿元人民币，由德国[[冯·格康，玛格及合伙人建筑师事务所|gmp建筑师事务所]]设计。该项目的主体育馆共有座位约18000个，可满足篮球，[[冰球|冰球]]，排球，[[室内足球|室内足球]]，网球，体操等16类室内运动比赛的需求<ref>{{Cite news|url=http://www.chinanews.com/ty/2011/06-29/3144569.shtml|title=深圳大运中心体育馆可满足16种比赛需求|last=|first=|date=2011-06-29|work=深圳特区报|accessdate=2017-10-06}}</ref>。 目前是[[CBA联赛|CBA联赛]][[深圳新世纪篮球俱乐部|深圳马可波罗队]]的和加拿大女子冰球联赛（CWHL）深圳昆仑鸿星队和深圳万科阳光队的主场<ref>{{Cite news|url=https://www.sohu.com/a/157959606_148781|title=昆仑鸿星两支队 主场落户深圳|last=|first=|date=2017-07-18|work=北京青年报|accessdate=2017-10-06}}</ref>，也是中国大陆地区继上海[[梅赛德斯-奔驰文化中心|梅赛德斯-奔驰文化中心]]、北京[[五棵松体育馆|五棵松体育馆]]和[[广州国际体育演艺中心|广州国际体育演艺中心]]之后第四个举办[[NBA海外赛|NBA中国赛]]的场馆<ref>{{Cite news|url=http://industry.sports.cn/news/cg/2015/0401/96104.html|title=体育场馆赛后运营难 深圳大运场馆各具活力|last=|first=|date=2015-04-01|work=深圳特区报|accessdate=2017-10-06}}</ref>。截至目前举办过2015年NBA季前赛[[洛杉矶快船队|洛杉矶快船队]]对阵[[夏洛特黄蜂队|夏洛特黄蜂队]]，以及2017年[[明尼苏达森林狼队|明尼苏达森林狼队]]对阵卫冕冠军[[金州勇士队|金州勇士队]]的比赛。大运中心主体育场建筑面积13.5万平米，可容纳6万名观众。举办过如[[2016年國際冠軍盃|2016年国际冠军杯]][[多特蒙德足球俱乐部|多特蒙德]]对阵[[曼彻斯特城足球俱乐部|曼城]]和[[2017年國際冠軍盃|2017年国际冠军杯]][[拜仁慕尼黑|拜仁慕尼黑]]对阵[[AC米兰|AC米兰]]的足球比赛。

龙岗体育中心是龙岗区内另一大型体育项目，该项目位于龙城街道西部，拥有包括龙岗国际自行车赛场和神仙岭国际网球中心等体育场馆和设施。龙岗国际自行车赛场包括主场馆、7公里的环水库公路赛道和6.8公里的山地赛道三部分，是世界首个按国际自盟2001最新标准建设的比赛场馆，也是国内以及亚洲第一个可同时举办自行车三项比赛（场地赛、公路赛及山地赛）的国际标准赛场。神仙岭国际网球中心每年1月初（WTA）和9月底（ATP）举办[[深圳网球公开赛|深圳网球公开赛]]，该赛事是WTA国际赛和ATP世界巡回赛250系列赛事。

位于龙岗区宝荷路的正中高尔夫球会举办过2014年[[沃尔沃中国公开赛|沃尔沃中国公开赛]]，并于2015年起，于每年4月举办深圳国际赛。

== 教育 ==
===大专院校===
龙岗区目前有高等院校三所，皆位于龙城街道。

[[深圳信息职业技术学院|深圳信息职业技术学院]]创办于2002年，是一所由深圳市人民政府举办的公办全日制高等院校，于2011年搬迁至龙岗区龙城街道。该校位于龙翔大道的新校区建成后曾被用作[[2011年世界大學生夏季運動會|2011年世界大學生夏季運動會]]的运动员村以及部分比赛场地。该校目前有在校学生16198人。

[[香港中文大學（深圳）|香港中文大學（深圳）]]是一所经国家教育部批准，按中外合作办学条例设立的研究型大学。校址位于龙岗区大运公园南侧，龙翔大道以北。港中大（深圳）于2014年正式成立并招生，目前有本科生和研究生共2000余人。该校实行全英语授课。

此外，由深圳市人民政府，[[北京理工大学|北京理工大学]]以及[[莫斯科国立大学|莫斯科国立罗蒙诺索夫大学]]合作举办的[[深圳北理莫斯科大学|深圳北理莫斯科大学]]于2017年成立并招生，该校永久校区目前仍在建设中。学校临时校区位于龙岗区龙城街道如意路的原深圳市体育运动学校内。

== 医疗服务 ==
位于龙岗街道的龙岗中心医院创立于1952年，是龙岗区首家[[三级甲等医院|三级甲等医院]]，也是深圳东部最大的一所综合性医院，提供约855张病床。辖区另外一家主要公立综合医院是龙岗区人民医院，创立于1993年，位于龙城街道，提供开放病床629张。另外，院址位于龙岗区宝荷大道的中国医学科学院肿瘤医院深圳医院，即深圳市肿瘤医院，是经国家卫生计生委批准、由中国医学科学院肿瘤医院（国家癌症中心）和深圳市政府联合创办的一所肿瘤专科医院，是深圳市政府医疗卫生“三名工程”重点项目，一期工程编制床位800张。以下为龙岗区医院列表：

== 旅游 ==
龙岗有[[龙岗客家民俗博物馆|龙岗客家民俗博物馆]]和[[大芬村|大芬村]]等景点。其中大芬村位於[[深圳|深圳]]龙岗区[[布吉街道|布吉]]，是一處著名的油畫村。在1980年代尾，由一位[[香港|香港]]画商[[黄江|黄江]]發起，村民原有200人，現在有1萬多外來人口從事美術工作。作品以仿[[西洋|西洋]][[油畫|油畫]]聞名，也有原創。此地有稱為[[大芬油畫村|大芬油畫村]]。
