'''龙邦口岸'''<ref>{{cite web|title=中华人民共和国政府和越南社会主义共和国政府关于中越陆地边境口岸及其管理制度的协定|url=http://www.fmprc.gov.cn/web/ziliao_674904/tytj_674911/tyfg_674913/t812101.shtml|publisher=[[中华人民共和国外交部|中华人民共和国外交部]]|date=2011-04-02}}</ref>是[[中华人民共和国|中国]][[广西壮族自治区|广西壮族自治区]]的一个国家级一类公路[[口岸|口岸]]<ref>{{cite web |author1=广西电子口岸 |title=龙邦口岸口岸介绍 |url=http://www.gxeport.gov.cn/flash/Statistics.aspx?portCode=LBGL |website=广西电子口岸 |accessdate=2019-02-16}}</ref>，定位为常年开放的公路客运货运口岸。位于[[百色市|百色市]][[靖西市|靖西市]][[龙邦镇|龙邦镇]]，对接[[越南社会主义共和国|越南社会主义共和国]][[高平省|高平省]][[茶岭县|茶岭县]]的[[茶岭口岸|茶岭口岸]]。目前是[[广西壮族自治区|广西壮族自治区]][[百色市|百色市]]唯一的国家一类口岸。<ref>{{cite web |author1=中华人民共和国商务部 |title=国务院批准广西龙邦口岸升格为一类口岸 |url=http://www.mofcom.gov.cn/article/resume/n/200301/20030100066162.shtml |accessdate=2019-01-23}}</ref><br>

== 历史 ==

2017年10月23日，中国龙邦边民互市贸易区建成开园，成为目前中越边境规模最大、设施配套最齐备、通关服务系统最先进的边民互市贸易区。<ref>{{cite web |author1=中新网 |title=中国龙邦边民互市贸易区建成开园 为中越边境规模最大 |url=http://www.chinanews.com/cj/2017/10-23/8358794.shtml}}</ref><br>
2018年12月19日，[[靖西市|靖西]]至[[龙邦镇|龙邦]]高速公路建成通车，为[[中国西南地区|中国西南地区]]通往东南亚各国再打开一条新的陆路通道。<ref>{{cite web |author1=凤凰网 |title=从靖西到龙邦口岸只要20分钟！ 广西出境高速公路网“再添一线”! |url=http://wemedia.ifeng.com/93912464/wemedia.shtml |accessdate=2019-01-24}}</ref>

== 位置及设施 ==

龙邦口岸位于中越边境742号界碑北侧约30米处，距[[靖西市|靖西]]市区42公里，距[[百色市|百色]]市区221公里，距[[广西壮族自治区|广西壮族自治区]]首府[[南宁市|南宁市]]302公里。
