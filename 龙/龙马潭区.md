'''泸州市龙马潭区'''是[[四川省|四川省]][[泸州市|泸州市]]的一个[[市辖区|市辖区]]，为泸州市主城三区之一、北部副中心，位于[[沱江|沱江]]-[[长江|长江]]一线以北。<ref name=":0">{{cite web |title=泸州市龙马潭区区情概况 |url=http://www.longmatan.gov.cn/qq/hzqh/content_30506 |publisher=泸州市龙马潭区人民政府 |accessdate=2018-09-02}}</ref>

== 区名由来 ==
龙马潭区得名于龙马潭。龙马潭位于清代[[泸州直隶州|泸州]]城东北二十里（今泸州市龙马潭区[[鱼塘镇|鱼塘街道]]），据[[清朝|清朝]][[常明|常明]]、[[杨芳灿|杨芳灿]]等纂修《四川通志》，[[唐朝|唐朝]]王昌于泸州遇一张姓落魄仙，传以道术后取马送其归家，马化龙入潭，王昌也仙去，故名龙马潭。相传下有龙窟。有[[宋朝|宋]]时所建冲虚、碧梧二观，近代重修。<ref name=":1" /><ref>{{cite book |author1=[[华国清|华国清]]等 |title=中国地方志集成·四川府县志辑32 光绪泸州直隶州志 |date=1992 |publisher=巴蜀书社 |location=成都 |isbn=7-80523-478-7 |edition=第一版 |accessdate=2018-09-03}}</ref>

==历史沿革==
1995年12月24日，[[中华人民共和国国务院|国务院]]批准{{notetag|国函〔1995〕132号}}设立泸州市龙马潭区，1996年1月26日，[[四川省人民政府|四川省人民政府]]批准{{notetag|省府发〔1996〕39号}}设立泸州市龙马潭区，辖泸州市原[[市中区_(泸州市)|市中区]]（今[[江阳区|江阳区]]）罗汉镇、鱼塘镇、小市街道办事处、高坝厂区办事处和[[泸县|泸县]]的胡市镇、石洞镇、特兴镇、金龙乡、安宁乡。1999年安宁乡撤乡建镇。2000年增设双加镇、长安乡、红星街道、莲花街道，高坝厂区与罗汉镇合署办公。2014年罗汉镇、鱼塘镇、安宁镇撤镇建街道，长安乡、金龙乡撤乡建镇{{notetag|四川省人民政府关于同意泸州市龙马潭区、叙永县调整部分乡镇行政区划的批复（川府民政〔2014〕26号）}}<ref name=":1">{{cite web |author1=龙马潭区信息办 |title=龙马潭区历史沿革 |url=http://www.longmatan.gov.cn/qq/lsyg288/content_17078 |publisher=泸州市人民政府 |accessdate=2018-09-02 |date=2013-01-11}}</ref>。2017年石洞镇撤镇设街道<ref>{{cite web |author1=泸州市政府办公室 |title=泸州市人民政府关于同意龙马潭区石洞镇撤镇设街道的批复（泸市府函〔2017〕196号） |url=http://www.luzhou.gov.cn/zw/zcfg/szfwj/lsfh/content_99519 |publisher=泸州市人民政府 |accessdate=2018-09-02 |date=2017-05-26}}</ref>。2018年特兴镇撤镇设街道<ref>{{cite web |author1=泸州市政府办公室 |title=泸州市人民政府关于同意龙马潭区特兴镇撤镇设街道的批复（泸市府函〔2018〕97号） |url=http://www.luzhou.gov.cn/zw/zcfg/szfwj/lsfh/content_376176 |publisher=泸州市人民政府 |accessdate=2018-09-02 |date=2018-04-11}}</ref>。

== 经济 ==
2016年，龙马潭区实现[[地区生产总值|地区生产总值]]205亿元，同比增长7.9%。社会消费品零售总额达60亿元，同比增长13.6%。城镇居民[[人均可支配收入|人均可支配收入]]达33280元，同比增长8.8%；农村居民人均可支配收入达16290元，同比增长9.8%。第一、第二、第三产业比重为5.0：70.3：24.7。<ref name=":0" /><ref>{{Cite web|url=http://www.longmatan.gov.cn/zw/zfwj/articlegzbgqtwj/content_33794|title=2016年龙马潭区政府工作报告|accessdate=2018-09-03|author=|date=2017-03-16|publisher=龙马潭区人民政府办公室}}</ref>
