'''龙泽站'''位于[[北京市|北京市]][[昌平区|昌平区]]，是[[北京地铁|北京地铁]][[北京地铁13号线|13号线]]的一个车站。车站编号是1307。

== 位置 ==
这个站位于[[八達嶺高速公路|八達嶺高速公路]]東北面。

== 结构 ==
本站在采用两层高架车站，一层为站厅，二层为站台

== 出口 ==
這個站有1個出口：A北

== 问题 ==
自13号线开通以来，由于接驳公交线路不足等原因，龙泽站周边长期存在[[非法营运车辆|非法营运车辆]]揽客的问题。<ref>{{cite news|url=http://news.xinhuanet.com/newscenter/2003-11/26/content_1199890_1.htm|title=公交车：想说爱你不容易——体验北京市民早出行|publisher=[[新华网|新华网]]|date=2003-11-26|accessdate=2016-06-30|deadurl=yes|archiveurl=https://web.archive.org/web/20041129120123/http://news.xinhuanet.com/newscenter/2003-11/26/content_1199890_1.htm|archivedate=2004-11-29}}</ref><ref>{{cite news|url=http://news.sina.com.cn/c/2013-11-18/022028735306.shtml|title=城铁龙泽站周边脏乱成顽症|publisher=[[北京日报|北京日报]]|date=2013-11-18|accessdate=2016-06-30}}</ref>此外，龙泽站出入口前的通道时常被售卖食品的商贩以一些自行车、电瓶车挤占。<ref>{{cite news|url=http://bjrb.bjd.com.cn/html/2016-06/06/content_40265.htm|title=乘客出入受阻 脚下黏糊糊|publisher=[[北京日报|北京日报]]|date=2016-06-06|accessdate=2016-06-30|deadurl=yes|archiveurl=https://web.archive.org/web/20160821205939/http://bjrb.bjd.com.cn/html/2016-06/06/content_40265.htm|archivedate=2016-08-21}}</ref>
