'''龙须藤'''（''Bauhinia championii''）又名'''菊花木'''、'''九龍藤'''、'''蠂藤'''<ref name="tw-tree">{{cite web
|url       = http://subject.forest.gov.tw/species/twtrees/book2/index-1.htm
|title     = 菊花木
|work      = 臺灣樹木  解說（二）
|editor1   = 呂福遠
|editor2   = 歐辰雄
|editor3   = 呂金誠 
|publisher = 行政院農業委員會
|year      = 1997
|page      = p9}}</ref>，是一種[[豆科|豆科]][[蘇木亞科|蘇木亞科]][[羊蹄甲屬|羊蹄甲屬]]的[[有花植物|有花植物]]。分布于[[越南|越南]]、[[台湾|台湾]]、[[印度尼西亚|印度尼西亚]]、[[印度|印度]]以及[[中国大陆|中国大陆]]的[[广东|广东]]、[[湖南|湖南]]、[[湖北|湖北]]、[[江西|江西]]、[[福建|福建]]、[[浙江|浙江]]、[[贵州|贵州]]、[[广西|广西]]等地，生长于海拔200米至1,000米的地区，一般生于丘陵灌丛中、山地疏林以及密林中，目前尚未由人工引种栽培。

==形態特徵==  
 
* [[常绿|常绿]][[藤本|藤本]]
* 單葉互生，頂端有很浅的分裂，与本属其他种区别，全缘 
* 请协助补充

== 别名 ==
