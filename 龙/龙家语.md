{{Infobox language|name=龙家语|nativename=Songnibao|states={{PRC}}|region=[[贵州省|贵州省]]|ethnicity={{tsl|en|Longjia people|龙家人}}|speakers=灭绝 ?|date=2011|familycolor=Sino-Tibetan|fam2=（未分类）|fam3=[[蔡家话|蔡家话]]–龙家语–[[卢人语|卢人语]]<ref name="Guizhou1984"/>|iso3=none|glotto=long1417|pronunciation=|iso1=|iso2=|script=|fam1=|fam4=|fam5=|vitality=}}'''龙家语''' （本名： ''{{IPA|suŋ˥ni˥mpau˨˩}}''）是[[贵州|贵州]]的一种[[汉藏语|汉藏语]]。它同[[蔡家話|蔡家話]]和[[卢人语|卢人语]]有着亲缘关系。龙家语可能已经[[絕跡語言|灭绝]]。

{{Tsl|en|Longjia people|龙家人}}放弃了他们原本的语言而改说[[西南官话|西南官话]]。根据《[[贵州省志|贵州省志]]》（2002）<ref name="Gazetteer">贵州省志. 民族志.(2002). 贵阳:貴州民族出版社.</ref> ，龙家语曾使用于[[大方县|大方县]]、 [[黔西县|黔西县]]和[[普定县|普定县]]。该语言被认为同[[蔡家話|蔡家話]]有亲缘关系<ref name="Gazetteer" />，并包含许多来自[[上古汉语|上古汉语]]的[[外来词|外来词]]。<ref name="Dafang">大方县年鉴 (1996:150-152)</ref>

== 分类 ==
贵州省民族识别工作队（1984）认为龙家语同[[蔡家話|蔡家話]]和[[卢人语|卢人语]]有着亲缘关系。<ref name="Guizhou1984">贵州省民族识别工作队. 1984. 南龙人（南京人-龙家人）族别问题调查报告.</ref> 但是，龙家语在汉藏语系中的分类尚不明确。郑张尚芳（2010） 认为[[蔡家話|蔡家话]]和[[白語|白语]]是姊妹语言<ref name="Zhengzhang2010">郑张尚芳. 2010. 蔡家话白语关系及词根比较. 选自 潘悟云、沈钟伟(编). 研究之乐-庆祝王士元先生七十五寿辰学术论文集，II，389-400. 上海:上海教育出版社.</ref>，而[[沙加尔|沙加尔]]认为蔡家话属于[[汉语族|汉语族]]，与[[瓦乡话|瓦乡话]]有着紧密的亲缘关系。<ref name="Sagart2011">Sagart, Laurent. 2011. [https://www.academia.edu/19534510/Chinese_dialects_classified_on_shared_innovations Classifying Chinese dialects/Sinitic languages on shared innovations]. Talk given at Centre de recherches linguistiques sur l’Asie orientale, Norgent sur Marne.</ref>

== 音系 ==
《普定年鉴》（1999年）称，龙家语有38个[[辅音|辅音]]和22个[[元音|元音]]（8个[[单元音|单元音]]和14个[[复元音|复元音]]）。《毕节年鉴》（1996年：143）报告说，龙家语有很多[[鼻化辅音|鼻化辅音]]。 在大方县，龙家语的本名是“松立保”。<ref name="Dafang"/>
