'''龙湖公园'''位于[[河北|河北]][[邯郸市|邯郸市]]主城区东部，南临人民路，北靠是丛台路，东临滏东大街，西依有“邯郸母亲河”之称的[[滏阳河|滏阳河]]，该园是以湖泊和植物造景为主，以“水林景色”为特色的开放式生态休憩公园。公园不但融入了具有现代园林景观的设计理念，而且还蕴含了当地的[[赵国|古赵文化]]的文化底蕴，利用声、光、电等现代科学技术手段，使地水体景观与园林艺术交相辉映{{参|name=参考2|{{cite web|title=松下投影机邯郸龙湖户外投影案例|url=http://projector.zol.com.cn/534/5341400.html|website=松下|publisher=中关村在线|accessdate=2017-05-07}}}}{{参|name=参考3|{{cite web|title=邯郸龙湖公园一期工程竣工 九大功能分区敞怀迎客|url=http://news.sina.com.cn/s/2006-09-27/104410121929s.shtml|website=新浪网|publisher=河北日报|accessdate=2017-05-07}}}}，每到夏季时节，龙湖夜晚客流量达3万人次以上{{参|name=参考1|{{cite web|author1=杨盖|title=汽车生活文化节 邯郸龙湖车展邀您共享|url=http://auto.cnfol.com/jiangjiacuxiao/20140716/18407397.shtml|website=汽车点评网|accessdate=2017-05-07}}}}。

== 命名 ==
龙湖里的湖水，来自于[[滏阳河|滏阳河]]，由公园西南进入湖中，流向东北，成湖于园中，在园内绕一圈最终从西北方向再次流到滏阳河，河道蜿蜒，犹如一条巨龙，故名“龙湖公园”。

== 修建历史和规模 ==
[[File:Longhu_Park_1.JPG|200px]]
龙湖公园最早是一片荒地，下大雨时，这里地势低洼雨水排不出去，逐步形成一个个小[[洼子|洼子]]。[[2005年|2005年]]，市政府决定截[[滏阳河|滏阳河]]上的[[柳林闸|柳林闸]]之水，引水建湖。首期工程光就挖湖堆山、砌围建岸动土达60万立方米，占地总面积有41.7公顷，扩展园中水体面积约8.2万平方米，最终建成由「南门入口广场区」、「景观湖区」、「市民休闲区」、「中心广场区」等9个功能区域{{参|name=参考3}}{{参|name=参考4|{{cite web|title=邯郸:龙湖公园正加紧建设|url=http://news.sina.com.cn/s/2005-09-27/09107045604s.shtml|website=新浪网|publisher=河北日报|accessdate=2017-05-07}}}}。

园内绿化面积在26.36公顷以上{{参|name=参考4}}，园内栽植乔灌木166种，各类植物在1.6万余株，还有例如[[枇杷|枇杷]]、[[美国海棠|美国海棠]]、[[木瓜树|木瓜树]]、[[大叶女贞|大叶女贞]]、[[紫玉兰|紫玉兰]]、[[白玉兰|白玉兰]]等许多较珍贵的树木花草，并且每棵植物上都被挂上关于植物介绍的小说明{{参|{{cite web|author1=郑建军|title=新鲜事！邯郸龙湖公园的植物有了“名片”|url=http://news.ifeng.com/gundong/detail_2011_09/03/8916912_0.shtml|website=凤凰网|publisher=长城网|accessdate=2017-05-07}}}}，草坪面积18万平米，建有造型各异的桥梁12座{{参|{{cite web|author1=李珊珊|title=邯郸市龙湖公园新栽6株大叶女贞树温暖过冬|url=http://handan.hebnews.cn/2014-12/08/content_4373050.htm|website=河北新闻网|publisher=中原商报|accessdate=2017-05-07}}}}{{参|{{cite web|author1=范文平|title=邯郸：龙湖公园园林工人为大叶女贞建“过冬房”|url=http://news.yuanlin.com/detail/2014121/202309.htm|website=中国园林网|publisher=邯郸晚报|accessdate=2017-05-07}}}}{{参|{{cite web|author1=郝群英|title=记者在龙湖公园看到|url=http://www.handannews.com.cn/new_epaper/hdwb/html/2014-03/11/content_8800.htm|website=邯郸之窗|publisher=邯郸日报|accessdate=2017-05-07}}}}。

随着滏阳河通航工程，龙湖公园在园内也修建了船舶码头，市民可以乘坐仿古游船欣赏滏阳河沿岸景色{{参|{{cite web|author1=赵耀光|title=邯郸市滏阳河游船复航 市民可乘船欣赏古城美景|url=http://hebei.hebnews.cn/2016-02/16/content_5339093.htm|website=河北新闻|publisher=燕赵晚报|accessdate=2017-05-07}}}}。

== 特色 ==
[[File:Park_view_of_Longhu_05.jpg|200px]]
公园在加强绿化和建湖的过程中，还注重其他景观区域的建设，例如建造了大型音乐喷泉、激光效果、水幕电影三部分组成的夜色水景。以龙为主题的大型音乐喷泉有「龙腾盛世」、「金龙吐水」、「蛟龙出水」、「天龙旋云」、「龙行天下」、「天宇翔龙」等{{参|name=参考3}}。湖内设计了主喷能达到108米，变换26种造型的大型音乐喷泉，还拥有70余台水泵，1000余盏水中灯{{参|{{cite web|author1=李珊珊|title=邯郸市龙湖公园对喷泉设施及景观灯进行维修|url=http://handan.hebnews.cn/2016-04/08/content_5439053.htm|website=河北新闻网|publisher=中原商报|accessdate=2017-05-07}}}}。春夏秋冬时节，各色花卉争奇斗艳。夜晚来临时，主喷头喷泉在音乐声中一柱冲天，特为壮观{{参|{{cite web|author1=李超|title=龙湖公园内的大型音乐喷泉经过修缮后再度喷发|url=http://www.handannews.com.cn/new_epaper/zysb/html/2016-07/08/content_130648.htm|website=邯郸之窗|publisher=邯郸日报|accessdate=2017-05-07}}}}。是市民休闲、健身、娱乐的去处之一{{参|{{cite web|title=来做追光族 邯郸“荧光夜跑团”为健康充电|url=http://news.ifeng.com/a/20150722/44224992_0.shtml|website=凤凰网|publisher=长城网|accessdate=2017-05-07}}}}。
