'''龙感湖管理区'''是位于[[湖北省|湖北省]][[黄冈市|黄冈市]][[黄梅县|黄梅县]]的一个[[行政管理区|行政管理区]]，它与'''龙感湖农场'''为[[一个机构两块牌子|两块牌子、一套班子]]<ref name="a1">{{Cite web |url=http://www.hbdfz.com.cn/detail/index/10191/138020 |title=龙感湖|accessdate=2018-10-29 |author= |date=2017-05-22 |work= |publisher=湖北地方网，来源：《湖北省湖泊志》|language=Zh-hans }}</ref>。龙感湖农场创建于1956年，经[[中华人民共和国国务院|中国国务院]]批准围湖垦建<ref name="a2">{{Cite web |url=http://www.hg.gov.cn/art/2017/6/16/art_7122_130229.html |title=龙感湖管理区简介|accessdate=2018-10-29 |author= |date=2017-06-16 |work= |publisher=黄冈市人民政府网站，来源：龙感湖管理区（农场）办公室|language=Zh-hans }}</ref>，系围垦[[龙感湖|龙感湖]]西南沼泽地而成。亦是[[湖北省人民政府|湖北省人民政府]]直属的14个大型[[国营农场|国营农场]]。农场总面积98.696平方千米。1996年9月1日<ref name="a1"/>，经批准设立黄冈市龙感湖管理区。2001年10月，管理区划归黄冈市管理。2006年，湖北省农垦改革后，黄冈市将龙感湖管理区视同县（市）级单位管理，授予县级人民政府行政许可和行政执法权<ref name="a2"/>。

== 辖区 ==

2013年时，下辖7个办事处、56个生产队和3个居民委员会<ref name="a1"/>。总人口5万人<ref name="a2"/>。

7个办事处分别是，[[严家闸办事处|严家闸办事处]]、[[芦柴湖办事处|芦柴湖办事处]]、[[洋湖办事处|洋湖办事处]]、[[沙湖办事处|沙湖办事处]]、[[春港办事处|春港办事处]]、[[塞湖办事处|塞湖办事处]]、[[青泥湖办事处|青泥湖办事处]]。
