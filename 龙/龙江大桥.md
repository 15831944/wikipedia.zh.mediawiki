'''龙江大桥'''也称作'''龙江特大桥'''，是[[云南省|云南省]][[保腾高速公路|保腾高速]]上的一座公路桥，是保腾高速的重点控制性工程<ref>{{cite web |url=http://society.people.com.cn/n1/2017/1118/c1008-29653792.html |title=桥  连起风景的风景 |author=李茂颖 |date=2017-11-18 |website=society.people.com.cn |publisher=人民网－人民日报 |access-date=2018-01-19 |quote=}}</ref>，属于双塔单跨的[[桥#按跨径分类|特大跨径]][[钢结构|钢]]{{le|箱梁桥|Box girder bridge|箱梁}}[[悬索桥|悬索桥]]<ref name="2017-12-14"></ref>，位于[[腾冲市|腾冲市]]和[[龙陵县|龙陵县]]交界。桥梁全长2,471米、宽33.5米，主跨度1,196米，项目概算投资19.55亿元<ref>{{cite web |url=http://www.mot.gov.cn/tupianxinwen/201604/t20160421_2016711.html |title=云南保腾高速公路龙江特大桥建成通车 |author= |date=2016-04-21 |website=www.mot.gov.cn |publisher=[[国家邮政局|国家邮政局]] |access-date=2018-01-20 |quote=}}</ref>。两个塔架分别高169.7米（保山岸）和129.7米（腾冲岸）<ref name="2016-04-22观察者"></ref>，索塔顶到江面最大差距为450米<ref name="2016-04-06薛丹">{{cite web |url=http://yn.people.com.cn/news/yunnan/n2/2016/0406/c228496-28091340.html |title=保腾高速龙江特大桥完成核载试验 预计五月前通车 |author=薛丹 |date=2016-04-06 |website=yn.people.com.cn |publisher=[[人民网|人民网]] |access-date=2018-01-19 |quote=}}</ref>，桥面距离河流{{convert|280|m|ft|abbr=on}}，使其成为[[世界最高桥梁列表_(桥面高度)|世界最高的桥之一]]<ref>{{cite web |url=http://www.dailymail.co.uk/news/peoplesdaily/article-3183778/Working-clouds-Breathtaking-photographs-Asia-s-largest-suspension-bridge-built-900-feet-stunning-river-valley.html |title=Working in the clouds: Breath-taking photographs show Asia's largest suspension bridge being built 900 feet above stunning river valley |author=EMILY CHAN |date=2015-08-03 |website=www.dailymail.co.uk |publisher={{le|DMG媒体|DMG Media|DMG Media}} |access-date=2018-01-19 |quote=}}</ref>，是亚洲山区跨径最大的钢箱梁悬索桥<ref>{{cite web |url=http://news.xinhuanet.com/photo/2016-05/19/c_128996541.htm |title=云南保山云雾飘渺 龙江大桥变“断桥” |author= |date=2016-05-19 |website=news.xinhuanet.com |publisher=中国新闻网 |access-date=2018-01-19 |quote=}}</ref><ref name="2016-04-06薛丹"></ref>，也是中国山区的第一座跨径超过千米的钢箱加劲梁悬索桥<ref name="2016-04-22观察者"></ref>。2016年5月1日通车，标志保腾高速全线通车，设计时速80公里，[[昆明市|昆明]]至[[腾冲市|腾冲]]的行车时间缩短到七个半小时<ref name="2017-12-14">{{cite web |url=http://uav.xinhuanet.com/2017-12/14/c_129765128.htm |title=【“飞阅”中国】航拍云南龙江大桥 飞架云端气势如虹 |author1=念新洪 |author2=陈畅 |date=2017-12-14 |website=uav.xinhuanet.com |publisher=新华网 |access-date=2018-01-19 |quote=}}</ref>。

==技术==

[[File:龙江特大桥主缆2.jpg|thumb]]

龙江大桥位于[[腾冲火山群|火山]]和[[云南地震带列表|地震]]带上，跨径布置为320米+1196米+320米<ref name="2015-10-31崔敏"></ref>，大桥主缆长度1,950米，两侧总计有65.9万米的338根索股<ref name="2016-04-22观察者">{{cite web |url=http://www.guancha.cn/Project/2016_04_18_357341_s.shtml |title=腾冲亚洲第一大桥即将通车：云雾飘渺 直通缅甸 |author= |date=2016-04-22 |website=www.guancha.cn |publisher=观察者网 |access-date=2018-01-19 |quote=}}</ref>，抗震等级按Ⅸ度设防<ref name="2015-07-31"></ref>，桥梁使用的缆索由[[奥盛集团|奥盛集团]]生产<ref>{{cite web |url=http://tech.ce.cn/news/201604/11/t20160411_10342281.shtml |title=奥盛缆索为云南龙江特大桥增添巍峨雄姿 |author= |date=2016-04-11 |website=tech.ce.cn |publisher=中国经济网 |access-date=2018-01-19 |quote=}}</ref>。龙江大桥共有97个钢箱梁梁段，主桥钢箱梁每个节段长12.4米、宽33.5米、高3米，重约154吨<ref name="2015-07-31"></ref>。桥梁建设共产生了5项技术创新，包括：
<br>1. 采用[[无人航空载具|无人机]]牵引先导索过[[瑞丽江|江]]
<br>2. 采用索股入鞍段预成型及架设技术
<br>3. 采用圆形缠丝加缠包带方式加除湿系统方式进行主揽防护
<br>4. 采用喷洒{{le|葡萄糖酸钠|Sodium gluconate}}作为{{le|缓凝剂|Retarder (chemistry)}}，配合水枪冲刷的方式进行锚碇[[混凝土|混凝土]]凿毛施工
<br>5. 在桥梁大体积混凝土中使用[[火山灰|火山灰]]作为混凝土外掺剂<ref name="2016-04-20陈鸿燕">{{cite web |url=http://news.xinhuanet.com/politics/2016-04/20/c_128914618.htm |title=云南龙江特大桥建成通车 |author=陈鸿燕 |date=2016-04-20 |website=news.xinhuanet.com |publisher=[[中央人民广播电台|央广网]] |access-date=2018-01-19 |quote=}}</ref>

==事故==

2016年4月22日上午11时15分，龙江特大桥龙陵岸引桥第五跨箱梁混泥土涂装（刷柒）项目施工过程中，吊篮角钢焊缝发生断裂后引起吊篮整体垮塌，从40米高空极速坠落，导致吊篮内正在施工的3名工人，1人当场死亡，2人送医院后抢救无效死亡，3人均为[[江苏省|江苏省]][[镇江市|镇江]]蓝舶科技股份有限公司员工。<ref>{{cite web |url=http://news.sohu.com/20160423/n445720121.shtml |title=云南龙江特大桥发生一起施工事故 3人死亡 |author= |date=2016-04-23 |website=news.sohu.com |publisher=昆明信息港 |access-date=2018-01-19 |quote=}}</ref>

==旅游==

保腾高速龙江大桥腾冲岸设置有[[公路服务区|服务区]]<ref name="2016-04-22观察者"></ref>，并将建设“龙江特大桥国际旅游区”<ref>{{cite web |url=http://www.yn.xinhuanet.com/travel/2018-01/19/c_136908406.htm |title=保山召开龙江特大桥国际旅游区项目建设推进工作会 |author= |date=2018-01-19 |website=www.yn.xinhuanet.com |publisher=保山市旅游发展委员会 |access-date=2018-01-19 |quote=}}</ref>。龙江大桥修建有游客观光步道，曾在2018年春节假期期间开放试运营，票价为每人30元，对保山本地人给予半价优惠<ref>{{cite web |url=http://www.baoshan.cn/2018/0224/16325.shtml |title=今年春节，龙江大桥彻底火了一把！ |author=<!--Staff writer(s); no by-line.--> |date=2018-02-24 |website=www.baoshan.cn |publisher=[[保山日报|保山日报]] |access-date=2018-03-17 |quote=}}</ref>。曾有人在龙江大桥成功完成低空[[跳伞|跳伞]]，然而桥梁管理部门表示将加强管理，不允许这样的危险活动再次发生<ref>{{cite web |url=http://www.yn.chinanews.com/news/2016/0429/13726.html |title=这个五一，咱们在云南玩点新鲜的 |author=韩帅南 |date=2016-04-29 |website=www.yn.chinanews.com |publisher=中国新闻网 |access-date=2018-01-19 |quote=}}</ref>。
