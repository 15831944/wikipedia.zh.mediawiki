'''龙华區'''是[[中华人民共和国|中华人民共和国]][[广东省|广东省]][[深圳市|深圳市]]下辖的一個[[市轄區|市轄區]]，辖区面积175.58平方公里，2010年末常住人口137.9万人（其中户籍人口11.4万人）。

== 地理 ==
龍華區位於深圳市中北部，東與[[龍崗區|龍崗區]]相鄰，南與[[福田區|福田區]]、[[南山区_(深圳市)|南山區]]、[[罗湖区|罗湖区]]相鄰，西與[[宝安区|寶安區]][[石岩街道|石岩街道]]、[[光明区|光明区]]相鄰，北與[[东莞市|東莞市]][[塘厦镇|塘廈鎮]]相鄰。{{深圳行政区划地图}}

== 历史 ==
“龍華”一名源於龍華墟，因早期移民建龍勝堂並於此建墟，取日漸繁華之意。

2011年12月30日，由[[寶安區|寶安區]]劃出<ref>{{Cite news|url=http://news.xinhuanet.com/politics/2011-10/27/c_111129220.htm|title=深圳筹划增设两个功能新区|last=|first=|date=2011-10-27|work=新華網|accessdate=2011-10-27}}</ref>并挂牌成立龍華新區<ref>{{Cite news|url=http://epaper.southcn.com/nfdaily/html/2011-12/27/content_7042756.htm|title=龙华、大鹏新区或本周成立|last=|first=|date=2011-12-27|work=南方日报|accessdate=2011-12-27}}</ref>，管委会驻[[觀湖街道|觀湖街道]]广场沿河路。

2016年10月11日，国务院批复同意设立龙华区<ref name=":0" />，管辖[[觀湖街道|观湖]]、[[观澜街道|观澜]]、[[龙华街道_(深圳市)|龙华]]、[[福城街道_(深圳市)|福城]]、[[民治街道|民治]]、[[大浪街道|大浪]]6个街道。2017年1月7日，龙华区正式挂牌成立<ref>{{Cite news|url=http://jb.sznews.com/html/2017-01/09/content_3704942.htm|title=龙华区坪山区揭牌成立|last=|first=|date=2017-01-09|work=晶报|accessdate=2017-01-10}}</ref>。

== 行政区划 ==
下辖观澜、龙华、大浪、民治、福城、观湖6街道，基层设有36个社区工作站和100个社区居民委员会。

== 功能組團 ==

=== 住宅區 ===
因為龍華區的[[民治街道|民治街道]]相對其他街道，最為接近[[福田區|福田區]]、[[羅湖區|羅湖區]]等商業中心。不少在原[[深圳經濟特區|深圳經濟特區]]內工作的人，選擇居於民治，使民治發展成一個以住宅為主的區域。

=== 商業區 ===
龍華區的商業區主要遍佈于[[深圳地铁|深圳地鐵]][[深圳地铁4号线|4號線]]沿途各站：[[民樂站_(深圳)|民樂站]]的上河坊購物樂園、[[白石龙站|白石龍站]]的星河COCO City、[[深圳北站_(地铁)|深圳北站]]的繽果空間、[[红山站_(深圳)|紅山站]]的九方購物中心等。

=== 旅遊區 ===

==== 羊台山 ====
{{main|羊台山}}
羊台山位於龍華區大浪街道、寶安區石岩街道、南山區桃源街道之間，於2004年「羊台疊翠」被評為[[深圳八景|深圳八景]]之一。主峰[[海拔|海拔]]581.7米 ，為[[深圳市|深圳市]]西部的最高點。

==== 鐵路 ====
