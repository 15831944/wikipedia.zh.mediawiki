'''龙云'''{{BD|1884年|11月19日|1962年|6月27日|L龙}}，[[譜名|譜名]]'''登雲'''，字'''志舟'''。[[云南|云南]][[昭通|昭通]]人，[[彝族|彝族]]人，族名'''纳吉鸟梯'''。是[[中华民国|中华民国]]初期[[滇军|滇军]]将领，亦是割据[[中国西南地区|西南地区]]的[[軍閥|軍閥]]，號稱'''云南王'''，曾任[[雲南省|雲南省]][[主席|主席]]、军事参议院院长、[[中華民國總統府戰略顧問|中华民国总统府战略顾问会]]副主任、[[中华人民共和国中央人民政府|中华人民共和国中央人民政府]]委员、[[中华人民共和国国防委员会|中华人民共和国国防委员会]]副主席、[[中国人民政治协商会议全国委员会|中华人民共和国全国政协]]常委等职。

[[File:Long_and_Chiang.jpg|缩略图]]第一次在雲南見面合照]]

== 早年 ==
龙云早年参加过反清[[辛亥革命#革命党：推翻满清|革命黨]]，1911年加入[[滇军|滇军]]，1914年毕业于[[云南讲武堂|云南讲武堂]]第4期，入云南都督[[唐继尧|唐继尧]]部，为唐所赏识，被唐从下级军官逐步提拔为侍卫队长，直至滇军第五军军长。

== 云南王 ==
[[File:Chaing_Kai_Sheck_and_Governor_Lung_Yuen_of_Yunnan_Province_in_1941.jpg|缩略图]]
* 1927年龙云与[[胡若愚_(雲南)|胡若愚]]等策动“二六政变”迫使[[唐继尧|唐继尧]]下台，之后在与胡若愚争夺云南统治权的斗争中获胜，1928年被[[国民政府|国民政府]]正式任命为云南省主席。除了國民政府的任命，龍雲在主政雲南時也表現許多服從國民政府的作為，像是在[[中原大戰|中原大戰]]期間其屬下滇軍在南京政府授命下攻擊廣西；以及在1928年清除雲南的共產黨，並多次在其控制區實施清黨，有效穩固其統治力。至1945年，共主政云南17年，被称为“云南王”。

*[[抗日战争_(中国)|抗日战争]]期间兼任陆军副总司令，龙云曾派遣其亲信[[卢汉|卢汉]]先后派遣滇军20多万赴抗战前线，参加了包括“[[台儿庄战役|台儿庄战役]]”等二十多次战役，雖然在其他军阀來看滇軍装备不良，但仍然在战斗中以惨烈的伤亡完成任務，同時云南在抗戰末期成為重要的兵源提供地，为抗战胜利作出巨大贡献。<ref>{{cite web|title=卢汉：四万子弟出云南 六十军血战台儿庄|url=http://news.ifeng.com/history/1/jishi/200811/1119_2663_884894_2.shtml|publisher=凤凰网历史频道综合|date=2009-11-19}}</ref>

*1938年12月19日，[[汪精卫|汪精卫]]夫妇、[[陶希圣|陶希圣]]等十余人，乘坐龙云包租的专机，离开昆明飞抵越南。<ref>{{cite web|title=汪精卫:从“爱国者”到汉奸|url=http://news.ifeng.com/history/special/wangjingwei/200912/1206_8848_1463946_6.shtml|publisher=凤凰网历史综合|date=2009-12-06}}</ref>

*1943年-1945年10月，[[中国共产党|中国共产党]]委派时任南方局宣传部长的[[华岗|华岗]]，到云南与龙云联络，试图争取其支持中共，双方建立了长期的沟通，龍雲本人則在1944年加入[[中國民主同盟|中國民主同盟]]，后因1945年10月蒋介石突然解除其武装被软禁而告终，但为后来龙云加入[[中华人民共和国|中华人民共和国]]新政府奠定了基础。<ref>{{cite web |url=http://www.xinhai.org/shi/191101835.htm |title=龙云}}</ref>

*1944年2月5日，龙云部冒充土匪，抢劫美国军械一事，美国因此一度不发中国武器；而蔣中正因此在日記內道出“龍雲跋扈，尤可憂慮”此類不滿之言。

*1945年10月3日，由於龍雲長時間與中國共產黨交往密切，同時抗戰勝利後雲南地位已由過去邊境重鎮變得無足輕重，促使[[蔣中正|蔣中正]]出兵鎮壓龍雲。鎮壓部隊由當時駐昆明的[[國民革命軍第五軍|國民革命軍第五軍]]主導，第五軍在優勢戰力下包圍五華山雲南省政府，並解除龍雲的省長職務。10月6日，龙云被國民革命軍移交飞赴成都出任“军事参议院院长”的虚职，由當時[[中華民國空降特戰部隊|傘兵第6隊]]與第9隊擔任儀隊，傘兵副司令張緒滋親任儀隊隊長，組成300人的儀隊到機場恭送龍雲赴成都上任，雲南的治權則轉交到他的親信盧漢手中。

*1946年5月。[[国民政府|国民政府]]还都[[南京|南京]]，龙云也跟隨政府至南京，並被軟禁在中央路156号洋房居住。而後[[军事参议院|军事参议院]]取消，龙云被任命为新成立的[[中華民國總統府戰略顧問|战略顾问委员会]]副主任。1948年12月8日，在[[陈纳德|陈纳德]]等人帮助下，龙云乘[[C-47運輸機|C-47運輸機]]从[[南京|南京]]到达广州，并乘船到达[[香港|香港]]，结束了三年两个月的软禁生活<ref name="《虎口脱险-陈纳德航空队运送龙云逃离南京纪实》">张增智：《虎口脱险-陈纳德航空队运送龙云逃离南京纪实》，《纵横》杂志，1984年第2期</ref>，在香港加入[[中國國民黨革命委員會|中國國民黨革命委員會]]。

* 在1949年10月中華人民共和國成立時，龍雲列入中央人民政府委員，中华人民共和国成立后1950年1月龍雲从香港赴[[北京|北京]]，加入[[中华人民共和国|中华人民共和国]]政府。历任[[中华人民共和国国防委员会|国防委员会]]副主席、[[中国人民政治协商会议|中国人民政治协商会议]]常委、[[民革|民革]]中央副主席、西南军政委员会副主席等职。1956年曾出访[[苏联|苏联]]、[[罗马尼亚|罗马尼亚]]、[[捷克斯洛伐克|捷克斯洛伐克]]、[[南斯拉夫|南斯拉夫]]等[[东欧|东欧]]国家。

== 保护西南联大 ==
龙云主政期间予雲南境內知識份子批評中央的言論自由，给[[西南联大|西南联大]]提供了相当程度的保护，直到其失势下台，炎黄杂志《龙云与民主堡垒西南联大》<ref>黎勤、李凌：龙云与民主堡垒西南联大，炎黄春秋杂志，2005年第4期    </ref>里面专文写道：
{{Quote|
西南联大所以被称为民主堡垒，其原因是多方面的……但还有一个必不可少的客观有利条件，这就是联大地处云南昆明，而当时云南省主席龙云对爱国民主运动持比较开明的态度，他自己也于1944年秘密加入民主同盟。有人说：'''如果西南联大在四川或其它蒋介石可以直接控制的地方，西南联大就不可能保持学术自由、兼容并包的原则'''，也不可能成为民主堡垒。这句话是符合实际的。
}}

《剑桥中华民国史1912-1949》下卷，也描写到龙云保护了西南联大的部分教授，“反对国民党独裁的联大教授从地方军阀云南省主席龙云得到支持，他阻碍了中央政府代理人控制大学校园政治思想的措施。”<ref>《剑桥中华民國史1912-1949》下卷，p473,中国社会科学出版社，1998</ref>，“当1945 年 11 月昆明学生领导反内战示威时，云南的政治气候已在变化；云南省长龙云虽然在战时与中央政府合作，但也保护了学术界。”<ref>《剑桥中华民國史1912-1949》下卷，p474,中国社会科学出版社，1998</ref>

具体在如下几次事件当中可见：
* “1941年3月上旬，蒋介石派[[三青团|三青团]]组织处长[[康泽|康泽]]到昆明，要抓中共控制的学生，遭到龙云抵制，康泽未能得逞，悻悻而去。”
* 1942年1月7日，西南联大学生發起讨孔游行，龙云又一次以“等因奉此”的公文旅行来应付，不采取实际行动，康泽又一次悻悻而去。

== 晚年 ==
===三子被殺===
1950年，中共云南军区指称[[龍繩曾|龍繩曾]]出任“滇东北军政长官”、“川滇黔康反共救国军”总司令，最终在老家昭通被解放军所杀，<span class="noprint" style="background-color:#FFEFD6;" title="{{lan|zh-hans=需要提供文献来源|zh-hant=需要提供文獻來源}}">一起被杀的还包括13岁、11岁的儿子、9岁的女儿和襁褓中的儿子</span><sup class="noprint Template-Fact">[[Wikipedia:列明来源|<span style="white-space: nowrap;" title="{{lan]]</sup>。只有一个儿子幸存逃过此劫，今在美国。当时为此事，朱德和周恩来受毛泽东之委托，曾到龙云家抚慰并称，中央定彻底调查。

===劃為右派===
1957年初，在[[整風运动|整風运动]]中，龙云曾在[[全國人大|全國人大]]的雲南組會議發言，主要表達了三層意思：「第一，[[蘇聯|蘇聯]]是我們[[社會主義陣營|社會主義陣營]]的老大哥。可是，他們太欺負我們的黨和國家了，借我們的債，要高[[利息|利息]]，要限期歸還。第二，雲南解放以前，雲南的農村已經遭到天災人禍的破壞，解放後應該讓人民休養生息。第三，我們[[涼山|涼山]]的彞族區域，還處在[[奴隸社會|奴隸社會]]階段，社會十分落後。應該循序漸進地把他們帶進[[社會主義|社會主義]]社會來。一來就搞[[階級鬥爭|階級鬥爭]]，實行[[土地改革|土地改革]]，我覺得為時尚早。」發言後，原滇軍舊部屬[[張沖|張沖]]就很擔心地私下對他說：「哎，你發言為什麼不事前找我商量一下，你對於當前的形勢一點也不明白，現在正在運動開始的時候，怎麼可以隨便潑冷水呢？」又說：「話說得是對的，可是這些話你只合私下向黨中央說，怎麼可以在群衆中間亂說呢？」盧漢回家後也說：「老主席（龍雲）太孟浪，他說的話雖然在情在理，但是在『[[反右|反右]]』的風頭上，怎麼能說這些話呢？要挨，要挨！」之後龍雲被扣上「反黨」、「反社會主義」兩項罪名，成為「大右派」，也受到昔日的部下、之後的[[中國民族列表|少數民族]]幹部批判，被指為「土皇帝」、「[[家天下|家天下]]」<ref>[http://history.people.com.cn/BIG5/n/2012/0724/c198307-18588554-5.html 龍雲：被打成右派的“雲南王”【5】--文史--人民網]</ref><ref>[http://magazine.sina.com/bg/southernpeopleweekly/260/20110713/1919112275.html 滇系三巨頭：從蔡鍔到龍雲 終_南方人物周刊_雜誌頻道_新浪網-北美]</ref>。

===身後===
1962年，龍雲在[[北京|北京]]去世。[[文化大革命|文化大革命]]期间，他的家被北航[[红卫兵|红卫兵]]抄家封门，夫人[[顾映秋|顾映秋]]也遭批斗，他在[[云南石林|石林]]上的题字也被刮掉。1980年其右派问题被平反，之后，其在[[云南石林|石林]]上的题字又被放在石林上。
