'''龙普腾事件'''或称'''朱文文事件'''是1964年间发生于中国[[湖南|湖南]]的一起由学生自杀引发，在[[中共湖南省委|中共湖南省委]]主导下引起的刑事[[冤案|冤案]]。被认为是1962年底开展的[[四清运动|四清运动]]中[[整肃|整肃]]教师的典型事件。[[湖南省|湖南]]全省受此事件牵连的干部、教师、学生达上千人<ref name="a1"/>。

== 事件过程 ==

'''朱文文'''（1946年－1964年6月<ref name="a1"/>），[[湖南省|湖南省]][[攸县|攸县]]人，1962年秋，[[攸县一中|攸县一中]]初中毕业，后考入[[茶陵县第一中学|茶陵一中]]，是公认的[[校花|校花]]。一次劳动课后，因不满班主任龙普腾（物理老师，时年29岁{{noteTag|1=1983年5月《湖南教育》报道称48岁，倒推至1964年为29岁。}}<ref name="a2"/>）批评她有[[资产阶级意识|资产阶级小姐意识]]，爱打扮，不爱劳动，而在离校约500米的铁牛潭投江自杀身亡。另说<ref name="a1"/>，朱文文申请[[助学金|助学金]]，班主任龙普腾上报学校，未获批准。两人发生争执，后朱文文又与同学争执。事后，朱文文自杀身亡。

== 事件处理 ==
家属在她死后要求中共茶陵县委“查明真相，追查凶手”，朱文文安葬于[[烈士陵园|烈士陵园]]，举行[[追悼会|追悼会]]，茶陵一中师生[[披麻戴孝|披麻戴孝]]为朱文文送葬。朱文文家属要求得到县委落实，但引起茶陵一中师生[[逆反心理|逆反心理]]，导致当地民众产生不同观点。<ref name="a1"/>

[[中共湖南省委|中共湖南省委]]的某位[[书记|书记]]{{noteTag|1=李若建原文仅称“一位省委书记”。1960年至1970年间，除湖南[[省委书记|省委书记]]除第一书记[[张平化|张平化]]一人外，另有书记处书记：[[徐启文|徐启文]]、[[于明涛|于明涛]]、[[华国锋|华国锋]]、[[周礼|周礼]]、[[周惠|周惠]]、[[胡继宗|胡继宗]]、[[谭余保|谭余保]]、[[李瑞山|李瑞山]]。}}在朱文文家属的申诉材料上，批示：“[[阶级斗争|阶级报复]]”，由湖南省、[[湘潭专区|湘潭专区]]、茶陵县三级政府组成联合工作组。工作组为迎合这位书记，将事件定性为反革命阶级报复事件。龙普腾的[[家庭成分|家庭成分]]由[[中农|中农]]改为[[富农|富农]]。龙普腾本人定性为[[反革命|反革命]]，判处[[无期徒刑|无期徒刑]]。对事件持不同意见的210人受株连[[处分|处分]]。61名中小学教师被清理。株连上千人<ref name="a1">{{Cite web |url=http://www.hybsl.cn/beijingcankao/beijingfenxi/2014-06-25/46944.html |title=李若建：改革开放前社会流动视角下中小学教师研究 |accessdate=2016-09-01 |author=李若建 |date=2014-06-25 |work= |publisher=胡耀邦史料信息网，来源：《开放时代》2014年第3期|language=Zh-hans }}</ref>。

据说其哥哥朱某当时在部队任指导员，朱某在朱文文死后纠缠不休，直接告到黨中央，为平息事件，国家主席[[刘少奇|刘少奇]]亲自批示：「严肃处理」，最後将班主任[[龙普腾|龙普腾]]判处无期徒刑。

== 平反 ==
