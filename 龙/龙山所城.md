'''龙山所城'''位于中国[[浙江省|浙江省]][[宁波市|宁波]][[慈溪市|慈溪市]][[龙山镇_(慈溪市)|龙山镇]]龙山所村，现存部分城墙遗址及护城河，1986年8月被公布为第三批慈溪县文物保护单位，2016年11月龙山所村被公布为第四批宁波市历史文化名村。

龙山千户所为明朝设置的卫所，洪武二十年（1387）信国公[[汤和|汤和]]奉诏建置，属[[观海卫城|观海卫]]。治所位于[[宁波府|宁波府]][[慈谿县|慈谿县]]北七十里，介于[[观海卫城|观海卫]]与[[镇海区_(宁波市)|定海县]]之间，东北为伏龙山，东南为石塘山<ref name="城池史">民国·杨泰亨修，冯可镛纂，《民国慈谿县志·卷二·建置·城垣》：龙山所城，郡治北七十里，明洪武二十年信国公汤和以定海县龙头场石塘团之址筑城凿池，建门一。永乐十六年都指挥使谷祥增高八尺，环置敌楼，增建东南西三门（卫志初制止开北门，谷祥塞北门，开东西南三门，又开水门一），各冠以楼，外罗月城，浚子池于城之外。城周回三里一十七步，高二丈五尺，广二丈，上列敌楼警铺各二十，雉堞八百五十六，吊桥三、构于东西南三门之外。池正濠周回五百六十二丈，广六尺，深二丈三尺，备濠五百一十丈，广一丈三尺，深一丈五尺。</ref>。
     
== 所城建筑 ==
龙山城墙原呈梯形，城周三里一十七步（约1467.2米），高二丈五尺（约8米），厚二丈（约6.4米）。设东西南城门三座，门外均设瓮城与吊桥，水门一座，城内设十字街。护城河有正濠与备濠之分，正濠周五百六十二丈（约1798.4米），宽六尺（约1.92米），深二丈三尺（约7.36米），备濠周五百一十丈（约1632米），广一丈三尺（约4.16米），深一丈五尺（约4.8米）<ref name="城池史" />。今城门已毁，城墙遗址尚存，南北高4米，东西高2-3米，宽8-10米
<ref>{{cite web |url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=983&type=2|title=《龙山所城》|accessdate=2018-04-30|work=|publisher=宁波文化遗产保护网}}</ref>。城内尚存[[朱祖燮故居|朱祖燮故居]]（环城路102号，现为慈溪市文物保护点）等具有代表性的传统民居<ref>{{cite web |url=http://epaper.cxnews.cn/html/2016-08/31/content_7_6.htm|title=《走进龙山所村 寻访老宅旧迹里的时光记忆》|accessdate=2018-04-30|work=|publisher=慈溪日报}}</ref>。
