'''{{PAGENAMEBASE}}'''是{{PRC admin/fullname|33/03/27/101/000|3}}下辖的一个[[行政建制镇|镇]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|33/03/27/101/000}}}}</ref>，被称为“中国农民第一城”。<ref>[http://news.xinhuanet.com/politics/2009-11/05/content_12391869.htm “中国第一座农民城”温州龙港新观察] {{webarchive|url=https://web.archive.org/web/20091108095826/http://news.xinhuanet.com/politics/2009-11/05/content_12391869.htm |date=2009-11-08 }}</ref>
'''龙港镇'''是联合国开发署可持续发展试点镇、国家新型城镇化综合试点镇、全国小城镇建设示范镇、全国小城镇综合改革试点镇、浙江省中心镇和小城市培育试点镇、浙江省城乡统筹现代商贸服务示范镇和温州市强镇扩权改革试点镇。<br/>
截至2015年，龙港镇辖17个社区，28个居民区，171个行政村，幅员面积172.05平方公里，总人口达50万。

== 历史沿革 ==
[[File:Cangnan_-_Pacao_-_P1210312.JPG|thumb]]
龙港镇原来只是[[平阳县|平阳县]]的经济中心[[鳌江镇_(平阳县)|鳌江镇]]对岸的一个村落，名叫'''方岩下'''。<br/>
1981年[[平阳县|平阳县]]南部分设[[苍南县|苍南县]]，由于缺少作为经济中心的城镇，地方政府给农民优惠政策，吸引农民在此兴建房屋，从事工商业，形成“中国农民第一城”。<br/>
1983年10月19日，浙江省人民政府（浙政[1983]148号)批复同意设立龙港镇，为苍南县直辖镇。<br/>
1984年，划沿江乡方岩村、河底高村；龙江乡金钗河村、江口村、下埠村组建龙港镇。<br/>
1984年6月，苍南县撤区建镇，实现以镇辖乡，龙港镇辖白沙乡、龙江乡、沿江乡、湖前乡4乡。<br/>
1985年2月，因镇管乡不符合宪法规定，又恢复区建制，白沙、龙江、沿江、湖前4乡复归宜山区。<br/>
1985年4月，设龙港区，辖白沙、龙江、沿江、湖前和海城乡共5乡<br/>
1988年10月，湖前镇改为直属镇。<br/>
1992年，沿江、龙江、白沙、海城四乡并入龙港镇。<br/>
2000年，撤销湖前镇、平等乡、江山乡，所属行政区域（除江山乡后垟增、梁宅2村）并入龙港镇。<br/>
2011年在温州市新一轮乡镇合并中，舥艚镇、芦浦镇、云岩乡并入龙港镇，调整后的龙港镇下辖3个社区、25个居民区、171个行政村，新版图达172.05平方公里，总人口达50万，初具一座中等城市规模。<br/>
2012年5月，龙港镇将12个办事处调整为17个社区。

2014年12月29日，龙港镇获批国家新型城镇化综合试点，成为全国第一批64个试点地区中仅有的两个镇级试点之一。

2016年，龙港镇将原有的17个社区整合为14个新型社区

2016年6月4日，成立苍南县龙港设市筹备工作领导小组，负责推进龙港“[[镇改市|镇改市]]”（即特大镇升格县级市）筹备工作

== “大部门制”改革 ==
根据试点工作部署，龙港镇按照层级减少、机构精简、成本节约的原则，把苍南县派驻龙港部门与镇内设机构及事业单位共41个机构合并为15个，原有1175名人员初步精简核定编制为736名。在社会管理层面，原有17个社区将整合为14个新型社区，人员从462名初步精简拟定编制为393名，逐步建立“市管社区”的体制。“大部门”实行以块为主的管理模式，取代过去“四大班子”的行政管理模式，避免出现“麻雀虽小五脏俱全”的浪费现象。

2016年6月4日，龙港镇15个大部门正式挂牌，投入实质性运行。<ref>{{Cite web|url=http://news.cnlg.cn/20160604/102991.htm|title=龙港15个“大部门”今挂牌 “镇改市”迈入新阶段 - 新闻中心 - 龙港网|accessdate=2016-06-04|work=news.cnlg.cn}}</ref>

== 人口 ==
全镇人口50万,主要通行汉语方言为苍南瓯语和蛮话。西南部份村落通用語言為閩南語。

== 经济 ==
龙港镇连同对岸历史较为悠久的[[鳌江镇|鳌江镇]]（人口20万），事实上已经形成一座中等规模的工商业城市，是整个温州南部诸县（苍南、平阳、[[泰顺|泰顺]]、[[文成|文成]]）以及福建北部[[福鼎|福鼎]]等地的经济中心。
2010年，实现GDP133.1亿元，工业总产值300.7亿元，社会消费品零售总额40. 7亿元，财政收入12.2亿元。
*龙港镇是全中国最重要的[[印刷|印刷]]基地之一。
