'''龙德格·普日布苏伦'''（Lundeg Purevsuren，{{bd|1964年|12月24日||}}）[[蒙古国|蒙古国]]人，蒙古国政治人物。

==生平==
1989年毕业于[[莫斯科国际关系学院|莫斯科国际关系学院]]。后来曾在[[德意志联邦共和国|德意志联邦共和国]]联邦外交部外交学院、[[新西兰|新西兰]][[惠灵顿维多利亚大学|惠灵顿维多利亚大学]]、[[美国|美国]][[夏威夷|夏威夷]]{{link-en|亚太安全研究中心|Asia-Pacific Center for Security Studies|亚太安全研究中心}}、[[哈佛大学|哈佛大学]][[肯尼迪学院|肯尼迪学院]]接受培训<ref name=nikkei>{{Cite web |url = http://future-of-asia.nikkei.jp/asia2016/eng/speaker29.html |title =Lundeg Purevsuren  |author =  |date =  |publisher =Nikkei  |language =  |accessdate = 2017-06-27 |quote =  }}</ref>。

1989年至1991年，在[[蒙古人民共和国|蒙古人民共和国]]外交部公共行政司工作。1991年至1993年，在德意志联邦共和国驻蒙古大使馆工作。1993年至1995年，在蒙古国外交部欧洲非洲司工作。1995年至2000年，在蒙古国驻[[德国|德国]][[波恩|波恩]]的大使馆工作。2001年至2009年，在蒙古国外交和贸易部欧洲司工作。2009年至2014年，任[[蒙古国总统|蒙古国总统]]的国家安全和外交政策顾问。2014年12月10日，[[国家大呼拉尔|国家大呼拉尔]]任命他为[[赛汗比勒格内阁|赛汗比勒格内阁]]的蒙古国对外关系部部长<ref name=nikkei/><ref>{{Cite web |url =http://www.chinanews.com/gn/2015/03-26/7161877.shtml  |title =  蒙古外长普日布苏伦将访华|author =  |date =2015-03-26  |publisher =中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。
