'''亓磊'''（{{lang-en|Stanley Qi}}），1983年出生于[[山东省|山东省]][[潍坊市|潍坊市]]，[[斯坦福大学|斯坦福大学]]生物工程系和化学与系统生物学系助理[[教授|教授]]，斯坦福大学多学科研究所ChEM-H学者，dCas9分子和CRISPR干涉（CRISPRi）技术的发明者，加州大学CRISPR基因编辑技术中国和[[欧盟|欧盟]]专利的共同发明人。<ref>{{cite web |title=Tsinghua alumni won the 2017 Sloan Research Award |url=http://news.tsinghua.edu.cn/publish/thunews/10303/2017/20170224092702742583332/20170224092702742583332_.html |website=Tsinghua University News |accessdate=24 February 2017}}</ref><ref>{{cite web |title=BioE Rising Star Seminar |url=http://bioeng.berkeley.edu/seminars/rising-star |website=berkeley bioengineering |accessdate=11 April 2018}}</ref>

== 人物经历 ==
2005年获清华大学物理与数学学学士学位，2016年赴美留学，师从[[朱棣文|朱棣文]]教授（1997年华人诺贝尔物理奖得主），2012年获得[[加州大学伯克利分校|加州大学伯克利分校]]生物工程博士学位。在2012年博士毕业后，他进入加州大学旧金山分校（UCSF）,作为合成生物学和系统生物学方向的独立研究员开展CRISPR技术开发的研究，并建立生物工程学实验室。他发明的CRISPR应用于转录和表观遗传调控的CRISPR干扰（CRISPRi）与CRISPR激活（CRISPRa）技术，将CRISPR技术应用在哺乳动物系统和人类基因调控编辑中，于2014年加入斯坦福大学。<ref>{{cite web |title=Stanley Lab in Stanford |url=http://med.stanford.edu/qilab.html}}</ref>

== 荣誉 ==
2013年，他的研究工作赢得了国家卫生署署长的早期独立奖。

2016年，获得皮尤学者称号，同年，被《细胞科学杂志》（The Journal of Cell Science）评为“受关注的细胞科学家”（Cell scientist to watch）称号，并被生物通评选为2016年生命科学十大风云人物。<ref>{{cite web |title=NIH Director's Early Independence Award Recipients 2013 Awardees |url=https://commonfund.nih.gov/earlyindependence/AwardRecipients13 |website=NIH}}</ref>

2017年，获得美国斯隆基金会（Alfred P. Sloan Foundation）颁发的斯隆研究奖<ref>{{cite web |title=Pew Biomedical Scholars |url=https://www.pewtrusts.org/en/projects/pew-biomedical-scholars/directory-of-pew-scholars/2016/lei-qi |website=Pew}}</ref><ref>{{cite web |title=Sloan Foundation  Past Fellows |url=https://sloan.org/past-fellows |website=Sloan Foundation Web}}</ref>
