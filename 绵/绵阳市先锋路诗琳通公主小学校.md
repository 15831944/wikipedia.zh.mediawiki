'''绵阳市先锋路诗琳通公主小学校'''（{{lang-th|โรงเรียนเหมียนหยางเซียนเฟิงลู่สิรินธร}}，{{lang-en|Mianyang Xianfenglu Sirindhorn Primary School}}<ref>{{cite web |url= http://www.thaiworld.org/th/thailand_monitor/answer.php?question_id=985|title=โรงเรียนประถมศึกษาเหมียนหยางเซียนเฟิงลู่สิรินธร ( 绵阳市先锋路诗琳通公主小学校) |last= |first= |date=2010-07-30 |website=thaiworld.org |publisher= |access-date= |quote=}}</ref>），原名'''绵阳市先锋路小学'''，位于[[四川省|四川省]][[绵阳市|绵阳市]][[涪城区|涪城区]]先锋路，始建于1965年。教学楼面积{{convert|7156.15|m2}}<ref name="sohu" />，可容纳2000人<ref name="cnr">{{cite web |url=http://www.cnr.cn/2010zhuanti/wcdzlzn/tpbd/201005/t20100515_506428942.html |title=中泰友谊之花：先锋路诗琳通公主小学 |last= |first= |date=2010-05-13 |website=中广网 |publisher= |access-date= |quote=}}</ref>。

2008年5月12日[[汶川大地震|汶川大地震]]导致学校设备严重损毁。2009年4月，[[泰国|泰国]][[诗琳通公主|诗琳通公主]]来绵阳市考察时，决定捐赠1000万元人民币重建先锋路小学，重建后改为现名。<ref name="xfjj" /><ref name="cnr" /><ref>{{cite web |url=http://www.gxnews.com.cn/staticpages/20100516/newgx4befd80b-2949612.shtml |title=[新汶川]泰国公主捐款绵阳建小学 |last= |first= |date=2010-05-16 |website=广西新闻网 |publisher= |access-date= |quote=}}</ref>

== 校门及校徽 ==
[[File:Xianfeng_logo.jpg|thumb]]
校门前有该学校的[[泰文|泰文]]及[[英文|英文]]题字；下方碑记是《先锋赋》，用[[文言文|文言文]]描述了先锋路小学的命名来源、校训、毁坏和重建记载<ref>[http://chuansong.me/n/2562690852017?jdfwkey=4ut6b 烹文煮字或咬文嚼字：句句得来不寻常]</ref>。校徽即[[诗琳通公主|诗琳通公主]]的{{link-en|皇家记号|Royal cypher|皇家标志}}，由[[蓝紫色|蓝紫色]][[泰文|泰文]]字母“{{lang|th|ส}}”和[[白色|白色]]泰文字母“{{lang|th|ธ}}”组成，上有一顶{{link-en|泰式王冠|Makuṭa}}。<ref>{{cite web |url= http://www.komchadluek.net/news/foreign/77324|title=ร.ร.เหมียนหยางเซียนเฟิงลู่สิรินธรหนึ่งน้ำใจมิตรแท้สู่แดนมังกร |last= |first= |date=2010-10-25 |website=komchadluek |publisher= |access-date= |quote=}}</ref><ref>{{cite web |url=http://www.chitraladaschool.ac.th/view_activity.php?Act_ID=241 |title=โครงการแลกเปลี่ยนวัฒนธรรมจิตรลดา-เหมียนหยาง ครั้งที่ 7 ณ มณฑลเสฉวน สาธารณรัฐประชาชนจีน |last= |first= |date= |website=吉拉达学校 |publisher= |access-date= |quote=}}</ref>

先锋教育标志是上红下蓝的圆形，中间是由“先锋”两字的[[汉语拼音|汉语拼音]]首字母“X”和“F”组成的白色图案，形似快速飞翔的[[老鹰|老鹰]]。

== 诗琳通公主访问小学 ==
=== 第一次 ===
2010年4月8日，诗琳通公主参加了该校的落成典礼，并向学校捐赠了10万元和一些书籍。

=== 第二次 ===
2011年4月9日，诗琳通公主第二次来到此校向学生赠书，与泰国的{{link-th|吉拉达学校|โรงเรียนจิตรลดา}}结为姊妹学校。随后四川的一位企业家{{谁}}向诗琳通公主赠送一尊[[普贤菩萨|普贤菩萨]]像。<ref>{{cite web |url=http://www.chinanews.com/gn/2011/04-09/2961721.shtml |title=泰国诗琳通公主再访四川震灾区小学(图) |last= |first= |date=2011-04-09 |website=中新网 |publisher= |access-date= |quote=}}</ref>

=== 第三次 ===
2018年4月7日，诗琳通公主第三次访问小学，她用流利的汉语演讲说：“好久没来了，我很想你们。这次来看到你们的校园更漂亮，孩子们进步更大了，我非常高兴。[[孔子|孔子]]说：‘[[q:论语#學而第一|学而时习之，不亦说乎？]]’希望你们爱学习、爱生活，不断进步”<ref>{{bilibili|av22232447|诗琳通公主说中文}}</ref>。全校师生为迎接公主制作了一个小花亭，让公主坐在其中观看师生表演。该表演与泰国吉拉达学校合作演出，其中包括歌曲《[[心爱的公主|心爱的公主]]》<ref>{{cite web |url=https://3g.163.com/dy/article/DEOIJDGM0517PFUK.html |title=声声不息 一份送给泰国诗琳通公主的礼物 |last= |first= |date=2018-04-07 |website=网易 |publisher=今日放猛料 |access-date= |quote=}}</ref>、《[[大象_(泰国儿歌)|大象儿歌]]》、《[[甜蜜蜜_(邓丽君歌曲)|甜蜜蜜]]》等。<ref>{{cite web |url=http://www.xflxx.com/index.aspx?menuid=4&type=articleinfo&lanmuid=6&infoid=1616&language=cn |title=泰王国诗琳通公主“回家”，激励学生关注学校发展 |last= |first= |date= |website=绵阳市先锋路小学 |publisher= |access-date= |quote=}}</ref>
