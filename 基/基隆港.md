'''基隆港'''是位於[[臺灣|臺灣]]北部[[基隆市|基隆市]]的[[港口|海港]]，為[[臺灣|臺灣]]四座國際[[商港|商港]]之一、以及[[北臺灣|北臺灣]]首要的[[海運|海運]]樞紐，營運機構為[[臺灣港務公司|臺灣港務公司]][[臺灣港務股份有限公司基隆港務分公司|基隆港務分公司]]。整個[[港區|港區]]被基隆[[市中心|市中心]]環繞，運輸方面以[[貨櫃|貨櫃]]為主、散貨為輔，並有數條國內外[[客輪|客輪]]航線固定彎靠。至2010年止，基隆港為世界第54大貨櫃港及臺灣第三大港。

== 港區環境及設施 ==
[https://www.taxicar-trip.com/ 基隆港]為利用自然之[[谷灣|谷灣]]地形所修築的港口，共有57座[[碼頭|碼頭]]、1個入港航道，港區總面積為572.17[[公頃|公頃]]。[[港灣|港灣]]形狀大致呈北寬南窄的漏斗狀，港區水域則分為外港、內港、牛稠港等3大區域。碼頭方面可分為東岸碼頭、西岸碼頭等2大部分，其中以西岸運量較大；使用類型則分為商用、軍用、漁用等3大類。

===碼頭簡介===
*貨櫃碼頭：15座
:設有三處貨櫃基地，配置有35至40噸可裝卸13至18排貨櫃之高性能式[[貨櫃|貨櫃]][[起重機|起重機]]<!-- Container crane -->，每年可裝卸200萬至300萬標準貨櫃箱（TEU）。
:#西岸貨櫃儲運場
:##北櫃場（舊稱第一貨櫃中心）
:##南櫃場（舊稱第三貨櫃中心）
:#東岸貨櫃儲運場（舊稱第二貨櫃中心）

*雜貨碼頭：24座
:設置有通棧、露置堆貨場，可供[[汽車|汽車]]、[[遊艇|遊艇]]、[[鋼鐵|鋼鐵]]等[[貨物|貨物]]裝卸之用。另設有[[水泥|水泥]]、卸[[煤|煤]]、[[石油|油品]]及其他散貨碼頭，配置水泥圓庫、自動卸煤機、化油儲槽、自動卸水泥設施等。

*客運碼頭：2座
:東岸及西岸各有一座（東2碼頭、西2碼頭），各設有一棟客運大廈。經常有國內外郵輪、客貨輪在此靠泊。

*其他用途碼頭：16座
:分別供港務公司所有港勤船，及工程船、軍艦、海巡艦、緝私艦等特種船隻靠泊。

===軍港===
基隆港東岸與西岸均有軍用碼頭。東岸為東5碼頭（入口標示為「海軍碼頭」）；東5碼頭入口對面設有海軍威海營區，是[[中華民國海軍|中華民國海軍]][[海軍一三一艦隊|一三一艦隊部]][[海軍基地|所在地]]；西岸則為西5—6與9—11B碼頭，由[[海軍基隆後勤支援指揮部|海軍基隆後勤支援指揮部]]（原海軍第三造船廠，簡稱「海三廠」）、[[陸軍後勤指揮部|陸軍第三地區支援指揮部]]運輸兵群第二營使用。

===漁港===
大沙灣以北，與[[和平島|和平島]]、[[八尺門|八尺門]]鄰近的水域為基隆港的[[渔业|漁]]用碼頭區（1998年正式劃為「漁業專用區」），又名「[[正濱漁港|正濱漁港]]」，為台灣重要[[遠洋漁業|遠洋漁業]]基地之一。

== 發展歷程 ==
[[File:Keelung-Garnot-Kelung-1894.jpg|thumb]]
[[File:Map_of_Kiirun(Keelung)_City_02.jpg|thumb]]
[[File:Kiirun_1_10000_1945.jpg|thumb]]

基隆港在昔日稱為「'''雞籠港'''」或「'''雞籠灣'''」。雞籠港在[[十七世紀|十七世紀]]即有外人足跡，[[西班牙|西班牙]]人[[台灣荷西殖民時期|佔領臺灣]]時就曾對雞籠港進行調查，並進行了部分建設。[[台灣清治時期|清治後期]]，西方列強的東來，逐漸開啟了雞籠港的發展。在[[同治|同治]]2年（1863年），基隆港以[[淡水區|淡水]]附港的名義正式開放為商港。而在1886年當時就任[[台灣巡撫|台灣巡撫]]的[[劉銘傳|劉銘傳]]進行基隆港的建港規劃，並委由當時的台灣首富[[林維源|林維源]]總辦基隆港的建港事宜，也將當時興建中的[[縱貫線|縱貫鐵路]]分出支線至港區；但後來建港工程因劉銘傳的離職而未全面實行，主要設施中只完成一座陸海聯運碼頭。後來統治台灣的[[中華民國政府|中華民國政府]]，將1886年認定為基隆港建立之年<ref>《基隆港建港百年紀念文集》，民國74年（1985年）11月9日出版，基隆港務局發行。</ref>。

[[台灣日治時期|日治時代]]，[[台灣總督府|日本當局]]計劃將基隆建設成為台灣與[[日本|日本]][[內地|本土]]的聯絡門戶，故基隆港正式開始進行現代化港口的建設，也就是從1899年到1944年間共五期的基隆港築港工程（第五期工程因[[第二次世界大戰|二戰]]爆發而未全部完工），清除了原本密佈內港區內的礁石，並在外港陸續整建了大型造船廠及[[軍港|軍港]]、[[漁港|漁港]]區等設施，從碼頭貨棧到港區[[鐵路|鐵路]]系統皆相當完備（現存的[[基隆港西岸碼頭倉庫|西二西三碼頭倉庫]]即在此時期興建）。這五期的築港工程，不但奠定了日後基隆港的發展基礎，也使基隆港在1970年代前穩坐台灣第一大港寶座。較特別的是，基隆港的築港工程並不由政府行政系統負責，而是由[[日軍|日本軍方]]主導，並被列為[[日本海軍|日本海軍]]列管軍港之一。

基隆港原由其所在的[[台北州|台北州]]設立「港務部」來管理，並由[[台灣總督府|台灣總督府]][[台灣總督府交通局海務部|交通局海務部]]監督，1941年[[太平洋戰爭|太平洋戰爭]]爆發後，改由直屬於台灣總督府的[[基隆港務局|基隆港務局]]負責。基隆港由於當時台灣主要物資吞吐港及海軍基地的地位，在大戰末期首當其衝，成為[[美軍|美軍]]轟炸的首要目標。港埠設施及港內停泊船隻皆毀損嚴重，港區幾成廢墟。

1945年，[[臺灣戰後時期|中華民國接管臺灣]]，續設基隆港務局，隸屬於[[臺灣省政府|台灣省政府交通處]]。港務局改組時以[[打撈|打撈]]港內一百多艘沉船及復建[[碼頭|碼頭]]、橋樑、船渠、船塢、運河、防波堤、修理場及倉庫等原有設施為主，1953年以後才開始增設港埠設施，如增建西29、30號碼頭、漁港區突堤碼頭、興建通棧倉庫等，並填築大沙灣海水浴場以興建船渠。

復建完成之後，基隆港開始復興。1961年至1974年間，港務局改建內港設施，並開始在外港及東岸興築碼頭<ref>{{Cite book|title=淘汰邊緣|last=胡|first=台麗|publisher=|year=|isbn=|location=|pages=}}</ref>。1974年至1981年間開始積極擴建外港，並先後興建兩座大型突堤碼頭，以消除內港瓶頸及島內成長的運輸需求；連接港區及[[中山高速公路|高速公路]]的東岸、西岸高架橋也興建於同時期。1982年至1992年，為因應貨櫃運輸時代的來臨，以改建及增建貨櫃碼頭為主。基隆港的營運量在1980年代到達高峰，到了1984年，基隆港更成為世界第七大貨櫃港。

發展至今，基隆港的碼頭總數從[[台灣日治時期|日治時期]]全座落在西岸的18座，擴增為現在的57座（西岸37座、東岸20座）。[[民國紀元|民國]]95年（2006年）年度全[[中華民國|中華民國]][[關稅|關稅]]總收入之中，經基隆港之收入為[[新台幣|新台幣]]821億餘元，佔總收入之60.54％<ref>關於[[中華民國|中華民國]]的[[海關|海關]]，詳見[[財政部關稅總局|財政部關稅總局]]條目。</ref>，足見基隆港在台灣經濟體系的重要地位。

基隆港的營運機構為[[臺灣港務股份有限公司基隆港務分公司|臺灣港務公司基隆港務分公司]]。其前身為基隆港務局，最初為[[臺灣省政府|臺灣省政府]]附屬之事業機構，1999年[[精省|精省]]後改隸[[中華民國交通部|中華民國交通部]]；2012年[[行政院組織再造|行政院組織再造]]實施後，中華民國交通部於同年3月1日起實施「政企分離」之航港管理作法，將各港務局公司化合併成立[[國營|國營]]之「臺灣港務公司」，原基隆港務局則改制為基隆港務分公司。

== 發展瓶頸與對策 ==
[[File:Ym_People_at_Keelung.jpg|250px]]貨櫃船]]
基隆港與基隆市唇齒相依，基隆市區的街廓即沿著港區發展，而早期基隆市民的生計更是與基隆港息息相關，昔日廣佈市區的「[[委託行|委託行]]」及[[酒吧|酒吧]]就是一例<ref>{{Cite web|url=http://www.books.com.tw/products/0010729869|title=靜寂工人：碼頭的日與夜|accessdate=2017-02-26|work=博客來}}</ref>。但進入1990年代之後，基隆港除了要面對國內外傳統港口的競爭，還要應付[[中國大陸|中國大陸]]東南各新興港口的快速崛起。而基隆港的港區規模已到達極限，卻因為港埠用地緊鄰市區及山區而無法擴建，港內大部分碼頭的吃水深度又無法停泊巨型貨櫃輪，導致許多船隻被迫彎靠[[高雄港|高雄港]]與[[台中港|台中港]]。{{Fact|基隆港的總運量於1990年代開始衰退，港口排名也在十年內，由世界前十大迅速跌到三十名以外。|time=2010-08-07T05:50:07+00:00}}

===興建新港===
基隆港於1980年代，正逢港埠營運的高峰期，當時港內船席不足的情形常常發生；基隆港務局於1980年代中期，為了解決基隆港的擁擠狀況，提出在基隆港以西的[[外木山|外木山]]海岸興建「基隆超級深水港」，即「基隆新港」的興建計畫，在基隆港營運狀況開始衰弱時，曾被認為是重振基隆港營運的最佳方法，但是之後一直無實際動作。在1997年，交通部以建設新港的難度過高、經費太過龐大等理由，否決了興建新港的提案，以在[[新北市|新北市]][[八里區|八里區]]興建淡水新港（即今[[台北港|台北港]]）作為替代方案。此舉引起了基隆市各界的不滿，認為此舉將會封殺了基隆港，甚至基隆市的未來發展。雖然如此，基隆市各界目前仍未放棄興建新港的可能。

===開放觀光===
近年來，為了提升競爭力，基隆港開始轉型為結合[[觀光|觀光]]、親水性之港口。港務局在2002年正式開放港區觀光，讓遊客可以搭船遊覽基隆的港埠風光。另外，為了吸引國際航運業者落腳，基隆港在2003年時設立了[[自由港|自由貿易港區]]<ref>[http://www.taiwanftz.nat.gov.tw/cwhe.asp 台灣自由貿易港區網站 - 自由貿易港區在哪裡] {{webarchive|url=https://web.archive.org/web/20080130092618/http://www.taiwanftz.nat.gov.tw/cwhe.asp |date=2008-01-30 }}</ref>。

===港市合一===
「[[港市合一|港市合一]]」的構想在1970年代時已經存在，<ref>{{cite news|title=專題報導「港市合一」面面觀|newspaper=聯合報|date=1972-03-04}}</ref>在1989年時已是選舉政見要點。<ref>{{cite news|title=基隆港市合一是政見的「主秀」|newspaper=經濟日報|date=1989-11-02}}</ref>為了促進基隆市港兩方的發展，基隆市各界自1990年代開始積極推動。2001年6月28日，港市合一的專責機構「基隆港管理委員會」正式成立，由[[基隆市市長|基隆市市長]]（時任市長為[[李進勇|李進勇]]）兼任主任委員。但交通部於2002年2月21日，函各港管理委員會暫行停止委員會運作<ref>[http://old.npf.org.tw/monthly/00207/theme-175.htm 《國家政策論壇》-「市港合一」回歸基本精神] {{webarchive|url=https://web.archive.org/web/20090417063641/http://old.npf.org.tw/monthly/00207/theme-175.htm |date=2009-04-17 }}</ref>。

== 未來展望 ==
由於現有港區短期內無法擴建，為了因應台北港將對基隆港的[[貨運|貨運]]業務造成的威脅、與鄰近國家港口的競爭，以及[[兩岸三通|兩岸三通]]後所帶來的[[客運|客運]]榮景，基隆港將從原本偏重[[貨物|貨物]]運輸，轉型成為客運、貨運並重的「加值型物流港」<ref>[http://www.klhb.gov.tw/Html/H01/H0105.aspx 基隆港務分公司-施政構想]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://www.cmri.org.tw/kmportal-deluxe/download/attdown/0/Mq2010-3-4.pdf 基隆港市共同發展策略]</ref>。未來基隆港將朝向「東客西貨」的方向來發展，東岸以客運與觀光遊憩為主，西岸則繼續作為貨運運輸的基地<ref>[http://nnk.ironman.tw/CH3/Ch3_4.html 北北基生活圈跨域空間發展整體策略規劃>現況分析>交通運輸]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；而配合基隆市中心的[[都市更新|都市更新]]計畫，未來基隆港部分碼頭的功能將進行調整，港區部分[[後線|後線]]區域也將開放商業開發，以達致基隆港、市發展之雙贏<ref name="基隆港未來規劃">[http://www.klhb.gov.tw/Html/H01/H0106.aspx 基隆港務分公司-基港局整體規劃及未來展望] {{webarchive|url=https://web.archive.org/web/20120419215352/http://www.klhb.gov.tw/Html/H01/H0106.aspx |date=2012-04-19 }}</ref>。
