'''基律纳'''（{{lang-sv|'''Kiruna'''}}、{{lang-se|Giron}}、{{lang-fi|Kiiruna}}），又译作'''基鲁那'''，是[[瑞典|瑞典]]最北部的一个城市，位于[[北博滕省|北博滕省]]，为[[瑞典萨米议会|瑞典萨米议会]]的所在地。

基律纳的名称来自于[[萨米语|萨米语]]''Giron''，意为雷鸟（ptarmigan），北面海裡的一种白色的鸟，鸟也被描绘到了城市的纹章上，和铁的标记在一起。铁代表了矿业在城市里占了很重要的地位。

== 地理 ==

基律纳建於1899年，並于1948年成为一个瑞典城市，曾经是世界上面积最大的城市。不过在1970年代瑞典自治区重新划分后，它变为世界上面积第二大的城市，面积为20,000平方公里，仅次于當時面積第一大城[[澳大利亚|澳大利亚]][[昆士兰|昆士兰]]洲的[[芒特艾萨|芒特艾萨]]市，后者有42,904平方公里。 

[[凯布讷山|凯布讷山]]是瑞典最高的山，海拔2,117米。基律纳自治区有6,000多个湖泊。[[托讷湖|托讷湖]]是其中最大的湖。7条河流穿过基律纳自治区，分别为[[卡利克斯河|卡利克斯河]]、[[托尔讷河|托尔讷河]]、[[莱尼奥河|莱尼奥河]]、[[劳塔斯河|劳塔斯河]]和[[维唐吉河|维唐吉河]]，而[[肯凯迈河|肯凯迈河]]和[[穆奥尼奥河|穆奥尼奥河]]是瑞典和[[芬兰|芬兰]]的分界线。无与伦比的地理风光还有著名的[[阿比斯库国家公园|阿比斯库国家公园]]，是和[[挪威|挪威]]的边界。

基律纳位于[[北极圈|北极圈]]以北145公里，从每年的5月30日到7月15日是[[极昼|极昼]]。[[极夜|极夜]]的日期要短一点，从每年的12月13日到次年的1月5日。

可以通过[[欧洲E10公路|欧洲E10公路]]，铁路和飞机到达基律纳。

由於主城區位於鐵礦上方，礦坑的巨大裂縫使基岩變形，目前基律納正將主城區遷往鄰近基岩穩固的地區。

== 工业 ==
[[铁矿|铁矿]]提炼是基律纳地区的支柱产业，城市很依赖[[LKAB|LKAB]]矿业公司。第二次世界大战期间，大量的铁矿从瑞典北部通过火车运到东海岸，再往下运，支持德国战争。

最近几年，基律纳努力改变城市对矿业的依赖程度转而发展科学，研发和政府相关职能。

== 风光 ==
Jukkasjärvi村有一个著名的[[冰旅馆|冰旅馆]]，一个完全由冰建造成的旅馆，每年都重新建造。
[[File:Kiruna_kyrka.jpg|thumb]]
1907年建造的基律纳的教堂也一样很有名气，是瑞典最大的木头建筑之一。教堂的表面是[[新哥特式|新哥特式]]风格，而祭坛是[[新艺术运动|新艺术运动]]风格。它曾被评为瑞典最佳外观的教堂和瑞典1950年前最重要的建筑。
