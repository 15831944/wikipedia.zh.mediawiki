'''基本書坊'''是位於[[台灣|台灣]]的[[華文|華文]][[男同志|男同志]]叢書[[出版社|出版社]]，2008年4月由作家[[邵祺邁|邵祺邁]]獨資創立。

== 命名 ==
取香港語「基」（Gay）、與書本的「本」（Books）兩字而成。英文名「Gbooks」，簡稱「GB」。

== 成立宗旨 ==
出版「gay味夠正、夠濃」的書籍，廣納各類書寫和議題，滿足華文男同志閱讀需求。尤其是在主流出版社無法出版、卻對同志有重要意義的出版品，需要有一固定「基地」使它得以面世。以書本為媒介和載體，散布、推廣性別平權意識，並使平權運動理念落實為實際行動。發掘並培育華人LGBT社群各項創作與媒體人才。保留同志歷史與文化，記錄當代同志生活樣態。
「取於社群、用於社群」，積極參與同志社群各種活動、贊助並支援各同志團體。<ref>出自2011年8月邵祺邁在成都愛白同志中心發表的演講「同志出版：軟硬兼施的運動」內容。參見：[http://www.facebook.com/note.php?note_id=186016061467982 大雪無痕的參與後記－好一座G片工場!]</ref>

== 趣聞軼事 ==
* 出版日期：經常選擇對作者具特殊意義、數字有諧音、或與該書主題關聯性強的日期。例如作者（或暗戀對象）的生日、54運動紀念日、520（我愛你）等。
* 「男X男」的正確念法：有讀做英文「X」、也有讀做中文「乘」音，目前咸認的最佳讀音為中文的「插」。
* 版權頁上的工作人員頭銜：點心主廚、舞術指導兩項為華文出版界創舉。
*《12P情慾相談室》為華人出版品中首度大膽以精液為題的封面。
