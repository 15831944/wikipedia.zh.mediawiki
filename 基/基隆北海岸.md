'''基隆北海岸'''，或稱'''大基隆地區'''，為位在[[台灣|台灣]]東北部的一個[[都會區|都會區]]，相當於[[日治時期|日治時期]]的[[台北州|台北州]][[基隆郡|基隆郡]]，又因以[[基隆市|基隆市]]為區域中心而得名。

「基隆北海岸」此一名稱，為[[基隆|基隆]]與台灣東北海岸地區之合稱。基隆北海岸地區的自然[[地標|地標]]，一個是座落在[[新北市|新北市]][[瑞芳區|瑞芳區]]的[[基隆山|基隆山]]，另一個則是位在[[基隆港|基隆港]]外海的[[基隆嶼|基隆嶼]]。其中，基隆嶼從台灣最北的[[石門區|石門區]]到[[鼻頭角|鼻頭角]]的沿海皆可看到。

== 範圍定義 ==
[[File:Taipei_Duhuiqu4.png|250px]]在基隆市、新北市境內管轄區，加總即為基隆北海岸之範圍]]
基隆在1875年開始設官統治之後，一直到[[台灣日治時期|日治時期]]設置[[基隆郡|基隆郡]]，在台灣東北部的[[富貴角|富貴角]]到[[三貂角|三貂角]]間的濱海聚落間，形成一個以基隆為中心的[[生活圈|生活圈]]，這個範圍就是一般所稱的大基隆地區。而在[[臺灣清治時期|清代]]與日治時期所設的[[基隆廳|基隆廳]]，轄區範圍甚至還包含[[汐止區|汐止]]。地理上的基隆北海岸，指石門區尖子鹿以東、貢寮區三貂角以北的海岸區域。

現在大基隆地區，包含基隆市和新北市東北角的六個區（[[金山|金山]]、[[萬里區|萬里]]、[[瑞芳|瑞芳]]、[[貢寮|貢寮]]、[[雙溪|雙溪]]、[[平溪|平溪]]），人口約有50萬左右，在[[中華民國內政部|內政部]]的全台14個地方生活圈規劃為「基隆生活圈」而自成一區，例如這些區皆為[[臺灣基隆地方法院|臺灣基隆地方法院]]之轄區。過去在[[中央氣象局|中央氣象局]]的氣象資料中，亦是採用基隆北海岸之稱呼。

== 都市體系 ==
=== 通勤圈及購物圈 ===
《臺灣地區都市體系之研究》一書中提到，以基隆為中心的通勤圈與購物圈資料：
*通勤圈：以基隆為通勤中心的地區有瑞芳一處
*購物圈：以基隆為購物中心的地區包含新北市的石門、金山、萬里、瑞芳、平溪、雙溪、貢寮等七區

=== 都市位階 ===
《臺灣地區都市體系之研究》中，大基隆地區的都市位階為：
*次區域中心：基隆
*一般市鎮：瑞芳<ref>行政院經濟設計委員會都市規劃處，民國六十四年五月《臺灣地區都市體系之研究》</ref>

而在台灣北部區域計畫（第一次通盤檢討）中基隆生活圈範圍為基隆市、金山、萬里、瑞芳、雙溪、平溪、貢寮
<br>都市階層為：
*地方中心：基隆
*一般市鎮：瑞芳
*農村集居中心：金山、萬里、平溪、雙溪、貢寮<ref>台灣北部區域計畫（第一次通盤檢討）</ref>

== 開發歷程 ==
'''1. 清治時期'''

1875年，[[清朝|清]]政府因應基隆開港之後，地位的日漸重要，也需要設置機關來負責通商等對外事務，遂設置[[台北城|台北]]分府通判，為基隆正式設官統治之始。1888年改設[[基隆廳|基隆廳]]，下轄四個[[堡|堡]]（類似今日的「[[鄉_(台灣)|鄉]]」），又有'''「金鷄貂石」'''之稱。
*(金) → 金包里（今[[金山區_(台灣)|金山]]、[[萬里區|萬里]]）
*(鷄) → 鷄籠（今[[基隆市|基隆市]]5區【[[信義區_(基隆市)|信義區]]、[[仁愛區|仁愛區]]、[[中正區_(基隆市)|中正區]]、[[中山區_(基隆市)|中山區]]、[[安樂區|安樂區]]安樂區部分區域】、[[瑞芳區|瑞芳]]北部）
*(貂) → 三貂嶺（今[[雙溪區|雙溪]]、[[貢寮區|貢寮]]）
*(石) → 石碇（今 [[汐止區|汐止]]、[[七堵區|七堵]]、[[安樂區|安樂區]]的[[鶯歌石_(基隆市)|鶯歌石]]以及[[港口_(基隆市)|港口]]、[[暖暖區|暖暖]]、[[平溪區|平溪]]、[[瑞芳區|瑞芳]]南部）

'''2. 日治時期'''

[[台灣日治時期|日治時代]]，[[台灣總督府|總督府]]以清代基隆廳的範圍為基礎，經過多次改制，1920年日治時全台最後一次行政區重劃時設置[[台北州|台北州]][[基隆郡|基隆郡]]，郡役所（郡公所）位於今日基隆市區的的信一路立體停車場。而港區及市區部分另設置[[基隆街_(基隆郡)|基隆街]]，在1924年與[[高雄市_(州轄市)|高雄]]同時升格為[[州轄市|市]]。

1950年全台縣市大重劃時，台北縣基隆區被廢除，除1947年已改隸基隆市外，其餘轄6個鄉鎮全部改為台北縣直轄。此後，行政區劃就未曾大幅更動過，而大基隆地區因此被分割由[[基隆市|基隆市]]及[[新北市|臺北縣]]各自管轄；2010年12月25日臺北縣升格為[[新北市|新北市]]後，原臺北縣所轄6個鄉鎮全部改為區，並由新北市直轄。

== 區域整合 ==
[[File:Yehliu.jpg|250px]]]]
大基隆地區居民的日常補給多仰賴基隆市，但基隆市周圍鄉鎮於戰後併入[[新北市|臺北縣]]，僅七堵鄉併入基隆市，導致週邊鄉鎮反由路程遙遠、位於[[板橋區_(台灣)|板橋市]]的台北縣政府管轄（詳見[[:File:Taipei_County.PNG|臺北縣行政區劃]]地圖）。故地方人士曾倡議設置「[[省三縣|基隆縣]]」，但因轄區太小及其他政治因素而屢遭[[中華民國政府|中央政府]]否決。

近年來又開始有將基隆市的管轄範圍擴大、或是直接併入[[台北市|台北市]]的聲音出現，在各級選舉中，更不時可在候選人政見看到成立「大基隆市」的構想。自2001年起[[北台灣|北台灣]]大部分縣市由[[泛藍|泛藍]]人士執政後，又出現基隆市、台北市、台北縣合併的呼聲，北台灣各縣市的首長也共同發表合作備忘錄，為未來縣市整合做準備。

2010年12月25日正式將台北縣升格為[[新北市|新北市]]，形成中心都市（基隆市）是省轄市，衛星鄉鎮屬[[直轄市|直轄市]]（新北市）的現象；仍屬[[台灣省|台灣省]]的基隆市，也因新北市阻隔形成[[飛地|飛地]]。
