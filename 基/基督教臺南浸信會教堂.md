'''基督教臺南浸信會教堂'''，為[[臺南市|台南市]][[中西區_(台灣)|中西區]]的一座教堂，建於民國42年（1953年），為戰後初期所建西方教堂建築。2016年進行文化資產審查時，台南浸信會原不願被列為文化資產，尚待日後拆除重建，經協調後登錄為歷史建築<ref>{{Cite web|url=http://www.cdns.com.tw/news.php?n_id=1&nc_id=135861|title=台南浸信會拒列歷史建築|accessdate=2019-04-13|author=|date=2016-12-21|publisher=中華日報}}</ref>。

== 沿革 ==
民國41年（1952年），美南[[浸信會|浸信會]]宣教士明俊德（Miss Bertha Smith）與張佩信創立台南浸信會，最初承租[[原臺南州會|台南市議會]]的禮堂作為聚會場所。本教堂於民國42年（1953年）3月動土興建，同年5月17日舉行獻堂感恩禮拜<ref>{{Cite web|url=http://www.lilyfruit.com.tw/left_4/content_20.php|title=台南浸信會|accessdate=2019-04-13|author=李文雄|date=|publisher=莉莉水果店}}</ref>。

== 建築特色 ==
教堂門廊由四根方柱及四根[[壁柱|附壁柱]]構成，[[柱头_(建筑)|柱頭]]有芼莨葉裝飾，雙斜坡屋頂，兩側各有八根扶壁，外牆以洗石子施作，為古典風格教堂建築，樸實典雅。內部木屋架保存良好，尚留有貓道，加上浸池、禮拜長椅等禮拜儀式設施，營造出獨有的歷史氛圍。<ref>{{Cite web|url=https://nchdb.boch.gov.tw/assets/overview/historicalBuilding/20180703000003|title=基督教臺南浸信會教堂|accessdate=2019-04-13|author=|date=|publisher=文化部文化資產局-國家文化資產網}}</ref>

== 其他 ==
基督教臺南浸信會教堂位於[[孔廟文化園區|孔廟文化園區]]內，周遭有多處歷史性建築。並且與[[臺南市立建興國民中學|建興國中]]圍牆間，有[[福安坑溪|福安坑溪]]流經，其河道仍可見。
