[[遗传学|遗传学]]中的推断是指对为观察到的[[基因型|基因型]]进行[[统计推断|统计推断]]。<ref>{{Template:Cite journal|last1 = Scheet|first1 = Paul|last2 = Stephens|first2 = Matthew|year = 2006|title = A Fast and Flexible Statistical Model for Large-Scale Population Genotype Data: Applications to Inferring Missing Genotypes and Haplotypic Phase|journal = The American Journal of Human Genetics|volume = 78|issue = 4|pages = 629–644|doi = 10.1086/502802|pmid = 16532393|pmc = 1424677}}</ref>这通过人群中的[[:en:Haplotype|单倍型]]来完成，比如[[国际人类基因组单体型图计划|国际人类基因组单体型图计划]]和[[千人基因组计划|千人基因组计划]]，这从而允许通过未检测变异与已检测变异的连锁关系进行推断。<ref>{{Template:Cite journal|last1 = Scheet|first1 = Paul|last2 = Stephens|first2 = Matthew|year = 2006|title = A Fast and Flexible Statistical Model for Large-Scale Population Genotype Data: Applications to Inferring Missing Genotypes and Haplotypic Phase|journal = The American Journal of Human Genetics|volume = 78|issue = 4|pages = 629–644|doi = 10.1086/502802|pmid = 16532393|pmc = 1424677}}</ref>基因型推断减少[[全基因组关联分析|全基因组关联分析]]相关位点数。
