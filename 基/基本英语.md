{{英語方言}}

'''基本英语'''（{{lang-en|'''Basic English'''}}）是一種[[人工語言|人工語言]]，它基於[[英語|英語]]的一種簡化版本而產生，由{{link-en|查尔斯·凯·奥格登|Charles Kay Ogden}}創造。在他於1930年所出版的《基本英语——规则和语法的一般约定》一書中有詳細的介紹。

奥格登说：「学习[[英语|英语]]要7年，学习[[世界语|世界语]]要7个月，而学习基本英语只要7星期。」因此，[[跨國公司|商業公司]]會將艱澀難懂但需要全球使用的文書寫成基本英語；而[[语言学校|语言学校]]也會為了讓學生短期內學會簡單的英語，而教授基本英语。

奥格登不使用重复意义的单词，而且需要所有国家都适用这些单词。他利用大规模的测试和调整来得到这些词汇。在[[语法|语法]]方面，他也做了简化，但是保留了英语通常的用法。

在[[第二次世界大战|第二次世界大战]]後，这种简化开始蕴含「推进[[世界和平|世界和平]]」的理念。但是由于缺乏规划，发展出了不同地方使用的简化版本。今天还存在的基本英语有850个单词作为初学英语者的单词词汇广传世界，特别是在[[亚洲|亚洲]]最流行。<ref>Edmond H Weiss ''The Elements of International English Style'', pp. 17-18, M. E. Sharpe, 2005 ISBN 978-0-7656-1572-5{{en}}</ref>

== 语法规则 ==
奥格登基本英语的语法规则是为了让人们只用850个单词就可以解决一般交流：
# 在单词词尾加''-s''作为复数，那些加''-es''和''-ies''的保留；
# 四个词缀：''-er''可以变成300多个名词，''-ing''和''-ed''变成形容词；
# 使用''-ly''变成副词；
# 对于数量，使用''more''和''most''，保留''-er''和''-est''；
# [[形容词|形容词]]的[[反义词|反义词]]是加[[前缀|前缀]]''un-'', ''im-''
# [[问句|问句]]使用相反的[[語序|词序]]，加上''do''；
# [[人称|人称]]和[[代詞|代名词]]使用和一般英语相同；
# 可以使用两个名词组合成[[新词|新词]]，比如''milkman''和''wordend''；还可以是名词加方位，例如''sundown''；
# [[计量单位|单位]]、[[數字|數字]]、[[貨幣|貨幣]]、[[月|月份]]、[[日|日子]]、[[年|年份]]、[[小时|钟点]]和[[国际语|国际语]]同一般英语形式；
# 使用专业和科技词汇。一些特殊词汇不属于基本英语，词性、复数等变化不遵守这个规范。

== 历史 ==
1944年，[[温斯顿·丘吉尔|温斯顿·丘吉尔]]在[[哈佛大学|哈佛大学]]的一个演讲裡，起初支持用基本英语作为世界语言，但是由于一个故事他放弃了，就是那句“blood, toil, tears and sweat”（热血、辛劳、眼泪和汗水）写成基本英语就是“blood, hard work, eyewash and body water”（血液、努力、淚液和体液）<ref>来自于[[时代教育副刊|时代教育副刊]][http://www.talkingto.co.uk/ttgo/html/ttgo_answ.asp?quesID=1923&CatID=187 名言集]</ref>。[[喬治·奧威爾|喬治·奧威爾]]的作品《[[一九八四|一九八四]]》中可能有对基本英语的模仿。书后对“[[新語_(一九八四)|新话]]”的注解可以认为暗含着对[[人工語言|构造语言]]的批评，尽管那在这本书的著作期间（1948年）呼声很高。

奥威尔的新话实际上更像是专为机器或网络设计的语言，不知晓网络的人士看不懂的。与之相比，奥格登的基本英语任何一个懂英语的人都能读懂，但它只能用作[[第二語言|第二語言]]。
