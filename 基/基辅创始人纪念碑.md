'''基辅创始人纪念碑'''（{{lang-uk|Пам'ятник засновникам Києва}}），又称'''基辅城创始人纪念碑'''<ref>{{cite journal |author1=景桂兰 |title=俄罗斯环境艺术 |journal=装饰 |date=2003 |issue=5 |pages=22-23}}</ref>、'''基辅城市奠基者纪念碑'''<ref name="环球网">{{cite web |title=乌克兰首都基辅：历史与浪漫的结合 |url=http://go.huanqiu.com/europe/2013-09/4362396_2.html |website=环球网 |accessdate=2019-04-17 |date=2013-09-17}}</ref><ref>{{cite web |title=乌克兰1991年版1 Karbovanets纸钞 |url=http://www.coin001.com/zhibiku.php?action=zhibishow&cate_id=175&id=6297 |website=纸币库 |accessdate=2019-04-17}}</ref>、'''基辅城市落成纪念碑'''<ref>{{cite web |author1=陶短房 |title=乌克兰雕塑见证时局变迁 示威者敲掉五角星标志 |url=http://money.163.com/14/0226/07/9M091PAE00253B0H.html |website=网易-环球时报 |accessdate=2019-04-22 |date=2014-02-26}}</ref>，坐落在[[乌克兰|乌克兰]]首都[[基辅|基辅]]的一尊纪念雕塑，是为纪念基辅建城1500年而在1982年建造的，其作者是雕塑家{{tsl|uk|Бородай Василь Захарович|瓦西里·扎哈罗维奇·博罗代}}，建筑师是尼古拉·米哈伊洛维奇·费先科（{{lang|ru|Николай Михайлович Фещенко}}）。

== 描述 ==
雕塑的主题内容是基辅创建者[[基伊|基伊]]、{{tsl|ru|Щек|塞克 (基辅历史)|塞克}}和{{tsl|ru|Хорив|霍里夫}}三兄弟和他们的妹妹{{tsl|ru|Лыбедь (легендарная личность)|雷别季}}。雕塑中，基伊手持[[弓箭|弓箭]]，塞克和霍里夫手持[[矛|长矛]]，而雷别季则站在船头向远处眺望。雕塑设计者博罗代是比照自己女儿的样貌创作雷别季的，他女儿是一名[[画家|画家]]及平面艺术家，年仅31岁时便因病去世<ref>{{Cite news|url=http://svoboda-news.com/svwp/%D1%87%D0%B8%D1%97-%D0%BE%D0%B1%D0%BB%D0%B8%D1%87%D1%87%D1%8F-%D0%B7%D0%BE%D0%B1%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D1%96-%D0%BD%D0%B0-%D1%81%D1%82%D0%BE%D0%BB%D0%B8%D1%87%D0%BD%D0%B8%D1%85-%D0%BF/|title=Чиї обличчя зображені на столичних пам’ятниках?|language=uk-UA|work=Свобода|accessdate=2018-04-06}}</ref>。

雕塑的基座是[[花岗岩|花岗岩]]，船长9米，三兄弟身高4.3米，雷别季身高3.8米。该纪念碑位于[[第聂伯河|第聂伯河]]畔{{tsl|uk|Наводницький парк|纳沃德尼茨基公园}}内，靠近[[第聂伯站|第聂伯地铁站]]和[[帕頓橋|帕顿桥]]。

2010年2月23日，由于腐蚀破损，塞克和霍里夫兄弟的高度变低<ref>{{cite web|url=http://gazeta.ua/index.php?id=328557|title=Gazeta.ua: На Дніпровській набережній впав пам'ятник засновникам Києва (ФОТО)|date=2010-02-24|archiveurl=http://www.webcitation.org/6HuseLMYG|archivedate=2013-07-06}}</ref><ref>{{cite web|url=http://kiev.pravda.com.ua/news/4b84ec129a3d6/|title=Українська Правда. Київ: Від пам’ятника засновникам Києва відвалилися Щек і Хорив.|date=2010-02-24|archiveurl=http://www.webcitation.org/6Husj1odE|archivedate=2013-07-06}}</ref>。2010年5月28日，纪念碑在经过3个月零5天的修复后重新开放，结果基辅市长{{tsl|uk|Черновецький Леонід Михайлович|列昂尼德·切诺韦茨基}}在揭幕仪式上忘记了城市创始人的名字<ref>{{cite web |title=Черновецкий открыл отремонтированный памятник «Хориву и их сестре Ще… Кию» |url=http://obkom.net.ua/news/2010-05-28/1752.shtml |website=Gazeta.ua |accessdate=2019-04-22}}</ref>。
