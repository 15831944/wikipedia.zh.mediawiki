'''基進黨'''是[[中華民國政黨|中華民國政黨]]之一，前身為'''基進側翼'''——[[臺灣|臺灣]]首個自稱[[臺獨左派|臺獨左派]]的政治團體<ref>{{Cite web |url = https://www.wealth.com.tw/home/articles/18797 |title = 陳奕齊讓基進黨暴紅的第一仗 |author = 郭瓊俐 |date = 2018年11月28日 |publisher = 財信雜誌社 |quote = 講到定位為台獨左派、剛性政黨的基進黨 }}</ref>。2012年成立，2014年中已發展有[[臺北市|臺北]]、[[臺南|臺南]]、[[新竹|新竹]]、[[臺中|臺中]]等地方組織，2015年7月正式申請成為全國性政治團體，2016年5月改為現名，並於內政部申請登記為政黨。總部位於[[高雄市|高雄]]，現任黨主席為[[陳奕齊|陳奕齊]]。

==歷史==
===緣起===
基進側翼的主張主要孕育於《超克藍綠》共筆部落格的新一專欄，2008年11月9日《吹響激進的號角－－成為統治集團眼中的「暴民」吧！！》<ref>[http://clique2008.blogspot.tw/2008/11/by-shinichi_9.html Shinichi，《吹響激進的號角－－成為統治集團眼中的「暴民」吧！！》，超克藍綠部落格]</ref>可視為第一份文獻：「…如今之際，唯有徹底放棄、告別[[民進黨|民進黨]]作為[[同路人|同路人]]，在台灣政治光譜上往『台灣人民』方向挺進並拉開台灣政治光譜的分佈，撐出一個激進有力的側翼…台灣人民的未來跟前途，才可以獲得確保！！」該文的寫作背景為[[海協會|海協會]]會長[[陳雲林|陳雲林]]來台時，發生暴警在街頭逞兇粗暴對付抗爭者。到了2010年，同作者的《一個真誠／真正具有「普賴度」(Pride)中間選民的宣言》<ref>[http://clique2008.blogspot.tw/2010/11/prideby-shinichi.html  Shinichi，《一個真誠／真正具有「普賴度」(Pride)中間選民的宣言》，超克藍綠部落格]</ref>一文，已可見基進側翼「[[三大神主牌|三大神主牌]]」，即政治民主化（反殖）、主權自主化（反帝）、社會自由化（反剝削）的提出。基進之意，在於強調立場上的堅定與根本。與取得政權為核心的政黨相較，在政治光譜上，必然是處於非主流的極端位置。

=== 起步與擴散 ===
2011年春正式進入政治宣講階段，[[台灣社|台灣南社]][[秘書長|秘書長]][[陳奕齊|陳奕齊]]（新一）陸續受邀於南社、台北Café Philo等介紹其政治主張。2012年4月27日，新一在《南之洛馱思》論壇講演「尋找後2012政治改革的阿基米德點」，正式為基進側翼的獨立組建吹起啟動的號角。

2013年至2014年初，除了在[[台南社區大學|台南社區大學]]開設「愛上政治」課程擴大基進側翼的骨幹之外，新一也開始進行全台宣講：[[雲科大|雲科大]]、[[台獨聯盟|台獨聯盟]]、台南Masa Loft Café、《如何阻撓國民黨繼續執政》座談會等等。其間基進側翼獲得台獨聯盟相當程度肯認，該聯盟為基進側翼在北部第一個合作伙伴。

直至2015年，基進側翼於台北、新竹、台中、雲林、台南等地各設有分部，總部位於高雄，事務所設於[[高雄市|高雄市]][[鹽埕區|鹽埕區]]大禮街48號。

===太陽花學運===
基進側翼經常被誤為是2014年[[太陽花運動|太陽花運動]]所催生的政團，這是因為此社會運動為基進側翼帶來不少曝光機會，該政團參與了台北、台南與高雄三地的群眾組建，新一於[[立法院|立法院]]外的政治宣講也頗獲佳評；3月30日的[[330反服貿遊行|330反服貿遊行]]中，基進側翼與超克藍綠共同印製的文宣《J'accuse...! 我控訴!》遭到現場糾察隊查扣，曾被媒體報導。另外，該政團在此運動之後進行核心成員的調整也受到新聞關注，只不過調整原因已與此運動無關了。

===太陽花學運之後===
2014年6月27日，中國國台辦主任張志軍至高雄參訪，基進側翼於漢來巨蛋會館外抗議，因警方強勢驅離而造成現場成員受傷。

2014年6月，基進側翼宣布參與年底的市議員選舉。

2014年8月17日，基進側翼舉辦「城市管線行腳—愛我高雄，城市零風險」遊行活動，並向高雄氣爆事故的罹難同胞默哀。

2015年1月18日成立「基進側翼全國聯合事務所」，並宣佈已向內政部申請成立政團<ref>{{cite news|author=涂建豐|title=第三勢力成型　基進側翼全國聯合事務所成立 |url=http://www.appledaily.com.tw/realtimenews/article/new/20150118/544143/ |accessdate=2015-01-19|newspaper=蘋果日報|date=2015-01-18}}</ref>。

===2016討黨產公投連署運動===
2015年3月30日，基進側翼和台獨聯盟、黨產歸零聯盟、[[黃國昌|黃國昌]]、[[林昶佐|林昶佐]]（Freddy）共推[[KMT|KMT]]討黨產公投<ref>{{cite news|author=林朝億|title=基進側翼推黨產公投 要大選投第3張票|url=http://newtalk.tw/news/view/2015-03-31/58422 |accessdate=2015-03-31|newspaper=新頭殼|date=2015-03-31}}</ref><ref>{{cite news|author=羅沛德|title=民團促黨產公投 國民黨：尊重|url=http://news.ltn.com.tw/news/politics/breakingnews/1273286|accessdate=2015-03-30|newspaper=自由時報|date=2015-03-30}}</ref>。

自從太陽花運動之後，[[第三勢力|第三勢力]]咸信「自己國家自己救」，儘管台灣青年公民，深刻地體認到在選舉中把台灣主要的民主障礙—中國國民黨—的政治勢力削弱，對台灣民主進程是相當重要的任務，但只有讓國民黨在地方與中央的公職席次削弱或不過半，遠遠不足以捍衛或促進台灣更健全的民主。

因此，「基進側翼」身為推動台灣政改的第三勢力中的一個政團，在2016此一場攸關台灣未來發展的大選前一年，發動「黨產公投連署運動」的用意，即是認為第三勢力應該為台灣的政治改革，立下制度性的重建基樁的可能，不該只是單單期待民進黨的上台執政。畢竟，民進黨若能在2016年拿回執政權，依舊得面對中國與美國的兩面壓力，以及內部中國國民黨的可能杯葛與阻礙之時，唯有立基於制度性重建的可能，方能擺脫過去阿扁任內的改革步伐受到各方勢力牽扯羈絆，導致各種重大民主轉型過程中的改革減緩或停擺。

而「黨產公投」此一2016年的第三張選票，其實正是為2016的新局作出制度性的因應。設若，2016黨產公投成案，此乃歷史上公投最可能過關的一次；若公投過關，則中國國民黨勢必得解散且不得重組，此時，台灣民主新局將據此展開。為了避免黨國勢力復辟，基進側翼同時主張曾為國民黨的退黨黨員和現任黨員，公投若通過，即便國民黨裂解或重生，那些皆永久不得參與任何選舉，即便部分成員已經退黨或已經認同進步理念(例如:楊實秋等人)。同時，黨政商利益夾纏、政策「尋租」(rent-seeking)所造就的貪腐、買票慣習與文化等等，將因為黨產公投的過關，一一進入清理狀態，讓台灣民主制度得以重構，並帶出[[向大眾透明化|向大眾透明化]](transparency)、[[社會徵信問責|社會徵信問責]](accountability)、[[回應公民需求|回應公民需求]](responsiveness)的新民主制度。此乃台灣二次民主化工程的再啟動。此外，部分成員主張，希望能以相同方式，要求新黨、親民黨、中華統一促進會等偏向於統一，或從國民黨分出政黨消失在臺灣政局，因為對部分成員而言，這些都是國民黨的延伸，不解散則餘孽依舊存在。

基於此，「基進側翼」即將在全台組建「爭取公民的第三張選票—黨產公投宣講團」，開拔至社區、公園、街頭，直接向公民訴求第三張選票的重要性。同時，基進側翼的宣講團，也將替各路支持利用黨產公投啟動台灣二次民主化的第三勢力的候選人進行助講，利用黨產公投議題的助講，串連起全台第三勢力的合作契機。<ref>{{cite news|author=基進側翼臉書專頁|title=「啟動民主進階版，公民作伙討黨產：爭取公民的第三張選票」基進全台助講團起跑新聞稿|url=https://www.facebook.com/radicalwing/photos/a.353905111425345.1073741831.348571125292077/507776979371490/?type=1}}</ref>

==理念==

===政治民主化===
台灣／中華民國還不是正常的民主國家，為何？因為正常的民主國家，不只是能選舉就好，還需要國民具備成熟的民主精神，並落實在其他制度上。以戰後德國為例，為反省威瑪共和導致納粹合法崛起的慘劇，新的聯邦憲法設置一道防衛機制：禁止法西斯政黨參政，甚至連主張法西斯都屬於犯罪行為。中國國民黨，就是台灣的法西斯政黨。威權時期的種種不義，都仍體現在今日的生活裡。洪仲丘案即為明證，因為國軍體系的腐敗，是國民黨體制的最高展現。因此我們主張，國民黨即便道歉、裂解、清算而重生，亦永久不得參與所有的政治選舉。否則台灣將只是從過去的威權時期，進化到學理上的「競爭威權」時期，也就是表面可以選舉，但國民黨永遠佔據最大贏面。<ref name="party doctine">{{Cite news|url = http://radicalwings.tw/about.php|title = 基進宗旨|last = 基进党|first = |date = |work = |accessdate = }}</ref>

===主權自主化===
基進側翼反對[[中華人民共和國|中華人民共和國]]與[[中華民國|中華民國]][[中國國民黨|中國國民黨]]、[[新黨|新黨]]的[[中华民族主义|中华民族主义]]，也反對[[民進黨|民進黨]]將台灣停滯於[[中華民國|中華民國]]現狀的立場（詳見[[台灣前途決議文|台灣前途決議文]]）。即同時反對中華人民共和國[[帝國主義|帝國主義]]的作為，也反對依循著中華民國政體所內建的文化觀與歷史觀，而在心態上自居為中國的[[藩屬|藩屬]]或[[派生|派生]]，而有礙獨立國格的建立。該政團認為，為了抵抗來自中華人民共和國的併吞壓力，必須主動發聲進行國際「承認政治」(recognition politics)；同時，也不能將[[統獨問題|統獨問題]]視為一個對等的選項而進行[[公投|公投]]，因為：不論公投結果如何，中國永遠在那裏，想當中國人的一方均可經個人選擇而實現其中國人的想望，但台灣人作為[[國籍|國籍]]身份卻有可能被[[多數暴力|多數暴力]]所剝奪。<ref name="party doctine"></ref>

===社會自由化===
社會自由化是跟[[經濟自由化|經濟自由化]]相對立的概念。兩大黨（即國民黨與民進黨）的經濟自由化、支持全球化、國際化立場，已經明顯造成經濟發展的利益為少數人所獨享，[[貧富差距|貧富差距]]逐年擴大，[[工時|工時]]高居世界第三名。其他黨（[[台聯|台聯]]、[[親民黨|親民黨]]、[[新黨|新黨]]...）雖然立場沒那麼支持自由化，甚至反對，但也有時候會向自由化妥協。該政團主張，依循著[[公平正義|公平正義]]原則，透過合理地分配台灣的經濟果實，提高社會整合（social inclusion/integration）；因為，一個不論貧富出身都有公平發展其才能的社會，才是國家經濟得以永續發展的關鍵所在。該政團也認為，透過這個過程，台灣人民將逐漸從勞動中釋放出來，成為擁有更多時間追逐自己生命夢想的自由人，即經由反抗不平等而達成社會自由化。<ref name="party doctine"></ref>然而於2018年初，面對民進黨政府修法放寬增加工時等作為，基進黨幹部表態贊成。

===區域平衡發展===
基進黨對於[[中國國民黨|中國國民黨]]於2008年重新執政後，推翻原有政策，將已南遷之[[體委會|體委會]]、[[漁業署|漁業署]]全部遷回[[台北市|台北市]]，而其餘原定但尚未執行之中央部會南遷計劃（例如：[[公廣集團|公廣集團]]）也全部終止，十分不滿。基進黨批評中國國民黨長期執政時[[重北輕南|將全台灣的政治、經濟、教育等資源過度集中於北台灣，尤其是台北市]]，而2008年重新執政後，其核心集中思維又再度恢復。<ref name="基進黨（基進側翼） 2017">{{cite web | title=20171008基進黨聲明稿【基進黨支持首都南遷和減壓】 | website=基進黨（基進側翼） | date=2017-10-08 | url=http://radicalwings.tw/news_info.php?SerialNo=163&Name=%E5%9F%BA%E9%80%B2%E9%BB%A8 | language=zh | access-date=2018-03-27 | deadurl=no | archiveurl=https://web.archive.org/web/20180327204604/http://radicalwings.tw/news_info.php?SerialNo=163&Name=%E5%9F%BA%E9%80%B2%E9%BB%A8 | archivedate=2018-03-27 }}</ref>

==爭議==

===黨內性騷擾===

基進黨發生過兩次[[性騷擾|性騷擾]]事件，分別發生於2014、2017年。基進黨Facebook粉絲專頁分別於2016、2018年對此發文聲明。

第一次性騷擾事件來源Facebook貼文已不可閱讀，僅剩基進黨聲明。<ref>{{Cite web |url = https://www.facebook.com/radicalwings/posts/714775165338336 |title = 基進黨針對李姓志工性騷擾事件聲明 |author = 基進黨 |date = 2016年9月16日 }}</ref>

第二次性騷擾事件的爆發開端是受害者吳雅惠於2018年8月30日性騷擾刑事判決<ref>{{Cite web |url = http://jirs.judicial.gov.tw/FJUD/index_1_S.aspx?p=6dIkB7SFzsqzBIbQfJkiRZyqBThQSXbL6h0YuHIxZxU%3d |title = 107年度易字第215號刑事判決 |author = 陳力揚 |date = 2018年8月30日 |publisher = 臺灣高雄地方法院 }}</ref>後，聽聞高雄基進黨傳來「吳雅惠是個行為不檢點的人」，情緒崩潰之下決定於11月16日上PTT發文《小黨志工的咪兔故事》公開案情經過。<ref>{{Cite web |url = https://www.facebook.com/yahuihuihui/posts/1836091993183592 |title = 這是我去年還在基進時，受到性騷擾的經過 |author = 吳雅惠 |date = 2018年11月21日 }}</ref>

2018年11月22日的基進黨聲明中說道「惟此事是屬個人私領域之事，基進側翼政團並無法介入個人情緒與感情」<ref>{{Cite web |url = https://www.facebook.com/radicalwings/photos/a.353905111425345/1214670982015416/ |title = 基進側翼政團聲明 |author = 基進黨 |date = 2018年11月22日 |quote = 惟此事是屬個人私領域之事，基進側翼政團並無法介入個人情緒與感情事由，僅能就相關團體規範之內容善進所有可能之處置。 }}</ref>，造成眾多網友不滿，認為基進黨試圖將性騷擾定位成私領域之事以撇清責任。吳雅惠也不滿基進黨沒有將性騷擾案例公開給黨員知道，以避免同樣的悲劇再次發生。

吳雅惠於Facebook發文後，隨即有人質疑為何特地要選在[[2018年九合一選舉|2018年九合一選舉]]投票前幾天才發文，認為吳有意打擊基進黨選情。吳雅惠在閱讀過基進黨聲明後又在11月22日於自己的Facebook做出三點回應，其中一點即認為基進黨的處理態度及事後聲明都讓她失望，反問：「為什麼對性騷擾議題採取如此立場的政黨，你還支持得下去？」<ref>{{Cite web |url = https://www.facebook.com/yahuihuihui/posts/1837291329730325 |title = 吳雅惠回應基進黨聲明 |author = 吳雅惠 |date = 2018年11月22日 }}</ref>

===左右派之爭===

即便基進黨自稱是個台獨左派[[剛性政黨|剛性政黨]]，台灣[[社運|社運]]圈、其它左派政黨及團體並不認為基進黨具有左派色彩。2018年6月19日由綠黨、社會民主黨、基進黨組成的[[社會福利國家連線|社會福利國家連線]]，由於基進黨的右派立場和其它兩黨的左派立場相異甚大，綠黨、社民黨黨員及支持者皆對此產生疑慮。<ref>{{Cite web |url = https://www.cmmedia.com.tw/home/articles/10474 |title = 第三勢力小黨組策略聯盟 只是形式大於實質？ |author = 孫偉倫 |date = 2018年6月20日 |publisher = 信傳媒 }}</ref>[[國際社會主義前進|國際社會主義前進]]更直批，基進黨具有[[右翼民粹主義|右翼民粹主義]]傾向。<ref>{{Cite web |url = https://www.socialism.org.tw/2018/07/06/tp0706/ |title = 風雨欲來－台灣國內外的資本主義危機將至 |author = 國際社會主義前進 |date = 2018年7月6日 |quote = 基進黨是一個具右翼民粹傾向的政黨，過往不時發表排外言論（例如將反中共與反中國人混雜起來，以「中國白蟻」這種國族歧視謔稱中國人），也支持不少新自由主義政策。基進黨這種民粹組織沒有一貫的立場，立場往往隨著民情和局勢搖擺，現在為了選票而打出左傾福利綱領。}}</ref>

許多人認為，基進黨過去在環保、土地正義、勞工權益、性別等領域皆持右派立場或有不良表現<ref>{{Cite web |url = https://www.storm.mg/article/451290 |title = 民進黨小弟連線 |author = 高忠義 |date = 2018年6月22日 |publisher = 風傳媒 |quote = 基進黨標榜台獨左派，但長期以來在南部的土地掠奪、勞動權利等議題上，卻甘願做為民進黨之側翼打手。例如：基進黨主席陳奕齊公開迴護高雄市政府強拆果菜市場、驅趕住在旗山大溝頂的弱勢民眾，也為高雄市政府在中央公園砍老樹讚聲，替民進黨惡修勞基法辯駁是台灣中小企業體質承受不起一例一休。}}</ref>，並於2016立委選舉前的宣講當中明確提出「內政交給民進黨，[[台獨|台獨]]交給基進黨」、「國會有113席立委，有性別、勞工、環保、土地代表時，難道不能有深獨的聲音？」<ref>{{Cite web |url = https://www.facebook.com/chenyichi.radicalwing/videos/455054281348595/ |title = 台中公園雨中開講--2016台灣的政治情勢與台灣人民的歷史任務 |author = 陳奕齊 |date = 2016年1月3日  }}</ref>，此處所謂的性別、勞工、環保、土地代表是指民進黨2016年推出的不分區立委名單，但[[不分區立委|不分區立委]]相較於區域立委在更大程度上必須聽令於黨的號令，加上民進黨的政治立場普遍被認為是右派<ref>{{Cite web |url = https://opinion.cw.com.tw/blog/profile/266/article/6497 |title = 勇敢脫去偽裝的外衣，明明是右派就別再「裝左」！ |author = 羅德水 |date = 2018年1月8日 |publisher = 獨立評論@天下 }}</ref>，顯然基進黨並不是那麼認同台獨以外議題的左派立場。

除了在[[2016年中華民國立法委員選舉|2016立委選前]]的新聞稿中明確提到「政治分工」的概念，讓[[蔡英文|蔡英文]]專心內政，由[[台灣團結聯盟|台聯]]扮演「抵抗中國侵入的力量」<ref>{{cite web |author1=陳奕齊 |title=20160108台聯不分區立委陳奕齊新聞稿 |url=https://www.facebook.com/chenyichi.radicalwing/videos/456599254527431 |website=FACEBOOK |accessdate=2016年1月8日}}</ref>；2018年2月7日《基進政治報》第38期再次提到「政治分工」，強調應由蔡英文主掌內政改革，基進黨則作為「內除國賊(「中國白蟻」與[[中國國民黨|中國國民黨]])、外抗強敵(抗中)」的角色<ref>{{cite journal |author1=陳奕齊 |title=台灣正迎來轉機──轉好還是轉壞？ |date=2018年2月7日 |issue=《基進政治報》第38期 |url=http://radicalwings.tw/newsletter_info.php?SerialNo=42}}</ref>。

雖然基進黨有提出性別平權主張，但性別平權議題其實顯現的是[[自由派|自由派]]、[[保守派|保守派]]差異，並不能以此延伸至其它領域的左右立場。因此基進黨的政治立場通常被認為屬於自由右派。

也有人認為基進黨不該被定義為極右派，認為基進黨「時常以荷蘭為效法對象的[[社會民主|社會民主]]路線」，極右派是「來自於過往網路論戰中的印象」。<ref>{{Cite web |url = https://ladopost.com/newsDetail5.php?ntId=37&nId=1384 |title = 社會福利國家連線將面臨的困境與挑戰 |author = 陳子瑜 |date = 2018年6月22日 |publisher = 六都春秋 }}</ref>

協調成立社會福利國家連線的前民進黨立委[[陳昭南|陳昭南]]更是將此連線視為左派群體，痛批[[時代力量|時代力量]]看不出是左派還右派，也看不出時力是反對黨。<ref>{{cite news |author1=林瑋豐 |title=出手整合第三勢力　陳昭南左罵柯文哲、右批時代力量 |url=https://www.storm.mg/article/451165?fbclid=IwAR0YhlhrgoRKOd74-I_OVF3hkKRkyb_dUVmX_ooIy6lNfxestemx-KYHZUs |accessdate=2018年6月19日 |agency=風傳媒}}</ref>

===用金錢打破金權政治===

為了打破金權政治，基進黨於2018年8月在高雄市區一棟大樓頂掛出三面附有照明的廣告帆布，廣告呼籲選民一起終結當前「大灑幣」的選舉遊戲，當下即遭受大量質疑「用金錢打破金權政治」<ref>{{Cite web |url = https://www.facebook.com/radicalwings/photos/a.353905111425345/1147253875423794 |title = 20180813 基進黨聲明稿【重鑄民主工具：用「看板」來怒吼？！】 |author = 基進黨 |date = 2018年8月13日 }}</ref><ref>{{Cite web |url = https://www.facebook.com/sanmin.PehUi/photos/a.543143852685277/689583614707966 |title = 【基進黨對選舉制度的控訴！】 |author = 陳柏惟 |date = 2018年8月12日 }}</ref>。隨後，基進黨又於蘋果日報及自由時報購買數則半版及全版廣告，總額超過新台幣數百萬元<ref>{{Cite web |url = https://www.facebook.com/impolite.tw/photos/a.587170101339406/2055194587870276 |title = 原來，你們一點也不窮 |author = 不禮貌鄉民團 |date = 2018年11月21日  }}</ref>，其中數則廣告內容順道批鬥同屬第三勢力的時代力量，此舉也引起爭議；時代力量高雄黨部主委[[陳惠敏|陳惠敏]]則不認為時力候選人戰力不足<ref>{{cite news |author1=周昭平 |title=基進黨「戰雞」群搶選票　諷時力「黃色小雞」 |url=https://tw.appledaily.com/new/realtime/20181116/1467830/ |accessdate=2018-11-16 |agency=台灣蘋果日報}}</ref>。
