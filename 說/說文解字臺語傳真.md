《'''說文解字臺語傳真'''》（[[白話字|閩南語白話字]]：'''''Soeh-Bûn Kái-Jī Tâi-Gí Thôan-Chin'''''），舊名《'''鬼谷子靈讖'''》，為[[台灣|台灣]]男[[藝人|藝人]][[張宗榮|張宗榮]]製作兼主持的[[現場直播|現場直播]][[電視節目|電視節目]]，在台灣多個[[有線電視|有線電視]]頻道播出，製作單位是張宗榮自組的「張宗榮電視節目製作公司」。

== 節目特色 ==

該節目主要是由[[電視|電視]][[觀眾|觀眾]]播打[[電話|電話]]，[[Call-in|Call-in]]至節目，向張宗榮「求籤」。節目播出的同時，張宗榮除了替觀眾朋友解答疑惑外，節目進行的同時，還會不定時播出〈'''工商服務時間'''〉{{efn|「工商服務」是「[[廣告|廣告]]」的另外一種稱呼。}}，向觀眾推廣[[藥品|藥品]]，直至節目接近尾聲為止。由於整體節目大多是以[[閩南語|閩南語]]進行，所以節目名稱才取名《'''說文解字臺語傳真'''》。

節目內容皆由張宗榮製作[[詩|詩]]與籤文。據張宗榮表示，製作該節目的目的在於「讓觀眾愛看、覺得有趣，能傳遞經驗、又能賺錢」；節目主軸的觀眾測字部分，則沿用古代的諸葛神數：就是凡字加384之數字，湊起來變成一首詩。

節目鼎盛時期，必須天天現場錄影。依照張宗榮的說法，該節目最重要的宗旨在於「當人迷惘時，可以點醒人生走回現實的指引」，並非一個[[算命|算命]]節目。

== 節目播出頻道 ==

該節目曾於[[蓬萊仙山衛星電視台|蓬萊仙山衛星電視台]]、[[國寶衛視|國寶衛視]]、[[台灣藝術電視台|臺灣藝術電視臺]]（包括其前身[[台藝衛星電視台|台藝衛星電視台]]）、[[八大第一台|GTV第1台]]等頻道播出。其中，蓬萊仙山、臺灣藝術電視臺自1995年起播放，共達十年之久。而國寶衛視播出時期，從2000年3月13日起，至同年9月13日為止，播出6個月的時間，一共播出185集。

== 歷屆主持人 ==

張宗榮親自擔任解籤者兼[[主持人|主持人]]，自稱「錢來也」（源自其早年參演的[[中華電視台|中華電視台]]閩南語[[連續劇|連續劇]]《俠士行》的同名[[角色|角色]]），以一襲藍色[[唐裝|唐裝]]的風格主持節目。助理主持人「大驕」與「小驕」，以[[旗袍|旗袍]]裝扮現身；「大驕」為固定助理，「小驕」不一定是固定的人。

== 節目進行方式 ==

節目開始時，「大驕」向觀眾解釋如何向「錢來也」詢問籤詩。觀眾求籤的方式，是先撥打[[電話|電話]]到[[攝影棚|攝影棚]]現場當中，先向大驕告知觀眾的[[姓名|本名]]、[[生肖|生肖]]等，並且解釋「為誰求籤」，還要告知三個[[漢字|字]]。這三個字，是不可以為[[英文字母|英文字母]]或[[阿拉伯數字|阿拉伯數字]]等非[[漢字|漢字]]的字；而且這三個字內，至少一定要有一個字要與求籤者本人有關係，「錢來也」才可以替觀眾解籤詩。「大驕」解說完畢後，現場開放觀眾Call-in。

觀眾Call-in至現場後，「大驕」會向觀眾詢問一些資訊。其中，詢問者的三個字、年齡及生肖，會由「錢來也」以[[毛筆|毛筆]]親筆寫在一張較小的[[宣紙|宣紙]]的右側。「錢來也」就會要求「大驕」將詢問者所提交的三個字的[[筆劃|筆劃]]各自統計出來，取每一個字的筆劃的個位數，開始進行籤詩的組合。例如，有一個觀眾的名字內有個「福」字，想詢問未來的姻緣如何，就可能會將「福、問、緣」三字提交給「錢來也」；「大驕」會將這三字的筆劃算出來後，各取每個字筆劃數的個位數：「福」有14劃，「問」有11劃，「緣」有15劃；各取三字筆劃的個位數，得知籤詩起始數為「415」，換算成詩籤本內的內容，就是從籤詩本內的第5頁第15個字開始。

「小驕」將數字告知「大驕」後，「大驕」就會以[[雷射筆|雷射指示筆]]在字上指示出來，並以[[普通話|普通話]]告訴「錢來也」對應的字是什麼。「錢來也」就會以毛筆把得知的字以由右至左的方式，寫在一張較大的宣紙上。接著，「小驕」就將上一個數字再加上999，推測下一個字為何。以「福、問、緣」取得的415為例，加上999，則為1,414，就是第15頁第14個字。接著，「大驕」一樣也會繼續並持續告知「錢來也」下一個字是什麼字。「錢來也」繼續以由右至左的方式，寫在較大的宣紙上。如此反覆，直至持續加總後，找不到應有的字，才完成一則籤詩。

每一篇籤詩的字數不一定，可以是[[律詩|律詩]]的字數，也可以是[[絕句|絕句]]的字數，或者類似典型的《[[詩經|詩經]]》的字數組合。這都是依詢問者給的三個字當中的每一個字的筆劃數的個位數而定。當然，「錢來也」也會按照求籤者的問題，配合籤詩解釋其內容，並且與觀眾互動，直至觀眾有所解答為止。

解完一位觀眾的籤詩後，觀眾掛斷電話，「錢來也」以閩南語說「繼續來求讖」，「大驕」會持續開放觀眾Call-in，直至節目結束為止。

每當「錢來也」寫完籤詩的第一句時，有時候會穿插預錄的〈工商服務時間〉，播映張宗榮親自向觀眾推廣[[藥物|藥物]]的閩南語[[廣告|廣告]]。該節目沒有其他商業廣告的贊助，[[收入|收入]]來源全部依賴該節目的藥品廣告收入。〈工商服務時間〉播映時，會附上中文[[字幕|字幕]]，也會在[[螢幕|螢幕]]角落加註「廣告」二字。在〈工商服務時間〉中，張宗榮不一定穿著藍色唐裝，有時也會穿著全套[[西裝|西裝]]，只有開場白「各位[[兄弟姊妹|兄弟姊妹]]啊」是幾乎沒改變過的。

因張宗榮本人以閩南語為主要語言，在大多數的情況下，整體節目是以閩南語進行，故電視機前的觀眾要撥打電話給「錢來也」、「大驕」與「小驕」的時候，必須儘量全程以閩南語進行對話。

== 模仿 ==

這個節目的特別性，也引起了好幾個節目或廣告的模仿，其中包含[[衛視中文台|衛視中文台]]《起笑一籮筐》（播出時間：1999年5月30日至2002年4月24日）、[[MUCH_TV|MUCH TV]]《搞笑Very Much》（播出時間：2001年2月12日至2002年3月15日）及[[中天電視|中天電視]]《[[全民大悶鍋|全民大悶鍋]]》。

另外一種模仿，就是[[台灣彩券|台灣彩券]]的[[公益彩券|公益彩券]]廣告「大樂透八期『爽』八億」（從2007年11月20日起至2007年12月14日為止的促銷活動），男主角就是模仿「錢來也」張宗榮。
