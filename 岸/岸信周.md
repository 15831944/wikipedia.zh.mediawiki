'''岸信周'''（生年不詳—1565年9月22日）是[[日本戰國時代|日本戰國時代]]的武將。父親是[[佐藤信連|佐藤信連]]。

最初以[[佐藤氏|佐藤氏]]為姓，家祖是[[藤原秀鄉|藤原秀鄉]]。

== 生平 ==
生年不詳。仕於[[齋藤氏|齋藤氏]]。[[天文_(後奈良天皇)|天文]]11年（1542年），在[[齋藤道三|齋藤道三]]於{{tsl|ja|大桑城}}流放主君[[土岐賴藝|土岐賴藝]]的戰鬥中從軍。天文16年（1547年）（一說天文13年（1544）），在[[加納口之戰|加納口之戰]]中，殺死[[織田信秀|織田信秀]]的外甥[[織田新十郎|織田新十郎]]，受道三下賜{{tsl|ja|感状|感狀}}。同年，再度參與進攻土岐賴藝。[[弘治_(後奈良天皇)|弘治]]2年（1556年），在[[長良川之戰|長良川之戰]]中，投向道三的嫡子[[齋藤義龍|義龍]]方。在義龍死後，仕於義龍的兒子[[齋藤龍興|龍興]]。

[[永祿|永祿]]8年（1565年），與[[關城_(美濃國)|關城]]城主[[長井道利|長井道利]]一同與投向[[織田信長|織田信長]]的{{tsl|ja|加治田城}}城主[[佐藤忠能|佐藤忠能]]戰鬥，敗於織田軍援軍，最後與一族一同在居城{{tsl|ja|堂洞城}}中自殺（[[堂洞合戰|堂洞合戰]]）。

== 子孫 ==
嫡男[[岸信房|信房]]在受信長派遣而來的勸降使者[[金森長近|金森長近]]面前，斬下小孩的首級，而乳母則帶著信房的弟弟秘密逃出城外。女兒（一說名字是榮）亦被乳母帶同逃離，在加治田栃洞區堂洞的東面「姥洞」（{{lang|ja|姥ヶ洞}}）藏匿。弟弟[[佐藤信貞|信貞]]亦在堂洞合戰中戰死。有子孫存續到現代。

== 逸話 ==
*關於從[[佐藤氏|佐藤氏]]改姓[[岸氏|岸氏]]的理由，在[[昭和|昭和]]8年（1933年）到{{tsl|ja|堂洞城}}調査的縣史跡調査委員[[伊藤信|伊藤信]]的報告書中寫下「[[天文_(後奈良天皇)|天文]]十六年九月二二日，勘解由參加[[加納口之戰|加納口之戰]]，向[[岐阜城|岐阜城]]進發時，[[鳶|鳶]]在天上飛翔，並掉下一羽，跌在信周的鎧袖上，在取下來看時，羽毛上有一「岸」字。實在是[[摩利支天|摩利支天]]的奇端，大喜，於是把家名改為岸」（{{lang|ja|天文十六年九月二二日勘解由加納口の戦に加はり、岐阜城を発する時、鳶天に飛び一羽落ちて信周の鎧袖に留まる、取り上げ見るに、羽に岸の文字あり。是れ実に摩利支天の奇端なりとて大いに喜び、則ち家名を改めて岸と称す}}）。在此戰中，殺死[[織田新十郎|織田新十郎]]，因功而受道三下賜感狀，在關市居住的子孫留下這個傳說。
*雖然信長希望以高祿重用，不過信周顯示出與主家一同終結的姿態。
*在堂洞合戰中，18度突進，完全不後退地戰鬥。妻子亦持長刀戰鬥，令人想起[[板額御前|坂額]]的勇戰。
*在戰死前，向身傍的妻子問「信房在做什麼」（{{lang|ja|信房はどうしているか}}）後，回答「北方被擊破，信房被認為戰死」（{{lang|ja|北の方は打ち破られて信房は討死と思われます}}），信周立即流涙，妻子大叫「[[武士_(日本)|武士]]在戰場上隕命是常事。我們都快點戰死吧」（{{lang|ja|武士が戦場で命をおとすは常の習い。さあ私たちも討死を急ぎましょう}}），並作下{{tsl|ja|辞世|辭世}}歌「{{lang|ja|先立つも暫し残るも同じ道、此の世の隙をあけぼのの空}}」，信周則對曰「{{lang|ja|待て暫し敵の波風きり払い倶にいたらん極楽の岸}}」，夫婦互刺而死。
