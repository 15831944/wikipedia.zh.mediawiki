'''岸'''（又稱'''濱'''），分為'''海岸'''、'''湖岸'''及'''河岸'''，是在[[水|水]]面和[[陆地|陆地]]接触处，经[[波浪|波浪]]、[[潮汐|潮汐]]、[[海流|海流]]等作用下形成的滨水地带，其中有眾多沈積物堆積而形成的岸稱為[[灘|灘]]。

== 海水作用 ==
[[File:Sea_defences_South_Coast.jpg|thumb]]
* [[波浪|波浪]]，常造成[[侵蝕|侵蝕]]
* [[海流|海流]]，常造成[[堆積|堆積]]
* [[潮汐|潮汐]]

== 海岸類型 ==
=== 依構成物質區分 ===
可分為[[岩岸|岩岸]]與[[沙岸|沙岸]]。沙岸海岸線平直而水淺。岩岸海岸線曲折而水深，較適合作[[港口|港口]]。

=== 依海面和陸地之相對高度區分 ===
可分為[[沉水海岸|沉水海岸]]與[[離水海岸|離水海岸]]。
* 沉水海岸為原本的陸地沉入海中形成，像原先的山谷形成的[[谷灣|谷灣]]，原先的冰河峽谷形成的[[峽灣|峽灣]]等等。
* 離水海岸為原本的海中地形從海中升起。

=== 華倫亭海岸分類 ===
# 海岸分類的目的是以簡單的指標，將複雜多變的海岸地形初步分類，以幫助掌握[[世界|世界]]上海岸地形的概況。[[華倫亭|華倫亭]]分類是諸多海岸分類法中的一種。
# 華倫亭（H. Valentin）依海岸所受的[[侵蝕|侵蝕]]、[[堆積|堆積]]、[[沉水|沉水]]及[[離水|離水]]四種作用。
# 其相互間消長所成的[[進夷|進夷]]（progradation）、[[退夷|退夷]]（retrogradation）及[[定止|定止]]（stationary ）三種現象作圖解分類，分法簡明。 

==== 進夷海岸 ====
* [[離水|離水]]侵蝕進夷（離水＞[[侵蝕|侵蝕]]） 
* 離水堆積[[進夷|進夷]]（離水＞堆積） 
* 離水堆積進夷（離水＝堆積） 
* 離水堆積進夷（離水＜堆積） 
* [[沉水|沉水]]堆積進夷（沉水＜堆積）

==== 退夷海岸 ====
* 沉水堆積退夷（沉水＞堆積） 
* 沉水侵蝕退夷（沉水＞侵蝕） 
* 沉水侵蝕退夷（沉水＝侵蝕） 
* 沉水侵蝕退夷（沉水＜侵蝕） 
* 離水侵蝕退夷（侵蝕＞離水） 

==== 定止海岸 ====
* 離水侵蝕定止（離水＝侵蝕）
* 沉水堆積定止（堆積＝沉水）

== 海岸微地貌 ==
=== 依形成營力區分 ===
==== 海蝕地形 ====
有[[海蝕柱|海蝕柱]]、[[海蝕崖|海蝕崖]]、[[海蝕平台|海蝕平台]]、[[海蝕門|海蝕門]]、[[海蝕洞|海蝕洞]]等等。

==== 海積地形 ====
* [[海灘|海灘]]，海水搬運[[沙|沙]]堆積於海岸形成。依沙粗細分成[[礫灘|礫灘]]和[[沙灘|沙灘]]（沙粒徑大於兩公釐為礫灘）。
* [[沙洲|沙洲]]，海水搬運沙堆積於海中形成的露出水面的陸地。

== 填海地型 ==
* [[填海|填海]]為人類於沿海地點，根據地形所進行的人為填海工程。

=== 人造海港 ===
* 將海岸規劃為船舶進出停置所規劃的人造[[海港|海港]]工程。

=== 人造海堤 ===
* 些許海岸地勢過低，評估地形許可規劃為防止海水倒灌所進行的人造[[海堤|海堤]]工程。
* 人造海堤可提供防止海水倒灌和並為人造觀海的景點。
