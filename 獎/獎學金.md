'''獎學金'''，是提供給特定[[學生|學生]]的[[金錢|金錢]][[獎勵|獎勵]]，頒發對象包括學業成績優異、操行良好或[[體育|體育]]、[[藝術|藝術]]等方面表現傑出者。

獎學金的額度不等，有些額度甚小，僅是象徵性意義；有些則額度甚大，足以支付學雜費、書籍費，甚至是生活費，讓學生得以在無經濟壓力下專心向學。

== 来源 ==
獎學金來源從私人捐資或是學校、政府、企業、團體組織等所提供。

当政府提供奖学金时，丧失公平，则会引发争议。在[[中华人民共和国高等教育|中国高等教育体制]]中，中国各级政府为追求教育国际化，引吸外籍学生，会给予他们远超本国学生、甚至本国公民人均收入的奖金学，享受[[超国民待遇|超国民待遇]]。以江苏省为例，自2014年起，[[江苏省教育厅|江苏省教育厅]]制定《留学江苏行动计划》，给予每名外籍学生5万至9万的政府奖学金<ref>{{Cite web |url=http://news.163.com/14/0829/07/A4Q3INEE00014Q4P.html|title=凭啥给留学生超国民待遇|accessdate=2018-06-03 |author=晏扬 |date=2014-08-29 |work= |publisher=网易，来源：大洋网-信息时报(广州)|language=Zh-hans }}</ref>。而当年，江苏的人均[[可支配收入|可支配收入]]不过2.7万元<ref>{{Cite web |url=http://js.people.com.cn/n/2015/0203/c360302-23779158.html|title=江苏2014年居民人均收入27173元 城乡差距趋小|accessdate=2018-06-03 |author=朱殿平 |date=2015-02-03 |work= |publisher=人民网|language=Zh-hans }}</ref>，仅是最低奖学金的一半。《留学江苏行动计划》公布后，引发争议。后，江苏省教育厅否认并非所有外籍学生均可享受，但奖学金数额确实如此<ref>{{Cite web |url=http://js.people.com.cn/n/2014/0906/c360307-22227969.html|title=外国留学生来江苏求学每年可获5万至9万资助|accessdate=2018-06-03 |author=蒋廷玉、耿联、杨频萍 |date=2014-09-06 |work= |publisher=人民网|language=Zh-hans }}</ref>，远超江苏居民人均收入。
