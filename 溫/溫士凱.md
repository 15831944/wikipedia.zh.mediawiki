'''溫士凱'''（{{lang-en|'''Da654654654}，{{bd|1967年|2月22日||}}），出生[[新竹縣|新竹縣]]的[[客家人|客家人]]，具有作家、廣播主持人、專業學者、演講師、美食及文創美學講師等多重身份，亦是[[講客廣播電台|講客廣播電台]]《世界異家來作客》的節目主持人。

著作「我在台東，心情，晴」<ref name="auto">{{Cite web |url = http://www.nani.com.tw/nani/jlearn/chin/multimedia/9jchin_2/5/9jc2_course_5.swf |title = 我在臺東心情晴課文}}</ref>一文，被選入[[國民中學|國民中學]][[國文|國文]]教科書課文; 「玩出國際觀」一文，被編入為[[高職|高職]]餐飲管理教科書課文。

榮獲[[第50屆金鐘獎|第50屆金鐘獎]]廣播節目社區節目類最佳節目主持人獎<ref name="auto1">{{Cite web|url =http://goldenbellawards.tavis.tw/50th/winnerlist-broadcast.htm=17114|title =第50屆廣播電視金鐘獎官方網站}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

== 生平 ==
溫士凱生長在[[台灣|台灣]][[新竹縣|新竹縣]][[竹東鎮|竹東鎮]]，是海陸腔客家人。1992前往[[美國|美國]][[加州|加州]][[半工半讀|半工半讀]]，先後完成加州[[格蘭岱爾學院|格蘭岱爾學院]]及[[俄亥俄州|俄亥俄州]][[克裏夫蘭州立大學|克裏夫蘭州立大學]]廣告藝術及設計學位，並旅居美國至2005年移居回台灣。

* 2011年及2009年受邀擔任[[行政院客委會|行政院客委會]] 「築夢計劃」評審委員。

* 2010年應邀為國際知名四季飯店創辦人伊薩多‧夏普先生所著的自傳「金色團隊」一書繁體中文版著推薦序，為該書唯一受邀的華人作家。

* 2010年擔任[[行政院客委會|行政院客委會]]「品客家」美食地圖審查委員。

* 2010年擔任統一左岸咖啡館「世界上最美好的假期」網路活動特別引言嘉賓。

* 2010年接受[[新加坡|新加坡]] [[電視|電視]]電視台「上食堂」節目擔任[[台灣|台灣]] 美食外景主持人。

* 2010年3月受邀「2010泰國週」美食大使，與駐台[[泰國|泰國]]代表Manasvanich共同示範泰國料理。

* 2009年5月接受[[日本|日本]][[NHK|NHK]]電視台[[關西|關西]]新聞部專訪。

* 2009年應邀率領台灣8對新人前往泰國普吉島參加「兩島一文化閩南古式婚裡」。

* 2009年應邀擔任桃園縣政府及桃園縣文化基金會為弱勢青年所舉辦的「達人希望學校」公益活動講師，為弱勢青年提供更寬廣的國際視野及學習空間。

* 2009年12月受邀擔任台東縣「完美伴旅」活動宣傳大使。
