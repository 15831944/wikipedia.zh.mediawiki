'''溫州公共自行車系統'''是[[中國|中國]][[浙江省|浙江省]][[溫州市|溫州市]]的一個公益性公共自行車租賃系統。類似于[[杭州公共自行車|杭州公共自行車]]，實行限時免費，階梯收費的政策<ref name="66wz">{{cite web|url=http://news.66wz.com/system/2012/08/13/103308919.shtml|title=鹿城5000辆公共自行车开始定制 9月出街（图）|publisher=温州网|date=|accessdate=2012-08-14|language=zh-cn}}</ref>。

== 標誌含義 ==
溫州公共自行車系統的標誌（Logo）是由合肥平行线设计工作室设计，融入了拉丁字母“WZ、LC、GG”，分別代表“溫州”、“鹿城”、“公共”。形似一隻小鹿，呼應溫州的別稱“[[鹿城|鹿城]]”，也代表了[[鹿城區|鹿城區]]。又像一輛[[自行車|自行車]]<ref name="66wz"/>。

== 租賃方式 ==
*市民在繳納押金300元跟預存租車費100元之後，可以辦理租車卡。
*市民可持身份證前往[[中國移動|中國移動]]營業廳，存入話費後辦理租車卡。
*2012年12月29日起可用市民卡租赁<ref>{{cite web|url=http://zj.people.com.cn/n/2012/1228/c186327-17930831.html|title=温州市民卡能借公共自行车了 可网上预约办理|publisher=人民网|date=|accessdate=2012-12-28|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20130122181418/http://zj.people.com.cn/n/2012/1228/c186327-17930831.html|archivedate=2013-01-22}}</ref>。市民卡租车需前往“市民卡服务大厅/社保局”签署《市民卡·公共自行车功能开通申请表》并缴纳保证金300元，再往市民卡电子钱包充值或圈存20元以上。

== 发展 ==
2013年2月7日，[[瓯海区|瓯海区]]各停车网点开始调试，随后两天，[[瓯海区|瓯海区]]、[[龙湾区|龙湾区]]的公共自行车系统相继开始运行<ref>{{cite web|url=http://news.66wz.com/system/2013/02/06/103531698.shtml|title=公共自行车三区通借通还|publisher=温州网|date=|accessdate=2013-03-16|language=zh-cn}}</ref>，将与[[鹿城区|鹿城区]]统一借还。

2016年9月，温州市区（鹿城区、瓯海区、龙湾区、洞头区）通用一张“温州市民卡”租车。市区内任意停车点租/还。
