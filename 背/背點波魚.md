'''背點波魚'''（[[学名|学名]]：''Rasbora dorsiocellata''）為[[輻鰭魚綱|輻鰭魚綱]][[鯉形目|鯉形目]][[鯉科|鯉科]]的其中一個[[種|種]]，是一個受歡迎的[[觀賞魚|觀賞魚]]的品種，更被列入由William T. Innes著名的著作《Exotic Aquarium Fishes》中。

== 分布 ==
本魚分布於[[馬來半島|馬來半島]]及[[印尼|印尼]][[蘇門答臘|蘇門答臘]]的酸性泥炭沼澤森林中的溪流。

== 特徵 ==
本魚體細長，體色為金綠色向腹部逐漸減弱為白色。背鰭上有一塊明顯的白邊黑斑；其他鰭的顏色較淺。雌魚的體側比雄魚深些，雄魚的鰭上會有一些明顯的顏色。體長可達6公分。

== 生態 ==
本魚棲息於水草茂密的溪流中，會在水草上產卵，喜群游，屬肉食性，以[[蠕蟲|蠕蟲]]、小型[[甲殼類|甲殼類]]為食，性情溫和。

== 經濟利用 ==
為觀賞性魚類。
