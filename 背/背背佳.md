'''背背佳'''是天津一品科技发展有限公司开发的一款矫正中小学生坐姿站姿的穿戴式医疗产品，用于治疗弯腰[[驼背|驼背]]等不正常姿态，基于[[天津大学|天津大学]]物理系袁兵教授设计的「英姿带」，原理是通过弹力带对穿戴者的姿态进行外力矫正，后来被指出医学效果与其广告宣传不符。<ref>{{cite web |author1=阵雨Fallin（ID：izhenyu163） |title=原来，这些童年广告骗了我们20多年 |url=https://new.qq.com/cmsn/20190309/20190309B0EV1O?pc |website=腾讯网 |publisher=浪潮工作室 |date=2019-03-09}}</ref>

== 营销 ==
背背佳于1997年8月开始在中国上市发售，并邀请了中国大陆家庭[[情景喜劇|情景喜劇]]《[[我爱我家_(家庭情景喜剧)|我爱我家]]》中賈圓圓的饰演者關凌作为[[代言人|代言人]]拍摄[[电视广告|电视广告]]，后又邀请中國大陸歌舞組合[[青春美少女|青春美少女]]组合、[[上海梦幻组合|上海梦幻组合]]以及[[超级女声|超级女声]]选手[[何洁|何洁]]等明星代言拍摄广告。所拍摄的电视广告投放在[[中央电视台|中央电视台]]以及各地方电视台。<ref>{{cite journal |author1=潘惠芳 |title=背背佳:年轻的第一品牌 |journal=中外管理 |date=2000年 |issue=01 |pages=62-64 |url=http://kns.cnki.net/KCMS/detail/detail.aspx?dbcode=CJFQ&dbname=CJFD2000&filename=ZHGN200001027&uid=WEEvREcwSlJHSldRa1FhdkJkVG1BVm9QNS9VRUdscmxTZDdPTEFwZUtJZz0=$9A4hF_YAuvQ5obgVAqNKPCYcEjKensW4IQMovwHtwkF4VYPoHbKxJw!!&v=MDcyOThNMUZyQ1VSTE9mWStabUZ5SG5WYnpJUHlYTVlMRzRIdEhNcm85SFk0UjhlWDFMdXhZUzdEaDFUM3FUclc= |issn=1002-6525}}</ref>

== 争议 ==
背背佳的产品设计目的是矫正青少年的姿态，避免出现驼背，以及对出现驼背的青少年进行矫正，使其恢复正常。但具体的效果却存在争议，一些使用了背背佳的青少年的驼背现象并没有得到改善，反而出现了[[腰椎|腰椎]]前凸或[[胸椎|胸椎]]后凸的副作用<ref>{{cite web |title=多名戴过背背佳孩子出现腰椎前凸胸椎后凸(图) |url=http://news.sina.com.cn/c/2006-10-30/063310357340s.shtml |website=新浪 |publisher=四川在线-天府早报 |date=2006-10-30}}</ref>。有骨科医师指出，背背佳无法矫正[[脊柱侧弯|脊柱侧弯]]，应到正规医院接受检查治疗<ref>{{cite journal |author1=吴翔 |title=驼背矫正,“背背佳”是帮手还是杀手? |journal=医疗保健器具 |date=2006年 |issue=12 |pages=40-41 |url=http://kns.cnki.net/KCMS/detail/detail.aspx?dbcode=CJFQ&dbname=CJFD2006&filename=YBQJ200612023&uid=WEEvREcwSlJHSldRa1FhdkJkVG1BVm9QNS9VRUdscmxTZDdPTEFwZUtJZz0=$9A4hF_YAuvQ5obgVAqNKPCYcEjKensW4IQMovwHtwkF4VYPoHbKxJw!!&v=MjY3Njc0SHRmTnJZOUhaNFI4ZVgxTHV4WVM3RGgxVDNxVHJXTTFGckNVUkxPZlkrWm1GeUhnVUw3TFBDL2FaTEc= |issn=1006-2653}}</ref><ref>{{cite journal |author1=泽敏 |title=背背佳:自封的矫姿专家 纠正驼背没根据 |journal=商品与质量 |date=2006年 |issue=45 |url=http://kns.cnki.net/KCMS/detail/detail.aspx?dbcode=CJFQ&dbname=CJFD2006&filename=SPZN200645006&uid=WEEvREcwSlJHSldRa1FhdkJkVG1BVm9QNS9VRUdscmxTZDdPTEFwZUtJZz0=$9A4hF_YAuvQ5obgVAqNKPCYcEjKensW4IQMovwHtwkF4VYPoHbKxJw!!&v=MTg5MzExVDNxVHJXTTFGckNVUkxPZlkrWm1GeUhnVWIvSk5qM1JZTEc0SHRmSXFvOUZZb1I4ZVgxTHV4WVM3RGg= |issn=1006-656X}}</ref>。
