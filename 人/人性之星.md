'''人性之星'''（{{lang-en|Humanity Star}}）是美国{{tsl|en|Rocket Lab|Rocket Lab|火箭实验室公司}}于2018年1月21日用{{tsl|en|Electron (rocket)|Electron (rocket)|电子号火箭}}从[[新西兰|新西兰]]发射的一颗人造卫星。

== 特点 ==
该卫星外观类似于迪斯科舞厅的闪光灯球，直径差不多1米，全身主要由[[碳纤维|碳纤维]]制成，外边镶上65片三角形镜面，可以不同角度高度反射太阳光至地球，成为夜空中最闪亮的“星星”之一。该卫星部署在低地轨道上，每隔90分钟绕地球一圈。<ref>{{cite news|title=美国一公司发射"人性之星"卫星 或成最亮星|url=http://news.163.com/photoview/00AO0001/2289194.html#p=D9AK591900AO0001NOS|accessdate=2018-01-31|work=网易|date=2018-01-29}}</ref>因其反光能力很强，其[[视星等|视星等]]最高可达7.0<ref name="heavens-satinfo">{{cite web |url=http://heavens-above.com/satinfo.aspx?satid=43168 |title=Humanity Star - Satellite Information |work=[[Heavens-Above|Heavens-Above]] |first=Chris |last=Peat |date=27 January 2018 |accessdate=27 January 2018}}</ref>。

== 目的 ==
该公司特地架设了追踪“人性之星”的网站，有意观测者在上面可以到查询卫星的实时位置。运行9个月之后，“人性之星”将开始慢慢脱离轨道，在地心引力的作用下坠入大气层中燃烧殆尽。该卫星设计者、火箭实验室创始人兼首席执行官彼得·贝克（Peter Beck）表示，这颗高度反射光的卫星没有实际用途，只是意在提醒人类有关气候变化和资源短缺等严峻问题，“希望当大家抬头仰望它时，就能想起卫星背后那辽阔无边的美丽宇宙，无论你正在世界上哪个角落、生活中发生了什么事，都能因此对事情产生不同的想法与行动。”<ref>{{cite news|title=「人性之星」繞地球放光　警醒世人反思問題|url=http://hk.on.cc/int/bkn/cnt/news/20180126/bknint-20180126184530741-0126_17011_001.html|accessdate=2018-01-31|publisher=东网|date=2018年01月26日}}</ref>
