'''人民英雄纪念碑'''位于[[中华人民共和国|中华人民共和国]][[中国首都|首都]][[北京市|北京市]][[天安门广场|天安门广场]]的中央，在[[天安门|天安门]]城楼、[[长安街|长安街]]、国旗旗杆以南，[[毛主席纪念堂|毛主席纪念堂]]以北，是1949年[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]为纪念[[中国近现代史|中国近现代史]]上的[[烈士_(中华人民共和国)|牺牲的人民英雄]]而建的[[纪念碑|纪念碑]]。1949年9月30日奠基，1952年8月1日开工，1958年5月1日揭幕，1961年成为第一批[[全国重点文物保护单位|全国重点文物保护单位]]。该纪念碑是[[中华人民共和国|中华人民共和国]]成立后第一个由国家兴建的大型纪念碑，也是第一个大型公共艺术项目。<ref name="lkq" />

== 纪念碑 ==
人民英雄纪念碑通高37.94米，占地约3100平方米，由碑身、须弥座和台基三部分组成。

碑体平面呈长方形，顶为[[盝顶|盝顶]]。碑身用413块[[花岗石|花岗石]]分32层垒砌而成。正面（北面）碑心是一整块石材，长14.7米、宽2.9米、厚1米、重103吨，镌刻着[[毛泽东|毛泽东]]题写的“'''人民英雄永垂不朽'''”八个鎏金大字。背面碑心由7块石材构成，内容為[[毛泽东|毛泽东]]起草、[[周恩来|周恩来]]书写的碑文，共150字，其文如下：（以下採用為碑文實際字體）

{{Cquote|人民英雄-{紀}-念碑<br>三年以-{来}-在人民解放-{战争}-和人民革命中-{牺}-牲的人民英雄-{們}-永垂不朽<br>三十年以-{来}-在人民解放-{战争}-和人民革命中-{牺}-牲的人民英雄-{們}-永垂不朽<br />由此上溯到一千八百四十年-{从}-那-{時}-起-{为}-了反-{对}-内外-{敌}-人-{争}-取民族-{独}-立和人民自由幸福在-{厯}-<br />次-{鬥争}-中-{牺}-牲的人民英雄-{們}-永垂不朽<br />一九四九年九月三十日　中-{國}-人民政治-{協}-商-{會議}-第一-{屆}-全-{体會議}-建立}}

碑文中的“三年以來”是指[[第二次国共内战|解放戰爭時期]]（1946—1949）；“三十年以來”是指​​以[[五四運動|五四運動]]起點的[[新民主主義|新民主主義]]革命時期（1919—1949），標誌著新民主義革命開始；“上溯到一千八百四十年”是指以[[鴉片戰爭|鴉片戰爭]]為起點的整個民主革命時期（1840—1949）；而1840年則是中國受侵略的開始，1840年鴉片戰爭，中國從此瀰漫著滾滾硝煙​​，成為了[[半殖民地|半殖民地]]半[[封建國家|封建國家]]，標誌著[[中國近代史|中國近代史]]開端。內部敵人：反動派，清政府；外部敵人：侵略者。

在碑身的左右两侧刻有五角星、松柏和旗帜组成的花纹图案。

碑身由[[林徽因|林徽因]]设计的两层[[须弥座|须弥座]]承托。上层小须弥座的四面刻[[牡丹|牡丹]]、[[荷花|荷花]]、[[菊花|菊花]]、[[垂幔|垂幔]]等拼成的八个花环，以示对烈士的崇敬之情。下层大[[须弥座|须弥座]]的束腰部镶嵌着十幅巨大的[[汉白玉|汉白玉]][[浮雕|浮雕]]，其中八幅作品反映了中国近现代史上的革命事件，按东南西北的顺序依次为“[[虎门销烟|虎门销烟]]”、“[[金田起义|金田起义]]”、“[[武昌起义|武昌起义]]”、“[[五四运动|五四运动]]”、“[[五卅运动|五卅运动]]”、“[[南昌起义|南昌起义]]”、“抗日游击战”和“[[渡江战役|胜利渡长江]]·解放全中国”。此外，在北面正中“胜利渡长江”的两侧还有两幅装饰性作品“支援前线”和“欢迎人民解放军”。这十座浮雕的高度均为2米，宽2至6.4米，总长40.68米，一共雕刻了180个人物，是由[[刘开渠|刘开渠]]等人设计创作的。

纪念碑的台基也分为两层，上层呈方形，下层呈[[海棠|海棠]]形，东西宽50.44米，南北长61.5米。两层台基的四面均为栏杆环绕，并设有台阶。整座人民英雄纪念碑共用1.7万块花岗石和汉白玉石砌成，面对[[天安门|天安门]]，肃穆庄严，雄伟壮观。它整合了明清以来的汉民族建筑与希腊罗马建筑的风格，并与北京这座城市的中轴线呼应。

== 兴建 ==
=== 简述 ===
1949年9月30日，[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]通过决议，提出“为纪念在[[第二次国共内战|人民解放战争]]和[[人民革命|人民革命]]、[[民族解放|民族解放]]、[[民主运动|民主运动]]中牺牲的人民英雄”，在北京建立“为国牺牲的人民英雄纪念碑”。当天下午在[[天安门广场|天安门广场]]上，[[毛泽东|毛泽东]]和全体政协代表为纪念碑奠基。此后开始了广泛的设计方案的征集活动。在建筑学家[[梁思成|梁思成]]和雕塑家[[刘开渠|刘开渠]]的主持下，确定了最终的建设方案。1952年8月1日起工程正式开工，到1958年4月落成，同年5月1日举行了隆重的揭幕典礼。

=== 奠基 ===
1949年9月30日，中国人民政治协商会议第一届全体会议闭幕。当天下午6时，參加[[中国人民政治协商会议第一届全体会议代表列表|中国人民政治协商会议第一届全体会议代表]]及首都各界群众代表共3000人在天安门广场南端举行了人民英雄纪念碑奠基典礼。

典礼在《[[义勇军进行曲|义勇军进行曲]]》中开始。[[周恩来|周恩来]]代表中国人民政治协商会议第一届全体会议主席团致词<ref>[http://www.globalview.cn/ReadNews.asp?NewsID=207 人民英雄纪念碑奠基前后，环球视野，于2011-10-1查阅] {{webarchive|url=https://web.archive.org/web/20130411202143/http://globalview.cn/ReadNews.asp?NewsID=207 |date=2013-04-11 }}</ref>：
{{Quote|我们中国人民政治协商会议第一届全体会议为号召人民纪念死者，鼓舞生者，特决定在中华人民共和国首都北京建立一个为国牺牲的人民英雄纪念碑。现在，1949年9月30日，我们全体代表在天安门外举行这个纪念碑的奠基典礼。}}

周恩来致词结束后，全体人员脱帽默哀。此后，[[中央人民政府主席|中央人民政府主席]][[毛泽东|毛泽东]]宣读了亲自撰写的碑文，并第一个为纪念碑的奠基石执锹铲土。

奠基石高1米7，其座高65厘米、宽100厘米、厚35厘米，质地为[[艾青石|艾青石]]。石上有铭文155字，[[隶书|隶书]]，由[[毛泽东|毛泽东]]起草，[[叶恭绰|叶恭绰]]书写，[[陈志敬|陈志敬]]镌刻。<ref>[http://news.sina.com.cn/c/2004-09-26/04544425148.shtml 人民英雄纪念碑奠基碑失踪50年 碑文与现在不同，新浪网，2004年09月26日]</ref>初稿为[[彭真|彭真]]所拟，原题为“中国人民解放战争和中国人民革命烈士纪念碑奠基典礼”。[[毛泽东|毛泽东]]对其加以修改，形成了终稿：<ref>[http://www.gmw.cn/content/2004-12/26/content_153578.htm 詹晓，人民英雄纪念碑的诞生，光明网，2004年12月26日]</ref>

== 修缮记录 ==
* 1971年，纪念碑北面原为[[鎏金|鎏金]]的“人民英雄永垂不朽”题字被改成[[红色|红色]][[玻璃钢|玻璃钢]]字。
* 1980年[[中华人民共和国国庆节|国庆节]]前，纪念碑拆除了红色玻璃钢字，更换了损坏断裂的石料，恢复了原有的鎏金字。
* 1999年6月，在天安门广场进行大规模改造时又对其进行了彻底的清洗和维护。
* 2006年，修复纪念碑存在的漏水、有裂缝、部分地方错位、风化严重等问题，是47年来首次大规模的修缮。<ref>[http://www.singtaonet.com/city/center/t20060209_135833.html 人民英雄纪念碑开始全面修缮]，星岛环球网引自《新京报》，2006年2月9日</ref>

== 历史事件 ==
* 1976年4月5日，天安门广场上发生以悼念已故总理[[周恩来|周恩来]]为目的的[[四五运动|四五运动]]，大量的花圈和诗歌布满了人民英雄纪念碑及广场。这一运动后被[[四人帮|四人帮]]镇压，但运动中的诗歌后来被搜集出版，称为“[[天安门诗抄|天安门诗抄]]”。<ref>[http://cimg.163.com/catchpic/1/1E/1E175714A7DFB61782202984E5736938.jpg 四五运动期间广场上抗议的民众，中央为读诗者]，[[网易|网易]]</ref>
* 1989年4月16日，北京高校的学生把已故前中共中央总书记[[胡耀邦|胡耀邦]]的遗照放在纪念碑上，也有人把[[大字报|大字报]]等贴在这上面。在随后整个[[六四事件|學生運動]]的过程中，纪念碑都是抗议者的指挥部和活动中心。当[[中国人民解放军|中国人民解放军]]戒嚴部隊在6月4日凌晨对天安门广场实施[[六四清场|清场]]时，聚集在纪念碑周围的数千学生是最后离开的。在此之前纪念碑的基座是开放的，任何人都可以上去，近距离参观纪念碑须弥座上的浮雕。而此后纪念碑基座则被护栏围起，不开放予公众。内有武警站岗。此外还有[[中国少年先锋队|少先队员]]的“少年先锋岗”，该岗自1986年起设立，2003年因[[SARS|SARS]]而中断，2006年起恢复。<ref>[http://news.xinhuanet.com/edu/2006-09/29/content_5151029.htm 人民英雄纪念碑将恢复少先队员站岗，新华网，2006年09月29日]</ref>

== 紀念活動 ==
=== 國慶節和烈士纪念日 ===
[[党和国家领导人|党和国家领导人]]，和各界代表，以往在10月1日[[中華人民共和國國慶節|中華人民共和國國慶節]]向人民英雄纪念碑敬献花篮。該儀式於2008年<ref>[http://news.xinhuanet.com/newscenter/2008-10/01/content_10138286.htm 胡锦涛等党和国家领导人向人民英雄纪念碑敬献花篮]，新华网，2008年10月1日</ref>、2010年<ref>[http://politics.people.com.cn/GB/1024/12870526.html 胡锦涛等党和国家领导人向人民英雄纪念碑敬献花篮] {{webarchive|url=https://web.archive.org/web/20111027182354/http://politics.people.com.cn/GB/1024/12870526.html |date=2011-10-27 }}，人民网，2010年10月1日</ref>、2011年<ref>[http://www.gov.cn/ldhd/2011-10/01/content_1961467.htm 胡锦涛等到天安门广场向人民英雄纪念碑敬献花篮]，中国政府网，2011年10月1日</ref>、2012年、2013年<ref>[http://news.sina.com.cn/c/2013-10-01/103128346440.shtml 2013年10月1日]</ref>举行（2009年則是在[[天安門廣場|天安門廣場]]舉行了[[中华人民共和国成立60周年庆典|中华人民共和国成立60周年庆典]]）。2014年8月31日，第十二届全国人大常委会第十次会议通过关于设立[[烈士纪念日|烈士纪念日]]的决定，规定每年9月30日，即人民英雄纪念碑奠基日为烈士纪念日，并规定每年9月30日国家举行纪念烈士活动。

=== 抗日戰爭勝利紀念日 ===
[[党和国家领导人|党和国家领导人]]和各界代表在9月3日[[抗日戰爭勝利紀念日|抗日戰爭勝利紀念日]]向人民英雄纪念碑敬献花篮。该仪式曾于1995年、2005年<ref>[http://politics.people.com.cn/GB/1026/3666029.html 为了和平的纪念 向人民英雄纪念碑敬献花篮仪式侧记，人民网，2005年09月04日]</ref>[[抗日战争|抗日战争]]胜利50、60周年之際举行。

=== 外国领导人敬献花篮 ===
人民英雄纪念碑也是外国领导人来京访问时敬献花篮的场所。
* 1991年8月，日本首相[[海部俊樹|海部俊樹]]在訪華期間，向人民英雄紀念碑獻了花圈，挽聯寫有“人民英雄永垂不朽”幾個字。<ref >[http://roll.sohu.com/20120425/n341535971.shtml “对日中关系要有大局观和决断力”，京報网，2012年04月25日]</ref>
