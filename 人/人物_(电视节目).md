《'''人物'''》，是从2001年起在[[中国中央电视台科教频道|中国中央电视台科学教育频道]]开播的电视节目，主要是讲述各职业人物生平故事，同时通过他们的人生传递[[思想|思想]]和[[知识|知识]]。

== 沿革 ==
2001年7月9日，《人物》正式在中国中央电视台科学教育频道开播，一直延续至今。每周星期四及星期五22点45分播出。

== 收视人群 ==
