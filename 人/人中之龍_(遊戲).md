{{vgname|人中之龍|ja={{jpn|j=龍が如く|hg=りゅうがごとく|rm=Ryūgagotoku|link=no}}|cn=如龙|tw=人中之龍|en=Yakuza}}是2005年[[SEGA|SEGA]]在[[PlayStation_2|PlayStation 2]]上所發售的[[動作冒險遊戲|動作冒險遊戲]]，為《[[人中之龍系列|人中之龍系列]]》的首作。在[[CERO|CERO]]的遊戲年齡分級上是十八禁的遊戲，之後被重新評估分級而變更為十七歲以上才能遊玩。PS2版本的銷量達100萬份以上<ref>https://ameblo.jp/nag0617/day-20090204.html</ref>。2015年9月15日，製作人[[名越稔洋|名越稔洋]]在SCEJA媒體發表會中宣布要在2016年1月21日推出本作在[[PlayStation_4|PlayStation 4]]平台的重製版：《[[人中之龍_(遊戲)#人中之龍_極|人中之龍 極]]》。截至2018年，自本作展開的整個《人中之龍系列》已在全球售出1100萬份<ref>https://www.segasammy.co.jp/english/ir/library/pdf/printing_annual/2018/ssh_ar18e_web.pdf</ref>。

== 概要 ==
再現了能夠在街道中自由移動的動作遊戲。武器和食料等項目相當多樣，使人能感受到其真實感。和主線無關的小遊戲也相當多，其中還有酒店的出現。能夠以步行來移動，採用即時制並且主要以使用武器或空手來戰鬥。在攻擊敵人時會讓「熱血值」上升，到達一定程度就能利用地形或武器來使用「極道技」。根據不同條件能夠使用各式各樣的極道技。

有許多和過去SEGA所發售的《[[莎木|莎木]]》（像是城市住民的全部對話都會呈現出來）及《[[火爆刑事|火爆刑事]]》（一對多的3D格鬥動作遊戲，像是本款遊戲裡面的掉在地上的刀和槍、椅子和桌子、看板和滅火器等背景物、吃飯時所用的胡椒都能作為武器使用）等遊戲相同的共通點。

== 故事大綱 ==
為了守護自己的朋友与心愛的女人而自願扛下殺了自己老大的罪名，坐了十年牢後回到神室町的主角「桐生一馬」與為了找尋母親的少女相遇。這個女孩的名字叫做「遙」，同時她的手中掌握了一筆行蹤不明的一百億元的關鍵资料，各方勢力都為了這筆錢都盯上了這名少女。為了守護遙而戰的桐生，也在戰鬥中開始找尋何謂「人活著的意義」。

== 遊戲中的主要用語 ==
; 堂島之龍
: 為身為堂島組成員時代的桐生的稱號。由來是來自於技術高超的刺青師・歌雕在桐生背後所刺的刺青而得名。而錦山的背後則是被刺上歌雕的作品「躍龍門之鯉」。

; 堂島組
: 桐生和錦山所屬的組織。在堂島組長死後，被風間組所吸收。

; 近江連盟
: 關西的一大黑社會組織。在2代中，關西的劇情裡成為一個重要的關鍵。

; 神室町
: 本作舞台中的街道名。街道模樣會讓人想到[[新宿|新宿]]的[[歌舞伎町|歌舞伎町]]。

《'''人中之龍 極'''》（{{jpn|j=龍が如く 極|hg=りゅうがごとく きわみ|rm=Ryūgagotoku Kiwami}} ）為世嘉於2016年1月21日在[[PlayStation_4|PlayStation 4]]及[[PlayStation_3|PlayStation 3]]發售的遊戲軟體{{refnest|group=注|中文版等海外版只發售PS4版<ref name="asia"/><ref name="korea"/>。}}。另外世嘉在2018年E3上公佈了本作將和《[[人中之龍0_誓言的場所|人中之龍0 誓言的場所]]》一同發售Windows PC版，通過[[Steam|Steam平台]]發行，發售日期為2019年2月19日。<ref>{{Cite web|url=https://www.pcgamer.com/yakuza-0-and-yakuza-kiwami-announced-for-pc/|title=Yakuza 0 and Yakuza Kiwami coming to PC|accessdate=2018-06-15|author=|date=|publisher=PCGamer}}</ref>

宣傳詞為「'''《人中之龍》釋放出，為了成人們的『究極』娛樂'''」。

作為人中之龍系列「10週年記念作品」，將第1部作品《人中之龍》加以重製，在「以10年間培養出來的最新技術重新製作」的概念下，除了將畫面及動作全部更新，並且增加許多新劇情。聲音部分也重新收錄原來九成演出者的配音，也有全新的演出者參與演出的角色<ref name="inside150920">{{Cite web|url=http://www.inside-games.jp/article/2015/09/20/91445.html|title=【TGS2015】《人中之龍 極》橫山P採訪！不僅是重製更是重生、遊戲不超越堅固的牆壁是不行的|publisher=インサイド|accessdate=2015-10-22}}</ref>。

基本系統及戰鬥方式都繼承《0》，但透過經驗值提升能力則是《1》中本來就有的東西<ref name="inside150920"/>。新要素有會在街中各處隨機碰上真島並且展開戰鬥的「真島趴趴走」，以及改編自《[[甲虫王者|甲虫王者]]》的小遊戲「昆虫女王」<ref>{{Cite web|url=http://ryu-ga-gotoku.com/news/detail/999.html|title=新系統「真島趴趴走」、「昆虫女王」介紹影片初公開！|publisher=龍が如く.com|accessdate=2015-11-23}}</ref>。

《極》中的角色：澤村由美與麗奈的cv改為[[坂本真綾|坂本真綾]]與[[田中敦子|田中敦子]]配音。

客串演出者有[[柔道家|柔道家]]及藝人[[篠原信一|篠原信一]]<ref>{{Cite web|url=http://www.famitsu.com/news/201512/12095235.html|title=《人中之龍6（暫定）》先行体驗版PV也公開了！ 《人中之龍 極》成品發表記者會的報導|publisher=ファミ通.com|accessdate=2015-12-12}}</ref>。

ps4遊戲特典則有《[[人中之龍6_生命诗篇。|人中之龍6 生命诗篇。]]》的体驗版，steam版本則無。
