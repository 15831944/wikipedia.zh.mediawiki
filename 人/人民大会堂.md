'''人民大會堂'''位於[[中華人民共和國|中華人民共和國]]首都[[北京市|北京市]][[西城区|西城区]][[天安門廣場|天安門廣場]]西側，建成於1959年。不僅是[[全國人民代表大會|全國人民代表大會]]召開的地點和[[全國人民代表大會常務委員會|全國人大常委會]]的會議和辦公地點，也是[[中華人民共和國|中華人民共和國]]黨和國家舉行政治、外交活動的場所，还对外承接会议与宴请及文艺活动。人民大會堂由[[全國人大常委會辦公廳|全國人大常委會辦公廳]]直属[[事业单位|事业单位]]人民大會堂管理局管理<ref>{{Cite web |url = http://www.gjsy.gov.cn/cxzl/ |title = 人民大会堂管理局 |author =  |date =  |publisher = 事业单位在线 |language =  |accessdate =2017-03-27  |quote =  }}</ref>。

== 歷史 ==
{{seealso|北京十大建筑|天安门广场}}
[[File:Great_Hall_of_the_People_under_construction_01.jpeg|thumb]]
1956年[[第一个五年计划_(中国)|第一個五年計劃]]提前完成後，[[中共中央|中共中央]]開始醞釀在北京建設一個較大的禮堂。1958年8月的[[北戴河會議|北戴河會議]]上，中共中央決定在北京建設包括萬人大會堂在內的多項重大建築工程，並要求在1959年10月中華人民共和國建國十週年之前投入使用。自決定修建至建成僅一年零一個月有餘，時間非常緊迫<ref name=shenbo>{{cite journal|title=人民大會堂建設紀實|author=沈勃|journal=建築創作|year=2014|issue=5|pages=24-25}}</ref>。

1958年9月初，北京市規劃局就萬人大會堂選址進行了规划，確定將萬人大會堂和[[中國國家博物館|革命歷史博物館]]分設於天安門廣場兩側。與此同時，全國各地的設計人員应邀到北京參與[[北京十大建筑|十大建築]]的設計工作<ref>{{cite journal|title=人民大會堂建設紀實|author=沈勃|journal=建築創作|year=2014|issue=5|pages=25-28}}</ref>。

在進行了多輪方案之後，針對天安門廣場佈局的問題，[[中共北京市委|中共北京市委]]召開會議，確定天安門廣場東西寬500米，大會堂的高度可以超過[[天安門|天安門]]。此後又確定大會堂正門不正對[[人民英雄紀念碑|人民英雄紀念碑]]。10月6日，第六稿設計方案送到[[中南海|中南海]]給[[中华人民共和国国务院总理|國務院總理]][[周恩來|周恩來]]匯報，確定了使用[[柱廊|柱廊]]而不是中式大屋頂的樣式。此後又經過全國范圍徵求意見，10月14日夜間由周恩來總理初步確定了設計方案。 <ref>{{cite journal|title=人民大會堂建設紀實|author=沈勃|journal=建築創作|year=2014|issue=5|pages=28-36}}</ref>

經過勘探，大會堂基礎範圍確定北側避開[[元大都|元大都]]護城河河道，南側試圖避開[[金代|金代]]運糧河道但未能完全避開<ref>{{cite journal|title=人民大會堂建設紀實|author=沈勃|journal=建築創作|year=2014|issue=5|pages=37-38}}</ref>。10月10日，基礎範圍內的民房拆遷基本完成。在基礎圖紙基本完成後，10月28日大會堂基礎工程開工<ref>{{cite journal|title=人民大會堂建設紀實|author=沈勃|journal=建築創作|year=2014|issue=5| pages=40}}</ref>。

作為一個邊設計邊施工的工程，開工後全國專家仍然在就建築外觀等問題進行討論。[[梁思成|梁思成]]等認為中選方案形式的古典主義風格，是“西而古”，既不代表中國民族風格，又不能代表[[中国共產黨|中国共產黨]]和中國人民。周恩來總理舉了[[塔|塔]]自[[印度|印度]]傳入中國後中國化，成為中國建築的例子，指出應該秉持國際主義精神，“古今中外，一切精華，兼容並蓄，皆為我用”<ref>{{cite journal|title=人民大會堂修建始末|author=張鎛|journal=建築創作|year=2014|issue=5|pages=158-159}}</ref>。

1959年9月9日凌晨，[[中國共產黨中央委員會主席|中國共產黨中央委員會主席]][[毛澤東|毛澤東]]到工地視察。視察時，北京市副市長[[萬里|萬里]]提出萬人大會堂尚未正式命名。經過一番議論，毛澤東最終敲定名稱為“人民大會堂”<ref>{{cite journal|title=人民大會堂建設紀實|author=沈勃|journal=建築創作|year=2014|issue=5|pages =80-81}}</ref>。9月10日，人民大會堂正式竣工交付使用<ref>{{cite journal|title=人民大會堂建設紀實|author=沈勃|journal=建築創作|year=2014|issue=5|pages=81- 82}}</ref>。

[[文化大革命|文化大革命]]期间，由于受到“[[破四旧|破四旧]]”等极左思潮影响，人民大会堂部分厅室遭到严重破坏，有的厅室除留下一架屏风外，四壁空空如也。国画家[[潘天寿|潘天寿]]、[[吴作人|吴作人]]的作品被从墙上摘下，弃置仓库十年。部分家具被私人占用。布置在台湾厅内的用台湾草编织的沙发也被[[江青|江青]]占为己有。青海厅大型挂壁国画《祁连新貌》被“[[四人帮|四人帮]]”成员之一搬到[[清华大学|清华大学]]装饰他的“接见厅”，直至1979年上半年才找回。北京画家[[俞致贞|俞致贞]]创作的大型屏风重彩工笔画《荷花》文革后被重新发现时，已经变成了一幅大语录牌<ref name=kaifang/>。

1959年人民大会堂建成后，每周都设有一两天的开放日，对公众开放。[[文化大革命|文化大革命]]爆发后，人民大会堂被封闭起来，仅用于举行大型会议及中央领导人办公、接见活动，不再对外开放。文化大革命结束后，人民大会堂对外开放的问题被重新提起，一些中央领导“几次会上会下都提起过。”1978年，[[中共十一届三中全会|中共十一届三中全会]]上，人民大会堂封闭的做法再次遭到部分老同志批评。1979年1月27日晚，中断了15年的[[首都春节联欢文艺晚会|首都群众春节联欢会]]在人民大会堂举行，[[中共中央政治局委员|中共中央政治局委员]][[邓颖超|邓颖超]]代表中共中央宣布：“人民大会堂将向各界群众开放。”1979年7月15日起，人民大会堂重新对公众开放。此后人民大会堂通过收取参观门票，以及在人民大会堂内开设小卖部等的收入，减轻了经费压力。随后人民大会堂加快商业化，对外陆续开放了国宴服务、场地租赁服务等，到1980年代末在行政经费上已达到自收自支。1986年7月4日，男高音歌唱家[[鲁契亚诺·帕瓦罗蒂|帕瓦罗蒂]]在人民大会堂举办个人音乐会，成为首位在人民大会堂演出的外国人，这次商业演出的巨大成功为人民大会堂举办商业演出提供了信心<ref name=kaifang>{{Cite web |url = http://cpc.people.com.cn/GB/85037/8574002.html?wij |title =  人民大会堂对外开放幕后|author =  |date =  |publisher =  中国共产党新闻网|language =  |accessdate =2017-03-27  |quote =  }}</ref>。

2007年12月19日，經[[北京市人民政府|北京市人民政府]]批准，[[北京市規劃委員會|北京市規劃委員會]]、[[北京市文物局|北京市文物局]]將人民大會堂建築列入《[[北京優秀近現代建築保護名錄|北京優秀近現代建築保護名錄]]（第一批）》。

== 建筑 ==
人民大會堂坐西朝東，位於天安門西南，與國家博物館正對。人民大會堂東西寬206米，南北長336米，高46.5米，地上四層，地下一層，佔地面積15萬平方米，建築面積17.18萬平方米。外觀為[[新古典主義建築|新古典主義]]與[[蘇維埃|蘇維埃]]風格的結合，並使用了中國風格的裝飾。建築平面呈「山」字形，外部是淺黃色花崗岩牆面，上有黃綠相間的琉璃瓦屋簷，下有5米高的[[花崗岩|花崗岩]]基座，四周共有134根圓形廊柱。總建築面積比[[北京故宮|北京故宮]]的全部建築面積還大，是全世界最大的會堂式建築。

=== 功能布局 ===
人民大會堂自北向南分為三段。一段主要功能為宴會廳；二段東部為中央大廳，西部為萬人禮堂；三段則為全國人大常委會會議和辦公區<ref>{{cite journal|title=人民大會堂修建始末|author=張鎛|journal=建築創作|year=2014|issue=5|pages=162-163}}</ref>。一段一層為交誼廳，二、三層為兩層高的宴會廳。二段全部高四層，其中東部一層和三層各設一個中央大廳，二層和四層為大廳上空和環形走廊；而西部為萬人禮堂，佔據四層高度。三段呈四方的環形，中間沒有建築，四周為辦公和小會議廳；2009年在環中部偏南進行了增建。由於工期緊張，人民大會堂只設置了台基下的地下室。

=== 宴会厅 ===
宴会厅位于人民大会堂的北墙。从北门进入，经风门厅、过厅到交谊大厅。交谊大厅面积为4,500平方米，[[大理石|大理石]]铺地，四周的明柱和壁柱用桃红色大理石镶砌。

風門廳俗稱北大廳，是中國政府為邦交國元首訪問舉行歡迎儀式的室內場地；東側有國賓會談廳，西側有國賓宴會廳，分別是國家黨政領導人與來賓舉行會談和歡迎宴會的地方。南端有八米寬、62級的[[漢白玉|漢白玉]]階梯通往二層宴會廳；大樓梯頂端有迎賓廳，是黨政領導人在宴會前歡迎貴賓及賓主合影留念之處。迎賓廳置有人民大會堂內最大的一幅國畫《江山如此多嬌》，畫心高5.65米，寬9米，由著名畫家[[傅抱石|傅抱石]]和[[关山月_(画家)|關山月]]以[[毛澤東|毛澤東]]《沁園春·雪》詞意為題材創作，畫題為毛澤東所書。廳內舖有人民大會堂最大的一塊地毯，寬16米，長23米，重三噸多。

宴會廳位於二樓，東西長102米，南北寬76米，高15米，面積7,000多平方米，可以舉辦五千人的宴會或一萬人的酒會。廳內東、西、北環繞寬敞的二層休息廊，奶黃色的牆壁和巨大的圓形廊柱裝飾著瀝粉貼金花飾，南面有供講話和文​​藝演出用的主席台。宴會廳頂天花中央鑲嵌著由水晶玻璃組成的吸頂大花燈，周圍是具有中國民族特色的點金石膏雕塑和棋盤式的彩色藻井。整個大廳金碧輝煌，是舉行盛大國宴和國慶招待會的地方。　

===118廳===
人民大會堂的118廳因為距離大會堂西南門和萬人禮堂的主席台比較近，成為當年[[毛澤東|毛澤東]]最常使用的廳室，常年在此辦公和接見外賓，也是人民大會堂中唯一以原始號碼命名的廳室。118廳布置簡單，沒有奢侈家具：大廳掛4幅畫，擺1組沙發，臥室裡放1張床，室內有張不大的辦公桌和椅子。為保證毛澤東出席大會不發生任何意外，時任國務院總理[[周恩來|周恩來]]在徵得毛澤東主席的同意後親自布置，將118廳作為毛澤東的臨時住地，安排醫護人員在該廳室之地下室設立可應急的供氧中心，還在118廳至大會堂主席台之間，搭設供氧管路。

=== 三楼中央大厅（金色大厅） ===
金色大廳的正式名稱是“三樓中央大廳”，是中華人民共和國[[黨和國家領導人|黨和國家領導人]]接見外國政要及各國駐華使節遞交[[國書|國書]]的場所，也是領導人舉行新聞發布會的場所。 

1993年3月31日下午，國務院總理[[李鵬|李鵬]]和副總理[[朱鎔基|朱鎔基]]、[[鄒家華|鄒家華]]、[[錢其琛|錢其琛]]在三樓中央大廳舉行記者招待會，自此，全國“[[兩會|兩會]]”期間[[國務院總理記者會|國務院總理記者會]]形成常態化、制度化。每年“兩會”時，總理記者招待會通常在三樓中央大廳舉行。 

2010年全國“兩會”期間，因報名參訪的記者很多，[[全國政協|全國政協]]新聞發布會也首次由人民大會堂新聞發布廳移至三樓中央大廳舉行。 <ref name="zgw">[http://www.china.com.cn/guoqing/2013-03/17/content_28266632.htm 讓世界感受中國風情揭秘人民大會堂“金色大廳”，中國網，2013 -03-17]</ref>

由於總理記者招待會在此舉行，記者們見該大廳雕樑畫棟，富有皇家氣象，便稱之為“金色大廳”。隨著“金色大廳”一名的傳開，後來媒體在報導時均稱其為“金色大廳”。 <ref name=zgw/>

該大廳始建於1959年，是位於人民大會堂三層的中央大廳，也是大禮堂和小禮堂的前廳，原設計為休息廳，最初由[[常沙娜|常沙娜]]設計，風格淡雅細緻。 2000年，該大廳曾進行了簡單裝修。<ref>王煒鈺俞靖芝尹思謹王曉東，亦空間亦裝飾人民大會堂金色大廳室內改造工程，室內設計與裝修2010年03期</ref>2009年，該大廳開始全面裝修改造，[[清華大學建築系|清華大學建築系]]教授[[王煒鈺|王煒鈺]]負責該工程。王煒鈺曾經承擔過[[毛主席紀念堂|毛主席紀念堂]]、[[中國國家博物館|中國革命歷史博物館]]、[[全國人大常委會|全國人大常委會]]會議大廳、人民大會堂香港廳、澳門廳的設計裝修任務。 <ref name=zgw/>2010年3月1日，裝修一新的金色大廳作為人大新聞發布廳使用。 <ref>[http://www.iqilu.com/html/zt/china/2010lianghui/pic/2010/0303/191796.html 人民大會堂金色大廳佈置巨幅書法《中華頌》，齊魯網，2010- 03-03]</ref>

該大廳南北長，東西窄，正門開在東側正中間。整個大廳在2009年裝修前由20根白色[[大理石|大理石]]柱構成，裝修之後，白色大理石柱變為雕有金色祥雲圖案。 2009年裝修改造前，該大廳陳列有1幅油畫、5幅國畫。 2010年裝修改造完工後，該大廳陳設的藝術作品有很大變化。 <ref name="zgw" />
* 屏風：2009年裝修改造前，正門進門處擺放有一個屏風《唐人擊鞠圖》。 2010年改為“牡丹屏風”。 <ref name="zgw" />
* 主畫：位於西牆上。 2009年裝修改造前，懸掛[[劉宇一|劉宇一]]創作的油畫《偉業千秋》，描繪了[[毛澤東|毛澤東]]、[[周恩來|周恩來]]等老一輩[[黨和國家領導人|黨和國家領導人]]率群眾進入人民大會堂。 2010年改為[[陳奕純|陳奕純]]創作的大型壁畫《國色天香》。 <ref>陳奕純，人民大會堂金色大廳主畫《國色天香》，《安徽文學》2010年第6期</ref><ref>[http://www.takefoto.cn/viewnews-50981.html人民大會堂主畫作者：名家陳奕純的新人姿態，北晚新視覺，2013-10-24]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
* 西牆北側：2010年新掛國畫《鶴陣飛翔》。 <ref name=zgw/>
* 西牆南側：2010年新掛國畫《孔雀》。 <ref name="zgw" />
* 南牆：2009年裝修改造前，大廳南牆上懸掛三幅國畫，分別繪有迎客松、漓江山水、梅花圖。 2010年改安有長18米、高3.72米的[[蕪湖|蕪湖]]鐵畫《中華頌》，由[[張志和|張志和]]（[[國家行政學院|國家行政學院]]教授、[[中國書法家協會|中國書法家協會]]理事）用兩個月創作，為楷書大字，由蕪湖的鐵藝大師[[儲金霞|儲金霞]]率弟子鍛造為鐵畫。 <ref name="zgw" />2013年，該大廳南北兩側改換藝術作品，原來南牆上的《中華頌》被替換為國畫《黃河，母親河》，長18米，高3.72米，由[[王西京|王西京]]（陝西美術家協會主席）於2013年8月底創作完成，9月初運至北京，陳列在金色大廳，2013年10月19日在人民大會堂陝西廳召開了“王西京為人民大會堂創作巨幅作品《黃河，母親河》捐贈儀式暨座談會”。 <ref name=xhwwt/><ref name="wyw"/>
* 北牆：2009年裝修改造前，大廳北牆上為兩幅潑墨國畫《天馬歡歌圖》、《唐人辭意圖》，均為長18米、高3米。 2010年改安有[[鐘開天|鐘開天]]創作的大型壁畫《壯麗中華》，長18米，寬3.72米。 <ref name=zgw/>2013年，北牆上改掛國畫《長江万裡圖》，由[[袁運甫|袁運甫]]、[[袁加|袁加]]創作於2013年。 <ref name=xhwwt>[http://www.js.xinhuanet.com/2014-03/14/c_119763567.htm 李克強被問“執政一年感受” 笑答"問題太大" ，新華網，2014- 03-14] {{webarchive|url=https://web.archive.org/web/20140314211226/http://www.js.xinhuanet.com/2014-03/14/c_119763567.htm |date=2014-03-14 }}</ref><ref name=wyw>[http://www.cflac.org.cn/ys/xwy/201311/t20131101_229274.html 王西京為人民大會堂創作巨幅作品《黃河,母親河》捐贈儀式暨座談會在京成功舉辦，中國文藝網，2013-11-01]</ref>
* 後面：大廳後面2010年新掛兩幅國畫作品。一幅是[[盧志學|盧志學]]2009年創作的國畫《山高水長》，長6米，寬3.5米。另一幅是反映[[天山天池|天山天池]]的作品。 <ref name=zgw/>

=== 国家接待厅 ===
人民大會堂南端主要部分是全國人大機關辦公場所，一樓設有國家接待廳，是[[黨和國家領導人|黨和國家領導人]]接待貴賓和[[中華人民共和國國家元首列表|中國國家元首]]接受外國新任駐華使節呈遞[[國書|國書]]之處，面積為550平方米。設計富有民族傳統風格。頂部造型是瀝粉貼金棋盤式藻井，懸掛四盞宮燈式水晶吊燈。四周牆壁飾織錦軟包，主牆面上是像徵悠久[[中華文明|中華文明]]的巨幅[[國畫|國畫]]《大河上下·浩浩長春》。
