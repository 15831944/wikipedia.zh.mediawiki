'''人民当家作主'''是[[中华人民共和国|中华人民共和国]]常用的政治词汇，其本质是[[协商民主|协商民主]]，不是“人人当家作主”<ref>{{Cite web|url =http://politics.people.com.cn/n/2014/0818/c1001-25481772.html|title =民主，让人民当家作主（对话价值观(2)·(民主篇)）|accessdate =2014-08-18|author = |date =|publisher =人民网|language =}}</ref>而是要把人民的利益放在第一位，处处为人民着想<ref>{{Cite web|url =http://news.163.com/17/1020/06/D1607PHO000187VI.html|title =让人民当家作主的路越走越宽广|accessdate =2017-10-20 |author = |date =|publisher =人民网|language =}}</ref>，是[[社会主义|社会主义]][[民主政治|民主政治]]的本质与[[核心|核心]]<ref>{{Cite web|url =http://cpc.people.com.cn/xuexi/n1/2017/1106/c385474-29628690.html|title =习近平：人民当家作主是社会主义民主政治的本质和核心|accessdate =2017-11-06|author = |date =|publisher =人民网|language =}}</ref><ref>{{Cite web|url =http://news.ifeng.com/a/20171106/52985487_0.shtml|title =把人民当家作主落到实处|accessdate =2017-11-06|author = |date =|publisher =凤凰资讯|language =}}</ref>。中国的[[马克思主义|马克思主义]]学者认为，世界上没有完全相同的[[政治制度|政治制度]]模式，政治制度不能脱离特定社会政治条件和历史文化传统来抽象评判，不能照搬照抄西方政治模式。因此，中国特色社会主义民主政治也全然不同于西方民主政治<ref>{{Cite web|url =http://cpc.people.com.cn/n1/2017/1105/c64387-29627402.html|title =深刻领会健全人民当家作主制度体系、发展社会主义民主政治的重大部署|accessdate =2017-11-05|author = |date =|publisher =人民网|language =}}</ref>。[[中共十九大|中共十九大]]把坚持[[人民当家作主|人民当家作主]]作为[[习近平新时代中国特色社会主义思想|新时代]]坚持和发展[[中国特色社会主义|中国特色社会主义]]的基本方略之一，十九大报告指出，发展社会主义民主政治就是要体现人民意志、保障人民权益、激发人民创造活力，用制度体系保证人民当家作主<ref>{{Cite web|url =http://news.ifeng.com/a/20171106/52986701_0.shtml|title =用制度体系保证人民当家作主|accessdate =2017-11-06|author = |date =|publisher =凤凰资讯|language =}}</ref>。

[[习近平|习近平]]在庆祝人民代表大会成立60周年大会上指出：“中国这样一个有5000多年文明史，几亿人口的国家建立起人民当家作主的新型政治制度，在中国政治发展史乃至世界发展史上都是具有划时代意义的”<ref>{{Cite web|url =http://politics.people.com.cn/n/2014/1006/c1001-25781475.html|title =坚持和发展人民当家作主的好制度——写在全国人民代表大会成立60周年之际|accessdate =2014-10-06|author =徐显明 |date =|publisher =人民网|language =}}</ref>。
