《'''人民铁道'''》（[[汉语拼音|汉语拼音]]：Rénmín Tiědào；[[英语|英语]]：People's Railway Daily）是[[中国铁路总公司|中国铁路总公司]]（原[[中华人民共和国铁道部|铁道部]]）主管的一份[[报纸|报纸]]，以宣传报道[[中国政府|中国政府]]的[[铁路|铁路]]政策为主，报头为[[中华人民共和国|中华人民共和国]]早期领导人[[毛泽东|毛泽东]]手迹，现社址位于[[北京市|北京市]][[海淀区|海淀区]]北蜂窝3号。[[中华人民共和国|中国]]统一刊号：CN11—0096，国外代号：D958。<ref name="f">{{cite web |url=http://www.chuanmeijia.com/2009/0419/2082.htm |title=人民铁道报 |accessdate=2010-05-12 |publisher=中国传媒家 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20100330101913/http://www.chuanmeijia.com/2009/0419/2082.htm |archivedate=2010-03-30 }}</ref>

== 历史 ==
1949年5月1日于[[中华人民共和国|中国]]北京创刊，第一期发行数量为5000份；由于受中国内部政治局势影响，该报于1957年和1966年两次停刊，1979年1月1日复刊。 <ref name="f"/> 

== 经营发行 ==
1991年1月1日自办发行，发行方式分订阅和旅客列车流动售卖两种。同大多数报刊一样，[[广告|广告]]刊登费是该报的主要经营收入，2010年该报每期彩色整版广告刊登费为11-17万元。<ref>{{cite web|url=http://www.allchina.cn/FilterMedia/PrintPaper/PM_detail_Paper_6257.html|title=人民铁道报2010年广告价格|accessdate=2010-06-28 |publisher=媒体资源网 |language=zh-hans }}</ref>
