'''人行道'''（'''行人專用道'''）是專門為了[[行人|行人]]而鋪設的道路。

== 說明 ==

* 分佈：在[[城市|城市]]，人行道是非常普遍的，一般街道旁均有人行道。在乡村人行道比较少，在野外則更只有人通行而踏出的自然歩道。
* 通行：根据各地不同的交通规则，在有些地区[[自行车|自行车]]也可以在人行道上行走。
* [[街道傢具|街道傢具]]：根据城市规划和现状的不同人行道的宽度可以相差非常大。人行道上还可能有其它的设施比如[[路灯|路灯]]、[[交通信号|交通信号]]、行人用[[指示牌|指示牌]]、[[广告牌|广告牌]]、[[消火栓|地上消火栓]]、[[道路交通標誌|道路交通標誌]]、裝飾用[[地面燈|地面燈]]等等。一些人行道上还有树木、花草和供行人休息的长板凳。在有些地方街道两旁的店铺可以在人行道上放货摊、座椅、遮阳伞等等。
* [[無障礙環境|無障礙環境]]：在发达国家许多地区的法律要求人行道上移除所有不便利[[残疾人|残疾人]]使用的设施，因此在过马路的地方人行道专门降低到马路同一个水平，以便[[轮椅|轮椅]]过马路；也有在人行道上安裝[[導盲磚|導盲磚]]，稱[[盲道|盲道]]。
* 路緣利用：人行道与马路之间的地方稱'''[[路緣|路緣]]'''，是機動車輛[[車道|車道]]的最邊緣處，實現了人車安全的隔離效果。一般會利用路緣的緩衝地帶性質，加建裝置或裝飾品：加裝[[欄杆|欄杆]]以隔離人車，路面設[[阴沟|阴沟]]和[[下水道|下水道]]来排洩[[降水|降水]]，或栽植[[行道樹|行道樹]]作[[綠化|綠化]]。
*改善[[生態環境|生態環境]]：使用透水鋪面，降水可從人行道表面直接滲透進入[[土壤|土壤]]，成為[[水循環|水循環]]的一部份。<ref name="透水鋪面" >{{cite web |url=http://www.aseed.org.tw/information%20service-4/resources.pdf |title=結構性透水鋪面與永續地下水資源及生態環境保育 |publisher=臺灣永續生態工法發展協會}}</ref>。

== 功能 ==

# 交通功能
#* 行人步行之流暢、安全、人車分離
# 商業功能
#* 最小型、覆蓋最廣的商業[[集市|集市]]，比[[購物廣場|購物廣場]]的存在更實際（後者需要在大面積空地建設，前者在有人行道的地方就能形成）
# 社會功能
#* 社交、遊憩、休憩。
# 都市空間、防災功能
#* 適度開放空間可防災，並具環境保育空間。
