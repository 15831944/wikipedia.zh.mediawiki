《'''人生交叉點'''》（{{lang-en|Human Crossing／Human Scramble}}、{{lang-ja|人間交差点}}）是原作[[矢島正雄|矢島正雄]]，並是日本漫畫家[[弘兼憲史|弘兼憲史]]的[[漫畫|漫畫]]作品，曾改编成[[電影|電影]]、[[電視劇|電視劇]]和[[動畫|動畫]]。漫畫中一系列的故事都有關人們的生活，從中學會生活教訓，並吸取經驗。本書在1985年獲得日本第三十屆《[[小學館漫畫賞|小學館漫畫賞]]》<ref>{{Cite web |url=http://comics.shogakukan.co.jp/mangasho/rist.html |title=小学館漫画賞: 歴代受賞者 |publisher=小學館 |language=ja |accessdate=2012年11月18日 |deadurl=yes |archiveurl=https://www.webcitation.org/5msLIPidX?url=http://comics.shogakukan.co.jp/mangasho/rist.html |archivedate=2010年1月18日 }}</ref>。

== 簡介 ==
人的一生就像在一條充滿分岔點的路上行走，當面對每一個十字路口時，不同的抉擇會導致不同的命運。故事內容有涉及「死」<ref>全232話中、半數以上即125話都包含</ref>的相當多，包括各種謀殺，自殺，死亡，被殺，意外死亡，其中有如[[墮胎|墮胎]]等富爭議題目，並有現場的血腥謀殺和絞死的場景詳細描述。

== 漫畫 ==
小學館出版的人生交叉點系列出版於1981年10月1日至1991年3月1日<ref name="Sho1">{{cite web|url=http://skygarden.shogakukan.co.jp/skygarden/owa/solc_dtl?isbn=4091803911|title=人間交差点（ヒューマンスクランブル） 1|publisher=[[小學館|小學館]]|language=ja|accessdate=2012年11月18日}}</ref><ref name="Sho27">{{cite web|url=http://skygarden.shogakukan.co.jp/skygarden/owa/solc_dtl?isbn=4091817572|title=人間交差点（ヒューマンスクランブル） 27|publisher=[[小學館|小學館]]|language=ja|accessdate=2012年11月18日}}</ref>，單行本一共19冊。並在1994年11月17日至1996年6月15日<ref>{{cite web|url=http://skygarden.shogakukan.co.jp/skygarden/owa/solc_dtl?isbn=409192011X|title=人間交差点（ヒューマンスクランブル） 1|publisher=[[小學館|小學館]]|language=ja|accessdate=2012年11月18日}}</ref><ref>{{cite web|url=http://skygarden.shogakukan.co.jp/skygarden/owa/solc_dtl?isbn=4091921299|title=人間交差点（ヒューマンスクランブル） 19|publisher=[[小學館|小學館]]|language=ja|accessdate=2012年11月18日}}</ref>再版重印。

== 動畫 ==
在2003年4月5日至同年6月28日深夜於東京電視台播出。動畫每集都是一個獨立故事，彼此的相連性不大。香港[[無綫電視|無綫電視]][[翡翠台|翡翠台]]則於2005年1月9日起深夜首播。
