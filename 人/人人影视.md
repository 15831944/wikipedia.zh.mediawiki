'''人人影视'''（[[英文|英文]]：'''YYeTs'''，YY指人人，ETS是表示'''E'''nglish '''T'''V '''S'''hows）是中国大陆的影视[[字幕组|字幕组]]网站。

==简介==
网站前身'''YYeTs字幕组'''是于2003年由[[加拿大|加拿大]]华裔留学生小鬼神创立。2006年6月开放了YYeTs美剧论坛，2007年正式更名为'''人人影视'''。梁良作为人人影视的管理员，兼任动画组的组长。为爱好者组织且对外宣称为非商业性组织<ref name=yes>{{cite web|url=http://www.collegedaily.cc/blog/1213/|title=人人影视正式关闭，由加拿大留学生创建至今已10年，背后故事多|format=|work=北美留学生日报|accessdate=2014-12-21|deadurl=yes|archiveurl=https://web.archive.org/web/20141214152913/http://www.collegedaily.cc/blog/1213/|archivedate=2014-12-14}}</ref>。是中國國內最早創立、影響最大的字幕組之一。宗旨为：分享、學習、進步。常年提供国外影视剧和公开课的翻译字幕和资源下载。

网站由于版权问题多次受到关站威胁。2009年，人人影视就因为盗版资源被查，当时包括“伊甸园”、“BT中国联盟”和“悠悠鸟”等中国国内前几大BT网站在内的111家视听节目服务网站都被关停整改。人人影视当时宣布网站转型，放弃视频下载，只提供字幕服务。2013年4月25日[[世界知识产权日|世界知识产权日]]前后，受到影视网站[[思路网|思路网]]被查封事件的影响，人人影视宣布暂时关站<ref name=yes/>。

2014年10月，人人影视在[[美国电影协会|美国电影协会]]公布的一份全球范围内的音像盗版调查报告中被点名<ref name="mpaa">[http://www.mpaa.org/wp-content/uploads/2014/10/MPAA-Filing-to-USTR-on-Worlds-Most-Notorious-Markets.pdf Re: Request for public comment on the 2014 Special 301 Out of Cycle Review of Notorious Markets. ]，Motion Picture Association of America, Inc.[2014-10-24].</ref>。人人影视在微博上回应称由于受到版权压力，人人影视网站将在11月底彻底清除所有无版权资源下载链接，但微博很快被删除。22日下午网站发布公告称暂时关站，屏蔽了中国大陆IP，能通过更改[[Hosts文件|Hosts文件]]来达到访问的目的。此后几日部分网民反映使用中国大陆IP亦能正常访问人人影视网站，而公告则有了更改，并称中国区网站正式关闭，日后可能建立国际版网站<ref>{{cite web |url=http://news.163.com/14/1129/16/AC7TIOP100014SEH.html |title=人人影视"满血复活" 服务器或位于新加坡和韩国 |format= |work=网易新闻中心 |accessdate=2014-12-21}}</ref>。12月开始人人影视关闭了所有国际版服务器，用国外IP访问会提示DNS查找失败<ref>{{cite web |url=http://news.mydrivers.com/1/345/345580.htm |title=人人影视国际版将关闭 美剧迷圣地彻底倒坍 |format= |work=驱动之家 |accessdate=2014-12-21}}</ref>。人人影视在微博上称网站正式关闭。

2015年1月中旬，有网友发现人人影视官网现在仍处在关闭状态，但其主页却出现了神秘的倒计时。网站对该倒计时的标语为“即将蚕变，敬请期待！”<ref>{{cite web |url=http://media.people.com.cn/n/2015/0120/c120837-26416456.html |title=人人影视官网现倒计时"即将蚕变"被疑将有大动作 |accessdate=2015-01-20}}</ref>。

2016年2月6日中午，人人影视在新浪微博宣布，转型美剧社区，更名为“人人美剧”重装上线。<ref>[http://news.163.com/15/0206/15/AHPGFF8P00014SEH.html 人人影视变身美剧社区“复活”，获李开复创新工场天使投资]</ref>

2016年12月16日，人人影视字幕组宣布：正式退出「人人美剧」，现与「人人视频」已无任何关系。<ref>{{Cite web|url=http://www.yyets.com/page2nd-announcement.html|title=重要声明：我们与人人视频已无任何关系|accessdate=2017年9月3日|author=|date=|publisher=}}</ref>
