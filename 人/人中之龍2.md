{{vgname|人中之龍2|ja=龍が如く2|en=Yakuza 2|diff=yes|cn=如龙2|tw=人中之龍2}}是[[SEGA|SEGA]]在2006年12月7日在[[PlayStation_2|PlayStation 2]]上所發售的遊戲。為《[[人中之龍_(遊戲)|人中之龍]]》的續篇。2017年12月7日在PS4平台推出本作的重製版《人中之龍 極2》。<ref>[https://unwire.hk/2017/09/25/ryugagotokukiwami2/game-channel/ 人中之龍 極2 發售日]</ref>

== 概要 ==

故事為《[[人中之龍_(遊戲)|人中之龍]]》一年後。這次的舞台不但有前作[[東京|東京]][[歌舞伎町|歌舞伎町]]，並且擴大舞台到[[大阪|大阪]][[道頓堀|道頓堀]]及[[新世界_(大阪)|新世界]]。這次遊戲的主要戰鬥系統，能從前方拉到後方作360°的全方位攻擊，另外武器及極道技的種類也都大幅增加。

小遊戲除了前作的棒球打擊場之外、還加入了[[高爾夫|高爾夫]]・[[保齡球|保齡球]]等運動項目，而且還能在裡面享受到遊玩[[將棋|將棋]]、[[麻將|麻將]]及[[柏青嫂|柏青嫂]]的樂趣，超越了只是玩免費小遊戲的階段。另外前作大受好評的[[酒店|酒店]]嬢也增加到十個人，在她們下班後還能享受到與她們約會的樂趣。

本作的PS2版本銷量突破100萬<ref>http://www.vgchartz.com/game/4762/yakuza-2/?region=All</ref>。並在2007年度的[[日本遊戲大賞|日本遊戲大賞]]得到優秀賞的肯定。

== 故事大綱 ==

從東城會的「消失的一百億事件」的一年後，桐生和由美的女兒遙一直過著平凡安穩的日子。

但是近江連合瞄準了已經弱化的東城會，為了順利將其合併並進軍[[關東|關東]]地區而不停準備著。作為第五代會長的寺田為了渡過這個危機，特地與桐生商量，但是卻遭到近江連合的刺殺而倒下。

於是桐生繼承寺田的意志，為了拯救東城會而返回神室町。然後親自將寺田所託付的書信送到關西的近江連合那邊，試圖阻止兩者相爭並希望和平解決衝突，但在那裡卻被人稱「關西之龍」的近江連合直參鄉龍會會長——鄉田龍司阻攔。

另外在東城會中還有一個長久以來一直被隱藏的「不能說出的事件」，在經過二十多年後，敵人私底下的復仇也同時進行著。

最後能制霸神室町的究竟會是關東，還是關西？而企圖在幕後操控這一切的黑手究竟是誰？桐生與龍司的雙龍之戰中，誰又能成為真正的「傳說之龍」？

== 劇中主要用語 ==

; 大阪之城
: 並非是知名的[[大坂城|大坂城]]，在遊戲中所指的是千石組的根據地。當一開始出現時是大坂城先分開成兩半，接著從地上升起另一座金色的大坂城。外壁為純金製，城内有忍者和鎧武者常駐著，裡面還養了兩隻老虎，是以《風雲!たけし城》（台譯：《百戰百勝王》）為原型來炫耀其近幾乎不合理的樣子。

; 真拳（ジングォン）派
: [[韓國人|韓國人]]的海外黑幫組織。被堂島宗兵評為有著「不可小覷」的實力，並且還有著做事旁若無人和可怕的殘暴性。20幾年前的聖誕節遭東城會消滅（這次事件的生存者有金大津（キム・デジン）、池頻敏（ジ・ヨンミン）、朴會宗（パク・フェジョン）等三人），此後就沒有任何活動。這次他們所引起的騷動就是為了要向東城會復仇，並且與有著同樣目的鄉龍會及千石組一起聯手侵略神室町。

; 神室町
: 這次以[[東京|東京]]為舞台的街道。從前作開始繼續登場。其模樣會讓人想到[[歌舞伎町|歌舞伎町]]，裡面有稱為「劇場前」等地名，還設置著類似於實際的[[歌舞伎町|歌舞伎町]]正面入口的招牌。

; 蒼天堀
: 以[[大阪|大阪]]為舞台的街道。其模樣會讓人想到[[道頓堀|道頓堀]]。[[かに道楽|かに道楽]]和[[づぼらや|づぼらや]]等名店也會在這個街道中登場。但是著名的固利果看板並沒有出現。閒話一句，在這個城市遭遇戰鬥時，桐生將敵人從橋上丟到水上所使用的極道技名稱為「蒼天堀dive」。

; 新星町
: 另一個以大阪舞台為原型的街道。從原來[[通天閣|通天閣]]塔下的場景，會讓人想到是以[[新世界_(大阪)|新世界]]為原型所創作出來。原本會出現作為該處象徵物的[[比利肯|比利肯]]像，但在《極2》中被刪去。

《'''人中之龍 極2 '''》（りゅうがごとく きわみ 2）為世嘉於2017年12月7日於[[PlayStation_4|PlayStation 4]]發售的遊戲軟體。發售日期與11年前人中之龍2的發售日相同，並搶先在台灣PSN上公布相關訊息。<ref>{{Cite web|url=https://gnn.gamer.com.tw/7/151627.html|title=《人中之龍 極2》搶先曝光 重製 PS2 名作《人中之龍 2》|publisher=巴哈姆特|accessdate=2017-08-25}}</ref>。

本作沿用6代最新的「龍引擎」並改善當時的缺點後重新打造遊戲畫面。並追加真島吾朗視點的故事，並能操作他進行遊玩。同時也對遊戲要素做出多種強化，並帶來更豐富的遊樂場所及支線故事。並且也發售繁體中文版及韓語版。<ref>{{Cite web|url=https://gnn.gamer.com.tw/4/151704.html|title=《人中之龍 極2》正式發表 配合 PS4 硬體水準重新打造經典《人中之龍 2》|publisher=巴哈姆特|accessdate=2017-08-26}}</ref>。

2中的部分角色狹山薰、倉橋涉、別所勉、高島遼改由[[久川綾|久川綾]]、[[木下鳳華|木下鳳華]]、[[木村祐一|木村祐一]]以及[[白竜|白竜]]等演員配音，至於後三位角色連同瓦次郎的面貌，則以配音的演員長相為原型而重新製作。

其他客串的角色，除了有[[青山光|青山光]]，[[橋本梨菜|橋本梨菜]]等2名[[寫真偶像|寫真偶像]]<ref name="famitsu171013">{{Cite web|url=https://www.famitsu.com/news/201710/13144013.html|title=《人中之龍 極2》公布能夠拍攝寫真偶像青山光＆橋本梨菜的新小遊戲！也有訪問影片|publisher=ファミ通.com|accessdate=2017-10-14|date=2017-10-13}}</ref>以外，還有[[摔角選手|摔角選手]]<ref group="注">[[武藤敬司|武藤敬司]]，[[蝶野正洋|蝶野正洋]]，[[長州力|長州力]]，[[天龍源一郎|天龍源一郎]]，[[藤波辰爾|藤波辰爾]]等。</ref>以及[[AV女優|AV女優]]<ref group="注">[[明日花綺羅|明日花綺羅]]，[[AIKA_(AV女優)|AIKA]]，[[高橋聖子|高橋聖子]]，[[桃乃木香奈|桃乃木香奈]]，[[三上悠亞|三上悠亞]]等5名女優。然後，三上在《6》中，也在小遊戲「線上聊天」出現過。</ref>等各5位登場<ref name="famitsu170921">{{Cite web|url=https://www.famitsu.com/news/201709/21142264.html|title=《人中之龍 極2》武藤敬司・蝶野正洋・長州力・天龍源一郎・藤波辰爾 VS 桐生一馬・真島吾朗實現!?【TGS2017】|publisher=ファミ通.com|accessdate=2017-09-21|date=2017-09-21}}</ref><ref name="famitsu171020">{{Cite web|url=https://www.famitsu.com/news/201710/20144472.html|title=明日花綺羅，三上悠亞，高橋聖子，桃乃木香奈，AIKA在《人中之龍 極2》演出影像公開！|publisher=ファミ通.com|accessdate=2017-10-26|date=2017-10-20}}</ref>。

本作採用《[[人中之龍6_生命詩篇。|人中之龍6 生命詩篇。]]》中的「龍引擎」進行重製，並改善當時的缺點後，再以《6》的系統為本作基礎<ref name="famitsu170826"/>，但武器装備等要素則是《2》本來就有的系統<ref name="dengeki170922">{{Cite web|url=http://dengekionline.com/elem/000/001/596/1596737/|title=《人中之龍 極2》收錄《電腦戰機》！ “人中之龍工作室”【TGS2017】舞台報告|publisher=電撃オンライン|accessdate=2017-10-26|date=2017-09-22}}</ref>

追加要素方面，收錄有《2》中未描寫到的真島吾朗的新劇情，該段故事中也能操作真島進行遊戲 <ref name="famitsu170826"/>。

小遊戲方面，新增保鑣任務，追加有將《6》的幫派創造者加以改良的「真島建設<ref name="famitsu170921"/>」、將《[[人中之龍0_誓言的場所|人中之龍0 誓言的場所]]》的風化島加以改良的「新・風化島<ref name="famitsu171020"/>」，一邊與女孩子對話一邊進行寫真攝影的「寫真偶像攝影工作室<ref name="famitsu171013"/>」以外，遊戲中心裡面也能遊玩初代《[[電腦戰機|電腦戰機]]》、《觀瀑樂園》等新遊戲<ref name="dengeki170922"/><ref>{{Cite web|url=http://dengekionline.com/elem/000/001/597/1597972/|title=桐生一馬在廁所裡面進行無仁義之戰!? 《人中之龍 極2》中“廁所小遊戲”登場【TGS2017】|publisher=電撃オンライン|accessdate=2017-10-26|date=2017-09-23}}</ref>。

2019年3月31日，通過ＥＳＲＢ評級，確定登入PC。

2019年4月11日宣布，在2019年5月9日正式登入steam，字體只支援英文和日文。

2019年5月17日，發行人中之龍極2 steam的組合包(如果再5/9前購買遊戲本體，組合包免費贈送。)
