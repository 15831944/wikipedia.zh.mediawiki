{{engname|《人鱼朵朵》|The Shoe Fairy}}是一部[[李芸婵|李芸婵]]执导、[[徐若瑄|徐若瑄]]主演的[[台湾电影|台湾电影]]，於2006年上映。获得2005年[[第42届金马奖|第42届金马奖]]最佳艺术指导奖（[[王逸飞|王逸飞]]）和最佳视觉效果提名（[[楊漢昶|楊漢昶]]、[[林志超|林志超]]）<ref>{{cite web |url=http://app.atmovies.com.tw/movie/movie.cfm?action=filmdata&film_id=fstw40235140 |language= |title=人魚朵朵 |publisher=开眼电影 |date=2006年 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20131016215701/http://app.atmovies.com.tw/movie/movie.cfm?action=filmdata&film_id=fstw40235140 |archivedate=2013-10-16 }}</ref>。

== 剧情简介 ==
影片充满[[童话|童话]]故事的影子，如《[[糖果屋|糖果屋]]》、《[[杰克与魔豆|杰克与魔豆]]》、《[[快乐王子|快乐王子]]》等，饱含童话与幻想色彩。

朵朵出生后双脚不能走路。后来，她接受了[[手术|手术]]终于能够走路，并获得了生命中的第一双鞋。之后，她就像被下了魔咒，不停地搜集美丽的鞋子。

朵朵长大后到童书出版社上班。通过神秘[[插画家|插画家]]的介绍，她认识了她的白马王子——牙医维孝，双方结了婚，从此他们能过上快乐的生活吗？

== 创作 ==
本片是[[刘德华|刘德华]]投资的“亚洲新星导”计划的第一部作品。是李芸婵从事电影业10年来的首部长片，并将格调定位成一个美丽的童话。

本片的部分情节被疑似抄袭根据日本作家[[村上春树|村上春树]]的同名短篇小说改编成的影片《[[托尼瀑谷|托尼瀑谷]]》，两片在台湾的同一档期上映。李芸婵表示自己从高中时就很喜欢村上的小说，但否认主动抄袭，她说情节类似是“潜移默化”的结果<ref>{{cite web |url=http://data.ent.sina.com.cn/movie/moviedetail.php?id=3690&detail=background |language= |title=人鱼朵朵幕后花絮 |publisher=新浪 |date= |accessdate= }}{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
