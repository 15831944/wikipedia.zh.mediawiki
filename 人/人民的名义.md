《'''人民的名义'''》是2017年推出的一部以[[政治|政治]]和[[反腐|反腐]]为题材的[[中国大陆电视剧|中国电视剧]]，于2017年3月28日起在[[湖南卫视|湖南卫视]][[金鹰独播剧场|金鹰独播剧场]]档次播出<ref>{{Cite web |url=http://ent.xinmin.cn/2017/03/16/30900994.html# |title=2017反腐大剧《人民的名义》3月28日登陆湖南卫视 |access-date=2017-03-18 |archive-url=https://web.archive.org/web/20170330015804/http://ent.xinmin.cn/2017/03/16/30900994.html# |archive-date=2017-03-30 |dead-url=yes }}</ref>，由[[陆毅|陆毅]]、[[张丰毅|张丰毅]]、[[吴刚_(中国演员)|吴刚]]、[[许亚军|许亚军]]、[[张志坚_(演员)|张志坚]]、[[柯蓝|柯蓝]]、[[胡静|胡静]]领衔主演<ref>[http://ent.sina.com.cn/v/m/2017-03-17/doc-ifycnpiu8909879.shtml 《人民的名义》陆毅变最强检察官 亮剑反腐猛打虎]</ref><ref>[http://ent.sina.com.cn/v/m/2017-03-17/doc-ifycnpiu8930953.shtml 《人民的名义》胡静变商界女强人 挑战神秘狠角色]</ref>。剧情讲述了检察官与贪官污吏之间因贪腐问题展开的殊死较量，最终贪腐集团被绳之以法的故事<ref name=henan />。该剧播出后反响极大，创下了近10年来[[中国省级卫视电视剧收视率排名表|中国国产电视剧的最高收视记录]]<ref name=收视率 />。

== 制作与发行 ==
=== 摄制背景 ===
1990年代至2000年代初，[[中国大陸|中国大陸]]拍摄过大量反腐题材影视剧。随着反腐题材影视剧数量攀升，其内容和质量日益良莠不齐。在2004年度全国电视剧题材规划会上，[[国家广电总局|国家广电总局]]、[[中国中央电视台|中央电视台]]、中国电视艺术委员会及全国各电视剧制作单位的代表表示要解决反腐、涉案剧过多、过滥的问题，国家广电总局决定进一步压缩警匪剧、反腐剧的规划立项比例。2004年，国家广电总局下发通知，要求全国所有电视台在晚间黄金时段不得播放渲染凶杀暴力的涉案题材影视剧<ref>{{Cite web|url =http://news.xinhuanet.com/newmedia/2004-04/20/content_1428981.htm|title =广电总局：涉案反腐剧均属于黄金时间禁播范围|author =|date =2004-04-20|publisher =新华网|accessdate =2017-04-19|deadurl =yes|archiveurl =https://web.archive.org/web/20041127172049/http://news.xinhuanet.com/newmedia/2004-04/20/content_1428981.htm|archivedate =2004-11-27}}</ref><ref>{{Cite web|url =http://news.xinhuanet.com/newmedia/2004-03/30/content_1391307.htm|title =犯罪手法腐败内幕描写升级 压缩此类电视剧立项|author =|date =2004-03-30|publisher =新华网|accessdate =2017-04-19|deadurl =yes|archiveurl =https://web.archive.org/web/20050407215309/http://news.xinhuanet.com/newmedia/2004-03/30/content_1391307.htm|archivedate =2005-04-07}}</ref>。此后，反腐剧进入约十年的“沉寂期”。2004年至2014年，仅有《[[我主沉浮|我主沉浮]]》、《[[高纬度战栗|高纬度战栗]]》等少数几部反腐题材电视剧播出<ref name=zgqnw>{{Cite web |url =http://news.youth.cn/jsxw/201704/t20170406_9423467_1.htm  |title = 《人民的名义》导演：反腐剧能不能写 尺度多大？ |author =  |date = 2017-04-06 |publisher =中国青年网   |accessdate =2017-04-19 }}</ref>。

随着2013年之后[[中共十八大以来的反腐败工作|中共十八大以来的反腐败工作]]的开展，反腐成为社会关注的热点。最高人民检察院影视中心认为可尝试恢復制作反腐剧。2014年底，[[最高人民检察院|最高人民检察院]]影视中心在征得[[国家新闻出版广电总局|国家新闻出版广电总局]]电视剧司司长[[李京盛|李京盛]]的同意后，邀请政治小说家[[周梅森|周梅森]]创作一部反映现实的反腐题材电视剧。这是最高人民检察院影视中心同期制作的多部现实主义反腐剧之一（另有《[[天网行动2015|天网行动2015]]》、《[[跨国追贪|跨国追贪]]》等）。2015年国家新闻出版广电总局电视剧司、国家新闻出版广电总局监察局的先后表态也被业界视为“有条件解禁”反腐、涉案剧的信号。随着2015年一大批法制题材电视剧播出，特别是《[[刑警队长|刑警队长]]》在晚间黄金时段播出，2004年的禁令被部分打破<ref name=zgqnw/><ref name=zgjjw>{{Cite web |url =http://www.ce.cn/culture/gd/201507/28/t20150728_6059751.shtml  |title = 反腐剧“有条件解禁” 近半年广电总局备案达11部 |author =  |date = 2015-07-28 |publisher =中国经济网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://ent.sina.com.cn/v/m/2015-10-22/doc-ifxizwsf8751631.shtml |title =广电总局：法制剧要反腐也要倡廉  |author =  |date = 2015-10-22 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

=== 融资与拍摄 ===
经过2004年的禁令限制，制作公司、投资方对制作反腐剧仍心有余悸。另外，在国家广电总局对反腐剧的“有条件解禁”下，反腐剧的审查门槛比2004年之前显著提高。再者，2015年至2016年热播剧前十榜上，现实主义题材剧几乎没有，而反腐剧的制作难度又远超一般现实题材剧，这一时期主旋律影视剧、政治类题材影视剧的票房和收视率普遍都很低。因此周梅森完成《人民的名义》剧本后，本剧经历了多轮投资方和发行方集体撤资，先后撤资的投资方共计50多家。本剧开机时仍有2000万元人民币资金缺口。最终仅剩的6家投资方里，除发起投资方凤凰传奇外，其余5家均为第一次投资电视剧的初创公司。其中，本剧制片人[[高亚麟|高亚麟]]作为创始人的嘉会文化投资占比40%，本剧导演[[李路_(导演)|李路]]的弘道影业投资占比10%，大盛国际和上海利达影业投资分别占比15%，最后一家投资方为北京正和顺影业<ref name=zgjjw/><ref name=huo/>。

====取景====
2016年2月至6月，本剧於[[北京|北京]]、[[南京|南京]]、[[泰州|泰州]]、[[句容|句容]]、[[重庆|重庆]]等地拍摄取景，其中95%左右的镜头是在[[南京|南京]]拍摄完成。剧中的汉东省检察院是在东南大学大礼堂拍摄，国家部委大楼则是东南大学图书馆。第十五集中，剧中出现的一个空镜头为重庆嘉滨路黄花园大桥<ref>{{cite web|url=http://news.163.com/17/0409/15/CHJE88MG00018AOR.html|title=《人民的名义》曾来重庆取景？黄花园大桥大美江景入镜！|publisher=重庆晨报上游新闻|date=2017-04-09|accessdate=2017-04-09}}</ref>。

陈海儿子小皮球的学校是在南师附中新城小学南校区取景<ref>{{cite web|last1=13465|first1=ck|title=南京版《人民的名义》十级考题，答对一半算我输！_新闻频道_中华网|url=http://news.china.com/news100/11038989/20170502/30479817_all.html|website=news.china.com}}</ref>。

=== 发行 ===
本剧的发行也历经波折，与本剧接触过的中央电视台、[[东方卫视|东方卫视]]、[[浙江卫视|浙江卫视]]、[[江苏卫视|江苏卫视]]等多家电视台最终均因出价太低而没能购买。直到2016年5月，投资1.2亿元人民币的本剧才被以播出娱乐节目见长的[[湖南卫视|湖南卫视]]以2.2亿元人民币买下，购剧协议中约定2017年1月15日交片，一旦未通过国家新闻出版广电总局审查，超过一天罚五万元人民币，连续十天后，合同自动终止。制片方和投资方对如期通过审查均无把握。但经新任国家新闻出版广电总局电视剧司长毛羽支持，本片迅速通过审查，这才避免了合同自动终止。本片2017年3月在湖南卫视首播前，几乎是零宣传<ref name=zgjjw/><ref name=huo/><ref>{{Cite web |url =http://ah.people.com.cn/n2/2017/0410/c358329-29994725.html  |title =《人民的名义》为什么没在央视播？湖南卫视2.2亿买下  |author =  |date = 2017-04-10 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://cjrb.cjn.cn/html/2017-04/18/content_5606555.htm |title =  《人民的名义》编剧、小说作者周梅森：没有机构授意影响剧组创作|author =  |date = 2017-04-18 |publisher = 长江日报 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.ifeng.com/a/20170404/50886021_0.shtml |title = 大尺度反腐剧《人民的名义》厉害了！小长假在朋友圈刷屏 |author =  |date = 2017-04-04 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.163.com/17/0410/08/CHL84KCE0001875P.html |title = 《人民的名义》曾吓跑几十家投资方 被小公司捡漏 |author =  |date =2017-04-10  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://wemedia.ifeng.com/13919099/wemedia.shtml |title = 《人民的名义》 紧追几步，跟上时代 |author =  |date = 2017-04-28 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

2017年3月28日，《人民的名义》首集在湖南卫视金鹰独播剧场播出。两周后，[[PPTV聚力|聚力传媒]]（PPTV）与制片方及湖南卫视签署协议，用“1个多亿，接近2亿元”的价格购买了该剧的网络独播权及分销权<ref name=daodi>{{Cite web |url =http://www.sohu.com/a/137471503_465299  |title = 《人民的名义》到底创造了多少收益 |author =  |date = 2017-04-30  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。随后在网络平台，聚力传媒（PPTV）采用“以剧换剧”的形式，将该剧分销给[[芒果TV|芒果TV]]、[[优酷|优酷]]、[[爱奇艺|爱奇艺]]、[[腾讯|腾讯]]等多家平台。根据猫眼数据统计，《人民的名义》上线26天，累计播放量已达116.5亿<ref name=huo>{{Cite web |url =http://media.people.com.cn/n1/2017/0424/c40606-29230253.html  |title =《人民的名义》怎样火的 高亚麟：多亏了年轻观众  |author =  |date = 2017-04-24 |publisher =  人民网  |language =  |accessdate =  |quote =  }}</ref>。

由于《人民的名义》收视率火爆，所以制片方和投资方、湖南卫视、聚力传媒都获得了巨大的商业收益<ref name=daodi/>。

==剧情==
国有资源管理部二处长趙德漢（[[侯勇|侯勇]]饰）被人举报受贿上亿元后，[[最高人民检察院反贪污贿赂总局|最高人民检察院反贪总局]]侦查处长侯亮平（[[陆毅|陆毅]]饰）前来搜查时，看到的却是一位长相憨厚、衣着朴素的农民在简陋破败的旧房里吃炸酱面，而后调查组发现他的别墅内藏有巨额贿款。与案件牵连甚紧的汉东省京州市副市长丁义珍（[[许文广|许文广]]饰），以反侦察手段乘坐[[美国联合航空|美国联合航空]]23432号航班逃往[[洛杉矶国际机场|洛杉矶]]，流亡美国和非洲。最终，案件线索定位于由京州光明峰项目引发的国企大风集團服装厂的股权争夺（又称一一六事件）。

在调查行动中，汉东省检察院反贪局长、侯亮平的大學同學陈海（[[黃俊鵬|黃俊鵬]]饰）遭到人为制造的“车祸”暗算，成为植物人。而精明干练的侯亮平临危受命，加上当时汉东“政治生态恶劣”，接任汉东省检察院反贪局长，继续调查行动。而在汉东省政坛，以省委副书记、政法委书记高育良（[[张志坚_(演员)|张志坚]]饰）为代表的“汉大帮”，以省委常委、京州市委书记李达康（[[吴刚_(中国演员)|吴刚]]饰）为代表的“秘书帮”也相争多年，新任省委书记沙瑞金（[[张丰毅|张丰毅]]饰）也打破政治的平衡局面。经过检察官与腐败分子的殊死较量后，最终，以前任省委书记赵立春为核心的贪腐集团被绳之以法<ref name=henan>{{Cite web|url =http://henan.china.com.cn/culture/2017/0424/4643355.shtml|title =观众有福了 反腐剧《国家行动》来了!比《人民的名义》还震撼 反腐打黑引关注|author =|date =2017-04-10|publisher =中国青年网|language =|accessdate =|quote =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url = http://cul.sohu.com/20170111/n478444445.shtml |title = “中国政治小说第一人”──著名作家周梅森长篇反腐小说《人民的名义》出版 |author =  |date =2017-01-11  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

此外，剧中光明区信访办接待窗口在现实中也多有存在，民众需要蹲着才能办事情，被戏称为“丁义珍”式窗口<ref>{{Cite web |url=http://news.163.com/17/0418/10/CIA3ICS80001875P.html|title=湖南株洲火车站派出所现"蹲式窗口" 市民蹲着办证|publisher=株洲网| date=2017-04-18 | accessdate=2017-04-20}}</ref><ref>{{Cite web |url=http://www.thepaper.cn/newsDetail_forward_1666400|title=江西吉安房管局回应“丁义珍”式办事窗口：已整改，将添凳子|publisher=株洲网| date=2017-04-19 | accessdate=2017-04-20}}</ref>。例如在电视剧热播期间，郑州市人力资源和社会保障局被网友曝出接待大厅窗口与《人民的名义》中如出一辙，官方紧急做出公告回应<ref>{{Cite web |url=http://www.thepaper.cn/newsDetail_forward_1662789|title=郑州市社保局回应“窗口办事要蹲着”：已将便民座椅复位固定|publisher=澎湃新闻| date=2017-04-14 | accessdate=2017-04-20}}</ref>，于3天后将窗口前的玻璃撤掉并增加座椅<ref>{{Cite web |url=http://photo.dahe.cn/2017/04-18/108396799.html|title=郑州市社保局“跪式窗口”玻璃围栏已撤掉|publisher=大河网| date=2017-04-18 | accessdate=2017-04-20}}</ref>。之后，全国多地的蹲式窗口被曝光。编剧周梅森在采访中称，窗口情节的灵感来自自己在江苏亲眼所见，至今仍未整改。对于多次举报、曝光解决不了的问题，却通过一部电视剧解决了，周梅森称出乎意料，并表示文艺作品不承担解决社会问题的功能，它只影响和引领世道人心<ref>{{Cite web |url=http://dz.xdkb.net/html/2017-04/23/content_459830.htm|title=现代快报·ZAKER南京“以网友的名义”专访周梅森|publisher=现代快报| date=2017-04-23 | accessdate=2017-04-26}}</ref>。

该剧播出后，剧中的一些取景地也成为了“[[网红|网红]]”旅游地<ref>{{Cite web |url=http://news.163.com/17/0430/06/CJ8GOVHH00018AOP.html|title=《人民的名义》取景地南京成了“网红”旅游地|publisher=华商报(西安)| date=2017-04-30 | accessdate=2017-04-30}}</ref>。

== 争议 ==
=== 完整劇集非法外流事件 ===
2017年4月13日，正在播出的《人民的名義》全片遭到非法外流，洩露視頻畫面上標註有「[[审查制度|送審]]樣片」字樣。據導演、總製片人李路稱，當日晚11時許，有200多名[[网络水军|水軍]]在全世界的網路上散佈該劇集全片，還有人在[[微博|微博]]、[[微信|微信]]朋友圈、[[淘寶網|淘寶網]]等處兜售遭洩露的視頻資源。對此，湖南衛視官方表示將徹查此事<ref>{{Cite web|url=http://ent.ycwb.com/2017-04/14/content_24642089.htm|title=《人民的名义》全集泄露 湖南卫视称将彻查到底_金羊网新闻|accessdate=2017-04-16|work=ent.ycwb.com|archive-url=https://web.archive.org/web/20170416232130/http://ent.ycwb.com/2017-04/14/content_24642089.htm|archive-date=2017-04-16|dead-url=yes}}</ref>。4月15日，李路在新聞發佈會上表示，目前他正在配合公安機關在網上搜索目標<ref>{{Cite web|url=http://appnews.legaldaily.com.cn/fz_jhxt/index.php?s=/Public/shareNews/nid/13101/isorder/0|title=200多水军散布《人民的名义》盗版 导演李路：我两天都没有睡觉|accessdate=2017-04-16|work=appnews.legaldaily.com.cn}}</ref>。[[国家版权局|国家版权局]]版权管理司司长于慈珂就此事件表示，这种行为应予以制止和查处<ref>{{Cite web|url=http://news.sina.com.cn/o/2017-04-25/doc-ifyepnea5010499.shtml|title=国家版权局回应《人民的名义》盗播：予以制止和查处
|accessdate=2017-04-25|work=中国新闻网}}</ref>。另外，在《人民的名义》播完后，全国一些十八线地市县市台亦开始通过下载片源等手段来编排该剧在当地的二轮播出，湖南卫视已表示该剧将压制版权，不售给其他电视台，同时严查地方台盗播该电视剧的行为。

=== 剧组疑遭诽谤 ===
2017年4月2日，环球网财经频道转载一篇未署名文章（文章来源标明百姓生活网，但至4月3日百姓生活网上并未搜到相关文章），称某博主“恶意诽谤《人民的名义》”，剧组“通过公安和网络管理部门，在不到四个小时就锁定了嫌疑人”，对其批评教育后封禁了其微博<ref name="环球">{{cite news|url=http://finance.huanqiu.com/roll/2017-04/10419284.html|title=恶意诽谤《人民的名义》触犯法律 “天花”博主被封号|publisher=环球网财经频道|date=2017-04-02|accessdate=2017-04-08}}</ref>。被质疑的正午阳光影业通过官方微博发布声明，称“某博主黑《人民的名义》后被国家机关调查”事件与该影业无关，“甚至并不清楚上述事件是否客观存在”<ref>{{cite news|url=http://ent.sina.com.cn/v/m/2017-04-03/doc-ifycwymx3578075.shtml|title=被疑诽谤《人民的名义》遭调查 正午阳光：荒唐|publisher=新浪娱乐|date=2017-04-03|accessdate=2017-04-08}}</ref>。

===遭侵权指控===
2017年11月1日，[[上海市浦东新区人民法院|上海市浦东新区人民法院]]受理[[刘三田|刘三田]]（笔名南嫫）起诉《人民的名义》侵权案，请求原告向周梅森及制片单位等八被告索赔1800万元，并要求停止电视剧的一切播出、复制、发行、信息网络传播以及小说出版。刘三田指，《人民的名义》侵犯了其原创长篇小说《暗箱》的著作权，无论是核心事件、叙事结构都高度近似。而《人民的名义》作者周梅森则否认了侵权指控，并表示“严重损害了本人的名誉权”。事发后，网友表示支持周梅森，同时指责刘三田炒作<ref>{{cite news|url=http://www.sohu.com/a/202380727_148781|title=《人民的名义》遭抄袭指控？周梅森坚决否认 |publisher=北京青年报|date=2017-11-05|accessdate=2017-11-05}}</ref>。
