'''人性'''（英文：Humanity），一般指人的本性。作为[[人|人]]应有的正面、积极的[[品性|品性]]，比如慈爱、善良，包含所有正义性的[[人类|人类]][[价值观|价值观]]。以人为本，也包括对“人”个体的尊重，无论国家功能机器还是社会资源对于“人”的不可凌驾性，不走[[極端民族主義|極端]]路線。通常所说的“人性”，也指[[同情论|同情心]]與[[同理心|同理心]]，是隶属于人性这一定义中的[[分支|分支]]。但人性中的不受歡迎的性質，應該也屬於人性的一部分卻多被刻意怱略掉，這能看出人普遍的良好[[期望|期望]]。

关于人性的问题，一直是学术上争论不休的一个话题，目前没有一个确切的结论。但是大部分人相信人性的存在，相信这是一种人类的特有标签。区别于“[[人格|人格]]”。

人性，作為人類特性解釋者，很有可能需要一並研究心和腦的功用，即[[思想|思想]]和[[心態|心態]]等範疇中的普遍存在，又歷久不衰的特質特性。

== 现代社会学定义 ==
现代[[社会学家|社会学家]]则认为世界上只有一个[[人类|人类]]，所以只有一种人性。这是不同民族之间能够交流、达成理解的[[前提|前提]]。但根据对婴儿反复实验，显示人性是[[进化|进化]]的结果，与生俱来的有：由于祖先生存的险恶环境衍生的对生存需要，对胜利的[[渴望|渴望]]，以及对与自己相似伴侣的渴望，对同类间自己地位的关心，由于始前人类的习惯和同理心对还有同类帮助的[[冲动|冲动]]。从人脑的结构来看这些都直接对[[前额叶|前额叶]]造成影响，但可以被出生后的记忆和周围事物所改变。
另有說；個別受後天影響的「人性」應分開稱作「個人人性」（[[個性|個性]]），或這人或這群、族、片土地上的等等人的人性特質，以作有別於「人類人性」（方便[[哲學|哲學]]或[[人類學|人類學]]等課題上的研究總稱人性human nature之用）的解釋。

== 中国文化的定义 ==
在[[中国文化|中国文化]]中，对人的本性，大致有四种观点：

# [[性善论|人性本善论]]，以[[儒家|儒家]][[孟子|孟子]]为代表。《[[三字经|三字经]]》开篇即为“人之初，性本善，性相近，习相远”。
# [[性恶论|人性本恶论]]，以[[儒家|儒家]][[荀子|荀子]]为代表，有“人之性恶，其善伪也”的论断。
# 人性无善无恶论，代表人物为[[告子|告子]]，他认为人性无善无不善，“生之谓性”，“食色，性也”<ref>{{cite book|author=[[孟子|孟轲]]|title=《[[孟子_(著作)|孟子]]·告子》|quote=食色性也。仁，内也，非外也。义，外也，非内也。}}</ref>。
# 人性既善又恶，代表人物是[[西汉|西汉]]的[[扬雄|扬雄]]，他认为“人之性也，善恶混。修其善则为善人，修其恶则为恶人”<ref>{{cite book|author=[[扬雄|扬雄]]|title=《法言·卷三·修身》}}</ref>。

== 其他的定义 ==
[[马克思主义|马克思主义]]虽承认人性的存在，但否认存在普遍抽象的人性：只有从人的社会性和阶级性出发，才能得出对人性的正确解释，并由此断言，在阶级社会中没有超阶级的人性。所以无产阶级有一个人性，资产阶级有另一个人性。这两种人性是不可调和的，只能通过斗争解决。

（西方谚语）：正因为人性本恶，所以民主是必须的；正因为人性向善，所以民主是可能的。

== 中国共产党的评论 ==
 	
中共党校刊物《[[学习时报|学习时报]]》刊文《[http://blog.sciencenet.cn/home.php?mod=attachment&filename=%B9%D8%D3%DA_%B5%B3%D0%D4_%D3%EB_%C8%CB%D0%D4_%D6%AE%B1%E6_%D1%CF%B7%A2%CF%E9.pdf 关于“党性”与“人性”之辨]》称，中国共产党的“党性”是人性的优化、升华和结晶。文章指出“党性”与“人性”又有不同品质。共产党的党性特质是人类社会发展的引领、代表，为社会先进者所执所遵，与人性属不同层面之物。政党是人组成的，党性不可能超然于世、脱离人性。党性植根于人性之壤，又是人性之优化、升华和结晶。<ref>{{cite news|title=党校刊物谈“党性”与“人性”引网民嘲讽|url=http://www.bbc.com/zhongwen/simp/china/2013/08/130819_party_school_on_dangxing|accessdate=2017-07-27|agency=BBC|date=2013-08-19}}</ref>
 
河南内乡县人民法院的宣传标语引发广大的争议，标语称“论党性不论人性，论规矩不论初犯，论主观不论客观，论业绩不论原由”。该院院长成延洲回应称，人性包含许多丑恶的东西，如“好色、贪婪、淫荡”等，“而党性却是人性中光辉一面的结晶体”，所以“当党性和人性重合时，既论党性又论人性；当党性和人性并列时，先论党性后论人性；当党性和人性冲突时，我们坚决只论党性不论人性”。<ref>{{cite news|author1=于盟童|title=中国法院院长：党性人性冲突时坚持党性|url=https://www.voachinese.com/a/chinese-judge-says-party-princlple-before-human-nature-20160516/3332419.html|accessdate=2017-07-27|agency=美国之音|date=2016-05-17}}</ref>
