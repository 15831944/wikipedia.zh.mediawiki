《'''人民政协报'''》是[[中国人民政治协商会议全国委员会|中国人民政治协商会议全国委员会]]的[[机关报|机关报]]，是中国人民政治协商会议全国委员会主办主管的全国性的[[统一战线|统一战线]]报纸<ref name=zhaopin/>。'''人民政协报社'''<ref>{{Cite web |url = http://www.gjsy.gov.cn/cxzl/ |title =人民政协报社  |author =  |date =  |publisher =事业单位在线  |language =  |accessdate =2018-02-11  |quote =  }}</ref>是[[全国政协办公厅|全国政协办公厅]]直属[[事业单位|事业单位]]。

==简介==
1982年3月20日，在全国政协副秘书长、学委会副主任[[聂真|聂真]]主持下，全国政协学委会办公室起草了《关于创办〈政协导报〉的意见》，《政协导报》是最初提出的报名之一，另有《人民政协报》、《人民论坛报》、《民主协商报》等。4月9日，全国政协学委会办公室邀请政协会刊编委会委员以及各[[民主党派|民主党派]]宣传部门负责人座谈。4月14日，[[全国政协机关党组|全国政协机关党组]]邀请各民主党派报刊负责人座谈。在这两次座谈会的基础上，确定报名为《人民政协报》。9月16日，全国政协机关党组向[[中共中央统战部|中共中央统战部]]、[[中共中央书记处|中共中央书记处]]提出《关于创办〈人民政协报〉的请求报告》。11月9日，《人民政协报》试刊号第一期出版。随后又出版第二期试刊，内部发行。11月24日，全国政协五届五次会议在《政协常委会工作报告》中宣布创办《人民政协报》的决定。12月18日，中共中央办公厅对请示报告答复称：“全国政协机关党组关于创办《人民政协报》的请示，已经中央书记处批准。”并称“《人民政协报》主要任务是坚持[[四项基本原则|四项基本原则]]，执行党的统一战线政策，努力把报纸办好”<ref>{{Cite web |url =http://www.rmzxb.com.cn/ztlb/rmzxb30/dsj30/t20130329_491422.htm  |title =一九八二年  |author =  |date = 2013-04-01 |publisher = 人民政协网 |language =  |accessdate =  |quote =  }}</ref>。

1983年1月10日，全国政协机关党组提出关于出版《人民政协报》的几点意见。1月12日，全国政协机关党组就出版该报的问题向[[中共中央宣传部|中共中央宣传部]]、中共中央统战部提出报告。3月9日，全国政协秘书长[[彭友今|彭友今]]、副秘书长[[孙起孟|孙起孟]]致信[[全国政协主席|全国政协主席]][[邓小平|邓小平]]，请他题写报名<ref name=yjbsan>{{Cite web |url =http://www.rmzxb.com.cn/ztlb/rmzxb30/dsj30/t20130329_491421.htm  |title =一九八三年  |author =  |date = 2013-04-01 |publisher = 人民政协网 |language =  |accessdate =  |quote =  }}</ref>。

1983年4月6日，《人民政协报》创刊。对开四版，每周三出版一期，首期发行量3.5万多份。报名由邓小平题写，[[党和国家领导人|党和国家领导人]]以及全国政协领导[[陆定一|陆定一]]、[[许德珩|许德珩]]、[[胡厥文|胡厥文]]、[[史良|史良]]、[[王昆仑|王昆仑]]、[[胡愈之|胡愈之]]等为创刊题贺词。按照中共中央“《人民政协报》成立[[党组|党组]]”的指示，该报成立了由[[萨空了|萨空了]]、[[徐盈|徐盈]]、[[徐亦安|徐亦安]]组成的党组，接受全国政协机关党组领导。报社还成立了萨空了、徐盈、[[王揖|王揖]]、[[余焕春|余焕春]]、[[沈求我|沈求我]]、[[叶笃义|叶笃义]]、[[虞效忠|虞效忠]]、[[黄森|黄森]]、[[徐亦安|徐亦安]]、[[张西洛|张西洛]]组成的编辑委员会。总编辑萨空了，副总编辑徐亦安、张西洛<ref name=yjbsan/>。

1997年1月1日，该报由周三刊改为周六刊，成为日报。[[全国政协副主席|全国政协副主席]][[何鲁丽|何鲁丽]]致信祝改日报成功。[[叶选平|叶选平]]、[[杨汝岱|杨汝岱]]、[[王兆国|王兆国]]、[[钱伟长|钱伟长]]、[[孙孚凌|孙孚凌]]为改日报题词祝贺<ref name=yjjq>{{Cite web |url =  http://www.rmzxb.com.cn/ztlb/rmzxb30/dsj30/t20130329_491407.htm|title =一九九七年  |author =  |date = 2013-04-01 |publisher = 人民政协网 |language =  |accessdate =  |quote =  }}</ref>。1999年1月1日，由周六刊改为周七刊<ref>{{Cite web |url = http://www.rmzxb.com.cn/ztlb/rmzxb30/dsj30/t20130329_491405.htm |title =一九九九年  |author =  |date = 2013-04-01 |publisher = 人民政协网  |language =  |accessdate =  |quote =  }}</ref>。2000年1月1日，版面调整为周一至周五对开八版，周六对开四版<ref name=ellling>{{Cite web |url = http://www.rmzxb.com.cn/ztlb/rmzxb30/dsj30/t20130329_491404.htm |title = 二○○○年 |author =  |date = 2013-04-01 |publisher =  人民政协网|language =  |accessdate =  |quote =  }}</ref>。

2000年2月25日，该报发布《人民政协报》可上网阅读的消息，在全国政协网页上新增了《人民政协报》栏目<ref name=ellling/>。2014年3月1日，经全新改版的人民政协网正式上线运行。人民政协网是全国政协主管主办的新媒体，由人民政协报社全资子公司——人民政协报网络有限公司负责运营<ref>{{Cite web |url = http://www.rmzxb.com.cn/aboutus/index.shtml |title =关于我们 |author =  |date =  |publisher = 人民政协网 |language =  |accessdate =  2018-02-08|quote =  }}</ref>。

2002年9月20日，总建筑面积2万平方米的人民政协报大厦在西八里庄路69号落成，全国政协副主席[[宋健|宋健]]、[[陈锦华|陈锦华]]、[[张克辉|张克辉]]，秘书长[[郑万通|郑万通]]出席竣工典礼并剪彩<ref>{{Cite web |url =http://www.rmzxb.com.cn/ztlb/rmzxb30/dsj30/t20130329_491402.htm  |title = 二○○二年 |author =  |date = 2013-04-01 |publisher =  人民政协网 |language =  |accessdate =  |quote =  }}</ref>。2003年1月16日，全国政协主席[[李瑞环|李瑞环]]到报社看望全体员工。2004年起，该报扩版出彩报<ref name=ellsan/>。2005年12月30日，人民政协报社社委会成立并举行第一次会议<ref name=ellw/>。

《人民政协报》坚持“立足统战，面向社会”的方针，为各级政协组织履行职能及政协委员参政议政服务<ref name=zhaopin>{{Cite web |url =  http://epaper.rmzxb.com.cn/detail.aspx?id=339331 |title = 人民政协报社招聘公告 |author =  |date =  2014-05-14|publisher =人民政协网  |language =  |accessdate =  |quote =  }}</ref>。

2013年4月3日，[[中共中央政治局常委|中共中央政治局常委]]、[[全国政协主席|全国政协主席]][[俞正声|俞正声]]到人民政协报社调研，提出“把《人民政协报》办成一份委员想看、爱看，开卷有益的大报”<ref name=zhuhe>{{Cite web|url = http://www.chinadaily.com.cn/hqgj/jryw/2013-04-09/content_8709107.html|title = 热烈祝贺《人民政协报》创刊三十周年|author = |date = 2013-04-09|publisher = 中国日报|language = |accessdate = |quote = }}{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
