《'''人生一串'''》，由[[哔哩哔哩|哔哩哔哩]]和[[华扬联众|旗帜传媒]]联合出品的一部[[中式烤串|中国市井烧烤文化]]主题美食纪录片。该片于2018年6月20日开始在哔哩哔哩全网獨家播映，每周三晚20点更新，共6集，每集30分钟。

== 拍摄制作 ==
据总导演陈英杰介绍，剧组在2016年开始策划，2017年年初开始田野调查，3月底开始拍摄，6月份主要拍摄结束，经过补拍、后期剪辑、修改后，到2018年6月才得以播出<ref>{{cite web |author1=刘璐 |author2=冯万艺 |title=《人生一串》收官！导演却说和世界杯同步“其实并没有太大把握” |url=https://www.jfdaily.com/news/detail?id=96908 |website=上观新闻 |accessdate=2018-07-26 |archive-url=https://web.archive.org/web/20180726063159/https://www.jfdaily.com/news/detail?id=96908 |archive-date=2018-07-26 |dead-url=no }}</ref>。第二集的主题是暗黑系美食，由于内容猎奇曾差点被摄制组摒弃，后来陈英杰发现在一般人眼中罕见的的食材其实正是当地人真实的日常，之所以会有人觉得食材黑暗，某种程度上却源于一种生活经历的欠缺<ref>{{cite web |title=接地气的一串 才是中国的深夜食堂 |url=http://www.jhnews.com.cn/2018/0724/819892.shtml |website=金华新闻网 |accessdate=2018-07-26 |archive-url=https://web.archive.org/web/20180726063158/http://www.jhnews.com.cn/2018/0724/819892.shtml |archive-date=2018-07-26 |dead-url=no }}</ref>。

《人生一串》共涉及30余家烧烤店铺，据称是由导演组从32个省市近500家烧烤店铺中挑选出来的<ref>{{cite web |title=能把人看饿的节目 了解一下 |url=http://www.prcfe.com/news/2018/0724/268647.html |website=中国财经新闻网-云南信息报 |accessdate=2018-07-26 |archive-url=https://web.archive.org/web/20180726063157/http://www.prcfe.com/news/2018/0724/268647.html |archive-date=2018-07-26 |dead-url=no }}</ref>。

== 音乐 ==
片尾曲《如去年一样伤悲》由[[旧北乐队|旧北乐队]]演唱，旧北乐队成员在第三集《来点解药》中有出镜<ref name="yichang"/>。

== 评论 ==
《杭州日报》评论该片文案逗趣，解说词[[中二|中二]][[音MAD|鬼畜]]，配音也未使用播音腔，而是选择了满是市井气息的大烟嗓，这些就像烧烤配[[孜然|孜然]]一样刚刚好<ref>{{cite web |title=烟火气中撸起人生一串 诞生于网络的美食纪录片为何大受年轻人欢迎？ |url=http://zjnews.zjol.com.cn/zjnews/hznews/201807/t20180726_7866336.shtml |website=杭州日报 |accessdate=2018-07-26 |archive-url=https://web.archive.org/web/20180726063159/http://zjnews.zjol.com.cn/zjnews/hznews/201807/t20180726_7866336.shtml |archive-date=2018-07-26 |dead-url=no }}</ref>。荔枝娱评称该片镜头简单直接，直接对准美食，即便在屏幕前都可以感到扑鼻的香气<ref>{{cite web |title=《人生一串》：走，撸串去！ |url=http://news.jstv.com/a/20180722/1532225171158.shtml |website=荔枝网 |accessdate=2018-07-26 |archive-url=https://web.archive.org/web/20180722063436/http://news.jstv.com/a/20180722/1532225171158.shtml |archive-date=2018-07-22 |dead-url=no }}</ref>。

截至2018年8月5日，bilibili站有超过1.2万人评价，评分为9.8分；豆瓣网超过2.8万人评价，评分为9.0分<ref>{{cite web |title=看完9.8分的《人生一串》，你得需要这份“重庆烧烤地图” |url=http://www.cqcb.com/belle/hot/2018-07-24/980259.html |website=渝报 |accessdate=2018-07-26 |archive-url=https://web.archive.org/web/20180726063158/http://www.cqcb.com/belle/hot/2018-07-24/980259.html |archive-date=2018-07-26 |dead-url=no }}</ref>。
