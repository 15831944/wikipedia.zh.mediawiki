《'''人鱼帝国'''》（{{lang-en|'''Empires of the Deep'''}}）是中国大陆和美国联合出品的[[3D|3D]][[魔幻片|魔幻片]]、[[奇幻片|奇幻片]]、探险片，导演厄文·克什纳，主演欧嘉·科瑞兰蔻、史蒂夫·波利茨、施艳飞、让·米歇尔·卡萨诺瓦。<ref>{{cite web |url=http://ent.163.com/13/0422/01/8T1EIHCT000300B1.html |title=《人鱼帝国》将映 3D视效大师揭秘魔幻海底世界 |accessdate=2013 |publisher=网易 |work= |language=zh-hans}}</ref>该片曾说会在2013年上映，但是没有<ref name=" " >{{cite web |url=http://movie.mtime.com/81176/reviews/1383730.html |title=《人鱼帝国 Empires of the Deep》首曝正式预告海报+伪预告片 |accessdate=2013 |publisher=时光网 |work= |language=zh-hans}}</ref>。人鱼帝国經過8個劇作家、構築40個版本的草稿，並請來參演過[[殺手47|殺手47]]、[[007大破量子危機|007大破量子危機]]等[[好萊塢|好萊塢]]大片的烏克蘭超模兼女星[[欧嘉·柯瑞兰寇|欧嘉·柯瑞兰寇]]演出，但這部片在電影節放映時被評為中國影史上「最貴又最難看」的電影，至今仍沒有片商願意發行<ref>[https://asia.nikkei.com/magazine/20140814-The-odd-couple/Cover-Story/Hollywood-dances-to-Beijings-beat Hollywood dances to Beijing's beat - Nikkei Asian Review]</ref>。

==剧情==
[[古希腊|古希腊]]时期，男孩阿拉特斯在成长期间被神秘力量煎熬，他必须前往海洋深处的人鱼帝国救出自己的父亲，在这段探险过程中，他战胜了人鱼、海盗和怪兽，最终也战胜了自己。

==攝製過程==
该片一共花费了7年时间来制作，投资达到了1.3亿元<ref>{{cite web |url=http://www.chinanews.com/yl/2012/10-26/4280097.shtml |title="人鱼帝国"烂尾2年导演换3茬 纯国货硬变"混血" |accessdate=2012 |publisher=中新网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.ifeng.com/HOT/yaowen/detail_2013_04/23/24554968_0.shtml |title=《人鱼帝国》七年磨一剑 |accessdate=2013 |publisher=凤凰网 |work= |language=zh-hans}}</ref>，背后的最大出資者是中国[[房地产|房地产]]大亨[[姜红宇|姜红宇]]，本人熱愛电影，過去從未有電影製片经验，但他声称自己看过4,000部电影，想制作一部「非常奇幻的爱情悲剧」，影片的剧本也由姜红宇亲自执笔，但其后在10个好莱坞编剧帮助下修改了40多遍。《人魚帝國》早在2006年就宣佈籌拍，直到2010年初才正式開機。《人魚帝國》號稱邀請了《[[阿凡達|阿凡達]]》後期[[特效|特效]]團隊，拍攝期間拖欠演職員工資、更換導演等問題接連發生。一個法國導演、兩個美國導演先後離開劇組，之後由第四位導演完成拍攝。[[2012年|2012年]]10月，影片[[預告|預告]]片首度曝光，但反應不佳<ref>[http://ent.sina.com.cn/m/f/2012-10-25/13193771941.shtml 《人鱼帝国》：“烂尾楼”折腾成“伪混血”|《人鱼帝国》_影音娱乐_新浪网]</ref>。。

该片曾在[[美国|美国]]、[[泰国|泰国]]、[[南中国海|南中国海]]等地取景<ref>{{cite web |url=http://ent.163.com/13/0424/01/8T6JDMM500034C67.html |title=《人鱼帝国》亮相北京电影节 3D大师为雅安祈福 |accessdate=2013 |publisher=网易 |work= |language=zh-hans}}</ref>。2013年4月，该片展现在[[北京电影节|北京电影节]]<ref>{{cite web |url=http://news.china.com.cn/rollnews/ent/live/2013-04/23/content_19648606.htm |title= 《人鱼帝国》亮相北京电影节 暑期档与观众见面|accessdate=2013 |publisher=中国网 |work= |language=zh-hans}}</ref>，但正式在[[院線|院線]]公映的時間迄今仍遙遙無期。
