《'''人在江湖：尊敬之路'''》（{{lang-en|'''''The Sopranos: Road to Respect'''''}}）是一款2006年的電子遊戲，由美國開發商7 Studios負責開發，遊戲改編自[[HBO|HBO]]電視劇《[[人在江湖_(美國電視劇)|人在江湖]]》（1999年-2007年）。遊戲於2006年11月7日發售。<ref>{{Cite web |url = https://www.ign.com/articles/2006/11/22/the-sopranos-road-to-respect-review |title = THE SOPRANOS: ROAD TO RESPECT REVIEW |date = 2006-11-21 |publisher = IGN |language = en}}</ref>

故事線發生在電視劇第五季和第六季之間，集中講述{{link-en|Big Pussy Bonpensiero|Big Pussy Bonpensiero}}的私生子Joey LaRocca。<ref>{{Cite web |url = https://www.giantbomb.com/the-sopranos-road-to-respect/3030-8694/ |title = The Sopranos: Road to Respect |publisher = Giant Bomb |language = en}}</ref>
