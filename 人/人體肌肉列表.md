== 頭部和頸部的肌肉 ==
[[頭|頭]]和頸部的[[肌肉|肌肉]]在人體的肌肉中發揮著重要的作用，它們支撐著人體的[[頭部|頭部]]，保護著人體重要的[[感覺器官|感覺器官]]。人體頭部的肌肉能夠使人做出各種臉部表情。人體頸部的肌肉有很多條，它們支撐著人體的頭部，並使頭部做出各種動作。

===頭部的肌肉===
頭部的肌肉可以分為[[面肌|面肌]]和[[咀嚼肌|咀嚼肌]]。其中面肌為扁又薄的皮肌，位置表淺，大多起自於[[頭顱|頭顱]]的不同部位，止於臉部[[皮膚|皮膚]]。人類的面肌較其他動物發達，這與人類[[大腦|大腦]][[皮質|皮質]]的高度進化、思維和語言活動有關。人類的咀嚼肌主要包括[[顳肌|顳肌]]、[[咬肌|咬肌]]……等肌肉，它們分布於[[下頜關節|下頜關節]]周圍，收縮時使[[下頜骨|下頜骨]]運動，參與[[咀嚼|咀嚼]]。

===頸部的肌肉===
[[頸部|頸部]]位於[[頭部|頭部]]、[[胸部|胸部]]和[[上肢|上肢]]之間。頸部的[[肌肉|肌肉]]主要由前方的[[舌骨下肌群|舌骨下肌群]]，外側的[[胸鎖乳突肌|胸鎖乳突肌]]，和後方的[[斜角肌群|斜角肌群]]組成。其中胸鎖乳突肌位於頸部兩側，當胸鎖乳突肌一側收縮時，它能夠使頭部左右擺動，當胸鎖乳突肌兩側同時收縮時，能夠使人的頭部後仰。[[斜角肌群|斜角肌群]]位於[[脊柱|脊柱]]頸段的兩側，包括[[前斜角肌|前斜角肌]]、[[中斜角肌|中斜角肌]]和[[後斜角肌|後斜角肌]]。

== 軀幹的肌肉 ==
[[軀幹|軀幹]]是連結[[頭|頭]][[頸|頸]]、[[上肢|上肢]]和[[下肢|下肢]]的部分，分為上方的[[胸部|胸部]]和下方的[[腹部|腹部]]。軀幹前面的淺層[[肌肉|肌肉]]主要是[[胸大肌|胸大肌]]和[[腹外斜肌|腹外斜肌]]，[[胸大肌|胸大肌]]<ref group="注">[[胸大肌|胸大肌]]位於[[軀幹|軀幹]][[胸部|胸部]]的位置，正確來講胸大肌應該被列為[[軀幹|軀幹]]的肌肉。可是用[[解剖學|解剖學]]的角度來講，胸大肌的尾端附著在[[肩膀|肩膀]]的[[鎖骨|鎖骨]]，胸大肌的腹內脊肌還延伸到[[手臂|手臂]]的[[肱骨|肱骨]]，所以在醫學上來說，[[胸大肌|胸大肌]]是位於[[上肢|上肢]]的肌肉，而不是[[軀幹|軀幹]]的肌肉。參考資料：[http://www.jirou.com/jiroulilun/lixue/2015/0910/10226.html 胸大肌的正確位置]</ref>是一塊大而薄的肌肉，它可以使[[手臂|手臂]]向前擺動、旋內和彎曲，另外它還可以在人們吊單槓時，幫助人們把身體向上拉。軀幹前面的深層[[肌肉|肌肉]]主要是[[肋間外肌|肋間外肌]]和[[腹直肌|腹直肌]]。軀幹背部的淺層肌肉主要是[[斜方肌|斜方肌]]和[[背闊肌|背闊肌]]，[[斜方肌|斜方肌]]呈三角形，其左右兩側相合成為斜方形，斜方肌的收縮能促使[[肩胛骨|肩胛骨]]向[[脊柱|脊柱]]靠近，另外斜方肌能夠向後拉伸[[頭|頭]]和[[肩膀|肩膀]]，使肩膀保持平穩。[[背闊肌|背闊肌]]是全身最大的[[扁闊肌|扁闊肌]]，它可以使[[肱骨|肱骨]]內收、旋內和後伸的作用，當人們做引體向上運動時，[[背闊肌|背闊肌]]可以幫助我們提升[[軀幹|軀幹]]。軀幹背部的深層肌肉主要是[[小菱形肌|小菱形肌]]和[[大菱形肌|大菱形肌]]，[[菱形肌|菱形肌]]呈菱形扁平狀，它包括小菱形肌和大菱形肌，菱形肌可以使肩胛骨向脊柱靠近並稍微向上提。 

===軀幹前面的肌肉===
[[肋骨|肋骨]]圍成的籠狀[[胸廓|胸廓]]構成了軀幹上部的[[支架|支架]]。[[腹壁|腹壁]]則僅由扁平的[[腹肌|腹肌]]形成，腹肌的[[肌腱|肌腱]]在腹部正中央融合形成[[腹白線|腹白線]]。

===軀幹背部的肌肉===
背部的淺層[[肌肉|肌肉]]可以帶動人體的[[手臂|手臂]]和[[肩膀|肩膀]]的運動。背部深層的[[豎脊肌|豎脊肌]]兩側收縮時可以拉伸[[脊柱|脊柱]]、抗重力和維持平衡，如果只有單側[[豎脊肌|豎脊肌]]收縮，就可以使人體的脊柱彎曲。[[豎脊肌|豎脊肌]]為[[背部|背部]]的肌肉中最長最大的肌肉，其作用是使脊柱後伸和仰頭，它對維持人體的直立姿勢起著重要的作用。

===腹壁===
[[腹外斜肌|腹外斜肌]]：覆盖腹部两侧最外层的[[肌肉|肌肉]]，它扁平而宽，呈不规则的四边形。从下面八根[[肋骨|肋骨]]起，斜下向前附着在[[髂骨|髂骨]]的前外嵴和穿过腹直肌鞘到达白线。

[[腹内斜肌|腹内斜肌]]：呈三角形，并且比它上面的腹外斜肌小而薄。它起自{{link-en|腹股溝韌帶|Inguinal ligament}}和[[髂骨|髂骨]]前内嵴。它的下三分之二和腹外斜肌以及内面的腹横肌纤维附着于白线。上三分之一则终于最下面的六根肋骨。

[[腹横肌|腹横肌]]：扁平成三角形。其纤维水平走向。位于腹内斜肌和下层的横筋膜之间。起于Poupart韧带，髂骨内唇，腰筋膜和下六根肋骨软骨部分的内侧。终于白线，在腹直肌后面。

[[腹直肌|腹直肌]]：扁平而长。它们起于[[耻骨|耻骨]]，在白线的两侧通过腹部向上附着于第5，6和7肋骨的软骨部分。该肌肉被三腱划通过。腹直肌为上面提到的侧腹壁的三种肌肉的纤维形成的厚实鞘所包绕。

[[錐肌|錐肌]]：呈小的三角形，位于下腹部腹直肌的前方。它位于下腹，在腹直肌的前面。它起于耻骨，终于白线，途中经过[[脐|脐]] 。

腹膜腔的後壁內，有著[[髂肌|髂肌]]（或稱胯肌，{{lang|en|iliacus muscle}}）、[[腰方肌|腰方肌]]（{{lang|en|quadratus lumborum muscle}}）、[[腰大肌|腰大肌]]與[[腰小肌|腰小肌]]等。髂肌與腰大肌統稱{{link-en|髂腰肌|Iliopsoas}}（{{lang|en|iliopsoas muscle}}）。

== 上肢的肌肉 ==
[[上肢|上肢]]的肌肉可以分為[[手臂|手臂]]的肌肉和[[手部|手部]]的肌肉。其中手臂的肌肉可以分為上臂的肌肉和前臂的肌肉。上臂的肌肉以內側肌和外側肌間隔為界，分前、後兩群。前臂的肌肉分布於[[尺骨|尺骨]]和[[橈骨|橈骨]]的周圍，分前、後兩群，數目眾多，大多為[[長肌|長肌]]，其中肌腹多集中在前臂的上半部，下半部則形成細長的[[肌腱|肌腱]]。手部的肌肉皆為短小的肌肉，全部都集中於手的掌面，能讓[[手指|手指]]運動，完成精細的技巧性動作，可以分為外側群、中間群和內側群。

===上臂的肌肉===
上臂的肌肉可以分為前、後兩群。上臂肌前群的淺層的肌肉主要是[[肱二頭肌|肱二頭肌]]，[[肱二頭肌|肱二頭肌]]屬於[[長肌|長肌]]，呈梭形，它的起端有兩個頭，因此而得名，肱二頭肌可以使[[肘關節|肘關節]]屈伸，此外肱二頭肌還可以協助[[肩關節|肩關節]]屈伸。深層的肌肉主要是[[肱肌|肱肌]]，[[肱肌|肱肌]]位於肱二頭肌的深面，它可以使肘關節屈伸，從而進一步促使[[前臂|前臂]]屈伸。上臂肌後群的肌肉主要是[[肱三頭肌|肱三頭肌]]，[[肱三頭肌|肱三頭肌]]的作用是使[[肘關節|肘關節]]伸展。

===前臂肌前群===
[[手臂|手臂]]的肌肉可以分為上臂的肌肉和[[前臂|前臂]]的肌肉。前臂的肌肉可以分為前、後兩群。其中前臂的前群肌肉為屈肌群，位於[[前臂|前臂]]的前面，總共有9塊，分為淺、深兩層。淺層的肌肉有[[肱橈肌|肱橈肌]]、[[旋前圓肌|旋前圓肌]]、[[橈側腕屈肌|橈側腕屈肌]]、[[掌長肌|掌長肌]]、[[指淺屈肌|指淺屈肌]]和[[尺側腕屈肌|尺側腕屈肌]]。深層的肌肉有[[拇長屈肌|拇長屈肌]]、[[指深屈肌|指深屈肌]]和[[旋前方肌|旋前方肌]]。前臂肌前群的肌肉可以讓手臂屈肘、屈腕、屈指和前臂旋前……等功能。

===前臂肌後群===
前臂的後群[[肌肉|肌肉]]為伸肌群，位於[[前臂|前臂]]的後面，總共有10塊，也分為淺、深兩層。淺層的肌肉有[[橈側腕長伸肌|橈側腕長伸肌]]、[[橈側腕短伸肌|橈側腕短伸肌]]、[[指伸肌|指伸肌]]、[[小指伸肌|小指伸肌]]和[[尺側腕伸肌|尺側腕伸肌]]。深層的肌肉有[[旋後肌|旋後肌]]、[[拇長展肌|拇長展肌]]、[[拇短伸肌|拇短伸肌]]、[[拇長伸肌|拇長伸肌]]和[[食指伸肌|食指伸肌]]。前臂肌後群的肌肉可以讓手臂伸肘、伸腕、伸指和前臂旋後……等功能。

===手部的肌肉===
[[手部|手部]]的[[肌肉|肌肉]]都十分短小，其功能是使手部運動。手部的肌肉可以分為外側、內側和中間。內側的肌肉主要有[[拇短展肌|拇短展肌]]、[[拇短屈肌|拇短屈肌]]、[[小指展肌|小指展肌]]和[[小指短屈肌|小指短屈肌]]……等。外側和中間的肌肉主要有[[拇指對掌肌|拇指對掌肌]]、[[拇收肌|拇收肌]]、[[小指對掌肌|小指對掌肌]]、[[指間掌側肌|指間掌側肌]]和[[骨間背側肌|骨間背側肌]]。這些肌肉可以使手掌和[[手指|手指]]的[[關節|關節]]彎曲和伸展。

== 下肢的肌肉 ==
[[人體|人體]][[下肢|下肢]]的肌肉主要包括[[腿部|腿部]]的肌肉和[[腳部|腳部]]的肌肉。其中人體腿部的肌肉能完成行走、跑動和爬行……等各種動作，而人體腳部的肌肉則主要起著維持[[足弓|足弓]]和行走的作用。腿部的肌肉可以分為[[髖肌|髖肌]]、[[大腿肌|大腿肌]]和[[小腿肌|小腿肌]]。腳部的肌肉可以分為[[足背肌|足背肌]]和[[足底肌|足底肌]]。

===髖肌前群===
[[髖肌|髖肌]]多起使於[[骨盆|骨盆]]內面或外面，跨越[[髖關節|髖關節]]，止於[[股骨|股骨]]，分為前、後兩群。髖肌前群主要包括[[髂腰肌|髂腰肌]]和[[闊筋膜張肌|闊筋膜張肌]]。[[髂腰肌|髂腰肌]]由[[髂肌|髂肌]]和[[腰大肌|腰大肌]]合併而成，它可以使髖關節前屈和旋外的作用。當人們做[[仰臥起坐|仰臥起坐]][[下肢|下肢]]固定時，髂腰肌可以使[[人體|人體]]的[[軀幹|軀幹]]和[[骨盆|骨盆]]前屈。

===大腿肌前群===
[[大腿肌|大腿肌]]分布於[[股骨|股骨]]周圍，分為前、後和內側3群。其中大腿肌前群的肌肉有[[縫匠肌|縫匠肌]]和[[股四頭肌|股四頭肌]]。[[縫匠肌|縫匠肌]]是全身最大的肌肉，它不僅能使[[大腿|大腿]]旋轉，還能使大腿屈伸。[[股四頭肌|股四頭肌]]是全身[[體積|體積]]最大的肌肉，它能使人在跑步、攀登和踢動時讓[[膝蓋|膝蓋]]伸直。事實上，股四頭肌並非只是一塊肌肉，它是由[[股直肌|股直肌]]、[[股內側肌|股內側肌]]、[[股外側肌|股外側肌]]和[[股中間肌|股中間肌]]組成。

===小腿肌前群===
[[小腿肌|小腿肌]]與[[上肢|上肢]]的[[前臂|前臂]]的肌肉比較，粗大但是數目少，分為前、後和外側3群，它參與維持人體的直立姿勢和行走。小腿肌前群主要包括[[脛骨前肌|脛骨前肌]]、[[拇長伸肌|拇長伸肌]]和[[趾長伸肌|趾長伸肌]]。[[脛骨前肌|脛骨前肌]]能產生足內翻，[[拇長伸肌|拇長伸肌]]和[[趾長伸肌|趾長伸肌]]能使[[腳趾|腳趾]]屈伸，此外[[趾長伸肌|趾長伸肌]]還能產生屈踝運動。

===髖肌後群===
髖肌後群位於[[臀部|臀部]]，又稱為臀肌，主要包括[[臀大肌|臀大肌]]、[[臀中肌|臀中肌]]、[[臀小肌|臀小肌]]、[[梨狀肌|梨狀肌]]……等[[肌肉|肌肉]]。這些豐滿的肌肉隆起形成臀部，起著支撐[[人體|人體]]的作用。[[臀大肌|臀大肌]]位於臀部淺層，其作用是使[[髖關節|髖關節]]旋外，同時它還能伸直[[軀幹|軀幹]]，防止軀幹前傾，以維持[[身體|身體]]的平衡。[[臀中肌|臀中肌]]位於[[臀大肌|臀大肌]]的深面，其作用是使[[髖關節|髖關節]]外展。[[梨狀肌|梨狀肌]]的作用是使髖關節旋外。

===大腿肌後群和大腿肌內側群===
大腿肌後群的肌肉包括[[股二頭肌|股二頭肌]]、[[半腱肌|半腱肌]]和[[半膜肌|半膜肌]]，這些肌肉可以使[[大腿|大腿]]和[[小腿|小腿]]彎曲，而且還可以使已經彎曲的小腿內旋。大腿肌內側群的肌肉位於大腿的內側，總共有5塊，為[[恥骨肌|恥骨肌]]、[[內收長肌|內收長肌]]、[[內收短肌|內收短肌]]、[[內收大肌|內收大肌]]和[[股薄肌|股薄肌]]，這些肌肉可以使大腿向外旋和向內收。

===小腿肌後群和小腿肌外側群===
小腿肌後群位於[[脛骨|脛骨]]、[[腓骨|腓骨]]及[[骨間膜|骨間膜]]的後面，可以分為淺、深兩層。淺層的肌肉主要有[[腓腸肌|腓腸肌]]和[[比目魚肌|比目魚肌]]，[[腓腸肌|腓腸肌]]的功能是彎曲、伸直和扭轉腳跟，當我們坐著並且輕拍[[小腿|小腿]]時，就可以看到這個部位的肌肉在晃動。[[比目魚肌|比目魚肌]]聯結小腿的肌肉和足骨，其功能是強化[[腳踝|腳踝]]，以防止身體向前傾倒。深層的肌肉主要有[[趾長屈肌|趾長屈肌]]、[[脛骨後肌|脛骨後肌]]和[[拇長屈肌|拇長屈肌]]，[[趾長屈肌|趾長屈肌]]和[[拇長屈肌|拇長屈肌]]的作用是使[[腳趾|腳趾]]跖屈。小腿肌外側群包括[[腓骨長肌|腓骨長肌]]和[[腓骨短肌|腓骨短肌]]，[[腓骨短肌|腓骨短肌]]的作用是使腳心外翻並且協助腳趾跖屈。

===腳部的肌肉===
[[腳部|腳部]]的肌肉分為[[足背肌|足背肌]]和[[足底肌|足底肌]]。其中足背肌比較弱小，主要包括[[趾短伸肌|趾短伸肌]]和[[拇短伸肌|拇短伸肌]]。足底肌與手掌的肌肉相似，也分為內側、外側和中間3群，但是卻沒有[[對掌肌|對掌肌]]。足底肌除了能配合小腿肌運動各個[[足關節|足關節]]外，還有維持[[足弓|足弓]]的作用。
