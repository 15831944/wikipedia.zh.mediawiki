'''宣戰'''是指一個國家宣告該國与其他（可多於一個）國家處於[[戰爭|戰爭]]狀態。

== 背景 ==
[[文藝復興|文藝復興]]時，已有國家作出正式的宣戰聲明，自此，[[民主國家|民主國家]]已接受宣戰作為一個[[外交|外交]]手段。

在[[國際法|國際法]]中，宣戰聲明代表該等國家互相承認處於敵對狀態，並規範各自的[[軍隊|軍隊]]的行為。[[海牙公約|海牙公約]]是影響這些聲明的主要多邊條約。

[[第一次世界大戰|第一次世界大戰]]後，1919年成立的[[國際聯盟|國際聯盟]]，和1928年在巴黎簽訂的[[非戰公約|非戰公約]]，皆顯示世界各國都致力避免大戰造成的浩劫。然而，[[第二次世界大戰|第二次世界大戰]]仍然爆發了，於是各國於戰後成立了[[聯合國機構|聯合國機構]]以防止國家透過宣戰來發動侵略。

== 聯合國與戰爭 ==

為迫使國家放棄以[[戰爭|戰爭]]解決紛爭，[[聯合國憲章|聯合國憲章]]的起草者嘗試限制會員國只能在特定環境下方能發動戰爭，尤其只是為了[[自衞|自衞]]。

但是，1950年6月25日，[[北韓|北韓]]进攻[[南韓|南韓]]後，聯合國卻參與了[[朝鮮戰爭|朝鮮戰爭]]。[[聯合國安全理事會|聯合國安全理事會]]在[[蘇聯|蘇聯]]代表因抗議[[聯合國|聯合國]]拒絕接納[[中華人民共和國|中華人民共和國]]為新成員國而自1950年1月起缺席的情況下，動議以13對1（[[南斯拉夫|南斯拉夫]]投了反對票）的表決結果得到通過。於是[[美國|美國]]和15個國家組成了「聯合國軍」。美國[[總統|總統]][[哈利·S·杜魯門|哈利·S·杜魯門]]在6月29日的[[記者會|記者會]]上，稱這種軍事行為為「{{tsl|en|Police action|警察行動}}」，而非「戰爭」。<ref>[http://teachingamericanhistory.org/library/index.asp?document=594 http://teachingamericanhistory.org/library/index.asp?document=594]</ref>

聯合國通過了一些[[聯合國安全理事會決議|安全理事會決議]]，以宣告一些戰爭為合乎國際法的。其中一個決議為關於1991年[[波斯灣戰爭|波斯灣戰爭]]的[[聯合國安理會687號決議|第687號決議]]。

== 不宣而戰 ==
在大部分的[[民主|民主]]國家，須[[立法部門|立法部門]]通過後方可進行宣戰。[[美國憲法|美國憲法]]只說明「[[美國國會|國會]]有權力……宣告戰爭……」，而沒有提及宣戰聲明的形式，故一些人認為經國會通過並授權的軍事行為是「宣戰聲明」。這個概念並無在美國法律制度中證實。[[榮·保羅|榮·保羅]]和一些人則爭論明確的宣戰聲明實際上是憲法要求。<ref>{{cite web |url=http://www.house.gov/paul/press/press2002/pr100402.htm |title=存档副本 |accessdate=2007-04-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20070406153209/http://www.house.gov/paul/press/press2002/pr100402.htm |archivedate=2007-04-06 }}</ref>

聯合國參與韓戰後，很多民主國家把他們的戰爭行為標籤成其他東西，如「軍事行動」或「武裝回應」，例如在美國參與[[越南戰爭|越南戰爭]]時。[[法國|法國]]和一些國家的軍事制度包括很多[[殖民地|殖民地]]，他們以警衛行動來干涉其前殖民地的事務，因為它們不能被視為「國際紛爭」。

不宣而戰是一種歸避憲法對宣戰者的保護，同時避免受到[[戰爭法|戰爭法]]的規限。不使用「戰爭」字眼似乎更符合[[公共關係|公共關係]]。故此，政府多數不再發表宣戰聲明，改為委婉地稱其行動為「警衛行動」和「獲授權使用軍隊」。

== 獲授權使用軍隊 ==
「獲授權使用軍隊」可避免傳統對發動戰爭的限制，故常被用作「宣戰」的代替品。一般來說，必須獲眾多立法機構通過正式聲明，但「獲授權使用軍隊」可容許選舉出來的國家元首直接調動軍隊，而無須太多諮詢。此外，國際間對宣戰越來越多規限，「獲授權使用軍隊」可避免宣戰聲明的負面影響。

獲授權使用軍隊在民主國家中很常見。例如：美國在20世紀後半部直接參與了很多軍事事件，但從未正式宣戰。越戰和[[伊拉克戰爭|伊拉克戰爭]]中，美國國會授權使用軍隊，而不發表宣戰聲明。如上述，一些人質疑這些立法程序是否合乎憲法。

== 仍生效的宣戰 ==
一些宣戰聲明仍然生效，但它們並不代表敵對狀態，而是沒有簽訂和平條約去終止聲明。

* [[第二次國共內戰|第二次國共內戰]]1945-1949年10月1日[[中國共產黨|中國共產黨]]在[[北京|北京]]建立[[中華人民共和國|中華人民共和國]]。[[中華民國|中華民國]]政府則退往[[臺灣|臺灣]]，[[兩岸分治|兩岸分治]]，法律上仍在戰。
* [[敘利亞|敘利亞]]和[[以色列|以色列]]在[[贖罪日戰爭|贖罪日戰爭]]後，仍然在戰。
* 伊拉克拒絕簽署[[第一次中东战争|第一次中东战争]]的停戰協定，與以色列仍然處於戰爭狀態；後因[[伊拉克戰爭|伊拉克戰爭]]使伊拉克受到美國武力保護，並跟以色列一樣納入美軍保護傘下，減緩了對立態勢，但因為宗教民族等層面的意識形態差異而未完全結束實質的對立。
* 由於[[納戈爾諾-卡拉巴赫|納戈爾諾-卡拉巴赫]]事件的死結，[[亞美尼亞|亞美尼亞]]和[[阿塞拜疆|阿塞拜疆]]未能和平解決。
* 第二次世界大戰後，雖然[[蘇聯|蘇聯]]（[[俄羅斯|俄羅斯]]）和[[日本|日本]]并未缔结和约。不過1956年10月《[[日苏联合宣言|日苏联合宣言]]》声明：自宣言生效之日起，结束两国战争状态。因此，日苏（俄）战争状态已结束。
* 1982年英國和阿根廷間爆發的[[福克蘭戰爭|-{zh-cn:马岛战争;zh-sg:福克兰战争;zh-hk福克蘭戰爭;zh-tw:福克蘭戰爭;}-]]，英軍在戰爭中獲勝並保全對-{zh-cn:马尔维纳斯群岛;zh-sg:福克兰群岛;zh-hk:福克蘭群島;zh-tw:福克蘭群島;}-的實質主權，但阿根廷方面雖戰敗卻未向英軍正式投降，並繼續堅稱-{zh-tw:福島;zh-hk:福島;zh-cn:马岛;zh-sg:福岛;}-為其領土，且雙方並未簽署停戰協定。
* 2012年4月18日，[[苏丹|苏丹]]总统[[奧馬爾·巴希爾|巴希尔]]宣布向[[南苏丹|南苏丹]]正式宣战，这是目前世界上最新的宣战声明。
