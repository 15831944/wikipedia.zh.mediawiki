'''宣道會陳元喜小學'''（{{lang-en|'''Christian Alliance H C Chan Primary School'''}}）是一所位於[[香港|香港]][[新界|新界]][[沙田|沙田]][[城河東|城河東]]的[[基督教|基督教]][[香港小學教育|小學]]，1984年開辦，辦學團體是[[香港九龍塘基督教中華宣道會|香港九龍塘基督教中華宣道會]]。現任校長為楊潤玲女士。

== 歷史 ==
宣道會陳元喜小學原為半日制小學，下午校已於2007年9月與台山商會黃達道紀念學校合併並遷入[[頌安邨|頌安邨]]，改為全日制的[[宣道会台山陈元喜小学|宣道会台山陈元喜小学]]，而位於[[秦石邨|秦石邨]]的上午校則改為全日制上課。

===命名===
陳元喜小學是以[[陳元喜|陳元喜]]醫生為命名，學校內亦有以[[陳朱素華|陳朱素華]]為命名的陳朱素華樓、陳元喜夫人朱素華堂<ref>{{cite web|url=http://www.cahcc.edu.hk/school_profile/school_history/|title=簡史|publisher=宣道會陳元喜小學|language=zh-hk|accessdate=2018年10月31日}}</ref> 。據資料，陳元喜與夫人陳朱素華，亦捐地興建同樣是[[香港九龍塘基督教中華宣道會|香港九龍塘基督教中華宣道會]]旗下的[[上水宣道小學|上水宣道小學]]，但該校已由上水搬至粉嶺<ref>{{cite journal|url=http://www.apsss.edu.hk/mag/11.pdf|title=第45屆畢業暨頒獎典禮|date=2008年1月|journal=校訊|publisher=上水宣道小學|language=zh-hk|accessdate=2018年10月31日}}</ref>。該教會亦舉辦[[宣道會陳朱素華紀念中學|宣道會陳朱素華紀念中學]]，位於粉嶺暉明路。

== 學生 ==
大約有700多人

== 辦學理念 ==
本校以「敬畏耶和華是智慧的開端，認識至聖者便是聰明。」（箴言第九章十節）為校訓，以聖經的真理作為教育的基礎，結合現代教育理論與方法，提供優質教育，使學生能明辨是非，追尋真理，在靈、德、智、體、群、美六育方面得到均衡發展，邁向豐盛人生。

== 信念 ==
「教養孩童，使他走當行的路，就是到老他也不偏離。」
（箴言第二十二章六節）
每個人都是按著上帝的形像被造，
各有獨特與尊貴的地方，
藉著基督教教育能使人去惡從善，
發揮天賦潛能，榮神益人。

== 聯繫中學 ==
[[宣道會鄭榮之中學|宣道會鄭榮之中學]]本為宣道會陳元喜小學與[[宣道會台山陳元喜小學|宣道會台山陳元喜小學]]的聯繫中學，但自2016-2017學年起，中學與小學的聯繫經已解除。
