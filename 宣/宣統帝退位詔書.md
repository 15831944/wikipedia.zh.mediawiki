《'''宣統帝退位詔書'''》是由[[清朝|清朝]]最後一位[[清朝君主列表|君主]]、同時也是[[中國|中國]]自[[秦始皇|秦始皇]]創立[[中国皇帝|皇帝制度]]以來的最後一位被承认的皇帝宣統帝[[溥仪|溥仪]]於1912年2月12日（[[宣統|宣統]]三年[[十二月廿五|十二月廿五]]）所頒布的[[退位|退位]][[詔書|詔書]]，标志着大清帝國統治[[中國歷史|歷史]]之终结，也同时标志着在中国实行了2000多年的帝制宣告结束。計自[[努尔哈赤|努尔哈赤]]建[[后金|後金國]]，至溥儀退位，國祚長達296年；若自[[順治帝|順治帝]][[清兵入關|清兵入關]]起算，則為268年。

1911年10月[[武昌起義|武昌起義]]之後，[[辛亥革命|辛亥革命]]爆發，[[湘|湘]]、[[山西省|晉]]、[[贛|贛]]、[[陝西省|秦]]等[[行省|行省]]相繼追隨革命，[[滿族|滿族]][[大臣|大臣]][[蔭昌|蔭昌]]鎮壓革命，但由於[[北洋軍|北洋軍]]皆[[袁世凱|袁世凱]]舊部，不易指揮，所以戰鬥失效。清廷命蔭昌[[署理|署理]]陸軍大臣，以袁世凱為[[清朝內閣總理大臣|內閣總理大臣]]征討革命軍，袁世凱率[[北洋軍|北洋]]第一軍總統官[[馮國璋|馮國璋]]、北洋第二軍總統官[[段祺瑞|段祺瑞]]奉命出征，但袁世凱權衡利弊，上台後採取剿撫兼施的策略，進而準備和革命黨談判，经过[[南北议和|南北议和]]，袁世凱獲得[[黃興|黃興]]保證，將由自己擔任[[總統|大總統]]，[[清室优待条件|清室优待条件]]基本达成，但清朝核心內部依然糾葛，北洋军將領段祺瑞在袁世凱授意之下，與四十九名文武大員發了《[[北洋五十將乞共和電|北洋五十將乞共和電]]》<ref name="徐">{{cite journal|1=|title=1925年12月30日，北洋军阀皖系将领徐树铮于廊坊车站遭冯玉祥仇杀|url=http://k.sina.com.cn/article_3579599314_d55c59d20010025u8.html}}</ref>，又[[段祺瑞等要求共和第二電|接連通電]]<ref name="徐" />脅迫。當時溥儀年僅6歲，無[[行為能力|行為能力]]，由[[隆裕太后|隆裕皇太后]][[臨朝稱制|臨朝稱制]]。另外，隆裕皇太后又頒布兩個相關詔書。

== 起草人 ==
有说退位詔書起草人為[[張謇|張謇]]；但究竟何人执笔起草，一直未有定论。1912年2月22日，《[[申报|申报]]》标题为《清-{后}-颁诏逊位时之伤心语》的报道：“此次宣布共和，清谕係由前清[[學部_(清朝)|学部]]次官[[张元奇|张元奇]]拟稿，由[[徐世昌|徐世昌]]删订润色，于廿五日早九鐘前清-{后}-升[[养心殿|养心殿]]後，由[[袁世凯|袁世凯]]君进呈。清-{后}-阅未终篇已泪如雨下，随交[[世续|世续]]、[[徐世昌|徐世昌]]盖用[[玉璽|御宝]]。”<ref>{{cite web |url = http://history.people.com.cn/GB/205396/17108476.html |title = 张耀杰：是谁起草了清帝逊位诏书 |publisher = 人民网，原载于《文史参考》2012年第4期（2月下）|author = 张耀杰 |date = 2012年2月14日 |accessdate  =2012-03-02 |language = zh-hans }}</ref>

== 相关文献 ==
溥仪一生的退位詔書共有3道，兩道是以大清皇帝身份頒布，除了1912年第一道退位詔書外，溥儀於1917年的[[辮軍復辟|辮軍復辟]]失敗後，也頒布退位詔書。

1945年[[蘇聯入侵滿洲|蘇聯入侵滿洲]]，溥仪也曾以[[大满洲帝国|大满洲帝国]][[康德_(满洲国)|康德]]皇帝身份頒布[[滿洲國皇帝退位詔書|退位詔書]]。
