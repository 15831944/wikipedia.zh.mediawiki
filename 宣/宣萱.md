'''宣萱'''（'''{{lang|en|Jessica Hester Hsuan}}'''，{{bd|1970年|8月18日|catIdx=H}}），[[香港|香港]]著名[[女演員|女演員]]，視-{后}-級人馬，前無綫當家花旦。現為[[古天樂|古天樂]]旗下經理人公司藝員。<ref>{{cite web |url=https://www.orientalsunday.hk/最新娛聞/合體拍劇-古天樂瞓身撐宣萱/ |title=古天樂瞓身撐宣萱|publisher=新地|date=2018-01-19}}</ref>

==生平==
宣萱英文名“Jessica”是爸爸替她取的，“Hester”是媽媽給她另外取的一個名字，英文全名為：Jessica Hester Hsuan。宣萱儿时先後就读[[瑪利諾修院學校|瑪利諾修院學校]]（小學部）和[[香港培道中學|香港培道中學]]，13歲的时候前往英國留學，就讀英國南部女子寄宿學校罗丁中学（Roedean School）。1992年毕业于[[倫敦大學|倫敦大學]][[帝國理工學院|帝國理工學院]]（Imperial College London, University of London）并获得[[材料工程學|材料工程學]]學士學位。

1992年，宣萱因接拍某品牌朱古力广告而被[[無綫電視|無綫電視]]发掘，随后成为該台旗下的合约艺人，仅仅演了几部电视剧之后，便於95年凭借着《[[壹号皇庭IV|壹号皇庭IV]]》崭露头角，随后主演的《[[天地男儿|天地男儿]]》、《[[难兄难弟|难兄难弟]]》、《[[天地豪情|天地豪情]]》、《[[宠物情缘|宠物情缘]]》、《[[刑事侦缉档案IV|刑事侦缉档案IV]]》等多部高收视剧集，更是顺利把她推向了无线的当家花旦的后位。1999年, 宣萱憑《[[刑事侦缉档案IV|刑事侦缉档案IV]]》中的「武俏君」一角勇奪[[無綫電視|無綫電視]]《[[1999年度萬千星輝賀台慶|萬千星輝頒獎典禮1999]]》「[[萬千星輝頒獎典禮最佳女主角|最佳女主角]]」一奬，成為[[香港|香港]]視-{后}-。

2000年代初，宣萱與[[郭可盈|郭可盈]]、[[蔡少芬|蔡少芬]]及[[陳慧珊|陳慧珊]]被譽為[[無綫電視|無綫電視]]「四大花旦」，而宣萱憑著《[[洗冤录|洗冤录]]》、《[[寻秦记|寻秦记]]》、《[[流金岁月|流金岁月]]》、《[[戇夫成龙|戇夫成龙]]》、《[[老婆大人|老婆大人]]》、《[[阿旺新傳|阿旺新傳]]》等剧集继续巩固她的无线一线女星的地位，並先后六次晉身《[[壹電視大獎|壹電視大獎]]》「十大電視藝人」三甲位置、五次勇奪[[無綫電視|無綫電視]]《[[萬千星輝頒獎典禮|萬千星輝頒獎典禮]]》「我最喜爱的电视角色」以及多個香港和亞洲地區的電視奬項，連續數年穩佔[[無綫電視|無綫電視]]頭號花旦之位。2008年曾拍攝[[亞洲電視|亞洲電視]]之電視劇《[[生命有明天|生命有明天]]》。當年曾為[[無綫電視|無綫電視]]監製[[林志華|林志華]]的常用演員。2010年重返[[無綫電視|無綫電視]]首次拍攝宮廷劇《[[萬凰之王|萬凰之王]]》，飾演反派皇后[[鈕祜祿·伊蘭|鈕祜祿·伊蘭]]。2012年離開[[無綫電視|無綫電視]]。

2015年憑電影《[[我來自紐約|我來自紐約]]》林淑嫻（Sophia）一角，奪得第七屆澳門國際電影節最佳女配角。

2016年以自由身形式重返「娘家」[[無綫電視|無綫電視]]拍攝該台首部[[4K|4K]]電視劇《[[不懂撒嬌的女人|不懂撒嬌的女人]]》<ref>{{cite web |url=http://bka.mpweekly.com/focus/local/【《不懂撒嬌的女人》職場大鬥法】收視女王宣萱 |title=收視女王宣萱與暖男林文龍再合作|publisher=明報周刊|date=2017-04-27}}</ref>。2017年5月，宣萱領衍主演的劇集《[[不懂撒嬌的女人|不懂撒嬌的女人]]》播出，於劇中飾演女強人「凌敏」，觀眾及網民大讚宣萱演技出色，「凌敏」Mall姐一角大受歡迎，宣萱的演技備受讚賞，劇集獲得收視口碑佳績。自劇集播出以來，觀眾都只關注宣萱與[[林文龍|林文龍]]於劇中的劇情發展<ref>{{cite web |url=http://hk.on.cc/hk/bkn/cnt/entertainment/20170513/bkn-20170513170022111-0513_00862_001.html|title=征服女強人！挫敗宣萱獲林文龍溫柔擁吻！|publisher=東網|date=2017-05-13}}</ref> ，令觀眾留意到她的演技，Mall姐一角令宣萱的演技再次大受好評，大多觀眾及網民均認為她在劇中的表現完全力壓同劇第二女主角[[唐詩詠|唐詩詠]]<ref>{{cite web |url=http://hd.stheadline.com/news/realtime/ent/1119185/|title=唐詩詠首奪視-{后}- 網民反問：佢憑咩？|publisher=頭條日報|date=2018-01-22}}</ref> ，獲網民在討論區狂讚的宣萱被推舉為民選視-{后}-

==生活==
宣萱的圈中好友甚多，包括[[古天樂|古天樂]]、[[羅嘉良|羅嘉良]]、[[吳啟華|吳啟華]]、[[歐陽震華|歐陽震華]]、[[王喜|王喜]]、[[蔡少芬|蔡少芬]]、[[胡定欣|胡定欣]]、[[陳錦鴻|陳錦鴻]]、[[呂頌賢|呂頌賢]]、[[陳慧珊|陳慧珊]]、[[張智霖|張智霖]]、[[郭晉安|郭晉安]]、[[張家輝|張家輝]]、[[馬德鐘|馬德鐘]]、[[楊怡|楊怡]]、[[李克勤|李克勤]]、[[苗僑偉|苗僑偉]]等。宣萱也是著名愛[[狗|狗]]人士。

==趣聞==
宣萱自1994年參演無綫電視劇集《[[餐餐有宋家|餐餐有宋家]]》開始，每逢她演出的古裝、時裝劇甚至電影，她都被安排做人質被綁架，截至2017年為止，宣萱竟被脅持了25次，大破影視界紀錄，堪稱「人質王」。<ref>[http://www.youtube.com/watch?v=qLUpn1KVIyE 視頻：《飛虎》炒舊橋　無錢18年冇新意 宣萱被脅持24次封人質王]</ref>
