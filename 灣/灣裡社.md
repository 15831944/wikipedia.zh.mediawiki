'''灣裡社'''是[[臺灣日治時期|臺灣日治時期]]位於[[臺南州|臺南州]][[新化郡|新化郡]][[善化庄|善化庄]]([[台灣製糖株式會社|台灣製糖株式會社]]灣裡製糖所內{{r|新}}(今[善化糖廠]旁善糖國小校內涼亭一帶{{r|金}})的構內[[神社|神社]]。

== 歷史 ==
灣裡社選址於 [[臺南州|臺南州]][[新化郡|新化郡]][[善化庄|善化庄]]曾文{{r|新}}47番地([[台灣製糖株式會社|台灣製糖株式會社]]灣裡製糖所內)(今善糖國小校內涼亭一帶){{r|金}}，是[[臺灣製糖株式會社|臺灣製糖株式會社]]灣裡製糖所（今[[善化糖廠|善化糖廠]]）的構內[[神社|神社]]。昭和九年版的《新化郡概況》記載說該神社是在昭和五年（1930年）8月投入4500多圓所興建的，而《南瀛糖廠誌》所寫的是該年5月動工，10月完工，但兩邊記載的鎮座式都是該年10月20日舉行{{r|金}}{{r|新}}{{r|廠}}。祭祀[[天照大神|天照大神]]、[[明治天皇|明治天皇]]、[[北白川宮能久親王|北白川宮能久親王]]{{r|金}}{{r|廠}}{{r|新}}，例祭日10月20日{{r|新}}{{r|金}}。灣裡社的拜殿為神明造<ref>{{Cite book|author=金子展也|title="台湾旧神社故地への旅案内―台湾を護った神々"中142頁最上方的圖片"往時的神社寫真"背景是神明造的拜殿|location=東京|publisher=神社新報社|date=2015-09-01|pages=142頁|ISBN=978-4-908128-02-8|language=ja}}</ref>，本殿很可能也是。

而當時灣裡製糖所（今[[善化糖廠|善化糖廠]]）曾經營窄軌鐵路「善化＝神社前」營業線（戰後稱[[善化街線|善化街線]]），其中神社指的就是灣裡社。

== 現況 ==
灣裡社本殿原址位於善糖國小內涼亭{{r|金}}。現僅存被移到在善化糖廠中山橋附近的狛犬1對{{r|廠}}<ref>{{Cite web |url = https://www.google.com.tw/maps/@23.154082,120.2901913,3a,15y,6.4h,81.86t/data=!3m6!1e1!3m4!1sjpe_Kx-GAyfKZADm8sOi7g!2e0!7i13312!8i6656?hl=zh-TW|title =google地圖街景中的灣裡社狛犬現在位置|date =2015-07|language=zh(台灣)}}</ref>和善糖文物館旁庭園出土很可能屬於灣裡社的石燈籠構件"竿"、"中台"各1枚{{r|金}}<ref>{{Cite web|url =http://www.peopo.org/news/103432|title = 善化糖廠出土石燈籠構件|author =內溝文史工作室|date =2012-11-24|publisher =PeoPo公民新聞|language=zh-TW}}</ref>。善糖文物館前作為入口設立了木造[[鳥居|明神鳥居]]1基，但與灣裡社無關。
