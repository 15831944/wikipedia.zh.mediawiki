'''[[謝赫_(稱號)|謝赫]]·塔米姆·本·哈邁德·本·哈利法·阿勒薩尼'''<ref>[http://www.apdnews.com/news/29083.html 卡塔尔国王传位王储] {{webarchive|url=https://web.archive.org/web/20160304211920/http://www.apdnews.com/news/29083.html |date=2016-03-04 }}，亚太日报，2013年6月26日</ref>（{{lang-ar|الشيخ تميم بن حمد آل ثاني}}，{{bd|1980年|6月3日|||catIdx=T}}）為現任[[卡達|卡達]][[酋長國|酋長國]]的[[埃米爾|埃米爾]]，同時也是前任[[卡塔尔埃米尔|卡達埃米爾]][[哈邁德·本·哈利法·阿勒薩尼|哈邁德·本·哈利法·阿勒薩尼]]的第四個兒子。

== 生平 ==
他於1980年6月3日出生在卡達[[杜哈|杜哈]]，為哈邁德·阿勒薩尼其第二任妻子{{Link-en|穆扎·賓特·納賽爾·米斯奈德|Mozah bint Nasser Al Missned}}所生下的第二個孩子<ref name="卡達政府">{{Cite web |url= http://www.qatarembassy.net/apparent.asp |title= His Highness Sheikh Tamim Bin Hamad Al-Thani. |author=  |date=  |publisher= {{Link-en|卡達政治|Politics of Qatar|卡達政府}} |language= en |accessdate= 2013年6月26日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20100612090507/http://qatarembassy.net/apparent.asp |archivedate= 2010年6月12日 }}</ref><ref>{{Cite web |url= http://www.lemonde.fr/proche-orient/article/2013/06/25/le-prince-tamim-un-emir-jeune-et-prudent-a-la-tete-du-qatar_3436042_3218.html |title= Le prince Tamim, un émir jeune et prudent à la tête du Qatar |author= {{Link-fr|班傑明·巴托洛繆|Benjamin Barthe}} |date= 2013年6月25日 |publisher= 《[[世界報_(法國)|世界報]]》 |language=  |accessdate= 2013年6月26日 |quote=  }}{{fr}}</ref>。

塔米姆·本·哈邁德·阿勒薩尼在1998年畢業於[[桑德赫斯特皇家軍事學院|桑德赫斯特皇家軍事學院]]後，便在卡達武裝部隊擔任少尉軍官。之後他開始於卡達政府擔任許多不同領域的職位，這包括有卡達最高教育理事會主席和卡達各機構負責人等<ref name="卡達通訊社">{{Cite web|url=http://www.qnaol.net/QNAEn/Qatar/Heir_Apparent/Pages/default.aspx |title=His Highness the Heir Apparent |author= |date= |publisher={{Link-en|卡達通訊社|Qatar News Agency}} |language=en |accessdate=2013年6月26日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20130509112526/http://www.qnaol.net/QNAEn/Qatar/Heir_Apparent/Pages/default.aspx |archivedate=2013年5月9日 }}</ref>；與此同時他也致力於促進國內眾多體育賽事的發展，並且擔任了卡達奧林匹克委員會主席與[[國際奧林匹克委員會|國際奧林匹克委員會]]代表成員之一<ref>{{Cite web |url= http://portal.www.gov.qa/wps/portal/about-qatar/therulingfamily/!ut/p/c5/rZC5mpswAISfJQ9gS0ICpBJzWQ43GAMNH15f3N7Y4Xr6eJMi3xbb7Uw5xT8zIAMvd8VQXotn2XdFAxKQSTk3VdUMicRhpKiQ_0SxZoWuAE0ZHEACSR5W850v9RJUS-BGWmA9In-0NR0-tY1l183Oric70jiOIJxtncCn7I_IZig2fF3ZPt7Kq_0D7EBWHtv1-Nau4ZpIRBQECYmYMoxl6UXKPrpIkAmKTck-sDb01WUrio4pYEjIvxx-IQWCFGTy_y2u4BuQq4LiqWSP6SuPvnHLXxZzqYoohaYpGxjyULI2RhBjFeFvZX36DTMGKSOIyZQiJoKDly6W-uCjrmzyLBnSAl883PfXpBMU1AzhrSrSTPdOtXccDD8exnG02Sny3rt2WxnTr-NRcORQCc4pvA2ZtvDSmwsn6psbnrnenPbdnabxQeJTbu3nQPqt1dYK5Xm4ityw0cnFJxZ9P2jSarkmfOfp02Qz09DulYKW4FmshDMHzrZvz-DeDnV9cT5M_gAX7w44/dl3/d3/L2dBISEvZ0FBIS9nQSEh/ |title= The Ruling Family |author= |date= |publisher= {{Link-en|卡達政治|Politics of Qatar|卡達政府}} |language= en |accessdate= 2013年6月26日 |quote= }}</ref>。2013年6月25日哈邁德·阿勒薩尼宣布[[遜位|遜位]]後便由其繼任卡達埃米爾與卡達武裝部隊最高統帥<ref>{{Cite web |url= http://www.aljazeera.net/news/pages/ca216440-2e12-47c9-8f4b-39082fa34ad2 |title= أمير قطر يسلم الحكم لولي عهده |author= |date= |publisher= [[半島電視台|半島電視台]] |language=  |accessdate= 2013年6月26日 |quote=  }}{{ar}}</ref><ref>{{Cite web |url= http://hilalpost.com/fr/2013/06/25/officiel-emir-du-qatar-quitte-le-pouvoir-au-profit-de-son-fils/ |title= Officiel: L'emir du Qatar quitte le pouvoir au profit de son fils ! |author= |date= 2013年6月25日 |publisher= The Hilal Post |language=  |accessdate= 2013年6月26日 |quote=  }}{{fr}}</ref>，這也使得其除了成為[[阿拉伯世界|阿拉伯世界]]中最年輕的國家領導人<ref>{{Cite web |url= http://www.spiegel.de/politik/ausland/tamim-bin-hamad-ist-der-neue-emir-von-katar-als-nachfolger-von-hamad-a-907764.html |title= Machtwechsel in Katar: Gernegroß II. |author= Christoph Sydow |date= 2013年6月25日 |publisher= 《[[明鏡|明鏡]]》 |language=  |accessdate= 2013年6月26日 |quote=  }}{{de}}</ref><ref>{{Cite web |url= http://www.elmundo.es/elmundo/2013/06/24/internacional/1372067871.html |title= El emir de Qatar abdica en su hijo, el príncipe Sheikh Tamim |author=  |date= 2013年6月24日 |publisher= 《[[世界报_(西班牙)|世界報]]》 |language=  |accessdate= 2013年6月26日 |quote=  }}{{es}}</ref><ref>{{Cite web |url= http://www.skai.gr/news/world/article/235798/katar-o-emiris-paradidei-tin-exousia-ston-gio-tou-seihi-tamim-/ |title= Κατάρ: Ο εμίρης παραδίδει την εξουσία στον γιό του σεΐχη Ταμίμ |author=  |date= 2013年6月25日 |publisher= {{Link-en|Skai TV|Skai TV}} |language=  |accessdate= 2013年6月26日 |quote= }}{{el}}</ref>，同時也是卡達有史以來最年輕的埃米爾與當前世界上最為年輕的君主<ref name = "卡達通訊社"/>。

他目前已經結婚並且有2個妻子並且生下2個女兒與2個兒子，並且亦擁有如[[巴黎聖日耳曼足球俱樂部|巴黎聖日耳曼足球俱樂部]]等地投資股份，同時曾獲得過如[[法國|法國]][[法國榮譽軍團勳章|榮譽軍團勳章]]等勛銜<ref name = "卡達政府"/>。
