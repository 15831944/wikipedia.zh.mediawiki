'''塔什干'''（{{lang-uz|Toshkent,Тошкент}}；{{lang-ru|link=no|Ташкент}}）是[[乌兹别克斯坦|乌兹别克斯坦]]首都，是全国的政治、经济、文化和科研中心，也是[[塔什干州|塔什干州]]的首府（分为11个部分，但不包括塔什干）。塔什干市面积334.8km²，2013年共有人口2,340,900人，就其人口而言是[[独联体|独联体]]国家中第5大城市。	

在1596年，[[烏茲別克人|烏茲別克人]]與[[哈薩克汗國|哈薩克汗國]]作戰失利，簽約把塔什干城與附近草原交給[[哈薩克汗國|哈薩克汗國]]240年，城中也有很多[[哈薩克人|哈薩克人]]定居，後來於18世紀，[[浩罕汗國|浩罕汗國]]成立，由一位名為[[羽奴思|羽奴思]]的[[和卓|和卓]]奪回塔什干，再回到烏茲別克人手中（之前哈薩克[[中玉茲|中玉茲]]汗[[阿布賚|阿布賚]]曾經出兵奪回此地）。

== 词源 ==

塔什干意为“石头城”。中国[[唐代|唐代]]时将其列为西域诸国之一，曾被译作“赭石”、“赭時 ”、“者舌”、“拓支”、“察赤”或意译为“石国”。这个地区突厥化后这些名称被转化为与突厥语相同意思的"Tash"，意为石头。

== 气候 ==
塔什干有着[[半干旱|半干旱]]的大陆性气候。从每年6月到9月是漫长而炎热干旱的夏季，而每年的12月到来年2月是短暂而寒冷的冬季，並會出現降雪和結冰天氣現象。塔什干的日間温度在6月到8月间可能会非常高。由於全年平均降雨量只有100到200毫米之间，塔什干和烏茲別克大部分土地是贫瘠并且干旱的。大部分的降雨量集中在冬季和春季，而从7月到9月是非常干旱的。

== 交通 ==
===航空===
*'''[[塔什干國際機場|塔什干國際機場]]'''（{{Airport codes|TAS|UTTT}}）
:塔什干國際機場為烏茲別克航空的航空樞紐港、烏茲別克境內最大機場、以及中亞最繁忙的機場。

===铁路===
[[File:Tashkent_Station.JPG|thumb]]]]
*'''[[塔什干站|塔什干站]]'''，为塔什干的铁路枢纽，有[[土西铁路|土西铁路]]经过。
*'''[[塔什干地鐵|塔什-{干}-地鐵]]'''
:塔什-{干}-地鐵為塔什-{干}-的城市軌道交通系統，開通於1977年11月6日，是[[中亞|中亞]]首座城市軌道交通系統。

== 体育 ==
塔什干最著名的足球俱乐部是[[塔什干棉农|塔什干棉农]]和[[本尤德科足球俱乐部|本尤德科足球俱乐部]]，他们都参加乌兹别克斯坦足球联赛。

== 國際組織 ==
