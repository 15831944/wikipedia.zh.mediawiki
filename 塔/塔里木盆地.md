'''塔里木盆地'''（{{lang-ug|تارىم ئويمانلىقى}} / Tarim oymanliqi / Tarim oymanliⱪi）位于[[中國|中國]]西北部的[[新疆维吾尔自治区|新疆]]地区，是中国面积最大的内陆[[盆地|盆地]]。塔里木盆地与[[准噶尔盆地|准噶尔盆地]]中间由[[天山山脉|天山]]隔开，南北相对，分布于[[昆仑山脉|昆仑山]]、[[天山山脉|天山]]、[[阿尔金山脉|阿尔金山]]之间。东西长1500公里，南北宽约600公里，总面积达53万平方公里<ref name="塔里木盆地">[http://www.chinaxinjiang.cn/quqing/dl/8/t20100607_604279.htm 塔里木盆地]</ref>，海拔高度在800至1300米之间，地势西高东低，盆地的中部是著名的[[塔克拉玛干沙漠|塔克拉玛干沙漠]]，边缘为山麓、[[戈壁|戈壁]]和[[綠洲|绿洲]]（[[冲积平原|冲积平原]]）。
[[File:Tarimrivermap.png|300px]]

== 地貌 ==
塔里木盆地北部有眾多風蝕墩與風蝕凹地相間組成的[[雅丹地貌|“雅丹”地貌]]，大致與主風向平行。因形狀似龍，頂部是鹽結塊，外表呈白色，故《[[汉书|汉书]]·地理志》中稱為“白龍堆”。<ref name="國家地理">{{cite book|title=《中國國家地理精華》|year=2007年|location=吉林出版集團有限公司|isbn=9787807206309|pages=p.17}}</ref>

== 地形地勢 ==
塔里木盆地輪廓呈菱形，地勢西高東低，向北微傾。從外圍向中心可分為高山帶、山前礫質洪積帶、沖積平原帶及沙漠。東部[[羅布泊|羅布泊]]是盆地最低處，也是積鹽中心，海拔780米。<ref name="國家地理" />
==氣候==
[[File:1964-01_1964年_塔里木盆地.jpg|thumb]]
由于塔里木盆地远离海洋，周围又有高大山脉环抱，来自海洋的湿润气流不易到达，故塔里木盆地地区年降雨量极少，西缘的[[乌什县|乌什]]为85毫米，[[阿克苏市|阿克苏]]为57毫米；北缘的[[库车县|库车]]为63毫米，[[库尔勒市|库尔勒]]为52毫米；南缘从西向东，[[阿图什市|阿图什]]为78毫米，[[喀什市|喀什]]为65毫米，[[和田市|和田]]为35毫米，[[若羌县|若羌]]为17毫米。<ref name="塔里木盆地">[http://www.chinaxinjiang.cn/quqing/dl/8/t20100607_604279.htm 塔里木盆地]</ref>

==考古==
[[File:Miran_fresco1.jpg|300px]]
塔里木盆地[[小河墓地|小河墓地]]已發掘出土[[肉身菩薩|墓葬與木乃伊]]，工具，工程陶瓷，彩陶和其他[[藝術品|藝術品]]。[[米兰遗址|米兰遗址]]就位于塔里木盆地中的塔克拉玛干沙漠中。在塔里木盆地的[[和田地区|和田地区]]发现了汉晋至南北朝年代的[[汉传佛教|佛教遗址]]。<ref>[http://www.chinanews.com/cul/2012/02-29/3708957.shtml 新疆和田发现佛教遗址 位于塔里木盆地南沿]</ref>

== 資源 ==
塔克拉瑪干沙漠位於塔里木盆地中心，幾乎終年不雨，被認為含有儲量豐富的[[石油|石油]]和[[天然气|天然气]]<ref>[http://www.cnki.com.cn/Article/CJFDTOTAL-TTSD200503007.htm 中国西北部造山型盆地构造特征与油气赋存]</ref><ref>[http://roll.sohu.com/20140201/n394453638.shtml 塔里木油田生产超过一亿吨]</ref>。
