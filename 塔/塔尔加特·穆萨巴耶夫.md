'''塔尔加特·阿曼格尔季耶维奇·穆萨巴耶夫'''（{{lang-ru|Талгат Амангельдиевич Мусабаев}}；{{lang-kk|Талғат Амангелдіұлы Мұсабаев|Talgat Amangeldyuly Musabayev}}，{{bd|1951年|1月7日|}}）是哈萨克斯坦第2位宇航员，苏联/俄罗斯第79位宇航员，世界第309位宇航员<ref>{{cite web |title=Позывные космонавтов |url=http://www.gctc.ru/main.php?id=156 |publisher=GCTC |archiveurl=https://web.archive.org/web/20190421055051/http://www.gctc.ru/main.php?id=156 |archivedate=2019-04-21 |language=ru}}</ref>。[[俄罗斯联邦英雄|俄罗斯联邦英雄]]，[[哈薩克斯坦英雄|哈薩克斯坦英雄]]。历任哈萨克斯坦国家航天局局长（2007年-2016年），哈萨克斯坦教育科学部部长（2010年-2013年），哈萨克斯坦总统顾问（2016年-2017年），[[哈萨克斯坦议会|哈萨克斯坦议会]]参议院议员（2017年7月-）。
==早年==
1951年1月7日出生在[[苏联|苏联]][[哈萨克苏维埃社会主义共和国|哈萨克苏维埃社会主义共和国]][[阿拉木圖州|阿拉木圖州]]卡尔加里。1974年毕业于[[里加|里加]]民航学院“飞行器无线电设备维护”专业，后在[[阿拉木图|阿拉木图]]布伦达伊联合航空公司担任工程师。1993年毕业于[[阿赫图宾斯克|阿赫图宾斯克]]军事飞行学校<ref name=bioru>{{cite web |title=Мусабаев, Талгат Амангельдиевич |url=http://www.warheroes.ru/hero/hero.asp?Hero_id=2976 |publisher=Герои страны |accessdate=2019-04-22 |language=ru}}</ref>。

==宇航员生涯==
1990年5月11日入选宇航员分队，后在[[加加林宇航员培训中心|加加林宇航员培训中心]]接受基础航空飞行训练，1991年9月13日获得测试宇航员资格<ref name=bioru/>。
===和平号空间站===
1998年1月29日至8月25日，他担任EO-25任务飞行工程师，在[[和平号空间站|和平号空间站]]执行任务<ref name=NASA-bio>{{cite web|url=https://history.nasa.gov/SP-4225/documentation/mir-summaries/nasa7/biographies/biographies.htm#musabayev|title=Documents – NASA-7 biographies|publisher=NASA}}</ref>。

===EP-1===
他的第三次任务是作为[[国际空间站|国际空间站]]“EP-1”短期访问任务的指令长。2001年4月28日乘坐[[联盟TM-32|联盟TM-32]]，2001年5月6日乘坐[[联盟TM-31|联盟TM-31]]返回，总飞行时间为7天22小时4分钟<ref name=EA-EP4>{{cite web|url=http://www.astronautix.com/flights/mirep4.htm|title=Mir EP-4|publisher=Encyclopedia Astronautica|deadurl=yes|archiveurl=https://web.archive.org/web/20111011182802/http://astronautix.com/flights/mirep4.htm|archivedate=2011-10-11}}</ref>。该任务以携带世界首位自费“[[太空游客|太空游客]]”[[丹尼斯·蒂托|丹尼斯·蒂托]]而著称<ref>{{cite web |title=首名太空游客升空 搅热外空旅游美梦 |url=http://www.china.com.cn/chinese/TR-c/31927.htm |publisher=[[北京晨报|北京晨报]] |accessdate=2019-04-22 |archiveurl=https://web.archive.org/web/20060620011049/http://china.com.cn/chinese/TR-c/31927.htm |archivedate=2006-06-20 |date=2001-04-29}}</ref>。

==退役==
2003年成为俄罗斯联邦陆军航空兵部队战斗训练部队负责人。2005年开始担任哈萨克斯坦和俄罗斯合资公司“[[巴伊杰列克观景塔|巴伊杰列克]]”总干事。2007年2月被任命为哈萨克斯坦共和国教育科学部航空航天委员会主席<ref>{{cite web|url=http://www.inform.kz/eng/article/2178551|title=Kazakhstan has own space exploration course :Talgat Musabayev|publisher=Kazinform|date=2009-04-13}}</ref>。2007年6月27日加入[[祖国之光|祖国之光]]政党<ref>{{cite web |title=Летчик-космонавт Т. Мусабаев вступил в ряды партии «Нур Отан» |url=http://www.zakon.kz/our/news/news.asp?id=30109212 |accessdate=2019-04-22 |archiveurl=https://web.archive.org/web/20071214125855/http://www.zakon.kz/our/news/news.asp?id=30109212 |archivedate=2007-12-14 |language=ru}}</ref>。2007年至2016年担任哈萨克斯坦国家航天局局长，2010年至2013年担任哈萨克斯坦教育科学部部长<ref>{{cite web |title=Талгат Мусабаев назначен сенатором |url=https://tengrinews.kz/kazakhstan_news/talgat-musabaev-naznachen-senatorom-322259/ |website=tengrinews.kz |archiveurl=https://web.archive.org/web/20190422150604/https://tengrinews.kz/kazakhstan_news/talgat-musabaev-naznachen-senatorom-322259/ |archivedate=2019-04-22 |language=ru |date=2017-07-13}}</ref>。2016年1月至2017年7月13日担任哈萨克斯坦共和国总统顾问。2017年7月起成为[[哈萨克斯坦议会|哈萨克斯坦议会]]参议院议员<ref>{{cite web |title=Мусабаев Талгат Амангельдиевич |url=https://kapital.kz/dossier/musabaev-talgat/ |website=kapital.kz |accessdate=2019-04-22 |date=2017-07-21}}</ref>。
