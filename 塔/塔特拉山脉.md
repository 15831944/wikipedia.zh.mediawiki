'''塔特拉山脉'''（{{lang-pl|Tatry}}；{{lang-sk|Tatry}}；{{lang-en|Tatra}}），是一条位于[[中欧|中欧]]的[[山脉|山脉]]。塔特拉是[[喀尔巴阡山脉|喀尔巴阡山脉]]中的最高山脉，是[[斯洛伐克|斯洛伐克]]与[[波兰|波兰]]的边界山脉。最高峰[[格尔拉赫峰|格尔拉赫峰]]，海拔2655米。<ref>{{cite web

==概述==
[[File:Tatry_NASA_World_Wind.jpg|thumb]]]]
[[File:Tatra_mountains_western_side_2.jpg|thumb]]
[[File:PanoramaTatr.jpg|thumb]]

塔特拉山脉兴起于[[阿尔卑斯造山运动|阿尔卑斯造山运动]]，是一条相对年轻的山脉，主要分为高塔特拉山和低塔特拉山，具有典型的[[阿尔卑斯|阿尔卑斯]]式地形，多冰斗、溶洞、山岳湖泊和悬谷。<ref>{{cite web
|url= http://news.xinhuanet.com/travel/2011-08/15/c_121859504_2.htm
| language= zh
| title= 斯洛伐克 来自塔特拉山上的风
|publisher= 新华网
|date= 2011年8月15日
|accessdate= 2013年1月7日}}</ref>塔特拉是[[喀尔巴阡山脉|喀尔巴阡山脉]]中的最高山脉，[[格尔拉赫峰|格尔拉赫峰]]是喀尔巴阡山脉的最高峰，也是喀尔巴阡山脉内少数高于2500米的山峰之一。

因其景观特征和便利的交通与基础设施，塔特拉山受到游客和科研人员的喜爱，塔特拉山脉发展成为受欢迎的[[冬季运动|冬季运动]]区及度假胜地，被称为“波兰的冬季首都”。包括[[扎科帕内|扎科帕内]]、[[波普拉德|波普拉德]]、[[上塔特拉山镇|上塔特拉山镇]]。
高塔特拉有24座（或25座）超过海拔2500米的山峰，连同[[南喀尔巴阡山脉|南喀尔巴阡山脉]]，成为整个1200公里长的喀尔巴阡山脉范围内唯一的高山景观的代表性地段。

==气候==
塔特拉山脉位于[[欧洲|欧洲]]中部的[[温带|温带]]，是[[气团|气团]]活动的重要阻碍。多山地形导致了该地区内多样化的气候。

;风速
[[顶峰|顶峰]]平均风速为6米/秒。
*北坡吹偏南风。
*塔特拉山底吹西风。
*[[焚风|焚风]]最常发生在十月和五月。
*最大风速288公里/小时（1968年5月6日）。<ref>{{cite web

;温度
全年温度变化范围在-40°C到33°C间，顶峰192天低于0°C。

;降水
最高的[[降水|降水]]记录在北坡，六月和七月的降水量达250毫米。一年降雨天数在215天至228天之间。发生[[雷暴|雷暴]]的天数平均每年36天。

;冰雪覆盖
顶峰冰雪覆盖的最大厚度合计：
*在波兰 - 355厘米；
*在斯洛伐克 - 410厘米。
有些山峰常年有冰雪覆盖，[[雪崩|雪崩]]频发。

2011年3月13日，波兰塔特拉山区发生雪崩，造成3人死亡。<ref >{{cite web

==植被==
[[File:Leontopodium_alpinum_detail.jpg|thumb]]]]
塔特拉山脉分布着物种多样的植物，有1000余种[[维管植物|维管植物]]、450余种[[苔藓|苔藓]]、700余种[[地衣|地衣]]、900余种[[真菌|真菌]]。
塔特拉山脉有5个气候植被带：
*1300米以下：喀尔巴阡山[[榉木|榉木]]森林，几乎没有灌木层，草本层占据了大部分。
*1300 - 1500米：[[云杉|云杉]]林，灌木层发育差，苔藓是重要组成部分。
*1500 - 1800米：山松及各种草本植物。
*1800 - 2300米：高海拔草原。
*2300米以上：裸露的岩石，除地衣外几乎没有植物。

==动物==
[[File:Rupicapra_rupicapra_-_Jahňaci_štít.jpg|thumb]]

塔特拉山脉栖息着多种动物，是54种[[缓步动物门|缓步动物]]、22种[[涡虫纲|涡虫]]、100种[[轮形动物门|轮形动物]]、22种[[桡脚类|桡脚类]]、162种[[蜘蛛|蜘蛛]]、81种[[软体动物|软体动物]]、43种[[哺乳动物|哺乳动物]]、200种鸟类、7种两栖类和爬行类动物的家园。

最值得注意的哺乳类动物有塔特拉羚羊、[[旱獭|旱獭]]、山白䶄、[[棕熊|棕熊]]、[[狼|狼]]、[[欧亚猞猁|欧亚猞猁]]、[[欧洲马鹿|欧洲马鹿]]、[[西方狍|西方狍]]及[[野猪|野猪]]；知名鱼类有[[溪红点鲑|溪红点鲑]]和[[花足杜父鱼|花足杜父鱼]]。

[[特有种|特有种]]节肢动物包括[[石蛾|石蛾]]，Xysticus alpicola蜘蛛和[[弹尾虫|弹尾虫]]。

==人类活动==
[[File:Tatry_w_drodze_do_Murowańca.jpg|thumb]]

在18世纪和19世纪，塔特拉山脉被用来放牧和采矿。为开辟道路，许多樹木被砍伐。这些活动虽然停止了，但其影响仍清晰可见。[[俄斯特拉发|俄斯特拉发]]和[[克拉科夫|克拉科夫]]的工业区亦对环境造成损害。2012年开始，波兰的志愿者每年会进行清理垃圾的工作。<ref>{{cite news 
| url = http://wiadomosci.onet.pl/regionalne/podhale/sprzatali-tatry-zebrali-tony-smieci,1,5176706,region-wiadomosc.html 
| title = Sprzątali Tatry. Zebrali tony śmieci 
| language = Pl
| trans_title =  
| work = 波兰通讯社
| date = 2012年7月1日}}</ref>
斯洛伐克塔特拉山国家公园成立于1949年（738公里²），215.56公里²的波兰塔特拉山国家公园成立于1954年。这两个相连的国家公园在1992年被联合国教科文组织列为人与生物圈计划下的生物圈保护区。

==文化==
身為中歐地區重要的地標，塔特拉山脈對於鄰近國家具有相當程度的文化影響。例如斯洛伐克的國歌，就是名為《[[塔特洛山上電光閃閃|塔特洛山上電光閃閃]]》。除此之外，總部設於[[科普日夫尼采|科普日夫尼采]]（Kopřivnice）的[[捷克共和國|捷克]]主要汽車品牌[[太拖拉|太拖拉]]（Tatra），就是因為車廠設於塔特拉山腳下而將山名作為品牌名稱。
