'''塔特·泰勒'''（{{lang-en|'''Tate Taylor'''}}）是一名[[美国|美国]]男演员、编剧、电影制片人和导演。他知名於執導《[[寫出友共鳴|寫出友共鳴]]》、《{{link-en|激樂人心|Get on Up (film)}}》和《[[列車上的女孩_(2016年電影)|列車上的女孩]]》。

== 早年 ==
他出生并成长于[[密西西比州|密西西比州]]的州府[[杰克逊_(密西西比州)|杰克逊]]。他是女作家Kathryn Stockett的好友。两人自入读[[幼儿园|幼儿园]]时就已认识<ref>http://www.dailymail.co.uk/femail/article-2024269/The-Help-Kathryn-Stockett-Tate-Taylors-friendship-took-book-big-screen.html</ref>。

== 生涯 ==
他在[[纽约|纽约]]和[[洛杉矶|洛杉矶]]工作了15年。

他曾导演过的电影有''Pretty Ugly People''，以及2011年的《[[相助|-{zh-cn:相助; zh-tw:姊妹; zh-hk:寫出友共鳴;}-]]》，本片是根据Kathryn Stockett写的小说《-{zh-cn:相助; zh-tw:姊妹; zh-hk:寫出友共鳴;}-》（''The Help''）改编而成，小说出版于2009年2月，而早在2008年6月他就获得了电影改编权<ref>http://arts.nationalpost.com/2011/08/10/qa-the-help-hits-close-to-home-for-director-tate-taylor/{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 個人生活 ==
