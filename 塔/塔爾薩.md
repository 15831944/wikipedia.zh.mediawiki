'''塔爾薩'''（{{lang-en|'''Tulsa'''}}）是[[美國|美國]][[奧克拉荷馬州|奧克拉荷馬州]]東北部的[[塔爾薩縣|塔爾薩縣]]首府，也是該州人口第二多[[城市|城市]]，2006年人口387,807，2013年估計人口398,121。塔爾薩附近是[[阿肯色河|阿肯色河]]。這個城市的總面積是483.9 km²（186.8 mi²），總面積的2.24％（10.9 km²）為水域。塔爾薩是美國第45大的城市和[[北美洲|北美洲]]第93大城市。因昔日盛產[[石油|石油]]，故有「世界石油之都」美譽。塔爾薩在[[奧克拉荷馬市|奧克拉荷馬市]]的東北99英里。塔爾薩大約是美國[[華盛頓特區|華盛頓特區]]三倍的大小。

== 歷史 ==
塔爾薩原本是印第安人的村莊。1882年，興建[[圣路易斯_(密苏里州)|聖路易斯]]至[[舊金山|舊金山]]的鐵路期間，白人開始定居。1898年建市。後來發現石油，發展迅速。

== 地理与气候 ==
塔尔萨属[[副热带湿润气候|副热带湿润气候]]，四季分明，降水充足而5、6月降雨量最多。冬季温和，时而偏向温暖，日最高气温低于或等于{{convert|0|C}}的平均日数为8.8天，日最低气温低于或等于{{convert|0|C}}的平均日数为71天，低于或等于{{convert|-10|C}}的平均日数为6.9天；夏季酷暑漫长，日高温达{{convert|30|C}}的日数年均有100天，达{{convert|100|F|disp=flip}}的有11天。<ref name = "NWS Tulsa, OK"/>最冷月（1月）均温{{convert|37.7|F|1|disp=flip}}，极端最低气温{{convert|−16|F|0|disp=flip}}（1930年1月22日）。<ref name = "NWS Tulsa, OK"/>最热月（7月）均温{{convert|82.9|F|1|disp=flip}}，极端最高气温{{convert|115|F|0|disp=flip}}（1936年8月10日）。<ref name = "NWS Tulsa, OK"/>无霜期平均为223天（3月29日至11月6日）；可测量降雪平均期为12月15日至2月22日。<ref name = "NWS Tulsa, OK"/>年均降水量约{{convert|41.0|in|sigfig=3|disp=flip}}；年极端最少降水量为{{convert|23.24|in|disp=flip}}（1956年），最多为{{convert|69.88|in|disp=flip}}（1973年）。<ref name = "NWS Tulsa, OK"/>年均降雪量为{{convert|9.6|in|cm|disp=flip}}；冬季极端最少降雪量微少以致难以测量（上次为1910–11年），2010–11年的降雪量最多，积累降雪量为{{convert|26.1|in|cm|disp=flip}}。<ref name = "NWS Tulsa, OK"/>
