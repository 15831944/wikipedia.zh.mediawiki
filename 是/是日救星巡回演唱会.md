《'''是日救星巡回演唱会'''》，是台湾女歌手[[徐佳莹|徐佳莹]]的第四次个人大型巡回演唱会<ref>{{Cite web |url=http://www.chinatimes.com/realtimenews/20161220005312-260404 |title=徐佳瑩32歲生日 宣布明年北高攻蛋 |accessdate=2017-05-29 |author=中央社 |date=2016-12-20 |website=中時電子報}}</ref>，首场于2017年4月22日在[[台北小巨蛋|台北小巨蛋]]揭开序幕<ref name="台北场">{{Cite web |url=https://www.kkbox.com/tw/tc/column/live_reviews-15-413-1.html |title=「讓我的音樂拯救你們」—徐佳瑩 是日救星演唱會 |accessdate=2017-05-29 |date=2017-04-27 |website=KKBOX}}</ref>，至2018年12月15日在[[高雄市|高雄]]作为最终场结束本次巡演，此次巡演共有两轮及10场演出。
