《'''即使是我也會哭泣的啊'''》（{{jpn|j=僕だって泣いちゃうよ|hg=ぼくだってないちゃうよ|rm=Boku datte naichauyo}}）是日本女子[[日本偶像|偶像]]組合[[NMB48|NMB48]]的第19張[[單曲|單曲]]，於2018年10月17日由[[YOSHIMOTO_R_and_C|laugh out loud! records]]發行。

== 概要==
單曲分為TYPE-A、TYPE-B、TYPE-C、TYPE-D和劇場盤。《即使是我也會哭泣的啊》和《浪漫的再見》收錄於所有版本，其中《即使是我也會哭泣的啊》的音樂錄影帶和舞蹈版音樂錄影帶收錄於劇場盤以外的所有版本，劇場盤則收錄了《不會逃離夢想》。《謊言機器》和其音樂錄影帶、演唱會片段「NMB48 LIVE TOUR 2018 in Summer@東京[[中野太陽廣場|中野太陽廣場]]（2018.7.30）中的《overture(NMB48 ver.)》《HA !》《[[大蔥鴨們|大蔥鴨們]]》《[[欲望者|Good Timing]]》《[[1830m|而且不叫酪梨啦…]]》《{{Link-ja|365天的紙飛機|365日の紙飛行機}}》《[[絕滅黑髮少女|青春的單圈時間]]》」收錄於初回版TYPE-A，通常版TYPE-A則改為收錄《謊言機器》的製作特輯，而沒有收錄演唱會片段；《True Purpose》和其音樂錄影帶、演唱會片段「NMB48 LIVE TOUR 2018 in Summer@兵庫{{Link-ja|神戶國際會館|神戸国際会館}}（2018.9.3）中的《overture(NMB48 ver.)》《[[絕滅黑髮少女|絕滅黑髮少女]]》《[[北川謙二_(單曲)|北川謙二]]》《[[世界的中心是大阪_～難波自治區～|想像的詩人]]》《[[Don't_look_back!|Don't look back!]]》《大蔥鴨們》《[[我不在|我不在]]》《[[輕咬少女|輕咬少女]]》《[[純情U-19|純情U-19]]》」收錄於初回版TYPE-B，通常版TYPE-B則改為收錄《True Purpose》的製作特輯，而沒有收錄演唱會片段；《職務質問》和其音樂錄影帶、演唱會片段「NMB48 LIVE TOUR 2018 in Summer@廣島{{Link-ja|廣島縣立文化藝術會堂|広島県立文化芸術ホール|上野學園會堂}}（2018.8.30）中的《overture(NMB48 ver.)》《[[Ja-Ba-Ja|犯蠢]]》《[[Team_A_5th_Stage「戀愛禁止條例」|NMB參上!]]》《[[欲望者|別無藥可救]]》《[[輕咬少女|輕咬少女]]》《[[欲望者|欲望者]]》《[[RIVER_(AKB48單曲)|飛機雲]]》《[[Team_K_2nd_Stage「青春女孩」|約好了唷]]》《[[大聲鑽石|大聲鑽石]]》」收錄於初回版TYPE-C，通常版TYPE-C則改為收錄《職務質問》的製作特輯，而沒有收錄演唱會片段；《想你忘記》和其音樂錄影帶、影片「山本彩的軌跡 2010-2018」收錄於TYPE-D，通常版TYPE-D則改為收錄《即使是我也會哭泣的啊》的製作特輯。劇場盤以外的初回版均附有活動參加券，通常版則是[[生寫真|生寫真]]，而劇場盤則是個別[[握手會|握手會]]的參加券<ref>{{cite web |url=http://nmb48.com/discography/19th_single.html |language=ja |script-title=ja:19th Single 「僕だって泣いちゃうよ」 |publisher=[[NMB48|NMB48]] |accessdate=2018-10-24  |archive-url=https://web.archive.org/web/20180927113956/http://www.nmb48.com/discography/19th_single.html |archive-date=2018-09-27 |dead-url=no }}</ref>。

2018年9月13日，NMB48在[[SHOWROOM|SHOWROOM]]宣佈推出單曲<ref>{{cite web |url=https://natalie.mu/music/news/299602 |language=ja |script-title=ja:NMB48山本彩ラストシングルでセンター抜擢、選抜メンバー18名発表 |publisher=[[natalie_(網站)|natalie]] |date=2018-09-13 |accessdate=2018-10-24  |archive-url=https://web.archive.org/web/20180925180528/https://natalie.mu/music/news/299602 |archive-date=2018-09-25 |dead-url=no }}</ref>。9月25日，單曲的音樂錄影帶在[[Youtube|Youtube]]上正式公開，另外也同時公佈了由[[紀嘉良|紀嘉良]]拍攝的封面<ref>{{cite web |url=https://www.oricon.co.jp/news/2120245/full/ |language=ja |script-title=ja:圧巻ダンスで山本彩を送り出す! NMB48「僕だって泣いちゃうよ」MV&ジャケ写解禁 |publisher=[[Oricon|Oricon]] |date=2018-09-25 |accessdate=2018-10-24  |archive-url=https://web.archive.org/web/20181012134954/https://www.oricon.co.jp/news/2120245/full/ |archive-date=2018-10-12 |dead-url=no }}</ref>。9月26日，NMB48研究生在NMB48劇場公演首次演唱了由[[山本彩|山本彩]]作曲的《不會逃離夢想》<ref>{{cite web |url=https://www.hochi.co.jp/entertainment/20180926-OHT1T50233.html |language=ja |script-title=ja:“プロデューサー”山本彩　作曲した「夢は逃げない」をＮＭＢ研究生が初お披露目 |publisher=[[體育報知|體育報知]] |date=2018-09-26 |accessdate=2018-10-24  |archive-url=https://web.archive.org/web/20180927082539/https://www.hochi.co.jp/entertainment/20180926-OHT1T50233.html |archive-date=2018-09-27 |dead-url=no }}</ref>。10月2日，NMB48在[[千葉縣|千葉縣]][[幕張展覽館|幕張展覽館]]結成8週年紀念公演「NMB48 8th Anniversary LIVE」中首次演唱了單曲<ref>{{cite web |url=https://www.oricon.co.jp/news/2120655/full/ |language=ja |script-title=ja:NMB48、8周年ライブで山本彩卒業シングル初披露「私だって泣いちゃうよ」 |publisher=Oricon |date=2018-10-02 |accessdate=2018-10-24  |archive-url=https://web.archive.org/web/20181017203044/https://www.oricon.co.jp/news/2120655/full/ |archive-date=2018-10-17 |dead-url=no }}</ref>。10月16日至28日，為了紀念單曲發賣，NMB48在[[大阪市|大阪市]][[心齋橋|心齋橋]][[HMV|HMV]]開設「NMB48 POP UP SHOP」，展示了單曲的服裝，亦有發售單曲的週邊產品<ref>{{cite web |url=https://www.oricon.co.jp/news/2121400/full/ |language=ja |script-title=ja:NMB48ショップが心斎橋に期間限定オープン さや姉卒業記念グッズ販売&衣装展示 |publisher=Oricon |date=2018-10-12 |accessdate=2018-10-24  |archive-url=https://web.archive.org/web/20181012160222/https://www.oricon.co.jp/news/2121400/full/ |archive-date=2018-10-12 |dead-url=no }}</ref>。

== 反應 ==
在《[[告示牌_(雜誌)|告示牌]]》Top Singles Sales榜中，單曲銷出265,583張，取得週榜第1位<ref>{{cite web|url=http://www.billboard-japan.com/d_news/detail/68964/2 |language=ja|script-title=ja:【ビルボード】NMB48『僕だって泣いちゃうよ』が265,583枚を売り上げ週間シングル・セールス首位獲得 |date=2018-10-22 |publisher=[[告示牌_(雜誌)|告示牌]] |accessdate=2018-10-24}}</ref>，[[日本熱門100金曲榜|日本熱門100金曲榜]]亦同樣取得第1位<ref>{{cite web|url=http://www.billboard-japan.com/d_news/detail/69039/2 |language=ja|script-title=ja:【ビルボード】26.5万枚を売り上げたNMB48「僕だって泣いちゃうよ」が総合首位獲得　BUMP「話がしたいよ」4位、米津「Flamingo」16位に初登場 |date=2018-10-24 |publisher=告示牌 |accessdate=2018-10-24}}</ref>。在[[Oricon公信榜|Oricon公信榜]]中，單曲以約23.8萬銷量，取得週榜第1位，自《[[榴槤少年|榴槤少年]]》起連續8張單曲取得第1位，總計則是17張單曲取得冠軍<ref>{{cite web|url=https://www.oricon.co.jp/news/2122002/full/ |language=ja|script-title=ja:NMB48、山本彩の卒業シングルが8作連続通算17作目の1位 |publisher=Oricon |date=2018-10-23 |accessdate=2018-10-24}}</ref>。
