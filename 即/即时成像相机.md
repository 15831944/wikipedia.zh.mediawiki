'''即时成像相机'''（'''Instant Camera'''），俗稱'''拍立得'''<ref>[https://tw.appledaily.com/new/realtime/20190318/1535047/  Canon首款拍立得曝光　機身薄如相印機]</ref><ref>[https://tw.appledaily.com/new/realtime/20181118/1468442/ 讓iPhone直接變拍立得　搖一搖顯影更快]</ref> 、'''立可拍'''<ref>[https://www.ettoday.net/news/20181109/1301831.htm#ixzz5izNj9Pn2 進博會少女心爆發！凱蒂貓立可拍、100%羊駝毛吊飾...萌點滿滿]</ref><ref>[https://www.chinatimes.com/newspapers/20180922000728-260112 《厲可拍》7分鐘奪1命]</ref>（另有一種拋棄式相機「Quicksnap」，俗稱 「即可拍」，非「立可拍」），是一种拍摄后相片能够即时快速成像的[[照相机|照相机]]，在按下快门后数秒内便能够自动从机身内吐出照片，并在几分钟内完成显影<ref>{{cite web |url=http://www.fujifilm.com.cn/products/film_camera/instant/instant.html |title=存档副本 |accessdate=2013-08-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20131207054604/http://www.fujifilm.com.cn/products/film_camera/instant/instant.html |archivedate=2013-12-07 }}，富士胶片公司的一次成像相机介绍页面。</ref>。因为其快速显影的优势，在[[胶卷|胶卷]]时代有着众多的使用者。但是由于[[数码摄影|数码摄影]]的普及，已经逐渐淡出市场<ref>[http://tech.163.com/08/1128/13/4RRCPSF3000915BD.html]，宝丽来公司的停产</ref>。
[[File:Kodak_Instant_Camera_EK6_(Sofortbildkamera).JPG|thumb]]

== 优点 ==
能够很快的得到拍摄结果，做到了即拍即得，能够做到相片的快速分享。对于傳統相機需胶片冲洗节省了许多时间。對於[[數位相機|數位相機]]，市場上雖有少數可隨身攜帶、使用電池的小型相片[[印表機|印表機]]，但便利性仍不如即時成像相機。

== 缺点 ==
相纸价格比较昂贵，例如[[宝丽来|宝丽来]]公司由于停产，原厂的过期相纸价格在[[德国|德国]]售价近20[[欧元|欧元]]<ref>[http://tech.163.com/08/1128/13/4RRCPSF3000915BD.html]</ref>。而专门为宝丽来相机复产的[[不可能计划|不可能计划]]（The Impossible Project）公司的相纸价格亦为23.49美元<ref>[http://shop.the-impossible-project.com/shop/]，不可能计划公司的网络销售页面。</ref>，而且由于技术原因只有八张。
