《'''流浪地球'''》是2019年上映的中国3D[[科幻片|科幻]][[冒險片|冒險電影]]，改编自[[刘慈欣|刘慈欣]]所著[[流浪地球|同名小说]]，由[[郭帆|郭帆]]执导，[[吳京_(演員)|吴京]]特别出演，[[屈楚萧|屈楚萧]]、[[赵今麦|赵今麦]]、[[李光洁|李光洁]]、[[吴孟达|吴孟达]]领衔主演，[[雷佳音|雷佳音]]友情主演<ref name="huanqiu-dingdang">{{cite web |title=《流浪地球》定档2019大年初一中国科幻冒险启程 |url=http://ent.huanqiu.com/movie/yingshi-neidi/2018-10/13365367.html |website=[[环球网|环球网]] |accessdate=2018-10-30 |language=zh-hans |date=2018-10-26 |archive-url=https://web.archive.org/web/20181030023238/http://ent.huanqiu.com/movie/yingshi-neidi/2018-10/13365367.html |archive-date=2018-10-30 |dead-url=no }}</ref><ref>{{cite web |title=流浪地球演职员 |url=http://movie.mtime.com/218707/fullcredits.html |website=[[时光网|时光网]] |accessdate=2019-02-09 |archive-url=https://web.archive.org/web/20190209124539/http://movie.mtime.com/218707/fullcredits.html |archive-date=2019-02-09 |dead-url=no }}</ref>。本片主要敘述[[太陽|太陽]]即將膨脹吞噬[[太陽系|太陽系]]之時，人類社會啟動「流浪地球」計劃並傾盡所有力量製造「行星發動機」驅使[[地球|地球]]逃離太陽系前往新家園的途中，地球經過[[木星|木星]]附近時所發生的故事。

[[主要拍攝|主要拍攝]]於2017年5月在[[山东省|山东省]][[青岛市|青岛市]]開始，並於同年9月結束，進入後期製作。该片于2019年2月5日（农历[[己亥|己亥]]年[[大年初一|大年初一]]）在中国大陆上映。美国流媒体巨头[[Netflix|Netflix]]于2019年2月宣布获得除中国大陆地区外的全球流媒体播放权，并将会翻译成28种语言。该片将于2019年5月6日上线[[Netflix|Netflix]]。

== 剧情 ==
人类预测在不久的將來，太陽將膨脹吞噬整個[[太陽系|太陽系]]，地球面臨毀滅。為求生存，人類社會开启「流浪地球」計劃，製造一万座「行星發動機」和两千座“转向发动机”, 試圖帶著地球一起逃出[[太陽系|太陽系]]前往新家園。人类建造和發射了“领航者号”国际[[空间站|空间站]]，为地球提供预警、领航、通讯服务。<ref>{{Cite book|chapter=流浪地球·编年史|title=流浪地球电影制作手记|author=朔方 等|publisher=人民交通出版社|year=2019|isbn=978-7-114-15160-6|location=|pages=}}</ref>

2075年，为利用[[引力弹弓效应|引力弹弓效应]]，地球航行到木星附近。航天员刘培强的儿子刘启带着妹妹韩朵朵从[[北京|北京]]地下城来到地表，用外公韩子昂的车卡偷开运载车，被发现并被关押于[[济宁|济宁]]派出所。韩子昂赶到并行贿看守未果，被一同关押。

[[File:Jupiter's_Great_Red_Spot_-_July_8_1979_(26008230898).jpg|缩略图]]]]
此时，[[木星|木星]]引力突然增大，地球地壳变动，五千余座行星发动机停机，地球失去足夠的自推力而坠入木星，将因进入木星的刚体[[洛希极限|洛希极限]]而解体。联合政府紧急动员，刘启、韩朵朵、韩子昂和牢友Tim四人从倒塌的派出所逃出，他们乘坐的运载车被CN171-11救援队（队长为王磊）征用，前去救援[[杭州|杭州]]一号行星发动机。经过[[上海|上海]]时运输车被毁，其後在運送火石的过程中韩子昂死亡，多名救援队员牺牲。之后刘启、韩朵朵、Tim离开王磊小队，找到另外一支救援队的运载车，决定执行该救援队既定的救援[[苏拉威西|苏拉威西]]三号转向发动机的任务，前往苏拉威西。王磊小队得知杭州一号行星发动机损坏且杭州4号行星发動机的地下城被岩浆毁灭，加入了刘启等人。即将到达苏拉威西时，他们得知苏拉威西三号转向发动机已被其他救援队重启。“领航者号”国际空间站[[人工智能|人工智能]]（名为“MOSS”）随后以联合政府名义全球广播：大部分发动机已经修复，但不足以将地球推离木星，地球将在七天后解体。广播后，刘启想到可以点燃混有木星原有的氢气和从地球吸引而来的氧气的木星大气，从而製造足夠的衝擊波推开地球。

“领航者号”国际空间站上的刘培强即将进入休眠时，与刘启失联。刘培强强行打开休眠舱，发现空间站正在远离木星，刘培强说服被MOSS唤醒来阻止他的航天员马卡洛夫，他们经空间站外通过太空行走前往空间站主控室，马卡洛夫在途中宇航员头盔被击碎而死。刘培强在主控室得知，木星引力引力激增发生时，MOSS在三号紧急预案启动后的0.42秒後即推算得出“流浪地球计划”失败，启动携带人类文明记录逃离的“火种计划”。地面上的刘启等人决定修改发动机程序，用苏拉威西三号转向发动机点燃木星大气，刘培强说服联合政府为王磊小队打开全球广播呼唤支援，并共享了点燃木星方案。三座行星发动机实行了点燃木星方案，但它们的喷射流距木星大气仍有五千公里而無法點燃。刘培强向联合政府申请牺牲空间站，MOSS随即屏蔽空间站通讯。刘培强關閉消防系統並以酒烧毁MOSS的核心节点讓其失效，以夺得空间站控制权，之並释放空间站的休眠仓，经联合政府同意，刘培強犧牲自己手動驾驶空间站进入喷射流。空间站的爆炸点燃了木星大气，地球成功被推离木星。

在前期制作阶段，郭帆首先需考虑电影的重心和与原著的关系，并指出「小说我们可以选择不同的视角，比如刘慈欣经常会选上帝视角、宇宙视角看待人类，时间线很宏大，跨越几千年。但电影很难去表达这个，电影如果花两小时表达详尽的世界观，就变成科教片了」。所以他认为电影只能将小说故事作为背景，并须把故事核心放在人物和情感上。在这基础上，剧组开始考虑原著哪一部分更适合作为电影背景。对于这一点，郭帆强调「电影需要具象化的呈现，必须要找一个能够让观众觉得很有意思的对比，或者能有视觉冲击力的画面」。最后，他决定将通过[[木星|木星]][[引力加速|引力加速]]到逃逸的故事作为电影背景，并最终将这一个在原著中仅仅只占约五段话篇幅的故事进行几乎完全的改编，作为故事主体。选择木星的原因也正是因为其视觉冲击力：地球对人类来讲十分庞大，但当把我们的家园和体积是地球一千三百多倍的木星比较时，就变得非常渺小、特别脆弱。<ref name="jiemian-interview" />

在确定木星这个背景后，剧组即开始考虑世界观，这也是郭帆认为筹备环节中最艰难一部分。整个世界观的设定耗时八个月，从社会层面的政治、经济、文化、教育、娱乐、饮食，到自然科学层面的地球物理环境的变化、地球与其他星体的关系、如何脱离太阳，剧组都进行了相关的考虑，更邀请不少[[中国科学院|中国科学院]]的学者和教授共同探讨，确定能够建立一套严谨的世界观。同时，剧组也编写了一部一百年的[[编年史|编年史]]，包括自2017年起一百年间的大事，将现实世界和电影世界串联在一起。<ref name="jiemian-interview" />除文本外，三百多人的概设团队和美术团队亦历经了十五个月的设计、绘制、规划和搭建，制作并呈现了三千张概念设计——包括行星发动机、地下城、运载车等所有场景的细节构思——和八千多张分镜头画稿。<ref name="mtime-kaiji">{{cite web |title=刘慈欣又一科幻神作将搬上银幕 《流浪地球》青岛开机 《同桌的你》导演执导 |url=http://news.mtime.com/2017/05/31/1569801.html |website=[[时光网|时光网]] |accessdate=2018-11-02 |language=zh-hans |date=2017-05-31 |archive-url=https://web.archive.org/web/20181103131144/http://news.mtime.com/2017/05/31/1569801.html |archive-date=2018-11-03 |dead-url=no }}</ref><ref name="mtime-teji">{{cite web |title=“做科幻电影不仅仅是没有钱这么简单” 《流浪地球》曝科幻特辑 揭秘筹拍四年幕后 |url=http://news.mtime.com/2018/11/14/1586116.html |website=[[时光网|时光网]] |accessdate=2018-11-14 |language=zh-hans |date=2018-11-14 |archive-url=https://web.archive.org/web/20181114100512/http://news.mtime.com/2018/11/14/1586116.html |archive-date=2018-11-14 |dead-url=no }}</ref>
<!--《流浪地球》的剧本在创作的近三年中经历了五十多稿的调整和修改。-->

《流浪地球》将是原著作者[[刘慈欣|刘慈欣]]第一部真正从文字到影像的作品，而在电影几乎快拍摄完成时，郭帆才第一次和刘慈欣见面讨论<ref name="jiemian-interview" />。刘慈欣表示相当包容电影改编，并对其寄予厚望。他称：“电影和小说是很不一样的，它们遵循的艺术规律完全不同，我认为电影主要创作者应该是导演、编剧、制片人，作为原作者应该给予他们极大的信任与自由。”<ref name="huanqiu-dingdang" />

=== 后期制作 ===
