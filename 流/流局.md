'''-{流局}-'''（又称'''-{荒牌}-'''），[[麻将|麻将]]术语，是除了和牌以外牌局的结束。

== 荒牌的规则 ==
每次牌局下[[墙牌|墙牌]]（未摸的牌）只剩7墩（14张）的情况下，必须留下而不能继续摸，若每多[[杠_(麻将术语)|杠]]一次则多留一张。这种情况下如果无人和牌，就需要重新开始一轮牌局，庄家继续坐庄。

== 日本麻将的流局 ==
===通常情況下的流局===
当除了14张王牌以外的所有壁牌（山）都已经被摸打完，而又没有人和牌的情况下，即构成{{lang|ja|荒牌平局（ホワンパイピンチュー、こうはいへいきょく）}}，也被简称为荒牌。不过在发生{{nihongo|[[流局满贯|流局满贯]]|流し満貫（ながしマンガン）}}的情况下一般不认为是荒牌而是当作自摸和牌处理。

===中途流局===
在中途若是達成某些條件，直接該局流局。此時不計算未聽牌的罰符，並且下局直接親家連莊。
==== 九種九牌 ====
{{nihongo|'''九種九牌'''|キューシュキューハイ}}，又稱為{{nihongo|九種么九牌倒牌|キューシュヤオチューハイトウハイ}}。是親家或閒家摸到第一張自摸牌時，手牌有九種（含）以上不同的么九牌。和[[雙立直|雙立直]]、[[地和|地和]]……等一樣，在第一次自摸之前，有吃、碰、杠發生的話就不成立。

有部分的麻將玩家拿到九種九牌不會直接倒牌，而是直接湊成[[國士無雙|國士無雙]]或做流局滿貫。

也有部分閒家會利用九種九牌用來阻擋前面的閒家或親家的[[二重立直|雙立直]]聽牌。

==== 四風子連打 ====
親家和閒家的第一張捨牌均為相同風牌時成立。在第一巡若有任何人吃、碰、杠的話就不成立。
起源來自某些麻將規則裡裡西風連打（一路歸西）的規則，若是四家都打出{{牌画|西}}的話，會被稱做「四歸西」，而「歸西」就是「上[[西天|西天]]」的意思，而「上西天」也有著「送死」的意思，若是四個玩家皆打出{{牌画|西}}，對於迷信兆頭的玩家來說，為了避免壞兆頭通常會直接流局重洗，後來就衍生成了四風連打的規則。

在日本麻將裡，則是被日本人解釋成「四」與「死」是同音的，會使日本人會認為「四風連打」看起來就像是「死風連打」，所以也同樣會因為兆頭而直接流局。


雖然在牌桌上不太常見，但有時四風連打也會被北家利用拿來阻斷前三家的[[二重立直|雙立直]]聽牌。

==== 四開槓 ====
同一局若有兩名（含）以上的玩家進行過槓，並且槓的次數為四次時成立（由於嶺上牌只有四張，所以一局最多只能槓四次。而若由同一名玩家進行四次槓，則算作[[四槓子|四槓子]]）。
有幾種情況及時有過四次槓依然不算四開槓：
* 第四次槓後直接[[嶺上開花|嶺上開花]]自摸。
* 第四次槓時被[[搶槓|搶槓]]。
* 第四次杠完打出的牌被荣和

==== 四家立直 ====
四名玩家均進行立直時成立，但若是最後一名玩家進行立直時的捨牌放槍的話則不成立。<br/>
對於講究技巧的日本麻將，若是四家通通立直就變成了只能摸打的狀態，換言之，就是想要和牌只能靠運氣，再加上與四風連打的原因相同，四家立直聽起來就像「死家立直」，一般來說遇見這種情況都會直接強制流局。<br/>
部分地區以及部分的日本麻將網路遊戲沒有四家立直的規則，在無此規則下，若四名玩家均進行立直該局繼續進行。<br/>
四家立直成立后，四名玩家皆应展示手牌。如果有玩家存在虚假立直，则按[[犯规_(日本麻将)|犯规]]处理。

==== 三家和 ==== 
某一玩家的捨牌被其他三名玩家和牌時成立。
也有沒有三家和的規則，若有{{nihongo|頭跳|頭跳ね}}的規則，則由上家和牌；若沒有頭跳規則，則三名玩家均能和牌，此時又被稱之為{{lang|ja|トリプルロン}}。

若是沒有頭跳與三家和的規則，則和牌點數照常計算，而收取立直棒的規則則遵照頭跳規則。
