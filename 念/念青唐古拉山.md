'''念青唐古拉山脉'''（{{bo|z=གཉན་ཆེན་ཐང་ལྷ་|w=gnyan chen thang lha|p=Nyainqêntanglha|t=Nyenchenthanglha}}，意为{{fact|“灵应草原神”}}）位于[[西藏|西藏自治区]]中东部。近东西走向。西侧[[冈底斯山脉|冈底斯山脉]],东侧[[横断山脉|横断山脉]]。全长1400公里，平均宽80公里。平均海拔5000～6000米。

== 地理 ==
[[File:Nyainqentanglha_from_Qingzang_Railway.jpg|thumb]]所見的念青唐古拉山脉]]
念青唐古拉山脉是[[雅鲁藏布江|雅鲁藏布江]]与[[怒江|怒江]]分水岭。也是高原上寒冷气候带与温暖（凉）气候带的分界线。主峰[[念青唐古拉峰|念青唐古拉峰]]海拔7162米。

== 地质 ==
山脉形成于燕山运动晚期，为一系列向东逆冲的褶皱山带，沿山带南侧均有深大断裂通过。

== 冰川 ==
念青唐古拉山脉有7080条冰川，总面积达10,701平方公里，在中国的几大山脉中冰川面积排名第二。山脉西段降水少，冰川属于亚大陆型冰川，而山脉东段受印度洋西南季风影响降水多，雪线海拔低，是中国最大的海洋型冰川地区，冰川分布集中，冰川数占了整个山脉冰川总数的2/3，冰川面积占了整个山脉冰川总面积的5/6，也是地球上中低纬地区冰川作用最强的中心之一。<ref>{{Cite magazine | url=http://www.dili360.com/cng/article/p5350c3d68ab0815.htm | title=念青唐古拉 | magazine=中国国家地理 |date=2011年|issue=第1期 | author=李炳元}}</ref>

其中有27条冰川长度超过10公里。如易贡八玉沟的[[卡钦冰川|卡钦冰川]]长达33公里，冰川末端海拔仅2530米，为西藏最大冰川，也是中国最大的海洋性冰川。

== 气候 ==
山脉东段受印度洋西南季风影响，降水多；山脉西段位于半干旱气候地区，降水少。　

==宗教信仰==
[[苯教|苯教]]信仰的山神主要有4位：雅拉香波（{{lang-en|Yarlha Shampo}}）山神、库拉卡日（{{lang-en|Kula Kari}}）山神、诺吉康桑山神、念青唐古拉山神，分别居住于西藏的东南西北四方。四大山神加上其他五位山神是“世界形成之九神”。<ref>{{Cite web | url=http://xz.people.com.cn/GB/139192/9418231.html | title=世间护法神念青唐古拉|author=三木才 （藏族）|date=2009年6月5日}}</ref>
