'''子辰'''（洋名：'''Anthony Tsang'''，{{bd|1980年|4月4日|}}），生于[[英属香港|英属香港]] ，祖籍[[山东|山东]][[临沂市|临沂市]]，籍贯[[中国|中国]][[香港|香港]]，现于数家私募基金任职董事总经理，现任道和环球集团(香港上市公司代码：00915) 大股东之一<ref>{{Cite web|url=http://www.21cbr.com/article/75732.html|title=道和环球集团投资人子辰（Anthony Tsang）：“资本玩家”的投资门道|accessdate=2018-11-20|author=|date=|work=21世纪商业评论|publisher=}}</ref>。

== 家庭 ==
=== 父母 ===
父亲是一名在香港经营自己诊所的医生，于1997年病逝，母亲为一名金融界别人士。
=== 兄弟姐妹 ===
一弟

== 任职 ==

前任道和环球集团（香港上市公司)执行董事<ref>{{Cite web|url=https://www.sohu.com/a/152736893_403354|title=子辰：在香港除了腾讯、美图，还有“对面”|accessdate=2018-11-20|author=|date=2017-06-28|work=搜狐网|publisher=}}</ref>

前任中国北方金融集团(香港上市公司)中国区总裁

前任天胤集团控股有限公司(美国上市公司)总裁

== 受邀演讲及社会服务 ==

子辰曾出席CCTV赢在中国活动嘉宾、阿里巴巴直播采访嘉宾、中国小姐评委等<ref name="multiple2">{{cite web |url=http://ent.ifeng.com/zz/detail_2012_10/10/18149926_0.shtml|title=2012中国小姐评委子辰：不会因为漂亮而加分. |accessdate=2012年10月10日|work=|publisher=凤凰网娱乐|language=zh-hans}}</ref>；同时也受邀到国内及香港等高等学府讲学<ref name="multiple1">{{cite web|url=http://zj.sina.com.cn/news/xfgz/2015-07-02/1640269797.html|title=子辰:A股会让你跳楼吗|accessdate=2015年7月2日|work=|publisher=新浪网财经频道|language=zh-hans}}</ref>。
