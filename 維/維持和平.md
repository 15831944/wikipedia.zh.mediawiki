'''維持和平'''（{{lang-en|Peacekeeping}}）的活動，旨在創造有利於維持持久[[和平|和平]]的條件。經研究發現，'''維和'''能減少[[戰場|戰場]]上[[平民|平民]]的死亡，並降低再次引發[[戰爭|戰爭]]的風險。<ref name=":0">{{cite web|website=United Nations Peacekeeping |url=http://peacekeeping.un.org/en/department-of-peacekeeping-operations |title=Department of Peacekeeping Operations(DPKO) |publisher=United Nations Peacekeeping |date= |accessdate=2012-10-03 |archiveurl = https://web.archive.org/web/20171212123531/http://peacekeeping.un.org/en/department-of-peacekeeping-operations |archivedate = 2017-12-12 |deadurl =no}}</ref><ref>{{cite web|url=https://www.un.org/en/peacekeeping/|title=United Nations Peacekeeping|publisher=|accessdate = 2014-10-23 |publisher=United Nations Peacekeeping |website=United Nations Peacekeeping |archiveurl = https://web.archive.org/web/20171212122436/https://peacekeeping.un.org/en |archivedate = 2017-12-12 |deadurl = no}}</ref><ref>{{cite web |url = http://www.un.org/zh/peacekeeping/operations/peacekeeping.shtml |accessdate = 2017-12-12 |website = 聯合國維持和平 |publisher = [[聯合國|聯合國]] |title = 什么是维持和平? |language = zh-hans |archiveurl = https://web.archive.org/web/20171212123138/http://www.un.org/zh/peacekeeping/operations/peacekeeping.shtml |archivedate = 2017-12-12 |deadurl = no}}</ref>
