'''維景灣畔'''（{{lang-en|'''Ocean Shores'''}}）是[[香港|香港]][[新界|新界]][[西貢區|西貢區]]其中一個最大的私人屋苑，位於[[調景嶺|調景嶺]]。維景灣畔設有17座（但不設第4及14座，故合共15座），合共提供5,728個住宅單位。屋苑由[[啟勝管理|啟勝管理]]負責管理。

==簡介==
維景灣畔地段屬新界西貢區[[調景嶺|調景嶺]]，屋苑前身為[[紹榮鋼鐵|紹榮鋼鐵廠]]。[[太古地產|太古地產]]與[[新鴻基地產|新鴻基地產]]合作，以各佔一半權益的方式，在1997年以127億元收購調景嶺紹榮鋼鐵的地皮，及後再就地皮補價58億元，總投資額達250億元<ref>{{cite news|title=賣地一鋪勁賺　70億|url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20060718&sec_id=4104&subsec_id=11867&art_id=6137357|accessdate=2011-01-23|newspaper=《[[蘋果日報_(香港)|蘋果日報]]》|date=2006-07-18}}</ref>。以可建樓面約400萬方呎計算，每呎發展成本約6,000元。屋苑佔地逾500,000平方呎，三期共建15座住宅，提供逾5,000個單位。大廈以單列排開的設計。屋苑設有住客會所、水上樂園、園藝花園及兒童遊樂設施等。由於維景灣畔由兩大地產商發展，用料高質，會所設施完善，管理優異，鄰近港鐵站之餘同時亦自成一角，背山面海，提供非常優質的居住環境，住戶亦可經[[澳景路|澳景路]]離開將軍澳區直達油塘(駕駛者可以通過此路前往東區海底隧道及市區)，而無需使用將軍澳隧道，其樓價在[[將軍澳|將軍澳]]區內屬龍頭屋苑，截至2018年10月呎價高達港幣18,000元。現時維景灣畔的住戶多為中產或更高收入家庭。

== 購物 ==
維景灣畔一期有一間[[超級市場|超級市場]]；三期有一個約3.3萬方呎小型商場，設有[[診所|診所]]、[[美容院|美容院]]、洗衣店及補習學校。維景灣畔居民多到附近商場，例如[[都會駅|都會駅]]、[[彩明商場|彩明商場]]和[[將軍澳市中心|將軍澳市中心]]購物。

商場早年由發展商拆售，先後曾數度轉手，其中，王夏陽於2013年3月以1.48億元購入上述商場樓面，呎價5,230元，現分成13個舖位出租，回報約6厘。

2013年11月維景灣畔Fusion新店開幕，位於第一期商場LG1層。

== 交通配套 ==
屋苑設有行人天橋連接[[港鐵|港鐵]][[調景嶺站|調景嶺站]]上蓋的[[城中駅|城中駅]]商場，因此居民主要利用港鐵往返。由於調景嶺站是港鐵[[將軍澳線|將軍澳線]]及[[觀塘線|觀塘線]]的轉車站，因此前往[[九龍|九龍]]或[[香港島|香港島]]均比將軍澳區其他屋苑方便。

屋苑有一條非專用通路<ref>http://www.landsd.gov.hk/en/legco/pgicos/SaiKung.pdf</ref>（[[澳景路|澳景路]]）通往藍田[[碧雲道|碧雲道]]。駕駛者可以通過此路前往[[東區海底隧道|東區海底隧道]]及市區。

維景灣畔一期較遠離調景嶺站，居民須步行5至15分鐘前往。

== 環境 ==
屋苑建於沿岸位置，景色優美，自成一角，居住環境清靜。鄰近海濱長廊，動靜皆宜。

== 治安 ==
保安嚴格，管理公司盡責查核探訪者身份，有效杜絕不明外來人士。

== 廣告特色 ==
在維景灣畔第三期推出之時，由於單位由太古地產及新鴻基地產合資興建，廣告上都刻意強調這特別的關係，例如利用攣生幼兒作為[[廣告|廣告]]角色，並加上大量[[疊字|疊字]]，如：合合拍拍、便便利利之類。
