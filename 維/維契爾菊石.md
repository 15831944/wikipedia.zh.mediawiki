'''維契爾菊石'''（[[學名|學名]]：{{lang|la|'''''Witchellia'''''}}）是生存在[[中侏羅世|中侏羅世]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]。其[[化石|化石]]較為罕見，分布於[[美國|美國]]、[[澳大利亞|澳大利亞]]、[[西班牙|西班牙]]和[[突尼西亞|突尼西亞]]等地<ref name="Fossilworks">{{Cite web |url = http://fossilworks.org/bridge.pl?a=taxonInfo&taxon_no=258008 |title = Fossilworks: Witchellia |language = en |accessdate = 2018-08-06 }}</ref>。牠們和同樣很罕見的[[佩勒寇德菊石|佩勒寇德菊石]]（''Pelekodites''）及{{tsl|en|Dorsetensia|多塞特菊石}}有親緣關係。<ref name="Witchellia">{{Cite web |url = http://liusir.pixnet.net/blog/post/390438074 |title = Witchellia 維契爾菊石 (英國 skye島) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-06 }}</ref>

== 特徵 ==
維契爾菊石的殼體扁平，側面的肋條較平滑而不突出，中央的臍部較小，腹側稍平坦，兩條縱向的溝形成於龍骨兩側。<ref name="Witchellia" />
