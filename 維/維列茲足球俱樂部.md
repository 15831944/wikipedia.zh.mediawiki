'''維列茲足球俱樂部'''（''Fudbalski Klub Velež''）是[[波黑|波黑]]的一家[[足球|足球]]俱樂部。球隊的主場位於[[莫斯塔爾|莫斯塔爾]]。設立于1922年6月22日<ref>{{cite web
 |publisher = FK Velez Mostar
 |title = FUDBALSKI KLUB "VELEŽ", MOSTAR
 |url = http://www.fkvelez.ba/o_klubu.php
 |accessdate = 2008-04-22
 |archiveurl = https://web.archive.org/web/20080329091349/http://www.fkvelez.ba/o_klubu.php
 |archivedate = 2008-03-29
 |deadurl = yes
 }}</ref>。球隊的名稱則是來自于附近的山峰。俱樂部目前的主場是Vrapčići球場，過去的主場是Bijeli Brijeg球場。球隊目前參加[[波赫超級足球聯賽|波赫超級足球聯賽]]的賽事。球隊在南斯拉夫時期得到了來自塞爾維亞人、波斯尼亞人和克羅埃西亞人三個主要民族的廣泛支持。波黑獨立之後維列茲的主要支持者則是波斯尼亞人。維列茲也參加過聯盟杯的賽事。
