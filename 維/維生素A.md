'''维生素A'''又称'''維他命A'''、'''抗干眼病维生素'''，是人類的必需[[營養素|營養素]]之一。維生素A的[[前体|前体]]是存在于多种植物中的[[胡萝卜素|胡萝卜素]]。

維生素A並非單一的一種化合物，而是有許多不同的型態。动物能将胡萝卜素在体内转化为维生素A贮藏在[[肝脏|肝脏]]中；通常是以[[醇|醇]]類的方式存在，稱作[[視黃醇|視黃醇]]，活性也是最高；但也有一些屬於[[醛|醛]]類，稱作[[視黃醛|視黃醛]]；另外還有一些屬於[[酸|酸]]類，稱作[[視黃酸|視黃酸]]。

視黃醇與視黃醛主要掌管桿細胞的視覺循環，而視黃酸主要是掌管人體內上皮組織分化有關，因此有些視黃酸衍伸物（俗稱的A酸）常用於皮膚疾病上的治療，另外有一種稱作視黃酯，其為人體內儲存脂溶性維他命A的主要型式。

== 維生素A的發現 ==
維生素A的發現可以追溯至1906年，當時的研究証實除[[碳水化合物|碳水化合物]]、[[蛋白質|蛋白質]]及[[脂肪|脂肪]]以外，另有一些因素以維持牛隻的健康。1917年，其中一種物質被威斯康辛大學麥迪遜分校的Elmer McCollum及[[耶魯大學|耶魯大學]]的Thomas Osborne發現。由於當時[[水溶性|水溶性]]的[[維生素B|維生素B]]剛被發現，研究員決定將名字改為「脂溶性維生素A」（維生素A）。

==作用==
維生素A是[[视网膜|视网膜]]内感光色素的组成部分。是保護眼晴和增進視力不可缺少的營養素，若缺乏的話，會引致結膜的表皮角質化，這會阻塞淚腺，從而導致乾眼症，甚至引致結膜炎。維生素A不足會引發[[夜盲症|夜盲症]]，因為視網膜內感光色素缺乏之故。

維生素A也是骨髓细胞分化时的调节因素，包括骨髓中的造血細胞都需要維生素A的調節

[[上皮组织|上皮组织]]分化亦需維生素A。美容業使用的蝦紅素，即是維生素A家族成員之一。

=== 補充可提高吸收利用的營養素 ===
膽鹼、必需脂肪酸、鋅、維生素C、D、E

== 颜色反应 ==
维生素A可以与多种含氯化合物在[[氯仿|氯仿]]中反应，生成带有颜色的溶液，这种颜色反应可以用于元素定性分析或油脂中维生素A的检测。常见的能与维生素A发生颜色反应的化合物有：
# <chem>SbCl3</chem>与维生素A反应显示的蓝色可以在三分钟以内不褪去。
# <chem>SnCl4</chem>最初可以得到深蓝色溶液，随后溶液迅速变为紫色。
# <chem>FeCl3</chem>使溶液显出带有荧光的紫红色，但由于该体系难以保持无水，故多次进行反应时可能会得到不一样的结果。
# 向维生素A的油溶液中加入极细的<chem>AlCl3</chem>粉末，溶液中显示紫红色，随后转变为棕色。但当溶于氯仿的油中含有干燥的氯化氢或光气且加入的<chem>AlCl3</chem>为细粉末时会得到持续时间相近的紫色溶液。
# 当<chem>SiCl4</chem>加入维生素A的鱼肝油溶液中溶液呈现玫瑰红色，但该反应分辨率不高。
# 与<chem>POCl3</chem>得到的溶液为蓝色，之后迅速转为红色。
以上反应得到的溶液其颜色均为蓝色再加上不同比例的红色，Rosenheim、 Drummond [1925] 和 Takahashi ''et al.'' [1925]认为蓝色是由维生素A结构导致的，而较浅的红色、黄色被掩蔽。<ref>{{Cite journal|title=Nutrition classics from the: Biochemical Journal 20: 497-501, 1926. LXIV. Colour reactions attributed to vitamin A. By Francis Howard Carr and Ernest Arthur Price|url=https://www.ncbi.nlm.nih.gov/pubmed/4587544|date=October 1973|journal=Nutrition Reviews|issue=10|volume=31|pages=312–313|issn=0029-6643|pmid=4587544}}</ref>
