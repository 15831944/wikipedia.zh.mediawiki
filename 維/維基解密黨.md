'''維基解密黨'''（[[英語|英語]]：'''WikiLeaks Party'''）是2013年成立的[[澳洲|澳洲]][[政黨|政黨]]，創立目的是為了[[維基解密|維基解密]]創辦人[[朱利安·阿桑奇|朱利安·阿桑奇]]參選[[澳洲參議員|澳洲參議員]]。

== 歷史 ==
=== 創辦新政黨 ===
2012年3月，朱利安·阿桑奇宣布即將競選2013年澳洲參議員選舉<ref name=ABC>{{cite web|title=Julian Assange to run for Senate|url=http://www.abc.net.au/news/2012-03-17/julian-assange-to-run-for-senate/3895958|publisher=ABC News|accessdate=7 March 2013}}</ref>，並在稍後宣布將成立一個新政黨。<ref name=RT>{{cite web|title=Assange to run for Australian Senate, start Wikileaks party|url=http://rt.com/news/juilan-assange-wikileaks-senate-900|publisher=rt.com|accessdate=7 March 2013}}</ref>

為了組織成立新政黨，澳洲國內的行動人士John Shipton、Omar Todd等人一起加入協助，並於2013年4月正式取得500位繳費黨員資格，得以向澳洲選舉委員會登記。<ref>{{cite web |url=http://www.digitaljournal.com/article/347434 |title=WikiLeaks Party raises senate bid, expands to two more states|last1=Sewell |first1=Anne|date=7 April 2013 |work=[[Digital_Journal|Digital Journal]]|accessdate=7 April 2013}}</ref>該黨也宣布將在三個省份派出候選人，包括朱利安·阿桑奇。

=== 朱利安·阿桑奇 ===
朱利安·阿桑奇為避免因為[[性侵|性侵]]官司引渡回[[瑞典|瑞典]]後又因為政治因素而被[[引渡|引渡]]至[[美國|美國]]受審洩漏機密文件案，2012年6月19日逃入{{Link-en|厄瓜多駐英國大使館|Embassy of Ecuador, London}}尋求[[政治庇護|政治庇護]]，而同年8月16日[[厄瓜多|厄瓜多]]官方批准他的請求。阿桑奇若踏出厄瓜多大使館，則會立刻遭到英國警方逮捕，因此他等同於被困在大使館裡。他將在[[維多利亞省|維多利亞省]]參選，若成功當選參議員，仍否返國將成為英、澳、厄三國的外交爭議；若到時阿桑奇仍不能回到澳洲就職，維基解密黨也準備好候補人選以代替阿桑奇職位。<ref>[http://news.rti.org.tw/index_newsContent.aspx?nid=416105 維基解密黨公布競選大計(台灣央廣)]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://www.epochtimes.com/b5/13/4/7/n3840587.htm 維基解密澳洲組黨，將競選三省參議員(台灣大紀元時報)]</ref>
