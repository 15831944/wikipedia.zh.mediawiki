'''維拉·瑪提'''（[[塞尔维亚文|塞尔维亚文]]：'''Веран Матић'''，{{bd|1962年|}}<ref name=IPI>{{cite web |url=http://www.freemedia.at/awards/veran-matic.html |title=Veran Matic, Serbia |publisher=International Press Institute |accessdate=14 August 2012 |archiveurl=https://www.webcitation.org/69ukyLhuA?url=http://www.freemedia.at/awards/veran-matic.html |archivedate=2012年8月14日 |deadurl=yes }}</ref>）是被[[英國廣播公司|英國廣播公司]]新聞稱為“南斯拉夫獨立廣播的先驅”的[[貝爾格萊德|貝爾格萊德]]廣播電台[[B92|B92]]的[[行政總裁|行政總裁]]。<ref name=BBC185>{{cite web |url=http://news.bbc.co.uk/2/hi/europe/754340.stm |title=B92: Belgrade's impartial voice |date=18 May 2000 |publisher=BBC News |archivedate=2012年8月14日 |archiveurl=https://www.webcitation.org/69umXIrgZ?url=http://news.bbc.co.uk/2/hi/europe/754340.stm |accessdate=14 August 2012 |deadurl=yes }}</ref>

於1993年，瑪提獲得[[保護記者委員會|保護記者委員會]]頒發[[国际新闻自由奖|国际新闻自由奖]]。<ref name="CPJ"/>於2000年，[[國際新聞協會|國際新聞協會]]把他列為[[世界新聞自由英雄|世界新聞自由英雄]]之一。

==職業生涯==
瑪提畢業於[[贝尔格莱德大学|贝尔格莱德大学]]，並獲得一個世界文學學士學位。他於1984年首次參與獨立青少年媒體，並於1989年5月與薩薩·米爾科維奇共同創立了獨立廣播電台[[B92|B92]]。<ref name=IPI/><ref name=BBC185 />此電台播放各種音樂並提供時事報道，特別是對總統[[斯洛博丹·米洛舍維奇|斯洛博丹·米洛舍維奇]]的批評。<ref name=BBC185 />因此，南斯拉夫政府採取各種行動妨礙或禁止B92進行廣播。<ref name=BBC243 />於1996年11月，B92被禁止廣播。作為回應，B92進行網上廣播。<ref name=BBC302344/>

雖然B29的敵人是[[斯洛博丹·米洛舍維奇|斯洛博丹·米洛舍維奇]]的政府，但是瑪提仍然反對1990年[[北約轟炸南斯拉夫|北約轟炸南斯拉夫]]的行動。於三月底，他於B92的網站上發表了一份名為《轟炸嬰兒和其洗澡水》的聲明。聲明中，他指出「北約正在履行自己末日的預言：每枚擊中地面的導彈加劇了北約本應防止的人道災難」。<ref>{{cite web |url=http://news.bbc.co.uk/2/hi/europe/310452.stm |title=Serbia closes B92 radio station |date=2 April 1999 |publisher=BBC News |archivedate=2012年8月14日 |archiveurl=https://www.webcitation.org/69un3KT1R?url=http://news.bbc.co.uk/2/hi/europe/310452.stm |accessdate=14 August 2012 |deadurl=yes }}</ref>不久，米洛舍維奇政府以「傳輸強度超過允許水平」的理由禁止電台廣播。馬提被帶走調查，而B92的工作人員則被禁止使用自己的手機或電腦。<ref name=BBC243>{{cite web |url=http://news.bbc.co.uk/2/hi/world/monitoring/302624.stm |title=Belgrade bans independent B92 |date=24 March 1999 |publisher=BBC News |archivedate=2012年8月19日 |archiveurl=https://www.webcitation.org/6A2VozeaH?url=http://news.bbc.co.uk/2/hi/world/monitoring/302624.stm |accessdate=19 August 2012 |deadurl=yes }}</ref>B92的工作人員指出「他們禁止我們廣播的真正原因是我們讓人們了解現在到底發生了什麼事」。而面對著禁令，B92仍然繼續通過衛星電台和互聯網進行直播。<ref name=BBC302344>{{cite web |url=http://news.bbc.co.uk/2/hi/europe/302344.stm |title=Radio B92 - off the air but on the Net |date=24 March 1999 |publisher=BBC News |archivedate=2012年8月19日 |archiveurl=https://www.webcitation.org/6A2jYsqnc?url=http://news.bbc.co.uk/2/hi/europe/302344.stm |accessdate=19 August 2012 |deadurl=yes }}</ref>

同年4月2日，B29的管理權被轉移到米洛舍維奇的[[塞爾維亞社會黨|塞爾維亞社會黨]]手中。當B92的記者拒絕配合新的管理者時，他們就被解僱。因此，瑪提創立了另一個名為B2-92的獨立電台。於2000年10月，B29與B2-92合併，而瑪提亦重新獲得B29的管理權。<ref name=IPI/>

==獎勵和表彰==
於1993年，瑪提獲[[保護記者委員會|保護記者委員會]]頒發[[国际新闻自由奖|国际新闻自由奖]]。<ref name="CPJ">{{cite web|url=http://www.cpj.org/attacks96/frontmatter/ipfa.html|title=Journalists Receive 1996 Press Freedom Awards|publisher=Committee to Protect Journalists|archivedate=2012年8月11日|archiveurl=https://www.webcitation.org/69pT569fI?url=http://www.cpj.org/attacks96/frontmatter/ipfa.html|accessdate=11 August 2012|deadurl=yes}}</ref>五年後，瑪提、塞納·碧根尼和[[維克多·伊凡銑克|維克多·伊凡銑克]]代表所有前南斯拉夫獨立媒體接受[[奧洛夫·帕爾梅獎|奧洛夫·帕爾梅獎]]。<ref>{{cite web |url=http://www.palmefonden.se/index.php?sid=1&pid=25&tid=58 |title=The Olof Palme Prize 1998 |publisher=Olof Palme Fund |archivedate=19 August 2012 |archiveurl=http://www.webcitation.org/6A2zPBu4L |accessdate=19 August 2012}}</ref>

於2000年，[[國際新聞協會|國際新聞協會]]把瑪提列為[[世界新聞自由英雄|世界新聞自由英雄]]之一。國際新聞協會稱讚他和B92人員「能夠面對塞爾維亞政府施壓，並對在其區域內發生的悲劇提供準確的、中立的觀點」。<ref name=IPI />於2009年，瑪提獲[[法國政府|法國政府]]亦頒發[[榮譽勳章|榮譽勳章]]，並把他稱為「一個為新聞自由而戰的優秀記者」。<ref>{{cite web |url=http://www.b92.net/eng/news/society-article.php?yyyy=2009&mm=10&dd=22&nav_id=62531 |title=Đinđić, Matić awarded Legion of Honor |date=22 October 2009 |publisher=B92 |archivedate=19 August 2012 |archiveurl=http://www.webcitation.org/6A2vzH996 |accessdate=19 August 2012}}</ref>

於2011年，塞爾維亞雜誌《你好》的讀者把瑪提選為「年度最人道的名人」。在過去的一年裡，瑪提創立的B92基金資助了塞爾維亞醫院提升嬰兒科設備，並把這次行動稱為「為嬰兒戰鬥」，因此獲塞爾維亞大眾稱讚。<ref>{{cite web |url=http://www.b92.net/eng/news/society-article.php?yyyy=2011&mm=12&dd=12&nav_id=77747 |title=Magazine declares B92 Fund head "most humane celebrity" |publisher=B92 |date=12 December 2011 |accessdate=19 August 2012 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>於2012年11月，瑪提獲頒維特茲新聞終身成就獎。<ref>{{cite web |url=http://www.b92.net/eng/news/society-article.php?yyyy=2012&mm=11&dd=10&nav_id=83099 |title=B92 wins five Press Vitez awards" |publisher=B92 |date=10 November 2011 |accessdate=13 November 2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20121115091007/http://www.b92.net/eng/news/society-article.php?yyyy=2012&mm=11&dd=10&nav_id=83099 |archivedate=2012年11月15日 }}</ref>同年12月，塞爾維亞經理協會把瑪提評為「最佳經理」。<ref>{{cite web |url=http://www.tanjug.rs/news/70767/best-managers-awarded.htm |title=Best managers awarded" |publisher=Tanjug |date=25 December 2012 |accessdate=25 December 2012 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>同月，瑪提獲UEPS頒發新理念獎。<ref>{{cite web |url=http://www.b92.net/eng/news/society-article.php?yyyy=2012&mm=12&dd=28&nav_id=83882 |title=B92 Fund head Matić wins UEPS award" |publisher=B92 |date=28 December 2012 |accessdate=28 December 2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20130323230045/http://www.b92.net/eng/news/society-article.php?yyyy=2012&mm=12&dd=28&nav_id=83882 |archivedate=2013年3月23日 }}</ref>

於2013年2月，瑪提獲頒其中一個國家最高榮譽，斯雷滕勳章。頒獎者指出「瑪提推行人道主義援助活動「為嬰兒戰鬥」向塞爾維亞的公民提供了福利」。<ref>{{cite web|url=http://www.b92.net/eng/insight/pressroom.php?yyyy=2013&mm=02&nav_id=84715 |title=Veran Matić honored with Sretenje Medal" |publisher=B92 |date=28 February 2012 |accessdate=28 December 2012}}</ref>
