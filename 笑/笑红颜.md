{{engname|《笑紅顏》|Battle of the Beauty}} 是湖南经视文化传播有限公司继《大丫鬟》《宫》后推出的又一力作，这是国内首部民国宅斗大戏。该剧由《大丫鬟》的导演何澍培执导。由刘松仁、胡静、陈楚河、翁虹、戴春荣、白珊、岳跃、于毅、陆翊等人主演。是关于江南民国第一家族的宅斗大戏。上映时间：2012年4月22日在2012年5月在湖南卫视播出。[[爱奇艺|爱奇艺]]亦于2012年4月23日起在网络上播出。[[腾讯视频播放器|腾讯视频]]亦于2012年4月24日起在网络上播出。馬來西亞[[Astro全佳HD|Astro全佳HD]]在2013年3月2日下午二時以五集播出，並在每星期一至五19：00一集重播。其後改為2013年3月9日起下午一時五集播出，重播時間不變。廣東[[珠江频道|珠江频道]](粤语)2012年11月5日至11月21日播出。

該劇原由製作公司命名為《'''斗红颜'''》，開播前被湖南卫视要求更改為現名<ref>[http://life.jschina.com.cn/system/2012/04/22/013190082.shtml 《斗红颜》更名《笑红颜》，中国江苏网轉載自龙虎网，2012-04-22]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

== 花絮 ==
该剧的发布会上，[[胡静|胡静]]当场痛哭，她自己说是因为[[南洋|南洋]][[吉隆坡|吉隆坡]]老公和她吵架阻扰她参加发布会而造成的，并且许诺只在[[北京|北京]]逗留一下午晚上必须回吉隆坡才获得老公同意准许其参加发布会。<ref>{{cite web |url= http://ent.sina.com.cn/v/m/2012-04-25/18243615367.shtml|title=胡静宣传《笑红颜》飙泪 临行前与老公吵一架 |publisher=新浪网 |accessdate=2012年4月25日 |work= |language =}}</ref>晚上，胡静的微博发布其南洋家人不准她拍戏，但是她表示不会放弃演艺事业。<ref>{{cite web |url=http://www.cnr.cn/qxwtp/201204/t20120426_509512120.shtml |title=胡静泪洒舞台自曝与老公吵架 暗示不会放弃拍戏 |publisher=中国广播网 |accessdate=2012-04-26 |work= |language =}}</ref>

== 社会反映 ==
该剧播出后，遭到观众和网友的质疑，认为该剧直接原原本本地抄袭2006年电视剧《[[血色残阳|血色残阳]]》，只换了演员而已。<ref>{{cite web |url=http://www.chinanews.com/yl/2012/04-26/3848277.shtml |title= "笑红颜"被指抄袭 主创：不介意比对，是再创作|publisher=中国新闻网 |accessdate=2012年4月26日 |work= |language =}}</ref><ref>{{cite web |url=http://et.21cn.com/gundong/etscroll/2012/04/25/11607273.shtml |title=《笑红颜》抄袭《血色残阳》 主创：是再创作 |publisher=21cn |accessdate=2012-04-25 |work= |language= |archiveurl=https://web.archive.org/web/20161105230213/http://et.21cn.com/gundong/etscroll/2012/04/25/11607273.shtml |archivedate=2016-11-05 |deadurl=yes }}</ref>
