'''漳州市'''（{{lang-nan|Tsiang-tsiu-tshī}}）是[[中华人民共和国|中华人民共和国]][[福建省|福建省]]下轄的[[地級市|地級市]]。位于北緯23.6—25度，東經117—118度[[漳州平原|漳州平原]]上，鄰近[[厦門|厦門]]、[[汕頭|汕頭]]两个經濟特区，是福建最南端的一个地級市。全市土地面积1.29万平方公里，海域面积1.86万平方公里，总人口498万人，其中市区建成区面积64平方公里，常住人口64万人。1986年12月8日[[中华人民共和国国务院|国务院]]确定并公布为第二批[[国家历史文化名城|国家历史文化名城]]。

== 歷史 ==
{{main|漳州府}}
漳州是历史文化名城，早于一万年前就有先民在这片沃土上繁衍生息。[[秦始皇|秦始皇]]二十五年（前222年），列入秦中央版图，属[[闽中郡|闽中郡]]。汉代，以梁山（今地名）为界，分属南越、[[闽越|闽越]]，延至[[隋|隋]][[开皇|开皇]]十二年（592年）归于一统。唐高宗[[总章|总章]]二年（669年），闽南发生叛乱，[[嶺南|嶺南]]行軍[[總管|總管]][[陳政|陳政]]父子奉命平乱。此时，他们率领[[河南|河南]]光、申、蔡三州7000多名将士入漳，连同随军眷属，带来[[中原文化|中原文化]]。唐垂拱二年（686年）建州，迄今已有1300余年历史。

唐[[垂拱|垂拱]]二年（686年），[[陳政|陳政]]之子[[陳元光|陳元光]]疏言：“周官七閩，宜增為八，請建一州，泉、潮間”。陳獲旨得准，在[[泉州|泉州]]與[[潮州|潮州]]之間建有漳州，並受命嶺南行軍總管、世守漳州[[刺史|刺史]]。陳元光又疏部曲[[許天正|許天正]]、[[馬仁|馬仁]]等幹略，請授[[司馬|司馬]]等職。陳元光即率眾等在漳州剪除荊棘，營農積穀，奏立行臺，四境遂無枹鼓之聲。明朝時，漳州之[[月港|月港]]已經是闽南大镇，人货萃聚。<ref>朱纨《阅视海防事》：“泉州之安海、漳州之月港乃闽南大镇，人货萃聚。”（《明经世文集》卷205《朱中丞甓余集》）</ref>

民国初年属汀漳道，后撤道，各县直属省辖，又设立行政督察区，民国末时，今漳州市大部分地区属第五行政督察区。

1949年，解放军占领漳州，1950年成立龙溪专区，后改龙溪地区，为漳州市前身。

1951年，析龙溪县城关设县级漳州市。

1960年，龙溪县与海澄县合并为龙海县。

1985年，龙溪地区改设地级漳州市，原县级漳州市改设芗城区。

1993年，龙海县改设县级龙海市。

1996年，析芗城区和龙海市部分区域设龙文区。

此外，原属漳州市龙海市的[[东碇岛|东碇岛]]目前被台湾当局占领，划为金门县所属。
== 地理 ==
漳州為福建省最南端的[[地級市|地級市]]，與[[廣東省|廣東省]]為鄰。漳州西北多山，东南滨海，属南[[亚热带季风气候|亚热带季风气候]]，年平均温度21℃，年降雨量1500毫米。

漳州地处[[福建|福建]]东南。[[陆域|陆域]]南北长187公里，东西宽127公里，面积12607平方公里，介于北纬23°34′至25°15′、东经116°54′至118°08′之间。东邻福建省厦门市、东北与福建省泉州市接壤，西北与福建省龙岩市毗邻，西与广东省梅州、潮州两市交界，东南与台湾隔海相望。

[[博平岭|博平岭]]横亘于西北，[[戴云山|戴云山]]餘脉深入北部境内。[[平和县|平和县]]的[[大芹山|大芹山]][[主峰|主峰]]海拔1544.5米，为全市第一高峰。九龙江全长258公里，为福建第二大河。流域面积14741平方公里，在漳州境内流域面积7586平方公里。此外还有[[鹿溪|鹿溪]]、[[漳江|漳江]]、[[东溪|东溪]]等主要河流。[[九龙江|九龙江]]中下游平原面积720平方公里，是省内最大[[平原|平原]]。

== 語言 ==
{{main|漳州话|客家話}}
漳州通行[[閩南語|閩南語]][[漳州话|漳州话]]，語言學歸類上屬於[[閩南語|閩南語]]的[[泉漳片|泉漳片]]，保留較多[[中古漢語|中古漢語]]特色，擁有豐富[[文白异读|文白异读]]。西部毗鄰[[闽西|闽西]]地方，與[[客家地区|客家地区]]過渡，通行[[客家話|客家話]]。分佈點，如：[[詔安縣|詔安縣]]秀篆、官陂、葛霞、太平、紅星等鄉鎮、[[雲霄縣|雲霄縣]]、[[南靖縣|南靖縣]]、[[平和縣|平和縣]]的部分地區。但全市主體人口仍為閩南人，佔九成以上。

== 文化 ==
漳州文化生活豐富，並擁有大量國家級非物質文化遺產。漳州市的漳州[[木版年画|木版年画]]、漳州木偶头雕刻、[[歌仔戏|歌仔戏]]（薌劇）、漳州[[布袋木偶戏|布袋木偶戏]]、[[锦歌|锦歌]]、漳州蔡福美传统制鼓技艺、漳州八宝印泥、漳浦县的漳浦剪纸、龙海市的[[保生大帝|保生大帝]]信俗、华安县的高山族拉手舞及东山县的东山歌册被列入[[国家级非物质文化遗产名录|国家级非物质文化遗产名录]]。此外、漳州文化还包括有[[潮剧|潮剧]]、[[剪紙|剪紙]]、[[灯謎|灯謎]]、[[車鼓弄|大車鼓舞蹈]]等。

== 宗教 ==
*'''佛教'''：[[南山寺_(漳州)|南山寺]]、普照寺、三平寺、古林寺、東明寺、靈通寺、瑞竹岩寺
*'''道教'''：[[浦頭大廟|浦頭大廟]]、[[路邊威惠廟|路邊威惠廟]]、[[官園威惠廟|官園威惠廟]]、[[漳浦威惠廟|漳浦威惠廟]]、[[雲霄威惠廟|雲霄威惠廟]]、[[東山關帝廟|東山關帝廟]]、[[白礁慈濟宮|白礁慈濟宮]]、[[霞東書院|霞東書院]]、通元廟、岱仙岩、初華坊
*'''天主教'''：后坂玫瑰聖母堂、東坂後天主堂、林下露德聖母堂、漳州市耶稣聖心堂 、長泰法蒂聖母堂、南靖天主教堂等。
*'''基督新教'''：接官亭禮拜堂、東坂後禮拜堂、福音堂、北門禮拜堂 、下沙禮拜堂等。

* [[彭沖_(政治人物)|彭冲]]

* [[杨骚|杨骚]]
* [[陳忠和|陳忠和]]：2008年中國女排主教練

* [[林正杰|林正杰]]：前[[民主進步黨|民主進步黨]]創黨黨員之一，現在為[[中華統一促進黨|中華統一促進黨]]領袖之一
* [[方舟子|方舟子]]：學者、科普作家。
