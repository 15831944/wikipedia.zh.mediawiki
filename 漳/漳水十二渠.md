'''漳水十二渠'''，又名'''引漳十二渠'''、'''漳水渠'''、'''西門渠'''，[[戰國|戰國]]初年[[魏國|魏國]][[西門豹|西門豹]]、[[史起|史起]]開鑿之水利，是戰國初年第一個大型引水渠<ref>{{Cite web|url= http://www.chiculture.net/0808/html/a03/b04/c12/0808c12.html |title= 中國古代水利 西門渠 |author= 中國文化研究院 |publisher= 燦爛的中國文明 網站 }}</ref>。此渠引[[漳河_(衛河)|漳水]]溉[[鄴|鄴]]（今河北省南端磁縣、臨漳縣一帶），以富魏之[[河內郡_(中國)|河內]]<ref name="漢書">[[s:漢書/卷029|漢書 溝洫志]]</ref>，為魏國初年強盛之基礎。

== 河伯娶婦 ==
戰國初年，[[魏文侯|魏文侯]]先後任用[[李悝|李悝]]、[[吳起|吳起]]等變法。前400年（魏文侯二十五年）<ref>[[s:史記三家註/卷044|史記 魏世家]]</ref>，魏侯派[[西門豹|西門豹]]任鄴令。由於漳水經常氾濫，鄴地[[三老|三老]]、[[廷掾|廷掾]]勾結[[巫祝|巫祝]]製造「[[河伯|河伯]]娶妻」騙局，藉機敲詐百姓中飽私囊。西門豹於是以其人之道，還治其人之身，借口奉送之美女實在醜，派巫嫗、其弟子及三老先後下河通知河伯，美其名為「請河伯寬限幾日，重新找個美女」。巫嫗與其弟子一個個被投之河中，廷掾、長老等貪官污吏被嚇得魂不附體，皆叩頭至額血流地。鄴吏民大驚恐，從此不敢復言為河伯娶婦。之後，西門豹率領民眾開鑿漳水十二渠，引河水灌民田。當其時，民治渠少煩苦，不欲也。西門豹曰：「民可以樂成，不可與慮始。今父老子弟雖患苦我，然百歲後期令父老子孫思我言。」<ref name="史記"> [[s:史記三家註/卷126#西門豹|s:史記三家註/卷126#西門豹]]</ref>

== 史起續修 ==
晉代[[左思|左思]]《[[魏都賦|魏都賦]]》云，「西門溉其前，史起灌其後」。[[魏襄王|魏襄王]]時期，鄴令[[史起|史起]]決漳水以灌溉鄴田。當時，鄴民大怨，欲凌辱史起，史起不敢出而避之。魏王乃使他人繼續為之。水已行，民大得其利，相與歌之曰：「鄴有聖令，時為史公，決漳水，灌鄴旁，終古斥鹵，生之稻粱。」<ref name="呂氏春秋">[[s:呂氏春秋/卷十六#樂成|s:呂氏春秋/卷十六#樂成]]</ref><ref name="漢書"/>

== 影響 ==
十二渠不僅使鄴城因灌溉而連年豐收，並且減輕漳水氾濫的威脅。令西漢到隋代幾百年間，鄴城安陽一帶繼續為政經重地，[[袁紹|袁紹]]、[[曹操|曹操]]、[[東魏|東魏]]、[[北齊|北齊]]曾都鄴城。

漢代有長吏以為十二渠橋經絕[[馳道|馳道]]，欲合渠水，且至馳道合三渠為一橋。鄴民人父老不肯聽長吏，以為西門君所為也，賢君之法式不可更也，長吏終聽而擱置計劃<ref name="史記"/>。

東漢未年，曹操以漳水渠為基礎建成'''天井堰'''，「又堨漳水，迴流東注，號天井堰。二十里中，作十二墱，墱相去三百步，令互相灌注，一源分爲十二流，皆懸水門。」<ref name="水經注">[[s:水經注/10|水經注 濁漳水 清漳水]]</ref>
