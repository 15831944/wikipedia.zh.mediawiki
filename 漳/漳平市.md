'''漳平市'''（{{lang-nan|漳平市|poj=Chiuⁿ-phîng-chhī|bp=Zniūpíng-cî}}）位于[[福建省|福建省]]西南部，[[九龙江|九龙江]]北溪上游，[[北纬|北纬]]24°54′- 25°47′，[[东经|东经]]117°11′- 117°44′。漳平市隶属[[龙岩市|龙岩市]]管辖，是华南重要的交通枢纽，亦是福建省重要的花卉和茶叶产地。

== 历史 ==
[[明|明]][[成化|成化]]六年1470年析龙岩县（现龙岩市）置漳平县，1956年宁洋县撤销，县城双洋等地入属。1990年8月撤县建市。

===名称由来===
《[[读史方舆纪要|读史方舆纪要]]》载：“县居漳上游，千山中地稍平衍，因曰漳平”<ref> {{cite web |url=http://www.zp.gov.cn/TourInfo/ShowArticle.asp?ArticleID=44 |title=你知道漳平的行政变迁吗？ |accessdate=2008-05-29 |date=2006-05-15 |publisher=漳平市人民政府 |language=zh-hans }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 地理 ==
东连[[永春县|永春县]]、[[安溪县|安溪县]]，东南与[[华安县|华安县]]交界，南与[[南靖县|南靖县]]接壤，西与[[新罗区|新罗区]]相连，北与[[永安市|永安市]]交界，东北与[[大田县|大田县]]毗邻。

漳平地势从南、北向河谷倾斜。苦笋林尖[[海拔|海拔]]1666米，为全市最高峰。主要河流[[九龙江|九龙江]]北溪。年平均气温20.3℃，年降水量1496毫米，无霜期317天。

== 物产 ==
* 矿产有金、银、铁、煤、钨、石灰石、铅锌、铜、稀土、石墨、大理石等20多种。
* 福建省潘洛铁矿由潘田、洛阳两矿区组成。洛阳矿区位于漳平市芦芝乡大深村；该矿建于1958年，现为本省最大的铁矿石生产基地，国家中型铁矿山。40多年来，该矿职工艰苦创业，将潘洛铁矿建成为拥有各种矿山机械设备，采矿、选矿、运输机械化的成龙配套的现代化中型铁矿山，设计年采铁矿能力90万吨，年选矿能力40万吨。
* 土特产主要有笋干、水仙茶、明姜、花卉、青仁乌豆、水蜜桃、芦芝柚、竹藤编。
