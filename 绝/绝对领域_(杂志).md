《'''绝对领域'''》是[[中国大陆|中国大陆]]发行的一本[[ACG|ACG]][[杂志|杂志]]，由开明文教音像出版社和北京漫动天地文化传媒有限公司联合出版，由北京艺堂印刷有限公司和北京旺都印务有限公司印刷。杂志的内容主要是对[[日本动漫|日式动漫]]和[[美少女游戏|美少女游戏]]的介绍和点评。2013年9月，此杂志被判定为非法出版物而停刊。<ref name="新华网" />

==刊物内容==
《绝对领域》以ACG为主题，刊登跟动漫和美少女游戏有关的介绍文章、点评文章和[[攻略|攻略]]等，不仅有日本制作的ACG作品的介绍，如《[[Rewrite|Rewrite]]》和《[[白色相簿2|白色相簿2]]》等作品，也有中国制作的ACG作品的介绍，如《[[雾之本境|雾之本境]]》、《[[雪之本境|雪之本境]]》等作品。杂志采用全彩色印刷和单色印刷混合的方式，并附带有[[光盘|光盘]]和[[长筒袜|长筒袜]]、[[胖次|胖次]]等实物赠品。<ref>{{cite web |title=《绝对领域》VOL.17 4月8日上市！送塑形吊带背心，价格不变！ |url=http://www.ddc.com.cn/books/newspaper/2013/0409/242.html |website=DDC传媒 |accessdate=2018-07-14 |date=2013-04-09 |deadurl=no |archiveurl=https://web.archive.org/web/20150909201301/http://www.ddc.com.cn/books/newspaper/2013/0409/242.html |archivedate=2015-09-09 |df= }}</ref><ref>{{cite web |author1=少年jump |title=中国动漫期刊《绝对领域》杂志每期内容回顾 |url=http://www.chncomic.com/info/201310/28513.html |website=中国国际动漫网 |accessdate=2018-07-14 |date=2013-10-08 |deadurl=no |archiveurl=https://web.archive.org/web/20180714162653/http://www.chncomic.com/info/201310/28513.html |archivedate=2018-07-14 |df= }}</ref><ref>{{cite web|title=国内动漫游戏杂志《绝对领域》附送女性内裤“胖次”|url=http://www.xi1.cn/2012/0406/5300.shtml|website=中国动漫产业网|accessdate=2018-07-14|date=2012-04-06|deadurl=no|archiveurl=https://web.archive.org/web/20180714162652/http://www.xi1.cn/2012/0406/5300.shtml|archivedate=2018-07-14|df=}}</ref>

==争议==
2013年9月16日，有读者向中华人民共和国[[全国“扫黄打非”工作小组办公室|“扫黄打非”工作小组办公室]]联合举报中心举报《绝对领域》杂志的封面和内容因打[[色情|色情]]擦边球而不适合青少年阅读。经过审查后，[[中华人民共和国国家新闻出版广电总局|中华人民共和国国家新闻出版广电总局]]和北京市文化市场行政执法总队判定该杂志的内容不适合青少年阅读，同时杂志的出版和印刷也没有上报审批，此外还存在着多个刊物共用一个刊号的现象。于是出版商和印刷商均受到没收[[非法所得|非法所得]]、[[罚款|罚款]]以及停顿整改的处罚，该杂志也因此被[[中共中央宣传部|中共中央宣传部]]勒令停刊。此次停刊事件也被[[中国中央电视台|中国中央电视台]]的《[[焦点访谈|焦点访谈]]》节目所报道。<ref name="新华网">{{cite web|title="扫黄打非"集中整治少儿出版物 《菠萝志》被查处|url=http://www.xinhuanet.com/zgjx/2013-09/30/c_132763003.htm|website=新华网|accessdate=2018-07-14|date=2013-09-30|deadurl=no|archiveurl=https://web.archive.org/web/20180714162654/http://www.xinhuanet.com/zgjx/2013-09/30/c_132763003.htm|archivedate=2018-07-14|df=}}</ref><ref>{{cite web |author1=魂弹 |title=百态｜《二次元狂热》100期纪念，与它结缘的人们 |url=http://www.anitama.cn/article/a6246d99d18747d1 |website=Anitama |accessdate=2018-07-14 |date=2017-06-25 |deadurl=no |archiveurl=https://web.archive.org/web/20180102232725/http://www.anitama.cn/article/a6246d99d18747d1 |archivedate=2018-01-02 |df= }}</ref>
