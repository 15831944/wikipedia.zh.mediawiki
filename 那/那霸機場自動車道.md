'''那霸機場自動車道'''（{{jp|j=那覇空港自動車道|hg=なはくうこうじどうしゃどう|rm=Naha kūkō jidōshadō|lead=yes|extra2={{lang-en|Naha Airport Expressway}}}}），同時也是'''國道506號'''（{{jp|j=国道506号|hg=こくどう506ごう|rm=Kokudō 506 gō|extra2={{lang-en|National Route 506}}}}），是[[日本|日本]][[沖繩縣|沖繩縣]][[那霸市|那霸市]]至[[中頭郡|中頭郡]]之[[高規格幹線道路|高規格幹線道路]]，為構成沖繩本島高速交通體系和{{link-ja|那霸都市圈|那覇都市圏}}的骨幹道路之一，將[[那霸機場|那霸機場]]連接至[[沖繩島|沖繩島]]各處，全長約20公里，被視為進出沖繩縣的[[玄關|玄關]]口<ref>{{cite web | url=http://www.hido.or.jp/14gyousei_backnumber/2013data/1307/1307chiiki-ogb.pdf | script-title=ja:那覇空港自動車道豊見城東道路の整備について | publisher={{lang|ja|内閣府　沖縄総合事務局　開発建設部　道路建設課}} | date=2013-07 |accessdate=2014-04-09}}</ref>。

本道路由[[西日本高速公路|西日本高速公路株式會社]]（NEXCO）負責運行和維修保養事宜<ref>{{cite web | url=http://corp.w-nexco.co.jp/corporate/disclosure/h18/pdfs/haebaru.pdf | script-title=ja:一般国道５０６号（那覇空港自動車道（南風原道路））に関する維持、修繕その他の管理の仕様書 | date=2007 | accessdate = 2014-04-09}}</ref>。

== 概要 ==
那霸機場自動車道一共分為三段：
* 小祿道路：那霸空港IC－豐見城・名嘉地IC，目前仍在計劃中，預計2020年前完成。
* 豐見城東道路：豐見城・名嘉地IC－南風原南IC，免費路段。
* 南風原道路：南風原南IC－西原JCT，2009年3月28日起免費開放。

=== 作為國道506號 ===
根據「一般國道之路線指定政令」與「一般國道之指定區間指定政令」，國道506號的概要如下（政令記載為[[市町村|市町村]]名）：
* 起點 : 那霸機場
* 終點 : 沖繩縣中頭郡西原町（字池田田味名474番3（西原JCT））
* 重要經過地 : 沖繩縣島尻郡豐見城村、同郡南風原町
* 指定區間 : 那霸市字鏡水水溜屋原1011番1 - 沖繩縣中頭郡西原町字池田我喜又408番1

== 歷史 ==
[[File:Naha_Airport_Expressway.jpg|250px]]上）]]
* 1987年6月30日：開始設置本[[高規格幹線道路|高規格幹線道路]]的構思。
* 1989年8月8日：南風原道路、豐見城東道路基本計劃完成。
* 1989年12月25日：收費道路「那霸機場自動車道（南風原道路）」開始建造<ref>1989年（平成元年）12月24日日本道路公団公告第63号「有料道路「那覇空港自動車道（南風原道路）」工事開始公告」</ref>。
* 1990年7月17日：南風原道路、豐見城東道路都市計劃完成。
* 1990年：開始為南風原道路徵地。
* 1992年：開始建造南風原道路。
* 1992年4月1日：分配編號為「国道506号」<ref>1992年（平成4年）4月3日政令第104号「一般国道の路線を指定する政令の一部を改正する政令」</ref>。
* 1993年：開始為豐見城東道路徵地。
* 1996年：開始建造豐見城東道路。
* 2000年6月23日：西原JCT－南風原南IC路段被編入一般国道506号，預定作為[[快速道路|自動車専用道路]]運行<ref>2000年（平成12年）6月22日建設省告示第1550号「自動車専用道路を指定する件」</ref>。
* 2000年6月27日：收費道路「那霸機場自動車道（南風原道路）」西原JCT－南風原南IC路段工程完成<ref>2000年（平成12年）6月26日日本道路公団公告第30号「有料道路「那覇空港自動車道（南風原道路）」工事一部完了公告」</ref>。
* 2000年6月28日：收費道路「那霸機場自動車道（南風原道路）」西原JCT－南風原南IC路段開通，道路收費落實將維持40年至2040年6月27日<ref>2000年（平成12年）6月26日日本道路公団公告第31号「有料道路「那覇空港自動車道（南風原道路）」の料金の額及び徴収期間の公告」</ref>。
* 2003年4月25日：南風原道路的豐見城IC－南風原南IC路段工程完成<ref>2003年（平成15年）4月24日日本道路公団公告第43号「有料道路「那覇空港自動車道（南風原道路）」工事完了公告」</ref>。
* 2003年4月26日：豐見城IC - 南風原南IC路段被編入一般国道506号，預定作為自動車専用道路運行<ref>2003年（平成15年）4月25日沖縄総合事務局告示第30号「自動車専用道路を指定する件」</ref>、收費道路「那覇空港自動車道（南風原道路）」豐見城IC－南風原南IC路段開通<ref>2003年（平成15年）4月25日沖縄総合事務局告示第31号「道路に関する件」</ref>。
* 2005年7月15日：日本道路公團考慮延長南風原道路的收費時間至2050年，以彌補844億日圓的道路建造費用當中未能獲得公共事業費補貼的29億日圓<ref>{{cite web |url=http://ryukyushimpo.jp/news/storyid-4338-storytopic-4.html |script-title=ja:２０５０年まで有料継続検討　那覇空港道南風原道路 |publisher=琉球新報 |date=2005-07-15 |accessdate=2014-04-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20140413135235/http://ryukyushimpo.jp/news/storyid-4338-storytopic-4.html |archivedate=2014年4月13日 |df= }}</ref>。 
* 2006年3月31日：落實收費道路「那霸機場自動車道（南風原道路）」道路收費將維持至2009年8月19日<ref>2006年（平成18年）3月31日西日本高速道路株式会社「高速道路の料金の額及び徴収期間の変更公告」</ref>。
* 2008年3月14日：豐見城・名嘉地JCT－南風原南IC路段被編入一般国道506号，預定作為自動車専用道路運行<ref>2008年（平成20年）3月14日沖縄総合事務局告示第15号「自動車専用道路を指定する件」</ref>。
* 2008年3月22日：豐見城・名嘉地JCT－南風原南IC路段開通<ref>2008年（平成20年）3月21日沖縄総合事務局告示第20号「道路に関する件」</ref>。
* 2009年3月28日：近6公里長<ref>{{cite web |url=http://www.qab.co.jp/news/200903287788.html |script-title=ja:那覇空港道 きょうから無料 |publisher=琉球朝日放送| date=2009-03-28 | accessdate=2014-04-10}}</ref>的南風原道路路段開始免費開放<ref>{{cite web | url=http://corp.w-nexco.co.jp/corporate/release/kyushu/h21/0226/ | script-title=ja:那覇空港自動車道（南風原道路）平成21年3月28日(土曜)午前0時に無料開放 | date=2009-02-26 | accessdate=2014-04-10}}</ref>。
* 2014年3月30日：由於需要處理一枚位於南風原町的未爆[[美軍|美軍]]砲彈，南風原南IC為安全起見需要關閉兩小時<ref>{{cite web | url=http://www.47news.jp/localnews/okinawa/2014/03/30_9.html | script-title=ja:南風原町宮城で30日不発弾処理 | date=2014-03-29 | accessdate=2014-04-10}}</ref>。
* 2014年3月31日：豐見城IC－南風原南IC路段從雙線行駛更改為四線行駛<ref>{{cite web | url=http://www.dc.ogb.go.jp/nankoku/topix/h25kisya/140214_r506tomihigasiichibukaitsu/index.html | script-title=ja:記者発表「国道506号豊見城東道路 一部区間４車線開通那覇空港自動車道の豊見城ＩＣ～南風原南IC 間が４車線開通！～平成２６年３月３１日開通～」 | date=2014-02-14 | accessdate=2014-04-09}}</ref>。
