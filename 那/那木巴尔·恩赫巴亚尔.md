'''那木巴尔·恩赫巴亚尔'''（{{lang-mn|Намбарын Энхбаяр}}，{{bd|1958年|6月1日|||catIdx=E}}），生于[[乌兰巴托|乌兰巴托]]，[[蒙古国|蒙古国]]政治人物，[[蒙古人民革命党|蒙古人民革命党]]前[[总书记|总书记]]和[[蒙古人民革命党中央委员会主席|主席]]，曾任[[蒙古总统|蒙古总统]]、[[蒙古总理|蒙古总理]]、[[国家大呼拉尔|国家大呼拉尔]]（国会）主席。

==生平==
1958年6月，那木巴尔·恩赫巴亚尔生于乌兰巴托市。1980年自[[苏联|苏联]][[莫斯科|莫斯科]][[高尔基文学院|高尔基文学院]]毕业。1985年到1986年，留学[[英国|英国]][[利兹大学|利兹大学]]<ref name=xhw>{{Cite web|url =http://news.xinhuanet.com/ziliao/2003-05/22/content_882425.htm|title =蒙古国前总统恩赫巴亚尔|author =|date =2003-05-22|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20151030023835/http://news.xinhuanet.com/ziliao/2003-05/22/content_882425.htm|archivedate =2015-10-30}}</ref>。

1980年到1990年，他在蒙古作家协会历任翻译、编辑、执行秘书。1990年到1992年，担任蒙古政府文化艺术发展委员会第一副主席。1992年8月到1996年，担任[[蒙古国|蒙古国]]文化部部长。1992年到2004年，担任[[国家大呼拉尔|国家大呼拉尔]]委员<ref name=xhw/>。

1996年7月，恩赫巴亚尔出任[[蒙古人民革命党|蒙古人民革命党]][[总书记|总书记]]。1997年6月到2005年，担任[[蒙古人民革命党|蒙古人民革命党]][[蒙古人民党中央委员会主席|主席]]。2000年7月到2004年6月，担任[[蒙古总理|蒙古国总理]]。2004年8月到2005年，担任[[国家大呼拉尔|国家大呼拉尔]]主席。2005年5月，当选为[[蒙古国总统|蒙古国总统]]，同年6月24日就任，任至2009年6月23日。2009年5月24日，他在大选中失败，成为第一位未能连任的蒙古国总统<ref name=xhw/>。

在担任总统期间，恩赫巴亚尔致力于把蒙古从原[[社会主义国家|社会主义国家]]转变为自由民主主义国家。[[美国|美国]]曾经在其任内向蒙古国提供2亿8500万美元的援助<ref name=zyrb>{{Cite web |url = http://chinese.joins.com/gb/article.do?method=detail&art_id=125674 |title = 前蒙古总统恩赫巴亚尔逃亡至韩国|author =  |date =2014-10-06  |publisher = 中央日报 |language =  |accessdate =  |quote =  }}</ref>。

2010年，[[蒙古人民革命党|蒙古人民革命党]]恢复原党名[[蒙古人民党|蒙古人民党]]。该党前主席恩赫巴亚尔在2011年另起炉灶，注册成立了新的“[[蒙古人民革命党_(2011年)|蒙古人民革命党]]”，而且当选该党主席<ref name=zgxww>{{Cite web |url = http://www.chinanews.com/gj/2012/07-25/4058628.shtml |title = 蒙古国总理巴特包勒德辞去蒙古人民党主席职务 |author =  |date = 2012-07-25 |publisher =  中国新闻网|language =  |accessdate =  |quote =  }}</ref>。

2012年4月13日，因涉嫌腐败问题，恩赫巴亚尔在乌兰巴托被警方逮捕<ref>{{cite web|url=http://www.zaobao.com/gj/gj120414_012.shtml | language= | title=蒙古前总统涉贪被捕 |publisher=联合早报 |date=2012-04-14 |accessdate= }}</ref>。他面对包括挪用原本献给佛寺的赠与，及非法将一家酒店私有化等多项贪污指控。2012年8月2日，贪污罪名成立，恩赫巴亚尔被判处有期徒刑四年，并处没收部分财产<ref>{{cite web|url=http://www.zaobao.com/gj/gj120804_008.shtml | language= | title=蒙古前总统贪污入狱 |publisher=联合早报 |date=2012-08-04 |accessdate= }}</ref>。恩赫巴亚尔对该判决不服，最终上诉到蒙古最高法院。2012年12月7日，蒙古最高法院终审判决，判处恩赫巴亚尔有期徒刑两年六个月<ref name=xhw/>。

在关押期间，恩赫巴亚尔大部分时间都住在供政府高级官员医疗用的第二综合医院。2013年8月，恩赫巴亚尔因健康原因获得赦免。此后，恩赫巴亚尔主要逗留[[韩国|韩国]]，治病并对外活动。2014年，恩赫巴亚尔和家属获得了韩国国籍。他是第一位逃往韩国的外国前任和现任国家元首<ref name=zyrb/>。

在[[2017年蒙古国总统选举|2017年蒙古国总统选举]]，本来蒙古人民革命党在2017年5月5日推选恩赫巴亚尔为总统候选人<ref>{{Cite web |url =http://news.xinhuanet.com/world/2017-05/06/c_1120929808.htm  |title = 蒙古国三大政党全部推选出总统候选人 |author =  |date =2017-05-06  |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref>。但蒙古国最高法院在申请截止前2天突然宣布拒绝恩赫巴亚尔参选，蒙古人民革命党紧急闭门协商，赶在申请截止前数小时提出[[赛音呼·钢巴特尔|赛音呼·钢巴特尔]]作为替代者参选<ref name=yuanye>{{Cite web |url = http://www.guancha.cn/YuanYe/2017_06_26_415063_3.shtml |title = 袁野：蒙古总统大选前瞻——内亚大陆的茶杯风暴 |author =  |date = 2017-06-26 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>。

恩赫巴亚尔会[[英语|英语]]、[[俄语|俄语]]，曾经翻译过蒙古传统史诗、[[英国|英国]][[狄更斯|狄更斯]]等西方作家的作品。恩赫巴亚尔已婚，夫人是朝勒蒙，有子女4人<ref name=xhw/>。
