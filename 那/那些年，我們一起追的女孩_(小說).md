《'''那些年，我们一起追的女孩'''》是[[臺灣|臺灣]]著名作家[[九把刀|九把刀]]的半自傳{{link-ja|青春小說|青春小說|青春}}[[愛情小说|愛情小说]]，於2011年翻拍成[[那些年，我們一起追的女孩|同名電影]]。

== 故事 ==
高中生[[柯景腾|柯景腾]]成绩很差而且调皮捣蛋，班主任于是想出一个办法来提高他的成绩和管束他，把他托付给成績优秀的女生沈佳仪来管理。沈佳仪的管制和约束，使得柯景腾的成绩步步高升，越来越好，同时柯景腾也爱上了这个成績优秀、长相美丽的女生，却因为羞涩而一直不敢表白，而沈佳仪却一心读书，并不想恋爱，虽然最后柯景腾借助机会表白了，沈佳儀也一同表白了，但是两人最终还是没有在一起。

== 书中人物 ==
*[[柯景腾|柯景腾]]，高中生，起初成绩很差，而且调皮捣蛋，在沈佳仪的帮助下变成了成绩优等生，全級前三十名的學生。

*沈佳宜，高中生，成绩优秀，为人善良，乐于助人，思想成熟。

演员[[王传一|王传一]]：“这本书，我花了很久的时间才看完，但是看完的当下涌上心头的却是满满的温暖。”<ref name=" cat" />

== 翻译 ==
伴随日版电影翻拍，日本[[讲谈社|讲谈社]]推出了《那些年，我們一起追的女孩》的日文翻译版，翻译者是阿井幸作与[[泉京鹿|泉京鹿]]。发售日是[[2018年|2018年]][[8月10日|8月10日]]。
