==生平==
皮明勇是[[历史学|历史学]]硕士<ref name=sanming>{{Cite web |url =http://china.caixin.com/2017-09-07/101140523.html  |title = 军事科学院领导层集中调整 三名将领履新 |author =  |date = 2017-09-07 |publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref>。研究生毕业后被分配到[[中国人民解放军军事科学院|中国人民解放军军事科学院]]工作，长期从事战争理论和军事战略研究<ref name=chengyuan>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1785804 |title = 军事科学院新领导班子成员渐次亮相，贺福初、皮明勇任副院长 |author =  |date =2017-09-06   |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。曾任军事科学院战略部副研究员<ref name=sanming/>。2013年，任中国人民解放军军事科学院科研指导部部长<ref>{{Cite web|url=http://news.xinhuanet.com/mil/2014-03/12/c_126256406.htm|title=皮明勇：甲午战败实为“文化力”之败|author=|date=2014-03-12|publisher=新华网|language=|accessdate=|quote=}}</ref>。2017年，任新调整组建的中国人民解放军军事科学院副院长<ref name=chengyuan/>。
