'''乐正绫'''是[[上海禾念|上海禾念]]推出的一款采用[[Vocaloid|Vocaloid]][[语音合成|语音合成]]技术的[[库_(计算机)|资料库]]及附属的[[虚构角色|虚构角色]]。歌声样本取自中国网络歌手[[祈Inory|祈Inory]]，以拥有元气少女的声线为特点，角色由[[插画家|插画家]]ideolo设计。上海禾念在2015年7月17日发布其首个产品。

== 角色 ==
乐正绫是一个有着深红色短发，红棕色眼睛，身着汉服风格服装的少女。她的原型是第一届“Vocaloid™ China”形象募集活动中的{{lang|ja|ハオ}}绘制的优胜作品“绫彩音”，创作灵感来自“[[博麗靈夢|博麗靈夢]]”角色。<ref>{{Cite book|year=2013|title= VOCALOID中文插畫本_Project V|pages=102|publisher=|isbn= }}</ref>

*身高：160厘米

;象征物
乐正绫的象征物是[[中国五声音阶|中国五声音阶]]中的“-{徵}-”，代表符号是梵文“{{lang|sa|त}}”。《汉书·律历志》中介绍：“-{徵}-为火声，居南方，时序为夏。”以及《黄帝内经·素问·阴阳应象大论》：“心在音为-{徵}-。”其体现了乐正绫的性格和她作为角色给人的印象。<ref>{{cite web |url= https://tieba.beidu.com/p/3823728760 |title= [细节+科普]关于DEMO曲和声库版绫帅_乐正绫吧_百度贴吧 |author= jarryestel |date= 2015-06-13 |language= zh-hans }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>其烛火状的产品[[标志|标志]]中也体现了这一理念。

== 宣传和发行 ==
2011年11月20日，Vocaloid™ China Project启动，公布征集人物形象、活动的计划。

2012年1月17日，Vocaloid China公布了包括“绫彩音”（乐正绫形象原型）在内的五个中文形象入围作品。3月21日，画师ideolo根据入围作品进行再创作后，公布了包括“乐正绫”等五位正式形象，并公布形象详细说明。6月6日至11月12日推出了五篇由七灵石动漫画有限公司制作宣传视频，视频中大致地讲述了乐正绫与洛天依相遇后的故事。

公司变动后重组的[[Vsinger|Vocanese]]经过三个月的运作，于2014年7月15日公布了乐正绫的声库制作计划，表示在雅马哈的技术支持下制作，并预计投入100万元人民币资金。<ref>{{cite web |url= http://weibo.com/3638133220/BdHNPC0rN |title= 大家好。小V今天想发布一下公司的运营规划... 来自VOCANESE - 微博 |author= Vocanese|date= 2014-07-15|publisher= 上海禾念|language= zh-hans }}</ref>度过筹备期后，Vocanese在8月2日确定开始进入制作期，并启动为期18天的乐正绫原声募集活动。8月20日活动结束并在8月26日公布三位候选人：菊花花，祈Inory，[[幽舞越山|幽舞越山]]。<ref>{{cite web |url= https://www.facebook.com/vocanese/photos/a.563579297044313.1073741828.555588054510104/693079564094285/ |title= VOCANESE - ☆速報☆ 8月26日於北京舉辦的“戰音Lorra”＆“樂正綾”的聲源選拔會的結果公布啦！... - Facebook |author= Vocanese|date= 2014-09-05|publisher= 上海禾念|language= zh-hans }}</ref>9月11日，测试声库录制完成。<ref>{{cite web |url= http://weibo.com/1663365680/Bmvyk19CS |title= @乐正绫_官博 和@战音Lorra_官博 的测试声... 来自人形兎 - 微博 |author= 人形兎|date= 2014-09-11|language= zh-hans }}</ref>9月30日，释放测试试听音频，开启网络投票。10月15日，公布获胜者为祈Inory，也开始进行声库的录制工作。

2015年5月28日，公司再变动后重组的Vsinger公布声库制作已完成，并正在进行包装工作。7月1日，产品在Vsinger的淘宝店铺上开启预售。7月17日，上海禾念在第十届萤火虫动漫游戏嘉年华上正式发售产品《VOCALOID™3 Library 乐正绫》，并公布新详细设定<ref>{{cite web |url= http://weibo.com/5582156776/CrwRDF45s |title= #V家动态# 从2012到2015，3年时间，诞生了... 来自Vsinger - 微博 |author= Vsinger|date= 2015-07-17|publisher= 天矢禾念|language= zh-hans }}</ref>，但引起众多粉丝不满，官方在听取粉丝意见后表示修改<!--虽然没正式消息说修改，但在官网公布的看得出改回去了-->。

== 同人声库 ==
在言和征集形象活动同期，出现以“乐正绫”角色为原本制作的，用于[[Utau|Utau]]的连续音类型声库的开发项目，开发团队都由<!---被伤透了心的--->乐正绫的粉丝组成，中之人为歌手泠鸢yousa。2014年2月2日，放出了已开发10%声库的[https://web.archive.org/web/20170923195537/http://5sing.kugou.com/m/Song/Detail/fc/11962418 试听曲]，预计半年后公开下载。<ref>{{cite web |url= https://tieba.beidu.com/p/2845227083 |title= 【绫utau音源制作组】关于阿绫utau连续音音源的说... |author= lww338 |date= 2014-02-12 |language= zh-hans }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>但之后随着上海禾念推出官方的开发计划而结束。
