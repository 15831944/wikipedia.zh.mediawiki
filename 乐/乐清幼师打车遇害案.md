在网友号召大家卸载滴滴的浪潮中，多位明星也宣布加入抵制。演员[[王传君|王传君]]发布了一张卸载滴滴App的截图，并配文“难道还要一直留着么？ ”而[[章子怡|章子怡]]则发微博反问滴滴打车的“滴”是不是“滴血”的意思。之后，[[马思纯|马思纯]]、[[何穗|何穗]]也在微博为受害者发声，分别称“地狱空荡荡，魔鬼在人间。就算保护自己，恐是无助”、“这炎炎夏日，新闻事件，看得太让人寒心了，出离愤怒！”<ref>{{cite web |title=滴滴事件众星发声 章子怡怒怼王传君卸载软件 |url=http://www.sohu.com/a/250100107_114941 |website=搜狐娱乐 |accessdate=2018-08-27 |date=2018-08-26}}</ref><ref>{{cite web |title=章子怡王传君质问滴滴平台 马思纯等声援被害女生 |url=http://ent.sina.com.cn/s/m/2018-08-26/doc-ihifuvph0645315.shtml |website=新浪网 |accessdate=2018-08-27 |date=2018-08-26}}</ref>。

《[[人民日报|人民日报]]》评论认为，滴滴顺风车事件显示各种新业态新经济崛起的时候，单靠政府难以建立严密的风险控制体系，互联网企业需要承担相应的社会责任<ref>{{cite web |title=网络平台不能只有“资本思维” |url=http://opinion.people.com.cn/n1/2018/0827/c1003-30251786.html |website=人民网 |accessdate=2018-08-27 |date=2018-08-27}}</ref>。《[[环球时报|环球时报]]》总编辑[[胡锡进|胡锡进]]则坦承“滴滴的确应该整改”，“但一个行业出问题，如果就不搞了，也未必就是办法”“那样的话，中国不知道有多少行业应该关掉”<ref>{{cite web |title=滴滴网约车再酿命案引质疑　下线顺风车业务 |url=https://www.bbc.com/zhongwen/simp/chinese-news-45318172 |website=BBC中文网 |accessdate=2018-08-27 |date=2018-08-27}}</ref>。

2018年連續發生兩起司機殺人強姦案後，[[央視|央視]]專題表示整理過去三年見報的新聞滴滴就有50起[[性騷擾|性騷擾]]、搶劫、強姦新聞，包含發生命案的這兩起。而北京衛視引用澎湃新闻調查發現這三年多來顺风车部門女总经理黄洁莉治理下，滴滴有意的將軟體功能跨出交通服務往一種[[相親|相親]]軟體的方向做發展，包含在七夕情人節推出一些活動，平日也鼓勵司機選擇異性乘客接單並在坐車過程有更多互動，黄洁莉曾演講表示要把搭車打造成「性感場景」，之後出現不少公司宣傳廣告是「我們相遇的浪漫故事」「濕了、硬了」「第一次見面就免單，我早晚也是你的」這類標語廣告。<ref>[https://item.btime.com/41rnj85j8169uhb0o1o3vtt6nhm 北京衛視-約車還是約會?]</ref>乐清案後黄洁莉被公司開除。<ref>[http://www.hnr.cn/news/shxw/201808/t20180827_3132820.html 滴滴被免职高管曾定义顺风车有社交属性，是“性感的场景”]</ref>

華龍網報導，滴滴順風車司機在接單前在軟體上可看到其他司機對乘客的評價。網路上一張截圖顯示，有司機給出「聲音很甜美」、「顏值爆表」等評價，還有一些露骨留言，例如「美女下車時絲襪容易走光看得想入非非」、「她有男朋友了」等。除了評價，順風車主頁面還設置「免單」功能，如果兩人聊得非常合拍，<ref>[http://tv.cctv.com/2018/08/27/VIDEc7xn8dOIQN8tGpFScsUh180827.shtml 央視-血滴、泪滴……“滴滴”警报声响起]</ref>車主可以不收錢。根據滴滴[[大數據|大數據]]分析，男車主願意給女乘客免單的比例達67%，是男乘客的2倍，此外，晚上10至11時是一天中最容易發生免單的時間段。為提高夜間安全滴滴恢復服務後曾短暫實施晚上10點過後至早上6點，禁止叫車服務，6月中則改為恢復晚上10點至12點及早上5至6點可叫車，但禁止男性司機載女性乘客（雙方需同一性別），但因中國開車司機多為男性，導致市場出現反映調整前，女司機沒增加前女用戶晚間根本叫不到車。<ref>[http://finance.sina.com.cn/china/2018-05-12/doc-ihamfahx0715536.shtml 人民日报：约车非“约会” 乘客个人信息岂能泄露]</ref>

==暴露的问题==
===私下接单===
由于私下接单可以省下本应支付给滴滴公司的费用，乘客和司机都可以得到实惠。所以取消订单、私下接单已成为行业潜规则。但私下接单存在更大的安全隐患<ref>{{cite web |title=顺风车司机：取消订单、私下接单是圈内潜规则|url=http://www.ce.cn/xwzx/gnsz/gdxw/201808/27/t20180827_30134512.shtml|website=中国经济网  |accessdate=2018-08-27 |date=2018-08-29}}</ref>。
===客服业务外包===
有调查{{who}}发现，滴滴客服工作被外包，滴滴客服并非滴滴自家员工。客服工作过程中，多以「非常抱歉给您带来不好的用户体验」、「我们会提交给专员处理」、「有结果会第一时间通知您」一类的话敷衍客服投诉<ref>{{cite web |title=接线员并非滴滴员工 招聘信息曝光客服业务大量外包|url=http://www.ittime.com.cn/news/news_23848.shtml |website=新浪科技|accessdate=2018-08-27 |date=2018-08-29}}</ref><ref>{{cite web |title=滴滴客服自述：我们就是滴滴和乘客之间的“挡箭牌”|url= http://www.ittime.com.cn/news/news_23890.shtml|website= 《人物》 |accessdate=2018-08-28|date=2018-08-29}}</ref>。
