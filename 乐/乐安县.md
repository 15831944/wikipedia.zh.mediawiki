'''乐安县'''是[[中国|中国]][[江西省|江西省]][[抚州市|抚州市]]所辖的一个[[县|县]]。总面积为2413平方公里，2003年人口为34.9万。

== 历史沿革 ==
[[南宋|南宋]][[绍兴|绍兴]]十九年（公元1149年）设县，属抚州，元代属抚州路，明清属抚州府。建国后属抚州地区、抚州市管辖。

== 行政区划 ==
乐安县下辖9个镇5个乡1个民族乡。

== 地理 ==
以山地、丘陵为主

== 交通 ==
* [[向乐铁路|向乐铁路]]

== 经济 ==
2002年2月，经国务院批准，乐安县被列为新阶段扶贫开发工作重点县。

2008年全县生产总值完成20.82亿元，财政总收入1.6366亿元。
