'''广州快速公交运输系统'''（{{lang-en|Guangzhou Bus Rapid Transit}}），俗称'''广州BRT'''，缩写'''GBRT'''，舊稱'''中山大道快速公交（BRT）试验线'''，位于[[中华人民共和国|中国]][[广州市|广州市]]东部，为广州首条建成的[[快速公交系统|快速公交系统]]，设有31条路线及26对车站，2010年2月10日开始免费试营运并于当月20日正式营运。

== 概况 ==
[[File:GBRT_Tianhe_Sports_Center.JPG|thumb]]
呈东西走向，西起天河区[[天河路|天河路]]，沿[[中山大道_(广州)|中山大道]]、[[黄埔东路|黄埔东路]]，东至[[黄埔区|黄埔区]][[开发大道|开发大道]]，其中天河路2.8公里、中山大道13公里、黄埔东路7.1公里。系统全长22.9[[公里|公里]]，全线规划路宽60米，道路中央为双向两条BRT-{zh-cn:公交专用道; zh-hk:巴士專線; zh-tw:公車專用道;}-（车站处4条），外侧为双向6至10条社会车道。设置站台26座，其中对开式15座，错位式11座，站台长度由55米至220米不等；站台为封闭式，并设有电子闸门。其营运、管理由广州市交通站场建设管理中心专门设立的广州快速公交运营管理有限公司负责，主要是负责营运系统的统一调度和管理工作。

== 历史 ==
[[File:GBRT_Gangding_Lane.jpg|thumb]]
自2005年开始，广州市建设部门展开快速公交概念研究，工程于2008年11月30日动工，总投资7.2亿元。廣州BRT自起建就爭議不斷，由于施工期间需要对天河路及中山大道部分路面及人行道进行围蔽施工，令该路段交通异常[[交通堵塞|拥堵]]，BRT因此也被戲稱為「不讓通」<ref>{{cite news|url=http://epaper.nfdaily.cn/html/2010-01/29/content_6816936.htm|title=打的开会遇堵车委员笑谈“不让通”(图)|publisher=南方日报|date=2010-01-29}}{{Dead link|date=2019年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，甚至影響到[[深圳市|深圳市]]關於是否建BRT的決策<ref>{{cite news|url=http://gcontent.oeeee.com/b/14/b147a61c1d07c1c9/Blog/6de/e95f23.html|title=深圳该不该建BRT？|publisher=南方都市报|date=2010-02-01|deadurl=yes|archiveurl=https://web.archive.org/web/20150604035444/http://gcontent.oeeee.com/b/14/b147a61c1d07c1c9/Blog/6de/e95f23.html|archivedate=2015-06-04}}</ref>。

2010年2月10日，廣州BRT開通，頭十天免費給市民搭乘<ref>{{cite news|url=http://gzdaily.dayoo.com/html/2010-02/04/content_858547.htm|title=BRT本月10日开通 市民免费乘坐10天|publisher=广州日报|date=2010-02-04|deadurl=yes|archiveurl=https://web.archive.org/web/20100209044056/http://gzdaily.dayoo.com/html/2010-02/04/content_858547.htm|archivedate=2010-02-09}}</ref>。自開通以來，其日均客流量达85万人次，最高达96万人次，单向高峰小时截面通过量达2.99万人次<ref name="广州获BRT系统金牌标准">{{cite news|url=http://www.ycwb.com/ePaper/ycwb/html/2013-09/23/content_257965.htm?div=-1#4942067-tsina-1-59766-92738d0edd726fe2f6cbcc3f15ecbfce|title=广州获BRT系统金牌标准|publisher=羊城晚报|date=2013-09-23|deadurl=yes|archiveurl=https://web.archive.org/web/20130924030606/http://www.ycwb.com/ePaper/ycwb/html/2013-09/23/content_257965.htm?div=-1#4942067-tsina-1-59766-92738d0edd726fe2f6cbcc3f15ecbfce|archivedate=2013-09-24}}</ref>。

广州BRT于2011年荣获可持续交通发展奖，为国内交通项目首获此奖者<ref>{{cite news|url=http://news.ycwb.com/2010-12/29/content_3224215.htm|title=广州BRT被授予“世界交通可持续发展奖”|publisher=羊城晚报|date=2010-12-29|access-date=2013-09-23|archive-url=https://web.archive.org/web/20110103001010/http://news.ycwb.com/2010-12/29/content_3224215.htm|archive-date=2011-01-03|dead-url=yes}}</ref>；2012年获联合国应对气候变化“灯塔奖”<ref>{{cite news|url=http://epaper.nfdaily.cn/html/2012-12/06/content_7148950.htm|title=广州BRT领回联合国“灯塔奖”|publisher=南方日报|date=2012-12-06}}{{Dead link|date=2019年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；2013年获评为BRT系统“金牌标准”，成为[[亚洲|亚洲]]唯一被评为“金牌标准”的BRT系统<ref name="广州获BRT系统金牌标准"/><ref name="再建BRT首选环市路">{{cite news|url=http://epaper.nfdaily.cn/html/2013-09/23/content_7227722.htm|title=再建BRT首选环市路|publisher=南方日报|date=2013-09-23|access-date=2013-09-23|archive-url=https://web.archive.org/web/20140308121643/http://epaper.nfdaily.cn/html/2013-09/23/content_7227722.htm|archive-date=2014-03-08|dead-url=yes}}</ref>。

目前廣州市官方有建更多BRT路段的意願，國外專家認為[[环市路|环市路]]、[[东风路_(广州)|东风路]]、[[黄埔大道|黄埔大道]]、[[解放路_(广州市)|解放路]]、[[机场路_(广州)|机场路]]、[[三元里大道|三元里大道]]、[[新港路|新港路]]，甚至[[广州大桥|广州大桥]]拓宽后的[[广州大道|广州大道]]，都比較適合建BRT，其中[[环市路|环市路]]為最適<ref name="再建BRT首选环市路"/>。

== 乘坐方式 ==
乘客可通过行人[[隧道|隧道]]、[[天桥|天桥]]和[[斑马线|斑马线]]进入和离开月台，在BRT通道内需要在月台投币或使用[[羊城通|羊城通]]拍卡入站，所有线路统一票价[[人民币|人民币]]2元。进入月台后按指引在指定闸門候车，倡导前门上车、后门下车（实际上乘客可在任何车门上落车），在BRT通道26站内均可免费换乘同方向其他路线。原线夜班巴士及不参与该系统营运的巴士路线则保留在道路两边巴士站点停靠，不进入专用通道行駛。其他细则与普通巴士无异。

== 路线 ==
在BRT投入运营之前，行经中山大道的有61条巴士<ref>《地铁五号线配套公交优化及BRT线路设置情况表》、《广州快速公交导乘手册》，广州市交委</ref>，其中31条路线纳入BRT通道，其余30条进行路线调整，避开BRT通道。具体路线采用30条灵活路线加1条摆渡线，其中摆渡路线从黄埔区夏园至天河体育中心，全程行走BRT通道，其余的30条公交路线为部分路段行走BRT通道。所有行经BRT的巴士线统一以“B”字开头。

== 意外事件 ==
{{update|time=2017-04-07}}
2014年7月1日，一辆巴士在广州BRT体育中心站内突然逆行，撞上对面一辆入站的巴士，造成人员受伤。据目击者称，当时正落大雨，这部巴士突然逆行，撞掉了同方向一辆公交车的后视镜，撞到对面入站的一辆巴士。当时该巴士上约有十余名乘客，有部分人员受伤，附近多部救护车赶到现场急救。现场消息称，肇事巴士车头严重变形，大部分挡风玻璃被破坏。在车站内中间护栏部分已被撞烂，车站的部分挡风玻璃损毁。车站自东向西方向已单向封锁，乘客只出不入。<ref>{{cite news|title=广州两公交车在BRT体育中心站相撞 伤者送医院|url=http://pic.cnr.cn/2012tppd/yangguangdujiapic/201307/t20130702_512946874_3.shtml|accessdate=2015-03-21|newspaper=央广网}}</ref>
