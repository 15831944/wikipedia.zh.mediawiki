'''广州动漫星城广场'''（简称：'''动漫星城'''）是中国[[广东省|广东省]][[广州市|广州市]][[越秀区|越秀区]]的一家地下[[主题商场|主题商场]]，共有三层，位于[[广州地铁|广州地铁]][[公园前站|公园前站]]附近，属于[[北京路_(越秀区)|北京路]]商业圈，自2006年8月18日开始营业。商场既有与[[日本动漫|日本动漫]]等[[ACG|ACG]]文化相关的店铺，也有其他主题的店铺，如餐饮店、服装店、化妆品店等。<ref name="官网">{{cite web |title=星城概况 |url=http://www.comicity.cn/index.php?s=/News/News/xcjj/type/1 |website=动漫星城官方网站 |access-date=2019-02-16 |archive-url=https://web.archive.org/web/20180706215540/http://www.comicity.cn/index.php?s=%2FNews%2FNews%2Fxcjj%2Ftype%2F1 |archive-date=2018-07-06 |dead-url=no }}</ref>

== 建筑构造 ==
动漫星城的地理位置处于[[人民公园_(广州)|人民公园]]的地下，有负一层、负二层和负三层共三层空间，其中个别区域有夹层空间<ref>{{cite web |title=广东置地三家子公司整合拍卖 留下三只下蛋的“金鸡” |url=http://news.winshang.com/html/010/2378.html |website=赢商网 |publisher=广州日报 |date=2011-12-04 |quote=开业至今，算上夹层合计4层的地下商场租户换了一拨又一拨}}</ref>，每一层分为东区和西区两个区域。其建筑面积大约为32000[[平方米|平方米]]。<ref name="官网" />

== 投资 ==
动漫星城最初由[[广东置地投资有限公司|广东置地投资有限公司]]旗下的广州天源投资管理有限公司所投资，投资金额为6亿元[[人民币|人民币]]<ref>{{cite web |title=动漫星城盛大开城 首个地下商业圈初成形 |url=http://news.163.com/06/0819/12/2OSUF22O0001124J.html |website=网易新闻 |publisher=南方网 |date=2006-08-19 |access-date=2019-02-13 |archive-url=https://web.archive.org/web/20060822134256/http://news.163.com/06/0819/12/2OSUF22O0001124J.html |archive-date=2006-08-22 |dead-url=no }}</ref>。2007年3月，[[中华人民共和国财政部|中华人民共和国财政部]]广东专员办检查[[交通银行|交通银行]]的财政情况时查出时任行长刘昌明伪造财务报表向广东置地的董事长王璟（又名王胜）非法贷款，后来广东置地也因此摘牌<ref>{{cite web |title=广东置地涉46亿骗贷案 拍卖动漫星城等3个项目还债 |url=http://news.winshang.com/html/010/1721.html |website=赢商网 |publisher=时代周报 |date=2011-11-27}}</ref><ref>{{cite web |title=动漫星城老板王胜涉巨额违规贷款 |url=http://news.163.com/09/1219/02/5QS44VE5000120GR.html |website=网易新闻 |publisher=南方网 |date=2009-12-19}}</ref>。2009年，广州天源亏损1300万元人民币，接下来的两年中，虽扭转亏损但获利不多。2011年12月，广州天源被[[拍卖|拍卖]]，最终[[广东粤海控股有限公司|广东粤海控股有限公司]]以3.28亿元人民币的价格中标，并由其子公司广东天河城（集团）股份有限公司经营<ref>{{cite web |author1=林晓丽 |title=天河城接手动漫星城 |url=http://news.163.com/14/0310/15/7L231T8Q00014AED.html |website=网易新闻 |publisher=大洋网-广州日报|date=2011-12-12 }}</ref><ref>{{cite web |title=广州动漫星城、名城商业广场、珠江新城地块股权捆绑竞拍 |url=http://news.winshang.com/html/010/1573.html |website=赢商网 |publisher=南方都市报 |date=2011-11-25}}</ref><ref>{{cite web |title=广州动漫星城、都汇168项目投资商100%股权遭拍卖 |url=http://news.winshang.com/html/010/1456.html |website=赢商网 |date=2011-11-11}}</ref><ref>{{cite web |title=动漫星城三项目亏损兼负债50亿 粤投接盘 |url=https://finance.qq.com/a/20111209/004901.htm |website=腾讯财经 |publisher=南方都市报 |date=2011-12-09}}</ref><ref>{{cite web |title=天河城收购广州动漫星城、名城商业广场项目40%权益 |url=http://news.winshang.com/html/010/1988.html |website=赢商网 | publisher=搜狐焦点网| date=2011-11-29}}</ref>。

== 营业状况 ==
动漫星城开业之初以动漫为主题，但现在以动漫为主题的商店已经大幅减少，萎缩为负三层西区的动漫专卖区，出售游戏、[[手办|手办]]、[[轻小说|轻小说]]、漫画等ACG主题的商品。其余区域则是其他主题的商店，如餐饮店、服装店化妆品店等。而虽然动漫星城位于地铁站的出口，交通较为便利，但是顾客数量并不是很多，特别是深楼层的顾客数量更少。一些小商家对此表示担忧，担心其营业利润抵不上铺租。<ref>{{cite web |title=“动漫”不再 广州动漫星城最高租金“3000+” |url=http://gz.house.163.com/16/1113/10/C5OBG5ET008782AS.html |website=网易房产 |publisher=羊城晚报(广州) |date=2016-11-13}}</ref><ref>{{cite web |title=广州地下商业广场调查：部分地下商城人气冷清 |url=http://news.winshang.com/html/013/1396-2.html |website=赢商网 |publisher=大洋网 |date=2012-10-31}}</ref>
