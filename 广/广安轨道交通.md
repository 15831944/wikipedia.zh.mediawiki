'''广安轨道交通'''（{{lang-en|'''Guang'an Rail Transit'''}}）是服务于[[中华人民共和国|中国]][[四川省|四川省]][[广安市|广安市]]的[[单轨铁路#跨座式单轨铁路|跨座式单轨]]系统，目前修建的[[广安云轨示范线|广安轨道交通1号线]]主要作为[[邓小平故居|邓小平故里景区]]旅游连接线。广安轨道交通第一条线路[[广安云轨示范线|广安轨道交通1号线]]于2018年8月22日开始试运行<ref>{{cite web |title=广安轨道交通（云轨）1号线将于2018年8月22日试运行|url=http://www.chengtu.com/forum.php?mod=viewthread&tid=566031}}</ref>，使得广安市成为四川省继成都市后第二个拥有城市轨道交通的城市。

== 在建线路 ==
=== 1号线 ===
{{main|广安云轨示范线}}
'''广安轨道交通1号线'''（又被称为'''广安小平故里连接线'''）一期全长9.955km，设有8座车站（其中长乐街站为预留车站）<ref>{{cite web |title=广安轨道交通1号线站点公布 |url=http://www.sohu.com/a/242408836_213861}}</ref>，自广安广门乡出发，向东再向北呈“L”型途径广安南站、广安中心城区抵达邓小平故里游客中心。全线为高架线路，在广门乡设有广门车辆段。

北延伸段全长9.3公里，设站6座，为全高架线路。该线将在南端与现有游客中心站相衔接。

广安轨道交通1号线預計2019年7月開通。

=== 2号线 ===
'''广安轨道交通2号线'''一期全长7.7公里，设站6座，为全高架线。该线将向东南跨越渠江，经奎阁新区达广安港。该线路站点为：影视城站-新区北站-奎阁站-广前大道站-港前大道站-广安港站。
