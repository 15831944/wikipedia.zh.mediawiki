'''广州地铁22号线'''是[[广州地铁|广州地铁]]的建设中路线之一。代表色暫定為<small> <font color={{广州地铁标志色|22}}>█</font> </small>普魯士藍。该线连接位于[[荔湾区|荔湾区]]的[[白鹅潭|白鹅潭]]和[[番禺区|番禺区]]的[[番禺广场站|番禺广场]]，并计划与[[广州地铁18号线|18號線]]共线运营至[[南沙区|南沙区]]万顷沙。<ref name=":0">{{Cite web|url=http://www.gzggzy.cn/cms/wz/view/index/layout3/index.jsp?siteId=1&channelId=503&infoId=493810|title=广州市城市轨道交通近期建设规划（2016-2022年）线路设计等前期研究十八、二十二号线工程设计咨询 招标公告|accessdate=2017-01-20|author=|date=|publisher=}}</ref>线路定位[[通勤鐵路|市域快线]]。現時屬於廣州市新一輪軌道交通（2016-2022）建設規劃方案的線路之一，已於2017年3月下旬獲國家發改委批覆立項。目前已经与[[广州地铁18号线|18号线]]同步开始掘进施工。<ref>{{Cite web|url=http://www.gzmtr.com/ygwm/xwzx/gsxw/201808/t20180829_58557.html|title=广州地铁 全程为你--公司新闻|accessdate=2018-09-03|work=www.gzmtr.com}}</ref>

== 概况 ==
22號線大致呈西北─東南走向，起始於荔灣區芳村（白鵝潭站），大致沿花地大道、花蕾路和東漖北路行走，然後大致轉向東南至西塱，再沿環翠南路、東新高速行走，再大致轉向西南，經陳頭崗進入廣州南站。隨後線路大致往東行進，沿廣明高速、市廣路、東環路、番禺大道北抵達番禺廣場站。隨後與18號線共線運行，前往南沙區萬頃沙。

線路全長31公里，均為地下線，設8座車站，其中换乘站4座；平均站间距6.3km，最大站间距7.5km，为陈头岗至西朗区间；最小站间距3.5km，为广州南站至陈头岗区间。西三和东沙工业园為预留車站，初期暫不開通。線路在番禺廣場站預留拆分南延條件，計劃未來另覓線位南延，不再與18號線共線；同時白鵝潭站亦預留延長條件，初步計劃未來往北延長，途經廣州火車站、以及新建的國鐵[[棠溪站|棠溪站]]，前往[[廣州白雲國際機場|白雲機場]]T3航站樓。

22號線將設兩個停車場，分別位於[[番禺區|番禺區]]东新高速公路以西，石北工业路以北，大石水道以东的石壁村内及[[番禺區|番禺區]]番禺大道北以東、市南路以北、草龍沙河二街以西，而陳頭崗停車場內亦設立主變電站供本線供電；而車輛段則與[[廣州地鐵18號线|18號线]]共用，位於[[南沙區|南沙區]]蕉门水道以西，X298县道以东、南沙七涌以南，八涌以北。本線和[[廣州地鐵18號线|18號线]]控制中心將設於隴枕停車場內。

== 歷史 ==
22號線在最初規劃時，起点为[[客村站|客村站]]，大致往南行走，經新光快速、番禺大道前往番禺廣場後進入南沙區，最終抵達明珠灣起步區南部的萬頃沙。該線按照快線設計，設站較少，以滿足南沙區快速來往番禺中心城區和廣州市區的功能。

后来，與原规划18号线交換編號，22號線改為承擔快速連接廣州南站至南沙，並確定從廣州南站延長至白鵝潭，番禺廣場以南一段則改為先與18號線共線。2017年3月下旬，22號線獲得立項批覆，此時白鵝潭站從靠近[[石圍塘站_(地鐵)|石圍塘站]]改為靠近[[芳村站|芳村站]]。4月中旬，根據22號線第二次環評公示的內容，22號線增加兩個車站，使車站總數增至8個<ref name=二次环评>[http://www.gzmtr.com/ygwm/gsgk/xxjs/201704/t20170420_51171.html 广州市城市轨道交通二十二号线工程 环境影响评价第二次公示]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

18號線和22號線原本均属于远期规划线路，但由於需要配合南沙自貿區發展，已決定將優先建設。22號線將按照快線設計，以實現南沙区與廣州南站、广州中心城区的快速轨道交通联系。<ref>{{Cite news|url =http://news.dayoo.com/guangzhou/201512/22/139995_45684383.htm|title =壕气！投三千来亿，广州未来15年新建15条地铁线|last =|first =|date =2015-12-22|work =|accessdate =|newspaper =[[广州日报|广州日报]]|deadurl =yes|archiveurl =https://web.archive.org/web/20151223155548/http://news.dayoo.com/guangzhou/201512/22/139995_45684383.htm|archivedate =2015-12-23}}</ref><ref>{{Cite news|url = http://gz.southcn.com/content/2015-12/22/content_139383442.htm|title = 广州新一轮地铁规划将杀到 15条线路详解|last = |first = |date = 2015-12-22|work = |accessdate = |newspaper = [[南方日报|南方日报]]}}</ref>

== 车站 ==
22號線計劃設8座車站，其中4站與已知既有和規劃線路換乘。

===換乘站===
* '''[[白鵝潭站|白鵝潭站]]'''：換乘[[广州地铁28号线|28号线]]，及前往[[芳村站|芳村站]]换乘[[广州地铁1号线|1号线]]和[[广州地铁11号线|11号线]]。1號線建设时未作任何预留，22號線車站位於現有芳村站西側，预计为站廳通道换乘。28號線為遠期線路，預留情況未知。
* '''[[西塱站|西塱站]]'''：換乘[[广州地铁1号线|1号线]]、[[廣佛地鐵|廣佛地鐵]]及[[广州地铁10号线|10号线]]。1號線和廣佛線建设时均未作任何预留，預計為站廳通道換乘；與10號線的預留情況未知。
* '''[[西三站|西三站]]'''：換乘[[广州地铁19号线|19号线]]。19號線為遠期線路，預留情況未知。
* '''[[陳頭崗站|陳頭崗站]]'''：换乘[[佛山地铁4号线|佛山地铁4号线]]。佛山地鐵4號線為遠期線路，預留情況未知。
* '''[[广州南站_(地铁)|广州南站]]'''：换乘[[廣州地铁2号线|2号线]]、[[廣州地铁7号线|7号线]]及[[佛山地铁2号线|佛山地铁2号线]]。22号线車站位於廣州南站東廣場，並以長通道連接既有地鐵車站。
* '''[[番禺廣場站|番禺廣場站]]'''：换乘[[广州地铁3号线|3号线]]、[[广州地铁17号线|17号线]]及[[广州地铁18号线|18号线]]。因3號線建造时未作预留，预计需经通道换乘；17號線為遠期線路，預留情況未知。

===獲預留結構的車站===
* '''[[棠溪站_(地鐵)|棠溪站]]'''：未來新建的[[棠溪站|棠溪火車站]]將會同步建設包括22號線在內的地鐵樞紐換乘站，可换乘[[广州地铁12号线|12号线]]、[[广州地铁24号线|24号线]]及[[佛山地铁8号线|佛山地铁8号线]]，同時預留一條新規劃線路接入；以及前往[[石潭站|石潭站]]換乘[[广州地铁8号线|8号线]]。因22號線車站位於西廣場，與位於東廣場的12號線、24號線及佛山8號線有一定的距離，只能通過站廳換乘；新規劃線路預留站位則同樣位於西廣場，可與22號線實現站台樓梯節點換乘；同時車站將設有換乘通道，連接至8號線石潭站實現換乘<ref>[http://www.gzggzy.cn/cms/wz/view/index/layout3/index.jsp?siteId=1&channelId=503&infoId=567632 广州白云（棠溪）站综合交通枢纽一体化建设工程施工总承包项目土建工程1标、3标招标公告] 2018-9-30</ref>。

== 列车 ==
全線預定使用8辆编组市域快線列车，最高运行速度160km/h<ref name=二次环评 />。
== 行車安排 ==
二十二号线初期采用站站停运营模式，並與18號線番禺廣場至萬頃沙段共線運行；近、远期南延独立运营后采用快慢車的运营模式。
