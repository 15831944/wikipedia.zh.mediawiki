'''广东肇庆中学'''，也称'''肇庆中学'''，简称'''肇中'''，曾名为'''肇庆府中学堂'''、'''省立肇庆中学'''、'''省立第七中学'''、'''肇庆市第三中学'''，是一所位于[[广东省|广东省]][[肇庆市|肇庆市]][[端州区|端州区]]的百年老校。该校建立于清光绪三十一年（1905年），是肇庆市历史悠久的中学之一，其前身为于明万历元年（1573年）创立的'''两广端溪书院'''。该校目前直属于肇庆市教育局，是广东省十六所重点中学之一，也是肇庆市唯一的省重点中学。

广东肇庆中学是一所历史悠久、业绩辉煌的省重点中学，坐落在风景秀丽的国家级历史文化名城——肇庆端州。素有“西江最高学府”之称。是全省十六所重点中学之一。以良好的校风、雄厚的师资、鲜明的特色、显著的效益而享誉岭南。<ref name=history></ref>

1994年被评为首批省一级学校，1997年被确定为首批全国现代教育技术实验学校，2002年被定为“十五”期间教育部重大教育科研基地和共青团全国青少年科技活动示范单位，2004年被定为广东省新课程改革试验样本校。2007年10月，学校以优秀等级通过广东省普通高中教学水平评估，成为广东省首批国家级示范性普通高中。2008年至今先后被评为“广东省安全文明校园”、“广东省依法治校示范校”、“广东省书香校园”、“广东省中小学校本培训示范校”、 “全国消防安全教育示范学校”、“广东省青少年科学教育特色学校”、“广东省创先争优‘南粤先锋’先进基层党组织”、“广东省德育示范学校”、“第五届和谐校园先进学校”。<ref name=history></ref>

2017年11月，获第一届全国文明校园称号。<ref>{{Cite news|url=http://www.xinhuanet.com/politics/2017-11/17/c_1121973853.htm|title=第五届全国文明城市、文明村镇、文明单位和第一届全国文明校园名单|author=新华社|date=|work=|accessdate=}}</ref>

== 历史沿革 ==

*1573年，[[明朝|明朝]][[万历|万历]]元年，分巡岭西道副使[[李材|李材]]在高要县学宫西侧创办'''端溪书院'''。
*1708年，[[清朝|清朝]][[康熙|康熙]]四十七年，[[两广总督|两广总督]][[赵宏灿|赵宏灿]]改端溪书院为'''天章书院'''，藏书丰富，为两广规模较大的学府。
*1752年，清朝[[乾隆|乾隆]]十七年，文学家、学者全祖望担任肇庆天章书院主讲。
*1757年，清朝乾隆二十二年，肇庆知府[[吴绳年|吴绳年]]主持扩建天章书院，并再名为'''端溪书院'''。端溪书院是一组一路五进的建筑群。第一进是面阔三间的[[硬山|硬山]]门厅，左右衬厅各一间，门厅前有[[照壁|照壁]]一幅，照壁左右各有牌坊一座。第二进是广德堂，硬山顶，面阔三间，左右衬厅亦各一间。第三进是棪天阁，[[歇山顶|歇山顶]]，面阔五间，两层间有腰檐和平座，楼下大厅称教忠堂。阁前东侧是[[监院|监院]]、西侧是书库。第四进是宣教堂，面阔五间，东西为厨房，堂前是天井，天井前有门一座，门前东西各有七间斋舍。第五进是后楼，面阔九间，中间三个开间上层称景贤阁，下层称全先生祠（祀[[全祖望|全祖望]]），东边三个开间是更衣所，西边三个开间是祭器所。第四进和第五进间是庭院，自南向北沿中轴线依次布置有澂鑑亭（面阔三间，卷棚顶）、半月形莲池、近光亭（即后来的爱莲亭，面阔三间，歇山顶）。庭院东西各有斋舍九间。经过民国十二年（1923）军阀混战时火灾和抗日战争时日机轰炸的洗劫后，书院内原有的建筑和藏书几乎尽毁。
*1905年，清朝[[光绪|光绪]]三十一年，广肇罗道道台[[蒋式芬|蒋式芬]]和肇庆知府多龄在端溪书院原址创办'''肇庆府中学堂'''，这是肇庆地方政府最早创办的近代化中学。也为现代肇中的正式开端。
*1912年，[[民國紀年|民国]]元年，改名为'''省立肇庆中学'''。
*1925年，民国十四年，改名为'''省立第七中学'''。
*1932年，民国二十一年，省立第七中学东邻的高要县学宫划入学校范围以扩建校舍（同时将肇庆府学宫划给高要县管理）。
*1935年，民国二十四年，复名为'''省立肇庆中学'''。
*抗日战争期间，校址曾几度迁移，1939年2月迁往德庆县金林乡，1944年又迁往广宁县木格罗家祠，1945年9月迁回原址。
*1948年，民国三十七年，五月肇中学生罢课4天，反对学校随意处分学生，反对滥收学米、膳费。
*1949年11月，学校正式定名为'''广东肇庆中学'''。
*1964年5月，被广东省教育厅定为省重点中学。
*1971年2月，改名为'''肇庆市第三中学'''。
*1978年4月，复名为'''广东肇庆中学'''。
*2001年9月，实行初高中分开办学。并把高中部迁移到仙女湖畔的新校区，实施全寄宿教学。

== 学生活动 ==
广东肇庆中学的课余活动众多，至2018年春季为止，已举办了六届健康体育节，两届人文国学节，两届文化艺术节，两届科技创新节，四届文化创意市集等项目，各节期间的系列活动有“我能我秀”才艺表演赛，十大歌手，校园辩论赛等。

==设施及资源==

=== 初中部 ===
[[File:Gdzqzx_Old_PlayG.jpg|thumb]]
初中部有基本教学楼3幢，实验楼1幢，图书楼1幢（又名爱莲楼），电教楼1幢，行政楼1幢（校友楼），教师宿舍、学生宿舍各1幢。

初中部有一大一小两个操场、一个乒乓球场和若干可供体育活动的空地。

初中部的范围基本上包括了原高要县学宫和端溪书院。有一条用麻石铺成的主校道，被称为石板路，具有历史价值，部分石板上仍有模糊的浮雕纹样，内容有松、鹿、朝阳、卷云等。而石板路旁常常用作集结点的芒果树更是肇中的象征。

==== 学校食宿 ====
初中部饭堂主要服务住校老师及非本地的住宿生，规模较小。

初中部宿舍主要供非本地或归家不方便的学生寄宿，八人一室，没有空调。男女宿舍靠楼层分离。

===高中部===
高中部有教学楼两幢，一幢是高一高二教学楼（中间包含行政楼），另一幢是高三楼；有图书楼一幢；饭堂两幢；学生宿舍两幢；教师宿舍两幢和体育馆一幢.

====图书馆====
高中部图书馆在高中部启用初期创立，图书主要由师生捐赠。
图书馆位于图书楼西区3楼至5楼，学生可凭学生卡到图书馆借书。每位学生最多借3本，若是帮图书馆做工就可借6本，为期15天，可续期一次。逾期不还将按“迟一天，禁一天”惩罚。

2010年9月21日上午，受[[强台风凡亚比_(2010年)|强台风凡亚比]]影响，图书楼部分装饰结构损毁，由于事发时正在上课，没有人在其下走动，故没有造成人员伤亡。

==== 篮球场 ====
篮球场是最热门的运动场所，校内共有8个标准场，按西到东，南到北，标为1—8号场，其中5号场（最近教书宿舍的）为教师专用，紧挨的6号场为新疆部专用。其余6个场地为高一到高三共有，每个年级各有两个。
