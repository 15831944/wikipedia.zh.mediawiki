'''广州市脑科医院'''（'''广州市精神病医院'''，俗稱'''芳村精神病院'''）位于[[广州市|广州市]]，于1898年由美国人[[嘉约翰|嘉约翰]]（[[:en:John_Glasgow_Kerr|John Kerr]]）在美國基督教長老會支持下以'''惠愛醫癲院'''（{{lang|en|Ophthalmic Hospital}}）<ref>{{zh-cn}}{{cite web |url = http://news.sina.com.cn/o/2008-11-28/040714797182s.shtml |title=全國首家精神病院110年前建於廣州}}</ref>為名创办，是中国與亞洲第一间精神病专科医院<ref>{{Cite web|url=http://www.gospeltimes.cn/index.php/portal/article/index/id/26017|title=广州基督教会为传教士医生嘉约翰及其家人举行遗骸落土礼|accessdate=|author=广州市基督教两会|date=2014年12月02日|publisher=}}</ref>，從建院至今已經有超過一百年的歷史，曾用名'''廣州市第十人民醫院'''<ref>因此亦被戲稱「傻十」</ref>。设[[芳村|芳村]]、[[江村|江村]]、[[荔湾|荔湾]]三个[[醫院#門診部門|门诊]]以及[[芳村|芳村]]、[[江村|江村]]两个[[醫院#住院部門|住院部]]<ref name="醫院簡介">[http://www.gzbrain.cn/subindex3.jsp?channelid=8&itemid=19&sampl=0 醫院簡介]{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }} 廣州市腦科醫院</ref>。廣州市腦科醫院亦是[[三級甲等醫院|三級甲等專科醫院]]<ref>[http://www.gzbrain.cn/subindex3.jsp?channelid=8&itemid=22&sampl=0 院長寄語] {{Wayback|url=http://www.gzbrain.cn/subindex3.jsp?channelid=8&itemid=22&sampl=0 |date=20171117174706 }} 廣州市腦科醫院</ref>、广州市首批[[医保|医保]]定点医疗机构<ref name = "醫院簡介" />。

該醫院為中國最早的一家西醫精神病醫院<ref>{{Cite web |url=http://www.xwgd.gov.cn/xwgd/News.shtml?p5=114988 |title=广州市文化广电新闻出版局-广州市文物保护单位-惠爱颠医院旧址 |accessdate=2013-06-19 |work=广州市第三次全国文物普查领导小组办公室 |publisher=广州市文化广电新闻出版局 |date= }}</ref>，對研究清代時期廣州地區引入西醫學術文化有較高的歷史價值。

=== 早期 ===
创办人[[嘉约翰|嘉约翰]]是[[美国|美国]][[俄亥俄州|俄亥俄州]][[基督教长老会|基督教长老会]]的[[牧师|牧师]]，于1854年间来广州行医与教学。在19世纪末大陆未曾有专门收治精神病患的医疗服务机构时，无法得到帮助的患者时常受虐待、嘲笑与打骂。嘉约翰见此，决心建院帮助这类群体。虽当时遭到[[两广总督|两广总督]][[张之洞|张之洞]]的阻挠，仍在[[芳村|芳村]]大沙地购得17亩地，完成志愿。
* 1898年2月28日，'''惠爱医癫院'''（'''惠愛醫院'''，{{lang|en|John G. Kerr Refuge for the Insane}}）正式开业，初设30张病床，当日仅收治到一男一女两位病人入院。
* 1927年3月1日，[[中華民國|中華民國]][[廣州市政府|廣州市政府]]接办惠爱医院，改名广州市立第二神经病院。
* 1935年6月，广州市政府全面收购惠爱医院物业，包括二院及[[广州市培英中学|培英中学]]初中部。12月26日，市府撤销第一神经病院，两院合并易名为广州市立精神病疗养院。
* 1938年10月广州沦陷。广州市政府撤退前，委托[[天主教广州总教区|天主教廣州代牧區]]代管精神病醫院。天主教派了三名[[修女|修女]]主管全院七年有余，直到[[中國抗日戰爭|抗战]]胜利。
* 1946年初，广州市政府将医院收回。
* 1949年12月，因[[中華民國政府|中華民國政府]]撤退至[[台灣|台灣]]，[[中国人民解放军广州市军事管制委员会|中国人民解放军广州市军事管制委员会]]派人接管了精神病院，並將其改名為廣州市市立精神病院。

=== 1949年後 ===
* 1955年5月改名為廣州市精神病院。
* 1958年10月改名為廣州市精神病防治院。
* 1969年[[文化大革命|文化大革命]]期間，改名為廣州市第十人民醫院。
* 1973年改名為廣州市精神病醫院。
* 2000年获准增挂廣州市腦科醫院院牌。

== 歷史建築 ==
[[File:Refuge_for_the_Insane_Canton.jpg|thumb]]
惠愛顛醫院舊址今僅存建院初期的行政辦公樓一幢，坐西北朝東南，兩層磚混結構的券廊建築，長21.1米，寬15米，建築佔地面積316.5平方米。兩層用簡潔方柱、四周券拱式柱廊圍繞，設瓶式欄杆，外牆紅磚砌築。在20世紀90年代修繕時，外牆改貼紅色瓷磚。

在廣州市文化廣電新聞出版局管轄的市文物保護單位名錄中，「惠愛醫癲院」被寫作「惠愛顛醫院」。<ref>{{Cite web |url=http://www.xwgd.gov.cn/xwgd/News.shtml?p5=114988 |title=惠爱颠医院旧址 |work=广州市第三次全国文物普查领导小组办公室 |publisher=廣州市文廣新局 }}</ref>

==文化影響==
由於芳村惠愛醫癲院是廣州乃至中國第一家精神病院，廣州人常把「芳村」一詞與精神病畫上等號。「去芳村」這個用語除了有「去芳村這個地方」的意思外，亦暗含着謾罵他人的意思，意味著一個人有精神問題。近年出現一些新的洐生詞，如「芳村需要你」、「芳村歡迎你」等。
