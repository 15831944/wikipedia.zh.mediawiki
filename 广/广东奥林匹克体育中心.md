'''广东奥林匹克体育中心'''位于[[中国|中国]][[广州市|广州市]][[天河区|天河区]][[东圃|东圃]]，是2001年举行的[[中华人民共和国第九届全国运动会|中华人民共和国第九届全国运动会]]的主赛场，和2010年举行的[[第十六届亚洲运动会|第十六届亚洲运动会]]主场馆以及[[2010年亚洲残疾人运动会|亚洲残疾人运动会]]的开、闭幕式场地，为新[[羊城八景|羊城八景]]之一<ref>[http://www.gz2010.cn/special/0078sp/20103dvenue.html 奥林匹克体育中心] {{webarchive|url=https://web.archive.org/web/20140602081450/http://www.gz2010.cn/special/0078sp/20103dvenue.html |date=2014-06-02 }}，广州2010年亚洲运动会官方网站，2010年月12月13日查阅。</ref>。

== 場館資料 ==
广东奥林匹克体育中心占地面积101万[[平方米|平方米]]，总建筑面积32.8万平方米，投资16.7亿元[[人民币|人民币]]，是[[广东省|广东省]]政府为承办2001年举行的[[中华人民共和国第九届全运会|第九届全国运动会]]而兴建的[[体育场|体育场]]馆。2010年，广东奥林匹克体育中心亦成为[[第十六届亚洲运动会|第十六届亚洲运动会]]的主場館。

== 体育设施及辅助设施 ==
[[足球|足球]]场、[[田径|田径]]场、[[马术|马术]]场、[[射箭|射箭]]场、[[棒球|棒球]]场、[[垒球|垒球]]场、[[曲棍球|曲棍球]]场、体育科技中心、新闻会议中心、药检中心、体育俱乐部。还有[[会所|会所]]、[[商场|商场]]、[[宾馆|宾馆]]以及大型休闲、娱乐、康复等设施。

== 奥林匹克体育场 ==
其主体建筑奥林匹克体育场位于奥林匹克体育中心南部，总占地面积14.56万平方米，可容纳观众80,012人。体育场自1998年12月30日动工，至2001年9月30日竣工，承办了[[中华人民共和国第九届全运会|九运会]]的开幕式和[[田径|田径]]、[[足球|足球]]等比赛项目。2002年以“五环晨曦”入选[[羊城八景|羊城八景]]。
