'''广东实验中学系列学校'''（又称“广东实验中学教育集团”，简称“大省实”、“省实系学校”），是指在中国广东省由直屬于廣東省教育廳的國家級重點中學[[广东实验中学|广东实验中学]]管理的一系列学校，是在2004年广东实验中学高中校区落成后以及2005年广东实验中学附属天河学校开设后形成的系列学校格局，学校分布在[[广东省|广东省]]的[[广州|广州]]、[[珠海|珠海]]等地。<ref>{{Cite web |url = http://www.sohu.com/a/252151968_712657 |title = “省实系”大揭秘！“一校九区”办学格局 |author = 卓越教育 |date = 2018-09-05 |publisher = 搜狐 |language = zh |accessdate = 2019-02-15 }}</ref>省实系学校办学形式各有不同，但将资源共享、学生管理、师资调配和教研共训等方面进行统一管理，并在部分学校实行集团化办学。各校的学校章程、规章制度、校园文化等方面均和广东实验中学实行统一标准。

截至2018年底，广东实验中学已形成“一花九叶”的一校九区办学格局。<ref>[http://www.gdsyzx.edu.cn/aboutus.php 广东实验中学简介]</ref>

== 历史 ==
* 2004年8月，于芳村坑口的广东实验中学新校区落成，翌年（2005年）高中部迁入，原校址改为初中部校区。
* 2004年4月29日，广东实验中学与广州市实验教育发展有限公司签订合作辦學協議書，宣布共同创办广东实验中学附属天河学校。<ref>{{cite news|language=zh-hans|author=杨华联|url=http://flash.gdsyzx.edu.cn/xiaokan/xyjj.pdf|title=广东实验中学与广州市实验教育发展有限公司联合创办广东实验中学附属天河学校|publisher=校园聚焦|pages=13-14|date=2005-01-15|accessdate=2019-02-12}}</ref>
* 2005年，广东实验中学附属天河学校开学。<ref name=history>{{Cite journal|author=广东教育杂志社|title=广东实验中学附属天河学校|journal=广东教育|year=2009|volume=综合版|issue=7|page=138}}</ref>
* 2008年，于[[佛山市|佛山市]][[顺德区|顺德区]]设立了广东实验中学顺德学校。<ref>[http://gdsysd.sdedu.net/page/IndexLeft/2/12 广东实验中学顺德学校简介]</ref>
* 2009年4月18日，广东实验中学与里水镇政府、中信保利达地产（佛山）公司签订合作办学意向书，共同举办广东实验中学南海学校。<ref>[http://www.gdsyzxnh.com/about.asp?sortid=36 广东实验中学南海学校筹建纪事]</ref>
* 2010年9月1日，广东实验中学附属南海学校开学。<ref>[http://www.gdsyzxnh.com/about.asp?sortid=36 广东实验中学南海学校筹建纪事]</ref>
* 2014年，于[[佛山市|佛山市]][[顺德区|顺德区]]设立了[[广东实验中学顺德学校#小学部|广东实验中学顺德学校附属小学]]。
* 2016年，珠海市人民政府于[[珠海市|珠海市]][[金湾区|金湾区]]设立珠海市金湾区第一中学并交给广东实验中学管理，实际名为[[广东实验中学珠海金湾学校|广东实验中学珠海金湾学校]]。
* 2018年2月26日，广东实验中学珠海金湾学校举行2018学年第二学期开学典礼暨学校启用仪式，此前该校借用珠海市金湾区金海岸中学的校舍进行办学。<ref>{{cite news |language = zh |author = 珠海电台先锋951 |url = http://baijiahao.baidu.com/s?id=1604571917544233796&wfr=spider&for=pc |title = 省实珠海金湾学校启用 有这八大特色你知道吗？ |publisher = 新浪看点 |date = 2018-02-27 |accessdate = 2019-02-15}}</ref>
* 2018年5月19日，[[广州市越秀外国语学校|广州市越秀外国语学校]]与[[广州市第二十一中学|广州市第二十一中学]]重组，并宣布筹建[[广东实验中学越秀学校|广东实验中学越秀学校]]。<ref>{{cite news |language = zh |author = 广州日报 |url = http://www.guangzhou.gov.cn/node_2190/node_2215/2018/05/26/1527305459727231.shtml |title = 21中、越秀外国语学校并入省实 |publisher = 中国广州网 |date = 2018-05-26 |accessdate = 2019-02-15}}</ref>
* 2018年6月22日，广东实验中学与广州广联教育发展有限公司签署合作协议，宣布共同举办[[广州市白云实验学校|广州市白云实验学校]]（[[白云省实|白云省实]]）。<ref>{{cite news |language = zh |author = 白云视野 |url = https://www.sohu.com/a/237346814_99933661 |title = 定了！白云区新开“白云省实”学校，落户夏茅，12年一贯制，九月招生！ |publisher = 搜狐 |date = 2018-06-22 |accessdate = 2019-02-15}}</ref>
* 2018年6月29日，荔湾区政府与广东实验中学签订合作共建广东实验中学荔湾学校框架协议，还签订合作共建教育集团的协议。<ref>{{cite news |language = zh |author = 广州日报 |url = http://k.sina.com.cn/article_2492261432_948ce438019007n63.html?from=news&subch=onews |title = 重磅！省实落户荔湾开办三所公办中小学！三校区共102个班 |publisher = 百度百家号 |date = 2018-06-29 |accessdate = 2019-02-15}}</ref>
* 2018年8月1日，越秀区教育局与广东实验中学签订合作办学协议书，宣布广东实验中学越秀学校正式成立并组建广东实验中学越秀教育集团。<ref>{{cite news |language = zh |author = 信息时报 |url = https://www.sohu.com/a/244642826_703551 |title = 省实越秀学校今日正式成立！华侨外国语并入省实越秀教育集团 |publisher = 搜狐 |date = 2018-08-01 |accessdate = 2019-02-15}}</ref>
* 2018年9月1日，广东实验中学越秀学校举办学校以及集团的揭牌儀式暨開學典禮<ref>{{cite news |language = zh |author = 信息时报 |url = http://news.sina.com.cn/o/2018-09-01/doc-ihinpmnr3265659.shtml |title = 越秀再添“牛校”！ 省实越秀集团今早正式揭牌 |publisher = 新浪新闻 |date = 2018-09-01 |accessdate = 2019-02-15}}</ref>，广州市白云实验学校（白云省实）举办开学典礼。
* 2018年11月8日，珠海市金湾区政府与广东实验中学在省实珠海金湾学校签订合作办学框架协议，宣布共同举办[[广东实验中学珠海金湾学校附属初级中学|广东实验中学珠海金湾学校附属初级中学]]。<ref>{{cite news |language = zh |author = 珠海特区报 |url = http://www.sohu.com/a/274427874_401397 |title = 广东实验中学在珠海设公办初中！30个班！地点敲定！ |publisher = 搜狐 |date = 2018-11-10 |accessdate = 2019-02-15}}</ref>
