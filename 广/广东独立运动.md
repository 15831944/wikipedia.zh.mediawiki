'''广东独立'''，簡稱'''粤獨'''，指的是认为[[广东|广东]]應不受外界管轄和干擾，建立独立自治的政治实体的主张与運動。
在中国近代，包括康有为弟子（后与康因主张而反目）[[欧榘甲|欧榘甲]]在内的等人均提出过这一主张。<ref>{{cite journal |last=宋德华 |first= |last2=刘雪琴 |first2= |date= |title=辛亥前期欧榘甲革命自立主张探析——以《新广东》为中心 |url= |journal=南方职业教育学刊 |volume= |issue= |pages=40-47 |doi= |access-date=2017-08-29 }}</ref>欧氏在其著作《[[新广东|新广东]]》中提出了建立“广东人之广东”的主张。1911年[[辛亥革命|辛亥革命]]，10月底，广东同盟会会员[[陈炯明|陈炯明]]、[[鄧鏗|鄧鏗]]和[[彭瑞海|彭瑞海]]等人在广东各地组织民军起义。11月9日，陈炯明率部光复[[惠州市|惠州]]，同日，广东宣布独立，成立[[中华民国粤省军政府|中华民国粤省军政府]]。1912年1月1日，中华民国成立，广东省成为中华民国中的一省。在[[中华民国|中华民国]]初年，受[[聯省自治|聯省自治]]思想影响，广东省起草了《[[广东省宪法草案|广东省宪法草案]]》<ref name="constitution">{{Cite book |title=《自下而上的立宪尝试：省宪评论》 |publisher=武汉大学出版社 |year=2010年 |isbn=9787307075603 |location=武汉市 |language=zh-cn}}</ref>，并于1921年12月19日由广东省议会通过。但是這個对广东省未来规划的主張並沒有獲得足夠支持，并且随着苏联势力介入远东以及国共两党北伐而流产。

== 历史 ==
=== 何启、李鸿章、卜力与流产的“两广独立” ===
[[清朝|清朝]]末年，内忧外患，[[义和团运动|义和团运动]]的迅速发展引起了清朝统治集团内部的分裂。[[慈禧太后|慈禧太后]]于1900年6月21日发布招抚义和团和向八国联军宣战的上谕。掌握地方大权的多数督抚强烈反对，坚决支持主和派，形成地方与中央公开对抗与分裂的局面。清政府宣战后，時任[[兩廣總督|兩廣總督]][[李鴻章|李鴻章]]、[[兩江總督|兩江總督]][[劉坤一|劉坤一]]、[[湖廣總督|湖廣總督]][[張之洞|張之洞]]、[[閩浙總督|閩浙總督]][[許應騤|許應騤]]、[[山東巡撫|山東巡撫]][[袁世凱|袁世凱]]等[[總督|總督]]、[[巡撫|巡撫]]等人奉行[[東南互保|東南互保]]，公然违背[[清政府|清政府]]的命令，不向外国开战。李鸿章更对清帝国向西方十一国宣战的诏书发出了极其强硬的回应：“此乱命也，粤不奉诏！”。此时东南各省抗拒朝廷的外交政策，自行与西方国家签订条约，实际上成为了脱离清中央政府控制的自治政体。而广东人素来有独特的身份认同，较早受现代化洗礼，此时渐渐出现分离主义诉求<ref> {{cite book |title=《民初的地方主义与联省自治》|year=1983年 |publisher=正中书局 |pages= |language = zh-tw}}</ref>。广东的士绅向李鸿章建议割据两广独立建国，往来粤港的商人亦寻求香港政府的协助，香港政府亦有意推动借此扩张英国在两广的利益。

与此同时，革命派已打算趁清廷势弱在岭南建立根据地。早在5、6月之交，当清廷尚未对各国宣战时，香港[[定例局|定例局]]非官守议员，同时也是首个华人爵士[[何启|何启]]就在请示港英政府后找到了留守香港的兴中会党人陈少白，建议兴中会与李鸿章合作，建立两广独立政府。陈少白表示赞同，随即与在日本的孙文联络，香港方面亦开始与李鸿章接触。

1900年6月初，李鸿章透过机要幕僚，广东士绅刘学询与孙文接洽，希望邀请孙到广州商谈合作事宜。刘学询对此计划亦十分热衷。随后，刘学询向孙文发出电文：“因北方拳乱，欲以粤省独立，思得足下为助，请速来粤协同进行”<ref>{{cite journal |last=董丛林 |first= |date= |title=李鸿章对“两广独立”的态度与庚子政局 |url= |journal=河北师院学报 |volume= |issue= |doi= |access-date=2018-10-3 }}</ref>。孙文当时正专注于准备在惠州起义，他对此的态度是机会主义式的，也认为不妨接触：“此举设有成，亦大局之福，故亦不妨一试”<ref name="sunys"> {{cite book |title=《孙中山年谱长编》|year=1991年 |publisher=中华书局 |pages= |language = zh-cn}}</ref>。6月15日后，清政府迭电谕令李鸿章北上述职，李鸿章心存观望，借故拖延。

何启的角色是要争取[[香港总督|香港总督]][[卜力|卜力]]的支持。何启首先与部分兴中会成员讨论了两广独立后改造中国的方案《华南志士致港督书》，再由何启译为英文交给卜力。何启起草的被称为《平治章程》的纲领支持在列强托管下重组中国的政治制度，内容包括“遷都於適中之地。如南京、漢口等處”，“於各省立一自治政府，以資分理”，将司法、教育制度西化，向外商开放中国市场，以及建立议会制度：

6月11日，孙文偕杨衢云、郑士良、宫崎寅藏等人自横滨启程，于17日抵香港海面，遇到了李鸿章派来迎接他的军舰。由于担心此乃清国诱捕他的陷阱，孙文不敢贸然上船，便派宫崎等三名享有治外法权的日本人代其赴会。当夜10时许，宫崎等人抵达刘学询在广州的公馆，双方开始秘密谈判。在谈判中，刘学询表示在列强攻陷北京前，李鸿章“不便有所表示”，暗示独立之事需待北京易手后进行。宫崎则提出，李鸿章应保障孙文安全，并向兴中会借款六万元作为双方合作基础。刘学询当即请示李鸿章后表示同意，并当面付款三万元。次日凌晨3时，长达五小时的秘密谈判结束，宫崎等人随即趁夜返回香港，孙文亦于当日启程，经河内前往新加坡。

为了维护英国在珠江流域的特殊利益，[[港英政府|港英政府]]也希望李鸿章留在广东，对李鸿章与革命派合作之事兴趣甚大，支持何提出的李鸿章与[[兴中会|兴中会]]合作实行兩廣独立的计划。卜力竭力推动李鸿章与孙文联合。卜力和孙文许诺李鸿章，如果独立成功，将推举李鸿章出任“两广共和国”的“总统”。李鸿章为稳定[[华南|华南]]局势，对这个计划颇为心动，与革命派进行谈判并给予贷款。

6月11日，孙文偕杨衢云、郑士良、宫崎寅藏等人自横滨启程，于17日抵香港海面，遇到了李鸿章派来迎接他的军舰。由于担心此乃清国诱捕他的陷阱，孙文不敢贸然上船，便派宫崎等三名享有治外法权的日本人代其赴会。当夜10时许，宫崎等人抵达刘学询在广州的公馆，双方开始秘密谈判。在谈判中，刘学询表示在列强攻陷北京前，李鸿章“不便有所表示”，暗示独立之事需待北京易手后进行。宫崎则提出，李鸿章应保障孙文安全，并向兴中会借款六万元作为双方合作基础。刘学询当即请示李鸿章后表示同意，并当面付款三万元。次日凌晨3时，长达五小时的秘密谈判结束，宫崎等人随即趁夜返回香港，孙文亦于当日启程，经河内前往新加坡。

为了维护英国在珠江流域的特殊利益，[[港英政府|港英政府]]也决心尽力阻止李北上，留在广东建国，支持何提出的李鸿章与[[兴中会|兴中会]]合作实行兩廣独立的计划。卜力竭力推动李鸿章与孙文联合。卜力和孙文许诺李鸿章，如果独立成功，将推举李鸿章出任“两广共和国”的“总统”。李鸿章为稳定[[华南|华南]]局势，对这个计划颇为心动，与革命派进行谈判并给予贷款。6月20日，署任港督[[加士居|加士居]]少将电报殖民地部，要求伦敦准许港府介入李鸿章的行程。但英国首相[[罗伯特·加斯科因-塞西尔，第三代索尔兹伯里侯爵|梳士巴利]]于22日下令禁止港府介入。港督卜力之后结束行程，于7月2日返港，即透过何启接触兴中会，并继续游说伦敦。

然而随着战局的恶化，清廷内主和声音开始出现，李鸿章于7月8日再次接到清廷电报。慈禧任命[[李鸿章|李鸿章]]为[[直隶总督|直隶总督]]兼[[北洋大臣|北洋大臣]]，李鸿章觉察到朝政有开始向有利于主和派方向发展的可能，决定改变态度，于是决定离粤北上<ref name = "沈晓敏 倪俊明／著">{{Cite book zh | author = 沈晓敏 倪俊明／著 | title = 《喋血南国：辛亥革命在广东》 | format =  | url = http://vip.book.sina.com.cn/pub/book.php?book=814081 | date = 2006年 | publisher = 广东人民出版社 | location = 广州 | id = ISBN 978-7-218-07223-4 | accessdate =  | pages = 頁数：237頁}}</ref>。离开前，他首先乘船从广州抵达香港拜会港督卜力，当面向港府表态<ref> {{cite book |title=China, Britain and Hong Kong, 1895-1945|year=1990年 |publisher=Chinese University Press |pages= |language = en}}</ref><ref name="cbgihkapcisc"/>。但卜力此时仍未放弃，亦得知孙文正从新加坡启程到香港，同时向伦敦发出一份紧急电报：

在两广独立计划失败以后，何启继续试图游说港英方面关于他改造中国王朝的思想与计划。7月21日，何啟报道称，卜力支持华南成立一個共和國。8月1日，何在英文报纸《德臣西報》上發表一篇以此政治綱領為依據的文章<ref> {{cite book |title=《中华民国革命秘史》|year=1981年 |publisher=广东人民出版社 |pages= |language = zh-cn}}</ref>，在8月10日更进一步在报刊上发表关于在中国南部建立“独立共和国”的文章<ref> {{cite book |title=《民权与民族主义 近代中国民权思想研究》|year=2016年 |publisher=中国政法大学出版社 |pages= |language = zh-cn}}</ref>。8月22日，何启在《[[德臣西报|德臣西报]]》上用“Sinensis”為筆名，“An Open Letter on the Situation”（關於局勢的公開信）為題发表了公开信总结自己的观点，信的收件人是“[[約翰牛|約翰牛]]先生”：

=== 广东独立协会 ===
清廷在多次对外战争中政府对外割地赔款签订[[不平等条约|不平等条约]]。1901年春，因传说清廷将把广东割让与法国。留日粤籍学生[[冯斯栾|冯斯栾]]、[[郑贯一|郑贯一]]、[[李自重|李自重]]、[[王宠惠|王宠惠]]、[[冯自由|冯自由]]及[[梁仲猷|梁仲猷]]等人发起了名为[[广东独立协会|广东独立协会]]的组织<ref>{{cite book |title=《中国近代史词典》|year=1982年 |publisher=上海辞书出版社 |location=上海市 |pages=34 |language = zh-cn}}</ref>，主张广东向清廷宣告独立，强调广东省是广东人的广东，命运应有广东人决定，而非清政府<ref>{{cite book |title=《法官外交家王宠惠》|publisher=福建教育出版社|author=祝曙光|language = zh-cn}}</ref>。在日粤侨响应者众，“参加者二百余人”<ref>{{cite book |title=《华侨革命开国史》|author=冯自由|language = zh-cn}}</ref>。组织发起者们当时亦曾拜访当时居于[[横滨|横滨]]的孙文商讨筹商办法。广东留日学生与[[兴中会|兴中会]]的合作便由此开始<ref name="fungbook">{{cite book |title=《革命逸史》|author=冯自由|year=1939年 |publisher=商务印书馆 |location=上海 |language = zh-cn}}</ref>。该会的成立推动了各省自立的风潮。这个组织虽然不是纯粹的留学生团体，但是以留学生为基干，由于事出于“割让广东”的谣言，而协会也是同乡会性质，该会成立不久后便停止活动。

=== 欧榘甲与《新广东》 ===
{{main|欧榘甲}}
清末的中国局势引发了知识分子及社会人士的思考。他们从各种立场、各种角度出发，提出了许多迥异的主张及方案。[[康有为|康有为]]及其弟子主张通过改革上书的方式，建立以[[中华民族|中华民族]]为主体的[[君主立宪制|君主立宪制]]大一统国家。而[[孙文|孙文]]则带领[[同盟会|同盟会]]以及其他组织在各地宣传革命，主张“驱逐鞑虏，恢复中华”。

[[甲午战争|甲午战争]]战败后，清廷被迫于该年4月签订《[[马关条约|马关条约]]》将[[台湾|台湾]]及周边岛屿割让予日本，遭到台湾住民强烈反对，台湾部分仕绅因而鼓动前清台湾巡抚[[唐景崧|唐景崧]]在该年5月25日宣布建国，成立了亚洲第一个以“民主国”自称的国家[[台湾民主国|台湾民主国]]。台湾民主国虽然仅短暂存在，却为后来各省独立自治的主张启迪了思路，并产生了密切影响<ref name="wongsiuyeung">{{cite journal |last=孙柏|date= |title=《黄萧养回头》作者为欧榘甲考―兼论欧榘甲在前期《新小说》作者群中的重要地位|url= |journal=戏剧艺术 |volume= |issue= |pages=29-35 |doi= |access-date=2017-08-29 }}</ref>。

在1902年，康有为弟子欧榘甲在其任总编辑的旧金山《大同日报》上，以笔名“太平洋客”连载一篇共五六萬字的长文《广东独立脱离满清羁绊之说》，分載數十日，主张广东独立及脫離滿清之必要，大為讀者歡迎。这组政论文很快由日本横滨的《[[新民丛报|新民丛报]]》转载，并由新民丛报社印成了题名《新广东》，又名《广东人之广东》的单行本出售，卷首有彩色的广东地图并附有美国[[独立厅|独立厅]]图。他在文中如此解释自己所取的标题以及主张：
{{Quote box
 |title = 
 |quote = 
夫治公事者不如治私事之勇，救他人者不如救其家人亲戚之急，爱中国者不如爱其所生省份之亲。人情所趋，未如何也。故窥现今之大势，莫如各省先行自图自立……吾广东人，请言自立自广东始。姑名是议曰“新广东”，以谂我广东人欲享新国之福份者。
 |source = ——欧榘甲：《新广东》。<ref name="newcan" >{{cite book |title=《辛亥革命前十年间时论选集》|year=1960年 |publisher=生活·读书·新知三联书店 |location=北京市 |pages= |language = zh-cn}}</ref>
|align = center|width = 100%|border = 3px|fontsize = 20|bgcolor =  lightgray|title_bg = gray |quoted =5px|qalign = left|salign = right}}{{clear}}
欧榘甲认为“中国人言自立者，以台湾为起点”，并以[[谭嗣同|谭嗣同]]、[[唐才常|唐才常]]等创办的《[[湘报|湘报]]》“为中国各省报言自立者始”。他在文中列出了他所认为的四点广东的“自立特质”：“人才之出众”，“财力之雄厚”，“地方之握要”及“户口之繁殖”。并提出了“自立报馆”，“自立学堂”及“秘密社会”（利用[[会党|会党]]组织）三大促成独立的手段。欧榘甲同时认为广东的三大民系，[[广府|本地]]，[[福佬|福佬]]和[[客家|客家]]均为广东共同体中不可分离的一部分：
{{Quote box
 |title = 
 |quote = 
此三者种族，同出一源，不过因声音而异……三者种族，智识心思，脑输角度，形体精神，不相上下。即以其族谱而言，其祖先莫不由中原丧乱，越岭南迁。故本地之族，多由南雄而至广肇；客家之族，多由赣州而至；嘉惠福佬之族，多由江浙而至福潮。其声音之异，亦由所居之地而变迁焉……然则三者同为种族，无可疑也。
 |source = ——欧榘甲：《新广东》。<ref name="newcan" />

欧榘甲在文后亦进一步提出了内地十八省的解体方案，以及广东与广西两地建成联邦的想法：
{{Quote box
 |title = 
 |quote = 
广西之人本粤东流寓而生，疆土既相毘连，情义复相亲挚，于联邦最为易事，此地理人种，关于天然界者也。不独两广为然，既就今中国本部总督所辖之地，而分立为国土，亦人种地理天然界之相合者。如直隶总督所辖直隶、山东、山西、河南四省为一独立国，两江总督所辖江苏、安徽、江西三省为一独立国，两湖总督所辖湖北、湖南为一独立国，云贵总督所辖云南、贵州，陕甘总督所辖陕西、甘肃，闽浙总督所辖浙江、福建（昔兼台湾为三省，今割于日本），皆两省为一独立国，四川总督所辖四川一省为一独立国。置之欧罗巴强国中（除俄不计，除属地不计），疆域之广，未有能及之者也。即不然，而活因河流江流海流，分为北中南三大部分：阴山以南，黄河以北，诸省合为一独立国；黄河以南、扬子江以北，诸省合为一独立国；扬子江南岸、南洋北岸，诸省合为一独立国。三大干并立，固近世非常之雄国也。然而不论南部、中部、北部，亦不论诸省相合为一与否，而苟有独立之一省，起于其间，则南省必归于南部，中省必归于中部，北省必归于北部，可无疑也。即或因声音风俗政体之异，北省不归北部而归中部，中省不归中部而归南部，南省不归南部而归中部，而苟能自立，虽任其意之所向可也。
 |source = ——欧榘甲：《新广东》。<ref name="newcan" />

欧榘甲的文章掀起了广泛的讨论和关注。此书又令他被康有为严词斥责，有“雲樵離經畔道，應逐出門牆”之语。康又致书欧榘甲说：“得汝书，头痛不可言，汝等迫吾死地”，“惟有与汝等决绝，分告天下而已”。康有为为此接连撰写了竭力反对的数篇文章，斥责他的[[十八省|十八省]]各自分立的主张，论述[[印度|印度]]亡國便是由於各省自立。1902年，[[韩文举|韩文举]]以笔名“忧患余生生”在《[[新民丛报|新民丛报]]》的“捫虱談虎錄”中赞扬欧榘甲的独立理论，认为这是能使中国本部的华人达到自立的方法，并主张中国各地的人亦可以加入此种运动：
{{Quote box
 |title = 
 |quote = 
吾讀至是，乃大服。吾願[[楚|楚]]人謀新楚，[[蜀|蜀]]人謀新蜀，[[吳|吳]]人謀新吳，[[越|越]]人謀新越，[[甌|甌]]人謀新甌，乃至[[燕|燕]]、[[齊|齊]]、[[秦|秦]]、[[晉|晉]]、[[滇|滇]]、[[黔|黔]]。
 |source = ——韩文举：評論《新廣東》。<ref>{{cite journal |last=忧患余生生|date=1902-8-18 |title=捫虱談虎錄：评论《新广东》|url= |journal=新民叢報 |volume=14 |issue= |pages=55-56 |doi= }}</ref>
|align = center|width = 100%|border = 3px|fontsize = 20|bgcolor =  lightgray|title_bg = gray |quoted =5px|qalign = left|salign = right}}{{clear}}

1903年，在《新广东》问世后一年，留学日本的湖南反清人士[[杨守仁|杨毓麟]]（长沙人）仿照《广东人之广东》的标题，以笔名“湖南人之湖南”发表《[[新湖南|新湖南]]》一文。杨毓麟的立论深受欧榘甲影响，亦主张[[中國本部|中國本部]]解体。文章开篇即称：“太平洋客著《新广东》，三户之愤民，读而韪之”，并宣称“广东倡之，吾湖南和之；广东鼓之，吾湖南舞之。吾于广东，如骖之靳也”<ref name="newxiang">{{Cite book |title=《杨毓麟集》 |publisher=岳麓书社 |year=2008年 |isbn=9787807611097| page=26|location=长沙 |language=zh-cn}}</ref>。楊氏甚至将清帝国的解体比喻为罗马帝国崩溃、将[[湖南独立|湖南独立]]比作欧洲民族国家建构。<ref>{{Cite book |title=《蹈海志士：杨毓麟传》 |publisher=岳麓书社 |year=2011年 |isbn=9787807617594 |location=长沙 |language=zh-cn}}</ref>

1905年5月8日，清廷军机处致函各省督抚严行查禁《新广东》、《新湖南》及《新民丛报》等宣扬变法或革命的“悖逆”书刊<ref name="newcanptol">{{cite journal |last=夏晓虹|date= |title=《新广东》：从政治到文学 |url= |journal=学术月刊 |volume= |issue= |pages=108-126 |doi= |access-date=2017-08-29 }}</ref>。冯自由在其回忆录《革命逸事》中也提及，《新湖南》“鼓吹湘省脱离满清独立之说甚力，与粤人欧榘甲著之《新广东》同风行于世”<ref name="fungbook"/>。值得注意的是，[[陈炯明|陈炯明]]后来提出“[[联省自治|联省自治]]”的主张，很大程度受到了欧的启示<ref name="guangzuzhongxue">{{cite book |title=《家国之光：欧榘甲与光祖中学》|author=罗更晓，刘中国，余俊杰|pages=4|language = zh-cn|publisher=花城出版社}}</ref>。

1908年，已与康有为决裂的欧榘甲与刘士骥等人希望在广西等地投资实业。欧到纽约为振华公司招股时，亦曾对人说:“吾等今日之做振华股, 不过欲他日图两广之地步耳。吾蓄谋已十有余年, 振华、广美两公司若成, 吾之目的可达也。”1909年，康有为一派与欧榘甲等人的矛盾白热化，包括刘士骥在内的多人被暗杀。欧不得不“出入率以死士相随”，战战兢兢度日。然而康有为抢先一步，举报欧借商谋乱，图为两广独立，密谋革命<ref>{{cite book |title=《庚子勤王与晚清政局》|author=桑兵 |pages=381 |publisher=北京大学出版社|language = zh-cn}}</ref>。康有为在告密信中指欧榘甲“伪托于保党中, 欲以暗移人心, 既以反背党旨, 为党魁所大责不容, 则又巧变面目”，“欲谋乱, 东西粤、云南三省遍布心腹, 运购军伙, 非藉招商股筹数十万不能措办”，“反心盜據”，“藉巡抚之势力, 以招商劫商, 因广西之荒僻而谋乱, 欲据两粤滇黔而自立”<ref>{{cite book |title=《康有为与保皇会》|author=上海市文物保管委员会 |pages=337|language = zh-cn}}</ref>。康有为因此擬請大理院勒令交還股款并请求朝廷“立拿乱首欧榘甲”。欧榘甲不得不走向隐匿生活，最后去世。

=== 辛亥广东脱清独立 ===
1911年湖北武昌起义后，广东各界“策划广东自保”。11月9日，[[广东咨议局|广东咨议局]]宣布广东脱清独立。广东的士绅阶层在晚清的咨议局改革中曾初尝现代政治参与的滋味，对于他们来说，辛亥革命是一个让他们主导实践广东独立自主的机遇。<ref name="TsuiShingYan" >{{cite book |title=《香港，鬱躁的家邦：本土觀點的香港源流史》|author=[[徐承恩|徐承恩]]|year=2017年 |publisher=左岸文化|pages=237 |language = zh-cn}}</ref>广东独立后担任广东民团总长的刘永福就在安民公告中如是说：
{{Quote box

夫吾粤，东接闽，西连桂，北枕五岭，南滨大洋，风俗语言嗜好与中原异，固天然独立国也。秦之[[赵佗|赵佗]]，隋之[[冯盎|冯盎]]、[[邓文进|邓文进]]，元之[[何真|何真]]，皆乘变乱时代，崛起一方，安辑人民，巩卫疆圉。今兵力强厚，独立之局告成矣，所以谋善后者，何止万端！……广东省，广东人之广东，斯言闻之熟矣。

=== 聯省自治思潮 ===
中華民國初年，受聯省自治思潮影響，政治家与地方实力军人提出聯邦制的中國政治制度改革方案，主張各省自治立憲。當時主政廣東的陳炯明亦贊同此種政治制度，並推動當時的廣東議會起草了《廣東省憲法草案》，規定了廣東的軍事與財政獨立自治制度以及與中華民國邦聯之間的關係。而在湖南，包括毛澤東等人則發起了湖南獨立運動，反對“大中華民國”，主張“湖南共和國”以及中国各省獨立建國，得到中國內外包括[[胡適|胡適]]和[[杜威|杜威]]等人士支持<ref>{{Cite web |url= https://thestandnews.com/politics/練乙錚-毛澤東曾提倡湖南獨立建國-港獨非大逆不道/ |title= 練乙錚：毛澤東曾提倡湖南獨立建國 港獨非大逆不道 |publisher= 立場新聞 |date= 2015-05-07 }}</ref>：
{{Quote box

==流行文化==
1902-04年，欧榘甲在《[[新小说|新小说]]》上以“新广东武生”为笔名发表了改良粤剧剧本《[[黄萧养回头|黄萧养回头]]》（全名《新串班本黄萧养回头全套》）。剧中旗帜鲜明地宣扬了《新广东》中的广东独立思想，并将欧氏在《新广东》中提出的独立策略加以戏剧化演绎。在欧榘甲设置的剧情中，主人公黄种强是明代广东农民起义领袖[[黄萧养|黄萧养]]转世后的身份。剧名中所谓“回头”即再世的意思。欧榘甲利用了广东民间“九牛浮水面，萧养转回头”“大石沉底，白鹅浮游，三十年后，萧养回头”的民谣达到宣传的目的。本剧的剧情便是黄种强率领诸人带领广东独立的故事，而黄种强也主张“将来广东独立，全靠斯人。”欧榘甲甚至还在剧中呼应了当时的[[菲律宾独立战争|菲律宾独立战争]]，在戏本中，[[阿奎纳多|阿奎纳多]]最终决定了“我非律宾保中国广东独立自治”的国策。<ref name="wongsiuyeung"/><ref name="newcanptol"/>

日本畅销军事题材小说家{{Link-ja|森詠|森詠}}有两部作品中均提及了广东独立的情节。在其1995-2003年出版的军事小说《新日本中国战争》中，森詠虚构了一个[[邓小平|邓小平]]逝世后中国发生内部权力斗争导致地区冲突的情节。因为在对[[李登辉|李登辉]]领导下的[[台湾独立|台湾独立]]的态度分歧，广东当局宣布了独立。中国内战由此爆发<ref>{{Cite book |author=森詠| title=《新・日本中国戦争〈第4部〉中国分裂》 |publisher=学習研究社 |year=1996年 |isbn=9784054006805|language=ja}}</ref>。而在森詠后来写的以虚构的2020年代为背景的军事小说《新编日本中国战争—怒涛的世纪》中，广东在中国与日本美国的军事冲突后期也宣布了独立<ref>{{Cite book |author=森詠| title=《新編 日本中国戦争 怒濤の世紀 第十一部 中国崩壊》|publisher=文芸社  |year=2017年 |isbn=9784286183930|language=ja}}</ref>。在两部小说中的最后，“广东共和国”的军队均加入了美国、日本与台湾对中国作战的一方。

在中国异议作家[[王力雄|王力雄]]的小说作品《[[黄祸_(小说)|黄祸]]》中，因为[[中华人民共和国|中华人民共和国]]内部权力斗争的缘故，包括广东在内的东南沿海诸省为维护自身利益﹐宣告实行“一国多制”脱离北京当局。此举最终引发了最后扩散为全球性战争的中国南北战争<ref>{{Cite book |author=王力雄| title=《黄祸》|publisher=Locus Publishing Company|year=2016年|language=zh-cn}}</ref>。

在以中华民国为背景的回合制策略游戏[[民国无双|民国无双]]里，玩家可以选择包括[[粤系|粤系]]、[[中華民國治藏歷史|西藏]]、[[满洲国|满洲国]]及[[东突厥斯坦第二共和国|东突厥斯坦第二共和国]]等在内的中国近代各势力。游戏以1925年、1935年、1946年的东亚为背景，玩家可选择任何势力进入游戏。在1925年的剧本中，[[国民政府|国民政府]]很快会像历史上那样，与粤东[[陈炯明|陈炯明]]的粤系爆发战争。游戏的设计者允许选择粤系的玩家进行改变历史的操作，例如统一东亚、统一中国、联省自治、达致金融垄断，或是建立独立的“广东共和国”。若玩家选择了建国为胜利目标，最终“广东共和国”将会在战后加入联合国。该游戏发布一个月，中华人民共和国文化部便以“严重影响国家文化安全”为由查禁<ref>{{Cite web |url =https://sites.google.com/site/kowloonia/faction/yue-jun |title = 粵系-民国无双官方网站 |publisher = ChengLap }}</ref>。

2015年，中國國家發改委將廣東省的廣州、[[惠州|惠州]]、[[東莞|東莞]]及深圳[[光明新區|光明新區]]列入試點，改善外來人口的福利，估計成本达1490億元人民幣。 此举激發起网络上的骂战。有廣州網民在微博上宣示不滿，認為此計劃沒有考慮過原住民感受，並批評外省人沒有決心融入，不學習廣州的文化，指“無義務將廣州的資源分给外省”，更有人高呼“廣州打響獨立戰爭！一人一票決定我地的未來！” 而外省网民则指廣州人不包容，認為應該允許全國人自由遷徙，更表示“沒有外來人口，廣州經濟將癱瘓”<ref>{{Cite web |url = http://www.passiontimes.hk/article/02-12-2015/21160 |title = 廣州將改善外來人福利　當地網民不滿高呼要獨立 |date = 2015-02-12 |publisher = 热血时报 |accessdate = 2015-2-12 }}</ref>。

2018年3月12日，媒体报道廣州多處公共設施被人寫上繁體字“廣州獨立 香港加油”，“广东独立”，“Free Cantonia”的標句，遍布交通工具及行人路之上，包括出現於[[大元帥府|大元帥府]]巴士站、新鴻花園巴士站、以及多個巴士坐位的椅背上。部分反对香港独立的亲中媒体，例如香港[[东网|东网]]，认为这是香港的“獨立歪風”蔓延到內地的表现，亦认为在公共地方胡亂塗鴉有違公德<ref>{{Cite web |url = http://hk.on.cc/hk/bkn/cnt/news/20180312/bkn-20180312111957470-0312_00822_001.html |title = 獨立歪風蔓延　廣州多處現「廣獨」字句 |date = 2018-03-12 |publisher = 香港东网 |accessdate = 2018-03-12 |archive-url = https://web.archive.org/web/20180312144940/http://hk.on.cc/hk/bkn/cnt/news/20180312/bkn-20180312111957470-0312_00822_001.html |archive-date = 2018-03-12 |dead-url = yes }}</ref>。有維權人士接受外媒採訪時表示，這是由於當地民眾不堪忍受中共打壓粵文化，對民眾的訴求置若罔聞而採取的反抗行動。廣州維權人士廖劍豪则在接受自由亞洲電台採訪時表示曾在廣州市地鐵[[農講所站|農講所站]]附近見过一張A4紙寫着“廣東獨立”。廖认为这种现象的出现是一種社會進步。不過，也有广州维权人士[[区伯|区伯]]表示，尽管希望推動民主進步，但亦反對分裂國家，认为一個國家的人民应该團結一致<ref>{{Cite web|url=https://www.rfa.org/cantonese/news/independence-03122018090537.html|title=广州出现独立标语　并称与香港同行|date = 2018-03-12 |publisher = 自由亚洲电台}}</ref>。澳门理工学院的政治学家Arnado Gonçalves在接受法广访问时亦表示，尽管广东人有其独特的身份认同，但也质疑此种运动的自洽性<ref>{{Cite web |url =http://pt.rfi.fr/mundo/20180313-china-independentistas-em-cantao |title = China: independentistas em Cantão ? |date = 2018-03-13 |publisher = RFI |language=pt}}</ref>。

2019年1月1日，在[[香港元旦大游行|香港元旦大游行]]中，支持香港独立的[[香港民族陣綫|香港民族陣綫]]游行队伍中亦出现了支持广东独立的木棉花旗帜，队伍中也包括[[西藏独立|西藏独立]]、[[台湾独立|台湾独立]]、[[东突厥斯坦独立运动|东突厥斯坦独立]]、[[南蒙古独立运动|南蒙古独立]]、[[满洲独立|满洲独立]]、[[闽越|闽越]]独立、[[巴蜀|巴蜀]]独立和[[上海|上海]]独立等旗帜<ref>{{Cite web |url = https://thestandnews.com/politics/%E8%BF%91%E7%99%BE%E4%BA%BA%E5%8F%83%E8%88%87%E6%B8%AF%E7%8D%A8%E9%81%8A%E8%A1%8C-%E8%A6%81%E6%B1%82%E8%A7%A3%E6%94%BE%E8%A5%BF%E8%97%8F-%E9%87%8B%E6%94%BE%E8%A2%AB%E9%97%9C%E6%8A%BC%E6%96%B0%E7%96%86%E7%B6%AD%E6%97%8F%E4%BA%BA/ |title = 數十人參與港獨遊行　要求解放西藏　釋放被關押新疆維族人 |date = 2019-01-01 |publisher = 立場新聞}}</ref>。除港独口号外，示威者亦高叫「廣東、廣西、福建獨立」等口號<ref>{{Cite web |url = https://hk.on.cc/hk/bkn/cnt/news/20190101/bkn-20190101150221562-0101_00822_001.html |title = 元旦遊行又現港獨旗　梁頌恆憂被捕拒入政總 |date = 2019-01-01 |publisher = 香港东网}}</ref>。

== 假想情況 ==
1979年4月，中共中央工作会议上，时任[[广东省委|广东省委]]第一书记的[[习仲勋|习仲勋]]提出希望中央放权与广东省。主持会议的[[华国锋|华国锋]]疑惑习仲勋要什么权力。习仲勋当场说：“如果广东是一个‘独立的国家’，可能几年就搞上去了，但是在现在的体制下，就不容易上去。” 这番发言在会议上引起大反响<ref>{{Cite web |url = http://history.m4.cn/2013-01/1200356.shtml |title = 习仲勋：如果广东是个“独立国家”早就搞上去了  |date = 2013-01-31 |publisher = 四月网 |accessdate = 2017-08-31 }}</ref>。

2008年9月11日，《[[南方都市报|南方都市报]]》发表以《假如广东是一个独立经济体》为题的整版报道，文章说：“让我们想象一下，假如广东是一个国家……我们将试从局外人角度，以一种新的方式来观察中国”，并计算出若将中国各省视为独立国家，“‘广东共和国’将成为世界第14大经济体。”<ref>{{Cite web |url = http://finance.southcn.com/content/2008-09/11/content_4593860.htm |title = 假如广东是独立经济体 经济总量位列世界第14 |publisher = 南方网 |accessdate = 2008-09-11 }}</ref>
