'''广水市'''位于[[中华人民共和国|中国]][[湖北省|湖北省]]东北部，是[[随州市|随州市]]代管的一个[[县级市|县级市]]。有鄂北门户和“湖北书法之乡”之称。原为应山县。广水市标为火凤。

== 历史沿革 ==
[[南朝宋|南朝宋]]析随县置'''[[永阳县|永阳县]]'''，为建县之始，隶[[司州|司州]][[随阳郡|随阳郡]]。[[南梁|南梁]][[大同_(南梁)|大同]]二年（536年）置[[应州_(南梁)|应州]], 又置[[平靖郡|平靖郡]]领永阳县。

[[隋朝|隋朝]][[开皇|开皇]]三年（583年）省平靖郡，由应州领[[平靖县|平靖县]]和永阳县。开皇十八年（598年），改永阳县为'''[[應山縣|应山县]]'''，这是“应山”一名的最早来历。该县得名应为市中心有一山名为应山，后讹传为印山，即今日市区中心印台山。
[[大业_(年号)|大业]]二年（606年）废应州，将平靖县并入应山县，隶属[[安陆郡|安陆郡]]。

[[唐朝|唐朝]][[武德|武德]]四年（621年）复置应州，领应山县、[[礼山县_(隋朝)|礼山县]]；武德八年（625年）又废应州，将礼山县并入应山县，属[[淮南道|淮南道]][[安州_(西魏)|安州]]。

[[北宋|北宋]]属[[荆湖北路|荆湖北路]][[德安府|德安府]]，[[开宝|开宝]]年间废吉阳县，吉阳山以北并入应山县。[[元朝|元朝]]隶[[随州_(古代)|随州]]，并属荆湖北路德安府。[[明朝|明]][[清朝|清]]两朝延其制，惟[[雍正|雍正]]七年后直属[[德安府|德安府]]，不再隶属随州。

民国时隶属变动频繁，先后直属于[[湖北省_(中華民國)|湖北省]]、属[[江汉道|江汉道]]、复直属湖北省、属湖北省第五行政督察区、属第三行政督察区。

1949年3月30日，中共政权占领应山县，此后属[[孝感专区|孝感专区]]。1959年改属[[武汉市|武汉市]]。1961年复属孝感专区。1988年12月撤销应山县，设广水市，以下辖广水镇（现[[广水街道|广水街道]]，位于[[京广铁路|京广铁路]]上）命名，但县治不变。2000年7月15日，广水市改由随州市代管。<ref>{{cite web |url=http://www.zggs.gov.cn/2005-07/29/cms241345article.shtml |title=存档副本 |accessdate=2009-09-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20100515025735/http://www.zggs.gov.cn/2005-07/29/cms241345article.shtml |archivedate=2010-05-15 }}</ref>

== 地理 ==
广水市位于湖北省北部偏东，北与[[河南省|河南省]][[信阳市|信阳市]]毗邻，东邻[[大悟县|大悟县]]，南接[[孝昌县|孝昌县]]、[[安陆市|安陆市]]，西连[[随州市|随州市]][[曾都区|曾都区]]。东西宽57.5公里，南北长78.7公里，面积2647平方公里。

广水地处[[桐柏山|桐柏山]]脉南麓、[[大别山|大别山]]脉西端, 属低山丘陵地带。最高处[[大贵山|大贵山]]海拨907.8米，最低处平林的河床海拔仅37米。<ref>http://www.zggs.gov.cn/2005-07/29/cms241348article.shtml{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 区划人口 ==
现辖3个街道办事处、10镇、4乡以及中华山、大贵寺、花山3个国营林场、三潭风景区和广水经济开发区。

2000年, 全市年末总人口为879216人。<ref>{{cite web |url=http://www.zggs.gov.cn/2005-07/29/cms241344article.shtml |title=存档副本 |accessdate=2009-09-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20100515025400/http://www.zggs.gov.cn/2005-07/29/cms241344article.shtml |archivedate=2010-05-15 }}</ref>

该区域主要语言是汉语方言西南官话。
该区域在太平天国叛乱后有部分人口移居至安徽，主要分布于宁国市大部及周边郎溪县、广德县、宣城市、泾县。
宁国市大部及周边郎溪县、广德县、宣城市、泾县方言和应山方言基本一致。

== 交通 ==
*[[京广铁路|京广铁路]]经市境东部武胜关、广办、杨寨，设[[广水站|广水站]]
*[[汉丹铁路|汉丹铁路]]经市境西部长岭、马坪两镇，1980年代设马坪站、长岭站、平林站，后汉丹线提速，撤销三站。
*[[107国道|107国道]]经市境东部武胜关、广办、杨寨，到[[武汉|武汉]][[天河国际机场|天河国际机场]]车程1小时
*[[316国道|316国道]]经本市西部长岭镇、马坪镇穿镇而过；长岭镇距武汉150公里、距襄阳180公里。
*S28高速（麻竹）贯穿市境东西，东与G4（京珠高速）相接、西与G70（汉十高速）相接，横跨107、316两条国道，有杨寨、广水（市府所在地应山）、关庙三个出入口。

== 经济 ==
2008年国民生产总值106.42亿元，比上年增长14.3%。

现有中型企业6家，小型企业154家。以食品加工、冶金铸造、建材、医药化工、造纸包装、通用设备制造、纺织服装为支柱产业。<ref>{{cite web |url=http://www.zggs.gov.cn/2009-04/13/cms750817article.shtml |title=存档副本 |accessdate=2009-09-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20100515025412/http://www.zggs.gov.cn/2009-04/13/cms750817article.shtml |archivedate=2010-05-15 }}</ref>
