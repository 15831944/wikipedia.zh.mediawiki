'''广东警官学院'''（{{lang-en|'''Guangdong Police College'''}}）前身為'''广东省公安干部学校'''，于1949年11月成立，經歷過多次的改革至今成為[[學院|學院]]，位於[[中國|中國]][[廣東省|廣東省]]，設有兩座校區，分區位於[[滨江|滨江]]和[[嘉禾|嘉禾]]，占地面积共37万[[平方米|平方米]]<ref>[http://gaoxiao.xuanxiao.com/daxue/1000.html 广东警官学院的基本信息] {{webarchive|url=https://web.archive.org/web/20121213065613/http://gaoxiao.xuanxiao.com/daxue/1000.html |date=2012-12-13 }}</ref>。廣東警官學院有約6,600名[[全日制|全日制]][[學生|學生]]，其中[[本科|本科]]學生有約5,600名，[[专科|专科]]學生約1,000人，体改班学员約1,000可及各类成人本科及专科學生約1,600人。自2004年升格成為广东警官学院以後，有逾10,000名學生在此[[畢業|畢業]]，

== 歷史 ==
1958年，广东省公安干部学校与[[廣東省政法干校|廣東省政法干校]]、[[廣州市公安学校|廣州市公安学校]]、[[廣州市行政干部学校|廣州市行政干部学校]]和[[廣東省民警干训大队|廣東省民警干训大队]]合并成立'''广东省政法干部学校'''。1964年，廣東省政法干部學校停止運作。

1972年，根据廣東省委指示，廣東省政法干部學校恢復辦理教學。

1983年，廣東省政法干部學校改變組織成为'''广东省公安司法管理干部学院'''，开展成人[[专科|专科]][[学历|学历]][[教育|教育]]。

1992年，在广东省公安司法管理干部学院的基础上，增加设立'''广东公安高等专科学校'''；1998年，为了深化[[高校|高校]]体制改革，[[資源增值|資源增值]]，经過[[廣東省政府|廣東省政府]]批准，全[[中国公安|中国公安]]系统中唯一的一所国家级重点中专学校'''广东省人民警察学校'''合并入广东公安高等专科学校內。2004年5月，经過[[教育部|教育部]]的批准，在广东公安高等专科学校的基础上建立了广东警官学院，是廣東省属於[[本科|本科]]层次的普通高等公安院校，也是[[华南地区|华南地区]]第一所公安本科院校<ref>[http://kaoshi.edu.sina.com.cn/college/c/11110.shtml 广东警官学院]</ref>。

==組織==
广东警官学院有教职員約500名。

== 設施 ==
广东警官学院特別重視科技研究事務，設立有多種[[實驗室|實驗室]]及模擬實驗室，包括[[脫氧核糖核酸|脫氧核糖核酸]]、[[法證|法證]][[攝影|攝影]]、痕迹检验、安全防范、[[法医学|法医学]]、[[罪案現場|罪案現場]]勘查、[[消防|消防]]、模拟[[审讯|审讯]]等，共53座。其中[[刑事科学技术实验教学中心|刑事科学技术实验教学中心]]被评級为广东省省级实验教学示范中心，并且设立有广东天正司法鉴定中心。广东警官学院圖書館的馆藏图书（包含[[电子图书|电子图书]]）達750,000冊。此外，广东警官学院設立有[[标准|标准]]的[[运动场|运动场]]、[[警察|警察]][[战训|战训]]大楼、[[射击场|射击场]]及[[游泳馆|游泳馆]]等训练场馆及设施等。

== 系部 ==
广东警官学院至2019年共有10个教学系部：
*法律系
*侦查系
*治安系
*刑事技术系
*警务指挥战术系
*网络信息安全系
*公共管理系
*思想政治理论课教学部
*基础课教研部
*警体研修训练部
