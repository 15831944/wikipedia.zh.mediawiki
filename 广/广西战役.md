'''广西战役'''，是[[第二次国共内战|第二次国共内战]]末期，[[中国人民解放军第四野战军|中国人民解放军第四野战军]]主力和[[中国人民解放军第二野战军|中国人民解放军第二野战军]]第4兵团，与[[中華民國國軍|国军]][[白崇禧|白崇禧]]集团的[[战役|战役]]。<ref name="军事I，第308页">中国大百科全书总编辑委员会《军事》编辑委员会，《中国大百科全书•军事I》，北京，中国大百科全书出版社，1989年5月上海印刷，第308页，ISBN 7500002424</ref>

== 背景 ==
1949年10月，[[衡宝战役|衡宝战役]]后，[[华中军政长官公署|华中军政长官公署]]军政长官白崇禧部5个[[兵团|兵团]]12个[[军|军]]共15万余人，由[[湘南|湘南]]退守[[广西|广西]]，企图在退至粤桂边境及湛江地区的[[广州|广州]][[绥靖公署|绥靖公署]]主任[[余汉谋|余汉谋]]部约4万人策应下，以[[桂林|桂林]]为中心，沿全县至[[柳州|柳州]]铁路两侧组织防御，阻止解放军入桂。第四野战军司令员[[林彪|林彪]]指挥9个军共40余万人，遵照中共中央军委主席[[毛泽东|毛泽东]]关于大迂回大包围作战方针，决心分3路向广西突进，求歼白崇禧集团于广西境内。<ref name="军事I，第308页"/>

== 经過 ==
10月31日，解放军发起战役。11月6日，解放军西路第13兵团第38、第39军由湘西南出发，11月10日分别攻占[[靖县|靖县]]、[[通道县|通道]]两城。第十七兵团仓皇西逃，第13兵团乘胜追击，于11月15日经[[黔南|黔南]]楔入广西，向[[河池|河池]]、[[百色|百色]]疾进，切断白崇禧西逃云南、贵州之路。南路第4兵团第13、第14、第15军和第15兵团第43军，分别于11月10日和11月15日向[[廉江县|廉江]]、[[茂名县|茂名]]、[[信宜县|信宜]]推进，切断白崇禧海上逃路。

北路第12兵团第40、第41、第45军进至湘桂边境，一部于11月10日袭占[[全县|全县]]县城。这时，第二野战军已突破白崇禧川黔防线，一部已逼近[[贵阳|贵阳]]、[[遵义|遵义]]。白崇禧在处境孤立，西逃无望情况下，急调第三兵团、第十一兵团共5个军由[[荔浦县|荔浦]]、[[恭城县|恭城]]南下[[玉林|玉林]]、[[岑溪|岑溪]]地区，企图在余汉谋残部配合下，发动“南线攻势”，以打通撤往[[海南岛|海南岛]]道路，并令第一兵团、第十兵团由[[桂北|桂北]]南撤。第四野战军前指决心趁机在粤桂边境围歼其第三兵团、第十一兵团，令南路部队以一部兵力在廉江至信宜一线占领有利地形阻击敌人，主力隐蔽待机；令西、北两路部队迅速南进，策应南路作战。11月24日，白崇禧第三兵团、第十一兵团开始向廉江、茂名、信宜等地实施猛烈攻击，受挫。11月27日，解放军南路主力从廉江、茂名、信宜突然出击，并乘胜猛追，攻占[[容县|容县]]、玉林、[[陆川|陆川]]等地，歼灭第十一兵团一部。

11月30日，第43军一举突入[[博白县|博白]]县城，歼灭白崇禧第三兵团部，俘华中军政公署军攻副长官兼第三兵团司令官[[张淦|张淦]]。随后，会同第4兵团主力将第三兵团所辖3个军大部围歼于博白、陆川地区。此时，余汉谋残部乘隙袭占廉江城，中国人民解放军第13军立即回师歼其一部。解放军西、北两路亦急速南下，在桂滇黔边纵队配合下，先后攻占[[河池|河池]]、桂林、柳州、[[梧州|梧州]]等，继又进至东兰、宾阳、贵县一线。白崇禧“南线攻势”失败后，仓皇撤退。令第一兵团、第十兵团分别撤到[[南宁|南宁]]、[[钦州|钦州]]地区，第十一兵团残部退向钦州，企图从海上逃跑。为全歼白崇禧残部于国境内，12月3日起，解放军各路部队继续猛追，先后占领[[百色|百色]]、南宁、[[灵山|灵山]]、[[合浦|合浦]]等地。12月7日，将华中军政长官公署和第一、第十兵团及其他残部大部围歼于钦州和[[小董圩|小董圩]]地区。至12月14日，先后占领[[友谊关|镇南关]]和爱店，战役结束。<ref name="军事I，第308页"/>

== 结果 ==
此役，国军被歼约17.3万人，2万余人窜逃国外。<ref name="军事I，第308页"/>白崇禧個人撤至海南岛，国军万余人撤入[[越南|越南]]，余部均被歼灭。解放军占领广西全省和广东省钦州（今属[[广西壮族自治区|广西壮族自治区]]）、[[雷州半岛|雷州半岛]]等地区，为尔后[[云南省|云南省]]和[[海南岛战役|海南岛战役]]创造埋下伏笔。<ref>[http://news.xinmin.cn/rollnews/2009/09/25/2630026.html 西野征程：广西战役]</ref>解放军缴获各种[[炮|炮]]1250余门，轻重机枪4300余挺，各种[[枪|枪]]4.8万余支、[[汽车|汽车]]1,170余辆，各种[[炮弹|炮弹]]5.6万余发，[[子弹|子弹]]580万发<ref name="军事科学">《中国人民解放军全史》军事历史研究部 编，[[军事科学出版社|军事科学出版社]]，2000年，ISBN 7-80137-315-4，卷“中国人民解放军战役战斗总览”。</ref>。
