'''广岛都市学园大学'''（{{lang-en|'''Hiroshima Cosmopolitan University'''}}）位于[[广岛县|广岛县]][[广岛市|广岛市]]南区宇品西5丁目13-18，为日本[[私立大学|私立大学]]，1974年建校，2009年设置。

== 沿革 ==
1974年，“广岛都市学园大学”的前身建立。

2009年，广岛都市学园大学正式设置招生。

== 校址 ==
广岛县[[广岛市|广岛市]][[南区_(广岛市)|南区]]，广岛市内私立大学唯一看护师养成四年制大学。

== 院系 ==
