'''广安云轨示范线'''又称'''广安云轨'''或'''广安轨道交通1号线'''，是[[中华人民共和国|中华人民共和国]][[四川省|四川省]][[广安市|广安市]]与[[比亚迪|比亚迪]]合作建设的[[單軌鐵路|跨座式单轨]]线路，于2017年1月提出建设，預計在2019年7月开通。线路全长9.955千米，设置了8个站点，全部为高架站。开通后，该线路将作为比亚迪的技术示范线和广安市旅游客运及市区交通线路广安轨道交通1号线运用。同时，广安市也依托该示范线，规划了自身的远期城市轨道交通系统。

== 规划与建设 ==
* 2017年1月12日，广安市与[[比亚迪|比亚迪]]签署协议，准备在广安市建设跨座式单轨技术示范线，并在广安设立“云轨”生产基地。<ref>{{Cite web|url=http://www.gatv.com.cn/21/53773_1.shtml|title=广安“云轨”线路初步确定|accessdate=2017-08-01|author=吴一龙 陈志宏 李硕|date=2017-01-21|publisher=广安新闻网}}</ref>
* 2017年1月22日，广安初步确定了其示范线为广安南站-思源广场-协兴园区游客中心走向。<ref>{{Cite web|url=http://www.rail-transit.com/Detail_News.aspx?id=39983|title=『广安』云轨预计2017年8月开通|accessdate=2017-08-01|author=|date=2017-01-25|publisher=中国轨道交通网}}</ref>
* 2017年2月26日，广安云轨交通示范线开工。<ref>{{Cite web|url=http://www.gakfq.gov.cn/news/qy/2017-03-01/1173.html|title=广安云轨交通正式开工|accessdate=2017-08-01|author=李平|date=2017-03-01|publisher=广安经济技术开发区}}</ref>
* 2017年3月9日，广安市规划局对广安邓小平故里景区旅游连接线项目路线7个站点进行了公示。<ref name=":0">{{Cite web|url=http://www.gasghj.gov.cn/guihuashenpi/ghdt/2017-03-09/409.html|title=广安邓小平故里景区旅游连接线项目选址公示|accessdate=2017-08-01|author=|date=2017-03-09|publisher=广安市规划局}}</ref>
*2017年4月25日，线路开始市民广场-游客中心段的桩基施工。
* 2017年4月底，比亚迪广安轨道梁厂开始试生产。<ref>{{Cite web|url=http://news.xinhuanet.com/local/2017-04/27/c_129576173.htm|title=比亚迪首条“云轨”旅游示范线即将投运|accessdate=2017-08-01|author=黄辉 康建林|date=2017-04-27|publisher=四川日报}}</ref>
*2017年12月31日，广安比亚迪云轨车辆厂生产的第一列单轨列车下线。
*2018年1月17日，首列三节编组列车被吊装上轨道。<ref>{{Cite news|url=http://news.ifeng.com/a/20180118/55248367_0.shtml#p=1|title=四川广安：云轨列车车厢吊装上线即将运行|author=邱海鹰|date=2018-01-18|work=凤凰网|accessdate=}}</ref>
*2018年7月18日，广安轨道交通开始2018年社会公开招聘。

== 线路 ==
广安云轨示范线全长约9.955千米，设有8个站点，自广安[[广门乡|广门乡]]出发，向东再向北呈“L”型途径[[广安南站|广安南站]]、广安中心城区抵达[[邓小平故居|邓小平故里]]游客中心。全线为高架线路，在广门乡设有广门车辆段。<ref>{{Cite web|url=http://www.gatig.com.cn/gongsigaikuang/qixiagongsi/2018-06-07/1025.html|title=广安市云轨交通有限公司|accessdate=|author=|date=|publisher=}}</ref><ref name=":0" />

=== 换乘 ===
在“广安南站”站可通过国铁[[广安南站|广安南站]]换乘国铁[[兰渝铁路|兰渝铁路]]列车。未来，在轨道交通1号线北延伸段影视城站将可以与轨道交通2号线实现换乘。

=== 远期规划 ===
该线路远期规划有北延伸段，另规划有轨道交通2号线。它们将和以现有示范线组成共同组成广安市城市轨道交通网络。

* 轨道交通1号线北延伸段全长9.3公里，设站6座，为全高架线路。该线将在南端与现有游客中心站相衔接。
* 轨道交通2号线一期全长7.7公里，设站6座，为全高架线。该线将向东南跨越渠江，经奎阁新区达广安港。该线路站点为：''影视城站-新区北站-[[奎阁|奎阁]]站-广前大道站-港前大道站-[[广安港|广安港]]站''。

== 列车 ==
“[[云轨|云轨]]”列车实质上为跨座式单轨列车，最高时速80km/h，比亚迪声称拥有其知识产权。与传统车辆相比，它减轻了轨道梁和车辆的重量，自身带有储电系统且刹车时可回收电力。其列车两端呈流线型，外形与中国铁路[[和谐号CRH3型电力动车组|和谐号CRH3型电力动车组]]有相似之处。<ref>{{Cite web|url=http://money.163.com/16/1223/14/C8VQ0RH9002580S6.html|title=比亚迪为何豪赌云轨|accessdate=2017-08-01|author=张瑜 单素敏|date=2016-12-23|publisher=瞭望东方周刊}}</ref>

== 争议 ==
许多学者对于“云轨”系统使用场景受限和载客量、兼容性以及安全问题提出了质疑。<ref>{{Cite web|url=http://www.umschina.com/view/282.html|title=“云轨”是公交升级、缓解拥堵的良药吗？|accessdate=2017-08-01|author=彭佳 俞梦晓 杨明 杨涛|date=2017-03-14|publisher=南京城交院}}</ref>
