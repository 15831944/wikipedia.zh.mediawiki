'''广西民族博物馆'''，位于[[广西壮族自治区|广西壮族自治区]][[南宁市|南宁市]]青秀山文化风景区，是广西壮族自治区级专题性民族文化博物馆。

==简介==
2001年，[[广西壮族自治区人民政府|广西壮族自治区人民政府]]决定建设广西民族博物馆。2001年10月，[[广西壮族自治区文化厅|广西壮族自治区文化厅]]成立了广西民族博物馆项目领导小组，并且设广西民族博物馆建设办公室。2003年1月，广西壮族自治区人民政府主席[[李兆焯|李兆焯]]在广西壮族自治区人大一次会议上作政府工作报告，将广西民族博物馆列为广西壮族自治区当年新建开工的重大项目之一。2003年3月，广西壮族自治区人民政府第三次主席办公会要求把广西民族博物馆建成一流的省级民族博物馆。2003年3月，广西民族博物馆馆址定在南宁市青秀山风景区内，占地面积130亩，投资2.5亿元人民币，是[[中华人民共和国|中华人民共和国]]成立以来广西最大的文化投资项目。2003年4月，广西壮族自治区文化厅决定组建广西民族博物馆建设办公室。2003年8月，[[广西壮族自治区机构编制委员会|广西壮族自治区机构编制委员会]]同意成立广西民族博物馆筹建办公室。2003年9月，广西壮族自治区文化厅任命广西壮族自治区文化厅文物处处长覃溥兼任广西民族博物馆馆长<ref name=dshji>{{Cite web |url = http://www.amgx.org/news-2084.html |title = 广西民族博物馆大事记 |author =  |date = 2017-02-10 |publisher = 广西民族博物馆 |language =  |accessdate =  |quote =  }}</ref>。

2003年9月，由广西民族博物馆承办的全区民族文物调查征集培训班在南宁市举行。2003年12月，广西民族博物馆建筑设计方案竞标会在南宁市举行，广西建筑综合设计院的设计方案入选。2003年12月，广西生态博物馆建设与发展模式高级研修班在[[河池市|河池市]][[南丹县|南丹县]]举行。2004年9月，广西壮族自治区文化厅决定成立广西民族博物馆建设领导小组。2004年10月，广西民族博物馆土石方工程开工。2006年3月2日，广西民族博物馆建设工程开工。2006年5月，广西民族博物馆获得[[国家发改委|国家发改委]]补助中央预算内投资8000万元人民币。2007年12月15日，主楼工程主体封顶。2008年12月11日，广西民族博物馆开馆仪式举行。2009年5月1日，正式对公众免费开放<ref name=dshji/>。2013年10月29日，广西民族博物馆与[[广西师范学院|广西师范学院]]合作成立的“广西影视人类学研究中心”挂牌仪式在广西民族博物馆举行<ref>{{Cite web |url =http://news.gxnews.com.cn/staticpages/20131031/newgx5271cb15-8873268.shtml  |title =广西影视人类学研究中心在广西民族博物馆挂牌  |author =  |date = 2013-10-31 |publisher =广西新闻网  |language =  |accessdate =  |quote =  }}</ref>。2016年12月30日，广西民族博物馆召开第一届理事会成立大会，广西民族博物馆是中国民族类博物馆成立理事会首家试点单位<ref name=dshji/><ref>{{Cite web |url = http://news.xinhuanet.com/local/2017-01/03/c_129429520.htm |title = 广西民族博物馆成立理事会 |author =  |date = 2017-01-03 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。2017年，广西民族博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。

广西民族博物馆位于南宁市青秀山风景区内，毗邻[[邕江|邕江]]，馆区由主楼、广西传统民居文化生态展示园组成。主楼建筑面积29370平方米，外形取自铜鼓。馆内常设6个固定陈列：《五彩八桂——广西民族文化陈列》、《穿越时空的鼓声——铜鼓文化》、《壮族文化展》、《多彩中华——中华民族文化展》、《缤纷世界——世界民族文化展览》、《昨日重现——百年老物件展》。此外还设有两个临时展厅<ref name=jj/>。

广西民族博物馆馆藏文物三万多件（套），其中国家三级以上文物三千多件（套），分为民族服饰、银器、生产生活用具、民间工艺、宗教信仰、织锦刺绣等，以青铜器、竹木器、银饰、纺织品为主<ref name=jj>{{Cite web |url = http://www.amgx.org/news-3703.html |title =广西民族博物馆简介  |author =  |date = 2016-06-01 |publisher = 广西民族博物馆 |language =  |accessdate =  |quote =  }}</ref>。

==生态博物馆==
