'''广州南站'''地铁站是[[广州地铁2号线|广州地铁2号线]]和[[广州地铁7号线|7号线]]兩線的換乘站兼終點站，位於[[广州市|广州市]][[番禺区|番禺区]]国铁[[广州南站|广州南站]]地下，2010年9月25日投入使用。

本站與位於附近的2號線大洲停車場、7號線大洲车辆段接軌。

== 車站結構 ==
===車站樓層===
广州南站地铁站现时位处国铁[[广州南站|广州南站]]地下一层至地下二层，地面为国铁站到达平台、南站东广场和西广场，这些位置均设有地铁站的出入口。地下一层为车站站厅，設有票務設施，客務中心及商店；地下二层为2号线及7号线站台层。

===站厅===
目前广州南站地铁站站厅设於国铁广州南站地下一層。为方便行人进出，站厅的东边的中心大部分地区被划分为收费区，收費區內設有多組专用电梯（[[升降机|升降机]]）、[[扶手電梯|扶手電梯]]及樓梯供乘客往返不同线路的月台。

受本站预留結構并变更的影响，目前站厅西边的大部分地区被划分为非收费区，在站厅西边处东侧，则设有售票机及云购票机等售票设施，以舒缓靠近收费区的售票机大排长龙的状况。

另外，本站的公共[[洗手間|洗手間]]设于站厅靠近D出入口旁。而广州地铁首设的母婴室则设于站厅靠近J出入口旁。

====車站商店及自助服務====
現時本站提供不同類型的商店，例如有[[便利商店|便利店]]、[[麵包糕餅店|麵包糕餅店]]、[[書店|書店]]、凉茶店、零食店等。

与其它车站相比，本站的自助服務有[[自動售賣機|自動售賣機]]、[[中国建设银行|中国建设银行]][[自动柜员机|自动柜员机]]以及自动售卡充值机等。車站內亦設有“好易”机，可充值羊城通，缴纳水电费以及交通罚款等服务。

===月台與換乘===
目前广州南站地铁站設有2個[[側式月台|側式月台]]及1個[[島式月台|島式月台]]，全部設有月台幕門。兩個側式月台分別為2號線的2號月台和7號線的3號月台；岛式月台兩側分別為2號線的1號月台和7號線的4號月台。

上述的月台配置使本站成為一個能夠提供部分方向[[跨月台換乘|跨月台換乘]]的車站。因此7號線抵站乘客能夠直接走到對面月台換乘2號線前往廣州市區；而2號線抵站乘客要換乘至7號線則需要繞經站廳<ref name="7号线开通">[http://www.huaxia.com/gdtb/gdyw/mttt/2016/12/5135293.html 广州地铁七号线28日开通 这份攻略请收好！]</ref>。
    
由於2號線規劃已宣告結束，不會再有任何延伸計劃，因此2號線在本站開通前的折返線上作出了超級特別設計：在往石壁方向的一端設一條渡線供列車站前折返；而在往大洲停車場方向的一端，下行路軌與上行路軌之間設有一條單渡線供列車站後折返。同時上行路軌在渡線前方分岔出一條路軌，與上行路軌自身一同進入大洲停車場，兩者作為出入段線；下行路軌則直接被中斷，該路軌南端為車站2號線信號電力設備供應區，因此無法再繼續延伸。2號線北端終點站嘉禾望崗站亦是類似設計，因此本站和嘉禾望崗站都成為2號線的永久終點站。2號線列車偶爾也會在非常特殊的情況下通過站前渡線直接駛入1號月台同時折返。

7号线方面，往石壁方向一端以及往大洲车辆段一端各设置一组交叉渡线，目前前者供7号线列车进行站前折返，后者供初期7号线在正常运营时进行站后折返。另外，在往大洲车辆段的一端交叉渡线后面设有两条连接正线的出入库线，正線南端盡頭亦预留正在建设中的西延伸延长。

==利用情況==
本站連接國鐵的廣州火車南站，亦鄰近广州南汽車客運站，本站继[[广州火车站|广州火车站]]、[[广州火车东站|广州火车东站]]之后，又一个成为連接全國各地及廣州市區各地的陸路交通樞紐。因此本站是廣州地鐵較為繁忙的車站之一，客流全天都維持在较高的水平，同時不少乘客亦會攜帶大量行李進出站，不过由於本站面积较大，而且是2号线，目前7号线的始发站之一，站內亦很少出现擁擠。若遇上節假日，或者一年一度的[[春运|春运]]及[[暑运|暑运]]，本站會出現大量人潮，严重时将会视情况实行客流控制措施<ref>[http://news.163.com/18/0102/01/D740ESLD00018AOP.html 广州地铁单日客流量首破千万]</ref>。

本站是2/7号线换乘站之一，由于本站是2號線的始發站，且7號線抵站乘客能直接同台換乘至2號線的缘故，部分由7号线换乘2号线的乘客为了上车时有位置坐，以及避免在石壁站換乘時繞經站廳，會选择在本站换乘<ref name="7号线开通" />。

== 历史 ==
[[File:Guangzhou_South_Railway_Station.jpg|thumb]]
{{main|广州南站}}
2003年，广州市规划选址于番禺石壁建设一个较广州火车站、广州东站规模大的新火车站。经过设计方案论证，最终于2004年12月30日动工兴建。

而作为配套设施之一的地铁车站，在2号线拆解后规划的2003年方案中，南部則改為延長至廣州新客站（現[[廣州南站|廣州南站]]），且考虑未来与7号线以及[[佛山地铁2號線|佛山地铁2號線]]进行换乘，最终于2004年随新客站工程动工而开工建设。2010年1月30日广州南站局部启用，地铁车站直到2010年9月25日新2號線和8號線通車才正式啟用。

在开通初期，本站2号线往嘉禾望岗方向的月台（即1号月台），现在4号月台一侧用本站色系的木板墙壁封闭，因此导致在乘客眼中，1号月台为侧式月台的设计。

另外本站在车站初期的設計中，7号线和佛山2號線的月台均设置在现有站厅的地下二层，两线均使用站前折返的设计。因此在车站落成时在站廳东北边偏东南侧，以及西南边中心处預留装设通往月台的扶手电梯、楼梯以及升降机会有预留安装位。即下面列出的設計：

不过，後來7號線最終確定使用6节B型列车编组，同時落實7号线將繼續往西延伸，因此7號線建設時佔用了原预留给佛山2号线的月台部分作為7号线的公共區及设备区，站廳原預留給佛山2號線的所有节点已经拆除欄杆，同時铺设永久的陶瓷地砖。佛山2号线车站最終改為在广州火车南站西广场北侧另外兴建。

2016年12月28日，因应7号线首期通车，本站7号线的两个站台正式启用，同时本站成为广州2号线和7号线的换乘车站。

== 未來發展 ==
===廣州地鐵7號線西延段===
7号线将会由本站繼續向西南方向延伸，進入[[佛山市|佛山市]][[順德區|順德區]]，並以[[美的大道站|美的大道站]]作為終點站。屆時，廣州南站將從7号线西端終點站改為中途站，目前的4号月台（即終點落客站台）将改为美的大道方向的站台。
===廣州地铁22号线===
[[廣州地铁22號線|22號線]]設有广州南站，與2號線、7号线及佛山地鐵2號線交匯<ref>[http://gd.qq.com/a/20170809/003752.htm 广州地铁22号线拟今年开工 祈福新村设站点]</ref>。預計22號線車站將設於火車南站东广场南侧的地下，為地下四層的島式月台車站。目前該車站作為廣州南站軌道交通銜接工程的一部分，正在進行工程招標<ref>{{Cite web |url=http://www.gdzbtb.gov.cn/zhaobiao12/201411/t20141107_267042.htm |title=广州南站广州市轨道交通衔接工程 施工图审查 |access-date=2015-03-16 |archive-url=https://web.archive.org/web/20150402211356/http://www.gdzbtb.gov.cn/zhaobiao12/201411/t20141107_267042.htm |archive-date=2015-04-02 |dead-url=yes }}</ref>。

===佛山地铁2号线===
[[佛山地铁2號線|佛山地铁2號線]]亦將設有广州南站，與广州2號線、7号线及22號線交匯。由于原預留給該線的月台已被广州7号线用於进行拓宽改造擴編，目前佛山2號線車站被遷移至火車南站西廣場偏北处建设，同時亦會增建一條換乘通道接入廣州2號線、7號線車站。

===文化牆===
