'''广东药科大学附属第一医院'''，暨'''广东药科大学臨床醫學院'''，原為'''广州铁路中心医院'''，建于1950年，是一所具[[医疗|医疗]]、[[教学|教学]]等各種功能的综合性医院，等級為[[三級甲等醫院|三級甲等]]<ref name = "醫院概況">{{Cite web |url=http://www.gy120.net/new/about.asp?id=5 |title=醫院概況 |accessdate=2012-07-27 |work=广东药科大学附屬第一醫院 |publisher= |date= }}</ref>。共有两个门诊部，分别为共和门诊与农林门诊，另有江海分院。

== 历史沿革<ref name = "醫院概況"/> ==
* 1950年，'''广州铁路中心医院'''始建，隶属于[[广州铁路集团|广州铁路集团]]。
* 1993年，成为三级甲等医院。
* 2004年，移交[[广东药学院|广东药学院]]，更名'''广东药学院附属第一医院'''。<ref>{{cite news |author= |coauthors= |title=粤铁路学校医院将一次性转制划给地方政府 |url=http://news.sina.com.cn/c/2004-09-08/10323616553s.shtml |work=新快報 |publisher=新浪網 |date=2004-09-08 |accessdate=2012-07-27 }} </ref>
* 2008年，与广东药学院临床医学院合并统一。
* 2016年，因广东药学院更名为[[广东药科大学|广东药科大学]]，医院也更名为'''广东药科大学附属第一医院'''。
=== 臨床醫學院<ref name = "院系專業">{{Cite web |url=http://www.gy120.net/new/lin.asp?id=39 |title=院系專業 |accessdate=2012-07-27 |work=广东药科大学臨床醫學院 |publisher= |date= }}</ref> ===
* 1979年10月，广东医药学院临床教学部成立
* 1980年，本科临床医学专业开办
* 1983年，临床医学系成立
* 2005年6月，临床医学院成立
* 2008年9月，与廣藥附一统一
