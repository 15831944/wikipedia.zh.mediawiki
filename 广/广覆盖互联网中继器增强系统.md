'''广覆盖互联网中继器增强系统'''（{{lang-en|'''Wide-coverage Internet Repeater Enhancement System'''，'''WIRES'''}}）是一个{{tsl|en|Yaesu (brand)|八重洲公司}}创建的连接至[[互联网|互联网]]的、使用[[网络协议通话技术|网络协议通话技术]]（VoIP）的{{tsl|en|amateur radio repeater|业余无线电中继台}}事实标准，允许使用这些中继台的任何业余无线电台通过VoIP彼此通信。<ref>{{Cite web |url=http://www.yaesu.com/jp/en/wiresinfo-en/introduction.html |title=Introducing to WIRESTM-II |accessdate=2017-07-14 |author=Yeasu |language=en |archiveurl=https://web.archive.org/web/20170714135735/http://www.yaesu.com/jp/en/wiresinfo-en/introduction.html |archivedate=2017-07-14 |deadurl=yes }}</ref>

WIRES使用[[双音多频|双音多频]]信号（DTMF）通过互联网与其他联网的WIRES电台建立连接 。<ref>{{Cite web |url=http://www.yaesu.com/jp/en/wiresinfo-en/guide.html |title=Operating Reminders for WIRES-II |accessdate=2017-07-14 |author=Yeasu |language=en |archiveurl=https://web.archive.org/web/20170714135944/http://yaesu.com/jp/en/wiresinfo-en/guide.html |archivedate=2017-07-14 |deadurl=yes }}</ref>由于WIRES并未使用任何私有的信号或协议，任何支持DTMF、带有DTMF键盘的电台都可以用于建立连接。
