'''广德中学'''，是[[安徽省|安徽省]][[宣城市|宣城市]]的一所[[高级中学|高级中学]]，为安徽省示范高中。

== 校史 ==
广德中学前身为1903年创办的“广德州中学堂”。1912年，改为广德县中学校。1982年，被列为县重点中学。1997年，被命名为地区级重点完全中学。2001年，被命名为安徽省示范高中。<ref name=jj>{{cite web|url=http://www.ahgdzx.com.cn/contents/3/170.html|title=安徽省广德中学简介|publisher=安徽省广德中学|accessdate=2018-07-03}}</ref>

== 现状 ==
学校占地250余亩，现有学生3526名。教职工223人，专任教师194名，其中正高级教师1人，特级教师5人，市学科带头人12人，高级教师93人。<ref name=jj/>

== 知名校友 ==
* [[许杰|许杰]]：原地质部副部长，中国科学院院士
* [[徐有芳|徐有芳]]：原林业部部长
