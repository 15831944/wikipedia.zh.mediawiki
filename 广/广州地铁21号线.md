'''广州地铁21号线'''是广州地铁的路线之一，屬通勤鐵路。大致呈东西走向，连接[[增城区|增城区]]、[[黃埔區|黃埔區]]与廣州市区。代表色是深藍色。21號線將是首批按照「大站快線」建設的線路之一。

2018年12月28日，开通镇龙西至增城广场段共26.2公里；剩餘員村至鎮龍西段共36.3公里預計需留待2019年開通。<ref name="EIA2III" />

== 概要 ==
21号线起于天河区的員村及天河公园西南角，经过天府路、棠德南路、大觀南路及規劃高唐大道，進入[[黃埔區|黃埔區]]（原萝岗区）科学城。再往東沿科學大道、開泰大道、科豐路和水西路行進，然後轉入原廣汕公路（324國道）東側，穿越沿線山嶺。隨後大致沿原廣汕公路（324國道）行進，經過东部山水新城、增城區中新鎮、朱村鎮（規劃教育城），進入增城大道後於荔城街道增城广场終止。而为了在开通初期与现有轨道交通衔接，初期线路利用[[广州地铁11号线|11号线]]的区间由[[天河公园站|天河公园]]延伸至[[员村站|员村]]，待11号线开通时再拆解改回天河公园站起点。

线路全长约62.5公里，其中地下线长约40公里，穿山隧道7公里，地上线15公里。共设置21座车站，其中地下车站17座，高架车4座，共有7座换乘站。<ref name="EIA2III">{{cite web | url=http://www.eiafans.com/thread-417496-1-1.html | title=广州市轨道交通二十一号线工程（员村～增城广场） 环境影响公众参与公告(第二次) （2012年9月） | date=2012-09-07 | accessdate=2012-09-07}}</ref>平均站间距3.685公里，最大站间距7.325公里，为坑貝至鳳崗区间，最小站间距1.38公里，为黄村至大观南路区间。<ref>[http://www.gzmtr.com/bid/ztbgl/bid_project_info.jsp?proj_id=2012070039&note_id=1&note_type=1 广州市十二五轨道交通线网前期深化研究（设计、设计咨询）招标公告] {{webarchive|url=https://web.archive.org/web/20131020021623/http://www.gzmtr.com/bid/ztbgl/bid_project_info.jsp?proj_id=2012070039&note_id=1&note_type=1 |date=2013-10-20 }}</ref>。

== 历史 ==
=== 规划 ===
广州地铁21号线黄村—水西段原本属于[[广州地铁4号线|4号线]]北延段，2010年底，为了促进东部发展，这一段規劃被独立并向两端延长，成为21号线。

2012年8月13日，环保部开始第二次环评公示，与此前公布的站点不同的是，从17个站点变成了18个，增加了天河智慧城站；此外，换乘站点也变成了7个，串联起9条地铁和城际。<ref name="EIA2I XKB">{{cite web | url=http://news.xkb.com.cn/guangzhou/2012/0812/217781.html | title=广州地铁21号线新增智慧城站 2016年建成开通 | publisher=新快网 | work=[[新快报|新快报]] | date=2012-08-12 | accessdate=2012-09-07 | deadurl=yes | archiveurl=https://web.archive.org/web/20140808042728/http://news.xkb.com.cn/guangzhou/2012/0812/217781.html | archivedate=2014-08-08 }}</ref>同年8月14日，环评公示修改，珠吉路站拆分為神舟路站和科学广场站（現科學城站）。从18个站点变成了19个。<ref name="EIA2II">{{cite web | url=http://www.eiafans.com/thread-402786-1-1.html | title=广州市轨道交通二十一号线（东部新城快线）项目环境影响公众参与公告(第二次) | date=2012-08-14 | accessdate=2012-09-07}}</ref>9月7日，环评公示再次修改。中新站西移，新增中新东站（現坑貝站）。同时，镇龙车辆段亦往南移了一点。<ref name="EIA2III" />2013年5月6日，根据公开的施工监理招标公告，水西—长平站前区间改为地下。<ref>{{Cite web|url=http://www.gzzb.gd.cn/cms/wz/view/other/jgy.jsp?infoId=209491&channelId=15&siteId=1|title=广州市轨道交通二十一号线土建工程施工监理服务项目6、8、9、10标公开招标公告|accessdate=2018-10-19|author=|date=|work=www.gzzb.gd.cn|publisher=}}</ref>

2015年4月29日，廣州地鐵發佈車站設計招標公告，明確21號線再增加一個朱村東站（現朱村站）。<ref>{{Cite web|url=http://bid.gzmtr.com/bizzbgg/bid_ztgg_jszb/201504/t20150429_39252.html|title=广州市轨道交通二十一号线朱村东站设计招标公告|accessdate=2018-10-19|author=|date=|work=bid.gzmtr.com|publisher=}}</ref>

=== 建设 ===
2012年4月13日，21号线现场查勘工作开展。2014年2月，二十一号线工程可行性研究报告获得廣東省发改委批复，可以正式动工建设。截至同年6月底，已有10個車站和5個區間動工，其它工點亦在进行管线迁改和前期准备工作。

2018年7月30日，镇龙至增城广场段轨行区移交运营部门进行调试<ref>{{Cite web|url=http://m.sohu.com/a/244562136_267106|title=广州地铁21号线镇龙至增城广场段移交运营调试-新闻频道-手机搜狐|accessdate=2018-10-19|work=m.sohu.com|language=zh-cn}}</ref>。

2018年8月，天河公園站主體結構封頂，標誌着二十一號線所有車站完成主體結構工程。

二十一號線鎮龍西站至增城廣場9個車站及軌行區已於2018年10月30日從建造事業總部將屬地管理權、調度指揮權、設備操作維護使用權移交給營運事業總部，隨即進入營運調試階段。

2018年12月26日廣州地鐵在微博公佈，二十一號線鎮龍西站至增城廣場站將於12月28日12:28起開通試運營<ref name="启用">[http://kb.southcn.com/content/2018-12/26/content_184606570.htm 官宣！2日后，广州地铁新线“三线齐发”]南方网 2018年12月26日</ref>。

2018年12月28日12:28，21號線鎮龍西站至增城廣場正式開通試運營。

2019年1月，天河智慧城站至大觀南路站盾构隧道完工，标志着21號线全线隧道贯通。

== 车站 ==
21号线共设21座车站，其中地下车站17座，高架车站4座，共有7座换乘站。

此外，全线设置一段两场，在黃埔区九龙镇设镇龙车辆段，在黃埔区水西村南侧设水西停车场，在增城市山田村东侧设象岭停车场。<ref>{{Cite web|url=http://gd.sina.com.cn/news/bm/2012-08-13/07396252.html|title=广州地铁21号线有望2016年开通 共设18站点_新浪广东城事_新浪广东|accessdate=2018-10-19|work=gd.sina.com.cn|archive-url=https://web.archive.org/web/20121205172229/http://gd.sina.com.cn/news/bm/2012-08-13/07396252.html|archive-date=2012-12-05|dead-url=yes}}</ref>

值得一提的是，神舟路站、水西站、金坑站、镇龙西站、中新站、朱村站、山田站会设置越行线。

=== 换乘站 ===
* '''[[员村站|员村站]]'''：换乘[[广州地铁5号线|5号线]]。因建设时未作预留，预计为站厅换乘。同时此站在11号线建成后将划归11号线。
* '''[[天河公园站|天河公园站]]'''：换乘[[广州地铁11号线|11号线]]、[[广州地铁13号线|13号线]]。三線車站統一設計，同步建成。其中与11号线为平行换乘，两者与13号线預計為站台節點换乘。同时此站在11号线建成后将成為21号线的終點站。
* '''[[黄村站_(广州)|黄村站]]'''：换乘[[广州地铁4号线|4号线]]。两线大致平行，但因建设时未作预留，且21號線的車站比4號線車站偏北，预计为站厅通道换乘。
* '''[[大观南路站|大观南路站]]'''：换乘[[广州地铁19号线|19号线]]。后者为远期线路，預計只會預留通道换乘條件。
* '''[[苏元站|苏元站]]'''：换乘[[广州地铁6号线|6号线]]。两者分期建设，T字型节点换乘。本站設計時並不是轉乘站，後來在6號線動工後突然確定為轉乘站，才緊急停工並更改設計作出預留。
* '''[[水西站_(广州)|水西站]]'''：换乘[[广州地铁7号线|7号线]]。本站設計時並不是轉乘站，後來在21號線動工後突然確定為轉乘站，才更改設計作出預留。
* '''[[镇龙站|镇龙站]]'''：换乘[[广州地铁14号线|14号线]]支线（知识城线）和[[穗莞深城际轨道交通|穗莞深城际线]]。14號線和21號線两者同期建设，雙島四線式平行换乘。
* '''[[增城广场站|增城广场站]]'''：换乘规划的[[广州地铁16号线|16号线]]。后者为远期线路，预计会预留T字型节点换乘。

==行車安排==
21號線鎮龍西至增城廣場段通車後至全線通車前採取單一路線營運，所有列車皆往返鎮龍西站及增城廣場站之間，只有各站停車服務，全日行車間隔為8.5分鐘一班。

待全線通車後，21號線預計將採取快慢車形式運營，根據廣州地鐵提供的資料<ref>{{Cite web|url=http://www.gzmtr.com/ygwm/xwzx/gsxw/201402/t20140227_34031.html|title=二十一号线将按快慢车设计 全线20站初期快车拟停9站|accessdate=2018-10-19|author=广州地铁|date=2014-02-27|publisher=}}</ref>，屆時21號線初期的快慢車停站情況如下：
* '''慢車為：員村──增城廣場，各站停車'''
* '''快車只停以下大站：員村─天河公園─棠東─黃村─大觀南路─蘇元──鎮龍──鳳崗──增城廣場'''

地铁设计院表示，设计上快慢车运行比例为1:2，这样的考虑，既可以满足站站停乘客的基本需求，也能让乘坐快车的市郊乘客确保1小时内到达市中心的出行目标。

初期高峰小时开行12对（5分鐘一班），近期高峰小时开行20对（3分鐘一班），远期高峰小时开行24对（2分30秒一班）。

==乘客量==
21號線現時只開通東段，只能換乘14號線支線，未能直接進入廣州市區。雖然14號線支線加開了鎮龍來往嘉禾望崗的快車，以儘量縮短21號線沿線來往市區的時間，但因路線迂迴，換乘次數多，對沿線居民吸引力有限。目前21號線客流僅以來往沿線鎮街及荔城的乘客為主，預計待全線開通後客流才能有大幅增長。

== 列车 ==
{{seealso|廣州地鐵二十一號線列車}}
21号线计划使用6节编组B型列车，最高运行速度120km/h。初期配屬列車33列（198輛）。

21號線相關列車合約已被[[中車株洲電力機車|中車株洲電力機車]]公司中標。除首列列車（21x001-21x002）於[[中車株洲電力機車|中車株洲電力機車]]生產外，其餘列車均在中車株洲電力機車公司屬下的廣州中車軌道交通裝備公司生產。

== 建设意外 ==
2018年1月25日下午5时左右，蘇元至水西左線盾构区间，盾构机带压开仓动火作业时，焊机电缆线短路引起火灾，造成3名仓内工作人员失蹤，而施救过程中土仓压力急速下降，掌子面土体失稳，突发坍塌，最後造成3人死亡。事故發生後，施工單位虛報意外發生時間是「2018年1月26日上午7时左右」，以及意外事件為「盾构机在换刀作业时隧道突发塌方」。該標段施工单位為中铁十四局集团有限公司，监理单位為广东至艺工程建设监理有限公司。<ref>{{cite news|url=http://www.zaobao.com/realtime/china/story20180201-831927|title=广州地铁隧道作业塌方致三人死亡|publisher=聯合早報|date=2018-02-01}}</ref><ref>{{Cite web|url=http://www.gz.gov.cn/gzgov/s16755/201808/0faed371a05a4e2c9f257f8622842c81.shtml|title=广州市人民政府门户网站 - 广州市黄埔区中铁十四局集团隧道工程有限公司“1·25”较大坍塌事故调查报告|accessdate=2018-10-19|work=www.gz.gov.cn|language=zh-CN}}</ref>

== 未来发展 ==
=== 天河公园至员村段拆解 ===
由于预计在21号线通车之时，途经21號綫規劃終點天河公园站的[[广州地铁11号线|11号线]]和[[广州地铁13号线|13号线]]兩條市區線路仍未通车，導致天河公園終點站成為「斷頭路」。同時途经世界大观站换乘的[[广州地铁19号线|19号线]]属于远期规划线路，而分别于黄村和蘇元換乘的4号线和6号线，因為运力較小亦无法承受较大的换乘客流。因此在初期将先行建设并優先借用属于11号线的天河公园—员村区间，暫時把該線與5號綫交匯，從而令21號線接入5號線，更好地融入市区线网。待未來11号线开通前夕，天河公园—员村区间将会拆離21号线，回归成为11号线的一部分。

===预留延伸===
21号线增城廣場站將预留延伸條件，未來可繼續往東延伸，跨過增江進入東岸的增江街道及更遠地區。
