'''广州白云国际机场'''（{{airport codes|'''CAN'''|'''ZGGG'''}}），通稱'''广州白云机场'''、'''广州机场'''，简称'''白云机场'''，是[[中华人民共和国|中华人民共和国]][[广东省|广东省]][[广州市|广州市]]的一座民用[[机场|机场]]，地处[[白雲區_(廣州市)|白云区]][[人和镇_(广州市)|人和镇]]和[[花都区|花都区]][[新华街道_(广州市)|新华街道]]、[[花山镇_(广州市)|花山镇]]、[[花东镇|花东镇]]交界处<ref>{{Cite web|url=https://www.google.com/maps/place/%E5%BB%A3%E5%B7%9E%E7%99%BD%E9%9B%B2%E5%9C%8B%E9%9A%9B%E6%A9%9F%E5%A0%B4/@23.3959128,113.3057812,17z/data=!3m1!4b1!4m5!3m4!1s0x3402e130a02328f3:0xa2467af91f7f31a3!8m2!3d23.3959079!4d113.3079699|title=谷歌地圖：廣州白雲國際機場|accessdate=2018-07-10}}</ref>，距广州市中心的直线距离约30公里，于2004年8月5日正式启用<ref name="白云机场介绍">{{Cite web|url=http://www.gbiac.net/gywm/airportintroduce|title=白云机场介绍|accessdate=2017-02-27}}</ref>。其两个机场代码均沿用自[[广州白云国际机场_(1933年-2004年)|原广州机场]]，[[IATA代码|IATA代码]]则源于广州一词的[[郵政式拼音|拉丁字母習慣拼法]]“{{lang|en|Canton}}”的缩写。白云机场目前为[[中国南方航空|中国南方航空]]、[[海南航空|海南航空]]、[[联邦快递|联邦快递]]、[[深圳航空|深圳航空]]的[[枢纽机场|枢纽机场]]<ref name="中國三大機場">{{Cite web|url=http://baijiahao.baidu.com/s?id=1563270843916477&wfr=spider&for=pc|title=中国最美的三大机场|accessdate=2017-03-30|author=百家號}}</ref>及[[中国国际航空|中国国际航空]]的[[重点机场|重点机场]]，是中国三大国际枢纽机场之一<ref name="世界机场排名">{{Cite web|url=https://zh.wikipedia.org/wiki/%E5%85%A8%E7%90%83%E6%9C%BA%E5%9C%BA%E5%AE%A2%E8%BF%90%E5%90%9E%E5%90%90%E9%87%8F%E5%88%97%E8%A1%A8#2016.E5.B9.B4.E5.88.9D.E6.AD.A5.E4.B8.8D.E5.AE.8C.E5.85.A8.E6.95.B0.E6.8D.AE|title=世界机场排名（不完全数据）|accessdate=2017-02-27|author=|date=|publisher=}}</ref>。

截至2017年底，白云机场航线网络已覆盖全球210多个通航点，其中国际及地区航点近90个，有超过75家中外航空公司在此运营<ref name="白云机场介绍"></ref>，年起降量达46.52万架次；2017年白云机场旅客吞吐量达6583.69万人次，位居[[全球機場客運吞吐量列表|世界十三]]<ref name="世界机场排名"></ref>、[[中华人民共和国机场列表|中国第三]]<ref name="中国机场排名">{{Cite web|url=http://www.caac.gov.cn/XXGK/XXGK/TJSJ/201702/t20170224_42760.html|title=2016年民航机场生产统计公报|accessdate=2017-02-27|author=中国民航局|date=|publisher=|archive-url=https://web.archive.org/web/20150823011202/http://www.caac.gov.cn/XXGK/XXGK/TJSJ/201702/t20170224_42760.html|archive-date=2015-08-23|dead-url=yes}}</ref>；货运吞吐量165.22万吨，位居[[全球机场货运量列表|世界前二十]]<ref name="世界机场排名"></ref>、中国第三<ref name="中国机场排名"></ref>。

== 历史 ==
===舊廣州白雲國際機场===
[[File:广州老白云国际机场夜景3711711.jpg|thumb]]
{{Main|广州白云国际机场 (1933－2004年)}}
中華民國二十一年（1932年）冬，国民革命军第一集团军空军开始在广州的远郊棠下、棠溪、岗贝、萧岗乡征地，始建廣州航空站，因為东面有[[白云山_(广州)|白云山]]，所以也稱白雲機場。<ref name="第一代航站楼">{{cite web|url=http://byss.by.gov.cn/html/2017-3-3/6396_41625.html?c=1&id=41625|title=第一代航站楼：见证广州民航事业的发展|publisher=白云时事|date=2017-03-03}}</ref>

中華民國二十二年（1933年）夏季落成，同年11月完工，广东航空学校随即从天河航空站迁入，成为飞行学员飞行训练的基地。<ref name="第一代航站楼"></ref>

中華民國二十七年（1938年10月），广州沦陷后，廣州航空站被日军侵占。之后，日军对机场进行了扩建，将跑道向北延长至1200米、加宽为100米、加厚为0.1米，并增建一条与跑道同等长宽厚的副跑道，机场总面积扩大到134万平方米。<ref name="老白云机场的轨迹">{{cite web|url=http://2016.163.com/16/0813/13/BUBPA4RT00050IU4.html|title=老白云机场的轨迹|publisher=羊城晚报|date=2016-08-13}}</ref>

中華民國三十六年（1947年）3月，国民政府交通部民用航空局接管廣州航空站，主要供[[中国航空|中国航空]]、[[中央航空|中央航空]]和由美国人陈纳德经营的国民政府行政院善后救济总署空运队的飞机起降。<ref name="第一代航站楼"></ref>

中華民國三十七年（1948年）12月，改為民航國際機場，国民政府交通部民航局投入巨资对廣州航空站进行扩建，于1949年5月2日竣工，7月1日投入使用，工程投资总额为金圆券6800万元。<ref name="老白云机场的轨迹"></ref>

1950年3月，[[中國人民解放軍空軍|中國人民解放軍空軍]]中南军区司令部航空处将廣州航空站移交给新成立的中国民航广州办事处。<ref name="老白云机场的轨迹"></ref>

1950年至1959年間，改為[[中國人民解放軍空軍|中國人民解放軍空軍]]使用。<ref name="老白云机场的轨迹"></ref>

1959年，广州飞行队由天河航空站迁至廣州航空站，改为军民共用机场。 <ref name="白雲機場歷史">{{cite web|url=http://travel.sina.com.cn/air/2008-08-19/163117153.shtml|title=广州白云国际机场的历史|publisher=[[新浪|新浪]]|date=2008-08-19}}</ref>

1963年2月，[[中國人民解放軍空軍|空军]]搬离廣州航空站並重新改為民航機場。同年10月，白云机场扩建工程修建委员会和总指挥部成立，並為廣州航空站更名，正式得名廣州白云机场。<ref name="白雲機場歷史"></ref>

1964年至1967年4月間，機場進行全面擴建，新增面积72.53万平方米，跑道从2000米延长至到2500米，才具備再與外國通航的條件。<ref name="白雲機場歷史"></ref>

1980年代，機場又分別對儲油庫、[[停机坪|停机坪]]、[[候机楼|候机楼]]、[[登机桥|登机桥]]和維修機庫等設施進行改造和擴建，達到國際一級機場的標準。[[改革开放|改革开放]]后白云机场发展迅猛，其旅客吞吐量和起降架次曾连续8年位居中国大陸第一，经过数次扩建但仍远远无法满足需求，择新址建设新机场势在必行。

1992年，機場進行民航體制改革，机场作为一个独立的经济实体运作。

1997年，机场第六次改扩建工程完工，可承受日旅客吞吐量由1.2万人次增加到4.1万人次<ref name="白雲機場歷史"/>，设计年旅客吞吐量由439.2万人次调至1500.6万人次。

2002年，舊廣州白雲國際機场的旅客吞吐量達到1601.44萬人次、貨郵行吞吐量達到59.26萬噸。<ref>{{cite web|url=http://news.163.com/14/0805/07/A2S9IU1I00014Q4P.html|title=一夜转场 十年飞跃|publisher=[[信息时报|信息时报]]|date=2014-08-05}}</ref>

2004年8月5日，服务了广州72年的舊白云机场随新白云机场的启用而关闭。

===新廣州白雲國際機场===
1992年，开始进行新白雲机场的选址工作，经过多年准备，最终选址距市区北部28公里的花都区新华街道、花山镇、花东镇及白云区人和镇的交界处，占地规模比原机场大近5倍。<ref>{{cite web|url=http://www.sohu.com/a/135433691_526351|title=东方乐园与老白云机场：消失了的风景|publisher=[[搜狐|搜狐]]：历史现场|date=2017-04-21}}</ref>

2000年8月，新机场正式动工，耗资198亿元人民币。

2004年8月2日，新白云机场竣工，并于5日零时正式启用。新机场也被称为“新白云”，以区别于旧机场，但这并非正式名称的一部分。<ref>{{cite web|url=http://news.southcn.com/gdnews/sd/200602120059.htm|title=见证“十五”：建新机场十月夷平五座山|publisher=南方网|date=2006-02-12}}</ref>

2005年9月，广州白云国际机场在全国民航首先创立"异地城市候机楼模式"，率先在东莞建立了内地第一个异地候机楼。<ref>{{cite web|url=http://travel.cnr.cn/2011lvpd/gny/20140805/t20140805_516121656.shtml|title=新白云机场10年 挺进全球16强|publisher=广州日报|date=2014-08-05}}</ref>

2010年2月9日，为迎接[[2010年亚洲运动会|广州亚运会]]，广州白云国际机场一号航站楼东三西三指廊正式启用，<ref>{{cite web|url=http://news.southcn.com/g/2010-02/08/content_9076404.htm|title=白云机场驾车接客规定明起大调 机场东三西三指廊启用|publisher=南方网|date=2010-02-08}}</ref>设计年旅客吞吐量从2500万人次调至3500万人次。

2012年7月20日，国家发改委批准了广州白云国际机场扩建工程可行性研究报告。<ref>{{cite web|url=http://www.caac.gov.cn/XWZX/DFDT/201208/t20120810_19816.html|title=广州白云国际机场扩建工程可研报告获国家发改委批准|publisher=中国民用航空局|date=2012-08-10}}</ref>

2012年8月3日，广州白云国际机场扩建工程正式动工，扩建内容包括第三跑道和滑行道系统、二号航站楼一期建设及相关附属设施等。<ref>{{cite web|url=http://www.caac.gov.cn/XWZX/MHYW/201208/t20120809_13273.html|title=广州白云国际机场扩建工程正式开工|publisher=中国民用航空局|date=2012-08-09}}</ref>

2015年2月，新白云机场第三跑道投入使用。<ref>{{cite web|url=http://news.sina.com.cn/c/2015-02-06/050231488802.shtml|title=广州白云机场第三跑道启用|publisher=新浪新闻|date=2015-02-06}}</ref>

2018年4月，新白云机场二号航站楼正式投入使用<ref name="白雲機場二號航廈啓用">{{cite web|url=http://baijiahao.baidu.com/s?id=1598784115483455061&wfr=spider&for=pc|title=白云机场T2航站楼启用|publisher=广州日报|date=2018-04-26}}</ref>，设计年旅客吞吐量从3500万人次调至8000万人次。

== 机场概况 ==
[[File:Guangzhou_Baiyun_Airport_Control_Tower.jpg|210px]]
广州白云国际机场是中国三大国际枢纽机场之一<ref name="中國三大機場"></ref>，所在地广州是华南地区最大的进出口岸和重要的交通枢纽，截止2017年底，白云机场航线网络已覆盖全球210多个通航点，其中国际及地区航点超过90个，通达全球50多个国家和地区，覆盖东南亚、连接欧美澳，辐射内地各主要城市；已有近75家航空公司在白云机场运营，其中外航和地区公司超过50家。从白云机场起飞，6个小时内可飞抵亚太20多个城市和地区<ref name="白云机场介绍"></ref>。

白云机场现有三条跑道，一号航站楼总面积52.3万平方米，138个客机位，45个货机位，由主楼、连接楼、指廊和高架连廊组成，共分四层。第三层为出发及候机大厅，第二层为到达夹层，第一层为到达及接机大厅，负一层则通往地铁及停车场。出发及候机大厅位于航站楼第三层(3F)，共有8个进出口。的士和大巴均可直接停靠在航站楼出发大厅门口。二号航站楼总面积88.07万平方米，商业面积4.3万平米，值机柜台397个。<ref>{{Cite web|url=http://www.gbiac.net/gywm/airportsituation|title=机场概况|accessdate=2017-02-27|author=|date=|publisher=}}</ref>

自2004年转场运营以来，广州白云国际机场连续实现旅客吞吐量的快速增长，成功实现多次跨越：2004年转场当年，即过2,000万人次，2007年又突破3,000万人次，2010年一举跨越4,000万人次大关，以每3年旅客吞吐量净增1,000万人次的增速发展。2016年吞吐量接近6,000万人次，2017年吞吐量6580万人次，2018年吞吐量6972万人次<ref>{{Cite web|url=www.caac.gov.cn/PHONE/XXGK_17/.../t20190305_194972.html|title=2018年民航机场生产统计公报 - 中国民用航空局|accessdate=2019-04-17|author=|date=|publisher=}}</ref>。

作为中国“十二五”规划和《珠三角规划纲要》重点建设项目，白云机场扩建工程于2012年8月正式动工，建设项目包括第三条跑道及滑行系统、T2航站楼、站坪及空管、供油等配套设施，设计年旅客吞吐量为8,000万人次、货邮吞吐量250万吨、飞机起降量62万架次。2015年2月5日，第三跑道投入运营；2018年4月26日，T2航站楼落成并投入使用。此外，与白云机场扩建工程同步建设的还有白云机场综合交通枢纽工程。按照规划，广佛环线、穗莞深2条城际轨道将并线引入1号和2号航站楼，广清城际轨道则通过广州北站和白云机场之间的轨道交通实现与机场对接。广东省机场管理集团相关负责人表示，白云机场未来将建成集民航、城轨、地铁、高速公路等多种交通运输方式于一体的立体式综合交通枢纽。<ref>{{Cite news|url=http://weibo.com/ttarticle/p/show?id=2309614015578842018517|title=广州白云机场扩建 目标是全球重要航空综合枢纽|last=|first=|date=|work=|accessdate=}}</ref>

目前以白云机场作为基地航空公司的有[[中国南方航空|中国南方航空]]、[[深圳航空|深圳航空]]、[[海南航空|海南航空]]、[[联邦快递|联邦快递]]、[[九元航空|九元航空]]及[[广东龙浩航空|龙浩航空]]<ref name="中國三大機場"></ref>。

==== 二号航站楼 ====
[[File:Guangzhou_Baiyun_International_Airport_Terminal_2_Departure_Lobby.jpg|210px]]
二号航站楼由广东省建筑设计院主导设计，此项目由陈雄和Mark Molen再次担任主设计师<ref name="白雲機場的設計師"></ref>。同一号航站楼一样由主楼、连接楼、指廊和高架连廊四大建筑结构组成，但建筑面积高达65.87万平米，一期规划商业面积4.3万平方米。二号航站楼的登机桥有58座，值机岛13个，旅客座椅高达11万张。在候机区里，在原有基础设计上，改为混流设计，国际流程和国内流程的办理更为简单，采用“指廊式＋前列式”混合构型，具有飞机停靠面长，近机位多、近距离大机位多、旅客步行距离适中、国际国内互换方便等优点<ref>{{Cite web|language=zh|title=新白云国际机场T2航站楼设计|publisher=社会洞察力、陈雄|accessdate=2018-03-03|url=http://baijiahao.baidu.com/s?id=1593874005789029575&wfr=spider&for=pc|archivedate=2018-07-03}}</ref>。设计引入可在国内与国际运营之间进行转换的混合机位概念，在流线最便捷、流程最短的前列式机位设置国内/国际可切换的混合机位，提高机位使用灵活性及旅客处理效率，满足国际业务快速增长的需求。<ref name="白雲機場二號航廈啓用"></ref>
另外，二号航站楼出发和到达国际边检区域均设有自助通关设备，其中出境自助通道约20条，入境自助通道约30条，符合条件的旅客可自助验证，通过“刷护照”“[[人脸识别|刷脸]]”出入境，实现10秒快速通关。<ref name="白雲機場二號航廈啓用"></ref>

此外，白雲機場二號航站楼于大中華地區内首创“2+X”语言显示模式，于登机口、行李提取區的屏幕上除[[中文|中文]]、[[英文|英文]]以外以目的地、經停地或始發地的官方文字显示該地的地名，例如前往/經停或始發自[[仁川國際機場|仁川國際機場]]的航班，登机口與行李提取區的屏幕上的目的地、經停地或始發地名稱首行顯示簡體中文“首爾仁川”，外語名稱行則英文“SEOUL”和韓語“인천”滾動顯示<ref>{{cite web|url=http://news.ycwb.com/2018-06/08/content_30026259.htm|title=国内首创“2+X”语言显示！白云机场2号航站楼航显系统还有这些新功能～|publisher=白云机场发布|date=2018-06-08|accessdate=2018-06-16}}</ref>。唯前往以及來自[[香港國際機場|香港]]、[[臺灣桃園國際機場|臺北桃園]]、[[臺中清泉崗機場|臺中]]、[[高雄國際機場|高雄]]、[[東京國際機場|東京羽田]]、[[成田國際機場|東京成田]]、[[关西国际机场|大阪]]和[[福岡機場|福岡]]的航班在屏幕上的外語名稱行不顯示[[繁體中文|繁體中文]]或[[日本汉字|日本汉字]]，只顯示[[英文|英文]]，但前往以及來自[[中部國際機場|名古屋]]的航班在屏幕上的外語名稱行雖不顯示[[日本汉字|日本汉字]]但显示日語[[平假名|平假名]]。

二号航站楼于2019年1月17日荣获[[Skytrax|Skytrax]]“全球五星级航站楼”认证<ref>{{Cite web|title=广州白云机场T2获“全球五星航站楼”认证|url=https://baijiahao.baidu.com/s?id=1622900864194157342&wfr=spider&for=pc|work=中國新聞網|accessdate=2019-03-29}}</ref>。

=== 貨運設施 ===
==== 联邦快递亚太转运中心 ====
[[File:FedEx_apron_(ZGGG).png|210px]]
2005年7月13日，[[联邦快递|联邦快递]]与白云国际机场签署了在机场内兴建全新亚太转运中心的协议，新的转运中心取代了联邦快递原本设于[[菲律宾|菲律宾]][[苏比克湾国际机场|苏比克湾国际机场]]的枢纽地位<ref name="fedex2005">{{cite press release
|url=http://news.van.fedex.com/intl/fr?node=5160|title=FedEx Expands Leadership in Asia Pacific with Plan to Build Region's Largest Air Cargo Hub in Guangzhou, China|publisher=FedEx|date=2005-07-13|accessdate=2008-09-04}}</ref>。

新的亚太转运中心占地63公顷，总建筑面积达82,000平方米<ref name="fedex2005"/>。投入运作后从业人员超过800名，每周营运136个航班，在20个[[亚洲|亚洲]]主要城市间提供运送服务，并通过这些城市连接全球超过220个国家和地区<ref>{{cite press release|title=FedEx Express Guangzhou Hub Fact Sheet|publisher=FedEx|date=2010-05|accessdate=2010-05-19|url=http://news.van.fedex.com/files/Fact%20sheet%20-%20FedEx%20APAC%20Hub_GZ_May2010_E.doc}}</ref>。

转运中心的一大特点是联邦快递拥有自己的机坪[[控制塔|控制塔]]，对于一家国际航空快递运输公司而言，这在全中国也是首开先河。联邦快递可自主在地面完成控制飞机起降、停机安排及优先上落货等程序。此外，该中心的主要分拣区由16条高速分拣带、7条用于货物装载的输送带以及90部主要和次要文件分拣设备组成，每小时可处理2.4万件货物<ref name="fedex2008"/> 。

转运中心于2006年开始奠基兴建，原定于2008年12月26日投入使用。但经过数月的测试后，联邦快递于2008年12月17日宣布将开业日期推迟至2009年上半年，转运中心方可全面投入营运。联邦快递称，修改后的开业日期“为联邦快递提供了必要的时间来充分测试所有的系统和流程，以及与广州当局密切合作，确保所有必要的批准到位”<ref name="fedex2008">{{cite press release|url=http://news.van.fedex.com/node/12029|title=New FedEx Asia-Pacific Hub Prepares for Opening|publisher=FedEx|date=2008-11-17|accessdate=2008-11-18}}</ref>。

2008年12月17日，转运中心完成了其首次试运作。一架联邦快递[[麦道MD-11|麦道MD-11]]飞机从菲律宾苏比克湾国际机场起飞，于凌晨5:50分顺利降落广州白云国际机场。该航班由全新的运转中心负责全部地面运作，使用了联邦快递控制塔和新分拣系统。在地面流程成功完成后，该航班准点飞往其最终航点巴黎夏尔·戴高乐机场。这条亚欧航线在试运作期间每周执飞4次。新转运中心从2009年2月6日正式营运<ref name="fedextest">{{cite press release|url=http://news.van.fedex.com/intl/cn?node=12371|title=FedEx Initiates Flight Operations Test at its New Asia Pacific Hub in Baiyun International Airport in Guangzhou|publisher=FedEx|date=2008-12-17|accessdate=2008-12-19}}</ref>。

2009年2月6日清晨，在最后一个航班飞往[[台湾|台湾]]后，联邦快递关闭了其位于菲律宾北部已运作了13年的的苏比克湾亚太转运中心，其核心业务已全数转移至白云机场<ref>{{cite web|url=http://www.google.com/hostednews/afp/article/ALeqM5jHsE_zHv0oatAy5umP6gEiIg-ocQ|title=FedEx closes hub in Philippines|publisher=''[[Agence_France-Presse|AFP]]''|date=2009-02-06|accessdate=2009-02-07}}</ref>。飞抵新亚太转运中心的首个航班来自[[印第安纳波利斯国际机场|印第安纳波利斯国际机场]]，这架以MD-11执飞的航班在经停巴黎夏尔·戴高乐机场后于夜里23:10降落白云机场，标志着新的亚太转运中心的开放并全面投入运作<ref>{{cite press release|url=http://news.van.fedex.com/Guangzhou|title=FedEx Express Opens New Asia Pacific Hub in Guangzhou, China|publisher=FedEx|date=2009-02-09|accessdate=2009-02-19}}</ref>。

=== 跑道及停机坪 ===
[[File:Baiyun_Airport_T1_and_T2.jpg|210px]]
白云国际机场一期工程场内占地1,434公顷，其中飞行区约860公顷，其他区域占地574公顷。此外，中转油库和导航工程占地约17公顷。

其中，总平面规划有五条[[跑道|跑道]]，海拔50英尺（15米）。一期建有二条跑道，分别位于航站楼的东西两侧。间距按2,200米规划，能满足二条平行仪表跑道的独立平行进近要求。东跑道长度按4,000米宽60米规划，一期按3,800米建设，满足现有目前全球最大型的客机[[空中巴士|空中巴士]][[A380|A380]]全重起降，同时并考虑未来更大机型的起降可能。西跑道长度按3,600米宽45米建设，满足4E类以下飞机的全重起飞和降落。在东跑道的东侧680米处建有第三条跑道。该跑道长3,800米宽60米，标准为4F级，能够起落包括A380在内的大型客机起降，于2012年8月20日动工建设，2014年7月31日完工<ref>[http://news.carnoc.com/list/302/302402.html 白云机场第三跑道及配套工程顺利通过行业验收]</ref>。
机场三跑建成后重新对三条跑道进行命名：第一跑道（西跑道）02L/20R改为01/19、第二跑道02R/20L改为02L/20R、第三跑道为02R/20L。

停机坪包括三部分：客机坪、货机坪、航空公司停机坪(包括维修机坪)。停机坪目前共提供超过60个近机位（可通过登机桥直接登机）和28个远机位（需乘摆渡车登机），停机坪总面积超过90万平方米。

== 远期规划 ==
{{GZ future infrastructure}}
{{update|章節|time=2019-01-05T23:40:00+08:00}}
2017年2月，位于国家级临空经济示范区范围内的广州白云国际机场最新的规划方案获规委会通过。地跨白云和花都两区的国家级临空经济示范区总面积达135 .5平方公里，116平方公里的空港经济区位于示范区范围内，而白云机场最新获通过的规划范围则在空港经济区范围内，占地约41.64平方公里。规划范围东面毗邻流溪河，西至大广高速，南到机场南侧规划路，北至机场横十六路。

根据规划，近期将有13个重点项目支撑白云机场建设国际航空枢纽，这些项目涵盖上述四大功能。具体来说，交通功能的重点项目是T2航站区扩建工程(1,800亩)、机场三期扩建工程(四跑、五跑、T3航站区)，穗莞深、广佛环、广清城际项目，FBO商务航空服务基地。航空货运的重点项目包括联邦快递亚太转运中心、华南地区空陆联运集散中心、超级货站、顺丰速运项目。商业商务配套功能包括航空会展、航空会展商业商务配套、高增枢纽国际商务综合配套(1,200亩)、穗佳物流总部。机务维修制造功能包括新科宇航广州飞机维修基地。

白云机场现状的交通条件不尽如人意，规划对外部交通和内部交通系统都进行了优化。对外交通从三个方面做了优化——包括与珠三角的区域联系、与中心城区和三大战略枢纽间的交通联系以及临空经济示范区的交通体系，对内部交通联系来说，规划预期的效果是：通过摆渡巴士、城际、有轨电车等方式实现机场内部的快捷联系，航站楼之间的出行时间控制在10分钟内，各功能区之间则控制在15分钟内。具体讲通过5条轨道和4个枢纽实现机场内外的连接，5条轨道包括：1条城际线，穗莞深/广佛环城际(在建)，联系3个航站楼；3条地铁线，现状的3号线、规划的白云机场-广州北站连接线联系T1/T2/T3和高增枢纽；规划的18号线联系T2/T3/三大战略枢纽；此外还有1条规划的有轨电车线，联系T3/高增枢纽。

4个枢纽则包括现状的T1(城际、地铁、机场大巴、小汽车、出租车)、T2(城际、地铁、机场大巴、小汽车、出租车)、规划的T3(高铁、城际、地铁、机场大巴、小汽车、出租车)和高增枢纽(地铁、有轨电车、机场大巴、小汽车)。

交通站场也为四大枢纽进行了配套，具体来说，规划公共停车场9个(新增3个)、机场大巴客运站3个(新增1个)。

=== 高速公路 ===
[[广州机场高速公路|机场高速公路]]随着新广州白云国际机场启用时就陆续开通，由广州市区往新机场只需30分钟左右，现有三元里、新市等多个出口，而在已航航站楼落成后最后规划的連結廣州市白雲區黃石東路的空港大道已经於2014年底通車。

=== 地面综合交通中心（GTC） ===
广州白云国际机场内有一座总建筑面积达22.2万平方米地面综合交通中心（GTC）位于一号航站楼和二号航站楼之间，其设有人行连接通道与二号航站楼互通，是目前华南地区最大的综合交通枢纽。包含一个室内停车楼和两个露天停车场，具备市内大巴、城际大巴、地铁、城轨换乘等多种功能。主体为地上四层，地下两层。其中负一、负二层主要作为过夜车辆停放区，地上一层、二层主要作为接客车辆停车区，三层、四层主要作为送机车辆停车区。每一层停车区都用不同的颜色区分开。

==== 巴士与城市候机楼 ====
現時，機場的巴士服務可分為两類。分別為市区巴士、城际巴士、旅游巴士、职工巴士以及场区穿梭巴士。于二号航站楼，市区巴士和长途巴士的候车区、上下客区在GTC西侧设立有巴士车位18个。旅游巴士、职工巴士和场区穿梭巴士候车区、上下客区在GTC东侧设立有巴士车位16个；于一号航站楼的A1-A2、A3-A4、A5-A6、B11-B13号门均设有市区、长途大巴的候车区、上下客区。
;场区穿梭巴士
场区穿梭巴士对旅客开放的仅有一号航站楼与二号航站楼往返线，其候车区、上下客区设在一号航站楼北二楼出发厅的10号门和二号航站楼三楼出发厅的42号门。场区内的其他穿梭巴士（如客运航站楼到货运区的穿梭巴士、机场来往员工宿舍的穿梭巴士）则只对机场内的持证人员开放。
;市区巴士
市区巴士均由機場二号客運大樓GTC开出，在机场内途经一号航站楼A区和B区后經[[广州机场高速公路|机场高速公路]]直接前往广州市各區。所有市区巴士到达机场后都将先停靠二号航站楼GTC，停车下客后再继续前往一号航站楼。

乘搭的士前往白云机场是较为方便的选择，但费用较为昂贵。从广州市越秀、白云、荔湾较近机场的地区搭[[廣州的士|的士]]前往新机场需100至120元[[人民币|人民币]]左右。广州出租车起步价为12元，起租里程2.5公里，续租价为每2.6元/公里。出租车载客行程满35公里后，续租价为每3.9元/公里。搭乘出租车过程中，产生路桥费、高速公路路费的由乘客支付。

一号航站楼到达区的的士上客点于国际到达区的A8号门外，国内到达区的B3、B6和B8号门外；二号航站楼则于国际到达区的52号门外，国内到达区的50号门外，其中国际旅客出租车上客区设有出租车上客车位18个，国内旅客出租车上客区设有出租车上客车位20个，GTC内有供的士调度疏导的停车场。

==== 停车场 ====
一号航站楼与二号航站楼的停车收费为同一标准，露天停车场与停车楼都是前15分钟免费，24小时内进入的车辆限享受两次免费时段。
;白云机场在一号航站楼南、北两边设有4个停车场，1座停车楼，总面积达18万多平方米，车位4700多个，停车场（楼）实行24小时开放。

P1停车场为国际到达停车场，可停小汽车697辆。该场距A区到达最近，步行6分钟即可到达，场内设有到达航班电子显示屏和洗手间，开辟专门人行通道前往A区。P2停车场为出发、过夜停车场，可停小汽车1364辆。该停车场紧挨航站楼，场内设置A、B两个停车区。其中，A、B停车区分别标识了A1-A13及B1-B12车道编号，场内设有洗手间及便民服务点。P3停车楼为过夜停车场，可停小汽车1413辆，停车楼位于航站楼和白云机场铂尔曼酒店之间，共有三层。停车楼各层根据不同街、道设计标识了不同图案。其中，负一层为水果停车区；一层为生肖停车区；二层为建筑物停车区，旅客可根据停车区域图案及车位编号停车、找车。场内设有电梯、洗手间及便民服务点。P4停车场为国内到达停车场，可停小汽车602辆，大中巴士91辆。场内设置了A、B两个小车停车区及大中巴停车区。P4停车场东西两侧设有人行天桥及垂直升降电梯。出发旅客通过搭乘位于A、B到达区的直达电梯前往航站楼，即可办理登机手续，步行约15分钟。场内东西侧均设有到达航班电子显示屏。P5停车场为国内到达停车场，可停小汽车633辆，由该场前往B区到达接客步行约6分钟。

;而二号航站楼设置了2个停车场、1座停车楼，均位于二号航站楼的正南方向，从一楼GTC入口或者二层连接楼可依次到达P8停车楼和P6、P7停车场。

其中P6、P7为室外停车场，共有停车位1356个，P8为停车楼，约有停车位3771个，分为A、B两座停车楼。三个停车场共用3个入口和2个出口，2个入口位于出港高架上桥位的西侧，另外1个入口连接出港高架的引桥（可直达P8停车场四楼），出口位于出港高架下桥位的东侧。所有进出二号航站楼停车场的车辆，统一由东入口进场西出口出场。

== 爭議 ==
=== 機場廣播 ===
白雲機場的電腦語音廣播及熱線電話只提供[[普通話|普通話]]和[[英語|英語]]，不提供[[粵語|粵語]][[廣州話|廣州話]]服務，這也與廣州城市公共交通服務（[[廣州地鐵|地鐵]]、公共巴士及火車等）全部提供粵語服務的做法大相迳庭。機場地勤公司為增加[[本土化|本土化]]服務，曾在2005年11月22日起加入粵語廣播，但僅以人工廣播方式進行，<ref>{{cite news|url=http://news.sina.com.cn/s/2005-11-22/16027509787s.shtml|title=白云机场增设粤语广播 |publisher=羊城晚報 |date=2005-11-22|accessdate=2019-02-03}}</ref>可措施在實行一段時間後就突然被取消，至今未見恢復。

=== 高昂物價 ===
從新白雲機場開始營運起，其進駐商家提供的餐飲、百貨服務，因價錢均高於市區數倍而一直備受旅客詬病。機場管理方為此在2009年開始採取壓抑物價的措施，要求便利店、特產店等商店內設立「機場限價商品專區」。機場向商家提供了一份詳細的備選商品名錄，主要針對飲用水、飲料、餅乾、即食麵、八寶粥等商品。具體規定是：<ref>{{cite web |title = 广州白云机场曾经的诺言：设限价商品专区 |url = http://www.ycwb.com/ePaper/xkb/html/2012-11/29/content_24463.htm |publisher = 新快報 |date = 2012-11-29 |accessdate = 2014-07-16 |deadurl = yes |archiveurl = https://web.archive.org/web/20140726114734/http://www.ycwb.com/ePaper/xkb/html/2012-11/29/content_24463.htm |archivedate = 2014-07-26 }}</ref>
# 商家必須從中至少選擇兩種礦泉水，單價不能超過3元人民幣；
# 選擇三種飲料，單價不能超過5元；
# 選擇兩種餅乾，單價不能超過5元；
# 即食麵和八寶粥必須選三款，單價不能超過6元。

除此以外，還要求屬於限價範圍的餐飲品種必須要在營業時間內均有供應，令旅客在機場大部分食肆吃飯只需二、三十元即可購買。

2010年，機場候機樓公司相關負責人表示，[[麥當勞|麥當勞]]即將進駐，屆時旅客用25元至30元的價格就可以買到一個豐盛的套餐。並稱麥當勞白雲機場分店的全部食品價格與市區相同，將真正做到「同城同價」。此後[[肯德基|肯德基]]、[[仙踪林|仙踪林]]、[[真功夫|真功夫]]等也進駐機場。

2012年，《[[新快報|新快報]]》記者實地調查發現，機場內八成便利店和特產店等零售商舖均未設限價專櫃。即使是有提供專櫃的店鋪，限價貨架均擺在隱蔽位置，「機場限價專區」的標籤被價目牌遮擋，供應中的商品亦並未如規定中齊全。而真功夫、肯德基食物的售價同樣高於市區，後者在當時記者的對比下貴了31%。<ref>{{cite web |title=机场“限价令”成一纸空文80%零售店未依规设限价专柜 |url=http://www.ycwb.com/ePaper/xkb/html/2012-11/29/content_24465.htm |publisher=新快報 |date=2012-11-29 |accessdate=2014-07-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20140726102708/http://www.ycwb.com/ePaper/xkb/html/2012-11/29/content_24465.htm |archivedate=2014-07-26 }}</ref>廣州市區15元的[[蘭州拉麵|蘭州拉麵]]，在機場以套餐形式販售，售價78元，比市價貴5倍。<ref>{{cite web |title = 餐饮又贵又难吃，不是太饿就忍着 |url = http://www.ycwb.com/ePaper/xkb/html/2012-11/29/content_24462.htm |publisher = 新快報 |date = 2012-11-29 |accessdate = 2014-07-16 |deadurl = yes |archiveurl = https://web.archive.org/web/20140726103654/http://www.ycwb.com/ePaper/xkb/html/2012-11/29/content_24462.htm |archivedate = 2014-07-26 }}</ref>

據報，[[臺北松山機場|台北松山]]/[[台灣桃園國際機場|桃園機場]]、[[新加坡樟宜機場|新加坡樟宜機場]]的商家均提供同城同價的保證。<ref>{{cite web |title = 国际机场比较：中国香港、台北和新加坡机场均保证同城同价 |url = http://www.ycwb.com/ePaper/xkb/html/2012-11/29/content_24475.htm |publisher = 新快報 |date = 2012-11-29 |accessdate = 2014-07-16 |deadurl = yes |archiveurl = https://web.archive.org/web/20140726083057/http://www.ycwb.com/ePaper/xkb/html/2012-11/29/content_24475.htm |archivedate = 2014-07-26 }}</ref>實際上，白雲機場店鋪的租金與天河區黃金地段[[正佳廣場|正佳廣場]]内一般位置的租金水平相若。<ref>{{cite web |title = 高租金造就高物价吗？与广州旺地比较：白云机场铺租低过天河城旺 |url = http://www.ycwb.com/ePaper/xkb/html/2012-11/29/content_24473.htm |publisher = 新快報 |date = 2012-11-29 |accessdate = 2014-07-16 |deadurl = yes |archiveurl = https://web.archive.org/web/20140726094548/http://www.ycwb.com/ePaper/xkb/html/2012-11/29/content_24473.htm |archivedate = 2014-07-26 }}</ref>
