'''广州法租界'''，近代中国4个[[在华法租界|在华法租界]]之一（另外3个是[[上海法租界|上海法租界]]、[[天津法租界|天津法租界]]和[[汉口法租界|汉口法租界]]）。同时也是[[广州|广州]]的2个租界之一，与[[广州英租界|广州英租界]]同在[[沙面岛|沙面岛]]上。今为广州市[[荔湾区|荔湾区]][[沙面街道|沙面街道]]。
== 历史 ==
[[File:Shameen_Canton.jpg|thumb]]
1861年开辟。[[第二次鸦片战争|第二次鸦片战争]]后，英法两国占领广州，选定了邻近传统的外贸区[[廣州十三行|廣州十三行]]（当时已被烧毁）西南面，[[珠江|珠江]][[白鹅潭|白鹅潭]]边上的小沙洲作为租借地，人工填筑成[[沙面岛|沙面岛]]，其中法租界占有东部的1/5，面积66亩。东部有一桥与华界相通。

1943年2月23日，[[维希法国|法国维希政府]]宣布，同意放弃在华租界。6月5日，[[汪精卫政权|汪精卫政权]]同日收回[[天津法租界|天津法租界]]、[[汉口法租界|汉口法租界]]和广州法租界。[[第二次世界大战|二战]]结束后，新成立的法国[[戴高乐|戴高乐]]政府亦宣布将沙面法租界交还[[中华民国|中华民国]]。

== 居民 ==  
广州法租界仅供[[法国|法国]]来华做外贸生意的[[洋人|洋人]]居住，不允许[[华人|华人]]入居。

== 管理 ==
===法国领事馆===
1888年，法租界当局开始建筑领事馆；1890年，将领事馆由[[广州|广州]]城内迁入[[沙面岛|沙面岛]]。
== 市政 ==
广州法租界开辟之初，由于当时[[法国|法国]]政府正集中力量在广州兴建[[石室圣心大教堂|石室圣心大教堂]]，因此广州法租界经营起步较晚。至1889年10月，[[法国|法国]]政府将沙面法租界内空余土地进行拍卖，并开始大兴土木进行市政建设，除兴建洋楼馆舍外，还设有公园、医院、教堂等公共设施。到19世纪末，已形成井然有序的独立街区，俨然一座独立于广州城之外的法国小城。
===金融===
1888年，法国人在沙面法租界建筑[[东方汇理银行|东方汇理银行]]广州分行。
===宗教===
[[File:Our_Lady_of_Lourdes_Chapel.jpg|thumb]]
天主教[[沙面露德圣母堂|沙面露德圣母堂]]建于1889年，奉露德圣母为该堂主保。
== 现状 ==
广州法租界所在区域，包括当年的西式建筑以及绿化带，大致保存完整。1996年，被定为[[全国重点文物保护单位|全国重点文物保护单位]]。
