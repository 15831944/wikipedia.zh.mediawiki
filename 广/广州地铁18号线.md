'''广州地铁18号线'''是[[广州地铁|广州地铁]]的建设中路线之一，連接位於[[天河區|天河區]]的[[廣州東站_(地鐵)|廣州東站]]和[[南沙區|南沙區]]的萬頃沙。代表色暫定為粉紫色。線路定位[[通勤鐵路|市域快線]]，連接南沙新區內的萬頃沙樞紐和明珠灣起步區，番禺區中心城區和萬博商務區，以及廣州市區內的琶洲西區電商區，珠江新城以及廣州東站。現時屬於廣州市新一輪軌道交通（2016-2022）建設規劃方案的線路之一，已於2017年3月下旬獲國家發改委批覆立項。

== 概要 ==
18號線大致呈南北走向，起始於南沙區的萬頃沙樞紐，大致沿西北方向行走，經鳳凰大道、番禺大道南前往番禺廣場。然後繼續沿番禺大道北行走，沿番禺大橋下穿洛浦島和珠江後航道。然後沿華南快速經過海珠濕地公園東側後大致轉向正北行走，經過琶洲西區之後轉向西，於珠江下方經海心沙島東端轉入冼村路，再沿體育東路行進。隨後下穿天河體育中心，並抵達終點站廣州東站。番禺廣場以南至萬頃沙一段初、近期與[[廣州地鐵22號線|22號線]]共線運營。線路亦預留延伸條件，計劃未來往北延伸至[[廣州白雲國際機場|白雲機場]]T3航站樓及[[清遠市|清遠市]]；以及往南延長進入[[中山市|中山市]]。

18號線全長約62.7公里，均為地下線，設9座車站，其中换乘站8座；平均站间距7.8km，最大站间距26.0km，为横沥至番禺广场站区间；最小站间距2.3km，为石榴岗至琶洲西区区间。

本綫將分別設置一個與[[廣州地鐵22號線|22號線]]共用車輛段和停車場，車輛段位於[[南沙區|南沙區]]蕉门水道以西，X298县道以东、南沙七涌以南，八涌以北；停車場位於[[番禺區|番禺區]]番禺大道北以東、市南路以北、草龍沙河二街以西。本線將設置萬頃沙主變電所和擴建十一號線赤沙滘主變電站對十八號線供電。本線和[[廣州地鐵22號線|22號線]]控制中心將設於陇枕停车场內。

== 歷史 ==
===規劃===
起初18号线起点为[[广州南站_(地铁)|广州南站]]，往東途經[[祈福新邨|祈福新邨]]再大致轉向南，經過番禺廣場後進入南沙區。途經欖核、大崗、橫瀝三鎮並設站，最終抵達明珠灣起步區南部的萬頃沙。亦有計劃從廣州南站北延，前往荔灣區芳村的[[石圍塘站|石圍塘]]，按照快線設計，設站較少，以滿足南沙區快速來往廣州南站及番禺中心城區的功能，且計劃換乘2、3號線，進一步進入廣州市中心城區。

后来，與原规划22号线交換編號，18號線改為承擔快速連接廣州市區至南沙，以及分流部分3號線客流的作用。當時規劃市區段行經珠江新城和金融城之間的馬場一帶，後來改為行走冼村路，深入珠江新城腹地，亦以求更有力分流3號線的客流壓力。南段亦從行經欖核、大崗兩鎮改為沿番禺大道南行走，18號線走向基本定型。

18號線和22號線原本均属于远期规划线路，但由於需要配合南沙自貿區發展，已決定將優先建設。18號線將按照快線設計，以實現南沙区與广州中心城区的快速轨道交通联系，“从南沙到东站要确保30分钟内到达”。<ref>{{Cite news|url =http://news.dayoo.com/guangzhou/201512/22/139995_45684383.htm|title =壕气！投三千来亿，广州未来15年新建15条地铁线|last =|first =|date =2015-12-22|work =|accessdate =|newspaper =[[广州日报|广州日报]]|deadurl =yes|archiveurl =https://web.archive.org/web/20151223155548/http://news.dayoo.com/guangzhou/201512/22/139995_45684383.htm|archivedate =2015-12-23}}</ref><ref>{{Cite news|url = http://gz.southcn.com/content/2015-12/22/content_139383442.htm|title = 广州新一轮地铁规划将杀到 15条线路详解|last = |first = |date = 2015-12-22|work = |accessdate = |newspaper = [[南方日报|南方日报]]}}</ref>

2017年4月，18號線進行第二次環評公示<ref name=EIAII>[http://www.jsti.com/index.php/show/index/771 广州市城市轨道交通十八号线工程 环境影响评价第二次公示]</ref>，其後坊間有不同的聲音分別要求增加體育東路站（即[[體育中心站_(廣州)|體育中心站]]）<ref>{{Cite news|url =http://news.dayoo.com/guangzhou/201704/19/139995_51135061.htm|title =广州地铁18号线能加个体育东站吗？|last =|first =|date =2017-04-19|work =|accessdate =|newspaper =[[广州日报|广州日报]]|deadurl =yes|archiveurl =https://web.archive.org/web/20170519001544/http://news.dayoo.com/guangzhou/201704/19/139995_51135061.htm|archivedate =2017年5月19日|df =}}</ref>和魚窩頭站<ref>{{Cite news|url = http://news.ycwb.com/2017-04/20/content_24681492.htm|title =榄核大岗诉求如何实现？鱼窝头加个站可缓解|last = |first = |date = 2017-04-20|work = |accessdate = |newspaper = [[羊城晚報|羊城晚報]]}}</ref>等車站，廣州市發改委更回應媒體稱會研究是否增加體育東路站<ref>{{Cite news|url =http://news.dayoo.com/guangzhou/201704/20/139995_51140373.htm?from=timeline&isappinstalled=0|title =广州地铁18号线能加个体育东站吗？|last =|first =|date =2017-04-20|work =|accessdate =|newspaper =[[广州日报|广州日报]]}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。但5月10日廣州市國土規劃委員會在新聞發佈會回應稱，為滿足南沙副中心到廣州市區30分鐘快速直達的時空目標，需要嚴格控制沿線車站數量，且作為時速高達160公里的快線，車站不可能設得太密<ref>{{Cite news|url =http://news.dayoo.com/guangzhou/201705/10/139995_51222965.htm|title =地铁18号线还能否增加站点？广州国规委：不可能|last =|first =|date =2017-05-10|work =|accessdate =|newspaper =[[广州日报|广州日报]]}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。因而18號線維持現時設置9站的方案，未能再增設新車站。

== 车站 ==
18號線計劃設9座車站，除沙溪站外的其餘8站均為換乘站。
===換乘站===
* '''[[广州东站_(地铁)|广州东站]]'''：换乘[[广州地铁1号线|1号线]]、[[广州地铁3号线|3号线]]及[[广州地铁11号线|11号线]]。因1號線和3號線建设时均未作任何预留，预计为通道换乘，11號線建設時則會預留與18號線站台節點換乘和站廳換乘。
* '''[[冼村站|冼村站]]'''：換乘[[廣州地鐵13號线|13號线]]。18號線的車站位於冼村路，與位於黃埔大道南側的13號線車站呈L型，預計為站台節點換乘和站廳換乘。
* '''[[琶洲西區站|琶洲西區站]]'''：換乘[[广州地铁19号线|19号线]]、[[广州地铁28号线|28号线]]，以及前往[[磨碟沙站|磨碟沙站]]换乘[[广州地铁8号线|8号线]]。19號線及28号线為遠期線路，預留情況未知。8號線建造時未作預留，但琶洲西區站距磨碟沙站只有200多米，預計將設有通道連通兩個車站實現換乘。
* '''[[石榴崗站|石榴崗站]]'''：换乘[[广州地铁11号线|11号线]]。預留情況未知。
* '''[[南村萬博站|南村萬博站]]'''：换乘[[广州地铁7号线|7号线]]及[[佛山地铁4号线|佛山地铁4号线]]。7號線建造时未作预留，预计需经通道换乘；佛山地鐵4號線為遠期線路，預留情況未知。
* '''[[番禺廣場站|番禺廣場站]]'''：换乘[[广州地铁3号线|3号线]]、[[广州地铁17号线|17号线]]及[[广州地铁22号线|22号线]]。因3號線建造时未作预留，预计需经通道换乘；17號線為遠期線路，預留情況未知。
* '''[[橫瀝站_(地鐵)|橫瀝站]]'''：换乘[[广州地铁15号线|15号线]]及[[广州地铁22号线|22号线]]。18號線和22號線近期共用本站月台。15號線為遠期線路，預留情況未知。
* '''[[萬頃沙站|萬頃沙站]]'''：换乘[[广州地铁15号线|15号线]]及[[广州地铁22号线|22号线]]。18號線和22號線近期共用本站月台。15號線為遠期線路，預留情況未知。
