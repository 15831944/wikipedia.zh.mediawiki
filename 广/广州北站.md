'''广州北站'''位于[[中华人民共和国|中国]][[广东省|广东省]][[广州市|广州市]][[花都区|花都区]][[新华街道_(广州市)|新华街道]]站前路，于1908年启用，为[[中国铁路广州局集团有限公司|中国铁路广州局集团有限公司]]广深铁路股份有限公司直属的客货运[[二等站_(中鐵)|二等站]]。经过铁路有[[京广铁路|京广铁路]]和[[京广高速铁路|京广高速铁路]]。

[[Category:中国大陆高普铁共站车站|Category:中国大陆高普铁共站车站]]

== 历史 ==
[[File:粵漢線新街.jpg|thumb]]
*1908年，广州北站建成启用。建成时名为'''新街站'''。
*1995年，改名为'''花都站'''。
*1999年9月15日，花都站改名为广州北站，原位于[[白云区_(广州市)|白云区]][[江高镇|江高镇]]的广州北站更名为[[江村站|江村站]]。
*2009年，广州北站进行改造，并在年底开始成为[[武广客运专线|武广客运专线]]临时始发终到站。

== 车站设施 ==
[[File:Guangzhou_North_Railway_Station_Gate_3.jpg|thumb]]
[[File:GUANGZHOU_NORTH_RAILWAY_STATION_1.JPG|thumb]]
[[File:Guangzhou_North_Railway_Station_Platform_3.JPG|thumb]]
广州北站京广铁路部分建有基本站台1座、中间站台1座，京广客运专线部分建有供和谐号停靠的中间站台2座，京广线共有4条到发线2条正线，武广客运专线2条到發线2条正线。另有京广客运专线与京广铁路联络线接驳京广铁路郭塘站、棠溪站及广州站。屆時，当棠溪站、广州站及广州东站的改造工程完成後，行驶於京广高速铁路的高铁列车可以通過京广铁路联络线直通到相关改造后這三座车站，甚至可以直通到东莞、深圳或[[香港西九龍站|香港西九龍]]。

== 使用情况 ==
广州北站是[[京广铁路|京广铁路]]的[[中间站|中间站]]，有K9，K10，K641，K9016等部分快速列车（以管内列车居多）停靠此站。并有[[T8371/T8372|T8371/T8372]]次特快列车停靠。2009年12月20日武广客运专线开通后，有若干和谐号动车组停靠。预计在2016年广州站的改扩建计划中，广州北站将与广州南站、广州东站和[[佛山站|佛山站]]分流广州站的多数客运列车。<ref>{{cite news|url=http://www.ycwb.com/ePaper/ycwb/html/2014-04/14/content_420442.htm?div=-1|title=广州火车站不惑拆迁|publisher=[[羊城晚报|羊城晚报]]|date=2014-04-14|accessdate=2015-02-03|deadurl=yes|archiveurl=https://web.archive.org/web/20140415054902/http://www.ycwb.com/ePaper/ycwb/html/2014-04/14/content_420442.htm?div=-1|archivedate=2014-04-15}}</ref>

== 始发车次 ==
目前广州北站未有任何普通列车始发车次。

2009年12月26日至2010年1月30日，广州北站曾临时作为武广客运专线南端的始发终到站。

== 接驳交通 ==
=== 地铁 ===
[[广州地铁9号线|广州地铁9号线]]已经于2017年12月28日开通，在国铁站西侧设[[广州北站_(地铁)|广州北站]]，但受[[广清城轨|广清城轨]]建设影响，目前该地铁站无法直达国铁广州北站。因此乘客需要在[[花城路站|花城路站]]下车，沿秀全大道徒步前往，或乘坐接驳专线巴士前往国铁广州北站。

{{广州地铁九号线各站|nocat}}

=== 公共汽車 ===
花都区内多条公交路线途径广州北站附近，本站临近花都客运站和站前路公交总站，有多条公交路线到达花都区各镇及广州市区，详见[[花都巴士线路表|花都巴士线路表]]、

花都客运站：位于秀全大道，离广州北站仅200米，主要负责花都区部分镇、广州市区主要客运站以及从化的公交及客运班车线路

站前路公交总站：位于站前路，广州北站以北云山桥底，线路主要发往花都区部分镇、广州神山以及广州中山八路公交总站（705路）

火车站公交站：有若干花都市区公交线路途经和始发（如1路、9路等）

=== 城际客运 ===

广州北站汽车客运站

另外，花都区长途汽车客运站（主要负责发往广东省内外长途班车线路）虽临近广州北站，但其位置位于铁路以西，需换乘公交及出租绕道前往。

== 改造 ==
2009年10月10日，广州北站开始进行为期2个月的施工改造，工程主要为新建中间天桥接驳站台和站楼内部改造等，施工期间广州北站暂停客运服务。同年12月20日，广州北站改造完工。室内共设14个售票窗口（北侧普速类车售票窗口6个和南侧武广高铁列车售票窗口8个，同时还配有4台武广高铁车票自动售票机）和4个候车室（1楼1、2候车室为普速列车候车室；3楼3、4候车室为武广高铁列车候车室）。

2015年，广州北站再次进行改造，届时该站将成为城际轨道枢纽站、广州北部交通中心、“空铁联运”重要枢纽点。同时对广州北站汽车客运站升级改造，建设为国家一级A类客运站<ref name=2018年启动>{{Cite web |url=http://society.people.com.cn/GB/n1/2016/0921/c1008-28728684.html |title=广州火车站改造  有望2018年启动 |accessdate=2016-09-21 |publisher=广州日报 |date= 2016-09-21}}</ref>。
