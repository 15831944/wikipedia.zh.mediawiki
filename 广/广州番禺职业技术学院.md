'''广州番禺职业技术学院'''（[[英文|英文]]：Guangzhou Panyu Polytechnic，PYP，简称：番职院）是全国首批、[[广州市属|广州市属]]第一所公办全日制普通高等职业院校，全国首批28所“[[国家示范性高等职业院校建设计划|国家示范性高等职业院校建设计划]]”院校之一，广东省示范性高等职业院校立项建设单位。实行[[广州市|广州市]]和[[番禺区|番禺区]]共建共管的体制。校区位于[[中国|中国]][[广州市|广州市]][[番禺区|番禺区]][[沙湾镇|沙湾镇]]青山湖。开设有42个紧贴[[珠三角|珠三角]]地区产业结构和社会发展需求的专业，以“一技之长+综合素质”为培养高技能人才的目标。

== 概况 ==
番职院原名为'''番禺理工学院'''，于1993年筹建。学院占地2061[[亩|亩]]，校舍建筑面积26.76万平方米，教学仪器设备总值（单价800元以上）9346余万元，图书馆面积2.9万平方米，馆藏纸质图书87.39万册、电子图书462175GB、纸质专业期刊1205种、电子专业期刊5128种，建有148个校内专业实训室和389个校外实训基地。
校园依山傍水、环境清幽，毗邻[[番禺滴水岩森林公园|番禺滴水岩森林公园]]，现有[[全日制|全日制]]在校生10000余人。其中约有60%为[[广州|广州]]本地生源。

== 办学历史 ==
*1993年[[番禺|番禺]]政府为满足经济迅速发展对人才的需要，决定在番禺筹建一所以工科为主，文理兼容的专科学校，同年9月获广东省人民政府正式批准筹建，校区选址原[[沙湾镇|沙湾镇]]青山湖农场。
*[[1993～1994年|1993～1994年]]学院挂靠[[广州大学|广州大学]]，招收中英文秘书、会计、酒店管理和计算机应用4个专业200人，借用[[广东番禺中学|广东番禺中学]]上课。
*1995年获[[广东省|广东省]]人民政府批准设立，学院首期工程竣工，9月在新校址招生、授课。
*随着国家大力发展[[职业教育|职业教育]]以及[[广州市|广州市]][[番禺区|番禺区]]经济建设和社会发展对[[高技能人才|高技能人才]]的需求，1997年9月[[中华人民共和国教育部|教育部]]正式批准备案，改名为番禺职业技术学院。
*2003年被[[广州市|广州市]]政府确定为市属高等职业教育龙头院校。
*2005年4月通过[[中华人民共和国教育部|教育部]]人才培养工作水平评估。
*2006年12月被教育部、[[中华人民共和国财政部|财政部]]确定为“[[国家示范性高等职业院校建设计划|国家示范性高等职业院校建设计划]]”首批28所立项建设院校之一。并被直接列为2008年度广东省示范性高等职业院校立项建设单位。
*为了体现[[广州市|广州市]]和[[番禺区|番禺区]]共建体制，提高学院服务广州经济和社会发展的能力，促进学院更好更快发展，2008年6月获[[中华人民共和国教育部|教育部]]批准备案，更名为广州番禺职业技术学院。

== 学以致用 ==
校训“[[学以致用|学以致用]]”是学校[[办学传统|办学传统]]与[[办学目标|办学目标]]的凝练和概括，同时也是对全体师生提出的期望和要求。在继续思考与探索下，提出了[[高等职业教育|高等职业教育]]办学应有的主要特征，即“服务性、职业性、开放式、大众化”。学校以“一技之长+综合素质”为人才培养目标的[[核心理念|核心理念]]。

== 毕业生就业 ==
自2001年[[广东省教育厅|广东省教育厅]]首次面向社会公布高校毕业生就业率以来，毕业生就业率一直稳居全省高校前列。
*2001年就业率位居全省高职高专类院校第二，达98.21%
*2002年就业率位居全省高职高专类院校第一，达94.17%
*2003年就业率位居全省高职高专类院校第一，达99.14%
*2004年就业率位居全省所有本专科院校第一，达99.73%
*2005年就业率位居全省所有本专科院校第一，达99.81%
*2006年就业率位居全省所有本专科院校第二，达99.84%
*2007年就业率位居全省所有本专科院校第二，达99.85%
*2008年就业率位居全省所有本专科院校第一，达99.88%
