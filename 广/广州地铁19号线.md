'''广州地铁19号线'''（{{lang-en|'''Line 19'''}}）是[[广州地铁|广州地铁]]的规划中路线之一，大致呈西南-東北走向。

== 概要 ==
19号线属于远期规划线路，目前尚未有可靠资料能提供它的车站列表。但从其他线路的规划中可知，该线大致走向为自西南向东北，同时规划中的[[广州地铁18号线|18号线]][[琶洲西區站|琶洲西區站]]、[[广州地铁21号线|21号线]][[大观南路站|大觀南路站]]亦会预留与该线的换乘。<ref name="L8北环评参与">[http://www.scies.com.cn/UploadPic/Files_2012326174755292.pdf 广州市轨道交通八号线北延段（文化公园～白云湖）工程环境影响评价公众参与] {{Wayback|url=http://www.scies.com.cn/UploadPic/Files_2012326174755292.pdf |date=20160305083806 }}</ref><ref name="L14环评报告">{{Cite web |url=http://www.scies.org/UploadPic/Files_2012813174822832.pdf |title=广州市轨道交通十四号线一期及知识城支线工程 环境影响报告书（简本） |access-date=2012-10-06 |archive-url=https://web.archive.org/web/20140808045127/http://www.scies.org/UploadPic/Files_2012813174822832.pdf |archive-date=2014-08-08 |dead-url=yes }}</ref><ref name="十二五设计招标">[http://www.gzzb.gd.cn/fileStorage/jyzx/bbfj/2012/7/xmbaqgfj/bb_2012-2884_huangzf_1341990332332.doc 广州市十二五轨道交通线网前期深化研究（设计、设计咨询）招标文件]</ref>

== 歷史 ==
按照2010年的旧规划，19号线从沙头至莲花山，中途先后与[[广州地铁3号线|3号线]]、[[广州地铁4号线|4号线]]及当时规划的[[广州地铁17号线|17号线]]、[[广州地铁15号线|18号线]]（后改为[[广州地铁15号线|广州地铁15号线]]）交汇换乘<ref>{{cite news|title=广州地铁规划2020落实方案|url=http://jt.gz.bendibao.com/news/20101029/73002.shtml|date=2010年10月29日|publisher=本地宝交通}}</ref>。

而后在其他线路的规划公示中这条线路已经不见了踪影，但2011年后又能隐约看到调整后的广州地铁19号线，走线已与原规划有所差别。當時規劃19號線起始於荔灣區穗鹽東路環城高速交叉處，沿穗鹽路、花蕾路前進，轉入明心路。隨後過江進入海珠區，沿同福路、仲愷路、濱江東路延伸，之後在[[广州大桥_(广州)|廣州大橋]]引橋西側大致沿蕙蘭路、友和路往東，進入藝洲路、雙塔路。隨後在[[珠江啤酒廠|珠江啤酒廠]]東側轉向北，過江進入天河區。沿馬場路、潭村路向東北偏北延伸，掠過[[暨南大學|暨南大學]]和[[華南師範大學|華南師範大學]]东邊，大致沿[[華南快速幹線|華南快速幹線]]繼續往東北偏東延伸。隨後進入匯景北路，大致向東行進，掠過[[匯景新城|匯景新城]]、[[华南农业大学|華南農業大學]]和[[广氮花园|廣氮花園]]。下穿大觀南路後大致沿奧體路、悅景路、科林路繼續往東延伸，最終到達科豐路口。

此外，按照之前的线路走向，19号线远期可能会在[[植物园站|植物园站]]接上预留了拆解条件并已按6B编组建设的[[广州地铁6号线|6号线]]二期，但从后来其它线路的规划图中得知，19号线又调整为继续向東至世界大觀与[[广州地铁21号线|21号线]]交汇。<ref name="十二五设计招标" />

2018年，19號線更改走線，海珠區讓出線位予28號線。因此本線經過琶洲西區站後繼續向南延伸，經江海大道，最後到達西三站，遠期延伸至佛山三山新城。<ref>[https://news.sina.cn/2018-09-15/detail-ihkahyhx3160654.d.html?from=wap 广州8条新地铁重磅曝光！站点逐一列明，以后没那么挤了！]大洋网-广州日报</ref>

== 车站 ==
根据规划图目测推断，該線會有以下的換乘站：<ref name="L8北环评参与" /><ref name="L14环评报告" />

* '''[[西三站|西三站]]'''：换乘[[广州地铁22号线|22号线]]。22號線初期為預留車站，预留情况不详。
* '''[[南浦站_(广州)|南浦站]]'''：换乘[[广州地铁2号线|2号线]]。因建造時未作任何預留，預計需經通道換乘。
* '''[[厦滘站|廈滘站]]'''：换乘[[广州地铁3号线|3号线]]。因建造時未作任何預留，預計需經通道換乘。
* '''[[赤崗站|赤崗站]]'''：换乘[[广州地铁8号线|8号线]]和[[广州地铁12号线|12号线]]。12號線仍在設計階段，預留情況未知；8號線建造時未作任何預留，預計需經通道換乘。
* '''[[琶洲西區站|琶洲西區站]]'''：换乘[[广州地铁18号线|18号线]]和[[广州地铁28号线|28号线]]，以及前往[[磨碟沙站|磨碟沙站]]换乘[[广州地铁8号线|8号线]]。18號線仍在設計階段，預留情況未知；8號線建造時未作預留，但本站距磨碟沙站只有200多米，預計將設有通道連通本站18號線和磨碟沙站實現換乘。
* '''[[潭村站|潭村站]]'''：换乘[[广州地铁5号线|5号线]]。两线十字形交汇，5号线建设时未作任何预留，预计需通过站厅通道换乘。
* '''[[馬場站_(廣州)|馬場站]]'''：换乘[[广州地铁13号线|13号线]]。13号线二期仍在設計階段，预留情况未知。
* '''[[華景路站|華景路站]]'''：换乘[[广州地铁11号线|11号线]]。11号线設計時未作出預留，換乘方式未知。
* '''[[大观南路站|大观南路站]]'''：换乘[[广州地铁21号线|21号线]]<ref name="十二五设计招标" />。两线十字交汇，但21号线仅作通道换乘考虑<ref>[http://www.gzzb.gd.cn/cms/wz/view/other/jgy.jsp?infoId=213766&channelId=15&siteId=1 广州市轨道交通二十一号线工程【施工5标】土建工程招标]</ref>。
* '''[[姬堂站|姬堂站]]'''：换乘[[广州地铁7号线|7号线]]和[[广州地铁28号线|28号线]]。7号线二期仍在設計階段，预留情况不详。
