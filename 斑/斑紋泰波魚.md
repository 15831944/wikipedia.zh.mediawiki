'''斑紋泰波魚'''（[[学名|学名]]：'''''Boraras maculatus'''''）為[[輻鰭魚綱|輻鰭魚綱]][[鯉形目|鯉形目]][[鯉科|鯉科]]的其中一個[[種|種]]。

== 分布 ==
本魚分布於[[泰國|泰國]]、[[馬來西亞|馬來西亞]]、[[新加坡|新加坡]]及[[印尼|印尼]][[蘇門答臘|蘇門答臘]]的溪流。

== 特徵 ==
本魚魚體外型不勻稱，尾柄特別細長。側腹上的黑斑與[[大點波魚|大點波魚]]的斑紋相似。體成橙色，魚鰭呈淺紅色，背鰭前面幾根鰭條為黑色和粉紅色，雄魚的側腹較平。體長約2.5公分，為世界最小的[[鲤科|鲤科]]魚種之一。

== 生態 ==
本魚棲息於森林溪流。肉食性，以[[蠕蟲|蠕蟲]]與小型甲殼動物為食。

== 經濟利用 ==
為觀賞性魚類，可和其他種類混養。於水族市場俗稱為小丑燈等。

==面臨威脅==
[[IUCN紅色名錄|IUCN紅色名錄]]指本物種分佈廣且缺乏任何已知的主要廣泛威脅而將之到為[[無危物種|無危物種]]。[[世界魚類數據庫|世界魚類數據庫]]則指本物種面臨[[熱帶雨林|熱帶雨林]]開發、[[污染|污染]]和[[水族貿易|水族貿易]]引起的非法採捕的威脅，於[[新加坡|新加坡]]的族群被認為為[[極危|極危]]，而於[[馬來西亞|馬來西亞]]亦已罕見或有可能已[[滅絕|滅絕]]。
