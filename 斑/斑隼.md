'''斑隼'''（学名：''Falco moluccensis''）在[[生物分类学|生物分类学]]上是[[隼|隼]]形目隼科隼属中的一个种，属于小型[[猛禽|猛禽]]。

== 分布和棲息地 ==
主要遍布[[澳大利亞|澳大利亞]]、[[東洋界|東洋界]]、以及大部分[[華萊士區|華萊士區]]，斑隼棲息於草原上的散生樹林，樹輕而繁茂適合牠棲息，主要活動範圍為森林附近的草原。牠偶爾會沿著伐木道路穿透森林，有時棲息在森林地區空地。在人的居住區域有時可以看見牠的蹤影。<ref>[http://www.globalraptors.org/grin/SpeciesResults.asp?specID=8243 Global Raptor Information Network - Moluccan Kestrel]</ref>

== 習性 ==
=== 掠食 ===
斑隼主要以小的[[哺乳動物|哺乳動物]]、[[鳥|鳥]]、[[蜥蜴|蜥蜴]]和[[昆蟲|昆蟲]]為主食。
