== 玊與玉 ==
玊隨著時間變化，古代所認為的部分瑕疵已經於現在被認為是一種玉的特點。

於古時，玊的種類可分為：

* 飄花；
* 雜色（又稱髒；包含「皮色」）；
* 水線；
* 綹裂；
* 黑點；
* 漿；
* 棉（或稱「玉花」；其中包含「雪花棉」）。

其中，雜色、皮色、雪花棉與飄花已在現代被重新界定，不再定為瑕疵<ref>{{Cite web|url=http://wemedia.ifeng.com/89547853/wemedia.shtml|title=这些翡翠有“瑕疵”|accessdate=2018-12-31|author=|date=|work=wemedia.ifeng.com|publisher=凤凰新媒体}}</ref>。

== 軼事 ==
2018年5月11日，[[澳門衛生局|澳門衛生局]]在會議廳召開的新聞發佈會時，[[仁伯爵綜合醫院|仁伯爵綜合醫院]]求診者處處長於水牌上的名字為「梁玊萍」，但是水牌上對應的[[粵拼|粵語譯音]]寫著「{{Lang|en|IOK}}」（粵拼的「玉」發音）。求證於澳門衛生局聯係人後，對方承認是水牌上的中文字打錯了，應該就是粵語譯音所對應的“玉”字。<ref>{{Cite web|url=http://www.mcfocus.com.mo/news.jsp?jid=189&lid=966&nid=4354|title=到底是“玊”還是“玉”？|accessdate=2018-12-31|author=梅小語|date=2018-05-11|work=www.mcfocus.com.mo|publisher=澳門焦點報}}</ref>
