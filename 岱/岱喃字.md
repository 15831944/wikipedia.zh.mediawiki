'''岱喃字'''（{{lang-vi|Tày Nôm}}）是[[越南|越南]][[岱依族|岱依族]]和[[儂族|儂族]]曾經使用的一種文字。其接近越南主體民族[[越族|越族]]所用的[[喃字|喃字]]，同是基於[[漢字|漢字]]及其造字原理。

== 简史 ==
岱族是具有民數最多、大多數分佈在越南北部的上游和中游地區的越南少數民族，集中在[[高平省|高平]]、[[涼山省|涼山]]、[[宣光省|宣光]]、[[河江省|河江]]等北部各省。

據一部分學者考證，岱喃字可能於[[越南南北朝|越南南北朝]]時期的[[莫朝|莫朝]]時產生。由於當時控制越南北方[[高平|高平]]地區的莫朝一直重視儒學，當地的其他民族的文化水平也得到提高。在此情況下兩位司[[天管樂|天管樂]]的岱依人'''[[儂瓊文|儂瓊文]]'''和'''[[閉文鳳|閉文鳳]]'''創造了岱喃字。此二人後被譽為岱喃字之父。<ref>何文書、呂文盧，《岱儂文化》，河內文化出版社，1981年</ref>至今尚流傳的《論紅顏四季》皇朝龍德元年抄本上，還明記他們倆的姓名和職稱。

== 示例 ==
* <span style="font-size:125%;">罷七</span>（合字）：七也。從七，罷聲。
* <span style="font-size:150%;">𠃩</span>：九也。從九，㐱聲。[[喃字|喃字]]讀音''chín''。
* <span style="font-size:125%;">邁十</span>（合字）：十也。從十，邁聲。
* <span style="font-size:150%;">𤾓</span>：百也。從百，林聲。今音ㄌㄧㄢˊ/lián或ㄌㄧㄢˇ/liàn，國語字讀音''chăm''、''trăm''。
* <span style="font-size:125%;">彥千</span>（合字）：千也。從千，彥聲。
* <span style="font-size:150%;">𪚚</span>：於物之中也。從中，龍聲。喃字讀音''trong''。
* <span style="font-size:150%;">𠄧</span>：與也。從二，巴聲。喃字讀音''và''、''vài''。
* <span style="font-size:150%;">𠸜</span>：名也。從名，先聲。喃字讀音''tên''。
* <span style="font-size:150%;">𧵑</span>：有財也。從貝，古聲。（為虛詞時，其用法似[[標準現代漢語|現代漢語]]「的」字。）''của''。
* <span style="font-size:150%;">𠓨</span>：於物之內也。從入，包聲。喃字讀音''vào''。
* <span style="font-size:125%;">罷由</span>（合字）：由也。從由，罷聲。
* <span style="font-size:150%;">𣎃</span>：月份也，從月，尚聲。（不指代為天體之「月」。）喃字讀音''tháng''。
