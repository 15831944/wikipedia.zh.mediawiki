'''班蘭特斯足球會'''（{{lang-hy|Ֆուտբոլային Ակումբ Բանանց}}），是一家位於[[亞美尼亞|亞美尼亞]]的足球會。現於國內頂級的[[亞美尼亞足球甲級聯賽|亞美尼亞足球甲級聯賽]]角逐。

== 球會榮譽 ==
* [[亞美尼亞足球甲級聯賽|亞美尼亞足球甲級聯賽]]
: 冠軍(1)：2013-14
: 亞軍(4)：2003-04, 2006-07, 2007-08, 2010-11
* [[亞美尼亞獨立盃|亞美尼亞獨立盃]]
: 冠軍(2)：1992, 2007
: 亞軍(5)：2003, 2004, 2008, 2009, 2010
* [[亞美尼亞超級盃|亞美尼亞超級盃]]
: 冠軍(1)：2014
: 亞軍(5)：2005, 2008, 2010, 2011, 2016
