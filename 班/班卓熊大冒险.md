{{vgname|班卓熊大冒险|en=Banjo-Kazooie|ja=バンジョーとカズーイの大冒険}}是[[Rare_Ltd.|Rare]]开发、[[任天堂|任天堂]]发行的一款[[平台动作游戏|平台]][[动作冒险游戏|动作冒险游戏]]，是[[班卓熊系列|班卓熊系列]]的首部作品。它于1998年在[[任天堂64|任天堂64]]上发行，2008年重新在[[Xbox_Live_Arcade|Xbox Live Arcade]]上发行。

游戏设定在虚构的螺旋山，那儿住着一名叫班卓的熊和一只叫小笛的鸟。巫婆Gruntilda绑架了班卓的妹妹Tooty，想通过转换装置偷走她的美貌。班卓和小笛于是冒险去营救Tooty。

《班卓熊大冒险》在商业和口碑上均获得成功，并成为任天堂平台上卖得最好的游戏之一。<ref name = USsales>{{cite web | url = http://www.the-magicbox.com/Chart-USPlatinum.shtml | title = The Magic Box - US Platinum Chart Games. | publisher = The Magic Box | accessdate = 2009-12-08}}</ref><ref>{{cite web|url=http://www.japan-gamecharts.com/n64.php |title=Nintendo 64 Japanese Ranking |publisher=Japan Game Charts |date=2008-04-10 |accessdate=2009-12-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20081230005328/http://www.japan-gamecharts.com/n64.php |archivedate=2008-12-30 }}</ref>
