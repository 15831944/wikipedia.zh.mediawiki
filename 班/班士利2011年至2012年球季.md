'''班士利2011年至2012年球季'''（{{lang-en|'''2011–12 Barnsley F.C. season'''}}）是[[巴恩斯利足球俱乐部|巴恩斯利足球俱乐部]]自1898年加入[[英格蘭足球聯賽|英格蘭足球聯賽]]後第103個球季，亦是連續第6個處於第二級聯賽的球季，上季以第17位結束球季。

== 記事 ==
* 2011年5月12日領隊[[马克·罗宾斯|马克·罗宾斯]]雖然帶領球隊取得近5年最佳排名的17位，但與東主帕特里克·佳尼（Patrick Cryne）未能就新球季的組軍預算及營運前景達成協議，遭球隊提前終止合約<ref>{{cite news |language=en |url=http://news.bbc.co.uk/sport2/hi/football/teams/b/barnsley/9484019.stm |title=Manager Mark Robins set to leave Barnsley |publisher=''[[BBC_Sport|BBC Sport]]'' |date=2011-05-12 |accessdate=2011-05-13}}</ref>，翌日佳尼澄清沒有要求羅賓斯離職<ref>{{cite news |language=en |url=http://www.football-league.co.uk/championship/news/20110513/robins-still-boss-cryne_2293322_2359724 |title=Robins still boss - Cryne |publisher=[[英格蘭足球聯賽|''football-league.co.uk'']] |date=2011-05-13 |accessdate=2011-05-14}}</ref>。
* 2011年5月15日領隊[[马克·罗宾斯|马克·罗宾斯]]透過[[英格蘭足球聯賽領隊協會|英格蘭足球聯賽領隊協會]]以新球季只獲有限資金帶領球隊而決定請辭<ref>{{cite news |language=en |url=http://news.bbc.co.uk/sport2/hi/football/teams/b/barnsley/9484019.stm |title=Mark Robins resigns as Barnsley manager |publisher=''[[BBC_Sport|BBC Sport]]'' |date=2011-05-15 |accessdate=2011-05-16}}</ref>。
* 2011年6月1日委任前[[罗奇代尔足球俱乐部|羅奇代爾]]領隊[[凯斯·希尔|-{zh-hans:凯斯·希尔; zh-hk:基夫·希路;}-]]（Keith Hill）接掌球隊<ref>{{cite news |language=en |url=http://news.bbc.co.uk/sport2/hi/football/13595001.stm |title=Barnsley appoint Rochdale boss Keith Hill as manager |publisher=''[[BBC_Sport|BBC Sport]]'' |date=2011-06-01 |accessdate=2011-06-01}}</ref>。

== 球衣 ==
本季球衣改由[[Nike|-{耐克}-]]（-{Nike}-）製造及供應，主場球衣採用簡單款式，維持傳統的紅衫白褲紅襪，而作客的白衫黑褲白襪，其上衣具有直幼暗紋，而主要[[贊助|贊助]]商是「CK Beckett」。
