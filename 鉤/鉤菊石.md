'''鉤菊石'''（[[學名|學名]]：{{lang|la|'''''Ancyloceras'''''}}），又名'''勾菊石'''，是生存在下[[白堊紀|白堊紀]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]。為[[鉤菊石科|鉤菊石科]]的模式屬。其[[化石|化石]]分布於世界各地。<ref name=fa>[http://fossilworks.org/bridge.pl?a=taxonInfo&taxon_no=84902 Paleobiology Database]</ref><ref name=fb>Sepkoski, Jack [http://strata.geology.wisc.edu/jack/showgenera.php?taxon=231&rank=class Sepkoski's Online Genus Database – Cephalopoda]</ref><ref name=fc>[http://www.fossilmuseum.net/Fossil_Galleries/Ammonites/Ancyloceras/Ancyloceras-heteromorph.htm Fossil Museum]</ref>

== 特徵 ==
和[[鉤菊石科|鉤菊石科]]下許多與本屬有親密關係的菊石一樣，鉤菊石的外殼有很大一部分沒有盤捲，形成鉤狀的外形，因而得名。殼表的橫向肋條隨著殼體的成長，由細密逐漸變得較粗而疏；此外，後期殼體的表面可見清晰的瘤狀殼飾之發育。<ref name="digimuse">{{Cite web |url = http://digimuse.nmns.edu.tw/Demo_2011/showMetadata.aspx?ObjectId=0900000180274747&TypeKind=suMeta&Type=invertebrates&Part=2-2&Domain=gg&Field=fi&Language=CHI |title = digimuse.nmns.edu.tw/Demo_2011/showMetadata.aspx?ObjectId=0900000180274747&TypeKind=suMeta&Type=invertebrates&Part=2-2&Domain=gg&Field=fi&Language=CHI |language = zh |accessdate = 2018-07-18 }}</ref>
