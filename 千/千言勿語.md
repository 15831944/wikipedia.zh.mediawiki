《'''千言勿語'''》（{{lang-en|'''''A Thousand Words'''''}}）是一齣2012年上映的[[美國|美國]]喜劇電影。由《[[w:en:Norbit|我愛籮霸]]》及《[[大哥號玩轉地球|大哥號玩轉地球]]》導演[[w:en:Brian_Robbins|拜恩·羅拔斯]]執導。主角為美國諧星[[艾迪·墨菲|艾迪·墨菲]]。

== 故事簡介 ==
能說善道的傑克（艾迪·墨菲 飾）為了達到目的往往不惜任何代價。然而在得罪了某心靈導師之後，他發現他的生命竟仰賴一株有一千片葉子的魔法樹，每一片葉子代表他這輩子剩下能說的一個字。於是，無可救藥的傑克不再說話並用其他不可取的方式與人溝通。

== 評價 ==
根據[[香港蘋果日報|香港蘋果日報]]2012年3月13日報導，《千言勿語》封塵四年後終於2012年3月9日正式在美國上映，影片不止突破性在影評網 [[Rotten_Tomatoes|Rotten Tomatoes]] 首次得零分劣評，更僅收630萬美元（約4,900萬港元），與4,000萬美元（約3.12億港元）的成本相距甚遠，相信會令電影公司勁蝕<ref>{{cite web |url= http://hk.apple.nextmedia.com/entertainment/art/20120313/16150463|title=愛迪梅菲爛片勁蝕 |accessdate=2012-03-13 }}</ref>。由於美國票房慘淡, 香港代理發行電影商沒有安排電影於戲院上映, 只安排影碟發行。
