《'''千歲Get You!!'''》（{{lang-ja|ちとせげっちゅ!!}}）是[[日本|日本]]漫畫家{{link-ja|真島悦也|真島悦也}}的[[四格漫畫|四格漫畫]]作品。

於[[竹書房|竹書房]]的漫畫雜誌月刊『[[まんがライフオリジナル|まんがライフオリジナル]]』在2002年9月号初次掲載。

『まんがライフオリジナル』的連載在2007年8月号完結，姊妹雜誌的『[[まんがライフMOMO|まんがライフMOMO]]』繼續連載中。（参照[[#連載|連載]]）。

在2005年發售[[廣播劇CD|廣播劇CD]]。

== 登場角色 ==
; 樱庭千歲（{{lang|ja|さくらば ちとせ}}）
: [[配音員|声]] - [[堀江由衣|堀江由衣]]（廣播劇CD）/ [[仲谷明香|仲谷明香]]（電視動畫）
: 本作的主角，精力充沛的11歲小學生。在學校就讀5年2班。非常喜歡柏原宏志。
: 有卡奇色的頭髮，左側頭髮用髮夾夾着。
: 5月5日出生。名字源自「千歲糖」（5月5日和千歲糖並沒有關係，是作者的誤解。）。
: 愛吃[[布丁|布丁]]、[[糖果|糖果]]和炸雞。
: 在情人節時送了很多[[巧克力|巧克力]]給宏志。
: 在刊載雜誌舉辦的人氣讀者投票中，只獲得三位。
; 柏原 宏志（{{lang|ja|かしわばら ひろし}}）
: 声 - [[伊藤健太郎|伊藤健太郎]]（廣播劇CD）/ [[羽多野涉|羽多野涉]]（電視動畫）
: 在免步村民生科工作，一位年輕的地方公務員。受千歲、藤麻子及外甥女美香所愛。
: 曾與藤麻子相親一次，因無緣再約會而衷心祝福藤麻子能早日覓得如意郎君。
: 在人氣讀者投票中獲得了第六位。
; 美咲（{{lang|ja|みさき}}）
: 声 - [[能登麻美子|能登麻美子]]（廣播劇CD）/ [[德井青空|德井青空]]（電視動畫）
: 在人氣讀者投票中獲得了第二位。千歲的同班同學，重度戀父情意結患者，生氣時連藤麻子老師也能即時擊倒，甚至能口部發射死光。
; 柊 雛子（{{lang|ja|ひいらぎ ひなこ}}）
: 声 - [[佐藤利奈|佐藤利奈]]（廣播劇CD）/ [[MAKO|MAKO]]（電視動畫）
: 千歲的同班同學，みさき的親友。家裡有錢的大小姐，最愛角色扮演。
: 在人氣讀者投票中，擊敗了主角及親友取得第一名。
; 藤 麻子（{{lang|ja|ふじ あさこ}}）
: 声 - [[生天目仁美|生天目仁美]]（廣播劇CD）/ [[三石琴乃|三石琴乃]]（電視動畫）
: 千歲的班主任，喜歡柏原宏志，經常生氣地制止千歲纏繞柏原。曾經與柏原相親一次，但聚少離多。
; 晚輩老師（{{lang|ja|後輩先生}}）
: 声 - [[伊藤静|伊藤静]]（廣播劇CD）/ [[山口立花子|山口立花子]]（電視動畫）
: 藤麻子的學妹，留短髮及戴眼鏡。
; 茂先生（{{lang|ja|シゲさん}}）
: 声 - [[緒方賢一|緒方賢一]]（廣播劇CD）/ [[辻親八|辻親八]]（電視動畫）
: 柏原宏志的上司。因千歲經常闖到公司纏擾宏志而下令宏志好好看管千歲。
; 校長先生
: 千歲學校校長，本名不明。
: 早會的時話總是很多。
; 柏原美香
: 声 - [[明坂聰美|明坂聰美]]（電視動畫）
: 柏原宏志的外甥女（姐姐的女兒）。金色長髮，紮雙馬尾，小學二年級生。因從英國留學回國，所以英語非常好（日記也是以英文書寫）。
: 愛上舅舅柏原宏志，對宏志直呼其名，視千歲及藤麻子為情敵。

=== 宠物 ===
; 波奇（{{lang|ja|ポチ}}）
: 宏志飼養的[[狗|狗]]。是隻全身雪白的日本狗。性格溫順。

; コタロー
: 麻子養的大花猫。
: 本來コタロー是千歲在它還是小猫的时撿到的，可是給了麻子收養，因此她便繼續養它。
: 最近，與コタロー相似的猫在麻子身邊出現。
