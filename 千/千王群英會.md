《千王群英会》（英語：The Shell Game II）是[[香港电视广播有限公司|无綫电视]]於1981年4月13日首播，由[[谢賢|谢賢]]、[[汪明荃|汪明荃]]、[[周潤發|周潤發]]、[[劉兆銘|劉兆銘]]及[[任達華|任達華]]等主演，共20集，主題曲《千王群英会》是由[[顾嘉煇|顾嘉煇]]作曲编曲、[[黃霑|黃霑]]填词、[[汪明荃|汪明荃]]主唱。<ref>[https://bka.mpweekly.com/focus/local/20170609-61367  新劇《賭城群英會》有當年《千王之王》影子嗎？]</ref><ref>{{Cite web |url=http://www.wongwanchoi.net/series/game.htm |title=千王群英會 |access-date=2018-02-12 |archive-url=https://web.archive.org/web/20170615170714/http://www.wongwanchoi.net/series/game.htm |archive-date=2017-06-15 |dead-url=yes }}</ref><REF>[https://www.xuehua.us/2018/11/09/%E9%A6%99%E6%B8%AF%E8%B5%8C%E7%89%87%E4%B8%8D%E6%98%AF%E5%8F%AA%E6%9C%89%E7%94%B5%E5%BD%B1%E3%80%8A%E8%B5%8C%E7%A5%9E%E3%80%8B%E3%80%8A%E8%B5%8C%E5%9C%A3%E3%80%8B%EF%BC%8C%E8%BF%98%E6%9C%89%E4%B8%8B/   香港赌片不是只有电影《赌神》《赌圣》，还有下面八部经典电视剧]</REF>
2018年11月26日(星期一)早上十點[[翡翠台|翡翠台]]重播該劇。

==故事大綱==

八年前，南神眼羅四海、北千手卓一夫 南北兩大千王退隱江湖。

八年後，卓一夫病逝，千門再起風雲 ....

阿龍（周潤發）略懂千術，到上海挑戰名聞四海的天下第一金手、新北千王屠一笑（謝賢），大敗而回，屠賞識其膽色頭腦，收龍為徒。二人同在上海發展，認識了熱心文化工作的霍思靈（汪明荃），霍品性善良，清麗脫俗，龍展開熱烈追求，但霍卻對屠有好感，而屠曾經歷婚姻挫折，故對霍之心意，只好 埋藏心底 .....

屠與龍不久成為上海最有威望的老千，他倆樂於助人，到處受人尊敬，但卻招惹杜天（秦煌）處處與屠鬥法，杜落敗，手上地盤、財富亦被屠取去。

杜與隱退多年的老千王仇大千（劉兆銘）之妻于蘭（呂有慧）合謀，利用仇之子仇鏗（黃允財），迫仇東山復出挑戰屠，屠更得悉霍欲與龍成親，大受打擊，接納仇挑戰 .....。
