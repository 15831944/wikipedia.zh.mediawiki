'''千岛湖'''即'''新安江水库'''，位于[[中華人民共和國|中国]][[浙江省|浙江省]][[杭州市|杭州市]]西南部的[[淳安县|淳安县]]和[[建德市|建德市]]境内，是1959年在[[新安江|新安江]]建德[[马铜官峡|马铜官峡]]筑坝兴建水电站而形成的[[人工湖|人工湖]]，面积达567.40平方公里<ref>{{Cite web|url=http://www.sbsm.gov.cn/article//dlxxshgb/201107/20110700086263.shtml|title=浙江省测绘与地理信息局关于启用浙江省主要河流长度、流域面积、主要湖泊面积数据的公告|publisher=浙江省测绘与地理信息局|date=2010-09-30|accessdate=2011-09-05|deadurl=yes|archiveurl=https://web.archive.org/web/20131202231114/http://www.sbsm.gov.cn/article//dlxxshgb/201107/20110700086263.shtml|archivedate=2013-12-02}}</ref>。水库蓄水后未被淹没的山峰形成岛屿，当水库水位运行于108米高程（黄海）时，2500平方米以上岛屿共1078个，因此新安江水库得名千岛湖。1982年被国务院确定为[[国家级风景名胜区|国家级风景名胜区]]<ref name="千岛湖生态保护"/>。1984年12月15日，经浙江省地名委员会批准，千岛湖作为新安江水库的风景区名。<ref>{{cite web|url=http://www.1078e.com/tour_News_show.asp?Newsid=2034&page=|title=见证60年:新安江水库更名为千岛湖一转折迎来新天地|publisher=网上千岛湖|date=2009-09-10|accessdate=2015-10-25|author=邹楚环}}{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

千岛湖整个湖区分为中心、东南、西北、西南和东北五个地理风格约有差异的区域，其中的中心和东南湖区已经开发成为商业旅游区。

== 建設歷史 ==
* 1947年（民国36年）11月19日～11月24日，[[浙江省|浙江省]][[钱塘江海塘工程局|钱塘江海塘工程局]]会同浙江省水利局、全国水力发电工程总处、水利部和[[中央地质调查所|中央地质调查所]]，由[[汪胡桢|汪胡桢]]等7人组成钱塘江上游调查团，先后察勘浙江、[[安徽|安徽]]交界处的[[街口|街口]]、建德[[罗桐埠|罗桐埠]]和[[桐庐|桐庐]][[七里泷|七里泷]]等处坝址，淳安[[贺城|贺城]]至建德[[梅城|梅城]]间和桐庐至[[杭州|杭州]]间的河道形势，编成《钱塘江上游察勘报告》，认为宜在街口建造高坝，装机8万千瓦，常年发电量4亿千瓦时；并调节洪水，减轻下游洪灾损失。形成街口、邵村、罗桐埠3级开发方案。
* 1953年4月，以浙江水力发电工程处为基础，成立[[华东水力发电工程局|华东水力发电工程局]]，承担新安江等水力发电站的勘测设计前期工作。
* 1954年华东水力发电工程局（现中国水利水电第十二工程局有限公司）勘测处新安江地质队毛水根等人开始勘探街口、黄江潭、云头、邵村、芹坑、铜官、罗桐埠和白沙等8处坝址。之后水力发电建设总局在比较8个坝址不同组合的1级、两级和3级开发方案后，选出罗桐埠1级开发和罗桐埠、黄江潭两级开发两个方案。[[中共|中共]]中央华东局书记[[谭震林|谭震林]]在听取水力发电建设总局汇报后，对1级开发持肯定态度。
* 1955年10月7日，[[国家建设委员会|国家建设委员会]]批准1级开发罗桐埠电站方案。
* 1956年5月，成立[[电力工业部|电力工业部]][[新安江水力发电工程局|新安江水力发电工程局]]（后更名中国水利水电第十二工程局），[[王醒|王醒]]任局长，[[徐洽时|徐洽时]]任总工程师。
* 1956年8月20日，新安江水库开始动工兴建。全线整修杭州至淳安公路，并加固桥梁，建上坝公路；同年10月动工兴建[[兰溪|兰溪]]至铜官专用铁路。
* 1957年4月，电站主体工程开工，开始清基开挖。
* 1958年7月，[[黄坛口水库|黄坛口水电站]]开始向新安江工地供电，为施工期主要电源。
* 1959年，新安江水库建设叠遇左岸大坍方、洪水破围堰、材料供应混乱、混凝土出现严重质量事故。
* 1959年4月，[[周恩来|周恩来]]总理视察工地时题词：“为我国第一座自己设计和自制设备的大型水力发电站的胜利建设而欢呼！”
* 1959年9月，新安江截流，水库开始蓄水。淹没了淳安、[[遂安|遂安]]的贺城、狮城、威坪镇、茶园镇和港口三个城镇（1958年，淳安、遂安两县合并为现在的淳安县），淹没了49个乡的1377个自然村，淹没了耕地良田超过三十万亩和城镇工商企业两百多家。
* 1960年4月22日，新安江水电厂建成，第一台7.25万千瓦机组开始发电。
* 1960年9月26日，新安江水电厂和华东电网并网供电。
* 1962年～1963年，陆续在右坝头0、1号段发现多条裂缝。
* 1965年12月，枢纽土建工程全部竣工。
* 1966年7月12日，杭州市大雨，新安江水库水位上升，大坝启闸泄洪。
* 1967年6月～9月,发现右岸坡观测点持续沉陷，1号坝段混凝土栏杆断裂。
* 1970年5月，新安江大坝右坝肩边坡变形问题讨论会上，提出边坡变形和页岩被淘刷及泥化有关，并认为3号坝段扬压力和渗漏偏大的主要原因是厚层页岩未处理好。
* 1977年，9台机组全部安装完毕，总容量66.25万千瓦。
* 1977年10月22日，最后一台机组投入运行。
* 1983年7月起，原上海院及原水电建设局总工[[马君寿|马君寿]]多次致函原[[电力工业部|电力工业部]]生产司，提出要对页岩进行强化处理。
* 1986年1月，在[[北京|北京]]举行的新安江大坝基础处理技术讨论会，提出“若长期使页岩泥化、不断造成岩层结构松弛形成局部空洞，后果严重，将危及大坝安全，因此要加固坝基防止泥化和管涌的发展”。
* 1990年5月，开始历时2年的首次大坝安全定期检查，认定为正常坝，但存在缺陷急需处理。

== 影响 ==
1959年，新安江水电站建成蓄水，淹没2座县城，8个镇，39个乡，1377个自然村，255家企业，270268间房屋，307838亩耕地。289951人离别故土，移往[[浙江省|浙]]、[[江西省|赣]]、[[安徽省|皖]]各地。淳安县比较平坦的地方全部淹没，剩下了比较高的山地，现在的淳安县城千岛湖镇也是建立在山顶上。

== 地理 ==
库区地形以低山丘陵为主，最高海拔967米。山地土壤主要是红壤、黄壤和岩性土。<ref name="千岛湖天然次生林">{{cite journal |author=余树全，李翠环，姜礼元等 |title=千岛湖天然次生林群落生态学研究 |journal=浙江林学院学报 |year=2002 |volume=19 |issue=2 |pages=138-142 |url=http://www.tbotany.csdb.cn/WEB/pdf/%C7%A7%B5%BA%BA%FE%CC%EC%C8%BB%B4%CE%C9%FA%C1%D6%C8%BA%C2%E4%C9%FA%CC%AC%D1%A7%D1%D0%BE%BF.PDF }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="千岛湖森林资源">{{cite journal |author=徐高福，章德三，张立功等 |title=千岛湖森林资源动态变化分析 |journal=林业调查规划 |year=2005 |volume=30 |issue=2 |pages=25-28 |url=http://www.cqvip.com/qk/90257b/2005002/15618753.html }}</ref>

== 植被 ==
库区的原生植被为亚热带常绿阔叶林，建设水库时原生植被被严重破坏，现存植被是建库后经过封山育林并人工补植形成的。自1963年以来，先后在千岛湖进行过5次森林资源调查，森林覆盖率由23.8％增加到89.4％。湖区森林面积为14.23平方千米, 小片[[常绿阔叶林|常绿阔叶林]]与大面积的[[马尾松|马尾松]]林和针阔混交林镶嵌状分布。<ref name="千岛湖天然次生林"/><ref name="千岛湖森林资源"/>

== 水体 ==
千岛湖最吸引人的还是其超过170亿立方米的一级水体，能见度达到10米以上，而且在多数湖区可以直接饮用。在建德市与千岛湖镇建有[[农夫山泉|农夫山泉]]生产基地，直接抽取湖中深层水。

== 旅游 == 
2006年接待游客180万人，门票收入达1.31亿元，旅游经济总收入19.1亿元<ref>[http://www.chinasus.org/case/tourism/20090401/46860-4.shtml 千岛湖进贤湾旅游度假区块概念性规划]</ref>，2009年旅游经济总收入38.1亿元。目前淳安直接从事旅游业有1.5万人，带动间接从业达到6万多人，占到全县总就业人数的27％。<ref name="引领低碳生态旅游">[http://news.daynews.com.cn/shxw/1024133.html 千岛湖——引领低碳生态旅游] {{webarchive|url=https://web.archive.org/web/20131127024807/http://news.daynews.com.cn/shxw/1024133.html |date=2013-11-27 }}</ref>

==水下遗迹==
{{main|遂安县城|淳安县城}}
淳安自古便是交通要道，且这一地区土地肥沃，所以在古代这一地区经济发达，人口稠密。
1959年水库建成后，将淳安，遂安两座县城及周边的面积的古村落、古民居淹没在水下，使得这些遗迹保存完好。其中淳安县更是以从古代至近代的牌坊而出名，被称为“牌坊博物馆”。据国家海洋局测绘显示，淳安县城城墙周长2.75公里，高度两米，有城门五座。
