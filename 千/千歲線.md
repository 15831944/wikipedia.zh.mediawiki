'''千歲線'''（{{jpn|j=千歳線|hg=ちとせせん|rm=Chitose sen|f=y}}）是一條屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）的鐵道路線（[[幹線|幹線]]）。[[北海道|北海道]]（[[膽振綜合振興局|膽振管內]]）[[苫小牧市|苫小牧市]]的[[沼之端站|沼之端站]]起與北海道（[[石狩振興局|石狩管內]]）[[札幌市|札幌市]][[白石區|白石區]]的[[白石站_(JR北海道)|白石站]]連結的本線，以及北海道（石狩管內）[[千歲市|千歲市]]的[[南千歲站|南千歲站]]起分岔至同市的[[新千歲機場站|新千歲機場站]]的支線（機場線）。

== 概要 ==
沼之端站與[[室蘭本線|室蘭本線]]接續，白石站與[[函館本線|函館本線]]接續。

此線是JR北海道的道內完結路線當中唯一全線[[電氣化鐵路|電氣化]]的路線。札幌市起往[[道南|道南]]、[[道東|道東]]的主要幹線，與此同時也是作為前往新千歲機場的[[機場聯絡軌道系統|連接路線]]、作為札幌[[近郊|近郊]]的[[通勤鐵路|通勤輸送]]路線、作為{{link-ja|札幌都市圈|札幌都市圏}}與往道外的[[物流|物流]]路線等多種性質合併的路線。尤其是南千歲以北的路段，是北海道內少數的班次過密運行圖的路線。另外，此線路段也有[[日本貨物鐵道|日本貨物鐵道]]（JR貨物）的[[貨物列車|貨物列車]]運行。

包括支線的全線，連同室蘭本線的[[苫小牧站|苫小牧站]]－沼之端站間一併成為[[智能卡|IC]][[乘車卡|乘車卡]]「[[Kitaca|Kitaca]]」的使用範圍。不過也存在通過{{link-ja|札幌啤酒庭園站|サッポロビール庭園駅}}、[[植苗站|植苗站]]的{{link-ja|普通列車|普通列車}}。然而，{{link-ja|美美站|美々駅}}使用人數的減少導致在2017年（[[平成|平成]]29年）3月4日廢除。

支線路段加算南千歲站－新千歲機場間的車費，是計算距離後的車費再加算140日圓。

== 歷史 ==
1926年（大正15年）8月21日，北海道鐵道'''札幌線'''舖設完成<ref name="history 1">{{cite web |url=http://www.city.chitose.hokkaido.jp/shishi_hp/paper/sikotsu04_01.htm |language=ja |title=小さな私鉄の大きな変身－千歳線八十年の歩みを振り返る－「1.北海道鉱業鉄道から北海道鉄道へ」「2.新聞の記事に見る全通、認可、開業の日」 |publisher=引自網站 |accessdate=2017-09-11 |archive-url=https://web.archive.org/web/20080408111056/http://www.city.chitose.hokkaido.jp/shishi_hp/paper/sikotsu04_01.htm |archive-date=2008-04-08 |deadurl=yes }}</ref>。當時興建的目的為連接當時的[[鐵道省|國鐵]][[室蘭本線|室蘭本線]]及[[函館本線|函館本線]]，協助運輸在岩見澤附近煤礦地區開採的煤炭，由室蘭本線經過本路線至其他地區。開業時沼之端站至苗穗站之間每天來回有5班列車，並主要使用[[客貨混編列車|客貨混編列車]]。當時由沼之端站至苗穗站，以及千歲站至苗穗站的時間分別需要2小時20分及1小時30分<ref name="history 2">{{cite web |url=http://www.city.chitose.hokkaido.jp/shishi_hp/paper/sikotsu04_02.htm |language=ja |title=小さな私鉄の大きな変身－千歳線八十年の歩みを振り返る－「3.札幌線の停車場」「4.戦前・戦中の動力車等」 |publisher=引自網站 |accessdate=2017-09-11 |archive-url=https://web.archive.org/web/20080407210630/http://www.city.chitose.hokkaido.jp/shishi_hp/paper/sikotsu04_02.htm |archive-date=2008-04-07 |deadurl=yes }}</ref>。

由於日膽地區（日高、膽振）的居民持續的需求，室蘭本線的苫小牧站至沼之端站、以及函館本線的苗穗站至札幌站分別於1934年（昭和9年）10月1日及1940年（昭和15年）10月26日開始有旅客列車行駛，當時由苫小牧站至札幌站的班次為每天來回4班列車，需要時間為約1小時30分<ref name="history 2"/>。此外，1935年（昭和10年）12月1日本線開始使用[[柴聯車|柴聯車]]。

[[第二次世界大戰|第二次世界大戰]]期間，為了強化戰爭時的運輸體制，當時的第十二航空艦隊司令部於1943年（昭和18年）8月1日購入鄰近千歲海軍航空基地的札幌線，成為國鐵千歲線<ref name="history 2"/><ref name="history 3">{{cite web |url=http://www.city.chitose.hokkaido.jp/shishi_hp/paper/sikotsu04_03.htm |language=ja |title=小さな私鉄の大きな変身－千歳線八十年の歩みを振り返る－「5.戦時買収」「6.千歳市内における引き込み線」 |publisher=引自網站 |accessdate=2017-09-11 |archive-url=https://web.archive.org/web/20080408111100/http://www.city.chitose.hokkaido.jp/shishi_hp/paper/sikotsu04_03.htm |archive-date=2008-04-08 |deadurl=yes }}</ref>。第二次世界大戰終結後，苫小牧站至札幌站的班次只恢復至每天來回1班列車，而苫小牧站至苗穗站、及苫小牧站至千歲站或惠庭站則有數班列車，直至1950年（昭和25年）10月1日才恢復至戰時的每天來回4班列車。而由1953年（昭和28年）12月開始所有旅客及混合列車皆由札幌站為起點及終點站<ref name="history 4">{{cite web |url=http://www.city.chitose.hokkaido.jp/shishi_hp/paper/sikotsu04_04.htm |language=ja |title=小さな私鉄の大きな変身－千歳線八十年の歩みを振り返る－「7.RTO(railroad transportation office)の時代」「8.国鉄白紙改正」 |publisher=引自網站 |accessdate=2017-09-11 |archive-url=https://web.archive.org/web/20080408111106/http://www.city.chitose.hokkaido.jp/shishi_hp/paper/sikotsu04_04.htm |archive-date=2008-04-08 |deadurl=yes }}</ref>。

戰前，連接札幌與函館的路線主要為經過小樽及俱知安的函館本線。而千歲線的道床則相對薄弱<ref name="history 3"/>，北廣島站至苗穗站之間的路段亦有狹窄的急彎<ref name="history 5">{{cite web |url=http://www.city.chitose.hokkaido.jp/shishi_hp/paper/sikotsu04_05.htm |language=ja |title=小さな私鉄の大きな変身－千歳線八十年の歩みを振り返る－「9.複線化と線路移設」「10.鉄空一貫（まとめとして）」 |publisher=引自網站 |accessdate=2017-09-11 |archive-url=https://web.archive.org/web/20080408111111/http://www.city.chitose.hokkaido.jp/shishi_hp/paper/sikotsu04_05.htm |archive-date=2008-04-08 |deadurl=yes }}</ref>。不過，函館本線長萬部站至小樽站之間也要經過很多斜坡及山路，因而稱為「山路」；與之對比的是較平坦的室蘭本線長萬部站至沼之端站及千歲線，稱為「海線」。因此戰爭期間及戰後會對路線進行改良以加強運輸能力。

1950年代開始設定優等列車，1960年代之後，雖然海線路線長度比山線長超過30 km<ref group="*">2013年（平成25年）時，札幌站至函館站的營業距離，沿海線為318.7 km，沿山線為286.3 km，相差32.4 km。</ref>，但是由於路線的周邊環境比較適合高速運輸，令本線確立為函館至札幌的主要幹道。1961年（昭和36年）10月1日的時間表修訂後，特急列車「[[超級大空號列車|大空]]」開始行駛。而最初的行駛路線並沒有停靠千歲線內任何一個-{車站}-<ref name="history 4"/>，只停靠苫小牧及札幌。

由1965年（昭和40年）9月22日開始千歲線進行[[複線鐵路|複線化]]，直到1969年（昭和44年）9月25日為止，惠庭站至北廣島站、美美站至千歲站、植苗站至美美站、沼之端站至植苗站已先後完成複線化。1973年（昭和48年）9月9日，配合新札幌副都心的開發計劃，北廣島站至苗穗站這段因為線形造成運輸瓶頸的路段進行改建並同時複線化，從而全線完成複線化。1980年（昭和55年）10月1日，全線完成[[電氣化鐵路|電氣化]]（[[交流電|交流電]]20,000 [[伏特|V]]、50 [[赫茲|Hz]]）並開設千歲機場站（現在的南千歲站）<ref name="history 5"/>，此時本線作為札幌都市圈的重要運輸路線及前往千歲機場（舊機場）的機場鐵路，在運輸能力方面進一步加強。

1981年（昭和56年）10月1日[[石勝線|石勝線]]開始營運<ref name="history 5"/><ref>{{cite web |url=http://photodb.hokkaido-np.co.jp/detail/0090263563 |language=ja |title=石勝線きょう開業 |publisher=フォト北海道 |date=1981-10-01 |accessdate=2017-09-08 |archive-url=https://web.archive.org/web/20151018130401/http://photodb.hokkaido-np.co.jp/detail/0090263563 |archive-date=2015-10-18 |deadurl=yes }}</ref>。石勝線連接千歲機場站及千歲線，作為札幌連接新得、帶廣及釧路地區的路線。過去根室本線的優等列車需要從札幌沿函館本線北上，到達瀧川後再沿根室本線南下到達新得，而石勝線開業後可以沿千歲線及石勝線前往新得，成為前往道東地區的較方便的選擇。

1986年（昭和61年）11月1日，函館本線長萬部站至小樽站的山線定期優等列車（特急、急行）全部廢除後，千歲線成為連接札幌及函館的唯一主要路線，包括特急「[[北斗號列車|北斗]]」在內的寢台特急、夜行急行及高速貨物列車頻繁地使用此路線，而路線亦進行結構強化及適應高速化的工程。

1987年（昭和62年）4月1日進行[[國鐵分割民營化|國鐵分割民營化]]，本路線由[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）繼承。1992年（平成4年）7月1日，千歲線支線（機場線）與[[新千歲機場|新千歲機場]]<ref group="*">機場本身已於1988年（昭和63年）7月20日正式投入使用。</ref>客運大樓同日開始投入使用<ref name="airport">{{cite web |url=http://photodb.hokkaido-np.co.jp/detail/0090366276 |language=ja |title="新玄関"きょう開業－新千歳空港ターミナルビル、JRが初めて乗り入れ |publisher=フォト北海道 |date=1992-07-01 |accessdate=2017-09-08 |archive-url=https://web.archive.org/web/20151018130743/http://photodb.hokkaido-np.co.jp/detail/0090366276 |archive-date=2015-10-18 |deadurl=yes }}</ref>。

2017年（平成29年）5月15日，JR北海道於官方網站公佈，由於[[新札幌車站_(JR北海道)|新札幌站]]及[[千歲車站_(北海道)|千歲站]]的抗震標準為1970年（昭和45年）之前適用的標準，而近期新札幌站出現混凝土剝落的情況，因此2017年度至2021年代將會對架空天橋進行進一步的抗震工程<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2017/170511-2.pdf |language=ja |title=高架橋の耐震補強工事の実施について |publisher=JR北海道 |date=2017-05-11 |accessdate=2017-09-08 |archive-url=https://web.archive.org/web/20170513204105/http://www.jrhokkaido.co.jp/press/2017/170511-2.pdf |archive-date=2017-05-13 |deadurl=yes }}</ref>。
=== 北海道鐵道札幌線 ===

* 1926年（大正15年）8月21日 - 北海道鐵道'''札幌線'''沼之端站至苗穗站路段（62.6 km<ref name="history 1"/>）開始營運。該路段的'''植苗站'''、'''美美站'''、'''千歲站'''、'''惠庭站'''、'''島松站'''、'''北廣島站'''、'''上野幌站'''（初代）、'''大谷地站'''、'''月寒站'''、'''東札幌站'''亦同時設立<ref name="history 2"/>。
* 1931年（昭和6年）7月25日 - 苗穗站至東札幌站路段[[鐵路電氣化|電氣化]]（[[直流電|直流電]]1,500 [[伏特|V]]），{{Link-ja|定山溪鐵道|定山渓鉄道線}}的[[柴聯車|柴油列車]]開始駛入苗穗站。
* 1934年（昭和9年）10月1日 - [[室蘭本線|室蘭本線]]的[[苫小牧車站|苫小牧車站]]至沼之端站路段開始有旅客列車駛入<ref name="history 2"/>。
* 1935年（昭和10年）12月1日 - 苗穗站至苫小牧站開始使用[[柴油機車|柴油機車]]（501型，即國鐵40350型）<ref name="history 2"/>。
* 1940年（昭和15年）10月26日 - [[函館本線|函館本線]]的苗穗站至札幌站路段開始有旅客列車駛入<ref name="history 2"/>。

=== 國有鐵道 ===
* 1943年（昭和18年）8月1日 - [[鐵道省|鐵道省]]購入北海道鐵道並國有化。沼之端站至苗穗站路段改稱為'''千歲線'''<ref>{{cite web|url=http://dl.ndl.go.jp/info:ndljp/pid/2961465/3 |language=ja |title=鉄道省告示第204号 |publisher=[[國立國會圖書館|國立國會圖書館]]數碼館藏 | date=1943-07-26 |accessdate=2017-09-08 }}</ref>。
* 1945年（昭和20年）
** 4月1日 - 上野幌站（初代）開始處理貨物及行李等業務。
** 11月4日 - [[室蘭車站|室蘭車站]]至苗穗站路段開始有同盟國專用列車行駛<ref name="history 4"/>。
** 12月25日 - 植苗站開始處理貨物及行李等業務。
* 1946年（昭和21年）11月10日 - 上野站至札幌站之間的同盟國專用列車改為行駛千歲線<ref name="history 4"/>。
* 1949年（昭和24年）6月1日 - 隨著日本國有鐵道法實施，由[[日本國有鐵道|日本國有鐵道]]（國鐵）接管。
* 1955年（昭和30年）6月1日 - 全線使用[[柴聯車|柴油列車]]。
* 1957年（昭和32年）
** 8月12日 - 定山溪鐵道的自行研製柴油列車開始駛入札幌站。
** 10月1日 - 定山溪鐵道的旅客列車停止駛入苗穗站。同時苗穗站至東札幌站之間的電氣化設備（直流電1,500 V）移除。
* 1958年（昭和33年）
** 3月1日 - 千歲站至惠庭站之間設置'''長都臨時乘降場'''。
** 7月1日 - 長都臨時乘降場升格為'''長都站'''。
* 1959年（昭和34年）
** 6月6日 - 行駛札幌站至[[樣似站|樣似站]]（經千歲線、[[日高本線|日高本線]]）的準急列車「襟裳」開始行駛。
** 9月22日 - 行駛札幌站至室蘭站、虻田站（現在的{{Link-ja|洞爺車站|洞爺駅}}）的準急列車「千歲」開始行駛。
* 1960年（昭和35年）
** 7月1日 - 行駛函館站至札幌站（經室蘭本線、千歲線）的柴油驅動急行列車「鈴蘭」開始行駛。
** 8月1日 - 植苗站停止貨物處理。
* 1961年（昭和36年）
** 1月14日 - 北廣島站至上野幌站（初代）之間設置'''西之里信號場'''（初代）。
** 9月20日 - 大谷地站停止貨物處理。
** 10月1日 - 隨著更改時間表，行駛[[函館車站|函館車站]]至[[旭川車站|旭川車站]]的特急列車「[[超級大空號列車|大空]]」開始行駛。從此之後，主要列車皆行駛千歲線。
* 1964年 - 行駛函館站至[[釧路車站|釧路車站]]、[[網走車站|網走車站]]（經室蘭本線、千歲線）的特急列車「[[鄂霍茨克號列車|鳳凰]]」開始行駛。
* 1965年（昭和40年）
** 9月22日 - 千歲站至惠庭站路段完成[[複線鐵路|複線化]]。
** 10月1日 - 行駛函館站至旭川站（經室蘭本線、千歲線）的特急列車「[[北斗號列車|北斗]]」開始行駛。

* 1981年（昭和56年）3月5日 - 沼之端站至白石站及室蘭本線的沼之端站至室蘭站路段引入[[調度集中系統|調度集中系統]]（CTC）<ref>{{cite web |url=http://photodb.hokkaido-np.co.jp/detail/0090265211 |language=ja |title=室蘭．千歳線CTCが始動 |publisher=フォト北海道 |date=1981-03-06 |accessdate=2017-09-08 |archive-url=https://web.archive.org/web/20151018141500/http://photodb.hokkaido-np.co.jp/detail/0090265211 |archive-date=2015-10-18 |deadurl=yes }}</ref>。

== 其他 ==
2016年（平成28年），JR北海計劃將行走千歲線的快速列車「機場」由現時每小時4班列車（15分鐘時間間隔）增加至每小時5班列車（12分鐘時間間隔），並同時檢討札幌貨物總站與本線相關[[聯絡線|聯絡線]]的[[交通立體化|交通立體化]]事宜。這是因為札幌貨物總站的貨物列車頻繁地駛入本線，而往千歲方向的貨物列車必須短暫地使用一小段札幌方向的路軌，以現時的鐵路配線情況對於增加班次有一定困難。此外，亦曾考慮增加列車的車卡數目，但這需要對-{車站}-月台進行大規模改建，因此不在考慮之列<ref>{{cite web |url=http://dd.hokkaido-np.co.jp/news/economy/economy/1-0330572.html |language=ja |title=快速エアポート増発、財源が課題　立体交差など百数十億円 |publisher=[[北海道新聞|北海道新聞]] |date=2016-10-25 |accessdate=2017-09-11 |archive-url=https://web.archive.org/web/20161025083513/http://dd.hokkaido-np.co.jp/news/economy/economy/1-0330572.html |archive-date=2016-10-25 |deadurl=yes }}</ref>。

而日本職業棒球隊伍[[北海道日本火腿鬥士|北海道日本火腿鬥士]]檢討及考慮興建新的主場館，以取代現時的主場館[[札幌巨蛋|札幌巨蛋]]。而[[北廣島市|北廣島市]]亦提出「球場構想」，在北廣島站約1.5公里距離的地方興建「北廣島總合運動公園」以吸引火腿鬥士於該市興建新球場，日本火腿集團已於2018年3月正式決定在北廣島市興建球場，預定2023年完工啟用<ref>[http://www.cna.com.tw/news/aspt/201803260273-1.aspx 日職棒火腿隊新球場 落腳北廣島市]</ref>。而為了方便前往新球場的遊客，該市要求JR北海道考慮於千歲線設置新車站，而JR北海道亦將此方案列入考慮之列<ref>{{cite web |url=http://dd.hokkaido-np.co.jp/news/area/sapporo/1-0350759.html |language=ja |title=北広島市、日ハムに新球場構想提案　誘致への熱意アピール |publisher=北海道新聞 |date=2016-12-21 |accessdate=2017-09-11 |archive-url=https://web.archive.org/web/20161221094456/http://dd.hokkaido-np.co.jp/news/area/sapporo/1-0350759.html |archive-date=2016-12-21 |deadurl=yes }}</ref><ref>{{cite web |url=http://dd.hokkaido-np.co.jp/sports/baseball/fighters/1-0350560.html |language=ja |title=北広島市が日ハム誘致案　新球場構想 |publisher=北海道新聞 |date=2016-12-21 |accessdate=2017-09-11 |archive-url=https://web.archive.org/web/20161221093838/http://dd.hokkaido-np.co.jp/sports/baseball/fighters/1-0350560.html |archive-date=2016-12-21 |deadurl=yes }}</ref>。
