《'''千千闕歌'''》是香港女歌手[[陳慧嫻|陳慧嫻]]於1989年推出的一首經典歌曲，改編自[[日本|日本]]歌手[[近藤真彥|近藤真彥]]的《{{lang|ja|[[:ja:夕焼けの歌|夕焼けの歌]]}}》，作曲{{link-ja|馬飼野康二|馬飼野康二}}、填詞[[林振強|林振強]]、編曲[[盧東尼|盧東尼]]，收錄於同年推出的大碟《[[永遠是你的朋友|永遠是你的朋友]]》，是其代表作之一。此曲推出後獲香港各大媒體熱播，在以播放率計算排名的[[叱咤903|叱咤903]]專業推介中取得三星期冠軍的佳績，亦獲得多個樂壇大獎，如[[1989年度叱咤樂壇流行榜頒獎典禮|1989年度叱咤樂壇流行榜頒獎典禮]]我最喜愛的歌曲大獎，其後更紅遍華人社會，唱至街知巷聞。

==歷史==
此曲的原版是[[日本|日本]]歌手[[近藤真彥|近藤真彥]]推出的《[[夕阳之歌_(近藤真彦单曲)|夕阳之歌 (近藤真彦单曲)]]》，作曲者[[馬飼野康二|馬飼野康二]]。1988年，陳慧嫻宣布即將退出樂壇後，[[寶麗金|寶麗金]]為她製作一系列的「離別」歌曲，而《千千闕歌》便是其一，改編歌詞由著名已故填詞人[[林振強|林振強]]操刀，全然寫出離別的哀傷感，加上動人的旋律，令此曲甫推出即成大熱歌曲，全城熱播。

另外，此曲亦被改編成其他粵語版，如[[梅艷芳|梅艷芳]]的《[[夕陽之歌|夕陽之歌]]》、[[張智霖|張智霖]]及[[許秋怡|許秋怡]]的《夢斷》及[[Blue_Jeans|Blue Jeans]]的《無聊時候》，而國語版則是[[台灣|台灣]]女歌手[[李翊君|李翊君]]的《風中的承諾》。

==「闕」與「闋」==
《千千闕歌》出版時誤植為「闕」，實為別字<ref>{{Cite web |url = http://blog.ifeng.com/article/31810319.html |title = 「一闋」，非「一闕」 |author = 黃志華 |date = 2014年02月11日 |publisher = 鳳凰博報 |archiveurl=https://web.archive.org/web/20160809210050/http://blog.ifeng.com/article/31810319.html |archivedate=2016-08-09}}</ref>，林振強的歌詞手稿原為《千千闋歌》<ref> {{Cite book | author = 洋葱嫂 | title = 洋葱頭背後的林振強 | location = 香港 | publisher = 壹出版有限公司 | date = 2013年11月 | ISBN = 9789888059898 | language = 中文 }} </ref>。「闕」是指古代宮門外兩邊供瞭望的樓臺<ref>{{Cite web |url = http://dict.revised.moe.edu.tw/cgi-bin/cbdic/gsweb.cgi?o=dcbdic&searchid=W00000006693 |title = 教育部重編國語辭典修訂本 |date = 2015 |publisher = 中華民國教育部 }}</ref>；而「闋」才是計算歌、詞、曲的單位<ref>{{Cite web |url = http://dict.revised.moe.edu.tw/cgi-bin/cbdic/gsweb.cgi?o=dcbdic&searchid=W00000006692 |title = 教育部重編國語辭典修訂本 |date = 2015 |publisher = 中華民國教育部 }}</ref>。

==1989年金曲金獎爭議==
由於梅艷芳和[[Blue_Jeans|Blue Jeans]]所屬的[[華星唱片|華星唱片]]和索尼音樂娛樂同時推出《夕陽之歌》和《無聊時候》，因此出現了「鬧三胞」事件，而《夕陽之歌》則成為電影《[[英雄本色III夕陽之歌|英雄本色3]]》的主題曲。

雖然梅艷芳貴為樂壇一姐，但由於《千千闕歌》較入屋及易上口，加上陳慧嫻十分動人的演繹，令《千千闕歌》的流行程度更勝一籌。陳慧嫻憑此曲於[[1989年度叱咤樂壇流行榜頒獎典禮得獎名單|叱咤頒獎禮]]獲得2715票以壓倒性姿態獨得「我最喜愛的歌曲大獎」。而商台高層[[俞琤|俞琤]]更直指此獎項為「真正代表民意」。

但在[[1989年度十大勁歌金曲得獎名單|TVB勁歌金曲頒獎禮]]中，主辦單位竟將金曲金獎頒予《夕陽之歌》，當時傳聞是由於主辦單位希望將此大獎留給「自己人」，因此選擇頒給當時隸屬TVB旗下[[華星唱片|華星]]的梅艷芳，而陳慧嫻亦在寶麗金（環球集團附屬公司）的要求下未有留在台上等待宣佈頒獎一刻，可見此獎項令三家唱片公司極度不滿。

==其他版本==
由於《千千闕歌》大熱，很多著名歌手都曾翻唱過，較為人熟悉的是天王[[張國榮|張國榮]]在1990年的告別演唱會中的翻唱，為一時佳話。

2015年，[[廣州恆大|廣州恆大]]球迷歡送領隊[[納比|納比]]，萬人齊唱《千千闕歌》。<ref>[https://www.youtube.com/watch?v=xhbYRQU39lM 廣州球迷萬人合唱《千千闕歌》 Farewell Song by Guangzhou Fans]</ref>

有「翻版陳慧嫻」之稱的[[糖妹|糖妹]]在2016年大金大家庭演唱會翻唱此歌。

另外，影星[[成龍|成龍]]亦是《千千闕歌》的超級歌迷，在歌曲當紅時曾經在片場不斷播放《千千闕歌》的錄音帶<ref>[https://www.youtube.com/watch?v=z5dbm0uvrjA 星級會客室 - 陳慧嫻]</ref>。
