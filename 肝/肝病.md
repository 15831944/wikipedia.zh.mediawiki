'''肝病'''（{{lang-en|'''Hepatic disease'''、'''Liver disease'''}}）是指[[肝臟|肝臟]]疾病，或是肝臟受損的症狀。<ref name="medl">{{Cite web|title = Liver Diseases: MedlinePlus|url = https://www.nlm.nih.gov/medlineplus/liverdiseases.html|website = www.nlm.nih.gov|access-date = 2015-06-20}}</ref>

==肝病種類==
[[File:Ground_glass_hepatocytes_high_mag_cropped_2.jpg|thumb]]
[[File:Primary_biliary_cirrhosis_intermed_mag_much_cropping.jpg|thumb]]
[[File:Buddchiari2.PNG|thumb]]
肝病種類超過一百種以上。症狀包括黃疸及體重減輕。以下是一最常見的肝病<ref>{{Cite web|title = Liver disease – NHS Choices|url = http://www.nhs.uk/Conditions/liver-disease/Pages/Introduction.aspx|website = www.nhs.uk|access-date = 2015-06-20}}</ref>：
* {{tsl|en|Fascioliasis|Fascioliasis}}，是由{{tsl|en|Fasciola|片形屬}}的寄生蟲（大部份是[[牛羊肝吸蟲|牛羊肝吸蟲]]）造成的寄生蟲感染<ref>{{Cite web|title = CDC – Fasciola|url = https://www.cdc.gov/parasites/fasciola/|website = www.cdc.gov|access-date = 2015-06-20}}</ref>。
* [[肝炎|肝炎]]是指肝臟的發炎，可能因為病毒造成（{{tsl|en|viral hepatitis|病毒性肝炎}}），也有可能因為{{tsl|en|hepatotoxicity|肝毒性}}物質造成（例如{{tsl|en|alcoholic hepatitis|酒精性肝炎}}），或是因為免疫（{{tsl|en|autoimmune hepatitis|自體免疫性肝炎}}）或是遺傳所造成<ref>{{Cite web|title = Hepatitis: MedlinePlus|url = https://www.nlm.nih.gov/medlineplus/hepatitis.html|website = www.nlm.nih.gov|access-date = 2015-06-20}}</ref>。
*{{tsl|en|Alcoholic liver disease|酒精性肝病變}}是因為{{tsl|en|Alcohol abuse|酒精濫用}}造成的肝臟表現，包括[[脂肪肝|脂肪肝]]、酒精性肝炎及[[肝硬化|肝硬化]]。像是「藥物引發」肝病或是「毒性」肝病的類似詞語則用來描述因為藥物引發的肝病<ref>{{Cite web|title = Alcoholic liver disease: MedlinePlus Medical Encyclopedia|url = https://www.nlm.nih.gov/medlineplus/ency/article/000281.htm|website = www.nlm.nih.gov|access-date = 2015-06-20}}</ref>。
* [[脂肪肝|脂肪肝]]（肝臟{{tsl|en|steatosis|脂肪變性}}）是一種可逆的症狀，是指大量[[三酸甘油酯|三酸甘油酯]]空泡累積在肝細胞中<ref>{{Cite web|title = Hepatic steatosis|url = http://medical-dictionary.thefreedictionary.com/Hepatic+steatosis|access-date = 2015-06-20}}</ref>。{{tsl|en|Non-alcoholic fatty liver disease|非酒精性脂肪肝}}是和肥胖及[[代謝症候群|代謝症候群]]有關的疾病<ref>{{Cite web|title = Non-alcoholic fatty liver disease – NHS Choices|url = http://www.nhs.uk/conditions/fatty-liver-disease/Pages/Introduction.aspx|website = www.nhs.uk|access-date = 2015-06-20}}</ref>
* 會損害肝臟的{{tsl|en|Hereditary disease|遺傳性疾病}}包括會在體內累積[[铁|铁]]質的[[血色沉着病|血色沉着病]]<ref>{{Cite web|title = Hemochromatosis: MedlinePlus|url = https://www.nlm.nih.gov/medlineplus/hemochromatosis.html|website = www.nlm.nih.gov|access-date = 2015-06-20}}</ref>以及[[肝豆狀核變性|肝豆狀核變性]]。肝臟損害也是{{tsl|en|alpha 1-antitrypsin deficiency|α1-抗胰蛋白酶缺乏症}}<ref>{{Cite web|title = Alpha-1 Antitrypsin Deficiency: MedlinePlus|url = https://www.nlm.nih.gov/medlineplus/alpha1antitrypsindeficiency.html|website = www.nlm.nih.gov|access-date = 2015-06-20}}</ref>以及[[龐貝氏症|龐貝氏症]]<ref>{{Cite book|title = Glycogen Storage Disease Type II (Pompe Disease)|url = https://www.ncbi.nlm.nih.gov/books/NBK1261/|publisher = University of Washington, Seattle|date = 1993|location = Seattle (WA)|pmid = 20301438|first = Nancy|last = Leslie|first2 = Brad T.|last2 = Tinkle|editor-first = Roberta A.|editor-last = Pagon|editor-first2 = Margaret P.|editor-last2 = Adam|editor-first3 = Holly H.|editor-last3 = Ardinger|editor-first4 = Stephanie E.|editor-last4 = Wallace|editor-first5 = Anne|editor-last5 = Amemiya|editor-first6 = Lora JH|editor-last6 = Bean|editor-first7 = Thomas D.|editor-last7 = Bird|editor-first8 = Cynthia R.|editor-last8 = Dolan|editor-first9 = Chin-To|editor-last9 = Fong}}</ref>的臨床特徵。
* 在[[家族性澱粉樣物多發性神經病變|家族性澱粉樣物多發性神經病變]]中，肝臟產生突變的轉甲狀腺素蛋白，會有嚴重的神經變性及心臟病效應。肝臟移植是其中一種治療方式<ref>{{Cite web|title = Transthyretin amyloidosis|url = http://ghr.nlm.nih.gov/condition/transthyretin-amyloidosis|website = Genetics Home Reference|access-date = 2015-06-20}}</ref>。
* [[吉爾波特症候群|吉爾波特症候群]]是有關[[胆红素|胆红素]]代謝的基因性疾病，在少部份人身上會出現，會造成輕微[[黄疸|黄疸]]<ref>{{Cite web|title = Gilbert syndrome|url = http://ghr.nlm.nih.gov/condition/gilbert-syndrome|website = Genetics Home Reference|access-date = 2015-06-20}}</ref>。
* [[肝硬化|肝硬化]]是指肝臟中形成纖維組織（[[纤维化|纤维化]]），取代肝臟中死亡的肝細胞，肝細胞死亡的原因包括病毒性肝炎、飲酒過量、其他肝毒性物質。肝硬化會導致慢性[[肝衰竭|肝衰竭]].<ref>{{Cite web|title = Cirrhosis: MedlinePlus Medical Encyclopedia|url = https://www.nlm.nih.gov/medlineplus/ency/article/000255.htm|website = www.nlm.nih.gov|access-date = 2015-06-20}}</ref>。
* 原發性[[肝癌|肝癌]]最見的有[[肝細胞癌|肝細胞癌]]及[[膽管癌|膽管癌]]，而肝{{tsl|en|angiosarcoma|血管肉瘤}}比較少見（許多肝臟惡性腫瘤是從腸胃道或是肺臟、腎臟原發癌症轉移的繼發性病灶）<ref>{{Cite web|title = Liver cancer – Hepatocellular carcinoma: MedlinePlus Medical Encyclopedia|url = https://www.nlm.nih.gov/medlineplus/ency/article/000280.htm|website = www.nlm.nih.gov|access-date = 2015-06-20}}</ref>。
* {{tsl|en|Primary biliary cirrhosis|原發性膽汁性肝硬化}}是{{tsl|en|bile canaliculus|膽小管}}的嚴重{{tsl|en|autoimmune|自身免疫性}}疾病<ref>{{Cite web|title = Primary biliary cirrhosis: MedlinePlus Medical Encyclopedia|url = https://www.nlm.nih.gov/medlineplus/ency/article/000282.htm|website = www.nlm.nih.gov|access-date = 2015-06-20}}</ref>。
* [[原發性硬化性膽管炎|原發性硬化性膽管炎]]是嚴重的慢性[[膽管|膽管]][[炎症|炎症]]，可能和器官的自身免疫性有關<ref>{{Cite web|title = Sclerosing cholangitis: MedlinePlus Medical Encyclopedia|url = https://www.nlm.nih.gov/medlineplus/ency/article/000285.htm|website = www.nlm.nih.gov|access-date = 2015-06-20}}</ref>。
* [[布加综合征|布加综合征]]是因為{{tsl|en|hepatic vein|肝静脉}}{{tsl|en|Vascular occlusion|血管闭塞|闭塞}}造成的臨床症狀<ref>{{Cite web|title = Hepatic vein obstruction (Budd-Chiari): MedlinePlus Medical Encyclopedia|url = https://www.nlm.nih.gov/medlineplus/ency/article/000239.htm|website = www.nlm.nih.gov|access-date = 2015-06-20}}</ref>。
