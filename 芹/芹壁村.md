'''芹壁村'''是一處位於[[中華民國|中華民國]][[馬祖列島|馬祖北竿島]]西部的村落，行政上屬於[[連江縣_(中華民國)|連江縣]][[北竿鄉|北竿鄉]]。芹壁舊名為鏡港，又稱鏡澳，在早期是一座小漁村，村裡的芹壁天后宮香火鼎盛，為當地重要信仰中心之一。此村保留有許多馬祖早期特色古厝，類似於地中海風號格，因此又被稱為「馬祖地中海」。<ref name="mediterranean">{{cite web |url= http://www.matsu-nsa.gov.tw/User/Attraction-Content.aspx?a=53&lang=1 |title=芹壁聚落 |publisher= 馬祖國家風景區 |access-date= 2014-12-22 }}</ref>另村莊也有步道，可直達[[上村_(北竿鄉)|上村]]。<ref name="matsu-nsa">{{cite web |url= http://www.matsu-nsa.gov.tw/Backpacker/Article.aspx?a=539 |title=芹壁步道 |publisher= 馬祖國家風景區管理處|access-date= 2014-12-22 }}</ref>村外以北海域不遠處有一座名為[[龜島_(連江縣)|龜島]]的島嶼，在芹壁村海岸即可觀見。<ref name="mediterranean"/>

==交通==
芹壁村位於環島北路中途段，往北連接[[橋仔村|橋仔村]]，往南則是到坂里水庫。<ref>{{cite news |url= http://www.matsu-news.gov.tw/2010web/news_detail_101.php?CMD=open&UID=179331 |title=歷時1年　北竿環島北路改善工程　近期完工 |publisher= 馬祖日報 |date= 2016-12-12 |access-date= 2016-12-12 }}</ref>從[[塘岐村|塘岐村]]及[[白沙村_(連江縣)|白沙村]]接可搭乘公車抵達。<ref>{{cite web |url= http://www.matsu-news.gov.tw/ftp_data/n-bus.htm |title=北竿 公車站班車發車時刻表 |publisher= 馬祖日報 |access-date= 2016-12-12 }}</ref><ref>{{cite web |url= http://hotels.matsu.idv.tw/bus_nb.php |title=南、北竿公車 |publisher= 馬祖住宿通 |access-date= 2016-12-12 }}</ref>
