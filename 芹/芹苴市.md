'''芹苴市'''（{{vie|v=Thành phố Cần Thơ|hn=城庯芹苴}}<ref>汉字写法见于《[[大南实录|大南实录]]》。“{{lang|vi|苴}}”字標準讀音為“{{lang|vi|thư}}”，此處“{{lang|vi|thơ}}”為越南南方口音。</ref><ref>《[[南国地舆|南国地舆]]》又写作“勤书”，“勤”与“芹”同音，“书”与“苴”同音。</ref>）是[[越南|越南]]的5個[[直轄市|直轄市]]之一，在[[後江省|後江省]]南面，是[[越南|越南]]第四大城，也是[[湄公河三角洲|湄公河三角洲]]上最大的城市，下轄5郡4縣。離[[胡志明市|胡志明市]]約160公里。被[[安江省|安江省]]、[[後江省|後江省]]、[[堅江省|堅江省]]、[[永隆省|永隆省]]及[[同塔省|同塔省]]包圍。

==歷史==
芹苴市在[[越南共和国|越南共和国]]时期属于[[丰盈省|丰盈省]]。越南统一后，和巴川省组成[[后江省|后江省]]。1991年12月26日，越南政府通过决议，将后江省分设为[[芹苴省|芹苴省]]和[[朔庄省|朔庄省]]2省。其中，芹苴省包括今天的芹苴市和后江省。2003年11月26日，越南政府通过决议，将芹苴省分设为直辖市芹苴市和新的后江省2个省级行政单位。

== 行政區劃 ==
芹苴市分成5郡和4縣。
* [[平水郡|平水郡]]（Quận Bình Thủy）
* [[丐𪘵郡|丐𪘵郡]]（Quận Cái Răng）
* [[寧橋郡|寧橋郡]]（Quận Ninh Kiều）
* [[烏門郡|烏門郡]]（Quận Ô Môn）
* [[禿衂郡|禿衂郡]]（Quận Thốt Nốt）
* [[豐田縣_(芹苴市)|豐田縣]]（Huyện Phong Điền）
* [[紅旗縣|紅旗縣]]（Huyện Cờ Đỏ）
* [[泰來縣_(越南)|泰來縣]]（Huyện Thới Lai）
* [[永盛县_(芹苴市)|永盛縣]]（Huyện Vĩnh Thạnh）

== 氣候 ==
* 屬於熱帶季風氣候，6月~11月是雨季、12月~5月是旱季。
* 年平均濕度83%
* 年降雨量1,635mm
* 年均溫27 °C。

== 教育 ==
* 芹苴大學，以農學院聞名。
* 九龍江三角洲稻米研究學院

== 旅遊 ==
[[File:CanThoFloatingMarket.jpg|thumb]]
芹苴市以鄉村生活聞名，遊客可以品嚐當地以[[柚子|柚子]]、[[龍眼|龍眼]]、[[波羅蜜|波羅蜜]]、[[芒果|芒果]]、[[橘子|橘子]]、[[榴槤|榴槤]]等不同水果所作成的風味餐。

== 航空交通 ==
有[[芹苴機場|芹苴機場]]提供航空服務，除了有通往[[內排機場|河內]]、[[峴港國際機場|峴港]]等國內航點外，春節期間因應農曆新年[[臺灣|臺灣]]越南籍配偶返鄉習俗，[[越南航空|越南航空]]有直飛班機由[[桃園國際機場|桃園國際機場]]來往芹苴市。
