'''電磁鐵'''是可以通[[电流|电流]]來产生[[磁力|磁力]]的装置，在[[電力|電力]]普及的社會中是一項不可缺少的工具，屬[[非永久磁鐵|非永久磁鐵]]，與[[永久磁鐵|永久磁鐵]]同為[[磁鐵|磁鐵]]的一種

== 原理 ==
[[File:VFPt_Solenoid_correct2.svg|thumb]]後可產生如一[[磁鐵棒|磁鐵棒]]的[[磁場|磁場]]。圖中的圓圈為導線截面，點代表電流出螢幕，叉代表流入螢幕；附箭頭的橢圓圓圈是磁力線。]]
當[[直流電|直流電]]通過[[導體|導體]]時會產生[[磁場|磁場]]，而通過作成[[螺線管|螺線管]]的導體時則會產生類似棒狀[[磁鐵|磁鐵]]的磁場。在螺線管的中心加入一[[磁性物質|磁性物質]]則此磁性物質會被[[磁化|磁化]]而達到加強磁場的效果。

一般而言，電磁鐵所產生的磁場強度與直流電大小、線圈圈數及中心的導磁物質有關，在設計電磁鐵時會注重線圈的分佈和導磁物質的選擇，並利用直流電的大小來控制磁場強度。然而線圈的材料具有[[電阻|電阻]]而限制了電磁鐵所能產生的磁場大小，但隨著[[超導體|超導體]]的發現與應用將有機會突破現有的限制。
一般而言，線圈越多越強。

== 歷史 ==
1820年，[[丹麥|丹麥]][[物理學家|物理學家]][[漢斯·奧斯特|漢斯·奧斯特]]（1777年－1851年）首先發現載流導線的電流會產生磁場，使臨近的磁針改變方向。

[[File:Sturgeon_electromagnet.png|thumb]]
[[File:Industrial_lifting_magnet.jpg|thumb]]。]]
西元1825年，[[英國人|英國人]]{{link-en|威廉·思特金|William Sturgeon}}（William Sturgeon，1783年-1850年）將通有電流的金屬線纏繞在絕緣的棒上，發明了電磁鐵。

[[美國人|美國人]][[物理學家|物理學家]][[約瑟·亨利|約瑟·亨利]]（Joseph Henry, 1797年-1878年）在得知這個消息後，在[[軟鐵|軟鐵]]芯上纏繞密集的線圈，使用電流不大的電池通電後，便能吸起一[[噸|噸]]重的[[鐵|鐵]]塊。

== 特性 ==
* 電磁鐵的磁力大小可以改變。
* 電磁鐵的磁極方向可以改變。
* 電磁鐵可以吸引鐵、鈷、鎳等製品。
