'''电导率'''（{{lang-en|electric conductivity}}）是表示物质传输[[电流|电流]]能力强弱的一种測量值。當施加[[電壓|電壓]]於[[導體|導體]]的兩端時，其[[電荷載子|電荷載子]]會呈現朝某方向流動的行為，因而產生電流。電導率 <math>\sigma\,\!</math> 是以[[歐姆定律|歐姆定律]]定義為[[電流密度|電流密度]] <math>\mathbf{J}\,\!</math> 和[[電場|電場]]強度 <math>\mathbf{E}\,\!</math> 的比率：
:<math>\mathbf{J} = \sigma \mathbf{E}\,\!</math> 。

有些物質會有[[異向性|異向性]] ({{lang|en|anisotropic}}) 的電導率，必需用 3 X 3 [[矩陣|矩陣]]來表達（使用數學術語，第二階[[張量|張量]]，通常是[[對稱矩陣|對稱的]]）。

電導率是[[电阻率|电阻率]] <math>\rho\,\!</math> 的[[倒數|倒數]]。在[[國際單位制|國際單位制]]中的單位是[[西门子_(单位)|西門子]]/[[公尺|公尺]] (S·m<sup>-1</sup>)：
:<math>\sigma = {1\over\rho}\,\!</math> 。  

[[電導率儀|電導率儀]] ({{lang|en|electrical conductivity meter}}) 是一種是用來測量溶液電導率的儀器。

== 电传导性 ==
'''电传导性''' ({{lang|en|electric conductivity}}) 是物质可以传导[[电子|电子]]的性质。按物质是否具有电传导性，可把物质分为[[导体|导体]]，[[半导体|半导体]]和[[绝缘体|绝缘体]]。

* 導體：[[金屬|金屬]]、[[电解质|电解质]]溶液，一般有很高的電導率，很低的電阻率。
* 絕緣體：像[[玻璃|玻璃]]、干燥的[[木材|木材]]、[[塑料|塑料]]、[[橡胶|橡胶]]或[[真空|真空]]這類物質的電導率很低，電阻率很高。
* 半導體：電導率在導體和絕緣體之間。在不同的狀況下，電導率會有很大的變化。例如，暴露於電場或某種頻率的[[光波|光波]]，最重要地，溫度和半導體材料的成份。

固態半導體的摻雜程度會造成電導率很大的變化。增加摻雜程度會造成高電導率。[[水|水]][[溶液|溶液]]的電導率高低跟其內含[[溶質|溶質]][[鹽|鹽]]的[[濃度|濃度]]有關，或其它會分解為[[電解質|電解質]]的化學雜質。水樣本的電導率是測量水的含鹽成分、含離子成分、含雜質成分等等的重要指標。水越純淨，電導率越低（電阻率越高）。水的電導率時常以'''電導係數'''來紀錄；電導係數是水在 25°C 溫度的電導率。

== 複電導率 ==
假若物質處於隨時間而變的電場，則電導率為複數（對於各向異性物質，電導率為複矩陣），稱為[[導納率|導納率]] ({{lang|en|admittivity}}) 。這方法可以應用於[[電阻抗斷層成像|電阻抗斷層成像]] ({{lang|en|electrical impedance tomography}}) ，一種工業或[[醫學成像|醫學成像]]。

另外一種計算[[交流|交流]]的方法使用實值的電導率（跟時間有關）和實值的[[電容率|電容率]]。電導率越大，物質會越快吸收交流訊號，物質越[[透明|不透明]]。

== 電導率與溫度的關係 ==
電導率與溫度紧密相關。金屬的電導率隨著溫度的增高而降低。半導體的電導率隨著溫度的增高而增高。在一段溫度值域內，電導率可以被近似為與溫度成[[正比|正比]]。為了要比較物質在不同溫度狀況的電導率，必須設定一個共同的參考溫度。電導率與溫度的相關性，時常可以表達為，電導率對上溫度線圖的[[斜率|斜率]]，用方程式寫為
:<math>\sigma(T) = {\sigma_0 \over 1 + \alpha (T - T_0)}\,\!</math>；

其中，<math>T_0\,\!</math> 是參考溫度，<math>T\,\!</math> 是測量溫度，
<math>\sigma\,\!</math> 是物質的電導率，<math>\sigma_0\,\!</math> 是物質在參考溫度的固定參考電導率（通常在室溫），<math>\alpha\,\!</math> 是物質的'''溫度補償斜率'''。

最常見的水的溫度補償斜率大約為 2 %/°C 。但是，它可以在 (1 to 3) %/°C 值域之間。這數值與[[地球化學|地球化學]]有關，可以很容易地在實驗室裡測量出來。

在非常低溫的狀況（在[[絕對零度|絕對零度]]附近），有些物質的電導率會變得非常高。稱這現象為[[超導現象|超導現象]]。
