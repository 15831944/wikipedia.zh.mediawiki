英文為'''Video Game Development Party'''。以[[電玩|電玩]]業界的產品、對業界結構的策略來將電玩廠商進行分類。

== 第一方開發廠 ==
通常簡稱為第一方或第一廠。指自行推出[[電玩主機|電玩主機]]的電玩廠商；如[[任天堂|任天堂]]。一旦退出電玩主機開發，就不再算第一方開發廠。如[[SEGA|SEGA]]、[[雅达利|雅达利]]。

== 第二方開發廠 ==
通常簡稱為第二方或第二廠。以接受他廠委託，代工開發遊戲為主要[[業務|業務]]。多半擁有相當程度的遊戲技術以維持競爭力，但苦無資金、[[通路|通路]]甚至自家[[品牌|品牌]][[產品|產品]]。常由資深遊戲開發者成立，憑業界人脈取得[[外包|外包]]訂單。

相較於第一方開發廠的定義明確，第二方開發廠就略顯模糊，比較像是為了與前者區分才產生的分類。[[任天堂|任天堂]]尤其擅用此體系，如1986年12月<ref>[http://www.intsys.co.jp/company/history/index.html INTELLIGENT SYSTEMS CO., LTD.　沿革<!-- Bot generated title -->]</ref>由[[任天堂|任天堂]]開発第一部成立的[[Intelligent_Systems|INTELLIGENT SYSTEMS]]日後專為[[任天堂|任天堂]]陣營提供[[聖火降魔錄|聖火降魔錄]]等獨占軟體，可能是最早的第二方開發廠。日後如開發[[任天堂明星大亂鬥|任天堂明星大亂鬥]]的[[HAL研究所|HAL研究所]]、開發[[神奇寶貝|神奇寶貝]]的[[GAME_FREAK|GAME FREAK]]也都由[[任天堂|任天堂]]扶植，解決90年代末自家[[電玩主機|電玩主機]]欠缺[[軟體|軟體]]支援的問題 。

在[[SCE|SCE]]陣營則稱此為衛星公司（{{lang-en|satellite company}}，{{lang-ja|サテライトカンパニー}}）<ref>[http://www.passion-ceo.net/contents/hp0043/index.php?No=885&CNo=43 ゲーム業種の情熱社長|INTI CREATES社長專訪<!-- Bot generated title -->]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。如[[LEVEL-5|LEVEL-5]]。

因為欠缺獨立性不利於發展，所以除非已被第一方開發廠大量持股或與他廠關係密切，否則成氣候的第二方開發廠往往會推出自家產品打響品牌<ref>[http://www.1up.com/features/the-man-who-created-street-fighter 1up: The Man Who Created Street Fighter|第二方開發廠Dimps社長因轉型接受專訪<!-- Bot generated title -->]</ref>。轉型為第三方開發廠。

== 第三方開發廠 ==
{{main|游戏开发者#第三方开发商}}
通常簡稱為第三方或第三廠。在策略上獨立自主，以加入[[電玩主機|電玩主機]]陣營販售自家[[產品|產品]]為[[業務|業務]]的一般獨立[[軟體|軟體]]廠。

成功的第三方開發廠往往有知名的遊戲系列，甚至在加入[[電玩主機|電玩主機]]陣營能從第一方開發廠取得種種優惠。如當年[[哈德森|哈德森]]（HUDSON）、[[南夢宮|南夢宮]]（NAMCO）、[[太東|太東]]（TAITO）、[[卡普空|卡普空]]（CAPCOM）、[[杰力可|杰力可]]（JALECO）和[[科樂美|科樂美]]（KONAMI）六家软件开发商加入开发[[FC遊戲|FC遊戲]]，就享有自行生產卡匣等後進難望其項背的優惠，被当时业界称为“六大软件商”。

== 爭議 ==
以歷史來看，第三方開發廠可能還早於第二方開發廠。原本在[[電腦|電腦]]業界第一方是指硬體主機廠商，第二方是消費者，第三方是軟體週邊廠商。再從[[任天堂|任天堂]]後來縮減六大软件商優惠，活用第二方開發廠來看，這種在硬體廠與軟體廠間重新劃分的分類可能是為了形容[[任天堂|任天堂]]的戰略佈局對業界的影響而日漸被定義下來的。

另外，第三方開發廠亦有接單代工他廠遊戲的時候，如[[From_Software|From Software]]替[[萬代南夢宮|萬代南夢宮]]開發[[異世紀機器人大戰系列|異世紀機器人大戰系列]]。或[[Arc_System_Works|Arc System Works]]替[[CAPCOM|CAPCOM]]開發[[戰國BASARA|戰國BASARA]] X。
