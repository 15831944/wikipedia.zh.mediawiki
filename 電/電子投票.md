'''電子投票'''，是一種投票設備，使用者直接按鈕，就可以投票給自己所支持的候選人。

== 優點 ==
* 可以減少排隊時間
* 可以使得投票更方便
* 可以使開票結果更正確
* 可以使開票更快
* 在計票方面可以減少人力
* 行動不方便者，可以在家使用

== 缺點 ==
* 可能有不法人士篡改程式，導致開票結果不正確。引起[[選舉舞弊|選舉舞弊]]的爭議。<ref>{{cite web | url=https://www.hbarel.com/analysis/cyber/secure-e-voting-is-hard-to-get | title=Why secure e-voting is so hard to get | author=Hagai Bar-El | deadurl=no | archiveurl=https://web.archive.org/web/20150912070825/https://www.hbarel.com/analysis/cyber/secure-e-voting-is-hard-to-get | archivedate=2015-09-12 | df= }}</ref>
* 無法留存實體證明，有爭議時不能請公正人士重新驗票。
* 非現場投票者，投票人易受他人監視控制，失去一人一票的[[公平|公平]]性。
* 連線投票的確認身份也等於可能失去[[匿名|匿名]]性。
* 監票困難。
* 造價昂貴。
* 原本在排隊期間，資源較少的小黨派可以宣揚自己的政見，現在則不行，各國投票所也已經逐漸不允許此類行為。
* 選民思考要投給誰的時間較短。
* 選民選前若未研究議題，現場投的時間依然沒縮短。
