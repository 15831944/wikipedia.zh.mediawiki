'''電阻器'''（Resistor），泛指所有用以產生[[電阻|電阻]]的[[電子|電子]]或[[電機|電機]]配件。電阻器的運作跟隨[[歐姆定律|歐姆定律]]，其'''電阻值'''定義為其[[電壓|電壓]]與[[電流|電流]]相除所得的[[比值|比值]]。

:<math>R = {V \over I}</math>

其中
:I是流過[[導體|導體]]的電流，單位是[[安培|安培]]
:V是導體兩端的電位差，單位是[[伏特|伏特]]
:R是導體的電阻，單位是[[歐姆|歐姆]]

電阻器是電子電路中常見的元件，實際的電阻可以由許多不同材質構成，包括薄膜、水泥或是高電阻系數的鎳鉻合金（{{link-en|電阻絲|resistance wire}}）。電阻器也可整合到[[積體電路|積體電路]]中，特別是類比IC，也可以整合到混合式集體電路或印刷電路中。

電阻器的機能可以用其電阻來表示，常用的電阻器阻值範圍超過9個[[數量級|數量級]]。電阻器阻值有一定的誤差範圍，在電子電路中使用電阻器時，需考慮使用電阻器的允許誤差和應用是否符合，若是一些精密的電路，可能也需要考慮電阻器的[[溫度係數|溫度係數]]。電阻器也會標示其最大[[功率|功率]]，此數值需大於電阻器在電路中預期的能量消耗，尤其在電力電子應用中更需考慮。大功率的電阻器一般會需要[[散熱片|散熱片]]。在高壓電路中也需考慮電阻器可承受的最大電壓，電阻器的工作電壓一般沒有下限，但電阻器的電壓若超過其最大電壓，可能在電流流過時使電阻器燃燒。

[[File:Real-Sch-Rezistoru.png|right]]

實際的電阻器會有串聯的雜散[[電感|電感]]及並聯的雜散[[電容|電容]]。在高頻應用時這些規格就相當重要。在[[低噪音放大器|低噪音放大器]]或是{{link-en|前級擴大機|pre-amp}}的應用中，電阻的[[雜訊|雜訊]]也需要考慮。電阻器的雜散電感、雜訊及溫度係數都和電阻器製造商使用的技術有關。一般廠商生產的一系列電阻器會使用某特定技術，不會針對個別電阻器標示使用的技術。一系列電阻器也可能以其形狀因數來區分，也就是零件的大小，以及引腳或端子的位置，這些在實際電路板佈線時都需考慮到。

電阻器的主要用途包括：
* 控制某一部份電路的電壓和電流[[比例|比例]]。<br />如果該段電路的電壓是固定的，電阻器能製造固定電流；<br />如果該段電路的電流是固定的，則電阻器能製造固定的電壓。
* 分配電路不同部份的電壓比例。
* 限制流經某一段電路的電流。
* 釋放[[熱能|熱能]]。發熱線便是根據電阻器的這個特性而產生出來的。
* 通过电阻器自身的一些特性，釆集环境信息。比如根据阻值随温度变化特性，来釆集环境温度。

== 理想電阻器 ==
在一個理想的電阻器裏，電阻值不會隨電壓或電流而改變，亦不會因電流的突然變動而改變。真實的電阻器無法達到這一點。現今的內部設計使電阻器在極端的電壓或電流（以至其他環境因素，例如[[溫度|溫度]]）下能表現相對小的電阻值變化。

=== 現實電阻器的限制 ===
{{see also|定值電阻}}
每一個電阻器均有其承受的電壓或電流的上限（主要取決於電阻器的[[體積|體積]]）。如果電壓或電流超出了這個範圍，首先電阻器的電阻值會改變（在一些電阻器中可以有劇烈的變動），繼而令電阻器因過熱等情況而損毀。大部份電阻器會標示額定的電功率，另外一些則會提供額定的電流或電壓。

另外，現實的電阻器本身除電阻外，亦擁有微量的[[電感|電感]]或[[電容|電容]]，使其表現與理想的電阻器有所差異。

== 電阻器的種類 ==
根據構造，電阻可分為以下幾類：

* [[定值電阻|定值電阻]]：以帶電阻物質或[[線圈|線圈]]構成、且不會因任何環境或人為因素而變量的電阻。現時常見的定值電阻有顏色條紋用以識別電阻值、誤差等資料。定值電阻兩端多帶有連接線，以方便裝嵌；部份在[[集成電路|集成電路]]中的定值電阻屬鑲嵌形式。
* [[可變電阻|可變電阻]]：泛指所有可以手動改變電阻值的電阻器。根據使用的場合，可變電阻有'''電壓分配器'''、'''電位器'''等別稱。常見的可變電阻有''三''個連接端。不同的連接配置可使該種電阻以可變電阻、分壓計，或定值電阻的方式運作。
* [[光敏電阻|光敏電阻]]：跟隨[[光線|光線]]的強弱而改變電阻值。
* [[熱敏電阻|熱敏電阻]]：跟隨[[溫度|溫度]]的高低而改變電阻值。
* [[壓敏電阻|壓敏電阻]]：又稱變阻器，一種跟隨[[電壓|電壓]]的高低而改變電阻值的配件，通常由[[壓敏陶瓷|壓敏陶瓷]]製成。
* 一種以半導體製成的電阻器擁有[[負數|負數]]的[[電阻#溫度對電阻的影響|溫度係數]]，能紓緩電子線路中的溫度影響。
* 除[[超導體|超導體]]以外的所有[[导电体|导电体]]均帶有一定電阻。

== 電阻器的標示 ==
電阻器的標示方式會隨其種類而不同，若是功率較大的水泥電阻或繞線電阻，會在外殼上直接印出阻值及功率。軸向電阻會在電阻外殼用[[電阻色碼|電阻色碼]]表示阻值，而[[SMD|SMD]]電阻有些會用數字標示其阻值，若尺寸較小的，也可以用英文字母及數字來表示阻值。

較大尺寸的[[表面黏貼技術|表面黏貼]]電阻（1608或更大的）會在上面印製數字標示其阻值。一般誤差（5%）的表面黏貼電阻會用三位數的數字表示其阻值，前兩位數是阻值的前二位[[有效数字|有效数字]]，第三位為十的次方（也就是後面要加幾個零）。例如：

若阻值小於100，會直接寫100, 220, 470。最後的零表示是十的零次方，也就是1，例如：

|-

不過有時也會直接表示為''10''或''22''，以避免誤認。

若阻值小於10，會用英字文母表示小數點。例如：

== 與電阻器相關的計算 ==
=== 電阻器的能量消耗 ===
電阻器消耗能量的[[功率|功率]]是電阻器兩端的[[電壓|電壓]]和流經的[[電流|電流]]的[[乘積|乘積]]；

:<math>P = I \cdot V = I^2\cdot R = \frac{V^2}{R} </math>

上式中，P為功率（以[[瓦|瓦]]計算），I為電流（以[[安培|安培]]計算），V為電壓（以[[伏特|伏特]]計算），R為電阻（以[[歐姆|歐姆]]計算）。後兩條等式是第一條等式演化而成。

至於電阻器在一段時間消耗的總能量則可以如下的[[積分|積分]]式表示：

:<math>W = \int_{t_1}^{t_2} v(t) i(t)\, dt</math>

其中，<math>t_1</math>是起始時間，<math>t_2</math>是終結時間，W是總消耗能量。

如平均功率超出電阻器的額定功率，電阻器的電阻會偏離原有的值，繼而電阻器會因過熱而燒毀。

''有關更多的資料，可參閱[[串聯|串聯]]和[[並聯|並聯]]。''

以下是一列[[串聯|串聯]]起來的電阻器：

[[File:Resistorsseries.png|File:Resistorsseries.png]]

電路兩端的總電阻值為各電阻器的電阻之[[和|和]]，即
:<math>R = R_1 + R_2 + ... + R_n \,</math>

以下是一組[[並聯|並聯]]的電阻器：

[[File:Resistorsparallel.png|File:Resistorsparallel.png]]

由於所有電阻的電壓相同，根據歐姆定律，它們的電流與電阻屬[[反比|反比]]，故
:<math> \frac{1}{R_\mathrm{eq}} = \frac{1}{R_1} + \frac{1}{R_2} + \cdots +  \frac{1}{R_n}</math>

並聯電路的電阻關係可以雙豎線"<math>\|</math>"表示（有如平面[[幾何|幾何]]中的[[平行|平行]]關係），故兩個並聯的電阻的總電阻可以此等式表示：
:<math> R_\mathrm{eq} = R_1 \| R_2 = {R_1 R_2 \over R_1 + R_2} </math>

串聯和並聯僅為電路網絡的其中兩個模式。較複雜的電路網絡難以使用簡單的串聯、並聯的方式表示。

例如一個呈[[正立方體|正立方體]]狀的網絡，要量度兩個[[頂點|頂點]]之間的電阻，需要使用聯立方程式求解未知數的方式計算；但如立方體的每一條[[稜|稜]]均設有相同的電阻器，則任何頂點之間的電阻均為單一電阻器的六分之五。

==失效模式==
在妥善設計的電路中，電阻器的失效率一般會比半導體或[[電解電容|電解電容]]要低。電阻器的受損一般都是因為電阻器上的平均功率超過電阻器可以散熱的功率，因此造成電阻器過熱。這可能是電路以外的異常原因所造成，但常常是電路中其他零件異常所造成（例如電晶體短路）。將電阻器運作在接近其功率額定的功率，會減短電阻器的壽命．或是使其阻值發生變化。較安全的電路設計一般會選用較大功率額定的電阻，以避免此問題出現。

當過熱時，碳膜電阻器的電阻值可能會增加或是減少<ref>{{cite web
  | title = Electronic components – resistors
  | work = Inspector's Technical Guide
  | publisher = US Food and Drug Administration
  | date = 1978-01-16
  | url = http://www.fda.gov/ora/Inspect_ref/itg/itg31.html
  | accessdate = 2008-06-11

  | archivedate = 2008-04-03
  | deadurl = yes
  }}</ref>，
當在接近其額定功率運轉時，碳膜電阻器或氧化膜電阻器可能會開路，金屬膜電阻器或是繞線電阻器也可能有類似情形，但機率較低。

電阻器也可能因為機械應力或是惡劣的環境因素而失效。例如若電路未密封，可能會因濕氣造成繞線電阻的腐蝕。

現在已經知道表面贴装電阻會因為硫渗入電阻內部塗層而失效，硫會和內部的銀生成不導電的[[硫化銀|硫化銀]]，電阻值會持續昇高。抗硫及防腐蝕的電阻應用在汽車、工業甚至軍事應用。ASTM B809是一份測試元件抗硫渗入的工業標準<REF>[http://www.ptsn.net.cn/standard/std_query/show-astm-1105-1.htm ASTM B809-1995用湿硫黄蒸气（"硫黄华"）作金属涂层中孔隙率测试的测试方法]</REF>。
