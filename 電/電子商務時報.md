== 历史沿革 ==
1999年7月——《電子商務時報》網路媒體正式成立。 

2003年4月——與「[[中時集團|中時集團]]全球華文行銷知識庫」合作。 

2009年1月——與「[[聯合新聞網|聯合新聞網]] udn.com數位資訊」合作。 

2013年1月——因應社群媒體的發展潮流，推出《電子商務時報》Facebook粉絲頁。 

2013年11月——與[[國立政治大學|國立政治大學]]「服務創新電子報」合作。

2017年9月——《電子商務時報》官網第四次改版。 

== 內容服務 ==
《電子商務時報》的報導內容主要針對[[电子商务|電子商務]]領域，以及與其相關的[[數字科技|數位科技]]、[[网络营销|網路行銷]]等議題。透過提供讀者全方位的電商資訊，讓讀者對電子商務的發展能有更全面的掌握，其主要內容涵蓋如下<ref>{{Cite web|url=http://www.ectimes.org.tw/aboutus.aspx|title=電子商務時報．Electronic Commerce Times|accessdate=2018-10-21|work=www.ectimes.org.tw}}</ref>：

* 產業動態：報導每日國內外重要的電子商務、數位科技、傳播行銷等面向之相關資訊。
* 專題報導（週報）：選取近期最具話題性的電商熱點，進行專題式深入報導。
* 獨家觀點：由特約專家針對電商領域的熱門議題，提出個人觀點及評論。
* 活動訊息：提供國內外與電子商務相關之學術研討會、演講等活動訊息。

== 團隊編制 ==
在組織結構上，《電子商務時報》共設有[[總編輯|總編輯]]、副總編輯、[[财务总监|財務長]]、公關長、教學長等職，並由資訊管理學系（所）組建工程師團隊提供技術支持。

由於該媒體的創辦人梁定澎（[[國家講座|國家講座]]教授）<ref name=":0" />為行銷傳播管理研究所（原傳播管理研究所）首任所長<ref>{{Cite web|url=http://imc.nsysu.edu.tw/files/15-1104-109644,c13280-1.php?Lang=zh-tw|title=系所簡史 - 國立中山大學行銷傳播管理研究所|accessdate=2018-10-22|work=imc.nsysu.edu.tw|language=zh-tw}}</ref>，故而長期提供該所的學生參與實習，主要負責新聞採編之工作<ref>{{Cite web|url=http://imc.nsysu.edu.tw/files/15-1104-150046,c15203-1.php?Lang=zh-tw|title=實習 - 國立中山大學行銷傳播管理研究所|accessdate=2018-10-21|work=imc.nsysu.edu.tw|language=zh-tw}}</ref>。
