'''電源成本'''（{{lang-en|Cost of electricity by source}}），是一個計算在[[輸電網路|輸電網路]]之中，各種[[發電|發電]]系統所產生的總容量與它們產生電力的成本比例。它們包括建造[[發電廠|發電廠]]的初期成本與[[折現率|折現率]]，持續運轉時的[[營業成本|營業成本]]（[[燃料|燃料]]費用、[[維護|維護]]費用），以及多年後停用發電廠時的最終善後處理費用。這類的計算，協助決策者，研究人員和其他引導討論和決策。

==成本因素==
===自營運電廠===
*初期費用
*:發電廠建造所需土地與建造費用（設計、環評、興建、監造）
*运营成本
*:燃料費與廢棄物處理費用、維修費和人事費用，設備折舊與初期費用按使用年限攤提，以及預先攤提善後費用
*善後費用
*:永久關閉發電廠所需費用

===外購電源===

==計算==
<math>
\mathrm{LEC} = \frac{\sum_{t=1}^{n} \frac{ I_t + M_t + F_t}{\left({1+r}\right)^t} }{\sum_{t=1}^{n} \frac{E_t}{\left({1+r}\right)^{t}} }
</math>

* <math>\mathrm{LEC}</math> = 發電成本均價/平均壽命
* <math>I_t</math> = 在''t''年的投資支出
* <math>M_t</math> = 在''t''年的操作和維護支出
* <math>F_t</math> = 在t年的燃料費支出
* <math>E_t</math> = 在t年發電
* <math>r</math>   = 年度實際[[貼現率|貼現率]]
* <math>n</math>   = 系統使用壽命
