《'''電影版妖怪手錶：光影之卷 鬼王的復活'''》（{{lang-ja|映画 妖怪ウォッチ シャドウサイド 鬼王の復活}}）是《[[妖怪手錶|妖怪手錶]]》[[劇場版|劇場版]]系列的第四部作品。《[[墓場鬼太郎|墓場鬼太郎]]》的角色也跨界出演電影中<ref>{{Cite web |url=https://www.cinematoday.jp/news/N0094384|title=妖怪ウォッチ×ゲゲゲが映画で実現！鬼太郎役に初代声優・野沢雅子が復帰|publisher=cinematoday|date=2017-09-13|accessdate=2019-02-23|language=ja}}</ref><ref>{{Cite web |url=https://natalie.mu/comic/news/248558|title=鬼太郎が映画「妖怪ウォッチ」に登場！ちゃんちゃんこ姿の鬼太ニャンも|publisher=[[Natalie_(網站)|Natalie]]|date=2017-09-13|accessdate=2019-02-23|language=ja}}</ref>。日本於2017年12月16日上映<ref>{{Cite web |url=https://www.cinematoday.jp/news/N0092220|title=映画『妖怪ウォッチ』第4弾、主人公らキャラ一新！30年後の物語に|publisher=cinematoday|date=2017-06-15|accessdate=2019-02-23|language=ja}}</ref><ref>{{Cite web |url=https://eiga.com/news/20170615/15/|title=「妖怪ウォッチ」主要キャラ激変！劇場版第4弾は“強面ジバニャン”登場ホラーコメディ|publisher=eiga|date=2017-06-15|accessdate=2019-02-23|language=ja}}</ref>。

== 票房 ==
電影在上映首周时票房为3.99億日元。<ref>https://www.animenewsnetwork.com/news/2017-12-19/yo-kai-watch-shadowside-film-debuts-at-no.2-with-399-million-yen/.125448</ref> 在第二和第三周时居票房排行榜第二位。<ref>https://www.animenewsnetwork.com/news/2018-01-01/yo-kai-watch-shadowside-anime-film-stays-at-no.2-in-2nd-weekend-at-box-office/.125952</ref><ref>https://www.animenewsnetwork.com/news/2018-01-14/yo-kai-watch-shadowside-anime-film-stays-at-no.2-in-3rd-weekend-at-box-office/.126164</ref><ref>https://www.animenewsnetwork.com/news/2018-01-14/love-chunibyo-and-other-delusions-take-on-me-anime-film-opens-at-no.6/.126387</ref> 时间截至至第八週时，电影總銷售額超過1,863,413,400日元。<ref>https://www.animenewsnetwork.com/news/2018-01-23/mazinger-z-infinity-falls-to-no.9-yo-kai-watch-to-no.8-at-japanese-box-office/.126801</ref>
