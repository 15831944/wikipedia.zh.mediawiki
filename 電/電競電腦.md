'''電競電腦'''（Gaming computer），又稱'''電競配備'''<ref name="appledaily37130464">{{cite news|title=電競配備優 遊戲更順手|url=http://www.appledaily.com.tw/appledaily/article/finance/20160327/37130464/|agency=蘋果日報|date=2016-03-27}}</ref>，也是{{link-en|硬體效能評測|Benchmark (computing)}}的一種評測級別及廠商標示的一種產品級別，稱作'''電競級'''<ref>{{cite news|title=PC核心零組件 掛上「電競級」成最佳識別|url=http://www.digitimes.com.tw/tw/dt/n/shwnws.asp?CnlID=13&packageid=6095&id=0000286032_PV21HFDS2OUE7830KYG51&cat=40&ct=1|agency=DIGITIMES中文網|date=2012-06-05}}</ref>，屬於[[個人電腦|個人電腦]]的一種，該種電腦在內部配備的[[硬體|硬體]]是針對运算需求高的[[电子游戏|电子游戏]]及[[電子競技|電子競技]]玩家而设计。電競電腦與一般电脑的区别在于通常配有較高階的硬體。

== 配備標準 ==
電競級的配備標準是因各家硬體效能評測排行而定，或是因各家廠商的產品標示而定，但不見得可信，因為沒有公認的判定標準。

=== 內部 ===
例如有高階[[處理器|處理器]]或可支援[[超頻|超頻]]、一張或多張高階[[显示卡|显示卡]]或可支援輸出至[[液晶電視|液晶電視]]、具有[[多通道記憶體技術|多通道記憶體技術]]的[[主機板|主機板]]、機箱內裝有[[電腦水冷|水冷式系統]]協助超頻運作、硬體通過VR ready認證等等。

=== 外部 ===
例如有USB[[遊戲手把|遊戲手把]]及特殊的電競[[機箱|機箱]]、電競[[鍵盤|鍵盤]]、電競[[滑鼠|滑鼠]]與電競[[耳機|耳機]]等等，這些產品常見為黑色系主題的特殊造型設計，並搭上[[背光|背光]]<ref name="appledaily37130464" />。

== 與電子遊戲機的關係 ==
電競電腦的競爭對手為[[電子遊戲機|電子遊戲機]]。

== 與家用電腦的關係 ==
[[家用電腦|家用電腦]]同樣是以玩遊戲為特色的電腦，在1980年代日益普及，後來被淘汰。

== 與高配電腦的差別 ==
電競電腦著重游戏性，而高配电脑著重性能最大化，并将游戏作为硬體效能評測的對象。二者的不同导致配備開销差别很大，常见的误解是電競電腦等于昂贵的高配电脑，而实际上显示卡厂商大部分利润源自中低端产品<ref>{{cite web |last=D. |first=Pablo |url=http://www.gameplanet.co.nz/features/132550.20081110.GPU-battle-4670-vs.-9600GSO/ |title=GPU battle: 4670 vs. 9600GSO |date=2008-10-11 |accessdate=2011-01-21 |publisher=[[Gameplanet_(New_Zealand)|Gameplanet]]}}</ref>。
