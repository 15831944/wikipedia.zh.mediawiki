閱讀器又稱為[[電子閱讀器|e-Reader]]。簡單地說，是一種用來閱讀[[電子書|電子書]]的可攜式電子裝置<ref>{{cite web |author1=BOOX.HK |title=電子書閲讀器知道更多一些 |url=http://www.boox.hk/article/know_more_about_ereader.html |website=www.boox.hk |publisher=BOOX.HK}}</ref>。

== 定義 ==
[[File:WikiReader013.png|thumb]]在[[經濟部工業局|經濟部工業局]]數位學習與數位典藏產業推動計劃中，將電子書載具定義為「學習終端」的一種，指涉應用於學習或因應學習需求所開發之學習用的終端系統或產品。[http://jlis.lis.ntu.edu.tw/article/v10-2-5.pdf 大學生閱讀行為與電子書閱讀器需求之初探研究]

电子阅读器是专门用于显示以书籍、杂志、报纸和其他印刷品为来源的书面材料的数位版本的便携式、低能耗设备<ref>{{cite web |author1=BOOX.HK |title=電子書閲讀器知道更多一些 |url=http://www.boox.hk/article/know_more_about_ereader.html |website=www.boox.hk |publisher=BOOX.HK}}</ref>。目前电子阅读器的主要产品有：亚马逊的Kindle、索尼的Reader series、Barnes & Noble出品的Nook、中国汉王的电纸书，还有一些设计目的旨在电子阅读市场的多用途设备，类似en Tourage的eDGe阅读器和苹果公司的iPad等。<ref>《新媒体概论》中国人民大学出版社匡文波</ref>

== 歷史發展 ==
'''2009年以来，电子阅读器在全球市场由温和发展转变为快速的發展。'''

⒈2009年2月9日，亚马逊发布了Kindle升级版Kindle2，它采用16级灰度显示屏，内置2GB容量，能够存储1500本全文字格式的书籍。

⒉同年，汉王科技股份有限公司“一号工程”的汉王电子阅读器10余款产品销量占领了[[中華人民共和國|中國]]电子阅读器95%的市场份额。汉王成为继亚马逊之后，全球排名第二的电子阅读器厂商。{{請求來源}}

某些研究指出手机阅读更有生命力。<ref>《新媒体概论》中国人民大学出版社匡文波</ref>

== 使用方式 ==
基于电子纸技术的电子阅读器（E-paper Based E-book Reader）是一种平板式阅读器。大部分的电子阅读器相当于一本薄薄的平装书，能储存约上百本电子图书，具有重量轻、容量大、电池使用时间长、屏幕大等特点，是办公无纸化的新选择。部分电子阅读器具有调节字体大小的功能，并且能显示JPEG、GIF格式的黑白图像和Word文件，实现RSS新闻订阅。

电子纸显示屏通过反射环境光达到可视效果，因此看上去更像普通纸张。这种显示屏的能效非常高，因为它一旦开启，就不再需要电流来维持文字的显示，而只有翻页时才消耗能量。<ref>《新媒体概论》中国人民大学出版社匡文波</ref>
