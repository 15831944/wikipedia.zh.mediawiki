'''電椅'''（Electric Chair）是[[美國|美國]]於20世紀經常使用的一項[[死刑|死刑執行]]工具。

== 歷史 ==
電椅的發明者名義上是{{tsl|en|Alfred P. Southwick|阿尔弗雷德·索思威克}}（Alfred P. Southwick），他曾受僱於[[托马斯·爱迪生|愛迪生]]。實際上就是愛迪生本人發明的。爱迪生發明电椅就是为了打击对手尼古拉·特斯拉的[[交流电|交流电]]公司——亦即[[西屋公司|西屋公司]]，为了证明交流电的危害性，最早他以[[狗|狗]]、[[猫|猫]]、[[猴子|猴子]]、[[马|马]]當成測試品，最後证明了交流电的危险性，他宣布：“交流电只适于处死待领场的狗或[[屠宰场|屠宰场]]的动物，还有处决死囚。”但最后，他的[[直流电|直流电]]公司还是倒闭了，因為交流电在使用上适合远距离传输，比直流电更具竞争优势。

[[美国最高法院|美国最高法院]]经过討論，认定这种新的行刑方式没有违背[[美国宪法|美国宪法]]八项修正案关于禁止“残酷和非人道处罚”的原则。1889年6月4日美國[[紐約州|紐約州]]政府於正式採用電椅作為處決工具，1890年8月6日美國奥本监狱執行了第一例電椅死刑，[[威廉·凯姆勒|威廉·凯姆勒]]因殺死自己的女友，成為第一位電椅受刑人。

威廉·凯姆勒在行刑前曾对监狱长说：“別着急，好好幹！”凯姆勒被通上第一次电流，电压为1000伏，凯姆勒渾身颤抖，十七秒後仍然活着；很快地，凯姆勒第二次被通上电流，电压這次为2000伏，身上的[[肌肉|肌肉]]开始烧焦，但仍然活着。第三次通過电流時，凯姆勒終於一命嗚呼，他全身幾乎被烧成了[[炭|炭]]，[[血管|血管]]破裂，發出惡臭。一名目睹這次行刑的[[記者|記者]]表示：“這是個可怕嚇人的場面，遠遠比起[[絞死|絞死]]糟糕。”西屋公司其後表示：“相信[[斬首|斬首]]應該會比這好一點。”一家报纸声称：“交流电无疑将使[[绞刑|绞刑]][[刽子手|刽子手]]在纽约州失业。”有趣的是，電燈是1879年在紐約被發明的，兩者僅相差10年。

== 處決過程 ==
顧名思義，電椅是以[[电流|電流]]使犯人因[[觸電|觸電]]而死亡的工具。

在使用電椅時，犯人會坐在椅上，頭和腿上各拴著一個[[電極|電極]]。當行刑者為電椅通電後，約8[[安培|安培]]的電流會通過犯人，使其內臟受損而死。

在理論上，犯人在電流通過後的極短時間內失去知覺，但有時電椅本身或用以提供電流的[[變壓器|變壓器]]會被強大電流所破壞，而犯人則痛苦地在地上呻吟。曾有犯人因此向[[法院|法院]]上訴，指自己已受刑而要求獲釋。但上訴遭到駁回，犯人在一年後終再被電椅處死。

== 衰落 ==

由於以電椅處決犯人常遺下燒焦的屍體及一陣惡臭，容易令主事者不安，[[美國|美國]]於1980年代開始便以[[毒針注射|毒針注射]]等方式取代電椅。

直至2004年為止，[[美國|美國]]僅有6個州容許以電椅作為執行[[死刑|死刑]]的方法。

截至2014年，電椅剩下是在[[阿拉巴馬州|阿拉巴馬州]]，[[佛羅里達州|佛羅里達州]]，[[南卡羅來納州|南卡羅來納州]]和[[維珍尼亞州|維珍尼亞州]]這4個州執行的可選形式。

美國東岸時間2013年1月16日（周三）晚上9時08分，葛里森（Robert Gleason）被送至電椅處決。這是美國2013年首度執行死刑，也是自2010年以來，再度用電椅處決死囚。
