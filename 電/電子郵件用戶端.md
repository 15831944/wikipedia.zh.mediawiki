'''電子郵件用戶端'''通常指使用[[IMAP|IMAP]]/[[APOP|APOP]]/[[POP3|POP3]]/[[SMTP|SMTP]]/[[ESMTP|ESMTP]]/协议收发[[电子邮件|电子邮件]]的[[软件|软件]]。用户不需要登入邮箱就可以收发邮件。

== 電子郵件用戶端 ==
{{seealso|电子邮件客户端列表}}
世界上有非常多种著名的邮箱客户端，比如[[Microsoft|Microsoft]] [[Microsoft_Office|Office]]的[[Outlook|Outlook]]、[[Mozilla|Mozilla]] [[Mozilla_Thunderbird|Thunderbird]]、[[The_Bat!|The Bat!]]、[[Becky!|Becky!]]以及中国的客户端三剑客[[FoxMail|FoxMail]]、[[Dreammail|Dreammail]]和[[KooMail|KooMail]]等。

== 標準 ==
接收電子郵件的常用協議是[[POP3|POP3]]和[[IMAP|IMAP]]，發送電子郵件的常用協議是[[SMTP|SMTP]]。

另一個大部分郵箱客戶端支持的重要標準是[[MIME|MIME]]，它是用來發送[[電子郵件附件|電子郵件附件]]的。

== 限制客户端的使用 ==
由于客户端通常不能带来网页版邮箱的广告点击，因此很多邮箱提供商开始限制邮箱客户端的使用，例如微软的[[MSN|MSN]]和[[Hotmail|Hotmail]]，[[雅虎|雅虎]]的[[雅虎电邮|雅虎电邮]]（非雅虎中国），开始限制客户端的使用并且使其成为收费项目。
