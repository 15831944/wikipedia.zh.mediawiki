'''指南宮'''，位於[[台灣|台灣]][[台北市|台北市]][[文山區|文山區]][[木柵|木柵]][[指南山|指南山]]一帶山麓，為台灣[[道教|道教]]聖地之一，俗稱'''仙公廟'''，以主祀[[八仙|八仙]]中的「[[呂仙公|呂仙公]]」（即[[呂純陽|呂純陽]]、[[呂洞賓|呂洞賓]]）緣故而得名。指南宮除為道教宮殿，亦按「[[三教|三教]]合流」的理念，兼祀[[儒教|儒]]、[[佛教|釋]]。

指南宮並為[[中華道教學院|中華道教學院]]的所在地，目前籌措成立道教大學等事項。

== 歷史沿革 ==
指南宮奉祀的主神呂純陽祖師，係於光緒八年（1882），[[艋舺縣丞|艋舺縣丞]]王斌林來台履任時，[[分靈|分靈]]渡海迎奉來台。初時呂仙祖由[[艋舺|艋舺]]（今[[台北市|台北市]][[萬華區|萬華區]]）王家崇奉「玉清齋」，開堂濟世，[[扶鸞|仙鸞]]顯化，靈應著名，各地百姓紛踴前來，祈求神靈保佑，解厄消災。

其後景尾街（即今[[文山區|文山區]][[景美|景美]]）一度瘟疫，許多鄉民因而不治；仕紳自艋舺恭迎呂仙祖寶像至景尾奉祀，並結成'''肫風社'''，教化行善之理，幸得神恩庇蔭，瘟疫止息。更有雙目[[失明|失明]]而重見光明者、多年未有子息而求助呂仙公後得子者，遂令前往頂禮膜拜的信眾日多。由是，地方仕紳感於神澤廣被，倡議建廟，普獲善信支持。

乃於光緒庚寅之歲（光緒十六年，1890年），眾善信敦請[[神諭|神諭]]擇地，穴選[[木柵|木柵]]猴山坑，因而披荊斬棘，啟建山林，在現址建指南宮，此是指南宮開基立祀的根源，即今日純陽寶殿；於今已逾百廿餘載。

1920年至1959年，指南宮曾多次擴建。1991年開始，以不改變殿宇本貌為原則，指南宮進行為時七年的重建本殿工程。

== 主要建築 ==
指南宮為儒道釋[[三教合流|三教合流]]的宏偉叢林。從最早供奉呂祖的純陽寶殿外，陸續完成了大成殿、凌霄寶殿、大雄寶殿、地藏王寶殿，可謂五殿同立。

以道教為主體的純陽寶殿又稱本殿，奉祀[[呂純陽|呂純陽]]祖師，中懸著「天下第一靈山」的牌匾；凌霄寶殿內有三清殿、三官殿等奉祀[[玉皇上帝|玉皇上帝]]、[[三清道祖|三清道祖]]與[[三官大帝|三官大帝]]。

大雄寶殿又稱[[佛祖|佛祖]]殿，奉祀[[三寶佛|三寶佛]]：[[釋迦牟尼佛|釋迦牟尼佛]]、[[阿彌陀佛|阿彌陀佛]]、[[藥師佛|藥師佛]]與[[觀音菩薩|觀音菩薩]]，[[佛教|佛教]]為主體。

指南宮後山的大成殿亦稱[[孔廟|孔廟]]，為原來之清心閣所改建，奉祀[[至聖先師|至聖先師]][[孔子|孔子]]、[[孟子|孟子]]、[[曾子|曾子]]。

各殿的設計建築，巍峨雄偉，壁畫石刻，雕樑畫楝，全是名家精心傑作，充分表現中國傳統的建築藝術風格。石雕為[[張火廣|張火廣]]作品<ref>[http://nrch.cca.gov.tw/ccahome/website/site20/contents/011/cca220003-li-wpkbhisdict002457-0735-u.xml 臺灣歷史辭典 - 十一畫 - 張火廣]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

此外，還有奉祀[[土地公|土地公]]的福德祠，祠前有依地理風水而建的七星池，沿著池旁的石階往上走，可到指南宮迴廊，飽覽[[木柵|木柵]]觀光茶園的景色。而遍植奇花異卉的指南宮觀光樂園，豢養了各種珍禽異獸及天然鳥園，由於[[木柵動物園|木柵動物園]]的開放，現不對外開放。

* '''純陽寶殿'''：為主殿，主祀[[呂仙公|呂仙公]]。內有[[五陽祖師|五陽祖師]]殿、祭改壇、龍神殿。
* '''凌霄寶殿'''：[[李重耀|李重耀]]所設計的二層樓建築，主祀[[三清道祖|三清道祖]]、[[玉皇大帝|玉皇大帝]]、[[王母娘娘|王母娘娘]]與諸位[[帝君|帝君]]、[[星君|星君]]。
* '''大雄寶殿'''：為[[佛教|佛教]]道場，廣場前門依儀規設有[[天王殿|天王殿]]，即供有[[彌勒佛|彌勒佛像]]與[[四大天王|四大天王像]]。
* '''大成殿'''：主祀[[孔子|孔子]]、[[孟子|孟子]]、[[曾子|曾子]]。
* '''地藏王寶殿'''：主祀[[地藏王菩薩|地藏王菩薩]]

== 宏教及服務 ==
與[[中華民國|中華民國]][[道教總會|道教總會]]合作成立的「[[中華道教學院|中華道教學院]]」，設於凌霄寶殿內，設有圖書館、研究室、宿舍及活動中心，以文化和教育來弘揚道學。頌揚母愛的慈母像、為慎終追遠而設的往生祿位。舉行祈福大典，如1998年12月3日至12月12日進行的「護國祈安羅天大醮」，集合世界各地道教會長及會員，共同祈禱風調雨順、國泰民安。指南宮登山步道及與[[台北市|台北市]]政府合作開發位於指南宮後方的指南露營場，是休閒活動的好地方，亦是假日時各社團青年活動的極佳場所。

== 慈善服務 ==
配合政府救濟政策，興辦貧民救助，各項急難救濟，冬令賬濟，免費供膳，設置獎學金，濟助清寒學子等慈善服務工作，均不遺餘力。1995年，文山區成立老人輔導中心，指南宮專案挑起輔導事宜。

== 計劃 ==
* 1.繼成立[[中華道教學院|中華道教學院]]後，使道教精神發揚得更光大。
* 2.積極籌劃成立文化館，讓後輩能在追溯過往文化之餘，發揚新文化。
* 3.籌建一座義診醫院，服務社會大眾。

==交通方式==
前往指南宮之大眾運輸系統，除了[[貓空纜車|貓空纜車]]設站停靠其後山停車場附近外，另有往返於公館捷運站及指南宮站之530公車可以前往，平日每15~20分發車，假日每30分發車。交通尚稱便利。
旅客下車後，先以緩坡上行再接續數百階梯，即可居高臨下飽覽台北市美景，並一探成立已逾百年之天下第一靈山恢宏建築。
