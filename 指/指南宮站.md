'''指南宮站'''位於[[台灣|台灣]][[台北市|台北市]][[文山區|文山區]]，為[[貓空纜車|貓空纜車]]的纜車車站。纜車車站位於[[指南宮|指南宮]]後山附近，因而得名。

== 車站概要 ==
纜車車站位於[[指南宮|指南宮]]後山附近，因而得名。

== 車站結構 ==
高架車站，外觀呈T字型。

== 歷史 ==
* 2007年7月4日：貓空纜車指南宮站隨著貓空纜車正式通車而啟用。
* 2008年10月1日：因颱風造成基座淘空，有安全疑慮，於10月1日下午三點起無限期停駛，直到改善完成、安全無虞為止。
* 2010年3月23日~2010年3月27日：重新恢復通車測試營運，每天限定免費運行1萬人次。
* 2010年3月30日：正式恢復營運。
* 2014年6月15日：迎仙亭水池不銹鋼鍍鈦五爪金龍雕塑“金龍獻瑞”揭幕<ref>[http://fun.taipei.gov.tw/ct.asp?xitem=77617704&CtNode=13550&mp=100005 全國唯一五爪鈦金龍 現身指南風景區] 臺北市政府工務局大地工程處新聞稿、2014年6月14日</ref><ref>[http://travel.ettoday.net/article/367564.htm 貓纜新地標「金龍獻瑞」亮相　用超值票多玩一站免加價] 東森旅遊網、2014年6月14日</ref>。
