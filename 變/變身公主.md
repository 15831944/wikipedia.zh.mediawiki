《'''變身公主'''》（{{lang-ja|プリンセス・プリンセス}}）是一部以描寫男孩們之間的友情為故事軸心的[[日本|日本]][[少女漫畫|少女漫畫]]作品，作者是[[津田美樹代|津田美樹代]]（{{lang|ja|つだみきよ}}）。目前於日本新書館「月刊Wings」雜誌上連載完結，單行本全5卷。續篇『'''變身公主+（PLUS）'''』於「月刊Wings」2006年5月號到2007年1月號連載完畢，單行本全1卷。第一部於2006年被改編成[[動畫|動畫]]在日本[[朝日電視台|朝日電視台]]首播，2006年4月5日-6月21日放映完結，全12話。同年6月更被改編成[[日劇|日劇]]『PRINCESS PRINCESS D』。台灣青文出版社取得日本新書館授權台灣中文版，正式譯名為『'''變身公主'''』。
* 變身公主, 中華電信MOD, 龍華動畫台(21頻道), 2013年4/4~6/13日, 周一至周五, 下午5點到6點.

== 故事簡介 ==
有著姣好容姿的轉校生河野亨，在他轉學的第一天就被學生會推選為「公主」的候補。而所謂「公主」制度，就是挑選美麗的高一學生，請他們於學校各種重大活動時穿著女裝，而且服裝還是嬌俏的蘿莉裝、護士服或女傭服，從而滋潤男校裡枯燥的生活! 起初河野亨決拒絕，可是，當他聽聞成為「公主」的學生將可得到以金錢為主的各種優待後，便答應成為「公主」的一員。學園裡，已有較先成為「公主」的四方谷裕史郎和豐實琴，然而二人的態度卻截然不同，四方谷十分享受「公主」的樂趣，但是實琴由始至終打算從「公主」逃離。以三人作為中心，加上其他個性强烈的美型配角，搞笑的女性向學園物語就此展開。

== 主要角色 ==

===公主們===
*河野亨：[[福山潤|福山潤]] / [[佐藤健|佐藤健]]
:在故事的一開始，他以轉學生的身分轉入此一男校，並馬上成為此校的公主之一。他似乎很熱心的接下當公主的這份工作，因為當了公主後有許多的免費東西與奢侈品。當他的父母死於意外時，他的舅舅便收養了他。他似乎很不情願與他的家人相處在一起，其主要原因為他的表妹－沙耶加(Sayaka)。

:他就像裕史郎一樣利用他的公主工作來矯正他與家庭之間的關係，而其影響程度就和他當公主時對那些學校男生的影響程度是一樣的。

*四方谷裕史郎：-{[[朴璐美|朴璐美]]}- / [[藤田玲|藤田玲]]
:東邊的公主。

:他是個有自信的男生，並且在穿著公主服裝面對其他人這件事情上完全沒有問題。他取笑實琴是因為他覺得實琴的反應很好玩。他遠離他的家庭是因為他覺得他的母親、繼父和他同母異父的弟弟在一起會是個完美的家庭。

:當實琴開始抱怨公主的工作時，他和亨常常組成一組來對抗實琴。這常常讓實琴在最終終於同意他們意見，特別是當實琴不想讓他的女朋友知道他當公主一事的時候，亨與裕史郎以此一手段來對付他。

*豐實琴：[[柿原徹也|柿原徹也]] / [[鎌苅健太|鎌苅健太]]
:西邊的公主。

:他比其他兩位公主更不情願穿上公主裝，因為他不想讓他的女朋友－吉川惠知道。他是一個可靠的角色因為他時常以出固執與抱怨的態度來面對所有公主的工作，但是即使他不願意他仍然會和其他兩位公主一起去完成。當校外人士發現他是位公主時，他通常會感到非常害羞。他是最活躍的，這點可以從他吼罵亨與裕史郎的方式發現。實琴的嗓音是無音不全的，在當上公主之前他從來沒有唱過。

:他也很關心他周遭那些靠近他的朋友，尤其是他的女朋友。當他的女朋友在他身邊時，他常常會變成一個和他平常表現不同的角色，這常使得亨與裕史郎為此而開懷大笑。

:儘管他很會抱怨而且也不太情願，然而，他最後終於習慣了他公主的工作，這點可以從某一次他想像到他和恵一起穿著[[哥德蘿莉|哥德蘿莉]]風的服裝時看出。

:《變身公主》並不是實琴第一次出現在[[津田美樹代|津田美樹代]]的漫畫中，他第一次出現的漫畫是－《[[續‧革命之日|續‧革命之日]]》。其主角是實琴的女朋友－吉川惠。

===其他角色===
*-{坂本秋良}-：[[保志總一朗|保志總一朗]] / [[足立理|足立理]]
:他是亨他們班的班長（1－D）。因為他哥哥的關係再加上他自己在行為與功課方面做得不錯的緣故，所以許多學生也尊稱他為「坂本大人」。當學生們看到他時，他們都會對他鞠躬。與亨和裕史郎關係不錯。

:因為差不多同樣的原因，有定看中他為下一任的學生會會長，並讓他有一定的機會發揮他那領導的精神。

:除了他，其餘的5位家庭成員都很美麗。在過去，秋良總是被外人在背後說，他的長相和他的家人不配。這使得秋良覺得它不屬於這個家。當他的家人發現了這點以後，立即與他討論並幫他解決這一問題。自從他是個心胸寬廣和自然單純的男孩，他的兄弟姊妹（尤其是他的哥哥和姊姊）常常為了引他-{注}-意而吵架。

:秋良與他的家人在[[津田美樹代|津田美樹代]]早期的作品－《[[家族情結|家族情結]]》中為主要角色。

*-{名田庄薫}-：[[勝杏里|勝杏里]] / [[山本康平|山本康平]]

*-{坂本春海}-：[[野島健兒|野島健兒]] / ---

:基本上來說春海是個很酷與常常微笑的人。儘管如此，每當他看到他弟弟時，他總是變得很激動也很容易哭。
