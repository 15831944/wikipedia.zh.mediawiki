'''變裝'''（{{lang-en|Cross-dressing, CD}}）是人類一項受予複雜解釋的行為，指任何人為任何理由、動機穿著「被視為屬於另一個[[性別|性別]]」的服裝，有時也會搭配[[化妝|化妝]]。成語「易弁而釵」就是男扮女裝的意思。而實際是男性的[[偽娘|偽娘]]雖不一定要變裝，但也常透過這樣的裝扮來表現自己或更突顯女性美。如果連容貌也改變成特定的異性，則進入到[[易容|易容]]的範疇。

有異裝行為的人，可概稱為較中性用語的'''異裝者'''（變裝者、易裝者）；而喜歡異裝或有此癖好者，則被稱為'''異裝癖'''（易服癖、變裝癖、易裝癖、異性裝扮癖）或'''戀異性裝癖'''。許多行為雖然看似相同，但因人而異而常會有不同的成因理解和行為解釋，因此這些稱呼其明確的定義頗為模糊。常見的狀況為工作於特定職業的[[反串|反串]]行為（尤其是演員），或是對異性服裝的[[戀物癖|戀物情節]]所產生的異裝。[[跨性別者|跨性別者]]是基於[[性別認同|性別認同]]而做出認同性別的打扮，並不屬於「變裝」的範疇。

== 異裝的動機 ==
由於人、理由、性別、衣物都相當的複雜多變，因此異裝也是相當複雜多變。很多人異裝並不是為了[[性_(生物學)|性]]或是[[戀物癖|戀物癖]]的理由。同樣的，儘管異裝也是一種[[跨性別|跨性別]]的行為，不是每一個異裝的人都有跨性別的感覺或認同。同時，為達到男女平等目的而進行變裝以表達對男女不平等的不滿。甚至一種行為是否是異裝也很複雜，例如某些人認為男性穿耳洞戴[[耳環|耳環]]是異裝（他們所處環境之風氣只有女士才戴耳環而又默認之故引申出此念），某些人卻認為不是，這兩類人對異裝的定義就不同，或保持[[性別中立|性別中立]]或者[[性別酷兒|性別酷兒]]。這很視乎於社會與民俗對於衣著所造就的風氣與潮流。

異裝一詞的意義未必與理由無關。像是某些跨性別者就把異裝當作一種身份、認同。異裝一詞的意義甚至也不限於性別，也有[[Cosplay|Cosplay]]、[[角色扮演|角色扮演]]、[[化粧舞會|化粧舞會]]扮演行為等意思。於舞會、化妝舞會等類似場合上，以一特定時代地點、階級、歷史或虛擬的人物為主題的服裝打扮，也是變裝的一種。<ref>[http://db1x.sinica.edu.tw/caat/caat_rptcaatc.php?_op=?SUBJECT_ID:300224229 藝術與建築索引典－化妝舞會裝]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }} 於2011年3月31日查閱</ref>

研究異裝的理由、動機有不少問題。歷史上的異裝案例，除非是明顯與一個特定事件有關，或由異裝者刻意強調其異裝動機，通常都很難確定異裝者之動機。而且現在作出的性別分類，像是同性戀、跨性別，在過去的異裝案例上未必有意義。而且即使在現在，異裝的人的動機也很難確定，常出現說法反覆的情形。

== 有關「跨性別」之「性別認同」所作的裝扮 ==
某些人是為自身的[[性別認同|性別認同]]認同另一個性別而穿著與[[性別指定|性別指定]]不同的衣物。像是某些[[跨性別者|跨性別者]]就覺得在日常生活中與人交流時為求心安理得與自在，而穿著心理認同性別的衣物，以表達自己正確的[[心理性別|心理性別]]，繼而在自己與他人交流時希望他人適度按照「男女社群話題之異」來相應調整話題。<br />跨性別包括了許多次分類，比如變性者（transsexual）、異裝者（cross-dresser）、異裝症者（transvestites）、異裝國王（drag kings）與[[變裝皇后|異裝皇后]]（drag queen）。

通常戀異性裝癖者或是異裝癖（transvestic fetishist）不包含在這裡面，因為在大部分的情況下這並不是一個性別議題（gender issue）（不過在實踐上很難分）。

在本文中為行文方便起見，'''異裝'''是指行為而與理由動機無關。但是異裝一詞的意義未必與理由無關。某些[[跨性別者|跨性別者]]就以「異裝」或是意義相關的'''CD'''（cross-dressing）、'''TV'''（transvestic）來稱呼相關的行為或人物。在此，異裝可說就是一種身分認同。

== 戀異性裝癖者 ==
人類異裝的動機，有因為性別認同障礙而異裝，也有因為純粹戀異性裝癖而異裝。戀異性裝癖不能劃到跨性別類别裡来，原因如下：
* 跨性別者乃認為自己[[心理性別|心理性別]]和[[性別指定|性別指定]]不一致，並基於在「社會普遍習慣把男女衣著風格作鮮明區分」的風氣下務求取得大眾對其內心真正性別即[[心理性別|心理性別]]之認同，而穿著與生理性別不同的裝扮。他們對異性服飾並不會有慾望，假如人類都恢復一絲不掛，他們就不會再有異裝的需要和擔憂，故此以戀異性裝癖並不適用於判定到跨性別者上。
* 戀異性裝癖的異裝則並非因為如跨性別者般其內心不認同自己的生殖性别，而是純粹迷戀該等服飾之功能以達求滿足自身性慾幻想之目的。

很多戀異性裝癖者跟人間既有的刻板印象不同，不是跨性別也不是同性戀。而許多人就只是單純把自己定義為跨性別者，儘管他們也可能符合以上所述的其中一個分類。

=== 演員與表演 ===
{{main|反串}}
男演員演出女性角色在中國傳統戲曲中有悠久的歷史。[[京劇|京劇]]大師[[梅蘭芳|梅蘭芳]]就以此聞名。民國初年開始有女演員演出男性角色，粵劇名伶[[任劍輝|任劍輝]]就是經常扮演男角的女演員。傳統戲曲中的'''反串'''原本是指演出與自身本工的行當不同的戲，但傳統戲曲以外的演員若扮演異性角色也可稱為'''反串'''。[[中華人民共和國|中華人民共和國]]的第一任[[國務院總理|國務院總理]][[周恩來|周恩來]]在就讀[[天津市南開中學|天津市南開中學]]的時期，曾參與「[[南開新劇|南開新劇]]」（南開中學的[[話劇|話劇]]團隊），並多次反串飾演[[女性|女性]][[角色|角色]]。<ref>[[天津市南開中學|天津市南開中學]]校史館內</ref>

而較知名的男扮女演員則計有香港已故藝人[[張國榮|張國榮]]，其異裝動機乃基於在「社會普遍習慣把男女衣著風格作鮮明區分」的風氣下務求以衣著風格取得大眾對其[[心理性別|心理性別]]和[[戀愛傾向|戀愛傾向]]的認知與認同。在[[長江七號|長江七號]]電影中，[[周星馳|周星馳]]的契女[[徐嬌|徐嬌]]、[[黄蕾|黄蕾]]和[[姚文雪|姚文雪]]反串演出男性角色<ref>[http://ent.sina.com.cn/m/2007-11-20/19351800693.shtml 《長江七號》「星光幫」──徐嬌]</ref><ref>[http://ent.sina.com.cn/m/2008-01-20/16431884624.shtml  《長江七號》「星光幫」── 黃蕾]</ref><ref>[http://ent.sina.com.cn/m/2008-01-20/17031884632.shtml 《長江七號》「星光幫」──姚文雪]</ref>；而[[韓永華|韓永華]]則反串演出女性角色<ref>[http://ent.sina.com.cn/m/2008-01-20/16531884627.shtml 《長江七號》「星光幫」──韓永華]</ref>。

== 社會壓迫 ==
有些女性認為男性易服後難以辨認其[[性別|性別]]，造成她們防範男性[[非禮|非禮]]有困難，然而並沒有證據顯示異裝者有較高比例為性騷擾加害者。大多數加害者往往外表上都符合於主流社會認知，將服裝穿扮視為增加性騷擾防範困擾的說法顯得牽強。

有些地方的法律禁止人民作異性打扮，如新加坡（但現況可能放寬）以及伊斯蘭教國家等。如果異裝行為對別人構成不安，例如女性化男生進入女洗手間/更衣室等，執法機關也有可能以遊蕩罪或類似罪名向異裝者提出告訴。<ref>[http://orientaldaily.on.cc/cnt/news/20110923/00176_072.html 法庭：入女更衣室 易服男罰款 東網]</ref>

1960到1990年代間，[[新加坡|新加坡]]法律曾經以當時社會兩性著裝習慣為規範訂立法律，並強制所有於其境內外出現或駐留的人士遵守，把「社會普遍習慣把男女衣著打扮風格作鮮明區分」的風氣貫徹硬性執行。例子有：
* 規定男士頭側頭髮長度不得足以蓋住耳朵及膊<ref>[http://farm2.static.flickr.com/1323/1404171425_1cff328a84.jpg?v=0 Males with long hair will be attended to last]</ref>，而在公眾場所亦時常有「長髮男子不受歡迎」等告示；
* 規定男士不得穿著修裁標緻的[[衣衫|衣衫]]、[[裙子|裙子]]、[[絲襪|絲襪]]、[[褲襪|褲襪]]、[[高跟鞋|高跟鞋]]等「被視為屬於女性穿著」的服裝；
* 規定男士不得佩戴耳環與任何標緻飾物等「被視為屬於女性佩戴」的飾物。
此法令主要針對[[嬉皮|嬉皮]]。當時，[[新加坡|新加坡]]境內任何民眾儀容一旦未符合上述所有規定者即屬違法。
