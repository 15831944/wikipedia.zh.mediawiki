《'''變相怪傑'''》（{{lang-en|'''''The Mask'''''}}）是一部于[[1994年電影|1994年]]首映的[[美國電影|美國]][[超級英雄電影|超級英雄電影]]，內容改編自[[黑馬漫畫|黑馬漫畫]]系列《[[面具_(漫畫)|面具]]》，描述一名小白領在戴上了一個具有超能力的面具後，變成[[超級英雄|超級英雄]]的經歷。

電影由{{Tsl|en|Chuck Russell|卓克·羅素}}執導，[[占·基利|占·基利]]及[[金美倫·戴雅絲|金美倫·戴雅絲]]等主演；此片亦為[[金美倫·戴雅絲|金美倫·戴雅絲]]的出道作品。電影大受歡迎，亦獲提名數個獎項，包括[[金球獎_(影視獎項)|金球獎]][[金球獎最佳音樂及喜劇片男主角|最佳音樂及喜劇片男主角]]（[[占·基利|占·基利]]）及[[第67屆奧斯卡金像獎|第67屆]][[奧斯卡金像獎|奧斯卡]][[奧斯卡最佳視覺效果獎|最佳視覺效果獎]]。

== 故事内容 ==
史丹利是一个老实本分的银行职员，经常遭到上司的呵斥，同事的奚落。直到有一天他接待了一个叫做婷娜的金发美女，史丹利见到她顿时对她颇有好感，然而令史丹利没有想到的是婷娜竟然是为黑帮头目多利安抢劫银行而来打探地形的探子。后来，史丹利受到邀请，与好友一同去市里的夜总会参加Party，却又因为没有地位而被拒之门外，不巧的是还在婷娜面前出尽了洋相，随后在回家的路上汽车却又散了架，总之一切倒霉的事全都发生在他身上。但意外的是史丹利无意间获得了一个外形古怪的面具，失意的他回到家后，戴上了神奇的面具，一瞬间他变成了一个拥有奇特力量的緑面怪侠，随后他利用这股力量痛快地发泄了一夜，第二天他又恢复了自己原本的形象。由于太过思念婷娜，并打算拯救她，史丹利便再次戴上面具，成功的劫走了多利安在银行所抢劫的大量钱财，然后前往夜总会与婷娜一同疯舞了一场，随后气急败坏的多利安发现自己的女友居然和绿色面具男在一起，他试图阻止他们却又不幸招来了警察，糟糕的是史丹利在逃跑时居然留下了证据。在这之后，警察和黑帮都在通缉緑面怪侠，并调查他的真身身份……

== 荣誉 ==
这部电影票房非常賣座，北美院线狂揽一亿一千九百万，海外票房达三亿五千万，成为当时仅次于[[蝙蝠侠|蝙蝠侠]]的票房第二高的超级英雄电影。尽管票房被之后的几部超级英雄电影超过，这部电影依然极受欢迎。
== 參註 ==
{{reflist}}
原作漫畫中，摩登大聖為以[[小丑_(漫畫)|小丑]]（[[蝙蝠俠|蝙蝠俠]]反派）作藍本的殺人狂魔，為史丹利‧伊卜吉解放壓力的姿態。电影《變相怪傑》最初本来被策划成一部恐怖片，但拍摄过程中在金·凱瑞（Jim Carrey）手中，卻變成以搞出各種惡作劇戲弄別人的瘋癲英雄的喜剧片。
