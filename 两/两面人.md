'''两面人'''或'''两面派'''是[[中国共产党|中国共产党]]对某些党员的特定称谓，多以定义在[[中华人民共和国政府|政府]]内担任官职的贪腐党员<ref>{{Cite web |url=http://fanfu.people.com.cn/n/2015/0826/c64371-27520244.html |title=认清政治上的“两面人”：贪官AB面的八个特征 |accessdate=2018-11-02 |author=崔小粟|date=2015-08-26|work= |publisher=人民网-中国共产党新闻网|language=Zh-hans }}</ref>，以及党内[[反中共|反共主义者]]<ref name="a1"/>。根据2018年版《[[:s:中国共产党纪律处分条例|中国共产党纪律处分条例]]》第五十一条的定义<ref>{{Cite web |url=http://www.12371.cn/2018/08/27/ARTI1535321642505383.shtml |title=中国共产党纪律处分条例（全文）（2018年修订，自2018年10月1日起施行） |accessdate=2018-11-02 |author=|date=|work= |publisher=共产党员网|language=Zh-hans }}</ref>，“对党不忠诚不老实，表里不一，阳奉阴违，欺上瞒下”，即是“搞两面派，做两面人”，对两面派、两面人“情节较轻的，给予警告或者严重警告处分；情节较重的，给予撤销党内职务或者留党察看处分；情节严重的，给予开除党籍处分。”党员处分时，'''两面人'''被视为严厉的措辞<ref>{{Cite web |url=http://politics.people.com.cn/n1/2018/0213/c1001-29823215.html |title=给鲁炜定性的这些词你应该牢记 |accessdate=2018-11-03 |author=|date=2018-02-13|work=责编：曹昆、曾伟 |publisher=人民网，时政频道|language=Zh-hans }}</ref><ref>{{Cite web |url=https://news.qq.com/a/20180213/023743.htm |title=侠客岛：鲁炜，被中央定性用词“最狠”的大老虎 |accessdate=2018-11-03 |author=|date=2018-02-13|work=责任编辑：leidu |publisher=腾讯网|language=Zh-hans }}</ref>。2018年2月，[[中共中央宣传部|中共中央宣传部]]原副部长、[[中央网信办|中央网信办]]原主任[[鲁炜|鲁炜]]落马时，即定性为“对[[中共中央|党中央]]极端不忠诚”的“典型的‘两面人’”<ref>{{Cite web |url=http://www.gov.cn/xinwen/2018-02/13/content_5266651.htm |title=中央宣传部原副部长、中央网信办原主任鲁炜严重违纪被开除党籍和公职 |accessdate=2018-11-03 |author=|date=2018-02-13|work=责任编辑：方圆震 |publisher=中国政府网，来源：新华社|language=Zh-hans }}</ref>。

在中国现行政治体系下，政府官员通常兼有中共党员身份。加入中国共产党，几乎是成为[[中华人民共和国公务员|公务员]]的必经之路。因此入党可能含有投机成分，而非信仰本身。政府机构中的两面人是为普遍现象<ref name="a1"/>。无论是学生党员或担任公职的党员，持有反共、反中思想者亦为常见。知名党员公开表达反共情绪亦非个案。比如说:[[毕福剑酒桌唱戏事件|毕福剑酒桌唱戏事件]]，以及[[任志强事件|任志强事件]]。有文章指，产生两面人的原因，有人性因素，更是政治制度制约不足的结果，包括考核、监督机制的缺位<ref>{{Cite web |url=http://news.163.com/16/0126/06/BE82C3AU00014AED.html |title=官场两面人 |accessdate=2018-11-02 |author=张晓政|date=2016-01-26|work=责任编辑：王晓易_NE0011 |publisher=网易，来源：山西新闻网-山西日报(太原)，据《月读》2016年第1期|language=Zh-hans }}</ref>。香港《[[東方日報_(香港)|东方日报]]》指出<ref name="a1">{{Cite web |url=https://hk.on.cc/hk/bkn/cnt/commentary/20180426/bkn-20180426000416645-0426_00832_001_cn.html |title=党员博士生精日　官场两面人辈出 |accessdate=2018-11-02 |author=|date=2018-04-26|work= |publisher=東網|language=Zh-hans }}</ref>，中共党内更多是隐形两面人，中共现有八千多万党员，究竟有多少两面人，“恐怕只有‘神知道’”。如果按《[[中国共产党党章|中国共产党党章]]》的标准，每年查处数万名贪官，就等于每年揭发数万名“两面人”。

2017年10月，[[中国共产党总书记|中国共产党总书记]][[习近平|习近平]][[中共十九大|中共十九大]]报告中提出“坚决反对搞两面派、做两面人”的观点<ref>{{Cite web |url=http://paper.people.com.cn/rmrb/html/2018-04/04/nw.D110000renmrb_20180404_3-07.htm |title=人民日报新知新觉：坚决反对搞两面派做两面人 |accessdate=2018-11-02 |author=王寿林|date=2018-04-04|work= |publisher=人民网，来源：《人民日报》（2018年04月04日 07版）|language=Zh-hans }}</ref>。2018年1月，[[中纪委|中纪委]]提出清除党内两面人、两面派，是2018年反腐的新动向<ref>{{Cite web |url=http://www.chinanews.com/gn/2018/01-17/8425798.shtml |title=2018反腐败新动向：中纪委剑指“两面人两面派” |accessdate=2018-11-03 |author=|date=2018-01-17|work= |publisher=中国新闻网|language=Zh-hans }}</ref>。[[王丹|王丹]]认为习近平提出打击两面人的原因，是为提高党内各级官员忠诚度，以巩固权力<ref>{{Cite web |url=https://www.rfa.org/mandarin/pinglun/wangdan/wd-01192018110542.html |title=谁是中共内部的“两面人”？（王丹） |accessdate=2018-11-03 |author=王丹|date=2018-01-19|work=网编：安克 |publisher=自由亚洲电台网站|language=Zh-hans }}</ref>。
