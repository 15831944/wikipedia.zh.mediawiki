'''泷川亚里沙'''（{{Jpn|j=瀧川 ありさ|kana=たきがわ ありさ|rm=Takigawa Arisa}}，{{Bd|1991年|5月8日}}），[[日本|日本]]女性[[创作歌手|创作歌手]]，出生于[[东京都|东京都]]。2015年出道，现为[[日本索尼音乐娱乐|日本索尼音乐娱乐]]旗下艺人。

==经历==
泷川从小就喜爱音乐，先后受到[[The_Blue_Hearts|THE BLUE HEARTS]]、[[松任谷由实|松任谷由实]]和[[山下达郎|山下达郎]]等音乐人的影响。就读中学时加入轻音乐部，开始演奏吉他和鼓，并创作歌曲。此后与高冈纯和石川美奈子组成乐队“A”（2010年改名“The alps”），开始音乐活动，获得了多个音乐竞赛的奖项。2012年10月乐队解散。

在乐队解散一年后，泷川开始了个人活动。2015年3月推出第一张单曲《Season》而主流出道<ref>{{cite web |title=新人シンガーソングライター・瀧川ありさ、デビュー曲が「七つの大罪」EDに |url=https://natalie.mu/music/news/133428 |website=音楽ナタリー |accessdate=2019-03-20}}</ref>。此曲被用作动画片《七大罪》的结尾曲。2015年一共发布三张单曲，11月举行了个人首场专场演唱会<ref>{{cite web |title=瀧川ありさ、超満員のO-nestで「うれしいな」連発の初ワンマン |url=https://natalie.mu/music/news/167241 |website=音楽ナタリー |accessdate=2019-03-20}}</ref>。

2016年获得[[日本金唱片大獎|日本金唱片大獎]]Best5新人<ref>{{cite web |title=ゴールドディスク大賞は嵐が2年連続4回目、アレキ・AKB48らも各賞受賞 |url=https://natalie.mu/music/news/177777 |website=音楽ナタリー |accessdate=2019-03-20}}</ref>。同年发售单曲《Again》<ref>{{cite web |title=瀧川ありさ4月に4thシングル「Again」、予約イベントでお渡し会も |url=https://natalie.mu/music/news/173788 |website=音楽ナタリー |accessdate=2019-03-20}}</ref>和《不褪色的眼神》<ref>{{cite web |title=瀧川ありさ、5thシングルは「色褪せない瞳」 |url=https://natalie.mu/music/news/191241 |website=音楽ナタリー |accessdate=2019-03-20}}</ref>，11月发售首张专辑《at film.》，并开始专辑发售巡回演唱会<ref>{{cite web |title=瀧川ありさ、デビュー曲から最新シングルまで収めた1stアルバムリリース |url=https://natalie.mu/music/news/201860 |website=音楽ナタリー |accessdate=2019-03-20}}</ref>。

2017年发售双A面单曲《No side/ONE FOR YOU》，其中《No side》是翻唱松任谷由实的歌曲。这首歌被用作动画片《ALL OUT!!》结尾曲<ref>{{cite web |title=「ALL OUT!!」新テーマ曲でCIVILIANが新曲書き下ろし＆瀧川ありさがユーミンカバー |url=https://natalie.mu/music/news/213557 |website=音楽ナタリー |accessdate=2019-03-20}}</ref>。

2018年发售首张迷你概念专辑《东京》，以出生成长的东京为主题<ref>{{cite web |title=瀧川ありさ“東京”をテーマにしたコンセプトミニアルバム発表、ワンマンも決定 |url=https://natalie.mu/music/news/279470 |website=音楽ナタリー |accessdate=2019-03-20}}</ref>。
