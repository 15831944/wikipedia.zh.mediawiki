'''让-玛丽·勒庞'''（{{lang-fr|Jean-Marie Le Pen}}，{{IPA-fr|ʒɑ̃ ma.ʁi lə.pɛn}}；{{bd|1928年|6月20日|||Le Pen, Jean-Marie}}），[[法国|法国]]政治家，1972年創立[[极右翼|极右翼]]政黨[[民族阵线_(法国)|民族阵线]]。

== 生平 ==
勒龐出生在[[法國|法國]][[布列塔尼|布列塔尼]]地区的一个海邊小鎮，父親是一名漁夫，1942年父親的漁船誤觸英軍水雷炸毁身亡，勒庞在青少年時因此成為孤兒。1947年進入[[巴黎大學|巴黎大學]]學習[[法律|法律]]。勒龐於1972年建立了自己的政黨──[[民族阵线_(法国)|民族阵线]]，並擔任黨魁至2011年。

勒龐自1974年開始即多次參選[[法國總統|法國總統]]，除1981年法国总统選舉因沒有取得足夠連署未能成為候選人外，1988年、1995年、2002年、2007年的法國總統選舉他都有參與。

[[1995年法国总统選舉|1995年法国总统選舉]]中，勒龐得到了15%的选票，雖然沒有當選，但他的得票排名第三，已經使[[法国|法国]]和[[欧洲|欧洲]]民众大感惊奇及恐慌。

[[2002年法國總統選舉|2002年法國總統選舉]]，他在首轮选举中击败了選前呼声很高的時任[[法国总理|法国总理]]、左派社會黨的[[利昂内尔·若斯潘|利昂内尔·若斯潘]]，進入了第二輪投票，更是震惊整个舆论界，左派發動示威抗議他的政黨，並把選票投給他的政敵、右派共和聯盟的時任總統席哈克。席哈克最終以絕對優勢擊敗成功連任。

[[2007年法國總統選舉|2007年法國總統選舉]]，勒龐五度參選，在左派社會黨的[[塞格琳·羅雅爾|塞格琳·羅雅爾]]與右派[[人民運動聯盟|人民運動聯盟]]的[[尼古拉·薩爾科齊|尼古拉·薩爾科齊]]和中間派民主黨的[[弗朗索瓦·貝魯|弗朗索瓦·貝魯]]多方強棒競選下，勒龐獲得10%的選票，排名第四未進入第二輪選舉。

2011年1月16日，勒龐卸任黨魁一職，由小女儿[[玛丽娜·勒庞|玛丽娜·勒庞]]繼任。

2015年8月21日，勒龐由於與其女兒兼現任领袖玛丽娜·勒庞領導的民族陣線政見不合，他因堅持帶[[新納粹|新納粹]]色彩的[[反猶太主義|反猶太主義]]政治立場，與黨的立場不符，被民族陣線開除黨籍。

在[[2015年11月巴黎襲擊事件|2015年11月巴黎襲擊事件]]發生後，勒龐要求重新啟用[[斷頭台|斷頭台]]用來處決[[恐怖份子|恐怖份子]]。<ref>{{Cite web |url= http://www.thedailybeast.com/articles/2015/11/25/monsieur-le-pen-wants-to-guillotine-terrorists-in-france.html|title= Monsieur Le Pen Wants to Guillotine Terrorists in France|accessdate=2016-07-02 |work= |publisher= The Daily Beast|date= 2015-11-25}}</ref><ref>{{Cite web |url= http://www.express.co.uk/news/world/621098/Reinstate-French-death-penalty-after-Paris-attacks-says-former-NF-leader-Jean-Marie-Le-Pen|title= Far-right French politician wants to bring back death penalty and BEHEAD terrorists|accessdate=2016-07-02 |work= |publisher= Daily Express|date= 2015-11-21}}</ref>

== 政治主張 ==
* 民族阵线主張大幅限制[[移民|移民]]進入法國，尤其是[[非洲|非洲]]及[[穆斯林|穆斯林]]移民进入[[法国|法国]]。
* 反对[[欧洲联盟|欧洲联盟]]和[[歐元|歐元]]，提倡[[貿易保護主義|貿易保護主義]]，以利保护国内经济发展和民众就业。
* 反对[[堕胎|堕胎]]、[[同性婚姻|同性婚姻]]、[[安樂死|安樂死]]，整頓治安、[[法律與秩序|法律與秩序]]、恢復[[死刑|死刑]]等。
