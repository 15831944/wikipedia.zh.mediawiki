'''尚·皮亞傑'''（{{lang-fr|'''Jean Piaget'''}}，{{bd|1896年|8月9日|1980年|9月16日|catIdx=Piaget, Jean}}），全名'''尚·威廉·弗里茲·皮亞傑'''（{{lang-fr|'''Jean William Fritz Piaget'''}}），[[瑞士|瑞士]]人，是近代最有名的[[發展心理學|發展心理學]]家，同時也是[[哲學家|哲學家]]。他的[[认知发展理论|认知发展理论]]成為了這個學科的典範。皮亞傑早年接受[[生物學|生物學]]的訓練，他在大學讀書時就已經開始對心理學有興趣，并曾涉獵心理學早期發展的各個學派如[[病理心理學|病理心理學]]、[[精神分析學|精神分析學]]、[[卡爾·榮格|荣格]]的[[潛意識心理學|潛意識心理學]]和[[西格蒙德·弗洛伊德|弗洛伊德]]的學說。皮亚杰从1929年到1975年在[[日内瓦大学|日内瓦大学]]担任心理学教授。

==生平==
皮亚杰出生在[[瑞士|瑞士]]法语区的[[纳沙泰尔|纳沙泰尔]]，父亲亚瑟是[[纳沙泰尔大学|纳沙泰尔大学]]的中世纪文学教授。他是个早熟的孩子，对[[生物学|生物学]]，特别是[[软体动物|软体动物]]很有兴趣，中学毕业之前在这方面已经发表了许多论文。他漫长的科学生涯开始于1907年他11岁时发表关于患白化病的[[麻雀|麻雀]]的短文。此后70年中他写了60多本书和一千五百篇多论文。

这时他从瑞士搬到[[法国|法国]]，在Grange-aux-Belles [[阿尔弗雷德·比奈|阿尔弗雷德·比奈]]（比奈[[智力测验|智力测验]]的发明者）开办的男校教书。1921年，他回到瑞士，担任日内瓦[[卢梭学会|卢梭学会]]会长。

1923年，他和Valentine Châtenay结婚，他们的3个孩子都是他研究的对象。

==认知发展阶段==
{{Main|认知发展论}}

皮亞傑最著名的學說，是他把兒童的[[認知發展|認知發展]]分成以下四個階段：
# 感知运动阶段（感覺-動作期，Sensorimotor, 0-2岁）靠感觉获取经验。1歲時發展出物體恆存的概念，以感覺動作發揮图式的功能；
# 前运算阶段（前運思期，Preoperational, 2-7岁）已經能使用語言及符號等表徵外在事物，具推理能力但不符邏輯，不具保留概念，缺乏可逆性，以自我為中心，直接推理，集中注意力，
# 具体运算阶段（具體運思期，Concrete Operational, 7-11歲）了解水平線概念，能使用具體物之操作來協助思考；
# 形式运算阶段（形式運思期，Formal Operational, 11-16歲）開始會類推，有邏輯思維和抽象思維。

這四個階段有以下的特色：
# 發展順序不變，但具有個別差異。
# 具有普遍性（''不具''文化特別性）。
# 依賴認知發展，但可普遍化為其它功能。
# 各發展階段都是在邏輯上有組織的整體。
# 各階段的順序是自然的階層（所有成功發展的階段都會有前面階段的元素參與合作，但後一階段比起前面的階段，更加不同，而且更加統整）。
# 每個階段，在思考模式上會表現出質的不同，而不僅僅是量的差異。

皮亞傑提出認知發展過程或建構過程有四個核心概念：
# [[基模（图式）|基模（图式）]]（schemas）：將知識形像化，成為一個概念模型。知識可以透過同化或順化來達成。
# [[同化|同化]]（assimilating）：將新知識和舊有知識類比，並作出關連。
# 調適（[[順化|順化]]，accommodating）：將舊知識的概念模型改變調適，以容納新的內容。
# [[平衡|平衡]]（equilibrium）
這種形像化過程不會完全只有同化或順化的過程，而是兩者之間的一個平衡點。找出這個平衡點的過程就是平衡。平衡有三種：
#第一種平衡是同化和順化之間的聯繫。
#第二種平衡是基模與基模之間的平衡。
#第三種平衡是一種調節個體部分知識與整體知識之間關係的平衡。

雖然皮亞傑的理論有部份跟[[維高斯基|維高斯基]]（Vygotzky）的理論相似，但相信皮亞傑的理論應該跟維高斯基的理論沒有關連。
皮亞傑的[[發展心理學|發展心理學]]理論廣泛流傳。在其中，[[哲學家|哲學家]]，也是[[社會理論家|社會理論家]]的[[哈贝马斯|哈贝马斯]]將皮亞傑的理論結合到他的研究中，而以[[溝通行動論|溝通行動論]]聞名。

==皮亚杰的儿童心理观==
皮亚杰把儿童看成建造自己的个人知识理论的''小哲学家''和科学家。

[[西摩爾·派普特|西摩爾·派普特]]就是根據皮亞傑的理論而開發了可讓兒童學習使用電腦的[[Logo程序设计语言|Logo程式語言]]。[[Alan_Kay|Alan Kay]]以皮亞傑的理論為基礎，開發了[[Dynabook|Dynabook]]編程概念。以上這兩項技術，都是現今我們日常使用的[[图形使用者介面|圖形使用者介面]]（GUI）的基礎技術，使電腦從文字界面過渡至圖形界面，改寫了電腦技術的發展路向。
