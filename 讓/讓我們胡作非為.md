'''讓我們胡作非為'''（{{lang-en|'''Let's Misbehave'''}}）是[[科爾·波特|科爾·波特]]於1927年創作的一首歌曲，原本是為他的第一部大製作《{{link-en|巴黎 (1928年音樂劇)|Paris (1928 musical)|巴黎}}》的女主角而準備的。雖然在百老匯開幕前用"{{link-en|Let's Do It, Let's Fall in Love|Let's Do It, Let's Fall in Love}}"取而代之，但是百老匯明星{{link-en|伊蓮·波多妮|Irène Bordoni}}做了這首歌曲的一個留聲機錄音。這首歌在1928年因{{link-en|艾爾文·阿倫森|Irving Aaronson|艾爾文·阿倫森和他的指揮官}}樂團而著名。這歌曲可能在《[[海上情緣|海上情緣]]》的1962年重演中變得更出名。

這首歌是在1927年由Banjo Buddy（又名Harold Sandelman）為{{link-en|Brunswick Records|Brunswick Records|Brunswick}}唱片公司而錄製的。<ref>{{Cite web |url = https://www.discogs.com/Banjo-Buddy-Lets-Misbehave/release/7931803 |title = Banjo Buddy – Let's Misbehave |publisher = Discogs |language = en}}</ref>

== 現代使用 ==
艾爾文·阿倫森的歌曲版本在兩部[[活地·亞倫|活地·亞倫]]的電影中被使用：1972年電影《[[性愛寶典|性愛寶典]]》的開頭和結尾，以及1994年電影《[[子彈橫飛百老匯|子彈橫飛百老匯]]》的結尾。1981年歌舞電影《''{{link-en|Pennies from Heaven (1981 film)|Pennies from Heaven (1981 film)|Pennies from Heaven}}''》中由[[基斯杜化·華堅|基斯杜化·華堅]]演唱這首歌。它也出現在1994年電影《''{{link-en|Timecop|Timecop}}''》中。最近期，它出現在2013年電影《[[大亨小傳_(2013年電影)|大亨小傳]]》<ref>{{Cite web |url = https://www.theatlantic.com/entertainment/archive/2013/05/a-grating-em-great-gatsby-em/275744/ |title = A Grating Great Gatsby |author = Christopher Orr |date = 2013-05-10 |publisher = The Atlantic |language = en}}</ref>。它也出現在[[肯·伯恩斯|肯·伯恩斯]]的2011年[[PBS|PBS]]紀錄片《[[禁酒令_(迷你連續劇)|禁酒令]]》中。
