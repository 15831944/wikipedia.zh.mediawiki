'''伍珀塔尔空中铁路'''（{{lang-de|Wuppertaler Schwebebahn}}），简称“'''空铁'''”（Schwebebahn），是一条位于[[德国|德国]][[伍珀塔尔|伍珀塔尔]]的[[悬挂式单轨铁路|悬挂式单轨铁路]]。
其全名为“电气高架（悬挂铁路）工程，-{zh-hans:渥伊根·朗恩;zh-tw:渥伊根·朗恩;zh-hk:愛根·朗恩;zh-mo:愛根·朗恩;}-系统”（''Anlage einer elektrischen Hochbahn (Schwebebahn), System Eugen Langen''）。<ref>{{cite web |url=http://www.ag-sdd.de/ausgew_erw/rundgaenge/vollansicht/rundgang_5/spk_9.htm |title=存档副本 |accessdate=2011-01-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20110718195017/http://www.ag-sdd.de/ausgew_erw/rundgaenge/vollansicht/rundgang_5/spk_9.htm |archivedate=2011-07-18 }}</ref>
该系统是世界上最老的电气高架悬挂式铁路，且系统特立独行。

==历史==
由-{zh-hans:{{link-en|渥伊根·朗恩|Eugen Langen}};zh-tw:{{link-en|渥伊根·朗恩|Eugen Langen}};zh-hk:{{link-en|愛根·朗恩|Eugen Langen}};zh-mo:{{link-en|愛根·朗恩|Eugen Langen}};}-为出售给[[柏林|柏林]]而设计<ref>{{cite web |url=http://home.arcor.de/nobe/faz980825.html |title=存档副本 |accessdate=2013-02-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20141105161612/http://home.arcor.de/nobe/faz980825.html |archivedate=2014-11-05 }} ''Die Große Erneuerung ist schon arg in Verzug'', Frankfurter Allgemeine Zeitung, 25. August 1998 {{de icon}}</ref>，该系统在1897年与1903年之间于[[Barmen|Barmen]], [[Elberfeld|Elberfeld]]和[[Wuppertal-Vohwinkel_station|Vohwinkel]]两站间建设，第一段轨道于1901年开通。该“空铁”至今仍作为公共交通形式运营，每年客运量为2500万人次（2008年数据）。<ref>{{cite web |url=http://www.wsw-online.de/unternehmen/Download/Geschaeftsberichte/gb2008.pdf |title=存档副本 |accessdate=2010-01-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20100215085000/http://www.wsw-online.de/unternehmen/Download/Geschaeftsberichte/gb2008.pdf |archivedate=2010-02-15 }} Annual report, page 44 {{de icon}}</ref>

空铁全程{{convert|13.3|km|mi}}，其中 Oberbarmen 和 Sonnborner大街之间的{{convert|10|km|mi|disp=or}}的路段位于[[伍泊河|伍泊河]]上方{{convert|12|m|ft}}处，Sonnborner大街和 Vohwinkel 之间的{{convert|3.3|km|mi|disp=or}}位于峡谷道路上方的{{convert|8|m|ft}}处。<ref name="Urbanrail">{{cite web |title=UrbanRail.Net > Europe > Germany > Wuppertaler Schwebebahn (suspension railway) |url=http://www.urbanrail.de/eu/vrr/wuppertal.htm |accessdate=20 February 2007 |archiveurl=https://web.archive.org/web/20070219130551/http://www.urbanrail.de/eu/vrr/wuppertal.htm |archivedate=2007年2月19日 |deadurl=yes }}</ref><ref name="Uniwuppertal">{{cite web | title=Uni Wuppertal – Wuppertal's Suspension Railway: overview and history | url=http://www.uni-wuppertal.de/wuppertal/schwebebahn/allgemein-en.html | accessdate=20 February 2007 | archiveurl=https://web.archive.org/web/20061008161818/http://www.uni-wuppertal.de/wuppertal/schwebebahn/allgemein-en.html | archivedate=2006年10月8日 | deadurl=yes }}</ref>铁路于某处跨越[[聯邦公路46|A46]]高速公路。全程需要30分钟。<ref name="Uniwuppertal" />空铁接受[[萊茵－魯爾運輸聯合體|萊茵－魯爾運輸聯合體]]（{{lang-de|Verkehrsverbund Rhein-Ruhr}}，簡稱「VRR」）系统内其他公司出售的车票。

== 技术==
[[Image:Antriebssatz_einer_aktuellen_Schwebebahn_Projekt_Erste_neue_Schwebebahn_1.jpg|thumb]]
[[File:Wuppertal_kaiserwagen.jpg|thumb]]
车厢悬挂于支撑钢梁下的一条单轨上，以DC600伏驱动的电机驱动车厢上方的轮子，电力来源于另外一条铁轨。<ref name="Stadtwerke">{{cite web|title=Wuppertaler Stadtwerke AG – English|url=http://www.wsw-online.de/schwebebahn/sprachen/english|accessdate=20 February 2007|deadurl=yes|archiveurl=https://web.archive.org/web/20061009002516/http://www.wsw-online.de/schwebebahn/sprachen/english|archivedate=2006年10月9日}} </ref>

支撑钢梁和钢轨由486根支柱和桥段支撑。两端的终点站同时作为车厂和灯泡线使用。<ref name="Uniwuppertal" />

现在一共有1970年代制造的27组<ref name="Stadtwerke" />2节编组<ref name="Urbanrail" />的车辆投入运营。<ref name="Uniwuppertal" />每节车厢为24米长，有4个门，48个座位，以及供130名乘客站立的空间。<ref name="Urbanrail" />最高时速为 60 km/h，平均旅速为 27 km/h。<ref name="Uniwuppertal" />

车辆中有辆''Kaiserwagen'' (皇帝专用车)，是当年[[威廉二世_(德國)|威廉二世]]于1900年10月24日试乘时坐过的车厢。该车在特殊场合下也会投入运营。<ref name="Uniwuppertal" />

== 事故==

;1917年1月15日
:Oberbarmen 和 Wupperfeld 站之间一列车与另外一列计划外停车的列车相撞，导致被撞列车的后一节车厢脱轨，两人受轻伤。<ref>{{cite web|title=http://www.schwebebahn-wtal.de|url=http://www.schwebebahn-wtal.de/|accessdate=21 February 2007}}</ref>之后为列车安装了使脱轨几乎不可能发生的安全装置。<ref name="Uniwuppertal" />
;1950年7月21日
:Althoff 马戏团组织了一场公开活动，将一头小象于 Alter Markt 车站放上了列车。当小象在列车里来回乱撞时，小象从列车里掉出，掉入伍泊河中。<ref>Article in Westdeutsche Zeitung, March 2009</ref>那头小象、两名记者、以及一名乘客受了轻伤。受伤的小象后来被命名为[[Tuffi|Tuffi]]，为意大利语中“跳水”的意思。运营商和马戏团领导后来都被罚款。
;1968年9月11日
:一辆卡车撞上一根支柱，导致一段铁轨掉落。当时区间内没有列车，之后促使管理者将支柱落地处装上混凝土墙。 
;1997年3月25日
:技术故障导致两列车于 Oberbarmen 站追尾，相撞的是一列工程车和皇帝专用车。14人受伤，不过没有列车出轨。
;1999年4月12日
:该系统内唯一一次致命事故发生于 Robert-Daum-Platz 站，当时该站正处于维修状态中。工人忘记在夜里工作完成后将铁轨上的铁爪去掉。一列东向列车以50千米每小时的速度撞上铁爪，脱轨，从10米高处掉入河中，导致5人丧生，49人受伤。<ref name="Uniwuppertal" />打捞工作花了3天3夜。事故发生8周后空铁才恢复运营。这次事故的损失估计在800万德国马克左右。 
:事故发生后的司法程序确认事故不是由于系统的技术故障导致，而是由于工人的粗心大意才发生；工人在列车开出车库后的10分钟内就慌忙离开工地，工地监工也没有尽到责任。
:伍珀塔尔地区法院判决施工经理和导致事故发生的工人无罪。工地监工因为玩忽职守，被以5例过失杀人罪和37例过失致人伤害罪判处有罪，但是在2001年1月31日时获得假释。<ref>http://www.hrr-strafrecht.de/hrr/4/01/4-289-01.php3</ref>
;2008年8月5日
:列车与一辆卡车上安装的升降臂相撞，导致一节车厢的地板被拉出10米长的缝隙。卡车司机受重伤，列车司机和一些乘客因为脑震荡被送医。<ref>http://www.spiegel.de/reise/aktuell/0,1518,570258,00.html</ref>
;2013年10月17日
:一节100米长的供电轨道掉到[[Bundesstraße_7|7号联邦道路]]上，压坏了一些停在那里的车，导致道路关闭。该市的消防部门不得不从停驶的列车中救出70名乘客。无人受伤。<ref>http://www.derwesten.de/region/wuppertaler-schwebebahn-steht-nach-unfall-bis-mindestens-montag-still-id8573070.html</ref>
