《'''阳痿美国'''》（又名《'''审判美国'''》），著名文学家、历史学家、演讲家[[李敖|李敖]]的[[戏剧|戏剧]]文学作品。简体字、繁体字均在2010年出版<ref>{{cite web |url=http://news.sina.com.cn/c/2010-09-15/223618117592s.shtml |title=李敖发布新书《阳痿美国》 |publisher=新浪网 |accessdate= 2010年9月15日 |work= |language =}}</ref><ref>{{cite web |url=http://news.cnr.cn/xwtz/201009/t20100915_507055067.html |title=李敖在台北发布新书《阳痿美国》 |publisher=中国广播网 |accessdate=2010-09-15 |work= |language =}}</ref> ，前者由[[李敖出版社|李敖出版社]]出版，後者由[[中信出版社|中信出版社]]出版；但是简体版由于[[中華人民共和國政府|中華人民共和國政府]]的阻扰，到2011年5月1日把书名改为《审判美国》才得以出版面世。李敖在[[新浪微博|新浪微博]]中说道：“同样一本书，在海峡左边叫《审判美国》，海峡右边叫《阳痿美国》，多妙啊！还没审判到[[美国|美国]]，却先审判到[[阳痿|阳痿]]！当年我做[[排长|排长]]的时候，郑班长禁止他手下的小兵看一本叫《青春花朵》的书，理由是书中有‘[[月经|月经]]’两个字；郑班长啊，做班长真委屈了你。”“为了这本书，多少唇舌说破了，多少良机错失了，多少成品被封杀成纸屑了；总算折腾270天后，自阉书名，得以风行祖国了。”“可惜大官人不喜欢美国阳痿，虽然‘阳痿’二字是[[学术名词|学术名词]]；几经疏通，最后蒙准改名出版。阳痿美国岂易事哉，你得先阉了自己。”繁体版为652页，但是简体版被删节了大量内容而只剩388页。

== 内容 ==
李敖在书中是“上帝李”，审判[[美国历史|美国历史]]上的43位[[总统|总统]]，指出每位总统的丑陋言行和不堪的一面，同时控诉了美国的[[民主|民主]]和[[正义|正义]]的虛假。<ref name="duck " >{{cite web |url= http://sh.sina.com.cn/book/new/2011-01-11/09251099.html|title=李敖新书《阳痿美国》震撼登场 |publisher=新浪网 |accessdate= 2011年1月11日 |work= |language =}}</ref>

李敖表示，他的民主思維曾經被美國的[[政治宣傳|政治宣傳]]欺騙，特別是旁觀中國大陸從[[文化大革命|文化大革命]]、[[改革開放|改革開放]]、被[[西方國家|西方國家]][[經濟封鎖|經濟封鎖]]及[[航空母艦|航空母艦]]威脅之後露出[[中國模式|中國模式]]的曙光，讓他反思美式民主的某些問題，並對[[中國共產黨|中國共產黨]]的領導更具好感；此書即是這種時代及思維背景下的作品。

== 艺术特色 ==
全书使用戏剧写成，模拟《[[圣经|圣经]]》上“[[最后的审判|最后的审判]]”的“[[法庭|法庭]]”，美国历史上的每届总统都是[[被告|被告]]；而上帝李是[[法官|法官]]，数落他们的罪状和虚伪的一面。<ref name="duck " />

== 社会反响 ==
李敖在发表《阳痿美国》时说：“美国做了很多坏事；但美国的[[媒体|媒体]]、[[电影|电影]]很强势，把美国做的坏事冲淡了、让人们都淡忘了。虽然它做了不少坏事，但人们还是喜欢美国、希望自己过美国那样的生活。中国人真要过美国人的生活，每个家庭拥有两辆[[轿车|轿车]]，这能行吗？一旦全世界都以美国为标准、都像美国那样生活，那样快速消耗资源，就会消耗三个[[地球|地球]]的资源，就必须多排掉9个地球的污染。”<ref>{{cite web |url=http://news.ifeng.com/opinion/gundong/detail_2011_05/24/6591325_0.shtml |title=“阳痿”美国，美国不能再作恶！ |publisher=凤凰网 |accessdate= 2011年5月24日|work= |language =}}</ref><ref>{{cite web |url=http://news.163.com/11/0110/03/6Q0PISND00014AED.html |title=李敖新书《阳痿美国》登场 |publisher=网易 |accessdate= 2011-01-10|work= |language =}}</ref>

《阳痿美国》入选为《[[亚洲周刊|亚洲周刊]]》2010年度[[亞洲週刊中文十大好書|中文十大好书]]非小说类。

2011年9月13日，[[名嘴|名嘴]][[楊憲宏|楊憲宏]]主持李敖出版社在[[台北喜來登大飯店|台北喜來登大飯店]]舉辦的李敖新書《你笨蛋，你笨蛋》發表會，盛讚李敖的快筆，說李敖從《陽痿美國》到《你笨蛋，你笨蛋》四本書一共寫了一百六十萬字，其著述之力，當代幾乎無人能比<ref>{{cite news|url=http://www.enews.com.tw/?app=view&no=6743|title=李敖《你笨蛋，你笨蛋》|publisher=華人世界時報|author=應瑋漢|date=2011-09-14|accessdate=2014-12-28}}</ref>。
