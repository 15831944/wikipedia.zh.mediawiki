'''阳泉市'''是[[中华人民共和国|中华人民共和国]][[山西省|山西省]]下辖的[[地级市|地级市]]，位于山西省东部，[[太行山脉|太行山脉]]西侧。山西东部经济，文化，政治中心。

== 历史 ==
[[旧石器时代|旧石器时代]]，今阳泉市辖区境内已有人类在此居住。平定县西北部的枣烟、大梁丁以及盂县西烟镇均曾发现过旧石器<ref>{{cite book|title=平定县志（1992年版）、盂县志（1992年版）}}</ref>。

[[西周|西周]]至[[春秋|春秋]]时期，阳泉属于[[晋国|晋国]]。

[[战国|战国]]时期，阳泉属于[[赵国|赵国]]。

[[秦朝|秦]]统一中国后，阳泉归太原郡管辖。

[[西汉|西汉]][[建元_(西汉)|建元]]元年（公元前140年），设上艾县，县治在今天的平定县张庄镇新城村，阳泉归辖于上艾县。

[[唐|唐]][[天宝_(唐朝)|天宝]]元年（742年），上艾县更名广阳县，县治迁至今[[昔阳县|昔阳县]]广阳村。

[[北宋|北宋]][[太平兴国|太平兴国]]四年（979年），广阳县更名为平定县，县治迁回平定县。

[[金朝|金]]，平定县改称平定州。

1900年前，现在的阳泉市区是平定州境内的一个名叫“漾泉”的小山村，也有人认为，这里只是一片名为沙江口的荒滩。

1903年[[石太线|石太铁路]]通车后，采矿业随之兴起。

[[民国|民国]]初年，改平定州为平定县，阳泉归辖于平定县。

1936年，阳泉附近形成了3万余人的小镇。

1947年，中共设立阳泉市，但辖区不包括平定县和盂县。

1983年，[[平定县|平定县]]、[[盂县|盂县]]归辖于阳泉市。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为1368502人<ref>{{cite book |author=阳泉市统计局|title=《阳泉市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加95386人，增长7.49%。其中，男性人口为710304人，占51.9%；女性人口为658198人，占48.1%。总人口性别比（以女性为100）为107.92。0－14岁人口为214449人，占15.67%；15－64岁人口为1044127人，占76.30%；65岁及以上人口为109926人，占8.03%。

=== 方言 ===
市区及平定县方言属于[[晋语|晋语]][[大包片|大包片]]，盂县方言属于[[晋语|晋语]][[并州片|并州片]]。但也有学者认为，阳泉方言整体属于并州片下的阳泉片（也称平辽小片）<ref>{{cite web|url=http://www.cnki.com.cn/Article/CJFDTOTAL-FYZA198602000.htm|title=山西方言的分区}}</ref>。

== 物产 ==
矿藏主要有[[无烟煤|无烟煤]]、[[铝矾土|铝矾土]]、[[硫铁矿|硫铁矿]]、[[石灰石|石灰石]]、耐火粘土、[[铁矿石|铁矿石]]、[[大理石|大理石]]、[[石膏|石膏]]、[[石棉|石棉]]、[[石英|石英]]、[[云母|云母]]、[[长石|长石]]、[[硅石|硅石]]、[[花岗石|花岗石]]、[[蛭石|蛭石]]、[[水晶|水晶]]等，还有[[镍|镍]]、[[铬|铬]]、[[锑|锑]]、[[铜|铜]]、[[钠|钠]]、[[磷|磷]]等稀有元素，阳泉人形象地以黑、白、黄矿为豪（黑煤，白铝矾土，黄硫铁矿）。

== 经济 ==
[[File:阳泉网通大厦.jpg|thumb]]

* 工业以煤炭开采、炼铁为主，兼有矿山机械、[[电力|电力]]、[[化工|化工]]、电子仪表、[[纺织|纺织]]等行业。
* 农村的乡镇工业成为全市经济的支柱。

== 交通 ==
[[石太线|石太复线]]贯穿该市，铁路支线四通八达，是[[华北地区|华北地区]]最大的铁路货运站。公路网也很发达，[[太旧高速|太旧高速]]从境内穿过。
== 名胜古迹 ==
有唐代平阳公主率军驻守的[[娘子关|娘子关]]、[[陽泉關王廟|陽泉關王廟]](陽泉關帝廟)、赵简子古城遗址、汉淮阴侯韩信驻军遗址、藥林寺、烈女祠、百团大战纪念碑、清凉寺、摩崖石刻、蒲台阁、梁家寨温泉、[[藏山|藏山]]等。
