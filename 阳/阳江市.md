'''阳江市'''（[[邮政式拼音|邮政式拼音]]：{{lang|la|'''Yeungkong'''}}<ref>中华民国《1947年广东政区及邮政式广东地名一览》，大部分根据粤语标准音广州话拼写，其他部分由潮汕、客家、官话或其他方音拼写而成</ref>。）是[[中华人民共和国|中华人民共和国]][[广东省|广东省]]下辖的[[地级市|地级市]]，位于广东省西南沿海。现辖[[江城区|江城区]]、[[阳东区|阳东区]]、[[阳西县|阳西县]]、[[阳春市|阳春市]]（县级市），东与[[江门市|江门市]]相邻，西与[[茂名市|茂名市]]交界，北与[[云浮市|云浮市]]相接，南临[[南海|南海]]。

阳江正在阳东区兴建大型[[核电站|核电站]]——[[东平核电站|东平核电站]]。阳江人均GDP位居粤东西北第一位，也是粤东西北首个人均GDP超过全国平均水平的地级市。

== 城市名称 ==
阳江的名称来源于阳江的母亲河——“[[漠阳江|漠阳江]]”，而漠阳江的名称又源“莫阳”，阳江最早叫“莫阳”。因为阳江古时候属于[[古越|古越]]，居住的多是越人。古越语中，“牛”为“莫”，（[[陽江話|阳江话]]现在称“牛”还叫“牛咩”）；“羊”越语为“阳”，和太阳的“阳”。“莫阳”就是牛羊的意思，据此推测古代的阳江曾是牛羊聚居的地方，或者是牛羊集市。而漠阳江江边现有一村，叫牛场村。阳江处于江之北，所以叫阳江。（中国古代地理立位，山北水南曰阴，山南水北曰阳）。至于阳春，古代“青”、“春”谓之东，阳春在江东边，所以名叫阳春。

阳江作为古高凉县、郡的治所，已经有1000多年历史，但相当长的一段时间内，“阳江”还只是个地域概念，未有具体的城市存在。宋代以前的阳江，作为一个地方的文化、经济、政治中心，也不过是一个地形相对高起来的山坡或土墩而已。[[隋|隋]][[大业|大业]]二年（公元606年），阳江正式始为阳江县。[[宋|宋]][[绍圣|绍圣]]四年（1097年）知州[[丁琏|丁琏]]（番禺人）依鼍山（现在江城第一小学周围）筑起一座土城，这就是阳江城的老祖宗。长期以来，阳江的治地中心都在鼍山脚下，因此阳江又称鼍城。<ref>《阳江地方志》</ref>

== 地理 ==
[[File:Yangjiang1.jpg|thumb]]
丘陵面积占陆地总面积的26.03%，山地面积占42.73%，平原面积占22.17%。海岸线长341.5千米，主要岛屿有30个，岛岸线长49.3千米。地势由北向南倾斜，依山傍海，东北有天露山屏障，西北有云雾山环绕。境内最高山峰为望夫山脉的[[鹅凰嶂|鹅凰嶂]]（广东省第二高峰），海拔1337米，最长河流为[[漠阳江|漠阳江]]（广东六大江河之一），全长199千米，南北贯穿全市，自北向南流入南海。<ref>《阳江指南地图》（广东省地图出版社，阳江国土资源局）</ref>

== 旅游 ==
[[File:Yangjiang3.JPG|thumb]][[File:Yangjiang2.jpg|thumb]]
阳江市是广东省著名的滨海旅游城市，旅游资源丰富，有滨海沙滩、峰林溶洞、温泉瀑布、湖光山色和灿烂的人文景观。
*[[海陵岛|海陵岛]]、阳春春湾——凌霄岩为省级旅游度假区，海陵岛的{{link-yue|大角湾|大角灣}}于2001年又被评为4A级国家旅游区。
*拥有阳江温泉、阳春春都温泉、阳西咸水温泉
*广东[[海上丝绸之路|海上丝绸之路]]博物馆将保存“南海一号”等古沉船。

== 物产 ==
*历史上手制[[风筝|风筝]]较多，有南国风筝之乡之称，特色风筝有“灵芝”。
*阳江是目前中国国内主要的刀剪生产出口地。  
*阳江地方特产有[[豆豉|豆豉]]、[[蠔|蠔]]、[[蓋蘇文魚|蓋蘇文魚]]、[[禾蟲|禾蟲]]和全国闻名的阳江[[十八子菜刀|十八子菜刀]]，阳西的凉果，阳春的春砂仁和马水桔。
*南药：春砂仁、益智、檀香、沙姜、藿香等。
*特色小吃：阳江猪肠碌，乒乓糕，杏仁茶，糕点“狗利仔”“酒杯印”。
*其中阳江刀具、豆豉、猪肠碌被众多阳江人“戏誉”为“阳江三件宝”（一说是小刀、豆豉、漆器，但生活中民众更乐于接受前一说法）。

== 名人 ==
* 曾躍鱗：南宋淳熙五年（ 公元一一七八年）高中進士，是有史可稽陽江第一名進士，當過福建福州府羅源縣主簿，盡忠職守，深受當地百姓稱讚
*[[邓琳|邓琳]]（1829-1893），字文石，号鼍阳居士.清代阳江著名教育家、公益家、艺术家。
*[[關山月|關山月]]，原名關澤霈，陽江市江城區埠場人，中國當代著名中國國畫家，美術教育家，嶺南畫派藝術大師，全國人大代表。
*[[蘇天賜|蘇天賜]]，被稱為中國當代油畫家的領軍人物。
*[[何士德|何士德]]，当代音乐家、作曲家、指挥家，新中国电影音乐奠基人之一。
*[[阮退之|阮退之]]，著名詩人。
*[[曾庆存|曾庆存]]，中国科学院大气物理研究所研究员，人阳江市江城区岗列玉沙村。地质学家曾庆丰的弟弟。
*[[劉齊|劉齊]]，奪取蹼泳世界冠軍最多的運動員。
*[[郑源|郑源]]，1982年11月2日生于广东阳江，华语流行男歌手、音乐人、唱作人、词曲创作者、华语乐坛情歌界代表性人物之一，北京戏曲学院音乐教授，毕业于星海音乐学院。
*[[欢子|欢子]]（Fandy Su），1984年9月25日生于广东阳江，中国流行男歌手、演员、词曲创作人、音乐人、企业家、索吻时代公司创始人，毕业于星海音乐学院 。
*[[梁季芙|梁季芙]]（1891～1961年），原名梁名渠，又名梁歡，江城平岡朝東村人。原為首飾工匠，對[[種金屬特性較有研究，工藝精湛，在當地頗有名氣。
* 林天蔚教授，香港珠海大學歴史系教授；香港古代史鼻祖
