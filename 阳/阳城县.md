'''阳城县'''是[[山西省|山西省]][[晋城市|晋城市]]的一个县，位于山西省南端中段，面积1968平方公里，人口約40万，共辖17乡（镇）及1个办事处。

== 历史 ==
阳城古称'''濩泽'''，[[北魏|北魏]]时始称阳城。

== 地理 ==
地势西高东低。

主要河流[[沁河|沁河]]。

== 行政区划 ==
{{PAGENAMEBASE}}下辖10个[[行政建制镇|镇]]、7个[[乡_(中华人民共和国)|乡]]、2个[[类似乡级单位|类似乡级单位]]（1个办事处、1个林场）：{{PRC admin/children|14/05/22}}。

== 交通 ==
* [[侯月铁路|侯月铁路]]
* 晋阳高速公路
* 阳翼高速公路

== 人口 ==
2010年第六次全国人口普查阳城县常住人口388788人。

== 经济 ==
阳城的煤炭产业和旅游业较发达。

2011年地区生产总值达到139.86亿元，财政总收入25.34亿元。<ref>[http://www.jconline.cn/Contents/Channel_8187/2012/0323/802427/content_802427.htm 阳城县2012年政府工作报告]</ref>
