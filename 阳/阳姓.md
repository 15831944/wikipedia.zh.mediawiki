'''陽姓'''是中文姓氏之一。根據中華民國內政部户政司民國107年（2018年）6月的數據，臺灣陽姓人口共有2,692人，為全國姓氏人數排名第184名。<ref>內政部戶政司編，《全國姓名統計分析》，臺北：內政部出版。2018年10月初版一刷</ref> 

==起源==
# 源于'''姬姓'''：出自源于东周时期周景王之子姬樊的封地，属于以国名为氏。在[[周景王|周景王]]时期，将自己的小儿子樊封在阳邑(今河南济源)，世称阳樊。周景王少子的后裔因为躲避周乱来到[[燕国|燕国]]，并以封邑名称为姓氏。
# 源于'''[[鲜卑族|鲜卑族]]姓氏，代北莫胡卢氏'''：后汉化改为阳姓。
# 源于姬姓，[[春秋|春秋]]时期[[鲁国|鲁国]][[季孙氏|季孙氏]]的家臣中有一人名叫[[阳虎|阳虎]]，因封邑在[[阳关|阳关]]（今[[山东|山东]][[泰安|泰安]]南），便以封邑为姓。
# 源于[[姬姓|姬姓]]，出自春秋时期晋国太傅[[阳处父|阳处父]]，属于封邑名称为氏。
# 源于[[芈姓|芈姓]]，出自春秋时期楚国令尹子瑕，属于封邑名称为氏。
# 源于[[契丹族|契丹族]]，出自宋朝时期西辽政权纳喇氏部族，属于汉化改姓为氏。
# 源于蒙古族，出自蒙古[[乃蛮部|乃蛮部]]太阳汗之后，属于汉化改姓为氏。
# 源于避难改姓，出自南宋时期开国男杨大巽，属于避难改姓为氏。
# 源于[[满族|满族]]、[[回族|回族]]、[[侗族|侗族]]、[[土家族|土家族]]、[[布依族|布依族]]等少数民族，属于汉化改姓为氏。
# 源于台灣原住民[[阿美族|阿美族]]氏族名'''Pacidal'''，原義是[[太陽|太陽]]。

==名人==
=== 古代 ===
*[[阳虎|阳虎]]，姬姓陽氏，名虎，東周春秋時期鲁國人，孟孙氏的族人，季孙氏的家宰。
*[[阳处父|阳处父]]，東周春秋時期晉國人，效命於晋文公、晉襄公，涉入所谓“易中军”的人事异动，最后身死名裂。
*[[阳休之|阳休之]]，阳尼重从孙，字子烈。东魏武定年间，官至黄门朗。北齐天统年间，官拜吏部尚书。北周武帝年间，官拜州刺史。著有《幽州人物志》。尼重从孙弼，字世辅，长于吏事，本州别驾加轻车将军。尼重从孙伯庆，汝南太守。
*[[阳城|阳城]]，唐代陕州夏县（今山西省夏县）人。唐德宗时，他出任道州（今湖南省道县）刺史。

=== 近現當代 ===
*[[陽介仁|陽介仁]]（1959年9月16日－）阿美族，為中華民國棒球選手，曾效力於中華職棒[[味全龍|味全龍]]及[[興農牛隊|興農牛隊]]，屬低肩下勾型[[投手|投手]]。
*[[陽建福|陽建福]]（1979年4月22日－）阿美族，為中華民國棒球選手，目前效力於中華職棒義大犀牛隊，是擅長[[滑球|滑球]]的投手。曾經是興農牛時期球隊所倚重的本土[[先發投手|先發投手]]。
*[[陽東益|陽東益]]（1978年12月20日－）阿美族，為中華民國棒球選手，曾效力於中華職棒[[統一獅隊|統一獅隊]]，守備位置為[[游擊手|游擊手]]。2010年底，因戰力外遭球隊釋出。
*[[陽森|陽森]]（1981年3月25日－）阿美族，為中華民國棒球選手，中華民國[[台東縣|台東縣]]人，曾效力於中華職棒統一7-ELEVEn獅隊，守備位置為[[二壘手|二壘手]]。2010年底統一球團以借貸關係複雜為由開除。
*[[陽豐隆|陽豐隆]]（1981年12月25日）阿美族，為中華民國棒球選手，原名陽聖凱，陽建福的胞弟；陽介仁的表侄；陽東益、陽森的表弟。右投右打[[內野手|內野手]]。
*[[陽耀勳|陽耀勳]]，阿美族，與弟弟陽岱鋼、陽耀華合稱陽家旅日三兄弟。曾為日本職棒[[福岡軟體銀行鷹隊|福岡軟體銀行鷹隊]][[左投|左投手]]，後入{{Link-en|艾頓那曲球|Altoona Curve}}，而後加盟國訓中心棒球隊。現為[[Lamigo桃猿隊|Lamigo桃猿隊]][[外野手|外野手]]。
*[[陽岱鋼|陽岱鋼]] - 陽耀勳之弟（原名陽仲壽），阿美族。現為[[日本職棒|日本職棒]][[中央聯盟|中央聯盟]][[讀賣巨人隊|讀賣巨人隊]]外野手。
*[[陽耀華|陽耀華]]（1984年11月20日）阿美族，陽品華，原名陽耀華，與哥哥陽耀勳、弟弟陽岱鋼（原名陽仲壽）三兄弟。目前陽耀華被[[愛媛橘子海盜隊|愛媛橘子海盜隊]]指名，加入了愛媛橘子海盜隊。
（以上皆為阿美族[http://twbsball.dils.tku.edu.tw/wiki/index.php/%E5%8F%B0%E6%9D%B1%E9%99%BD%E5%AE%B6 陽家班]）
*[[陽詠存|陽詠存]]（Esther Yang，1987年5月2日－），漢族，是中華民國影視演員、流行音樂歌手、凱渥模特兒經紀公司旗下的模特兒。
