'''罗伯特·詹姆斯·李·“鮑勃”·霍克'''（{{lang-en|Robert James Lee "Bob" Hawke}}，{{bd|1929年|12月9日|||Hawke}}），前任[[澳大利亚总理|澳大利亚总理]]（1983-1991年）、1956年的澳洲[[羅德學者|羅德學者]]。霍克曾四度带领[[澳大利亚工党|澳大利亚工党]]获得联邦大选的胜利，執政八年九個月，也是迄今为止在出身工党的总理中任期最长的一位。

== 早期生活 ==
鮑勃·霍克出生於[[澳大利亞|澳大利亞]][[南澳大利亞州|南澳州]]博德敦（Bordertown），其中他的叔叔阿尔伯特·霍克更是时任[[西澳大利亚州|西澳大利亚州]]州长（1953-1959年）。后霍克毕业于[[西澳大学|西澳大学]]，并获得了法律与学术的双学位。18岁时，霍克遭遇的的一次车祸改变了他的命运，这促使他开始往更高的目标进发。1954年，在[[羅德獎學金|羅德獎學金]]的贊助下，霍克留学[[牛津大学|牛津大学]][[牛津大学大学学院|大学学院]]，改为研究澳大利亚的工资评定系统，在此期间，霍克创下喝[[啤酒|啤酒]]的[[世界纪录|世界纪录]]（现已被打破），这后来成了许多男性澳大利亚选民支持他的一个原因。

1956年，與妻子Hazel Masterson結婚，育有四個子女，並獲得[[澳洲國立大學|澳洲國立大學]]獎學金，前往[[首都|首都]][[坎培拉|坎培拉]]進修法律，並在入學後，成為學生校務會議代表。1970年代起與Blanche d'Alpuget發展[[婚外情|婚外情]]。後與妻子Hazel Masterson分居，但因為免損害其政治仕途才一直未有對外公開，而其妻子亦一直履行第一夫人職責，直至1991年霍克卸任後方對外宣布分居，並於1995年正式離婚。1995年與Blanche d'Alpuget結婚。

== 政治生涯 ==
在完成学业不久后，霍克开始进入澳大利亚工会理事会（ACTU）工作，为条件不太好的劳工而工作。1969年，霍克当选为全澳总工会主席，从此成为一位具有知名度的人物。1973年霍克成为澳大利亚工党全国主席，并很快作为了当时的[[惠特拉姆|惠特拉姆]]政府中的杰出人物。

[[File:Bob_Hawke_in_1980.jpg|150px]]
1980年，霍克當選聯邦下議院議員，正式涉足联邦政坛。随即他获时任工党领袖[[比尔·海登|比尔·海登]]邀请进入[[影子内阁|影子内阁]]，成为影子就业及劳资关系部长，其欢迎度很快就胜过海登。1982年，时任总理[[马尔科姆·弗雷泽|马尔科姆·弗雷泽]]企图提前大选，而海登领导下的工党却并未有足够的支持夺回执政权，霍克取而代之成为党首的传言也开始散播。

1982年，海登宣布进行党团的领导权投票，但最终霍克以5票之差败给海登。但是很快随着工党在补选中的败北，海登地位再次不保，1983年2月，海登在党内劝说下被迫辞职，而霍克则顺利自动当选工党领袖，宣布代表工黨参选总理，此时离投票已不到一个月。

== 联邦总理 ==
1983年3月5日，工党以压倒优势大败[[澳大利亚自由党|自由党]]，霍克成功当选总理，展開工黨長達13年的執政，当天恰巧也是霍克父亲的生日。霍克上任后，其实行了新的[[澳大利亚内阁|内阁体制]]，只有最重要的部分部长才参加内阁，其余一般部长不参加内阁会议。与此同时，工党党团也建立起更加正式稳定的派系体系，这很大程度上改造了工党党团的运作。

== 政績 ==
霍克是工黨的右派人物，其任內一項最重要的政績，就是加快澳洲國營企業的民營化。正是他在1989年出訪[[韓國|韓國]]時，提出建立亞洲太平洋經濟合作組織（[[APEC|APEC]]）的概念并實施，被認為是近15年來引導了亞洲和太平洋地區經濟的重要舉措。此外，他亦與任內使[[澳洲元|澳洲元]]加入[[匯率|自由浮動機-{制}-]]。

他是君主立憲制的支持者，多次粉碎了共和派的計劃，使澳洲仍然在[[英聯邦|大英國協]]內，並在1986年使澳洲的司法獲得終審權，成立[[澳洲高等法院|澳洲高等法院]]。但他的性格，也成為下台的原因。1991年，連續擔任四屆總理的霍克在時任財政部長、自由派的[[保羅·基廷|保羅·基廷]]多次挑戰丁終在黨團會議中失利，由基廷接任。

1992至1995年間返回母校，擔任[[澳洲國立大學|澳洲國立大學]]（ANU）太平洋研究和社會科學學院兼職教授。

== 趣聞 ==
霍克有一個綽號叫“哭泣的總理”，事緣於1989年[[六四天安门事件|六四天安门事件]]發生當日，中共血腥鎮壓學生運動，他在電視直播發表澳洲對此事的評論時，淚灑當場。
