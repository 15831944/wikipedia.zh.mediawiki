'''{{ruby-py|鄠|-{zh-cn:hù;zh-tw:ㄏㄨˋ;zh-hk:wu<sup>6</sup>}-}}邑区'''，原名'''鄠县'''，1964年[[因用字生僻难认而更名的中国地名列表|更名]]為'''户县'''，位於[[中华人民共和国|中华人民共和国]][[陕西省|陕西省]]中部，[[西安市|西安市]]西南部，是西安市市轄區。鄠邑区位于[[关中平原|关中平原]]中部，[[秦岭|秦岭]]北麓、[[渭河|渭河]]南岸，东接[[长安区|长安区]]、西连[[周至县|周至县]]。辖1个街道办事处、13个建制镇、1个景区管理局、518个行政村、21个社区居民委员会，总面积1282平方公里，总人口60.66万人。<ref>{{Cite web|url=http://www.huxian.gov.cn/ptl/def/def/index_4_271.html|title=走进户县新|accessdate=2017-01-02|work=www.huxian.gov.cn|deadurl=yes|archiveurl=https://web.archive.org/web/20170102172019/http://www.huxian.gov.cn/ptl/def/def/index_4_271.html|archivedate=2017年1月2日|df=}}</ref>

== 历史沿革 ==
[[公元前|公元前]]21世纪至前16世纪，[[夏|夏]]朝为“[[有扈氏|有扈氏]]”部落。这是鄠邑区建置最早的方国，也是“鄠邑区”地名的由来，[[夏朝|夏朝]]的重要事件[[甘之戰|甘之戰]]和[[甘誓|甘誓]]都發生在鄠邑区附近。[[商|商]]时设“崇国”；[[周|周]]时，作“丰邑”；[[秦|秦]]代改“扈”为“鄠”；[[西汉|西汉]]时设“鄠县”。历代延续。1964年9月10日，因鄠字生僻，经国务院批准，将“鄠县”更名为“户县”。原隶属于[[咸阳|咸阳]]地区行政公署。二十世纪八十年代初，同[[周至县|周至县]]一起，划归[[西安市|西安市]]管辖。2016年11月24日，国务院批复同意户县撤县改区，恢复鄠字，设立西安市鄠邑区，以原户县的行政区域为鄠邑区的行政区域，鄠邑区人民政府驻甘亭街道东街7号<ref>{{cite web|title=陕西省人民政府关于同意西安市调整部分行政区划的批复|url=http://www.shaanxi.gov.cn/gk/zfwj/64026.htm|date=2016-12-30 |accessdate=2017-11-30}}</ref>。2017年9月9日，鄠邑区举行撤县设区揭牌仪式<ref>{{cite web |url=http://news.hsw.cn/system/2017/0909/904166.shtml |title=鄠邑今正式撤县设区揭牌 成为西安第11个建制区 |author1=屈妍 |date=2017-09-09 |website=华商网 |publisher=华商报 |access-date=2017-09-09 |quote=}}</ref>。

== 气候与资源 ==
鄠邑区属暖热带半湿润大陆性季风气候区，四季冷暖干湿分明，光、热、水资源丰富。是农业生产和多种经营的好地区。县境内有36条大小河流，均源出秦岭北麓，出山后汇成涝河、新河、太平河、高冠河四大水系、分布全区，贯通南北，涝河北流入渭河，其余三大水系汇入沣河。
 
肥沃的土地是鄠邑区的资源，耕地总面积55万亩，林地面积52.4万亩，木材蓄积量311万立方米。鄠邑区已发现矿藏地及矿藏线索30余处，以大理石储量最大。鄠邑区水资源相对丰富，秦岭山前地带地下300—500米深处还有40度左右的热水。野生动物种类丰富，其中包括属国家一级保护动物的[[羚牛|羚牛]]，属国家二级保护动物的[[长尾雉|长尾雉]]。药材有340种，药用价值和产量大的有[[菖蒲|菖蒲]]、[[乌药|乌药]]、[[柴胡|柴胡]]、[[天麻|天麻]]、[[金银花|金银花]]、[[贝母|贝母]]、[[五味子|五味子]]等，鄠邑区是[[陕西省|陕西省]]乌药主产地之一。

== 人口 ==
全区总人口近57万，其中农业人口47万人，占总人口的84%。民族，以汉族为主，还有回、满等16个少数民族千余。

== 文化 ==
鄠邑区历史悠久，文化氛围浓郁。这里有世界上最早、规模最宏大的[[汉朝|汉]]代“国家铸币厂”——[[钟官城|钟官城]]遗址；世界最早的天文台金峰寺，埋藏[[唐|唐]][[玄奘|玄奘]]法师遗骨的紫阁寺。还有[[草堂寺|草堂寺]]、[[重阳宫|重阳宫]]、[[公输堂|公输堂]]等古迹。高冠瀑布、紫阁峪更是留下了[[李白|李白]]、[[杜甫|杜甫]]等大诗人的名篇佳作。

自1970年代以来，鄠邑区以其秀美的山川景色和良好的人文环境，孕育出了名闻遐迩的“[[户县农民画|户县农民画]]”。并涌现出了一大批诸如[[李凤兰_(画家)|李凤兰]]、[[潘晓玲|潘晓玲]]等著名农民画家；而鄠邑区，也以农民画乡而饮誉海内外。

== 经济发展 ==
直到1950年代，鄠邑区才开展了大规模的工业建设。1970年代中后期，在时任[[中共|中共]]户县县委书记[[苏耀先|苏耀先]]的领导下，户县展开了大规模的“园田化”建设。新修并拓宽道路、绿化街道、打建饮水设施等。目前鄠邑区已经形成了以制药、造纸、化工、电力、建材、机械制造、农副产品加工、工艺铸造等为基础的较为完备的工业产业体系。

== 健康卫生 ==
户县为流行性[[出血热病毒|出血热]]的传统疫区，该区出血热病例报告与其他地方相比较多<ref>[https://3g.163.com/all/article/E3IUU4770001875P.html 高校老鼠肆虐学生感染出血热死亡:床铺沾满老鼠屎] 中国青年网 2018-12-21 </ref><ref>[http://html.rhhz.net/zhlxbx/20180904.htm 渭河流域2005－2015年肾综合征出血热流行特征及环境危险因素分析.]  李雪, 刘昆, 谷旭, 袁筱婕, 邵中军. 中华流行病学杂志, 2018, 39(9): 1159-1164</ref>。1956年出现首例病人后，疫区不断扩大，涉及全县所有乡镇。在1974年秋冬户县宋村等地就爆发过出血热，短短几个月累计284人发病，仅宋村就死亡20个壮劳力<ref>[http://www.china.com.cn/chinese/feature/381995.htm 姜素椿鲜为人知的故事] 孙晶岩 黄显斌 生活时报2003-08-06</ref>。在1970~80年代，这里的冬春季流行性出血热发病率高，伤亡也重<ref>[http://www.jrzx.com/article.php?cid=261&id=835 孙文雅：传承中医 心系百姓] 2018-07-27 各界导报-各界新闻网 </ref>。2010年统计发病282例，死亡6例，发病率48.34/10万，病死率2.13%<ref>[http://yxrmyy0662.com/Item/17.aspx 2016年11月7日新闻发布会材料三：做好公卫工作 为百姓健康安全保驾护航] 阳西县人民医院</ref>。
