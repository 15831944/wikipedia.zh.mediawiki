'''程宽德'''（{{bd|1937年||1991年|12月25日}}），藏名'''泽仁多吉'''（{{bo|z=ཚེ་རིང་རྡོ་རྗེ།|w=Tsering Dorjee}}），[[四川|四川]][[德格|德格]]人，[[中国人民解放军|中国人民解放军]]军官。

== 生平 ==
程宽德出生于[[中华民国|中华民国]][[四川省_(中華民國)|四川省]][[西康行政督察区|西康行政督察区]][[德格县|德格县]]（今中华人民共和国[[四川省|四川省]][[甘孜藏族自治州|甘孜藏族自治州]]德格县）。父亲为[[汉族|汉族]]，曾在[[中国国民党|中国国民党]]某部任[[中校|中校]]副官，后加入[[中国工农红军|中国工农红军]]，因负伤留在了四川。母亲是德格当地的[[康巴人|康巴藏人]]。1950年，[[中国人民解放军第18军|中国人民解放军第18军]]途经德格时，13岁的程宽德参军，并随之入藏，担任翻译。1954年1月，作为[[中国共产党|中国共产党]]要培养的新生力量，程宽德曾被送到西藏军区干部学校学习藏语文，同年9月到成都西南民族学院政治班学习<ref name=a123 />。1955年12月毕业后，分配在西藏军区则拉宗边防总站工作，任见习参谋、参谋。

1956年10月1日，作为[[西藏军区|西藏军区]]代表，去[[北京|北京]]参加国庆阅兵，并因此登上《[[解放军画报|解放军画报]]》封面<ref name="三代">{{cite web |author1=唯色 |title=我的相册——我家三代 |url=http://woeser.middle-way.net/2008/11/blog-post_1258.html |website=看不见的西藏 |date=2008-11-26 |access-date=2018-11-03 |archive-url=https://web.archive.org/web/20180723212510/http://woeser.middle-way.net/2008/11/blog-post_1258.html |archive-date=2018-07-23 |dead-url=no }}</ref><ref name="追思">{{cite web |author1=王沁鸥 |title=拉萨烈士陵园里的追思 |url=http://www.xinhuanet.com/mil/2018-04/06/c_129844847.htm |website=新华网 |accessdate=2018-11-03 |date=2018-04-06}}</ref>。1957年4月30日，加入中国共产党。同年5月，调西藏军区情报部任参谋，并在西藏地方干部学校西藏军区藏语文队学习藏语文两年。1959年调到西藏军区政治部联络部任助理员，参加了平息西藏的[[1959年西藏叛乱|武装叛乱战斗]]和收容俘虏的工作。1962年10月参加了[[中印边境自卫反击战|中印边境自卫反击战]]，并在琼结、隆子等地作收容印军俘虏的工作。1964年2月，参加西藏人民广播电台组建和开展《对流落国外的藏族同胞的广播》节目。1966年2月任边防科副科长<ref name=a123 />。

[[文化大革命|文化大革命]]期间，因部队里派系斗争，被调往四川省甘孜藏族自治州的驻军单位<ref name="三代" />。1970年6月，任四川省[[康定军分区|康定军分区]][[道孚县|道孚县]][[人民武装部|人民武装部]]副部长。1976年6月，担任人民武装部部长<ref>{{Cite book|title=道孚县志|last=四川省道孚县志编纂委员会编纂|first=|publisher=成都：四川人民出版社|year=1998.02|isbn=7-220-03781-3|location=|pages=415}}</ref>。1979年1月，任西藏军区生产建设师和日喀则军分区政治部副主任。1979年11月，任四川省军区甘孜军分区副参谋长。1983年7月，任该军分区参谋长。1988年9月，授予大校军衔。1989年12月，任军分区副司令员兼参谋长。1990年6月，任[[中国人民解放军西藏军区|西藏军区]][[拉萨军分区|拉萨军分区]]副司令员<ref name="三代" /><ref name="镜头">{{cite news|url=https://cn.nytimes.com/china/20160831/tibet-culture-revolution-photography/|title=镜头里的西藏文革（一）：跟随父亲的影迹|newspaper=[[纽约时报|纽约时报]]中文网|author=罗四鸰|date=2016年8月31日|access-date=2018-11-03|archive-url=https://web.archive.org/web/20180723212401/https://cn.nytimes.com/china/20160831/tibet-culture-revolution-photography/|archive-date=2018-07-23|dead-url=no}}</ref>。1991年12月25日，因高血压病引起突发性脑溢血，抢救无效，在[[西藏军区总医院|西藏军区总医院]]去世，享年54岁<ref name=a123>{{Cite book|title=甘孜藏族自治州军事志|last=四川省甘孜军分区《军事志》编纂委员会|first=|publisher=四川省甘孜军分区《军事志》编纂委员会|year=1999.05|isbn=|location=|pages=388-389}}</ref>。

== 家人 ==
[[File:Cheng_Kuande_and_His_Wife.jpg|缩略图]]
1965年9月，程宽德结婚。妻子是[[日喀则|日喀则]]地区藏人，曾在拉萨藏干校、[[中央政法干部学校|中央政法干部学校]]学习，其父亲曾是[[噶厦|噶厦]][[噶伦|噶伦]]、[[昌都总管|昌都总管]][[拉鲁·次旺多吉|拉鲁·次旺多吉]]的管家<ref name="三代" />。

长女[[程文萨|程文萨]]（藏名茨仁唯色），1966年7月出生，女诗人、作家。另有女儿程文红<ref name="追思" />，及一个1971年在道孚出生的儿子<ref name="三代" />。
