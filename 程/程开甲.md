'''程开甲'''（{{bd|1918年|8月3日|2018年|11月17日|catIdx=C程}}），[[江苏省|江苏省]][[吴江市|吴江市]]（今[[江苏省|江苏省]][[苏州市|苏州市]][[吴江区|吴江区]]）人。[[中华人民共和国|中华人民共和国]]核武器技术专家，[[中国科学院|中国科学院]]院士，被誉为“[[两弹一星|两弹一星]]”的元勋。2018年11月17日上午在北京病逝，享年100岁<ref name="death">{{cite news|url=https://mil.news.sina.com.cn/china/2018-11-17/doc-ihmutuec1128787.shtml|title=101岁“中国核司令”程开甲逝世：获两弹一星功勋章|date=17 Nov 2018|work=澎湃新闻}}</ref>。

== 生平 ==
1931年，13岁的程开甲离开家乡江苏吴江，考入距家20公里的嘉兴[[秀州中学|秀州中学]]。1937年程开甲获浙江省教会中学英文演讲比赛第一名。<ref>{{Cite web |url=http://www.sohu.com/a/276222109_578930|title=秀州中学沉痛送别程开 |accessdate=2018-11-17 |work=搜狐教育- 搜狐网}}</ref>1941年毕业于[[浙江大学|浙江大学]]物理系，后留校任教至1946年，留校期间完成了多项重要的相对论和基本粒子的研究<ref>{{Cite web |url=https://news.sina.com.cn/2018-11-17/doc-ihmutuec1127137.shtml |title=“中国核司令”程开甲院士逝世 享年101岁 |accessdate=2018-11-17 |work=新浪新闻 - 新浪网}}</ref>。1946年赴[[英国|英国]][[爱丁堡大学|爱丁堡大学]]留学，为物理学家[[马克斯·玻恩|马克斯·玻恩]]的研究生，期间主要从事超导电性双带理论的研究。1948年获博士学位。之后任英国皇家化学工业研究所研究员。1950年回到浙江大学物理系任教。1952年调到[[南京大学|南京大学]]物理系，工作近十年，曾任物理系副系主任。1960年起参与中国第一颗原子弹研究，1962年调任[[中国人民解放军国防科学技术委员会|国防科委]][[中国核试验基地|核试验基地]]研究所副所长、所长。他是中国第一颗[[原子弹|原子弹]]研制的开拓者之一、[[核武器|核武器]]试验事业的创始人之一，[[核试验|核试验]]总体技术的设计者<ref>{{Cite web|url=http://www.people.com.cn/GB/keji/25509/34159/34161/2918544.html|title=两弹一星功勋：核武器专家程开甲院士|date=2004-10-14|publisher=人民网}}</ref>。后任[[国防科工委|国防科工委]]科学技术委员会正军职常任委员<ref name=jing/>。

1984年，调任北京，任国防科工委（总装备部）科技委常任委员、顾问，国家超导专家委员会顾问<ref>{{Cite web|url=http://news.sina.com.cn/2018-11-17/doc-ihmutuec1127137.shtml|title=“中国核司令”程开甲院士逝世 享年101岁|accessdate=2018-11-17|author=|date=|work=新浪新闻|publisher=}}</ref>。

1999年，获中共中央、国务院、中央军委授予“[[两弹一星|两弹一星]]”功勋奖章。2014年1月10日，和化学物理学家[[张存浩|张存浩]]同获[[国家最高科学技术奖|中国国家最高科学技术奖]]<ref>{{cite web|url=http://news.163.com/14/0110/10/9I7IP7400001124J.html|title=2013年国家最高科学技术奖得主：张存浩、程开甲|date=2014-01-10|publisher=网易}}</ref>。

2017年7月，中央军委主席[[习近平|习近平]]签署命令，授予程开甲等10位同志“[[八一勋章_(2017年)|八一勋章]]”<ref name="jing">{{Cite web|url=http://news.ifeng.com/a/20170728/51522244_0.shtml|title=景海鹏、程开甲、王刚等10人被授予“八一勋章”|date=2017-07-28|publisher=凤凰网}}</ref>。2017年7月28日，中央军委颁授“八一勋章”和授予荣誉称号仪式在北京[[八一大楼|八一大楼]]举行，中共中央总书记、国家主席、中央军委主席习近平向程开甲等10位“八一勋章”获得者颁授勋章和证书。这是“八一勋章”首次颁授。程开甲同时获授“忠诚奉献、科技报国的‘两弹一星’元勋”荣誉称号<ref>{{Cite web|url=http://news.ifeng.com/a/20170728/51524452_0.shtml|title=习近平向“八一勋章”获得者颁授勋章和证书|date=2017-07-28|publisher=凤凰网}}</ref>。2018年3月，获颁“影响世界华人大奖”终身成就奖<ref>{{Cite web|url=http://phtv.ifeng.com/a/20180330/44670935_0.shtml|title=程开甲院士获颁“影响世界华人大奖”终身成就奖|accessdate=2018-11-17|author=|date=|work=凤凰网卫视|publisher=}}</ref>。2019年2月，当选为2018年[[感动中国|感动中国]]年度人物<ref>{{Cite web|url=http://china.zjol.com.cn/gnxw/201902/t20190219_9484409.shtml|title=程开甲钟扬等荣膺 感动中国年度人物|accessdate=2019-02-19|author=|date=2019-02-19|work=浙江在线|publisher=}}</ref>。
