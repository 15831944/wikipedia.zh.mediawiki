'''程冬瑤'''，（英文名︰Yuna Ching），[[香港|香港]]出生，是香港聾人機構「[[龍耳|龍耳]]」的手語翻譯員，亦是[[龍耳電視|龍耳電視]]的手語主播<ref>{{cite web |language=zh-hk |url=http://www.silencetv.hk/program/%E9%BE%8D%E8%80%B3%E9%9B%BB%E8%A6%96%E6%89%8B%E8%AA%9E%E6%96%B0%E8%81%9E-5/ | title=龍耳電視手語新聞 |publisher=[[龍耳電視|龍耳電視]] |date=2018-04-27 |accessdate=2018-08-06}}</ref>及主持<ref>{{cite web |language=zh-hk |url=http://www.silencetv.hk/program/%E9%BD%8A%E9%BD%8A%E8%B7%A8%E9%9A%9C%E7%A4%99%EF%BC%9A%E8%AE%93%E6%84%9B%E3%80%8C%E5%8C%85%E3%80%8D%E5%9C%8D/ | title=齊齊跨障礙：讓愛「包」圍 |publisher=[[龍耳電視|龍耳電視]] |date=2018-04-27 |accessdate=2018-08-06}}</ref>。現她符合《[[香港手語翻譯員名單|香港手語翻譯員名單]]》基本條件的手語翻譯員。她亦有和另一手語翻譯員[[馬芬燕|馬芬燕]]參與[[香港電台|香港電台]]電視節目「[[早辰。早晨|早辰。早晨]]」<ref>{{cite web |language=zh-hk |url=http://www.rthk.hk/tv/dtt31/programme/thismorning/episode/493335 | title=手語直播節目︰居屋定價；前特首梁振英撰文反駁《明報》前總編輯劉進圖文章 |publisher=[[香港電台|香港電台]] |date=2018-06-28 |accessdate=2018-07-19}}</ref>及[[無綫電視|無綫電視]]新聞節目「[[手語新聞報道|手語新聞報道]]」<ref>{{cite web |language=zh-hk |url=http://news.tvb.com/programmes/newsreportwithsignlanguage/5b4cb3efe60383f57d01797e | title=手語新聞報道 |publisher=[[無綫新聞|無綫新聞]] |date=2018-07-16 |accessdate=2018-07-19}}</ref>的[[手語傳譯|手語傳譯]]工作。
