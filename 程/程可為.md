'''程可為'''（英文名：'''Rainbow Ching Hor Wai'''，{{bd|1950年|10月10日}}<ref name=tvb/>），[[本名|本名]]'''楊蓮桂'''，人稱「程大姑」，香港出生；是無綫早期的二線女演員，更曾是無線旗下備受力捧的花旦之一。本地[[甘草演員|甘草演員]]，现为[[香港無線電視|TVB]]部頭合約女艺员。

<br />

== 个人履历 ==
1972年中五畢業後，她曾投考不同的工作，包括空姐、船艙臨時護理員、[[麗的電視|麗的電視]]（[[亞洲電視|亞洲電視]]前身）及[[無綫電視|無綫電視]]，當時兩份空姐工作均被取錄，惟因家人認為相比而言，當年演員的工作比較空姐工作較多留在香港，相對更安全<ref name=":0">{{Cite news|url=https://topick.hket.com/article/2294474/%E3%80%8A%E7%A6%8F%E7%88%BE%E6%91%A9%E5%B8%AB%E5%A5%B6%E3%80%8B%E7%A8%8B%E5%8F%AF%E7%82%BA%E6%9B%BE%E7%95%B6%E8%BC%94%E8%AD%A6%E8%A3%9C%E8%B2%BC%E6%97%A5%E5%B8%B8%E9%96%8B%E6%94%AF%E3%80%80%E5%85%A5%E8%A1%8C47%E5%B9%B4%E5%BE%9E%E4%B8%8D%E8%A8%88%E8%BC%83%E8%A7%92%E8%89%B2%E5%A4%A7%E5%B0%8F%E5%85%A8%E5%8A%9B%E4%BB%A5%E8%B5%B4%E3%80%80?mtc=10016|title=《福爾摩師奶》程可為曾當輔警補貼日常開支　入行47年從不計較角色大小全力以赴|author=鄭嘉瑤|date=2019/03/22|work=TOPick|accessdate=2019/04/02|publisher=香港經濟日報}}</ref>，於是她最後決定投身演藝行業，參加了1972年第1期[[無綫電視藝員訓練班|無綫電視藝員訓練班]]，與[[甘國亮|甘國亮]]、[[許紹雄|許紹雄]]等人為同學。<ref>{{Cite news|url=https://skypost.ulifestyle.com.hk/article/2290355/%E7%A8%8B%E5%8F%AF%E7%82%BA%E6%8D%B1%E5%88%B0%E6%82%A3%E7%94%B2%E4%BA%A2%E5%87%B8%E7%9C%BC%20%E5%BE%A1%E7%94%A8%E6%85%88%E6%AF%8D%E5%AF%A7%E9%A1%98%E5%96%AE%E8%BA%AB%E9%9D%A0%E8%87%AA%E5%B7%B1|title=程可為捱到患甲亢凸眼 御用慈母寧願單身靠自己|author=|date=2019/03/11|work=晴報|accessdate=2019/04/02}}</ref>藝名為她本人在訓練班時期所取，意為「將來的前程樣樣都可以有所作為」。<ref name=":0" />

畢業後加入[[無綫電視|無綫電視]]<ref name="a">[http://programme.tvb.com/variety/greenroom/episode/20151119/ 今日VIP 2015.11.19 - 程可為]</ref>，是無綫早期的二線女演員，更曾是無線旗下備受力捧的小花之一（其餘小花為[[莊文清|莊文清]]、[[高妙思|高妙思]]、[[盧宛茵|盧宛茵]]等）一直至1980年代中。在訓練班學習以至成為演員的早期，她曾經投考[[香港輔警|香港輔警]]，以幫補收入。<ref name=":0" />

程可為曾在1984年移民[[温哥華|温哥華]]，並於1996年回流香港<ref name=interview>{{cite web |author1=林通賢 |title=【一日四劇】無綫御用阿媽程可為恥笑頒獎禮：畀嚿膠我做咩？ |url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/44516/%E4%B8%80%E6%97%A5%E5%9B%9B%E5%8A%87-%E7%84%A1%E7%B6%AB%E5%BE%A1%E7%94%A8%E9%98%BF%E5%AA%BD%E7%A8%8B%E5%8F%AF%E7%82%BA%E6%81%A5%E7%AC%91%E9%A0%92%E7%8D%8E%E7%A6%AE-%E7%95%80%E5%9A%BF%E8%86%A0%E6%88%91%E5%81%9A%E5%92%A9 |publisher=香港01|accessdate=2019/04/02}}</ref>，參演劇集至今。{{Fact|回流前曾在加拿大任職地產經紀。|time=2019-04-02T02:59:48+00:00}}

她擁有多年的演出經驗，演出不少大獲好評的角色，例如《[[紫禁驚雷|紫禁驚雷]]》中威風八面、氣勢凌人的[[博爾濟吉特氏|博爾濟吉特氏]]（孝莊太-{后}-、太皇太-{后}-）、《[[萬凰之王|萬凰之王]]》的崔嬤嬤、《[[耀舞長安|耀舞長安]]》的[[長孫皇后|長孫皇后]]、《[[初五啟市錄|初五啟市錄]]》的唐玉卿及《[[守業者|守業者]]》的陳婉琴等。一名資深的甘草演員，由於她經常飾演主角的母親或長輩，因此她以慈母形象深入民心。然而她本人對飾演很多次的母親角色的印象都不深，反而對《[[深宮計|深宮計]]》狡猾的蔡仲屏（蔡尚宮）更記憶猶新。<ref name=interview/>

她曾在2003年、2004年、2005年及2011年分別憑劇集《[[俗世情真|俗世情真]]》內飾演的「馬翠花」、《[[烽火奇遇結良緣|烽火奇遇結良緣]]》內飾演的「柳金花」、《[[血薦軒轅|血薦軒轅]]》內飾演的「何氏」、《[[青出於藍|青出於藍]]》內飾演的「文美鳳」、《[[心花放|心花放]]》內飾演的「鄧雪君」及《[[紫禁驚雷|紫禁驚雷]]》內飾演的「孝莊文皇后」獲得當年的[[萬千星輝頒獎典禮|萬千星輝頒獎典禮]]「最佳女配角」提名。

近年為無綫電視監製[[梁材遠|梁材遠]]的常用演員。

她是一名資深的甘草演員，擁有多年的演出經驗，演技早已得到廣泛的觀眾認同，她通常飾演主角的母親或長輩，在《[[師奶強人|師奶強人]]》、《[[流金歲月|流金歲月]]》、《[[烽火奇遇結良緣|烽火奇遇結良緣]]》、《[[老友狗狗|老友狗狗]]》、《[[情越雙白線|情越雙白線]]》、《[[東山飄雨西關晴|東山飄雨西關晴]]》、《[[怒火街頭|怒火街頭]]》、《[[我的如意狼君|我的如意狼君]]》及《[[守業者|守業者]]》等劇飾演的慈母形象，早已深入民心。

此外，她一直默默耕耘，演出不少大獲好評的角色，包括《[[宮心計|宮心計]]》中狡猾的蔡仲屏（蔡尚宮）、《[[紫禁驚雷|紫禁驚雷]]》中威風八面、氣勢凌人的[[博爾濟吉特氏|博爾濟吉特氏]]（孝莊太-{后}-、太皇太-{后}-）、《[[萬凰之王|萬凰之王]]》的崔嬤嬤、《[[耀舞長安|耀舞長安]]》的[[長孫皇后|長孫皇后]]、《[[初五啟市錄|初五啟市錄]]》的唐玉卿及《[[守業者|守業者]]》的陳婉琴等，深得觀眾肯定。她曾在2003年、2004年、2005年及2011年分別憑劇集《[[俗世情真|俗世情真]]》內飾演的「馬翠花」、《[[烽火奇遇結良緣|烽火奇遇結良緣]]》內飾演的「柳金花」、《[[血薦軒轅|血薦軒轅]]》內飾演的「何氏」、《[[青出於藍|青出於藍]]》內飾演的「文美鳳」、《[[心花放|心花放]]》內飾演的「鄧雪君」及《[[紫禁驚雷|紫禁驚雷]]》內飾演的「孝莊文皇后」獲得當年的萬千星輝頒獎典禮「最佳女配角」提名。

近年為無綫電視監製[[梁材遠|梁材遠]]的常用演員。
