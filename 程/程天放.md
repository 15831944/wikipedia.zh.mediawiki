'''程天放'''（{{bd|1899年||1967年|11月29日|catIdx=Cheng程}}），原名'''學愉'''，字'''佳士'''，號'''少芝'''，[[江西省|江西省]][[新建縣|新建縣]]人，生於浙江[[杭州|杭州]]，[[程矞采|程矞采]]曾孙，[[中華民國|中華民國]]政治人物、教育家、外交官。<ref name=xu>{{Cite book|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref name=kmt>{{Cite book|author = 劉国銘主編|title = 中国国民党百年人物全書|year = 2005|publisher = 团結出版社|isbn = 7-80214-039-0}}</ref><ref name=zx>{{Cite book|author = 東亜問題調査会|title = 最新支那要人伝|year = 1941|publisher = [[朝日新聞社|朝日新聞社]]|isbn = }}</ref>

== 生平 ==
=== 从五四運動到AB团 ===
[[File:程天放.jpg|thumb]]
1912年（[[民国紀元|民国]]元年），程天放入省城洪都中学学习。其後转入私立[[心远中学|心远中学]]。1917年，入[[上海|上海]][[復旦公學|復旦公學]]。1919年（民国8年）[[五四運動|五四運動]]爆发，程天放出任上海学生聯合会副評議長。1919年秋，当选为上海学生聯合会会長。1919年，自[[復旦大学|復旦大学]]毕业，赴[[美国|美国]]留学，入[[芝加哥大学|芝加哥大学]]哲学系，后来转入[[伊利诺伊大学|伊利诺伊大学]]政治学系。1922年获得学士学位后，赴[[加拿大|加拿大]]，入[[多伦多大学|多伦多大学]]从事研究。其間，1923年加入[[中国国民党|中国国民党]]，并任该党机关报《醒獅日報》总編輯兼该党加拿大東部支部長。1926年获政治学博士，同年8月归国。<ref name=xu/><ref name=kmt/><ref name=zx/>

归国後，程天放在[[復旦大学|復旦大学]]及[[大夏大学|大夏大学]]担任教授。1927年（民国16年）1月，任[[中国国民党|中国国民党]][[江西省_(中華民国)|江西省]]党部執行委員兼宣传部部長。翌月，任江西省政府教育厅厅長。其間，程天放作为反共的「[[AB团|AB团]]」的领导人，同江西省内的[[中国共产党|中国共产党]]勢力进行了激烈的权力斗争。4月，程天放被中国共产党方面逮捕，即将被处决。同年，江西省政府主席[[朱培徳|朱培徳]]开始接近[[蒋介石|蒋介石]]的[[南京国民政府|南京国民政府]]，同年5月末程天放被释放。<ref name=xu/><ref name=kmt/><ref name=zx/>

=== 教育行政部門的活動 ===
[[File:Cheng_Tianfang2.jpg|thumb]]》中的程天放照片]]
1927年12月，程天放任[[国立中央大学|国立中央大学]]教授兼[[国民政府|国民政府]]参事。1928年（民国17年），任中央軍校軍官团政治总教官兼政治部主任。同年末，任[[安徽省_(中華民国)|安徽省]]政府教育厅厅長。1929年（民国18年）1月，任[[安徽大学|省立安徽大学]]校長。3月，当选中国国民党第三届候补中央執行委員。<ref name=xu/><ref name=kmt/><ref name=zx/>

1931年（民国20年）6月，程天放出任中国国民党中央党部宣传部副部長。同年12月，兼任江西省政府教育厅厅長。1932年秋，兼任陸海軍总司令部党政委員会委員。1932年12月，当选中国国民党第四届候补中央執行委員。1933年（民国22年）2月，任[[湖北省_(中華民国)|湖北省]]教育厅厅長。同年，任[[国立浙江大学|国立浙江大学]]校長。1934年（民国23年）9月，调任[[国立中央政治学校|国立中央政治学校]]教務主任。11月，任[[江蘇省_(中華民国)|江蘇省]]教育厅厅長。1935年（民国24年）6月，被任命为中国駐德国大使。同年11月当选中国国民党第五届中央監察委員。<ref name=xu/><ref name=kmt/><ref name=zx/>

[[抗日战争|抗日战争]]爆发後的1939年（民国28年）3月，程天放任[[国立四川大学|国立四川大学]]校長。同年9月，任[[三民主義青年团|三民主義青年团]]中央監察会監察。1943年（民国32年），任[[国立中央政治学校|国立中央政治学校]]教育長，兼任国防最高委員会常務委員。1945年（民国34年）5月，当选中国国民党第六届中央監察委員。9月，任[[联合国教育、科学及文化组织|联合国教育、科学及文化组织]]憲章制定会議中国代表。其後，历任中国国民党監察委員会常務委員、[[立法院|立法院]]立法委員、立法院外交委員会委員、中国国民党中央宣传部部長。<ref name=xu/><ref name=kmt/><ref name=zx/>

=== 台湾的活動 ===
[[File:Xinjian_Wangshan_Tuku_20120626-12.jpg|thumb]]]]
[[第二次国共内战|第二次国共内战]]末期，程天放随国民政府撤往[[台湾|台湾]]。1950年（民国39年）3月，升任[[中華民国教育部|教育部]]部長，任职至1954年（民国43年）5月，为整顿台湾的教育環境作出贡献。1958年（民国47年），被任命为[[考試院|考試院]]副院長。此外，还兼任[[总統府国策顧問|总統府国策顧問]]，并当选中国国民党第八、第九届中央評議委員。<ref name=xu/><ref name=kmt/><ref>{{cite web|title=程天放|author=|url=http://www.ncnews.com.cn/jdnc/ncmr/t20060529_45806.htm|publisher=南昌新闻网|date=2006-05-29|access-date=2012-07-10|archive-url=https://web.archive.org/web/20110723122342/http://www.ncnews.com.cn/jdnc/ncmr/t20060529_45806.htm|archive-date=2011-07-23|dead-url=yes}}</ref>

1967年11月29日，程天放在[[纽约|纽约]]逝世。享年69岁（满68歳）。<ref name=xu/><ref name=kmt/>
