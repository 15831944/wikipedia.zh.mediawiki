'''（糊涂王）查理三世'''（'''{{lang-fr|Charles III le Simple}}'''，{{bd|879年|9月17日|929年|10月7日}}）[[法國君主列表|西法蘭克國王]]（898年－929年在位）。他属于已衰落的[[加洛林王朝|加洛林王朝]]，是国王[[路易二世_(西法兰克)|路易二世]]的遗腹子。

== 生平 ==

查理三世在其同父异母的兄长[[卡洛曼二世_(西法兰克)|卡洛曼二世]]去世时因年幼未能即位，而由東法蘭克國王[[胖子查理|胖子查理]]统治法国。胖子查理去世后，查理三世又被排除于王位继承人之外。[[罗贝坦王朝|罗贝坦王朝]]的[[巴黎伯爵|巴黎伯爵]][[厄德一世_(西法兰克)|厄德]]继承国王的权力，而一小部分有势力的贵族在893年年宣布查理才是合法的法国国王。然而直到898年厄德去世后，查理三世的王位才被大多数诸侯承认。

[[北欧海盗|北欧海盗]]对西法蘭克的侵袭已有很长一段时间，他们在[[诺曼底|诺曼底]]建立一个小型殖民地。为安抚这支危险的力量，查理三世于911年授予北欧人首领[[羅洛_(諾曼第)|羅洛]]以公爵称号。在罗洛的封地上最终出现强大的[[诺曼底公国|诺曼底公国]]。同年查理三世趁東法蘭克王國的最後一個加洛林家族的國王[[孩童路易|孩童路易]]死去而陷于内讧之机，把[[洛林|洛林]]并入西法蘭克。

920年，西法蘭克贵族在[[苏瓦松|苏瓦松]]集会，宣布取消对查理三世的效忠宣誓。922年诸侯们推举卡佩家族的[[法兰西公爵|法兰西公爵]]罗贝尔为国王（即[[罗贝尔一世_(法兰西)|罗贝尔一世]]）。查理三世在苏瓦松战胜罗贝尔一世<ref>罗贝尔阵亡。</ref>。然而诸侯们又推举罗贝尔的女婿[[勃艮第公爵|勃艮第公爵]][[拉乌尔一世_(西法兰克)|拉乌尔]]继续与他作对。查理三世在923年的叛乱中被弗尔芒杜瓦伯爵[[埃尔贝尔二世|埃尔贝尔二世]]俘虏，关押于[[佩罗讷|佩罗讷]]的城堡中。

== 死亡 ==

在监禁的第6年的929年，10月7日，查理死于[[佩罗讷|佩罗讷]]的城堡，享年50岁。

== 配偶与子女 ==

夏尔先后与[[芙雷德温尼|芙雷德温尼]]、[[威塞克斯的埃德吉芙|威塞克斯的埃德吉芙]]结婚，所有子女中只有[[路易四世_(法兰西)|路易四世]]当上了国王。
