'''查島鸚鵡'''（''Cyanoramphus forbesi''），又名'''佛氏黃額長尾鸚鵡'''，是[[查塔姆群島|查塔姆群島]]內特有的一種稀有[[鸚鵡|鸚鵡]]。

== 分佈及棲息地 ==
查島鸚鵡棲息在[[查塔姆群島|查塔姆群島]]及[[匹特島|匹特島]]的矮樹林。由於獵殺、[[家貓|家貓]]的入侵及失去棲息地，牠們的數量於1938年在[[芒格爾|芒格爾]]只餘下100隻。雖然後來芒格爾進行植林計劃而令牠們的數量有所回升，但卻都仍只限於生活在芒格爾之內。<ref>{{cite web |url=http://www.birdlife.org/datazone/search/species_search.html?action=SpcHTMDetails.asp&sid=9814&m=0 |title=Chatham Parakeet - BirdLife Species Factsheet |publisher=BirdLife International |accessdate=7 April 2009}}</ref>

== 食性 ==
查島鸚鵡主要吃[[無脊椎動物|無脊椎動物]]，但也會吃[[花朵|花朵]]、[[種子|種子]]及[[葉子|葉子]]。
