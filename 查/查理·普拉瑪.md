'''查理·普拉瑪'''（{{lang-en|'''Charlie Plummer'''}}，{{bd|1999年|5月24日}}）是一名美國電影和電視劇演員，普拉瑪在演出電視劇《[[酒私風雲|酒私風雲]]》和《''{{link-en|Granite Flats|Granite Flats}}''》前曾在一些短片中開始童星的生涯。他在電影《''{{link-en|King Jack|King Jack}}''》中飾演主角Jack。

2017年，他在[[列尼·史葛|列尼·史葛]]的驚悚電影《[[萬惡金錢|萬惡金錢]]》中飾演的{{link-en|約翰·保羅·蓋提三世|John Paul Getty III}}，以及在[[安德魯·海格|安德魯·海格]]的電影《[[賽馬皮特|賽馬皮特]]》中飾演主角。

== 個人生活 ==
普拉瑪出生於[[紐約州|紐約州]]的[[波啟浦夕市_(紐約州)|波啟浦夕市]]，是女演員Maia Guest與編劇/監製John Christian Plummer的兒子，他們在1996年5月結婚。<ref>{{Cite web |url = https://www.nytimes.com/1996/05/05/style/wedding-maia-guest-and-john-plummer.html |title = WEDDING;Maia Guest and John Plummer |date = 1996-05-05 |publisher = The New York Times |language = en}}</ref>

== 生涯 ==
小孩時，普拉瑪在一些短片如《''Frank''》（2010年）《''Three Things''》（2011年）和《''Alan Smithee''》（2012年）中演出。2011年，普拉瑪在[[HBO|HBO]]的電視劇《[[酒私風雲|酒私風雲]]》中演出8集，他在[[酒私風雲_(第二季)|第2季]]、[[酒私風雲_(第三季)|第3季]]和[[酒私風雲_(第四季)|第4季]]中飾演Eli Thompson的兒子Michael。<ref>{{Cite web |url = http://www.shockya.com/news/2015/05/25/tribeca-2015-interview-charlie-plummer-cory-nichols-and-christian-madsen-talk-king-jack-exclusive/ |title = Tribeca 2015 Interview: Charlie Plummer, Cory Nichols and Christian Madsen Talk King Jack (Exclusive) |author = Karen Benardello |date = 2015-05-25 |publisher = Shockya |language = en}}</ref>2012年，普拉瑪首次在長篇電影《[[搖滾不滅|搖滾不滅]]》中飾演配角。這部電影由[[大衛·雀斯|大衛·雀斯]]執導，並於2012年12月21日由[[派拉蒙優勢影業|派拉蒙優勢影業]]負責上映。<ref>{{Cite web |url = http://www.boxofficemojo.com/movies/?id=davidchase2012.htm |title = Not Fade Away |publisher = boxofficemojo |language = en}}</ref>2013年，普拉瑪加入了{{link-en|BYU TV|BYU TV|BYUtv}}的電視劇《''{{link-en|Granite Flats|Granite Flats}}''》，他在其中飾演主角Timmy Sanders。<ref>{{Cite web |url = http://highlandscurrent.com/2013/04/15/granite-flats-is-a-family-affair-for-guest-and-plummer-times-2/ |title = Granite Flats Is a Family Affair for Guest and Plummer ‘Times 2’ |author = Alison Rooney |date = 2013-04-15 |publisher = Highlands Current |language = en}}</ref>他出演了全三季的24集，電視劇在[[Netflix|Netflix]]上首播，並於2015年6月25日正式完結。

在2015年，普拉瑪在電影《''{{link-en|King Jack|King Jack}}''》中扮演主角Jack，與Cory Nichols和{{link-en|克里斯蒂安·馬德森|Christian Madsen}}一同演出，該片由Felix Thompson執導。<ref>{{Cite web |url = http://www.shockya.com/news/2015/05/25/tribeca-2015-interview-charlie-plummer-cory-nichols-and-christian-madsen-talk-king-jack-exclusive/ |title = Tribeca 2015 Interview: Charlie Plummer, Cory Nichols and Christian Madsen Talk King Jack (Exclusive) |author = Karen Benardello |date = 2015-05-25 |publisher = shockya.com |language = en}}</ref>這部電影於2015年4月17日在[[翠貝卡電影節|翠貝卡電影節]]上首映。他是重啟電影《[[蜘蛛俠：強勢回歸|蜘蛛俠：強勢回歸]]》中進行過主角[[蜘蛛俠|蜘蛛俠]]篩選測試的頭六名演員之一<ref>{{Cite web |url = https://www.hollywoodreporter.com/heat-vision/spider-man-hopefuls-screen-test-798995 |title = Spider-Man Hopefuls Screen-Test in Atlanta as Marvel Decision Nears (Exclusive) |author = Borys Kit |date = 2015-05-30 |publisher = The Hollywood Reporter |language = en}}</ref>，後來索尼和漫威選擇了[[湯姆·荷蘭|湯姆·荷蘭]]擔任這個角色。<ref>{{Cite web |url = https://news.marvel.com/movies/24758/sony_pictures_and_marvel_studios_find_their_spider-man_star_and_director/ |title = Tom Holland Cast as the Web-Slinger, Jon Watts to Direct the Next Spider-Man Adventure |date = 2015-06-23 |publisher = marvel.com |language = en |access-date = 2018-04-13 |archive-url = https://web.archive.org/web/20170816004635/https://news.marvel.com/movies/24758/sony_pictures_and_marvel_studios_find_their_spider-man_star_and_director/ |archive-date = 2017-08-16 |dead-url = yes }}</ref>

普拉瑪與[[李察·基爾|李察·基爾]]、[[史提夫·庫根|史提夫·庫根]]和[[羅娜·蓮妮|羅娜·蓮妮]]共同演出驚悚電影《''{{link-en|The Dinner (2017 film)|The Dinner (2017 film)|The Dinner}}''》（2017年），並由{{link-en|歐林·穆佛曼|Oren Moverman}}執導電影。在2017年，他在[[列尼·史葛|列尼·史葛]]的驚悚電影《[[萬惡金錢|萬惡金錢]]》中飾演被綁架的繼承人{{link-en|約翰·保羅·蓋提三世|John Paul Getty III}}，與[[米雪·威廉絲|米雪·威廉絲]]、[[基斯杜化·龐馬|基斯杜化·龐馬]]和[[麥克·華堡|麥克·華堡]]一起演出。

2015年12月，普拉瑪與演員[[瑪麗莎·托梅|瑪麗莎·托梅]]和[[提摩西·奧利芬|提摩西·奧利芬]]一起在獨立電影《''{{link-en|Behold My Heart|Behold My Heart}}''》中演出。<ref>{{Cite web |url = https://variety.com/2015/film/news/marisa-tomei-timothy-olyphant-charlie-plummer-behold-my-heart-1201656579/ |title = Marisa Tomei, Timothy Olyphant Starring in ‘Behold My Heart’ |author = Dave McNary |date = 2015-12-08 |publisher = Variety |language = en}}</ref>
