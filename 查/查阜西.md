'''查阜西'''（1895-1976），名'''鎮湖'''，又名'''夷平'''，字'''阜西'''，[[江西|江西]][[修水|修水]]人，是中國著名[[古琴|古琴]]演奏家、理論家，是最重要的[[古琴琴譜|古琴琴譜]]的蒐集者之一。他因為善彈《[[瀟湘水雲|瀟湘水雲]]》而又被稱為“'''查瀟湘'''”，與[[張子謙|張子謙]]、[[彭祉卿|彭祉卿]]並稱“浦東三傑”。<ref>{{Cite web|url=http://www.cglgq.com/wz/39.htm|title=张子谦先生在近现代琴史上的贡献|accessdate=2014-12-31|author=[[成公亮|成公亮]]|date=2008|work=|publisher=}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 在航空領域也有建樹，是[[兩航起義|兩航起義]]的主要組織策劃者之一。此外，他還是第一至四屆[[全國人大代表|全國人大代表]]。

==生平==
査阜西在[[光緒|光緒]]二十一年（1895年）出生於[[江西|江西]][[修水|修水]]山口镇来苏村，少年時開始學習古琴。1919年左右前往[[煙臺海軍學校|煙臺海軍學校]]、航空學校學習。1925年加入[[中國共產黨|中國共產黨]]<ref name="兩航起義"/>，1930年代成立[[今虞琴社|今虞琴社]]，並出版《[[今虞琴刊|今虞琴刊]]》，聯繫全國各地的琴人。1945年受邀前往美國[[加州大學|加州大學]]、[[美国国家地理协会|美国国家地理协会]]、[[华盛顿音乐俱乐部|华盛顿音乐俱乐部]]、[[美国国会图书馆|美国国会图书馆]]進行古琴學術演講，並演奏古琴。其論述在《[[美国音乐学协会公报|美国音乐学协会公报]]》上出版，同時也發行了一系列古琴音樂。他將藏於國會圖書館中的《[[神奇秘谱|神奇秘谱]]》、《[[太音大全|太音大全]]》複製下來帶回了中國。<ref name="中國江西網"/>

直到1950年代，查阜西在航空領域還很活躍，曾任[[中央航空運輸公司|中央航空運輸公司]]副总经理<ref name="兩航起義">{{Cite web |url=http://mil.news.sina.com.cn/s/2009-11-09/0950573100.html|title=纪念两航起义60周年：两航起义始末|accessdate=2014-06-27 |work= |publisher=中國民航報|date=2009-11-09}}</ref>。1949年受[[周恩來|周恩來]]委任，查阜西到[[香港|香港]]組織了[[兩航起義|兩航起義]]。<ref name="中國江西網">{{Cite web |url= http://www.jxnews.com.cn/people/system/2014/05/26/013123492.shtml|title=江西查阜西为我国近代琴坛领袖 开创我国古琴音乐对外交流先例 |accessdate=2014-06-27 |work= |publisher=中國江西網|date=2014-05-29}}</ref>

1954年，查阜西成立[[北京古琴研究會|北京古琴研究會]]，出版《[[古琴研究|古琴研究]]》、《[[琴論綴新|琴論綴新]]》。1958年作爲[[中國歌劇團|中國歌劇團]]古典音樂顧問前往日本演出，同年出版《[[存見古琴曲譜揖覽|存見古琴曲譜揖覽]]》<ref name="學報">{{cite journal|author = [[陳長林|陳長林]]|title=查阜西先生对现代琴学的贡献|journal=《中央音樂學院學報》 |issue=1996年02期  |url=http://www.cnki.com.cn/Article/CJFDTotal-ZYYE602.005.htm|accessdate=2014-06-27 }}</ref>
。後來亦曾前往蘇聯的[[五聲音階|五聲音階]]國際會議。<ref>{{Cite web |url= http://info.guqu.net/guqindashi/20060728214317_528.html|title=古琴艺术大家查阜西|accessdate=2014-06-27 |work= |publisher=中國古曲網|date=2006-07-28}}</ref>

1949年之後他曾擔任[[中國音樂家協會|中國音樂家協會]]副主席，在1950年代間，查阜西前往全國各地訪問各位琴人，錄取了來自86位琴家的262首琴曲，他亦曾推薦各地琴家們去各大院校任教，自己也受聘爲[[中央音樂學院|中央音樂學院]]民樂系主任<ref name="學報"/>。1961年開始編纂巨著《[[琴曲集成|琴曲集成]]》，後來因[[文化大革命|文化大革命]]中斷，此書到2010年才全部出齊<ref name="中國江西網"/><ref name="學報"/>。
