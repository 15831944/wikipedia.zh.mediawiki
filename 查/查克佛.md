'''查克佛'''（{{lang-en|Mark Drakeford}}，{{bd|1954年|9月19日|||Drakeford, Mark}}）是一位[[威爾斯|威爾斯]]的政治人物，黨籍為{{le|威爾斯工黨|Welsh Labour|工黨}}，現任[[威爾士首席部長|威爾士首席部長]]。

2018年秋，時任首席部長[[莊家偉|莊家偉]]宣布辭任工黨黨魁以及首席部長之職務，查克佛為第一名宣布競逐的參選人。2018年12月6日，查克佛於{{le|2018年威爾斯工黨黨魁選舉|2018 Welsh Labour Party leadership election|黨魁選舉}}次輪以取得53.9%選票的姿態獲勝，成為新任黨魁。<ref>{{cite news|url=https://www.bbc.com/news/uk-wales-politics-46471728|work=BBC|title=Welsh Labour leadership: Mark Drakeford set to be Wales’ first minister|language=en}}</ref>12日，獲[[威爾斯國民議會|國民議會]]議員提名首席部長一職，翌日上任。<ref>{{cite news|url=https://www.bbc.com/news/uk-wales-politics-46537794|title=Mark Drakeford confirmed as new Welsh first minister|work=BBC|language=en}}</ref>

成為首席部長前，曾任財政部長、-{zh-cn:退出欧洲联盟; zh-hk:脫離歐盟; zh-tw:脫離歐洲聯盟;}-事務部長以及衛生部長。<ref>{{cite news|url=https://news.mingpao.com/ins/%E5%9C%8B%E9%9A%9B/article/20151201/s00005/1448979655466/%E8%8B%B1%E5%9C%8B%E5%A8%81%E7%88%BE%E6%96%AF%E7%A0%B4%E5%A4%A9%E8%8D%92%E4%BF%AE%E4%BE%8B-%E6%89%80%E6%9C%89%E6%88%90%E4%BA%BA%E8%87%AA%E5%8B%95%E6%88%90%E5%99%A8%E5%AE%98%E6%8D%90%E8%B4%88%E8%80%85|title=英國威爾斯破天荒修例　所有成人自動成器官捐贈者 (23:05)|work=明報新聞網}}</ref>
