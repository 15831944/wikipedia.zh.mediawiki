'''查爾斯街'''，其大多數路段被編入馬里蘭州139號州道，從[[巴爾的摩|巴爾的摩市]]至巴爾的摩郡的[[陶森|陶森]]，其北端在州際公路695號附近接於貝隆那大街（ Bellona Avenue），南端則是巴爾的摩市的聯邦山（Federal Hill）。查爾斯街是巴爾的摩市的幹道之一，於市區有單行道路段；單行道路段的對向交通由聖保羅街（St. Paul Street）、馬里蘭大街（Maryland Avenue）、大教堂街（Cathedral Street）、自由街（Liberty Street）等道路負責。

雖然並不算是巴爾的摩市真正的東西分界點，查爾斯街仍被認為是巴爾的摩市的東西分界線。任何與查爾斯街交叉而過的街道，其兩側門牌號碼都由本街街口處起算，街道也由此區分東段與西段。<ref>http://books.google.com/books?id=iIEr6DrWY3MC&pg=PA2&dq=%22charles+street%22baltimore&lr=&as_brr=3&ie=ISO-8859-1&output=html&cd=20</ref> 整條查爾斯街都是馬里蘭景觀道路之一，稱為「查爾斯街景觀道路（Charles Street Scenic Byway)」，也是國家景觀道路之一，稱為「巴爾的摩查爾斯史蹟街（Baltimore's Historic Charles Street）」。<ref name="Maryland Byways">{{cite web|title=Maryland Byways|publisher=Maryland State Highway Administration|url=http://www.marylandroads.com/oed/MarylandScenicByways.pdf|format=PDF|accessdate=2011-01-24}}</ref><ref name=nsbp>{{cite web|url=http://byways.org/explore/byways/2271|title=Baltimore's Historic Charles Street|publisher=[[National_Scenic_Byways_Program|National Scenic Byways Program]]|accessdate=2012-03-28}}</ref>


==路線==
馬里蘭州道139號由巴爾的摩市的美國國道1號/40號卡車專用道（常用的名字是北大街（North Avenue））起算，北向車道沿查爾斯街北行，南向車道則為聖保羅街。


==歷史==
1730年查爾斯街初建時稱為福瑞斯特街（Forest 或 Forrest Street）。<ref>http://books.google.com/books?id=6tF4AAAAMAAJ&pg=PA53&vq=%22charles+street%22&dq=%22charles+street%22baltimore&lr=&as_brr=3&ie=ISO-8859-1&output=html</ref><ref name="charlesstreet.net">{{cite web |url=http://www.charlesstreet.net/history.cfm |title=存档副本 |accessdate=2010-06-27 |0=2010-08-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20100820073831/http://www.charlesstreet.net/history.cfm |archivedate=2010-08-20 }}</ref> 南端位於今日的隆巴德街（ Lombard Street ）。<ref>http://books.google.com/books?id=6tF4AAAAMAAJ&pg=PA48&vq=%22charles+street%22&dq=%22charles+street%22baltimore&lr=&as_brr=3&ie=ISO-8859-1&output=html&source=gbs_search_r&cad=1</ref> 至少在1761年時本路就稱為查爾斯街了。 <ref name="charlesstreet.net"/> 1858年，瓊斯瀑布溪（Jones Falls）氾濫，查爾斯街在今日的賓州車站附近的橋樑被捲走。<ref>http://books.google.com/books?id=6tF4AAAAMAAJ&pg=PA213&vq=%22charles+street%22&dq=%22charles+street%22baltimore&lr=&as_brr=3&ie=ISO-8859-1&output=html&source=gbs_search_r&cad=1</ref><ref>http://books.google.com/books?id=6tF4AAAAMAAJ&pg=PA209&vq=%22charles+street%22&dq=%22charles+street%22baltimore&lr=&as_brr=3&ie=ISO-8859-1&output=html&source=gbs_search_r&cad=1</ref>
位於巴爾的摩郡，稱為查爾斯街大街（Charles Street Avenue）最先鋪上路面的路段，是1915年在[[陶森|陶森]]市中心的西邊，從切薩皮克大街（Chesapeake Avenue）向北到加帕路（Joppa Road）的路段。<ref name="1915 src report"/><ref name="1921 map"/> 本路於巴爾的摩郡的其他路段在1928年鋪上路面並在1933年編為馬里蘭州州道139號。<ref name="1928 map"/> 1934年交通量的增加使得馬里蘭州道路委員會（Maryland State Roads Commission）建議將本路的巴爾的摩郡路段拓寬，由{{convert|14|to|20|ft|m}} 至 {{convert|30|ft|m|}}。<ref name="1934 src report"/> 1950年在巴爾的摩市，於北大街至第29街之間，州道139號分為兩條單行道，北向為查爾斯街，南向為馬里蘭大街<ref name="1950 map"/> 1954開始計畫將馬里蘭州州道139號從巴爾的摩市向北拓寬至州道134號，<ref name="1954 src report"/> 並開始在本公路將來與695號州際公路交會處進行先期工程；工程於1955年正式動工。<ref name="1954 src report"/><ref name="1956 src report"/> 1956年開始，馬里蘭州道139號從州道134號路口處開始至陶森城大道（Towsontown Boulevard）止，拓寬為有分隔島的四線道，由陶森城大道路口至州際公路環狀線路段則改線新築成有分隔島的四線道。<ref name="1958 src report"/> 工程由州際公路交流道到加帕路段，以及由州道134號到陶森城大道路段，在1957年完工。<ref name="1957 map"/> 改線計畫（包括加帕路跨越州道139的橋樑）則在1958年完工。<ref name="1958 src report"/><ref name="1958 map"/><ref name="NBI Joppa Road"/> 1959年新線劃為馬里蘭州道139號，而查爾斯街大街舊的路段在1961年移交給郡政府負責。<ref name="1959 map"/><ref name="1961 map"/> 1963年州道139號的南向車道改指定為聖保羅街，其範圍乃由北大街至聖保羅街的北端－－亦即與查爾斯街交會處；不過聖保羅街本身由第33街以北就是一條雙向有分隔島的大道。<ref name="1963 map"/> 1999年，州道139號北端與貝隆那大街交會處改建為圓環。<ref name="Maryland HLR 1999"/><ref name="Roundabouts"/> 後來為了重建州道139號與州際公路695號的交流道，圓環被移除，改建成一般的路口，工程於2008年開始，2012年完成。<ref name="MDSHA Charles Street Interchange"/>
