'''查克·費尼'''（英語：Chuck Feeney，{{bd|1931年|4月| |}}），1931年出生在[[美國|美國]][[新澤西州|新澤西州]]，建立了環球免稅集團（DFS）。查克·費尼一直默默行善，捐出數十億的美金，被稱為“慈善界的[[詹姆士·龐德|詹姆士·龐德]]”。<ref name = ' Chuck ' >{{Cite web |url = https://www.christiantimes.cn/news/13412/%E4%BA%BF%E4%B8%87%E5%AF%8C%E7%BF%81%E5%9F%BA%E7%9D%A3%E5%BE%92%E6%85%88%E5%96%84%E5%AE%B6%E6%9F%A5%E5%85%8B.%E8%B4%B9%E5%B0%BC%E4%BD%99%E7%94%9F%E6%A2%A6%E6%83%B3%EF%BC%9A%E6%97%A0%E7%89%B5%E6%97%A0%E6%8C%82%E5%9C%B0%E5%8E%BB%E8%A7%81%E4%B8%8A%E5%B8%9D | title =亿万富翁基督徒慈善家查克.费尼余生梦想：无牵无挂地去见上帝 | publisher = 基督時報 |language =zh |accessdate = 2019-03-24 }}</ref>

==生平==
查克·費尼出生在美國新澤西州伊麗莎白市，袓父是[[愛爾蘭|愛爾蘭]]的移民，父親在保險公司當任業務員，母親是一位護士，他的父母個性都是樂於幫助別人，他受到[[基督教|基督教]]信仰的家庭教育影響，過著節儉的生活。查克·費尼在1948年加入美國空軍，在[[日本|日本]]和[[韓國|韓國]]服役4年。退役之後進入[[康奈爾大學|康奈爾大學]]就讀酒店管理，1956年到[[法國|法國]]的[[格勒諾布爾大學|格勒諾布爾大學]]就讀政治學。
查克•費尼於畢業後先在美國艦隊駐紮的濱海自由城辦夏令營，服務隨軍的子女。之久利用老兵身份開始推銷免稅商品。在這個期間遇到了康奈爾大學的校友羅伯特·米勒，開始擴大合作。於1960年兩人成立了環球免稅集團（DFS），到1964年時，店面已經涵蓋了27個國家，DFS發展快速，目前已是世界最大的連鎖免稅店集團。<ref name = ' Chuck '/>

==慈善事業==
1981年查克·費尼捐給母校康奈爾大學70萬美元，之後成立了大西洋慈善基金。他要求受捐贈的機構不能為他放置感謝牌，且必須簽署保密協議，不能透露他的捐款。在1984年，查克·費尼將自己持有的38.75%的DFS股份，轉到了大西洋慈善基金。目前他仍每年捐出大約4億美元，大西洋慈善基金會在美國、[[澳大利亞|澳大利亞]]、[[越南|越南]]、[[百慕達|百慕達]]、[[南非|南非]]和愛爾蘭的教育、科學、醫療、老齡化和公民權利方面，共捐贈了62億美元，並且該基金預計將在2020年完成他所有的捐款之後結束。[[比爾·蓋茲|比爾·蓋茲]]和[[華倫·巴菲特|華倫·巴菲特]]的捐款行為，也有受到查克·費尼的影響。<ref name = ' Chuck '/><ref >{{cite news| title=Chuck Feeney: The Billionaire Who Is Trying To Go Broke| url = https://www.forbes.com/sites/stevenbertoni/2012/09/18/chuck-feeney-the-billionaire-who-is-trying-to-go-broke/#32265e5e291c | accessdate=2019-03-24 | agency=Forbes | date=2012-09-18 }}</ref>
