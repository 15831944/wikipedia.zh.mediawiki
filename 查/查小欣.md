'''查小欣'''（{{bd|1958年|8月4日}}，原名'''黎小明'''），是[[香港|香港]]娛樂新聞工作者暨[[節目主持|節目主持]]。於1980年畢業於[[香港浸會書院|香港浸會書院]]，及後30多年一直從事文化娛樂報道的工作，與不少香港和兩岸藝人頗為熟稔。

== 簡介 ==
查小欣過去是《[[明報|明報]]》的娛樂新聞記者，後來成為編輯。之後她曾於多家八卦雜誌社工作過，並曾創立《[[頭條周刊|頭條周刊]]》。但因為《頭條周刊》在創刊號中刊登了應屆[[亞洲小姐|亞洲小姐]]曾小燕沒有穿內褲的照片而惹上官非。雜誌社於翌年倒閉，後來她加入[[商業電台|商業電台]]主持節目《茶煲裡的查篤撐》。[[科網股|科網股]]熱潮時，亦曾開辦網上娛樂新聞入門網站[[show8.com|show8.com]]，但在科網股爆破後亦倒閉，亦曾投得[[亞洲電視|亞洲電視]]娛樂新聞節目的製作權，製作《[[八串騷|八串騷]]》節目。往後十多年於商台主持《[[巴巴閉邊個夠我查篤撐|巴巴閉邊個夠我查篤撐]]》，直到2012年因[[陳志雲|陳志雲]]擔任商台行政總裁，其原時段節目被[[吳綺莉|吳綺莉]]的新節目取代<ref>[http://hk.ent.yahoo.com/news/%e5%90%b3%e7%b6%ba%e8%8e%89%e5%8f%96%e4%bb%a3%e6%9f%a5%e5%b0%8f%e6%ac%a3-%e9%99%b3%e5%bf%97%e9%9b%b2%e8%b8%a2%e8%b5%b0-%e6%9f%a5%e7%af%a4%e6%92%90-212237049.html 吳綺莉取代查小欣 陳志雲踢走《查篤撐》] {{webarchive|url=https://web.archive.org/web/20120815005859/http://hk.ent.yahoo.com/news/%E5%90%B3%E7%B6%BA%E8%8E%89%E5%8F%96%E4%BB%A3%E6%9F%A5%E5%B0%8F%E6%AC%A3-%E9%99%B3%E5%BF%97%E9%9B%B2%E8%B8%A2%E8%B5%B0-%E6%9F%A5%E7%AF%A4%E6%92%90-212237049.html |date=2012-08-15 }}《明報》2012年8月12日</ref><ref>[http://www.hkheadline.com/ent/ent_content.asp?track=news&contid=197425&srctype=n 空降商台「換血」不止 陳志雲攬盧覓雪踢走查小欣]《頭條日報》2012年7月5日</ref>。她的前夫是以「夏天為大地帶來雨水」香港政府[[渠務署|渠務署]]廣告旁白聞名的前電台主持人[[李學斌_(電台主持)|李學斌]]，兩人共育有一子。

查小欣和香港著名媒体人[[卢觅雪|卢觅雪]]，一并被称为香港的“[[狗仔队之母|狗仔队之母]]”，她们二人都曾将[[狗仔队|狗仔队]]文化带入[[香港|香港]]和[[两岸三地|两岸三地]]的华人社会。<ref>[http://news.xinhuanet.com/ent/2006-12/11/content_5466573.htm 世上没有秘密 查小欣：我就是香港狗仔第一人]《新闻晨报》2006年12月11日</ref><ref>[http://ent.cn.yahoo.com/ypen/20110610/409049.html 解密查小欣：香港狗仔队“鼻祖”的私生活] {{webarchive|url=https://web.archive.org/web/20110614014514/http://ent.cn.yahoo.com/ypen/20110610/409049.html |date=2011-06-14 }}《雅虎娱乐》2011年6月10日</ref>

== 報導風格 ==
查小欣深挖娛樂圈藝人隱私的報導風格，引起廣泛爭議。不少人認為這是仗著新聞自由，踐踏隱私。不過查小欣則認為，這是一名專業的新聞從業員應該做的事，只不過她所涉獵的範疇是娛樂圈人物，而非政商界人物而已。查小欣更指，演藝人員、公眾人物、政商名流的平台是公眾給予，必須被公眾監督，沒有隱私概念。並稱，其與及一眾娛樂新聞記者均公私分明，私下與藝人交往，則從不將藝人當做是非籌碼，但報導卻要顧及道德原則和社會責任。<ref>[http://ent.southcn.com/8/2009-09/01/content_5673237.htm 查小欣：科技进步的今天，已难再有隐私] {{Wayback|url=http://ent.southcn.com/8/2009-09/01/content_5673237.htm |date=20150402095133 }}《南方网》2009-09-01 </ref><ref>[http://ent.sina.com.cn/r/m/2012-09-27/11243754018.shtml 查小欣：当娱记立场坚定不怕歧视]《南方都市报》2012年09月27日</ref>

== 爭議性的言論 ==
*2002年10月18日，查小欣曾在上午電台節目內誤報七、八十年代香港著名歌手[[羅文|羅文]](籮記)死訊而備受批評(其後於當晚逝世)，後來經查小欣道歉後，事件才告平息。事源可能是因為當時電話訪問羅文所屬唱片公司[[Music_Plus|Music Plus]]主席[[楊受成|楊受成]]，查小欣誤會了楊受成的說話所致。

*另外，在2008年初的[[陳冠希裸照事件|陳冠希裸照事件]]上，查小欣曾發表文章，認為淫照不可能因[[陳冠希|陳冠希]]疏忽，託維修公司修理而流出（又稱「明眼人事件」）。豈料事件正是陳氏疏忽所致，令查小欣遭受網民恥笑。

=== 節目被腰斩 ===
2012年，查小欣獲電視高層[[余詠珊|余詠珊]]小姐邀請，加入香港電視圈，在[[香港無線電視|香港無線電視]]開設娛樂新聞評論節目《[[爆足一周|爆足一周]]》。節目移植電臺廣播模式，吹彈淺唱，公開談及明星極為隱私的家庭婚姻等議題，被網民指責沉悶無聊，在無根無據的前提下，肆意揶揄践踏別人隱私，並且以各式陰謀論的观点为娱乐圈和社会大众带来負能量。最终节目开播四集后被腰斩，成为最短命的娱乐新闻报道节目之一。<ref>[http://ent.ifeng.com/tv/news/zongyi/detail_2012_10/29/18629954_0.shtml 查小欣《爆足一周》暂停播 怕被告不敢随便爆料]《凤凰娱乐》2012年10月29日</ref><ref>[http://et.21cn.com/tv/tvb/zixun/2012/10/22/13342488.shtml 查小欣《爆足一周》收视创新低 观众要求腰斩] {{Wayback|url=http://et.21cn.com/tv/tvb/zixun/2012/10/22/13342488.shtml |date=20150512151824 }}《21CN娱乐》2012-10-22</ref>
