'''查尔斯·鲍威尔'''（{{lang|en|Charles Powell, Baron Powell of Bayswater}}，{{bd|1941年|7月6日||}}）[[英国|英国]]人，英国外交官、政治家、商人。

==生平==
鲍威尔先后在[[坎特伯雷|坎特伯雷]]{{link-en|坎特伯雷圣埃德蒙学校|St Edmund's School Canterbury|座堂唱诗班学校}}、{{link-en|国王学校 (坎特伯雷)|The King's School, Canterbury|国王学校}}、[[牛津大学新学院|牛津大学新学院]]接受教育，1963年毕业于牛津大学新学院并获现代历史学士学位。

1963年，加入[[英国外交部|英国外交部]]。首个职位是1965年担任英国驻[[芬兰|芬兰]]大使馆三等秘书，后晋升为该大使馆二等秘书。1968年回到[[伦敦|伦敦]]，在[[英国外交及联邦事务部|英国外交及联邦事务部]]度过三年。1971年，任英国驻[[美国|美国]]大使馆一等秘书，并成为大使的私人秘书。1974年调到[[波恩|波恩]]在英国驻[[联邦德国|联邦德国]]大使馆工作，1980年调到[[布鲁塞尔|布鲁塞尔]]任{{link-en|英国驻欧盟常任代表列表|List of Permanent Representatives of the United Kingdom to the European Union|英国驻欧盟常任代表}}的顾问。后来，他被借调到[[唐宁街十号|唐宁街十号]]，担任[[英国首相|英国首相]][[撒切尔夫人|撒切尔夫人]]的私人秘书（1983年至1990年），随后担任英国首相[[约翰·梅杰|约翰·梅杰]]的私人秘书（1990至1991年）。 在为撒切尔夫人工作期间，他成为她最值得信赖的外交政策助手之一，并帮助安排了富有争议的同[[沙特阿拉伯|沙特阿拉伯]]之间的[[亚玛玛|亚玛玛]]军火交易<ref name="bae files">{{cite news |title=BAE Files: Charles Powell|url=https://www.theguardian.com/world/2007/jun/07/bae20|publisher=BBC|date=2007-06-07|accessdate=2008-03-18 | location=London | first=David | last=Leigh}}</ref>。他还参与了中英两国有关[[香港回归|香港回归]]问题的谈判事务，曾陪同撒切尔夫人在1982年访华就香港问题同[[邓小平|邓小平]]会谈、1984年再度访华签署关于香港问题的《[[中英联合声明|中英联合声明]]》<ref name=wy>{{Cite web |url =  http://money.163.com/13/1114/08/9DKJBAUN00253B0H.html#from=keyscan|title = 查尔斯·鲍威尔:中国学生缺乏社会经历 |author =  |date =2013-11-14  |publisher =网易  |language =  |accessdate =  |quote =  }}</ref><ref name=mishu/>。

自1992年以来，他一直是国际商人。他利用在政府任职期间取得的经验，接手了几个董事和顾问职位<ref name="bae files"/>。他是天箭座资产管理有限公司（Sagitta Asset Management Limited）和[[路易威登|路易威登]]英国有限公司的董事长。他还曾担任{{link-en|菲利普斯 (拍卖商)|Phillips (auctioneers)|菲利普斯美术拍卖公司}}董事长兼[[怡和洋行|怡和控股有限公司]]高级总监。他还担任[[卡特彼勒|卡特彼勒]]公司、[[路易威登|路易威登]]集团、[[文华东方酒店集团|文华东方国际有限公司]]、[[迅达集团|迅达控股有限公司]]、{{link-en|蒂夫克|Tiphook|蒂夫克公司}}、[[德事隆|德事隆公司]]和{{link-en|希布 (企业)|Hibu|耶尔集团公司}}、[[地中海航运公司|地中海航运公司]]等公司的董事。他也是{{link-en|亨利·杰克逊协会|Henry Jackson Society|亨利·杰克逊协会}}的签约者。在军火行业，他担任过以下职位：[[罗尔斯·罗伊斯股份有限公司|罗尔斯·罗伊斯股份有限公司]]、{{link-en|塔莱斯集团|Thales Group|塔莱斯集团}}、[[英国宇航系统|英国宇航系统]]和[[德事隆|德事隆]]的顾问<ref>{{cite web|url=https://www.caat.org.uk/resources/influence/person/104/meetings |title=Resources - Influence - Person - 104 - Charles Powell |publisher=CAAT |date=2015-09-14 |accessdate=2017-02-18}}</ref><ref name=wy/>。

2000年，他被封为终身贵族（勋爵）<ref name=mishu>{{Cite web |url = http://news.sohu.com/20141027/n405510927.shtml |title =  撒切尔夫人前秘书：香港自治程度远超英方预期|author =  |date =2014-10-27  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。2010年代，他是[[英国上议院|英国上议院]]中立议员、英国宪法特别委员会成员，[[牛津大学赛德商学院|牛津大学赛德商学院]]托管理事会主席。他还兼任[[复旦大学管理学院|复旦大学管理学院]]首届国际顾问委员会委员等职务<ref name=wy/>。2014年10月，英国上议院议员查尔斯·鲍威尔在接受[[英国广播公司|英国广播公司]]采访时表示，香港的自治程度远超当年[[英属香港|英方治理香港时期]]以及当年英方就香港问题同中国谈判时的预期，香港参加[[占中行动|占中行动]]的抗议者“不切实际”<ref name=mishu/><ref>{{Cite web |url = http://world.people.com.cn/n/2014/1008/c157278-25787372.html |title =英国上院议员查尔斯·鲍威尔:“占中”行为无视现实  |author =  |date = 2014-10-08 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
