'''查理十世'''（[[法语|法语]]：Charles X；{{bd|1757年|10月9日|1836年|11月6日|catIdx=C}}），本名'''查理-菲利普'''（[[法语|法语]]：Charles-Philippe），是[[法国|法国]][[波旁復辟|波旁復辟]]后的第二个[[法國君主列表|法兰西及纳瓦尔国王]]（1824年8月16日–1830年8月2日在位）。他是[[路易十五|路易十五]]之孙、[[路易十六|路易十六]]及[[路易十八|十八]]之弟，歷經其兄路易十八的[[波旁復辟|十年復辟]]之後，67歲（1824年）時才继承王位。由于他对[[天主教|天主教]]的强烈热情和贵族政治的厌恶，引起人民的强烈不满，引發1830年[[七月革命|七月革命]]，查理被迫[[逊位|逊位]]，流亡英國。

== 早年 ==
1757年10月9日，查理-菲利普出生於[[凡尔赛宫|凡尔赛宫]]，是[[路易，法国王太子_(1729年–1765年)|法兰西王太子路易]]和[[萨克森的玛丽·约瑟芬_(1731年–1767年)|萨克森的玛丽·约瑟芬]]所生倒数第二个儿子，也是[[路易十五|路易十五]]的孙儿，出生後，路易十五授予孙儿[[阿图瓦伯爵|阿图瓦伯爵]]的封号，此封号也是查理-菲利普大半生最为人用的称呼。因为他唯一的弟弟夭折，查理-菲利普便成为家中幼子。

查理-菲利普早年便失去双亲，因此他幼年被祖父[[路易十五|路易十五]]教养。祖父死后，他的哥哥[[路易十六|路易十六]]继承王位。

== 青年 ==
1773年11月，16岁的查理-菲利普同17岁的[[萨克森的玛丽·特丽莎公主|萨克森的玛丽·特丽莎公主]]结婚。尽管查理-菲利普的妻子被认为相貌丑陋，但婚后生活很顺利，婚后两年便生下了儿子[[路易十九|路易-安托万]]。

由于查理-菲利普当时两个还在人世的哥哥都没有孩子，而且查理-菲利普酷似祖父，使得查理-菲利普及其后裔有继承王位的可能，而被时人视为是法国王室的最引人注目的人物。直到路易十六和[[玛丽·安托瓦内特|玛丽·安托瓦内特]]的长子[[路易-約瑟夫_(法國王太子)|路易-约瑟夫]]，公众才减弱了对查理-菲利普的关注。

查理-菲利普是个极端守旧保守主义者，[[法国大革命|法国大革命]]期间，查理-菲利普被迫逃亡国外。1793年，路易十六被斩首，查理-菲利普便成了无冕之王[[路易十七|路易十七]]的叔叔。路易十七于1793年夭折，查理-菲利普的哥哥[[路易十八|普罗旺斯伯爵]]成为王位继承人。查理-菲利普在1795年至1814年间住在[[英国|英国]]。

== 成為法國波旁復辟國王 ==
[[路易十八|普罗旺斯伯爵]]日后复辟王朝，成为路易十八，查理-菲利普成为路易十八的主要顾问和出谋划策者，在他周围集中了一批最保守的旧贵族。由于路易十八膝下无子，查理-菲利普便成为哥哥的继承人。

1824年，路易十八逝世，查理-菲利普继承王位，是为查理十世，當時他已經67歲。

查理十世即位后，对[[法国大革命|法国大革命]]和[[拿破仑战争|拿破仑战争]]时所受的折磨难以释怀，因而对[[自由主义|自由主义]]极度怨恨，对新秩序加以抗拒。查理的次子伯利公爵[[阿图瓦的查理-费迪南德|阿图瓦的查理-费迪南德]]被激进份子所杀，使他对自由主义更恨之入骨。查理相信君权神授论，认为君主应拥有绝对权力。曾言宁伐木为生，亦不屑成为英式立宪君主，反映其对君主专政的一承不变。一般学者认为，固执、傲慢的性格正是其失败之根源。查理领导着一个极端保守的、主张恢复革命前阶级特权的政治集团，他的这种倾向在其继承王位后表现得更加明显。他在即位后推行一系列反民主政策，这使他十分不得人心。由于他对[[天主教|天主教]]的强烈热情和贵族政治的厌恶，引起人民的强烈不满。

1830年8月9日，法国发生[[七月革命|七月革命]]，查理十世的统治被推翻，查理被迫[[逊位|逊位]]，再度流亡英國。法国王位最终由他的堂弟奥尔良公爵[[路易-菲利普一世|路易-菲利普]]继承。

== 逝世 ==
晚年，查理由英國返回大陆，1836年11月6日逝世，享年79歲。

查理死后，他的儿子路易-安托万和孙子[[亨利五世_(法兰西)|亨利]]先后声称自己为法兰西王位的继承人。亨利死后，没有留下任何后代，宣告着路易十五的男系后人断绝。
