'''查尔斯·弗朗西斯·里克特'''（{{lang-en|'''Charles Francis Richter'''}}，{{bd|1900年|4月26日|1985年|9月30日|catIdx=Richter, Charles Franci}}），美国[[物理学家|物理学家]]、地震学家，生于[[美国|美国]][[俄亥俄州|俄亥俄州]]{{link-en|欧弗佩克|Overpeck, Ohio}}。曾任职于[[加州理工学院|加州理工学院]]与同事[[賓諾·古登堡|賓諾·古登堡]]共同创立了判定地震规模的[[里氏地震规模|里氏地震规模]]，并于1935年首次使用这一度量方式。

== 生平 ==
查尔斯·弗朗西斯·里克特于1900年4月26日出生于[[美国|美国]][[俄亥俄州|俄亥俄州]]{{link-en|欧弗佩克|Overpeck, Ohio}}的一个农村家庭<ref name=Ml1 />。里克特还拥有[[德国|德国]]血统，他的曾祖父由于在[[德意志1848年革命|德意志1848年革命]]中受到迫害从[[巴登-符腾堡州|巴登-符腾堡州]]逃亡至美国<ref name=intro1 />。在里克特年轻时就饱受了人间痛苦。他的父母在他幼时便离了婚，随后他被外祖父抚养长大，在1909年跟随着整个家庭移居至[[洛杉矶|洛杉矶]]，这也使得他在年轻时患有[[精神分裂症|精神分裂症]]<ref name=intro2 />。1920年，里克特从[[斯坦福大学|斯坦福大学]]本科毕业。1928年，他在[[加州理工学院|加州理工学院]]获得了理论物理学的[[博士|博士]]学位，但在此前一年，他就早已收到了来自{{link-en|卡内基科学研究所|Carnegie Institution for Science}}的聘函。因此，里克特开始接触[[地震学|地震学]]。

1985年9月30日，里克特因[[充血性心力衰竭|充血性心力衰竭]]于美国[[加利福尼亚州|加利福尼亚州]][[帕萨迪纳_(加利福尼亚州)|帕萨迪纳]]逝世，享年85岁<ref name =FA2 />。

== 科研生涯 ==
查尔斯·弗朗西斯·里克特于1927年收到了来自[[罗伯特·密立根|罗伯特·密立根]]的研究助理聘函，自此开始在{{link-en|卡内基科学研究所|Carnegie Institution for Science}}的帕萨迪那地震研究所工作，也正是在此结识了他的合作伙伴[[宾诺·古登堡|宾诺·古登堡]]<ref name =FA2 />。他在哈里·伍德教授的指导下开始学习地震学知识，如真相分析等，从事测量地震图和确定[[震中|震中]]的工作<ref name=intro3 />。1931年开始，帕萨迪那地震研究所开始了出版定期地震目录的准备工作，这促使里克特加快了解决如何确定地震大小问题的进程<ref name=intro3 />。为解决这一问题，1932年，里克特和古登堡共同发明了一种用于描述地震释放能量大小的标度，即[[里氏震级|里氏震级]]<ref name=Ml2 />。此后，古登堡的每项重要工作，几乎都得到了里克特的参与和合作，他们两人合著的论文多达160多篇<ref name=intro3 />。1937年，他回到了[[加州理工学院|加州理工学院]]担任助理教授一职。在[[第二次世界大战|二战]]后，他晋升为副教授，最终于1952年正式成为了教授，在那里度过了他的余生<ref name=intro2 /><ref name =FA2 />。

1959年，里克特作为交流学者前往[[东京大学|东京大学]]工作了一年，这是他唯一一次离开加州理工学院的工作经历，自此他再也没有享受过任何公休<ref name=intro2 />。1970年，里克特正式退休<ref name =FA1 />。

== 显著科研成果 ==
1932年，里克特与同事古登堡共同发明了一种用于描述地震释放能量大小的标度，即[[里氏震级|里氏震级]]<ref name=Ml2 />。地震的震级频度分布被认为是[[地震|地震]]的最重要性质之一，而里克特与古登堡确定的公式被认为是传统地震学中最遵从震级频度分布的关系式<ref name=gs />。在{{link-en|1971年圣费尔南多地震|1971 San Fernando earthquake}}发生后，重灾区因为引用了里克特发表的警告而防止了更多的伤亡出现<ref name =FA1 />。

== 个人生活 ==
查尔斯·弗朗西斯·里克特承认自己是一名活跃的[[天然主义|裸体主义者]]。他曾与他的妻子前往过多个{{link-en|裸体娱乐|Nude recreation|裸体社区}}。里克特同时也是一名[[无神论者|无神论者]]<ref name=Ml1 />。
