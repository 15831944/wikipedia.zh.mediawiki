本列表為'''[[查德|查德]]'''駐'''[[中華民國|中華民國]]'''歷任大使名錄。現已斷交。
==歷史==
*1997年8月12日，中華民國與查德恢復大使級外交關係。

*2006年8月5日，中華民國與查德斷交，查德與中華人民共和國復交。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=8742DCE7A2A28761&sms=491D0E5BF5F4BC36&s=2C11BF7FD929A3AD |author = 公眾外交協調會 |date = 2006年8月9日 |title = 外交部說明處理與查德斷交情形及政府立場記者會 |publisher = 中華民國外交部}}</ref>。
