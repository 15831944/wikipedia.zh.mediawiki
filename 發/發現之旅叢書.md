《'''發現之旅'''》（中譯版別名：{{lang|fr|Voyage D’exploration}}<ref>{{lang|fr|Voyage D’exploration}}：法文，意為「探索之旅」，也可解作「發現之旅」，標記在中譯本封面左上角（包括正體版和簡體版），但在法國原版中從來沒有使用過這個標題。</ref>）是法國[[加利瑪出版社|加利瑪出版社]]於1986年推出的[[圖鑑|圖鑑]]式[[百科全書|百科全書]][[文庫_(出版)|文庫集]]《{{lang|fr|'''Découvertes Gallimard'''}}》（{{IPA-fr|dekuvɛʁt ɡalimaːʁ}}，直譯「加利瑪發現文庫」，<ref>這個譯名取自本系列第53冊《馬內：「我畫我看到的！」》，書中原作「加利馬發現文庫」，是一種更準確的譯法。因為「叢書」對應的英文是「{{lang|en|book series}}」，而《{{lang|fr|Découvertes Gallimard}}》是一套{{lang|fr|collection}}，同 {{lang|en|book series}} 是兩種概念。比起「叢書」，「文庫」的意思更接近「{{lang|fr|collection}}」。</ref>「{{lang|fr|Découvertes}}」即「發現」之意；英國譯名：{{lang|en|New Horizons}}；美國譯名：{{lang|en|Abrams Discoveries}}）的中文譯本，由[[中華民國|臺灣]][[時報文化|時報文化]]翻譯發行。首卷中譯本《文字與書寫：思想的符號》於1994年5月31日推出，<ref>{{cite web |url=http://www.readingtimes.com.tw/ReadingTimes/ProductPage.aspx?gp=productdetail&cid=mcea(SellItems)&id=XB0001 |title=「發現之旅」系列第1冊：文字與書寫(XB0001)──思想的符號 |publisher=[[時報文化|時報文化]] |website=www.readingtimes.com.tw |accessdate=2017-04-03}}</ref>歷時12年陸續編輯發行，截止到2006年2月20日出版的《達利：超現實主義狂想天才》，<ref>{{cite web |url=http://www.readingtimes.com.tw/ReadingTimes/ProductPage.aspx?gp=productdetail&cid=mcda(SellItems)&id=XB0081 |title=達利(XB0081)──超現實主義狂想天才 |publisher=[[時報文化|時報文化]] |website=www.readingtimes.com.tw |accessdate=2017-04-03}}</ref>共計發行81冊。相較法文原版的700多冊（包含番外系列），這只是很小的一部份。法國原版的首部作品《{{lang|fr|À la recherche de l’Égypte oubliée}}》—由埃及學家[[尚·韋寇|尚·韋寇]]所著—在中譯版裡被作為第2冊《古埃及探祕：尼羅河畔的金字塔世界》。

== 概觀 ==
[[File:Presentación_de_La_escritura,_pp._4–5.jpg|thumb]]的著作《{{Link-en|大事記|Froissart's Chronicles}}》中的纖細畫。]]
《發現之旅》是{{Link-en|加利瑪青年出版社|Gallimard Jeunesse}}總裁{{Link-en|皮耶·馬爾尙|Pierre Marchand (editor)}}首創的「圖像閱讀」形式叢書，並締造了1980年代以來最成功的「圖像閱讀書」風潮。<ref>{{cite web |url=http://tpi.culture.tw/content-53-PStatusCtr-publishContent/706 |title=談"e"時代圖書行銷的觀念與做法(下) |work=臺灣出版資訊網 |accessdate=2017-04-23}}</ref>「發現」的圖書都是以32[[開本|開本]]大小（125 × 178 mm）的{{tsl|en|Pocket edition|口袋書}}樣式設計，以厚質帶光澤的[[銅版紙|雪銅紙]]印刷，<ref name=古文明系列 />，並配以豐盛的全彩插圖。每一冊敍述一個主題，作者都是邀請此一主題的學術權威執筆，均用{{tsl|en|Monograph|專題論文}}的形式寫作，內容涵蓋歷史、考古、宗教、音樂、美術、美學、科學等世界文明精華，<ref name=發現之旅>{{cite web |url=http://www.readingtimes.com.tw/timeshtml/ad/0903_discover/ |title=發現之旅系列簡介 |publisher=[[時報文化|時報文化]] |website=www.readingtimes.com.tw |accessdate=2017-04-03}}</ref>並有為數眾多的人物傳記、自然演化、地理變遷和古文明探檢史，<ref>{{cite web |url=http://www.readingtimes.com.tw/timeshtml/promote/renaissance/p06.html |title=發現之旅-藝術大師全系列 |publisher=[[時報文化|時報文化]] |website=www.readingtimes.com.tw |accessdate=2017-04-04}}</ref><ref>{{cite web |url=http://www.readingtimes.com.tw/ReadingTimes/ProductPage.aspx?gp=productdetail&cid=mcge(SellItems)&id=1ZD0142 |title=發現之旅-自然奧秘演化五書(1ZD0142) |publisher=[[時報文化|時報文化]] |website=www.readingtimes.com.tw |accessdate=2017-04-04}}</ref><ref name=古文明系列>{{cite web |url=http://www.books.com.tw/activity/new_prod/2003090501.html |title=發現之旅：古埃及、古文明系列，尋訪千年來的迷霧 |work=[[博客來|博客來]]新品活動 |accessdate=2017-04-04}}</ref>是一套[[百科全書|百科全書]]式的文庫合集。目前參與著述的作者已達502人，<ref name=Découvertes>{{cite web |url=http://www.gallimard.fr/Divers/Plus-sur-la-collection/Decouvertes-Gallimard/(sourcenode)/116085 |title=Collection Découvertes Gallimard |publisher=[[加利瑪出版社|加利瑪出版社]] |website=www.gallimard.fr |accessdate=2017-04-03|language=fr}}</ref>他們擅以[[時間軸|時序]]和歷史視角來闡述主題。比如，中譯本首卷《文字與書寫：思想的符號》就運用[[編年體|編年體]]追述自[[美索不達米亞文明|美索不達米亞文明]]以降的五千年[[文字史|文字史]]。這種敍述方式幾乎遍及整套書集，可說「歷史」與「編年紀」是構成《發現之旅》文本的基本元素。配給插圖的說明文也必須具備自身的訊息量，不能重複{{tsl|en|Body text|正文}}。作者要以一套主流書集的標準來寫作，創作出有一定水準的文本，此外，文本需與插圖完美配合。簡體中譯本第49冊《[[:commons:滿滿的書頁：書的歷史|滿滿的書頁：書的歷史]]》的作者布魯諾·布拉塞爾（{{lang|fr|Bruno Blasselle}}），巴黎[[阿瑟納爾圖書館|阿瑟納爾圖書館]]館長，描述了他為「發現」文庫工作的體驗：「對作者而言，撰寫一本『發現』有如被困在陷阱裡，迫使你無法按照自己已然習慣的計劃去工作，你必須超越自己的計劃。」<ref>{{cite episode |title=Dixième anniversaire de la collection Découvertes |series=Qu'est-ce qu'elle dit, Zazie ? |network=[[法國電視三台|法國電視三台]] |date=1997-05-15 |language=fr}}</ref>根據時報出版者的說法，譯介這套書系的構想是「在題材上擺脫狹隘的實用主義，就一個人智慧的發展方面提供多元選擇；在寫作角度上能跨越中國本位，以及近代過度來自美、日文化的影響，爲讀者提供更接近完整世界觀的思考角度；在設計與製作上，能夠呼應影像時代的視覺需求。」<ref>{{Cite book |last=Vercoutter |first=Jean |others=吳岳添/譯 |date=1994年5月31日 |edition=2002年1月1日第十六刷 |title=《古埃及探祕：尼羅河畔的金字塔世界》 |series=發現之旅 |volume=02 |url= |location=臺北 |publisher=時報文化 |page=第225頁 |chapter=編者的話 |language=zh-TW |isbn=957-13-1161-8 《發現之旅》的封面設計是其特色之一，法文本舊版的封面採用黑色背景，彩色印刷，有很強的光澤度，類似軟精裝本。新版封面則使用不具光澤感的啞光壓層設計，但拋棄了單一的黑色背景，顏色上更豐富多彩，並依據不同題材制定{{tsl|en|Color code|色碼}}。一張全幅面彩圖（{{lang|fr|image plein format}}）用於封面正面，一張和主題相關的小型圖片用於書脊，背面也附一張圖片以及「發現」文庫的[[獅身人面像|獅身人面像]][[標識|標識]]，以此讓「發現」在視覺效果上很明顯和其他智識性書籍區分開來。「發現」的其他語言版本也多採用加利瑪的設計，有了這種強烈的視覺辨識度，即使是不同語言版本的「發現」書籍，人們也可以輕鬆識別。不過也有例外，比如[[俄文|俄文]]版的「發現」書集就使用了迥然不同的封面設計。

此系列的單書都控制在200頁左右，並附有約120至200張源自各種古代藝術品和[[文化遺產|文化遺產]]的彩色插圖，由義大利分色製版。<ref name=發現之旅 />圖片使用四至七種顏色印製，有時也會使用[[金色|金色]]和[[銀色|銀色]]，比如中譯本第54冊《華格納：世界終極的歌劇》就使用了一種帶金屬質感的濃鬱金色；第21冊《吸血鬼：暗夜-{zh-hant:裡; zh-tw:裡; zh-hk:裡; zh-mo:裡; zh-sg:里; zh-cn:里; zh-hans:里}-尋找生命》使用了銀色。打開書以後，首先映入眼簾的是8–10幅佔據整頁的插圖、照片或影視劇照，有時輔以文字，一般是從文學作品中摘錄的段落或詩句。[[倫敦|倫敦]]{{tsl|en|Thames & Hudson|泰晤士和漢德森出版社}}（「發現」的英國版發行商）的總裁傑米·坎博林（{{lang|en|Jamie Camplin}}）稱這一系列開卷插圖是「受到電影影響的[[預告片|片花]]」。<ref name=In>{{Cite web |url=http://www.eyemagazine.com/feature/article/in-the-beginning-was-the-picture |title=In the beginning was the picture |author=Jim Davies |work={{link-en|Eye Magazine}}|date=1994 |accessdate=2017-09-30|language=en-GB}}</ref>例如中譯本第68冊《埃及學家商博良：破解古埃及文的天才》，迎接讀者的開卷畫面是一套取自[[商博良|商博良]]發表於1836年的手抄本《[[埃及語法|埃及語法]]》裡的彩頁。插圖設計師從雜誌中萃取靈感，將圖片製作的輪廓鮮明，即使是黑色背景，也會飾以金邊或銀邊。

[[File:Inside_page_view_of_‘Shakespeare_–_Comme_il_vous_plaira’_(Traditional_Chinese_edition).jpg|thumb]]
內頁[[版面設計|版面設計]]是「發現」的另一特色。根據不同主題，正文被劃分為三至八章節，各章節用新聞手法構造，附有導語和副標題，讀起來頗有時事報導的感覺。正文有時會被插頁打斷，好似幕間休息。所謂插頁是一幅佔滿整兩頁的跨頁圖片，或稱「圖像插入」，也有較小型的跨頁圖則是和文本結合在一起。除了新聞報紙的手法，電影元素對「發現」的影響也不容小覷。部份單書包含四折拉頁，展開可看到諸如宮殿等一類大型建築或自然地理的全景圖，彷彿大銀幕上的投影。閱讀「發現」給人的感覺就像是看電影，文本在這裡成為了畫外音，有時文學的意味濃郁。至於那些沒有文字說明的圖片，則像是電影中的定格鏡頭。<ref>{{cite news |editor=藜芙荷 |date=2007年2月8日 |title=【法國】伽利瑪的「發現」之旅 |url=http://blog.udn.com/jason080/760828 |newspaper=中國圖書商報 |location=北京 |access-date=2018年10月12日}}</ref>這種影像[[蒙太奇|蒙太奇]]的圖文編排，<ref>{{cite web |url=http://www.glt.org.tw/download/GLT-news-letter-pdf/GLT%20NO%2017.pdf |title=亞維儂—劇場的王國 |publisher=[[牯嶺街小劇場|牯嶺街小劇場]]文化報 |website=glt.org.tw |accessdate=2017-04-27}}</ref>使得每一冊都猶如一本小型[[圖鑑|圖鑑]]，抑或一部書本化的紀錄片。不同國家的版本有時會對書中內容稍作改動以使其更能迎合當地市場需要，比如在中譯本《古埃及探祕：尼羅河畔的金字塔世界》書末所附的「古埃及年表」內加註了中國歷代年表以茲對照；在法文版第26冊《{{lang|fr|Le ciel, ordre et désordre}}》中有一套8頁的[[螢光|螢光]]星空圖，於中譯本《星空：諸神的花園》（第9冊）裡替換成由[[臺北市立天文科學教育館|臺北市立天文科學教育館]]研究助理洪景川專為其繪製的「臺灣的星空」圖。<ref>{{Cite book |last=Verdet |first=Jean-Pierre |date=1994年5月31日 |title=《星空：諸神的花園》 |series=發現之旅 |volume=09 |url=http://www.readingtimes.com.tw/ReadingTimes/ProductPage.aspx?gp=productdetail&cid=rtxb(SellItems)&id=XB0009 |location=臺北 |publisher=時報文化 |page=193 |language=zh-TW |isbn=9571311545 |quote=129–136　星圖。洪景川繪。}}</ref>

每冊書的最後1/4部份為「見證與文獻」（{{lang|fr|Témoignages et Documents}}），不同於正文，這部份始終以黑白印刷，但依舊使用光澤度很高的雪銅紙。不過某些語言的譯本改用了普通紙張來印刷這部份，比如中譯本就是如此。「文獻」檔如同一部小型[[選集|選集]]，提供原始手稿與資料、索引、插圖出處等。「見證與文獻」通常由作者或出版社蒐集的檔案或材料表組合而來，每組資料起始會附上導言和簡短說明文。全冊插圖的來源會以頁數順序在「圖片目錄與出處」（{{lang|fr|Table des illustrations}}）一欄中列明。根據描述的主題，「見證與文獻」可能以一份[[年代學|編年表]]、{{tsl|en|Filmography|影片集錦|電影列表}}、[[音樂唱片分類|唱片列表]]、[[目錄學|書籍列表]]或[[索引|索引]]作結。

對細節的重視也是「發現」文庫的一大特點，在選擇[[字體|字體]]方面，{{link-en|Trump Mediaeval}} 用於正文；{{lang|en|Barnattan}}用於副標題；{{link-en|Zapf Dingbats|Zapf Dingbats|Zapf}} [[Dingbat|Dingbat]]s 用於法文的角形引號「{{lang|fr|«»}}」；[[義大利體|義大利體]]用於插圖的說明文並將[[首字放大|首字放大]]，最後一行添加[[下劃線|下劃線]]。正體字中譯本方面，正文一律採取字形更新前的[[新細明體#字形|細明體]]（[[舊字形|舊字形]]）；[[圓體|圓體]]用於插圖說明文，首字放大和下劃線均予以保留；[[標楷體|標楷體]]用於「見證與文獻」的導言和說明文。此外，前期發行的書目在取字上更為傳統，如取「{{lang|ja|靑}}」不取「青」、取「{{lang|ja|爲}}」不取「-{為}-」、取「{{lang|ja|兪}}」不取「俞」、取「{{lang|ja|緖}}」不取「-{緒}-」（「者」字多一點）、取「⺬」不取「⺭」等等，後期反之。「發現」的大多書目是在法國北部城市[[埃夫勒|埃夫勒]]印刷，其次就是在義大利。義大利印刷人賈尼·史達羅（{{lang|it|Gianni Stavro}}）長期以來為「發現」提供最新的印刷技術，他也是這套書集的固定國際合作人。基於牢固與否的考慮，所有書都採用線裝。<ref name=à-la-française>{{Cite web |url=https://www.ricochet-jeunes.org/articles/decouvertes-gallimard-ou-la-culture-encyclopedique-la-francaise |title=Découvertes Gallimard ou la culture encyclopédique à la française |author=Françoise Hache-Bissette |work=ricochet-jeunes.org |year=2002 |accessdate=2017-09-30 |language=fr}}</ref>加利瑪誓言要讓這套書集成為「世上最美的口袋文庫集」({{lang|fr|« La plus belle collection de poche du monde. »}})。<ref name=fête>{{YouTube|N5r-Eic9mSI|Découvertes Gallimard fête son n° 200 (1994)}}</ref> 

[[File:Découvertes_Gallimard,_séries_avec_des_codes_couleurs.jpg|thumb]]
早期的《發現文庫》由18個系列構成，分別是：考古（{{lang|fr|Archéologie}}）、宗教（{{lang|fr|Religions}}）、歷史（{{lang|fr|Histoire}}）、文學（{{lang|fr|Littérature}}）、哲學（{{lang|fr|Philosophie}}）、繪畫（{{lang|fr|Peinture}}）、建築（{{lang|fr|Architecture}}）、雕塑（{{lang|fr|Sculpture}}）、科學（{{lang|fr|Sciences}}）、技術（{{lang|fr|Techniques}}）、電影（{{lang|fr|Cinéma}}）、[[地方誌|地方誌]]（{{lang|fr|Mémoire des lieux}}）、[[自然史|自然史]]（{{lang|fr|Histoires naturelles}}）、傳統知識（{{lang|fr|Traditions}}）、探索世界（{{lang|fr|Invention du monde}}）、生活的藝術（{{lang|fr|Art de vivre}}）、音樂與舞蹈（{{lang|fr|Musique et danse}}）、體育與競技（{{lang|fr|Sports et jeux}}）。現在改組成7大主題，並配置色碼：藝術（紅色）、考古（褐色）、歷史（藍色）、宗教（深藍）、文學（白色）、文化與社會（黃色）、科學與技術（綠色）。

文字的故事始於五千年前的[[蘇美|蘇美]]，在[[美索不達米亞|美索不達米亞]]，也就是兩河流域，人們發明了書寫。他們不僅使用這些記號表述事物，也用來記錄自己的名字。[[楔形文字|楔形文字]]、[[埃及象形文字|埃及象形文字]]、[[阿拉伯字母|阿拉伯字母]]、中國的[[漢字|漢字]]：歷代的[[書吏|書吏]]彙編出人類的記憶。公元前14世紀，[[腓尼基|腓尼基]]人發明了字母。自此，只要將不同字母組合就能表達一種語言的全部發音。15世紀發明[[活字印刷術|活字印刷術]]以後，書寫的歷史便與[[文字設計|文字設計]]密不可分，也讓書寫的式樣更加普及和多元化。

語言學者兼符號學者[[喬治·尚|喬治·尚]]在《文字與書寫：思想的符號》一書中追溯文字同書寫的歷程，這個故事中包含了諸多發明家和解讀古代語言的學者。此書原名《{{lang|fr|L’écriture, mémoire des hommes}}》，意為「書寫：人類的記憶」，出版於1987年，是《{{lang|fr|Découvertes Gallimard}}》文庫的第24種書目，屬於「考古」系列（{{lang|fr|Archéologie}}），在上述文庫的中譯版本《發現之旅》叢書中則是「開幕」之作（第1冊）。通過古代遺跡、出土文物、歷史文獻記錄，作者以考古的視角為讀者闡述文字的歷史；通過歷代抄寫員、語言學家、古語釋讀者等等的事蹟，作者由古至今以[[編年體|時間線]]串連起整個故事。書中詳細介紹了各種各樣的寫作工具和[[介質|介質]]，比如石頭、粘土、紙莎草紙、羊皮紙、紙張、刀片、畫筆、蘆葦筆、鵝毛筆、鋼筆、印刷機等等。並討論了這些不同的寫作方式和印刷工具如何影響書寫內容的發展，及其是否可以大量流通，以及流通的方式和渠道。

此外，該書也描述了在不同年代和地域，書寫及參與這項工作的人員所代表的社會意義：古代美索不達米亞的書吏或中世紀的抄寫員究竟是何種身份？古羅馬帝國時代受過良好教育（識字）的公民和現代社會的公民有區別嗎？諸如此類，作者引領讀者把文字和書寫納入一個整體的人類社會和文化現象中去思考它們的本質。書中引用法國作家兼學者{{Link-en|安田樸|René Étiemble}}所說的話：「人類生生死死，代代相繼，已逾百萬年，但我們學會書寫的歷史僅有6000年。」依賴文字的發明和運用，6000年的歷史積纍於人類的記憶中，而沒有文字的歷史與時間則已無法追溯、無法丈量。所以，「文字的歷史幾乎就是人類的歷史本身」，其發展過程漫長而複雜。文字創造者石破天驚的稟賦，古文字破譯者釋讀神祕符號的才華，還有這些文字符號自身的美與暗示，構成了一個令人心醉神迷的奇妙世界。<ref>{{cite web |url=http://www.ewen.com.cn/cache/books/107/bkview-107423-308345.htm |title=書寫者的永恆形像——讀《文字與書寫：思想的符號》 |author=萬宇 |date=2003-08-21 |website=ewen.com.cn |access-date=2018-11-10}}</ref>

根據這套文庫的一貫準則，書中安排大量彩色插圖，印刷在極具光澤度的銅紙上——不同時代的繪畫、版畫、素描、古埃及紙莎草圖、中世紀[[泥金裝飾手抄本|泥金手抄本]]、文藝復興至近代的書籍裝飾圖案、印刷字體圖片、考古文物的照片…… 幾乎有如一本「[[視覺文學|視覺小說]]」。開卷「片花」是取自[[尙·傅華薩|尙·傅華薩]]的著作《{{Link-en|大事記|Froissart's Chronicles}}》中的一套繪製異常精美的纖細畫，其下配有文字說明。正文部份分成六個章節：
* 第一章：素樸的開端（{{lang|fr|Une humble naissance}}）——縱觀文字的發展史，其開篇幾乎都是相同的，遠古先民想出的第一批書寫符號都是圖畫。有人或許以為先民創製符號是為了保存他們的古老傳說，但文字出現的原因其實非常實際，比如保存賬目。隨後出現的猜字畫謎則是文字發展的關鍵。伴隨美索不達米亞文明的興起，書吏這一職業也應運而生。
* 第二章：神的發明（{{lang|fr|Une invention des Dieux}}）——當楔形文字在美索不達米亞全境傳播時，與其相鄰的古埃及和遙遠的古中國也產生了各自的文字系統。雖然處在世界的兩端，但這兩個民族都將文字視為天神所賜之物。
* 第三章：字母的革命（{{lang|fr|La révolution de l’alphabet}}）——公元前1000年之際，文字的演化史出現關鍵變革，那就是字母的發明。這要歸功於腓尼基人，一個居住在地中海東岸的航海與貿易民族。[[腓尼基字母|腓尼基字母]]後來傳入希臘，派生出[[希臘字母|希臘字母]]；從希臘字母又誕生了[[拉丁字母|拉丁字母]]，由此開啓西方文明的濫觴。在東方，腓尼基字母派生了[[阿拉米字母|阿拉米字母]]，其下又有[[希伯來字母|希伯來字母]]，及至今日阿拉伯世界所使用的[[阿拉伯字母|阿拉伯字母]]。中古時期的基督教和伊斯蘭教世界，「文字」一詞（法：{{lang|fr|Écriture}}；英：{{lang|en|Scripture}}）很多時候就是專指《[[聖經|聖經]]》和《[[古蘭經|古蘭經]]》。
* 第四章：從抄寫員到印刷工（{{lang|fr|Des copistes aux imprimeurs}}）——古羅馬帝國崩潰後，歐洲人繼承了羅馬人的語言和文字，即拉丁語及其書寫系統。西羅馬帝國滅亡5個世紀以後，法蘭克國王[[查理曼|查理曼]]宣佈基督教就是羅馬文明的繼承者，並在他的王國內努力復興行將消失的知識和文化。中世紀的抄寫員就此誕生，他們大多數都是修道院裡的僧侶。直到15世紀，[[約翰尼斯·古騰堡|約翰尼斯·古騰堡]]發明了[[活字印刷術|活字印刷術]]。
* 第五章：書籍的印製（{{lang|fr|Les hommes du livre}}）——受益於古騰堡印刷術的成功，抄寫員和手寫的藝術通通成為歷史。在其後的文藝復興、[[巴洛克時代|巴洛克]]和[[啟蒙時代|啟蒙時代]]，文字的世界，憑藉印刷術開始空前擴張。不過為了記錄思想，筆仍是不可或缺的工具。自19世紀以降，鋼筆、原子筆、打字機相繼問世。
* 第六章：古文字學者（{{lang|fr|Les déchiffreurs}}）——最後一章講述19世紀古文字釋讀者的故事，包括著名的古埃及文破譯者[[尚-法蘭索瓦·商博良|尚-法蘭索瓦·商博良]]，對楔形文字的破譯，[[克里特|克里特]]島上發現的仍未破譯的三種文字：[[線形文字A|線形文字A]]、[[線形文字B|線形文字B]]和[[斐斯托斯圓盤|斐斯托斯圓盤]]，以及作者最後提到在[[復活節島|復活節島]]上發現的神祕符號，其含義至今無人知曉。

此書的第二部份「見證與文獻」（129–207頁）蒐集彙編了有關文字與書寫的研究文章和文獻的摘錄，分成10個部份：1. 文字與城市；2. 鉛字的藝術；3. 數字的形像；4. 音樂的記錄；5. 工具與材料；6. 書法與文字的遊戲；7. 遠東的書寫藝術；8. 希臘字母及其他；9. 文字的禮讚；10. 《[[吉爾伽美什|吉爾伽美什]]》史詩。第8部份「希臘字母及其他」中還列出了取自[[狄德羅|狄德羅]]和[[達冷柏|達冷柏]]主編的《[[百科全書，或科學、藝術和工藝詳解詞典|百科全書]]》裡的希臘字母表、哥德字母表、俄羅斯字母表、日本假名表、漢字部首表、古代東方字母表、阿拉伯字母表、亞美尼亞字母表，以及坦米爾或通行於印度西海岸南段的[[馬拉巴爾|馬拉巴爾]]字母表。最後附文字地理分佈圖、圖片目錄與出處以及索引。

《文字與書寫：思想的符號》是《{{lang|fr|Découvertes}}》文庫最為暢銷的書目之一，<ref name=à-la-française />被翻譯成英國英文、美國英文、西班牙文、義大利文、巴西葡萄牙文、捷克文、斯洛伐克文、丹麥文、挪威文、瑞典文、德文、荷蘭文、波蘭文、匈牙利文、斯洛維尼亞文、土耳其文、希臘文、俄羅斯文、正體中文、簡體中文、日文以及韓文，並多次再版。該書初版有224頁，2007年發行的新版本，經過修訂以後只有160頁。

《加利瑪發現文庫》的誕生源於皮耶·馬爾尙的理念，這套文庫集最初命名為「{{lang|fr|Les chemins de la connaissance}}」（學識之道）。馬爾尙早在1972年他剛進入加利瑪出版社工作時就已有了這個想法，他說：「我為這套文庫集醞釀了14年之久，多虧《你是書中的主人公》叢書的成功，我們如今可以開啟新的航程…… 這是有史以來的第一套口袋百科全書…… 我們敢打賭，一旦你翻開這套書，無論你閱讀的是何種題材，也無論你讀到哪一頁，你都不忍心再把書合上。」<ref>{{cite magazine |last= |first= |date=1986年11月17日 |title=Comme un film documentaire |trans-title= |url= |language=fr |magazine=Livres Hebdo, n° 47 |location=巴黎 |publisher=Electre S.A. }}</ref>在當時，出版全彩且每頁插圖的口袋百科全書被認為是毫無理智之擧，不過「發現」仍然在1986年誕生了，來自{{tsl|en|Éditions Robert Laffont|勞勃·拉豐出版社}}的伊莉莎白·德·法茜（{{lang|fr|Élisabeth de Farcy}}）擔任文庫集的總監，她著手招募作者以及蒐集圖像文件。法茜在接受法國《{{tsl|en|La Croix|十字報}}》採訪時說：「圖像在『發現』中佔據很重要的位置，如同其在藝術作品中的效用。」<ref name=Élisabeth>{{Cite web |url=http://www.la-croix.com/amp/276508 |title=Élisabeth de Farcy, directrice de la collection « Découvertes Gallimard » : « Une collection de poche à l'ambition encyclopédique » |date=2006年11月13日 |publisher=十字報 |accessdate=2017年10月3日 |language=fr}}</ref>

「發現」的預覽版，或稱「第0期圖書」（{{lang|fr|numéro zéro}}）於1986年的夏季被送到500家書店，之後於同年11月正式推出。首發12冊，每冊印行25000本。<ref>{{cite magazine |last= |first= |date=1986年11月8日 |title=Spécial Gallimard jeunesse |trans-title= |url= |language=fr |magazine=Livres Hebdo, supplément au n° 50 |location=巴黎 |publisher=Electre S.A. }}</ref>{{link-en|BBDO}}廣告公司為「發現」寫下標語：「在第一頁和最後一頁之間，你將『發現』無窮的內容」。書集發行之前，馬爾尙並沒有做[[市場研究|市場研究]]，這位擅長[[自主學習|自修]]的總裁在法國知名電視節目《{{tsl|fr|Ça se discute|大家來討論}}》上解釋道：「這項計劃已醞釀經年，它萌生自我對知識開始渴求之時。毫無疑問，一個人需有精深的自修才能眞正意識到百科全書的重要性。我們一定都有過這樣的經歷：意識到必須建立我們自己的文化，尋求可靠的參考依據，做出明確的陳述或聲明。至於『發現』，我不需要做市場研究，從一開始，我希望的就是給予公眾我所需要的書籍。」<ref>{{cite magazine |date=1990年12月 |title=Mon tout, c'est Découvertes Gallimard |trans-title= |url= |language=fr |magazine={{tsl|en|Lire (magazine)|閱讀 (雜誌)|閱讀}} |location=巴黎 |publisher= }}</ref>雖然「發現」起初對讀者的定位是青少年，但這些書很快也在成人世界受到歡迎。這套書集隨之進入快速成長期，五年內便推出了105種書目。1989年迎來一家[[西班牙|西班牙]]的合作出版社，這是「發現」的第一位國際合作商，19個國家將會在未來的歲月陸續加入這項計劃。<ref name=Découvertes />1992年，在「發現」出版了151種書目以後，加利瑪對[[比利時|比利時]][[荷蘭語|荷蘭語]]作家{{tsl|nl|Vic de Donder|維克·德·棟德}}所寫的關於[[海妖|海妖]]和[[人魚|人魚]]的作品《{{lang|nl|De lokroep van de zeemeermin}}》（簡體中譯本第22冊《海妖的歌》）產生了興趣。但這家巴黎出版社有些許猶豫：「很有趣的題材，但你要如何為其配置插圖呢？」棟德隨之展示了大約1000張圖像文件，是他花了多年蒐集而來。這套令人印象深刻的圖片打消了加利瑪的疑慮，棟德成為「發現」叢書的第一位非法語作家。<ref>{{cite news |last=Vlaminck |first=M. |date=1992年10月24日 |title=Gallimard haalt De Donder binnen |trans-title= |url= |language=nl |newspaper={{tsl|en|Het Nieuwsblad|新聞報 (比利時)|新聞報}} |page=15 |location={{tsl|en|Groot-Bijgaarden|格魯特-拜嘉登}} |access-date= }}</ref>

20世紀80年代末至90年代初的幾年間是「發現」的全盛時期，作者大多來自學術界或是[[策展人|策展人]]。有關藝術家的著作，特別是畫家或音樂家，常常伴隨大型展覽出版或銷售。比如中譯本第65冊《馬諦-{}-斯：揮灑絢麗色彩》就是配合1993年在[[龐畢度中心|龐畢度中心]]擧辦的畫展「[[亨利·馬諦斯|亨利·馬諦斯]] 1904–1917」而發行；<ref name=à-la-française />來自「{{lang|fr|Hors série}}」的《{{lang|fr|Élisabeth Vigée Le Brun}}》於2016年在[[紐約市|紐約市]][[大都會藝術博物館|大都會藝術博物館]]擧辦的[[伊莉莎白·薇姬·勒布倫|伊莉莎白·薇姬·勒布倫]]畫展出售。<ref>{{Cite web |url=http://www.versopolis.com/panorama/127/elisabeth-louise-vige-le-brun-and-the-art-of-seduction |title=Elisabeth Louise Vigée-Le Brun and the Art of Seduction |author=Diane Joy Charney |publisher=Versopolis |accessdate=2017-10-03 |language=en }}</ref>也有很多書目是因應時事而創作，如中譯本第21冊《吸血鬼：暗夜-{zh-hant:裡; zh-tw:裡; zh-hk:裡; zh-mo:裡; zh-sg:里; zh-cn:里; zh-hans:里}-尋找生命》就是因應[[法蘭西斯·柯波拉|法蘭西斯·柯波拉]]的賣座電影《[[吸血鬼：真愛不死|吸血鬼：真愛不死]]》出版。時隔十八年後，又在《[[暮光之城系列|暮光之城系列]]》小說及電影大獲成功的推動下於2010年發行了此書的修訂版。<ref>{{Cite web |url=http://www.brunogerelli.info/Histoires-de-vampires-a-Claix.html |title=Histoires de vampires à Claix |author=Bruno Gerelli |website=www.brunogerelli.info |date=2011-07-17 |accessdate=2017-12-06|language=fr}}</ref><ref>{{Cite web |url=http://www.gallimard.fr/Catalogue/GALLIMARD/Decouvertes-Gallimard/Decouvertes-Gallimard/Culture-et-societe/Le-reveil-des-vampires |title=''Sang pour sang : Le réveil des vampires'', collection Découvertes Gallimard (n° 161) |publisher=[[加利瑪出版社|加利瑪出版社]] |website=www.gallimard.fr |accessdate=2017-12-06|language=fr}}</ref>法文原版第222冊《{{lang|fr|Un Conservatoire pour les Arts et Métiers}}》因應{{tsl|en|Conservatoire national des arts et métiers|法國國立工藝學院}}兩百週年紀念出版。「發現」也會因應時事而調整作品的完成時限，有些書目便是在極其有限的時間內出版，如法文版第60冊《{{lang|fr|Mémoires du Louvre}}》（[[羅浮宮|羅浮宮]]的記憶）是配合[[羅浮宮金字塔|羅浮宮金字塔]]於1989年的落成典禮；第77冊《{{lang|fr|Les Temples de l’opéra}}》（歌劇殿堂）獻給[[巴士底歌劇院|巴士底歌劇院]]於1990年落成。這兩部作品都是在六至八週內完成所有出版程序，而正常情況下需要花費二至三個月的時間。除了因時事而生的作品，這套書集也有一些令人好奇的書目，像是以[[紅髮|紅髮]]為主題的第338冊《{{lang|fr|Roux et rousses : Un éclat très particulier}}》；<ref name=à-la-française />介紹[[情色|情色]]史的第521冊《{{lang|fr|Histoire de l’érotisme : De l’Olympe au cybersexe}}》。某些題材可能不利銷售但被認為有必要為其創作書目：有關[[透視|透視]]（第138冊《{{lang|fr|La Perspective en jeu : Les dessous de l’image}}》）；[[人體|人體]]的圖解（第185冊《{{lang|fr|Les images du corps}}》）；[[矯飾主義|矯飾主義]]（第457冊《{{lang|fr|Le maniérisme : Une avant-garde au XVIᵉ siècle}}》）。<ref name=Élisabeth />不過近幾年間，以特定國家、城市或某個大型機構的歷史為主題的書目漸漸取代了藝術家，同時也包含諸多關於[[社會學|社會學]]和宗教題材的書目。例如，第204冊《{{lang|fr|New York : Chronique d’une ville sauvage}}》（簡體中譯本第56冊《紐約：一座野性城市的編年史》）是翻譯並改編自[[美國英語|英文]]的作品，第205冊也是關於城市歷史的書目：《{{lang|fr|La magie blanche de Saint-Pétersbourg}}》（[[聖彼得堡|聖彼得堡]]的白色魔法）；以[[同性戀|同性戀]]為主題的第417冊《{{lang|fr|La longue marche des gays}}》，有關[[藥物濫用|藥物濫用]]等社會問題的第423冊《{{lang|fr|Les drogues : Une passion maudite}}》；研究{{tsl|en|Marian devotions|聖母敬禮}}和[[聖母顯現|聖母顯現]]的書目：第401冊《{{lang|fr|La Vierge : Femme au visage divin}}》和第524冊《{{lang|fr|Le monde de Lourdes}}》。也有為數不少的書目獻予作家，如第381冊《{{lang|fr|Marcel Proust : La cathédrale du temps}}》（中譯本第58冊《普魯斯特：建構時光大教堂》）。不太主流的題材反而有更大的需求量，也許是因為當時的競爭相對較小，因此，一本關於[[熙篤會|熙篤會]]僧侶的書（第95冊《{{lang|fr|Le rêve cistercien}}》）是1990年的暢銷書之一。而其他一些更顯主流的題材，比如[[淘金熱|淘金熱]]（第34冊《{{lang|fr|La fièvre de l’or}}》）和[[海盜|海盜]]（第45冊《{{lang|fr|Sous le pavillon noir : Pirates et flibustiers}}》）的銷售成績則不太理想。<ref name=à-la-française />一般來說，來自「藝術」和「考古」兩個系列的書目是最受歡迎的。《古埃及探祕：尼羅河畔的金字塔世界》和《文字與書寫：思想的符號》是「發現」最暢銷的兩種書目，<ref name=Découvertes />均出自「考古」系列。截止到2001年，前者在世界範圍內售出50多萬冊。至於「體育與競技」系列，其中只有4本最受喜愛：[[環法自由車賽|環法自由車賽]]（第81冊《{{lang|fr|La saga du Tour de France}}》），[[足球|足球]]的歷史（第83冊《{{lang|fr|La balle au pied : Histoire du football}}》），[[奧林匹克運動會|奧林匹克運動會]]（第133冊《{{lang|fr|Jeux Olympiques : La flamme de l’exploit}}》）和[[橄欖球類運動|橄欖球]]的歷史（第164冊《{{lang|fr|Voyous et gentlemen : Une histoire du rugby}}》）。<ref name=à-la-française />到[[1990年代|1990年代]]中期，競爭開始出現，新書目的創作量和發行量逐漸減少，但部份題材被收入了番外篇《'''{{lang|fr|Découvertes Gallimard Hors série}}'''》。<ref name=Découvertes />1994年，「發現」完成了第200種書目《{{lang|fr|Voyages en Utopie}}》（駛向[[烏托邦|烏托邦]]）的創作，加利瑪於3月25日在{{tsl|en|Musée national des Monuments Français|法國國立古蹟博物館}}為書集擧辦了慶功宴。<ref name=fête />

如今，每年大概有15種書目會根據當前的文化和科學研究對內容進行更新，新版書目的設計也作了很多改變。第一次是自1998年9月發行的第359種書目《{{lang|fr|Le Théâtre de Boulevard : « Ciel, mon mari ! »}}》起，調整了外觀設計，拋棄了封面的黑色背景，並將字體印刷得更醒目。第二次是從2000年3月出版的第390冊《{{lang|fr|L’Esprit des savoirs}}》開始，調整了內頁[[版面設計|版面設計]]。經過再版的舊書目，刪去了「見證與文獻」部份的某些內容，整體頁數也被縮減。<ref name=Découvertes />這套文庫集在誕生後的幾年內即與一種表面化又快速轉換的視覺文化（{{lang|fr|culture visuelle du zapping}}）連繫在一起，「發現」為此重申了自己的初衷：圖像並非凌駕於文本，而是同文本結合在一起，令閱讀體驗更豐富更具活力。<ref name=Découvertes />書集從最新科技中受益良多，現在全部使用[[桌面出版|桌面出版]]編輯。由於{{link-en|QuarkXPress}}軟體是在1987年才發佈，「發現」的最初30種書目都是用[[照相排版|照相排版]]。一本「發現」的目前價位，根據頁數的多少，介於8.40至15.90[[歐元|歐元]]之間（於臺灣為[[NT$|NT$]]250–280；於香港是[[HK$|HK$]]83–93；在中國是[[CN¥|CN¥]]28–40），考慮到「發現」書籍的品質，這被認為是相當便宜的價格。<ref name=à-la-française />

[[File:Encyclopedie_de_D'Alembert_et_Diderot_-_Premiere_Page_-_ENC_1-NA5.jpg|thumb]]》是第一部法文百科全書，由[[狄德羅|狄德羅]]、[[達冷柏|達冷柏]]和其他多名編撰者合作完成。它是現代百科全書形式的源頭之一，也是首部有諸多[[百科全書派|具名的編撰者]]合作的作品，或許還是最著名的早期百科全書。《加利瑪發現文庫》將它的第100種書目《{{lang|fr|Le Roman vrai de l’Encyclopédie}}》（百科全書的實事小說）獻予這部[[啟蒙時代|啟蒙時代]]的著作。]]
「發現」至今已有20多種語言的譯本，包括英文、西班牙文、俄羅斯文、阿拉伯文、中文、日文、韓文等。{{tsl|en|Gallimard Jeunesse|加利瑪青年出版社}}的現任總裁海德薇·帕絲奎（{{lang|fr|Hedwige Pasquet}}）說：「這套價格平易近人的口袋百科全書實現了它的[[人文主義|人文主義]]夢想，讓人人都有機會學習各種領域的知識，而且這些知識都是與時俱進的。[[狄德羅|狄德羅]]和[[達冷柏|達冷柏]]定會對此非常贊同……」<ref name=Découverte>{{Cite web |url=http://www.bibliomonde.com/collection/decouverte-749.html |title=Découverte |work=bibliomonde.com |accessdate=2017年10月4日 |language=fr}}</ref>根據法國《{{tsl|fr|Livres Hebdo|讀書週刊}}》的記錄，截止到1999年，這套「法國式文獻書庫」在全世界擁有超過二千萬冊的銷售量。<ref>{{Cite magazine |last= |first= |date=1999年1月 |title=Livres Hebdo (nº 321) |trans-title= |url= |language=fr |magazine=Livres Hebdo, n° 321 |location=Paris |publisher=Electre S.A. |accessdate= }}</ref>自90年代起開始進入[[東歐|東歐]]和亞洲市場，尤其在[[日本|日本]]和[[俄羅斯|俄羅斯]]。前者是首個出版「發現」的東亞國家，自1990年開始截止到2017年已發行166種書目；<ref>{{Cite web |url=https://www.sogensha.co.jp/book_series_sai/ayumi/index.html |title=「知の再発見」双書の歩み|publisher=創元社 |accessdate=2017-10-04|language=ja}}</ref><ref>{{Cite web |url=https://www.sogensha.co.jp/productlist/detail?id=1686 |title=「知の再発見」双書166『ヴュイヤール──ゆらめく装飾画』|publisher=創元社 |accessdate=2017年10月4日 |language=ja }}</ref>後者四年內推出了約100種書目。為了解決國際知識產權和藝術品的複製權問題，國際出版社首先確定要發行多少種書目，然後再根據自身的編輯流程來選擇題材，並分攤高昂的攝影作品版權費。因此，美國的{{tsl|en|Abrams Books|亞伯蘭斯出版社}}為他們的英譯本「發現」《{{lang|en-US|Abrams Discoveries}}》選擇了眾多與傳統文化題材相關的書目，如第56冊《{{lang|fr|À la recherche de la Rome antique}}》（中譯本第27冊《羅馬考古：永恆之城重現》），第86冊《{{lang|fr|La Naissance de la Grèce}}》（中譯本第3冊《希臘的誕生：燦爛的古典文明》），第360冊《{{lang|fr|La redécouverte de la Chine ancienne}}》（再探[[中華文明|中華文明]]史）等；而日本的{{link-ja|創元社}}更偏好具有原創性質的書目，如第57冊《{{lang|fr|Les sorcières, fiancées de Satan}}》（中譯本第45冊《女巫：撒旦的情人》）。<ref name=à-la-française />西班牙是第一個引進「發現」的國家，來自[[馬德里|馬德里]]的{{tsl|es|Aguilar (editorial)|阿奇拉出版社}}於1989年首發了12種書目；<ref>{{Cite magazine |date=1989年11月12日 |title=AGUILAR UNIVERSAL – La primera enciclopedia ilustrada a todo color en formato de bolsillo |trans-title= |url=http://hemeroteca.abc.es/nav/Navigate.exe/hemeroteca/madrid/blanco.y.negro/1989/11/12/101.html |language=es |magazine={{tsl|en|Blanco y Negro (magazine)|黑與白 (雜誌)|黑與白}} |location=馬德里 |publisher={{tsl|es|Editorial Católica|天主教出版社}} |accessdate=2017年10月4日 }}</ref>在[[義大利|義大利]]，從1992年開始，伊萊塔/加利瑪出版社（{{lang|it|Electa/Gallimard}}）七年內推出了128種書目；在[[英國|英國]]，{{tsl|en|Thames & Hudson|泰晤士和漢德森出版社}}於1992年5月首發12冊，<ref name=In />至2008年共發行107種書目；在[[美國|美國]]，於1992至2008年間出版了111種書目；在[[南韓|南韓]]，[[時空社|時空社]]從1995年起截止到2017年12月發行了142種書目。<ref>{{Cite web|url=https://www.sigongsa.com/books/bookView.php?bookcode=SB002076&catecode=01010300 |title=디스커버리총서 1 - 문자의 역사 |website=www.sigongsa.com |publisher=時空社 |language=ko |accessdate=2017-10-04}}</ref>此外，國際版本通常會合作印製以節省花費，並支援發行量少的國家。<ref name=à-la-française />比如英國和美國就合作分攤印刷費和翻譯費，美國版再根據{{tsl|en|American and British English spelling differences|美式及英式英語拼法差別|詞彙拼寫}}和慣用語的差異進行重新編輯，反之亦然。<ref name=In />

除了與眾多國外出版社合作，「發現」也和各種本國的大型機構建立夥伴關係。最矚目的是自1989年以來與{{tsl|en|Réunion des Musées Nationaux|法國國家博物館聯合會}}的合作：截止到2001年，共同推出了32種書目，外加一種英文書目（第277冊《{{lang|en|Corot: Extraordinary landscapes}}》）、[[印象派|印象派]]{{lang|en|[[Boxset|Boxset]]}}、四種「{{lang|fr|Hors série}}」書目。合作的目的之一是可以共攤成本並共享收入，其二是結合聯合會的博物館資源與出版社的[[內容傳遞網路|資訊傳遞]]能力，令作品在質量和效率上都大大提升。當一部作品通過聯合會伴隨大型展覽推出，就能使銷售量大增。其他合作機構還包括{{tsl|en|French Alternative Energies and Atomic Energy Commission|原子能與替代能源委員會}}（第282冊《{{lang|fr|L’atome, de la recherche à l’industrie : Le Commissariat à l’Énergie Atomique}}》），{{tsl|en|Crédit Mutuel|國民互助信貸銀行}}（第379冊《{{lang|fr|L’odyssée de l’euro}}》），[[萊雅集團|萊雅集團]]（第405冊《{{lang|fr|Les vies du cheveu}}》—簡體中譯本無序號《秀髮絲語》），以及有數百年歷史的{{tsl|fr|Arjowiggins|奧若維津士紙業公司}}（第369冊《{{lang|fr|Le papier : Une aventure au quotidien}}》）等等。「發現」有時會在「致謝」中明確提及合作方（第379冊），但通常只有隱晦的提示（第405冊），甚至低調的保持沈默（第389冊《{{lang|fr|Vive l’eau}}》）。<ref name=à-la-française />

文獻紀錄式書籍的出版在法國已經衰落多年，{{tsl|en|Market saturation|市場飽和}}是一大主因，來自其他[[傳播媒體|媒介]]的競爭是另一大原因。特別是現今發達的[[網際網路|網際網路]]，令一部份年輕世代無意再購買實體書籍。儘管《發現文庫》有極其出色的公眾形象，但出版社仍不免擔心遭受眾多新興事物的排擠。在國際上，書集無疑獲得很大成功，但在它的家鄉法國，銷量下降，情況正在惡化，新書目的創作量和印刷量都在縮減。「發現」的創始人皮耶·馬爾尙在為加利瑪出版社工作了27年之後，於1999年3月1日轉投了[[阿歇特出版公司|阿歇特出版公司]]。對「發現」來說，這無疑是極大的損失，好在已有10多年的發行經驗，且書集的經濟效益一直不錯，因而積累下一筆可觀的資金，讓「發現」最終度過難關。<ref name=à-la-française />2011年，為慶祝加利瑪出版社一百週年紀念，「發現」推出了它的第一本[[iPad|iPad]][[電子書|電子書]]：第569種書目《{{lang|fr|Gallimard : Un éditeur à l’œuvre}}》（加利瑪：孜孜不倦的出版商）。<ref name=Découvertes />

時至今日，唯有《{{lang|fr|Découvertes Gallimard Hors série}}》系列還在持續更新。這套文庫集的主體部份《{{lang|fr|Découvertes Gallimard}}》已經停止新書目的創作，只於每年再版部份舊書目，再版的書目一般會重新配置封面圖。

== 圖像 ==
[[File:Measuring_Temple_of_the_Frescos.jpg|thumb]]
圖像在「發現」中佔據很重要的位置，這套叢書從雜誌的內頁[[版面設計|版面設計]]上獲得很多靈感。但在[[1980年代|1980年代]]，還不存在[[桌面出版|桌面出版]]和照片數位化技術，所有複雜的書籍實物模型全部手工製作，插圖師不得不往返於博物館、圖書館、畫廊等地蒐集圖像文件。如今，科技的發展簡化了這些程序，但新的困難也在不斷出現，使用一張圖片所需的手續越來越多，情況更趨複雜。<ref name=Élisabeth />有關當代題材的插圖所需的花費更為高昂，因為出版社必須通過攝影經銷公司取得圖片的正當使用權。未經發表的圖片永遠作為首選，除了從博物館或圖書館搜尋資料，「發現」的插圖師們也持有一筆資源。19世紀英國探險家{{tsl|en|Frederick Catherwood|費德利克·加瑟伍德}}的一套描繪[[馬雅文明|馬雅]]遺跡的畫作被用於第20種書目《{{lang|fr|Les cités perdues des Mayas}}》（中譯本第8冊《馬雅古城：湮沒在森林-{zh-hant:裡; zh-tw:裡; zh-hk:裡; zh-mo:裡; zh-sg:里; zh-cn:里; zh-hans:里}-的奇蹟》）中，這些繪畫就是由一名插圖師提供。<ref name=à-la-française />

一般來說，有關藝術、文明、考古的題材不難配置插圖，但如果要為「[[疼痛|疼痛]]」（第370冊《{{lang|fr|La douleur : Un mal à combattre}}》）或「[[永續發展|永續發展]]」（第495冊《{{lang|fr|Le développement durable : Maintenant ou jamais}}》）這樣的主題蒐集插圖，情況會變得相當「挑剔講究」，因為必須要避免毫無新意的重複圖像或單調乏味的圖片。插圖師需要拓寬搜尋領域，利用歷史文獻、藝術作品、{{tsl|en|Film still|電影劇照}}等資源來解決這個問題。<ref name=Élisabeth />

== 反響 ==
法國週刊《[[電視全覽|電視全覽]]》讚揚《發現之旅》為「結合了影視作品的懸念與新聞業界的效率，並以文學予其氣質，藝術增其華美」；<ref name=Découvertes />德國《{{tsl|en|Die Zeit|時代週報 (德國)|時代週報}}》評價這套叢書是「一場充滿意義、驚奇滿載又令人激動的探險之旅」。<ref>{{cite web |url=http://www.zeit.de/1990/52/abenteuer-geschichte |title=Abenteuer Geschichte: Eine spannende neue Reihe |author={{link-de|Gerhard Prause}} |date=1990-12-21 |publisher={{link-en|Die Zeit}} 雖然「加利瑪發現」書系以其獨特品質和精緻的視覺設計受到讚揚，但這套叢書在定位上遭遇很多問題。許多書商對分類頗感困擾：究竟是作為成人閱讀書籍，還是青少年書籍？應該歸類在口袋書一欄，還是[[人文科學|人文科學]]的紀錄性文獻叢書？最後只能按題材將這些書分散安置。由於競爭激烈，大多書商都拒絕再將「發現」陳列在書店裡最顯眼的位置。另一大原因是由於加利瑪出版社的庫存相當緊張，「發現」的幾百冊書目從來都不齊全，這一點無法讓顧客感到滿意。同時，定位的問題也潛伏在報刊等印刷媒介方面。比如，就私人而言，大多記者都很樂意於從報刊宣傳機構獲取「發現」最新出版的書目，但他們對在自己的報導中大篇幅介紹這樣一本難以歸入[[隨筆|隨筆]]類別的小書明顯缺乏熱情。即使這本書是新近面世而非舊書再版，情況也不會有多好。<ref name=à-la-française />

== 發售型態 ==
