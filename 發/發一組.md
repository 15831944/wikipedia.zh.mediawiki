'''發一組'''是[[一貫道|一貫道]]在[[臺灣|臺灣]]傳播的十八組線之一，由[[韓恩榮|韓恩榮]][[老前人|老前人]]創建，[[母壇|母壇]]{{Efn|一貫道來臺組線多以在中國大陸的道壇為單位，故有此名}}為[[天津市|天津]]同興壇。道務中心在[[南投縣|南投縣]][[埔里鎮|埔里鎮]][[天元佛院|天元佛院]]。韓恩榮[[歸空|歸空]]{{Efn|一貫道道親逝世的用語}}後，續留該組的支線由[[發一崇德|發一崇德]]支線的[[陳鴻珍|陳鴻珍]][[前人_(一貫道)|前人]]領導，在她歸空後則各自發展，仍以天元佛院為橫向聯繫中心。<ref name=":0">{{Cite journal|title=發一探源──天津一貫道「同興壇」研究|author=林榮澤|url=http://www.iktcds.edu.tw/Chong-De-School/img/research/201508-04-%E7%99%BC%E4%B8%80%E6%8E%A2%E6%BA%90--%E5%A4%A9%E6%B4%A5%E4%B8%80%E8%B2%AB%E9%81%93%E3%80%8C%E5%90%8C%E8%88%88%E5%A3%87%E3%80%8D%E7%A0%94%E7%A9%B6.pdf|date=2015-08|journal=一貫道研究|publisher=[[一貫道崇德學院|一貫道崇德學院]]|accessdate=2019-01-29|issue=|doi=|others=|year=|volume=4|language=zh-tw|pmid=|archiveurl=https://web.archive.org/web/20180820085622/http://www.iktcds.edu.tw/Chong-De-School/img/research/201508-04-%E7%99%BC%E4%B8%80%E6%8E%A2%E6%BA%90--%E5%A4%A9%E6%B4%A5%E4%B8%80%E8%B2%AB%E9%81%93%E3%80%8C%E5%90%8C%E8%88%88%E5%A3%87%E3%80%8D%E7%A0%94%E7%A9%B6.pdf|archivedate=2018-08-20|deadurl=no}}</ref>{{Rp|45}}

該組是十八組線中較有系統架構的宗教組織，亦是一貫道中知識份子最多的一組。<ref name=":1">{{Cite book|url=https://hdl.handle.net/11296/8447u9|title=台灣一貫道發一崇德雲林道場發展史|last=董坤耀|publisher=國立中正大學|accessdate=2019-01-29|year=2010}}</ref>{{Rp|10}}

== 名稱 ==
[[File:韓萬年.jpg|缩略图]]]]
依據韓恩榮之子<ref name=":1" />{{Rp|53}}[[韓萬年|韓萬年]][[點傳師|點傳師]]所述，「發一」這個名稱是1947年左右，中國人民對一貫道產生反感，師母[[孫素貞|孫素貞]]於是將師尊[[張天然|張天然]]用的求道願文中「一貫大道」改為其他名稱，天津地區被改為「發一大道」，取「發揚一貫」之意。「發一」名稱在今日的臺灣，雖依然可作為母壇位居天津的諸支線代稱，但已演變成為「同興壇」到臺教徒發展的支線專稱。<ref name=":0" />{{Rp|46}}

== 歷史 ==
同興壇由韓恩榮在1939年於其經營的天津大德隆染織廠內開設，距離張天然在天津的「天真總壇」甚近，所以逐漸成為天津地區道務推展核心。<ref name=":0" />{{Rp|57-58}}1939年夏，後來[[發一靈隱|發一靈隱]]的創建[[前人_(一貫道職級)|前人]][[李鈺銘|李鈺銘]]在同興壇入道；次年[[發一崇德|發一崇德]]創建前人[[陳鴻珍|陳鴻珍]]在天津明德壇入道，後來多在同興壇辦道<ref name=":0" />{{Rp|59}}。1947年初，韓恩榮在同興壇神職人員的聚會上提起到臺傳道事宜，劉振奎前人等人率先前往臺灣，次年韓恩榮亦前往臺灣<ref name=":0" />{{Rp|72}}。1947、48年間，約有20餘名神職人員來臺<ref name=":0" />{{Rp|73}}。

初到臺灣的發一組人員由韓雨霖領導，在台北市東門市場經營「同德商行」維生並居住在該店內。1949年6月，來自[[雲林縣|雲林縣]][[斗六|斗六]]的李清賀在[[斗六車站|斗六車站]]旁買了一幢建築經營「青年照相館」，在其內設立發一組在臺首間佛堂，後因「官考」{{Efn|一貫道對於政府取締的稱呼}}而先後遷至太和旅社、[[斗六崇修堂|斗六崇修堂]]。<ref name=":1" />{{Rp|52-53}}

韓恩榮[[歸空|歸空]]後，續留發一組的支線由陳鴻珍領導；陳鴻珍歸空後，則各自發展，仍以天元佛院為橫向聯繫中心。<ref name=":0" />{{Rp|45}}2011年，[[無極老母|無極老母]][[降壇|降壇]]指示「發一組受命者是兒萬年」，韓萬年依此成為發一組第三代領導人<ref name="韓萬年天命">{{cite web |author1=施春兆 |title=明本宮「天命道統真傳」活動報導 |url=http://www.iktcds.edu.tw/Chong-De-School/act-each-unit-21.htm |publisher=一貫道崇德學院 |accessdate=2019-04-03|archiveurl=https://web.archive.org/web/20190403123729/http://www.iktcds.edu.tw/Chong-De-School/act-each-unit-21.htm|archivedate=2019-04-03}}</ref>。

== 支線 ==
發一組在一貫道十八組線中是較有系統架構的宗教組織，亦是一貫道知識份子最多的一組<ref name=":1" />{{Rp|10}}。其下有[[發一崇德|發一崇德]]、[[發一天恩|發一天恩]]、[[發一靈隱|發一靈隱]]、[[發一天元|發一天元]]四個較大的支線，及發一光耀、發一奉天、發一德化、發一同義、發一慈濟、發一慧圓（慧音）、發一慈法七個較小的支線。<ref name=":0" />{{Rp|45}}

=== 發一崇德 ===
[[File:Sanbao_Temple_三寶宮.jpg|thumb]]雲林道場六和區的廟宇佛堂三寶宮]]

{{Main|發一崇德}}
崇德支線是發一組十一條支線中較大四者之一，亦是發展中最具規模的一條支線。其發跡於雲林道場。<ref name=":1" />{{Rp|10}}
