'''發現者'''（{{lang-en|Discoverer}}），是一套由[[連震東|連震東]]基金會籌劃，華暘傳播公司製作的一系列紀錄片，獲[[中華民國教育部|中華民國教育部]]推薦爲優良文化教育節目<ref>{{Cite web |url=http://www.lnanews.com/news/FGS%E3%80%8A%E7%99%BC%E7%8F%BE%E8%80%85%E3%80%8B%E4%B8%89%E6%9C%88%E5%9C%A8%E4%BA%BA%E9%96%93%E8%A1%9B%E8%A6%96%E9%A6%96%E6%92%AD.html |title=FGS《發現者》三月在人間衛視首播 |accessdate=2017-12-22 |work= |publisher=人間通訊社 |date= }}</ref>，曾於[[中華民國|中華民國]][[公共電視文化事業基金會|公共電視]]分為104集播出<ref>{{Cite web |url=http://web.pts.org.tw/php/mealc/main.php?XMAENO=607 |title=節目名稱：發現者 |accessdate=2017-12-22 |work= |publisher=公共電視 |date= }}</ref>，爾後又於[[東森電視台|東森電視台]]重播<ref>{{Cite web |url=http://forums.chinatimes.com/special/second/87EE0903.htm |title=連惠心鍾情電視 淡然「孕」事 |accessdate=2017-12-22 |work= |publisher=中國時報 |date= }}</ref>。

==概述==
發現者紀錄片為首套由臺灣自製的歷史文化紀錄片，自1996年起，由[[連戰|連戰]]之女[[連惠心|連惠心]]擔綱策劃<ref name=工商報導>{{Cite web |url=http://www.epochtimes.com/b5/13/9/11/n3961570.htm |title=【工商報導】新唐人《發現者》 記錄人類文明與榮耀之路 |accessdate=2017-12-22 |work= |publisher=大紀元時報 |date=2013-09-11 }}</ref>，據悉籌拍《發現者》乃打算做成臺灣版的《[[Discovery|Discovery]]》<ref>{{Cite web |url=https://tw.appledaily.com/lifestyle/daily/20040802/21127473 |title=連戰女兒 違規兼差 |accessdate=2017-12-22 |work= |publisher=[[蘋果日報_(台灣)|蘋果日報]] |date=2004-08-02 }}</ref>，內容全數以實地取景拍攝。紀錄片早期外景主持人包括[[任賢齊|任賢齊]]、[[九孔|九孔]]、[[許傑輝|許傑輝]]等人，製作團隊曾遠赴[[埃及|埃及]]出外景<ref>{{Cite web |url=https://tw.appledaily.com/finance/daily/20060410/22528061 |title=任賢齊《心太軟》 翻紅飆男兒淚 |accessdate=2017-12-22 |work= |publisher=[[蘋果日報_(台灣)|蘋果日報]] |date=2006-04-10 }}</ref><ref>{{Cite web |url=https://tw.news.yahoo.com/%E5%B0%8F%E9%BD%8A%E5%94%B1-%E5%86%8D%E5%87%BA%E7%99%BC-%E6%85%B0%E6%A7%8D%E5%82%B7-213000201.html |title=小齊唱〈再出發〉慰槍傷 |accessdate=2017-12-22 |work= |publisher=中時電子報 |date=2011-12-04 }}</ref>。
《發現者》從人類文明起源開始，進而闡述世界各國文明歷程，涵蓋13個主題：《人類文明起源之旅》、《西方文明起源之旅》、《中國文明之旅》、《回教文明、東方印度與佛陀文明之旅》、《羅馬文明興衰之旅》、《基督教文明之旅》、《絲路文明之旅》、《失落的文明之旅》、《世紀藝文之旅》、《近代歐洲躍進之旅》、《俄羅斯文明之旅》、《日本文化之旅》。由沙鷗國際多媒體推出DVD共91集，亦以各種主題分輯推出<ref>{{Cite web |url=http://www.ylib.com/epaperhistory/Best100Epaper/090902juedm.htm |title=《發現者》DVD |accessdate=2017-12-22 |work= |publisher=遠流出版公司 }}</ref>。

==內容簡介==
* 《人類文明起源之旅》（1–10集）：上古人類文明之源起綜述，繼而分述[[美索不達米亞文明|美索不達米亞文明]]、[[古埃及文明|古埃及文明]]及[[歐洲|歐洲]][[巨石文化|巨石文化]]。
* 《西方文明起源之旅》（11–18集）：[[克里特文明|克里特文明]]、[[古希臘文明|古希臘文明]]、[[希臘化|希臘化]]文明時期以及[[波斯帝國|波斯帝國]]文化。
* 《中國文明之旅》（19–32集）：從[[中國史前文化|遠古時代]]至[[清朝|清朝]]的中國文明歷程，本系列最後一集講述[[臺灣荷蘭統治時期|臺灣荷蘭統治時期]]的歷史文化。
* 《東方印度與佛陀文明之旅》（33–37集）：[[古印度|古印度]]文明同[[印度佛教|印度佛教]]文化。
* 《羅馬文明興衰之旅》（38–44集）：薈萃[[古羅馬|古羅馬]]歷代文化，源起、發展、興衰、滅亡。第40集專述[[龐貝古城|龐貝古城]]。
* 《基督教文明之旅》（45–55集）：從[[猶太教|猶太教]]時期開始，囊括[[羅馬帝國|羅馬帝國]]時代、[[拜占庭帝國|拜占庭帝國]]時代及至[[中世紀歐洲|中世紀歐洲]]的[[基督宗教|基督宗教]]歷程。
* 《回教文明之旅》（56–58集）：[[伊斯蘭教|伊斯蘭教]]發展歷程及[[鄂圖曼帝國|鄂圖曼帝國]]文化。
* 《絲路文明之旅》（59–64集）：絲綢之路上的文明發展，[[敦煌|敦煌]]、[[西域|西域]]古國及[[絲路|絲路]]宗教。
* 《失落的文明之旅》（65–71集）：[[美洲|美洲]]三大文明[[瑪雅|瑪雅]]、[[印加|印加]]、[[阿茲特克|阿茲特克]]的興衰史。
* 《世紀藝文之旅》（72–75集）：述說[[莎士比亞|莎士比亞]]、[[文藝復興|文藝復興]]、[[巴洛克|巴洛克]]藝術與文化及[[俄羅斯|俄羅斯]]等主題。
* 《近代歐洲躍進之旅》（76–80集）：[[地理大發現|地理大發現]]、[[啟蒙時期|啟蒙時期]]、[[法國大革命|法國大革命]]、[[工業革命|工業革命]]與19世紀諸多的文化藝術流派。
* 《俄羅斯文明之旅》（81–85集）：從俄羅斯誕生開始，[[維京|維京]]血統、[[基輔|基輔]]受洗、俄羅斯統一、[[彼得大帝|彼得大帝]]西化、[[沙皇俄國|沙皇俄國]]的黃金時代至末代沙皇[[尼古拉二世|尼古拉二世]]。
* 《日本文化之旅》（86–88集）：由講述早期歷史「[[天照大神|太陽之子]]」、中世的「[[戰國時代_(日本)|日本之戰國霸業]]」同「近代[[日本|日本]]」組成。

* 不分系列（89–91集）：由講述[[第一次世界大戰|第一次世界大戰]]的「戰爭與和平」、[[第二次世界大戰|第二次世界大戰]]的「蒼白之路」與最終集的「發現未來」所組成。
