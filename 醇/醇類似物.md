'''醇類似物'''是泛指任何有類似於[[醇類|醇類]]或[[羥基|羥基]]結構的[[有機化合物|有機化合物]]，例如[[硫醇|硫醇]]、[[酚|酚]]等。大部分的醇類似物都有類似的性質，例如易揮發和刺激性等性質。
== 酚 ==
[[File:Fenol.png|thumb]]
{{main|酚}}
酚是一種具有[[羥基|羥基]]的[[芳香烴|芳香烴]]，不屬於醇類，但它依然屬於醇類似物。最常見的是[[石炭酸|石炭酸]]([[苯酚|苯酚]])。
{{clear}}

== 硫醇 ==
[[File:Thiolgruppe.svg|thumb]]
{{main|硫醇}}
硫醇是指醇的[[氧|氧]]被[[硫|硫]]取代或[[羥基|羥基]]的[[氧|氧]]被[[硫|硫]]取代成[[巰基|巰基]]的一種醇類似物。特別的是，
由於具又臭味，常被用於加入石油氣或天然氣等瓦斯內中的警覺劑，用以警示瓦斯泄漏。
{{clear}}

== 硒醇 ==
[[File:Generic-selenol-2D.png|thumb]]
{{main|硒醇}}
硫醇是指醇的[[氧|氧]]被[[硒|硒]]取代的一種醇類似物。最有名的是氨基酸[[硒半胱氨酸|硒半胱氨酸]]。
{{clear}}

== 碲醇 ==
[[File:Generic-tellurol-2D.png|thumb]]
{{main|碲醇}}
硫醇是指醇的[[氧|氧]]被[[碲|碲]]取代的一種醇類似物。
{{clear}}

== 釙醇 ==
[[File:Generic-Polonol-2D.png|thumb]]
硫醇是指醇的[[氧|氧]]被[[釙|釙]]取代的一種醇類似物。雖然還沒有任何一種釙醇被制得，但仍可以推測其性質。<ref>{{cite journal |last1=Weigel |first1=F. |year=1959 |title=Chemie des Poloniums |journal=[[Angewandte_Chemie|Angewandte Chemie]] |volume=71 |pages=289–316 |doi=10.1002/ange.19590710902}}</ref>
{{clear}}

== 氰醇 ==
{{main|羟腈}}
氰醇（cyanoalcohol），即羟腈，[[有机化合物|有机化合物]]的一类，是醇的類似物之一，是[[醛|醛]]或[[酮|酮]]分子中[[羰基|羰基]]发生[[氰化氢|氰化氢]][[加成反应|加成反应]]生成的化合物。

== 系统命名 ==
在英文的[[IUPAC命名法|IUPAC]]系统当中，烷烃链去掉末尾的“e”，加入“ol”，如：“methanol”和“ethanol”，若是[[氧|氧]]被取代類似物則將該元素字尾-ium去掉，加上“ol”再加入字尾，而不用去掉烷烃链末尾的“e”，如：“methaneselenol”，另外，
硫醇是例外，直接加入“thiol”即可，如：“methanethiol”。<ref name=reusch-alcohols>{{cite web | author = William Reusch | work = VirtualText of Organic Chemistry | title = Alcohols | url = http://www.cem.msu.edu/~reusch/VirtualText/alcohol1.htm#alcnom | accessdate = 2007-09-14 | deadurl = yes | archiveurl = https://web.archive.org/web/20070919162404/http://www.cem.msu.edu/~reusch/VirtualText/alcohol1.htm#alcnom | archivedate = 2007-09-19 }}</ref>
其餘則與醇相同。
