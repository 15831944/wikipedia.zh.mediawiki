《'''暗黑殺神'''》（{{lang-en|'''''Diablo'''''}}）是一部2015年加拿大與美國合拍的心理驚悚西部電影，由{{link-en|勞倫斯·羅克|Lawrence Roeck}}執導和編劇。主演是[[史葛·伊士活|史葛·伊士活]]、[[華頓·哥金斯|華頓·哥金斯]]、[[卡蜜拉·貝兒|卡蜜拉·貝兒]]和[[丹尼·葛洛佛|丹尼·葛洛佛]]。<ref>{{Cite web |url = https://www.hollywoodreporter.com/news/scott-eastwoods-diablo-nabbed-by-840589 |title = Scott Eastwood's 'Diablo' Nabbed by Orion Releasing, Momentum |author = Etan Vlessing |date = 2015-11-16 |publisher = The Hollywood Reporter |language = en}}</ref><ref>{{Cite web |url = https://variety.com/2016/film/reviews/diablo-review-scott-eastwood-1201674952/ |title = Film Review: ‘Diablo’ |author = Andrew Barker |date = 2016-01-08 |publisher = Variety |language = en}}</ref>本片是西部片偶像[[奇連·伊士活|奇連·伊士活]]的兒子史葛·伊士活第一部演出的西部片。
