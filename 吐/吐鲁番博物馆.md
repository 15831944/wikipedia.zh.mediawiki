'''吐鲁番博物馆'''，位于[[新疆维吾尔自治区|新疆维吾尔自治区]][[吐鲁番市|吐鲁番市]]木纳尔路1268号，是吐鲁番市属综合性博物馆。

==简介==
1989年，吐鲁番博物馆由[[吐鲁番地区|吐鲁番地区]]投资80万元人民币建成，1990年，正式成立了吐鲁番地区博物馆，是隶属吐鲁番地区文物局的全额拨款[[事业单位|事业单位]]，正科级。占地面积5000平方米，内有陈列展厅4个，约1650平方米，常设展厅包括：出土文物陈列厅、巨犀化石陈列厅、古尸陈列厅。1999年，吐鲁番地区博物馆被定为新疆维吾尔自治区爱国主义教育基地。2008年3月30日，吐鲁番博物馆正式免费开放。2008年6月2日，因吐鲁番博物馆搬迁而闭馆<ref name=tsw>{{Cite web |url = http://www.ts.cn/special/2010-mztjjy/2010-12/03/content_5415742.htm |title = 吐鲁番博物馆 |author =  |date = 2010-12-03 |publisher =天山网  |language =  |accessdate =  |quote =  }}</ref>。

2006年9月，吐鲁番博物馆新馆开工建设<ref name=tsw/>。2009年9月26日，新馆正式对外开放。新馆建筑面积13291.3平方米，展陈面积4126平方米，地上三层，东北角设有一座56米高的观光塔。弧形外墙嵌有反映吐鲁番地区四千年历史的浮雕<ref name=gywm>{{Cite web |url = http://www.tlfbwg.net/gywm/show/27.aspx |title =关于我们  |author =  |date =2014-09-02  |publisher = 吐鲁番博物馆 |language =  |accessdate =  |quote =  }}</ref>。

吐鲁番博物新馆首批对外开放的展厅是“吐鲁番地区通史陈列厅”、“巨犀化石陈厅”。此后又陆续开放了“吐鲁番地区出土文书陈列厅”、“古钱币陈列厅”、“沉睡千年——吐鲁番出土干尸及其随葬遗物展陈列厅”等。吐鲁番博物馆的馆藏规模在整个新疆维吾尔自治区仅次于[[新疆维吾尔自治区博物馆|新疆维吾尔自治区博物馆]]，馆藏文物20325件，其中一级文物79件、二级文物61件、三级文物148件。<ref>{{Cite web |url=http://kydww.sach.gov.cn/cms/wwj/area.do?cmd=areaDetail&c_unit_code=65000021800002 |title=全国可移动文物登录网：吐鲁番博物馆 |access-date=2017-08-22 |archive-url=https://web.archive.org/web/20170822094037/http://kydww.sach.gov.cn/cms/wwj/area.do?cmd=areaDetail&c_unit_code=65000021800002 |archive-date=2017-08-22 |dead-url=yes }}</ref>

2017年，吐鲁番博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。
