{{日本作品|只要長得可愛，即使是變態你也喜歡嗎？|可愛ければ変態でも好きになってくれますか？}}是{{tsl|ja|花間燈}}寫作、sune負責插畫的日本[[輕小說|輕小說]]作品。2017年11月於《[[月刊龍時代|月刊龍時代]]》的2017年12月號開始連載[[漫畫|漫畫]]，CHuN作畫<ref>{{Cite web|url=http://ln-news.com/archives/64051/post-64051/|title=『可愛ければ変態でも好きになってくれますか？』＆『異世界のんびり農家』のコミカライズ連載が本日発売のドラゴンエイジ12月号より同時スタート|work=Ln-news|date=2017-11-09|accessdate=2019-03-26|language=ja}}</ref>。2018年5月25日發行[[廣播劇CD|廣播劇CD]]<ref>{{Cite web|url=http://www.koepota.jp/news/2018/04/26/0105.html|title=天﨑滉平、今村彩夏、小澤亜李ら出演！ドラマCD「可愛ければ変態でも好きになってくれますか？」5月25日発売！|work=Koepota|date=2018-04-26|accessdate=2019-03-26|language=ja}}</ref>。

2019年2月20日決定改編為[[電視動畫|電視動畫]]<ref>{{Cite web|url=https://www.animatetimes.com/news/details.php?id=1550650524|title=『可愛ければ変態でも好きになってくれますか？』がテレビアニメ化！ メインキャラクターのビジュアルと、下野紘さんら出演声優3名が発表に！|work=animate Times|date=2019-02-20|accessdate=2019-03-26|language=ja}}</ref>，2019年3月24日在「AnimeJapan 2019」的活動會場中宣布於7月播出<ref>{{Cite web|url=https://www.animatetimes.com/news/details.php?id=1553386408|title=『可愛ければ変態でも好きになってくれますか？』追加声優に野水伊織さん・本渡楓さん・河本啓佑さん！ 放送時期は2019年夏に決定|work=animate Times|date=2019-03-24|accessdate=2019-03-26|language=ja}}</ref>。

== 故事簡介 ==
桐生慧輝希望交到女朋友。某天他收到了一封裡面夾著內褲的情書，而根據慧輝的推測，有四位女孩是最有可能寄這封情書的對象，當慧輝不斷地去追尋真相時，卻發現他身邊的女孩子有著不為人知的秘密。

=== 主要角色 ===
; {{nihongo|桐生慧輝|桐生 慧輝（きりゅう けいき）|聲：[[天崎滉平|天崎滉平]]<ref name="drama"/>／[[下野紘|下野紘]]<ref name="anime"/>}}
: 作品男主角，書法社社員並擔任圖書委員。是個妹控。對自己身邊的女孩都是變態這件事感到懊惱鬱悶。嚮往與普通的女孩談普通的戀愛。
; {{nihongo|朱鷺原紗雪|朱鷺原 紗雪（ときはら さゆき）|聲：[[今村彩夏|今村彩夏]]<ref name="drama"/>／[[竹達彩奈|竹達彩奈]]<ref name="anime"/>}}
: 慧輝的學姐，書法社社長。長相貌美而受到慧輝傾慕，實際上自己想要成為慧輝的寵物。
; {{nihongo|古賀唯花|古賀 唯花（こが ゆいか）|聲：[[小澤亞李|小澤亞李]]<ref name="drama"/>／[[日高里菜|日高里菜]]<ref name="anime"/>}}
: 慧輝的學妹，金髮碧眼的女孩。是學校的圖書委員，興趣是讀書與繪本。平時表現得楚楚可憐，實際上有施虐癖氣質，希望慧輝成為自己的奴隸。
; {{nihongo|南條真緒|南条 真緒（なんじょう まお）|聲：[[松田颯水|松田颯水]]<ref name="drama"/>／[[野水伊織|野水伊織]]<ref name="anime"/>}}
: 慧輝的同班同學。有在畫以慧輝和翔馬為對象的[[BL_(和製英語)|BL]]同人誌。
; {{nihongo|桐生瑞葉|桐生 瑞葉（きりゅう みずは）|聲：[[桑原由氣|桑原由氣]]<ref name="drama"/>／[[本渡楓|本渡楓]]<ref name="anime"/>}}
: 慧輝的妹妹，擅長家事的可愛女孩。真實身份就是送慧輝情書和內褲的女孩，其實是暴露狂，有嚴重的戀兄情結。和慧輝沒有血緣關係。
; {{nihongo|秋山翔馬|秋山 翔馬（あきやま しょうま）|聲：[[田丸篤志|田丸篤志]]<ref name="drama"/>／[[河本啟佑|河本啟佑]]<ref name="anime"/>}}
: 慧輝的男性友人，所屬社團是網球社，運動萬能。長相帥氣因而受女生歡迎，本人實際上是[[蘿莉控|蘿莉控]]。
