'''活塞男事件'''於2008年8月6日深夜在[[香港|香港]][[九龍|九龍]][[觀塘區|觀塘區]][[藍田_(香港)|藍田]][[碧雲道|碧雲道]][[五桂山_(香港)|五桂山]]山邊的[[藍田公園|藍田公園]]發生，一名中年男子於[[三號強風訊號|三號強風訊號]]生效期間，在藍田公園內的[[仰臥起坐|仰臥起坐]]鐵板運動時，懷疑利用鐵板[[自慰|自慰]]，其[[陽具|陽具]]被緊套在板上的圓孔內，無法自拔而報警求助。除了[[香港傳媒|香港傳媒]]報道消息，亦有多間外國媒體報道事件，事主被[[網民|網民]]起了「活塞男」等等[[外號|外號]]，亦引伸了大量後續的[[惡搞|惡搞]]創作，並獲得[[達爾文獎|達爾文獎]]的榮譽表揚<ref>[http://darwinawards.com/stupid/stupid2008-08.html 2008 At-Risk Survivor: Bench Press] - Darwin Awards</ref>。

== 釋義 ==
{{See|活塞}}

活塞是在[[氣缸|氣缸]]內滑動的中空圓柱，活塞的外徑約等於氣缸的內徑。功能是改變氣缸包圍的容積，或是傳遞氣缸內流體所施加的壓力。在事件中，「活塞」被用作比喻事主利用鐵板自慰時的動作，尤如活塞氣缸內滑動一樣，亦有活活被塞著之意。

== 經過 ==
2008年8月6日晚上，一名42歲辛姓男子<ref name="20080813_NextWeekly">[http://www.hk-fd.net/mag/view.php?mag=next&issue=963&site=11500323 活塞男繼續買鹹碟] {{webarchive|url=https://web.archive.org/web/20080828040843/http://www.hk-fd.net/mag/view.php?mag=next&issue=963&site=11500323 |date=2008-08-28 }}，[[壹週刊|壹週刊]]第963期，2008年8月13日。</ref>冒雨在[[藍田公園|藍田公園]]觀景台位置使用健身設施[[仰臥起坐|仰臥起坐]]時，懷疑將[[陽具|陽具]]困在設施約一吋直徑（約2.5公分）的圓孔內而無法拔出，最終報警求助。

[[救護總區|救護人員]]趕抵現場後為事主注射[[鎮靜劑|鎮靜劑]]後仍然未能夠為事主脫困，幾經擾攘，救護人員最後決定將事主連人帶板（即將鐵板拆除）抬出送院搶救。期間事主對記者的採訪相當抗拒<ref>[[明報|明報]]：[http://hk.news.yahoo.com/article/080807/4/7kwe.html 半夜除褲做運動 中年漢子孫根「被困」]，2008年8月7日</ref><ref>[http://hk.news.yahoo.com/article/080807/3/7kz9.html 星島日報：疑服藥後做掌上壓 下體卡小孔　壯漢連人帶板送院]，2008年8月8日</ref>。

== 事後 ==
* 該男子出院後被[[狗仔隊|狗仔隊]]跟蹤，被[[偷拍|偷拍]]到他前往[[灣仔|灣仔]][[東方188商場|東方188商場]]購買[[色情片|色情片]][[數碼多功能影音光碟|數碼多功能影音光碟]]<ref name="20080813_NextWeekly" />。曾對記者表示他當時之所以被困是因為被債主逼他還債，使用手段以致<ref name="20080813_NextWeekly" />。由於其回應與事情有所出入，令人懷疑其當時是否有所隱瞞。
