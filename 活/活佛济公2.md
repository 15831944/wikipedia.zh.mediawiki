{{engname|《活佛济公2》|The Legend of Crazy Monk 2}} 是2011年7月27日上星于[[中国大陆|中国大陆]]的由[[陈浩民|陈浩民]]主演[[济公|济公]]的[[电视剧|电视剧]]。
部分劇情與1995年台灣版濟公(周明增版)相似。

== 剧情 ==
该剧据清郭小亭著《济公全传》及民间故事改编，由林添一执导，简远信编剧，由《血魔出世》、《贞节牌坊》、《画中仙》、《木马缘》、《双退婚》 、《万应佛堂》、《疯女劫》、《雪女传说》 、《鬼郎君》 等9个故事组成。继第一部后，讲述前世为降龙尊者的济公除恶扬善，拯救百姓苦难，与以乾坤洞主为首的恶势力斗智、斗勇的故事。

== 社会影响 ==
该剧的演员造型比较雷人、搞怪，台词穿越而高效，颠覆了传统佛教神仙在人民心中的形象，被观众誉为该片的亮点。<ref>{{cite web |url= http://ent.sina.com.cn/v/2011-03-08/12443248601.shtml|title=《济公活佛》造型雷人 林江国版哪吒笑翻观众 |accessdate= 2011年|work=|publisher=新浪网 |language = zh-hans}}</ref>[[陈浩民|陈浩民]]解释说：“绝对不是为了恶搞，‘不要迷恋哥，哥只是个传说’、‘你妈喊你回家吃饭’这样的话，我们拿来用只是为了增加亲切感。”<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2010-09-02/11363073640.shtml |title=江苏卫视《活佛济公》再掀三版济公大PK |accessdate= 2010年|work=|publisher=新浪网 |language = zh-hans}}</ref>

== 评价 ==
观众总结了三位演员中不同济公的特色：[[游本昌|游本昌]]版的是“传神”济公，[[周星驰|周星驰]]版的是“无厘头”济公，[[陈浩民|陈浩民]]版的是“潮”济公。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2010-09-12/09373084526.shtml |title= 《活佛济公》收视爆红 郑亦桐冷艳出镜颠覆本色|accessdate= 2010年|work=|publisher=新浪网 |language = zh-hans}}</ref>
