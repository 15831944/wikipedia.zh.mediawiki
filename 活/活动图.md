'''活动图'''（{{lang-en|activity diagram}}）是[[工作流|工作流]]的图形化表示。<ref>{{cite web|url=http://highered.mcgraw-hill.com/sites/0077110005/student_view0/glossary.html|title=Glossary of Key Terms|accessdate=2017-08-02|language=en}}</ref>活动图的主要由活动和动作构成，也可以支持分支选择、迭代、并行。在 [[UML|UML]] 中，活动图主要用于为计算性和组织性过程（即工作流）建模，相关活动之间的数据流也在其覆盖范围之内。<ref>UML Revision Task Force. OMG Unified Modeling Language Specification, Version 1.4 (final draft). February 2001.</ref><ref>J. Rumbaugh, I. Jacobson, and G. Booch. The Unified Modeling Language Reference Manual. Addison-Wesley, 1999.</ref>

活动图由若干不同形状的图形组成，图形之间由箭头连接。
<ref>OMG Unified Modeling Language Superstructure Specification, version 2.1.1. Document
formal/2007-02-05, Object Management Group, February 2007. http://www.omg.org/cgi-bin/doc?formal/2007-02-05.</ref>
其中比较重要的形状有：

* 圆角矩形：表示「动作」；
* 菱形：表示并行活动的开始（分离）和结束（汇合）；
* 矩形：表示「决策」；
* 黑圆：工作流的开始（初始节点）；
* 黑圈套住的黑圆：工作流的结束（结束节点）。

箭头从开始一路指向结束，用来表示活动的先后顺序。
