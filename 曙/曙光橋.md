'''曙光橋'''是位於臺灣花蓮縣[[花蓮市|花蓮市]][[美崙溪|美崙溪]]河口的一座景觀橋梁，其名稱源自站於其上東望[[太平洋|太平洋]]可迎接第一道[[曙光|曙光]]。<ref name="花蓮站歷史沿革"/><ref name="文化資產"/> 該橋起造於[[台灣日本時代|日本時代]]，當時為鐵路路線[[花蓮臨港線|花蓮港支線]]的鐵道橋'''米崙溪橋'''，橫跨美崙溪以聯絡[[花蓮港|花蓮港]]與[[花蓮市|花蓮市]]。<ref name="花蓮縣政府"/><ref name="文化資產"/>在該鐵道支線廢除後，花蓮縣政府將其改建為木棧橋，供行人及自行車通行。<ref name="文化部文化資產局"/>

== 沿革 ==
昔日，[[花蓮臨港線|花蓮臨港線]]自舊花蓮車站出發往[[花蓮港車站|花蓮港車站]]前進，途經聯絡[[花蓮港|花蓮港]]與[[花蓮市|花蓮市]]的美崙溪橋，沿途停靠民利、新村、[[美崙車站|米崙]]及花蓮港站<ref name="文化資產" />。隨著[[北迴鐵路|北迴鐵路]]竣工，花蓮車站西遷，整個鐵路路線的業務也跟著遷移，昔日臨港線鐵道及鐵橋隨即停止使用。<ref name="文化資產" />2000年（民國89年），花蓮縣縣長[[王慶豐|王慶豐]]規劃「南北濱自行車道」，以南濱公園為起點沿舊臨港支線至北濱公園，再至花蓮港北側管制站，全長四公里；在獲[[行政院環保署|行政院環保署]]經費補助及[[臺灣鐵路管理局|臺灣鐵路管理局]]同意使用土地後<ref name="文化資產" />，花蓮縣政府將原本鐵道橋鋪上木棧道，成為提供[[自行車|自行車]]及行人行走的景觀橋<ref name="花蓮縣政府" />。2005年（民國94年）3月經登錄列為花蓮縣歷史建築。<ref name="文化資產" /><ref name="文化部文化資產局" />

== 周遭歷史 ==
美崙溪河口在[[清朝|清朝]]時負有運送官兵到[[豐川|豐川]]保護原住民的功能，因此又稱「陸軍港」<ref name="花蓮站歷史沿革" />；這裡也是花蓮歷史上[[漢人|漢人]]第一次登陸的地點。<ref name="花蓮站歷史沿革" />
