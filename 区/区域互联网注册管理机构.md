'''区域互联网注册机构'''（{{lang|en|'''R'''egional '''I'''nternet '''R'''egistry}}，{{lang|en|'''RIR'''}}），是管理世界上某特定地区[[Internet|Internet]]资源的组织。[[Internet|Internet]]资源包括[[IP地址|IP地址]]（包含[[IPv4|IPv4]]和[[IPv6|IPv6]]）和使用在[[BGP|BGP]][[路由|路由]]中的[[自治系统|自治系统号]]（Autonomous System number）。

现在世界上有五个正在运作的区域互联网注册管理机构：
[[Image:Regional_Internet_Registries_world_map.svg|thumb]]
* {{le|美洲互联网号码注册管理机构|American Registry for Internet Numbers}}（{{lang|en|American Registry for Internet Numbers}}，{{lang|en|ARIN}}）<ref name=arin>[http://www.arin.net American Registry for Internet Numbers]</ref>管理[[北美|北美]]和部分[[加勒比地区|加勒比地区]]事务
* {{le|欧洲IP网络资源协调中心|Réseaux IP Européens Network Coordination Centre}}（{{lang|en|RIPE Network Coordination Centre}}，{{lang|en|RIPE NCC}}）<ref name=ripe>[http://www.ripe.net RIPE Network Coordination Centre]</ref>管理[[欧洲|欧洲]]、[[中东|中东]]和[[中亚|中亚]]地区事务 
* [[APNIC|亚太网络信息中心]]（{{lang|en|Asia-Pacific Network Information Centre}}，{{lang|en|APNIC}}）<ref name=apnic>[http://www.apnic.net Asia-Pacific Network Information Centre]</ref>管理[[亚洲|亚洲]]和[[太平洋|太平洋]]地区事务
*{{le|拉丁美洲及加勒比地区互联网地址注册管理机构|Latin America and Caribbean Network Information Centre}}（{{lang|en|Latin American and Caribbean Internet Address Registry}}，{{lang|en|LACNIC}}）<ref name=lacnic>[http://www.lacnic.net Latin American and Caribbean Internet Address Registry]</ref>管理[[拉丁美洲|拉丁美洲]]和部分[[加勒比地区|加勒比地区]]事务
* {{le|非洲网络信息中心|AFRINIC}}（{{lang|en|African Network Information Centre}}，{{lang|en|AfriNIC}}）<ref name=afrinic>[http://www.afrinic.net African Network Information Centre]</ref>管理[[非洲|非洲]]事务

== 建立RIR的原因 ==
每个连接到符合[[网际协议|IP协议]]网络的设备都需要一个IP地址。IP地址和自治系统号是有限的资源。这意味着它们终有一天会消耗完。这就需要一个有效率和中立的管理机构来管理这些资源以确保能公平地分配IP地址和自治系统号，从而防止资源的[[囤积|囤积]]。

== 区域互联网注册管理机构和IANA之间的关系 ==
[[IANA|互联网地址分配机构]]（IANA）将网络资源委任区域互联网注册管理机构管理，然后相对的，为了未来能进一步地再委任[[ISP|ISP]]和最终用户的资源，区域互联网注册管理机构遵循着他们的区域政策。总体上来说，区域互联网注册管理机构和号码资源组织（NRO）<ref name=nro>[http://www.nro.net/ Number Resource Organization]</ref>组成了一个整体，以代表他们的利益，并采取联合行动，协调它们在全球范围内的活动。NRO与[[ICANN|ICANN]]签订协议创立{{le|地址支持组织|Address Supporting Organization}}（Address Supporting Organisation），从而在ICANN的框架内开展协调全球IP寻址的政策。
