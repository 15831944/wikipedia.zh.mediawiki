'''蝸牛子醬'''（'''''caviar d'escargots'''''，亦稱為 '''蝸牛珍珠'''）<ref name="Fletcher"/><ref name="Winter"/>是一種[[魚子醬|魚子醬]] ，包括新鮮或者處理過的陸生蝸牛卵。

在其天然狀態下，蝸牛卵都是無色的。<ref name="Britton"/>處理過後，蝸牛子醬呈現奶油色、粉白色或者白色，其大小一般在直徑3-4毫米。<ref name="Bertrand"/>一些蝸牛卵甚至可以達到直徑3-6 毫米。<ref name="Winter"/>某些專門生產[[法國蝸牛|蝸牛]]的商業蝸牛養殖場也包括生產蝸牛子醬。<ref name="Winter"/>在2014年九月，某一品牌蝸牛子醬的零售價格甚至達到每50克罐裝超過150歐元。

== 特色 ==
生蝸牛卵的外殼往往是精巧而易碎的。<ref name="Bertrand"/>這些卵經過[[巴斯德消毒法|巴斯德消毒]]以便於[[食品保存|保存]]。<ref name="Bertrand"/>然而，經過巴氏殺菌的蝸牛卵被認為會影響其風味，<ref name="Bertrand"/>因此某些處理後的罐裝蝸牛子醬不採用巴氏殺菌，而改用[[鹽水|鹽水]]作為[[防腐劑|防腐劑]]。<ref name="Bertrand"/>一些生產商甚至使用風味食鹽調製的鹽水來增添產品的風味。

蝸牛子醬的風味有時被描述為會讓人聯想到烤[[蘆筍|蘆筍]]，而在其他情况下也有人認為與烤[[蘑菇|蘑菇]]相似。<ref name="Tepper"/>某些評論認為蝸牛子醬所特有微妙的口味類似於木香調，<ref name="Bertrand"/>其具有強烈的大自然味道，並且相較於魚子醬口感更脆些。<ref name="Soteriou"/>蝸牛子醬亦可搭配其他餐點，像是{{link-en|吐司點|Toast point}}、[[酸奶|酸奶]]和[[香檳|香檳]]食用。<ref name="Bertrand"/>它也可以入湯， 或者製作其他料理。

== 蝸牛養殖場 ==
{{link-en|蝸牛養殖|Heliciculture}}包含了飼養與採收專門供人類消耗用的產品。一些商業蝸牛養殖場收集和處理蝸牛子醬，然後包裝銷售給消費者。<ref name="Bertrand"/><ref name="Soteriou"/>有時蝸牛被飼養於人工控管的溫度、光照和氣候等等條件的溫室，以提高蝸牛的產卵量。<ref name="Bertrand"/><ref name="Soteriou"/>由於蝸牛是[[雌雄同體|雌雄同體]](同時具有[[雄性|雄性]]及[[雌性|雌性]][[性器官|性器官]])，所有蝸牛均可產卵。<ref name="Bertrand"/><ref name="Soteriou"/>蝸牛一般於產卵後會將其卵埋入土中。 一種採集的方法是將產卵的蝸牛放置於鋪了土壤或砂的盒子裡，並且將其收集。<ref name="Soteriou"/>蝸牛卵的產量相較於魚卵而言是相當微薄的，例如，以[[鱘魚|鱘魚]]為例，一個蝸牛每年大約產下四公克的卵，而一條鱘魚每年則能產下最多18公斤的卵。<ref name="Britton"/>

== 市場 ==
1987年八月在[[美國|美國]]，報告指出Brut d'Escargot品牌蝸牛子醬的市場零售價格為{{Convert|40|$/oz|$/kg}}。 <ref name="Britton"/>在當時，其售價相當於{{link-en|歐洲鰉魚子醬| Beluga caviar
}}。<ref name="Britton"/>在2007年十二月，一罐50克De Jaeger品牌的蝸牛子醬，養殖場位於[[法國|法國]][[蘇瓦松|蘇瓦松]]，零售價則是80[[歐元|歐元]]。<ref name="Bertrand"/>在2014年九月，一罐50克的Viennese Snails品牌的蝸牛子醬，產地位於[[奥地利|奥地利]][[維也納|維也納]]，零售價超過150歐元。<ref name="Soteriou"/>一罐50克罐大約相當於兩湯匙。<ref name="Britton"/>某些蝸牛農場直接出售其蝸牛子醬給餐廳。<ref name="Winter"/>
