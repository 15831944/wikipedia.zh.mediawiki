'''弗拉基米尔·尼古拉耶维奇·切洛梅'''（[[俄语|俄语]]：'''{{lang|ru|Влади́мир Никола́евич Челоме́й}}'''，1914年6月30日[[波兰|波兰]][[谢德尔采|谢德尔采]] - 1984年12月8日[[莫斯科|莫斯科]]），前[[苏联|苏联]]力学家、火箭工程师。

切洛梅出生于沙皇俄国的谢德尔采（今属波兰），乌克兰族。在3个月大时，他的家人逃往波尔塔瓦，到了第一次世界大战时又迁往谢德尔采。 当切洛梅12岁时，全家又搬到基辅。1932年，切洛梅被送往基辅理工学院（既后来的基辅航空学院），在那里他凸显了一个优秀学生的特质。 1936年，
他出版了自己的第一本书《向量分析》。在基辅理工学院学习期间，切洛梅旁听了很多高端理论的讲座，包括数学分析，微分方程，数学物理，弹性力学理论，并在基辅大学力学讲座上发表演说。当时在乌克兰苏维埃社会主义共和国科学院工作的杜里奥列维.奇维塔在这时发现了切洛梅的学术潜力。切洛梅对力学和振动理论的兴趣非常浓厚即使在他休息的时候也会去思考这方面的问题。 1937年，切洛梅以优异的成绩从基辅理工学院毕业，并进入力学研究所进行教学和科研工作。

== 二战 ==
从伟大的卫国战争开始时，切洛梅曾任职于位于莫斯科巴拉诺夫中央航空发动机研究所（TsIAM）在莫斯科，他在1942年创造的第一个苏维埃脉动喷气发动机，独立的纳粹德国类似的当代发展。  1944年夏天，人们知道纳粹德国利用V1巡航导弹对英格兰南部发起攻击。在1944年10月19日之后，由苏联国家国防委员会和人民委员航空工业阿列克谢沙胡林的决定，切洛梅被任命为董事兼植物N51构造（先前主任尼古拉波利卡尔波夫前去世后很短的时间）。 切洛梅是设计，制造和测试尽早第一枚苏联巡航导弹。早在1944年12月，这枚导弹，代号为10X，是早期的测试发射佩特利亚科夫P- 8和图波列夫TU - 2型飞机

==切洛梅设计局和学术生涯 == 

[[切洛梅设计局|切洛梅设计局]]和学术生涯之后，他成功的10X，苏联特别设计局在设计试点较少的飞机（OKB- 52）是根据切洛梅的领导。 1955年，切洛梅被任命为切洛梅设计局首席结构工程师，他在那里继续从事巡航导弹的研究工作。  切洛梅继续他的科学研究，从[[鲍曼莫斯科国立技术大学|鲍曼莫斯科国立技术大学]]科学博士学位。在1951年完成专业论文后，他在1952年成为该校教授。  1958年，OKB-52提出了多级洲际弹道导弹的建议。尽管他们的UR- 200型火箭的设计被否决的米哈伊尔 扬格利的[[R-36|R-36]]（[[北约代号|北约代号]]SS - 9悬崖），他们的UR-100的设计被接受。切洛梅的设计局是一般机械零件，工业部为首的谢尔盖阿法纳西耶夫

== 宇宙飞船 ==

1959年，切洛梅被任命为航空设备总工程师。OKB- 52，以及设计洲际导弹，开始工作，宇宙飞船，并在1961年开始研究一个更大功率的洲际弹道导弹[[UR-500|UR-500]]。  1962年，切洛梅年成为[[苏联科学院|苏联科学院]][[力学|力学]]部院士。  切洛梅成为“月球竞赛科罗廖夫的内部竞争对手”。 切洛梅建议，功率强劲的UR- 500用于发射一个小型两人登月飞船，并成功地获得雇用赫鲁晓夫的家人对他的建议表示支持。他还声称，UR- 500可以被用来发射军用空间站。 继[[赫鲁晓夫|赫鲁晓夫]]的卸任，切洛梅的和[[科罗廖夫|科罗廖夫]]的项目合并，但苏联登月计划继续进行。UR- 500（也称为质子）首次发射定于于1967年3月10日举行。 虽然UR-500火箭从没有像切洛梅所希望的用来发送宇航员到月球的，质子已广泛用于发射苏联的卫星，以及所有苏联的空间站，并包括对前三个国际空间站组件的两个模块（[[国际空间站|国际空间站]]）。 如名为Polyot人造地球卫星还设计了切洛梅的OKB。与先前的工艺，甚至切洛梅第一卫星Polyot - 1（1963）和Polyot - 2（1964）能够改变其运行轨道自己。他还领导了卫星研制的质子。在20世纪70年代切洛梅的设计局工作的阿尔马兹[[礼炮2号|礼炮2号]]，[[礼炮3号|礼炮3号]]和[[礼炮5号|礼炮5号]]也成为了礼炮号，和平号轨道站的基础上，与星辰号空间站。为了支持他的阿尔马兹站，切洛梅设计的TKS，作为[[联盟号|联盟号]]的选择。从未坐过的TKS按计划，但衍生工具的[[礼炮7号|礼炮7号]]和[[和平号轨道空间站|和平号轨道空间站]]飞行的模块。1984年切洛梅在[[莫斯科|莫斯科]]去世。
