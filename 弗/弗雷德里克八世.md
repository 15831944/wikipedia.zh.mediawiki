'''弗雷德里克八世'''（'''{{lang|da|Frederik VIII.}}'''，{{bd|1843年|6月3日|1912年|5月14日|catIdx=F}}），[[丹麦君主列表|丹麦国王]]（1906年—1912年在位）。他是克里斯蒂安九世和[[路易丝公主_(黑塞)|路易丝王后]]的大儿子。生于[[哥本哈根|哥本哈根]]。1906年1月克里斯蒂安国王去世他即位。

===婚姻及子嗣===
在1869年7月28日弗雷德里克迎娶瑞典的[[路易莎公主|路易莎公主]]（{{lang|en|Lovisa of Sweden}}，1851年10月31日—1926年3月20日），洛维莎是瑞典国王[[卡尔十五世|卡尔十五世]]的女儿。他们有四个儿子和四个女儿：

# 丹麦王储[[克里斯蒂安十世|克里斯蒂安]] （1870年—1947年），後來的君主克里斯蒂安十世。
# 次子[[哈康七世|卡尔王子]] （1872年—1957年），后来的[[挪威国王|挪威国王]][[哈康七世|哈康七世]]
# [[路易丝公主_(丹麦)|路易丝公主]] （1875年—1906年）
# [[哈拉尔王子_(丹麦)|哈拉尔王子]] （1876年—1949年）
# [[英格公主|英格公主]] （1878年—1958年）
# [[提亞公主|提亞公主]] （1880年—1945年）
# [[古斯塔夫王子|古斯塔夫王子]] （1887年—1944年）
# [[達格瑪公主|達格瑪公主]] （1890年—1961年）
