'''弗拉基米爾·弗基西維奇'''（'''Vladimir Vukićević'''，{{bd|1979年|4月29日|||}}），是一位出生於[[塞爾維亞|塞爾維亞]]的[[美國|美國]][[軟體工程師|軟體工程師]]，曾參與過許多[[開放原始碼|開放原始碼]]專案開發工作。主要知名於開發開放原始碼[[圖形庫|圖形庫]]，包括[[Mozilla|Mozilla]]專案中使用的圖形庫。

==生涯==
[[File:Vlad_Vukicevic.jpg|thumb]]
2006年，弗基西維奇開始利用''[[Canvas_(HTML元素)|canvas]]''的HTML元素來開發OpenGL 3D原型，他稱為Canvas 3D<ref>[http://blog.vlad1.com/2007/11/26/canvas-3d-gl-power-web-style/ "Canvas 3D GL Power, Web Style", ''Vladimir Vukićević — Words''] {{webarchive|url=https://web.archive.org/web/20110717224855/http://blog.vlad1.com/2007/11/26/canvas-3d-gl-power-web-style/ |date=2011-07-17 }}</ref>。這項工作導致[[Khronos_Group|Khronos Group]]在內部成立了一個名為“Accelerated 3D on the Web”的工作小組，以便為[[OpenGL|OpenGL]]和OpenGL ES 2.0功能建立一個免授權費的標準API<ref>"Khronos launches initiative for free standard for accelerated 3D on Web", ''Khronos Press Release,'' {{cite web |url=http://www.khronos.org/news/press/releases/khronos-launches-initiative-for-free-standard-for-accelerated-3d-on-web/ |title=存档副本 |accessdate=2011-03-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20110307153836/http://www.khronos.org/news/press/releases/khronos-launches-initiative-for-free-standard-for-accelerated-3d-on-web/ |archivedate=2011-03-07 }}</ref>，其中產生了[[WebGL|WebGL]]規範。

2008年，弗基西維奇公開批評Apple在[[Safari|Safari網頁瀏覽器]]中使用專有[[API|API]]來提高性能，使[[Firefox|Firefox]]等第三方應用程序無法解決類似的性能問題<ref>{{cite news|last=Broersma|first=Matthew|title=Apple slammed over programming secrets|url=http://www.infoworld.com/d/mobilize/apple-slammed-over-programming-secrets-860|accessdate=2011-03-04|newspaper=Techworld.com|date=2008-03-03}}</ref>。

弗基西維奇對[[Firefox_for_Android|Firefox Mobile／Fennec]]做出了許多重要貢獻，包括[[TraceMonkey|TraceMonkey]] [[JavaScript|JavaScript]]引擎的[[ARM|ARM]]後端<ref>{{cite news|last=Gruener|first=Wolfgang|title=Firefox mobile web browser scheduled for 2010|url=http://www.tgdaily.com/software-features/39398-firefox-mobile-web-browser-scheduled-for-2010|accessdate=2011-03-04|newspaper=TG Daily|date=2008-09-19}}</ref>和Fennec的[[Android|Android]]初始端口<ref>{{cite news|last=Claburn|first=Thomas|title=Mozilla Tests Mobile Firefox For Android|url=http://www.informationweek.com/news/government/showArticle.jhtml?articleID=224700134|accessdate=2011-03-04|newspaper=InformationWeek|date=2010-04-29}}</ref>。

弗基西維奇還合著（與Stuart Parmenter）2004年的[[APNG|APNG]]規範。

弗基西維奇擔任Khronos WebGL工作組的主席<ref>{{cite web|title=Khronos Releases Final WebGL 1.0 Specification|url=http://www.khronos.org/news/press/releases/khronos-releases-final-webgl-1.0-specification|publisher=Khronos Group|accessdate=2011-03-04|deadurl=yes|archiveurl=https://web.archive.org/web/20110306042639/http://www.khronos.org/news/press/releases/khronos-releases-final-webgl-1.0-specification|archivedate=2011-03-06}}</ref>，並在2017年之前擔任[[Mozilla|Mozilla]]工程總監<ref>{{cite web|last=Vukićević|first=Vladimir|title=Games Meeting Cancelled|url=https://groups.google.com/forum/?fromgroups=#!topic/mozilla.community.games/QPZKiBkFAb0/|accessdate=2012-07-26}}</ref>。

他現在是Unity Technologies新興技術高級軟件工程師<ref>{{Cite web|url=https://www.linkedin.com/in/vladimirvukicevic/|title=LinkedIn Profile|last=Vukićević|first=Vladimir|date=|website=LinkedIn|archive-url=|archive-date=|dead-url=|access-date=}}</ref>。
