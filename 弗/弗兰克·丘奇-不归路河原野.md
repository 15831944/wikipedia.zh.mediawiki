'''弗兰克·丘奇-不归路河原野'''（{{lang-en|Frank Church—River of No Return Wilderness}}<ref name="usda">{{cite web |title=Frank Church River of No Return Wilderness |url=https://www.fs.usda.gov/detail/scnf/specialplaces/?cid=stelprdb5360033 |website=美国农业部 |accessdate=2018-09-03}}</ref>）是美国[[爱达荷州|爱达荷州]]一处受保护的{{tsl|en|wilderness area|荒原地区}}，1980年间由[[美国国会|美国国会]]设立，1984年更名为目前的名称，以此纪念[[美国参议院|美国参议院]]议员[[弗兰克·丘奇|弗兰克·丘奇]]。该荒原是美国除阿拉斯加州外最大的一片荒原，<ref>{{cite web |title=美国各州的昵称（43）：爱达荷——土豆州、珍宝州及其他 {{!}} ShareAmerica |url=https://share.america.gov/zh-hans/the-gem-state/ |website=ShareAmerica |accessdate=2018-09-03 |language=zh-CN |date=2017-11-09}}</ref>面积{{convert|2,366,757|acre|km2}}。<ref name="usda" />
