'''弗朗茨二世'''([[德语|德语]]：'''{{lang|de|Franz II}}'''，{{bd|1768年|2月12日|1835年|3月2日|catIdx=F}})，[[神圣罗马帝国|神圣罗马帝国]]的末代[[神圣罗马皇帝|皇帝]](1792年-1806年在位)，[[奥地利帝国|奥地利帝国]]的第一位[[奥地利皇帝|皇帝]](1804年-1835年在位，称'''弗朗茨一世'''，[[德语|德语]]：'''{{lang|de|Franz I}}''')。神圣罗马帝国皇帝[[利奥波德二世_(神聖羅馬帝國)|利奧波德二世]]与皇后西班牙的玛利亚·路易莎之子。

弗朗茨二世就任神圣罗马帝国皇帝時，德意志城邦割據的情況已經十分嚴重，神圣罗马帝国皇帝早已失去實權。[[法國大革命|法國大革命]]和[[拿破崙|拿破崙]]的崛起加劇了德意志諸城邦的分化，帝國的前景並不樂觀。[[第二次反法同盟|第二次反法同盟]]結束後，神聖羅馬帝國喪失了全部意大利領地，瑞士更成立[[赫爾維蒂共和國|赫爾維蒂共和國]]，掠奪了奧地利大公國在阿爾卑斯山中部的領地。为了对抗拿破仑即将称帝的野心弗朗茨二世皇帝于1804年8月10日成立奥地利帝国，自称奥地利皇帝。[[第三次反法同盟|第三次反法同盟]]後，弗郎茨二世在[[奧斯特里茲戰役|奧斯特里茲戰役]]慘敗，神聖羅馬帝國首都[[維也納|維也納]]被法軍三次攻陷，奧地利軍被全殲，雙方簽訂[[普雷斯堡和約|普雷斯堡和約]]，萊茵邦聯創立，1806年8月6日弗郎茨二世被迫放弃神圣罗马帝国皇帝的封号，只保留奥地利皇帝頭銜。至此，历经了844年的神圣罗马帝国彻底宣告终结。弗朗茨二世再度被迫臣從於拿破崙和沙皇[[亞歷山大一世(俄國)|亞歷山大一世]]的[[提爾西特和約|提爾西特同盟]]中。不可干涉[[第七次俄土戰爭|第七次俄土戰爭]]。

弗朗茨二世發起了四次反[[拿破仑|拿破仑]]的神聖同盟，都被拿破崙打敗，而且大大削弱了奧地利的國力，間接導致日後[[普魯士|普魯士]]的崛起。1806年，拿破仑強行解散了神圣罗马帝国，歷八百年的神圣罗马帝国滅亡，奧地利成為[[萊茵邦聯|萊茵邦聯]]的一部分。弗朗茨二世被迫以和亲的方式，把女儿[[瑪麗·路易莎|瑪麗·路易莎]]嫁给拿破仑。從此[[哈布斯堡王朝|哈布斯堡王朝]]的勢力大幅度衰落，無法恢復對全歐洲的影響力。

弗朗茨二世在1814年拿破崙倒台後，立即主持召開了[[維也納會議|維也納會議]]，在首相[[梅特涅|梅特涅]]領導下成立維也納體系和[[神聖同盟|神聖同盟]]，企圖恢復[[歐洲|歐洲]]的舊「秩序」，以抵消[[自由主義|自由主義]]的影響，但被迫制定憲法，承認部分改革。在[[1848年革命|1848年革命]]後，維也納體系被推翻，弗朗茨二世用盡一生精力維繫的奧地利帝國崩潰，匈牙利、波希米亞、巴爾幹各國民族主義紛起。

法国大革命中被断头的[[玛丽·安托瓦内特|玛丽·安托瓦内特]]王后是弗朗茨二世的姑姑。

== 家庭 ==
弗朗茨一共结婚四次， 十三名子女皆由第一和第二位妻子所出。 当中只有七名活到成年。

* 第一位妻子: [[符腾堡|符腾堡]]公国的伊丽莎白郡主〔1767-1790〕。产下一女后逝世。终年23岁。
# 露多维卡(1790年2月18日-1791年6月24日)。周岁夭折。
* 第二位妻子: 姑表妹[[两西西里王国|两西西里王国]]的玛丽亚·特丽莎公主（1772-1807）。育有十二名子女。享年35岁。
# [[瑪麗·路易莎|瑪麗·路易莎]](1791年12月7日-1847年12月17日)。1814年-1847年间担任帕尔马女公爵。享年56岁。她一共结过三次婚：1810年4月1日嫁给法国皇帝拿破仑一世(1769年8月15日-1821年5月5日)；1821年8月8日或9月7日下嫁奥地利的亚当·阿达尔贝·冯·奈贝格伯爵(1775年4月8日-1829年2月22日)；1834年2月17日下嫁法国的邦贝勒伯爵查理·何内 (1785年11月6日-1856年5月30日)
# [[斐迪南一世_(奧地利)|斐迪南一世]](1793年4月19日-1875年6月29日)。1835年3月2日继位为奥地利皇帝。娶撒丁王国萨伏依王朝的玛丽亚·安娜公主(1803年9月19日-1884年5月4日)。无嗣。享年82岁。
# 玛丽·卡罗莲(1794年6月8日-1795年3月16日)。未满1岁夭折。
# 卡罗莲(1795年12月9日-1799年6月30日)。未满4岁夭折。
# 玛丽·利奥波丁娜 (1797年1月22日-1826年12月11日)。1817年嫁给巴西皇帝[[佩德羅一世_(巴西)|佩德罗一世]]兼葡萄牙国王佩德罗四世(1798年10月12日-1834年9月24日)。死于流产。享年35岁。
# 玛丽.克雷门丁(1798年3月1日-1881年9月3日)。1816年嫁给了舅舅两西西里王国的萨勒诺亲王利奥波多(1790年7月2日-1851年3月10日)。享年83岁。
# 约瑟夫(1799年4月9日-1807年6月30日)。8岁夭折。
# 玛丽.卡罗莲(1801年4月8日-1832年5月22日)。1819年嫁给[[萨克森|萨克森]]国王斐特烈·奥古斯特二世(1797年5月18日-1854年8月9日)。享年31岁。
# [[弗兰茨·卡尔大公|弗兰茨·卡尔大公]](1802年12月7日-1878年3月8日)。1824年娶巴伐利亚王国的苏菲公主(1805年1月27日-1872年5月28日)。1848年宣誓放弃继承权。为奥匈帝国开国皇帝[[弗朗茨·约瑟夫一世|弗朗茨·约瑟夫一世]]之父。享年75岁。
# 玛丽·安妮(1804年6月8日-1858年12月28日)。曾患有严重痴呆和面部畸形。享年54岁。
# 约翰(1805年8月29或30日-1809年2月19日)。未满4岁夭折。
# 阿玛丽(1806年4月6日-1807年4月9日)。周岁夭折。
* 第三位妻子: 堂妹玛丽亚·露多维卡（1787-1816）。无嗣。死于肺痨，享年29岁。
* 第四位妻子: [[巴伐利亚|巴伐利亚]]王国的夏洛特公主（1792-1873）。无嗣。享年81岁。
