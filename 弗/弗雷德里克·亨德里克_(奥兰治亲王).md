'''弗雷德里克·亨德里克'''（{{lang-nl|Friderik Hendrik}}，{{bd|1584年|1月29日|1647年|3月14日|Hendrik}}），[[尼德兰|尼德兰]]政治家和军事统帅，[[联省共和国|联省共和国]][[荷蘭省督|執政]]（1625年4月25日~1647年3月14日在位）。他的封号包括[[荷蘭君主列表|奧蘭治親王]]和[[拿骚|拿骚]]伯爵。

== [[荷蘭省督|執政]]生涯 ==
弗雷德里克·亨德里克是奥兰治亲王[[威廉一世_(奥兰治)|沉默者威廉]]之幼子，母親是法國[[胡格諾|胡格諾]]貴族[[路薏絲·德·科利尼|路薏絲·德·科利尼]]（1555-1620年），她是法國1570年代初的新教權臣——[[科利尼|科利尼]]之女（科利尼死於1572年的[[聖巴托羅繆大屠殺|聖巴托羅繆大屠殺]]）。

弗雷德里克·亨德里克的年轻时代，在兄长联省[[荷蘭省督|執政]][[拿騷的毛里茨|拿騷的毛里茨]]的军队里接受军事训练。他像对待父亲一样尊敬这位兄长。1625年沒有合法子嗣的莫里斯去世后，弗雷德里克·亨德里克繼承奧蘭治親王，並出任包括[[荷兰省|荷兰省]]、[[乌得勒支省|乌得勒支省]]、[[西蘭省|西蘭省]]、[[奥费赖塞尔省|奥费赖塞尔省]]和[[格尔德恩省|格尔德恩省]]的[[联省共和国|联省共和国]][[荷蘭省督|執政]]（相当于继承了莫里斯的职务）。在其统治后期，[[格罗宁根|格罗宁根]]与[[德伦特|德伦特]]两省也加入了这个行列（1640年）。

他擁有兄長毛里茨所缺乏的智慧與心機，一方面調解兄長遺留的教派對立問題，獲得兩方的一致擁戴而廣收人心；另一方面牢牢掌握共和國的政策，特別是建立與[[法國|法國]]的同盟，以便共同征服西屬尼德蘭（[[南尼德蘭|南尼德蘭]]）。

== [[三十年戰爭|三十年戰爭]] ==
弗雷德里克·亨德里克具有杰出的军事才能，他在与[[西班牙|西班牙]]的决定性斗争中为尼德兰作出了巨大贡献。他于1625年开始领导全部联省共和国军队，投入[[三十年戰爭|三十年戰爭]]。这支军队先后[[赫龍洛之圍_(1627年)|攻陷了西班牙统治的赫龍洛]]（1627年）、[[斯海尔托亨博斯|斯海尔托亨博斯]]（1629年）、[[马斯特里赫特|马斯特里赫特]]（1632年）、[[布雷达|布雷达]]（1637年）、[[根特|根特]]（1644年）、[[胡斯特|胡斯特]]（1645年）；为了迫使西班牙让步，弗雷德里克·亨德里克于1635年与[[法国|法国]]结成同盟。他领导准备与西班牙簽署最后和约的工作。由于弗雷德里克·亨德里克的努力，尼德兰最终实现了与西班牙缔结一份给予其长久为之奋斗的权利的条约，即[[尼德兰-西班牙和约|尼德兰-西班牙和约]]（1648年1月），結束了[[八十年戰爭|八十年戰爭]]，荷蘭正式獨立（[[三十年戰爭|三十年戰爭]]也在同年結束）。

在弗雷德里克·亨德里克的执政时期，為荷蘭贏得富強地位，尼德兰的国力和影响力达到了顶峰。1628年他派出船長[[皮特·彼得松·海因|皮特·彼得松·海因]]（Piet Pieterszoon Hein）捕獲價值數千萬盾的西班牙寶藏船，使得尼德蘭的經濟實力逐漸超越[[西班牙帝國|西班牙帝國]]；1639年，西班牙利用英王[[查理一世|查理一世]]的內部危機（即將爆發[[英國內戰|英國內戰]]），使用英國的港口作攻荷基地，組織「第二次[[無敵艦隊|無敵艦隊]]」，進攻荷蘭。結果西班牙艦隊在[[唐斯戰役|唐斯戰役]]中被尼德蘭海軍擊敗，西班牙的[[日不落帝國|日不落帝國]]開始崩解（1640年[[葡萄牙|葡萄牙]]因此脫離西班牙獨立）。可以說1639年後，弗雷德里克·亨德里克和盟友法王[[路易十三|路易十三]]並駕齊驅，成為歐洲最強大富有的國家領袖；他也和法國首相[[黎塞留|黎塞留]]，並列為全歐最有權力的兩位政治家。

== 提升家族地位 ==
弗雷德里克·亨德里克在国内企图仿效其他[[欧洲|欧洲]]国家的君主专制制度，这尤其表现在他于1631年通過继承法案，使[[联省共和国|联省共和国]]的[[荷蘭省督|執政]]一职由[[荷蘭君主列表|奧蘭治家族]]世袭。

他也利用英王[[查理一世|查理一世]]的[[英國內戰|內部危機]]，在1641年安排[[幼子威廉|幼子威廉]]與查理一世之女{{link-en|玛丽公主|Mary, Princess Royal and Princess of Orange}}（Mary Stuart）聯姻。這次重要的王朝之間的聯姻將奧蘭治家族的利益與英國王室——首先是[[斯圖亞特王室|斯圖亞特王室]]，然後是[[漢諾威王室|漢諾威王室]]——緊密地聯繫在一起。[[荷蘭君主列表|奧蘭治家族]]的地位由於這次聯姻而提高，一大票準王公的貴族侍臣，聚集在腓特烈·亨利的周圍。

== 對立浮現 ==
{{main|荷蘭省督}}
弗雷德里克·亨德里克雖然擅長與[[荷蘭省|荷蘭省]]商人協調而未與之衝突，但是在1640年代，[[荷蘭省|荷蘭省]]的議會派以外力減輕為理由，開始恢復對共和國政策上的主導作用，弗雷德里克·亨德里克逐漸因折衝斡旋而身心疲憊。1647年亨德里克過世，其子[[威廉二世_(奥兰治)|威廉二世]]（1626-1650年）繼為[[荷蘭省督|執政]]後，就與荷蘭省的議會派商人發生激烈衝突。懷恨在心的荷省商人，趁著1650年威廉二世驟逝的良機，取消了五個省的[[荷蘭省督|執政]]之位，廢除1631年的繼承法案，奧蘭治家族一度中衰。後來到了1672年，才因為突發的[[災難年|災難年]]事件，使亨利的孫子[[威廉三世|威廉三世]]重登[[荷蘭省督|執政]]，威廉三世後來更在1688年西征英國，發動[[光榮革命|光榮革命]]而成為英王（1689-1702年在位）兼荷蘭[[荷蘭省督|執政]]。
