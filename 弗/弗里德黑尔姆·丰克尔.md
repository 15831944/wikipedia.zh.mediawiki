'''弗里德黑尔姆·丰克尔'''（{{lang-de|Friedhelm Funkel}}，{{bd|1953年|12月10日|catIdx=Funkel, Friedhelm}}）是一位[[德国|德国]]前[[足球运动员|足球运动员]]及现任[[足球教练|足球教练]]，自2016年3月起执教于当时仍身处[[德乙联赛|德乙联赛]]的俱乐部[[福尔图娜杜塞尔多夫|福尔图娜杜塞尔多夫]]，并于[[2017年至2018年德国足球乙级联赛|2017-18赛季]]率队成功升入[[德甲|德甲]]。

==球员生涯==
在完成批发贸易从业人员的职业培训后，丰克尔自1973年起在[[乌丁根05|乌丁根05]]开始了自己的职业生涯。从1980年至1983年间，这位拥有出色进球能力的中场曾在[[凯泽斯劳滕足球俱乐部|凯泽斯劳滕]]效力，然后又返回[[克雷费尔德|克雷费尔德]]。他作为球员所取得的最大成就是代表乌丁根05在{{le|1985年德国足协杯决赛|1985 DFB-Pokal Final}}中以2比1击败[[拜仁慕尼黑|拜仁慕尼黑]]，从而捧得[[德国足协杯|德国足协杯]]。经过320场[[德甲|德甲比赛]]并射入83球后，丰克尔于1990年在乌丁根结束了自己的球员生涯。其代表乌丁根参加的254场德甲比赛和59个入球至今仍是该俱乐部的最高纪录。此外，他也曾于[[1974年至1975年德国足球乙级联赛|1974-75赛季]]和1976-1979赛季期间代表乌丁根参加过152场[[德乙联赛|德乙联赛]]并射入66球。<ref>{{cite web|url=http://www.kicker.de/news/fussball/bundesliga/vereine/1-bundesliga/1989-90/kfc-uerdingen-05-28/10824/spieler_friedhelm-funkel.html|title=Funkel, Friedhelm – Spielerstatistik|publisher=[[踢球者|kicker]]|accessdate=2011-04-02}}</ref>

从1981年3月31日至9月22日，丰克尔曾代表{{ld|德国国家足球B队|Deutsche B-Fußballnationalmannschaft|德国B队}}出场4次，并于5月21日在[[不来梅|不来梅]]以3比0战胜[[爱尔兰国家足球队|爱尔兰]]的比赛中射入自己唯一的国际比赛入球。<ref>Hohensee/Huber/Matheja: ''kicker Almanach 2008'', Copress Sport, München 2007, ISBN 9783767909052</ref>

==执教生涯==
在球员生涯末期，丰克尔便以球员兼教练身份主持家乡球队{{ld|诺伊斯草地体育俱乐部|VfR Neuss|诺伊斯}}的日常训练。至[[1990年至1991年德国足球甲级联赛|1990-91赛季]]末，他在德甲尚余两轮比赛的情况下正式出任乌丁根05主教练。<ref>[http://www.tagesspiegel.de/sport/art272,2915295 18 Jahre Profitrainer – eine Chronik Von Neuss bis Frankfurt am Main] bei tagesspiegel.de</ref><ref>[http://www.transfermarkt.de/vfr-neuss/platzierungen/verein/975 Trainerhistorie bei Neuss]</ref>随着这支克雷费尔德俱乐部先后于1991年、1993年和1996年降入德乙，他都能两次率队立即重返德甲（1992年、1994年）。而在经历了1996年的降级之后，丰克尔转投德乙俱乐部[[杜伊斯堡足球俱乐部|杜伊斯堡]]<ref name="MSV Duisburg - Trainerhistorie">{{cite web |title=MSV Duisburg |url=http://www.kicker.de/news/fussball/2bundesliga/vereine/2-bundesliga/2018-19/msv-duisburg-1/trainer.html |website=kicker.de |publisher=kicker |accessdate=2018-07-18 }}</ref>。

1996年4月8日，丰克尔作为{{ld|汉斯·邦加茨|Hans Bongartz}}的继任者，在杜伊斯堡展开执教工作。随后不久，他便率队成功升入德甲，并连续三个赛季保级成功。1998年，他的球队甚至进入了{{le|1998年德国足协杯决赛|1998 DFB-Pokal Final|德国足协杯决赛}}，却以1比2不敌拜仁慕尼黑。杜伊斯堡于[[1999年至2000年德國足球甲級聯賽|1999-2000赛季]]的表现则不尽如人意；当丰克尔的球队跌落积分榜末席，他于2000年3月24日离任。]<ref name="MSV Duisburg - Trainerhistorie"/>

2000年9月19日，丰克尔接掌了当时排名尚处于降级区的德甲球队[[汉莎罗斯托克|汉莎罗斯托克]]。在他的管理下，罗斯托克在[[2000年至2001年德国足球甲级联赛|当季德甲]]提前三轮保级成功。至[[2001年至2002年德国足球甲级联赛|2001-02赛季]]，其胞弟{{ld|沃尔夫冈·丰克尔|Wolfgang Funkel}}亦被聘为球队的助理教练，但在经历了前半程的低迷表现后，两兄弟于2001年12月1日双双遭到解雇。<ref name="Hansa Rostock - Trainerhistorie">{{cite web |title=Hansa Rostock |url=http://www.kicker.de/news/fussball/3liga/vereine/3-liga/2018-19/hansa-rostock-3/trainer.html |website=kicker.de |publisher=kicker |accessdate=2018-07-18 }}</ref>

2002年2月14日，丰克尔出任德甲球队[[科隆足球俱乐部|科隆]]的主教练。尽管战绩良好（主场保持不败），他却无法阻止科隆历史上第二次降入德乙。但他仍能够率队在[[2002年至2003年德國足球乙級聯賽|2002-03赛季]]立即重返德甲。此后由于表现不佳，丰克尔2003年10月30日辞任科隆主教练。<ref name="1. FC Köln - Trainerhistorie">{{cite web|title=1. FC Köln|url=http://www.kicker.de/news/fussball/bundesliga/vereine/1-bundesliga/2014-15/1-fc-koeln-16/trainer.html|publisher=kicker|accessdate=2015-02-26}}</ref>

丰克尔于[[2004年至2005年德國足球乙級聯賽|2004-05赛季]]初期接过德乙球队[[和睦法兰克福|和睦法兰克福]]的教鞭；并成功率队升入德甲。[[2005年至2006年德国足球甲级联赛|随后一个赛季]]，法兰克福在德甲以第14名完赛，同时还进军了{{le|2006年德国足协杯决赛|2006 DFB-Pokal Final}}，但以0比1不敌拜仁慕尼黑。<ref>{{cite web|url=https://www.dfb.de/dfb-pokal/spieltag/?spieledb_path=%2Fmatches%2F559718|title=DFB-Pokal, 2005/2006, Finale |publisher=DFB|date=2006-04-29|accessdate=2018-08-27}}</ref>这是丰克尔自1998年（率杜伊斯堡）以来作为主教练参加的第二场足协杯决赛，当时的对手同样为拜仁慕尼黑。由于拜仁直接获得了[[2006–07年欧洲冠军联赛|欧洲冠军联赛]]的参赛权，法兰克福得以以杯赛亚军身份参加[[2006–07年欧洲联盟杯|2006-07赛季欧洲联盟杯]]。尽管表现不错，但他们未能从分组赛中突围。法兰克福在[[2006年至2007年德国足球甲级联赛|2006-07赛季德甲]]再度以第14名完赛，并闯入了德国足协杯的半决赛。[[2007年至2008年德国足球甲级联赛|2007-08赛季]]则以第9名完赛，使之成为丰克尔在执教生涯中最成功的排位。由于在[[2008年至2009年德国足球甲级联赛|2008-09赛季]]下半程的表现不佳，丰克尔于赛季末受到了媒体的广泛批评，而球迷也在主场比赛中发出将其解雇的呼声，丰克尔遂向俱乐部申请提前终止原本应于2010年6月30日届满的合同。俱乐部同意了他的请求。<ref>{{Cite web |url=http://www.eintracht.de/aktuell/27207/ |title=存档副本 |access-date=2018-08-27 |archive-url=https://web.archive.org/web/20140408220612/http://www.eintracht.de/aktuell/27207/ |archive-date=2014-04-08 |dead-url=yes }}</ref>凭借五个完整的赛季任期，芬克尔与{{ld|埃里希·里贝克|Erich Ribbeck}}并列成为法兰克福队史上任教时间最长的主教练<ref name="Eintracht Frankfurt - Trainerhistorie">{{cite web|title=Eintracht Frankfurt|url=http://www.kicker.de/news/fussball/bundesliga/vereine/1-bundesliga/2013-14/eintracht-frankfurt-32/trainer.html|publisher=kicker|accessdate=2014-01-16}}</ref>。

2009年10月3日，丰克尔成为[[柏林赫塔|柏林赫塔]]的主教练，他是接替了不久前刚被解职的[[吕西安·法夫尔|吕西安·法夫尔]]<ref>[https://www.welt.de/sport/fussball/article4716430/Friedhelm-Funkel-wird-neuer-Hertha-Trainer.html ''Friedhelm Funkel wird neuer Hertha-Trainer'']</ref>。至2010年5月，排名积分垫底而降级的俱乐部宣布<ref>[http://www.faz.net/aktuell/sport/fussball/bundesliga/1-1-in-leverkusen-hertha-bsc-berlin-steigt-aus-der-bundesliga-ab-1985314.html ''Hertha BSC steigt aus der Bundesliga ab'']</ref>，它们将不会延长与丰克尔在赛季末到期的合同<ref>{{Cite web |url=http://www.herthabsc.de/index.php?id=413&tx_ttnews%5Btt_news%5D=7104&cHash=6bfd41f474 |title=存档副本 |access-date=2018-08-27 |archive-url=https://web.archive.org/web/20100514105507/http://www.herthabsc.de/index.php?id=413&tx_ttnews%5Btt_news%5D=7104&cHash=6bfd41f474 |archive-date=2010-05-14 |dead-url=yes }}</ref>。

至[[2010年至2011年德国足球乙级联赛|2010-11赛季]]，丰克尔又接过同样从德甲降级的球队[[波鸿足球俱乐部|波鸿]]的教鞭<ref>{{Cite web |url=http://www.sport1.de/de/fussball/fussball_bundesliga2/newspage_240088.html |title=存档副本 |access-date=2018-08-27 |archive-url=https://web.archive.org/web/20100524142454/http://www.sport1.de/de/fussball/fussball_bundesliga2/newspage_240088.html |archive-date=2010-05-24 |dead-url=yes }}</ref>。波鸿在他的率领下以第三名完赛，并获得了参加[[2010年至2011年德國足球甲級聯賽#升班／降班附加賽|升级附加赛]]的资格，却以1比2的总比分不敌[[普鲁士门兴格拉德巴赫|普鲁士门兴格拉德巴赫]]。丰克尔于2011年9月14日在[[2011年至2012年德国足球乙级联赛|赛季仍进行]]期间辞职。当时波鸿连续4场比赛失利，在七轮过后位居积分榜第17名。<ref>{{cite web|url=http://www.goal.com/de/news/831/2-liga/2011/09/14/2664468/vfl-bochum-beurlaubt-friedhelm-funkel |publisher=goal.com|title=''VfL Bochum beurlaubt Friedhelm Funkel''|date=2011-09-14|accessdate=2018-08-27}}</ref>在离任仅五天后，丰克尔便于2011年9月19日获得新工作，接任[[阿勒曼尼亚亚琛|阿勒曼尼亚亚琛]]的主教练并签订了一份期至2013年的合同<ref>[http://www.alemannia-aachen.de/aktuelles/nachrichten/details/Geballte-Erfahrung-auf-der-Trainerbank-21363m/ ''Friedhelm Funkel neuer Chefcoach der Alemannia – Vertrag bis 2013'']</ref>。然而，在亚琛经历了五连败且身处直接降级区后，丰克尔于2012年4月1日遭到解雇<ref>[http://www.alemannia-aachen.de/aktuelles/nachrichten/details/Funkel-beurlaubt-Aussem-uebernimmt-21541c/ ''Funkel beurlaubt, Aussem übernimmt'']</ref>。

截至2011年5月，丰克尔作为球员和教练共参加了超过1100场德甲和德乙比赛，成为德国职业联赛参赛场次最多的纪录保持者。<ref>{{cite web|url=http://www.wz-newsline.de/home/sport/fussball/funkel-wir-sind-aussenseiter-1.660333 |publisher=wz-newsline.de|title=''Funkel: „Wir sind Außenseiter“''|date=2011-05-15|accessdate=2018-08-27}}</ref>

从2013年9月起，丰克尔出任[[慕尼黑1860|慕尼黑1860]]主教练<ref>[http://www.spiegel.de/sport/fussball/1860-muenchen-vor-funkel-verpflichtung-a-920960.html ''2. Fußball-Bundesliga: 1860 München bestätigt Funkel als neuen Trainer'']</ref>。尽管俱乐部和丰克尔于2014年3月便宣布将在赛季末分手<ref>{{Cite web |url=http://www.tsv1860.de/aktuell/news/friedhelm-funkel-und-die-loewen-gehen-getrennte-wege |title=存档副本 |access-date=2018-08-27 |archive-url=https://web.archive.org/web/20140407091037/http://www.tsv1860.de/aktuell/news/friedhelm-funkel-und-die-loewen-gehen-getrennte-wege |archive-date=2014-04-07 |dead-url=yes }}</ref>，丰克尔还是于4月初便在主场以0比3不敌[[卡尔斯鲁厄体育俱乐部|卡尔斯鲁厄]]后便被提前解雇。他的继任者是{{ld|马库斯·冯·阿伦|Markus von Ahlen}}。<ref>{{Cite web |url=http://www.tsv1860.de/aktuell/news/1860-stellt-cheftrainer-friedhelm-funkel-frei |title=存档副本 |access-date=2018-08-27 |archive-url=https://web.archive.org/web/20140407094614/http://www.tsv1860.de/aktuell/news/1860-stellt-cheftrainer-friedhelm-funkel-frei |archive-date=2014-04-07 |dead-url=yes }}</ref>

自2016年3月中旬起，丰克尔接替{{ld|马库斯·库尔茨|Marco Kurz}}担当德乙俱乐部[[福尔图娜杜塞尔多夫|福尔图娜杜塞尔多夫]]主教练<ref>[http://www.f95.de/aktuell/news/profis/detail/18889-friedhelm-funkel-neuer-cheftrainer/c5a740c4326426b9f9a3259fd5176c54/ ''Friedhelm Funkel neuer Cheftrainer''] auf f95.de vom 14. März 2016, abgerufen am 14. März 2016</ref>。他于2018年4月28日成功率队升入德甲<ref>{{cite web|url=http://sports.163.com/18/0428/21/DGGS55NM00058781.html|title=杜塞时隔5年重返德甲!坐拥两日本国脚+前拜仁门神|publisher=网易体育|date=2018-04-28|accessdate=2018-08-27}}</ref>。
