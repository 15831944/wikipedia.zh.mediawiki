《'''微博之星'''》是自2013年起於[[香港|香港]]舉辦的一個[[網站|網站]]年度綜合頒獎典禮，由[[新浪香港|新浪香港]]主办。獎項候選名單是根據微博活躍度、傳播力、提及量及搜尋度而產生，並設立網上投票環節，作為其中一個頒發獎項的參考數據之一，並向受歡迎的[[娛樂圈|娛樂圈]]人物、企業及媒体頒發相關年度獎項。

2019年起頒獎禮更名為「微博星耀盛典」，頒授對象擴大至亞洲及歐洲等地，並增設[[网紅|网紅]]KOL的獎項。

== 評審準則 ==
「微博之星」頒獎典禮的評審準則，分為三部分：
# 候選名單根據微博活躍度、傳播力、提及量及搜尋度而產生
# 候選藝人的[[資訊|資訊]]，在新浪微博內的瀏覽量
# 網上投票環節的得票

「微博星耀盛典」頒獎典禮的評審準則：
# 通過總結過去一年的微博數據，圍繞港台地區、日韓、東南亞、北美、歐洲及澳新等地，評選年度微博星耀及人氣之星。

==獎項==
自2013年起，「微博之星」頒獎典禮的獎項，分為四部分：
# 微博給力獎項（音樂、电影、电视組別）
# 十大香港娛樂名人
# 十大香港企業
# 十大香港媒体
