'''微山湖'''，亦名'''南四湖'''，位于[[中国|中国]][[山东|山东]]、[[江苏|江苏]]交界处，由微山湖、[[昭阳湖|昭阳湖]]、[[南阳湖|南阳湖]]、[[独山湖|独山湖]]四个彼此相连的湖泊组成，归[[山东省|山东省]][[微山县|微山县]]管辖。微山四湖南北长120[[公里|公里]]，东西最宽处达25公里，水域[[面积|面积]]达1266平方公里，是中国北方最大的淡水湖泊<ref>[http://news.xinhuanet.com/society/2011-10/05/c_122121169.htm 中国“生态治理”北方最大的淡水湖微山湖]</ref>。狭义上，微山湖是指1960年在湖腰建成了拦湖大坝南的下级湖，同昭阳湖、独山湖和南阳湖构成广义微山湖。

微山湖位于[[东经|东经]]116度34分－117度21分，[[北纬|北纬]]34度27分－35度20分，像一条狭长的玉带，镶嵌在[[中国|中国]][[山东省|山东省]]南部，归山东[[济宁市|济宁市]][[微山县|微山县]]管辖。北接孔孟之乡、礼仪之邦、运河之都[[济宁|济宁]]、[[曲阜|曲阜]]、[[邹城|邹城]]，南临帝王之乡、五省通衢[[徐州|徐州]]，西通[[鱼米之乡|鱼米之乡]][[鱼台|鱼台]]、[[沛县|沛县]]、[[铜山|铜山]]，东连煤城[[滕州|滕州]]、[[枣庄|枣庄]]、[[峄城|峄城]]。湖区内有“微园”游乐小区、山东省水上运动集训场、湖滨公园、体育中心、微山湖宾馆、医疗康复中心、文化娱乐中心等。微山县内有11万伏变电站2座，3.5万伏变电站7座。

交通方面，[[京杭大运河|京杭大运河]]、[[京沪铁路|京沪铁路]]、[[104国道|104国道]]（[[京福公路|京福公路]]）、[[104省道|104省道]]均从微山县境内穿过，县城与周边各县市、县乡镇均公路相连。现有沿湖港口80多处，可直达[[长江|长江]]各口岸。航空方面，北靠[[济宁机场|济宁机场]]，南临[[徐州机场|徐州机场]]。

微山湖的形成是在4亿年前，华北地区整体下降为浅海和湖沼。在700万年以来，由于地壳强烈运动，大面积形成凹陷。鲁中山西诸水潴积于此形成涝洼区（据《邹城市志》），为微山湖的诞生创造了条件。另一个重要原因，由于黄河不断决溢淤积抬高泗水西岸的高地，[[黄河|黄河]]水长期占据此地形成了湿地，现在的微山湖就形成于明代万历年间的黄河决口。

== 风景 ==
微山湖是中国著名的六大淡水湖泊之一，风光秀丽，山、岛、森林、湖面、渔船、芦苇荡、荷花池，构成了特有的画面，是个天然的大公园。其中尤以有“花中仙子”之称的荷花最为耀眼，有的多达几拾万亩，所以又有人把这里叫作—中国荷都。{{Fact|time=2012-05-12T16:40:00+00:00}}

== 物产 ==
[[File:A_Cornor_of_Weishan_Lake.JPG|left]]
微山湖物产丰富，向来有“日出斗金”的说法，湖内有各种鱼，虾类七、八拾种，水生植物四十余种，水禽、鸟类达八十余种，被命名为中国“麻鸭之乡”和“[[乌鳢|乌鳢]]之乡”{{Fact|time=2012-05-12T16:40:00+00:00}}。湖区水资源门类比较齐全。鱼类现有78种，以鲤鱼为主，经济鱼类有[[鲫鱼|鲫鱼]]、黄鱼（鱼桑）、乌鳢、[[红鳍鲌|红鳍鲌]]、长春鳊和鲤鱼6种，底栖动物包括软体动物、节枝动物、不节动物、昆虫等63种（科），其资源总量为98876吨，浮游植物共116种，其中优势种14种，浮游动物248种，优势种共32种，水生维管束植物74种，全湖现存为304万吨，渔业生物饵料相当丰富。

微山湖矿产资源丰厚。煤炭已探明储量127亿吨，是中国重点煤炭基地之一，且埋藏浅、煤层厚，大多为优质气煤、肥煤。稀土已探明储量1275万吨，具有含磷、铁等杂质少、品位高、冶炼工艺简单等优点，是目前国内发现的唯一典型氟碳铈澜矿资源。此外，还有大量的石灰岩、煤矸石、黄沙等资源。

== 人文 ==
[[File:Location_of_Weishan_Lake.jpg|left]]
[[微山岛|微山岛]]是湖中第一大岛，文化古迹众多，境内有[[商朝|殷]][[周朝|周]][[微子墓|微子墓]]、[[漢朝|汉]]初[[张良墓|张良墓]]、[[春秋時代|春秋]][[目夷墓|目夷墓]]、[[伏羲陵|伏羲陵]]（庙）、[[仲子路庙|仲子路庙]]、[[六合泉|六合泉]]、[[圣母泉|圣母泉]]以及大量的庙宇亭台、古碑刻石、汉画像石等古迹，还有铁道游击队纪念碑。

微山湖畔的两城乡还有中国远古[[三皇五帝|三皇五帝]]之[[伏羲|伏羲]]的庙宇——伏羲庙，庙后有一池泉名为圣母池泉。马坡乡还是《[[梁山伯与祝英台|梁山伯与祝英台]]》故事发生的地点，当地仍然有[[明|明]]代遗留下来的[[梁祝合墓|梁祝合墓]]。

=== 革命老区 ===
[[抗日战争|抗日战争]]时期，以微山湖为根据地的“微湖大队”、“运河支队”、“铁道游击队”等革命武装，出没在芦苇荡里，活跃于津浦铁路线上。[[刘知侠|刘知侠]]的长篇小说《铁道游击队》即取材于此。一曲《弹起我心爱的土琵琶》使微山湖名扬中外。铁道游击队主要活动地区为现今[[枣庄市|枣庄市]]，1945年10月，在枣庄和[[临城|临城]]的1000多日军向一支不足百人的抗日游击武装投降。这是有史以来军事受降中十分罕见的一幕。

=== 名人 ===
*'''[[微子|微子]]'''，名启，是商王[[帝乙|帝乙]]的长子，[[商纣王|商纣王]]的同母庶兄。做子为[[宋国|宋国]]始祖，死后葬于宋国留邑（密县治，今微山岛西湖内）东山上，即今[[微山岛|微山岛]]上。

* '''[[目夷|目夷]]'''，字子鱼，是春秋时人，殷微子的 17 世孙，[[宋襄公|宋襄公]]的庶兄，是著名的政治家和军事家目夷墓位于微山东峰，距微子墓2.5公里处。现存墓为圆上堆，墓前立一石碑，是[[宋神宗|宋神宗]][[熙宁|熙宁]]五年（1072年）徐州知州[[傅尧俞|傅尧俞]]所立，正面阴刻篆文：“宋贤目夷君墓”。碑长方形抹角，高2.3米，宽0.9米，厚0.4米。

* '''[[张良|张良]]'''，字子房，[[戰國時代|战国]]时[[韩国|韩国]]（今河南颍川）人，是为[[刘邦|刘邦]]出谋划策，屡建功业的开国元勋，是一个难得的军师。[[刘邦|刘邦]]灭了[[项羽|项羽]]建汉称帝，高祖六年（前201年），大封功臣，高祖说。“运筹帷幄中，决胜千里外，子房功也。”叫张良选择 3 万户作为封地，[[张良|张良]]不敢接受，说：“我和帝初见面时在留，把我封在留就可以了。”因此，张良被称为留侯。张良多病，晚年避居留城封地。死后葬于留城东微山岛上。现在，墓前还有一个很大的村庄叫“墓前村”。

== 边界争议 ==
微山湖区自清朝以来，时常发生[[土客冲突|土客争斗]]事件。由于过去[[黄河|黄河]]下游经常泛滥成灾，来自山东南部等黄泛区的[[流民|灾民]]便开始移民到微山湖区，从而引发了湖区土客争斗的紧张局势，史称[[湖团案|湖团案]]。据记载，1865年，曾国藩奉命镇压[[捻军|捻军]]余部，到湖西沛县时，以为这里因为争土地发生[[分類械鬥|械斗]]的渔民是捻军余部，前来察看，才发现并非如此。于是曾国藩骑马划边为界，开挖边沟形成今天的“一溜十八团”（后来演变为十八段，现在的[[五段镇|五段镇]]等地名即来源于此），也称“边里”，从此百姓停止械斗，各自劳作，但“段里段外（或称边里边外）”这一界限仍然存在，例如边里边外在过去是不通婚的。<ref name=":0">{{Cite web|url=http://www.bjnews.com.cn/lifestyle/2012/12/19/240074.html|title=沛县 随微山湖水浮动在苏鲁省界_走读中国_新京报网|accessdate=2016-12-16|last=www.bjnews.com.cn|work=www.bjnews.com.cn|deadurl=yes|archiveurl=https://web.archive.org/web/20131022001335/http://www.bjnews.com.cn/lifestyle/2012/12/19/240074.html|archivedate=2013-10-22}}</ref>

中华人民共和国成立后，湖区位于山东与[[平原省|平原]]省界，其后由于[[徐州市|徐州地区]]划归江苏省又成为山东与江苏省界。1953年为了便于管理湖区而成立山东省微山县，将湖面、湖中岛屿以及湖区附近的几个乡镇全部划归该县，与江苏方面除少数地区外都是以湖西岸为界。由于湖水在不同季节会发生涨落，因此微山湖西岸的苏鲁两省边界时而发生变动，当地经常说，“涨水涨到的地方是山东的，水退退到的地方是江苏的”。冬季湖水退去后湖底可用于耕种[[小麦|小麦]]，而夏季水多的时候又可以收获[[芦苇|芦苇]]、养鱼、种藕等，因此常造成西岸的江苏省沛县农民与管理湖区的山东省微山县农民发生抢种抢收一类的冲突甚至械斗。

1959年微山湖水位下降后，微山湖区的群众又开始为争夺原为湖底的湖田湖产发生纠纷<ref name=":0" />。直到2006年，该地区共发生因边界纠纷引发的大规模械斗400余起，死伤800余人。国家有关部门曾多次派出工作组前往调处，并于1967年、1984年、1985年多次对湖区边界进行了调整，但终因矛盾双方坚持各自利益，微山湖地区始终未能平静。<ref name=":1">{{Cite web|url=http://news.xinhuanet.com/politics/2006-05/29/content_4613702.htm|title=?t望：地方创新化解边界纠纷 百年纷争恩怨一朝了|accessdate=2016-12-16|work=news.xinhuanet.com}}</ref>

2005年初，[[徐州市|徐州]]、[[济宁市|济宁]]两市召开第五次苏鲁接边地区维护稳定会议，专门就多年湖区纠纷死亡人员尸体处理进行协商，最终确定互不追究对方，2005年5月底前各自火化遗留尸体，并由各自县乡财政出资抚恤死者亲属。双方均在协议规定的期限内，彻底解决了遗留尸体的火化问题。<ref name=":1" />2006年以后，由于两地调解机制逐步完善，并且努力改变湖区农村经济收入单一的状态，湖区群众对湖田湖产的依赖减少，再加上很多农民离开当地外出打工，微山湖地区才开始出现和谐局面。<ref name=":0" />
