'''微鯉'''是[[鲤形目|鯉形目]][[鲤科|鯉科]][[微鲤屬|微鲤屬]]的一種，是目前已知[[世界上最小的魚類|世界上最小的魚類]]之一，也是最小的[[鯉科|鯉科]]魚類。

== 特徵 ==
身體半透明，且不具有頭骨，魚鰭可用來攀附。體型及微細小，雌性最大長度為1.03公分，雄性略小，僅0.98公分，是目前已知最小的魚類。另外，目前發現的最小成體僅有0.79公分，在[[脊椎动物|脊椎動物]]中，體型與[[胖嬰魚|胖嬰魚]]一樣僅大於在2012年發現的[[阿马乌童蛙|阿馬烏童蛙]]。由於體型嬌小，於2006年才被發現。

== 分布 ==
僅分布於[[印度尼西亚|印尼]]的[[蘇門答臘|蘇門答臘島]]和[[民丹島|民丹島]]。

== 習性 ==
主要居住在泥炭沼澤中，好酸性水域。目前對此種魚所知甚少。
