{{Vgname|微軟模擬列車|en=Microsoft Train Simulator|cn=微软模拟火车|tw=微軟模擬列車}}是一個能在微軟[[Microsoft_Windows|視窗]]作業系統運行的模擬鐵道遊戲，由英國工作室Kuju Entertainment所開發。該套軟體於2001年7月發行於[[美國|美國]]、[[歐洲|歐洲]]、[[香港|香港]]、[[日本|日本]]、[[台灣|台灣]]等地。曾於2003年公佈開發第二代，但後來取消開發微軟模擬列車2計劃<ref>美國遊戲網站Gamespot文章，http://www.gamespot.com/pc/sim/trainsimulator2/news_6094833.html</ref>。而在2007年1月，微軟再於遊戲官方網頁宣佈使用「[[微軟模擬飛行|微軟模擬飛行]]X」的遊戲引擎開發全新的系列作品。<ref>官方網頁中公佈。</ref>

== 原始版本 ==

=== 路線 ===

遊戲內含以下鐵路路線：

* [[英國|英國]]卡萊爾 - 塞特爾鐵路線，遊戲中收錄1920年代從[[卡萊爾|卡萊爾]]至[[塞特爾|塞特爾]]的整段路線，遊戲中出現列車由疾速蘇格蘭鐵路公司（Flying Scotsman Railway）營運。現在路線是英國全國鐵路網的一部分。
* [[奧地利|奧地利]]阿爾貝格鐵路線，遊戲中收錄1920年代從[[因斯布魯克|因斯布魯克]]至聖安東（St. Anton）的路段。遊戲中出現列車由東方捷運公司（The Orient-Express）營運。
* [[美國|美國]][[東北走廊|東北走廊]]線，遊戲中收錄從[[華盛頓|華盛頓]]至[[費城|費城]]的路段。遊戲中出現列車由美铁公司(Amtrak)營運。
* 美國蒙大拿山隘路線，從謝爾比（Shelby）至白鯨（Whitefish）。由BNSF鐵路公司營運。
* [[日本|日本]]小田急小田原線，從[[新宿站|新宿站]]至箱根湯本站。由[[小田急電鐵|小田急電鐵株式會社]]營運。
* 日本肥薩線，遊戲中收錄從[[八代站|八代站]]至吉松站的路段。由[[九州旅客鐵道|九州旅客鐵道]]（JR九州）營運。 {| class="wikitable" !线路名称 !线路提供商 !游戏中玩家默认驾驶的列车 !AI默认驾驶的列车 !终点站，始发站 !线路里程（km） |- |Hisatsu Line |JR九州 |Kiha31 |Kiha140 | | |- | | | | | | |- | | | | | | |}

=== 編輯器及工具 ===

遊戲提供了編輯器及工具，可供玩家修改遊戲中已有的路線、景物等等。玩家亦可以透過工具建設全新的鐵路路線，甚至遊戲中的任務（Activities）和駕駛室視點（Cabview）在遊戲中使用。
中国玩家们已经制作出了近乎完全的车辆包和大量线路任务，可以满足大陆玩家的要求

== 非官方的附加檔案 ==

由於遊戲提供了各種供玩家修改、新增遊戲元件的編輯器及工具，大量玩家製作出於現實中出現的各種鐵路路線以及鐵路車輛，亦有玩家製作虛構的路線和車輛。與「微軟模擬飛行」同樣，這些非官方的附加檔案分為需付費購買的「附加檔案組」（Add-on pack）或是可於愛好者網站中免費下載（通常需要付費的附加檔案會比可供免費下載的附加檔案更細緻）
