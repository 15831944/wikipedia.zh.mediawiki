'''微甜女孩WAYSugar'''，為[[臺灣|臺灣]][[女子演唱團體|女子演唱團體]]，天然系女子團體，是由三位具有各自獨特風格的女孩組合而成。<ref>{{cite web|url=http://www.chinatimes.com/newspapers/20180108000662-260112|publisher=中时电子报|title=愛波咒肇逃犯一生鼻毛外露|author=蔡琛儀|date=2018-01-08|accessdate=2018-01-11}}</ref>

== 團名 ==
WAYsugar前三個字母取自於三位團員英文字母字首，成員包括'''W'''awa娃娃、'''A'''pple愛波以及'''Y'''ibi牙牙。<br>
而微甜女孩這個名字，一開始的發想是希望比其他女團更有自己的個性，而不是只有甜美，剛好三人都有各自獨特的個性，希望透過微甜女孩走出自己的路，除了甜美之外也有許多不同的個性。

== 欠款風波 ==

2019之農曆年假過後，突然有多家香港及澳門的公司先後發出聲明，指謫微甜女孩及其經紀公司絕頂創意商行欠款。

2019年2月11日，香港商 ThisVO 公司的[[余嘉允|余嘉允]]發出聲明<ref>{{cite web|url=https://www.facebook.com/ThisVOTalents/photos/a.1835522546544420/2055990724497600/?type=3&theater|publisher=ThisVO.com|title=ThisVO 嚴正聲明 / A Solemn Declaration from ThisVO|author=|date=2019-02-11|accessdate=2019-02-11}}</ref>，指於數月前（2018年11月），台灣絕頂創意商行負責人張智棋先生（藝名：張大同）（即[[微甜女孩|微甜女孩]] WAYsugar之經理人）向該公司借款以作該女子組合之來港經營費。由2018年11月至今，曾使用不同藉口來推延還款。ThisVO 公司指已將有關事宜委託法律團隊追討相關賠償，亦同意把事件由法律部門交由警方調查，同時不排除將於適當時候入禀[[香港|香港]]或[[台灣|台灣]]之[[司法機構|司法機構]]將事件訴諸法律處理，更不排除提請將該等公司[[清盤|清盤]]及解散，希望該公司盡快正視及解決問題。

同日，澳門的簡約娛樂有限公司亦發出聲明<ref>{{cite web|url=https://www.facebook.com/Vangoghentertainmenthk/photos/a.1527697587560361/2177928119203968/?type=3&theater|publisher=簡約娛樂有限公司|title=簡約娛樂有限公司嚴正聲明|author=|date=2019-02-11|accessdate=2019-02-11}}</ref>，指微甜女孩之公司及經理人亦曾經向該公司借款，但簡約娛樂並無就範。及後，聲明中指簡約娛樂之𣄃下藝人收到張大同恐嚇，使其工作及心理均受到影響。外界差測被恐嚇的藝人為澳門歌手[[小瑜|小瑜]]。

2019年2月12日，香港梵高娱乐有限公司亦发出與 ThisVO 雷同的声明<ref>{{cite web|url=https://www.facebook.com/Vangoghentertainmenthk/photos/a.1527697587560361/2177928132537300/?type=3&theater|publisher=梵高娱乐有限公司|title=梵高娱乐有限公司嚴正聲明|author=|date=2019-02-12|accessdate=2019-02-12}}</ref>，揚言追讨台湾绝顶创意商行张大同先生多项宣传费用。金額約為人民幣五萬多元。並要求絕頂創意商行向該公司公開致歉，以為更早前誣指該公司虛假文書的言論負責。

欠款風波發生之後，多家港澳台媒體均有報道<ref>{{cite web|url=http://www.fhcjzk.com/caijing/2019/0212/3463.html|publisher=凤凰财经周刊|title=台湾女子组合《微甜女孩》经理人一夜成名，多家公司为他开年|author=|date=2019-02-12|accessdate=2019-02-12}}</ref>。更有網民翻查以往報道，發現微甜女孩以及其經紀公司，早已有欠款的前科<ref>{{cite web|url=https://tw.appledaily.com/new/realtime/20180315/1315152/|publisher=台灣蘋果日報|title=女團hen衰！被爆欠尾款還被酸「重鹹女孩」
|author=|date=2018-03-15|accessdate=2018-03-15}}</ref><ref>{{cite web|url=https://star.ettoday.net/news/1130906|publisher=東森新聞|title=高雄女團被酸「不值20萬」　公司出面還原欠款事件！|author=|date=2018-03-15|accessdate=2018-03-15}}</ref>。

2019年2月27日，欠款事件再次發生變化。香港 ThisVO 公司指台灣絕頂創意商行負責人張智棋先生（藝名：張大同）依然以不同借口拖延還款。並指該公司已多次就有關帳項作無條件延期，但張先生方面一直未有誠意解決問題。態度更開始轉壞。ThisVO 公司指曾於數月內收到三次由張先生向本公司發出的「已[[匯款|匯款]]」通知，張先生更有向 ThisVO 公司總經理稱「已從香港[[恒生銀行|恒生銀行]]轉帳到ThisVO所使用的香港[[東亞銀行|東亞銀行]]帳戶<ref>{{cite web|url=https://streamable.com/8bcm1|publisher=ThisVO|title=張大同和ThisVO的Whatsapp語音對話錄音|author=|date=2019-02-2|accessdate=2019-02-27}}</ref>（註：香港銀行是不允許非客戶在分行進行跨行轉帳）」。但至今一直未有一分一毫到帳，在多次追問下亦從來無法提供任何收據及付款證明。張先生亦於上一份聲明發出後向 ThisVO 公司稱將會於2月25日（星期一）提供匯款收據，最後亦再次以原因拖延。香港 ThisVO 公司再次發出嚴正聲明及最後通牒<ref>{{cite web|url=https://www.facebook.com/ThisVOTalents/posts/2079408065489199|publisher=ThisVO.com|title=ThisVO 嚴正聲明及最後通牒|author=|date=2019-02-27|accessdate=2019-02-27}}</ref>，指該公司已經開始執行對「台灣 絕頂創意商行（微甜女孩 WAYsugar之經理公司）」的強制[[清算|清算]]（強制清盤）法律安排。

2019年2月28日，即ThisVO公司向微甜女孩之經理人發出最後通牒的一天後，ThisVO 在[[臉書|臉書]]帖文表示絕頂創意一方曾於2月25日表示已再次向ThisVO公司匯款，需時一週到帳。但ThisVO公司致電香港[[星展銀行|星展銀行]]確認時，銀行依舊表示沒有任何處理中或已處理的存入匯款。因此，ThisVO公司代表已於當天早上以「商業詐騙」為由，向[[高雄市政府警察局|高雄市政府警察局]]報案。並已收到該局的「受理通知信」，該局亦向ThisVO答應將盡速辦理有關案件。<ref>{{cite web|url=https://www.facebook.com/ThisVOTalents/posts/2081237368639602|publisher=ThisVO.com|title=ThisVO 帖文指就絕頂創意欠款一事，已以「商業詐騙」為由，向高雄市政府警察局報案|author=|date=2019-02-28|accessdate=2019-02-28}}</ref>
