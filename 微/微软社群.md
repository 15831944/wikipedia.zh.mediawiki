'''微软社群'''是一個免費的線上社群和討論論壇，您可以在此詢問和回答關於使用[[Microsoft|Microsoft]]產品的問題。<ref>[http://news.thewindowsclub.com/microsoft-community-updated-to-include-surface-windows-rt-forums-56245/ twcn-technews 2012-10-26]</ref><ref>[http://www.internet2.edu/blogs/detail/8581 Internet2 and Microsoft Enhance NET+ Offerings inter2]</ref>
==創建帳號==
利用原本的[[Hotmail|Hotmail]]信箱帳號，[[MSN|MSN]]帳號及[[Microsoft_Outlook|Microsoft Outlook]]帳號可直接登入，如果沒有請自行建立一個。
==詢問、了解、解決==
你可以利用搜尋欄位來找尋你的問題，或者建立一個全新的問題。有許多消費者軟體的認可專家(每天使用我們產品的使用者)，以及來自各行各業的其他新使用者會樂於為你解答。
經由回復，討論，來建立聲望及獲得徽章。你也可以對任何解答投下支持及反對。當任何回答者回復你的問題時將透過電子郵件傳送給你。

==社群==
社群是一個個社團，由類別下面區分。例如Internet Explorer有[[Microsoft_Edge|Microsoft Edge]]，IE11~7，其他版本如IE6~1等。社群中有貢獻者則可成為[[角色|角色]]。並獲得徽章。<ref>{{Cite web |url=https://www.nintex.com/blog/mvps-share-tips-helping-microsoft-community/ |title=Ninetex blog |access-date=2016-09-11 |archive-url=https://web.archive.org/web/20160920142645/https://www.nintex.com/blog/mvps-share-tips-helping-microsoft-community/ |archive-date=2016-09-20 |dead-url=yes }}</ref><ref>[http://www.thewindowsclub.com/tips-become-microsoft-mvp-mcc the windows club]</ref>
==角色==
*社群仲裁者 : 社群成員中最有貢獻者。
*支援工程師 : 由微軟聘請的解答專家。
*論壇仲裁者 : 由微軟聘請以標示最佳解答、管理不當使用內容，並維護社群正常運作的專家。
*Wiki作者 : 是展現傑出的寫作與專門技術的社群成員。
*論壇負責人 : 負責特定類別的微軟員工。
*微軟最有價值專家 : 具有MVP徽章，可與微軟進行密切聯繫。
*Microsoft員工 : 具有微軟徽章，是參與社群的員工或在微軟裡面的員工
*Windows測試人員 : 具有Insider徽章。透過存取以前版本來進一步了解錯誤進而改善。
