{{日本作品|微笑標的|笑う標的|Laughing Target}}，是[[日本|日本]][[漫畫家|漫畫家]][[高橋留美子|高橋留美子]]於[[小學館|小學館]]少年漫畫雜誌《[[週刊少年Sunday|週刊少年Sunday]]》1983年立春增刊號發表的1回完結短篇[[奇幻漫畫|奇幻]]、[[恐怖|恐怖]][[驚悚|驚悚]][[漫畫|漫畫]]。並於1987年3月21日發售全1話的[[OVA|OVA]]動畫。

== 作品概要 ==
{{expand section}}
本作是高橋留美子早期在「[[週刊少年Sunday|週刊少年Sunday 1983年立春增刊號]]」發表一話方式完結、走嚴肅劇情路線的短篇作品，以及女性情念的視野為主題的驚悚漫畫。

== 故事簡介 ==
{{expand section}}
漫畫原作與OVA的兩種版本的結局雖然有所出入，但是女主角最後死亡的這點設定依然不變。

== OVA版 ==
1987年3月21日，以「OVA系列'''『Rumic World'''（<small>{{lang-ja|るーみっくわーるど}}</small>）'''』第3彈『微笑標的』'''」的名稱在日本當地發售，動畫由[[Studio_Pierrot|Studio Pierrot]]製作。另一方面，該動畫作品由於邀請當紅偶像歌手{{link-ja|松本伊代|松本伊代}}參加配音引發話題，但此話題卻沒有讓動畫成功得到熱烈回響，因此結果與本作品的故事結局一樣都以悲劇收場。

該動畫後來有在[[美國|美國]]的Central Park Media<ref>{{cite web|url= http://www.animenewsnetwork.com/buried-treasure/2008-09-25/the-laughing-target|title=Buried Treasure The Laughing Target|author= Justin Sevakis|date=2008-09-25|accessdate=2014-08-31|work=[[動畫新聞網|Anime News Network]]|language=en}}</ref>進行販售，目前已經結束公開販售。
