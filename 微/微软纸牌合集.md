'''微软纸牌合集'''（{{lang-en|'''Microsoft Solitaire Collection'''}}）是[[Microsoft_Studios|Microsoft Studios]]在[[Windows_8|Windows 8]]上发布的一款[[扑克牌|扑克牌游戏]]合集，包括以前版本的[[Windows|Windows]]中内置的[[微軟新接龍|FreeCell]]，[[連環新接龍|Spider]]，[[接龍_(紙牌遊戲)|Klondike]]，和全新的TriPeaks和Pyramid两种新样式。目前可从[[Windows_Store|Windows Store]]中下载。<ref>{{cite web|title=Windows 应用商店 中适用于 Windows 的 Microsoft Solitaire Collection 应用|url=http://apps.microsoft.com/webpdp/zh-cn/app/1a36fd17-5161-4651-ae2d-13384e427ea8}}</ref>游戏可使用[[Xbox_Live|Xbox Live]]账号登录以联机游戏。

== 五种纸牌游戏变体 ==
Microsoft Solitaire Collection包括以前版本的[[Windows|Windows]]中内置的[[空当接龍|FreeCell]]，[[蜘蛛纸牌|Spider]]，[[接龍_(紙牌遊戲)|Klondike]]，和全新的TriPeaks和Pyramid两种新样式，共计五种游戏变体。其中[[空当接龙|FreeCell]]、[[蜘蛛纸牌|Spider]]和[[接龍_(紙牌遊戲)|Klondike]]分别就是[[Windows_7|Windows 7]]中的[[空当接龙|空当接龙]]、[[蜘蛛纸牌|蜘蛛纸牌]]和[[接龍_(紙牌遊戲)|接龍]]。

=== 新的TriPeaks和Pyramid ===
* Tripeaks包括不同的纸牌挑战，试试看在无法移动前你能支持多久。
* 在Pyramid中组合两张相加为13的纸牌来消去他们。

== 与Xbox Live 结合 ==
现在可以登录到Xbox Live 来获得成就、与好友线上比赛,提交您的分数,追踪个人游戏设置和统计信息。Microsoft Solitaire Collection 中的 “开始游戏”“暂停”功能还会将您的记录上传到您的服务器，方便您在其他Windows 8和Windows RT 设备上立即还原游戏进度，不会丢失游戏进度和数据。
