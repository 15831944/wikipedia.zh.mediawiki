'''微軟研究院'''（{{lang-en|Microsoft Research}}，[[縮寫|縮寫]]为MSR）是[[微軟公司|微軟公司]]在1991年創立硏究不同的[[计算机科学|计算机科学]]主題與問題的分部。微軟研究院其中一個固定的目標為「支持長期的電腦科學硏究而不受產品週期所限」<ref>http://research.microsoft.com/aboutmsr/overview/default.aspx</ref>。

微軟研究院是目前世界頂尖的研究中心之一，有很多在電腦科學、[[物理學|物理學]]、[[數學|數學]]受到高度公認的專家及許多著名科學獎項得主在微軟研究院，包括[[圖靈獎|圖靈獎]]得主[[東尼·霍爾|東尼·霍爾]]、[[詹姆斯·尼古拉·格雷|詹姆斯·尼古拉·格雷]]，[[菲爾茲獎|菲爾茲獎]]得主Michael Freedman，[[沃爾夫獎|沃爾夫獎]]得主Laszlo Lovasz，MacArthur Fellow獎金得主Jim Blinn與Dijkstra Prize獎得主Leslie Lamport。

==研究領域==
微軟研究院的研究範疇可以被歸類為10大項：<ref>http://research.microsoft.com/research/default.aspx</ref>
# [[演算法|演算法]]與理論
# 硬體發展
# [[人機互動|人機互動]]
# 機械的學習、適應與智慧
# [[多媒體|多媒體]]與影像技術
# 資料搜索、擷取與[[知識管理|知識管理]]
# 資訊安全與加密技術
# 社會計算
# 軟體發展
# 系統、[[计算机体系结构|计算机体系结构]]、可攜系統與網路

微軟研究院有提供被稱為「微軟學者」的人才培育專案，微軟研究院每年會提供微軟學者獎學金與在國際、研究院的學習機會給學生。

==全球各地的研究院==

微軟研究院於[[班加羅爾|班加羅爾]]、[[北京|北京]]、[[劍橋|劍橋]]、[[矽谷|矽谷]]、[[雷德蒙德_(華盛頓州)|雷德蒙德]]與[[舊金山|舊金山]]擁有實驗室。<ref>http://research.microsoft.com/en-us/research/default.aspx</ref>

* '''微軟雷德蒙德研究院'''：1991年設立於美國[[華盛頓州|華盛頓州]]微軟雷德蒙德園區（Redmond，微軟總公司所在地），雷德蒙德研究院常與來自英國劍橋、中國北京、舊金山的研究員合作交流。
* '''[[微軟亞洲研究院|微軟亞洲研究院]]'''（MSRA）：1998年在北京成立，有上百位來自全球的研究人員與科學家，還有200多位客座研究員與學生支援研究，主要研究改良的[[使用者介面|使用者介面]]、網路與[[無線通訊|無線通訊]]，和新世代多媒體與亞洲訊息處理技術等。
* '''微軟英國劍橋研究院'''：1997年成立於英國劍橋，有超過百名的研究人員，主要研究[[機械學習|機械學習]]、[[資訊安全|資訊安全]]、[[資料探勘|資料探勘]]、[[作業系統|作業系統]]、編程與網路技術等。微軟英國研究所也與[[劍橋大學|劍橋大學]]的計算機研究室有密切的聯繫。
* '''微軟矽谷研究院'''：2001年成立於[[矽谷|矽谷]]。研究人員約25人，主要研究[[分布式運算|分布式運算]]，還有包括資訊安全、[[可靠性|可靠性]]、協定、[[資源定址器|資源定址器]]等研究。2006年8月與微軟灣區研究中心（BARC）合併。
* '''微軟印度研究院'''：2005年1月成立於[[班加羅爾|班加羅爾]]，研究人員約50人，並有大批實習生支援研究。主要再進行多領域的基礎研究及應用研究，並與多所學校與研究機構有密切交流。
