{{vgname|毁灭战士|en=Doom|遊戲標準字設計為''DOOM''}}，正式發表前通常被稱為{{vgnameq|毀滅戰士4|en=Doom 4}}，是一款各平台上市的[[第一人稱射擊|第一人稱射擊]][[電子遊戲|電子遊戲]]，
由[[id_Software|id Software]]進行開發，[[貝塞斯達軟體|貝塞斯達軟體]]發行。游戏採用[[id_Software|id Software]]新开发之{{link-en|id Tech 6|id Tech 6}}引擎，<ref name=carmackrage>{{Cite news|url=http://www.gameinformer.com/News/Story/200708/N07.0803.1731.12214.htm?Page=2 |title=QuakeCon 2007: John Carmack Talks ''Rage'', id Tech 5 And More |last=Berghammer |first=Billy |date=2007-08-03 |publisher=''Game Informer'' |accessdate=2007-08-24 |archiveurl=https://web.archive.org/web/20071029050912/http://www.gameinformer.com/News/Story/200708/N07.0803.1731.12214.htm?Page=2 |archivedate=2007-10-29 |deadurl=yes }}</ref>
本遊戲是[[毀滅戰士系列|毀滅戰士系列]]久違的回歸作，並且是繼2004年發行之[[毀滅戰士3|毀滅戰士3]]以來首次大筆投資開發的遊戲。已在2016年5月13日于[[Microsoft_Windows|Microsoft Windows]]、[[PlayStation_4|PlayStation 4]]和[[Xbox_One|Xbox One]]平台上同时推出。<ref name=hollensheadkotaku>{{cite web|last=Totilo|first=Stephen|title=Doom 4 Targeted For Console And PC, Wolfenstein In ‘Thinking About It’ Phase|url=http://kotaku.com/5613904/doom-4-targeted-for-console-and-pc-wolfenstein-in-thinking-about-it-phase|publisher=Kotaku}}</ref>游戏之后在2017年11月10日登陆[[任天堂Switch|任天堂Switch]]平台，这是系列继《{{le|毁灭战士64|Doom 64}}》之后时隔20年再次登录任天堂的游戏平台。

== 遊戲內容 ==
=== 單人 ===
根據[[id_Software|id Software]][[執行製作人|執行製作人]]{{link-en|马蒂·斯特拉顿|Marty Stratton}}表示，毀滅戰士將是一款以「惡魔超怪，槍械超壞，移動超快」作為主軸的遊戲<ref>{{cite web|url=http://metro.co.uk/2015/06/15/first-doom-gameplay-footage-is-super-gory-fun-5245833/|title=First Doom gameplay footage is super gory fun|date=2015-06-15|accessdate=2015-07-03|work=[[Metro_(magazine)|Metro]]}}</ref>。遊戲內會有各式各樣的槍械武器，玩家能夠於遊玩時搜集並可經由和其他玩家交易來免費取得。諸多經典武器，
包含雙管霰彈槍以及[[BFG_9000|BFG 9000]]（電漿炮）都將重出江湖。此外，舉凡能將怪物鋸成兩半的[[電鋸|電鋸]]等等近戰武器也會一同出現。眾多自原作現身的怪物也將再度降臨，像是骷髏亡靈（Revenant）、巨形人魔（Mancubus）以及機械巨魔（[[Cyberdemon|Cyberdemon]]）；有些怪物則是經過重新設計。

== 开发 ==
《毁灭战士4》于2008年5月7日被宣布，<ref name="ign_announce">{{cite web |author=Jason Ocampo |date=2008-05-07 |accessdate=2008-05-07 |url=http://pc.ign.com/articles/872/872097p1.html |title=''Doom 4'' Announced |publisher=[[IGN|IGN]]}}</ref>在这之前，[[约翰·卡马克|约翰·卡马克]]曾在2007年8月3日的[[QuakeCon|QuakeCon]]上暗示此游戏的存在。<ref name=carmackrage/>

2014年2月19日，贝塞斯达软件宣布向预定了《[[德軍總部：新秩序|德軍總部：新秩序]]》的玩家，开放《毁灭战士4》的Beta版本，名为“Doom”的Beta版在所有受支持的平台上均可运行。<ref>{{cite web|url=http://www.bethblog.com/2014/02/19/get-psyched-wolfenstein-release-date-first-details-on-doom-beta/ |title=Get Psyched: Wolfenstein release date; First details on Doom beta|first=Matt|last=Grandstaff|work=Bethblog.com|publisher=[[Bethesda_Softworks|Bethesda Softworks]]|date=2014-02-19|accessdate=2014-03-01}}</ref>在2014年的QuakeCon上，id Software[[執行製作人|執行製作人]]{{link-en|马蒂·斯特拉顿|Marty Stratton}}宣布《毁灭战士4》正式重命名为《毁灭战士》。<ref>{{cite web|url=https://www.vg247.com/2014/07/18/doom-4-details-quakecon-2014-reboot/|title=Why id Software is calling it "Doom" and not "Doom 4"|publisher=[[VG247|VG247]]|date=2014-07-18|accessdate=2016-06-12}}</ref>

續集《毀面戰士：永恆》（Doom Eternal）將於2019年上市。
