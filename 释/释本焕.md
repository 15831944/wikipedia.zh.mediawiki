'''释本焕'''（{{bd|1907年||2012年|4月2日|catIdx=Ben本焕}}），俗名'''张志山'''，出生于[[湖北省|湖北省]][[黄冈县|黄冈县]]（出生地今位于湖北省[[武汉市|武汉市]][[新洲区|新洲区]]），是[[中国佛教协会|中国佛教协会]]第8届理事会名誉会长，被誉为“中国佛门泰斗”，歷經晚清、民國、中华人民共和国三个阶段。<ref name="shu1 " >{{cite web |url= http://news.sohu.com/20120402/n339634872.shtml|title=中国佛教协会第8届理事会名誉会长本焕长老示寂 |publisher=搜狐网 |accessdate= 2012年|work= |language =zh-cn}}</ref><ref name="zxwa " >{{cite web |url=http://www.chinanews.com/cul/2012/04-02/3793751.shtml |title=中国佛门泰斗本焕长老在弘法寺圆寂 |publisher=中国新闻网 |accessdate= 2012年|work= |language =zh-cn}}</ref><ref name="shfb " >{{cite web |url=http://fo.ifeng.com/zhuanti/shijiefojiaoluntan2/people/200903/0321_350_54020.shtml |title=本焕长老：中国佛教协会常务理事 |publisher= 凤凰网|accessdate= 2009年|work= |language =zh-cn}}</ref>

== 生平 ==
[[File:_Ben_Huan_about_1950s.jpg|thumb]]

1907年，释本焕出生于今[[湖北省|湖北省]][[武汉市|武汉市]][[新洲区|新洲区]]。<ref name="zxwa " /><ref name="shfb " />

7岁时，被送入[[私塾|私塾]]读书，在[[私塾|私塾]]学习了6年之后，他在杂货店当学徒。<ref name="shfb " />

1930年，在新洲[[报恩寺|报恩寺]]出家，然后到[[武昌區|武昌]][[宝通寺|宝通寺]]受戒，随后到[[扬州|扬州]][[高旻寺|高旻寺]]拜师于[[汉传佛教|汉传佛教]]高僧[[释来果|释来果]]，学法七年，深研[[禅宗|禅宗]]。<ref name="zxwa " /><ref name="shfb " />

30岁时，从[[河北|河北]][[正定|正定]][[临济寺|临济寺]]起香，三步一拜朝礼[[五台山|五台山]]，历经三个月的风雨，路程300多里，终于来到了五台山。<ref name="zxwa " /><ref name="shfb " />随后用三个月的时间，朝拜了五台山上的五个台顶。<ref name="zxwa " /><ref name="shfb " />最后驻锡于[[五台山|五台山]]广济茅篷（即今[[碧山寺|碧山寺]]）。<ref name="zxwa " /><ref name="shfb " />在碧山寺修行期间，用自己的血书写了《[[普贤行愿品|普贤行愿品]]》等19部[[佛经|佛经]]，共计20余万字。<ref name="zxwa " /><ref name="shfb " />

1939年，出任[[碧山寺|碧山寺]]第三代方丈。<ref name="shfb " />

1947年，游历天下禅院，朝拜了[[北京|北京]][[弥勒院_(北京西直门南小街)|弥勒院]]、[[天津|天津]][[居士林|居士林]]、[[上海|上海]][[普济寺|普济寺]]。<ref name="shfb " />

1949年，继承[[汉传佛教|汉传佛教]]高僧[[释虚云|释虚云]]法脉，为[[临济宗|临济宗]]第四十四代传人。<ref name="zxwa " /><ref name="shfb " />随后出任[[南华寺|南华寺]]方丈。<ref name="shfb " />

1958年，[[中华人民共和国|中华人民共和国]]开始“[[反右倾|反右倾]]”运动，释本焕蒙冤入狱，在狱中被关押达22年，佛教居士[[赵朴初|赵朴初]]曾多次奔走营救释本焕。<ref name=" fhsm" >{{cite web |url= http://phtv.ifeng.com/program/nyjs/200911/1130_5364_1456313.shtml|title=本焕长老103岁生日 数十万人赴深圳祝寿 |publisher=凤凰网 |accessdate= 2009年|work= |language =zh-cn}}</ref>
[[File:Ben_Huan_1950.jpg|thumb]]
1978年[[中共十一届三中全会|中共十一届三中全会]]后，释本焕恢复了名誉和人身自由，并且获彻底平反。<ref name=" fhsm" />1980年，出任[[丹霞山|丹霞山]][[别传寺|别传寺]]方丈。1987年，出任[[广州|广州]][[光孝寺|光孝寺]]方丈。<ref name="shfb " />

释本焕曾任[[中国佛教协会|中国佛教协会]]常务理事、中国佛教协会咨议委员会主席、湖北省佛教协会名誉会长、广东省佛教协会名誉副会长、深圳市佛教协会会长、韶关市佛教协会名誉会长、广东省政协委员。一生奉行[[人间佛教|人间佛教]]，对社会[[慈善|慈善]]事业做出了巨大贡献，他捐献了希望学校9所、平民医院2家。<ref name=" fhsm" />

2012年4月2日，释本焕在深圳市[[弘法寺_(深圳市)|弘法寺]]圆寂，世寿106岁。<ref name="shu1 " /><ref name="zxwa " />长老荼毗开窑后，经专家鉴定，在铁栏融化的情况下，全身骨骼完整，皆为舍利。目前泰国、柬埔寨、澳大利亚、老挝等国内外众多寺庙纷纷修建塔庙，恭请本老舍利永久安奉。[[尼泊尔|尼泊尔]][[中华寺_(尼泊尔)|中华寺]]建有其舍利塔。
