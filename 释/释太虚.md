'''釋太虚'''（{{bd|1890年|1月8日|1947年|3月17日|catIdx=T太}}），俗名'''吕沛林'''，出生于[[浙江省|浙江]][[海宁县_(明朝)|海宁县]]长安镇，[[籍贯|籍贯]]浙江[[崇德县|崇德县]]（今[[桐乡市|桐乡市]]），著名[[佛教|佛教]][[臨濟宗|臨濟宗]][[禪師|禪師]]、哲学家，是[[印順法師|印順法師]]、[[東初法師|東初法師]]、[[大勇法師|大勇法師]]、[[慈航法師|慈航法師]]的入門師父，被推崇為[[人間佛教|人間佛教]]的創辦人，受到佛教後學們的崇敬。

== 艰辛童年 ==
太虚出生于浙江[[海宁县_(明朝)|海宁]]长安镇一个木工家庭，俗名'''吕沛林'''，父母早逝，从小跟随外祖母生活，体弱多病，生活拮据。曾跟随笃信佛道的外祖母到[[九华山|九华山]]、[[普陀山|普陀山]]等名山古刹[[进香|进香]]，故对佛教素有好感。在小舅的指导下，沛林读书识字，并曾在百货商店当过学徒。

== 出家求学 ==
世间生活的困苦，使呂沛林对晨钟暮鼓的出家生活更加向往，1904年5月，沛林终于离开长安镇，最后在[[吴江县|吴江县]][[平望|平望]][[平望小九华寺|小九华寺]]隨士達法師[[剃度|剃度]][[出家|出家]]，[[臨濟宗|臨濟宗]]法名「唯心」，后由师祖[[奘年法师|奘年]]立表字「太虚」。同年12月，随奘年于[[曹洞宗|曹洞宗]][[宁波|宁波]][[天童寺|天童寺]]受戒，[[得戒师|得戒师]][[敬安法师|敬安法师]]，[[尊证师|尊证师]][[道阶法师|道阶法师]]。在传戒过程中，由于太虚记忆力超群，短时间内将各种戒本背得烂熟，答问自如，众戒师「咸以法器相许」。

随后几年，在敬安法师、歧昌法师、道阶法师等高僧的教导下，太虚开始学习经论，包括《[[法华经|法华经]]》、《[[楞严经|楞严经]]》等，同时也翻阅了许多传记，包括《[[指月录|指月录]]》、《[[高僧传|高僧传]]》、《[[弘明集|弘明集]]》等。由于太虚记忆力超常，口才出众，众名师对其多有属意。

1907年，太虚前往[[慈溪|慈溪]][[慈溪西方寺|西方寺]]阅[[大藏经|大藏经]]，阅《[[大般若经|大般若经]]》有悟。1909年来到[[南京|南京]]，就读于[[杨仁山|杨仁山]]居士创办的[[祇洹精舍|祇洹精舍]]。

== 革命启蒙 ==
1908年，在僧人[[华山(僧人)|华山]]的影响下，太虚阅读了不少革命著作，包括[[康有为|康有为]]《[[大同书|大同书]]》、[[梁启超|梁启超]]《[[新民说|新民说]]》、[[章太炎|章太炎]]《[[告佛弟子书|告佛弟子书]]》等。同年夏，结识了革命僧人[[栖云|栖云]]，开始与[[革命党|革命党]]人来往。

1910年应栖云邀请，前往革命党人云集的[[广州|广州]]，于寺院[[狮子林|狮子林]]设佛学[[精舍|精舍]]讲学，后任双溪寺[[住持|住持]]，期间阅读了[[托尔斯泰|托尔斯泰]]、[[巴枯宁|巴枯宁]]、[[马克思|马克思]]等人的译著，与革命党人多有交往，并不时参與革命党秘密集会。1911年4月，[[同盟会|同盟会]][[黄花岗起义|黄花岗起义]]爆发，旋即失败，太虚作诗《弔黄花岗》，内有“南粤城里起战争，隆隆炮声惊天地！为复民权死亦生，大书特出一烈字”，后被官方察觉，派兵围捕，太虚避居《平民报》报馆。

== 风雨护教 ==
[[File:Taixu2.jpg|thumb]]》中的太虛]]
=== 背景===
==== 国家的命运 ====
[[清朝|清朝]]后期，以非正统儒家思想为主导的传统集权社会已经走到了尽头，暴露出日益尖锐的社会问题，上层统治阶级占据着大部分的土地，腐败堕落；广大的下层百姓，在苛捐杂税、传统伦理的压榨下，在绝望中奄奄一息。作为近代科学发展典范的欧美，在对贫穷国家方面，并没有施以援手，反而更加贪婪地掠夺这些弱势国家的物质利益，这使得贫苦的中国民众更加雪上加霜。

同治年间，[[洪秀全|洪秀全]]打着"[[拜上帝教|拜上帝教]]"的旗号，建立[[太平天國|太平天國]]，发动起大批贫农試圖推翻[[滿清|滿清]]，一度占据了中国东南部的十多个省份，所到之处，[[寺院|寺院]]、[[道观|道观]]、[[廟宇|廟宇]]等这些他们看来的"异教"，都一概摧毁，这场长达十多年的暴乱，几乎造成[[中華文化|传统文化]]（包括佛教）的断层。

==== 佛教的命运 ====
佛教在这样的动荡不安中，亦是日趋沒落。由于取消了考试取僧的制度，僧人文化素质良莠参杂。在宗法社会的阴影下，原本鲜活的佛教体制，演变为"[[子孙庙|子孙庙]]"、"房头患"。在物质利益的驱动下，僧人不是忙于经论学习、参禅修持，而是汲汲于赶经忏，沦为职业治丧者。

整个社会在要求革新，上层士大夫也在尝试改良，他们把矛头指向了佛教，制造舆论，鼓吹占寺夺产来兴办教育。面对这样的情势，有识之士纷纷采取措施来挽救佛教。[[杨仁山|杨仁山]]居士在南京创办了[[金陵刻经处|金陵刻经处]]，刻印流通了大量急缺的佛教典籍。有的地方开始尝试建立佛教组织，来为佛教呼吁；建立新式的佛教教学机构，培养弘法的人才。太虚法师，在这两个方面，都努力地进行了探索与开拓。

=== 组织佛教===
[[File:Taixu_Standing.jpeg|thumb]]
====「大闹金山」事件====
1912年[[元旦|元旦]]，[[中华民国|中华民国]]在南京宣告成立。太虚随后在南京发起成立佛教协进会，于[[毗庐寺|毗庐寺]]设立筹备处。在[[社会党|社会党]]成员的引荐下，[[孙中山|孙中山]]在总统府接见了太虚，并托其秘书[[马君武|马君武]]与之详谈。太虚详细阐述了佛教改革、筹备协进会的情况，得到马君武的赞可。<!--1911年，赴[[广州|广州]][[白云山|白云山]]住持[[双溪寺|双溪寺]]?-->

此时，仁山也来到南京，他准备上书教育部，要求将[[金山寺|金山寺]]改办成僧学堂以造就僧才。在与太虚商议后，决定同往镇江，联合众人，召开协进会的成立大会，共图改革大业。

1912年1月，太虚等人抵达金山寺，他们拜访了[[方丈|方丈]]青权、[[知客|知客]]霜亭等，讨论协进会事宜。由于改革佛教的主张，可能会触及守旧势力的利益，而且仁山与这些人早有宿怨，但青权等人碍于革命风潮、太虚等人多势众，协进会还是在金山召开了。

与会者僧众有二三百人，各界来宾有三四百，其中以社会党人为多，太虚被推选为会议主席，讲明设会宗旨，宣读会章，接着仁山发言，他指责传统的[[剃度制|剃度制]]、[[子孙庙|子孙住持制]]，认为其压制有识僧人的成长，要求予以废除。扬州僧人寂山随即登台演说，批驳仁山无端冒犯祖制，胡作非为。仁山听罢，怒火中烧，再次登台，力数扬州、镇江诸山长老昏庸无能、专谋私利。仁山的发言，受到了来宾的欢迎，仁山趁机提出，要以金山寺开办佛教学堂，全部寺产充作办学经费，方丈青权表示无法忍受，寂山高声呼打，但由于社会党人压阵，甚至有人用手杖击打寂山头颅，青权等人最后只好忍耐表面接受。

太虚对混乱的会场，自觉不妥，随后宣布散会。当晚，仁山带领二十多个同学接管金山寺。太虚将镇江诸事务交付仁山住持，自己回到南京。几天后，霜亭带领几十个人趁夜进入佛协会，将仁山等人打成重伤。在舆论的谴责下，青权、霜亭等人被判刑，但随后因为政府大赦，青权等人又回到金山寺，重新担任住持。

协进会以失败而告终，因为这则事件，太虚虽然名声大震，但名誉受到了影响。

1914年至1916年间，太虚在浙江[[普陀山|普陀山]]闭关于[[锡龄禅院|锡龄禅院]]，由[[印光|印光]]为其封关。

=== 兴办教育 ===
出关后，太虚赴[[日本|日本]]、[[台湾|台湾]]和[[东南亚|东南亚]]考察当地佛教，回国后鼓吹佛教革新，创办《觉社书》杂志，后改名《[[海潮音|海潮音]]》月刊。1920年，太虚前往武汉讲解[[心经|心经]]，受到当地僧众拥戴，成立汉口佛教会。

==== 武昌佛学院 ====
1921年，住持[[杭州|杭州]][[净慈寺|净慈寺]]，后因受到非议，年底再次前往武汉，受[[私立武昌中华大学|私立武昌中华大学]]之聘，讲解[[哲学|哲学]]和[[因明学|因明学]]。1922年，任[[武昌佛学院|武昌佛学院]]院长，次年创建[[世界佛教联合会|世界佛教联合会]]，任会长。1928年，出访[[欧洲|欧洲]]，传播佛教教义。

== 晚年 ==
[[File:Xiamen_Nanputuo_20120226-21.jpg|thumb]]的太虚大师之塔]]
[[抗日战争_(中国)|抗日战争]]爆发后，前往东南亚各地组织当地华侨募捐，支持抗战。1943年，和[[于斌|于斌]]、[[冯玉祥|冯玉祥]]、[[白崇禧|白崇禧]]等人成立宗教联谊会。抗战胜利后，被国民政府授予抗戰胜利勋章。

1947年3月17日，太虚在[[上海|上海]][[玉佛寺|玉佛寺]]因[[脑溢血|脑溢血]]圆寂，葬于浙江[[奉化|奉化]][[雪窦山|雪窦山]]。
