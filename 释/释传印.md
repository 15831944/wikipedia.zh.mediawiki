'''释传印'''{{BD|1927年|1月30日|||catIdx= L吕}}原名'''吕毓岱'''，字'''月川'''，法名'''宣传'''，出生于[[辽宁|辽宁]][[庄河|庄河]]光明山，毕业于[[中国佛学院|中国佛学院]]，[[日本|日本]][[京都|京都]][[净土寺佛教大学|净土寺佛教大学]]留学生，2010年出任[[中国佛教协会|中国佛教协会]]会长。<ref name="lion " />

==简历==
1927年，释传印出生于[[辽宁|辽宁]][[庄河|庄河]]光明山。<ref name="lion " />

1947年，释传印拜师于[[庄河县|庄河县]][[普化寺|普化寺]][[释崇仁|释崇仁]]。<ref>{{cite web |url=http://fo.ifeng.com/special/chuanyinzhanglaozt/beiminzhiluchujia/detail_2010_03/26/407981_0.shtml |title=1947年 我在老家庄河普化寺出家 |publisher=凤凰网 |accessdate=2010年3月26日 |work= |language =zh-cn}}</ref>

1954年，释传印参与开发[[江西|江西]][[云居山|云居山]]。<ref>{{cite web |url = http://fo.ifeng.com/special/chuanyinzhanglaozt/beiminzhiluchujia/detail_2010_03/26/407990_0.shtml |title = 1954年 几经周折终于去了江西云居山 |publisher  =凤凰网 |accessdate = 2010年3月26日 |work = |language = zh-cn}}</ref>

1955年，[[释虚云|释虚云]]给其受[[具足戒|具足戒]]<ref>{{cite web |url=http://fo.ifeng.com/special/chuanyinzhanglaozt/beiminzhiluchujia/detail_2010_03/26/407994_0.shtml |title=1955年 依虚云老和尚受具足戒 |publisher = 凤凰网 |accessdate = 2010年3月26日 |work = |language = zh-cn}}</ref>。

1956年，释传印担任释虚云的侍者，成为[[沩仰宗|沩仰宗]]第九世传人，法名'''宣传'''<ref name="lion " >{{cite web |url = http://fo.ifeng.com/special/chuanyinzhanglaozt/beiminzhiluchujia/detail_2010_03/26/407576_0.shtml |title = 传印长老简历 |publisher = 凤凰网 |accessdate = 2010年3月26日|work= |language = zh-cn}}</ref>。

1960年，释传印到[[中国佛学院|中国佛学院]]读书，学习[[英语|英语]]、[[日语|日语]]和[[巴利文|巴利文]]<ref>{{cite web |url = http://fo.ifeng.com/special/chuanyinzhanglaozt/ziwochaoyu/detail_2010_03/26/408048_0.shtml |title = 1960年9月 推荐我到中国佛学院读本科 |publisher = 凤凰网 |accessdate = 2010年3月26日 |work = |language = zh-cn}}</ref>。同年，母亲和姐姐出家，母亲法名隆祥，姐姐法名能述。

1966年，[[文化大革命|文化大革命]]爆发，释传印被下放到[[云山|云山]]种地<ref>{{cite web |url = http://fo.ifeng.com/special/chuanyinzhanglaozt/jianshouxinyang/detail_2010_03/26/408062_0.shtml |title = 1966年 文革来了，我到云居山垦殖场劳动，坚守吃素 |publisher = 凤凰网 |accessdate=2010年3月26日  |work= |language =zh-cn}}</ref>。晚上在当地的弥陀寺偷偷礼佛<ref>{{cite web |url = http://fo.ifeng.com/special/chuanyinzhanglaozt/jianshouxinyang/detail_2010_03/26/408082_0.shtml |title = 在农场8年 只能偷着礼佛 |publisher=凤凰网 |accessdate=2010年3月26日 |work= |language =zh-cn}}</ref>。

1974年，释传印到[[旸岭山|旸岭山]]种地<ref>{{cite web |url=http://fo.ifeng.com/special/chuanyinzhanglaozt/jianshouxinyang/detail_2010_03/26/408096_0.shtml |title = 1974年 为了修行礼佛，我一头扎进了人烟稀少的旸岭山 |publisher = 凤凰网 |accessdate= 2010年3月26日 |work= |language = zh-cn}}</ref>。

1978年，文化大革命结束，释传印被召回[[北京|北京]]<ref>{{cite web |url = http://fo.ifeng.com/special/chuanyinzhanglaozt/jueduirenwuzhongshenyuanding/detail_2010_03/26/408109_0.shtml |title = 1978年 我从江西到北京轰动了当地 |publisher= 凤凰网|accessdate=2010年3月26日 |work= |language =zh-cn}}</ref>。

1979年，[[国务院宗教事务局|国务院宗教事务局]]政策研究室主任[[赵宏伟|赵宏伟]]推荐他到日本留学<ref>{{cite web |url= http://fo.ifeng.com/special/chuanyinzhanglaozt/jueduirenwuzhongshenyuanding/detail_2010_03/26/408116_0.shtml|title=1979年 国家宗教局推荐我去日本进修 |publisher=凤凰网|accessdate=2010年3月26日 |work= |language =zh-cn}}</ref>。

1981年，释传印到[[日本|日本]][[京都|京都]][[净土寺佛教大学|净土寺佛教大学]]留学，同时考察日本佛教<ref>{{cite web |url= http://fo.ifeng.com/special/chuanyinzhanglaozt/jueduirenwuzhongshenyuanding/detail_2010_03/26/408122_0.shtml|title= 1981年 中央支持恢复佛教教育 我去日本留学考察|publisher=凤凰网 |accessdate=2010年3月26日 |work= |language =zh-cn}}</ref>。

1983年，释传印在[[中国佛学院|中国佛学院]]任教<ref name="lion " />。

1999年，释传印出任[[中国佛教协会|中国佛教协会]]副会长<ref name="lion " />。

2010年，释传印出任中国佛教协会会长兼[[中国佛学院|中国佛学院]]院长<ref name="lion " />。
