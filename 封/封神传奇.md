《'''封神传奇'''》（{{lang-en|League of Gods}}）是[[博纳影业|博纳影业]]和[[中国星集团|中国星集团]]联合出品的一部中国3D魔幻动作片，改编經典中國神話《[[封神演义|封神演义]]》，由[[李连杰|李连杰]]、[[梁家辉|梁家辉]]、[[范冰冰|范冰冰]]、[[黄晓明|黄晓明]]、[[古天乐|古天乐]]、[[Angelababy|Angelababy]]、[[文章_(演员)|文章]]、[[向佐|向佐]]、[[许晴|许晴]]、[[祖鋒|祖鋒]]、[[安志杰|安志杰]]主演，[[许安|许安]]执导，中国大陆、香港于2016年7月29日公映。

== 剧情 ==
商朝末代，紂王（梁家輝 飾）被妖妃妲己（范冰冰 飾）所迷惑，聯同申公豹（古天樂 飾）召喚黑暗力量，驅使滅世黑龍降生人間，一時間妖孽横行，禍國殃民。為阻止昏君滅世，姜子牙（李連杰 飾）派遣「雷震子」姬雷（向佐 飾）、哪吒（文章 飾）與 「二郎神」楊戩（黃曉明 飾），合力尋找光明之劍，斬妖除魔，並協助周武王率領西岐大軍攻打朝歌。不料妲己施法令姜子牙中了逆生咒，朝歌大戰迫在眉睫，黑龍降世已成定局，天下陷入滅世危機之中……

== 制作 ==

== 发行 ==
2016年4月21日，博纳影业总裁[[于冬|于冬]]、中国星的[[向华强|向华强]]、[[陈岚|陈岚]]，以及主演李连杰、向佐出席在[[北京|北京]]举办的定档发布会，发布会上播放了30秒预告片，并宣布影片将于7月29日上映。<ref name="定档"/>5月中旬，发布首款海报。<ref name="海报">{{cite web|url=http://news.mtime.com/2016/05/17/1555508.html | language= | title=《封神》首发“逆天”海报 |publisher=Mtime时光网 |date=2016-05-17 |accessdate= }}</ref>7月24日，主创在[[北京|北京]]举行首映礼，出品人于冬、向华强、陈岚，导演许安、杨龙，以及李连杰、范冰冰、祖鋒、向佐、[[黄岳泰|黄岳泰]]等出席。<ref >{{cite web|url=http://www.1905.com/news/20160725/1050668.shtml#p1 | language= | title=《封神》红毯群星缺席 范冰冰、李连杰卖力撑场 |publisher=1905电影网 |date=2016-07-25 |accessdate= }}</ref>

== 评价 ==
電影在中國上映後，評價慘不忍睹，在豆瓣僅得3.0的低分。普遍批評即使擁有強大的卡司陣容和一定程度的3D特效，但劇情實在毫無用心。<ref >{{cite web|url=http://www.1905.com/news/20160731/1054066.shtml#p1 | language= | title=《封神传奇》：超越《富春山居图》华语片"奇迹" |publisher=1905电影网 |date=2016-07-31 |accessdate= }}</ref>
