氰基丙烯酸乙酯(ECA)，又稱為 2-氰基-2-丙烯酸乙酯，它是一種無色液體，具有粘度低。它可溶於[[丙酮|丙酮]]、[[丁酮|丁酮]]、[[硝基甲烷|硝基甲烷]]和[[二氯甲烷|二氯甲烷]]。在潮濕的環境下氰基丙烯酸乙酯會發生聚合反應。

== 製備 ==
利用[[甲醛|甲醛]]與氰基乙酸乙酯縮合反應，合成氰基丙烯酸乙酯：<br />
CH<sub>2</sub>(CN)CO<sub>2</sub>Et  +  CH<sub>2</sub>O   →  CH<sub>2</sub>=C(CN)CO<sub>2</sub>Et  +  H<sub>2</sub>O
<br />
然後讓該聚合物(CH<sub>2</sub>=C(CN)CO<sub>2</sub>Et)再加熱進行[[裂化反应|裂化反应]]形成單體(即氰基丙烯酸乙酯)。
== 應用 ==
氰基丙烯酸乙酯用於黏合各種材料。它在醫學也應用於液體繃帶和無縫線手術上，但是它較低毒性的正丁基氰基丙烯酸酯和辛基氰基丙烯酸酯少使用。 現成的非醫療級膠水不適合醫療應用，因為它可能含有溶劑（如甲醇），並在聚合過程中產生熱量。<ref name="miracleglue.com">[http://www.miracleglue.com/wounds.htm]</ref><br />
固化後，其樹脂軟化的溫度在超過150°C以上。氰基丙烯酸乙酯樹脂使用溫度為-54°C到82 °C之間，而[[相对电容率|相对电容率]]為3.33在1 MHz之下<ref name="miracleglue.com"/>。

== 安全 ==
在美國，氰基丙烯酸乙酯(ECA)的 [http://en.wikipedia.org/wiki/Threshold_Limit_Value 閾限值] 是 2 ppm。在高溫下會解聚，並產生強烈刺激性氣體產品影響肺部和眼睛。
