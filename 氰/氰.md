氰（'''Cyanogen'''）旧译作'''𩇛'''，也称'''氰气'''，[[化学式|化学式]]为([[碳|C]][[氮|N]]){{sub|2}}，是[[碳|碳]]和[[氮|氮]]的[[化合物|化合物]]（N≡C—C≡N）。可用于[[有机合成|有机合成]]，也用作消毒、杀虫的[[熏蒸剂|熏蒸剂]]。

氰在[[标准状况|标准状况]]下是无色[[气体|气体]]，带苦杏仁气味。[[燃烧|燃烧]]时呈桃红色火焰，边缘侧带蓝色。氰溶于[[水|水]]、[[乙醇|乙醇]]、[[乙醚|乙醚]]。

氰的化学性质与[[卤素|卤素]]很相似，是[[拟卤素|拟卤素]]（或类卤素）的一种。氰气会被[[氧化还原反应|还原]]为毒性极强的[[氰化物|氰化物]]。氰在高温下与[[氢气|氢气]]反应生成[[氰化氢|氰化氢]]。与[[氢氧化钾|氢氧化钾]]反应生成[[氰化钾|氰化钾]]和[[氰酸钾|氰酸钾]]。氰加热至400°C以上[[聚合|聚合]]成不溶性的白色固体(CN)<sub>x</sub>。

氰是[[草酰胺|草酰胺]]的脱水产物，是[[草酸|草酸]]衍生的[[腈|腈]]：
:H<sub>2</sub>NC(O)C(O)NH<sub>2</sub>  →  N≡C—C≡N  +  2 H<sub>2</sub>O

== 制备 ==
方法一：在高温下[[碳|碳]]与[[氮|氮]]直接化合成氰

:N<sub>2</sub>+2C -高温→ (CN)<sub>2</sub>+71千卡

方法二：氰可由加热[[氰化汞|氰化汞]]制得：
:2 Hg(CN)<sub>2</sub> → (CN)<sub>2</sub> + Hg<sub>2</sub>(CN)<sub>2</sub>

方法三：二价铜盐，如[[硫酸铜|硫酸铜]]与氰化物反应产生不稳定的氰化铜，很快分解生成氰和[[氰化亚铜|氰化亚铜]]：<ref>{{cite journal

:2KCN → 2K + (CN)<sub>2</sub>
