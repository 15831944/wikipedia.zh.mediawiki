'''習近平畫像潑墨事件'''指2018年7月一位网名为“feefeefly”（本名[[董瑤瓊|董瑤瓊]]）的女子用[[Twitter|Twitter]]在上海[[海航集團|海航]]大廈前直播其對[[中国共产党|中国共产党]][[一党专政|一党专政]]的不滿，並批評中共暴政實行的“腦控”壓迫、朝[[中国共产党中央委员会总书记|中共中央总书记]][[習近平|習近平]]頭像潑墨，因而受到關注<ref name="自由時報電子報 2018">{{cite web | title=抗議暴政！直播對習近平畫像潑墨 女網友行蹤成謎 - 國際 | website=自由時報電子報 | date=2018-07-05 | url=http://news.ltn.com.tw/news/world/breakingnews/2478575 | language=zh | access-date=2018-07-06}}</ref>。後來證實潑墨者名為董瑤瓊，中國湖南省株洲市攸县人，在上海房地產公司做中介工作。

潑墨事件隨即在網絡上快速散播，並引起网民迴響。有网民貼出對習近平肖像潑墨、擲泥土的照片，表達對中國共產黨的高壓統治之不滿。<ref name="自由時報電子報 218">{{cite web | title=接力響應潑墨行動 網友：以後買墨水得實名制登記| website=自由時報電子報 | date=2018-07-06 | url=http://news.ltn.com.tw/news/world/breakingnews/2480361 | language=zh | access-date=2018-07-07}}</ref>

== 事件經過 ==
2018年7月4日，清晨6时左右，在[[上海|上海]][[贯君|海航]]大厦前说：“...反对[[习近平|习近平]][[独裁|独裁]][[专制|专制]]暴政，反对中国共产党对我实施的脑控压迫...反对（此时向[[习近平|习近平]]画像泼墨）...我对他恨之入骨...看到没有，这是我的行为，反对[[习近平|习近平]][[专制|专制]]、暴政，反对共产党对我实施的脑控压迫。我要求国际组织介入进来，我会配合调查取证。因为很多脑控迫害取证是需要才能介入的。...对，我今天就泼墨他了，我看他能把我怎么地，[[习近平|习近平]]，我在这里等着你来抓我。我就一个人，反对中国共产党，反对共产党[[独裁|独裁]]、暴政、[[专制|专制]]。我今天就站在这里，我让你来抓我。后面就是[[贯君|海航]]大厦，[[习近平|习近平]]你的资产，好吗。我就在你的资产面前，泼你的墨。看到没有，看到你丑陋的脸没有？”。<ref>{{cite news|title=抗議暴政！直播對習近平畫像潑墨 女網友行蹤成謎|url=http://news.ltn.com.tw/news/world/breakingnews/2478575|accessdate=2018-07-25|publisher=[[自由时报|自由时报]]|date=2018-07-05}}</ref>

下午3時30分左右，董瑤瓊在Twitter發出其帳號被不明人士註銷前的最後一則推特PO文称“現在我的門外有人一群制服的人。待會換好衣服就出去。我沒有罪。有罪的是傷害我的人和組織。”

之後推特上的文章開始被刪除，接著本人便失去联系。中國異議藝術家[[華涌|華涌]]个人猜测說，董瑤瓊的推特離奇消失，估計是她被中共控制後，控制她的人所為。<ref name="自由時報電子報 2018"/>《[[自由亞洲電台|自由亞洲電台]]》報導該報記者曾透過Twitter聯繫上董瑤瓊，但她拒絕接受採訪，不久她的推特帳號關閉，行蹤成謎。<ref name="自由時報電子報 2018"/>「潑墨」一詞成為網路禁詞<ref name="聯合新聞網 2018"/>。

[[華涌|華涌]]在湖南找到董瓊瑤的父親挖煤礦工董建彪後，得知其女已被中共當局以「攻擊國家領導人」罪名抓捕，董建彪說“如果是我，我也會潑”、“我認為她的行為蠻好的，她不是為自己，是代表底層人，她說的是事實”、“我女兒這次可能命都保不住了，習主席把自己當成[[皇帝|皇帝]]，過去得罪了皇帝是要株連九族的…”、“得罪了這麼高的權貴，如果判她死刑，我願意去頂，如果頂不了，我也不想活了”、“我就要保住女兒一條命”。直播這一事態並表明“希望全世界熱愛自由的人們，替我和董建彪先生呼籲[[言論自由|言論自由]]無罪…”的華涌和董父也立即於半夜被破門而入抓走。<ref>{{Cite web |url = http://trad.cn.rfi.fr/%E4%B8%AD%E5%9C%8B/20180714-%E8%AD%A6%E6%96%B9%E7%96%91%E4%BB%A5%E6%94%BB%E6%93%8A%E5%9C%8B%E5%AE%B6%E9%A0%98%E5%B0%8E%E4%BA%BA%E7%BD%AA%E5%90%8D%E6%8A%93%E6%8D%95%E4%B8%8A%E6%B5%B7%E6%BD%91%E5%A2%A8%E5%A5%B3%E5%AD%90 |title = 警方疑以攻擊國家領導人罪名抓捕上海潑墨女子 |agency = [[法國國際廣播電台|法國國際廣播電台]] |author = |date = 2018年7月14日 |language = zh-tw}}</ref><ref>{{Cite web |url = http://news.ltn.com.tw/news/world/breakingnews/2488011 |title = 女向習近平像潑墨後人間蒸發 父親和異議人士被抓 |agency = [[自由時報|自由時報]] |author = |date = 2018年7月14日 |language = zh-tw}}</ref>
