'''唯美主义'''（{{lang-en|aestheticism}}）主張[[藝術哲學|藝術哲學]]應獨立於所有[[哲學|哲學]]之外，[[藝術|藝術]]只能以藝術自身的標準來評判<ref>{{cite web|url=http://db1x.sinica.edu.tw/caat/caat_rptcaatc.php?_op=?SUBJECT_ID:300055955|title=藝術與建築索引典—唯美主義 (哲學運動及態度)|accessdate=2011年3月14日}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

'''唯美主义运动'''（英语：{{lang|en|aesthetic movement}}）在汉语中有“'''美学运动'''”、“'''唯美运动'''”等不同译法，是於19世纪后期出现在[[英国|英国]]艺术和文学领域中的一场组织松散的[[反社會|反社會]]的[[社會運動|運動]]，发生于[[维多利亚时代|维多利亚时代]]晚期，大致从1868年延续至1901年，通常[[学术界|学术界]]认为唯美主义运动的结束以[[奥斯卡·王尔德|奥斯卡·王尔德]]被捕为标志。

== 緣起 ==
唯美主义运动呈現著頹廢風氣，也有著對當時席捲著歐洲的[[樂觀進步主義|樂觀進步主義]]的反思。英国的[[颓废派|颓废派]]作家们受[[瓦尔特·佩特|瓦尔特·佩特]]的影响非常大。佩特在1867年至1868年之间发表了一系列文章，主张人们应该热情的拥抱生活，追求生活的艺术化。颓废主义者们接受了这一观点。[[法国|法国]][[哲学家|哲学家]][[维克多·库辛|维克多·库辛]]和[[奥菲尔·戈蒂埃|奥菲尔·戈蒂埃]]在[[法国|法国]]推广了这一[[观念|观念]]，提出“为艺术而艺术”的口号，并声称[[艺术|艺术]]与[[道德|道德]]之间没有关联。

== 主張 ==
唯美主义运动中的作家和艺术家认为：艺术的使命在于为人类提供感观上的愉悦，而非传递某种道德或情感上的信息。因此，唯美主义者们拒绝接受[[约翰·罗斯金|约翰·罗斯金]]和马修·阿诺德提出的“艺术是承载道德的实用之物”的[[功利主义|功利主义]]观点。相反，唯美主义者认为[[艺术|艺术]]不应具有任何说教的因素，而是追求单纯的[[美感|美感]]。他们如痴如醉的追求[[艺术|艺术]]的“美”，认为“美”才是[[艺术|艺术]]的[[本质|本质]]，并且主张[[生活|生活]]应该[[模仿|模仿]][[艺术|艺术]]。

== 唯美主义文学特征 ==
[[唯美主义运动|唯美主义运动]]的主要[[特征|特征]]包括：追求建议性而非陈述性、追求感官享受、对[[象征|象征]]手法的大量应用、追求[[事物|事物]]之间的关联感应——即探求[[语汇|语汇]]、[[色彩|色彩]]和[[音乐|音乐]]之间内在的联系。

人们一般认为唯美主义和彼时发生在[[法国|法国]]的[[象征主义|象征主义]]或[[颓废主义运动|颓废主义运动]]同属一脉，是这场国际性文艺运动在英国的分支。这场运动是反[[维多利亚|维多利亚]]风格风潮的一部分，具有后[[浪漫主义|浪漫主义]]的特征。

唯美主义有時被與[[惡魔主義|惡魔主義]]連結，但是對惡魔主義和頹廢藝術不一定相符，倒不如說是感性的和[[文藝復興運動|文藝復興運動]]一起進行。有時也和[[神秘主義|神秘主義]]結合。

== 唯美主义文学代表人物 ==
唯美主义视[[浪漫主义|浪漫主义]][[诗人|诗人]][[约翰·济慈|约翰·济慈]]和[[雪莱|雪莱]]为先驱，也受到了[[拉斐尔前派|拉斐尔前派]]的影响。

在英国，唯美主义最杰出的代表人物是[[阿尔杰农·查尔斯－斯温伯恩|阿尔杰农·查尔斯－斯温伯恩]]（{{tsl|en|Algernon Swinburne|Algernon Swinburne}}），和[[奥斯卡·王尔德|奥斯卡·王尔德]]，这两人都接受过[[法国|法国]][[象征主义|象征主义]]的影响。

和唯美主义运动有关联的艺术家包括[[詹姆斯·麦克尼尔·惠斯勒|詹姆斯·麦克尼尔·惠斯勒]]和[[但丁·加布里埃尔·罗塞蒂|但丁·加布里埃尔·罗塞蒂]]。

== 对唯美主义文学的評價 ==
[[英國|英國]][[詩人|詩人]]兼評論者，被譽為[[維多利亞時代|維多利亞時代]]中期[[反叛|反叛]][[詩人|詩人]][[象徵|象徵]]的阿尔杰农·斯温伯恩對此派系的評論為「這個畫派的意義是『美』本身」，說明了唯美主義的[[本質|本質]]。

[[法國天主教|法國天主教]][[玫瑰十字會|玫瑰十字會]]的[[創始人|創始人]]、[[神秘小說|神秘小說]][[作家|作家]]和[[藝術|藝術]][[評論家|評論家]][[約瑟芬·佩拉當|約瑟芬·佩拉當]]（{{tsl|en|Joséphin Péladan|Joséphin Péladan}}），其對此派系評價為「美是產生[[感情|感情]]讓[[觀念|觀念]][[昇華|昇華]]的歡樂」。

唯美主义也曾经受到来自杂志《笨拙》以及吉尔伯特和苏利文的小[[歌剧|歌剧]]《忍耐》的嘲讽。

== 唯美主义视觉艺术 ==
唯美主义思潮对[[室内设计|室内设计]]也产生了影响。唯美主义的[[室内设计师|室内设计师]]们喜欢以[[孔雀|孔雀]][[羽毛|羽毛]]和[[蓝|蓝]][[白|白]]相间的[[中国|中国]][[瓷器|瓷器]]作为装饰。

在英国，它发生于[[维多利亚时代|维多利亚时代]]晚期，大致从1868年延续至1901年。设计史研究中往往也将同时在美国出现的具有相似倾向的设计潮流归属于唯美运动。

有必要指出的是，尽管唯美运动和[[艺术与工艺运动|艺术与工艺运动]]有时候显得针锋相对，但二者并行发展，并且共同处于维多利亚时代的[[折衷主义|折衷主义]]气氛中，设计师之间常常颇有交往，彼此互相影响，有时候使用着共同的设计语言。唯美运动的核心思想认为，艺术的使命在于为人类提供感观上的愉悦，而非传递某种道德或情感上的信息。因此，唯美主义者拒绝接受[[约翰·拉斯金|约翰·拉斯金]]那种把艺术与道德相联系的观点。他们认为艺术或其他装饰不应具有任何说教的因素，提倡“为艺术而艺术”，如痴如醉地追求超然于生活的纯粹美，追求形式完美和艺术技巧。他们过着[[波希米亚主义|波希米亚主义]]（Bohemianism）的生活，即使贫穷，也热爱美甚于热爱生活。简言之，就是“唯美是求”。<ref name="测试">{{cite book zh |author=颜勇、黃虹 编著 |title=《西方设计：一部为生活制作艺术的历史》|pages=第212-222页 |location=长沙 |publisher=湖南科學技术出版社 |year=2010年 |ISBN=9787535764799}}</ref>
