《'''唯有妳'''》（{{lang-en|''I Run to You''}}）是美国乡村音乐团体[[怀旧女郎|怀旧女郎]]錄製的一首歌曲，于2009年1月发行。2009年11月11日，该歌曲获得了當年乡村音乐协会单曲奖。

== 詞曲 ==
《唯有妳》是一首中速乐曲，使用[[电吉他|电吉他]]和{{tsl|en|string section||部分弦乐}}伴奏。在其中，叙述者——主唱查尔斯·凯利（Charles Kelley）和希拉里·斯科特（Hillary Scott）表示，他们互相幫助来脫離世界消极和偏见的那一面。

==  音乐录像带 ==
歌曲的音乐录像带由Adam Boatman执导，于2009年3月发行。视频展示了一连串的事件，描绘了一个善行是如何诱发另一个善行的。在视频的开端，一个母亲给一个在人行道上弹吉他的男人钱。男人随后走进了一所咖啡店，并向一个把钱包落在家里的商人贷款。那个商人然后帮助一个不慎跌倒在地的女人捡起了散落在地的一叠纸。后来，那个女人坐在椅子上等候公共汽车。母亲从一开始就到达公共汽车站，她将自己的座位让给了女人。视频还录下了三位演唱者在小酒店的房间内唱歌时的场景。

2010年7月30日，为欧洲定做的第二版音乐录像带在[[Parlophone|Parlophone]]的[[YouTube|YouTube]]页面上首次亮相。<ref>[https://www.youtube.com/watch?v=uKo-RqEZcD4 YouTube - Parlophone - Lady Antebellum - I Run To You (Official)]</ref>该视频由克里斯托弗·锡姆斯（Christopher Sims）执导。<ref>[http://www.simsdirector.com/video?ladyantebellum_iruntoyou Christopher Sims - Director - Lady Antebellum - "I Run to You"] {{webarchive|url=https://web.archive.org/web/20110716072927/http://www.simsdirector.com/video?ladyantebellum_iruntoyou |date=2011-07-16 }}</ref>

==乐评==
Engine 145的布雷迪·弗彻（Brady Vercher）给了歌曲一个“踩”，将歌曲称为“爱的模糊信念”，并补充说“要麼是因為他們沒有提出好的理念，讓他们在歌词中舒暢地敞开（心扉），要麼是他們壓根沒有認真寫歌词，只是把這首歌的歌詞當做支撐旋律的一種手段。这首歌偶然間顯得有些故作神秘。”<ref name="engine145">{{cite web |url=http://www.engine145.com/lady-antebellum-i-run-to-you/ |title=Lady Antebellum - "I Run to You" |last=Vercher |first=Brady |date=2009-02-04 |accessdate=2009-02-21 |work=Engine 145 |deadurl=yes |archiveurl=https://web.archive.org/web/20141218104023/http://www.engine145.com/lady-antebellum-i-run-to-you/ |archivedate=2014-12-18 }}</ref>乡村世界（Country Universe）的丹·米利肯（Dan Milliken）将这首歌评为B级，他表示这首歌“甜蜜地進行着，聽不出比起本質更為重要的（事物）”。<ref>{{cite web|url=http://www.countryuniverse.net/2009/01/24/review-lady-antebellum-i-run-to-you/|title=Lady Antebelllum - "I Run to You"|last=Milliken|first=Dan|date=2009-01-24|work=Country Universe|accessdate=2009-03-13}}</ref>

{{tsl|en|Paul Worley|保罗·沃利|保罗·沃利}}和其他相关人员一起制作了专辑并为之弹了吉他，他表示“这首歌表达了對仇恨、偏见、消极和激烈竞争的反对，
而這些最终為爱所救赎！它拥有一种不可抗拒的旋律和一种令人摇头晃脑的律动。还有什么（比这个）更好的？”<ref name="liner"/>

== 榜单成绩 ==
唯有妳于2009年1月31日在美国《告示牌》的热门乡村音乐榜上首次亮相。在榜单上坚守了26个星期之后，这首歌于2009年7月25日当周成为怀旧女郎第一热门的单曲。这首歌也成了怀旧女郎在[[告示牌百强单曲榜|告示牌百强单曲榜]]上的首个40强，最高排名为第27名。在告示牌的2009年年终榜单中，唯有妳被排为了第一名。<ref>{{cite web|url=http://www.billboard.com/charts/year-end/2009/hot-country-songs|title=Year End Charts - Year-end Singles - Hot Country Songs|accessdate=2009-12-11|date=|publisher=Billboard.com}}</ref>
