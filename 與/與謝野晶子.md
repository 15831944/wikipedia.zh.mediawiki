'''與謝野 晶子'''（{{bd|1878年|12月7日|1942年|5月29日}}），是[[明治|明治]]至[[昭和|昭和]]時期活躍的[[詩人|詩人]]、[[作家|作家]]、[[思想家|思想家]]。舊姓「鳳」（ほう）。戶籍名為「志よう」。其[[筆名|筆名]]中的「晶」字，是取自「志よう（しょう）」。丈夫為[[與謝野鐵幹|與謝野鐵幹]]（本名與謝野寬）。她一生著述颇丰，被[[日本|日本]][[作家|作家]][[田边圣子|田边圣子]]评为：“一千年才出现一个的天才。” <ref name=" smgs" />

== 生平 ==
=== 读书生涯 ===
与谢野晶子出生于[[大阪|大阪]][[堺市|堺市]]，家里經營日式和果子羊羹老舖。<ref name=" smgs" >{{cite web |url= http://www.jnocnews.jp/news/show.aspx?id=28492|title= 明治新女性——与谢野晶子|accessdate= 2009年|work=|publisher=日本新华侨报网 |language = zh-hans}}</ref> 父亲[[凤宗七|凤宗七]]和母亲生了三兄妹，二哥夭折。<ref name=" smgs" /> 与谢野晶子出生后被送到[[姨妈|姨妈]]家养育了三年。<ref name=" smgs" /> 接回家后被父亲视为男童养育，四岁就送到小学，因其发蒙尚不成熟，六岁再次送到小学。<ref name=" smgs" /> 小学毕业考入[[京都府立第一女子高中|京都府立第一女子高中]]。<ref name=" smgs" /> 她因为不满学校的课程老旧，自己在家里帮父母做事，家中又藏书甚丰，于是边工边读。<ref name=" smgs" /> 在此期间，她饱读了[[清少纳言|清少纳言]]、[[紫式部|紫式部]]、[[尾崎红叶|尾崎红叶]]、[[幸田露伴|幸田露伴]]、[[樋口一叶|樋口一叶]]等日本文坛大家的作品以及[[中国古典文学|中国古典文学]]作品，[[白居易|白居易]]是其中之一。<ref name=" smgs" />

=== 创作生涯 ===
16岁起，她开始在各大刊物发表文章，作品多数为描写男欢女爱。<ref name=" smgs" /> 

[[明治|明治]]32年，[[与谢野铁干|与谢野铁干]]在[[东京都|东京都]]成立了新诗社，创办《[[明星|明星]]》刊物，与谢野晶子的作品多数被《明星》选用发表。<ref name=" smgs" />

== 爱情婚姻 ==
[[明治|明治]]33年8月，[[與謝野鐵幹|與謝野鐵幹]]为了扩大《[[明星|明星]]》的影响力，去[[关西|关西]]发表演讲，和与谢野晶子多次见面，两人心生爱慕，私定终身，不过此时与谢野铁幹和前妻尚未解决离婚问题。<ref name=" smgs" />

明治34年6月，两人[[同居|同居]]。<ref name=" smgs" /> 两人的事迹很快传遍文坛，《[[明星|明星]]》发表了与谢野晶子的和歌集《[[乱发|乱发]]》，两件事一同被社会[[保守|保守]]人士借口“卖妻”、“[[强奸|强奸]]”、“蛊惑[[处女|处女]]发疯”等罪名大张挞伐，并直接导致了《[[明星|明星]]》被停刊。<ref name=" smgs" />

== 文学风格 ==
与谢野晶子的早期作品抨击了[[日本|日本]][[旧社会|旧社会]]的虚伪[[道德|道德]]，强烈呼唤真爱真情和[[自由恋爱|自由恋爱]]。<ref name=" smgs" /> 多数讴歌了[[肉体|肉体]][[感官|感官]]之美，深切表现了少女情窦初开的[[思春|思春]]情态。<ref name=" smgs" /> 如《[[善恶草|善恶草]]》的[[思春|思春]]歌<ref name=" smgs" /> ：{{quote|讲道君子哟，<br />柔嫩肌肤空在身，<br />热血徒澎湃，<br />无人触摸无人爱，<br />人不亦寂寞难捱？}} 后期在[[日俄战争|日俄战争]]中的作品则表现了[[反战|反战]]和关注个人生命的主题。<ref name=" smgs" /> 如写给弟弟的作品《君勿死去》<ref name=" smgs" /> ：{{quote|叹身处旅顺包围军中之弟<br />你呀！不要死！<br />圣上自己不出征，却叫别人的孩子去流血，去为野蛮杀人而送命，还要说这种死光荣！<br />人都说圣上慈悲为怀，可这件事又怎能叫人想得开通？
}}
