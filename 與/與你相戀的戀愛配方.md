{{日本作品|與你相戀的戀愛配方|あなたに恋する恋愛ルセット}}是UNiSONSHiFT在2017年5月26日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=http://moedigi.com/game/2017/04/unisonshift-rusetto-master-up/|title=お菓子が結ぶ、あまくてあまーい、ラブストーリー。『あなたに恋する恋愛ルセット』がマスターアップ|website=moedigi|date=2017-04-29|accessdate=2019-04-06|language=ja}}</ref>。

== 故事簡介 ==
私立六華星學園有著各式各樣的專門課程，男主角二宮春馬從小就喜歡甜點因此決定選擇製菓學科，在課程中春馬認識了同樣喜歡製作甜點的女孩們。隨著時間經過，春馬除了精進了自己的技術外，更和女孩們變得要好<ref>{{cite web|url=https://www.tgsmart.jp/article.aspx?a=19607|title=【発売直前！】ユニゾンシフトの最新作は"お砂糖200％増し"がコンセプトのイチャラブADV！|website=TG Smart|date=2017-05-19|accessdate=2019-04-06|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|二宮春馬|二宮 春馬（にのみや はるま）}}
: 作品男主角，非常喜歡甜點而選擇了製菓學科，雖然沒有製作甜點的經驗，但熟悉甜點的知識。
; {{nihongo|橘乃乃香|橘 ののか（たちばな ののか）|聲：{{lang|ja|みたかりん}}<ref name="MAIN_CHARA">{{cite web|url=http://unisonshift.amusecraft.com/products/project31/character01.html|title=MAIN CHARA|publisher=UNiSONSHiFT|accessdate=2019年4月23日|language=ja}}</ref>}}
: 蛋糕店的女兒，個性活潑，當認真做一件事時便會全心投入，尤其對於甜點更是如此。喜歡看少女漫畫，對戀愛有所憧憬。
; {{nihongo|大園柚姬|大園 柚姫（おおぞの ゆうき）|聲：[[風音|風音]]<ref name="MAIN_CHARA"/>}}
: 出身名家的大小姐，個性不驕傲，對任何人都很溫柔。因為知道製菓學科的花梨老師相當有名而選擇該課程，實際上並無製作甜點的經驗。
; {{nihongo|鍵由楓花|鍵由 楓花（かぎよし ふうか）|聲：和葉<ref name="MAIN_CHARA"/>}}
: 和菓子老店「鍵屋」的女兒，對人很有禮貌，總是散發著悠閒自在的氣質，為了擴充自己的眼界選擇製菓學科。
; {{nihongo|白咲美繪瑠|白咲 美絵瑠（しろさき みえる）|聲：[[Miru|Miru]]<ref name="MAIN_CHARA"/>}}
: 學園長的女兒，因為怕麻煩而經常翹課。實際上擁有靈敏的味覺，只要吃一口食物便能給予評價。

=== 其他角色 ===
; {{nihongo|花梨老師|かりん先生|聲：[[遠野Soyogi|遠野Soyogi]]<ref name="SUB_CHARA">{{cite web|url=http://unisonshift.amusecraft.com/products/project31/character02.html|title=SUB CHARA|publisher=UNiSONSHiFT|accessdate=2019年4月23日|language=ja}}</ref>}}
: 製菓學科的講師，以前曾在法國留學並在有名的糕點店擔任西點師傅，有著極為高明的技術。經常做女性打扮但實際是男性。
; {{nihongo|白咲莉賽特|白咲 リゼット（しろさき リゼット）|聲：奏雨<ref name="SUB_CHARA"/>}}
: 私立六華星學園的學園長，美繪瑠的母親。性格溫厚且經常帶著笑容，但生氣時非常可怕。
