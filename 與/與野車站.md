'''與野車站'''（{{jpn|j=与野駅|hg=よのえき|rm=Yono eki|f=y}}）位於[[日本|日本]][[埼玉縣|埼玉縣]][[埼玉市|埼玉市]][[浦和區|浦和區]]{{link-ja|上木崎|上木崎}}一丁目，是[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）的[[鐵路車站|鐵路車站]]。

== 停站路線 ==
此站停靠的路線在軌道名稱上有[[東北本線|東北本線]]與[[武藏野線|武藏野線]]（大宮支線）（詳情參見路線條目與{{link-ja|鐵道路線的名稱|鉄道路線の名称}}），與書類上的分歧站。與旅客列車有關、只有行走東北本線的[[電車線與列車線|電車線]]的[[京濱東北線|京濱東北線]]列車停靠，旅客導覽不顯示「東北（本）線」。

== 車站構造 ==
本站是一地面車站，月台為[[島式月台|島式月台]]1面2線設計，站房則是[[跨站式站房|跨站式站房]]設計。由於站長室位於東口側，故本站於大部分京濱東北線車站不同，是以北行線月台為1號月台。

== 历史 ==
* 1912年（[[大正|大正]]元年）11月1日：應當地（原與野町）居民要求，將浦和－大宮段中間的大原號誌站（1906年（[[明治|明治]]39年）4月16日增設）升格為'''与野站'''。開業時，為方便居民使用，只於面向與野町的西側設有出口（其後才增設東口），是少數沒有面向原主要道路（本站東側為中山道）的出口的車站。
* 1973年（[[昭和|昭和]]48年）4月1日：武藏野線貨物支線開業。
* 1987年（昭和62年）4月1日：[[國鐵分割民營化|國鐵分割民營化]]，本站由JR東日本接手經營。
* 2001年（[[平成|平成]]13年）11月18日：智能卡系統[[Suica|Suica]]正式投入服務。
