《'''與僧侶交合的色慾之夜'''》（{{lang-ja|僧侶と交わる色欲の夜に…}}）是{{lang|ja|真臣レオン}}的[[日本|日本]][[漫畫|漫畫]]作品，在ComicFesta等漫畫網站連載，星雲社的Clair TL Comics已出版4冊[[單行本|單行本]]。

== 故事簡介 ==
深谷美樱在参加[[同学会|同学会]]时，与曾经的初恋对象九条重逢了。然而九条脱了帽子之后，美樱才发现他是个[[光头|光头]]，原来他继承了家族的[[寺庙|寺庙]]成为了[[僧侶|僧侶]]。美樱原本以为僧侶应该不能[[恋爱|恋爱]]、不能[[结婚|结婚]]，然而九条却与她发生了[[性关系|性关系]]。

== 登場角色 ==
; {{nihongo|深谷美櫻|深谷美桜|聲：[[原舞香|原舞香]]}}
: 大學4年生。初中時暗戀着隆秀。努力的直率性格。對隆秀感情覺得迷惑。
; {{nihongo|九條隆秀|九条隆秀|聲：[[鈴城葉|鈴城葉]]}}
: 美櫻初中時代的初戀對象。現在繼承自家的寺院，成為僧侶。責任感很強，但在戀愛方面很笨拙。
; {{nihongo|九條雪隆|九条雪隆|聲：[[恋津田蓮也|恋津田蓮也]]}}
: 隆秀的弟弟。離開了家，和美櫻就讀於同所大學。開朗而飄飄然的性格。
; {{nihongo|九條清隆|九条清隆|聲：{{lang|ja|狂々パーマ}}}}
: 隆秀的父親，寺院的住持。性格沉穩。
; {{nihongo|九條栞|九条栞|聲：{{link-ja|秋山ねりね}}}}
: 隆秀的母親。喜歡可愛的東西。
