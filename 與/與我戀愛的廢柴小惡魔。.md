{{日本作品|與我戀愛的廢柴小惡魔。|僕と恋するポンコツアクマ。}}是SMILE在2015年9月25日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=https://www.game-style.jp/special/backnumber/201508/21/00pon_sm.php|title=Game-Style「與我戀愛的廢柴小惡魔。」特集|website=Game-Style|date=2015-08-21|accessdate=2018-10-11|language=ja}}</ref>。2016年5月27日<ref>{{Cite web|url=https://www.game-style.jp/monitor/201604/14/02pse_sm.php|title=Game-Style「與我戀愛的廢柴小惡魔。超級H」特集|publisher=Game-Style|date=2016-04-14|accessdate=2018-10-11|language=ja}}</ref>發售[[Fan_disc|Fan disc]]《與我戀愛的廢柴小惡魔。超級H！》（{{lang|ja|僕と恋するポンコツアクマ。すっごいえっち！}}）。2018年10月27日在[[Steam|Steam]]平台發行。

== 故事簡介 ==
桐谷梨梨愛和母親桐谷怜香吵架後，被怜香要求搬到學園的女子宿舍生活，而和梨梨愛住在同個屋簷下的藤真蓮與妹妹藤真咲莉也被命令搬到宿舍。到了宿舍後，蓮知曉了宿舍的房客都不是普通人類，甚至連青梅竹馬梨梨愛跟妹妹咲莉也都分別是夢魔和天使。而蓮住在這個宿舍的目的便是要幫助這些女孩矯正不足的性格<ref>{{cite web|url=https://www.game-style.jp/soft/201508/07/01pon_sm.php|title=欠点だらけのポンコツアクマっ娘たちを、濃密な愛とエッチで更生させろ！ 『僕と恋するポンコツアクマ。』9月25日発売！ |website=Game-Style|date=2015-08-07|accessdate=2018-10-11|language=ja}}</ref>。

== 登場角色 ==
; {{nihongo|藤真蓮|藤真 蓮（ふじま れん）}}
: 作品男主角，就讀青耶馬學園二年級。和妹妹咲莉兩個人一起住在梨梨愛家。
; {{nihongo|桐谷梨梨愛|桐谷 梨々愛（きりたに りりあ）|聲：[[小倉結衣|小倉結衣]]}}
: 蓮的青梅竹馬，就讀青耶馬學園二年級。實際身分是[[夢魔|夢魔]]，喜歡看少女漫畫。
; {{nihongo|藤真咲莉|藤真 咲莉（ふじま えみり）|聲：{{lang|ja|橘まお}}}}
: 蓮沒有血緣關係的妹妹，就讀青耶馬學園一年級。[[情結#日本次文化|兄控]]。實際身分是[[天使|天使]]。年幼時曾當過偶像。
; {{nihongo|綾崎優|綾崎 優（あやさき ゆう）|聲：{{lang|ja|歩サラ}}}}
: 轉學生少女，就讀青耶馬學園二年級。實際身分是[[雪女|雪女]]。個性寡言且缺乏感情變化，非常怕冷。
; {{nihongo|有瀨川阿爾爾|有瀬川 アルル（ありせがわ アルル）|聲：上田朱音}}
: 蓮的學妹，就讀青耶馬學園一年級。實際身分是[[木乃伊|木乃伊]]。不喜歡穿衣服，因此很羨慕能以自然姿態生活的動物。
; {{nihongo|葵雅妃|葵 雅妃（あおい みやび）|聲：{{lang|ja|手塚りょうこ}}}}
: 蓮的學姐，就讀青耶馬學園三年級。實際身分是[[魔女|魔女]]。未來的夢想是成為藥劑師，但因為不擅長用功讀書而被雙親反對。
: 曾經當過讀者模特兒，在校內非常有人氣。
; {{nihongo|柴崎凜|柴崎 凛（しばさき りん）|聲：{{lang|ja|藤咲ウサ}}}}
: 蓮的學妹，就讀青耶馬學園一年級。實際身分是虎女。
; {{nihongo|龍泉寺愛花|竜泉寺 愛花（りゅうせんじ あいふぁ）|聲：[[白雪碧|白雪碧]]}}
: 蓮的班導師，實際身分是龍族的女教師。
; {{nihongo|桐谷怜香|桐谷 怜香（きりたに れいか）|聲：[[川島梨乃|川島梨乃]]}}
: 梨梨愛的母親，青耶馬學園的學園長。
