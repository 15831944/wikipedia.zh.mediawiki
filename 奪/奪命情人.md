{{engname|《奪命情人》|'' Deadly Lovers ''}}是1980年代由香港[[電視廣播有限公司|電視廣播有限公司]]出品的一部[[電視電影|電視電影]]，由[[黎明|黎明]]、[[劉美君|劉美君]]、[[邵仲衡|邵仲衡]]、[[杜麗莎|杜麗莎]]等演出，於1988年進行拍攝及製作，1989年3月12日在[[無綫電視|無綫電視]][[翡翠台|翡翠台]]播出<ref>{{Cite web |url= http://hkmdb.com/db/movies/view.mhtml?id=12420&display_set=big5|title =
奪命情人（1988）| publisher =香港影庫|language = 中文}}</ref><ref>{{Cite web |url= https://www.hkfact.com/movie/20079|title =電影－奪命情人| publisher =香港電影反刪減小組|language = 中文}}</ref>。

==劇情介紹==
故事講述韋德明是一名律師，與妻子玉育有一名兒子子輝，一家三口生活安定。某天，龍突然從外國回港，並以玉的裸照向玉勒索。原來龍是玉的舊情人，兩人以前是不良少年，曾拍下裸照，後來龍因為犯了事而逃走到外國。玉擔心裸照的事會影響韋德明的聲譽，更害怕韋德明知道她的往事而拋棄她，唯有接受龍的勒索。龍向黑幫尋仇失敗，反被索償，再次向玉勒索，玉走投無路，唯有向韋德明說出真相，韋德明知悉事件後非常憤怒，帶著兒子離開玉，此時龍駕車想要撞向韋德明，玉的親親前來找龍算帳，卻被龍殺死。警方發出通緝令追捕龍，龍變得精神失常，誓殺害韋德明一家三口，眾人展開了一段驚險的故事<ref>{{Cite web |url = http://ent.163.com/05/1109/17/224R9N4H00031MF2.html |title =黎明早期演繹的角色形象的幾大類別|author =王曉易|date =2005年11月9日|publisher =網易娛樂|language =中文}}</ref>。
