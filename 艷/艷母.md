《'''艷母'''》（{{lang-ja|艶母}}）是[[雅綴|雅綴]]在1998年連載的[[日本|日本]][[成人漫畫|成人漫畫]]，並在2003年推出[[OVA|OVA]][[成人動畫|成人動畫]]系列。

== 登場角色 ==
; {{nihongo|雨宮美沙子|雨宮 美沙子（あまみや みさこ）|聲：[[二見渚|二見渚]]}}
: 洋介的妻子，一彥的繼母。
; {{nihongo|悅美子|悦美子（えみこ）|聲：[[御影由宇|御影由宇]]}}
: 美沙子的妹妹。落入雨宮一彥的陷阱而得知他和美沙子那扭曲的關係。
; {{nihongo|雨宮一彦|雨宮 一彦（あまみや かずひこ）|聲：平井真澄 / [[平井達矢|平井達矢]]}}
: 洋介的兒子。
; {{nihongo|雨宮洋介|雨宮 洋介（あまみや ようすけ）|聲：Back麗男}}
: 美沙子的丈夫。
