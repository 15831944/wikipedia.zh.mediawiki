[[教宗|教宗]]'''依諾增爵八世'''（{{lang-la|Innocentius PP. VIII}}，{{bd|1432年||1492年|7月25日|catIdx=I}}），又翻譯'''諾森八世'''，原名Giovanni Battista Cibo，[[意大利|意大利]]人，於1484年8月29日—1492年7月25日岀任教宗。

== 早年 ==
諾森八世出生于義大利[[热那亚|热那亚]](Genova)，1467年被提升为[[薩沃納省|薩沃納省]](Savona)主教，1472年出任[[莫尔费塔|莫尔费塔]](Molfetta)大主教，1473年被提升为[[樞機|樞機]]。

== 在位期间 ==
[[思道四世|思道四世]]死后，枢机会议内部派别争斗非常严重，[[罗马|罗马]]街头甚至爆发动乱，在諾森八世当选前夜，他在[[教宗選舉|教宗選舉]]中[[賄選|賄選]]才得以当选。他本人身体多病，又没有多少威信，因此他当任时[[天主教会|天主教会]]的政治主要是由他的前任思道四世的侄子[[朱利安诺·德拉·罗韦雷|朱利安诺·德拉·罗韦雷]]来操纵。后来罗韦雷自己也当选教宗，是为[[儒略二世|儒略二世]]。

諾森八世当选后于1484年发表了一部[[教宗诏书|教宗诏书]]，尤其激起了在[[德意志|德意志]]的[[獵巫運動|獵巫運動]]的迫害，和加强了[[西班牙|西班牙]]的[[宗教裁判所|宗教裁判所]]。

同时他还呼吁组织[[十字军|十字军]]，但实际上为了改善他的[[经济|经济]]状况，他与[[奥斯曼帝国|奥斯曼帝国]][[苏丹_(称谓)|苏丹]][[巴耶塞特二世|巴耶塞特二世]]的关系不错。他扣押了巴耶塞特二世的兄弟和最强的竞争者[[傑姆蘇丹|傑姆蘇丹]]，为此巴耶塞特二世每年向他进贡（估计四万個[[杜卡特|杜卡特]]），还交给諾森八世传说中刺入[[耶稣|耶稣]]胁部的[[命运之矛|命运之矛]]。

虽然如此，由于諾森八世生活奢侈腐败，他依然常常缺钱，有时他甚至[[典当|典当]]他的教宗皇冠和[[教廷|教廷]]的宝藏。

在[[意大利|意大利]]内部的政治上他与[[那不勒斯|那不勒斯]]国王的冲突尤其严重。军事力量强大的那不勒斯国王拒絕向諾森八世交税，而法国国王[[查理八世_(法兰西)|查理八世]]又违约不肯帮助依諾增爵出兵。1486年諾森八世只好与那不勒斯谈和，但是此后不久那不勒斯又撕毁了和约。最后1492年諾森八世的儿子与[[美第奇家族|美第奇家族]]通婚，同时他将他的一个孙女嫁给那不勒斯国王的一个叔叔才平息了这场战争。

== 其他 ==

諾森八世有众多儿女，当时罗马的一首歌嘲笑地说：“他产了八个无用之子，以及同样多的女儿，他真的可以被称为'''罗马之父'''”（拉丁语：Octo nocens pueros genuit, totidemque puellas; hunc merito poterit dicere Roma patrem）
'''罗马之父'''是[[教皇|教皇]]之意，[[雙關|雙關]]語。他也大量使用[[裙带关系|裙带关系]]，为自己的亲人谋权图富。

諾森八世有许多非常激烈的批评者。其中最有势力的是[[多明我会|多明我会]]教士[[薩佛納羅拉|薩佛納羅拉]]，萨佛纳羅拉非常激烈地批评[[教宗国|教宗国]]的腐败和奢侈。由于萨佛纳羅拉正确地预言了諾森八世的死日，这使得他获得了许多追随者。

当时的一本反教宗的[[野史|野史]]《[[罗马城日记|罗马城日记]]》（Diarium urbis Romae）记载了许多关于諾森八世的[[轶事|轶事]]，其中包括他死前命人割开三个十岁的童男的[[动脉|动脉]]来为他[[输血|输血]]以获取这些童男的青春。但是这些报道没有第三资料的证实。
