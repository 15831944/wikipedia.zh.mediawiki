[[物理学|物理学]]中，'''物質波'''（即'''德布羅意波'''）係指所有[[物質|物質]]的[[波|波]]（见[[波粒二象性|波粒二象性]]）。

德布羅意說明了[[波長|波長]]和[[動量|動量]]成反比；[[頻率|頻率]]和[[總能|總能]]成正比之關係，是[[路易·德布羅意|路易·德布羅意]]於1923年在他的博士論文提出的。

第一德布羅意方程指出，粒子波長λ（亦稱「德布羅意波長」）和動量p的關係：（下式中[[普朗克常數|普朗克常數]]h、粒子[[靜質量|靜質量]]m、粒子[[速度|速度]]v、[[勞侖茲因子|勞侖茲因子]]γ和[[真空|真空]][[光速|光速]]c）
: <math>\lambda = \frac{h}{p} = \frac{h}{\gamma mv} = \frac{h}{mv} \sqrt{1 - \frac{v^2}{c^2} }</math>

第二德布羅意方程指出頻率ν和總能E的關係：
: <math>\nu = \frac{E}{h} = \frac{\gamma\,mc^2}{h} = \frac {1}{\sqrt{1 - \frac{v^2}{c^2}}} \cdot \frac{mc^2}{h}</math>

這兩個式子通常寫作
: <math>p = {h \over \lambda} = \hbar {2\pi \over \lambda} = \hbar k \,</math>
: <math>E = h \nu = \hbar \cdot 2\pi \nu = \hbar \omega \,</math>

== 實驗證明 ==
1927年，[[克林頓·戴維森|克林頓·戴維森]]與[[雷斯特·革末|雷斯特·革末]]在[[貝爾實驗室|貝爾實驗室]]將[[電子|電子]]射向[[鎳|鎳]]結晶 ，發現其[[繞射|繞射]]圖譜和後來[[基本粒子|基本粒子]]也被證實有波的性質。1999年，[[富勒烯|富勒烯]]被測出有波的性質。<ref>http://www.univie.ac.at/qfp/research/matterwave/c60/index.html</ref>

== 大型物件的波長 ==
理論上，不只亞原子粒子有波的性質。

例如：投球手以40米每秒投出一個質量為0.15公斤的棒球。這個球的波長為

: <math>\lambda = \frac {6.626 \times 10^{-34} \mbox{ kg} \cdot \mathrm{m}^2  / \mathrm{s}}{0.15 \ \mathrm{kg} \times 40 \ \mathrm{m/s}} = 1.10 \times 10^{-34} \ \mathrm{m} </math>

這比原子核的直徑10<sup>−15</sup>米更小，直趨[[普朗克長度|普朗克長度]]10<sup>−35</sup>。因此，現時的技術是無法觀察出其波動性質的。
