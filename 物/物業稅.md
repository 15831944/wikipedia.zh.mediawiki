'''物業稅'''是[[政府|政府]]向[[地產|地產]]物業徵收的一種[[財產稅|財產稅]]，通常向物業的業主或租戶等使用者徵收。負責徵收物業稅的政府機構可能會對物業進行估值，並以物業價值的一個[[百分比|百分比]]作為應繳的物業稅額。世界各國徵收物業稅的方式及稅率各有不同。

== 中國的物業稅 ==
在[[中國大陸|中國大陸]]，物業稅是一種新生事物，它跟現行[[房產稅|房產稅]]、[[城市房地產稅|城市房地產稅]]、[[土地增值稅|土地增值稅]]以及[[土地出讓金|土地出讓金]]等都各有異，也有同，有繁多稅種，而物業稅定義還在學術探討階段。<ref>{{cite news|title=中国征物业税关注三问题 可能买得起房住不起|url=http://www.heb.chinanews.com.cn/news/fc/2005-11-25/118.html|website=中国新闻网-河北新闻|accessdate=2017-01-19|date=2005年11月25日}}</ref>

== 香港的物業稅 ==
在[[香港|香港]]，也有一種稅項名為物業稅（英文也名叫Property Tax）。

根據香港《[[香港稅務條例|稅務條例]]》第112章，第5B條《物業稅的徵收》，它是一種[[直接稅|直接稅]]的[[入息稅|入息稅]]，業主只是持有香港物業是不會被課稅，除非物業被租出而產生的代價收入，是課稅年度該代價收入的八成淨金額（即是“應評稅淨值”）將被課物業稅。另外，香港還有[[香港差餉|差餉]]、[[香港地租|地稅或地租]]（視乎適用條例）兩種物業稅項。

== 台灣的物業稅 ==
在[[中華民國|中華民國]]，包括[[房屋稅|房屋稅]]、[[地價稅|地價稅]]、[[土地增值稅|土地增值稅]]、契稅以及[[奢侈稅|奢侈稅]]等。

=== 房屋稅 ===
房屋稅在中華民國屬於地方稅，由地方政府課徵，房屋稅是對房屋所有人在持有期間所課徵的財產稅，課稅範圍包括房屋及附屬於該房屋並增加其使用價值之建築物。所謂的房屋，除一般通稱的房屋外，尚包含其他形狀特殊而供住宅、工作、營業等固定於土地上的建築物在內，如夾層房屋及各種形狀之散裝倉庫、油槽及加油亭等均是，稅率根據用途(自住、住家用途、營業用途)，以及持有戶數而有所不同。

<code>
房屋稅 = 房屋課稅現值 * 稅率
    房屋課稅現值 = 核定單價 × 面積 ×（1 - 折舊率 × 折舊年數）× 街路等級調整率（地段率）
　　核定單價 = 標準單價 ×（1 ± 各加減項之加減率）± 樓層高度之超高或偏低價  
</code>

== 美国的房產稅 ==
每年需要缴纳的基本税费有：房产税、物业费、房屋保险费，且因为房屋所在的地区、房屋物业类型等因素而有不同。
*房地产税（Property Tax或Real Estate Tax），也称为物业税，或简称为地税，是地方针对房地产征收的郡税，市税，学区税的总称。美国各个州税率不同，在美国，只要你拥有房子，你就要交纳房地产税，在美国税率通常为1-3%，[[加州|加州]]大约在1.2-2.0%之间。[[乔治亚州|乔治亚州]]是税比较低的地区，[[亚特兰大|亚特兰大]]为例——每年约是房产估价的1%。
**房產稅最低的是Alabama--AL的0.32%
**房產稅最高的是Texas--TX的1.76%
*房屋保险，一般的区域，一栋三十万到五十万的房子，它的房屋保险费每年通常在us$600到$900之间。
*社区费（如果是住宅大廈則称管理费、物业费）
*庭院的维护：一些地方政府会限制住宅前方草坪的高度、淸潔
