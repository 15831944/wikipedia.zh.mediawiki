'''逐次逼近模拟数字转换器'''（{{lang-en|'''Successive approximation ADC/SAR ADC'''}}）<br>
逐次逼近模拟数字转换器[[模拟数字轉換器|模拟数字轉換器]]領域裏的其中一種分類。<br><br>
標準的逐次逼近模拟数字转换器架構包含：<br><br>
1. [[計數器|計數器]]。<br>
2. [[数字模拟轉換器|数字模拟轉換器]]。<br>
3. [[電壓比較器|電壓比較器]]，对数字模拟转换器输出的电压与输入电压进行比较。<br>
4. 存放转换结果的[[寄存器|寄存器]]。<br>
<br>
== 原理 ==
（以 12-bit 逐次逼近模拟数字转换器為例）：<br>
1. 计数器复位。<br>
2. 使得转换结果寄存器输出为转换器分辨率的一半，对 12-bit SAR ADC而言即为2048。<br>
3. 将转换结果寄存器的内容送给[[数字模拟轉換器|数字模拟轉換器]]。<br>
4. 倘若输入电压高于数字模拟转换器输出的电压，则转换结果寄存器的值加上 2^(转换器分辨率 - 计数器值)，倘若输入电压低于数字模拟转换器输出的电压，则将转换结果寄存器的值减去 2^(转换器分辨率 - 计数器值)。<br>
5. 计数器加一，重复3、4、5。当计数器达到 12（转换位数时），转换完成。转换结果寄存器即存储了[[量化|量化]]后的数字代码。<br>
<br>
由此可见，SAR ADC的转换次数与设计分辨率有关。如此转换器要达到 1MSPS 的转换速度时，至少需要 12MHz 的时钟才能够驱动。<br>

== 優點 ==
在[[IC|IC]]設計裏，使用面積較小，功率消耗相对較低。<br>

== 缺點 ==
1. 轉換速度相对较慢。<br>
2. 由于一次转换需要多次比较，因此对[[数字模拟轉換器|数字模拟轉換器]]速度、精度要求较高。<br>
