在[[趣味數學|趣味數學]]中，'''循環單位'''是由1組成的數如1, 11, 111, 1111等。

1966年，A.H. Beiler稱這類數為'''repunit'''，表示''rep''eated ''unit''。

對於n≥1，循環單位可以這樣定義：
:<math>R_n^{(b)}={b^n-1\over b-1}\qquad\,\!</math>

亦可以用[[遞歸|遞歸]]的方法：
:<math>R_0=0\,\!</math>
:<math>R_n=bR_{n-1}+1\,\!</math>

其中<math>b\,\!</math>是[[进位制|进位制]]的底。在這篇文章，循環單位都是指[[十进制|十进制]]中的。

==循環單位的平方==
<math>R_1\,\!</math>至<math>R_b\,\!</math>的循環單位，<math>R_n\,\!</math>的[[平方|平方]]有一個很有趣的性質，它們都會得出由1到<math>n\,\!</math>的數字順序組成的[[回文数|回文数]]。例如十进制中的：
<pre>
        1×1        =        1
       11×11       =       121
      111×111      =      12321
     1111×1111     =     1234321
    11111×11111    =    123454321
   111111×111111   =   12345654321
  1111111×1111111  =  1234567654321
 11111111×11111111 = 123456787654321
111111111×111111111=12345678987654321
</pre>

而上述原則於十進制，只在<math>n<10\,\!</math>的情況下才能生效，因為在<math>n>9\,\!</math>的情況下，<math>R_n\,\!</math>的平方已經不能組成迴文數。例如：
<pre>
      11111111111×1111111111      =      1234567900987654321
     111111111111×11111111111     =     123456790120987654321
    1111111111111×111111111111    =    12345679012320987654321
   11111111111111×1111111111111   =   1234567901234320987654321
  111111111111111×11111111111111  =  123456790123454320987654321
 1111111111111111×111111111111111 = 12345679012345654320987654321
11111111111111111×1111111111111111=1234567901234567654320987654321
...
</pre>
雖然在<math>9<n<19\,\!</math>的情況下，<math>R_n\,\!</math>的平方不能組成迴文數，卻有著固定的結構：

#如果<math>n=10\,\!</math>，前綴：123456790，後綴：0987654321
#如果<math>n>10\,\!</math>，前綴：123456790，中段：從1開始順序數數，直至得出<math>n\,\!</math>與9的差，再倒數至2，後綴：0987654321

==循環單位兼質數==
當<math>n</math>能被大於1的<math>k</math>整除時，<math>R_k|R_n</math>（例如<math>111111111 = 111 \times 1001001</math>），因此若<math>R_n</math>是質數，<math>n</math>必須是質數。

現在已知<math>n = 2, 19, 23, 317, 1031</math>時，<math>R_n</math>是質數，而<math>n = 49081, 86453</math>的<math>R_n</math>則可能是[[偽素數|偽素數]]。
