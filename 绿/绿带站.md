'''绿带站'''（{{lang-en|Greenbelt station}}）是一個位於[[美國|美國]][[马里兰州|马里兰州]][[乔治王子县_(马里兰州)|乔治王子县]]的一座[[华盛顿地铁|华盛顿地铁]][[华盛顿地铁绿线|绿线]]和[[马里兰区域通勤铁路|马里兰区域通勤铁路]]的鐵路車站。该站是[[华盛顿地铁|华盛顿地铁]][[华盛顿地铁绿线|绿线]]的东北终点站。

该站位于[[绿带城|绿带城]]的西北边缘（在[[伯温高地|伯温高地]]，[[贝兹维尔|贝兹维尔]]和[[大学公园市|大学公园市]]的北部附近），就在樱桃木巷（{{lang-en|Cherrywood Lane}}）旁边，在[[495号州际公路_(首都环线)|首都环线]]附近。它的停车场有超过3,300个空位<ref>{{Cite web |url=https://www.wmata.com/service/parking/parking-details.cfm?stationid=80 |title=停车：绿带站 |website=华盛顿都会区交通局 |access-date=2019年1月25日}}</ref>。该站也有方便的[[495号州际公路_(首都环线)|环线]]入口和出口。绿带站被本地人和通勤族使用。另外，到[[巴尔的摩/华盛顿瑟古德·馬歇爾国际机场|巴尔的摩/华盛顿国际机场]]的[[Metrobus_(Washington,_D.C.)|都城巴士]]B30平日快线让通勤族连接到[[巴尔的摩|巴尔的摩]]的地区运输服务。绿带站，为[[美国国家航空航天局|美国国家航空航天局]][[戈达德太空飞行中心|戈达德太空飞行中心]]的雇员和客人，是最方便的站。通勤族可以上[[乔治王子郡公车|乔治王子郡公车]]15X线<ref>{{Cite web |url=https://www.princegeorgescountymd.gov/DocumentCenter/View/4127/Route-15X-Map-and-Schedule?bidId= |title=15X线 |website=乔治王子县官方网站 |access-date=2019年1月25日}}</ref>或者[[Metrobus_(Washington,_D.C.)|都城巴士]]G14线<ref>{{Cite web |url=https://www.wmata.com/rider-guide/stations/upload/station_bus_maps/pdfs/Greenbelt.pdf |title=从绿带站的公车服务 |website=华盛顿都会区交通局 |access-date=2019年1月25日}}</ref>去[[戈达德太空飞行中心|戈达德中心]]。

==马里兰区域通勤铁路列车服务==
[[Image:GreenbeltMARCstation.JPG|thumb]]月台 (朝南)]]

[[马里兰区域通勤铁路|马里兰区域通勤铁路]]列车，在[[康登线|坎登线]]上，停靠于绿带车站。[[马里兰区域通勤铁路|通勤铁路]]的轨道跟[[华盛顿地铁|华盛顿地铁]]的轨道平行。该车站有两个[[侧式月台|侧式月台]]。它也是对残疾人易接近的，但是没有各自大楼，厕所，电话，售票机或暖气，而没有人员。[[康登线|坎登线]]连接[[华盛顿特区|华盛顿特区]]的[[华盛顿联合车站|联合车站]]跟[[巴尔的摩|巴尔的摩]]的[[坎登车站|坎登车站]]。

==历史==
[[华盛顿地铁|地铁]]服务到[[绿带城|绿带城]]开始于1993年12月11日。同时，另三个站也正好于[[乔治王子县|乔治王子县]]开始服务：[[大学公园-马里兰大学站|大学公园站]]，[[乔治王子广场站|乔治王子广场站]]和[[西海恩兹维尔站|西海恩兹维尔站]]。到[[乔治王子县|乔治王子县]]的延伸增加了7.96英里（12.81公里）的轨道到[[华盛顿地铁|华盛顿地铁]]捷运系统。目前[[华盛顿地铁|华盛顿都会区交通局]]有初步计划一直延伸绿线，从绿带站到[[劳雷尔|劳雷尔]]，[[马里兰州米德堡|米德堡]]，而可能[[巴尔的摩/华盛顿瑟古德·马歇尔国际机场|巴尔的摩/华盛顿国际机场]]。

由于Rush+节目，[[华盛顿地铁黄线|黄线]]曾经高峰时间营运从[[托腾堡站|托腾堡]]到绿带站。但是，在2017年6月25日[[华盛顿地铁|华盛顿都会区交通局]]停止了[[华盛顿地铁黄线|黄线]]服务到绿带站<ref>{{Cite web |url=https://www.wmata.com/service/status/details/upload/June25FareServiceChanges_PrintableBrochure_English.pdf |title=25日票价及服务改变可打印的小册子 |date=2017年}}</ref>。
