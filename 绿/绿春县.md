'''绿春县'''是[[中华人民共和国|中华人民共和国]][[云南省|云南省]][[红河哈尼族彝族自治州|红河哈尼族彝族自治州]]下属的一个[[县|县]]；位于红河州西南部，[[泗兰江|泗兰江]]、[[小黑江|小黑江]]流贯。县政府驻[[大兴镇_(绿春县)|大兴镇]]。面积3167平方公里，2002年人口20万人。

== 歷史 ==
1958年以[[元阳县|元阳县]]、[[红河县|红河县]]、[[墨江县|墨江县]]的部分地区设置。

== 地理 ==
县境内全是山区，最高海拔2637米，最低海拔320米。年均气温17.9℃，年均降雨量2234毫米。与越南莱州省山水相连，国境线长153公里，属中越自卫还击战时的前沿阵地，也是国家首批确定的重点扶持开发县，按国务院确定的贫困标准，尚有贫困人口134587人，其中深度贫困人口59068人。

== 资源 ==
一是土地资源。每平方公里人口密度仅75人，有海拔1000米以下的热区面积100多万亩。
二是水能资源。县境内水能蕴藏量达168万千瓦，可开发量120万千瓦以上，目前已引资50亿元开发80万千瓦。
三是矿产资源。有丰富的金、铜、砷、铅、锌、铁等矿带，占全县国土面积的37.5%。
四是生物资源。县境内的黄连山国家级自然保护区面积达97万亩，有数十种国家重点保护动植物，全县森林覆盖率达60.1%，累计发展绿色产业110万亩。
五是文化旅游资源。绿春是哈尼族最集中的家园，是哈尼标准语音所在地和哈尼原始宗教文化圣地，自然生态旅游资源和原生态民族文化资源积淀深厚，民风民俗独具魅力，民族文化生态旅游发展潜力巨大。

== 行政區劃 ==
绿春县辖8乡1镇，即：[[大兴镇_(绿春县)|大兴镇]]、[[戈奎乡|戈奎乡]]、[[牛孔乡|牛孔乡]]、[[大水沟乡|大水沟乡]]、[[大黑山乡|大黑山乡]]、[[半坡乡_(绿春县)|半坡乡]]、[[骑马坝乡|骑马坝乡]]、[[三猛乡|三猛乡]]和[[平河乡_(绿春县)|平河乡]]。下轄81个村委会，10个社区，738个村民小组，781个自然村，总人口23万人。

== 民族 ==
全县世居民族有哈尼、彝、瑶、傣、拉祜、汉六种，少数民族人口占总总人口的98.7％，其中哈尼族人口占87.4％。
