'''绿黄色社会'''（{{jpn|j=緑黄色社会|kn=りょくおうしょくしゃかい|rm=RyokuōshokuShakai}}）是[[日本|日本]]的一支男女混合4人组[[流行乐队|流行乐队]]，2012年在日本[[爱知县|爱知县]]组成。现属[[史诗唱片日本|EPIC Record]]旗下。简称「リョクシャカ」（RyokuShaka）。

==成员==
===现任成员===
*长屋晴子（{{jpn|j=長屋 晴子|kn=ながや はるこ|rm=Nagaya Haruko}}，{{Birth date and age|1995|5|28}}－），担当[[主唱|主唱]]，[[吉他|吉他]]，[[作词|作词]]，[[作曲|作曲]]。
*小林壹誓（{{jpn|j=小林 壱誓|kn=こばやし いっせい|rm=Kobayashi Issei}}，{{Birth date and age|1996|2|11}}－），担当[[吉他|吉他]]，[[和声|和声]]，作曲。
*peppe（{{jpn|j=ぺっぺ|kn=|rm=}}，本名神谷香奈江（{{jpn|j=神谷 香奈江|kn=かみや かなえ|rm=Kamiya Kanae}}），{{Birth date and age|1995|12|6}}－），担当[[键盘乐器|键盘]]，和声，作曲。
*穴见真吾（{{jpn|j=穴見 真吾|kn=あなみ しんご|rm=Anami Shingo}}，{{Birth date and age|1998|1|27}}－），担当[[电贝司|贝司]]，和声，作曲。

==经历==
2012年，以高中同学和发小的身份，长屋、小林、神谷、衫江、穴见五人组成乐队。2013年8月参加在[[日比谷野外音乐堂|日比谷野外音乐堂]]由[[SCHOOL_OF_LOCK!|SCHOOL OF LOCK!]]与[[索尼音乐|索尼音乐]]举办的“{{link-ja|闪光Riot|閃光ライオット}}”青年音乐节，获得大会第二名<ref>{{Cite web |url=https://www.musicman-net.com/artist/25597 |title=「閃光ライオット2013」グランプリは、フィッシュライフに決定}}</ref>。自主制作的CD《{{lang|ja|ノンカテゴリ}}》（无类别）、《{{lang|ja|ラストプレゼント}}》（最后的礼物）、《{{lang|ja|リアリズム}}》（现实主义）在现场发售。

2015年12月10日，鼓手衫江退出乐队，之后乐队一直在无鼓手的情况下活动。

2017年1月11日，乐队首次面向全国发行的[[迷你专辑|迷你专辑]]《Nice To Meet You??》发售<ref>{{Cite web |url=https://natalie.mu/music/news/210701 |title=名古屋発・緑黄色社会、SHE IS SUMMERら迎えた自主企画で初の全国盤発売を発表}}</ref>。同年4月7日首场专场演唱会在[[名古屋市|名古屋市]]{{link-ja|ell.FITS ALL|Electric Lady Land#ell.FITSALL}}举行。8月2日发售第二张迷你专辑《ADORE》<ref>{{Cite web |url=http://skream.jp/news/2017/06/ryokusyaka_adore_release.php |title=平均年齢20歳の愛知発4ピース・バンド 緑黄色社会、8/2に2ndミニ・アルバム『ADORE』タワレコ限定リリース決定}}</ref>。11月至12月举办了首次巡回，共在[[东京|东京]]、[[大阪|大阪]]和名古屋举行3场。

2018年3月发行第一张完整[[专辑|专辑]]《緑黄色社会》<ref>{{Cite web |url=https://natalie.mu/music/news/265697|title=緑黄色社会が初のフルアルバム詳細発表、過去ナンバー含む全10曲収録 }}</ref>，专辑发行后乐队举行了共计6场的专场巡回演唱会，而专辑同名主打曲《绿黄色社会》被提名入围2019年CD店大奖<ref>{{Cite web |url=https://skream.jp/news/2018/09/cd-shop-2018_1st-nominate.php |title=第11回CDショップ大賞2019、1次ノミネート作品としてWANIMA、オーラル、ヤバT、ユニゾン、Nulbarich、髭男、cero、緑黄色社会ら15タイトル発表}}</ref>。11月7日发行第3张迷你专辑《满出来的水去了那里》<ref>{{Cite web |url=https://skream.jp/news/2018/10/ryokushaka_anokoromitahikari_mv.php |title=緑黄色社会、11/7リリースのミニ・アルバム『溢れた水の行方』より「あのころ見た光」MV公開}}</ref>。此后将举行年内第二次共计7场的巡回<ref>{{Cite web |url=http://www.billboard-japan.com/d_news/detail/67431/2 |title=緑黄色社会、2018年2度目のワンマンツアー【溢れた音の行方】開催}}</ref>。

===乐队名的由来===
乐队最初就想以汉字作为乐队名。长屋在喝一种蔬菜汁时，小林看见就说了一句“绿黄色蔬菜”，而其他成员把“绿黄色蔬菜”误听成了“绿黄色社会”，就此成为了乐队名<ref>{{Cite web |url=http://www.m-on.press/music/0000091749 |title=緑黄色社会、新世代のポップセンスを届ける4人の素顔に迫る【M-ON! Recommend】}}</ref>。
