'''丘普朗·阿瑞昆'''（{{lang-th|เฌอปราง อารีย์กุล}}、{{RTGS|''Choeprang Arikun''}}、{{IPA-th|t͡ɕʰɤː.praːŋ ʔaː.riː.kun|IPA}}，{{bd|1996年|5月2日|||catIdx=Areekul, Cherprang}}）是[[泰國|泰國]]偶像藝人，為女子偶像團體[[BNK48|BNK48]] Team BIII成員，同時擔任BNK48的總隊長<ref>{{Cite news|url=https://music.mthai.com/news/newsinter/257589.html|title=คาวาอี้!! เปิดตัว BNK48 วงน้องสาวสัญชาติไทยของ AKB48|date=2017-02-15|accessdate=2018-03-31|language=th|location=Bangkok|website=music.mthai.com|publisher=MThai.com|archive-url=https://web.archive.org/web/20180401003834/https://music.mthai.com/news/newsinter/257589.html#|archive-date=2018-04-01|dead-url=no}}</ref>。她也是BNK48最早亮相的29名一期生之一<ref name="ManagerJuly2017">{{cite news|url=https://www.manager.co.th/Home/ViewNews.aspx?NewsID=9600000065594|title=น่ารักสิบผ่าน! “เฌอปราง อารีย์กุล” กัปตันคนแรกของ BNK48|publisher=Manager Online|date=2017-07-07|accessdate=2018-02-19|language=th|website=manager.co.th|location=Bangkok}}</ref>。

自入團之前，她就讀於[[瑪希敦大學|瑪希敦大學]]{{link-en|瑪希敦大學國際學院|Mahidol University International College|國際學院}}，並肩負研究助理的工作，與指導教授撰寫學術論文<ref name = "roong-aroon"/>。也因此，入團後她出席了許多泰國國內及國外的科學活動，甚至泰國航太單位宣稱將與他國合作，於2019年將她的[[肖像|肖像]]隨著[[人造衛星|人造衛星]]送上太空<ref name = "newtv"/>。此外，她也是一名[[Cosplay|Cosplay]]愛好者<ref name = "hello"/>。

== 經歷 ==
=== 偶像 ===
[[File:Cherprang_Areekul-2018.jpg|thumb]]
丘普朗本身是[[日本動畫|日本動畫]]迷，在觀看以[[AKB48|AKB48]]作為題材的動漫《[[AKB0048|AKB0048]]》時初次認識了[[AKB48集團|48 Group]]。她開始了解這個團體真實存在，並在不久之後成為粉絲<ref name="ManagerJuly2017"/>，她所尊敬的成员包括曾任NMB48总队长的[[山本彩|山本彩]]<ref name="mdpr">{{cite news|url=https://mdpr.jp/music/detail/1773511|script-title=ja:海外グループBNK48注目のキャプテン・CHERPRANG（チャープラン）、初ランクイン「みんなを愛しています」＜第10回AKB48世界選抜総選挙＞|publisher=モデルプレス|date=2018-06-16|accessdate=2018-11-13|language=ja|website=モデルプレス}}</ref>。

== 個人 ==
丘普朗來自[[泰國南部地區|泰國南部地區]]的[[宋卡府|宋卡府]][[合艾|合艾]]，原本一家人生活於山丘上<ref name="praew">{{cite web|url=http://praew.com/people/172879.html|title=ชวนโอตะมุง! 9 เรื่อง (ไม่) ลับของเฌอปราง กัปตันคนเก่งแห่ง BNK48|date=2018-05-24|accessdate=2018-07-10|language=th|location=Bangkok|website=praew.com|publisher=Amarin Printing and Publishing|author=Aunso|archive-url=https://web.archive.org/web/20180709190119/http://praew.com/people/172879.html#|archive-date=2018-07-09|dead-url=no}}</ref>。之後她搬至[[曼谷|曼谷]]居住<ref name = "praew"/>。她在曼谷的一所{{link-en|另類學校|alternative school}}隆阿倫學校（Roong Aroon School）接受[[中等教育|中等教育]]<ref>{{cite web|url=https://readthecloud.co/thoughts-13/|title=ครูของครู ความคิดของครูผู้เชื่อในธรรมชาติ การสอนแบบพระพุทธเจ้า และศักยภาพในเด็กทุกคน|author=ทรงกลด บางยี่ขัน|publisher=The Cloud|accessdate=2018-03-23|archive-url=https://web.archive.org/web/20180323155853/https://readthecloud.co/thoughts-13/#|archive-date=2018-03-23|dead-url=no}}</ref>。她認知到自己過著太平淡的生活，而不知道人生的目標為何，因此和母親商談後決定進入另類學校就讀<ref name = "hello">{{cite journal|title=‘เฌอปราง อารีย์กุล’ ไอดอลนักวิทยาศาสตร์แห่ง BNK48|url=https://th.hellomagazine.com/education/the-talented-cherprang-bnk48/|language=th|location=Bangkok|author=Editorial|journal=Hello Magazine|publisher=Hello Magazine|date=2018-05-14|accessdate=2018-06-22}}</ref>。在學校，她發現自身的數學與自然表現優於美術，對科學實驗等相關事務具有熱忱，並受到她的自然老師啟發，希望未來能成為一名科學家<ref name = "hello"/>。除此之外，每年學校表演活動的鼓手及啦啦隊員也都由她擔任<ref name = "hello"/>。

在另類學校完成學業後<ref name="roong-aroon">{{cite web|url=http://www.roong-aroon.ac.th/?p=2162|title=มัธยมศึกษาปีที่ ๖ รุ่นที่ ๑๒ ปีการศึกษา ๒๕๕๖|publisher=Roong Aroon School|date=2014-06-11|accessdate=2018-03-23|website=roong-aroon.ac.th|location=Bangkok|language=th|archive-url=https://web.archive.org/web/20180324040905/http://www.roong-aroon.ac.th/?p=2162#|archive-date=2018-03-24|dead-url=no}}</ref>，她獲得[[瑪希敦大學|瑪希敦大學]]{{link-en|瑪希敦大學國際學院|Mahidol University International College|國際學院}}的錄取。於學校她主修化學、副修物理<ref name="campus-star">{{cite web|url=https://campus.campus-star.com/star/56058.html|title=นิยามเด็กเนิร์ดนักเคมี ฉบับกัปตันเฌอปราง BNK48 วิทยาลัยนานาชาติ ม.มหิดล|author=teenpublishing|language=th|location=Bangkok|publisher=campus.campus-star|website=campus.campus-star|accessdate=2018-03-20|date=25 November 2017|archive-url=https://web.archive.org/web/20180320230545/https://campus.campus-star.com/star/56058.html#|archive-date=2018-03-20|dead-url=no}}</ref><ref name = "dailynews"/>，她曾表示[[物理化學|物理化學]]、[[量子化學|量子化學]]及[[太空科學|太空科學]]是她最有興趣也是未來最期望投入研究的領域<ref name = "praew"/>。同時，在大學內她發現自己對於[[Cosplay|Cosplay]]充滿興趣，而後成為Cosplayer參與活動<ref name = "hello"/>。

在大學二年級和三年級期間，她肩負研究助理的工作<ref name = "hello"/>，撰寫了一篇有關[[藍瓶實驗|藍瓶實驗]]的學術論文，並刊載於2017年11月的《{{link-en|皇家學會開放科學|Royal Society Open Science}}》<ref>{{cite web|url=http://blogs.royalsociety.org/publishing/musical-notes-and-chemistry/|title=Musical notes and a chemistry experiment|author-first=And|author-last=Dunn|publisher=The Royal Society|date=2018-03-21|accessdate=2018-03-23|website=blogs.royalsociety.org|archive-url=https://web.archive.org/web/20180323155327/http://blogs.royalsociety.org/publishing/musical-notes-and-chemistry/#|archive-date=2018-03-23|dead-url=no}}</ref><ref>{{cite web|url=http://rsos.royalsocietypublishing.org/content/4/11/170708|title=A DFT investigation of the blue bottle experiment: E∘half-cell analysis of autoxidation catalysed by redox indicators|publisher=[[皇家學會|皇家學會]]|accessdate=2018-11-13|date=2017-11-08|language=en|archive-url=https://web.archive.org/web/20180622060308/http://rsos.royalsocietypublishing.org/content/4/11/170708#|archive-date=2018-06-22|dead-url=no}}</ref>。由於她的科學學識背景，被邀請參加許多學術活動，例如2018年3月29日[[天宮一號|天宮一號]][[再入|再入]]的研討會<ref>{{cite news|url=http://www.bangkokbiznews.com/news/detail/797140|title='เฌอปราง BNK48' ร่วมนับถอยหลังเทียนกง-1ตกสู่โลก บทสรุปจากห้วงอวกาศ|website=bangkokbiznews.com|date=2018-03-28|accessdate=2018-06-21|language=th|location=Bangkok|publisher=กรุงเทพธุรกิจ|archive-url=https://web.archive.org/web/20180621194445/http://www.bangkokbiznews.com/news/detail/797140#|archive-date=2018-06-21|dead-url=no}}</ref>，以及同年6月1日的同步輻射光研究所成立十週年活動<ref name="newtv">{{cite news|url=http://www.newtv.co.th/news/16494|title=โอตะปลื้ม! โครงการส่งภาพ “เฌอปราง BNK48” สู่อวกาศ|publisher=New18|date=2018-06-03|accessdate=2018-06-21|website=newtv.co.th|language=th|location=Bangkok|archive-url=https://web.archive.org/web/20180621194156/http://www.newtv.co.th/news/16494#|archive-date=2018-06-21|dead-url=no}}</ref>。此外，她也跟隨泰國代表隊前往[[英國|英國]]參加2018年{{link-en|切爾滕納姆科學節|Cheltenham Science Festival}}的科學傳播比賽<ref>{{cite web|url=http://www.most.go.th/main/th/34-news/news-gov/7233-famelab|title=รอบชิงชนะเลิศ “FameLab” เวทีค้นหาตัวแทนประเทศไทยไปร่วมแข่งขันบนเวทีสื่อสารวิทยาศาสตร์ระดับโลก “Cheltenham Science Festival”|publisher=Ministry of Science and Technology|accessdate=2018-06-21|date=n.d.|website=most.go.th|language=th|location=Bangkok|archive-url=https://web.archive.org/web/20180621194341/http://www.most.go.th/main/th/34-news/news-gov/7233-famelab#|archive-date=2018-06-21|dead-url=no}}</ref>。
