'''日本工农学校'''，[[中国共产党中央委员会|中国共产党中央委员会]]于1940年10月在[[陕西|陕西]][[延安|延安]][[宝塔山|宝塔山]]创办的一所改造日军[[俘虏|俘虏]]的专门学校，教育对象是日军俘虏和投诚的日军士兵，校长化名林哲，是[[日本共产党|日本共产党]]的创始人和主要领导人之一、[[国际共产主义运动|国际共产主义运动]]著名活动家[[野坂参三|野坂参三]]。

== 背景 ==
[[File:Zhou_Enlai_and_Sanzo_Nosaka.jpg|thumb]]
1937年抗日战争全面爆发后，[[朱德|朱德]]、[[彭德怀|彭德怀]]就曾联名签署八路军关于对日俘虏政策的规定，对俘虏的日军进行优待，愿意回国的提供路费。1937年[[平型关战役|平型关战役]]后，八路军在战场上俘虏的日军官兵逐渐增多。<ref name="jiema"/>中共的对日战俘政策由优待释放向教育感化转变。1940年6月，中共中央发出对日军俘虏工作的指示，要求注意选择少数日军俘虏中的进步分子，给以较长期的训练。<ref>{{cite web |author1=彭珊珊 |title=中共改造日本战俘的“延安经验”曾震惊美国 |url=http://cul.qq.com/a/20141212/000657.htm |website=腾讯网-澎湃新闻 |access-date=2018-08-04 |archive-url=https://web.archive.org/web/20180804152242/http://cul.qq.com/a/20141212/000657.htm |archive-date=2018-08-04 |dead-url=no }}</ref>

1940年3月，时任日本共产国际代表的野坂参三随[[周恩来|周恩来]]、[[任弼时|任弼时]]等人秘密由[[苏联|苏联]]来到延安。野坂参三到延安后与中共相关领导人商谈了有关日军战俘的教育管理问题。<ref name=wang/>

== 历史 ==
[[File:Students_of_the_Japanese_Worker_Peasant_School.jpg|thumb]]

经过几个月的调查研究，野坂参三在1940年10月向[[毛泽东|毛泽东]]写信建议中共中央在延安成立一所教育改造日军俘虏的专门学校。据野坂参三的提议，[[中共中央|中共中央]]、[[中央军委|中央军委]]决定，由[[八路军总政治部|八路军总政治部]]在延安建立了一所以日军战俘为主体的特殊学校，用以开展打倒[[法西斯|法西斯]]的国际统一战线工作。<ref name=wang>{{cite web |author1=王仕琪 |title=日本战俘为啥怀念延安日本工农学校? |url=http://dangshi.people.com.cn/n/2012/1026/c85037-19395318.html |website=中国共产党新闻网-人民政协报 |access-date=2018-08-04 |archive-url=https://web.archive.org/web/20130120084132/http://dangshi.people.com.cn/n/2012/1026/c85037-19395318.html |archive-date=2013-01-20 |dead-url=no }}</ref>毛泽东亲自定名“日本工农学校”。学校的具体工作由八路军政治部负责领导。<ref name="jiema">{{cite web |title=解码抗战时在延安创立的日本工农学校 |url=http://www.xinhuanet.com/book/2015-11/20/c_128449168.htm |website=新华网-中国纪检监察报 |access-date=2018-08-04 |archive-url=https://web.archive.org/web/20180804152246/http://www.xinhuanet.com/book/2015-11/20/c_128449168.htm |archive-date=2018-08-04 |dead-url=no }}</ref>

1941年5月15日，日本工农学校举行开学典礼。林哲（野坂参三当时的公开名）{{efn|1943年5月下旬，野坂参三在毛泽东建议下开始公开活动，名字换成其在苏联用过的化名“冈野进”。}}任校长，[[八路军第120师|八路军第120师]]第359旅政治部敌工科科长[[赵安博|赵安博]]任副校长。1943年4月后改由总政敌工部副部长[[李初梨|李初梨]]兼任副校长。除野坂参三外，各课教员有[[王学文|王学文]]、[[何思敬|何思敬]]、[[李初梨|李初梨]]、[[廖体仁|廖体仁]]、[[江右书|江右书]]等，均为曾留学日本的精通日语的中国人。<ref name="jiema"/>

学校位于宝塔山下，共有16个窑洞和一栋不大的平房。全体工作人员包括校长和炊事员在内仅二十来人。随着抗日战争形势的发展，日本工农学校及其学员数量也在增多。延安日本工农学校最初有学员11人，到1945年8月增至300多人；<ref name="jiema"/>山西、山东和华中等[[抗日根据地|抗日根据地]]也先后成立了总分校或分校。<ref>{{cite web |title=【历史】延安曾有个日本战俘学校 |url=http://www.dzwww.com/shandong/sdnews/201704/t20170404_15716952.htm |website=大众网 |access-date=2018-08-04 |archive-url=https://web.archive.org/web/20180804152244/http://www.dzwww.com/shandong/sdnews/201704/t20170404_15716952.htm |archive-date=2018-08-04 |dead-url=no }}</ref>
