'''日田彥山線'''（{{jpn|j=日田彦山線|hg=ひたひこさんせん|rm=Hita-Hikosan sen|f=y}}）是一條連結[[日本|日本]][[福岡縣|福岡縣]][[北九州市|北九州市]][[小倉南區|小倉南區]]的{{link-ja|城野站 (JR九州)|城野駅 (JR九州)|城野站}}至[[大分縣|大分縣]][[日田市|日田市]]的{{link-ja|夜明站|夜明駅}}，屬於[[九州旅客鐵道|九州旅客鐵道]]（JR九州）的鐵路線（[[地方交通線|地方交通線]]）。

== 路線資料 ==
* 管轄（事業類別）：九州旅客鐵道（[[鐵路公司|第一種鐵道事業者]]）
* 路線距離（{{link-ja|營業距離|営業キロ}}）：68.7公里
* [[軌距|軌距]]：1067毫米
* 道床：全線[[道碴|有碴軌道]]
* 站數：24個（包括起終點站）
** 若只限屬於日田彥山線的車站，排除起終點站（城野站屬於日豐本線，夜明站屬於久大本線<ref>『停車場変遷大事典 国鉄・JR編』JTB 1998年</ref>）則為22個。
* 複線路段：沒有（全線單線）
* 電氣化路段：沒有（全線{{link-ja|非電氣化|非電化}}）
* [[閉塞_(鐵路)|閉塞方式]]：特殊自動閉塞式（軌道回路檢知式）
* {{link-ja|運轉指令所|運転指令所}}：博多綜合指令中心<ref name = "交通20130205">交通新聞2013年2月5日</ref>
* 最高速度：85公里/小時

全線由{{link-ja|九州旅客鐵道鐵道事業本部|九州旅客鉄道鉄道事業本部|本社鐵道事業本部}}直轄。
