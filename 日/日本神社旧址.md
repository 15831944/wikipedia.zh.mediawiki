'''日本神社旧址'''（五台山一号建筑）位于[[中国|中国]][[江苏省|江苏省]][[南京市|南京市]][[鼓楼区_(南京)|鼓楼区]]五台山1号，[[五台山体育中心|五台山体育中心]]东侧，原为[[侵华日军|侵华日军]]仿照东京[[靖国神社|靖国神社]]而建的'''南京神社'''，设计者为[[高见一郎|高见一郎]]，为同类建筑中规模最大的一处，现存建于1940年2月至1941年底的两栋砖木结构建筑，其中坐北面南者为正殿，坐东面西者为侧殿（又被称作大庙和小庙），正殿中原供奉天照大神像及在战争中阵亡的校官以上军官灵位，侧殿则用于供奉尉官以下军官和普通士兵灵位。日本战败后，国民政府将其改作'''中国抗战阵亡将士纪念堂'''，正殿用于供奉在战争中阵亡的中国军队将士灵位，侧殿则用于陈列自侵华日军中缴获的战利品，正殿前仍存有[[蒋中正|蒋中正]]手植[[黑松|黑松]]及石碣<ref>{{cite web |url=http://news.163.com/12/0816/09/8916CO7800014AED.html|title=《五台山上“日本神社”成为侵略罪证》|accessdate=2012-09-14|work=于峰|publisher=金陵晚报}}</ref>。

建筑均采用日式[[和风建筑|和风建筑]]风格，柱跗式台基、明黄色墙壁、赭黄色窗户、方形外廊柱，歇山顶上覆以黑色瓦，院内草坪上另存有若干件日式风格石质灯座及底部有雕花纹饰的旧式路灯。建筑现由江苏省体育局管理使用，内部均已被改造<ref>{{cite web |url=http://news.163.com/12/0717/02/86J4O70E00014AED.html|title=《五台山体育中心有座日本神社》|accessdate=2012-09-14|work=肖雷|publisher=扬子晚报}}</ref>。
