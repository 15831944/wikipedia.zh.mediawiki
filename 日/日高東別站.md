'''日高東別站'''（{{jpn|j=日高東別駅|hg=ひだかとうべつえき|rm=Hidaka-tōbetsu eki|f=y}}）位於[[北海道|北海道]][[日高郡|日高郡]][[新日高町|新日高町]]靜內東別，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|ヒツ}}'''。

名稱中「日高」來自於舊國名「[[日高國|日高国]]」，「東別」源於[[阿伊努語|阿伊努語]]的「toy-pet」，意思是有可食土的河川<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/8190P.pdf|title=アイヌ語地名リスト|accessdate=2017-08-17|author=北海道庁|date=|publisher=北海道庁}}</ref>。

== 車站構造 ==
[[側式月台|側式月台]]1面1線的地面站。沒有站房，只有一座小型的候車室。

== 車站周邊 ==
附近多為農家。特產是東別草莓。
* [[北海道道796號西端春立線|北海道道796號西端春立線]]
* 日高國際鄉村俱樂部

== 歷史 ==
* 1958年（昭和33年）7月15日 - 設立，只處理旅客業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
