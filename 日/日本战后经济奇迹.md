[[二战|二战]]后[[日本经济|日本经济]]经历了高速的增长时期，称为'''日本战后经济奇迹'''。

== 背景 ==
二战后，世界迎来了[[冷战|冷战]]时期，日本作为[[美国|美国]]抵御[[苏联|苏联]]影响的桥头堡，因此受到了美国极大的[[经济援助|经济援助]]，又加上[[朝鲜战争|朝鲜战争]]的爆发以及日本戰後的重建和更新需求使日本[[重工业|重工业]]复苏，加上沒有養[[軍隊|軍隊]]的束縛（[[日本國憲法第九條|《日本國憲法》第九條]]），这些都开启了经济奇迹的大门。日本经济的特色包括：合作厂商、[[供应商|供应商]]、[[经销商|经销商]]与[[银行|银行]]紧密相连成一个团体，因而这个团体被称为[[财阀|财阀]]。财阀往往与政府有着良好的关系。公司与工厂裏强大的[[工会|工会]]和[[终身雇用|终身雇用]]制度保证了员工的利益。那些刺激经济的政策通过[[通商產業省|通商產業省]]出台。<ref>《日本的崛起——昭和經濟史》</ref>

== 美国的援助 ==
二战之后，[[通货膨胀|通货膨胀]]和[[失业|失业]]导致日本经济的全面崩溃。为了稳定日本的局势，在[[美国政府|美国政府]]的主持下，[[盟軍最高司令官總司令部|盟軍最高司令官總司令部]]（GHQ）在日本的经济复苏发挥了至关重要的作用：GHQ官员认为，经济发展不仅可以防止日本再次出现[[军国主义|军国主义]]，并且可以防止[[共产主义|共产主义]]。1950年，由于朝鲜战争的爆发，日本成为[[美军|美军]][[军需|军需]]生产和维修的基地，美国政府支付巨额的特殊采购，这些特殊采购占当年日本[[出口贸易|出口贸易]]的27%。美国还帮助日本加入了[[关税及贸易总协定|关税及贸易总协定]]（GATT），尽管[[英国|英国]]强烈反对。

== 政府的贡献 ==
1950年代后期，日本的经济完全恢复。1956年，日本政府制定“[[电力五年计划|电力五年计划]]”，建立电力工业和用石油取代[[煤炭|煤炭]]，由此带来的良好影响带动了[[耐久財|耐久財]]消费，出现战后第一次经济发展高潮，时称[[神武景气|神武景气]]。<ref> 鬼頭宏 『図説人口で見る日本史　縄文時代から近未来社会まで』 PHP研究所、2007年7月、168-170頁</ref>
[[File:Public_Pawn_Shop_circa_1960.jpg|缩略图]]
1958年后，日本政府开始引导企业生产如汽车、电视等家用电器和钢铁，出现了第二次经济发展高潮，称为[[岩户景气|岩户景气]]。
[[File:Kisarazu-port.JPG|thumb]]位於千葉縣的煉鋼廠，60年代日本礦金屬相關業達到單年25%的成長]]
[[File:1966_Datsun_Sunny_01.jpg|thumb]]象徵]]
当日本申办成功[[1964年夏季奥林匹克运动会|1964年夏季奥林匹克运动会]]后，日本政府耗资达30亿美元，用作兴建运动场馆，以及修建相关配套如交通设施及供水系统。日本为了东京奥运的直接场馆投资为295亿日元，间接投资（公路、地下铁等交通建设，上下水道铺建等）则达9,600亿日元。房地产市场迅速发展，再一次拉动了经济，时称[[奧林匹克景氣|奥运会景气]]。

但奥运会结束后日本经济出现了放缓，于是日本政府决定发行战后的第一次建设国债刺激经济。在这期间，有不少大企业合并，同时私家车和彩色电视快速普及，国民的所得水平快速提高，时称[[伊奘諾景氣|伊奘諾景氣]]。

1960年代日本开始了[[國民所得倍增計劃|國民所得倍增計劃]]，10年的时间让日本的国民生产总值增加到26兆日圆，但实际的经济成长远超过预期：仅六年的时间，国民平均所得就达到了倍增的计划。日本政府制定法规保护私营部门，有效地管理经济危机和扩大贸易。
