'''日本道路交通情報中心'''（{{jpn|j=日本道路交通情報センター|hg=にほんどうろこうつうじょうほうセンター|rm=nihon dōro kōtsū jōhō sentā|f=y}}；{{lang-en|Japan Road Traffic Information Center}}），簡稱'''JARTIC'''，是[[日本|日本]]提供道路交通資訊的專責機構，其委託各[[都道府縣|都道府縣]]交通部門取得交通訊息，並將資訊收集、彙整並提供道路交通訊息給一般大眾。1970年成立，2013年4月1日由[[財團法人|財團法人]]轉型為公益財團法人。<ref name=history/>

== 概要 ==
日本道路交通情報中心成立於[[1970年|1970年]]（[[昭和|昭和]]45年）1月，成立目的為「提供道路交通資訊，並利用資訊預防事故和救災」。<ref name="job">{{Cite Web|url=http://www.jartic.or.jp/guide/whatjob.html|title=成立目的|publisher=日本道路交通情報中心|access-date=2013-02-11}}{{ja}}</ref>日本道路交通情報中心的主要業務有：
# 藉由各交通相關單位（包含各都道府縣交通部門、[[警察廳_(日本)|警察廳]]、[[警視廳|警視廳]]、[[國土交通省|國土交通省]]及[[高速道路株式會社法|各高速道路株式會社]]）及該單位的142個分部提供的道路交通資訊，將資訊加以統整並以媒體發布給大眾。<ref name=job/>
# 研究道路交通資訊，並提供給相關單位及大眾。<ref name=job/>
# 進行[[交通安全|交通安全]]教育。<ref name=job/>
[[2013年|2013年]]（[[平成|平成]]25年）4月1日後，日本道路交通情報中心轉型為[[財團法人|公益財團法人]]組織。<ref>{{Cite web|url=http://www.jartic.or.jp/guide/about/disclos/pdf/keikaku.pdf |title=事業計畫書|format=PDF|publisher=日本道路交通情報中心|access-date=2013-02-11}}{{ja}}</ref>

== 訊息放送 ==
使用者可以利用藉由[[電話|電話]]<ref name="tel">{{Cite web|url=http://www.jartic.or.jp/jartic_web/person/telephone.html|title=查詢道路交通情報電話號碼|publisher=日本道路交通情報中心|access-date=2013-02-11}}{{ja}}</ref>、[[廣播|廣播]]<ref name="broadcast">{{Cite web |url=http://www.jartic.or.jp/jartic_web/person/broadcast.html |title=廣播、放送道路交通情報時間|publisher=日本道路交通情報中心|access-date=2013-12-17}}{{ja}}</ref>、[[電視|電視]]<ref name=broadcast/>、[[網路|網路]]<ref>{{Cite web|url=http://www.jartic.or.jp/|title=即時路況|publisher=日本道路交通情報中心|access-date=2018-11-24}}{{ja}}</ref>、[[行動電話|行動電話]]<ref>{{Cite web |url=http://www.jartic.or.jp/jartic_web/others/faqs.html|title=FAQ|publisher=日本道路交通情報中心|accessdate=2013-02-11}}{{ja}}</ref>及[[道路交通信息通信系統|道路交通信息通信系統]]（簡稱'''VICS'''）<ref>{{Cite web|publisher=國土交通省{{link-ja|國土技術政策綜合研究所|国土技術政策総合研究所}}ITS（智能運輸）研究室|url=http://www.nilim.go.jp/japanese/its/its_research/vics/|archivedate=2018-05-18|language=ja|title=VICS|archiveurl=https://web.archive.org/web/20180518103915/http://www.nilim.go.jp/japanese/its/its_research/vics/}}</ref>獲取路況資訊。雖然這些路況資訊公開的，但是以[[本田汽車|本田汽車]]為首的一些企業開始出現一種避免使用日本道路交通情報中心提供的路況資訊的風氣，理由是因為二次使用JARTIC提供的路況資訊每年需要和JARTIC簽訂約數百萬日圓的契約，因此[[本田汽車|本田汽車]]獨自發展了獨立的交通資訊收集系統。<ref>{{Cite web |url=http://www.jartic.or.jp/jartic_web/company/online.html|title=向企業提供交通情報細則及收費|publisher=日本道路交通情報中心|access-date=2013-02-11}}{{ja}}</ref>

=== 電話及行動電話 ===
日本道路交通情報中心提供使用電話及行動電話查詢路況資訊的服務。在日本國內撥打050-369-6666（電話）或#8011（行動電話、[[個人手持式電話系統|個人手持式電話系統]]（PHS））即可查詢全日本路況。<ref name=tel/><ref>{{Cite Web|url=http://www.jartic.or.jp/jartic_web/pdf/telNumberList.pdf|publisher=日本道路交通情報中心|title=全國及各都道府縣路況查詢電話表|access-date=2018-11-25|format=pdf}}{{ja}}</ref>

=== 廣播 ===
一些廣播的聽眾可在每日的固定時段收聽到來自日本道路交通情報中心的路況報導。通常是由日本道路交通情報中心提供資訊並由主持人播報，但有時也會連線到中心由中心的路況播報員播報。<ref>{{cite book |author={{link-ja|室井昌也|室井昌也}}|date=2014-11|title=交通情報の女|isbn=978-4-8460-1385-1|page=p.52-53|publisher=論創社}}{{ja}}</ref>

=== 電視 ===
[[日本放送協會|日本放送協會]]（NHK）每天早晨和傍晚在[[首都圈_(日本)|首都圈]]、[[中京圈|名古屋都市圈]]、[[近畿地方|近畿地方]]、[[福岡縣|福岡縣]]和[[北海道|北海道]]的[[帶廣|帶廣]]和[[函館|函館]]於各提供一次最新的路況資訊。[[東北放送|東北放送]]的觀眾可以在週一至週五早上7點51分的{{link-ja|觀看!宮城|ウォッチン!みやぎ}}節目收看關於[[宮城縣|宮城縣]]及[[仙台市|仙台市]]的路況資訊。[[千葉電視台|千葉電視台]]與合作的對象[[神奈川電視台|神奈川電視台]]及[[埼玉電視台|埼玉電視台]]在每天7點過後會聯合播出[[千葉縣|千葉縣]]、[[神奈川縣|神奈川縣]]、[[埼玉縣|埼玉縣]]及[[東京都#區部（特別區）|東京都區部]]的路況報導。

==軼事==
雖然大部分時候路況資訊是直接由日本道路交通情報中心提供，但在特定的情況下[[日本廣播電台列表|各電台]]主持人提供的路況資訊是由總台提供，而非由日本道路交通情報中心直接提供。另外，[[調幅廣播|調幅廣播]]（AM）和[[調頻廣播|調頻廣播]]（FM）在提供路況資料的時候對日本道路交通情報中心的稱呼不盡相同。AM電台稱為道路交通情報中心，FM電台稱為JARTIC。<ref>但是有2個例外：[[關西電台|關西電台]]雖然是AM電台，但使用JARTIC這個名稱。同樣也是AM電台的[[ABC電台|朝日放送電台]]，則在2個節目中的路況報導中使用JARTIC的名稱，分別是{{link-ja|STAR☆MUSIC☆SUNDAY (朝日放送)|STAR☆MUSIC☆SUNDAY|STAR☆MUSIC☆SUNDAY}}和{{link-ja|Cheers! (朝日放送)|Cheers! (ラジオ番組)|Cheers!}}。</ref>而直接直呼其正式名稱的電台大多是廣播時段較短的地方電台。另外，常可發現以下現象：雖然電視和廣播宣稱此路段為「順暢」，但此路段實際上為「壅塞」（日語：{{lang|ja|空いています}}）。這是因為「壅塞」的定義為「整條公路每小時的車流量達到極端地少」。
