'''日笠陽子'''（{{bd|1985年|7月16日|catIdx=Hikasa Yōko}}）是[[日本|日本]][[女性|女性]][[聲優|聲優]]、[[旁白|旁白]]、[[歌手|歌手]]，[[I'm_Enterprise|I'm Enterprise]]所屬，[[神奈川縣|神奈川縣]]出身。身高155cm，[[血型|血型]]O型。

== 概要 ==
=== 經歷 ===
因為憧憬在《[[美少女戰士_(動畫)|美少女戰士]]》和《[[新世紀福音戰士|新世紀福音戰士]]》中演出完全不同角色（[[月野兔|月野兔]]和[[葛城美里|葛城美里]]）的[[三石琴乃|三石琴乃]]而想成為聲優<ref name="日ナレ紹介_後"> {{Cite web |date= 2009-06-03 |url= http://www.animate.tv/school/interview_details.php?s_id=A01&id=1251108986 |title= animatetv專訪 日本播音演技研究所 日笠陽子 後編}}</ref><ref name="声優プリンセス">{{Cite journal|author= |year= 2009 |title= PICK-UP FRESH VOICE ACTRESS |journal= 声優プリンセス |pages= pp.79-81 |publisher= 近代映画社}} ISBN 978-4-7648-7153-3 </ref>。高中畢業後，在服裝系專修學校學習的同時，進了[[日本播音演技研究所|日本播音演技研究所]]（21期生）<ref group="註">決定進日本播音演技研究所的理由是課程不會影響學業，同時憧憬的[[三石琴乃|三石琴乃]]也是從這裡畢業的。</ref><ref name="日ナレ紹介_後"/>。本科生時通過「免費新人育成考試」成為特待生。

2007年出演《[[Sketch_book～素描簿～|Sketchbook～full color's～]]》的根岸水面電視動畫出道。2009年出演《[[K-ON!|K-ON!]]》的秋山澪一角之後受到關注。之後出演《[[最後大魔王|最後大魔王]]》的服部絢子等，然後在《[[世紀末超自然學院|世紀末超自然學院]]》首次主演擔任神代瑪雅一角。

在2010年3月6日發表的第4回[[聲優獎|聲優Awards]]上，以《[[K-ON!|K-ON!]]》劇中結成的團體「放學後Tea Time」的一員，與[[豐崎愛生|豐崎愛生]]、[[佐藤聰美|佐藤聰美]]、[[壽美菜子|壽美菜子]]、[[竹達彩奈|竹達彩奈]]一起獲得歌唱賞。

2013年為電視動畫《[[進擊的巨人|進擊的巨人]]》演唱片尾曲，同時宣布將以歌手身份出道<ref>[http://www.cinematoday.jp/page/N0048456 「けいおん!」日笠陽子、アニメ「進撃の巨人」エンディングテーマでソロデビュー決定]</ref>。
同年2月8日官網開通，首單《進擊的巨人》片尾曲「{{lang|ja|[[美丽而残酷的世界|美しき残酷な世界]]}}」，以及連續三個月的音樂作品發售時間<ref>[http://www.oricon.co.jp/news/music/2021488/full/ “てへぺろ”日笠陽子、5・8ソロ始動！ 怒とうの3ヶ月連続リリース]</ref>。同年官網於4月14日更新消息，10月27日在{{link-ja|STUDIO COAST|STUDIO COAST}}舉辦首場個人演唱會<ref>[http://www.ota-suke.jp/news/95278 ■ 日笠陽子のライブが10月に新木場STUDIO COASTで開催決定]</ref>。

2015年12月30日，於自身Blog宣布結婚消息<ref>{{Cite web |url=http://blog.livedoor.jp/hiyonikki/archives/2070038.html|title= ごあいさつ |work= 日笠陽子のひよっ子記}}</ref>。

=== 特色 ===
諸如上述的《[[世紀末超自然學院|世紀末超自然學院]]》的神代瑪雅、《[[最後大魔王|最後大魔王]]》的服部絢子、《[[K-ON!|K-ON!]]》的秋山澪等，經常出演強氣的女性。雖然經常演性格像男生一般的角色以及早熟、大姐系的角色，不過也演過像《[[WORKING!!|WORKING!!]]》的小鳥遊泉這樣陰氣的角色。在動畫以外，還有零星做外國片配音和播音類的工作。

因在廣播節目中經常很興奮地進行各種模仿、表演各種才藝，而被聽眾和聲優夥伴叫做搞笑藝人。另外在廣播《{{lang|ja|A&G REQUEST デジスタ}}》中開始的用來欺負日笠的「日笠以前是不良少女」的玩笑也廣泛傳播開來，在《[[K-ON!|K-ON!]]》的網路廣播《{{lang|ja|らじおん!}}》中也常被搬出來<ref>廣播《{{lang|ja|おどろき戦隊 モモノキファイブ}}》、超!A&G+《{{lang|ja|A&G REQUEST デジスタ}}》、《{{lang|ja|らじおん!}}》等。</ref>。

在各個廣播節目中經常使用獨創的玩笑「'''{{lang|ja|てへぺろ(・ω<)}}'''」<ref group="註">《{{lang|ja|おどろき戦隊モモノキファイブ}}》第7回（2009年5月21日）裡首次披露。</ref>，共演者也經常會用。之後這個玩笑被[[SEIGURA|SEIGURA web]]企劃的「SEIGURA web選取的流行語2009」（第1回）選上，作為作者受到表彰<ref>{{Cite web |date=2010-03-10 |url=http://www.seigura.com/topics/show/1373 |title=日笠陽子獲獎！ 「SEIGURA web選取的流行語2009」:SEIGURA web |work=[[SEIGURA|SEiGURA web]] |publisher=[[SEIGURA|SEIGURA]] |deadurl=yes |archiveurl=https://web.archive.org/web/20100322213536/http://www.seigura.com/topics/show/1373 |archivedate=2010-03-22 }}</ref>。

=== 人物 ===
中學、高中六年都在[[壘球|壘球]]部，中學1年時是擔任[[中外野手|中外野手]]<ref name="日ナレ紹介_後"/>，之後一直擔任[[游擊手|游擊手]]和[[投手|投手]]<ref>K-ON! web radio『{{lang|ja|らじおん!}}』第12回、[[聲優Animedia|聲優Animedia]]（2009年5月号）等。</ref>，為右投左打。此外因為跑的快，還經常被選為接力賽跑最後一棒的選手<ref>[[超!A&G+|超!A&G+]]『A&G REQUEST デジスタ』第24回。</ref>。午休都在圖書館度過，也曾擔任過圖書委員<ref>聖痕鍊金士radio!〜米海洛普學園廣播社〜.Lantis web radio. 2010-02-12（第6回）放送. </ref>。

日笠本人推薦<ref name="声優プリンセス"/>的暱稱「{{lang|ja|ひよっち}}」是以前事務所同期的[[古川小百合|古川小百合]]幫忙想的。此外，在各個廣播節目中經常使用「{{lang|ja|てへぺろ}}」這個自創的搞笑用語，也經常被共演者使用。

特長是裁縫、模仿<ref name="日ナレ紹介_前"/>等。另外，擁有毛筆3段、硬筆3段、[[芳香療法|芳香療法]]檢定2級等資格<ref name="profile">{{Cite web |url= http://www.imenterprise.jp/data.php?id=64 |title= 日笠陽子 公式profile |work= I'm Enterprise公式網站 |publisher= [[I'm_Enterprise|I'm Enterprise]]}}</ref><ref name="日ナレ紹介_後"/>。

因在profile的特長裡寫了模仿而經常被人要求表演模仿<ref name="日ナレ紹介_前"/>，在廣播中模仿過[[崖上的波妞|崖上的波妞]]、[[出川哲朗|出川哲朗]]、[[織田裕二|織田裕二]]等。不過本人說其實模仿只是停留在興趣範圍<ref name="日ナレ紹介_前">{{Cite web |date= 2009-06-23 |url= http://www.animate.tv/school/interview_details.php?s_id=A01&id=1251765015 |title= animatetv專訪 日本播音演技研究所 日笠陽子 前編}} </ref>。另外在廣播《{{lang|ja|おどろき戦隊 モモノキファイブ}}》的節目版塊中演出老人、小孩、時尚女生、魔王、怪人等角色。

和在《{{lang|ja|A&G REQUEST デジスタ}}》中共演的[[大龜明日香|大龜明日香]]、在《[[K-ON!|K-ON!]]》等作品中共演的[[佐藤聰美|佐藤聰美]]、事務所同期的[[折戶真理|折戶真理]]等很要好，從BLOG中也能看出。

相當愛玩遊戲，經常和聲優夥伴一起玩遊戲。很喜歡《[[魔物獵人|魔物獵人]]》，從《魔物獵人P2》開始一直有在玩<ref>銀河に吠えろ!宇宙GメンTAKUYA声優木曜市（2009年7月9日放送回）</ref>。《[[魔物獵人3|魔物獵人3]]》也在發售後和[[Wii|Wii]]一起購入，經常在線和折戶真理一起玩。此外還擁有[[PlayStation_3|PS3]]、[[Xbox_360|Xbox 360]]、[[PlayStation_Portable|PSP]]、[[任天堂DS_Lite|DS Lite]]等遊戲主機<ref>{{Cite web |url= http://blog.livedoor.jp/hiyonikki/archives/884076.html |title= ぴよぴよ |work= 日笠陽子のひよっ子記 }}</ref>。

在《{{lang|ja|らじおん!}}》的企劃中，和自己演出的角色一樣實際進行樂團活動。日笠擔當BASS，並且配合秋山澪的設定使用左撇子用BASS<ref>K-ON! web radio『らじおん!』、超!A&G+『A&G REQUEST デジスタ』、『[[SEIGURA|SEIGURA]]』（2009年5月号）付録「2009新作アニメインタビュー&アフレコ 春の祭典BOOK」、『VOiCE Newtype』（No.032, 45 - 47ページ）等。</ref><ref name="声優プリンセス"/>（日笠自身是慣用右手），同時還擔任歌手。本人說雖然做起來很困難，但同時也很快樂<ref>{{Cite journal|author= 田中尚道 |date=2009年8月号增刊 |title= <kirakira girls> K-ON!（豐崎愛生×日笠陽子×佐藤聰美×寿美菜子×竹達彩奈） |journal= VOiCE Newtype|issue= 33 |pages= pp. 31-33 |publisher= [[角川書店|角川書店]]}}{{ASIN|B002DZC468}}</ref>。

《[[機動戰士GUNDAM_SEED|機動戰士GUNDAM SEED]]》系列的FAN，喜歡的角色是[[迪亞哥·艾爾斯曼|迪亞哥·艾爾斯曼]]、尼可·阿瑪菲。此外還把演出了系列主要角色的[[拉克絲·克萊因|拉克絲·克萊因]]和[[米婭·坎貝爾|米婭·坎貝爾]]的[[田中理惠_(聲優)|田中理惠]]視為憧憬的對象<ref>『アニメ星人』第11回。</ref>。擁有作品相關的全部CD<ref>超!A&G+『A&G REQUEST デジスタ』。</ref>。

印象深刻的作品是電視動畫出道作《[[Sketch_book～素描簿～|Sketchbook～full color's～]]》（根岸水面）<ref name="声優プリンセス"/>，參加面試時以很活力的感覺進行了表演，監督說「就保持這個感覺來演」而被選上了<ref name="声優プリンセス"/>。

喜歡的歌手有[[angela|angela]]、[[下川美娜|下川美娜]]、[[川嶋愛|川嶋愛]]等。在angela和下川作為guest出演《{{lang|ja|A&G REQUEST デジスタ}}》時很積極地表示是她們的FANS。也曾於同一節目在川嶋愛作為guest出演之際，在本人面前因為過於感動而哭出來。

曾經與[[大龜明日香|大龜明日香]]一齊到西武巨蛋看NANA MIZUKI LIVE DIAMOND 2009<ref>{{Cite web |url= http://blog.livedoor.jp/hiyonikki/archives/825706.html |title= きのうは |work= 日笠陽子のひよっ子記 }}</ref>。因在《[[戰姬絕唱SYMPHOGEAR|戰姬絕唱SYMPHOGEAR G]]》與[[水樹奈奈|水樹奈奈]]合唱角色歌，在NANA MIZUKI LIVE CIRCUS 2013獲得[[水樹奈奈|水樹奈奈]]的邀請在安可時候合唱「{{lang|ja|不死鳥のフランメ}}」<ref>{{Cite web |url= http://blog.livedoor.jp/hiyonikki/archives/1927538.html |title= らいぶげすと！ |work= 日笠陽子のひよっ子記 }}</ref><ref>{{Cite web |url= http://www.mizukinana.jp/blog/2013/08/post-2282.html |title= LIVE CIRCUS☆TRICK６西武ドーム１日目っ！ |work= 水樹奈々　公式サイト　NANAPARTY }}</ref><ref>{{Cite web |url= https://twitter.com/SYMPHOGEAR/status/363638114343866368|title=戰姬絕唱SYMPHOGEAR G公式Twitter}}</ref>。
