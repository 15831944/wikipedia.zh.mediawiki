'''日高光啓'''（{{jpn|j=日高 光啓|hg=ひだか みつひろ}}，1986年12月12日）是[[日本|日本]]的[[創作歌手|說唱歌手]]、[[作詞家|作詞家]]和[[演員|演員]]，日本男女[[舞蹈流行音樂|舞蹈流行音樂]]組合[[AAA_(團體)|AAA]]的四號成員。個人活動名義「'''SKY-HI'''」。[[千葉縣|千葉縣]]出身。早稻田實業學校畢業，曾就讀[[早稻田大學|早稻田大學]]社會科學部。曾屬組合Mother Ninja，名義「'''Geeny'''」。<ref>[http://talent.yahoo.co.jp/pf/detail/pp228464 日高光啓]Yahoo（人物）</ref><ref>[http://news.aol.jp/2014/12/18/hwz_skyhi/ ラッパーSKY-HI／パフォーマーAAA日高光啓　マルチに活躍するアーティストの仕事術とは]AOL Online Japan</ref>

== 經歷 ==
1986年冬，日高光啓出生在[[日本|日本]]的[[千葉縣|千葉縣]]。他是個早產兒，左耳患有先天性的聽覺障礙。<ref name=dailymusic92220>[https://web.archive.org/web/20150403205826/https://dailymusic.auone.jp/daily_interview/92220 SKY-HI 【前編】DAILY MUSIC presents SKY-HI公開取材  DAILY MUSIC]</ref>他的父親是一位國際航班的[[飛行員|飛行員]]。婴兒時期，曾在[[花王|花王]]merries紙尿褲廣告出演過兒童模特兒。

日高光啓曾希望長大成為[[足球運動員|足球運動員]]，並接受了足球訓練，但當他上到中學的時候，因為受到左耳的聽覺障礙影響，已經完全聽不清足球教練所說的話。<ref name=dailymusic92220 /> 

日高光啟放棄了足球夢想，選擇了自己第二喜歡的音樂。但同樣的音樂，日高聽到的感覺與其他人並不一樣，曾想過放棄。日高的父親鼓勵他“因為耳朵的關係，你可以聽到與眾不同的音樂，豈不更好！”，這之後日高開始以音樂作為自己的目標。<ref name=dailymusic92220 /> 
 
2001年，日高光啓加入了[[傑尼斯事務所|傑尼斯事務所]]的[[小傑尼斯|小傑尼斯]]。

2003年，日高光啓與[[西島隆弘|西島隆弘]]、[[浦田直也|浦田直也]]、[[末吉秀太|末吉秀太]]、[[與真司郎|與真司郎]]、通過愛貝克思的AAA男成員限定甄選。

2004年，日高光啓退出傑尼斯事務所。

2005年，與[[AAA_(團體)|AAA (團體)]]的成員[[西島隆弘|西島隆弘]]、[[浦田直也|浦田直也]]、[[末吉秀太|末吉秀太]]一同擔任過[[鈴木亞美|鈴木亞美]]的伴舞者。

2006年11月與akiko（アキコ）和[[m.o.v.e|m.o.v.e]]的[[motsu|motsu]]共組新團[[Mother_Ninja|Mother Ninja]]，在其中以Geeny自稱。

2007年8月22日 [[Mother_Ninja|Mother Ninja]]發行首張專輯スポンテイニアス。

2013年 以SKY-HI名義正式展開個人活動，舉行全國巡迴演唱會，於8月7日發行首張單曲「[[愛BLOOM_/_RULE|愛BLOOM / RULE]]」

== 人物 ==
===音樂風格===
* 在AAA中主要擔當說唱（Rap），而在大部分AAA的歌曲中的Rap詞都是由日高自己填詞的。但也曾被AAA的成員說過「日高因為會使用難的詞語而會令他們不明白當中意思」。
===藝人關係===
* 在AAA結成以前就認識[[宇野實彩子|宇野實彩子]]，但兩人沒有什麼實際交際，而是因為宇野的一個同班女同學是日高的朋友，有著一個共同的中間人。因為認識的早，兩人說話沒有什麽顧及，稱呼對方的時候基本都不加敬稱，而是直呼姓氏。 
*2005年，日高和宇野實彩子從各自就讀的男子高中和女子高中畢業，一起組織了兩所學校十名左右熟人參加的畢業紀念派對，這件事是日高求學生涯最难忘的事情。<ref>{{cite web |url= http://www.oricon.co.jp/music/interview/090210_03_03.html#closeup|title =出逢い別れ☆AAAのリアルな思いとは？！バレンタイン学生時代の思い出も！！？|accessdate=2009-02-10|publisher=ORICON STYLE ミュージック}}</ref> 
* 在AAA是[[末吉秀太|末吉秀太]]最常打交道的一個成員，末吉曾和另一名成員[[西島隆弘|西島隆弘]]在公司發生爭執，當時末吉將西島按倒在地，但被日高制止，末吉當時表示自己是看在日高的情面上。後來日高和末吉、西島也建立了三人之間的友誼。
