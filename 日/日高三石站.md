'''日高三石站'''（{{jpn|j=日高三石駅|hg=ひだかみついしえき|rm=Hidaka-mitsuishi eki|f=y}}）位於[[北海道|北海道]][[日高郡|日高郡]][[新日高町|新日高町]]三石旭町，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|ヒツ}}'''。以前是急行「襟裳」的停靠站。

== 車站構造 ==
[[側式月台|側式月台]]1面1線的地面站。以前是[[相對式月台|相對式月台]]2面2線，可會車。站房內設有食堂。目前站房與「交流衛星三石（{{lang|ja|ふれあいサテライト三石}}）」併設。

== 車站周邊 ==
* [[國道235號|國道235號]]
* 新日高町役場三石廳舍（原為三石町公所）
* 靜內警察署三石派出所
* 三石郵局（與日本郵便苫小牧支店三石集配中心併設）
* 日高信用金庫三石支店
* 日高漁業協同組合三石支所
* 道南巴士「三石站前」站牌

== 歷史 ==
* 1933年（[[昭和|昭和]]8年）12月15日 - 設立。
* 1977年（昭和52年）2月1日 - 廢止貨物處理的業務。
* 1984年（昭和59年）2月1日 - 廢止行李處理的業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
* 1993年（[[平成|平成]]5年） - 站房與「交流衛星三石」併設。
