'''日本樂高樂園'''（{{lang-ja|レゴランド・ジャパン}}）是位於[[日本|日本]][[名古屋市|名古屋市]][[港區_(名古屋市)|港區]][[金城碼頭|金城碼頭]]二丁目的一個[[樂高積木|樂高積木]]的[[主題樂園|主題樂園]]，也是世界第九個開業的樂高樂園。日本樂高樂園的主要以2至12歲的兒童為目標顧客，在這裡不僅可遊玩樂高積木，還有體現樂高世界觀、能使遊客主動體驗的遊樂設施。日本樂高樂園開園於2017年4月1日，2018年4月15日水族館「SEA LIFE」開幕，同月28日，「日本樂高樂園酒店」（Legoland Japan Hotel）開幕，園區正式改名為「日本樂高樂園度假區」（LEGOLAND Japan Resort）。
[[File:Legoland_Japan_Hotel_20180331-01.jpg|thumb]]

== 價格調整 ==
Legoland於2018年7月19日宣布價格調整。將夏季、年頭年末等繁忙期，與平常非繁忙期劃分，繁忙期1 day pass 大人票價為6,900yen（HKD$479、NTD$1903），非繁忙期則減至5,000yen（HKD$347、NTD$1379）；1 day pass小童票價原價為5,300yen（HKD$368、NTD$1462），繁忙期減價至4,500yen（HKD$312、NTD$1241），非繁忙期低至3,700yen（HKD$257、NTD$1020）。<ref>{{Cite web |url = https://travel.ulifestyle.com.hk/news/detail/24329 |title = 日本LEGOLAND價格調整 |date = 2018-07-19 |publisher = U Travel}}</ref>
