{{Japanese|japanese=首都|kana=しゅと|romaji=shuto}}
'''[[日本|日本]]'''的'''[[首都|首都]]'''主要隨著[[天皇|天皇]]的[[居住地|居住地]]而定，現今[[De_facto|事實上]]為'''[[东京|东京]]'''或'''[[東京都|東京都]]'''，因其是[[皇居|天皇居所]]、[[日本皇室|皇室]]與[[日本政府|中央政府]]的所在地。但由於東京的首都地位沒有[[按照法律的|法律依据]]<ref>{{Cite news| url=http://news.livedoor.com/article/detail/11960641/| title=実は正しい？「日本に首都とかねーよ」に騒然|date=2016-09-01| accessdate=2016-09-02| publisher=[[活力門|livedoor]]| language=ja}}</ref>，日本各界定義首都的標準也不一致（例如皇室居所位置、天皇[[御座|御座]]位置、[[中央政府所在地|政府駐地]]等），加上昔日[[朝廷|朝廷]]所在的[[京都|京都]]仍留有[[京都御所|舊皇宮]]、天皇御座等皇室設施，因而存在着首都是京都还是东京之争。

現行的[[日本法律|日本法律]]雖然沒有直接指定首都位置，但近代部分的[[詔書|詔書]]與廢止法律提及東京或[[東京都|東京都]]為首都。而日本的《{{link-ja|首都建设法|首都建設法}}》在1956年便被废止，<ref>{{Cite web |url = http://news.mydrivers.com/1/497/497625.htm |title = 11区人民震惊了：谷歌曝光日本根本没有法定首都 |author = 小路 |date = 2016-09-01 |publisher = 驱动之家 |language = 中文 |accessdate = 2017-02-22 }}</ref>取代其的《{{link-ja|首都圈整備法|首都圏整備法}}》則將東京都與周圍地區定為日本政治、經濟、文化等領域中心的「[[首都圈_(日本)|首都圈]]」。<ref>{{Cite journal| url=http://www.sangiin.go.jp/japanese/annai/chousa/rippou_chousa/backnumber/2009pdf/20090113196.pdf| format=PDF| title=首都を定める法律| publisher=日本{{tsl|ja|参議院事務局|參議院事務局}}| journal=立法と調査| issue=288| date=2009-01| accessdate=2016-09-02| author=藤田弓子| language=ja}}</ref>
