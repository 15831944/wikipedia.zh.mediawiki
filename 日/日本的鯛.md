'''日本海魴'''（學名：{{lang|la|''Zeus faber''}}），又稱'''鏡魚'''、'''鏡鯧'''、'''遠東海魴'''、'''馬頭鯛'''、'''的鯛'''、'''魴魚'''、'''豆的鯛'''，是[[輻鰭魚綱|輻鰭魚綱]][[海魴目|海魴目]]的其中一[[種|種]]。在台灣的高級餐廳菜單中稱為'''鲂魚'''，英文俗名為John Dory，音譯為'''多利魚'''。

台灣坊間低價、去骨去皮的多利魚、魴魚或雪斑魚<ref>[http://vivialwaysin.pixnet.net/blog/post/37917440 也想來說說關於食品添加物-我們喜歡吃的魴魚到底怎麼選呢? @ 寶貝的寶貝餐 :: 痞客邦 PIXNET ::]</ref>，皆為東南亞、越南淡水養殖的[[低眼無齒𩷶|低眼無齒𩷶]]和[[博氏巨鯰|博氏巨鯰]]（學名：{{lang|la|''Pangasius bocourti''}}，俗稱「巴沙魚」，也被用来假冒為[[龍脷|龍脷]]魚）<ref>[https://cafe.icook.tw/t/622 真的吃到高級法式多利魚嗎? 什麼是多利魚? 如何分辨真偽? - 《愛料理》討論區] {{webarchive|url=https://web.archive.org/web/20151004172732/https://cafe.icook.tw/t/622 |date=2015-10-04 }} 目前市面上多以另一種魴魚(淡水鯉科三角魴)及東南亞越南巴沙魚(芒鯰)的肉片來偽稱為『多利魚』。...台灣市面上最常偽稱多利魚或鲂魚的品種是越南芒鯰(巴沙魚-Basa)所做成的魚片。</ref>，假冒高價的深海產魴魚(以日本馬頭鯛每公斤400元為例，每片250公克的魴魚約100元，若以巴沙魚頂替，每片250公克的鯰魚只要25元)，使很多人誤以為「鯰魚」就是魴魚、多利魚或國宴魚<ref>{{cite web| url=https://cafe.icook.tw/t/622| title=魴魚(多利魚)| accessdate=2014-09-10| deadurl=yes| archiveurl=https://web.archive.org/web/20151004172732/https://cafe.icook.tw/t/622| archivedate=2015-10-04}}</ref>。最簡單的分辨方法是日本的鯛的魚片帶有魚皮，呈三角形；鯰魚的魚片沒有魚皮，呈長條形(長舌狀)。

== 分布 ==
分布於[[太平洋|太平洋]]西岸至[[印度洋|印度洋]]間。遠洋[[洄遊|洄遊]]性魚類。

== 深度 ==
水深上下限40至200公尺。

== 特徵 ==
本魚臀鰭棘4枚，背鰭硬棘部基底無棘狀板。沿背鰭軟條部和臀鰭之基底有5至7枚大形棘狀板。胸鰭短；腹鰭延長，後端達臀鰭起點之後；尾鰭截平。鱗片十分細小，牢牢附著於體側，體高甚大且側扁，吻深裂，頭大型。胸鰭上有一大黑斑，體呈[[淡褐色|淡褐色]]，體側有10條左右的長條狀縱帶，分布不平均，口部朝上且可伸出，頜可伸縮，上下頜齒小，腭骨則無齒。體長可達50公分。

== 生態 ==
本魚大多棲息於深海之泥質海底，以[[魚類|魚類]]、[[浮游生物|浮游生物]]為食物，而在夏秋兩季則可能會游到礁石附近。

== 經濟利用 ==
中大型魚類，常為底拖網捕獲，肉為鮮美，極具食用價值。
