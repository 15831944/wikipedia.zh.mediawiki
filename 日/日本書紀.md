《'''日本書紀'''》（{{jpn|j = 日本書紀|hg = やまとふみ/やまとぶみ|rm = Yamato Pumi/Yamato Bumi|kyu = |hgt = }}<ref group="註">一般讀「{{jpn|hg = にっぽんしょき|rm = Nippon Shoki|kyu = |hgt = |link = no}}」或「{{jpn|hg = にほんしょき|rm = Nihon Shoki|kyu = |hgt = |link = no}}」。</ref>）是[[日本|日本]]留傳至今最早之[[正史|正史]]，[[六國史|六國史]]之首，原名《日本紀》，[[舍人親王|-{舍}-人親王]]等人所撰，於西元681年至720年（[[養老_(日本年號)|養老]]4年）完成。記述[[神代|神代]]乃至[[持統天皇|持統天皇]]時代的歷史。全三十卷，一和二卷讲神代，三到三十卷从[[神武天皇|神武天皇]]讲到[[持統天皇|持統天皇]]，採用[[漢文|漢文]][[編年體|編年體]]寫成。[[系譜|系譜]]一卷，如今已亡佚。

== 歷史 ==
編纂的起因在於經[[壬申之亂|壬申之亂]]而取得[[政權|政權]]之[[天武天皇|天武天皇]]欲向外宣示自身之[[皇統|皇統]]而下令編成。

依《[[古語拾遺|古語拾遺]]》所述，「上古之世，未有文字，貴賤老少，口口相傳，前言往行，存而不忘。」在編纂《日本書紀》之前，並沒有正式的文獻記載，僅有口耳傳說，且[[日本|日本]]與[[朝鮮半島|朝鮮]]相同，當時以[[漢文|漢文]]作為正式的官方語言，考慮到文字之華美性，是以編纂《日本書紀》時，多半借用中國典籍上之文字，拿來描述上代所傳承之傳說，是其一大特色。

在《日本書紀》之外，當時亦有《[[古事記|古事記]]》《[[風土記|風土記]]》等等各種書籍傳世，比照其紀錄，或有重疊，或有相違，是紀錄[[口傳文學|口傳文學]]時必定產生之現象。由此可以想像，當時的傳說已有多種版本，非一時、一地、一人所造，而《日本書紀》之目的，即在編列出為準之「[[定本|定本]]」。

於《古事記》序中有-{云}-：「余聞諸家所傳[[帝紀|帝紀]]、[[本辭|本辭]]，有詭於事實者，或多加虛偽。若今不修其謬，唯恐數年之後，旨趣亡散佚矣。[[帝紀|帝紀]]、[[本辭|本辭]]，乃我國組織之理，[[天皇|天皇]]政治之基，故欲撰[[帝紀|帝紀]]、檢[[舊辭|舊辭]]，去偽、定真，使傳後世。」目的之一，在令上古[[傳說|傳說]]、[[神話|神話]]、[[歷史|歷史]]統一，有利於[[天皇|天皇]]政權。

就《日本書紀》之撰寫背景，其[[史觀|史觀]]以[[天皇|天皇]]家與[[中臣氏|中臣氏]]（[[藤原氏|藤原氏]]）為中心，而在[[齋部廣成|齋部廣成]]之《古語拾遺》提出後[[齋部|齋部]]‧[[忌部|忌部]]氏之史觀後，其他氏族亦相仿行之，遂有[[物部氏|物部氏]]史觀之《[[先代舊事本紀|先代舊事本紀]]》以及[[姓氏錄|姓氏錄]]等文獻問世。

篡改部分在於抹煞了「[[大友皇子|大友皇子]]」的[[即位|即位]]紀錄，將[[天智天皇|天智天皇]]死後定為[[天武天皇|天武天皇]][[元年|元年]]，主張[[天武天皇|天武天皇]]繼位之正統性。後來在其餘[[史料|史料]]中找到[[大友皇子|大友皇子]]之即位紀錄，是以於[[明治|明治]]三年，追諡[[弘文天皇|弘文天皇]]。

==影响==
[[大日本帝国|大日本帝国]]时期的国家格言，“[[八紘一宇|八紘一宇]]”即出自此书 「上則答-{乾}-靈授国之德，下則弘皇孫養正之心。然後，兼六合以開都，掩八紘而為宇，不亦可乎？」（{{lang|ja|上は則ち乾霊の国を授けたまいし徳に答え、下は則ち皇孫の正を養うの心を弘め、然る後、六合を兼ねて以て都を開き、八紘を掩いて宇と為さん事、亦可からずや。}}）<ref>{{Cite wikisource|title=日本書紀/卷第三#營造橿原宮}}</ref>平安时代后，新天皇登基必须要开设书纪讲席，878年后，添加了“日本纪竟宴”。
由于虚构成分较多，[[家永三郎|家永三郎]]在《日本文化史》中说《日本書紀》不是史书而是“氏族社会的产物”。

在[[天智天皇#日本國誕生|日本國誕生]]後，「日本書紀」成為使用日本國國號，建立日本國認同的第一本書籍，本書特色是將670年創立的「日本國」國號與「天皇」頭銜，追溯到西元前660年的神武天皇即位時期，並使用同一個「日本」國號，以主張全日本列島自古以來屬於天皇統治的政治區域；書中並進一步說明天皇與[[隼人|隼人]]君主雙方的祖先是由降臨[[鹿兒島縣|鹿兒島]]的[[霧島山|霧島山]]的神明後代。<ref name = "日本史的誕生：東亞視野下的日本建國史，原文名稱：日本史の誕生: 千三百年前の外圧が日本を作った_2016"> {{ cite web | title = 日本史的誕生：東亞視野下的日本建國史，原文名稱：日本史の誕生: 千三百年前の外圧が日本を作った | url = https://www.books.com.tw/products/0010730076 | publisher = 八旗文化 | author = 岡田英弘，譯者：陳心慧 | isbn = 9789869356220 | language = zh-tw | date = 2016-10-05 }} </ref>
