'''外務省'''（{{jpn|j=外務省|hg=がいむしょう}}、英語: Ministry of Foreign Affairs、縮寫：MOFA）為[[日本|日本]]對外關係事務的最高主管機關，也是日本自1885年（[[明治|明治]]十八年）實施[[內閣制|內閣制]]以來，唯一沒有變動的中央機關。
依據{{link-ja|外務省設置法|外務省設置法}}第3條規定，[[日本國政府|日本國政府]]以和平及安全為主軸推動對外關係，本身積極參與國際事務，隨著國際情勢變動而調整其外交政策，以增進國家及人民之利益。日本積極躍升國際主要大國地位，如成為[[聯合國|聯合國]][[安全理事會|安全理事會]]常任理事國此一目標，其外交系統內亦有派系，以語言可分為主要的知美、[[知華派|知華]]、知俄三派；以學校出身可分為東大派（[[東京大學|東京大學]]畢）、如水會（[[一橋大學|一橋大學]]畢）、外大派（[[東京外國語大學|東京外國語大學]]畢）和大凰會（[[創價大學|創價大學]]畢）等主要派系。

== 概要 ==
為了達成上述外務省設置法第3條之目標，外務省主掌[[外交政策|外交政策]]、外交使節、通商航海、條約等[[國際法|國際法]]規締結與運用，與外國政府交涉、情報收集、分析及發送、在外日本人保護與文化宣傳活動等國家對外事務。

1885年（[[明治|明治]]十八年）實施[[內閣制|內閣制]]以來，雖然歷經太政官達（內閣職權）、外務省官制、舊外務省設置法等不同設置根據法，然而省名未曾變動，是唯一沒有變動的中央機關。

而「外務省」的名稱最早可追溯自1869年（明治2年）8月15日開始的[[太政官制|太政官制]]（二官六省制）。

外務省首長為[[外務大臣_(日本)|外務大臣]]，[[內部部局|內部部局]]有大臣官房與綜合外交政策局等10局，[[審議會|審議會]]有外務人事審議會與海外交流審議會，[[設施等機關|設施等機關]]有外務省研修所，{{link-ja|特別機關|特別の機関}}有在外公館。

2017年（平成29年）4月1日外務省主管之[[獨立行政法人|獨立行政法人]]有[[國際協力機構|國際協力機構]]與[[國際交流基金|國際交流基金]]<ref>[http://www.soumu.go.jp/main_content/000408998.pdf 独立行政法人一覧（平成29年4月1日現在）]</ref>。

外務省所發行的文宣有《外交》<ref>[http://www.mofa.go.jp/mofaj/press/pr/gaikou/index.html 外交専門誌「外交」]　外務省</ref>與《外交論壇》（都市出版發售）<ref>「[http://www.toshishuppan.co.jp/gaikou.html 外交のページ]」　都市出版、2012年8月7日閲覧。</ref>。

外務省的標誌是來自於1937年（昭和12年）制定的職員徽章圖案，由[[漢字|漢字]]的「外」所轉變而成<ref name="sonota">「その他」『[http://www.mofa.go.jp/mofaj/annai/honsho/shiryo/qa/sonota_02.html 外務省: 外交史料　Q&A　その他]』外務省。</ref>。

== 所掌事務 ==
外務省設置法第4條名列29項所掌事務。主要如下。

* [[外交政策|外交政策]]（外務省設置法第4條第1項第１號）
* 與外國政府交涉（第2號）
* [[聯合國|聯合國]]等國際機構（第3號）
* 締結條約（第4號）
* [[國際法|國際法]]規的解釋與實施（第5號）
* 涉外法律事項（第6號）、
* 國際情勢的資訊收集及分析，以及外國與國際機構等相關調査（第7號）
* 保護並提升日本國民在海外的法律上或經濟上之利益（第8號）
* 海外國人的生命安全（第9號）
* 海外國人的身分相關事項（第10號、第11號）
* 發行[[日本護照|日本護照]]與[[旅遊警示|海外旅行]]及海外移居（第12號）
* [[簽證|簽證]]（第13號）
* 流日[[外國人|外國人]]待遇（第14號）
* 關於海外事務的國內公關事務（第15號）
* 關於日本事務的海外公關事務（同號）
* 向外國介紹[[日本文化|日本文化]]（第16號）
* 外交文書（第17號）
* 派遣[[外交官|外交官]]及[[領事|領事]]官（第18號）
* 接受外交官及領事官與國際機構要員（第19號）
* 勲章、{{link-ja|記章|記章}}、{{link-ja|日本榮典|日本の栄典}}的國際調解（第20號）
* 儀典等外交禮儀（第21號）
* 外交史料編撰（第22號）
* [[外地_(大日本帝國)|外地]]整理事務（第23號）
* [[政府開發援助|政府開發援助]]（第24～26號）

海外[[日本人|日本人]]因[[犯罪|犯罪]]行為在該地判處[[死刑|死刑]]時，作為國人保護的一環，外務省（[[外交代表機構|在外公館]]職員）應安排會面等措施。

== 廳舍 ==
外務省廳舍位在[[東京都|東京都]][[千代田區|千代田區]][[霞關|霞關]]二丁目二番一號，由本館（中央、南廳舍）、北館（北廳舍）及新廳舍排成口字型組成<ref>[http://www.mlit.go.jp/gobuild/kasumi_map_04_tyousya_gaimu.htm 官庁営繕 : 外務省庁舎] [[國土交通省|國土交通省]]、2012年8月7日閲覧。</ref>。

本館（中央、南廳舍）是地上8層、地下1層的L型建築物，1970年（昭和45年）完工，內部分為中央廳舍與南廳舍。北館（北廳舍）是地上8層、地下1層的建築物，1960年（昭和35年）完工。

新廳舍在南廳舍與北廳舍之間、中央廳舍對側，為地上7層、地下3層的建築。北廳舍與中央廳舍之間，以及南廳舍與中央廳舍之間有聯絡通道。

以前南廳舍有科學技術廳與[[公平交易委員會_(日本)|公平交易委員會]]入駐，[[中央省廳再編|中央省廳再編]]後搬出。

2002年（平成14年）初至2003年（平成15年）底，廳舍進行耐震工程，外務省另租[[港區_(東京都)|港區]][[芝公園|芝公園]]的住友不動產芝公園塔作為臨時廳舍。

===飯倉別館（飯倉公館）===
飯倉別館靠近外務事務次官官邸，位於[[東京都|東京都]][[港區|港區]][[麻布台|麻布台]]一丁目五番三號。名稱源自於當地歷史地名飯倉町。

1971年飯倉別館建成以後，與外國元首及外交部長會談或特別會議均在此處召開，在此館建成以前多是在[[迎賓館|迎賓館]]舉行。該館一部份為{{link-ja|外務省外交史料館|外務省外交史料館|外交史料館}}所用，收藏[[幕末|幕末]]至開國時期的外交歷史檔案。

附近還有舊事務次官公邸麻布台別館。

== 歷史 ==
[[File:Gaimusho_in_early_Meiji_era.jpg|thumb]]
1871年（明治3年），外務省從[[銀座|銀座]]遷入霞關，原先使用江戶時代的[[福岡藩|福岡藩]]黑田邸上屋敷，然而在1877年（明治10年）2月1日燒毀。

1881年（明治14年），由Charles Alfred Chastel de Boinville}設計的新廳舍竣工。

1993年（平成5年），新設綜合外交政策局與國際情報局。

1997年（平成9年），外務省透過借調[[國立國會圖書館|國立國會圖書館]]調査立法考査局的專門調査員松井啓，違反「圖書館自由宣言」，收集調查[[在野黨|在野黨]]國會議員在國立國會圖書館的外交、安全保障問題相關閱覽紀錄<ref>「[http://www.47news.jp/CN/201102/CN2011021901000542.html 議員の依頼資料を極秘報告　国立図書館出向の外務官僚]」 共同通信、2011年2月19日。</ref><ref>[http://www.jcp.or.jp/akahata/aik10/2011-02-22/2011022201_03_1.html 国会図書館スパイ問題　真相の徹底糾明を　市田書記局長　証人喚問を要求] しんぶん赤旗 2011年2月22日</ref><ref>[http://www.jcp.or.jp/akahata/aik10/2011-02-22/2011022202_01_1.html 国会図書館への出向者　２０年余で７人送る　外務省　幹部職員切れ目なく] しんぶん赤旗 2011年2月22日</ref>。整件事在2011年（平成23年）2月公開。

2001年（平成13年）4月，[[小泉純一郎|小泉純一郎]]就任[[內閣總理大臣|內閣總理大臣]]，[[第1次小泉內閣|第1次小泉內閣]]成立，[[田中真紀子|田中真紀子]]議員（前總理[[田中角榮|田中角榮]]長女）就任[[外務大臣_(日本)|外相]]。田中外相啟動人事凍結政策引發外務官僚反彈，陸續爆出外務省機密費挪用問題、[[公款|公款]]流用嫌疑、賄絡等醜聞，田中外相因此稱外務省為『'''伏魔殿'''』，呼籲改革。

之後田中外相與外務[[官僚|官僚]]不斷對立。田中外相透過不斷更換[[事務次官|事務次官]]企圖改革外務省，2001年（平成13年）9月11日發生[[九一一襲擊事件|九一一襲擊事件]]後，[[外交政策|外交政策]]改由[[總理大臣官邸|官邸]]主導，外相被排除在外。

接著，在[[阿富汗戰爭_(2001年)|阿富汗戰爭]]重建協助活動中，外務省疑似受到眾議員{{link-ja|鈴木宗男|鈴木宗男}}施壓，將[[非政府組織|NGO]]排除在復興會議之外，田中外相表示事務方已證實確有此事，然而事務次官{{link-ja|野上義二|野上義二}}卻否認提供資訊。引起鈴木議員、田中外相、外務省3方的鬥爭。2002年（平成14年）1月，[[小泉純一郎|小泉純一郎]]宣布更換田中外相與野上事務次官。

2002年（平成14年）3月11日，被指控對外務省施加過度壓力的鈴木宗男議員前往國會作證<ref>[http://www.shugiin.go.jp/internet/itdb_kaigiroku.nsf/html/kaigiroku/001815420020311022.htm 第154回国会 予算委員会 第22号] {{Wayback|url=http://www.shugiin.go.jp/internet/itdb_kaigiroku.nsf/html/kaigiroku/001815420020311022.htm |date=20150924122109 }} 2002年（平成14年）3月11日</ref>。

[[川口順子|川口順子]]大臣時代的2004年（平成16年）8月1日，儀典長（次官級）降格為大臣官房儀典長（局長級），領事移住部升格為{{link-ja|領事局|領事局}}，國際情報局改組為統括官組織（{{link-ja|國際情報統括官組織|国際情報統括官組織}}），條約局改組為國際法局。

2006年（平成18年）8月1日，各單位重新編制。為強化[[印度|印度]]與[[東協|東協]]關係，因此在[[亞洲大洋洲局|亞洲大洋洲局]]下設「南部亞洲部」。另一方面，經濟協力局及大臣官房國際社會協力部（[[政府開發援助|ODA]]相關事務）整合新設「國際協力局」。

2012年（平成24年）1月18日，[[野田內閣_(第1次改造)|野田第1次改造內閣]]（[[野田佳彥|野田佳彥]]總理）外相[[玄葉光一郎|玄葉光一郎]]在大臣記者會中首次引進[[英語|英語]]即時口譯，將大臣發言與國內外記者的詢問翻譯為[[日語|日語]]與英語，記者會中可借耳機聽取即時口譯<ref>読売新聞2012年1月19日13S版4面、[http://sankei.jp.msn.com/politics/news/120118/plc12011818190010-n1.htm 外相の定例記者会見に英語同時通訳] {{Wayback|url=http://sankei.jp.msn.com/politics/news/120118/plc12011818190010-n1.htm |date=20120119125209 }}</ref>。

2017年（平成29年）3月，在文部科學省違反再就職等規定事件中，爆出外務省職員違法安插至國立大學法人[[東京外國語大學|東京外國語大學]]擔任特聘教授。

2018年（平成30年）7月1日，各單位再次調整<ref>外務省組織令の一部を改正する政令(平成30年政令第193号）による改正。</ref>。亞洲大洋洲局的東北亞課分為東北亞第一課及東北亞第二課。第一課負責韓國情勢、日韓合作等，第二課負責朝鮮情勢、日朝關係等。此外，由於考慮到亞洲大洋洲相關外交政策的綜合企劃立案及調整的必要性，亞洲大洋洲局地域政策課改組為地域政策參事官（組織令上為大臣官房的參事官）<ref>[https://www.mofa.go.jp/mofaj/press/release/press4_006172.html 外務省HP　>会見・発表・広報 > 報道発表 > アジア大洋州局の組織改編]</ref>。

=== 特別機關 ===
外務省設有特別機關[[外交代表機構|在外公館]]（法律第6條）。在外公館分為[[大使館|大使館]]、公使館、總領事館、[[領事館|領事館]]、常駐代表團與日本政府在外事務所五種（法律第6條第2項、第3項）。

== 財政 ==
2018年度（平成30年度）一般會計最初預算的外務省所管預算為6,967億16萬4千日圓<ref name="予算" />。其中，外務本省5498億4705萬3千日圓、在外公館1468億5311萬1千日圓。

國會、裁判所、會計檢査院、內閣、內閣府、復興廳、總務省、法務省、外務省、財務省、文部科學省、厚生勞動省、農林水產省、經濟產業省、國土交通省、環境省及防衛省共管東日本大震災復興特別會計。

== 職員 ==
2017年7月1日的外務省一般職在職人數有5,842人（其中女性1,647人）<ref>「[http://www.cas.go.jp/jp/gaiyou/jimu/jinjikyoku/jinji_toukei.html 一般職国家公務員在職状況統計表（平成29年７月１日現在）  一般職国家公務員在職状況統計表（平成29年７月１日現在）]」</ref>。行政機關職員定員令規定之外務省員額包含特別職167人在內共有6,146人<ref name="定員令"
>「[http://elaws.e-gov.go.jp/search/elawsSearch/elaws_search/lsg0500/detail?lawId=344CO0000000121_20180401_430CO0000000075&openerCode=1 行政機関職員定員令（昭和44年5月16日政令第121号）]」（最終改正：平成30年8月31日政令第246号）</ref>。

外務省一般職職員俸給由國家公務員法與一般俸給法規定，然而在外公館職員（以下稱在外職員）則適用特別法{{link-ja|駐外使館名稱及位置暨駐外外交領事人員俸給法|在外公館の名称及び位置並びに在外公館に勤務する外務公務員の給与に関する法律}}（外務公務員給与法）。

外務省的一般職職員是非現業{{link-ja|國家公務員|国家公務員}}，勞動基本權中的爭議權與團體協約締結權不受{{link-ja|國家公務員法|国家公務員法}}認可。由於具有團結權，職員可依國公法規定自由成立或加入[[工會|勞動組合]]「{{link-ja|職員團體|職員団体}}」（國公法第108條之2第3項）。2018年3月31日現在，[[人事院|人事院]]沒有登錄的職員團體<ref>[http://www.jinji.go.jp/hakusho/pdf/29_1-3-6.pdf 平成29年度　年次報告書(公務員白書)　「第1編第3部第6章:職員団体 - 資料6-2;職員団体の登録状況。2018年3月31日現在。]</ref>。

專職職員的採用試驗有國家公務員採用綜合職試驗、一般職大學畢業程度試験（技術系）、一般職高中畢業程度試驗、以及獨自的專門職試驗外務省專門職員採用試驗（人事院規則8―18第3條）。此外，任期制職員有{{link-ja|專門調査員|専門調査員}}、在外公館派遣員、現地採用職員等。
