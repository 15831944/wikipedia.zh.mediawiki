'''高中棒球錦標賽'''，全名'''全國高等學校棒球選手權大會'''<ref>「選手權大會」即中文所稱的「{{link-en|錦標賽|Championship}}」。</ref>（{{jpn|j=全国高等学校野球選手権大会|hg=ぜんこくこうとうがっこうやきゅうせんしゅけんたいかい|rm=zenkoku kōtōgakkō yakyū senshuken taikai|f=y}}），是[[日本|日本]]的全國[[日本高中棒球|高中棒球]]聯賽之一，每年8月於[[兵庫縣|兵庫縣]][[西宮市|西宮市]]的[[阪神甲子園球場|阪神甲子園球場]]舉行；一般俗稱「'''夏季大會'''」（{{lang|ja|夏の大会}}）或「'''夏季甲子園'''」（{{lang|ja|夏の甲子園}}）<ref>[http://news.163.com/12/0809/12/88FDNATA00014JB6_2.html]，第94届夏季甲子园8日开幕。</ref>，以區別於同樣在該球場舉行的[[日本高中棒球選拔賽|高中棒球選拔賽]]（即春季甲子園）。起始於1915年，是同類型比賽中歷史最悠久的。

此聯賽由[[朝日新聞社|朝日新聞社]]與公益財團法人{{link-ja|日本高等學校野球聯盟|日本高等学校野球連盟}}（高野連）共同主辦，冠軍隊伍會被授予深紅色的{{link-ja|優勝旗|優勝旗}}。原則上參賽資格採一[[都道府縣|府縣]]一校制，不過由於[[北海道|北海道]]和[[東京都|東京都]]的學校較多，所以各被劃分割為兩個地方賽區，因此各會有兩名代表校參賽。各學校經過地方預賽後，由優勝校代表該賽區至甲子園參賽（共49隊）。此聯賽每5年（除以5餘3的年份）會辦一次盛大的紀念大賽，[[埼玉縣|埼玉縣]]、[[千葉縣|千葉縣]]、[[神奈川縣|神奈川縣]]、[[愛知縣|愛知縣]]、[[大阪府|大阪府]]、[[兵庫縣|兵庫縣]]各會有兩名代表校參賽（共55隊）。

== 概要 ==
[[File:Ceremonial_First_Pitch_1915.jpg|thumb]]社長{{link-ja|村山龍平|村山龍平}}開球。]]

[[大正|大正]]4年（1915年）由[[朝日新聞社|朝日新聞社]]發起的全國中等學校優勝野球大會在[[大阪府|大阪府]]{{link-ja|豐中運動場|豊中グラウンド|豐中球場}}展開了第一回的比賽，從此開啟了日本高中棒球聯賽的歷史。大正6年（第三回）比賽場地遷移到兵庫縣西宮的{{link-ja|鳴尾球場|鳴尾球場}}。大正13年（1924年），[[阪神甲子園球場|甲子園球場]]落成（當年為[[甲子|甲子]]年，因而命名），並從該年（第十回）開始將比賽場地遷移至此，「甲子園」也因此成為日本高中棒球的代名詞。

早期甲子園比賽沒有年齡限制，只要有學籍就可以參賽，但在1920年豐國中學派出已在法政大學有出場經驗的小方二十世選手出場，所以自1922年起，限制出場年齡為18歲以下。

2018年的大賽是夏季甲子園的[[第100回全國高等學校野球選手權紀念大會|第100屆紀念大賽]]，同時也是[[平成|平成]]時代的最後一屆比賽；應援節目《{{link-ja|熱鬪甲子園|熱闘甲子園}}》同期主題歌則為[[嵐|嵐]]的「[[夏疾風|夏疾風]]」。

== 賽制 ==
地方預賽、全國大賽都是採用單敗淘汰制。

== 电视转播 ==
[[日本放送协会|日本放送协会]]（NHK）自1954年起进行本次大会的电视转播，目前已经持续了57年。现在的本大会转播已经实现-{zh-cn:高清化;zh-tw:高畫質化}-、立体声化、解说专业化。同时NHK通过NHK WORLD PREMIUM这一频道实施部分重要赛事的国际直播。以下是NHK的转播时间表：

;NHK综合频道：
平日 8:15-11:54、13:05-18:00
周六 9:00-11:54、13:50-18:00
周日 8:35-9:00、10:05-11:54、13:05-18:00
上午9点以后会插入整点新闻播报。

;教育频道：
平日 11:54-13:05、18:00-比赛结束
土曜日 8:35-9:00、11:54-13:50、18:00-比赛结束
日曜日 9:00-10:05、11:54-13:05、18:00-比赛结束
根据各地播出需要，NHK地方支局会有针对性地自主延长转播时间

第93回（2011年）因比赛时间提前、周六周日在上午8:00（周六教育台、周日综合台）开始播出。而平日仍是8：15分播出。

在民营电视台方面，[[全日本新闻网|全日本新闻网]]会选择性地转播相当一部分地方赛事，其中位于大阪的[[朝日放送|朝日放送]]会完整地转播本大会自一回战以来的全部赛事。但是位于东京的[[朝日电视台|朝日电视台]]不会在综合频道转播赛事。此外，还有[[埼玉电视台|埼玉电视台]][[千叶电视台|千叶电视台]][[群马电视台|群马电视台]]等独立电视台，TBS旗下的[[山梨电视台|山梨电视台]][[千叶电视台|千叶电视台]]，还有CX旗下的[[富山电视台|富山电视台]]，以及相当一部分有线电视频道会进行转播。
