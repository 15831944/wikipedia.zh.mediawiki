'''[[日本|日本]][[人口|人口]]'''根據[[總務省|總務省]]統計局資料，為1億2659万2千人（2018年1月確定值）<ref>[http://www.stat.go.jp/data/jinsui/new.htm 日本統計局]</ref>，是世界人口第十一大國。在20世紀之后，由於日本進入[[後工業社會|後工業社會]]，[[人口|人口]]生育率降低，醫療條件提高；加上日本一向對外來[[移民|移民]]管制甚嚴格，導致日本成為世界上[[人口老化|人口老化]]程度最高的國家之一，同時日本也是人均壽命最長的國家之一。目前，日本總人口與勞動人口仍持續減少。

== 分布情況 ==
日本是高度[[城市化|城市化]]的國家，目前只有5%的居民為農業人口。大約有8000萬人居住在[[本州島|本州島]][[太平洋|太平洋]]側和[[九州島|九州島]]北部地區。[[東京都|東京都]]的人口數字超過1300萬，其他如[[橫濱市|橫濱市]]有3,555,473人；[[大阪市|大阪市]]有2,624,129人；[[名古屋市|名古屋市]]有2,190,549人；[[札幌市|札幌市]]有1,854,837人；[[神戶市|神戶市]]有1,513,967人；[[京都市|京都市]]有1,466,163；[[福岡市|福岡市]]有 1,556,369人；[[川崎市|川崎市]]有1,290,426人，[[北九州市|北九州市]]有1,000,211人。

== 出生率 ==
2007年2月，日本出現了四十年來首次的出生率负成長。日本由于出生率降低，未來50年內將進入人口負成長時期，不少人認為這會影響社會的安定。一些人口學家則認為日本人口减少带来的好处是使日本已經不再是個擁擠的國家，人口减少对生态环境有利。

== 問題 ==
=== 幽灵人瑞 ===
根據[[厚生勞動省|厚生勞動省]]的調查結果顯示，截至2014年9月1日全日本100歲以上的人瑞高達5萬8820人，較去年增加4423人<ref>[https://tw.news.yahoo.com/%E6%97%A5%E6%9C%AC%E7%99%BE%E6%AD%B2%E4%BA%BA%E7%91%9E%E5%A2%9E-%E9%80%A344%E5%B9%B4%E5%89%B5%E6%96%B0%E9%AB%98-065521929.html 日本百歲人瑞增 連44年創新高 中央通訊社]</ref>，然而當中有人瑞曾被發現是已死亡的“幽灵人瑞”，日本的老人家退休後可領取國民年金，有家人為了繼續領取老人家的年金和其他福利，就假裝老人家依然在生，不交「死亡届」，以他的印鑑和戶口續領年金。對此[[日本警察|日本警察]]已經開始進行詳細調查<ref>{{cite web | url=http://www.asahi.com/special/playback/TKY201007290276.html | title=都内男性最高齢１１１歳、３０年前に死亡か　自宅に遺体 | publisher=[[朝日新聞|朝日新聞]] | date=2010-07-29 | accessdate=2014-03-30}}</ref><ref>{{cite web|title=“幽灵寿星”拷问日本人情|url=http://informationtimes.dayoo.com/html/2010-09/06/content_1111848.htm|publisher=信息时报|accessdate=2014-03-29|deadurl=yes|archiveurl=https://web.archive.org/web/20100912104533/http://informationtimes.dayoo.com/html/2010-09/06/content_1111848.htm|archivedate=2010-09-12}}</ref>。並且因為逃稅、未能全面掌握國民資料、福利被濫用等問題，日本開始討論引入[[My_Number|My Number]]制度。<ref>[http://www.storm.mg/lifestyle/72873 風傳媒：社會大改革，日本人為什麼不太歡迎My Number？]</ref>

=== 自殺人口 ===
日本警察廳於2010年5月13日公佈的調查結果顯示，2009年日本共有32854人自殺，這是連續12年自殺人數突破3萬人。在各年齡層中，二三十歲的年輕人自殺率創歷史新高。根據統計數據，日本2009年的自殺人數較2008年增加了596人，這是自1978年開始統計以來連續12年突破3萬人大關。

在各年齡段分佈中，20多歲年輕人的自殺率為十万分之24.1，30多歲為十万分之26.2，比例創歷史新高。不同年齡段的自殺比率均比2008年略有增加，其中20多歲的自殺者較前年增加了0.9%；40多歲較前年增加了5.9%；50多歲人的自殺率較前年增加了2.0%，已佔到自殺者總數的19.8%；60 多歲人的自殺率較前年增加了3.9%。20-40歲人群的自殺率已高於歷史紀錄。

根據有關部門羅列出的52項自殺原因，排在第一位的是[[健康|健康]]問題，達15867人，其中死於“抑鬱症”的人排在首位，達6949人。排在第二位的是經濟、生活問題，達8377人。其中“失業”導致自殺的人數為1071；“就業失敗”為354人；“生活苦悶”為1731。這三類人群在所有自殺者中約佔4 成。另外，“無業者”則佔自殺人群的57%。

在各年齡層的自殺原因中，“[[失業|失業]]”和“生活苦悶”的比例提高。此外，超過50歲的中老年人“感到[[孤獨|孤獨]]”者也逐漸增加。

== 人口密度 ==
日本的人口密度為每平方公里337人，位居世界第30位，高于[[印度|印度]]，低于[[比利時|比利時]]。

== 年齡結構 ==
與其他后工業化國家一樣，日本面臨人口老化的問題。在1989年，老齡人口占日本人口比例為11.6%，預計在2030年會攀升到25.6%，然而事實上，現在這一數字已經達到22.6%。

== 宗教信仰 ==
[[日本國憲法|日本國憲法]]保障國民的[[宗教自由|宗教自由]]，不設國教，故[[日本宗教|日本宗教]]呈現多元性。但基於歷史因素，[[神道教|神道教]]與[[日本佛教|日本佛教]]是[[日本|日本]]的主要[[宗教|宗教]]，文化上，多數日本人同時信仰神道與[[佛教|佛教]]，[[日本|日本]]神道教約佔人口比率99%，[[佛教徒|佛教徒]]約佔人口的八成<ref>[[中央情報局|中央情報局]][[2012年|2012年]]調査</ref>。但也有許多年輕的[[日本人|日本人]]雖然依然參加宗教儀式，但卻自認為[[不可知論者|不可知論者]]或[[無神論者|無神論者]]。
