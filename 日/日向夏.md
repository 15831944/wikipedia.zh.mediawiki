'''日向夏'''（[[片假名|片假名]]: ヒュウガナツ，[[学名|学名]]：''Citrus tamurana''），是一種[[无患子目|无患子目]][[芸香科|芸香科]][[柑橘属|柑橘属]]的[[常绿植物|常绿]][[灌木|灌木]]植物，其果實可供食用。

== 命名相關 ==
其名字中的[[日向國|日向]]是指古代位在今[[宮崎縣|宮崎縣]]的一個[[令制國|令制國]]。

== 概略 ==
日向夏因其生長地位處氣候類型為[[亞熱帶氣候|亞熱帶氣候]]的[[九州_(日本)|九州]]地區而得以茁壯生長；其果實呈圓形至稍長橢圓形，其成熟者外皮顯淡黃，內裡飽含具有甜味之果汁，其中亦有果籽在核心之處。 

== 起源與產出 ==
該物種最初發現於[[宮崎縣|宮崎縣]]；主要產出於宮崎県、[[高知県|高知県]]、愛媛県、熊本県、[[伊豆半島|伊豆半島]]、神奈川県。
