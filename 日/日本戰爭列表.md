== 江戶時代（1603－1867） ==

*[[大坂之役|大坂之役]]（1614－1615年）
**大坂冬之陣（1614年）
:[[德川家康|德川家康]]藉[[方廣寺鐘銘事件|方廣寺鐘銘事件]]向[[豐臣氏|豐臣氏]]宣戰，因無法攻下[[真田幸村|真田幸村]]在[[大坂城|大坂城]]南方構築的[[真田丸|真田丸]]，[[德川家康|德川家康]]動用[[國崩|國崩]]砲擊大坂的天守閣，雙方暫時停戰。
**大坂夏之陣（1615年）
:因豐臣家拒絕德川幕府在政治上的要求，[[德川家康|德川家康]]決定再次出兵，最後德川軍勝利，[[豐臣氏|豐臣氏]]滅亡。
*[[島原之亂|島原之亂]] 1637-1638年
:德川幕府初期的大型[[一揆|一揆]]，觸動到[[德川家光|德川家光]]到[[九州|九州]]親征，可說是對[[基督教|基督教]]的戰鬥。

*[[薩英戰爭|薩英戰爭]]（1863年）
:因[[生麥事件|生麥事件]]，[[英國|英國]]向[[薩摩藩|薩摩藩]]報仇，派出軍艦炮轟薩摩藩所在地的[[鹿兒島|鹿兒島]]。

*[[下關戰爭|下關戰爭]]（1864年）
:[[長州藩|長州藩]]封鎖[[馬關海峽|馬關海峽]]，阻止外國船隻通過。四國[[列強|列強]]（[[英國|英國]]、[[法國|法國]]、[[美國|美國]]、[[荷蘭|荷蘭]])組織聯軍，強行重開海峽，與長州守軍發生激烈衝突。

*[[禁門之變|禁門之變]]（1864年）
:長州藩為了控制[[京都|京都]]政局，向京都進軍。被[[京都守護職|京都守護職]]為首的幕府軍擊敗，將長州勢力逐出京都。

*[[長州征伐|長州征伐]]（1864－1866年）

*第一次[[長州征伐|長州征伐]]（1864年）
:因長州藩在禁門之變中向[[京都御所|京都御所]]開炮，成為朝敵，幕府組織諸藩向長州進軍。長州藩內部分裂成親幕的俗論派及倒幕的正義派，俗論派打敗正義派並控制長州政局後，向幕府投降。

:第二次[[長州征伐|長州征伐]]（1866年）
:長州正義派的[[高杉晉作|高杉晉作]]起兵叛亂，擊敗俗論派控制下的長州藩政府。幕府為了消滅倒幕的正義派，再次向長州進軍。長州藩最終戰勝幕府，幕府以[[孝明天皇|孝明天皇]]去世為藉口退兵。

== 近代（1867年至今） ==
*[[戊辰戰爭|戊辰戰爭]]（1868－1869年）
:以[[薩摩藩|薩摩藩]]、[[長州藩|長州藩]]為首的討幕派對[[江戶幕府|江戶幕府]]發動的討伐戰爭，也是近代日本“建國”的統一戰爭。戰事爆發後，因幕府軍向天皇御錦旗開火，使[[德川慶喜|德川慶喜]]為首的幕府軍成為「朝賊」，德川慶喜與新政府談判後放棄抵抗([[江戶開城|江戶開城]])，幕府勢力急轉直下，之後政府軍又贏得了東北戰爭，幕府最後的殘餘勢力在北方組織政權([[蝦夷共和國|蝦夷共和國]])，但亦在[[箱館戰爭|箱館戰爭]]中遭到擊潰。最後政府軍贏得全面性勝利，成功將“天皇”權威推展至全國每一處土地。

*[[西南戰爭|西南戰爭]]（1876年）
:日本近代史上最後一次內戰。以[[西鄉隆盛|西鄉隆盛]]為首的[[薩摩|薩摩]]官兵對[[明治政府|明治政府]]宣戰，政府軍則派遣大批部隊圍剿叛軍。開戰之初西鄉軍佔上風，相繼攻佔多處重鎮。但政府大軍進入九州後，在數場激戰中擊潰人少的西鄉軍。最後西鄉隆盛自殺，戰事結束。

*[[日清戰爭|中日甲午戰爭]]（1894年－1895年）
:中日史上第四次大規模軍事衝突，也是日本近代史上首度境外大規模作戰。起因於朝鮮內亂，支持新黨的日本出兵鎮壓亂事，卻與支持舊黨的滿清發生衝突，之後清軍增援，雙方於陸地海上正式開戰。海陸戰役日軍皆大敗清軍，並佔領清軍北洋艦隊主基地[[威海衛|威海衛]]，清軍投降。雙方於和戰之間簽訂《[[馬關條約|馬關條約]]》，戰事結束。
*[[義和團運動|八国联军战役]] 日本作為八國聯軍之一出兵跟清軍和義和團交戰。
*[[日露戰爭|日俄戰爭]]（1904－1905年）
:是日本和俄羅斯帝國為爭奪在朝鮮半島和中國東北地區利益而引發的戰爭，自1904年2月8日日本海軍夜襲旅順港外俄羅斯艦隊([[仁川海戰|仁川海戰]])開始，至1905年5月27日的[[對馬海峽海戰|對馬海峽海戰]]為止，雙方至9月1日簽署休戰議定書停戰，10月14日締結日俄《朴茨茅斯和約》，結束戰爭，日本獲得勝利，得到了中國東北的控制權力與穩固對朝鮮的統治，而俄羅斯帝國從此衰敗，進而在1905年發生俄國革命。

*[[第一次世界大戰|第一次世界大戰]]（1917－1918年）

*[[九一八事變|九一八事變]]（1931年）
:中日史上第五次大規模軍事衝突。1931年日本關東軍以柳條湖事件為由，發動進攻推翻了中國東北的軍閥，並佔領整個滿州。而中方在[[國民政府|國民政府]]的不抵抗政策下棄守，全員撤退至長城之內。不久，關東軍以復興滿清為條件，說服溥儀回到東北並建立了滿州國。

*[[一·二八事變|一·二八事變]]（1931－1932年）
:又稱為第一次上海事變。因在上海租界的日僑受到排日人士的襲擊，日方遂出兵上海，迫使[[國民政府|國民政府]]一度遷都洛陽。此戰中、日雙方各投入大量部隊參戰，導致戰事陷入膠著。最後，雙方簽訂協議，日軍自上海撤退。

*[[第二次世界大戰|第二次世界大戰]]（1937－1945年）
:第二次世界大戰日本身居要角，同時出兵中國、東南亞和太平洋。起初進攻勢如破竹，在各地取得重大的成果。但隨後[[美國|美國]]參戰，日方逐漸不敵美國強大的國力與軍事力，而自各地節節敗退。最後在美國動用[[原子彈|原子彈]]、蘇聯對日宣戰等因素下，日本宣布投降，由美國為首的[[聯合國|聯合國]]接收管理。依區域的不同，日軍參與的戰事可分成下列三個部分。

*[[中国抗日战争|中国抗日战争]] （中日戰爭）（1937－1945年）
:中日史上第六次大規模軍事衝突，也是規模最大、最慘烈的一次。戰事之初日軍連戰告捷，幾乎完全佔領中國東半部國土，迫使[[國民政府|國民政府]]遷都[[重慶|重慶]]，與盤據西北的[[中國共產黨|中國共產黨]]合作抵抗。之後戰事陷入膠著，戰爭後期[[國民政府|國民政府]]在美方支援下重建軍力，逐漸收復部分領土，並馳援[[緬甸|緬甸]]戰事。最後美國動用[[原子彈|原子彈]]，日本宣布投降，中國境內戰事隨之結束。
