'''日比谷站'''（{{jpn|j=日比谷駅|hg=ひびやえき|rm=Hibiya eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[千代田區|千代田區]][[有樂町|有樂町]]一丁目，屬於[[東京地下鐵|東京地下鐵]]、[[東京都交通局|東京都交通局]]（[[都營地下鐵|都營地下鐵]]）的[[鐵路車站|鐵路車站]]。

== 停靠路線 ==
此站有東京地下鐵[[日比谷線|日比谷線]]、[[千代田線|千代田線]]與都營地下鐵[[三田線_(都營地下鐵)|三田線]]3條路線停靠。日比谷線車站編號為'''H 07'''，千代田線為'''C 09'''，三田線為'''I 08'''。

由於鄰近[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[山手線|山手線]]、[[京濱東北線|京濱東北線]]與東京地下鐵[[東京地下鐵有樂町線|有樂町線]]的[[有樂町站|有樂町站]]，因此是有樂町線的轉乘站。雖然能與JR線轉乘，但對千代田線來說距離稍遠。

日比谷線、千代田線往有樂町線有樂町站方向可利用都營三田線剪票口附近的通道。該通道由東京都交通局管轄，導覽基本上以都營線為基準。本站往東側的[[銀座站_(日本)|銀座站]]與[[東銀座站|東銀座站]]、北側的[[大手町站_(東京都)|大手町站]]、[[東京站|東京站]]皆有地下通道連結。

都營地下鐵三田線延伸至目黑以前，[[東京都交通局6300形電車|6300形]]的自動廣播轉乘內容也包含JR線。<!--各線とも有楽町線の有楽町駅との接続駅でもある。東京地下鉄の両路線からは主に三田線の構内通路を経由して乗り換えることになる。-->

== 歷史 ==
* 1964年（[[昭和|昭和]]39年）8月29日：[[帝都高速度交通營團|帝都高速度交通營團]]（營團地下鐵）日比谷線車站開業。
* 1971年（昭和46年）3月20日：營團地下鐵千代田線車站開業。
* 1972年（昭和47年）6月30日：都營地下鐵6號線車站開業。
* 1974年（昭和49年）10月30日：營團地下鐵有樂町線的有樂町站開業，開始轉乘業務。
* 1978年（昭和53年）7月1日：都營地下鐵6號線改稱三田線。
* 2004年（[[平成|平成]]16年）4月1日：營團地下鐵民營化。日比谷線、千代田線車站由東京地下鐵繼承。
* 2018年（平成30年）10月13日：千代田線月台啟用[[月台閘門|月台閘門]]。

== 車站結構 ==
=== 東京地下鐵 ===
日比谷線為[[側式月台|側式月台]]2面2線的[[地下車站|地下車站]]。剪票口位於地下2樓，月台位於地下3樓，之間有電梯與電扶梯可利用。

千代田線為[[島式月台|島式月台]]1面2線的地下車站。剪票口位於地下1樓，月台位於地下2樓，之間有電梯與電扶梯可利用。

日比谷線霞關側與千代田線二重橋前側之間有連絡通道直通。由於付費區內沒有其他連絡通道，若從不同路線的剪票口進站，須經由月台與連絡通道前往搭車月台。

每年10月{{link-ja|鐵道之日|鉄道の日}}前在[[日比谷公園|日比谷公園]]舉行「鐵道節」（鉄道フェスティバル）時，千代田線霞關側剪票口附近也會設置攤位。

=== 都營地下鐵 ===
島式1面2線的地下車站。剪票口位於地下1樓、月台位於地下2樓，之間有電梯與電扶梯可利用。

站內設有三處剪票口。

2016年3月31日以前，本站是日比谷站務管理所所在站，管理三田線[[三田站_(東京都)|三田站]] - [[千石站|千石站]]間各站。2016年4月1日起，[[東京都交通局|東京都交通局]]實施車站管區制，本站成為{{link-ja|日比谷站管區|日比谷駅管区}}所在站，前述的日比谷站務管理所廢止<ref>[http://www.reiki.metro.tokyo.jp/reiki_honbun/ag10115671.html 東京都交通局駅務管区処務規程]</ref>
<ref>現仍管理三田站～千石站，但[[春日站_(東京都)|春日站]]不在管轄範圍內，白金高輪站、白金台站由[[東京地下鐵|東京地下鐵]]管轄，目黑站由[[東京急行電鐵|東京急行電鐵]]管轄。</ref>。

大手町側的付費區外自由通道是與千代田線[[二重橋前站|二重橋前站]]共用。剪票口外有定期券售票處以及便利商店型商店「Grand Merci」（グランメルシー）。

2001年3月起數年，製藥公司三共（現[[第一三共|第一三共]]）在站內展示壁面廣告。

另外，本站剪票口附近有至地上的電梯，但由於是帝國劇場所有，有限制使用時間。

== 站名由來 ==
來自於車站附近的「日比谷公園」。

由於也鄰近皇居與皇居外苑，曾計畫作為站名但遭當時政府與[[宮內廳|宮內廳]]否決，因此改為「日比谷」。
