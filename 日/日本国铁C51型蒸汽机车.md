'''C51型'''是[[日本|日本]][[鐵道省|铁道省]]以及[[日本國有鐵道|日本国有铁道]]所使用的[[煤水车|煤水车]]式干线客运用[[蒸汽機車|蒸汽机车]]。本型机车是铁道省对从[[美国机车公司|美国机车公司]]订购的[[日本國鐵8900型蒸汽機車|8900型蒸汽机车]]的国产化改进，因此该车型最初被称为'''18900型'''，后于1928年6月改称'''C51型'''。<ref>{{Cite web|url=http://www.kurogane-rail.jp/sl/clococ51.html|title=C51型煤水车式蒸汽机车|accessdate=|author=|date=|publisher=}}</ref>

== 制造 ==
铁道省于1919年至1928年间订制本型机车289辆，由[[汽車製造_(公司)|汽车制造]]（240辆）、[[三菱重工|三菱造船]]（22辆）、铁道省滨松工厂（18辆）制造。

== 运用 ==
=== 日本国内 ===
由于C51型机车在牵引力、高速性能和可靠性方面实现了高标准，在1920年代到1930年代成为日本国内各干线的客运机车主力。1930年到1934年间[[超特急列車|超特急列車]][[燕號列車|燕號列車]]东京到[[名古屋市|名古屋]]区间的机车交路由本型机车担当，使得东京到[[神户市|神户]]的旅行时间减少9小时<ref>{{Cite book|title=日本蒸汽机车|last=直高|first=广田|publisher=讲谈社|year=1972年|isbn=0-87011-185-X|location=|pages=57页和96页}}</ref>。而配属于{{link-ja|品川机务段|品川機関区}}的C51 236和C51 239两辆机车亦被指定为[[御召列车|御召列车]]本务机车。

随着旅客输送量的增加以及钢制客车的投入运用使得旅客列车的牵引定数在增加，加上{{link-ja|C53|国鉄C53形蒸気機関車}}和{{link-ja|C59|国鉄C59形蒸気機関車}}投入运用，C51型机车遂转入地方线路运用。战后，C51型机车于1966年最终退役。

=== 中国国内 ===
1939年由[[大日本帝國陸軍|日本陆军]]征调16辆C51型机车，改造为[[標準鐵軌|标准轨距]]，交由[[華中鐵道|华中铁道]]运用于南京-上海路线，编为パシナ型<ref name=smrps7>Pashina，意指Pacifics第7型，但与[[南滿洲鐵道|南满州铁道]]パシナ型（即胜利7型蒸汽机车）为不同车型。</ref>。于中华人民共和国成立后改称为'''ㄆㄒ9型（PX9）'''，后再改称为胜利9型（SL9）。<ref>{{Cite book|title=中国蒸汽机车世纪集影|last=|first=|publisher=中国铁道出版社|year=2001年|isbn=7-113-04148-5|location=|pages=}}</ref>

== 事故 ==
{{main|松川事件}}
[[1949年|1949年]]8月17日凌晨3点9分<ref name=acci>当时适属美军占领期间，故正值[[夏時制|夏时制]]采行，其实际发生时间，应为凌晨2点9分</ref>，[[日本國有鐵道|国铁]][[東北本線|东北本线]][[福島縣|福岛县]][[松川車站_(日本)|松川站]]－[[金谷川車站|金谷川站]]间发生列车翻覆意外，本务机C51 133号机车、机后的2节行李车、1节邮政车和2节客车脱轨，造成司机等三位乘务人员死亡。根据出事现场调查结果，证实为铁轨曾经人为破坏，警方于事件发生后，陆续逮捕20名嫌疑人员，但全员于1963年的判决中，被判罪名不成立。由于迄今仍无法确认真正的犯人为何，故与同年内稍早发生之[[下山事件|下山事件]]及[[三鷹事件|三鹰事件]]，并称为“{{link-ja|国铁三大谜案|国鉄三大ミステリー事件}}”。

== 保存 ==
日本国内有4辆C51型机车静态展示：
*C51 5：曾静态展示于{{link-ja|青梅铁道公园|青梅鉄道公園}}，后送往[[埼玉縣|埼玉县]][[埼玉市|埼玉市]][[鐵道博物館_(日本)|铁道博物馆]]静态展示。
*C51 44：静态展示于{{link-ja|秋田综合车辆中心|秋田総合車両センター}}
*C51 85：静态展示于{{link-ja|鹿儿岛综合车辆中心|鹿児島車両センター}}
*C51 239：静态展示于[[京都鐵道博物館|京都鐵道博物館]]
应当注意的是，4辆静态展示的C51型机车中，C51 44、C51 85和C51 239这3辆机车皆被解剖展示。C51 239在被移交[[京都鐵道博物館|京都鐵道博物館]]前身[[梅小路蒸汽火車博物館|梅小路蒸汽火車博物館]]时，锅炉周围是新造和安装，但由于是静态展示，内部没有完全恢复。
