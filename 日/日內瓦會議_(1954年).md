'''日内瓦会议''' (La Conférence de Genève) 是1954年4月至7月在[[瑞士|瑞士]][[日内瓦|日内瓦]][[万国宫|万国宫]]召开的一次国际性多边会议。其议程是关于重建[[印度支那|印度支那]]和[[朝鮮半島|朝鲜]]的和平的问题。

==背景==
[[第二次世界大战|第二次世界大战]]结束后[[法国|法国]]企图在印度支那恢复其殖民统治，这导致了[[第一次印度支那战争|第一次印度支那战争]]的爆发。虽然法国受到[[美国|美国]]的支持，但在[[陈赓|陈赓]]率领的中国军事顾问团的[[援越抗法|援越抗法]]的帮助下，法国军队于1954年被[[胡志明|胡志明]]领导的[[越南共产党|越南独立联盟]]的人民军打败。1954年5月3日[[奠边府战役|奠边府战役]]结束后，法国{{le|拉尼埃|Joseph Laniel}}主战内阁倒台，[[孟戴斯·弗朗斯|孟戴斯·弗朗斯]]主和派组阁。1954年5月8日日内瓦印度支那和平解决方案开始谈判。<ref>[http://dag.fmprc.gov.cn/chn/dajmkf/dawz/t308974.htm 张素林，从日内瓦会议看周恩来的外交艺术，载 档案与史学2004年第4期] {{webarchive|url=https://web.archive.org/web/20090904154633/http://dag.fmprc.gov.cn/chn/dajmkf/dawz/t308974.htm |date=2009-09-04 }}</ref>

==会议經過==
在會議僵持不下期間，1954年7月3日-5日，中華人民共和國總理[[周恩來|周恩來]]在广西[[柳州|柳州]]看望越南民主共和国主席[[胡志明|胡志明]]，勸他接受「兩個越南」。北越接受了建議，各方最後达成协议。<ref>[http://www.yami.net.cn/Files/Interesting126.html 1954年的日内瓦会议埋下了中越战争的隐患]</ref><ref>{{cite web | url=http://cpc.people.com.cn/GB/69112/75843/75873/5168769.html | title=周恩来与1954年印支问题日内瓦会议——从多边外交视角进行的研究}}</ref><ref>{{cite web | url=http://www.hprc.org.cn/pdf/DANG701017.pdf | format=PDF | title=周恩来在日内瓦会议期间为恢复印度支那和平进行的努力 | publisher=当代中国研究所}}</ref>

经过8次全体会议、23次限制性会议的讨论。1954年7月21日九國外長（[[柬埔寨王国|柬埔寨王国]]、[[越南民主共和国|越南民主共和国]]、[[法国|法国]]、[[老挝王国|老挝王国]]、[[中华人民共和国|中华人民共和国]]、[[越南共和国|南越]]、[[苏联|苏联]]、[[英国|英国]]和[[美国|美国]]）共同达成《日内瓦会议最后宣言》，內容為：
# 自[[一七線|北緯十七度]]分割南北越南。
# 法兰西联邦军队与越南人民军分别在300天内分别进入分界线南北各自的集结区
# [[法國|法國]]撤出[[越南|越南]]，並承認他前殖民地三邦變為獨立國。
# 在越南境內不得設任何[[軍事基地|軍事基地]]。
# 南北越為[[中立國|中立國家]]，不得與任何國家締結軍事[[同盟|同盟]]，不得進口[[軍火|軍火]]，不得依附任何集團國家。
# 南北越在1956年7月以前實行普選，由[[普選|普選]]再統一南北越。
# 沿著北緯17度在南北两侧各不超过5公里劃出[[非軍事區|非軍事區]]。
# 由[[加拿大|加拿大]]、[[印度|印度]]、[[波蘭|波蘭]]三國成立一個國際監督委員會，負責監督雙方履行停戰協定中各項條款。
# 各國尊重三邦獨立和主權完整，不干涉三國[[內政|內政]]。<ref>袁文靖，越南戰爭史，台北：國際現勢周刊社，民國七十年十二月，頁四。</ref>

但后来只有[[法国|法国]]和[[北越|北越]]签署了这个协议。[[越南|越南]]被分为南北两个地区，这两个地区于[[越南战争|越南战争]]后统一合并。

除了美国与南越，各方谈判代表声明赞同这一宣言。美国代表声明不参与这一宣言，但“根据联合国宪章，它将不为协议执行设置障碍，也不以武力破坏协议”。南越表示不能接受这一协议。

==中英關係==
1950年1月6日，英国外交大臣[[欧内斯特·贝文|贝文]]致电[[中华人民共和国|中华人民共和国]][[周恩来|周恩来]]外长，承认[[中華人民共和國中央人民政府|中華人民共和國中央人民政府]]为“中国法律上之政府”，愿意同[[中華人民共和國|中華人民共和國]]建立外交关系，在未任命[[大使|大使]]之前，派[[胡階森|胡階森]]为临时代办。而中華人民共和國政府僅承認胡為“大不列颠及北爱尔兰联合王国政府派来[[北京|北京]]就贵我两国建立外交关系问题进行谈判的代表”。<ref>英国及锡兰政府愿与我建外交关系  我分别复电同意进行谈判，人民日报1950年1月10日，第1版</ref>

在日内瓦會議上，中英雙方展開了一系列磋商。1954年6月17日，中英双方同时发表公报：“[[中华人民共和国中央人民政府|中华人民共和国中央人民政府]]和[[联合王国|联合王国]]协议，[[中央人民政府|中央人民政府]]派遣[[代办|代办]]驻在[[伦敦|伦敦]]，其地位和任务与[[英国|英国]]驻[[北京|北京]]代办的地位和任务相同。”<ref>[http://dag.fmprc.gov.cn/chn/dajmkf/dawz/t308980.htm 张素林，新中国的半外交关系，載 世界知識2005年第1期] {{webarchive|url=https://web.archive.org/web/20090505161855/http://dag.fmprc.gov.cn/chn/dajmkf/dawz/t308980.htm |date=2009-05-05 }}</ref>
