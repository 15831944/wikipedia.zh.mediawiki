'''日本军国主义'''（{{lang-ja|軍国主義}}，{{lang|ja|ぐんこくしゅぎ}}）指的是[[大日本帝國|大日本帝國]]時期日本的一種[[意識形態|政治意識形態]]。

== 詞源和定義 ==

== 起源 ==

1868年（慶應4年）1月1日[[大日本帝國陸軍|日本陸軍]]成立，1869年（明治2年）1月1日[[大日本帝國海軍|日本海軍]]成立，1870年（明治3年）1月1日[[皇軍|皇軍]]正式成立，1873年（明治6年）實行[[徵兵|徵兵]]。1876年（明治9年）因降低[[日本武士|武士]]地位引發[[西南戰爭|西南戰爭]]，戰後[[武士道|武士道]]精神在日軍中得到恢復和發展。

[[昭和|昭和]]前期，日本兩派軍閥[[皇道派|皇道派]]及[[統制派|統制派]]爭奪對於軍部及政府內閣的控制權引發多場政變及動盪，皇道派冒然發動[[二二六事件|二二六事件]]後遭到壓抑，這場鬥爭最終由統制派勝出。

[[二戰|第二次世界大戰]]期間，日本军国主义在統制派首腦[[東條英機|東條英機]]領導下令日本走上对外扩张的道路。

== 內容 ==
*军国主义实质上的领袖是[[日本天皇|日本天皇]][[昭和天皇|昭和天皇]]
*军国主义配合[[皇民化运动|皇民化运动]]和[[内地延长主义|内地延长主义]]等殖民政策，奴役殖民地人民
*军国主义的执行者[[武部六藏|武部六藏]]、[[古海忠之|古海忠之]]、[[冈村宁次|冈村宁次]]、[[东条英机|东条英机]]等将领都服从于[[日本军|日本军]]
*军国主义的主要特征是狂热的[[日本民族主义|日本民族主义]]和[[种族主义|种族主义]]及[[民族优越感|民族优越感]]思想，军国主义分子多数都是极端民族主义分子
*军国主义源于日本的传统[[武士道|武士道]]、[[八纮一宇|八纮一宇]]、[[神道教|神道教]]等，都为民族主义下的军国主义提供温床

==圖謀復辟==

===和平憲法===
2012年起，日本首相[[安倍晉三|安倍晉三]]極力推動修改被視為和平憲法的[[日本國憲法第九條|日本國憲法第九條]]，並提出於2020年完成。此舉被很多人視為日本重新走向軍國主義。<ref>[https://cn.nytimes.com/asia-pacific/20170504/japan-constitution-shinzo-abe-military/zh-hant/ 安倍晉三欲在2020年前修訂和平憲法]紐約時報.2017-5-4</ref>

===日本教育===
2017年，右翼教育機構"森友學園"以奇低的價格從日本政府手中買下土地，被揭發與首相[[安倍晉三|安倍晉三]]有關。<ref>[http://www.bbc.com/zhongwen/trad/world-39373353 日本「愛國」學校醜聞 安倍越陷越深]BBC.2017-3-23</ref>
