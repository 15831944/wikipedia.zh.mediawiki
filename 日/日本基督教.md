'''[[日本|日本]]'''的'''[[基督教|基督宗教]]'''歷史，切確記載起始於16世紀[[安土桃山時代|安土桃山時代]]由[[日本天主教|天主教]][[傳教士|傳教士]]進行的[[傳福音|傳教]]活動，但因基督教與日本原有的[[神道|神道教]]及[[日本佛教|佛教]]發生衝突，在[[豐臣秀吉|豐臣秀吉]]當政時開始受到迫害，[[江戶時代|江戶時代]]更被大規模禁止，導致日本[[基督徒|基督徒]]的活動被迫轉入地下；至[[明治維新|明治維新]]後，日本才重新開放基督教傳播，[[新教|新教]]、[[正教會|正教會]]等其他[[基督教派系|基督教宗派]]也在此時開始傳入日本。現今根據[[日本政府|日本政府]]在2017年底的統計，日本信奉基督宗教的人口約191萬，佔日本[[日本人口|全國居住人口]]（含旅日外籍人士）約1.1%<ref>{{Cite book|url=http://www.bunka.go.jp/tokei_hakusho_shuppan/hakusho_nenjihokokusho/shukyo_nenkan/pdf/h29nenkan.pdf|title=宗教年鑑　平成29年版|pages=第35頁|publisher=[[文化廳|文化廳]]|date=|accessdate=|language=ja}}</ref>。

== 戰國時代至安土桃山時代 ==
[[File:Stain_glass_window_of_St_Francis_of_Xavier_in_Béthanie.JPG|right]]

基督教[[聶斯脫里派|聶斯脫里派]]（中国稱為[[景教|景教]]）於5世紀頃、由[[秦河勝|秦河勝]]等人傳入日本。但是、歷史的文記錄不多、不明確之處很多。

[[西班牙|西班牙]]籍的[[天主教|天主教]][[耶穌會|耶穌會]]傳教士[[方濟·沙勿略|方濟·沙勿略]]首先將基督教傳播到日本。1549年8月15日（[[聖母升天節|聖母升天節]]），沙勿略在日本朋友彌次郎的引介之下，攜同兩位耶穌會士（島來斯神父、斐迪南修士）經[[麻六甲海峽|麻六甲海峽]]，輾轉抵達日本[[九州_(日本)|九州]]南部的[[鹿兒島|鹿兒島]]。沙勿略成為第一位踏上日本國土的天主教傳教士。年底，鹿兒島已經有一百五十人、附近地區有四百五十人領洗入教。

沙勿略採用{{link-en|教會本土化|Inculturation}}的方式傳教。沙勿略登陸日本後，馬上發現日本的情況和[[印度|印度]]的大異其趣，日本人的生活習慣，思想文化原來比印度的複雜多了；他意識到不能把在印度使用的傳教方法搬到日本，在這裡首先必須學習日本語言，認識日本文化，哲學思想，並採用日本人的風俗習慣，而且要花費很長的時間才足以使一個人皈依基督。沙勿略從日本寫回[[歐洲|歐洲]]的信立刻被公布出來，吸引了無數人的興趣，也產生極大的共鳴。一時，他被推崇為現代的傳教士。

儘管沙勿略得到地方大名的支持，並打算到[[京都|京都]]謁見[[天皇|天皇]]並取得日本全國傳教之權，但兩者均未成功。他在1551年離開日本前往[[中國|中國]]傳教。這時日本有教友一千。

[[織田信長|織田信長]]對從西洋傳來的基督教非常歡迎。織田信長在[[安土城|安土城]]設有基督教的教堂，並且容許基督教的傳教士到日本傳教。信長統治市民的政策相對[[豐臣秀吉|豐臣秀吉]]和[[德川家康|德川家康]]比較寬鬆，對基督教也不例外。所以在安土桃山時代的基督教發展十分蓬勃，直到江戶時代初期才逐漸減退。

[[豐臣秀吉|豐臣秀吉]]時代開始後、基督教勢力擴大。十六世紀末葉相繼發生信奉天主教的[[吉利支丹大名|吉利支丹大名]]與[[吉利支丹|吉利支丹]]燒燬本地[[寺廟|寺廟]]和[[神社|神社]]並迫害僧侶的事件；相對的，各地信奉佛教或神道的[[大名|大名]]迫害吉利支丹的事情也時有所聞。此外，[[葡萄牙|葡萄牙]]商人將日本人當作[[奴隸|奴隸]]販賣到海外的事情爆發後，豐臣秀吉遂於[[天正_(日本)|天正]]15年（公元1587年）頒佈《[[伴天連追放令|伴天連追放令]]》，定天主教為[[邪教|邪教]]。不過著眼於[[南蠻貿易|南蠻貿易]]的實際利益，並未大規模地迫害該教。因此天主教仍在暗地裡[[布道|布道]]傳教，[[洗禮|受洗]]成為吉利支丹的日本人也不會被公家強迫還俗。不過，[[文禄|文禄]]5年（公元1596年）10月發生西班牙船艦「聖菲利浦號」漂流至[[土佐國|土佐國]]的{{link-ja|聖菲利浦號事件|サン＝フェリペ号事件}}，豐臣秀吉再度於12月8日頒布禁教令。耶穌會後進入日本傳教的[[方濟各會|方濟各會]]非常活躍，對於禁教令相當挑釁，故[[京都|京都]][[奉行|奉行]][[石田三成|石田三成]]下令逮捕京都內的方濟各會成員與基督徒，並予以處刑。

== 江戶時代 ==
德川家康統一天下後，承認基督教傳播。德川幕府第三代將軍[[德川家光|德川家光]]力主鎖國政策，1616年下令禁止[[長崎|長崎]]和[[平戶|平戶]]兩港口外的對外貿易。大名們失去貿易收入，轉向對領地民眾增加稅收來維持收入。[[島原藩|島原藩]]主[[松倉重政|松倉重政]]於1621年開始建築[[島原城|島原城]]，並鎮壓天主教徒。1637年10月，島原的[[天草四郎時貞|天草四郎時貞]]率領人民，包围[[富岡城|富岡城]]，爆發[[島原之亂|島原之亂]]。之後幕府軍連同九州眾大名，攻克原城。

幕府怀疑西方传教士与起义有关，于是驱逐葡萄牙商人，并严格禁绝天主教。幸存的教徒被迫成為了地下教徒。寛永十年（公元1633年）[[江戶幕府|江戶幕府]]採取[[鎖國政策|鎖國政策]]後、頒布了「禁教令」。基督教徒被迫更改信仰以及被流放到外國。後來[[江戶幕府|江戶幕府]]為了阻止外國傳教，決意與親天主教的西班牙及葡萄牙斷交。

== 明治維新後 ==
明治維新後重新允許基督教傳播，但[[明治憲法|明治憲法]]规定日本以神道教為國教。

1947年制定的[[日本國憲法|日本國憲法]]，完全保障宗教自由，然而基督教的影響力已不如當年在安土桃山時代傳入時強大。二戰後，日本的基督教人口只佔百分之6，而真正上教堂的信徒則僅僅只有百分之1而已，影響力遠不及日本當地的神道教及佛教。
