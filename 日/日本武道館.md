'''日本武道館'''（{{jpn|j='''日本武道館'''|hg=にっぽんぶどうかん|rm=Nippon Budōkan}}）是[[日本|日本]]的一个为鼓勵傳統的[[武道|武道]]及身心磨練的[[道場|大道場]]而興建的設施，坐落於[[東京都|東京都]][[千代田區|千代田區]][[北之丸公園|北之丸公園]]。日本武道館亦是其業主[[財團法人|財團法人]]的名稱。通稱為「'''武道館'''」。

== 概略 ==
日本武道館是為1964年舉行的[[1964年夏季奥林匹克運動會|東京奧運]]會場之一而建設，因而得名。同年10月3日落成。設計師為[[山田守|山田守]]，採用八角形的設計，屋頂樣貌則仿自[[富士山|富士山]]。門口的「武道舘」匾額，為財團法人日本武道館首任會長[[正力松太郎|正力松太郎]]之手筆。

1964年東京奧運期間，至10月20日共4日間在此舉行[[柔道|柔道]]比賽。之後主要做為柔道、[[劍道|劍道]]、[[空手道|空手道]]等[[武道|武道]]活動場地使用，同時成為[[舞蹈|舞蹈]]、[[樂團|樂團]]、演武會等表演，以及[[演唱會|演唱會]]與[[格闘技|格闘技]]（[[職業拳擊|職業拳擊]]、[[職業摔跤|職業摔跤]]、[[K-1|K-1]]、[[PRIDE|PRIDE]]）的舉行會場、[[大學|大學]]等的大規模[[入學禮|入學禮]]、[[畢業禮|畢業禮]]會場等廣泛使用。

毎年4月29日舉行的[[全日本柔道選手權大會|全日本柔道選手權大會]]（無差別級競賽，日本全國冠軍決定戰）、11月3日[[全日本劍道選手權|全日本劍道選手權]]、11月下旬的[[自衛隊音樂節|自衛隊音樂節]]皆在此舉行。[[日本天皇|日本天皇]]與[[日本內閣總理大臣|首相]]固定出席的[[全國戰歿者追悼式|全國戰歿者追悼式]]也於每年的8月15日（日本[[日本投降|二戰投降日]]）在此地舉行。

== 館内設備等 ==
* 大道場（[[競技場|競技場]]）上方天井常態掛置[[日本國旗|日本國旗]]。
* 大體育館舖設木板，作為柔道競技場使用時，會舖設[[疊|疊]]，在演唱會舉行時則放置椅子。在天井的日本國旗在演唱會或活動舉行時會降下。柔道場使用的數百枚疊，收納於體育館的地底下。（建築物的B3F）
* 一階固定席3199席，2、3階固定席7846席，立見席（3階）480席，競技場區最大2946席，最大共14471席。<ref name="gaiyou">[http://www.nipponbudokan.or.jp/about/gaiyou.html 日本武道館建物概要（アリーナ面積・座席表）]</ref>
* 大體育館亦作其他，小規模的柔道場與劍道場。
* 各武道的學生連盟事務所與[[國際武道大學|國際武道大學]]事務所設於此。
* 2020年將作為第32屆夏季奧林匹克運動會柔道與空手道比賽場館，在此之前日本武道館將首次進行改建。

== 音樂活動 ==
由於悠久的歷史（1964年建成。而最早的巨蛋[[東京巨蛋|東京巨蛋]]於1988年才建成），武道館是眾多世界頂尖樂團及歌手演出過，並錄製下「Live In Budokan」現場專輯的場地，也因此為全世界所知。故此地有著一種「聖地」及「最高殿堂」的象徵意義，而不單單只是一個體育館，能夠在此地演出是眾多歌手的目標。

* [[披頭四樂隊|披頭四樂隊]]於1966年6、7月間第一次踏上日本即是在此舉行公演，是第一個在此演出的流行樂樂團。
* [[老虎樂隊|老虎樂隊]]在1968年3月10日在此公演，是首個在武道館演出的日本樂隊。後1971年1月24日《The Tigers Beautiful Concert》告別演唱會亦在此舉行。
* [[齊柏林飛船|齊柏林飛船]]於1971年9月在此首次來日公演。
* [[深紫|深紫]]於1972年，1973年在此舉行日本公演。1972年之演出錄製為『Made in Japan』現場專輯。
* [[木匠兄妹|木匠兄妹]]1972年6月2日進行了首次的武道館公演。
* [[Eric_Clapton|Eric Clapton]]1974年10月31日首次來日公演。自此開始至今(2016年)，除1985年，1991年以外（因與披頭四成員[[George_Harrison|George Harrison]]合作音樂會），42年來每次巡迴演唱都會來到武道館，總場數破百場。1979年12月3日之公演錄製為『Just One Night』現場專輯發表。
* [[皇后樂團|皇后樂團]]1975年4月19日首次來日公演。
* [[西城秀樹|西城秀樹]]於1975年11月3日，舉行了首個日本人在武道館的個人演唱會。[[尾崎豐|尾崎豐]]原定在1992年6月16、17日舉行其首個武道館演唱會，但因意外離世而取消。 
* [[聲優|聲優]][[椎名碧流|椎名碧流]]於1997年2月22及23日在此舉行個人公演，是第一位在此演出的聲優。
* [[法蘭克·辛納屈|法蘭克·辛納屈]]1985年在此舉行演唱會。
* [[鄧麗君|鄧麗君]]曾在1985于日本東京武道館開唱，是首位在此場地演出的華人歌手。
* [[楊麗花|楊麗花]]曾在1983年和2001年于日本東京武道館演出[[唐伯虎點秋香|唐伯虎點秋香]]和[[梁山伯與祝英台|梁山伯與祝英台]]，是首位在此場地演出[[歌仔戲|歌仔戲]]的藝人。
* [[王菲|王菲]]也曾在日本東京武道館開唱。1999年，在日本武道館舉辦了兩場演唱會，2001年又在同場館舉辦了兩場演唱會。
* [[女子十二樂坊|女子十二樂坊]]曾在2004年於日本武道館舉行專場音樂會。
* [[周杰倫|周杰倫]]2008年世界巡迴演唱會日本站，2月16、17日在東京武道館展開連唱兩天的演出，是首位在此場地演出的華人男歌手。
*[[神话_(组合)|神話]]在2006年9月24日在日本東京武道館舉行《SHINHWA 2006 JAPAN TOUR INSPIRATION》演唱會，是首位在此場地演出的韓國團體。
* [[谷山紀章|谷山紀章]]於2010年5月3日以歌手KISHOW名義和吉他手[[飯塚昌明|飯塚昌明]]所組成的樂團[[GRANRODEO|GRANRODEO]]在此舉辦成立五周年紀念的武道館演唱會，也是第一位在武道館開唱的男性聲優（聲優界內為第六位）。
* [[泰勒·史薇芙特|泰勒·史薇芙特]]於2011年2月17日舉行[[愛的告白世界巡迴演唱會|愛的告白世界巡迴演唱會]]東京站第二場。
* [[宮野真守|宮野真守]]於2013年音樂活動屆滿五周年之際，以首位男性聲優SOLO歌手身份成功登上武道館舉辦個人公演。
* [[AKB48|AKB48]]姐妹團[[HKT48|HKT48]]於2013年4月27日，主流出道第39天初次登上武道館，為[[女子偶像團體|女子偶像團體]]之紀錄（《[[AKB48集團臨時總會～黑白分明！～|AKB48集團臨時總會～黑白分明！～]]》）。
* [[SILENT_SIREN|SILENT SIREN]]於2015年1月17日，以主流出道2年2個月的時間打破了[[SCANDAL|SCANDAL]]以5年7個月創下的紀錄, 成為女子樂團首次登上武道館的最快紀錄(截至2017年)。
* [[张根硕|张根硕]] 2015年6月2日-3日 举办  《The Cri Show Ⅲ》
* [[五月天|五月天]] 2015年8月28、29日舉辦《[[五月天_Just_Rock_It！就是世界巡迴演唱會|Just Rock It！就是世界巡迴演唱會]]》，是首位在此場地演出的華人樂團。2017年2月3日、4日举办《Re：DNA》演唱会。2018年5月19日、20日舉辦《[[人生無限公司巡迴演唱會|LIFE人生無限公司巡迴演唱會]]》<ref>{{Cite web |url = http://www.bin-music.com/events/mayday/2017MaydayLiFE/ |title = 五月天 LiFE 人生無限公司 巡迴演唱會 }}</ref>
*[[初音未来|初音未来]] 2015年9月4及5日举办《魔法未来2015》演唱会，是首位在日本武道馆举办演唱会的虚拟偶像。
*[[iKON|iKON]] 2016年2月15及16日舉辦《iKON Japan Tour 2016》，在日本出道後僅33天便在日本武道館舉辦演唱會，打破史上海外藝人用時最短的記錄
*ClariS 2017年2月10日舉辦 《ClariS 1st 武道館コンサート～2つの仮面と失われた太陽～》
*[[Versailles|Versailles]] 2017年2月14日情人節當天舉辦該樂團首度於日本武道館的演唱會，許多外國人慕名而來，日本內閣總理大臣夫人安倍昭恵也到場支持。
*[[M.S.S_Project|M.S.S Project]] 2017年2月18日舉辦《M.S.S Project ～光と闇のファンタジア～ FINAL》，是第一組以遊戲實況者的身分在武道館表演。
*[[BLACKPINK|BLACKPINK]] 2017年7月20日舉辦《BLACKPINK Premium Debut Showcase》，成為首組尚未在日本出道便於此地舉行公演的海外藝人。
*[[After_the_Rain|After the Rain]] 2017年8月9、10日連續兩日舉辦《After the Rain 日本武道館2017 -Clockwise/Anti-Clockwise-》，是首組以Nico唱見出身在此演出的音樂組合。
*[[Aikatsu!|Aikatsu!]]2018年2月27、28日舉辦《Aikatsu Music Festa 2018》，亦是Aikatsu歌唱擔當活動卒業前最後專場演唱會
*[[i☆Ris|i☆Ris]]2017年4月5日舉辦的《i☆Ris 4th Anniversary Live~418~》是4週年演唱會
