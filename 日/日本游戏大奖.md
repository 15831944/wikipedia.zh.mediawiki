'''日本游戏大奖'''（{{lang-ja|日本ゲーム大賞}}），是每年一度的日本[[电子游戏|电子游戏]]奖项，于1996年开始举办，主办方为[[计算机娱乐供应商协会|计算机娱乐供应商协会]]。
[[File:Japan_Game_Awards_2007_booth,_Tokyo_Game_Show_20070919.jpg|thumb]]

== 概要 ==
奖项主要分为年度作品部门、未来部门、业余部门三类，每个部门内还分别有各个奖项。
; 年度作品部门 : 对象为在限定期间内于日本国内发售的家用机游戏或电脑PC游戏。
; 未来部门 : 从第6回开始新设的部门，评选对象为在限定期间内发表或展出于[[东京电玩展|东京电玩展]]的未发售游戏。根据在东京电玩展上实施的投票，得票多的作品会被授予奖项。由于评选的是未发售作品，实际上反应的是玩家对游戏的期待程度<ref>{{Cite web|url =http://awards.cesa.or.jp/overview/overview_b_2.html|title =フューチャー部門｜表彰内容と選考基準｜|accessdate =2015-10-21|author =|date =|publisher =|language=ja |deadurl =yes|archiveurl =https://web.archive.org/web/20151022021557/http://awards.cesa.or.jp/overview/overview_b_2.html|archivedate =2015-10-22}}</ref>。另外，没有参展的厂商的作品不在评选范围内。
; 业余部门 : 2007年新设的部门，对象为日本国内的业余游戏制作团体或个人。
