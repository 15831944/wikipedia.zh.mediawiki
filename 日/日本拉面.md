日式'''拉麵'''（{{jp|j={{Audio|help=no|Ja-Ramen.oga|ラーメン}}|rm=Rāmen|lead=yes}}）是[[日本料理|日本料理]]所使用的[[麵條|麵條]]與[[麵食|麵食]]種類之一，多以切製而非拉製而成。因屬於中文[[外來語|外來語]]，故常用[[片假名|片假名]]來拼音“{{lang|ja|'''ラーメン'''}}”，[[日文汉字|日文汉字]]写法有“拉麺”、“老麺”、“柳麺”三种，在日本其他常见名称包括“'''{{lang|ja|中華そば}}'''”、“'''{{lang|ja|支那そば}}'''”或“'''{{lang|ja|南京そば}}'''”。

== 历史 ==
日本最早关于中国面条的记载是[[明朝|明朝]]遗臣[[朱舜水|朱舜水]]流亡到日本后，用面条來款待日本[[江戶時代|江戶時代]]的[[大名|大名]]——[[水户藩|水户藩]]藩主[[德川光圀|德川光圀]]，但並未普及。

“{{lang|ja|ラーメン}}”的名称的来源有多种说法。被最多人认同的说法是来自[[汉语|汉语]]的“[[拉面|拉面]]”，所以汉字写做「拉面」。另外的说法包括：老面、卤面、柳面 。

在[[明治|明治]]时代早期，拉面是[[横滨中华街|横滨中华街]]常见的食品。1900年代，来自[[上海|上海]]和[[广东|广东]]的中国人在日本卖[[切面|切面]]，配以简单的汤底和配料。在[[昭和|昭和]]年间，拉面在日本開始流行。那时拉面被叫做“{{lang|ja|支那そば}}”或“{{lang|ja|南京そば}}”（以中国都市命名乃强调其为中国产，而非特指南京）。

[[第二次世界大战|第二次世界大战]]之后，来自[[美国|美国]]的廉价面粉和从中国战场回来的士兵使得中国风味的面条大行其道。同时日本政府官方禁止使用一切关于“[[支那_(日语)|支那]]”的词汇，故民間習慣稱呼改為“{{lang|ja|中華そば}}”。“{{lang|ja|そば}}”是日本使用[[荞麦|荞麦]]製作的[[荞麦面条|荞麦面条]]。

1910年出生于[[大日本帝國|大日本帝國]][[台灣|台灣]][[嘉義廳|嘉義廳]]的[[安藤百福|安藤百福]]，在1958年发明的[[方便面|-{zh-hans:方便面;zh-hant:泡麵;}-]]（即食麵）叫“拉面”，创立[[日清|日清]]食品公司。拉面成为流行的方便食品。由此“拉面”这个词的使用也超过了其他的名称的使用。1980年代，日本拉面成了日本饮食文化的代表之一，日本各地都有人研发出别具地方风味的拉面。

近年来，由于对[[疯牛病|-{zh-hans:疯牛病;zh-hant:狂牛病;}-]]的恐惧，日本人对和拉面的顾客層相同之另一快餐食品「[[牛丼|-{zh-hans:牛肉盖浇饭;zh-hk:日式牛肉飯;zh-tw:牛丼;}-]]」的消费明显下降，拉面成了上班族的第一午餐选择。

而全球第一間取得米芝蓮一星的拉麵店「蔦」亦創新地以松露醬配搭拉面，把別具特色的拉面帶給世界<ref>[https://bkb.mpweekly.com/cu0003/20170523-33741 當松露遇上拉麵]</ref>。

== 拉麵的製作 ==
[[File:Fresh_ramen_noodle_001.jpg|thumb]]

早期最普遍的拉麵是加上[[日式叉燒|日式叉燒]]、[[筍子|筍子]]的[[醬油|醬油]]口味，但現在拉麵的口味也越來越多樣化。

將使用手工或機械製作的麵條煮好，加上利用[[骨頭|豬骨]]（日文：豚骨）或雞肉、蔬菜、小魚乾等熬煮的湯頭，大多都會再搭配[[日式叉燒|日式叉燒]]、[[筍子|筍子]]、[[蔥|蔥花]]等配料。

大部分的麵條都是使用[[麵粉|麵粉]]（小麥粉、強力粉）、[[水|水]]、盐，以及「[[鹼水|鹼水]]」（{{lang-ja|かんすい}}，又被音譯為「甘素」）為原料，顏色大多是黃色。鹼水是指[[碳酸钾|碳酸钾]]和[[碳酸钠|碳酸钠]]的混合物（有時也會加入[[磷酸|磷酸]]）。這是由於曾有人使用[[內蒙古|內蒙古]]的湖水來製作麵條，結果發現麵條變得更加好吃，因此研究了湖水的成份之後，發展出了這樣的配方。鹼水是屬於[[碱|碱性]]，會讓麵粉中的[[穀蛋白黏膠質|穀蛋白黏膠質]]產生性質變化，讓麵條具有光澤感和增加彈性，也會讓麵粉中的[[黃酮類|黃酮類]]變成黃色，讓麵條具有獨特的顏色。

日本[[第二次世界大戰|戰後]]有一段時期，出現了許多品質惡劣的「鹼水」，並且對健康可能會有不良的影響。現在[[日本農林規格|日本農林規格]]（JAS）制定了成份的規定，鹼水已經不再有安全性的問題了。此外，也有因為不喜歡鹼水那種獨特的味道，而改用[[雞蛋|雞蛋]]取代的做法。水含量（加水率）較高的麵條較柔軟，耐浸泡；水含量低的面条比较有劲，能够更多地吸收汤的味道，但容易泡烂。一般麵條中水的比例大約為20%到40%。

面条在各地粗细有所不同，但通常一家店只使用一种面。拉面店可以自己制作面条，也可以专门的面厂订做。订做时可以指定原料比例、制作方法、晾晒时间等等，以保持拉面店自己的特点。

== 湯底 ==
[[File:Japanese_Salt_flavor_Sapporo_Ramen.JPG|thumb]]
[[File:Misora-men2222222.jpg|thumb]]
拉麵的[[湯底|湯底]]大多有基本的調味材料，再添加不同的額外材料，成為各式各樣的湯頭。此外使用各地區不同的食材，也產生當地獨特的口味，使拉麵成為深入日本各地的普遍食物。

汤底的常见原料包括：[[鸡肉|鸡肉]]、[[猪骨|猪骨]]、[[牛|牛]]骨、柴鱼乾（鰹節）、[[青花魚|青花魚]]乾、[[小鱼乾|小鱼乾]]、[[海带|海带]]、炒[[黄豆|黄豆]]、[[香菇|香菇]]、[[洋葱|洋葱]]、[[葱|葱]]等等。拉面汤通常需要连续炖煮数小时甚至数天。有些拉面店使用或混用成桶买进的商业拉面汤，这种做法方便且可以降低成本，但专门的拉面饕客可以吃出其中的区别。

湯底的口味一般來說可分為[[醬油|醬油]]味、[[豚骨|豚骨]]（豬骨）味、[[鹽|鹽]]味、[[味噌|味噌]]味。

此外，也有像[[擔擔面|擔擔面]]一般使用[[唐辛子|唐辛子]]（[[辣椒|辣椒]]）的辣味和[[芝麻|芝麻]]口味的湯底、類似[[生馬麵|生馬麵]]（日本一種地方麵食）的[[醋|醋]]味湯底，以及歐式風味的[[番茄|番茄]]湯底，甚至也有[[咖哩|咖哩]]的口味。

亦有不同味道的演變，例如魚介味，主要用作混搭雞、豚湯底，配以昆布能更有效帶出鮮味<ref>[https://bkb.mpweekly.com/cu0003/20180317-68931 關於拉麵，有說「豬骨退場，魚介上位」是真的嗎？]</ref>。
=== 醬油味 ===
[[本州|本州]]最主流的风味，使用日本酱油、[[鸡|鸡]]肉和[[蔬菜|蔬菜]]。可隨個人喜好加入[[辣油|辣油]]和[[胡椒|胡椒]]。

=== 豚骨味 ===
最早發源自[[九州_(日本)|九州]]，使用猪骨长期熬炖出白色的浓汤。也可加酱油成为「豚骨酱油味」。

=== 鹽味 ===
麵汤清澈，源于[[大正|大正]]时期的[[北海道|北海道]][[函馆|函馆]]，因此又被稱為「[[函館拉麵|函館拉麵]]」。和其他风味相比，更能突出湯底材料本身的味道。

=== 味噌味 ===
使用雞肉或豬肉熬製湯底，再以日本传统的[[味噌|味噌]]酱调味。

== 其他元素 ==
除了面条和湯底，拉面的其他重要元素包括面码、酱料、香油。

通常是把湯底放在碗中，加以特制的酱料，再把煮好的面放进汤中端给食用者。面上通常會放上的配料包括：[[叉烧|叉烧]]肉、[[海苔|海苔]]、[[豆芽|豆芽]]、[[白菜|白菜]]、[[鸡蛋|鸡蛋]]、[[蒜|蒜]]末、[[笋|笋]]、[[鱼板|鱼板]]、[[玉米|玉米]]粒、[[雪菜|雪菜]]、[[土豆|土豆]]、燉肉、[[酸梅|酸梅]]等等。最后可以再加一些香油或者[[香辛料|香辛料]]。

== 日本各地方口味的拉麵 ==
日本的“三大拉面”為：[[北海道|北海道]][[札幌拉面|札幌拉面]]、[[福冈县|福冈]][[博多拉面|博多拉面]]和[[福岛县|福岛]][[喜多方拉面|喜多方拉面]]。此外，东京拉面、和歌山拉面、熊本拉面、长崎拉面等也都很有名。

=== [[北海道|北海道]] ===
*[[釧路拉麵|釧路拉麵]]（[[釧路市|釧路市]]）：湯底主要使用[[柴魚|柴魚]]，再加上[[昆布|昆布]]（即[[海带|海带]]）、小魚乾等魚類、豬骨、雞肉等，是屬於醬油風味的拉麵。味道十分清新，不會產生後勁。麵條原料中水的比例較高，是捲曲的極細麵條。

*北見拉麵（[[北見市|北見市]]）：主要使用[[洋蔥|洋蔥]]熬煮的醬油湯底。
[[File:Asahikawara-men.jpg|thumb]]
*[[旭川拉麵|旭川拉麵]]（[[旭川市|旭川市]]）：
:使用魚類和豬骨、雞肉等熬製的醬油風味。因為發源自寒冷的地區，因此為了避免熱度快速流失，添加了[[豬油|豬油]]。口味較為清淡。
:麵條為水含量較低的捲曲細麵，很容易將湯汁附著於麵上。
:雖然以醬油口味為主流進行宣傳，但也有很受歡迎的味噌口味，因此旭川拉麵等于醬油拉麵的這個認知是錯誤的。和[[札幌|札幌]]的味噌口味不同，旭川的[[味噌湯|味噌湯]]底是強調味噌獨特的甜味。
:在附近的上川町也有所謂的「上川拉麵」，但基本上和旭川拉麵類似。
*[[札幌拉麵|札幌拉麵]]（[[札幌市|札幌市]]）：
:最早起源的是鹽味拉麵，但後來卻是味噌口味成為代表。是日本各地風味拉麵中最早獲得全國性知名度的口味。
:為了增加味噌湯底的濃郁口感，加入了大量的豬油和[[蒜|蒜]]。湯底中還加入了炒過的蔬菜，將這些蔬菜與味噌一起融化之後還可以澆在白飯上，成為口味獨特的「味噌丼飯」。
:麵條的含水量很高。除了味噌口味之外，也有許多店家提供醬油口味和鹽味的拉麵，但比起其他地方較鹹和較辣。
[[File:Siora-men.jpg|thumb]]
*[[函館拉麵|函館拉麵]]（[[函館市|函館市]]）：
:以鹽味的湯底為主流，是在日本其他地區都沒有類似的當地風味。
:用小火熬煮出淡淡的[[豬骨湯|豬骨湯]]底，再加上鹽來調味，口味相當清淡。
:麵條是在拉麵中少見的「直條麵」。也有加上[[乾酪|乾酪]]粉的吃法。

== 食用 ==
[[File:High_school_students_eating_Ramen.JPG|thumb]]
[[File:Ramen_with_half_sized_fried_rice（ラーメン半チャーハンセット）.JPG|250px]]
在日本，经营拉面的有流动拉面摊、专门的“拉面屋”（通常门面较小，可以坐十几个人）、和一些普通饭馆。

食用拉面需趁热吃，在日本吃面时发出大声的唏声也不会有人认为吃相不雅。

在日本，一碗拉面通常是卖几百[[日元|日元]]。拉面店通常兼营[[炒飯|炒飯]]和[[锅贴|锅贴]]，二、三百日元一份几只锅贴。一碗拉面不够的顾客可以花一、二百日元加大面条的量（汤和面码不变），或要一份锅贴。

拉面的经营者通常会提供一些免费的[[朝鲜泡菜|朝鲜泡菜]]、[[胡椒|胡椒]]等香辛料、[[蒜酥|蒜酥]]等，供食客调味。

== 即食麵 ==
在美国2006年，十几[[美分|美分]]就可以买到一包最便宜的[[即食麵|即食麵]]。常见的品牌有[[东洋水产|东洋水产]]出品的Maruchan，[[札幌一番|札幌一番]]（Sapporo Ichiban）和日清（Nissin）。

在[[墨西哥|墨西哥]]等国家，一些迎合当地口味的[[即食麵|即食麵]]大受当地人的欢迎。

有不少西方人认为即食麵是日本人在20世纪对人类做出的最重要的发明之一。

== 其他 ==
*在欧美一些[[亚洲|亚洲]]人集中的地区，正宗的[[日本拉面店|日本拉面店]]近年来像雨后春笋一样大量开张，大有超过传统的[[寿司|寿司]]店的趋势。很多日本餐馆也兼售拉面。

*1994年在日本[[新横浜|新横浜]]落成的“新横浜拉面博物馆”，以重现1958年（发明方便拉面的那一年）的[[东京|东京]]而闻名。

*香港亦有不少拉麵店及日式食店有售拉麵，但大部分店舖所售的拉麵都是由來自香港的南天製麵麵廠出產。該麵廠由日本人創辦及主理，已有十三年歷史，每日生產超過一萬份拉麵<ref>[https://bkb.mpweekly.com/cu0003/20180319-69074 拉麵麵條「香港製造」 愈新鮮愈好？]</ref>。

== 流行文化 ==
*1999年：日本[[漫画|漫画]]《[[拉面王|拉面王]]》，是讲述一个上班族藤本浩平如何开创有自己风味的拉面的故事。
*2009年：[[好萊塢|好萊塢]]拍攝的電影《[[拉麵女孩|拉麵女孩]]》，就是描述一位美國女孩料理拉麵的酸甜喜樂故事。
*2018年：《[[愛吃拉麵的小泉同學|愛吃拉麵的小泉同學]]》，描述把吃拉麵當人生樂趣的小泉同學的故事，先後動畫化與真人化。
