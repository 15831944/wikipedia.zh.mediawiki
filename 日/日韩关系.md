'''日韩关系'''或'''韩日关系'''是指[[韩国|韩国]]与[[日本|日本]]的双边关系。1951年10月，韩日双方在美国的积极调解下，开始了邦交正常化的预备性会谈。但由于双方对日占领时期历史问题分歧过大，韩日双方在经历了13年零8个月的谈判后，才于1965年6月22日最终签署《[[韩日基本关系条约|韩日基本关系条约]]》，建立外交关系。

==歷史==
===古代===
{{main|日本与朝鲜半岛的关系}}
日本与[[朝鲜半岛|朝鲜半岛]]的最早的接触被认为可以追溯到公元前3世纪，据记载当时有人从[[百济|百济]]、[[新罗|新罗]]、[[伽耶|伽耶]]移民到[[日本|日本]]的[[九州地区|九州地区]]<ref name="koreajoongangdaily.joinsmsn.com">[http://koreajoongangdaily.joinsmsn.com/news/article/article.aspx?aid=2918261 ‘왜구 조선인설’도 폐기 … 근·현대사는 접점 못 찾아]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。同时，佛教是从[[朝鲜半岛|朝鲜半岛]]的[[新罗|新罗]]传入[[日本|日本]]的。當時百濟南部及新羅即現時韓國境內。

1592年，[[日本|日本]]入侵[[朝鲜王朝|朝鲜]]，[[万历朝鲜之役|萬曆朝鮮戰爭]]爆发，1894年, 日本在[[甲午战争|甲午战争]]後势力开始向[[朝鲜半岛|朝鲜半岛]]急剧扩张。1904年，日本与[[大韩帝国|大韩帝国]]签订第一个保护国条约。 1910年8月22日，《日韩合并条约》签订，[[朝鲜半岛|朝鲜半岛]]被日本吞并。

1945年8月15日[[日本|日本]][[无条件投降|无条件投降]]，日本对[[朝鲜半岛|朝鲜半岛]]35年的统治结束。8月21日[[苏联红军|苏联红军]]占领[[平壤|平壤]]并建立了[[苏联军事政府|苏联军事政府]]，8月25日美军登陆[[仁川|仁川]]。9月8日美国远东军司令部在[[朝鲜半岛|朝鲜半岛]]南部成立[[驻朝鲜美国陆军司令部军政厅|驻朝鲜美国陆军司令部军政厅]]（USAMGIK）。

===《旧金山和约》===
1945年8月，日本无条件投降后，韩国和日本都处在美国的占领之下。韩日之间就急需解决的问题需通过美国占领当局来协调处理。对于日本在韩国的遗留财产，美国发布军政法令宣布禁止转移或处理公有部分，并将私有部分划归为美军政当局所有。大韩民国政府成立后，美军政府根据1948年9月11日签订的《关于韩美财政及财产的协定》，将没收的日本财产转交给了新成立的韩国政府。1945年8月27日，驻日盟军最高统帅设定了“关于日本人捕鱼和捕鲸作业的成文区域”，即“{{Link-ko|麦克阿瑟线|맥아더 라인}}”，以保护水产资源和限制日本的渔业活动。但是，日本的渔船经常越线到韩国海域捕鱼。1947年2月，韩国开始扣留越线的日本渔船并将其转交[[釜山|釜山]]美军政机关处理。大韩民国成立后，韩国对越线日本渔船态度更加强硬，甚至发生过炮击事件。{{r|董向荣|方秀玉|page1=276|page2=154}}

随着冷战格局在亚洲的形成，美国迅速从压制日本的政策转变为扶植其成为美国的重要盟友。在制定《对日媾和条约》时，韩国对有关领土和撤销“{{Link-ko|麦克阿瑟线|맥아더 라인}}”，以及日本在韩财产等问题向美国提出了强烈不满。但美国对韩国的交涉反应冷淡。《[[旧金山和约|旧金山和约]]》模糊了[[独岛|独岛]]（竹岛）的主权归属。此后，日韩两国对独岛的主权问题争论不休。对于日本对其在韩财产及其财产请求权问题的处理，《旧金山和约》也没做明确的规定，而是要双方协商解决。在韩日邦交正常化的谈判过程中，日本为此提出了将韩国对日请求权与韩国取得的在韩日本财产相互抵消的“抵消論”，使韩国在谈判中处于不利地位。美国出于对冷战的考虑，将日本确定为其在亚洲政策的核心目标，从而在姌和条约中偏袒日本，限制韩国的要求，使韩日纠纷失去了一次重要的解决机会。{{r|执政党|董向荣|page1=60-61|page=277-278}}

===韩日邦交正常化===
为实现以日本为中心的亚洲[[冷战|冷战]]布局，美国积极调解韩日这两个亚洲盟国的关系。1951年10月，韩日双方在美国的敦促下，开始了邦交正常化的预备性会谈。但前五轮的会谈进展缓慢{{r|董向荣|执政党|方秀玉|page1=278|page2=62|page3=155-156}}。1959年，日本将7.5万余名韩侨遣返给朝鲜，而不是韩国，使李承晚政府大为光火{{r|方秀玉|page =168}}。[[朴正熙|朴正熙]]上台后，韩国政治文化发生了很大的变化，在对日问题上也采取了更为务实的态度。1961年10月20日，在第六轮会谈上，首席代表被换成了经济界人士，双方都希望通过经济合作来解决分歧。11月，朴正熙在访问美国途中与时任日本首相[[池田勇人|池田勇人]]就财产请求权举行了两次会谈。1962年10月和11月，[[金钟泌|金钟泌]]两次访问日本，与日本[[大平正芳|大平正芳]]外相就财产请求权问题达成初步协议。日本通过向韩国提供无偿、有偿的经济合作资金解决财产请求权问题。双方还同意以12海里基线取代“[[李承晚线|李承晚线]]”。不过，韩日两国对战争补偿款项的名称问题上严重分歧。韩方希望使用“索赔权”一词，而日方坚持“经济合作资金”一词。在第六轮谈判中，韩日两国国内的反对声音高涨。迫于压力，第六轮韩日会谈被迫中断。1963年底，朴正熙通过竞选当选韩国总统后，其政权的合法性得到了加强。1964年11月，[[佐藤荣作|佐藤荣作]]上台。双方在邦交正常化上表现出更加积极和务实的态度。1964年12月，双方举行了第七次会谈。1965年2月，时任日本外相椎名应韩国外务部长[[李东元|李东元]]的邀请访问韩国，双方草签了《[[韩日基本关系条约|韩日基本关系条约]]》。美国对这一重大进展非常满意，时任[[美国总统|美国总统]][[林登·约翰逊|林登·约翰逊]]为此对朴正熙大为赞扬{{r|董向荣|方秀玉|page1=278-279|page2=}}。

1965年6月22日，韩日双方在经历13年零8个月的谈判后，在日本首相官邸举行《韩日基本关系条约》、《关于旅日朝鲜侨民法律地位和待遇的协定》、《日韩渔业协定》、《日韩关于解决对日财产请求权和经济合作的协定》、《日韩关于文物及 合作的协定》等文件的簽署儀式。{{r|方秀玉|董向荣|page1=172|page2=280}}2005年1月17日，韩国公开了韩日会谈记录等外交文件。在第六轮会谈时，韩方为包括930081名幸存者、77603名死难者家属、25000名伤者的总计1032684名被征用者，向日方提出了总计3.64亿美元的补偿款。补偿额度为幸存者每人200美元，死者家属1650美元，伤者2000美元。双方最终商定，日本提供3亿美元无偿援助、2亿美元有偿援助，以及3亿美元商业贷款，“一次性解决”受害者索赔问题。韩国政府放弃“索赔权”，接受“经济合作”。不过，朴正熙政府并没有将赔偿金-{}-发放给受害者，而是用来发展经济，无偿资金主要被投向农、林、渔和原材料部门，贷款则被用于建设钢铁、公路等基础设施项目上。1975-1977年，朴正熙当局只是根据《对日民间索赔补偿法》等相关法律，在有限的范围内发放了少量的补偿金。{{r|方秀玉|董向荣|page1=172|page2=282-283}}

===建交后的日韩关系===
1965年，日韩建交后，两国关系得到了很大的发展。20世纪70年代，由于[[金大中绑架事件|金大中绑架事件]]和韩裔日本人[[文世光|文世光]]刺杀[[朴正熙|朴正熙]]未遂事件使两国关系走入低谷。后经美国的调解和双方努力，两国关系得到了恢复。{{r|董向荣|page=276}}

1982年，[[中曾根康弘|中曾根康弘]]出任日本首相后，韩日关系开始升温。1983年1月1日，中曾根康弘应[[全斗焕|全斗焕]]的邀请访问韩国。这是日本首相首次出访韩国。日本向韩国提供了40亿美元的贷款，帮助韩国解决20世纪70年代末经济危机所带来的困难。1984年，全斗焕对日本进行了回访。这也是韩国总统对日本的首次访问，使韩日关系发展进入一个新的阶段。{{r|方秀玉|朴钟锦|page1=194|page2=232}}

[[金泳三|金泳三]]执政时期（1993－1998），韩日关系得到进一步的发展，两国间的矛盾和障碍有所化解，日本政要在“道歉问题”上有了一定的突破。1992年，时任[[日本首相|日本首相]][[宫泽喜一|宫泽喜一]]访韩时承认日军在二战期间曾征召[[慰安妇|慰安妇]]，表示对“被征召的朝鲜妇女经受的难以用笔墨和言词形容的痛苦表示由衷的歉意并进行反省”。1993年，[[细川护熙|细川护熙]]访韩时首次表示[[日韩合并|日韩合并]]为“殖民统治”，对朝鲜半岛人民所经历的“各种各样的令人难以忍受的痛苦和悲伤”，他想以“加害人的身份从内心进行反省并深表歉意”。[[村山富市|村山富市]]执政时，他对日本在韩国的殖民统治 “表示由衷的歉意和深刻的反省”。韩日历史矛盾的化解使两国关系提高到了一个新的高度。金泳三政府提出“新的韩国与新的日本要手挽手，真正地建立一个面向未来、面向世界的互补性的伙伴关系”。{{r|方秀玉|朴钟锦|page1=211|page2=232}}

[[金大中|金大中]]执政时期（1998－2003），韩日关系进入了蜜月期。1998年，时任[[日本首相|日本首相]][[小渊惠三|小渊惠三]]提出“20世纪发生的事情，要在20世纪内解决”的政见，并邀请金大中访日。<ref >{{cite web |url =http://www.chinanews.com/hb/2010/08-26/2493119.shtml  |title=亚洲周刊：日本拒为对华侵略道歉背后 |publisher =中国新闻网 |date=2010-08-26}}</ref>同年10月，金大中在访日期间与日本签署了建立面向二十一世纪新型伙伴关系的《韩日伙伴关系共同宣言》。日本首次以联合宣言的形式向韩国就历史问题表示“痛切的反省和道歉”。<ref>{{cite web |url =http://www.people.com.cn/GB/guoji/209/4893/4895/20010508/459798.html |title=历史不容歪曲正义必胜邪恶 |publisher =人民网 |date=2001-05-08}}</ref><ref>{{cite web |url=http://www.china.com.cn/chinese/HIAW/136663.htm |title=日本概况 |publisher=中国网 |accessdate=2015-04-12}}</ref>{{r|方秀玉||page=218}}韩国同意在五年内分三个阶段解禁对日本文化产品的进口<ref>{{cite web |url=http://www.people.com.cn/GB/guoji/14549/1913572.html |title=韩国再次解禁日本文化 |publisher=人民网 |date=2003-06-13}}</ref>。韩日两国还签订了新的《韩日渔业协定》，双边关系得到提升{{r|金大中|page=460-461}}。2002年，韩日两国联合举办了[[2002年世界杯足球赛|世界杯足球赛]]<ref>{{cite web |url =http://sports.sohu.com/20090202/n262003753.shtml |title=FIFA官网宣布申办新政韩日世界杯或将独一无二 |publisher=搜狐网 |date=2009-02-02}}</ref>{{r|方秀玉|page=218}}。

[[卢武铉|卢武铉]]执政时期（2003－2008），兩國首腦曾進行互訪。但由于[[日本历史教科书问题|日本历史教科书问题]]、[[小泉纯一郎|小泉纯一郎]]参拜[[靖国神社|靖国神社]]、殖民统治时期受害者赔偿问题，又使得两国关系迅速转冷到几乎是自70年代以来的低点。日本谋求成为[[联合国|联合国]][[安理会常任理事国|安理会常任理事国]]进一步加剧了双方的矛盾。{{r|董向荣|方秀玉|page1=276|page2=224-225}}在[[安倍晉三|安倍晉三]]2006年成為首相後，韓日關系改善。

[[李明博|李明博]]执政时期（2008－2013），韩日关系得到支持。2008年2月，时任日本首相[[福田康夫|福田康夫]]在出席[[李明博|李明博]]就职典礼时，双方表示重启首脑定期会晤机制。2008年4月，在李明博访日期间，双方确定开创“更加成熟的伙伴关系”新时代。{{r|董向荣|page=281}}2011年8月，李明博在国内压力下，首次向日本提出[[慰安妇|慰安妇]]历史遗留问题。但在李明博12月份访日期间，日本对此没有做出具体答复。{{r|朴钟锦|page=298-299}}2012年8月10日，时任韩国总统李明博在[[韩国光复节|韩国光复节]]纪念日前夕登上[[独岛|独岛]]，成為首個登島的韩國總統，引发新一轮的日韩领土之争。

[[朴槿惠|朴槿惠]]执政时期（2013－2017），2013年2月22日，日本[[岛根县|岛根县]]举行“竹岛日”纪念活动，使韩日关系变得更为紧张。{{r|朴钟锦|page=313}}[[朴槿惠|朴槿惠]]出任韩国总统后，並無像過去的韓國總統般先訪美國和日本的作法，而是訪問美國和中國。她强调历史认识对双方建立面向未来的合作关系至关重要，并敦促日本采取积极措施解决慰安妇历史遗留问题。{{r|朴钟锦|page=313}}朴槿惠任內未曾訪問日本，其與日本首相[[安倍晉三|安倍晉三]]的關系亦不佳。但兩國2015年在慰安婦問題上達成《{{ja-link|韩日慰安妇协议|慰安婦問題日韓合意}}》。

[[文在寅|文在寅]]执政时期（2017－），一改朴槿惠時期的疆硬政策，雖然他反對慰安婦協議，但同意與日本首相[[安倍晉三|安倍晉三]]進行會面及對話。2017年9月文在寅在海參崴與安倍見面時表達了改善日韓關係的意願，提出「穩定管控」歷史的未決問題。與日本保持良好關係對於朝鮮問題的解決不可或缺。雖説如此，文在寅的「雙軌政策」還是讓日本難以接受。他以牢記歷史問題的「進步係政治家」的面孔，對日本發表帶有敵意的言論後，又以「國家元首」的姿態表達「面向未來」。<ref>[https://zh.cn.nikkei.com/columnviewpoint/column/28956-2018-02-12-05-00-20.html?start=1 文在寅與安倍：兩張面孔] [[日經中文網|日經中文網]] 2018-02-12</ref>但2018年下半年起，日韩关系因[[韩国大法院|韩国大法院]]判决两起日本企业赔偿二战强征的韩国劳工<ref>[http://www.xinhuanet.com/mil/2018-10/31/c_129982582.htm 韩国大法院判决日企业赔偿二战时期强征的韩国劳工]</ref><ref>[http://world.huanqiu.com/exclusive/2018-10/13403700.html 耗时13年跨国诉讼案宣判！韩法院判日企赔偿二战强征的韩国劳工]</ref><ref>[http://world.people.com.cn/n1/2018/1129/c1002-30432791.html 韩法院再就劳工诉讼案令日企赔偿]</ref>、韩国政府解散{{ja-link|和解与治愈基金会|和解・癒やし財団}}迫使《{{ja-link|韩日慰安妇协议|慰安婦問題日韓合意}}》实质性冻结<ref>[http://www.xinhuanet.com/world/2018-11/21/c_1123747470.htm 韩国宣布将解散韩日协议设立的“慰安妇”基金会]</ref><ref>[https://news.sina.com.cn/w/2018-12-29/doc-ihqhqcis1300952.shtml 慰安妇协议签署三周年 日本再次要求韩国履行协议]</ref>、韩日[[火控雷达|火控雷达]]争端逐渐恶化<ref>[http://www.xinhuanet.com/world/2018-12/28/c_1123921364.htm 日本称其巡逻机受到韩国军舰火控雷达照射]</ref>。韩国国会议长{{ja-link|文喜相}}甚至要求日本天皇就慰安妇道歉，招致日本强烈抗议<ref>[http://news.haiwainet.cn/n/2019/0213/c3541093-31495731.html 要天皇"向慰安妇道歉"惹怒安倍 韩国议长耿直回应！]</ref>。

==经贸关系==
韩日建交后，双边贸易得到迅猛发展。1966年，韩日贸易额达到3.6亿美元，首次超过了韩美贸易额（3.49亿美元）。1979年，双边贸易额突破了100亿美元。{{r|董向荣|page=281}}2013年，日本为韩国第二大进口国和第三大出口国<ref name=cia>{{cite web |url=https://www.cia.gov/library/publications/the-world-factbook/geos/ks.html |title=Korea, South |publisher= CIA World Factbook|accessdate=20115-09-18 |language=en}}</ref>。韩国为日本第六大进口国和第三大出口国<ref>{{cite web |url=https://www.cia.gov/library/publications/the-world-factbook/geos/ja.html |title=Japan |publisher= CIA World Factbook|accessdate=20115-09-18 |language=en}}</ref>。
