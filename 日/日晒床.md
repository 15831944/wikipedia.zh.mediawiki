'''日晒床'''，又稱'''太陽床'''，一種模擬[[日光浴|日光浴]]的器材，可以讓[[人類|人類]][[肌膚|肌膚]]暴露在[[人工|人工]]產生的[[紫外線|紫外線]]下，使[[黑色素|黑色素]]產生而讓[[皮膚|皮膚]]變黑的一種裝置。

== 结构 ==
日晒床结构如同[[贝壳|贝壳]]，分为底座和舱盖两部分。底座固定，舱盖可以开合，方便“日晒者”进出。底座和舱盖内侧排列有多根[[紫外线|紫外线]]灯管。

== 使用方法 ==
使用时人平躺在日晒床上，暴露需照射部位[[皮肤|皮肤]]，闭合舱盖打开[[电源|电源]]，[[紫外线|紫外线]]灯管发出的[[紫外线|紫外线]]模拟[[太陽光|自然日光]]中[[紫外线|紫外线]]对[[人体|人体]]进行照射。隔天照射十分钟左右，连续十次即可让[[皮肤|皮肤]]呈小麦色。

== 健康問題 ==
雖然日晒床被人们廣泛使用，但於它釋出的物質會致癌，因此[[世界衛生組織|世界衛生組織]]轄下的[[國際癌症研究機構|國際癌症研究機構]]（IARC）曾於1992年把太陽燈及日晒床列為「可能致癌物」，後於2009年7月改為「肯定致癌物」。

IARC在《刺血針腫瘤學》（Lancet Oncology）[[期刊|期刊]]上，發布結合二十項[[研究|研究]][[分析|分析]]的新[[報告|報告]]，把太陽燈升級為最高級別的[[致癌物|致癌物]]。根據新的評級，日晒床及其他來源的[[紫外線|紫外線]][[輻射|輻射]]，跟[[吸煙|吸煙]]、[[石棉|石棉]]、[[乙型肝炎|乙型肝炎]]及打掃煙囪一樣，肯定會[[致癌|致癌]]。其致命性相當於吸入[[芥子氣|芥子氣]]甚至[[砒霜|砒霜]]，對[[年輕人|年輕人]]尤其高危。

負責研究的Vincent Cogliano表示，一個[[健康|健康]]的人沒道理去使用它。三十歲前定期照太陽燈的人，患[[黑色素瘤|黑色素瘤]]皮膚癌的[[風險|風險]]，比沒有照燈的人高出七成半。

[[香港衛生署|香港衛生署]]表示正密切關注有關報告，並呼籲公眾避免胡亂使用。
