'''日驻华大使座车国旗被抢事件'''，2012年8月27日下午四时许，日本驻华大使[[丹羽宇一郎|丹羽宇一郎]]座车在[[北京市|北京市]]内环线上行驶返回[[日本駐華大使馆|日本駐華大使馆]]时，在北京四环路上被两辆曲线行驶的白色[[宝马|宝马]]车和黑色[[奥迪|奥迪]]车（分别悬挂京、皖车牌）尾随，随即在车流量较少路段對大使座車進行拦截逼停，迫使丹羽宇一郎的座车停下。其中一辆轿车上的一名男子将插在丹羽宇一郎座车上的[[日本国旗|日本国旗]]夺走，随即离开，未对日使馆人员造成伤害。<ref>{{Cite news|url=http://news.163.com/12/0828/20/8A17GFI600014JB6.html|title=日本驻华大使对袭击事件表示遗憾|publisher=网易网|date=2012-08-28}}</ref>

有报道称，奥迪车上的一男一女与宝马车上的两男事发前并不相识，因「看到日本国旗后，​​非常生气」，所以拦停车辆，抢夺日本国旗。中国公安机关在事后，对其中两名男子处以5天[[行政拘留|行政拘留]]的处分，9月5日将两人释放。有分析认为，该事件无疑是「中国国内反日情绪高涨的产物」<ref>{{cite web|url=http://news.wenweipo.com/2012/09/03/IN1209030046.htm |title=攔截丹羽者:看到日本國旗生氣 |publisher=文汇网，来源：环球网援引日本《产经新闻》9月3日报道|author=|date=2012-09-03 |accessdate=2012-09-08 |language=Zh-hans}}</ref>。

==民间热议==

由于事情发生时，正值[[钓鱼岛及其附属岛屿主权问题|中日钓鱼岛领土问题]]争端升级，普遍认为此事件的发生与日本在钓鱼岛领土争议问题上的态度有关。很有可能是围绕钓鱼岛问题的[[反日|反日]]活动。关于事件的定性，网民之间存在激烈的讨论。<ref>{{Cite news|url=http://www.changjiangtimes.com/2012/08/408231.html|title=日本大使在京遇袭国旗被夺 网友热议“爱国”还是“害国”|publisher=长江商报|date=2012-08-28}}</ref>随即还爆发了关于此前[[启丰二号船员2012年登上钓鱼岛事件|中国人登岛事件]]定性的讨论。

===正面定性===
部分网民对此持支持赞成态度，盛赞夺旗男子。时间后一天，8月28日，在北京有一名女子向日本大使馆投掷塑料瓶，随即被警方控制。<ref>{{Cite news|url=http://slide.news.sina.com.cn/c/slide_1_2841_25980.html|title=北京一名女子向日本大使馆投掷塑料瓶|publisher=新浪网|date=2012-08-29}}</ref>

===负面定性===
媒体方面总体持批评态度。由人民日报社主办的《环球时报》发表社评：拔日本大使座车的国旗是胡来。并描述夺旗男子：“是胡来，而且很笨很蠢。”<ref>{{Cite news|url=http://news.sina.com.cn/pl/2012-08-28/002625046582.shtml|title=环球时报社评：拔日本大使座车的国旗是胡来|publisher=新浪网|date=2012-08-29}}</ref>中国前国务委员[[唐家璇|唐家璇]]在8月29日举办的“纪念中日邦交正常化40周年国际学术研讨会”上也表示，夺旗行为“不是理性的爱国行为。”

网络上，部分网友认为夺旗行为是对亲华派[[丹羽宇一郎|丹羽宇一郎]]的打击，可能会使日本亲华派遭到打击，继而扶起日本反华派。部分网友也对丹羽宇一郎在华的工作表示肯定。

== 各方反应 == 
{{Flagicon |PRC}} [[中华人民共和国外交部发言人|中华人民共和国外交部发言人]]办公室：“中方有关部门正在对此进行认真调查。中国政府一贯认真履行《[[维也纳外交关系公约|维也纳外交关系公约]]》，保护外国驻华使馆和人员的安全。”<ref>{{Cite news |url = http://news.qq.com/a/20120828/000063.htm |title = 外交部称正在调查日驻华大使座车国旗被抢事件 |publisher = 腾讯网 |date = 2012-08-28}}</ref>随后，中国方面宣布控制了参与夺旗的人员，但未透露人数。

{{Flagicon |JPN}} [[日本外务大臣|日本外务大臣]][[玄叶光一郎|玄叶光一郎]]：“非常遗憾，国旗代表国家的尊严。日本已进行严正抗议，要求不要再发生类似问题，强烈要求中方开始刑事调查。”日方并声称已拍下夺旗男子的照片和车号，要求中方刑事调查<ref>{{Cite news |url = http://news.sina.com.cn/c/2012-08-28/080425051740.shtml |title = 日本外相要求中方依法对拔旗事件展开刑事调查 |publisher = 新浪网 |date = 2012-08-28}}</ref>。日本外务副大臣[[山口壮|山口壮]]28日抵达北京，将[[野田佳彦|野田首相]]书写的亲笔信转交给中国国家主席[[胡锦涛|胡锦涛]]。

{{Flagicon |USA}} [[美国国务院|美国国务院]]发言人[[维多利亚·纽兰|维多利亚·纽兰]]在美东时间8月21日的例行记者会上表示：“我们关注此事。如果日本大使的车辆遭到袭击的报告是真实，那将受到格外关注，特别是由于他拥有外交地位。”<ref>{{Cite news |url = http://news.eastday.com/c/20120828/u1a6819523.html |title = 美方称关注日本驻华大使车辆在京遇袭事件 |publisher = 东方网 |date = 2012-08-28}}</ref>维多利亚·纽兰在28日的例行记者会上在答记者问时表示美国采用日语读音作为钓鱼岛官方名称<ref>{{Cite news |url = http://www.people.com.cn/h/2012/0829/c25408-1-1242771452.html |title = 美国宣布采用日语读音作为钓鱼岛官方名称 |publisher = 人民网 |date = 2012-08-28}}</ref>。

==相关片段==

据相关采访，丹羽宇一郎在夺旗事件发生后选择低调处理，在当天下午5时许，按照规定仅向[[中国外交部|中国外交部]]亚洲司司长[[罗照辉|罗照辉]]提出抗议。未进一步扩大事态。但由于当日网络上出现事件现场目击者上传的照片，日本驻中国大使馆不得不在当天晚上7时许，向在华日本媒体通报了此事件，于是引起了日本国内舆论的哗然。<ref>{{Cite news|url=http://blog.sina.com.cn/s/blog_4cd1c1670102dxvs.html?tj=1|title=徐静波听丹羽大使讲述“袭车事件”|publisher=徐静波博客|date=2012-08-29}}</ref>
