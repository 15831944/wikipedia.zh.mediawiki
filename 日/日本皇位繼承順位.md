'''日本皇位繼承順序'''是依據[[皇室典範|皇室典範]]內規定的{{link-ja|皇位繼承|皇位継承}}順序所排出的[[日本皇室|日本皇室]]繼承皇位的順位。

==相關法條==
現行[[日本國憲法|日本國憲法]]中規定日本皇位為[[世襲|世襲]]制，而關於世襲方法則[[國會|國會]]所議決的皇室典範內容來決定。

皇室典範中規定只有男性皇族才擁有皇位繼承權，繼承順序依序為<ref>{{cite web |location=[[东京都|东京都]] |publisher=[[宫内厅|宫内厅]] |title=皇位繼承 |url = http://www.kunaicho.go.jp/about/seido/seido03.html |language=ja }}</ref>：
# 皇長子
# 皇長孫
# 皇長子之其他子孫
# 皇次子及其子孫
# 其他皇子孫
# 皇兄弟及其子孫
# 皇伯叔父及其子孫
# 其他皇族
