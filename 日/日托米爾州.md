'''日托米爾州'''（{{lang-uk|Житомирська область|Zhytomirs'ka oblast'}}）是[[烏克蘭|烏克蘭]]北部的一州。总面積29,832平方公里，2011年总人口1,266,938。首府[[治托米亞|日托米爾]]。

== 行政区划 ==
日托米尔州辖有23个[[区_(乌克兰)|区]]、5个市。

==人口==
日托米尔州是[[波兰裔乌克兰人|波兰裔乌克兰人]]最主要的聚集地，人口约4.9万人。

===年龄结构===
: 0-14岁：15.8% {{increase}}（男 103,194/女 97,617）
: 15-64岁：68.1% {{decrease}}（男 420,285/女 444,803）
: 65岁以上：16.1% {{decrease}}（男 65,301/女 138,472，2013年官方统计）

===年龄中位数===
: 全部人口：39.4 岁 {{increase}}
: 男性人口：36.1 岁 {{increase}}
: 女性人口：42.6 岁 {{increase}}（2013年官方统计）

== 经济 ==
日托米尔州当地经济以[[花岗岩|花岗岩]]矿业、其他建筑用石、林业、农业、机械加工业为主。该州北部地区受[[切尔诺贝利核事故|切尔诺贝利核事故]]影响较大。
