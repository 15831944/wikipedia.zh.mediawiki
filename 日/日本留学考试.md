'''日本留學試驗'''（The '''E'''xamination for '''J'''apanese '''U'''niversity Admission for International Students，簡稱'''EJU'''）是由[[日本|日本]]獨立[[行政法人|行政法人]]日本學生支援機構所主辦的一項[[考試|考試]]。<ref name="a2">參考：{{ja}}[http://www.jasso.go.jp/eju/whats_eju.html {{lang|ja|日本留学試験とは}}]（日本留學試驗說明）。</ref>

日本留學試驗主要是以希望進入日本的[[大學|大學]]、大學院或是[[專科學校|專科學校]]的外國[[留學生|留學生]]為對象，評估其[[日本語|日本語]]能力以及基礎學力為目的之共同考試。目前日本有相當數量的大學、短期大學和專科學校等，承認日本留學試驗的成績。這項考試在日本及亞洲不少地方都設有考場。<ref name="a2">參考：{{ja}}[http://www.jasso.go.jp/eju/whats_eju.html {{lang|ja|日本留学試験とは}}]（日本留學試驗說明）。</ref>不過，在中國大陸，並沒設有考場。

== 概略 ==
在施行日本留學試驗以前，如果日本以外的人士想要進入日本的大學、專科學校等[[高等教育|高等教育]]機關就讀，大多需要接受多種不同的考試，其中最基本的就是「自費外國人留學生統一考試」。自2002年開始，日本留學試驗正式施行，定於每年6月及11月在日本國內外舉辦兩場考試，同時廢除了原有的「自費外國人留學生統一考試」。<ref name="a2">參考：{{ja}}[http://www.jasso.go.jp/eju/whats_eju.html {{lang|ja|日本留学試験とは}}]（日本留學試驗說明）。</ref>

日本留學試驗的主考科目以[[日語|日語]]、[[理科|理科]]、[[數學|數學]]和綜合科目等四個項目，可以依照日本的各大學所指定的必考科目來選擇。除了日語的科目必須指定以日文出題為主之外，其他科目出題的語言可以自己選擇以英文或是日文為主。<ref name="a1">參考：{{ja}}[http://www.jasso.go.jp/eju/jp_subject.html {{lang|ja|日本留学試験出題科目}}]（日本留學試驗出題科目）。</ref>

== 接受日本留學試驗成績的日本院校 ==
*大學：約400多所；短期大學：約80所；大學院：約40多所；専門學校：約100多所。<ref>參考：{{ja}}[http://www.jasso.go.jp/eju/use.html {{lang|ja|日本留学試験利用校}}]（接受日本留學試驗成績的學校一覽）。</ref>

== 有效时间 ==
考試成績的有效期限一般而言是两年，不過需要以报考的大学而定。每所大學的規定都不盡相同，所以必須事前先確認清楚各大學的募集要項。考生可自由选择两次考试分数高的一次成绩报考大学，但是只能以同一次考试的成绩為主。例如不能把同年第1回的[[数学|数学]]成绩和第2回的[[日语|日语]]和[[理科|理科]]成绩搭配提交。另外，某些大学在录取时会指定要考生当年或去年特定一次考试的成绩，这些都需要考生到各大学咨询详细情况。

== 考试科目 ==
有些大学的某些专业会指定考生去考某些科目。
*[[日语|日语]]：选择题400分，记述（作文）50分，含说明时间共计120分钟。
*#記述問題（作文）（最高分50分、30分钟）
*#读解（阅读）（最高分200分、40分钟）
*#听读解（看材料答的听力）（最高分100分）
*#听解（无材料的听力）（最高分100分）
和[[日本语能力测试|日本语能力测试]]不同，日本留學試驗之日语考试更注重生活日语，而[[日本语能力测试|日本语能力测试]]涉及很多语法类的知识性内容。这主要是为考验留学生能否适应将来在日本的生活。

文理分（'''以下的考试范围以日本教材表示'''）
*[[数学|数学]]：从以下两个项目中选择一个项目。一个项目200分，考试时间共80分钟。
**项目1：数学コース1：考试范围「数学I（データの分析を除く）」、「数学Ａ」
**项目2：数学コース2：考试范围「数学I（データの分析を除く）」、「数学Ａ」、 「数学II」、「数学Ｂ（数列・ベクトルのみ）」、「数学III」
*[[理科|理科]]：从以下三科目中选择两个科目。每个科目100分，共200分，考试时间共80分钟。
**[[物理|物理]]  「物理基礎・物理」
**[[化学|化学]]  「化学基础・化学」
**[[生物|生物]]  「生物基础・生物」
*综合科目：200分，80分钟。
:[[政治|政治]]・[[经济|经济]]・現代社会・[[地理|地理]]・近现代[[歴史|歴史]]的综合問題。范围有限。

以上日语、数学、理科和综合科目均为选择科目，考试报名的时候可从这4个科目中选择1科目或多科目，但大多数大学的报考条件都需要考日语科目，另外报考理系专业还需要数学コース2与理科成绩，对应的，报考文系专业则需要数学コース1与综合科目成绩，但也有一些大学的文系专业只需要日语一个科目成绩，或日语与综合科目两个科目的成绩。

== 评分标准 ==
为了避免题目因难易造成不公平的問題發生，日本留學試驗基于[[项目反应理论|项目反应理论]]來评分，即解答的統計理論而採分。如此可以避免問題的難易度的不同，而產生比較有利或是不利的狀況發生。利用這樣的评分标准與目前的[[TOEIC|TOEIC]]和[[TOEFL|TOEFL]]考试相同。<ref>參考：{{ja}}[http://www.jasso.go.jp/eju/touka.html {{lang|ja|日本留学試験における得点等化について}}]（關於日本留學試驗成績的採分說明）。</ref>

== 成绩發布 ==
考试成绩将通过[[明信片|明信片]]告知考生（从2018年开始，考试从网上申请的考生可从网上查询成绩，但仍可收到实体成绩明信片）。考生将准考号填入各大学的入学申请书上后，大学会以此从[[日本学生支援機構|日本学生支援機構]]获得考生成绩和作文。如此各大学也可以参考标准自行批阅考生的作文。
