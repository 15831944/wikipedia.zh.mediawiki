'''日高幌別站'''（{{jpn|j=日高幌別駅|hg=ひだかほろべつえき|rm=Hidaka-horobetsu eki|f=y}}）位於[[北海道|北海道]][[浦河郡|浦河郡]][[浦河町|浦河町]]字西幌別，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|ヒヘ}}'''。為了與[[登別市|登別市]][[幌別車站|幌別車站]]區分，故在站名前加上「日高」。

== 車站構造 ==
* [[側式月台|側式月台]]1面1線的地面站。
* 站房內有餐廳、西幌別簡易郵局和商店，規模較大。

== 車站周邊 ==
* [[國道236號|國道236號]]、{{Link-ja|國道336號|国道336号}}
* [[浦河警察署|浦河警察署]]西幌別派出所
* 西幌別簡易郵局
* 日高東農業協同組合（JA日高東）幌別事業所
* {{link-ja|浦河優駿Village AERU|うらかわ優駿ビレッジAERU}}
* 浦河町立浦河東部小學
* 浦河町立浦河第二中學
* 日高種畜牧場
* 日本中央競馬会日高牧場
* 清河町立伏木田光夫美術人館
* 清河町立鄉土博物館
* 巴士站
** JR北海道巴士：往[[浦河站|浦河站]]、向別、上野深或往[[樣似站|樣似站]]、樣似營業所前
** 日交巴士：往上杵臼、鵜苫澤

== 歷史 ==
* 1937年8月10日 - 設立。
* 1960年4月1日 - 業務委託化。
* 1977年2月1日 - 廢止貨物和行李處理的業務。無人化。
* 1987年4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
