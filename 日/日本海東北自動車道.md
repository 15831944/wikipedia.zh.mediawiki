'''日本海東北自動車道'''（{{jpn|j=日本海東北自動車道|hg=にほんかいとうほくじどうしゃどう|rm=Nihonkai Tōhoku jidōshadō|f=y}}）是[[新潟縣|新潟縣]][[新潟市|新潟市]][[江南區_(新潟市)|江南區]][[新潟中央系統交流道|新潟中央系統交流道]]為起點，經過[[山形縣|山形縣]]，終於[[秋田縣|秋田縣]][[秋田市|秋田市]][[河邊系統交流道|河邊系統交流道]]的[[日本高速公路|高速公路]]。

[[縮寫|略稱]]為'''日本海東北道'''（{{lang|ja|にほんかいとうほくどう}}、NIHONKAI-TOHOKU EXPWY）、'''日東道'''（{{lang|ja|にっとうどう}}、NITTO EXPWY）。主線上的[[道路交通標誌|道路標識]]有這兩種的略稱方式，但[[無線電|無線電]]等交通資訊多以日本海東北道稱呼之。

從象潟交流道到本莊交流道為[[國道7號_(日本)|一般國道]]快速道路，其他路段則為[[高速自動車國道|高速自動車國道]]。

截止至2015年10月，新潟中央系統交流道（新潟縣新潟市江南區） - [[朝日真秀場交流道|朝日真秀場交流道]]（新潟縣[[村上市|村上市]]）間、溫海溫泉交流道（山形縣[[鶴岡市|鶴岡市]]） - [[酒田湊交流道|酒田湊交流道]]（山形縣[[酒田市|酒田市]]）間、象潟交流道（秋田縣[[仁賀保市|仁賀保市]]） - 河邊系統交流道（秋田縣秋田市）間已經通車。

高速公路路線編號與[[秋田自動車道|秋田自動車道]]河邊系統交流道 - 小坂系統交流道區間共同編為「E7」。

== 概要 ==
本道路是連接[[裏日本|日本海側]]縱貫軸之[[新潟县|新潟縣]]、[[山形縣|山形縣]][[庄内地方|庄内地方]]、[[秋田縣|秋田縣]]等三地域的高速公路。與內陸側的[[東北自動車道|東北自動車道]]和[[表日本|太平洋側]]的[[常磐自動車道|常磐自動車道]]共同組成縱貫[[东北地方_(日本)|東北地方]]的一條路線。幾乎全線皆與[[國道7號_(日本)|國道7號]]並行，若全線通車後將是從[[近畿地方|近畿圏]]與[[中京圈|中京圏]]道[[北東北|北東北]]的最短路線。

[[荒川胎内交流道|荒川胎内交流道]] - [[朝日真秀場交流道|朝日真秀場交流道]]間、溫海溫泉交流道 - [[鶴岡系統交流道|鶴岡系統交流道]]間，[[本莊交流道|本莊交流道]] - [[岩城交流道|岩城交流道]]採用'''[[新直轄方式|新直轄方式]]'''管理。前述區間除了鶴岡西交流道 - 鶴岡系統交流道區間外<ref>鶴岡西IC - 鶴岡JCT間にはインターチェンジがないため、この区間を走行する場合は鶴岡JCT以北の有料区間を走行する必要がある。</ref>，其他皆為'''[[免費|免費]]'''。

通過豪雪地帶，[[冬季|冬季]]時經常以普通輪胎的車輛為對象實施[[雪鍊|雪鏈管制]]。

從[[2010年|2010年]][[6月28日|6月28日]]開始，收費路段的新潟中央系統交流道 - 荒川胎内交流道、岩城交流道 - 河邊系統交流道間實施[[高速公路免費化社會實驗|免費化社會實驗]]，[[2011年|2011年]][[3月11日|3月11日]]發生[[2011年日本東北地方太平洋近海地震|東北地方太平洋沖地震]]（[[東日本大震災|東日本大震災]]），政府為了為確保復興費用，同年[[6月19日|6月19日]]結束實驗<ref name="mlit">{{PDFlink|[http://www.mlit.go.jp/common/000146807.pdf 国土交通省 1次補正を踏まえた高速道路の料金について]}}PDF</ref>。但是自翌日起，以東日本大震災的受災者支援與復興、復舊支援為目的的部分車輛在通行同區間時的免費措施開始實行，2012年3月31日日本海東北道免費措施終止<ref name="nexco20110617">[http://www.e-nexco.co.jp/road_info/important_info/h23/0617/ NEXCO東日本 お客さまへの大切なお知らせ 東北地方の高速道路無料措置通行に関するよくあるご質問について] {{webarchive|url=https://web.archive.org/web/20111126230753/http://www.e-nexco.co.jp/road_info/important_info/h23/0617/ |date=2011-11-26 }}
</ref>。

此外，長時間作為基本計畫區間的朝日真秀場 - 溫海溫泉間、遊佐 - 象潟間的兩區間在2013年（平成25年）5月15日分別以日本海沿岸東北自動車道（國道7號）朝日溫海道路、遊佐象潟道路的名義事業化<ref name="2013jigyoka">{{Cite web|url=http://www.thr.mlit.go.jp/sakata/road/nitiendo/|title=日本海沿岸東北自動車道 県内各事業区間の状況|accessdate=2013年6月30日|publisher=国土交通省 東北地方整備局 酒田河川国道事務所}}</ref>，全線通車指日可待。

== 歷史 ==
* [[1994年|1994年]]（[[平成|平成]]6年）[[7月28日|7月28日]] : 北陸自動車道（當時）新潟中央系統交流道 - 新潟龜田交流道通車，同時與磐越自動車道連接。
* [[1997年|1997年]]（平成9年）[[10月30日|10月30日]] : 山形自動車道（當時）庄内朝日交流道 - 酒田交流道開通。
* 1997年（平成9年）[[11月13日|11月13日]] : 北陸自動車道（當時）新潟龜田交流道 - 新潟機場交流道開通。
* [[2001年|2001年]]（平成13年）[[7月7日|7月7日]] : 秋田機場交流道 - 河邊系統交流道開通。
* 2001年（平成13年）[[8月9日|8月9日]] : 山形自動車道（當時）酒田交流道 - 酒田湊交流道開通。
* [[2002年|2002年]]（平成14年）[[5月26日|5月26日]] : 新潟機場交流道 - 聖籠新發田交流道通車，同時間新潟中央系統交流道 - 新潟機場交流道從北陸自動車道改稱日本海東北自動車道。
* 2002年（平成14年）[[10月20日|10月20日]] : 聖籠新發田交流道 - 中條交流道通車。
* 2002年（平成14年）[[10月26日|10月26日]] : 岩城交流道 - 秋田機場交流道通車。
* [[2005年|2005年]]（平成17年）[[10月1日|10月1日]] : [[日本道路公團|日本道路公團]]民營化，改由[[東日本高速公路|東日本高速道路]]株式會社管理。
* [[2006年|2006年]]（平成18年）[[2月7日|2月7日]] : 荒川交流道（現為荒川胎内交流道）- 朝日交流道成為新直轄區間。
* [[2007年|2007年]]（平成19年）[[3月30日|3月30日]] : 廢除岩城收費站，改設秋田機場主線收費站。
* 2007年（平成19年）[[4月1日|4月1日]] : 豐榮智慧型交流道啟用。
* 2007年（平成19年）[[9月17日|9月17日]] : 兩前寺臨時出入口 - 岩城交流道通車，大内系統交流道與岩谷道路連接。
* [[2009年|2009年]]（平成21年）[[3月14日|3月14日]] : 救護車退出路（併設加治川紫雲寺巴士站）啟用。
* 2009年（平成21年）[[6月30日|6月30日]] : 酒田中央系統交流道、鶴岡西交流道、三瀨交流道、五十川交流道、朝日第二交流道、村上第二交流道、神林交流道等各交流道追加設置得到國土交通大臣的許可<ref name="mlit20090630">[http://www.mlit.go.jp/report/press/road01_hh_000081.html 高速自動車國道へのインターチェンジの追加設置について] - 國土交通省道路局 2009年6月30日</ref>。
* 2009年（平成21年）[[7月18日|7月18日]] : 中條交流道 - 荒川胎内交流道通車。
* [[2010年|2010年]]（平成22年）[[2月2日|2月2日]] : 發表以收費路段（新潟中央系統交流道 - 荒川胎内交流道、岩城交流道 - 河邊系統交流道）為對象的高速公路免費化社會實驗方案<ref>{{PDFlink|[http://www.mlit.go.jp/common/000057575.pdf 平成22年度高速公路免費化社會實驗計畫（案）]}}PDF - 國土交通省 2010年2月2日</ref>。
* 2010年（平成22年）[[2月15日|2月15日]] : 隨著荒川胎内交流道 - 神林岩船港交流道路段在2010年[[3月28日|3月28日]]通車，暫稱之朝日交流道、朝日第二交流道、村上第二交流道、村上交流道、神林交流道等各交流道以朝日真秀場交流道、朝日三面交流道、村上山邊里交流道、村上瀨波溫泉交流道、神林岩船港交流道作為其正式名稱並且一併發表<ref>{{PDFlink|[http://www.hrr.mlit.go.jp/niikoku/kisha/pdf/2010021502kaituu.pdf 日本海東北自動車道 荒川胎内交流道〜神林岩船港交流道 平成22年3月28日通車]}}PDF - 國土交通省北陸地方整備局 2010年2月15日</ref>。
* 2010年（平成22年）3月28日 : 荒川胎内交流道 - 神林岩船港交流道通車。
* 2010年（平成22年）4月1日 : 荒川胎内交流道 - 神林岩船港交流道間的管理單位從新潟國道事務所變更為羽越河川國道事務所<ref>{{PDFlink|[http://www.hrr.mlit.go.jp/niikoku/kisha/pdf/2010032901kanri.pdf 自4月1日起，日本海沿岸東北自動車道由羽越河川國道事務所管理]}}PDF - 國土交通省北陸地方整備局 2010年3月29日</ref>。
* 2010年（平成22年）[[6月28日|6月28日]] : 以收費路段（新潟中央系統交流道 - 荒川胎内交流道、岩城交流道 - 河邊系統交流道）為對象的高速公路免費化社會實驗開始<ref>{{PDFlink|[http://www.mlit.go.jp/common/000116387.pdf 平成22年度 高速公路免費畫社會實驗]}}PDF - 國土交通省 2010年6月15日</ref>。
* [[2011年|2011年]]（平成23年）[[3月27日|3月27日]] : 神林岩船港交流道 - 朝日真秀場交流道間通車。
* 2011年（平成23年）[[6月19日|6月19日]]：政府為了確保在2011年[[3月11日|3月11]][[東日本大震災|東日本大震災]]的復興費用，結束收費路段（新潟中央系統交流道 - 荒川胎内交流道、岩城交流道 - 河邊系統交流道）正在實行的免費化社會實驗，並暫時凍結以後的社會實驗。隨著此事，上述路段再度收費。
* 2011年（平成23年）[[6月20日|6月20日]] : 以東日本大震災的受災者支援、復興或復舊支援為目的，部分車輛可免費通行收費路段（新潟中央系統交流道 - 荒川胎内交流道、岩城交流道 - 河邊系統交流道）的措施開始實施。
* 2011年（平成23年）[[8月23日|8月23日]] : 國土交通省預計在9月發表朝日真秀場 - 溫海（現溫海溫泉）間、遊佐 - 象潟間兩區間整備案之檢討<ref>[http://www.47news.jp/CN/201108/CN2011082301000920.html 日本海沿岸道、早期に全線完成へ 9月に整備案検討開始] - 47NEWS 2011年8月23日</ref>。
* [[2012年|2012年]]（平成24年）[[3月24日|3月24日]] : 溫海溫泉泉交流道 - 鶴岡系統交流道通車並連接山形自動車道。同時間鶴岡系統交流道 - 酒田湊交流道間從山形自動車道改稱日本海東北自動車道。
* 2012年（平成24年）[[3月31日|3月31日]] : 結束以東日本大震災受災者支援、復興與復舊支援為目的的免費措施<ref>[http://www.e-nexco.co.jp/road_info/important_info/h23/0617/ NEXCO東日本 お客さまへの大切なお知らせ 東日本大震災に伴う東北地方の高速道路の無料措置通行方法について] {{webarchive|url=https://web.archive.org/web/20111126230753/http://www.e-nexco.co.jp/road_info/important_info/h23/0617/ |date=2011-11-26 }}
</ref>。
