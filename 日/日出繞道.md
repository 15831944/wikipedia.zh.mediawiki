'''日出繞道'''（{{jpn|j=日出バイパス|hg=ひじバイパス|rm=Hiji baipasu|f=y}}）為[[東九州自動車道|東九州自動車道]]連接[[速見交流道|速見交流道]]與[[日出交流道|日出交流道]]，總長9.0公里的[[快速道路|快速道路]]，路線名為[[國道10號(日本)|國道10號]]。全線由[[西日本高速公路|西日本高速公路]]以一般收費道路型式管理，同時為全國路線網的一環。在日出交流道與連接的大分機場道路共同構成地域高規格道路大分機場道路。

高速公路路線編號與[[大分機場道路|大分機場道路]]共同被編為 {{Ja Exp Route Sign|E97}} <ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/list/index.html|title=高速道路ナンバリング一覧|accessdate=2017年2月26日|publisher=国土交通省}}</ref>。

== 概要 ==
* 起點：大分縣速見郡日出町南畑字札之辻4927番地之267（[[速見交流道|速見交流道]]）
* 終點：大分縣速見郡日出町藤原字里2353番地之10（[[日出交流道|日出交流道]]）
* 全長：9.0km
* 規格：第1種第3級
* 設計速度：80km/h
* 道路收費期限：2050年（[[平成|平成]]62年）8月15日

== 歷史 ==
* 2002年3月30日：全線開通。
* 2010年2月2日：被指定為高速公路免費化社會實驗指定公路之一。

== 路線狀況 ==
=== 通行費 ===
2009年7月起的收費。
* 小型汽車等：250日圓
* 普通車：300日圓
* 中型車：350日圓
* 大型車：500日圓
* 特大車：850日圓
=== 利用日出繞道時的通行費徵收方法 ===
[[日出交流道|日出交流道]]未設置[[收費站|收費站]]。
; 經日出繞道利用東九州自動車道
: 於速見交流道內入手通行券（上載「速見本線」），抵達目的地離開高速公路時合併徵收日出繞道的通行費。
; 經東九州自動車道利用日出繞道
: 從出發地道速見交流道為止的通行費與日出繞道的通行費合併徵收。
; 僅利用日出繞道
: 上下線皆於速見交流道徵收通行費。
