日本的[[平面媒體|平面媒體]]和[[電子媒體|電子媒體]]相當蓬勃。廣播事業共分為[[公共廣播|公共廣播]]、[[商業廣播|商業廣播]]兩種，其中除了[[日本廣播協會|日本廣播協會]]（NHK）、[[放送大學學園|放送大學學園]]之外，所有的日本[[電台|電台]]及[[電視台|電視台]]都是[[商業媒體|商業媒體]]。[[日語|日語]]稱商業廣播為「民間放送」（{{lang|ja|みんかんほうそう}}；「放送」即[[中文|中文]]「廣播」之意），簡稱「民放」。

== 電視 ==
日本是世界人均看電視時間最長的國家，電視對日本國民的生活有很大的影響。全日本有六家全國電視網：
# [[NHK|NHK]]（日本放送協会）- 屬於[[公共媒體|公共媒體]]；與其他電視臺不同，NHK擁有兩個[[無線電視|無線電視]]頻道。
# [[NNN|NNN]]·[[NNS|NNS]] - 以[[日本電視台|日本電視台]]為中心的電視網，立場較保守；[[讀賣新聞|讀賣新聞]]的關係企業。
# [[JNN|JNN]] - 以[[東京廣播公司|TBS]]（東京放送）為中心的電視網，為日本第一個[[電視聯播網|電視聯播網]]；[[每日新聞|每日新聞]]的關聯播放者。
# [[FNN|FNN]]·[[FNS|FNS]] - 以[[富士電視台|富士電視台]]為中心的電視網；[[富士產經集團|富士產經集團]]的核心企業。 
# [[ANN|ANN]] - 以[[朝日電視台|朝日電視台]]為中心的電視網；[[朝日新聞|朝日新聞]]的關係企業。
# [[TXN|TXN]] - 以[[東京電視台|東京電視台]]為中心的電視網；[[日本經濟新聞|日本經濟新聞]]的關係企業。


在日本，[[有線電視|有線電視]]相對弱勢，很多觀眾滿足于六大無線電視網，不會再花錢收看有線電視。此外，[[衛星電視|衛星電視]]在日本的發展較大，並擁有一定規模的收視戶。多數國家有線電視裝機率遠高於衛星電視，但日本則二者比率差距不大。但日本的衛星電視分成兩種，一種是地上數位電視型，透過地面的衛星接收站傳送地面衛星訊號到日本各家戶中，此種接收成本較小，便能接收當地的電視節目。另外一種是需要安裝衛星接收碟的衛星電視，俗稱BS，高畫質影像傳輸，接收成本較高。

== 報紙 ==
日本是世界報紙大國，全世界發行量最高的兩份報紙讀賣新聞和朝日新聞都是日本報紙。報紙在人們生活中有很高的地位，根據一份民意調查，日本人最信任的事物即為報紙。日本有五家全國性綜合報紙，大部分從19世紀日本近代化後就問世，是日本現代大眾傳播事業的骨幹，日本現今主要的電影公司、民營電台與電視台、和廣告公司都找得到其與這幾家大報的淵源。

# [[讀賣新聞|讀賣新聞]]：中間偏右，發行量位居世界第一。與日本電視台是關聯企業。與廣告公司[[博報堂|博報堂]]有關。 
# [[朝日新聞|朝日新聞]]：左派報紙，發行量居于第二位。很多文學作品在該報刊載。與朝日電視台是關聯企業。與[[東映|東映]]映畫有關。
# [[每日新聞|每日新聞]]：左派報紙，發行量第三，東京放送是其關聯播放電視台。與[[松竹|松竹]]映畫和廣告公司[[電通|電通]]有關。
# [[日本經濟新聞|日本經濟新聞]]：經濟專業報紙，與東京電視台是關聯企業。 
# [[產經新聞|產經新聞]]：發行量位居全日本第七。保守與[[日本民族主義|民族主義]]立場較強，與富士電視台是關聯企業。與[[東寶|東寶]]映畫有關。

除了以上的六家全國性報紙外，還有[[東京新聞|東京新聞]]、[[中日新聞|中日新聞]]、[[西日本新聞|西日本新聞]]、[[北海道新聞|北海道新聞]]和[[河北新報|河北新報]]等地方性報紙。體育報紙在日本也有很高的地位，主要的體育報紙有[[東京體育|東京體育]]、[[體育報知|體育報知]]等。
