'''日本人'''（{{jpn|j=日本人|hg=にほんじん／にっぽんじん|rm=Nihonjin / Nipponjin|f=h}}）指擁有[[日本|日本]][[國籍|國籍]]者或[[日本列島|日本列島]]的各民族或族群。

== 民族 ==
{{main|日本民族问题|日本民族主義}}
[[File:Japan_glaciation.gif|thumb]]
日本多數民族为[[大和民族|大和民族]]，另外还有[[阿伊努族|阿伊努]]、[[琉球族|琉球]]、[[朝鲜族|朝鲜族]]等[[少数民族|少数民族]]，阿伊努族人是北海道的[[土著民族|土著民族]]，琉球族則是在[[琉球國|琉球國]]被日本吞并後成為日本冲绳群岛的一个少数民族。如今，國籍上的日本人與民族上的日本人基本傾向於相同，但有時「日本人」一詞專指大和民族。

== 起源 ==
{{refimprovesect|time=2017-08-31}}
現發現最古老的日本人人骨，大約屬於2万年前至3万年前的[[冰河期|冰河期]]時代。一般認為[[繩文人|繩文人]]最早於冰河期時代到達日本列島。根據一份由2013年由日本的遺傳學家神澤秀明（Hideaki Kanzawa）的研究，繩文人與大洋洲的[[巴布亞人|巴布亞人]]與[[美拉尼西亞人|美拉尼西亞人]]在[[遺傳因子|遺傳因子]]上最為接近。而且，繩文人和大洋洲的諸民族都有與[[丹尼索瓦人|丹尼索瓦人]]混血的相似性，這也間接的証實了[[繩文人|繩文人]]、大洋洲諸民族與[[澳洲人種|澳洲人種]]（如藏族人）間的共同關係。冰河期時代的[[北海_(大西洋)|北海]]被冰所覆蓋，於庫頁島、北海道、日本列島間進行移動明顯是可行的。另外，由渡過南海而來的[[南島語族|南島語族]]与[[越人|越人]]、[[西南夷|西南夷]]。

日本人與韓國人的Y染色體DNA單倍型類群，有个别基因类似，如高比例的單倍型類群O-M176及一定比例的單倍型類群C-M130。而單倍型類群O-M176在中國大陸的南北方都是不常見的Y染色體DNA單倍型類群。关西一帯的日本人由於鄰近朝鮮半島，而根据新的遗传路径:这类基因是日本人移民到朝鲜半岛时候留在朝鲜半岛的，[[弥生人|弥生人]]在[[遺傳因子|遺傳因子]]上的影響較大。关東的日本人，雖然弥生人在[[遺傳因子|遺傳因子]]上的影響仍舊較大，但[[繩文人|繩文人]]也對[[遺傳因子|遺傳因子]]有顯著的影響弥生人與繩文人不断反复混血形成日本人。最後，日本歷史上的繩文人如蝦夷族漸漸融合成大和民族的一分子。少數蝦夷族只能退居東日本的邊陲地道與北海道（[[隼人|隼人]]則殘留在[[南九州|南九州]]一帶）。

日本人的體質與語言是在公元前400年形成，但真正定型是在十二世纪[[蝦夷人|蝦夷人]]投降后為止。早期的弥生人與朝鮮半島的移民有一定的血緣關係與緊密的連繫，如日本的[[倭國|倭國]]與朝鮮半島的百濟王室的聯姻關係。日本在古墳時代的大王的[[古墳|古墳]]，有著被稱為[[前方後圓墳|前方後圓墳]]的特殊建築型式。而朝鮮半島南方也有為數不少的類似的[[前方後圓墳|前方後圓墳]]（見[[朝鮮半島的日本式古墳|朝鮮半島的日本式古墳]]）。

自從1996年開始中日兩國的[[考古學家|考古學家]]、[[人類學家|人類學家]]和醫學專家聯合組成的中日人骨共同調查團隊所做的考古研究發現，在數十萬年前來自[[中國東北|中國東北]]的原始人類遷入朝鮮半島居住，一些又遷徙至日本。考古學和人類學觀點認為日本民族是主要由古代中國[[漢族|漢族]]、長江下游的[[吳越人|吳越人]]、中國南方沿海人和少量西伯利亞[[通古斯人|通古斯人]]、少量[[南太平洋|南太平洋]]群島的[[馬來人|馬來人]]以及[[中南半島|中南半島]]的印支人融合而來，逐漸遷移到日本融合衍變而來的<ref name="第三只眼">{{Cite book |title=第三只眼看日本 |author=元坤 |date=2010年12月 |publisher=中國廣播電視出版社 |year=2010年12月 |isbn=978-7-5043-6298-8 |editor=王佳 |location=中国北京|page=p.6}}</ref>。

在中國[[西周|西周]]滅[[商朝|商]]之後，商朝遺臣[[箕子|箕子]]到朝鮮半島與當地土著建立了「箕氏侯國」。[[戰國|戰國]]時[[秦國|秦國]]滅[[燕國|燕國]]，[[齊國|齊國]]和[[越國|越國]]時，大量燕國人、齊國人和越國人逃到朝鮮半島（现在的韩国和朝鲜）。前3世紀至15公紀間，大量中國人移居朝鮮半島，這以後直到[[清朝|清朝]]時，每當中國北方出現戰亂和災害，也陸續有中國人向朝鮮半島遷徙，後來又從那裡向東、向南到了日本，或從[[中國東部|中國東部]]直接跨海到日本<ref name="第三只眼"/>。

现代日本人是在幕府被推翻后才真正出现（之前日本人只認自己是藩的成员）。在一項顱骨測量研究中，婆羅洲，越南，蘇祿，爪哇和蘇拉威西島按此順序靠近日本人。

== 两大父系DNA ==
[[File:Migraciones_humanas_en_haplogrupos_de_ADN-Y.PNG|thumb]]
