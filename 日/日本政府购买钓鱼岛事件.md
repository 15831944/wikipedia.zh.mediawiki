'''日本政府购买钓鱼岛事件'''（{{lang-ja|'''尖閣諸島国有化'''}}），源于[[日本|日本]][[东京都知事|东京都知事]][[石原慎太郎|石原慎太郎]]的施政。石原慎太郎为了获得[[東京都廳|东京都政府]]对[[钓鱼岛|钓鱼岛]]的直接性管辖和管理，发起日本国内民众捐款从私人手中购买钓鱼岛活动。<ref>{{cite web |url=http://news.qq.com/a/20120515/000320.htm |title=东京购钓鱼岛捐款破5亿 或已达到购买价位下限 |accessdate=2012年5月15日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>最终，2012年9月10日，[[日本政府|日本政府]]正式以20.5亿日元（人民币1.66亿元，[[新臺幣|新臺幣]]7億元）从[[栗原弘行|栗原弘行]]手中收购钓鱼岛及其附属岛屿[[南小岛|南小岛]]、[[北小岛|北小岛]]，在同年9月11日付款和登记。

== 事件背景 ==
{{main|釣魚台列嶼主權問題}}
[[大日本帝国|大日本帝国]]吞并[[琉球国|琉球国]]时期，声称[[福冈县|福冈县]][[八女市|八女市]]出身的实业家[[古贺辰四郎|古贺辰四郎]]为钓鱼岛的开拓者和岛主。<ref name="cob " >{{cite web |url=http://news.qq.com/a/20120417/001303.htm |title= 石原提收购钓鱼岛设想 日本政府持静观审视态度|accessdate=2012年4月17日 |publisher=腾讯网 |work= |language =zh-cn}}</ref><ref name="duck " >{{cite web |url= http://news.sohu.com/20120418/n340953733.shtml|title=石原：中国反对日购买钓鱼岛 几乎等同对日宣战 |accessdate=2012年4月18日 |publisher= 搜狐网|work= |language =zh-cn}}</ref>

1885年10月，大日本帝国外务大臣在公文中写道，“近时清朝报纸揭载我政府欲占据台湾附近的清朝属岛的传闻，对我国抱以疑忌，并促清朝政府予以注意。我政府若于此时遽尔公然建立国标，反易招致清朝的疑忌”。<ref name=" fox" >{{cite web |url=http://news.qq.com/a/20120521/000558.htm |title= 国立政治大学邵汉仪投书《华尔街日报》日本明治政府承认清朝拥有钓鱼岛|accessdate= 2012年5月18日|publisher= 腾讯网|work= |language =zh-cn}}</ref>

1892年1月，大日本帝国[[冲绳知事|冲绳知事]]在公文书中写道：“再次踏查各岛之时机尚未到来”。<ref name=" fox" />

1894年5月，大日本帝国内务省公文书中写道：“自1885年由[[琉球县|琉球县]]属警部派出之调查以来，期间未再进行实地调查”。<ref name=" fox" />

1895年，古贺辰四郎从大日本帝国政府租借钓鱼岛30年，并说是“皇国大捷之结果”。<ref name="cob " /><ref name="duck " /><ref name=" fox" />

1932年，古贺辰四郎的儿子[[古贺善次|古贺善次]]用1万5000日元（今2500万元日元）从大日本帝国政府购买除[[大正岛|大正岛]](中國稱為「赤尾嶼」)之外的各岛屿。<ref name="cob " /><ref name="duck " />

1970年后，古贺家以4600万日元的价格出售钓鱼岛的所有权给[[埼玉县|埼玉县]]的一位好友[[栗原弘行|栗原弘行]]。<ref name="cob " /><ref name="duck " />
