'''日向'''（'''{{lang|ja|ひゅうが}}'''，當時由於使用舊假名遣，所以艦名的表示為'''{{lang|ja|ひうが}}'''）是[[大日本帝国海軍|大日本帝国海軍]]的[[戰艦|戰艦]]，[[伊勢型戰艦|伊勢型戰艦]]2號艦。於[[太平洋戰爭|太平洋戰爭]]中期，改造成[[航空戰艦|航空戰艦]]，「航空戰艦」只是因方便上的稱呼，於正式的艦籍中仍為戰艦。艦名的來源為日本[[宮崎縣|宮崎縣]]的[[令制國|舊國名]]，因此艦內神社所供奉的是來自[[宮崎神宮|宮崎神宮]]的分神<ref>[[#海軍生活|#海軍生活]]525頁</ref>。該艦名在帝國海軍只有1代，於戰後由[[海上自衛隊|海上自衛隊]]的[[護衛艦_(日本)|護衛艦]]｢[[日向號護衛艦|日向]]（{{lang|ja|ひゅうが}}）｣所繼承。

== 艦歷 ==
「日向」為[[三菱重工業長崎造船所|三菱合資會社三菱造船所]]（現・三菱重工長崎造船所）<!-- 當時還沒有三菱重工業-->建造，於1915年（大正4年）5月6日以「'''第六號戰艦'''」為名動工<ref>「軍艦日向製造1件」、[[#歴群26伊勢型|#歴群26伊勢型]]98頁</ref>，1917年（大正6年）1月27日，由[[東伏見宮依仁親王|東伏見宮依仁親王]]親臨舉行「進水式」（下水典禮）<ref>「進水式」p.45、[[#歴群26伊勢型|#歴群26伊勢型]]98頁</ref>，1918年（大正7年）4月30日，「'''軍艦 日向'''」正式服役<ref>[[#歴群26伊勢型|#歴群26伊勢型]]101頁</ref>。

原設計為[[扶桑型戰艦|扶桑型戰艦]]的4號艦，但由於財政問題令預定完工日期大幅延遲，所以有機會將扶桑型的缺點改善，主砲位置亦有所變更。可是在完成的時候，搭載38厘米砲的[[伊麗莎白女王級戰艦|伊麗莎白女王級戰艦]]已在英國服役。歷代艦長之中，包括了後來成為[[聯合艦隊司令長官|聯合艦隊司令長官]]及指揮[[捷一號作戰|捷一號作戰]]等的[[豐田副武|豐田副武]]，也有負責航空戰艦改裝案進程，後來成為[[第四航空戰隊|第四航空戰隊]]司令官，並與「日向」有很深關係的[[松田千秋|松田千秋]]，此外亦有[[宇垣纏|宇垣纏]]、[[西村祥治|西村祥治]]等不少戰史上的重要人物。

== 兩次砲塔爆炸 ==
竣工後，配屬於[[第一艦隊_(日本海軍)|第一艦隊]]第一戰隊，並為最新銳的戰艦，在「[[長門號戰艦|長門]]」竣工之前為[[聯合艦隊|聯合艦隊]][[旗艦|旗艦]]。1940年（昭和15年）6月22日至7月10日期間，作為[[滿州國|滿州國]]康德帝（[[愛新覺羅溥儀|愛新覺羅溥儀]]）訪日時的座乘艦<ref>[[#海軍生活|#海軍生活]]541-545頁</ref>。[[太平洋戰爭|太平洋戰爭]]開戰時，「伊勢」所屬於第一艦隊第二戰隊，並為戰隊旗艦。

另一方面，「日向」經常發生嚴重的事故。在服役之後的1919年（大正8年）10月24日，於[[房總|房總]]近海演習期間，第3砲塔發生爆炸事故<ref>「軍艦日向3番砲塔爆発の件(1)」p.7</ref>。在1924年（大正13年）9月17日於第四砲塔彈藥庫發生火災<ref>「軍艦日向四砲塔弾庫火災1件」p.4</ref>。另於1942年（昭和17年）5月5日，在[[伊予灘|伊予灘]]演習中，在進行第七齊射期間，於第5砲塔中再次發生爆炸事故<ref>[[#海軍生活|#海軍生活]]533頁</ref>。這個在砲塔內爆炸的事故，在爆炸時的一瞬間被拍攝下來，該影片仍為現存<ref group="注">該影片於「日向」主砲射擊演習期間，在其後樓並面向第五・六號砲塔的方位攝影，在進行編輯及[[日本的審查制度|審查]]期間並不發現出了問題，還在[[新聞電影|新聞電影]]中公開（[[日本新聞|日本新聞]]第112號・1942年（昭和17年）7月29日公開[http://www.youtube.com/watch?v=OWSUpQkzMps]）後成為趣聞。</ref>。眼之所見，其程度只是發砲後的砲煙稍為奇怪，在外觀上亦沒有明顯的損傷。在艦橋上的艦長及砲術科，只是感覺到5號砲塔所發射的主砲彈落在對出50米的海面很不協調外，並沒有感到有任何異樣<ref>[[#海軍生活|#海軍生活]]534頁</ref>。之後，收到主砲發令所報告5號砲塔火發生火災，並立即在火藥庫注水避免爆沈，該意外中共有55名死者，8名重傷者<ref>[[#海軍生活|#海軍生活]]536-537頁</ref>。而事發的原因，是主砲在裝填後，尾栓在沒有完全緊閉下，電流就在火管<!--並不是雷管-->流動並為裝藥點火，因此砲彈並沒有向前射擊，反而將砲管內的壓力成為逆流衝進砲塔內<ref name="海軍生活538">[[#海軍生活|#海軍生活]]538頁</ref>。由火管內突然有電流流出，被說成5號砲塔特有的「怪癖」<ref name="海軍生活538"/>。

在出現眾多死傷者後，「日向」返回吳，並將受損的第五砲塔撤去，並在該處增設4座[[九六式二十五毫米高角機槍|25毫米3聯裝機槍]]，亦另外搭載當時新開發的[[假稱二號電波探信儀二型|假稱二號電波探信儀二型]]（對水上22號電探、即對水面雷達）以作試驗，在[[中途島海戰|中途島海戰]]的一環中前往阿留申群島。在這戰鬥中[[第1航空艦隊_(日本海軍)|南雲機動部隊]]在喪失4艘主力航空母艦後被毀滅，而主力部隊在沒有與敵方戰鬥的情況下回到日本。「日向」的雷達在遇到惡劣天氣的歸途上一直為艦隊保持正確航路，自此[[松田千秋|松田千秋]]艦長不斷向周圍的人講述雷達如何有效<ref>[[#歴群26伊勢型|#歴群26伊勢型]]162頁</ref>。

在發生兩次砲塔爆炸，一次彈藥庫火警等極危險的事故下仍然相安無事，對比起戰艦「[[河內號戰艦|河內]]」與「[[陸奥號戰艦|陸奥]]」等爆沈的例子來說，可說是非常幸運<ref group="注">「[[三笠號戰艦|三笠]]」也曾在1906年遭遇過爆沈事故，最後幸好能成功[[打撈|打撈]]並修復。而美國的[[愛荷華級戰艦|愛荷華級戰艦]]「[[愛荷華號戰艦_(BB-61)|愛荷華]]」號，於1989年曾發生二號砲塔爆炸的事故，並且令多人死亡。</ref>。再者，於[[中途島海戰|中途島海戰]]損失4艘航空母艦及第五砲塔因事故而拆除，此後令「日向」及同型艦「伊勢」的命運發生很大轉變。（詳細參考[[伊勢型戰艦|伊勢型戰艦]]）

== 航空戰艦「日向」 ==
為填補航空母艦的戰力，日本海軍決定將[[扶桑型戰艦|扶桑型戰艦]]及[[伊勢型戰艦|伊勢型戰艦]]改裝成航空母艦。但由於時間不配合，扶桑型沒有實施任何改裝，而伊勢型也沒有進行全面的改造，只是將後部的5號及6號主砲撤除，並新設飛機庫及[[飛行甲板|飛行甲板]]，成為[[航空戰艦|航空戰艦]]<ref>[[#歴群26伊勢型|#歴群26伊勢型]]163頁</ref>。為了減輕重量，艦上所有的副砲[[50口徑三年式14厘米砲|50口徑三年式14厘米砲]]全部被撤除。這些副砲全部被轉到陸上當砲台使用，並為此在[[吳鎮守府|吳鎮守府]]編成第六特別陸戰隊重砲隊<ref>[[#海軍生活|#海軍生活]]560頁</ref>。但是，新安裝的飛行甲板的長度還不足正規航空母艦的一半，因此並不能給艦載機進行著艦動作。不過，飛行甲板上的空間還是能夠進行航空機整備及起飛作業。而主砲撤去後其彈藥庫的空間，被改成航空機用燃料庫及武器庫<ref>[[#歴群26伊勢型|#歴群26伊勢型]]146頁</ref>。「日向」所帶領的[[第四航空戰隊|第四航空戰隊]]預定配備所屬於[[第六三四海軍航空隊|第六三四海軍航空隊]]的
[[水上偵察機|水上偵察機]]「[[瑞雲水上偵察機|瑞雲]]」及[[艦載轟炸機|艦載轟炸機]]「[[彗星俯衝轟炸機|彗星二二型]]」等的主力機及部隊，當中有14架彗星及8架瑞雲將會編配到「日向」<ref name="旗艦行動5">[[#旗艦行動|#旗艦行動]]p.5</ref>。該艦隊的運用思想，是使用彈射器將彗星射出，其後降落到在攻擊後有體機損耗的航空母艦上，或陸上基地。亦有說法指出伊勢型航空戰艦將會與近乎同速度的航空母艦「[[信濃號航空母艦|信濃]]」（由大和型戰艦三號艦改造成的航空母艦）組成航空戰隊<ref>[[#歴群26伊勢型|#歴群26伊勢型]]159-160頁「伊勢型航空戦艦の運用思想」</ref>。

「日向」的改裝由1943年5月開始，11月18日完成。<ref>[[#歴群26伊勢型|#歴群26伊勢型]]164頁</ref>。其間，較先完成航空戰艦改裝的「伊勢」，在飛機庫內裝載[[大和型戰艦|大和型戰艦]]的46厘米砲彈，前往楚克錨地進行物資及彈藥輸送。「日向」在改造完成後與「伊勢」共同被編入第十一水雷戰隊，在日本內地毎日進行訓練。1944年5月1日[[第四航空戰隊|第四航空戰隊]]編成，「日向」為[[松田千秋|松田千秋]][[少將|少將]]所乘坐及第四航空戰隊的旗艦<ref name="旗艦行動5"/>。6月7日因需要強化機槍的火力而進入吳工廠內船塢，其間美軍向塞班島方面進攻，並與日本軍之間爆發[[馬里亞納海戰|馬里亞納海戰]]<ref name="旗艦行動5"/>。此時「日向」與「伊勢」的進行中的工程被中止，並為出擊作準備，但趕不及參與該海戰。<ref name="旗艦行動5"/>。

10月，美軍向菲律賓方面發動進攻。10月20日，「日向」參加[[雷伊泰灣海戰|捷一號作戰]]從日本出擊，預定搭載所屬的[[第六三四海軍航空隊|第634航空隊]]的飛機因[[台灣沖航空戰|台灣沖航空戰]]的關係而轉用到菲律賓方面，所以「日向」與「伊勢」雖然配屬航空戰隊但搭載機一架也沒有<ref>[[#旗艦行動|#旗艦行動]]p.8</ref>。小澤的前衛部隊由松田支隊所編成，10月24日因尋求與美軍艦隊砲戰的機會而南下，但並沒有遇上敵人，翌25日0700時與本隊再合流<ref name="旗艦行動9">[[#旗艦行動|#旗艦行動]]p.9</ref>。於25日的[[雷伊泰灣海戰#恩加尼奥角海战|恩加尼奧角海戰]]中，小澤本隊的4艘航空母艦（[[瑞鶴號航空母艦|瑞鶴]]、[[瑞鳳號航空母艦|瑞鳳]]、[[千歲號航空母艦|千歲]]、[[千代田號航空母艦|千代田]]）全部被擊沈。其後美軍軍機集中攻擊仍然健在的「日向」與「伊勢」，但由於兩艦長使用了松田少將所提出的航空攻擊回避術，因此在兩艦長的巧妙回避運動下，加上改裝成航空戰艦後大幅增強對空火力成功回避美軍的攻擊。最終在記錄中有6架美軍軍機被擊落，艦身並沒有重大損傷，但仍有1人戰死，8人負傷<ref>[[#日向捷号詳報|#日向捷号詳報]]p.41、[[#第4航戦詳報|#第4航戦詳報]]p.48</ref>。「日向」共發射了主砲三式彈112發、12.7厘米高射砲彈659發、25毫米機槍彈28970發及噴進砲彈250發<ref>[[#日向捷号詳報|#日向捷号詳報]]p.14</ref>。於10月29日，返回日本<ref name="旗艦行動9"/>。

1945年（昭和20年）2月，由於參加戰略物資輸送作戰「[[北號作戰|北號作戰]]」，彈射器被撤除，更有一部分機槍被拆走而提供給現地部隊，回程時使用飛機庫的空間滿載石油、橡膠及錫等運回戰略物資已經很稀少的日本。全艦隊在沒有任何損傷的狀態下返回日本，行動像奇跡般的成功，而「日向」、「伊勢」及輕巡洋艦「[[大淀號輕巡洋艦|大淀]]」所輸送物資總量，連一隻中型貨物船也超不過。

== 結局 ==
[[file:Backus,_This_was_Hyuga.jpg|thumb]]
[[file:Aerial_photo_of_battleship_Hyuga_after_being_sunk_at_Kure.jpeg|thumb]]
北號作戰完結後，「日向」一直停泊在[[吳港|吳軍港]]，在1945年3月1日被指定為第一預備艦，其後在3月19日在[[吳軍港空襲|吳軍港空襲]]中被3發炸彈（二號砲塔左舷後部、左舷鍋爐室、艦後部）擊中<ref name="歴群26伊勢170">[[#歴群26伊勢型|#歴群26伊勢型]]170頁</ref>。受到損傷的「日向」在4月20日被指定為第四預備艦，5月1日為改為特殊警備艦，並在吳港外（[[情島|情島]]近海）作浮砲台<ref name="歴群26伊勢170"/>。於7月24日的[[轟炸吳市_(1945年7月)|吳軍港空襲]]中受到美軍航空母艦艦載機以波狀攻擊，艦長[[草川淳|草川淳]]少將（追授[[中將|中將]]）[[戰死|戰死]]<ref name="歴群26伊勢171">[[#歴群26伊勢型|#歴群26伊勢型]]171頁</ref>。當攻擊發生時，約半數乘組員已經退艦，艦上留有約1000人，當中戰死204人，重輕傷者約600人<ref name="歴群26伊勢171"/>。7月26日，「日向」以水平狀態坐底<ref name="歴群26伊勢171"/>。<!--着底の直接原因は命中弾によるものではなく、多数の至近弾による{{要出典範囲|date=2011年7月|弾薬誘爆によるものとされる。}}-->

在戰後的1947年（昭和22年）7月，在完成解體後其艦歷正式完結。尚且，在情島坐底時的「日向」，於戰後被美軍拍攝為彩色影像，今日仍然可以觀察到當時嚴重的受損情況。現在「日向」曾裝備過的[[航海燈|航海燈]]及[[軍艦旗|軍艦旗]]於[[廣島縣|廣島縣]]吳市內的[[大和博物館|大和博物館]]所收藏。
