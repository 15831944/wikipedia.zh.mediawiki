'''內閣總理大臣'''（{{jpn|j=内閣総理大臣|hg=ないかくそうりだいじん|rm=Naikaku sōri daijin|kyu=內閣總理大臣|f=y}}）是[[日本|日本]]最高[[政府首脑|行政首長]]，主要職責為領導[[日本內閣|內閣]]的運作，主持{{link-ja|閣議 (日本)|閣議 (日本)|內閣會議}}，以及代表內閣向[[國會_(日本)|國會]]提出施政報告。其須符合文民及國會議員身分，經國會[[內閣總理大臣指名選舉|選出]]後再交由[[天皇|天皇]]任命。中文常稱該職為'''日本首相'''，在日本則簡稱為'''總理大臣'''（{{jp|j=総理大臣|hg=そうりだいじん|rm=Sōri daijin|help=yes}}）、'''[[總理|總理]]'''（{{jp|j=総理|hg=そうり|rm=Sōri|help=yes}}）、'''[[首相|首相]]'''（{{jp|j=首相|hg=しゅしょう|rm=Shushō|help=yes}}）等，尊稱「[[閣下|閣下]]」<ref>[http://www.kantei.go.jp/cn/ 日本国首相官邸网页]，2015-6-7</ref>。現任（第98任）為[[自由民主党_(日本)|自民黨]]籍的[[安倍晋三|安倍晋三]]，自2012年12月26日起就職。

== 概述 ==
===創設之初===
日本設置內閣總理大臣之職位，是在1885年隨著[[日本內閣|內閣]]制度的創立而開始的，以取代過往在[[律令制|律令制]]中執掌國家行政權的[[太政大臣|太政大臣]]（[[三公|三公]]之首），比日本第一部現代憲法《[[大日本帝國憲法|大日本帝國憲法]]》（明治憲法）施行時間1890年11月29日還早，第一任為1885年12月22日就任的[[伊藤博文|伊藤博文]]。

事實上，《大日本帝國憲法》並沒有明文設置「內閣總理大臣」這個官職，僅設置「[[國務大臣_(日本)|國務大臣]]」，即內閣閣員；內閣總理大臣職務的法源依據，則是來自先後於1885年及1889年頒布的《{{link-ja|內閣職權|内閣職権}}》與《{{link-ja|內閣官制|内閣官制}}》，後者便規定內閣總理大臣為「各（國務）大臣之首領」，主要功能為統合整個內閣，但也僅有「{{link-en|同輩者之首|Primus inter pares}}」的地位，在法律上對於其他國務大臣沒有任免權。如果內閣成員間發生意見分歧導致國政無法推行，只能走向{{link-ja|內閣總辭|内閣総辞職|總辭}}一途。由於當時的法律同時規定{{link-ja|軍部|軍部}}大臣（[[陸軍大臣|陸軍]]與[[海軍大臣_(日本)|海軍]]大臣）必須由[[軍部大臣現役武官制|現役軍人擔任]]，導致軍部可以此干預內閣總理大臣、乃至於整個內閣的人事權，這種現象在[[昭和|昭和]]初期[[日本法西斯主義|日本法西斯主義]]抬頭後尤其明顯。

===明治憲法情形===
在《大日本帝國憲法》的體制下，內閣總理大臣是由[[天皇|天皇]]親自任命的（此方式稱為「{{link-ja|大命降下|大命降下}}」），但因不是憲法規定的職位，其任命方式曾因時局變遷而多次改變。最初是由[[元老_(日本)|元老]]向天皇薦舉，因而常由推翻[[江戶幕府|江戶幕府]]的[[明治维新|明治维新]]有功人員出任；直到[[大正民主|大正民主]]時期，在當時[[自由民主制|自由民主]]浪潮的氛圍下，則多由[[日本眾議院|眾議院]]多數黨的[[黨魁|黨魁]]出任，此慣例被稱為「{{link-ja|憲政之常道|憲政の常道}}」；進入昭和時代後，隨著日本逐漸進入[[大東亞戰爭|戰時]]體制、以及最後一位[[元老|元老]][[西園寺公望|西園寺公望]]的逝世，改為由[[內大臣府|內大臣]]召集的「{{link-ja|重臣會議|重臣会議}}」舉薦，又轉變成現役[[軍人|軍人]]或軍部相關人士出任的局面。

根據《大日本帝國憲法》，政治實權掌握在[[天皇|天皇]]手上，而天皇又是[[天照大神|祖神]]的象徵，因此內閣總理大臣變成「有責無權」的情形，在[[大日本帝國|大日本帝國]]時代的內閣任期反而並不穩定。

===現行憲法情形===
[[第二次世界大戰|第二次世界大戰]]後，《大日本帝國憲法》被現行的《[[日本國憲法|日本國憲法]]》取代，內閣總理大臣的職務才連同整個內閣組織列入憲法之中。在以[[議會制君主立憲制|議會制君主立憲制]]為基礎的新憲法中，內閣總理大臣被賦予許多舊憲法時代沒有的政治權力，例如可自行任免國務大臣，而不需要天皇或[[國會_(日本)|國會]]的同意，使其握有實權、而真正成為國家的最高[[政府首腦|行政首長]]。首位依據《日本國憲法》就任的內閣總理大臣，為1947年5月24日上任的時任[[日本社會黨|日本社會黨]]委員長[[片山哲|片山哲]]。

== 職位變更形式 ==
{{expand-section}}

===選出與就職===
根據《日本國憲法》規定，內閣總理大臣由國會議員在[[日本參議院|參]][[日本眾議院|眾]]兩院議員中提名人選，並召開聯席會議以進行表決（正式名稱為「[[內閣總理大臣指名選舉|內閣總理大臣指名選舉]]」，通稱為「首班指名」），後再由天皇進行形式上的任命。為了避免戰前[[軍政府|軍人干政]]的狀況重演，根据《日本国宪法》第66条第2项，內閣總理大臣必須由「文民」擔任；此處對「文民」沒有明確定義，通常被認為是指未曾在舊[[日軍|日軍]]服役、或是不具備現役軍人（現指在[[自衛隊|自衛隊]]{{link-ja|自衛官|自衛官|服役}}）身分者<ref>{{lang|ja|1973年（昭和48年）12月19日（72回国会）の衆議院建設委員会において、大村襄治政府委員（内閣官房副長官）は「政府といたしましては、憲法第六十六条第二項の文民につきましては、「旧陸海軍の職業軍人の経歴を有する者であって、軍国主義的思想に深く染まっていると考えられるもの」、それから「自衛官の職に在る者」、この二つを判断の基準にいたしているわけでございます。」と答弁している。}}</ref>。

因为内阁总理大臣[[间接选举|非直选]]产生，而是由國會從其議員中選出，因而多由國會多數黨、或是[[聯合政府|聯合政府]]成員政黨的黨魁出任。特別的是，《日本國憲法》施行以來的內閣總理大臣均由眾議院議員出任，這已形成一種憲政慣例。《日本國憲法》虽然有规定内阁总理大臣的产生程序，却没明确规定其任期，但是各党的党章均對党魁任期有所规定，因而等同間接規範了內閣總理大臣的任期；例如從創黨以來長期維持[[一党独大|一党独大]]局面的[[自由民主党_(日本)|自民党]]，就规定[[自由民主党总裁|党总裁]]3年一任，原则上只能连任两届。而國會議員的任期限制，也間接規範了內閣總理大臣擔任時間。

===卸任與解職===

== 權責 ==
根据《日本国宪法》以及其他法令，总理大臣的权利和义务规定如下：
*组织[[内阁|内阁]]，任命并任意罢免[[國務大臣_(日本)|国务大臣]]（《日本国宪法》第68條）
*允许对在任国务大臣[[起诉|起诉]]（《日本国宪法》第75條）
*代表内阁向[[日本國會|國會]]提出议案（《日本国宪法》第72條）
*代表内阁向國會报告一般国内事务与[[外交|外交]]关系（《日本国宪法》第72條）
*代表内阁指挥监督各个行政部门（《日本国宪法》第72條）
*与专职国务大臣共同签署法律及政令（《日本国宪法》第74條）
*主持内阁会议（《内阁法》第4條第2项）
*指定内阁总理大臣及专职国务大臣的代理人（《内阁法》第9条、10條）
*擔任{{link-ja|皇室會議|皇室会議}}議長（《[[皇室典範|皇室典範]]》第29条、33条）
*發佈[[緊急命令|緊急命令]]（《警察法》第71條）
*執掌[[自衛隊|自衛隊]]的[[總司令|最高指揮監督權]]（《自衛隊法》第7條）
*在{{link-ja|間接侵略|間接侵略}}或其他[[緊急狀態|緊急狀態]]，一般警察不可能維持治安時，[[治安出動|命令自衛隊出動維持治安]]（《自衛隊法》第78條）
*{{link-ja|選舉管理委員會 (日本)|選挙管理委員会|中央選舉管理會}}委員之任命及罷免（《公職選舉法》第5條之2）
*[[日本放送協會|日本放送協會]]經營委員會委員之任命（《放送法》第16條）
