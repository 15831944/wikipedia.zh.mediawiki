'''云栖小镇'''地处[[杭州|杭州]][[西湖区|西湖区]]西南，位于全国12个国家级旅游度假区之一的[[杭州之江国家旅游度假区|杭州之江国家旅游度假区]]核心区块，辖区面积3.5平方公里，地域涉及原凌家桥、石龙山、良户等村社，是浙江省特色小镇的发源地，2014年10月，时任浙江省省长的[[李强_(1959年)|李强]]在考察云栖小镇时，肯定了云栖小镇的发展模式，并首次提出了特色小镇的概念<ref>{{cite web |title=管委会主任复盘云栖小镇炼成记：7000万到10个亿 |url=http://3g.163.com/dy/article/D4AOLPM40518H2UA.html |website=网易 |accessdate=2018-10-17}}</ref>。

云栖小镇前身是2002年经杭州市规划局批准设立的杭州转塘科技园区。<ref>[http://hzcloudtown.com/web-site/xzdt/gyxz_list.jsp?type=1&xzjj=on 云栖小镇简介]</ref>云栖小镇是第二批省级特色小镇<ref>[http://tsxz.zjol.com.cn/ycnews/201809/t20180913_8262083.shtml 5个！浙江省政府正式命名第二批省级特色小镇]</ref>，云栖大会永久举办地，是以云计算产业为主导的特色小镇。辖区内有2所高等院校，[[西湖大学|西湖大学]]云栖校区（杭州西湖高等研究院）、中共杭州市委党校。国家3A级旅游景区，免费参观<ref>[http://www.zj.xinhuanet.com/2017-11/16/c_1121967354.htm 杭州云栖小镇上榜国家3A级旅游景区-新华网]</ref>。2017年度财政收入超7亿元人民币<ref>[http://hzcloudtown.com/web-site/xzdt/gyxz_list.jsp?type=1&xzjj=on 云栖小镇简介]</ref>。设置的管理机构为：杭州云栖小镇管理委员会（西湖区下属的正处级机构），现任主任、党委书记：吕钢锋。

== 云栖大会 ==
云栖大会的前身是阿里云开发者大会，2013年开始移师至云栖小镇举办，2015年更名为云栖大会，当年主会场迁入新建的云栖小镇国际会展中心。云栖大会是目前中国科技届的主流盛会，2018年云栖大会到场人数12万人<ref>{{cite web |title=2018云栖大会开幕 预计到场人数超过12万 |url=http://finance.ifeng.com/a/20180919/16514928_0.shtml |website=凤凰网}}</ref>。

== 主要道路 ==
辖区内横向主要道路有，浮山街、石龙山街和河山街等。纵向道路主要有科海路、山景路、云梦路和凌周线等。
