'''云南南滚河国家级自然保护区'''是[[中华人民共和国|中华人民共和国]][[云南省|云南省]][[临沧市|临沧市]]的一个[[国家级自然保护区|国家级自然保护区]]，地跨[[沧源佤族自治县|沧源佤族自治县]]和[[耿马傣族佤族自治县|耿马傣族佤族自治县]]，距离[[中缅边界|中缅边界]]最近仅2.4千米<ref>{{cite book |author1=沧源佤族自治县民族中学地理室 |author2=沧源佤族自治县交通局 |date=2002年12月 |title=《沧源地理》 |url= |location=[[昆明|昆明]] |publisher=云南科技出版社 |page=35 |isbn=7-5416-1753-9 |author-link= }}</ref>。自然保护区的森林覆盖率达94.3%，被誉为“[[北回归线|北回归线]]以南最后一块绿洲”<ref name="概况">{{cite book |author1=沧源佤族自治县概况编写组 |author2=沧源佤族自治县修订委员会 |date=2007年11月 |title=《沧源佤族自治县概况》 |url= |location=[[北京|北京]] |publisher=[[民族出版社|民族出版社]] |page=197 |isbn=978-7-105-08553-8 |ignore-isbn-error=true |author-link= }}</ref>，自建立以来从未出现过大的森林火灾。南滚河国家级自然保护区是[[白掌长臂猿|白掌长臂猿]]在中国的唯一分布地，主要保护对象为[[亚洲象|亚洲象]]和热带季雨林景观。保护区的自然环境和植被近似[[西双版纳|西双版纳]]，动物分布种类与之大体相同；又因与缅甸为邻，动物常出入国境有利于促进物种交流，因此也具有东南亚动物区系的特征。<ref name="中国林业网">{{cite web |url=http://nghnr.forestry.gov.cn/business/htmlfiles/nghcybhq/jqgk/index.html |title=保护区概况 |author=云南省沧源南滚河国家级自然保护区 |date= |website=nghnr.forestry.gov.cn |publisher=中国林业网 |access-date=2018-01-12 |quote=}}</ref><ref name="人民网">{{cite web |url=http://yn.people.com.cn/news/yunnan/n2/2016/1028/c375157-29221248.html |title=云南沧源南滚河国家级自然保护区生态环境健康发展 |author1=赵淑芳 |author2=肖红军 |author3=赵文明 |date=2016-10-28 |website=yn.people.com.cn |publisher=人民网-云南频道 |access-date=2018-01-12 |quote=}}</ref><ref name="沧源佤族自治县志">{{cite book |author=沧源佤族自治县地方志编撰委员会 |date=1998年12月 |title=《沧源佤族自治县志》 |url= |location=[[昆明|昆明]] |publisher=云南民族出版社 |page= |isbn=7-5367-1498 |ignore-isbn-error=true |author-link= }}</ref>{{rp|237,241}}

==地理地质==

南滚河国家级自然保护区总面积为508.87平方千米，范围为北纬23°09′~23°40′、东经98°57′~99°26′之间，西起[[沧源佤族自治县|沧源县]]小母空山顶，经[[班洪乡|班洪村]]到[[耿马傣族佤族自治县|耿马县]][[勐简乡|三家村]]，东起沧源县登革洛河岔口到耿马县石楼梯，南起沧源县[[勐角傣族彝族拉祜族乡|勐角]]林场，北到耿马县三叉河<ref name="国务院办公厅">{{cite web |url=http://www.gov.cn/gongbao/content/2003/content_62403.htm |title=国务院办公厅关于扩大云南南滚河等 国家级自然保护区范围的通知 |author=[[中华人民共和国国务院办公厅|国务院办公厅]] |date=2003-08-26 |website=www.gov.cn |publisher=中国政府网 |access-date=2018-01-12 |quote=}}</ref>。保护区位于[[横断山脉|横断山系]][[怒山|怒山山脉]]的南延部分，属滇西纵谷区，是[[云贵高原|云贵高原]]向[[缅甸|缅甸]][[掸邦高原|掸邦山地]]的过渡地带，地跨[[澜沧江|澜沧江]]、[[怒江|怒江]]两大水系<ref>{{cite web |url=http://www.zgqxb.com.cn/xwbb/hynews/201012/t20101208_14282.htm |title=国家林业局批准云南南滚河国家级自然保护区生态旅游规划 |author= |date=2010-12-08 |website=www.zgqxb.com.cn |publisher=新华网 |access-date=2018-01-12 |quote=}}</ref>。[[南滚河|南滚河]]是云南境内[[怒江|怒江]]下游仅次于[[勐波罗河|勐波罗河]]、[[南定河|南定河]]的第三大支流，水网丰富<ref>{{cite journal |author1=周伟 |author2=许亚 |author3=张庆 |author4=郑玲 |date= |title=南滚河国家级自然保护区鱼类资源及其评价 |url=http://www.cncdiversitas.org/sites/default/files/23_4.pdf |journal=《第六届全国生物多样性保护与持续利用研讨会论文集》 |volume= |issue= |pages=203-209 |doi= |access-date=2018-01-12 }}</ref>。

地质上属古滇西地槽的中三江褶皱系的一部分，在[[中生代|中生代]][[印支构造期|印支运动]]和[[燕山构造期|燕山运动]]时褶皱成陆，后受[[喜马拉雅构造期|喜马拉雅山运动]]影响，[[第三纪|第三纪]]开始形成高原雏形，自[[第四纪|第四纪]]起高原强烈提升，河流急剧下切，形成山河相间，沟谷纵横的中山山地复杂地貌。<ref>{{cite web |url=http://www.cangyuan.gov.cn/show-20-10527-1.html |title=云南南滚河国家级自然保护区沧源管理局展开规模性资源调查 |author=田震 |date=2016-10-09 |website=www.cangyuan.gov.cn |publisher=沧源县人民政府 |access-date=2018-01-12 |quote=}}</ref>

==历史==

1979年5月9日，[[临沧地区|临沧地区]]林业局批复《关于加强沧源县[[班老乡|班老]]原始森林中的珍贵野生动物、植物资源所需人员经费的报告》，同意在专管机构正式成立之前配备脱产管护人员。同年9月14日，沧源县林业局正式向地区林业局申报设置自然保护区；10月25日，地区行署向云南省政府呈报《关于建立南滚河自然保护区的请示报告》；12月3日，省人民政府向[[中华人民共和国国务院|国务院]]上报《关于建立南滚河自然保护区的报告》。1980年3月16日，国务院批复省政府，同意将[[南滚河|南滚河]]流域的部分地区划为自然保护区。1980年7月11日，省政府批准设立“云南省沧源佤族自治县南滚河自然保护区管理所”，另设一个受管理所领导的森林派出所。<ref name="沧源佤族自治县志"></ref>{{rp|246}}

南滚河自然保护区是云南省首批建立的保护区之一，最初仅包含沧源县西部[[班洪乡|班洪乡]]和[[班老乡|班老乡]]的部分地区，面积70.825平方千米，1995年被国务院批准为[[国家级自然保护区|国家级自然保护区]]。2003年保护区范围扩大，新增加面积438.05平方公里，澜沧江省级自然保护区的大青山——窝坎大山子片区356.26平方公里并入南滚河自然保护区，剩余地区为南郎片区8.66平方公里，芒库河片区73.13平方公里<ref name="国务院办公厅"></ref>。南滚河国家级自然保护区在2009年被认定为云南省科普教育基地<ref name="云南科普网">{{cite web |url=http://www.ynkp.cn/kpjd/show/725.aspx |title=南滚河国家级自然保护区 |author=云南省科学技术情报院 |date= |website=www.ynkp.cn |publisher=云南科普网 |access-date=2018-01-12 |quote= }}{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2016年成为第五批全国“生物多样性保护示范项目”区域之一。<ref>{{cite web |url=http://nghnr.forestry.gov.cn/publicfiles/business/htmlfiles/nghcybhq/stzy/201507/39161.html |title=自然保护 |author=云南省沧源南滚河国家级自然保护区 |date= |website=nghnr.forestry.gov.cn |publisher=中国林业网 |access-date=2018-01-12 |quote=}}</ref><ref>{{cite web |url=http://www.cangyuan.gov.cn/show-19-11434-1.html |title=云南南滚河国家级自然保护区沧源片区成为第五批全国“生物多样性保护示范项目”区域之一 |author=田震 |date=2017-02-18 |website=www.cangyuan.gov.cn |publisher=沧源县人民政府 |access-date=2018-01-12 |quote=}}</ref>

==生物资源==

南滚河国家级自然保护区在中国植物区系的分区上属于古热带植物的滇、缅、泰地区；植被区划属于热带季雨林、雨林区，西部（偏干性）季雨林、雨林亚区，北热带季节性雨林、半常绿季雨林地带中的滇西南河谷山地半常绿季雨林区<ref name="沧源佤族自治县志"></ref>{{rp|239}}。有记录的[[维管植物|维管植物]]共有2,439种，分属222科，1,066属。其中[[蕨类植物|蕨类植物]]43科，94属，150种，[[种子植物|种子植物]]179科，972属，2,289种。植物名录中属珍稀濒危保护植物的共计216种，占保护区维管植物种类的8.86%，包括[[国家重点保护野生植物名录|国家级保护植物]]18种，其中蕨类植物5种，被子植物13种。云南省级保护植物35种，均为被子植物。其他还有[[苔藓植物|苔藓植物]]37科，83属，165种；大型[[真菌|真菌]]4个纲，22科，56属，102种。森林以[[热带雨林|热带雨林]]、热带季雨林为主，重要植物有[[桫椤|桫椤]]、[[长蕊木兰|长蕊木兰]]、[[见血封喉树|箭毒木]]、[[千果榄仁|千果榄仁]]、[[董棕|董棕]]、[[铁杉|铁杉]]、[[铁力木|铁力木]]、[[四数木|四数木]]、[[云南石梓|云南石梓]]等。<ref name="生物资源">{{cite web |url=http://nghnr.forestry.gov.cn/publicfiles/business/htmlfiles/nghcybhq/stzy/201507/39156.html |title=南滚河沧源管理局自然资源 |author=云南省沧源南滚河国家级自然保护区 |date= |website=nghnr.forestry.gov.cn |publisher=中国林业网 |access-date=2018-01-12 |quote=}}</ref><ref name="云南科普网"></ref><ref>{{cite book |author=徐天铎 |title=《畅游世界系列：畅游大西部》 |url=https://books.google.ie/books?id=Eo2aDQAAQBAJ&pg=PT427&lpg=PT427&dq=%E5%8D%97%E6%BB%9A%E6%B2%B3%E5%9B%BD%E5%AE%B6%E7%BA%A7%E8%87%AA%E7%84%B6%E4%BF%9D%E6%8A%A4%E5%8C%BA&source=bl&ots=WznswqLHoC&sig=gX4aUpbcisq5WPjCYSYLWQ2frsU&hl=zh-CN&sa=X&ved=0ahUKEwjnm6GF_dLYAhVpJMAKHdhJA0U4MhDoAQhAMAU#v=onepage&q=%E5%8D%97%E6%BB%9A%E6%B2%B3%E5%9B%BD%E5%AE%B6%E7%BA%A7%E8%87%AA%E7%84%B6%E4%BF%9D%E6%8A%A4%E5%8C%BA&f=false |location=[[北京市|北京]] |publisher=[[中国轻工业出版社|中国轻工业出版社]] |page= |date=2011年 |isbn=978-7-5019-8317-9}}</ref>

动物地理区划上，南滚河保护区属于[[东洋界|东洋界]]、中印亚界、华南区的滇南山地亚区，是热带动物科属分布的北界，按生态类型划分属热带森林、灌木、草地——农田生态地理动物群<ref name="沧源佤族自治县志"></ref>{{rp|241}}。[[哺乳动物|哺乳动物]]有10目，30科，79属，102种；[[两栖动物|两栖动物]]3目8科19属26种，[[爬行动物|爬行动物]]2目12科30属40种；[[鸟类|鸟类]]18目43科264种；[[鱼类|鱼类]]6目10科29属40种；[[昆虫|昆虫]]10目84科610种。其中被列入国家重点保护野生动物名单的共计63种。保护区内栖息有[[亚洲象|亚洲象]]、[[孟加拉虎|孟加拉虎]]、[[金钱豹|金钱豹]]、[[白掌长臂猿|白掌长臂猿]]、[[黑冠长臂猿|黑冠长臂猿]]、[[林麝|林麝]]、[[豚鹿|豚鹿]]等19种[[国家一级重点保护野生动物名录|国家一级保护动物]]，[[猕猴|猕猴]]、[[短尾猴|短尾猴]]、[[小熊猫|小熊猫]]等44种[[国家二级重点保护野生动物名录|国家二级保护动物]]。南滚河国家级自然保护区是白掌长臂猿在中国的唯一分布地，以及亚洲象的唯二分布地之一（另一处为[[西双版纳傣族自治州|西双版纳]]）。<ref name="生物资源"></ref><ref>{{cite journal |author=肖亮中 |date=2002年 |title=生态孤岛 南滚河 |url=http://www.dili360.com/cng/article/p5350c3d6a7f8c91.htm |journal=《[[中国国家地理|中国国家地理]]》 |volume=09 |issue= |pages= |doi= |access-date=2018-01-12 }}</ref><ref name="人民网"></ref><ref>{{cite web |url=http://zjbwg.cdstm.cn/index.php?m=sevenbooks&a=showall&placeid=452&typeid=71 |title=云南南滚河国家级自然保护区 |author=云南南滚河国家级自然保护区 |date= |website=zjbwg.cdstm.cn |publisher=中国数字科技馆 |access-date=2018-01-12 |quote=}}</ref>
