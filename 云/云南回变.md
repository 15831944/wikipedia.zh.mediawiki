'''雲南回變'''，又稱'''杜文秀起义'''，是1856年发生在[[清朝|清朝]][[云南省|云南省]]的一次大規模[[回族|回族]]穆斯林武裝反抗[[清朝政府|清朝政府]]统治的[[起事|起事]]。1872年被[[清军|清军]]镇压。

== 背景 ==
19世纪，为争夺土地和矿权，云南的[[汉族|汉族]][[土地主|地主]]商人和[[回族|回族]]地主商人之间发生的冲突，这些冲突日益激化。

[[道光|道光]]二十六年（1846年），[[保山县|保山县]]发生永昌汉、回互鬥事件，官府袒护汉族，纵汉杀回，[[杜文秀|杜文秀]]家人全部遇难。杜文秀等人先后前往[[昆明|昆明]]和[[北京|北京]]上诉，都未能得到公正待遇。

== 过程 ==
[[咸丰_(年号)|咸丰]]六年（1856年），[[臨安府_(雲南)|临安府]]（[[治所|治所]]在今[[建水县|建水县]]）因矿权纠纷发生汉、回冲突，冲突蔓延到昆明等地。云南巡抚下令对“滋事”回民，格杀无论。此举激起云南各地回民的反抗。8月，[[杜文秀|杜文秀]]在蒙化（今[[巍山|巍山]]）小围埂村聚众起义。起义军攻克大理，并以此为根据地建立政权，杜文秀任“总统兵马大元帅”，宣传响应[[太平天国|太平天国]]，建立大理政權，廢除清朝年號，制定「遙奉太平天國南京號召，革命滿清」和「聯合回漢一體，豎立義旗，驅逐韃虜，恢復中華，剪除貪污，出民水火」的綱領，規定各民族之間一律平等的原則<ref>{{cite book|author1=李健彪|title=《执著岁月: 回族史与伊斯兰文化》|date=2000|publisher=西安出版社}}</ref>。

同治六年（1867年），杜文秀政权已占有云南一半的地区。同年杜文秀派二十万大军包围[[昆明|昆明]]，但久攻不下。

同治八年（1869年）清军反攻，起义军失利，退回滇西，这次失利成为起义的转折点。

同治十一年（1872年），清军围困[[大理|大理]]，杜文秀服毒自杀，起义失败。

== 影响 ==
{{more|1894年香港鼠疫}}

云南回变期间，部份雲南人口遷移，相信鼠疫是此時隨之遷移至人口較多之兩廣一帶；后来再流行至全球。
