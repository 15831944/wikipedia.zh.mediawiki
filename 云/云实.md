'''云实'''（[[学名|学名]]：{{lang|la|''Caesalpinia decapetala''}}）为[[豆科|豆科]][[云实属|云实属]]的植物，是一種密生且枝幹帶有直鉤狀或倒鉤狀刺的灌木。豆莢狀果實及樹皮富含[[鞣质|鞣质]]、苦澀莖幹及根部可做藥用，屬藥用植物。

== 形态 ==
落叶[[灌木|灌木]]，高2～4米。树皮暗红色，密生倒钩刺。二回羽状复叶，长20-30厘米，羽片3-10对，对生，基部有刺1对，每羽片有小叶7-15对，膜质，长圆形，先端圆，微缺，基部钝，两边均被短柔毛，有时毛脱落。

[[总状花序|总状花序]]顶生，长15-30厘米；总花梗多刺；花左右对称，花梗长2-4厘米，花易脱落；萼片5，长圆形，被短柔毛；花瓣5，黄色，盛开时反卷；[[雄蕊|雄蕊]]10，分离，花丝中部以下密生茸毛；[[子房|子房]]上位。

长椭圆形[[荚果|荚果]]近木质，先端具尖喙，沿腹缝线膨大成狭翅，成熟时沿腹缝开裂，无毛，栗褐色；褐色长圆形种子6-9颗。花、果期4-10月。

== 分布 ==
分布于[[温带地区|温带地区]]、[[亚洲|亚洲]]热带以及[[中国大陆|中国大陆]]的[[河南|河南]]、[[江西|江西]]、[[甘肃|甘肃]]、[[广东|广东]]、[[广西|广西]]、[[四川|四川]]、[[江苏|江苏]]、[[福建|福建]]、[[湖南|湖南]]、[[湖北|湖北]]、[[云南|云南]]、[[河北|河北]]、[[浙江|浙江]]、[[陕西|陕西]]、[[贵州|贵州]]、[[安徽|安徽]]等地。生长于海拔150米至2,100米的地区。一般生長在深山老林，或平原、丘陵、山坡灌丛中以及河旁等區域。目前尚未由人工引种栽培。<ref name=tannincdecaptetala>大陸中心／綜合報導,"網傳「斗米蟲」是最好補品　460一條有錢無處買"[http://www.ettoday.net/news/20121116/128377.htm],ETtoday 新聞雲,2012年11月16日/13:42.</ref>

== 别名 ==
「百鳥不停」、药王子（湖北宜昌）、铁场豆（福建）、马豆、水皂角、天豆。

==蛀虫==
一種生活在雲實樹莖中的蠹蟲稱為云实蛀虫，也是一種可增強人體免疫力的藥用蛀虫。蟲體為蜷縮狀，伸開有8至9公分長。別名「斗米蟲」，在《[[本草綱目|本草綱目]]》、《[[神農本草經|神農本草經]]》、《[[中國藥典|中國藥典]]》等典籍均有記載。<ref name=tannincdecaptetala/>
