'''云南驿机场'''是[[第二次世界大战|第二次世界大战]]期间建于今[[云南省|云南省]][[大理白族自治州|大理白族自治州]]内的一座[[军用机场|军用机场]]，二战时的机场与如今的云南驿机场并非同一机场，现址位于[[祥云县|祥云县]][[云南驿镇|云南驿镇]]果城村委会北屯村。二战时期在此建有2座机场，云南驿机场和'''北屯机场'''，原云南驿机场已荒废并被开垦为农田<ref>{{cite web |url=http://xw.kunming.cn/a/2015-07/01/content_3952467.htm |title=寻访抗战重镇云南驿机场 |author=刘潺 |date=2015-07-01 |website=xw.kunming.cn |publisher=新华网云南 |access-date=2018-07-28 |quote=}}</ref>。如今的云南驿机场实为原北屯机场（祥云县人民政府发布的净空保护区域通告<ref group="注">《关于禁止在云南驿机场净空保护区域内放飞鸟（鸽）类动物的通告》</ref>中，使用的机场名为云南驿机场<ref name="净空保护区域通告">{{cite web |url=http://kuaibao.qq.com/s/20180703B1CUXC00?refer=spider |title=祥云这些净空保护区域范围，禁止放养放飞鸟类动物！ |date=2018-07-03 |website= |publisher=祥云时讯 |access-date=2018-07-28 |quote=}}</ref>）。

==历史==
1939年初，北屯机场动工，同年7月底完工，耗资15,750元[[中华民国货币_(大陆时期)|新滇币]]。机场长800米宽600米，面积64万平方米。建成后成为中华民国航空委员会空军学校云南驿初级班的教练机场，北屯机场也是[[云南驿机场_(二战时期)|云南驿机场]]的辅助机场，主要负责起降[[道格拉斯DC-3|DC-3]]、[[C-47运输机|C-47]]等型号的运输机<ref>{{cite web |url=http://www.lotour.com/zhengwen/1/lg-jc-9922.shtml |title=驼峰航线上的一片祥云 |last= |first= |date= |website=www.lotour.com |publisher=乐途旅游 |access-date=2017-10-01 |quote=}}</ref>，1944年曾遭[[陸軍飛行戰隊_(日本陸軍)|日军]]轰炸。<ref name="大理州交通志">{{cite book |author=大理白族自治州交通局 |date=1991 |title=大理白族自治州交通志 |url= |location=昆明 |publisher=云南人民出版社 |page=P 210 |isbn=7-222-00770-9 |author-link= }}</ref>

[[中华人民共和国|中华人民共和国]]成立后北屯机场成为[[中国人民解放军|中国人民解放军]]的军用机场并被保留下来。1961年进行扩建，跑道增长至3,000米。曾是[[中国人民解放军空军第十航空学校|中国人民解放军空军第十航空学校]]第四训练团的驻地。1985年，机场停止使用。<ref name="大理州交通志"></ref>

2018年[[谷歌地图|谷歌地图]]的卫星影像显示，北屯机场已再次启用，名称更改为“云南驿机场”<ref name="净空保护区域通告" />，停机坪中有战斗机停放。<ref>{{Google maps|url=https://www.google.cn/maps/@25.4407672,100.7359452,3526m/data=!3m1!1e3 |title=北屯机场 |accessdate=2018-07-25}}</ref>
