'''云山学校'''，位于[[湖南省|湖南省]][[长沙市|长沙市]][[宁乡县|宁乡县]][[横市|横市]][[水云山|水云山]]下面。<ref name="book" >{{cite book |language =zh-hans|author=黄海潮、江宏照|title=《宁乡史地》|year=2002年|publisher=南方出版社|location=海南|pages=第49-50页|isbn=7-80660-538-X}}</ref>

== 简介 ==
[[清朝|清朝]][[同治|同治]]三年（1864年）[[宁乡|宁乡]]人[[刘典|刘典]]建立云山书院。<ref name=" web">{{cite web|url=http://www.xxcb.cn/show.asp?id=995973|title=《百年云山书院“变脸”成学校》|accessdate=2009-11-26|work=|publisher=潇湘晨报|language=zh-hans|deadurl=yes|archiveurl=https://archive.is/20130424161929/http://www.xxcb.cn/show.asp?id=995973|archivedate=2013-04-24}}</ref>云山学校的前身是云山书院。云山学校坐落于[[水云山|水云山]]下。左右有环抱的[[天马山|天马山]]，前面是[[沩水|沩水]]。云山学校仿佛是龙头矗立于此。<ref name="book" />

== 建筑 ==
云山学校是典型的[[江南|江南]]院宅。建筑布局为“国”字大院，占地7200平方米。有讲堂、[[文昌阁|文昌阁]]、先师堂等房屋一共158间，白墙青瓦。还有池塘、树木。院内现存有[[清朝|清朝]][[陕西巡抚|陕西巡抚]][[刘典|刘典]]书写的“倡建云山书院碑”。学校正房前有[[谢觉哉|谢觉哉]]书写的“云山完小”四个大字，周围有[[谢觉哉|谢觉哉]]、[[王凌波|王凌波]]种植的“黄金树”等树木。<ref name="book" />

== 校友 ==
云山学校是[[宁乡县|宁乡县]]革命的摇篮。[[何叔衡|何叔衡]]、[[姜梦周|姜梦周]]、[[王凌波|王凌波]]、[[谢觉哉|谢觉哉]]曾经都在这里读书，同时也在这里教书，被誉为“云山四友”和“[[宁乡四髯|宁乡四髯]]”。<ref name="book" />
