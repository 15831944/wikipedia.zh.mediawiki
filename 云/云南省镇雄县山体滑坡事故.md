'''云南省镇雄县山体滑坡灾害'''，是指2013年1月11日上午8时20分，[[云南省|云南省]][[昭通市|昭通市]][[镇雄县|镇雄县]][[果珠乡|果珠乡]]高坡村赵家沟村民组发生的特大[[山体滑坡|山体滑坡]]事故。事故共造成46人死亡，2人受伤。房屋、牲畜、道路交通等损失共计4550余万元。<ref>{{cite web | url=http://news.sina.com.cn/c/2013-01-13/025426021273.shtml| title=云南山体滑坡事故前雨雪持续1个月无预警 | publisher=新浪新闻网 | accessdate=01.13, 2013 |}}</ref><ref>{{cite web| url=http://www.zx.gov.cn/ReadNews.asp?NewsID=23064&BigClassName=镇雄新闻&SmallClassName=时政要闻&SpecialID=0| title=我县召开“1.11”山体滑坡应急救援第二次新闻发布会| publisher=镇雄县委政府网| accessdate=2013年1月12日| 5=}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==事故原因==
据滇东北地质灾害防治专家组组长姜兴武说：<ref>{{cite web | url=http://news.hexun.com/2013-01-12/150094242.html | title=专家解析云南镇雄山体滑坡四原因 | publisher=和讯新闻网 | accessdate=2013-01-12 |}}</ref>
{{quote|
#滑坡区地形陡峻，坡度达35度到50度，地形条件不利于斜坡稳定，为滑坡形成提供了有效的滑移空间；
#滑坡区是第四纪残坡体堆积物，以粘性土为主，厚度大，自身稳定性差，是滑坡形成的内在因素；
#连续雨雪天气是滑坡发生的直接诱因，最近一个月，该区域持续降雨降雪，堆积物水分渗透浸泡后达到饱和，在重力作用下发生滑坡；
#[[2012年彝良地震|彝良县“9·7”地震]]对滑坡形成也有一定影响。
}}
但官方给出的纯属天灾、不存在人祸因素的事故报告深受当地村民与外界的广泛质疑。当地村民称专家组用不足24个小时作出的事故报告太过草率，希望能与专家“面对面辩论”。<ref>{{cite web | url=http://news.qq.com/a/20130114/001022.htm| title=云南滑坡地500米内14个煤窑 村民称有人祸因素 | publisher=腾讯网 | accessdate=2013年1月14日 |}}</ref>矿工王翔（化名）说：“我在高坡煤矿干了5年多，矿里什么情况我一清二楚，滑坡发生的地点，下方就是煤矿的采空区，本来有煤，但是被采空了，我非常确定。”这一说法得到在场另一名矿工的证实。<ref>{{cite web | url=http://news.qq.com/a/20130115/000356.htm | title=镇雄多名矿工称非常确定滑坡正下方是采空区 | publisher=腾讯网 | accessdate=01,15,2013 |}}</ref>1月15日，数十名遇难者家属和村民签署联名信，准备致函国家煤监局、国土资源部等单位，希望组成联合调查组，重新调查高坡村滑坡灾难。家属代表称，他们不信任云南省地质专家做出的滑坡调查结论。<ref name=" xxcb">{{cite web|url=http://www.zj.xinhuanet.com/newscenter/InAndAbroad/2013-01/16/c_114381485.htm|title=云南镇雄72名村民联名要求重新调查滑坡事件|accessdate=2013年1月16日|publisher=新华网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20130619032934/http://www.zj.xinhuanet.com/newscenter/InAndAbroad/2013-01/16/c_114381485.htm|archivedate=2013年6月19日}}</ref>

==事故过程==
云南省昭通市镇雄县果珠乡高坡村赵家沟村民小组最近十来天，持续下雪，9日晚和10日更是一直下大雪。1月11日上午8点20分左右，突然发生山体滑坡、泥石流。滑坡断面垂直约500米，坡面有700—800米。<ref>{{cite web | url=http://news.sina.com.cn/c/2013-01-11/172526014918.shtml| title=目击者称云南镇雄县滑坡就像卡车卸沙石 | publisher=新浪新闻网 | accessdate=01,11,2013 |}}</ref>
据《[[人民日报|人民日报]]》报道，此次山体滑坡体长120米、宽110米、厚16米，滑坡范围共涉及16户群众、68人，其中被山体掩埋14户、46人，被掩埋房屋63间、畜厩和厕所31间、猪59头、牛5头，毁坏耕地500余亩。<ref>{{cite web | url=http://news.sina.com.cn/c/2013-01-11/221526015800.shtml| title=云南昭通山体滑坡长120米宽110米 | publisher=新浪新闻网 | accessdate=01,11,2013 |}}</ref>

==救援==
事故发生后，[[镇雄县|镇雄县]][[林口彝族苗族鄉|林口彝族苗族鄉]]组织的第一支救援力量最先到达现场[[搜救|搜救]]，随后相继有30余支救援队伍前来支援，救援人员达2000余人。<ref>{{cite web | url=http://news.qq.com/a/20130112/000596.htm| title=云南镇雄山体滑坡事故46具遇难者遗体全部找到 | publisher=腾讯网 | accessdate=01,12, 2013 |}}</ref>灾情发生４个小时，事故现场就集结了400人开展搜救，灾情发生８小时，现场已集结上千人参与搜救。11日19时20分左右，云南省省长[[李纪恒|李纪恒]]率领工作组到达现场指挥搜救。<ref>{{cite web | url=http://news.xinhuanet.com/2013-01/12/c_114345242.htm| title=搜救，争分夺秒——云南镇雄山体滑坡搜救纪实 | publisher=新华网 | accessdate=01,12, 2013 |}}</ref>12日11时50分许，最后1名失踪者遗体被找到，遇难人数确认为46人，搜救工作结束。<ref>{{cite web | url=http://news.xinhuanet.com/2013-01/12/c_114345223.htm| title=悲情赵家沟--来自云南镇雄山体滑坡现场的调查报告 | publisher=新华网 | accessdate=01,12, 2013 |}}</ref>

==事故后处置==
1月13日，在未经遇难者家属签字同意、部分家属没见到亲人最后一面的情况下，遇难者遗体在镇雄县殡仪馆匆忙全部火化，引起遇难者家属不满和愤怒。<ref>{{cite web | url=http://news.qq.com/a/20130115/000048.htm| title=云南镇雄就匆忙火化遗体致歉 称暂未考虑补偿 | publisher=腾讯网| accessdate=01,15,2013 |}}</ref>1月14日，鎮雄縣委、縣政府向遇難者家屬及親屬誠懇道歉，並責令縣民政局及殯儀館作出深刻的書面檢查。<ref>{{cite web | url=http://politics.people.com.cn/BIG5/n/2013/0115/c14562-20202385.html| title=雲南鎮雄就“強行火化遇難者遺體”道歉 | publisher=人民网| accessdate=2013年1月15日|}}</ref>镇雄县“1·11”山体滑坡应急救援指挥部负责人14日介绍说，由于工作中考虑不够周全，对遇难者家属及亲属造成了伤害，县应急救援指挥部积极开展疏导和安抚工作，并代表县委、县政府向遇难者家属及亲属作出了诚恳道歉。<ref>{{cite web| url=http://www.morningpost.com.cn/szb/html/2013-01/15/content_205083.htm| title=镇雄县就强行火化遗体道歉| publisher=北京晨报| accessdate=2013年1月15日| 5=| deadurl=yes| archiveurl=https://archive.is/20130424193001/http://www.morningpost.com.cn/szb/html/2013-01/15/content_205083.htm| archivedate=2013年4月24日}}</ref>
1月15日《[[潇湘晨报|潇湘晨报]]》发表评论文章称：“强行火化之类的工作作风，是一种人祸和权力的‘滑坡’”。<ref>{{cite web | url=http://epaper.xxcb.cn/XXCBA/html/2013-01/15/content_2678361.htm| title=强行火化是权力“滑坡” | publisher=潇湘晨报| accessdate=2013年1月15日|}}</ref>
