{{vgname|還願|en=Devotion}}{{NoteTag|英語Devotion的字義是「奉獻」、「祈禱」、「忠誠」。}}是一款由[[赤燭遊戲|赤燭遊戲]]發行於Steam平台的[[第一人称视角游戏|第一人稱視角]][[恐怖遊戲|恐怖]][[益智游戏|解謎遊戲]]，遊戲選於2019年2月19日（恰為該年農曆[[上元節|上元節]]）開放下載<ref name="ltn" /><ref name="nownews" />。游戏描述了1980年代的[[臺灣|臺灣]][[臺北|臺北]]的一個小公寓中，一個試圖遵循傳統觀念的三口之家周邊發生的诸多事件<ref name="gamicsoft" />。遊戲之名稱「[[還願|還願]]」，是[[佛教|佛教]]、[[道教|道教]]的一種宗教行为。在信徒[[禱告|禱告]]祈求的願望實現後，会以此報答回饋[[神仙|神仙]]、[[聖者|聖者]]甚至[[妖魔|妖魔]]等。遊戲以1980年代的[[臺灣|臺灣]]為故事背景，並融入1970年代、1980年代常見事物（如歌謠、選秀節目）與[[臺灣民間信仰|臺灣民間信仰]]概念。遊戲採用[[第一人称视角游戏|第一人稱視角]]來進行，玩家將扮演家庭男主人杜豐于的角色進行遊戲<ref name="gamicsoft"/>。玩家一開始需在四個不同時間軸、類似而不相同並彼此相連的舊公寓場景中搜尋線索、[[推理|推理]]解謎，劇情將會隨著各時間軸中的線索與道具揭曉後推進。

== 剧情概述 ==
故事從杜家公寓內的客廳開始，玩家操控主角[[劇作家|劇作家]]杜豐-{于}-，為尋找其女兒杜美心的下落，在1980年到1987年間的杜家公寓裡進行探索，隨著找到的線索逐步揭露故事全貌。

當紅玉女歌手鞏莉芳在與知名剧作家杜豐-{于}-結婚後息影，但婚後由於杜豐-{于}-的劇本逐漸不合時代潮流，家庭經濟每況愈下，鞏莉芳雖意圖復出歌壇，貼補家計，卻被[[大男人主義|大男人主義]]的杜豐-{于}-強烈反對，而使夫妻間的爭執日益擴大。杜美心以「和媽媽一樣成為大明星」為願望，被家裡寄予重望，但她卻在參加[[選秀節目|選秀節目]]《七彩星舞台》演唱時失常，後由於家庭失和與自身的壓力開始喘不過氣、無法唱歌。

起初發作時，杜豐-{于}-向冒牌神棍何老師請益，並請示「慈孤觀音」。美心在服藥及杜豐-{于}-的陪伴下逐漸康復。只是杜豐-{于}-誤以為是美心得到慈孤觀音的庇蔭才得以康復，這也讓杜豐-{于}-更加信任慈孤觀音的法力。而後美心的病情因不明原因再度惡化，至醫院檢查後仍查無生理病因，醫師建議轉診[[精神科|精神科]]。由於在80年代的台灣，精神疾病被視為一種惡疾，有鑑於此不願相信女兒是「神經病」的杜豐-{于}-再次求助於何老師與慈孤觀音，並更加聽信於偏方及宗教儀式。鞏莉芳意圖將丈夫導回正軌，不料卻被走火入魔的杜豐-{于}-視作妖魔鬼怪，還被逼迫去慈孤觀音面前懺悔。鞏莉芳無奈之下只得離家，等待重返演藝圈生計穩定後再將美心及丈夫接到單純的地方過活。與此同時，杜豐-{于}-在何老師的指引下，透過[[觀落陰|觀元辰宮]]進入自己的元辰宮，並向慈孤觀音獻上自己的右眼、舌頭、鮮血，在觀元辰宮結束後將杜美心關進廁所，進行為期七日的瘋狂儀式──活體浸泡蛇酒，且中途禁止開門與進行任何打擾。

一開始杜豐-{于}-請示何老師時，何老師還十分有耐心的告訴他此儀式「輕則當日，重則七日」。等到儀式接近第七天杜豐-{于}-再次打電話找何老師時，電話卻再也無人回應。驚覺大事不妙的杜豐-{于}-再次來到何老師居所時，卻發現人去樓空，只留下一卷信眾在電話中怒罵何老師是[[神棍|神棍]]的卡帶。最後，萬念俱灰的杜豐-{于}-一個人打開了廁所的門，隨後冒出一陣白光，炙烈的白光中顯現出一個如夢似幻的世界，有滿滿的紙摺鬱金香，有紙風車，有杜豐-{于}-成堆的稿紙，有眾多的遊戲設施，以及杜美心逐漸遠去的背影<ref name="NOWnews評" />。

== 角色 ==
===遊戲正篇===
; 杜豐-{于}-（演員：姚舜庭，配音：[[程伯仁|程伯仁]]／[[陳敬恆|陳敬恆]] ）
: 玩家主要操作人物，於六零、七零年代時獲獎無數的[[劇作家|劇作家]]，在與鞏莉芳結婚後育有女兒杜美心。因劇本逐漸不受觀眾及導演歡迎，且不願更改鋪張浪費的生活方式，致使家庭生計出問題。在美心罹患精神病後求助於何老師與「慈孤觀音」，因美心的病情好轉，便每個月給付「[[香油錢|功德金]]」供養慈孤觀音，使生活更加困頓。在鞏莉芳為了家計打算重回歌壇時，因放不下男人的尊嚴而與妻子爭執並離異，美心因此再次深陷重病，杜豐-{于}-便按照何老師的指引，讓美心在蛇酒中浸泡七天。在無法聯絡上何老師後發現她是神棍。

; 鞏莉芳（演員、配音：[[陳品伶|陳品伶]]，演唱：[[何夏|何夏]]）
: 知名玉女歌星，在嫁給杜豐-{于}-後息影，是女兒杜美心嚮往的目標。在家計出問題後時常向母親調頭寸救急，並曾一度想回娘家休養，由於始終無法勸諫丈夫，還被其視為妖魔鬼怪，最終與丈夫離異，離異後，復出演藝圈，想在演藝圈賺夠錢後，接一家人到平靜的地方過日子，但事與願違。

; 杜美心（演員：[[林芷瑈|林芷瑈]]，配音、演唱：[[劉芷融|劉芷融]]）
: 杜豐-{于}-及鞏莉芳之女，小學生。受到母親的影響而嚮往成為歌星，但在參加選秀節目《七彩星舞台》時失利，又因父母時常為錢爭執，產生極大的心理壓力，被醫師建議轉診精神科。但因杜豐-{于}-深信慈孤觀音，便帶她向慈孤觀音求援，最終美心被浸泡在蛇酒中長達七日。杜美心深信繪本《花與愛》中的情節，認為用紙摺的鬱金香佈滿整個房間就能讓自己痊癒，但深信慈孤觀音的父親不願陪自己摺紙。

; 何老師（配音：王娟）
: 於公寓凶宅中設立「慈孤壇」的神媒，奉祀「慈孤觀音」供信徒參拜。但其實何老師只是一個打著慈孤觀音名號，提供信徒旁門左道的神棍。1987年，同樣奉祀慈孤觀音之陸恭銘創立「陸心助法會」，並完全切割陸心助法會與慈孤壇和杜家慘案之關係。

===前導現實互動遊戲（ARG）出現之角色===
; 簡晶菁
: 陸心助法會成員。大學生。曾為陸心製作簡報檔與單曲CD《迎接新明天》。於[[Instagram|Instagram]]發布諸多參與陸心助法會活動之貼文，及一則自己遭到跟蹤之影片，為遊戲重要起始線索之一。其母親簡月瑛對於其經常不回家之情形表示不解，於是和正在調查陸心的鞏劭華合作。但簡晶菁實為陸心助法會為引出鞏劭華與內鬼（後得知為柯文心）而設之誘餌。

; 鞏劭華
: 糾纏簡晶菁之男子，實為鞏莉芳之姪子。對於當年表姊杜美心突然失蹤及家族內傳言杜家出事感到困惑，於是著手調查陸心助法會之黑幕與救助受其洗腦者，卻不慎驚動到陸心而遭[[人肉搜索|人肉搜索]]出其個人資料，之後下落不明，疑遭陸心成員圍剿。

; 柯文心
: 陸心助法會中區幹部。於簡晶菁遭鞏劭華糾纏時出面制止，並拾獲鞏劭華掉落之手機。但在查閱鞏劭華手機內容後得知陸心助法會過往黑幕，於是決定暗中幫助鞏劭華，化名「吳阿娥」於[[YouTube|YouTube]]發布簡晶菁遭騷擾之影片，並透過陸心招募新生的活動暗示陸心內幕予更多人知曉，卻因此使鞏劭華的個人資料遭參與玩家翻出，被潛伏於對話群組中的陸心臥底全數獲悉，自己的內鬼身分亦因此曝光而自身難保。

; 洪欣偉
: 陸心助法會中區幹部。帶領玩家進行陸心助法會體驗活動「清心法」。其父親表示其加入陸心之後「整個人都變了」，於是尋求鞏劭華之協助。

== 開發 ==
赤燭遊戲於2018年2月首次公布新遊戲《還願》<ref name="初發表" />，並於2018年7月公布首支宣傳影片<ref name="宣传片" />。

合作創辦人暨美術設計陳敬恆在2019年[[台北國際電玩展|台北國際電玩展]]接受採訪時表示，本遊戲由江東昱擔任主要製作人，在其提議下團隊決定以第一人稱3D視角製作遊戲。所有成員均為此花了功夫在學習[[Unity_(游戏引擎)|Unity]]引擎。遊戲主體的開發以12人規模的團隊進行<ref name="是否有多結局" />。

===評價===
在[[匯總媒體|匯總媒體]]上，《還願》在[[Metacritic|Metacritic]]根據11條評論獲得85/100分，屬「大致正面評價」<ref name=mc/>。而[[GameRankings|GameRankings]]則是根據4條評論得到83/100分<ref name=gr/>。

评价方面，[[IGN|IGN]]的大衛·傑格諾（{{lang|en|David Jagneaux}}）給予了《還願》8.2分（满分10分），他讚揚其從頭到尾都是一款出色的心理恐怖遊戲，充滿超現實的意象和使人不寒而慄的玩偶，以及精彩的環境故事，但傑格諾同時亦批評有些愚蠢的謎題和令人苦悶的緩慢行走速度，使得遊玩時間比實際的要長，還指出在很多方面來說，《還願》並不是一款恐怖遊戲，而是偵探故事<ref name="IGN"/>。IGN日本分部的今井晉則給予了《還願》9.8分（滿分10分）的高分，评论称赞了游戏的临场感、逼真的音效、强烈的恐怖感以及多种画风描述之下的游戏剧情，但也指出其“后半段游戏的展开偏重於依赖独角戏”的缺点<ref name="IGNJP" />。而[[GameSpot|GameSpot]]的理查·維克林（{{lang|en|Richard Wakeling}}）打出9/10分，形容赤燭喚起了玩家對《[[P.T.|P.T.]]》的恐懼情感，畫出了一幅具創意、發人深省與陰險的家庭生活肖像畫，讚揚其對細節的關注使公寓本身有了逼真的個性，以及不怕面對沉重的主題去敘述一個引人入勝的故事，但他亦指出遊戲後期的追逐場景較為沉悶<ref name="GSpot"/>。

{{link-en|Metro (英國報紙)|Metro (British newspaper)|Metro}}讚揚《還願》是本世代最棒[[恐怖遊戲|生存恐怖遊戲]]之一，但只有三小時的遊戲流程是其缺點<ref name="Metro" />。[[新浪|新浪]]新聞在评论中认为这款游戏将一家三口連同房子組成了一個會隨著交互、觸發而不斷改變場景的家，佐以台灣腔和閩南口音，还原了80年代的台灣生活场景，并认为这是一款不輸於《[[寂靜嶺|寂靜嶺]]》的國風恐怖遊戲<ref name="新浪評論" />。由於爭議事件的爆發，大量中國大陸玩家給予負評，導致其Steam評價一度降為“多半差評”，其后则回升至「褒貶不一」<ref>{{cite web |title=Steam - Devoltion |url=https://store.steampowered.com/app/1006510/Devotion/#app_reviews_hash |publisher=Steam |accessdate=2019-02-23 |archive-url=https://web.archive.org/web/20190222161847/https://store.steampowered.com/app/1006510/Devotion/#app_reviews_hash#app_reviews_hash |archive-date=2019-02-22 |dead-url=no }}</ref><ref name="爭議" />。
