《'''哇!妮莉'''》（英語：Whoa, Nelly!）是[[加拿大|加拿大]]女歌手[[妮莉·費塔朵|妮莉·費塔朵]]的第一張錄音室專輯，於2000年10月24日透過{{le|夢工廠唱片|DreamWorks Records}}發行。

專輯獲得樂評家整體正面的評價，包括稱讚專輯創新地融合了各類曲風與音效，以及費塔朵的嗓音<ref>{{cite web |url=http://www.metacritic.com/music/whoa-nelly!/nelly-furtado |title=Whoa, Nelly! – Nelly Furtado |publisher=[[Metacritic|Metacritic]] |accessdate=2011-07-23 }}</ref>。專輯其後獲提名為[[格林美獎|格林美獎]]最佳流行演唱專輯。

《哇!妮莉》至今發行了四首商業單曲，包括打進[[美國|美國]][[告示牌百強單曲榜|告示牌百強單曲榜]]前10位的《{{le|I'm Like a Bird}}》與《{{le|Turn Off the Light}}》。專輯最高於美國[[告示牌200大專輯榜|告示牌200大專輯榜]]登上第24位，並獲[[美國唱片業協會|美國唱片業協會]]認證為雙白金唱片。截至2009年，專輯已經於全球售出超過700萬張<ref>{{cite web |url=http://www.dailymail.co.uk/home/moslive/article-1215843/Why-I-turned-Playboy-breakdown-Nelly-Furtado-reveals-all.html |title=Why I turned down Playboy and had a breakdown: Nelly Furtado reveals all] |publisher=Daily Mail |date=2009-09-26 |accessdate=2010-10-07}}</ref>。

== 發行背景 ==
費塔朵高中二年級時在[[多倫多|多倫多]]遇到了嘻哈團體瘋狂芝士的團員塔利斯·紐柯克，並在團體的專輯《Join the Ranks》中擔任歌曲《Waitin' 4 the Streets》的主唱<ref name=nelstar>{{cite web|url=http://www.nelstar-project.com/about/nelstar|title=Nelstar* (Nelly Furtado) Biography|work=Nelstar-Project.com|accessdate=2005-12-09|deadurl=yes|archiveurl=https://web.archive.org/web/20060101212435/http://www.nelstar-project.com/about/nelstar/|archivedate=2006-01-01}}</ref>。1996年從高中畢業後，費塔朵搬到了多倫多，並與紐柯克組成了一個[[神遊舞曲|神遊]]團體Nelstar。然而她認為神遊曲風有過多種族隔離的意味，並認為這樣的曲風不能代表她的個性，也無法突顯她的歌唱實力<ref name=nelstar/>。因此，她離開了這個團體回到了家鄉。而她在回到家鄉前參加了1997年的歌唱才藝比賽<ref name=nelstar/>。

費塔朵在歌唱才藝比賽中的表現引起加拿大節奏藍調樂團{{le|哲學家皇帝 (樂團)|The Philosopher Kings|哲學家皇帝}}的主唱{{le|傑拉德·伊頓|Gerald Eaton}}與結他手{{le|布萊恩·韋斯特|Brian West (musician)}}的注意，伊頓與韋斯特幫她製作了一捲試唱帶，這捲帶子為她贏得了一紙唱片合約<ref>{{cite web |url=http://www.umusic.com.tw/album.php?id=2844|title=Whoa, Nelly! + Folklore (2 For 1) / 哇!妮莉 + 故事書 (2 合 1 雙碟) - Nelly Furtado / 妮莉費塔朵|work=[[環球音樂_(台灣)|環球音樂]]|accessdate=2017-06-13}}</ref>。1999年，費塔朵發行了第一張單曲《Party's Just Begun (Again)》，這張單曲收錄在「{{le|強迫入境|Brokedown Palace}}」的電影原聲帶裡。歌曲發行後，伊頓與韋斯特擔任專輯的製作人，與費塔朵一同創作了部分歌曲，並參與製作專輯大部分歌曲。

== 商業成績 ==
《哇!妮莉》獲得國際性的商業成功。專輯最高登上[[加拿大|加拿大]]專輯榜亞軍位置，並獲認證為四白金唱片。在美國，專輯最高登上告示牌200大專輯榜第24位，並獲美國唱片業協會認證為雙白金唱片。專輯打進[[荷蘭|荷蘭]]與[[瑞士|瑞士]]專輯榜前10位，並打進[[澳洲|澳洲]]、[[挪威|挪威]]與[[紐西蘭|紐西蘭]]專輯榜前5位。專輯其後於紐西蘭獲[[新西蘭唱片音樂協會|唱片音樂協會]]認證為三白金唱片，並於[[葡萄牙|葡萄牙]]跟瑞士獲認證為白金唱片。專輯於[[比利時|比利時]][[法蘭德斯|法蘭德斯]]區與[[華隆大區|華隆大區]]皆登上最高第20位，並於[[法國|法國]]登上最高第46位。
