'''坑口站'''（{{lang-en|'''Hang Hau Station'''}}）是[[香港|香港]][[港鐵|港鐵]][[將軍澳綫|將軍澳綫]]的一個地底車站，位於[[新界|新界]][[西貢區|西貢區]][[將軍澳|將軍澳]][[坑口|坑口]]，於2002年8月18日正式啟用，車站以淺藍色為主要色調。坑口站在[[康城站_(香港)|康城站]]啟用前是港鐵系統中最東端車站。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Hang Hau Station 2018 03 part2.jpg | caption1 = 大堂
| image2 = Hang Hau Station 2013 08 part2.JPG | caption2 = 此站採用新款式的八達通增值機，而且與售票機分開擺放
}}  
由於此站採用對向式月台設計，大堂比將軍澳綫其他車站稍大，令兩個月台的扶手電梯能夠在大堂內互通。大堂內提供各類型的商店供乘客購物或進食，例如有[[便利店|便利店]]、[[麵包糕餅店|麵包糕餅店]]、[[時裝店|時裝店]]、[[診所|診所]]等。此外，坑口站也有不少自助服務設施供乘客使用，包括[[自動櫃員機|自動櫃員機]]、[[自動販賣機|自動售賣機]]及自動照相機<ref name="shops"/>。客務中心位於大堂中央，旁邊設有[[香港郵政|香港郵政]][[郵箱|郵箱]]，方便乘客在車站內投寄郵件<ref>{{Cite web|title= 鐵路四處遊－郵箱服務|url=http://www.mtr.com.hk/chi/getting_around/eib.html|work=港鐵網頁|publisher=港鐵公司|accessdate=2014年5月1日|language=zh-hk}}</ref><ref>{{Cite web|title=郵箱位置表－西貢區|url=http://app1.hongkongpost.hk/CGI/location/box_content.jsp?region=%27New%20Territories%27&lang=zh_TW#saikung|work=香港郵政網頁|publisher=香港郵政|accessdate=2014年5月1日|language=zh-hk|deadurl=yes|archiveurl=https://web.archive.org/web/20140415110401/http://app1.hongkongpost.hk/CGI/location/box_content.jsp?region=%27New%20Territories%27&lang=zh_TW#saikung|archivedate=2014年4月15日}}</ref>。近B出入口的非付費區大堂設有「[[會員服務站|會員服務站]]」，供乘客享用港鐵友禮會的會員優惠<ref>{{Cite web|title= 鐵路四處遊－「-{會員服務站}-」|url=http://www.mtr.com.hk/ch/customer/services/csp.html|work=港鐵網頁|publisher=港鐵公司|accessdate=2016年9月26日|language=zh-hk}}</ref>。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Hang Hau Station 2018 03 part3.jpg | caption1 = 1號月台﹙往寶琳方向﹚
| image2 = Hang Hau Station 2018 03 part4.jpg | caption2 = 2號月台﹙往北角方向﹚

坑口站有2個月台，採用[[側式月台|側式月台]]設計，以對向式月台排列<ref name=HAH_SS>{{cite web|title=坑口站位置圖|url=http://www.mtr.com.hk/jplanner/images/layouts/hah.pdf|work=港鐵網頁|publisher=港鐵公司|accessdate=2014年5月1日|language=zh-hk}}</ref>，中間以牆壁分隔，並設有廣告板，月台亦設有[[月台幕門|月台幕門]]。

此站月台尾端設有緊急出口樓梯，為市區綫非轉車站較為少有及罕見的設計<ref>{{cite web|author=MTR之今昔|title=坑口站|url=http://mtr.hk365day.com/images/TKOL/HAHE.jpg|accessdate=2012-08-10}}</ref>。另外，列車由坑口站前往寶琳站途中由於需要從雙線轉入單線（[[寶琳站|寶琳站]]僅有一個月台），故列車可能會為了保持[[閉塞_(鐵路)|安全距離]]、等待信號或在控制中心要求而在未到站時減速，甚至停下，或在坑口站月台等候，直至前方列車離開寶琳站月台及駛過兩線二合為一的[[轉轍器|轉轍器]]後方可前進。

此外，月台尾端亦與其他港鐵新建車站一樣，均有預留空間供車長下車工作，與[[觀塘綫|觀塘綫]]（不包括[[黃埔站_(香港)|黃埔站]]至[[何文田站|何文田站]]及[[油塘站|油塘站]]至[[調景嶺站|調景嶺站]]）、[[荃灣綫|荃灣綫]]、[[港島綫|港島綫]]（不包括[[西營盤站|西營盤站]]至[[堅尼地城站|堅尼地城站]]）及[[東鐵綫|東鐵綫]]（不包括[[羅湖站_(香港)|羅湖站]]及[[落馬洲站|落馬洲站]]）舊式車站設計不同。

==鄰接車站==
坑口站是將軍澳綫的中途站。如果乘客需要在該站往來[[康城站_(香港)|康城站]]，需先乘搭列車到[[將軍澳站|將軍澳站]]，然後前往對面月台轉乘往康城方向的列車。

==利用狀況==
坑口站鄰近[[東港城|東港城]]、[[蔚藍灣畔|蔚藍灣畔]]、[[南豐廣場|南豐廣場]]、[[厚德邨|厚德邨]]、[[安寧花園|安寧花園]]等大型屋苑或屋邨，同時也要兼顧[[清水灣半島_(住宅)|清水灣半島]]、[[將軍澳工業邨|將軍澳工業邨]]、[[香港科技大學|香港科技大學]]及[[西貢市中心|西貢市中心]]的轉乘客，因此成為將軍澳綫8個車站中最多出入閘人流的一個。坑口站也鄰近[[將軍澳醫院|將軍澳醫院]]和[[西貢將軍澳政府綜合大樓|西貢將軍澳政府綜合大樓]]，在該處看病或上班的市民也會使用此站。自康城站通車後，雖然部份[[將軍澳工業邨|將軍澳工業邨]]的乘客已經改用康城站；但因康城站班次疏落（繁忙時間每七分鐘一班車，非繁忙時間每十二分鐘一班，而且非繁忙時間只能到達調景嶺站<ref>{{cite news|title=康城站啟用 居民嫌班次疏|url=http://news.sina.com.hk/news/812/3/1/1212190/1.html|accessdate=2012-08-11|newspaper=星島日報|language=zh-hk|date=2009-07-27}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>），故仍有不少乘客經此站乘搭班次較頻密的列車。

坑口站與將軍澳站之間全為彎曲路軌，列車駛經時噪音嚴重，是整條[[將軍澳綫|將軍澳綫]]中噪音最高的一段。2003年[[民建聯|民建聯]]在英製列車駛經此段時，在車廂內測得噪音高達100.2[[分貝|分貝]]<ref>{{cite news|title=逾百分貝 將軍澳支線噪音最高|url=http://www.thmalex.com/hkris/rnc/new_page_18.htm|accessdate=2012-08-11|newspaper=明報|date=2003-08-29|deadurl=yes|archiveurl=https://web.archive.org/web/20140502010840/http://www.thmalex.com/hkris/rnc/new_page_18.htm|archivedate=2014-05-02}}</ref> 。港鐵稱已經採取措施，降低噪音；2002年購入的[[港鐵Rotem電動列車|韓製列車]]在行走此路段時，噪音明顯比20世紀的[[港鐵都城嘉慕電動列車_(直流電)|英製列車]]為低。在2010年將軍澳綫採用韓製列車之前，乘客因為車廂內噪音嚴重，有時無法聽到車廂內的廣播，甚至無法使用手提電話；改用韓製列車後，因列車採用密封程度較佳的[[內崁式門|內崁式車門]]，車廂內噪音明顯減少。<ref>{{cite news|title=港鐵將軍澳線噪音可達100分貝|url=http://www.youtube.com/watch?v=IPZpDnQEYmo|accessdate=2012-08-11|newspaper=亞洲電視|date=2009-04-09}}</ref> 

此站上蓋住宅為[[蔚藍灣畔|蔚藍灣畔]]，基座商場為[[連理街|連理街]]，而車站旁邊亦設有隧道的通風口。港鐵於2012年推出搭十送一優惠計劃時，乘客亦可選擇前往連理街商場換領[[港鐵免費單程票|免費單程車票]]，從而起到分流作用。<ref>{{cite web|author=港鐵公司|title=連環優惠第一回 搭十送一|url=http://mtr.com.hk/chi/whatsnew/ride10_2012.html|accessdate=2012-08-11}}</ref>

==接駁交通==

==歷史==
在將軍澳綫落實前，坑口站月台東南方（即往將軍澳站方向）的管道在上方修建[[培成路|培成路]]時已預先興建，因此在鐵路工程期間毋須封閉及挖掘該路段，使車站周邊以單程行車為主的道路得以維持正常行車。至於車站所在地已在規劃[[將軍澳新市鎮|將軍澳新市鎮]]時以鐵路預留地的方式預留，曾用作露天收費停車場，直到興建將軍澳綫前被收回。將軍澳綫的動工儀式和嘉年華正是在該處舉行。
