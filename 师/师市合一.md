'''师市合一'''是[[新疆生产建设兵团|新疆生产建设兵团]]与[[新疆维吾尔自治区|新疆维吾尔自治区]]的一种行政体制。兵团的师在其驻地成立县级市，这些市由自治区[[省直辖县级行政区|直辖]]，而由兵团直接管理。平时为民，战时为兵。市委书记同时是师[[政委|政委]]，市长同时就是[[师长|师长]]。过去，工商、税务、交通、环保等公共事务由驻地的地、州政府办理，现在都要归口到新成立的县级市。县级市成立[[人民代表大会|人民代表大会]]，政府工作接受[[人大代表|人大代表]]监督。这几个农建师也通过县级市获得了合法的财政收入与税收留成。

== 历史 ==
新疆生产建设兵团长期面临一个困境：是[[政府|政府]]卻要[[纳税|纳税]]。兵团各师要向所在地的[[地方政府|地方政府]]纳税，导致兵团自身各级没有合法的行政财政地位。

1984年9月，新疆维吾尔自治区党委、人民政府决定农八师和[[石河子市|石河子市]]实行一个党委领导、政企分设的体制。

进入21世纪后，兵团推广农八师的石河子市模式，在另外八个师的驻地又组建了八个[[县级市|县级市]]，实行“师市合一”管理模式。在[[中华人民共和国国务院|国务院]]批复后，兵团“积极与有关部门联系，争取上级领导的支持”，对自治区、有关地州做了一年多大量艰苦工作后，[[新疆维吾尔自治区人民政府|新疆维吾尔自治区人民政府]]终于行文同意这几个县级市挂牌成立。

== 未来发展 ==
根据“十二五”规划，新疆生产建设兵团将按'''“师建城市、团场建镇”'''进一步加快城镇化。2020年，兵团的城镇化率将从2009年的48%增加到70%。一方面，在每个师部建一个或多个市，即“一师一(多)市”；另一方面，在条件成熟团场重点发展小城镇，将各师所辖团场全部设立为建制镇。<ref name=a>{{cite web |url=http://politics.people.com.cn/n/2014/0225/c1001-24452488.html |title=国务院批准新疆设双河市 兵团城镇化将一师一市 |publisher=新京报 |date=2014-02-25 |access-date=2016-12-03}}</ref>
