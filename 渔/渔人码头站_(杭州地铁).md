'''渔人码头站'''是[[杭州地铁9号线|杭州地铁9号线]]的一个站点，位于钱江路与之江路交叉口。<ref>{{cite web |title=地铁5号线二期、9号线一期详细站点位置来了 哪站离你近？_新蓝网 |url=http://n.cztv.com/news/12674668.html |website=新蓝网}}</ref>

== 结构 ==
渔人码头主体沿钱江路布置，为地下二层岛式车站，双柱三跨箱形框架结构，有效站台宽度12.6米、长120米，车站主体长283.2米，标准段宽21.3米。站前设单渡线。车站总建筑面积约17035平方米<ref>{{cite web |title=省发展改革委关于杭州地铁9号线一期工程初步设计的批复 |url=http://www.zjdpc.gov.cn/art/2017/8/22/art_808_1732461.html |website=浙江省发展改革委员会}}</ref><ref>{{cite web |title=杭州地铁9号线一期工程土建施工SG9-2标段土建施工SG9-2标段（中央公园站～钱江路站区间、钱江路站、钱江路站～渔人码头站区间、渔人码头站）招标文件公示 |url=http://www.bidchance.com/info.do?channel=calgg&id=22125592 |website=招标网}}</ref>
