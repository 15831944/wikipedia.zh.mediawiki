'''侯云德'''（{{bd|1929年|7月13日||}}），[[江苏|江苏]][[常州|常州]]人，中国医学病毒学专家，[[中国工程院|中国工程院]]院士，现任[[中国疾病预防控制中心|中国疾病预防控制中心]]病毒病预防控制所院士实验室主任，传染病国家重大专项技术总师，曾任中国工程院副院长。<ref  name=":0">{{Cite web|url=http://www.cae.cn/cae/html/main/colys/93593570.html|title=中国工程院-侯云德|accessdate=2018-01-11|author=|date=|work=www.cae.cn|publisher=}}</ref>

侯云德被称为中国[[干扰素|干扰素]]之父。<ref>{{Cite web|url=http://tv.cctv.com/2018/01/08/VIDEmhwV5ea4uZ7SZh8W0xv4180108.shtml|title=《大家》 20180108 2017年度国家最高科学技术奖获得者_CCTV节目官网-CCTV-10_央视网(cctv.com)|accessdate=2018-01-11|author=|date=|last=|work=tv.cctv.com|publisher=}}</ref>

== 履历<ref name=":0" /> ==
* 1955年3月，[[同济医学院|同济医学院]]医本科毕业。
* 1962年，就读于原苏联莫斯科苏联医学科学院病毒学研究所，同时获得副博士和医学博士学位。
* 1994年，当选[[中国工程院|中国工程院]][[中国工程院院士|院士]]。
* 2018年1月，获国家最高科学技术奖<ref>{{cite web |url = http://www.xinhuanet.com/tech/2018-01/09/c_1122231183.htm |title = 道固远，笃行可至——病毒斗士侯云德院士荣获国家最高科学技术奖_新华网 |accessdate=2018-01-11 |website = www.xinhuanet.com }}</ref>。
