'''侯马市'''是[[中华人民共和国|中华人民共和国]][[山西省|山西省]]的一个[[县级市|县级市]]，由[[临汾市|临汾市]]代管。位于山西省西南部，面积221平方公里，人口约24万。
[[曲沃县|曲沃县]]将于与[[侯马市|侯马市]]将同城，合并后称[[侯马市|侯马市]]，驻地为原曲沃县境内。 
== 历史 ==
前585年，[[晋景公|晋景公]]以新田“土厚水深，居之不疾，有汾、浍以流其恶，且民从教，十世之利”，将晋国都城迁至[[新田乡_(侯马市)|新田]]。至前376年，韩、赵、魏三分其地，静公废为庶人，晋国绝祀，共历13公，209年。[[明|明]]洪武八年（1375年），绛州金台驿迁此，设侯马驿。

1958年成立侯马市，1963年撤销；1971年8月再次建市。

== 交通 ==
侯馬市是山西南部重要交通中心，[[同蒲铁路|同蒲铁路]]、[[侯西鐵路|侯西鐵路]]、[[侯月铁路|侯月铁路]]在此交會，是山西、河南、陝西之間來往的重要樞鈕。
*铁路：大西高铁(大同---西安）[[南同蒲铁路|南同蒲铁路]]（[[大同市|大同]]－[[风陵渡|风陵渡]]）、[[侯西鐵路|侯西鐵路]]（侯马－[[西安|西安]]）、[[侯月铁路|侯月铁路]]（侯马－[[月山|月山]]）[[侯马站|侯马站]]
*公路：{{中国高速公路名|G55}}、[[侯平高速公路|侯平高速公路]]（S75）、{{中国国道名|108}}、{{中国省道名|235|P=山西}}、{{中国省道名|236|P=山西}}

== 名胜 ==
*[[彭真|彭真]]故居
*[[晋国|晋国]]古都博物馆：馆内展示了在侯马出土的晋国及各朝代珍贵文物五百余件。其中部分馆藏文物“空首布币”、“晋侯鼎”等是世界罕见的珍品。
*庙寝遗址公园：是晋国晚期宗庙建筑群遗址。公园绿地间点缀着侯马出土的晋国文物图案。是山西省唯一以保护地下文物为主的遗址公园。
*铸铜遗址公园：位于牛村古城东南（今平阳机械厂厂区、生活区及周围），是侯马晋国遗址的重要组成部分。
*金代董氏墓：[[董玘坚|董玘坚]]、[[董明|董明]]的仿木结构墓地。
