==政策立場==
===勞動政策===
*侯友宜於2018年8月1日透過行動政見發表勞動政策<ref>{{cite web |url=https://tw.news.yahoo.com/%E5%8E%9A%E5%8F%8B%E8%AA%BC-%E8%A1%8C%E5%8B%95%E6%94%BF%E8%A6%8B%E7%99%BC%E8%A1%A8-%E4%BE%AF%E5%8F%8B%E5%AE%9C3%E5%8B%9E%E5%8B%95%E6%94%BF%E7%AD%96%E4%BF%9D%E9%9A%9C%E5%8B%9E%E5%B7%A5-072512847.html |title=「厚友誼」行動政見發表！侯友宜3勞動政策保障勞工 |date=2018-08-01 |accessdate=2018-08-02 |archive-url=https://web.archive.org/web/20180803074517/https://tw.news.yahoo.com/%E5%8E%9A%E5%8F%8B%E8%AA%BC-%E8%A1%8C%E5%8B%95%E6%94%BF%E8%A6%8B%E7%99%BC%E8%A1%A8-%E4%BE%AF%E5%8F%8B%E5%AE%9C3%E5%8B%9E%E5%8B%95%E6%94%BF%E7%AD%96%E4%BF%9D%E9%9A%9C%E5%8B%9E%E5%B7%A5-072512847.html |archive-date=2018-08-03 |dead-url=no }}</ref>。
*加強勞動檢查：加強勞動檢查，推動勞資政多方對話，政府必須扮演主導角色，促進勞資溝通。
*推動零工經濟協助方案：提供就業市場與產業現況、勞動法令等相關資訊，開設免費訓練專班，受訓期間補貼交通與食宿，為非典型工作朋友創造更好的就業環境。
*擴大勞工參與式預算：邀請青年勞工以公民審議方式討論部分預算之執行方式，共同決定如何使用並提供施政方向。

===社福政策===
*侯友宜認為未來除了要設立更多公托中心，也想利用閒置教室，以及公益回饋空間，或是活動中心市場，把質量都帶出來，更強調應該要跟數位結合，讓托育的方式更加智慧，也要讓偏鄉地區試辦小型公托，挹注托育資源<ref>{{cite web |url=https://tw.news.yahoo.com/%E9%97%9C%E5%BF%83%E5%85%AC%E5%85%B1%E6%89%98%E8%82%B2-%E4%BE%AF%E5%8F%8B%E5%AE%9C-%E7%82%BA%E4%B8%8B-%E4%BB%A3%E7%B9%BC%E7%BA%8C%E6%89%93%E6%8B%9A-061214773.html |title=關心公共托育 侯友宜：為下一代繼續打拚 |date=2018-07-24 |accessdate=2018-07-24 |archive-url=https://web.archive.org/web/20180727115348/https://tw.news.yahoo.com/%E9%97%9C%E5%BF%83%E5%85%AC%E5%85%B1%E6%89%98%E8%82%B2-%E4%BE%AF%E5%8F%8B%E5%AE%9C-%E7%82%BA%E4%B8%8B-%E4%BB%A3%E7%B9%BC%E7%BA%8C%E6%89%93%E6%8B%9A-061214773.html |archive-date=2018-07-27 |dead-url=no }}</ref>。
*在企業自辦托育方面，侯認為目前補助企業新設兩百萬元的誘因不夠，他主張上修補助到四百萬元。另外侯也主張政府提供獎勵誘因，鼓勵公寓大廈釋出空間，好強化「社區化托育普及化」<ref>{{cite web |url=http://www.chinatimes.com/realtimenews/20180712004510-260407 |title=侯友宜：4年公托倍增 善用app智慧托育 |date=2018-07-12 |accessdate=2018-07-12 |archive-url=https://web.archive.org/web/20180712185829/http://www.chinatimes.com/realtimenews/20180712004510-260407 |archive-date=2018-07-12 |dead-url=no }}</ref>。
*侯友宜談[[幸福保衛站|幸福保衛站]]是他在副市長任內最有感的政策。透過4大超商提供需要幫助的孩子，也找出更多高風險家庭弱勢學童，成為社會關懷的一種新型態<ref>{{cite web |url=https://udn.com/news/story/6656/3210418 |title=侯友宜談幸福保衛站很自豪 曾找回失聯已久的少年 |date=2018-06-21 |accessdate=2018-06-21 |archive-url=https://web.archive.org/web/20180624033955/https://udn.com/news/story/6656/3210418 |archive-date=2018-06-24 |dead-url=no }}</ref>。
