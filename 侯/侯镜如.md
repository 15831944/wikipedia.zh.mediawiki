'''侯镜如'''（{{bd|1902年|10月17日|1994年|10月25日|catIdx=H侯}}），号心朗，[[河南|河南]][[永城|永城]]人。

== 早年經歷 ==
1902年10月17日出生，1919年至1923年就学于[[河南留学欧美预备学校|河南留学欧美预备学校]]，1924年5月考入[[黄埔军校|黄埔军校]]，成为第一期学员。1925年10月第二次东征期间加入[[中国共产党|中国共产党]]。

1927年8月1日，侯镜如时在[[贺龙|贺龙]]的第20军任教导员团团长，随部队参加了[[南昌起义|南昌起义]]。

== 中年經歷 ==
1935年4月，[[国民政府|国民政府]]授予侯镜如[[陆军|陆军]][[少将|少将]][[军衔|军衔]]，担任[[国民革命军第三十军|国民革命军第三十军]]第三十师参谋长，后又调任第三十师第八十九旅旅长。1937年[[抗日战争|抗日战争]]爆发，因其在[[忻口战役|忻口战役]]的出色表现，升任为[[国民革命军第九十一军|国民革命军第九十一军]][[参谋长|参谋长]]。1938年初，调任[[国民革命军第九十二军|国民革命军第九十二军]]第二十一师师长，先后参加了[[徐州会战|徐州会战]]、[[长沙会战|长沙会战]]等战役。1943年，升任[[国民革命军第九十二军|国民革命军第九十二军]]军长，领[[中将|中将]]军衔，并率部参加了[[湘西会战|湘西会战]]。

1948年，[[国共内战|国共内战]][[辽沈战役|辽沈战役]]时期，侯镜如担任第十七兵团司令负责救援[[锦州|锦州]]国军。行动开始前，侯镜如已经秘密与中共联络，宣称援救锦州打进去也出不来，故意放慢动作，推迟11天才率领部队达到<ref name="刘统">{{cite book |author = 刘统 |title = 中国革命战争纪实·解放战争·东北卷 |date = 2007-7 |location = 北京 |publisher = 人民出版社 |ISBN = 978-7-01-004261-9|page=580}}</ref><ref>{{cite web|url=http://paper.people.com.cn/hqrw/html/2013-02/06/content_1205701.htm?div=-1 |title= 国民党中将侯镜如之子揭秘：父亲不算真正的“卧底”|author=陈晓燕、刘畅| publisher=环球人物|date=2013-2-4|accessdate=2018-4-7 }}</ref>。

1949年，侯镜如到[[香港|香港]]去探望[[母亲|母亲]]，一度滞留香港。1952年7月，返回[[中国大陆|中国大陆]]，担任[[中央人民政府政务院|中央人民政府政务院]]参事。1955年5月，侯镜如加入了[[中国国民党革命委员会|中国国民党革命委员会]]。1960年8月，补选为民革中央常务委员。

== 晚年經歷 ==
1981年，在民革五届二中全会上，被选为中央副主席。1988年2月被选为[[黄埔军校同学会|黄埔军校同学会]]会长，9月被选为[[中国和平统一促进会|中国和平统一促进会]]会长。1989年3月增选为[[全国政协副主席|全国政协副主席]]。1992年的民革第八届全国代表大会上，被选为中央名誉主席。
