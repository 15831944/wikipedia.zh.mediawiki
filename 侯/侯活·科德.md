'''侯活·科德'''（{{lang-en|'''Howard Korder'''}}）是美國編劇和劇作家。他是1988年[[成年禮|成年禮]]舞台劇《''Boy's Life''》的作者，獲得提名一項{{link-en|普立茲獎戲劇獎|Pulitzer Prize for Drama}}。<ref>{{Cite web |url = http://www.pulitzer.org/prize-winners-by-year/1988 |title = 1988 Pulitzer Prizes |publisher = The Pulitzer Prizes |language = en}}</ref>他的戲劇《''Search and Destroy''》在1995年被{{link-en|狙擊手 (電影)|Search and Destroy (film)|改編成電影}}。<ref>{{Cite web |url = https://www.ibdb.com/broadway-production/search-and-destroy-4664 |title = Search and Destroy |publisher = Internet Broadway Database |language = en}}</ref>他曾寫過《{{link-en|蘭德的激情|The Passion of Ayn Rand (film)}}》和《{{link-en|合法侵入|Lakeview Terrace}}》的劇本。他也是《[[酒私風雲|酒私風雲]]》的編劇之一。
