'''侯为贵'''（{{bd|1942年|}}），[[陕西省|陕西]][[西安市|西安]]人，毕业于[[南昌大学|南昌大学]]，[[中兴通讯|中兴通讯]]公司创始人<ref name="赵凡禹,燕君2017">{{cite book|author=赵凡禹,燕君|title=任正非正传|url=https://books.google.com/books?id=KfgBDgAAQBAJ&pg=PT116|date=28 January 2017|publisher=崧博出版事業有限公司|page=116|id=GGKEY:KTR2W9T2F09}}</ref>。

== 个人生平 ==
1942年，侯为贵出生于陕西省西安市；20世纪80年代初，被派往[[美国|美国]]负责技术引进。其后到航天691厂任职。1985年，侯为贵被派往深圳，创办中兴通讯前身，中港合资企业[[中兴通讯|中兴半导体公司]]，并担任该公司[[总经理|总经理]]。1993年，组建中兴新通讯设备有限公司。1997年，侯为贵成立了中兴通讯股份有限公司，并担任公司副董事长和总经理。2004年1月,侯为贵担任中兴通讯股份有限公司董事长，中兴新通讯设备有限公司副董事长，中兴先通设备有限公司董事长。同年，中兴通讯在[[香港交易所|香港交易所]]上市。2013年4月3日，担任中兴通讯第六任董事长。2016年1月15日，侯为贵卸任中兴通讯董事长。2018年，侯为贵重新出任中兴董事长<ref>{{cite web |author1=沙行 |title=77岁侯为贵：中兴危局 老将出马 |url=http://tech.sina.com.cn/csj/2018-04-23/doc-ifzqvvrz8016896.shtml |website=新浪 |accessdate=2019-02-20}}</ref><ref>{{cite book|title=从中国制造到中国创造|url=https://books.google.com/books?id=Fr8wAQAAIAAJ|year=2007|publisher=航空工业出版社|isbn=978-7-80183-995-4|pages=172–174}}</ref><ref name="ByrneRhonda))2016">{{cite book|author1=Rhonda Byrne|title=力量|url=https://books.google.com/books?id=mOR5CwAAQBAJ&pg=PA116|year=2016|publisher=右灰文化傳播有限公司|page=116|id=GGKEY:Y8AX0QQJ466}}</ref>。

== 个人荣誉 ==
侯为贵曾获得“CCTV中国经济年度人物评选”、“2004年CCTV中国经济年度人物”、“中国信息产业十大年度经济人物”、“中国十大科技领袖”等荣誉。
