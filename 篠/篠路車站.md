'''篠路站'''（{{jpn|j=篠路駅|hg=しのろえき|rm=Shinoro eki|f=y}}）是位於[[北海道|北海道]][[札幌市|札幌市]][[北區_(札幌市)|北區]]篠路4-{条}-7丁目2，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[札沼線|札沼線]]（學園都市線）的[[鐵路車站|鐵路車站]]。車站編號為'''G08'''。電報略號為'''{{lang|ja|ノロ}}'''。

== 概要 ==
1934年在當時{{Link-ja|篠路村|篠路村}}的核心地帶開始營運。營運初期，附近主要是農業地區，主要栽種[[蕎麥|蕎麥]]、[[洋蔥|洋蔥]]及飼養[[家牛|牛]][[馬|馬]]等，亦處理貨物運輸。1957年至1974年期間，亦負責運輸於生振地區的茨戶油田生產的[[石油|石油]]。1970年代後期，車站周圍開始開發成住宅地區。

== 歷史 ==
* 1934年（昭和9年）11月20日 - [[鐵道省|國有鐵道]]'''札沼南線'''[[桑園車站|桑園車站]]至[[石狩當別車站|石狩當別車站]]路段開始營運，本車站同時開始營運。
* 1935年（昭和10年）10月3日 - 隨著石狩當別站至[[浦臼車站|浦臼車站]]路段開始營運，札沼南線與'''札沼北線'''合併並改稱為'''札沼線'''<ref name=school>{{cite web|url=http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |language=ja |title=札沼線（学園都市線）の電化について |publisher=JR北海道 | date=2009-09-09 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20091006234745/http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |archive-date=2009-10-06 }}</ref>。
* 1949年（昭和24年）6月1日 - 本路線由[[日本國有鐵道|日本國有鐵道]]（國鐵）繼承。
* 1979年（昭和54年）2月1日 - 停止貨物裝卸服務。
* 1984年（昭和59年）2月1日 - 停止行李裝卸服務。
* 1987年（昭和62年）4月1日 - 由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理。

* 1990年（平成2年）11月4日 - 站房改建。
* 1991年（平成3年）3月16日 - 札沼線的暱稱設定為「'''學園都市線'''」<ref name=school/><ref name=school2>{{cite web |url=http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |language=ja |title=札沼線（学園都市線）の電化開業時期について |publisher=JR北海道 |date=2011-10-03 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20111205160350/http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |archive-date=2011-12-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20111205160350/http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |archivedate=2011-12-05 }}</ref>。
* 1995年（平成7年）3月16日 - 札沼線（學園都市線）[[太平車站_(日本)|太平車站]]至本車站路段[[複線鐵路|複線化]]，於本車站附近增設上行線。
* 1997年（平成9年）3月22日 - 札沼線（學園都市線）本車站至[[愛之里教育大車站|愛之里教育大車站]]路段複線化<ref name=school2/>，本車站至[[拓北車站|拓北車站]]增設上行線。
* 2000年（平成12年） - 札沼線[[桑園車站|桑園車站]]至[[石狩月形車站|石狩月形車站]]路段引入自動進路制御裝置（PRC）。
* 2006年（平成18年）1月 - 引入自動列車導向裝置。並設置按方向的導向用液晶顯示屏。
* 2007年（平成19年）10月1日 - 設定車站編號（'''G08'''）<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |language=ja |title=駅番号表示（駅ナンバリング）を実施します |publisher=JR北海道 |date=2007-09-12 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20070930015220/http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |archive-date=2007-09-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20070930015220/http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |archivedate=2007-09-30 }}</ref>。
* 2008年（平成20年）10月25日 - 開始使用[[儲值車票|智能車票]]「[[Kitaca|Kitaca]]」<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |language=ja |title=Kitacaサービス開始日決定について |publisher=JR北海道 |date=2008-09-10 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20080913103748/http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |archive-date=2008-09-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20080913103748/http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |archivedate=2008-09-13 }}</ref>。
* 2009年（平成21年）12月10日 - 札沼線（學園都市線）桑園站至[[北海道醫療大學車站|北海道醫療大學車站]]開始進行[[電氣化鐵路|電氣化]]<ref>{{cite web|url=http://blog.hokkaido-np.co.jp/tamatetsu/oldentries/archives/2012/05/post_105.html |language=ja |title=学園都市線　電化カウントダウン |publisher=北海道新聞 | date=2012-05-30 |accessdate=2017-08-25 }}</ref>。
* 2012年（平成24年）6月1日 - 札沼線（學園都市線）桑園站至北海道醫療大學站正式電氣化（[[交流電|交流電]]20,000[[伏特|V]]、50[[赫兹|Hz]]）<ref>{{cite web|url=http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |language=ja |title=学園都市線電化開業に伴う電車の投入（第一次）について |publisher=JR北海道 | date=2012-03-14 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20120925203437/http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |archive-date=2012-09-25 }}</ref><ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |language=ja |title=平成24年10月ダイヤ改正について |publisher=JR北海道 |date=2012-08-03 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20120925203427/http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |archive-date=2012-09-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20120925203427/http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |archivedate=2012-09-25 }}</ref>。
* 2014年（平成26年）3月4日 - 由於車站職員錯誤理解末班車的時間，比預定時間中的將車站出入口鎖上，令部份乘客未能乘坐末班車而需要使用計程車到達目的地<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2014/140305-1.pdf |language=ja |title=最終列車にご乗車いただけなかった事象について |publisher=JR北海道 |date=2014-03-05 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20150109124115/http://www.jrhokkaido.co.jp/press/2014/140305-1.pdf |archive-date=2015-01-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20150109124115/http://www.jrhokkaido.co.jp/press/2014/140305-1.pdf |archivedate=2015-01-09 }}</ref>。
* 2017年（平成29年）2月10日 - 東面入口設置[[驗票閘門|自動閘機]]。

== 車站構造 ==
現時為[[側式月台|側式月台]]2面2線的地面車站。2號線旁設有供[[冬季服務車輛|除雪車]]的留置線。此外，札沼線（學園都市線）進行電氣化期間，於本車站內設置變電裝置。

由[[桑園車站|桑園車站]]管理的業務委託車站（北海道JR Servicenet），東面出口設有[[綠窗口|綠窗口]]（營業時間5時40分至23時00分）、[[自動售票機|自動售票機]]及[[驗票閘門|自動閘機]]。而西面入口則設有簡易售票機、簡易自動閘機及[[Kitaca|簡易Kitaca閘機]]。

== 車站周邊 ==
