《'''忍者刺客'''》（{{lang-en|'''Ninja Assassin'''}}）2009年11月25日上映于[[美国|美国]]的[[动作片|动作片]]。

== 剧情 ==
忍者家族欧祖努是一个恐怖组织，专门刺杀各国政要和政客，危害到世界的和平，这个组织位于[[日本|日本]]。雷藏是一个孤儿，从小就到了欧祖努这个组织里，学到了一身武艺，精通[[忍者|忍者]]中的各种技艺，他从小就被视为组织中不可多得的人才来培养，但是忍者宗主的残暴和暴虐使得他从小就被蒙上了阴影。他的青年时期，忍者家族的女忍者衣澄爱上了他，最后为了自由而逃走，被抓回来之后，在众目睽睽之下被同门师兄弟一剑穿心而死。雷藏逃离了组织，去了[[德国|德国]]隐居起来，但是忍者家族不停地派出忍者追杀他，正好此时，欧洲警察密卡·克莱蒂和他的同伴在追踪欧洲政要被暗杀的谜团，两人相互认识，成为信任的好友。最后，雷藏亲手杀了忍者家族的宗主，获得了自由。

== 花絮 ==
电影拍摄期间，郑智薰借着对已故母亲的怀念苦苦熬过了艰苦的训练生活。<ref>{{cite web |url=http://ent.qq.com/a/20091111/000165.htm |title=Rain演《忍者刺客》靠思念亡母熬过地狱式特训 |publisher=腾讯网 |accessdate= 2009年11月11日 |work= |language =}}</ref> 为了拍摄该片，他也曾一年没有喝酒。<ref>{{cite web |url=http://ent.sina.com.cn/m/f/2009-11-17/00402771962.shtml |title=Rain为演《忍者刺客》“禁酒”一年 |publisher=新浪网 |accessdate=2009年11月17日 |work= |language =}}</ref>

== 票房 ==
郑智薰的电影在中囯[[山东|山东]]获得了满满的票房，被聘为“山东省形象大使”。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2009-12-21/16212817881.shtml |title=《忍者刺客》大卖 主演Rain将在山东迎接新年 |publisher= 新浪网|accessdate= 2009年12月21日 |work= |language =}}</ref>

电影在[[香港|香港]]的票房排行第三名。<ref>{{cite web |url= http://ent.163.com/09/1207/15/5PUMAB7800033BU3.html|title=《2012》香港票房三连冠 《忍者刺客》冲至第3 |publisher=新浪网 |accessdate= 2009-12-07|work= |language =}}</ref>

== 社会影响 ==
忍者刺客是一部沉默暗杀的片子，但是剧中声音过于喧嚣，显得吵闹。
