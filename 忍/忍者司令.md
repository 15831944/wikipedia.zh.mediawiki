== 劇情 ==
1968年，在[[越戰|越戰]]期間，由約翰·亨特領導的[[美國陸軍特種部隊|美國陸軍特種部隊]]「蜥蜴老菸槍」（{{lang|fr|Lizard Smokers}}）遭到了一群忍者伏擊。在混亂中，霍普金斯在伏擊中失去了他的右臂，奧斯卡·科瓦爾斯基被數隻[[迅猛龍|迅猛龍]]圍攻，而約翰則被[[越共|越共]]捕獲。在被囚禁的過程中，約翰從俘虜他的尹上校那學習了[[忍術|忍術]]。約翰的訓練完成後，尹還他自由，同時向約翰警告，要小心一個背叛他的教誨的紅色忍者。當美軍的汽油彈從空中投下後，村莊被摧毀，而尹也為此死去。

1986年，隱居於[[加拿大|加拿大]]的約翰接受了霍普金斯的拜訪，並獲悉約翰的前妻洛莉已遭到謀殺，而他的女兒珍妮被一名為非法軍火商工作的神秘人士綁架。約翰與霍普金斯到[[洛杉磯|洛杉磯]]與[[FBI|FBI]]合作，假扮成[[美國空軍|美國空軍]]與那名非法軍火商作交易。但當約翰發現對方是前[[蘇聯陸軍|蘇聯陸軍]]上校金斯基後，眾人遭到金斯基的忍者人馬伏擊。在金斯基逃脫前，霍普金斯被紅色忍者所殺。

約翰根據金斯基留下的訊息而前往[[中美洲|中美洲]]的瓦爾佛德（{{lang|fr|Val Verde}}）國，並打算在金斯基的別墅拯救他的女兒，但發現他的老戰友科瓦爾斯基還活著並已成為一名生化人。珍妮偷偷地在金斯基的褲子裡放了一枚手榴彈而殺死了他，但科瓦爾斯基突然和她一起在原定消失。約翰透過決鬥擊敗了紅色忍者，並發現對方面具下的真面目是洛莉。洛莉解釋說，自己的另一個未來版本就是先前已被殺害的版本，且自越戰以來，金斯基為忍者提供了[[時間旅行|時間旅行]]的技術。她告訴約翰，珍妮已被送到1998年，金斯基正在訓練他的軍隊建立一個新的世界秩序。在洛莉要約翰拯救珍妮後，他將洛莉斬首並{{le|蘇生 (時空英豪)|Quickening (Highlander)|使用了她最後的能量}}前往1998年的後末日世界。

在他到達時，約翰與成年的珍妮團聚。她透露，自己重新編寫了科瓦爾斯基的程式，而使他成為夥伴。約翰還發現霍普金斯還活著並駕駛著一輛[[龐蒂克火鳥|龐蒂克火鳥Trans Am]]外型的時間機器。

在片尾片段中，約翰、珍妮、霍普金斯與科瓦爾斯基試圖回到1986年，但最終被困在金斯基編寫及製作的電子遊戲中。他們必須找到逃離遊戲世界的方法，以阻止金斯基用他的軍隊征服世界。

== 製作 ==
導演班傑明·康斯是一位法國電影人，他在十多年間都在執導電子遊戲預告片和獨立短片。2016年，他撰寫了《忍者司令》的劇本並花了5000美元與他的朋友一起製作和拍攝鏡頭<ref name="Action Elite"/><ref name="Screen Anarchy"/>。《忍者司令》大多在法國[[蒙佩利爾|蒙佩利爾]]拍攝。與使用特效合成背景的《[[功夫厲|功夫厲]]》（2015年）不同的是，《忍者司令》使用了實境拍攝及真實的特技、化妝<ref name="Kickstarter" /><ref name="Action Elite" /><ref name="Screen Anarchy" />。

== 發行 ==
《忍者司令》於2018年12月21日在[[YouTube|YouTube]]上發行，包括英語、法語和波蘭語版本<ref name="Action Elite" /><ref name="Screen Anarchy" />。該片於2018年12月31日推出捷克語版本<ref>{{cite web |url=https://www.youtube.com/watch?v=AyqzXZChxVM |title=''Commando Ninja'' (Czech Dub) |first=Benjamin |last=Combes |publisher=[[YouTube|YouTube]] |date=2018-12-31 |accessdate=2018-12-31}}</ref>。此外，德國玩具製造商{{lang|de|Goodleg Toys}}宣布，他們將於2019年2月推出《忍者司令》的[[可動人偶|可動人偶]]。該可動人偶的{{le|工業模型|Kitbashing|模型}}風格來自於1980年代的各種玩具系列，如{{le|大英雄 (玩具)|G.I. Joe|大英雄}}、{{le|太空超人系列|Masters of the Universe|太空超人}}和{{le|WWF孩之寶可動人偶|WWF Hasbro action figures|WWF}}。其預購在電影發行後開始<ref>{{cite web |url=http://www.goodlegtoys.de/post/181265901295/official-commando-ninja-x-goodleg-toys-figures-to |title=Official Commando Ninja x Goodleg Toys figures to be released with the movie on 21/12/18! |publisher=Goodleg Toys |date=2018-12-20 |accessdate=2018-12-26}}</ref>。
