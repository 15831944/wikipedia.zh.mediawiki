'''应公长老寿塔'''，位于[[北京市|北京市]][[房山区|房山区]][[韩村河镇|韩村河镇]]天开村北，是[[汉传佛教|汉传佛教]]高僧应公长老的墓塔，[[北京市文物保护单位|北京市文物保护单位]]。

==简介==
应公俗姓赵，名谱应，是[[辽朝|辽朝]][[天开寺_(北京)|天开寺]]的首任住持，是[[元朝|元朝]][[至元|至元]]年间重修天开寺的高僧，在应公住持天开寺期间，[[元世祖|元世祖]]忽必烈颁圣旨护持天开寺<ref name=qingshan>{{Cite web |url =http://news.163.com/16/0923/15/C1LKBTP300014AED.html  |title =青山绿水花深处  |author =  |date =2016-09-23  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref name=guta>{{Cite web |url = http://www.fjnet.com/jjdt/jjdtnr/200410/t20041008_1911.htm |title =古塔巡礼：游房山古塔群 寻古人“花边事”  |author =  |date = 2004-10-08 |publisher = 佛教在线 |language =  |accessdate =  |quote =  }}</ref>。应公长老寿塔，俗称“和尚塔”，建于[[元朝|元朝]][[大德|大德]]五年二月（1301年）。坐北朝南，通高12米，为六角形五级檐式砖塔，[[须弥座|须弥座]]上砌有三层莲花。柱形塔身正面开有拱门，门楣上端嵌有铭文“应公长老寿塔”。塔身东南面、西南面各嵌铭一方。正面设有假门，其他各面设有假窗。假窗上部设有挂落。塔身以上是五级密檐，各层檐下都有仿木结构的砖制[[斗拱|斗拱]]<ref>{{Cite web |url =  http://www.bj.chinanews.com/news/2005/2005-12-29/1/8339.html|title = 北京古塔 |author =  |date = 2005-12-29 |publisher = 北京网 |language =  |accessdate =  |quote =  }}</ref>。

应公长老寿塔是应公的埋骨处。应公长老寿塔的所在地，原是天开寺的塔院，葬有天开寺历代高僧，僧塔林立。[[中华人民共和国|中华人民共和国]]时期，1959年修建天开水库，应公长老寿塔位于天开水库东侧临水库大坝处。后来天开水库被废弃而干涸。2012年，在天开水库原址上完成了“天开花海”一期工程，2015年全部工程建成。应公长老寿塔成为“天开花海”中的一个景点<ref name=qingshan/><ref name=guta/>。

2004年4月到7月，房山区进行了[[周吉祥塔|周吉祥塔]]、[[玉皇塔_(北京)|玉皇塔]]、[[照塔|照塔]]和应公长老寿塔四座古塔与[[万佛堂、孔水洞石刻及塔|万佛堂]]修缮工程<ref>{{cite web|title=房山四塔、万佛堂等修缮工程正在进行|url=http://www.huaxia.com/zhwh/whxx/2004/06/84363.html|publisher=华夏经纬网|date= 2004-06-17|accessdate=2017-08-21}}</ref>。
