'''应天府'''是[[明朝|明朝]]设置的[[府_(行政区划)|府]]，为明代初期[[首都|首都]]，後改[[留都|留都]]。即今[[南京市|南京市]]。

== 历史 ==
[[元朝|元朝]]时，为[[集庆路|集庆路]]，属[[江浙行省|江浙行省]]。

[[至正|至正]]十六年（1356年、太祖丙申年），[[朱元璋|朱元璋]]攻占集庆路，三月改集庆路为应天府。至正二十四年（1364年）即吳王位，王都應天府。元至正二十六年（1366年）拓建[[建康城|建康城]]<ref>《明太祖实录》“丙午八月庚戌朔，拓建康城”</ref>{{NoteTag|南宋建康知府史正志、馬光祖前後浚濠增筑建康府城。}}。明洪武元年（1368年）八月，朱元璋稱帝，仍以建康城為京城，二年九月始建[[南京城|新京城]]，六年八月建成。都城内的[[宫城|宫城]]（即[[明故宫|明故宫]]）称[[紫禁城|紫禁城]]。洪武十一年，称'''京师'''。

[[建文|建文]]元年（1399年）燕王[[朱棣|朱棣]]反叛他的姪子[[建文帝|建文帝]]，發動了[[靖難之變|靖難之變]]，建文四年（1402年）燕兵破應天府城，朱棣[[登基|登基]]，是為[[明成祖|明成祖]]，明年改元永樂。

[[永乐_(明朝)|永樂]]元年（1403年），朱棣決定北平府改為[[順天府|順天府]]（今[[北京|北京]]），立為[[陪都|陪都]]北京{{NoteTag|相當于[[明中都遗址|中都]][[鳳陽府|鳳陽府]]及後來的[[承天府|承天府]]。}}。

永樂十九年（1421年）[[明成祖|明成祖]]正式迁都，史稱[[永乐迁都|永乐迁都]]，南京應天府成为[[陪都|陪都]]。

[[南明|南明]][[弘光|弘光]]元年（1645年），清軍佔領江南，改應天府為[[江寧府|江寧府]]。

== 官制 ==
洪武初年，應天府設[[知府|知府]]、[[同知|同知]]、[[通判|通判]]及[[經歷|經歷]]、[[知事|知事]]、[[照磨|照磨]]等官。後添設[[治中|治中]]、[[推官|推官]]。洪武三年、陞格為[[正三品|正三品]]衙門。賜銀印。同時，將知府改為[[府尹|府尹]]。同知更為[[府丞|府丞]]。洪武二十七年、添設[[檢校|檢校]]。

== 行政区划 ==
应天府下领八县。

== 人口 ==
[[洪武|洪武]]二十六年（1393年）编户一十六万三千九百一十五，口一百十九万三千六百二十。[[弘治_(明朝)|弘治]]四年（1491年），户一十四万四千三百六十八，口七十一万一千三。[[万历|万历]]六年（1578年），户一十四万三千五百九十七，口七十九万五百一十三。
