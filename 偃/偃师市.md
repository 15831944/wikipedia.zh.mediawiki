'''偃师市'''位于东经112°6′15″～112°58′24″和北纬34°27′30″～34°50′00″之间，为[[中華人民共和國|中国]][[河南省|河南省]]下辖[[县级市|县级市]]。

总面积668.58平方公里，人口61.6万（2013年）；辖11镇（城关、[[山化镇|山化]]、[[首阳山镇|首阳山]]、[[邙岭乡|邙岭]]、[[岳滩镇|岳滩]]、[[翟镇|翟镇]]、[[顾县镇_(偃师市)|顾县]]、[[缑氏|缑氏]]、[[府店镇|府店]]、[[大口乡|大口]]、[[高龙|高龙]]）、1个工业区（偃师市工业区）、210个行政村、32个社区；市区面积16平方公里，市区人口14万。

偃师市前身为偃师县。战国末年，秦灭周後设偃师县，先後隶属三川郡、河南郡、河南府、河洛道、豫西行政长官公署、河南省第十行政区督察专员公署、太岳五专署、豫西一专署、洛阳专区、[[洛阳地区|洛阳地区]]、[[洛阳市|洛阳市]]。1993年底，经国务院批准撤县建市；1994年4月8日，偃师市正式挂牌，由河南省委托洛阳市代管。

== 地理交通 ==

位于河南省西部，东邻[[巩义市|巩义市]]，东南部与[[登封市|登封市]]为邻，西接洛阳市[[洛龙区|洛龙区]]，西北接壤[[孟津县|孟津县]]，南部和[[伊川县|伊川县]]以万安山为界，北隔[[黄河|黄河]]与[[孟州市|孟州市]]相望。距[[世界文化遗产|世界文化遗产]][[龙门石窟|龙门石窟]]、中岳[[嵩山|嵩山]]，[[佛教|佛教]]圣地[[白马寺|白马寺]]、[[少林寺|少林寺]]，[[关公|关公]]埋首之地[[关林|关林]]均近在咫尺，上述几地距市境都只有几公里。

[[310国道|G310]]线、[[207国道|G207]]线在偃师市交会；[[陇海铁路|陇海铁路]]在偃师境内设偃师、首阳山两个火车站；G30[[连霍高速公路|连霍高速公路]]经过偃师北部，并设出入口；[[郑西客运专线|郑西客运专线]]从偃师市中部地区穿过。

== 行政区划 ==

2004年，偃师市辖13个镇、4个乡（城关镇、首阳山镇、岳滩镇、翟镇镇、佃庄镇、顾县镇、高龙镇、庞村镇、寇店镇、李村镇、诸葛镇、缑氏镇、府店镇，山化乡、邙岭乡、大口乡、佛光乡），面积948平方公里，人口约82万。

2005年，佛光乡撤销，其行政区域并入府店镇。当年年底，偃师市辖13个镇、3个乡。

2009年，诸葛镇划归洛阳市洛龙区；2010年，李村镇划归洛阳市洛龙区；2011年，庞村镇划归洛阳市洛龙区；2012年，佃庄镇、寇店镇划归洛阳市洛龙区。但在实际操作中，洛龙区并未管辖这5个镇，而是由洛阳市政府的派出机构伊洛工业园区管委会（对外名字为“[[伊滨区|伊滨区]]”，是未经民政部批准的非法区划）管辖这5个镇、106个村、280平方公里土地、26万人口。

2012年，撤销城关镇，成立商城街道、工业区街道；撤销首阳山镇，成立首阳山街道。

2014年，偃师市辖3个街道、9个镇，共210个行政村、32个社区。

== 经济 ==

偃师为较具实力的[[县级行政区|县市]]之一，其经济总量位居河南省各县（市）前列，自1993年以来，多次进入全国百强县（市）行列。

2013年，完成地区[[生产总值|生产总值]]（GDP）354亿元，[[固定资产投资|固定资产投资]]199亿元，[[公共财政预算收入|公共财政预算收入]]12.12亿元，[[社会消费品零售总额|社会消费品零售总额]]116.8亿元，[[城镇居民人均可支配收入|城镇居民人均可支配收入]]22357元，[[农民人均纯收入|农民人均纯收入]]12596元。

在全省108县（市）首次经济发展质量总体评价中，偃师综合评价排名第七位，发展结构排名第六位，发展效益和民生幸福均列第四位。

偃师市民营经济发达，主要产业有[[布鞋|布鞋]]制造（主要分布在城关镇、山化镇）、[[三轮摩托车|三轮摩托车]]制造（主要分布在岳滩镇）、[[针织|针织]]（主要在翟镇镇）、[[钢制办公家具|钢制办公家具]]（主要在庞村镇、李村镇、高龙镇）等。

大型国有及外资企业主要在火力发电行业，现有火电厂3座，分别是[[大唐首阳山电厂|大唐首阳山电厂]]、[[华润电力控股|华润]]热电公司和[[华润电力首阳山有限公司|华润电力首阳山有限公司]]，总装机容量367万[[千瓦|千瓦]]。偃师成为中部地区重要[[能源|能源]]基地，同时，也带来严重[[污染|污染]]问题。

农业方面，种植业以[[小麦|小麦]]、[[玉米|玉米]]为主，兼有[[畜牧|畜牧]]养殖业、[[花卉|花卉]]蔬菜业、葡萄种植（主要在缑氏镇）等，特产有“[[银条|银条]]”（主要在城关镇东寺庄、西寺庄村）。

== 历史文化 ==
[[商朝|商朝]]开国之主[[商汤|汤]]的墓地位于山化镇蔺窑村；商末周初名臣[[伯夷|伯夷]]、[[叔齐|叔齐]]曾在偃师首阳山劝阻周武王伐纣，后不食周粟而死于首阳山；[[周武王|周武王]]东征伐纣凯旋，在此筑城“息偃戎师”而得名，先后有[[夏朝|夏]]、[[商朝|商]]、[[东周|东周]]、[[东汉|东汉]]、[[曹魏|曹魏]]、[[西晋|西晋]]、[[北魏|北魏]]等七个朝代在偃师建都。传说中，还有五帝时代的帝喾高辛氏建都与高庄村。大秦相国[[吕不韦|吕不韦]]墓位于首阳山偃师市第一高中校园内；东汉时，[[班固|班固]]在偃师著《汉书》，[[张衡|张衡]]在此发明[[地动仪|地动仪]]、候风仪，[[蔡伦|蔡伦]]在首阳山镇纸庄村改进[[造纸术|造纸术]]，西晋历代帝王陵在首阳山；唐代“诗圣”杜甫墓在城关镇前杜楼村；北宋史学家[[司马光|司马光]]在诸葛镇独乐园编修《[[资治通鉴|资治通鉴]]》……大批历史名人都在这里成就了英名伟业。
[[File:刘根造像碑1392.jpg|thumb]]
境内有7处全国重点文物保护单位，分别是夏都[[二里头遗址|二里头遗址]]（[[斟鄩|斟鄩]]）、尸乡沟商城遗址（[[西亳商城遗址|西亳商城遗址]]）、[[唐恭陵|唐恭陵]]、滑国故城、升仙太子碑、刘国故城、洛南东汉帝陵。另外，与洛阳、孟津共有[[汉魏洛阳故城|汉魏洛阳故城]]遗址以及[[邙山古墓群|邙山古墓群]]）、25处河南省文物保护单位、44处市县级文物保护单位。市区商城路建有偃师商城博物馆，收藏上古文物精品。

偃师自古人杰地灵，是唐代佛学大师[[玄奘|玄奘]]和北宋名相[[吕蒙正|吕蒙正]]、今中国书法家协会主席张海的故乡。

三位书法大师安葬于偃师市山化镇：东汉末年中国历史上首位书法家[[钟繇|钟繇]]及其子三国时曹魏名将[[钟会|钟会]]安葬于东屯村，唐代书法家[[颜真卿|颜真卿]]陵园位于汤泉村，明清之际书法家王铎埋骨于石家庄村。

偃师还是[[客家|客家先民首次南迁]]出发地和[[丝绸之路|丝绸之路]]东方起点之一。
