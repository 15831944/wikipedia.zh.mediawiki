'''兖州之战'''是汉末军阀[[曹操|曹操]]和[[呂布|呂布]]争夺[[兖州|兖州]]的战役。战事持续了百余日，胜负未分。

== 战役背景 ==
192年，[[青州|青州]][[黄巾军|黄巾军]]余党入侵兖州，占领[[东平|东平]]和[[任城|任城]]。兖州刺史[[刘岱|刘岱]]想进攻他们，被济北相[[鲍信|鲍信]]谏止。刘岱不听鲍信警告，正面迎战黄巾军，最终败死。

此时，曹操的谋士[[陈宫|陈宫]]敦促他夺取兖州作为扩张的根据地，并主动说服[[刘岱|刘岱]]旧部归顺曹操。鲍信素与曹操为友，来到[[东郡|东郡]]请曹操出任兖州刺史。

曹操攻黄巾于[[寿张|寿张]]，但未能取胜。曹操改革军制，严正军令，激励将士。他认识到叛军靠劫掠养活自己，并没有稳定的补给线。因此，他突袭叛军，阻止其劫掠，取得最终胜利，迫使叛军北退。曹操追击叛军并再败叛军于济北。包括10万百姓在内的30万黄巾军投降曹操。曹操改编降卒，组建[[青州兵|青州兵]]，并完善[[屯田制|屯田制]]以使百姓能养活自己和军队。

193年，曹操父[[曹嵩|曹嵩]]被徐州牧[[陶谦|陶谦]]部下[[张闿_(东汉)|张闿]]所杀。曹操为复仇攻打陶谦，扫荡其辖地，在徐州杀死包括百姓在内的数千人。[[陶谦|陶谦]]退守[[郯城|郯城]]，曹操不能攻克，军队补给短缺后只得撤军。

次年，曹操再伐陶谦，攻陷琅琊、东海二郡的很多县。此时，曹操下属[[张邈|张邈]]与陈宫背叛曹操，协助吕布接手曹操的大本营兖州。曹操决定放弃讨伐陶谦，回师复夺兖州。

== 战役 ==
曹操军和吕布军僵持至少一百天，曹军处于劣势。但因饥荒爆发，吕布放弃了兖州刺史之位。曹操围吕布于[[濮阳|濮阳]][[巨野|巨野]]。曹操谋士[[荀彧|荀彧]]和[[程昱|程昱]]守御[[鄄城|鄄城]]、[[范县|范县]]、[[东阿|东阿]]，只有两城防御坚固，于是曹操决定回军。吕布杀至，围鄄城，不能取胜，遂西进屯濮阳，后东进屯[[山阳|山阳]]。

在战斗中，呂布让所部骑兵打头阵，冲击曹操的青州兵。青州兵惊慌奔逃，曹军阵形完全散乱。曹操目睹混乱，看到火光迸发，亲自冲锋向前，但落马且左掌烧伤。因曹军众将未见主公，担心其安危，曹军在进寨前暂停了。曹操竭力振奋部属士气，下令立即准备围城器械，以再次围攻吕布。

张邈跟随呂布，留下弟弟[[广陵|广陵]]太守[[张超_(东平)|张超]]照看在[[雍丘|雍丘]]的家眷。曹操围雍丘数月，破城，杀张超及张氏全家。张邈求助[[袁术|袁术]]被拒后被部卒所杀。同时，蝗灾爆发，饿死多人，还出现了人相食的局面。吕布也用尽了储备、马料和谷物，双方都只得撤军。<ref>《三国志·魏书一·武帝纪第一》</ref><ref>《三国志·魏书七·吕布（张邈）臧洪传第七》</ref>

后曹操围吕布于濮阳，最初支持呂布的濮阳大户田氏转而效忠曹操，使曹军入城。曹操在东门放火以示无返回之意，结果遭到攻击并战败。吕布的一些骑手抓住了曹操却不知其身份。曹操看见一人骑黄马，便诈称此人为曹操，吕布的骑手便撇下他去追骑黄马者。曹操冲出着火的东门，逃离濮阳。<ref>[[袁暐|袁暐]]《[[献帝春秋|献帝春秋]]》</ref>

两年内，曹操收复兖州全境，败呂布于巨野。呂布东逃投靠接替陶谦掌管徐州的[[刘备|刘备]]。

== 后果 ==
兖州之战最终导致了198年曹操刘备联军攻打呂布的[[下邳之战|下邳之战]]。呂布战败并被曹操下令处决。

== 演绎 ==
[[罗贯中|罗贯中]]的历史小说《[[三国演义|三国演义]]》也提及了兖州之战的战况，佔兩個篇幅（〈第十一回〉和〈第十二回〉），描述如下：

=== 曹操初败 ===
在第一场遭遇战中，呂布部将[[臧霸|臧霸]]、[[张辽|张辽]]与曹操部将[[乐进|乐进]]、[[夏侯惇|夏侯惇]]交锋。呂布不耐烦地拿着[[方天画戟|方天画戟]]冲锋上前。夏侯惇和乐进逃走；呂布持续进攻，曹操被迫后撤三四十里扎营。

次日，曹操从四面袭击[[高顺|高顺]]、[[魏续|魏续]]和[[侯成|侯成]]把守的呂布营寨，呂布军伤亡惨重。曹操入寨，遭遇高顺。时已四更，天将破晓，曹操听闻呂布亲自率军从西面杀来援助高顺，就弃寨而走。呂布率高顺、魏续、侯成追击曹操败军。曹操派[[于禁|于禁]]和乐进阻拦，未果。曹操向北进军时，又陷入张辽和臧霸的伏击。曹操派[[吕虔|吕虔]]和[[曹洪|曹洪]]迎敌，也战败了。曹操向西逃，又遭遇呂布部将[[郝萌|郝萌]]、[[曹性|曹性]]、[[成廉|成廉]]、[[宋宪|宋宪]]。曹操危急之际呼救，[[典韦|典韦]]来助。曹操得以逃回本寨。

一次曹操攻呂布的濮阳时，杀入城中，发现中计了。陈宫设下圈套，堵住了曹操的退路。张辽、臧霸率军从东西两面进攻曹操。曹操向北门逃奔，被曹性、郝萌阻截，转向南门，又遇到高顺、侯成。典韦、[[李典|李典]]、乐进在敌军中冲杀寻找主公。曹操与典韦会合后再次冲奔北门，正撞见寻找曹操的呂布。但呂布误认他人为曹操并追杀之，使曹操得以在典韦和[[夏侯渊|夏侯渊]]保护下从北门逃离。

=== 战局逆转 ===
曹操濮阳战败的次日，呂布听闻曹操在五更时分被火烧伤致死。他立即聚集军马经[[马陵|马陵]]进攻敌营。在经过山道时，战鼓响，曹军伏兵四起。呂布死战得脱，回濮阳，坚守不出。

曹操下次攻呂布时，以夏侯惇、夏侯渊、李典、乐进、吕虔、于禁六将为主将。呂布独自出城迎战，遭到围攻，被迫撤回。但濮阳大户田氏背叛吕布，倒向曹操。田氏关闭城门，不让吕布入城，吕布只得逃走。陈宫从东门带吕布家眷逃出，后与吕布会合。曹操复夺濮阳。

吕布在[[定陶|定陶]]寻求张邈、张超庇护。吕布出城哨探曹营，见曹营接近密林，担心有伏兵而撤退。后他决定放火逼出曹军，但惊讶地发现林中没出现一个士兵。吕布听见敌军的战鼓声，寨后杀出一支军队，吕布前去迎战。夏侯惇、夏侯渊、典韦、[[许褚|许褚]]、李典、乐进率曹军攻击吕布。吕布溃败，所部三分之二被杀。吕布部将成廉也被射死。吕布残部回报陈宫，陈宫和高顺聚集吕布所部弃定陶而走。张超自杀，张邈投奔袁术。当吕布后来重聚人马时，兖州东北已被曹操所得。

吕布并未被吓倒，在陈宫建议下想再次效力[[袁绍|袁绍]]。袁绍却派[[颜良|颜良]]率军五万攻吕布。吕布无力与袁绍周旋，逃往徐州投刘备，受到刘备的热情欢迎。
