《'''拜托了☆双子星'''》（{{lang-ja|おねがい☆ツインズ}}）是一部[[日本动画|日本动画]]作品，是《[[拜托了☆老师|拜托了☆老师]]》的续篇。

== 人物介紹 ==
;{{nihongo|神城麻郁|神城 麻郁（かみしろ まいく）|配音：[[浪川大輔|浪川大輔]]}}
: 本篇男主人公，16歲的高校一年級學生。在孤兒院長大，靠編程打工維持生活，性格成熟，感覺上有點酷。

;{{nihongo|宮藤深衣奈|宮藤 深衣奈（みやふじ みいな）|配音：[[中原麻衣|中原麻衣]]}}
: 本篇女主人公之一，16歲的高校一年級學生。同樣在孤兒院長大，性格卻很活潑、開朗，戀愛同盟的發起者（第六集）。

;{{nihongo|小野寺樺戀|小野寺 樺恋（おのでら かれん）|配音：[[清水愛|清水愛]]}}
: 本篇女主人公之一，16歲的高校一年級學生。感覺上她對男人免疫力很弱，很容易暈倒。

;{{nihongo|織部椿|織部 椿（おりべ つばき）|配音：[[根谷美智子|根谷美智子]]}}
: 麻郁的前輩二年級學生，擔任[[學生會|學生會]]副會長職務，高中一年級跟在準備考試來高中的康生偶然遇見，在和不知人生該如何前進的康生的交談中她告訢康生應該繼續上學，於是就開始交往還約會了好幾回但有一次發生了一件事（請看第八集）她跟康生疏遠了，在認識了麻郁後喜歡上他，但在第八集跟莓談完後面前自己的感情跟康生和好再次交往。

;{{nihongo|島崎康生|島崎 康生（しまざき こうせい）|配音：[[鈴村健一|鈴村健一]]}}
: 相貌長得很帥的麻郁的同班同學，有點BL傾向，但實際上是因為半年前在準備考試來高中的時候跟椿偶然遇見，在和不知人生該如何前進的他的交談中椿告訴他應該繼續上學，於是就開始交往，還約會了好幾回，但有一次發生了一件事（請看第八集）椿跟疏遠了，因此想壓抑自己對女性的興趣，把麻郁當成了比朋友更進一步的朋友，但後來跟椿和好了再次交往。

;{{nihongo|風見瑞穗|風見 みずほ（かざみ みずほ）|配音：[[井上喜久子|井上喜久子]]}}
: 《[[拜託了老師|拜託了老師]]》的女主人公，宇宙人，麻郁所在學校的老師，後來和草薙桂結婚。

;{{nihongo|草薙桂|草薙 桂（くさなぎ けい）|配音：[[保志總一朗|保志總一朗]]}}
: 《[[拜託了老師|拜託了老師]]》的男主人公，麻郁的前輩三年級學生，風見瑞穗的老公，因患有停滯病停滯三年，目前年齡20歲。

;{{nihongo|森野莓|森野 苺（もりの いちご）|配音：[[田村由香里|田村由香里]]}}
: 學生會會長，三年級學生，就是前作《[[拜託了老師|拜託了老師]]》中跟桂一樣患有停滯病的女孩，停滯六年，目前年齡23歲。

;{{nihongo|四道跨|四道 跨（しどう またぐ）|配音：[[三浦祥朗|三浦祥朗]]}}
: 晴子的哥哥，三年級學生，長得一般的他，這次他的春天會來嗎？

;{{nihongo|四道晴子|四道 晴子（しどう はるこ）|配音：[[新谷良子|新谷良子]]}}
: 四道跨的妹妹，天真爛漫而且很可愛。

== 戀愛同盟 ==
在第6話，知道彼此對麻郁戀愛的深衣奈和樺木戀愛結成的同盟。

同盟規章
# 在弄清楚哪一個是麻郁的親人之前都不能跟麻郁告白。
# 當得知自己和麻郁是親人後要馬上報告
# 是親人的一方要幫助另一方和麻郁戀愛
# 雙方合力不讓其他女孩子接近麻郁
# 我們要待在神城麻郁的身邊
