'''拜罗伊特音乐节'''（[[德语|德语]]：'''Bayreuther Festspiele'''）是每年一度在德国[[拜罗伊特节日剧院|拜罗伊特节日剧院]]举行的[[音乐节|音乐节]]，演出内容为作曲家[[理查德·瓦格纳|理查德·瓦格纳]]的[[歌剧|歌剧]]作品，因此又称'''理查德·瓦格纳音乐节'''。音乐节一般在7月25日至8月28日举行。

== 演出内容 ==
音乐节由瓦格纳本人发起，1876年8月13日首次开幕，上演的剧目是四联剧《[[尼伯龙根的指环|尼伯龙根的指环]]》。自此每年都会从瓦格纳的《[[漂泊的荷兰人|漂泊的荷兰人]]》、《[[帕西法尔|帕西法尔]]》等十部乐剧中选择若干部演出。他早期的三部歌剧：《{{link-de|仙女 (歌剧)|Die Feen}}》、《{{link-de|禁恋|Das Liebesverbot}}》和《[[黎恩济|黎恩济]]》被他本人从节目单上剔除，因为这三部剧并不符合他后来提出的“乐剧”的要求。

== 组织机构 ==
自1973年起，音乐节的主办者为“拜罗伊特理查德·瓦格纳基金”（Richard-Wagner-Stiftung Bayreuth），基金成员有[[德意志联邦共和国|德意志联邦共和国]]、[[巴伐利亚州|巴伐利亚州]]、[[拜罗伊特|拜罗伊特市]]、拜罗伊特之友协会、巴伐利亚州基金、上弗兰肯基金、[[上弗兰肯行政区|上弗兰肯行政区]]以及瓦格纳的家族成员。基金会的负责人为拜罗伊特市长。音乐节从1986年起由拜罗伊特音乐节有限公司承担（Bayreuther Festspiele GmbH）。

== 人事 ==
艺术指导：
* 1876－1882年：[[理查德·瓦格纳|理查德·瓦格纳]]
* 1883－1906年：{{le|科西玛·瓦格纳|Cosima Wagner}}
* 1908－1930年：[[齐格弗里德·瓦格纳|齐格弗里德·瓦格纳]]
* 1931－1944年：{{le|维尼弗里德·瓦格纳|Winifred Wagner}}
* 1951－1966年：{{le|维兰德·瓦格纳|Wieland Wagner}}、{{le|沃尔夫冈·瓦格纳|Wolfgang Wagner}}
* 1967－2008年：沃尔夫冈·瓦格纳
* 2009－2015年：{{le|埃娃·瓦格纳-帕斯屈耶|Eva Wagner-Pasquier}}、{{le|卡塔琳娜·瓦格纳|Katharina Wagner}}
* 2015年起：卡塔琳娜·瓦格纳

音乐节现任音乐总监是指挥家[[克里斯蒂安·蒂勒曼|克里斯蒂安·蒂勒曼]]。
