《'''拜托了☆老师'''》（'''{{lang-ja|おねがい☆ティーチャー}}'''），是一部由[[製作委員會方式|Please!]]製作的[[动画|动画]]作品。該部動畫於2002年1月10日至3月28日在[[WOWOW|WOWOW]]播放；[[Animax_Asia|Animax Asia]]引進版權並於[[香港|香港]]及[[東亞|東亞]]等地區播放，[[JET綜合台|JET日本台]]引進版權並於[[台灣|台灣]]播放。另有[[漫畫|漫畫]]版本，由[[林家志弦|林家志弦]]作畫。

== 登場人物 ==
; {{nihongo|草薙桂|草薙 桂（くさなぎ けい）||[[聲優|聲]]：[[保志總一朗|保志總一朗]]}}
: 本作的男主角，[[長野縣|長野縣]]縣立木崎高等學校一年級學生。因為患上「停滯病」（一種在極為痛苦悲傷下引發的病，讓患者自身處於安詳的世界）的關係，雖然已經18歲，卻只有15歲的外表。意外撞見瑞穗乘著[[宇宙船|宇宙船]]降臨地球，曾經和瑞穗說出自己患了停滯病的秘密，而之後也為了保住瑞穗的秘密，和瑞穗結婚；雖然是莫名其妙的開始，但在同居的那段時間裡，兩人漸漸喜歡上彼此。可是後來被銀河總部消除有關瑞穗的記憶，一度忘記瑞穗，但因為瑞穗事先留下一盒「[[百奇|POCKY巧克力棒]]」最後也有想起來，也和瑞穗再過一段愉快的新婚生活。
; {{nihongo|風見瑞穗|風見 みずほ（かざみ みずほ）||[[聲優|聲]]：[[井上喜久子|井上喜久子]]}}
: 本作的女主角。是擁有[[人|地球人]]父親和[[外星人|外星人]]母親的[[混血兒|混血兒]]，父親在她幼時逝世。在木崎高等學校擔任國文老師（井上喜久子擁有[[初中|初中]]國文教師資格），美麗溫柔又聰明，愛上了外表年齡15歲的桂。喜歡吃「POCKY巧克力棒」，因為那是她和爸爸的小小回憶。
: 被銀河總部派駐地球擔任監視員，不小心被桂發現她是外星人，而和桂[[結婚|結婚]]，但之後也愛上了桂。與桂結婚後，只在戶籍資料上[[日本夫婦別姓問題|改姓草薙]]。可是後來銀河總部為了懲罰瑞穗，而消除地球上有關瑞穗的任何記憶。但瑞穗不顧一切的回到地球，事先在桂身上留下一盒「POCKY巧克力棒」也讓桂找回消失的那段記憶。向馬利耶下命令時最常講的一句話：「{{nihongo|這是最優先事項喔！|最優先事項よ！}}」
; {{nihongo|風見初穗|風見 はつほ（かざみ はつほ）||[[聲優|聲]]：[[高田由美|高田由美]]}}
: 瑞穗的媽媽。喜歡吃「POCHY牛奶巧克力棒」。
: 與瑞穗的爸爸在宇宙中相遇，進而相戀。整體上與瑞穗非常的相似，害桂常常誤以為是瑞穗。[[美魔女|在外表上也看不出來年齡很大]]，反而還異常的年輕。在桂跟瑞穗的感情生活方面給了相當多的幫助，在他們的感情生活方面有大的影響。本身挺喜歡桂。
; {{nihongo|風見真穗|風見 まほ（かざみ まほ）||[[聲優|聲]]：[[興梠里美|興梠里美]]}}
: 瑞穗的妹妹。喜歡吃「POCHY草莓棒」。有非常強烈的[[戀姐情結|戀姐情結]]，非常愛護瑞穗，任何地方都很為她著想，但是也因此使得桂剛開始吃了不少的苦，但是後來被桂和瑞穗的真情所感動而答應他們結婚。本身很喜歡桂。
; {{nihongo|馬利耶|まりえ||[[聲優|聲]]：[[金田朋子|金田朋子]]}}
: 瑞穗的宇宙船的控制系統。
; {{nihongo|咪魯魯|みるる||[[聲優|聲]]：[[佐久間玲|佐久間玲]]}}
: 真穗的宇宙船的控制系統。
; {{nihongo|緣川小石|縁川 小石（へりかわ こいし）||[[聲優|聲]]：[[川澄綾子|川澄綾子]]}}
: 桂的同學，家裡經營「緣川商店」。本來喜歡桂，但最後卻喜歡上正臣。
; {{nihongo|森野莓|森野 苺（もりの いちご）||[[聲優|聲]]：[[田村由香里|田村由香里]]}}
: 桂的同學，是該班的班長。性格[[無口|無口]]，[[直覺|直覺]]敏銳，某意義系列中最強的人物。外觀年幼，其實跟桂一樣罹患停滯病，停滯6年，戶籍資料上的年齡是21歲。希望好友小石和楓能得到幸福，支援著她們的戀愛。個性上在外表和跟小石她們一起的時候看不出來，但是內心深處其實是很[[自閉|自閉]]的，而且說話也蠻尖酸刻薄，但是她非常的重視[[朋友|朋友]]。
; {{nihongo|水澄楓|水澄 楓（みすみ かえで）||[[聲優|聲]]：[[大原沙耶香|大原沙耶香]]}}
: 桂的同學。因得自[[外國人|外國人]]祖父的[[遺傳|遺傳]]，有紅色頭髮與高佻的身材。在第5集之後就跟漂介是男女朋友，但是在睡夢中常常夢見漂介每次都緊緊地抱著她，在後來當然也實現了願望跟漂介在一起。
; {{nihongo|間雲漂介|間雲 漂介（まぐも ひょうすけ）||[[聲優|聲]]：[[岩田光央|岩田光央]]}}
: 桂的同學，學業成績優秀。在第5集之後就跟楓是男女朋友。他想要跟他的哥哥一樣走上[[政治|政治]]的舞台，改變現在的腐敗政治。
; {{nihongo|四道跨|四道 跨（しどう またぐ）||[[聲優|聲]]：[[三浦祥朗|三浦祥朗]]}}
: 桂的同學，喜歡用[[望遠鏡|望遠鏡]]看夜空中的星星。喜歡瑞穗。
; {{nihongo|山田正臣|山田 正臣（やまだ まさおみ）||[[聲優|聲]]：[[杉田智和|杉田智和]]}}
: 木崎高等學校技術老師。個性灰暗，沉默寡言。
; {{nihongo|江田島稔|江田島 みのる（えだじま みのる）||[[聲優|聲]]：[[内田直哉|内田直哉]]}}
: 桂的叔叔，[[婦產科|婦產科]][[醫師|醫師]]。
; {{nihongo|江田島このは|江田島 このは（えだじま このは）||[[聲優|聲]]：[[佐久間玲|佐久間玲]]}}
: 桂的嬸嬸，稔的妻子兼婦產科診所[[護士|護士]]。
; {{nihongo|天堂|天堂（てんどう）||[[聲優|聲]]：[[仲野裕|仲野裕]]}}
: 木崎高等學校校長，妻子是該校校友。
; {{nihongo|草薙梢|草薙 こずえ（くさなぎ こずえ）||[[聲優|聲]]：[[南央美|南央美]]}}
: 桂的姊姊，跟桂一樣罹患停滯病，停滯3年，就讀初中時在桂眼前[[自殺|自殺]]。在動畫第11話出現在桂的夢中。
; {{nihongo|裁判官|裁判官||[[聲優|聲]]：[[江原正士|江原正士]]}}
: 銀河總部的裁判官，認定瑞穗在地球嚴重違規，消除地球上有關瑞穗的任何記憶。

== 背景 ==
《星空的邂逅》主要在日本[[長野縣|長野縣]][[大町市|大町市]][[木崎湖|木崎湖]]取景，木崎高等學校校舍以[[舊制松本高等學校|舊制松本高等學校]]校舍為藍本。
