'''珀玥'''（{{lang|en|'''The Garrison'''}}），是位於[[香港|香港]][[新界|新界]][[沙田|沙田]][[大圍|大圍]][[美田路|美田路]]的單幢住宅，樓高共23層，由[[遠東發展|遠東發展]]發展，[[仲量聯行|仲量聯行]]管理，預計於[[2019年3月|2019年3月]]入伙。

== 歷史 ==
珀玥 前身為官地，[[2013年6月|2013年6月]]時任[[發展局局長|發展局局長]][[陳茂波|陳茂波]]宣怖於當年7至9月推出9幅地皮，其中包括此位於[[大圍|大圍]][[美田路|美田路]]佔0.07[[公頃|公頃]]的住宅地皮<ref>[http://www2.hkej.com/property/article/id/90144 6幅小型住宅地登場]</ref>，至[[2014年4月|2014年4月]]，[[地政總署|地政總署]]推出地皮招標，終獲8家財團入標，包括[[新鴻基地產|新鴻基地產]]、[[盛洋投資|盛洋投資]]、[[宏安集團|宏安集團]]、[[遠東發展|遠東發展]]及[[信和置業|信和置業]]等，雖然地皮位於[[大圍|大圍]]市中心，唯由於地皮呈三角形，剪裁有一定難度，加上項目需提供停車位，需建於地庫及或興建汽車升降台，不排除會削減用作地舖的樓面，故市場估計會直接拉低項目價值<ref>[http://paper.takungpao.com/resfile/PDF/20140405/PDF/a2_screen.pdf 大圍地盤呈三角形]</ref>，當時市場估值約1.58億至2.42億元<ref>[https://news.cnyes.com/news/id/896312 遠東發展(00035.HK)1.5億奪大圍美田路地皮]</ref>，最終於同月[[地政總署|地政總署]]宣怖[[遠東發展|遠東發展]]以1.48億元投得地皮，平均每[[平方呎|平方呎]]樓面地價低至2,817元，為市場預期下限<ref>[https://news.now.com/home/finance/player?newsId=97926 發展商避險 大圍住宅地低價批出]</ref>。

[[2014年5月|2014年5月]][[遠東發展|遠東發展]]表示有關地皮考慮發展作商住用途，以用盡[[地積比率|地積比率]]，當時計劃準備總共約以三億五千萬[[港元|港元]]以發展上層成以開放式及一房間隔為主的酒店式精品住宅，樓下則作商業用途<ref>[http://std.stheadline.com/daily/news-content.php?id=160617&target=2 遠展大圍建精品豪宅混合商鋪]</ref>，項目最終於[[2017年8月|2017年8月]]動工。

== 簡介 ==
珀玥 共設23層（包括地庫1層但不設4、13、14及24樓），於5至25樓提供118伙[[實用面積|實用面積]]介乎201方呎至404方呎的1房至2房住宅單位，當中逾98%為1房戶，其屬[[沙田區|沙田區]]內最細的1房戶，項目另設2伙頂層2房特色單位<ref>[https://ps.hket.com/article/2107522/%E9%81%A0%E6%9D%B1%E7%8F%80%E7%8E%A51%E6%88%BF201%E5%91%8E%E7%B4%B0%E7%B5%95%E6%B2%99%E7%94%B0 遠東珀玥1房201呎細絕沙田]</ref>，其餘樓層分別設有
* 地庫1層：[[停車場|停車場]]（2個商戶停車位、1個商戶電單車停車位、8個單車停車位、9個住客停車位、1個住客訪客暢通易達停車位、2個住客訪客停車位及1個住客電單車停車位）
* 地下：住宅入口大堂、商舖、[[停車場|停車場]]（1個商戶上落貨車位、1個住客上落貨車位及1個商戶停車位）
* 1樓：商舖
* 2樓：機電層
* 3樓：[[會所|住客康樂設施]]（面積約4,611[[平方呎|平方呎]]，包括健身室、瑜珈室及戶外燒烤場等）

項目於[[2018年5月|2018年5月]]獲批[[預售樓花同意書|預售樓花同意書]]，至[[2018年7月|2018年7月]]發展商上載[[售樓說明書|售樓說明書]]及公怖售價，當中售樓說明書有所出錯，包括平台單位圖則少畫了門、漏計露台面積，消息指發展商在上樓書前已知出錯，惟趕推盤先行上載，及後再作出修訂；至於一房單位折實售價由422.6萬元至685.9萬元，平均呎價19,800元至24,100元，被指「貴絕大圍上車屋苑」<ref>[https://hk.finance.appledaily.com/finance/daily/article/20180706/20441936 無視娥六招　呎售逾2萬 地價每呎$2,871 大圍新盤　麵包貴麵粉6倍]</ref>。

至7月中旬開售當中112伙，結果收票近2300票，超額認購近20倍<ref>[https://hk.finance.appledaily.com/finance/realtime/article/20180714/58440964 【擔遮掃新盤】大圍珀玥賣112伙 沽出逾9成]</ref>，唯有報導指有[[地產代理|地產代理]]呼籲幫忙入飛認購，若抽中會把單位留予他人購買，故被指「入票數字水分甚高」<ref>[https://hk.finance.appledaily.com/finance/realtime/article/20180711/58430035 【大圍新盤】網傳代理call客谷飛 珀玥收逾1,400票水份超高]</ref>
，最終開售當日下午8時項目沽出106伙，佔可供發售單位約95%，套現約5.9億[[港元|港元]]<ref>[http://hk.on.cc/hk/bkn/cnt/finance/20180714/bkn-20180714170951325-0714_00842_001.html 【珀玥開售】珀玥賣112伙 即日沽106伙]</ref>。而[[示範單位|示範單位]]曾設於[[黃大仙|黃大仙]][[現崇山#商場|現崇山商場]]，當中展出的已裝修單位拆去房與廳及房與廁的牆身，有傳媒指是「掩人耳目，扮有空間」<ref>[https://hk.finance.appledaily.com/finance/realtime/article/20180705/58403073 【房策後新盤】珀玥1房206呎示位 睡房不適合高人]</ref>。

== 命名 ==
發展商於[[2018年|2018年]][[5月10日|5月10日]]正式公佈項目中英文名稱，表示「項目具交通、購物及優美環境於一身，仿如罕珍明珠，綻放光芒，故中文名以「珀玥」為名，英文名「Garrison」則有駐地及要塞的意思。」<ref>[https://ps.hket.com/article/2070154/%E9%81%A0%E6%9D%B1%E6%B2%99%E7%94%B0%E9%A0%85%E7%9B%AE%E5%91%BD%E5%90%8D%E7%82%BA%E7%8F%80%E7%8E%A5%20%E9%80%BE9%E6%88%90%E7%82%BA1%E6%88%BF 遠東沙田項目命名為珀玥 逾9成為1房]</ref>
