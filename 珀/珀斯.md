'''珀斯'''（{{lang-en|'''Perth'''}}）是[[西澳大利亞州|西澳大利亞州]]的[[首府|首府]]，也是[[澳大利亞|澳大利亞]][[澳大利亚城市人口列表|第四大城市]]。根據2008年6月的[[澳大利亞人口|人口統計]]，珀斯[[都會區|都會區]]的人口共有202萬人，是澳洲第四大城市，人口增长率高於國家統計的平均水準。

由於地處[[澳大利亞洲|澳洲大陸]]西岸[[地中海氣候|地中海氣候]]地區，擁有溫和的氣候與[[天鵝河_(西澳)|天鵝河]]岸的別緻景色，使珀斯得以成為非常受歡迎的觀光[[旅遊|旅遊]]目的地。

珀斯擁有廣闊的居住空間及高水準的生活質素，於每年的[[世界最佳居住城市|世界最佳居住城市]]評選中都是名列前茅，反映出珀斯無論居住環境，生活質素及社會福利等都是极佳的城市。珀斯人的友善态度世界公认，曾於2003年獲得世界最友善城市称号<ref>[http://economy.enorth.com.cn/system/2002/08/16/000398323.shtml 2003年世界最友善城市]</ref>，得到世界性的讚賞及認同。

== 歷史 ==
[[File:Flag_of_Western_Australia.svg|thumb]]
[[File:The_Foundation_of_Perth.jpg|thumb]]

雖然在珀斯之前，[[英國陸軍|英軍]]早就在1826年時在西澳洲南面海岸的{{link-en|喬治國王峽灣|King George Sound}}（後來改名為「[[奧班尼|奧班尼]]」）建立了一個基地，以避免傳言中[[法國|法國]]要兼併澳大利亞西部的可能，但珀斯仍然是整個澳大利亞西部第一個建立的殖民城鎮。1829年建城的珀斯剛開始時是自由屯墾殖民地──{{tsl|en|Swan River Colony|天鵝河殖民地}}的首府。珀斯是由{{link-en|詹姆斯·史特靈|James Stirling}}爵士所命名，並選擇此作為新城鎮。史特靈是[[蘇格蘭|蘇格蘭]]航海家，與當時英國殖民地大臣──[[美利爵士|美利爵士]]（Sir George Murray）的願望一致，決議以墨瑞的出生地和他在[[英國下議院|英國下議院]]裡，議會席位的所在地──[[珀斯-金罗斯|珀斯郡]]（{{lang|en|Perth Shire}}）命名為珀斯。該殖民地在1850年之後改名為「西澳大利亞」（簡稱「西澳」），並開始成為被[[英國|英國]]放逐海外的[[囚犯|囚犯]]所寄居的第二故鄉，以補足當地農業與商業發展過程中非常欠缺的人力資源。

1900年，經公民投票，西澳在1901年正式加入了[[澳洲聯邦|澳洲聯邦]]，亦是澳大利亞最後一個同意加入聯邦的[[殖民地|殖民地]]。當時其它前殖民地為了說服西澳洲願意加入，作出了各種讓步，包括從東部建築[[鐵路|鐵路]]幹線（經Kalgoorlie）連接到珀斯等。
 
1933年，在[[公民|公民]][[投票|投票]]下，西澳表決脫離澳洲聯邦，當時以大比數傾向於贊成獨立。但是，表決前不久剛好正值國會[[選舉|選舉]]，結果當時的政府下台。而新上任的政府並不贊成獨立行動。但鑒於公民投票的結果，新政府仍然向[[英國|英國]]訴請獨立，只是申請被駁回。

在發展歷程中，珀斯得以繁榮的主要原因，是成為[[天然資源|天然資源]]產業的一個重要服務中心。作為一個最接近對[[金|金]]、[[鐵礦|鐵礦]]、[[鎳|鎳]]、[[鋁土|鋁土]]、[[金剛石|金剛石]]、[[礦物沙|礦物沙]]、[[煤炭|煤炭]]、[[油|油]]以及[[天然氣|天然氣]]龐大儲備的城市，大多世界的主要能源和工程企業在珀斯皆設有辦公室。

踏入1990年代，大多數時間都是由{{link-en|理查德·科特|Richard Court}}所領導的自由黨政府當政，珀斯亦從這時期開始步向繁榮發展。

== 地理 ==
[[File:Perth_locator-MJC.png|thumb]]的位置（紅點所示）]]
珀斯中心坐标为[[南緯|南緯]]31度52分48秒，[[東經|東經]]115度52分58秒。全市面積5,386平方千米。西面濒临[[印度洋|印度洋]]，距非洲海岸约7350千米；東面是內陸沙漠地帶。

珀斯属[[地中海气候|地中海气候]]，四季较为分明。夏季干燥,较为炎热；冬季溫和湿润，在天气晴好时气温不会太低，只有清晨及晚間可能比較冷凉，气温日较差有時可高達20℃，但气温降至0℃及以下的情况极其少见。最熱月為2月，平均气温24.5℃，极端最高气温46.2℃（1991年2月23日）；最冷月为7月，平均气温13.0℃，极端最低气温-0.7℃（2006年6月17日）。年平均降水量746.9毫米。

== 政府體制 ==
[[File:governmentWesternAustralia.jpg|thumb]]
根據澳大利亞憲法，西澳是依照[[西敏制|西敏制]]（即[[英國|英國]]式[[議會|議會]]制）的形式來施行[[三權分立|三權分立]]的管治的。除了被聯邦分割了些立法權和司法權外，其它方面保持完全獨立，其中：
* [[立法權|立法權]]取決於西澳的議會，即西澳立法委員會與西澳立法局。
* [[行政權|行政權]]取決於行政委員會，成員包括州長和資深部長。行政權由西澳州長和內閣行使，由西澳總督代表英國女皇任命。
* [[司法權|司法權]]由西澳最高法院和次級法院體系行使，但澳大利亞高等法院有最後的司法權。在澳大利亞憲法授予聯邦政府權力制定為法律的情況下，其它聯邦法庭亦有一定的司法權。

西澳州議會位於首府珀斯，隸屬西澳州議會的[[都會區|都會區]]劃分為超過三十多個[[西澳大利亞州地方政府區域|地方政府區域]]。

== 經濟 ==
[[File:City-of-Perth.jpg|thumb]]]]
在过去，相比於[[澳洲|澳洲]]其他主要大[[城市|城市]]，珀斯有着較高的生活水準。原因在于，珀斯[[都會區|都會區]]各行各業的一般工資虽略低於其他大城市，但[[房地產|房地產]]價格遠低於[[澳洲東岸|東岸城市]]。但自2005年至2006年的兩年間，情況卻正在轉變，珀斯的[[房地產|房地產]]價格大幅度上升，而其他東岸城市的升幅卻很輕微，[[悉尼|悉尼]]更有下跌的趨勢，使得珀斯與各城市的生活水準日漸拉近。

珀斯一如其他東岸的城市，一般屬於都會區內的職位都能提供，惟職位供應的數量普遍較[[悉尼|悉尼]]及[[墨爾本|墨爾本]]略少。

[[西澳大利亞|西澳]]擁有極豐盛的[[天然資源|天然資源]]——[[煤|煤]]及[[金屬|金屬]]矿藏。許多採礦業及與[[礦物|礦物]]有關的行業，在珀斯設有[[總部|總部]]。雖然採礦的工程都在珀斯以外的區域進行，但珀斯是提供工程師及專業人材技術支援的主要來源。距離珀斯30公里以南之[[奎那那|奎那那市]]的一所龐大[[煉油廠|煉油廠]]，所需的工程及化學專業人員是來自珀斯，亦為珀斯帶動了就業市場。

此外，[[農業|農業]]及[[旅遊業|旅遊業]]亦在珀斯的經濟中占有相当份额。

== 居住概況 ==
[[File:ScarborougHouse.jpg|thumb]]
珀斯主要分為240多個[[郊區|區域]]，基本上每個區也有住宅。由於土地遼闊，房子一般都寬敞舒適，大部份設有[[花園|花園]]，[[西澳洲政府|當地政府]]更規定業主要在花園種植花草以美化環境。房屋一般以紅瓦屋頂的[[平房|平房]]，或比較新式的特色樓房，或再高尚些的[[別墅|別墅]]為主，亦有少數的[[公寓大廈|大廈]]（Apartment）座落於某些比較城市的區域，如珀斯，{{tsl|en|East Perth|東珀斯}}、[[:en:Subiaco,_Western_Australia|Subiaco]]等。一般来说日落海岸、西区和天鹅河附近的郊区房价较贵，近年来内城区的一些郊区的房价呈高速上涨趋势，如 West Leederville、East Perth等。房价较低地区则是一些离海滩，天鹅河及市中心相对较偏远的郊区，如东面的Midland、Guildford；南面的Armadale。由于矿产资源丰富，珀斯房价过去十多年间高速上涨，但是由于最近几年矿业不景气，珀斯房地产进入调整期。

== 旅遊 ==
=== 費利曼圖 ===
{{main|費利曼圖}}
[[File:FremantleMonument(alfeewusy).jpg|thumb]]海軍紀念碑]]
[[費利曼圖|費利曼圖]](Fremantle)距離珀斯市以南約19公里（12英里）的一個海港城市，澳洲西海岸天鵝河入海口。費利曼圖於1829年成為天鵝河畔的第一個居民點。在1929年被設立為市，現約有2.6萬人口。

費利曼圖是一個古意盎然的海港城市。該市的大部分建築物已被[[西澳洲政府|當地政府]]列入保護古蹟，故整個城市也仍舊保留著一百年前的風貌。当地值得参觀的景點有：
* 西澳軍事博物館（[[:en:Army_Museum_of_Western_Australia|Army Museum of Western Australia]]）
* 費利曼圖舊監獄（[[:en:Fremantle_Prison|Fremantle Prison]]）
* 費利曼圖市場（[[:en:Fremantle_Markets|Fremantle Market]]）
等等。

=== 海灘 ===
[[File:ScarboroughBeach(alfeewusy).jpg|thumb]]
珀斯有很多著名的[[海灘|海灘]]，其景色之優美程度足以和澳洲東岸的[[黄金海岸_(澳大利亚)|黃金海岸]]相媲美。由眾多著名海灘連接而成之海岸線名為[[日落海岸|日落海岸]]，該海岸覆蓋範圍包括南面的費利曼圖海岸以北至[[君达乐|君达乐]]（Joondalup），當中包括的海灘多達20幾個，如：
* 布萊頓海灘（[[:en:Brighton,_Western_Australia|Brighton Beach]]）
* 科茨洛海灘（[[:en:Cottesloe,_Western_Australia|Cottesloe Beach]]）
* 斯旺伯恩海灘（[[:en:Swanbourne,_Western_Australia|Swanbourne Beach]]，著名[[天體|天體]]沙灘）
* 洛里厄特海灘（[[:en:Floreat,_Western_Australia|Floreat Beach]]）
* 斯卡伯勒海灘（[[:en:Scarborough,_Western_Australia|Scarborough Beach]]，著名[[衝浪|衝浪]]聖地）
* 城市海灘（[[:en:City_Beach|City Beach]]）
* 北部海灘（[[:en:North_Beach,_Western_Australia|North Beach]]）
* 沃特曼海灘（[[:en:Waterman,_Western_Australia|Waterman Beach]]）
* 索倫托海灘（[[:en:Sorrento,_Western_Australia|Sorrento Beach]]）
等等。

=== 國王公園 ===
[[File:WildFlowers.jpg|thumb]]
國王公園（[[:en:Kings_Park,_Western_Australia|Kings Park]]）座落於天鵝河畔的伊萊山（[[:en:Mount_Eliza,_Western_Australia|Mt. Eliza]]）之上，佔地相當廣闊達400公頃，盤踞整個山頭，公園內茂樹林立，樹高比天，壯麗非常。園內的設計一方面盡量保持著原始大自然氣息，有古老的觀光電車穿梭往來園內，及騎[[駱駝|駱駝]]遊覽公園等項目，都令人覺得有回歸原始的感覺.另一方面又增加了不少[[現代化|現代化]]設施以方便[[遊客|遊客]]，如設有在柏斯非常著名的5星級餐廳Frasers Restaurant、燒烤爐，[[溫室|溫室]]及[[觀景台|觀景台]]更是俯瞰全珀斯市[[天際線|全景]]的最佳位置。原始與現代化交織而成的Kings Park,卻有著以想不到和諧與協調。特別一提的是每年的9~11月是珀斯的[[春天|春天]]，是觀賞野花盛開的季節，每年那個時候都會在Kings Park舉行野花節給遊客觀賞，無需他們跑到遙遠的西澳郊區才可欣賞到。[[公園|公園]]內原本廣大的綠色草原，頓然變成一片色彩萬千的花海，非常漂亮悅目。

=== 中心商業區 ===

== 人口 ==
1980年代早期時，珀斯超過[[阿德萊德|阿德萊德]]，成為澳洲人口第四多的城市。2014年珀斯人口為2,021,200人。市內人口主要為歐裔（英裔、愛爾蘭裔、意大利裔），約3%是華人，一大部分是近30年來從東南亞（馬來西亞、新加坡、越南、印尼）移民到珀斯的華人。也有一定數量的印度裔和南非（白人）裔移民。

== 交通 ==
[[File:PerthRailway.jpg|thumb]]
[[File:PerthRailwayStationUndergroundCentral.jpg|thumb]]
[[File:Aerial_view_of_Fremantle.JPG|thumb]]

珀斯的國際及國內[[珀斯機場|機場]]都在市中心以東的[[:en:Redcliffe,_Western_Australia|Redcliffe]]區。距離市中心的商業區分別達十七與十二公里。另一個名為[[:en:Jandakot_Airport|詹達科特機場]]在市中心以南的詹達科特郊區，專為一般航空學而設。道路方面，珀斯有一套完善公路網與三條[[高速公路|高速公路]]、九條[[:en:List_of_Highways_in_Perth|都會區高速公路]]，並未設有收費站。

市區的公共運輸－巴士、城郊列车和小型邮轮，由[[:en:Transperth|Transperth]]提供。其中包括免費的[[:en:Perth_Central_Area_Transit|CAT]]巴士（'''Central Area Transit'''，中央區域運輸交通），方便人們在市中心的活動。四條CAT巴士路線（紅、藍、黃、綠）貫穿珀斯市東西、南北。珀斯為降低污染，除CAT以外，在繁華的市中心亦提供免費的公共運輸系統，搭其他公車只要上車和下車都是在市中心範圍內完全免費。

位於市中心的珀斯鐵路總站，是富有古西方建築風格的[[地標|地標]]，亦是中途轉乘站，提供五條幹線共六十九個站，大部份主要地區也能到達。由總站珀斯開出至五個端點站分別為西南面的[[費利曼圖|費利曼圖]]，北面的[[君達樂|君達樂]]及[[畢拿_(西澳大利亞)|畢拿]]，東面的[[:en:Midland_railway_line,_Perth|米德兰]]、東南面的[[:en:Armadale_railway_line,_Perth|阿馬代爾]]及南面的[[曼杜拉|曼杜拉]]。若無火車到達的地方亦有巴士行駛，以確保盡量提供公共運輸服務於各區；惟一般巴士班次較少。

在市中心有两个大型公车中转站，珀斯巴士埠（'''Perth Busport'''）和伊丽莎白港巴士总站（'''Elizabeth Quay Bus Station'''），其中珀斯巴士埠和[[:en:Perth_Train_Station|珀斯鐵路總站]]是相連的。另外在費利曼圖的公车转運站，亦與費利曼圖鐵路總站連接。

另外，999路公车和998路公车是环城车，路途经过大多珀斯境内的学校，其中包括珀斯市內的四所大學，主要为方便学生而设立的学校公车线。

珀斯機場有公共巴士到達，40號線以及935號線可以到達三號及四號客運大樓，而380號線則可以到達一號及二號客運大樓。

珀斯的车票是时间制，一般车票可以在指定時間内无限换乘同區間的公车或者铁路，時效為两小时至三小时不等。

== 唐人街 ==
[[File:Northbridge1.jpg|thumb]]
北橋區（[[:en:Northbridge,_Western_Australia|Northbridge]]）是珀斯著名的[[唐人街|唐人街]]，这里有很多华人餐厅和华人商店，是华人聚集最多的地方之一。

北橋區除了是珀斯的唐人街，該區亦是當地人的文化及娛樂場所的集中地。離市中心步行只需數分鐘路程。每逢週末或假日北橋區的酒吧、夜總會和咖啡廳便擠滿了客人，非常熱鬧。是澳洲其中一個最大的各式餐廳集中地之一。超過一百間餐廳可供選擇，特別集中在Lake Street及James Street，種類由[[中國菜|中國菜]]、[[印度菜|印度菜]]、[[希臘菜|希臘菜]]、[[埃及菜|埃及菜]]、[[泰國菜|泰國菜]]、[[越南菜|越南菜]]至[[意大利菜|意大利菜]]等。此外，[[西澳博物館|西澳博物館]]、[[西澳藝術館|西澳藝術館]]、以及位於亞歷山大圖書館大樓（[[:en:Alexander_Library_Building|Alexander Library Building]]）内的西澳州立圖書館（[[:en:State_Library_of_Western_Australia|Western Australia State Library]]）等文化設施亦集中在此地區。
