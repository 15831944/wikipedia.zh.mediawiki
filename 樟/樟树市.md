'''樟树市'''位于[[江西省|江西省]]中部，[[鄱阳湖平原|鄱阳湖平原]]的南缘，是隶属于[[宜春市|宜春市]]的一个[[县级市|县级市]]。

== 历史沿革 ==
[[南唐|南唐]][[升元|升元]]二年（公元938年），析[[高安|高安]]之建安、修德，新淦之崇学三乡置清江县，治潇滩镇（今[[临江镇|临江镇]]）。

北宋[[淳化|淳化]]三年（公元992年），以清江、新淦、新喻三县置[[临江军|临江军]]，清江为[[附郭县|附郭县]]首。元改临江路。

明改临江府。清沿明制。

1949年6月19日，解放军占领清江，县人民政府置[[樟树镇_(清江县)|樟树镇]]，7月15日迁临江镇，1950年3月27日又迁回樟树镇。

1988年10月26日，撤清江县及樟树镇，设樟树市<ref>[http://www.gov.cn/test/2007-03/23/content_559151.htm 中华人民共和国行政区划（1988年）]</ref>。

1989年1月13日，同意樟树市撤销[[樟树镇_(清江县)|樟树镇]]设立淦阳、鹿江、吴家巷、薛溪四个办事处<ref>[http://www.xzqh.org/html/show/jx/10401.html 1989年江西省行政区划]</ref>。

== 地理 ==
东南部、西北部为山地丘陵，中间[[赣江|赣江]]及[[袁水|袁水]]两岸为平原。

袁水在市区西汇入赣江。

樟树市属[[亚热带季风气候|亚热带季风气候]]。

== 药都 ==
樟树是与河北[[安国|安国]]、安徽[[亳州|亳州]]齐名的药都，自古即有“药不过樟树不灵”一说。“[[樟树帮|樟树帮]]”制作的中药[[饮片|饮片]]曾享有盛誉。

== 交通 ==
* 铁路：[[浙赣铁路|浙赣铁路]]经市区，设[[樟树站|樟树站]]；[[京九铁路|京九铁路]]经东南部，设[[樟树东站|樟树东站]]。
* 公路：[[105国道|105国道]]，[[沪昆高速公路|沪昆高速公路]]和[[赣粤高速公路|赣粤高速公路]]在昌傅镇交汇。
* 水路：赣江干流纵贯本市。市区有赣江铁路大桥和赣江公路大桥各一座，两桥之间为港区。
* 航空：东南有军用机场。

== 经济 ==
* 药：樟树现仍是重要的中药材市场，每年都要举行药交会。药企有樟树制药厂、仁和集团。
* 酒：市区东郊有四特酒厂。
* 盐：市区南郊有盐矿。
