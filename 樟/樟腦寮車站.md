'''樟腦寮車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[竹崎鄉|竹崎鄉]]，為[[行政院農業委員會林務局|林務局]][[阿里山森林鐵路|阿里山森林鐵路]][[阿里山線|阿里山線]]之[[鐵路車站|鐵路車站]]。

== 車站構造 ==
站房最左有候車的[[坐位|坐位]]，中間有買票的窗口，右面是[[辦公室|辦公室]]。看來像是木建的，其實只是在外牆加上原木，站房旁設有供蒸汽機車加水用的[[水鶴|水鶴]]，不過現已停用。

== 利用狀況 ==
*阿里山線（主線）的停靠站。2016年1月3日~2017年9月1日，周日辦理312次中興號與1次阿里山號[[列車交會|列車交會]]，為台灣唯一可使用之[[折返式鐵路|折返式車站]]<ref>{{cite web|url=http://www.cna.com.tw/news/ahel/201601030030-1.aspx| title=阿里山鐵路 加開週日獨立山區間車| publisher=中央通訊社 |date=2016-01-03|accessdate=2016-01-13 |language=中文}}</ref><ref>{{cite news|url=http://www.railway.gov.tw/Alishan-tw/index.aspx?sn=18065&n=20641|title=森鐵本線時刻表|publisher=阿里山森林鐵路管理處}}</ref>。

== 車站週邊 ==
*有民居和商店。
*獨立山步道

== 歷史 ==
*1912年10月1日：設立「樟腦寮驛」。
