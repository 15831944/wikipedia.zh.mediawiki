'''厘'''，也作'''釐'''，一个传统的中国重量单位，一-{zh-hans:厘; zh-hant:釐}-为“[[分_(質量單位)|分]]”的十分之一，“[[钱_(重量单位)|钱]]”的百分之一，“[[两|两]]”的千分之一。

根据[[清朝|清朝]][[康熙|康熙]]年間制定發布，[[中華民國|中華民國]][[北洋政府|北洋政府]]繼續的[[營造尺庫平制|營造尺庫平制]]， 0.0000625斤 = 0.001兩 = 0.01錢 = 0.1分 = 1釐 = 10毫 = 0.037301克。

根據1884年《[[香港法例|香港法例]]》第22條的定義，一-{zh-hans:厘; zh-hant:釐}-等於0.0013[[安士|安士]]。現時[[香港度量衡制度|香港度量衡制度]]中，根據香港現時的《度量衡條例》，一-{zh-hans:厘; zh-hant:釐}-等於十分之一分或一萬六千分之一[[斤|斤]]，即3.779936375[[毫克|毫克]]<ref name = "hk_law">{{cite web|title = 《度量衡條例》 |work = 香港法例 | url = http://www.legislation.gov.hk/blis_ind.nsf/FB2D3FD8A4E2A3264825647C0030A9E1/7A87923654646266482564AF000FABF8?OpenDocument}}</ref>。
