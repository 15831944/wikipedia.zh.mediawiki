'''汽車道'''（{{jpn|j=汽車道|hg=きしゃみち}}、英文：''Kishamichi Promenade''）位於[[日本|日本]][[神奈川縣|神奈川縣]][[橫濱市|橫濱市]]，是利用[[櫻木町站|櫻木町站]]前至{{link-ja|新港 (橫濱市)|新港 (横浜市)|新港}}地區的鐵道廢線所整備的步道。1997年啟用，是屬於港灣環境整備設施（港灣綠地）<ref>{{Cite web|url=http://www.city.yokohama.jp/me/reiki/honbun/ag20208491.html|title=横浜市港湾施設使用条例第2条第2項の規定に基づく港湾施設の告示（昭和43年5月1日 告示第94号）|date=1968-5-1|publisher=横浜市総務局|accessdate=2013-3-10}}</ref>，也是2002年設定的散步路線「開港之道」起點<ref>[http://www.natsuzora.com/may/town/kaikonomichi.html 開港の道]（横浜線沿線 街角散歩）</ref><ref>[http://matiere.at.webry.info/200805/article_1.html ヨコハマ ～ 「開港の道」バーチャル散歩]（matiere）</ref>。有部分在[[橫濱港未來21|港未來地區]]（17街區）的'''運河公園'''內。

== 概要 ==
汽車道原先名為'''Wiener Promenade'''（{{jpn|j=ウィンナープロムナード}}），1996年（[[平成|平成]]8年）舉行名稱公募後改為現名。汽車道使用的是1911年（[[明治|明治]]44年）開通的舊橫濱站與新港埠頭之間的臨港線（通稱'''稅關線'''）廢線跡，部分區段還可見到軌條。

這條臨港線最初是用來運輸貨物，中途經過新港埠頭內的橫濱港貨物所（之後的橫濱港站），最後連結至[[橫濱稅關|橫濱稅關]]內的貨物所。1920年（[[大正|大正]]9年）起，本條路線開始在[[舊金山|舊金山]]航路出航日運行客運列車。戰後[[美軍|美軍]]曾暫時接收，1952年（[[昭和|昭和]]27年）返還後持續辦理客運業務至1961年（昭和35年）[[冰川丸|冰川丸]]最後一次航行為止。

之後，這條貨物支線在1986年（昭和61年）廢除<ref group="注">高島至新港埠頭內的橫濱港信號場<small>（舊橫濱港站）</small>，在文書上的旅客營業廢止日為1987年。</ref>。1989年（平成元年）舉行{{link-ja|橫濱博覧會|横浜博覧会}}時，主辦單位開設從會場入口櫻木町站附近（稱為日本丸站）至[[山下公園|山下公園]]冰川丸附近（稱為山下公園站）的客運列車（[[柴聯車|柴聯車]]）<ref group="注">橫濱博覧會委員會取得限期的鐵道業者執照。</ref>，博覧會結束後再次廢除。1997年（平成9年），橫濱市整備為汽車道。

汽車道有三座橋梁連結{{link-ja|日本丸 (初代)|日本丸 (初代)|日本丸}}側與新港地區兩座[[人工島|人工島]]<ref name="Kishamichi">「汽車道パンフレット」（横浜市港湾局）</ref>。這些橋樑皆是由開通當時所架設的美製與英製[[桁架橋|桁架橋]]改建而成。

== 橋梁 ==
汽車道上的橋樑稱為港1號橋梁〜港3號橋梁，現為{{link-ja|橫濱市認定歷史建造物|横浜市認定歴史的建造物}}<ref>{{Cite web|url=http://www.city.yokohama.lg.jp/toshi/design/m09/building/|title=認定歴史的建造物一覧|publisher=横浜市都市整備局|accessdate=2013-3-10}}</ref>。

=== 港1號橋梁 ===
1907年（明治40年）與港2號橋梁一起由{{link-en|美國橋梁公司|American Bridge Company}}所製作的桁架橋。1909年（明治42年）由[[鐵道院|鐵道院]]架設<ref name="Bridge-1">{{Cite web|url=http://www.city.yokohama.lg.jp/toshi/design/m09/building/n33.html|title=歴史的建造物30|publisher=横浜市都市整備局|accessdate=2013-3-10}}</ref>。港1號橋梁由兩個30英尺板樑橋和一個100英尺桁架橋組成<ref>現設置的橋梁說明板</ref>。橋寬與複線相當。2012年（平成24年）10月24日至2013年（平成25年）3月15日進行補修工程。
; 概要<ref name="Bridge-1"/>
: 構造概要：鋼桁架橋
: 設計者：鐵道院
: 製作者：美國橋梁公司

=== 港2號橋梁 ===
100英尺的複線桁架橋。與港1號橋梁於同年製作<ref name="Bridge-1"/>。
; 概要<ref name="Bridge-2">{{Cite web|url=http://www.city.yokohama.lg.jp/toshi/design/m09/building/n34.html|title=歴史的建造物31|publisher=横浜市都市整備局|accessdate=2013-3-10}}</ref>
: 構造：鋼桁架橋
: 設計者：鐵道院
: 製作者：美國橋梁公司

=== 港3號橋梁 ===
原是1906年（明治39年）架設的{{link-ja|北海道炭礦鐵道|北海道炭礦鉄道}}[[石勝線|夕張線]]{{link-ja|夕張川|夕張川}}橋梁的100英尺桁架橋，1928年（昭和3年）轉用於{{link-ja|橫濱第二合同廳舍|横浜第二合同庁舎|橫濱生絲檢査所}}的大岡橋梁<ref name="Kishamichi"/><ref name="Bridge-3">{{Cite web|url=http://www.city.yokohama.lg.jp/toshi/design/m09/building/n35.html|title=歴史的建造物32|publisher=横浜市都市整備局|accessdate=2013-3-10}}</ref>。現在將夕張川橋梁桁架橋的一部份移至汽車道保存。此座橋樑為英國製，桁架高度較低<ref name="Kishamichi"/>。
; 概要<ref name="Bridge-3"/>
: 構造概要：板樑橋（鋼製華倫桁架橋）
: 設計者：鐵道院
: 製作者：川崎造船所兵庫分工場

== 周邊綠地整備（運河公園） ==
Navios橫濱西側（{{link-ja|橫濱WORLD PORTERS|横浜ワールドポーターズ}}前）的港未來地區17街區為'''運河公園''' <span style="font-size:x-small;">{{coord|35|27|10.49|N|139|38|18.48|E|scale:10000|name=運河公園}}</span>。汽車道過港3號橋梁後進入運河公園內再穿過Navios橫濱下方往紅磚倉庫方向前進。

運河公園內有{{link-ja|日本丸紀念公園|日本丸メモリアルパーク}}與前往{{link-ja|象鼻公園|象の鼻パーク}}、{{link-ja|大棧橋|大さん橋}}方向的水上巴士碼頭<ref>[http://keihinferry.co.jp/waterbus/index.html 京浜フェリーボート：水上バス]</ref>。公園南側[[運河|運河]]對岸是正在進行[[都市更新|再開發]]的{{link-ja|北仲通地區|北仲通地区}}。
