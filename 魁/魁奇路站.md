'''魁奇路站'''是[[廣佛地鐵|廣佛地鐵]]的一座車站，位于[[佛山市|佛山市]][[禅城区|禅城区]]汾江南路魁奇路口北侧的地底。该站是佛山市区的地铁、[[公共汽车|公共汽车]]和[[出租车|出租车]]的交通换乘站。<ref>[http://www.cngdfs.com/station-quiqilu.htm 魁奇路站]</ref> 车站于2010年11月3日启用。

===站廳===
魁奇路站站厅中西侧被划分为收费区，周围被行人通道以及车站商店包围，站厅收费区内设有[[扶手电梯|扶手电梯]]和楼梯供乘客前往车站月台。

本站的专用电梯由于位于非付费区内，乘客使用时需要工作人员协助使用。

商店及自助服务方面，本站现时设有便利店和面包店，以及中国银行自動櫃員機、售卖机以及好易自助终端等设施。

===月台===
广佛线設有一個[[島式月台|島式月台]]，位於汾江南路地底。佛山地铁2号线则设有两个侧式月台，位于魁奇一路地底。
另外，广佛地铁站台以南设有交叉渡线，在开通初期以本站为终点站时，会利用该渡线折返所有列车。而在广佛地铁二期开通后，本站成为中途站，日常运营不再利用该渡线折返列车，继续往南开往新城东站。

== 利用状况 ==
本站周围都是住宅区，亦有不少位于本站南侧的居民通过公交等方式接驳本站，故本站主要客流以工作日早晚通勤客流为主，其他时段客流较低，令本站显得十分冷清。

== 历史 ==
本站的命名来源于附近的魁奇一路。

2002年廣佛地鐵的走向初步確定，魁奇路站確定為廣佛地鐵佛山段的其中一個中途站。2007年6月28日，廣佛地鐵首通段正式動工。

2010年10月28，29日，本站隨廣佛地鐵首通段全線開放給廣佛兩市的市民進行試乘。11月3日，廣佛地鐵首通段開通試運營，本站正式啟用。

== 未来发展 ==
{{main|佛山地铁2号线}}
佛山地铁2号线将设置有魁奇路站，并与广佛地铁换乘。

根据现有资料显示，2号线车站将位于现有广佛线车站上方，采用侧式月台，與廣佛線站廳同層<ref>[http://219.136.222.219/servlet/WzDownLoadFile?id=zhen5923ffa9899c 广佛无线招标文件(会签稿)]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2號線開通後站廳將會被路軌分割為南北兩個站廳，而站厅东侧将会扩建，增加换乘设施。
