'''魁北克市'''（{{lang-fr|'''Ville de Québec'''}}，{{lang-en|'''Quebec City'''}}），又称'''魁北克城'''，是一座位於[[加拿大|加拿大]][[魁北克|魁北克]]南部的城市，[[人口|人口]]约70多万，是[[魁北克省|魁北克省]][[首府|首府]]。主要使用[[法语|法语]]。魁北克市是[[墨西哥|墨西哥]]以北唯一的留有古老[[城墙|城墙]]的[[北美洲|北美洲]]城市。當中的[[魁北克歷史城區|魁北克歷史城區]]也是[[联合国教科文组织|联合国教科文组织]][[世界文化遗产|世界文化遗产]]（1985年）。

== 地理位置 ==
魁北克市位于圣劳伦斯谷地，整个城市坐落于[[圣劳伦斯河|圣劳伦斯河]]北边。控制[[新法蘭西|新法蘭西]]的水路入口，因而有'''美洲的[[直布羅陀|直布羅陀]]'''之稱。下部在河边，上部在戴蒙岬角（Cap Diamant）上；目前两部之间有路、阶梯和[[缆索铁路|缆索铁路]]。距魁北克最大城市[[蒙特利尔|蒙特利尔]]约275公里。

== 人口 ==
魁北克市地区710,700（2004年统计数字），这个数字包括了郊区城市人口。市区人口约169,076（2001年）。大部分说法语或者英法双语。[[白人|白人]]人口占97.3%，信[[天主教|天主教]]人口占91.2%。

== 简史 ==
[[File:Quebec_city_view_2005-02-14.JPG|left]]
此地的原始居民为[[印第安|印第安]]各民族（统称为[[原住民|原住民]]），这些民族在至少10,000年前便在加拿大居住。

[[法国|法国]]探险家[[萨缪尔·德·尚普兰|萨缪尔·德·尚普兰]]于1608年建立魁北克市，此地成为法国[[殖民地|殖民地]][[新法兰西|新法兰西]]的首府。在英法[[七年战争|七年战争]]中的1759年在魁北克市[[亚伯拉罕平原|亚伯拉罕平原]]战役，法国守军战败投降，魁北克市以及新法兰西的相当大的一部分（见[[1763年巴黎条约|1763年巴黎条约]]）从此成为[[英国|英国]]殖民地。
