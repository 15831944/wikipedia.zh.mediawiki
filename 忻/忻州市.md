'''忻州市'''是[[中华人民共和国|中华人民共和国]][[山西省|山西省]]下辖的[[地级市|地级市]]，位于山西省中北部，西隔[[黄河|黄河]]与[[陕西|陕西]]、[[内蒙古|内蒙古]]相望，东临[[太行山|太行山]]与[[河北省|河北省]]接壤。

== 歷史 ==
[[File:Tomb_of_Northern_Qi_Dynasty_in_Jiuyuangang,_Xinzhou,_Mural_25.jpg|thumb]]中的狩猎图壁画]]
忻州地区，[[春秋|春秋]]时大部分属晋地，[[战国|战国]]属[[趙國|趙]]，[[秦代|秦]][[漢朝|漢]]属太原，雁门，太平等郡，[[隋朝|隋]]為新興郡、雁門郡，[[唐代|唐]]、[[五代|五代]]、[[宋朝|宋]]為忻州雁門郡，[[金朝|金]]、[[元朝|元]]、[[明朝|明]]、[[清朝|清]]為忻州、代州、保德州，[[民国|民国]]初属雁门道，後分别属山西第一、第二专员公署。[[中華人民共和國|中華人民共和國]]成立後成立[[忻縣專區|忻縣專區]]。1958年底与[[雁北专区|雁北专区]]合并为[[晋北专区|晋北专区]]，1961年复置[[忻县专区|忻县专区]]。1970年改专区为地区，1978年改地區為行政公署。2000年6月14日，經[[中華人民共和國國務院|国务院]]批准：撤銷[[忻州地區|忻州地區]]和縣級忻州市，設立地級忻州市。

== 自然资源 ==
忻州已探明储量的矿产达50余种，其中[[煤炭|煤炭]]800亿吨，占全省的13%；[[铝土矿|铝土矿]]2.5亿吨，占全省的27%；[[铁矿|铁矿]]18亿吨，占全省的48%；[[金矿|金矿]]1.9万公斤，占全省的74%；[[钛|钛]]矿7600万吨，为全国第二大矿床。

== 经济 ==
忻州是一个欠发达地区，全市有11个[[国家扶贫开发工作重点县|国家扶贫开发工作重点县]]。依托丰富的能源矿产资源，全市初步建立起[[煤炭|煤炭]]、[[电力|电力]]、[[冶金|冶金]]、[[化工|化工]]、装备制造等工业体系。煤炭资源整合后保留矿井81座。
