'''瓦列里·伊万诺维奇·托卡列夫'''（{{lang-ru|Валерий Иванович Токарев}}，{{bd|1952年|10月29日|}}）是[[俄罗斯|俄罗斯]]已退役[[宇航员|宇航员]]。
==早年==
1952年10月29日出生于[[阿斯特拉罕州|阿斯特拉罕州]][[卡普斯京亚尔靶场|卡普斯京亚尔靶场]]，父亲伊万·帕夫洛维奇·托卡列夫于1972年因车祸去世<ref>{{cite web |title=Biographical Data |url=https://web.archive.org/web/20170212055414/https://www.jsc.nasa.gov/Bios/htmlbios/tokarev.html |publisher=[[NASA|NASA]] |accessdate=2019-04-15}}</ref>。他21岁从[[斯塔夫罗波尔|斯塔夫罗波尔]]高等航空军事学校毕业，成为一名[[歼击机|歼击机]][[飞行员|飞行员]]。1983年，他以优异成绩毕业于[[阿斯特拉罕州|阿斯特拉罕州]]试飞员培训中心，并于1987年通过空军选拔宇航员的医疗检查。之后接受了多项严格培训，掌握了40多种飞机和直升机的驾驶技术，最终于1997年正式加入[[加加林宇航员培训中心|加加林宇航员培训中心]]的宇航员队伍，成为俄罗斯第91位宇航员，也是世界上第388位宇航员<ref>{{cite web |title=美俄宇航员护航世界第三名太空游客 |url=http://tech.sina.com.cn/d/2005-10-02/1448733481.shtml |publisher=[[新浪|新浪]]科技 |archiveurl=https://web.archive.org/web/20190415054444/http://tech.sina.com.cn/d/2005-10-02/1448733481.shtml |archivedate=2019-04-15 |date=2005-10-02}}</ref>。

===STS-96===
1999年5月27日[[美国东部时间|美国东部时间]]上午6时49分，托卡列夫等宇航员在美国[[肯尼迪航天中心|肯尼迪航天中心]]乘坐[[发现号航天飞机|发现号航天飞机]]飞往国际空间站，完成了自己的首次太空飞行。此次任务时长为9日19时13分<ref>{{cite web |title=Mission Archives STS-96 |url=https://www.nasa.gov/mission_pages/shuttle/shuttlemissions/archives/sts-96.html |publisher=NASA |accessdate=2019-04-15}}</ref>。

2005年9月3日，他作为联盟TMA-7号飞船的指令长、[[远征12|远征12]]任务飞行工程师，与美国宇航员[[威廉·麦克阿瑟|威廉·麦克阿瑟]]搭乘联盟TMA-7号载人飞船抵达国际空间站<ref>{{cite web |title=俄联盟飞船30日升空 宇航员将在轨工作178天 |url=http://wap.xjtu.edu.cn/info/1012/11175.htm |publisher=[[西安交通大学|西安交通大学]]科技在线 |archiveurl=https://web.archive.org/web/20190415065353/http://wap.xjtu.edu.cn/info/1012/11175.htm |archivedate=2019-04-15 |date=2006-03-18}}</ref>。

莫斯科时间2005年11月7日18时32分（北京时间23时32分），他和麦克阿瑟走出国际空间站，开始舱外任务。他们在空间站美国舱段外安装了一个摄像机并在站外拆除了一些设备。在完成全部任务后于莫斯科时间23时54分（北京时间8日凌晨4时54分）返回空间站。整个过程共持续5小时22分钟<ref>{{cite web |author=魏忠杰 |title=俄美两名宇航员成功在空间站外安装摄像机 |url=http://world.people.com.cn/GB/41218/3837116.html |publisher=[[新华社|新华社]] |archiveurl=https://web.archive.org/web/20190415045230/http://world.people.com.cn/GB/41218/3837116.html |archivedate=2019-04-15 |location=[[莫斯科|莫斯科]] |date=2005-11-08}}</ref>。

11月18日，他和麦克阿瑟驾驶[[联盟TMA-7|联盟TMA-7]]载人飞船，将其从[[码头号对接舱|码头号对接舱]]移到[[曙光号功能货舱|曙光号功能货舱]]上，以保证顺利从码头号对接口钻出进行太空行走任务。2005年3月20日，他和麦克阿瑟驾驶飞船离开空间站曙光号功能货舱，与[[星辰号服务舱|星辰号服务舱]]重新完成对接，为空间站4月1日迎接[[联盟TMA-8|联盟TMA-8]]载人飞船让出了曙光号功能货舱对接口<ref>{{cite web |title=国际空间站上的俄“联盟”飞船重新“靠港” |url=http://www.taikongmedia.com/Item/Show.asp?m=1&d=14987 |publisher=中国太空网 |archiveurl=https://web.archive.org/web/20190415062504/http://www.taikongmedia.com/Item/Show.asp?m=1&d=14987 |archivedate=2019-04-15 |date=2006-03-21}}</ref>。

莫斯科时间2006年4月9日凌晨0时28分，乘坐有宇航员托卡列夫、麦克阿瑟和太空游客[[马库斯·蓬蒂斯|马库斯·蓬蒂斯]]的联盟TMA-7载人飞船与国际空间站星辰号服务舱分离，约2个半小时后开始减速，20分钟后飞船轨道舱、仪器舱和返回舱分离并进入地球大气层。飞船返回舱在距离地面10公里处打开[[降落伞|降落伞]]，最终于凌晨3时47分（北京时间7时47分）在[[哈萨克斯坦|哈萨克斯坦]][[阿尔卡雷克|阿尔卡雷克]]市东北部实现软着陆<ref>{{cite web |author1=魏忠杰 |title=“联盟TMA-7”载人飞船返航 降落在哈萨克斯坦境内 |url=http://www.cnsa.gov.cn/n6758823/n6759010/c6775978/content.html |publisher=[[国家航天局|国家航天局]] |archiveurl=https://web.archive.org/web/20190415063712/http://www.cnsa.gov.cn/n6758823/n6759010/c6775978/content.html |archivedate=2019-04-15 |date=2006-04-10}}</ref>。
