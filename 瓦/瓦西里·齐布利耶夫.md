'''瓦西里·瓦西里耶维奇·齐布利耶夫'''（{{lang-ru|Васи́лий Васи́льевич Цибли́ев}}，{{bd|1954年|2月20日|}}）是[[俄罗斯|俄罗斯]]已退役[[宇航员|宇航员]]，曾2次前往[[和平号空间站|和平号空间站]]执行任务，共进行了6次太空行走。
==早年==
1954年2月20日出生于苏联[[克里米亚|克里米亚]][[基洛夫斯基区|基洛夫斯基区]]奥列希夫卡（Ореховка）。1975年毕业于[[哈尔科夫|哈尔科夫]]空军学院，之后在空军服役<ref name=bio>{{cite web |title=Летчик-космонавт РФ |url=https://web.archive.org/web/20071102043407/http://www.gctc.ru/biogr/cibleev.html |publisher=[[加加林宇航员培训中心|GCTC]] |accessdate=2019-04-20 |language=ru}}</ref>。

==宇航员生涯==
1987入选宇航员分队。1987年12月到1989年在[[加加林宇航员培训中心|加加林宇航员培训中心]]接受基础训练。1989年8月30日获得测试宇航员资格<ref name=bio/>。
===第一次===
1993年7月1日至1994年1月14日在和平号空间站执行任务。期间一共进行了5次太空行走，总时长共计14时13分<ref name=bio/>。

===第二次===
1997年2月10日至1997年8月14日，齐布利耶夫担任“前往和平号空间站第23次远征”任务的指挥长<ref>{{cite web |title=Vasily Tsibliev |url=https://spaceflight.nasa.gov/history/shuttle-mir/people/p-c-tsibliev.htm |publisher=NASA |accessdate=2019-04-20 |date=2013-08-23}}</ref>。4月29日，他与美国同行[[杰瑞·林恩格|杰瑞·林恩格]]身着俄制太空服，一前一后走出空间站，展开联合太空行走<ref>{{cite web |title=探秘俄罗斯“星城”：美国宇航员第二个家 |url=http://news.ifeng.com/world/3/200811/1115_2592_879325_1.shtml |publisher=新华网 |archiveurl=https://web.archive.org/web/20190420144312/http://news.ifeng.com/world/3/200811/1115_2592_879325_1.shtml |archivedate=2019-04-20 |date=2008-11-15}}</ref>。

6月25日，他在引导[[進步號太空船|進步號]]货运飞船与和平号空间站对接时，未控制速度，使飞船撞向了空间站的[[光谱号|光谱号]]增压舱。空间站的一块[[太阳能电池板|太阳能电池板]]阵列被撞掉<ref>{{Cite book|title = This New Ocean: The Story of the First Space Age|url = https://books.google.com/books?id=TLKCmJSFJnYC|publisher = [[兰登书屋|Random House Publishing Group]]|date = 2010-09-29|isbn = 9780307765482|language = en|first = William E.|last = Burrows}}</ref> ，机舱[[压力|压力]]开始降低<ref>{{Cite book|title = Surviving the Extremes: A Doctor's Journey to the Limits of Human Endurance|url = https://books.google.com/books?id=2czSJB5BSHoC|publisher = Macmillan|date = 2004-01-20|isbn = 9781429976114|language = en|first = Kenneth|last = Kamler}}</ref>。

[[亚历山大·拉祖特金|拉祖特金]]等队员迅速切断了与损坏舱体的连接，空间站内部压力开始稳定。但是太阳能[[电池|电池]]装置的损坏使[[散热器|散热器]]保护盖移位。他们关闭了[[陀螺|陀螺]]控制系统，并人工操作大型姿态控制[[推力|推力]]器，使空间站实现机动，让其他舱体上的电池板朝向太阳以获取[[电能|电能]]。最终空间站成功脱险<ref>{{cite web |title=十次大胆的太空救援 |url=http://www.sohu.com/a/118843396_418353 |publisher=搜狐 |accessdate=2019-04-20 |archiveurl=https://web.archive.org/web/20180102184157/http://www.sohu.com/a/118843396_418353 |archivedate=2018-01-02 |date=2016-11-13}}</ref>。

==退役==
1997年获得莫斯科国立测量学和地图学大学学位。1998年6月19日从宇航员退役<ref>{{cite web |title=Циблиев, Василий Васильевич |url=http://www.warheroes.ru/hero/hero.asp?Hero_id=2282 |publisher=Герои страны |accessdate=2019-04-20 |language=ru}}</ref>。2000年4月6日开始担任[[加加林宇航员培训中心|加加林宇航员培训中心]]副主任<ref name=bio/>。
