'''瓦列里·格里戈里耶维奇·科尔尊'''（{{lang-ru|Вале́рий Григо́рьевич Ко́рзун}}，{{bd|1953年|3月5日|}}）是[[俄罗斯|俄罗斯]]已退役[[宇航员|宇航员]]，曾两次前往太空，四次进行太空行走任务。
==早年==
1953年3月5日出生于苏联[[罗斯托夫州|罗斯托夫州]][[紅蘇林|紅蘇林]]。1974年毕业于[[卡恰|卡恰]]高等軍事航空學校，毕业后在[[苏联空军|苏联空军]]服役，历任飞行员，高级飞行员，空军中队指挥官。他能熟练驾驶6种飞机，累计飞行时间超过1500小时<ref name=ru>{{cite web |title=Valeriy Grigoryevich KORZUN |url=https://www.energia.ru/en/iss/iss05/korzun.html |publisher=[[科罗廖夫能源火箭航天集团|SP Korolev RSC Energia]] |accessdate=2019-04-21}}</ref>。1987年获得[[加加林空军学院|加加林空军学院]]授予的指挥官资格，同年入选[[加加林宇航员培训中心|加加林宇航员培训中心]]测试宇航员候选人。1987年12月开始接受基础航空飞行培训，1989年6月获得测试宇航员资格<ref name=nasabio>{{cite web |title=Valery Grigoryevich Korzun |url=https://web.archive.org/web/20160801084441/http://www.jsc.nasa.gov/Bios/htmlbios/korzun.html|date=2002 |publisher=NASA |accessdate=2019-04-20}}</ref>。
==宇航员生涯==
1991年5月20日至7月10日，他接受了“和平号空间站远征队”的专业培训。然而由于计划的改变和队员的重组，训练被迫终止。1991年10月至1992年3月，他作为“和平号空间站远征11”任务的备用人员接受了培训。1995年10月至1996年7月，他接受了俄[[美国|美]][[法国|法]]“和平号空间站远征22”任务备用指挥长专门培训。后由于主指挥长生病，他被任命为该任务指挥长<ref name=ru/>。
===EO-22===
1996年8月17日至1997年3月2日，他担任“和平号空间站远征22”任务指挥长，在[[和平号空间站|和平号空间站]]度过了196天17小时，期间一共进行了两次太空行走<ref>{{cite web |title=День рождения Героя России, летчика-космонавта РФ Валерия Корзуна |url=http://www.gctc.ru/print.php?id=1871 |publisher=GCTC |archiveurl=https://web.archive.org/web/20190421063127/http://www.gctc.ru/print.php?id=1871 |archivedate=2019-04-21 |date=2019-03-05|language=ru}}</ref>。

===远征5===
2002年6月5日21时23分，[[奋进号航天飞机|奋进号航天飞机]]从美国[[佛罗里达州|佛罗里达州]][[肯尼迪航天中心|肯尼迪航天中心]]发射升空，搭载着分别来自俄美法3个国家的7名机组人员，其中科尔尊和[[谢尔盖·特列晓夫|谢尔盖·特列晓夫]]、美国女宇航员[[佩吉·惠特森|佩吉·惠特森]]是[[远征5|远征5]]任务队员<ref>{{cite news |url=http://www.cas.cn/xw/gjdt/200906/t20090608_611769.html|title=美国“奋进”号航天飞机7日与国际空间站对接 |archivedate=2019-04-21 |agency=[[新华网|新华网]] |publisher=[[中国科学院|中国科学院]] |date=2002-06-07 |archiveurl=https://web.archive.org/web/20190421051901/http://www.cas.cn/xw/gjdt/200906/t20090608_611769.html}}</ref>。

2002年10月11日，科尔尊等俄罗斯宇航员在空间站接受了[[俄罗斯联邦国家统计局|俄罗斯联邦国家统计局]]的[[人口普查|人口普查]]，他们填写了[[复印|复印]]的人口普查表格，并回答了普查员的有关提问。这些表格最后会在博物馆展出。该普查是自1991年[[苏联解体|苏联解体]]以来俄罗斯的首次人口普查<ref>{{cite web |title=俄罗斯3名宇航员在太空飞行中接受人口普查 |url=http://mil.news.sina.com.cn/2002-10-12/87850.html |publisher=新华社 |archiveurl=https://web.archive.org/web/20190421050353/http://mil.news.sina.com.cn/2002-10-12/87850.html |archivedate=2019-04-21 |date=2002-10-12}}</ref>。
