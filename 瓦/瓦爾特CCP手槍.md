'''瓦爾特CCP'''（{{Lang-en|'''C'''oncealed '''C'''arry '''P'''istol}}，意為：{{Link-en|隱蔽攜帶|Concealed carry}}手槍）是一把由[[德国|德国]]槍械製造商[[卡尔·瓦尔特运动枪|卡尔·瓦尔特运动枪]][[有限公司|有限公司]]為了民用市場而特別研製、生產和推出的一款具有半自動扳機和阻鐵系統、適合隱藏下隨身攜帶的[[半自動手槍|半自動手槍]]，發射[[9×19mm魯格彈|9×19毫米]][[手枪|手枪]][[子彈|子彈]]。<ref name="HGM-WaltherCCP">{{cite web|last1=Tarr|first1=James|title=Walther CCP 9mm Review|url=http://www.handgunsmag.com/reviews/compacts/walther-ccp-9mm-review/|website=handgunsmag.com|accessdate=2017-02-16}}</ref><ref name="WaltherUSA">{{cite web | url=http://www.waltherarms.com/ccp/ |title=Walther CCP |publisher=Walther Firearms |date=2014 |access-date=2014-10-08}}</ref>

== 歷史 ==
2014年3月，德國卡爾·華瑟運動槍有限公司推出該槍<ref name="A4S">{{cite web | url=http://www.all4shooters.com/de/Spezial/Messen-2014/IWA-2014/IWA-2014-Kurzwaffen/IWA-2014-Walther-CCP-Kurzwaffe/ |title=IWA 2014 / Neuvorstellung von WALTHER - die halbautomatische Pistole CCP in 9 x19 mm |author=all4shooters.com Team |publisher=all4shooters.com |date=2014-03-10 |access-date=2014-10-08}}</ref>。最顯著的特點，與[[瓦爾特P99手槍|瓦爾特P99]]或[[瓦爾特PPQ手槍|瓦爾特PPQ]]無擊錘與相應的待擊一樣，在於CCP採用了完全內部化的平移擊針擊發。

== 設計細節 ==
瓦爾特CCP藉由[[反沖作用_(槍械)|氣體延遲反沖式槍機]]操作，利用從點燃子彈所產生的高壓燃氣的壓力，將其引導至槍管前方的一小孔，進入有活塞的小口徑長圓筒，以減緩和延遲槍機的向後開放運作。這種設計與[[HK_P7手槍|HK P7]]的設計幾乎完全相同。只是，瓦爾特把這個系統叫作「軟[[後座力|後座力]]」（{{Lang-en|Softcoil}}）。與[[瓦爾特P99手槍|瓦爾特P99]]或[[瓦爾特PPQ手槍|瓦爾特PPQ]]不同的是，CCP具有超薄的手動拇指保險；另外還有一個額外的內部保險用作[[撞針|擊針]]阻塊。符合[[人因工程学|人體工程學]]的握把是PPQ的握把的縮短版本<ref name="A4S"/>。[[手槍套筒|套筒]]是由[[钢|钢]]製成，有黑色或銀色兩種可選。可拆卸式[[彈匣|彈匣]]容量為8發。CCP亦在套筒下、[[扳機護環|扳機護圈]]前方的{{Link-en|防塵蓋|Dust cover (firearms)}}整合了一條[[皮卡汀尼導軌|MIL-STD-1913戰術導軌]]以安裝各種[[戰術燈|戰術燈]]、[[雷射瞄準器|雷射瞄準器]]和其他戰術配件。其總長度為163毫米（6.42英吋）、寬度為30毫米（1.18英吋）、高度為130毫米（5.12英吋），而槍管長度則是90毫米（3.54英吋）。

與CCP的規模相比，5.5磅的扳機扣力和0.27英寸的扳機行程都相對較高。<ref name="A4S"/>

不完全分解或完全分解時都需要使用為新型CCP而提供的專屬工具，但據說可以使用其他設備代替。瓦爾特已經準備了一份“測試”彈藥清單，並提出了最佳功能的握把說明，可根據要求提供給手槍的擁有者，並附有保密／專有聲明。

== 召回 ==
2017年3月，瓦爾特發布了召回所有CCP手槍的通知書，因為他們聲稱CCP的某種潛在缺陷可能會導致該槍如掉落地上<ref>{{cite web|title=Walther USA CCP Recall|url=http://www.waltherarms.com/walther-resources/ccp-recall/|accessdate=2018-01-21|archiveurl=https://web.archive.org/web/20170726123410/http://www.waltherarms.com/walther-resources/ccp-recall/|archivedate=2017-07-26|deadurl=yes}}</ref>，無論手動保險上鎖與否，都會擊發進而走火。結果已經令瓦爾特發布了一個自願性召回，並會將所有受影響的CCP設計升級以消除掉落走火的風險。
