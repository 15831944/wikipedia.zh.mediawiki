'''瓦拉明'''（[[波斯语|波斯语]]：'''{{lang|fa|ورامین}}'''）是[[伊朗|伊朗]]北部[[德黑兰省|德黑兰省]][[瓦拉明郡|瓦拉明郡]]的首府，根据2011年的普查，人口为218,991人。

==历史==
瓦拉明被认为是在[[波斯古经|波斯古经]]中提到的城市，Varena。

==工业==
瓦拉明糖精炼厂建于1934-1935年，是伊朗和中东第一家炼油厂。近年来，由于原料不足，已经减少了生产。<ref>{{Cite web |url=http://www.parsine.com/fa/news/57078/قديمي-ترين-کارخانه-قند-ايران-تعطيل-شد|script-title=fa:قديمي ترين کارخانه قند ايران تعطيل شد. |work=www.parsine.com |language=fa}}</ref><ref>{{Cite web |url=http://www.tebyan.net/Social/SevenContinents/IranTour_Geographical_/2010/7/8/130041.html|script-title=fa:یادگارهای جناب نیکلای|work=www.tebyan.net |language=fa}}</ref>

瓦拉明榨油厂建于1938-1939年，是伊朗第一家生产植物油的厂家。<ref>{{Cite web|url=http://www.khoec.com/Default.aspx?tabid=368|script-title=fa:روغنکشی شماره ی یک ورامین|work=www.khoec.com|language=fa|deadurl=yes|archiveurl=https://www.webcitation.org/6CJFkA7tv?url=http://www.khoec.com/Default.aspx?tabid=368|archivedate=2012-11-20}}</ref>

==手工艺==
[[File:Varamin_Carpet_8.jpg|thumb]]
瓦拉明地毯是著名的地毯的一种，具有重复图案的几何图案。许多地毯专家将其视为纯粹的波斯地毯。<ref>{{Cite web |url=http://www.little-persia.com/?action=Varamin|title= Varamin carpets in Little Persia |work=www.little-persia.com }}</ref>

==教育==
[[伊斯兰阿扎德大学|伊斯兰阿扎德大学]]设有瓦拉明-皮什瓦分部，拥有86个专业和15000名学生。<ref>http://iauvaramin.ac.ir/fa/?page_id=227</ref>
