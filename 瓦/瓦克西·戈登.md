'''瓦克西·戈登'''（{{lang-en|'''Waxey Gordon'''}}，本名'''艾文·懷茲勒'''（'''Irving Wexler'''），{{bd|1888年|1月19日|1952年6月24日}}）是一名美國黑幫，專門從事{{link-en|朗姆酒飛車|Rum-running|非法私酒}}和非法賭博。在[[禁酒時期|禁酒令]]期間是[[阿諾·羅斯汀|阿諾·羅斯汀]]的一名合夥人，他在羅斯汀死後被捲入權力鬥爭中。羅斯汀的伙伴[[查理·盧西安諾|查理·盧西安諾]]和[[邁爾·蘭斯基|邁爾·蘭斯基]]向當局提供證據，導致他被監禁十年。

== 流行文化 ==
瓦克西·戈登是[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》中的次要角色，由[[尼克·桑多|尼克·桑多]]飾演他。<ref>{{Cite web |url = https://www.thoughtco.com/who-was-waxey-gordon-2076724 |title = Waxey Gordon |author = Ariela Pelaia |date = 2016-01-12 |publisher = thoughtco.com |language = en}}</ref>

1960年電視劇《{{link-en|鐵面無私 (1959年電視劇)|The Untouchables (1959 TV series)|鐵面無私}}》的集數"The Waxey Gordon Story"中，由{{link-en|尼希米·佩索夫|Nehemiah Persoff}}飾演戈登。<ref>{{Cite web |url = http://www.imdb.com/title/tt0737848/ |title = The Untouchables (1959–1963): The Waxey Gordon Story |publisher = IMDb |language = en}}</ref>
