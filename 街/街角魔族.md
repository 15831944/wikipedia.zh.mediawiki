{{日本作品|街角魔族|まちカドまぞく}}是由日本漫画家{{link-ja|伊藤出云|伊藤いづも}}创作的[[四格漫画|四格漫画]]作品，在《[[Manga_Time_Kirara_Carat|Manga Time Kirara Carat]]》2014年8月号和9月号上做客连载，並於2014年11月号起正式开始连载。
== 简介 ==
故事的舞台设定在虚构的城市“多魔市”，主要讲述突然觉醒了魔族之力的暗之魔族的末裔吉田优子与光之一族的退役魔法少女千代田桃之间的日常校园生活。

== 登场角色 ==
; {{nihongo|吉田优子|吉田 優子（よしだ ゆうこ）}}
: 本作主角，15岁，生日为9月28日，就读于樱之丘高中一年级D班<ref>单行本第一卷第8页</ref>，身材矮小，对人说话总是使用[[敬语|敬语]]，学习和运动都不擅长。
: 在某天早上突然发现自己长出了角和尾巴，才知道自己并不是人类，而是暗之魔族，而自己总是长不高和家境贫寒的原因是受到光之一族的魔法诅咒，想要解除诅咒就只有拿到魔法少女的血液，为了振兴家族和能使自己长高而挑起了打败光之魔法少女的使命。

; {{nihongo|千代田桃|千代田 桃（ちよだ もも）}}
: 与优子同校的光之魔法少女，15岁，生日为3月25日<ref>单行本第一卷角色介绍页</ref>，就读于樱之丘高中一年级A班，头上戴着魔法少女的发饰，现已退役。
: 擅长运动，也有着十分强大的肌肉力量，身体能力比常人高得多，可以徒手捏碎握力计。
: 故事开始时间前六年凭借自己魔法少女的身份拯救了世界。

; {{nihongo|莉莉丝|リリス}}
: 优子的祖先，预测年龄4000到5000岁，来自[[美索不达米亚|美索不达米亚]]的褐色皮肤的梦魔<ref>单行本第一卷第93页</ref>，具有潜入并操控他人梦境的能力，平时待在一尊雕像中。

; {{nihongo|佐田杏里|佐田 杏里（さた あんり）}}
: 优子的同班同学，跟优子关系很好，有着半长的黑发，初中跟千代田桃一起上学，目前隶属于樱之丘高中网球部。

; {{nihongo|小仓志温|小倉 しおん（おぐら しおん）}}
: 杏里的朋友，就读于一年级C班<ref>单行本第一卷第21页</ref>，戴着眼镜，头发用带子扎在脑后，创建了“诅咒与恶魔研究部”（{{lang|ja|呪術とか悪魔とかの研究部}}）。

; {{nihongo|阳夏木蜜柑|陽夏木 ミカン（ひなつき ミカン）}}
: 在千代田桃魔力减弱时前来帮助桃的魔法少女，高一学生，有着橙色的头发，原本是居住在其他地方的稳健派魔法少女，得知了樱之丘高中有魔族后就转到了樱之丘高中，也是少数对优子直呼其名的人。

; {{nihongo|吉田清子|吉田 清子（よしだ せいこ）}}
: 优子的母亲，性格顽皮，跟女儿优子很合得来，在作品中由于受到光之一族的魔法诅咒故而家境贫寒。

; {{nihongo|吉田良子|吉田 良子（よしだ　りょうこ）}}
: 优子的妹妹，自称为“良”，穿着跟姐姐优子不同的校服。
