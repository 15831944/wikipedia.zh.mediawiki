'''街亭之戰'''是[[三國|三國]]時期[[諸葛亮北伐|諸葛亮北伐]]的其中一場戰事，發生於諸葛亮第一次北伐戰爭期間，也是這次北伐戰爭中一場決定性戰事。魏[[太和_(曹叡)|太和]]二年（228年），諸葛亮[[馬謖|馬謖]]領諸軍在重要陣地[[街亭|街亭]]抵抗[[曹魏|曹魏]]將領[[張郃|張郃]]，但被張郃擊敗。蜀漢於是撤退，結束第一次北伐。

== 背景 ==
[[曹魏|魏]][[太和|太和]]二年（228年，[[蜀漢|蜀漢]][[建興_(劉禪)|建興]]六年），[[諸葛亮|諸葛亮]]率軍攻打曹魏，聲稱由[[斜谷|斜谷]]道攻取[[郿縣|郿縣]]，並派[[趙雲|趙雲]]、[[鄧芝|鄧芝]]為疑軍，據[[箕谷|箕谷]]，此軍勢引魏明帝[[曹叡|曹叡]]令大將軍[[曹真|曹真]]前去防守郿縣抵禦趙雲等之攻擊；諸葛亮則親率諸軍進攻[[祁山|祁山]]，發動第一次北伐。諸葛亮北伐令[[天水|天水]]、[[南安|南安]]和[[安定郡|安定]]三郡[[天水之乱|叛魏響應]]，[[關中|關中]]震動，震驚曹魏，[[魏明帝|魏明帝]]更親身到[[長安|長安]]督戰，並派[[張郃|張郃]]抗擊進攻祁山的諸葛亮。

== 經過 ==
諸葛亮攻祁山時，眾人都建議用宿將[[魏延|魏延]]、[[吳懿|吳懿]]等人任主将，但諸葛亮不聽，任命自己的參軍[[馬謖|馬謖]]領導諸軍，後於軍事重地街亭防禦曹魏將領[[張郃|張郃]]的進攻。諸葛亮下令馬謖守住[[街亭|街亭]]，殊不知馬謖違反諸葛亮節度，舉措煩擾，更加捨棄水源，選擇登上[[南山|南山]]據守而非佔據山下的城鎮；禆將軍[[王平_(三国)|王平]]曾多番規勸，但馬謖都不接納。張郃到後，包圍山上，又斷絕山上馬謖軍的水源，並且大舉進擊，大破馬謖軍，士卒四散，潰不成軍。此時王平命自己所領的軍隊鳴鼓自守，張郃懷疑有伏兵，不敢進逼，王平得以有時間收拾殘軍，並率領敗軍撤回。馬謖失守街亭後，諸葛亮失去重要據點，進退無據，無法再戰，於是遷[[西縣|西縣]]一千多戶撤回[[漢中|漢中]]。後馬謖與將領[[李盛|李盛]]、[[張休_(蜀漢)|張休]]等人被處死，[[黃襲|黃襲]]被奪兵。

== 影響 ==
*街亭之戰失敗令第一次北伐功敗垂成，日後[[蜀漢|蜀漢]]數次北伐未能獲得比本次北伐一度令三郡投降還要更大的戰果。
*本來自劉備征孫權失敗而病逝後，蜀漢都沒有甚麼侵擾魏蜀邊界的行動，曹魏都沒有作出甚麼準備，以為蜀漢已無力侵邊；故此當諸葛亮北伐的消息傳來時，令朝野震動，更加令三郡叛離。戰後曹魏亦加強了對蜀漢的防衛，包括曹真預計諸葛亮會攻[[陈仓城|陳倉]]，於是命將軍[[郝昭|郝昭]]守治陳倉城，以致年底（即229年年初）諸葛亮[[陈仓之围|進圍陳倉]]時無功而還。及後亦因曹魏多作防備而令北伐難以斬獲。
