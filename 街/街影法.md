[[File:XRL_WKT_(21).JPG|缩略图]][[渡船街_(香港)|渡船街]]的[[文華新村|文華新村]]（八文樓）]]
'''街影法'''，是[[英屬香港|港英政府]]在1969年[[香港|香港]]《建築物（規劃）規例》中加入的建築要求，確保街道日照和空氣流通，不過此限制於1987年便取消。

== 背景 ==
[[File:Setback_tower_options,_1916.jpg|缩略图]]1916年對街道建築的限制]]
香港曾經歷過不少[[香港瘟疫列表|瘟疫]]、如19世紀末的[[1894年香港鼠疫|鼠疫]]等，港英政府明白公共衛生的重要性，而[[1960年代香港|1960年代]]香港剛轉型作轉口港，被列為疫埠會對經濟以至整個社會都會造成衝擊，因此由[[英国|英國]]引入[[街影|街影]]的建築概念，促成香港街影法的規劃，保持公共空間的日照和通風。<ref>{{Cite web|url=http://hk.on.cc/hk/bkn/cnt/news/20170727/bkn-20170727060038058-0727_00822_001.html|title=搜查線：街影法係咩？　斜面大廈關佢事|accessdate=2018-09-24|author=|date=2017-07-27|publisher=東方日報}}</ref>

== 內容 ==
街影法是針對20層以下的建築物的'''街影投射面積'''（street shadow area）<ref>{{Cite web|url=http://www.popd.hk/%E9%A6%99%E6%B8%AF%E6%80%AA%E6%A8%93%E5%A4%A7%E5%85%A8%E3%80%80%E6%8F%AD%E8%A7%A3%E8%BF%B7%E6%80%9D%EF%BC%88%E4%B8%8A%EF%BC%89/|title=本土樓宇奇觀（五）樓梯樓你見過未？|accessdate=2018-09-24|author=Angel|date=2016-09|publisher=metro Pop}}</ref>，不同情況有不同限制，內容大致如下：

* 大廈與地面之間要呈特定的'''日光角度斜面'''（daylight plane）
* 以[[街道|街道]]中心作為起點，向兩邊的樓宇劃一條76度角向上斜的線

== 斜面樓 ==
[[File:杜誌台_Tonnochy_Towers_灣仔_Wan_Chai_謝斐道_Jaffe_Road_杜老誌道_Tonnochy_Road,_2018.jpg|缩略图]][[謝斐道|謝斐道]]杜誌台]]
'''斜面樓'''，是對[[香港|香港]]於1969至1987年實施街影法時所衍生的一種建築物類型的泛稱。

此類建築物為符合當時規定同時用盡[[地積比率|地積比率]]，會逐漸遞減頂樓的[[建筑面积|建築面積]]大小，形成梯級的外觀。

== 常見地點 ==
斜面樓在[[香港|香港]][[1990年代香港|1990年]]代前人口較密集的[[香港市區|市區]]較普遍：

* [[香港島|港島]]：[[灣仔|灣仔]]丶[[西營盤|西營盤]]
* [[九龍|九龍]]：[[深水埗|深水埗]]丶[[旺角|旺角]]

== 香港現況 ==
[[File:West_Kowloon_Olympic_Station_Buildings_20110510.jpg|缩略图]]是眾多地區其中最多[[屏風樓|屏風樓]]的地方]]
香港的樓宇開始以高樓形式興建，建築限制在1987年廢除，繼而進入「[[屏風樓|屏風樓]]」的建築年代，發展商都在有限的[[容积率|地積比率]]下以高度增加樓宇面積。
