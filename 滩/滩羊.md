'''滩羊'''，中国特有的一个[[裘皮|裘皮]]用[[绵羊|绵羊]]品种，由[[蒙古羊|蒙古羊]]经长期的自然选择和人工选育而形成的一个长脂尾、粗毛型的品种。滩羊头部有褐色或黑色花斑，毛细长而拳曲，呈波浪形，躯干多为白<ref>{{cite book |editor=郝迟，盛广智，李勉东 |title=汉语倒排词典 |date=1987-09 |publisher=黑龙江人民出版社}}</ref>。滩羊一般是一年一羔或两年三羔<ref>{{cite web |title=滩羊，柳暗花明谱新曲 |url=http://www.nx.xinhuanet.com/jdwt/ty/index.htm |website=新华网 |accessdate=2018-07-27 |archive-url=https://web.archive.org/web/20061213134534/http://www.nx.xinhuanet.com/jdwt/ty/index.htm |archive-date=2006-12-13 |dead-url=no }}</ref>。滩羊适于荒漠、半荒漠干旱草原，主产于[[宁夏|宁夏]]的[[盐池县|盐池]]、[[同心县|同心]]和[[灵武市|灵武]]等地区，在[[甘肃|甘肃]]、[[陕西|陕西]]、[[内蒙古|内蒙古]]三省区与宁夏毗邻的地区也有出产。<ref name="tian">{{cite journal |author1=田晓梅 |title=滩羊的保种选育技术 |journal=中国畜牧业 |date=2010 |issue=15 |pages=42-43}}</ref>2006年，滩羊被[[中華人民共和国农业部|中華人民共和国农业部]]列入《[[国家级畜禽遗传资源保护名录|国家级畜禽遗传资源保护名录]]》，属于国家级畜禽遗传资源保护品种<ref>{{cite web |title=中华人民共和国农业部公告  第662号 |url=http://www.gov.cn/gzdt/2006-06/10/content_306243.htm |website=中国政府网 |accessdate=2018-07-27 |archive-url=https://web.archive.org/web/20160602221153/http://www.gov.cn/gzdt/2006-06/10/content_306243.htm |archive-date=2016-06-02 |dead-url=no }}</ref>。

滩羊所产一月龄羔羊裘皮<ref name="cnki">{{cite web |title=宁夏滩羊 |url=http://xuewen.cnki.net/R2006050690002178.html |website=cnki学问 |publisher=宁夏百科全书 |accessdate=2018-07-25 |archive-url=https://web.archive.org/web/20180725122830/http://xuewen.cnki.net/R2006050690002178.html |archive-date=2018-07-25 |dead-url=no }}</ref>，惯称“二毛皮”，与[[枸杞|枸杞]]、[[甘草|甘草]]、[[贺兰石|贺兰石]]和[[发菜|发菜]]并称“宁夏五宝”。2014年，[[二毛皮制作技艺|二毛皮制作技艺]]被确定进入“国家级非物质文化遗产代表性项目名录”<ref>{{cite web |title=国务院关于公布第四批国家级 非物质文化遗产代表性项目名录的通知 |url=http://www.gov.cn/zhengce/content/2014-12/03/content_9286.htm |website=中国政府网 |accessdate=2018-07-26 |archive-url=https://web.archive.org/web/20150111080004/http://www.gov.cn/zhengce/content/2014-12/03/content_9286.htm |archive-date=2015-01-11 |dead-url=no }}</ref>。宁夏盐池有“中国滩羊之乡”之城，“盐池滩羊”在2016年被[[国家质检总局|国家质检总局]]批准为[[地理标志保护产品|地理标志保护产品]]<ref>{{cite web |author1=杨之汀 |title=国家质检总局批准盐池滩羊为地理标志保护产品 |url=http://news.eastday.com/eastday/13news/auto/news/china/20160222/u7ai5319275.html |website=东方网 |accessdate=2018-07-25 |archive-url=https://web.archive.org/web/20180725153525/http://news.eastday.com/eastday/13news/auto/news/china/20160222/u7ai5319275.html |archive-date=2018-07-25 |dead-url=no }}</ref>。

== 名称 ==
一说因产于[[贺兰山|贺兰山]]东麓和[[黄河|黄河]]两岸的草滩一带而得名<ref>{{cite web |title=滩羊 |url=http://xuewen.cnki.net/R2006060010002191.html |website=CNKI-中国名食百科 |access-date=2018-07-27 |archive-url=https://web.archive.org/web/20180727054859/http://xuewen.cnki.net/R2006060010002191.html |archive-date=2018-07-27 |dead-url=no }}</ref>，一说因自然放牧时羊群远看像一片一片的碱滩而得名<ref>{{cite web |title=宁夏盐池滩羊养殖系统 |url=http://www.moa.gov.cn/ztzl/zywhycsl/dspwhyc/201709/t20170918_5818747.htm |website=农业部 |accessdate=2018-07-25 |archive-url=https://web.archive.org/web/20180725153621/http://www.moa.gov.cn/ztzl/zywhycsl/dspwhyc/201709/t20170918_5818747.htm |archive-date=2018-07-25 |dead-url=no }}</ref>。

== 外貌特征 ==
成年滩羊的体格中等，体质结实，腰背平直，胸较深，四肢端正，蹄质坚实。尾根部宽大，尾尖细圆，呈三角形，下垂过[[跗節|跗关节]]。鼻梁稍微隆起，耳朵有大、中、小三种。公羊有螺旋形[[犄角|角]]，母羊一般没有角或者有小角。毛色纯白有光泽，头部多有褐色、黑色或黄色斑块。前后躯的毛股的表现一致，自然长度8厘米以上，呈毛辫状<ref name="tian"/>。

滩羊以皮毛颜色区别，有白、黑之分，但实际养殖中几乎均为白色滩羊，黑色滩羊数量很小。据调查，宁夏黑色滩羊主要分布在同心县的[[韦州镇|韦州]]、豫旺，中卫的[[红泉镇|红泉]]，盐池的[[惠安堡镇|惠安堡]]等地区，自1970年代以来饲养量不断下降，至2008年调查统计时仅存500多只<ref>{{cite journal |author1=赵金宇, 任德新, 张鑫荣 |title=宁夏白、黑滩羊种质及生产性能研究与分析 |journal=甘肃畜牧兽医 |date=2009 |issue=6}}</ref>。

== 人类利用 ==
=== 裘皮 ===
滩羊是裘皮羊品种，所产一月龄羔羊裘皮，惯称“二毛皮”，毛股具5个至7个半圆形弯曲<ref name="cnki"/>，有“二毛皮九道弯”的说法。二毛皮可用于制作男女冬装。滩羊皮的鞣制工艺非常复杂，以山西省交城县的滩羊皮鞣制工艺为例，有洗、泡、晒、铲、钉、鞣、吊、压、裁、缝等20余道工序，另加入黄糜、皮硝等辅料，完全依靠手工操作<ref name="jiaocheng">{{cite web |title=交城滩羊皮鞣制工艺（交城）（传统技艺）（0201） |url=http://www.shanxi.gov.cn/sq/lsrw/msgj/201804/t20180412_406639.shtml |website=山西省人民政府网 |accessdate=2018-07-27 |archive-url=https://web.archive.org/web/20180727054900/http://www.shanxi.gov.cn/sq/lsrw/msgj/201804/t20180412_406639.shtml |archive-date=2018-07-27 |dead-url=no }}</ref>。

=== 羊毛 ===
滩羊毛的纤维细长均匀，绒毛轻柔蓬松，富有弹性，是制作毛毯、披肩围脖、毛衫等的高级原料<ref name="jiaocheng"/>。

=== 滩羊肉 ===
滩羊以肉质鲜美、风味独特而著称。滩羊肉的脂肪分布均匀<ref>{{cite journal |author1=权海涛 |author2=杨晓燕 |author3=陈海燕 |author4=马天海 |title=不同月龄宁夏滩羊背最长肌挥发性风味成分比较 |journal=养殖与饲料 |date=2011 |issue=11}}</ref>，肌肉纤维细腻，肉质鲜嫩，吸水性极高，膻味较其他羊种相比较轻<ref>{{cite journal |author1=马志胜 |title=宁夏滩羊不同部位肉中挥发性风味物质分析 |journal=畜牧兽医科学 |date=2017 |issue=12 |page=24}}</ref>。据测定，每100克滩羊肉中含有蛋白质13.6克，脂肪34.8克，碳水化合物1.1克，以及钙、铁、磷、钾等矿物质，同时还含有一定的维生素、烟酸、氨基酸等营养成分<ref>{{cite journal |author1=张晓梅 |title=宁夏滩羊文献分析 |journal=安徽农业科学 |date=2010 |issue=38}}</ref>。
