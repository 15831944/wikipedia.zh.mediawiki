'''硒酸钠'''(Sodium selenate)是具有化学式{{Chem|Na|2|SeO|4}}的[[无机化合物|无机化合物]]，区分于亚硒酸钠。 它以无水盐、七水合物和十水合物的形式存在。<ref>Kamburov, S.; Schmidt, H.; Voigt, W.; Balarew, C. "Similarities and peculiarities between the crystal structures of the hydrates of sodium sulfate and selenate" Acta Crystallographica B (Struct Sci Cryst Eng Mater.) 2014, volume 70, pp. 714-22. {{Doi|10.1107/S2052520614007653}}</ref> 它们都是白色水溶性固体。十水合物是[[混合维生素补充剂|混合维生素补充剂]]和[[饲料|牲畜饲料]]中常见的成分，是硒的来源。 无水盐用于生产玻璃。虽然硒酸盐毒性更大，但硒酸钠和硫酸钠的许多物理性质相似。 <ref name="Ullmann">Bernd E. Langner "Selenium and Selenium Compounds" in Ullmann's Encyclopedia of Industrial Chemistry, 2005, Wiley-VCH, Weinheim. {{Doi|10.1002/14356007.a23_525}}.</ref> 

== 生产 ==
硒酸钠通过硒的氧化产生，首先用[[硝酸|硝酸]]，产生[[亚硒酸|亚硒酸]]。 将亚硒酸中和以形成[[亚硒酸盐|亚硒酸钠]]。 亚硒酸钠被在[[鹽基|碱性]]介质中[[过氧化氢|的过氧化氢]][[氧化还原反应|氧化]]以形成[[硒酸盐|硒酸盐]] ，然后将其[[喷雾干燥|喷雾干燥]] 。 <ref> Bjornberg，A。;美国Martensson; Paulsson KM（Boliden Aktiebolag）。生产硒盐的方法。 1986年8月12日的美国专利4,605,544。 </ref> 

: Se + 2HNO<sub>3</sub> → H<sub>2</sub>SeO<sub>3</sub> + NO + NO<sub>2</sub> 
: H<sub>2</sub>SeO<sub>3</sub> + Na<sub>2</sub>CO<sub>3</sub> →Na<sub>2</sub>SeO<sub>3</sub> + H<sub>2</sub>O + CO<sub>2</sub> 
: Na<sub>2</sub>SeO<sub>3</sub> + H<sub>2</sub>O<sub>2</sub> →Na<sub>2</sub>SeO<sub>4</sub> + H<sub>2</sub>O 

它是在[[永斯·贝采利乌斯|永斯·贝采利乌斯]]于1817年发现[[硒|硒]]后不久制备的。 

== 工业用途 ==

=== 玻璃制造业 ===
硒酸钠最早的应用之一是[[玻璃|玻璃]]工业。 硒在玻璃中产生红色色调。熔融玻璃用硒酸钠处理后用[[三氧化二砷|三氧化二砷]]处理以还原硒酸钠并提供元素硒。 硒酸钠也用作玻璃生产中的脱色剂。它赋予玻璃的红色色调与制造过程中由[[氧化亚铁|氧化亚铁]]产生的绿色色调互补。当一起使用时，两种化合物产生无色玻璃。 <ref>惠特克，MC;工业与工程化学。 1912年，第7页，第4页。 539-540 </ref> 

=== 农药 ===
硒酸钠是一些用于对抗[[螨|螨虫]]、[[蚜虫|蚜虫]]和[[粉蚧|粉蚧]]的[[殺蟲劑|杀虫剂中]]的常见成分。 对于大多数[[昆虫|昆虫]]，剂量为10mg/kg足以致命。<ref> Krieger，K。农药毒理学手册：第1卷;学术出版社：加利福尼亚州圣地亚哥，2001年</ref> <ref>汉森，B。; Lindblom，SD;勒夫勒，ML; Pilon-Smits，E。;新的植物学家。 2004,3,162。Pg。 655-662。 </ref> 它也用于一些[[杀真菌剂|杀菌剂]] 。 

=== 作物的生物强化 ===
硒酸钠有效地用于作物的生物强化，因此强化食物/饲料以减轻人和牲畜的硒缺乏。它可以作为叶面喷雾或通过生根培养基施用，例如添加到肥料中。 

== 膳食补充剂 ==
由于其硒含量和高溶解度，硒酸钠是非处方维生素补充剂的常见成分。  硒是一种痕量[[礦物質|必需元素]]。 在预混动物饲料中，硒酸钠和亚硒酸盐也很常见。 两种化合物都没有表现出吸收的硒量的差异。 美国FDA规定动物饲料含量不超过5ppm硒含量。<ref> Podoll，KL;伯纳德，JB; Ullrey，DE; DeBar，SR; Ku，PK; Magee，WT; Journ。动物科学。 1992年，70年，第6页，1965年至1970年。 </ref> 对此的争议发生在2009年，当时一组21匹马球马因不正确混合的[[營養補充品|膳食补充剂]]死于硒中毒。 <ref>希普利，艾米“马球马的死亡来自错误混合的补充”华盛顿邮报，华盛顿，2009年4月29日，第1页。 </ref> 

== 毒理学 ==
美国FDA和欧盟目前将硒酸钠分类为有毒，主要是摄入或吸入。对大鼠的测试显示剂量为1.6mg/kg是致命的。对于70千克（154磅）的人，该剂量相当于112毫克，或者560粒200微克药丸。长期接触硒酸钠会导致严重的肺、肾和肝损伤。 <ref name="Ganther"> Ganther，HE;加利福尼亚州鲍曼; Journ。营养。 1962年，77。P. 408-414 </ref> 

饮食中过量暴露于硒导致称为硒中毒的病症。在血液水平大于100微克/分升时发生硒中毒。 症状包括胃肠不适、脱发、白斑、大蒜呼吸气味、疲劳、烦躁和轻度神经损伤。<ref>膳食补充剂情况说明书：硒。 [http://ods.od.nih.gov/factsheets/selenium http://ods.od.nih.gov/factsheets/selenium（2011年10月19日]访问）。 </ref>
