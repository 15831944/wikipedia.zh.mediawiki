'''吹上站'''（{{jpn|j=吹上駅|hg=ふきあげえき|rm=Fukiage eki|f=y}}）是[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）和[[日本貨物鐵道|日本貨物鐵道]]（JR貨物）的[[鐵路車站|鐵路車站]]，位於[[埼玉縣|埼玉縣]][[鴻巢市|鴻巢市]]吹上本町一丁目。

== 車站結構 ==
本站是一[[地面車站|地面車站]]，站房採[[跨站式站房|跨站式站房]]設計。站內設有[[綠色窗口|綠色窗口]]、自動閘機、劃位車票自動售賣機等設備，此外更有[[Kiosk_(便利商店)|Kiosk便利商店]]、[[小吃店|小吃店]]等店舖。

本站有兩座月台，1號月台是[[側式月台|側式月台]]，2、3號月台則是[[島式月台|島式月台]]設計。

== 貨運業務 ==
本站原兼營客貨運，國鐵民營化後貨運服務由JR貨物管理。1980年，JR貨物暫停本站大部分貨運服務，只維持自[[根岸站|根岸站]]經本站的關東菱油（新日本石油系，原三菱石油系）專用線至本站附近的吹上油庫的班次。自2002年3月專用線被廢除後，本站不再有定期貨運列車班次停靠，並降格為臨時貨運站。

== 历史 ==
* 1885年（[[明治|明治]]18年）3月1日－[[日本鐵道|日本鐵道]]吹上站開業。
* 1901年（明治34年）6月－忍馬車鐵道（後稱[[行田馬車鐵道|行田馬車鐵道]]）於本站前設站。
* 1906年（明治39年）11月1日－日本鐵道被國有化，成為[[日本國有鐵道|日本國鐵]]的車站。
* 1922年（[[大正|大正]]11年）－行田馬車鐵道結業。
* 1980年（[[昭和|昭和]]55年）8月21日－除自專用線開出的貨運列車外，停止所有貨運業務。
* 1981年（昭和56年）10月20日－站房改建為跨站式站房。
* 1985年（昭和60年）3月14日－停止行李託運服務。
* 1987年（昭和62年）4月1日－[[國鐵分割民營化|國鐵分割民營化]]，本站由JR東日本和JR貨物接手管理。
* 2001年（[[平成|平成]]13年）
** 11月18日－智能卡系統[[Suica|Suica]]正式投入服務。
** 12月1日－[[湘南新宿線|湘南新宿線]]開業。
* 2002年（平成14年）3月23日－廢除所有自本站開出的定期貨運列車班次。
