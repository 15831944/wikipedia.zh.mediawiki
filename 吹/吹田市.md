'''吹田市'''（{{jpn|j=吹田市|hg=すいたし|rm=Suita shi|f=y}}）是位于[[日本|日本]][[大阪府|大阪府]]北部的一个[[城市|城市]]，轄區南側屬於[[大阪平原|大阪平原]]，北側為{{Tsl|ja|千里丘陵}}，1970年[[日本万国博览会|日本万国博览会]]曾在此舉辦。

[[日本職業足球聯賽|日本職業足球聯賽]]的[[大阪飛腳|大阪飛腳]]即以此地的[[市立吹田足球场|市立吹田足球场]]作為主場。

== 歷史 ==


現在的吹田市轄區在在令制國時代大部分區域都屬於[[攝津國|攝津國]][[島下郡|島下郡]]，西南部則屬於[[豐島郡_(大阪府)|豐島郡]]；在[[江戶時代|江戶時代]]大部分區域為{{Tsl|ja|淀藩}}的領地，但也有部分區域分屬幕府直屬、旗本、{{Tsl|ja|高槻藩}}、[[岡田藩|岡田藩]]、{{Tsl|ja|岡部藩}}的領地。

[[明治維新|明治維新]]後，這些區域曾先後隸屬大阪府司農局、大阪府北司農局、[[攝津縣|攝津縣]]、[[豐崎縣|豐崎縣]]、[[兵庫縣|兵庫縣]]、高槻縣、淀縣、半原縣、岡田縣，直到1871年的第一次府縣整併中才全數被劃入[[大阪府|大阪府]]。

1889年實施[[町村制|町村制]]時，現在的吹田市轄區分屬：{{Tsl|ja|岸部村}}、{{Tsl|ja|山田村 (大阪府三島郡)|山田村 (大阪府三島郡)|山田村}}、'''吹田村'''、{{Tsl|ja|千里村 (大阪府)|千里村 (大阪府)|千里村}}、{{Tsl|ja|新田村 (大阪府)|新田村 (大阪府)|新田村}}、{{Tsl|ja|豊津村 (大阪府)|豐津村 (大阪府)|豐津村}}共六個行政區劃；吹田村在1908年升格為'''{{Tsl|ja|吹田町}}'''，1940年再與千里村、岸部村、豐津村合併為'''吹田市'''，1950年代又陸續併入了山田村和新田村的部分區域，成為現在的吹田市。<ref>{{Cite web| url=http://www.city.suita.osaka.jp/home/kurashinojoho/_79352/_79353.html| title=吹田市の沿革| publisher=吹田市政府| language=ja| accessdate=2018-12-13}}</ref>

2000年日本開始實施[[特例市|特例市]]制度後，吹田市也於2001年成為特例市，但隨著特例市制度的廢除，目前吹田市計畫於2020年再升格為[[中核市|中核市]]。

== 交通 ==

轄內有多條軌道交通路線通過，包括[[西日本旅客鐵道|西日本旅客鐵道]]的[[東海道本線|東海道本線]]、[[阪急電鐵|阪急電鐵]]的[[千里線|千里線]]和[[京都本線|京都本線]]、[[大阪地下鐵|大阪地下鐵]]的[[御堂筋線|御堂筋線]]以及與其[[直通運行|直通運行]]的[[北大阪急行電鐵|北大阪急行電鐵]][[南北線_(北大阪急行電鐵)|南北線]]、[[大阪高速鐵道|大阪高速鐵道]]的[[大阪單軌電車線|本線]]和[[國際文化公園都市單軌電車線|彩都線]]，於2019年3月開始營業的西日本旅客鐵道[[大阪東線|大阪東線]]延伸段也通過轄區南部。

此外，這些軌道業者在轄內有大多設有[[機廠|機廠]]，包括西日本旅客鐵道的{{link-ja|吹田総合車両所|吹田綜合車輛所}}、[[日本貨物鐵道|日本貨物鐵道]]的{{link-ja|吹田機関区|吹田機関区}}、[[大阪市高速電氣軌道|大阪市高速電氣軌道]]的東吹田檢車場（[[堺筋線|堺筋線]]的機廠）、[[大阪高速鐵道|大阪高速鐵道]]的單軌電車車輛基地、[[北大阪急行電鐵|北大阪急行電鐵]]的桃山台機廠。

高速公路則有[[名神高速道路|名神高速道路]]、[[中國自動車道|中國自動車道]]、[[近畿自動車道|近畿自動車道]]在轄區東北部的[[吹田系統交流道|吹田系統交流道]]交會，同時也是中國自動車道、近畿自動車道的起點，除了往南往東可進入大阪市市區外，往北可通往[[名古屋市|名古屋市]]，往西可到中國地方。

轄內客運主要依賴{{Tsl|ja|阪急バス|阪急巴士}}所經營的客運路線，同時{{Tsl|ja|京阪バス|京阪巴士}}和{{Tsl|ja|近鉄バス|近鐵巴士}}也有經營往返周邊城市的交通路線。

== 教育 ==

轄內的高等教育學校包括[[大阪大学|大阪大学]]、[[大阪學院大學|大阪學院大學]]、[[关西大学|关西大学]]、{{Tsl|ja|千里金蘭大学|千里金蘭大學}}、{{Tsl|ja|大和大学|大和大學}}；其中大阪大学的前身可以追溯至江戶時代1838年由醫生兼西洋學者[[緒方洪庵|緒方洪庵]]所開設的[[蘭學|蘭學]][[適塾|適塾]]，1931年被整併為「大阪帝國大學」，為當時日本本土的七所[[帝國大學|帝國大學]]之一，二次大戰結束後，於1937年改為現在的「大阪大學」。<ref name="OsakaUniversity">{{cite web |url=http://www.osaka-u.ac.jp/ja/guide/about/history|script-title=ja:大阪大学の歴史 |accessdate=2014-09-11|publisher={{lang|ja|大阪大学}}}}{{ja}}</ref>

[[关西大学|關西大學]]則可追溯到1825年成立的漢學塾{{Tsl|ja|泊園書院}}及1886年成立的{{Tsl|ja|関西法律学校|關西法律學校}}，1905年改組為私立關西大學，後再於1920年改為現在的名稱<ref>{{cite web |url=http://www.kansai-u.ac.jp/global/guide/history.html|script-title=ja:関西大学のあゆみ |accessdate=2014-09-11|publisher={{lang|ja|関西大学}}}}{{ja}}</ref>；目前與關西地區幾所具有知名度的私立大學被合稱為「[[關關同立|關關同立]]」。

* [[妙高市|妙高市]]（[[新潟縣|新潟縣]]）
*: 吹田市於2002年8月28日與{{Tsl|ja|妙高高原町}}締結友好協定，而後妙高高原町於2005年合併為妙高市。
* [[若狹町|若狹町]]（[[福井縣|福井縣]][[三方上中郡|三方上中郡]]）
*: 吹田市於2004年11月13日與{{Tsl|ja|三方町}}締結友好協定，而後三方町於2005年合併為若狹町。
* [[高島市|高島市]]（[[滋賀縣|滋賀縣]]）
*: 吹田市最初是與{{Tsl|ja|今津町 (滋賀県)|今津町 (滋賀縣)|今津町}}展開交流，今津町於2005年合併為高島市後，雙方於同一年締結友好協定。
* [[能勢町|能勢町]]（[[大阪府|大阪府]][[豐能郡|豐能郡]]）
*: 兩地於2005年8月29日締結友好協定。
* [[香美町|香美町]]（[[兵庫縣|兵庫縣]][[美方郡|美方郡]]）
*: 兩地於2008年11月12日締結友好協定。

=== 海外 ===
* {{flagicon|Sri Lanka}} [[莫拉圖瓦|莫拉圖瓦]]（[[斯里蘭卡|斯里蘭卡]][[西部省_(斯里蘭卡)|西部省]][[可倫坡區|可倫坡區]]）
*: 兩地於1982年7月20日締結友好協定。
* {{flagicon|Australia}} [[賓士鎮市|賓士鎮市]]（[[澳洲|澳洲]][[新南威爾斯州|新南威爾斯州]]）
*: 兩地於1989年3月9日簽署友好協定，但賓士鎮市已於2016年合併為{{Tsl|en|Canterbury-Bankstown Council|坎特伯雷－賓士鎮議會}}。
