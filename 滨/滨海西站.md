'''滨海西站'''原名'''滨海站'''，位于[[天津市|天津市]][[滨海新区|滨海新区]]，是[[天津滨海新区|天津滨海新区]]最大的地面火车站<ref name="xiayue">[http://www.bh.gov.cn/bhxw/system/2010/07/19/010056671.shtml 滨海新区：滨海站交通枢纽站房下月开工] {{webarchive|url=https://web.archive.org/web/20150402090340/http://www.bh.gov.cn/bhxw/system/2010/07/19/010056671.shtml |date=2015-04-02 }}，滨海新区网，2010-07-19，于2010-7-30查阅。</ref>，与[[天津站|天津站]]、[[天津西站|天津西站]]、[[滨海站|滨海站]]并称为[[天津铁路|天津铁路]]四大主客运枢纽<ref>[http://www.chinahighway.com/news/2010/425237.php 滨海火车站：打造综合交通枢纽中心]，中国公路网，2010-07-23，于2010-7-30查阅。</ref>，[[津秦客运专线|津秦客运专线]]、[[环渤海城际铁路|环渤海城际铁路]]、[[京滨城际铁路|京滨城际铁路]]、[[天津地铁|地铁]]市域Z2线、滨海B2线、滨海B3线将在此相汇。滨海西站已于2008年开始前期土地整理工作，并已于2013年12月1日随[[津秦客运专线|津秦客运专线]]一并开通<ref name="xiayue"/>。

== 位置 ==
滨海西站交通枢纽位于[[滨海新区|滨海新区]]海洋高新区金海湖畔，距离[[滨海新区|滨海新区]]核心约12公里，东临海川路、西邻海德路、南临[[天津经济技术开发区|天津经济技术开发区]][[泰达|泰达]]第九大街、北临滨庸道。

== 车站结构 ==
滨海西站的主站房分地上2层，地下1层，总面积约8万平方米，是[[津秦客运专线|津秦客运专线]]上的新建车站。

滨海西站按[[环渤海城际铁路|环渤海城际铁路]]和[[津秦客运专线|津秦客运专线]]两个高速分场设置，共8台18线，最高聚集人数可达2500人，一期工程建设[[津秦客运专线|津秦客运专线]]站台，并为[[环渤海城际铁路|环渤海城际铁路]]预留站台施工位置。

== 车站功能 ==
滨海西站是[[天津铁路|天津铁路]]四大主客运枢纽之一，将承担滨海新区[[津秦客运专线|津秦客运专线]]高速客流的始发终到任务和[[环渤海城际铁路|环渤海城际铁路]][[滨海新区|滨海新区]]至[[黄骅|黄骅]]段的城际铁路客流的始发终到任务<ref name="xiayue"/>。

== 配套设施 ==
滨海西站站区内规划3条轨道交通线，分别为市域Z2线([[武清区|武清]]—[[汉沽|汉沽]])、滨海B2线(滨海西站——[[天津经济技术开发区|临港工业区]])、滨海B3线(滨海西站——南部新城)。滨海西站主站房建成后，其南侧和北侧广场规划8条和4条公交线路，北广场东侧占地2万平方米的长途汽车站将设置到达车位5个、发车位20个，同时南北两侧的两处社会停车场规划停车泊位990个。
