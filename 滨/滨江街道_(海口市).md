'''滨江街道'''是[[中华人民共和国|中华人民共和国]][[海南省|海南省]][[海口市|海口市]][[琼山区|琼山区]]的一个街道。原街道办工委书记丁敏涉嫌利用职务便利收受76万“好处费”，在2015年4月底因涉嫌受贿犯罪被立案侦查<ref>{{cite web |url=http://www.sohu.com/a/31333064_161791 |title=海口市琼山区滨江街道办事处原工委书记丁敏 涉嫌受贿76万受审 |author=王燕珍 |date=2015-09-10 |website= |publisher=南国都市报 |access-date=2018-03-01 |quote=}}</ref>；原街道办副主任陈高明利用其分管职务之便，受贿134.29万元的财务，在2017年被[[海南省高级人民法院|海南省高院]]以受贿罪判刑<ref>{{cite web |url=http://www.hntqb.com/html/2017-12/08/content_6_1.htm |title=海口滨江街道办原副主任陈高明 利用查违建受贿134万判7年 |author=吴兴 |date=2017-12-08 |website=www.hntqb.com |publisher=海南特区报 |access-date=2018-03-01 |quote=}}</ref>。

==行政区划==

滨江街道下辖以下地区：

东门社区、城东社区、下坎社区、铁桥社区和博桂社区。<ref>{{cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2016/46/01/07/460107003.html |title=2016年统计用区划代码和城乡划分代码：滨江街道 |author= |date=2016 |website= |publisher=[[中华人民共和国国家统计局|中华人民共和国国家统计局]] |access-date= |quote=}}</ref>
