'''濱松市'''（{{jpn|j=浜松市|hg=はままつし|rm=Hamamatsu shi|f=y}}）是位於[[日本|日本]][[靜岡縣|靜岡縣]]西部[[遠江國|遠州]]地區的[[都市|都市]]，也是靜岡縣第2個[[政令指定都市|政令指定都市]]，人口約79萬，為靜岡縣內人口最多的[[市|市]]<ref>{{cite web |url=http://toukei.pref.shizuoka.jp/jinkoushugyouhan/data/02-030/documents/2608jk.xls |script-title=ja:平成26年8月市町別推計人口 |accessdate=2014-09-03|publisher={{lang|ja|静岡県}} |language=ja}}</ref>，轄區面積為1,558.04平方公里，是日本面積第二大的市，僅次於[[岐阜縣|岐阜縣]][[高山市|高山市]]<ref>{{cite web |url=http://www.gsi.go.jp/common/000077734.pdf |script-title=ja:市区町村別面積の順位 大小各20位 |accessdate=2014-09-03|publisher={{lang|ja|国土地理院}} |language=ja}}</ref>。其為日本重要的工業都市，[[鈴木_(公司)|鈴木公司]]、[[本田技研工業|本田技研工業]]、[[山葉|山葉]]等日本大型企業都是創業於此。

濱松最早是[[江戶時代|江戶時代]]的[[城下町|城下町]]，也是當時[[東海道|東海道]]的[[宿場町|宿場町]]之一。1911年7月1日，濱松正式建市。2005年7月1日，濱松市和附近的多個市町村進行合併，市域面積大幅擴大。2007年4月1日，濱松市與[[新潟縣|新潟縣]][[新潟市|新潟市]]同時升格為政令指定都市<ref>{{cite web |url=http://www.soumu.go.jp/main_sosiki/jichi_gyousei/bunken/shitei_toshi-ichiran.html |script-title=ja:指定都市一覧 |accessdate=2014-09-03|publisher={{lang|ja|総務省}} |language=ja}}</ref>，。現在全市分為[[中區_(濱松市)|中區]]、[[東區_(濱松市)|東區]]、[[西區_(濱松市)|西區]]、[[南區_(濱松市)|南區]]、[[北區_(濱松市)|北區]]、[[濱北區|濱北區]]、[[天龍區|天龍區]]7個[[市轄區|區]]。

== 历史 ==
濱松市歷史悠久。在[[濱北區|濱北區]]根堅洞窟發現了約1萬4000年至1萬8000年之前的人骨，顯示濱松自這一時期開始就有人類活動<ref>{{cite web |url=http://gakusyu.shizuoka-c.ed.jp/shakai/shizuoka_bunkazai/seibu_nekata.html |script-title=ja:根堅遺跡 |accessdate=2014-09-05|publisher={{lang|ja|静岡県総合教育センター}} |language=ja}}</ref>。在[[中區_(濱松市)|中區]]還發現了約4,000至3,000年前，繩紋時代後期的遺跡<ref>{{cite web |url=http://www.city.hamamatsu.shizuoka.jp/hamahaku/04smzkap/kouen.html |script-title=ja:蜆塚公園 |accessdate=2014-09-05|publisher={{lang|ja|浜松市}} |language=ja}}</ref>。進入[[奈良時代|奈良時代]]後，濱松一帶地區的人類活動更加活躍。這一時期濱松被稱為「濱津」，這亦是今日「濱松」這一名稱的起源<ref name="History1">{{cite web |url=http://www.city.hamamatsu.shizuoka.jp/koho2/intro/nenpyo01.html |script-title=ja:年表|accessdate=2014-09-05|publisher={{lang|ja|浜松市}} |language=ja}}</ref>。1570年時，德川家康攻占濱松地區的曳馬城，並將其改名為[[濱松城|濱松城]]<ref name="History2">{{cite web |url=http://www.city.hamamatsu.shizuoka.jp/koho2/intro/ayumi01.html |script-title=ja:浜松の歴史|accessdate=2014-09-05|publisher={{lang|ja|浜松市}} |language=ja}}</ref>。在江戶時代，濱松是一座[[城下町|城下町]]。江戶幕府有6位[[老中|老中]]都是出身自濱松城，使得濱松城有「出世城」之稱<ref name="History2" />。此外，由於濱松位於[[東海道|東海道]]途中，因此濱松在江戶時代曾是一座宿場町，大小旅館林立<ref name="History1" />。1709年時，濱松有1,386戶，人口有4,336人<ref name="History1" />。
[[File:Hamamatsu_downtown_in_Meiji_and_Taisho_eras.JPG|220px]]
[[明治維新|明治維新]]之後，隨著[[廢藩置縣|廢藩置縣]]的進行，1871年12月，濱松成為{{link-ja|濱松縣|浜松県}}的縣廳所在地<ref name="History1" />。不過在1876年，濱松縣被合併入靜岡縣，濱松市亦被劃入靜岡縣<ref name="History1" />。1887年，日本首台[[風琴|風琴]]就是在濱松市製造<ref>{{cite web |url=http://www.yamaha.co.jp/design/ud2010/declaration/history.html |script-title=ja:これまでのヤマハの歩み|accessdate=2014-09-05|publisher={{lang|ja|ヤマハ}} |language=ja}}</ref>，從此開始了濱松市樂器之城的歷史。1889年，[[東海道本線|東海道本線]][[濱松站|濱松站]]開業，濱松市的交通狀況大為改善。1889年4月1日，濱松町正式成立<ref name="History2" />。1911年，濱松町又升格成為濱松市，當時濱松市的人口有36,782人<ref name="History2" />。20世紀前期是濱松市工業的起步期。隨著[[山葉|山葉]]的企業的發展，濱松市的樂器產業日益發展。同時濱松亦建有眾多軍需工廠，這亦確立了濱松在戰後發動機工業的基礎。但在1944年，濱松市在東南海地震中嚴重受災。加上二戰期間，擁有眾多軍需工廠的濱松市多次受到美軍空襲。二戰期間濱松市共受到27次空襲和艦砲射擊、有約5000人在空襲中死亡或受傷<ref name="History2" />。濱松市因戰火而幾乎化為廢墟。

濱松市在戰後迅速開始復興。濱松市在戰後多次和附近市町村合併，市域面積和人口都大幅增加。1982年，濱松市的人口超過50萬<ref name="History2" />。1964年，[[東海道新幹線|東海道新幹線]]正式開通並途徑濱松市<ref name="History1" />。之後在1969年，東名高速公路亦實現竣工<ref name="History1" />。這兩條交通動脈的完成大幅改善了濱松市的聯外交通。濱松市的經濟在戰後亦有大幅發展。1946年，[[本田宗一郎|本田宗一郎]]在濱松創建了本田技術研究所，並且製造了本田公司的第一輛機車<ref>{{cite web |url=http://hamamatsu-daisuki.net/keyword10/bike.html |script-title=ja:オートバイ|accessdate=2014-09-05|publisher={{lang|ja|はままつ旅百花}} |language=ja}}</ref>，開啟了濱松市機車製造業的歷史。隨著本田取得成功，[[鈴木公司|鈴木織機]]和[[山葉|日本樂器製造]]也開始製造機車，濱松市的機車製造業迅速發展，成為繼樂器之後濱松市的另一大支柱產業。1982年，濱松市建立了科學園區<ref>{{cite web |url=http://hamamatsu-daisuki.net/keyword10/bike.html |script-title=ja:産業立地政策、地域産業政策の歴史的展開|accessdate=2014-09-05|publisher={{lang|ja|産業立地}} |language=ja}}</ref>，科技亦開始成為濱松市的重要產業<ref>{{cite book |author={{lang|ja|平岡昭利・野間晴雄}} |script-title=ja:《地図で読む百年　中部Ⅰ》 |year=2000 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=978-4772250474}}{{ja}}</ref>{{rp|89-96}}。1996年，濱松市成為[[中核市|中核市]]。2005年，濱松市和[[濱北市|濱北市]]、[[天龍市|天龍市]]、引佐郡{{link-ja|引佐町|引佐町}}、{{link-ja|細江町|細江町}}、{{link-ja|三日町|三ケ日町}}、濱名郡{{link-ja|雄踏町|雄踏町}}、{{link-ja|舞阪町|舞阪町}}、{{link-ja|磐田郡|磐田郡}}{{link-ja|佐久間町 (靜岡縣)|佐久間町|佐久間町}}、{{link-ja|水窪町|水窪町}}、{{link-ja|龍山村 (靜岡縣)|龍山村 (静岡県)|龍山村}}、周智郡{{link-ja|引佐町|引佐町}}{{link-ja|春野町 (靜岡縣)|春野町 (静岡県)|春野町}}合併。2007年，濱松市正式成為[[政令指定都市|政令指定都市]]。

== 地理 ==
[[File:Tenryu_river_aerial_photo.jpg|260px]]
濱松市位於日本的太平洋沿岸工業帶，地處[[東京特別區|東京]]和[[大阪市|大阪]]的中間位置<ref>{{cite web|url=http://www.city.hamamatsu.shizuoka.jp/sangyoshinko/invite/location/location.html|script-title=ja:浜松市の強み・魅力|accessdate=2014-09-05|publisher={{lang|ja|浜松市}}|language=ja|title={title}|archive-url=https://web.archive.org/web/20140905144403/http://www.city.hamamatsu.shizuoka.jp/sangyoshinko/invite/location/location.html|archive-date=2014-09-05|dead-url=yes}}</ref>。濱松市是日本面積第二大市，僅次於岐阜縣高山市。而寬闊的面積亦使得濱松市擁有多樣的地貌。濱松市北部地區屬[[赤石山脈|赤石山系]]，大多為山林所覆蓋，人口也較為稀少。濱松市的森林面積達103,000公頃，市域面積的約68%被森林所覆蓋<ref name="Geography">{{cite web |url=https://www.env.go.jp/air/sensory/jireishu/kk_jirei2009/pdf/06_5_Hamamatsu.pdf |script-title=ja:5．浜松（静岡県浜松市）|accessdate=2014-09-05|publisher={{lang|ja|環境省}} |language=ja}}</ref>。市區則主要集中於南部的平原地區。濱松市最南部的{{link-ja|遠州灘|遠州灘}}海岸則擁有日本少見的連綿沙灘。[[天龍川|天龍川]]流經濱松市東部地區，是濱松市最重要的河流<ref>{{cite web|url=http://www.cbr.mlit.go.jp/hamamatsu/gaiyo_kasen/gaiyo_kasen02.html|script-title=ja:天竜川の大きさなど|accessdate=2014-09-05|publisher={{lang|ja|国土交通省中部地方整備局浜松河川国道事務所}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20150509053733/http://www.cbr.mlit.go.jp/hamamatsu/gaiyo_kasen/gaiyo_kasen02.html|archivedate=2015-05-09}}</ref>。[[濱名湖|濱名湖]]位於濱松市西部，面積70.4平方公里，平均水深4.8米，南部深度較淺北部則湖水較深<ref>{{cite web |url=http://www.pref.shizuoka.jp/kankyou/ka-070/hamanako/hamanako/chiri.html|script-title=ja:自然保護課/浜名湖の地理|accessdate=2014-09-05|publisher={{lang|ja|静岡県}} |language=ja}}</ref>。由於有海水流入，[[濱名湖|濱名湖]]是一座汽水湖，水產資源亦十分豐富。

=== 氣候 ===
和靜岡縣的大多數地區一樣，濱松市氣候溫暖。濱松市的年均降水量達1,875.5毫米，年均日照時間達2,156.8小時，年平均氣溫則有16攝氏度。但在冬季，濱松市時有較強的西北季節風吹拂，使得體感溫度比實際溫度要低<ref name="Geography" />。在濱松市的山區地區，夏季時有焚風吹襲，導致氣溫有時會急劇升高。而在冬季，雖然山區地區的天龍區水窪町被指定為豪雪地帯，市中心則氣候溫暖極少降雪。

== 政治 ==
在日本眾議院選舉中，濱松市分屬三個選舉區。[[東區_(濱松市)|東區]]和[[南區_(濱松市)|南區]]、[[中區_(濱松市)|中區]]的大部分地區屬於[[靜岡縣第8區|靜岡縣第8區]]；[[西區_(濱松市)|西區]]、[[北區_(濱松市)|北區]]、[[濱北區|濱北區]]及[[天龍區|天龍區]]的大部分地區和[[中區_(濱松市)|中區]]和[[南區_(濱松市)|南區]]的小部分地區屬於[[靜岡縣第7區|靜岡縣第7區]]；[[天龍區|天龍區]]的小部分地區則與[[磐田市|磐田市]]、[[袋井市|袋井市]]、[[掛川市|掛川市]]、[[菊川市|菊川市]]、[[御前崎市|御前崎市]]的一部分以及[[森町_(靜岡縣)|森町]]同屬[[靜岡縣第3區|靜岡縣第3區]]。在傳統上來看，濱松市是[[自由民主黨_(日本)|自民黨]]較強的地區。現在濱松市三個選舉區的眾議院議員也都是自民黨籍的議員<ref>{{cite web |url=http://www.pref.shizuoka.jp/senkan/shugiin2012/kai/kai-shou/kai-3ku.html|script-title=ja:衆議院小選挙区　開票結果（静岡県第3区）|accessdate=2014-09-05|publisher={{lang|ja|静岡県選挙管理委員会}} |language=ja}}</ref><ref>{{cite web |url=http://www.pref.shizuoka.jp/senkan/shugiin2012/kai/kai-shou/kai-7ku.html|script-title=ja:衆議院小選挙区　開票結果（静岡県第7区）|accessdate=2014-09-05|publisher={{lang|ja|静岡県選挙管理委員会}} |language=ja}}</ref><ref>{{cite web |url=http://www.pref.shizuoka.jp/senkan/shugiin2012/kai/kai-shou/kai-8ku.html|script-title=ja:衆議院小選挙区　開票結果（静岡県第8区）|accessdate=2014-09-05|publisher={{lang|ja|静岡県選挙管理委員会}} |language=ja}}</ref><ref>{{cite web |url=http://www.yomiuri.co.jp/election/shugiin/2017/kaihyou/ya22.html|script-title=ja:【小選挙区】静岡県 |accessdate=2017-10-24|publisher={{lang|ja|読売新聞}} |language=ja}}</ref>。	

而在市政方面，現任濱松市長是{{link-ja|鈴木康友|鈴木康友}}。他曾是[[民主黨_(日本)|民主黨]]的眾議院議員。2007年5月1日，他參加濱松市市長選舉並以203,923票當選<ref>{{cite web|url=http://go2senkyo.com/election/8011|script-title=ja:浜松市長選挙 結果|accessdate=2014-09-05|publisher={{lang|ja|ザ選挙}}|language=ja|title={title}|archive-url=https://web.archive.org/web/20140905143854/http://go2senkyo.com/election/8011|archive-date=2014-09-05|dead-url=yes}}</ref>。2011年市長選舉時，由於沒有競爭對手登記參選，鈴木無投票當選，連任濱松市長。這也是日本的政令都市市長選舉中首次出現無投票當選<ref>{{cite web |url=http://www.nikkei.com/article/DGXNASFS2701B_X20C11A3000000/|script-title=ja:浜松市長に鈴木氏再選　政令市初の無投票当選|accessdate=2014-09-05|publisher={{lang|ja|選挙ドットコム}} |language=ja}}</ref>。鈴木康友在2015年和2019年也成功連任<ref>{{cite web |url=https://go2senkyo.com/local/senkyo/18507|script-title=ja:浜松市長選挙 |accessdate=2019-04-11|publisher={{lang|ja|日本経済新聞}} |language=ja}}</ref>。濱松市議會共有46個議席，但現在實質上只有44名市議員。議會的最大黨派是自民黨，擁有21個議席。此外[[公明黨|公明黨]]、[[日本共產黨|日本共產黨]]、[[社會民主黨_(日本)|社民黨]]等政黨也在濱松市議會擁有議席<ref>{{cite web |url=http://www.city.hamamatsu.shizuoka.jp/gikai/iinkai/meibo.html|script-title=ja:議員名簿（50音順）|accessdate=2014-09-05|publisher={{lang|ja|浜松市}} |language=ja}}</ref>。

== 經濟 ==
=== 農業 ===
濱松市的農業產出額位居日本第四，總農家數位居日本第一，是日本前列的農業大市<ref>{{cite web |url=http://www.city.hamamatsu.shizuoka.jp/nousei/shinko/agri/noshin_h20/ss3.html|script-title=ja:第3章 浜松市農業振興のビジョン|accessdate=2014-09-05|publisher={{lang|ja|浜松市}} |language=ja}}</ref>。就生產金額來看，濱松市最重要的農產品是[[柑橘|柑橘]]，其次依次是[[大米|大米]]、[[菊花|菊花]]、[[牛|肉牛]]、[[蜜瓜|蜜瓜]]、[[茶葉|茶葉]]和[[牛奶|牛奶]]<ref>{{cite web |url=http://www.city.hamamatsu.shizuoka.jp/nousei/shinko/agri/noshin_h20/ss2_1.html|script-title=ja:第2章 浜松市農業の現況(1)|accessdate=2014-09-05|publisher={{lang|ja|浜松市}} |language=ja}}</ref><ref>{{cite web |url=http://www.city.hamamatsu.shizuoka.jp/nousei/portal/nousanbutsu/index.html|script-title=ja:浜松市の農作物|accessdate=2014-09-05|publisher={{lang|ja|浜松市}} |language=ja}}</ref>。濱松的柑橘頗為著名，濱名湖柑橘銷售至日本各地，其中又以{{link-ja|三日柑橘|三ケ日みかん}}最具盛名，是濱松柑橘中的高級品種<ref>{{cite web|url=http://www.mikkabimikan.jp/about/|script-title=ja:教えて！三ヶ日みかん|accessdate=2014-09-05|publisher={{lang|ja|JAみっかび　三ヶ日町柑橘出荷組合}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20141210105809/http://www.mikkabimikan.jp/about/|archivedate=2014-12-10}}</ref>。濱松的另一代表名產是[[鰻魚|鰻魚]]。濱名湖的鰻魚養殖業開始於1900年<ref name="Chishi">{{cite book |author={{lang|ja|藤田佳久・田林明}}|script-title=ja:《日本の地誌7　中部圏》 |year=2007 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-16767-2|language=ja}}</ref>{{rp|222}}<ref>{{cite web |url=http://hamamatsu-daisuki.net/keyword10/eel.html|script-title=ja:うなぎ|accessdate=2014-09-05|publisher={{lang|ja|はままつ旅百花}} |language=ja}}</ref>，現在仍然是日本最重要的鰻魚養殖產地之一<ref name="Unagi1">{{cite web |url=http://www.hamamatsu-books.jp/category/detail/4e07e927abc48.html|script-title=ja:ウナギ|accessdate=2014-09-05|publisher={{lang|ja|浜松情報BOOK}} |language=ja}}</ref><ref name="Unagi2">{{cite web|url=http://www.city.hamamatsu.shizuoka.jp/miryoku/kawarahan/2011/014.html|script-title=ja:100年以上の歴史を持つ“うなぎ養殖”発祥の地・浜松|accessdate=2014-09-05|publisher={{lang|ja|浜松市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140905123221/http://www.city.hamamatsu.shizuoka.jp/miryoku/kawarahan/2011/014.html|archivedate=2014-09-05}}</ref>。濱松市的林業也頗為發達，天龍區北部地區是東海地區著名的林業地帶<ref>{{cite web|url=http://www.city.hamamatsu.shizuoka.jp/nousei/shinko/forestry/forest/index.html|script-title=ja:ようこそ、天竜美林へ|accessdate=2014-09-05|publisher={{lang|ja|浜松市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140905122638/http://www.city.hamamatsu.shizuoka.jp/nousei/shinko/forestry/forest/index.html|archivedate=2014-09-05}}</ref>。
=== 工業 ===
[[File:YAMAHA_(headquarters_1).jpg|260px]]
濱松市是日本重要的工業都市，[[鈴木公司|鈴木]]和[[山葉|山葉]]等日本位居前列的大企業現在仍將總部設在濱松市。2010年時，濱松的工業品出產金額達1兆9795億日元<ref>{{cite web |url=http://shinkinkeizai.jp/research/research201116.html|script-title=ja:浜松市の工業統計の推移|accessdate=2014-09-05|publisher={{lang|ja|静岡県西部地域しんきん経済研究所}} |language=ja}}</ref>。濱松市現代產業的發展開始於纖維工業。在1935年時，纖維產業產品的生產金額占到濱松總工業生產金額的70%<ref name="Chishi" />{{rp|217}}。在戰後，樂器產業和機車產業迅速發展，和纖維產業並列成為濱松市三大產業<ref>{{cite web |url=http://shinkinkeizai.jp/research/research200821.html|script-title=ja:浜松市三大産業の変遷|accessdate=2014-09-05|publisher={{lang|ja|静岡県西部地域しんきん経済研究所}} |language=ja}}</ref>。在樂器產業方面，日本首架[[風琴|風琴]]和[[鋼琴|鋼琴]]都是在濱松製造<ref>{{cite web |url=http://hamamatsu-daisuki.net/industry/history.html|script-title=ja:浜松地域の産業の歴史|accessdate=2014-09-05|publisher={{lang|ja|はままつ旅百花}} |language=ja}}</ref>。

現在日本三大樂器企業山葉、[[河合樂器製造公司|河合樂器製造公司]]和[[樂蘭公司|樂蘭公司]]的總部均位於濱松市<ref>{{cite web |url=http://www.hamamatsu-books.jp/category/detail/4d221037495ea.html|script-title=ja:楽器産業|accessdate=2014-09-05|publisher={{lang|ja|浜松情報BOOK}} |language=ja}}</ref>。濱松樂器產業最主要的產品是[[鋼琴|鋼琴]]。在山葉將其鋼琴工廠自濱松遷至掛川工廠之前，濱松市佔去日本鋼琴市場生產份額的100%<ref>{{cite web |url=http://gakusyu.shizuoka-c.ed.jp/shakai/shizuoka_kougyou/hamamatu_011_gakki.html|script-title=ja:浜松市の楽器産業|accessdate=2014-09-05|publisher={{lang|ja|静岡県総合教育センター}} |language=ja}}</ref>。除了鋼琴之外，濱松市的管樂器產量亦居於日本前列。濱松市各樂器企業還將其生產電子樂器的經驗應用在其他產業上，使得濱松市在日本半導體產業中亦有一席之地<ref name="Chishi" />{{rp|221}}。

機車產業是濱松另一大支柱產業，日本四大機車生產企業中，[[鈴木公司|鈴木]]、[[本田技研工業|本田]]和[[山葉|山葉]]都是發祥於濱松。機車製造業除了帶動零部件產業一併發展外，亦促進了汽車產業的成長。

除了三大產業之外，濱松的科技產業、光電產業亦頗為發達。1926年，[[高柳健次郎|高柳健次郎]]在濱松進行[[電視機|電視機]]映像實驗，是為日本電視產業之始<ref>{{cite web |url=http://www.hamamatsu-books.jp/category/detail/4d22220ddb997.html|script-title=ja:浜松の放送産業|accessdate=2014-09-05|publisher={{lang|ja|浜松情報BOOK}} |language=ja}}</ref>。此外，濱松市亦是日本首個[[底片|底片]]和國產客運[[飛機|飛機]]的誕生地<ref>{{cite web |url=http://www.city.hamamatsu.shizuoka.jp/miryoku/nihonhatsu.html|script-title=ja:浜松の日本初・日本一|accessdate=2014-09-05|publisher={{lang|ja|浜松市}} |language=ja}}</ref>。

=== 服務業 ===
濱松市的郊外地區人口較多，[[日本人口集中地區|人口集中地區]]人口佔總人口的比重相對較低，加上汽車社會的進展，使得濱松市中心的商業陷入蕭條。相反的是郊外有眾多大型購物中心。現在濱松市內的百貨商店僅有濱松站前的{{link-ja|遠鐵百貨店|遠鉄百貨店}}一家<ref>{{cite web |url=http://www.entetsu.co.jp/release/20090811entetsu.pdf|script-title=ja:新商業ビル建設及びギャラリーモール整備計画について|accessdate=2014-09-05|publisher={{lang|ja|遠州鉄道}} |language=ja}}</ref>。

在金融業方面，{{link-ja|遠州信用金庫|遠州信用金庫}}和{{link-ja|濱松信用金庫|濱松信用金庫}}這兩家金融機構的總部設在濱松。此外[[三井住友銀行|三井住友銀行]]、[[三菱東京UFJ銀行|三菱東京UFJ銀行]]、[[瑞穗銀行|瑞穗銀行]]、[[里索那銀行|里索那銀行]]、[[靜岡銀行|靜岡銀行]]、{{link-ja|駿河銀行|スルガ銀行}}、{{link-ja|清水銀行|清水銀行}}、{{link-ja|靜岡中央銀行|静岡中央銀行}}、{{link-ja|愛知銀行|愛知銀行}}、{{link-ja|名古屋銀行|名古屋銀行}}在濱松亦設有支行。

由於濱松市內{{link-ja|巴西僑民|在日ブラジル人}}眾多，因此[[巴西銀行|巴西銀行]]在濱松開有支行<ref>{{cite web |url=http://www.bb.com.br/portalbb/page22,3669,3669,22,0,7,8.bb?codigoNoticia=20976|script-title=ja:浜松支店|accessdate=2014-09-05|publisher={{lang|ja|ブラジル銀行}} |language=ja}}</ref>、[[巴西外交部|巴西外交部]]亦於濱松市設有[[總領事館|總領事館]]，為該市唯一一個[[外國駐日本機構列表|外國駐日本機構]]。

=== 藝術 ===
濱松是一座樂器之都，因此音樂文化特別發達。濱松現在舉辦有濱松國際鋼琴大賽<ref>{{cite web |url=http://www.hipic.jp/hipic/|script-title=ja:浜コンについて |accessdate=2014-09-05|publisher={{lang|ja|浜松国際ピアノコンクール}} |language=ja}}</ref>和濱松世界青少年合唱節<ref>{{cite web |url=http://www.hcf.or.jp/bunka/world_youth_music_festival/outline/|script-title=ja:開催趣旨|accessdate=2014-09-05|publisher={{lang|ja|浜松世界青少年音楽祭2014 - 浜松市文化振興財団}} |language=ja}}</ref>等音樂活動，試圖打造濱松成為音樂之都<ref>{{cite web |url=http://www.city.hamamatsu.shizuoka.jp/miryoku/jiman/music.html|script-title=ja:音楽の都へ|accessdate=2014-09-05|publisher={{lang|ja|浜松市}} |language=ja}}</ref>。濱松市還設有濱松市樂器博物館<ref>{{cite web |url=http://www.gakkihaku.jp/aboutus/outline/|script-title=ja:設立・沿革|accessdate=2014-09-05|publisher={{lang|ja|浜松市楽器博物館}} |language=ja}}</ref>，是日本首個公立的樂器博物館<ref>{{cite web |url=http://hamamatsu-daisuki.net/keyword10/music.html|script-title=ja:音楽|accessdate=2014-09-05|publisher={{lang|ja|はままつ旅百花}} |language=ja}}</ref>。除此之外，濱松市還有{{link-ja|濱松市美術館|浜松市美術館}}<ref>{{cite web |url=http://www.city.hamamatsu.shizuoka.jp/artmuse/|script-title=ja:浜松市美術館|accessdate=2014-09-09|publisher={{lang|ja|浜松市}} |language=ja}}</ref>、{{link-ja|濱松科学館|浜松科学館}}<ref>{{cite web|url=http://www.hamamatsu-kagakukan.jp/kannai/kannai/ayumi.html|script-title=ja:開館からの歩み|accessdate=2014-09-05|publisher={{lang|ja|浜松科学館}}|language=ja|title={title}|archive-url=https://web.archive.org/web/20140910200310/http://www.hamamatsu-kagakukan.jp/kannai/kannai/ayumi.html|archive-date=2014-09-10|dead-url=yes}}</ref>等藝術文化設施。

=== 飲食 ===
濱松市的飲食文化亦頗具特色。濱松市與[[宇都宮市|宇都宮市]]並列為日本[[餃子|餃子]]文化最興盛的兩座都市，而在2011年及2012年，濱松市連續兩年成為日本人均食用餃子數量最多的城市<ref>{{cite web |url=http://news.mynavi.jp/news/2013/02/04/033/|script-title=ja:静岡県・浜松市、宇都宮を抑え餃子消費量日本一に|accessdate=2014-09-05|publisher={{lang|ja|マイナビ}} |language=ja}}</ref>。濱松市內有眾多以餃子聞名的餐館，餃子已深入濱松的飲食文化之中。和日本其他地區多使用[[大白菜|大白菜]]不同，濱松餃子較多採用[[捲心菜|捲心菜]]作為內餡，因此味道較甜，在日本的餃子中獨樹一幟<ref>{{cite web |url=http://hamamatsugyouza.jp/about.html|script-title=ja:浜松餃子の歴史と特徴|accessdate=2014-09-05|publisher={{lang|ja|浜松餃子学会}} |language=ja}}</ref>。濱松飲食文化的另一象徵是[[鰻魚|鰻魚]]。濱松不僅是日本重要的鰻魚產地，亦是日本人均鰻魚[[蒲燒|蒲燒]]消費量最多的都市<ref name="Unagi1" />。由於濱松地處東京和大阪之間，在濱松可以同時品嚐到關東風和關西風的鰻魚蒲燒，是為濱松鰻魚蒲燒的特色<ref name="Unagi2" />。

=== 教育 ===
濱松市內現在有三所國立、公立大學，分別是[[靜岡大學|靜岡大學]]濱松校區（設有工學院和信息學院）<ref>{{cite web |url=http://www.shizuoka.ac.jp/campuslife/campus/hamamatsu/index.html|script-title=ja:浜松キャンパス|accessdate=2014-09-05|publisher={{lang|ja|静岡大学}} |language=ja}}</ref>、[[濱松医科大學|濱松医科大學]]和{{link-ja|靜岡文化藝術大學|静岡文化芸術大学}}。濱松市內還有{{link-ja|濱松大學|浜松大学}}和{{link-ja|濱松學院大學|浜松学院大学}}等四座私立大學。濱松市現在一共有26所高中，58所初中（其中市立初中49所<ref>{{cite web |url=http://www.city.hamamatsu.shizuoka.jp/somu/you-gakkou/chu-ichiran.html|script-title=ja:中学校一覧|accessdate=2014-09-05|publisher={{lang|ja|浜松市}} |language=ja}}</ref>）、102所小學（其中市立小學101所<ref>{{cite web |url=http://www.city.hamamatsu.shizuoka.jp/somu/you-gakkou/shou-ichiran.html|script-title=ja:小学校一覧|accessdate=2014-09-05|publisher={{lang|ja|浜松市}} |language=ja}}</ref>）。

== 交通 ==
濱松市內最重要的車站是[[濱松站|濱松站]]。[[東海旅客鐵道|東海旅客鐵道]]（JR東海）有三條鐵路路線途徑濱松市，分別是[[東海道本線|東海道本線]]、[[飯田線|飯田線]]和[[東海道新幹線|東海道新幹線]]。濱松市當地還有兩家小型的鐵道公司。{{link-ja|遠州鐵道|遠州鉄道}}經營的{{link-ja|遠州鐵道鐵道線|遠州鉄道鉄道線}}是濱松市民重要的通勤、通學交通手段<ref>{{cite web|url=http://www.entetsu.co.jp/tetsudou/akaden_library/kobore.html|script-title=ja:赤電こぼれ話|accessdate=2014-09-05|publisher={{lang|ja|遠州鉄道}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20141114031013/http://www.entetsu.co.jp/tetsudou/akaden_library/kobore.html|archivedate=2014-11-14}}</ref>。{{link-ja|天竜浜名湖鉄道|天龍濱名湖鐵道}}則經營有[[天龍濱名湖線|天龍濱名湖線]]<ref>{{cite web |url=http://www.tenhama.co.jp/station/|script-title=ja:路線と駅|accessdate=2014-09-05|publisher={{lang|ja|天竜浜名湖鉄道株式会社}} |language=ja}}</ref>。這條線路曾是日本國鐵的鐵路路線。濱松市內雖曾有市營公車運行，但已在1986年廢止。現在濱松市的公車大多由遠鐵巴士公司運行。在公路交通方面，[[東名高速公路|東名高速公路]]和[[新東名高速公路|新東名高速公路]]均途徑濱松市。濱松市內還有[[國道1號_(日本)|國道1號]]、{{link-ja|國道42號|国道42号}}、{{link-ja|國道150號|国道150号}}、{{link-ja|國道152號|国道152号}}、{{link-ja|國道257號|国道257号}}、[[國道301號|國道301號]]等重要道路通過。濱松市內雖然有一座{{link-ja|濱松機場|浜松基地}}，但為[[航空自衛隊|航空自衛隊]]用機場，不對一般民眾開放。距離濱松最近的機場是[[中部國際機場|中部國際機場]]<ref>{{cite web|url=http://www.centrair.jp/access/area/city.html|script-title=ja:主要都市へのアクセス|accessdate=2014-09-05|publisher={{lang|ja|中部国際空港 セントレア}} |language=ja}}</ref>和[[靜岡機場|靜岡機場]]<ref>{{cite web|url=http://www.mtfuji-shizuokaairport.jp/access/index.html|script-title=ja:交通アクセス|accessdate=2014-09-05|publisher={{lang|ja|富士山静岡空港}} |language=ja}}</ref>。
