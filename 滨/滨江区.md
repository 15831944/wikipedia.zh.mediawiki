'''滨江区'''是[[中國|中國]][[浙江省|浙江省]][[杭州市|杭州市]]下辖的一个区，位于[[钱塘江|钱塘江]]南岸。1996年12月12日由原浙江省[[萧山市|萧山市]]西兴镇、长河镇和浦沿镇合并成立，滨江区东面与南面与[[萧山区|萧山区]]相临。

== 简介 ==
'''杭州高新技术产业开发区'''建于1990年3月，1991年3月经国务院批准为国家级高新区。滨江区1996年12月经国务院批准设立。开发区管委会与滨江区政府实行[[一個機構兩塊牌子|两个名称、一套人马]]。

== 行政区划 ==
区政府驻江南大道100号，位于西兴街道。

== 经济 ==
[[File:201607_Binjiang_Overview.jpg|thumb]]
[[File:Binjiang_District_15.JPG|thumb]]
2014年，全区实现地区生产总值692.84亿元，增长11.5%；财政总收入169.21亿元，增长20.4%，财政总收入增幅连续三年位居杭州市第一。全区信息经济总收入1273.43亿元，增长30%。全区规上工业销售产值突破千亿元，达1019.52亿元，增长21.8%；实现工业增加值350.03亿元，增长17.1%；万元工业增加值综合能耗0.07吨标煤，为杭州市平均水平的十分之一。

=== 特色产业 ===

目前已初步形成“两强两优两新”的特色产业格局（“两强”指通信设备制造业和软件业，“两优”指集成电路设计制造业和数字电视产业，“两新”指动漫产业和网络游戏产业）；2006年高新技术产业实现技工贸收入885亿元，占全区技工贸总收入的78.3％<ref>[http://www.hhtz.gov.cn/hhtz/DesktopDefault.aspx?tabid=8297157a-989f-4469-bef2-b5aa1b2bd6f9 走进高新（滨江）产业园区] {{webarchive|url=https://web.archive.org/web/20110925153231/http://www.hhtz.gov.cn/hhtz/DesktopDefault.aspx?tabid=8297157a-989f-4469-bef2-b5aa1b2bd6f9 |date=2011-09-25 }}</ref>。具体如下：

*通讯设备制造业
*软件产业
*集成电路设计制造业
*数字电视产业
*动漫、网络游戏产业

== 教育 ==
滨江高教园区位于[[浦沿街道|浦沿街道]]，拥有[[浙江中医药大学|浙江中医药大学]]、[[浙江警察学院|浙江警察学院]]、[[浙江艺术职业学院|浙江艺术职业学院]]、[[浙江机电职业技术学院|浙江机电职业技术学院]]和[[杭州医学院|杭州医学院]]共计五所院校。

[[杭二中|杭二中]]和[[杭州市长河高级中学|长河高中]]是浙江省一级重点中学。

== 交通 ==
[[File:Binjiang_District_14.JPG|缩略图]]
滨江区与江北的杭州主城区通过三座[[钱塘江|钱塘江]][[桥梁|桥梁]]相连，由西到东分别为[[钱塘江大桥|钱塘江大桥]]、[[复兴大桥|复兴大桥]]和[[西兴大桥|西兴大桥]]。另外滨江通过西边的[[钱江七桥|钱江七桥]]（之江大桥）和[[钱江五桥|钱江五桥]]与[[西湖区|西湖区]]的[[双浦镇|双浦镇]]相连。

[[江南大道_(杭州)|江南大道]]为滨江区内东西走向的主干道路；北连[[复兴大桥|复兴大桥]]的时代大道为南北走方的主干路。其他东西向的主干路还有滨盛路、滨安路、滨康路和南环路。

[[杭州地铁1号线|杭州地铁1号线]]在滨江区设有[[江陵路站|江陵路站]]、[[滨和路站|滨和路站]]、[[西兴站|西兴站]]和[[滨康路站|滨康路站]]。[[杭州地铁4号线|杭州地铁4号线]]在滨江区设有[[联庄站|联庄站]]、[[中医药大学站_(杭州市)|中医药大学站]]、[[杨家墩站|杨家墩站]]和[[浦沿站|浦沿站]]。规划中的[[杭州地铁6号线|杭州地铁6号线]]长25公里，主体位于滨江区内，西起[[浦沿街道|浦沿]]，经浦沿路、东信大道与中兴路，连接公共次中心，沿江边连接[[钱江世纪城|钱江世纪城]]到钱江文化产业园。

== 医院 ==
滨江目前有武警浙江医院、[[浙医二院滨江院区|浙医二院滨江院区]]、[[浙江大学医学院附属儿童医院|浙江大学医学院附属儿童医院]]三家大型医院。
