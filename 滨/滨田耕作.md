'''滨田耕作'''（{{jpn|j=浜田耕作|hg=はまだ こうさく}}，1881年2月22日 - 1938年7月25日[[7月25日|<nowiki/>]] <ref>[http://dl.ndl.go.jp/info:ndljp/pid/2959963/16 「彙報　官庁事項　官吏薨去及卒去」『官報』第3472号、昭和13年7月30日、p.1218]</ref>），号'''青陵'''（せいりょう），日本[[考古学|考古学家]]。[[京都大学|京都帝国大学]]教授、[[校長|校長]]。被称为「日本近代考古学之父」。他的儿子{{tsl|ja|浜田敦|滨田敦}}也是京都大学教授，研究日语。

== 生平 ==
岸和田藩上級[[藩士|藩士]]滨田家的长子，出生于[[大阪府|大阪府]][[岸和田市|岸和田市]]。就读于大阪府立北野中学校（現{{Link-ja|大阪府立北野高等学校|大阪府立北野高等学校}}），后被开除，转学到[[東京府|东京府]]旧制早稻田中学校（現{{Link-ja|早稻田中学校・高等学校|早稲田中学校・高等学校|早稻田中・高等学校}}）。之后，在[[舊制第三高等學校|第三次高等學校]] <ref>{{Citation|title=第三高等学校一覧 明治35年9月起明治37年8月止|url=http://dl.ndl.go.jp/info:ndljp/pid/812907/69 |year=1902|page=127|publisher=第三高等学校}}</ref>（現在的[[京都大学|京都大学]]综合人间学部）、[[东京大学|东京帝国大学]]专攻艺术史，1905年毕业<ref>{{Citation|title=東京帝国大学一覧 従明治39年至明治40年|url=http://dl.ndl.go.jp/info:ndljp/pid/813182/319|year=1907|page=208|publisher=東京帝国大学}}</ref>。之后，留学欧洲继续研究[[考古学|考古学]]。返回日本后，任[[京都大学|京都帝国大学]]考古学研究室第一任教授。1918年，被授予[[文学博士|文学博士]][[学位|学位]]<ref>[http://dl.ndl.go.jp/info:ndljp/pid/2953994/6 『官報』第1880号、大正7年11月8日、p.155]</ref>。他在《通論考古学》一书中，给考古学下了定义：<ref>{{Citation|title=通論考古学|url=http://dl.ndl.go.jp/info:ndljp/pid/964457/27 |year=1922|ref=harv|author=濱田耕作|page=11|publisher=大鐙閣}}</ref>“考古学是研究人类的物质遗物（基于人类的过去）的学问。”这本书长期以来一直作为考古学教科书而广受欢迎，它提高了日本考古学的水平，并使考古学在日本传播开来。

他发现了{{Link-ja|梅原末治|梅原末治}}、{{Link-ja|末永雅雄|末永雅雄}}、{{Link-ja|小林行雄|小林行雄}}等人，形成了[[京都學派|京都学派]]的考古学。他将欧洲考古研究方法纳入日本传统考古方法中，并进一步调查了中国和朝鲜半岛的亚洲[[遗址|遗址]]，为日本考古研究的发展做出了巨大贡献。

[[1937年|1937年]]就任京都帝国大学校長。1938年春后，生病住入京都帝国大学病院治疗，恰逢发生了清野事件（[[醫學院|医学部]][[教授|教授]]清野謙次从古寺盗取古文書等被逮捕、起訴，被从京大免職）。滨田耕作因感到自己有责任而在7月初辞职，尽管处于病危状态，他还是出院返回家中，最终因[[肾萎缩|肾萎缩]]而导致[[尿毒症|尿毒症]]并发去世。

死後埋葬在[[京都市|京都市]][[左京區|左京区]][[法然院|法然院]]墓地。1988年，岸和田市和朝日新聞社共同以他的名字设立了滨田青陵奖，授予取得考古学、歴史、美術等优秀研究成果的人。

== 年譜 ==
* [[1881年|1881年]]2月22日，出生于大阪府岸和田市。
* [[1894年|1894年]]，进入大阪府立第一尋常中学校学习。
* [[1898年|1898年]]，受到退校处分，转入早稻田中学。
* [[1899年|1899年]]，进入第三高等学校学习。
* [[1902年|1902年]]，进入東京帝国大学文科大学史学科学习。
* [[1905年|1905年]]，大学毕业後，进入研究生院。参与美術雜誌“{{Internal link helper/ja|国华（日本期刊）|国華|国华}}”的编辑。
* [[1909年|1909年]]，与野村琴寿結婚。就任京都帝国大学文科大学講師。
* [[1913年|1913年]]，就任京都帝国大学文科大学助教授，前往欧洲留学。
* [[1916年|1916年]]，负责京都帝国大学文科大学、考古学講座。
* [[1917年|1917年]]，就任京都帝国大学文科大学教授。
* [[1925年|1925年]]，与[[島村孝三郎|島村孝三郎]]、{{Link-ja|原田淑人|原田淑人}}等人一起創立东亚考古学会。
* [[1931年|1931年]]，被推举为{{Link-ja|帝国学士院|帝国学士院}}会員。
* [[1937年|1937年]]，就任京都帝国大学总長。
* [[1938年|1938年]]7月25日逝去，葬于京都鹿谷法然院。
