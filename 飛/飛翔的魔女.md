《'''飛翔的魔女'''》（{{lang-ja|ふらいんぐうぃっち}}，{{lang-en|Flying Witch}}）是[[石塚千尋|石塚千尋]]原作的[[日本|日本]][[漫畫|漫畫]]作品，由[[講談社|講談社]]出版，於2012年8月9日發售的《[[別冊少年Magazine|別冊少年Magazine]]》2012年9月號開始連載<ref>{{Cite web|url=http://www.amazon.co.jp/dp/B008PUN0OU|title=《別冊少年Magazine》2012年9月號|publisher=[[日本亞馬遜|amazon.co.jp]]|accessdate=2016-05-18|language=ja}}</ref>。

== 故事簡介 ==
來自[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]]的見習魔女「木幡真琴」是位年僅15歲的少女，為了遵循新人魔女修練的傳統，她帶著黑貓使魔「千戶」來到位於[[青森縣|青森縣]][[弘前市|弘前市]]的表兄妹「圭」和「千夏」家中寄宿，展開充滿驚喜的魔女日常。

== 登場人物 ==
=== 主要人物 ===
; {{nihongo|木幡真琴|こわた まこと||聲：[[篠田南|篠田南]]}}
: 生日2月22日，血液O型，身高160cm。
: [[魔女|魔女]]，高中一年級，和阿圭同班。活潑、充滿好奇心。嚴重路癡。目前為了魔女的傳統而寄住在遠房親戚家，並學習各種魔女的工作。有時會在阿圭家中迷路。喜歡吃美食，阿圭每次推薦一些沒吃過的料理時會很興奮。在原本為阿圭奶奶生前耕作的田種菜。

; {{nihongo|倉本圭|くらもと けい||聲：[[菅原慎介|菅原慎介]]}}
: 生日2月2日，血液O型，身高175cm。
: 高中一年級，真琴的堂兄，真琴都稱其為“阿圭”。個性穩重，十分可靠，講話有條理，做事時會先列出一些注意事項。不放心讓真琴自己出門。非常擅長料理、園藝。每年會固定幾天和千夏去山上採野菜。害怕鬼，有一次進鬼屋時全程不敢看，緊抓著真琴不放。不相信許願。平時會看電影。

; {{nihongo|倉本千夏|くらもと ちなつ||聲：[[鈴木繪理|鈴木繪理]]}}
: 生日8月8日，血液O型，身高127cm。
: 圭的妹妹，茜的弟子。天真活潑。一開始有點害怕真琴。喜歡真琴做的魔女長袍。不喜歡吃野菜。被帶來春天的使者的長相嚇到，以為是壞人，之後了解原因。帶來春天的使者則送一盆牽牛花給千夏作為賠禮。跟真琴養的貓千戶處得不錯。

; {{nihongo|木幡茜|こわた あかね||聲：[[葵井歌菜|葵井歌菜]]}}
: 生日5月1日，血液A型，身高170cm。
: 魔女，真琴的姊姊，古銅色肌膚、白色頭髮為其特色。喜歡別人的八卦，是個非常優秀的魔女，在魔女界算是有名的。喜歡到處旅行流浪，目前因為告一段落，也借住到阿圭家，順便教導妹妹一些魔術。因為千夏想成為魔女而成為師傅，決定讓千夏先從見習魔女開始。

; {{nihongo|石渡那央|いしわたり なお||聲：[[三上枝織|三上枝織]]}}
: 生日12月21日，血液A型，身高162cm。
: 真琴的同班同學，與圭是中學時代的同學。家是經營酒鋪的，會幫忙送貨以賺取零用錢。喜歡番茄，討厭青蛙。真琴原本在野外發現稀有的植物風茄想送給她，但因為風茄十分怪異而拒絕。

; {{nihongo|犬養|いぬかい||聲：[[日野麻里|日野麻里]]}}
: 生日10月2日，血液B型，身高170cm。
: 茜的朋友，秋田县出身。因醉酒誤食茜做的魔法藥而變的長的像狗，唯有晚上才能恢復成原樣。曾經長期設占卜小攤來等待木幡茜，但結果等到了她的妹妹真琴，之後還是找到了茜並在其提醒後想起是自己喝醉誤食導致。長相是阿圭喜歡的類型。因白天為狗的樣子而改上大學夜間部。已经从茜得到解药，但是为了讨侄儿开心而没使用。

; {{nihongo|椎名杏子|しいな あんず||聲：[[井口裕香|井口裕香]]}}
: 生日7月1日，血液AB型，身高150cm。
: 魔女，初中生。是普通人無法進入的咖啡廳店主的女兒。與母親和幽靈小姐一起做招待。喜歡研究考古，看見古老建築物之類的東西就會興奮起來。使魔为一只叫奧羅爾的貓頭鷹。

=== 倉本家 ===
; {{nihongo|倉本奈奈|くらもと なな||聲：[[井上喜久子|井上喜久子]]}}
: 圭與千夏的母親，為繪本作家。善於掌握年輕人的風格，對大多事物不感到動搖。

; {{nihongo|倉本啟司|倉本啓司||聲：[[小形滿|小形滿]]}}
: 圭與千夏的父親，為務農人、類似奈奈的寬容性格。操一口濃濃的[[津輕市|津輕]]方言，與真琴和茜溝通需要透過圭和千夏的翻譯。

=== 咖啡廳 ===
; {{nihongo|杏子母親|杏子の母||聲：[[早水理沙|早水理沙]]}}
: 咖啡廳「孔庫魯西歐」的店主，不拘小節的強力魔女。為了營造與「彼岸」交流的場所對咖啡廳施加魔法，普通人從外觀看來只是廢墟。

; {{nihongo|雛|ひな|聲：[[金元壽子|金元壽子]]}}
: 生日6月24日，血液A型，身高164cm。
: 幽靈，[[明治|明治]]39年（1906年）出生。是咖啡廳裡的服務生，也會做甜點。因容易害羞而把自己隱形，僅用紙條和客人聯絡。阿圭一開始很害怕她。因真琴做的魔法陣而現出原形。

=== 使魔 ===
; {{nihongo|奇多|チト||聲：[[茅野愛衣|茅野愛衣]]}}
: 黑貓，真琴的使魔。活了17年（相當於人類84歲）。聽得懂人話，喜歡進行小冒險，愛捉弄狗。
: 因被直說胖了一圈，而賭氣的站在直的肩膀上。

; {{nihongo|肯尼|ケニー||聲：[[佐倉綾音|佐倉綾音]]}}
: 白貓，茜的使魔。熟悉人類學、考古學，博學。性格很平靜。

; {{nihongo|艾爾|アル||聲：[[小澤亞李|小澤亞李]]}}
: 倉鼠，犬養的使魔。

=== 其他 ===
; {{nihongo|帶來春天的使者|春の運び屋さん||聲：[[櫻井孝宏|櫻井孝宏]]}}

== 用語 ==
; {{nihongo|魔女|魔女}}
: 本作主角的身份，從古以來的習俗，15歲就是成人，得一個人踏入社會。而且魔女的身分不能向外人透露。
: 在東北地區有很多魔女，因為自然環境佳、資源豐富，方便施展魔術，深受好評。
: 常帶著動物作為使魔。
: 從普通人變成魔女的過程非常辛苦，甚至可能斷送普通人的人生。
: 魔女進入森林時會異常地興奮。

; {{nihongo|[[曼德拉草|曼德拉草]]|マンドレイク}}
: 又稱風茄，非常珍稀，散發氨的氣味，拔出時會發出巨大的聲音，一般人聽到可能會因此死亡。毒性強，但去毒能做成萬能藥，也能泡茶。

; {{nihongo|孔庫魯西歐|コンクルシオ}}
: 是一間開了200年的咖啡店，為杏子的媽媽所開，她對咖啡店使用魔法隱形，故一般人看不到咖啡店，只會看到殘破的建築物。需用二禮二拍一禮（參拜神社的參拜方法）才能解除。動物或魔法相關人士能進去此咖啡店。

=== 網路動畫 ===
標題名為{{日本作品|飛翔的魔女Puchi|ふらいんぐうぃっちぷち}}，內容是和正篇相關的小插曲及廣告的2分鐘宣傳短片，並收錄在BD和DVD。2016年3月19日至6月24日於[[YouTube|YouTube]]和日本電視台的子公司[[VAP|VAP]]官方網站進行更新，第二話（4月1日更新）之後隔週星期五更新。2017年4月現在，發佈《飛翔的魔女Puchi》BD&DVD Vol.1發售記念篇為主題的短片<ref>{{cite web|url=http://www.flyingwitch.jp/movie/index.html|title=STORY|work=【Movie】電視動畫「飛翔的魔女」公式官網|accessdate=2016-05-26|language=ja}}</ref>。登場人物則是用二頭身加3D建模作設計，而使魔的體型在此相當於真琴的一半。

==== 製作人員 ====
* 導演－市川量也
* 分鏡、演出－甲田菜穗
* 劇本－赤尾凸
* Puchi人物設定－關繪理奈
* 音響監督－岩浪美和
* 音樂－出羽良彰
* 動畫製作－{{lang|ja|ドメリカ}}
* 製作－[[VAP|VAP]]
