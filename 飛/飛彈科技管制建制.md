'''飛彈科技管制建制'''（{{lang-en|Missile Technology Control Regime, MTCR}}）於1987年4月成立。目標是規範與管制國家的[[飛彈|飛彈]]技術輸出與擴散。飛彈科技管制建制主要始於美國前總統雷根1982年簽署的第70號國家安全命令。<ref>United States, ''National Security Directive'', No. 70 November 30, 1982, {{cite web |url=https://www.reaganlibrary.archives.gov/archives/reference/Scanned%20NSDDS/NSDD70.pdf |title=存档副本 |accessdate=2016年10月26日 |deadurl=yes |archiveurl=https://web.archive.org/web/20161224084548/https://reaganlibrary.archives.gov/archives/reference/Scanned%20NSDDs/NSDD70.pdf |archivedate=2016年12月24日 |df= }}.</ref>隨後美國便與西方國家進行協商，最初的參與國有美國、英國、德國、法國、義大利、加拿大與日本，<ref>MTCR, [http://mtcr.info/partners/ ''MTCR Partners''].</ref>
目前為只有35個會員國<ref><[http://www.cna.com.tw/news/aopl/201606270355-1.aspx 印度順利加入飛彈技術管制協定]>，《[[中央通訊社|中央通訊社]]》，2016年6月27日，引用日期：2016年9月1日，[https://web.archive.org/web/20160628135557/http://www.cna.com.tw/news/aopl/201606270355-1.aspx 存檔頁面]。</ref>
== 規定 ==
只要系統載重超過500公斤、射程超過300公里，可搭載[[大規模殺傷性武器|大規模殺傷性武器]]（WMD）的系統與技術都將管制，列點如下。
#完整火箭系統：弹道导弹、太空發射載具、探測火箭
#無人駕駛航空器：巡弋飛彈、無人機、遠遙控飛機
