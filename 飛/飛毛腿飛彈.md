'''飞毛腿导弹'''（Scud）是一个已经被大众接受了的词汇，指[[苏联|苏联]]在[[冷战|冷战]]时期开发并被广泛出口的一系列的[[战术|战术]][[弹道导弹|弹道导弹]]。这个名称是从[[北約代號|北約代號]]为'''SS-1飞毛腿'''得来的，它包括了苏联的从'''R-11'''、'''R-17'''、 '''R-300厄尔布鲁士'''的一系列导弹。飞毛腿这个名字还被媒体指别的国家根据苏联原型广泛发展的许多种导弹。如在美国有时指飞毛腿SCUD被泛指为任何国家的不是从西方原型发展出来的[[弹道导弹|弹道导弹]]。

== 研制 ==
[[File:9P117_9K72.jpg|right]] (9P117)发射车与8K14火箭]]

第一次「飞毛腿（Scud）」这个词是[[北约|北约]]将R-11[[弹道导弹|弹道导弹]]称为'''飞毛腿-A'''型（SS-1b）。更早的{{link-en|R-1导弹|R-1 (missile)}}被[[北约|北约]]称为'''SS-1 Scunner'''是一个非常不一样的设计，几乎是[[德国|德国]][[V2火箭|V2火箭]]的直接翻版。[[R-11|R-11]]同样采用了V-2的技术，但它是一个全新的设计，更小，与V-2和R-1的形状不一样。R-11是由[[S.P.__Rocket_and_Space_Corporation_Energia|Korolyev OKB]]<ref name="Wade R-17">{{cite web Korolev|url=http://www.astronautix.com/lvs/r17.htm |title=R-17 |accessdate=2008-02-13 |last=Wade |first=Mark |publisher=[[Encyclopedia_Astronautica|Encyclopedia Astronautica]]}}</ref>[[:en:Makeyev_OKB|Makeyev OKB]] 设计并在1957年服役的。R-11最有革命性的创新是[[Aleksei_Mihailovich_Isaev|A.M. Isaev]]设计的引擎，比V-2的多室设计（multi-chamber design）远为简单，并且用防震荡折流板（anti-oscillation baffle）防止间歇燃烧。这是苏联太空火箭所使用的更大的引擎的设计先驱。

后来发展的变种有1961年的'''R-300厄尔布鲁士/飞毛腿-B（ SS-1c）'''和1965年的'''飞毛腿-C（SS-1d ）'''，都可以携带5-80千吨的常规高爆的[[核武器|核武器]]或[[化学武器|化学武器]]（加料的[[VX_(神經毒劑)|VX]]）弹头。1980年代发展的'''飞毛腿-D（SS-1e）'''可以携带一个常规高爆弹头，一个[[燃料空气炸弹|燃料空气]]弹头，40个反[[跑道|跑道]]炸弹或者100个5千克的人员杀伤性小炸弹。

所有型号都是11.25米长（除了飞毛腿Scud-A，短了一米）直径0.88米。推动引擎是用煤油和硝酸的单引擎（飞毛腿-A）或者用[[UDMH|UDMH]]和[[RFNA|RFNA]]（Russian SG-02 Tonka 250）的引擎（别的型号）。

最大速度5马赫。

== 型号 ==
===苏联===
====R-11====
1951年，由[[Sergey_Korolyov|Sergey Korolev]]领导的[[OKB-1|OKB-1]]的工程师[[Victor_Makeev|Victor Makeev]]开始设计R-11导弹系统。1953年4月18日首飞，使用伊萨耶夫设计的煤油+硝酸作为燃料的引擎。1953年12月13日，位于[[Zlatoust|Zlatoust]]的[[SKB-385|SKB-385]]工厂开始生产该型号。1955年6月，Makeev成为SKB-385的总设计师。1955年7月，R-11正式入役。<ref name="Wade R-11">{{cite web |url=http://www.astronautix.com/lvs/r11.htm |title=R-11 |accessdate=2008-02-17 |last=Wade |first=Mark |publisher=[[Encyclopedia_Astronautica|Encyclopedia Astronautica]]}}</ref>1958年4月1日，携带核弹头的R-11M入役。发射系统型号为8K11。<ref>Zaloga, p.7</ref>

R-11M最大射程270 km，如果携带50kt核弹头则最大射程为150 km.<ref>Zaloga, p.4</ref> <ref name="Jane's">{{cite web |url=http://www.janes.com/security/international_security/news/misc/sws_scud010426.shtml |title=SS-1 `Scud' (R-11/8K11, R-11FM (SS-N-1B) and R-17/8K14) |accessdate=2008-02-12 |date=26 April 2001 |work= |publisher=[[Jane's_Information_Group|Jane's Information Group]] |archiveurl = https://web.archive.org/web/20071215141138/http://www.janes.com/security/international_security/news/misc/sws_scud010426.shtml <!-- Bot retrieved archive --> |archivedate = 2007-12-15}}</ref>

潜射型号R-11FM (SS-N-1 Scud-A)于1955年2月在试飞,1955年9月在[[Project_611|Project 611]] (祖鲁级)潜艇上试射。型号研制于1955年8月转移给Makeev的SKB-385。<ref name="Wade R-11"/>1959年实战部署在611级及[[Project_629|Project 629]] (高尔夫级)弹道导弹常规潜艇. 服役期间，发射77次，其中59次成功。<ref>{{cite web |url=http://www.fas.org/nuke/guide/russia/slbm/r-11fm.htm |title=R-11FM / SS-1b Scud |accessdate=2008-02-19 |date=July 13, 2000 |publisher=[[Federation_of_American_Scientists|Federation of American Scientists]]}}</ref>


====R-17==== 
[[File:Wz8K14_RB3.jpg|thumb]]
R-11的后继型号是R-17 (SS-1C 飞毛腿-B), 并在1970年代改称R-300，产量估计7000套。服役于32个国家，还有4个国家仿制。<ref name="Jane's"/>1961年首飞，1964年入役。<ref name="Wade R-17"/>

R-17可携带常规、核、化学或集束弹头。<ref name="Jane's"/>最初使用履带式底盘作为2P19[[transporter_erector_launcher|transporter erector launcher]] (TEL), 但不久改为Titan Central Design Bureau的轮式底盘，1967年入役。<ref>Zaloga, pp.14-15</ref> [[MAZ-543|MAZ-543]]车辆作为9P117 ''Uragan''.使用液压动力把导弹起竖，这需要4分钟才能发射。<ref name="Jane's"/>

====飞毛腿-C====
Makeyev  OKB研制的R-17增程型，西方称为飞毛腿-C，1965年在Kapustin Yar卡普斯京亚尔靶场首飞。, 射程500–600 km，但战斗荷载降低了。当时苏联有更好的替代装备，如[[TR-1_Temp|TR-1 Temp]] (SS-12 Scaleboard),因此飞毛腿-C并没有被苏军列装。<ref name="Zaloga1">Zaloga, p.17</ref>

====飞毛腿-D==== 
R-17 VTO增强了R-17的精度。The Central Scientific Research Institute for Automation and Hydraulics (TsNIAAG)1968年开始这一项目。1979年9月首飞。1989年入役，型号为9K720 Aerofon。<ref name="Z_19">Zaloga, p.19</ref> 此时，苏军有了更为先进的[[OTR-21_Tochka|OTR-21 Tochka]] (SS-21) 与[[R-400_Oka|R-400 Oka]] (SS-23), 飞毛腿-D针对出口市场。<ref name="Z_19"/>

====劳动-1====

===伊拉克===
"飞毛腿"这个名字也被用来指同一种导弹经[[伊拉克|伊拉克]]修改的版本。修改后射程增加，当许多导弹向[[以色列|以色列]](40)和[[沙特阿拉伯|沙特阿拉伯]](46)发射时，这一点尤为突出。美国制造的[[爱国者导弹系统|爱国者导弹系统]]据说成功地拦截了导弹，但是许多评论家声称爱国者导弹的作用被严重夸张了，实际上有85%的失败率。这些导弹是伊拉克最有危险的进攻性武器之一，尤其是对于以色列而言。携带生化弹头的可能性也被广泛关注。

最终，飞毛腿导弹直接造成了一名以色列人，以及28名美国士兵（导弹击中了位于沙特阿拉伯首都[[利雅得|利雅得]]的一座[[宾夕法尼亚州|宾夕法尼亚州]][[国民警卫队|国民警卫队]]兵营）的死亡。针对飞毛腿导弹的搜索耗用了自愿联盟空军大约三分之一的力量。借助于“运输-起竖-发射”（TEL）三用车，这些导弹可以被非常机动地运输，并且难以被捕捉；美国和英国的特种部队成员经常承担在敌军的防线后方搜索并破坏这些导弹的任务。对于上述集团而言，消除来自飞毛腿导弹的威胁是有着特殊重要性的。正如以色列曾经扬言，如果侵袭继续，便将加入对伊拉克的战争，但这会分裂反以阿拉伯国家组成的攻打伊拉克的联盟，也会大大加重美国和英国的战争代价。

伊拉克发展了四种改型：飞毛腿导弹（Scud）、远程飞毛腿导弹（longer-range Scud或Scud LR）、[[侯赛因|侯赛因]]（Al Hussein）和[[阿巴斯|阿巴斯]]（Al Abbas）。除了那些几乎未经改造的武器之外，这些导弹并不成功，因为它们往往在飞行过程中崩解，并且只能搭载小弹头。
