{{engname|《飛虎》|''First Option''}}為於1996年首映的[[香港電影|香港電影]]，由[[陳嘉上|陳嘉上]]執導，由[[王敏德|王敏德]]和[[梁詠琪|梁詠琪]]主要演出。

== 劇情 ==
電影描述[[香港海關|香港海關]]及[[皇家香港警察隊|皇家香港警察隊]][[特別任務連|特別任務連]]（[[特別任務連|飛虎隊]]）與販毒集團的鬥爭，海關毒品調查科人員在掃蕩販毒據點時，漏了有「亞洲冰-{后}-」之稱的主犯。接下來特別任務連依照[[情報|情報]]，來到廢棄[[工廠|工廠]]，卻遭到[[地雷|地雷]][[埋伏|埋伏]]和[[狙擊手|狙擊手]]的伏擊，特別任務連損失慘重，同時又調查出人員所中的[[子彈|子彈]]，原來是[[員警|員警]]才有的特製槍械所發。後來查出男主角 Don 的好友——[[美國緝毒局|美國緝毒局]](DEA)探員 Rick 竟是販毒團夥的真正頭目，他在正準備運毒品出海，特別任務連人員和[[關員|關員]]深入離島叢林，與[[海豹突擊隊|海豹突擊隊]]作戰，終於將 Rick 擒獲。

== 獎項 ==
《飛虎》獲得[[金馬獎|金馬獎]]最佳男主角、最佳剪輯與最佳音效提名<ref>[http://www.starblvd.com/cgi-bin/Movie/MV_Film?file=FirstOption]</ref>，獲得第3屆[[香港電影評論學會大獎|香港電影評論協會]]最佳導演獎。<ref>{{cite web |url=http://www.lesliecheung.com.cn/archive/hkfilm/view-348.html |title=存档副本 |accessdate=2010-07-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20100731152153/http://www.lesliecheung.com.cn/archive/hkfilm/view-348.html |archivedate=2010-07-31 }}</ref>
