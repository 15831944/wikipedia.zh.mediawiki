'''飛特立航空'''為[[台灣|台灣]]經營商務專機及醫療救援專機業務的通用航空業<ref>{{cite web |url=https://www.ettoday.net/news/20130903/265377.htm |title=搶 550 億頂級客層商機　飛特立航空推商務、醫療包機 |date=2013-09-03 |publisher=東森新聞雲 |language=zh-tw}}</ref>，成立於2010年。樞紐機場為[[臺北市|臺北市]][[松山機場|松山機場]]及高雄市[[高雄國際機場|高雄國際機場]]，維護基地設在高雄市高雄國際機場。飛特立航空主要業務包括商務專機、醫療專機、地勤服務、飛機採購、託管、銷售及維護管理等。2018年除開始執行[[金門縣|金門縣]]的緊急醫療後送業務<ref>{{cite web |url=https://udn.com/news/story/7266/3147629 |title=飛特立航空 全球救援發表全台最大醫療專機隊 |date=2018-05-17 |publisher=聯合新聞網 |language=zh-tw}}</ref>外同年7月與飛行訓練學校安捷航空簽約，未來機師缺額將優先錄取安捷航空結業機師，以因應公司2021年預計擴充機隊之機師需求<ref>{{cite web |url=http://www.chinatimes.com/newspapers/20180608000671-260107 |title=飛特立擴版圖 優先錄用安捷機師|date=2018-06-08 |publisher=中時電子報 |language=zh-tw}}</ref>。

== 飛安事件 ==

*2014年3月25日，一架{{link-en|霍克400|Hawker 400}}噴射機（國籍標誌及登記號碼：B-95995）執飛[[松山機場|松山機場]]途經[[金門機場|金門機場]]至[[馬祖|馬祖]][[南竿機場|南竿機場]]之包機載客任務。由航管引導至南竿機場北方，準備在其降落在21跑道前先進行「[[儀器著陸系統|儀器進場]]」程序，由於飛航路徑為「由北向南」，但航機在進場前上空時，因故無法執行「儀器進場」程序，飛航組員又恰巧目視到[[北竿機場|北竿機場]]的21[[跑道|跑道]]（距離約2.3浬），將其誤判為南竿機場21跑道（距離約6.8浬）後，隨即進行目視降落<ref>{{cite web|url=https://www.nownews.com/news/20150501/1682305 |title=飛南竿誤降北竿...　私人包機遭提25點飛安改善建議|publisher=今日新聞 |date=2015-05-01 |language=zh-tw}}</ref><ref>{{cite web|url=https://www.asc.gov.tw/main_ch/docDetail.aspx?uid=227&pid=227&docid=687 |title=事故調查報告-飛特立航空公司B-95995航機誤降落於馬祖北竿機場 |publisher=中華民國行政院飛航安全委員會 |date=2015-04-30 |language=zh-tw}}</ref><ref> [https://www.asc.gov.tw/upload/acd_att/ASC-AOR-15-04-001.pdf ASC-AOR-15-04-001 飛安事故調查報告]飛航安全調查委員會 2014-03-25</ref>。事故發生後，航機及乘員皆無損傷。
