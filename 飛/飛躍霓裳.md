{{engname|《飛躍霓裳》|'' A Friend in Need ''}}是[[電視廣播有限公司|電視廣播有限公司]]在1988年播出的電視劇，由[[梅小青|梅小青]]監製，[[黎明|黎明]]、[[羅慧娟|羅慧娟]]、[[戚美珍|戚美珍]]、[[任達華|任達華]]、[[梁珮玲|梁珮玲]]、[[邵仲衡|邵仲衡]]等演出，於1988年4月18日至5月13日期間在[[無綫電視|無綫電視]][[翡翠台|翡翠台]]播出<ref>{{Cite web |url =https://programme.mytvsuper.com/tc/116786/%E9%A3%9B%E8%BA%8D%E9%9C%93%E8%A3%B3|title = 飛躍霓裳| publisher =myTV SUPER|language =中文}}</ref>。

==劇情簡介==
劇情講述林欣出生於小康家庭，中學預科畢業後由於未能升讀大學，於是投身社會，在時裝製衣公司工作。姜舜苓是林欣的上司，工作能幹，對初出茅廬的林欣照顧有加，讓林欣在公司學習時裝設計，林欣視之為恩師。林欣有一個要好的男朋友方浩明，自從林欣投身社會之後，與正在就讀大學的方浩明產生了隔膜，方浩明與林欣的舊同學阮潔莊交往甚密，後來更奉子成婚，使到林欣非常傷心，鄰居關學文前來開解林欣。原來關學文對林欣早有愛意，但由於性格內向，一直不敢向林欣表白。林欣在工作的時候認識了攝影師夏祖蔭，夏祖蔭追求林欣，兩人墮入愛河，卻招來上司姜舜苓的妒忌，在工作上處處打擊林欣。林欣與夏祖蔭很快便因為性格不合而分手，林欣在公司得不到重視，決定辭職，事業和愛情皆失意，幸好得到關學文的關心。這時候，林欣發現一直以來默默支持自己的關學文，原來就是理想的對象<ref>{{Cite web |url = https://drive.google.com/open?id=17Hp9swDtmh-rr-h3SV7HvKUskVrFcM75 |title =翡翠劇集－飛躍霓裳劇情及人物介紹|publisher =歡樂無線雜誌|language =中文}}</ref><ref>{{Cite web |url = https://drive.google.com/open?id=1IgI1mm8VbtNv8UzZVV9v8y7G8mWssNsO |title =翡翠劇集－飛躍霓裳人物介紹|publisher =歡樂無線雜誌|language =中文}}</ref>。
