《'''飛虎奇兵'''》（[[有線兒童台|有線兒童台]]譯名），[[台灣|台灣]]中視譯名《'''奪寶奇謀'''》（日本原名：'''{{lang|ja|国際共同制作 モンタナ・ジョーンズ}}'''，英文： '''{{lang|en|Montana Jones}}''' ；後改：'''{{lang|ja|冒険航空会社（アドベンチャー・エアライン）モンタナ}}'''，英文： '''{{lang|en|Adventure Airline Montana}}''' ），為由[[意大利|意大利]][[REVER|REVER]] 、[[日本|日本]]{{lang|ja| NHK エンタープライズ}} 和 {{lang|ja|NHK ソフトウェア}}「共同製作」的冒險類[[電視動畫|電視動畫]]，全輯共52集。

日本[[NHK綜合頻道|NHK綜合頻道]]首播於1994年4月2日，1995年4月8日播完；重播於[[NHK教育頻道|NHK教育頻道]]，自2003年4月7日，2004年3月22日再終，但只播了49集。

台灣[[中視|中視]]首播於1995年9月6日至1995年11月24日，52集全部播映完畢。香港首播於[[香港有線電視兒童台|香港有線電視兒童台]]，一九九六年左右。以動畫截圖組合而成的[[漫畫|漫畫]]版台灣中文版[[單行本|單行本]]由[[長鴻出版社|長鴻出版社]]發行，全套共10集。

==故事大綱==
在1930年代的[[美國|美國]][[波士頓|波士頓]]海岸，開辦航空運輸的鍾斯與[[考古學家|考古學家]]阿發，收到（{{lang|ja|ギルト博士}}）的指令，到世界各地取回稀世珍奇的寶藏，以免落入由芝魯王率領的盜寶團手中。一場橫渡五洋七洲、穿越機關陷阱、比腦力比手力的冒險競賽，就此展開。

==主要人物==
===已方===
; 鍾斯（{{lang|ja|モンタナ・ジョーンズ}}）
:（配音：[[大塚明夫|大塚明夫]]）
: 阿發的堂或表兄。非常外向，冒險精神、身手及自信心都超強，經常受（{{lang|ja|ギルト博士}}）所托，將世界各地的寶藏帶回博物館（[[大英博物館|大英]]？），常與芝魯王及其手下鬥智鬥力。擁有水上飛機「姬蒂」，用作遊行及運輸，但航空會社經常赤字……

; 阿發（{{lang|ja|アルフレッド・ジョーンズ}}）
:（配音：[[中尾隆聖|中尾隆聖]]）
: 鍾斯的堂或表弟。非常內向，精於研究、閱讀及考古，隨鍾斯同行並輔助他尋寶。喜歡吃自己媽媽弄的意粉（阿發媽媽於[[波士頓|波士頓]]開餐館），又常常說「搭鍾斯架飛機，真係九條命都唔夠……」，因為鍾斯在必要時會進行花式飛行，加上阿發會暈機浪，另外又不懂游泳。

; 瑪麗莎（{{lang|ja|メリッサ・ソーン}}）
:（配音：[[岩男潤子|岩男潤子]]）
: 外交官，到最後才知是（{{lang|ja|ギルト博士}}）的女兒。善多國語言，因為不喜歡平凡的生活而與鍾斯和阿發一起冒險。與阿發一樣，瑪麗莎也會不滿鍾斯的危險駕駛。

; ？（{{lang|ja|ギルト博士}}）
:（配音：[[糸博|糸博]]）
: 博物館（[[大英博物館|大英]]？）的一人，也是阿發的上司，幾乎從未露面，每次也是將一隻會自爆的[[唱片|黑膠唱片]]寄給鍾斯，告訴他任務的內容。

===敵方===
; 芝魯王（{{lang|ja|ゼロ卿}}）
:（配音：[[大友龍三郎|大友龍三郎]]）
: 盜寶團的首領，每次都與鍾斯等人展開角力，每次都失敗……

; ？（{{lang|ja|ニトロ博士}}）
:（配音：[[瀧口順平|瀧口順平]]）
: 機械科學家，為芝魯王製造了不少機器來尋寶和對付鍾斯等人。對自己的製成品付出很多感情。

; ？（{{lang|ja|スリム}}）
:（配音：[[櫻井敏治|櫻井敏治]]）
: 芝魯王的手下之一，很強壯，但不聰明。

; ？（{{lang|ja|スラム}}）
:（配音：[[長島雄一|長島雄一]]）
: 芝魯王的手下之一，很狡猾，但不忠心。

==備註==
* 《'''飛虎奇兵'''》與另一套動畫《[[名偵探福爾摩斯|名偵探福爾摩斯]]》（{{lang|ja|[[:ja:名探偵ホームズ|名探偵ホームズ]]}}）皆是以[[擬人化動物|擬人化動物]]或[[獸人|獸人]]為人物角色，不過前者主要是[[貓科|貓科]]動物（故用「飛虎」一詞），後者則為[[犬科|犬科]]動物。

* 以上兩套動畫，{{lang|ja|マルコ・パゴット}}和{{lang|ja|ジー・パゴット}}皆有參與，並也是 [[REVER|REVER]] 與日本其他製作部的「共同製作」。

* 這套動畫的靈感可能是來自一[[美國|美國]]著名的冒險考古家人物角色[[印第安那瓊斯|印第安那瓊斯]] ([[:en:Indiana_Jones|Indiana Jones]]) 。
