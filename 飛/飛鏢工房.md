香港[[原創|原創]][[品牌|品牌]]飛鏢工房於2015年7月正式成立，始創人徐詠琳於2016年獲得香港青年工業家獎<ref>[http://www.yiah.org/?_page=yiah&_func=main&_lang=zh-hant&_para 《香港青年工業家獎2016》]</ref>，秉持時尚感、生活化、智能化及高品質的理念，以「將飛鏢帶入屋」為概念，透過自家應用程式(Guz World)連接全世界玩家，是首個品牌推出輕便式支援網絡對戰，型號分別有GUZ，GUZ2，GUZ Sigma 和 GUZ Devil。

== 電子飛鏢靶-GUZ ==

GUZ<ref>[https://unwire.hk/2015/07/28/guz-board/wireless-home/ 《唔使落Pub！依莉詩試玩 G Board智能飛鏢靶》]</ref>是<<飛鏢工房>>品牌，透過自家應用程式(Guz World)連接全世界玩家，是首個品牌推出輕便式支援網絡對戰，型號分別有GUZ，GUZ2，Guz Sigma和Guz Devil。


'''GUZ'''/'''GUZ2'''

申請專利 - 方形電子飛鏢靶。擁有可換式面蓋和可利用應用程式(Guz World)控制飛鏢靶上LED燈光，飛鏢靶設有LED自動提示功能。代2016年獲得由香港電子業商會頒發之電子遊戲類別金獎。


'''Sigma'''

於2016年推出，與另外2款GUZ一樣，皆支對戰。外形與傳靶無異，與GUZ2相比沒有LED和可換式面蓋，為系列中之低檔機。


'''Guz devil'''

於2年年尾推出，與GUZ Sigma同款不同色，<<飛鏢工房>>限量生產了66套，推出一個月隨即售罄，於2017年

== 應用程式-Guz World ==
GUZ電子鏢靶透過應用程式Guz World連線進行網絡對戰，Guz World前身是Guz Board(於2016年12月31日正式下架)。新增2款原創遊戲(Battleship和Guzy Party)，支援外置鏡頭和最多8位玩家同時對戰。

== 飛鏢-Mu(μ) ==

Mu(μ)的宗旨是精準，飛鏢重量偏差值控制在±0.05g ，也會替專業飛鏢選手設計和推出專屬飛鏢。


'''Mu(μ)5'''/'''Mu(μ)5 Diamond'''

直筒形，首家研發鏢內鏍紋技術(已申請專利)，可用90%鎢任意調較飛鏢重量，用家可根據個人需要或習慣調較飛鏢重量，其幻彩鏢身每支獨立製造，每支都是獨一無二。
