'''飛鳥鈴'''（{{lang-ja|飛鳥りん}}，1997年1月8日 - ），[[日本|日本]]的[[AV女優|AV女優]]，[[京都府|京都府]]出身<ref name="SOD"/>。曾是「[[Soft_On_Demand|SOD]]」專屬女優<ref name="SOD"/>。所屬於「{{link-ja|Eightman Production|モデルプロダクションエイトマン}}」事務所<ref name="エイトマン">{{Cite web |url=http://www.8man.jp/models/asuka/ |title=所属タレント 飛鳥 りん |publisher=エイトマン |accessdate=2017-03-12}}</ref>。

== 人物 ==
* 她喜歡[[偶像_(日本)|偶像]]，興趣是[[料理|料理]]、看[[動漫|動漫]]、[[室內設計|室內設計]]平面圖鑑賞和[[地圖|地圖]]鑑賞<ref name="SOD"/>。

* 原本她很喜歡偶像也嚮往很久了，而且在出道之前已經通過了某個偶像團體的試鏡，但是後來又被她拒絕了<ref name="サイゾー">{{Cite web |url=http://www.cyzo.com/2016/07/post_28959_entry.html |title=美少女アイドルが脱いだ！　飛鳥りん、AVデビュー作は5時間の大長編！|publisher={{link-ja|日刊サイゾー|サイゾー}} |date=2016-07-26 |accessdate=2017-03-12}}</ref>。

* 最嚮往的女優是[[紗倉真菜|紗倉真菜]]，兩人有許多次見面的機會，也一起錄廣播節目，覺得對方真的很可愛<ref name="サイゾー"/>。

* 學生時期曾在[[女僕咖啡廳|女僕咖啡廳]][[兼職|打工]]<ref name="サイゾー"/>。

* 在出道之前會研究紗倉真菜的AV。覺得拍AV最困難的場景是精油按摩<ref name="サイゾー"/>。

== 簡歷 ==
2016年3月，她申請了「[[Soft_On_Demand|SOD]]」的[[偶像_(日本)|偶像]]培育計畫，以專屬偶像的身分活動<ref>{{cite web|url=http://avno1.playno1.com/article-19510-1.html#.WwhJpEiFOUk |title=SOD的救世主? - 飛鳥りん |publisher=PLAYNO.1玩樂達人 |date=2016-04-02}}</ref>。

2016年9月1日，在日本[[AV片商|AV片商]]「[[Soft_On_Demand|SOD Create]]」以專屬女優身分[[AV|AV]][[出道|出道]]<ref>{{cite web|url=http://www.playno1.com/article-20658-1.html#.WwhlzUiFOUk |title=AV界也有穿越劇！飛鳥りん終於被插進去了！ |publisher=PLAYNO.1玩樂達人 |date=2016-07-25}}</ref>。

2018年10月18日，在個人[[Twitter|Twitter]]上宣布於SOD Create畢業，並於11月發行畢業作品<ref>[https://twitter.com/asukarin0108/status/1052901538849411072 @asukarin0108 下午8:37 - 2018年10月18日]</ref>。
