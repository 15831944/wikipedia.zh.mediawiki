《'''飛行器的執行週期'''》（{{lang-en|''The Silent Star Stone''}}）是中國歌手[[郭頂|郭頂]]的專輯，由[[環球音樂|環球音樂]]於2016年11月29日發行。郭頂在2009年《微微》發行後陷入迷惘，他認為自己沒有能力將專輯製作的更好，因而選擇退居幕後學習。受到科幻電影的影響，他以想像力、探索及時間軸為新專輯概念，專輯製作約歷時兩年。歌曲在作曲上較為即興，歌詞則較不重口號性及邏輯性，藉此讓歌曲更有想像力。

專輯發行後，樂評在風格及題材元素給予普遍好評，但對於郭頂的歌曲[[唱腔|詮釋方式]]則褒貶不一。專輯亦在[[第28屆金曲獎|第28屆金曲獎]]獲6項提名。

== 背景與概念 ==
郭頂在2009年前作《微微》發行後陷入迷惘，他認為自己沒有能力將專輯製作的更好，因而選擇退居幕後「找自我」以及學習詞曲製作，七年間除了寫自己的作品，也為[[周筆暢|周筆暢]]、[[薛之謙|薛之謙]]等歌手製作歌曲，這段幕後經歷讓他體悟音樂創作必須「跟隨自己內心才能快樂」，最終完成這張「放飛了自我」的專輯<ref name=bjnews>{{cite web |author1=倪雪莹 |author2=杨畅 |title=郭顶：这可能是我最后一张“好听”的专辑 - 文化 - 新京报网 |url=http://www.bjnews.com.cn/culture/2017/06/26/448083.html |publisher=《[[新京報|新京報]]》 |accessdate=2018-10-16 |date=2017-06-26 |archive-url=https://web.archive.org/web/20181016125524/http://www.bjnews.com.cn/culture/2017/06/26/448083.html |archive-date=2018-10-16 |dead-url=no }}</ref><ref name=kkbox>{{cite web |author1=dato |title=音樂融入他的靈魂裡，他挑出來寫成歌——專訪郭頂 |url=https://www.kkbox.com/tw/tc/column/interviews-0-956-1.html |publisher=[[KKBOX|KKBOX]] |accessdate=2018-10-17 |date=2017-06-23}}</ref>。專輯製作過程中，郭頂廣泛接觸了具想像力的作品，包括科幻電影《[[星際效應|星際效應]]》，以及講述未來人類與人工智能相戀故事的《[[雲端情人|-{zh:雲端情人;zh-hans:云端情人;zh-hant:雲端情人;zh-cn:她;zh-hk:觸不到的她;zh-mo:觸不到的她;zh-sg:云端情人;zh-tw:雲端情人;}-]]》，他表示電影這種綜合視覺、音樂、技術及人類情感的創作形式帶給他啟發，讓他想製作一張近似於此的專輯。{{r|bjnews|weibo}}

相較先前專輯多描寫日常，新專輯以想像力、探索及時間軸為概念{{r|kkbox}}<ref name=weibo>{{cite web |author1=林怡妘 |title=郭顶金曲奖最想拿的奖没入围 希望观众还相信音乐 |url=http://ent.sina.com.cn/y/yneidi/2017-06-20/doc-ifyhfnqa4493331.shtml |work=[[新浪|新浪]] |publisher=新浪新聞 |accessdate=2018-10-16 |date=2017-06-20 |archive-url=https://web.archive.org/web/20181016131847/http://ent.sina.com.cn/y/yneidi/2017-06-20/doc-ifyhfnqa4493331.shtml |archive-date=2018-10-16 |dead-url=no }}</ref>，自開頭《淒美地》到最後一首《下一次在進站》完成完整「週期」故事，描寫主題包含社會關懷及人類極端情感等多方嘗試，郭頂希望「每個人聽到歌曲後會投射自己心裡的想像」{{r|weibo}}。專輯名稱「飛行器的執行週期」概念則來自科幻小說家[[特德·姜|特德·姜]]《{{tsl|en|The Lifecycle of Software Objects|軟體物件的生命週期}}》書名。英文名「The Silent Star Stone」意為「寂靜星石」，是收錄曲《保留》的其中一句歌詞；專輯封套特別將英文名稱中的「H、E、R、E」4字使用不同顏色印刷點出，意指「有個『在這裡』的人，在把訊息傳給每個可能聽到他的人」。{{r|bjnews}}

== 錄製與發行 ==
{{Quote box|quoted=true|border=1|salign=right|align=left|width=215px|quote=这张专辑本身也在讲人工智能，人工智能和人的最大差别是，人有时候无法控制自己的情绪，不会那么快变得平稳和稳定。所以我们在做的时候就有意去抓住人在情绪冷的时候和情绪热的时候那种不可控制的感觉，就是人性化的感觉。|source=－ 郭頂談及專輯製作{{r|20170301sohu}}}}
專輯概念成形後，郭頂放棄原先寫好的風格迥異歌曲，轉而「探索他的宇宙」，他深受[[披頭四樂團|披頭四樂團]]和{{tsl|en|Blue Cheer|藍調喝採}}的作品影響，在這張專輯中嘗試1960、1970年代的音樂風格{{r|bjnews}}。郭頂稱這是張「非正確」的專輯，作曲較為即興，且不特別使用具口號性的歌詞，詞與詞之間的邏輯也較為跳躍，藉此讓歌曲更有想像；歌曲錄音方面，他使用收集來的舊設備進行現場同步收音，保留產生的粗糙音效，同時減少後製機械修正，讓歌曲以較為粗糙而原始的方式呈現{{r|20170301sohu}}。為配合專輯整體風格郭頂調整了[[唱腔|唱腔]]，改以「冷峻、沙啞的方式」詮釋歌曲。{{r|bjnews|20170301sohu}}

==歌曲==
在《淒美地》創作時，霧霾嚴重、呼吸困難以致郭頂對於生存「產生了一些質疑」，藉以歌曲表達儘管有目標，然而前途路漫漫，或許永無抵達可能的「悲壯情緒」{{r|bjnews}}。歌曲設定為專輯中第一首歌曲，因郭頂認為它適合「作為一個故事的開篇」；歌詞風格介於詩歌與白話之間，他稱是「白話文詩歌」<ref>{{cite web |author1=大发 |title=郭顶《飞行器的执行周期》：七年沉淀积累 |url=http://ent.sina.com.cn/y/yneidi/2016-12-30/doc-ifxzczff3443590.shtml |publisher=新浪 |accessdate=2018-10-20 |date=2016-12-30 |archive-url=https://web.archive.org/web/20181020021101/http://ent.sina.com.cn/y/yneidi/2016-12-30/doc-ifxzczff3443590.shtml |archive-date=2018-10-20 |dead-url=no }}</ref>。[[立場新聞|立場新聞]]編輯認為編曲上「突出了鼓的力道，像推你（聽者）不斷地前進」<ref name=standnews>{{cite web |author1=田中小百合 |title=郭頂去得更遠的飛行計劃 |url=https://thestandnews.com/culture/郭頂去得更遠的飛行計劃/ |publisher=[[立場新聞|立場新聞]] |accessdate=2018-10-16 |date=2017-06-01 |archive-url=https://web.archive.org/web/20181016125316/https://thestandnews.com/culture/%E9%83%AD%E9%A0%82%E5%8E%BB%E5%BE%97%E6%9B%B4%E9%81%A0%E7%9A%84%E9%A3%9B%E8%A1%8C%E8%A8%88%E5%8A%83/ |archive-date=2018-10-16 |dead-url=no }}</ref>。第三首歌曲《把你的外套留在深巷》是描寫如何與自我的孤獨相處，郭頂稱這首「些許偏執」的歌曲展現他「從音樂出發點」的理念，是他在專輯中最喜歡的歌曲。<ref>{{cite news |author1=麻乐 |title=郭顶：“做音乐的过程和人的成长是有点类似的”-南方都市报·奥一网 |url=http://epaper.oeeee.com/epaper/C/html/2017-06/15/content_37163.htm#article |work=《[[南方都市報|南方都市報]]》 |accessdate=2018-10-18 |date=2017-06-15 |archive-url=https://web.archive.org/web/20181018152707/http://epaper.oeeee.com/epaper/C/html/2017-06/15/content_37163.htm#article |archive-date=2018-10-18 |dead-url=no }}</ref><ref>{{cite web |author1=謝濬如 |title=音樂是種尋找同類的過程、讓老派混搭摩登：專訪郭頂 |url=https://ysolife.com/guo-ding-interview/ |publisher=樂手巢 |accessdate=2018-10-16 |date=2017-05-22 |archive-url=https://web.archive.org/web/20181016130724/https://ysolife.com/guo-ding-interview/ |archive-date=2018-10-16 |dead-url=no }}</ref>

第四首《想著你》歌詞表達即使會分離，仍然相信會再次相遇的思念之情<ref name="GQ">{{cite web |author1=Eric Hung |title=一位不能忽視的音樂創作才子─郭頂《飛行器的執行週期》 |url=https://www.gq.com.tw/blog/hitodjerichung/detail-3680.html |website=GQ瀟灑男人網 |publisher=《[[GQ|GQ]]》 |accessdate=2018-10-19 |date=2017-01-24 |archive-url=https://web.archive.org/web/20170712010838/http://www.gq.com.tw/blog/hitodjerichung/detail-3680.html |archive-date=2017-07-12 |dead-url=no }}</ref>。第五首《水星記》以八大行星中距離[[太陽|太陽]]最近的[[水星|水星]][[軌道_(力學)|遵循固有軌跡]]，比喻人與人之間忽近忽遠的關係<ref name=cecn>{{cite web |author1=傅云鹏 |title=郭顶全新音乐大碟《飞行器的执行周期》发行在即 |url=http://ent.ce.cn/news/201611/15/t20161115_17818237.shtml |website=中國經濟網 |publisher=《[[經濟日報_(中華人民共和國)|經濟日報]]》 |accessdate=2018-10-20 |archiveurl=https://web.archive.org/web/20170926142712/http://ent.ce.cn/news/201611/15/t20161115_17818237.shtml |archivedate=2017-09-26 |date=2016-11-15}}</ref>，郭頂「猜測水星是不是特別迷戀太陽，不然幹嘛靠它那麼緊，用那種方式折磨自己」，藉以表達「有時心的距離特別可怕」<ref>{{cite web |author1=唐千雅 |title=金曲失6金吸人氣　郭頂險些打敗五月天 |url=https://www.mirrormedia.mg/story/20170714ent018/ |publisher=《[[鏡週刊|鏡週刊]]》 |accessdate=2018-10-16 |date=2017-07-21 |archive-url=https://web.archive.org/web/20181016125033/https://www.mirrormedia.mg/story/20170714ent018/ |archive-date=2018-10-16 |dead-url=no }}</ref>，他稱這首歌曲並不是情歌，而是當自己向著某人或某地靠近時，「面對自己特別渺小（......）的私密情緒」；歌曲在製作時，使用了一種具「宇宙感」的合成器。{{r|20170301sohu}}

第六首《在雲端》是首{{tsl|en|Motown#Motown Sound|摩城音樂|摩城式}}[[靈魂樂|靈魂樂]]混合[[藍調|藍調]]風格歌曲<ref name=sinchew>{{cite news |author1=月貝凡 |title=那一場風花雪月的白話詩歌：郭頂《飛行器的執行周期》 |work=《[[星洲日報|星洲日報]]》 |date=2017-04-16 |location=馬來西亞}}</ref>。第八首歌曲《每個眼神都隻身荒野》在風格上以[[節奏藍調|節奏藍調]]為基底，結合復古[[英倫搖滾|英倫]][[輕搖滾|輕搖滾]]{{r|sinchew}}。第九首歌曲《有什麼奇怪》則傳達每個人都是擁有自己運行軌道的個體，不須在意旁人眼光的概念<ref name=sohu>{{cite web |title=郭顶出镜微电影大结局 用好音乐见证幸福时刻 |url=http://www.sohu.com/a/154266976_115487 |publisher=[[搜狐|搜狐]] |accessdate=2018-10-16 |date=2017-07-04 |archive-url=https://web.archive.org/web/20181018015301/http://www.sohu.com/a/154266976_115487 |archive-date=2018-10-18 |dead-url=no }}</ref>，風格融入了[[放克|放克]]與[[迪斯科|迪斯科]]{{r|GQ}}。尾曲《下次再進站》描寫前往未知時內心的感受<ref>{{cite interview |interviewer=[[馬世芳|馬世芳]] |title=《音樂五四三》 |publisher=[[台灣全民廣播電台|台灣全民廣播電台]] |date=2017-04-18  |accessdate=2018-10-19}}</ref>，是首節奏藍調風格歌曲。{{r|sinchew}}

==音樂錄影帶==
《水星記》[[音樂錄影帶|音樂錄影帶]]由黃婕妤執導，劇情描述在未來科技中，一對母女與人工智能生活的故事<ref name=sohumv>{{cite web |title=郭顶概念性MV获肯定 《做个梦给你》温情上线 |url=http://music.yule.sohu.com/20170411/n487675497.shtml |publisher=搜狐 |accessdate=2018-10-18 |date=2017-04-11 |archive-url=https://web.archive.org/web/20181018100552/http://music.yule.sohu.com/20170411/n487675497.shtml |archive-date=2018-10-18 |dead-url=no }}</ref>。郭頂最初的畫面構想是縮小了自己，在「微觀」的視角中觀察這個世界，不過為了與黃婕妤完成共同創作，他最終並未單方面提供概念，而是依黃婕妤對歌曲的解讀來創作{{r|kkbox}}。延續《水星記》故事的音樂影片《做個夢給你》則於2017年4月11日公開。{{r|sohumv}}

《把你的外套留在深巷》由陳霈芙執導，該片入選2017年柏林時尚影展（Berlin Fashion Film Festival）{{r|sohumv}}。劇情影片《在這 淒美地》由吳仲倫執導，講述盲眼少年在夢境中，找尋心中目標的故事。<ref>{{cite web |title=郭顶《在这凄美地》MV首发 全金榜献唱《水星记》 |url=http://ent.sina.com.cn/y/ygangtai/2017-05-08/doc-ifyeychk7138466.shtml |publisher=[[新浪|新浪]] |accessdate=2018-10-17 |date=2017-05-08 |archive-url=https://web.archive.org/web/20181018014130/http://ent.sina.com.cn/y/ygangtai/2017-05-08/doc-ifyeychk7138466.shtml |archive-date=2018-10-18 |dead-url=no }}</ref><ref>{{cite web |title=郭顶携导演吴仲伦深度解读《凄美地》故事版《在这凄美地》今日首播 |url=http://ent.cctv.com/2017/05/08/ARTItc2q3W27QdYgaCZclnzH170508.shtml |publisher=[[央視網|央視網]] |accessdate=2018-10-18 |date=2017-05-08 |archive-url=https://web.archive.org/web/20181018150659/http://ent.cctv.com/2017/05/08/ARTItc2q3W27QdYgaCZclnzH170508.shtml |archive-date=2018-10-18 |dead-url=no }}</ref>

== 反響 ==
===媒體評論===
《[[星洲日報|星洲日報]]》編輯月貝凡認為專輯「深受西洋音樂文化的熏陶和影響，卻仍不失其濃烈的個人特色」{{r|sinchew}}。《[[GQ|GQ]]》編輯Eric Hung形容專輯「如同封面色調，明顯感受到一股冰冷的金屬氛圍，但從郭頂的歌聲又能感受明顯溫度」{{r|GQ}}。[[中時電子報|中時電子報]]則認為專輯風格包含[[搖滾|搖滾]]、[[謠曲|謠曲]]、[[節奏藍調|節奏藍調]]與[[放克|放克]]，專輯整體的「概念與執行力非常完整」。<ref>{{cite web |author1=尤嬿妮 |author2=楊宗翰 |author3=黃保慧 |title=金曲歌王預測 林宥嘉先聲奪人 |url=https://www.chinatimes.com/newspapers/20170613000667-260112 |website=[[中時電子報|中時電子報]] |accessdate=2018-10-18 |date=2017-06-13 |archive-url=https://web.archive.org/web/20181018121620/https://www.chinatimes.com/newspapers/20170613000667-260112 |archive-date=2018-10-18 |dead-url=no }}</ref>

[[關鍵評論網|關鍵評論網]]的DJ Ken Lin認為郭頂的詮釋「不特別炫技，保有最初的真誠感」，他形容這是張揉合[[藍調|藍調]]、[[鄉村音樂|鄉村音樂]]以及太空飄渺元素「極為浪漫的一張專輯」<ref>{{cite web |author1=DJ Ken Lin大叔 |title=大叔DJ看28屆金曲獎：各項入圍名單中誰最該得獎呢？ |url=https://www.thenewslens.com/article/71607 |publisher=[[關鍵評論網|關鍵評論網]] |date=2017-06-23 |access-date=2018-10-17 |archive-url=https://web.archive.org/web/20181016133014/https://www.thenewslens.com/article/71607 |archive-date=2018-10-16 |dead-url=no }}</ref>。[[立場新聞|立場新聞]]編輯田中小百合則有不同觀點，他認為專輯以科幻概念為題，在故事與執行上「仍差一道火候」，詮釋方式上「較為地造作」，但他同時指出在現今以分工制為主的流行樂壇中，郭頂獨自主導整張專輯製作擁有「能夠被平衡、調和得好」的優勢，更稱「對編曲的重視」令專輯「馬上被提升」。{{r|standnews}}

[[搜狐|搜狐]]音樂的朱爾摩斯讚揚專輯「不再執著於表達男女之愛」的與眾不同<ref>{{cite web |author1=朱尔摩斯 |title=一个太少，我们还能奢求更多个郭顶吗？ |url=http://music.yule.sohu.com/20170526/n494546758.shtml |publisher=[[搜狐|搜狐]] |accessdate=2018-10-19 |date=2017-05-26 |archive-url=https://web.archive.org/web/20181019074725/http://music.yule.sohu.com/20170526/n494546758.shtml |archive-date=2018-10-19 |dead-url=no }}</ref>。《[[南方都市報|南方都市報]]》將專輯選為2016年十大華語專輯第7名，編輯小樹讚賞專輯「詩意與聲響氣氛緊貼」。<ref>{{cite news |title=十大华语专辑 |url=http://epaper.oeeee.com/epaper/C/html/2017-01/05/content_759.htm |work=《[[南方都市報|南方都市報]]》 |accessdate=2018-10-18 |language=en |date=2017-01-05 |archive-url=https://web.archive.org/web/20181018151632/http://epaper.oeeee.com/epaper/C/html/2017-01/05/content_759.htm |archive-date=2018-10-18 |dead-url=no }}</ref>
