'''孫鄰'''（{{bd|202年||249年|catIdx=S}}），[[表字|字]]'''公達'''，[[吳郡|吳郡]][[富春|富春]]（今[[浙江|浙江]][[富陽|富陽]]）人。東吳宗室，[[孫權|孫權]]堂兄[[孫贲|孫贲]]子。

== 生平 ==
孫鄰個性精細敏捷，年小時就有好名聲。九歲時父親去世，代領父職[[豫章|豫章]][[太守|太守]]，並承襲其爵位，進封'''都鄉侯'''，因为年幼的缘故不久官职为[[顧邵|顧邵]]代行。孫鄰自此在豫章近二十年，期間曾經討平叛賊，治郡亦有功績，於是獲召到[[武昌|武昌]]（今[[湖北|湖北]][[鄂州市|鄂州市]]）任繞帳督。

其時[[太常|太常]][[潘濬|潘濬]]亦在武昌管理[[荊州_(古代)|荊州]]事務，而重安縣縣長[[舒燮|舒燮]]因犯罪下獄，潘濬因與其有曾過節，於是打算按法處死他，當時已經有人為他說情，但潘濬並未有意赦免舒燮，孫鄰勸道：「昔日[[舒邵|舒仲膺]]與[[舒伯膺|舒伯膺]]因報親友被殺之仇而爭著認罪，天下人都稱許他們的義行，成了一時美談；而且舒仲膺那時還有歸效我國的意願。今天你殺了他的子弟，假若天下統一了，皇帝北巡時就必有中原人士問舒仲膺後代的下落，得來的回答時卻是潘承明殺了舒燮，你說該如何是好？」潘濬聽後立即打消懲處舒燮的念頭，舒燮也逃過一劫。

孫鄰後遷夏口沔中督，威遠將軍。[[赤烏|赤烏]]十二年（249年），孫鄰去世。

== 評價 ==
韋曜《吳書》：雅性精敏，幼有令譽。

陳壽《三國志》：夫親親恩義，古今之常。宗子維城，詩人所稱。況此諸孫，或贊興初基，或鎮據邊陲，克堪厥任，不忝其榮者乎！故詳著雲。

== 子女 ==
*[[孙震_(三国)|孫震]]，孫鄰子，無難督。[[晉滅吳之戰|晉滅吳之戰]]時任護軍<ref>干寶《晉紀》作護軍，《晉書·王渾傳》作大將軍，今從《晉紀》</ref>，為[[周浚|周浚]]等諸晉將擊敗，與[[張悌|張悌]]一同戰死。<ref>《晉書·王渾傳》：「吳丞相張悌、大將軍孫震等率眾數萬指城陽，渾遣司馬孫疇、揚州刺史周浚擊破之，臨陣斬二將，及首虜七千八百級，吳人大震。」</ref>
*[[孫苗|孫苗]]，孫鄰子，夏口沔中督，父親逝後襲其爵位。
*[[孫旅|孫旅]]，孫鄰子。
*[[孫述|孫述]]，孫鄰子，曾為武昌督，平荊州事。
*[[孫諧|孫諧]]，孫鄰子，城門校尉。
*[[孫歆|孫歆]]，孫鄰子，樂鄉督。晉滅吳之戰時於樂鄉為晉將[[周旨|周旨]]所俘。<ref>《資治通鑑·卷八十一》：「（周）旨等發伏兵隨歆軍而入，歆不覺，直至帳下，虜歆而還。」</ref>
