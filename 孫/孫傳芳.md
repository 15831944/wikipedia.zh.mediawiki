'''孫傳芳'''{{BD|1885年|4月17日|1935年|11月13日|catIdx=S孙}}，字'''馨远'''，[[山東省|山東省]][[泰安县|泰安县]][[范镇|范镇]]人，[[中华民國|中华民國]]军事将领，[[直系军阀|直系]]領導人之一。<ref name=zhang>張振鶴「孫传芳」{{Cite book<!--|和書-->|author = [[中国社会科学院|中国社会科学院]]近代史研究所|title = 民国人物传 第1卷|year = 1978|publisher = [[中華書局|中華書局]]|isbn = }}</ref><ref name=ren>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>孫传芳常常面带笑容，但面善而心狠，故人称“笑面虎将軍”。<ref name=zhang/><ref name=ren/>

==生平==
=== 在湖北的崛起 ===
1904年（[[光緒|光緒]]30年），孫傳芳毕业于[[保定陸軍軍官学校|北洋陸軍速成学堂]]，获官費赴[[日本|日本]]学习軍事。1908年（光緒34年），自[[陸軍士官学校_(日本)|陸軍士官学校]]第6期毕业。归国後的1909年（[[宣統|宣統]]元年），接受[[陸軍部_(清朝)|陸軍部]]考试，成绩优秀。此后，孫傳芳加入[[王占元|王占元]]所率的北洋陸軍近畿第2鎮，任步兵队第5標教官。<ref name=zhang/><ref name=ren/>

[[民初|民初]]時代，第2鎮改编为第2師，驻扎[[湖北省_(中華民国)|湖北省]]。孫傳芳逐渐升至第3旅旅長。1917年（[[民国紀元|民国]]6年），被任命为第21混成旅旅長。1920年（民国9年）7月的[[直皖战争|直皖战争]]中，孫傳芳随直系的王占元俘虏了[[皖系|皖系]]的[[長江上游总司令|長江上游总司令]][[吴光新|吴光新]]，孫傳芳随后接任[[長江上游总司令|長江上游总司令]]。1921年（民国10年）8月，王占元在湖北省失势。孫傳芳接任第2師師長，成为[[两湖巡阅使|两湖巡阅使]][[吴佩孚|吴佩孚]]的部属。<ref name=zhang/><ref name=ren/><ref name=liu>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = 中華書局|isbn = 7-101-01320-1}}</ref>

=== 直系强人 ===
[[File:Sun_Chuanfang.jpg|thumb]]
1922年（民国11年）4月的[[第一次直奉战争|第一次直奉战争]]後，孫传芳成为直系成員，公开要求皖系擁立的大总統[[徐世昌|徐世昌]]辞职。結果，6月徐世昌辞职，[[黎元洪|黎元洪]]再度出任大总統。1923年（民国12年）1月，孫传芳率軍赴[[福建省_(中華民国)|福建省]]，被任命为福建軍務督理（简称“福建督理”）。<ref name=zhang/><ref name=ren/><ref name=liu/>

1924年（民国13年）9月，[[第二次直奉战争|第二次直奉战争]]的前哨战[[江浙战争|江浙战争]]打响。孫传芳支援直系的[[江蘇省_(中華民国)|江蘇]]督軍[[齐燮元|齐燮元]]，击败皖系的[[浙江省_(中華民国)|浙江]]督軍[[盧永祥|盧永祥]]。此后，孫传芳被任命为[[閩浙巡閱使|閩浙巡閱使]]兼浙江軍務督理。<ref name=zhang/><ref name=ren/>
[[File:Sun_Chuanfang2.jpg|thumb]]
自此，江蘇、浙江为孫传芳、齐燮元、盧永祥三人勢力鼎立。第二次直奉战争中，直系敗北，[[段祺瑞|段祺瑞]]建立[[中华民国临时政府_(段祺瑞)|中华民国临时政府]]，[[奉系|奉系]]的[[張宗昌|張宗昌]]为支援盧永祥而南下。为此，孫传芳于1925年（民国14年）1月和齐燮元结成同盟，组织「江浙聯軍」迎撃。段祺瑞为笼络孫传芳而任命其为浙江督办。孫传芳遂抛弃齐燮元，齐燮元陷于孤立而发表下野宣言。<ref name=zhang/><ref name=ren/><ref name=liu/>

同时，为了扩大在南方的勢力，奉系对孫传芳依然保持警戒。孫传芳乃秘密同[[馮玉祥|馮玉祥]]领导的[[国民軍|国民軍]]结成同盟。1925年10月，奉系的[[山東省_(中華民国)|山東]]督軍張宗昌、江蘇督軍[[楊宇霆|楊宇霆]]开始进攻孫传芳（[[浙魯战争|浙魯战争]]、[[奉浙战争|奉浙战争]]）。孫传芳取得胜利。同年11月，孫传芳在[[南京市|南京]]宣布就任浙閩蘇皖贛五省聯軍总司令。当時，应孫传芳之邀，[[章太炎|章太炎]]任（政治）顧問、[[蒋方震|蒋方震]]（蒋百里）及[[冈村寧次|冈村寧次]]任高等軍事顧問。<ref name=zhang/><ref name=ren/>

=== 在北伐中敗北 ===

1926年（民国15年）7月，[[中国国民党|中国国民党]]开始[[国民革命军北伐|国民革命军北伐]]，[[吴佩孚|吴佩孚]]遭到攻撃。孫传芳为对抗[[孫文|孫文]]的[[三民主義|三民主義]]而提出「愛国家、愛人民、愛敵人」的「'''三愛主義'''」。9月，击败吴佩孚的北伐軍进攻[[江西省|江西省]]，孫传芳赴江西省迎撃，但遭到敗北，撤至南京。此後，孫传芳改善了同奉系的关系，11月和張宗昌共同拥立[[張作霖|張作霖]]任[[安国軍|安国軍]]总司令，孫传芳任副司令兼五省聯軍总司令。<ref name=zhang/><ref name=ren/><ref name=liu/>

[[File:Sun_Chuanfang6.jpg|thumb]]

但是，这没有挽回其面对[[国民革命軍|国民革命軍]]时所处的劣勢。1927年（民国16年）3月，孫传芳的南京失守。4月12日，[[四一二事件|四一二事件]]，孫传芳趁机反击但遭到失败。1928年（民国17年）4月，孫传芳軍在山東省被歼灭，孫传芳逃往北方。此后，孫传芳的軍事及政治势力完全喪失。<ref name=zhang/><ref name=ren/>

敗北後，孫传芳一度投[[張学良|張学良]]麾下，但因感到人身危险而迁居[[大連市|大連]]。1931年（民国20年）[[九一八事变|九一八事变]]爆发后，孫传芳迁居[[天津英租界|天津英租界]]，整日诵念[[佛经|佛经]]。1935年（民国24年）11月13日，孫传芳正在[[天津佛教居士林|天津佛教居士林]]誦佛经，[[施剑翘|施剑翘]]{{NoteTag|她是魯軍司令官[[施從濱|施從濱]]的女儿。她父親在浙魯战争之際，被孫传芳俘虏了，并将其枪殺，暴尸三日三夜。<ref name=zhang/><ref name=ren/>}}从背後枪击孫传芳，孫传芳死亡。得年51（满50歳）。葬于[[北京|北京]][[西山_(北京)|西山]][[十方普觉寺|卧佛寺]]东南，即[[孙传芳墓|孙传芳墓]]，现位于[[北京植物园|北京植物园]]内。<ref name=zhang/><ref name=ren/>

==家庭==
孫生有子女七人，其中子四人，分别为孙家裕、孙家震、孙家均、[[孫家勤|孫家勤]]；女三人。<ref>[[顾祖彭|顾祖彭]]，恪威上将军总浙闽苏皖赣五省军务孙君神道碑，1938年1月</ref>四子[[孫家勤|孫家勤]]为画家，乃[[張大千|張大千]]之[[弟子|弟子]]，曾任[[台灣師範大學|台灣師範大學]]美術系教授。
