== 家族历史 ==
祖父[[孙昌|孙昌]]曾跟随叔叔孙中山革命，1917年奉孙中山命乘船押送军饷赴[[广州|广州]][[黄埔|黄埔]]时殉职，时年34岁。父亲[[孙乾|孙乾]]曾在1927年代表国家参加了在上海举办的第七届[[远东运动会|远东运动会]]担当排球国手，后任[[中山县|中山县]]县长4年，政绩颇著。<ref>{{Cite press release |language = zh-cn |author = 网易新闻 |url = http://news.163.com/10/1012/00/6IOL7MP800014JB6.html |title = 孙中山曾侄孙口述家史 |date = 2010-10-11 }}</ref>
