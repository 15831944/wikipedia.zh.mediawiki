'''孫國棟'''（{{bd|1922年|10月9日|2013年|6月26日|catIdx=S孫}}），[[廣東|廣東]][[番禺|番禺]]人。著名[[中國|中國]]歷史學家，教育家、哲學家、前[[香港中文大學|香港中文大學]]歷史系與中文系主任<ref>{{Cite web|url=http://www1.hkej.com/dailynews/articlePrint/id/679550|title=悼孫國棟老師|newspaper=[[信報|信報]]|publisher=南馳志、雷競璇|date=2013-07-04|language=zh-hant}}</ref>及[[新亞書院|新亞書院]]文學院[[院長|院長]]、香港[[新亞中學|新亞中學]]前校董會成員（即於1973至1983年擔任校監一職）。孫氏專於研究唐宋史，為唐宋史大家。尤以唐宋社會[[門第|門第]]變遷、唐宋政制與官制的研究最為史學界推祟。另外，他是歷史學家[[錢穆|錢穆]]的弟子。<ref>{{Cite web|url=http://www.chinainperspective.com/ArtShow.aspx?AID=21771|title=深切悼念孫國棟教授|newspaper=中國民主教育基金會|publisher=中國民主教育基金會 理事會及評審委員會|date=2013-06-30|language=zh-hant}}</ref>而前新亞中學校長及現任校監屈啟秋（1973年香港中文大學歷史系）則是他的學生。

== 生平 ==
孫國棟年輕時在1944年參與[[抗日戰爭|抗日戰爭]]<ref>{{Cite web|url=http://cdef.org/?tabid=173|title=以歷史文化為鑑的孫國棟、何冰姿夫婦|newspaper=中國民主教育基金會|publisher=余健文|date=|language=zh-hant}}</ref>。在戰爭勝利後他退伍回[[南京|南京]]復學。1949年[[中國共產黨|中國共產黨]]掌政，他與夫人[[何冰姿|何冰姿]]流亡至[[香港|香港]]。其後便在香港私立的[[嶺東中學|嶺東中學]]工作。由於他曾投稿《人生雜誌》，為雜誌創辦人[[王道|王道]]賞識，於是獲邀擔任《人生雜誌》編輯。又任《中南日報》主筆。孫國棟因《人生雜誌》的關係而認識[[錢穆|錢穆]]、[[唐君毅|唐君毅]]、[[牟宗三|牟宗三]]、[[徐復觀|徐復觀]]等名家，亦因此而激發其以學術為終生志業之心。1955年，他於[[新亞研究所|新亞研究所]]師從[[錢穆|錢穆]]治國史，並成為[[新亞研究所|新亞研究所]]第一屆畢業生。1964年後他擔任[[新亞書院|新亞書院]]歷史系主任、董事、文學院院長，[[香港中文大學|香港中文大學]]歷史系主任兼新亞研究所所長。孫氏於新亞研究所畢業後，隨即到新亞書院講授歷史。1983年，他辭去香港中文大學系主任職務，到美國過退休生活。<ref>{{Cite web|url=http://www.chinainperspective.com/ArtShow.aspx?AID=21771|title=深切悼念孫國棟教授|newspaper=中國民主教育基金會|publisher=中國民主教育基金會 理事會及評審委員會|date=2013-06-30|language=zh-hant}}</ref>[[八九民運|八九民運]]期間，孫國棟與[[何冰姿|何冰姿]]積極參與海外的支緩活動。不久[[六四事件|六四事件]]發生，他撰文寫道：「他們（北京學子）以青春之年而赴死，固然可憫；然而生命的價值，原不問久暫，與其卑微猥瑣而苟且偷生，何如充實光輝、軒昂磊落而赴死！」以表現其對他們的敬重。又贊曰：「凜凜千古，烈烈英魂，大哉生命，取義成仁！」

自六四事件以來，孫國棟夫婦為[[中國民主教育基金會|中國民主教育基金會]]擔任評審委員，從不間斷地為[[中華人民共和國民主運動|中國民主進程]]發展而努力。由1989年起到1995年之前，他定期遠赴[[加利福尼亞州大學柏克萊分校|加利福尼亞州大學柏克萊分校]]為香港的學生作文化講座，傳播中國文化，定居於[[美國|美國]][[舊金山灣區|舊金山灣區]]。

2005年，孫國棟經歷親人屢遭殺害後，決定遷回香港，並獲新亞書院安置於其學生宿舍知行樓地下舍監室居住，直至終老（中大另一個獲同等禮待的資深前教員只有[[胡秀英|胡秀英]]教授，亦已於2012年去世）。

孫國棟以錢穆的弟子自居，以錢穆《[[國史大綱|國史大綱]]》為體編輯[[中華文化|中華文化]]教科書，非常重視歷史精神；孫國棟晚年與[[柏楊|柏楊]]論戰，反對柏楊的「[[醜陋的中國人|醜陋的中國人]]」之說，也批評《[[柏楊版資治通鑑|柏楊版資治通鑑]]》的膚淺<ref>{{cite news|url=http://www.chinatimes.com/realtimenews/20170731003779-260409|title=滅香與改教科書 學者大嘆禮崩樂壞|author=江飛宇|publisher=[[中時電子報|中時電子報]]|date=2017-07-31|accessdate=2017-08-12|language=zh-tw}}</ref>。

2013年6月26日，孫國棟於香港逝世，享年九十一歲。<ref>{{Cite web|url=http://hk.apple.nextmedia.com/news/art/20130628/18315486|title=歷史學家孫國棟逝世|newspaper=[[蘋果日報|蘋果日報]]|publisher=[[蘋果日報|蘋果日報]]|date=2013-06-28|language=zh-hant}}</ref>同年8月3日於[[香港殯儀館|香港殯儀館]]舉殯。

== 著作 ==
孫國棟筆下的著作數目多，例如《評柏楊》、《生命的足跡》、《我的抗日從軍行》、《永遠的校長·江平教授八十華誕慶賀文集》、《唐代中央重要文官遷轉途徑研究》、《唐宋史論叢》、《慕稼軒文存》等。

== 家庭 ==
1947年8月1日，孫國棟在[[廣州市|廣州市]]與廣東省三水縣人[[何冰姿|何冰姿]]舉行婚禮。二人育有二子二女。長子孫經武為一名化學系博士，曾居[[特拉華州|特拉華州]]（Delaware）。而長女孫經文畢業於[[香港中文大學|香港中文大學]]歷史系，長期從事歷史教學工作；也曾為中國歷史教育學會會員，為爭取中學保留中國歷史科及改革歷史課程而努力。次子孫緯武為美國[[華盛頓大學|華盛頓大學]]物理學系教授兼香港中文大學物理系榮譽教授。至於幼女孫允文是[[加州理工學院|加州理工學院]]博士，現居於[[南加州|南加州]]。

2003年4月，孫國棟的女兒孫經文及女婿吳家望在[[沙田|沙田]][[排頭村|排頭村]]寓所內遭他們僱用的[[菲律賓|菲律賓]]籍男傭工 Magno Cruz Manalili 謀殺。至翌年12月，犯人在[[菲律賓|菲律賓]]落網。<ref>{{Cite web|url=http://paper.wenweipo.com/2011/05/18/YO1105180011.htm|title=涉殺孫國棟女 男菲傭拒認罪|newspaper=[[文匯報|文匯報]]|publisher=[[文匯報|文匯報]]|date=2011-05-18|language=zh-hant}}</ref>2010年，兇手被引渡到香港受審。<ref>{{Cite web|url=http://the-sun.on.cc/cnt/news/20100612/00407_031.html|title=涉殺孫國棟女兒男菲傭願引渡返港|newspaper=[[太陽報|太陽報]]|publisher=[[太陽報|太陽報]]|date=2010-06-12|language=zh-hant}}</ref>
