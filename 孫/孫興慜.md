'''孫興慜'''（{{lang-ko|'''손흥민'''}}，{{bd|1992年|7月8日||}}），是[[韓國|韓國]][[足球員|足球員]]，出生於[[江原道_(韓國)|江原道]][[春川市|春川市]]，現時效力[[英超|英超]]球隊[[托特纳姆热刺足球俱乐部|熱刺]]，他也是[[韩国国家足球队|韩国国家足球队]]的主力球員。

孫興慜在效力[[德甲|德甲]]球隊時已得到很高評價，[[德國|德國]]名宿[[碧根鮑華|-{zh-cn:贝肯鲍尔;zh-tw:貝肯鮑爾;zh-hk:碧根鮑華;}-]]就很看好他的實力，「他不但能取得漂亮的入球，還能製造入球。他是很出色的球員，我真的很喜歡他的球風。」

== 踢球風格 ==
孫興慜是一個頗爲全面的進攻球員，左右脚皆能靈活運用，能適應前鋒、前腰和翼衛等多個位置，因而在賽場上經常擔任策動進攻的重要核心；孫興慜亦擁有極爲優越的奔跑爆發力，效力熱刺時經常一人帶球於邊路突進甩開前去防守的敵隊球員，造成極具威脅的反擊。

== 球會生涯 ==
=== 汉堡 ===
孫興慜沒有在韓國[[經典K聯賽|經典K聯賽]]開始自己的職業球員生涯，16歲加入德甲球队[[汉堡足球俱乐部|汉堡]]的青年隊。

孙兴慜在2010年夏季进入[[汉堡足球俱乐部|汉堡]]一线队，并在和[[車路士足球會|車路士]]的热身赛中打进絕殺球，帮助汉堡2比1絕殺对手，一战成名。

孙兴慜在热身赛的精彩表现使得[[弗朗茨·贝肯鲍尔|-{zh-hans:弗朗茨·贝肯鲍尔;zh-hk:法蘭斯·碧根鮑華;zh-tw:法蘭茲·貝肯鮑爾;}-]]称赞他是韩国的[[盖德·穆勒|-{zh-hans:盖德·穆勒; zh-tw:蓋德·穆勒; zh-hk:格得·梅拿;}-]]，并表现他将有可能成为下一个[[車範根|車範根]]<ref>http://www.skysports.com/story/0,19528,11881_6491780,00.html [[天空體育台|天空體育台]] 2010年11月6日. Retrieved 2010年11月9日.</ref>。

但孫興慜的德國之路並不是一帆風順，在加盟漢堡一年之後，他並沒有得到這支德國球隊的喜愛，他成為自由球員回流市場，而且在接下來的6個月裡，他都沒有加盟任何一支球隊。但即便是身為自由球員，他還是被韓國U17國家隊徵召，並且在U17世界盃上證明了自己，他絕對是一名能夠在歐洲球壇掀起波瀾的一個好球員。在U-17世界杯的5場比賽中，孫興慜射入了3球，幫助球隊一路殺到了八強。也正是因為他在U-17世界杯上的出色表現，使得漢堡重新簽回了這名亞洲球員。

但孙兴慜在之后热身赛遭受跖骨骨折，这也使他在德甲赛场的亮相推迟到2010年10月30日。这一天他在汉堡2比3负于[[科隆足球俱乐部|科隆]]的比赛中首次出场，并在第24分钟打进职业生涯的首个进球。这个进球也打破39年前由[[曼弗雷德·卡尔茨|曼弗雷德·卡尔茨]]创造的汉堡球员德甲进球最年轻纪录<ref>http://kr.news.yahoo.com/sports/uefa/view?aid=2010110600100559719</ref><ref>{{cite web |url=http://news.sportsseoul.com/read/soccer/891447.htm |title=存档副本 |accessdate=2010-11-06 |0=2010-11-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20101104050213/http://news.sportsseoul.com/read/soccer/891447.htm |archivedate=2010-11-04 }}  Retrieved 6 November 2010. </ref>、歐洲頂級聯賽最年輕的韓國球員、第一個在歐洲聯賽完成[[帽子戲法|帽子戲法]]的韓國球員。11月4日，汉堡俱乐部宣布和孙兴慜续约至2014年6月30日。

=== 勒沃库森 ===
2013年6月14日，[[勒沃库森足球俱乐部|勒沃库森]]确认将以创俱乐部转会纪录的1000万欧元身价签入孙兴慜，孙兴慜和勒沃库森签约5年<ref>http://sports.news.naver.com/sports/index.nhn?category=worldfootball&ctg=news&mod=read&office_id=436&article_id=0000001055</ref>。

孙兴慜接受德國媒體採訪時，他很滿意2014年的表現：「我很開心應該沒有辜負球隊的期望，但我也有段時間不在狀況內，不過這就是足球。而且這一季進球來到兩位數，這聽起來確實也比9好很多。」

在比賽中，他能夠利用自己的速度輕易突破對手的防守，然後想辦法幫助球隊取得入球。孫興慜另外一個突出屬性是他的門前射門能力，他在對慕遜加柏的比賽中用一腳遠射攻開了對手大門。而在今季之前，他的很多入球都來自於左路內切，但在今季這名韓國攻擊手幾乎只在左路活動。這名韓國國腳能夠在前場適應多個位置，在效力漢堡的時候，孫興慜有時候會出現在中鋒的位置上，但在利華古遜由於[[斯特凡·基斯林|-{zh-hans:基斯林;zh-hk:傑斯寧;zh-tw:基斯林;}-]]的存在，讓他並沒有什麼機會出現在這個位置之上。

=== 熱刺 ===
孫興慜在2015年8月28日以2200萬英鎊加盟[[熱刺|熱刺]]，穿起7號球衣，並成為史上最貴亞洲球員。

2017年11月5日，孫興慜攻入[[英格兰足球超级联赛|英超]]第20球，超越前輩[[朴智星|朴智星]]，成為英超入球最多亞洲球員。

2019年4月，18-19赛季欧冠四分之一决赛中，孙兴慜两回合累计打入3球，帮助热刺淘汰曼城，历史性晋级欧冠四强。次回合比赛结束后，孙兴慜的欧冠总进球数来到12球，超越乌兹别克斯坦名宿马克西姆-沙茨基赫（11球），成为欧冠历史上进球最多的亚洲球员。

== 国家队生涯 ==
由于在德甲赛场的出色表现，没有国家队经验的孙兴慜在2010年底被选入[[韩国国家足球队|韩国国家队]]参加[[2011年亚洲杯足球赛|2011年亚洲杯足球赛]]的名单<ref>[http://sports.sohu.com/20101225/n278502401.shtml 日韩公布亚洲杯名单 本田圭佑朴智星领衔最强阵]</ref>。12月30日，孙兴慜在和[[叙利亚国家足球队|叙利亚]]的友谊赛中第一次代表国家队出战。

[[2014年世界盃足球賽|2014年世界盃足球賽]]，南韓出戰[[阿爾及利亞国家足球队|阿爾及利亞]]，上半場即遭對手以3球血洗，換邊後至第50分鐘，孫興慜在前場接獲隊友[[奇誠庸|奇誠庸]]的長傳，停球後一個假晃轉身，盤球後即冷不防左腳抽射，球穿越阿爾及利亞門將的胯下破網，孫興慜這記入球展現高超的個人技藝，再次證明自己今年歐洲-{}-聯賽百大球星唯一亞洲球員的身手。最終，南韓還是無法將比數進一步拉近，以2-4不敵北非球隊阿爾及利亞。

2016年代表韓國奧運隊出戰[[2016年夏季奧林匹克運動會男子足球比賽|奧運男足決賽周]]，跟隨韓國U23代表隊出戰[[2016年夏季奧林匹克運動會|里約熱內盧奧運會]]，孫興慜又謂：「我恨不得和隊友並肩作戰，並嘗試盡快跟奧運隊會合，我要協助球隊連續兩屆奪得奧運獎牌。」孫興慜為奧運男足決賽周上場4次、攻入2球，在男子足球爭奪四強賽的比賽上，韓國奧運足球隊則以0比1負於洪都拉斯，8強止步。

2018年夏天，孙兴慜代表韓國隊出戰[[2018年世界盃足球賽|2018年世界盃足球賽]]決賽周，他分別在對戰[[墨西哥国家足球队|墨西哥]]與[[德国国家足球队|德國]]的比賽當中各射入一球。

[[2018年亞洲運動會男子足球比賽|2018年亞運男足]]決賽，韓國U23代表隊成員之一的孫興慜成功協助韓國U23於決賽擊敗日本U23代表隊，同時韓國U23代表隊選手免除兵役<ref>[https://www.hk01.com/即時體育/230241/亞運足球-韓國加時勝日本衛冕男足金牌-孫興慜免除兵役 【亞運足球】韓國加時勝日本衛冕男足金牌　孫興慜免除兵役]</ref>。

== 家庭 ==
孫興慜的父親{{le|孫雄政|Son Woong-jung}}是一名在球壇有很大影響力的人物，所以孫興慜的成功很大程度上源於父親對他的鞭策。
