'''孫嘉禾'''（英文：Coco，{{bd|1972年|1月24日|catIdx=Sun孫}}），出生於[[台灣|台灣]][[花蓮縣|花蓮]]，目前是[[MOMO購物台|MOMO購物台]]的'''[[電視購物|電視購物]]專家'''，節目主持LIVE銷售成功率超高，慧眼獨具並用嚴謹的態度分析產品，有層次的堆疊商品優點，常常引起奇蹟般爆線的訂購量，人稱＂[[電視購物|電視購物]]專家一姐＂。

== 學歷 ==
1989年孫嘉禾於自[[花蓮女中|花蓮女中]]畢業。

== 簡歷 == 
*2002年－迄今，富邦[[MOMO購物台|MOMO購物台]][[電視購物|電視購物]]專家
*2001年，傳銷業講師
*1993年－2000年，[[南山人壽|南山人壽]]保險業務員
*1990年－1992年，[[和泰汽車|和泰汽車]]汽車銷售員
*1989年，[[長榮航空|長榮航空]]空姐

== 經歷 ==
'''孫嘉禾'''17歲從[[花蓮女中|花蓮女中]]畢業，因為聯考只考上私立大學，孫嘉禾放棄了繼續求學，選擇直接踏入職場。隨即考上[[長榮航空|長榮航空]]，成為[[長榮航空|長榮航空]]第一期＂寶寶班＂的員工。19 歲那年，她當選第一屆[[蓬萊|蓬萊]]仙子選美冠軍，因而被汽車業者相中，開始做汽車銷售。短短一年，就贏得了「[[Camary|Camary]]（和泰汽車某車款）皇后」的美譽。20歲作[[南山人壽|南山人壽]]保險業務員，在保險界一待8年，前6年是她的黃金期（後來因為健康問題而處於休養狀態）。29歲轉行做SPA傳銷，之後到另一家傳銷公司擔任講師，主題跟銷售技巧和開發個人潛能有關。

32歲由於長期在[[香港|香港]]、[[泰國|泰國]]、[[中國大陸|中國大陸]]等地奔波，以致[[腰椎|腰椎]]出現問題，此時正好親友聽聞[[富邦|富邦]]在徵選購物專家，她便決定轉換跑道，由於個人在[[富邦金控|富邦金控]][[蔡明忠|蔡明忠]]面前展現強烈企圖心，因此在兩千名徵選者中脫穎而出，成為[[富邦|富邦]][[MOMO購物台|MOMO購物台]]的開台元老，參與[[電視購物|電視購物]]節目主持。目前[[富邦|富邦]][[MOMO購物台|MOMO購物台]]開台滿九年，她也作[[電視購物|電視購物]]九年，曾經創下一分鐘產值11萬的佳績，是目前[[富邦|富邦]][[MOMO購物台|MOMO購物台]]的資深[[電視購物|電視購物]]專家。<ref>[[遠見雜誌|遠見雜誌]]－前進的動力[http://www.gvm.com.tw/Board/content.aspx?go=cover&ser=14217 60秒產值11萬 電視購物專家 孫嘉禾 不成熟市場，愈有機會] {{webarchive|url=https://web.archive.org/web/20141007032100/http://www.gvm.com.tw/Board/content.aspx?go=cover&ser=14217 |date=2014-10-07 }}</ref>

賣過七千種商品，幾乎每檔都達陣。一款包包熱賣三年，賣出八萬個。每分鐘創造二十萬產值，一人年創五億元業績，穩坐[[MOMO購物台|MOMO購物台]]一姐地位。

2002年加入富邦[[MOMO購物台|MOMO購物台]]，成為電視購物專家，任職至今。<ref>[http://enews.url.com.tw/enews/32368 富邦momo銷售天后孫嘉禾]</ref>
2008年12月份，孫嘉禾被控介入華航已婚機師楊俊銘的婚姻，涉嫌妨害家庭。孫嘉禾表示不知道楊俊銘已經婚，同時反控楊妻李筱萍涉嫌傷害。
