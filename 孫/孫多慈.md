'''孫多慈'''（{{Bd|1913年||1975年||catIdx=T蔡}}<ref name=":0">{{Cite web|url=http://www.nmh.gov.tw/zh/exhibition_2_3_22_729.htm|title=回眸有情─孫多慈百年紀念展|accessdate=2017-11-07|author=|date=2015-02-09|last=|work=www.nmh.gov.tw|publisher=國立歷史博物館}}</ref>），[[又名|又名]]'''孫韻君'''，是一位出身安徽[[壽縣|壽縣]]的[[藝術教育|藝術教育]]家。

== 生平 ==
1913年，孫多慈[[出生|出生]]於[[安徽省_(中華民國)|安徽省]]壽縣；她的[[父親|父親]]是孫傳瑗，他曾在[[孫傳芳|孫傳芳]]手下擔任[[秘書|秘書]]，後於[[大學|大學]]擔任[[教師|教職]]及[[行政|行政]]職務。<ref name=":1">{{Cite news|url=http://www.chinatimes.com/newspapers/20130817000957-260115|title=孫多慈百年回眸 依然有情|author=|last=|date=|work=|newspaper=中時電子報|accessdate=2017-11-07|language=zh-TW}}</ref><ref name=":2">{{Cite book|title=回眸有情─孫多慈百年紀念展|last=|first=國立歷史博物館編輯委員會|publisher=國立歷史博物館|year=2013|isbn=9789860374452|location=|pages=208}}</ref>
[[File:Sunduoci_1951_NYC.jpg|缩略图]]與[[胡適|胡適]]（右一）、[[汪亞塵|汪亞塵]]（左一）、王少陵（後排左一）等拍攝的合照。]]
1930年，孫多慈參加[[國立中央大學|國立中央大學]]入學考試，但未被錄取；自此，她開始在該校{{Link-en|旁聽|Academic audit}}藝術相關課程，她的作品也受到[[徐悲鴻|徐悲鴻]]賞識。1931年，孫多慈考入該校。此後，徐悲鴻曾勉勵孫多慈效仿{{Link-en|羅莎・邦賀|Rosa Bonheur}}等女性藝術家。1935年，孫多慈[[畢業|畢業]]並計畫申請[[公費留考|公費留學]]；同時，徐悲鴻委請[[宗白華|宗白華]]為《孫多慈描集》作[[序|序]]。當時，孫多慈與徐悲鴻的關係引起眾人矚目和議論。<ref name=":1" /><ref name=":2" />

[[中國抗日戰爭|中國抗日戰爭]]爆發後，孫多慈一路遷徙於安徽[[安慶|安慶]]、湖南[[長沙|長沙]]、廣西[[桂林|桂林]]及浙江[[麗水|麗水]]，並在1941年與時任[[浙江省政府|浙江省政府]]教育廳長的許紹棣[[結婚|結婚]]。<ref name=":1" /><ref name=":2" />

1949年，孫多慈隨[[家人|家人]]遷居[[臺灣|臺灣]]，並前往[[歐美|歐美]]各地精進技藝，又在[[香港|香港]]、[[菲律賓|菲律賓]]、[[美國|美國]]、[[泰國|泰國]]、[[伊朗|伊朗]]等地舉行展覽；同時，她也在[[臺灣師範大學|臺灣師範大學]]擔任教職直至1971年[[退休|退休]]為止。<ref name=":1" />

1963年，孫多慈應[[張其昀|張其昀]]邀請籌組[[中國文化學院|中國文化學院]]美術系，並擔任該系第一任主任。<ref name=":0" /><ref name=":2" />

1975年，孫多慈因[[癌症|癌症]]逝世。<ref name=":1" /><ref name=":2" />

== 家庭 ==
孫多慈與許紹棣育有許玨方等[[子女|子女]]。<ref name=":1" />

== 創作風格 ==
孫多慈的創作形式包含[[油畫|油畫]]及[[水墨畫|水墨畫]]等，並繼承徐悲鴻的[[寫實主義|寫實]]主張。<ref name=":1" />

== 代表作品 ==
* 《瓶汲》（1930年）
* 《狮》（1934年）
* 《自畫像》（1934年）
* 《景寧風景》（1943年）
* 《[[泰国公主|泰国公主]]》（1961年）
* 《春城無處不飛華》（1971年）
