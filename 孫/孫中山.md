'''孫中山'''（{{bd|1866年|11月12日|1925年|3月12日|catIdx=S孫}}{{notetag|孫文生日採用公定日期與公認說法<ref>{{cite web |url=http://laws.mol.gov.tw/chi/flaw/FLAWDOC01.asp?lsid=FL014931&lno=23 |title=勞動基準法施行細則第二十三條 |publisher=勞工部法令查詢系統 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304035214/http://laws.mol.gov.tw/chi/flaw/FLAWDOC01.asp?lsid=FL014931&lno=23 |archivedate=2016-03-04 }}</ref>，為其逝世後，廣東[[中國國民黨中央黨部|中國國民黨中央黨部]]向時居[[澳門|澳門]]的孫氏[[元配|元配]][[盧慕貞|盧慕貞]]求證生辰，盧氏據[[訂婚|訂婚]]時交換之[[八字|八字]]查出，孫生於[[同治|同治]]五年（1866年）[[農曆|農曆]]十月初六[[寅時|寅時]]（[[公曆|公曆]]11月12日）<ref name=吳相湘 />{{rp|18}}。然而孫在1897年11月應英國[[劍橋大學|劍橋大學]]漢學教授[[翟理斯|翟理斯]]（Herbert Allen Giles）之請所著自述，則自稱生於1866年農曆十月十六日，與訂婚八字相差10天。學者[[黃季陸|黃季陸]]推測十月十六的「十」字可能是「初」字筆誤<ref>{{cite book |author=黃季陸 |chapter=〈關於國父生辰的考證〉 |title=《研究中山先生的史料與史學》 |location=台北 |publisher=中華民國史料研究中心 |date=1975-12-12 }}</ref>。}}），名'''文'''，[[幼名|幼名]]'''帝象'''，[[譜名|譜名]]'''德明'''，[[表字|字]]'''載之'''，[[號|號]]'''逸仙'''、'''日新'''。流亡日本時，曾[[化名|化名]]'''中山樵'''<ref name=海 />{{rp|2936}}，故通稱'''孫中山'''。孫中山為[[廣府人|廣府人]]，出生於[[大清帝國|大清帝國]][[廣東省_(清)|廣東省]][[廣州府|廣州府]][[香山縣|香山縣]][[翠亨村|翠亨村]]，祖籍[[广东省|广东省]][[东莞縣|东莞縣]]。清末民初[[醫師|醫師]]、[[政治家|政治家]]、[[革命家|革命家]]、[[哲学家|哲学家]]，是[[中華民國|中華民國]]奠基人，亦是[[中國國民黨|中國國民黨]]之創黨人。提倡[[三民主義|三民主義]]，一般認為是為中國歷史結束帝制的代表人物及思想領袖。

其童年曾受[[太平天国|太平天国]]的影响而有[[革命|革命]]思想。28岁上書[[清朝|清朝]]重臣[[李鴻章|李鴻章]]建言[[改革|改革]]，卻遭遇李的冷落<ref>{{cite web |url=http://military.dwnews.com/news/2014-06-10/59477084.html |title=遭李鸿章冷落孙中山失望下发起革命 |publisher=多维新闻网 |date=2014-06-10 |access-date=2014-06-11 |archive-url=https://web.archive.org/web/20140712063537/http://military.dwnews.com/news/2014-06-10/59477084.html |archive-date=2014-07-12 |dead-url=yes }}</ref>，[[日本|日本]]命令將兵遣師，[[甲午戰爭|甲午戰爭]]爆發<ref name=師 >{{cite book |quote=近者日本命將遣師，侵入吾土，除宅居戰地之人民外，罕有知中日開釁之舉也者。 |authors=孫文自述、甘作霖譯 |chapter=《伦敦被难记》布里斯特耳1897年英文初版，上海商務印書館1912年5月中文譯版 |location=北京 |year=1981 |title=《孫中山全集》第一卷 |publisher=中華書局 }}</ref>{{rp|52}}。1894年11月24日，孙中山在[[檀香山|檀香山]]成立[[兴中会|兴中会]]（[[中國國民黨|中國國民黨]]前身）<ref name=孫中山全集 >{{cite book |title=《孫中山全集》第一卷 |location=北京 |year=1981 |publisher=中華書局 }}</ref>{{rp|52}}。1895年，兴中会发动[[乙未广州起义|乙未广州起义]]<ref name=孫中山全集 />{{rp|53}}。1905年，在[[日本|日本]][[東京|東京]]組成[[中國同盟會|中國同盟會]]，孫被推為[[總理|總理]]<ref name=海 >{{cite book |title=《辭海》（1989年版） |publisher=[[上海辭書出版社|上海辭書出版社]] |year=1989 |isbn=7532600831 |editor=辭海編輯委員會 }}</ref>{{rp|2936}}，主張使用[[青天白日旗|青天白日旗]]作為革命旗幟，反對使用黃興提出的[[井字旗|井字旗]]；確定「驅除[[韃虜|韃虜]]，恢復中華，建立民國，平均地權」，並提出[[三民主義|三民主義]]學說<ref name=海 />{{rp|2936}}。1911年12月29日，被十七省代表在[[南京|南京]]推选为[[中華民國臨時大總統|中華民國臨時大總統]]，1912年1月1日在南京宣布就职，建立[[中華民國臨時政府_(南京)|中华民国臨时政府]]<ref name=海 />{{rp|2936}}。1919年，将[[中华革命党|中华革命党]]改组为[[中国国民党|中国国民党]]<ref name=海 />{{rp|2936}}。1940年4月，[[國民政府|國民政府]]明令尊稱孫中山先生為「[[中華民國國父|中華民國國父]]」<ref>{{cite web |url=http://www.president.gov.tw/Default.aspx?tabid=1064 |title=國父孫中山先生 }}</ref>。中国国民党永久保留其於黨內的「[[中國國民黨主席|總理]]」職銜。[[中國共產黨|中國共產黨]]尊其為「中國近代民主革命的偉大先行者」<ref name=海 />{{rp|2936}}<ref>{{cite web |url=http://news.ifeng.com/history/special/sunzhongshan/200906/0628_7072_1223479.shtml |title=毛泽东：孙中山是伟大的革命先行者 }}</ref>。

== 名字 ==
孫除了本名與下表所使用的[[名字|名字]]外，亦用過大量[[中文|中文]]、[[英文|英文]]、[[日文|日文]][[化名|化名]]與[[筆名|筆名]]<ref>{{cite web |url=http://www.sunyat-sen.org:1980/b5/sunyat-sen.org/sun2007/show.php?id=182 |title=孫中山的名號稱謂獵趣 |publisher=孙中山故居纪念馆 |accessdate=2010-07-29 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20160306234103/http://www.sunyat-sen.org:1980/b5/sunyat-sen.org/sun2007/show.php?id=182 |archivedate=2016-03-06 }}</ref>，多是為了擺脫[[通緝|通緝]]或隱蔽身份而取，部分則在於宣揚[[革命|革命]]或是表達人生期望<ref>{{cite web |url=http://www.sszx.org.cn/sunyjView.aspx?id=969 |title=辛亥年孫中山在倫敦化名考 |author=李纾 |accessdate=2018-01-12 |language=zh-cn }}</ref>。在歐美，孫以其號「逸仙」或全稱「孫逸仙」行。

此外，孫曾自稱為[[洪秀全|洪秀全]]第二，並認為洪氏為「反清英雄第一人」。有人認為這是由於孫氏接受西式教育，不受傳統忠君觀念束縛，才敢於如此自稱；中华民国歷史教科书亦採用此說。但孙後也曾批洪秀全不知民權、民主。

另外，孫在香港西醫書院中讀書時，常當眾倡言反清，聞者多膽怯走避，與[[陳少白|陳少白]]、[[尢列|尢列]]、[[楊鶴齡|楊鶴齡]]三人常住香港，「聽夕往還，所談者莫不為革命之言論，所懷者莫不為革命之思想，所研究者莫不為革命之問題。四人相依甚密，非談革命則無以為歡，數年如一日。故[[港澳|港澳]]間之戚友交遊，皆呼予等為『[[四大寇|四大寇]]』」<ref name=孫文學說第八章/>。

[[清政府|清政府]]公文中，皆在其名字「文」上加上三點水[[部首|部首]]，貶稱其為「'''孫汶'''」。“汶汶”一詞，出自《[[史記|史記]]·[[屈原|屈原]][[賈誼|賈生]]列傳》：“人又谁能以身之察察，受物之汶汶者乎？”《[[史記索隱|史記索隱]]》說“汶汶，猶昏暗也”，《[[史記集解|史記集解]]》引[[王逸|王逸]]說是“蒙垢污”，或說“玷污也”。通常清政府對於犯人其在名字旁會多加水字旁部首，以視為江洋草莽大盜，如刺殺[[馬新貽|馬新貽]]的「張文祥」被稱為「張汶祥」。

辛亥革命後，孫與[[黃興|黃興]]合稱為'''孫黃'''。

孫文於[[1922年|1922年]][[六一六事變|六一六事變]]之中於[[永豐艦|永豐艦]]上炮擊廣州城，造成廣州城傷亡百餘人，從此有了「'''孫大炮'''」的名號<ref>http://sophist4ever.pixnet.net/blog/post/24705418-%E3%80%90%E6%98%A8%E5%A4%A9%E7%9A%84%E7%82%AE%E7%81%AB%E5%B1%A0%E5%9F%8E%E3%80%91%E4%B9%9F%E8%AB%87%E5%AD%AB%E5%A4%A7%E7%82%AE%E9%80%99%E5%80%8B%E7%B6%BD%E8%99%9F%E7%9A%84%E7%94%B1 【昨天的炮火屠城】也談孫大炮這個綽號的由來!</ref>，常被反對者使用<ref>[http://view.news.qq.com/original/legacyintouch/d518.html 孙中山不该被讽刺为“孙大炮”]</ref>。

孫於1925年3月12日病逝後，當時於[[北京|北京]][[中山公园|中央公園]][[北京社稷坛|社稷壇]]舉行公祭時，[[河南省_(中華民國)|豫]]軍總司令[[樊鍾秀|樊鍾秀]]特致送巨型素花橫額（闊丈餘，高四、五尺），當中大書「[[國父|國父]]」二字，他的唁電輓幛，均稱「國父」，這是孫在公開場合被尊稱為「國父」之始<ref name="由來">{{cite web |language=zh-tw |publisher=孫中山學術研究資訊網 |title=國父的由來 |url=http://sun.yatsen.gov.tw/content.php?cid=S01_02_01 }}</ref>。

[[中國抗日戰爭|中國抗日戰爭]]中期，1938年1月16日，[[日本首相|日本首相]][[近衛文麿|近衛文麿]]發表 “不以國民政府為對手”，積極拉攏[[汪精衛|汪精衛]]；1940年3月29日，汪精衛和日本達成協議成立「中華民國國民政府還都南京」。為了爭取正統地位，1940年4月，中國國民黨中央常务委员会第143次会议决议：尊称本党总理为国父，以表尊崇。根据中常会决议，[[国民政府|国民政府]]以孙「倡导国民革命，手创中华民国，更新政体，永奠邦基，谋世界之大同，求国际之平等，光被四表，功高万世」。同年4月1日，通令全国尊稱孫为中华民国国父<ref name=命/>{{rp|707}}。同年5月29日，国民政府又发布明令，规定在政府[[公家|公家]]机关、民众团体应一律改称国父，在中国国民党党内称国父或总理均可，民间已印就之图书文字，不必强令改易。

1941年5月29日，[[汪精卫政權|汪精衛的南京国民政府]]中央政治委員會第49次會議通過由[[陳公博|陳公博]]提出的“手創中華民國之中國國民黨總理孫中山先生應尊稱為中華民國國父”的議案，自此之後，孙中山即在[[中国历史|中国历史]]上确立其“国父”的地位<ref name=由來/>。规定此後有关“公牍、教科书籍、报纸、刊物及一切文字称述总理或孙先生时，均应改称国父”。

中国共产党和[[中华人民共和国|中华人民共和国]]政府并未确立中华人民共和国国父，因[[毛澤東|毛澤東]]建立中华人民共和国「取代」中华民国，故不稱孙中山为中华民国国父。[[中華人民共和國政府|中華人民共和國政府]]或[[中國共產黨|中國共產黨]]在正式場合或文書上提及孫中山時，通常使用稱號是「'''中国民主革命的偉大先行者'''」，從未稱孫中山為國父，现时，中国共产党和中华人民共和国政府不使用“国父”、“国母”等词。但中國大陆民間仍称孙中山为国父。

孫祖先系出金陵，其遠祖固，宋代進士<ref name=命/>{{rp|258}}。祖籍广东省东莞县<ref>{{cite web |language=zh-cn |publisher=孫中山故居紀念館 |title=《孫中山的家世——資料與研究》 |url=http://www.sunyat-sen.org/sun2007/sunjiasi/sunjiansi1.htm |accessdate=2015-09-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20170701223218/http://www.sunyat-sen.org/sun2007/sunjiasi/sunjiansi1.htm |archivedate=2017-07-01 }}</ref>。[[元朝|元末]]大亂，玄孫常德乃遷至廣東[[南雄|南雄]][[珠璣巷|珠璣巷]]，不久沿著廣東北部珠江流域之北江南下，最後定居於廣東省東莞縣之圓頭山鄉，是為來粵之一始祖，稱一世祖<ref name=命/>{{rp|258}}。二世祖貴華移居同縣瀕珠江東岸之長沙鄉<ref name=命/>{{rp|258}}。至明代，五世祖禮贊已嫌長沙鄉人多，耕地不足，無法謀生，就橫渡珠江往西岸，到達[[香山縣|香山縣]]之涌口村定居<ref name=命/>{{rp|258}}。到十一世祖瑞英，涌口村又患人滿，乃於清代乾隆時期西移，找尋原始土地開墾以謀生，結果到達當今翠亨村邊緣之逕仔蓢開基<ref name=命/>{{rp|258}}。根据孙文故居《孙氏家谱》所载，始祖、二世、三世、四世祖，都在东莞上沙乡居住，五世礼赞公才从上沙迁走，先辈族谱仍以上沙作为“故乡”。

孫文高祖父孫殿朝（1745年－1793年），高祖母林氏；曾祖父孫恒輝（1767年－1801年），曾祖母程氏；祖父孫敬賢（1789年－1850年），祖母黃氏（1792年－1869年）。父親[[孫達成|孫達成]]（1813年9月26日－1888年3月23日），母親楊氏（1828年－1910年），長兄[[孫德彰|孫德彰]]（1854年－1915年），二姊孫金星（1857年－1860年），三兄孫德佑（1860年－1866年），四姊孫妙茜（1863年－1955年），妹孫秋綺（1871年9月3日—1912年4月18日）<ref name=命/>{{rp|283}}。溯自常德至廣東，孫是為第十八代<ref name=命/>{{rp|258}}。

孫文的祖先務農為生，曾祖父孫恒輝家有田產十餘畝，當時屬小康之家，22歲時生孫敬賢，但於孫敬賢14歲即過世。

祖父孫敬賢依寡母為生，篤信[[堪輿|堪輿]]，醉心[[術士|術士]]們的風水之說，又常遊山玩水，以致家道中落<ref>{{cite book |author=尚明軒 |title=《孫中山傳》 |publisher=思文化 |year=2014 }}</ref>。孫敬賢有三個兒子， 依序是孫達成（1813年－1888年，孫中山的父親）、孫學成（1826年－1864年）、孫觀成（1831年－1867年）<ref name=命 />{{rp|275}}。

孫家到孫達成時代，所有祖先遺留下來之耕地已所剩無多，他在翠亨村無法謀生，1830年他跑到[[澳門|澳門]]一所鞋店當學徒<ref name=命/>{{rp|275}}。1854年孫達成32歲返回翠亨村成親<ref name=命/>{{rp|275}}，與位於翠亨村東北約4公里、廣府人聚居之崖口鄉隔田村楊勝輝家女兒結婚<ref name=三/>{{rp|85}}。婚後，在家鄉翠亨村依靠耕作養家糊口。孫達成既種水稻也種番薯，但一家大小平常都是吃番薯，又為村人補鞋為副業，待1866年孫出生，孫達成還必須在晚上打更幫補，白天種地晚上打更，日夜不得休息<ref name=命/>{{rp|286}}。

二叔父孫學成年輕時，與同鄉到[[美國|美國]][[加州|加州]][[淘金|淘金]]謀生，至34歲（1860年）才回鄉耕作，可惜38歲 (1864年) 即因病過世。三叔父孫觀成則於年輕時，前往[[上海|上海]]做工，至29歲（1860年）才回鄉，耕田為生。1864年春，孫達成兄弟三人曾共同批耕其先祖孫瑞英遺留在逕仔蓢一帶山上荒田，種植果樹。孫觀成因家境艱難，再次到[[上海|上海]]謀生，後(1867年)在上海附近船上病死，享年37歲。孫文的二個叔父早年均因為貧窮必須遠赴他鄉做工謀生，而且在他出生前後因病去逝，孫看到清朝的績弱不振，受到外國強敵的欺侮，對於自己從事反清革命的動機，曾說過：「幼時的境遇刺激我……我如果沒出生在貧農家庭，我或不會關心這個重大問題」<ref>{{cite book |authors=宮崎寅藏著、[[陳鵬仁|陳鵬仁]]譯 |title=《宮崎滔天論孫中山黃興》 |location=台北 |publisher=[[正中書局|正中書局]] |year=1977 }}</ref>{{rp|6}}。

== 早年經歷 ==
1866年11月12日（[[清|清]][[同治|同治]]五年十月六日<ref name=吳相湘>{{cite book |author=吳相湘 |title=《孫逸仙先生傳》上冊 |location=台北 |publisher=遠東圖書公司 |year=1982 }}</ref>{{rp|18}}），孫在廣東省廣州府香山縣南萌鎮[[翠亨村|翠亨村]]誕生<ref name=三/>{{rp|116}}；乳名帝象，上村塾讀書時取名文，成親時取字德明<ref name=三>{{cite book |author=黄宇和 |title=《三十歲前的孫中山——翠亨、檀島、香港1866-1895》 |publisher=中華書局（香港） |year=2011 |isbn=978-988-8104-67-3 }}</ref>{{rp|4}}。父親孫達成在澳門一家鞋店當學徒，32歲才與家鄉的楊勝輝的女兒結婚。孫中山是家中排行第五的孩子，上面有二兄（[[孫德彰|孫德彰]]、孫德佑）及二姊（孫金星、孫妙茜），其中二兄孫德佑及長姊孫金星均在幼年過世，下面有一個妹妹孫秋綺。孫中山家裡貧窮，他出生時，父親已經53歲，母親38歲，長兄孫眉與他相差12歲。1869年10月9日，祖母黃氏去世；由於家窮，乃兄[[孫眉|孫眉]]到鄰鄉南蓢地主程名桂家當長工<ref name=三/>{{rp|4}}。

1871年9月3日，妹秋綺生；由於家計實在無法維持，孫眉赴[[檀香山|檀香山]]當華工<ref name=三/>{{rp|4}}。11月12日，童年孫與姐姐妙茜上山打柴，無論從金檳榔山山頂或犛頭尖山山腰，遙望[[珠江|珠江]]河口，都能看到金星門雲集大批走鴉片煙土之外國商船<ref name=三/>{{rp|4}}。

1872年，孫入翠亨村[[私塾|私塾]]，接受[[國學|國學]]啟蒙教育<ref name=言>{{cite book |author=孫中山 |editor=葉匡政 |title=《孫中山非常言：論道1896－1925》 |location=北京 |publisher=人民日報出版社 |year=2011 |edition=第1版 }}</ref>{{rp|204}}。孫說過：「革命思想之成熟固予長大後事，然革命之最初動機，則予在幼年時代鄉關宿老談話時已起。宿老者誰？[[太平天國|太平天國]]中殘敗之老英雄是也。」<ref name=命/>{{rp|226}}

孫眉與母舅楊文納前往檀香山，最初他在農場打工，後來向檀香山政府領地，自己在[[茂宜縣|茂宜]]開墾經營農場({{coord|20|41|30.58|N|156|22|42.47|W|region:US_type:landmark}})，事業有成。1876年，孫眉在檀香山做工匯款回家，家境慢慢好轉，孫得以進入村塾讀書；第一位老師是位癮君子，煙癮發作時就曠課；由於老師煙癮頻頻發作，孫也沒法好好讀書<ref name=三/>{{rp|4}}。

1877年6月9日，[[孫眉|孫眉]]從[[檀香山|檀香山]]回到翠亨村結婚，在家住3個月<ref name=命/>{{rp|389}}。7月15日，孫眉在翠亨村成親<ref name=三/>{{rp|4}}。9月22日，孫眉再度赴檀香山，孫擬隨兄赴檀，未果<ref name=三/>{{rp|5}}。

=== 首往夏威夷 ===
1879年5月21日<ref name=命/>{{rp|303}}，孫隨母坐駁艇到澳門坐英輪「格蘭諾去」號（S. S. Grannoch）赴檀香山<ref name=三/>{{rp|5}}，孫回憶道：「始見輪舟之奇，滄海之闊，自是有慕西學之心，窮天地之想」<ref>吳倫霓霞等編：《孫中山在港澳與海外活動史蹟》，廣州：[[中山大學|中山大學]]孫中山研究所、香港：[[香港中文大學|香港中文大學]][[聯合書院|聯合書院]]</ref>{{rp|7}}。1879年孫母親前往檀香山探望長子孫眉，孫眉較孫年長12歲，1871年即離鄉背井，到夏威夷闖蕩<ref name=白>{{cite book |author=[[白吉爾|白吉爾]]（Marie-Claire Bergère）著、溫哈溢譯（第九章協力林添貴，人物小傳協力楊詩韻） |title=《孫逸仙》 |location=台北市 |publisher=[[時報出版|時報出版]] |date=2010-06-21 |isbn=978-957-13-5208-4 }}</ref>{{rp|34}}。6月，孫到檀香山茂宜島茄荷蕾埠孫眉開設商店當店員<ref name=言/>{{rp|204}}。後入盤羅河學校習算術等科<ref name=言/>{{rp|204}}。9月<ref name=三/>{{rp|5}}，孫到一所由[[英國聖公會|英國聖公會]][[主教|主教]]{{le|韋禮士|Alfred Willis}}主持之教會學校[[伊奥拉尼書院|意奧蘭尼書院]]當寄宿生，採全[[英語|英語]]授課<ref name=白/>{{rp|35}}。意奧蘭尼學校，是英國聖公會畢斯浦（Espiscopal Bishop）與威爾士夫人共同於1862年創立，1872年韋禮士牧師繼任校長，翌年遷移校址於白地斯街（Bates Street），擴建房舍<ref name=吳相湘/>{{rp|29}}。學校強制學生必須每天禱告，每週日必須作禮拜，孫開始認識[[基督教|基督教]]儀軌、聖歌，以及閱讀《[[聖經|聖經]]》<ref name=白/>{{rp|35}}。孫在內修讀[[英語|英語]]、[[英國歷史|英國歷史]]、[[算術|算術]]、「[[代數|代數]]、[[幾何學|幾何學]]、[[生理學|生理學]]、[[拉丁文|拉丁文]]與繪圖」等科目<ref name=命/>{{rp|319-320}}。
[[File:Sun_Yat_Sen's_Young_Time2.png|thumb]]

孫在意奧蘭尼書院讀三年書，學習英語，在1883年7月頒獎典禮上得到文法二等獎<ref name=白/>{{rp|35}}。1882年7月27日，孫畢業於夏威夷[[伊奥拉尼書院|伊奥拉尼書院]]<ref name=言/>{{rp|204}}。孫英文文法這一科考第二名<ref name=三/>{{rp|219}}，獲[[夏威夷王國|夏威夷王國]][[國王|國王]][[卡拉卡瓦|卡拉卡瓦]]在毕业典礼上親頒一本中国书籍作为獎品，出色的英文水平为孙中山之后的革命活动助力甚多<ref name="何，2016">{{cite book |author1=何虎生 |title=《孙中山传》 |year=2016 |publisher=中国工人出版社 |location=北京市 |page=16 |edition=第二版 |accessdate=2018-06-14}}</ref>。孫眉認為年輕人應該先完成教育<ref name=白/>{{rp|35}}。

1883年，火奴魯魯華裔知識分子何寬、李昌等組織中西擴論會<ref name=三/>{{rp|5}}。1883年1月15日<ref name=三/>{{rp|5}}，孫與其他教會學校子弟進入[[美國公理會|美國公理會]]創辦之[[奧阿厚書院|奧阿厚書院]]就讀，繼續學習，特別是基礎醫學和[[法律|法律]]<ref name=白/>{{rp|35}}。孫繼續學業（相當於大學預科）<ref name=註2>{{cite web |url=http://sun.yatsen.gov.tw/content.php?cid=S01_01_02_03 |title=國父的求學 |publisher=孫中山學術研究網 }}</ref>。6月30日，奧阿厚書院學年結束，孫坐船回到孫眉在離島所開之牧場<ref name=三/>{{rp|5}}。7月初<ref name=三/>{{rp|5}}，入學不超過三個月，由於孫加入基督教意向不減，又經常勸說在夏威夷工作的[[華僑|華僑]]工人不要膜拜[[關聖帝君|關聖帝君]]神像，兄長憂其觸犯眾怒，二怕雙親斥責，遂斷絕資助，將17歲的孫送回翠亨村家鄉<ref name=吳相湘/>{{rp|16-31}}。孫在1883年夏天從夏威夷回到中國，當他坐遠洋船從[[火奴魯魯|火奴魯魯]]到香港後，孫即上岸轉坐中國內河返回翠亨村<ref name=命/>{{rp|390}}。孫坐輪船到香港轉坐淇澳島之帆船回鄉，抗議清朝官吏向該船乘客勒索<ref name=三/>{{rp|5}}。7月底，孫回到翠亨村<ref name=三/>{{rp|5}}。秋，孫即與同村青年[[陸皓東|陸皓東]]結成好朋友，並經常共同談論時政之腐敗，直到深夜<ref name=命/>{{rp|492}}。8月中<ref name=三/>{{rp|5}}，孫與同鄉友人陸皓東一起「搗毀[[偶像|偶像]]」，破壞翠亨村村中的[[北帝與北帝廟|北帝廟]]神像，不為鄉人所容<ref name=吳相湘/>{{rp|15}}，經崖口、淇澳逃離家鄉前往香港<ref name=三/>{{rp|5}}。陸皓東與孫是同鄉，在上海接受教育<ref name=白/>{{rp|35}}。

=== 入香港拔萃書室及轉學中央書院 ===	
1883年3月31日，美國綱紀慎會[[喜嘉理|喜嘉理]][[牧師|牧師]]在[[香港|香港]]登陸<ref name=三/>{{rp|5}}。8月18日，喜嘉里牧師收留孫在其香港中環之傳道所寄宿<ref name=三/>{{rp|5}}。秋，孫赴香港入讀[[英國聖公會|英國聖公會]]主辦的[[拔萃男書院|拔萃書室]]（當時英文稱為 Diocesan Home and Orphanage，拔萃收容所暨孤兒院）<ref name=註5>[http://www.lib.hku.hk/syshk/D.html 中山史蹟徑]，孫在香港（香港大學圖書館）</ref>為日校走讀生<ref name=三 />{{rp|259-261}}。10月29日，孫在喜嘉理牧師宿舍重逢檀香山傳教士、曾任教於奧阿厚書院之芙蘭·諦文（Frank Damon）及其父母<ref name=三/>{{rp|5}}。是年冬，孫與[[陸皓東|陸皓東]]一同到基督教綱紀慎會（又稱[[公理會|公理會]]）位於港島[[必列者士街街市|必列者士街的佈道所]]，由美國[[公理會|公理會]][[傳教士|傳教士]]喜嘉理牧師於1884年5月4日為他主持[[洗禮|洗禮]]加入教會<ref name=hager17>"...At present there are some seven members in the interior belonging to our mission, and two here, one I baptized last Sabbath,a young man who is attending the Government Central School. We had a very pleasant communion service yesterday..." - Hager to Clark, 5 May 1884, ABC 16.3.8: South China v.4, no.17, p.3</ref><ref name=hager18>"...We had a pleasant communion yesterday and received one Chinaman into the church..." - Hager to Pond, 5 May 1884, ABC 16.3.8: South China v.4, no.18, p.3 postscript</ref>（陸則於1884年10月19日受洗<ref name=hager22>"...Yesterday, 19 October, we also celebrated the Lord's supper in Hongkong and received one on profession of faith. He had heard about the gospel from one of our members who is a student in Hongkong, and he seems to be a quiet but sincere believer..." Hager to Clark, 18 October 1884, ABC 16.3.8; South China v.4, no.22, p.6. 信件日期雖為1884年10月18日，但似乎喜嘉理在開始寫信後，花了數天才完成。</ref>）；在[[中華基督教會公理堂|其所屬教會]]存留之[[受洗|受洗]]名单中，[[陸皓東|陸皓東]]署名「中桂」（名單中排第四）<ref name=命/>{{rp|365}}，孙署名「日新」（名單中排第二），「日新」取《[[大學_(經傳)|大学]]》「苟日新，日日新，又日新」之意<ref name=命/>{{rp|365}}。厥後[[區鳳墀|區鳳墀]]據「日新」二字，為孫改號「逸仙」<ref name=命/>{{rp|365}}。喜嘉里牧師與孫情誼真摰，欣賞孫對宗教熱忱，孫追隨他四處宣教<ref name=白/>{{rp|36}}。

1884年4月15日<ref name=命/>{{rp|362}}，孫轉學香港[[中央書院|中央書院]]（今[[皇仁書院|皇仁書院]]）<ref name=言/>{{rp|204}}。這是一所中學，學生多半是各國中產階級子弟，課程以英文為主，中文為輔<ref name=白/>{{rp|35}}。孫用孫帝象姓名註冊，學號二七四六<ref name=吳相湘/>{{rp|35}}。得知孫[[皈依|皈依]][[基督|基督]]後，孫眉把他召回夏威夷，試圖藉由切斷弟弟所有奧援<ref name=白/>{{rp|36}}。11月，孫奉孫眉急召赴檀香山，交還家產，芙蘭·諦文為孫籌募盤川<ref name=三/>{{rp|5}}。孫在中央書院讀書兩年半期間，[[受洗|受洗]]成為[[基督徒|基督徒]]、第一次婚姻、與孫眉激烈衝突<ref name=白/>{{rp|35-36}}。1885年5月，芙蘭·諦文為孫籌足盤川，自檀香山回到香港復課<ref name=三/>{{rp|5}}。幾個月後，在基督教友人幫助得以於1886年返回香港，由於孫在夏威夷長待幾個月，其中學生涯結束，孫並未拿到中央書院畢業文憑<ref name=白/>{{rp|36}}。

1886年7月，孫未畢業就離開中央書院<ref name=三/>{{rp|5}}。[[邵元沖|邵元沖]]說：「總理自言，幼時旅港肄業，所習多專於英文，嗣而治漢文，不得合用之本，見校中藏有華英文合璧[[四書|四書]]，讀而大愛之，遂反覆精讀，即假以漢文之教本，且得因此而窺治中國[[儒教|儒教]]之哲理。又英譯本釋義顯豁，無[[漢學|漢學]]註疏之繁瑣晦澀，領解較易。總理既目識心通，由是而對[[中國文化|中國文化]]，備致欽崇，極深研幾，以造成畢生學術之基礎。」<ref>{{cite book |author=邵元沖 |chapter=〈總理學記〉 |title=《孫中山生平事業追憶錄》 |location=北京 |publisher=人民出版社 |editors=尚明軒、王學莊、陳崧合編 |year=1986 }}</ref>{{rp|694}}

=== 首次結婚 ===
1885年5月26日，孫奉父命回鄉與同縣外壆村（今珠海市外沙村）[[盧慕貞|盧慕貞]]成親<ref name=三/>{{rp|5}}<ref name=註2/>。盧慕貞自幼纏足，盡心操持家務，照顧公婆和兒女<ref name=下>{{cite book |year=2015 |title=《中國近代被消失的八十年終極版》（下） |location=香港 |publisher=Hong Kong New Media |isbn=978-988-14176-8-8 |authors=蕭若元說、劉文慧統籌、Jo Cheung撰稿及編輯、-{于}-非資料搜集 }}</ref>{{rp|205}}。出生於累世積善的書香門第之家，父親盧耀顯曾至美國檀香山經商致富。二人婚後生了一子二女，即長子[[孫科|孫科]]、長女[[孫娫|孫娫]]與次女[[孫婉|孫婉]]；由於孫中山終日在外奔波，所以和元配聚少離多<ref name=下/>{{rp|205}}。1915年9月1日，孫提出與盧慕貞離婚，盧表示：「孫先生為革命奔走海外，到處流浪，身心為之交瘁，既然現有人願意照料他的生活，我願意成全其美，與先生離婚」<ref name=命/>{{rp|605}}。

== 從醫與革命 ==
[[File:Si_Da_Kou.jpg|160px]]。攝於香港華人西醫書院]]

1886年9月<ref name=三/>{{rp|5}}，孫持[[喜嘉理|喜嘉理]][[博士|博士]]介紹信，進入由（美國）長老教會海外基督使團嘉約翰（John Kerr）博士主持之[[廣州|廣州]][[博濟醫院|博濟醫院]]附設醫科學校習醫<ref name=白/>{{rp|37}}，開始鼓吹革命<ref name=言/>{{rp|205}}。10月，孫認識英國倫敦傳道會宣教師區鳯墀，區鳯墀為其改名逸仙<ref name=三/>{{rp|6}}。孫與基督徒同學[[鄭士良|鄭士良]]結為知己<ref name=白/>{{rp|37}}。鄭士良是上海富商之子，曾就讀於廣州德國教會學校，結交不少廣東會黨人士<ref name=白/>{{rp|37}}。

1887年10月1日，西醫書院教務長孟生醫生在[[香港大會堂|香港大會堂]]宣佈該院成立<ref name=三/>{{rp|6}}。10月3日，孫在[[香港華人西醫書院|香港華人西醫書院]]（College of Medicine for Chinese, Hong Kong，即現今為[[香港大學醫學院|香港大學醫學院]]）正式上課，第一課是[[康德黎|康德黎]]醫生講授之[[解剖學|解剖學]]<ref name=三/>{{rp|6}}。學堂隸屬於倫敦傳道會，由聞名香港之菁英代表[[何啟|何啟]]出資籌辦，該校教務長是[[蘇格蘭|蘇格蘭]]名醫[[康德黎|康德黎]]<ref name=白/>{{rp|37}}。孫曾回憶：「予在廣州學醫甫一年，聞香港有[[香港華人西醫書院|英文醫校]]開設，予以其學課較優，而地方較自由，可以鼓吹革命，故投香港學校肄業。」<ref name=孫文學說第八章>{{cite book |author=孫文 |chapter=第八章〈[[:s:心理建設/第八章|有志竟成]]〉 |title=《[[孫文學說|孫文學說]]——知難行易（心理建設）》 }}</ref>就讀香港西醫書院五年間，孫每於學課餘暇，皆致力於革命之鼓吹，常往來於香港、澳門之間；當時在香港只[[陳少白|陳少白]]、[[尢列|尢列]]、[[楊鶴齡|楊鶴齡]]三人，而上海歸客則陸皓東而已，與陳、尢、楊三人常住香港，聽夕往還，談革命之言論，懷革命之思想，研究革命之問題<ref name=孫文學說第八章/>。孫對[[達爾文|達爾文]]之[[進化論|進化論]]發生興趣<ref>{{cite book |author=[[蔣永敬|蔣永敬]] |title=《國父革命運動史要及其思想之演進》 |publisher=正中書局 |year=1975 }}</ref>{{rp|14}}。楊鶴齡家先在澳明做生意有成，在[[港島|港島]]成立楊耀記商店，「嘗在店內獨闢一樓，為友朋聚集談話之所。……孫、陳、尢、楊四人每日在楊耀記高談造反覆滿，興高采烈，時人咸以四大寇稱之。」<ref>{{cite book |author=馮自由 |title=《革命逸史》第一冊 |publisher=中華書局 |location=北京 |year=1981 |edition=重版 |chapter=〈興中會四大寇訂交始末〉 }}</ref>{{rp|8-9}}四大寇聚談之楊耀記，「同志[[鄭士良|鄭士良]]、[[陸皓東|陸皓東]]等來往廣州、上海過[[香港|港]]時，亦常下榻其間，故該店可稱革命黨人最初之政談俱樂部。」<ref>{{cite book |author=馮自由 |title=《華僑與辛亥革命》 |publisher=中國社會科學出版社 |location=北京 |year=1981 |chapter=〈華僑革命開國史〉 }}</ref>{{rp|2}}孫中山、陳少白、尢列、楊鶴齡相依甚密，非談革命則無以為歡，數年如一日；故港澳間之戚友交遊，皆呼為「四大寇」，為革命言論之時代<ref name=孫文學說第八章/>。

1887年冬孫父達成公病篤，孫回鄉伺奉湯藥，孫眉聞訊亦自檀香山歸，兄弟芥蒂盡釋<ref name=三/>{{rp|378}}。1888年3月24日，孫父達成公在翠亨村病逝，享年76歲<ref name=言/>{{rp|205}}，孫眉對孫加倍愛護備至，凡所需學費，均允源源供給<ref name=三/>{{rp|6}}。

1890年1月，孫介紹[[陳少白|陳少白]]入讀香港西醫書院<ref name=三/>{{rp|6}}。在香港讀書時，孫與[[楊鶴齡|楊鶴齡]]、陳少白、[[尢列|尢列]]倡導者革命，時人稱為「[[四大寇|四大寇]]」<ref name=言/>{{rp|205}}。陳少白在〈四大寇名稱之由來〉稱：「每遇休暇，四人輒聚楊室暢談革命，慕[[洪秀全|洪秀全]]之為人。又以成者為王，敗者為寇，洪秀全未成而敗，清人目之為寇，而四人之志，猶洪秀全也，因笑自謂我儕四人，其亦清廷之大寇乎，其名由是起，蓋有慨乎言之也。時孫先生等尚在香港醫學堂肄業，而時人亦以稱之，實則縱談之四大寇，固非盡從事於真正之革命也。而乙未年廣州之役，楊與尤皆不與焉」<ref name=命/>{{rp|418-419}}。同年孫上書[[鄭藻如|鄭藻如]]，建議禁煙及改良農業和教育等<ref name=三/>{{rp|6}}。12月30日，孫陪康德黎夫婦往廣州痲瘋村研究痲瘋病<ref name=三/>{{rp|6}}。

1891年3月27日，孫與40多名香港道濟會堂年輕教友組織教友少年會<ref name=三/>{{rp|6}}。10月20日，子[[孫科|孫科]]在翠亨村誕生<ref name=三/>{{rp|6}}。

1892年7月，孫以首屆成績第一名畢業於西醫書院<ref name=言/>{{rp|205}}<ref>「1892年，余與孫先生同時畢業於雅麗氏醫院，余年廿一，孫先生年廿六。同班三十餘人，僅吾二人及格而已。」見《江英華回憶錄》</ref>。並獲[[香港總督|香港總督]][[威廉·羅便臣|威廉·羅便臣]]親自頒獎。9月22日，在恩師康德黎醫生帶領下，與同期畢業之江英華同赴廣州<ref name=三/>{{rp|6}}。9月23日，英國駐廣州領事引見[[兩廣總督|兩廣總督]][[李瀚章|李瀚章]]，李命填寫三代履歷，孫拂袖而去<ref name=三/>{{rp|6}}。9月24日，在恩師康德黎醫生帶領下，孫與江英華同方香港<ref name=三/>{{rp|6}}。畢業後，孫在[[澳門|澳門]][[鏡湖醫院|鏡湖醫院]]當醫師<ref name=言/>{{rp|205}}。1892年12月18日，與澳門鏡湖醫院簽訂〈揭本生息贈藥單〉，貸款2,000銀元開設中西藥局<ref name=三/>{{rp|6}}。

1893年春，遷設藥局於廣州，改名東西藥局，藉醫術掩護革命<ref name=言 />{{rp|205}}。1893年7月，孫在澳門議事亭前地十六A號開設「孫醫館」<ref name=三/>{{rp|6}}。7月29日，其澳門中西藥局開業<ref name=三/>{{rp|6}}。中文廣告〈春滿鏡湖〉刊登於1893年9月26日葡文《澳門回聲》稱：「大國手孫逸仙先生，我華人而業西醫者也。性情和厚，學識精明。向從英美名師游，洞窺秘奧。現在鏡湖醫院贈醫數月，甚著功效。但每日除贈醫外，尚有診症餘閒在。先生原不欲酌定醫金，過為計較。然而稱情致送，義所應然。今我同人，為之釐訂規條，著明刻候，每日由十點鐘起至十二點鐘止，在鏡湖醫院贈醫，不受分文，以惠貧乏。復由一點鐘至三點鐘止，在寫字樓候診。三點鐘以後，出門就診。其所訂醫金，俱係減贈。他如未訂各款，要必審視其人其症，不事奢求，務祈相與有成，俾盡利物濟人之初志而已。……」<ref name=命 />{{rp|428-429}}孫行醫於澳門、廣州<ref name=海/>{{rp|2936}}。

據云「（孫）先生在[[香港|港]]曾接納三點會首領，並親自切實調查其實力，約定時間在茶樓飲茶，先生入時，凡起立者即會員。先生如約前往，至十餘處，每處茶客起立者百數十人，喜出望外。實則其頭目事先邀集工人充數，為一騙局」<ref>{{cite book |editor=陳錫祺主編 |title=《孫中山年譜長編》上冊 |publisher=中華書局 |location=北京 |year=1991 |chapter=田桐〈革命閑話〉，《太平雜誌》第一卷第二號 }}</ref>{{rp|88}}。

=== 興中會 ===
{{Main|兴中会|乙未廣州起義|倫敦蒙難記}}
[[File:李鐵夫孫中山12345.jpg|缩略图]] 畫布油畫 93×71.7cm|右]]
1894年1月，孫藉廣州城外南關之聖教書樓懸壺、西關之冼基設東西藥局<ref name=三/>{{rp|6}}。同月，孫在翠亨村寫就〈[[s:上李鴻章書|上李鴻章書]]〉<ref name=三/>{{rp|6}}，主張「富強之大經，治國之大本」在於「人能盡其才，地能盡其利，物能盡其用，貨能暢其流」<ref name=言/>{{rp|206}}。並決定親赴天津面呈給[[直隸總督|直隸總督]][[李鴻章|李鴻章]]<ref name=白/>{{rp|49}}。2月15日，廣州《中西日報》東西藥局廣告：「……大醫生孫君逸仙……舊歲底因事返澳度年，今已由澳回省。」<ref>{{cite book |editor=陳錫祺主編 |title=《孫中山年譜長編》上冊 |publisher=中華書局 |location=北京 |year=1991 |chapter=廣州《中西日報》廣告，1894年2月15日，轉錄於嶺南大學孫中山博士紀念醫院籌備委員會編：《總理開始學醫與革命運動五十周年紀念史略》，廣州嶺南大學，1935 }}</ref>{{rp|71}}。

1894年3月31日，孫中山長女[[孫娫|孫娫]]誕生。

6月1日，前澳門海防同知魏恆致盛宙懷函：「荔孫世丈大人賜覽：久違矩訓，馳繫實深。侄卸前山篆回省，值台旌已先期遄發，未獲面別，殊深悵疚。茲懇者：香山縣醫士孫生名文號逸仙，人極純謹，精熟歐洲掌故；政治、語言、文字皆精通，並善中西醫術，知者甚多，妒者亦復不少。現擬遠遊京師，然後作歐洲之遊。久仰[[盛宣懷|令兄觀察公]]德望，欲求一見，知侄與世丈處，既有年誼世好，又蒙青照有素，特囑函懇賞賜書於令兄觀察公前先容，感激之情，不啻身受者矣。侄賦閑省寓，毫無善狀，幸上下人口平安，堪以告慰。省中新政，諒已早有風聞。茲不多贅。匆匆泐佈，敬請崇安，唯照不莊。興里侄恆頓首。廿八日。」<ref name=料>{{cite book |author=沈渭濱 |title=《辛亥革命史叢刊》第一輯 |publisher=中華書局 |location=北京 |year=1980 |chapter=〈1894年孫中山謁見李鴻章一事的新資料〉 }}</ref>{{rp|89}}6月中，棄醫，偕[[陸皓東|陸皓東]]至[[天津|天津]]上書李鴻章<ref name=言/>{{rp|206}}。孫拿魏恆之信，啟程前往上海，並與上海電報局領班生陸皓東結伴同行<ref name=三/>{{rp|525}}。6月13日，盛宙懷鑒於魏恆情面而接見孫，並致函其堂兄盛宣懷：「敬稟者：頃有滬堂教習唐心存兄之同窗孫逸仙兄，係廣東香山縣人，精熟歐洲醫理，並由廣東前山同知魏直牧函託求轉吾哥俯賜吹植。附呈原件，（尚）祈𧦴閱。特此稟達。恭叩福安。弟宙懷謹稟。初十日。」<ref name=料/>{{rp|93}}在得到盛宙懷推薦信後，孫在上海巧遇[[鄭觀應|鄭觀應]]，結果鄭觀應也為孫謁見李鴻章之事寫推薦信：「杏翁仁兄方伯大人閣下敬肅者：敝邑有孫逸仙者，少年英俊，曩在香港考取英國醫士，留心西學，有志農桑生殖之要術，欲遊歷法國講求養蠶之法，及遊西北省履勘荒曠之區，招人開墾，免致華工受困於外洋，其志不可謂不高，其說亦頗切近，而非若狂士之大言欺世者比。茲欲北遊津門，上書傅相，一白其胸中之素蘊，弟特敢以尺函為其介，俾其叩謁台端。尚祈進而教之，則同深紉佩矣。專肅，敬請勛綏，唯祈  鈞鑒不備  教小弟制鄭觀應頓首。  再肅者：孫逸仙醫生擬自備資斧，先遊泰西各國，學習農務，藝成而後返國，與同志集資設書院教人；並擬遊歷新疆、瓊州、臺灣，招人開墾，囑弟墾我公代求傅相，轉請總署給予遊歷泰西各國護照一紙，俾到外國向該國外部發給遊學執照，以利遄行。想我公有心世道，必俯如所請也。肅此，再叩勛綏不備。  教小弟名心又肅。」<ref name=料/>{{rp|94}}[[王韜_(思想家)|王韜]]有一朋友在李鴻章幕下當文案，王韜就寫信介紹孫到天津，孫就到天津去見文案；軍書旁午，文案把孫大文章送到李鴻章，不知是否見過，後來李鴻章說打仗完以後再見，孫知道沒有辦法，回到上海<ref>{{cite book |author=陳少白 |chapter=〈興中會革命史要〉 |title=《中國近代史資料叢刊——辛亥革命》第一冊 |location=上海 |publisher=上海人民出版社 |year=1981 }}</ref>{{rp|27}}。提出革新政治主張，被拒絕<ref name=海/>{{rp|2936}}。旋遊歷北京、武漢，觀察形勢<ref name=言/>{{rp|206}}。7月25日，[[甲午戰爭|甲午中日戰爭]]爆發<ref name=三/>{{rp|6}}。李鴻章拒絕與孫面談，請願書後來刊載於在1894年9、10月號之[[上海|上海]]《[[萬國公報_(傳教士報紙)|萬國公報]]》<ref name=白/>{{rp|49}}。孫和陸皓東離開天津，在香港短暫逗留後，孫於夏末乘船前往夏威夷<ref name=白/>{{rp|61}}。10月，孫自上海經日本赴[[檀香山|檀香山]]<ref name=言/>{{rp|206}}。

1894年11月24日，孫在檀香山[[歐胡島|歐胡島]]募款<ref>{{cite book |author=[[羅家倫|羅家倫]] |chapter=〈興中會成立日期之史的考訂〉 |title=《中山先生行誼》下冊 |location=臺北 |publisher=臺灣書局 |year=1995 |edition=初版 }}</ref>，在檀香山成立中國有史以來第一個現代革命團體[[興中會|興中會]]<ref name=三/>{{rp|6}}，在檀香山卑涉銀行（Bishop Bank）華籍經理何寬住宅裡，召集僑社人士李昌、劉祥、程蔚南、鄧蔭南、鄭金、黃亮、黃華恢、鍾木賢、許直臣、卓海、李祿、李多馬、林鑑泉、鄭照、劉壽、鍾宇（工宇）、曹采、劉卓、宋居仁、陳南、夏百子、李𣏌、侯艾泉等二十四人<ref name=興>{{cite book |author=[[馮自由|馮自由]] |title=《興中會組織史》 |series=《革命逸史》第四集 |year=1946 }}</ref>{{rp|1-23}}。提出「振興中華」口號和「驱除[[鞑虏|鞑虏]]，恢复中國，创立合眾政府」<ref>{{cite book |title=《國父年譜》增訂本上冊 |publisher=[[中國國民黨中央委員會黨史委員會|中國國民黨中央委員會黨史委員會]] |date=1969-11-24 }}</ref>{{rp|60}}<ref>{{cite book |author=馮自由 |title=《華僑革命開國史》 |publisher=臺灣商務印書館 |year=1953 |quote=「聯盟人某省某縣人某某，驅除韃虜，恢復中國，創立合眾政府，倘有二心，神明鑒察。」 }}</ref>{{rp|26}}政綱<ref name=海 />{{rp|2936}}。後來在香港，誓詞於1895年改為「驅逐韃虜，恢復中華，創立合眾政府」。永和泰商号[[經理|經理]][[劉祥_(興中會)|劉祥]]、卑涉銀行（Bank of Bishop and Co.,Ltd.）華人經理[[何寬|何寬]]為首任正副主席，黃華恢為司庫，程蔚南、許直臣為正副文案<ref name=興/>{{rp|3}}。據〈興中會會員及收入會銀時日與進支數簿〉，從1894年11月24日至1895年9月2日，先後共有112人加入興中會<ref>{{cite book |title=《孫中山在夏威夷：活動和追隨者》 |author=馬兗生 |location=北京 |publisher=世界知識出版社 |year=2003 }}</ref>{{rp|19}}。

=== 第一次廣州起義 ===
1895年1月，孫中山與[[鄧蔭南|鄧蔭南]]等同志返回香港籌謀革命<ref name=言/>{{rp|206}}。興中會香港總會設立於1895年2月18日，糾集孫故交與同窗，其中包括陸皓東、陳少白、鄭士良<ref name=白/>{{rp|62}}。1892年3月13日，香港[[輔仁文社|輔仁文社]]成立，舉[[楊衢雲|楊衢雲]]為會長，[[謝纘泰|謝纘泰]]為秘書<ref name=三/>{{rp|6}}；孫在領導權之爭上面臨楊衢雲挑戰，楊之抱負和自信與孫旗鼓相當<ref name=白/>{{rp|63}}。

2月21日，在香港[[中環|中環]][[士丹頓街|士丹頓街]]13號成立興中會，以「乾亨行」做掩護<ref name=三/>{{rp|6}}。「乾亨」名號是黃詠商所訂，黃父名勝，任香港議政局議員，與大律師何啟有戚誼，何介紹黃與孫相識，黃由是入會<ref name=吳相湘/>{{rp|119}}。同日香港興中會與香港輔仁文社合併，仍稱興中會，會長（總辦）一職未決<ref name=三/>{{rp|6}}。3月1日，孫拜會日本駐香港領事中川恆次郎，請其援助起義，未果<ref name=三/>{{rp|6}}。3月16日，首次幹部會議決定先攻取廣州，並採用陸皓東設計之[[青天白日旗|青天白日旗]]為起義軍旗<ref name=言/>{{rp|206}}。孫從事革命之秋，教會懼其波及，宣言把孫驅逐出堂<ref name=命/>{{rp|480}}。3月18日，《[[德臣西報|德臣西報]]》發表社論，指出革命黨準備成立君主立憲<ref name=三/>{{rp|6}}。5月30日，謝纘泰在英文《德臣西報》致光緒皇帝公開信，促請憲政改革<ref name=三/>{{rp|6}}。孫廣徵同志，定[[重陽節|重陽節]]（10月26日）為起義之日。

新組織「會長」人事案，延宕直至10月10日，是日孫、楊和其各人馬對峙，劍拔弩張，經過妥協，「會長」職位給楊衢雲，孫則獲得廣州起義現場指揮權<ref name=白/>{{rp|63}}。與會者利用傳統宗教信仰為誓「驅除韃虜，恢復中華，創立合眾政府，倘有貳心，神明鑒察。」。興中會選舉[[首領|會長]]（稱總辦，時稱「伯理璽天德」，音譯自[[英文|英文]]「{{lang|en|President}}」），楊衢雲志在必得，孫退讓，楊當選<ref name=三/>{{rp|7}}。

孫利用官方許可，在10月初設立「農學會」<ref name=白/>{{rp|67}}。10月初，香港警方獲線報，謂有三合會份子招募壯勇赴廣州<ref name=三/>{{rp|7}}。朱貴全偕兄朱某及邱四聲言招募壯勇，每名月給糧銀10元<ref name=三/>{{rp|7}}。朱貴全之兄招得400苦力，自己先行他往，壯勇由朱貴全帶領<ref name=三/>{{rp|7}}。孫於汕頭及西江沿岸募集兩軍，同時向廣州進逼<ref name=孫中山全集/>{{rp|53}}。兩軍期於1895年10月某日，一由西南，一由東北，同時向廣州進發<ref name=孫中山全集/>{{rp|53}}。10月25日，朱湘假其弟朱淇之名向緝捕委員李家焯自首；革命黨人原定當天從香港乘夜輪去廣州，因募勇不足，未能成行<ref name=三/>{{rp|7}}。

[[乙未广州起义|乙未广州起义]]定1895年10月26日[[重陽節|重陽節]]舉行<ref name=命/>{{rp|499}}。可是因為事先洩密，接濟未至而起義失敗<ref name=言/>{{rp|206}}。原定起義主力之所謂「敢死隊三千人」沒從香港開到廣州<ref name=三/>{{rp|7}}。清晨6時，在廣州各路人馬首領紛紛來到起義總部王家祠討口令待命，孫人馬已聚集在廣州碼頭等著迎接來自香港之船隻、點收軍火並與傭兵盟友會面<ref name=白/>{{rp|68}}，當天早上8時許，孫方接楊衢雲電報說：「貨不能來」，當時通用有線電報很慢，兩天收到已經很好<ref name=命/>{{rp|499}}。晨8時後，決定取消起義，遣散埋伏水上及附近準備響應之會黨<ref name=三/>{{rp|7}}。孫覆電楊衢雲曰：「貨不要來，以待後命。」<ref name=三/>{{rp|7}}黃昏，陳少白乘「泰安」夜航返香港，孫留廣州善後<ref name=三/>{{rp|7}}。緝捕委員李家焯擬逮捕孫，向兩廣總督[[譚鍾麟|譚鍾麟]]請示；譚鍾麟以孫為教會中人，着李家焯不可魯莽從事<ref name=三/>{{rp|7}}。晚，孫與區鳳墀宣教師連袂赴河南王煜初牧師兒子王寵光婚宴；李家焯探勇掩至婚宴，仍不敢動手，反被孫奚落一番<ref name=三/>{{rp|7}}。10月27日，孫在河南宣教師區鳳墀福音堂與來自香港及廣州本土教眾[[主日|主日]][[崇拜|崇拜]]；禮成，孫扮女裝在教眾掩護下避過探勇坐自備小汽船赴唐家灣轉乘轎子到澳門<ref name=三/>{{rp|7}}。李家焯突然掩至雙門底王家𧙈農學會拿獲陸皓東、程懷、程次3人<ref name=三/>{{rp|7}}。大約400人連同軍火，已在10月27日登上「保安號」（SS. Powan），「保安號」在10月28日清晨被縣令帶令駐防軍逮個正著<ref name=白/>{{rp|68-69}}。不料會員部署略定，忽有密電馳至，謂西南、東北兩軍中途被阻<ref name=孫中山全集/>{{rp|54}}。兩軍既不得進，則应援之勢已孤，即起事之謀已敗<ref name=孫中山全集/>{{rp|54}}。出於萬全考量，孫決定延遲整個行動，並解散雇用人馬<ref name=白/>{{rp|68}}。孫坐轎設法逸逃澳門，再從澳門前往香港<ref name=白/>{{rp|69-70}}。10月29日，孫自澳門坐船抵達香港<ref name=三/>{{rp|8}}。

11月，孫斷然改裝偕陳少白等赴日本，旋組織興中會分會於[[橫濱|橫濱]]<ref name=言/>{{rp|206}}。孫與陳少白、鄭士良同坐「廣島丸」於11月2日由香港出發駛往日本<ref name=命/>{{rp|544}}。11月7日，陸皓東等為首多數成員被捕處刑，孫則受到通緝<ref name=言/>{{rp|206}}。陸皓東即便被嚴刑拷打，也拒不供出同謀名單，他在赴義前稱<ref name=白/>{{rp|69}}：「……吾方以外患為之日迫，欲治其標；孫則主滿仇之必報，思治其本。……要之：今日非廢滅滿清，決不足以光復漢族；非誅除漢奸，又不足廢滅滿清。故吾等尤欲誅一二狗官，以為我漢人當頭一棒。今事雖不成，此心甚慰。但一我可殺，而繼我而起者而不可盡殺。公羊既殁，九世含寃。異人歸楚，吾說自驗。吾言盡矣，請速行刑」<ref name=吳相湘/>{{rp|131}}。11月12日，孫在[[神戶|神戶]]上岸，隨行遇有鄭士良與陳少白<ref name=白/>{{rp|70}}。翌日當地報紙新聞標題：「支那革命黨首領孫逸仙到達日本」<ref name=吳相湘/>{{rp|145}}。孫利用蟄伏日本期間，徹底改變外型：剪掉[[剃髮令|辮子]]、不再薙髮，並脫下中式長袍換穿歐式[[西服|西服]]<ref name=白/>{{rp|71}}。

從1895年建立香港興中會，至1911年辛亥革命成功16年間，孫發動[[孫中山領導起義|十次武裝起義]]中，有六次是香港興中會和同盟會香港分會，以香港為基地秘密發動。香港既是指揮和策劃中心，又是經費籌集與轉匯中心、軍火購製與轉運中心，海內外革命[[同志|同志]]聯絡與招募中心，也是每次起義失敗後革命黨人避難場所<ref>{{cite book |author=[[劉蜀永_(歷史學家)|劉蜀永]] |title=《香港的歷史》 |location=北京 |publisher=新華出版社 }}</ref>{{rp|57-70}}。孫動員秘密會黨，並重新利用興中會以及香港的革命根據地<ref name=白/>{{rp|93}}。

孫把陳少白留在東京，派鄭士良返回香港，旋即啟程前往夏威夷<ref name=白/>{{rp|72}}。1896年1月，孫再赴檀香山<ref name=言/>{{rp|206}}。孫眉安排寡母，連同孫妻小四人，前往他居住之[[茂宜島|茂宜島]]，就近照料他們生活<ref name=白/>{{rp|39}}。孫與母親、長子孫科以及一雙女兒重逢<ref name=白/>{{rp|72}}。3月4日，香港政府認為孫在港從事反清活動，威脅香港安定和秩序，於是簽發驅遂令，禁止孫氏5年內踏足香港境內<ref>Order of banishment，1896-03-04，by the Governor in Council, the Colony of Hong Kong</ref>。6月，自檀香山赴美國<ref name=言/>{{rp|206}}。香港政府下令驅逐孫、楊衢雲、陳少白出境，5年內禁止在香港居留<ref name=三/>{{rp|8}}。9月，自美國赴英國<ref name=言/>{{rp|206}}。

=== 倫敦蒙難、完成三民主義體系 ===	
{{main|倫敦蒙難記}}
[[File:Sun_Yat-Sen_plaque.JPG|thumb]]

1896年9月23日孫從紐約坐船往英國<ref name=命 />{{rp|529}}。晚9時50分到達倫敦[[聖潘克拉斯車站|聖班司車站]]（St Pancras），再僱馬車到達斯屈朗街胥旅社（Haxells Hotel, Strand）投宿，已近翌日子夜<ref name=吳相湘 />{{rp|151}}。孫收到昔日恩師[[康德黎|康德黎]]邀請，於1896年9月30日抵達倫敦<ref name=白 />{{rp|72}}。10月1日早，孫就離開旅社，前往波蘭區覃文省街46號，晤見康氏伉儷<ref name=吳相湘 />{{rp|152}}。10月11日，孫被滿清駐[[英國|英國]][[公使|公使]][[龚照瑗|龚照瑗]]誘捕<ref name=言 />{{rp|206}}。根據歷史學者蕭若元和權威學者黃宇和闡述，孫中山最初於滿清駐倫敦使館外游盪，並計劃爭取領使館內華人支持革命，1896年10月10日，孫終得領館內相熟朋友批准，於第二天容許孫中山進入滿清使館裏巡視。翌日，使館工作人員得悉孫中山將要前來，並設置埋伏，將孫中山拘捕<Ref>蕭若元,[http://www.memehk.com/player-1215-%E8%95%AD%E9%81%99%E9%81%8A-%E7%AC%AC75%E9%9B%86#　"蕭遙遊 第75集"], ''MemeHK'', 2014-02-24</Ref><Ref>黃宇和, ''孙逸仙伦敦蒙难眞相'' (上海书店出版社, 2004)</Ref>。行動計畫是逮捕孫，將他祕密遣送回國，接受法律制裁<ref name=白 />{{rp|73}}。孫得機會說服公使館內一名英國僕人柯尔，将自己比作遭[[土耳其人|土耳其人]]迫害的[[亚美尼亚|亚美尼亚]]基督徒，称中国皇帝也要杀中国基督徒，孫給柯爾20英鎊現金，並允諾事後再給他一千英鎊，从而获得其協助，將便條交給[[康德黎|康德黎]]，上頭寫著：「予於前星期日，被二華人始則施以誘騙，繼則復驟加強暴，將予幽禁於中國使館中。一、二日後，使館將特僱一船，解予回國，回國後必被斬首，奈何？」<ref name=白 />{{rp|74}}10月22日，《環球報》（The Globe）登載康德黎長篇訪問特刊<ref name=白 />{{rp|75}}。經康德黎奔走媒體後，[[英國外交部|英國外交部]]認為[[公法|公法]]上清廷無權在英國領土上拘捕孫，使得此事成為國際事件。孫12天始脫險，撰《[[倫敦蒙難記|倫敦蒙難記]]》<ref name=言 />{{rp|206}}。孫向報紙投稿，召開多次記者會，接受多次訪問，並嬴得並遍支持<ref name=白 />{{rp|75}}。11月，孫在倫敦應[[英國劍橋大學|英國劍橋大學]][[漢學家|漢學家]][[翟理斯|翟理斯]]邀請，撰寫《[[倫敦蒙難記|倫敦蒙難記]]》<ref name=白 />{{rp|75}}。

1897年，孫在倫敦[[大英博物館|大英博物館]]研究並考察[[歐洲政治|歐洲政治]]，完成三民主義思想體系<ref name=言 />{{rp|206}}。

=== 創中西學校、化名中山樵 ===
1897年7月1日<ref name=命 />{{rp|529}}，孫自倫敦經[[加拿大|加拿大]]，8月16日抵達日本橫濱<ref name=命 />{{rp|529}}，旋創[[中西學校|中西學校]]<ref name=言 />{{rp|206}}。日本民黨領袖[[犬養毅|犬養毅]]遣[[宮崎寅藏|宮崎寅藏]]、[[平山周|平山周]]迎接孫<ref name=領導風格 >{{cite book |author=[[蔣永敬|蔣永敬]] |chapter=〈孫中山三大領導風格〉 |editor=[[呂芳上|呂芳上]]主編 |title=《論民國時期領導精英》 |location=香港 |publisher=商務印書館 |year=2009 }}</ref>{{rp|36}}。引至[[東京|東京]]相會，一見如故，抵掌談天下事，大為快慰<ref name=領導風格 />{{rp|36}}。平山周某次帶孫文至東京數寄屋橋外的旅館「對鶴館」住宿要登記姓名時，認為寫日本式名字較安全，平山周想起日前經過日比谷「中山」侯爵邸，於是幫孫文填「中山」的姓，孫文自己則寫上名字為「樵」<ref name=命 />{{rp|24}}，於是孫開始使用「中山樵」的化名，孫中山因此得名<ref name=言 />{{rp|206}}。更因犬養毅之介紹，廣交日本朝野賢豪<ref>{{cite book |author=孫中山 |chapter=〈孫文學說〉第八章〈有志竟成〉 |title=《國父全集》第一冊 |location=台北 |publisher=近代中國出版社 |year=1989 }}</ref>{{rp|49}}。被納入犬養毅羽翼後，孫定居東京，由宮崎寅藏、平山周為伴，身分為宮崎和平山中文教師<ref name=白 />{{rp|88}}。透過他們，孫再結識[[頭山滿|頭山滿]]、進步黨黨魁[[大隈重信|大隈重信]]、[[黑龍會|黑龍會]]{{tsl|ja|山田良政|山田良政}}等人<ref name=白 />{{rp|85}}。11月，孫在日本與宮崎寅藏筆談記錄，孫曾有「僕常持一都四京（五大鎮）之說：武漢（都）、西京（重慶）、東京（江寧）、廣州（南京）、順天（北京）」<ref>{{cite book |author=彭澤周 |title=《近代中日關係研究論集》 |location=台北 |year=1978 |publisher=藝文出版社 }}</ref>{{rp|237}}。12月，命陳少白成立興中會分會於[[台北|台北]]<ref name=言 />{{rp|206}}。

1897年8月16日，孫從英國取道加拿大到達日本橫濱，橫濱華僑溫炳臣有一名15歲女僕[[淺田春|淺田春]]，曾上小學、中學，又上私立英文學校<ref name=命 />{{rp|612}}。快要畢業時，父親突然去世，被迫出來做工，成為溫炳臣小女傭<ref name=命 />{{rp|612}}。日本[[靜岡縣|靜岡縣]]清水町人，懂中文及英文。孫到來後，溫炳臣讓她照顧孫<ref name=命 />{{rp|612}}。是年孫31歲。日本當局也讓她隨時報告孫之行止，1901年以患病為由，回到靜岡老家<ref name=命 />{{rp|612-613}}。淺田春1902年夏過世時，孫頗為傷心。

=== 奔走建立革命組織 ===
[[File:Sun_Yat_Sen_in_Japan_1898.png|thumb]]的合照。前排左起︰安永東之助、楊衢雲、[[平山周|平山周]]、末永節、[[內田良平|內田良平]]；後排左起︰可兒長一、小山雄太郎、[[宮崎寅藏|宮崎寅藏]]、孫中山（秘書）<ref>{{cite news |url=http://www.metrohk.com.hk/index.php?cmd=detail&id=162000 |title=FOCUS 香港烈士楊衢雲被遺忘的辛亥革命 |location=香港 |publisher=《[[都市日報|都市日報]]》 }}</ref><ref>{{cite journal |url=http://www.mingpaomonthly.com/cfm/Archive2.cfm?File=201104/inf/01a.txt |title=回首香江醫療百年 |journal=[[明報月刊|明報月刊]] |issue=二零一一年四月號 |deadurl=yes |archiveurl=https://web.archive.org/web/20150922153303/http://www.mingpaomonthly.com/cfm/Archive2.cfm?File=201104%2Finf%2F01a.txt |archivedate=2015-09-22 }}</ref>、清藤幸七郎、大原義剛。]]

楊衢雲輾轉往來於[[東南亞|東南亞]]、[[南非|南非]]後，於1898年春天前往東京與孫會合<ref name=白 />{{rp|94}}。1898年春夏，孫來往於日本東京、長崎、神戶、[[馬關|馬關]]、橫濱等地，結識[[菲律賓|菲律賓]]起義軍代表{{tsl|en|Mariano Ponce|馬里亞諾·彭西}}<ref name=言 />{{rp|207}}。秋冬，與亡命日本之[[梁啟超|梁啟超]]會談聯合反清，未成<ref name=言 />{{rp|207}}。1899年春夏，孫繼續在日本作革命聯絡策動，並為菲律賓起義軍購運軍械<ref name=言 />{{rp|207}}。12月，孫命陳少白創辦《[[中國日報_(香港)|中國日報]]》於香港<ref name=言 />{{rp|207}}。報紙經費由日本人提供，報紙開辦之初，先是得到[[何啟|何啟]]大力襄助，爾後香港富商之子[[李紀堂|李紀堂]]做靠山<ref name=白 />{{rp|94}}。據《英國殖民地部檔案》編號129卷339，[[香港總督|香港總督]][[梅含理|梅含理]]曾於1913年給英國殖民地大臣報告，提及[[香港立法局議員|香港立法局議員]]何啟和[[韋玉|韋玉]]曾協助革命黨人：「無可爭辯的證據顯示，遠在革命爆發之前，這兩位先生都已得到革命黨人的信任。革命黨的會議經常在韋玉先生家裡舉行……在革命前的好幾個月，他聽任自己的住宅被用為密謀反對清廷的會議場所，而且參與了密謀者的機密……在革命前好幾個月內，何啟亦參與革命黨人的機密，實際上充當他們的顧問，指導他們的活動程度，以免違反香港法律條文的規定。」<ref name=中3 />{{rp|108}}

=== 第二次起義：惠州起義 ===
[[File:Sun_Yat_Sen_1900.png|160px]]

孫一面重啟香港之革命根據地，一面聯絡並擴大秘密會黨之人脈<ref name=白 />{{rp|94}}。1900年，中國發生[[庚子拳亂|庚子拳亂]]，引來[[八國聯軍|八國聯軍]]入侵中國。孫將網絡拓展工作部分託付給日籍顧問和同志，部分託付給湖南改革派畢永年<ref name=白 />{{rp|95}}。陳少白邀請廣東三合會一位大老黃福赴香港，他在陳少白接受「白扇」頭銜之儀式中擔任主持<ref name=白 />{{rp|95}}。1899年11月[[李鴻章|李鴻章]]接任[[兩廣總督|兩廣總督]]，想北京混亂失序之際另起一個獨立政府<ref name=白 />{{rp|96}}。孫藉此機會聯繫李鴻章，希望能籌劃南方諸[[行省|行省]]獨立，成立類似[[美利堅合眾國|美利堅合眾國]]政府，李鴻章也答應與其會見。但在日本友人協助下，卻發覺不過是清廷陷阱。在劉學詢召喚下，孫於1900年6月8日搭船啟程，隨行者還有日本顧問，日本顧問抵達[[香港|香港]]後便開始替孫行事，與劉學詢協商，可是不久後李鴻章決定放棄分離主義念頭，奉朝廷之命回[[北京|北京]]協調[[庚子條約|庚子條約]]，於1900年7月17日趕赴上海<ref name=白 />{{rp|97}}。1900年6月，孫主動求見法國駐越北大使哈爾蒙德（Jules Harmond），6月21日孫抵達西貢，攜帶哈爾蒙德大使給法屬印度支那總督韜美（Paul Dourmer）之信<ref name=白 />{{rp|128}}。同年夏，孫先是抵達香港（並未上岸），然後再到安南、新加坡、上海<ref name=白 />{{rp|104}}。1900年9月，孫與日本友人及原香港興中會核心人物先赴香港，但被禁入境後轉往[[台灣|台灣]]，故孫命鄭士良等人在[[廣東|廣東]][[惠州|惠州]]三洲田（今[[深圳市|深圳市]][[鹽田區|鹽田區]]三洲田村）一帶發動起義（史稱[[惠州起義|惠州起義]]）<ref name=海 />{{rp|2936}}。朝廷軍隊在沙灣遭大敗，起義隊伍直抵鎮隆，北距惠州僅20公里<ref name=白 />{{rp|105}}。10月17日，鄭士良人馬接獲孫下達新指示，轉而揮兵廈門，他們在陌生之境長途行軍遭到清兵追擊，10月23日日本懾於俄國反彈，已放棄在廈門建立據點之念頭，並撤退先前承諾之援助，鄭士良和三合會各龍頭只得逃逸香港，丟下各自人馬，任他門歸巢返鄉<ref name=白 />{{rp|107}}。由于未能按计划与新安、虎门义军会合，该地义军被迫解散。四战四捷后义军枪支弹药无法获得补给，郑士良急电孙，孙在台湾致电宫崎寅藏速送枪弹，由于采购枪械事物的中村弥六从中渔利，所购全是废枪。起义失败告终<ref>{{cite book |authors=沈晓敏、[[倪俊明|倪俊明]] |title=《喋血南国：辛亥革命在广东》 |publisher=广东人民出版社 }}</ref>。鄭士良左右手[[史堅如|史堅如]]在廣州攻擊兩廣總督，10月28日廣州發生爆炸案，兩廣總督脫逃，史堅如在11月9日遭斬首<ref name=白 />{{rp|107}}。楊衢雲在教室中被擊斃；鄭士良於用餐後暴卒<ref name=白 />{{rp|107-108}}。孫返回日本<ref name=白 />{{rp|108}}。

1902年，宮崎滔天出版孫自傳《三十三年落花夢》，1903年中文譯本問世<ref name=白 />{{rp|141}}。同年春，孫受[[章炳麟|章炳麟]]之邀，出席「支那亡國二百四十年紀念會」<ref name=白 />{{rp|142}}。孫在日本和章炳麟談論時又曾涉及革命發難以及成功後建都地點，孫以為「謀本部，則武昌；謀藩服，則西安；謀大洲，則伊犂。視其規模遠近而已」，孫並且說明：「定鼎者，南方誠莫武昌若。尚濱海之建都者，必遏遠武昌。夫武昌揚雲於大江，東趨賓山（江蘇省），四日而極，足以轉輸矣，外鑑諸鄰國，[[柏林|柏林]]無海，[[江戶|江戶]]則曰海堧爾；內海雖咸，亦猶大江也……江沔之在上游，其通達等是矣，何必傅海！夫北望襄樊以鎮撫河雒。鐵道既佈，而行理及於長城，其斥候至窮朔者，金陵之絀，武昌之贏也」<ref name=吳相湘 />{{rp|100}}。孫抵達越南，在華僑中建立革命組織，在[[河內|河內]]結交[[法國|法國]]官員<ref name=言 />{{rp|21}}。從1902年12月至1903年5月，孫停留河內，結交某些法國政治、外交、軍事人士<ref name=白 />{{rp|127}}。

1903年6月，孫抵達橫濱<ref name=白 />{{rp|135}}。同年夏，孫與15歲的[[大月薰|大月薰]]訂立婚約，並應孫要求，轉到公立英和女子學校讀書<ref name=命 />{{rp|613}}。1903年7月，兩人舉行婚禮，1905年8月大月熏懷上孫孩子<ref name=命 />{{rp|613}}。後生一女[[宮川富美子|宮川富美子]]。1903年夏，孫在日本[[青山|青山]]秘密創辦一所軍事學校，訓練遭日本軍事機構排除之中國學生<ref name=白 />{{rp|142}}。被吸收妁15名粵籍學員，青山軍事學校運作不超過4個月<ref name=白 />{{rp|143}}。孫改革命誓詞為「驅除韃虜，恢復中華，創立民國，平均地權」。

=== 夏威夷出生证明 ===
孫於1903年9月26日啟程前往夏威夷<ref name=白 />{{rp|136}}。為求工作方便，孫揚棄先前立場，加入[[三合會|三合會]]<ref name=白 />{{rp|137}}。1904年1月11日，孫中山在檀香山加入[[洪門|洪門]][[致公堂|致公堂]]，成為致公堂「洪棍」。3月31日，孫帶著地方堂口給三藩市三合會之介紹信，啟程前往美國<ref name=白 />{{rp|137-138}}。

为规避《[[排华法案|排华法案]]》<ref name=累計七年 />，孙預先取得偽造文件<ref name=白 />{{rp|138}}，夏威夷準州政府當局發出的[[出生证明|出生证明]]{{notetag|證書申稱「孫逸仙現住在茂宜島的庫拉。見證人證明孫逸仙於1870年11月24日（「11月24日」是[[興中會|興中會]]成立之日）出生在夏威夷群島。證書上的照片與本人相似。」出生證明書上簽署的日期為1904年3月14日。<ref name=累計七年 /><ref name=KS/>}}和美國護照<ref name=非偽造 /><ref name=邮票 /><ref name=累計七年 />。取得該證時親友們做出了孙中山在夏威夷出生的證詞<ref name=非偽造 >{{cite news |author1=陳艷群 |title=孫中山出生證件 非偽造 |agency=世界新聞網 北美華文新聞、華商資訊 |date=2011-06-08 }}</ref><ref name=累計七年 >{{cite news |author1=陳艷群 |title=孫中山先後六次赴檀島 累計七年 |agency=世界新聞網 北美華文新聞、華商資訊 |date=2011-06-08 }}</ref>。

1904年4月6日<ref name=白 />{{rp|138}}，孙从檀香山赴美国，为阻止孙入境，清政府向美国举报孙[[护照|护照]]有问题<ref name=邮票 />。1904年4月6日，孙上岸后，被美国官员扣留并审问，他写下《自述证言》<ref name=邮票 >{{cite web |url=http://www.chinanews.com/cul/2013/05-17/4827760.shtml |title=孙中山曾与林肯同登外国邮票 中国地图为背景 |publisher=中國新聞網 }}</ref><ref name=假證明 >[http://www.ait.org.tw/zh/dr-sys-and-us.html 孫中山與美國特展]，[[美國在台協會|美國在台協會]]舉辦</ref>。《自述证言》{{notetag|1904年之《自述证言》原件现藏于位于加州舊金山的[[美国国家档案局|美国国家档案局]]，目前仅广州[[孙中山大元帅府纪念馆|孙中山大元帅府纪念馆]]有复制件。}}<ref name=邮票 />中称：自己于1870年出生在{{link-en|夏威夷伊娃|Ewa Villages, Hawaii}}<ref name=KS>Worden, R. L. (1971). K'ang Yu-wei, Sun Yat-sen, et al. and the Bureau of Immigration. Ch'ing-shih wen-t'i 2(6), 1-10. Johns Hopkins University Press.</ref>，三四岁时被父母带回中国。

1904年4月16日，旧金山移民局作出“拒绝孙中山入境”的决定。[[移民局|移民局]]做出这个决定的理由是：孙中山曾经在1896年以中国公民的身份入境美国旧金山。华裔报人[[伍盘照|伍盘照]]和舊金山[[洪門|洪門]][[致公堂|致公堂]]保釋及代聘[[律師|律師]]方才免被遣送回中國。1904年4月26日，孙中山委託华盛顿的拉斯頓與席頓律师事务所（Ralston & Siddons），向美国移民局提起上诉。1904年4月28日美国“商务及劳工部”作出复议决定：准许孙中山入境美国，理由是早在1896年的时候，[[夏威夷|夏威夷]]尚非美国属地，是[[夏威夷共和国|夏威夷共和国]]时期，孙不可能拿夏威夷发的美国护照，所以1900年4月30日生效的“美国移民新法案”，自动赋予孙中山“[[美国公民|美国公民]]”的身份，取得“夏威夷护照”，或者说“美国殖民地护照”，又称“疆省护照”。所以准許。6月孫動身巡迴全美十多個城市<ref name=白 />{{rp|138}}。之後孫到[[美國東岸|美國東岸]]尋求[[華僑|華僑]]支持革命。9月底孫抵達[[紐約|紐約]]<ref name=白 />{{rp|139}}。孫首度發表對外宣言，希望博得外國人士對革命的支持與好感，但並未取得甚大成果。不過，當時在香港有《中國日報》，在夏威夷有《檀山新報》，在三藩市有《大同日報》，最後在新加坡有《圖南日報》<ref name=白 />{{rp|139-140}}。

1904年冬，孙中山携[[朱和中|朱和中]]赴[[德国|德国]][[柏林|柏林]]，访问清國籍華裔留德学生。在[[宾步程|宾步程]]的带动下，王发科、王相楚等留学生加入兴中会。在柏林逗留数日后，孙中山与[[朱和中|朱和中]]赴[[法国|法国]][[巴黎|巴黎]]，留法学生唐豸、[[胡瑛|胡瑛]]、[[汤芗铭|汤芗铭]]、向国华等十多人应邀入[[兴中会|兴中会]]。孙中山在巴黎住在利倭尼街的瓦克拉旅馆。一天，[[汤芗铭|汤芗铭]]、向国华、王发科、王相楚四人结伴来找孙中山去附近的咖啡馆喝咖啡，王发科、王相楚二人中途离席，到瓦克拉旅馆盗走孙中山的行箧中的党人名单及入盟书、法国殖民大臣致[[法屬印度支那|法屬印度支那]]总督的信函。此为王发科、王相楚、汤芗铭、向国华四人合伙设计。得手后，四人到清国驻法国公使馆告密，公使[[孙宝琦|孙宝琦]]训斥四人称：“你们加入革命党，是叛清朝；今来自首，又叛革命党。且陷害同学，人格何在！”孙宝琦收阅這四人盗取的文件后称，“今后你们要好好念书，安分守己，不要胡闹。”留法学生夏坚仲是孙宝琦的亲戚，得知消息后赶到使馆向孙宝琦疏通。孙宝琦乃将入盟者的宣誓书烧掉，将其余文件交给夏坚仲，夏坚仲通过邮局将文件寄还孙中山。但是，孙宝琦从法国殖民大臣致法屬印度支那总督的信函中得知孙中山的安南起义计划后，即赴[[法国外交部|法国外交部]]交涉，导致孙中山的起义计划未能实现<ref>{{cite book |url=http://history.people.com.cn/GB/205478/15499794.html |author=胡为雄 |title=《帝制的终结（下）︰辛亥天变》 |publisher=当代中国出版社 |year=2011 |ISBN=9787801709967 }}</ref>。1905年2月，孫抵達巴黎<ref name=白 />{{rp|131}}。同年5月中旬，[[孙中山|孙中山]]以[[社会主义|社会主义]]追随者的姿态访问了设在[[比利时|比利时]][[布鲁塞尔|布鲁塞尔]]的[[第二国际|第二国际]]书记处，与第二国际执行局主席[[王德威尔德|王德威尔德]]和书记[[胡斯曼|胡斯曼]]进行会谈，提出接纳他的[[兴中会|兴中会]]为第二国际成员的请求，王德威尔德、胡斯曼等人认为当时落后的中国还不具备社会主义革命的条件未予批准<ref>王炯华：《[http://cnolo.com/China/nu/2007424/20133643279.htm 略论中国国民党之社会民主党性质] {{webarchive|url=https://web.archive.org/web/20081201234308/http://www.cnolo.com/China/nu/2007424/20133643279.htm |date=2008-12-01 }}》，中国在线</ref><ref>{{cite book |author1=段雲章 |title=中山先生的世界觀 |page=257 |url=https://books.google.com.tw/books?id=RDS-riXYSXMC&pg=PA257&lpg=PA257&dq=王德威尔德+孙中山+1905年&source=bl&ots=x7oEZmMZTv&sig=ACfU3U2jbAsU6iOH35VHbSZhTfkjwSYQiw&hl=zh-CN&sa=X&ved=2ahUKEwiVpP7ej_XgAhUDV7wKHWocBIIQ6AEwAnoECAYQAQ#v=onepage&q=王德威尔德%20孙中山%201905年&f=false |accessdate=2019-03-09}}</ref>。

=== 組建同盟會 ===
[[File:同盟會成立.jpg|160px]]
{{main|中国同盟会}}
1905年中再赴[[遠東|遠東]]。7月19日抵日本[[橫濱|橫濱]]，當即由[[宮崎寅藏|宮崎寅藏]]和[[楊度|楊度]]等人介紹認識了[[黃興|黃興]]，並開始籌劃聯合各革命組織<ref name=大拼盤 >{{cite journal |author=唐德剛 |title=〈同盟會這個革命大拼盤〉 |journal=《[[明報月刊|明報月刊]]》1991年12月號 |location=香港 |publisher=[[明報雜誌有限公司|明報雜誌有限公司]] }}</ref>{{rp|125}}。7月30日，在日本極端民族主義之[[黑龍會|黑龍會]]總部舉行大型會議，有70人出席，會中決議成立新組織，並奉孫為領導，新組織名為「中國同盟會」<ref name=白 />{{rp|145}}。8月13日，孫在東京向留學生作公開演說，大為轟動；演說會場是東京富士見樓（麴町區飯田河岸），會場能容納千人<ref name=領導風格 />{{rp|38}}。大會由宋教仁主持，有逾700名學生雲集在大廳內，還有許多學生不得其門而入，簇擁在門外，聆聽孫首度在東京公開發表演說<ref name=白 />{{rp|145}}。屆時爆滿，後來者猶絡繹不絕，門外擁擠不通；警吏令封門，諸人在外不得入，甚為喧嘩；又開門聽其進場<ref name=領導風格 />{{rp|38}}。

1905年8月20日，在日本人[[內田良平|內田良平]]牽線下，結合孫中山的興中會、[[黃興|黃興]]與[[宋教仁|宋教仁]]等人的[[華興會|華興會]]、[[上海|上海]][[蔡元培|蔡元培]]、[[章炳麟|章炳麟]]與[[吳敬恆|吳敬恆]]等人的[[愛國學社|愛國學社]]、[[張繼_(中華民國)|張繼]]的[[青年會|青年會]]等組織，在日本[[東京|東京]]成立[[中國同盟會|中國同盟會]]<ref name=大拼盤 />{{rp|126}}。成立大會是在日本國會議員阪本金彌之宅邸，在場300人聆聽孫宣讀新組織章程<ref name=白 />{{rp|146}}。中國革命同盟會成立於東京，其誓詞為：「驅除韃虜，恢復中華，建立民國，平均地權」，次年發表「軍政府宣言」，復將此誓詞列為革命建國之「四綱」<ref name=綱 >{{cite book |author=[[郭廷以|郭廷以]] |title=《近代中國史綱》 |location=香港 |publisher=[[中文大學|中文大學]]出版社 |year=1986 |version=第三版 }}</ref>{{rp|366}}。孫定立中華民國之名稱，而公布黨員，使之各回本省，鼓吹革命主義<ref>{{cite book |author=孫中山 |title=《國父全集》第一冊 |location=台北 |publisher=中國國民黨中央委員會黨史委員會校訂 |year=1973 }}</ref>{{rp|498}}。孫被推舉為總理，黃興為執行部庶務，地位形同副總理<ref name=白 />{{rp|146}}。

[[中國同盟會|中國同盟會]]以華興會機關刊物《二十世紀之支那》改組成為《[[民報|民報]]》<ref name=白 />{{rp|146}}。在發刊詞首次提出「[[三民主義|三民主義]]」學說，與[[梁啟超|梁啟超]]、[[康有為|康有為]]等改良派激烈論戰。《民報》是在1905年11月26日創刊，並於1908年10月10日出版第二十四期後遭日本當局下令封禁<ref name=白 />{{rp|169}}。

同盟會宣言開宗明義，呼籲需要一場「國民革命」，由「一國之人」著手進行，且應分兩階段展開：第一階段是驅逐滿人，第二階段是恢復中國主權；滿人乃關外之東夷，他們征服中國、奴役漢人，讓漢人「為亡國之民者二百六十年於斯」，唯有驅逐滿人，才有可能「光復中華」，「中國者，中國人之中國，中國之政治，中國人任之」<ref name=白 />{{rp|172}}。同盟會創立時發表之宣言，是革命意識形態即三民主義之雛形，爾後又藉黃興、章炳麟之助，孫在1906年秋天進一步修訂，即所謂《中國同盟會革命方略》<ref name=白 />{{rp|167}}。正式宣示所進行者為國民革命與編定「同盟會革命方略」，並將創立者為“中華民國”；並舉所誓之四綱，制定「軍法之治，約法之治、憲法之治」三道程序<ref name=註6 >{{cite web |url=http://www.boxun.com/hero/200801/orthodox/5_1.shtml |title=興中會、中國同盟會、黃花崗起義、烈士陵園（1895-1911） |publisher=博讯文坛 |accessdate=2010-01-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20090428093525/http://www.boxun.com/hero/200801/orthodox/5_1.shtml |archivedate=2009-04-28 }}</ref>。宣言主要方針是在《民報》上發展出來，文章由不同作者執筆，觀點具個人色彩，有時還彼此矛盾，確實構成三民主義教義主體，一般認為表達孫之政治思想<ref name=白 />{{rp|167}}。

同盟會其實一直是革命團體間一個極為鬆散之聯盟，其中某些革命團體多年後變得公開與孫為敵<ref name=白 />{{rp|167}}。孫之門徒以排滿[[種族主義|種族主義]]取代反帝國主義做為現代民族主義之基礎，全然將民主等同共和體制，諸如此類<ref name=白 />{{rp|168-169}}。他們處理大多數問題，都是從同盟會敵視梁啟超之觀點來分析<ref name=白 />{{rp|169}}。

=== 第三次到第九次起義 ===
{{main|十次革命起義}}
1905年10月，自日本經[[西貢|西貢]]往[[馬賽|馬賽]]、[[巴黎|巴黎]]<ref name=經緯 >{{cite journal |author=唐德剛 |title=〈武昌起義的經緯〉 |journal=《明報月刊》1992年3月 |location=香港 |publisher=明報雜誌有限公司 }}</ref>{{rp|94}}。1906年4月，孫出席同盟會在新加坡成立分會<ref name=白 />{{rp|163}}。

1907年2月，要做為同盟會象徵旗幟設計，孫滿意[[青天白日旗|青天白日旗]]，[[黃興|黃興]]則想要有中國古風、象徵[[平均地權|平均地權]]政策之「井」字旗<ref name=白 />{{rp|161}}。根據宋教仁所言，孫態度令黃非常不滿：「蓋孫文素日不能開誠布公，虛心坦懷待人，作事近於專制跋扈，有令人難堪處故也。」<ref name=白 />{{rp|161}}一個月後，日本當局受滿清政府施壓，決定驅逐孙離境<ref name=白 />{{rp|161}}。為表達善意，日方除發布驅逐令之外，還以外交辭令表達歉意，並饋贈孫大筆款項<ref name=白 />{{rp|161}}。日本政府送其路費6,000元，日本商人铃木久五郎送其10,000元。孫收款後，留2,000元作《[[民报|民报]]》经费<ref>朱育和等著：《辛亥革命史》，北京：[[人民出版社|人民出版社]]</ref>。孫自3月離開東京<ref name=白 />{{rp|162}}。由於此事未經同盟會內部商議，於是引起會內分裂。合計[[日本政府|日本政府]]與[[右派|右派]]人士共贈款2.3萬日元，相當於現今1千萬[[美金|美金]]<ref>{{cite web |url=https://books.google.com.tw/books?id=nS07BAAAQBAJ&lpg=PT270&ots=W4nSnfAgXn&dq=%E5%AD%AB%E6%96%87%20%E9%85%92&hl=zh-TW&pg=PT273#v=onepage&q=%E5%AD%AB%E6%96%87%20%E9%85%92&f=false |title=日本近現代史の「裏の主役」たち：北一輝、大川周明、頭山満、松井石根……「アジア主義者」の夢と挫折 |author=[[田原總一朗|田原總一朗]] |date=2014-06-13 |publisher=[[小學館|小學館]] }}</ref>。1907年3月至1908年1月<ref name=白 />{{rp|164}}，孫先抵越南[[河内|河内]]，继续策划革命起义，1907年5月，孫命[[余丑|余丑]]發動[[潮州黃岡起義|潮州黃岡起義]]，歷六日而敗，是為第三次起義。

6月，孫命[[鄧子瑜|鄧子瑜]]起義於[[惠州|惠州]][[惠州七女湖起義|七女湖]]，歷十餘日而敗，是為第四次起義。

1907年7月6日，[[光復會|光復會]][[徐錫麟|徐錫麟]]在安徽安慶起義，是為[[安慶起義|安慶起義]]，失敗殉難。

1907年9月1日，粤西靠近廣西與越北之市鎮[[防城起義|防城起義]]<ref name=白 />{{rp|193}}，起義軍由秘密會黨首領[[王和順|王和順]]起義於[[欽州|欽州]]，是為第五次起義。

10月，孫私自委託宮崎滔天在東京幫他做財務處理，洽商購買軍火，形式上把同盟會其他領袖排除在軍事機密外<ref name=白 />{{rp|162}}。12月<ref name=白 />{{rp|194}}，孫親赴[[廣西|廣西]]，指揮[[黃明堂|黃明堂]]發動[[鎮南關起義|鎮南關起義]]，再告失敗，是為第六次起義。

在法國外交部壓力下，孙於1908年3月被逐出[[越南|越南]]<ref name=白 />{{rp|195}}。南下南洋之後，在[[胡漢民|胡漢民]]、[[汪兆銘|汪兆銘]]等支持下，孫在[[新加坡|新加坡]][[晚晴园|晚晴园]]另成立同盟會總部，后搬迁到[[槟城|槟城]]。

1908年，同盟會[[陶成章|陶成章]]在新加坡《南洋總匯新報》發表《孫文罪狀》。指責孫文「謊騙營私」，在[[滙豐銀行|滙豐銀行]]有巨額存款，貪污兩萬[[銀元|銀元]]的革命經費，有「殘賊同志」、 「蒙蔽同志」、「敗壞全體名譽」等三種十二項「罪狀」，要求「開除孫文總理之名，發表罪狀，遍告海內外」。陶成章在1912年被[[陳其美|陳其美]]命令部下[[蔣介石|蔣介石]]暗殺<ref>{{cite book |title=《蔣主席》 |chapter=第八章〈槍殺陶成章〉 |author=[[鄧文儀|鄧文儀]] |publisher=勝利出版社 |year=1945 |location=上海 }}</ref>。1909年11月，章炳麟猛烈反擊偽《民報》，呼籲海外華人不要捐資給孫；陶成章譴責乾綱獨斷，再度要求同盟會開除孫<ref name=白 />{{rp|163}}。

1908年3月27日，[[黃興|黃興]]由[[法屬印度支那|法屬印度支那]]率革命軍進攻[[钦州市|欽州]]，發動[[欽廉上思之役|欽廉上思之役]]，是為第七次起義。4月，[[黃明堂|黃明堂]]發動[[雲南|雲南]][[河口|河口]]起義，是為第八次起義。

1908年10月，《民報》遭日本發出禁令<ref name=白 />{{rp|162}}。11月，孙中山在泰国期间访问美国驻泰国公使馆，面见美国公使汉密尔顿·金（Minister Hamilton King），要求美国政府给他颁发一本“美国本土护照”。1909年3月11日，美国国务院向汉密尔顿·金公使下发这样的决定：孙中山没有资格登记为美国公民或申请美国护照，理由如下：“美国公民享有权利，但也有责任和义务，他非但没有尽美国公民的义务，反而是全力投入中国的政治，尤有甚者，他是鼓动反政府的领袖，而本國政府與[[中國政府|中國政府]]有友好关系”。

1910年2月<ref name=白 />{{rp|198}}，[[倪映典|倪映典]]發動[[廣州新軍起義|廣州新軍起義]]，是為第九次起義。3月，孫抵檀香山，改組興中會為中國同盟會檀香山分會<ref name=言 />{{rp|56}}。

=== 第十次起義：黃花崗起義 ===
{{main|黃花崗起義}}
1911年4月28日，[[溫生才|溫生才]]暗殺廣州副[[都統|都統]][[孚琦|孚琦]]<ref name=白 />{{rp|202}}。4月27日（農曆三月二十九日），[[趙聲|趙聲]]、[[黃興|黃興]]等人領導廣州[[黃花崗起義|黃花崗起義]]<ref name=言 />{{rp|66}}。革命黨人一度占領[[兩廣總督|兩廣總督]]衙門，之後黃興負傷<ref name=白 />{{rp|202}}。由於起义军和政府军力量懸殊，黃花崗起義很快失敗，大批革命黨被捕犧牲<ref name=言 />{{rp|66}}。是役戰死者為被捕57人，被捕後就義者29人<ref name=言 />{{rp|566}}。共為86人<ref name=言 />{{rp|66}}。後收殮到72具烈士遺骸，合葬於黃花崗，是為第十次起義。

=== 武昌起義 ===
{{main|辛亥革命}}
[[File:WuchangUprising.jpg|thumb]]，又称“红楼”]]

1911年5月，清廷宣布「鐵路國有」，四川民眾紛起抗議，成立「保路同志會」，派代表到北京請願，還舉行罷市、罷工和罷課，[[四川總督|四川總督]][[趙爾豐|趙爾豐]]強硬鎮壓，引起激烈反抗，清廷急調湖北新軍到四川鎮壓，造成湖北守備空虛，革命黨人乘機於10月起義<ref name=中3 >{{cite book |title=《中國歷史3》（中學三年級適用） |year=2013 |authors=編審陳佳榮、呂振基，作者譚松壽、羅國潤、黃家樑、陳志華 |location=香港 |publisher=現代教育研究社 |isbn=978-962-11-1257-6 }}</ref>{{rp|103}}。1911年8月，「中部[[同盟會|同盟會]]」在上海成立<ref name=白 />{{rp|165}}。[[武昌起義|武昌起義]]前夕，湖北革命團體一同集結在[[文學社|文學社]]（湖北新軍革命團體）與[[共進會|共進會]]旗幟下，文學社多招募自駐紮武昌之新軍軍官和士兵，共進會則是同盟會異議組織地方分支<ref name=白 />{{rp|217}}。10月10日（農曆八月十九日）[[武昌起義|武昌起義]]，武昌駐軍亂起，找上守備統領[[黎元洪|黎元洪]]出任[[湖北軍政府|湖北軍政府]][[都督|都督]]<ref name=白 />{{rp|218}}。經過一夜血戰，終於在第二天早晨攻佔總督衙門，革命軍於是佔領武昌<ref name=中3 />{{rp|103}}。

辛亥革命發生之際孫中山正在美國進行演說募款之旅，美國時間十月九日下榻[[猶他州|猶他州]]小鎮[[奧格登_(猶他州)|奧格登]]的Hotel Marion，<ref>歷史資料：[http://www.orangenews.hk/culture/system/2016/09/05/010041013.shtml 【聊歷史】武昌起義爆發時，孫中山在美國「刷盤子」？]{{zh-hk}}</ref>。十月十日下榻丹佛（或譯典華）的布朗皇宮飯店(Brown Palace Hotel)，該飯店並存有孫文當時的入住記錄。<ref>歷史資料：[http://blog.udn.com/kinking/6198328 論武昌起義當天孫文在端盤子一說與史實不符]{{zh-tw}}</ref> 根據孫文自述：「十餘日前，在途中已接到黃克強在香港發來一電，因行李先運送至此地，而密電碼則置於其中，故途上無由譯之。是夕抵埠，乃由行李檢出密碼，而譯克強之電。其文曰：『居正從武昌到港，報告新軍必動，請速匯款應急』等語。時予在典華，思無法可得款，隨欲擬電覆之，令勿動。惟時已入夜，予終日在車中體倦神疲，思慮紛亂，乃止。欲於明朝睡醒精神清爽時，再詳思審度而後覆之。乃一睡至翌日午前十一時，起後覺饑，先至飯堂用膳，道經迴廊報館，便購一報攜入飯堂閱看。坐下一展報紙，則見電報一段曰：『武昌為革命黨佔領。』如是我心中躊躇未決之覆電，已為之冰釋矣。乃擬電致克強，申說覆電延遲之由，及予以後之行蹤。」<ref>歷史資料：[https://zh.wikisource.org/wiki/心理建設/第八章 建國方略 心理建設]{{zh-tw}}</ref>

起義後幾週之間，革命席捲華中、華南，各省紛紛宣布脫離中央<ref name=白 />{{rp|220}}。自武昌起義後，袁世凱被迫回籍養病，清廷命陸軍大臣[[蔭昌|蔭昌]]率軍南下平亂，但其所率之軍多為袁世凱所訓練，不受節制，清廷至此不得已再起用袁為[[內閣總理大臣|內閣總理大臣]]，節制諸軍<ref name=下 />{{rp|1}}。

10月14日，倫敦《每日紀事報》刊出〈孫逸仙講述他第一次起事反抗中國皇朝之故事〉：「我們的最高理想是，藉著《[[聖經|聖經]]》以及我們所理解的歐美教育作為手段，向我們痛苦的國人傳達一個信息：公正的法律會為他們帶來幸福，文明會為他們帶來歡慰。」<ref name=命 />{{rp|568}}孫10月20日抵達倫敦，先與康德黎夫婦敍舊<ref name=白 />{{rp|223}}。11月11日，孫抵倫敦，就武昌起義成功接受《濱海雜誌》採訪<ref name=言 />{{rp|57}}，至11月20日離開<ref name=言 />{{rp|58}}。據統計，自1894年到1911年之間發動革命起義事件，計有29次之多<ref>[[李筱峰|李筱峰]]：〈孤峰對話〉，2006-04-12</ref>。

== 建立民國 ==
1904年底，孫收到中國旅歐學生資助，轉往[[歐洲|歐洲]]活動，在[[倫敦|倫敦]]、[[巴黎|巴黎]]、[[布魯塞爾|布魯塞爾]]等地中國留學生中活動宣傳革命，並從留學生中籌得款項。受到清政府全力追緝影響，自1907年起孫便長期居留歐美各國。1909年5月，孫中山自新加坡赴馬賽轉巴黎，在巴黎住了一個多月<ref name=經緯 />{{rp|95}}。7月，孫抵布魯塞爾，馬君武前往迎接<ref name=言 />{{rp|54}}。8月，孫自[[布魯塞爾|布魯塞爾]]抵達英國倫敦<ref name=言 />{{rp|52}}。11月，孫自英國抵美國紐約<ref name=言 />{{rp|65}}。至1911年期間，孫大部分時間在旅途之上，曾環繞地球多次，在各國華僑、留學生中籌劃革命經費及外國政府支持，然而所得極為有限。

另一方面，同盟會及其週邊組織快速擴張規模，並於1910年1月成立同盟會美洲地區總會，期望能吸收更多海外華僑參與革命。1911年2月，孫在[[加拿大|加拿大]][[溫哥華|溫哥華]]設立「洪門籌餉局」為革命籌款<ref name=言 />{{rp|64}}。5月3日，孫在美國芝加哥召集會議，商討起義善後及再圖大舉等問題<ref name=言 />{{rp|67}}。7月，孫到美國委陌林埠<ref name=言 />{{rp|68}}。武昌起義時，孫中山人在美國進行募款活動而不在中國{{notetag|据[[唐德剛|唐德剛]]表示：據某私人記述，武昌起義成功後，當時接到電報的孫中山正在[[科羅拉多州|科羅拉多州]]典華城（今譯[[丹佛|丹佛]]）友人盧瑞連開的餐館中當「企檯」端盤子<ref>{{cite book |author=唐德剛 |title=《晚清七十年（5）袁世凱、孫文與辛亥革命》 |isbn=9573235153 }}</ref>。但事實上，武昌起義時，孫中山人不在科羅拉多州的丹佛，而是在猶他州鹽湖城附近的Ogden，並用飯店信箋寫信給友人談募款之事。而他與革命黨人舊金山致公總堂的黃雲蘇在1911年10月10日（中國的10月11日）才到丹佛，並下榻知名的{{link-en|布朗皇宮飯店|Brown Palace Hotel (Denver, Colorado)}}（Brown Palace Hotel），該飯店現在仍保留孫中山當時的入住簽名簿，並展示在會議室與飯店[[Facebook|Facebook]]專頁中。入住飯店後，孫中山才收到武昌起義的電報<ref>{{cite web |date=2011-10-14 |editor=關春英 |url=http://gb.cri.cn/1321/2011/10/14/5791s3403104.htm |title=透過文物說歷史：辛亥革命爆發時孫中山正在美國丹佛市為革命籌募經費 |location=北京 |publisher=[[中國國際廣播電台|中國國際廣播電台]] }}</ref>。}}。11月2日，孫从纽约出发。11月10日，在國外得悉「武昌已經為革命軍佔領」消息後，孫抵倫敦開展外交活動<ref name=言 />{{rp|70}}。孫在倫敦草擬《告世界書》<ref name=言 />{{rp|70}}。遊說英法美德銀行團代表終止貸款給清政府，并委托维加炮厂经理道森与英国外交大臣会谈，要求英国政府停止对清政府的一切借款，制止日本政府援助清政府，取消英属各地放逐令以便取道回国，得到英国政府同意。此后，四国银行代表团同意停止给清政府借款。11月23日，孙会晤[[东方汇理银行|东方汇理银行]]总裁西蒙（Stabislas Simon）<ref name=白/>{{rp|224}}。孫要求“重掌握海关及其税收”“取消厘金”，给中国革命政府贷款，但遭到拒绝。孙中山还到法国参议院拜访了参议员、前外长{{le|毕盛|Stephen Pichon}}和众议院的一些议员，希望法国政府承认中华民国，并与前总理[[乔治·克列孟梭|乔治·克列孟梭]]诚挚会晤，访问了法国在野人士。得到西方国家“皆极表同情于我”的印象。

=== 臨時大總統 ===
[[File:孫中山先生.jpg|thumb]]
1911年11月24日，孫从法国[[马赛|马赛]]启程，乘“狄凡哈号”邮轮，经停[[馬來亞|馬來亞]][[槟城|槟城]]、[[新加坡|新加坡]]、[[香港|香港]]。袁世凱率領大軍入北京組閣，並迫[[載澧|載澧]]退歸藩邸，繼又迫[[載濤|載濤]]交出禁衛軍，清室軍政大權盡握袁手中，袁立命[[馮國璋|馮國璋]]全力攻[[漢陽|漢陽]]<ref name=下冊 />{{rp|1}}。

12月14日，各省[[諮議局|諮議局]]代表雲集南京，共計44人，分別代表17個省分<ref name=白 />{{rp|220}}。孫於12月21日抵達香港稍事逗留<ref name=白 />{{rp|225}}。12月25日，孫到達[[上海|上海]]<ref name=言 />{{rp|72}}。孫與《大陸報》主筆談話稱：「革命不在金錢，而全在熱心。」<ref name=命 />{{rp|667}}孫向英國政府要求三事：「一、止絕清廷一切借款；二、制止日本援助清廷；三、取消各處英屬政府之放逐令，以便予取道回國。三事皆得英政府允許。」<ref name=命 />{{rp|647}}12月26日，孫與黃興等在[[上海孫中山行館舊址|寶昌路408號（今淮海中路650弄3號）行館]]召開中國同盟會最高幹部會議，商討組織臨時政府方案<ref name=言 />{{rp|73}}。在南北和谈进行之际，12月29日，同盟会连夜赴南京召集代表开会，提出成立政府，结果孫中山以16票當選為[[中華民國大總統|中華民國臨時大總統]]<ref name=經緯 />{{rp|96}}。1912年1月1日（辛亥年十一月十三日）晚上10時，在[[南京|南京]]總統府舉行中華民國臨時大總統就職典禮<ref name=言 />{{rp|72}}。孫宣誓就任<ref name=言 />{{rp|72}}。是年1月底，組成臨時[[參議院|參議院]]（[[國會|國會]]）。但当时的《[[纽约时报|纽约时报]]》报道对{{link-en|这次选举的代表|wikisource:The New York Times/1911/12/31/DR. SUN'S ELECTION IS NOT TAKEN SERIOUSLY}}的合法性提出疑问<ref>{{cite web|url=http://query.nytimes.com/mem/archive-free/pdf?res=990CE3DC123AE633A25752C3A9649D946096D6CF|title=DR. SUN'S ELECTION IS NOT TAKEN SERIOUSLY|publisher=《[[纽约时报|纽约时报]]》|date=1911-12-31|language=en}}</ref>。

南京臨時政府定用[[陽曆|陽曆]]，及用[[五色旗|五色旗]]為[[國旗|國旗]]<ref name=下冊 >{{cite book |authors=袁大衛、陸迪河編著 |title=《中國歷史問題精解》下冊 |location=香港 |publisher=現代教育研究社 |year=1993 |isbn=962-11-2592-8 }}</ref>{{rp|1}}。中華民國建國之初，北洋政府採用「五色旗」作為國旗，旗面按順序為紅、黃、藍、白、黑的五色橫條，分別表示漢、滿、蒙、回、藏五族共和，所選用的五色為五個民族傳統上所喜愛的顏色。

當時满清政权并未终结。1月3日，袁世凯指使手下[[冯国璋|冯国璋]]、[[段祺瑞|段祺瑞]]等48名北洋将领联名通电，反对共和制和南京临时政府，拥护君主立宪制，并以军事实力向临时政府施压。受到此事影响，南京方面内部主战情绪空前高涨，临时政府于是接受孙中山的用兵主张。1月11日，南京临时政府决定组建北伐军海陆军，孙宣布自任总指挥，黄兴任北伐军陆军参谋长，[[林述庆|林述庆]]任北伐军津浦路总司令，[[孙岳|孙岳]]为北伐军淮扬线总司令，[[柏文蔚|柏文蔚]]为第一军军长，[[蓝天蔚|蓝天蔚]]为第二军军长，并成立海军舰队。在总统府设北伐大本营，实际由黄兴主持。大本营的机构直到3月下旬取消。首战，北伐军在津浦路一线告捷，粤军于[[蚌埠|蚌埠]]、[[固镇|固镇]]、[[宿州|宿州]]一线大败清军，[[张勋|张勋]]军望风而逃，徐州不战而下。同时，[[安徽|安徽]]、[[河南|河南]]前线及奋勇军也传来捷报。1月16日，海军先锋进抵[[烟台|烟台]]，沪军占领[[登州|登州]]。北京为之震动。面对北伐军的胜利，袁世凯感到恐慌。英国、日本、美国、德国派军舰在长江示威，并阻止民军在[[秦皇岛|秦皇岛]]登陆。日本向内蒙古、东北增兵。由于军费匮乏，粮草、饷械奇缺。时值严冬，北伐军御寒衣物不足，冻死者众多，军队士气大受影响。而六路大军只在名义上受临时政府指挥，实际上各自为战，进军受阻。与此同时，临时政府陷入严重的财政危机，举步维艰。尽管孙仍然坚持主张北伐，但是临时政府内部上层如[[张謇|张謇]]、[[汤寿潜|汤寿潜]]、[[程德全|程德全]]等开始出现主张停战议和的声音，其他人认为可以把大总统之位让与袁世凯，换取清帝退位。袁世凯上台后用《临时约法》对其限制，袁必不敢胡作非为。[[汪精卫|汪精卫]]认为孙中山不愿议和是因为贪恋总统之位，黄兴扬言要剖腹自杀，虽然该议和主张仍遭到了地方军政府及群众团体的激烈反对。孙中山最终依然选择妥协<ref name=何，2016 />{{rp|213}}。曾訪問過孫之記者提到「他真誠、慎思的外表」，以及「安靜、沉穩的舉止」<ref name=白 />{{rp|230}}。

2月12日，清廷[[宣統帝退位詔書|下詔]][[宣統帝|宣統帝]]遜位，詔書文本由南京方面草擬，再傳給袁世凱<ref name=白 />{{rp|234}}。期间，袁世凯曾遭[[炸弹|炸弹]]暗杀，但行动失败，1月18日，孙以提出《五条要约》方式向袁世凯摊牌。孫不願延長內戰，雙方遂派代表在上海議和，孫提出：一、清帝退位；二、袁絕對贊同共和；三、孫自清帝退位後辭職；四、由參議院選舉袁為臨時大總統；五、袁被選後，當宣誓遵守《臨時約法》，袁乃嗾使[[段祺瑞|段祺瑞]]等電清廷實行共和<ref name=下冊 />{{rp|1-2}}。经过反复修改又于1月22日以公诸报端的方式将南北双方幕后谈判全部曝光，令袁世凯极其尴尬和不满，也让议和全权代表伍廷芳尽失颜面。2月13日，袁世凯通电共和，根据南方独立各省及部分革命党人自始至终的意愿，孙提出辞职咨文，准备将总统一职讓位给袁世凯。2月15日，孫中山祭[[明孝陵|明孝陵]]時向文武官員講話<ref name=言 />{{rp|88}}。同日孫宣布辭去總統職位<ref name=白 />{{rp|235}}。

=== 臨時約法 ===
1912年3月6日，南京臨時參議院決議，允許[[袁世凱|袁世凱]]在北京就職。3月8日，以孫中山為首的南京臨時政府，由臨時參議院通過頒佈具有「憲法」性質的《[[中華民國臨時約法|中華民國臨時約法]]》，取代《[[中華民國臨時政府組織大綱|中華民國臨時政府組織大綱]]》，3月11日公佈實施。《臨時約法》確定人民之權利義務及臨時政府參議院之組織與職權，並規定《約法》施行後10個月內成立國會<ref name=下冊 />{{rp|1}}。行政權由總統與內閣分享，且閣員須由國會通過任命<ref name=白 />{{rp|237}}。內閣由國務總理及各部總長組成，法院為司法機關<ref name=用 >{{cite book |editor=現代教育研究社編輯委員會 |title=《會考版中國歷史（中學五年級適用）》 |location=香港 |publisher=現代教育研究社 |year=1993 |author=王爾敏校訂 |isbn=962-11-2588-X }}</ref>{{rp|4}}。3月10日袁世凱在北京[[石大人胡同|石大人胡同]][[前清|前清]][[外務部|外務部]]公署就任臨時大總統，3月25日，袁世凱任命之內閣總理[[唐紹儀|唐紹儀]]抵南京組織新內閣<ref name=言 />{{rp|86}}。參議院為防袁野心，特設「責任內閣制」，以唐紹儀為國務總理，並設國務會議以行使職權，但袁不能容，唐被迫辭職，以[[陸徵祥|陸徵祥]]代之<ref name=下冊 />{{rp|3}}。3月29日，孫出席臨時政府各部總次長、衛戌總督、各軍師旅司令官舉行餞別宴會<ref name=言 />{{rp|82}}。1914年5月，臨時約法遭罷廢，新憲法除去所有行政權約束，並無限期延長總統任期<ref name=白 />{{rp|273}}。

=== 解任臨時大總統 ===
1912年4月1日，孙在南京正式解除臨時大總統職務，並把總統寶座讓給袁世凱<ref name=命 />{{rp|649}}。孫中山辭職後不再過問政治<ref name=用 />{{rp|5}}，决心“尽瘁社会上事业”，开始着手社会革命。4月3日，孙中山离开南京，到全国各地参观并宣传民生主义。4日，在接受《[[文汇报_(上海)|文汇报]]》记者提问时，孙中山说：“政治上革命今已如愿而尝矣；后当竭力从事于社会上革命，社会革命比政治革命愈属重大，且非兵力所能援助，必须以和平手段从事”。6日，参观[[江南制造局|江南制造局]]。4月9日，孫應鄂軍都督[[黎元洪|黎元洪]]之邀抵武漢<ref name=言 />{{rp|85}}，与廖仲恺、胡汉民等由上海经南京前往，武昌市因此万人空巷。孙中山在武汉与辛亥革命首义战士和湖北同盟会会员见面，10日参观楚望台，并出席湖北军政界代表300人参加的欢迎会，并发表演讲，阐述官吏与军人的职责，并解释自己辞职的原因，强调“功不必自我成，名不为自我居”。4月20日，孙中山抵达[[福州|福州]]，参观马尾船政局。25日抵达[[广州|广州]]，在广东女子师范第二校发表演说，指出“中国四万万同胞皆应受教育，此师范学校所宜急办者也，而女子师范尤为重要。”孫在其廣東之行中說：「吾國乃社會主義之民國，欲行社會主義。吾等皆是真誠的社會主義領袖。」<ref name=白 />{{rp|244}}5月15日，前往黄花岗七十二烈士墓地致祭文。5月27日，孫抵翠亨村<ref name=言 />{{rp|81}}，与夫人和大哥团聚<ref name=何，2016 />{{rp|227-228}}。参观途中，孙中山愈加认为中国唯一的出路是发展实业，而建筑铁路是“发展中国财源的第一要策”。6月，[[唐紹儀|唐紹儀]][[內閣|內閣]]垮台<ref name=白 />{{rp|243}}。6月中旬，孙中山回到上海与黄兴商量，草拟了一份修筑铁路的计划。袁世凱邀請南方領袖到北京與他會晤，孫於8月24日抵達首都<ref name=白 />{{rp|243}}。9月25日，雙方達成共識，公布一則聯合聲明，提出建立中央集權之政府，去軍事化，開放外國資本，促成政黨和解，穩定政局<ref name=白 />{{rp|243}}。

=== 國民黨理事長 ===
1912年8月，宋教仁以同盟會為基礎，聯合[[統一共和黨|統一共和黨]]、[[國民共進會|國民共進會]]及國民公黨，改組成[[國民黨_(1912–1913)|國民黨]]，推舉孫中山任理事長，黃興、[[宋教仁|宋教仁]]等為理事，實際黨務由宋教仁主持<ref name=用 />{{rp|14}}。孫擔任總理，黃興為協理<ref name=白 />{{rp|241}}。8月25日，孫在北京舉行的國民黨成立大會中被選為理事長，孫以“決不願居政界，惟願作自由國民”，即委宋教仁為代理理事長。1913年11月，國民黨殘存勢力被解散<ref name=白 />{{rp|273}}。

=== 中国铁路督辦 ===
1912年9月10日，袁世凱「特授孫文以籌辦全國鐵路全國鐵路全權將擬築之路先與各國商人商議借款招股事宜」<ref>《任命狀》第360號，2012-09-10</ref>。袁世凱任命孫為鐵路督辦，每月可支領經費3萬元<ref name=白 />{{rp|250}}。孙自己出任[[中国铁路总公司|中国铁路总公司]]总理，设总部于上海<ref>{{cite web |editor=李寒 |url=http://www.cnr.cn/junshi/jszl/wjls/200908/t20090803_505419121.html |title=中国军史：1925年孙中山逝世 |publisher=中国广播网}}</ref>，希望透過開放外資，盡快完成建設中國鐵路，自认为有相當詳細之規劃<ref>《國父全集》第二冊，臺北：近代中國，1989年</ref>{{rp|257-258、800、816、825-826}}<ref>徐高阮：《中山先生的全面利用外資政策》，台北：臺灣商務印書館，1963年5月</ref><ref>王爾敏：《孫中山與中華民國》，台北：國家圖書館出版品，2011年4月</ref>{{rp|137-151}}。1912年10月，孫委托[[王寵惠|王寵惠]]與英商簽定廣州至[[重慶|重慶]]及[[蘭州|蘭州]]的鐵路線，袁橫加阻撓，不予批准<ref name=綱 />{{rp|416-419}}。

=== 討袁護法 ===
{{Main|二次革命|護法運動|中日盟約}}

1913年2月11日，孫赴日本考察，藉以籌措外資興建鐵路<ref name=言 />{{rp|93}}。2月13日抵達日本後，孫受到熱情接待<ref name=白 />{{rp|253}}。2月至6月，袁世凱提供巨額，誘使國民黨員另組政團，有國事維持會、政友會、超然議員社等從國民黨分裂出來<ref name=用 />{{rp|6}}。3月4日晚，孫在東京設宴答謝日本各界人士，牧野外相等內閣閣員應邀出席<ref name=言 />{{rp|98}}。3月11日，孫與[[宋嘉樹|宋嘉樹]]、[[戴季陶|戴季陶]]等參觀大阪《[[每日新聞|每日新聞]]》社<ref name=言 />{{rp|91}}。3月14日，孫與戴季陶、[[馬君武|馬君武]]、宋嘉樹、[[何天炯|何天炯]]等出席[[神戶|神戶]]國民黨交通部正支部長[[吳錦堂|吳錦堂]]在寓所所設午餐會<ref name=言 />{{rp|94}}。

3月20日，宋教仁在上海滬寧車站遇刺<ref name=命 />{{rp|666}}，兩天後重傷不治<ref name=用 />{{rp|14}}。3月22日，孫在長崎獲悉宋教仁在上海被刺消息<ref name=言 />{{rp|99}}。暗殺宋教仁的人，從查獲之宋案兇手與北京來往電文中<ref name=用 />{{rp|14}}，有观点认为主使行刺的是袁世凯<ref name=近代史 >{{cite book |author=李守孔 |title=《中國現代史》 |location=台北 |publisher=[[三民書局|三民書局]] |year=1973 }}</ref>{{rp|499}}，而直接布置暗殺的則是國務總理趙秉鈞<ref name=海 />{{rp|2936}}<ref name=表 >{{cite book |authors=[[陳布雷|陳-{布}-雷]]等編著 |title=《蔣介石先生年表》 |location=台北 |publisher=[[傳記文學_(雜誌)|傳記文學]]出版社 |date=1978-06-01 }}</ref>{{rp|6}}。也有观点認為凶手是[[陈其美|陈其美]]<ref>{{cite book zh |author=张耀杰 |title=《谁谋杀了宋教仁》 |publisher=团结出版社 |isbn=9787512606722 |url=http://hb.qq.com/a/20120628/000311.htm |pages=327 |year=2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20140924042313/http://hb.qq.com/a/20120628/000311.htm |archivedate=2014-09-24 }}</ref>，还有观点认为孙中山的嫌疑最重<ref>{{cite web |title=楚望台：宋教仁先生是被谁刺杀的？ |url=http://www.21ccom.net/articles/lsjd/article_201001203104.html |date=2010-01-20 |publisher=[[共识网|共识网]] |archiveurl=https://web.archive.org/web/20160304231245/http://www.21ccom.net/articles/lsjd/article_201001203104.html |archivedate=2016-03-04 |deadurl=yes }}</ref>，以宋教仁被暗杀和善后大借款为借口<ref>中央政府與國民黨之問題，《[[大公报|大公报]]》，1913-06-01</ref>。

1913年3月23日，孫參觀長崎[[三菱|三菱]]造船所後，下午即乘船返國<ref name=言 />{{rp|99}}。孫自從3月25日返回上海後，便極力主張公開與北京政府決裂，並以武力對抗袁世凱<ref name=白 />{{rp|255}}。同月，孫與黃興等在上海橫濱正金銀行商討集資討袁<ref name=言 />{{rp|96}}。孫中山力主興師討袁，但黃興認為兵力不足，主張循法律途徑解決<ref name=用 />{{rp|6}}。袁竟不經國會通過，擅與英、法、德、俄、日五國銀行團簽訂2,500萬鎊之善後大借款，名義用於平亂<ref name=下 />{{rp|4}}。提倡五权宪法的孙却反对法律程序解决，力主南方各省起兵反袁，是為[[二次革命|二次革命]]，开民国武力解决政治争端先河<ref name="二">{{cite web|language=zh-tw|publisher=孫中山學術研究資訊網|title=二次革命與討袁護法|url=http://sun.yatsen.gov.tw/content.php?cid=S01_01_04_02}}</ref>。袁罷免反對善後大借款之三[[都督|都督]][[李烈鈞|李烈鈞]]、[[胡漢民|胡漢民]]、[[柏文蔚|柏文蔚]]，致引起二次革命<ref name=下 />{{rp|3}}。「二次革命」因勢單力薄，在兩個月內被各個擊破<ref name=用 />{{rp|6}}。7月8日，李烈鈞佔領湖口要隘，宣布獨立，李烈鈞為江西討袁軍總司令，黃興7月15日任江蘇討袁軍總司令，柏文蔚任安徽討袁軍總司令，陳其美任上海討袁軍總司令<ref name=下冊 />{{rp|5}}。8月2日，孫由上海乘[[德國|德國]]船舶潛逃至[[福州|福州]]。反袁失敗後，孫喬裝化名逃亡，經台灣[[基隆|基隆]]乘坐日本貨船「信濃丸」號，1913年8月7日到達日本[[門司區|門司區]]<ref name=命 />{{rp|671}}。孫動身前往日本，於8月8日上岸<ref name=白 />{{rp|259}}。9月初，袁世凱已弭平所有叛亂<ref name=白 />{{rp|259}}。10月15日，袁以北京總檢查名義通緝孙及二次革命首要<ref>{{cite web |language=zh-tw |publisher=孫中山學術研究資訊網 |title=国父生平事迹简表·民國2年 |url=http://sun.yatsen.gov.tw/content.php?cid=s01_01_01_01 }}</ref>。在[[日本內閣|日本內閣]]默許下，孫經門司、[[神戶|神戶]]，最後從橫濱進入東京{{notetag|孫中山致日本有司小池張造私函原件內曾提出以二十一條相匹敵之條件，以換取聯日、討袁。該函件由日本學者松末英紀教授從日本戰前檔案中影印，並在1986年於孫中山誕辰一百二十週年學術討論會中發表}}<ref>[[哥倫比亞大學|哥倫比亞大學]]教授C. Martin Wilbur著作SUN Yat-Sen: Frustrated Patriot（New York: Columbia University Press: 1976）內亦有引英日文史料證明</ref><ref name=海 />{{rp|2936}}。

=== 組建中華革命黨 ===
中華革命黨於1913年底開始籌建<ref name=白 />{{rp|274}}。1914年1月，孫准許陳其美東北之行<ref name=白 />{{rp|283}}。7月，孫在日本召集國民黨員<ref name=用 />{{rp|33}}，改組國民黨為[[中華革命黨|中華革命黨]]<ref name=表 />{{rp|6}}。並兩次發表討袁宣言<ref name=二 />。「中華革命黨」宣佈「以掃除專制政治，建設完全民國為目的」，「以實行民權、民生兩主義為宗旨」，規定入黨者都要按指印、立誓約，絕對服從孫中山，黨員人數最多時只500人左右<ref name=用 />{{rp|33}}。每個人都要發誓，「為救中國危亡，拯生民困苦，願犧牲一己之身命自由權利。」<ref name=白 />{{rp|274}}每個黨員之地位取決於入黨時間：凡在革命開始前之前入黨者，即為「首義黨員」，日後為「元勳公民」，將被賦予「特權」；凡在革命開始之後、革命政府成立之前入黨者，即為「協助黨員」，日後成為「有功公民」，享有選舉權與被選舉權；革命勝利之後始入黨者，即為「普通黨員」，日後為「先進公民」，只有選舉權，沒有被選舉權<ref name=白 />{{rp|275}}。孫提案遭到革命黨人強力抵抗，特別是黃興<ref name=白 />{{rp|276}}。部分同時流亡日本的國民黨員對此反對，原同盟會中重要人物如[[黃興|黃興]]、[[李烈鈞|李烈鈞]]、[[柏文蔚|柏文蔚]]、[[譚人鳳|譚人鳳]]、[[陈炯明|陈炯明]]等俱未有加入。7月8日，中華革命黨成立大會在東京舉行，孫選任總理<ref name=言 />{{rp|101}}。{{cn|7月}}，[[蒋介石|蒋介石]]赴[[滿洲|東北]]策动讨袁军事時，孫中山对蒋表示，“日本人如果不将东北和台湾交还我们，并保护[[朝鲜|朝鲜]]的独立，我们国民革命运动是不能停止的，你要将这个意思告诉日本将领”。后在东北时，蒋将孙的话表达予「一位日本联队长」：“听了我的话大为不满，面红耳赤而去。第二天就请我离开东北。”<ref>{{cite journal |author=李云汉 |title=〈蒋中正先生与台湾〉 |journal=《近代中国》1995年10月 |issue=第109期 }}</ref>{{rp|163}}11月29日，黑龍會領袖內田良平上呈日本政府一份關於解決中國問題之備忘錄，他在文中提議從袁世凱手中救出中國，然後與中國結盟，將中國置於日本保護下；他推薦支持兩個有助推翻袁世凱之團體：以[[肅親王|肅親王]]為核心之北方君主主義者，和南方革命黨人孫<ref name=白 />{{rp|280}}。

中華革命黨在各地組織活動，由於缺乏群眾基礎而屢遭挫折；1919年10月，孫中山又把中華革命黨改組成中國國民黨<ref name=用 />{{rp|33}}。

[[File:Sun-Soong_wedding_photo.jpg|thumb]]于[[日本|日本]][[东京|东京]]结婚]]

孫與盧慕貞分居多年，協議離婚<ref name=言 />{{rp|102}}。1915年10月25日，孫與[[宋慶齡|宋慶齡]]在日本東京結婚<ref name=言 />{{rp|102}}。

=== 護國之役 ===
{{main|護國之役}}
袁在二次革命失敗後，乃組織「公民團」，脅迫議員選己為正式總統，解散國會，設立「約法會議」，廢止《臨時約法》<ref name=下冊 />{{rp|3}}。1915年，袁世凱慫恿籌組[[籌安會|籌安會]]，鼓吹恢復帝制<ref name=白 />{{rp|273}}。1月18日，日本向北京政府提出要求，事關山東、南滿和內蒙古東部權利讓渡，和日本軍事和政治顧問任用<ref name=白 />{{rp|281}}。3月14日，孫致書[[日本外務省|日本外務省]]大臣，向日本政府承諾優於[[二十一條|二十一條]]要求之權利讓渡<ref name=白 />{{rp|282}}。11月，陳其美刺殺上海鎮守使[[鄭汝成|鄭汝成]]，12月試圖破壞黃浦江之中國軍艦<ref name=白 />{{rp|284}}。袁進行「國體投票」，要求各區、各省之國民代表，投票贊成改行君主政體，又指使參議議員上書擁戴袁為皇帝<ref name=下冊 />{{rp|6}}。12月12日，袁世凱宣佈接受帝位，改國號為[[中华帝国|中华帝国]]，以明年為「洪憲」元年；前雲南都督[[蔡鍔|蔡鍔]]潛離北京，繞道回雲南，於12月底聯同李烈鈞、雲南都督[[唐繼堯|唐繼堯]]宣佈雲南獨立，組成「護國軍」，向四川、貴州、廣西三路出兵<ref name=用 />{{rp|9}}。12月25日，雲南護國軍訴請其他各省響應討袁<ref name=白 />{{rp|286}}。1916年3月22日，袁世凱被迫取消帝制<ref name=言 />{{rp|108}}。袁部屬如段祺瑞、馮國璋、[[陳宦|陳宦]]、[[湯薌銘|湯薌銘]]、[[陳樹藩|陳樹藩]]等相繼反袁<ref name=下冊 />{{rp|6}}。4月9日，孫、宋慶齡等在日本友人田中昂寓所舉行小型聲討袁世凱集會（又稱「帝制取銷一笑會」）<ref name=言 />{{rp|108}}。4月27日，孫為討袁大計，離日返國<ref name=言 />{{rp|111}}。5月1日，孫抵達[[上海公共租界|上海公共租界]]<ref name=言 />{{rp|111}}。5月9日，孫发表《[[s:討袁宣言|討袁宣言]]》，號召推翻袁世凯。5月18日，袁世凱買兇刺殺陳其美<ref name=白 />{{rp|286}}。6月6日，袁世凱逝世<ref name=言 />{{rp|112}}。7月17日，孫在上海張園舉行茶話會，與各界人士研討袁死後之形勢和建設方針<ref name=言 />{{rp|112}}。10月31日，黃興逝世<ref name=言 />{{rp|120}}。11月24日，孫與前來吊唁之友人在上海哈同花園<ref name=言 />{{rp|120}}。

=== 張勳復辟 ===
總統府內務總長[[孫洪伊|孫洪伊]]為[[黎元洪|黎元洪]]信任，國務院秘書長[[徐樹錚|徐樹錚]]則為段祺瑞之門人，二人各為其主，遇事衝突，造成府、院之爭<ref name=下冊 />{{rp|8}}。1917年5月11日，段祺瑞派出「請願者」包圍國會<ref name=白 />{{rp|289}}。6月，徐州[[張勳|張勳]]率軍5,000人北上，強迫黎元洪解散國會<ref name=用 />{{rp|11}}。7月1日，張勳在北京重新擁立清廢帝[[溥儀|溥儀]]，歷時12天，史稱[[張勳復辟|張勳復辟]]<ref name=言 />{{rp|121}}。皖系軍閥[[段祺瑞|段祺瑞]]乘機糾集舊部，組織「討逆軍」攻入北京，張軍敗退，溥儀再度宣佈退位<ref name=用 />{{rp|11}}。段祺瑞起兵驅逐張勳後，控制北京政府，拒絕恢復《中華民國臨時約法》和[[中华民国初年国会|國會]]<ref name=言 />{{rp|121}}。黎元洪去職，由馮國璋繼任總統，段祺瑞復任國務總理，以「再造共和」功臣自命<ref name=用 />{{rp|11}}。

=== 第一次護法運動 ===
{{main|護法之役}}
[[File:Hubei_Military_Government.jpg|thumb]]
段祺瑞違反臨時約法，通過對德宣戰案，舉借日債，企圖擴張勢力；孫以段違背憲法，乃自上海赴廣東，倡導護法，各省亦多響應<ref name=下冊 />{{rp|7-8}}。1917年7月6日，孫等南下掀起護法運動<ref name=言 />{{rp|121}}。孫中山即以維護臨時約法為號召，聯合西南軍閥，在廣州組織軍政府<ref name=用 />{{rp|11}}。時任[[廣東省|廣東省]]省長[[朱慶瀾|朱慶瀾]]邀孙赴[[粤|粤]]，並派[[程璧光|程璧光]]之獨立[[海軍|海軍]]載孫中山、[[唐紹儀|唐紹儀]]、汪兆銘、[[伍廷芳|伍廷芳]]等人和部分国会议员南下[[廣州|廣州]]。7月17日，孫抵達廣州<ref name=言 />{{rp|121}}。7月21日，海軍總司令指揮麾下15艘軍艦駛離上海，開赴廣州，當時廣州已聚集130位國會代表，約占總數三分之一<ref name=白 />{{rp|290}}。9月1日，國會非常會議選舉孫中山為[[护法军政府|中華民國軍政府]]海陸軍[[大元帥|大元帥]]<ref name=言 />{{rp|122}}。9月2日，[[陸榮廷|陸榮廷]]即通電國會議員和廣州士紳、商人，稱：「既然總統仍在位，則毋須另立政府。大元帥這令人難解的名稱尤其有問題。我們是簡樸之人，對權力沒有任何野心，只想盡本分，而且我們不會走錯誤的一步。因此我們不願意配合當前動作。」<ref name=白 />{{rp|293}}9月10日，孫正式就任<ref name=言 />{{rp|122}}。9月13日，廣州軍政府對德宣戰<ref name=白 />{{rp|291}}。孫在廣州組織軍政府，謀求[[北伐|北伐]]。蔣介石撰上對北洋軍作戰計劃及滇粵兩軍對於閩浙單獨作戰之計劃<ref name=表 />{{rp|7}}。此計劃受到孫讚許。

孫號召國會議員南下，召開國會非常會議，展開[[護法運動|護法運動]]（亦稱“三次革命”），誓師[[北伐|北伐]]<ref name=二 />。由於南方軍閥本無誠意護法，故不久即醞釀議和<ref name=用 />{{rp|11}}。但廣州護法政府逐漸由旧桂、滇系軍人控制，孫實力有限，甚至「政令不出[[士敏土|士敏土]]廠（[[孙中山大元帅府纪念馆|大元帥府]]）」。孫曾嘗試發動兵變而未果。1918年5月，軍政府改組，實權掌握在軍閥手中<ref name=用 />{{rp|11}}。軍政府以七總裁取代大元帥，領導南方與西南各省<ref name=白 />{{rp|294}}。

[[File:Sun_Yat-Sen's_House.jpg|thumb]]
孫中山見事不可為，惟有黯然離去<ref name=用 />{{rp|11}}。孫抵達上海後，與先到之宋慶齡住進華僑捐贈之莫利哀路29號寓所（今香山路7號、上海孫中山故居）<ref name=言 />{{rp|141}}。1918年至1919年，孙中山闭门谢客，撰写文稿，决心改变现状。宋庆龄随时帮助他收集有价值的书和报纸杂志，协助查阅资料，著书立说。作为秘书，宋庆龄还负责协助起草电函稿件，翻译英文，并加紧学习德文、俄文，准备联系苏联方面<ref name=何，2016 />{{rp|303}}。

== 繼續革命 ==
{{main|中國國民黨|陳炯明|六一六事變}}
=== 成立中國國民黨 ===
1919年10月10日<ref name=白 />{{rp|297}}，孫改組中華革命黨，擴大吸收黨員，成立[[中國國民黨|中國國民黨]]<ref name=表 />{{rp|8}}<ref name="改">{{cite web |language=zh-tw |publisher=孫中山學術研究資訊網 |title=改組政黨與北伐 |url=http://sun.yatsen.gov.tw/content.php?cid=S01_01_04_03 }}</ref>。1920年，[[陳炯明|陳炯明]]奉孫中山之命率領粵軍從福建返廣東，打敗盤踞在廣東之桂系軍閥，重建廣東根據地<ref name=用 />{{rp|33}}。同年秋，孫試圖與湖南自治派之[[趙恆惕|趙恆惕]]談判結盟，並透過湖南，爭取和四川、雲南結盟<ref name=白 />{{rp|314}}。11月9日頒布之中國國民黨總章規定，中國國民黨以實施三民主義、創立五權憲法為目的<ref name=白 />{{rp|297}}。11月28日，孫設法返回廣州<ref name=白 />{{rp|312}}。

=== 第二次護法運動 ===
1921年初，舊國會在廣州重開，因不足法定人數，故稱國會非常會議，推舉孫中山為中華民國政府非常大總統<ref name=用 />{{rp|33}}。孫設法號召舊國會225名議員，於4月7日在廣州集會，推選總統<ref name=白 />{{rp|312}}。4月2日，廣州非常國會取消軍政府，改總裁-{制}-為總統-{制}-。4月29日廣州政府外交部長接受《北華捷報》專訪，即明白表示：「任何人要和我方達成協議，必須同意由他（孫）擔任臨時大總統，以待召開國會、正常選舉。我們沒有其他條件。」<ref name=白 />{{rp|313-314}}5月，孫中山宣佈就職，轄區僅有廣東，實權在[[陳炯明|陳炯明]]之手<ref name=用 />{{rp|33}}。第二次護法運動開始。7月24日，孫與宋慶齡在廣州出席「出征軍人慰勞會」<ref name=言 />{{rp|137}}。10月，廣州非常國會批准孫北伐計劃<ref name=白 />{{rp|314}}。

1922年4月，孫不滿廣東省政府不支持北伐，決定罷黜陳炯明廣東省長和粵軍總司令職位<ref name=白 />{{rp|317}}。5月，[[直奉战争|直奉战争]][[奉系|奉系]]遭[[直系|直系]]擊敗<ref name=白 />{{rp|314}}。6月2日，[[徐世昌|徐世昌]]在壓力下退讓，宣布辭職<ref name=白 />{{rp|314}}。直系恢复约法和国会。在粤国会议员纷纷北上。孫中山就任非常大總統，積極策劃北伐，陳炯明志在割據，多方阻撓<ref name=用 />{{rp|33}}。陳炯明主张停战，实行[[聯省自治|聯省自治]]，而孙主张继续北伐，最終產生激烈衝突。1922年夏，孫自桂回粵，免陳炯明職<ref name=表 />{{rp|8}}。6月16日，[[葉舉|葉舉]]炮打孫寓所<ref name=白 />{{rp|318}}。陳炯明兵變，[[宋慶齡|宋慶齡]]不願與孫同行，認為「目標太大，容易被叛軍發現」，覺得個人不致有何危險，對孫說：「中國可以沒有我，不可以沒有你。我請求你先走吧！」<ref name=命 />{{rp|611-612}}陳炯明發動[[六一六事变|六一六事变]]，圍攻總統府悬赏捕杀大总统，叛军将总统府焚毁，府中所藏孙心爱的大量藏书和未经发表的著述亦被焚毁，孫与卫兵登上永豐艦與之作戰，后被困黄埔，叛军不断炮击，孙及部属给养困难，孤立无援，处境极为艰难。6月29日，蒋介石在料理后事之后同汪精卫坐外国轮船冒死赶来救援，日夜陪伴左右并出谋划策，孙大为感动振奋。在永丰舰经过[[车歪炮台|车歪炮台]]前往[[白鹅潭|白鹅潭]]时，两岸叛军集中炮火进行攻击。孙在甲板上镇定指挥，吩咐士兵节约子弹并瞄准射击。由于炮火密集，士兵卫士多人倒毙，经其他人苦劝，孙中山才回到舰艇舱内<ref name=何，2016 />{{rp|300-301}}。英國只肯派一艘炮艇供孫調遣，孫因而得以在8月9日離開珠江，安抵香港，再轉往上海<ref name=白 />{{rp|319}}。8月離開廣州往上海<ref name=用 />{{rp|33}}。9月至12月，孫在上海寓所3次召集在上海各省中國國民黨負責人[[廖仲愷|廖仲愷]]、[[汪精衛|汪精衛]]、[[胡漢民|胡漢民]]、[[張繼_(中華民國)|張繼]]、[[楊庶堪|楊庶堪]]討論改組中國國民黨問題<ref name=言 />{{rp|148}}。孫和[[吳佩孚|吳佩孚]]談判，盼望能回到北京；孫同時與[[張作霖|張作霖]]秘密交涉，從張作霖得到若干經費<ref name=白 />{{rp|320}}。

=== 聯俄容共 ===
{{main|聯俄容共}}
在1912年〈中國的民主和民粹主義〉中，[[列寧|列寧]]讚揚孫「真摯的民主精神」和「對群眾的溫暖同情」，不過批評孫之小資產階級，以及「意圖在中國阻止資本主義」之天真野心<ref name=白 />{{rp|322}}。1913年4月〈中國的政黨鬥爭〉，列寧觀點更尖銳：他將國民黨之積弱不振歸因於「從未能吸引足夠廣大的中國群眾加入革命」，又指出國民黨領袖之「弱點」在於「他是個夢想家且優柔寡斷」<ref name=白 />{{rp|322-323}}。

1919年底，波布夫（Popoff）[[上校|上校]]和孫接觸，形容孫是「一個老派軍閥，認為捨武力即無法拯救中國」<ref name=白 />{{rp|324}}。1920年7月共產國際第二次代表大會上，馬林熱切支持列寧觀點，相信[[統一戰線|統一戰線]]政策，並渴望見到在中國實現<ref name=白 />{{rp|323}}。[[羅易|羅易]]不認同統一戰線構想，稱孫是「一個權謀家……不實際……絕對的反動派」<ref name=白 />{{rp|324}}。1920年秋，孫時常在上海寓所接見[[共產國際|共產國際]]東方部代表[[吳廷康|吳廷康]]<ref name=白 />{{rp|323}}。第二次護法運動失敗後，因沒有西方及日本金援支持，拒絕原因是不看好孫文政治資本，孫中山開始考慮與[[蘇聯|蘇聯]]支持的[[中國共產黨|中國共產黨]]合作之可能<ref name="聯俄容共">{{cite web |language=zh |title=孫中山“聯俄容共” |url=http://www.rocidea.com/roc-892.aspx |author=三鑒齋 }}</ref>。1921年12月，孫在桂林接見過共產國際官方代表[[馬林_(共產國際)|馬林]]<ref name=白 />{{rp|317}}。孫對俄國革命及列寧本人表示讚賞，很想多瞭解俄國新政權之成就，特別是對新經濟政策相當好奇<ref name=白 />{{rp|323}}。1922年1月為反制[[華盛頓會議|華盛頓會議]]，在[[莫斯科|莫斯科]]召開之遠東勞動者大會上，共產國際首領[[季諾維也夫|季諾維也夫]]大肆抨擊中國國民黨之資產階級作風，以及孫尋求美國援助之執拗<ref name=白 />{{rp|324}}。1922年8月，蘇聯政府秘密通過馬林與孫中山聯繫<ref name=用 />{{rp|33}}；馬林和[[李大釗|李大釗]]與孫中山多次會晤，討論振興中國國民黨之問題<ref name=用 />{{rp|29}}。11月至2月，共產國際第四次代表大會通過《綜論東方問題》，強調中國共產黨先天體質不良，主張暫與中國國民黨及若干地方領導人妥協，並提議組成統一戰線<ref name=白 />{{rp|324-325}}。

1923年1月12日，共產國際執委會通過決議，認定中國國民黨為中國共產黨唯一之夥伴，並指示中國共產黨在中國國民黨內部運作，同時要保有組織和活動自主<ref name=白 />{{rp|325}}。1月17日至1月26日<ref name=白 />{{rp|325}}，蘇聯政府全權代表[[越飛|越飛]]到上海與孫中山會談，到月底發表《[[孫文越飛宣言|孫文越飛聯合宣言]]》，標誌確立孫中山「聯俄」政策，主要內容有：<br>1.蘇聯願援助中國完成統一大業；<br>2.蘇聯願拋棄沙俄對華之不平等條約，另行開始中蘇交涉；<br>3.中東鐵路問題由中蘇協商，暫仍維持現狀；<br>4.蘇聯無意使外蒙脫離中國，蘇軍也不必立時撤退<ref name=用 />{{rp|33}}。孫以為共產組織，甚至蘇維埃制度，事實均不能引用於中國；因中國並無使此項共產制度或蘇維埃制度可以成功之情況也；此項見解，越飛完全同感；且以為中國最要最急之問題，乃在民國的統一之成功，與完全國家的獨立之獲得<ref name=中3 />{{rp|144}}。孫接受與莫斯科同盟，但有兩大保留：一、中國不會轉向共產主義，俄國將支持中國（孫）之國家統一及獨立奮鬥；二、越飛重申俄國放棄沙皇時代依條約在華所得之一切特權<ref name=白 />{{rp|325-326}}。1月26日，孫通電各大軍閥，重申其自願裁軍之計劃，另外再提議委請列強出任第三方代表團；代表團負責監督裁軍，也負責籌募、運作用於國家經濟及政治改造之國際基金<ref name=白 />{{rp|320-321}}。

孫中山引入蘇聯金援改造中國國民黨、建立黨軍的政策<ref name=聯俄容共 />。雙方發表聯合宣言，聲明共產組織及蘇維埃制度不能引用於中國，但蘇聯願意援助中國完成統一及取得完全之獨立<ref name=用 />{{rp|29}}。孫越宣言俄方承認中國對外蒙古[[主權|主權]]，承諾不在中國進行共產革命，認為共產主義制度不適合中國國情<ref name=聯俄容共 />。孫接受臨時協議，把中東鐵路委由蘇聯管理，並准許蘇聯軍隊占領外蒙古<ref name=白 />{{rp|326}}。與[[共產國際|共產國際]]合作後，蘇俄方面給予孫大量[[軍火|軍火]]和[[銀錢|銀錢]]援助，並派出軍事顧問幫助孫創立黨軍北伐。孫中山派遣參謀長蔣中正率領代表團訪蘇，考察政治和軍事，又聘用蘇聯代表鮑羅廷為政治顧問，協助改組中國國民黨<ref name=用 />{{rp|29-30}}。1923年8月16日，孫中山派遣蔣中正訪問蘇聯，向革命成功推翻俄羅斯臨時政府的蘇聯學習建立革命軍隊的經驗。9月，蘇共中央挑選[[米哈伊尔·马尔科维奇·鲍罗廷|鮑羅廷]]出任孫之政治顧問，負責執行與中國國民黨之統一戰線策略<ref name=白 />{{rp|334}}。蘇聯新特使鮑羅廷於10月6日抵達廣州<ref name=白 />{{rp|333}}。10月9日至10月17日之間，鮑羅廷和中國共產黨五度私下會談，會中他表明改組政策即是滲透政策：「在報上，我講的是國民黨，但對我們而言，我講其實是最終共產黨影響力的增加……絕不能忘記，我們實際在做的工作（投入穩定國民黨的），就是為了穩定共產黨，這個目標應該要永久記住。」<ref name=白 />{{rp|337}}10月13日，孫批准鮑羅廷提出之改組方案：訂定新黨章、在廣州和上海組建核心小組，再到全國各地成立地方黨部，最後召開黨之全國代表大會通過變革，並推選一組中央執行委員會<ref name=白 />{{rp|337}}。10月18日，孫以[[陸海軍大元帥大本營|陸海軍大元帥大本營]]名義，委任鮑羅廷為中國國民黨組織訓練員<ref name=言 />{{rp|160}}。10月25日，孫指派9位同志組成臨時中央執行委員會，其中一人具有共產黨員身分<ref name=白 />{{rp|337}}。10月底，滇、桂系部隊作戰失利向廣州撤退，陳炯明軍一路尾追到廣州城外<ref name=白 />{{rp|338}}。鮑羅廷在11月13日臨時中執會痛批中國國民黨領導人未能動員群眾，稱：「以農民為主廣東民眾對前線戰爭依然持消極態度！」<ref name=白 />{{rp|338}}鮑羅廷組織兩支志願隊參加城防工作：趕赴前線者約有500人，大部分由共產黨員訓練<ref name=白 />{{rp|339}}。11月18日夜，軍事情勢逆轉，陳炯明部倉惶撤退<ref name=白 />{{rp|339}}。11月25日，孫在〈在廣州大本營國民黨員的演說〉稱：「吾黨與他（鮑羅廷）等所主張，皆是三民主義，主義既是相同，但吾黨尚未有良好方法，所以仍遲遲不能成功。他等氣魄厚，學問深，故能想出良好方法。吾等想革命成功，一定要學他，吾等在革命未成功之前，既是人自為戰，今後應該結合團體而戰，為有紀律的奮鬥；因為要學他的方法，所以我請鮑君做黨的訓員，使之訓練吾黨同志……望各同志犧牲自己成見，誠意去學他的方法。」<ref>{{cite book |title=《孫中山全集》第八卷 |publisher=中華書局 |year=1981 }}</ref>{{rp|437-438}}11月29日，[[鄧澤如|鄧澤如]]等11人聯名上書孫，攻擊中國共產黨，反對改組中國國民黨，他們指責中國共產黨幫助孫確定反帝反軍閥之綱領，使中國國民黨遭受「國際之仇怨」，「在國內斷絕實力派之協助」；孫不為所動，並堅持說：「你們如果反對[[國共合作|國共合作]]，我就解散國民黨，我自己參加共產黨去。」<ref name=中3 />{{rp|144}}12月29日，孫接受[[列寧|列寧]]和共產國際協助重建[[大元帥府|大元帥府]]，共產國際派鮑羅廷到廣州作孫[[顧問|顧問]]，以[[蘇聯共產黨|蘇聯共產黨]]模式重組中國國民黨<ref name=聯俄容共 />。

對於中國共產黨黨員，孫願意容納於中國國民黨內，一來是為了爭取蘇聯援助，二來是為集中力量，壯大國民革命聲勢<ref name=用 />{{rp|29}}。1924年1月20日至1月30日，[[中國國民黨第一次全國代表大會|中國國民黨第一次全國代表大會]]在廣州召開<ref name=白 />{{rp|340}}。孫召開全代會，在1月20日上午之開幕演說上，他強調黨改組之必要，以及將黨變成革命、建國工具之目標<ref name=白 />{{rp|343}}。孫力排中國國民黨內反共勢力干擾，堅持“[[聯俄容共|聯俄容共]]”，但是国民党上层自大会一开始就对孙中山的联共政策意见不一，争论激烈。意见分三派，一派主张联共，一派主张溶共，还有一派主张分共。在国民党改组过程中及改组以后，不断有人表示反对，有当面发难、联名上书、集会抗议、会场上抗议等。[[冯自由|冯自由]]、刘成禹、[[胡毅生|胡毅生]]、[[李石曾|李石曾]]、[[戴季陶|戴季陶]]等公开表示反对国民党改组。孙中山于是表示：“你们怕共产党，不赞成改组，可以退出国民党呀！你们不赞成改组，那就解散国民党，我个人可以加入共产党。”甚至孙中山的儿子[[孙科|孙科]]因为发表了反苏反共的言论，不支持改组，被孙中山亲自把名字从第一届国民党中央委员会名单中劃掉<ref name=何，2016 />{{rp|333-334}}<ref name=聯俄容共 />。大會主席團由胡漢民、汪精衛、李大釗等5人組成，蘇聯顧問鮑羅廷也出席大會；大會確立聯俄、容共扶助農工三大政策，並決議中國共產黨黨員可以個人身份加入中國國民黨，但須服膺三民主義及遵守黨綱黨紀；與會代表百多人，包括20多名中國共產黨黨員<ref name=用 />{{rp|30}}。大會後鮑羅廷離開廣州好幾個月<ref name=白 />{{rp|347}}。從春天起，中國國民黨中執會農民部和農民運動講習所負責制訂、推行中國國民黨之農村政策：農民部當家是秘書[[彭湃|彭湃]]，由中共農民委員會和鮑羅廷在幕後操控<ref name=白 />{{rp|353}}。在[[廖仲愷|廖仲愷]]倡儀下，5月1日大型慶祝會於廣州籌開，孫在會中演講，再度呼籲大家為民族解放奮鬥，而把階級鬥爭之主題擺到次要地位<ref name=白 />{{rp|355}}。

中共黨員威脅拒絕統一戰線即是與蘇聯絕裂，但孫不肯在鮑羅廷回來前解決爭端<ref name=白 />{{rp|349}}。幾經急電催促，鮑羅廷在6月20日回到廣州<ref name=白 />{{rp|349}}。6月25日，鮑羅廷與溫和派反對領袖會談時，承認共產黨的確構成「黨（國民黨）內有黨」，但他暗示是取得蘇援代價：「國民黨已經死了、埋了……新成員，如具有團體精神共產黨員，透過為老同志灌輸競爭活力，昭示著新國民黨誕生……國民黨的中央組織仍然鬆散，命令也得不到全體黨員遵行。在這種情況下，不能期待共產黨放棄他們自己的組織。」<ref name=白 />{{rp|349-350}}上海、北京同志函電湧至，抗議共產黨干預、操縱，尤其是他們力促通過中蘇條約之宣傳；[[張繼_(中華民國)|張繼]]和[[汪精衛|汪精衛]]在廣州公開發表反共演說，並發起要求驅逐危險盟友之請願連署<ref name=白 />{{rp|350}}。7月15日，[[沙面島|沙面]]全體華工罷工<ref name=白 />{{rp|356}}。孫統一戰線之工具性概念成形，是毛澤東審慎務實主義之前兆；毛澤東曾痛批教條「比大便更不如」，只肯保留馬列主義有用之部分<ref name=白 />{{rp|357}}。8月3日，孫在[[中山大學|廣州大學]]演講民生主義時批評[[馬克思主義|馬克思主義]]，駁斥馬克思主義之[[剩餘價值|剩餘價值]]說，也摒棄[[階級鬥爭|階級鬥爭]]之重要角色；和鮑羅廷對話後一星期，孫再次演講民生主義，推翻先前批評，再次主張民生主義即是社會主義<ref name=白 />{{rp|350}}。8月21日，孫〈在廣州農民運動講習所第一屆畢業禮演說〉，對講習所學員發出警告，無法在中國發動階級鬥爭，反之應該要用平和方法，以合作為訴求：「所以大家此時去宣傳，一定要很謹慎……你們更要聯絡全體的農民來同政府合作，慢慢商量來解決農民同地主的方法，讓農民可以得利益，地主不受損失。」<ref>{{cite book |title=《孫中山全集》第十卷 |publisher=中華書局 |year=1981 }}</ref>{{rp|558}}。同年秋，農民協會和地方民團間之衝突已演變成激戰<ref name=白 />{{rp|354}}。8月23日，中執會通過決議，承認中共黨員在中國國民黨內之特別責任，解除張繼、[[謝持|謝持]]控訴他們派系活動之罪名<ref name=白 />{{rp|350}}。同年，蘇聯支持[[外蒙古|外蒙古]]成立[[蒙古人民共和国|蒙古人民共和国]]，未從外蒙古撤軍。孫中山認為共產主義與[[民生主義|民生主義]]毫無衝突，民生主義已包括共產主義在內<ref name=用 />{{rp|34}}。1925年2月，鮑羅廷決定派廣州政府軍增援農民協會對抗地方民團<ref name=白 />{{rp|354}}。

1922年底，孫中山策動駐廣西之滇軍、桂軍及駐西江之部分粵軍趕走陳炯明，奪回廣州<ref name=用 />{{rp|33}}。1923年1月16日，拥护孙中山的[[滇系軍閥|滇軍]][[杨希闵|杨希闵]]部、[[桂系|桂军]][[刘震寰|刘震寰]]部、和倒戈粤军合组西路讨贼军击败陳炯明部，陳炯明退守[[东江|东江]]<ref>{{cite web |url=http://sun.yatsen.gov.tw/hero_detail.php?id=60 |title=〈人物誌-陳炯明〉 |publisher=孫中山學術研究資訊網 }}</ref>。2月，孫中山回廣東，再任[[大元帅|大元帅]]，3月正式成立大元帥府<ref name=用 />{{rp|33}}。3月2日，孫以大元帥名義，特任楊庶堪為大本營秘書長，[[朱培德|朱培德]]為大本營參軍長兼大本營護軍司令<ref name=言 />{{rp|165}}。當時陳炯明佔據東江一帶及潮州、汕頭等地，與孫成相持之局<ref name=用 />{{rp|33}}。4月20日，孫督師北伐討賊，與參謀長[[蔣介石|蔣介石]]從廣州火車北上<ref name=言 />{{rp|168}}。8月14日，為紀念廣州蒙難一週年，孫與宋慶齡同該艦官兵重登永豐艦<ref name=言 />{{rp|161}}。11月9日，[[鄧澤如|鄧澤如]]等人聯名上書孫<ref name=言 />{{rp|170}}。

1924年1月1日，孫與宋慶齡在廣州大元帥府舉行慶祝中華民國成立12週年暨授勳典禮<ref name=言 />{{rp|189}}。3月18日，孫署名公告〈關於實行黨紀的命令〉，提醒黨員遵守黨紀：黨員「應由所在地之最高黨部直接組織，並指揮之」、「不得假借名義，自由行動」<ref>{{cite book |title=《孫中山全集》第九卷 |publisher=中華書局 |year=1981 }}</ref>{{rp|618}}。孫在鮑羅廷和黨內領導人陪同下，於6月16日主持開學典禮<ref name=白 />{{rp|360}}。6月，[[黃埔軍校|黃埔軍校]]正式開學，第一期有學員600多人，孫中山兼任軍校總理，任命蔣中正為校長，廖仲愷為黨代表；蘇聯派出多名軍官作顧問，並安排中國共產黨黨員入軍校就讀或任教<ref name=用 />{{rp|34}}。6月16日，孫在黃埔軍校開學禮演說稱：「中國革命遲遲不能成功，就是因為沒有真正革命武裝隊伍。為了早日完成革命的使命，我們才下決心改組國民黨，建立黨軍，使革命與民眾力量相結合，這就是創辦黃埔陸軍軍官學校主旨。」<ref name=中3 />{{rp|142}}軍校參照蘇聯紅軍制度實行「軍事與政治並重」之方針，學生除學習軍事外，還要學習革命理論，以提高政治覺悟<ref name=用 />{{rp|31}}。

9月3日，[[江浙战争|江浙战争]]爆发。孫試圖重新發動北伐，成為統一戰線之絆腳石、革命動員一致行動之異聲<ref name=白 />{{rp|347}}。9月18日，孫發表《中國國民黨北伐宣言》：「原夫[[反革命|反革命]]之發生，實繼承專制時代之思想，對內犧牲民眾之利益，對外犧牲國家利益，以保持其過去時代之地位。觀於袁世凱之稱帝，張勳之復辟，[[馮國璋|馮國璋]]、[[徐世昌|徐世昌]]之毁法，[[曹錕|曹錕]]、[[吳佩孚|吳佩孚]]之竊位盜國，十三年來連續不絕，可知其分子雖有新陳代謝，而其傳統思想始終如一。此等反革命之惡勢力，以北京為巢窟，而流毒於各省。……此戰之目的不僅在推倒軍閥，尤在推倒軍閥所賴以生存的帝國主義。蓋必如是。然後反革命之根株乃得永絕，中國乃能脫離[[次殖民地|次殖民地]]之地位，以造成[[自由|自由]][[獨立|獨立]]之[[國家|國家]]也。」<ref name=中3 />{{rp|147}}中國國民黨內部出現批評、抗議，孫科辭去廣州市長，廖仲愷放棄廣東省省長，蔣介石也拒絕讓黃埔師生參加北伐，只派一連兵到韶關保衛孫<ref name=白 />{{rp|364}}。9月4日，孙在广州组建北伐討賊軍，以[[譚延愷|譚延愷]]为[[总司令|总司令]]，联合[[卢永祥|卢永祥]]、[[张作霖|张作霖]]、段祺瑞，“共抗[[直系軍閥|直系]]”，准备北伐。9月[[第二次直奉戰爭|第二次直奉戰爭]]爆發，孙中山指示[[于右任|于右任]]、[[焦易堂|焦易堂]]、[[徐謙|徐謙]]等秘密与[[冯玉祥|冯玉祥]]、[[胡景翼|胡景翼]]、[[孙岳|孙岳]]联系，对其做争取工作。孙中山送给冯玉祥六千本《三民主义》、一千本《[[建国方略|建国方略]]》和《建国大纲》。冯玉祥将这些书发给其官兵阅读学习。10月下旬直系第三軍軍長[[馮玉祥|馮玉祥]]，率軍自長城前線回師[[北京政變|佔領北京]]，軟禁總統[[曹錕|曹錕]]，馮玉祥部和胡景翼、孫岳部組成[[國民軍|國民軍]]<ref name=用 />{{rp|43}}。10月9日，孫寫信給蔣：「革命委員會當要馬上成立，以對付種種非常之事。漢民、精衛不加入，未嘗不可。蓋今日革命，非學俄國不可。而漢民已失此信仰，當然不應加入，於事乃為有濟；必加入，反多妨礙，而兩失其用，此固不容客氣也。精衛本亦非俄派之革命，不加入亦可。我黨今後之革命，非以俄為師，斷無成就。而漢民、精衛恐皆不能降心相從。且二人性質俱長於調和現狀，不長於徹底解決。現在之不生不死局面，有此二人當易於維持，若另開新局，非彼之長。故只好各用所長，則兩有裨益。若混合做之，則必兩無所成。所以現在局面由漢民、精衛維持調護之。若至維持不住，一旦至於崩潰，當出快刀斬亂麻，成敗有所不計。今之革命委員會，則為籌備以出此種手段，此固非漢民、精衛之所宜也。故當分途以做事，不宜拖泥帶水以敷衍也。此覆。再：明日果有罷市之事，則必當火速將黃埔所有械彈運韶，再圖辦法。如無罷市，則先運我貨前來，商械當必照所定條件分交各戶可也。若兄煩於保管，可運至兵工廠或河南行營暫存俱可。即候毅安
孫文十月九日」<ref name=命 />{{rp|693-694}}。10月10日，广州发生[[陈廉伯|商团事变]]。孫於10月12日決定鎮壓商團，同時又決定不返回廣州<ref name=命 />{{rp|687}}。10月14日，孫從韶關調兵遣將：「令警衛軍、工團軍、農民自衛軍、飛機隊、甲車隊、兵工廠衛隊、陸軍講武學校、滇軍幹部學校學生，統歸蔣司令指揮」<ref name=命 />{{rp|688}}。10月15日，孙回师广州击溃商团武装。10月23日，[[馮玉祥|馮玉祥]]發動[[北京政变|北京政变]]<ref name=白 />{{rp|413}}。10月30日，孫因而捨棄在韶關建立之總部，返回廣州<ref name=白 />{{rp|413}}。馮玉祥、張作霖、段祺瑞分別致電孫，請他北上共商國是<ref name=用 />{{rp|43}}。11月3日，孫〈在黃埔軍官學校的告別演說〉，批評黃埔軍校學生抗議者缺乏紀律，「殊不知革命的目的」<ref>{{cite book |title=《孫中山全集》第十一卷 |publisher=中華書局 |year=1981 }}</ref>{{rp|270}}。11月4日，為孫舉行餞行大會<ref name=白 />{{rp|415}}。11月，孫應邀北上討論國是，提出「召開國民會議和廢除[[不平等條約|不平等條約]]」兩大號召，同[[帝國主義|帝國主義]]和[[北洋軍閥|北洋軍閥]]段祺瑞、張作霖等作鬥爭<ref name=海 />{{rp|2936}}。孫發表北上宣言，主張召開國民會議，以謀求中國之和平統一，對外廢除一切不平等條約，對內實行民治<ref name=用 />{{rp|43}}。11月17日，孫自北京抵達上海，與宋慶齡在上海寓所接見各界歡迎代表<ref name=言 />{{rp|191}}。段祺瑞被推為「臨時執政」，宣稱「外崇國信」，承認一切不平條約繼續有效<ref name=用 />{{rp|43}}。11月24日，孫與宋慶齡抵達日本[[神戶|神戶]]<ref name=言 />{{rp|195}}。針對孫中山主張召開國民會議，段祺瑞宣佈召開善後會議，只限各省首領、各省區長官、有功勳及特殊資望者，不包括人民代表<ref name=用 />{{rp|43}}。11月25日，孫在神戶與前來拜訪之[[頭山滿|頭山滿]]長談<ref name=言 />{{rp|194}}。11月28日，在神戶演講《大亞洲主義》<ref name=言 />{{rp|217}}。

12月4日，孫抵天津，受到盛大歡迎，因勞累並受風寒，身體感覺不適<ref name=言 />{{rp|217-218}}。年底周恩來出任黃埔軍校政治部主任，葉劍英、惲代英等也擔任要職<ref name=用 />{{rp|34}}。12月31日，孫扶病至北京，發表《入京宣言》，及對歡迎民眾之書面談話<ref name=言 />{{rp|218}}。然後孫在[[北京協和醫學院|北京協和醫學院]]附屬醫院接受治療<ref name=白 />{{rp|421}}。1925年1月，孫堅持速開國民會議以解決國是，段祺瑞卻主張召開善後會議<ref name=言 />{{rp|218}}。

== 逝世 ==
1925年1月1日，孫中山抵達[[北京|北京]]後病情發作，入[[北京协和医院|協和醫院]]住院治療。1月26日，協和醫院進行手術，證實孫罹患肝癌末期<ref name=白/>{{rp|421}}。[[張學良|張學良]]說孫中山病重時，在天津見過一回，他對[[唐德剛|唐德剛]]說：「他的病，你知道怎麼來的？他就是因為見[[張作霖|我父親]]以後病的，冷的。他本來有病，見我父親的天很冷，大概在屋子裡很熱，感冒了；他病發了。」孫中山對張學良說：「現在國家的責任就在你們這些年輕人身上，而且，尤其是你們東北的年輕人，介乎這日、俄兩大之間，你們很難應付。」<ref>{{cite book |authors=張學良口述、唐德剛著 |title=《張學良口述歷史》 |location=台北 |publisher=遠流出版 |date=2009-03-01 |edition=初版 }}</ref>{{rp|317}}

1925年2月，[[善後會議|善後會議]]在北京召開，國共兩黨均激烈反對，4月善後會議結束，一無所成<ref name=用/>{{rp|43}}。2月18日孫中山自協和醫院出院，移居[[鐵獅子胡同|鐵獅子胡同]][[行轅|行轅]]<ref name=言/>{{rp|218}}。在內憂外患中，孫扶病入京，苦無成果，終於病情惡化<ref name=用/>{{rp|35}}。孫改以[[中醫|中醫]]治療，先後經由著名中醫[[陸仲安|陸仲安]]、[[唐堯欽|唐堯欽]]、周樹芬三人共同診視。1925年2月26日，停止服用[[中藥|中藥]]，改回用西醫治療。此期間[[湯爾和|湯爾和]]與[[汪精衛|汪精衛]]對中西醫治療爆發爭論，湯爾和在《[[晨報_(北京)|晨報]]》上發表〈關於孫中山病狀的疑問〉，對中醫治療加以指責。孫在[[顧維鈞|顧維鈞]]家裡簽署《國事遺囑》<ref name=白/>{{rp|421}}。3月11日，孫中山在[[遺囑|遺囑]]上簽名，稍休後説幾句話勉勵各同志：「……生死本不足念，但所抱定的主義，還沒有完全實現，總是遺憾。希望你們努力奮鬥，使國民會議早日開成，達成實行三民主義和五權憲法的目的，我在九泉之下也安心了。」<ref name=劉/>{{rp|251}}。孫中山呼吸愈來愈困難，精神疲倦，在重複說著：「和平」、「奮鬥」、「救中國」；醫生因病人説話太多，就請孫入睡<ref name=劉/>{{rp|251-252}}。3月11日晚上6時半，孫中山手腳變冷，已不能再出聲說話，醫生説：已沒有脈搏，請大家注意逝世時間；第二天上午3時，孫又醒來一次<ref name=劉/>{{rp|252}}。3月12日9時30分<ref name=言/>{{rp|218}}，孫中山心臟停止跳動，呼吸停止<ref name=劉/>{{rp|252}}，在北京逝世，享年59歲<ref name=海/>{{rp|2936}}。

孫臨終時表達最後願望：用[[基督教|基督教]]儀式送終；結果宋慶齡和孫科不顧中國國民黨人之強烈反對，而堅持在北京協和醫院用基督教儀式為孫舉行私人喪禮之後，才讓中國國民黨中央在北京舉行公開喪禮<ref name=三/>{{rp|169}}。孫中山的確實死因，外界普遍認為是肝癌，近年根據[[協和醫院|協和醫院]]存檔的病理檢查報告，認為孫原发[[胆囊癌|胆囊癌]]，癌細胞转移到[[肝臟|肝臟]]並救治無效<ref>[http://big5.chinanews.com:89/shipin/2011/06-16/news40012.html 揭秘孫中山最後的日子：死於膽囊癌] {{webarchive|url=https://web.archive.org/web/20111124184313/http://big5.chinanews.com:89/shipin/2011/06-16/news40012.html |date=2011-11-24 }}</ref>。3月13日，《[[南華早報|南華早報]]》在社論中說：「他（孫）屬於人類極為少數的幾個人，他們為了一個理念，敢於挑戰幾乎一切。當[[最後審判|最後審判]]來臨的時候，他們是屬於[[天國|天國]]的人。」<ref name=命/>{{rp|495}}3月15日大殮，3月19日移靈於[[北京中山公園|北京中央公園]]社稷壇，供各界瞻仰致敬<ref name=言/>{{rp|218}}。沿路護送民眾有12萬人，3月20日開弔，民眾前去弔祭更有幾十萬人<ref name=劉>{{cite book |title=《國父傳》 |author=劉中和編著 |location=台北 |publisher=益群書店 |year=1984 |isbn=9575520858 |series=中華偉人傳記叢書 }}</ref>{{rp|253}}。4月2日出殯，由中央公園出發到北京西郊三十里之香山碧雲寺；還有30萬民眾把馬路填塞滿滿，直到北京[[西直門|西直門]]；還有2萬多人步行直跟到香山[[碧雲寺|碧雲寺]]<ref name=劉/>{{rp|253}}。孫奉厝於北京[[西山_(北京)|西山]]碧雲寺內石塔中<ref name=言/>{{rp|218}}。

=== 遺囑 ===
{{main|總理遺囑}}
[[File:国父遗嘱.png|thumb]]

1925年3月11日，孫簽字於汪精衛執筆的《[[總理遺囑|政治遺囑]]》和《家事遺囑》上<ref name=言/>{{rp|218}}。據《家事遺囑》原件謂：「余因盡瘁國事，不治家產。其所遺之書籍、衣物、住宅等，一切均付吾妻宋慶齡，以為紀念。-{余}-之兒女已長成，能自立，望各自愛，以繼余志。此囑。孫文三月十一日補簽。中華民國十四年二月二十四日。筆記者[[汪精衛|汪精衛]]。證明者[[宋子文|宋子文]] [[鄒魯|鄒魯]] [[邵元沖|邵元沖]] [[孔祥熙|孔祥熙]] [[吳敬恒|吳敬恒]] [[何香凝|何香凝]] [[孫科|孫科]] [[戴季陶|戴季陶]] [[戴恩賽|戴恩賽]]。」在貪腐成風之年代，孫光明磊落，著實令孫之競爭者和軍閥相形見短；雖然在其他方面，孫之行動和政治風格常常無異他們<ref name=白/>{{rp|423}}。孫在彌留之際，由汪精衛代筆<ref>{{cite web |url=http://book.sina.com.cn/wangjingweiyuchen/excerpt/sz/zt/2008-11-26/1702247809.shtml |title=執筆人汪精衛為何篡改孫中山先生遺囑？（圖）_讀書頻道_新浪網 }}</ref>簽署了兩份遺囑。孫弥留之际提到国事的遺言是“革命尚未成功，同志仍须努力！”<ref name="遺囑">{{cite web |url=http://news.china.com/zh_cn/history/all/11025807/20050404/12214592.html |title=“革命尚未成功”——孙中山遗嘱全解密 |date=2005-04-04 |accessdate=2007-10-13 |publisher=中华网 |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20071116054238/http://news.china.com/zh_cn/history/all/11025807/20050404/12214592.html |archivedate=2007-11-16 }}</ref>。

5月16日，中國國民黨一屆三中全會決議接受先生遺囑<ref name=言/>{{rp|218}}。

=== 遺體 ===
孙中山生前曾说，自己死后，希望能像苏联领袖[[列宁|列宁]]那样，保留遗体供民众瞻仰。他病故的第二天，国民党便与[[苏联政府|苏联政府]]联系，希望帮助为孙中山作遗体防腐处理和訂製安葬列宁时使用的那种[[水晶棺|水晶棺]]。但那时苏联人认为，列宁是不可比拟的领袖人物，其他人不可能再享受这种殊荣。因此未能提供防腐技术，但同意提供水晶棺。當時苏联政府赠送的水晶棺，现保存在北京碧云寺的中山纪念堂，供人瞻仰。孙中山病逝于北京铁狮子胡同行馆。苏联用火车运来的水晶棺到北京后，发现孙中山遗体因初期防腐技术处理不好，遗体皮肤开始变黑，已不能长期保存生前的状态，使用水晶棺已无意义。同时专家认为苏联的水晶棺仅适用于气候寒冷干燥的北方，在炎热潮湿的中国南方，遗体最多只能保存20年，所以最终未能使用水晶棺。孙中山的遗体最早安置在北京碧云寺内。四年后，南京的陵墓完工。

1929年6月1日，孫中山遗体从北京迁葬南京[[中山陵|中山陵]]<ref>{{cite web |url=http://news.ifeng.com/history/zhongguojindaishi/detail_2012_05/30/14926393_0.shtml?_from_ralated |title=孙中山遗体改殓易棺之谜：苏联水晶棺最终未使用 |publisher=凤凰网 |date=2012-05-30 }}</ref>。中山陵的墓室室内是一座半球形，中央是直径约4.3米的大理石坑，深1.6米，[[大理石|大理石]]坑四周围着一圈精致的白色大理石栏杆。石坑的正中棺椁上安放着就是孙中山身着[[中山装|中山装]]的[[汉白玉|汉白玉]]卧像，卧像与其真人比例大小一样，由[[捷克|捷克]]雕塑家高畦雕刻。

== 思想與信仰 ==
{{三民主義}}
{{main|孫中山與宗教}}
[[File:TianXiaWeiGong.png|thumb]]

孫文的思想大部分是他將中國道統和[[西洋|西洋]][[歐美|歐美]]各家學說綜合整理而來，但是也有少部分見解是其所獨創。

孫文：“中國有一個[[道統|道統]]，[[堯|堯]]、[[舜|舜]]、[[禹|禹]]、[[汤_(商)|湯]]、[[周文王|文]]、[[周武王|武]]、[[周公旦|周公]]、[[孔子|孔子]]相繼不絕，我的思想基礎，就是這個[[道統|道統]]，我的革命，就是繼承這個正統思想，來發揚光大！”<ref name="藍培綱">{{cite web | language = zh-tw |publisher=大紀元 |title=植樹節憶孫中山先生 |url=http://www.epochtimes.com/b5/8/3/11/n2040478p.htm |author=藍培綱 |accessdate = 2010-11-18 }}</ref>

[[民族主義|民族]]、民權、民生的「[[三民主義|三民主義]]」<ref name="三民主義">{{cite web |language = zh-tw |publisher =維基文庫 |title=三民主義 |url=http://zh.wikisource.org/wiki/%E4%B8%89%E6%B0%91%E4%B8%BB%E7%BE%A9 |author=孫中山 }}</ref>。

「人民有權，政府有能」的「權能區分」；廣興福利，大有為式的「萬能政府」。

人民之權又稱為政權，有[[選舉|選舉]]、[[罢免选举|罷免]]、[[創制|創制]]、[[公民投票|複決]]四種。

政府之能又稱為治權，[[立法部門|立法]]、[[行政部门|行政]]、[[司法部門|司法]]、[[考試院|考試]]、[[監察院|監察]]五權分立，是為「[[五權憲法|五權憲法]]」。

人民行使政權的基本訓練與條件：[[民權初步|民權初步]]，即議事規則。

中央與地方的「均權制度」，以縣為單位的「[[自治地方|地方自治]]」。

在經濟思想方面，提倡社會互助理論，曾說「人類進化之原則與物種進化之原則不同，物種以競爭為原則，人類則以互助為原則。」似是受到[[查爾斯·達爾文|查爾斯·達爾文]]進化論與俄國[[克魯泡特金|克魯泡特金]]的無政府主義思想的影響。

政治現代化建設方面，主張漸進主義的[[軍政|軍政]]、[[國民政府建國大綱|訓政]]、[[憲政|憲政]]三階段說。軍政時期優先消滅軍閥土匪，應實行軍管。訓政時期優先基礎建設與[[民權初步|民權初步]]訓練，應實行一黨執政。憲政實行之條件是全國半數以上縣市具有[[選舉|選舉]][[罢免选举|罷免]]地方首長之條件，公民具有發動創制複決之條件，則選舉召開[[國民大會|國民大會]]制訂憲法，還政於民，實行多黨競爭的現代政治制度。

[[甲午战争|甲午战争]]以前，孙之政治主张类似洋务派，提出“人尽其才，地尽其力，物尽其用，货畅其流”之革新。失败后，认识到扶清抗洋行不通。只有改革制度，才能成功救中国，于是走上革命道路。

以[[王道|王道]]為基礎的區域合作和政治架構：[[大亞洲主義|大亞洲主義]]，以及向國際投資開放的[[實業計畫|實業計畫]]。

人生哲學：「不做大官，要做大事。」

在哲學上，提出「知難行易」說，批判「知之非艱，行之惟艱」的保守思想<ref name=海 />{{rp|2936}}。

[[大同_(思想)|天下為公]]。

孫中山讚揚[[霍元甲|霍元甲]]「欲使國強，非人人習武不可」之信念和將霍家拳公諸於世的高風亮節，親筆寫下「尚武精神」四個大字，惠贈[[精武體育會|精武體育會]]<ref>{{cite web |language= zh |publisher=中華人網 |title=俠客行-霍元甲 |url=http://www.greatchinese.com/famous/xiake/huoyuanjia.htm |accessdate=2010-08-28 }}</ref>。

孫是模範愛國者、世界主義之現代化推動者；孫之備受推崇，原因是訴諸外國資本之大型經濟發展計劃之倡導者<ref name=白 />{{rp|432}}。

孫中山出生在一個信奉[[中国宗教|中國傳統宗教]]的家庭，主要信仰是[[佛教|佛教]]、[[道教|道教]]，幼名「帝象」，「帝」字乃親人為其請求「[[北帝|北帝]]」（真武玄天上帝）神護佑之意。後來孫中山就讀西方教會學校，14歲時在[[夏威夷|夏威夷]]讀書曾接觸[[基督教|基督教]]西方[[傳教士|傳教士]]，逐漸接受基督教，並且因為信仰緣故不惜放棄在檀香山學習。在香港拔萃書室开始信奉基督，並於1884年間（並非如一些史料所說之1883年）與[[陸皓東|陸皓東]]先後在[[美國公理會|美國公理會]]香港綱紀慎會堂[[受洗|受洗]]正式成为[[基督徒|基督徒]]<ref name=hager17 /><ref name=hager18 /><ref name=hager22 /><ref>{{cite web |url=http://www.zsda.gov.cn/plus/view.php?aid=284516 |title=孫中山在香港加入基督教時的受洗名單，孫日新即孫中山、陸中桂即陸皓東 |deadurl=yes |archiveurl=https://web.archive.org/web/20131014172807/http://www.zsda.gov.cn/plus/view.php?aid=284516 |archivedate=2013-10-14 }}</ref>。此時極為排斥[[中國民間信仰|中國民間信仰]]及其[[神祇|神祇]]如「[[北帝|北帝]]」、「[[關帝|關帝君]]」。但是这种说法是不成立的，因为孙中山逃亡在[[马来西亚|马来西亚]]时，担任当地的[[洪门|洪门]]领袖，及宗庙领导人，曾经几次受到逮捕也是透过[[槟城|槟城]]宝福社大伯公庙的密道逃跑。

以下為其早年對宗教之言論：

孫中山曾抨擊西方「用宗教來耗奪中國人的精神」<ref>《中國國民黨為九七國恥宣言》，1924-09-07</ref>，有人從其言論中提出孫中山信基督教或不信基督教的各種說法。孫中山晚年指出宗教「[[迷信|迷信]]」的弊端，1923年的公開演講中說「就宗教和科學比較起來，科學自然較優」。因為「科學的知識，不服從迷信，對於一件事，須用觀察和實驗的方法，過細去研究，研究屢次不錯，始認定為知識。」<ref>{{cite book |editor=[[中国社会科学院|中国社会科学院]]近代史研究所 |title=《孫中山全集》第8卷 |location=北京 |publisher=中华书局 |date=1981-08-01 }}</ref>{{rp|316}}在政治上，主張[[政教分離|政教分離]]、[[宗教自由|宗教自由]]。

孫中山論宗教和民族的關係：
{{Quote|「大凡人類奉拜相同的神，或信仰相同的祖宗，也可結合成一個民族。宗教在造成民族的力量中也很雄大，像[[阿拉伯|阿拉伯]]和[[猶太|猶太]]兩國，已經亡許久，但是阿拉伯人和猶太人，至今還是存在。他們國家雖亡，而民族之所以能夠存在的道理，就是因為各有各的宗教。大家都知道現在的[[猶太人|猶太人]]，散在各國的極多，世界上極有名的學問家，像[[卡尔·马克思|馬克斯]]、[[愛因斯坦|愛因斯坦]]，都是猶太人。在像現在英美各國的資本勢力，也是被猶太人操縱。猶太民族的天資是很聰明的，加以宗教之信仰，故雖流離遷徙於各國，猶能維持其民族于長久。[[阿拉伯人|阿拉伯人]]所以能夠存在的道理，也是因為他們有[[穆罕默德|穆罕默德]]的宗教。其他信仰[[佛教|佛教]]（誤，事實上是[[印度教|印度教]]）極深的民族，像[[印度|印度]]，國家雖然亡到英國，種族還是永遠不能消滅。」}}

孫中山注意[[信仰|信仰]]的力量，以三民主義信仰團結革命同志。他曾說：「宗教之所以能夠感化人的道理，便是在他們有一種主義，令人信仰。普通人如果信仰主義，便深入刻骨，便能夠為主義去死。因為這個原因，傳教的人往往為本教奮鬥，犧牲生命亦所不辭。」「我們國民黨要革命的道理，是要改革中國政治，實行三民主義和五權憲法。我們的這種主義，比宗教的主義還要切實。因為宗教的主義，是講將來的事和在世界以外的事；我們的政治主義，是講現在的事和人類有切膚之痛的事。宗教是為將來[[靈魂|靈魂]]謀幸福的，政治是為眼前肉體謀幸福的。說到將來的靈魂，自然是近於空虛；講到眼前的肉體，自然有憑有據。那麼宗教徒宣傳空虛的道理，尚可收到無量的效果；我們[[政黨|政黨]]宣傳有可憑據的道理，還怕不能成功嗎？！」民族主義是三民主義的內容之一，孫中山曾論及[[民族主義|民族主義]]思想以及民族和[[信仰|信仰]]的關係：「大凡人類奉拜相同的神，或信仰相同的[[祖宗|祖宗]]，也可結合成一個[[民族|民族]]。」「[[中國人|中國人]]黃色的原因，是由於根源黃色血統而成。祖先是什麼血統，便永遠遺傳成一族的人民，所以血統的力是很大的。」

孫中山論“三民主義信仰”：
{{Quote|「人類對於一件事，研究當中的道理，最先發生思想；思想貫通以後，便起信仰；有了信仰，就生出力量。所以主義是先由思想再到信仰，次由信仰生出力量，然後完全成立。何以說三民主義就是救國主義呢？因三民主義系促進中國之國際地位平等、政治地位平等、經濟地位平等，使中國永久適存於世界，所以說三民主義就是救國主義。」}}

其最后一任妻子[[宋庆龄|宋庆龄]]回忆道：「孫中山明確地告訴我，他從來不信甚麽[[上帝|上帝]]，他也不相信[[傳教士|傳教士]]（他們不是「偽善者」就是「受了誤導」）」<ref>1966年4月13日宋慶齡致[[伊斯雷尔·爱泼斯坦|伊斯雷尔·爱泼斯坦]]的信。見《宋慶齡書信集》下，北京[[人民出版社|人民出版社]]，1999年，第652頁</ref>宋庆龄认为，孙中山“从来没有感觉到耶稣教义和他自己的[[三民主义|主义]]之间有任何矛盾。对他来说，[[社会福音运动|社会革命]]就是把基督教义付诸实施。”<ref>《斯诺文集》第一卷，北京：新华出版社，1984年版，第109页</ref>

**第二任妻子：[[宋慶齡|宋慶齡]]（1893年1月27日－1981年5月29日），祖籍[[海南|海南]]省[[文昌市|文昌市]]（原廣東省文昌縣）,無子女。1915年10月25日孫與其在日本東京結婚，向同志聲言：「我愛我國，我愛我妻。我不是神，我是人。我是革命者，我不能受社會惡習所支配。」<ref name=命 />{{rp|611}}宋從父親與孫之交談中，「得悉我們的民國處在很大危險之中，因為袁世凱陰謀推翻它。一些國家在道義上和財政上支持著袁世凱，因為他們被其狡猾的外交手腕及其手下陰險毒辣的宣傳所欺騙。我國民眾之聲被壓制。革命事業似乎無望。孫的某些追隨者，在絕望中把革命事業看作失敗的事業而放棄了」<ref name=命 />{{rp|609}}。

== 紀念 ==
{{Main|孫中山紀念}}
=== 臺灣 ===
孫中山當初便因[[甲午戰爭|甲午戰爭]]清廷戰敗，割讓[[台灣|台灣]]，而加深其推翻[[清廷|清廷]]的革命決心。孫中山一生訪台四次（備註：孫中山1924年最後一次登台，只隨船泊港於[[基隆|基隆]]，並未上岸），尋求[[日本|日本]][[臺灣總督府|臺灣總督府]]幫助其所需的人力與金錢、[[軍火|軍火]]，也曾被[[軟禁|軟禁]]於梅屋敷<ref>梅屋敷旅社位於[[台北車站|台北車站]]附近。</ref>，今日成為[[逸仙公園|國父史蹟紀念館]]，連附近的道路一併改稱「中山北路」。

中華民國[[台北市|台北市]]的[[國立國父紀念館|國立國父紀念館]]，採用孫中山三次來臺灣的說法：

第一次到臺灣：1900年孫中山計劃在[[惠州起義|惠州起義]]，本來想從香港內渡但不成功，於是折回日本，轉渡臺灣，擬由臺灣設法潛渡內地，當時臺灣總督[[兒玉源太郎|兒玉源太郎]]十分贊同中國的革命，再加上義和團事變導致八國聯軍，中國北方已經陷於無政府的狀態。於是兒玉總督命令民政長官[[後藤新平|後藤新平]]與孫中山接洽，允諾起事之後，可以相助。孫中山於是一面擴充原有計畫，就地加聘軍官，一面令鄭士良即日發動，並更改原定計畫，不直逼省城，而先占領沿海一帶，多集黨眾，等待孫中山的到來，等到有武器的支援後，然後再行攻取省城。不料惠州起義後不久，日本內閣改組，新任首相[[伊藤博文|伊藤博文]]的對華政策，與前任[[山縣有朋|山縣有朋]]內閣大為不同，禁止臺灣總督與中國革命黨接洽，又禁止武器出口及日本軍官投入革命軍，於是孫中山的計畫乃遭到失敗。孫中山在1900年9月25日，化名吳仲，從日本神戶搭3,450噸的客輪臺南丸來臺。同行的有清藤幸七郎、[[宮崎寅藏|宮崎寅藏]]、[[內田良平|內田良平]]、平岡浩太郎、[[平山周|平山周]]、尾崎行昌等人。9月28日抵[[基隆|基隆]]進駐臺北，與臺灣總督[[兒玉源太郎|兒玉源太郎]]及民政長官[[後藤新平|後藤新平]]會晤。10月8日在臺北新起町設立革命總司令部指揮所，在此策劃惠州起義。新起町在割歸日本以前稱新起街，是艋舺後起的主要街道，現為長沙街。惠州之役失敗，孫中山聞訊，於11月10日仍舊化名「吳仲」，乘橫濱丸自基隆啟航返回日本，後藤新平同行。孫中山在臺前後停留44天，也獲得[[菲律賓|菲律賓]]製[[中古|中古]]武器一批，曾與[[楊心如|楊心如]]、[[吳文秀|吳文秀]]等會面。該次會面，也為兒玉總督於同年出兵佔領[[廈門|廈門]]的遠因之一。

[[File:Yixian_Park_entry_20130709.jpg|thumb]]
第二次到臺灣：民國2年（1913年）討袁失敗後，8月5日孫中山化名汪國權，與胡漢民等隨員二人，搭信濃丸秘密來臺，由臺灣總督派員接待，住進御成町梅屋敷。梅屋敷是臺灣歷史悠久旅社，建於1896年，1900年由新竹州山中移來古梅木200株栽作梅林，先為一小屋，後增建「吾妻別館」兼作料理業。孫中山身著黑色洋服，由日人護衛村田省藏隨行，進入梅屋敷，由主人大和宗吉、藤井悟一郎及佣人村上百惠接待孫先生用餐。餐後孫先生，揮毫「同仁」、「博愛」橫幅相贈，署名「孫文」。民國35年（1946年）10月10日，因臺灣省黨部主任委員李翼中之請，保留梅屋敷成立「國父史蹟紀念館」。民國43年（1954年）11月於紀念館中設立紀念亭，蔣中正親題碑文「匡復中華的起點，重建民國的基地」，以為中國國民黨建黨六十年紀念。紀念館現由臺北市公園路燈管理處管理。孫中山在臺期間，曾與翁俊明晤商但無活動。8月中旬孫中山仍搭信濃丸到神戶。2005年，台灣統派知名學者[[李敖|李敖]]曾以孫中山下榻梅屋敷的史實，質疑孫逸仙1913年的台灣之旅，有所謂嫖妓行為。該考證曾引起部分台灣[[榮民|榮民]]的嚴重抗議。一般學者則認為，該史實所稱的高砂族藝妓，應只是單純歌唱舞蹈表演的[[台灣原住民|台灣原住民]]。

第三次到臺灣：民國7年（1918年）6月，孫中山辭去護法軍政府大元帥後，由廣州搭船，經過汕頭，然後搭乘天草丸由臺灣轉赴日本。7日下午4時抵基隆，胡漢民、戴季陶隨行，目的是想和臺灣同胞見面發表意見，宣傳主義，以喚起民族意識，但遭臺灣總督府阻擾，臺灣官憲只派員到船中招待，下午5時即改乘信濃丸前往神戶。

1925年3月12日孫中山過世。消息傳來，台灣一些知識分子立即著手籌備孫中山的哀悼會，並且創作訃文、輓歌與弔文，準備在會上宣讀，不料台灣總督府得知以後，約談一些籌備會的成員，要求他們不要在會上發表演說、宣讀訃文與弔文、唱歌……極盡刁難之能事，所幸經過有志之士的周旋與妥協，哀悼會還是照常舉行。當時，一位重要的新文學作家[[張我軍|張我軍]]使用中國[[白話文|白話文]]寫了一首〈孫中山先生弔詞〉<ref>這篇弔詞後來收錄在1965年3月1日《[[傳記文學_(雜誌)|傳記文學]]》第6卷第3期，以及張我軍的相關著作。</ref>，原本打算在會上宣讀，卻因當局禁止而作罷。

1940年（民國29年），[[中華民國國民政府|中華民國國民政府]]通令全國，讚揚孫文「倡導國民革命，手創中華民國，更新[[政體|政體]]，永奠邦基，謀世界之大同，求國際之平等」，尊稱其為「[[中華民國國父|中華民國國父]]」或「中國國父」。

[[臺灣戰後時期|中華民國接收臺灣後]]，臺灣各地也常見為了紀念孫中山“[[中山路|中山路]]”、“[[中山公園|中山公園]]”、“[[中山堂|中山堂]]”等名稱的街道或紀念建築，[[台北|台北]][[國立故宮博物院|國立故宮博物院]]為紀念孫文百年冥誕命而名為“[[中山博物院|中山博物院]]”，亦於高雄市西子灣復校[[國立中山大學|國立中山大學]]。

至今在臺灣的[[中華民國政府|中華民國政府]]依然尊孫中山為國父，並於主要[[公家|公家]]機關及各級學校懸掛其遺像，臺灣民主化後，已無硬性規定，但不少的地方及公園仍有其銅像。

中華民國現今通行的新臺幣，拾圓、伍拾圓硬幣及壹佰圓紙鈔鑄印有孫中山肖像，[[中央銀行_(台灣)|中央銀行]]亦會發行孫文紀念幣。

孙中山于1912年卸任[[臨时大总统|臨时大总统]]后，曾与东莞上沙村孙氏族人合照。照片中，孙坐在前排正中，而参加此次会面的上沙孙族就有30多人。此照片至今仍保存在广东省中山市[[翠亨村|翠亨村]]孙氏宗祠，可供游客观览。此外，还有两副昔日香山赠送上沙的对联（实物已不存）。

由於孫中山晚年對中国共产党採取聯合政策，以及在他過世之後、其遺孀宋慶齡更是通過實際行動表達对共產黨支持，故孙中山亦為自1949年之後、在中國主流舆论中仍享有崇高名望的中國國民黨革命人士之一。

[[中華人民共和國|中華人民共和國]]成立後，原先常見街道名稱“[[中正路|中正路]]”大多被改，而“[[中山路|中山路]]”和“[[中山公園|中山公園]]”等常見名稱卻多保留至今。現今中國共產黨官方對孫中山評價为「中國近代民主革命的偉大先行者」。

1989年1月26日，中国于东南极大陆拉斯曼丘陵建立[[中国南极中山站|中国南极中山站]]，该站是中国的第二个南极科学考察站。站中设有“孙中山先生纪念堂”。

[[File:孙中山与人民英雄纪念碑.jpg|thumb]]
2005年起北京[[天安門廣場|天安門廣場]]逢重大節日——[[勞動節|勞動節]]與[[中華人民共和國國慶日|中華人民共和國國慶日]]，孙中山的巨幅画像被树立在[[人民英雄纪念碑|人民英雄纪念碑]]正前方。

中國共產黨稱孫中山為「杰出的爱国主义者和[[民族|民族]][[英雄|英雄]]、中国民主革命的伟大先行者」<ref>{{cite web |editor=余英杰 |url=http://www.gov.cn/ldhd/2006-11/12/content_440200.htm |title=胡锦涛在孙中山先生诞辰140周年纪念大会上的讲话 |publisher=新华社 |location=北京 |date=2006-11-12 }}</ref>。

在孫中山140周年誕辰日，[[中共中央總書記|中共中央總書記]]、[[中華人民共和國主席|中華人民共和國主席]][[胡錦濤|胡錦濤]]發表講話指出，在孫中山組織領導和革命精神感召下，於1911年辛亥革命推翻清朝的統治，結束在中國的君主專制制度，孫中山為中國人民和[[中華民族|中華民族]]建立歷史性功勞，復興中華民族，並盛讚他為「中華民族的民族英雄」<ref>{{cite news |editor=余英杰 |url=http://news.xinhuanet.com/politics/2006-11/12/content_5320326.htm |title=孫中山先生誕辰140周年紀念大會在京舉行 胡錦濤講話 |publisher=新華網 |date=2006-11-12 }}</ref>。

== 評價 ==

孫早年創立三民主義，為革命運動制定綱領<ref name=會 >{{cite book |editor=現代教育研究社編輯委員會 |title=《會考版中國歷史（中學五年級）（教師手冊）》 |location=香港 |publisher=現代教育研究社有限公司 |year=1993 }}</ref>{{rp|39}}。他和他的战友领导了辛亥革命，推翻了满清276年对华夏的殖民统治和民族压迫，光复了神州大地。晚年復吸納新思想，賦予三民主義以新解釋，推動[[國民革命|國民革命]]發展<ref name=會 />{{rp|39}}。策劃反清起義，策動反袁、護法，致力打倒[[北洋軍閥|北洋軍閥]]和[[帝國主義|帝國主義]]，為爭取中國獨立、自由和統一而奮鬥<ref name=會 />{{rp|39}}。

1892年7月，孫畢業於[[香港|香港]][[香港西醫書院|西醫書院]]，並且在學成績滿分，第一名畢業，由教務長康德黎博士頒發畢業證書。當時有十三位學生入學，但到畢業時僅有兩位學生成績合格能夠畢業，孫氏及江英華。香港西醫書院在當時尚是草創，仍未能獲得香港當局的承認，無法取得香港的行醫執照{{notetag|見香港1892年教育年報（電子版載於http://sunzi1.lib.hku.hk/hkgro/view/s1893/1383.pdf），第13頁，20條，譯文大意：經過五年訓練，西醫學院的首兩名學生於1892年7月畢業。雖然他們的資格仍未為政府承認，但一班獨立的考官認可他們可以在藥物、外科及接生方面執業。}}。香港西醫書院成立初期，畢業生只能以華人醫生資格在港行醫（當時法例無規管華人醫生）；並無法例上醫生資格。在當時的遠東地區仍屬比較有開拓性的醫學院，孫畢業時，北洋大臣李鴻章曾有意將其延攬到其欲創辦的北洋西醫書院<ref name=吳相湘/>。

許多反清革命家都持外國護照，尤其是有了倫敦遇難的經驗，孫在1904年3月14日（倡導革命時期）以出生在夏威夷為理由申請美國公民，但在1904年4月7日從夏威夷前往舊金山時，遭到美國移民局官員懷疑并扣留，孫聘請律師{{notetag|亦有說法是得到舊金山[[洪門|洪門]]致公堂保釋並代聘律師。}}，找了親友作證並打贏官司<ref name=吳相湘/>，保留美國國籍。美國此時正實施[[美国排华法案|排華法案]]。中國人除政府官員，少部份留學生外，一律禁止入境。孫中山是以宣誓方式取得[[夏威夷|夏威夷]]出生紙後，在美国政府的帮助下<ref>{{cite news|url=http://www.kwongwah.com.my/news/2011/06/06/76.html|publisher=《光华时报》|date=2011-06-06|title=美在台协会首次出示文件证明　孙中山是美国人}}</ref>，申請得到美國護照入境，以免被遣送回大清帝国，並讓孫之後可到美國東岸尋求各国人民支持革命。

1915年10月25日，[[宋慶齡|宋慶齡]]離家私奔，在東京由日本律師和田瑞作證，同孫結婚。宋慶齡之父[[宋嘉澍|宋嘉澍]]聞之大怒，不能諒解，宋嘉澍夫婦追到日本，據日本人士回憶：宋嘉澍在大門口叫喊：「我要見搶走我女兒的總理！」孫出來後，宋突然往地上一跪，磕了幾個頭說：「我不懂事的女兒就拜託你了，請千萬多關照！」之後轉頭就走。

二次革命失敗後，孫與一些革命同志逃亡日本。當時有對前途失信心的人，想拿孫的八字去給命相師算一算，看將來是否有革命成功之日；換句話說，如果孫八字不好，那麼革命便不會成功，他們可能就準備要各奔前程，另謀出路了。但當他們去詢問時，孫罵他們不該這麼迷信，並且說：「若我八字不好，難道我們就不革命了嗎？」接著，他告訴那些意志不堅的人說，他的「八字」是「打倒軍閥，繼續革命」。

1923年2月20日，香港大學學生會請58歲的孫回[[香港大學|香港大學]]做報告，在[[陳友仁|陳友仁]]陪同下乘汽車抵達母校發表演講<ref>[http://www.chinareviewnews.com/doc/1013/7/8/5/101378515.html?coluid=7&kindid=0&docid=101378515 辛亥革命百年慶典　香港應積極參與]，中國評論新聞網</ref>。有問到孫於何時及如何而得革命思想及新思想，他說：「……我之此等思想發源地即為香港。至於如何得之，則我於三十年前在香港讀書，暇時輒閒步市街，見其秩序整齊，建築閎美，工作進步不斷，腦海中留有甚深之印象。……由市政之研究進而為政治之研究。研究結果，知香港政府官員皆潔己奉公，貪贓納賄之事絕無僅有，此與中國情形正相反。蓋中國官員以貪贓納賄為常事，而潔己奉公為變例也。……於是覺悟鄉村政治乃中國政治中之最清潔者，愈高則愈齷齪。……中國對於世界他處之良好事物皆可模倣，而最要之先著，厥為改變政府。現社會中最有力之物，即為一組織良好之政府。……我既自稱革命家，社會上疑議紛起，多所誤會，其實一中國式之革命家究不過抱溫和主義，其所主張者非極端主義，乃爭一良好穩健之政府。……學友諸君乎！諸君與-{余}-同受教育於此英國屬地，並在同一之學校，吾人必須以英國為模範，以英國式之良政治傳播於中國全國。」<ref>{{cite web|author=孫中山|title=我之革命思想發源地為香港——在香港大學的演說|date=1923-02-20|url=https://bbs.powerapple.com/forum.php?mod=viewthread&tid=2752866}}</ref>

據[[李敖|李敖]]引述[[陳鵬仁|陳鵬仁]]譯著《宮崎滔天論孫中山與黃興》一書記載，第29任[[日本首相|日本首相]][[犬養毅|犬養毅]]有一次問孫：一生最喜歡什麼，孫毫不猶疑說：「Revolution」(革命），再問：那麼第二呢？孫笑而不答，一再追問，孫答說：「Woman」（女人），犬養毅於是拍着手說，很好，並問：再其次呢？孫答：「Book」（書）。
