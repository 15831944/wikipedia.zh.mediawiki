'''孫正義'''（[[日语|日语]] : {{lang|ja|孫 正義}}，[[韩语|韩语]] : 손정의，{{bd|1957年|8月11日|catIdx=Son, Masayoshi}}），生於[[日本|日本]][[佐賀縣|佐賀縣]][[鳥栖市|鳥栖市]]，[[軟銀集團|軟銀集團]]的創辦人兼[[社長|社長]]，[[軟銀|軟銀]]股份有限公司[[會長|會長]]。據《[[福布斯|-{zh-cn:福布斯; zh-sg:资本家; zh-hk:福布斯; zh-tw:富比士;}-]]》雜誌報導，他在2011年擁有81億美元淨資產，從而名列日本富豪榜第二位。孫正義在《福布斯》2019年億萬富翁排行榜中名列第43位，資產達到216億美元。<ref>{{Cite web |url = https://www.forbes.com/profile/masayoshi-son/?list=billionaires#671e786e3818 | title = #43 Masayoshi Son | publisher = Forbes |language =en |accessdate = 2019-03-24 }}</ref>

孫正義曾在20世纪90年代[[互聯網泡沫|互聯網泡沫]]的[[雅虎|雅虎]]估价最高时，超过[[微软|微软]]创办人[[比尔·盖茨|-{zh-tw:比爾·蓋茲; zh-hk:比爾·蓋茨; zh-cn:比尔·盖茨;}-]]成为一天时间世界首富。<ref>[http://www.people.com.cn/GB/paper81/3947/471220.html 孙正义：做过一天世界首富的人]</ref>依靠软银数次对[[阿里巴巴集團|阿里巴巴集團]]的控股与投资回报，成為日本首富。<ref>[http://economy.gmw.cn/newspaper/2014-09/19/content_100992912.htm 孙正义投资阿里获利2500倍：巴菲特望洋兴叹]</ref>

== 生平 ==
為[[韓裔日本人|韓裔日本人]]第三代。其祖父[[孫鐘慶|孫鐘慶]]，由[[朝鮮半島|朝鮮半島]]移居日本九州。其父孫三憲，改用日本姓氏「安本」，定居於日本[[佐贺县|佐贺县]][[鳥栖市|鳥栖市]]朝鮮人聚落中，是一位企業家，以釀造私酒致富。其母李玉子，家中生有四名子女，孫正義為次男。孫正義出生時，取名安本正義。因為身為朝鮮後裔，幼年時受到同年紀日本人的歧視。在和日本[[麦当劳|麥當勞]]社長[[藤田田|藤田田]]會面過後，孫氏聽取其建議開始學習[[英语|英語]]和[[计算机科学|電腦科學]]。

16歲時，孫氏移居[[加利福尼亚州|加利福尼亞州]]，後來進入[[加州大學伯克利分校|加州大學伯克利分校]]，主修[[经济|經濟]]，還選讀了一些電腦課程。他被一份雜誌提及的[[微型晶片|微型晶片]]吸引着了，19歲的他已認為電腦科技將帶起下一輪的商業革命。

留意到一切和微型晶片有關的東西都大有作為，孫氏決定要每天思考至少一個企業想法。他取得一個翻譯裝置的專利，以一百萬[[美元|美元]]賣給[[夏普公司|夏普]]，這個專利的應用包括夏普的Wizard系列[[个人数码助理|个人数码助理]]。

孫氏將《[[太空侵略者|太空侵略者]]》這個遊戲引入了大學內。1980年畢業。1981年，孫氏回到日本，創立軟體銀行，發展了有關電腦的媒體及軟件。

1980年代末期，孫氏仍不是日本公民。他的日本妻子隨他姓孫。結婚後，法律承認了她的新姓氏，孫正義隨之而成為日本公民而不用改姓傳統的日本姓氏。1991年，孫正義[[歸化|歸化]]為日本人{{Fact|time=2018-05-10T13:07:00+00:00}}。

== 家族 ==

其本貫為[[一直孫氏|一直孫氏]]，據孫正義本人描述<ref>參見日本的雑誌《[[文藝春秋|文藝春秋]]》1999年11月号記事。</ref>，出自[[春秋時代|春秋時代]]的著名[[兵法|兵法家]][[孫武|孫武]]的一族<ref>參見《和讯IT消息》2008年08月02日記事。「第二届APEC工商咨询理事会亚太中小企业峰会召开，软银集团董事长兼总裁孙正义透露了自己的家族渊源。他說道：「我在23代之前，我的家族、我的先人也是中国人。所以说我的姓是姓孙，我的姓是一个中国的姓氏，因为我的家族从23代前从中国移到了韩国，然后再迁到了日本。」1957年8月11日，孙正义出生，孙正义的父母在九州经营柏青哥生意，祖父辈从中国福建经韩国移民至日本当矿工，并取日本姓氏安本。</ref>。他的后代移居到[[朝鮮半島|朝鮮半島]]，传到22世孫鐘慶，从[[韓國|韓國]][[大邱市|大邱市]]渡海定居在[[日本|日本]]，孙家在日本定居已有三代。他的家族说：「孫氏和[[韓国|韓国]]固有的孫氏不一样，我祖籍和[[漢民族|漢民族]]的[[孫姓|孫氏]]属于同一根源。”<ref>《孫家的遺伝子》孫泰蔵著、角川書店 、2002年、日本、ISBN 4-04-883764-8</ref>
