'''孙亚夫'''（{{bd|1952年|3月||}}）生于[[福建|福建]][[福州|福州]]，[[中华人民共和国|中华人民共和国]]官员。<ref name=gtb>[http://www.gwytb.gov.cn/gtb/sunyafu/ 孙亚夫，中共中央台湾工作办公室、国务院台湾事务办公室，于2013-04-30查阅] {{webarchive|url=https://web.archive.org/web/20171012150632/http://www.gwytb.gov.cn/gtb/sunyafu/ |date=2017-10-12 }}</ref>

==生平==
孙亚夫生于1952年3月。1969年1月参加工作，成为[[福建省|福建省]][[顺昌县|顺昌县]][[插队|插队]][[知识青年|知识青年]]。1972年12月，成为[[福州市|福州市]]钢铁磷肥厂工人。其间，1976年7月加入[[中国共产党|中国共产党]]。1978年2月，成为[[厦门大学|厦门大学]]历史系学生，1982年2月毕业。1982年3月，成为[[外交学院|外交学院]]国际关系史专业硕士研究生。1984年12月毕业获得硕士学位后，留校担任外交学院国际关系史教研室教员。<ref name=gtb/>

1985年11月，任[[全国人大华侨委员会|全国人大华侨委员会]]办公室秘书。1990年4月起，历任[[中央对台工作领导小组|中央对台工作领导小组]]办公室副处长、处长。1991年1月，任[[中共中央台办|中共中央台办]]、[[国务院台办|国务院台办]]处长。1992年8月，升任[[中共中央台办|中共中央台办]]、[[国务院台办|国务院台办]]协调局副局长，[[海峡两岸关系协会|海峡两岸关系协会]]副秘书长。1994年6月，改任[[中共中央台办|中共中央台办]]、[[国务院台办|国务院台办]]研究局副局长。1997年6月，升任[[中共中央台办|中共中央台办]]、[[国务院台办|国务院台办]]研究局局长。其间，1998年起兼任[[海峡两岸关系协会|海峡两岸关系协会]]副会长。2000年7月，出任[[中共中央台办|中共中央台办]]、[[国务院台办|国务院台办]]主任助理兼研究局长。2004年2月至2013年11月，任[[中共中央台办|中共中央台办]]、[[国务院台办|国务院台办]]副主任。<ref name=gtb/>2012年3月，任[[厦门大学|厦门大学]]台湾研究中心主任<ref>{{Cite web |url = http://www.chinanews.com/edu/2012/03-31/3790959.shtml |title = 国台办副主任孙亚夫受聘厦大台湾研究中心主任 |author =  |date = 2012-03-31 |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。2013年2月1日，当选[[中国人民政治协商会议|中国人民政治协商会议]]第十二届全国委员会委员，任全国政协港澳台侨委员会委员<ref>{{Cite web |url =  http://cpc.people.com.cn/gbzl/html/121000033.html|title =  孙亚夫|author =  |date =  |publisher =中国领导干部资料库  |language =  |accessdate = 2017-10-12 |quote =  }}</ref>。
