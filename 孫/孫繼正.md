'''孫繼正'''（{{bd|1962年|3月17日|CatIdx=S孫}}），[[台北市|台北市]][[北投區|北投]]人<ref>[https://www.ettoday.net/news/20171101/1042971.htm?feature=todaysforum&tab_id=268 罷免黃國昌為選舉？　安定力量：2018不參選公職], Ettoday, 2017年11月1日</ref>，為[[安定力量|安定力量]]聯盟主席，崇光金品公司代表人，<ref>{{cite web|title=崇光金品股份有限公司|url=http://datagovtw.com/company.php?id=84286958|website=臺灣政府資料開放平臺}}</ref>曾到基督教台北豐盛教會講道<ref>{{Cite web|url=http://abundant.org.tw/modules/tadnews/index.php?nsn=591|title=基督教台北豐盛教會 - 鼎 爐 之 中-孫繼正弟兄|accessdate=2017-03-16|last=豐盛教會|work=abundant.org.tw|language=zh-hant}}</ref>，現為罷免[[新北市第十二選舉區|新北市第十二選舉區]]立法委員[[黃國昌|黃國昌]]之代表人物。<ref>{{Cite news|url=http://www.storm.mg/article/207008|title=罷免黃國昌難過關，出於這幾個原因-風傳媒|accessdate=2017-03-14|language=zh-hant}}</ref>

== 生平 ==
孫繼正的父親為警官，小妹早逝。與音樂創作人[[黃國倫|黃國倫]]有私交，曾一同上《[[康熙來了|康熙來了]]》。<ref>{{Cite web|url=http://www.goodtv.tv/vod/prog_id/533/get_vid/13036?url=533/541/0|title=真情部落格 走過千帆 - 孫繼正|accessdate=2017-03-14|work=好消息電視台|language=zh-TW}}</ref>

== 政治活動 ==
2016年12月[[北北基安定力量聯盟|北北基安定力量聯盟]]成立，由孫繼正擔任主席，[[游信義|游信義]]為秘書長，12月24日，聯盟在新北市汐止區發動罷免黃國昌的行動。<ref>{{cite news|url=http://udn.com/news/story/8375/2190406|title=罷免黃國昌 提案人數過門檻|author=}}</ref>

2017年3月，孫繼正透露安定力量聯盟已獲內政部正式立案，轉型為全國性政治團體；但他亦表示，他個人將不會從政參選。<ref>{{Cite news|url=https://udn.com/news/story/6656/2336642|title=「拔昌」列車啟動 黃國昌：尊重|last=聯合新聞網|first=|date=|work=|newspaper=聯合新聞網|accessdate=2017-03-14}}</ref>

2017年7月，在中國國民黨智庫「[[國家政策研究基金會|國家政策研究基金會]]」，與[[王大師|王大師]]、[[孫立群|孫立群]]、[[曾銘宗|曾銘宗]]及[[葉慶元|葉慶元]]一起召開記者會，不滿[[臉書|臉書]]遭封鎖。<ref>{{cite news|title=批臉書染綠頻封鎖藍軍言論，國民黨基金會要唐鳳踹共|url=http://www.storm.mg/article/305386|agency=風傳媒|date=2017-07-27}}</ref>

2017年10月，接受《[[中國評論通訊社|中國評論通訊社]]》專訪表示：「黃國昌的所作所為證明他違背民意、操弄民粹、態度傲慢」。<ref>{{cite news|title=堅定反同婚 基督徒發動罷免欲扳倒黃國昌|url=http://www.crntt.tw/doc/1048/5/4/0/104854007.html?coluid=93&kindid=2910&docid=104854007|agency=台灣中評網|date=2017-10-24}}</ref>

2019年3月，宣布參選新北第12選區立委，並再度劍指時代力量立委黃國昌，高喊「黃國昌嘜造，直球對決。」<ref>{{cite news|title=孫繼正選立委再度劍指黃國昌 時力：尊重|url=https://udn.com/news/story/6656/3726322?from=udn-ch1_breaknews-1-0-news|agency=聯合新聞網|date=2019-3-29}}</ref>

== 爭議 ==
2003年1月25日，崇光金品公司因違反政府採購法，被[[臺灣臺北地方法院檢察署|臺灣臺北地方法院檢察署]]檢察官提起公訴，科罰金新台幣200,000元。<ref>{{cite web|title=臺灣臺北地方法院刑事簡易判決 九十二年度簡字第四四一五號|url=http://djirs.judicial.gov.tw/fjud/index_1_S.aspx?p=QqUvN3ACdcNy2pzI1WVDEdGnrfTrF7HVSo5TR%2fRcoQA%3d|website=中華民國司法院}}</ref>
