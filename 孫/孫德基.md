'''孫德基'''{{GBSHK}}{{JP}}（{{bd|1953年}}），前香港[[審計署_(香港)|審計署署長]]，[[香港會計師公會|香港會計師公會]]前會長。

孫德基是[[香港會計師公會|香港會計師公會]]資深會員，[[美國註冊會計師協會|美國註冊會計師協會]]及[[伊利諾伊州註冊會計師協會|伊利諾伊州註冊會計師協會]]會員，前任[[安永會計師事務所|安永會計師事務所]]遠東區主席及首席合伙人（於2010年離職）。孫德基打破回歸後審計署署長一直由公務員出任首長的慣例，現為[[北京市政協|北京市政協]]委員，與[[梁振英|梁振英]]同是首批藉改革開放北上的香港專業人士。<ref name="first">{{Cite web |url= http://www.sharpdaily.hk/article/news/20120629/107192/ |title= 候任審計署長　孫德基有污點 |publisher= 
爽報  }}</ref>

== 公共事务 ==
在公共事務方面，孫德基擔任過[[香港按揭證券有限公司|香港按揭證券有限公司]][[非执行董事|非执行董事]]，[[强制性公積金計畫管理局|强制性公積金計畫管理局]]非执行董事，[[香港科技大學|香港科技大學]]校董及[[香港聯交所主板及創業板上市委員會及投資顧問委員會|香港聯交所主板及創業板上市委員會及投資顧問委員會]]委員。同时，孫德基也在2003年之際被[[香港公开大學|香港公开大學]]授予榮譽[[商業管理学|商業管理学]]博士學位<ref>{{Cite web |url= http://finance.sina.com.cn/g/20050519/14201603412.shtml |title= 安永中国区新任主席孙德基重视国企客户 |publisher= 
新浪财经 | date =2005年5月19日 | }}</ref>，并於2006年獲香港特別行政區政府頒授[[銅紫荊星章|銅紫荊星章]]及于2008年被委任为[[太平紳士|太平紳士]]<ref>{{Cite web |url= http://finance.qq.com/a/20110604/001654.htm |title= 安永孙德基介绍 |publisher= 腾讯财经 | date =2011年6月4日 | }}</ref>。

== 争议 ==
[[人民力量|人民力量]]認為，孫德基與[[梁振英|梁振英]]關係良好，可能導致孫德基在執行審計署署長時政治不中立，同時也質疑孫德基在安永任職期間，安永被雅佳清盤人控告專業失當及索償，可能間接成為他會計師專業生涯中的污點，因此人民力量反對他出任署長。<ref name="first" />不過，孫德基指出該[[民事訴訟|民事訴訟]]中，他的[[專業操守|專業操守]]完全沒有受到質疑。
