'''孫振環''' ({{bd|1898年||1972年||}})，[[河北省|河北省]][[沧州市|滄州]]人，中國武術家，前[[廈門通背武術社|廈門通背武術社]]社長，有「[[鼓浪屿|鼓浪嶼]][[霍元甲|霍元甲]]」稱號。<ref>{{Cite news|url=http://www.taihainet.com/news/xmnews/shms/2016-06-13/1725728.html|title=通臂拳弟子何丙仲忆一代宗师孙振环“武功了得 用长矛击黄豆”|author=林少蓉|date=2016-6-13|work=台海網|accessdate=2018-9-6}}</ref>

== 習武經過 ==
早年師從滄州孫振坤、左東君習[[通臂拳|通背]]、[[劈掛拳|劈掛]]，練成「通背鐵沙掌」<ref>{{Cite book|title=講武論道第二章|last=洪|first=敦耕|publisher=天地圖書|year=2013|isbn=9789882198944|location=Hong Kong|pages=24-31}}</ref>。後在左東君推薦下，從江德灣處學「[[六合拳|六合]]槍法」，[[劉德寬|劉德寬]]的「七門槍」，[[馬良_(民國)|馬良]]的「[[潭腿|潭腿]]」，又向其他武林老前輩學習「七門槍」、「秋大刀」、「[[虎尾三節棍|虎尾三節棍]]」等武藝。

== 生平 ==
1989年生於河北省滄州[[盐山县|鹽山縣]]孫八里村一個武術世家，自小練武。武成後，最初經劉德寬介紹下在[[天津市|天津]]充任鏢師，但因為仗義懲戒惡霸「花豹子」，得罪了當地惡勢力，只好轉往西北謀生。

其時，[[冯玉祥|馮玉祥將軍]]正在[[西安市|西安]]廣招天下豪傑，孫振環便投身軍旅，憑著一身功夫，成為西北軍武術教官，馮玉祥讚他是「西北第一把跤」。

後來，[[日本|日本]]揮軍西北，孫振環在[[宋哲元|宋哲元]]將軍的指揮下，雖然屢立戰功，但軍力懸殊，所在部隊終被擊潰。

1934年為了避開日軍偵緝，孫振環輾轉來到廈門，受聘於[[厦门市|廈門]][[中南銀行|中南銀行]]為鏢師，並在當地安頓，停止漂泊生活。

翌年，開辦「通臂武術社」，廣收門徒。並在[[精武体育会|精武體育會]]廈門分會與英華中學教武術。<ref>{{Cite book|title=中国武术人名辞典|last=昌|first=沧|publisher=人民体育出版社|year=1993|isbn=7500909519|location=China|pages=}}</ref>

1953年，新中國成立後，孫振環重開通臂武術社 , 將之設在鼓浪嶼泉州路。

1958年，被福建省體委聘為全省武術集訓隊總教練，負責挑选和訓練全省各地的武術精英。

[[文化大革命|文革]]期間，好景不常，通臂武術社被迫關閉。

1972年6月3日凌晨，因突發心肌梗塞，與世長辭，終年74歲。<ref name=":0">{{Cite web|url=https://www.daoyoupt.com/i2755/|title=【历史名人】武林英豪孙振环先生|accessdate=2018-9-6|author=荀青云|date=2015-08-12|publisher=}}</ref>

== 徒弟 ==
孫振環一生授徒上萬人，其中不乏武術界的精英，如[[洪敦耕|洪敦耕]]、[[施載煌|施載煌]]、黄傳勛(黃传勋)、楊漢國、孫誌慶、高毅等，都是全省乃至全國知名的武術家。

== 評價 ==
孫在廈門期間，為人正義，他的許多軼事，在老一輩廈門人中廣為流傳。

歐美及東南亞一帶武壇頗具影響力的《[[黑帶_(雜誌)|黑帶]]》、《[[武林周刊|武林周刊]]》等雜誌，把孫振環稱之為「德藝雙馨」、「近代武林的一代宗師」。

廈門文史學家何丙仲先生概括孫振環一生：「仁者之魂，勇者之魄。獻身體壇，垂範藝德」。<ref name=":0" />
