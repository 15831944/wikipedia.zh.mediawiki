'''三星Galaxy Note 9'''是由三星電子設計、開發和銷售的Android智能手機，是三星Galaxy Note系列的旗艦手機。於2018年8月9日亮相，作為[[三星Galaxy_Note_8|三星Galaxy Note 8]]的繼承者<ref>{{Cite news|url=https://www.express.co.uk/life-style/science-technology/968896/Samsung-Galaxy-Note-9-release-date-UK|title=Galaxy Note 9 release revealed and this is date Samsung fans should put in their diary|last=Snelling|first=David|date=2018-06-03|work=Express.co.uk|access-date=2018-06-04|language=en}}</ref>。

== 歷史 ==
許多Galaxy Note 9功能在正式發布之前就被洩露了，包括它的S-Pen。<ref>{{cite web|url=https://m.gsmarena.com/samsung_galaxy_note9_specs_and_features_revealed_through_detailed_handson-news-31949.php|title=Samsung Galaxy Note9 specs and features revealed in detailed hands-on|publisher=}}</ref><ref>{{cite web|url=https://www.theverge.com/circuitbreaker/2018/6/27/17509830/samsung-unpacked-galaxy-note-9-event-august-9th-announced|title=Samsung announces Galaxy Note 9 event for August 9th|publisher=}}</ref>

== 規格 ==
=== 硬件 ===
Samsung note 9採用6.4英寸[[1440p|1440p]] [[Super_AMOLED|Super AMOLED]]顯示屏，寬高比為18.5：9。前面的設計與Note 8類似，使用三星Infinity Display。

Samsung note 9在美國、加拿大、日本、南美洲、中國大陆、香港和澳門用[[高通骁龙|高通骁龙]]845，在歐洲、南韓、臺灣用[[三星Exynos|三星Exynos]] 9810。有128或512 GB的存儲選項。128 GB型號具有6 GB RAM，而512 GB具有8 GB RAM。所有型號都有一個microSD卡插槽，可能最大限度地將手機存儲空間擴大到接近1TB的存儲空間。

機身顏色:-{zh-hant:湛海藍; zh-cn:寒霜蓝}-,薰衣紫,-{zh-hant:霧金銅; zh-cn:玄镜铜;}-,-{zh-hant:午夜黑; zh-cn:丹青黑;}-,雲朶銀(只限美國售賣),初雪白(只限臺灣,香港,於11月23日公開售賣)。

Note 9具有IP68防水防塵功能和USB-C連接器，支持Samsung DeX，無需底座，手機還配有3.5 mm耳機插孔以及支持Dolby Atmos的AKG調諧立體聲揚聲器。

Note 9通過Qi標準具有快速無線充電功能。

Note 9的電池已經從以前的三星Galaxy手機大幅升級; 它有4000毫安時的電池。

相機系統是雙攝像頭設置，其中一個鏡頭具有與S9 +相機系統類似的雙光圈（f / 1.5和f / 2.4）系統，但設置是水平翻轉而不是垂直翻轉，類似於Note 8的設置。

指紋傳感器已經移動到攝像機設置下方，就像S9和S9 +一樣，而不是像Note 8那樣在它旁邊。 <ref>{{Cite journal|date=2017-08-07|title=Samsung Galaxy Note 9 Will Feature In-Screen Fingerprint Sensor|url=https://www.highbeam.com/doc/1G1-503305345.html|journal=|author=|access-date=2018-08-11|archive-url=https://web.archive.org/web/20180810011323/https://www.highbeam.com/doc/1G1-503305345.html|archive-date=2018-08-10|dead-url=yes}}</ref><ref>{{Cite news|url=https://www.techradar.com/news/samsung-galaxy-note-9-what-we-want-to-see|title=Samsung Galaxy Note 9 release date, price, news and leaks|work=TechRadar|access-date=2018-05-10|language=en}}</ref><ref>{{Cite web|url=https://mobilizujeme.cz/stitky/samsung-galaxy-note-9|title=Samsung Galaxy Note 9 - Note 9將從銅管散熱改為碳纖維熱管，散熱效果更好。

==== S-Pen ====
S Pen現在具有藍牙功能，包括點擊其上的按鈕（長按，單按或連按）來執行某些操作，例如在演示文稿中向前或向後移動或拍照，並通過應用程序為應用程序提供第三方SDK支持。

S-Pen現在有一個電池（基本上是一個超級電容器），在手機內部充電，三星聲稱使用30分鐘（或按鈕最多200次點擊），只需40秒的充電時間。 <ref>{{Cite web|url=https://www.samsung.com/us/mobile/galaxy-note9/s-pen/|title=Galaxy Note9 S Pen - Stylus Pen with Android Remote Control {{!}} Samsung US|website=Samsung Electronics America|language=en|access-date=2018-08-10}}</ref>

=== 軟件 ===
Note 9附帶Android Oreo 8.1，三星Samsung Experience 9.5。

相機軟件已得到進一步改進，現在包括智能相機功能，以幫助識別場景。
