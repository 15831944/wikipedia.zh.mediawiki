'''三韩'''是公元前2世纪末至公元后4世纪左右[[朝鲜半岛|朝鲜半岛]]南部三个部落联盟，包括[[马韩|马韩]]、[[辰韩|辰韩]]和[[弁韩|弁韩]]。<ref>《后汉书·东夷列传》第七十五：“韩有三种”。</ref>《三国志·魏志·东夷传》中提到：“韩在带方之南，东西以海为限，南与倭接，方可四千里。 ”按照秦汉时期一里为415.8米的标准来计算，那么三韩部落所占据的总面积应该不下十七万平方公里，已将大半个朝鲜半岛都囊括在了其中。 其中[[马韩|马韩]]是三韩中最强大的，被三韩拥立为“辰王”，定都目支国，统领三韩之地。<ref>《后汉书/卷85》：马韩最大，共立其种为辰王，都目支国，尽王三韩之地。</ref>4世纪[[马韩|马韩]]已被[[百济|百济]]兼并。[[辰韩|辰韩]]被[[新罗|新罗]]兼并，[[弁韩|弁韩]]则演化成[[伽倻|伽倻]]后又被[[新罗|新罗]]兼并。

“三韩”一词最早在中国的典籍《后汉书·东夷列传》中出现。三韩所在的具体位置存在争议。不同时期，三韩的边界也不断的变化。但大體上认为在朝鮮半島的南部，大致为今天的大韩民国。《[[后汉书|后汉书]]》曰三韩之地“皆古之[[辰国|辰国]]也”<ref>《后汉书/卷85》：（三韩） 凡七十八国，伯济是其一国焉。大者万余户，小者数千家，各在山海闲，地合方四千余里，东西以海为限，皆古之辰国也。</ref>

「韓」（{{kor|k=한|rm=han}}）在現代[[韩语|韩语]]中被賦予“伟大”或“领袖”的意思。[[大韩民国|大韩民国]]的国名中就含有“韩”字。「韓」後來演變為[[朝鮮族|朝鮮族]]的別名。三韩是现在[[朝鲜族|朝鲜族]]的直系祖先，也是如今韩国国名得来的原因。

== 概况 ==
{{朝鮮半島歷史|隱藏=1}}
《三国志》中[[辰韩|辰韩]]與秦人有關，也有说法沿用了[[辰国|辰国]]的“辰”字。 [[弁韩|弁韩]]的另一个称呼“弁辰”也使用“辰”这个字。另外[[马韩|马韩]]的统治者在一段时间称自己为辰王表示对整个三韩的统治。

[[马韩|马韩]]是三韩中最大的一个，由54个部落构成。大致位于[[全罗道|全罗道]]，[[忠清道|忠清道]]和[[京畿道|京畿道]]。

[[辰韩|辰韩]]主要位于[[洛东江|洛东江]]以东，由12个城邦组成。其中的一个城邦后来吸收了其它城邦发展成[[新罗|新罗]]。

[[弁韩|弁韩]]由12个城邦组成，发展成[[伽倻|伽倻]]后被[[新罗|新罗]]吸收。弁韩位于洛东江以南和以西。三韓领袖叫天君，有祭天的义务。

== 技术 ==
铁制工具使农业耕作更加容易，从而也促进了朝鲜半岛的农业发展。[[全罗道|全罗道]]在当时已发展成[[水稻|水稻]]生产中心。

== 典籍中的三韩 ==
《後漢書·東夷列傳》第七十五：「韓有三種：一曰馬韓、二曰辰韓、三曰弁辰。馬韓在西，有五十四國，其北與樂浪，南與倭接，辰韓在東，十有二國，其北與[[濊貊|濊貊]]接。弁辰在辰韓之南，亦十有二國，其南亦與[[倭|倭]]接。凡七十八國，伯濟是其一國焉。大者萬余戶，小者數千家，各在山海間，地合方四千余里，東西以海為限，皆古之[[辰國|辰國]]也。馬韓最大，共立其種為辰王，都目支國，盡王三韓之地。其諸國王先皆是馬韓種人焉。」

雖然[[朝鮮半島|朝鮮半島]]古代國名多變，但是「韓人」、「韓民」之稱一直在民間延續，在中韓兩國的法律文書中也時有出現，比如：19世紀80年代的《中朝商民水陸貿易章程》中就有「華民」、「韓人」之稱，19世紀[[甲午戰爭|甲午戰爭]]前夕，清光緒帝的《[[:s:光緒二十年對日宣戰詔書|對日宣戰詔書]]》中，也有「著李鴻章嚴飭派出各軍，迅速進剿，厚集雄師，陸續進發，以拯'''韓民'''于塗炭」之稱。

== 與漢四郡的關係 ==
[[File:Ancient_Korea_Taihougun.png|thumb]]
[[File:Map_of_The_east_barbarian_0.png|left]]'''三韩'''諸国与汉四郡位置。]]
[[汉四郡|汉四郡]]中的[[乐浪郡|乐浪郡]]似乎与三韩的每个城邦都有联系。 

[[真番郡|真番]]郡治[[霅縣|霅縣]]與其餘無考的七縣為開[[辰國|辰國]]北部所置。[[三國志|三國志]]魏志馬韓傳云：“從事吳林以樂浪本統三韓，分割辰韓八國以與樂浪。”韓國學者[[李丙燾|李丙燾]]認為，此處“辰韓八國”可能就是真番郡南部沒入辰國的八縣，包括霅縣<ref>李丙燾，《真番郡考》，周一良譯，《禹貢》期刊2-{卷}-10期</ref>。

昭帝[[始元|始元]]五年（前82年），罷真番郡，以北部七縣入樂浪郡，隸屬于南部都尉。霅縣及其南部無考的七縣可能于此時沒于辰國，後入馬韓。[[漢獻帝|獻帝]][[建安_(漢獻帝)|建安]]中，割據[[遼東|遼東]]的公孫康分樂浪郡屯有縣以南地置[[帶方郡|帶方郡]]<ref>三國志魏志東夷傳</ref>，辖带方、列口、長岑、昭明、含資、提奚、海冥七縣<ref>[[續漢書|續漢書]]郡國志</ref>，即西漢時真番郡北部故地。

[[曹魏|魏]][[曹芳|齊王]][[正始|正始]]年閒，魏國遣[[毌丘儉|毌丘儉]]等伐[[辰韓|辰韓]]，恢復了對臨屯郡故地的統治，東濊、辰韓諸部落多由帶方郡管轄。[[西晉|西晉]][[太康_(晉)|太康]]中改昭明縣為南新縣<ref>[[晉書|晉書]]地理志</ref>。[[晉愍帝|晉愍帝]][[建兴_(晋愍帝)|建興]]二年（314年），帶方郡沒入[[高句麗|高句麗]]。此後中原王朝在朝鮮半島的直接統治暫時中斷，直至[[唐代|唐代]]初年滅高句麗。

三韩与汉四郡有着广泛的贸易往来。朝鲜半岛各地都有[[汉朝|汉朝]]的钱币出土。

==與古朝鮮的關係==
中国史书《[[尚书大传|尚书大传]]》是最早记载[[商周|商周]]时代的移民[[箕子朝鲜|箕子朝鲜]]方国的文献，同时也是最早提及朝鲜的文献，据推断大约在公元前900年左右。朝鮮有史料記載的年代遠遠長過三韓，卫满朝鲜時曾經阻止三韓等部落朝貢[[漢朝|漢朝]]。

==與今韓國的關係==
1897年[[甲午戰爭|甲午戰爭]]清朝戰敗后，[[日本|日本]]与中国[[清朝|清朝]]签署[[馬关条約|馬关条約]]之後，中國承認朝鮮是自主独立國家，不再主張朝鮮之[[宗主國|宗主國]]，1897年，朝鮮王宣布建制稱帝，[[李氏朝鮮|李氏朝鮮]]将本国国号改为「[[大韩帝国|大韩帝国]]」，「韓」第一次從民間進入國號。 

1910年大日本帝国将[[韓國併合|韓國併合]]後、该地域的称呼被改回为「朝鮮」。之后的1919年，[[李承晩|李承晩]]（後来韩国的首任总统）、[[金九|金九]]等独立運動家流亡中國後成立了[[大韩民国临时政府|大韩民国临时政府]]，该[[流亡政府|流亡政府]]的名称采用了「大韓」的名称。

== 轶话 ==
《[[三国史记|三国史记]]》，《[[三国遗事|三国遗事]]》和《[[帝王韵记|帝王韵记]]》将马韩说成是[[高句丽|高句丽]]。此观点最早由[[新罗|新罗]]儒学和历史学者[[崔致遠|崔致遠]]提出的。

[[李氏朝鲜|李氏朝鲜]]晚期的“实学”学者则强调在地理上讲马韩应用[[百济|百济]]更为相关。

“三韩”一般为古代[[朝鲜半岛|朝鲜半岛]]南部的[[马韩|马韩]]、[[辰韩|辰韩]]、[[弁韩|弁韩]]之总称，后泛指[[朝鲜|朝鲜]]。但是在一些汉人史书或者小说笔记中也常指代[[辽东|辽东]]其他民族。

在[[唐朝|唐朝]]时一些投靠唐朝的[[高句丽族|高句丽族]]或者[[靺鞨|靺鞨]]将领也在历史上被称为“三韩贵种”，如[[李多祚|李多祚]]。[[辽|辽]][[开泰|开泰]]（公元1012—1020年）中，[[辽圣宗|辽圣宗]]伐[[高丽|高丽]]，以俘户置[[高州|高州]]，又以其中三韩遗员置[[三韩县|三韩县]]，属[[中京道|中京道]]。[[金|金]]属[[北京路|北京路]][[大定府|大定府]]，址在今之[[内蒙古|内蒙古]][[赤峰市|赤峰市]]东。[[顾炎武|顾炎武]]《日知录·外国·三韩》条谓：“今人谓辽东为三韩者，……原其故，本于天启初失辽阳以后，奏章之文遂有谓辽人为三韩者，外之也。今江人乃以之自称，夫亦自外也矣。”

“古营州”人，“三韩人”在辽金以后的中国东北文人笔下乃指辽东。清以来，常以“三韩”作辽东之代称。如[[曹寅|曹寅]]祖籍[[辽阳|辽阳]]，[[韩炎|韩炎]]《有怀堂文稿》卷六《织造曹使君寿序》谓“以余所见，三韩曹使君子清，乃诚善读书者”，可为例证。在清人所著[[满洲源流考|满洲源流考]]中，三韩被认为是三汗的讹传，认为三韩为满族人的来源之一，地理位置在“[[奉天|奉天]]东北，吉林一带，壤接朝鲜”，辽代也在此设立国[[三韩县|三韩县]]。

《[[日本書紀|日本書紀]]》中也将百濟、新羅、高句麗三国称呼为[[三韓|三韓]]。
