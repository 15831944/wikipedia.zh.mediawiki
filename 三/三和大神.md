'''三和大神'''是指栖身在中国[[广东省|广东省]][[深圳市|深圳市]][[龙华新区|龙华新区]]景乐新村北区三和人力市场的一群打工者<ref>{{Cite web |url = http://views.ce.cn/view/ent/201808/21/t20180821_30081638.shtml |title = 围观“三和大神”不该止于猎奇 |author =  |date = 2018年08月21日 |publisher = 中国经济网 |language = zh-cn |accessdate = 2018年9月7日  }}</ref>，他们以打短工或者日结<ref group="注">日结是指当日完成工作后当日结算工资</ref>为生，号称做一天可以玩三天。这些打工者有的甚至没有身份证<ref group="注">有的是被偷有的则是被自己以100-200元的价格卖掉</ref>，身负债务<ref group="注">通常是由于赌博或者网贷</ref>，与家人鲜有来往<ref>{{Cite web |url = http://www.bjnews.com.cn/wevideo/2018/08/20/500291.html |title = 传说中的深圳“三和大神” 日薪百元 工作一天玩三天 |author =  |date = 2018-08-20 |publisher =  新京报|language =zh-cn  |accessdate = 2018-9-7  }}</ref>。失去了身份证的大神，由于无法购票，甚至无法离开三和地区。三和大神的定义同时也有精神方面的因素，往往有着过一天算一天的豁达精神，不畏死，不惧穷、苦、脏。

由于三和人才市场治安混乱<ref>{{Cite web |url =http://news.163.com/17/0828/22/CSV7IE8E000187VE.html |title = 深圳重拳治理“三和大神” 有人身上携带20张身份证 |author =  |date =2017-08-28|publisher = 网易新闻|language =zh-cn  |accessdate = 2018-9-7  }}</ref>，2017年8月，三和人才市场被警方整治<ref>{{Cite web |url = https://www.thepaper.cn/newsDetail_forward_1781483 |title = 深圳三和劳务市场被整治，警方刑拘多人 |author =  |date = 2017-09-1 |publisher = 澎拜新闻|language =zh-cn  |accessdate = 2018-9-7  }}</ref><ref>{{Cite web |url = http://www.sznews.com/news/content/2017-08/28/content_17123519.htm |title = 招工诈骗、黑网吧……深圳不要这样的“三和” |author =  |date = 2017-08-28 |publisher = 深圳新闻网 |language = zh-cn |accessdate = 2018-9-7 |archive-url = https://web.archive.org/web/20170828231301/http://sznews.com/news/content/2017-08/28/content_17123519.htm |archive-date = 2017-08-28 |dead-url = yes }}</ref><ref>{{Cite web |url =https://www.jfdaily.com/news/detail?id=69708|title = “城中村整治最严模式”已半年，深圳再无三和“大神”？ |author =  |date =2017-10-31|publisher = 上观新闻|language =zh-cn  |accessdate = 2018-9-7  }}</ref>。

==术语==
* 挂逼：是指遇到了特别困难的情况，比如身无分文，有时也指身亡<ref>{{Cite web |url = http://news.163.com/17/0830/14/CT3I1J3E000187VE.html|title = 深圳“三和”纪事 |author =  |date =2017-08-30 |publisher = 网易新闻|language =zh-cn  |accessdate = 2018-9-7  }}</ref>。
** 挂逼麵：4元人民幣一碗的清汤麵
** 挂逼床：15元人民幣一晚的床位
* 团饭：在网上向他人乞讨以求吃一顿饭
* 大水：一种便宜且容量大的瓶装水，最便宜的為清藍牌大水，兩公升一大瓶約2元人民幣。

==产生原因==

# 无学历：由于没有高等教育的学历，或者虽然上了大学，但是学了冶金、材料、采矿等专业，无法从事较为体面、轻松的工作，如工程师、会计师等。
#* 很多大神没有高中学历，因此也无法就职于待遇较好的大厂，如鴻海。也即无法就职为长期、正式工。
# 无技术：由于没有技术，也无法从事技工类工种，如焊接工。
# 无资本：由于毫无投资资本，也就无法开店经营，如经营早餐店等。
# 无法承受恶劣的工作环境：与第一代农民工不同，大神们由于各种原因，无法承受长期、恶劣的工作环境。例：一天工作16个小时，一周工作6天，从事电子组装工作等。
#不幸：由于各种偶然或者不偶然的因素，失去了进一步向上的可能性，从而辗转流落到三和。例：由于打工而延误了入学，被取消大学学籍，不得不辗转流落到三和。
