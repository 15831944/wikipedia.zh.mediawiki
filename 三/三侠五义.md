《'''三侠五义'''》，原名《'''忠烈侠义传'''》，是[[清朝|清朝]]人根據說唱藝術家[[石玉崑|石玉崑]]之唱本改編而成的長篇[[章回小說|章回小說]]，共一百二十回。

清朝[[咸丰|咸丰]]年间，石玉崑口头创作《'''龙图公案'''》；其後有人節錄成書，題名《'''龙图耳录'''》。[[光緒|光緒]]年間，又有人加以改編後重新出版，書名《'''[[忠烈侠义传|忠烈侠义传]]'''》，又名《'''三侠五义'''》。「三俠」是指南俠[[展昭|展昭]]、北俠[[歐陽春|歐陽春]]、雙俠[[丁兆兰|丁兆兰]]、[[丁兆蕙|丁兆蕙]]兄弟（共算一俠）；“五义”則指鑽天鼠[[卢方|卢方]]、徹地鼠[[韩彰|韩彰]]、穿山鼠[[徐庆|徐庆]]、翻江鼠[[蒋平|蒋平]]、锦毛鼠[[白玉堂|白玉堂]]，这五鼠弟兄。

清朝学者[[俞樾|俞樾]]刪改部分章節，並认为原书中主要有七个侠客：除了上述四人外，尚有小俠艾虎、黑妖狐智化及小诸葛沈仲元，一共是七人，故重新再版時改名为《'''七侠五义'''》。

== 創作背景 ==
据[[說書|說書]]的艺人世代相传：石玉昆受[[禮亲王|禮亲王]]之托，以[[明清|明清]]以来流传的《[[包公案|包公案]]》故事为基础，根据王爷家挂着画中的[[钟馗|钟馗]]與[[兰花|兰花]]、玉猫、挂画钉子及家中的老鼠，构建了书中的主要正反派人物，以三侠五义辅保[[包拯|包拯]]除暴安良、惩奸除恶为情节主线编缀成书。石玉昆在书中用了很多中国传统学问，比如[[易经|易经]]、[[八卦|八卦]]、[[五行|五行]]、儒学等，分别以“仁”、“义”、“礼”、“智”、“信”塑造出性格迥异，但是又侠肝义胆的五鼠弟兄；另外书中还大量反映清朝中后期民俗。

== 内容简介 ==
[[包拯|包拯]]幼时，其二嫂几度加害未死。长大后入都会试，途中得识侠士[[展昭|展昭]]，共除金龙寺恶僧，至隐逸村助李员外擒歹徒。后经殿试中进士，点翰林，出任[[定远县|定远县]]令。在任上智断墨斗案、扇坠案、乌盆案，官声大振，却因審理烏盆案時用刑不慎夾死兇手而被上司革职。归故里十年，又进京求官，得丞相王苞举荐，任[[开封府|开封府]]尹。

在开封任上得主簿公孙策之助，又智断尸蛊案。恤民情，参奏国舅庞昱不法，乃奉旨下陈州查赈放粮。至三星镇破女头男尸案。至陈州捉刺客，铡[[庞昱|庞昱]]。放赈三年，陈州大治。草桥天齐庙遇落难李太后，得知[[宋真宗|真宗]]朝刘妃、[[郭槐|郭槐]][[狸猫换太子|狸猫换太子]]故事。包公送李-{后}-还朝，夜审郭槐，使沉冤得雪，升任丞相。

继而又断葛登云谋霸范仲禹妻害命案等；展昭金殿试艺，钦封四品护卫，赐封号御猫；茉花村遇双侠丁兆兰、丁兆蕙，与其妹月华比剑联姻。白玉堂三试颜查散<ref>《七俠五義》中「顏查散」作「顏春敏」。</ref>，结为兄弟。柳家庄颜查散蒙冤，白玉堂开封府寄柬留刀，包公明断此案。白玉堂大内忠烈祠题诗，杀恶太监郭安。“五鼠”闹东京。范仲禹奉旨审太师庞吉。白玉堂盗包公三宝，展昭入陷空岛得三宝，白玉堂归案受封。欧阳春太岁庄除马刚。众侠大破邓家堡，拿花蝴江冲，杭州知府倪继祖庵堂认母，私访霸王庄落难。艾虎出世。欧阳春等大破霸王庄，救太守，拿马强。姚成诬告欧阳春，白玉堂奉旨办案。智化盗冠，丁兆蕙栽赃，艾虎告状，五堂会审，四衣库首领马朝贤正法。卧虎沟沙龙救乡亲，众侠义大破黑狼山。施俊娶真假牡丹。

颜查散奉旨巡查襄阳王赵珏反情，三侠五义下襄阳，白玉堂三探冲霄楼，捐躯铜网阵。逆水寒潭蒋平捞取邓车盗走的钦差金印。侠义诈君山。 此后接《[[小五义|小五义]]》、《[[续小五义|续小五义]]》。

== 評書名家 ==
民国年间，评书名家[[王杰奎|王杰奎]]、[[金杰华|金杰华]]，[[金杰立|金杰立]]、[[刘杰谦|刘杰谦]]、[[马连登|马连登]]善说此书。[[王杰奎|王杰奎]]的《三侠五义》在电台播出时听众人数很多，一些人力车夫宁可不拉车挣钱也要在大喇叭下面听广播，王因此得到“净街王”的美名。其后三侠五义的名家还有傅阔增、李鑫荃等人。

== 《三侠五义》和《七侠五义》 ==
石玉昆说三侠五义很受欢迎。据当时报纸记载：他说书的场子“进园中一望书无座位”、观众“恨不能近身受得先生崇”。其口述内容部分被其弟子和听众记录成册，最初名曰《龙图耳录》，大约在同治以前出版。后来因为买的人太多，又多次再版，名称变换频繁，有叫《龙图公案》的，有叫《忠烈侠义传》的。清朝学者[[俞樾|俞樾]]买到此书，认为原书中主要有七个侠客：除了欧阳春、展昭，丁兆兰、丁兆蕙应拆为两个侠客，再加上书中的“暗侠”黑妖狐智化、“盗侠”小诸葛沈仲元、“小侠”艾虎一共是七人，重新再版定名为《七侠五义》。

== 三侠五义的影响 ==
作为中国最早出现的武侠作品之一，三侠五义对中国近代评书、武侠小说乃至文学艺术影响深远，称得上是开山鼻祖，由此掀起了各类武侠题材文学作品的高潮。此后武侠公案、[[短打评书|短打评书]]盛极一时，例如《[[五女七贞|五女七贞]]》、《[[永庆升平|永庆升平]]》，民国《[[三侠剑|三侠剑]]》、《[[雍正剑侠图|雍正剑侠图]]》等纷纷问世，清末民初亦有大量知识分子投身武侠小说创作，写了很多脍炙人口的佳作，比如[[王度庐|王度庐]]的《[[卧虎藏龙|卧虎藏龙]]》，[[还珠楼主|还珠楼主]]的《[[蜀山奇侠传|蜀山奇侠传]]》，一直到港台的[[金庸|金庸]]、[[古龙|古龙]]的武侠小说都在它的影响之下。

== 白眉大侠与三侠五义 ==
[[单田芳|单田芳]]说的白眉大侠跟三侠五义、[[小五义|小五义]]并无多大关系，只是沿用了其中部分人物。全套三侠五义可从包公出世说到小小五义，当年王杰奎在电台足可以说三年。一般人得不全，很多版本到了三分之一处冲霄楼、诈君山之后就各自为政了。单田芳录制的三侠五义，七杰小五义等评书涉及后半部小五义的内容不多，征讨宁夏国、复夺陷空岛根本没有，而是以白眉大侠徐良为主角另外敷衍出一套情节。

== 傳說的開封府刑具 ==
[[File:开封包公祠.JPG|thumb]]
[[File:The_Memorial_Temple_of_Bao_Zheng_in_Hefei_02_2012-06.JPG|thumb]]内的龍頭鍘、虎頭鍘和狗頭鍘]]
龍頭鍘（火龍鍘）、虎頭鍘、狗頭鍘（犬頭鍘），是[[戲曲|戲曲]]、[[小說|小說]]中，包拯所使用的[[刑具|刑具]]，置於[[開封府|開封府]][[公堂|公堂]]上。現在很多[[包公廟|包公廟]]都設有仿製品。在包公的治理之下，龍虎狗頭鍘鍘殺[[十恶_(法律)|十惡]]不赦的罪犯。

;龍頭鍘
罪大惡極、犯死刑的[[宗室|皇親]][[外戚|國戚]]、[[王公|王公]][[諸侯|諸侯]]。

;虎頭鍘
犯死刑的[[大臣|大臣]]、[[官吏|官吏]]。

;狗頭鍘
犯死刑的[[平民|平民]][[百姓|百姓]]。
