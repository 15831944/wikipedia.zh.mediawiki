'''三元桥站'''位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]]，是[[北京地铁|北京地铁]][[北京地铁10号线|10号线]]与[[北京机场轨道交通线|机场线]]的[[换乘站|换乘车站]]。10号线车站和机场线车站同时进行建设施工，2006年11月15日机场线车站主体结构率先完工<ref>[http://www.bjpopss.gov.cn/bjpssweb/n23072c51.aspx 机场快轨三元桥车站实现主体完工] {{webarchive|url=https://archive.is/20131125051136/http://www.bjpopss.gov.cn/bjpssweb/n23072c51.aspx |date=2013-11-25 }}，北京社科规划网站，2006-11-16。</ref>，2007年4月28日10号线车站主体结构完工<ref>[http://news.xinhuanet.com/fortune/2007-04/28/content_6041404.htm 北京地铁十号线车站主体结构工程全部完工]，新华网，2007年04月28日。</ref>，并于2008年7月19日随着10号线一期和机场线同时通车投入运营<ref>[http://www.chinanews.com/olympic/news/2008/07-19/1317659.shtml 北京地铁奥运支线机场线10号线三线今日开通]，中国新闻网，2008年07月19日。</ref>。

在最初的规划中，机场快速轨道线在[[三元桥|三元桥]]附近不设站，后来修改规划，据市规划委的批复方案，在三元桥增设了一站，可与10号线换乘。<ref>[http://beijing.qianlong.com/3825/2005/03/20/1860@2558565.htm 机场轨道增设三元桥站] {{webarchive|url=https://web.archive.org/web/20131202231702/http://beijing.qianlong.com/3825/2005/03/20/1860%402558565.htm |date=2013-12-02 }}，千龙网，2005-03-20。</ref>

== 位置 ==
三元桥站地处三元桥立交桥区东北角，站位位于平行的[[京密路|京密路]]与[[首都机场高速公路|首都机场高速公路]]间地下，机场线与10号线在三元桥站沿京密路与机场高速（东北-西南走向）基本呈平行状布置。

== 结构 ==
机场线与10号线三元桥车站均为地下二层车站，岛式站台设计，主体为框架结构，明挖施工<ref>{{cite journal|author=|title=三元桥站土方开挖施工方案|journal=}}</ref>。
在三元桥站10号线与机场线的站厅中部设置一条联络通道，要换乘机场线的乘客得从10号线站厅通过连接非付费区的双向通道进入机场线站厅，通道距离只有40米左右。因两线票制不同，故为非付费区换乘，乘客换乘需再入闸缴费。

== 装饰 ==
机场线车站设计空间突出飞行的流线型造型，以蓝色为主色调。<ref>[http://news.zhulong.com/read86733.htm 解读首都机场线之三元桥站]，筑龙网，2008年8月12日。</ref> 
10号线车站站厅装饰的世界地图图案上用不同颜色灯光反映世界能源的分布状况。

== 出入口 ==
车站设六个出入口，D口设有通地面垂直电梯。在三元桥站的 (D) 出入口，凤凰城外设有一块630平方米的自行车存车场。
