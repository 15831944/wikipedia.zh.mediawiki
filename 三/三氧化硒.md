'''三氧化硒'''（化学式SeO<sub>3</sub>）是[[硒|硒]](VI)的[[氧化物|氧化物]]。为无色潮解性[[晶体|晶体]]。

== 制备 ==
实验室通过用无水[[硒酸钾|硒酸钾]](K<sub>2</sub>SeO<sub>4</sub>)和[[三氧化硫|三氧化硫]]反应制备三氧化硒： 
K<sub>2</sub>SeO<sub>4</sub> + SO<sub>3</sub> → SeO<sub>3</sub> + K<sub>2</sub>SO<sub>4</sub> 

三氧化硒在热力学上不稳定，逐渐转变为[[二氧化硒|二氧化硒]]。<ref>翻译自：http://en.wikipedia.org/wiki/Selenium_trioxide</ref>

== 结构 ==
该固体由环状的四聚体Se<sub>4</sub>O<sub>12</sub>构成。在[[气相|气相]]游离出[[单体|单体]]。三氧化硒的[[分子构型|分子构型]]是平面三角形。三氧化硒是[[非极性分子|非极性分子]]，在硒和氧之间是双键。[[键角|键角]]都是120°，没有单独的[[未共用电子对|未共用电子对]]。

== 化学性质 ==
三氧化硒和硒酸和所有的硒酸盐一样，是强氧化剂。三氧化硒对热不稳定。

== 用途 ==
三氧化硒用来制造光电电池和太阳能装置。
