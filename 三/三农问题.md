'''三农'''问题特指[[中華人民共和國|中華人民共和國]]的[[农业|农业]]、[[农村|农村]]和[[农民|农民]]问题，这是一个从事行业、居住地域和主体身份的三位一体的问题。

“三农”作为一个[[概念|概念]]则由[[经济学家|经济学家]][[温铁军|温铁军]][[博士|博士]]于1996年正式提出<ref>[http://www.china-village.org/ReadNews.asp?NewsID=3261&newsnameID=20&newsname=%E5%A8%93%E2%95%85%E6%90%A7%E9%8D%90 中國農村網:三农问题：非不能也，而不为也 ——温铁军博士答记者问] {{webarchive|url=https://archive.is/20070928154644/http://www.china-village.org/ReadNews.asp?NewsID=3261&newsnameID=20&newsname=%E5%A8%93%E2%95%85%E6%90%A7%E9%8D%90 |date=2007-09-28 }}</ref>，自此渐渐被[[媒体|媒体]]和[[官方|官方]]广泛引用；2000年初，[[湖北省|湖北省]][[监利县|监利县]]棋盘乡党委书记[[李昌平|李昌平]]给[[朱镕基|朱镕基]][[总理|总理]]写信提出“农民真苦，农村真穷，农业真危险”<ref>[http://news.sina.com.cn/c/2003-03-11/0607941447.shtml 中国青年报:农业部长评说农民真苦 农村真穷 农业真危险]</ref>以及出版《[[我向总理说实话|我向总理说实话]]》后，2001年三农问题的提法写入文件，正式成为中華人民共和國理论界和官方决策层引用的[[术语|术语]]；[[中共中央|中共中央]]于2003年正式将“三农”问题写入工作报告<ref>[http://www.people.com.cn/BIG5/jingji/1037/3158573.html 人民網:特別策劃：中央一號文件再次求解“三農”問題]</ref>。

== 内容 ==
*'''农业问题'''：主要是农业生产经营的问题，具体而言就是如何实现农业产业化。集中表现为农业生产经营的市场化程度低，农产品价格波动性大；农业生产主要依靠小规模农户，难以获得规模经济；粮食安全问题始终不容忽视。
*'''农村问题'''：集中表现为户籍制度导致的城乡二元分割，城乡的经济、文化水平差异较大。形象比喻为中国的[[城市|城市]]像[[欧洲|欧洲]]，农村像[[非洲|非洲]]。
*'''农民问题'''：主要是农民收入低、城乡收入差距大，农民的整体文化素质较低，农民[[权利|权利]]得不到保障等。

“农村是中华民族的摇篮，农业是国民经济的基础，农民是我们的衣食父母。失去了这三者，也就失去了立国之本。”，認為[[三農問題|三農問題]]事關全局<ref>[http://www.snzg.cn/article/2010/0415/article_18237.html  破解“三农”：论新阶段的农民增收]</ref>

== 背景 ==
[[File:Village_-_South_of_Changjiang_River.jpg|thumb]]
'''[[城镇|城镇]]和[[乡村|乡村]]的收入差距增大'''。根据[[中华人民共和国国家统计局|国家统计局]]发布的统计公报，2006年[[中華人民共和國|中華人民共和國]]城镇居民人均[[可支配收入|可支配收入]]11759元，农村人均纯收入3587元，二者比例为3.28:1（2005年为3.22:1）；而1985年城镇居民人均[[工资|工资]]690元，[[农村|农村]][[人均纯收入|人均纯收入]]397元，二者差距为1.74:1，20年时间城乡收入差距扩大了85％。

自改革开放以来，农村地区实行的是统分结合的'''[[家庭联产承包责任制|家庭联产承包责任制]]'''，并开始推动[[农村产业化|农村产业化]]进程。同时中国农业产品开始面对外国农业大国的竞争，单一的家庭生产单位无力进行市场竞争。

[[邓小平|邓小平]][[南巡讲话|南巡讲话]]后，中国经济开始起飞，与同期中国总体经济的发展相比，第一产业的发展程度远不及第二、[[第三产业|第三产业]]发展。农民认为种田没有足够的收入，因而农村地区开始出现[[民工|农民进城打工]]的现象，但城市并没有足够的就业岗位提供，从而形成大量集聚在城内而没有工作的农民工群体。随之而来的就是农村地区生产积极性下降，农业投入开始减少，引发更多的农民进城“找工作”，形成恶性循环。由于农业的发展将直接影响国家经济的正常发展，而农村人口占中国总人口的3/4，他们的总体生活情况是中国总体生活情况的最主要体现。农村的发展将是实现中国共产党提出的“三步走”战略最大的挑战，从而使三农问题成为当前社会中显著的问题。

== 三農問題的衍生影響 ==
在社會發展中的的發展差異使[[貧富懸殊|貧富懸殊]]差距擴大。在城市化的農地轉建地發展過程中，農民領取補償款和安置房後轉化非農業人口在城鎮謀生，變成市民，但由於生長環境和教育水準、思維方式還是有別於城市居民，有許多格格不入現象三農問題成為全國性的社會現象。

關於中國農村二十一世纪初的研究報告《来自中国社会底层的报告》，報導記錄[[大别山区|大别山区]]农村土地失衡、环境污染、乱摊派、人口拐卖、村匪地霸、封建迷信、製假售假、吃祭食、媒婆帮等使农民受害的社会现象，作者呼吁，[[中国改革开放|中国改革开放]]20多年来，经济取得进步，但必须重視农村问题的严重性，牢固农业大国的基础<ref name="叶照青朱大印2004">{{cite book|author1=叶照青|author2=朱大印|author3=耿昌军|title=来自中国社会底层的报告|url=http://book.cnxianzai.com/books/695/index.shtml|accessdate=12 April 2014|year=2004|publisher=长江文艺出版社|isbn=9787535427229|deadurl=yes|archiveurl=https://web.archive.org/web/20140413144622/http://book.cnxianzai.com/books/695/index.shtml|archivedate=2014年4月13日}}</ref>。

2017年起[[鬥魚TV|鬥魚TV]]直播平台開闢了公益性質的正能量區「新三農」板塊，透過直播科技增進民眾尤其是年輕世代看待三農的視角。<ref>[http://hb.people.com.cn/n2/2017/0909/c337099-30714090.html “网络直播”促三农发展]</ref>

== 解决措施 ==
*'''推行[[农村税费改革|农村税费改革]]'''。2003年3月，为了减轻农民负担，促进农民收入恢复增长，推进农村经济持续发展和农村社会全面进步，解决制约中国发展的农业生产落后的瓶颈问题，中国政府在原有的于2000年首先在[[安徽|安徽]]进行的农村税费改革试点的基础上，发出《国务院关于全面推进农村税费改革试点工作的意见》的文件。
*'''推行[[新农村建设|新农村建设]]目标'''。2005年10月11日[[中国共产党|中共]]第十六届中央委员会第五次全体会议通过的《中共中央关于制定国民经济和社会发展第十一个五年规划的建议》提出了“'''[[新农村建设|建设社会主义新农村]]'''”目标，其核心内容是发展农业和建设农村，增加政府对[[农业|农业]]和[[农村|农村]]投入，改善[[基础设施|基础设施]]包括乡村道路建设，强调以工促农、以城带乡，基本建立[[农村合作医疗制度|农村合作医疗制度]]，巩固九年制[[义务教育|义务教育]]，对农村[[学生|学生]]免收杂费。
**2005年11月29日至12月1日[[中国共产党中央委员会|中共中央]]经济工作会议，确定2006年总体工作要求中，提出“着重推进社会主义新农村建设”的要求；
**2005年12月23日国务院常务会议决定，2006年开始全部免除西部地区农村义务教育阶段学生学习杂费，2007年扩大到中部和东部地区；
*'''取消[[农业税|农业税]]'''。2005年12月29日，十届[[全国人大常委会|全国人大常委会]]第十九次会议经表决决定，[[s:中华人民共和国农业税条例|农业税条例]]自2006年1月1日起废止，中国[[农民|农民]]告别有2600年历史的“皇粮国税”。
*'''推出“[[农民工|农民工]]”维权方案'''。2006年1月18日国务院第122次常务会议原则上通过了《[[s:国务院关于解决农民工问题的若干意见|国务院关于解决农民工问题的若干意见]]》。

== 城乡二元体制改革 ==
[[File:A_man_with_two_oxen_in_Yangshuo,_China.jpg|thumb]]
加大行政手腕力度，控制农业生产资料的价格，采用新政策保证农民最低收入。解决城市[[农民工|农民工]]歧视现象，安排适合农民工的工作岗位。向农村地区推行'''产业化'''调整，鼓励成立'''[[乡镇企业|乡镇企业]]'''就地解决农民就业问题。推行小[[城镇化|城镇化]]，采用减少农民的数量的方法来解决农民问题。
