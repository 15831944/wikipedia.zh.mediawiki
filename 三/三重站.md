'''三重站'''位於[[台灣|台灣]][[新北市|新北市]][[三重區|三重區]]，為[[台北捷運|台北捷運]][[中和新蘆線|中和新蘆線]]與[[桃園捷運|桃園捷運]][[桃園捷運機場線|機場線]]之交會[[捷運車站|捷運車站]]。

==車站概要==
三重站位於捷運路與後埔街（疏洪東路）口的捷運路地下，接近捷運路與重新路、重陽路的交叉口，次地名為「過田」，車站代碼為'''O15'''。鄰近二重疏洪道東側雖未位於三重鬧區中央地帶，但由於此處作為桃園捷運系統機場線[[轉乘站|轉乘站]]之故，故命名為「三重站」，[[台語|台語]]廣播唸作「'''三重埔'''」（Sann-tîng-poo／Saⁿ-tēng-po·）<ref>http://www2.cna.com.tw/News/aLOC/201201110291.aspx {{Wayback|url=http://www2.cna.com.tw/News/aLOC/201201110291.aspx |date=20120114015349 }} 新莊線三重站 台語改三重埔</ref>。

桃園捷運機場線原計畫於三重福音街與後埔街（疏洪東路）口，後埔街（疏洪東路）上設立福音站，但隨中和新蘆線的設立，福音街更名為捷運路，且台北捷運將本站命名為三重站，故一併變更為此名；車站代碼為'''A2'''。啟用後已經成為[[三重區|三重區]]民眾前往[[台北車站|台北車站]]最快的交通方式，可不必再從[[中和新蘆線|中和新蘆線]]轉乘[[淡水信義線|淡水信義線]]。

== 車站構造 ==
成為高架{{rint|tw|tym}}{{rint|taoyuan|airport}}（[[桃園機場捷運|桃園機場捷運]]）與地下{{rint|tw|trts}}（[[台北捷運|台北捷運]]）交會車站，共有4個出入口。
*{{rint|tw|trtc}}[[台北捷運|台北捷運]]{{中和新蘆線}}中和新蘆線：地下四層車站，站體長221.4公尺，寬21.55公尺，深度19.2公尺，一個[[島式月台|島式月台]]<ref>http://www2.dorts.gov.tw/human/artnet/arch_xinzhuang.htm {{Wayback|url=http://www2.dorts.gov.tw/human/artnet/arch_xinzhuang.htm |date=20101112035230 }} 捷運車站建築設計
後續路網 新莊線-縣轄段各車站透視圖說明</ref>。
*{{rint|tw|tym}}[[桃園捷運|桃園捷運]]{{rint|taoyuan|airport}}[[機場線|機場線]]：是一座架設在河堤上、樓高四層設有一座島式月台之高架車站，站體長約120公尺，寬約19公尺，與台北捷運所屬站體共用一處出入口，站內設有3座無障礙電梯。目前第1月台為普通車往[[環北站|中壢]]方向、第2月台為普通車往[[臺北車站_(桃園捷運)|台北]]方向，直達車與普通車共用一條軌道，但直達車會快速通過此站。

=== 車站出口 ===
三重站出口1位於捷運新蘆線西端，和機場捷運三重站用天橋連接，出口2、3位於車站東端，[[無障礙|無障礙]]電梯位於出口1。
* 出口1A：大臺北都會公園.幸福水漾公園([[桃園捷運|桃園捷運]][[桃園國際機場捷運|機場線]]出入口，可透過非付費區連通至新蘆線)
* 出口1：疏洪東路一段[[File:Accessibility-directory.svg|20px]](捷運新蘆線、機場線共用出入口)
* 出口2：重新路四段(捷運新蘆線出入口，可透過非付費區連通至機場線)
* 出口3：捷運路(捷運新蘆線出入口，可透過非付費區連通至機場線)

== 歷史 ==
[[中正機場捷運|中正機場捷運]]計畫中，原預計站名為'''福音'''（取自捷運路舊名「福音街」）。政府收回自辦後，調整桃園捷運機場線站名與交會的台北捷運新莊線上的車站相同。
*2012年1月5日：台北捷運新莊線三重站隨著[[輔大站|輔大]]至[[大橋頭站|大橋頭]]段正式通車而啟用<ref>[http://www.trtc.com.tw/ct.asp?xItem=14302580&ctNode=24508&mp=122031 台北大眾捷運股份有限公司 ─ 郝龍斌、朱立倫共同宣布台北捷運新莊線大橋頭站至輔大站將於元月5日正式通車，並比照蘆洲線模式，實施1個月的試乘優惠措施] {{webarchive|url=https://web.archive.org/web/20120108150737/http://www.trtc.com.tw/ct.asp?xItem=14302580&ctNode=24508&mp=122031 |date=2012-01-08 }}</ref><ref>[http://www2.cna.com.tw/News/FirstNews/201201020021.aspx 新莊線 5日下午2時通車]</ref>。
*2013年6月29日：新莊線全線通車至[[迴龍站|迴龍站]]。
*2017年3月2日，隨著機場捷運「[[臺北車站_(桃園捷運)|臺北車站]]—[[環北站|環北]]」段正式通車而啟用<ref name="通車營運計畫" />。
