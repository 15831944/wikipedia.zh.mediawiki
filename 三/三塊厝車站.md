'''三塊厝車站'''位於[[高雄市|高雄市]][[三民區|三民區]]的[[三塊厝_(高雄市)|三塊厝]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]（及過去[[屏東線|屏東線]]）的[[鐵路車站|鐵路車站]]，舊站房目前為高雄市市定古蹟。配合[[高雄市區鐵路地下化計畫|高雄市區鐵路地下化計畫]]新建車站，於2018年10月14日重新啟用。

== 車站構造 ==
===地面舊站===
*[[台灣日據時期|日據時期]]留下的木造站房一座。
*[[車站月台|月臺]]一座（[[屏東線|屏東線]]上下行單線交會[[島式月臺|島式月臺]]）。

===地下化新站===
*[[岸式月台|岸式月台]]（[[縱貫線_(南段)|縱貫線]]）東西正線通過，無待避線。
*地下化入口採木造日式風格，向舊站站體致敬。

== 保存狀況 ==
*僅存一木造站房及尚未拆除的軌道。(目前因為進行[[高雄市區鐵路地下化計畫|高雄市區鐵路地下化]]工程，[[縱貫線_(南段)|縱貫線]]正線以外的軌道均已拆除)

== 利用狀況 ==
*在今日[[高雄車站|高雄車站]]設立前的[[台灣日治時期|日治時期]]，此站為高雄中學校（今[[高雄市立高雄高級中學|高雄市立高雄高級中學]]）師生及附近居民主要門戶。
*後因高雄車站自港邊遷至今址，距本站過近而客運大幅下滑，最後步入廢站命運。
*此站客運業務位於屏東線上。亦即[[縱貫線_(南段)|縱貫線]]列車不停靠此站，亦不設月台。
*配合[[高雄市區鐵路地下化計畫|高雄鐵路地下化]]及[[臺鐵捷運化|臺鐵捷運化]]，已於2018年10月14日在縱貫線恢復客運。
*位於三德西街90巷內，介於[[自立路_(高雄市)|自立陸橋]]與[[中華路_(高雄市)|中華路]]車行地下道之間。
* 目前為簡易站。
* 停靠狀態：[[臺鐵區間車|區間車]]皆停靠。
* 開放使用[[一卡通_(台灣)|一卡通]]、[[悠遊卡|悠遊卡]]及[[Icash|icash 2.0]] 付費。

== 歷史 ==
*1908年2月1日：設「臨時停車場」。
*1941年6月21日：升為三等站。
*1986年9月26日：廢除。
*2018年10月14日：隨著[[高雄市區鐵路地下化計畫|高雄市區鐵路地下化]]工程完工而啟用。
