'''三氯化磷'''，分子式：{{磷}}{{氯}}{{Sub|3}}。一种无色透明液体，沸点76℃，在潮湿空气中能水解成[[亚磷酸|亚磷酸]]和[[氯化氢|氯化氢]]，发生白烟而变质，须密封贮藏。<br />
: PCl<sub>3</sub> + 3H<sub>2</sub>O → H<sub>3</sub>PO<sub>3</sub> + 3 HCl

遇[[乙醇|乙醇]]和水起水解反应，<br />

遇氧能生成[[三氯氧磷|三氯氧磷]]。<br />

:2PCl<sub>3</sub> + O<sub>2</sub> → 2 POCl<sub>3</sub>

在水存在下，可与[[甲醇|甲醇]]反应生成[[亞磷酸二甲酯|亞磷酸二甲酯]]。<br />

反應分兩步進行：<br />
1.PCl<sub>3</sub> + 3H<sub>2</sub>O → H<sub>3</sub>PO<sub>3</sub> + 3HCl <br />
2.H<sub>3</sub>PO<sub>3</sub> + 2CH<sub>3</sub>OH → HOP(OCH<sub>3</sub>)<sub>2</sub> + 2H<sub>2</sub>O <br />

總反應為<br />

PCl<sub>3</sub> + 2CH<sub>3</sub>OH + H<sub>2</sub>O → HOP(OCH<sub>3</sub>)<sub>2</sub> + 3HCl<br />
