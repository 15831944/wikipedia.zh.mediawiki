'''三江侗族自治县'''是[[中国|中国]][[广西壮族自治区|广西壮族自治区]][[柳州市|柳州市]]所辖的一个[[自治县|自治县]]，地处湘、桂、黔三省（区）交界地。

==历史==
北宋[[崇宁|崇宁]]四年（1105年）于[[融水县|融水县]]三口寨置怀远军，后置怀远县，治所在今[[丹洲镇|丹洲镇]]。后几经兴废。明[[洪武|洪武]]十三年（1380年）复县。1914年因与安徽怀远县重名，更名为三江县。1949年11月间，[[中国人民解放军第四野战军|中国人民解放军第四野战军]]攻占三江县<ref>{{Cite web |url=http://www.people.com.cn/GB/historic/1122/3974.html |title=1949年11月22日 桂林解放 |accessdate=2014-09-05 |author= |date= |work= |publisher=人民网|language=Zh-hans }}</ref>。1952年12月3日成立县级侗族自治区，1955年9月改为侗族自治县。<ref>{{cite web|title=历史沿革|author=|url=http://www.sjx.gov.cn/structure/zjsj/lsyyzw_2139_1.htm|publisher=三江县政府网站|deadurl=yes|archiveurl=https://web.archive.org/web/20140905050841/http://www.sjx.gov.cn/structure/zjsj/lsyyzw_2139_1.htm|archivedate=2014-09-05}}</ref>

== 行政区划 ==
三江侗族自治县辖3个[[镇|镇]]、9个[[乡|乡]]、3个[[民族乡|民族乡]]：
* 镇：[[古宜镇|古宜镇]]、[[丹洲镇|丹洲镇]]、[[斗江镇|斗江镇]]
* 乡：[[林溪乡|林溪乡]]、[[和平乡|和平乡]]、[[老堡乡|老堡乡]]、[[程村乡|程村乡]]、[[梅林乡|梅林乡]]、[[洋溪乡|洋溪乡]]、[[良口乡|良口乡]]、[[独峒乡|独峒乡]]、[[八江乡|八江乡]]
* 民族乡：[[同乐苗族乡|同乐苗族乡]]、[[高基瑶族乡|高基瑶族乡]]、[[富禄苗族乡|富禄苗族乡]]

==地理==
总面积为2454[[平方公里|平方公里]]。地处[[东经|东经]]108°53′—109°52′，[[北纬|北纬]]25°22′— 26°2′，湘、桂、黔三省（区）交界地。县城[[古宜镇|古宜镇]]距离柳州市203公里，距[[桂林市|桂林市]]167公里。

境内最高峰为西南部与融水交界处的白云山，海拔1448米。

境内主要河流有[[融江|融江]]（老堡乡上游称[[都柳江|都柳江]]）及其支流。

===气候===
属[[亚热带|亚热带]]南岭湿润气候区，山地谷地气候区。

==交通==
* 铁路：[[焦柳铁路|焦柳铁路]]（境内设有[[三江县站|三江县站]]等火车站）、[[贵广客运专线|贵广客运专线]]
* 公路：[[209国道|209国道]]和[[321国道|321国道]]在境内交汇；[[三江至北海高速公路|三江至北海高速公路]]（桂高速S31，三北高速）起于此

==人口==
2010年[[第六次全国人口普查|第六次全国人口普查]]三江侗族自治县总人口367707人，常住人口297244人。<ref>{{cite web |title=柳州市2010年第六次全国人口普查主要数据公报  |author= |url=http://www.gxtj.gov.cn/tjsj/tjgb/201107/t20110715_1073.html |publisher=广西壮族自治区统计局 }}</ref>

主要少数民族有[[侗族|侗族]]、[[苗族|苗族]]、[[瑶族|瑶族]]等。

==经济==
三江经济以农业为主，工业不发达。近年来大力发展旅游业等第三产业。

==风景名胜==
三江是侗族的主要聚居区，有大量的侗族特色建筑如风雨桥和鼓楼，其中林溪乡的[[程阳永济桥|程阳永济桥]]、独峒乡的[[岜团桥|岜团桥]]、八江乡的[[马胖鼓楼|马胖鼓楼]]已被列为[[全国重点文物保护单位|全国重点文物保护单位]]。位于良口乡的[[和里三王宫|和里三王宫]]是一座侗族庙宇，兼具汉、侗两族建筑特色，已被列为[[广西壮族自治区文物保护单位|广西壮族自治区文物保护单位]]。

三江老县城在丹洲镇，始建于明代的[[丹洲古城|丹洲古城]]仍有一些保存较好的古迹，也被列为广西壮族自治区文物保护单位。
