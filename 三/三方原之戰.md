'''三方原之戰'''是[[元龟|元龟]]三年十二月二十二日（1573年1月25日）在[[日本|日本]][[三方原|三方原]]（今[[濱松市|濱松市]]內）爆發的[[戰爭|戰爭]]，兩軍的统帅為[[德川家康|德川家康]]和[[武田信玄|武田信玄]]。这是武田信玄以[[上洛|上洛]]為目的的一場戰爭。武田軍最終取得勝利，繼續往西前進，但是中途武田信玄病情惡化，在回歸领地[[甲府|甲府]]（今[[山梨县|山梨县]]）途中病死。

== 前端 ==
在由[[織田信長|織田信長]]扶植的[[室町幕府|室町幕府]]將軍[[足利義昭|足利義昭]]決意鏟除信長後，足利義昭以外交結交近畿週邊地區的有力[[大名|大名]]組成了[[信長包圍網|信長包圍網]]。義昭多次要求信玄上洛，向織田軍攻擊。可是信玄沒有實質行動，卻花了兩年的時間部署外交，以下就是例子。

*在[[北條氏康|北條氏康]]死後，與後繼者[[北條氏政|北條氏政]]修補關係。
*此外為免北條出現異心，對[[常陸國|常陸國]]大名[[佐竹義重|佐竹義重]]發信文書。
*利用[[石山本願寺|石山本願寺]]的[[一向一揆|一向一揆]]（信玄的正室[[三條夫人|三條]]的妹妹是[[本願寺顯如|本願寺顯如]]的妻子。）牽制越中地區北陸大名[[上杉謙信|上杉謙信]]。
*同時催促[[淺井長政|淺井長政]]、[[朝倉義景|朝倉義景]]、[[三好三人眾|三好三人眾]]等大名出兵，準備攻擊織田。

== 出陣 ==
武田軍在十月九日在[[甲府|甲府]]出發，兵分三路。開始向[[織田信長|織田信長]]的同盟[[德川家康|德川家康]]的軍队攻擊。第一隊為信玄的本隊，兵力二萬二，當中的二千是北條方增援的部隊。第二隊為[[山縣昌景|山縣昌景]]的部隊，五千兵力，攻擊東三河，攻落武節城、[[長篠城|長篠城]]後往[[遠江國|遠江國]]方向攻略(德川部署遠江約七千人，目前普遍算法[[山縣昌景|山縣昌景]]的五千人算三方原參戰軍，而遠江的德川軍卻不算？)。第三隊為[[秋山信友|秋山信友]]五千，攻擊[[美濃國|美濃國]]的岩村城。主力軍方面，前線將領[[馬場信春|馬場信春]]，率領五千兵力攻擊二俁城（在今[[濱松市|濱松市]]），以測試德川軍的戰力，可是其城池非常堅硬，一直無法突破。約兩個月後，才開始使用計謀破壞城內的供水系統，十二月十九日，二俁城守方開城投降。十二月二十一日（1573年1月14日），與[[山縣昌景|山縣昌景]]的部隊會合，並在祝田之坂佈陣，在此佈陣的目的是有利於武田軍前進，因為此路是往[[三河國|三河]]及[[遠江國|遠江]]的交匯，加上武田軍與[[濱松城|濱松城]]非常接近。

當時，根据德川軍的情报，武田軍的動向並非攻擊[[德川家康|德川家康]]的根据地[[濱松城|濱松城]]，而是西進[[京都|京都]]。當時陣中不少家臣向家康提議，讓武田軍向西前進，可是家康決定迎擊，他派遣了一萬一千的部隊（其中三千為織田軍的增援，[[佐久間信盛|佐久間信盛]]、[[瀧川一益|瀧川一益]]、[[平手汎秀|平手汎秀]]、[[林秀貞|林秀貞]]、[[水野信元|水野信元]]，[[林秀貞|林秀貞]]為織田家筆頭大老，加上另外四將僅能支援三千）。向祝田之坂前進。十二月二十二日早上，兩軍已經在三方原完成佈陣。

== 佈陣 ==
兩軍的佈陣如下：

* 武田軍：第一陣為[[小山田信茂|小山田信茂]]、山縣昌景以及[[內藤昌豐|內藤昌豐]]。第二陣為[[小幡信貞|小幡信貞]]、[[馬場信春|馬場信春]]及[[武田勝賴|武田勝賴]]。第三陣為武田信玄的本陣。陣形為魚鱗之陣。
* 德川軍：陣形為鶴翼之陣左翼為[[石川數正|石川數正]]、[[本多忠勝|本多忠勝]]、[[松平家忠|松平家忠]]及[[小笠原長忠|小笠原長忠]]；右翼為[[酒井忠次|酒井忠次]]、[[瀧川一益|瀧川一益]]、[[平手汎秀|平手汎秀]]及[[佐久間信盛|佐久間信盛]]。中央為大將德川家康。

== 開戰 ==
兩軍對峙了一段時間（當中兩軍實際的行動情況不明），武田軍部將[[小山田信茂|小山田信茂]]匯報德川軍的缺乏戰意。大概五時左右，根據《[[甲陽軍鑑|甲陽軍鑑]]》記載，小山田部（三千五百人）的投石隊以擲石的方法向守城石川隊攻擊，激怒德川軍，石川隊受到重創。石川隊撤退後，終於使[[德川家康|德川家康]]命令發動總攻擊，根據《[[甲陽軍鑑|甲陽軍鑑]]》跟《[[三河物語|三河物語]]》的紀錄，武田軍的[[山縣昌景|山縣昌景]]、[[小山田信茂|小山田信茂]]兩隊都被[[德川家康|德川家康]]擊退，但是兩軍鬥志和兵力有一定的差距，在[[武田勝賴|武田勝賴]]支撐住德川軍的進攻後，[[武田信玄|武田信玄]]回師迎擊，甚至包括負責運輸的甘利隊也參戰，进行全軍攻擊，取得了優勢。雙方只是戰鬥約兩小時左右，約下午六時德川軍開始撤退。當時，德川軍撤退得非常危險，[[德川家康|德川家康]]的替身[[夏目吉信|夏目吉信]]戰死，此外擔任家康殿軍的[[本多忠真|本多忠真]]戰死，具傳家康甚至在撤退時嚇得失禁脫糞，可見得家康非常狼狽。織田軍當中，大將[[平手汎秀|平手汎秀]]戰死。當信玄軍殺到[[濱松城|濱松城]]外，德川軍使用空城計，信玄軍見德川軍完全沒有防備，所有城門打開。[[武田信玄|武田信玄]]軍前線部隊还聽到了鼓聲。信玄恐防有詐，下令停止攻擊。結果，家康安然无事，避過了此劫。

戰後，武田軍之後沒有再攻打家康當時的居城[[濱松城|濱松城]]，在祝田之坂往西前進，向三河進攻，兩個月後在[[野田城之戰|野田城之戰]]擊退德川的守備之後，[[武田信玄|武田信玄]]病情惡化，武田主力軍回歸，回歸途中在[[信濃|信濃]]伊那郡死亡。

== 其他 ==
[[File:Mikatagaharasenekizou.jpg|thumb]]
織田的增援軍[[佐久間信盛|佐久間信盛]]因為在這場戰爭失去鬥志而沒有參戰，雖然信長沒有立即懲罰他，但是後來在1580年8月被[[織田信長|織田信長]]問罪，原因是他的部隊不加入戰鬥，導致信長另一名部屬[[平手汎秀|平手汎秀]]戰死，[[佐久間信盛|佐久間信盛]]的侄子[[佐久間盛政|佐久間盛政]]同樣被信長放逐。

在此戰後，傳說[[德川家康|德川家康]]下令畫師狩野探幽繪畫一幅愁眉苦臉的畫像，後世稱為「顰像」，提醒自己不要出現更難堪的情況，此畫像現存{{link-ja|德川美術館|徳川美術館}}。但已被證實此圖所繪乃是[[長篠之戰|長篠之戰]]的家康像，被敘述為三方原之戰後是1936年時新愛知新聞及[[大阪每日新聞|大阪每日新聞]]這兩家報紙的誤傳。
