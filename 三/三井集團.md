'''三井集團'''（{{jpn|j=三井グループ|hg=みついグループ|rm=Mitsui Gurūpu|f=y}}，{{lang|en|''Mitsui Group''}}），是一個[[日本|日本]]的企業联合体，乃由[[二戰|二戰]]前屬[[三井財閥|三井財閥]]的各企業在战后組合而成的联盟，別稱「'''二木会'''」。

== 概要 ==
在二戰前曾經是日本最大[[財閥|財閥]]的[[三井財閥|三井財閥]]，在[[日本戰敗|日本戰敗]]后與其他財閥一同被[[GHQ|GHQ]]命令解散。解散后，與同時是舊財閥企業重組而成的[[三菱集團|三菱集團]]和[[住友集團|住友集團]]相比，三井集團的結成時間相對較晚、加上由於本來作爲集團資金最大來源的[[三井銀行|三井銀行]]和[[帝国銀行|帝国銀行]]的分裂，使集團規模大爲縮小。同時，集團内的企業也沒有趕上日本[[日本战后经济奇迹|高度經濟成長期]]的[[重化學工業|重化學工業]]化，使得集團的地位與戰前的財閥相比大爲減弱。就表面上來説，相比其他幾大集團，現在的三井集團内各企業的關聯性比較鬆散。雖然如此，由於資本上的關係，諸如[[豐田汽車|豐田汽車]]和[[東芝|東芝]]，以及因與東芝而建立起關係的[[IHI|IHI]]等公司依然屬於本集團。同時，集團内部也保留著一定的歷史傳承，如三井系列企業管理層專用的俱樂部，甚至在三井記念病院内還保留著内管理層專用的特別病室。

近年的動向包括有，三井礦山因接受[[新日本製鐵|新日本製鐵]]（現：[[新日鐵住金|新日鐵住金]]）的企業重建支援，改名為「日本COKES工業」而脫離了本集團。在2008年4月，作爲元三井財閥始祖之一「越後屋呉服店」後代的[[三越|三越]]，與大股東為[[三菱東京UFJ銀行|三菱東京UFJ銀行]]（舊[[三菱銀行|三菱銀行]]）的[[伊勢丹|伊勢丹]]合并組成了「三越伊勢丹控股公司」。另一方面，在金融界則有如[[さくら銀行|さくら銀行]]（舊三井銀行）和[[住友銀行|住友銀行]]合併，組成了現時日本銀行界三巨頭之一的[[三井住友銀行|三井住友銀行]]、和因接受了豐田的資本支援而合并的[[三井住友海上火災保險|三井住友海上火災保險]]、[[愛和誼損害保險|愛和誼損害保險]]和[[日生同和損害保險|日生同和損害保險]]等(現MS&AD保險控股)。與此同時，還有如[[富士膠片|富士膠片]]和[[TBS|TBS]]等新加入的成員。

值得一提的是，三井集團旗下其中一派的二木会，成員企業大多為沒有冠名[[三井|三井]]的傍系企業，現在如何加强與這些企業的聯係也成爲了集團今後的主要課題。

== 商標 ==
[[File:Mitsui_Bussan_logo.svg|thumb]]
三井集團的商標為一呈菱形狀的「井」字内包圍著一個「三」字<ref>企業によっては「三」の文字が直線でなく筆文字であったり、線の太さが異なったりする。なお三井物産のみ丸がない。[http://www.mitsuipr.com/history/column/04/index.html コラム「三井を読む」第4回　商いの大理想を新たな店章に託して]を参照。</ref>。這個標記原則上雖然被大部分冠名「三井」的企業使用，但也有如三井住友銀行、三井住友海上和[[商船三井|商船三井]]<ref>前身の一つ「三井海上火災」も、大正海上から社名変更後も井桁マークを使用していなかった。</ref>等因與其他集團公司合并而沒有使用這個標記的特例。而最近，三井廣報委員会也沒有使用這個標記。

同時，即使是冠名為「三井」的企業當中，與三菱集團的象徵商標「三菱」和住友集團的象徵商標「井紋」相比，一直以來三井集團商標的采用率也沒有前兩者高<ref>例、[[三井銀行|三井銀行]]の八重桜の中に井桁に「三」など。</ref>。

== 三井集團主要各社 ==
三井集團企業數量龐大，在衆多企業中，舊[[三井銀行|三井銀行]]、[[三井物産|三井物産]]和舊[[三井礦山|三井礦山]]三家企業曾經被稱為「三井集團御三家」。戰后由于三井本社被[[三井不動産|三井不動産]]吸収合併，後者取代舊三井礦山成爲了新的御三家成員。
