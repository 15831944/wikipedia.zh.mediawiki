'''三埤車站'''是位於[[臺灣|臺灣]][[高雄市|高雄市]][[路竹區|路竹區]]、隸屬於[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]的一座[[鐵路車站|鐵路車站]]，現在已經遭到[[廢棄鐵路車站|廢止]]。

== 歷史 ==
1953年3月1日時，[[臺灣鐵路管理局|臺灣鐵路管理局]]為了便利旅客乘車而在[[路竹車站|路竹車站]]與[[岡山車站|岡山車站]]間增設三埤車站，該車站等級為[[招呼站_(台鐵)|招呼站]]並且只停靠汽油客車。而在1950年時，臺灣省議會議員[[朱萬成|朱萬成]]、[[黃占岸|黃占岸]]、[[黃國秀|黃國秀]]和[[朱清景|朱清景]]等人則提議在三埤車站設置簡易候車室以及安排售票員等措施<ref>{{Cite book | author = [[臺灣省議會|臺灣省議會]] | title = 決議案辦理情形 | location = 臺灣霧峰 | publisher = 《台灣省議會公報》第二卷第十九期 | date = 1998年12月21日 | pages = 第824頁 | ISBN =  | accessdate = 2014年5月3日 | url =  | language = zh-hant | quote =  }} </ref>。然而由於營運期間由於乘客人數相對稀少以及搭配[[台鐵柴油客車|台鐵柴油客車]]停駛之政策，大約在1970年至1971年時三埤車站遭到裁撤，不過廢站後部份月臺仍然被保存下來<ref>{{Cite web |url = http://trstour.com/san-pi.htm |title = 三埤車站 |author =  |date =  |publisher = 驛站之旅 |language = zh-hant |accessdate = 2014年5月3日 |quote =  }}</ref>。

== 使用 ==
三埤車站營運時為不派遣鐵路車站員工駐守以及協助辦理行車業務的招呼站，並且僅供汽油客車和[[柴聯車|柴聯車]]停靠。三埤車站本身是一個[[地面車站|地面車站]]，而主要結構為兩座分別位於[[平交道|平交道]]兩側的[[側式月台|側式月台]]，到了今日儘管遭到裁撤後仍然能夠分辨水泥磚造的月臺遺跡。另外三埤車站的月臺還設計成位於平交道兩側，其中海側月臺位於平交道北方、而山側月臺則位於平交道南方<ref>{{Cite web |url = http://ticket.myweb.hinet.net/station/a04,03-10.htm |title = 三埤站 |author =  |date =  |publisher = 看橋工房 |language = zh-hant |accessdate = 2014年5月3日 |quote =  }}</ref>。
