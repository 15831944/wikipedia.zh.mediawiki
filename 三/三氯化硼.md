'''三氯化硼'''（{{硼}}{{氯}}{{Sub|3}}），是一种有强烈臭味的[[气体|气体]]，用于制各种硼化合物，也用作有机合成[[催化剂|催化剂]]、[[硅酸盐|硅酸盐]]分解时的助溶剂以及对钢铁进行硼化等。

== 備製 ==

硼和[[鹵素|鹵素]]反應產生硼的三[[卤化物|卤化物]]。但在工業上，三氯化硼是以[[氧化硼|氧化硼]]與碳在500 ℃中直接氯化製得。

: B<sub>2</sub>O<sub>3</sub> + 3 C + 3 Cl<sub>2</sub> → 2 BCl<sub>3</sub> + 3 CO
