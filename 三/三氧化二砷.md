'''三氧化二砷'''（[[學名|學名]]：{{lang|en|Arsenic trioxide}}，藥品名：{{lang|en|Asadin}}），俗稱'''砒霜'''、白砒、鹤顶红<ref name="j-tw-20091231-99">[http://www.taiwan-pharma.org.tw/JTP/101/146-150.html 砒霜的臨床應用] {{webarchive|url=https://web.archive.org/web/20141012231054/http://www.taiwan-pharma.org.tw/JTP/101/146-150.html |date=2014-10-12 }}，高雄長庚紀念醫院藥劑部中藥組藥師 陳治諶、賴建璋、林口長庚紀念醫院中醫藥劑部藥師 楊榮季，《藥學雜誌電子報》99期，2009-12-31 </ref>，分子式As<sub>2</sub>O<sub>3</sub>，是最具商業價值的[[砷|砷]][[化合物|化合物]]及主要的砷[[化學|化學]]開始物料，也是最古老的[[毒物|毒物]]之一，無臭無味，外觀為白色[[霜|霜]]狀粉末，故稱砒霜。這是經某幾種指定的礦物處理過程所產生的高毒性副產品，例如採金礦、高溫蒸餾[[砷黃鐵礦|砷黃鐵礦]]（毒砂）並冷凝其白煙等。

== 化學特性 ==
三氧化二砷是[[兩性氧化物|兩性氧化物]]，但酸性超过碱性。溶于[[鹼性|鹼性]][[溶液|溶液]]會產生亞砷酸鹽；較難溶于[[酸|酸]]，但又會溶于[[浓鹽酸|浓鹽酸]]，生成三氯化砷。
: As<sub>2</sub>O<sub>3</sub> +  6 HCl  →  2 AsCl<sub>3</sub>  + 3 H<sub>2</sub>O
三氧化二砷會和[[氧化劑|氧化劑]]（例如[[臭氧|臭氧]]、[[過氧化氫|過氧化氫]]及[[硝酸|硝酸]]）起化学反应生成[[五氧化二砷|五氧化二砷]]（As<sub>2</sub>O<sub>5</sub>）。甚可以[[還原|還原]]為砷，也可以还原生成[[氢化砷|氢化砷]]（AsH<sub>3</sub>）。

== 製備 ==
三氧化二砷一般可以用兩種方法製備：[[氧化|氧化]][[三硫化二砷|三硫化二砷]]或[[水解|水解]][[三氯化砷|三氯化砷]]：
:<math> \rm 2 As_2S_3 + 9O_2 {=\!=\!=\!=} 2As_2O_3 + 6SO_2 </math>
:<math> \rm 2AsCl_3 + 3H_2O {=\!=\!=\!=} As_2O_3 + 6HCl </math>

== 分子結構 ==
四面體分子，在α-型、液態或氣態，是As<sub>4</sub>O<sub>6</sub>。

== 毒物學 ==
古時候常用砒霜做[[毒藥|毒藥]]，但由於技術上的限制，砒霜中會混有大量的[[硫|硫]]或[[硫化物|硫化物]]，呈现红色，故又称“鹤顶红”。[[銀|銀]]易與硫起化學變化生成黑色的[[硫化銀|硫化銀]]，因此常被用來試毒。由此可看出銀不過是檢驗出砒霜中的硫，而如今技術的進步，所製造的砒霜不再含硫，因此銀也無法準確檢驗砒霜。

三氧化二砷极易被[[呼吸系統|呼吸系統]]、[[消化系統|消化系統]]及[[皮膚|皮膚]]吸收。縱然是吸入它的粉末或[[氣體|氣體]]，還是皮膚接觸到它，已可以知道它的毒性。可通過[[甲基化|甲基化]]二甲胂酸及排[[尿|尿]]先消除一些[[半衰期|半衰期]]為一至二日的砷，但仍有相當的份量（在重複接觸時為30–40%）會併入骨、肌肉、頭髮、指甲及所有含有豐富角質素的組織中，需超過一個月才能消除。

通過攝食會發生急性砷中毒，第一症狀是出現消化系統問題：[[嘔吐|嘔吐]]、腹部疼痛及[[腹瀉|腹瀉]]常帶[[血|血]]。不致命的劑量也可使人[[痙攣|痙攣]]、[[心臟|心臟]][[血管|血管]]產生問題、[[肝臟|肝臟]]及[[腎臟|腎臟]][[發炎|發炎]]及血液不尋常凝結，跟隨着的是[[指甲|指甲]]上出現白線及脫髮等症狀。攝取低劑量也可出現肝臟和腎臟的問題，以及使[[皮膚|皮膚]][[色素|色素]]轉變。

吸入或接觸三氧化二砷，也是急性砷中毒的原因。急性砷中毒的第一症狀是[[呼吸道|呼吸道]]或皮膚受到嚴重刺激，其後便是長期的[[神經|神經]]問題。即使三氧化二砷溶液已稀釋，若觸及[[眼睛|眼睛]]，仍非常危險。

以下的人較易患上砷中毒：在工作上經常接觸三氧化二砷的（例如在金屬煉製廠中工作）、經常飲用含有大量砷的[[水|水]]（0.3–0.4 ppm）的居民，以及長期服用砷基藥物的病人。

三氧化二砷被視為[[人類|人類]]的[[致癌|致癌]][[物質|物質]]。在[[美國|美國]]、[[日本|日本]]和[[瑞典|瑞典]]，一個研究指出鑄[[銅|銅]]廠的工人比普通群眾患[[肺癌|肺癌]]的機會率高出六至十倍。無論是通過飲用的食水還是作醫療用途的藥物，長期攝食三氧化二砷都可導致[[皮膚癌|皮膚癌]]。同一研究指出若女性於工作時需接觸三氧化二砷或與鑄銅廠是比鄰，會產生[[生殖|生殖]]問題，例如：高流產率、剛出生的嬰孩比較輕、先天畸形等。

曾有網路謠言指稱，蝦類海產與[[維生素C|維生素C]]經混食後，會於體內自行合成砒霜而致人於死。但經計算，若要在體內合成足以造成急性中毒的劑量，所食用之蝦類海產與維生素C均須遠逾常人腸胃容量，故在实际上極不可能。最壞的結果是因慢性中毒而造成肝腫大，但其嚴重程度遠小於養殖海產體內的過量[[抗生素|抗生素]]所造成的問題。<ref>{{cite web|url=http://rumor.nownews.com/2007/11/12/515-80275.htm|archivedate=2015-04-28|title=維他命C配蝦子=吃砒霜？　專家一笑|4=網路追追追|5=NOWnews 今日新聞網 http://rumor.nownews.com/2007/11/12/515-80275.htm#ixzz3Yc2DnqHV|deadurl=yes|archiveurl=https://archive.is/20150428140017/http://rumor.nownews.com/2007/11/12/515-80275.htm}}</ref>

在[[歐盟|歐盟]]的「[[關於化學品註冊、評估、許可和限制法案|關於化學品註冊、評估、許可和限制法案]]」（REACH）中，已將三氧化二砷列為高度關注物質(SVHC)。<ref>[http://echa.Europa.eu/consultations/authorisation/svhc/svhc_cons_en.asp Deadlink] {{webarchive|url=https://web.archive.org/web/20111118220145/http://echa.europa.eu/consultations/authorisation/svhc/svhc_cons_en.asp |date=2011-11-18 }}</ref>

== 用途 ==
[[File:Arsenic_trioxide_with_label.JPG|right]]

=== 药劑 ===
# 製备砷基[[殺蟲劑|殺蟲劑]]的原料，包括[[亞砷酸鈉|亞砷酸鈉]]、砷酸鈉及二甲胂酸鈉。
# 制备[[农药|农药]]、杀鼠剂等。
# 製备配藥學及[[獸醫|獸醫學]]的產品，例如：{{tsl|en|Neosalvarsan|阿斯凡納明}}。

=== 工業 ===
# [[搪瓷|搪瓷]]及[[玻璃|玻璃]]的脫色劑，也能使玻璃、陶瓷产品更透明光滑。
# [[木|木]]的[[防腐劑|防腐劑]]。
# 在[[冶金學|冶金學]]中，它用于阻止[[氫|氫]]的再[[結合|結合]]。
# 製备[[元素|元素]]形式的砷、砷[[合金|合金]]及砷[[化合物半导体|化合物半導體]]的原料。

=== 醫學用途 ===
用作抵抗性急性前骨髓性[[白血病|白血病]]的[[化學療法|化學療法]]。這種藥名為Trisenox®，成份為三氧化二砷，是一次性的針劑；每劑容量為10[[毫克|毫克]]，[[稀釋|稀釋]]後作[[靜脈注射|靜脈注射]]。也可用于治療一些對其他[[藥物|藥物]]治療無反應的白血病患者。

製藥廠Cephalon把三氧化二砷命名為 Trisenox® 並售賣之。這种藥用作[[化療藥物|化療藥物]]醫治對一線藥物沒有反應的原發性作用。有人猜想[[三硫化二砷|三硫化二砷]]能引誘[[癌|癌]][[細胞|細胞]]自毀。因為砷有[[毒性|毒性]]，所以用此[[藥|藥]]的[[風險|風險]]非常高。

三氧化二砷也应用在中医和现代医学，也称为“白砷”，可以经[[呼吸道|呼吸道]]、[[皮肤|皮肤]]及[[消化道|消化道]]吸收。用以治疗对[[全反式维甲酸|全反式维甲酸]]和蒽环霉素化疗无反应或复发的急性早幼粒细胞白血病。微量时可治療皮膚症或改進血液，但0.06[[克|克]]是危險劑量，0.125-0.25[[克|克]]的三氧化二砷已足以致人於死地。

2010年，[[香港大學醫學院|香港大學醫學院]]發現，若劑量適當，砒霜的潛在毒性會解除，尤其是對心臟的副作用會大大減低，並成功把砒霜研發成為處方的口服藥物，已治愈超過一百名[[血癌|血癌]]病人。患者已接受此項治療，五年內存活率達七成。在此等病症中，口服砒霜已替代了骨髓移植作為最主要療法。口服砒霜還能應用於治療其他癌症，包括[[淋巴瘤|淋巴癌]]和[[多發性骨髓瘤|多發性骨髓瘤]]。<ref>[http://www.ke.hku.hk/eng/highlights/2010/trioxide],HKU Knowledge Exchange Highlights 2010]</ref>

[[牙科|牙科]]方面，极小剂量的三氧化二砷可用于杀死[[龋齿|龋齿]]内的牙神经。

1996年，[[科学_(期刊)|Science雜誌]]稱“這是繼[[全反式維甲酸|全反式維甲酸]]{{NoteTag|[[维甲酸综合症|维甲酸综合症]]}}之後，又一個令人感到震撼的發現”<ref>Mervs J.：Ancient Remedy Performs New Tricks。J Science, 1996;273(2): 578。</ref>。1999年，美國[[FDA|FDA]]批准As<sub>2</sub>O<sub>3</sub>治療複發性和難治性{{tsl|en|Acute promyelocytic leukemia|急性早幼粒細胞白血病}}（APL）、多發性骨髓瘤（MM）、[[骨髓增生异常综合征|-{zh-hans:骨髓增生异常综合征;zh-hant:骨髓增生異常綜合症;}-]]（MDS）。2002年，Asadin獲得臺灣衛生署核准上市。<ref name="j-tw-20091231-99" />
