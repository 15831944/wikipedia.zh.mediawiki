'''三色刺蝶魚'''，俗名'''美國石美人'''，為[[輻鰭魚綱|輻鰭魚綱]][[鱸形目|鱸形目]][[鱸亞目|鱸亞目]][[蓋刺魚科|蓋刺魚科]]的其中一個[[種|種]]。

== 分布 ==
本魚分布於西[[大西洋|大西洋]]區，包括[[美國|美國]]、[[墨西哥|墨西哥]]、[[宏都拉斯|宏都拉斯]]、[[尼加拉瓜|尼加拉瓜]]、[[貝里斯|貝里斯]]、[[哥斯大黎加|哥斯大黎加]]、[[巴拿馬|巴拿馬]]、[[加勒比海|加勒比海]]各島嶼、[[哥倫比亞|哥倫比亞]]、[[委內瑞拉|委內瑞拉]]、[[蘇利南|蘇利南]]、[[法屬圭亞那|法屬圭亞那]]、[[蓋亞那|蓋亞那]]、[[巴西|巴西]]等海域。

== 深度 ==
水深3~92公尺。

== 特徵 ==
本魚幼魚和成魚體色相同，只是顏色分布部位不同。幼魚體色為黃色，背鰭後部下面的側腹上有藍邊黑斑塊，覆蓋了成魚3/4的魚體，只有頭、胸、頸背部呈黃色。黑色的背鰭和臀鰭的邊緣為紅色及黃色，胸鰭、腹鰭及尾鰭呈黃色。背鰭硬棘14枚，軟條17~19枚；臀鰭硬棘3枚，軟條18~20枚。體長可達35公分。

== 生態 ==
本魚棲息於岩礁與[[珊瑚|珊瑚]]豐富的區域。稚魚常在[[火珊瑚|火珊瑚]]中出現。肉食性，以[[被囊類|被囊類]]、[[海綿|海綿]]、[[珊瑚|珊瑚]]與[[藻類|藻類]]為食。

== 經濟利用 ==
為高經濟價值的魚類，較少當食用魚。有報告指出其體內含有雪卡魚毒。
