'''三貂嶺車站'''位於[[台灣|台灣]][[新北市|新北市]][[瑞芳區|瑞芳區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[宜蘭線|宜蘭線]]、[[平溪線|平溪線]]的[[鐵路車站|鐵路車站]]。由於原本是為了平溪線與[[正線|正線]]匯集時的[[閉塞_(鐵路)|閉塞]]控制而設置的工作用車站，並不作為對外營運用途，三貂嶺車站是台灣唯一沒有道路可以直接抵達的鐵路車站，而是與鄰近的[[公路|公路]]隔著[[基隆河|基隆河]]相望。欲在本站搭車的旅客，必須自上游一段距離的[[鐵路橋|鐵路橋]]處利用沿著鐵路線旁的人行步道跨越[[河床|河床]]之後，徒步抵達本站。

== 車站構造 ==
*[[岸式月台|岸式月台]]兩座。
*車站建設在河岸與山壁間，腹地狹窄而無法建設天橋及地下道，往[[七堵車站|七堵]]、[[瑞芳車站|瑞芳]]的乘客須在站務人員引導下跨越軌道乘車。
*山側月台因山壁滲水終年潮濕。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。僅停靠[[臺鐵區間車|區間車]]。
*[[平溪線|平溪線]]於本站以南自[[宜蘭線|宜蘭線]]分歧。
*全台唯一無公路可抵達之車站。
*站內號誌樓廢棄後曾屹立一段時間<ref>{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>，今已拆除<ref>[https://web.archive.org/web/20140106043245/http://railway.tw/?p=426 平溪線的情調就要毀了！] @ 2013.02.20</ref>。
*本站設有「鐵路之旅－小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。
