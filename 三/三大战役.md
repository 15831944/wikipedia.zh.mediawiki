'''三大會戰'''，[[中國共產黨|中國共產黨]]称为'''三大戰役'''，指的是在1948年9月至1949年1月間，[[中国人民解放军|中国人民解放军]]與[[中華民國國軍|中華民國國軍]]之間在[[第二次國共内戰|第二次國共内戰]]中的三次關鍵[[戰役|戰役]]。三次會戰是指[[遼西會戰|遼西會戰]]、[[徐蚌會戰|徐蚌會戰]]与[[平津會戰|平津會戰]]，均以中國共產黨的胜利告终。

1947年7月，解放軍轉入戰略進攻<ref name="軍事I，第472頁">中國大百科全書總編輯委員會《軍事》編輯委員會，《中國大百科全書•軍事I》，北京，中國大百科全書出版社，1989年5月，第472頁，ISBN 7500002424</ref>。由1948年起，解放軍在三大战役结束后，國軍損失超過150万，精锐部队丧失殆尽，[[中國国民党|中國国民党]]在中國大陆的统治逐漸崩溃。而[[中國共產黨|中國共產黨]]勢力經年累月積攢，及在抗战中磨练的灵活的游击战术之優勢完全顯現，開始向全中國进一步进军。以至1950年打下[[海南島|海南島]]與[[西康|西康]]，中國共產黨統治整個中國大陸。

== 遼瀋會戰（辽沈战役） ==
{{main|辽沈战役}}
从1948年9月12日至11月2日（52天），[[林彪|林彪]]、[[罗荣桓|罗荣桓]]率领的[[东北野战军|东北野战军]]向[[中國東北|中國東北]]的[[锦州|锦州]]、[[长春|长春]]等地發起進攻，与國軍[[東北剿匪總司令部|東北剿匪總司令部]](總司令[[衛立煌|衛立煌]])麾下的主力部隊进行了五十二天的作战，國軍傷亡被俘共计47万2千餘人，東北全境為中共所佔，从而使解放军之優勢顯現，此战役称为'''辽沈战役'''，國府方面称作「'''遼瀋會戰'''」。<ref>[http://lov.vac.gov.tw/Protection/Content.aspx?i=25&c=3 行政院退輔會榮民文化網--遼瀋會戰]</ref>

== 徐蚌會戰（淮海战役） ==
{{main|淮海战役}}
从1948年11月6日至1949年1月10日（66天），[[刘伯承|刘伯承]]、[[陈毅|陈毅]]、[[粟裕|粟裕]]、[[邓小平|邓小平]]等率领的[[华东野战军|华东野战军]]和[[中原野战军|中原野战军]]60万人，在淮河、[[海州|海州]]一线附近与国军[[徐州剿匪总司令部|徐州剿匪总司令部]](總司令[[劉峙|劉峙]])所屬的主力部隊和[[华中剿匪总司令部|华中剿匪总司令部]](總司令[[白崇禧|白崇禧]])所屬的增援部队共80万人进行战略决战，以伤亡13万4千余人的代价，消灭及改编國軍[[黄百韬|黄百韬]]、[[黄维|黄维]]、[[邱清泉|邱清泉]]、[[李弥|李弥]]、[[孙元良|孙元良]]等部55万5千人。战役结束后，解放军控制了长江以北的广大地区，直接威脅首都[[南京|南京]]。此战役称为淮海战役，历时65天，奠定了解放长江以南各省的基础，對中國形勢有深遠影響。粟裕在此战役中战功卓著，毛泽东评“粟裕同志立了第一功”。

== 平津會戰（平津战役） ==
{{main|平津戰役}}
从1948年12月5日至1949年1月31日（64天），林彪、羅榮桓与[[聶榮臻|聶榮臻]]等人率东北野战军和[[华北野战军|华北野战军]]共100万，包围了國軍[[华北剿匪总司令部|华北剿匪总司令部]](總司令[[傅作义|傅作义]])統領之50餘萬國軍，先后通过新保安、天津等战役，消灭了国军部分主力，最终解放军与傅作义达成协议，傅作义开城投降。通过这次战役，國軍[[陈长捷|陈长捷]]、[[傅作义|傅作义]]等部共52万餘人被歼灭及改编，解放军佔領了[[北平|北平]]、[[天津|天津]]及[[張家口|張家口]]等地。这一战役被称为[[平津會戰|平津战役]]。
