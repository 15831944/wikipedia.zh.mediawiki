'''三宮·花時計前站'''（{{jpn|j=三宮・花時計前駅|hg=さんのみや・はなどけいまええき|rm=Sannomiya-Hanadokei-mae eki|f=y}}）是一個位於[[日本|日本]][[兵庫縣|兵庫縣]][[神戶市|神戶市]][[中央區_(神戶市)|中央區]]，屬於[[神戶市營地下鐵|神戶市營地下鐵]][[海岸線_(神戶市營地下鐵)|海岸線]]的[[鐵路車站|鐵路車站]]，也是該線的始發站。[[車站編號|車站編號]]是'''K01'''。

站名的「花時計」來自{{link-ja|神戶市役所|神戸市役所}}北側的{{link-ja|花時計 (神戶市)|花時計 (神戸市)|花時計}}。

== 歷史 ==
* 2001年（[[平成|平成]]13年）7月7日 - [[神戶市營地下鐵|神戶市營地下鐵]][[海岸線_(神戶市營地下鐵)|海岸線]]的此站－[[新長田站|新長田站]]間開通的同時開業。

== 車站結構 ==
地下2樓設有大堂與閘口，地下3樓設有月台。月台是島式兼港灣式1面2線。往新長田方向的列車發車。有地下通道，可與轉乘預定站（三宮站〈阪急、阪神是神戶三宮站，JR西日本是三之宮站〉），與[[舊居留地·大丸前站|舊居留地·大丸前站]]接續。

== 使用情況 ==
2016年度的1日平均'''上車'''人次為'''7,349人'''<ref>[http://www.city.kobe.lg.jp/life/access/transport/kotsu/gaiyou/index.html 神戸市：神戸市交通局　事業概要]</ref>。海岸線車站次於[[和田岬站|和田岬站]]、[[新長田站|新長田站]]排行第3位，開業以後有緩緩上昇傾向。
* 2000年度計劃時，2005年度的預測乘車人次有21,652人<ref name="jigohyoka">{{PDFlink|[http://www.city.kobe.lg.jp/life/access/transport/subway/kaigannsen/img/houkokusyo.pdf 地下鉄海岸線事後評価]}} - 神戸市、2007年3月</ref>。不過現在的乘車人次只有計劃時的3分之1程度。
