'''三民車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]]-{[[玉里鎮|玉里鎮]]}-，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。本站配合「花東新車站運動」計畫改建完工。

== 車站構造 ==
三民車站是設置有[[岸式月台|岸式]]與[[島式月台|島式月台]]各一座的地面車站，其中島式的第二月台為建站時原有，岸式的第一月台則是後來增建的。在平日列車多停靠島式的第二月台。2014年時配合[[交通部鐵路改建工程局|交通部鐵路改建工程局]]主導的「花東效能提升計畫」進行站房與站內設施的改良的同時，於第二月台與站房之間增設了一條配置有電梯的[[行人地下道|行人地下道]]，在此之前搭車旅客皆須直接徒步穿越鐵軌才能抵達月台。

== 營運狀況 ==
根據台鐵的分級，三民屬於乙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]等級，僅停靠[[臺鐵區間車|區間車]]，是台東線停靠班次最少的車站（單日南下北上各6個班次）。三民車站南距臨站[[玉里車站|玉里]]11.5公里，北距[[瑞穗車站_(臺灣)|瑞穗]]則為9.2公里，是台東線沿線站距區間最長、最孤立的一座客運車站。三民車站站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。2016年配合政策進行票務設備升級之後，開始提供使用[[一卡通_(台灣)|一卡通]]、[[悠遊卡|悠遊卡]]及[[icash|icash 2.0]]等[[電子票卡|電子票卡]]繳付車資的服務。

== 歷史 ==
三民車站所在的地區原名為「迪階」或「迪佳」，後來在[[台灣日治時期|日治時期]]改名為「三笠」。
*1917年11月1日：設「'''三笠驛'''」，地名由來以當地山姿優婉、彷彿[[奈良市|奈良]][[春日山_(奈良縣)|三笠山]]，而取名為「三笠」。
*1946年：[[國民政府|國民政府]]接管台灣後，因原站名和地名「三笠」具有強烈日本色彩，行政區域改為[[玉里鎮|玉里鎮]]三民里，故站名配合地名改為「'''三民'''」。
*2012年：配合「花東效能提升計畫」，三民車站進行外觀重新拉皮與相關設備更新的改建工程，由楊炳國建築師事務所及英商[[奧雅納|奧雅納工程顧問公司]]所規劃設計<ref>[http://visionf2012.pixnet.net/blog/post/29218580 花東新車站改造運動(三民站介紹)]</ref>。
*2014年7月11日：車站改建工程完成{{R|SL201-4}}。
*2016年6月28日：啟用多卡通刷卡機。
*2017年9月16日：本站由[[三等站_(台鐵)|三等站]]降為乙種[[簡易站|簡易站]]，並指定由[[玉里車站|玉里站]]管理。
