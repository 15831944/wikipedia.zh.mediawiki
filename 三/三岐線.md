'''三岐線'''（{{jpn|j=三岐線|hg=さんぎせん|rm=Sangi sen|f=y}}）是一條連結[[日本|日本]][[三重縣|三重縣]][[四日市市|四日市市]]的{{link-ja|富田站 (三重縣)|富田駅 (三重県)|富田站}}至三重縣[[員辨市|員辨市]]的{{link-ja|西藤原站|西藤原駅}}，屬於[[三岐鐵道|三岐鐵道]]的鐵路線。
== 路線資料 ==
* 路線距離（{{link-ja|營業距離|営業キロ}}）：
** 三岐線：富田－西藤原間 26.5公里（富田－三岐朝明間1.0公里只限貨物營業）<ref name="sone26 3">[[#sone26|『歴史でめぐる鉄道全路線 公営鉄道・私鉄』26号 3頁]]</ref>
** 近鐵連絡線：三岐朝明信號場－近鐵富田間 1.1公里<ref name="sone26 3"/>
* [[軌距|軌距]]：1067毫米<ref name="sone26 3"/>
* 站數：三岐線15個、近鐵聯絡線1個（全部包括起終點站，信號場除外）<ref name="sone26 3"/>
* 複線路段：沒有（全線單線）<ref name = "現有私鉄概説_33" />
* 電氣化路段：全線（直流1500V）<ref name = "現有私鉄概説_33" />
* 最大坡度：25[[千分比|‰]]（伊勢治田－東藤原間、東藤原－西野尻間）<ref name = "現有私鉄概説_33">[[#現有私鉄概説|「現有私鉄概説 三岐鉄道」p.33]]</ref>
* 最小曲線半徑：200公尺<ref name = "現有私鉄概説_33" />。
* [[閉塞_(鐵路)|閉塞方式]]：自動閉塞式<ref name = "現有私鉄概説_33" />
* 最高速度：70公里/小時<ref name="sone26 25"/>
