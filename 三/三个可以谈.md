'''三个可以谈'''是2002年11月，时任[[中共中央总书记|中共中央总书记]]的[[江泽民|江泽民]]在[[中共十六大|中共十六大]]报告中提出的为[[统一中国|统一中国]]，如何与中华民国（台湾）进行政治谈判的主张。

江泽民主张在[[一個中國_(中華人民共和國)|一个中国]]前提下，什么问题都可以谈。一是“可以谈正式结束两岸敌对状态问题{{efn|name=ab1}}”；二是“可以谈台湾地区在国际上与其身份相适应的经济文化活动空间问题”；三是“也可以谈台湾当局的政治地位等问题”<ref name="a1">{{Cite web |url=http://news.sohu.com/49/43/news204604349.shtml |title=张铭清：“三个可以谈”拓展两岸政治谈判空间|accessdate=2019-03-29 |author= |date=2002-11-27 |work= |publisher=搜狐网，中国新闻网|language=Zh-hans }}</ref>。

1992年，江泽民在[[中共十四大|中共十四大]]报告中即提出“在一个中国前提下，什么问题都可以谈，包括就两岸正式谈判的方式问题同台湾方面进行讨论，找到双方都认为合适的办法”。1995年1月，江泽民提出对台八项主张，被归纳为[[江八点|江八点]]。1997年，[[中共十五大|中共十五大]]报告，再度重申<ref name="a1"/>。

== 评价 ==

2002年中共十六大报告发表后的11月27日，[[国台办|国台办]]新闻发言人[[张铭清|张铭清]]在新闻发表会上表示，“三个可以谈”表明了己方对台方针政策的一贯性、连续性，体现了大陆方面在两岸政治谈判问题上的积极态度和极大诚意。“三个可以谈”使两岸政治谈判的议题进一步具体化、更具操作性。如第一个“可以谈”结束两岸敌对状态{{efn|name=ab1}}。“三个可以谈”进一步拓展了两岸政治谈判的空间，为打开两岸政治僵局指明了道路，为两岸同胞展现了共同争取和平统一的光明前景<ref name="a1"/><ref>{{Cite web |url=http://www.gwytb.gov.cn/zlzx/16thCongress/201101/t20110126_1734507.htm |title=国务院台办就十六大有关台湾问题的论述举行新闻发布会(节录)|accessdate=2019-03-29 |author= |date=2002-11-27 |work= |publisher=国台办网站|language=Zh-hans }}</ref>。
