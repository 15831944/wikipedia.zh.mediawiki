== 基本信息 ==
* 中文名：[[三氯化镱|三氯化镱]]、氯化镱、氯化镱(III)
* 化学式：YbCl<sub>3</sub>
* CAS号：10361-91-8（无水）、10035-01-5（六水）、19423-87-1（水合）
* 英文名：Ytterbium(III) chloride、Ytterbium chloride、Ytterbium trichloride
* 外观：白色粉末

== 化学反应方程式 ==
*被还原剂还原：
: 2 YbCl<sub>3</sub> + H<sub>2</sub> → 2 [[二氯化镱|YbCl<sub>2</sub>]] + 2 HCl<ref>无机化学丛书 第七卷 钪 稀土元素. 科学出版社. (1) 二卤化物的制备方法. pp 196</ref>
*和配体结合：
: YbCl<sub>3</sub> + 3 [[四氢呋喃|THF]] → YbCl<sub>3</sub>·3THF<ref>钱长涛, 王兵, 邓道利,等. 四氢呋喃配位的三氯化镱[YbCl<sub>3</sub>·3THF]的分子结构[J]. 结构化学, 1993, 12(1):18-21.</ref>
*复分解反应：
: YbCl<sub>3</sub> + 3 NaOH → Yb(OH)<sub>3</sub>↓ + 3 NaCl
: YbCl<sub>3</sub> + (NH<sub>4</sub>)<sub>3</sub>PO<sub>4</sub> → 3 NH<sub>4</sub>Cl + YbPO<sub>4</sub>↓
: YbCl<sub>3</sub> + 3 AgNO<sub>3</sub> → Yb(NO<sub>3</sub>)<sub>3</sub>+ 3 AgCl↓
: YbCl<sub>3</sub> + 3 AgClO<sub>4</sub> → Yb(ClO<sub>4</sub>)<sub>3</sub> + 3 AgCl↓
