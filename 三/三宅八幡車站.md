'''三宅八幡車站'''（{{jpn|j=三宅八幡駅|hg=みやけはちまんえき|rm=Miyake-hachiman eki|f=y}}）是位於[[日本|日本]][[京都市|京都市]][[左京區|左京區]]的[[叡山電鐵|叡山電鐵]][[叡山本線|叡山本線]]的[[鐵路車站|車站]]。車站編號為'''E07'''。

== 車站結構 ==
[[相對式月台|相對式月台]]2面2線的[[無人車站|無人車站]]。往出町柳車站方向的月台有大約一節車廂長的遮蓬，往八瀨比叡山口車站方向的月台中央則有較小的遮蓬和長椅。柵欄和柱子等等為了因應三宅八幡宮而被塗成朱紅色。主要的出肉口味在月台的八瀨比叡山口車站側，站內平交道（第1種甲）也在這一側，但在往出町柳方向的月台的出町柳側也有小小的出入口。

上下行的月台都沒有無障礙設施。從地面到月台的上只有階梯，使用輪椅必需要有人協助。

== 歷史 ==
* 1925年（[[大正|大正]]14年）9月27日　開業。當時為[[京都電燈|京都電燈]]的車站。
* 1942年（[[昭和|昭和]]17年）3月2日　由於公司的讓渡，成了[[京福電氣鐵道|京福電氣鐵道]]的車站。
* 1986年（昭和61年）4月1日　由於公司的讓渡，成了叡山電鐵的車站。

== 其他 ==
[[File:Stele_of_Miyakehachiman_Jinja.JPG|thumb]]
站名由來的三宅八幡宮離[[叡山電鐵鞍馬線|鞍馬線]]的[[八幡前車站_(京都府)|八幡前車站]]比較近，但是三宅八幡正式的參道（表參道）是在三宅八幡車站這一側，在車站前可以看到位於北側的三宅八幡宮的朱紅色大[[鳥居|鳥居]]。另外，車站前立有刻上「三宅八幡神社」的石柱，此神社名一直使用到二戰剛結束時為止。
