'''三山國王'''是發源於[[中國|中國]][[廣東|廣東]][[粵东|粵东]]地區之信仰，為[[潮州府|潮州府]]住民，包含[[潮州民系|潮州福佬人]]（潮汕人）、潮州[[客家人|客家人]]和漳州府南部（漳浦、平和、雲霄、詔安），包含漳州[[閩南民系|福佬人]]、漳州客家人的精神信仰，隨著當地移民向外擴展，成為[[粵东|粵东]]、[[東南亞|東南亞]]、[[香港|香港]]及[[臺灣民間信仰|臺灣民間信仰]]之一。三山國王實指現[[廣東|廣東]][[揭陽市|揭陽市]][[揭西縣|揭西縣]][[河婆镇|河婆镇]]北面的三座山——[[巾山_(三山國王)|巾山]]、[[明山_(三山國王)|明山]]、[[獨山_(三山國王)|獨山]]的三位[[山神|山神]]。祖廟位於今廣東省[[揭陽市|揭陽市]][[揭西縣|揭西縣]]縣城旁，當地舊稱霖田都，故該廟多被稱為霖田祖廟。三山國王的有不少傳說，一說是[[宋太祖|宋太祖]][[趙匡胤|趙匡胤]]借得三山的神力幫助，後才將[[劉張之亂|劉張之亂]]平息，遂封三山國王。據[[臺灣|臺灣]]鄉村禮俗記載，臺灣三山國王聖誕定為每年[[二月廿五|二月廿五]]。而粵東地區三山國王出游則定在正月，一般是[[正月初十|正月初十]]或[[正月十五|十五]]。

== 三王簡介 ==
[[File:田心宫正像1.jpg|thumb]]
[[File:臺南三山國王廟_鎮殿獨山國王.jpg|thumb]] 鎮殿獨山國王。]]
相傳[[唐代|唐代]]文豪[[韓愈|韓愈]]被貶為[[潮州|潮州]]刺史時，當地洪水泛濫成災，居民向這三座山的山神祈求止雨，果然雨過天晴，韓愈便奉為「三山神」，加以祭祀。到了[[北宋|北宋]]時期，三位山神屢次顯靈助宋軍平亂，[[宋太宗|宋太宗]]就分別賜封三位山神為清化威德報國王（大國王，巾山國王）、助政明肅寧國王（二國王，明山國王）、惠威弘應豐國王（三國王，[[獨山國王|獨山國王]]），合祀為「三山國王」。

以下列出三山國王各自姓氏及生誕<ref>出自臺南三山國王廟文宣</ref>。
*巾山國王：大王，姓連名傑，字清化，生誕是農曆[[二月二十五日|二月二十五日]]
*明山國王：二王，姓趙名軒，字助政，生誕是農曆[[六月二十五日|六月二十五日]]
*[[獨山國王|獨山國王]]：三王，姓喬名俊，字惠威，生誕是農曆[[九月二十五日|九月二十五日]]

== 三山国王庙 ==
在粤东三山国王庙常俗称为王爷宫或王公宫。
===粤东===
据不完全统计，[[粤东|粤东]]地区各地已知的三山国王庙超过200座。，其中[[汕头|汕头]]65座、[[揭阳|揭阳]]60座、[[潮州|潮州]]25座、[[梅州|梅州]]45座、[[汕尾|汕尾]]109座。据2012年实地调查，汕头市[[濠江区|濠江区]]三山国王庙有38座。由此，前述不完全统计，粤东三山国王庙超过200座是十分保守的。
*[[揭陽市|揭陽市]][[揭西县|揭西县]][[霖田祖庙|霖田祖庙]]（始建于隋，至今有1400年历史）
*[[潮州市|潮州市]][[饶平县|饶平县]]鸿埕大庙（始建于宋，至今有800多年历史）
*[[汕頭市|汕頭市]][[潮阳区|潮阳区]]棉城蛇脐古庙（始建于明，有500多年历史）
*汕頭市[[澄海|澄海]]银砂古庙（始建于明，有600多年历史，1996年重建）
*汕頭市[[达濠|达濠]]王爷宫始建于明朝，清乾隆五十年改建三山亭，1985年遭火灾，遂重建，2009年末重修
*汕頭市达濠青篮田心宫，始建于明崇祯五年（1632年），1987年重修
