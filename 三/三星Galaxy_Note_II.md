'''Samsung Galaxy Note II'''是[[韓國|韓國]][[三星電子|三星電子]]在2012年中推出的[[Android|Android]]行動通訊產品，擁有5.5英寸螢幕，介乎大螢幕[[智慧型手機|智慧型手機]]與[[平板電腦|平板電腦]]之間，為[[Samsung_Galaxy_Note|Samsung Galaxy Note]]的後繼機。而Galaxy Note II同期的最大競爭對手是 iPhone 5。

== 歷史 ==
Galaxy Note II在2012年8月29日於[[德國|德國]][[柏林國際廣播展|IFA]]展中推出，拥有5.5英寸HD [[Super_AMOLED_PLUS|Super AMOLED PLUS]]屏幕，并配备了S Pen[[手写笔|手写笔]]，于2012年10月在全球各个市场发售。上市后37天内即售出超过三百万部，两个月之后销量超越五百万部。

== 設計 ==
Galaxy Note II在設計上與[[Samsung_Galaxy_S_III|Samsung Galaxy S III]]的圓潤風格相似，不同於第一代Galaxy Note的方框設計。S Pen位置位於機背左下角，方便拔出。正面底部設有目錄鍵（Menu）、主按鍵（Home）和返回鍵（Back）。

Galaxy Note II是Note系列手機當中，唯一未有配備康寧[[大猩猩玻璃|大猩猩玻璃]]的産品。

=== S Pen繪圖筆 ===
:Galaxy Note II附送S Pen觸控筆，比上一代觸控筆改良，並提供更多S Pen的獨有程式。改良後的S Pen直徑8mm，有更好的手握感。筆頭改用橡膠材質，由上一代256層壓力感應增加至1024層，也避免刮損螢幕。<ref>{{cite web|url=http://chinese.engadget.com/2012/08/29/samsung-galaxy-note-ii-getting-redesigned-s-pen/|title=Samsung Galaxy Note II 的 S Pen 得到重新設計|date=2012-08-30|accessdate=2012-08-30}}</ref>
:繪圖晶片為日本的[[Wacom|Wacom]] W8008 1227KU203，Galaxy Note亦是Bamboo Stylus觸控筆的其中一隻測試品。

=== 軟體<ref>{{cite web|url=http://chinese.engadget.com/2012/08/29/samsung-galaxy-note-ii-getting-redesigned-s-pen/|title=Samsung Galaxy Note II 的 S Pen 得到重新設計|date=2012-08-30|accessdate=2012-08-30}}</ref> ===
使用界面預設為三星的[[Samsung_TouchWiz|TouchWiz]]界面，加入支援S Pen的功能。
*浮窗预览（Air View）: 讓S Pen筆尖不接觸螢幕的情況下，像滑鼠般在螢幕上操作資料夾和圖片庫等。
*快捷命令（Quick Command）: S Pen手勢操控，向上滑會顯示功能表。
*提笔速记（Popup Note）: S Pen可於通話時拔出，Popup Note會自動顯示，使用者可以記下筆記。

==自燃事件==
2016年9月23日，印度民航局官员表示，在一架印度航班上，一台Note 2手机冒烟起火。由于发现及时，这次事故并没有酿成大祸。事发后，印度民航局对三星Note系列手机发布禁令，规定该公司的手机上飞机后必须关机，甚至可能会禁止乘客携带<ref>{{cite web|url=http://tech.ifeng.com/a/20160924/44457299_0.shtml|title=三星手机又爆炸了 这次是Note2在印度航班上起火|date=2016年9月24日|accessdate=2016年9月26日}}</ref>。

2017年11月13日，一名山東省壽光市的民眾，當日中午Galaxy Note II在辦公桌上突然噴出烈焰，手機在桌上旋轉180°，隨後不斷冒出陣陣白色煙霧，過程都被監視器拍下來，當事人指出，該手機為2013年購入，至今使用原廠電池，從未更換過<ref>{{cite web|url=http://news.ltn.com.tw/news/world/breakingnews/2258629|title=三星Galaxy Note II傳爆炸 手機電池射出紅色烈焰|date=2017年11月19日|accessdate=2017年11月19日}}</ref>。
