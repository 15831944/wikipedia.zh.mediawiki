== 中國 ==
* '''[[三國|三國]]'''時代（220年－280年），[[中國歷史|中國歷史]]上[[東漢|東漢]]末年[[曹魏|魏]]、[[蜀汉|蜀]]、[[東吳|吳]]三足鼎立的時代。
; 史書
*《[[三國志|三國志]]》，[[陳壽|陳壽]]編纂的歷史書。
{{See also|三国志 (消歧义)}}
; 小說
*《[[三國演義|三國演義]]》，[[羅貫中|羅貫中]]的章回小說。
{{See also|三国演义 (消歧义)}}
*《[[三国机密|三国机密]]》，[[马伯庸|马伯庸]]的章回小说。
; 電視劇
*《[[三国演义_(电视剧)|三国演义 (电视剧)]]》，由[[孙彦军|孙彦军]]、[[鲍国安|鲍国安]]、[[唐国强|唐国强]]、[[吳曉東|吳曉東]]、[[魏宗万|魏宗万]]等主演，1994年播映。
*《[[三國_(電視劇)|三國 (電視劇)]]》，由[[于和伟|于和伟]]、[[陈建斌|陈建斌]]、[[陸毅|陸毅]]、[[張博|張博]]、[[倪大红|倪大红]]等主演，2010年出品。
*《[[大军师司马懿|大军师司马懿]]》，由[[吴秀波|吴秀波]]、[[于和偉|于和偉]]、[[王洛勇|王洛勇]]、[[丁海峰|丁海峰]]、[[王伯昭|王伯昭]]等主演，2017年出品。
*《[[三国机密之潜龙在渊|三国机密之潜龙在渊]]》，由[[马天宇|马天宇]]、[[韩东君|韩东君]]、[[万茜|万茜]]、[[董洁|董洁]]、[[王阳明|王阳明]]、[[董璇|董璇]]等主演，2018年出品。

== 朝鮮半島 ==
*[[朝鲜|朝鲜]][[前三國時代|前三國時代]]
*[[朝鲜三国时代|朝鲜三国时代]]，[[朝鮮歷史|朝鮮歷史]]上[[高句麗|高句麗]]、[[新羅|新羅]]、[[百濟|百濟]]鼎立的时代（57年－668年）
*[[後三國|後三國]]，朝鲜历史上[[泰封|泰封]]、新罗、[[后百济|后百济]]鼎立的时代（892年－936年）
*《[[三國史記|三國史記]]》，[[高丽|高丽]]史家金富轼奉高麗仁宗之命在1145年以漢文记载朝鲜三国时代的历史。
*《[[三國遺事|三國遺事]]》，繼《三國史記》之後，高麗時代僧侶一然所編撰的史書。

*《[[三国记|三国记]]》，6世紀中背景，韓國1992年-1993年出品。
