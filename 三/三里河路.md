'''三里河路'''是位于中国[[北京市|北京市]][[西城区|西城区]]西部的一条道路。<ref name=bjdmd>北京地名典，中国文联出版社，2008年，第170页</ref>

==简介==
三里河路北起[[西直门外大街|西直门外大街]]，南到[[复兴门外大街|复兴门外大街]]。因三里河而得名。[[侯仁之|侯仁之]]所著《步芳集》载：[[金世宗|金世宗]][[大定|大定]]十一年（1171年），从金口（位于[[石景山|石景山]]西北麻峪）开渠，引[[永定河|永定河]]水，经[[车箱渠|车箱渠]]故道，称“金口河”（遗迹至今仍可见，讹称为“[[金沟河_(北京)|金沟河]]”），其下游经今[[朱各庄|朱各庄]]附近折向东南，引入当时的[[金中都|金中都]]北护城河（今[[西便门|西便门]]外三里河故道便是其遗址的一段），径直向东，经[[通州区|通州城]]北，注入白河（该段即今[[通惠河|通惠河]]最初的前身）。此河道虽开成，但并未取得预期效果，原因是“泥淖淤塞，积滓成浅，不能胜舟”。<ref name=bjdmd/>

三里河，即金口河的故道。三里河名字中的“三里”，并非指距明清北京城垣三里，而是指距[[金中都|金中都]]城垣三里。金中都的北垣在今[[西便门|西便门]]附近。<ref name=bjdmd/>

过去三里河路以东为沙沟，旧时为南沙沟村。再向东为三里河村，旧时有小河自西北向东南流，1950年代建设三里河居民区后淤塞。有说法称，这条小河就是三里河。<ref name=bjdmd/>

从[[中国科学院|中国科学院]]西南侧开始，三里河路顺着西侧的[[永定河引水渠|永定河引水渠]]稍微向西弯曲，随后又改向北。从此弯曲处直接向北，有三里河东辅路，一直通至[[月坛南街|月坛南街]]。三里河东辅路西侧为绿地，东侧的[[测绘出版社|测绘出版社]]、[[中国机械工程学会|中国机械工程学会]]、[[中国科学院|中国科学院]]等单位的门牌号仍然属于三里河路。

三里河路西侧、[[钓鱼台国宾馆|钓鱼台国宾馆]]东墙外有大片的[[银杏|银杏]]林，每到秋季这里是北京市民欣赏银杏的胜地<ref>{{Cite web |url =http://news.sina.com.cn/c/2017-10-29/doc-ifynfrfn0087436.shtml  |title =钓鱼台银杏大道进入“热闹”观赏期  |author =  |date = 2017-10-29 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。
