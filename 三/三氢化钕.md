== 制备 ==
它可由金属钕在氢气气氛中加压加热反应得到：
: 2 Nd + 3 H<sub>2</sub> → 2 NdH<sub>3</sub>

==性质==
三氢化钕是六方晶系的蓝色晶体，晶胞参数a=0.385 nm, c=0.688 nm。

它可以和水反应：
: NdH<sub>3</sub> + 3 H<sub>2</sub>O → Nd(OH)<sub>3</sub> + 3 H<sub>2</sub>↑
