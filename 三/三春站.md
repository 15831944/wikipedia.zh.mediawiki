'''三春站'''（{{jpn|j=三春駅|hg=みはるえき|rm=Miharu eki|f=y}}）位于[[福岛县|福岛县]][[田村市|田村市]][[三春町|三春町]]大字平泽字担桥，是[[东日本旅客铁道|东日本旅客铁道]]（JR東日本）管辖的[[磐越东线|磐越东线]]上的[[鐵路車站|鐵路車站]]<ref name="station"/>。2002年入选[[东北车站百选|东北车站百选]]。

== 历史 ==
* 1914年7月21日 - 作为平西线重点站投入运营<ref name="station"/>。
* 1915年3月21日 - 三春站 - 小野新町站站间开通。
* 1987年4月1日 - [[国铁分割民营化|国铁分割民营化]]后成为JR东日本车站。
* 1993年
**当年 - 伴随磐越东线营业所成立，归属磐越东线营业所管辖。
**12月 - 三春町出资，站房改造<ref>{{Cite web |url=http://miharu-mk.com/aboutus |title=三春まちづくり公社について |publisher=株式会社三春まちづくり公社 |accessdate=2017-05-05}}</ref>。
* 2005年12月10日 - 伴随磐越东线营业所废止，本站成为管理站<ref name="station"/>。
* 2009年3月14日 - 支持IC卡"[[Suica|Suica]]"使用<ref name="station"/>。
* 2016年4月1日 - [[三春站|三春站]]业务委托化，管理站业务交还给[[郡山車站_(福島縣)|郡山站]]<ref name="station"/>。

== 车站结构 ==
[[岛式站台|岛式站台]]1台2线的[[地上车站|地上车站]]<ref name="station"/>。站台与站房间通过地道连接<ref name="station"/>。

[[郡山站_(福島县)|郡山站]]管理，JR东北综合服务受托的业务委托站<ref name="station"/>。站房内设有绿色窗口、简易"[[Suica|Suica]]"闸机、简易Suica自动售票机。站房设有快餐和土产发售商店<ref name="station"/>。
