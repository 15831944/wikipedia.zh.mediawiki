'''三磷酸腺苷'''（{{lang-en|'''adenosine triphosphate''', '''ATP'''}}；也称作'''腺苷三磷酸'''、'''[[腺嘌呤|腺嘌呤]]核苷三磷酸'''）在[[生物化學|生物化學]]中是一种[[核苷酸|核苷酸]]，作为[[細胞|細胞]]内[[能量|能量]]传递的“[[分子|分子]][[通货|通货]]”，储存和传递[[化学能|化学能]]。ATP在[[核酸|核酸]]合成中也具有重要作用。它也是RNA序列中的鳥嘌呤二核苷酸，在DNA進行轉錄或複製時可做為替補。

== 化學性質 ==
ATP由[[腺苷|腺苷]]和三個[[磷酸基|磷酸基]]所組成，[[化學式|化學式]]C<sub>10</sub>H<sub>16</sub>N<sub>5</sub>O<sub>13</sub>P<sub>3</sub>，結構簡式C<sub>10</sub>H<sub>8</sub>N<sub>4</sub>O<sub>2</sub>NH<sub>2</sub>(OH)<sub>2</sub>(PO<sub>3</sub>H)<sub>3</sub>H，[[分子量|分子量]]507.184。三個磷酸基團從腺苷開始被編爲α、β和γ磷酸基。ATP的化學名稱爲5'-三磷酸-9-β-D-呋喃核糖基腺嘌呤，或者5'-三磷酸-9-β-D-呋喃核糖基-6-氨基嘌呤。

ATP在[[缓冲溶液|非缓冲水溶液]]中不稳定，会水解为[[ADP|ADP]]和[[磷酸|磷酸]]。这是因为ATP分子中的P-O-P键比形成的磷酸[[键能|键能]]小，且产生了产物间和水间的[[氢键|氢键]]释放能量，使得反应放热而自发进行。在ATP与ADP的水溶液的[[化学平衡|化学平衡]]中，ATP最终会几乎完全转化为ADP。在达到平衡以前，发生该水解反应整个系统[[吉布斯能|吉布斯能]]变化量小于零，这意味着该体系可以对外界做非体积[[功|功]]。事实上，活细胞会通过呼吸作用维持ATP的浓度在ADP的五倍左右。在这种条件下，ATP水解提供的能量足以供其合成代谢所需。<ref name=Nicholls>{{cite book |author=Ferguson, S. J.; Nicholls, David; Ferguson, Stuart |title=Bioenergetics 3 |publisher=Academic |location=San Diego |year=2002 |isbn=0-12-518121-3 |edition=3rd}}</ref><ref>{{cite journal|author=林海斌 |title=是断键释能？还是水解释能？——ATP的能量来源|journal=《化学教学》|volume= |issue= |pages=74-76|date=2009年1月}}</ref>

== 生物合成 ==
在细胞中ATP的[[摩尔浓度|摩尔浓度]]通常是1-10mM。<ref>Beis I.，和Newsholme E.A.，（1975）。</ref>
ATP可通過多種細胞途徑產生。最典型的如在[[粒線體|粒線體]]中通過[[氧化磷酸化|氧化磷酸化]]由[[三磷酸腺苷合酶|三磷酸腺苷合酶]]合成，或者在植物的[[葉綠體|葉綠體]]中通過[[光合作用|光合作用]]合成。ATP合成的主要能源爲[[葡萄糖|葡萄糖]]和[[脂肪酸|脂肪酸]]。每分子葡萄糖先在[[细胞质基质|细胞质基质]]中產生2分子[[丙酮酸|丙酮酸]]同時產生2分子ATP，最終在粒線體中通過[[三羧酸循環|三羧酸循環]]（或称[[柠檬酸循环|柠檬酸循环]]）產生最多38分子ATP。脂肪酸氧化分解進入[[柠檬酸循环|柠檬酸循环]]，长链脱除也可以用于氧化磷酸化分解产生ATP，一般為108個ATP。

=== 糖解途径 ===
{{main|糖解作用}}
在糖解途径（Glycolytic Pathway）中，一个葡萄糖分子被分解，反应过程中生成两个ATP分子，反应式为：
:C<sub>6</sub>H<sub>12</sub>O<sub>6</sub> + 2 NAD<sup>+</sup> + 2 ADP + 2 H<sub>3</sub>PO<sub>4</sub> → 2 NADH + 2 [[丙酮酸|C<sub>3</sub>H<sub>4</sub>O<sub>3</sub>]] + 2 ATP + 2 H<sub>2</sub>O + 2 H<sup>+</sup>

=== 三羧檸檬酸循环途径(又名"檸檬酸循環") ===
{{main|三羧酸循环|氧化磷酸化}}
[[File:Citric_acid_cycle_with_aconitate_2_zh-hans.svg|thumb]]
在[[线粒体|线粒体]]中，丙酮酸被氧化为[[乙酰辅酶A|乙酰辅酶A]]，经精确控制的“燃烧”会产生总和为两个ATP分子的能量。
三羧酸循环（檸檬酸循環）全部反映的'''总和'''可表示为：
:Acetyl-CoA + 3 NAD<sup>+</sup> + FAD + GDP + P<sub>i</sub> + 2 H<sub>2</sub>O → CoA-SH + 3 NADH + 3 H<sup>+</sup> + FADH<sub>2</sub> + GTP + 2 CO<sub>2</sub>

[[File:MgATP2-small.gif|thumb]]，鎂-ATP，360度旋轉。]]

=== β-氧化 ===
{{main|β-氧化}}
脂肪酸也可以由β-氧化分解为乙酰辅酶A，一樣進入[[柠檬酸循环|柠檬酸循环]]產生能量。每个β-氧化的循环還为乙酸长链脱去两个碳原子并制造各一个NADH和FADH<sub>2</sub>分子，也可以用于氧化磷酸化分解产生ATP，因為脂肪酸氧化可以重複多次，能量產量更大。

=== 无氧分解 ===
{{main|发酵}}
无氧分解或称[[发酵|发酵]]是和糖酵解有些相似的过程。这个过程需要在没有氧氣作为电子[[受体|受体]]时产生能量。在大部分[[真核生物|真核生物]]体内，[[葡萄糖|葡萄糖]]同时被作为能量储存单位和电子供体。从葡萄糖分解为[[乳酸|乳酸]]的方程式为：
: C<sub>6</sub>H<sub>12</sub>O<sub>6</sub><math>\to</math> 2CH<sub>3</sub>CH(OH)COOH + 2 ATP

== ATP循环 ==
人体每天的能量需要[[水解|水解]]100-150摩尔的ATP，即相当于50至75千克這麼多。这意味着人一天将要分解掉相当于他体重的ATP。但人体中ATP的总量只有大约0.1[[莫耳|摩尔]]（51公克左右），所以每个ATP分子每天要被重复利用1000-1500次。ATP不能被储存，因为ATP在合成后必须于短时间内被消耗。

== ATP檢測 ==

由於所有存活的生物（包括微生物）體內都含ATP，而其含量幾乎十分穩定，所以於環境中採集標本並計量ATP含量，可以間接反映環境中微生物的數量。這對於餐飲業，食品製造及加工業（如奶制品廠），醫療業等對微生物控制比較着緊的行業，ATP測量提供了一個十分便利方案。<ref>http://atp.holisticphysio.com/</ref> ATP估量只要數分鐘就可完成，相反，傳統的細菌培養測試動輒要2至4天才完成，其時受污染產品已流出市面。ISO 22000食品安全管理系統的[[危害分析重要管制點|危害分析重要管制點]] (HACCP)體系都建議使用這即時評估方法。

== 其它三磷酸苷 ==

活细胞中也有其他的高能三磷酸盐如[[三磷酸鸟苷|三磷酸鸟苷]]。能量可以在这些三磷酸盐和ATP中由磷酸[[激酶|激酶]]催化反应之类的反应转移：当磷酸键被[[水解|水解]]的时候能量就会被释放。这种能量可以被多种酶、肌动蛋白和运输蛋白用于细胞的活动。水解还会生成自由的磷酸盐和[[二磷酸腺苷|二磷酸腺苷]]。[[二磷酸腺苷|二磷酸腺苷]]又可以被进一步水解为另一个磷酸离子和[[一磷酸腺苷|一磷酸腺苷]]。ATP也可以被直接水解为[[一磷酸腺苷|一磷酸腺苷]]和[[焦磷酸盐|焦磷酸盐]]，这个反应在水溶液中是高效的[[不可逆反应|不可逆反应]]。

=== ADP與GTP的反應 ===

:[[二磷酸腺苷|ADP]] + [[三磷酸鸟苷|GTP]] → ATP + [[二磷酸鸟苷|GDP]]
:[[二磷酸腺苷|二磷酸腺苷]] + [[三磷酸鸟苷|三磷酸鸟苷]] → '''三磷酸腺苷''' + [[二磷酸鸟苷|二磷酸鸟苷]]
<!--
:C_{10}H_{15}N_5O_{10}P_2 + C_{10}H_{16}N_5O_{14}P_3 → C_{10}H_{16}N_5O_{13}P_3 + C_{10}H_{15}N_5O_{11}P_2
-->
ATP可能会被作为[[纳米技术|纳米技术]]和[[灌溉|灌溉]]的能源。人工[[心脏起搏器|心脏起搏器]]可能受益于这种技术而不再需要[[电池|电池]]提供动力。
