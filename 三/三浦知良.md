'''三浦知良'''（{{bd|1967年|2月26日|catIdx=Kazuyoshi, Miura}}），是一名[[日本|日本]]足球运动员。他在1993年成为日本首位获得[[亚洲足球先生|亚洲足球先生]]荣誉的球员。<ref>{{cite web |url=http://www.rsssf.com/miscellaneous/as-poy.html |title=Asian Player of the Year |publisher=RSSSF |date= |accessdate=2009年8月8日 |language=en }}</ref>目前效力于日本J2联赛的[[横滨FC|横滨FC]]。他在2000年退出[[日本国家足球队|日本国家队]]，留下了日本国家队进球数第二的成绩（89场、55球）。

三浦知良的父親是納谷宣雄，在三浦知良四岁时与其母亲三浦由子[[離婚|離婚]]。三浦知良與兄長[[三浦泰年|泰年]]一同由母親撫養，並改隨母姓三浦。<ref name="xinhua">{{cite web |url=http://news.xinhuanet.com/sports/2009-03/15/content_11012778.htm |title=三浦知良破门刷新日联赛纪录 42岁老将续写传奇 |publisher=[[新华网|新华网]]（转自[[新浪网|新浪体育]]） |date=2009年3月15日 |accessdate=2009年8月8日 |language=zh }}</ref>

== 俱乐部生涯 ==
1967年，三浦知良出生于足球氛围非常浓郁的[[静冈市|静冈市]]。他读小学时就进入伯父纳谷义郎率领的城内FC，开始接触足球。1982年，年仅15岁的三浦知良入读静冈学园高中不到一年就离开学校，只身远渡[[巴西|巴西]]，成为[[圣保罗|圣保罗]]{{Link-en|尤文图斯竞技|Clube Atlético Juventus}}的一名球员。1986年，他与[[桑托斯足球俱乐部|桑托斯]]签下了自己的第一份职业合同。他先后效力过包括[[帕尔梅拉斯足球俱乐部|帕尔梅拉斯]]、[[库里蒂巴足球俱乐部|库里蒂巴]]在内的多家巴西球队，之后于1990年回到日本。<ref name="xinhua"/>

回国后，三浦知良加盟日本足球联赛的[[东京绿茵|读卖队]]。读卖队后来脱离[[读卖新闻|读卖新闻]]，在1993年[[J联赛|J联赛]]启动时更名为[[东京绿茵|川崎绿茵]]。三浦知良的成名历程几乎与J联赛的创办同步。在读卖队以及更名后的川崎绿茵效力期间，三浦知良与日本国家队的常客[[拉莫斯·瑠偉|拉莫斯·瑠偉]]、[[北澤豪|北澤豪]]等人并肩作战，帮助球队连续四年赢下联赛冠军头衔。其中，读卖队夺得最后两届日本足球联赛冠军，而川崎绿茵夺得头两届J联赛冠军。三浦知良还在1993年获得[[J联赛年度最有价值球员|J联赛年度最有价值球员]]和[[亚洲足球先生|亚洲足球先生]]这两项荣誉称号。<ref name="xinhua"/>

1994－1995赛季，三浦知良成为日本首位加盟[[意甲|意甲]]球队的足球运动员，效力于[[热那亚板球与足球俱乐部|热那亚]]。他在这个赛季出场21次，仅在热那亚与[[桑普多利亚|桑普多利亚]]的热那亚德比中打进一球。1995年夏天，他回到川崎绿茵，一直效力该队至1998赛季末。1999年，三浦知良再次登陆欧洲联赛，这次他加盟的是[[萨格勒布迪纳摩足球俱乐部|萨格勒布迪纳摩]]。但不久后，他便回到日本，先后效力[[京都不死鸟|京都不死鸟]]和[[神户胜利船|神户胜利船]]，并最终在2005年转会[[横滨FC|横滨FC]]。<ref name="xinhua"/>

2005年，三浦知良曾被横滨FC短期租借至[[悉尼FC|悉尼FC]]，参加了[[A联赛|A联赛]]以及[[2005年国际足联世界俱乐部冠军杯|2005年国际足联世界俱乐部冠军杯]]。他在悉尼FC的第二场比赛中打进两球，帮助球队以3比2战胜联赛领头羊[[阿德莱德联足球俱乐部|阿德莱德联]]。

自2008年，三浦知良便多次刷新日本J聯賽最年長入球紀錄，現時紀錄被刷新至50歲0個月14日。

== 国家队生涯 ==
三浦知良在1990年9月26日日本队与[[孟加拉国家足球队|孟加拉队]]的[[亚运会|亚运会]]比赛上首次代表国家队出场。从1990年到2000年，他共为日本国家队出场89次，踢進55球。<ref name="goals">{{cite web |url=http://www.rsssf.com/miscellaneous/miura-intlg.html |title=Kazuyoshi "Kazu" Miura - Goals in International Matches |publisher=RSSSF |date= |accessdate=2009年8月8日 |language=en }}</ref>

2012年，三浦知良代表日本參加[[世界盃五人制足球賽|世界盃五人制足球賽]]。<ref name="SINA">{{cite web |url=http://sports.sina.com.cn/c/2012-11-03/00096281410.shtml |title=五人制世界杯鸣锣开战 三浦知良圆世界杯之梦(图) |publisher=SINA |date=2012年11月3日 |accessdate=2012年12月22日 |language=zh}}</ref>
