《'''三国食货志'''》是[[陶元珍|陶元珍]]在[[武汉大学|武汉大学]]历史系的毕业论文，发表于1934年，曾登载于《[[燕京学报|燕京学报]]》。该作品曾得到[[胡适|胡适]]、[[傅斯年|傅斯年]]的赞赏<ref name=Lu2017>{{cite web |author1=陆漓 |title=《居延汉简笺证》背后的陶元甘 |url=https://www.thepaper.cn/newsDetail_forward_1755492 |website=澎湃新闻 |accessdate=2019-03-19}}</ref><ref>{{cite web |title=【回到三國】三國時代活着已是奇蹟　貴族都要醒目兼有運先生存到 |url=https://www.hk01.com/%E7%86%B1%E7%88%86%E8%A9%B1%E9%A1%8C/290108/%E5%9B%9E%E5%88%B0%E4%B8%89%E5%9C%8B-%E4%B8%89%E5%9C%8B%E6%99%82%E4%BB%A3%E6%B4%BB%E7%9D%80%E5%B7%B2%E6%98%AF%E5%A5%87%E8%B9%9F-%E8%B2%B4%E6%97%8F%E9%83%BD%E8%A6%81%E9%86%92%E7%9B%AE%E5%85%BC%E6%9C%89%E9%81%8B%E5%85%88%E7%94%9F%E5%AD%98%E5%88%B0 |website=HK01 |accessdate=2019-03-19}}</ref>。

《三国食货志》是中国近代较为重要的三国时代经济史研究著作之一<ref>{{cite journal|title=读稀见书随笔四则|url=https://books.google.com/books?id=fjcKAAAAIAAJ|author=刘远游|year=1989|issue=4|journal=复旦学报: 社会科学版|page=94}}</ref><ref>{{cite book|title=東岳論叢|url=https://books.google.com/books?id=yWQ4AAAAIAAJ|year=1984|publisher=東岳論叢雜志社|page=75}}</ref>。

==写作过程==
[[陈寿|陈寿]]写作的《[[三國志|三國志]]》有传无志，陶元珍在[[国立武汉大学|国立武汉大学]]就学时，遂以补三国食货志为毕业论文，得[[李剑农|李剑农]]指导完成，[[上海商务印书馆|上海商务印书馆]]在1935年出第一版，1989年[[台湾商务印书馆|台湾商务印书馆]]再版<ref name=Lu2017/>。

== 内容 ==
三国食货志共有九篇：
*第1篇 自序
*第2篇 户口
*第3篇 劳动
*第4篇 土地
*第5篇 农业
*第6篇 货币与物价
*第7篇 交通与都市
*第8篇 工商业
*第9篇 人民生活与国家财政
