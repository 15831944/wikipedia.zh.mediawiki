'''三角座'''（Triangulum, {{pronEng|traɪˈæŋgj<s>ʊ</s>ləm}}）是一個北天[[星座|星座]]，面积131.85[[平方度|平方度]]，占全天面积的0.320%，在全天88个星座中，[[星座面积列表|面积排行]]第七十八位。三角座中亮于5.5等的恒星有12颗，最亮星为[[三角座β|天大将军九]]（三角座β），视星等为3.00。每年10月23日子夜三角座中心经过上中天。它是現代88個星座中的其中一個，也是被[[托勒密|托勒密]]列出的48個星座之一。

== 特征 ==
三角座最亮的三颗恒星组成了几乎等腰的瘦长三角形，在远离灯光污染的地方很容易辨认出来。

三角座有一个[[梅西耶天体|梅西耶天体]]：M33，位于[[三角座α|娄宿增六]](三角座α)西面。
[[Image:TriangulumCC.jpg|thumb]]

== 历史及神话 ==
三角座的设立来源于[[托勒密|托勒密]]的著作。在早期也曾被稱為“Sicilia”，因爲[[西西里島|西西里島]]之神[[刻瑞斯_(羅馬神祇)|Ceres]]據説曾哀求[[朱比特|朱比特]]把西西里島放到天上。

== 恒星 ==
* [[三角座β|天大将军九]]（三角座β），三角座最亮的恒星，[[视星等|视星等]]3.00，距离110[[光年|光年]]，拥有一個靠得很近的較暗的伴星。
* [[三角座α|娄宿增六]](三角座α)，视星等3.41，距离65光年，位于三角形的顶点，也有一顆靠得很近的較暗的伴星。
* [[三角座6|三角座6]]，目视[[双星|双星]]，可由中型望遠鏡分成一顆黃色的亮星和一顆藍色的暗星<ref>Robert Burnham, Jr (1978).  ''Burnham's Celestial Handbook'', Dover Publications, New York.  ISBN 0-486-24065-7</ref>。主星视星等5.2，伴星为6.9，复合星等5.1。主星是[[半规则变星|半规则变星]]，亮度介于5.19-5.28之间。
* [[三角座R|三角座R]]，[[变星|长周期变星]]，亮度最大时为5.4，最小时为12.6，变光周期为266.9日。

== [[深空天體|深空天體]] ==
[[File:M33.jpg|right]]
* [[三角座星系|三角座星系]]（M33），又称为纸风车星系，是[[本星系團|本星系團]]四个[[螺旋星系|螺旋星系]]中的最小的一个，直径约5.5万光年，距離[[銀河系|銀河系]]257万[[光年|光年]]。其視星等為5.8，亮度足以在天氣良好的情況下由肉眼看到；其表面亮度低，因此需在低光害环境中观察。若光害嚴重，就連小型天文望遠鏡或雙筒望遠鏡也無能爲力。
* [[NGC_925|NGC 925]]，螺旋星系，長度10[[角分|角分]]，視星等为12。
* [[NGC_672|NGC 672]]，[[棒旋星系|棒旋星系]]，長度5角分、視星等为11.6。
