== 營運狀況 ==
[[File:A_wood_sculpture_in_Sanyi_Station_(Taiwan).jpg|thumb]]
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]，
**[[觀光列車|環島觀光列車]]不停靠。
**[[臺鐵區間車|區間車]]：皆停靠。
**自強號：皆不停靠。
**對號列車停靠率：0%；所有車種停靠率：53.5%。
**少數與[[成追線|成追線]][[臺鐵區間車|區間車]]在此折返。
**區間快車：停靠南下(逆行)2011次和北上(順行)2042次
*站內設施 
**列車資訊自動播音服務系統[[客家語|客家語]]、[[閩南語|閩南語]]、[[國語|國語]]等三種。
**售票窗口：設有兩個。
**[[自動閘門|自動閘門]]：尚未設置。
**[[自動售票機|自動售票機]]：設有一台。
**即時時刻表液晶電視服務：目前僅設有一部。
**列車資訊LED顯示系統：剪票口兩面、各月台一面。
**站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
[[File:Platform_3,_TRA_Sanyi_Station_20131109.jpg|thumb]]
*1903年10月7日：苗栗=三叉河間通車。
*1903年10月7日：設「三叉河驛」。
*1920年10月7日：改稱「三叉驛」。
*1945年10月：更名為「三叉站」。
*1955年3月1日：改為「三義站」沿用至今。
*1995年9月：原木造站房拆除，移至[[卓蘭鎮|卓蘭鎮]]小雪霸瀑布休閒村重組，目前在休閒村內保存再利用。<ref>[http://www.appledaily.com.tw/appledaily/article/supplement/20080504/30514653/ 苗栗 卓蘭 探三義車站 觀瀑賞山城風光]</ref>
*1998年9月：本站以南由舊山線(三義=后里)，改由新山線的長隧道代替。
*2000年3月31日：新站房落成。
*2010年6月5日：舊山線復駛，本站為起點站。
*2015年6月30日：啟用多卡通刷卡機。
*2017年9月:本站月台已墊高為115公分，無階化完成
