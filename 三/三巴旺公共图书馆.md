'''三巴旺公共圖書館''' （{{lang-en|Sembawang Public Library}}）是一所位于[[三巴旺|三巴旺]]规划区{{link-en|新加坡圖書館管理局|National Library Board}}旗下的[[公共圖書館|公共圖書館]]。此图书馆位于{{link-en|Sun Plaza购物中心|Sun Plaza (Singapore)}}里的五楼。2000年图书馆正式开幕，2014年经翻新后重新开幕。

==历史==
2000年8月三巴旺公共圖書館正式开幕，现任[[新加坡内务部|新加坡内务部]]部长[[尚穆根|尚穆根]]主持了开幕典礼。<ref>{{Cite web|url = http://infopedia.nl.sg/articles/SIP_260_2005-01-03.html|title = Sembawang Community Library|first = Jenny|last = Tien|deadurl = yes|archiveurl = https://web.archive.org/web/20120527123608/http://infopedia.nl.sg/articles/SIP_260_2005-01-03.html|archivedate = 2012-05-27|df = |language=en}}</ref>

2014年头圖書館进行为期7个月的翻新，2014年底以「海事业」的设计主题重新开放。<ref>{{cite web|url=https://www.straitstimes.com/singapore/ahoy-sembawang-public-library-reopens-with-new-theme-and-wider-range-of-books|title=Ahoy! Sembawang Public Library reopens with new theme and wider range of books|first=|last=|date=2014-11-05|publisher=|language=en|deadurl=no|archiveurl=https://web.archive.org/web/20180514083337/https://www.straitstimes.com/singapore/ahoy-sembawang-public-library-reopens-with-new-theme-and-wider-range-of-books|archivedate=2018-05-14|df=}}</ref>
