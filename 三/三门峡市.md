'''三门峡市'''是[[中华人民共和国|中华人民共和国]][[河南省|河南省]]下辖的[[地级市|地级市]]，位于河南省西部。东接[[洛阳市|洛阳市]]，南界[[南阳市|南阳市]]，西与[[陕西省|陕西省]][[渭南市|渭南市]]、[[商洛市|商洛市]]相邻，北隔[[黄河|黄河]]与[[山西省|山西省]][[运城市|运城市]]相望。地处豫西山区和黄土丘陵地带，市区位于[[黄河|黄河]]南岸，[[洛河|洛河]]和[[涧河|涧河]]流经境内。总面积10309平方公里，人口226.87万。三门峡是[[陇海铁路|陇海铁路]]上重要的城市，伴随着黄河第一坝——[[三门峡水利枢纽|三门峡水利枢纽]]的建设而崛起的一座新兴城市。

== 名称由来 ==
相传[[大禹|大禹]]治水，使神斧将高山劈成“人门”、“神门”、“鬼门”三道峡谷，河道中由鬼石和神石将河道分成三流，如同有三座门，三门峡由此得名。三门峡水库建成后，将神石和鬼石作为坝基。

== 历史 ==
古为[[夏朝|夏]][[商朝|商]]王朝统治的中心区域。[[西周|西周]]时属[[焦国|焦国]]、[[東虢國|虢国]]。[[周成王|周成王]]时，[[周公|周公]]和[[召公|召公]]以'''陕原'''为界，分周而治。《[[括地志|括地志]]》记：“周、召分陕，不因城名，从原而界也”。[[春秋时期|春秋时期]]先属虢，后属[[晋国|晋]]。[[战国|战国]]分属[[韩国_(战国)|韩]]、[[秦国|秦]]、[[魏国_(战国)|魏]]等国。[[秦朝|秦]]置[[陕县|陕县]]，属[[三川郡|三川郡]]。[[西汉|西汉]][[汉高帝|高帝]]二年（前205年）改三川郡为[[河南郡|河南郡]]。[[元鼎|元鼎]]四年（前113年）于[[函谷关|函谷关]]边置[[弘农郡|弘农郡]]，治[[弘农县|弘农县]]（今[[灵宝市|灵宝市]]东北），陕县属之。[[东汉|东汉]]及[[魏晋|魏晋]]沿袭。

[[北魏|北魏]][[太和_(北魏)|太和]]十一年（487年）置[[陕州_(古代)|陕州]]，治陕县（今[[陕州区|陕州区]]）。太和十八年（494年）废陕州。北魏[[宣武帝|宣武帝]]改陕县为[[北陕县|北陕县]]。[[东魏|东魏]][[天平_(东魏)|天平]]初复置陕州。[[西魏|西魏]][[大统|大统]]三年（537年）再废。[[北周|北周]]北陕县复名陕县，[[北周明帝|明帝]]年间复置陕州。

[[隋朝|隋]][[大业|大业]]三年（607年）废陕州，陕县属[[河南郡|河南郡]]。[[义宁|义宁]]元年（617年）于陕县复置弘农郡。[[唐朝|唐]][[武德|武德]]元年（618年）改弘农郡为陕州。[[天宝|天宝]]元年（742年）改陕州为[[陕郡|陕郡]]。[[乾元|乾元]]元年（758年）复为陕州。[[天祐|天祐]]元年（904年）改陕州为[[兴唐府|兴唐府]]。[[唐哀帝|唐哀帝]]时恢复陕州。
[[File:1962-03_1962年_三门峡市.jpg|thumb]]
[[北宋|北宋]]陕州属[[永兴军路|永兴军路]]。[[金代|金代]]属[[南京路_(金朝)|南京路]]。[[元代|元代]]属[[河南府路|河南府路]]。[[明朝|明]][[洪武|洪武]]元年（1368年）废陕县入陕州，属[[河南府|河南府]]。[[清朝|清]][[雍正|雍正]]二年（1724年）改陕州为[[陕州直隶州|陕州直隶州]]。[[民国|民国]]二年（1913年）废府改县，陕县属[[豫西道|豫西道]]，次年改为[[河洛道|河洛道]]。1928年废道制。民国二十一年（1932年）陕县属河南省第十一行政督察区。

[[中华人民共和国|中华人民共和国]]成立后，陕县属[[陕州专区|陕州专区]]，为专署驻地。1952年陕州专区并入[[洛阳专区|洛阳专区]]。1957年3月，经国务院批准，析陕县会兴等七乡设立三门峡市，由省直辖。1962年改由[[洛阳专区|洛阳专区]]管辖。1969年洛阳专区改称[[洛阳地区|洛阳地区]]。1986年1月，撤销洛阳地区，三门峡市升为地级市，将[[渑池|渑池]]、[[陕县|陕县]]、[[灵宝|灵宝]]、[[卢氏县|卢氏]]4县划归三门峡市管辖，义马市由三门峡市代管；同年5月设立[[湖滨区|湖滨区]]。2015年2月，撤销陕县，设立[[陕州区|陕州区]]。

== 地理 ==
三门峡市位于河南西部，[[豫|豫]]、[[晋|晋]]、[[陕|陕]]三省交界，东连[[洛阳|洛阳]]，南接[[南阳|南阳]]，西与陕西[[渭南|渭南]]接壤，北隔[[黄河|黄河]]与山西[[运城|运城]]相望。

三门峡市三面环山，一面环水，地势西南高、东北低，海拔高度一般在300—1500米之间，位于灵宝境内的老鸦岔海拔2413.8米，是河南省最高峰。全市总面积10496平方公里，其中山区占54.8%，丘陵占36%，平原占9.2%。三门峡市地处中纬度内陆区，属暖温带大陆性季风气候。历年平均气温13.2℃，年平均日照2354.3小时，平均日照率51%，历年无霜期184——218天，平均年降雨量550——800毫米。全市河流水系除黄河外，还有[[洛河|洛河]]、[[弘农河|弘农河]]、[[老灌河|老灌河]]三大河流和众多支流，分属[[黄河|黄河]]、[[长江|长江]]两大水系。

三门峡下游河中心有一块巨石在河道中挺立，经河水猛烈冲击，屹立几千年而不倒，叫做“[[砥柱山|砥柱山]]”，中国的[[成语|成语]]“[[中流砥柱|中流砥柱]]”就由此而来。
