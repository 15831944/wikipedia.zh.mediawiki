'''三氯乙烯'''是工業常用[[溶劑|溶劑]]。它無色，有毒性、透明、粘性低、不燃燒、易揮發，具有芳香味的液體，對[[神經|神經]]有[[麻醉|麻醉]]作用。純三氯乙烯分解緩慢。當有[[紫外線|紫外線]]照射三氯乙烯與[[氧氣|氧氣]]混合物時，加速三氯乙烯[[分解|分解]]。

三氯乙烯主要用作[[金属|金属]]脱脂和[[羊毛|羊毛]]及织物的干洗剂。[[树脂|树脂]]、[[沥青|沥青]]、[[煤焦油|煤焦油]]、[[醋酸纤维|醋酸纤维素]]、[[硝化纤维|硝化纤维素]]、[[橡胶|橡胶]]和涂料等的[[溶剂|溶剂]]。在医药上曾用作[[麻醉剂|麻醉剂]]，[[农药|农药]]上是合成[[一氯乙酸|一氯乙酸]]的原料。
三氯乙烯屬第一種有機溶劑。

== 生產 ==
* 1970年代前：

:[[乙炔|HC≡CH]] + 2 [[氯|Cl<sub>2</sub>]] → [[1,1,2,2-四氯乙烷|Cl<sub>2</sub>CHCHCl<sub>2</sub>]]

:2 Cl<sub>2</sub>CHCHCl<sub>2</sub> + [[氫氧化鈣|Ca(OH)<sub>2</sub>]] → 2 ClCH=CCl<sub>2</sub> + [[氯化鈣|CaCl<sub>2</sub>]] + 2 H<sub>2</sub>O

*在300-500℃，加熱四氯乙烷，加入[[氯化鋇|氯化鋇]]或[[氯化鈣|氯化鈣]]：

:Cl<sub>2</sub>CHCHCl<sub>2</sub> → ClCH=CCl<sub>2</sub> + [[鹽酸|HCl]]

* 現時：先以[[三氯化鐵|三氯化鐵]]為催化劑：

:[[乙烯|CH<sub>2</sub>=CH<sub>2</sub>]] + Cl<sub>2</sub> → [[1,2-二氯乙烷|ClCH<sub>2</sub>CH<sub>2</sub>Cl]]

400℃左右時，再加入氯：

:ClCH<sub>2</sub>CH<sub>2</sub>Cl + 2 Cl<sub>2</sub> → ClCH=CCl<sub>2</sub> + 3 HCl

最常見的[[催化劑|催化劑]]是[[氯化鉀|氯化鉀]]和[[氯化鋁|氯化鋁]]或[[碳|碳]]。[[四氯乙烷|四氯乙烷]]是副產品。
