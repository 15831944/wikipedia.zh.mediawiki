'''三軍總醫院松山分院'''，舊名'''空軍總醫院'''，[[編號|編號]]'''807醫院'''，是[[中華民國|中華民國]]在[[臺北市|臺北市]]的一所國軍醫療機構，目前下轄於[[國防部軍醫局|國防部軍醫局]]的一所區域暨乙類教學醫院。

== 概要 ==
三軍總醫院松山分院是臺北市[[松山區_(臺北市)|松山區]]代表的大型醫療機構之一<ref>[http://news.gpwb.gov.tw/news.aspx?ydn=026dTHGgTRNpmRFEgxcbfc4F8J3ahtuG7OxCtCB9AyWjSfuwcke697cHl1SO5xIkfcq8KWCe7vXeOSpQ1MyefEfhDeWy%2biBWceAVkxuorPI%3d 軍民健康守護者 軍醫專業團隊獲肯定] {{webarchive|url=https://web.archive.org/web/20160304200709/http://news.gpwb.gov.tw/news.aspx?ydn=026dTHGgTRNpmRFEgxcbfc4F8J3ahtuG7OxCtCB9AyWjSfuwcke697cHl1SO5xIkfcq8KWCe7vXeOSpQ1MyefEfhDeWy%2BiBWceAVkxuorPI%3D |date=2016-03-04 }} （[[青年日報|青年日報]]、2008年1月31日）</ref>，以臺北市松山地區居民及台北、宜蘭地區國軍人員為主要核心服務範圍。平時負責臺灣北部地區官兵健康維護，戰時支援第三作戰區收療受傷官兵及民眾，並執行各項醫療任務。

== 歷史 ==
* 1945年12月1日：'''第四空軍醫院'''成立於[[南京|南京]]。
* 1949年1月20日：隨空軍總部遷臺，更名臺北市空軍總醫院，位於臺北市蘭州街。
* 1958年4月1日：更名'''空軍總醫院'''，同年12月1日遷入健康路現址。
* 1959年11月25日：舉行落成典禮<ref>[http://www.cnavista.com.tw/shop/Stores_app/Browse_Item_Details.asp?Store_Id=103&Shopper_Id=47017191747254701&page_id=23&Cat_id=%AA%C5%ADx%20%ADx%C2%E5%20%C2%E5%B0%7C%20%A5~%BB%AB%B3X%B5%D8%20%B0%D1%C6%5B%20%A8%E5%C2%A7%20%BA%F2%AB%E6%C2%E5%C0%F8%20%C2%E5%C0%F8%B3%5D%ACI%20%C2%E5%C3%C4%BD%C3%A5%CD&IndexNo=19591125009500 空軍總醫院落成典禮]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }} 好望角·中央社數位照片平臺</ref>。
* 1970年6月12日：小兒科病房落成，由[[陳香梅|陳香梅]]與[[飛虎隊|飛虎隊]]盟友普萊斯柯德等捐贈，以普萊斯柯德亡子小彼德之名命名為彼德·普萊斯柯德紀念兒童醫院（Peter Prescott Memorial Children Hospital Unit）<ref>[http://www.cnavista.com.tw/shop/stores_app/Browse_Item_Details.asp?store_id=103&Shopper_Id=1130742224161130&page_id=23&Cat_id=%20&IndexNo=19700612009400&path1=%20&sno=197006120094000001L%2EJPG&p=1 空軍總醫院小兒科病房落成 由賴名湯夫人剪綵]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }} 好望角·中央社數位照片平臺</ref>。
* 1995年7月1日：依精進軍醫組織案，原隸屬三軍總部之四、五級醫院取消軍種番號，改直隸[[國防部參謀本部|參謀本部]]，更名'''國軍八零七總醫院'''。
* 1998年7月1日：實施[[精實案|精實案]]，國軍醫院番號改地區命名，更名'''國軍松山醫院'''。
* 2002年3月1日：原隸屬參謀本部之國軍醫院，改隸為國防部軍醫局之附屬機構。
* 2003年5月5日：奉命為[[嚴重急性呼吸道症候群|SARS]]專責醫院<ref>[http://www.nownews.com/2003/05/19/10974-1456474.htm 陳水扁戴口罩了 松山醫院成專責醫院今啟用] （Nownews、2003年5月19日）</ref>，5月19日改裝完成，設置102間負壓式隔離病房，成為SARS病患治療的中繼站<ref>[http://www.libertytimes.com.tw/2003/new/may/20/today-sars20.htm 總統到松醫 全程戴口罩]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }} （[[自由時報|自由時報]]、2003年5月20日）</ref>。至同年9月16日，始恢復正常門診。
* 2005年11月1日：依精進案規劃，國軍松山醫院提升為軍陣醫學之總醫院型態，更名'''國軍松山總醫院'''。
* 2013年1月1日：依國防部「醫療機構整合」計劃，更名為'''三軍總醫院松山分院'''。

== 組織架構 ==
國軍松山總醫院本部下轄有醫療部、醫勤室、衛保室<ref>2006年7月1日更名為'''衛材補給保養室'''，負責藥品衛材補給、醫療裝備汱舊更新編列計劃、醫療裝備保修任務、醫療器材、敷料消毒作業等任務。</ref>、行政室、民診處、主計室及勤務隊等七大單位。

醫療部則分成急診醫學科、眼科、牙科、復健科、耳鼻喉科、護理部、臨床藥劑科、放射科、精神科、家醫科、麻醉科、加護中心、社區醫學部、航空醫學暨體檢部、婦產科、小兒科、臨床病理科、內科部及外科部。

內科部下又細分為一般內科、心臟內科、腸胃內科、神經內科及腎臟內科。外科部則細分骨科、一般外科、整型外科、泌尿外科及神經外科。
