'''三宝鸟'''（[[学名|学名]]：{{lang|la|''Eurystomus orientalis''}}）又名'''-{zh-cn:佛法僧;zh-tw:三寶鳥;}-'''，为[[佛法僧科|佛法僧科]][[三寶鳥屬|三寶鳥屬]]的[[鸟类|鸟类]]。该物种的模式产地在[[爪哇|爪哇]]。<ref name='id2331'/>

== 特征 ==
体长约28厘米。喙短而阔，口角深裂呈红色，只有尖端微黑。通体羽色多蓝绿色，头部和尾部为黑色。成鸟的喉部具显著亮蓝色，幼鸟缺少，头部和背部杂有褐色羽。

== 亚种 ==
*'''三宝鸟普通亚种'''（[[学名|学名]]：{{lang|la|''Eurystomus orientalis calonyx''}}）。在[[中国大陆|中国大陆]]，分布于东部沿海、西至[[甘肃|甘肃]]、[[四川|四川]]、[[西藏|西藏]]、[[云南|云南]]、[[广西|广西]]等地。该物种的模式产地在[[印度|印度]]。<ref name='id2332'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=2332| title =三宝鸟普通亚种| work =《中国动物物种编目数据库》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-04-04| deadurl =yes| archiveurl =https://web.archive.org/web/20160305052719/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=2332| archivedate =2016-03-05}}</ref>

== 名称 ==
「三寶鳥」或「佛法僧」的名稱來自日本。夜間森林中“but po so”（音似日語“佛法僧”。[[佛寶|佛寶]]、[[正法_(佛教)|法寶]]與[[僧寶|僧寶]]為[[佛教|佛教]]中之「[[三宝_(佛教)|三寶]]」）的鳥鳴聲，被認為是此鳥所發，故名。1935年發現此鳥鳴聲實為[[普通角鴞|普通角鴞]]所發。
