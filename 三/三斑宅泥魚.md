'''三斑宅泥魚'''，又稱'''三斑圓雀鯛'''，俗名為厚殼仔、三點白，為[[輻鰭魚綱|輻鰭魚綱]][[鱸形目|鱸形目]][[隆頭魚亞目|隆頭魚亞目]][[雀鯛科|雀鯛科]]的其中一[[種|種]]。

== 分布 ==
本魚分布於[[印度|印度]][[太平洋|太平洋]]區，包括[[東非|東非]]、[[馬達加斯加|馬達加斯加]]、[[紅海|紅海]]、[[馬爾地夫|馬爾地夫]]、[[印度|印度]]、[[斯里蘭卡|斯里蘭卡]]、[[日本|日本]]、[[台灣|台灣]]、[[中國大陸|中國大陸]]、[[菲律賓|菲律賓]]、[[越南|越南]]、[[泰國|泰國]]、[[印尼|印尼]]、[[馬來西亞|馬來西亞]]、[[澳洲|澳洲]]、[[所羅門群島|所羅門群島]]、[[斐濟群島|斐濟群島]]、[[馬紹爾群島|馬紹爾群島]]、[[密克羅尼西亞|密克羅尼西亞]]、[[馬里亞納群島|馬里亞納群島]]、[[法屬玻里尼西亞|法屬玻里尼西亞]]、[[夏威夷群島|夏威夷群島]]、[[東加|東加]]、[[萬那杜|萬那杜]]、[[諾魯|諾魯]]等海域。<ref name="fishbase-trimaculatus">{{FishBase_species|genus=Dascyllus|species=trimaculatus|year=2007|month=5}}</ref>

== 深度 ==
水深1~55公尺。

== 特徵 ==
本魚體呈圓形而側扁，幼體呈黑[[褐色|褐色]]，頭頂正中有一[[白色|白色]]圓斑，體側在第七至第九背鰭棘下方的側線上亦有一珠白色圓斑。成魚體呈[[暗褐色|暗褐色]]，上述白斑則消失，背鰭鰭條後半部及胸鰭顏色淡，其他各鰭[[黑色|黑色]]。吻短而鈍圓。口中型；兩頜齒小而呈圓錐狀，靠外緣之齒列漸大且齒端背側有不規則之絨毛帶。背鰭硬棘12枚、背鰭軟條14~16枚、臀鰭硬棘2枚、臀鰭軟條14~15枚。體長可達15公分。

== 生態 ==
本魚幼魚常和[[海葵|海葵]]共生，其對海葵的刺細胞已有免疫，成魚則成群地在礁石突出處或大石塊附近活動。屬雜食性，以[[藻類|藻類]]及動物性[[浮游生物|浮游生物]]為食。領域性極強，在警戒或交配時，體色會轉變成灰白色。

== 經濟利用 ==
為可愛的[[觀賞魚|觀賞魚]]，尤其是小魚躲在海葵觸手中，相當可愛，很少人食用。
