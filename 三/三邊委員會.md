'''三極委員會'''（[[英語|英語]]：'''Trilateral Commission'''），中文又譯'''三邊委員會'''，<ref name = TChome/>是一國際性[[非政府組織|非政府組織]]，非黨派的討論性團體。於1973年7月，由[[大衛·洛克菲勒|大衛·洛克菲勒]]建立。目的是建立[[美洲|美洲]]、[[歐洲|歐洲]]和[[日本|日本]]更緊密的合作關係。

== 歷史 ==
因倍感北美、歐洲和日本等亞太國家地區間的不和諧，三邊委員會於是成立，其宗旨為促進世界各地實質性的政治和經濟對話。其成立宣言：
*國際間相互的依存性日益加深，成為當代世界的一項不容否認的事實。它超越和影響著每個國家的制度……有鑑於在世界經濟和大規模的國際關係上，背負著巨大的重擔。並在自身的利益和世界各地負有特殊責任的基礎上，制訂有效的合作目標。所以重要的是，要在世界各國、日本、西歐和北美發展更多的合作關係。<ref name=FAQ />
*要有效地解決常見的問題，日本、西歐和北美將有更密切的磋商和合作。並在平等互利的基礎上，協調各國制定和執行政策上的問題，確保他們的共同利益……避免他們的相互合作與單方面行動損害到其他地區。並利用現有的國際和區域組織，進一步提高他們的影響力。<ref name=FAQ />
*委員會希望擔當啟發性的角色，以及其他國家和地區自由交換意見的渠道。並進一步關注發展中國家和東西方關係的改善。<ref name=FAQ>{{cite web|title= F&Q|url= http://www.trilateral.org/go.cfm?do=Page.View&pid=9|accessdate=2012-11-30}}</ref>

[[哥倫比亞大學|哥倫比亞大學]][[教授|教授]][[茲比格涅夫·布里辛斯基|茲比格涅夫·布里辛斯基]]與國際事務專家[[大衛·洛克菲勒|大衛·洛克菲勒]]，以及以下人士，共同組建了此組織：
* [[亨利D·歐文|亨利D·歐文]] （[[布魯金斯學會|布魯金斯學會]]外交政策研究員）
* [[喬治·S·富蘭克林|喬治·S·富蘭克林]]
* [[羅伯特·R·鮑伊|羅伯特·R·鮑伊]]
* [[杰拉德C.史密斯|杰拉德C.史密斯]]
* [[馬歇爾·霍恩布洛爾|馬歇爾·霍恩布洛爾]] 
* [[威廉·斯克蘭頓|威廉·斯克蘭頓]]
* [[埃德溫·賴肖爾|埃德溫·賴肖爾]]（[[哈佛大學|哈佛大學]]教授）
* [[馬克思·科恩斯塔姆|馬克思·科恩斯塔姆]]
其他創始成員，包含後來的[[美國聯邦準備理事會|美國聯邦準備理事會]]負責人[[亞倫·葛林斯班|亞倫·葛林斯班]]和[[保羅·沃爾克|保羅·沃爾克]]。

1973年10月，在[[東京|東京]]的三邊委員會，啟用了一年兩次會議的時間表。1976年5月，第一次全體會議在北京舉行。<ref>{{cite web|title= publication|url= http://www.trilateral.org/go.cfm?do=Page.View&pid=8|accessdate=2012-11-30}}</ref>正是通過這些早期的會議，將中國整合到全球的政治對話系統中。在這些交流之前，中國在國際舞台上相當孤立。<ref>{{cite web |url=http://www.trilateral.org/go.cfm?do=Page.View&pid=9#12 |title=The Trilateral Commission FAQ |publisher=The Trilateral Commission |year=2011 |accessdate=2012-07-17}}</ref>組織自成立以來，持續發行《三邊對話》（Trialogue）的官方雜誌。<ref>{{cite web|title= publication|url= http://www.trilateral.org/go.cfm?do=Page.View&pid=7|accessdate=2012-11-30}}</ref>

== 會員構成 ==
會員被分為三個地區。代表北美大陸的120個成員：[[加拿大|加拿大]]20位、[[墨西哥|墨西哥]]13位和[[美國|美國]]87位。<ref>{{cite web|title=America |url=http://www.trilateral.org/go.cfm?do=Page.View&pid=12 |accessdate=2012-11-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20121021204534/http://www.trilateral.org/go.cfm?do=Page.View&pid=12 |archivedate=2012-10-21 }}</ref>[[歐洲|歐洲]]個別國家的上限是20位，目前已經達到規定上限總數的170名，歐洲大陸上幾乎每一個國家都有名額：[[德國|德國]]18位、[[法國|法國]]18位、[[義大利|義大利]]及[[英國|英國]]12位、[[西班牙|西班牙]]12位，剩餘名額為其他國家所分有。<ref>{{cite web|title=Europe |url=http://www.trilateral.org/go.cfm?do=Page.View&pid=11 |accessdate=2012-11-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20130509143411/http://www.trilateral.org/go.cfm?do=Page.View&pid=11 |archivedate=2013-05-09 }}</ref>起初，[[亞洲|亞洲]]和[[大洋洲|大洋洲]]，只有[[日本|日本]]代表出席會議。2000年，增加為85名成員，並擴增為亞太集團，由117個成員組成：日本75位、[[韓國|韓國]]11位、[[澳大利亞|澳大利亞]]和[[紐西蘭|紐西蘭]]7位、[[東盟|東盟]]國家15位、[[香港|香港]]和[[中國|中國]]7位。<ref>{{cite web|title= Asia|url= http://www.trilateral.org/go.cfm?do=Page.View&pid=13|accessdate=2012-11-30}}</ref>

== 批評 ==
在右派方面，一些思想家、政治家批評三邊委員會是侵犯國家主權。保守的前共和黨參議員[[貝利·高華德|貝利·高華德]]，在其著作中批評：「他們想控制和鞏固在政治、金融、知識產權、宗教上的權力。」。<ref>{{cite book |author=Goldwater, Barry |title=With No Apologies |coauthors=coauthored with Stephen Shadegg |publisher=Berkley |year=1980 |page=299 |isbn=0-425-04663-X}}</ref>在左派方面，語言學家[[諾姆·杭士基|諾姆·杭士基]]認為，此會發布的一份名為「民主危機」的報告，提出方案解決1960年代的民主問題，內容表露出「自由意識形態的[[國家資本主義|國家資本主義]]統治精英」的思想。杭士基還認為，委員會在[[吉米·卡特|吉米·卡特]]的管理政策上有不當影響。<ref>"[http://www.chomsky.info/books/priorities01.htm The Carter Administration: Myth and Reality]", Noam Chomsky</ref>
