'''三間村'''位於[[臺灣|臺灣本島]]的西側，是臺灣[[嘉義縣|嘉義縣]][[新港鄉|新港鄉]]轄下的行政區，屬於[[平原|平原地形]]，面積約278公頃，行政區下設8鄰，設籍人口542戶，總計約1701人。

== 地理 ==
三間村於嘉義縣新港鄉的東南方<ref name=":0">{{Cite web|url=http://sixstar.moc.gov.tw/blog/95aa46vg/mapGuideAction.do?method=doMapGuide|title=嘉義縣新港鄉三間社區發展協會|accessdate=2017-07-04|author=|date=|publisher=}}</ref>，為平原地形、[[熱帶乾濕季氣候|熱帶潮濕]]氣候<ref>{{Cite journal|title=阮琡婷(2002)《嘉義縣新港鄉中洋村、三間村的區域地理研究》國立高雄師範大學碩士論文，頁1|author=阮琡婷|url=|journal=《嘉義縣新港鄉中洋村、三間村的區域地理研究》|issue=|doi=|others=|year=|volume=|page=|pmid=}}</ref>，東鄰[[民雄鄉|民雄鄉]][[山中村|山中村]]，西連新港鄉[[中洋村|中洋村]]，全村由[[三間厝|三間厝]]、[[港尾|港尾]]以及[[鑽石新城|鑽石新城]]三個聚落所組成，村內的作物種類以[[水稻|水稻]]為主，近年來村內興起養豬業<ref name=":1">{{Cite web|url=http://gn.moi.gov.tw/GeoNames/GNMap/map_Admin/map/MapMain.aspx#|title=地名資訊服務網|accessdate=2017-07-04|author=|date=|publisher=|deadurl=yes|archiveurl=https://web.archive.org/web/20170225130550/http://gn.moi.gov.tw/geonames/GNMap/map_Admin/map/MapMain.aspx|archivedate=2017-02-25}}</ref>，土地面積約278公頃<ref name=":0" />。

三間村由發源自牛稠山的咬仔竹圳及牛稠溪分別貫穿村內和流經村子的南端，因而構成便利的灌溉網，也奠定了三間村以農立村的基礎<ref name=":0" />。

== 聚落歷史 ==
三間村於1723-1735年間([[雍正帝|清雍正時期]])有官民合力開鑿的十四甲圳灌溉，是新港鄉最早有紀錄的人工水利設施<ref>{{Cite journal|title=阮琡婷(2002)《嘉義縣新港鄉中洋村、三間村的區域地理研究》國立高雄師範大學碩士論文，頁14。|author=阮琡婷|url=|journal=《嘉義縣新港鄉中洋村、三間村的區域地理研究》|issue=|doi=|others=|year=|volume=|page=|pmid=}}</ref>。

三間村水利興盛，之前有「九十六甲埤」舊址在津港尾一帶，湖面廣達五十公頃以上，是先民開墾三間厝及民雄鄉山中村時，賴以灌溉的主要水源，但因日漸淤積而被人們填塞<ref name=":0" />。

三間村於新巷時期屬[[牛稠溪堡|牛稠溪堡]]第九區，1920年([[1920年|大正九年]])時，三間聚落改為[[新巷庄|新巷庄]]的行政區域；1950年([[1950年|民國39年]])以後變更為嘉義縣新港鄉的轄區<ref name=":1" />。

== 教育 ==
三間村因無教育機構，所以嘉義縣政府將三間村的學區劃分在嘉義縣新港鄉[[福德村|福德村]]的[[新港國小|新港國小]]、[[新港國中|新港國中]]<ref>{{Cite web|url=http://edu.cyhg.gov.tw/News_Content.aspx?n=8B04CBD4471A6D45&s=FA655C580A05CA56|title=嘉義縣政府教育處|accessdate=2017-07-05|author=|date=|publisher=}}</ref>。

== 文化資產 ==
* 三間永興宮：永興宮是當地的信仰中心，因始創建於[[六斗|六斗]]，故又俗稱[[六斗廟|六斗廟]]，位於三間村36號，1736年創立，主祀神明為[[保生大帝|保生大帝]]<ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/shingang/1007016-YXG/view|title=三間永興宮-文化資源地理資訊系統|accessdate=2017-07-04|author=|date=|publisher=}}</ref>，於誕辰日(每年農曆3月15日)當天會舉行繞境，繞境結束後緊接著是「[[過火|過火]]」儀式，是村中重要的祭祀活動<ref name=":2">{{Cite journal|title=阮琡婷(2002)《嘉義縣新港鄉中洋村、三間村的區域地理研究》國立高雄師範大學碩士論文，頁98|author=阮琡婷|url=|journal=《嘉義縣新港鄉中洋村、三間村的區域地理研究》|issue=|doi=|others=|year=|volume=|page=|pmid=}}</ref>。

* 鎮安宮：位於工業區旁，由[[華興元帥|華興元帥]]鎮殿，鎮安宮所在地原有一棵榕樹，後來因有一孩童在榕樹下起乩，材民才決定在該處建造廟宇<ref name=":1" />。

== 景點 ==
* 三隻小豬地標：為了讓社區有新的氣象，村民們沿[[縣道166號|縣166]]道路營造入口意象，以改善社區的環境，包括以長青樹作為精神地標，長青樹取其老樹新生，也象徵了社區要向前走的意象<ref name=":3">{{Cite web|url=http://www.ttvs.cy.edu.tw/kcc/960310ya/san1.htm|title=新港三間村社區營造|accessdate=2017-07-04|author=|date=|publisher=}}</ref>。

* 小火車公園：將[[臺灣糖業鐵路|台糖小火車]]舊火車站旁雜草叢生的空地，整理打造成三間牛稠小火車公園，並設置文化牆，展出社區的老照片以及社區村民的寫生作品<ref name=":3" />。

* 消失的[[十八甲野蓮埤|十八甲野蓮埤]]：因三間村曾經擁有十八甲野蓮埤的溫沃土地，所以在舊鐵道下挖空來建造蓮花池<ref name=":4">{{Cite web|url=http://www.hkfce.org.tw/chinese/03_column/03_detail.php?kid=29&cid=323|title=快樂向前走三間村-新港文教基金會|accessdate=2017-07-04|author=|date=|publisher=}}</ref>。

* 新港自行車道：舊時台糖鐵道，在停車後，轉型成自行車道，總長度為4.8公里，從新港市中心開始，沿路經過[[菜公村|菜公村]]、中洋村，最後抵達三間村<ref>{{Cite web|url=http://www3.nccu.edu.tw/~101356028/tourism_2.html|title=新港輕旅行-新港自行車道|accessdate=2017-07-04|author=|date=|publisher=}}</ref>。

== 其他特色 ==
由嘉義縣新港鄉[[農會|農會]]所輔導的「整合鄉村社區組織計畫」中，讓社區營造有一定的成果，並得到「全國特優社區」之美名<ref name=":4" />。

三間村於冬至前一天會舉行「謝平安」的祭祀活動，且由各鄰輪流[[擲筊|擲筊]]決定由誰負責祭祀相關事宜；農曆正月初九則擲筊決定頭家仔，負責在農曆3月15日前兩天清廟<ref name=":2" />。
