'''三林东站'''（工程名称'''东明路站'''，原称'''三林公园站'''）位于[[上海|上海]][[浦東新区|浦東新区]][[三林鎮|三林鎮]]，车站位于[[三林路|三林路]][[东明路|东明路]]，车站主体位于三林路地下，横跨路口，呈近东西方向。为[[上海轨道交通11号线|上海轨道交通11号线]]北段二期的车站，于2013年8月31日启用<ref>{{Cite web |url= http://www.shmetro.com/node49/201308/con113351.htm |title= 11号线二期8月31日起通车试运营 |author= 上海地铁运管中心 |date=2013-08-29 |language= zh-cn |accessdate= 2013-08-29 }}</ref>。

== 公交换乘 ==
84、572区间、755、780、795、986区间、1003、周南线、南南线

== 车站出口 ==
*1号口：三林路东，东明路北
*2号口：三林路东，东明路南
*4号口：三林路西，东明路北
