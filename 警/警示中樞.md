[[大脑|大脑]]是[[人类|人类]]特有区别以其他[[生物|生物]]有关[[智能|智能]]、[[行为|行为]]和[[思维|思维]]的特色[[器官|器官]]。关于人类大脑的[[功能|功能]][[分类|分类]]和通俗的功能[[控制|控制]]的研究已经经历了几百年，但是人类大脑的功能还没用被完完全全地被发现或者说理论地阐述。就目前所知，大脑中枢的功能十分复杂，随着[[行为学|行为学]]、[[心理学|心理学]]和其他边缘科学研究的不断加入，[[大脑中枢|大脑中枢]]的一些不为人所知的功能渐渐浮出水面，其中包括'''警示中枢'''（Guard Center）。

== 起源 == 
2007年，一个新的中枢分类被提出来，被命名为“警示中枢”，提出此命名的[[广州|广州]]一所[[医学|医学]]院校的学者认为这是一个早已被人觉察但却从来没被独立出分析的大脑中枢，而且不局限于人脑，也就是说这是所有拥有大脑的生物所持有的。

== 原理 ==
在结合国内外一系列针对性或非针对性的研究后得出，大脑警示中枢（Guard Center）的现实表现确实存在。其功能表现为：让人有准备或无准备地[[焦躁|焦躁]]、忐忑、不安、[[恐惧|恐惧]]等，或者是让人有[[意识|意识]]或无意识地做某些动作或者行为。
   
比如，在当你集中精神做某件事的时候或突然感到来自[[心理|心理]]的[[干扰|干扰]]，通俗地说为“突然觉得不对劲”。但是当你停下来仔细观察周围时或许会发现根本没有任何异样。其实这是因为身边某些长年累月被大脑其他中枢熟悉了的现象突然发些改变所致的，甚至是微小的改变。

举个例子，在雷声大作的夜晚你可以尽情地安睡不被吵醒，但是你却可能因为轻微的脚步声而被惊醒甚至醒来后你不知道为何会被惊醒。这就是警示中枢发挥的作用。

== 功能 ==
警示中枢主要负责“被动思维”，其功能有两个：
* 第一，负责收集[[视觉|视觉]]、[[听觉|听觉]]和[[触觉|触觉]]范围内的信息，形成环境像，并储存、
* 第二，但发现某个环境像中的某一点或者某些点跟之前储存的环境像发些区别时，发出[[信号|信号]]，这些信号随即引起焦虑、不安或者恐惧。

== 延伸 ==
由此可以解释幾种现象：[[抑郁|抑郁]]、[[敏感|敏感]]、[[强迫症|强迫症]]及某些[[精神分裂|精神分裂]]。
  
由于大脑警示中枢受损，失去功能，或让某些人产生反应迟钝、对周边环境改变不敏感、淡漠的现象，类似抑郁症状，这或许是抑郁的一个起因。
   
由于大脑警示中枢受到某些外来的强烈[[刺激|刺激]]（噩耗、喜讯、药物刺激等），使得警示中枢异常放电，导致某些人出现无故的恐惧、伤心和哭闹，类似精神分裂。
   
由于经常受到周边环境的影响到底大脑警示中枢高敏感，总会莫名其妙地不安，导致总是不断重复某些动作，类似强迫症。
