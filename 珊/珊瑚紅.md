== 珊瑚紅 ==

作為[[網頁顏色|網頁顏色]]的'''珊瑚紅'''是[[橙色|橙色]]系、粉紅色系顏色。

英語中最早於1513年時使用珊瑚紅一詞作為顏色名稱。<ref>{{cite book|title=A Dictionary of Color|last=Maerz|first=Aloys John|publisher=McGraw-Hill Book Company|year=1930|isbn=|location=New York|page=193|postscript=|author2=Paul, Morris Rea}}</ref>

{{clear}}

== 人類文化中的珊瑚紅 ==
; 體育
珊瑚紅是[[邁阿密海豚|邁阿密海豚]]的球隊顏色之一。
; 文學
佩脫拉克的詩作經常將女人的嘴唇描摹為珊瑚紅色。
