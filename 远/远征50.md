'''远征50'''是前往[[国际空间站|国际空间站]]的第50次[[国际太空站远征队列表|长期考察组活动]]。

莫斯科时间2016年10月30日凌晨3时35分<ref>{{cite web |title=ЦУП. ПИЛОТИРУЕМЫЙ КОРАБЛЬ «СОЮЗ МС» ОТСТЫКОВАЛСЯ ОТ МКС И ОТПРАВИЛСЯ К ЗЕМЛЕ |url=https://www.roscosmos.ru/22819/ |accessdate=2019-04-16 |language=ru |date=2016-10-30}}</ref>，载有[[阿纳托利·伊万尼申|阿纳托利·伊万尼申]]、[[大西卓哉|大西卓哉]]和[[凯瑟琳·鲁宾斯|凯瑟琳·鲁宾斯]]3名宇航员的[[联盟MS-01|联盟MS-01]]飞船与国际空间站分离，标志着[[远征49|远征49]]任务的结束和远征50任务的开始<ref>{{cite web|url=http://en.roscosmos.ru/20656/|title=Soyuz MS crew return|accessdate=2016-10-31|publisher=Roscosmos}}</ref>；留在空间站的{{le|罗伯特·金布罗|Robert Kimbrough}}、[[安德烈·鲍里先科|安德烈·鲍里先科]]和[[谢尔盖·尼古拉耶维奇·雷日科夫|谢尔盖·雷日科夫]]转移到本次远征50任务，其中金布罗担任指令长<ref>{{cite web|url=http://www.collectspace.com/news/news-102816a-space-station-expedition-50.html|title='Golden' expedition: 50th commander takes charge of space station|publisher=collectspace.com|date=2016-10-28 |access-date=2016-11-22}}</ref>。

北京时间2016年11月18日凌晨4时20分，载有[[奥列格·诺维茨基|奥列格·诺维茨基]]、[[佩姬·惠特森|佩姬·惠特森]]和[[托马·佩斯凯|托马·佩斯凯]]的[[联盟MS-03|联盟MS-03]]飞船在哈萨克斯坦的[[拜科努尔发射场|拜科努尔发射场]]成功发射升空，不久后到达国际空间站<ref>{{cite web |title=俄罗斯“联盟”飞船今载3名国际宇航员发射升空 |url=http://m.news.cctv.com/2016/11/18/ARTI8rUvHio8UmOYxVypQCTD161118.shtml |publisher=[[CCTV|CCTV]] |archiveurl=https://web.archive.org/web/20190416143134/http://m.news.cctv.com/2016/11/18/ARTI8rUvHio8UmOYxVypQCTD161118.shtml |archivedate=2019-04-16 |date=2016-11-18}}</ref>，空间站进入6人满员状态<ref>{{Cite web|url=https://blogs.nasa.gov/spacestation/2016/11/19/welcome-aboard-new-arrivals-make-six-expedition-50-crew-members/|title=Welcome Aboard! New Arrivals Make Six Expedition 50 Crew Members {{!}} Space Station|website=blogs.nasa.gov|access-date=2016-11-24}}</ref>。

2017年4月10日，金布罗、鲍里先科、雷日科夫乘坐[[联盟MS-02|联盟MS-02]]返回地球<ref>{{Cite news|url=https://www.nasa.gov/mission_pages/station/expeditions/expedition50/index.html|title=Expedition 50|last=Garcia|first=Mark|date=2015-02-12|work=NASA|language=en|access-date=2017-03-16}}</ref>，标志着本次远征的结束。惠特森、皮沙特、诺维茨基转移到[[远征51|远征51]]，其中惠特森担任指令长<ref>{{cite web|url=https://blogs.nasa.gov/spacestation/2017/06/01/station-changes-command-before-friday-crew-return/ |title=Station Changes Command Before Friday Crew Return |accessdate=2017-06-18|publisher=NASA |date= 2017-06-01 |last=Garcia |first=Mark}}</ref>。
