'''远东广播公司'''（[[英语|英语]]：Far East Broadcasting Company，简称FEBC）是一個总部位于[[美国|美国]]的[[国际广播|国际广播]]机构，主要向世界各地传播[[基督教|基督教]]方面的内容。於1945年開始广播，目前使用149种语言，日播音时间超过650小时，使用[[FM|FM]]、[[中波|中波]]和[[短波|短波]]播音。

== 广播目标 ==
让听众通过收听广播方式，让[[耶稣|耶稣]]走进他们的内心深处<ref name="febc.org">{{cite web |url=http://febc.org/about/core_values.html |title=存档副本 |accessdate=2008-08-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20080623214149/http://febc.org/about/core_values.html |archivedate=2008-06-23 }}</ref>。

== 历史 ==
1945年[[太平洋战争|太平洋战争]]结束<ref>{{cite web |url=http://febc.org/about/history.html |title=存档副本 |accessdate=2008-08-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20080624044619/http://www.febc.org/about/history.html |archivedate=2008-06-24 }}</ref>，随着[[驻日美军|驻日美军]]和[[驻韩美军|驻韩美军]]等驻[[亚洲|亚洲]]美军的存在，使得John Broger和Bob Bowman认为有必要在亚洲设立一个基督教广播机构，来满足这些人的收听需要，顺便可以将[[福音书|福音书]]带给数百万亚洲民众，于是电台最终选择在中国上海播音。<br/>然而在1948年，随着中国内战的局势发生变化，为了不让电台停播，于是便决定迁移至另一个适合的国家去落地，经过多日的寻找，John Broger最终决定将电台的新地址落地在位于[[南中国海|南中国海]]东侧的[[菲律宾|菲律宾]]。<br/>由于此时的菲律宾刚刚独立，在各方面都需要得到[[美国|美国]]的资助，故十分同意远东广播公司在此落地。最终在1948年6月4日于[[马尼拉|马尼拉]]开始播音《东方的呼声》，使用呼号为[[DZAS|KZAS]]，而对中国的[[良友电台|良友电台]]从菲律宾以短波向中国大陆广播也于1949年7月29日开始。<br/>除覆盖东亚地区之外，目前已经扩展至[[东南亚|东南亚]]和[[俄罗斯|俄罗斯]]，拥有41个广播站。此外还有姐妹广播机构[[远东福音广播公司|远东福音广播公司]]对[[南亚|南亚]]、[[非洲|非洲]]及中东地区广播。

== 慈善活动 ==
在远东广播公司成立至今，已举办过多起慈善活动，如提供奖学金给贫困学生、提供食物给贫困居民等。
