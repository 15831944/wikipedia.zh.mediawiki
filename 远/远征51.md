'''远征51'''是前往[[国际空间站|国际空间站]]的第51次[[国际太空站远征队列表|长期考察组活动]]。2017年4月10日, [[联盟MS-02|联盟MS-02]]飞船与国际空间站分离, 标志着本次远征的开始。这艘飞船返回时接回了[[远征50|远征50]]的三名宇航员{{le|罗伯特·金布罗|Robert Kimbrough}}、{{le|安德烈·博赖森科|Andrei Borisenko}}、{{le|谢尔盖·尼古拉耶维奇·雷日科夫|Sergey Nikolayevich Ryzhikov}}，另外三名宇航员[[佩吉·惠特森|佩吉·惠特森]]、{{le|托马斯·皮沙特|Thomas Pesquet}}、{{le|奥勒根·诺维斯基|Oleg Novitskiy}}转移到本次远征，惠特森担任指令长<ref>{{cite web|url=https://blogs.nasa.gov/spacestation/2017/06/01/station-changes-command-before-friday-crew-return/|title=Station Changes Command Before Friday Crew Return|accessdate=18 June 2017|date=1 June 2017|last=Garcia|first=Mark|publisher=NASA}}</ref>。

2017年4月20日, [[联盟MS-04|联盟MS-04]]飞船与国际空间站对接成功，本次发射减少了1名俄罗斯宇航员，只有两名宇航员{{le|费奥多尔·雅奇金|Fyodor Yurchikhin}}、{{le|杰克·费舍尔|Jack D. Fischer}}进入空间站，空间站为五人状态<ref>{{cite web|url=http://www.space.com/34736-nasa-russia-reduced-space-station-crew.html|title=NASA, Russia Set Flights for Trimmed-Down Space Station Crew|author=Irene Klotz|date=16 November 2016|publisher=space.com|access-date=21 November 2016}}</ref>。

4月初，惠特森的任务被从6月延长到9月<ref>{{cite web|url=http://www.cnbeta.com/articles/tech/599841.htm|title=破纪录：女宇航员Peggy Whitson在空间站多待了三个月}}</ref><ref>{{cite web|url=https://www.nasa.gov/press-release/nasa-astronaut-peggy-whitson-adds-three-months-to-record-breaking-mission|title=NASA Astronaut Peggy Whitson Adds Time to Record-Breaking Mission|accessdate=April 25, 2017|date=April 5, 2017|last=Northon|first=Karen|publisher=}}</ref>。2017年6月2日，诺维斯基，皮沙特乘坐[[联盟MS-03|联盟MS-03]]飞船返回地球，标志着本次远征的结束<ref name="end">{{Cite web|url=https://blogs.nasa.gov/spacestation/2017/06/01/station-changes-command-before-friday-crew-return/|title=Station Changes Command Before Friday Crew Return|accessdate=18 June 2017|date=1 June 2017|last=Garcia|first=Mark|publisher=NASA}}</ref>。惠特森、费舍尔、雅奇金转移到[[远征52|远征52]]，雅奇金成为指令长<ref>{{Cite web|url=https://blogs.nasa.gov/spacestation/2017/06/01/station-changes-command-before-friday-crew-return/|title=Station Changes Command Before Friday Crew Return|accessdate=18 June 2017|date=1 June 2017|last=Garcia|first=Mark|publisher=NASA}}</ref>。
