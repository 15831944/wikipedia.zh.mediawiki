《'''財神到'''》，粤语[[农历新年|贺年]]歌曲，由[[许冠杰|许冠杰]]、[[黎彼得|黎彼得]]创作。《财神到》的时代背景是1970年代初，当时的香港社會开始出现财神形象。此曲反映了香港社會流行的愛財心態，又勸勉人們不要迷信、不要為非作歹，在香港受到歡迎。

== 创作背景与历程 ==
1970年代初，香港社会开始出现[[财神|财神]]形象和相关的文化作品，而当时社会大众普遍重视农历新年，《財神到》就是在这个时代背景下誕生<ref name="rthk"/>。《財神到》由香港歌手許冠傑作曲，許冠傑、黎彼得一同作词，最早收录在許冠傑1978年發表的[[財神到_(專輯)|同名专辑]]中<ref name="rthk"/><ref name="朱耀偉"/>。此曲先有歌词、后有曲调，这种音乐创作手法称为“问字取腔”，广泛用于1970年代始的粤语音乐中<ref name="黃志華"/>。

== 音乐特色 ==
《財神到》既有中国传统文化，又有西方音乐的元素，脍炙人口<ref name="hk01"/>。歌词主要讲述财神降世，并提到人们盼望新年如意吉祥的祝福<ref name="陆仁"/>。此外，歌詞也反映了中国人传统的发财愿望、香港社會流行的爱财心态，并表现了人们在生活穷困时希望能够发财的现象<ref name="hk01"/><ref name="陆仁"/><ref name="鏡報"/>。当时的香港音乐界已流行创作贺年歌曲，黎彼得将《財神到》的歌词以“恭喜”外的其他内容为主题，从而做到创新<ref name="rthk"/>。此曲有劝勉人们不要迷信、不要为非作歹的歌词，黎彼得认为这样可以带来共鸣与正能量<ref name="rthk"/>。

== 反响 ==
《財神到》一曲发行后便在香港受到欢迎，所属专辑在1978年举行的[[第二屆香港金唱片頒獎典禮|第二屆香港金唱片頒獎典禮]]得到白金唱片荣誉<ref name="rthk"/>。此曲成为香港历久不衰的贺年歌曲，一直传唱了数十年<ref name="hk01"/>。此外，此曲还从香港流传到[[广州|广州]]、继而传唱到[[岭南|岭南]]地区一带，在这些地区广泛流行起来<ref name="陆仁"/>。《財神到》亦普及於海外各地的华人社群，例如[[巴西|巴西]]就曾经有电台广播此曲<ref name="rthk"/>。
