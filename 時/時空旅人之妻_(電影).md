《'''時空旅人之妻'''》（{{lang-en|'''''The Time Traveler's Wife'''''}}）是一部2009年的美國[[愛情片|愛情]]電影。本片改編自[[奧黛麗·尼芬格|奧黛麗·尼芬格]]於2003年所創作的[[時空旅人之妻|同名愛情小說]]。由{{tsl|en|Robert Schwentke|勞勃·史溫凱}}執導，[[艾力·班納|艾力·班納]]、[[瑞秋·麥亞當斯|瑞秋·麥亞當斯]]和[[榮·利文斯通|榮·利文斯通]]主演。

==劇情==
一場車禍，母親喪生，而孩子卻奇蹟般倖存，他赤裸著身體獲救，然而救人者竟然是來自未來的自己。

原來，亨利·德泰伯尔（Henry DeTamble，[[艾力·班納|艾力·班納]]飾），這位芝加哥圖書館的一名管理員患上慢性時間錯位症，正在忍受的是一種遺傳而來的“時間混亂”所帶來的麻煩和痛苦，这是一種極其罕見的特殊體質，使得他圍繞著屬於自己的時間線，隨時隨地都會消失或出現。

不斷地在時空當中來來回回地穿梭，最可怕的是，他沒辦法控制自己的能力，只能無奈地任憑命運的擺佈。而且，他每次穿越都赤身裸体，經常鬧出笑話。

在無數次的穿越中，他時常會穿越到同一個地點，那就是女主角克萊爾（Clare Abshire，[[瑞秋·麥亞當斯|瑞秋·麥亞當斯]]飾）家别墅外的樹林中。在那裏他認識了當時才6歲的克萊爾（{{tsl|en|Brooklynn Proulx|布鲁克琳·普劳克斯}}飾），並且告訴她自己是一個時空穿梭者。

而克萊爾也在一次次與亨利的相遇中逐漸愛上了他，夢想著在未來能夠嫁給他，直到他們在圖書館的第一次相遇，他們的時間轴第一次在現实中同步了……他們在克服了層層地阻礙之後，共同邁進了婚姻的殿堂……

然而這段婚姻所衍生出來的問題和複雜性，卻因為亨利對自己的無能為力而增加了很多，他沒辦法控制自己在同一個時空逗留的時間長短，所以他和他的愛人永遠都沒有辦法保持步調一致……

==製作==
本電影的改編權由電影公司[[B計劃娛樂|B計劃娛樂]]及[[新線影業|新線影業]]於原著小說未出版之前就已經取得。<ref name="Rubin rewriting Time">{{cite web |title = "Rubin rewriting 'Time'" |url = http://variety.com/2007/film/news/rubin-rewriting-time-2-1117956555/ |publisher = [[Variety|Variety]] "Dave McNary"(January 2, 2007) }}</ref>

原著小說作者[[奧黛麗·尼芬格|奧黛麗·尼芬格]]於訪問中表示於寫作期間曾想像過如果拍成電影會變成怎樣。<ref name="Interview: Audrey Niffenegger">{{cite web |title = "Interview: Audrey Niffenegger" |url = http://chicagoist.com/2007/05/09/interview_audrey_niffenegger.php |publisher = Chicagoist (2007-05-09) |deadurl = yes |archiveurl = https://web.archive.org/web/20140901155945/http://chicagoist.com/2007/05/09/interview_audrey_niffenegger.php |archivedate = 2014-09-01 }}</ref>當被問到對於現正被改編的電影版本有甚麼期望，[[奧黛麗·尼芬格|奧黛麗·尼芬格]]則回應「自己腦海中已經有一個想像中的電影版本，所以有點害怕這個想像中的版本會被刪改，這令我覺得有點害怕及毛骨悚然，因為畢竟那些角色已經不屬於我自己。<ref name= "Niffenegger's first book, and it's about time">{{cite web |title = "Niffenegger's first book, and it's about time" |publisher =  [[National_Post|National Post]] (December 3, 2003). [[LexisNexis|LexisNexis]] (subscription required)}}</ref>

2003年9月，工作室聘請了美國[[編劇|編劇]]{{link-en|傑瑞米·李文|Jeremy Leven}}來撰寫小說的改編劇本，<ref name= "Feud for thought">{{cite web |title =  "Feud for thought" |url = http://variety.com/2003/film/news/feud-for-thought-1117892085/ |publisher = [[Variety|Variety]] "Michael Fleming" (2003-09-07)}}</ref>著名導演[[史提芬·史匹堡|史提芬·史匹堡]]及[[大衛·芬查|大衛·芬查]]雖曾簡略表示過對劇本有興趣，但卻始終沒有協商過其製作。<ref name="Schwentke finds time for NL's 'Wife'">{{cite web|title="Schwentke finds time for NL's 'Wife'" |url=http://www.hollywoodreporter.com/hr/content_display/film/news/e3invDY7O/Fa2e%2B5R%2BMGGMZsw%3D%3D |publisher=[[The_Hollywood_Reporter|The Hollywood Reporter]] "Borys Kit; Nicole Sperling" (2006-11-01) |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20061108172656/http://www.hollywoodreporter.com/hr/content_display/film/news/e3invDY7O/Fa2e%2B5R%2BMGGMZsw%3D%3D |archivedate=2006-11-08 }}</ref>

2005年3月，另一位導演[[葛斯·范·桑|葛斯·范·桑]]與工作室協商執導本電影，<ref name= "Van Sant Helming Time Traveler's Wife">{{cite web |title = "Van Sant Helming Time Traveler's Wife" |url = http://www.comingsoon.net/news/topnews.php?id=8808 |publisher = ComingSoon.net. 2005-03-17}}</ref>但其後卻不了了之。

直至2006年11月，導演{{tsl|en|Robert Schwentke|勞勃·史溫凱}}最終被聘請成為本片執導。<ref name="Schwentke finds time for NL's 'Wife'">{{cite web|title="Schwentke finds time for NL's 'Wife'" |url=http://www.hollywoodreporter.com/hr/content_display/film/news/e3invDY7O/Fa2e%2B5R%2BMGGMZsw%3D%3D |publisher=[[The_Hollywood_Reporter|The Hollywood Reporter]] "Borys Kit; Nicole Sperling" (2006-11-01) |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20061108172656/http://www.hollywoodreporter.com/hr/content_display/film/news/e3invDY7O/Fa2e%2B5R%2BMGGMZsw%3D%3D |archivedate=2006-11-08 }}</ref>

2007年1月，[[新線影業|新線影業]]聘請了同為美國編劇{{link-en|布魯士·祖兒·羅賓斯|Bruce Joel Rubin}}重新編寫傑瑞米·李文的改編劇本，<ref name="Rubin rewriting Time"/>[[艾力·班納|艾力·班納]]及[[瑞秋·麥亞當斯|瑞秋·麥亞當斯]]正式於同年的4月選為男女主角。<ref name=  "New Line finds its cast on 'Time'">{{cite web |title =  "New Line finds its cast on 'Time'" |url = http://variety.com/2007/film/news/new-line-finds-its-cast-on-time-2-1117963263/ |publisher = [[Variety|Variety]] "Michael Fleming; Dave McNary" (2007-04-17) }}</ref>

於同年9月10日，開始於[[多倫多|多倫多]]及[[咸美頓|咸美頓]]拍攝。<ref name= "Domestic film: In production">{{cite web |title = "Domestic film: In production" |url = http://www.hollywoodreporter.com/movies/production-listings |publisher = [[The_Hollywood_Reporter|The Hollywood Reporter]] }}</ref><ref name= "Internet Movie Database – List of Films shot in Hamilton, Ontario">{{cite web |title = "Internet Movie Database – List of Films shot in Hamilton, Ontario" |url = http://www.imdb.com/search/title?endings=on&&locations=Hamilton,+Ontario,+Canada |publisher = [[IMDb|IMDb]] }}</ref>

本來預計於2008年秋季上映，但其後卻在沒有任何官方解釋之下推遲上映。[[瑞秋·麥亞當斯|瑞秋·麥亞當斯]]之後表示檔期的延遲是由於需要補拍或重拍一些鏡頭，同時亦加了一個額外鏡頭以草地為背景，可惜由於季節不同的關係而在等待<ref name="Spotlight on Rachel McAdams">{{cite web |title = "Spotlight on Rachel McAdams" |url = http://www.ew.com/ew/article/0,,20273013,00.html |publisher = [[Entertainment_Weekly|Entertainment Weekly]] "Valby, Karen" (2009-04-24)}}</ref>，而且[[艾力·班納|艾力·班納]]於另一套2009年的電影[[星空奇遇記|星空奇遇記]]中修剪了頭髮，需時長出頭髮，所以基本上是因為等待季節及艾力的頭髮而延遲了。<ref name= "How Eric Bana's shaved Trek head held up Time Traveler's Wife">{{cite web |title = "How Eric Bana's shaved Trek head held up Time Traveler's Wife" |url = http://www.scifiwire.com/2009/03/how-eric-banas-shaved-tre.php |publisher = [[Sci_Fi_Wire|Sci Fi Wire]] "Fred Topel" (2009-03-27) }}</ref>。最終這部電影由[[華納兄弟|華納兄弟]]於2009年8月14日上映。<ref name= "The Time Traveler's Wife Will Finally See Release in August">{{cite web |title = "The Time Traveler's Wife Will Finally See Release in August" |url = http://slashfilm.com/the-time-travelers-wife-will-finally-see-release-in-august/ |publisher = "Sciretta, Peter" (2009-03-12) }}</ref><ref name= "Warners moves up 'Traveler's">{{cite web |title = "Warners moves up 'Traveler's" |url = http://www.hollywoodreporter.com/hr/content_display/news/e3ia9c64386847182ffd0417818b48a123b |publisher = [[The_Hollywood_Reporter|The Hollywood Reporter]] "Carl DiOrio" (March 16, 2009) }}</ref>

==電影原聲帶==
本電影的配樂於2008年秋季由加拿大電影配樂家[[麥可·唐納|麥可·唐納]]聯同{{link-en|Hollywood Studio Symphony|Hollywood Studio Symphony}}於{{link-en|Ocean Way Recording|Ocean Way Recording}}製作的。<ref name= "Mychael Danna scores The Time Traveler's Wife">{{cite web |title = "Mychael Danna scores The Time Traveler's Wife" |url = http://www.scoringsessions.com/news/154 |publisher = ScoringSessions.com "Dan Goldwasser" (2008-12-09) }}</ref>

選用[[生命之屋合唱團|生命之屋合唱團]]的歌曲"{{link-en|Broken (生命之屋合唱團歌曲)|Broken (Lifehouse song)|Broken}}"作為預告片及宣傳片的主題曲，電視廣告則選取了沒有被收錄於原聲大碟中的{{link-en|Carolina Liar|Carolina Liar}}的歌曲"Show Me What I'm Looking For"。

此外，本電影亦收錄了由[[加拿大|加拿大]][[獨立搖滾|獨立搖滾]]樂隊{{link-en|Broken Social Scene|Broken Social Scene}}翻唱[[歡樂分隊|歡樂分隊]]的"{{link-en|Love Will Tear Us Apart|Love Will Tear Us Apart}}"。

官方原聲大碟以下載形式於2009年8月11日由{{link-en|水塔音樂|WaterTower Music|新線唱片}}發佈，實體[[CD|CD]]則由[[迪卡唱片公司|迪卡唱片公司]]發行，但一般而言只能從美國以外的銷售商購得。
