====任務型區域立委====
任務型區域立委不為個人競選，只負責不分區政黨票。

分別是[[新北市第五選舉區|新北市第五選舉區]]郭柏瑜、[[桃園市第二選舉區|桃園市第二選舉區]]張康儀、[[彰化縣第三選舉區|彰化縣第三選舉區]][[張益勝|張益勝]]<ref>{{Cite web |url = http://www.chinatimes.com/newspapers/20151126000685-260107 |title = 洪遊江、張益勝 登記選立委 |author = 吳敏菁 |date = 2015年11月26日 |publisher = 中時電子報 |language = zh-hant |accessdate = 2015年12月24日 |quote = 張益勝（36歲）首次參選，昨天下午3點30分，委託代理人代表時代力量政黨完成登記，參選理念是要求政黨透明化，支持時代力量的主張。 }}</ref>、[[雲林縣第二選舉區|雲林縣第二選舉區]][[王煒婷|王煒婷]]<ref>{{Cite web|url=http://udn.com/news/story/7876/1359221 |title=雲林立委／群像：8年級生王煒婷 要當監督者 |author=李京昇 |date=2015年12月6日 |publisher=聯合報 |language=zh-hant |accessdate=2015年12月24日 |quote=時代力量在雲林縣立委第二選區，推出王煒婷參選，她是「任務型區域立委」，不設競選總部，全力搶攻政黨票；八年級生的王煒婷年僅25歲，老家在古坑鄉荷苞村，從小喜愛繪畫，現在是平面設計師，因認同時代力量的理念，進而開啟她的政治之路，盡好本分是她此次參選目標。 |deadurl=yes |archiveurl=https://web.archive.org/web/20151223235943/http://udn.com/news/story/7876/1359221 |archivedate=2015年12月23日 }}</ref>、[[臺南市第四選舉區|臺南市第四選舉區]]蔡郁芝、[[高雄市第七選舉區|高雄市第七選舉區]][[陳惠敏_(博士)|陳惠敏]]、[[高雄市第八選舉區|高雄市第八選舉區]]陳函谷。結果雖均落選，得票數仍偏高。

====全國不分區立委====
創黨初期，時代力量全國不分區[[立委|立委]]排名名單開放給全國公民自己透過[[網際網路|網路]]一人一票選出，而不是此黨的黨主席或中央常務委員會（中常委）決定<ref>[http://www.newpowerparty.tw/candidates 時代力量不分區立委，名單自己選，排名自己排]</ref>，但符合提名資格的五位人選中，前四位的黃國昌、馮光遠、洪慈庸、林昶佐已經在區域立委參選名單，第五名[[姚立明|姚立明]]則是婉拒推薦，所以於2015年10月20日下午，時代力量主席團發布聲明表示由於該機制並未能有效產生不分區[[立法院|立法委員]]選舉候選人，將中止該活動，並改由主席團推薦及黨員連署的方式產生候選人，並經黨員及時代之友經由網路進行排序選出<ref>{{Cite web|url = https://www.facebook.com/newpowerparty/posts/1064840070254015|title = 【時代力量主席團聲明】2015.10.20|accessdate = 2015-10-20|author = 時代力量主席團|date = |publisher = }}</ref>。

時代力量推出共六位不分區名單，並於2015年11月20日正式公告黨員及時代之友網路投票結果。由於憲法中之婦女保障名額相關規定及[[柯劭臻|柯劭臻]]律師之禮讓，故由[[高潞·以用·巴魕剌|高潞·以用·巴魕剌]]擔任時代力量不分區候選人第1順位。二到六名依序為[[徐永明|徐永明]]、[[鄭秀玲|鄭秀玲]]、[[柯劭臻|柯劭臻]]、[[林依瑩|林依瑩]]，及[[柯一正|柯一正]]<ref>[http://www.storm.mg/article/74317 時代力量不分區公布 前原民台記者高潞．以用名列首位]</ref>。

時代力量政黨票取得了73萬1461票，得票率衝破5%門檻達到6.1%，可以分得2席全國不分區立法委員席次。<ref name="ltn2016">[http://news.ltn.com.tw/news/politics/breakingnews/1575069?Slots=Sps 民進黨狂掃68席「完全執政」 國民黨僅剩35席]</ref>

====民意調查====
{{main|2016年中華民國立法委員選舉#民意調查}}
選戰早期，部分民意調查顯示時代力量未能在政黨票單獨跨過百分之五的當選門檻，可能無法參與分配不分區立委席次，除非和其他政黨共組參政聯盟。據早期民進黨內部民調顯示，時代力量支持率達6.9%；其中在台北市及新北市支持度更達一成以上。但自由時報民調顯示時代力量支持度只有1.96%。

然而自2015年12月中下旬起，支持度便均有穩定達5%以上，有時甚至高於親民黨。2016年1月起，支持度衝高至六席全上的可能性。但選前受到[[周子瑜事件|周子瑜事件]]以及民進黨喊告急，強力催票影響，泛綠支持者集中政黨選票給[[民進黨|民進黨]]，造成不分區選期不如預期<ref>[http://news.ltn.com.tw/news/politics/breakingnews/1575354 催票、子瑜事件 時代力量不分區不如預期],2016.01.16 自由時報</ref>。
