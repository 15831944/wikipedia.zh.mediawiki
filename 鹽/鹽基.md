在[[香港|香港]]和[[台湾|台湾]]，'''鹽基'''或'''鹼基'''，有時也稱做'''鹼'''<ref>[http://tw.britannica.com/MiniSite/Article/id00007259.html 大英百科全書線上繁體中文版]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>（儘管「鹼」具有多種意義）。這是根據[[酸鹼理論|布朗斯特-劳里學說]]關於酸和鹽基的部份，鹽基可以簡單想像成吸收[[質子|質子]]的物質。額外的定義包括提供[[孤立電子對|孤立電子對]]（由[[劉易斯|劉易斯]]提出），以及是氫氧根離子的來源（由[[阿伦尼乌斯|阿伦尼乌斯]]提出）。

鹽基也可想像為與[[酸|酸]]化學相對。酸和鹽基的反應稱為[[中和作用|中和作用]]。鹽基可想像為與[[酸|酸]]化學相對是因為酸增加了[[水合氫離子|水合氫離子]]（H<sub>3</sub>O<sup>+</sup>）在水中的[[濃度|濃度]]，反之鹽基降低這種濃度。鹽基與酸反應生成[[水|水]]和[[鹽|鹽]]。鹽基的一般性質包括：
* '''味道'''：[[苦味|苦味]]（相對於[[酸|酸]]的[[酸味|酸味]]以及[[醛|醛]]和[[酮|酮]]的[[甜味|甜味]]）
* '''觸感'''：具黏滑和像肥皂的感覺
* '''活潑性'''：對有機物呈[[苛性|苛性]]，與酸性或可還原的物質劇烈反應
* '''導電性'''：水溶液以及熔融鹽基電離而導電
*將紅色[[石蕊試紙|石蕊試紙]]變為藍色

== 鹽基的定義 ==
法國化學家{{link-en|侯艾爾|Guillaume François Rouelle}}於1754年製定了鹽基這一概念。他注意到當時的酸大多是揮發性液體（例如[[乙酸|乙酸]]），與特定物質化合便會轉變為固態鹽。這些特定物質便是鹽的「''根基''」<ref>''The Origin of the Term Base''  William B. Jensen Journal of Chemical Education · 1130第83卷，2006年8月8日</ref>並因此得名。[[鹼|鹼]]（Alkali）是鹽基的一個特殊例子，會在水溶液中釋出[[氫氧根離子|氫氧根離子]]。'''阿累尼烏斯鹽基'''是指水溶而水溶液[[pH值|pH值]]經常大於7的鹽基。還有其他更廣義和深層的[[酸鹼理論|酸鹼定義]]。

== 鹽基和pH值 ==
量度（不純的）水的[[pH值|pH值]]，便是測試它的[[酸性|酸性]]。純水中，大約每千萬個水分子便有一個根據以下方程式電離為水合氫離子和氫氧根離子：

:2H<sub>2</sub>O(l) → H<sub>3</sub>O<sup>+</sup>(aq) + OH<sup>−</sup>(aq)

[[濃度|濃度]]即是以[[摩爾濃度|摩爾濃度]]計算（''M''或[[摩爾|摩爾]]每立方分米（mol/ dm<sup>3</sup>））水合氫離子和氫氧根離子的濃度；是為水的[[離解常數|離解常數]]，數值為10<sup>−7</sup> ''M''。pH值定義為−log [H<sub>3</sub>O<sup>+</sup>]；因此，純水的pH值是7。（這些數據均取於23 °C，並於不同溫度時有所差別。）

鹽基接收（或去除）溶液中的水合氫離子，或釋放氫氧根離子到溶液中。兩者均降低了水合氫離子在水中的濃度，從而提升pH值。相反，酸釋放水合氫離子或接收氫氧根離子而降低pH值。

例如，1莫耳的氫氧化鈉（40 [[克|克]]）溶於1公升的水中，氫氧根離子的濃度便是1摩爾每公升。所以氫離子濃度為每公升[H<sup>+</sup>] = 10<sup>−14</sup> 莫耳，而pH值 = −log 10<sup>−14</sup> = 14.

'''鹼度係數'''（'''pK<sub>b</sub>'''）是鹼性的測量，並和[[酸度係數|酸度係數]]（'''pK<sub>a</sub>'''）有以下關係pK<sub>a</sub> + pK<sub>b</sub> = 14.

[[鹼濃度|鹼濃度]]是量度溶液相對碳酸鹽或碳酸氫鹽中和酸的能力。

== 與酸的中和作用 ==
當鹽基溶解於水中，[[氫氧化鈉|氫氧化鈉]]离解為氫氧根離子和鈉離子：

:NaOH → Na<sup>+</sup> + OH<sup>−</sup>

同樣地，[[氯化氫|氯化氫]]於水中生成水合氫離子和氯離子：

:HCl + H<sub>2</sub>O → H<sub>3</sub>O<sup>+</sup> + Cl<sup>−</sup>

當兩種溶液混合，水合氫離子（H<sub>3</sub>O<sup>+</sup>）和氫氧根離子（OH<sup>−</sup>）化合生成水分子：

:H<sub>3</sub>O<sup>+</sup> + OH<sup>−</sup> → 2 H<sub>2</sub>O

如果以相同份量的氫氧化鈉溶液和氫氯酸混合，鹽基和酸會恰好完全中和，於溶液中生成氯化鈉，即[[食鹽|食鹽]]。

弱鹽基，如蘇打和蛋白，用於酸洩漏事件中的中和劑。以強鹽基如[[氫氧化鈉|氫氧化鈉]]或[[氫氧化鉀|氫氧化鉀]]中和會引起劇烈的放熱反應，同時鹽基也製造了和酸洩漏同等的破壞。

== 非金屬氧化物的鹼性 ==
[[碳酸鈉|碳酸鈉]]和[[氨|氨]]亦是鹽基，但它們都沒有氫氧根離子（OH<sup>−</sup>）。這是因為兩種都在溶解時接收氫離子（H<sup>+</sup>）：

:Na<sub>2</sub>CO<sub>3</sub> + H<sub>2</sub>O → 2 Na<sup>+</sup> + HCO<sub>3</sub><sup>−</sup> + OH<sup>−</sup>
:NH<sub>3</sub> + H<sub>2</sub>O → NH<sub>4</sub><sup>+</sup> + OH<sup>−</sup>

== 鹽基的強弱 ==
強鹽基是一種會完全[[水解|水解]]的鹽基，並提升[[pH值|pH值]]到14或以上。強鹽基，像強酸一樣，會侵襲生物組織並引致灼傷。它們的反應不同，所以強酸具腐蝕性，而強鹽基則是[[苛性度|苛性的]]。[[超強鹽基|超強鹽基]]是一類特別的鹼性化合物而[[非親核鹼|非親核鹼]]（暫無譯名）是一類缺少親核性的特殊強鹽基。而[[弱鹽基|弱鹽基]]包括用於清潔的[[氨|氨]]。

=== 強鹽基 ===
強鹽基是鹼性化合物所以能夠以酸鹼反應將弱鹽基去質子化。酸度係數大於及等於13的化合物都稱為強鹽基。常見例子包括鹼金屬和鹼土金屬的氫氧化物如氫氧化鈉及氫氧化鈣。非常強的鹽基甚至能在缺少水時將微酸性的烴去質子化。

== 鹽基作為異構催化劑 ==
鹼性物質可用作[[化學反應|化學反應]]的異構[[溶解度|不可溶]][[催化劑|催化劑]]。例如[[氧化鎂|氧化鎂]]，[[氧化鈣|氧化鈣]]以及[[氧化鋇|氧化鋇]]等金屬氧化物以及[[氧化鋁|氧化鋁]]上的[[氟化鉀|氟化鉀]]和某幾種[[沸石|沸石]]。大量[[過渡金屬|過渡金屬]]製造良好催化劑，其中不少是鹼性的。鹼性催化劑可用於[[氫化|氫化]]，[[共價鍵|雙键]]的移動，[[米尔温–庞多夫–韦尔莱还原反应|米尔温–庞多夫–韦尔莱还原反应]]，[[Michael反應|Michael反應]]，以及許多反應。

== 鹽基和鹼的混淆 ==
鹽基和鹼這兩個術語交換使用，這是由於大部分的鹽基都是鹼。「測量泥土中的[[鹼性|鹼性]]」實際是量度[[pH值|pH值]]。有時錯誤地將[[氨|氨]]歸類為鹼。

注意不是大部分[[碱金属|碱金属]]鹽都是鹼，只有本身是鹼性的鹽才是鹼。

儘管大部分[[電負性|電陽的]]金屬[[氧化物|氧化物]]都是鹼性，只有可溶的鹼金屬及鹼土金屬氧化物才可稱為鹼。

鹼是鹼金屬及鹼土金屬的鹼性鹽，但根據辭典[http://en.wiktionary.org/wiki/alkali][http://dictionary.reference.com/search?q=alkali], 對鹼的定義卻有相矛盾，包括：

* 所有可溶的鹽基[https://web.archive.org/web/20060523041052/http://www.tiscali.co.uk/reference/encyclopaedia/hutchinson/m0029936.html][http://www.thefreedictionary.com/alkali]，這應稱作[[阿累尼烏斯鹼|阿累尼烏斯鹼]]。
* 鹽基的水溶液[http://www.krysstal.com/acidbase.html] 。
