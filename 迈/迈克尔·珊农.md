'''麥可·-{zh-cn:科比特;zh-hk:高伯特;zh-tw:柯伯特;}-·夏儂'''（{{lang-en|Michael Corbett Shannon}}，{{bd|1974年|8月7日|||catIdx=S珊农}}）是一名美国演员和音樂家。他憑在《[[浮生路|浮生路]]》（2008年）和《[[迷離字戀|迷離字戀]]》（2016年）的角色而獲得兩次提名[[奧斯卡最佳男配角獎|奧斯卡最佳男配角獎]]，此外憑《[[居住正義|居住正義]]》（2014年）的角色而獲得提名[[美國演員工會獎最佳男配角|美國演員工會獎]]和[[金球獎最佳電影男配角|金球獎]]。他憑《''{{link-en|Long Day's Journey into Night|Long Day's Journey into Night}}''》得到提名{{link-en|東尼獎最佳話劇男配角|Tony Award for Best Featured Actor in a Play}}。

夏儂的第一部演出電影是《[[偷天情緣|偷天情緣]]》（1993年），而在《[[八里公路|八里公路]]》（2002年）中的演出得到了廣泛的注意。他以電影上的多才多藝而聞名，他著名的電影包括《[[珍珠港_(電影)|珍珠港]]》（2001年）、《[[重案夢幻組2|重案夢幻組2]]》（2003年）、《{{link-en|恐怖幻象|Bug (2006 film)}}》（2006年）、《{{link-en|賊兄弟連環劫|Before the Devil Knows You're Dead}}》（2007年）、《[[冷面梟雄|冷面梟雄]]》（2012年）、《[[超人：鋼鐵英雄|超人：鋼鐵英雄]]》（2013年）和《[[總統與歌神|總統與歌神]]》（2016年）。夏儂是[[傑夫·尼科爾斯|傑夫·尼科爾斯]]的慣常合作者，出現在他所有的電影中：《{{link-en|獵槍往事|Shotgun Stories}}》（2007年）、《[[歷劫重生|歷劫重生]]》（2011年）、《[[爛泥_(電影)|爛泥]]》（2012年）、《[[通天眼|通天眼]]》和《[[愛侶_(電影)|愛侶]]》（兩部都是2016年）。他也知名於[[HBO|HBO]]的時代劇集《[[酒私風雲|酒私風雲]]》（2010年–2014年）中飾演[[尼爾森·范奧登|尼爾森·范奧登]]<ref>{{Cite web |url = http://collider.com/michael-shannon-boardwalk-empire-zod-man-of-steel-interview/ |title = Michael Shannon Talks BOARDWALK EMPIRE, MAN OF STEEL, and Future Projects |author = Christina Radish |date = 2011-10-31 |publisher = Collider |language = en}}</ref><ref>{{Cite web |url = https://www.huffingtonpost.com/2011/10/31/boardwalk-empire-michael-shannon_n_1066594.html |title = ‘Boardwalk Empire’ Star Michael Shannon Talks Character Development And Being Hated |author = Jessie Heyman |date = 2011-10-31 |publisher = HuffPost |language = en}}</ref>，為他獲得提名三項[[美國演員工會獎|美國演員工會獎]]。
