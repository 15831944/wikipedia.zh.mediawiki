'''莱宁·波尔泰雷·莫雷诺·加西斯''' （{{lang-es|Lenín Boltaire Moreno Garcés}}：{{bd|1953年|3月19日|||Moreno, Lenin}}）出生于厄瓜多尔边境城市新罗卡堡 ，[[厄瓜多尔|厄瓜多尔]]政治家，现任厄瓜多尔总统。他曾经于2007年至2013年在科亚雷政府担任副总统。他的名字“莱宁”得名于[[俄国|俄国]]革命家[[列宁|列宁]]。<ref name=ax> {{cite web | title = 厄瓜多尔新总统轮椅上就职 曾遭枪击瘫痪妻子不弃| url = http://www.dzwww.com/xinwen/guojixinwen/201705/t20170525_15963639.htm }} </ref>

2016年10月1日，作为执政党[[主权祖国联盟运动|主权祖国联盟运动]]的候选人参与2017年厄瓜多尔总统大选。2017年4月2日，莫雷诺当选为厄瓜多尔总统，同年5月25日就职。<ref> {{cite web | title = 莫雷诺当选厄瓜多尔总统 竞争对手尚未承认败选| url = http://news.163.com/17/0405/11/CH8MQND1000187V5.html }} </ref>

莫雷诺在45岁时因枪击致使残疾而必须依赖轮椅行走，他也是目前世界上唯一担任国家元首的残障人士。<ref name=ax> {{cite web | title = 厄瓜多尔新总统轮椅上就职 曾遭枪击瘫痪妻子不弃| url = http://www.dzwww.com/xinwen/guojixinwen/201705/t20170525_15963639.htm }} </ref>
