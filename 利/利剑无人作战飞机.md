'''利剑无人作战飞机'''（{{lang-en|'''Sharp Sword'''}}）简称'''利剑无人机'''或'''利剑'''，是[[中华人民共和国|中华人民共和国]][[沈阳飞机设计研究所|沈阳飞机设计研究所]]设计、[[洪都航空工业集团|洪都航空工业集团]]生产的大型[[隐形战机|隐形]][[无人战斗航空载具|无人]][[攻击机|攻击机]]，采用[[飞翼布局|飞翼布局]]及隐身低流损蛇形[[进气道|进气道]]。<ref>{{cite web |url=https://www.youtube.com/watch?v=f-kcrrYJQ84 |title=永恒的追求 |publisher=中国航空报 |date=2011-04-30 |accessdate=2013-05-10}}</ref>利剑总设计师为中航工业沈阳所副所长刘志敏。<ref>{{cite web |url=http://www.hangkong.com/2014/0714/15547.html |title=中航工业刘志敏：坚守研制一线 建功重点型号 |publisher=中国航空报 |date=2014-07-14 |accessdate=2016-04-28 |archive-url=https://web.archive.org/web/20160513204803/http://www.hangkong.com/2014/0714/15547.html |archive-date=2016-05-13 |dead-url=yes }}</ref>

== 历史 ==
[[File:Лиянь.jpg|thumb]]
利剑无人作战飞机项目于2009年启动<ref name="guancha1"></ref>，首架于2012年12月13日下线，<ref>{{cite web |url=https://www.youtube.com/watch?v=ooBcRMZzxMg|title=央視-利劍在握 |publisher=中国航空在线 |date=2012-12-24 |accessdate=2013-05-10}}</ref>随后开始进行地面测试。该机先进低隐身反射进气道于2013年5月在车台进行了历时一个多月的与[[发动机|发动机]]匹配的试车摸底试验。<ref>{{cite web|url=http://www.grsx.org/2013/05/27/15/wx/|title=汪祥：心系祖国 情系航空|publisher=贵人善行|date=2013-05-27|accessdate=2014-03-10}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>2013年11月21日13时许，利剑无人作战攻击机在西南某试飞中心成功完成首飞。這也標誌著中國成為继[[美国|美国]]、[[法国|法国]]、[[英国|英国]]之后第四個成功完成專用无人作战飞机首飛的國家。<ref>[http://www.appledaily.com.tw/realtimenews/article/international/20131122/296737/%E5%82%B3%E4%B8%AD%E5%9C%8B%E7%94%A2%E7%84%A1%E4%BA%BA%E6%A9%9F%E3%80%8C%E5%88%A9%E5%8A%8D%E3%80%8D%E3%80%80%E9%A6%96%E5%BA%A6%E8%A9%A6%E9%A3%9B%E6%88%90%E5%8A%9F 傳中國無人機“利劍首飛” 蘋果日報]</ref>

“利剑”无人作战飞机验证机采用中国国产中等推力涡扇發動機。<ref>{{citeweb|title=航空报国酬壮志  长天映照赤子心——记贵州航空发动机研究所副总设计师韩冬|publisher=中工网|date=2015-04-27|url=http://gz.workercn.cn/23345/201504/27/150427112947712.shtml|deadurl=yes|archiveurl=https://web.archive.org/web/20150923060631/http://gz.workercn.cn/23345/201504/27/150427112947712.shtml|archivedate=2015-09-23}}</ref>

2016年5月24日，中航工业官方媒体《中国航空报》关于沈阳飞机设计研究所副总设计师张子军的报道，提到了某项验证项目“以全面达标的优异成绩通过验收”，取得了重大成果，突破了无人机领域多项关键技术，填补了国内空白。观察者网军事评论员认为，这一验证项目应为“利剑”隐身无人机；鉴于“利剑”的演示验证项目定位，该机通过验收有可能意味着这一项目正式结束，转为技术储备。有传言称，目前中国正在研发一款战略轰炸机，其部分技术特点与B-2较为类似。如果传言属实，那么“利剑”的部分技术也有可能应用于这一战略轰炸机上的可能。<ref name="guancha1">[http://www.guancha.cn/military-affairs/2016_05_26_361826.shtml “利剑”隐身无人机或已通过验收 相关技术可能用于下一代轰炸机]</ref>
