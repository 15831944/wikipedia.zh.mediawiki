利奥王朝也在474年-480年短暂统治西部帝国的[[达尔马提亚|达尔马提亚]]行省。

== 从[[利奥一世|利奥一世]]至[[利奥二世|利奥二世]] （457-474） ==
{{main|利奥一世}}
{{main|利奥二世}}
[[马尔西安|马尔西安]]死后，[[狄奥多西王朝|狄奥多西王朝]]在东部帝国的统治结束，[[利奥一世_(拜占庭)|利奥一世]]被阿兰裔军事指挥官[[Aspar|阿斯帕]]推上皇位，后者与当时西部的[[里西莫|里西莫]]一样，大权独揽，扮演着皇位操纵者的角色。阿斯帕曾认为利奥是个听话的傀儡皇帝，但随着年龄的增长，后者逐渐摆脱了阿斯帕的掌控。在471年阿斯帕及其子阿尔达布尔在一场叛乱中被杀后，东部帝国又完全回到了皇帝的完全掌控之中，并在接下来的几个世纪中维持在继任皇帝手中。<ref>Treadgold (1997), 152–155</ref>

在利奥一世即位之时，西部帝国几近崩溃。尽管西部皇帝[[马约里安|马约里安]]曾试图力挽狂澜，但在470年前夕，西部帝国的统治范围已经骤减至仅限于[[高卢|高卢]]北部、意大利及[[伊利里亚|伊利里亚]]行省一部分的残缺领土。利奥一世曾试图从[[汪达尔人|汪达尔人]]手中夺回[[阿非利加_(羅馬行省)|北非]]诸省，但以失败告终。<ref name="C553">Cameron (2000), 553</ref> 此后北非诸省长期脱离帝国统治，直至[[查士丁尼一世|查士丁尼一世]]在6世纪初期将其收复。

一反之前的皇帝由军事指挥官拥立的传统，利奥一世是最早被[[君士坦丁堡普世牧首|君士坦丁堡牧首]]加冕的罗马皇帝，这也反映了当时东部帝国教会与皇帝间的等级关系。 这种加冕方式将最终彻底取代由军事指挥官拥立的模式，并在中世纪时期成为惯例。

为巩固与[[伊蘇利亞|伊苏里亚人]]的同盟，利奥一世将女儿{{tsl|en|Ariadne (empress)||阿里阿德涅}}许配给了伊苏里亚人塔拉希科迪萨（Tarasicodissa），后者于466年改名[[芝諾_(羅馬帝國)|泽诺]]，也就是后来的同名皇帝。泽诺之子[[利奥二世_(拜占庭)|利奥二世]]在利奥一世死后即位，但仅仅在位11个月就去世了，皇位回到了其父泽诺手中。

== [[芝諾_(羅馬帝國)|泽诺]]和[[巴西利斯库斯|巴西利斯库斯]] (474-491) ==
{{main|芝諾 (羅馬帝國)}}
{{main|巴西利斯库斯}}
在泽诺统治时期，[[西羅馬帝國|西罗马帝国]]最终灭亡。西部帝国的结束时间尚有争议，传统说法认为，当公元476年日耳曼裔将军[[奧多亞塞|奥多亚塞]]将受其控制的小皇帝[[罗慕路斯·奥古斯都|罗慕路斯·奥古斯都]]废黜，但不再另立傀儡时，西罗马帝国正式解体。但此后一位之前曾遭废黜的西部皇帝[[尼波斯|尤利乌斯·尼波斯]]在泽诺的支持下就任西部皇帝，奥多亚塞认可了他的帝位并同意担任尼波斯政权在意大利的总督一职。 尤利乌斯·尼波斯没有重返意大利，而是继续作为名义上的西部皇帝统治[[達爾馬提亞|达尔马提亚]]地区，直到他于480年去世。480年也标志了西部帝国的彻底灭亡。

尼波斯死后，泽诺并没有再次任命一个新的西部皇帝，而是自称罗马帝国唯一的皇帝，在法理上时隔85年（狄奥多西一世去世之时）之后首次重新一统东西帝国。此后这个“罗马帝国”再未分裂。

在尼波斯去世之时，奥多亚塞本承认了与泽诺的臣属关系，但此后逐渐脱离帝国自立，于是泽诺与[[東哥德人|东哥特人]]（东哥特人之前已在[[默西亚行省|默西亚行省]]定居下来）首领[[狄奧多里克大帝|狄奥多里克]]进行协商，授予他"意大利地区军事长官"（''magister militum per Italiam''）的职位。在493年奥多亚塞死后，从小就在君士坦丁堡长大的狄奥多里克开始以东哥特人的名义统治意大利诸省。泽诺一方面认可了狄奥多里克所建立的[[東哥德王國|东哥特王国]]，另一方面则在名义上维持了对西部的统治权，并替东部帝国扫除了一个潜在的蛮族威胁。<ref name="BEv2">{{cite encyclopedia|title=Byzantine Empire|encyclopedia=Encyclopædia Britannica}}</ref>

泽诺曾在475年被[[巴西利斯库斯|巴西利斯库斯]]短暂地赶下皇位，但在12个月之后，他重登帝位，并将巴西利斯库斯全家关在一个地下蓄水池内。<ref>Elton, Hugh (10 June 1998). </ref>

== [[阿纳斯塔修斯一世|阿纳斯塔修斯一世]] (491-518) ==
{{main|阿纳斯塔修斯一世}}
[[阿纳斯塔修斯一世|阿纳斯塔修斯一世]]本是位年迈的罗马裔民政长官，他于491年与泽诺的遗孀阿里阿德涅成婚，并继承泽诺成为罗马皇帝。阿纳斯塔修斯一世是个优秀的改革家和管理者，他进一步完善了[[康斯坦丁一世|康斯坦丁一世]]所建立的货币体系，设定了当时帝国最常用货币——[[福利斯|福利斯铜币]]的标准重量。<ref>Grierson (1999), 17</ref> 阿纳斯塔修斯还取消了国内广受争议、四年征收一次的贸易税（chrysargyron）。诸如此类的货币改革使得国库逐渐充盈，在阿纳斯塔修斯一世去世时，国库中已有145150千克(320000磅)的金币储备。

继承阿纳斯塔修斯一世的是[[查士丁一世|查士丁一世]]，后者则开启了一个新的王朝——[[查士丁尼王朝|查士丁尼王朝]]。
