'''利益團體'''（{{lang-en|Interest group}}），又称'''压力团体'''<ref>{{cite news|url=http://mail.knu.edu.tw/ccchen/data/LectureNotes/303010102/Chapter-12.pdf|title=压力集团|date=|publisher=開南大學}}</ref>或'''倡议团体'''，是指具有相同利益並向[[社會|社會]]或[[政府|政府]]提出合理訴求，以爭取團體及其成員利益、影响[[公共政策|公共政策]]的[[社会团体|社会团体]]。<ref name="千華數位文化陽曄2014">{{cite book|author1=千華數位文化|author2=陽曄|author3=[升大學測、指考]|title=升大學測、指考公民與社會科[精準命中題庫+歷年試題完全破解]|url=https://books.google.com/books?id=E80aAwAAQBAJ&pg=PA42|date=9 January 2014|publisher=千華數位文化|isbn=978-986-315-745-8|pages=42–}}</ref>

利益團體是现代政治体系的重要特征，在[[二战|二战]]以后蓬勃兴起。利益集团可能介入政治生活，也可能不介入政治生活，而只有介入政治生活，并试图通过影响政府的政策和行为来实现自己利益的集团才是政治团体。美国著名政治学家[[大卫·杜鲁门|大卫·杜鲁门]]在《[[政治过程：政治利益与公共舆论|政治过程：政治利益与公共舆论]]》一书中，将利益集团界定为：一个具有共同态度的群体，通过影响政府而向社会中的其他群体提出一定的利益要求或某种声明。<ref name="王鸿志著2013">{{cite book|author=王鸿志著|title=政治狂澜的浪花: 台湾第三势力研究|url=https://books.google.com/books?id=Y0FADwAAQBAJ&pg=PA52|year=2013|publisher=九州社 崧博出版社|isbn=978-7-5108-2325-1|pages=52–}}</ref>

== 分類 ==

倡导團體可以分為經濟性利益團體和公共利益團體。前者關心的乃是成員的經濟利益，這類的團體皆為功能性的組合。他們成立的目的是旨在團結該階級或組織的力量來保障自身利益，並且為該[[階級|階級]]或[[组织_(社会学)|組織]]提供討論和交流的平台。其中最主要的類型，是[[商人|商人]]、[[勞工|勞工]]、[[農民|農民]]組成的各種團體，諸如[[商會|商會]]或[[工會|工會]]，其次是各種專業人員所組成的團體，如醫療學會、教育人士協會等。

後者主要是爭取其成員非經濟性的共同[[理想|理想]]，他們成立的目的主要是爭取公共利益或透過成立利益團體來向大眾推廣其理想。英國學者稱這類團體為[[促創團體|促創團體]]，以別於一般的經濟性利益團體。[[環保組織|環保組織]]、爭取[[婦女權益|婦女權益]]及主張保障[[弱勢社群|弱勢社群]]的組織均屬於公共利益團體。

== 左右政府決策的方法 ==
与[[政党|政党]]的诉求目的是取得政权不同，一般而言，不同類型的利益團體會透過影響[[政府|政府]]決策來保障其自身利益。

利益團體會以以下方法影響政府決策，如：
# [[遊說|遊說]]：在現代[[民主國家|民主國家]]，遊說為利益團體最重要與最直接的活動方式。是指利益團體以文字、一詞向政府官員表達個人或團體對政府政策的意見，藉此影響立法與政策的制定之行為。
# [[宣傳|宣傳]]：設法影響社會輿論以獲得民眾的支持與同情，便成為利益團體活動的重要方式之一。這類工作通常是由受過專門訓練的人員或公共關係專家透過各式各樣的[[傳播媒體|傳播媒體]]，以文字、圖畫、廣告或活動等方式，來增加民眾對於其目標之認同與支持。
# 助選：利益團體雖不推舉公職候選人，仍可透過各種助選方式幫助候選人當選後履行承諾。部分利益團體利用“[[政治獻金|政治獻金]]”左右候選人的政見，企圖影響選舉的結果。此外，助選方式還包括動員該團體成員投票支持或為其義務宣傳等。
# [[集會遊行|集會遊行]]：是民主國家中言論自由保障的一環，透過這樣的動員、直接參與的方式，可以增強團體成員的向心力，並藉由大眾傳播達到宣傳效果，進而匯聚民意向政府施壓、迫使政府讓步。
# 司法途徑：利益團體也可能採取司法訴訟或聲請[[大法官|大法官]]釋憲來表達訴求，以連署的方式要求立委聲請大法官釋憲。經由法院判決或釋憲結果，不僅能爭取到特定權益，還可能引起輿論關注而提高能見度。

而利益團體運作的成效，取決於本身的組織資源(包括內部資金、人力的多寡)、內部成員的向心力、領導者的風格、組織內部是否了解行政與立法機構的運作模式，以及組織本身在政府部門的信譽等因素。

== 影響力 ==
以下為影響利益團體影響力的因素：

* 組織及[[動員能力|動員能力]]
* [[領導|領導]]能力
* 公眾支持度
* [[財政資源|財政資源]]
* [[社會地位|社會地位]]
* 與[[政府|政府]]的關係；如與政府關係密切或與[[執政黨|執政黨]]的[[政治理念|政治理念]]接近者較易保障其利益

== 功能 ==
* 代表及反映社會中不同階級的利益及意見
* 監察政府運作：部份利益團體在[[立法機關|立法機關]]中擁有議席，有必要代表公眾監察政府的[[決策|決策]]，如指出政府政策的流弊，並向政府施壓。
* 推動公民意識及[[教育|教育]]公眾：利益團體會製造話題，以圖引起社會關注，並從中教育相關的知識
* 提高政府[[問責性|問責性]]
* 提供參與[[社會事務|社會事務]]的機會
* 推動[[民主|民主]]及[[人權|人權]]發展
* 促進社會[[多元化|多元化]]及[[自由化|自由化]]
