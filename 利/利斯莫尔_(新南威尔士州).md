'''立斯摩爾'''（'''Lismore'''）是個[[澳大利亞|澳大利亞]][[新南威爾斯州|新南威爾斯州]]有著[[亞熱帶氣候|亞熱帶氣候]]的小城市，乃澳東沿海的主要城鎮之一；人口約2萬7（2006年）。

== 教育 ==
[[南十字星大學|南十字星大學]]的校總區位於本市，今為澳大利亞提供遠程教學排名第七大的高等學府。

== 夜生活 ==
立斯摩爾與地處澳洲最東部的[[拜倫灣|拜倫灣]]比鄰。當地的溫山飯店（Winsome Hotel）是個同性戀人於夜晚聚會的著名地點。鄰近的夜店「The One」與「The Standard」也同樣著名。此外，從週四至週六，南十字星大學的「大學酒吧」則有為數眾多的大學生在此歡聚和跳舞。而在[[中央商務區|中央商業區]]（CBD）則有許多專為一般社會人士所開設的社交型夜店。

== 地方美食 ==
如果觀光客-{欲}-享用一頓美好的晚餐，在橋街（Bridge street）的「兩千頭牛餐廳」（20,000 cows restaurant）會是個絕佳的好去處。而「Fire in the Belly」比薩餐廳則是於2006年榮獲「澳大利亞最佳比薩大獎」的著名餐館。此外，本市由於鄰近咖啡豆產區，所以市區的咖啡館多採用當地所生產的咖啡，以利地方經濟。
