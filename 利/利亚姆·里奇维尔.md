'''利安·馬菲·列治維'''（{{lang-en|'''Liam Matthew Ridgewell'''}}，{{bd|1984年|7月21日|catIdx=Ridgewell, Liam}}）是一名[[英格蘭|英格蘭]]職業足球員，司職[[後衛_(足球)|後衛]]，場上最佳位置是擔任中堅，亦可勝任左閘，現時效力[[英冠|英冠]]球會[[赫爾城足球俱樂部|候城]]。列治維出身於[[阿士東維拉足球會|阿士東維拉]]，於2002年曾短暫借用到[[伯恩茅斯足球俱乐部|般尼茅夫]]，其後於2007年轉投同市死敵[[伯明翰足球會|伯明翰城]]，是球隊贏得[[2011年英格蘭聯賽盃決賽|2011年聯賽盃]]的冠軍功臣之一。

== 生平 ==
=== 球會 ===
;早年
列治維生於[[倫敦|倫敦]][[Bexleyheath|博克斯雷希斯]]（Bexleyheath）<ref>{{cite web |url=http://www.skysports.com/football/player/0,19754,11694_222093,00.html |title=Liam Ridgewell |publisher=''Sky Sports'' |accessdate=8 September 2010}}</ref>，初時在[[西汉姆联足球俱乐部|韋斯咸]]接受足球訓練，其後於2001年2月轉投[[阿士東維拉足球會|阿士東維拉]]<ref name=WestHam>{{cite web |title=Ridgewell relishes Hammers return |url=http://www.whufc.com/articles/20100609/ridgewell-relishes-hammers-return_2236884_2055691 |publisher=[[西汉姆联足球俱乐部|''www.whufc.com'']] |accessdate=1 August 2011 |archive-url=https://web.archive.org/web/20100613024035/http://www.whufc.com/articles/20100609/ridgewell-relishes-hammers-return_2236884_2055691 |archive-date=2010-06-13 |dead-url=yes }}</ref>。列治維是維拉在2002年5月擊敗[[埃弗顿足球俱乐部|愛華頓]]贏得[[英格蘭足總青年盃|英格蘭足總青年盃]]錦標的成員之一，於比賽中他成功拑制[[韦恩·鲁尼|朗尼]]是球隊取勝的主因<ref name=Youthcup>{{cite web|title=Liam Ridgewell|url=http://www.carling.com/football/aston-villa-fc/liam-ridgewell.html|publisher=www.carling.com|accessdate=1 August 2011}}</ref>。

;阿士東維拉
在列治維於2002年10月為[[英格蘭19歲以下足球代表隊|英格蘭U19]]取得入球賽和南斯拉夫U-19後一天<ref>{{cite web |title=England Under 19's fixtures and results |url=http://www.ntfc.co.uk/page/NewsDetail/0,,10425~295188,00.html |publisher=www.ntfc.co.uk |accessdate=1 August 2011}}</ref>，他獲外借到[[伯恩茅斯足球俱乐部|般尼茅夫]]汲取經驗，於10月13日至11月13日期間上陣5場。直到2003年1月4日列治維才首次為維拉一隊上陣，於[[英格蘭足總盃|英格蘭足總盃]]1-4負於[[布莱克本流浪者足球俱乐部|布力般流浪]]以後補身份登場。2006年9月10日列治維在對[[西汉姆联足球俱乐部|韋斯咸]]取得個人歷來首個入球。
