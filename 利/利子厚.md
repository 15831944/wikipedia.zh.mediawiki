'''利子厚'''{{JP}}（'''Michael Lee'''，{{bd|1961年|5月13日|catIdx=}}），是[[利漢釗|利漢釗]]、[[利定昌|利定昌]]的堂弟，[[利銘澤|利銘澤]]、[[利孝和|利孝和]]、[[利榮傑|利榮傑]]侄兒，[[利承武|利承武]]堂叔父。現任[[香港賽馬會|香港賽馬會]][[董事|董事]]<ref>[http://www.hkjc.com/chinese/news/news_2006083113817.htm 2006年8月，利子厚新任香港賽馬會董事]</ref>，前[[香港|香港]][[上市公司|上市公司]][[希慎興業|希慎興業]]的公司董事總經理，[[利希慎家族|利希慎家族]]第五代成員，[[父親|父親]]及[[母親|母親]]是[[利榮達|利榮達]]及[[方笑寬|方笑寬]]，祖父是「[[鴉片|鴉片]]大王」[[利希慎|利希慎]]。

== 在任功過 ==
利子厚在2003年任希慎興業[[董事|董事]][[總經理|總經理]]。公司業務以物業投資、出租為主，其間[[董建華|董建華]]年代，[[香港經濟|香港經濟]]不好景，再者如[[國際金融中心_(香港)|國際金融中心]]等同行[[寫字樓|寫字樓]]業主競爭客源。利子厚在任前，1996年希慎以36億港元購入的中環[[娛樂行|娛樂行]]，在2005年9月27億元出售，損手9.4億，可算少數目。2007年5月8日股東大會通過，利子厚退任希慎興業職務。<ref>[http://www.mpfinance.com/htm/Finance/20070404/News/ec_eca1.htm 2007年5月8日，利子厚退任希慎興業職務]</ref> <ref>[http://kui.name/NEWS/HK/2007-04-04/56957.html 利子厚帶領希慎，走過經濟低谷。]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

== 家庭狀況 ==
利子厚於2012年6月和拍拖三年的蒙古女友Munkhzul Odkhuu結婚。<ref>[http://orientaldaily.on.cc/cnt/news/20120528/00176_078.html 中環出更：利子厚娶外蒙妻「土」製婚禮]</ref>但後來離婚，於2016年和[[龐卓貽|龐卓貽]]（[[紹榮鋼鐵|紹榮鋼鐵]]龐熙女兒）低調再婚。

== 名下馬匹 ==
利子厚是[[香港賽馬會|香港賽馬會]]會員及董事，他的名下馬匹包括熱鬥勇士（與葉家祺、華國威共同擁有）、熱鬥戰士（與葉家祺、何志堅共同擁有）、奧馬精神、野外桃源、惺惜犬馬（個人名下）、擦出火、百戰百勝、天下為攻（與陳衍里醫生共同擁有，2018年[[主席短途獎|主席短途獎]]冠軍）、勁酒王（駿馬團體名下）、愛馬寶、愛馬奔騰、醉翁意（愛馬者團體名下，成員包括林詩鍵、林棣權、霍廣行、盧楚鏘等）、喜悅繽紛（好玩得團體名下，成員包括林詩鍵、林詩棋、林棣堅、霍廣行、盧楚鏘、羅德榮等）。
