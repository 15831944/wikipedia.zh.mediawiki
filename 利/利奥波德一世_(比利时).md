'''利奥波德一世'''（[[荷蘭語|荷蘭語]]、[[德語|德語]]：'''Leopold I'''，{{lang-fr|'''Léopold I<sup>er</sup>'''}}，{{bd|1790年|12月16日|1865年|12月10日|}}），原名'''利奥波德·乔治·克里斯蒂安·弗里德里希'''（{{lang-de|Leopold Georg Christian Friedrich}}）；
出生于[[德国|德国]][[科堡|科堡]]，德國薩克森-科堡-薩爾菲德公爵。1831年7月21日成為比利时的第一位国王，開創了比利時[[薩克森-科堡-哥達王朝|薩克森-科堡-哥達王朝]]。

他的子女包括比利時國王[[利奧波德二世|利奧波德二世]]和墨西哥王后[[瑪麗-夏洛特|瑪麗-夏洛特]]。還是英國女皇[[维多利亚_(英国君主)|維多利亞]]的丈夫的叔叔及堂姐夫（其第一任妻子夏洛特是維多利亞的堂姐）和舅父。

== 早年生活 ==
利奥波德一世是德国萨克森-科堡-萨尔菲德公爵的幼子。他首先开始从军，1795年五岁的他竟然升为上校，1802年俄罗斯陆军将军。1806年[[拿破仑|拿破仑]]占领他父亲的公爵领地后他去[[巴黎|巴黎]]。拿破仑向他提供作为自己的助手的地位，但利奥波德拒绝。他代替他的哥哥接受了公爵地位而成为拿破仑的反对者。1815年他晋升为[[元帅|元帅]]。

1816年5月2日他与当时国王[[乔治三世|乔治三世]]的孙女、王储[[乔治四世|乔治四世]]的女儿[[夏洛特公主_(威尔士)|夏洛特公主]]结婚。他的妻子夏洛特本有望继承英国王位。但在1817年11月6日，夏洛特因[[產科三重悲劇|難产逝世]]。这样一来英国的王位继承人就空缺了，利奥波德促成了他的孀居姐姐[[维多利亚公主_(萨克森-科堡-萨尔费尔德)|维多利亚]]与[[乔治三世|乔治三世]]的四子[[爱德华王子_(肯特及斯特拉森公爵)|爱德华王子]]的婚姻，维多利亚与爱德华的女儿后来成为了著名的[[维多利亚女王|维多利亚女王]]。

1829年利奥波德与一个酷似夏洛特的演员结婚，出于社会上的反对，这个婚姻既没有进行宗教的、又没有进行公开的婚礼仪式。据文献这个婚姻于1831年结束。

== 比利時国王 ==
1830年[[希腊|希腊]]推选利奥波德为其国王，但利奥波德没有接受这个王位。1830年10月4日比利时从[[荷兰|荷兰]]独立后比利时国会问利奥波德他是否愿意成为比利时国王。1831年7月17日利奥波德在比利时德帕内登陆，7月21日他向比利时宪法宣誓，这一天从此成为比利时的国庆节。一周半后荷兰军队入侵比利时。八年后，1839年荷兰承认《[[1839年伦敦条约|伦敦条约]]》，承认比利时的独立。

1832年8月9日他与[[法国|法国]]国王[[路易·腓力|路易·菲利普]]的长女[[路易丝王后_(比利时)|路易斯-玛丽公主]] 结婚，两人有三男一女：
# [[路易·菲利普王储（比利时）|路易·菲利普·利奥波德·维克多·恩斯特]]（1833年—1834年，早夭）
# [[利奥波德二世_(比利时)|利奥波德二世]]（1835年—1909年）传位侄子阿尔贝一世
# 菲利普（1837年—1905年）佛兰德伯爵，阿尔贝一世之父
# 玛丽-夏洛特（1840年—1927年），1857年与[[墨西哥|墨西哥]]皇帝[[马西米连诺一世_(墨西哥)|马西米连诺一世]]结婚

利奥波德一世本人终身保持他的[[新教|新教]]信仰，但他的王后和儿女与比利时大部分居民一样信仰[[天主教|天主教]]。此外他还与他的情婦有两个儿子。

利奥波德一世赞成和帮助建立了[[歐洲大陸|欧洲大陆]]上的第一条铁路（从[[布鲁塞尔|布鲁塞尔]]到[[梅赫伦|梅赫伦]]）。1835年5月5日他参加了这条铁路的通车典礼。

1840年他促成了维多利亚女王的婚姻。1842年他试图发表一个控制妇女和儿童劳工的法律，但被议会驳倒。由于他的外交和内政上的妥善处置，[[1848年革命|1848年革命]]没有波及比利时。
