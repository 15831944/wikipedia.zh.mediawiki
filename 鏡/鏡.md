'''鏡子'''，也稱'''鑑'''，是一種表面光滑，具[[反射_(物理學)|反射]]光線能力的物品。最常見的鏡子是平面鏡，常被人們利用來整理儀容。在科學方面，鏡子也常被使用在和[[望遠鏡|望遠鏡]]、[[雷射|雷射]]、工業器械等儀器上。

== 成像原理 ==
[[File:Mirror_image_reversal.svg|thumb]]
不論是平面鏡或者是非平面鏡（凹面鏡或凸面鏡），光線都會遵守[[反射定律|反射定律]]而被面鏡[[反射|反射]]，反射光線進入眼中後即可在[[視網膜|視網膜]]中形成視覺。

在平面鏡上，當一束平行光束碰到鏡子，光束會被平行地反射出去，此時的成像和眼睛所看到的像相同。平面鏡成的像所在位置並無實際物體或光線，所以形成的是[[虛像_(光學)|虛像]]。

== 鏡子的運用 ==
[[File:MFSTWLEndOfPlaform.JPG|200px]]
[[File:ICAC_Building_Identification_Parade_Suite_View1.jpg|thumb]]
[[File:道路安全反射鏡.JPG|thumb]]
*[[梳妝鏡|梳妝鏡]]：被用來協助[[化妝|化妝]]、刮鬍子及梳髮等整理儀容的工具。
*[[儀器|儀器]]：許多光學儀器、例如[[望遠鏡|望遠鏡]]及[[顯微鏡|顯微鏡]]的光路中，會利用鏡子來進行反射。
*安全：例如交通工具的後視鏡和照後鏡。有些道路的轉角會擺置凸面鏡，提醒往來行人注意安全。
*[[单向玻璃|單面反光鏡]]：例如證人在警局認人手續中，用作辨認疑犯用的單面鏡。 [https://web.archive.org/web/20060903120841/http://phlife.7456.net/show/2/23/]

== 玻璃镀银制镜技术 ==
=== 原料及要求 ===
# [[玻璃|玻璃]]应是平整无缺，中间无气泡的透明玻璃；
# [[硝酸银|硝酸银]]，含量在99.5%以上；
# [[氨水|氨水]]，[[浓度|浓度]]为25—28%；
# [[酒石酸钾钠|酒石酸钾钠]]，要化学纯。

=== 制作方法 ===
# 清洗[[玻璃|玻璃]]：按规格裁好[[玻璃|玻璃]]后，先用[[自来水|自来水]]冲洗正反两面，然后将铁红粉带水涂在要镀的一面，待干后擦去铁红粉，水洗干净。再用微量的[[氯化锡|氯化锡]][[溶液|溶液]]擦洗[[玻璃|玻璃]]要镀的面。洗后用水冲净残余的[[氯化亚锡|氯化亚锡]]。最后用干净的水（最好用蒸馏水）冲一下[[玻璃|玻璃]]。
# 镀银：将洗干净的[[玻璃|玻璃]]平放在水平的木架或木条上，取银液一份和还原液一份搅拌匀倒上。药液以不流掉为度。约每平方[[公尺|米]]2[[分升|分升]]左右。待其渐渐在[[玻璃|玻璃]]上反应出银镜，将多余的药液倒掉，用水冲洗，倒上百分之一的[[明胶|明胶]]晾干。干后再在上面涂一层铁红底漆或其它防锈漆液便成了镜子。

=== 药液配方 ===
# 银液：蒸馏水（冷开水也可）2500毫升，[[硝酸银|硝酸银]]25克，[[氨|氨]]水18.5毫升（经化学反应澄清为止）。
# 还原液：蒸馏水（冷开水也可）2500毫升，[[酒石酸钾钠|酒石酸钾钠]]25克，上液加热澄清后再放入[[硝酸银|硝酸银]]0.5%，药液守滤后备用。
# [[明胶|明胶]]液：水1000毫升，[[明胶|明胶]]10克，隔水蒸化。
# 铁红底[[漆|漆]]加适量[[香蕉水|香蕉水]]溶液。
