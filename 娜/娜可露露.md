{{vgname2|娜可露露|ja=ナコルル|aka=娜考露露、娜考璐璐}}是由日本電子遊戲商[[SNK|SNK]]所推出；[[格鬥遊戲|對戰格鬥遊戲]]系列[[侍魂系列|侍魂]]裡所登場的女性角色。

== 作品描述 ==
在遊戲設定中，娜可露露是居住於[[北海道|北海道]]裡名叫神威（{{lang|ja|カムイ}}）的[[阿伊努人|阿伊努人]]部落裡的少女；並擔任著部落中與神靈溝通的巫女。外觀上為黑色長髮、身著阿伊努人傳統花樣服飾。性格上娜可露露喜愛著大自然的一切事物，並有著能傾聽大自然聲音的特殊能力，而娜可露露不喜殺戮；因此在面臨衝突時會盡可能避免以武力解決。在私下娜可露露擅長調理「{{lang|ja|ラタシケプ}}」；一種混合野菜與油脂的阿伊努人傳統飲食。<ref>{{cite web |url=http://www.d1.dion.ne.jp/~latel/samurai/q_a/mest3.html |script-title=ja:天草降臨ファンブックより抜粋 |work={{lang|ja|侍魂の部屋}} |accessdate=2016年2月6日}}</ref>

而遊戲劇情中娜可露露會踏上戰鬥旅程是因有感到一股邪惡勢力將摧毀她所居的自然家園，為了守護族人娜可露露便攜帶著他父親留下的短刀齊齊烏西（{{lang|ja|チチウシ}}）、以及她的獵鷹夥伴瑪瑪哈哈（{{lang|ja|ママハハ}}）踏上了戰鬥旅程。雖然在侍魂系列第二作《[[真侍魂_霸王丸地獄變|霸王丸地獄變]]》中，娜可露露的角色結局是她為了讓遭受摧毀的自然還原，自願犧牲性命讓神祇利用她的生命之力將大地重生，但往後SNK仍安排娜可露露在侍魂後續作品登場。而為了避免遊戲劇情與《霸王丸地獄變》「娜可露露犧牲生命」的結局產生矛盾；侍魂第三作《[[侍魂_斬紅郎無雙劍|斬紅郎無雙劍]]》、或第四作《[[侍魂_天草降臨|天草降臨]]》的時間點都是安排在《霸王丸地獄變》之前。而SNK在1999年推出、故事線是發生在侍魂初代遊戲20多年後的《[[劍客異聞錄_甦醒的蒼紅之刃_侍魂新章|甦醒的蒼紅之刃]]》娜可露露仍有登場，SNK則將當時《霸王丸地獄變》的結局解釋為娜可露露並未死去；而是將靈魂與大地之母合為一，且娜可露露在該遊戲中是化身為阿伊努人民間傳說裡的小妖精[[克魯波克魯|克魯波克魯]]模樣登場。<ref>{{cite web |url=http://www.samuraispirits.net/game/info/sswrdrama/drama2.html |title=Samurai Shodown Warriors Rage Mini Drama Translation |work=Samuraispirits Net |accessdate=2016年2月6日}}</ref>

娜可露露在遊戲裡的戰鬥方式是以短刀搭配她敏捷的速度來攻擊；並且指揮她的獵鷹夥伴瑪瑪哈哈協助突擊對手。<ref name="neogeomuseum">{{cite web |url=http://neogeomuseum.snkplaymore.co.jp/whats/ |title=What'S Neogeo? |work=Neogeo Museum |date=2010年2月27日 |accessdate=2016年2月6日}}</ref>在侍魂第三作《斬紅郎無雙劍》則增加[[北海道狼|蝦夷狼]]{{nihongo|西克魯|シクルゥ}}作為娜可露露的新動物夥伴。

在娜可露露與侍魂系列人物的關係中；遊戲主角日本劍豪[[霸王丸|霸王丸]]是與娜可露露多次合力打倒邪惡勢力的戰友，而來自法國的女劍客{{nihongo|夏洛特|シャルロット}}則是她敬仰的女性，另自美國來到日本學習忍術的美國忍者{{nihongo|加爾福特|ガルフォード}}是對娜可露露抱持好感的男性。娜可露露的妹妹{{nihongo|莉姆露露|リムルル}}最初是在侍魂系列第二作《霸王丸地獄變》擔任背景角色，從第三作《斬紅郎無雙劍》起則成為也可讓玩家使用的角色。

=== 紫娜可露露 ===
多數格鬥遊戲在選擇同位角色對戰時，兩方操控的人物在外型上僅有色調的不同。而娜可露露是唯一在侍魂系列眾遊戲人物裡，第二位角色（2P）會在勝利畫面裡與第一位角色呈現不一樣的表情。{{ref|SS3}}

在2P版本娜可露露的勝利畫面中，多呈現著較兇悍、或是狂傲笑容的神情。另2P版本娜可露露的色調上是黝黑皮膚、搭配紫色條紋花樣的衣裝外觀，遊戲界則多以{{nihongo|「紫娜可露露」|紫ナコルル}}作為稱呼。

紫娜可露露隨後常成為《侍魂》週邊動漫作品、或是其它非官方[[二次創作|二次創作]]所採用的題材，她有被描述成是藏在娜可露露內心裡另一個較黑暗、偏激、好鬥的人格；或是孿生的姊妹，由[[島本和彥|島本和彥]]繪製的侍魂漫畫便有採用紫娜可露露在劇情中。另外日本女性漫畫家[[七瀨葵|七瀨葵]]有多次發表以娜可露露和紫娜可露露之間關係為主題的[[同人誌|同人誌]]，因受到許多愛好者的正面回應；而被SNK邀請替1999年推出的動畫{{nihongo|《侍魂2 ～阿斯拉斬魔傳～》|SAMURAI SPIRITS 2 〜アスラ斬魔伝〜}}擔任腳色設計，<ref>{{cite book |title=Manga Design |publisher=Taschen GmbH |author=Masanao Amano |isbn=978-3-822-82591-4 |pages=384頁 |date=2004年8月27日}}</ref>在該動畫也有特別提及紫娜可露露的存在。

=== 蕾拉 ===
在2001年SNK發行的以娜可露露為主角的冒險遊戲{{nihongo|《娜可露露～給你的禮物～》| ナコルル~あのひとからのおくりもの~}}裡，以先前的紫娜可露露為基礎新增一位名為{{nihongo|蕾拉|レラ}}的角色。

「蕾拉」的讀音在[[阿伊努語|阿伊努語]]裡有「[[風|風]]」的意思，<ref>{{cite web |url=http://www.tsuruga.com/concept/bunka.html |script-title=ja:阿寒という文化について |work=Akan Yuku No Sato Tsuruga  |date=年月日 |accessdate=2016年2月6日}}</ref>遊戲中她被設定為娜可露露內心裡「為了保護內心重視的事物、可以不擇任何手段」的一面，並且脫離了娜可露露本尊；成為時而現身於娜可露露面前現身、時而消失像鬼魅般的神秘形體。蕾拉內心雖與娜可露露同樣喜愛著自然以及家人朋友，但在面對阻礙目的或威脅安危的敵手時；蕾拉可以不猶豫地去奪取對方生命，而蕾拉不同於先前紫娜可露露會有狂傲的神情；她是被設定為總是面貌冷酷的少女。後續在2003年的侍魂系列遊戲《[[侍魂_零|零]]》裡，SNK也讓蕾拉登場成為可被玩家使用的角色。

而往後為了與娜可露露作出區別，原先是娜可露露的動物夥伴之一蝦夷狼西克魯便變更為是與蕾拉配對。

== 衍生登場作品 ==
侍魂系列裡娜可露露是少數能有個人主題作品的角色，SNK除了在2001年發行以她為主角的遊戲《娜可露露～給你的禮物～》外，也將該遊戲隔年另製成[[OVA|OVA]]動畫發行。<ref>{{cite web |url=http://myanimelist.net/anime/1937/Nakoruru__Ano_Hito_kara_no_Okurimono |title=Nakoruru: Ano Hito kara no Okurimono |work=MyAnimeList |accessdate=2016年2月6日}}</ref>

娜可露露也常被SNK挑作為[[跨界作品|跨界作品]]對戰格鬥遊戲的人選，像是《[[CAPCOM_VS._SNK_千年之戰|CAPCOM VS. SNK 千年之戰]]》、<ref>{{cite web |url=http://uk.ign.com/articles/2000/11/13/capcom-vs-snk-millennium-fight-2000 |title=Capcom vs. SNK: Millennium Fight 2000 |work=[[IGN|IGN]] |author=Jeremy Dunham |date=2000年11月13日 |accessdate=2016年2月6日}}</ref>《[[時空亂鬥|時空亂鬥]]》、《[[拳皇XIV|拳皇XIV]]》等等都有娜可露露的出現。<ref>{{cite web |url=http://www.4gamer.net/games/088/G008877/20130613097/ |script-title=ja:「LOST SAGA」にサムスピの「ナコルル」が6月20日から参戦。お供の鷹，ママハハと連携して攻撃を繰り出す |work=4Gamer.Net |author=Yusuke |date=2013年6月13日 |accessdate=2016年2月6日}}</ref>

== 反應 ==

=== 評價 ===
{{quote box|quoted=true|width=30%|salign=center|quote=儘管她外型嬌小且性格溫和；娜可露露可說是格鬥遊戲裡最初的「柔弱但靈敏」風格角色之一；此一特性也讓她在侍魂系列裡與其他陽剛男性角色群突顯出來。娜可露露內心潛在著兩個自我意識讓她時而與獵鷹瑪瑪哈哈合作；或是時而與蝦夷狼西克魯搭配之情況是她吸引人的特質。娜可露露可謂是SNK眾多遊戲角色裡的傑作之一；並且不是單純靠著賣弄性感模樣方式來得到大家的歡迎。|source=出於電子遊戲網站[[GamesRadar|GamesRadar]]在2012年7月20日對於娜可露露的評論。<ref name="Top7">{{cite web |url=http://www.gamesradar.com/top-7-best-fighting-game-characters/ |title=The Top 7… Best fighting game characters |work=[[GamesRadar|GamesRadar]] |author=Lucas Sullivan |date=2012年9月10日 |accessdate=2016年2月6日}}</ref>}}

自侍魂系列登場後，娜可露露在日本國內或海外都有獲得到不差的人氣。<ref name="neogeomuseum"/>其中西班牙的電子遊戲《Hobby Consolas》表示如果SNK的格鬥遊戲[[餓狼傳說系列|餓狼傳說系列]]的女角代表是[[不知火舞|不知火舞]]；那侍魂系列無疑是娜可露露，且娜可露露是不以性感的方式來呈現出她迷人的一面。<ref>{{cite web|url=https://archive.org/stream/HobbyConsolas055/Hobby_Consolas_055#page/n111/mode/2up/ |title=Hobby Consolas 055 |publisher=Hobby Consolas |accessdate=2016年2月6日}}</ref>而她愛護自然的形象有被電子遊戲網站[[GamesRadar|GamesRadar]]形容為「像是日本封建時代版本風格的[[地球超人|地球超人]]」<ref name="Top7"/>

而她在遊戲裡角色性能；有被美國的電子遊戲雜誌《[[GamePro|GamePro]]》指出娜可露露在最初《侍魂》首代登場裡性能強度太弱，破壞了該款作品的遊戲強弱平衡度。<ref>[[GamePro|GamePro]] 1993年10月, (51): 114頁.</ref>另一方面《[[電子遊戲月刊|電子遊戲月刊]]》則反向表示儘管娜可露露在該款遊戲裡外形並不高大、但卻會是個給予人致命的角色。<ref> Electronic Gaming Monthly (EGM Media, LLC). 1993年9月, (50): 167頁.</ref>

另娜可露露的動物夥伴獵鷹瑪瑪哈哈曾被雜誌《Complex》評為格鬥遊戲裡最令人感到誇張的十大戰鬥武器之一、<ref>{{cite web |url=http://ca.complex.com/pop-culture/NaN/Invalid%20date/the-10-craziest-weapons-in-fighting-games/4 |title=The 10 Craziest Weapons in Fighting Games |work=Complex |author=Rich Knight |date=2012年1月21日 |accessdate=2016年2月6日 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>以及被[[音樂電視網|音樂電視網]]頻道的專欄列為電子遊戲中最偉大的鳥類角色名單裡。<ref>{{cite web |url=http://www.mtv.com/news/2456668/help-us-name-the-greatest-bird-in-video-game-history/ |title=Help Us Name The Greatest Bird In Video Game History  |work=[[音樂電視網|音樂電視網]] |author=Tracey John |date=2008年4月6日 |accessdate=2016年2月6日}}</ref>

=== 社會採用 ===
在1994年日本[[三鷹市|三鷹市]]市公所的水道部門有請動畫家[[結城信輝|結城信輝]]繪製一幅請民眾關心水資源的公益海報；而該海報的內容主題人物便是採用娜可露露。<ref name="neogeomuseum"/>

SNK也多次採用娜可露露作為公司的形象角色，如2007年便以娜可露露和SNK旗下另款格鬥遊戲《[[餓狼傳說|餓狼傳說]]》的主角[[泰瑞·柏格|泰瑞·柏格]]；作用為公司所成立的社會公益團體{{nihongo|「娜可露露與泰瑞俱樂部」|ナコルル＆テリークラブ}}之吉祥物角色，其中喜愛自然的娜可露露是作為該團體支持環保運動的代表。<ref>{{Cite web|url=http://www.snkplaymore.co.jp/nakoruru_terry_club/index.php |archiveurl=https://web.archive.org/web/20071030035702/http://www.snkplaymore.co.jp/nakoruru_terry_club/index.php |script-title=ja:ナコルル＆テリークラブ |archivedate=2007年10月30日 |accessdate=2016年2月6日 |deadurl=yes }}</ref>另2011年日本發生[[2011年日本東北地方太平洋近海地震|東北地方太平洋地震]]災情後，SNK有請曾任侍魂遊戲角色繪師的[[白井影二|白井影二]]繪製一張娜可露露祈禱圖像；作為公司替災後社會聲援的代表。<ref>{{cite web |url=http://www.kotaku.jp/2011/03/jishin_nakoruru.html |script-title=ja:大自然の祈り。白井影二さんの『侍魂』ナコルル応援イラスト |work=[[Kotaku|Kotaku]] |author=金本太郎 |date=2011年3月16日 |accessdate=2016年2月6日}}</ref>
