'''渥太华强奸危机中心'''（{{lang-en|Ottawa Rape Crisis Centre}}）是一家加拿大机构, 致力于结束一切形式的性暴力<ref>{{cite web |url=http://orcc.net/index.html |title=存档副本 |accessdate=2011-03-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20110602092828/http://www.orcc.net/index.html |archivedate=2011-06-02 }}</ref>。渥太华强奸危机中心于1974年成立，是加拿大境内{{what|第三家强奸危机操作中心}}<ref>[http://ocrcc.wildapricot.org/Resources/Documents/Newsletter-Spring2010%20(3).pdf]</ref>。该组织的理念是主动地反种族主义、反压迫和女权主义<ref>[http://www.volunteerottawa.ca/vo-clean/index.php?/eng/i_want_to_volunteer/community_organizations&startwith=%20&agencyId=694] {{webarchive|url=https://web.archive.org/web/20110706212437/http://www.volunteerottawa.ca/vo-clean/index.php?%2Feng%2Fi_want_to_volunteer%2Fcommunity_organizations&startwith=&agencyId=694 |date=2011-07-06 }}</ref>。

== 服务 ==
渥太华强奸危机中心为16岁以上的女性提供服务。这些服务包括24小时危机热线、咨询服务和公共教育项目。同时也向遭受性侵犯者的朋友、家人和伴侣提供咨询服务。多种项目和工程也在进行中，例如在渥太华卡尔顿拘留中心与被监禁的女性一起工作<ref>{{cite web |url=http://www.sexualassaultsupport.ca/ottawa |title=存档副本 |accessdate=2011-03-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20110706202854/http://www.sexualassaultsupport.ca/ottawa |archivedate=2011-07-06 }}</ref>，在当地高中里实行的女孩聊天项目<ref>{{cite web |url=http://www.girlsactionfoundation.ca/en/member/ottawa-rape-crisis-centre |title=存档副本 |accessdate=2011-03-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20110706181837/http://www.girlsactionfoundation.ca/en/member/ottawa-rape-crisis-centre |archivedate=2011-07-06 }}</ref>。对于有移民和难民背景的年轻女性<ref>[http://www.unb.ca/fredericton/arts/centres/mmfc/_resources/pdfs/6onyalo.pdf]</ref>，还有一个安全城市网站，为年轻人提供互动的机会。

== 历史 ==
1974年12月15日，渥太华强奸危机中心由一小群女性正式开张。在此之前，渥太华社区内没有任何组织为遭受过性侵犯的女性提供服务。该中心是当时加拿大仅有三家的强奸危机中心之一<ref>[http://www.volunteerottawa.ca/vo-clean/index.php?/eng/i_want_to_volunteer/community_organizations&startwith=%20&agencyId=694] {{webarchive|url=https://web.archive.org/web/20110706212437/http://www.volunteerottawa.ca/vo-clean/index.php?%2Feng%2Fi_want_to_volunteer%2Fcommunity_organizations&startwith=&agencyId=694 |date=2011-07-06 }}</ref>。另外两家机构位于多伦多和温哥华<ref>{{cite web |url=http://www.ottawacharities.com/ottawa-rape-crisis.php |title=存档副本 |accessdate=2011-03-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20101128015518/http://ottawacharities.com/ottawa-rape-crisis.php |archivedate=2010-11-28 }}</ref>。
