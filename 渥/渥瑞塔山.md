'''渥瑞塔山'''（Monte Verità，字面意为真理之山）是[[瑞士|瑞士]][[提契诺州|提契诺州]][[阿斯科纳|阿斯科纳]]的一个山丘，高{{convert|350|m|ft|sp=us|disp=or}}）。自20世纪初以来，曾有许多不同的乌托邦和文化的活动或社团立足于此。在生活革命运动（Lebensreform）期间<ref>Landmann (1979), p. 7</ref><ref name="DD">{{cite web|last1=Dailey|first1=Dan|title=Wandervogel - Frequently Asked Questions|url=http://www.wandervogel.com/wandervogel_faq.html|website=www.wandervogel.com|accessdate=2016-04-19}}</ref>，此处开始成为热门目的地。

==历史==
1900年，一个[[安特卫普|安特卫普]]商人的儿子亨利·奥伊顿克芬和他的伴侣艾达·霍夫曼买了阿斯科纳的一座山，建立了渥瑞塔山素食者合作聚落。聚落最初按照原始社会主义的原则建立，但以后拥护素食主义，并且开办渥瑞塔山疗养院，一处日光浴设施<ref>Landmann (1979), p. 13-20</ref>。

居民们"憎恶私有财产，实行严格的道德规范、素食和裸体，并拒绝婚姻和服装<ref name = "segall">{{cite web |url=http://www.segalselfbuild.co.uk/news/waltersegalbycol.html |title=WALTER SEGAL - Community Architect |author=Colin Ward |publisher=Diggers and Dreamers: A Directory of Alternative Living |accessdate=2008-09-18}}</ref>。

[[无政府主义|无政府主义]]医生Raphael Friedeberg于1904年迁往阿斯科纳，并吸引许多其他无政府主义者到该地区。被这座山吸引的艺术家和其他名人包括[[赫尔曼·黑塞|赫尔曼·黑塞]],<ref>{{cite web | url=http://www.monteverita.org/en/43/monte-verita.aspx | title=Monte Verità | publisher=Monte Verità.org | accessdate=2014-02-26}}</ref> [[卡尔·荣格|卡尔·荣格]]、[[埃里希·玛利亚·雷马克|埃里希·玛利亚·雷马克]]、[[斯特凡·乔治|斯特凡·乔治]]、[[艾莎道拉·邓肯|艾莎道拉·邓肯]]、[[保罗·克利|保罗·克利]]、[[鲁道夫·斯坦纳|鲁道夫·斯坦纳]]、[[恩斯特·托勒尔|恩斯特·托勒尔]]、[[亨利·范·德费尔德|亨利·范·德费尔德]]、[[埃里希·米萨姆|埃里希·米萨姆]]等人<ref>Landmann (1979), p. 59-60</ref>。

[[File:Hotel_monte_verità.jpg|thumb]]
从1913年到1918年，鲁道夫·拉班在渥瑞塔山开办"艺术学校"。1917年，东方圣殿骑士团（Ordo Templi Orientis）团长西奥多·罗伊斯（Theodor Reuss）召开会议，包含许多主题，包括没有民族主义的社会、妇女权利, 神秘共济会，以及作为艺术、仪式和宗教的舞蹈。<ref>Landmann (1979), p. 144-146</ref> 

从1923年到1926年，渥瑞塔山开设酒店。1926年由男爵Eduard von der Heydt获得<ref name = "heydt">{{cite web |url=http://www.ymago.net/photo/monteverita/heydt.html |title=Eduart von der Heydt |accessdate=2008-09-18}}</ref>。第二年，建造一个新的现代主义风格的酒店。1964年Heydt 逝世，此处遂成为[[提契诺州|提契诺州]]的财产。 

==现状==
渥瑞塔山目前拥有[[苏黎世联邦理工学院|苏黎世联邦理工学院]]会议中心''Centro Stefano Franscini''，以及一个包括三座建筑的博物馆: Casa Anatta，一个平顶的木建筑，用作素食聚落的总部，现在展出聚落的历史；Casa Selma，一个小型建筑，曾为日光浴疗养院；和全景画所在的建筑。这座小山也是一个茶园和日本茶馆的所在地。
