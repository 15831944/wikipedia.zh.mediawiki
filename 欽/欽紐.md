'''欽紐'''（{{my|m=ခင်ညွန်|y=hkang nywan.}}，{{bd|1939年|10月11日|||K}}），[[緬甸|緬甸]]軍政府前領導人物，前[[緬甸總理|緬甸總理]]及上將，擁有華人血統<ref>{{cite web |url=http://www.saag.org/papers12/paper1161.html |title=存档副本 |accessdate=2006-05-22 |0=2005-12-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20070927200005/http://www.saag.org/papers12/paper1161.html |archivedate=2007-09-27 }}</ref>，於2003年8月被委派出任總理一職，至2004年10月他被迫以「健康問題」為理由辭職，由[[梭溫|梭溫]]接替，其後遭到拘禁。

== 總理 ==
在任總理期間，在處理反對派領袖[[昂山素季|昂山素季]]的問題上，欽紐顯得比較溫和，與昂山素季會面後主張釋放她，與強硬派、軍政府最高領袖[[丹瑞|丹瑞]]的意見不合，促使他被罷黜，另任命梭溫為總理。

2004年年底，欽紐遭親丹瑞的軍方高層[[軟禁|軟禁]]。2005年7月，受到包括贪污、受贿在内的8项指控，欽紐被一个特别法庭判处可缓期执行的44年徒刑，从此开始被[[软禁|软禁]]在家。

== 被釋放 ==
2012年1月13日，新上台的缅甸文人政府释放了651名被判刑人员，其中就包括钦纽<ref>{{cite web|url=http://news.163.com/12/0113/16/7NLOS06B00014JB5.html?from=index | language= | title=缅甸大赦 前总理钦纽获释 |publisher=新华网 |date=2012-01-13 |accessdate= }}</ref>。
