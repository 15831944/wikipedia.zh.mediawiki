'''姬豬'''（[[学名|学名]]：''{{lang|la|Porcula salvania}}''），或稱侏儒豬、迷你豬<ref>{{cite web |title=侏儒豬 |url=http://terms.naer.edu.tw/detail/1077177/ |website=國家教育研究院 雙語詞彙、學術名詞暨辭書資訊網 |accessdate=2018-06-19}}</ref><ref>{{cite web |title=華盛頓公約附錄一、二、三列管之動植物物種清單說明 |url=https://www.trade.gov.tw/App_Ashx/File.ashx?FilePath=../Files/PageFile/241_76962/華盛頓公約附錄一二三清單(old+version).pdf |website=經濟部國際貿易局 |accessdate=2018-06-19}}</ref><ref>{{cite web |title=濒危野生动植物种国际贸易公约附录I、附录II和附录III(2017年8月) |url=http://www.cites.org.cn/uploadfile/article/uploadfile/201804/20180411050030363.pdf |website=中华人民共和国濒危物种科学委员会}}</ref>，是一種在[[印度次大陸|印度次大陸]]生活的小型豬，是[[猪科|猪科]]动物中最小的一种，曾经遍布[[印度|印度]]、[[尼泊尔|尼泊尔]]和[[不丹|不丹]]的高濕草原地帶。 然而人類的活動已經大大破壞了侏儒豬的自然棲息地，目前只分布在[[阿薩姆邦|阿薩姆邦]]，總野生種群估計少於150隻。

== 特征 ==
[[File:PorculaSalvaniaSmit.jpg|thumb]]
[[File:PorculaSalvaniaAdultWolf.jpg|thumb]]
姬豬约55－71cm长，20－30cm高，尾巴约2.5 cm，体重6.6－11.8公斤。皮肤棕黑色。小猪出生时是粉灰色，然后变为棕色底，黄色条纹。它们头尖长，额头和脖子后面有较长的毛。成年雄性上口部的虎牙伸出唇外。

野生姬豬挖出一个浅沟，垫上植物为窝，白天燥热时躲在窝里。主食为草树根，昆虫，小鼠类和小爬虫。

它们寿命约为8年，1－2岁时性成熟。台风季之前交配，孕期为100天，一胎3－6头小猪。

牠也是[[侏儒豬蝨|侏儒豬蝨]]的特定寄主。
