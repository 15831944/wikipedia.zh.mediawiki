《'''知識新天地'''》（{{lang-en|'''Knowledge for All'''}}）是[[香港公開大學|香港公開大學]]製作的電視節目，並由[[香港|香港]]的免費電視台播放<ref name="hkcna-20151014">{{cite news|title=無線明珠台將播放新一輯《進修新天地》|url=http://www.hkcna.hk/content/2015/1014/405769.shtml|publisher=香港新聞網|date=2015-10-14}}</ref>。[[電視|電視]]作為香港大眾吸收新知及增進學識的主要途徑，香港公開大學自1989年成立以來即已製作電視資訊節目，在1989年推出時名為《進修天地》，後來以《進修新天地》為名繼續推出<ref name="whb-20151015">{{cite news|title=「公大講堂」月底推 張信剛談「絲路」|url=http://paper.wenweipo.com/2015/10/15/ED1510150008.htm|publisher=文匯報|date=2015-10-15}}</ref>。節目的內容涵蓋[[語言|語言]]與[[文學|文學]]、[[歷史|歷史]]與[[文化|文化]]、[[家庭|家庭]]及[[兒童|兒童]]發展、[[社會科學|社會科學]]、[[醫療保健|醫療保健]]、[[教育|教育]]、[[金融|金融]]與[[投資|投資]]、[[市場學|市場學]]與[[管理學|管理學]]、[[公司治理|企業管治]]、[[科學|科學]]與[[科技|科技]]，以及[[電腦|電腦]]與[[資訊科技|資訊科技]]等多個範疇<ref name="ouhk-200111">{{cite journal|title=「進修新天地」10月起在明珠台播放|journal=公開大學通訊|date=2001年11月|volume=10|issue=4|url=http://www.ouhk.edu.hk/~openlink/current/0111/c_etv.htm}}</ref>。為推廣「知識全民共享」的理念，節目於2018年9月2日起更名為《'''知識新天地'''》（{{lang-en|'''Knowledge for All'''}}）<ref name="ouhk-20180823">{{cite news|title= 香港公開大學與全港市民共享知識 明珠台九月二日起播放全新電視節目《知識新天地》 |url= http://www.ouhk.edu.hk/wcsprd/Satellite?CCNAME=CCNEWS&PAGE=1&STU_ANNOUNCE=false&YEAR=2018&c=C_PAU&cid=1385191263961&d=Touch&dis=11&lang=chi&pagename=OUHK%2FtcPortalPage2014 |publisher=香港公開大學|date=2018-08-23}}</ref>。

== 節目環節 ==
「公大講堂」名人演講系列是《知識新天地》從2015年10月起加入的新環節，由公大邀請多位大師級學者專家、知名人士到公開大學演講廳就不同話題主講<ref name="whb-20151015a">{{cite news|title=「公大講堂」月底推 張信剛談「絲路」|url=http://paper.wenweipo.com/2015/10/15/ED1510150007.htm|publisher=文匯報|date=2015-10-15}}</ref>，主題包括[[香港歷史|香港歷史]]、[[中國文化|中國文化]]、新聞時事和健康保健等<ref name="whb-20161012">{{cite news|title=「公大講堂」論過客看港史|url=http://paper.wenweipo.com/2016/10/12/ED1610120008.htm|publisher=文匯報|date=2016-10-12}}</ref>。 [[諾貝爾文學獎|諾貝爾文學獎]]得主[[莫言|莫言]]到訪公開大學出席公大文學論壇的演說<ref name="st-20121022">{{cite news|title=莫言談文學 公大特播論壇片段|url=http://std.stheadline.com/daily/news-content.php?id=392925&target=2|publisher=星島日報|date=2012-10-22}}</ref>，也在《知識新天地》中播放。

== 傳媒合作 ==
《知識新天地》雖然是香港公開大學製作的電視節目，但利用大氣電波廣播電視節目就一直需要與本地免費電視台進行合作，[[無綫電視|無綫電視]]是公開大學播放《知識新天地》的最主要合作機構。2015年10月14日，無綫電視助理總經理杜之克，香港小姐[[麥明詩|麥明詩]]及[[郭嘉文|郭嘉文]]<ref>{{cite journal|title=懷念讀書的日子 麥明詩望繼續進修|journal=東周網|url=http://eastweek.my-magazine.me/main/46718?keyword=%E8%A1%A8}}</ref>、演員[[翟威廉|翟威廉]]與[[何基佑|何基佑]]，蒞臨公開大學出席新一輯《進修新天地》的啟動儀式<ref name="hkcna-20151014" />。

== 播放時間 ==
由2015年10月起<ref name="sengpou-20151015">{{cite news|title=明珠台逢周四播《進修新天地》|url=http://www.sengpou.com/news_detail.php?news_id=1436&element_index=8|publisher=星報|date=2015-10-15}}</ref>，新一輯《進修新天地》每逢星期日於[[無綫電視明珠台|無綫電視明珠台]]播放<ref name="ouhk-20151014">{{cite news|title=香港公開大學與全港市民共享知識 明珠台十月起共拓進修新天地|url=http://www.ouhk.edu.hk/wcsprd/Satellite?pagename=OUHK/tcPortalPage2014&CCNAME=CCNEWS&c=C_PAU&cid=1385178848765&dis=11&YEAR=2015&PAGE=1&STU_ANNOUNCE=false&lang=chi|publisher=香港公開大學|date=2015-10-14}}</ref>。播放時間為早上9時至下午1時，每周四小時的節目，一般分為4節至6節涵蓋不同的課題。另外，[[樂視網|樂視網]]設有專屬頻道（Ch.620）提供節目重溫<ref name="ouhk-otv-1">{{cite web|title=進修新天地電視節目|url=http://www.ouhk.edu.hk/wcsprd/Satellite?pagename=OUHK/tcSingPage&c=C_PAU&cid=191182021200&lang=chi|publisher=香港公開大學|accessdate=2017-10-12}}</ref>。
