'''铑'''（舊譯'''錴'''）符号'''Rh'''，[[元素|元素]]之一，[[原子序|原子序]]45，只有一个穩定的同位素<sup>103</sup>Rh。由英國化學家[[威廉·海德·伍拉斯顿|威廉·海德·伍拉斯顿]]（William Hyde Wollaston）于1803年发现，并以其一种玫瑰色的[[氯|氯]]化合物命名，可由该化合物于[[王水|王水]]反应而得。英文「Rhodium」的希腊语意为"玫瑰"。

铑是坚硬的银白色[[过渡金属|过渡金属]]，耐腐蚀，可在[[铂|铂]]矿发现，十分稀有，亦在一些铂合金中用作[[催化剂|催化剂]]。

== 形状性质 ==
质地坚硬，不受[[酸|酸]]的侵蚀。不溶于硝酸，微溶于王水。反射率高。常见价态+3。

== 发现 ==
1803年，继W.H. Wollaston 发现钯之后，他从产自南美洲的粗铂矿提取出铑。他使用王水，NaOH，氯化铵等物质，将钯转化为玫瑰红色沉淀。

== 来源 ==
由[[镍|镍]]生产的副产品获得。

銠為[[鉑|鉑]]礦中的的稀少成分，產量純粹取決有多少鉑礦開採出來，開採的鉑礦越多，從中得到的銠雜質也越多。若市場需求大於供給銠的價格就會高漲，因只為增加銠供給而採更多的鉑礦不符合經濟效益。<ref>看得到的化學，Theodore Gray著，大是文化 ISBN 978-986652667-1</ref>

== 用途 ==
用作高科技仪器的[[防磨涂料|防磨涂料]]和[[催化剂|催化剂]]、[[铑铂合金|铑铂合金]]用于生产[[热电偶|热电偶]]。也用于镀在车前灯反射镜、[[电话|电话]][[中继器|中继器]]、钢笔尖、[[内燃機|内燃機]][[車輛|車輛]]的[[觸媒轉換器|觸媒轉換器]]及[[白金_(合金)|白金]]首饰等。[[威爾金森催化劑|威爾金森催化劑]]是一種铑的[[配合物|配合物]]，可用于烯烃的氢化还原。

在[[核反应|核反应]]中用含铑的[[探测仪|探测仪]]测量[[中子流|中子流]]水平。

还可用于[[首饰|首饰]]和装饰品以及高級[[音響|音響]]用[[介面_(資訊科技)#硬體介面|端口]]的鍍層，一般會先鍍一層[[銀|銀]]再鍍一層铑，因铑的惰性能達到防止[[氧化|氧化]]、抗磨損的效果。
