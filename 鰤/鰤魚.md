'''鰤魚'''或'''五條鰤'''（[[学名|学名]]：''{{lang|la|Seriola quinqueradiata}}''），又名'''青甘鱼'''、'''平安鱼'''、'''油甘魚'''，是鱸形目鰺科的海水魚。在新加坡從英語「Amberjack」直譯作「'''琥珀魚'''」<ref>{{cite episode |date=2013-04-30 |url=http://mytv.tvb.com/foodandtravel/japanflavour/144433 |episode=《[[J_Food好吃|J Food好吃]]》 |channel=[[J2|J2]] }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是一種分布于[[台湾|台湾]]、[[日本|日本]]附近海域的魚。鰤魚在日本有100種以上的名字<ref>{{cite web |url=http://www.cn.asahibeer.co.jp/fun/food/ueno03.html |title=存档副本 |accessdate=2007-06-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20070927224719/http://www.cn.asahibeer.co.jp/fun/food/ueno03.html |archivedate=2007-09-27 }}</ref>。在[[台灣|台灣]]則是被稱為'''青甘鰺'''、'''青魽'''、'''青甘'''，是日本料理店常見的生食材料。鰤魚是屬於溫水性一種<ref>{{cite web |url=http://www.yzagri.gov.cn/infodetail.php?ej=368c23c5a2c95efc0ff7fae055a3de16&infoid=1122022114 |title=存档副本 |accessdate=2007-06-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20070929082152/http://www.yzagri.gov.cn/infodetail.php?ej=368c23c5a2c95efc0ff7fae055a3de16&infoid=1122022114 |archivedate=2007-09-29 }}</ref>。

==簡介==

===分佈地點與[[產卵期|產卵期]]===
* [[臺灣|臺灣]]：[[臺灣島|臺灣本島的東北角與]]南方[[海域|海域]]。
* [[日本|日本]]：[[日本海|日本海]]、[[近畿地方|關西]]地區。
** 其產卵期視乎其生活區域，一般為每年的二月至五月。

===生存環境===
* '''鰤魚'''多生活於高鹽度的水域，其身體對鹽度有較強的適應力（最宜範圍：20‰－25‰）。
* '''鰤魚'''多生活於溫水溫的水域，（最宜範圍：24℃－26℃）〔[[養殖場|養殖場]]溫度範圍彈性更大〕。

===食物===
隨著[[生長|生長]][[長度|長度]]漸漸長大，所吃的食物會漸漸不同：4[[釐米|釐米]]以下：是吃[[甲殼動物|甲殼動物]]；其後陸續可以吃[[小魚|小魚]]、[[鯖魚|鯖魚]]、[[烏賊|烏賊]]、[[沙丁魚|沙丁魚]]等魚類為食。

==養殖歷史與方式==

===養殖方式===
[[海水|海水]][[網箱|網箱]]養殖（養殖1年到2年）。

===養殖歷史===
[[日本|日本]]是最早開始養鰤魚的國家（ 大約四百年前，'''七尾'''及'''宇出津漁港'''就開始了<ref>{{cite web |url=http://www.hot-ishikawa.jp/f-lang/chn-trad/t_mikaku/mi_noto/minoto.htm |title=存档副本 |accessdate=2007-06-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20070624032942/http://www.hot-ishikawa.jp/f-lang/chn-trad/t_mikaku/mi_noto/minoto.htm |archivedate=2007-06-24 }}</ref>）。

近幾年[[中華人民共和國|中國]]亦有進行鰤魚養殖。

青甘鰺養殖企業已經在21世紀大為改觀 ，通過提高技術系泊筏移動到的外海及製粒飼料進行水產養殖。

這種先進技術的，可以在兩年內育肥“青甘鰺”，也出現了開展雄心勃勃的嘗試出貨為“青甘鰺”的公司。
這種漁民的努力結出果實是最近幾年發生逆轉的現象，是水產養殖產品可能比天然產品還良好。但對於養殖青甘，消費者仍然存有以前寄生蟲及抗生素殘留的印象，但美味的養殖青甘鰺已成為野生產品一個挑戰。

===主要疾病===
[[虹彩病毒感染症|虹彩病毒感染症]]，[[結核病|假結核病]]、[[諾卡氏菌病|諾卡氏菌病]]、[[溶血性链球菌|溶血性链球菌]]等<ref>{{cite web |url=http://www.seasun.com.cn/upload/japan16tb-2.doc |title=存档副本 |accessdate=2007-06-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20070930055122/http://www.seasun.com.cn/upload/japan16tb-2.doc |archivedate=2007-09-30 }}</ref><ref>https://web.archive.org/web/20070930055122/http://www.seasun.com.cn/upload/japan16tb-2.doc</ref>。

==健康方面==
                                                         
鰤魚含有豐富的[[二十二碳六烯酸|DHA]]，是對人體很健康的食物。
