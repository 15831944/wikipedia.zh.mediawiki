'''血汗攻闖'''，[[香港|香港]]獨立[[樂隊|樂隊]]，成立於2014年。

== 樂團簡介 ==
根據樂隊[[社交網站|社交網站]]，血汗攻闖此樂隊原名為「[[血汗工廠|血汗工廠]]」，成立於2014年年初，由[[社運|社運]]人士所組成，其中成員三度更迭，後於2014年10月26日香港社運電影節固定了四人陣容，及更改名字為「血汗攻闖」，演奏類別被歸納為[[饒舌金屬|饒舌金屬]]及[[新金屬|新金屬]]，樂隊歌曲題材主要集中在評論[[社會現象|社會現象]]及[[政治理念|政治理念]]倡議為主。

2015年，該樂隊參與[[嶺南大學_(香港)|嶺南大學]]學生舉辦的[[音樂會|音樂會]]，因其中一首曲目《Fuck The Police》內容牽涉辱罵[[警察|警察]]和大量不雅用語，引起風波，上載相關片段的媒體網站亦有大量兩極性的評語充斥，而因樂隊中有成員為註冊社會工作者，亦引來大量反對聲音認為樂隊及個別成員「教壞細路」、指控樂隊成員為「廢偽社工」<ref>{{cite news |url = https://thestandnews.com/politics/%E4%B8%8D%E6%BB%BF%E6%A0%A1%E5%9C%92%E5%94%B1%E7%B2%97%E5%8F%A3%E6%AD%8C%E7%BD%B5%E8%AD%A6-%E5%BB%BA%E5%88%B6%E7%B5%84%E7%B9%94%E4%BF%83%E8%A7%A3%E6%95%A3%E5%B6%BA%E5%A4%A7%E5%AD%B8%E7%94%9F%E6%9C%83/ |title = 【粗口歌罵警】何君堯：嶺大應解散學生會　否則政府須整頓管理層 |publisher = 立場新聞 |pages = https://thestandnews.com/ |date = 2015-04-27  |accessdate = 2/10/2017}}</ref>。樂隊事後於各網上媒體平台發表聲明<ref>{{cite news |url =http://www.inmediahk.net/node/1034039 |title = 血汗攻闖就「黑暗時代抗爭音樂會」表演歌曲所引發爭議之聲明 |publisher = 香港獨立媒體 |pages = http://www.inmediahk.net/ |date = 2015-05-06  |accessdate = 2/10/2017}}</ref>，僅向受影響的主辦方及學生致歉，明言不會就歌曲帶辱罵成份向警務人員道歉及表示「企硬」。後來風波不了了之。

2016年，樂隊發佈正式版的《Fuck The Police》，由香港音樂人[[周博賢|周博賢]]混音。歌曲正式發佈後卻與上次大為不同：沒有太多反對和抨擊聲音在歌曲頁面。同年，亞洲音樂雜誌《Unite Asia》將樂隊音樂類型定義為饒舌金屬<ref>{{cite news |url = http://uniteasia.org/6265-2/ |title = Hong Kong rap metal band 血汗攻闖 Cantonese version of a fiery Fuck The Police track |publisher = Unite Asia |pages = https://archive.is/mHz6j |date = January 20, 2016 |accessdate = 2/10/2017 |deadurl = yes |archiveurl = https://archive.is/20160507023604/http://uniteasia.org/6265-2/ |archivedate = 2016年5月7日 }}</ref>。
