《'''举起手来'''》（{{lang-en|'''Hands Up 2'''}}）2010年9月9日上映于[[中国大陆|中国大陆]]的战争喜剧片。<ref>{{cite web |url=http://news.163.com/10/0904/04/6FN86E8000014AED.html |title=《举起手来2》是闹剧？ |accessdate= 2010-09-04 |publisher=网易 |work= |language =zh-cn}}</ref>

== 剧情 ==
1945年，[[大日本帝国|大日本帝国]]面临全面溃败和战略收缩，他们从[[中华民国|中华民国]]掠夺了大量的文物，准备用一艘名叫阿多丸的舰船带回到日本本土，为了挽救文物，随后厨师郭大叔和女兵小刘与其展开了殊死搏斗。<ref name=" lion" >{{cite web |url=http://news.163.com/10/0908/10/6G289D9P00014AED.html |title=《举起手来2》明日上映 |accessdate=2010-09-08 |publisher= 网易|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/c/2010-09-03/13323075115.shtml |title=《举起手来2》将映 “举起手来”见潘长江郭达 |accessdate=2010年9月3日 |publisher= 新浪网|work= |language =zh-cn}}</ref>

== 花絮 ==
该片为纪念[[抗日战争|抗日战争]]胜利65周年的宣传片。<ref name=" lion" />导演冯小宁将日本侵略者设计成为“一群蠢猪”，以戏谑的电影手法，在[[中国共产党|中国共产党]]、[[中国国民党|中国国民党]]、[[美国|美国]][[飞虎队|飞虎队]]的合力之下，步步败退，颇具喜剧气氛。<ref>{{cite web |url=http://yule.sohu.com/20100920/n275143468.shtml |title=《举起手来2之追击阿多丸》：冯小宁存在的价值 |accessdate= 2010年9月20日 |publisher=搜狐网 |work= |language =zh-cn}}</ref>电影发行的最低票价为5人民币。<ref>{{cite web |url= http://www.chinanews.com/yl/2010/09-08/2518473.shtml|title= 郭达潘长江西安宣传《举起手来2》 最低票价5元|accessdate=2010年9月8日 |publisher=中国新闻网 |work= |language =zh-cn}}</ref>

该片的片尾还提及了中日之间的[[釣魚臺列嶼主權問題|釣魚臺列嶼主權問題]]并明确表明[[釣魚臺列嶼|釣魚臺列嶼]]属于中国：当一心想要返回日本老家的中村下等兵坐着美军潜艇发射的鱼雷在一个小岛上登陆时，在那裡遇到了一位渔民（由导演[[冯小宁|冯小宁]]饰演），他先是用[[日语|日语]]、之后是[[汉语|汉语]]问道：“（大叔），这里是[[日本|日本]]吗？（日语：'''{{lang|ja|おじさん、ここは日本ですか？}}'''）”（括号部分仅见于日语），然而那位渔民是中国人，用汉语回答道：“日本？你走错道儿啦，这叫[[釣魚臺|釣魚臺]]，是中国！”，说完，渔民拿起鱼叉，对着日本兵怒吼：“举起手来！！！”<ref>[http://tv.cctv.com/2013/08/26/VIDE1377488013948304.shtml 《经典电影》 20130825 《举起手来——追击阿多丸》（视频从1:16:00至1:16:51）]，央视网，2013-08-26 02:06</ref>
