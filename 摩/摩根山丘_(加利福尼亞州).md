'''摩根山丘'''（'''Morgan Hill'''），又称'''摩根希尔'''，是[[美国|美国]][[加利福尼亚州|加利福尼亚州]][[圣克拉拉县|圣克拉拉县]]下属的一个市。设市于1906年11月10日。起初主要为附近[[农民|农民]]、农场主和果园农夫的居住地，后逐渐扩展并演变为[[硅谷|硅谷]]的一个高科技工业卫星城。根据[[美国2010年人口普查|美国2010年人口普查]]，该市有人口37,882人。
== 地势 ==
[[File:Morgan_Hill,_CA.jpg|thumb]]
摩根山丘位于[[旧金山湾区|旧金山湾区]]东南部，[[圣荷西_(加利福尼亚州)|圣荷西]]市中心东南方大约39[[公里|公里]]处，[[吉尔罗伊_(加利福尼亚州)|吉尔罗伊]]市西北方大约21公里，往西距离[[太平洋|太平洋]]海岸大约24公里，西南部是[[圣塔克鲁兹_(加利福尼亚州)|圣塔克鲁兹]]市。市区坐落在[[圣克拉拉山谷|圣克拉拉山谷]]南部，[[海拔|海拔]]大约107[[米|米]]。市区内没有自然形成的湖泊和河流。该市位于[[地震带|地震带]]附近，是地震多发区。

== 气候 ==
该市气候受到太平洋的影响，属于[[地中海气候|地中海气候]]。全年气温区间在0.9℃（33.6°F）至32.3℃（90.2°F）之间。全年[[降水量|降水量]]约为480[[毫米|毫米]]。夏季气候极为干燥，冬天雨量较多，但极少有降雪。在夏季，[[雾|雾]]较为常见。晚上10时左右会有来自海湾的雾飘至该市上空，第二天上午10时左右逐渐散去。在冬天，阵雨发生的几率较大。该地区的地形不适合[[龙卷风|龙卷风]]产生，也较少出现雷暴或风暴天气。这里的地中海气候使得一些热带植物，如[[棕榈树|棕榈树]]，适合在这里生长。
== 人口 ==
=== 2000年 ===
根据[[2000年美国人口普查|2000年美国人口普查]]，2000年时，该市有10,846户，8,633个家庭。种族组成为：72.40%的[[白人|白人]]，6.02%[[亚裔|亚裔]]，1.71%[[黑人|非洲裔]]，1.08%[[原住民|原住民]]，0.23%太平洋岛屿居民，13.43%属于其它种族。5.13%同时来自两个或更多的种族。

2000年该市有44%的家庭拥有18岁以下的[[未成年人|未成年人]]。63.2%为有婚姻的家庭。

=== 2010年 ===
根据[[2010年美国人口普查|2010年美国人口普查]]，2010年时，该市有12,326户，9,696个家庭。平均每个家庭有3.39人。种族组成为：65.2%的[[白人|白人]]，10.2%[[亚裔|亚裔]]，2.0%[[黑人|非洲裔]]，0.9%[[原住民|原住民]]，0.3%太平洋岛屿居民，15.3%属于其它种族。6.2%同时来自两个或更多的种族。

2010年该市有44.9%的家庭拥有18岁以下的[[未成年人|未成年人]]。61.5%为有婚姻的家庭。

== 设施 ==
[[太平洋瓦电公司|太平洋瓦电公司]]为该市提供燃气和电力。

该市有一所公立高中和[[社区大学|社区大学]]，同时也有一些学生前往圣荷西市内的私立学校上学，如[[山谷基督教学校|山谷基督教学校]]。
