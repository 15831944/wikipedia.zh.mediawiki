{{摩尔多瓦历史}}

'''摩尔多瓦历史'''最早可追溯至1350年代。[[摩尔多瓦人|摩尔多瓦人]]与[[罗马尼亚人|罗马尼亚人]]同宗同文，都是[[达契亚|达契亚]]的子孙。13至14世纪，[[蒙古|蒙古]][[鞑靼人|鞑靼人]]侵入该地，后[[沙皇俄国|沙皇俄国]]将摩部分领土[[比薩拉比亞|比薩拉比亞]]划入俄罗斯版图。1859年1月，摩尔多瓦和[[瓦拉几亚|瓦拉几亚]]合并，成为新成立的[[罗马尼亚联合公国|罗马尼亚联合公国]]的一部分。20世纪初期受[[苏维埃俄国|苏维埃俄国]]控制并最终加入[[苏联|苏联]]，成为其中的一个[[加盟共和国|加盟共和国]]。1990年代独立至今，为欧洲内较为落后的国家。

== 史前 ==

== 古典和中世纪前期 ==
在有记载的古典时代，摩尔多瓦由多个部落所占据。其中主要是达西亚人，其他时期有[[巴斯塔奈人|巴斯塔奈人]]、[[萨尔马提亚人|萨尔马提亚人]]和[[斯基泰人|斯基泰人]]。在1至7世纪，南部时而由[[罗马帝国|罗马帝国]]以及之后的[[拜占庭帝国|拜占庭帝国]]所控制。

== 摩尔多瓦公国 ==
在中世纪，[[摩尔达维亚公国|摩尔达维亚公国]]于1359年建立。

== 俄罗斯帝国一部分 ==
除了[[德涅斯特河左岸|德涅斯特河左岸]]之外，现今摩尔多瓦共和国为历史上的[[比薩拉比亞|比薩拉比亞]]地区大部分。

作为[[第七次俄土战争|第七次俄土战争]]的结果，根据1812年5月28日，[[奥斯曼帝国|奥斯曼帝国]]和[[俄罗斯帝国|俄罗斯帝国]]的[[布加勒斯特条约_(1812年)|布加勒斯特条约]]，后者合并了摩尔多瓦公国的东部。<ref name="Mitrasca2002">{{Cite book|title=Moldova: a Romanian province under Russian rule : diplomatic history from the archives of the great powers |last=Mitrasca |first=Marcel  |authorlink= |year=2002 |publisher=Algora Publishing |location= |isbn=1-892941-86-4 |page=6 |pages= |url=https://books.google.com/books?id=mZogbSmBR-4C&pg=PA6}}</ref>这块地区称为比萨拉比亚。<ref>Mitrasca, pp. 10–11.</ref>

== 摩尔多瓦民主共和国 ==
在[[1905年俄国革命|1905年俄国革命]]之后，一个罗马尼亚民族主义运动在比萨拉比亚发展。它在1906年至1907年遇到挫折，但是在1917年，运动开始重新出现并且更为强烈。<ref>Mitrasca, pp. 28–29</ref>

== 大罗马尼亚一部分 ==
在1918年之后，比萨拉比亚处于罗马尼亚的管辖之下。这个是根据1920年[[巴黎协定|巴黎协定]]。新生的[[苏维埃俄国|苏维埃俄国]]并不承认[[罗马尼亚|罗马尼亚]]对于比萨拉比亚的统治。<ref>Wayne S Vucinich, ''Bessarabia'' In: ''[[Collier's_Encyclopedia|Collier's Encyclopedia]]'' (Crowell Collier and MacMillan Inc., 1967) vol. 4, p. 103</ref>[[比萨拉比亚|比萨拉比亚]][[苏维埃社会主义共和国|苏维埃社会主义共和国]]于1919年5月5日在[[敖德薩|敖德薩]]宣告成立。这是一个临时的工农[[流亡政府|流亡政府]]。它于1919年5月11日在[[蒂拉斯波尔|蒂拉斯波尔]]建立成为[[俄罗斯苏维埃联邦社会主义共和国|俄罗斯苏维埃联邦社会主义共和国]]一部分。

== 第二次世界大战和苏联 ==

== 1991年独立 ==
随着[[戈尔巴乔夫|戈尔巴乔夫]]的[[新思维|新思维]]改革和[[开放政策|开放政策]]气候，[[民族主义|民族主义]]气候在[[摩尔多瓦|摩尔多瓦]]升级。1989年，摩尔多瓦人民前线作为一个独立的文化和政治组织成立并得到官方承认。

== 独立后 ==

* 1990年9月2日德涅斯特河沿岸宣布独立；随后成立[[德涅斯特河沿岸摩尔达维亚共和国|德涅斯特河沿岸摩尔达维亚共和国]]。
* 1991年5月23日改称为“摩尔多瓦共和国”。
* 1991年8月27日摩尔多瓦共和国宣布独立。
* 1992年，爆发[[德涅斯特河沿岸战争|德涅斯特河沿岸战争]]，最终[[摩尔多瓦共和国|摩尔多瓦共和国]]战败，[[德涅斯特河沿岸|德涅斯特河沿岸]]维持独立。
* 1994年3月，[[公民投票|公民投票]]以压倒性的票数支持维持独立主权国家的地位，反对与[[罗马尼亚|罗马尼亚]]再统一。
* 2009年7月，[[摩尔多瓦共和国共产党人党|摩尔多瓦共和国共产党人党]]选举失利。
* 2016年2月17日，[[罗马尼亚|罗马尼亚]]支持摩尔多瓦加入[[欧盟|欧盟]]进程。
