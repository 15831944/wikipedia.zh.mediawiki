'''摩洛哥历史'''，[[卡普萨文化|卡普萨文化]]（Capsian culture）在西元前8000年将[[摩洛哥|摩洛哥]]带入到[[新石器时代|新石器时代]]，那时[[马格里布|马格里布]]不像今天这样贫瘠。[[柏柏尔|柏柏尔]]语言也许是和农耕差不多同时来到，被本地居民和带来这种语言的外来者所共同继承。现代DNA分析业已证明，现代摩洛哥人的基因池由多个人种而来，不仅有主要的种族如[[柏柏尔人|柏柏尔人]]和[[阿拉伯|阿拉伯]]人，也有[[腓尼基|腓尼基]]人、[[西班牙|西班牙]][[犹太人|犹太人]]和[[撒哈拉|撒哈拉]]以南地区（sub-Saharan）的[[非洲|非洲]]人。

== 史前摩洛哥 ==
考古发掘证实了祖先智人以及早期人类存在于今摩洛哥。

== 早期历史 ==
多个世纪以来，腓尼基人在摩洛哥沿岸定居。[[腓尼基|腓尼基]]人的到来，预示了摩洛哥北部许多个世纪被外来政权统治的历史，这个战略地带作为[[毛里塔尼亚-廷吉他纳|毛里塔尼亚-廷吉他纳]]。

在约公元前3世纪，毛里塔尼亚王国曾是地中海北非沿岸一个独立的王国。后纳入[[罗马帝国|罗马帝国]]的一部分。

== 罗马和罗马属地的摩洛哥 ==

5世纪时，今摩洛哥落入[[汪达尔人|汪达尔人]]之手，然后是[[西哥特|西哥特]]人，接續又是來自[[拜占庭|拜占庭]]的希腊人迅速占领，尽管如此在这时期占现代摩洛哥大部分的高山地区仍然未被占领，还保留在[[柏柏尔|柏柏尔]]人手中。

== 早期伊斯兰的摩洛哥 ==
阿拉伯人在7世纪时征服了摩洛哥，並带進了他们的文明和[[伊斯兰教|伊斯兰教]]，而多数柏柏尔人皈依了，成立了Nekor王国。并很快脱离了远在[[巴格达|巴格达]]的[[阿拔斯王朝|阿拔斯王朝]][[哈里发|哈里发]]的控制，在Idris ibn Salih领导下建立了[[伊德里斯王朝|伊德里斯王朝]]，摩洛哥成为学术中心和重要的政权所在地。

=== 伊德里斯王朝（788年－974年）===
[[伊德里斯王朝|伊德里斯王朝]]是一个以摩洛哥为中心的穆斯林王朝。<ref>{{Citation |last=Hodgson |first=Marshall |year=1961 |title=Venture of Islam |publication-place=Chicago |publisher=University of Chicago Press |pages=262}}</ref>

== 柏柏尔人王朝 ==
摩洛哥在一系列取代了阿拉伯[[伊德里斯王朝|伊德里斯王朝]]的柏柏尔王朝的统治下达到顶点，先是[[穆拉比德王朝|穆拉比德王朝]]，然后是[[穆瓦希德王朝|穆瓦希德王朝]]，並统治了西北非的大部分和西班牙的大部分地區，这一地区的一些小王国，先后被征服，然而在一系列长年的内战之后，帝国崩溃了。

=== 穆瓦希德王朝（1147年－1248年）===
[[穆瓦希德王朝|穆瓦希德王朝]]的统治者是摩洛哥人。

=== 马林王朝（1248年－1465年）===
[[马林王朝|马林王朝]]是由逊尼派穆斯林Zenata柏柏尔人建立。

=== 瓦塔斯王朝（1471年－1549年）===
[[瓦塔斯王朝|瓦塔斯王朝]]是一个统治摩洛哥的王朝。与马林王朝类似，他们来自于Zenata柏柏尔人。

1631年-1635年，[[默罕默德一世|默罕默德一世]]，Tafilalt 苏丹。1635年-1664年，[[默罕默德二世|默罕默德二世]]，Tafilalt [[苏丹|苏丹]]。

== 阿拉维王朝 ==
1659年Mohammed al-Hajj ibn Abu Bakr al-Dila'i征服萨阿德王朝。1664年10月22日，Ar-Rashid（1664年-1672年）在[[非斯|非斯]]自立为[[苏丹_(称谓)|苏丹]]，1668年12月7日征服并控制了[[马拉喀什|马拉喀什]]（Marrakesh）。

[[阿拉維王朝|阿拉維王朝]]最终取得了控制，摩洛哥面临西班牙和向西方席卷而来的[[奥斯曼帝国|奥斯曼帝国]]入侵，[[阿拉維王朝|阿拉維王朝]]仍成功的稳固了自己的地位，尽管王国比原来的国家面积要小，但仍是非常的富裕。

1684年，吞并[[丹吉尔|丹吉尔]]。

1777年，摩洛哥是第一个承认美国独立的国家，[[美-摩友好条约|美-摩友好条约]]是美国最早的稳固友好条约。由[[约翰·亚当斯|约翰·亚当斯]]和[[托马斯·杰斐逊|托马斯·杰斐逊]]签署，自1783年以来一直生效。丹吉尔的{{Link-en|丹吉爾美國公使館|American Legation, Tangier|美国公使馆}}（[[领事馆|领事馆]]）是美国政府有史以来最早的海外地产，那座建筑现在是丹吉尔美国公使馆博物馆。

1769年，从[[葡萄牙|葡萄牙]]手里征服[[杰迪代|杰迪代]]（El Jadida）。

1895年，从[[不列颠帝国|不列颠帝国]]手里买下[[尤比角|尤比角]]（Cape Juby）。

== 欧洲的影响 ==
15世纪[[葡萄牙|葡萄牙]]人成功控制[[大西洋|大西洋]]沿岸的努力，并没有影响到摩洛哥此[[地中海|地中海]]心脏地区。[[拿破仑战争|拿破仑战争]]后，[[埃及|埃及]]和[[北非|北非]][[马格里布|马格里布]]在本地长官控制下开始脱离[[君士坦丁堡|君士坦丁堡]]统治。[[欧洲|欧洲]]工业化后，马格里布有越来越大的殖民潜力，比未知的非洲其他地区更为富庶，并处在对[[地中海|地中海]]的出口产生战略影响的位置，摩洛哥变成欧洲列强眼中有重要性的国家。

[[法国|法国]]早在1903年就对摩洛哥表现强烈兴趣。在[[英国|英国]]支持下的法国對摩洛哥的影响，引起了[[德国|德国]]的不滿。1905年到1906年的[[第一次摩洛哥危机|第一次摩洛哥危机]]在[[阿尔赫西拉斯会议|阿尔赫西拉斯会议]]得到解决。法国的特殊地位得到正式承认，而摩洛哥的治安托管权交給给了法国和西班牙。

[[第二次摩洛哥危机|第二次摩洛哥危机]]由德国引起，加剧了欧洲列强间的紧张，但1912年3月30日签订的「[[非斯条约|非斯条约]]」，把摩洛哥变为法国的[[保护国|被保护国]]。同样，在[[非斯条约|非斯条约]]中，那一年的11月27日[[西班牙|西班牙]]得到了北部和南部地区（[[伊夫尼地区|伊夫尼地区]]）保护者的地位。

屬於[[分离主义|分离主义]]者的利夫共和国（Republic of the Rif）于1921年9月18日由利夫人宣布建国，1926年5月27日被西班牙和法国军队解散。

[[民族主义|民族主义]]的政党，随后在法国的保护国兴起，以[[二战|二战]]末英美-{zh-hans:发;zh-hant:發;}-表的[[大西洋宪章|大西洋宪章]]（其中讲到所有的民族有权选择他们政府的形式）等宣言作为摩洛哥独立的理据。

许多摩洛哥抵抗战士（Goumiere）在[[一战|一战]]和[[二战|二战]]中帮助了[[美国|美国]]。库特拉（Istiqlal）（独立党）在1944年发布的一份宣言，是人民最早的对独立的要求，独立党后来领导了大部分的民族主义运动。

1953年，法国驱逐了广受尊敬的苏丹[[穆罕默德五世_(摩洛哥)|穆罕默德五世]]（Sultan Mohammed V），扶植不受欢迎的[[穆罕默德·本·阿拉法|穆罕默德·本·阿拉法]]（Mohammed Ben Aarafa）取而代之，但他的统治被认为是非法的，于是激起对法国保护地位的抗争。1955年法国允许[[穆罕默德五世_(摩洛哥)|穆罕默德五世]]回国。在第二年，最终导致摩洛哥独立和谈的开始。

== 独立 ==
摩洛哥王国，在1956年3月2日从[[法国|法国]]控制下重获政治独立。同年7月4日法国正式放弃对摩洛哥的保护，通过1956年至1958年與西班牙谈判中取得的协议，摩洛哥收回了部分原先由西班牙控制的地区（雖然以军事手段取回其他西班牙占领区的努力并不成功）。

国际化的城市[[丹吉尔|丹吉尔]]，因1956年10月29日[[丹吉尔协议|丹吉尔协议]]的签署而重获名声。1961年3月3日[[哈桑二世_(摩洛哥)|哈桑二世]]成为摩洛哥国王，在他统治的時期是政治是动荡不安的，政府的鐵腕措施为这此时期赢得了「领导年代」的名声。

在邻国[[阿尔及利亚|阿尔及利亚]]於1962年自从法国独立后，[[廷杜夫省|廷杜夫省]]地区的边界冲突不断，并在1963年升级成为[[沙地战争|沙地战争]]（Sand war），摩洛哥为了建立[[大摩洛哥|大摩洛哥]]而入侵了这一地区，但战斗僵持数周，摩洛哥被迫撤军，边界没有任何更改，而边界问题仍然存在。而在边界得到了划定之後，摩洛哥再没有向阿尔及利亚提出过领土要求。

== 近代 ==
摩洛哥的历史和外交一个重要的主题是和[[西撒哈拉|西撒哈拉]]的斗争，摩洛哥自1975年末入侵和吞并了前[[西属撒哈拉|西属撒哈拉]]後，卻遭到[[西撒人阵|西撒人阵]]（Polisario Front）的军事抵抗，这是一个由[[阿尔及利亚|阿尔及利亚]]支持的[[撒哈拉人|撒哈拉人]]（Sahrawi）發起的民族主义运动。此地区最终的归属仍未解决，但目前摩洛哥对大部分地区有军事控制權，和[[阿拉伯撒哈拉民主共和国|阿拉伯撒哈拉民主共和国]]的停火协议自1991年起生效。

1990年代以来，政治的渐进改革导致了1997年的[[两院制|两院制]]立法体系建立，随着1999年国王[[哈桑二世_(摩洛哥)|哈桑二世 (摩洛哥)]]去世，思想更为自由的[[穆罕默德六世|穆罕默德六世]]繼任，即开始了成功的现代化改革，摩洛哥的人权纪录有了显著的改进。

2000年3月，妇女团体在[[卡萨布兰卡|卡萨布兰卡]]舉行了示威活動，要求改革妇女的法律地位，共有約四万名妇女参加此活動，呼吁禁止一夫多妻制和引入离婚法（当时离婚完全是屬於宗教程序）。尽管反对一方的示威活動吸引了五十万人参加，但改变现状的运动仍在穆罕默德国王的影响下持續進行，一项新的「摩洛哥个人身份法」（Mudawana，家庭法），在2004年初颁布，迎合了一些女权分子的要求。

2002年，摩洛哥占据无人的[[佩雷希尔岛|佩雷希尔岛]]（Perejil）。2003年5月16日，在卡萨布兰卡的一系列针对平民的炸弹袭击中，造成33名平民死亡，100多人受伤，随后对伊斯兰反政府者的镇压，受到了人权人士的批评。

2004年6月，[[美国|美国]]总统[[乔治·沃克·布什|布什]]赞扬了摩洛哥在[[反恐战争|反恐战争]]中的努力，确定摩洛哥为其主要的非[[北约|北约]]盟国之一，同一年稍晚時，摩洛哥與[[美国|美国]]以及其主要贸易伙伴[[欧盟|欧盟]]签订了[[自由贸易协定|自由贸易协定]]。

2005年，支持[[西撒哈拉|西撒哈拉]]独立的示威和骚乱在摩洛哥控制的[[阿尤恩|阿尤恩]]地區爆发，虽然有军队和警察的增援，但至今仍未能控制住抗议活动，而包含來自於「[[國際特赦組織|國際特赦組織]]」、「[[自由之家_(美國)|自由之家 (美國)]]」與「[[人權觀察|人權觀察]]」等組織的批評，主要在於其觀察到警察对示威者和独立人士的虐待行為。被参與者称为「独立起义」（Independence Intifada）的活動繼續受到[[西撒人阵|西撒人阵]]的支持，至今仍在持续中。
