'''摩尔体积'''是指单位[[物質的量|物质的量]]的某种物质于[[标准状态|标准状态]]（0℃、100kPa）的体积，也就是一[[摩爾_(單位)|摩尔]]物质在标准情况下的体积。

<math>V_m=\frac{V}{n}</math>

其中，<math>V_m</math>为摩尔体积，<math>V</math>为[[体积|体积]]，<math>n</math>为[[物质的量|物质的量]]。

==气体摩尔体积==

0 ℃（273.15 K）、100kPa称为[[标准状况|标准状况]](STP)。1摩尔的理想气体在STP下占有的体积为22.710953（21）L(CODATA 2010)，为标准摩尔体积。也就是说在标准状况下<math>V_m=22.710953</math>L/mol。

理想中一摩尔气体在标准大气压下的体积为22.4L，较精确的是：<math>V_m=22.41410</math>L/mol。
使用时应注意：

# 必须是在一定温度和压强下（101kpa 0℃）。
# “任何气体”既包括纯净物又包括气体混合物。
# 22.4升是个近似数值。
# 单位是升/摩尔，而不是升。<br />

单位物质的量的气体所占的体积叫做气体摩尔体积，相同气体摩尔体积的气体其含有的粒子数也相同。气体摩尔体积不是固定不变的，它决定于气体所处的温度和压强。如在25度101千帕时气体摩尔体积为24.5升/摩尔
在外界条件相同的情况下，气体的摩尔体积相同。<br />

气体摩尔体积<math>V_m</math>与<math>T</math>、<math>P</math>、<math>n</math>等之间关系　<br />

# 同温度、同压强下，<math>V</math>相同，则<math>N</math>相同，<math>n</math>相同
# 同温度、同压强下，<math>\frac {V_1}{V_2}=\frac {n_1}{n_2}=\frac {N_1}{N_2}</math>
# <math>PV=nRT</math>
