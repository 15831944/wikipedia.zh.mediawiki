{{作品名稱|name=''Skyscraper''|hk=高凶浩劫|tw=摩天大樓|cn=摩天营救}}是一部2018年[[美國|美國]][[動作片|動作]][[驚悚片|驚悚片]]，由[[勞森·馬修·托波|勞森·M·托波]]執導和編劇。其主演包括[[巨石強森|巨石強森]]、[[妮芙·坎貝爾|妮芙·坎貝爾]]、[[黃經漢|黃經漢]]、{{le|羅蘭·穆勒|Roland Møller}}、[[帕布羅·薛伯|帕布羅·薛伯]]、[[文峰|文峰]]、[[昆凌|昆凌]]和[[諾亞·泰勒|諾亞·泰勒]]。拍攝於2017年9月在[[加拿大|加拿大]][[溫哥華|溫哥華]]開始進行。該片由[[環球影業|環球影業]]定於2018年7月13日在美國上映。

== 劇情 ==
前[[美國海軍|美國海軍]]兼[[聯邦調查局|聯邦調查局]][[人質拯救隊|人質拯救隊長]]威爾·索耶，一次任務中沒能勸降匪徒而導致他引爆[[自殺攻擊|炸彈背心]]，不但人質犧牲、連他自己都受爆炸波及而失去左腿。他裝上[[義肢|義肢]]並退伍後，跟治療過他的女軍醫莎拉相愛，結婚後有兩個孩子喬吉雅和亨利。十年後，威爾在老同事班·吉萊斯比的介紹下陪家人蒞臨[[香港|香港]]，來到香港剛完工的-{[[摩天大樓|摩天大樓]]}-「明珠塔」擔任安全顧問。明珠塔由科技大亨趙隆基投資建造，有3500英呎和225層樓高，現已成為[[摩天大樓列表|世界第一高]]的-{摩天大樓}-，只需要威爾的安全鑒定通過就能對公眾開放上層的住宅區。威爾審核過全樓的完善[[滅火|滅火]]裝置和安全系統後，認為萬無一失而允許趙隆基開放上層區。趙隆基對威爾表示信心，交給他一台能[[遠程控制|遠程控制]]整棟樓安全系統的[[平板電腦|平板電腦]]，其只能靠威爾的{{link-en|面部識別系統|Facial recognition system|面部識別}}才能登陸。

威爾和班搭[[香港渡輪|渡輪]]前往明珠塔對岸的遠程控制中心時，威爾的包被一名小偷搶跑，而班將威爾帶回家包扎傷口時，卻發現原本會被偷的平板被威爾放在身上。班不久後突然掏出槍表示自己為[[恐怖分子|恐怖分子]]工作，試圖搶走平板卻被威爾反制且擊中胸口，臨死前叫威爾趕緊離開。恐怖分子随即趕到，威爾剛從後窗逃跑，女領隊小夏搶走他的平板，並且靠威爾的臉登陸明珠塔的系統。而以寇瑞斯·波薩率領的恐怖分子潛入明珠塔內部，開始在96層實施[[縱火|縱火]]，和闖入遠程控制中心的小夏裡應外合關閉所有滅火系統，讓火勢開始失控般的往上竄。趙隆基原本對自己大樓的安全系統表示強大信心，但見自己失去系統登陸權限而只好安排撤離。在下層，莎拉當天原本陪兩個兒女去動物園玩，但因亨利吃壞肚子而迫使一家人先回到明珠塔，導致她們受困於火災現場。

[[香港警務處|香港警務處]]的吳督察率領的緊急隊伍到場支援，安排滅火過程中也派警察追捕逃亡在外的威爾。被當成嫌犯的威爾得知妻兒都被困在明珠塔裡後，只好擺脫警察並冒險爬上離塔最近的施工用[[起重機|起重機]]，最後縱身一跳而幸運地爬回大樓內部尋找家人。當趙隆基準備全員搭上私人[[直升機|直升機]]撤離時，身為恐怖分子之一的{{link-en|保險經紀|Insurance broker}}皮爾斯，跟來到頂樓的波薩等人共同殺死所有保全後抓獲趙隆基。波薩原本準備搭上直升機撤離，保安隊長奧傑奇死前槍殺駕駛員而使直升機墜毀，也使趙隆基逃跑並躲回自己的[[空中别墅|頂層公寓]]。由於公寓外門堅固且使用獨立系統，造成波薩無法打開門，但當他得知威爾回到大樓裡時，有意想讓他來幫自己開門而派皮爾斯到下層去抓威爾的家人。威爾及時趕到家人身邊並將皮爾斯踢下樓掉進火海，他讓莎拉和亨利坐進電梯裡切斷纜繩，讓她們下墜至一樓前及時拉動[[電梯限速器|限速器]]而獲救。

喬吉雅跟家人走散後困在上方且被恐怖分子追趕，威爾雖然及時救下她，但父女倆一起被波薩抓獲。波薩用喬吉雅性命要挾威爾想辦法打開門抓回趙隆基，隨後挾持著喬吉雅前往頂樓，無從選擇的威爾被迫再次冒險往外墻爬，到大樓中心的巨大[[渦輪發動機|渦輪發動機]]內部打開[[子程序|子程序]]面板，打開門後及時回去到裡面找到趙隆基。趙隆基坦述他建造大樓時屢次受波薩[[敲詐|敲詐]]，因此在用[[電子商務|電子匯款]]付他錢的過程中植入追蹤程式，由此搜集到波薩所工作的每一位犯罪首腦的[[洗錢|洗錢]]等不法活動；所有資料已經被他存入一個隨身[[硬碟|硬碟]]中，同時也迫使波薩安排這次縱火來湮滅人證和物證。

莎拉和亨利獲救後向吳督察澄清威爾不是恐怖分子，同時協助警方指認波薩一行人的身份，最後推斷波薩他們即將在塔頂跳傘至河對岸的[[中環碼頭|中環碼頭]]工地。莎拉跟著吳督察及時趕到碼頭，跟在此接應的小夏的軍隊發生交火，莎拉幫忙制伏小夏後從她手裡取回威爾的平板。在塔頂，威爾帶著趙隆基和硬碟打算向波薩贖回女兒，而趙隆基故意演苦肉計使波薩分心，喬吉雅也跑回父親身邊。急切想拿回硬碟的波薩帶人追趕他們，所有人回到塔頂的標誌性球體內部分散，當中安置的多數[[動態捕捉|動態捕捉]]屏幕讓所有人真假難分。威爾殺死大部分波薩的士兵，趙隆基奮力抵抗卻中槍受傷，將唯一的手槍交給威爾去救女兒。波薩這時再度挾持喬吉雅並用一顆[[手榴彈|手榴彈]]威脅面前的威爾投降，但後來才發現面前的人還是影像，最後被身後的威爾踢下樓，直接被自己的手榴彈炸死。

威爾和喬吉雅父女團圓不久，火勢卻竄到頂樓而讓他們無處可逃，所幸得到平板的莎拉將整棟樓的安全系統重啟，恢復滅火系統才撲滅整棟樓的火災。救援直升機來到接走威爾三人，趙隆基表示他接下來要向警方上交硬碟並重建明珠塔。直升機到達河對岸後，威爾和莎拉一家人終於團圓，一家人在圍觀市民的歡呼聲中離開現場。

== 反響 ==
=== 評價 ===
[[爛番茄|爛番茄]]上持有47%的新鮮度，平均得分5.15／10<ref>{{Cite web |url=https://www.rottentomatoes.com/m/skyscraper_2018|title=Skyscraper (2018) |work=[[Rotten_Tomatoes|Rotten Tomatoes]] |accessdate=2018-07-16}}</ref>，不過觀眾新鮮度高達65%，平均得分3.7／5，在[[Metacritic|Metacritic]]上，電影獲得了51分<ref>{{cite web |url=http://www.metacritic.com/movie/skyscraper|title=Skyscraper Reviews |work=[[Metacritic|Metacritic]] |publisher=[[CBS_Interactive|CBS Interactive]] |accessdate=2018-07-12}}</ref>。據[[CinemaScore|CinemaScore]]調查，觀眾的評價在A+至F間落於「B+」<ref name=opening />。

=== 票房 ===
北美方面，《摩天大樓》與《[[尖叫旅社3：怪獸假期|尖叫旅社3：怪獸假期]]》及擴大上映的《{{le|抱歉打擾你|Sorry to Bother You (film)}}》同天上映並競爭，分析師預估《摩天大樓》在首週能從3782間戲院中獲得約3200萬至4000萬美元的票房<ref name=BOpreview>{{cite magazine |last=McNary |first=Dave|url=https://variety.com/2018/film/box-office/box-office-hotel-transylvania-3-summer-vacation-1202854033/ |title=‘Hotel Transylvania 3: Summer Vacation’ on Track for $40 Million-Plus Opening|magazine=[[Variety_(magazine)|Variety]] |publisher=Penske Business Media |date=2018-06-21 |accessdate=2018-06-27}}</ref>。但《摩天大樓》在週四晚間進取得僅195萬美元（輸給強森的4月電影《[[毀滅大作戰|毀滅大作戰]]》之240萬美元）並在第一天取得930萬美元，這使首週的預估值降低至2400萬美元。該片的美國首週票房最終開出了2490萬美元，位居當週第三，輸給《尖叫旅社3：怪獸假期》及《[[蟻人與黃蜂女|蟻人與黃蜂女]]》。[[Deadline_Hollywood|Deadline Hollywood]]網站認為，《摩天大樓》的表現之所以不佳，主因歸於觀眾早已在別部片看過片中的情節，且7月這個上映檔期是處於相當擁擠的夏季電影高峰，以及可能是因過於頻繁地看見巨石強森的電影而使人感到厭倦（雖然該網站發現有72%的觀眾是因為強森才買了《摩天大樓》的票）<ref name=opening>{{cite web|url=https://deadline.com/2018/07/dwayne-johnson-skyscraper-ant-man-and-the-wasp-hotel-transylvania-3-weekend-box-office-1202426089/|title=Why Dwayne Johnson Pic ‘Skyscraper’ Went Up In Flames At The B.O. With $25M+|website=[[Deadline_Hollywood|Deadline Hollywood]] |publisher=Penske Business Media |date=July 15, 2018 |first=Anthony |last= D'Alessandro |accessdate=July 15, 2018}}</ref>。

台灣方面，首週四天票房為新台幣6130萬元<ref name="tfi1829">{{cite web |title=全國電影票房2018年07/16-07/22統計資訊 |url=https://www.tfi.org.tw/Content/TFI/PublicInfo/全國電影票房2018年0716-0722統計資訊.pdf |website=[[國家電影中心|國家電影中心]] |date=2018-07-29}}</ref>；次週票房累計至新台幣1.31億元<ref name="tfi1829"/>；第三週票房累計至新台幣1.62億元<ref>{{cite web |title=全國電影票房2018年07/23-07/29統計資訊 |url=https://www.tfi.org.tw/Content/TFI/PublicInfo/全國電影票房2018年0723-0729統計資訊.pdf |website=[[國家電影中心|國家電影中心]] |date=2018-08-02}}</ref>；第四週票房累計至新台幣1.78億元<ref name="tfi1831">{{cite web |title=全國電影票房2018年07/30-08/05統計資訊 |url=https://www.tfi.org.tw/Content/TFI/PublicInfo/全國電影票房2018年0730-0805統計資訊.pdf |website=[[國家電影中心|國家電影中心]] |date=2018-08-09}}</ref>；最終全台票房為新台幣1.87億元<ref name="tfi1847">{{cite web |title=全國電影票房2018年11/19-11/25統計資訊 |url=https://www.tfi.org.tw/Content/TFI/PublicInfo/全國票房2018年1119-1125統計資訊.pdf |website=[[國家電影中心|國家電影中心]] |date=2018-11-29}}</ref>。
