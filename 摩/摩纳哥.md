'''摩納哥親王國'''（{{lang-fr|Principauté de Monaco}}），也譯作'''摩納哥[[公國|公國]]'''，中文通稱'''摩納哥'''，是一个位于[[欧洲|欧洲]]的[[城邦国家|城邦国家]]。摩納哥地处[[法国南部|法国南部]]，除了靠[[地中海|地中海]]的南部海岸線之外，全境北、西、東三面皆由[[法國|法國]]包圍，主要是由摩纳哥旧城和随后建立起来的週遭地区组成。作为世界上[[人口|人口]]稠密的国家之一，摩纳哥也是一个典型的[[微型国家|微型国家]]和[[城邦|城邦]]，得利於[[蒙特卡洛賭場|蒙特卡洛賭場]]及觀光收入，作為不向國民課稅的國家而聞名，有避稅天堂之稱。

== 历史 ==
{{main|摩納哥歷史}}
摩纳哥的地名源自於約公元前6世纪，由來自[[希臘|希臘]][[福基斯|福基斯]]（Phocaea）地區的福基斯人於該地區附近所建立的希臘殖民地——[[摩諾伊科|摩諾伊科]]（Monoikos）。根據[[希臘神話|希臘神話]]，相傳大力士[[海格力斯|海格力斯]]曾途經這裡，因此福基斯人在此地建了一個神廟祭祀海克力斯，稱為「Monoikos」（{{lang-el|Μόνοικος}}，意指獨棟的房屋或神殿），並逐漸轉變成此處的地名。

摩纳哥于1215年再次被作为[[熱那亞|熱那亞]]的殖民地创建。自1297年，当弗朗索瓦·格里马尔迪乔装成[[方济会|方济会]]（Franciscan）的[[修道士|修道士]]（在意大利语中称为''monaco''）占领了保护那块著名的礁石的要塞后，摩纳哥一直处于[[格里马尔迪王朝|格里马尔迪]]（Grimaldi）家族统治下，唯一的例外是自1793年到1814年，摩纳哥是被[[法国|法国]]控制的。1815年至1860年，[[维也纳会议|维也纳会议]]指定[[薩丁尼亞王国|薩丁尼亞王国]]为摩纳哥的保护国，直到1861年，摩纳哥的主权才通过法国-摩纳哥条约确定。

在1911年第一部宪法頒佈前，[[摩纳哥亲王|摩纳哥亲王]]曾经是至高无上的统治者。1918年7月，签署了一份条约，条约规定法国在摩纳哥提供有限的保护。这条约被写入《[[凡尔赛和约|凡尔赛和约]]》，建立了摩纳哥的政策应该按照法国的政治、军事、和经济利益进行调整。1919年同[[法国|法国]]签订的条约规定，一旦[[國家元首|國家元首]]逝世而没有男性后裔，摩纳哥将并入[[法国|法国]]。兩國於2002年另立新條約，當摩纳哥國家元首無人繼承，摩纳哥仍然為獨立國家，法國亦繼續提供防衛。第二次世界大戰中，1942年，摩纳哥被義大利短暫占領，1943年義大利投降後撤出摩納哥。

[[兰尼埃三世|兰尼埃三世]]于1949年在其祖父[[路易二世_(摩纳哥)|路易二世]]去世后继承了王位。1962年颁布了一部新的宪法，废除了死刑，保护了女性的投票权，并建立了最高法院以保证基本的自由。1993年，摩纳哥成为联合国的正式成员国，具有完全投票权。

2005年3月31日，由於他的父親兰尼埃三世長期臥病在床，阿尔贝王子開始負起[[摄政|摄政]]的任務。2005年4月6日兰尼埃亲王身故，阿尔贝王子即位，成为[[阿尔贝二世_(摩纳哥)|阿尔贝二世]]。阿尔贝二世于2005年7月12日正式成为摩纳哥的统治者，他的即位庆典在他的父亲兰尼埃三世举行葬礼的大教堂举行。他的即位有两次庆典，另一次庆典在2005年的11月9日举行。

== 政治 ==
{{main|摩納哥政治}}
現時摩納哥由摩納哥亲王[[阿尔贝二世_(摩纳哥)|阿爾貝二世]]統治，是一個[[君主立憲|君主立憲]]國家。

摩纳哥自1911年以来实行[[君主立宪制|君主立宪制]]，亲王为[[国家元首|国家元首]]。[[行政机构|行政机构]]为由[[政府首脑|首相]]主持的四人政府委员会（[[内阁|内阁]]）。首相由一名法国公民担任，由親王从法国政府提交的多名候选人中选择并任命。根据1962年宪法约定，首相与一院制的国家会议（[[摩納哥議會|摩納哥議會]]）分享权力。这一立法机构的24名成员通过普选从名单中产生，任期5年。

摩纳哥的地方事务由公共会议管理。公共会议由15名选举产生的成员组成，由市长主持。

== 地理 ==
[[File:MonacoView.jpg|thumb]]
[[File:Monaco_BW_2011-06-07_16-07-20.jpg|thumb]]
[[File:Monaco,_Monaco.JPG|thumb]]
{{main|摩納哥地理|摩纳哥行政区划|摩納哥填海工程}}
摩纳哥位于地中海北岸，氣候乾燥溫暖，長年陽光普照，屬典型的[[地中海氣候|地中海氣候]]。全國被法国东南部的[[濱海阿爾卑斯省|濱海阿爾卑斯省]]三面環繞，国土东西方向长3500米，最窄处200多米，面積約202公頃（2.02平方公里，比紐約中央公園面積更小），是全球第二小的國家，其中五分之一的面積都是透過填海得出，隨之建成的是兩個泊滿遊艇的著名海港—[[芳特維耶|芳特維耶]]港和Port Hercules。全國高樓林立，沒有河流，海拔最高點是164.4公尺，位於一個斜坡之上。

== 经济 ==
{{main|摩納哥經濟}}
摩纳哥主要的收入来源之一是[[旅遊业|旅遊业]]；每年都有许多人被摩纳哥的赌场和舒适气候所吸引。2001年，一个新的建筑项目扩展了主要港口碼頭，主要用于停靠豪华遊轮。摩纳哥经济发达，主要以博彩、旅遊和银行业为主，该国在服务业和小型的、高附加值的、无污染的工业的多种经营上取得了成功的開發。

政府没有针对个人收入課税。由于全国人口过少，政府在许多公共服务部门保持垄断地位，包括邮政、电信服务。现在[[摩纳哥电信|摩纳哥电信]]由[[大东电报局|大东电报局]]持有49%的股份，政府持有45%的股份，还有摩纳哥银行（{{lang|fr|Compagnie Monégasque de Banque}}）持有6%的股份。摩纳哥的生活水准很高，与附近法国的大都市大体相当。

无个人所得税的政策吸引了数量可观的富裕避稅移民，主要是来自其他欧洲国家，他们透过在摩纳哥以外的事業赚得其大部分的收入；著名人士如[[F1赛车|F1赛车]]赛車手吸引了大多数的注意，但是大多数的移民是商人。1865年建立的[[蒙特卡洛赌场|蒙特卡洛赌场]]促使摩纳哥从穷国发展成为了富裕国家。

摩纳哥不是[[欧盟|欧盟]]的成员国，但是通过与法国的关税联盟使用和与法国一样的货币——[[欧元|欧元]]。摩纳哥与欧盟有非常紧密的联系。摩纳哥获得了[[摩納哥的歐元硬幣|歐元硬幣]]的铸造权，即有在硬币的国家面上使用摩纳哥自己的设计的权利。
全球最貴呎價的物業就是位於摩納哥的阿熱爾山山腰上的la villa別墅,該物業因地理位置問題,雖然面積僅158平方米,但面臨整個法國南部海灣的景色,地勢也比其他物業高,被譽為擁有最漂亮景觀的物業,該物業於2017年以2.56億美元售出,每平方米售價高達162萬美元.

== 交通 ==
{{main|摩納哥交通}}

摩納哥沒有國際機場，最就近的為法國[[尼斯蔚藍海岸機場|尼斯蔚藍海岸機場]]（Nice Côte d'Azur International Airport）。由機場至摩納哥市可乘坐尼斯公共巴士110線［終點站為芒通（MENTON）］，而尼斯市中心另有100線公共巴士可到達摩納哥（終點站同樣為芒通），需時約40分鐘。110線的車費為22歐元，一年有效的來回票為33歐元。另亦可選乘[[法國國鐵|法國國鐵]]（SNCF）火車，由尼斯開出約15分鐘可抵達[[摩納哥蒙地卡羅車站|摩納哥蒙地卡羅車站]]（Monaco/Monte-Carlo station）。

摩納哥市內交通主要依賴公共巴士，共有5條巴士線穿梭市內，每位車費2歐元（另有日票每位5歐元可供選擇），路線如下：

Route 1 : Monaco Ville - Saint Roman -  Monaco Ville

Route  2 : Monaco Ville - Jardin Exotique -  Monaco Ville

Route 4 : Condamine - Saint Roman - Condamine

Route 5 : Fontvieille - Hôpital -  Fontvieille

Route 6 : Fontvieille - Larvotto -  Fontvieille

由於整個城市均依山而建，市內部份地區亦設有地下行人隧道及公用升降機，讓行人用以來往主要景點。<ref>[http://www.cg06.fr/fr/servir-les-habitants/deplacements/transport-collectifs/lignes-et-horaires/lignes-et-horaires/ 尼斯往蔚藍海岸地區公共巴士路線資料]</ref><ref>[http://www.visitmonaco.com/en 摩納哥旅遊網]</ref><ref>[http://www.cam.mc/informations-pratiques.php?lang=en 摩納哥公共巴士路線資料]</ref>

== 人口 ==
{{main|摩納哥人口}}
摩纳哥是现在[[世界|世界]]上[[人口|人口]]最稠密的[[国家|国家]]，常住人口有32,020人，其中6,089是摩納哥本国人。本国籍的摩纳哥人在自己的国家的居民中成为少数派，这种人口状况在世界各国都较为少见。常住人口中最多的是[[法国人|法国人]]，占32%，其次是[[意大利人|意大利人]]，占20%，本地的[[摩纳哥人|摩纳哥人]]只占19%，剩下的29%人口由来自其他125个[[国家|国家]]的[[侨民|侨民]]组成，这些人构成了摩纳哥的国际人口。

摩納哥的人均及单位面积警力為世界上最大的。摩纳哥也是世界上国土面积第二小的国家，位列[[梵蒂冈|梵蒂冈]]之后。

[[法语|法语]]是摩纳哥唯一的[[官方语言|官方语言]]，但也使用[[英语|英语]]、[[意大利语|意大利语]]和摩纳哥本地的[[摩纳哥语|摩纳哥语]]（由[[利古里亚语|利古里亚语]]衍生）。[[识字率|识字率]]达99%。[[罗马天主教|罗马天主教]]是法定的[[国教|国教]]，佔人口的90%，由[[天主教摩納哥總教區|天主教摩納哥總教區]]管理，但摩纳哥宪法保障信仰其他宗教的自由。
