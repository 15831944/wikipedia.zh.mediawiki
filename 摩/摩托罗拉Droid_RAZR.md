'''Motorola Droid RAZR'''是[[美國|美國]][[摩托羅拉|摩托羅拉]]公司於2011年推出的智慧型手機。該機的7.1mm厚度讓其成為市場上最薄之雙核心智慧型手機。（薄於[[Samsung_Galaxy_S_II|Samsung Galaxy S II]]的8.49mm）該智慧型手機也是第一隻採用[[qHD|qHD]] [[Super_AMOLED|Super AMOLED]]的智慧型手機。

==規格<ref>{{cite web |url= http://mobilesyrup.com/2011/10/18/rogers-releasing-the-motorola-razr-in-time-for-the-holidays/|title= Rogers releasing the Motorola RAZR "in time for the holidays", $149.99 on a 3-year |last=Hardy |first=Ian |date=18 October 2011|work= |publisher= Mobile Syrup|accessdate=18 October 2011}}</ref>==
*螢幕：4.3" [[qHD|qHD]] [[Super_AMOLED|Super AMOLED]] （分辨率为960x540）（[[三星電子|三星電子]]製造）
*處理器：[[OMAP|OMAP4430双核處理器]]
*相機：800萬畫素（配置LED閃光燈）前置摄像头为130萬畫素
*重量：127g
*厚度：0.71cm
*産地：{{CHN}}
*[[隨機存取記憶體|RAM]]：1GB
*[[ROM|ROM]]：16GB

==系统==
'''Droid Razr'''发布是采用[[Android|Android]] 2.3.6，根據[[摩托罗拉|摩托罗拉]]官方表示，Droid Razr將於2012年第二季升级至[[Android_4.0.4|Android 4.0.4]]。<ref>{{cite web |url=https://forums.motorola.com/pages/00add97d6c |title=存档副本 |accessdate=2012-05-21 |0=2012-05-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20120509091558/http://forums.motorola.com/pages/00add97d6c |archivedate=2012-05-09 }} Motorola Android Software Upgrade News</ref>。 2013年第一季升级至[[Android_4.1.2|Android 4.1.2]]。

==中国版本==
[[摩托罗拉|摩托罗拉]]於2011年底在[[中国|中国]]发布[[CDMA|CDMA]]（型号为XT928，电信定制机)和[[TD-SCDMA|TD-SCDMA]](型号为MT917，移动定制机）版本。<ref>[http://cn.engadget.com/2011/11/04/motorola-razr-china-xt910/ 摩托罗拉正式在中国发布 RAZR XT910]</ref>其中[[CDMA|CDMA]]版本为双模双待，并且搭配擁有1300万畫素的照像鏡頭以及雙[[LED|LED]]閃光燈，还有[[杜比|杜比]]音效加持。<ref>[http://cn.engadget.com/2011/11/13/droid-razr-china-td-scdma-cdma2000-gsm/ 中国电信及中国移动版 Droid RAZR 现身]</ref>

==Droid RAZR MAXX==
'''Droid RAZR MAXX'''為Droid Razr的[[电池|电池]]容量加大版本，該款手機之电池容量提升至3300mAh，但厚度增加至8.99mm。

==解锁Bootloader==
根據'''Engadget'''報導<ref>[http://www.engadget.com/2011/10/24/motorola-razr-open-for-unlocked-bootloader-business-if-the-carr/ Motorola RAZR open for unlocked bootloader business, if the carrier says so]</ref>，国际版的Droid Razr将解锁[[Bootloader|Bootloader]]，这将允许用户更方便的刷[[ROM|ROM]]以及获得[[Root|Root]]权限。
