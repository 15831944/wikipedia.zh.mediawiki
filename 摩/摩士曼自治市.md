'''摩士曼自治市'''是[[澳大利亞|澳大利亞]]的一個[[地方政府區域|地方政府]]，乃[[雪梨|雪梨]]地區的重要[[衛星城市|衛星城市]]之一。

== 人文 ==
根據澳大利亞統計局資料<ref>{{cite web
|publisher = [[:en:Australian_Bureau_of_Statistics|ABS]] 
|title=Regional Population Growth, Australia, 2005-06 (catalogue no.: 3218.0)
|url = http://abs.gov.au
|accessdate = 2007-03-12 }}</ref>，
2006年6月30日摩士曼自治市有居民28,414人是新南威爾斯州內人口數排名第70大城，比例佔州內總人口6,827,694的0.42%。2007年6月，《[[雪梨晨鋒報|雪梨晨鋒報]]》指出<ref>[http://www.epochtimes.com/b5/9/12/19/n2759381.htm Mosman工資收入居全國之首]</ref>，摩士曼居民的工資收入為澳大利亞第一，每年個人平均是10.84萬澳幣，與3年前相較，成長了30%。

== 市議會 ==
摩士曼自治市議會有議席9座，分由3個選區，各選出3位議員，市長非直選。<ref name=EC>{{cite web|url = http://www.lg.elections.nsw.gov.au/LGE2008/result.Mosman.html|title = Mosman Municipal Council|work = 2008 Election results| publisher = Electoral Commission NSW| accessdate = 2009-08-02}}</ref> 

==姊妹市==
摩士曼與同州的[[:en:Glen_Innes,_New_South_Wales|格林茵斯]]結為雙子城。
