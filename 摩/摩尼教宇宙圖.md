《'''摩尼教宇宙圖'''》（[[日語|日語]]：{{lang|ja|マニ教宇宙図}}）是一幅[[元朝|元朝]]時期的絹布彩繪，描繪[[摩尼教|摩尼教]]「十層天八層地」宇宙觀的整體像，繪製手法和藝術風格同《[[聖者傳圖_1|聖者傳圖 1]]》、《[[聖者傳圖_2|聖者傳圖 2]]》、《[[摩尼的父母|摩尼的父母]]》及《[[摩尼誕生圖|摩尼誕生圖]]》非常相似。此畫於2010年被[[京都大學|京都大學]]文獻語言學教授{{Link-ja|吉田豐 (語言學家)|吉田豊 (言語学者)|吉田豐}}發現，被他稱為「宇宙圖」，傳入日本的時間不明，現屬私人收藏<ref name=宇宙図 />。

== 概述 ==
該畫可能出自中國元朝的浙江、福建等南方地區畫師之手。吉田豐發現畫中有穿着摩尼教特有的紅邊白袍僧侶服飾的人物，在與中國新疆發現的摩尼教史料對比後，斷定該圖為摩尼教繪畫。此畫頂端被分隔開的區塊描繪的是摩尼教的天堂，或稱「光明王國」。主體部份頂端繪有兩個圓形區域，依左右順序分別為月宮和日宮<ref>{{cite web |url=https://blogimg.goo.ne.jp/user_image/46/78/c32050dc18514eb85114fae7ed9072ee.jpg |title=国内にマニ教宇宙図 |accessdate=2018年7月7日 |language=ja}}</ref>。往下就是用圓弧分隔的十層天，其中不僅描繪有天使、惡魔等形象，還有西方的[[黃道十二宮|黃道十二宮]]。再下方的人類世界（凡間）地面上矗立着蘑菇狀的「[[須彌山|須彌山]]」，最下方被解釋為摩尼教的地獄<ref name=宇宙図>{{cite web |url=https://www.nikkei.com/article/DGXNASDG2600R_W0A920C1CR8000/ |title=国内にマニ教「宇宙図」　世界初、京大教授ら確認 |date=2010年9月26日 |newspaper=[[日本經濟新聞|日本經濟新聞]] |accessdate=2018年7月7日 |language=ja}}</ref>。

== 補說 ==
[[File:Collection_of_Chinese_Manichaean_Hanging_Scrolls.jpg|thumb]]
目前已知的中國摩尼教絹畫掛軸包括該《宇宙圖》在內共八幅，可分為四大類：
; 單體神像兩幅（描繪摩尼和耶穌）
* [[摩尼像絹畫掛軸|摩尼像]]
* [[夷數佛幀|夷數佛幀]]
; 描繪[[救贖論|救贖論]]（{{lang|en|Soteriology}}）的畫軸一幅
* [[冥王聖幀|冥王聖幀]]
; 描繪先知論（{{lang|en|Prophetology}}）的畫軸四幅
* [[摩尼的父母|摩尼的父母]]
* [[摩尼誕生圖|摩尼誕生圖]]
* [[聖者傳圖_1|聖者傳圖 1]]
* [[聖者傳圖_2|聖者傳圖 2]]
; 描繪[[宇宙論|宇宙論]]（{{lang|en|Cosmology}}）的畫軸一幅
* 摩尼教宇宙圖
