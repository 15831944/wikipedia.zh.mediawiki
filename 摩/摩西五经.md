《'''五書'''》（{{lang-el|πέντετεῦχος}}，'''Pentateuch'''），常被稱為《'''摩西五經'''》（{{lang-en|Five Books of Moses}}，华人天主教会譯为《'''梅瑟五書'''》；[[正教會|東正教]]譯《'''摩伊息斯之五經'''》；唐朝[[景教|景教]]譯《'''牟世法王經'''》<ref>{{Cite web |url=http://www.vccstm.ca/Otm/issue88/0101cs02.htm |title=福音與中華：中文聖經的問世 |accessdate=2018年7月1日 |author=楊愛程 |newspaper=真理報 |archive-url=https://web.archive.org/web/20180701030523/http://www.vccstm.ca/Otm/issue88/0101cs02.htm |archive-date=2018年7月1日 |dead-url=yes }}</ref>），[[猶太教|猶太教]]稱為[[妥拉|妥拉]]（{{lang-he|תּוֹרָה}}，'''Torah'''），又稱為'''摩西律法'''（{{lang-he|תּוֹרַת־מֹשֶׁה}}，Torat Moshe），基督徒常稱為'''律法書'''，是《[[希伯来圣经|希伯来圣经]]》最初的五部經典。

==名稱起源==
《摩西五經》（{{lang-en|Pentateuch}}）源自於{{lang-el|πέντε}}（pénte，意為五）與{{lang|el|τεῦχος}}（teûkhos，意為書卷、法案），字面直譯為五卷書。相傳這五卷皆由[[摩西|摩西]]寫成，因此稱摩西五書。

== 概况 ==
全經用最古老的[[希伯來文|希伯來文]]寫成，是[[猶太教|猶太教]]最重要經典之一，也是西元前6世紀以前唯一的[[希伯來|希伯來]]法律匯編。摩西五經曾作為[[猶大國|猶大國]]的國家法律規範；猶大國亡國後仍以習慣法的形式自動調節猶太人的生活。傳統一向認為這是由[[摩西|摩西]]接受[[上帝|上帝]]的啟示而撰寫的，內容是古代的[[以色列人|以色列人]]民間故事，记载了以色列民族的起源，尤其是创世的上帝对他们的启示。

摩西五經主要思想是六個重要的教義：
# 上帝的創造
# 人的尊嚴與墮落
# 上帝的救贖
# 上帝的揀選
# 上帝的立約
# 上帝的律法

== 成书 ==
摩西五经的成书时间目前尚不能确定，可以视之为一些文献的结集，其中一些文献非常古老。

公元前6世纪以色列人被掳至巴比伦时，这些文献被汇编起来。这段时期是犹太人经历的一个极其重要的时期，他们搜集能够得到的关于以色列人的历史文献，汇编在一起，通过这样的方式来维持和肯定自身的信仰基础。他们认为，这些文献记载了他们成为上帝选民的历史，即使漂泊在外，这些历史都对他们的前途命运有着重要影响。但目前的文献尚不能确切地描述摩西五经的成书过程。

=== 底本學說  ===
{{Main|底本學說}}

[[圣经批评|圣经批评]]學者分析摩西五經，認為它們是由不同年代、不同來源的文件所編輯而成，並非[[摩西|摩西]]一人所獨撰。學者根據他們文句措辭的風格不同，將其分析比對出五個不同來源：
#[[耶和華文獻|耶和華文獻]]（The Jahwist），簡稱J典、或耶典，因文集均稱呼上帝為「[[耶和華|耶和華]]」而得名。約成書於前950年前後。
#[[伊羅興文獻|伊羅興文獻]]（The Elohist），簡稱E典、或伊典。這部分稱上帝為「[[伊羅興|伊羅興]]」，為複數形名詞，這是古[[迦南|迦南]]對神的稱法。約成書於前721年前後。
#[[申命記文獻|申命記文獻]]（The Deuteronomist），簡稱D典、或申典。約成書於前650年前後，由[[猶太|猶太]]王國的[[約西亞|約西亞]]王所主導。
#[[祭司文獻|祭司文獻]]（Priestly Book），簡稱P典、或祭司典。約成書於前538年前後，由從巴比倫回歸的[[以斯拉|以斯拉]]等[[文士|文士]]所主導。
#[[托拉|托拉]]編輯者（Torah redactors），約成書於前450年前後。

== 争议 ==
因為《[[申命記|申命記]]》記錄到摩西死後之事，其作者是否為摩西或者該經卷是否經後人增改，仍有爭議。（參考[[文本說|文本說]]/底本說）

== 主要經卷 ==
在《[[聖經|聖經]]》原文裡，五卷經文是不分開的，包括:
# 《[[創世記|創世記]]》（בראשית，Bereshit：「在起初」之意）、
# 《[[出埃及記|出埃及記]]》（שמות，Shemot：「名字」之意）、
# 《[[利未記|利未記]]》（ויקרא，Vayyiqra：「他呼叫」之意）、
# 《[[民數記|民數記]]》（במדבר，Bammidbar：「在沙漠」之意）和
# 《[[申命記|申命記]]》（דברים，Devarim：「言语」之意）。

五卷經書名字皆取自每卷經文第一句話。<ref name="托辣各卷名称皆为每卷首字？">[http://www.jewishjournal.com/you_tai_ren/item/the_first_word_and_the_name_of_each_book_20100321/ 托辣各卷名称皆为每卷首字？]</ref>
