'''杭州消防主题公园'''位于[[杭州经济技术开发区|杭州经济技术开发区]]，于2009年7月6日正式亮相。

== 介绍 ==
公园面积约500平米，具有[[消防|消防]]主题特色，使市民在休闲娱乐中，了解如何防范和应对[[火灾|火灾]]发生、如何进行火场逃生等常识<ref>{{cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2009/07/07/015649982.shtml | language= | title=杭州消防主题公园 |publisher=浙江在线新闻网站 |date=2009年7月7日 |accessdate= }}</ref>。公园入口处矗立着一个高12米、由一辆报废的32米云梯车和一辆报废的泡沫水罐消防车纯手工打造而成“消防金刚霄霄”<ref>{{cite web|url=http://news.163.com/10/1217/20/6O4PJ9T100014JB6.html | language= | title=“消防金刚”首次亮相 将镇守杭州消防主题公园 |publisher= 中国新闻网 |date=2010-12-17 |accessdate= }}</ref>。
[[File:杭州消防主题公园-消防金刚.jpg|缩略图]]

公园获2011杭州市民体验日“金城标奖”<ref>{{cite web |url=http://news.xiashanet.com/a/n/xs/68184.htm |language= |title=下沙消防主题公园 喜获杭州“金城标奖” |publisher=杭州网 |date=2011-06-23 |accessdate= |archive-url=https://web.archive.org/web/20160304202602/http://news.xiashanet.com/a/n/xs/68184.htm |archive-date=2016-03-04 |dead-url=yes }}</ref>。

== 地理位置 ==
公园西边、南边和北边分别沿[[文泽路|文泽路]]、[[学林街|学林街]]和[[学源街|学源街]]。

公园东侧为[[浙江理工大学|浙江理工大学]]生活区，南侧为理工大学校园，西侧为[[福雷德广场|福雷德广场]]。
