'''杭州湾跨海大桥'''是一座横跨[[中国|中国]][[杭州湾|杭州湾]]海域的跨海大桥，它北起[[浙江省|浙江省]][[嘉兴市|嘉兴市]][[海盐|海盐]][[郑家埭|郑家埭]]，南至[[宁波市|宁波市]][[慈溪|慈溪]][[水路湾|水路湾]]。<ref name="工程技术" />它是[[沈海高速公路|沈海高速公路]]的一部分，全长36公里，比连接[[巴林|巴林]]与[[沙烏地阿拉伯|-{zh-hans:沙特阿拉伯;zh-hant:沙特阿拉伯;zh-tw:沙烏地阿拉伯;zh-hk:沙地阿拉伯;zh-sg:沙地阿拉伯;zh-mo:沙地阿拉伯;}-]]的[[法赫德国王大桥|法赫德国王大桥]]长11公里，2008年建成时曾为世界上最长的跨海大桥。在浙江省高速公路规划中，杭州湾跨海大桥又被定名为“'''杭州湾宁波通道'''”，为未来[[浙江高速公路|浙江省高速公路网]]“两纵两横十八连三绕三通道”中的“三通道”之一。

== 历史 ==
杭州湾跨海大桥2002年经[[中华人民共和国国家发展和改革委员会|国家计委]]批准立项，2003年6月8日奠基，<ref>[http://unn.people.com.cn/GB/channel455/456/1230/200306/09/262997.html 宁波杭州湾跨海大桥奠基 总长36公里为世界之最]</ref>2007年6月26日全线贯通，<ref>[http://gtog.ningbo.gov.cn/art/2007/6/27/art_1005_23159.html 杭州湾跨海大桥全线贯通]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>2008年5月1日建成通车。通车后从[[浙江|浙江]][[宁波|宁波]]到[[上海市|上海]][[莘庄镇|莘庄]]的陆路距离只有179公里，缩短了120公里左右，大大缓解已经拥挤不堪的[[沪杭高速公路|沪杭]]和[[杭甬高速公路|杭甬高速公路]]的压力，也形成以上海为中心的江浙沪两小时交通圈。<ref>[http://gtog.ningbo.gov.cn/art/2007/6/27/art_1005_23159.html 杭州湾跨海大桥全线贯通]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2013年7月上海實業（集團）有限公司斥資14.66億元人民幣收購杭州灣跨海大橋23.0584%的股權。

== 投资 ==
杭州湾跨海大桥主要投资来自浙江省地方政府和浙江民营企业，没有依靠国家投资，是目前浙江的[[地标|地标]]之一。大桥总投资预计超过161亿元[[人民币|人民币]]。其中大桥部分长36公里，耗资118亿元；北岸连接线部分长29.1公里，投资17亿元；南岸连接线部分长55.3公里，投资34亿元。来自民间的资本占了总资本的一半，包括[[雅戈尔|雅戈尔]]、[[方太厨具|方太厨具]]、[[海通集团|海通集团]]等[[民营企业|民营企业]]都参与了对大桥的投资。大桥收费年限为30年，收费标准预计为80元/辆。

== 工程建设 ==
[[File:HangzhouBBTowers.jpg|thumb]]
杭州湾跨海大桥所处的杭州湾地区是世界三大强潮海湾之一。受到台风影响较大，水流湍急，潮流紊乱容易产生冲刷，南岸滩涂地下存在沼气，这些自然条件给大桥的工程建设带来了较大的困难。<ref name="工程技术">[http://www.cixi.gov.cn/art/2009/2/27/art_17884_277747.html 杭州湾大桥工程介绍]</ref>

为了克服桥长、地质条件复杂等困难，杭州湾跨海大桥建设中采用多种先进技术。其中，超长钢管桩、预应力混凝土箱梁、大型平板车梁上运梁工艺、[[GPS|GPS]]测量和防腐蚀工艺是大桥建设中的亮点。<ref name="工程技术" />

大桥桥宽33米，为双向六车道，设计时速100公里，设计使用寿命100年以上。大桥设有南、北两个航道，其中北航道宽325米，高47米，可通航3万5千吨级[[船舶|船舶]]；南航道宽125米，高31米，可通航3000吨级船舶。<ref>[http://unn.people.com.cn/GB/channel2/3/30/3648/3649/200306/09/263225.html 杭州湾跨海大桥项目概况]</ref>大桥设计中结合景观，桥面为S形，这种设计可以消除驾驶者的疲劳心理。<ref name="工程技术" />在离南岸约14公里处，设计有一个面积达1.2万平方米的交通服务救援海上平台，同时也是旅游休闲观光平台的[[杭州湾跨海大桥海中平台|杭州湾跨海大桥海中平台]]。<ref>[http://gtog.ningbo.gov.cn/art/2008/9/9/art_1005_224025.html 杭州湾跨海大桥海中平台改造项目开工] {{webarchive|url=https://web.archive.org/web/20090313122758/http://gtog.ningbo.gov.cn/art/2008/9/9/art_1005_224025.html |date=2009-03-13 }}</ref>

== 经济影响 ==
杭州湾跨海大桥使得[[慈溪市|慈溪市]]北部交通大为改观。早在2003年，慈溪市便在北部大桥登陆处设立慈溪经济技术开发区，后来升格为[[杭州湾新区|杭州湾新区]]。2010年，杭州湾新区正式挂牌成立，目标为建设浙江的“浦东”。<ref>[http://news.cnnb.com.cn/system/2010/04/16/006492133.shtml 杭州湾新区：全力推进浙江“新浦东”建设]</ref>

== 花絮 ==
[[File:Hangzhou_Bay_Bridge_Tour_Station.jpg|thumb]]
*2008年5月2日大桥开通次日，排队上桥的车辆甚至在距离主桥几十公里处排起难得一见的车辆长龙，迫于车流量过大的原因，当晚开放的大桥景观灯光被迫取消，只保留了路灯的灯光。<ref>[http://news.tfol.com/10026/10032/2008/5/3/10554748.shtml 杭州湾跨海大桥壮观 百车违规停留观光] {{webarchive|url=https://web.archive.org/web/20080612110559/http://news.tfol.com/10026/10032/2008/5/3/10554748.shtml |date=2008-06-12 }}</ref>
*大桥开通后的一段时间内，并不允许大货车通行。直到2008年10月11日方正式解禁。<ref>[http://news.eastday.com/m/20081011/u1a3912188.html 杭州湾跨海大桥今通货车 物流纽带作用真正发挥]</ref>
*2010年3月24日，海上观景平台由于施工不慎发生火灾，幸而没有造成人员伤亡。<ref>[http://hz.cheshi.com/zhejiang/201003/197944.shtml 杭州湾跨海大桥海上观景平台着火 27名被困人员获救]</ref>
*在公路桥落成之后，[[杭州湾铁路大桥|跨杭州湾铁路]]也已纳入议事日程。<ref>[http://nb.house.sina.com.cn/news/2009-12-31/15317097.html 砸下近百亿甬城铁路两小时飞沪杭宁]</ref>这条铁路大桥的走线将大致平行于目前的杭州湾跨海大桥。
