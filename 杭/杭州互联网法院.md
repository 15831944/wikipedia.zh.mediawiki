'''杭州互联网法院'''，牌子挂在[[杭州铁路运输法院|杭州铁路运输法院]]，是[[中华人民共和国|中华人民共和国]]乃至世界上首家互联网法院<ref name=ifeng>{{Cite web |url =http://tech.ifeng.com/a/20170818/44665264_0.shtml|title = 杭州互联网法院今早揭牌，全世界第一家互联网法院 |author =  |date = 2017-08-18 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

==沿革==
2015年4月，[[浙江省高级人民法院|浙江省高级人民法院]]确定由[[杭州市西湖区人民法院|杭州市西湖区人民法院]]、[[杭州市滨江区人民法院|杭州市滨江区人民法院]]、[[杭州市余杭区人民法院|杭州市余杭区人民法院]]这三家基层人民法院以及浙江省杭州市中级人民法院开展[[电子商务|电子商务]]网上法庭试点，其中杭州市西湖区人民法院审理[[互联网金融|互联网金融]]纠纷和电子督促案件，杭州市滨江区人民法院审理[[知识产权|知识产权]]纠纷案件，杭州市余杭区人民法院审理网络交易纠纷案件，浙江省杭州市中级人民法院审理前述纠纷的上诉案件。截至2017年4月，电子商务网上法庭已处理涉[[网购|网购]]、网络支付等纠纷将近2万件。[[浙江省|浙江省]]全省已有15家法院加入电子商务网上法庭平台。电子商务网上法庭已实现起诉、调解、立案（管辖异议）、举证、质证、开庭、判决等诉讼环节的全程网络化<ref name=zhw>{{Cite web |url =http://news.china.com/news100/11038989/20170815/31093899_all.html  |title = 国内首家互联网法院搅动一池春水 |author =  |date = 2017-08-15 |publisher = 中华网 |language =  |accessdate =  |quote =  }}</ref>。

2017年1月，浙江省高级人民法院院长[[陈国猛|陈国猛]]公开表示，浙江法院将推动设立杭州网络法院，促进网络法治的健全发展。杭州网络法院是基于杭州法院系统的电子商务网上法庭而建<ref name=weishm/>。

2017年全国[[两会|两会]]期间，[[全国政协委员|全国政协委员]]、[[中国民主促进会|民进]]浙江省委副主委[[赵光育|赵光育]]建议，应设立中国（杭州）-{互}-联网法院。他在提出建议时表示，近年来在杭州的电子商务平台自身处理的纠纷数以万计，杭州市各基层人民法院受理的电子商务案件已经自2013年的600多件增至2016年的上万件<ref name=weishm>{{Cite web |url =http://news.163.com/17/0626/19/CNSLQJR0000187VE.html  |title = 中央同意设立杭州互联网法院，为什么是杭州？ |author =  |date = 2017-06-26 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。

另一方面，自2016年起，杭州铁路运输法院开展涉及知识产权、互联网案件及部分行政案件跨区域管辖和审判的转型试点。2017年4月，[[最高人民法院|最高人民法院]]批复同意由杭州铁路运输法院集中管辖杭州地区的5类涉网案件。2017年5月1日至6月20日，杭州铁路运输法院共收到涉网案件申请1896件，其中正式立案的有1446件<ref name=zhw/>。

2017年6月26日，[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军事委员会主席|中央军委主席]]、[[中央全面深化改革领导小组|中央全面深化改革领导小组]]组长[[习近平|习近平]]主持召开中央全面深化改革领导小组第三十六次会议并发表讲话，会议审议通过了《关于设立杭州互联网法院的方案》。会议称，设立该法院“是司法主动适应-{互}-联网发展大趋势的一项重大制度创新。要按照依法有序、积极稳妥、遵循司法规律、满足群众需求的要求，探索涉网案件诉讼规则，完善审理机制，提升审判效能，为维护网-{}-络安全、化解涉网纠纷、促进-{互}-联网和经济社会深度融合等提供司法保障。”<ref>{{Cite web |url = http://www.chinacourt.org/article/detail/2017/06/id/2903251.shtml |title = 习近平主持召开中央全面深化改革领导小组第三十六次会议 |author =  |date = 2017-06-26  |publisher =  中国法院网|language =  |accessdate =  |quote =  }}</ref>

2017年8月16日，杭州市第十三届人民代表大会常务委员会第四次会议任命了杭州互联网法院院长、副院长、审判员<ref name=duqian>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1765200 |title =杭州市人大常委会会议任命杜前为杭州互联网法院院长  |author =  |date = 2017-08-16 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。2017年8月18日，杭州互联网法院正式揭牌<ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1766889|title = 中国首家互联网法院杭州揭牌：探索涉网案件诉讼规则 |author =  |date = 2017-08-18 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>，牌子挂在[[杭州铁路运输法院|杭州铁路运输法院]]<ref>{{Cite web |url =  https://www.sohu.com/a/165702504_733937|title = 中国首家互联网法院揭牌，10个问答让您成为杭州互联网法院知识达人！ |author =  |date = 2017-08-18 |publisher =登封普法|language =  |accessdate =  |quote =  }}</ref>。

==法院特色==
由于该院为互联网法院，所以在该院受理的所有案件，全部在互联网完成。该院设有“网上诉讼平台”，当事人在该平台使用手机号码注册账号后，可通过该平台递交起诉状和相应的证据材料，勾选所需依据的法律条文，之后系统会自动提取电子商务平台的当事人身份信息、网上交易过程及各类表单数据，全程起诉立案的时间为5分钟左右。同时，系统会自动计算诉讼费用，当事人可通过[[支付宝|支付宝]]或网银缴纳。随后将自动向被告送达诉讼信息，被告可登陆平台查看提出抗辩。该院平均开庭时间为25分钟，平均审限32天，开庭案件实现100%在线庭审、在线判决、网上送达，体现了以互联网方式审理互联网案件的效率、公开、便民优势。截至2017年8月15日，该院共立案2605件，审理结案1444件，全部为“在线审理”<ref name=fy>{{Cite web |url =  http://news.cnhubei.com/xw/gn/201708/t3889855_1.shtml|title = 首家互联网法院揭秘：平均开庭时间仅25分钟 打官司像“网购”一样便利 |author =  |date = 2017-08-18 |publisher =中国法院网|language =  |accessdate =  |quote =  }}</ref>。

该院的法庭与普通法院法庭相比，不设原告、被告席和书记员，仅设法官（审判员），且采用语音识别系统进行记录。法官面前设有一块大屏幕，在线隔空审理案件，与多人视频模式相似。而屏幕上显示原告与被告所处的地理位置<ref>{{Cite web |url =  http://news.china.com/socialgd/10000169/20170819/31131309.html|title =中国首家互联网法院：5分钟提交诉状 打官司像“网购” |author =  |date = 2017-08-19 |publisher = 中国青年网|language =  |accessdate =  |quote =  }}</ref>。庭审期间，法官可以随时在屏幕上点击查看证据材料等各类诉讼文件，系统还会自动梳理出类似案例供法官参考<ref name=jm/>。

==管辖==
根据[[最高人民法院|最高人民法院]]的实施方案，杭州互联网法院集中管辖杭州市辖区内基层人民法院有管辖权的下列涉互联网一审案件<ref name=fy/>：
* 1、互联网购物、服务、小额金融借款等合同纠纷；
* 2、互联网著作权权属、侵权纠纷；
* 3、利用互联网侵害他人人格权纠纷；
* 4、互联网购物产品责任侵权纠纷；
* 5、互联网域名纠纷；
* 6、因互联网行政管理引发的行政纠纷；
* 7、上级人民法院指定杭州互联网法院管辖其他涉互联网民事、行政案件。

未来，杭州互联网法院还将逐步拓展案件范围<ref name=jm>{{Cite web |url =  http://www.jiemian.com/article/1561370.html|title =中国首家互联网法院挂牌：哪些案件网上审？网上怎么审？ |author =  |date = 2017-08-19 |publisher =界面|language =  |accessdate =  |quote =  }}</ref>。
