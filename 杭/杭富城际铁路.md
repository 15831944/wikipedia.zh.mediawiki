'''杭富城际铁路'''，亦称'''[[杭州地铁6号线|杭州地铁6号线]]富阳段'''<ref>{{cite web|url=http://zjnews.zjol.com.cn/zjnews/hznews/201710/t20171027_5463959.shtml|title=杭州至富阳城际铁路有调整 计划和地铁6号线同步建成通车|publisher=都市快报}}</ref>，是中国[[浙江省|浙江省]][[杭州市|杭州市]]境内的一条在建地铁线路，起于[[西湖区_(杭州市)|西湖区]][[美院象山站|美院象山站]]，止于[[富阳区|富阳区]][[桂花西路站|桂花西路站]]，全长23.15公里，全部为地下线，设置11个地下车站，设计行车速度100公里/小时。

工程建设期至2020年，与[[杭州地铁6号线|6号线]]同步建成投用。城际铁路杭富线建成后将促进富阳区公共交通方式的多样化，推动富阳区融入杭州主城区。

==简介==
线路起于[[杭州地铁6号线|杭州地铁6号线]][[美院象山站|美院象山站]]，与其为叠岛换乘，出站后与6号线一同折向西南，并在象山路与320国道交叉口北侧与6号线交叉（6号线下穿杭富线），随后线路折回320国道南侧主路，于杭七中、浙江音乐学院前设[[音乐学院站|音乐学院站]]，于中村路交叉口处设[[中村站_(杭州地铁)|中村站]]。出站后线路继续沿国道南侧主路向西地下敷设，在美和院居住片区前设[[野生动物园东站|野生动物园东站]]。于科创路东侧设[[银湖站_(杭州地铁)|银湖站]]，于虎啸杏村设[[虎啸杏站|虎啸杏站]]，于九龙大道交叉口东北侧设[[受降站|受降站]]，跨过320国道后沿新320国道外北侧走行，于高渠路与新320国道交叉口东侧设[[富阳客运中心站_(杭州地铁)|富阳客运中心站]]。在高桥西路与金桥北路交叉口处设[[高桥站_(杭州地铁)|高桥站]]，于文心苑居住小区前设[[新桥站_(杭州地铁)|新桥站]]，于公园西路与金桥北路交叉口处设[[公望街站|公望街站]]，于五岔路口东方茂购物中心前设终点站[[桂花西路站|桂花西路站]]。桂花西路站向南预留远期延伸的工程条件。

线路全长约23.50公里，设站11座，平均站间距约2.3公里。线路在西湖区境内长度为6.76公里，全部为地下线，设站3座；富阳区境内长度为16.74公里，全部为地下线，设站8座。工程在富阳境内富阳客运中心站以北宋家塘村设车辆段一座；控制中心与6号线合建，利用七堡控制中心；供电系统采用110/35kV两级电压集中供电方式。工程在车辆段选址范围内新建一座主变电所，同时利用6号线彩虹变作为备用电源。<ref>{{Cite web|url = http://www.zjdpc.gov.cn/art/2016/11/4/art_808_1717507.html|title = 省发展改革委关于调整杭州至富阳城际铁路工程初步设计的批复|accessdate = |author =|date = 2017-10-24 |publisher = }}</ref>

== 历史 ==
本线原规划为独立线路，起点为枫桦西路站。在2016年的规划调整中，本线成为6号线主线的一部分，而原6号线双浦段3站则改为通过渡线连接，并规划在未来拆解。

*2016年2月20日，位于银湖站至受降站之间的试验段开工。试验段全长1.006公里，主要以高架方式沿320国道路中设置。<ref>{{cite web |url=http://news-hzrb.hangzhou.com.cn/system/2016/02/19/012965160.shtml |language= |title=杭州至富阳城际铁路工程正式开工 |publisher=杭报在线 |date=2016年2月19日 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20170114055542/http://news-hzrb.hangzhou.com.cn/system/2016/02/19/012965160.shtml |archivedate=2017年1月14日 }}</ref>
*2017年10月24日，《省发展改革委关于调整杭州至富阳城际铁路工程初步设计的批复》中取消文创园站至汽车北站站的高架区间，改为地下线<ref>{{Cite web|url = http://www.zjdpc.gov.cn/art/2016/11/4/art_808_1717507.html
|title = 省发展改革委关于调整杭州至富阳城际铁路工程初步设计的批复|accessdate = |author =|date = 2017-10-24 |publisher = }}</ref>。不久后试验段已建成的桥墩被全部拆除。
*2018年12月27日，本线车站正式名获批。<ref>{{cite web|url=https://mp.weixin.qq.com/s/Pa3_TYsURfF5-VVCv-Q50g|title=最新，杭州至富阳城际铁路站名获批！|publisher=杭港地铁|date=2018-12-27}}</ref>
