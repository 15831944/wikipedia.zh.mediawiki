'''圣母无原罪主教座堂（杭州）'''是位于中国[[杭州|杭州]]市中心[[天水街道|天水街道]][[中山北路|中山北路]]415号（临近[[武林广场|武林广场]]）的一座历史悠久的天主教堂，[[天主教杭州教区|天主教杭州教区]]的[[主教座堂|主教座堂]]，也是杭州市区目前唯一开放的天主教堂。

== 历史 ==
明[[万历|万历]]年间，工部员外郎、杭州籍人士[[李之藻|李之藻]]在北京皈依天主教，并由[[利玛窦|利玛窦]]施行[[洗礼|洗礼]]。万历三十九年（1611年），李之藻因父亲去世回乡守孝，经过南京时带上意大利耶稣会士[[郭居静|郭居静]]、[[金尼阁|金尼阁]]二神父，他们在杭州[[李之藻|李之藻]]的家中举行了第一台[[弥撒|弥撒]]，这一天（5月8日）被称为杭州天主教开教日。[[杨廷筠|杨廷筠]]是在杭州最早的皈依者之一，曾任监察御史，原是佛教徒，信仰天主教后立刻撤下了家中供奉的佛像。1616年，在南京教难期间，许多天主教徒逃往杭州，在[[李之藻|李之藻]]和[[杨廷筠|杨廷筠]]的家中避难。1627年（天启七年），弥额尔（[[杨廷筠|杨廷筠]]）出资在观巷购地建造杭州第一座天主教堂（今教堂略西处）<ref>{{cite web |url=http://www.cczj.org/company.asp?id=195 |title=存档副本 |accessdate=2013-09-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20160303200559/http://www.cczj.org/company.asp?id=195 |archivedate=2016-03-03 }}</ref><ref>《浙江省宗教志》资料汇编（一），1993年由《浙江省宗教志》编辑部出版</ref><ref>《浙江天主教》，郭慕天编著，1998年由《浙江省宗教志》编辑部出版</ref><ref>《宗教文化》，陈村富主编，1997年由东方出版社出版</ref><ref>《杭州市志》</ref>。李之藻、杨廷筠与[[徐光启|徐光启]]同被称为[[聖教三柱石|中国天主教三大柱石]]，扶持天主教最为热心，使得杭州成为明末天主教传教中心之一。

1659年（顺治十六年）－1661年，意大利传教士[[卫匡国|卫匡国]]在今址建造了圣母无原罪堂。

1730年，清朝[[雍正|雍正]]皇帝下令禁止天主教传播，教堂被改为寺庙。

1958年，杭州市天主教爱国会自选自圣了[[吴国焕|吴国焕]]神父为杭州教区主教。

1966年[[文革|文革]]期间，教堂被关闭，改作它用。

1982年12月12日，杭州圣母无原罪大堂举行复堂庆祝大礼弥撒。2011年1月，教堂成为[[浙江省文物保护单位|浙江省文物保护单位]]。

== 弥撒 ==
每周日早晨举行中文弥撒，周六晚举行英文弥撒。该教堂拥有2个唱诗班。
