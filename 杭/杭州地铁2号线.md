'''杭州地铁2号线'''，标识色为<span style="font-weight: bold; color: {{杭州地铁颜色|2}};">丹桂橙</span>，[[余杭|余杭]][[良渚站|良渚站]]至[[萧山|萧山]][[朝阳站_(杭州地铁)|朝阳站]]，大致呈西北-东南走向，线路全长48公里设33个站点，全部为地下线、地下车站。分为一期东南段13站，一期西北段11站，以及二期9站三个阶段建设。一期东南段于2014年11月24日开通运营。<ref >{{cite web|url=http://z.hangzhou.com.cn/2014/subway2/content/2014-11/24/content_5539081.htm | language= | title=杭州地铁2号线东南段试运营发出首列车 黑色站名表将被换下 |publisher=杭州网 |date=2014-11-24 |accessdate= }}</ref>2017年12月27日，线路全线通车。<ref name=":0">{{Cite web|url=http://hangzhou.zjol.com.cn/system/2017/12/26/021638551.shtml|title=重磅！杭州地铁2号线明天全线开通 从萧山直达良渚-杭州新闻|accessdate=2018-01-20|work=hangzhou.zjol.com.cn}}</ref>同时，杭州地铁移动支付过闸项目开始试运行，支持旅客以[[支付宝|支付宝]]扫码、[[中国银联|银联]]闪付方式快速通过闸机，无须排队购票。<ref>{{Cite web|url=http://www.huaxia.com/ztlx/zjxw/2017/12/5586496.html|title=杭州地铁2号线全线开通 今天下班你能坐地铁回家|accessdate=2018-01-20|work=www.huaxia.com}}</ref>

== 概况 ==
一期工程全长30.5公里，因当时[[杭州地铁1号线|1号线]]仍在施工，杭州市政府部门担心同时进行两条地铁工程可能导致道路交通瘫痪，因此将2号线一期路段分为东南段工程（朝阳站至钱江路站）和西北段工程（钱江路站至丰潭路站）分开建设<ref>{{cite news|title=从钱江路到丰潭路 杭州地铁2号线西北段8月开工|url=http://zjnews.zjol.com.cn/05zjnews/system/2011/05/12/017513047.shtml|date=2011-05-12|publisher=浙江新闻网}}</ref>。东南段长约18.2公里，于2008年9月28日先期开工，先开工建设一路、建设三路两对相邻的停靠站，东南段于2014年11月建成通车<ref >{{cite web|url=http://news.xinmin.cn/shehui/2014/06/06/24487022.html | language= | title=杭州地铁2号线东南段11月试运营 4号线首通段明年年初通车 |publisher=浙江在线 |date=2014-06-06 |accessdate= }}</ref>，11月18日起开始免费试乘5天，平均每天有3万多人次试乘。其中[[钱江世纪城站|钱江世纪城站]]因6号线施工和客流不足暂缓开通（列车越站通过）。行车间隔为8分45秒，平均停站40秒，[[钱江路站|钱江路站]]至[[朝阳站_(杭州地铁)|朝阳站]]单程行车时间约31分钟。<ref name="初定"/>

11月24日开通一条公交专线，专门连接地铁2号线[[钱江路站|钱江路站]]与[[杭州地铁1号线|杭州地铁1号线]]的[[近江站|近江站]]，该公交中间不设站点全程约15分钟，对接驳换乘的乘客免费。该接驳专线在地铁4号线首通段于2015年2月2日9:30开通之时停运。<ref >{{cite web|url=http://hznews.hangzhou.com.cn/chengshi/content/2014-11/17/content_5529190.htm | language= | title=地铁2号线东南段明起免费试乘5天 |publisher=杭州网 |date=2014-11-17 |accessdate= }}</ref><ref name="初定">{{cite web|url=http://zj.sina.com.cn/news/m/2014-11-20/detail-icczmvun0078164.shtml | language= | title=杭州地铁2号线与1号线接驳方案初定 |publisher=杭州日报 |date=2014-11-20 |accessdate= }}</ref>

一期西北段和二期计划于2017年建成。一期西北段中的庆春广场站至古翠路站部分计划于2017年6月开通<ref>{{cite web|url=http://downc.zmctc.com/zjshy/JSGCZtbMis2_ZJS/Pages/ZBGG/PriewGongGao.aspx?ProjectNo=2008-152-82&chriid=80321677 | language= | title=浙江省重点建设工程招标公告 |publisher=浙江重大工程交易网 |date=2016-03-02 |accessdate= 位于补充文件中}}</ref>，但由于工期延误，最终官方确定于7月3日下午15:30开通。

二期和三期工程（丰潭路至良渚）于2017年12月27日17:00开通。<ref name=":0" />

== 设计 ==
杭州地铁2号线主题设计为“杭城烟雨”。设计有氤氲云水感觉的湖蓝、浅绿色系。按照“一线一景类型化”的设计理念进行，每个站点各有风味。整体效果大气现代，细节处又优雅细腻。<ref>{{cite web |title=杭州地铁2号线主题为“杭城烟雨”标识色是橙色-建国路,天花板,学院路,赤橙黄绿青蓝紫,一脉相承-浙江在线-杭州新闻 |url=http://hangzhou.zjol.com.cn/hangzhou/system/2013/03/13/019205040.shtml |website=浙江在线}}</ref>但在最终车站建设中，大部分车站都取消了原有设计的颜色，改用黑白灰样式，并在部分车站改用全新设计。非特色站装修风格基本统一，只不过在颜色和局部细节上有不同。<ref>{{cite web |title=地铁2号线装修主题为"烟雨杭州" 能找到画或诗_大浙网_腾讯网 |url=http://zj.qq.com/a/20130313/000011.htm |website=腾讯大浙网}}</ref>另外在站台上还设置有少量水墨画风格的石凳供乘客临时休息。<ref>{{cite web |title=各站点出入口在哪里？怎么去最方便？_地铁2号线西北段开通_杭州网 |url=http://z.hangzhou.com.cn/gundong/2017dteh/content/2017-06/30/content_6591915.htm |website=杭州网}}</ref>

== 车辆段 ==

2号线拥有两个车辆基地，分别是[[蜀山车辆段|蜀山车辆段]]和[[双桥停车场|双桥停车场]]。
蜀山车辆段在[[朝阳站_(杭州地铁)|朝阳站]]以南，用于2号线列车的停放和各类养护，同时承担4号线列车的高级养护工作<ref>{{cite web|title=杭州地铁4号线二期工程先行段环境影响报告书全本公示|url=http://www.hzmetro.com/EptionUload/files/file/20180428/20180428090706_4052.pdf|website=杭州地铁}}</ref>。
双桥停车场位于[[良渚站|良渚站]]南侧，用于2号线列车的停放和普通养护。

== 历史 ==
[[File:20160925杭州地铁2号线站前折返进入钱江路站第一视角.gif|thumb]]实行站前折返]]
[[File:201608_Gouzhuang_Station_under_construction.jpg|thumb]]建设中（2016年8月）]]
2号线最初规划全长约48.817km，其中地下线37.567km，高架线10.88km，地上地下过渡段0.37km。共设车站35座，其中地下站29座，高架站6座。<ref>{{cite web|title=地铁时代给杭州带来什么|url=http://zzhz.zjol.com.cn/05zzhz/system/2008/09/29/009985997.shtml|website=新浪}}</ref><ref>{{cite web|title=杭州地铁2号线工程简介|url=http://www.ditiezu.com/thread-67921-1-1.html}}</ref>

*2008年7月14日，省发改委批复同意杭州地铁2号线一期工程初步设计。
*2008年9月28日，2号线东南段开工。

== 票价 ==

2号线东南段计价方式和1号线一样，按里程分段计价。算下来，长18公里的东南段的最高票价为5元。在免票和优惠购票政策方面，2号线东南段和1号线基本一样，普通乘客购买储值卡刷卡坐地铁享受票价9.1折优惠。至2017年底2号线全部建成，从朝阳站到良渚站全长48公里需时80分钟票价为9元。<ref >{{cite web|url=http://hznews.hangzhou.com.cn/chengshi/content/2014-10/08/content_5472848.htm | language= | title=杭州地铁2号线东南段票价出炉 |publisher=杭州网 |date=2014-10-08 |accessdate= }}</ref>

使用银行卡或移动终端(如Apple Pay、Samsung Pay、Huawei Pay等)进出闸机的乘客，在进站时会收到按最高票价完成的预授权金额，出站后按正常票价扣款。如果未能成功刷卡出站或出站记录与进站记录不匹配，则会强制扣除最高票价的金额或者卡内余额。对此情况，持卡人可于7日内向车站客服中心申诉。<ref name=":0" />
