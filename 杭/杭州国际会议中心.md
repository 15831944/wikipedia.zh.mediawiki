'''杭州国际会议中心'''（Hangzhou International Conference Center）位于[[杭州市|杭州市]][[钱江新城|钱江新城]]南端[[钱塘江|钱塘江]]畔，比邻[[杭州大剧院|杭州大剧院]]。

== 概况 ==
会议中心以[[会议|会议]]、超[[五星级|五星级]][[酒店|酒店]]为主要功能，提供会议、演出、演示、展示、住宿、休闲、餐饮等国际标准的综合性多功能会议和商务服务。总建筑面积13万平方米<ref>{{cite web |url = http://www.hangzhou.com.cn/20080923/ca1577973.htm | language= | title=杭州国际会议中心 |publisher=杭州网 |date=2008年9月26日 |accessdate= }}</ref>，其中会议部分2.1万平方米，酒店4.8万平方米，其余为后勤服务设施。

建筑总高度85米，主体建筑由12米的椭圆形裙房部分和直径85米的上部球体组成。中心外墙采用金色钛板金色玻璃幕墙，俗称“大金球”。<ref>{{cite web |title=杭州新地标①丨钱塘江边“大金球” 你曾与它同框吗 |url=https://zj.zjol.com.cn/news.html?id=827297 |website=浙江在线}}</ref>

== 结构功能 ==
* 裙房会议区：两层，设有可灵活分割的2000平米宴会厅、1000座国际会议厅、电视电话会议厅、圆桌会议厅、多功能新闻发布厅及若干中小会议室，包括媒体传播中心与商务中心等。首层与地下一层的'''[[波浪文化城|波浪文化城]]购物休闲广场'''可通过大台阶直接连通。
* 酒店区：会议中心的'''杭州[[洲际酒店|洲际酒店]]'''（InterContinental Hangzhou）于2010年12月27日开业，由[[钱江新城|钱江新城]]投资建设，国际著名的酒店管理公司——[[洲际酒店集团|洲际酒店集团]]负责运营管理，总投资15亿元，总建筑面积13万平方米，共19层高85米<ref>{{cite web| url=http://www.hangzhou.gov.cn/main/zwdt/tpxw/T340151.shtml| language=| title=杭州国际会议中心（洲际酒店）正式开业| website=“中国杭州”政府门户网站| date=2010-12-27| accessdate=| deadurl=yes| archiveurl=https://web.archive.org/web/20110505202955/http://www.hangzhou.gov.cn/main/zwdt/tpxw/T340151.shtml| archivedate=2011-05-05}}</ref>。大堂及餐饮位于球体下部，中区325间客房围绕着独特的穹状中厅设置，向大剧院一侧开敞的球面墙幕将阳光和广场上的景观引入。顶部为空中健身中心，包括景观游泳池和SPA。
* 地下一层为酒店后勤辅助配套设施和设备用房，地下二层为450个泊位的停车库。

== 杭州其他大型会展场所 ==
