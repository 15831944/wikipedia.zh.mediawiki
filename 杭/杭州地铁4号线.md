'''杭州地铁4号线'''，标识色为<span style="font-weight: bold; color: {{杭州地铁颜色|4}};">香樟绿</span>，  大致呈东北-西南走向，北起[[彭埠站|彭埠站]]，南抵[[浦沿站|浦沿站]]，贯穿整个[[钱江新城|钱江新城]]（江北区块）。首通段（东北段）于2015年2月2日投入试运营<ref name="open-operation">{{cite web|url=http://www.hangzhou.gov.cn/main/zwdt/bzbd/szcf/T511697.shtml |language= |title=地铁4号线首通段2月2日开通试运营！ |publisher=中国杭州 |date=2015-01-31 |accessdate=2015-02-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20150204075503/http://www.hangzhou.gov.cn/main/zwdt/bzbd/szcf/T511697.shtml |archivedate=2015-02-04 }}</ref>，西南段2018年1月9日开通。<ref name="开工">{{cite web|url=http://hznews.hangzhou.com.cn/chengshi/content/2013-11/14/content_4964351.htm | language= | title=杭州地铁4号线年内开工 建成后将串联1号线和2号线东南段 |publisher=杭州网 |date=2013-11-14 |accessdate= }}</ref>二期工程已纳入杭州地铁三期规划。

== 概况 ==
一期工程全长20.8公里，分为首通段工程（彭埠站至近江站）和西南段工程（近江站至浦沿站）。首通段长约9.6公里，于2008年开工，2015年2月2日投入试运营<ref name="open-operation" />。一期西南段长11.2公里有8个站，于2018年1月9日通车。

一期工程北起彭埠站，在火车东站和彭埠站与[[杭州地铁1号线|1号线]]形成双向同站台换乘，向南经新塘路、沿富春江路到[[钱江新城|钱江新城]]市中心与[[杭州地铁2号线|2号线]]形成单向同向同站台换乘，南抵[[浦沿站|浦沿站]]。该线是轨道交通的次干线，连通了1、2号线路，有多个换乘机会，便于把主干线上的客流运送到更远的地区，并加强对钱江新城的服务，它也连接了火车东站和[[杭州汽车南站|汽车南站]]等对外交通点。<ref>{{cite web|url=http://www.0571dt.cn/?page_id=198 | language= | title=杭州地铁4号线 |publisher=杭州地铁生活门户网站 |date= |accessdate=2012年10月 }}</ref>

二期工程起自彭埠站，向北转向明石路，沿明石路向北敷设，穿过德胜快速路后，继续沿明石路向北敷设，在明石路与笕石路交叉口，转向西沿笕石路敷设，下穿既有沪杭铁路后，沿杨家路、长大屋路向西敷设，之后沿东新路向北敷设，下穿上塘西北，沿着吴家角港向西北敷设，在杭钢地块，线路转向西，沿平炼路敷设，下穿运河后，线路沿勾运路向西敷设，在勾运路与西塘路交叉口，线路转向西南，沿金渡北路向西敷设至线路终点。线路全长23.4km，全部为地下线，共设车站14座。<ref>{{cite web|url=http://www.hzmetro.com/notice_show.aspx?ID=6194 | language= | title=杭州市轨道交通三期建设规划环评公众意见征询 |publisher=杭州地铁 |date= |accessdate=2015年12月 }}</ref>

按里程分段计价来算首通段最高票价为4元。4号线首班车发车时间为6:30，末班车发车时间为22:30，全天运行16个小时。运营初期，首通段列车行车间隔为8分30秒，彭埠站至近江站单程行车时间约21分钟。<ref >{{cite web|url=http://z.hangzhou.com.cn/2015/dt4h/content/2015-01/31/content_5634571.htm | language= | title=下周一可以坐上地铁4号线啦！首通段最高票价4元 可站内换乘 |publisher=杭州网 |date=2015年1月31日 |accessdate= }}</ref>

== 车辆段 ==

4号线拥有1个车辆基地，为[[七堡车辆段|七堡车辆段]]。
七堡车辆段在[[彭埠站|彭埠站]]和[[九和路站|九和路站]]之间，主要用于1号线、4号线列车的停放和养护。
4号线列车的高级养护位于2号线的[[蜀山车辆段|蜀山车辆段]]<ref>{{cite web|title=杭州地铁4号线二期工程先行段环境影响报告书全本公示|url=http://www.hzmetro.com/EptionUload/files/file/20180428/20180428090706_4052.pdf|website=杭州地铁}}</ref>。

== 事故 ==
*2014年7月31日上午，江锦路站至市民中心站区间右线盾构机进洞时（位于新业桥南侧河道下方（盾构区间从河底穿过）），现场发现洞门左上方出现渗漏水，新塘河河水水位增高，到上午11时，围堰积水突然消失进入市民中心站，随之围堰倒塌，新塘河河水灌入地铁车站导致市民中心站厅、站台被淹，并造成新业桥成为危桥。未造成人员伤亡。
*2015年8月28日上午9：33分，四号线南星桥站在地下连续墙施工中不慎挖破燃气管道（直径300mm）导致泄漏。事发后，施工单位通知消防、燃气公司。市地铁集团领导赶赴现场指挥抢险。9:39 ，消防到达现场疏散周边人员；9:55，燃气公司开始抢险，燃气管道阀门及时被关闭。16:20，受损燃气管道焊接作业已完成，16：50，相关燃气管道已恢复通气，此事件对沿线居民及其他用户供气的影响已全部消除。<ref>{{Cite web|url = http://hznews.hangzhou.com.cn/chengshi/content/2015-08/29/content_5901742_2.htm|title = 杭州地铁4号线工地天然气管道被挖破 城市管网为何会“受伤”？|accessdate = |author = |date = |publisher = }}</ref>事件未造成施工人员伤亡。
*2015年11月28日晚上8点左右，水澄桥站在建工地外围复兴路发生塌陷，复兴路交通中断。事故没有造成人员伤亡，但周边71户居民受影响连夜疏散，2000余户家庭管道煤气中断，塌陷的主要原因初步判断是地下连续墙渗水导致水土流失。<ref>{{Cite web|url = http://zjnews.zjol.com.cn/system/2015/11/29/020931638.shtml|title = 杭州地铁4号线水澄桥站昨晚揪住了一城人的心|accessdate = 2015-12-01|author = |date = |publisher = }}</ref>
*2016年7月8日22：30许，地铁4号线南段中医药大学站南基坑施工时，发生基坑土体突涌。现场施工人员10名，其中2人成功自救，8人不同程度困于淤泥当中。事发后，施工单位第一时间启动应急预案，组织人员进行救援。市地铁集团接报后，迅速赶赴现场指挥。市交警、特警、消防，以及滨江区等有关单位人员立即赶赴现场配合抢险。此事故最终造成4人死亡，4人受伤。事故未对地铁基坑安全造成影响。<ref>{{cite web|url=http://weibo.com/2219143801/DDXXFjr6q?type=repost | language= | title=201607080800突发事故通报 |publisher= 杭州地铁官方  |date= |accessdate= }}</ref><ref>{{cite web|url=http://weibo.com/2219143801/DE1Lj3Eo8?ref=home&rid=4_0_1_2598668098530849299&type=comment | language= | title=201607081800突发事故通报 |publisher= 杭州地铁官方  |date= |accessdate= }}</ref>

== 未来发展 ==
4号线二期工程计15站24.1公里 <ref>[http://www.zjdpc.gov.cn/art/2017/12/14/art_808_1738284.html 浙发改设计（2017）112号]</ref>
