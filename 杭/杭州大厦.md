'''杭州大厦'''（Hangzhou Tower），又称'''杭州大厦购物城'''，'''杭州大厦有限公司'''，1988年开业，是[[杭州市|杭州市]]市中心的一家经营高档奢侈品品牌与时尚流行商品的精品[[百货公司|百货公司]]与大型商场，集购物、美食、休闲娱乐、会议展览、住宿、办公写字楼于一体<ref name="简介">{{cite web |url=http://www.hangzhoutower.com/2011/about.asp |language= |title=购物城简介 |publisher=杭州大厦购物城 |date= |accessdate=2012年10月 |deadurl=yes |archiveurl=https://web.archive.org/web/20121025144616/http://www.hangzhoutower.com/2011/about.asp |archivedate=2012-10-25 }}</ref>。

== 简介 ==
它座落于杭州市最繁华地段——[[武林广场|武林广场]]西侧，分A座、B座、C座坤和店、D座华浙店四座大楼，横跨[[环城北路|环城北路]]。总建筑面积近14万平米<ref name= "简介"/>，共经营5万多种国内外高档名牌商品。

“生活·购物·享受”为其时尚经营理念，杭州大厦的效益连续多年位列中国大型零售企业（单店）前10位，创利已进入全国商界前三名，零售规模为[[浙江省|浙江省]]商业同行第一位<ref name="中心">{{cite web|url=http://mall.ppsj.com.cn/676D5DDE592753A6/ | language= | title=杭州大厦购物中心 |publisher=ppsj.com |date= |accessdate=2012年10月 }}</ref>。被誉为“杭州市窗口的一张亮丽名片”。<ref name="">{{cite web |url=http://life.dahangzhou.com/367.html |language= |title=杭州大厦购物中心 |publisher=大杭州 |date= |accessdate=2012年10月 |deadurl=yes |archiveurl=https://web.archive.org/web/20121002213245/http://life.dahangzhou.com/367.html |archivedate=2012-10-02 }}</ref>

杭州大厦已从单店品牌经营逐步走向规模化[[连锁|连锁]]经营，先后输出品牌管理杭州大厦－[[湖州|湖州]]凤凰购物中心、杭州大厦－[[西湖时代广场|西湖时代广场]]，以及杭州大厦－元华购物中心共三家大型商场，经营规模已位居浙江省第一名。<ref name= "中心"/>

在杭州举办[[2016年二十国集团峰会|2016年二十国集团峰会]]期间，包括[[加拿大总理|加拿大总理]]在内的多名与会国家领导人和代表团光临杭州大厦。
