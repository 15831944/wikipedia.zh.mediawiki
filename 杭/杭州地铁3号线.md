'''杭州地铁3号线'''长52.2公里，大致呈西南-东北走向，西起小和山和余杭中央公园，在[[百家园路站|百家园路站]]并线、沿[[天目山路|天目山路]]，经[[杭州汽车西站|汽车西站]]后转至西溪路、黄龙体育中心、曙光路至[[武林广场|武林广场]]、[[西湖文化广场|西湖文化广场]]与[[杭州地铁1号线|1号线]]形成双向同站台换乘。再向北转向河东路到长乐路（规划），后往东北方向转至沈半路、白石北路、新天地街、竹清路、华丰路、同协路、大农港路、天都路，终点为[[余杭区|余杭区]][[星桥大道站|星桥路]]，共设车站33座。远期经过藕花洲大街与9号线相交，止于临平红丰路站。<ref>{{Cite web|url = http://www.hzmetro.com/notice_show.aspx?ID=6194#midc|title = 杭州市轨道交通三期建设规划环评公众意见征询|accessdate = |author = |date = |publisher = }}</ref>

3号线经过[[西湖区|西湖区]]、[[上城区|上城区]]、[[拱墅区|拱墅区]]和[[余杭区|余杭区]]，位于市主要客流走廊之上，是轨道系统的骨干线。目的是加强市西面、东北面与市中心的联系。<ref>{{cite web|url=http://www.0571dt.cn/?page_id=196 | language= | title=杭州地铁3号线 |publisher=杭州地铁生活门户网站 |date= |accessdate=2012年10月 }}</ref>

预计2017年开工<ref>[http://hznews.hangzhou.com.cn/chengshi/content/2015-08/13/content_5881552.htm 杭州地铁3号线2017年开工 文一路地下通道2017年建成]</ref>。

== 走向 ==
3号线一期主线起点文一西路站，终点余杭区星桥路站，线路全长43.8km，平均站间距1.53km，全为地下线。设车站29座，其中换乘站9座，分别与杭州地铁1号线(2座车站换乘)、2号线、3号线小和山支线、4号线、5号线、10号线、杭临线换乘，另有2座车站（高教路、松木场）预留与远期线路换乘条件。3号线主线由文一西路站起起依次沿东西大道——水乡北路——五常大道——天目山路——西溪路——曙光路——体育场路——中山北路——河东路——长乐路——大关路——沈半路——白石北路——新天地街——竹清路——华丰路——同协路——大农港路——天都路敷设，终点站临平。线路依次经过了余杭区、西湖区、下城
区、拱墅区、江干区等5个城区。

3号线一期支线起点西湖区小和山站，终点百家园路站，线路长8.4km，平均站间距1.98km，全为地下线。设车站5座（含百家园路站），其中换乘站2座，分别与3号线主线、新增12号线换乘。3号线支线由小和山站起依次沿留和路和天目山路敷设。

3号线将在丁桥设一座车辆段，在小和山设一座停车场，大架修车辆基地与5号线五常车辆基地资源共享，控制中心利用1号线的七堡控制中心。在绿汀站附近设主变电所（3/5、杭临）、东新东路变电所（3/4/5共享）、丁桥变三座主变电所。

== 列车 ==
3号线车辆选型为AH型车，每节长19.52m，宽3.08m，高3.8m（含车轮）<ref>{{cite web|author1=中铁第四勘察院|title=杭州地铁3号线一期工程环境影响报告书全文公开本|url=http://www.hzmetro.com/EptionUload/files/file/20180409/20180409170830_3423.pdf|website=杭州地铁}}</ref> ，初、近、远期均采用6节编组、4动2拖，总长117.12m。最高运行速度80公里/小时；DC1500V架空接触网供电。初期配车数72列。<ref>{{cite web|title=省发展改革委关于杭州地铁3号线一期工程初步设计的批复|url=http://www.zjdpc.gov.cn/art/2018/1/12/art_808_1739283.html|website=浙江省发展和改革委员会}}</ref>

== 历史 ==
{{Expand section|date=2018年1月}}
杭州地铁3号线最早规划为自留下始发沿天目山路至武林广场，在武林广场、西湖文化广场两个站与1号线换乘之后，向北折向大关，再向东北折向丁桥、临平。
由于留下车辆段选址问题，在2010年《杭州市轨道交通线网规划（修编）》中，3号线仅为花坞（西溪湿地）至临平红丰路<ref>{{cite web|title=杭州地铁3号线为啥短了十公里 由于留下车辆段建设|url=http://zjnews.zjol.com.cn/system/2013/10/25/019666344.shtml|website=浙江在线}}</ref>，与9号线换乘站则从临平站改为邱山大街站<ref>{{cite web|title=地铁3号线西段将南移　终点为邱山大街站|url=http://www.yuhang.gov.cn/zjyh/jryh/news/201309/t20130906_872303.html|website=余杭政府门户网}}</ref>。而后，由于建设时序调整，3号线未能进入杭州市城市轨道交通第二期建设规划（2010-2016年）。

2013年，3号线古荡至武林门段从天目山路移至曙光路，新线路相比原线路多1个车站。<ref>{{cite web|title=杭州地铁3号线线位重大调整|url=http://hznews.hangzhou.com.cn/chengshi/content/2013-09/05/content_4882018.htm|website=杭州网}}</ref>

2016年，杭州市城市轨道交通第三期建设规划（2017-2022年）获批，3号线一期列入其中，为“主线：文一西路-星桥，支线：小和山-百家园路”的Y型交路，长度约52.2km。其中文一西路-百家园路段为原[[杭州至临安城际铁路|杭州至临安城际铁路]]二期规划调整而来，而小和山-百家园路段规划在未来割接给14号线。在实际设计过程中线路有所微调，从33个站增加为35个站。此时3号线预留西延至杭州西站，东延至临平汽车北站（后又延伸至北沙东路）的远期线路，与9号线远期的换乘站则从邱山大街站改回临平站。

2018年，《杭州市城市轨道交通第三期建设规划（2017-2022年）调整》发布，其中包含从文一西路站（不含）经杭州西站至吴山前村站的3号线北（西）延工程。<ref>{{cite web|title=杭州地铁三期新增机场轨道快线、3号线北延、5号线西延|url=http://zjnews.zjol.com.cn/zjnews/hznews/201803/t20180306_6735701.shtml|website=浙江在线}}</ref>
