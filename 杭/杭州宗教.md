'''杭州宗教'''介绍[[佛教|佛教]]、[[道教|道教]]、[[基督教|基督教]]和[[伊斯兰教|伊斯兰教]]在[[杭州|杭州]]的传播历史和现状。

== 佛教 ==
[[File:Great_statue_of_Sakyamuni_in_the_Mahavira_Hall_of_Lingyin_Temple,_Hangzhou.JPG|缩略图]]内部]]
[[佛教|佛教]]在历史上是对杭州影响最大的宗教。佛教在杭州的传播始于[[东晋|东晋]]，兴于[[五代|五代]]，盛于[[南宋|南宋]]（寺庙多达480所）。杭州的[[径山|径山]]、灵隐寺、净慈寺、三竺寺均名列禅、教、律三宗“五山十刹”前列。<ref name="佛教">{{cite web |url=http://mzj.hz.gov.cn/info_index/queryNewsDetail.do?newsId=11201&type=1 |language= |title=杭州佛教基本情况 |publisher=杭州民族宗教网 |date=2006年1月17日 |accessdate= |deadurl=yes |archiveurl=https://archive.is/20121203190416/http://mzj.hz.gov.cn/info_index/queryNewsDetail.do?newsId=11201&type=1 |archivedate=2012年12月3日 }}</ref>杭州最著名的[[寺院|寺院]]是并称南北两山之最的[[灵隐寺|灵隐寺]]与[[净慈寺|净慈寺]]<ref>{{cite web |url=http://www.hangzhou.com.cn/20050801/ca1177211.htm |language= |title=净慈寺 |publisher=杭州网 |date=2006年8月28日 |accessdate= |archive-url=https://web.archive.org/web/20100510081241/http://www.hangzhou.com.cn/20050801/ca1177211.htm |archive-date=2010-05-10 |dead-url=yes }}</ref>。位于[[西湖|西湖]]西北面的[[灵隐寺|灵隐寺]]坐落于[[飞来峰|飞来峰]]与[[北高峰|北高峰]]之间的灵隐山麓中，创建于[[东晋|东晋]]咸和元年（326年），是中国最早的[[佛教|佛教]][[寺院|寺院]]和中国十大古刹之一。也是杭州历史悠久、景色宜人的游览胜地。[[五代十国|五代十国]][[后周|后周]]显德元年（954年）所建的[[净慈寺|净慈寺]]位于西湖湖畔南屏山，与灵隐寺皆是[[汉族地区佛教全国重点寺院|汉族地区佛教全国重点寺院]]。

杭州市佛教协会成立于1956年8月<ref name= "佛教"/>。[[杭州佛学院|杭州佛学院]]是杭州市佛教协会于1998年创办的一所佛教学院，设有教理院、艺术院两个二级学院以及研究所，本科生学制四年，研究生学制三年。

== 道教 ==
杭州目前可统计的[[道教|道教]]信徒约42000人。杭州市道教协会成立于1985年。杭州历史上著名的宫观有[[玉皇山_(杭州)|玉皇山]]巅的[[福星观|福星观]]、西湖栖霞岭北麓的[[黄龙洞 (杭州)
|黄龙洞]]和[[吴山_(杭州)|吴山]]之巅的[[城隍庙|城隍庙]]<ref>{{cite web |url=http://mzj.hz.gov.cn/info_index/queryNewsDetail.do?newsId=11020&type=1 |language= |title=杭州道教基本情况 |publisher=杭州民族宗教网 |date=2006年1月17日 |accessdate= |deadurl=yes |archiveurl=https://archive.is/20121211041437/http://mzj.hz.gov.cn/info_index/queryNewsDetail.do?newsId=11020&type=1 |archivedate=2012年12月11日 }}</ref>。

== 基督教 ==
{{main|杭州基督教}}
[[File:Cathedral_of_the_Immaculate_Conception_of_Hangzhou_-_1.jpg|缩略图]]]]
[[明代|明代]]时期[[天主教|天主教]]传入杭州地区。杭州现有11所[[天主教堂|天主教堂]]，在职教职人员（主教、神父、修女）10人，信众二千多人<ref>{{cite web |url=http://mzj.hz.gov.cn/info_index/queryNewsDetail.do?newsId=11409&type=1 |language= |title=杭州市天主教基本情况 |publisher=杭州民族宗教网 |date=2006年1月17日 |accessdate= |deadurl=yes |archiveurl=https://archive.is/20121203152644/http://mzj.hz.gov.cn/info_index/queryNewsDetail.do?newsId=11409&type=1 |archivedate=2012年12月3日 }}</ref>；而据2004年初统计杭州有17.9万人信仰[[基督教|基督教]]（其中[[萧山区|萧山区]]占一半以上），市区开放有38处基督教活动场所<ref>{{cite web |url=http://mzj.hz.gov.cn/info_index/queryNewsDetail.do?newsId=11314&type=1 |language= |title=杭州基督教基本情况 |publisher=杭州民族宗教网 |date=2006年1月17日 |accessdate= |deadurl=yes |archiveurl=https://archive.is/20121203194655/http://mzj.hz.gov.cn/info_index/queryNewsDetail.do?newsId=11314&type=1 |archivedate=2012年12月3日 }}</ref>。位于[[中山北路|中山北路]]415号（临近[[武林广场|武林广场]]）的[[杭州圣母无原罪主教座堂|杭州圣母无原罪主教座堂]]是一座历史悠久的[[天主教堂|天主教堂]]，1661年由[[意大利|意大利]][[传教士|传教士]][[卫匡国|卫匡国]]建立。它不仅是[[天主教杭州教区|天主教杭州教区]]的主教座堂，也是杭州市区目前唯一开放的天主教堂，每周日早晨举行中文[[弥撒|弥撒]]，周六晚举行英文弥撒，教堂拥有2个[[唱诗班|唱诗班]]。2011年1月教堂成为[[浙江省文物保护单位|浙江省文物保护单位]]。

[[新教|新教]]教堂有：
*[[美北长老会|美北长老会]]：[[思澄堂|思澄堂]]、[[鼓楼堂|鼓楼堂]]
*[[美南长老会|美南长老会]]：[[天水堂_(杭州)|天水堂]]、[[湖山堂|湖山堂]]、[[杭州市城北堂|城北堂]]、笕桥堂等
*[[中国内地会|中国内地会]]：[[崇一堂|崇一堂]]
*[[英国圣公会|英国圣公会]]：[[信一堂|信一堂]]（1958年关闭，改为[[小学|小学]]）
*[[美北浸礼会|美北浸礼会]]：[[民众堂|民众堂]]（1958年关闭，1985年拆除）

== 伊斯兰教 ==
[[隋代|隋代]]时期从[[广州|广州]]、[[泉州|泉州]]等地来的阿拉伯商人将[[伊斯兰教|伊斯兰教]]传入杭州。据普查，杭州市有[[回族|回族]]等十个信仰伊斯兰教[[少数民族|少数民族]]共4000余人。<ref>{{cite web |url=http://mzj.hz.gov.cn/info_index/queryNewsDetail.do?newsId=11224&type=1 |language= |title=杭州伊斯兰教基本情况 |publisher=杭州民族宗教网 |date=2006年1月17日 |accessdate= |deadurl=yes |archiveurl=https://archive.is/20121203145035/http://mzj.hz.gov.cn/info_index/queryNewsDetail.do?newsId=11224&type=1 |archivedate=2012年12月3日 }}</ref>位于中山中路的[[凤凰寺|凤凰寺]]为中国[[伊斯兰教|伊斯兰教]]四大[[清真寺|清真寺]]之一，该寺创建于[[唐代|唐代]]，[[宋代|宋代]]毁于火灾，[[元朝|元朝]]时得到重建，1953年又经过大修，至今保存完好，2001年成为[[全国重点文物保护单位|全国重点文物保护单位]]，是杭州[[穆斯林|穆斯林]]的聚集中心<ref>{{cite web |url=http://www.aoohz.com/mingsheng/rw04.htm |language= |title=凤凰寺 |publisher=杭州旅游 |date= |accessdate=2012年12月 |archive-url=https://web.archive.org/web/20121210201812/http://www.aoohz.com/mingsheng/rw04.htm |archive-date=2012-12-10 |dead-url=yes }}</ref>。

由于近年杭州穆斯林人口增加，同时出于凤凰寺文物保护的需要，杭州于2011年在[[城东新城|城东新城]]修建[[杭州清真寺|杭州清真寺]]，并于2017年5月正式启用。<ref>{{Cite web|url = http://hzdaily.hangzhou.com.cn/hzrb/2017/06/27/article_detail_1_20170627A0212.html|title = 杭城穆斯林欢度开斋节|accessdate =2018-02-15 |author = |date =2017-06-27 |publisher =[[杭州日报|杭州日报]] }}</ref>杭州清真寺是杭州的第二座清真寺。
