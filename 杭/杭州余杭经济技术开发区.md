'''杭州余杭经济技术开发区'''是一个位于[[杭州市|杭州市]][[余杭区|余杭区]]的[[国家级经济技术开发区|国家级经济技术开发区]]。

== 简介 ==
开发区成立于1993年11月6日<ref name="升级">{{cite web|url=http://hzdaily.hangzhou.com.cn/hzrb/html/2012-09/30/content_1352771.htm | language= | title=杭州余杭经济开发区升级为国家级经济技术开发区 |publisher=杭州日报 |date=2012-09-30 |accessdate= }}</ref>。位于[[临平|临平]]城区以北，目前面积为51.34平方公里，托管23个社区，2008年底常住人口44852人，暂住人口62580人。<ref name="简介">{{cite web |url=http://www.hyeda.com.cn/er_article.aspx?classid=-2 |language= |title=园区简介 |publisher=杭州余杭经济技术开发区 |date= |accessdate=2012年10月 |deadurl=yes |archiveurl=https://web.archive.org/web/20120623115932/http://www.hyeda.com.cn/er_article.aspx?classid=-2 |archivedate=2012-06-23 }}</ref>

主要产业有生物医药业、装备制造业、绿色产业、通信电子和纺织服务业。<ref>{{cite web|url=http://news.xinmin.cn/domestic/gnkb/2012/09/30/16559403.html | language= | title=杭州余杭经济开发区晋升国家级开发区 |publisher=中国新闻网 |date=2012-09-30 |accessdate= }}</ref>

2012年9月29日升级为国家级开发区，成为[[浙江省|浙江省]]在2012年继[[义乌经济技术开发区|义乌开发区]]之后获得升级的第二个国家级开发区，也是浙江第15个国家级开发区。<ref name= "余杭"/>

== 主要公司 ==
园区有49家亿元以上规模企业<ref name= "升级"/>。区内有瑞士[[诺华制药|诺华制药]]集团、法国[[赛诺菲|赛诺菲]]制药集团、美国[[礼来制药|礼来制药]]集团、中国[[华润集团|华润集团]]等多家[[世界500强|世界500强]]企业入驻，也有[[旺旺食品|旺旺食品]]、南都电源、力达纺织、春风摩托、杏辉药业、天元生物、山鹰纸业、华鼎服装、[[诺贝尔陶瓷|诺贝尔陶瓷]]<ref name= "简介"/>、[[老板电器|老板电器]]、[[西奥电梯|西奥电梯]]等一大批国内外企业。2011年开发区工业总产值首次突破300亿元，实现税收18亿元。<ref name="余杭">{{cite web|url=http://www.qzwb.com/gb/content/2012-09/30/content_4147987.htm | language= | title=杭州余杭经济开发区升级为国家级开发区 |publisher=泉州网 |date=2012-09-30 |accessdate= }}</ref>2012年开发区成为浙江省唯一的“生物医药产业示范基地”。<ref name= "升级"/>
