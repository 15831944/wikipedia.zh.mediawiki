'''杭州清真寺'''位于中国[[浙江省|浙江省]][[杭州市|杭州市]][[江干区|江干区]]，是一座伊斯兰风格的[[清真寺|清真寺]]。杭州清真寺是中国最大的清真寺之一。

[[杭州市伊斯兰教协会|杭州市伊斯兰教协会]]也设置在清真寺内。<ref>{{Cite web|url = http://mzj.hangzhou.gov.cn/zjsw/zjtt/200601/t20060117_391028.html|title = 杭州市伊斯兰教协会|accessdate =2018-02-15 |author =民族处 |date =2017-06-17 |publisher =杭州市民族宗教事务局 }}</ref>

== 历史 ==
杭州原仅有[[凤凰寺|凤凰寺]]一座清真寺，近年杭州穆斯林人口增加，截止于2014年底有2万余之多，凤凰寺难以满足穆斯林的宗教活动需求，也难以落实对全国重点文物保护的要求，因此有了新建清真寺的想法。 

杭州清真寺于2011年被列入杭州市重点建设项目名单，<ref>{{Cite web|url = http://www.hzdpc.gov.cn/ztzl/ztzl_tzyzdxm/201106/t20110628_19555.html|title = 2011年杭州市重点建设项目名单|accessdate =2018-02-15 |author = |date =2011-06-28 |publisher =杭州市发展改革委员会 }}</ref>并于翌年获杭州市发展改革委员会批复，同年的10月12日举行了奠基仪式。清真寺于2016年完工，2017年5月正式启用。<ref>{{Cite web|url = http://hzdaily.hangzhou.com.cn/hzrb/2017/06/27/article_detail_1_20170627A0212.html|title = 杭城穆斯林欢度开斋节|accessdate =2018-02-15 |author = |date =2017-06-27 |publisher =[[杭州日报|杭州日报]] }}</ref>

== 建筑 ==
[[File:Gate_Stone_of_Hangzhou_Masjid.jpg|thumb]]
杭州清真寺是典型的中东阿拉伯风格建筑，占地10亩，总建筑面积约为16998平方米，包含地上五层，地下一层。<ref>{{Cite web|url = http://www.hzzx.gov.cn/taxd/content/2014-07/15/content_5362708.htm|title = 关于新建“杭州清真寺”寺名设为公交车站名的建议 |accessdate =2018-02-15 |author = |date =2014-07-15 |publisher =杭州政协新闻网 }}</ref>整个建筑采用金色铜质穹顶、米白色花岗岩，并设有两座[[宣礼塔|宣礼塔]]。清真寺内设有两层男女礼拜大殿、男女净房、多功能厅、伊斯兰文化展示厅、民族团结爱国教育展示厅，以及杭州市伊斯兰教协会办公区，集民族服务、宗教活动、爱国主义教育基地功能为一体。<ref>{{Cite web|url = http://mzj.hangzhou.gov.cn/zjsw/zjhdcs/201707/t20170714_699331.html|title = 伊斯兰教--杭州清真寺 |accessdate =2018-02-15 |author =民族处 |date =2017-06-14 |publisher =杭州市民族宗教事务局 }}</ref>
