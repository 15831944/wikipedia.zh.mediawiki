'''杭州高等教育'''介绍[[杭州市|杭州市]]的高等教育。

== 概况 ==
2009年，位于杭州的全日制普通高校有37所、含研究生在内的在校学生43万人，其中市属高校6所、含研究生在校生5.7万人。[[浙江大学|浙江大学]]、[[浙江工业大学|浙江工业大学]]、[[杭州电子科技大学|杭州电子科技大学]]、[[浙江理工大学|浙江理工大学]]和[[浙江工商大学|浙江工商大学]]是学生人数最多的五所大学。全市高等教育毛入学率53.6%。<ref name="概况">{{cite web |url=http://www.hangzhou.gov.cn/main/zjhz/hzlj/2010/jy/T350332.shtml |language= |title=杭州高等教育概况 |publisher=杭州网 |date=2011-04-13 |accessdate= }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

浙江大学2009年招收本科生5,495人，硕士研究生4,674人，博士研究生1,596人；毕业本科生5,322人，硕士研究生1,626人（其中留学生24人），博士研究生1,272人（其中留学生17人）；在校本科生2.23万人、研究生1.99万人、留学生3,600人（含国际交流学生）。<ref name= "概况"/>

== 高校 ==

部属高校：[[浙江大学|浙江大学]]（教育部直属，浙江省唯一的[[985工程|985工程]]高校）、[[中国美术学院|中国美术学院]]（原教育部直属，现文化部与浙江省共建）。

省属高校：[[浙江工业大学|浙江工业大学]]、[[浙江理工大学|浙江理工大学]]、[[杭州电子科技大学|杭州电子科技大学]]、[[浙江工商大学|浙江工商大学]]、[[中国计量大学|中国计量大学]]、[[浙江中医药大学|浙江中医药大学]]、[[浙江林学院|浙江林学院]]、[[浙江财经大学|浙江财经大学]]、[[浙江科技学院|浙江科技学院]]、[[浙江广播电视大学|浙江广播电视大学]]、[[浙江外国语学院|浙江外国语学院]]、[[浙江传媒学院|浙江传媒学院]]、[[中国工商银行杭州金融研修学院|中国工商银行杭州金融研修学院]]、[[浙江树人大学|浙江树人大学]]，以及2016年新成立的[[浙江音乐学院|浙江音乐学院]]。

市属高校：[[杭州师范大学|杭州师范大学]]、[[浙江大学城市学院|浙江大学城市学院]]、[[杭州职业技术学院|杭州职业技术学院]]、[[杭州广播电视大学|杭州广播电视大学]]。

== 浙江西湖高等研究院 ==
[[浙江西湖高等研究院|浙江西湖高等研究院]]成立于2016年12月，是中国首所民办高等研究院，为[[西湖大学|西湖大学]]（筹）的前身及筹建依托主体。它以开展科学研究和[[博士生|博士生]]培养为主。
