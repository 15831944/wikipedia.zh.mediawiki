'''杭临城际铁路'''、又称'''杭州地铁杭临线'''、'''杭州至临安城际铁路'''，在杭州地铁线路序号中编号为16<ref>{{cite web |title=杭临城际铁路列车“谍照”曝光 前脸系着一根“浅橙色飘带” |url=http://hznews.hangzhou.com.cn/chengshi/content/2019-04/18/content_7179213.htm |website=杭州网}}</ref>。东起于[[余杭区|余杭区]][[绿汀路站|绿汀路站]]，西止于[[临安区|临安区]][[九州街站|九州街站]]，全长34.7公里，设车站12座，其中地下站5座，高架站7座，设计行车速度120公里/小时，车型为4节B型车编组。2016年1月4日最早开工的试验段位于南峰站至青山湖科技城站之间，长2.78公里，全部为高架线路。全线平均速度通常会控制在40-70公里/小时。这条城际铁路建成后，从临安市区到杭州市中心可以控制在1小时以内。

工程建设期为2015年至2019年，计划与[[杭州地铁5号线|5号线]]同步建成投用。城际铁路杭临线建成后将串联临安区、余杭区、杭州主城区，将大大缩短临安、余杭西部与杭州主城的车程，有助于城西创新大走廊加快发展，推动临安融入主城区步伐，促进杭州城乡一体化<ref>{{Cite web|url = http://zjnews.zjol.com.cn/system/2016/01/03/020975128.shtml|title = 杭州至临安城际铁路开工 12个站点串起1小时交通圈|accessdate = |author = |date = |publisher = }}</ref>。

==简介==

杭州至临安城际铁路工程建设期定为 2015 年至 2019 年，全线线路走向为：线路起于临安区锦南新城九州街，在市文体中心前设[[九州街站|九州街站]]，以地下形式沿九洲街向东走行，之后向北拐入万马路延长线，过吴越街路口沿万马路北行，于城中街路口设[[临安广场站|临安广场站]]，出站后线路继续沿万马路北行，穿越万马路及苕溪南路东南象限地块及南苕溪后拐入苕溪北路沿路向东直行，斜穿东湖路与苕溪北路东侧地块后在线路与环城北路交汇处设[[农林大学站|农林大学站]]，出站后线路下穿科技大道并沿路南侧敷设，敷设方式由地下逐渐转为高架，并于高新技术产业园内科技大道与平湖路西南象限交叉口设[[青山湖站|青山湖站]]，之后线路继续沿科技大道南侧走行，在陈市线东侧设[[八百里站|八百里站]]，在西环路路口前拐入路中，在青山科技城临石路路口东侧设[[青山湖科技城站|青山湖科技城站]]，并在南苕溪临余桥以南两跨南苕溪后进入余杭区，线路沿 02 省道北侧绿化带向东走行，在规划南湖小镇片区，现南峰村设[[南峰站|南峰站]]，在跳头村设[[南湖站_(杭州地铁)|南湖站]]，在杭泰路路口东侧绿化带设[[中泰站|中泰站]]，在城东路路口设[[禹航路站|禹航路站]]，之后上跨余杭收费站匝道后向北拐入东西大道，沿道路走行，在规划凤新路路口设[[凤新路站|凤新路站]]，出站后线路绕过青枫墅园转入规划水乡北路，线路终于[[绿汀路站|绿汀路站]]，在此与杭州地铁 3 号线平行双岛同站台换乘，与杭州地铁 5 号线叠落垂直换乘。

线路总长约 34.8km，共设车站 12 座，平均站间距约 3.13km，其中地下站共 5 座（后增加为8座）。线路临安曲境内长度 20.7km，设站 6 座（地下站 3 座）；余杭区境内长度 14.3km，设站 6 座（地下站 5 座）。工程速度目标值为 120km/h，采用 B 型车 4 辆编组、直流 1500V 架空接触网受电。于临安区下泉村设车辆综合基地一座，控制中心设于杭州地铁控制中心内，本工程采用 35kV 分散供电方案<ref>{{Cite web|url =http://www.crfsdi.com.cn/sysWeb/upfiles/hpgs/20151112111501.pdf|title =杭州至临安城际铁路工程环境影响报告书|accessdate =|author =|date =|publisher =|deadurl =yes|archiveurl =https://web.archive.org/web/20160303035612/http://www.crfsdi.com.cn/sysWeb/upfiles/hpgs/20151112111501.pdf|archivedate =2016年3月3日|df =}}</ref>。

== 列车 ==
*制造商：中国中车南京浦镇制车辆有限公司、杭州中车城市轨道交通车辆有限公司
*外观：列车为鼓型车，以黑、白、浅橙为主色调，侧边顶部和中部各有一条浅橙色的细色带。外观与2号线列车类似，但在部分外观上有细微区别。
*材料：铝合金车体
*编组： 4B
*长×宽×高（米）：19×2.88×3.8
*供电方式：架空接触网受电[[直流|直流]]1500[[伏|伏]]
*最高速度：120[[千米每小时|千米每小时]]

== 历史 ==
*2017年9月14日，根据《省发展改革委关于杭州至临安城际铁路工程初步设计变更的批复》，更改了[[青山湖站|青山湖站]]（工程名为高新园区站）-[[凤新路站|凤新路站]]的部分设计，将青山湖站由高架站改为地下站。<ref>http://www.zjdpc.gov.cn/art/2017/9/14/art_808_1733386.html 省发展改革委关于杭州至临安城际铁路工程初步设计变更的批复</ref>
*2018年1月12日，根据《省发展改革委关于杭州至临安城际铁路工程余杭段初步设计变更的批复》，取消了余杭段的高架线，改为地下线，同时将[[中泰站|中泰站]]和[[禹航路站|禹航路站]]由高架站改为地下站。<ref>{{cite web|title=省发展改革委关于杭州至临安城际铁路工程余杭段初步设计变更的批复|url=http://www.zjdpc.gov.cn/art/2018/1/12/art_808_1739284.html|website=浙江省发展和改革委员会}}</ref>
*2019年3月25日，杭州市人民政府了批复杭州至临安城际铁路工程车站的站名，其中有8个站名与工程名不同。<ref>{{cite web |title=杭州市人民政府关于杭州至临安城际铁路工程车站站名的批复 |url=http://www.hangzhou.gov.cn/art/2019/3/28/art_1256295_31738047.html |website=中国杭州}}</ref>
