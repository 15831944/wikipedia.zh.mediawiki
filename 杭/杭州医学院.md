'''杭州医学院'''是位于[[浙江|浙江]][[杭州市|杭州市]]的一所高等院校，是浙江省卫计委直属的唯一一所医学院校，前身是1925年创建的浙江省立女子产科学校，历经浙江省立杭州高级医事职业学校、杭州卫生学校、浙江省卫生学校、浙江医学职业技术学院、浙江医学高等专科学校几个阶段。

== 历史 ==
1974年10月，因[[文革|文革]]停办的杭州卫生学校复校，并改名为浙江省卫生学校。

2002年9月，浙江医学职业技术学院迁往滨江高教园区。2004年5月，学院更名为浙江医学高等专科学校。2016年3月，浙江医专获[[教育部|教育部]]批复升格更名并开始招[[本科生|本科生]]。2016年10月8日，浙江医学高等专科学校正式更名为杭州医学院<ref>{{cite web |url=http://zj.sina.com.cn/edu/jyzx/2016-10-10/100329451.html |language= |title=郑继伟副省长等领导到杭州医学院揭牌暨调研指导 |publisher=新浪浙江教育 |date=2016年10月10日 |accessdate= |archive-url=https://web.archive.org/web/20161010212819/http://zj.sina.com.cn/edu/jyzx/2016-10-10/100329451.html |archive-date=2016-10-10 |dead-url=yes }}</ref>。现任院长吕建新。<ref name="揭牌">{{cite web|url=http://news.xinhuanet.com/local/2016-10/09/c_129314049.htm | language= | title=杭州医学院正式揭牌 |publisher= 浙江日报|date=2016年10月9日 |accessdate= }}</ref>

== 简介 ==
学校有[[滨江区|滨江]]、临安两个校区，总占地面积667.2亩，校舍建筑面积26.22万平米，全日制在校生5400余人。<ref name="招生"/>杭医临安校区位于[[临安区|临安区]][[锦南街道|锦南街道]]核心的柯家村区块，占地面积500亩，预计2017年开始招收新生2000人。<ref name="招生">{{cite web|url=http://news.gmw.cn/newspaper/2016-09/28/content_116576372.htm | language= | title=山清水秀的杭州医学院临安校区明年招生 |publisher=杭州网 |date=2016-09-28 |accessdate= }}</ref>现有341名专任教师，其中具有硕士研究生以上学历者173人，其中博士生67人。<ref name="简介"/>

2016年9月，学院迎来了首届300名本科新生，分别就读于临床医学、医学影像学、护理学、药学、医学检验技术5个专业。<ref name="揭牌"/>

== 专业设置 ==

本科专业（5个）：临床医学专业（五年制）、医学影像学专业（五年制，省高等教育重点专业）、医学检验技术专业、药学专业、护理学专业（四年制）。属于理科第二批招生。

专科专业（12个）：临床医学、口腔医学、康复治疗技术、医学检验技术、卫生检验与检疫技术、医学影像技术、卫生信息管理、卫生监督、健康管理、药学、护理、助产。<ref name="招生"/><ref name="简介">{{cite web| url=http://www.hzm.edu.cn/col/col22/index.html| language=| title=学校简介| publisher=杭州医学院| date=| accessdate=2016-10| deadurl=yes| archiveurl=https://web.archive.org/web/20161014182745/http://www.hzm.edu.cn/col/col22/index.html| archivedate=2016-10-14}}</ref>
