[[杭州市|杭州市]]位于[[浙江省|浙江省]]西北部，东临[[杭州湾|杭州湾]]，省内最大河流[[钱塘江|钱塘江]]从西南向东北方向流经全市大部分地区。按顺时针方向，依次与[[绍兴|绍兴]]、[[金华|金华]]、[[衢州|衢州]]、[[黄山市|黄山]]（[[安徽|安徽]]）、[[宣城|宣城]]（安徽）、[[湖州|湖州]]、[[嘉兴|嘉兴]]等地级市相邻。土地总面积16,596平方公里，市区面积8,000平方公里<ref>[http://www.hangzhou.gov.cn/main/zjhz/hzlj/2007/zsh/T232609.shtml 《杭州年鉴2007年版》]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。市域的四至分别为：北纬29°11′至30°34′；东经118°20′至120°37′。

== 地形 ==
[[File:120.14979E_30.26805N.png|left]]杭州地处[[长江三角洲|长江三角洲]]南沿和钱塘江流域下游，地形复杂多样。杭州市西部、中部和南部属浙西[[丘陵|丘陵]]区，主干山脉有[[天目山|天目山]]、白际山、昱岭、千里岗和[[龙门山|龙门山]]等，其中有多座海拔超过1500米的山峰，其中全市最高点在白际山的[[清凉峰|清凉峰]]（1787米）。东部属浙北平原（组成[[杭嘉湖平原|杭嘉湖平原]]和[[宁绍平原|宁绍平原]]），地势低平，海拔仅3~6米，河网和湖泊密布，较大河流有钱塘江、[[东苕溪|东苕溪]]和[[京杭大运河|京杭大运河]]，主要的水体包括[[新安江水库|新安江水库]]（又称“千岛湖”）、[[青山湖|青山湖]]和西湖等，具有典型的“江南水乡”特征。全市土地面积中，山地丘陵占65.6％，平原占26.4％，各类水体共占8％，故有“七山一水二分田”之说。

== 气候 ==

杭州市属[[亚热带季风气候|亚热带季风气候]]（Cfa），最明显的气候特征是冬夏[[季风|季风]]交替明显，四季分明，降水与日照充足。年平均气温15～17℃（西北山区9～15℃），全年一月份最冷，平均气温3～5℃（西北山区-3～3℃）；七月份最热，平均气温27～30℃（西北山区20～27℃）。极端最高气温42.9℃（1971年7月31日，建德），极端最低气温-20.6℃（1958年1月16日，仙人顶）。市区极端最高气温41.6℃（2013年8月9日），市区极端最低气温-9.6℃（1969年2月6日）。≥10℃[[积温|积温]]4700～5700℃（西北山区2500～4700℃），[[无霜期|无霜期]]230～260天（西北山区210～230天）。年平均降水量1100～1600毫米<ref>{{cite book| title =杭州市志| publisher =中华书局| pages =第一卷自然环境篇概述| url =http://www.hangzhou.gov.cn/main/zjhz/hzsz/citymark/328/T83002.shtml?catalogid=5375| isbn =7101015379| accessdate =2009-01-15}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url=http://www.hangzhou.gov.cn/main/zjhz/hzsz/citymark/331/T83011.shtml?catalogid=5375 |language= |title=第一节 气候要素 |publisher=杭州政府 |date=2003-02-04 |accessdate= }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。日最大降水量为246.4mm（2013年10月7日）。<ref>http://zj.qq.com/a/20131008/002393.htm</ref>杭州夏季较炎热，极端高温发生频率较高。1951～2013年市区年均≥35℃的高温天数25.4天，其中1951～1970年23.2天，1971～2000年21.4天，1981～2010年增至27.2天（仅次于[[福州|福州]]和[[重庆|重庆]]，居省会级城市第三位）。1981～2010年日最高气温≥38℃天数年均3.7天（仅次于[[重庆|重庆]]，居省会级城市第二位）。1951～2016年日最高气温≥40℃天数17天<ref>{{cite web|url=http://cdc.cma.gov.cn/home.do |language= |title=杭州气象资料 |publisher=中国气象科学数据共享服务网 |date= |accessdate=2013年5月 |deadurl=yes |archiveurl=https://web.archive.org/web/20150302053152/http://cdc.cma.gov.cn/home.do |archivedate=2015-03-02 }}</ref>。2013年7月，[[中国气象局|中国气象局]]研究人员根据对最近30年（1981～2010年）的气象资料综合分析认定，杭州35度以上高温的天数位居中国省会城市第三名，杭州也因此与[[福州|福州]]、[[重庆|重庆]]、[[南昌|南昌]]三个城市被网民冠名为“[[火炉城市|新四大火炉]]”<ref >{{cite web|url=http://news.e23.cn/content/2013-07-17/2013071700677.html | language= | title=新四大火炉诞生 古代夏天无空调风扇古人如何消暑? |publisher=华声在线 |date=2013-07-17 |accessdate= }}</ref>。1956年8月2日，受[[颱風溫黛_(1956年)|5612号台风]]影响，杭州拱宸桥观测站（海拔5.3米）最低本站气压陡降至958.7百帕，换算成海平面气压959.3百帕，为中国大陆有观测记录以来省会级城市海平面气压最低值<ref>{{cite web |url=http://www.hangzhou.gov.cn/main/zjhz/hzsz/citymark/331/T83011.shtml?catalogid=5375 |language= |title=第一节 气候要素 |publisher=杭州政府 |date=2003-02-04 |accessdate= }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url=http://cdc.cma.gov.cn/choiceStation.do |title=存档副本 |accessdate=2014-07-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20140224190039/http://cdc.cma.gov.cn/choiceStation.do |archivedate=2014-02-24 }}</ref>。

温暖湿润的气候有利于[[水稻|水稻]]和多种农作物的生长，也为[[杭州旅游业|杭州旅游业]]的发展提供了独特的生态环境。杭州夏季气候炎热湿润，而冬季阴冷潮湿。与此相对，春秋两季气候宜人，恰逢中国法定长假期间，是观光旅游的黄金季节。<ref>{{cite web |url=http://bc.hangzhou.com.cn/content/2012-10/25/content_4443606.htm |language= |title=秋天，是一年中旅游的黄金季节 |publisher=杭州网 |date=2012-10-25 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20131001204315/http://bc.hangzhou.com.cn/content/2012-10/25/content_4443606.htm |archivedate=2013-10-01 }}</ref><ref>{{cite web|url=http://tv.sohu.com/20070914/n252153661.shtml | language= | title=视频：浙江杭州四季分明 春秋是旅游的黄金季节 |publisher=东方卫视 |date=2007年9月14日 |accessdate= }}</ref><ref>{{cite web|url=http://www.daodao.com/Changshi-g298559-c8-Hangzhou_Zhejiang_Lvxingjijie.html | language= | title=杭州:旅行季节 |publisher=到到网 |date=2010-11-05 |accessdate= }}</ref>

== 自然资源 ==
杭州市自然资源和天然能源较丰富，其中水力资源首推[[新安江水库|新安江水库]]。该水库又称“千岛湖”，建成于1959年9月，是中国东部沿海地区最大的水库，库区面积570多平方公里，有大小岛屿1000余个。该水库下游的[[新安江水电站|新安江水电站]]是1949年中华人民共和国成立后建成的第一座大型水电站。

另外，杭州的[[红壤|红壤]]和[[水稻土|水稻土]]适宜[[茶叶|茶叶]]和[[水稻|水稻]]、[[油菜|油菜]]、棉麻、蔬菜等作物生长。临安天目山以其种类繁多的动植物品种被列为[[国家级自然保护区|国家级自然保护区]]。

矿产方面，主要有[[铁|铁]]、[[钼|钼]]、[[铜|铜]]、[[萤石|萤石]]、[[白云岩|白云岩]]、[[石灰岩|石灰岩]]、[[膨润土|膨润土]]等，另外，临安[[昌化|昌化]]特产的[[鸡血石|鸡血石]]，是中国用于图章用石的四大名石之一（其余三种为：[[寿山石|寿山石]]、[[青田石|青田石]]、[[巴林石|巴林石]]）。
