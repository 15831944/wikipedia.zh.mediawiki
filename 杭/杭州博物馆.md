'''杭州博物馆'''，位于[[浙江省|浙江省]][[杭州市|杭州市]][[上城区|上城区]]粮道山18号，是展现杭州历史变迁和城市文物珍藏的人文类综合性博物馆。

==简介==
杭州博物馆占地面积2.4万平方米，建筑面积1.3万平方米，展区面积7千平方米<ref>[http://www.hzmuseum.com/about.aspx?cid=1 杭博简介]</ref>。前身（南馆）为2001年10月开放的“杭州历史博物馆”，新馆（北馆）于2012年6月9日正式对外开放，被[[国家文物局|国家文物局]]评定为国家二级博物馆<ref>[http://www.sach.gov.cn/art/2014/5/29/art_1058_62701.html 2012年度全国博物馆名录] {{webarchive|url=https://web.archive.org/web/20150924114550/http://www.sach.gov.cn/art/2014/5/29/art_1058_62701.html |date=2015-09-24 }}</ref>。

杭州博物馆拥有藏品1.2万件，其中一级文物珍品81件（套）<ref>[http://www.zjww.gov.cn/museum/2006-01-11/48674052.shtml 浙江省文物局-杭州博物馆]</ref>，镇馆之宝是战国[[水晶杯|水晶杯]]，1990年10月在杭州半山镇石桥村出土，是迄今为止我国出土的早期水晶制品中最大的一件，是国宝级文物<ref>[http://www.zjww.gov.cn/news/2012-07-30/585281970.shtml 杭州日报：战国水晶杯：很穿越，很梦幻]</ref>。2002年1月18日，国家文物局公布了64件（组）珍贵文物为首批禁止出国（境）展览的文物名录，这只战国水晶杯就是其中之一<ref>[http://www.wenbao.net/html/whyichan/64th/64SHOU.htm 国家文物局发出通知：首批一级文物禁止出国（境）展览]</ref>。

2017年，杭州博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。
