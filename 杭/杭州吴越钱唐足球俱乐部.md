'''杭州吴越钱唐足球俱乐部'''（简称：杭州吴越钱唐）是位于[[浙江省|浙江省]][[杭州市|杭州市]]的一家职业足球俱乐部，成立于2004年。<ref name=":6">{{Cite web|url=http://jrzb.zjol.com.cn/html/2014-06/06/content_2687059.htm|title=浙企世界杯32强蓄势待发 连足协杯“黑马”都不敢懈怠|accessdate=2018-10-10|author=|date=2014-06-06|publisher=今日早报}}</ref>现作赛于[[中国足球协会会员协会冠军联赛|中冠联赛]]和[[中国足球协会室内五人制足球超级联赛|五超联赛]]。杭州吴越钱唐足球俱乐部的11人制足球主场是[[杭州市体育发展集团体育中心|杭州市体育发展集团体育中心]]。<ref>{{Cite web|url=http://www.sporthz.com/show/index/id/495.html|title=体育中心成浙超联赛杭州队主场|accessdate=2018-10-16|author=|date=2017-08-01|publisher=杭州市体育发展集团}}</ref>

== 俱乐部历史 ==
杭州吴越钱唐足球俱乐部的前身是成立于2004年9月8日的杭州城市天使队。2005年3月，球队获得杭州芝浦科技有限公司赞助，更名为杭州芝浦队。<ref name=":6" />

2013年11月26日，杭州芝浦业余足球俱乐部（登记号:浙杭050016）在杭州市民政局登记注册成立。<ref>{{Cite web|url=http://220.191.210.68/frame2/publish.jsp?pk_id=STMF_STDJ_NW1308000003|title=杭州芝浦俱乐部成立公示|accessdate=2018-10-10|author=|date=|publisher=杭州市社会组织管理信息网}}</ref>

2015年3月4日，经中国足球协会杯委员会审核通过，更名为杭州安格足球俱乐部。<ref>{{Cite news|url=http://sports.sina.com.cn/c/2015-03-04/17147532008.shtml|title=关于杭州芝浦和广西柳粤俱乐部更名的公告|author=|date=2015-03-04|work=新浪体育|accessdate=2018-10-10|archive-url=https://web.archive.org/web/20150402172212/http://sports.sina.com.cn/c/2015-03-04/17147532008.shtml|archive-date=2015-04-02|dead-url=no}}</ref>

2017年3月，杭州安格足球俱乐部、杭州启盟足球俱乐部、杭州T60足球俱乐部达成合作协议，三方约定以杭州安格足球俱乐部为主体，合并组建杭州吴越钱唐足球俱乐部，向2018赛季[[中国足球协会乙级联赛|中乙联赛]]参赛资格发起冲击。<ref>{{Cite web|url=http://hznews.hangzhou.com.cn/wenti/content/2017-03/28/content_6506052.htm|title=杭州三支业余足球队要“搞事情” 组建“吴越钱唐”目标冲击中乙|accessdate=2018-08-21|author=|date=2017-03-28|work=|publisher=杭州网}}</ref>

2017年7月11日，杭州吴越钱唐足球俱乐部代表浙江省参加在天津举行的[[中华人民共和国第十三届运动会|中华人民共和国第十三届运动会]]笼式足球项目，并在决赛中以3 : 2的比分击败陕西队，获得（男子五人制组）冠军。这也是浙江足球史上首个全运会冠军。<ref name=":4">{{Cite web|url=http://www.chinanews.com/ty/2017/07-11/8275147.shtml|title=创造历史 浙江男子笼式足球队勇夺第13届全运会冠军|accessdate=2018-08-21|author=|date=2017-07-11|work=|publisher=中国新闻网|archive-url=https://web.archive.org/web/20180821223138/http://www.chinanews.com/ty/2017/07-11/8275147.shtml|archive-date=2018-08-21|dead-url=no}}</ref>

2018年5月13日，杭州吴越钱唐足球俱乐部与[[杭州师范大学|杭州师范大学]]达成战略合作框架协议，共同推动足球运动发展，共建五人制和十一人制足球队。<ref>{{Cite web|url=http://sports.sina.com.cn/other/2018-05-13/doc-ihamfahx6423457.shtml|title=全运冠军队和杭师大战略合作 杭州足球要走学院派路线|accessdate=2018-08-21|author=|date=2018-05-13|work=|publisher=新浪体育}}</ref>杭州吴越钱塘还与[[湖北大学|湖北大学]]合作，由部分湖北大学校队师生球员与原俱乐部全运会冠军球队球员合作组队参加2018年五甲联赛。<ref>{{cite web |author1=新浪体育 |title=全运会冠军+大学生国脚 吴越钱唐要搞什么大事情？ |url=http://sports.sina.com.cn/china/zj/2018-03-22/doc-ifysnevk4805937.shtml |website=sports.sina.com.cn |access-date=2018-11-04 |archive-url=https://web.archive.org/web/20180329052449/http://sports.sina.com.cn/china/zj/2018-03-22/doc-ifysnevk4805937.shtml |archive-date=2018-03-29 |dead-url=no }}</ref><ref>{{cite web |author1=新浪体育 |title=一支业余足球俱乐部 让浙江重新有了超级联赛球队 |url=http://sports.sina.com.cn/china/futsal/2018-08-05/doc-ihhhczfa8796961.shtml |website=sports.sina.com.cn |access-date=2018-11-04 |archive-url=https://web.archive.org/web/20180913115413/http://sports.sina.com.cn/china/futsal/2018-08-05/doc-ihhhczfa8796961.shtml |archive-date=2018-09-13 |dead-url=no }}</ref>

2018年6月30日，杭州吴越钱唐足球俱乐部参加在宁夏[[石嘴山市|石嘴山市]]举行的2017-2018中国足球协会室内五人制足球甲级联赛总决赛阶段比赛，以6胜1平的不败战绩，勇夺五甲联赛冠军。并取得[[2018-2019中国足球协会室内五人制足球超级联赛|2018-2019中国足球协会室内五人制足球超级联赛]]的报名参赛资格。<ref name=":3">{{Cite news|url=http://www.thecfa.cn/wjxwzx/20180630/23889.html|title=五甲联赛全国总决赛结束 杭州吴越湖大勇夺联赛冠军|author=|date=2018-06-30|work=中国足球协会官网|accessdate=2018-08-21}}</ref><ref>{{cite web |author1=湖北大学体育学院、鲜文涛 |title=我校获得全国五人制足球甲级联赛总决赛冠军-湖北大学 |url=http://www.hubu.edu.cn/info/1246/28173.htm |website=www.hubu.edu.cn}}</ref>这也宣告了自2013年浙江黄龙足球队解散后<ref>{{Cite web|url=http://sports.qq.com/a/20130607/019324.htm|title=浙江黄龙足球队解散 五甲老牌劲旅无奈退出|accessdate=2018-10-08|author=|date=2013-06-07|publisher=腾讯体育}}</ref>，浙江再次拥有了五人制足球顶级联赛的球队。

2018年9月30日，杭州吴越钱唐足球俱乐部参加在山东省[[青岛市|青岛市]]举行的2018年室内五人制足球中国足球协会杯第四阶段的三四名决赛中，以6 : 4的比分击败上赛季五超联赛第10名的新疆师范大学队，夺得2018年中国五人制足协杯季军。<ref name=":5">{{Cite web|url=http://www.dongqiudi.com/archive/811551.html|title=即墨杯2018室内五人制中国足球协会杯圆满结束|accessdate=2018-10-10|author=五超联赛|date=2018-10-02|publisher=懂球帝}}</ref>

2018年10月13日，杭州吴越钱唐足球俱乐部参加在山西省[[太原市|太原市]]进行的2018年中冠联赛全国总决赛四分之一决赛次回合的比赛中，以1:1的比分战平山西信都，从而以3:2的总比分淘汰山西信都，晋级中冠联赛半决赛，并且取得2019年中乙联赛的优先报名参赛资格，接下来只要能通过中国足协的各项准入审查，杭州吴越钱唐足球俱乐部就能征战[[2019年中乙联赛|2019年中乙联赛]]。<ref>{{Cite web|url=http://zjtyol.zjol.com.cn/zhzx/201810/t20181014_8473296.shtml|title=杭州昨天有了第二家男足职业俱乐部 吴越钱唐冲乙成功！|accessdate=2018-10-14|author=|date=2018-10-14|work=|publisher=浙江在线体育|archive-url=https://web.archive.org/web/20181014091616/http://zjtyol.zjol.com.cn/zhzx/201810/t20181014_8473296.shtml|archive-date=2018-10-14|dead-url=no}}</ref><ref>{{Cite web|url=http://www.dongqiudi.com/archive/821453.html|title=中冠综述：杭州吴越钱唐晋级四强；湖北楚风合力3-0天津锐虎占先机|accessdate=2018-10-13|author=|date=2018-10-13|work=|publisher=懂球帝|archive-url=https://web.archive.org/web/20181013172424/http://www.dongqiudi.com/archive/821453.html|archive-date=2018-10-13|dead-url=no}}</ref>
