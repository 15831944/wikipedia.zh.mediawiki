'''杭州地铁5号线'''长51.65公里，西北起[[余杭区|余杭区]]老余杭镇，东南至[[萧山区|萧山区]][[杭州南站|杭州南站]]以东姑娘桥板块，大致呈西北-东南走向。它跨幅最大，穿越杭州的余杭区、西湖区、拱墅区、下城区、上城区、滨江区、萧山区共计7个区域。规划有16个换乘点，为换乘点最多的一条线路<ref >{{cite web|url=http://house.hangzhou.com.cn/lsxw/bbyw/content/2014-11/21/content_5535495.htm | language= | title=到2019年 杭州拥有1、2、4、5、6共5条线 杭州地铁真正成网 |publisher=钱江晚报 |date=2014年11月21日 |accessdate= }}</ref>。2014年12月20日，5号线位于余杭区的[[良睦路站|良睦路站]]、[[杭师大仓前站|杭师大仓前站]]和[[永福站|永福站]]率先开建，这3座车站计划总工期为三年，至2018年初已完工。<ref name="开工">{{cite web|url=http://ori.hangzhou.com.cn/ornews/content/2014-12/19/content_5576597.htm | language= | title=杭州地铁5号线开工 余杭未来科技城3个车站率先建 |publisher=杭州网 |date=2014-12-19 |accessdate= }}</ref>[[杭港地铁|杭港地铁]]将参与该线路的运营。首通段（创景路-善贤）预计于2019年6月30日开通。<ref>{{cite web |title=杭州地铁5号线隧道运行视频曝光！开通时间有底了！_ |url=http://zj.qq.com/a/20190226/001217.htm |website=腾讯大浙网}}</ref>

== 简介 ==
[[File:杭州地铁5号线城站站基坑.JPG|缩略图]]
[[File:杭州地铁5号线城站站基坑高空俯瞰.JPG|缩略图]]
杭州地铁5号线西起[[余杭区|余杭区]]的老余杭镇站，经凤新路、文二西路、创景路、余杭塘路、萍水街至桥西连接湖州路附近的运河商贸中心，再经西文街、东新路、建国北路、建国中路到[[杭州站|城站]]，把[[下城区|下城区]]居住公共中心、会展中心连接起来，再沿江城路、飞云江路过江后沿长河路向南，折向东至滨康路、金城路，最终经过[[杭州南站|杭州南站]]，终点为[[姑娘桥站|姑娘桥站]]。线路全长51.8公里，设车站38座，全为地下线。五常设一座[[五常车辆基地|五常车辆基地]]，萧山设一座[[姑娘桥停车场|姑娘桥停车场]]。控制中心利用[[杭州地铁1号线|杭州地铁1号线]]的七堡控制中心。<ref name="开工"/>

该线为轨道交通次干线，它连接1、2、3三条主干线，以便把主干线上的客流运送到更远的地区。<ref>{{cite web|url=http://www.0571dt.cn/?page_id=200 | language= | title=杭州地铁5号线 |publisher=杭州地铁生活门户网站 |date= |accessdate=2012年10月 }}</ref>

[[杭港地铁|杭港地铁]]将参与该线路的运营，作为其拓展计划的一部分。[[港铁公司|港铁公司]]于2016年5月递交为期二十五年的「[[政府和社会资本合作|公私合營]]」特许经营权标书<ref>{{cite web|url=http://www.mtr.com.hk/archive/corporate/ch/investor/mtr_2016_interim_chi_media_final.pdf|title=港鐵公司 二零一六年中期業績報告|date=2016-08-09|publisher=港铁}}</ref>，并于2017年6月26日与[[杭州市人民政府|杭州市人民政府]]签订地铁5号线PPP项目特许协议<ref>{{cite news|title=杭州市政府与港铁公司签订地铁5号线PPP项目特许协议|url=http://www.mtrhz.com.cn/dynamic/detail-1205.htm|publisher=杭港地铁|date=2017-06-27}}</ref>。根据协议，杭港地铁将负责5号线的机电设备工程部分的投资以及线路落成通车后25年的营运管理和维护<ref>{{cite news|title=港鐵奪杭州地鐵5號線特許經營權　總投資達109億|url=http://hd.stheadline.com/news/realtime/hk/938073/|publisher=头条日报|date=2017-06-27}}</ref>。

== 历史 ==
2013年6月20日，国家发改委正式批复《杭州市城市轨道交通近期建设规划（2013-2019年）》，其中包括杭州地铁5号线一期工程。按照规划，杭州地铁5号线一期工程于2013年8月底完成工程可行性报告；2013年底完成初步设计；2014上半年开工建设；2019年底建成通车。
*2013年10月15日，5号线一期工程香樟路-科技岛（现[[创景路站|创景路站]]）开始二次环评公示，根据环评公示，杭州地铁5号线一期工程工期60个月，预计2019年试运营。 
*2014年8月25日，可行性研究报告获浙江省发改委批复。
*2014年10月15日，初步设计获浙江省发改委批复。
*2014年12月20日，仓前站、常二路站、杭师大站正式开工，标志着杭州地铁5号线一期工程正式开工。 
*2015年8月27日，杭师大站开始基坑开挖，这是杭州地铁5号线一期工程第一个进入基坑开挖的车站。 
*2016年7月7日，杭师大站至常二路站区间工程继左线隧道贯通后，同年8月2日，区间右线隧道也顺利贯通，成为杭州地铁5号线一期工程首个实现双线贯通的区间隧道。

*2017年7月，建国北路站、江南大道站、沈半路站主体工程施工需要，沿线机动车道进行交通管制。
*2017年9月，二期工程初步设计获浙江省发改委批复，杭州地铁5号线二期工程起于绿汀路站、终到创新路站，全长3.2千米，即将开工建设，预计与2019年底与杭州地铁5号线一期工程同步建成投入使用。 
*2018年4月15日，中央公园站至仓前站区间双线贯通。  
*2018年6月8日，杭州市地铁集团有限责任公司发布了杭州地铁5号线一期、二期38座车站拟定站名方案，并向社会征求意见。
*2018年11月8日，5号线38座车站正式定名，其中有22个站名与工程名不同。

== 事件 ==
2018年10月17日15点30分左右，5号线右线南星桥站往长河站过江隧道（由北向南）掘进盾构工作位置的上方出现大量不明泡沫翻涌。市建委经调查分析，确定江面泡沫为地层中释放的沼气夹带施工盾构润滑用的发泡剂引起，隧道及左右线盾构机均处于安全状态，对周边取水口、海塘、桥梁等安全均无影响。17点半左右，白色泡沫已消失，施工单位已暂停该工艺施工。<ref>{{cite web |title=刚刚！官方发布关于钱塘江江面泡沫有关情况说明！_现场_浙青网-青年时报官网,全国青年报三强 |url=http://www.qnsb.com/news/index.php?m=content&c=index&a=show&catid=80&id=94783 |website=青年时报}}</ref>
