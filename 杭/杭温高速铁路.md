'''杭温高速铁路'''，简称'''杭温高铁'''，是[[浙江省|浙江省]]一条连接[[杭州市|杭州市]]、[[金华市|金华市]]、[[台州市|台州市]]和[[温州市|温州市]]之间的[[高速铁路|高速铁路]]。

== 项目进展 ==

杭温高铁一期工程尚未实质性开工，预计2018年内实质性开工，预计2022年建成通车。<ref>{{cite web |author1=温州市铁路与城市轨道交通建设办公室 |title=杭温高铁为何迟迟未见全面开建，根据目前进展预计何时全线进场施工，能否确保2021年通车 |url=http://wlwz.wenzhou.gov.cn/wzshow_62893.html |website=wlwz.wenzhou.gov.cn |accessdate=2018-08-25}}</ref>

== 运营 ==

=== 影响 ===

杭温高速铁路项目建成通车时，将结束[[仙居县|仙居县]]、[[磐安县|磐安县]]不通铁路的历史。

== 线路概况 ==

新建杭温铁路北起杭州西，南接杭深铁路永嘉站（温州北）至温州南，线路途经杭州、金华、台州、温州等4市9县（市、区）。杭州西至温州南全长约331.3公里。
