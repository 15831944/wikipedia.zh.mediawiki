'''杭州之江国家旅游度假区'''是位于[[中華人民共和國|中華人民共和國]][[杭州市|杭州市]]的[[国家旅游度假区|国家旅游度假区]]，1992年10月，经[[中华人民共和国国务院|国务院]]批准建立。度假区位于杭州西南部，南临[[钱塘江|钱塘江]]（即'''之江'''），北依[[五云山|五云山]]，面积9.9平方公里。度假区位于[[西湖区|西湖区]][[转塘街道|转塘街道]]辖区内，由杭州之江国家旅游度假区'''管委会'''负责管理<ref name="介绍">{{cite web |url=http://www.zftec.gov.cn/zfxxgk/xxgkml/kfqgz/gjjkfq/T205597.shtml |language= |title=杭州之江国家旅游度假区 |publisher=浙江省商务厅 |date= |accessdate=2012年11月 |deadurl=yes |archiveurl=https://archive.is/20121129231908/http://www.zftec.gov.cn/zfxxgk/xxgkml/kfqgz/gjjkfq/T205597.shtml |archivedate=2012-11-29 }}</ref>。

== 简介 ==
区内景色秀丽、环境幽雅，距离[[西湖|西湖]]、[[钱江潮|钱江潮]]、[[富春江|富春江]]、[[新安江|新安江]]、[[千岛湖|千岛湖]]等著名景点都不远，是杭州风景名胜区的最佳结合部<ref name= "介绍"/>。

在度假区内投资兴建旅游项目的外商投资企业可以享受到许多优惠政策<ref name= "介绍"/>。

目前规划面积已达50.68平方公里。度假区成立多年来，主要建成了[[杭州宋城|杭州宋城]]、西湖国际高尔夫等主题旅游项目，以及华庭云栖度假酒店、玫瑰园度假酒店、希尔假日酒店等许多度假单元，累计接待中外游客超过2900万人次。投入30多亿元进行基础设施建设，基础配套日趋完善，生态环境大幅改善。招商引资、旅游接待量、项目建设等各方面在全国12个[[国家旅游度假区|国家旅游度假区]]中均名列前列。<ref>{{cite web |url=http://www.hz-zj.com/aboutus/intro.aspx |language= |title=之江概况 |publisher=杭州之江国家旅游度假区 |date= |accessdate=2012年11月 |deadurl=yes |archiveurl=https://archive.is/20101017042931/http://www.hz-zj.com/aboutus/intro.aspx |archivedate=2010-10-17 }}</ref>

== 地理交通 ==
度假区距西湖6公里，距市中心10公里，距[[萧山国际机场|萧山国际机场]]25公里，距[[杭州火车站|杭州火车站]]13.5公里，距[[沪杭高速公路|沪杭高速公路]]入口处22公里。区内东西南北向主干道与支线路网已基本完成，路幅宽12－40米<ref name= "介绍"/>。

== 浙江文化城 ==
规划中的浙江文化城规划在之江度假区的核心区域，建在珊瑚沙水库的西侧，文化城占地1000亩，主要包括“一院四馆两公司”：

*一院，即已开工建设的[[浙江音乐学院|浙江音乐学院]]；
*四馆，指[[浙江图书馆|浙江图书馆]]新馆、[[浙江省博物馆|浙江省博物馆]]新馆、[[浙江非物质文化遗产馆|浙江非物质文化遗产馆]]和[[浙江文学馆|浙江文学馆]]；
*两公司，指浙江的一家[[传媒|传媒]]集团的文化产业项目，和一家[[出版|出版]]集团的文化产业项目。<ref >{{cite web|url=http://hznews.hangzhou.com.cn/chengshi/content/2013-01/30/content_4588675.htm | language= | title=2016年，之江地块将崛起一座浙江文化城 |publisher=杭州网 |date=2013-01-30 |accessdate= }}</ref>
