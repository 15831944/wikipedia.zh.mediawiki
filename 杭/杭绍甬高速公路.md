'''杭绍甬高速公路'''，又名'''杭甬高速复线'''，是西起[[杭州|杭州]][[下沙|下沙]]<ref name="下沙">{{cite web |url = http://www.hangzhou.com.cn/20081020/ca1606036.htm |language = zh-hans |title = 杭绍甬高速公路2010年提前开建 起点在下沙杭州境内长约25公里 |publisher=杭州网 |date=2008年11月21日 |accessdate= }}</ref>，沿[[杭州湾|杭州湾]]南岸由西往东，东至[[宁波|宁波]]的高速公路，途经杭州、[[绍兴|绍兴]]、宁波三地，并间接联系上了[[舟山|舟山]]。<ref name="启动">{{cite web |url = http://zjnews.zjol.com.cn/05zjnews/system/2011/08/11/017758635.shtml |language= zh-hans |title = 杭甬之间将再建一条高速 杭绍甬高速线位论证启动 |publisher=浙江在线 |date=2011年8月11日 |accessdate = }}</ref>其中，宁波段一期工程初步设计已获得[[中华人民共和国交通运输部|交通运输部]]批准，将于2018年内全面开工。<ref>{{Cite web |url=http://news.cnnb.com.cn/system/2018/02/22/008728085.shtml |title=杭甬高速复线初步设计获批 全市最大交通项目年内全面动工 |accessdate=2018-02-22 |author=中国宁波网 |date=2018-02-22}}</ref>

== 简介 ==
由于[[杭甬高速公路|杭甬高速公路]]（浙江首条高速公路，长145公里，1996年通车，2007年11月28日，全线实现双向八车道通行）和[[329国道|329国道]]承担的交通运输压力很大，所以需要再建一条杭州到宁波的高速公路。

杭绍甬高速规划全长128公里，双向六车道，将与[[嘉绍大桥|嘉绍大桥]]、[[杭州湾跨海大桥|杭州湾跨海大桥]]、舟山连岛工程[[金塘大桥|金塘大桥]]相接，并与[[杭甬高速公路|杭甬高速公路]]形成南北呼应。此外据地方媒体说法，建成后其车辆限速将全国首次突破120 km/h，提升至150 km/h。
