'''杭氧站'''（工程名为城市之星站）是[[杭州地铁5号线|杭州地铁5号线]]西北段的一个站点，位于杭州市下城区原杭州制氧机厂门口，东新路与规划胜景西路口，沿东新路布置，有9个出入口。2015年开工建设。<ref >{{cite web|url=http://hz.house.sina.com.cn/scan/2014-09-03/17514426178.shtml | language= | title=地铁5号线开工倒计时 15年1月开工19年4月底通车 |publisher=新浪房产 |date=2014-09-03 |accessdate= }}</ref><ref>{{cite web |title=杭州地铁5号线城市之星站动工开建 附其余各站点具体位置及施工进度 |url=https://mp.weixin.qq.com/s?__biz=MzAwODU1MzIwMw==&mid=2651351305&idx=2&sn=439f8e395b7da509977d717b0a7a7942&scene=4#wechat_redirect |website=口水杭州}}</ref>

== 车站设计 ==
杭氧站为地下二层岛式站台，车站全长483.6米，共设置9个出入口及4组风亭，采用明挖顺筑法施工。<ref>{{cite web |title=杭州地铁5号线城市之星站主体结构开工 |url=http://www.zjsgzw.gov.cn/art/2017/1/6/art_17_13486.html |website=浙江省国资委}}</ref>
