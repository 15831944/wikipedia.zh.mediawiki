'''杭州公园列表'''列举位于[[杭州市|杭州市]]的游憩[[公园|公园]]和[[主题公园|主题公园]]。

== 简介 ==
杭州最有名、最美丽的公园是位于[[西湖|西湖]]边的[[湖滨公园|湖滨公园]]<ref>{{cite web |url=http://zt-hzrb.hangzhou.com.cn/system/2012/03/18/011822144.shtml |language= |title=2011年杭州最美公园揭晓 湖滨公园市民公园当选 |publisher=都市快报 |date=2012-03-18 |accessdate= }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>和[[太子湾公园|太子湾公园]]。

== 主要公园 ==
*[[湖滨公园|湖滨公园]]，从南至北包括'''一公园'''、二公园、三公园、四公园、五公园和'''六公园'''，位于西湖东侧，东沿[[湖滨路|湖滨路]]<ref>{{cite web |url=http://www.cngoto.com/chi/search_detail3.asp?id=1331 |language= |title=湖滨公园 |publisher=天堂生活 |date= |accessdate=2012年11月 |deadurl=yes |archiveurl=https://web.archive.org/web/20120325074516/http://www.cngoto.com/chi/search_detail3.asp?id=1331 |archivedate=2012-03-25 }}</ref>；
*[[太子湾公园|太子湾公园]]，西湖西南侧，北沿[[南山路|南山路]]；
*[[中山公园|中山公园]]，位于西湖[[孤山_(西湖)|孤山]]岛上，1927年（民國16年）为纪念[[孫中山|孫中山]]先生而命名；
*[[城北体育公园|城北体育公园]]，位于[[大关街道|大关街道]]，西临绍兴路；
*紫荆公园
*学士公园
*长桥公园
*少年儿童公园，位于[[杭州动物园|杭州动物园]]北边，东临[[虎跑路|虎跑路]]；
*[[钱江新城森林公园|森林公园]]，位于[[钱江新城|钱江新城]]；
*[[钱江新城市民公园|市民公园]]，位于[[钱江新城|钱江新城]]；
*[[望江公园|望江公园]]，2012年年底开放；<ref>[http://www.hzmotv.com/forum.php?mod=viewthread&tid=210402&page=1&extra=#pid361622 年底运河隧道建成望江公园开放]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>
*[[杭州消防主题公园|杭州消防主题公园]]，位于[[下沙经济开发区|下沙经济开发区]]；
*沿江景观公园，位于下沙经济开发区，总面积近18万平方米，2013年年底开放；<ref >{{cite web|url=http://ori.hangzhou.com.cn/ornews/content/2013-10/24/content_4940441.htm | language= | title=钱塘江边的大花园 沿江景观公园今年年底开园 |publisher=杭州网 |date=2013-10-24 |accessdate= }}</ref>
*[[杭州白塔公园|杭州白塔公园]]，2014年5月1日开放
