'''勐角傣族彝族拉祜族乡'''简称'''勐角民族乡'''、'''勐角乡'''，是{{PRC admin/fullname|53/09/27/205/000|3}}下辖的一个[[民族乡|民族乡]]。勐角一词由[[傣语|傣语]]音译，“勐”为地方，“角”（方言读guo）为开创，意为“最先开创家业的地方”。乡政府驻勐角村，陆距沧源县城14.5千米，现任乡长是王官和，党委书记是涂安磊。乡境内有始建于公元1380年的[[金龙寺_(沧源)|金龙寺]]，以及因传统佤族聚落而闻名的旅游村寨[[翁丁村|翁丁村]]。<ref name="政区大典">{{cite book |author1=段丽元主编 |author2=[[中华人民共和国民政部|中华人民共和国民政部]] |title=《中华人民共和国政区大典 云南卷》 |url= |location=[[北京市|北京]] |publisher=[[中国社会出版社|中国社会出版社]] |page=1506-1507 |date=2016年05月 |isbn=978-7-5087-5306-5}}</ref><ref>{{cite web |url=http://fo.ifeng.com/a/20160216/41551371_0.shtml |title=勐角金龙寺 |author= |date=2016-02-16 |website=fo.ifeng.com |publisher=凤凰佛教 |access-date=2018-01-06 |quote=}}</ref><ref>{{cite web |url=http://iel.cass.cn/ztpd/ddlt/wyb/200810/t20081031_2763893.shtml |title=翁丁村之旅 |author=袁智中 |date=2008-10-30 |website=中国民族文学网 |publisher=文艺报 |access-date=2017-10-22 |quote=}}</ref>

==历史沿革==

早期历史上勐角归由[[勐角董土司|勐角董土司]]管辖。1945年，[[沧源设治局|沧源设治局]]实行废区划乡，新成立'''勐角镇'''。1954年改置为'''勐角区'''，辖境包括[[勐董镇|勐董]]。1965年勐董镇从勐角区划出。1968年在[[人民公社化运动|人民公社化运动]]中改置为'''勐角公社'''，次年更名'''红忠公社'''，1971年复名勐角公社。1984年设置勐角区，1988年撤区建乡成立勐角傣族彝族拉祜族乡。<ref name="县志">{{cite book |author1=李明富主编 |author2=沧源佤族自治县地方志编撰委员会 |date=1998年12月 |title=《沧源佤族自治县志》 |url= |location=[[昆明|昆明]] |publisher=云南民族出版社 |page=42 |isbn=7-5367-1498 |ignore-isbn-error=true }}</ref><ref>{{cite web |url=http://www.xzqh.org/html/show/yn/19051.html |title=勐角傣族彝族拉祜族乡 |author= |date=2010-06-03 |website=www.xzqh.org |publisher=行政区划网 |access-date=2018-01-06 |quote=}}</ref>

==地理==

[[File:翁丁村01.jpg|thumb]]的佤族聚落]]

勐角傣族彝族拉祜族乡位于沧源县中部，东南西北依次与[[糯良乡|糯良乡]]、[[勐董镇|勐董镇]]、[[班洪乡|班洪乡]]和[[勐来乡|勐来乡]]接壤。勐角乡总面积216.76平方千米，[[中华人民共和国第六次全国人口普查|2010年普查]]总人口13,990人，居民以[[佤族|佤族]]为主，其次是[[傣族|傣族]]、[[彝族|彝族]]、[[拉祜族|拉祜族]]和[[汉族|汉族]]，占比分别占总人口的59.63%、21.27%、8.32%、6.48%和3.9%。乡内主要坝子是[[勐角坝|勐角坝]]，山脉有[[窝坎大山山脉|窝坎大山]]、[[芒告大山山脉|芒告大山]]等，地势南北高中西低，最高点为窝坎大山（2,605米，这也是沧源县的最高点），最低点为新牙河与[[班洪乡|班洪]]交界处（680米）。主要河流有[[勐董河|勐董河]]、南角河、新牙河等，属[[澜沧江|澜沧江]]水系，新牙河上建有4座梯级水电站，主要水库有[[翁丁水库|翁丁水库]]等，在建有中型[[芒回水库|芒回水库]]。气候上属[[亚热带季风气候|亚热带季风气候]]，全年气候温和，立体气候显著，年均气温18.5℃，年均降水1602.1毫米。勐角乡的部分地区属[[南滚河国家级自然保护区|南滚河国家级自然保护区]]，2011年全乡森林覆盖率达71%。莲花塘拥有[[煤矿|煤矿]]储量1,500万吨，可开采面积15平方千米。<ref name="政区大典"></ref><ref name="概况">{{cite book |author1=沧源佤族自治县概况编写组 |author2=沧源佤族自治县修订委员会 |date=2007年11月 |title=《沧源佤族自治县概况》 |url= |location=[[北京|北京]] |publisher=[[民族出版社|民族出版社]] |page=33-34 |isbn=978-7-105-08553-8 |ignore-isbn-error=true |author-link= }}</ref>

==经济==

勐角傣族彝族拉祜族乡的经济以农业为主，2011年农民人均纯收入3,621元。2011年，勐角乡粮食种植面积3.92万亩，年产8,803吨；[[核桃|核桃]]4.73万亩，年产2.7吨；茶场8,273亩，年产99吨；{{le|烤烟|Flue-cured tobacco}}2,075亩，年产152吨，其他农作物还有[[甘蔗|甘蔗]]、[[蔬菜|蔬菜]]、[[油菜|油菜]]等。[[畜牧业|畜牧业]]方面，2011年末生猪和其他大牲畜共存栏27,492头。2011年政府财政收入177万元。<ref name="政区大典"></ref>

==基础设施==

勐角傣族彝族拉祜族乡内有小学8所，初中1所，在校学生共1,758人。医疗方面有卫生所1所、村卫生室9个，共有病床30张，专业卫生人员32人。交通方面，S314[[中华人民共和国公路等级|省道]][[314省道_(云南)|小沧公路]]经过勐角，其他有乡村公路24条共64.25千米，其中砂石路面18条共71.53千米、水泥公路4条1.93千米。<ref name="政区大典"></ref>
