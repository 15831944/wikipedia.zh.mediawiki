'''国营勐省农场'''是[[云南省|云南省]][[沧源佤族自治县|沧源佤族自治县]]的一个[[农场|农场]]，位于[[勐省镇|勐省镇]]境内，农场场部陆距沧源县城54.1千米，隶属于[[云南农垦集团|云南省农垦总局]]，也受沧源县政府机关的双重领导。现任农场主任是银韫喆，管委会党委书记是卫军明<ref>{{cite web |url=http://www.cangyuan.gov.cn/show-22-13424-1.html |title=沧源佤族自治县河长制工作实施方案 |last= |first= |date=2017-11-24 |website=www.cangyuan.gov.cn |publisher=沧源水务局 |access-date=2017-12-16 |quote=}}</ref>。

==历史==

1958年3月16日，[[中国人民解放军|中国人民解放军]]第11速成中学、[[中国人民解放军陆军第十四集团军|第14军]]直属单位、[[中国人民解放军陆军第四十师|第40师]]转业军官、下放干部和职工及家属共计510人（实到505人），响应[[上山下乡运动|上山下乡运动]]的号召，在[[大理市|大理]]14军的军部举行成立勐省农场的大会，3月20日农场人员抵达勐省。此后陆续接收附近村民、退伍及复员军人、支边群众进入农场系统，到1960年底共有11个生产队和修配厂、砖瓦厂等多个下属单位，总人口增加到1,674人。1961年[[芒卡坝|芒卡坝]]农场合并进入勐省农场，1967年实行军事管制。1970年[[云南生产建设兵团|云南生产建设兵团]]成立，勐省农场被整编为'''云南生产建设兵团第二师第九团'''，属[[中国人民解放军云南省军区|云南军区]]建制，改行[[团_(军队)|团]]、[[营|营]]、[[连|连]]三级管理。到1973年末，因陆续接收退伍军人、[[知青|知青]]等类人群，勐省农场员工达到4,620人。1974年10月撤销兵团，恢复农场建制，现役军人撤离农场，实行场、队两级管理，由[[云南农垦集团|省农垦总局]]领导，场下设13个生产队，以及机务队、试验站、水泥厂、电站、加工厂、卫生队、石灰队、直属队。1979年恢复勐省农场名称，知青返城，到1990年总人口减为4,599人，下设35个单位，其中茶叶生产队15个、农业生产队4个、工业单位9个、场直属单位7个。2012年勐省农场共辖有农业公司、茶叶公司、电力公司、水泥厂、建筑队、医院等单位，农业公司下有4个生产队，茶叶公司下有14个生产队、2个初制茶所、1个茶厂（石佛洞茶厂）。<ref name="沧源佤族自治县志">{{cite book |author=沧源佤族自治县地方志编撰委员会 |date=1998年12月 |title=《沧源佤族自治县志》 |url= |location=[[昆明|昆明]] |publisher=云南民族出版社 |page= |isbn=7-5367-1498 |ignore-isbn-error=true |author-link= }}</ref>{{rp|204-205}}<ref name="2012年代码">{{cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2012/53/09/27/530927501.html |title=2012年统计用区划代码和城乡划分代码：勐省农场 |last= |first= |date=2012 |website= |publisher=[[中华人民共和国国家统计局|中华人民共和国国家统计局]] |access-date= |quote=}}</ref>

==行政区划==

勐省农场是[[沧源佤族自治县|沧源佤族自治县]]下属的一个[[类似乡级单位|类似乡级单位]]，由勐省农场管理委员会管理，2012年曾调整下属机构<ref name="2012年代码"></ref><ref>{{cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2013/53/09/27/530927401.html |title=2013年统计用区划代码和城乡划分代码：勐省农场 |last= |first= |date=2013 |website= |publisher=[[中华人民共和国国家统计局|中华人民共和国国家统计局]] |access-date= |quote=}}</ref>，现下辖3个[[社区_(行政区划)|社区]]：机关社区、农司社区、石佛洞社区。<ref>{{PRC admin/ref sgc|53/09/27/401/000}}</ref>

==生产==

勐省农场在沧源县农业区划上属于“东北部低热河谷粮、胶（紫胶）、蔗、林、牧区”<ref>{{cite book |author=沧源佤族自治县概况编写组、修订委员会 |date=2007年11月 |title=《沧源佤族自治县概况》 |url= |location=[[北京|北京]] |publisher=[[民族出版社|民族出版社]] |page= |isbn=978-7-105-08553-8 |ignore-isbn-error=true |author-link= }}</ref>{{rp|91}}，种植有[[水稻|水稻]]、[[玉米|玉米]]、[[稻#旱稻与水稻的不同|旱稻]]、[[杂粮|杂粮]]等粮食作物，其中以水稻为主。1959年试种双季稻，1983年开始种植[[杂交水稻|杂交水稻]]，1988年推行与蔬菜交替种植的“两年五熟制”种植法。[[茶叶|茶叶]]在农场的生产中也占有重要地位，其他农作物还有[[橡胶|橡胶]]、[[甘蔗|甘蔗]]、[[水果|热带水果]]、[[蔬菜|蔬菜]]等。<ref name="沧源佤族自治县志"></ref>{{rp|206-209}}

畜牧渔业方面，勐省农场饲养有[[猪|猪]]、[[水牛|水牛]]和[[黄牛|黄牛]]，1967年后开挖鱼塘养殖了多种淡水鱼。<ref name="沧源佤族自治县志"></ref>{{rp|209-210}}

勐省农场下设有多个工业生产单位。1958年红砖厂建成，1964年在[[拉勐河|拉勐河]]建成电站一座，1970年建成水泥厂一座，此外还有农机站、供销商业服务公司等机构，供销公司的商业服务在1988年转交给个人承包经营。<ref name="沧源佤族自治县志"></ref>{{rp|210}}
