'''鞍池停车区'''（平假名：くらがいけパーキングエリア）是位於[[愛知縣|愛知縣]][[豐田市|豐田市]]的[[東海環狀自動車道|東海環狀自動車道]]之[[休息區|休息區]]，由[[中日本高速公路|中日本高速公路]]管理。此休息區同時設置[[智能IC|智能IC]]。

== 連接道路 ==
* [[東海環狀自動車道|東海環狀自動車道]]

== 历史 ==
* 2005年3月19日 - [[東海環狀自動車道|東海環狀自動車道]][[美濃關JCT|美濃關JCT]]至[[豐田東JCT|豐田東JCT]]開通，此休息站也同時啟用。
* 2008年2月16日 - 開始試用智能IC。
* 2009年4月1日 - 智能IC正式啟用。
* 2010年12月1日 - 為了讓大型車輛使用智能IC，智能IC暫時關閉進行工程。
* 2011年3月25日 - 智能IC重開。
