'''影响指數'''（英文：'''{{lang|en|Impact factor}}'''、缩写：'''{{lang|en|IF}}'''），或譯'''-{zh-cn:数; zh-tw:影響因子;zh-hk:影響指數}-'''、'''影響係數'''<ref>{{Cite web |url=http://www.lib.ntu.edu.tw/node/1334 |title=常問問題- 什麼是Impact Factor？如何查詢？ |publisher= [[國立臺灣大學|國立臺灣大學]]圖書館 |accessdate=2014-03-27}}</ref>，是{{Link-en|美国科学情報研究所|Institute for Scientific Information}}（ISI）的[[期刊引证报告|期刊引证报告]]（JCR）中的一项数据。指的是某一期刊的文章在特定年份或时期被引用的频率，是衡量[[学术期刊|学术期刊]]影响力的一个重要指标。由美国科学情報研究所创始人{{link-en|尤金·加菲得|Eugene Garfield}}在1960年代创立，其后为[[文献计量学|文献计量学]]的发展带来了一系列重大革新。

自1975年以来，每年定期发布于《[[期刊引证报告|期刊引证报告]]》（{{lang|en|JCR}}）。

== 计算方法 ==
影响指數即某期刊前两年（S, T）发表的论文在统计当年（U）的被引用总次数X（前兩年總被引次數）'''除以'''该期刊在前两年（S, T）内发表的论文总数Y（前兩年總發文量）。这是一个国际上通行的期刊评价指标。公式為：IF<sub>U</sub> =（X<sub>(S,T)</sub> / Y<sub>(S,T)</sub>）。
*例如，某期刊2010年影响指數的计算
*本刊2009年的文章在2010年的被引次数：48；本刊2009年的发文量：187
*本刊2008年的文章在2010年的被引次数：128；本刊2008年的发文量：154
*本刊2008-2009的文章在2010年的被引次数总计：176 =（48+128）
*本刊2008-2009年的发文量总计：341 =（187+154）
*本刊2010年的影响指數：0.5161 = （176÷341）
*意义：该指标是相对统计值，可克服大小期刊由于载文量不同所带来的偏差。一般来说，影响指數越大，其学术影响力也越大。

=== 基本IF值计算方法 ===
*以1992年为例，計算某期刊在該年的影響指數：
*X＝以1992年為基點、某期刊於1990和1991年''出版论文在1992年'''''被引用之总次数'''
*Y＝以1992年為基點、某期刊1990和1991年全部'''论文發文量的总和'''
*IF<sub>1992年</sub> ＝（X<sub>(1990年,1991年)</sub>  / Y<sub>(1990年,1991年)</sub>）

== 對研究上的影响 ==
*许多著名学术期刊会在其网站上注明期刊的影响指數，以表明在对应学科的影响力。如[[美國化學會志|美國化學會志]]等。
