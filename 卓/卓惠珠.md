'''卓惠珠'''<ref>{{Cite web|url=https://helpasperger.blogspot.tw/2014/01/blog-post.html|title=花媽卓惠珠介紹|accessdate=2017-11-06|work=helpasperger.blogspot.tw}}</ref>（{{bd|1963年|8月20日|catIdx=}}），暱稱花媽，為[[部落格|部落格]]「幫助高功能自閉與亞斯伯格症理解人際互動困難和情緒障礙<ref>{{Cite web|url=https://helpasperger.blogspot.tw/|title=幫助高功能自閉與亞斯伯格理解人際互動困難和情緒障礙|accessdate=2017-11-06|work=helpasperger.blogspot.tw}}</ref>」版主，以及臉書粉絲頁「幫助高功能自閉與亞斯伯格症<ref>{{Cite web|url=https://www.facebook.com/aspergerhouse|title=幫助高功能自閉與亞斯伯格|accessdate=2017-11-06|author=|date=|work=www.facebook.com|publisher=|language=zh-hant}}</ref>」管理人，台灣媒體稱之為亞斯教母。育有一子一女，其子於國小確診輕度自閉。為[[自閉症|自閉症]]及[[亞斯伯格症候群|亞斯伯格症]]領域著名人士。<ref>{{cite web|author = 李屏瑤  |title = 《山不轉，我轉！》花媽卓惠珠：面對亞斯孩子，家長的表現就是最好的身教 |url = http://okapi.books.com.tw/article/8438 |website = 博客來OKAPI閱讀生活誌 |publisher = [[博客來數位科技股份有限公司|博客來數位科技股份有限公司]] |date = 2016-01-29 |accessdate = 2017-11-06 |language = zh-tw |archiveurl = https://web.archive.org/web/20171106184552/http://okapi.books.com.tw/article/8438 |archivedate = 2017-11-06 |deadurl= no}}</ref><ref>{{cite web |author = 李佩芬 |title = 為泛自閉家族發聲，花媽卓惠珠：我懂得這中間的苦 |url = https://www.parenting.com.tw/article/5049225-為泛自閉家族發聲，花媽卓惠珠：我懂得這中間的苦/ |website = 親子天下雜誌<!--45期--> |publisher = 親子天下 |date = 2013-05<!-- (已於2016-04更新)--> |accessdate = 2017-11-06 |language = zh-tw |archiveurl = https://archive.fo/20171106185210/https://www.parenting.com.tw/article/5049225-%E7%82%BA%E6%B3%9B%E8%87%AA%E9%96%89%E5%AE%B6%E6%97%8F%E7%99%BC%E8%81%B2%EF%BC%8C%E8%8A%B1%E5%AA%BD%E5%8D%93%E6%83%A0%E7%8F%A0%EF%BC%9A%E6%88%91%E6%87%82%E5%BE%97%E9%80%99%E4%B8%AD%E9%96%93%E7%9A%84%E8%8B%A6/ |archivedate = 2017-11-06 |deadurl = no }}</ref>

被台灣媒體稱為亞斯教母，人稱花媽的卓惠珠，育有確診輕度自閉，難以區分為高功能自閉或亞斯柏格症的兒子，現已投入社會工作，擔任程式設計師，深知養育這類孩子的不易，因此長期經營「幫助高功能自閉與亞斯柏格」社群部落格及粉絲頁，協助有相同困擾的家長以及孩子。

曾經為了兒子，在國小擔任長期代課老師，在學校輔導室、教務處、資源班都擔任過相關職務。出版過四本自閉症相關書籍，辦過上千場輕度自閉相關課程，每年在各級學校受邀150場左右演講，2011年獲得「新北社會文化貢獻獎」，2015年擔任「新北真人圖館館特教講師」，2016年獲得「台灣部落格大獎文化藝術類首獎」。

2018年，經營《[https://www.youtube.com/channel/UC5pKrPGnW45d6Ui3EvmWWSg/videos 聽花媽說說話]》影音頻道。
2018年，擔任「希望基金會」教育委員會主任委員。
2019年，擔任「中華健康生活與運動協會」理事長。
