'''離心機'''是一種[[機械|機械]]，可藉由[[電動機|電動機]]或其他機械的帶動而高速轉動，產生數千倍於[[重力|重力]]的[[離心力|離心力]]，以加快[[液體|液體]]中顆粒的沉降速度，把樣品中不同[[沉降系數|沉降系數]]和[[密度|密度]][[質量|質量]]的物質分離。離心力的大與小，轉動速度、旋轉半徑岱以及物質的[[融質量|融質量]]而決定。<ref> [www.ch.ntu.edu.tw/~genchem99/doc/08T8.pdf]</ref>離心機廣泛運用於[[化學工程|化學工程]]、[[石油|石油]]、[[食品加工|食品加工]]、[[制藥|制藥]]、[[選礦工程|選礦工程]]、[[炭|炭]]、[[水處理|水處理]]、[[核能工業|核能工業]]和[[船舶|船舶]]等部門。 

==歷史==
* 早期應用—[[中國|中國]]古人會取一端套住陶罐的繩子並甩動以產生離心力，擠出陶罐裏的[[蜂蜜|蜂蜜]]。
* 近代—19世紀中葉的[[歐洲|歐洲]]，首先出現[[紡織品|紡織品]]脫水的三足式離心機，和[[制糖|制糖]]廠分離砂糖結晶用的上懸式離心機。這些早期離心機雖尚未全自動化，需以人力進行排渣，卻已有了現代離心機雛形。 

==原理==
利用向心[[加速度|加速度]]公式得知，[[密度|密度]]較大之物體在下，密度較小之物體在上。

*  '''離心過濾'''，使懸浮液受離心力，而對過濾介質施加壓力，液體通過過濾面，固體顆粒被截下，從而液-固分離。
* '''離心沉降'''，利用懸浮液(或乳濁液)密度不同的各成份在離心力場中迅速沉降分層的原理，以達成液-固(或液-液)分離。

===離心力計算===

'''相對離心力'''（RCF, relative centrifugal force)，以[[重力加速度|重力加速度]]<math>g</math>的倍數來表示；

[[每分鐘轉速|rpm]](revolution per minute，或r/min)表示離心機每分鐘的轉數。'''rpm'''與<math>g</math>之間的換算公式為
 
：'''RCF''' <math>= 1.119 \times 10^{-5} \times rpm^2  \times r</math>

其中<math>r</math> 表示離心機轉軸中心與離心管中心的距離，單位為[[公分|公分]]。
離心管的位置由[[轉子|轉子]]（rotor）決定，因此<math>r</math> 必須查閱相關轉子的參數而得。 

==應用==
===日常===
*烹飪時的食物處理，近來一些廚師發現離心機是很方便實用的廚房用品
*衣物脫水（[[洗衣機|洗衣機]]、[[脫水機|脫水機]]）

===實驗用===
* [[遺傳工程學|遺傳工程學]]、[[酶工程學|酶工程學]]
批次離心機(或瓶式離心機)是實驗室常用的離心機型。如果具超強之離心力，則可應用於超離心技術(微分離心技術)。
此即分離[[混合物|混合物]]中各成份，已成功應用於[[細胞|細胞]]內胞器如[[粒線體|粒線體]]、[[微粒體|微粒體]]、[[溶酶體|溶酶體]]等之分離，或各種[[蛋白質|蛋白質]]及[[核酸|核酸]]之分離。超離心技術是研究遺傳工程學、酶工程學必須的基礎。
* [[同位素|同位素]]分離
以[[氣體離心法|氣體離心法]]進行[[同位素|同位素]]的加工製造，核武所需的[[濃縮鈾|濃縮鈾]]即可藉由此方法製造出來，因此離心機在國際間列為受管制的設備。
* 航空技術
以離心力模擬[[重力加速度|重力加速度]]，以探求生物在極端狀況下體內物件位移。<ref>[[Mary_Roach|Mary Roach]], "Packing For Mars" (2012)</ref>

===工業用===
此作動機械雖泛稱遠心分離機，為區分工業用機械，取之'''離心式分離機'''或'''離心式[[過濾|過濾]]機'''，離心機應用極廣，切削液過濾系統主要用途是分離[[切削液|切削液]]/[[冷卻液|冷卻液]]裡在[[研磨|研磨]]加工過程中所產生的雜質與切屑，可分離的雜質不限於金屬，非金屬材質如: [[矽|矽]]晶、玻璃、[[石墨|石墨]]、不鏽鋼、[[石英|石英]]、鎢鋼、[[粉塵|粉塵]]...等非磁性切屑也可過濾。藉由[[離心力|離心力]]將雜質高速分離，使切削液能迅速的重複利用。不僅提升加工品質也友善環境。經由離心力分離的設計結構，可減少耗材濾袋的使用，具備[[環保|環保]]與綠色經濟的設計是現今產業的發展趨勢。<ref>{{cite web |author1=ChinMinn |title=全自動快速除屑離心式過濾機 |url=https://www.chinminn.com/zh-tw/product/centrifugal-filtration/automatic-chip-removal-centrifugal-machine |website=勤閔 |accessdate=2019-03-22}}</ref><ref>{{cite web |author1=楊聰橋 |title=勤閔全自動離心式過濾機 有效淨化切削液 |url=https://money.udn.com/money/story/6092/3674244 |website=經濟日報 |accessdate=2019-03-07}}</ref>

*[[蜂蜜|蜂蜜]]採集：將蜂巢箱內的蜂巢置入離心機之中，以手搖方式轉動，蜂巢中的蜂蜜會甩出而流入收集器之中。

==機器特性==
衡量離心分離機分離性能的重要指標是'''分離因數'''。表示為被分離物料在轉鼓內所受的[[離心力|離心力]]與其[[重力|重力]]的[[比值|比值]]，分離因數越大，通常分離越快，效果越好。

'''最大轉速'''，離心機在空載情況下的轉速。根據'''最大轉速'''的不同，離心機可更細分為：
# 低速離心機         ''10000 rpm· min <sup>-1</sup>''
# 高速離心機         ''10000 rpm · min一30000 rpm · min <sup>-1</sup>''
# 超高速離心機      '' 30000 rpm ·min<sup>-1</sup>''
