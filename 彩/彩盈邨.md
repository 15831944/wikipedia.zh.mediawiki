{{engname|彩盈邨|Choi Ying Estate}}是[[香港房屋委員會|香港房屋委員會]]轄下一個[[公共屋邨|公共屋邨]]，位於[[九龍|九龍]][[觀塘區|觀塘區]][[平山_(香港)|平山]]，由房屋署總建築師（3）設計，於2008年落成入伙，目前由[[卓安物業顧問有限公司|卓安物業顧問有限公司]]負責屋邨物業管理。

==簡介==
彩盈邨是「[[平山_(香港)|彩雲道及佐敦谷毗鄰的發展計劃]]」<ref>[http://www.cedd.gov.hk/tc/projects/major/kw/klnkaeb564cl.htm 土木工程拓展署]{{dead link}}</ref><ref>[http://www.hk-place.com/vp.php?board=1&id=2168-6 佐敦谷地盤平整工程照片]</ref>的一部分，根據該項計劃，政府在[[彩雲道|彩雲道]]及[[佐敦谷|佐敦谷]]一帶闢建地台，興建道路、行車天橋、行人天橋、雨水渠及污水渠等基本設施，並會開闢綠化地帶及改善附近地區現有的道路交匯處，並興建房屋、小學及中學的用地<ref>[http://www.dchome.net/viewthread.php?tid=461480 佐敦谷(發展計劃地盤)] - 風景區 dchome.net數碼天地論壇</ref>。

彩盈邨於2008年中落成入伙，共有五幢樓宇，提供約4,000個公屋單位，並設有一個小型商場「彩盈坊」，以及一個停車場。鄰近的屋邨有[[彩德邨|彩德邨]]、[[彩雲邨|彩雲邨]]、[[彩霞邨|彩霞邨]]、[[啟業邨|啟業邨]]、[[彩福邨|彩福邨]]等。

== 獲獎 ==
彩雲道的公共房屋發展項目在[[香港工程師學會|香港工程師學會]]和英國結構工程師學會聯合結構分部舉辦的「卓越結構大獎」中，奪得嘉許獎。

2011年3月28日是彩雲道的公共房屋發展項目最後一期建築工程在當日完成，涉及彩盈邨、[[彩福邨|彩福邨]]及[[彩德邨|彩德邨]]三個屋邨。[[香港房屋委員會|香港房屋委員會]]成功把荒廢的石礦場改變為13,000個居所，成為能容納超過35,000人居住的社區。經過規劃設計後，屋邨居民可以在綠意盎然的自然環境中，經過四通八達的行人通道，享用各式各樣的設施和綠化空間。

[[香港房屋委員會|香港房屋委員會]]在設計階段進行廣泛的環境研究，確保該項目的整體發展與四周環境協調。此外，他們亦顧及其他主要的設計問題，例如保留伸延至鄰近發展項目的觀景廊和風道。置身在這個發展項目中，天地和樹木等自然景物色彩分明，令三個屋邨與周遭環境融合之餘，又各具特色。總括而言，這項目象徵了房委會的良好公屋建築方針。<ref>{{Cite web |url=http://www.info.gov.hk/gia/general/201106/27/P201106270244.htm |title=房屋署署長在房屋委員會公開例會的發言全文 |date=2011-06-27 |author=香港政府新聞公報}}</ref>

== 商場 ==
彩盈邨設有小型商場，命名為「'''彩盈坊'''」。商場採用街舖式設計，並無裝設冷氣。場內合共只有5間商舖，包括[[百佳超級市場|百佳超級市場]]、[[OK便利店|OK便利店]]、一間診所及[[銀龍茶餐廳|銀龍茶餐廳]]，另一間商舖則用作商場管理處<ref>[http://www.housingauthority.gov.hk/hdw/content/document/b5/commercial/shoppingcentres/choiying.pdf 房委會 彩盈坊招商廣告]</ref>

== 事故 ==
* 2009年5月7日，彩盈邨盈康樓38樓一單位發生火警，一名76歲男住客吸入濃煙死亡<ref>[http://cablenews.i-cable.com/webapps/news_video/index.php?news_id=302377 牛頭角彩盈村火警老翁死亡]有線新聞</ref>。
* 2018年6月20日，彩盈邨盈順樓，上午11時左右，一名16歲少年爬出35樓的單位窗外，危坐外牆簷篷，家人連忙報警求助。警方及消防員趕到游說，並召談判專家協助。消防員在地面張開救生氣墊，同時在樓上單位準備飛將軍。少年曾經傾手提電話3至4次，至下午1時半左右，身處樓上單位的消防飛將軍，爬出窗外準備空降之際，少年仰望瞥見，竟然立即一躍而下，跌在地下梯級的欄杆，再反彈墮地，距離氣墊不足兩米，他的右腳掌折斷甩脫，當場重創昏迷，送院證實不治。警方正了解他不開心的原因，事發時少年的母親一直在單位內，她事後傷心激動，由親友陪同送院檢查。
