{{彩色電視廣播標準列表|[[洲|洲域]]}}世界各地區之電視系統規格差異極大，所做一個歸類。 <ref group="參">{{cite web|title=國立臺灣師範大學教育學院工教系 莊謙本先生的報告|url=http://140.122.100.145/ntnuj/j31/j31.asp?appl=j31-9.pdf}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref group="參">{{cite web|title=國立臺灣師範大學 師大學報|url=http://www.ntnu.edu.tw/acad/pub/j31/j31-9.htm|deadurl=yes|archiveurl=https://web.archive.org/web/20100412085843/http://www.ntnu.edu.tw/acad/pub/j31/j31-9.htm|archivedate=2010-04-12}}</ref>

世界上主要有五種格式，第一種是類比電視系統；第二種是數位地面廣播電視系統；第三種是數位有線電視系統；第四種是衛星電視系統；第五種是網路協定電視。

=== PAL ===
[[File:PAL-NTSC-SECAM.svg|thumb]]
* Phase Alternative Between Line[[西德|西德]]開發，於1967年成立。
* 除了北美，東亞部分地區使用NTSC制式，中東、法國及東歐採用SECAM制式以外，世界上大部份地區都是採用PAL。
* 有PAL、PAL-B、PAL-D、PAL-G、PAL-H、PAL-I、PAL-K、PAL-K1、PAL-N、PAL-M，10種格式。

=== NTSC ===
* National Television System Committee[[美國|美國]]開發，於1953年成立。
* 美國、加拿大、墨西哥等大部分美洲國家以及臺灣、日本、韓國、菲律賓等均採用這種制式。
* 有 NTSC、NTSC-C、NTSC -J、NTSC -M、NTSC -U、NTSC -Us，6種格式。
=== SECAM ===
* Se'quential Co'uleur A Me'moire[[法國|法國]]開發，於1967年成立。
* 採用SECAM制的國家主要為俄羅斯、法國、埃及以及非洲的一些法語系國家等等。
* 有 SECAM、SECAM -B、SECAM -D、SECAM -G、SECAM -K、SECAM -L，6種格式。

== 數位地面廣播電視系統  ==
[[File:Digital_broadcast_standards.svg|thumb]]
* 為數位電視節目地面整合式數位廣播有四個系統使用中。
* 中國(包括香港和澳門) 選擇了 DMB-T/H (數位多媒體廣播Terrestrial/Handheld) 當 DTV 標準。
* 在日本ISDB-T ([[地面電視|地面廣播電視)]] 系統由[[電波產業會|電波產業會]]標準化 (Association of Radio Industries and Businesses) (ISDB formats).
* A system which uses an [[8_VSB|8 VSB]] modulation (8 水平殘邊帶) for terrestrial broadcasting, standardized by ATSC, has been realized in the USA ([[ATSC_制式|ATSC 制式]]).
* The DVB Project has been realized in Europe by EBU (歐洲廣播聯盟)、ETSI (European Telecommunications Standards Institute) and the CENELEC (European Committee for Electrotechnical Standardization) (DVB formats MPEG-2/H.264 AVC). For terrestrial broadcasting the DVB-T (Digital Video Broadcasting Terrestrial) standard is used.
* 美規 ATSC 8-VSB 用於 6MHz 電視頻道。 
* 歐規 DVB-T COFDM 用於 6/7/8 MHz 電視頻道，目前台灣採用的是歐規 DVB-T 6MHz 。 
* 日規 ISDB-T COFDM 用於 6MHz 電視頻道 ( 屬歐規改良型 ) 。
* 中規 DMB-T/H TDS-OFDM 用於 8MHz 電視頻道 ( 屬歐規改良型 ) 。
