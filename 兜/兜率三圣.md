'''兜率三圣'''，又称'''弥勒三尊'''、'''弥勒三圣'''，是[[兜率天|兜率天]]的三位菩萨，主尊为[[弥勒佛|弥勒佛]]，左胁侍[[法华林菩萨|法华林菩萨]]，右胁侍[[大妙相菩萨|大妙相菩萨]]<ref>[http://buddhaspace.org/dict/fk/data/%25E5%25BD%258C%25E5%258B%2592%25E4%25B8%2589%25E5%25B0%258A.html 弥勒三尊]</ref><ref>{{cite web |url=http://www.pusa123.com/pusa/news/dujia/201122495.shtml |title=雪窦禅寺举行“兜率三圣”佛像开光仪式 |last= |first= |date=2011-09-20 |website=菩萨在线 |publisher=果云 |access-date= |quote=}}</ref>。

自[[宋代|宋代]]的[[大肚弥勒佛|大肚弥勒佛]]出现之后，兜率三圣在中国寺院中较罕见<ref>{{cite web |url= https://fo.ifeng.com/tupian/sheyingjia/detail_2013_06/08/26237150_8.shtml|title=奎子作品：新加坡最富丽堂皇的佛寺 佛牙寺 |last= |first= |date=2013-06-08 |website=凤凰网 |publisher= |access-date=2018-09-23 |quote=}}</ref>。在[[海南省|海南省]][[三亚市|三亚市]]的[[南山寺_(三亚)|南山寺]]、[[浙江省|浙江省]][[宁波市|宁波市]]的[[雪窦寺|雪窦寺]]和台湾[[南投县|南投县]][[大慈山弥勒道场|大慈山弥勒道场]]均供奉有兜率三圣。

== 形象造型 ==
*弥勒佛：
**头戴天冠，呈倚坐像。
**结[[手印|说法印]]，两手持莲花，右边莲花上有[[法轮|法轮]]，左边莲花上有贤瓶。

*法华林菩萨：
**右手持金轮，左手扶住轮的一角。

*大妙相菩萨：
**右手执[[毛笔|毛笔]]，左手持莲枝<ref>[http://www.dts.org.tw/3f.htm 三樓　大慈寶殿]</ref><ref>[http://xn--zqsv1f7yfwxhm2f.tw/%e6%84%9f%e6%81%a9%e8%8d%89%e9%9b%b2%e5%b8%ab%e5%a7%8a%e4%be%9b%e9%a4%8a%e6%85%88%e6%b0%8f%e3%80%81%e6%91%ba%e7%96%8a%e5%bc%8f%e3%80%90%e5%85%9c%e7%8e%87%e4%b8%89%e8%81%96%e3%80%91%e8%81%96%e5%83%8f/ 感恩草雲師姊供養慈氏、摺疊式【兜率三聖】聖像]</ref>。
