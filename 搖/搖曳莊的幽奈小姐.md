《'''湯搖莊的幽奈同學'''》（{{lang-ja|ゆらぎ荘の幽奈さん}}） 是由[[三浦忠弘|三浦忠弘]]所繪製的[[日本漫畫|日本漫畫]]，自2016年10號在《[[週刊少年Jump|週刊少年Jump]]》上連載。同年七月為止單行本銷量已達七萬本，此後以驚人的銷售速度，快速累積單行本，同年9月底，銷量已突破50萬本，5卷書腰表示系列突破100萬本<ref>[http://chomanga.com/archives/49049102.html 搖曳莊的幽奈小姐] {{webarchive|url=https://web.archive.org/web/20160806075345/http://chomanga.com/archives/49049102.html |date=2016-08-06 }}</ref>。2017年11月13日宣布動畫化<ref>{{Cite web|url=http://natalie.mu/comic/news/256431|script-title=ja:ミウラタダヒロのお色気ラブコメ「ゆらぎ荘の幽奈さん」アニメ化|date=2017-11-13|website=漫畫Natalie|accessdate=2017-11-13|language=ja}}</ref>，於2018年7月播放。

== 故事大綱 ==
擁有[[通靈|通靈]]能力的男主角'''冬空凩'''，入住了温泉公寓'''搖曳莊'''，在他的房間中有一位名為'''幽奈'''的[[地縛靈|地縛靈]]，除此之外、搖曳莊裡還住著多位各有魅力的美女。此後，冬空凩與諸多女性充滿笑料與鬧劇的生活便開始了……

=== 搖曳莊 ===
; {{nihongo|冬空凩|冬空 コガラシ（ふゆぞら コガラシ）|[[配音員|聲]]：[[小野友樹|小野友樹]]<ref name="character">{{cite web|url=https://yuragisou.com/character/|title=CHARACTER|work=電視動畫「湯搖莊的幽奈同學」官方網站|language=ja}}</ref>}}
: 本作的男主角，個性豪爽正直，喜欢打抱不平，生來便有[[通靈|通靈]]體質（容易被附身並且對於詛咒等負面性的術抗性超弱）並且非常困擾，虽然因此获得了無數的技能，但也因为被炒股上瘾的幽灵附体后身负巨债，一直居无定所。后来拜師學了除靈的方法，避免自身再度被幽靈隨意附身，自稱肉體派靈能師，第七代的「八咫鋼」。因为搖曳莊的老板承诺如果能成功除灵就让其终身免费居住，所以搬进了搖曳莊204号房。
: 自身的信条是不打女人，所以无法通过擅长的将对方痛扁一顿的方法除掉搖曳莊4號房的地缚灵（幽奈），只好与其同居。就读于汤烟高中1年4班，放学后会去打工赚取生活费。
: 由於曾被極爲強大的靈能師幽靈強迫收為弟子，於是經歷了地獄般的修行和試煉，最後讓冬空得到了連師傅都能打倒的強大神力跟肉體，就算是神級妖怪都能被其以拳頭一擊打倒。缺點是沒有學會其他術法，只能藉由毆打的方式除靈，一旦陷入不能揮拳的狀態就無計可施。對詛咒與咒術的抗性也很差，不擅長應付變身術，因為一旦被變成沒有手腳的物品就完全失去抵抗能力。
: 其師傅為御三家中「八咫鋼」家的後裔。而在八咫鋼家已絕後的現在，冬空成為唯一的傳人，因此在妖怪間被稱呼為八咫鋼。

: 不能通灵的人看不到幽奈的存在，也无法交谈，但可以用笔进行交流；通灵能力很弱的人类眼中，她只是一团白影（拍下的照片中也是如此），且声音如魔鬼一般，因此对其十分恐惧；能通灵的人则可以看到其高中女生的外貌，并能和其说话。遇到凩后，与凩达成了“凩帮助幽奈了却对世间的留念后，幽奈就升天成佛”的协议，开始了同居生活後在一連串事件後逐漸對凩產生感情並愛上凩。

: 其真正身份為'''天狐白叡'''以自己的女兒'''天狐幻'''為原型所創造的第七代「{{nihongo|'''天狐幻流齋'''|てんこ げんりゅうさい}}」，因無法繼承餓爛洞術式而被白叡視為失敗之作。

: 統帥[[信濃|信濃]]龍雅湖的黑龍神，是個黑髮的好色青年，性格幼稚，成天尋找美女做自己的妻子。曾将幽奈绑架回来，想让她做自己的妻子，但被凩阻止，之後便長期待在湖底龍宮苦修，鮮少出現在外界。出關後和凩於角鬥場內來了一場巔峰對決，戰敗，期望姊姊能夠得到屬於自己的幸福。

; {{nihongo|緋扇迦樓羅|緋扇 かるら（ひおうぎ かるら）}}
:統領西區妖怪軍團的[[大天狗|大天狗]]，實力超群足以1人輕鬆擊敗搖曳莊的實力，身懷神足通/他心通/天眼通絕技，關原之戰後迷上凩並把凩視為丈夫第一人選，在綁架凩逼婚失敗後努力以朋友方式逐漸攻略中。

; {{nihongo|巳虎神魔鶇|巳虎神 マトラ（みこがみ まとら）}}
:京都大妖怪鵺，西區妖怪軍團第一戰將，擁有超越朧的速度與宵之坂的力量，但對於瞬移一類的術並不擅長，一旦被丟在某處就只能借助交通工具往返。

; {{nihongo|[[土蜘蛛|土蜘蛛]]|土蜘蛛（つちぐも）}}
:被封印在橫濱山道的靈石中的大妖怪，擁有浦方家的最強式神都為之恐懼的強大靈力，疑似被幽奈生前所封印。當他見識到凩的強大實力後，自願回到靈石的封印中。

; {{nihongo|葛城米莉亞|}}
: 居住在東京一所豪宅的女性狐妖，與小柚同年。知道幽奈真正的身份。

;{{nihongo|緋扇我流馱|緋扇 我流馱（ひおうぎ がるうだ）}}
: 緋扇迦樓羅的父親，原本在關原之戰前是個性情勇猛剛烈的妖怪，但在凩的介入戰爭之後，性格變得溫和許多。現時擔任緋扇學園理事長，十分溺愛迦樓羅。

; 餓爛洞
: 傳說中的大惡靈。曾在百年前的日本肆虐，連魔境院逢牙也喪命於其手上。其真正身份為天狐家的初代當家'''天狐白叡'''創造的術式。

; {{nihongo|宵之坂釀之介|}}
: 吞子的父親。為御三家中西之「宵之坂」當家。西方陣營妖怪的統帥。

; {{nihongo|宵之坂酌人|}}
: 吞子的弟弟。

=== 其他人類 ===
;{{nihongo|白露蓮華|白露 蓮華（しらつゆ れんげ）}}
: 拉菲列女子学院会长兼高等部二年级生，為知名財閥白露集團會長的孫女。因為外型優雅深受全校憧憬。經過惡靈事件被冬空凩解圍後，對其有好感。

;{{nihongo|森野古湖乃|森野 古湖乃（もりの ここの）}}
: 拉菲列女子学院会書記，宮崎千紗希的兒時好友。委託冬空凩等人至學校除靈。

;{{nihongo|羽良嶼累|羽良嶼 累（はらしま るい）|聲：[[渡邊優里奈|渡邊優里奈]]}}
: 荒霸吐吞子的女編輯，留著覆蓋左眼的短髮，認為冬空凩是名十分認真的少年。

;{{nihongo|宮崎日和|宮崎 日和（みやざき ひまり）|聲：[[日笠陽子|日笠陽子]]}}
: 宮崎千紗希的母親，個性和藹可親，喜歡可愛事物，千紗希的容貌和身材皆遺傳自她。曾邀請搖曳莊成員們至家裡做客，以此契機對信樂小袖產生喜愛，還和千紗希教導雲雀豐胸的秘訣。
