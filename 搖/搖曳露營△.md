{{日本作品|搖曳露營△|ゆるキャン△}}是[[日本|日本]][[漫畫家|漫畫家]]{{tsl|ja|あfろ}}的[[漫畫|漫畫]]作品。以[[山梨縣|山梨縣]]一帶作舞台，描述[[露營|露營]]中的知識、野炊等戶外活動的魅力，以及以此作為娛樂的女高中生的溫和日常生活。

== 故事簡介 ==
=== 本篇「搖曳露營△」 ===
喜歡在淡季進行單人露營活動的'''志摩凜'''在一次於本栖湖的露營中，遇到了為一覽富士山而騎腳踏車來到本栖湖，但因睡著而陷入困境的'''各務原撫子'''。撫子在與凜交流過後對露營產生了興趣而加入學校的「野外活動同好會」，與同好會成員'''大垣千明'''與'''犬山葵'''開始了團體露營的活動。期間凜的好友'''齊藤惠那'''也因此對露營產生了興趣。故事便圍繞著「凜的單人露營」以及「野外活動同好會的露營」間展開。

=== 番外篇「房間露營△」 ===
因為想把露營當作社團活動，'''大垣千明'''和'''犬山葵'''成立了'''「野外活動同好會」'''，但因為活動預算金少得不足夠買齊露營所需用具，所以在創社的四月直至十一月都沒有什麼實際的活動紀錄。後來在新社員'''各務原撫子'''加入後，「野外活動同好會」終於開始了像樣的社團活動。

放學後到社辦研究露營的冷知識、玩起露營遊戲、聊著各種妄想的話題，幻想著10年、60年、1000年後<ref>「房間露營△」：放學後24、25、31</ref>的露營活動，故事就圍繞在社團每天的日常活動間展開。

== 劇中設定 ==
=== 故事的舞台相關 ===
: 本作以角色'''「住在山梨縣一帶」'''及'''「就讀本栖高中」'''為主要設定：
* 作品中'''露營場'''及'''溫泉設施'''、'''戶外活動用品店'''及'''超商'''、'''網站名稱'''等具有現實名稱的商標均以'''「架空的名稱」'''頂替掉。
* 有保留現實中'''觀光景點'''、'''地名'''、'''道路名'''及'''標誌'''的名稱。
* 主要登場人物的'''住家'''，是以實際'''沒有建築物'''的土地作設計藍圖。
* 作中雖然沒有作為設計本栖高中的原型，但上學路是參考現實'''身延町立下部中・小學（已廢校）'''的舊地一帶所設計。
* 角色前往購物的'''「超商 ZEBRA」'''，是以現實身延町的購物中心'''「{{tsl|ja|フレスポみのぶ}}」'''中的'''「SELVA 身延店」'''作為設計原型。
* 千明打工的'''「酒の川本 身延店」'''，是以上述所提及的身延町購物中心地區內曾經營過的一家酒品專賣店作原型設計。

: 就如電視動畫版第4話中滔滔不絕地發出提問的撫子一樣。
: 收集聽眾構想的感覺'''撫子可能會問到的無關緊要的問題'''，再強制原紗友里來回答。

* '''搖曳露營△[[俳句|５７５]]'''（音泉廣播獨有）
: 以「搖曳露營△」為主題，收集聽眾所構想的有關露營的'''５７５俳句'''{{notetag|５７５：俳句的特定格式，必須由五、七、五共十七個'''日文音'''組成。}}。
: 例如：{{ruby|看到了高山|山(やま)を見る}}　{{ruby|發著呆地瞭望著|ボーッと眺めて}}　{{ruby|已到了日落|日が暮れる}}
