'''庾凝绩'''（{{bd|9世纪||10世纪|catIdx=Y}}），[[五代十国|五代十国]]时期[[前蜀|前蜀]][[官员|官员]]。开国皇帝[[王建_(前蜀)|王建]]末年，任[[枢密使|枢密使]]，并在继任者末代皇帝[[王衍_(前蜀)|王衍]]在位初期继续有权。

== 背景 ==
庾凝绩家世不详，仅知为更知名的官员、在两位皇帝年间官拜[[宰相|宰相]]的[[庾传素|庾传素]]的再从弟。<ref name=SGCQ41>[https://archive.org/stream/06080952.cn#page/n160/mode/2up 《十国春秋》卷四十一]</ref>

== 王建年间 ==
王建年间，庾凝绩曾任[[翰林学士|翰林学士]]。<ref name=SGCQ41/>[[天汉_(前蜀)|天汉]]元年（917年）七月，庾凝绩时任翰林学士承旨，[[司徒|司徒]]、判枢密院事[[毛文锡|毛文锡]]正要把女儿嫁给庾传素的儿子，在枢密院聚集亲族奏乐而没有预先上表说明，被当权宦官飞龙使[[唐文扆|唐文扆]]弹劾。八月，王建贬毛文锡，罢庾传素为[[工部尚书|工部尚书]]，以庾凝绩权判内枢密院事。闰十月，任庾凝绩为[[吏部尚书|吏部尚书]]、内枢密使。[[光天_(王建)|光天]]元年（918年）五月，王建病重，他意欲以[[皇太子|皇太子]][[王宗衍|王宗衍]]托孤的养子马步都指挥使[[王宗弼|王宗弼]]指控唐文扆意欲夺权，王建贬唐文扆，招庾凝绩专管中外财赋、中书除授、诸司刑狱案牍，由宣徽南院使宦官[[宋光嗣|宋光嗣]]专管都城及行营军旅之事。王建临终不久又任宋光嗣为内枢密使，未详庾凝绩是否解职。六月王建死后，王宗衍继位，改名王衍。王建遗诏指定宋光嗣、王宗弼及其他养子[[王宗瑶|王宗瑶]]、[[王宗绾|王宗绾]]、[[王宗夔|王宗夔]]辅政，没有以庾凝绩辅政。<ref name=ZZTJ270>[[:s:資治通鑑/卷270|《资治通鉴》卷二百七十]]</ref>

== 王衍年间 ==
王衍继位后即杀唐文扆，宰相[[太傅|太傅]]、[[门下侍郎|门下侍郎]]、[[同平章事|同平章事]][[张格|张格]]也受到连累被贬[[维州|维州]]司户，庾凝绩奏徙张格于合水镇，令[[茂州|茂州]][[刺史|刺史]][[顾承郾|顾承郾]]寻张格罪过。王衍养兄[[王宗侃|王宗侃]]妻张氏因和张格同姓，欲保全他，对顾母说：“警告你儿子，不要替他人报仇，他日他人将归罪于你。”顾承郾从之。庾凝绩怒，在公事中寻隙归罪顾承郾。<ref name=ZZTJ270/>其睚眦必报如此。<ref name=SGCQ41/>

此后没有庾凝绩的记载。

== 评价 ==
*《十国春秋》论曰：凝绩以修却为能，度量不无少损。<ref name=SGCQ41/>
