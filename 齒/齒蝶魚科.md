'''齒蝶魚科'''為[[輻鰭魚綱|輻鰭魚綱]][[骨舌魚目|骨舌魚目]]的其中一[[科|科]]，本科僅有一屬一[[種|種]]，即'''齒蝶魚(''Pantodon buchholzi'')'''。

== 分布 ==
本魚分布於[[非洲|非洲]]中部及西部，包括[[貝南|貝南]]、[[中非|中非]]、[[喀麥隆|喀麥隆]]、[[剛果民主共和國|剛果民主共和國]]、[[尼日|尼日]]、[[加彭|加彭]]、[[剛果|剛果]]、[[查德|查德]]、[[奈及利亞|奈及利亞]]、[[塞拉利昂|塞拉利昂]]等。

== 特徵 ==
本魚體色為金黃色，沿體側有褐色亮斑和一道模糊的橫條紋。帶有斑紋的胸鰭生得相當大，從上往下看宛如[[蝴蝶|蝴蝶]]的翅膀。背鰭和臀鰭後位，臀鰭很大，尾鰭的中間鰭條向外延伸，腹鰭有細線狀鰭條；嘴大而上翻，一道深色條紋從嘴上方斜穿過[[眼睛|眼睛]]，側線鱗片26至30枚，背鰭軟條6枚；臀鰭軟條9至15枚，體長可達12公分。

== 生態 ==
本魚棲息在在沼澤、湖泊、小溪等平靜水域, 主要活動在水面表層，能用寬大的胸鰭離開水面跳躍, 尋找[[昆蟲|昆蟲]]或從掠食者中脫逃,屬雜食性，以[[昆蟲|昆蟲]]、[[甲殼類|甲殼類]]動物和[[魚類|魚類]]等為食。

== 經濟利用 ==
為觀賞魚類，飼養時魚缸須加蓋，以防魚跳出。
