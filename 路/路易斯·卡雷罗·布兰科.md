'''路易斯·卡雷罗·布兰科'''（{{lang-es|Luis Carrero Blanco}}，{{bd|1904年|3月4日|1973年|12月20日|catIdx=Blanco, Carrero}}），前西班牙海军上将，佛朗哥的密友和心腹。1973年12月20日死于[[埃塔|埃塔]]组织的暗杀。<ref>{{Cite web |url = http://news.163.com/05/1216/22/254INLQR00011EBF.html |title = 1973年12月20日 西班牙总理布兰科被炸身亡 |author =  |date = 2005-12-20 |publisher = 网易历史 |language = 中文 |accessdate = 2017-02-20 }}</ref>有人认为此次暗杀“改变了西班牙的历史”。<ref>{{Cite web |url = http://www.cankaoxiaoxi.com/mil/20160406/1118973.shtml |title = 西班牙如何终结独裁政权？美媒：炸死佛朗哥接班人 |author = 董磊 |date = 2016-04-06 |publisher = 参考消息网 |language = 中文 |accessdate = 2017-02-20 }}</ref>
==人物生平==
路易斯·卡雷罗·布兰科1904年3月4日出生于[[西班牙|西班牙]][[坎塔布里亚|坎塔布里亚]][[桑托尼亚|桑托尼亚]]。后进入海军学院，1918年是参加了[[里夫战争|里夫战争]]。

1929年时，路易斯·卡雷罗·布兰科与María del Carmen Pichot y Villa 结婚，他们先后有了五个孩子。<ref>[http://www.geneall.net/H/per_page.php?id=467858 Luis Carrero Blanco, 1. duque de Carrero Blanco] {{es icon}}</ref>

[[西班牙内战|西班牙内战]]爆发后，路易斯·卡雷罗·布兰科加入佛朗哥一派。
