'''路易斯斯密斯湖'''（{{lang-en|'''Lewis Smith Lake'''}}）是[[美国|美国]][[阿拉巴马州|阿拉巴马州]]北部的一个水库。它位于[[黑沃里尔河|黑沃里尔河]]支流锡普西河的上游，湖水面积超过 {{convert|21200|acre|km2}}。<ref name="odala">[http://www.outdooralabama.com/fishing/freshwater/where/reservoirs/smith/ Smith Lake - Freshwater Fishing in Alabama's Reservoirs<!-- Bot generated title -->] {{webarchive|url=https://web.archive.org/web/20071222075702/http://www.outdooralabama.com/fishing/freshwater/where/reservoirs/smith/ |date=2007-12-22 }}</ref>该湖平均深度为{{convert|66|ft|m}}，最深处达到{{convert|264|ft|m}}，水面海拔高度为{{convert|510|ft|m}}。<ref name="alap">{{cite web |url=http://www.alabamapower.com/lakes/smith.asp |title=存档副本 |accessdate=2012-07-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20120321234055/http://www.alabamapower.com/lakes/smith.asp |archivedate=2012-03-21 }}</ref>湖体有三个分叉，长达 {{convert|500|mi|km}}  <ref name="odala"/><ref name="alap"/>的湖岸线分布在[[卡尔曼县|卡尔曼县]]，[[沃克县_(亚拉巴马州)|沃克县]]和[[温斯顿县_(亚拉巴马州)|温斯顿县]]。 

== 工程概况 ==
该湖是由南方公司（{{nyse|SO}}）四个下属公司之一的[[阿拉巴马电力公司|阿拉巴马电力公司]]建设的[[路易斯斯密斯大坝|路易斯斯密斯大坝]]拦截河水而形成的。大坝横跨长度为{{convert|2200|ft|m}} ，最高处达到{{convert|300|ft|m}}，1957年11月25日开始施工，1961年9月5日投入使用。

路易斯斯密斯的名称是为了纪念路易斯·马丁·史密斯。他曾于1952年至1957年担任阿拉巴马电力公司总裁职务。<ref>http://encyclopediaofalabama.org/face/Article.jsp?id=h-1888</ref>

== 生态 ==
该湖是美国第二干净的湖，仅次于[[塔霍湖|塔霍湖]]。

该湖有丰富的渔获资源，是渔民的乐园。主要鱼类有[[大口黑鲈|大口黑鲈]]、[[银花鲈鱼|银花鲈鱼]]、[[黑斑刺盖太阳鱼|黑斑刺盖太阳鱼]]、[[蓝鳃太阳鱼|蓝鳃太阳鱼]]和[[钳鱼|钳鱼]]。<ref name="odala"/>

2008年4月，[[世界户外钓鱼大赛|世界户外钓鱼大赛]]（FLW）第二张分站赛在此举行。<ref>http://www.flwoutdoors.com/bassfishing/flw/schedule/2008/{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>来自[[佛罗里达州|佛罗里达州]][[贝县|贝县]][[巴拿马市(佛罗里达州)|巴拿马市]]的迈克尔·本纳特（MICHAEL BENNETT）以{{convert|27.5|lb|kg|1}}的成绩，获得冠军。

== 湖滨 ==
附近的城镇包括，卡尔曼县的[[不莱梅社区|不莱梅社区]]、[[克伦丘社区|克伦丘社区]]和[[道奇城镇|道奇城镇]]，沃克县的[[柯里社区|柯里社区]]和[[贾斯珀市_(阿拉巴马州)|贾斯珀市]]，温斯顿县的[[艾迪生_(亚拉巴马州)|艾迪生镇]]、[[阿尔利镇|阿尔利镇]]（Arley）、[[达布尔斯普林斯市|达布尔斯普林斯市]]。
