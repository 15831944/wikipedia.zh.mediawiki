'''路易一世（安茹）（Louis I of Anjou）'''（{{bd|1339年|7月23日|1384年|9月20日|catIdx=L}}），[[法国|法国]]国王[[约翰二世_(法兰西)|约翰二世]]和王后[[波希米亚的邦娜|波希米亚的邦娜]]的次子，{{sfn|Keane|2016|p=17}}法国王室安茹支系的始祖。邦娜在[[文森城堡|文森城堡]]生下他。1356年，他的父亲封他为安茹伯爵、曼恩伯爵，并分别于1360年和1370年升为[[安茹公爵|安茹公爵]]和图赖讷公爵。

1382年，作为那不勒斯女王[[乔万娜一世|乔万娜一世]]的养子，他继承了普罗旺斯和福卡尔基耶，也继承了她对那不勒斯和[[耶路撒冷|耶路撒冷]]的权利。当他率军进入意大利以行使对那不勒斯的继承权时，已是经历对英[[百年战争|百年战争]]的战场老手了。他在行军中去世，权利和爵位落在了以他命名的儿子[[路易二世_(安茹)|路易二世]]身上，后者一度成功统治那不勒斯。

== 百年战争 ==
1356年，路易在其兄太子夏尔（后来的[[查理五世_(法兰西)|夏尔五世]]）统领的部队中参加了[[普瓦捷战役|普瓦捷战役]]。兄弟俩几乎没有作战，在对抗中全军逃跑。尽管此举令他们蒙羞，逃跑却让他们免于被此战完胜的英格兰人所俘。约翰二世和路易的弟弟[[大胆腓力|大胆腓力]]却不幸被[[黑太子爱德华|黑太子爱德华]]率领的英军俘获。1360年，英法就赎金及议和条件达成布莱特尼条约，其中要求40个出身高贵的人质作为国王付清赎金的担保。10月，已是安茹公爵的路易随队航行到英格兰。由于法国经济不景气，分期债务被延期。路易因此处在英格兰监护下的时间长于预期的6个月。他试图和英王[[爱德华三世|爱德华三世]]谈判以换取自己的自由，谈判失败，决定逃跑。路易回到法国后，其父约翰二世反对他这一非骑士行为。约翰二世为此感到不名誉，且结合布莱特尼条约要求的赎金正在拖欠的事实，他回到英格兰被囚，以挽回名誉。

1380-1382年，路易是侄子[[查理六世_(法兰西)|夏尔六世]]的[[摄政|摄政]]。

== 那不勒斯国王 ==
1382年那不勒斯女王[[乔万娜一世|乔万娜一世]]死后，路易离开法国，要求那不勒斯王位。乔万娜没有子女，且敌视近支亲族，不愿让他们继承，已收养路易为继承人。路易继承她为普罗旺斯和福卡尔基耶伯爵。但路易尽管在[[阿维尼翁|阿维尼翁]]由[[对立教宗|对立教宗]][[克雷芒七世_(對立教宗)|克莱芒七世]]加冕为[[那不勒斯国王|那不勒斯国王]]，却被迫滞留在法国，乔万娜的军队被她的前继承人和同族堂兄[[卡洛三世_(那不勒斯)|杜拉佐的卡洛]]所败。1382年，乔万娜在圣费莱的狱中被卡洛杀；在对立教宗、法国、[[米兰|米兰]]的[[贝尔纳博·维斯康蒂|贝尔纳博·维斯康蒂]]、[[萨伏伊|萨伏伊]]的[[阿梅迪奥六世|阿梅迪奥六世]]等的支持下，路易用摄政期间获取的钱，发起了从卡洛手中夺取那不勒斯王位的远征。

此次远征的军队大约有4万，<ref>{{cite web |url=http://cronologia.leonardo.it/storia/aa1380a.htm |title=Papa Urbano VI e il Regno di Napoli |work=Cronologia |language=it |accessdate=November 7, 2011}}</ref>但并不成功。[[卡洛三世_(那不勒斯)|卡洛]]倚仗[[约翰·霍克伍德|约翰·霍克伍德]]和[[巴尔托洛梅奥·德阿尔维阿诺|巴尔托洛梅奥·德阿尔维阿诺]]的14,000雇佣军，将法军从那不勒斯转移到王国的其他地区，并用游击战术骚扰之。1383年3月1日萨伏伊国王阿梅迪奥六世在[[莫利塞|莫利塞]]驾崩后，他的军队开了小差。路易转而向侄子法国国王求助，后者派出一支由[[库锡勋爵安盖兰七世|库锡的安盖兰]]率领的军队，后者攻占了[[阿雷佐|阿雷佐]]，侵入[[那不勒斯|那不勒斯]]，但却在半途得知路易已于1384年9月20日在[[比谢列|比谢列]]突然去世。于是库锡勋爵很快把阿雷佐卖给[[佛罗伦萨|佛罗伦萨]]后班师回国。

== 婚姻和子女 ==
1360年7月9日，路易娶[[布列塔尼公爵夏尔|布列塔尼公爵夏尔]]和[[德勒的让娜|德勒的让娜]]之女吉斯女领主[[布卢瓦的玛丽|布卢瓦的玛丽]]{{sfn|Rohr|2016|p=30}}。他们的子女如下：
*玛丽（1370年-1383年后）；
*[[路易二世_(安茹)|路易二世]]（1377年-1417年）{{sfn|Rohr|2016|p=30}}；
*夏尔（1380年-1404年，[[昂热|昂热]]），塔兰托亲王，[[鲁西_(埃纳省)|鲁锡]]、[[埃唐普|埃唐普]]和[[然镇|然镇]]伯爵。

== 启示录挂毯 ==
1370年代末，路易安排了启示录挂毯的设立，即一系列描述《[[启示录|启示录]]》[[默示|默示]]故事的[[缂织壁毯|缂织壁毯]]。
