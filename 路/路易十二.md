'''路易十二（人民之父）Louis XII le Père du Peuple'''（{{bd|1462年|6月27日|1515年|1月1日}}）[[法国|法国]][[瓦卢瓦王朝|瓦卢瓦王朝]]国王（1498年—1515年在位）。他属于该王朝的奥尔良支系。即位前的封号是[[奥尔良公爵|奥尔良公爵]]（称路易二世，1465年起），[[瓦卢瓦公爵|瓦卢瓦公爵]]（也稱路易二世），[[那不勒斯國王|那不勒斯國王]]（路易吉三世）。

== 家世 ==
路易十二是[[奥尔良|奥尔良]]公爵[[查理一世_(奥尔良公爵)|查理一世]]和[[克莱沃的玛丽|克莱沃的玛丽]]的儿子，生于[[布卢瓦|布卢瓦]]。他的父亲是法国国王[[查理六世_(法兰西)|查理六世]]的侄子，也是法国最伟大的宫廷诗人之一。1465年父亲去世后，路易继承了奥尔良公爵的封号。1480年代他因為野心勃勃，意圖獲得法國攝政權，曾經被當時的攝政[[法兰西的安妮|法兰西的安妮]]關入鐵籠中。<ref name="巴亞爾">巴亞爾等著，《法國史圖說》（上海：上海人民出版社，2005），頁126-131</ref>

== 婚姻 ==

1476年，奥尔良公爵路易与他的堂侄女[[法兰西的让娜|法兰西的让娜]]订婚，后者是路易的堂兄法王[[路易十一|路易十一]]之女。让娜很虔诚但却是个残废。1498年当法国国王[[查理八世_(法兰西)|查理八世]]无嗣而终后，这个婚约被取消了，路易与查理八世的遗孀[[布列塔尼的安妮|布列塔尼的安妮]]结婚并得以继承王位。安妮是[[布列塔尼公爵|布列塔尼公爵]][[弗朗索瓦一世_(布列塔尼)|弗朗索瓦一世]]的唯一继承人，路易十二与她的政治婚姻确保了布列塔尼和[[勃艮第|勃艮第]]两公国留在法国版图之内。在布列塔尼的安妮去世后，路易十二最后娶了[[英格兰|英格兰]]国王[[亨利八世|亨利八世]]的妹妹[[玛丽·都铎|玛丽·都铎]]。<ref name="Sauvigny">G. de Bertier de Sauvigny著，《法國史》（台北：五南圖書出版公司，1989），頁101-102</ref>

== 意大利战争 ==
[[Image:Louis_-_Litterae_super_abrogatione_pragmatice_sanctionis_in_quarta_sessione_sacro_sancti_Lateranensis_concilii_publice_lecte_et_recitate,_1512_-_4592398.tif|thumb]]

路易十二全面继承了查理八世征服[[意大利|意大利]]的计划，他想达到的目标甚至比查理八世还要远大。[[意大利战争|意大利战争]]在他统治时期变成了许多大国卷入的[[欧洲|欧洲]]战争，并在此后的近半个世纪令法国无法自拔。

1499年在企图独霸意大利的[[威尼斯|威尼斯]]邀请之下，路易十二率大军进入意大利，进攻威尼斯在[[伦巴第|伦巴第]]地区发展势力的最大障碍[[米兰|米兰]]。他成功地征服了米兰，把米兰的统治者[[盧多維科·斯福爾扎|盧多維科·斯福爾扎]]一舉趕跑。1500年，路易十二与[[亞拉岡|亞拉岡]]国王[[费尔南多二世_(阿拉贡)|斐迪南二世]]签订密约（1500年[[格拉纳达条约|格拉纳达条约]]），瓜分[[那不勒斯|那不勒斯]]，并取得那不勒斯王位；但斐迪南二世最终打乱了他的计划。1502年路易十二与斐迪南二世发生冲突，随之1503年法军在加利良诺大败。到1504年，法国在那不勒斯的佔地與影响都被[[西班牙|西班牙]]給排挤出去，形成西、法二國分據南北的強國（法佔米蘭、西佔那不勒斯）的局面，但因為[[佛羅倫斯共和國|佛羅倫斯共和國]]自1494年後持續親法，因此法軍在義大利的影響力優於西班牙。1511年，[[教皇|教皇]][[尤利乌斯二世|尤利乌斯二世]]为免意大利落入法国之手，联合各城市及西班牙拼凑成反对法国的“[[神圣联盟|神圣联盟]]”。1512年路易十二在[[拉韦纳战役|拉韦纳战役]]中打败了联盟的军队，但随后因為[[斯福爾扎家族|斯福爾扎家族]]意圖奪回米蘭，法軍于1513年再次进攻米兰时，被[[瑞士|瑞士]]雇佣军击败（诺凡拉战役）。这次战役以后，路易十二几乎彻底退出了对意大利的争夺。<ref name="巴亞爾"/>

== 王位继承 ==
路易十二力图成为一位人民的好国王。他对此作出许多努力，如改革司法系统和减轻税赋，頒布嚴厲的法令來打擊軍人之不法。这些举动使他得到了“人民之父”的称誉（1506年由全國[[三級會議|三級會議]]頒給他的頭銜）。路易十二于1515年1月1日去世，被安葬在[[圣但尼修道院|圣但尼修道院]]。由于法国的王位继承严格遵照[[薩利克繼承法|薩利克繼承法]]排斥女性的原则，路易十二的王冠传给了他的堂侄兼女婿[[弗朗索瓦一世|昂古莱姆的弗朗索瓦]]（即弗朗索瓦一世）。<ref name="Sauvigny"/><ref name="巴亞爾"/>

路易十二的人格單純、偉大、真摯，他是為仁慈、平和、節儉的君主，也在他的政府其他層面烙下一個父權的性格。他的統治促成經濟的蓬勃成長，開始16世紀上半長期且全面的經濟上升與人口成長，這些都讓稅收大增的法國很容易就能負擔路易十二在[[義大利戰爭|義大利戰爭]]的龐大支出，也讓繼任的法蘭西斯一世能夠奢華地鋪張浪費、強化王權及文化建設。<ref name="Sauvigny"/>

[[马基雅弗利|马基雅弗利]]在他的[[君主论|君主论]]中，把路易十二作为失败君主的典型。

== 子女 ==
长女[[法蘭西的克洛德|克洛德]]（1499年10月14日-1524年7月20日），1514年5月18日嫁[[弗朗索瓦一世_(法兰西)|弗朗索瓦一世]]，有子女，成功繼承法國王位。
