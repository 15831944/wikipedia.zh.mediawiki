'''路易斯安娜州'''（{{lang-fr|État de la Louisiane}}; {{lang-en|State of Louisiana}}），簡稱'''路州'''，是[[美國|美國]]的一個[[州|州]]，位於[[美國墨西哥灣沿岸地區|墨西哥灣沿岸]]。此州以對比強烈的[[文化|文化]]、[[地理|地理]]景觀著名。如[[嘉年華會|嘉年華會]]的狂歡和荒野沼澤地中的寂靜；並混合了美國幾乎所有的重要文化元素，如[[印地安|印地安]]、[[西班牙|西班牙]]、[[法國|法國]]、[[英國|英國]]和[[非洲|非洲]]等文化。盛產[[石油|石油]]。

== 歷史 ==
[[File:New_Orleans_h76369k.jpg|缩略图]]-[[新奧爾良戰爭|新奧爾良戰爭]]]]
{{See also|路易斯安那購地}}

在17世紀被[[歐洲人發現|歐洲人發現]]以前，「路易斯安那」主要是美洲土著民居住。史載第一批前來該州的[[歐洲|歐洲]]人是在1528年開始，[[西班牙人|西班牙人]][[埃尔南多·德·索托|埃尔南多·德·索托]]乃第一個發現「路易斯安那」北方河谷的人。其時[[西班牙|西班牙]]人在[[密西西比河|密西西比河]]河口駐紮。

1682年，[[法國人|法國人]][[熱內-羅貝·卡瓦利爾|熱內-羅貝·卡瓦利爾]]宣佈「路易斯安那」是[[法國|法國]]屬地，以紀念[[法國國王|法國國王]][[路易十四|路易十四]]而命名。1762年，西班牙從法國手中得到「路易斯安那」殖民地，此後該地已有大量的[[法國|法國]]移民居住，因此该州至今都是美国比较主要的法语地区之一。至[[美國|美國]]於1776年立國後，1803年美國向法國購得「路易斯安那」，成為美國的一部分。日後，再劃分為數州，路易斯安那州僅為其中一部分。

== 法律與政府 ==
路易斯安那州的政府是行政、立法和司法三项都是有自主权，只有[[国防|国防]]与[[外交|外交]]例外。[[美國民主党|民主黨]]和[[美國共和党|共和黨]]在这州势均力敌。[[美國眾議院|美國眾議院]]有七個代表來自路易斯安那州，其中四個是共和黨人，而三個則是民主黨人。

路易斯安那州的法律主體大致是按[[法國|法國]]、[[德國|德國]]和[[西班牙|西班牙]]所使用的[[欧陆法系|欧陆法系]]，有別於其他[[美國各州|各州]]遵從的英式[[海洋法系|海洋法系]]。1997年該州乃第一個州份推行盟約婚姻法（covenant marriage），要求婚前諮詢及對[[離婚|離婚]]設定嚴苛的限制。

== 行政區劃 ==
{{main|路易斯安那州行政區劃}}
美國的路易西安納州一共有64個堂區，堂區與美國本土另外47個州以及夏威夷州的縣是同一個級別的行政單位，另外阿拉斯加州既沒有縣也沒有堂區，而是劃分成自治市和統計區。

== 地理 ==
[[File:Intracoastal_Waterway_Louisiana.jpg|thumb]][[河口|河口]]]]

路易斯安那州東接[[密西西比州|密西西比州]]，西連[[德克萨斯州|德克萨斯州]]，南濱[[墨西哥灣|墨西哥灣]]，北與[[阿肯色州|阿肯色州]]為鄰。

[[密西西比河|密西西比河]]流贯该州东部，河两岸是宽广的冲积平原，河口处形成广阔的三角洲，伸入墨西哥湾。由於屬[[密西西比河|密西西比河]]河的下游，大致上該州有兩類地形，其一是[[沖積平原|沖積平原]]，毗連南面海岸；其二則是[[高地|高地]]，主要位處北面。[[沖積平原|沖積平原]]平均海拔30米，是全美國海拔最低的州。最高點在西北部的[[德里斯基尔山|德里斯基尔山]]，海拔僅160米。

路易斯安那州属[[亚热带气候|亚热带气候]]，夏季湿热，冬季温和，8月份平均最高气温28度，一月份平均最低气温8度。該州平均年降水量在[[什里夫波特_(路易斯安那州)|什里夫波特]]为1200毫米，[[新奥尔良|新奥尔良]]和[[巴吞鲁日|巴吞鲁日]]之间為1600毫米。

森林覆盖率约为50%，主要树木为[[松|松]]、[[柏|柏]]。

== 經濟 ==

[[File:LafayetteCollage.jpg|thumb]]]]
[[File:Atchafalaya_Basin.jpg|thumb]]
[[File:Louisiana_State_Capitol_at_night.jpg|thumb]]
[[File:New_Orleans_Stoop_Sitters_Louisiana_Natl_Guard_2008.jpg|缩略图]]
2005年路易斯安那州的国民GDP为1680亿美元，排在全国第24位。人均收入是30952美元，位列全美第41位。

该州主要的农产品包括水产品（是全球最大的小龙虾生产地，产量约占世界总产量的90%）、棉花、大豆、牛、[[甘蔗|甘蔗]]、禽蛋类、奶制品和稻米。北部盛产[[棉花|棉花]]，南部广种[[甘蔗|甘蔗]]，西南部沼泽地区培植[[水稻|水稻]]。該州甘蔗產量列全国第三位，仅次于[[佛罗里达州|佛罗里达州]]及[[夏威夷|夏威夷]]兩個州。[[稻米|稻米]]产量則列全国第四位，次于[[阿肯色州|阿肯色州]]、[[加利福尼亚州|加利福尼亚州]]及[[德克萨斯州|德克萨斯州]]。1960年前[[棉花|棉花]]在农产值中占首位。[[墨西哥湾|墨西哥湾]]沿岸渔业发达，盛产[[虾|虾]]、大[[鲱鱼|鲱鱼]]、[[牡蛎|牡蛎]]和[[螃蟹|螃蟹]]。其中大鲱鱼产量居各州之冠，而大虾产量列全国第二位。蟹产量列第四位。

路易斯安那州的工业包括化工产品、[[石油|石油]]和[[煤炭|煤炭]]产品、食品加工、运输设备和纸制品。主要矿藏有[[石油|石油]]、[[天然气|天然气]]和[[硫磺|硫磺]]等。主要油田分布在[[什里夫波特|什里夫波特]]和[[门罗|门罗]]之间，以及[[墨西哥湾|墨西哥湾]]沿岸。該州石油产量在全國仅次于[[德克萨斯州|德克萨斯州]]。

Tabasco酱汁，有美国最大的辣酱生产商之一Mcllhenny公司在市场上销售，而这个公司则创办于Avery岛。

南路易斯安那港，位于新奥尔良和巴吞鲁日之间的密西西比河畔，是西半球停泊量最大，世界停泊量第4大的河港，同时也是世界上最大的散装货河港。

[[旅游|旅游]]和文化在该州经济中占有重要地位，尤其是在新奥尔良地区，每年在这两项上的收入据估计有52亿美元。该州也主办过很多重要的文化活动，如世界文化经济论坛，每年秋天在新奥尔良莫里尔会议中心举办。

[[新奥尔良|新奥尔良]]和[[什里夫波特|什里夫波特]]也是繁荣的电影工业中心。州政府积极的金融政策和促进措施使当地的电影产业步入了快速发展的轨道。在2007年末2008年初，一个300000平方英尺（28000平方米）的电影摄影棚将在Treme开启，和众多艺术级的产品设备，和一个电影培训学院。

路易斯安那州有3种个人所得税等级，从2%到6%不等。消费税是4%：3.97%的该州消费税和0.03%的旅游促进地区消费税。行政分支机构在州税以外还征收他们自己的消费税。该州还有使用税，包括被税务部分配给当地政府的4%（使用税）。财产税则按当地水平进行评估和征收。路易斯安那是个享受[[政府补贴|政府补贴]]的州，每支付1美元就可得到联邦政府的1.44美元的补助。

=== 联邦政府补贴和支出 ===
路易斯安娜的纳税人得到比州平均水平更高的联邦资金补贴（每1美元所要缴纳的联邦税）。2005年每1美元所征收的联邦税，该州公民在联邦支出方面收到大约1.78美元的补助。这在美国所有州中排名第4高，同时显示出自1995年每1美元的税得到1.35[[美元|美元]]联邦支出补助（全国范围内排第7）有一个上涨的过程。临近几个州在每1美元所征收的联邦税的联邦支出补助为：德克萨斯0.94美元、阿肯色1.41美元和密西西比2.02美元。联邦支出在2005年以来得到特别调高是因为[[卡特里娜飓风|卡特里娜飓风]]所带来的灾后重建工作导致的。

=== 能源 ===
路易斯安娜有丰富的石油和天然气。无论在陆上还是州境内的海面上都发现了丰富的[[石油|石油]]和[[天然气|天然气]]储量。此外，巨大的石油和天然气储量在联邦行政机构外部大陆架（OCS）所管制的[[墨西哥湾|墨西哥湾]]被发现（在该州的海上）。据能源信息管理局的消息，[[墨西哥湾|墨西哥湾]]的外部大陆架是美国最大的[[石油|石油]]生产区。除墨西哥湾外部大陆架，该州的石油产量排美国第4，[[石油|石油]]储量占全美总量的2%。该州[[天然气|天然气]]总量约占全国的5%。最近在Haynesville页岩构造带发现部分或整个的Caddo、Bossier、Bienville、Sabine、De Soto、Red River和Natchitoches教区的天然气储量使该地区成为世界第4大[[天然气|天然气]]田，一些气井最初可以日产2500万立方英尺的天然气。

该州是世界上第一个在水面上进行[[石油|石油]]钻探的地方，具体位置在州西北部的卡多湖。石油和天然气工业，就像运输和[[油气|油气]]精炼等辅助工业一样，自20世纪40年代后就成为了该州的主导产业。自1950年开始，该州受到[[美国内政部|美国内政部]]的几次控告，联邦政府努力取消该州的水下土地财产权。这些控制着大量的石油和天然气贮液器。当70年代的石油天然气繁荣时期时，振奋了该州的经济。同样的，当80年代的天然气和[[石油危机|石油危机]]时，很大程度上归因于联邦储备委员会的货币政策，该州的房地产、储蓄金和贷款以及当地[[银行|银行]]都迅速贬值。该州的经济就像其20世纪后半页的政策一样，没能摆脱能源工业对经济的影响。自上世纪80年代后，这些工业在[[侯斯頓|侯斯頓]]得到了整合。

== 人口 ==
2003年人口普查，路易斯安娜州有4,496,334人（含有大約20萬的法語人口）。目前路易斯安娜州的種族比例可分為：
* 62.5% 是白人
* 32.5% 非裔美國人 
* 2.4% 是[[拉美裔人|拉美裔人]]
* 1.2% 是亞裔美國人
* 0.6%是[[印第安人|印地安原住民]]
* 1.1 % 混合的種族

路易斯安娜州的前五大居民祖先為：
# [[非裔美國人|非裔美國人]]（32.5%）
# [[法國|法國]]或[[加拿大|加拿大]][[魁北克|魁北克]]人（16.2%）
# [[美國各州|美國各州]]移民（10.1%）
# [[德國|德國]]人（7.1%）
# [[愛爾蘭|愛爾蘭]]人（7%）

路易斯安娜是美國有較多[[法語|法語]]人口的州之一，參見[[美國的法語|美國的法語]]。

=== 宗教 ===
與其他南部州份情況相若，路易斯安那州大部份居民都信奉基督[[新教|新教]]，2014年皮尤研究中心數據指出新教徒佔成年人口57％，另26％為[[天主教|天主教]]徒，1％為[[耶和華見證人|耶和華見證人]]會友。無宗教信仰者佔成年人口13％。<ref>[http://www.pewforum.org/religious-landscape-study/state/louisiana/ Religious Landscape Study: Adults in Louisiana], Pew Research Center</ref>

在路易斯安娜州的新教徒中，前三大派別為：
* [[浸信会|浸信会]]－29%
* [[衛理公會|衛理公會]]－5%
* [[五旬节派|五旬节派]]－2%
