'''路随'''，又稱'''路隋'''（{{bd|776年||835年|8月16日|catIdx=L}}<ref name=AS>[http://db1x.sinica.edu.tw/sinocal/ 中央研究院两千年中西历转换] {{webarchive|url=https://web.archive.org/web/20100522200011/http://db1x.sinica.edu.tw/sinocal/ |date=2010-05-22 }}</ref><ref>[[:s:舊唐書/卷17下|《旧唐书》卷一十七下]]：壬戌，镇海军节度使路随卒。</ref>），[[表字|字]]'''南式'''，[[唐朝|唐朝]]官员，[[唐文宗|唐文宗]]年间任[[宰相|宰相]]。

== 背景 ==
路随生于[[唐代宗|唐代宗]][[大历|大历]]十一年（776年）。祖上[[阳平关|阳平]]人氏。高祖[[路节|路节]]为[[唐高宗|唐高宗]]兄越王[[李貞_(唐朝)|李贞]]府东阁史祭酒。曾祖[[路惟恕|路惟恕]]，官至[[睦州_(仁寿)|睦州]]刺史；祖父[[路俊之|路俊之]]，官终[[太子|太子]]通事舍人。<ref name=BT159>[[:s:舊唐書/卷159|《旧唐书》卷一百五十九]]</ref>

路随父[[路泌|路泌]]以博识好学闻名，通《[[五经|五经]]》，尤嗜好《[[诗|诗]]》、《[[易|易]]》、《[[左氏春秋|左氏春秋]]》。[[太尉|太尉]][[朱泚|朱泚]]反叛[[唐德宗|唐德宗]]，德宗被迫逃离[[长安|长安]]，奔[[奉天县_(唐朝)|奉天]]，再奔[[兴元|兴元]]，路泌随驾流亡，妻儿留在长安。流亡期间，中箭负伤。后他效力行在都知兵马使[[浑瑊|浑瑊]]，浑瑊很器重他，后来浑瑊任[[河中节度使|河中节度使]]，路泌也作为浑瑊的从事参与了其对另一叛将[[李怀光|李怀光]]的讨伐。[[贞元_(唐朝)|贞元]]三年（787年），路泌陪浑瑊与[[吐蕃|吐蕃]]大相[[尚结赞|尚结赞]]缔盟，尚结赞设伏劫盟，意图俘虏浑瑊；浑瑊逃跑，幕僚大多被俘杀。<ref>[[:s:資治通鑑/卷232|《资治通鉴》卷二百三十二]]</ref>路泌被俘并被带到吐蕃首都[[拉萨|拉萨]]。在拉萨期间，他通晓佛学，为吐蕃赞普所重待以宾礼，但至死未能获准返回中原。<ref name=BT159/><ref name=NBT142>[[:s:新唐書/卷142|《新唐书》卷一百四十二]]</ref>

路泌被俘时，路随还是孩童，长大后得知父亲被吐蕃所俘，日夜哭泣，坐时面向西面吐蕃方向，为哀悼父亲被俘而不吃肉，也不婚娶。<ref name=LA>[[:s:荐士於中书舍人书|李翱《荐士於中书舍人书》]]</ref>母亲告诉他，他长得像父亲，他便终身不照镜子。十九年（803年），吐蕃写信求和，路随给唐德宗上了三份表章请求答允求和，以至德宗派中使向他解释，德宗不信任吐蕃求和，在等待吐蕃进一步示诚。但数年间，吐蕃未能证明诚意，当时也就没有发生进一步的和谈。<ref name=BT159/><ref name=NBT142/>

== 早期仕途 ==
路随举[[明经|明经]]，授[[润州|润州]]参军事，为军阀[[镇海军节度使|镇海军节度使]][[李锜|李锜]]所制。李锜想羞辱路随，让路随知市事，路随怡然坐在市中监督事务。<ref>鉴于李锜于[[元和|元和]]二年（807年）反叛朝廷并败死，这些事必然发生在李锜败亡之前。见[[:s:資治通鑑/卷237|《资治通鉴》卷二百三十七]]。</ref>东都[[洛阳|洛阳]]留守[[韦夏卿|韦夏卿]]闻之，被路随感动，邀他入东都幕府，<ref>[[:s:新唐书/卷162|《新唐书》卷一百六十二]]</ref>路随因此名声更大了。<ref name=BT159/><ref name=NBT142/><ref>[[:s:故太子少保赠尚书左仆射京兆韦府君神道碑|吕温《故太子少保赠尚书左仆射京兆韦府君神道碑》]]</ref>

路随曾任宣歙来石军判官、试太常寺协律郎，不蓄养仆人、侍妾，以孝行被国子博士[[李翱|李翱]]称赞比作[[曾子|曾子]]、[[闵子骞|闵子骞]]，并得其推荐。<ref name=LA/>

== 唐宪宗年间 ==
此时，[[唐宪宗|唐宪宗]]在位，吐蕃不时提议求和。路随五次上表宪宗请求修好，还上表给宰相，希望路泌生还；宰相[[裴垍|裴垍]]和[[李藩|李藩]]都为他说话。和吐蕃最终议和后，五年（810年），应宪宗要求，吐蕃送还路泌及其同僚判官[[郑叔矩|郑叔矩]]的棺材、墓石和手书。宪宗追赠路泌[[绛州|绛州]]刺史，用公费安葬，累赠至[[太子太保|太子太保]]。<ref>[[:s:資治通鑑/卷238|《资治通鉴》卷二百三十八]]：庚申，吐蕃遣其臣[[论思邪热|论思邪热]]入见，且归路泌、郑叔矩之柩。</ref>路随为父服丧，以孝闻名，丧期满后，任[[左补阙|左补阙]]。一次宰相[[李绛|李绛]]建议宪宗多听取谏官们的意见，宪宗举了路随和[[韦处厚|韦处厚]]的例子，说自己常听他们的话。有识者从此敬服二人。路随又被任为史馆修撰，以耿直爽亮为人所称。<ref name=NBT142/>又历任起居郎、司勋员外郎，仍兼史馆修撰。<ref name=BT159/>

== 唐穆宗年间 ==
十五年（820年）正月，宪宗崩，子[[唐穆宗|唐穆宗]]继立。<ref>[[:s:資治通鑑/卷241|《资治通鉴》卷二百四十一]]：庚子，暴崩于中和殿。……闰月，丙午，穆宗即位于太极殿东序。</ref>穆宗登基后，升路随为司勋郎中。韦处厚入[[翰林院|翰林院]]为侍讲学士，知制诰[[韦表微|韦表微]]与韦处厚商议增选学士，荐路随，路随也升为侍讲学士。<ref>[[:s:新唐书/卷177|《新唐书》卷一百七十七]]</ref>三月，穆宗召路随和韦处厚在太液亭讲《毛诗·关雎》《尚书·洪范》等篇，讲完后都赐绯鱼袋。长庆二年（822年）四月，韦处厚因穆宗年轻不好学，与路随合《易》《书》《诗》《春秋》《礼》《[[孝经|孝经]]》《[[论语|论语]]》撰《六经法言》二十卷进献，<ref name=NBT59>[[:s:新唐书/卷59|《新唐书》卷五十九]]</ref><ref>[[:s:翰林院厅壁记|韦处厚《翰林院厅壁记》]]</ref><ref>《册府元龟》一作三年（823年）事。</ref>获赐锦彩二百匹、银器二百，路随得改[[谏议大夫|谏议大夫]]，韦处厚改中书舍人，都赐金紫，仍兼侍讲学士。闰十月，穆宗修《宪宗实录》，以宰相[[杜元颖|杜元颖]]监修，再次用路随、韦处厚充史职，与同僚[[赵暨|赵暨]]、史馆修撰[[宇文籍|宇文籍]]、韦表微、[[沈传师|沈传师]]、[[郑澣|郑澣]]等同修《宪宗实录》。<ref name=NBT58>[[:s:新唐书/卷58|《新唐书》卷五十八]]</ref><ref name=LS>[[:s:上宪宗实录表|:s:上宪宗实录表]]</ref>实录没修成时，穆宗允许路随、韦处厚隔日入史馆，平时仍然上朝参政。<ref name=BT159/><ref>[[:s:旧唐书/卷16|《旧唐书》卷一十六]]</ref><ref>[[:s:命路随韦处厚兼充史馆修撰敕|:s:命路随韦处厚兼充史馆修撰敕]]</ref>因路随与韦处厚称赞推荐，有文学和德行的江州司马[[韦辞|韦辞]]被提拔为户部员外。<ref>[[:s:旧唐书/卷160|《旧唐书》卷一百六十]]</ref>

== 唐敬宗年间 ==
四年（824年），穆宗崩，子[[唐敬宗|唐敬宗]]继立。<ref name=ZZTJ243>[[:s:資治通鑑/卷243|《资治通鉴》卷二百四十三]]</ref>路随以朝议郎、守谏议大夫、充翰林学士、上轻车都尉、赐紫金鱼袋被任为守[[中书舍人|中书舍人]]、[[翰林学士|翰林学士]]，勋赐如故。<ref>[[:s:授学士路隋等中书舍人制|李虞仲《授学士路隋等中书舍人制》]]</ref>当有人送金帛致贺时，路随回应：“我怎能因为公事获取私财？”将礼物都退还了。<ref name=BT159/><ref name=NBT142/>

[[宝历_(唐朝)|宝历]]二年（826年）十二月，敬宗因脾气为宦官和军将所恨，遭弑。弑君阴谋的为首宦官[[刘克明_(唐朝)|刘克明]]要路随以敬宗名义起草遗诏以穆宗弟绛王[[李悟_(绛王)|李悟]]权当军国事，但很快，枢密使[[王守澄|王守澄]]、[[杨承和|杨承和]]、[[神策军|神策军]]中尉[[魏从简|魏从简]]、[[梁守谦|梁守谦]]等当权宦官回击作乱者并屠戮之。<ref>[[:s:新唐书/卷208|《新唐书》卷二百零八]]</ref>王守澄与韦处厚共议后，以穆宗母[[懿安皇后|郭太皇太后]]名义拟诏以敬宗弟江王[[唐文宗|李涵]]（随后改名[[李昂_(唐朝)|李昂]]）为帝，即唐文宗。<ref name=ZZTJ243/>

== 唐文宗年间 ==
事后路随并未被处罚，可见未被视为弑杀敬宗的乱党。时任承旨的韦处厚在文宗继位后不久入相，路随受命接替他为承旨。路随后守[[兵部侍郎|兵部侍郎]]，仍知制诰、翰林学士。[[太和_(唐朝)|太和]]元年（827年）四月，文宗召路随以下对于大液殿，各赐锦彩银器。<ref name=CFYG>《[[册府元龟|册府元龟]]》</ref>二年（828年）十一月韦处厚死后，十二月，路随以中散大夫、守尚书兵部侍郎、知制诰、充翰林学士、上轻车都尉、赐紫金鱼袋拜正议大夫、守[[中书侍郎|中书侍郎]]，授[[同中书门下平章事|同中书门下平章事]]，为[[实质宰相|实质宰相]]，<ref name=ZZTJ243/><ref>[[:s:旧唐书/卷17上|《旧唐书》卷一十七上]]</ref><ref name=NBT8>[[:s:新唐书/卷8|《新唐书》卷八]]</ref>勋赐如故，<ref>[[:s:授路随中书侍郎同平章事制|:s:授路随中书侍郎同平章事制]]</ref>监修国史。<ref name=BT159/>

三年（829年）十二月，路随建议宰相不宜兼判度支类事务，引用腐败宰相[[杨国忠|杨国忠]]、[[元载|元载]]和[[皇甫镈|皇甫镈]]的例子，路随的上级同僚宰相[[裴度|裴度]]辞去度支，仅任宰相。<ref name=ZZTJ244>[[:s:資治通鑑/卷244|《资治通鉴》卷二百四十四]]：路隋言于上曰：“宰相任重，不宜兼金谷琐碎之务，如杨国忠、元载、皇甫镈皆奸臣，所为不足法也。”上以为然。于是裴度辞度支，上许之。</ref>路随奉命继续修撰《宪宗实录》，采用韦处厚记录的史料，再与史官[[苏景裔|苏景裔]]等广加拜访研究，历时约一年，于四年（830年）三月经光顺门进献《宪宗实录》四十卷及目录一卷，并上表，<ref name=LS/>得优诏，与史官司封郎中[[苏景胤|苏景胤]]、起居舍人[[陈夷行|陈夷行]]、顿田员外郎[[李汉_(唐朝)|李汉]]、右拾遗[[蒋系|蒋系]]获赐锦绣银器有差。<ref>[[:s:答路随等上宪宗实录诏|:s:答路随等上宪宗实录诏]]</ref>期间，路随等议论了实录不载[[永贞_(唐顺宗)|永贞]]元年（805年）九月去世的[[河阳节度使|河阳三城节度使]][[元韶_(唐朝)|元韶]]事迹之事，认为元韶没有出色的功绩，理应如此。<ref name=CFYG/>因先前[[韩愈|韩愈]]监修《[[唐顺宗|顺宗]]实录》时，写了很多宫禁之事，令宦官憎恨，他们常对皇帝说记录不实，卫尉卿[[周居巢|周居巢]]、谏议大夫[[王彦威_(唐朝)|王彦威]]、给事中[[李固言|李固言]]、史官苏景胤也都上言请求修正。文宗命路随修改顺宗实录，路随上表就实录不实之处婉言致歉，但为涉事官员的正直及其作为史官的独立立场辩护。文宗因而修改了命令，建议，德宗、宪宗年间宫禁之事不可确证者皆删除，其余不改。<ref>[[:s:允路随详正顺宗实录诏|:s:允路随详正顺宗实录诏]]</ref>同年，路随转[[门下侍郎|门下侍郎]]，加[[崇文馆|崇文馆]]大学士。五年（831年）正月，同僚宰相[[宋申锡|宋申锡]]被诬与皇弟漳王[[李凑|李凑]]谋反，路随与宋申锡和其他宰相[[李宗闵|李宗闵]]、[[牛僧孺|牛僧孺]]奉召入延英殿，宋申锡得知不被召见后自知获罪而退，路随等得知宋申锡谋反事后惊愕地互相看着。宋申锡、李凑都被贬。<ref>[[:s:旧唐书/卷167|《旧唐书》卷一百六十七]]</ref><ref>[[:s:新唐书/卷152|《新唐书》卷一百五十二]]</ref>二月，路随去紫宸殿奏事，到龙墀，摔倒了，命宦官搀扶。七年（833年），兼任文宗子[[太子|太子]][[李永_(唐朝)|李永]]的[[太师|太师]]；同年向文宗献上宪宗、穆宗实录。<ref name=NBT58/>十月，举[[开元|开元]]十七年[[张说|张说]]、[[源乾曜|源乾曜]]以[[唐玄宗|唐玄宗]]生日为千秋节的先例，建议文宗将生日定为庆成节，获准。八年（834年），路随告病辞职，不许。六月，与同僚宰相[[王涯|王涯]]奏请依旧制读时令。九月，册拜[[太子太师|太子太师]]，太子在明门见他。九年（835年）三月，文宗近臣[[郑注|郑注]]及尚书左丞[[王璠|王璠]]、户部侍郎李汉弹劾前宰相[[镇海军节度使|镇海军节度使]][[李德裕|李德裕]]与李凑傅母[[杜仲阳|杜仲阳]]勾结，文宗在蓬莱殿召宰相王涯、李固言、路随及王璠、李汉、郑注，路随不但不认可，还试图为李德裕辩护，说李德裕所为不至如此，如果属实，那么自己也应该有罪。议论稍息，他却因而为郑注所忌恨，李德裕被贬[[袁州|袁州]]长史，路随也不在奏章署名，因而在四月被罢相，<ref name=NBT8/><ref>[[:s:新唐书/卷180|《新唐书》卷一百八十]]</ref>遣出长安，拜检校尚书右仆射，任镇海军节度使、润州刺史、浙江西道观察等使，<ref>[[:s:旧唐书/卷174|《旧唐书》卷一百七十四]]</ref>尽管保有同中书门下平章事作为荣衔，却连面圣道别的机会也没有。<ref name=ZZTJ245>[[:s:資治通鑑/卷245|《资治通鉴》卷二百四十五]]：初，李德裕为浙西观察使，漳王傅母杜仲阳坐宋申锡事放归金陵，诏德裕存处之。会德裕已离浙西，牒留后李蟾使如诏旨。至是，左承王璠、户部侍郎李汉奏德裕厚赂仲阳，阴结漳王，图为不轨。上怒甚，召宰相及璠、汉、郑注等面质之。璠、汉等极口诬之，路隋曰：“德裕不至有此。果如所言，臣亦应得罪！”言者稍息。……    丙申，以门下侍郎、同平章事路隋同平章事，充镇海节度使，趣之赴镇，不得面辞。坐救李德裕故也。</ref>七月，路随在赴任镇海途中，在长江中流的船上去世。<ref>[[:s:旧唐书/卷17下|《旧唐书》卷一十七下]]</ref>册赠[[太保|太保]]，谥'''贞'''。<ref name=BT159/><ref name=NBT142/>

路随有学行，大度，作为谏官能直言，在内廷能匡助，不参与李宗闵、李德裕朋党之争，也不参与[[李训|李训]]、郑注的奸谋，可谓得君子中庸之道。<ref name=BT159/>

路随赴任镇海军时，征辟凤翔司马[[李敬彝|李敬彝]]，路随死后，李敬彝客居江淮，因此未能参与太和九年末的[[甘露之变|甘露之变]]，免于一死。<ref>[[:s:新唐书/卷179|《新唐书》卷一百七十九]]</ref>

[[唐武宗|唐武宗]][[会昌_(唐朝)|会昌]]二年（842年）十一月，敕令州府搜捕收回路随所修《宪宗实录》。<ref>[[:s:旧唐书/卷18上|《旧唐书》卷一十八上]]</ref>

[[宋真宗|宋真宗]]年间，[[孙奭|孙奭]]判[[国子监|国子监]]，曾写路随、韦处厚传给国子监说书[[贾昌朝|贾昌朝]]说：“君当以经术进，如二公。”<ref>[[:s:宋史/卷285|《宋史》卷二百八十五]]</ref>
