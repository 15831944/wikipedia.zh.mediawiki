'''路易维尔'''（'''Louisville'''，{{IPAc-en|audio=Looeevil.ogg|ˈ|l|uː|iː|v|ɪ|l}}，{{IPAc-en|local|audio=Loouhvull.ogg|ˈ|l|uː|ə|v|əl}} 或 {{IPAc-en|audio=Luhvull.ogg|ˈ|l|ʌ|v|əl}}）是[[肯塔基州|肯塔基州]]的最大城市。根據不同的計算方法，路易維爾為美國第十六或第二十七大城市。該市由[[喬治·克拉克|喬治·克拉克]]建立於1778年，並得名於[[法國|法國]]國王[[路易十六|路易十六]]。路易維爾以[[三冠賽|三冠賽]]的首場[[肯塔基打比大賽|肯塔基打比大賽]]而著名。

路易維爾位於肯塔基州中北部，與印第安納州只有一河之隔。該市為[[傑弗遜縣_(肯塔基州)|-{zh-hans:杰斐逊县; zh-hk:傑弗遜郡;}-]]的[[首府|首府]]，其範圍因2003年的合併而變得更大。受到中西部和南部的影響，路易維爾經常被認為是美國南部最北邊和北部最南邊的城市。

== 发音 ==
按路易维尔当地居民的习惯，该城名称Louisville中的“s”字母不发音，因其源于法语。所以标准的汉语音译为“路易维尔”。但美国还有多个州（包括[[科罗拉多州|科罗拉多州]]、[[佐治亚州|佐治亚州]]、[[密西西比州|密西西比州]]、[[内布拉斯加州|内布拉斯加州]]、[[俄亥俄州|俄亥俄州]]和[[田纳西州|田纳西州]]）均有名称为Louisville的城镇，这些Louisville中的“s”则都发音，读作/ˈluːɪsvɪl/，音译应为“路易斯维尔”。

== 人口 ==
根据2010年美国人口普查，该市人口为741,096人。

==歷史==
此城早在幾百年前，就已有土著聚居的記錄。而第一個[[歐洲|歐洲]]人抵達此城，是[[法國|法國]]探險家[[拉薩勒|拉薩勒]]，時為1669年。在[[美國獨立戰爭|美國獨立戰爭]]時代，來自[[英國|英國]]的[[喬治·克拉克|喬治·克拉克]]，就宣佈建立此城。

== 地理與氣候 ==
路易维尔气候代表美国中西部温带与东南部副热带的过度，属[[副热带湿润气候|副热带湿润气候]]（接近[[温带大陆性湿润气候|温带大陆性湿润气候]]），四季分明。冬季冷凉，潮湿，日照较少，日最高气温低于{{convert|0|C}}的平均日数为16天，日最低气温低于{{convert|-10|C}}的平均日数为10.2天，低于{{convert|-15|C}}的有2.7天；夏季炎热潮湿，日最高气温超过{{convert|30|C}}的日数年均有69天，{{convert|35|C}}以上的有8.0天。<ref name = "NWS Louisville, KY"/>最冷月（1月）均温{{convert|34.9|F|1|disp=flip}}，极端最低气温{{convert|−22|F|0|disp=flip}}（1994年1月19日）。<ref name = "NWS Louisville, KY"/>最热月（7月）均温{{convert|79.3|F|1|disp=flip}}，极端最高气温{{convert|107|F|0|disp=flip}}（上次出现于1936年7月14日）。<ref name = "NWS Louisville, KY"/>无霜期平均为207天（4月7日至10月30日）；可测量降雪平均期为12月8日至3月12日。<ref name = "NWS Louisville, KY"/>年均降水量约{{convert|44.9|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|23.88|in|disp=flip}}（1930年），最多为{{convert|68.02|in|disp=flip}}（2011年）。<ref name = "NWS Louisville, KY"/>年均降雪量为{{convert|12.5|in|cm|disp=flip}}；1988–89年的降雪量最少，积累降雪量只有{{convert|0.9|in|cm|disp=flip}}，1917–18年的降雪量最多，积累降雪量为{{convert|50.2|in|cm|disp=flip}}。<ref name = "NWS Louisville, KY"/>

==政府==
路易威爾的政府是採用市長議會制，是由市長和市議會共同管治。

全市第一次市長選舉是在1828年3月，當時設定為市長任期為三年。至1894年，改為每一任四年。

市議會共設有26個議員席位，各自代表市內不同區域。
