'''路易十八'''（{{bd|1755年|11月17日|1824年|9月16日|catIdx=L}}），[[法國國王|法國國王]]，是[[路易十六|路易十六]]的弟弟，封'''普羅旺斯伯爵'''。其姪[[路易十七|路易十七]]在狱中被[[保王党|保王党]]奉为国王。1795年，路易十七死于狱中，路易十八被奉为继承人。但由1795年至1824年在位期間，他有一大段時間並未居住在法國國內，實際在位約十年（1814-1824年），當中約三個月短暫流亡。

== 早年生活 ==
路易十八原名'''路易·斯坦尼斯瓦夫·沙維爾'''（Louis Stanislav Xavier），1755年11月17日於法國[[凡尔赛宫|凡尔赛宫]]內出生，是父親[[路易，法国王太子_(1729年–1765年)|路易·斐迪南]]（法王[[路易十五|路易十五]]世的長子）和母親[[萨克森的玛丽·约瑟芬_(1731年–1767年)|薩克森的瑪麗·約瑟芬]]的第四子，他出生時已被封為[[普羅旺斯|普羅旺斯]][[伯爵|伯爵]]，但隨後其兄長因故去世，他成為王位繼承人（大親王'''''Monsieur'''）。

== 法國大革命 ==
當[[法國大革命|法國大革命]]爆發時初期，在他的封地普羅旺斯所發生的反皇室事件較為溫和，但及後的反皇室浪潮漸漸升溫，路易·斯坦尼斯瓦夫·塞維爾（當時的普羅旺斯伯爵）亦受孤立，流放到[[威斯特伐利亞|威斯特伐利亞]]（Westphalia）生活。1791年發生了[[路易十六的出逃|路易十六的出逃]]事件，法王[[路易十六|路易十六]]被送上[[斷頭臺|斷頭臺]]處死，當得悉其兄死訊後，他宣佈立其姪为路易十七世，自己則為攝政。直至1795年6月8日[[路易十七|路易十七]]死於監獄，他自立為路易十八。

== 在位時期 ==
{{main|波旁復辟}}
由於法國的政權落入[[拿破崙|拿破崙]]的手上，路易十八自登基後便流浪到[[歐洲|歐洲]]多個國家居住，計有[[沙俄|沙俄]]和英國。他在[[英國|英國]]居住多年。

直至1814年，[[拿破仑|拿破仑]]在[[萊比錫戰役|萊比錫戰役]]战败，[[巴黎|巴黎]]陷落，宣布退位，路易十八在[[大英帝國|英]][[普魯士|普]]联军护送下回到[[巴黎|巴黎]]，[[复辟|复辟]][[波旁王朝|波旁王朝]]，史稱「[[波旁復辟|波旁復辟]]」，而拿破崙被流放[[厄爾巴島|厄爾巴島]]。1815年，因為政經條件的迅速惡化，民眾思念拿破崙，使他重掌政權，出現[[百日王朝|百日王朝]]，讓路易十八再度逃出法國。直到拿破仑同年的[[滑鐵盧戰爭|滑鐵盧戰爭]]失敗後，宣布退位並被英國流放[[聖海倫娜島|聖赫勒拿島]]，路易十八才又復位。

他在位期間採取妥協穩健的方針，理論上實施[[開明專制|開明專制]]，實際上執行[[君主立憲制|君主立憲制]]；路易睿智地尊重國會並重視經濟建設，因此人民逐漸接受並支持這位「跟著外國軍隊進來」的復辟國王。剛開始路易十八进行第一次议会政治试验，国王拥有行政权和立法动议权，不久又解散了议会。重新选举的议会中，擁護君主立憲的自由派占了多数，政治逐渐清明，負責能幹的大臣也極力去恢復法國的地位與國力（如[[塔列蘭|塔列蘭]]）。但是在1820年2月13日，王位繼承人[[查理十世_(法兰西)|阿图瓦伯爵]]之次子贝利公爵遇刺，极端派援引[[神聖同盟|神聖同盟]]的外力，趁机逼迫溫和派首相[[埃利_(德卡茲公爵)|德卡茲]]辞职。极端派组成内阁，強化[[天主教會|天主教會]]的力量並逐漸進行思想控制，許多市民称作“黑色恐怖”（因教士穿黑袍）。路易十八對此无力干预，自叹自己是一个“连马背也跨不上去的骑士”。

雖然信奉[[自由主義|自由主義]]的市民對[[查理十世_(法兰西)|阿图瓦伯爵]]日漸增加的保守勢力日漸疑慮和不滿；此時法國的農、工業卻因為好景氣的繁榮進步，因此以農民為首的勢力仍給予國王熱誠的愛戴。

== 去世 ==
1824年，路易十八一病不起，由於無子女，王位由67岁的弟弟[[查理十世_(法兰西)|阿图瓦伯爵（查理十世）]]即位。此時法國不但已經恢復國際地位與軍政實力，也重建了經濟繁榮。

即使路易十八在晚年逐漸屈從[[奧地利帝國|奧地利帝國]][[宰相|宰相]][[梅特涅|梅特涅]]的[[保守主義|保守主義]]，以及其弟查理十世的極端君權派，他很清楚[[絕對君主制|絕對君主制]]是不可能被法國市民所接受，因此他死前就憂慮地說：「我的弟弟恐怕難以死在這張床上」。果不其然，1824年[[查理十世_(法兰西)|查理十世]]即位後，很快因為經濟衰退的巨大衝擊，以及國王提倡[[神權統治|神權統治]]的[[專制主義|專制主義]]，在1830年爆發了[[七月革命|七月革命]]，波旁王朝告終。
