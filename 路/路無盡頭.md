《'''路無盡頭'''》{{sfn|居三元|張殿英|1993}}（{{lang-id|Jalan Tak Ada Ujung}}），或譯《路漫漫》{{sfn|梁立基|2003}}、《無盡頭的路》{{sfn|梁立基|1983}}，是一部由[[穆赫塔爾·盧比斯|穆赫塔爾·盧比斯]]創作的[[印度尼西亞文學|印尼小說]]，在1952年由[[圖書編譯局|圖書編譯局]]出版。故事發生在[[印度尼西亚独立革命|獨立戰爭]]時期，講述主人公伊薩老師如何在參加[[游擊隊|游擊戰]]、為[[印度尼西亞|共和國]]戰鬥期間，克服自己因為時局的變化而產生的恐懼情緒。這部小說是文學獎的得獎作品，並成為印尼國內外多篇論文的討論題材，甚至獲評為盧比斯最好的作品。

== 背景 ==
《路無盡頭》的作者[[穆赫塔爾·盧比斯|穆赫塔爾·盧比斯]]（1922年－2004年）是一位[[巴塔克人|巴塔克族]]作家，來自[[荷屬東印度|荷屬東印度]]蘇門答臘[[巴東_(印尼)|巴東]]（今屬[[印度尼西亞|印度尼西亞]][[西蘇門答臘省|西蘇門答臘省]]）{{sfn|Badan Bahasa, Mochtar Lubis}}。他在小學時期開始從事文學創作，至1941年[[荷屬東印度日佔時期|日本佔領東印度群島]]之後投身新聞事業。1945年[[印度尼西亞獨立宣言|印尼獨立]]後，盧比斯曾加入[[安塔拉通訊社|安塔拉通訊社]]等多家傳媒機構，報導[[印度尼西亞獨立革命|印尼獨立戰爭]]的戰況。{{sfn|Her|2010}}《路無盡頭》是盧比斯繼《沒有明天》（''{{lang|id|Tidak Ada Esok}}''）之後的第二部小說，花了6個月來完成{{sfn|Hill|2010|p=70-71}}。

== 故事大綱 ==
伊薩老師（Guru Isa）一直過着戰戰兢兢的生活——首先日本佔領者令群眾恐慌，然後印尼爆發獨立戰爭，形勢如火如荼。他害怕極了，結果變成了一個[[勃起功能障碍|阳痿]]患者，病情持續了幾年。不過，出於教師的職責，他不得不參加一場青年會議，討論獨立戰爭的形勢。他沒辦法拒絕，於是與會者叫他去當個[[郵務士|郵務士]]，在[[雅加達|雅加達]]運送書信和武器。

沒過多久，伊薩認識了一位叫做哈希爾（Hazil）的游擊隊戰士，他年紀輕輕，而且跟伊薩一樣，喜歡音樂，於是他們兩人就結為朋友，伊薩也感到更加放鬆。不過，在一起參加獨立戰爭的時候，伊薩又開始感到焦慮不安。一次把武器運送到雅加達城外以後，伊薩染上了[[疟疾|疟疾]]，哈希爾幫助伊薩的妻子法蒂瑪（Fatimah）照顧伊薩。

康復了以後，伊薩又可以出門教書，不過法蒂瑪知道伊薩不舉之後，感到失望，便和哈希爾偷情。伊薩在睡房的枕頭下找到哈希爾的[[煙斗|煙斗]]，知道妻子有外遇之後，怒火中燒，卻沒有跟法蒂瑪和哈希爾對質，而是疏遠大家，變得更不自信。

伊薩和哈希爾很快接到了指令，他們要向一大群離開戲院的荷軍士兵投擲[[手榴彈|手榴彈]]。雖然他們成功了，可是沒多久哈希爾就被荷軍拘捕了。本來伊薩打算離開雅加達，最後卻選擇承擔後果。他在牢房受盡了虐待也不發一言，倒是哈希爾跟他見面的時候，說自己「只是被打了一記耳光」就把游擊隊的動向和盤托出。聽罷了，伊薩就克服了恐懼，重拾信心，還恢復了男人的陽剛之氣。

== 人物形象 ==
據馬曼·S·瑪哈雅娜（{{lang|id|Maman S. Mahayana}}）、奧永·索菲安（{{lang|id|Oyon Sofyan}}）和阿赫瑪德·迪安（{{lang|id|Achmad Dian}}）所述，故事的主人公伊薩是一位[[小學|小學]]教師，喜歡聽音樂、看足球賽。他是法蒂瑪的丈夫，在社區很受歡迎。經歷過日佔時期和獨立戰爭的爆發之後，他的情緒變得越來越緊張，結果變得很容易受驚，遇到衝突的時候想盡力避開。故事的另一個主要人物哈希爾是一位年輕的游擊隊戰士，也是伊薩的朋友，起初對獨立戰爭很有熱忱。為了增加游擊隊的勝算，伊薩決定加入游擊隊，與哈希爾並肩作戰，哈希爾的表現很英勇，伊薩則顯得戰戰兢兢。兩人先後被荷軍逮捕以後，一方面伊薩受到折磨還是堅毅不屈，另一方面哈希爾不復昔日的勇敢，經過一番虐打以後，就把游擊隊的動向告訴給荷蘭人{{sfn|Mahayana|Sofyan|Dian|2007|p=117-120}}。

==  發行與評價 ==
《路無盡頭》在1952年由[[圖書編譯局|圖書編譯局]]出版，並於一年後獲得全國文化協商機構的文學獎{{sfn|KS|2010|p=157–159}}。[[H·B·耶辛|H·B·耶辛]]、[[A·德歐|A·德歐]]、[[穆罕默德·巴爾法斯|穆罕默德·巴爾法斯]]{{sfn|Balfas|1976|p=92}}等文學評論家都認為《路無盡頭》是盧比斯最好的作品。

這部小說在1968年由A·H·約翰斯（A.H. Johns）翻譯成英文，然後在1988年翻譯成中文，此外還翻譯成日文、韓文、意大利文、荷蘭文、印地文和泰文{{sfn|Hill|2010|p=210}}，並成為印尼國內外多篇論文的討論題材{{sfn|KS|2010|p=157–159}}。
