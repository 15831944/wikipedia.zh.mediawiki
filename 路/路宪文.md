'''路宪文'''（{{bd|1919年||1995年|}}）[[山西省|山西省]][[陵川县|陵川县]][[附城镇_(陵川县)|附城镇]]河东村人<ref name=mzz> {{Cite book | author = 山西省陵川县民政局 编，路书静、张天纯 主编 | title = 陵川县民政志 | location =  | publisher = 中国社会出版社 | date = 2001年 | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。[[中国共产党|中国共产党]]官员，[[信阳事件|信阳事件]]的主要责任人。

==生平==
路宪文的家乡是附城河东村。自幼在附城读书。1935年考入[[上党|上党]]乡村师范学校，在校期间加入[[牺盟会|牺盟会]]。后考入山西省牺盟会协助员培训班。结业后，起初任[[平顺县|平顺县]]总动员委员会协助员，后来回到[[陵川县|陵川县]]牺盟会，在陵川县牺盟会附城分会担任协助员<ref name=ma/>。

1937年，[[八路军|八路军]][[赵基梅|赵]][[谭甫仁|谭]]支队通过宣传队的方式进入附城进行抗日宣传，后由八路军工作团在附城秘密发展[[中国共产党|中国共产党]]党员。1937年，路宪文、侯景域成为宣传队队长钟远昌、指导员曾旭庆介绍入党的首批党员。1938年初，中共陵川县工委决定在陵川县牺盟会附城分会成立中共临时支部，支部书记由中共陵川县工委组织委员孟宪德兼任，路宪文任组织委员，侯景域任宣传委员。这也是陵川县历史上第一个中共基层党支部。1938年5月，中共陵川县委成立，路宪文任中共陵川县委宣传部部长<ref name=ma/><ref>{{Cite web |url =http://www.jconline.cn/Contents/Channel_14831/2013/1029/1007455/content_1007455.htm  |title = 中共陵川县委成立地旧址 |author =  |date = 2013-10-29 |publisher = 晋城在线 |language =  |accessdate =  |quote =  }}</ref>。

1939年12月，山西“[[晋西事变|十二月事变]]”发生。1939年12月中旬，[[中国国民党|中国国民党]]方面的[[山西省政府|山西省政府]]撤销了中国共产党党员[[张维汉|张维汉]]的陵川县县长职务，任命中国国民党方面的[[张生兆|张生兆]]为县长。中共方面的[[晋冀鲁豫边区|晋冀鲁豫边区]]主动与山西的中国国民党[[阎锡山|阎锡山]]当局谈判协商后达成协议，撤出了[[太行区|太行区]]太南陵川县等地，史称“太南撤军”。中共太南特委也调整了中共陵川县委的领导班子，原中共陵川县委宣传部部长路宪文调任[[八路军|八路军]]一二九师太南支队（[[赵基梅|赵基梅]]支队）驻陵工作团（民运工作团）主任兼中共陵川县委军事部部长，留在陵川县坚持地下斗争<ref name=guling>{{Cite web|url =http://www.lczf.cn/DocHtml/1/16/01/00034878.html|title =“新庄村留人小店”情报联络站|author =|date =2015-09-25|publisher =古陵之窗|language =|accessdate =|quote =}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。1940年起，任陵川县子弟兵团政治委员。1940年3月随军北撤后，历任中共平顺县委统战部部长、中共平南县委宣传部部长、平顺县武委会主任兼武工队副政委<ref name=ma>{{Cite web |url = http://www.jcszgdsxh.com/h-nd-2116-2_723.html |title =第二回 八路军开辟根据地 太行区设立陵高县  |author = 马栓贵 |date =2015-04-16  |publisher = 晋城党史网 |language =  |accessdate =  |quote =  }}</ref>。

1943年3月，中共[[晋冀豫区|晋冀豫]]四地委作出了开辟陵川县工作的决定之后，路宪文起初任中共陵川县委宣传部部长。1943年4月，路宪文受中共[[太行区|太行区]]党委、[[太行军区|太行军区]]和四专署、军分区的指示，率部深入陵川县开展了5次武装侦察。1943年5月，中共陵川县委组成陵西、陵东两支武装工作队深入陵川县侦察，路宪文任陵西队队长。1943年8月，被晋冀豫四专署提名为陵川县抗日民主政府代理县长<ref name=ma/>。

1943年9月，中共太行八地委、八专署、八军分区在陵川县成立。1943年10月，中共太行八地委、太行八专署决定在陵川县、高平县、晋城县交界处建立[[陵高县|陵高县]]，名称取自陵川的“陵”字和高平的“高”字，设立中共陵高县委、陵高县抗日民主政府<ref name=guling/><ref name=ma/>。1943年11月下旬某晚，新任命的陵高县抗日民主政府县长兼独立营营长路宪文，在八路军一二九师新一旅二团（当地人称“老二团”）三个连的配合下奇袭了附城，击溃并驱逐了日伪三区区长郭成印的日伪中队，在附城举行群众大会宣布中共陵高县委、陵高县抗日民主政府成立。当夜，新成立的中共陵高县委、陵高县抗日民主政府的机关工作人员分批秘密进驻新庄村<ref name=guling/>。

1945年，路宪文南下工作，历任中共[[沁阳县|沁阳县]]委书记兼独立营政委，南下干部大队四中队队长。1947年10月，任中共鄂豫一地委[[霍固县|霍固县]]委书记兼前敌指挥部政委（该霍固县为北霍固县，1947年10月设置）<ref>{{Cite web |url =http://www.gsw.gov.cn/html/hsly/fhsy/34689_3.html  |title =  固始县革命斗争史简介|author =  |date = 2010-09-01 |publisher = 固始网 |language =  |accessdate =  |quote =  }}</ref>。1949年1月，[[中国人民解放军|中国人民解放军]]攻占霍邱县城，霍固县奉命撤销，恢复[[霍邱县|霍邱县]]<ref>{{Cite web |url =http://www.gsw.gov.cn/html/zoujingushi/gushigailan/lishiyange/6291_2.html  |title =固始县历史沿革  |author =  |date =2008-10-09  |publisher =  固始网|language =  |accessdate =  |quote =  }}</ref>。路宪文任中共霍邱县委副书记兼三元店区委书记。后任中共鄂豫一地委党校党委副书记，中共[[商城县|商城县]]委副书记兼组织部长。1949年7月至1950年，任中共商城县委书记<ref>{{Cite web |url = http://www.scxx.ha.cn/scda/ShowArticle.asp?ArticleID=297 |title = 商城县档案馆指南 |author =  |date =2006-11-28  |publisher =商城县档案局  |language =  |accessdate =  |quote =  }}</ref>。此后历任潢川专员公署副专员兼财委主任，报务局局长。1952年至1954年，任中共信阳地委副书记。1954年6月，代理中共信阳地委书记。1954年10月至1958年，任中共信阳地委书记。1958年，中共信阳地委开始设置第一书记。1958年至1960年9月，路宪文任中共信阳地委第一书记兼信阳军分区第一政委<ref>{{Cite web |url =http://www.xydsw.gov.cn/Item/Show.asp?m=1&d=1593  |title = 信阳市历任地、市委书记名录一览表 |author =  |date = 2011-07-14  |publisher =  信阳党史网|language =  |accessdate =  |quote =  }}</ref>。

1960年9月，路宪文因“[[信阳事件|信阳事件]]”被撤职。1962年，[[河南省高级人民法院|河南省高级人民法院]]以“反革命报复罪”和“违法乱纪”判处路宪文有期徒刑三年。1964年，任河南省博爱县国营农场副厂长。1975年重新加入中国共产党<ref> {{Cite book | author =喬培華  | title =信陽事件  | location = 香港 | publisher = 開放出版社 | date =2009年  | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref><ref> {{Cite book | author = 余习广 主编 | title = 血屠：信阳事件认罪书集 | location =  | publisher =  | date =  | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。1980年2月至1983年9月，任河南省百泉农业专科学校（现为[[河南科技学院|河南科技学院]]）校长、校党委委员。1983年9月离休<ref>{{Cite web |url =  http://xyh.hist.edu.cn/piclist3.jsp?urltype=tree.TreeTempUrl&wbtreeid=1549|title = 历任领导 |author =  |date =  |publisher = 河南科技学院校友会 |language =  |accessdate =2017-05-27  |quote =  }}</ref>

1995年，路宪文在[[河南|河南]]病逝<ref name=mzz/>。
