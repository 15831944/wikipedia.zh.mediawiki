'''路甬祥'''（{{bd|1942年|4月28日|catIdx=L}}），[[浙江|浙江]][[慈溪|慈溪]]人；中国当代[[流体传动|流体传动]]及[[控制学|控制学]]专家，[[浙江大学|浙江大学]][[机械工程|机械工程]]系毕业，[[联邦德国|联邦德国]][[亚琛工业大学|亚琛工业大学]][[工程学|工程学]][[博士|博士]]。1974年5月加入[[中国共产党|中国共产党]]，曾任[[中国科学院|中国科学院]]院长；现任[[全国人大常委会副委员长|全国人大常委会副委员长]]，[[国际科学院理事会|国际科学院理事会]]（IAC）共同主席，[[中国机械工程学会|中国机械工程学会]]理事长，[[浙江大学|浙江大学]][[教授|教授]]、[[博士生导师|博士生导师]]，[[清华大学|清华大学]]兼职教授，[[香港大学|香港大学]]名誉教授等；[[中国科学院|中国科学院]]和[[中国工程院|中国工程院]]“[[中华人民共和国两院院士制度|双院士]]”。

== 早年经历 ==
路甬祥于1942年4月28日出生在[[宁波|宁波]][[慈溪|慈溪]]的一个西医家庭。1932年时，父亲[[路念慈|路念慈]]在[[慈溪|慈溪]]创办了一家医院，自任院长和全科医生；路甬祥的母亲李芳则是医院的[[助产士|助产士]]。根据《[[慈溪县方志|慈溪县方志]]》记载：“路念慈精于内外科... ...对贫病者免收诊金、酌收药费... ...”<ref>[http://www.cast.org.cn/n35081/n10371505/10378565.html 路甬祥：为国家和民族的崛起强筋壮骨] {{webarchive|url=https://web.archive.org/web/20100317164410/http://www.cast.org.cn/n35081/n10371505/10378565.html |date=2010-03-17 }}</ref> 路甬祥自幼勤奋，并受到家庭环境影响，信奉科学技术。1956年，14岁时，因成绩优异，被保送升入[[宁波市第二中学|宁波市第二中学]]高中部学习。路甬祥从不满足一般的课堂学习，在初中时，就开始自学高中课程；而在高中时，则开始自学大学基础课。1959年，他考入[[浙江大学|浙江大学]]机械系，攻读水力机械专业。

1964年，大学毕业的路甬祥被留在学校任教。但当时规定，大学毕业生都必须到农村去参加“[[社会主义教育运动|社会主义教育运动]]”，路甬祥就被分配到[[浙江|浙江]][[诸暨|诸暨]]做“[[社会主义教育运动|社教]]”工作。一年后回到[[浙江大学|浙江大学]]，任机械工程系助教、讲师。在他工作后不久，就和中学同学刁琳琳结婚。婚后，二人育有两个孩子。

== 继续深造 ==
在[[文革|文革]]结束后，中国的政治开始环境改善，对于学术活动也逐渐重视起来。因为在[[文革|文革]]中，路甬祥依旧坚持研究工作，他很快获得学校推荐，并通过了[[联邦德国|联邦德国]]洪堡基金会的遴选，赴德国深造。1979年2月7日，37岁的路甬祥赴[[亚琛工业大学|亚琛工业大学]]液压气动研究所，做研究员；并于1981年5月1日，获得工程科学博士学位。这是“[[改革开放|改革开放]]”后，[[联邦德国|联邦德国]]授予[[中华人民共和国|中华人民共和国]]公民的第一个工程科学博士学位。

== 回国创业 ==
1981年，路甬祥回国。此后，他一直任教于[[浙江大学|浙江大学]]；历任讲师、副教授、教授，流体传动研究室主任、机械工程系流体传动研究所所长等职。1985年出任[[浙江大学|浙江大学]]副校长，1988年升任校长。

1991年1月，任国务院学位委员会副主任委员。

1992年1月，路甬祥当选为[[中国科学院|中国科学院]]技术科学部委员；1993年，出任[[中科院|中科院]]副院长；1994年又升任常务副院长，同年还当选[[中国工程院|中国工程院]]院士；1997年7月起，任[[中国科学院|中国科学院]]院长<ref>[http://news.xinhuanet.com/ziliao/2002-03/05/content_300460.htm 新华网：路甬祥简历] {{webarchive|url=https://web.archive.org/web/20090305100948/http://news.xinhuanet.com/ziliao/2002-03/05/content_300460.htm |date=2009-03-05 }}</ref>，2011年2月卸任<ref>[http://news.xinhuanet.com/edu/2011-03/08/c_121164093.htm 纳米科技研究专家白春礼接任中国科学院院长] {{webarchive|url=https://web.archive.org/web/20140119000523/http://news.xinhuanet.com/edu/2011-03/08/c_121164093.htm |date=2014-01-19 }}</ref>。

期间，路甬祥还先后出任过[[中国科协|中国科协]]副主席、国家教委高等教育咨询委员会主席、中国科技史学会理事长、香港特别行政区“[[行政长官特设创新科技委员会|行政长官特设创新科技委员会]]”委员、[[发展中国家科学院|发展中国家科学院]]（TWAS）副院长、中国机械工程学会理事长、国际科学院理事会（IAC）共同主席、中国航空运动协会名誉主席等职。

2012年5月，任中国知识产权研究会誉理事长<ref>[http://www.cnips.org/gk_news.asp?NewsID=37 名誉理事长] {{webarchive|url=https://web.archive.org/web/20140426232647/http://www.cnips.org/gk_news.asp?NewsID=37 |date=2014-04-26 }}</ref>。2012年12月，作为[[胡锦涛|胡锦涛]]主席特使出使[[墨西哥|墨西哥]]，拜会墨西哥时新任总统[[培尼亚·涅托|培尼亚·涅托]]<ref>[http://news.xinhuanet.com/world/2012-12/02/c_113876120.htm 胡锦涛主席特使路甬祥会见墨西哥新任总统涅托] {{webarchive|url=https://web.archive.org/web/20121206085455/http://news.xinhuanet.com/world/2012-12/02/c_113876120.htm |date=2012-12-06 }}</ref>。

2013年12月，当选[[中国产学研合作促进会|中国产学研合作促进会]]会长<ref>[http://news.sciencenet.cn/htmlnews/2013/12/286720.shtm 路甬祥当选中国产学研合作促进会会长]</ref>。2014年10月，当选中国创新设计产业联盟会长<ref>[http://zjnews.zjol.com.cn/system/2014/10/11/020297410.shtml 中国创新设计产业战略联盟在杭成立 路甬祥担任会长]</ref>。

在政治上，是中共第十二届（[[1985年中国共产党全国代表会议|1985年9月中共全国代表会议]]增补）、十三届中央候补委员，第十四届、十五届、十六届、十七届中央委员，第十届、十一届全国人大常委会副委员长。

== 学术成就 ==
*是中国流体传动及控制学家，特别是在电液比例控制新技术方面<ref>[http://www.china.com.cn/chinese/2003/Mar/293790.htm 中国网：路甬祥]</ref>。
*在机械工程方面获得多项欧美专利；如“二通插装式电液比例调速装置”等。
*“创造性地提出了“系统流量检测力反馈”、“系统压力直接检测和反馈”等新原理，并将其应用于先导流量和压力控制器件，改变了已沿用100多年的弗利明－琴肯流量控制原理和40多年来传统的维克斯先导型压力控制原理，取得了“二通插装式电液比例流量控制装置”及“电液比例压力控制装置”等五项发明专利，使大流量和高压领域内的稳态和动态控制精度获得显著提高。并运用这些原理和机－电－液一体插装技术相结合，推广应用于阀控、泵控和液压[[马达|马达]]等，成功地研究开发了一系列新型电液控制器件及工程系统，该技术被认为是八十年代以来电液控制技术重要进展之一，被[[德国|德国]]、[[日本|日本]]、[[瑞士|瑞士]]等国列入教材、手册专著。”
*“主持开发研究了相应的CAD、CAT支撑系统，被广泛应用于许多工业部门。”
*创建[[浙江大学|浙江大学]]流体传动与控制研究所，为中国这一领域的重要研究中心之一。
*发表科学研究和工程教育论文250多篇，另有专著5部。
