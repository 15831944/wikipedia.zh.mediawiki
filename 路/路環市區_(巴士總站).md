'''路環市區'''（{{Lang-pt|'''VILA DE COLOANE'''}}、{{lang-en|'''Coloane Village'''}}）位於[[澳門|澳門]][[路環|路環]]恩尼斯總統前地的一個公車站。[[澳門新福利|新福利]]'''[[澳門巴士25路線|25]]'''和[[澳巴|澳巴]]'''[[澳門巴士50路線|50]]'''路線以該站為終點站。[[澳巴|澳巴]]'''[[澳門巴士N3路線|N3]]'''路線(正線)以該站為循環點。

== 歷史 ==
* 2017年5月6日，因[[澳門巴士25路線|25]]路線取消行駛路環市區至黑沙海灘的一段行程而設立本站，讓[[澳門巴士25路線|25]]、[[澳門巴士50路線|50]]路線作為該路線的總站。<ref>{{cite news |title=25路線周六起調整行程加密班次提升效率 |url=http://www.dsat.gov.mo/dsat/news_detail.aspx?data_category=1&a_id=3882 |publisher=[[交通事務局|交通事務局]] |date=2017-05-04}}</ref>
*2018年9月1日，[[澳門巴士N3路線|N3]]路線調整為循環線，本站僅作為中途站。

== 設施 ==

== 各車道安排 ==
只有一條車道

== 爭議 ==
本站設於一個雙線行車的細小迴旋處內，整個站點的長度僅約二十米，即表示第一部十二米巴士停站後，路面只剩下一線；第二部十二米巴士靠站後，整個迴旋處即告癱瘓，根本不能作總站。此情況逼使[[澳門巴士25路線|25路線]]的車長在連續駕駛接近1.5小時，由澳門最北的[[關閘總站|關閘]]總站駛至位於澳門南端的本站後，放棄在總站的中場休息時間，直接離站返回關閘，此運作模式與循環線無異。
