'''路氹邊檢大樓'''（{{lang-pt|'''Posto Fronteiriço de Cotai'''}}、{{lang-en|'''Cotai Frontier Post'''}}），俗稱'''蓮花口岸'''，位於[[路氹城|路氹城]]蓮花路，為[[澳門|澳門]]與[[中國|中國]][[廣東省|廣東省]][[珠海市|珠海市]]三條陸路邊境通道之一，也是兩地第二條的陸路邊境通道。實行24小時通關，是兩地唯一全天候開放的正式口岸。邊檢站通過[[蓮花大橋|蓮花大橋]]連接[[橫琴口岸|橫琴口岸]]。

== 沿革 ==
* '''路氹邊檢大樓'''於2000年3月28日隨着蓮花大橋通車啟用，是第二個澳門與珠海市陸上交通邊境。當時旅客通關時間爲上午9時至下午5時、貨運部分則爲上午7時至下午5時。路氹邊檢大樓的落成，分流了部分[[關閘邊檢大樓|關閘邊檢大樓]]中澳兩地客、貨物運送之壓力。

* 2002年1月1日，邊檢大樓延長通關時間，旅客及貨運通關時間爲上午9時至晩上8時<ref>{{cite news |language = zh-hant|author =  |coauthors = |url = |title = 橫琴延長通關促珠澳發展|publisher = 澳門日報|pages =  |date = 2002年1月2日 |accessdate = }}</ref>。當時途經大樓的巴士路線只會在通關時間進入，其餘營運時間則會跳過此站。

* 2005年9月17日，由於[[珠海市|珠海市]][[橫琴口岸|橫琴口岸]]旅客出入境大堂結構出現問題而關閉重建，路氹邊檢大樓也同步禁止非乘車旅客通行，但出入境的行車通道仍然繼續運作，因此貨運以及駕車過關沒有到影響。旅客出入境大堂直至2007年5月1日重新恢復。<ref>[http://www.gd.xinhuanet.com/newscenter/2007-04/30/content_9935012.htm 珠海横琴口岸“五一”恢复通关] {{webarchive|url=https://web.archive.org/web/20090304121012/http://www.gd.xinhuanet.com/newscenter/2007-04/30/content_9935012.htm |date=2009-03-04 }}</ref>

* 2014年12月18日，路氹邊檢大樓開放時間改為24小時開放，是澳門與[[中國大陸|中國大陸]]首個全天候開放的正式口岸，值得一提，雖然[[跨境工業區邊檢大樓|跨境工業區邊檢大樓]]為首[[澳門|澳門]]首個24小時通關的口岸，但它屬於專用口岸。

== 過關模式 ==
路氹邊檢站不同於關閘邊檢站的過關模式，由於中澳兩座邊檢大樓之間是經由蓮花大橋連接，而且蓮花大橋禁止行人通行，因此所有出入境人士必須於禁區乘搭專線巴士前往對岸之邊檢大樓。往返兩地的[[蓮花口岸專線巴士|專線巴士]]目前由[[岐關車路有限公司|岐關公司]]及[[澳門公共汽車有限公司|澳巴]]共同營運，車費為[[人民幣|人民幣]]、[[澳門幣|澳門幣]]、[[港幣|港幣]]6元，可以使用澳門通付款。
