'''路克·培格勒'''（{{lang-en|'''Luke Pegler'''}}）是一名[[澳洲|澳洲]]演員。

== 生涯 ==
培格勒曾經在兩部[[世界摔角娛樂電影製片廠|世界摔角娛樂電影]]中演出：《''{{link-en|The Condemned|The Condemned}}''》<ref>{{Cite web |url = http://www.comingsoon.net/movies/reviews/20061-the-condemned-2 |title = The Condemned |author = Edward Douglas |date = 2007-04-25 |publisher = comingsoon.net |language = en}}</ref>和《[[非禮勿視|非禮勿視]]》。2008年至2009年，他在《''{{link-en|Packed to the Rafters|Packed to the Rafters}}''》中飾演Daniel Griggs。
