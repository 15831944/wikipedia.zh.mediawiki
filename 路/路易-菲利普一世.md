'''路易-菲利普一世'''（{{lang-fr|'''Louis-Philippe I<sup>er</sup>'''}}，{{bd|1773年|10月6日|1850年|8月26日|catIdx=P}}），[[法国君主列表|法国国王]]（1830－1848年）。又稱「路易腓力」。

== 法国大革命前 ==
[[奥尔良公爵|奥尔良公爵]][[路易·腓力二世_(奥尔良公爵)|路易·腓力·约瑟夫]]之子。1789年繼承瓦卢瓦公爵，1793年其父被雅各宾派处决后继承奥尔良公爵及内穆尔公爵称号。

== 法国大革命期间 ==
1789年[[法国大革命|法国大革命]]爆发，他参加支持革命政府的进步贵族团体，翌年参加[[雅各宾俱乐部|雅各宾俱乐部]]。1792年4月法奥交战，他参加北方方面军，同年9月晋升少将。

== 流亡期间 ==
1793年4月他偕北方方面军指挥官杜穆里埃（Dumouriez）投奔奥军，并去[[瑞士|瑞士]]避难。同年11月其父被雅各宾派政府处决时，他成为奥尔良公爵。他在[[美国|美国]]住了两年多，然后决定返欧。他于1800年初到达[[英格兰|英格兰]]，在英格兰长期居住。1809年去[[西西里|西西里]]，与[[西西里和那不勒斯君主列表|那不勒斯国王]]的女儿玛丽·阿玛麗結婚。

== 法国国王 ==
1814年[[路易十八|路易十八]]第一次复辟时路易·腓力返回法国。1830年[[查理十世_(法兰西)|查理十世]]试图推行镇压法令，触发1830年的[[法國七月革命|法國七月革命]]（7月27日－7月30日）。7月31日，立法议会选举他为王国摄政。两天后查理退位，8月9日，路易·菲利普加冕为法国国王。他在右翼极端君主派和社会党人及其他共和党人之间采取中间路线，以巩固自己的权力。1846年工农业萧条，人民普遍不满，引發[[法國二月革命|法國二月革命]]。他於1848年2月24日[[逊位|逊位]]，隐居于英格兰的[[薩里郡|萨里]]并于1850年去世。
