'''路易十六'''（{{lang-fr|Louis XVI}}；{{bd|1754年|8月23日|1793年|1月21日|catIdx=Louis}}），原名'''路易-奥古斯特'''（Louis-Auguste），亦名'''路易-卡佩'''（Louis Capet），[[法国君主列表|法国国王]]，1774年即位，1791年后正式头衔更改为[[法国君主列表|法国人的国王]]，1792年被废黜，并于次年1月21日被送上[[断头台|断头台]]。其父[[法兰西的路易_(1729年–1765年)|法兰西太子路易]]为[[路易十五|路易十五]]之子及继承人，但于1765年早逝；1774年，路易继承其祖父王位。

路易在其统治前期试图根据[[开明专制|启蒙运动]]理念对法国进行改革。这些举措包括试图废除[[农奴制|农奴制]]和{{le|土地稅|taille}}，并对非天主教徒实行宗教宽容政策。{{le|法国贵族|French nobility}}反对改革，并成功抑止了这些措施的推行。路易亦在其[[重农主义|自由派]]大臣[[劳恩男爵安·罗伯特·雅克·杜尔哥|杜尔哥]]支持下放松了对[[粮食|粮食]]市场的管制，但这一措施导致了面包价格的上升，加之这一时期粮食歉收，食物短缺最终引发民众暴动。1776年起路易十六开始积极支持北美殖民者的[[美国独立战争|反英独立斗争]]，最终通过《[[巴黎条约_(1783年)|巴黎条约]]》得以实现。

债务和财政危机导致[[旧制度|旧制度]]愈发不得人心，最终促成了[[1789年法国三级会议|1789年法国三级会议]]的召开。法国中层阶级和下层阶级对法国贵族及法国君主专制愈发不满，路易十六及其王后[[玛丽·安托瓦内特|玛丽·安托瓦内特]]则成为了这一制度的代表人物。1789年，巴黎发生动乱，[[攻占巴士底狱|巴士底狱被冲击]]，[[法国大革命|法国大革命]]就此开始。

路易本性优柔寡断且具一定保守倾向，由此法国民众视其为旧制度暴政的象征，其受欢迎程度快速下滑。1791年6月的[[路易十六的出逃|瓦雷内出逃]]（君主立宪制建立四个月前）使民众愈发认为其希望借助[[法国大革命战争|外国入侵]]以保其政治地位，君主制的废除和共和制的建立愈显可能。

在[[第一次反法同盟|内战与外战]]交织之下，路易十六在1792年[[八月十日事件|八月十日事件]]中被逮捕，一个月后（9月21日）君主立宪制被废除，[[法兰西第一共和国|法兰西第一共和国]]建立。[[国民公会|国民公会]]对其进行审判，认定其犯[[背叛国家罪|叛国罪]]，并于1793年1月21日将是时的“公民路易-卡佩”（取至[[卡佩王朝|卡佩王朝]]创立者[[于格·卡佩|于格·卡佩]]；革命者认为卡佩当为路易的姓氏）送上[[断头台|断头台]]。路易十六为法国历史上唯一被处决的君主，其死亡宣告了延续近千年法国君主制的终结。

== 即位 ==
路易十六有一顆良好仁心與治國善念，但性格优柔寡断，即位后多次更換首相和部長，任由内阁内讧，从激进的改革到保守的节俭措施，政策沒有一定的連貫，但總的來說是朝向[[開明專制|開明專制]]。18世纪80年代法国陷入财政危机后，更经常靠製鎖與打猎消磨時間，有些人因此稱他為「鎖匠國王」。<ref name="陳文海">陳文海，《法國史》（北京：人民出版社，2004），頁212</ref>

即位後，路易十六因為參與了1775-1783年的[[美國獨立戰爭|美國獨立戰爭]]（由忠心能幹的外交大臣[[夏爾·格拉維耶|韋爾熱訥]]主導），欠下十數億里弗的國債，使得國債飆升至1787年的25億里弗（依照不同的計算方法，1788年已將近四十億里弗），面臨無力償債付息的困境。<ref name="Palmer">R.R.Palmer等著、孫福生等譯，《現代世界史（前篇）──從歐洲興起到1870年》（台北：五南，2012），頁401-403</ref>

== 法國大革命 ==
1789年5月，在首席财政大臣[[雅克·内克爾|雅克·内克爾]]等人的敦促下召开[[三级会议|三级会议]]，以解決迫在眉睫的國債危機。人民正面看待國王召開三級會議，稱他為「好國王路易」（或翻成「賢明王路易」）。<ref name="Palmer"/>開會時，三個等級發生劇烈爭執，第三等级提出[[制宪|制宪]]要求，但路易態度猶豫不決，加上1788年酷寒的大天災與同時開放英國製造品造成的經濟崩潰，导致了同年7月14日[[法国大革命|法国大革命]]的爆发。路易被迫签署《[[人权宣言|人权宣言]]》，10月后被迫从[[凡尔赛宫|凡尔赛宫]]遷居[[巴黎|巴黎]]（[[凡爾賽婦女大遊行|凡爾賽婦女大遊行]]）。此後一直到1791年六月[[路易十六的出逃|出逃]]前，路易都保持著高漲的民意支持度。公眾讚賞他願意放棄[[專制|專制]]王權，並能夠配合人民的要求而退居為[[立憲|立憲]]君王；但是路易對於[[國民制憲議會|國民制憲議會]]所實施的激烈反教會政策，內心越來越不滿，因為他是個篤實虔誠的[[天主教徒|天主教徒]]，不願意看到反教會政策撕裂了教士與人民（當時許多外省農民支持當地教會，與[[巴黎|巴黎]]的議會政府發生流血抗爭），於是他開始醞釀恢復有力的王權。

== 出逃失敗後被處決 ==
{{See also|路易十六的出逃|八月十日事件}}
1791年6月20日他化装成平民，全家一同乘马车[[路易十六的出逃|出逃]]。结果被发现后押送回巴黎，扣押在[[杜伊勒里宫|杜伊勒里宫]]，從此法國市民（[[無套褲漢|無套褲漢]]）開始敵視他。同年9月他批准[[法国1791年宪法|宪法]]，自称“法兰西人的国王”。1792年4月法國对[[奥地利|奥地利]]的[[哈布斯堡君主國|哈布斯堡君主國]]宣战，[[八月十日事件|8月10日巴黎眾人革命]]后他被捕，同年9月[[君主制|君主制]]被废除。

1793年1月16日－17日被[[国民公会|国民公会]]判处[[死刑|死刑]]，1月21日被送上[[断頭臺|断頭臺]]。諷刺的是，傳說路易十六当年曾亲自参与了[[斷頭臺|斷頭臺]]的设计，为減輕受刑者的痛苦，他還命人将铡刀改成[[三角形|三角形]]，没想到自己却死在这部杀人利器之下。他七歲的兒子[[路易十七|路易十七]]在獄中三年，一日僅有一餐，隨地大小便，連洗澡都得不到准許，最後被虐待致死。

== 評價 ==

1989年7月14日，法國慶祝革命200週年的慶典上，法國總統[[密特朗|密特朗]]表示，“路易十六是個好人，把他處死是件悲劇，但也是不可避免的。”；路易十六被處死時，他在外省仍有頗高的支持度，大多數的法國農民仍然敬愛並遵奉他（法國農民占人口的八成以上），但是以[[巴黎|巴黎]]為首的市民卻嚴重敵視並仇恨他，在觀賞處決時報以歡聲雷動，慶祝共和國的確立。如同[[羅伯斯比爾|羅伯斯比爾]]所說的：「路易必須死，因為共和必須生。<ref>https://zh.m.wikipedia.org/zh-tw/马克西米连·罗伯斯庇尔</ref>」
