'''路德維希菊石'''（[[學名|學名]]：{{lang|la|'''''Ludwigia'''''}}），又名'''洛德威菊石'''，是生存在[[中侏羅世|中侏羅世]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]。其[[化石|化石]]分布於[[歐洲|歐洲]]、[[伊朗|伊朗]]和[[突尼西亞|突尼西亞]]等地。<ref name="Fossilworks">{{Cite web |url = http://fossilworks.org/bridge.pl?a=taxonInfo&taxon_no=15136 |title = Fossilworks: Ludwigia |language = en |accessdate = 2018-08-03 }}</ref>

== 特徵 ==
路德維希菊石的殼體具有微彎而分岔的肋條，肋條非常突出，但隨著年紀增長，其外環會愈來愈光滑，致使肋條變得較不明顯。其平坦的腹側上具有單一的腹稜。牠們的外觀和其近親{{tsl|en|Graphoceras|葛拉佛菊石}}十分相似。<ref name="Ludwigia">{{Cite web |url = http://liusir.pixnet.net/blog/post/248637026 |title = Ludwigia 路德維希菊石(英國2) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-03 }}</ref>
