{{日本作品|老師！請讓我休息！|先生！休ませてください！|}}是{{lang|ja|[[吉辺あくろ|吉辺あくろ]]}}於[[COMIC_CUNE|COMIC CUNE]]創作的[[四格漫畫|四格漫畫]]，於2015年8月27日開始連載，2019年1月26日結束連載。

== 故事簡介 ==
保健室，據說這是一些學生放鬆的地方。睡覺、看書、吃零食……保健室是一個歡樂的空間。這裡是某所高校的保健室，在這裡經常出現的有個性陰沉的文學少女、隨和的不良少女、天然的體育系少女、體弱的大小姐、合法的蘿莉……。女高中生們的保健室翹課喜劇！

: 陰沉系（{{lang|ja|根暗系}}）文學少女。
: 只要到了不喜歡的課程就會到保健室翹課。

: 隨和的不良少女。
: 隨身攜帶起司魚糕（{{lang|ja|チーかま}}{{Notetag|{{lang|ja|株式会社丸善}}的註冊商標，使用起司和魚醬製成的糕狀食品。}}）。

: 體弱的大小姐。
: 保健室老師很擔心她。

: 班導師。
