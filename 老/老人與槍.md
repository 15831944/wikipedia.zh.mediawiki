《'''老人與槍'''》（{{lang-en|''The Old Man & the Gun''}}）是一部於[[2018年電影|2018年]]上映的美國[[犯罪片|犯罪]][[喜劇片|喜劇]][[劇情片|劇情片]]<ref name="Metacritic"/>，由[[大衛·羅利|大衛·羅利]]自編自導，劇情改編自《[[紐約客|紐約客]]》專欄作家{{link-en|大衛·葛蘭|David Grann}}的同名短篇文章。該片由[[勞勃·瑞福|勞勃·瑞福]]、[[凱西·艾佛列克|凱西·艾佛列克]]、[[丹尼·葛洛佛|丹尼·葛洛佛]]、{{link-en|蒂卡·桑普特|Tika Sumpter}}、[[汤姆·威茨|湯姆·威茲]]與[[西西·史派克|西西·史派克]]等人主演，劇情講述年老的銀行搶匪{{link-en|佛雷斯特·塔克|Forrest Tucker (criminal)}}（瑞福飾演）的故事。該片也是瑞福的演藝生涯的謝幕之作。

《老人與槍》於2018年9月28日在北美院線上映，所獲評價以正面為主。

== 劇情 ==

該片根據真人真事改編，講述前科累累的{{link-en|佛雷斯特·塔克|Forrest Tucker (criminal)}}（[[勞勃·瑞福|勞勃·瑞福]]飾演）在邁入老年時仍繼續搶銀行的故事<ref name="Trailer"/>。

== 角色 ==

* [[勞勃·瑞福|勞勃·瑞福]]飾演{{link-en|佛雷斯特·塔克|Forrest Tucker (criminal)}}，曾犯過不少搶案的老人，如今繼續搶銀行<ref name="Trailer"/>。2018年8月6日，瑞福宣布，該片將是他的息影之作，並表示，「我十分肯定地說，我的演出生涯到此為止了，因為我已經從21歲演到現在，我認為我演夠了」<ref name="Redford"/><ref name="Redford2"/><ref name="Redford3"/><ref name="Redford4"/>。不過對於未來是否會再出任導演，瑞福表示「再看看吧」<ref name="Redford2"/>。然而，2018年9月，瑞福在出席該片的首映會時表示自己很後悔宣布息影，並解釋道「如果我打算退休，我應該悄悄地離開，不應公開談論造成大家過度關注，我希望大家能聚-{焦}-在這部電影和我演出的角色」<ref name="Redford5"/><ref name="Redford6"/>。
* [[凱西·艾佛列克|凱西·艾佛列克]]飾演約翰·亨特，追捕佛雷斯特的警察<ref name="Trailer"/>。
* [[丹尼·葛洛佛|丹尼·葛洛佛]]飾演泰迪·葛林<ref name="Trailer"/>
* {{link-en|蒂卡·桑普特|Tika Sumpter}}飾演莫琳<ref name="Trailer"/>
* [[汤姆·威茨|湯姆·威茲]]飾演華勒<ref name="Trailer"/>
* [[西西·史派克|西西·史派克]]飾演珠兒，佛雷斯特的愛人<ref name="Trailer"/>。
* [[伊莉莎白·莫斯|伊莉莎白·莫斯]]飾演<ref name="Deadline.com"/><ref name="Deadline.com2"/>
* {{link-en|小以賽亞·惠特洛克|Isiah Whitlock, Jr.}}飾演吉恩·丹特勒警探<ref name="Deadline.com"/><ref name="Deadline.com2"/>
* {{link-en|基斯·卡拉丁|Keith Carradine}}飾演考爾德<ref name="Deadline.com3"/>

== 製作 ==

2010年，{{link-en|自身影業|Identity Films}}的{{link-en|安東尼·馬斯特莫羅|Anthony Mastromauro}}買下了《老人與槍》的版權<ref name="Identity"/>。此前，該片的版權由[[華納兄弟|華納兄弟]]握有<ref name="Identity"/>。該片改編自《[[紐約客|紐約客]]》專欄作家{{link-en|大衛·葛蘭|David Grann}}的同名短篇文章<ref name="Yahoo! 奇摩電影"/>。2016年10月，據報導，[[勞勃·瑞福|勞勃·瑞福]]與[[凱西·艾佛列克|凱西·艾佛列克]]將擔綱主演，而[[大衛·羅利|大衛·羅利]]則將自編自導該片<ref name="Variety"/><ref name="EW"/>。比爾·霍德曼（{{lang|en|Bill Holderman}}）、馬斯特莫羅、{{link-en|道恩·奧斯托夫|Dawn Ostroff}}、瑞福、傑瑞米·史特克勒（{{lang|en|Jeremy Steckler}}）與{{link-en|詹姆斯·D·史特恩|James D. Stern}}將擔任該片的製片人<ref name="Variety"/><ref name="EW"/>。2017年3月，[[丹尼·葛洛佛|丹尼·葛洛佛]]、{{link-en|蒂卡·桑普特|Tika Sumpter}}、[[汤姆·威茨|湯姆·威茲]]、[[西西·史派克|西西·史派克]]、[[伊莉莎白·莫斯|伊莉莎白·莫斯]]與{{link-en|小以賽亞·惠特洛克|Isiah Whitlock, Jr.}}簽約加入劇組<ref name="Deadline.com"/><ref name="Deadline.com2"/>。4月，據報導，{{link-en|基斯·卡拉丁|Keith Carradine}}將出演該片<ref name="Deadline.com3"/>。《老人與槍》的[[主體攝影|主體攝影]]於4月3日在美國[[俄亥俄州|俄亥俄州]]開始<ref name="Filming"/>。

== 音樂 ==

2017年5月中旬，據報導，該片的配樂將由曾與導演[[大衛·羅利|大衛·羅利]]合作過《{{lang|en|St. Nick}}》（2009年）、《{{link-en|險路謎情|Ain't Them Bodies Saints}}》（2013年）、《[[尋龍傳說|尋龍傳說]]》（2016年）及《{{link-en|鬼魅浮生|A Ghost Story}}》（2017年）的{{link-en|丹尼爾·哈特|Daniel Hart (musician)}}創作<ref name="Score"/>。

== 上映 ==

2017年3月，[[福斯探照燈影業|福斯探照燈影業]]取得了該片的發行權<ref name="Fox Searchlight"/>。《老人與槍》將於2018年9月在[[2018年多倫多國際影展|多倫多國際影展]]上首映<ref name="TIFF2"/>。該片原定於2018年10月5日在北美院線上映<ref name="Release"/>，但後來提早至2018年9月28日<ref name="Release2"/>。該片的首支官方預告於2018年6月釋出<ref name="Trailer"/>。[[美國電影協會|美國電影協會]]（MPAA）以片中有髒話為由將電影評為[[美国电影分级制度#美國電影協會電影分級|PG-13級]]，意味著片中含有部分家長可能認為不適合13歲以下觀眾觀看的內容，建議家長需特別注意<ref name="RT"/>。

== 反響 ==

=== 評價 ===

《老人與槍》獲得了多數影評人的好評。[[爛蕃茄|爛蕃茄]]上收集的210篇專業影評文章中，有193篇給出了「新鮮」的正面評價，「新鮮度」為92%，平均得分7.5分（滿分10分）<ref name="RT"/>，而基於另一影評網站[[Metacritic|Metacritic]]上的47篇評論文章，其中45篇予以好評，2篇褒貶不一，平均分為80（滿分100），綜合結果為「普遍良好的評價」 <ref name="Metacritic"/>。
