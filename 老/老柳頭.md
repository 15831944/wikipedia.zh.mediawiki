'''老柳頭'''（{{lang-en|Old Man Willow}}），又譯'''柳樹老頭'''，是[[托爾金|托爾金]]奇幻小說裡[[中土大陸|中土大陸]]的樹妖，位於[[老林|老林]]之內<ref>{{cite book|title=''The Complete Guide to Middle-earth''|last=Foster|first=Robert|publisher=[[Random_House|Random House]]|year=1978|origyear=1971|isbn=0-345-27547-0}}</ref><ref>{{Cite web|url=http://www.glyphweb.com/arda/o/oldmanwillow.html|title=Old Man Willow|accessdate=2018-02-20|publisher=阿爾達百科全書|language=en-GB}}</ref>。它十分仇恨一切在森林中行走的事物，能夠[[催眠|催眠]]路過的旅人使其昏昏欲睡，也能使用自己樹枝或[[樹根|樹根]]來攻擊對方。在[[第三紀元|第三紀元]]末期，[[魔戒持有者|魔戒持有者]][[佛羅多|佛羅多]]和他的夥伴在途經老林時就遭到老柳頭的襲擊。

老柳頭在《[[魔戒|魔戒]]》首部《[[魔戒現身|魔戒現身]]》以及《[[湯姆·龐巴迪歷險記|湯姆·龐巴迪歷險記]]》均有出場。

== 概述 ==
在小說中老柳頭被形容得非常巨大可怕，位於老林內的[[柳树|柳樹]]林中央；它的催眠歌謠類似於[[搖籃曲|搖籃曲]]。[[湯姆·龐巴迪|湯姆·龐巴迪]]（Tom Bombadil）提及老柳頭是老林中最可怕的樹，它希望將附近一帶的森林都收納在自己的力量之下<ref name=":0" /><ref>{{Cite web|url=https://web.archive.org/web/20050929224139/http://www.victorianweb.org/courses/fiction/65/tolkien/isaacs4.html|title=The Great Willow in the Old Forest|accessdate=2018-07-14|date=2005-09-29}}</ref>。

[[魔戒聖戰|魔戒聖戰]]時期，持有[[至尊魔戒|魔戒]]的[[哈比人|哈比人]]佛羅多和他的夥伴為了躲避[[戒靈|戒靈]]的追殺而選擇穿越老林，在路經老柳頭旁時受到了它的催眠。昏昏欲睡的佛羅多差點被老柳頭的樹根壓進河中，[[梅里雅達克·烈酒鹿|梅里]]和[[皮聘|皮聘]]則被老柳頭吸進它樹上的裂縫內。這時湯姆·龐巴迪及時到達，唱著歌將梅里和皮聘釋放出來。<ref name=":0">{{Cite book|title=魔戒首部曲：魔戒現身|last=托爾金|publisher=聯經|year=2012年12月|isbn=978-957-08-4100-8|location=臺北}}</ref>

托爾金本人曾在信件中指出：由於老林樹木的記憶中帶有許多創傷，因此它們對在森林中行走生物都是充滿敵意的。而這種仇恨是老林自己本身形成的，並非受到[[索倫_(魔戒)|索倫]]的影響；托爾金就曾對於[[英國廣播公司|英國廣播公司]]的[[廣播劇|廣播劇]]裡把老柳頭描繪成[[魔多|魔多]]盟友的做法表示不滿<ref>{{Cite news|url=http://www.theonering.net/torwp/2008/11/03/30350-essay-tolkiens-trees/|title=Essay: Tolkien’s Trees|author=Claudia Riiff Finseth|date=2008-11-04|newspaper=TheOneRing.net|accessdate=2018-02-20|language=en-US}}</ref>。

== 靈感 ==
後來有人推測，托爾金在[[牛津|牛津]]居住期間，附近一帶柳樹林立的河谷可能為他創作老柳頭提供了靈感<ref>{{Cite news|url=http://www.bbc.com/news/uk-england-oxfordshire-20894824|title=Tolkien inspired by Oxfordshire|last=Serck|first=Linda|date=2013|newspaper=BBC News|accessdate=2018-01-28|language=en-GB}}</ref>。

== 改編描述 ==
雖然老柳頭並沒有出現在[[彼得·傑克森|彼得·傑克森]]的《[[魔戒電影三部曲|魔戒電影三部曲]]》中，但系列電影中的《[[魔戒二部曲：雙城奇謀|魔戒二部曲：雙城奇謀]]》裡有一個與原著裡的老柳頭非常類似的場景：梅里和皮聘在[[法貢森林|法貢森林]]中被一個[[胡恩_(中土大陸)|胡恩]]（Huorn，野樹人）吞吃掉，但老[[樹人|樹人]][[樹鬍|樹鬍]]隨後趕來救出他們<ref>{{Cite web|url=http://tolkiengateway.net/wiki/Old_Man_Willow|title=Old Man Willow|accessdate=2017-11-25|publisher=Tolkien Gateway|language=en}}</ref>，該段樹鬍的[[台詞|台詞]]完全來自原著中龐巴迪斥責老柳頭的話<ref>魔戒二部曲：雙城奇謀DVD-特別收錄</ref>。

在電子遊戲《[[指環王Online：安格瑪之影|指環王Online：安格瑪之影]]》中，老柳頭守護著[[百合花|百合花]]<ref>{{Cite web|url=https://web.archive.org/web/20090503235331/http://www.lotro.com/component/content/article/334-feature-article-a-heros-guide-to-the-old-forest?start=3|title=Feature Article: A Hero's Guide to the Old Forest|accessdate=2018-07-14|publisher=LOTRO}}</ref>。

== 有關影響 ==
[[J·K·羅琳|J·K·羅琳]]小說《[[哈利波特|哈利波特]]》裡的[[渾拚柳|渾拚柳]]與托爾金小說裡的老柳頭極為相似，在《哈利波特》裡，渾拚柳被種植於[[霍格華茲|霍格華茲]]的[[禁忌森林|禁忌森林]]外，會痛打接近它的人<ref>{{Cite web|url=http://greenbooks.theonering.net/guest/files/050102.html|title=How Much Was Rowling Inspired by Tolkien?|accessdate=2018-07-14|last=Corvar|first=Calisuri and|publisher=TheOneRing.net}}</ref>。
