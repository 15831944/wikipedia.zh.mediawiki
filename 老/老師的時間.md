《'''老師的時間'''》（{{lang|ja|せんせいのお時間}}）是日本漫畫家[[百瀨球美|百瀨球美]]創作的[[四格漫畫|四格漫畫]]作品。於[[竹書房|竹書房]]的《まんがライフMOMO》連載。

== 故事簡介 ==
== 登場人物 ==
=== 2年A班 ===
; {{nihongo|鈴木 美香|鈴木 みか（すずき みか）||聲：[[南央美|南央美]]}}
: 作中主人公。通稱「美香老師」。身高148cm。雙魚座。血型B型。27歲。興趣是讀書。
: [[娃娃臉|娃娃臉]]，身高只有148cm的幼兒體型（其後在漫畫第9卷中的身體檢查中，測量出147cm，矮了1cm），身穿童裝的話會被誤認是中小學生，但內在是27歲的大人（如體能）。
; {{nihongo|北川 理央|北川 理央（きたがわ りお）||聲：[[山崎和佳奈|山崎和佳奈]]}}
: 通稱「北川」。水瓶座。AB型。興趣是秘密。成績優秀。
: 根據本人的發言，喜歡「嬌小、可愛、有點奇妙」的東西。身材出眾，金髮的美少女。[[女同性戀|女同性戀]]，[[蘿莉控|蘿莉控]]，對男性沒興趣。
; {{nihongo|工藤 雄一|工藤 雄一（くどう ゆういち）||聲：[[上田祐司|上田祐司]]}}
: 通稱「工藤」。天秤座。AB型。美少年，表示自己是[[男同性戀|男同性戀]]。
: 成績優秀。在當初參加高中入學考試時對末武[[一見鍾情|一見鍾情]]，導致考試時無法集中精神，勉強合格。
: 喜歡[[日本文學|日本文學]]，特別喜歡[[三島由紀夫|三島由紀夫]]的《[[假面的告白|假面的告白]]》，另外也喜歡作家[[稻垣足穗|稻垣足穗]]及{{link-ja|福永武彦|福永武彦}}。
: 想像力豐富，經常把自己和末武的對話曲解，最後興奮過度，導致[[流鼻血|流鼻血]]之類的變態行徑。
; {{nihongo|小林 茜|小林 あかね（こばやし あかね）||聲：[[川上倫子|川上倫子]]}}
: 通稱「小林」。白羊座。O型。興趣是時尚流行。自稱「{{nihongo|女高中生之王|キングオブ女子高生}}」，沒有時尚觸覺。
; {{nihongo|末武 健太|末武 健太（すえたけ けんた||聲：[[山口勝平|山口勝平]]}}
: 通稱「末武」。獅子座。B型。興趣是運動。足球部成員。三兄弟的老三。
; {{nihongo|關 讓治|関 譲治（せき じょうじ）||聲：[[子安武人|子安武人]]（廣播劇）／[[谷山紀章|谷山紀章]]（動畫）}}
: 通稱「關」。身高175cm。重量60kg。射手座。O型。興趣是自己。
; {{nihongo|富永 美奈子|富永 美奈子（とみなが みなこ）|| 聲：[[榎本温子|榎本温子]]（廣播劇）／[[植田佳奈|植田佳奈]]（動畫）}}
: 通稱「富永」。處女座。 A型。興趣是料理。料理Club成員。漫畫第6卷後被小林及關稱為「{{nihongo|阿富|トミー}}」。
: 看起來是容姿端麗的千金小姐系女高中生，但喜怒哀樂的表情不多。性格對任何事也很嚴厲，毒舌家。
; {{nihongo|中村 元|中村 元（なかむら げん）||聲：岩田光央}}
: 通稱「{{nihongo|老爹|おやじ}}」。金牛座。 O型。興趣是[[賭博|賭博]]（特別是賭[[賽車|賽車]]）及讀書（特別是[[科幻小說|科幻小說]]）。現時對相機有興趣。
; {{nihongo|流 靜|流 静（ながれ しずか）||聲：[[大谷育江|大谷育江]]}}
: 通稱「{{nihongo|班長|委員長}}」。天蠍座。 O型。性格認真，成績優異。戴著眼鏡、束著麻花辮的優等生，實際上有著狂熱歌迷的一面，特別是偶像團體「Sunyanpu」的克羅的超級狂熱者。
; {{nihongo|渡部 匠|渡部 匠（わたべ たくみ）||聲：[[岡野浩介|岡野浩介]]}}
: 通稱「渡部」。
; {{nihongo|安東尼·M·張伯倫|アンソニー・M・チェンバレン|Anthony M.Chamberlain|聲：[[森久保祥太郎|森久保祥太郎]]}}
: 通稱「安東尼」。
; {{nihongo|遠藤 久里子|遠藤 くりこ（えんどうくりこ）}}
