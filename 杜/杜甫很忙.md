'''杜甫很忙'''是2012年3月发生在[[新浪微博|新浪微博]]上的一个涂鸦、恶搞语文课本插图《杜甫像》的网络活动、话题，由中国大陆的[[高中|高中]]学生發起。“杜甫很忙”成为3月末中国互联网的热词之一，并被商业广告利用。某公关团队和[[新浪微博|新浪微博]]先后声称此次网络行销为其策划。

原图由[[蒋兆和|蒋兆和]]创作于1959年，在人教版《语文》(必修三)课本中作为[[杜甫|杜甫]]作品《登高》的配图。《杜甫像》1959年画作（131cm×90cm）现存于[[中国历史博物馆|中国历史博物馆]]，1962年画作（109X67cm）现存于成都[[杜甫草堂博物馆|杜甫草堂博物馆]]。

== 事件发展 ==
*3月21日，有网友在新浪微博发布了几张恶搞课本中杜甫图像的涂鸦图，引起上百网友回复。3月22日，新浪微话题组决定将其打造成一个微博话题，新浪微话题组配合“微博新鲜事”帐号、list推广、热门话题榜单等多个途径推广“杜甫很忙”话题，带动网友参与并转发，最终形成“轰动”效应。<ref name="认领">{{Citeweb|url=http://www.techweb.com.cn/internet/2012-03-31/1173885.shtml|title=“杜甫很忙”系人为策划 遭公关团队与新浪微博“认领”|date=2012-03-31|accessdate=2012-03-31|publisher=TechWeb.com.cn}}</ref>
*薛永昱声称“杜甫很忙”是其团体为某博物馆所作策划，网友纷纷将矛头指向成都[[杜甫草堂博物馆|杜甫草堂博物馆]]。杜甫草堂博物馆副馆长方伟回应道“博物馆需要[[营销|营销]]，但绝不是没有底线的营销”否认与之相关。<ref name="cdrb">{{Citeweb|url=http://www.cdrb.com.cn/html/2012-03/30/content_1543623.htm|title=网上杜甫很忙 网下草堂很冤|date=2012-03-30|accessdate=2012-03-31|publisher=成都日报|deadurl=yes|archiveurl=https://web.archive.org/web/20150514084513/http://www.cdrb.com.cn/html/2012-03/30/content_1543623.htm|archivedate=2015-05-14}}</ref>
*3月30日，[[成都商报|成都商报]]文化新闻部与成都杜甫草堂博物馆联合策划发起“万元征集杜甫创意画像”活动。<ref>{{Citeweb|url=http://e.chengdu.cn/html/2012-03/30/content_307937.htm|title=杜甫很忙 草堂喊你去画他|date=2012-03-30|accessdate=2012-03-31|publisher=成都商报}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==被创造出的形象==
此次事件中，杜甫被[[涂鸦|涂鸦]]为多种形象：

*端着[[狙击枪|狙击枪]]凝视远方
*开着[[坦克|坦克]]向前冲
*戴着[[墨镜|墨镜]]骑[[电动车|电动车]]出门
*骑着[[自行车|自行车]]去买菜
*吃上洋[[快餐|快餐]]
*跟美女翩翩起舞
*穿着运动服打[[篮球|篮球]]
*坐在桌前切[[西瓜|西瓜]]
*玩转千元[[安卓|安卓]]机
*使用[[笔记本电脑|笔记本电脑]]上网

== 策划者争议 ==
“杜甫很忙”成为微博热门话题之后，[[薛永昱|薛永昱]]声称该话题为其团队“帮一个博物馆做的策划”<ref name="认领"/>，有3张（一说为大部分<ref>{{Citeweb|url=http://www.bjd.com.cn/10jsxw/201203/31/t20120331_1631896.html|title=营销人士薛永昱称“杜甫很忙”是其策划|date=2012-03-31|accessdate=2012-03-31|publisher=中华工商时报|deadurl=yes|archiveurl=https://web.archive.org/web/20120421022335/http://www.bjd.com.cn/10jsxw/201203/31/t20120331_1631896.html|archivedate=2012-04-21}}</ref>）为其团队所画，其他涂鸦图为网友自发跟风<ref>{{Citeweb|url=http://www.chinanews.com/cul/2012/03-31/3788056.shtml|title=“杜甫很忙”背后：从传图到风靡网络仅用31小时|date=2012-03-31|accessdate=2012-03-31|publisher=华西都市报}}</ref>。随后，新浪微博运营经理李凯在问答网站[[知乎|知乎]]上发表文章阐述整个[[营销|营销]]过程，表明为新浪微话题组策划，而不是邀功的[[薛永昱|薛永昱]]<ref>{{cite web |url = http://www.zhihu.com/question/20141822 |title = 为什么「杜甫很忙」会在微博上掀起一阵流行风呢？这现象背后说明了什么？ |author = 李凯 |date = 2012-03-30 |website = 知乎 |accessdate = 2013-11-09}}</ref> 。
