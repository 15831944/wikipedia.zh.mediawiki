'''杜元穎'''{{BD|769年||833年||catIdx=D}}<ref name=BT17-2>《[[舊唐書|舊唐書]]》[[s:舊唐書/卷17下|卷17下]]</ref><ref>《[[旧唐书|旧唐书]]·文宗本纪》称杜元穎死在[[大和_(唐朝)|大和]]六年十二月丁未日，但是当年十二月没有丁未日。大和六年十二月主要是在833年1月。[[中央研究院|中央研究院]][http://sinocal.sinica.edu.tw 兩千年中西曆轉換]</ref>，封爵'''建安男'''，京兆杜陵（今[[陕西省|陕西省]][[西安市|西安市]]）人。中国[[唐朝|唐朝]]大臣，[[唐穆宗|唐穆宗]]时[[翰林学士|翰林学士]]、[[户部侍郎|户部侍郎]][[同中书门下平章事|同中书门下平章事]]。他的错误政策被指责使唐朝中央政府再次失去[[河朔|河朔]]。而后来担任[[西川节度使|西川节度使]]（总部设在今[[四川省|四川省]][[成都市|成都市]]），引发[[南诏|南诏]]入侵。

== 背景 ==
杜元穎生于[[唐代宗|唐代宗]][[大历|大历]]四年（769年）<ref name=BT163>《舊唐書》[[s:舊唐書/卷163|卷163]]</ref>。他是[[唐朝|唐朝]]初年名相[[杜如晦|杜如晦]]的后代，《舊唐書》说他是杜如晦的裔孙<ref name=BT163/>《[[新唐書|新唐書]]·杜如晦传》说他是杜如晦的五世孙<ref name=NBT96>《[[新唐書|新唐書]]》[[s:新唐書/卷096|卷96]].</ref>《新唐書·宰相世系表》说他是杜如晦叔叔[[杜淹|杜淹]]的六世孙，<ref name=NBT72>《新唐書》[http://www.guoxue123.com/shibu/0101/00xtsf/table/f46.htm 卷72]</ref>《[[資治通鑑|資治通鑑]]》称杜元颖是杜淹的第六代孙。<ref name=ZZTJ241>《資治通鑑》[[s:資治通鑑/卷241|卷241]]</ref>杜元穎五世祖杜敬同，担任[[中書舍人|中書舍人]]、東陽公。高祖父杜從則，为[[工部侍郎|工部侍郎]]。曾祖父杜自遠、祖父杜繁没有仕官记录，父亲杜佐担任大理正。杜元穎行十四，他的弟弟杜元絳出任[[太子宾客|太子宾客]]<ref name=NBT72/>。[[唐德宗|唐德宗]][[贞元_(唐朝)|贞元]]十六年（800年）杜元穎登进士第。[[唐宪宗|唐宪宗]][[元和_(唐朝)|元和]]元年（806年）登博学宏辞科。元和十一年（816年）复登茂才异等科。两次受辟[[节度使|节度使]]幕府。<ref name=BT163/>

== 唐宪宗时代 ==
[[唐宪宗|唐宪宗]]元和年间，杜元穎出任[[門下省|門下省]][[左拾遺|左拾遺]]，转任太常博士，充翰林学士，元和十二年（817年），改[[中書省|中書省]][[右补阙|右补阙]]，仍充翰林學士。杜元穎写文章文辞敏捷，受到宪宗的赞赏。<ref name=BT163/>平定[[吴元济|吴元济]]，<ref>《資治通鑑》[[s:資治通鑑/卷240|卷240]]</ref>起草诏令很多，朝廷赐给杜元穎绯鱼袋，<ref name=BT163/>四品、五品官员使用绯色官服及鱼袋，<ref>《新唐書》[[s:新唐書/卷024|卷24]]</ref>右补阙只是七品<ref>《新唐書》[[s:新唐書/卷047|卷47]]</ref>。元和十四年（819年）三月，加[[吏部|吏部]][[司勳|司勳]][[員外郎|員外郎]]，[[知制诰|知制诰]]<ref name=BT163/>。

== 唐穆宗时代 ==
元和十五年（820年）唐宪宗驾崩，其子[[唐穆宗|唐穆宗]]即位<ref name=ZZTJ241/>。召见杜元穎到思政殿，赐金紫，提拔他为中書舍人。穆宗让杜元穎担任戶部侍郎、学士承旨。<ref name=BT163/>[[长庆|长庆]]元年（821年），杜元穎以本官同中書門下平章事。<ref name=ZZTJ241/>穆宗封他为[[上柱国|上柱国]]、'''建安县男'''。从右补阙到侍郎，不到一年升任宰相，文学之臣得到升迁的速度，没有比得过杜元颖的。
<ref name=BT163/>

长庆元年（821年），[[盧龍节度使|盧龍节度使]]（治今北京市）[[刘总|刘总]]提出辞职，取消割据，归附中央。刘总提出将刘总一分为三，[[幽州|幽州]]、[[涿州|涿州]]、[[营州|营州]]为一道，请求任命[[张弘靖|张弘靖]]为节度使；[[平州|平州]]、[[蓟州|蓟州]]、[[妫州|妫州]]、[[檀州_(隋朝)|檀州]]为一道，请求任命[[平卢节度使|平卢节度使]][[薛平|薛平]]为节度使；[[瀛州_(北魏)|瀛州]]、[[莫州|莫州]]为一道，请求任命代理[[京兆尹|京兆尹]][[盧士玫|盧士玫]]为[[观察使|观察使]]。同时他还派了一个数量的人员,把部下骄纵强横难以管辖的将士，如都知兵马使[[朱克融|朱克融]]等人挑出来，送到京城，请朝廷奖励并予以提拔，让幽州人都能产生羡慕朝廷官禄的意向。唐穆宗没有完全实现刘总的计划，以卢士玫为瀛莫观察使。但剩下的州都由张弘靖管理。宰相[[崔植|崔植]]、杜元颖缺乏深谋远虑，不理解刘总的计划。朱克融等人在京城客居很久，窘迫到借衣讨食的地步，每天到中书省去请求授予官职，崔植、杜元颖不理。张弘靖为幽州节度使，勒令朱克融等人归回幽州，受张弘靖指使效力。朱克融等人十分怨怒。<ref name=ZZTJ241/>821年秋天，卢龙士兵哗变，废黜了把张弘靖，拥立朱克融为节度使。最后不仅卢龙，[[成德节度使|成德节度使]]（治今[[河北省|河北省]][[石家庄市|石家庄市]]正定县）、[[魏博节度使|魏博节度使]]（治今河北省[[邯郸市|邯郸市]][[大名县|大名县]]）也恢复了藩镇割据。朝廷再度丢失河朔地区，直到唐朝最终灭亡，一直未能收复。传统史家归咎于宰辅大臣杜元颖、崔植、[[王播|王播]]的无能。<ref name=ZZTJ242>《資治通鑑》[[s:資治通鑑/卷242|卷242]]</ref>

长庆二年（822年），[[宣武军|宣武军]]（治今[[河南省|河南省]][[开封市|开封市]]）发生军变，李㝏驱逐了节度使李愿。杜元穎和户部侍郎、判度支張平叔想要仿照河北的惯例，任命李㝏为节度使来安抚他。[[李逢吉|李逢吉]]主张打击李㝏：“河北藩镇割据跋扈，朝廷予以承认，是不得已的事。现在，如果连宣武也姑息，恐怕江淮以南的广大地区都要脱离朝廷。”杜元颖、张平叔和他争论说：“为什么要可惜几尺长的节度使符节，而不爱惜宣武一方百姓的生命呢？”唐穆宗接受了李逢吉的建议，李㝏很快被下属李質杀死，宣武军投降朝廷。<ref name=ZZTJ242/>

长庆三年（823年）[[九月初八|九月初八]]，唐穆宗任命[[中书侍郎|中书侍郎]]、同平章事杜元颖兼同平章事的荣誉职务，充任[[剑南西川节度使|剑南西川节度使]]（治今[[四川省|四川省]][[成都市|成都市]]），穆宗亲自到安福门为他饯行。<ref name=ZZTJ243>《資治通鑑》[[s:資治通鑑/卷243|卷243]]</ref>

== 唐敬宗时代 ==
长庆四年（824年），唐穆宗驾崩，其子[[唐敬宗|唐敬宗]]即位<ref name=ZZTJ243/>。唐敬宗年轻喜欢玩，追求奢侈，杜元穎为了取悦皇帝，经常向皇帝进贡罕见、有价值的珍宝。杜元穎因此加紧盘剥西川，这引起当地军民的不满。<ref name=BT163/>杜元颖不懂军事，却专门积蓄财产，减削士卒的衣食供给。<ref name=ZZTJ244>《資治通鑑》[[s:資治通鑑/卷244|卷244]]</ref>同时，李逢吉让知枢密[[王守澄|王守澄]]对敬宗说：“陛下所以能被立为皇太子，主要是李逢吉的功劳。像杜元颖、李绅这些人，都是要立深王李察（[[李悰|李悰]]）的。”唐敬宗后来发现[[裴度|裴度]]、杜元颖、[[李绅|李绅]]上疏请立自己为皇太子的上奏，才重新信任杜元颖等人。<ref name=ZZTJ243/>

== 唐文宗时代 ==
西南戍边的士卒衣食不足，纷纷到[[南诏|南诏]]境内去掠夺偷盗。南诏国赠送他们衣物和粮食，得以知晓西川的动静虚实。南诏国蒙嵯顛执掌朝政，密谋大举侵犯西川，西南的边防州郡多次向杜元颖报告，杜元颖不信。[[唐文宗|唐文宗]]大和三年（829年），嵯顛率兵来攻，边城毫无防备。南诏以西川的降卒为向导，袭击并攻陷了[[雟州|雟州]]（今[[四川省|四川省]][[凉山彝族自治州|凉山彝族自治州]][[西昌市|西昌市]]）、[[戎州|戎州]]（今[[四川省|四川省]][[宜宾市|宜宾市]]）。十一月二十八日，杜元颖派兵和南诏军队在[[邛州|邛州]]以南交战，西川兵大败。南诏乘胜攻占邛州。南诏径直抵达[[成都|成都]]城下，十二月初四，攻陷成都外城。杜元颖率领将士退守牙城，抵抗南诏军队。杜元颖几次想离城逃亡。南诏军前锋部队远抵东川节度使驻地[[梓州|梓州]]（今[[四川省|四川省]][[绵阳市|绵阳市]][[三台县|三台县]]）的西城。南诏军队驻留成都西城十天。退走时，大肆掠夺妇女和各种工匠几万人，以及各种珍宝奇货。南诏国工匠的技术水平可以和西川媲美。[[十二月初六|十二月初六]]，唐文宗贬杜元颖为[[邵州|邵州]]（今[[湖南省|湖南省]][[邵阳市|邵阳市]]）[[刺史|刺史]]。嵯颠派遣使者上表唐朝，说这次军事行动只是由于杜元颖不爱护士卒，士卒痛恨他，才争相做我的向导，请求我出兵诛杀杜元颖。要求唐朝处死杜元颖。十二月二十一日，唐文宗再次贬杜元颖为[[循州|循州]]（今[[广东省|广东省]][[惠州市|惠州市]]）[[司马|司马]]。新任西川节度使[[郭釗|郭釗]]抵达成都后，和南诏国签订友好条约，规定两国互不侵扰。<ref name=ZZTJ244/>杜元颖的属下判官崔璜贬为[[连州|连州]]司马，纥干臮贬为[[郢州_(西魏)|郢州]]长史，卢并贬为[[唐州_(唐朝)|唐州]]司马。大和六年十二月丁未日，杜元颖卒于循州贬所。杜元颖临死前，他上书乞求死后赠官，于是文宗追赠他为[[湖州|湖州]]刺史。杜元颖和[[李德裕|李德裕]]关系很好，[[唐武宗|唐武宗]][[会昌_(唐朝)|会昌]]年间，李德裕执政，恢复杜元颖的官爵。杜元颖的侄子、杜元絳的儿子[[杜审权|杜审权]]和杜审权的儿子[[杜让能|杜让能]]、杜让能的儿子[[杜晓|杜晓]]后来担任宰相。<ref name=NBT96/>《新唐书·艺文志》记载杜元颖五题一卷，《[[全唐诗|全唐诗]]》存诗一首。
