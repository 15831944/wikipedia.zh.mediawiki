'''杜湖'''是一个位于[[中华人民共和国|中国]][[浙江省|浙江省]][[慈溪市|慈溪市]]的湖泊，面积约为3700多亩，是慈溪市最大的湖泊。杜湖一部分改建为里杜湖水库，外杜湖单独成湖<ref name='《上林湖越窑》'>{{cite book |author=慈溪市博物馆 |authorlink= |coauthors= |title=《上林湖越窑》 |year=2002年10月 |publisher=科学出版社 |location= |isbn= }}</ref>。其中里杜湖是慈溪市最大的饮用水源水库。

== 地理位置 ==
{{Wide image|杜湖全景.jpg|1080px|<center>杜湖全景，摄于[[五磊山|五磊山]]山顶</center>}}
杜湖位于慈溪观海卫镇东南部，东、西、南三面环山（属慈溪南部丘陵）,北面与[[白洋湖_(慈溪)|白洋湖]]相邻。西北面是拥有上千年历史的—[[宓家埭|宓家埭]]古聚落和[[鸣鹤古镇|鸣鹤古镇]]。

== 里杜湖 ==
里杜湖水库总库容2370万立方米,有效库容1700万立方米<ref>[http://www.cnki.com.cn/Article/CJFDTotal-JJSL197902003.htm]</ref>。
其他，可详见[[里杜湖水库|里杜湖水库]]条目。
== 杜湖大坝 ==
=== 沿革 ===
* 1969年底，杜湖大坝动工兴建，由当时慈溪县政府组织建造。
* 1973年，大坝开始发挥效益。
* 1975年，工程全面竣工。
=== 大坝资料 ===
坝高17.5米，平均坝长440米，坝底宽80米,坝顶宽4米<ref>[http://www.cnki.com.cn/Article/CJFDTotal-JJSL197902003.htm]</ref>。坝南是里湖（水库），坝北是外湖，这种设计在中国国内是罕见的<ref>[http://202.107.212.146:838/nbwshj/index.php?option=com_content&view=article&id=504:2011-09-14-02-02-14&catid=50:m&Itemid=69]</ref>。

== 西埠头大桥 ==
位于里杜湖水库南侧，是连接西埠头于湖滨东路的重要通道。桥长60米。距离里杜湖南岸居民区600米。
