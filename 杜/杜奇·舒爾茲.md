'''杜奇·舒爾茲'''（'''Dutch Schultz'''，本名'''Arthur Simon Flegenheimer'''，{{bd|1901年|8月6日|1935年|10月24日}}）是一名1920年代和1930年代在[[紐約市|紐約市]]的[[美國猶太人|猶太裔美國]]黑幫，他在[[黑社會|有組織犯罪]]相關的活動中發財，包括非法私酒和彩票詐騙。舒爾茲曾被檢察官[[托马斯·杜威|托马斯·杜威]]領導的兩次逃稅審判削弱了勢力，他的非法勾當也受到黑幫[[查理·盧西安諾|幸運·盧西安諾]]所威脅。為了避免定罪，舒爾茲要求[[黑手黨委員會|黑手黨委員會]]批准殺死杜威，但被拒絕。當舒爾茲不服從委員會並企圖殺害杜威時，委員會便於1935年下令殺死他。<ref>{{Cite web |url = https://www.haaretz.com/jewish/jewish-mobster-gunned-down-in-newark-toilet-1.5447844 |title = This Day in Jewish History // 1935: A Jewish Mobster Is Gunned Down in a Newark Toilet |author = Goldy Levy |date = 2016-10-23 |publisher = haaretz.com |language = en}}</ref>
