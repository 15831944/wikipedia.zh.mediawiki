'''杜宪'''（{{bd|1954年|9月23日|||catIdx=D杜}}）女，祖籍[[湖北省|湖北省]][[秭归县|秭归县]]，生于[[北京市|北京市]]，[[中华人民共和国|中华人民共和国]]媒体人。

==生平==
杜宪于1954年生于北京<ref name="tsinghua"/>。16岁初中毕业，进入位于[[北京|北京]][[门头沟|门头沟]]、实施准军事化管理的北京人民轴承厂，在后勤连的大食堂工作7年<ref name="sina"/>。[[文化大革命|文化大革命]]结束后，1977年恢复[[高考|高考]]，杜宪报名参加了[[北京广播学院|北京广播学院]]（现[[中国传媒大学|中国传媒大学]]）门头沟招生点的初试。与此同时，由于当时[[中国中央电视台|中央电视台]]新闻播音人员青黄不接，[[李娟_(电视主播)|李娟]]、[[吕大渝|吕大渝]]等资深从业人员也亲自到北京广播学院门头沟招生点试图直接挑选播音员，选中了杜宪。但最后杜宪被北京广播学院录取，没有进入中央电视台<ref name="sina"/><ref name="wenzhai"/>。杜宪、[[薛飞_(电视主播)|薛飞]]等人是北京广播学院77级播音班的同学<ref name=duxue/>。

1982年，杜宪、薛飞毕业后被分配到中央电视台新闻部播音组，起初承担《[[新闻联播|新闻联播]]》20分钟的国内新闻录制任务。后来，随着[[赵忠祥|赵忠祥]]、[[刘佳_(主持人)|刘佳]]、[[吕大渝|吕大渝]]等老一代播音员退出一线，薛飞、杜宪开始成为《新闻联播》的主要播音员，同时也成了1980年代的名人<ref name="sina"/><ref name=duxue/>。1988年，杜宪成为[[中国人民政治协商会议第七届全国委员会委员名单|第七届全国政协委员]]<ref name="sina">{{cite web|title=《新闻联播》主持人杜宪|url=http://eladies.sina.com.cn/news/2009/0608/1524875272.shtml|publisher=新浪网|work=|date=2009-06-08|accessdate=2014-05-31|language=zh-cn}}</ref><ref name="政协">{{cite web|title=中国人民政治协商会议第七届全国委员会委员名单（2081人）|url=http://www.cppcc.gov.cn/2011/09/29/ARTI1317288700388280.shtml|publisher=中国政协网|work=|date=|accessdate=2014-06-01|language=zh-cn}}</ref>。1989年6月4日晚间，《新闻联播》由杜宪与[[张宏民|张宏民]]主持，播报了[[六四事件|北京戒严]]等消息，杜宪当天身着黑衣、语速缓慢<ref>{{cite book |author=国亚 |year=2005 |title=《一个普通中国人的家族史》第八章 躁动关头 1988～1991（二）|publisher = 中国广播电视出版社 |url = http://mjlsh.usc.cuhk.edu.hk/book.aspx?cid=2&tid=114 |isbn = 9787504344311 |accessdate = 2014-06-02 |language = zh-cn}}</ref><ref>{{cite web |title = 六四遭封殺央視主播薛飛出書追憶往事 |url = http://china.dwnews.com/big5/news/2014-02-09/59404570.html |publisher = 多維新聞 |date = 2014-02-09 |accessdate = 2014-06-02 |language = zh-hant |archive-url = https://web.archive.org/web/20140605051105/http://china.dwnews.com/big5/news/2014-02-09/59404570.html |archive-date = 2014-06-05 |dead-url = yes }}</ref>。

由于同情学生，杜宪、薛飞在1989年6月被调离中央电视台新闻部播音组，杜宪转入中央电视台经济部担任编辑，化名“皓月”<ref name=duxue>{{Cite web |url = http://news.ifeng.com/a/20140918/42022163_0.shtml |title = 杜宪和薛飞离开央视 两人曾是《新闻联播》主播 |author =  |date =2014-09-18  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。此后经济部每次评选先进，杜宪始终榜上有名<ref name="sina" />。1992年，杜宪应国际新闻广播电视交流中心（即“[[梅地亚中心|梅地亚中心]]”）之邀，出任电视片《中国小城镇》的主持人，跟随摄制组游历了[[云南|云南]]、[[贵州|贵州]]、[[四川|四川]]、[[西藏|西藏]]等地<ref name="wenzhai">{{cite web |author = 碧流春 |title =杜宪 幸福都是拼来的 |url = http://www.chinanews.com/gj/kong/news/2008/06-10/1277248.shtml |publisher = 中国新闻网|work=中国妇女报|date = 2008-06-10 |accessdate = 2014-05-30 |language = zh-cn}}</ref>。

1992年6月，杜宪辞职离开中央电视台。随后杜宪与“梅地亚中心”签订了5年的工作合约。1992年9月至同年底，杜宪应邀到[[美国|美国]][[佛罗里达大学|佛罗里达大学]]做访问学者，丈夫[[陈道明|陈道明]]随行<ref name="wenzhai"/><ref name="sina"/><ref name=duxue/>。1992年底，电视剧《[[北京人在纽约|北京人在纽约]]》的导演[[郑晓龙|郑晓龙]]、[[冯小刚|冯小刚]]邀请杜宪出演女主角“郭燕”，当时陈道明已被定为饰演“大卫”，和[[姜文|姜文]]等人已经进组，但不久杜宪、陈道明一道解约离开了剧组<ref name=duxue/>。

杜宪回国后，继续履行与“梅地亚中心”的工作合同。1993年夏，杜宪正式辞去“梅地亚中心”的职位，出任今日新闻广告有限公司艺术总监、[[常州|常州]]先奇影视制作中心董事长。同年，其著作《我在美国的106天》由长春出版社出版。1998年，杜宪又创办一家科技公司，任董事长<ref name="sina"/>。1997年开始，香港[[凤凰卫视|凤凰卫视]]邀请杜宪加盟。2000年1月，她受聘出任凤凰卫视节目主持人、策划，主持《只有一个地球》栏目，并先后任专题片《穿越风沙线》、《寻找远去的家园》、《永远的三峡》的监制和主持人<ref name="wenzhai"/><ref name=duxue/>。2002年，杜宪离开凤凰卫视<ref name=duxue/>。

后来，杜宪到母校[[中国传媒大学|中国传媒大学]]（前身为北京广播学院）播音主持艺术学院担任副教授<ref name="sohu"/><ref name="sina"/><ref name=duxue/>。2007年11月2日，杜宪与[[清华大学|清华大学]]签订协议，以父亲[[杜庆华|杜庆华]]所获“[[何梁何利奖|何梁何利奖]]”为基础，筹款55万[[人民币|人民币]]设立“杜庆华奖学金”，主要用于奖励[[清华大学航天航空学院|清华大学航天航空学院]]贫寒而优秀的学生。<ref name="tsinghua">{{cite web|title=“杜庆华奖学金”在清华大学设立|url=http://news.sciencenet.cn/htmlnews/200711517171659193389.html|publisher=科学网|work=清华大学|date=2007-11-05|accessdate=2014-05-31|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20150318082400/http://news.sciencenet.cn/htmlnews/200711517171659193389.html|archivedate=2015-03-18}}</ref>

==家庭==
*父：[[杜庆华|杜庆华]]，力学专家，[[清华大学|清华大学]]教授，[[中国工程院院士|中国工程院院士]]<ref name="tsinghua"/>。
*丈夫：[[陈道明|陈道明]]，[[演员|演员]]。1978年两人经介绍相识，1982年结婚，1985年生下女儿陈格，后者13岁时被送往[[英国|英国]]读书<ref name="sina"/><ref name="sohu">{{cite web|title=陈道明妻子杜宪近照曝光 现为中传媒副教授|url=http://news.sohu.com/20131105/n389616970.shtml|publisher=搜狐新闻|work=[[人民网|人民网]]|date=2013-11-05|accessdate=2014-05-30|language=zh-cn}}</ref>。
