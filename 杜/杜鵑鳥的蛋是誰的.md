《'''杜鵑鳥的蛋是誰的'''》，-{zh:簡體中文版譯作《'''布谷鳥的蛋是誰的'''》;zh-hans:繁体中文版书名为《'''杜鵑鳥的蛋是誰的'''》;zh-hant:簡體中文版譯作《'''布谷鳥的蛋是誰的'''》}-，是[[日本|日本]]作家[[東野圭吾|東野圭吾]]的[[長篇小說|長篇]][[推理小說|推理小說]]。2010年由[[光文社|光文社]]發行了單行本，2013年光文社發行文庫本。

2016年春季由[[WOWOW|WOWOW]]電視台撥出的改編為真人版連續劇。[[土屋太鳳|土屋太鳳]]主演<ref name="cinemacafe"/>。

== 登場人物 == 
; {{nihongo|緋田 宏昌|{{ruby|緋田|ひだ}} {{ruby|宏昌|ひろよし}}|Hida Hiroyoshi}}
: 前[[高山滑雪|高山滑雪]][[奧運|奧運]]選手，但未曾在國際性比賽留下好的名次。

; {{nihongo|緋田 風美|{{ruby|緋田|ひだ}} {{ruby|風美|かざみ}}|Hida Kazami}}
: 緋田宏昌的女兒，[[高山滑雪|高山滑雪]]選手。是高山滑雪界的明日之星，目前隸屬新世開發[[高山滑雪|高山滑雪]]部的成員。

; {{nihongo|緋田 智代|{{ruby|緋田|ひだ}} {{ruby|智代|ともよ}}|Hida Tomoyo}}
: 緋田宏昌的妻子。在風美小時候自殺身亡。

; {{nihongo|鳥越 伸吾|{{ruby|鳥越|とりごえ}} {{ruby|伸吾|しんご}}|Torigoe Shingo}}
: 被柚木發掘其才能，開始[[越野滑雪|越野滑雪]]，加入新世開發少年滑雪俱樂部。

; {{nihongo|鳥越 克哉|{{ruby|鳥越|とりごえ}} {{ruby|克哉|かつや}}|Torigoe Katsuya}}
: 鳥越伸吾的父親，前登山家。比起家人更愛高山。

; {{nihongo|高倉|{{ruby|高倉|たかくら}}|Takakura}}
: 新世開發高山滑雪部的教練，指導緋田風美。緋田宏昌選手時代的教練兼密友。

; {{nihongo|貝塚|{{ruby|貝塚|かいづか}}|Kaiduka}}
: 新世開發越野滑雪部的教練，指導鳥越伸吾，非常嚴格按表操課。

; {{nihongo|柚木 洋輔|{{ruby|柚木|ゆずき}} {{ruby|洋輔|ようすけ}}|Yuzuki Yousuke}}
: 新世開發運動科學研究所副所長，相信可經由基因的排序關聯性，尋找出各類型的頂尖運動好手。

; {{nihongo|小谷|{{ruby|小谷|こたに}}|Kotani}}
: 新世開發體育部的部長。對於柚木來說，小谷就是他的上司。

; {{nihongo|藤井|{{ruby|藤井|ふじい}}|Huzii}}
: 與伸吾在同場集訓的當地高中滑雪隊員。與伸吾同為高一學生，對人友善且熱愛滑雪，只是滑雪成績不佳。

; {{nihongo|黑澤|{{ruby|黒沢|くろさわ}}|Kurosawa}}
: 與伸吾在同場集訓的當地高中滑雪隊員。是藤井的學長，在越野滑雪項目的技術高超，是參加全國性大賽的常客，喜歡在訓練時與伸吾互相較量。

; {{nihongo|上條 伸行|{{ruby|上条|かみじょう}} {{ruby|伸行|のぶゆき}}|Kamijou Nobuyuki}}
: 「KM建設」的社長。因特殊的原因接近緋田父女。

; {{nihongo|上條 世津子|{{ruby|上条|かみじょう}} {{ruby|世津子|せつこ}}|Kamijou Setsuko}}
: 上條伸行的妻子。

; {{nihongo|上條 文也|{{ruby|上条|かみじょう}} {{ruby|文也|ふみや}}|Kamijou Fumiya}}
: 上條伸行的兒子。「KM建設」常務。患有某種疾病。

; {{nihongo|木原|{{ruby|木原|きはら}}|Kihara}}
: 警視廳刑事部搜查一課警部。

; {{nihongo|西島|{{ruby|西島|にしじま}}|Nishizima}}
: 警視廳刑事部搜查一課巡查部長。
