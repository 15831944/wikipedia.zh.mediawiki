'''兔頭哥'''（本名：粘世昕，網路代號：Tim885885，別名：兔頭），台灣[[實況主|實況主]]、戶外實況主、獨立遊戲創作者、遊戲語言翻譯家（英文→中文）、正體潤色者（簡體中文→正體中文）、實況資料庫站長。

因為玩《[[俠盜獵車手V|俠盜獵車手5]]》技術高強聞名，曾在大陸[[斗鱼TV|鬥魚TV]]直播台灣戶外風情爆紅，第一次直播即十萬人觀看。[[LIVEhouse.in|LIVEhouse.in]]上直播台北市長[[柯文哲|柯文哲]]騎單車[[一日雙塔|一日雙塔]]，在台灣[[Facebook|Facebook]]上首位[[現場直播|直播]]走訪各大鬼屋並於臉書上直播有3萬人同時收看，最終是在[[Twitch|Twitch]]直播《[[精靈寶可夢GO|精靈寶可夢GO]]》違規上[[新聞|新聞]]才開始爆紅。隨後加入[[周杰倫|周杰倫]]旗下知名電競團體[[J_Team|Jteam]]，是整個團體唯一生活類型實況主，但因為於[[YouTube|Youtube]]上播放侵權音樂電台而遭到解約。

== 經歷 ==
原始網路代號為小4昕（英文代號：God885885 或 Tim885885），於2002年前後在[[台灣論壇|台灣論壇]]上出現，隨後出現於[[遊戲基地|遊戲基地]]與[[巴哈姆特|巴哈姆特]]。主要出現板塊為《俠盜獵車手》板。

2003年，常做一些詳細教學文心得文。還有一系列的GTA遊戲精彩合集等等，很喜歡從事將英文的軟體或是遊戲[[模組_(遊戲)|模組]]、文章等等翻譯成中文。
*
2004年開始做各種有趣遊戲[[影片_(消歧義)|影片]]合集。當時沒有網路[[影音網站|影音網站]]，所以影片都是上傳免費空間供人下載觀看。

2005年登上[[YouTube|Youtube]]製作各種編輯過以[[俠盜獵車手系列|俠盜獵車手]]為主的遊戲，是早期台灣費心做遊戲影片的Youtuber。2006年因Youtub帳號忘了，改了新的帳號，即是現在所用的「兔頭」頻道。

2009年開始於[[Justin.tv|Justin.TV]]上直播各種日常生活，帶電腦到家門外做戶外直播。應是台灣最早專精作生活類型的直播主。

2011年上了大學之後則開始於Youtube做了一系列的網路直播教學影片，和其他各種教學文章幫助很多人，還有一些「真的是這樣嗎？實測後告訴你」之類的文章解開網路迷思。同年在 Justin.tv 連續一個月生活直播，沒告知室友，更衣裸體都直播出去。

2014年 加入[[實況主|實況主]]、知名[[YouTube|Youtuber]]老皮創立的「太陽工作室」。

== 架站經歷 ==
於國中時曾架設大型論壇 台灣GoGoBox聯盟 論壇，當時據稱開站兩個月日瀏覽量已經超過100萬人次但當時多個網站相繼因為盜版被抄，[[台灣論壇|台灣論壇]]、[[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]]。故以低價轉手他人。
* 2007年 架設 Xbox360成就網，是一個[[網誌|網誌]]類型的[[網站|網站]]。裡面有Xbox 360裡特有的成就系統的翻譯，當時有很多遊戲只有英文或日文，[[翻譯|翻譯]]成就目的讓玩家知道怎麼解鎖成就。
* 2009年 架設 小四仔的家，裡面盡是發一些廢文至今已廢站。
* 2012年 架設 兔頭的窩為主要個人網站。
* 2015年 架設 實況資料庫 做為實況主們專用的百科網站。
* 2016年 架設 [https://web.archive.org/web/20171220131726/http://twitchtw.com/ Twitch全中文妹子]網站，裡面專門收錄[[Twitch_(網站)|Twitch]]上的中文女實況主。

== 遊戲中文化經歷 ==
從小學時期就很喜歡將英文軟體修改為中文語言。隨後能力上升開始翻譯各種遊戲。
* 2010年 加入大陸知名翻譯網站，3DM漢化組。成立3DM正體潤色組，主要宗旨是將3DM漢化組翻譯好的簡體中文改為正體中文並修正兩岸用語成港澳台常用詞句等。
* 2012年 不再代表3DM正體潤色遊戲。但獲得3DM榮耀漢化組長頭銜。
* 2013年 至今還是為很多[[Steam|Steam]]上的大大小小遊戲做正體中文化。

== 替代役生涯 ==
於2014年7月入伍138梯次[[替代役_(中華民國)|替代役]]，躺醫護所到下單位，最終選上[[矯正署|矯正役]]於[[綠島監獄|綠島監獄]]服役半年後因[[精神官能症|精神官能症]]驗退。

== 獨立遊戲製作經歷 ==
由於大學要交畢業專題，兔頭獨自以聖約翰科技大學為背景用Unity引擎做的線上射擊遊戲。《聖約翰射擊遊戲》

在[[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]]的場外休憩區哈拉區和[[Komica|Komica]]匿名版上爆紅。另外收錄了[[總統府_(臺灣)|總統府]]和[[自由廣場_(臺北)|自由廣場]]、[[中正紀念堂|中正紀念堂]]地圖。完成畢業報告之後即停止開發荒廢至今。

== 爭議事件 ==
* 2008年因違反[[巴哈姆特電玩資訊站|巴哈姆特]] 哈拉區版規遭到全站禁貼二週處分<ref>[https://forum.gamer.com.tw/Co.php?bsn=60147&sn=8009 巴哈姆特全站禁貼公告]</ref>，於Wii Sports[[版主|版主]]職位也被卸下，及永久不能再申請版務職位。
* 2011年於[[奇摩|奇摩]]拍賣和[[露天拍賣|露天拍賣]]上販賣[[軟體|軟體]]序號後因為買家序號失效被肉搜，隨後宣稱已找[[警察|警察]]處理並退回所有款項，自己也是[[受害人|受害人]]。
* 2014年未經台灣代理[[授權|授權]][[翻譯|翻譯]]遊戲 [[總統萬歲5|總統萬歲5]] 遭到代理商警告。
* 於2016年年初[[Facebook|Facebook]]個人用戶[[現場直播|直播]]功能開放之後，即在兩周內每天晚上去不同地方的[[臺灣|台灣]]各大鬼屋[[現場直播|直播]]。如：新莊廢棄醫院、台北廢棄醫院、桃園廢棄陶瓷工廠、台北廢棄會館、新竹廢棄戲院等20處。從首次直播的2000人同時收看到最後一次直播同時最高3萬人同時收看。帶動台灣鬼屋直播熱潮，廣大網友開始爭相直播，[[:en:17_(app)|17]]直播上[[萬姐|萬姐]]也因此去直播鬼屋。隨後遭到[[廢墟攝影|廢墟]]愛好者與廢墟持有者警告而打消直播念頭。
* 2016年8月於[[Twitch|Twitch]]做戶外直播[[精靈寶可夢GO|精靈寶可夢GO]]違規騎[[YouBike微笑單車|Ubike]]單車邊持手機抓寶上新聞<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20160806/923522/ 實況主直播騎車抓寶可夢　網友：危險三寶]</ref>。
* 2016年8月於[[Twitch|Twitch]]做戶外直播精靈寶可夢GO時在台北捷運車廂內飲用飲料遭檢舉上新聞<ref>[https://www.youtube.com/watch?v=u04J3ER-t8w 捷運抓寶「喝飲料」　實況主違規自拍道歉 三立新聞台]</ref>。
* 2017年6月因於[[YouTube|Youtube]]上直播[[音樂|音樂]]電台，號稱[[全球|全球]]唯一精緻網路音樂電台，畫面上有[[KTV|KTV]]動態[[歌詞|歌詞]]、音譜峰值、[[時間表|時間表]]、[[背景|背景]]、和音樂標題與封面等，觸犯[[著作權|著作權]]遭到[[台灣唱片出版事業基金會|台灣唱片出版事業基金會]]提告<ref>[http://news.ltn.com.tw/news/life/breakingnews/2096376 網路音樂平台遭控侵權 實況主兔頭哥：鄭重道歉]</ref>，最終結果緩起訴處分。
* 2017年6月因前面的電台事件與[[周杰倫|周杰倫]]旗下知名電競團隊 [[J_Team|Jteam]] ，解約<ref>[https://www.facebook.com/jteamstreaming/posts/320281231737588 Jteam FB解約聲明]</ref>。
* 2018年6月27因在直播上談論[[Twitch|Twitch]]可利用工具獲得大量免費小奇點，遭到黑粉大量斗內洗錢並惡意檢舉讓[[Twitch|Twitch]]官方人員於後台查到有非法收益，而遭到[[Twitch|Twitch]]永久停權處分。
* 2018年9月5日因網路上有一套疑似檢舉者本人的說法，指稱[[Twitch|Twitch]]頻道之所以會被永久停權因為是《自己斗內給自己》，遭到[[粘世昕|粘世昕]]本人提告<ref>[https://www.facebook.com/Small4Home/posts/1601466369958401 Facebook粉絲專頁公告提告黑粉]</ref>。
