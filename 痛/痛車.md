'''痛車'''（{{lang-ja|痛車}}）指在車身外觀將[[ACG|ACG]]作品人物的圖案以彩繪或貼紙黏貼等方式裝飾上去的車輛，其發源於[[日本|日本]]，是ACG文化的一環。

痛車最常見的車種是[[汽車|汽車]]與[[速克達|速克達]]的[[摩托車|摩托車]]，因為車身有較大面積可裝飾ACG圖案；另外在[[腳踏車|腳踏車]]（稱為「痛單車」（{{lang|ja|痛チャリ}}））<ref>[http://www.akibablog.net/archives/2007/08/itachari-070806.html アキバblog 痛自転車が9台　アキバで「痛チャリ」オフ」 2007年8月6日]</ref>及[[大型車|大型車]]偶而可見，在大型車上的通常是相關廠商進行宣傳廣告用。

與[[cosplay|cosplay]]活動類似的是，痛車較常出現於如日本[[東京都|東京都]]的[[秋葉原|秋葉原]]、[[大阪|大阪]]的[[日本橋|日本橋]]等ACG商品販賣集中的地方，以及ACG、[[同人誌|同人誌]]展覽會。另外在一些汽車展以及痛車同好的聚會也會出現。

==起源==
「痛車」是「{{lang|ja|見ていて痛々しい車}}」之簡稱<ref name="sankei20081117">{{Cite news|language=ja |date=2008-11-17 |title=フェラーリも萌える「痛車」ビジネス |url=http://sankei.jp.msn.com/economy/business/081117/biz0811170842000-n1.htm |work=[[產經新聞|產經新聞]] |publisher=[[產業經濟新聞社|產業經濟新聞社]] |accessdate=2010-12-23 |archiveurl=https://web.archive.org/web/20090523174414/http://sankei.jp.msn.com/economy/business/081117/biz0811170842000-n1.htm |archivedate=2009-05-23 |deadurl=yes }}</ref><ref name="cinematoday20101222">{{Cite news|language =ja||date=2010-12-22|title=即完売！「けいおん！」痛車のオリジナルプラモデル&限定品が再販決定！|url=http://www.cinematoday.jp/page/N0029206|work=シネマトゥデイ|publisher=ウエルバ|accessdate=2010-12-23}}</ref><ref>{{Cite journal|language = ja|author = 川口盛之助|year = 2010年|title = 「What」も「How」も顧客にオープン|journal = クラウドものづくり|volume = 2010年冬号（『日経ものづくり』2010年12月号付録）|publisher = 日経BP|pages = pp.34-38}}</ref>。
「痛」在日本的俗語中有著「不忍直視，讓別人看到會非常丟臉」的涵義<ref>{{Cite web|url =http://jp.hujiang.com/new/p739940/|title = 一词多用：「痛い」还是「イタイ」|author = 沪江日语}}</ref><ref name="榎本2009">{{Cite book|language = ja|editor=榎本秋|date=2009-06-05|title=オタクのことが面白いほどわかる本|edition=第1刷|publisher=中経出版|isbn=978-4-8061-3358-2|pages=180,183頁}}</ref>，而有的[[御宅族|御宅族]]將自己的興趣暴露出來會讓人感到「痛」（不忍直視、丟臉），这是一种揶揄的说法。
也有人認為是因為裝飾車身所費不貲，痛到荷包，因此稱為「痛車」<ref>[http://www.pinktentacle.com/2007/04/itasha-pimped-rides-otaku-style/ Itasha: Pimped rides, otaku style]</ref><ref>http://www.gamebase.com.tw/news/news_content.php?sno=86679618&page=1</ref>。

2000年代之後，ACG文化在日本開始流行，因此也開始有人在車子上動腦。且隨著ACG文化的對外輸出，一起流傳到[[中國大陸|中國大陸]]、[[香港|香港]]、[[台灣|台灣]]<ref>[http://ascii.jp/elem/000/000/109/109681/ 痛車 in 台北!　世界は痛車に毒されはじめた……] (Itasha in Taipei! The world started to poisoned by itasha...), Ascii.jp report, Feb 2008</ref>、[[菲律賓|菲律賓]]<ref>[http://www.philanime.com/2009/03/philippine-itasha-cars/ Philippine Itasha Cars] {{webarchive|url=https://web.archive.org/web/20160304085820/http://www.philanime.com/2009/03/philippine-itasha-cars/ |date=2016-03-04 }}</ref>、[[美國|美國]]<ref>{{Cite web |url=http://www.dannychoo.com/detail/mac/eng/image/4523/America+Itasha.html |title=America Itasha |access-date=2009-08-19 |archive-url=https://web.archive.org/web/20090504211948/http://www.dannychoo.com/detail/mac/eng/image/4523/America+Itasha.html |archive-date=2009-05-04 |dead-url=yes }}</ref>及[[歐洲|歐洲]]等地。

==手法==
早期的痛車是以彩繪方式直接繪於車身，隨著彩色大圖輸出的技術日漸普遍，現今的痛車大多是將圖案輸出之後再黏貼於車身，較早期彩繪方式容易且方便撕下圖案更換。

痛單車則是在輪圈加上塑膠蓋，將ACG圖案貼在塑膠蓋上；另外也有在車架鋼管間的三角型區域裝上貼有ACG圖案的塑膠板，少數則直接在車架鋼管上貼圖案。

== 法律问题 ==
[[File:Kymco_Oreimo_itansha_at_Bahamut_Market_20110903a.jpg|缩略图]]
在一些国家，擅自改变车辆外观可能触犯法律。例如《[[中华人民共和国道路交通安全法|中华人民共和国道路交通安全法]]》第十六条规定，不得“擅自改变机动车已經登记的结构、构造或者特征”；《[[中华人民共和国道路交通安全法实施条例|中华人民共和国道路交通安全法实施条例]]》第十三条规定“机动车喷涂、粘贴标识或者车身广告的，不得影响安全驾驶”；一般来说，在中国大陆，车身喷绘、贴纸面积若超过车身面积的30%，需向交管部门报备，否则将会被视为非法[[改装车|改装车]]，并处以罚款<ref>{{cite web|author1=玩车教授|title=贴纸面积不要超过30% 不然就要罚款哟|url=http://auto.sohu.com/20151206/n429943297.shtml|website=搜狐|accessdate=2017-03-28|date=2015-12-06}}</ref>。在台湾，於2016年5月開放合法彩繪車輛，只需到[[交通部公路總局|交通部公路總局]]所轄[[監理所|監理所]]辦理車輛變更登記與臨時檢驗，在[[行車執照|行車執照]]上所登記的車色為「彩繪車」<ref>[https://tw.autos.yahoo.com/news/%E4%BA%A4%E9%80%9A%E9%83%A8%E5%AE%8C%E6%88%90%E4%BF%AE%E6%B3%95-%E5%BD%A9%E7%B9%AA%E8%BB%8A%E8%BC%9B%E5%90%88%E6%B3%95%E4%B8%8A%E8%B7%AF-044602784.html 交通部完成修法，彩繪車輛合法上路]</ref>。
