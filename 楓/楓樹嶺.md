'''楓樹嶺'''（[[英語|英語]]：Maple Ridge）是一個位於[[加拿大|加拿大]][[卑詩省|卑詩省]][[大溫哥華地區|大溫哥華地區]]的地方行政區。它位於[[菲沙河|菲沙河]]北岸，西鄰[[匹特草原|匹特草原]]，東接[[米遜_(卑詩省)|米遜]]，北臨[[菲沙河谷區域局|菲沙河谷區域局]]的己選區，南面則隔菲沙河與[[蘭里區|蘭里區]]和[[亞博斯福|亞博斯福]]相望。據2011年人口普查，楓樹嶺約有76,052名居民。<ref name="StatsCan2011"/>

楓樹嶺於1874年9月12日設立地方行政架構，其管轄範圍亦包括匹特草原。然而，匹特草原的居民卻於1892年申請脫離楓樹嶺並恢復成一個不設地方行政架構的社區。<ref name="PMHistory1">[http://www.pittmeadows.bc.ca/EN/main/residents/43/754.html Pitt Meadows - History] - 匹特草原官方網頁 歷史頁面</ref>卑詩省政府於1967年設立[[不列顛哥倫比亞省行政區劃|區域局]]此行政等級，並將楓樹嶺劃歸[[:en:Dewdney-Alouette_Regional_District|杜德尼-亞魯艾特區域局]]。該區域局於1995年解散後<ref>[http://www.env.gov.bc.ca/bcgn-bin/bcg10?name=14745 BCGNIS Geographical Name Details] {{webarchive|url=https://archive.is/20070815195405/http://www.env.gov.bc.ca/bcgn-bin/bcg10?name=14745 |date=2007-08-15 }}</ref>，楓樹嶺連同匹特草原改歸[[大溫哥華區域局|大溫哥華區域局]]。

== 交通 ==
[[File:Port_haney.PNG|thumb]]
楓樹嶺的公共交通服務是由[[運輸聯線|運輸聯線]]（TransLink）提供。連接[[溫哥華|溫哥華]][[溫哥華市中心|市中心]]和[[米遜_(卑詩省)|米遜]]的[[西岸快車|西岸快車]][[通勤鐵路|通勤鐵路]]於楓樹嶺設有兩個車站，分別為[[:en:Maple_Meadows_Station|楓樹草原站]]和[[:en:Port_Haney_Station|黑尼港站]]，而市内亦有多條公車綫。

市内的主要幹道為卑詩省[[:en:British_Columbia_Highway_7|7號省道]]（又稱洛歇公路），向西通往[[匹特草原|匹特草原]]，向東通往米遜。[[金穗大橋|金穗大橋]]則橫跨[[菲沙河|菲沙河]]，通往[[素里|素里]]和[[蘭里區|蘭里區]]。

== 教育 ==
楓樹嶺的公校是由[[:en:School_District_42_Maple_Ridge-Pitt_Meadows|42號校區]]管理。區内的公立教育設施包括18間小學、5間中學、1間成人教育中心以及1間社區學院<ref>http://www.sd42.ca/schools</ref>。42號校區亦同時管理匹特草原的公校。
