'''六家車站'''位於[[台灣|台灣]][[新竹縣|新竹縣]][[竹北市|竹北市]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[六家線|六家線]]的[[鐵路車站|鐵路車站]]。本站與[[高鐵新竹站|高鐵新竹站]][[共站|共站]]，2011年11月11日隨著工程完工而啟用。

== 站名由來 ==
本地舊地名為隘口，今日此地仍屬於-{[[隘口里|隘口里]]}-。日治時期1916年（大正5年）[[帝國製糖|帝國製糖]]開辦[[下山線|下山線]]客運業務，曾在附近設有隘口車站。1933年（昭和8年）停辦。

六家之名衍生自車站西方1.5公里的六張犁聚落（今東平里）。[[台灣日治時期|台灣日治時期]]1920年－1937年這一帶全屬-{[[六家庄|六家庄]]}-（相當於現今的『鄉』），後來六家-{庄}-與-{[[舊港庄|舊港庄]]}-合併成為-{[[竹北庄|竹北庄]]}-，即今[[竹北市|竹北市]]前身。不過六家之名仍然泛指附近區域，因此車站設立後即以此為名。

== 車站構造 ==
*高架車站，設[[島式月台|島式月台]]一座。(原車折返新竹站)
*[[鐵路車站#終點站|終端式車站]]。
*2樓車站大廳與{{rint|tw|thsr}}[[高鐵新竹站|高鐵新竹站]]間有通廊銜接。

== 利用狀況 ==
*本站為[[臺灣鐵路管理局車站等級#簡易站|乙種簡易站]]，由[[竹東車站|竹東站]]管理。
*僅停靠[[臺鐵區間車|區間車]]。
*位於[[高鐵新竹站|高鐵新竹站]]西南側，可直接轉乘高鐵。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*2011年11月11日：預定隨著六家線正式通車而啟用，並開始免費乘車六家線及內灣線。
*2011年11月25日：免費乘車活動結束。
*2013年9月30日：啟用多卡通刷卡機。
