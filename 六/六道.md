{{佛教}}
'''六道'''（{{lang-sa|{{IAST|ṣaḍ-gatīḥ}}}}），又名'''六趣'''（趣[[通假字|通]]趋，意为趋向的轮回道及环境）、'''六凡'''、'''六途'''或'''轮回六道'''等，佛教名詞，意指六種[[欲界|欲界]]、[[色界|色界]]及[[無色界|無色界]]眾生的種類型態或者說是境界，也是在[[凡夫|凡夫]]众生[[轮回|轮回]]之道途，它是佛教宇宙觀的一部份。六道（六凡）屬於[[十法界|十法界]]（十界），和四聖合稱為“四聖六凡”<ref>{{cite web|url=http://buddhapark.myweb.hinet.net/Series/Retribution/retribution-5.htm|title=第五篇深信因果輪迴念佛求生極樂}}</ref>。

六道可分為'''三善道'''和'''[[三恶道|三恶道]]'''（又称三途、三恶趣）。三善道為[[天_(佛教)|天]]、[[阿修羅_(佛教)|阿修罗]]、[[人道_(佛教)|人]]； 三恶道為[[畜生|畜生]]、[[饿鬼|饿鬼]]、[[地獄_(佛教)|地狱]]。早期經論說[[五道|五道]]，是指除了[[阿修羅_(佛教)|阿修罗]]以外的欲界五種眾生境界<ref>《佛學大辭典》【五趣】：（名數）又曰五惡趣，五道等。一地獄，二餓鬼，三畜生，四人，五天。文句十二曰：「從一至一，故名為趣。」無量壽經曰：「開示五趣，度未度者。」<br/>【五道】：（名數）為有情往來之所，故曰道。有五處：一地獄道，二餓鬼道，三畜生道，四人道，五天道。與五趣同。</ref>，因為[[阿修羅_(佛教)|阿修罗]]事實上遍佈在其它的五個道，所以沒有特別單獨列出，天道阿修羅也是一種神靈，因為好勇鬥狠，能與天神爭鬥，其他道的阿修羅只是指心性殘暴的眾生，有的有[[神通|神通]]。阿修罗雖為善道，擁有神通法力，因德不及天，故又曰非天；以其有憤恨之苦，尚甚于人，故有時被列入三恶道中，合称為'''四恶道'''。

[[四生|四生]]和六趣（趨）在漢傳佛教合稱'''生趣'''。因'''[[欲界|欲界]]'''（{{lang-sa|kāma-dhātu}}）爲[[欲界天|欲界天人]]、[[人道_(佛教)|人]]、[[阿修羅_(佛教)|阿修罗]]、[[畜生道|畜生]]、[[饿鬼道|餓鬼]]、[[地獄道|地獄]]这[[六道|六道]]众生杂居之地，故也可以欲界代指六道境界。

==經典出處==
佛經常說到五道或六道。如：
*《[[法华经|法华经]]》：“六道众生生死所趣。”
*《[[观佛三昧经|观佛三昧经]]》：“轮回六趣，如旋火轮。”
*《[[佛說輪轉五道罪福報應經|佛說輪轉五道罪福報應經]]》（[[劉宋|劉宋]][[三藏法師|三藏法師]][[求那跋陀羅|求那跋陀羅]]譯）：“因緣合會誰為親，五戒十善除去瞋，不望他許自為親，世間榮樂如浮雲。輾轉五道如車輪，莫計壽命惜金銀，天地尚壞何況身，奉持經戒是大珍。勿貪財色辱悞人，三界眾生如群羊，來去五道身壞傷，命速流水何有常。作惡甚近受罪長，泥犁地獄沸鑊湯，制心剛意離禍殃，犯罪入中痛難當。”

==流转机制==
{{main|轮回|{{le|轮回 (佛教)|Saṃsāra_(Buddhism)}}|佛教因果論}}
六道是生命轮回的六种大体分类，按照[[因果论_(佛教)|因果报应]]的自然规律，造[[善业|善业]]者终得善报，造[[恶业|恶业]]者终得恶报。善多恶少者又在因缘和合下，多往生善道；反之善少恶多者多下堕恶道。大善之人多生在[[天_(佛教)|天道]]，大恶之人多下[[地狱|地狱]]。

[[佛教|佛教]]相信，任何人若严格遵守[[五戒|五戒]]，来世可得[[六根|六根]]整然的人身。若在五戒上，再加行[[十善|十善]]，即可生到天界。若要出轮回，也需要以五戒十善为基础，按[[四聖諦|四聖諦]]、[[八聖道|八聖道]]、[[七覺分|七覺分]]、[[十二因緣|十二因緣]]等佛法修行，最终取[[阿罗汉|阿罗汉]]果及成就[[菩萨|登地菩萨]]乃至[[佛陀|佛陀]]果，便能不受後有，脫離輪迴。

== 三善道 ==
[[File:Bhavachakra.jpg|thumb]]。]]
* [[天_(佛教)|天道]]（[[天人_(佛教)|天人]]）：受人间以上胜妙果报之所。因在人道的人守十善而生天的天人，可以[[神足通|飛翔]]、有[[神通|神通]]，但是會有死亡，顯現[[天人五衰|天人五衰]]相，最終入輪迴。例如[[帝釋天|帝釋天]]示現天人五衰並預知自己將投胎為驢子的[[公案|公案]]<ref> {{Cite book | author =晉世沙門法炬共法立譯| title =《法句譬喻經》卷第一〈無常品第一〉|publisher =CBETA 電子佛典| url =http://buddhism.lib.ntu.edu.tw/BDLM/sutra/chi_pdf/sutra2/T04n0211.pdf | language =zh| quote =  昔者天帝釋五德離身，自知命盡當下生世間，在陶作家受驢胞胎。何謂五德？一者身上光滅，二者頭上華萎，三者不樂本坐，四者腋下汗臭，五者塵土著身。以此五事自知福盡，甚大愁憂。自念三界之中，濟人苦厄唯有佛耳，於是奔馳往到佛所。時佛在耆闍崛山石室中，坐禪入普濟三昧。天帝見佛，稽首作禮伏地，至心三自歸命佛法聖眾，未起之間其命忽出，便至陶家驢母腹中作子。時驢自解走瓦坏間破壞坏器，其主打之，尋時傷胎，其神即還入故身中，五德還備復為天帝。佛三昧覺，讚言：「善哉天帝！能於殞命之際歸命三尊，罪對已畢，不更勤苦。」}} </ref>。
* [[阿修羅_(佛教)|阿修羅道]]：大力神之生所。因常怀嗔心而好鬥，於天界向天神發起戰爭。
* [[人_(佛教)|人道]]：人类之生所，有四洲之分。能稍修[[五戒|五戒]]、[[不殺生|不殺生]]、不偷盜、不[[欲邪行|邪淫]]、不妄語、不飲酒，可生人道。
有些特殊的情況使得在三善道中福份不全，如天道中的[[天人五衰|五衰]]的天人，人道中貧苦殘弱者（例如慳吝不[[佈施|施]]來世即使生人道也可能得貧窮果報），阿修羅中被天神鎮壓者等。

== 三恶道 ==
{{main|三惡道}}
* [[畜生道|畜生道]]：傍生之生所。多以人界为依所，而眼可见。
* [[餓鬼道|餓鬼道]]：常求饭食之鬼类生处。与人界共处，而眼不可见。
* [[地獄_(佛教)|地獄道]]：恶人受苦痛之处，在地下受八热八寒之苦。

有些特殊的情況，使得在[[三惡道|三惡道]]中仍得享樂，此輩是生前積德甚多者，如畜生道中的[[那伽|神龍]]、[[摩睺羅伽|地龍]]、[[金翅鳥|金翅鳥]]，鬼道中的[[夜叉|夜叉]]、[[羅剎|羅剎]]，[[地獄道|地獄道]]中的[[牛頭馬面|牛頭馬面]]鬼差等<ref>《立世阿毘曇論》：「大地獄中，以大獄卒作王富自在。閻羅處地獄，以閻羅王作王富自在。一切禽獸及水羅剎，以婆婁那王作王富自在。諸蛇龍等，以婆脩吉龍王作王富自在。諸大龍者，以摩那思龍王作王富自在。諸象龍者，以婁闍耆利象王作王富自在。諸飛鳥者，以迦婁羅王作王富自在。四足步行眾生，以師子王作王富自在。一切鬼道，以鬼尊王作王富自在。一切脩羅道，以四脩羅王作王富自在：一羅睺、二波羅陀、三毘摩質多、四婆利毘盧遮。閻浮提中，以轉輪王作王富自在。一處王作王富自在。國眾尊老作王富自在。有時無王如劫初立。西瞿耶尼以轉輪王作王富自在。國眾尊老作王富自在。有時無王如劫初立。東弗婆提以轉輪王作王富自在。國眾尊老作王富自在。有時無王如劫初立。北欝單越以轉輪王作王富自在。有時無王。四天王處。以四大天王作王富自在。忉利天者。以三十三天作王富自在。夜摩天者。以脩夜摩王作王富自在。兜率陀天。以善足意王作王富自在。化樂天者以善化王作王富自在。他化自在天。以令自在王作王富自在。一切欲界中。以有惡魔王作王富自在。一千世界中。以大梵天王作王富自在。世間及諸天魔王所大梵處。沙門婆羅門及人天處。以如來阿羅訶三藐三佛陀法。然作王富自在。」
</ref>。

==五道==
在一些[[初期佛教|初期佛教]]經典中，皆只有提到'''五趣'''（或'''五道'''）<ref>
《雜阿含經》卷15〈406經〉：「佛告阿難：「盲龜浮木，雖復差違，或復相得。愚癡凡夫漂流'''五趣'''，暫復人身，甚難於彼。所以者何？彼諸眾生不行其義、不行法、不行善、不行真實，展轉殺害，強者陵弱，造無量惡故。是故，比丘！於四聖諦當未無間等者，當勤方便，起增上欲，學無間等。」」</ref><ref>《雜阿含經》卷16〈436經〉：「佛住王舍城迦蘭陀竹園。爾時，佛告諸比丘：「譬如五節相續輪，大力士夫令速旋轉。如是，沙門、婆羅門於此苦聖諦不如實知，此苦集聖諦、苦滅聖諦、苦滅道跡聖諦不如實知，輪迴五趣，而速旋轉，或墮地獄，或墮畜生，或墮餓鬼，或人、或天，還墮惡道，長夜輪轉。是故，比丘！於四聖諦未無間等者，當勤方便，起增上欲，學無間等。」　佛說此經已，諸比丘聞佛所說，歡喜奉行。」</ref><ref>《長阿含經》卷13〈20經〉：「彼以定心，清淨無穢，柔濡調伏，住無動處，一心修習見生死智證。彼天眼淨，見諸眾生死此生彼、從彼生此、形色好醜、善惡諸果、尊貴卑賤、隨所造業報應因緣皆悉知之。此人身行惡，口言惡，意念惡，誹謗賢聖，信邪倒見，身敗命終，墮三惡道；此人身行善，口言善，意念善，不謗賢聖，見正信行，身壞命終，生天、人中。以天眼淨，見諸眾生隨所業緣，往來'''五道'''，譬如城內高廣平地，四交道頭起大高樓，明目之士在上而觀，見諸行人東西南北，舉動所為皆悉見之。摩納！比丘如是，以定心清淨，住無動處，見生死智證。以天眼淨，盡見眾生所為善惡，隨業受生，往來'''五道'''皆悉知之，此是比丘得第二明。斷除無明，生於慧明，捨離闇冥，出智慧光，此是見眾生生死智證明也。」</ref><ref>《大毘婆沙論》卷172：「已說五趣一一差別，於彼中有阿素洛今當說。謂有餘部立阿素洛為第六趣，彼不應作是說，契經唯說有五趣故。」
</ref>。[[部派佛教|部派佛教]]傳統中，皆以五道為主，如[[說一切有部|說一切有部]]<ref name="大智度">《大智度論》卷10：「問曰：於五道眾生中，佛是天人師，不說三惡道，以其無福，無受道分故。是諸龍、鬼，皆墮惡道中。答曰：佛亦不分明說五道；說五道者，是一切有部僧所說；婆蹉弗妒路部僧說有六道。」</ref>，以及現代[[南传上座部佛教|南传上座部佛教]]的[[巴利三藏|巴利三藏]]等。<ref>郭忠生，[http://ccbs.ntu.edu.tw/FULLTEXT/JR-AN/an008_06.htm 六道與五道]</ref>

阿修羅並無獨立，依其出生的不同（[[化生|化生]]、[[胎生_(佛教)|胎生]]、[[卵生_(佛教)|卵生]]和[[濕生|濕生]]），納入天、人、餓鬼或畜生<ref>《楞嚴經·九》</ref>，地狱道没有阿修罗，天道阿修羅也屬於天道神靈，但與地居天人居住環境不同，心性也暴躁易怒，常嫉妒天人，與之爭戰。最早提出六道說法的可能是[[犢子部|犢子部]]<ref name="大智度" />，將阿修羅及[[揵闥婆|揵闥婆]]等列為[[阿修羅道|阿修羅道]]<ref>《大智度論》卷10：「復次，應有六道，何以故？三惡道一向是罪處，若福多罪少，是名阿修羅、揵闥婆等，生處應別。以是故，應言六趣。」</ref>。大眾部支持六道說，[[大眾部|大眾部]]中的支派如[[案達羅派|案達羅派]]、[[北道派|北道派]]皆採用六道，故逐漸形成了六道的說法，但《[[大毗婆沙论|大毗婆沙论]]》、《[[俱舍论|俱舍论]]》、《[[显宗论|显宗论]]》、《[[正理论|正理论]]》、《[[杂心论|杂心论]]》等诸论，都只设五趣。

== 七道 ==
《[[楞嚴經|楞嚴經]]》中，六道加上仙道為七趣<ref>《大佛頂首楞嚴經講記（上冊）》，[[釋倓虛|倓虛]]老法師，財團法人佛陀教育基金會</ref>{{rp|357}}<ref>《楞严经》卷九：“ 阿难，如是地狱、饿鬼、畜生、人及神仙、天、阿修罗，精研七趣，皆是昏沉诸有为相，妄想受生，妄想随业。”</ref>，是依眾生所生的環境而言。

==艺术表现==
[[有轮|有轮]]（{{lang-sa|भवचक्र|bhavachakra}}；{{lang-pi|भवचक्क|bhavachakka}}；{{bo|སྲིད་པའི་འཁོར་ལོ་|w=srid pa'i 'khor lo|t=sipé khorlo}}），对六道轮回的视觉表现形式，形象比拟出三有、二十五有等生死迷界犹如转轮不息。<ref>《佛光大辞典》【有轮】</ref>常见於藏传佛教[[唐卡|唐卡]]。
