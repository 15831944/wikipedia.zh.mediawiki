'''{{ruby-py|六|-{zh-cn:lù;zh-tw:ㄌㄨˋ}-}}安市'''是[[中华人民共和国|中华人民共和国]][[安徽省|安徽省]]下辖的[[地级市|地级市]]，位于安徽省西部。东临[[合肥市|合肥市]]，南邻[[安庆市|安庆市]]，西南接[[湖北省|湖北省]][[黄冈市|黄冈市]]，西毗[[河南省|河南省]][[信阳市|信阳市]]，北界[[阜阳市|阜阳市]]，东北与[[淮南市|淮南市]]接壤。地处[[皖西|皖西]]山地，[[大别山|大别山]]北麓丘陵地带，地势西南高，东北低，市境中部为[[长江流域|长江流域]]与[[淮河流域|淮河流域]]的分水岭。有[[淠河|淠河]]、[[史河|史河]]、[[沣河|沣河]]、[[东淝河|东淝河]]等河流，主要呈南北走向，上游建有[[梅山水库|梅山]]、[[响洪甸水库|响洪甸]]、[[佛子岭水库|佛子岭]]、[[磨子潭水库|磨子潭]]、[[龙河口水库|龙河口]]五大水库。全市总面积1.55万平方公里，人口474.1万。六安为全省主要产粮及产茶基地，盛产[[茶叶|茶叶]]，尤以[[六安瓜片|六安瓜片]]、[[霍山黄芽|霍山黄芽]]最为著名。

中國著名女作家[[張愛玲|張愛玲]]的經典小說《[[半生緣|半生緣]]》裡的女主角[[顧曼楨|顧曼楨]]的出生地就安徽省六安市

== 名称读音 ==
“六安”之“六”音本同“陆”（汉语拼音：lù）字，为古时地名，类似的读音还出现在[[南京市|南京市]][[六合区|六合区]]。但《[[现代汉语词典|现代汉语词典]]》自第六版起删去了这一读音，而当地民众仍习惯读作旧音，引发争议<ref>{{cite news|title=词典取消六安“lu”读法改读“liu” 网友抗议|url=http://book.sina.com.cn/news/c/2013-08-26/0845525913.shtml?wbf=more|accessdate=2013-08-26|newspaper=新安晚报|date=2013-08-26}}</ref> 。

== 历史 ==
六安别名“皋城”，盖因中国远古时代司法鼻祖[[皋陶|皋陶]]封地于此而得名。

六安之名首次出现于[[汉武帝|汉武帝]][[元狩|元狩]]二年前121年所置[[六安国|六安国]]，六安之名由此而始。六安国治所在六县，即今六安市境内。[[唐朝|唐朝]]时分属寿州和庐州，其间置舒城县和盛唐县。[[元朝|元朝]]末期始设[[六安州|六安州]]，归[[庐州府|庐州府]]管辖，六安州治所在六安城西侧临河处。

2007年初“[[合武高铁|合武高铁]]”工地上，六安王[[刘庆_(六安王)|刘庆]]墓葬的发现和发掘震动国际考古界。

历史上六安州城共建有4个正城门，分别为东门朝京门、南门德胜门、西门通济门、北门武定门。另外出于便捷还有两个偏门，即东南侧城墙的魁星阁、西北侧临河的便储门，而州治所，也就是当时六安州的政府行政机构，便是设立在便储门旁的鼓楼附近。

== 自然环境 ==
六安市位于东经115°20′－117°14′，北纬31°01′－32°40′之间，南部是山地，北部是平原热、水配合良好。

六安四季分明，气候温和，雨量适中，光照充足，无霜期长。年日照2000－2300小时，常年平均气温15.6℃，年平均降雨量1100毫米左右，主要集中在5－8月，占年总降水量的55%。

== 经济 ==
在[[长三角|长三角]]产业不断升级与安徽参与泛长三角合作的过程中，六安市作为皖江城市带承接东部产业西移的重要轴心，具有承东传西的重要意义。重点打造合六工业走廊的六安市产业承接集中区，融入以合肥为轴心的一轴两翼产业发展战略，是合肥经济圈重要组成部分。

2011年，全市实现农业总产值322.5亿元，全市规模以上工业累计完成工业增加值325.7亿元；全市完成固定资产投资558.1亿元；城市居民人均可支配收入17095元，城市居民人均消费性支出12993元；农村居民人均纯收入5643元。

== 交通 ==
境内有312、206、105国道和合（肥）九（江）、宁（南京）西（安）铁路。建设中的沪陕（上海—西安）、济广（济南—广州）、六武（六安—武汉）、合阜（合肥—阜阳）高速公路。[[宁西铁路|宁西铁路]]和高速电气化的[[合武铁路|合武铁路]]（即沪汉蓉铁路）从六安通过，并即将开工建设可直达汕头的"阜六铁路"，起点安徽阜阳，途径六安、安庆、江西景德镇、鹰潭、金溪、南城、南丰、广昌、宁都、瑞金、会昌、寻乌、广东梅州、汕头。北部淮河航运可直达长江。

[[合肥新桥国际机场|合肥新桥国际机场]]位于六安和合肥之间的交界处，距离六安市区仅40多公里。新桥国际机场是一座4E级国际机场，2020年目标年吞吐量大约达到950万人次，年货物吞吐量可达12.24万吨，年起降10.04万架次，高峰小时3400人次。
