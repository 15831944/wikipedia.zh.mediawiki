'''六学'''是部分[[中国|中国]]网友对[[中国中央电视台|央视]]版的[[西游记_(1982年电视剧)|电视剧《西游记》]][[孙悟空|孙悟空]]的扮演者[[六小龄童|六小龄童]]（本名章金莱）的一些言行进行模仿、[[恶搞|恶搞]]的[[网络迷因|网络迷因]]<ref name="huxiu">{{cite web|author=为你写一个故事 |url=https://www.huxiu.com/article/277243.html |title=刷遍微博的“六小龄童学”，究竟是什么？ |website=虎嗅 |date=2018-12-19 |accessdate=2018-12-22}}</ref><ref name="曾奕">{{cite web|author=曾奕 |url=http://culture.dwnews.com/renwen/news/2018-12-21/60107082.html |title=拆局：中国“美猴王”的变身 |website=多维新闻网 |date=2018-12-22 |accessdate=2018-12-22}}</ref><ref>{{cite web |author1=航通社 |title=自媒体人都在向六小龄童老师学习 |url=https://www.pintu360.com/a61694.html |website=品途商业评论 |accessdate=2018-12-23 |date=2018-12-14}}</ref><ref name="刘楚涵"/>。模仿六小龄童语气说话，或对其言论进行改编的网友则被称为“六学家”<ref name="huxiu"/>。

==兴起==
===起源===
[[File:Journey_to_the_West_on_Star_Reunion_116.JPG|200px]]
六小龄童出身于“章氏[[悟空戏|猴戏]]世家”，是1986年播映的电视剧《西游记》中孙悟空的扮演者。由于该电视剧深入人心，以至于到今天在不少人心中认定最经典的“孙悟空”就是六小龄童，也因此，大众经常能看到六小龄童评论西游记、孙悟空的相关话题<ref name=":1" />。

例如，他曾多次表示反对过度改编、恶搞《西游记》，希望在改编时多一些敬畏之心，传达[[正能量|正能量]]和真善美<ref name="猴年春晚">{{Cite news|url=http://culture.people.com.cn/n1/2016/0706/c1013-28527431.html|title=六小龄童忧虑"猴王"无传人 不上猴年春晚似失实得|access-date=2018-12-20|archive-url=https://web.archive.org/web/20160707155038/http://culture.people.com.cn/n1/2016/0706/c1013-28527431.html|work=广州日报|archive-date=2016-07-07|dead-url=no}}</ref>，却在恶搞《西游记》的多部电视剧和电影中出演角色，在痛斥恶搞的同时参与恶搞有[[双重标准|双重标准]]之嫌<ref name="张楠"/><ref name="二唯">{{cite web |author1=二唯 |title=一只孙悟空，让六小龄童挣了多少钱？ |url=https://www.caishimv.com/index.php/web/web_archives/info_details.html?id=9122 |website=财视Media |accessdate=2018-12-28 |archiveurl=https://web.archive.org/web/20181228030811/https://www.caishimv.com/index.php/web/web_archives/info_details.html?id=9122 |archivedate=2018-12-28 |date=2018-12-27}}</ref>。在电视剧《[[吴承恩与西游记|吴承恩与西游记]]》中<ref name="m1905"/>，他扮演的[[吴承恩|吴承恩]]<ref>{{cite web|author=娱乐大咖内参 |url=https://www.jiemian.com/article/526504.html |title=猴年春晚六小龄童缺席？你们还不知道IP时，他就开始运营IP周边了 |website=界面新闻 |date=2016-01-29 |accessdate=2018-12-27}}</ref><ref>{{cite web |url=http://ent.163.com/special/00033RHV/wuchengenxyj.html |title=电视剧《吴承恩与西游记》 立体西游记 |website=网易娱乐 |accessdate=2018-12-27 |archive-url=https://web.archive.org/web/20181227230200/http://ent.163.com/special/00033RHV/wuchengenxyj.html |archive-date=2018-12-27 |dead-url=no }}</ref>却类似[[耍猴|耍猴]]小丑<ref name=hx>{{cite web|author= |url=http://www.hxnews.com/news/yl/ylxw/201812/21/1675047_2.shtml |title=六小龄童被爆大量“黑料”，半生名声毁于一旦，究竟是真是假？（2） |publisher=海峡网 |date=2018-12-21 |accessdate=2018-12-21}}</ref>。在剧中，恶霸被美猴王附体的吴承恩打跑，甚至能一棒子打死[[红孩儿|红孩儿]]。某一集中，吴承恩伪装成猴子，几句话将俺答兵吓走，后被[[嘉靖帝|嘉靖帝]]封为[[明朝|大明]]的猴王<ref>{{cite web |url=https://www.iqiyi.com/v_19rrh8s2nc.html |title=吴承恩与西游记第27集 |publisher=爱奇艺•电视剧 |accessdate=2018-12-22 |archive-url=https://web.archive.org/web/20181222173637/https://www.iqiyi.com/v_19rrh8s2nc.html |archive-date=2018-12-22 |dead-url=yes }}</ref>。

他曾称其“拍《西游记》，没有替身，再苦再难的戏，都是自己上”，但《西游记》的摄像师[[王崇秋|王崇秋]]称至少有四个人扮演过孙悟空，六小龄童矢口否认并声称要把替身之一[[徐霆雷|徐霆雷]]告上法庭<ref name="m1905"/><ref name="金错刀"/><ref name="卢婷雯">{{cite web |author1=卢婷雯 |title=他当过六小龄童的替身 东山渔民临危受命，在《西游记》中饰演出海拜师的美猴王 |url=http://www.taihainet.com/news/fujian/yghx/2016-01-31/1661332.html |website=台海网 |accessdate=2018-12-22 |archiveurl=https://web.archive.org/web/20181222032529/http://www.taihainet.com/news/fujian/yghx/2016-01-31/1661332.html |archivedate=2018-12-22 |date=2016-01-31}}</ref><ref name="NE恶灵">{{cite web |author1=NE恶灵 |title=童年“美猴王”六小龄童被人扒出的黑历史（二） |url=https://news.china.com/news100/11038989/20170508/30498163.html |website=中华网 |accessdate=2018-12-22 |archiveurl=https://web.archive.org/web/20181213124748/https://news.china.com/news100/11038989/20170508/30498163.html |archivedate=2018-12-13}}</ref><ref>{{cite web |author1=娱乐的情人迷 |title=六小龄童拍《西游记》不用替身? 网友: 别逗了, 我们不是傻子 |url=https://www.63nf.com/Ent/4/139966.shtml |website=南方网 |accessdate=2018-12-22 |archiveurl=https://web.archive.org/web/20181222162150/https://www.63nf.com/Ent/4/139966.shtml |archivedate=2018-12-11 |date=2018-12-11}}</ref><ref name=d/>。

===兴盛===

==影响和回应==
===影响===
因为有太多的负面消息，六小龄童举办的个人自传的北京首发仪式及读者签售会活动取消了媒体采访环节。有文章评论称，六小龄童应该像孙悟空那样拿出精神和勇气主动面对媒体表明态度，“逃避是最坏的选择”<ref name="张楠">{{cite web |author1=张楠 |title=六小龄童签售会取消采访，负面消息频出“有小组织在黑我 |url=http://www.yangtse.com/app/ent/2018-12-31/657526.html |website=扬子晚报网 |accessdate=2019-01-03 |date=2018-12-31 |archive-url=https://web.archive.org/web/20190103210353/http://www.yangtse.com/app/ent/2018-12-31/657526.html |archive-date=2019-01-03 |dead-url=no }}</ref><ref>{{cite web |title=因负面消息过多 六小龄童签售会临时取消采访环节 |url=http://news.stnn.cc/ent/2018/1231/604279.shtml |website=星岛环球网 |publisher=北京青年报 |accessdate=2019-01-03 |date=2018-12-31 |archive-url=https://web.archive.org/web/20190103161219/http://news.stnn.cc/ent/2018/1231/604279.shtml |archive-date=2019-01-03 |dead-url=no }}</ref><ref name="陈家沛">{{cite web |author1=陈家沛 |title=六小龄童取消采访，逃避是最坏的选择 |url=http://hlj.rednet.cn/c/2019/01/02/4804693.htm |website=红网 |accessdate=2019-01-03 |date=2019-01-02 |archive-url=https://web.archive.org/web/20190103110455/http://hlj.rednet.cn/c/2019/01/02/4804693.htm |archive-date=2019-01-03 |dead-url=no }}</ref>。

===回应===
对于“六学”的兴起，六小龄童在接受媒体采访时回应称“不能挖苦我或者对我人格进行侮辱，现在有幕后问题，大家要适可而止！并不是某一个人对我的质疑，已经形成了一个非常不好的小组织了，号召怎么去黑（我）”，而且他建议政府尽快实行[[中国网络实名制|中国网络实名制]]<ref name="张楠"/><ref>{{cite web |title=六小龄童谈被黑：有小组织在黑我 希望网络实名制 |url=http://sd.people.com.cn/n2/2018/1227/c386915-32461116.html |website=人民网山东频道 |publisher=北青网娱乐 |accessdate=2019-01-03 |date=2018-12-27}}</ref>。

==评论==
《[[北京日报|北京日报]]》发表文章《六小龄童，对你也该念念[[紧箍咒|紧箍咒]]》，文中指出：这几年，除了每次在和西游记相关的公共事件中反复蹭热度，六小龄童屡屡宣传的中美合拍的西游题材电影总不见影子，反而是常常批判他人改编的西游题材作品，自诩为唯一正统。此做法并不能推动西游文化的传承，反倒体现了艺人心中的[[文化霸权|文化霸权]]<ref>{{cite news |author1=木婉清 |title=六小龄童，对你也该念念紧箍咒 |url=http://bjrb.bjd.com.cn/html/2018-12/05/content_303278.htm |accessdate=2018-12-22 |work=北京日报 |issue=第23929期 |publisher=北京日报社 |date=2018-12-05 |page=第12版：文化}}</ref>。

《[[新京报|新京报]]》发表文章《“六学”走红：为什么我们对[[黑话|黑话]]游戏欲罢不能？》称，“六学”的爆红伴随着六小龄童个人[[声誉|声誉]]的彻底坍塌，这和他最近几年的言行紧密相关：在“雷剧”中出演主角、批判他人对孙悟空的演绎、维护自己诠释《西游记》的正统地位、在悼念杨洁的视频中推广自己的电影……六小龄童的“六学”正如[[蔡依林|蔡依林]]的“[[淋文化|淋文化]]”，是网络流行文化体系中的黑话系统<ref>{{cite news |author1=安安 |title=“六学”走红：为什么我们对黑话游戏欲罢不能？ |url=http://www.bjnews.com.cn/culture/2018/12/25/533578.html |accessdate=2018-12-27 |work=新京报 |date=2018-12-25 |archive-url=https://web.archive.org/web/20181226233308/http://www.bjnews.com.cn/culture/2018/12/25/533578.html |archive-date=2018-12-26 |dead-url=no }}</ref>。
