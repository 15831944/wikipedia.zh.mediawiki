'''六朝事迹编类'''，史地杂记。南宋[[张敦颐|张敦颐]]撰。[[绍兴|绍兴]]三十年（1160年）撰成，14卷。

== 编者 ==
张敦颐，字养正，婺源（今属江西）人。绍兴八年（1138年）进士。历知舒、衡二州。

== 内容 ==
此书是张敦颐在建康（今南京）的江东路安抚使兼知建康韩仲通幕中任干办公事时，为补“图经、实录”之脱漏而作，书成由[[建康府学|建康府学]]刊刻。全书分总叙、形势、城阙、楼台、江河、山冈、宅舍、谶记、灵异、神仙、寺院、庙宇、坟陵、碑刻14门。作者以[[六朝|六朝]]以来80余种典籍与所存碑记相参订，材料详备。故其内容虽然超出六朝，上溯吴越，下及唐宋，致有"失于断限"之讥，也不能掩盖其较高的存史价值。

== 后世评价 ==
《[[四库全书總目提要|四库全书總目提要]]》认为作者宣扬北伐必败，过于卑懦，是主和派的论调，但也承认其说法至少比虚夸形胜者要切实。

== 版本 ==
*2卷本，丛书本有明《[[古今逸史|古今逸史]]》、清《[[四库全书|四库全书]]》、《[[丛书集成初编|丛书集成初编]]》。

*单行本有清初[[冯知十|冯知十]]家抄本、清[[不不通阁|不不通阁]]抄本、清嘉庆[[沈兆沄|沈兆沄]]校刊本、清[[道光|道光]]辛丑张氏宝德重刊本（1841年）、清抄本（[[中国国家图书馆|中国国家图书馆]]藏）。

*14卷本，清[[光绪|光绪]]十三年[[宝章阁|宝章阁]]仿绍兴建康府学刊本、清末刊本。

*14卷附识1卷本，清道光庚子[[金陵|金陵]]张氏校刊宋本（1840年）。

*1卷本，名《[[六朝事迹|六朝事迹]]》，皆丛书本，有《[[说郛|说郛]]》、《[[锦囊小史|锦囊小史]]》及明末《[[五朝小说|五朝小说]]》和民国[[扫叶山房|扫叶山房]]《[[五朝小说大观|五朝小说大观]]》中的《[[宋人百家小说琐记家|宋人百家小说琐记家]]》。

*解放后校点本，1989年11月[[南京出版社|南京出版社]][[王进珊|王进珊]]校点本、1995年[[上海古籍出版社|上海古籍出版社]][[张忱石|张忱石]]点校本。<ref>版本之多正反映其受重视的程度。所谓14卷、2卷、1卷，只是分卷、排列不同，内容还是一样的。</ref>
