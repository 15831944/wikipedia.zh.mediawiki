'''六枝特区'''是[[中华人民共和国|中华人民共和国]][[贵州省|贵州省]][[六盘水市|六盘水市]]下属的一个[[县级特区|县级行政特区]]，也是全国唯一一个县级特区。地处[[东经|东经]]105°08'-105°43'，[[北纬|北纬]] 25°59'-26°22'。与之接壤的县市有：东边[[镇宁县|镇宁县]]、[[普定县|普定县]]，南边[[关岭县|关岭县]]，西边[[水城县|水城县]]，西南边[[晴隆县|晴隆县]]、[[普安县|普安县]]，北边[[织金县|织金县]]、[[纳雍县|纳雍县]]。是全国重点产[[煤|煤]]地区，素有江南煤海之美称。<ref>{{cite web |url=http://www.gzlps.gov.cn/col/col18621/index.html|title=中国凉都·六盘水·六枝特区|accessdate=2014-04-28}} </ref>

== 历史 ==

1966年，[[六枝县|六枝县]]、[[普定县|普定县]]和[[镇宁县|镇宁县]]划出八个[[公社|公社]]设置六枝特区，同时六枝县复名[[郎岱县|郎岱县]]。1967年，六枝特区、[[盘县特区|盘县特区]]、[[水城特区|水城特区]]、郎岱县、[[水城县|水城县]]、[[盘县|盘县]]，组成[[六盘水专区|六盘水专区]]，六盘水即三特区首字。1970年，六盘水专区改称[[六盘水地区|六盘水地区]]，郎岱县并入六枝特区。1978年12月，六盘水专区改为[[六盘水市|六盘水市]]。<ref>{{Cite web|url=http://www.chiyou.name/page/whyz/dmtx/lps.htm|title=贵州地名来源探析 六盘水市|accessdate=2013-12-18|author=|date=|work=|publisher=黔途网|language=Zh-hans}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 地理 ==
六枝自然条件优越。境内气候宜人，属亚热带季风温暖湿润气候，气候温和，夏无酷暑，冬无严寒，雨量充沛，年均气温14．5℃，极端最高气温34．1℃，极端最低气温-5.6℃。年均日照1252.4小时，年总积温5700℃，年太阳辐射总量3614.6兆焦/平方米。年降水量1476.4毫米，无霜期294天。地貌属中高山类型，地势西北高、东南低。区内地形破碎、切割强烈，山高谷深，河流众多。主要山脉有乌蒙山脉、苗岭山脉等，平均海拔1350米，海拔最高点2126.9米，最低点609.5米。<ref>{{Cite web|url=http://www.gz.xinhuanet.com/2015-02/04/c_1114246460.htm|title=六枝特区概况|accessdate=2015-02-04|author=|date=|work=六枝特区委宣传部|publisher=新华网贵州频道地方网群|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20150509061732/http://www.gz.xinhuanet.com/2015-02/04/c_1114246460.htm|archivedate=2015-05-09}}</ref>

== 教育 ==
区内有四个县级初级中学，分别是六枝特区第二中学（位于友谊路）、六枝特区第三中学（位于文化路）、六枝特区第四中学（位于那平路）和六枝特区第五中学（位于那古坝）；一个县级高级中学——六枝特区第一中学（位于文化路），一个市级高级中学——六盘水市第二中学（位于四方坡）。
