在[[幾何學|幾何學]]中，'''六階六邊形鑲嵌'''是由[[六邊形|六邊形]]組成的[[雙曲面|雙曲面]][[正鑲嵌圖|正鑲嵌圖]]，在[[施萊夫利符號|施萊夫利符號]]中用{6,6}表示。六階六邊形鑲嵌即每個[[頂點|頂點]]皆為六個[[六邊形|六邊形]]的公共頂點，頂點周圍包含了六個不重疊的六邊形，一個六邊形內角120度，六個六邊形超過了360度，因此無法因此無法在平面作出，但可以在雙曲面上作出。

== 對稱性 ==
這個鑲嵌代表一個由六條鏡射線定義一個正六邊形基本域的[[萬花筒|萬花筒]]。這由六個三階交叉反射性在{{en-link|軌型符號|orbifold notation}}被稱為(*333333)。在考斯特表示法可表示為[6<sup>*</sup>,6]，從三個的鏡射線當中移除兩條穿過六邊形中心的鏡射線。 
這個[[萬花筒|萬花筒]]的奇數/偶數基本域可被視為是交替塗色的{{CDD|node_1|split1-66|branch}}鑲嵌：
: [[File:H2_tiling_366-2.png|240px]]

== 相關多面體與鑲嵌 ==
該鑲嵌在拓樸學中也和每個頂點有著六個面的多面體及鑲嵌相關， 施萊夫利符號皆為{n,6}，而考斯特符號為{{CDD|node_1|n|node|6|node}}，從n到無窮。
{{六階正鑲嵌}}
該鑲嵌在拓樸學上和頂點圖是（6<sup>n</sup>）的一系列的鑲嵌的一部份。
