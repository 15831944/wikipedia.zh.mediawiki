'''六氯合铱(III)酸钠'''是一种[[配位化合物|配位化合物]]，化学式为Na<sub>3</sub>[IrCl<sub>6</sub>]，它是暗绿色晶体，易溶于水，可以形成水合物。

==制备==
六氯合铱(III)酸钠可由三氯化铱和氯化钠在浓溶液中反应得到：
: 3 NaCl + IrCl<sub>3</sub> → Na<sub>3</sub>[IrCl<sub>6</sub>]
从溶液中可以结晶出Na<sub>3</sub>[IrCl<sub>6</sub>]·12H<sub>2</sub>O。
:<math>\mathsf{Na_3[IrCl_6]\cdot 12H_2O \ \xrightarrow{50^oC, vacuum}\ Na_3[IrCl_6] + 12H_2O }</math>

==化学性质==
六氯合铱(III)酸钠在高温下分解：
::<math>\mathsf{2Na_3[IrCl_6] \ \xrightarrow{800-1000^oC}\ 2Ir + 6NaCl + 3Cl_2 }</math>

它可以和硫酸反应，也能被热的硝酸氧化：
::<math>\mathsf{3Na_3[IrCl_6] + 6H_2SO_4 \ \xrightarrow{}\ Ir_2(SO_4)_3 + 3Na_2SO_4 + 12HCl\uparrow }</math>
::<math>\mathsf{Na_3[IrCl_6] + 4HNO_3 + H_2O \ \xrightarrow{100^oC}\ IrO_2\downarrow + NO_2 + 3NaNO_3 + 6HCl\uparrow }</math>

在惰性气氛中和碱反应，沉淀出[[三氧化二铱|三氧化二铱]]；在氧气存在下也可以被氧化为[[二氧化铱|二氧化铱]]：
::<math>\mathsf{2Na_3[IrCl_6] + 6NaOH \ \xrightarrow{CO_2}\ Ir_2O_3\downarrow + 12NaCl + 3H_2O }</math>
::<math>\mathsf{4Na_3[IrCl_6] + O_2 + 12NaOH \ \xrightarrow{CO_2}\ 4IrO_2\downarrow + 24NaCl + 6H_2O }</math>

它能够被氢气还原：
::<math>\mathsf{2Na_3[IrCl_6] + 3H_2 \ \xrightarrow{400-450^oC}\ 2Ir + 6NaCl + 6HCl }</math>

加热时能被氯气氧化：
::<math>\mathsf{2Na_3[IrCl_6] + Cl_2 \ \xrightarrow{400-500^oC}\ 2Na_2[IrCl_6] + 2NaCl }</math>
酸性溶液中的空气也能完成氧化过程。<ref>[https://core.ac.uk/download/pdf/35468497.pdf Structural and spectroscopic investigations of halogeno complexes of iridium (III) and iridium (IV)]. [2018-11-2]</ref>

和氯化钾反应生成钾盐沉淀：
::<math>\mathsf{Na_3[IrCl_6] + 3KCl \ \xrightarrow{10^oC}\ K_3[IrCl_6]\downarrow + 3NaCl }</math>
