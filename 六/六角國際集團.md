'''六角國際事業股份有限公司'''（'''La Kaffa International CO., Ltd'''，簡稱'''六角國際餐飲集團'''），是一家[[臺灣|臺灣]]的[[餐馆|餐飲]]事業[[公司集團|集團企業]]，總部位於臺灣[[新竹縣|新竹縣]][[竹北市|竹北市]]，由[[王耀輝|王耀輝]]等人創立於2004年。該企業旗下有「 '''[[Chatime日出茶太|「Chatime」日出茶太」]]'''、「 春上布丁蛋糕」、「 Bake Code 」烘焙密碼、「ZenQ」仙Q冰菓室、「La Kaffa」六角咖啡與「Chatime Lounge」茶時光等，並於2016年八月成立子公司「王座國際餐飲集團」管理旗下「杏子日式豬排」、「段純貞」牛肉麵、「大阪王將」、「初念」個人鍋專賣、「京都勝牛」、等多個餐飲品牌。

== 公司集團歷史 ==
* 2018年：成功進軍火鍋市場，創立「初念」個人鍋專賣，首家門市開立在一級美食戰區--台中。六角國際集團獲得[[京都勝牛|京都勝牛]]台灣代理權在台灣開分店，並於2018年10月5日於台北新光三越A11百貨開設台灣1號店。
*2017年3月28日：六角國際集團拿到[[大阪王將|大阪王將]]台灣代理權在台灣開分店，並於2017年3月28日在台南開設台灣1號店<ref>{{cite news|author=彭夢竺  |title=六角國際代理「大阪王將」　台灣首店插旗台南|url=https://www.nownews.com/news/20170327/2458972|accessdate=2017-03-27|newspaper=《[[NOWnews_今日新聞|NOWnews 今日新聞]]》 |date=2017-07-28}}</ref>。
* 2017年7月27日：跟馬爾地夫的合作夥伴Strada Pvt Ltd.餐飲集團，馬爾地夫開[[Chatime日出茶太|Chatime日出茶太]]分店<ref>{{cite news|author=姚舜  |title=六角拓版圖 日出茶太插旗馬爾地夫|url=http://www.chinatimes.com/newspapers/20170727000218-260206|accessdate=2017-07-27|newspaper=《[[工商時報|工商時報]]》 |date=2017-07-28}}</ref>。
* 2017年7月28日：代理的人氣牛肉麵店「段純貞」積極投資海外開分店，最快2017年底就會在上海開出第一家海外直營店，2018年可能在海外[[加拿大|加拿大]]、[[澳洲|澳洲]]開分店<ref>{{cite news|author=黃冠穎  |title=六角牛肉麵 搶進上海|url=https://money.udn.com/money/story/11074/2608401|accessdate=2017-07-28|newspaper=《[[經濟日報|經濟日報]]》 |date=2017-07-28}}</ref>。
*2017年：收購新竹名店「春上布丁蛋糕」，取得品牌經營權，保留現有經營團隊及品牌狀況下，加入六角提供的產品研發、展店技術服務等資源，未來規畫針對台灣市場進行拓展，亦規畫進軍海外布局。
*2016年：六角國際在台衝刺餐飲品牌，今年首季推出「段純貞牛肉麵」。段純貞牛肉麵是新竹起家，以四川口味的麻辣牛肉麵為口感，與台灣口味牛肉麵做出區隔。
*2008年：日出茶太進軍中國成立上海辦公室。
*2007年：成立「ZenQ仙Q甜品專賣店」。
*2005年：成立「Chatime日出茶太」。
*2004年4月20日：六角國際事業股份有限公司設立於新竹市，推出La Kaffa專業外帶咖啡店。
