'''六角義治'''（{{bd|1545年||1612年|11月14日}}，即[[天文_(後奈良天皇)|天文]]14年－[[慶長|慶長]]17年[[十月廿二|10月22日]]）為日本戰國時代[[近江國|南近江]]的大名。別名：四郎（通稱）、'''六角義弼'''。[[宇多源氏|宇多源氏]][[佐佐木氏|佐佐木氏]]嫡流[[六角家|六角家]]出生。諡號「鵬庵玄雅」、官位至[[右衛門督|右衛門督]]，[[六角義定|六角義定]]的大哥，有一子[[佐佐木定治|佐佐木定治]]。

== 繼位家督、觀音寺騷動 ==
六角義治為[[六角承禎|六角承禎]]([[六角義賢|義賢]])之長子、母為[[畠山義總|畠山義總]]之女，妻子为齋藤郁，[[齋藤義龍|齋藤義龍]]之女。1559年繼承父親的家督之位。但是六角義治是個相當愚蠢的人物，由於嫉妒[[六角家|六角家]]重臣[[後藤賢豐|後藤賢豐]]的影響力，於是永祿6年(1563年)六角義治殺害後藤賢豐，引發[[觀音寺騷動|觀音寺騷動]]，六角義治因此被家臣團放逐出[[觀音寺城|觀音寺城]]，直到家中重臣[[蒲生定秀|蒲生定秀]]、[[蒲生賢秀|蒲生賢秀]]的斡旋協調之下六角義治才得以回到觀音寺城。但是六角家已經因為觀音寺騷動造成相當大的內部分裂，力量因此嚴重衰退，1567年雖然六角義治頒布[[六角氏式目|六角氏式目]]以限制主君權力的方式企圖挽回人心，但也沒有發揮太大的效用。因此有被逼迫把家督讓與其弟[[六角義定|義定]]之一說。

== 流放義昭 ==
永祿8年（1565年），[[三好三人眾|三好三人眾]]與[[松永久秀|松永久秀]]殺害將軍[[足利義輝|足利義輝]]，其弟[[足利義昭|足利義昭]]逃亡藏身在六角家中。於是[[三好三人眾|三好三人眾]]用[[管領|管領]]的職位引誘義治，義治答應所以義昭遭到流放。也因此流落到織田家的[[足利義昭|足利義昭]]成為了[[織田信長|織田信長]]對六角家發動攻略、[[上洛|上洛]]的最好藉口。

== 衰退 ==
永祿11年（1568年）六角家與[[三好三人眾|三好三人眾]]之一的[[岩成友通|岩成友通]]一起對抗[[織田信長|織田信長]]的上洛大軍侵略，一度成功守住，但等守軍情勢稍緩又於隔年再度發動攻擊（[[觀音寺城之戰|觀音寺城之戰]]），因此棄城而逃。之後在六角勢力內的甲賀郡[[石部城|石部城]]與[[朝倉氏|朝倉]]、[[淺井氏|淺井]]、[[三好氏|三好]]聯軍從四處夾攻織田家（[[野田城、福島城作戰|野田城、福島城作戰]]），使信長苦不堪言。因此隨後信長要求朝廷介入和議，但和議不久，信長隨即廢棄和議攻下朝倉，直到毀滅了淺井才與六角家再度議和。之後又再於[[足利幕府|足利]]、[[武田氏|武田]]、[[上杉氏|上杉]]的[[信長包圍網|信長包圍網]]中以甲賀郡[[石部城|石部城]]為據點，義賢、義治父子一直以反對信長為目標而持續作戰，直到天正二年（1574年）石部城遭到織田軍攻陷後（[[石部城之戰|石部城之戰]]），六角義治逃往[[甲斐國|甲斐]]依附武田家。

== 晚年 ==
晚年六角義治投靠侍奉[[豐臣秀吉|羽柴秀吉]]與[[豐臣秀次|羽柴秀次]]，並且出任[[豐臣秀賴|豐臣秀賴]]的弓道師傅而結束一生。
