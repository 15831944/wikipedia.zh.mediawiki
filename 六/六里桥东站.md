'''六里桥东站'''位于[[北京市|北京市]][[丰台区|丰台区]]，是[[北京地铁|北京地铁]][[北京地铁9号线|9号线]]的一座车站。于2011年12月31日随9号线南段开通投入使用。

== 位置 ==
六里桥东站位于北京市区西南部，[[北京三环路|三环路]]内，[[广安路_(北京)|广安路]]（西南－东北向）与[[太平桥路_(北京)|太平桥路]]（约呈南北向）交口西侧，广安路－[[京石高速公路|京石高速公路]]与[[西三环南路_(北京)|西三环南路]]相交的[[六里桥|六里桥]]东，呈西南－东北走向。<ref>{{cite web|url=http://www.bjghw.gov.cn/web/static/articles/catalog_14300/2009-10-19/article_ff80808123fa497b0124529b8055000d/ff80808123fa497b0124529b8055000d.html|title=地铁9号线规划方案公告|publisher=[[北京市规划委员会|北京市规划委员会]]|date=2009-10-14|accessdate=2010-03-28|deadurl=yes|archiveurl=https://web.archive.org/web/20100327030037/http://www.bjghw.gov.cn/web/static/articles/catalog_14300/2009-10-19/article_ff80808123fa497b0124529b8055000d/ff80808123fa497b0124529b8055000d.html|archivedate=2010-03-27}}</ref>

== 结构 ==
该站为地下车站，[[岛式站台|岛式站台]]设计。车站色调为红色。<ref>{{cite web|url=http://www.bjjtw.gov.cn/gzdt/ywsds/201112/t20111209_58060.htm |title=9号线南段将于2011年年底开通试运营 |date=2011-12-09 |accessdate=2011-12-17 |publisher=[[北京市交通委员会|北京市交通委员会]] |deadurl=yes |archiveurl=https://web.archive.org/web/20131224115102/http://www.bjjtw.gov.cn/gzdt/ywsds/201112/t20111209_58060.htm |archivedate=2013-12-24 }}</ref>

== 速度 ==
北京西站到六里桥东线路时速大约在25公里／小时到35公里／小时。此路段因为下穿铁路所以减慢速度。
