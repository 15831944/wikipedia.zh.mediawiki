《'''六度集'''》又名《'''六度集經'''》、《'''六度無極經'''》、《'''度無極集'''》、《'''雜無極經'''》，收於[[大正藏|大正藏]]本緣部。係集錄[[佛陀|佛陀]]在過去世[[菩薩|菩薩]]生時之九十一則[[本生譚|本生譚]]故事（當中有三則非屬故事情節）<ref name="南華"/>，配合布施、持戒、忍辱、精進、禪定、智慧等六度[[波羅蜜多|波羅蜜]]而集成，其特色在於闡揚菩薩行慈悲救苦、捨己利他的故事，為最早漢譯「本生經」的代表<ref name="香光">{{cite web|author1=香光尼眾佛學院圖書館|title=六度集經解題|url=http://www.gaya.org.tw/library/readers/guide-ldjj.htm}}</ref>。

經中之本生譚大部分可見於[[巴利文|巴利文]]本及其他漢譯本。又本經原典已散佚不傳，然據其內容骨幹推知，應成立於二世紀時<ref name="佛光">{{cite web|title=六度集經|url=http://buddhaspace.org/dict/fk/data/%25E5%2585%25AD%25E5%25BA%25A6%25E9%259B%2586%25E7%25B6%2593.html|website=佛光大辭典}}</ref>。

== 題解 ==
「六度」（Paramita），即六[[波羅蜜多|波羅蜜]]，指布施、持戒、忍辱、精進、禪定、般若（明）等六種菩薩的修行法門<ref name="六度研究"/>。

== 傳譯 ==
三國時代吳‧[[康僧會|康僧會]]在太元元年至天紀四年間（二五一～二八０）所譯出<ref name="南華"/>。

== 內容 ==
本經依六波羅蜜之次第分類，與其他本生經之雜然列舉不同。內容分六章，即（１）布施度無極章（２）戒度無極章（３）忍辱度無極章（４）精進度無極章（５）禪度無極章（６）明度無極章。

其中，第一章中之〈普施商主本生〉及〈須大拏經〉分別相當於劉宋‧求那跋陀羅譯的《大意經》、西秦‧聖堅譯的《太子須大拏經》。第二章中之〈太子墓魄經〉相當於題為安世高譯的《太子墓魄經》及西晉‧竺法護譯的《太子墓魄經》。〈頂生聖王經〉相當於宋‧施護等譯的《頂生王因緣經》。第三章中之〈睒道士本生〉相當於西晉譯者佚名的《睒子經》及西晉‧聖堅譯的《睒子經》。第四章中之〈修凡鹿王本生〉相當於題為吳‧支謙譯的《九色鹿經》<ref name="香光"/>。

《六度集經》共有九則與《雜寶藏經》的主題或題材內容相似，且《雜寶藏經》中的本生故事多與《六度集經》有關。 有八則與《賢愚經》的主題或題材內容相似<ref name="南華"/>。

== 考證 ==
現傳本《六度集》當中，有一部分內容是提要。[[湯用彤|湯用彤]]認為該「提要」的部份，是康僧會所寫，他說：「審其內容，決為會所自製，非譯自胡本。」<ref name="六度研究"/>

《六度集經》有改寫及改譯的現象，根據陳洪的研究，它有三個演變的版本(康僧會的原編本、形成於南朝的改編本和形成於隋唐間的新編本)，是一部編譯之作，而非純為翻譯佛典<ref name="六度研究">{{cite journal|author1=釋天常|title=六度集研究|journal=中華佛學研究|date=1998|url=http://www.chibs.edu.tw/ch_html/chbs/02/chbs0203.htm}}</ref><ref name="南華">{{cite book|author1=陳瓔秦|title=《六度集經》菩薩行敘事之研究|publisher=南華大學文學系|url=http://nhuir.nhu.edu.tw/retrieve/21319/100NHU05076017-001.pdf}}</ref>。

== 影響==
康僧會將儒家的仁義忠孝悌等倫理道德，與佛教的五戒、十善、六度等觀念進行比附詮釋。在這一過程中，有意無意地改造了佛教出世修行的思想，以減少與儒家精神的扦格<ref>{{cite web|author1=吳海勇|title=《六度集經》的文學世界|url=http://baohuasi.org/gnews/2016820/2016820344222.html|access-date=2017-09-15|archive-url=https://web.archive.org/web/20170916052908/http://baohuasi.org/gnews/2016820/2016820344222.html|archive-date=2017-09-16|dead-url=yes}}</ref>。杜繼文認為《六度集經》以大乘的「六波羅蜜」，結合[[安世高|安世高]]傳來的「禪數」與[[孟子|孟子]]的「仁義」思想，形成特殊的「[[格義|格義]]」風格，承先啟後的開展中國的大乘佛教思想<ref name="南華"/>。

[[印順法師|印順法師]]認為《六度集經》中，「禪度」與「明度」的部分最少，但「布施」主題，共有二六則，佔近三成，與「忍辱」、「精進」一同展示了救濟眾生的慈心悲行。顯示「[[慈悲|慈悲]]」眾生，在六度的主軸地位。釋尊「本生」重於悲濟，立願在[[娑婆|穢土]]成佛，所以從[[淨土|淨土]]來聽法的，都讚歎釋尊的慈悲，以及在穢土修行的功德<ref>{{cite book|author1=印順|title=初期大乘佛教之起源與開展－第二項 六度集──重慈悲|url=http://yinshun-edu.org.tw/zh-hant/comment/reply/3451}}</ref>。
