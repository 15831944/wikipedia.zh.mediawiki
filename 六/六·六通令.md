'''六·六通令'''，是指1967年6月6日，由[[中共中央|中共中央]]、[[中华人民共和国国务院|国务院]]、[[中央军委|中央军委]]、[[中央文革小组|中央文革小组]]发布的[[通令|通令]]<ref>{{Cite book|title=历史的丰碑 中华人民共和国国史全鉴 7 军事卷|last=[[中共中央党校理论研究室|中共中央党校理论研究室]]编|first=|publisher=北京：[[中共中央文献出版社|中共中央文献出版社]]|year=2005.01|isbn=7-5073-1612-2|location=|pages=274}}</ref>。通令旨在纠正当时出现的打砸抢抄抓的歪风<ref>{{Cite book|title=二十世纪中国大事全书|last=[[张宏儒|张宏儒]]主编；华夏文化促进会、华夏图书研究所《二十世纪中国大事全书》编辑委员会编|first=|publisher=北京：北京出版社|year=1993.05|isbn=7-200-01707-8|location=|pages=590}}</ref>。

== 内容==
{{wikisource|中共中央、国务院、中央军委、中央文革小组通令}}
六·六通令一共有七条<ref>{{Cite book|title=1967年的故事|last=[[王新全|王新全]]，徐鹏堂主编|first=|publisher=延吉：延边大学出版社|year=2005.07|isbn=7-5634-2106-8|location=|pages=102}}</ref>：
# 除国家专政机关奉命依法执行必要的逮捕拘留任务外，任何团体和个人，都不准抓人，都不准私设公堂和变相地私设公堂。
# 各级党政军机关的档案文件和印章，任何团体和个人，都不准抢夺、窃取和破坏。
# 社会主义的国家财产和集体财产绝对不可侵犯，革命群众都有保护的责任，任何团体和个人，都不准侵占，不准砸抢，不准用任何借口进行破坏。
# 严禁武斗，严禁行凶打人，严禁在本单位和外单位打群架，严禁抢夺个人所有的财物。
# 除国家专政机关奉命依法执行任务外，任何团体和个人，都不准对任何团体和个人，进行搜家和抄家。
# 由卫戍部队和驻军负责保证上述各条的执行。自即日起，对于违反上述各条的，都应该严加处理。
# 各革命群众组织，应该成为执行本通令的模范，并协助卫戍部队和驻军进行政治思想工作。
