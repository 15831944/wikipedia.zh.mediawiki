'''六田登'''（{{lang-ja|六田 登}}，{{bd|1952年|7月23日<ref name="kino">{{cite web |url=http://kinobook.jp/07/p142.htm |script-title=ja:KINO Vol.7｜六田登 21世紀のマンガ·ロングインタビュー |publisher= |accessdate=2018-06-15 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20091124071349/http://kinobook.jp/07/p142.htm |archivedate=2009年11月24日 |df= }}</ref>||}}），[[日本|日本]]資深男性[[漫畫家|漫畫家]]、同人作家，[[京都精華大學|京都精華大學]]特任教授<ref>{{cite web |url=http://osaka.yomiuri.co.jp/edu_news/20080403kk01.htm |script-title=ja:京都精華大マンガ学部長に竹宮惠子さん |publisher= |accessdate=2018-06-15 |language=ja }}{{dead link|date=2018年4月}}</ref><ref>{{cite web |url=http://www.yomiuri.co.jp/index.htm |title=YOMIURI ONLINE |publisher= |accessdate=2018-06-15 |language=ja |archive-url=https://web.archive.org/web/20140322065533/http://www.yomiuri.co.jp/index.htm |archive-date=2014-03-22 |dead-url=yes }}</ref>。[[大阪府|大阪府]][[八尾市|八尾市]]出生<ref name="kino" />。

他的作品大多在[[小學館|小學館]]發表。代表作有《{{link-ja|衝鋒勝平|ダッシュ勝平}}》、《{{link-ja|極速悍將|F (漫画)}}》、《ICHIGO 二都物語》及《歌-{麿}-》等。其中，《衝鋒勝平》和《極速悍將》曾經被改編成同名[[電視動畫|電視動畫]]。

== 來歷 ==
六田登從高中時期開始，以參加鐵甲作畫會（<small>{{lang-ja|アストロ作画会}}</small>）和{{link-ja|作畫集團|作画グループ}}（<small>{{lang-ja|作画グループ}}</small>）的同人創作活動起家。

1971年，19歲的六田為了成為漫畫家，搭[[順風車|順風車]]前往東京發展<ref name="csprofile">{{cite web |url=http://www.rokuda-noboru.com/climbers.htm |title=六田先生の活動紹介 |publisher= |accessdate=2018-06-15 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20080509103945/http://www.rokuda-noboru.com/climbers.htm |archivedate=2008年5月9日 |df= }}</ref><ref name="csprofile2">{{cite web |url=http://www.rokuda-noboru.com/ |title=CLIMBER'S STUDIO |publisher=六田登的官方個人網站 |accessdate=2018-06-15 |language=ja }}</ref>。來到東京之後，當初主要負責旅遊指南版面和漫畫單行本之企劃與構成工作。在那之後，筆名'''{{lang|ja|ろくだのぼ～る}}'''（將漢字轉化成[[平假名|平假名]]）和{{link-ja|御廚哲美|御厨さと美}}2人一起在《{{link-ja|小學館學年別學習雜誌|小学館の学年別学習雑誌|小學六年生}}》（[[小學館|小學館]]）雜誌中的讀者版面單元「{{lang|ja|ハロー6ワイドショー}}」踴躍發表創作。

1978年，六田以短篇「最終測驗（<small>{{lang-ja|最終テスト}}</small>）」入圍第1回{{link-ja|小學館新人漫畫大賞|小学館新人コミック大賞|小學館新人漫畫大獎}}佳作，從此以漫畫家身份正式出道。並開始在少年漫畫雜誌《[[週刊少年Sunday|週刊少年Sunday]]》連載過《{{link-ja|衝鋒勝平|ダッシュ勝平}}》及《{{lang|ja|その名もあがろう}}》等許多少年路線的搞笑漫畫。

1985年，六田轉往青年漫畫雜誌《[[Big_Comic_Spirits|Big Comic Spirits]]》發表以懸疑為題材的「風炎」之後，全面在青年漫畫雜誌持續發表。

六田是出道至今滿40年的現役漫畫家。在六田的漫畫作品中，常見主角即使不斷受到打擊和挫折、走投無路時，卻抱著必死決心追逐著重要目標是他作品的特徵。

六田除了畫漫畫之外，還有出過繪本。

1991年，《{{link-ja|極速悍將|F (漫画)}}》入圍第36回[[小學館漫畫賞|小學館漫畫獎]]青年部門。

== 年譜 ==
* 1952年：在[[大阪府|大阪府]][[八尾市|八尾市]]出生。
* 1978年：以短篇「最終{{lang|ja|テスト}}」出道。
* 1979年：開始連載《{{link-ja|衝鋒勝平|ダッシュ勝平}}》。
* 1981年：《衝鋒勝平》電視動畫化（製作：[[龍之子製作公司|龍之子製作公司]]。播放電視台：[[富士電視網|富士電視網]]）。
* 1985年：開始連載《{{link-ja|極速悍將|F (漫画)}}》。
* 1988年：《極速悍將》電視動畫化（製作：[[Kitty_film|Kitty film]]／動畫實質製作：[[STUDIO_DEEN|STUDIO DEEN]]。播放電視台：富士電視網）。
* 1991年：《極速悍將》入圍第36屆[[小學館漫畫賞|小學館漫畫獎]]青年部門。
* 2002年：開始連載極速悍將續篇《{{link-ja|極速悍將 再生瑠璃|F REGENERATION 瑠璃}}》。
* 2009年：開始連載極速悍將外傳《{{link-ja|極速悍將 FINAL|F FINAL}}》。
