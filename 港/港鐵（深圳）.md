'''香港鐵路（深圳）有限公司'''（{{lang-en|'''MTR Corporation (Shenzhen) Limited}}'''），簡稱'''港鐵（深圳）'''，企業形象標誌為「'''{{lang|en|MTR}}'''」，是[[香港鐵路有限公司|香港鐵路有限公司]]（簡稱「港鐵公司」，其前身是[[地鐵有限公司|香港地下鐵路公司]]，2000年[[私有化|私有化]]後改名為[[地鐵有限公司|地鐵有限公司]]）全資擁有附屬公司，于2004年3月成立，負責開展[[深圳地鐵4號綫|深圳地鐵4號綫]]二期專案的建造工程、專案管理、物業發展以及全線開通後的車務營運工作。

== 概述 ==
[[File:Platform_of_Shenzhen_North_Station,_Long_Hua_Line.JPG|thumb]]
[[File:SZMetro_Shangmeilin.jpg|thumb]]
[[深圳地鐵|深圳地鐵]][[深圳地鐵4號綫|4號綫]]（龍華綫）由一期和二期組成，全長約20公里，是雙軌設計的市區鐵路，以港深邊界的福田口岸為起點，直達龍華中心區，共有15個車站。

4號綫一期長4.5公里，共有5個車站，由[[福田口岸站|福田口岸站]]至[[少年宮站|少年宮站]]，已于2004年12月通車。二期工程全長約15.8公里，包括大約5公里地下段和10.8公里地面段及高架段，共設有2個地下車站、1個地面車站和7個高架車站；已于2011年6月16日建成通車。

4號綫二期專案總投資額約為[[人民幣|人民幣]]58億元，是中國大陸國內採用[[BOT模式|BOT]]（Build-Operate-Transfer）建設營運轉移模式建設的地鐵專案，是[[港鐵公司|港鐵公司]]與內地合作發展軌道專案和深港跨界合作的重要里程碑。

== 歷史及發展 ==
*2004年3月，成立香港鐵路（深圳）有限公司。
*2009年1月，深圳地鐵4號綫[[特许经营|特许经营]]协议获中國[[发改委|发改委]]批复。
*2009年3月，4號綫《特许经营协议》正式签署。
*2010年7月1日，正式接管深圳地鐵4號綫一期的車務營運工作，並逐步开始进行一期和二期的设备与系统整合。
*2010年年底4號綫二期机电设备系统全面安装及测试，并实现轨通、电通。
*2011年6月16日，4號綫正式投入營運，[[民間興建營運後轉移模式|民間興建營運後轉移模式]]協議正式生效。
*2013年12月16日，港鐵在內地的第一個物業發展項目，位於龍華綫車廠上蓋的樓盤「[[薈港尊邸|天頌]]」舉行開工典禮。該項目是港鐵首次在香港以外地區獲得土地開發權，港鐵在香港實施多年的「鐵路加物業綜合發展經營」模式取得新突破。
*2014年2月13日，胡瑞華接任港鐵（深圳）總經理，劉焯民調任[[杭港地鐵|杭港地鐵]]公司出任總經理。

== 物業發展 ==
[[龍華車輛段|龍華車輛段]]一期項目名為[[薈港尊邸|天頌]]<ref>[http://www.tiarasz.com.cn/ 天頌官方網站]</ref>，由港鐵物業發展（深圳）有限公司發展，並由香港梁黃顧建築師事務所設計，佔地八萬九千平方米，規劃建築面積二十萬六千平方米，其中住宅建築面積十九萬平方米，而商業建築面積一萬平方米<ref>[http://sz.winshang.com/news-454802.html 赢商网-片区观察：盘点宝安中海锦城、港铁天颂等重点项目]</ref>。為提升住宅質素，港鐵特地舉辦雕塑比賽，勝出的作品會放在物業內展示<ref>[https://www.facebook.com/mtr.tiara 港鐵•天頌雕塑藝術大賞facebook群組]</ref>。為促銷樓盤，港鐵亦在[[東鐵綫|東鐵綫]]車廂及沿線車站刊登廣告。
