'''港珠澳大橋邊檢大樓'''（{{lang-pt|POSTO FRONTEIRIÇO DA PONTE HONG KONG-ZHUHAI-MACAU}}、{{lang-en|'''Hong Kong-Zhuhai-Macau Bridge Frontier Port'''}}），位於港珠澳大橋澳門邊檢大樓地面層的一個終點站，澳巴'''[[澳門巴士101X路線|101X]]'''路線和[[澳門新福利公共汽車有限公司|新福利]]'''[[澳門巴士102X路線|102X]]'''路線以該站為終點站。

== 沿革 ==
*本站於2018年10月24日，配合[[港珠澳大橋|港珠澳大橋]]啟用而投入運作。<ref>{{cite news |title=港珠澳大橋跨境通行政策資料 |url=http://www.dsat.gov.mo/bus/site/bus_news_detail.aspx?a_id=810E3F195D387B084CEB7BC9EB40490E |publisher=[[交通事務局|交通事務局]] |date=2018-10-19}}</ref>

== 歷史 ==

== 設施 ==
*候車亭。

== 各車道安排 ==
所有路線先在一樓離境大堂清客，再返回地面層上客。其中地面層採用鋸齒型設計。

* '''港珠澳大橋邊檢大樓''' ↺ [[亞馬喇前地_(巴士總站)|亞馬喇前地]]
