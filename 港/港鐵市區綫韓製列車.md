'''港鐵市區綫韓製列車'''（{{lang-en|K-Train}}），乃[[港鐵公司|港鐵公司]]所擁有的[[直流電|直流電]]傳動[[通勤型電動列車|通勤型列車]]。該款列車可通行於港鐵市區四綫{{NoteTag|包括[[觀塘綫|觀塘綫]]、[[荃灣綫|荃灣綫]]、[[港島綫|港島綫]]及[[將軍澳綫|將軍澳綫]]}}，於2002年投入服務時行走[[港鐵|港鐵]][[觀塘綫|觀塘綫]]，於2010年全部調往行走[[將軍澳綫|將軍澳綫]]。

== 簡述 ==
這款電動列車由南韓[[Rotem|Rotem]]生產（原名'''Ko'''rea '''Ro'''lling '''S'''tock Corporation，[[縮寫|縮寫]]：KOROS，2002年改稱Rotem（全名為'''R'''ailr'''o'''ading '''T'''echnology Syst'''em'''），2007年因被收購而改稱現代Rotem（又稱現代樂鐵），是南韓現代汽車集團成員之一），採用交流牽引電動機及嵌入式車門設計。另外，該類列車的拖卡位於頭尾兩端，並非夾於電動車組的中間。隔音效果比[[港鐵現代化列車|都城嘉慕列車]]和[[港鐵Adtranz-CAF電動列車|機場鐵路列車]]良好，性能也較佳。

== 歷史 ==
1998年，南韓[[Rotem|Rotem]]（當時稱為KOROS）-[[三菱|三菱]]財團投得[[將軍澳綫|將軍澳綫]]列車訂單（合約編號：TKE-C651，Tseung Kwan O Extension-Contract 651），並於2001年開始生產[[將軍澳綫|將軍澳綫]]列車，總數共13列8卡列車，（即104個車卡）。列車原本計劃於[[將軍澳綫|將軍澳綫]]行走，但後來發現新列車不能即時適應兩套新舊列車系統<ref>《未配合地鐵訊號 影響逾8萬客 韓製列車「慢行」 恐礙[[將軍澳線|將軍澳線]]》，原載《[[香港經濟日報|香港經濟日報]]》。2002年8月9日，第A03版（要聞）。</ref>，為保將軍澳綫能如期通車，地鐵最終安排13列該款列車行走[[觀塘綫|觀塘綫]]，同時地鐵重組大部分[[港鐵都城嘉慕電動列車（直流電）|現代化列車]]的車卡排列，把大部分於1988年至1989年期間從英國都城嘉慕（G-stock）和於1994年至1998年期間購入的GEC Alsthom所製造的車卡（H-stock），調動至將軍澳綫行走（拖卡除外）。[[將軍澳綫|將軍澳綫]]擁有多個急彎，如採用現代化列車行走車廂之內將有極高噪音，故特別訂製隔音明顯較佳的韓製列車，當時並無預計發生信號兼容問題。

於是有關列車在2003年至2004年初期間亦曾經被安排行走[[荃灣綫|荃灣綫]]、[[港島綫|港島綫]]，不過因維修問題而主力行走[[觀塘綫|觀塘綫]]，要到2010年4月8日才在[[將軍澳綫|將軍澳綫]]投入服務，以解決[[將軍澳綫|將軍澳綫]]的噪音問題。於2012年10月，亦曾再次前往[[荃灣綫|荃灣綫]]、[[港島綫|港島綫]]試車。

[[港鐵|港鐵]]為了解決將軍澳綫噪音問題及配合[[康城站_(香港)|康城站]]的啟用，原定於2009年起，將13列觀塘綫韓製列車列車調往[[將軍澳綫|將軍澳綫]]繼續服務，可是該類列車仍然未能完全適應兩套新舊列車系統<ref>「原訂於新站開通前調往將軍澳線服務的韓製列車，由於近日該線經常誤點，亦不能在油塘及坑口站準確停車，因此暫時全數返回觀塘線至八月」，[http://www.ushb.net/ 特區聯合交通網]，2009年7月12日查閱</ref>，為確保7月20日成功實施新的列車排序及7月26日[[康城站_(香港)|康城站]]能如期通車，最終使韓製列車繼續行走觀塘綫。列車訊號系統經過改良後，2009年8月，逢星期日會有兩列韓製列車行走將軍澳線。2010年4月8日，因應[[將軍澳車廠|將軍澳車廠]]外判商[[澳洲|澳洲]]的{{Link-en|United Goninan|United Goninan}}撤出，將基地分別搬至[[九龍灣車廠|九龍灣車廠]]和[[輕鐵車廠|輕鐵車廠]]，前者負責1988年至1989年期間從英國都城嘉慕（G-stock）和於1994年至1998年期間購入的GEC Alsthom所製造的車卡（H-stock）的日常維修保養，而後者配合輕鐵Comeng輕鐵列車翻新工程，因此2010年4月8日當天起[[將軍澳車廠|將軍澳車廠]]的[[港鐵都城嘉慕電動列車_(直流電)|現代化列車]]與[[九龍灣車廠|九龍灣車廠]]的韓製列車對調。同日起，13列韓製列車正式調往[[將軍澳綫|將軍澳綫]]行駛（但本批列車因車身出現生銹跡，故港鐵於調至[[將軍澳綫|將軍澳綫]]不久便進行車身洗酸工程），而[[觀塘綫|觀塘綫]]回復全綫均以[[港鐵都城嘉慕電動列車_(直流電)|現代化列車]]行駛，直至[[港鐵北車長客電動列車|中國製列車]]於2011年12月7日投入服務。

儘管所有韓製列車調往將軍澳綫行走，由於將軍澳車廠並未提供全面的維修服務，以停泊為主要功能，故列車在部份維修工作進行時需返回觀塘綫的九龍灣車廠進行，而維修或定期檢查工作，多安排在星期六及日進行。

2013年12月16日，由於[[將軍澳綫|將軍澳綫]]發生嚴重供電事故，令將軍澳綫停駛五小時，其中一列列車（A311/A312，當天在將軍澳綫行走車序025）被困過海路段，但因為困在受影響的供電區以外，重開過海路段並駛至[[北角站|北角站]]調頭後，直接經[[藍田站後備行車隧道|藍田站後備行車隧道]]進入[[藍田站|藍田站]]並改以[[油麻地站|油麻地]]為終點站作臨時運作，令該款列車相隔接近3年後再次在[[觀塘綫|觀塘綫]]載客，亦是首次以上下行方向倒轉於觀塘綫行走（即上行頭卡A312向[[油麻地站|油麻地]]方向；正常情況應該是下行頭卡即A311向[[油麻地站|油麻地]]方向）<ref>[//www.youtube.com/watch?v=kRLFwABs9KY 港鐵KTL K-Train (C311)北角至彩虹行車片段 (調景嶺供電故障)]</ref>。

2014年3月12日，一列韓製列車（A319/A320）於中午時分不載客在[[荃灣綫|荃灣綫]]上試車，為該款列車於[[將軍澳綫|將軍澳綫]]供電事故後首次於[[觀塘綫|觀塘綫]]及[[將軍澳綫|將軍澳綫]]以外路綫行走<ref>{{cite web|url=https://www.youtube.com/watch?v=9mhIUiiEE5Q|title=【**異曲同工】港鐵 TWL K-Train (A320/A319) 不載客駛經旺角站一台|first=|last=mtrA381382|date=2014-03-12|publisher=|accessdate=2018-10-27|via=YouTube}}</ref>；2015年6月韓製列車到[[港島綫西延|港島綫西延]]試車。

2018年4月起，所有列車車門編號貼紙換成接近九鐵款式。（已全數更換）

2018年8月，港鐵為A321/322試驗性安裝車載軌道檢測系統、車輪環和減噪音液體瓶；隨後將陸續在各編組加裝。

== 特點 ==
這款列車基本設計參考了西班牙製列車（Adtranz-CAF）而作出改良，例如鋁合金車身改成不銹鋼車身，將無動力車卡設於首節和尾節車輛；而同期[[九廣鐵路公司|九鐵]]引進的[[SP1900|SP1900]]列車，亦是首次採用類似配置。

因不銹鋼較地鐵列車傳統使用的鋁材為重，列車內牆、車門及機櫃皆改用一般用於飛機上的蜂窩板製造，不僅較傳統物料輕巧，更具防火及隔音之作用，高鐵列車（包括新幹線）皆已採用。列車設計着重減低噪音，除蜂窩板外，是款列車更特別採用懸浮式地板，安裝在橡膠軟墊上，有助吸收列車與路軌摩擦產生之聲浪。其靜音器和變速箱通過電腦模擬測試，以便取得最寧靜的效果。

;車門設計
這款列車與[[港鐵Adtranz-CAF電動列車|西班牙製列車]]相似，採用了由奧地利iFE Doors（Knorr-Bremse子公司）生產的[[嵌入式車門|嵌入式車門]]，不過嵌入時是用氣壓直向和緊密地把車門吸回關門的位置，與車身間之門隙比滑動式車門和[[Adtranz-CAF電動列車|西班牙製列車]]細小。隔音效果比[[都城嘉慕電動列車_(直流電)|現代化列車]]和[[Adtranz-CAF電動列車|西班牙製列車]]良好。特別限制了滑動式車門因車門與地台間空隙傳入的噪音。不過，正因這關門方式與[[Adtranz-CAF電動列車|機場鐵路列車]]一樣但不像機場鐵路列車留有一半隻手部空間的縫隙，而是強力地壓回車身，如果手部放在車身與車門重疊位置便會被車門夾到；2002年投入服務初期曾夾到多名乘客。

;車外設計
列車的工業設計和[[港鐵東涌綫韓製列車|港鐵東涌綫韓製列車]]一樣由英國Design Triangle負責。

;車廂設計
這款列車的內部設計類似東涌綫[[港鐵Adtranz-CAF電動列車|西班牙製列車]]。列車的玻璃由不同廠商製造。窗口玻璃由法國聖戈班Sekurit製造，車門玻璃主要由澳洲DMS Glass製造，座椅兩側的屏風玻璃由南韓Samwon公司負責製造。列車貫通道由瑞典Dellner Couplers AB提供，本型號列車的車廂與車廂之間的連貫通道較其他市區綫列車為寛敞。而車廂的旁板為蜂窩夾層式玻璃纖維面板，由南韓HANKUK公司負責製造。

主要機器配置
列車的牽引系統為三菱電機製造的IGBT-VVVF。三菱電機同時為這款列車提供牽引電動機、輔助逆變器（SIV）及車管理系統（TMS）。德國Knorr-Bremse（克諾爾）則負責提供制動系統及空氣壓縮機。

;其他
這款列車首次採用澳洲SIGMA(現稱Sigma Coachair)製造的RPR44MX1型空調機組，緊急逃生斜道由法國BARAT group提供。牽引電機由[[日本|日本]][[三菱電機|三菱電機]]提供。列車的廣播系統、車內揚聲器、列車緊急通話器皆由日本東亞特殊電機供應。列車採用的BSI車鈎由瑞典SAB-WABCO製造。

== 問題 ==
===車門及門膠===
於投入服務初期，車門關上速度比較快，車門合上後隨即向內推<ref>[https://www.youtube.com/watch?v=LYxZku1W4fg&t=130s]|2:10-2:13</ref>，而非現時的關上一半、等待一兩秒之後再合上之後再內推。因此，車門夾到乘客的情況時有出現。經過[[立法會|立法會]]討論後，[[地鐵公司|地鐵公司]]更換了新的車門軟件及控制晶片，更換後車門關閉比較以前慢，而東涌綫韓製列車則於投入服務時已經採用了新的車門控制組件。由於車門控制方式存儲於晶片當中，不能夠隨意更改，故此導致現時亦只能夠以比較慢的速度關門。針對此缺點，港鐵公司其後購入的中國製列車已經改回採用傳統的外置滑動式車門。

另外，港鐵公司亦曾經更換門膠；韓製列車的感應元件不敏感，於2012年6月8日，一列韓製列車在[[鰂魚涌站|鰂魚涌站]]夾到乘客的公事包後，仍然可以夾物開行。<ref>{{cite news|url=http://orientaldaily.on.cc/cnt/news/20120609/00176_014.html|title=車門咬公事包 港鐵男(火羅)爆|newspaper=東方日報|date=2012-06-09|accessdate=2012-06-09}}</ref>

== 未來發展 ==
為節省電源，港鐵已於2013年或之前，為前地鐵路線（不包括[[迪士尼綫|迪士尼綫]]）共136列列車，更換至採用[[發光二極體|發光二極體]]（LED）照明。<ref>{{cite news|author=港鐵|title=港鐵進一步擴展「LED」環保燈光節能計劃|url=http://www.mtr.com.hk/chi/community/cor/pr_LED%20Lighting_Chinese.pdf|accessdate=2012-12-25|date=2010-02-10}}</ref>未來會逐步提升現時所有[[將軍澳綫|將軍澳綫]]的Rotem列車的車廂設備，包括更換乘客資訊系統（如增設[[港鐵車廂電視|港鐵車廂電視]]）、採用新款吊環、增設動態路線圖、更換橄攬型扶手、安裝車載軌道檢測系統、車輪環、減噪音液體瓶及[[閉路電視|閉路電視]]。
