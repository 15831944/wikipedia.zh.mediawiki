==背景==
台灣、香港、澳門居民在[[中國大陸|中國大陸]]的身分證件只有[[台湾居民来往大陆通行证|台湾居民来往大陆通行证]]（即[[台胞证|台胞证]]）或是[[港澳居民來往內地通行證|港澳居民來往內地通行證]]（即[[回乡证|回乡证]]），然而這些證件在大陸進办理网上购票、自助取票、旅馆入住、金融业务等受到限制，因此[[中華人民共和國政府|中華人民共和國政府]]決定推出港澳台居民居住证，让港澳台居民能够享受到与中國大陸居民基本相同的公共服务和便利<ref name="aaa">{{cite web|url=http://www.chinanews.com/gn/2018/08-16/8601469.shtml|title=《港澳台居民居住证申领发放办法》将印发 9月起实施|accessdate=2018-08-17|author=罗攀|date=2018-08-16|publisher=中国新闻网|language=zh-cn|archiveurl=https://web.archive.org/web/20180831174833/http://www.chinanews.com/gn/2018/08-16/8601469.shtml|archivedate=2018-08-31}}</ref>。

==歷史==
{{维基文库|港澳台居民居住证申领发放办法}}
2018年8月16日上午10時，[[中國國務院|中國國務院]]舉辦新聞發佈會，[[中華人民共和國公安部|公安部]]副部長[[侍俊|侍俊]]、[[國台辦|國台辦]]副主任[[龙明彪|龍明彪]]、[[港澳辦|港澳辦]]副主任[[黄柳权|黃柳權]]共同出席。侍俊表示，[[国务院办公厅|國務院辦公廳]]將印發《港澳台居民居住證申領發放辦法》，並於9月1日正式實施<ref>{{cite web |language = zh-tw |url = https://www.ettoday.net/news/20180816/1236534.htm#ixzz5OR7RonuC |title = 不斷更新／20日即可領！台胞大陸居住證「採18碼」9月1日實施　 |date = 2018-08-16　 |accessdate = 2018-08-17 |publisher = ETtoday新聞雲 |archiveurl = https://web.archive.org/web/20180817130846/https://www.ettoday.net/news/20180816/1236534.htm#ixzz5OR7RonuC |archive-date = 2018-08-17 |dead-url = no }}</ref>。

2018年8月19日，国务院办公厅正式公佈《港澳台居民居住證申領發放辦法》。<ref name=居住证申领发放办法/>

自2018年8月26日開始，各省公安廳、各市公安局陸續發佈港澳台居民居住證受理點數量以及位置。至31日晚，公安部公布各地共设立港澳台居民申领居住证受理点6,572个<ref>{{cite web |language = zh-cn |url = http://www.chinanews.com/ga/2018/08-31/8615827.shtml |title = 公安部：各地共设立港澳台居民申领居住证受理点6572个 |publisher = 中国新闻网 |archivedate = 2018-08-31 |date = 2018-08-31　 |accessdate = 2018-08-31 |archiveurl = https://web.archive.org/web/20180831211929/http://www.chinanews.com/ga/2018/08-31/8615827.shtml |dead-url = no }}</ref>。

==樣式==
[[File:Hong_Kong,_Macao_and_Taiwan_residents'_residence_permit.jpg|thumb]]
港澳臺居民居住證樣式不同于中國大陸各地份颁发的[[居住证|居住证]]，而與[[中华人民共和国居民身份证|中华人民共和国居民身份证]]十分雷同，但缺少「[[中國民族列表|民族]]」一欄<ref>{{cite web |language = zh-hk |url = http://www1.hkej.com/dailynews/cntw/article/1924338/%E6%B8%AF%E6%BE%B3%E5%8F%B0%E5%B1%85%E6%B0%91%E5%B1%85%E4%BD%8F%E8%AD%89%E4%B8%8D%E6%A8%99%E6%B0%91%E6%97%8F|title = 港澳台居民居住證不標民族|publisher=信報財經新聞|archivedate=  |date =  2018-09-01　|accessdate = 2018-09-05 |archiveurl= }}</ref>。

為了方便港澳台居民與中國大陸對接，港澳台居民居住證結合居民身分證技術，確保整個中國大陸地區的各種身分識別系統設備都能識讀。<ref>{{cite web |language = zh-tw |url = https://news.tvbs.com.tw/world/974672 |title = 等同身分證　陸推18碼台灣居民居住證 |publisher = TVBS |archivedate = 2018-08-26 |date = 2018-08-16　 |accessdate = 2018-08-18 |archiveurl = https://web.archive.org/web/20180826014909/https://news.tvbs.com.tw/world/974672 |dead-url = no }}</ref>

* 證件正面：在彩虹印刷花紋和[[長城|長城]]的襯托下，[[中華人民共和國國徽|中華人民共和國國徽]]位於證件的左上角突出位置，證件名稱分兩行排列於國徽右側證件上方。
* 證件背面：在彩虹印刷花紋襯托下登載持證人身分信息。<ref>{{cite web|url=https://udn.com/news/story/7331/3340196|title=台灣居民居住證樣式出爐 跟大陸身分證超像|accessdate=2018-09-05|author=許依晨|date=2018-08-30|publisher=udn|language=zh-tw|archiveurl=https://web.archive.org/web/20180831174235/https://udn.com/news/story/7331/3340196|archivedate=2018-08-31}}</ref>

根據港澳台居民居住證申領發放辦法第七條，港澳台居民居住證的式樣，由[[中華人民共和國公安部|公安部]]商[[國務院港澳事務辦公室|國務院港澳事務辦公室]]、[[國台辦|國台辦]]制定。<ref name=居住证申领发放办法/>

港澳台居民居住证的內容有以下：
* 背面
** '''姓名'''
** '''性别'''（“男”或“女”）
** '''出生年月日'''（格式：“╳╳╳╳年╳月╳日”）
** '''住址'''（即居住地住址）
** 18位'''[[中華人民共和國公民身份號碼|公民身份号码]]'''
* 正面
** '''签发机关'''（居住地所处县级人民政府公安机关）
** 证件的'''有效期限'''（格式：“20╳╳年╳月╳日-20╳╳年╳月╳日”）
**'''签发次数'''（于'''有效期限'''右侧登列）
**'''通行证号码'''（港澳居民为9位，台湾居民为8位）

此外，于背面右侧印有持证人的彩色免冠照片。<ref name=居住证樣式>{{cite web |language = zh-cn |url = http://news.cctv.com/2018/08/30/ARTIC2zjoSIAINjA0s8Ur9q7180830.shtml |title = 北京市公安局：9月1日起港澳台居民可在京办理居住证|publisher=央视网|archivedate= 2018-08-31 |date =  2018-08-30　|accessdate = 2018-08-30 |archiveurl=https://web.archive.org/web/20180831174604/http://news.cctv.com/2018/08/30/ARTIC2zjoSIAINjA0s8Ur9q7180830.shtml }}</ref>

==功能==
港澳台居民居住证因採用身份證相同製作技術，在诸多方面跟第二代[[中华人民共和国居民身份证|中华人民共和国身份证]]具备相同功能，但不能作為入出境中国大陆的證件。入出境必须使用相应的旅行证件，港澳居民须申领[[回乡证|回乡证]]，台湾居民则须申领[[台胞证|台胞证]]。<ref>{{cite web |language = zh-cn |url = http://news.cctv.com/2018/08/16/ARTI27VTkJXUb0Yh33CzCQTh180816.shtml |title = 港澳台居民居住证如何申领？享哪些便利？权威回应|publisher=央視網|archivedate= 2018-08-31 |author=中国新闻网 |date =  2018-08-16　|accessdate = 2018-08-17 |archiveurl=https://web.archive.org/web/20180831175703/http://news.cctv.com/2018/08/16/ARTI27VTkJXUb0Yh33CzCQTh180816.shtml }}</ref>
