'''港鐵'''[[東鐵綫|東鐵綫]]在2010年6月15日前曾經兼營貨運服務<ref>{{cite news|author=港鐵|title=港鐵停辦貨運 提升客運服務|url=http://www.mtr.com.hk/eng/corporate/file_rep/PR-09-122-C.pdf|accessdate=2012-08-29|date=2009-10-29}}</ref>，因此東鐵綫沿線設有多個貨場，供處理貨運列車的始發、維修貨運列車（及機車）等。

== 紅磡貨場 ==
[[File:Hung_Hom_Seaside_pier_2012.jpg|250px]]
紅磡貨場位於[[紅磡站|紅磡站]]東面地面，內有香港郵政[[國際郵件中心|國際郵件中心]]，上蓋為[[香港體育館|香港體育館]]，後增建九鐵與長江實業合作地產項目[[國際都會|國際都會]]，大部份均為有蓋地方，但兩端仍設有露天部分，包括一個可供船隻停泊的碼頭。貨場裝有吊機，但用了一年即停用<ref>{{cite news|title=九鐵廢吊機浪費二千一萬|url=http://orientaldaily.on.cc/archive/20060321/new/new_a66cnt.html|accessdate=2012-08-29|newspaper=東方日報|date=2006-03-21}}</ref>。除供貨運之外，亦設有[[港鐵車廠#機車行車室|機車行車室]]，供柴油機車進行清潔、入油等工作；另外，亦設有電子工場。停止使用之前，曾經有東鐵綫列車於此停泊過夜，而每日的尾班[[廣九直通車|廣九直通車]]客車亦會暫泊於此。自港鐵於2010年停止貨運服務後，紅磡貨場只用作東鐵綫列車和機車停泊之用。

在2000年代，部份[[SP1900/1950型電動列車|千九列車]]亦於此處上岸。2011年7月起，為配合[[觀塘線延線|觀塘線延線]]及[[沙田至中環線|沙田至中環線]]工程，南面的大部份露天地方均被圍封，並於10月起興建紅磡躉船轉運站。

隨著[[沙田至中環綫|沙田至中環綫]]計劃的開展，本貨場於2012年7月16日正式關閉<ref>http://www.hkitalk.net/HKiTalk2/redirect.php?tid=698987&goto=lastpost#lastpost</ref>，將會改建成[[屯馬綫|屯馬綫]]的紅磡站新月台及車廠。<ref>[http://www.hyd.gov.hk/chi/road_and_railway/railway_projects/scl/gazettal/scheme/doc/gn7326/gl/SCL-G27B_Web.pdf 沙中綫圖則] - 路政署</ref>與此同時，廣九直通車改往[[港鐵何東樓車廠|港鐵何東樓車廠]]過夜。

== 何文田貨場 ==
何文田貨場，亦稱[[上水屠房#歷史|何文田牲口站]]，已建成[[香港理工大學#主校園|香港理工大學]]第八期校園（Z座）。原址位於東鐵綫[[旺角東站|旺角東站]]與[[紅磡站|紅磡站]]之間的隧道入口旁，前身為通往九廣鐵路英段[[尖沙咀站_(九廣鐵路英段)|尖沙咀站]]的正線，後來改以[[紅磡站|紅磡站]]為南行總站，正線變成側線，並成為貨場。而在沙田至中環線工程後，此貨場將成為紅磡列車停放處之附設機務段。

何文田貨場僅與北行路軌連接，需要前往紅磡貨場的列車需要經[[旺角東站|旺角東站]]調頭。

== 旺角貨場 ==
[[File:Mong_Kok_Freight_Terminal_entry.JPG|thumb]]
旺角貨場，位於東鐵綫[[旺角東站|旺角東站]]側（[[新世紀廣場|新世紀廣場]]地庫樓層），曾改為嘉士伯（香港）物流中心。在沙田至中環線工程開展後，此處已重新投入服務，成為機務段。

== 沙田貨場 ==
沙田貨場位於[[沙田站|沙田站]]1號月台旁（可以從月台清楚見到），九鐵年代原位於[[何東樓車廠|何東樓車廠]]的軌道工場因要興建[[御龍山|御龍山]]而曾被暫遷至沙田貨場，用作存放軌道、道碴和工程車等，直至2009年初軌道工場物資被遷回何東樓車廠，軌道工場亦被拆除。

== 火炭貨場 ==
[[File:Fotan_freight_yard.jpg|thumb]]]]
火炭貨場位於[[火炭站|火炭站]]1號月台旁，使用量偏低，以往只有運載危險品的貨卡（由中國內地抵港之石油罐車班次，目前已經終止）或抵港訪問的內地[[健康快車|健康快車]]使用。由於路軌設計缺陷，貨場僅連接北行路軌，貨運列車進入火炭貨場時有特定路線需要跟從。

[[港鐵東鐵綫現代列車|東鐵綫南北走廊新列車]]運到香港時，由[[葵青貨櫃碼頭|青衣九號貨櫃碼頭]]分三批（每日三卡）運到火炭貨場，並進行車卡併接、撕去保護膠紙及在車頭裝上港鐵標誌等工序，然後等待晚上停止服務後，由柴油機車牽引下，經北廠出入口運入何東樓車廠。而在沙田至中環線工程中，此貨場亦用於組裝東鐵線新訊號系統組件。

當[[港鐵中期翻新列車|港鐵中期翻新列車]]由2019年起退役時，本貨場將可能成為解體線（但最終解體線設於何處，仍未決定）；而在2017年11月22日凌晨，港鐵就在本貨場進行了一次舊車調離演習，由62及59號柴油機車牽引E92/E70列車進入擬設解體線，行駛路線與韓製新列車進廠相反。

== 上水屠房 ==
{{see also|上水屠房站}}
上水屠房站位於[[上水站_(香港)|上水站]]與[[羅湖站_(香港)|羅湖站]]之間，在牲口全面改由貨車運送之前，本站每天均有運載牲口的貨運列車停泊卸貨，但改用貨車運送之後，本站亦改為列車停放處（停車[[側線_(鐵路)|側線]]）。

== 羅湖貨場 ==
羅湖貨場位於[[羅湖站_(香港)|羅湖站]]以南，為邊境禁區範圍之內，為所有貨運列車必經之處，曾用作停泊[[港鐵伊藤忠近畿川崎列車|港鐵伊藤忠近畿川崎列車]]（KRS991批次，已於2008年調入[[西鐵綫|西鐵綫]]正式服務）；現時部分抵港[[港鐵東鐵綫現代列車|港鐵東鐵綫現代列車]]亦因車廠未有充足空間容納，而暫時停泊在此。
