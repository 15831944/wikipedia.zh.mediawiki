'''港墘假山'''是位于[[广东省|广东省]][[揭阳市|揭阳市]][[榕城区|榕城区]]进安街与天福东路交界的一座[[假山|假山]]，为1991年港墘村投资兴建<ref>{{cite web |url=http://www.jyrongcheng.gov.cn/View.aspx?q_aid=13017&ArticleType=Common&q_cid=7 |title=榕华街道打响天福东路拆迁攻坚战“第一炮” |last= |first= |date=2017-11-01 |website=揭阳市榕城区人民政府 |publisher= |access-date=2018-04-27 |quote=}}</ref>，曾经是揭阳市的[[地标|地标]]。

2017年10月25日上午9时，为扩建道路，时隔26年的港墘假山被拆除。<ref>{{cite web |url=http://www.sohu.com/a/200413432_168944 |title=岁月成空！拆不掉的虚线情怀，移不走的城心生活 |last= |first= |date=2017-10-26 |website=搜狐 |publisher= |access-date=2018-04-27 |quote=}}</ref><ref>{{cite web |url=http://www.jieyang.gd.cn/view.aspx?q_aid=283499 |title=榕城区社会各界众志成城推进“五路一河”项目 |last= |first= |date=2018-02-05 |website=揭阳市人民政府 |publisher= |access-date=2018-04-27 |quote=}}</ref>现在港墘假山的原位置是天福东路的一个路段<ref>{{cite web |url=http://jy.southcn.com/content/2019-01/24/content_184952640.htm |title=重磅！榕城区天福东路榕华段月底通车 |last= |first= |date=2019-01-24 |website=南方网 |publisher=揭阳日报 |access-date=2019-02-16 |quote=}}</ref>。
