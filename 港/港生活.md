'''港生活'''，亦稱為'''HK港生活'''，是香港一個'''網上香港生活資訊平台'''，主力發掘本地有趣玩樂活動好去處，為[[香港經濟日報集團有限公司|香港經濟日報集團有限公司]]<ref name="香港經濟日報集團有限公司">{{cite web  |url= http://www.hket.com/eti/info/group/  |title= 香港經濟日報集團 成員網站  |publisher= [[香港經濟日報集團|香港經濟日報集團]]  |deadurl= yes  |archiveurl= https://web.archive.org/web/20140706181002/http://www.hket.com/eti/info/group/  |archivedate= 2014-07-06  }}</ref>旗下[[U_Lifestyle|U Lifestyle]]<ref name="U Lifestyle網站 關於我們">{{cite web|url= http://www.ulifestyle.com.hk/aboutus  |title= U Lifestyle網站 關於我們 | publisher= [[U_Lifestyle|U Lifestyle]]網站  }}</ref>網站成員之一。

該網站免費提供最新玩、買、食資訊等豐富內容，包括：最Hit玩樂、購物及飲食情報及香港周圍遊等等，旨在推廣香港獨有文化特質<ref name="香港觀察：香港的文化特質">{{cite web|url=http://www.bbc.co.uk/zhongwen/trad/indepth/2010/09/100928_hkreview_culture_by_lingongqin.shtml|title=BBC 中文网 - 香港觀察 - 香港觀察：香港的文化特質|publisher=[[BBC|BBC]] 中文网}}</ref>。

除桌面版網頁外，該網站設有流動版網頁及[[手機應用程式|手機應用程式]]，網民可於使用流動裝置如手提電話去瀏覽。

<br />

== 歷史 ==
* HK港生活於2014年9月25日正式開始運作。

== 網站分類 ==

*'''港活動'''
**香港本地[[活動|活動]] 資料介紹，如[[演唱會|演唱會]]、[[表演|表演]]、[[商場|商場]]、[[展覽|展覽]]及[[嘉年華|嘉年華]]活動
**用家可按活動類型性質、地區、日期等資料搜尋

*'''港地方'''
**香港本地熱門[[地方|地方]]資料介紹
**用家可按地區搜尋

*'''香港周圍遊'''
**香港本地遊覽推介路線，綜合各個[[消閒|消閒]]好去處
**用家可按活動類型性質、地區等資料搜尋路線

*'''港熱話'''
**主要圍繞香港本地熱門討論話題
**分類包括：[[社會|社會]]、[[八卦|八卦]]、[[生活|生活]]、[[科技|科技]]、[[娛樂|娛樂]]、新奇等
*'''會員專享'''
**主要提供各類會員活動讓會員自由參與
