{{日本作品|攻殼機動隊|GHOST IN THE SHELL / 攻殻機動隊|Ghost in the Shell }}是一部[[日本|日本]][[動畫|動畫]][[電影|電影]]，導演是[[押井守|押井守]]，劇情改編自[[士郎正宗|士郎正宗]][[漫畫|漫畫]]《[[攻殼機動隊|攻殼機動隊]]》。《攻殼機動隊》由[[Production_I.G|Production I.G]]公司製作，也是[[攻殼機動隊|攻殼機動隊]]系列首作。《攻殼機動隊》是日本动画史上的里程碑，也是[[押井守|押井守]]的代表作之一。

2008年，為了紀念押井的新作電影《[[空中殺手|空中殺手]]》上映，將本片以新裝版《攻殻機動隊2.0》方式重映。《2.0》除了將部分畫面以新作3DCG呈現外，並將整體畫面色調由原版的藍、綠色系，調整為與《攻殼機動隊2 INNOCENCE》相同的偏橙色系，另外並調整為環繞音場（5.1ch/6.1ch），部分配音陣容與對白、配樂、音效，均在美國的Skywalker Sound的協助下修改。

==劇情簡介==
2029年，[[公安九課_(攻殼機動隊)|公安九課]]在市容很像[[香港|香港]]的[[日本|日本]]城市「新港市」中執行各種秘密任務。九課因為不顧正當程序的自作主張，強硬行事，往往與政府其他部門起衝突，例如負責[[外交|外交]]的公安六課就與公安九課不合。另一方面，公安九課之中的重要人物，全身是義體的[[賽伯格|機械生化人]][[草薙素子|草薙素子]]，對於自己的現狀一直隱隱感到不滿。

有一天，公安九課因為一件[[外國|外國]]政要的刺殺未遂案而開始追查神秘的[[駭客|駭客]]「傀儡師」。傀儡師能透過網路入侵人腦，植入虛擬記憶，再利用記憶被竄改的對象來達成目的。雖然公安九課沒有成功追查到傀儡師，但是藏有傀儡師的「[[靈魂|靈魂]]」的機械生化人卻逃脫製造公司的控制，並因此來到九課。公安六課得知消息後，趕來綁走傀儡師。因為草薙素子在事前就推測出公安六課的綁架行動，促使公安九課成功攔截傀儡師，並發現傀儡師並不是人類，而是公安六課設計來操縱人類的[[電腦程式|電腦程式]]「2501計畫」。2501計畫在意外中產生自覺意識，決定逃出公安六課以求生存，並且要求和草薙素子的靈魂「融合」以求進一步發展。草薙素子最後在同事巴特的協助與掩護下完成融合並且脫離政府掌握，下落不明，其意識與整個網絡融合。

==评价==
《攻殼機動隊》於日本上映時，票房出色，也獲得廣泛的正面評價。《攻殼機動隊》在[[爛番茄|爛番茄]]網站獲得94%正面評價，被一致評論為是一部 「令人震驚的現代動畫禮讚，劇情具有深沉與複雜的論述」<ref>{{cite web|url=http://www.rottentomatoes.com/m/ghost_in_the_shell/ |title=Ghost in the Shell |publisher=Rotten Tomatoes |date= |accessdate=2013-10-30}}</ref>。

Twitch Film影評尼爾斯·馬蒂爾斯（Niels Matthijs）稱讚《攻殼機動隊》, 認為這部動畫「不僅是日本動畫的經典，並與[[史丹利·庫柏力克|史丹利·庫柏力克]]執導的《[[2001太空漫遊|2001太空漫遊]]》及[[安德烈·塔爾科夫斯基|安德烈·塔爾科夫斯基]]執導的《[[飛向太空|飛向太空]]》一樣，其電影成就都超過原著，就像賦予一個已經廣受歡迎樂團新的意義」，並名列為他最愛的電影名單第48名<ref>{{cite web| url = http://twitchfilm.com/2012/01/review-ghost-in-the-shell-personal-favorites-48.html| title = GHOST IN THE SHELL (Personal Favorites #48)| accessdate = 2012-12-15| date = 2006-01-31| work = Twitch Film| deadurl = yes| archiveurl = https://web.archive.org/web/20121227081611/http://twitchfilm.com/2012/01/review-ghost-in-the-shell-personal-favorites-48.html| archivedate = 2012-12-27}}</ref>。《[[帝國雜誌|帝國雜誌]]》編輯克拉克·柯林斯（Clark Collis）則稱讚《攻殼機動隊》的製作品質<ref>{{cite web | url = http://www.empireonline.com/reviews/reviewcomplete.asp?FID=2929 | title =  Ghost in the Shell | accessdate = 2012-10-20 | last = Collis | first = Clark | work = Empire Online}}</ref> 。[[動畫新聞網|動畫新聞網]]的喬納森·梅斯（Johnathan Mays）稱讚《攻殼機動隊》[[電腦|電腦]]特效結合，認為這部電影也許是動畫史上最出色的作品<ref>{{cite web | url = http://www.animenewsnetwork.com/review/ghost-in-the-shell/dvd | title =  Ghost in the Shell DVD | accessdate = 2012-10-29 | last = Mays | first = Johnathan | work = Anime News Network}}</ref> 。[[海倫·麥康尼|海倫·麥康尼]]（Helen McCarthy） 在《500部不可忽視的動畫電影》中形容《攻殼機動隊》是動畫史上最出色的作品之一<ref>Helen McCarthy. ''500 Essential Anime Movies: The Ultimate Guide''. — Harper Design, 2009. — P. 20-21. — 528 p. — ISBN 978-0061474507</ref>。

《攻殼機動隊》是第一部在發行當時就獲得《[[告示牌|告示牌]]》影音排行榜冠軍的動畫<ref name="special edition"/>，《攻殼機動隊》是2006年[[DVD|DVD]]銷售成績最出色的電影之一<ref>{{cite web | url = http://www.animenewsnetwork.com/news/2006-01-31/top-selling-anime-dvd-movies| title = Top Selling Anime DVD Movies| accessdate = 2012-12-05 | date = 2006-01-31 | work = Anime News Network}}</ref>。《攻殼機動隊》在[[Total_Film|Total Film]]50大動畫排行榜中名列第35名<ref>{{cite web|url=http://www.totalfilm.com/features/50-greatest-animated-movies |title=Top 50 Greatest Animated Movies |accessdate=2012-10-20 |date=2010-11-22 |work=Total Film |deadurl=yes |archiveurl=https://web.archive.org/web/20140523152706/http://www.totalfilm.com/features/50-greatest-animated-movies |archivedate=2014-05-23 }}</ref>。《攻殼機動隊》在巫師動畫雜誌50大北美動畫排行榜中名列第4名<ref>{{cite web | date=2001-07-06 | title=Wizard lists Top 50 Anime | url= http://www.animenewsnetwork.com/news/2001-07-16/wizard-lists-top-50-anime | publisher=[[Anime_News_Network|Anime News Network]] | accessdate=2014-02-02}}</ref>。

==影響==
