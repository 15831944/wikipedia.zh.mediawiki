'''國立苗栗高級商業職業學校'''，簡稱'''國立苗商'''、'''苗栗高商'''、'''苗商'''，位於[[苗栗縣|苗栗縣]][[苗栗市|苗栗市]]光復路口及忠貞路口，與[[國立苗栗高級中學|國立苗栗高級中學]]及[[私立建臺高級中學|私立建臺中學]]相鄰，交通方便，為苗栗地區著名的高級中等學校，也是全國知名優質商業技術型高中之一。

== 校訓 ==
*精----精敏幹練、精益求精。
*誠----誠懇有禮、誠實守信。
*勤----勤儉刻苦、勤勉奮發。
*樸----樸素無辜、樸厚踏實。

== 沿革 ==
*創設於民國四十七年，初時借用苗栗初中教室上課，校名為「苗栗縣立苗栗高級商業職業補習學校」。 
*民國六十二年八月一日，改制為「臺灣省立苗栗高級商業職業學校」。
*民國八十九年二月一日，奉命改制為「國立苗栗高級商業職業學校」。

===女籃隊===
苗栗高商女籃隊在84、85學年度拿到殿軍，是她們隊史最佳戰績。近年來常能在HBL的8強舞台上看到她們的身影，但多處於挨打的劣勢。屈居劣勢的主因，為資源分配不均，球隊招生相對困難。儘管如此，山城娘子軍還是有一套屬於自己的玩法。

相較於對冠軍野心勃勃的列強，苗栗高商女籃隊並非不想挑戰4強，只是選擇相較務實地以「保8」作為賽季目標，並以「戰力銜接」作為永續經營的方針。在晉級8強後，除了可以力搏搶勝的場次外，其他時間拿來磨練陣中學妹，為下一個賽季提早做安排。

105學年度，苗栗高商趁著衛冕軍[[淡水商工|淡水商工]]兵敗如山倒，在8強賽最後一場比賽擊敗對手，拿下隊史第1次第7名。

106學年度，苗栗高商持續推進戰績，拿下第5名，差一點就能前進[[台北小巨蛋|台北小巨蛋]]。

107學年度，苗栗高商在預賽以分組第3躲過打外卡戰的命運，並在8強賽前6戰取得3勝3負，但很可惜在第7戰以52：58不敵[[北一女中|北一女中]]，再度與小巨蛋緣慳一面。
