'''國際地理奧林匹亞'''（[[英文|英文]]：'''International Geography Olympiad'''，[[縮寫|縮寫]]：'''iGeO'''）是一個為16到19歲學生所舉辦的[[國際科學奧林匹亞|國際科學奧林匹亞]]，由各國國內奧林匹亞選出的代表隊參加。競賽內容分為三個部分：手寫題、多媒體測驗、和野外實地考察。國際地理奧林匹亞測驗參賽選手空間模式和過程的觀念，包括邊界、中心、環域的概念，對旅遊、[[恐怖份子|恐怖份子]]、難民的認知，對食物、衣服、藥品、數位資訊貿易的瞭解，對人口成長、[[聖嬰現象|聖嬰現象]]、[[海嘯|海嘯]]、地震的掌握。本競賽由[[國際地理聯合會|國際地理聯合會]]舉辦，並從2012年開始由每兩年舉辦改為每年舉辦一次。<ref>[http://www.geoolympiad.org iGeO的官方網站]</ref>

== 歷史 ==
1994年在[[布拉格|布拉格]]舉辦的國際地理聯合會中，[[波蘭|波蘭]]和[[荷蘭|荷蘭]]提出了給15到19歲學生參加國際地理奧林匹亞的理念，並於1996年在荷蘭[[海牙|海牙]]舉辦了第一次競賽，共計有5國參加。在2008年[[突尼西亞|突尼西亞]][[迦太基|迦太基]]的比賽後，參賽國增加到了24國；到了2010年[[中華民國|中華民國]][[台北|台北]]的第八屆賽事，參賽國再成長，並由29個政治實體組成，包含：[[澳大利亞|澳大利亞]]、[[白俄羅斯|白俄羅斯]]、[[比利時|比利時]]、[[巴西|巴西]]、[[保加利亞|保加利亞]]、[[中華人民共和國|中華人民共和國]]、[[香港|香港]]、[[克羅埃西亞|克羅埃西亞]]、[[捷克|捷克]]、[[丹麥|丹麥]]、[[德國|德國]]、[[愛沙尼亞|愛沙尼亞]]、[[芬蘭|芬蘭]]、[[匈牙利|匈牙利]]、[[日本|日本]]、[[拉脱維亞|拉脱維亞]]、[[立陶宛|立陶宛]]、[[墨西哥|墨西哥]]、[[荷蘭|荷蘭]]、[[紐西蘭|紐西蘭]]、[[奈及利亞|奈及利亞]]、[[波蘭|波蘭]]、[[羅馬尼亞|羅馬尼亞]]、[[俄羅斯|俄羅斯]]、[[斯洛伐克|斯洛伐克]]、[[斯洛維尼亞|斯洛維尼亞]]、[[中華民國|中華民國]]（主辦國）、[[突尼西亞|突尼西亞]]、[[英國|英國]]。

== 目標 ==
國際地理奧林匹亞的目標包括：
* 推廣地理學。
* 刺激年輕學子彼此的瞭解。
* 刺激世界各國更高品質的地理教育。

==賽制==
每支代表隊包含最多4名參賽者，2名領隊，及若干名觀察員。其中參賽者年齡不可超過19歲(以當年7月1日為準)，且必須就讀於中學校(Secondary School)。而領隊必須擁有地理方面的專業知識及足夠的英文能力，以將題目翻譯成該國母語。

比賽分為筆試、田野考察測驗及多媒體測驗三部分，分別佔總分的40%、40%及20%。

參賽者會依其個人成績排先後次序，約一半參賽者會獲得獎牌。'''金牌'''、'''銀牌'''、'''銅牌'''的比例約為'''1：2：3'''。

英文為比賽的官方語言，故所有參賽者必須以英文作答試題。如參賽者的母語並非英文，他們可使用雙語詞典以助作答。

比賽題目環繞以下12個議題：
# 氣候及氣候變化
# 危害及危害管理
# 資源及資源管理
# 環境地理學及可持續發展
# 地貌，景觀及土地利用
# 農業地理學及糧食問題
# 人口及人口變化
# 經濟地理學及全球化
# 發展地理學及空間不平等
# 城市地理學，市區更新及都市規劃
# 旅遊業及旅遊管理
# 文化地理學及區域認同
