'''國民大學'''（[[諺文|諺文]]：'''국민대학교'''），創立於1946年9月，位於[[首爾特別市|首爾特別市]][[城北區|城北區]][[貞陵洞|貞陵洞]]設置校園，是一所綜合型私立大學。

== 簡介 ==
'''國民大學'''象徵理念為以校為家，學生人數為15,162人（''這是2018年的數據''），大學院人數為3,020人（''這是2018的數據''）。1946年國民大學院設立，1948年升格為國民大學，1971年遷移到漢城（今首爾）[[城北區|城北區]][[貞陵洞|貞陵洞]]校區，1976年設置了大學院碩士課程，1979年設置了大學院博士課程，1981年3月改編為大學，同時設置了教育大學院，1982年設置了經營大學院，1984年設置了行政大學院，1998年3月總共設置了10個單科大学、14個學部、35個専攻、18個學科，在校生有8826人，教授群有245人。

[[File:Kookmin3.jpg|thumb]]

== 特色专业 ==
视觉设计，汽车工学

== 强势专业 ==
机械汽车工学（韩国国内首位）
工业设计系，视觉设计系，陶瓷工艺系，金属工艺系，服装设计系，室内设计系，影像设计系
