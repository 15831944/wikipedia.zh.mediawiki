'''国际回信券'''（{{lang-en|International Reply Coupon, IRC}}）是由[[万国邮联|万国邮联]]发行的一种邮资凭据。

国际回信券服务于1906年万国邮联罗马会议上被正式提出。理论上，国际回信券可在任何一个万国邮联成员国兑换为等价于国际优先函件的最低邮资<ref>[http://www.upu.int/en/activities/international-reply-coupons/about-international-reply-coupons.html 万国邮联关于国际回信券的介绍页面（英文）]</ref>。

== 用途 ==
在写信给陌生人并期待其回信时，在[[信封|信封]]中附上回信邮资是一种通行的礼节。对于国内函件而言，通行的做法是在信内附上[[邮票|邮票]]；然而对于国际函件，本国邮票未必会被收件人所在国家承认，这便是国际回信券试图解决的问题。国际回信券並非郵票，但卻可用來向[[郵局|郵局]]兌換郵票，以方便不同國家的民眾，來和外國通信的一項郵政產物。

如果希望身处国外的收件人予以回信，但又不愿因国际邮资而增加其负担，或者出于礼节，可以将国际回信券一并寄出。收件人即可凭国际回信券回信而不用额外支付国际邮资。但在实际操作中，某些国家的邮局在兑换过程中需要额外补缴一定费用，如[[芬兰|芬兰]]<ref>{{Cite web|url=https://www.posti.fi/private/send/stamps.html|title=Stamps - Posti for Private Customers|accessdate=2018-07-27|work=www.posti.fi|language=en}}</ref>等国。

==券面设计==
在国际回信券的下一个发行周期到来之前，万国邮政联盟会征集下一版国际回信券的券面设计。此时，万国邮联会公布所需主题、截止日期及获选奖励。被选用的作品的设计者一般会得到10,000张国际回信券作为奖励。<ref>{{Cite web |url = http://www.upu.int/uploads/tx_sbdownloader/rulesOfTheInternationalReplyCouponDesignCompetitionEn.pdf |title = 2017-2021版国际回信券设计竞赛规则 |date = 2016-02-25 |language = en |accessdate = 2018-07-18}}</ref>

== 票据价值 ==
2013年万国邮联函件业务细则（{{lang-en|The Letter Post Regulations}}）中规定，国际回邮券的计价货币为[[国际货币基金组织|国际货币基金组织]][[特别提款权|特别提款权]]。每份国际回邮券的价值等于0.74份特别提款权<ref>{{Cite web |url=http://www.rrt.lt/download/19052/actregulationsletterpostfinalprotocolen(2013).pdf |title=万国邮联函件业务细则（英文） |access-date=2018-05-15 |archive-url=https://web.archive.org/web/20180516103151/http://www.rrt.lt/download/19052/actregulationsletterpostfinalprotocolen(2013).pdf |archive-date=2018-05-16 |dead-url=yes }}</ref>。

万国邮联的成员国可以向万国邮联以0.74特别提款权双向兑换国际回邮券，国际回邮券在成员国国内的销售办法由成员国自行决定，但销售价格不应低于0.74份特别提款权。

根據中華民國《[[中華郵政|郵政法]]》第一章第4條第9款之名詞定義，國際回信郵票券是一種「有價證券」<ref>{{cite web|url=https://www.post.gov.tw/post/internet/Law/index.jsp?ID=1701 |title=郵政法規 - 郵政法 |publisher=中華郵政全球資訊網 |date=2011年4月27日修正公布 |accessdate=2018年5月15日最後閱覽}}</ref>，但自1972年4月13日起，中華民國喪失萬國郵政聯盟會籍，被中華人民共和國所取代，[[中華郵政|中華郵政]]沒有發行國際回信郵票券，國際回信郵票券也無法向中華郵政的各郵局兌換[[中華民國郵票|中華民國郵票]]使用。

[[英國|英國]]的[[皇家郵政|皇家郵政]]因需求量減少而從2011年12月31日起停售國際回信券，但仍受理其他國家發售的國際回信券兌換[[英國郵票|英國郵票]]<ref>{{cite web|url=https://www.royalmail.com/reply-sender |title=Reply to sender |website=Upu.int |date= |accessdate=2017-12-08}}</ref>。
