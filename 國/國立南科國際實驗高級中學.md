'''國立南科國際實驗高級中學'''（簡稱'''南科實中''', 英文簡稱 '''NNKIEH'''），是一所位於[[臺灣|臺灣]][[臺南市|臺南市]][[南部科學工業園區|南部科學工業園區]]內<ref>[https://www.google.com.tw/maps/myplaces?hl=zh-TW&ll=23.107089,120.290744&spn=0.011171,0.017231&ctz=-480&t=m&brcurrent=3,0x346ef3065c07572f:0xe711f004bf9c5469,0&z=16 國立南科實中校園位置] Google Maps</ref>的完全中學。

== 沿革 ==
南科實中前身為原屬臺南縣立之南科國民小學及南科國民中學。2006年，兩校合併改制為國立[[實驗中學|實驗中學]]，同時將兩校原有的雙語班合併為1-12年級之雙語部，當時暫無高中學生，直至97學年度開始招收第一屆高中新生。此外，校園中另設有私立之南科國際幼稚園，其與南科實中無關。建校的目的主要在於提供區內事業單位、投資廠商、政府機關及鄰近之學術研究機構員工子女，以及海外歸國服務學人之子女的就學需求。

原有之縣立南科中小學初期由地方政府設置，而南科園區最大特色—區內的國際廠商眾多，為留住外籍員工，延攬更多海外人才進駐；礙於法規與經費限制，無法滿足園區員工需求，因此為了建構優質投資環境，[[南科管理局|南科管理局]]乃積極籌設成立實驗完全中學，目的在於提供科學園區員工子女優質的教育環境。

國中自86年籌設，由黃光明博士擔任籌備處主任，歷經郭達永先生與方崑合先生擔任校長；國小自87年籌設，由呂昌道先生擔任籌備主任及校長。民國95年合併改制為國立南科國際實驗高級中學，由戴旭璋博士擔任創校首任校長。目前學校包括高中部12班、國中部21班、國小部42班、雙語部12班，合計全校學生2359人，教職員工205人，為國內鮮有的四部一體的體制。

== 制服 ==
國中小除體育服統一外均無制服。高中部則有制服，不論男女皆是白色襯衫，冬季的時候男女皆有西裝式黑底滾灰邊外套，上繡校徽；女生裙子為深綠色格裙，男生為灰色長褲，男生暗紅色領帶女生深綠色領結；<ref name="南科實中制服樣式">[https://uniform.wingzero.tw/school/album/tw/420/1 Uniform Map 制服地圖-南科國際實驗高中]</ref>比較特別的是除重大集會外，否則並不硬性規定穿制服在學校內活動，日常在校主要是穿便服。

== 社團概況 ==
[[File:南科實中管樂團.jpg|thumb]]
中學部現有共32個社團，社團活動時間為星期一下午的第二、三節課， 一學期約有8~10次的上課時間。

社團分類：共分為五大類：
*學藝性社團：(文藝校刊、科學研究、辯論、動漫文化研究、電影欣賞評論、異國文化欣賞、模擬聯合國（雙語）)
*康樂性社團：(紫錐花、手工藝、熱舞、調飲、魔術)
*體育性社團：(網球、排球、羽球、桌球、籃球、籃球裁判、軟式棒壘球(國中)、足球（雙語）)
*音樂性社團：(管樂A、管樂B、音樂演奏、吉他、合唱、熱音)
*服務性社團：(童軍A、童軍B、康輔)
