'''國父紀念館站'''位於[[台灣|台灣]][[台北市|台北市]][[大安區_(臺北市)|大安區]]、[[信義區_(臺北市)|信義區]]交界處，為[[台北捷運|台北捷運]][[板南線|板南線]]（[[南港線|南港線]]）的[[捷運車站|捷運車站]]。

== 車站概要 ==
車站位於[[忠孝東路|忠孝東路]]與[[光復南路|光復南路]]口；車站代碼為'''BL17'''。站名取自所在地東南邊的[[國立國父紀念館|國立國父紀念館]]。值得一提的是，本站之英文站名「Sun Yat-Sen Memorial Hall」與[[廣州地鐵2号线|廣州地鐵2号线]]的[[紀念堂站|紀念堂站]]相同。

[[File:Platform_1,_Sun_Yat-sen_Memorial_Hall_Station_20060720.jpg|thumb]]

== 車站結構 ==
地下二層車站，一個[[島式月台|島式月台]]，五個出入口。本站設有[[半高式月台門|半高式月台門]]。

=== 車站出口 ===
出口1、2位於車站西端，出口3～5位於車站東端，[[無障礙|無障礙]]電梯位於出口3、4之間。
* 出口1：台北秀傳醫院、[[華視|華視]]
* 出口2：大陸大樓、[[交通部觀光局|觀光局]]
* 出口3：[[臺北市信義區光復國民小學|光復國小]]
* 出口4：國父紀念館
* 出口5：觀光局旅遊服務中心、[[松山菸廠|松山菸廠]]舊址([[松山文創園區|松山文創園區]])

== 利用狀況 ==
[[File:Platform_of_Blue_Line_in_Sun_Yat-sen_Memorial_Hall_Station.JPG|thumb]]
國父紀念館站位處繁榮的[[東區商圈|東區商圈]]，鄰近的[[國立國父紀念館|國父紀念館]]為台北市的重要景點，每逢[[跨年|跨年]]期間常有大批人潮進出，與[[市政府站_(臺北市)|市政府站]]同為跨年期間最擁擠的捷運站，常需彈性加開空車疏散人潮。為了加強乘客安全及防止人員入侵軌道的情形發生，[[台北捷運公司|台北捷運公司]]於2010年在本站、市政府站與[[圓山站|圓山站]]加設[[月台閘門|月台閘門]]<ref>{{cite web |url=http://www.libertytimes.com.tw/2009/new/feb/12/today-taipei9-3.htm |title=存档副本 |accessdate=2010-06-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20100428215251/http://www.libertytimes.com.tw/2009/new/feb/12/today-taipei9-3.htm |archivedate=2010-04-28 }} 防跳軌 捷運3站將設月台門- 自由電子報</ref><ref>http://61.222.185.194/?FID=6&CID=47840 台灣新生報 | 防跳軌 北捷增3站設月台門</ref>。

== 歷史 ==
*1999年12月24日：隨著南港線「[[市政府站_(臺北市)|市政府]]－[[西門站|西門]]」段以及[[台北捷運板橋線|板橋線]]「[[西門站|西門]]－[[龍山寺站|龍山寺]]」段正式通車而啟用。
*2010年10月1日：半高式月台門正式啟用。
