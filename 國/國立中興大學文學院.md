|name = 國立中興大學文學院|EnglishName = National Chung Hsing University <br> College of Liberal Arts|image = |院長 = 韓碧琴|type = 公立|established = 1969年|location = {{ROC}}[[台中市|台中市]]|address = 南區興大路145號|affiliation = [[國立中興大學|國立中興大學]]|Faculty = 75|Student = 1800 <ref> http://www.nchu.edu.tw/introduction.php?mid=776 </ref>|Undergraduate = 1263(含進修學士班)|Postgraduate = 537(含碩專班、博士班)|website = [http://www.cla.nchu.edu.tw/ www.cla.nchu.edu.tw/]|label_color = |border_color = |box_width = }}'''國立中興大學文學院'''（英語：College of Liberal Arts ，National Chung Hsing University），係[[國立中興大學|國立中興大學]]學院之一，成立於1969年<ref>{{Cite web|url =http://140.120.126.189/about.htm|title =國立中興大學文學院-本院簡介|accessdate =2016-01-29|author =|date =|publisher =|deadurl =yes|archiveurl =https://web.archive.org/web/20150217122957/http://140.120.126.189/about.htm|archivedate =2015-02-17}}</ref>，為國內歷史最為悠久文學院之一；現有中國文學、外國語文與歷史等三系所、圖書資訊學與台灣文學與跨國文化等二研究所及台灣與跨文化研究國際博士學位學程，並於2016年秋新增文化創意產業學士學位學程<ref>{{Cite web|url = https://www.youtube.com/watch?v=K2Ct4m4NDZI|title = 2015 國立中興大學文學院簡介|accessdate = 2016-01-29|author = |date = |publisher = 中興大學鹿鳴文化資產中心}}</ref>，現任院長為中文系韓碧琴教授。

文學院原坐落於弘道樓1976年，該處原為森林系之苗圃<ref>{{Cite web|url = http://for.nchu.edu.tw/index.asp?ind=2&no=1&id=4|title = 中興大學森林系系所沿革|accessdate = 2016-01-29|author = |date = |publisher = }}</ref>，後因[[921大地震|921地震]]受損、空間不足，經教育部核准拆除<ref>{{Cite web|url = http://ir.lib.nchu.edu.tw/bitstream/11455/80929/1/book20_7.pdf|title = 校園動態:弘道樓的今與昔及人文大樓工程|accessdate = 2016-01-29|author = |date = 2013-12-11|publisher = 中興大學校友聯絡中心}}</ref>，並獲教育部補助2.3億於原址興建人文大樓，2015年啟用<ref>{{Cite web|url = http://news.gpwb.gov.tw/news.aspx?ydn=2QR3ZY8DxoKPv973RMHrgSHTD4yz4GJwg/fjBD2Lhmnqg8rNdWvGr/LRSxVsmqEUnQoXGcFh2x+ffc4CTR2+EHtep2Qj0qZK39dVG+UsxyE=|title = 興大「人文大樓」揭牌|accessdate = 2016-01-29|author = 蕭宇廷|date = 2015-12-10|publisher = 青年日報}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web|url = http://secret.nchu.edu.tw/download/366followup.pdf|title = 第 366 次行政會議列管案執行情形報告|accessdate = 2016-01-29|author = |date = |publisher = 中興大學秘書室}}</ref>。
