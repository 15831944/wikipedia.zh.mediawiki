'''國立臺灣大學醫學院校區'''簡稱'''醫學院校區'''，主要是[[國立臺灣大學醫學院|國立臺灣大學醫學院]]所在的地方。校址位於[[臺灣|臺灣]][[臺北市|臺北市]][[中正區_(臺北市)|中正區 (臺北市)]][[仁愛路_(台北市)|仁愛路]]一段1號。

== 地理位置 ==
校區位於[[臺北市|臺北市]][[市中心|市中心]]，北接[[內政部|內政部]]、[[立法院|立法院]]、[[監察院|監察院]]、[[行政院|行政院]]等政府機關及[[台北車站|台北車站]]；南面[[外交部|外交部]]、[[交通部|交通部]]、[[國家圖書館|國家圖書館]]、[[中央氣象局|中央氣象局]]、[[國家音樂廳|國家音樂廳]]、[[國家戲劇院|國家戲劇院]]、[[中正紀念堂|中正紀念堂]]等重要機構；東鄰[[國立台灣大學社會科學院|國立台灣大學社會科學院]]舊校區（在社科院遷回校總區前合稱「城中校區」）、[[台北市立成功高級中學|台北市立成功高級中學]]和[[國立台北商業技術學院|國立台北商業技術學院]]等校；西近[[二二八和平公園|二二八和平公園]]、[[北一女中|北一女中]]、[[總統府|總統府]]、[[西門町|西門町]]等。

校區附近區域涵蓋多種建築設施和組織。

== 組織 ==
=== 醫學院 ===
國立臺灣大學醫學院為[[國立臺灣大學|國立臺灣大學]]的學院之一，成立於1897年，是台灣最早成立的[[醫學|醫學]]院校。大學部包含醫學系、牙醫學系、藥學系、醫學檢驗暨生物技術學系、護理學系、物理治療學系、職能治療學系，共計七個系，另有研究所二十六所。
==== 牙醫專業學院 ====
包含牙醫學系、臨床牙醫學研究所、口腔生物科學研究所，計一系二所。


位於[[台北市|台北市]][[中正區_(臺北市)|中正區]][[臺大醫院|臺大醫院]]西址九西走廊（原總督府醫學校舊址）

==== 藥學專業學院 ====
包含藥學系、臨床藥學研究所、藥物研究中心，計一系二所。


位於[[台北市|台北市]][[中正區_(臺北市)|中正區]]林森南路33號 (國立臺灣大學水森館)

=== 公衛學院 ===
包含公共衛生學系及七個研究所。


位於[[台北市|台北市]][[中正區_(臺北市)|中正區]]徐州路17號，3樓為物理治療學系，4樓為職能治療學系，其餘皆為公衛學院所有。


座落於[[臺北市|臺北市]][[中正區_(臺北市)|中正區]][[中山南路_(臺北市)|中山南路]]與[[仁愛路_(台北市)|仁愛路]]交叉口，9.81 公頃（原總督府醫學校舊址）。

=== 臺大醫院 ===
[[台北市|台北市]][[中正區_(臺北市)|中正區]]，7.79 公頃（原總督府醫學校舊址、台北病院舊址）
==== 附設分院 ====
* 臺大醫院公館分院 - [[台北市|台北市]][[大安區_(臺北市)|大安區]]（原國軍八一七醫院舊址）
* 臺大醫院北護分院 - [[台北市|台北市]][[萬華區|萬華區]]（原[[國立台北護理學院|國立台北護理學院]]附設醫院）
* 臺大醫院雲林分院 - [[雲林縣|雲林縣]][[斗六市|斗六市]]（原署立雲林醫院）
* 臺大醫院兒童醫療大樓 - 2008年11月啟用
* 臺大醫院金山分院 - [[新北市|新北市]][[金山區_(新北市)|金山區]]（原金山醫院）
* 臺大醫院新竹分院 - [[新竹市|新竹市]][[北區_(新竹市)|北區]]（原署立新竹醫院）
* 臺大醫院竹東分院 - [[新竹縣|新竹縣]][[竹東鎮|竹東鎮]]（原署立竹東醫院）
* 臺大醫院雲林分院虎尾院區 － [[雲林縣|雲林縣]][[虎尾鎮|虎尾鎮]]

== 古蹟/建築 ==
院區建築有[[日治|日治]]時期的二號館（今[[國立臺灣大學醫學人文博物館|國立臺灣大學醫學人文博物館]]），藥學館及藥理館；[[二戰|二戰]]後增建的藥理實驗室、聯合教學館、圓形教室以及男二與女六宿舍；最後為整建後的建築物基礎醫學大樓及診斷治療大樓等。
