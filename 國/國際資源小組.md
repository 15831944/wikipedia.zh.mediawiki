'''國際資源小組'''（{{lang-en|International Resource Panel}}）由[[聯合國環境署|聯合國環境署]]於2007年成立，旨在研究並分享改善全球資源利用相關知識。小組成員包含資源管理領域傑出的科學家，探討全球資源利用之關鍵議題，以及相關科學、技術及經濟社會領域之發展，並編制評估報告提供政策參考。同時，小組也在政策制定者，產業界和各組織之間建立聯繫管道，並就改善全球和在地資源管理提供建議，致力於引導我們避免過度消費，浪費和生態危害，並實現更加繁榮及可持續的未來。

根據預測，全球人口將在2050年達到92億，伴隨著全球經濟持續成長，國際資源小組有緊迫的任務協助改善資源利用及再利用的方式，並

轉向一個社會公平，經濟利益和環境健康兼顧的全球資源利用系統不無可能。

國際資源小組的具體任務是：

* 就自然資源的可持續利用性，以及其整個生命週期不同階段對環境的影響，提供獨立並專業的科學評估。
* 強化理解使經濟增長與環境退化脫鉤之方式。

== '''國際資源小組的重要性''' ==
全球資源開採量於從1900年的約70億噸增加到2000年的約550億噸<ref>{{Cite journal|title=Assessing Global Resource Use|url=http://www.resourcepanel.org/reports/assessing-global-resource-use|last=Manager|date=2017-10-30|journal=Resource Panel|language=en}}</ref>。資源利用和相關環境破壞增加的原因包括人口增長和經濟增長。各種採掘業造成不同的環境影響。燃燒化石燃料，如石油和天然氣，會造成局部空氣污染，並增加二氧化碳排放導致[[氣候變遷|氣候變遷]]。使用生物質，尤其是木材，往往會導致土地退化和生物多樣性喪失。

採集工業礦物，礦石和建築材料導致開採地點的土地退化和土壤污染，採礦，加工和運輸也導致大量的能源消耗。採掘後的廢物也經常對環境產生不量影響，甚至具有毒性。這些導致天然土地被轉變為農業用地，過度捕撈和[[生态系统|生態系統]]的分裂。而國際資源小組的目標，便是提供經[[同行評審|同行評審]]的科學評估，以指引更可持續和公平的方式。

== '''小組成員''' ==
國際資源小組由前[[歐洲委員會|歐洲委員會環境署]]JanezPotočnik和[[巴西|巴西]]前環境部長Izabella Teixeira共同主持。由來自各學術機構和不同科學領域超過35名專家組成，並由一個小型秘書處提供協助。

== '''指導委員會成員''' ==
國際資源小組由以各國政府、歐洲委員會和聯合國環境署組成的指導委員會提供支持。

國際資源小組的合作夥伴包括[[经济合作与发展组织|經濟合作與發展組織]]，國際商會，國際科學理事會，[[國際自然保護聯盟|國際自然保護聯盟]]，世界可持續發展工商理事會等。
