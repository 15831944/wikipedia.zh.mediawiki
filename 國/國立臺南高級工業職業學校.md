'''國立臺南高級工業職業學校'''，簡稱'''臺南高工、南工'''，位於[[臺灣|臺灣]][[臺南市|臺南市]][[永康區|永康區]]的一所技術型高級中等學校。創立於1941年4月1日，原名「'''台南州立台南工業學校'''」，初設機械、電氣、工業化學、土木、建築五科，招收國民學校畢業生，修業年限為五年，因值二次世界大戰，五年課程濃縮為四年。為臺灣的[[八大省工|八大省工]]之一，亦為'''臺南市工科高職的第一志願'''，並附設南工駕訓班、進修學校。該校為教育部土木與建築群科中心主辦學校。目前有改隸於[[國立成功大學|國立成功大學]]之計畫，巧合的是成功大學於日治時期也簡稱臺南高工。<ref>[http://web.ptivs.tnc.edu.tw/ncku_ptivs/DLFile/%E8%87%BA%E5%8D%97%E9%AB%98%E5%B7%A5%E6%94%B9%E9%9A%B8%E6%88%90%E5%A4%A7%E9%99%84%E5%B1%AC%E5%8D%97%E5%B7%A5%E8%A8%88%E7%95%AB%E6%9B%B8(4%E6%9C%8830%E6%97%A5%E6%9C%83%E8%AD%B0%E9%80%9A%E9%81%8E).pdf 國立臺南高級工業職業學校改隸為國立成功大學附屬臺南高級工業職業學校計畫書]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}，國立臺南高級工業職業學校，2014年4月30日</ref><ref>[http://tw.news.yahoo.com/南工成大合併案-229票同意通過-144500567.html 南工成大合併案 229票同意通過]，中華日報，2013年9月18日</ref><ref>[http://web.ncku.edu.tw/files/14-1000-116167,r1353-1.php 臺南高工改隸成大附屬單位 成大校務會議通過] {{Wayback|url=http://web.ncku.edu.tw/files/14-1000-116167,r1353-1.php |date=20131230233946 }}，國立成功大學，2013校園焦點，2013年12月25日</ref>。

==校史==
*1941年4月1日創立[[臺南州|臺南州]]立臺南工業學校，修業年限為五年。
*1945年[[二次大戰|二次大戰]]結束，改為'''臺灣省立臺南工業職業學校'''，修業年限三年。
*1946年初級部停招新生，開辦高級部。
*1951年9月改稱為'''臺灣省立臺南高級工業職業學校'''。
*1955年8月[[臺灣省立工學院|臺灣省立工學院]]附設工業職業學校併入臺南高工，並開始實施單位行業訓練，改機械科為機工科，電機科為電工科，增設建築木工科。
*1958年7月土木、化工科停招新生，改設製圖科，並附設工業職業補習學校。
*1959年奉令附設技藝訓練中心，訓練社會失業青年。
*1961年7月增設建築泥工科。
*1962年建築木工、建築泥工兩合併為建築工科。
*1963年增設電子修護科。
*1964年增設化驗科、汽車修護科。
*1966年增設建築製圖科。
*1967年化驗、電子修護兩科增收女生。
*1968年增設板金科及航空機具修護科（後改為飛機修護科）。
*1971年增設鑄工科（後改為鑄造科）。
*1972年增設空中教學補習學校。
*1973年興建自強館、圓形教學大樓。
*1987年增設資訊科。
*1998年改稱'''國立臺南高級工業職業學校'''。
*2013年通過投票改隸[[國立成功大學|國立成功大學]]。

== 科系 ==
'''共五群，十二科'''
===機械群===

*'''機械科''' 
培養學生能擔任各種機械操作、維護、機械製圖、電腦輔助設計與製造及數值控機械操作與機電整合、精密量測檢驗等。
*'''製圖科''' 
培養機械圖面繪製、閱讀及基礎設計人才為目標。
*'''板金科''' 
培養學生具備各類板狀金屬之製圖、剪切、成型、組裝及銲接等基本入行技能。
*'''鑄造科''' 
傳授鑄造產品之設計與製作之基本知識，訓練傳統鑄模製作、金屬熔鑄、材料試驗等之基本技能。

===動力機械群===
*'''飛機修護科''' 

傳授有關飛機修護行業所需之基本技術與知識以及培養航空維護及修配管理之知能，擁有南工最貴的教具(Airbus 350 模擬機)，目前科主任為王裕良先生(2018年時)。
*'''汽車科''' 
瞭解汽車檢驗及維修之基本知識和訓練汽車裝配、保養及修護之基本技能，並學習電工、電子等專業科目為未來電動車發展準備。
*'''汽車修護科''' 
106新增科系，實用技能班學程，學習內容與汽車科相同並具備汽車修護基本技能。

===電機電子資訊群===
*'''資訊科''' 
培養資訊工業所需技術人員為目標，舉凡電腦硬體的維修、設計及電腦軟體撰寫的能力，都是培訓學生的重點。
*'''電子科'''
傳授電子、通信技術之基本知識及技能，從"做中學"培養成為3C達人，學習數位電子、工業電子、音響技術、家庭電器維修、節能光電等等。
*'''電機科''' 
培育學生具有室內配線、低壓工配、數位邏輯電路及裝配檢修、工業配電盤裝配與維修、微電腦控制、靜止電機及旋轉電機、配線、操作與維修能力。
===土木建築群===
*'''土木科''' 
傳授有關土木建築業所需之工程技術與基本知識還有工程管理之相關專業知識與營建法規。
*'''建築科'''
培養學生具備土木與建築群共同核心能力以及健全建築相關產業之初級技術人才，並為相關專業領域之學習或較高層級專業知能之進修奠定基礎。

===化工群===
*'''化工科''' 
培育化工基層技術人員，使能勝任化工相關產業之化學分析、品管分析及現場操作、維護檢修工作。

==臺南高工學生聯會==
===班聯會===
加強學生與學校及班級間的聯擊，推動全校性的活動、增進校園和諧。經過8年的努力，南工廢除進校門必須穿著制服長褲之規定。<ref>
[http://www.ettoday.net/news/20160219/650236.htm 以前都跑給糾察追！　臺南高工8年短褲終於解禁]</ref>
===畢聯會===
處理畢業生共同的事務，擔任畢業生與學校溝通及爭取權力之橋樑，並得接受學校委託辦理其他相關活動為宗旨。
===糾察隊===
在上學與放學期間協助學生交通安全，並且維持學校之秩序。另有附屬儀隊，協助學校各大集會之開場、演出。

== 學校活動 ==
===運動會===
在每年11月初舉辦，為時兩天，有各項田賽、徑賽、趣味競賽以及最受歡迎的科際接力比賽。

===園遊會===
在每年4月初舉辦，各個班級可以自行決定想要販賣的物品。
===演講活動===
大部分會在禮拜三的社團課邀請各界講師到綜合活動中心演講。
===整潔競賽(大掃除)===
為南工最愛舉辦的活動之一，通常會在親師座談會,大型考試,校慶前以及學期末前舉行，使得學生在畢業前能夠習得專業的打掃能力。

===科盃===
科與科的球類競賽，有籃球、羽球、排球、桌球和壘球等比賽。

===班際拔河===
班與班的拔河比賽，各班可自行決定是否參加。

== 學校特色 ==
*歷史悠久，校地廣大（為臺南市全高職面積最大），景色優美。
*重視推廣教育，培養基本工業技術人才。
*設科完整共12科及職能班，本校為唯一有飛機修護科之國立職校。
*學校定位：升學導向型的工業職業學校。
*學校使命：與科技大學串連，培養實務操作能力強的工程師或管理者。
*提供全方位教育：包括知識、技能、價值、態度、生活常規，即教訓輔合一教育。
*營造優質的校園：校園淨化、綠化、美化，營造科技、人文、生態和環保的校園。
*加強英語文教學與活動，提升學生國際化能力。
