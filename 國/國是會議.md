'''國是會議'''是中華民國非正式的任務型[[政治|政治]][[會議|會議]]機構，由[[中華民國政府|政府]]針對於重要政治或經濟議題，由朝野政黨與民間人士共同召開，以決定國家重要的政策方針。其起源於[[李登輝政府|李登輝政府]]時期；在李登輝執政期間，曾召開過二次大型國是會議，加速了台灣[[民主化|民主化]]的發展。

國是會議的性質類似[[政治协商会议|政治协商会议]]，但非常設機構，也不具備任何法定職權，《[[中華民國憲法增修條文|中華民國憲法增修條文]]》也無提及。

==語源==
「國是」，為國家重大的政策方針；這個名稱，首見於《[[新序|新序]]》與《[[後漢書|後漢書]]》中「共定國是」一詞<ref>劉向《[[新序|新序]]・雜事二》：楚莊王問於孫叔敖曰：「寡人未得所以為國是也。」孫叔敖曰：「國之有是，眾非之所惡也。臣恐王之不能定也。」王曰：「不定獨在君乎？亦在臣乎？」孫叔敖曰：「國君驕士曰：『士非我無逌富貴。』士驕君曰：『國非士無逌安強。』人君或失國而不悟，士或至飢寒而不進，君臣不合，國是無逌定矣。夏桀殷紂，不定國是，而以合其取舍者為是，以不合其取舍者為非，故致亡而不知。」莊王曰：「善哉！願相國與諸侯士大夫共定國是，寡人豈敢以褊國驕士民哉！」</ref><ref>《後漢書》卷28上〈桓譚馮衍列傳〉：「昔楚莊王問孫叔敖曰：『寡人未得所以為國是也。』叔敖曰：『國之有是，眾所惡也，恐王不能定也。』王曰：『不定獨在君，亦在臣乎？』對曰：『“君驕士，曰士非我無從富貴；士驕君，曰君非士無從安存。人君或至失國而不悟，士或至饑寒而不進。君臣不合，則國是無從定矣。』莊王曰：『善，願相國與諸大夫共定國是也。』」</ref>。

==歷史==
===1990國是會議===
1990年，時任總統[[李登輝|李登輝]]會見[[野百合學運|野百合學運]]學生代表，同意召開國是會議，以解決重大憲政爭議。6月28日，在[[中國國民黨|中國國民黨]]與[[民主進步黨|民主進步黨]]的支持下，於台北市[[圓山大飯店|圓山大飯店]]召開此次會議。會中針對「國會改革」、「地方制度」、「中央政府體制」、「大陸政策與兩岸關係」、「憲法與臨時條款修正方式」等五組議題逐一討論，最後達成「終止[[動員戡亂時期|動員戡亂時期]]」、「回歸[[中華民國憲法|憲法]]」、「廢止《[[動員戡亂時期臨時條款|動員戡亂時期臨時條款]]》」、「修憲採取一機關兩階段方式」、「修憲以《[[中華民國憲法增修條文|中華民國憲法增修條文]]》名之」等共識，並一一透過法定體制逐一落實。

根據會議結論，1991年5月1日，總統令公布廢止《動員戡亂時期臨時條款》，國會代表[[中華民國國會在臺灣之全面選舉與罷免|恢復定期改選]]，「[[萬年國會|萬年國會]]」現象結束，台灣的民主化進入新階段。

1990年，民进党为参与国是会议而提出《民主大宪章实录》，其中主张总统由辖区国民直接选举产生，这促成国是会议就此达成共识，从而使1996年首次实现总统直选。民进党为参与国是会议而成立了「民主大宪章研究小组」，召集人为[[黄煌雄|黄煌雄]]，成员包括名誉召集人[[黄信介|黄信介]]，以及[[张俊宏|张俊宏]]、[[吕秀莲|吕秀莲]]、[[傅正|傅正]]、[[张俊雄|张俊雄]]、[[苏贞昌|苏贞昌]]、[[谢长廷|谢长廷]]、[[陈水扁|陈水扁]]、[[江鹏坚|江鹏坚]]、[[尤清|尤清]]、[[康宁祥|康宁祥]]，顾问为[[彭明敏|彭明敏]]、[[许信良|许信良]]。2017年9月24日，黄煌雄創辦的[[台灣研究基金會|台灣研究基金會]]主办的“总统直选、民主台湾”学术研讨会上，黃煌雄表示，当时民主大宪章研究小组缺乏对[[中国大陆|中国大陆]]全盘、有系统的省思，对[[中国共产党|中国共产党]]采取了疏离、戒慎恐惧的态度，导致没能对中国大陆的迅速崛起作出正确评估与掌握<ref>{{Cite web |url = http://www.crntt.com/doc/1048/2/3/5/104823573.html?coluid=93&kindid=2910&docid=104823573&mdate=0925004420 |title = 黄煌雄：当年推动直选缺乏对大陆全盘的省思 |author = 倪鴻祥 |date = 2017-09-25 |publisher = 中评社|language =  |accessdate =  |quote =  }}</ref>。

===1996國家發展會議===
1996年，李登輝[[1996年中華民國總統選舉|二次連任總統]]後，於12月23日到28日召開國家發展會議（國發會），由副總統[[連戰|連戰]]主持，討論「健全憲政體制」、「加速經濟發展」及「增進兩岸關係」三大議題。該會議做出的最大決議為[[省虛級化|省虛級化]]。
