'''國民革命忠烈祠'''是位於[[臺灣|臺灣]][[臺北市|臺北市]][[大直|大直]]的[[忠烈祠_(臺灣)|忠烈祠]]。為[[中華民國政府|中華民國政府]][[中央政府所在地|中央政府所在地]]之專祠<ref>{{cite web |title = 國民革命忠烈祠入祀辦法：第二條 |url = http://law.moj.gov.tw/LawClass/LawAll.aspx?PCode=F0140008 |publisher = 法務部[[全國法規資料庫|全國法規資料庫]]工作小組 }}</ref>，兼有[[中華民國首都|首都]]忠烈祠之功能<ref>{{cite paper |title = 忠烈祠研究－－「國殤聖域」建立的歷史沿革 |author = 蔡錦堂 |quote = 圓山忠烈祠為國民革命專祠，暫替代首都忠烈祠 |date = 2001-06 }}</ref>。建築型式仿[[北京故宮|北京故宮]][[太和殿|太和殿]]。[[中華民國總統|中華民國總統]][[蔣中正|蔣中正]]親題為'''國民革命忠烈祠'''，民間則習稱'''圓山忠烈祠'''、'''大直忠烈祠'''或'''臺北忠烈祠'''<ref>[http://library.taiwanschoolnet.org/cyberfair2006/huigogo/index2/index2.html 忠城烈事--國民革命忠烈祠]</ref>。

1969年完工落成而成為今貌，是中華民國全國崇祀國殤位階最高的場所，也是國際人士來台訪問時，向殉難英烈致敬的代表場所。入祀烈士經[[中華民國國防部|國防部]]造冊報請總統明令行之。該祠奉祀殉職官兵共40萬餘人<ref>[http://afrc.mnd.gov.tw/martyr/martyr_brief/martyr_mnd.html 祠祀葬厝 - 國民革命忠烈祠介紹] {{webarchive|url=https://web.archive.org/web/20110430052727/http://afrc.mnd.gov.tw/martyr/martyr_brief/martyr_mnd.html |date=2011-04-30 }}</ref>。

[[File:National_Martyrs_Shrine_(0732).JPG|right]]

圓山忠烈祠鄰近[[圓山大飯店|圓山大飯店]]，祠內由[[中華民國三軍儀隊|中華民國三軍儀隊]]擔任禮兵駐守，儀隊交接儀式於每日上午9:00至下午5:00的整點執行，其中下午5:00的儀式提早至下午4:40開始，操槍動作大約20分鐘；因為儀隊交接儀式，使圓山忠烈祠成為[[臺北市|臺北市]]特別受到外國[[旅客|旅客]]造訪的一處觀光景點。

==歷史==
該地址的前身是「[[台灣護國神社|台灣護國神社]]」，曾經奉祀[[日本|日本]][[靖國神社|靖國神社]]內之臺灣籍陣亡將士<ref>[http://library.taiwanschoolnet.org/cyberfair2006/huigogo/index2/index2.html 忠城烈事--國民革命忠烈祠]</ref>。

後來因[[蔣中正|蔣中正]]認為原建築之簡陋、有失莊嚴、不足以祭祀國民革命英烈，乃擬令由陸軍一級上將[[何應欽|何應欽]]將軍主持重建、於[[民國58年|民國58年]]（1969年）3月25日落成啟用，並依《國民革命忠烈祠入祀辦法》設置、管理，1969年由[[行政院|行政院]]制頒、1999年修訂的[[行政命令|行政命令]]《[[:s:忠烈祠祀辦法|忠烈祠祀辦法]]》第八條明訂「中央政府所在地之首都，建立首都忠烈祠，並得特准建立專祠專坊或專碑」，同辦法第二十四條並規定「各地忠烈祠應於每年三月二十九日及九月三日，依公祭禮節舉行公祭；首都忠烈祠，由[[中華民國總統|總統]]主祭」，典禮相當隆重且莊嚴<ref>{{cite journal |author = 饒賀凱、楊明原、宋欽益、林勇年、謝瑋鎧 |title =〈國家神域的建構：以高雄的神社與忠烈祠為對象〉|journal = 國家文化資料館 |page = 1 |language = zh-tw |year = 2003}}</ref>。
