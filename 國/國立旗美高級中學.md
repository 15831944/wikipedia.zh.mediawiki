'''國立旗美高級中學'''（'''{{Lang-en|National Chimei Senior High School}}'''，縮寫：'''CMSH'''），簡稱'''[[旗美高中|旗美高中]]'''，是一所位於[[高雄市|高雄市]][[旗山區|旗山區]]的國立高中，位於旗尾山下，三面被旗尾山所環繞，學生大多來自旗美九鄉鎮及部分高雄市區的學生，現任校長為彭昕鋐先生 。因位處[[旗山區|旗山區]]與[[美濃區|美濃區]]交界，故得此名。旗美高中共三十班，每一年級普通科七班，職業類科三班，分別為家政科、幼保科、流行服飾科。民國95年獲[[教育部|教育部]]選為「優質化高中」，每年由[[教育部|教育部]]補助五百萬元挹注教學資源。

==校史==
*1968年，'''臺灣省立旗美高級中學'''創立，由原高雄縣立旗山中學轉入高中二、三年級學生各3班，另招新生7班，總計13班。
*69學年度（1980年9月起），配合[[中華民國教育部|中華民國教育部]]發展[[職業教育|職業教育]]政策，增設綜合家政科。
*1985年，增設服裝科。
*78學年度（1989年9月起），開始試行[[綜合高中|綜合高中]]實驗。
*1997年，成立管樂隊。
*2000年7月，增設幼保科。
*2000年2月1日，改名為'''國立旗美高級中學'''，附設職業類科。
*2010年，管樂隊參加全國學生音樂比賽團體項目南區決賽榮獲優等（八連霸）。
*2010年起，開始辦理國際教育旅行。
*2013年，旗美高中首次出現73級分亮眼的學測成績。
*2013年，成立籃球隊。
*2014年，服裝科更設為流行服飾科。
*2018年10月16日，美作高校由副校長河本幸雄領隊，共41名師生到旗美高中參訪<ref>{{Cite web|url=https://udn.com/news/story/7327/3424656|title=日本高校生來台交流 體驗泡牛奶與編織頭髮|accessdate=|author=|date=|publisher=}}</ref>。

== 職科介紹 ==
旗美高中職業類科創立於1980年(民國69年)，為一兼具高中與高職特性之學校。初期僅設家政科，1985年(民國74年)成立服裝科，2000年(民國89年)增設幼兒保育科；服裝科為提升專業能力及因應時尚產業多元技能需求，積極擴充設備並新增美容整體造型、服飾行銷等多元學習課程，於2014年(民國103年)更設為流行服飾科。目前職業類科有流行服飾科、家政科、幼保科三科，每科各設一班<ref>{{Cite web|url=http://www.cmsh.khc.edu.tw/ftp/20170324025146.pdf|title=國立旗美高級中學群科課程綱要總體課程計畫書|accessdate=2018-11-27|author=|date=2017-03-24|publisher=}}</ref>。 

== 成就表現 ==

*2016年，紫錐花龍舟隊勇奪2016年愛河龍舟賽高中混合組第二名。
*2016年10月，日本[[岡山縣|岡山縣]][[美作高校|美作高校]]師生69人蒞校，進行課程交流。
*2017年5月30日，高雄[[愛河|愛河]]端午龍舟嘉年華高中混合組冠軍及季軍。
*2018年6月18日，旗美高中紫錐花服務隊競技青少年混合組第二名<ref>{{Cite web|url=https://our.pikapage.com/page?id=5abc40255455db22271e1acc#/|title=2018高雄愛河端午節龍舟賽（賽程、即時賽果）|accessdate=|author=|date=|publisher=}}</ref>。
*2018年10月27日，前往[[印尼|印尼]]進行教育與文化交流，為期八天七夜。

==校園簡介==
*交通住宿：
# 搭乘本校提供的高雄客運學生專車。(有高雄、楠梓、鳳山、里港、美濃等多處上處)
# 國道10號高雄支線終點直行高140區道，約一公里可到達本校。
# 搭乘高雄、屏東、台南客運至旗山轉運站(原旗山南站)或旗尾站再步行至本校(可徒步行走旗山地景橋)(可參照路面指標)。
# 本校擁有設備完善的男女宿舍，可登記住宿並提供晚自習場所。
# 提供社區運動設施。

*辦學目標：
# 升學輔導及多元化教學。
# 注重生活品德教育。
# 充實人文素養、藝術薰陶。
# 提升解決問題及社會適應能力。
# 學以致用，將所學貢獻於社會。
# 幫助他人，學習關懷身邊事物。
