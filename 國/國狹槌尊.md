《[[日本書紀|日本書紀]]》記述之'''國狹槌尊'''（{{lang|ja|くにのさつち の みこと}}）乃是[[日本神話|日本神話]]裡開天闢地的[[神祇|神祇]]，亦是[[神世七代|神世七代]]之一，別名為'''國狹立尊'''（{{lang|ja|くにのさたち の みこと}}）。

== 概要 ==
以下為各典籍記載之神世七代神名對照表：

{{神世七代對照表}}

《日本書紀》卷第一對於渾沌天地成形的過程紀錄了六種說法：
* 第一種說法為有一狀貌難言之物居於天地間，先化作[[國之常立神|國常立尊]]，次化作'''國狹槌尊'''（亦曰'''國狹立尊'''）與[[豐雲野神|豐國主尊]]。
* 第二種說法是國土初發展時，國中生物如尾芽之抽出，故先化作[[宇摩志阿斯訶備比古遲神|可美葦芽彥舅尊]]，接著是[[國之常立神|國常立尊]]和'''國狹槌尊'''。
* 第三種說法是天地混成時始有[[宇摩志阿斯訶備比古遲神|可美葦芽彥舅尊]]、[[國之常立神|國底立尊]]等神人。
* 第四種說法是天地初判始有國常立尊、'''國狹槌尊'''等俱生之神。
* 第五種說法乃天地未生似海上浮雲飄盪不定，其中生出如泥中冒出之葦芽之物，化成人後便稱[[國之常立神|國常立尊]]。
* 第六種說法為天、地甫成形時，有一種浮於半空、狀若葦芽之物化成[[天之常立神|天常立尊]]，接著又化出[[宇摩志阿斯訶備比古遲神|可美葦芽彥舅尊]]、[[國之常立神|國常立尊]]等二神。

== 解說 ==
神名裡的「槌（{{lang|ja|つち}}）」意為「[[土地|土地]]」。

== 信仰 ==
