'''國道八號'''又稱為「'''臺南支線'''」或「'''[[臺南環線|臺南環線]]'''」，1999年8月16日全線通車。全線位於[[臺南市|臺南市]]，西起[[安南區|安南區]]，經過[[安定區_(台灣)|安定區]]、[[新市區_(台灣)|新市區]]東抵[[新化區|新化區]]。

==簡介==
國道八號西起[[臺南市|臺南市]][[安南區|安南]]、[[安定區_(台灣)|安定]]區界處，往東於[[台南系統交流道|台南系統交流道]]與[[中山高速公路|國道一號]]交匯，至[[新化系統交流道|新化系統交流道]]與[[福爾摩沙高速公路|國道三號]]交匯，東至新化區[[台20線|台20線]]，其中[[台南端|台南端]]（0k）至[[南133線平交路口|南133線平交路口]]（4.2k）為[[中華民國快速道路#國道快速公路|國道快速公路]]<ref>[http://www.freeway.gov.tw/Publish.aspx?cnid=526&p=134 交通部臺灣區國道高速公路局法規及行政規則-高速公路及快速公路交通管制規則]</ref>（但禁止[[大型重型機車|大型重型機車]]行駛<ref>[http://www.freeway.gov.tw/Publish.aspx?cnid=183&p=346 國道快速公路開放及禁止550c.c.以上大型重型機車行駛路段資訊]</ref>），是安南區的對外交通要道。

[[台南端|台南端]]可接[[台17線#甲線|台17甲線]]往安南區並連接北區、中西區、安平區，另築有東西向之[[台17線#乙線|台17乙線]]，連接[[台17線|台17線]]，計畫名稱「國道八號銜接[[西濱公路|西濱公路]]聯絡道」，於2007年12月18日通車，臺南市政府命名為「[[台江大道|台江大道]]」。

雖然定位為國道，但同樣位於[[台南都會區|台南都會區]]的東西向快速公路[[台86線|台86線]]重要性卻不亞於前者。因為後者雖然同樣為東西向路線，但是更貼近市區，也串連[[高鐵台南站|高鐵台南站]]、[[台南機場|台南機場]]等重要交通樞紐。與國道一號、國道三號、國道八號連成一條環繞台南市市中心之路網，為臺灣第一個單一縣市內由高速公路、快速公路組成之環狀道路。

因為[[新市交流道|新市交流道]]聯絡南科之入口道路'''[[新港社大道|新港社大道]]'''，常造成台1線交通擁堵，後續計畫自國道8號新市交流道（西側出入口）往南架設聯絡道連接[[臺南都會區北外環道路|臺南都會區北外環道路]]，並跨越[[鹽水溪|鹽水溪]]與[[台1線|台1線]]立體交叉連接，道路名稱為[https://www.google.com.tw/maps/dir/23.063879,120.276368/23.053738,120.2714182/@23.0534793,120.2714519,17.93z?hl=zh-TW 國道8號南科聯絡道延伸省道台1線道路]，目前已完工通車。

國道八號全線及側車道與銜接之國八聯絡道（台17甲線安吉路）、台17乙線、南科聯絡道、新港社大道皆有路燈照明。

== 側車道 ==
台南端至台南系統交流道之間設有側車道，長度6公里，作為匯入出主線之平面車道，亦是重要的直行交通幹線，中途與多條公路平面交會，並與主線之間，[https://goo.gl/maps/fCVcmKQfbK62 多處設有加減速車道與左右轉車道]，路廊寬度40~56公尺。

台南端以南銜接台17甲線側車道，區道南133線以東為封閉式道路，並於台南系統交流道橋下銜接南科西向連絡道。

自海佃路口起，連接此側車道至[https://www.youtube.com/watch?v=xnxFiDMStsE 台19甲線之路段實施號誌長綠]，增進車輛續行效果，可大幅減少通行時間。

==相關計劃==
*新吉工業區聯外道路
**此計畫針對新吉工業區設立，於園區內規劃20米寬（公學路一段124巷拓寬）、30米寬（安新二路）及40米寬（安順-{庄}-大道）道路來負荷未來園區內車流量，以利向東銜接台南端紓解尖峰車流量，道路已於2018年12月11日通車。

==後續計畫==
* [http://goo.gl/maps/Kck16 南科西向聯絡道南段]
**平行[[中山高|中山高]]於[[台南系統交流道|台南系統交流道]]西側興建南北雙向分隔4車道，直接聯繫[http://goo.gl/maps/mTG4z 國道8號側車道]至鄉道南134線，往東車流無須利用[[港口交流道|港口交流道]]左轉至南科，避免車流直接進入港口村聚落。
*[[南133線平交路口|南133線平交路口]]立體化
**該路口為闢建時考量當地民意反映所留設，車禍頻傳，台南市政府專函請高速公路局辦理區道南133線與國道8號立體交會，設置鑽石型交流道，重新命名為港口交流道。<ref>{{Cite web |url=http://www.chinatimes.com/realtimenews/20170315005695-260405|title= 交長抵台南 市府爭取交通建設|date=2017-03-15 |publisher=[[中時電子報|中時電子報]] |accessdate=2017-03-15 }}</ref>。沿線周遭道路已開始更換預告牌為「港口交流道」<ref>[https://goo.gl/maps/6aG757H3FC92 2017年起國八側車道沿線預告牌已有「港口交流道」]</ref>。
**由於[[南133線平交路口|南133線平交路口]]距離新吉交流道及台南系統交流道過近，高速公路局基於交通運轉及安全考量，計畫僅將該路段高架立體化，並取消設置鑽石型交流道。未來鄰近的新吉交流道將可能增設東出西入之匝道，形成完整的鑽石型交流道。
*國道1號永康交流道聯絡道工程
**國道1號[[永康交流道|永康交流道]]聯絡道，又稱為安南永康新環線，連結永康交流道至國道8號。<ref>[http://www.epochtimes.com/b5/16/11/4/n8460869.htm 永康交流道聯絡道工程]</ref>
***南133線拓寬並往南延伸連結[[台南都會區北外環道路|台南都會區北外環道路]]，並銜接永康交流道聯絡道至永康交流道。
*高架道路延伸至安中路口
**台19線交流道以西路段高架化，與台江大道立體交會並增設台江大道交流道，路寬56公尺(預留台江大道高架道路銜接匝道)，安吉路中央分隔島架設高架道路至安中路口，路寬30公尺，預估總經費約55億5000萬元<ref>{{Cite web |url=http://www.tainan.gov.tw/tainan/dep_news.asp?id=%7B62A67EB2-A625-4546-9800-A0D05F4F6473%7D|title= 國八延伸高架 南市府工務局|date=2017-03-15 |publisher=[[臺南市政府|臺南市政府]] |accessdate=2017-03-15 }}</ref>。
*[[國道八號_(中華民國)|國道8號]]連接高架道路至西濱，由營建署辦理。<ref>[https://www.chinatimes.com/realtimenews/20190408003692-260405 「最後一哩路」環評初審通過 西濱快2026可望全線通車]</ref>
