'''國民革命軍第六十軍'''原屬[[大清|大清]][[雲南|雲南]][[都督|都督]][[唐繼堯|唐繼堯]]部[[滇軍|滇軍]]，其中80%官兵均為雲南人，絕大多數都是[[少數民族|少數民族]]，以[[彝族|彝族]]、[[白族|白族]]為主。

[[宣統|宣統]]三年（1911年），以[[同盟會|同盟會]]會員為核心的一批滇軍將領，響應[[辛亥革命|辛亥革命]]，推翻[[滿清|滿清]]在雲南之統治。

[[民國|民國]]4年（1915年），滇軍參加[[護國戰爭|護國戰爭]]，在[[中國近代史|中國近代史]]上留名。

民國26年（1937年）9月，當時雲南王[[龍雲|龍雲]]將原有的6個步兵[[旅|旅]]編組為國民革命軍第六十軍，[[盧漢|盧漢]]任[[軍長|軍長]]。

1937年10月10日，第六十軍下轄的第一八二、第一八三、第一八四共3個師。主要裝備法械，作為滇系主力部隊，裝備十分精良、齊全，是抗戰初期[[中國軍隊|中國軍隊]]當中裝備最強、人數最多的一個軍。

== [[抗日戰爭|抗日戰爭]]時期 ==
1937年10月5日，第六十軍在雲南[[昆明|昆明]]東南郊的巫家壩操場舉行誓師出征大會，隨後開赴對[[日本|日本]]作戰前線，在[[台兒莊大捷|台兒莊大捷]]中作出重大貢獻，傷亡13000多人。

1938年3月下旬，[[日本軍|日本軍]]進犯[[徐州|徐州]]，第六十軍奉命前往增援，負責防守[[禹王山|禹王山]]一線，經過27晝夜血戰，傷亡巨大，但陣地巍然不動。

1940年9月，日軍佔領[[越南|越南]]，第一八二師、第一八四師轉調雲南防守[[紅河州|紅河州]]一帶。

1945年8月15日，[[盧漢|盧漢]]親率[[國民革命軍第九十三軍|國民革命軍第九十三軍]]、第六十軍入越南，接受日軍[[投降|投降]]。

== [[第二次国共内战|第二次国共内战]]時期 ==
1945年11月，[[蔣中正|蔣中正]]又重新整編滇軍，將8個步兵師縮編為2個軍，即第九十三軍、第六十軍。同月，[[曾澤生|曾澤生]]被任命為第六十軍軍長。

1946年4月，第六十軍奉命海運[[中国东北地区|中国东北地区]]增援[[剿匪|剿匪]]作戰，第六十軍下轄第一八二師、第一八四師和暫編第二十一師。第一八二師駐防[[瀋陽|瀋陽]]、[[鐵嶺|鐵嶺]]、[[昌圖|昌圖]]歸[[孫立人|孫立人]]指揮。第一八四師駐防[[鞍山|鞍山]]、[[海城|海城]]等地歸[[廖耀湘|廖耀湘]]指揮，暫編第二十一師駐防[[撫順|撫順]]等地歸[[東北行營|東北行營]]直接指揮，由于分散使用，駐防鞍山、海城的第一八四師被包围，1946年5月30日阵前倒戈。

重建后的第一八四师在梅河口被全歼。

1947年11月，暂编第五十二师列入第六十军建制，但“人事、经理自成一系”。该师由原东北第四保安区改编为暂编第五十二师，师长刘伯中。后并编入东北交警总局吉林警务处，中央嫡系李嵩任师长。

1948年3月，第六十军放弃吉林市，撤守[[長春|長春]]城東，下轄暂编第二十一师、暂编第五十二师、第一八二师。

再次重建后的第一八四師驻辽西，与第六十軍分开使用，在1948年10月[[锦州之战|锦州之战]]中被歼灭。

[[蔣中正|蔣中正]]對這支少數民族部隊一直都不是很放心，再加上裝備給養都不如駐防長春城西的[[國民革命軍新編第七軍|國民革命軍新編第七軍]]，最終導致這支來自雲南的滇軍在[[辽沈战役|辽沈战役]]中倒戈。
 
1948年10月14日，[[曾澤生|曾澤生]]和其麾下暂编第二十一师師長[[隴耀|隴耀]]、第一八二师师长[[白肇學|白肇學]]等商定起义事宜。10月17日，第六十軍發動[[起义|起义]]，新編第七軍被迫投诚。国军第六十军被改编为[[中国人民解放军第50军|中国人民解放军第50军]]，辖第148、第149、第150师，曾泽生任军长至1970年。

== [[中华人民共和国|中华人民共和国]]時期 ==
1950年，以“[[中国人民志愿军|中国人民志愿军]]”名义入朝作战，参加第一次、第二次、第三次、第四次战役。1955年后撤出朝鲜，隶属于成都军区。

1967年，50军第149师（1948年由国军暂21师改编）使用18军52师番号，后改编为西藏军区第52旅；其余所属部队及军部番号在1985年裁军中被撤销。
