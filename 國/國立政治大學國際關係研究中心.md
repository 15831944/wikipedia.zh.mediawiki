'''國立政治大學國際關係研究中心'''，簡稱'''政大國際關係研究中心'''或'''政大國關中心'''，是[[冷戰|冷戰]]時期[[中華民國|中華民國]]成立的[[智庫|智庫]]中心<ref>{{cite journal |url = http://www.chinatimes.com/newspapers/20141014001163-260901 |title = 諾貝爾獎得主也遭監控 美中搶買馬英九AB檔案 |author = 王超群 |date = 2014年10月14日 |quote = 位於木柵山區的政治大學國關中心，早年極其神祕低調，尤其是冷戰時期，更是全球知名中共與蘇聯情報的交換據點，常有情治人員出入。雖然那一段輝煌歲月已逝，國關中心至今仍存放龐大的國安密檔，甚至還有許多最高等級的「絕對機密」！ |journal = 《[[周刊王|周刊王]]》 |issue = [http://www.nownews.com/p/2014/10/14/1454911 第027期]}}</ref>，今為[[亞洲|亞洲]]及[[太平洋地區|太平洋地區]]重要[[智庫|智庫]]之一。<ref name=penn>{{cite web|url=http://repository.upenn.edu/think_tanks/|title=
2016 Global Go To Think Tank Index Report|format=pdf|page=61|publisher=University of Pennsylvania|author=James G. McGann, Ph.D.|language=en|location=Philadelphia|date=2017-01-26}}</ref>

== 沿革==
1953年4月1日，「'''國際關係研究會'''」創立，主要研究[[中共|中共]]及[[國際|國際]]問題，提供[[中華民國政府|中華民國政府]]施政參考<ref name="intro">{{cite web |url = http://iir.nccu.edu.tw/intro/pages.php?ID=intro2&submenu=intro2&menu=mastermenu2 |title=歷史沿革 |publisher = 國立政治大學國際關係研究中心}}</ref>。

1961年，國際關係研究會改為「中華民國'''國際關係研究所'''」，擴大對國內外學術研究機構之聯繫與合作<ref name="intro" />。

[[吳俊才|吳俊才]]於1964年正式接任國際關係研究所主任，兩位副主任為[[立法委員|立法委員]]暨[[國際關係|國際關係]]學者[[鄧公玄|鄧公玄]]和[[司法行政部調查局|司法行政部調查局]]副局長[[郭華倫|郭華倫]]<ref name=sinica/>。

1975年7月1日，中華民國國際關係研究所改制為「'''國際關係研究中心'''」，隸屬[[國立政治大學|國立政治大學]]，繼續推動對[[中國大陸|中國大陸]]以及國際事務之研究<ref name="intro" />。

== 概況 ==
原由[[中國國民黨|中國國民黨]]主持，並與[[美國|美國]]的外交單位有密切合作，充作美國在中國共產黨研究的前哨站。[[中美斷交|中美斷交]]、臺灣[[解嚴|解嚴]]以後，國關中心於1996年8月1日完全併入政大建制。其[[圖書館|圖書館]]是[[臺灣地區|臺灣地區]]收藏[[中國共產黨|中國共產黨]]相關資料最豐富的圖書館，在學術研究上有一定重要性<ref name=sinica>{{cite journal |url = http://www.mh.sinica.edu.tw/MHDocument/PublicationDetail/PublicationDetail_1556.pdf |title = 敵前養士:「國際關係研究中心」前傳, 1937-1975 |author = 劉曉鵬 |journal = 中央研究院近代史研究所集刊 |issue = 82 |pages = 第145至第174頁 |date=2013年12月 |publisher = 中央研究院近代史研究所 |ISSN = 0578-2015}}</ref>。現受到[[中华教育文化基金会|中华教育文化基金会]]設立的中國政治學會圖書館基金所補助，該基金會也是[[國立清華大學|國立清華大學]]基金的管理者。

* 美國與歐洲研究所：前身為國際關係研究所及國際關係研究中心之「國際組」。1996年8月1日國研中心歸併[[政治大學|政治大學]]後，改編為「第一研究所」。研究領域包含外交、政治、經濟、安全及其他重要領域，主要專攻於[[美洲|美洲]]、[[歐洲|歐洲]]及[[非洲|非洲]]之情勢、國際關係、經濟及國際組織之研究。
* 亞洲太平洋研究所：最初亦為國際關係研究所「國際組」，後收編為「第二研究所」研究方向專攻於[[亞洲|亞洲]]、環[[太平洋|太平洋]]地區及[[大洋洲|大洋洲]]各國情勢、國際關係、經濟及國際組織之研究。
* 中國政治研究所：曾歷經「國際關係研究會」、「國際關係研究所」及「國際關係研究中心」等三個時期，在「國際關係研究會」及「國際關係研究所」時期分別為該「會」及「所」之下的「匪情研究組」。後改制為「第三研究所」，目前研究方向為亞太情勢的發展、安全經濟與政治、[[中共|中共]]於國際雙邊關係中對[[中華民國|中華民國]]地位等問題的主張之研究、[[習近平|習近平]]時代的挑戰、中共對外軍事衝突的政治運用之研究、政治參與在[[香港|香港]]特區的發展、[[世貿組織|世貿組織]]與[[中國|中國]]、全球化下的兩岸經濟關係、中共和平崛起、中共對外戰爭、中共十六大後之走向、新世紀的中國兩岸、[[馬克思|馬克思]]與[[恩格斯|恩格斯]]的文藝理論、臺灣海峽[[兩岸關係|兩岸關係]]報告、中共軍事思想發展、中國大陸國防工業的民間化。
* 中國社會經濟研究所：成立之初主要定位為國家智庫，隸屬「匪情研究組」，後改制為「第四研究所」，研究之領域包括[[中國大陸|中國大陸]]的社會、經濟、文化、教育、法律、區域研究等。其中國農村研究團隊為全台灣第一批進入中國深山[[農村|農村]]，進行蹲點研究，而且持續至今未中斷，累積了豐厚的田野資料。氣候變遷、水治理、西部大開發乃至兩岸鄉村治理與發展、經濟往來、[[中國大陸|中國大陸]][[高等教育|高等教育]]、農業及工業、非政府組織、國營企業等，皆是第四研究所的研究領域。
* 當代日本研究中心：政治大學當代[[日本|日本]]研究中心成立於2009年9月24日，本研究中心是由[[日本交流協會|日本交流協會]]以及[[日本國際交流基金|日本國際交流基金]]資助設立，主要目的希望強化國內的日本研究。並成為兩國之間的學術交流平台。
* 台灣非洲研究論壇：未來[[非洲|非洲]]國家在國際社會的影響力，將成為我國在外交困境下的援助力，因此非洲研究論壇的成立，可以讓非洲國家駐中華民國人員，能有更多的機會參與不同活動與國人交流。如此一來，不僅讓非洲國家更了解我國，也使得國民能更認識來自非洲的朋友的背景文化和事務，促進雙方文化的交流，並且了解兩岸問題在國際外交上和其他的方面所造成的困難。
