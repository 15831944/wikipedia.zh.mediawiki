'''國立中山大學復校史'''，係指[[國立中山大學|國立中山大學]]「[[在台復校|在台復校]]」的三十年運作歷史。

== 孫中山在廣東創校 ==
[[File:National_Sun_Yat-sen_University_Canton.jpg|thumb]]
1924年，[[中华民国陆海军大元帅大本营|中華民國陸海軍大元帥]]、[[中國國民黨|中國國民黨]]總理[[孫中山|孫中山]]先生為培育[[國民革命|國民革命]]人才，並因應積極建設[[現代化|現代化]][[國家|國家]]之需要，將「[[廣東高等師範學校|廣東高等師範學校]]」、「[[廣東公立法政大學|廣東公立法政大學]]」、「[[廣東公立農業專門學校|廣東公立農業專門學校]]」3所學校合併，改組成為[[國立廣東大學|國立廣東大學]]，並頒授「博學、審問、慎思、明辨、篤行」之校訓。孫中山翌年病逝後，廣東大學更名為「國立中山大學」，並於[[廣州|廣州]][[石牌|石牌]]建立新校址。

[[抗日戰爭|抗日戰爭]]爆發後，中山大學匆匆南遷，先後復課於[[雲南|雲南]]，又移校於[[粵|粵]]北，直至1945年才遷回石牌原址。1949年，[[國民政府|國民政府]]於[[國共內戰|國共內戰]]中敗走台灣，中華民國的國立中山大學被[[中華人民共和國|中華人民共和國]]改為新[[中山大學|中山大學]]。

*1905年，「[[廣東法政學堂|廣東法政學堂]]」創建
*1909年，「[[廣東農林試驗場|廣東農林試驗場]]」創建，「[[廣東公立醫學堂|廣東公立醫學堂]]」創建，「[[廣東公立警監學堂|廣東公立警監學堂]]」創建
*1910年，「[[廣東工藝局|廣東工藝局]]」創建
*1924年，「[[廣東高等師範學校|廣東高等師範學校]]」、「[[廣東公立法政大學|廣東公立法政大學]]」、「[[廣東公立農業專門學校|廣東公立農業專門學校]]」合併升格「'''[[國立廣東大學|國立廣東大學]]'''」
*1925年，「[[廣東公立醫科大學|廣東公立醫科大學]]」、「[[廣東公立工業專門學校|廣東公立工業專門學校]]」併入

*1928年3月，復名為「國立中山大學」
*1937年，「[[國立廣東法科學院|國立廣東法科學院]]」併入法學院；「[[廣東省立勷勤大學|廣東省立勷勤大學]]工學院」併入工學院
*1952年，原中山大學主體與「[[嶺南大學_(廣州)|嶺南大學]]」合併，校址遷至嶺南大學原址康樂園

== 中山大學校友籌畫復校 ==
[[File:Hsieh_Tung_Ming.jpg|thumb]]
[[File:Zheng_Yanfen.jpg|100px]]
===初期復校運動（1960年代）===
1962年1月，校友會理監事聯席會議決議組織「復校促進委員會」，積極進行復校運動。

1963年起，關切中山大學復校之校友先後在[[中國國民黨|中國國民黨]]第9、11次全國代表大會，[[謝東閔|謝東閔]]、[[鄭彥棻|鄭彥棻]]等11位中常委都分別提出復校案，也均經一致通過。

1964年[[國民大會|國民大會]]第4次會議通過114人提議，迅謀中山大學復校案，多名[[立法委員|立法委員]]校友歷年皆於[[立法院|立法院]]再三質詢，請政府從速在[[臺灣|臺]]恢復大學。

1965年[[教育部|教育部]]部長[[黃季陸|黃季陸]]為慶祝國父百年誕辰，曾於11月12日宣布「籌備中山大學復校」，並成立「國立中山大學重建委員會」負責籌畫。但因教育部編列之籌備復校預算無著，復校進行因而受阻。

===臺中之議（1971年）===
1971年初，教育部決定將[[省立中興大學|省立中興大學]]改制國立，[[臺灣省議會|臺灣省議會]]遂建議「將中興大學改名中山大學」，國民黨[[中國國民黨黨史史料編纂委員會|中央黨史會]]與[[設計考核委員會|設計考核委員會]]聯銜建議中央轉知從政同志一致支持。[[監察院|監察院]]院長、副院長暨[[監委|監委]]等多人聯銜致函[[行政院|行政院]]院長[[嚴家淦|嚴家淦]]支持本案。同時[[美國|美國]]、[[加拿大|加拿大]]、[[泰國|泰國]]、[[香港|香港]]等地校友會亦表態支持。

然而，教育部隨後復函省府：「關於省議會建議國立中山大學復校一案，當存備研究參考。」第二波復校運動由此無疾而終<ref>吳文曉編，〈國立中山大學校史（稿）〉，收入國立中山大學校友會編，《國立中山大學的回顧與展望》（臺北：1986年11月11日），頁甲九。</ref>。

===嘉義之議（1977年）===
歷經兩次失敗後，復校運動依舊不依不饒。[[海外|海外]]方面，香港同學會首先成立「國立中山大學校友促進復校委員會」，美國[[華僑|華僑]]（包括美國和加拿大）亦組織支援「國立中山大學復校委員會」。[[中華民國|國內]]方面，[[立委|立委]][[蕭天讚|蕭天讚]]於1977年向行政院質詢，提出「敬請迅速核准中山大學在嘉義復校案」。

同年，海外僑團歸國慶祝10月10日[[國慶|國慶]]大典時組成「美洲華僑支援國立中山大學復校代表團」並晉見[[教育部|教育部]]部長[[李元簇|李元簇]]，進行早日在臺恢復母校的陳情。李氏表示政府當於增設[[大學|大學]]時予以考慮。隨後，代表團應[[嘉義縣|嘉義縣]]地方人士之邀請，前往訪問，該縣地方人士[[林國川|林國川]][[博士|博士]]組成歡迎會，[[縣長|縣長]]、[[議長|議長]]、[[民意代表|民意代表]]等數10人皆全程參加，表示「歡迎中山大學在嘉義復校」，縣方願提供60[[甲|甲]]的土地建校。

===定址高雄市（1978年至1980年）===
[[File:Chiang_Ching-kuo_1948.jpg|thumb]]
====國大的提案====
1978年3月17日，國民大會第6次會議，校友代表[[鄧克一|鄧克一]]等39人，[[何文炯|何文炯]]等126人，[[嚴江津|嚴江津]]等36人，分別提出母校[[在台復校|在台復校]]三案，前一案決議送政府參辦，後兩案合併討論，並通過了「請政府迅速確實辦理」之議。

5月20日，躬逢[[中華民國|中華民國]]第6任[[總統|總統]][[蔣經國|蔣經國]]先生、[[副總統|副總統]][[謝東閔|謝東閔]]先生（校友）就職典禮，海外校友紛紛組團回國祝賀。慶祝會中，多人耳聞教育部有在[[南臺灣|南部]]增設[[國立大學|國立大學]]之議，復校希望由此轉濃。

校友會把握時機，向當局請求即將新設之南部大學「能夠命名為國立中山大學」。立委[[司徒政|司徒政]]向教育部質詢時亦提出相同之議論。[http://www.focat.org.tw/blog/ 華僑救國聯合總會]全體理事會議於10月24日通過請政府在臺設立國立中山大學之決議。[[監察院|監察院]]教育委員會也通過監察委員40餘人連署案，促請政府設立國立中山大學。至此，復校運動成為全台民意機構、海外華僑、地方人士之公意。

====澄清湖或西子灣====
1979年，教育部宣布「國立中山大學於[[高雄市|高雄市]]建校」。7月成立籌備處，聘請[[李煥|李煥]]先生為籌備處主任。

當局原擬將[[澄清湖|澄清湖]]40餘[[公頃|公頃]]大學預定地作為校址，經實地勘查後，以該地不敷建立一所[[綜合大學|綜合大學]]之用，且該地內有私人土地，徵用費時，因而作罷。9月18日教育部次長[[施啟揚|施啟揚]]與李煥宣布「國立中山大學決定在高雄[[西子灣|西子灣]]設校，惟澄清湖大學用地仍予保留」<ref>吳文曉編，〈國立中山大學校史（稿）〉，收入國立中山大學校友會編，《國立中山大學的回顧與展望》（臺北：1986年11月11日），頁甲一零。</ref>。

====高雄中山大學====
至此，捨棄了三個地點（臺中、嘉義、高雄澄清湖），進行了四次復校運動，歷經半甲子30年的時間之後，國立中山大學終於完成在台復校之宏願。然而，該校復校以來一直以1980年為校齡計算之起點，未曾舉行過囊括[[廣州|廣州]]時期校齡的周年校慶，而校徽、校旗與主要[[行政|行政]]教研人員亦皆未繼承廣州的[[中山大學|中山大學]]，其復校之名實，相較於[[國立清華大學|國立清華大學]]、[[國立交通大學|國立交通大學]]等校，仍有待商榷。

在嚴格的意義上，國立中山大學與[[中山大學|中山大學]]的關係，類似於美國[[華盛頓大學|華盛頓大學]]與[[聖路易斯華盛頓大學|聖路易斯華盛頓大學]]。其校名皆因「紀念偉人」而起，實質上是各自獨立的大學。

== 復校之後 ==
[[File:國立中山大學海研三號.jpg|200px]]
===初期：4系2所230人===
復校第一年設有[[三民主義|中山學術]]研究所、[[企業|企業]]管理研究所、[[中國文學|中國文學]]系、[[外國語文|外國語文]]學系、企業管理系及[[電機|電機]]工程學系。起初預定3年內發展為文理、工商、海洋3個[[學院|學院]]，計9個學系及3個[[研究所|研究所]]，5年內發展為文、理、工、管理、海洋5個學院，共計17個學系及更多的研究所。

1980年招考中山學術研究所及企業管理研究所碩士班，第一屆研究生各15名，應考生卻多達1700餘名，錄取者多為國內著名[[大學|大學]]畢業生。7月參加全台[[大學聯考|大學聯考]]，招收學生兩百人，錄取者亦多為國內著名[[高中|高中]]畢業生。

===近況：全台6強===
經六任校長之經營擘劃，復校近30年的國立中山大學至今計有6個[[學院|學院]]，19個[[學系|學系]](97學年成立光電工程學系)，35個[[碩士|碩士]]班、25個[[博士|博士]]班，現有學生人數近萬人，專任師資470餘位。2000年代成為全臺7所「研究型大學」之一，穩居國際主要[[大學排名|大學排名]]全臺8強地位，屬全臺250餘所大學中的佼佼者。

2014年，國立中山大學在[[世界大學學術排名|世界大學學術排名]]位列第426名，為'''全臺第6'''<ref>{{cite web|url=http://news.ner.gov.tw/index.php?act=culnews&code=view&ids=167093|title=國立教育廣播}}</ref>，也是4所「[[中字輩|中字輩]]大學」之首。同一年，首位[[中國大陸學生赴臺灣就讀|陸生]]台灣博士從本校亞太所畢業<ref>[http://www.chinatimes.com/newspapers/20140617000778-260107 全台首位 陸生博士出爐 - 中時電子報]</ref>。
