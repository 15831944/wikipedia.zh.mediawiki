== 學校簡介 ==
=== 歷史源流 ===
* 1992年8月1日，成立籌備處，並指派李福登[[博士|博士]]擔任籌備處主任，統籌各項事宜。
* 1995年7月1日，'''國立高雄餐旅管理專科學校'''正式成立，成為全國首創的餐旅專業學府。
* 2000年，該校正式升格為'''國立高雄餐旅學院'''，增設二技學制及專科進修部。
* 2005年，受高雄市政府教育局教委託成立管理"'''高雄市立餐旅國民中學'''"'''。
* 2010年6月15日，獲'''[[中華民國教育部|教育部]]'''審議通過，8月1日改名'''國立高雄餐旅大學'''。
* 2012年，增設學士後渡假管理學士學位學程、學士後烘焙管理學士學位學程。
* 2013年，停招學士後渡假管理學士學位學程、學士後烘焙管理學士學位學程。
* 2015年8月1日，高雄市立餐旅國民中學改制並改隸為「[[國立高雄餐旅大學附屬餐旅高級中等學校|國立高雄餐旅大學附屬餐旅高級中等學校]]」。
* 2016年8月1日：「餐旅管理研究所」與「餐旅教育研究所」整併，改名為「餐旅研究所」，並分為「管理組」與「教育組」；「旅遊管理研究所」與「運輸與休閒服務規劃碩士學位學程」整併為「觀光研究所」，並分「休閒旅遊組」與「運輸服務組」；「台灣飲食文化產業研究所」、「餐飲創新研發碩士學位學程」整併為「飲食文化暨餐飲創新研究所」。

== 學生活動 ==
===學生自治組織===
國立高雄餐旅大學全校性學生自治組織全名為「國立高雄餐旅大學學生會」。依據該校「學生自治團體設置及輔導辦法」學院；所、系、學位學程及科層級，成立學生自治團體，也就是俗稱之「系學會」。

===學生社團===
該校設「學生社團」。發展普通，可區分為：自治性社團、服務性社團、體育性社團、康樂性社團、學術性社團。<ref>{{Cite web |url=http://efolio.nkuht.edu.tw/CL_NKHC/index.aspx# |title=國立高雄餐旅大學 社團活動資訊暨管理系統 |accessdate=2018-01-13 |work=efolio.nkuht.edu.tw |archive-url=https://web.archive.org/web/20180114073708/http://efolio.nkuht.edu.tw/CL_NKHC/index.aspx |archive-date=2018-01-14 |dead-url=yes }}</ref>

== 學生實習爭議 ==
2016年7月起，有兩名三年級的此校學生受校方安排至台中永采烘焙坊實習，但數月下來[[工時規管|工時]]過長且工作的環境苛刻又薪水刻意給低，兩位學生無法負荷決定提前結束實習<ref>{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20170222/1061456/|title=血汗實習「重罰」2萬？　惡老闆起底|last=|first=|date=2017-02-22|work=|newspaper=蘋果日報|accessdate=2017-02-25|language=zh-TW}}</ref>。烘培坊業者因此指責學生以及校方對於實習工作的處理態度不佳，將提告要求兩位學生賠償業者在交接工作不完整導致的損失各59萬元<ref>{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20170224/1063267/|title=抓到了！血汗實習店索賠59萬證明曝光|last=|first=|date=2017-02-24|work=|newspaper=蘋果日報|accessdate=2017-02-25|language=zh-TW}}</ref>。此事於2017年2月被媒體廣為報導，永采實習被指為[[血汗工廠|血汗店家]]，引起民間輿論及政府重視。[[中華民國教育部|教育部]]技職司在事發後一個月內將清查所有學校的實習環境<ref name=":0">{{Cite news|url=http://www.epochtimes.com.tw/n201146/%E9%98%BB%E8%A1%80%E6%B1%97%E5%AF%A6%E7%BF%92--%E6%95%99%E9%83%A8-1%E5%80%8B%E6%9C%88%E7%B5%A6%E4%BA%A4%E5%BE%85.html|title=阻血汗實習  教部：1個月給交待|last=吳|first=旻洲|date=2017-02-24|work=|newspaper=台灣大紀元|accessdate=2017-02-25|location=台北}}</ref>，台中市政府勞工局則在調查後依照違反[[勞動基準法_(中華民國)|勞基法]]對永采開罰100萬元新台幣<ref>{{Cite web|url=http://m.appledaily.com.tw/realtimenews/article/new/20170225/1064466/|title=血汗烘培坊違法明確　中市府重罰100萬|accessdate=2017-02-25|author=黃任膺|date=2017-02-25|work=m.appledaily.com.tw|publisher=蘋果日報|location=台中}}</ref>。校方在溝通後決定將兩位學生轉往台中金典酒店與台中日月千禧酒店實習<ref>{{Cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20170225/37563638/|title=血汗名店超冷清 660元就「包場」|last=王|first=乙徹|date=2017-02-25|work=|last2=黃|first2=任膺|newspaper=蘋果日報|accessdate=2017-02-25|language=zh-TW|last3=吳|first3=慧芬}}</ref>，校方主秘石名貴表示對此次事件於校方內部進行調查檢討<ref name=":0" />。
