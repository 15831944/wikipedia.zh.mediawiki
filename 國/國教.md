'''国教'''或'''官方宗教'''、'''官方信仰'''，是指由[[国家|国家]]确立的特定[[宗教|宗教]]。拥有国教的国家并非一定为[[神权|神权]]国家，也不代表国教受到政府控制。国家或政府在当代社会中对公民施加国教影响的程度差别很大，从[[沙特阿拉伯|沙特阿拉伯]]、[[伊朗|伊朗]]的高度影响，到[[丹麦|丹麦]]、[[英格兰|英格兰]]、[[蘇格蘭|蘇格蘭]]、[[挪威|挪威]]、[[冰岛|冰岛]]和[[希腊|希腊]]的最低度影响或根本没有影响。

== 簡介 ==
历史上，在實行[[政教合一|政教合一]]的國家，国教可以對國家政權的政治和政策有很大影響力，有的國家內的人民更被強-{制}-成為国教之信徒<ref name="rbw">{{cite web |title = 國外宗教立法種種 |url = http://www.rbw.org.cn/article.aspx?ty=uuY&i=uUQ&pg=3&langu=f |publisher = 人本网 |accessdate = 2012-10-08}}</ref>。與此同時，國家政權也可以對國教有很大的影響力，有很多宗教的教義在政治需要下或者在政治領導人的個人喜好之下被任意解讀，宗教領袖被當權者隨意撤換，信徒們被迫接受。

[[東南亚|東南亚]]的一些国家将[[佛教|佛教]]确立为国教，而[[欧洲|欧洲]]地区的很多国家则把[[基督宗教|基督宗教]]的某個宗派确立为国教（[[天主教|天主教]]、[[新教|新教]]及[[東正教|東正教]]），[[阿拉伯国家|阿拉伯国家]]则把[[伊斯兰教|伊斯兰教]]（[[遜尼派|遜尼派]]）确立为国教。[[日本|日本]]在[[江戶時代|江戶時代]]，規定每戶家庭都必須歸屬在[[日本佛教|日本佛教]]某宗派之下，成為[[寺院|寺院]]的[[檀越|檀越]]，稱「寺檀制度」；[[大日本帝國|帝國時期]]則實施「[[國家神道|國家神道]]」，將[[神道|神道]]定為國民共通的[[信仰|信仰]]。實際上[[日本政府|日本政府]]的這兩次行為，相當於訂定神道與佛教為國教。

在[[宗教改革|宗教改革]]及[[啟蒙運動|啟蒙運動]]後，現代的[[世俗|世俗]]國家多實行[[政教分离|政教分离]]，官方宗教不再支配政府。在[[宗教自由|宗教自由]]的保障和[[世界人权宣言|世界人权宣言]]下，做為公民權利，人們可根據自己的意願選擇宗教信仰<ref name="rbw" />。同樣的，各種宗教也不再受政府支配。正是出於這個理由，[[基督新教|基督新教]]提出了[[政教分離|政教分離]]的概念，並且在世界各地大力推廣。

全世界以[[憲法|憲法]]或其他法律條文規定基督宗教為國教的有[[英格蘭|英格蘭]]、[[丹麦|丹麦]]、[[冰島|冰島]]、[[希臘|希臘]]等國家，但保障[[宗教自由|宗教自由]]。規定伊斯蘭教為國教的有[[伊朗|伊朗]]、[[叙利亚|叙利亚]]、[[沙特阿拉伯|沙特阿拉伯]]等多數[[中東|中東]][[阿拉伯國家|阿拉伯國家]]及亞洲的[[马来西亚|马来西亚]]（[[马来西亚宪法|马来西亚宪法]]中規定伊斯蘭教为馬国官方宗教而非國教，但國内仍然有伊斯蘭教國与[[世俗國家|世俗國家]]的争议，而國內保障[[宗教自由|宗教自由]]。）。規定佛教為國教的有[[泰國|泰國]]、[[不丹|不丹]]、[[柬埔寨|柬埔寨]]等國家。这三个宗教也常被合称为[[世界三大宗教|世界三大宗教]]。政教合一強制人民信奉國教（伊斯蘭教）的有[[伊朗|伊朗]]。

{{cn|另外，雖強調宗教自由及[[政教分离|政教分离]]，但許多西方國家如[[美國|美國]]，由於信奉[[基督宗教|基督宗教]]人口佔多數，在某些制度設計上，獨厚主要的宗教，如[[基督新教|基督新教]]宗派和[[羅馬天主教|羅馬天主教]]开辦的学校常常可以豁免[[稅金|稅金]]，[[土耳其|土耳其]]在公立學校會推廣伊斯蘭教义，也可視為一種隱性的國教表徵。}}

== 宗教势力最强和最弱的国家和地区 ==
{{Unreferenced section|time=2019-01-07T09:35:35+00:00}}
宗教势力包括[[多神教|多神教]]（polytheism）、[[一神教|一神教]]（monotheism）和[[泛神论|泛神论]]（pantheism）的宗教组织及其[[神职人员|神职人员]]。

所谓[[泛神论|泛神论]]（pantheism），为未设立具体神祇之教义崇拜（如日本的[[神道|神道]]、海地的[[巫毒|巫毒]]），非指[[无神论|无神论]]（explicit atheism）。
