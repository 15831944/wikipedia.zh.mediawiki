'''國民革命軍第六十六軍'''曾多次组建。

== 抗日戰爭初期 創建 ==
民國26年(1937年)以[[國民革命軍|國民革命軍]]第一五九師與國民革命軍第一六〇師編成國民革命軍第六十六軍，隸屬於[[國民革命軍第十二集團軍|國民革命軍第十二集團軍]]，[[葉肇|葉肇]]任軍長，國民革命軍第一五九師長為[[譚邃|譚邃]]，1938年10月[[陳驥|陳驥]]繼任師長，國民革命軍第一六〇師由葉肇兼任師長，1938年2月[[華振中|華振中]]繼任師長。

1938年4月譚邃接任軍長，10月譚邃病故，陳驥繼任軍長，下轄三個師:第一五一師，師長[[莫希德|莫希德]]，1939年1月[[林偉儔|林偉儔]]繼任師長；第一五九師，師長，1938年10月陳驥升任軍長，[[官禕|官禕]]繼任師長；第一六〇師，師長華振中，1939年1月[[宋士台|宋士台]]繼任師長。

1940年，第六十六軍被撤銷。

== 抗日戰爭中期 第一次重建 ==
1942年3月，又以第二[[補訓處|補訓處]]改編成第六十六軍，[[張軫|張軫]]任軍長，下轄新編第二十八師、新編第二十九師、新編第三十八師。第六十六軍編成後，即編入[[中國遠征軍|中國遠征軍]]入[[緬|緬]]作戰。第六十六軍駐防[[臘戍|臘戍]]、[[瓦城|瓦城]]一帶，1942年3月23日[[日本軍|日本軍]]開始進攻，以縱隊快速向細胞、臘戍穿插突進，第六十六軍分段截擊日軍。

1942年3月29日，日軍攻佔臘戍，新編第三十八師撤向[[印度|印度]]，新編第二十八師退回國內，第六十六軍番號撤銷。

新編第二十八師為1939年5月由[[軍委會別動隊|軍委會別動隊]]改編成，[[劉伯龍|劉伯龍]]任師長，隸屬[[重慶衛戍司令部|重慶衛戍司令部]]，1940年開赴[[貴州|貴州]]，隸屬[[滇|滇]][[黔|黔]][[綏靖公署|綏靖公署]]，1942年11月[[李士奇|李士奇]]繼任師長，1943年改隸屬[[國民革命軍第七十一軍|國民革命軍第七十一軍]]。

新編第二十九師為1939年5月以第二補訓處之一部改編成，[[馬維驥|馬維驥]]任師長，隸屬重慶衛戍司令部，1943年改隸屬[[國民革命軍第六軍|國民革命軍第六軍]]。

新編第三十八師為1938年12月以[[財政部|財政部]][[稅警總團|稅警總團]]改編成，[[孫立人|孫立人]]任師長，駐防[[貴州|貴州]][[都匀|都匀]]，在[[第一次滇緬戰役|第一次滇緬戰役]]後轉至印度，後改隸屬[[國民革命軍新編第一軍|國民革命軍新編第一軍]]。

== 抗日戰爭中期 第二次重建 ==
1940年10月以暫編第三十二師、暫編第三十三師、暫編第三十四師合編成[[國民革命軍暫編第九軍|國民革命軍暫編第九軍]]，以[[馮聖法|馮聖法]]為軍長。

1942年12月，暫編第九軍改編為第六十六軍。該軍編成後，在浙江又將暫編第三十五師編入該軍，隸屬[[抗日戰爭第三戰區|第三戰區]]，1942年8月改隸屬[[抗日戰爭第六戰區|第六戰區]]，12月改隸屬[[江防司令部|江防司令部]]。1943年2月，馮聖法他調，任命[[方靖|方靖]]為軍長，1943年2月第六十六軍撤銷。

暫編第三十二師，為1940年9月以[[浙江省|浙江省]][[國民自衛團|國民自衛團]]第一縱隊及[[浙江|浙江]]第六[[保安團|保安團]]為基幹編成．以[[黃權|黃權]]為師長。

暫編第三十三師，該師原為浙江[[抗日|抗日]]自衛部隊，1940年4月以其第一、第六、第八支隊和第五區總隊併編為[[抗日|抗日]]自衛第二縱隊，10月改編為暫編第三十三師，以[[蕭冀勉|蕭冀勉]]為師長，1943年9月[[周淘漉|周淘漉]]任師長，該師改隸屬[[國民革命軍第八十八軍|國民革命軍第八十八軍]]。

暫編第三十四師，1938年1月成立第三戰區[[遊擊司令部|遊擊司令部]]，該部以浙江省警員及各地常備隊、壯丁編成4個[[支隊|支隊]]，7月改編為浙江省國民自衛團[[總司令部|總司令部]]，1940年將所屬支隊併編為3個縱隊，3月將[[第三縱隊|第三縱隊]]改編為暫編第三十四師，[[彭鞏英|彭鞏英]]為師長，1942年3月[[朱奇|朱奇]]任師長，1943年2月以第十八師副師長[[吳嘯亞|吳嘯亞]]任師長，同時改隸屬[[國民革命軍陸軍第十八軍|國民革命軍陸軍第十八軍]]。

暫編第三十五師，以浙江地方部隊編成，[[勞冠英|勞冠英]]任師長。1942年底，該[[師|師]]調至第六戰區後，勞冠英被調任[[夔巫|夔巫]]師管區司令，該師被併編。

== 抗日戰爭末期 第三次重建 ==
1943年10月再次以[[國民革命軍暫編第九軍|國民革命軍暫編第九軍]]改編為第六十六軍，仍以[[方靖|方靖]]為軍長，改轄第一八五師、第一九九師，1944年8月[[方靖|方靖]]調任[[國民革命軍第七十九軍|國民革命軍第七十九軍]]軍長，副軍長[[宋瑞珂|宋瑞珂]]升任軍長。

第一八五師，[[師長|師長]][[方天|方天]]，原隸屬[[國民革命軍第九十四軍|國民革命軍第九十四軍]]，1940年12月方天升任[[國民革命軍第十八軍|國民革命軍第十八軍]]軍長，[[石祖黃|石祖黃]]接任師長，1943年[[李仲辛|李仲辛]]接任師長。

第一九九師，師長[[羅樹甲|羅樹甲]]，1940年5月羅樹甲升任副軍長，[[宋瑞珂|宋瑞珂]]接任師長，1943年2月、3月[[蘇令德|蘇令德]]、[[周天健|周天健]]相繼任該師師長，1944年9月[[彭戰存|彭戰存]]接任師長。

抗戰勝利後1946年5月第六十六軍撤銷，改編為整編第六十六師，由原軍長宋瑞珂任師長。1947年7月該師在[[魯|魯]]西南被殲，宋瑞珂被俘。

== 動員戡亂初期 第四次重建 ==
1947年9月重建該師，以[[李仲辛|李仲辛]]為師長。 11月[[羅賢達|羅賢達]]接[[李仲辛|李仲辛]]任師長。該師轄：整編第十三旅，[[旅長|旅長]]羅賢達，羅升任師長後，由[[張潔|張潔]]接任旅長；整編第一八五旅，旅長李仲辛，1946年[[塗煥陶|塗煥陶]]接李任旅長，1947年9月[[袁樾人|袁樾人]]接任旅長職；整編第一九九旅，旅長[[彭戰存|彭戰存]]，1946年由[[王士翹|王士翹]]任旅長，1947年9月張潔接任旅長，11月由[[蕭炳寅|蕭炳寅]]接任旅長職。

整編第六十六師，師長李仲辛。轄整編第十三旅，旅長羅賢達，1948年3月[[鄭潔|鄭潔]]接任旅長；整編第—八五旅，旅長[[周穆深|周穆深]]；整編第一九九旅，旅長張潔，1948年10月[[蕭炳寅|蕭炳寅]]接任旅長，該旅於1948年6月改隸屬[[國民革命軍第七十九軍|國民革命軍第七十九軍]]。

1948年6月，整編第六十六師及所轄第十三旅，在[[河南|河南]][[開封|開封]]被殲，師長李仲辛[[自殺|自殺]]。

== 動員戡亂末期 第五次重建 ==
1948年7月又重建該師，以羅賢達為師長，轄第十三旅，旅長[[羅有經|羅有經]]；第一八五旅，旅長袁樾人。

1948年9月恢復第六十六軍番號，羅賢達改任軍長，各旅長改為師長。

1949年初改隸第七[[綏靖區|綏靖區]]。是年4月，該軍在[[皖|皖]]南被殲。
