'''國際象棋'''（{{lang-en|chess}}），又稱'''-{zh-cn:西洋棋;zh-hk:西洋棋;zh-tw:國際象棋}-'''，是一種二人對弈的[[抽象策略遊戲|戰略]][[棋盘遊戲|棋盘遊戲]]，也是世界上最流行的遊戲之一。世界各地數以百萬計的人在家中、{{link-en|西洋棋俱樂部|chess club|俱樂部}}中、網路上以{{link-en|通信西洋棋|correspondence chess}}或比賽形式对弈。

國際象棋的棋盤由64個黑白相間的八乘八網格組成，棋盤上上到下的格子的左邊是1~8從左到右是A~H，每個棋子都有代號。王＝Ｋ后＝Ｑ車＝Ｒ馬＝Ｎ象＝Ｂ兵是由你所在的格子來判斷如果你在b5（是從左邊數過來第二個是b，從b的地方下往上數第五個就是b5）那之兵就用b來代表。每位玩家开局时各有16個棋子：一[[王_(國際象棋)|王]]、一[[后_(國際象棋)|-{后}-]]、兩[[车_(國際象棋)|车]]、兩[[马_(國際象棋)|马]]、兩[[象_(國際象棋)|象]]和八[[兵_(國際象棋)|兵]]，各具不同功能与走法。棋手行棋目标是將對方的王處在不可避免的威脅之下以[[將死|將死]]對方，也可以通过對方自知无望、主动认输而獲勝，另有相当多的情况可导致[[和局_(國際象棋)|和局]]和局有幾種第一，你要求和棋對方如果也同意的話你們就和棋了。第二，五十步沒有吃子動兵就和棋。第三，對方所有子力都不能動又輪到對方這樣叫做『逼和』。遊戲過程分三個階段：[[國際象棋開局|开局]]、[[國際象棋中局|中局]]、{{link-en|西洋棋殘局|Chess endgame|残局}}，共有10<sup>43</sup>至10<sup>50</sup>種棋局變化。

国际象棋棋子多用[[木|木]]或[[塑膠|塑膠]]製成，也有用石材制作；较为精美的石頭、[[玻璃|玻璃]]（[[水晶|水晶]]）或[[金屬|金屬]]製棋子常用作裝飾擺設。

== 歷史 ==
[[File:KnightsTemplarPlayingChess1283.jpg|thumb]]，1283年。]]
{{main|象棋史}}
國際象棋一般被認爲源自一種[[印度|印度]]的遊戲——[[恰图兰卡|恰图兰卡]]，7世紀時流傳至[[波斯|波斯]]成為[[波斯象棋|波斯象棋]]。[[穆斯林|穆斯林]]統治波斯後，它被帶到[[伊斯蘭|伊斯蘭]]的國家。[[九世紀|九世紀]]傳入南歐，10世紀時傳到[[西班牙|西班牙]]，11世紀傳到[[英國|英國]]。15世紀末，現代國際象棋的規則逐漸成形。現代的玩法與19世紀時的大致相同。由於流傳已久，因此在各地與各時期產生不少的[[國際象棋變體|國際象棋變體]]規則。

== 規則 ==
=== 棋盤與棋子 ===
{{Chess piece}}
國際象棋的棋盤為正方形，由32個深色和32個淺色方格交替排列組成。每邊8個方格。淺色棋格稱為“白格”，深色棋格稱為“黑格”。

擺放方式：棋手的棋盤右下角必須為白格。白棋王后置於白格；黑棋王后置於黑格。

國際象棋的棋子也分為兩種顏色：淺色棋子稱為“白棋”，執白棋的棋手稱為“白方”；深色棋子稱為“黑棋”，執黑棋的棋手稱為“黑方”。對弈雙方各有16枚棋子，分別為一國王、一王后、双主教、雙馬、雙車和八士兵。開局時即全部放在棋盤上規定的棋格內。

棋局由白方先下，對弈雙方輪流移動棋盤上既有的己方棋子，一步棋只能移動一個棋子（唯一的例外是「[[王車易位|王車易位]]」，是在一步棋同時移動王和車）。棋子要不就是移動到未被占據的棋格，要不就是占據對方棋子所在的棋格，並將對方棋子拿出棋盤，稱為「吃子」（這也有一個例外是「[[吃過路兵|吃過路兵]]」，己方的兵是落在對方被吃兵的後一格）。一個棋格內只能有一個棋子，棋子也不可重疊。

==== 三个特殊规则 ====
;士兵的[[升變|升變]]：
:當己方的士兵走到对方的底线（即最远离己方的一行）时，玩家可選擇把該士兵升級為車、馬、主教或王后，但不能變國王，也不能選擇不變。

;[[吃過路兵|吃過路兵]]：
:當一方的士兵从原始位置向前一步走兩格時，如果所到格的同一横线的相邻格有对方的士兵时，则后者可以立即吃掉相邻的前者，但是占据原来位置的斜前方那一格，而不是前者原来占据的那格。但“吃过路-{兵}-”的一方必须在对方走棋後的下一步立刻吃，否则就会失去该“吃過路-{兵}-”的机会。

==== 棋譜記法 ====
棋盘的每一个格子由一个坐标来表示。横向的一排方格称为一“行”，纵向的一排方格称为一“列”。

棋盘上由同色方格斜角相连而形成的斜行称为“斜线”。棋盘上的斜线共有26条，长短不一。最长的斜线有8格，最短的2格。斜线用始末两端的方格代号表示，例如a1-h8斜线、c1-h6斜线。a1-h8斜线和h1-a8斜线又称“主斜线”或“大斜线”。

为了便于记录棋谱，棋盘上离白方最近的那一行定为1，最远的那一行定为8。位于白方最左边的那一列定为a，最右边的那一列定为h。这样，白方左下角的第1格的坐标就是a1，白方王所在的原始位置就是e1，黑方-{后}-所在的原始位置就是d8。

;[[代数记谱法|代数记谱法]]：
:目前通用的是先写棋子再写由哪格走到哪格（士兵可以不写）。例如'''马g1-f3'''，表示马由g1格走到f3格。例如'''e2-e4'''，表示白方士兵初始的e2格向前走两步到e4格。

:当一方有两个同样的棋子可以到达同一个方格时，则需要写出所走棋子起始格的行或者列，例如车ad3表示在a3的车走到了d3。

:如果一方吃掉对方的棋子，则记谱时加上'''x'''。例如'''主教-b5xc6'''或'''主教xc6'''，表示原先位于b5格主教走到c6格，并且“吃”了对方的一个棋子。

:“将军”时，记谱时加上'''+'''。例如王后c5+表示王后走到c5格并且“将军”对方的國王。

:记录王车易位，以'''O-O'''表示短易位，以'''O-O-O'''表示长易位。

:记录吃过路-{兵}-，在着法後加上e.p.（en passant）

:记录士兵的升变，在着法後加上（=王后）、（=车）等。

:其他一些常用的记录棋谱的记号还有：
::'''#''' 或'''x'''：表示将死。
::'''++'''：表示双将。
::'''1/2-1/2'''：表示棋赛的结果是和棋。
::'''1-0'''：表示棋赛的结果是白方胜。
::'''0-1'''：表示棋赛的结果是黑方胜。

:记谱时有时还会用到评论记号：
::'''!'''：有利的着法；
::'''!!'''：非常有利的着法；
::'''?'''：错招；
::'''??'''：大错招；
::'''!?'''：後果不明的着法；
::'''∽'''：走任意一着棋。

:正式的计时比赛中，如果一方所剩的时间不到5分钟，则此方可以允许不再记谱。

=== 勝負和 ===
==== [[將軍_(國際象棋)|將軍]]與[[將死|將死]] ====
国际象棋的对局目的是把对方的國王将死。比赛规定：当一方的國王受到对方棋子攻击（威胁）时，即下一步就可以把對方的“國王”吃掉时，成为國王被照将，攻击方称为“叫将”、“照将”、“打将”或「将军」（check），此时被攻击方必须立即設法解除对“國王”的威脅（称为“应将”，一是把攻擊國王的棋子吃掉，二是移動“國王”到另一格去，三是用自己的其他棋子把攻击“國王”的棋子挡住。），如果无法避开将军，王即被'''将死'''（checkmate），棋局即告負。

==== [[和局_(國際象棋)|和棋]] ====
由[[世界国际象棋联合会|世界国际象棋联合会]] (World Chess Federation, FIDE)的规则所规定的「[[和局_(國際象棋)|和棋]]」局面有以下5种：

* 其中一方虽然其「國王」自身沒有被對方「將軍」，但是却没有合法的棋步可走。这种“和棋”称“[[欠行|逼和]]”。

* 任何一方以任何合法走法都无法将死对方。这种“和棋”称“死局”。例如國王对國王单马、或國王对國王单主教等等。

* 一方提出和棋要求，並得到另一方的同意。这种“和棋”称“协议和局”。

* 3次重复同一局面，包括两种情况：1)即将第三次（或以上）重复；2）刚刚第三次（或以上）重复。由有权提出方提出。

* 在连续的50个回合内，双方既没有棋子被吃掉，也没有士兵被移动过。由有权提出方提出。

==== 等级计分方法 ====
国际象棋的记分方式实行级别、积分制。级别分别是无级别、30-1级。每个级别需要积累100分。有关输赢后积分的详细计算方法如下（注意：输方扣分等于赢方得分）<ref>[http://games.sports.cn/server/guoxiang/2008-04-22/1442015.html 国际象棋简明规则]. 中国棋院在线,2008-4-22,[2011-11-27]</ref>：
*和局：双方现有积分相差10分以内，无积分；相差10-99分，则分低方得(10+差分/10)/2；相差100分以上，分低方得差分/4。 　　
*低级别方赢：双方现有积分相差100分以上，则得差分/2；相差10-99分，得10+差分/10；相差10分以内，得10分。
*高级别方赢：双方现有积分相差200分以上，无积分；相差100-199分，得2分；相差10-99分，得10-差分/12.5；相差10分以内，得10分。 　　
*下过50盘以后得分/失分会趋于稳定。

==== 等级设定 ====
按等级可分为[[特级大师|特级大师]]、[[国际大师|国际大师]]、[[棋联大师|棋联大师]]、[[棋协大师|棋协大师]]、[[候补大师|候补大师]]、[[女子特级大师|女子特级大师]]、[[女子国际大师|女子国际大师]]

== 開局 ==
[[国际象棋开局|开局]]，也称布局，是[[国际象棋|国际象棋]]游戏的第一阶段。一般来说，开局是从双方棋子的原始位置开始，直到第10-15个回合。

=== 开局的目標 ===
* 控制中心地区（不一定是用棋子去占据中心地区）。
*國王的安全得到保证。
* 尽快出子，使自己的子力能迅速到达进攻或防御的位置
* 建立起好的[[兵形|兵形]]，使各个士兵之间能互相呼应，以便进攻和防御。

=== 开局的完成 ===
* 双方主要子力已经基本出动，或者可以随时出动；就是说，主要子力的主要行动路线已经基本打开。
* 國王的位置已经确定；最好已经进入比较安全的位置。
* 局面结构已经基本定型。
* 双方攻守计划的主要趋向已经基本明确。

=== 开局的分类 ===

== 中局 ==
国际象棋'''[[中局|中局]]'''的主要目標是：
# 为残局打下基础和做好准备。如果一方不能在中局时“将死”对方，就要拖延到残局。这样，歼灭对方的兵和保护好自己的兵，就是中局非常重要的一个内容：因为兵可以升变。实际上，歼灭对方兵并且保护己方兵在开局的时候就需要加以考虑。
# 想办法去“将死”对方。
# 通过各种手段，包括各种战术（或战术组合）、各种兑子手段和各种赚子手段，削弱和消灭对方的进攻和防御的棋力。

== 殘局 ==
国际象棋的对局分为开局、中局和残局三个阶段。其中残局是重要的，具有决定性的阶段。当对局经过中局后，双方的棋子显著减少，局面相对简化的时候，对局就进入了残局阶段。不是所有的对局都会经过残局阶段，一些激烈攻防的对局，可能在进入残局之前，就于激烈的战斗中决定了胜负。但是大多数的对局都会进入残局阶段。

残局的基本任务是将自己的优势转化成为胜利或者弱化对方的优势从而取得和棋。专业棋手都要对残局进行严格的训练。很多理论家认为，首先学习残局会对初学者更好地学习国际象棋有巨大的帮助，因为残局不仅棋子数量相对较少，而且这样可以更好地理解每个棋子的特性和它们之间相互的协调，从而可以更好地学习和掌握开局和中局的战略和战术。

虽然初学者往往从单杀王开始学起，但是实战中遇到的残局往往是更加复杂的多子多兵的残局，作为专业棋手，对所有这些局面的把握都要经过严格的训练。

=== 一些常见的残局及其胜负形势 ===
* 单-{后}-、单车、双象、马象都可杀单王，但双马不行。

==== -{后}-类残局 ====
* -{后}-对车：一般能胜。
* -{后}-对马象：一般能胜，但有一种和棋形势。
* -{后}-对双象：一般能胜。另外也有一种和棋形势。
* -{后}-对双马：一般是和棋。
* -{后}-对车和弱子（弱子指马或象）：一般是和棋。
* -{后}-对双车：一般是和棋，但双方都有取胜可能。
* 双车和弱子对单-{后}-：一般能胜，但需超过50步。
* -{后}-和弱子对车和一个弱子：一般能胜。
* -{后}-和弱子对双车：-{后}-马对双车一般是和棋，-{后}-象对双车一般能胜，但最多需85步。
* -{后}-车对-{后}-车：尽管子力相当，在67.74%的情况下先走者能胜。
* 车和两个弱子对单-{后}-：一般是和棋。
* -{后}-和弱子对单-{后}-：一般是和棋。
* -{后}-对三个弱子：一般是和棋，对三个同色格象除外。
* 四个弱子对单-{后}-：一般能胜。

==== 车类残局 ====
* 车和弱子对单车：一般是和棋，但车象对单车有很大的取胜可能。
* 车对象：一般是和棋。
* 车对马：一般是和棋。
* 双车对两个弱子：一般能胜。
* 双象单马对单车：异色格象一般能胜，但最多需68步。
* 双马单象对单车：一般是和棋，但有一些取胜形势，最多需49步。
* 车象对双马：一般能胜，但最多需223步。
* 车马对双象：双象为同色格的话，一般能胜，但最多需243步。双象为异色格的话，一般是和棋。
* 车象对马象：如果这两个象色格相同，一般是和棋。如果这两个象色格不同，则一般能胜，但最多需98步。
* 双车对单车：一般能胜。

==== 其它残局 ====
* 双象对单马：一般能胜〔假设这两个象色格不同〕，但最多需66步。
* 两个弱子对一个弱子：一般是和棋，双象对单马除外。
* 三个弱子对一个弱子：一般能胜，但可能需要超过50步。
* 王兵对单王：
**1.有兵方先夺取升变控制格胜
**2.有兵方先“对王”胜，无兵方先“对王”和。

== 人機大戰 ==
編写一个能下国际象棋的计算机程序，曾是科学家们梦寐以求的目标。

基本上人們認為1997年超级电脑[[深藍_(電腦)|深蓝]]與当时的[[國際象棋世界冠軍|國際象棋世界冠軍]][[卡斯帕罗夫|卡斯帕罗夫]]的六盤決戰，是電腦擊敗人腦的開端。（參見條目：[[深蓝对卡斯帕罗夫|深蓝对卡斯帕罗夫]]）可是，雖然在1997年那場人機大戰，表面上是超級電腦戰勝了人類第一高手，但在具爭議性的第二局中，卡斯帕羅夫懷疑有人類大師影響深藍走棋。

同時，在最後幾步深藍竟然犯下大錯，為卡氏提供長將成和的機會，但卡氏卻沒有注意到而認輸，最後一局深藍更只是因卡氏顛倒走棋次序而獲得勝利。而且總計1996年和1997年深藍vs卡斯帕羅夫兩次對戰的成績，後者仍以6.5比5.5勝出。

而在2003年裏，卡斯帕羅夫分別再與超級電腦X3D Fritz 和 Deep Junior（比深藍強一倍）對弈，均能取得勝局，而這兩次對弈都以和局結束，可見1997年深藍擊敗卡斯帕羅夫的說服力不夠強。

另一位[[國際象棋世界冠軍|國際象棋世界冠軍]][[克拉姆尼克|克拉姆尼克]]亦曾與超級電腦{{illm|Deep Fritz|WD=Q901348}}對弈，時間分別為2002年和2006年。在2002年的對弈中，克拉姆尼克與Deep Fritz 8 以4:4戰和(2勝2負4和)；而在2006年，克拉姆尼克在人機大戰中，以2:4敗予Deep Fritz 10,。而此後也沒有這些人機對抗賽，因為人類始終難以在64格裏戰勝電腦，比賽的觀賞性也逐漸下降。

== -{zh-tw:西洋棋;zh-hk:象棋;zh-cn:象棋;}-與伊斯蘭教 ==
《[[穆斯林聖訓實錄|穆斯林聖訓實錄]]》28:5612記述[[穆罕默德|穆罕默德]]說：「玩象棋者，如同把手染入豬肉和豬血中。」<ref>[http://cmje.usc.edu/religious-texts/hadith/muslim/028-smt.php The Book of Poetry (Kitab Al-Sh`ir)], Translation of Sahih Muslim, Book 28.</ref>

1981年，[[伊朗伊斯蘭共和國|伊朗伊斯蘭共和國]]禁止玩-{zh-tw:西洋棋;zh-hk:象棋;zh-cn:象棋;}-，直至1988年才由[[伊朗最高領袖|最高領袖]][[霍梅尼|霍梅尼]]發佈教令解禁，容許非賭博性質的玩象棋行為<ref>[http://www.google.com/hostednews/afp/article/ALeqM5i126emdhs7EHvMjqUayaJeUetLtA Grandmaster's next move: A chess revival in Iran]，2010年1月20日。</ref>。2016年1月，[[沙特阿拉伯|沙特阿拉伯]]大[[穆夫提|穆夫提]]Abdul Aziz bin-Abdullah發佈[[伊斯蘭教令|教令]]禁止下象棋，認為象棋對弈製造仇恨、浪費時間和鼓吹賭博，不過該教令無法律約束力。<ref>{{Cite news |url = http://www.dailymail.co.uk/news/article-3410448/Now-Saudi-Arabia-bans-CHESS-Muslim-cleric-outlaws-game-waste-time-encourages-gambling.html |title = Saudi Arabia bans CHESS after Grand mufti says it's 'a waste of time' | work=Daily Mail Online |date=2016-01-21 }}</ref>
