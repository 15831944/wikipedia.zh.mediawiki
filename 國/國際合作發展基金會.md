'''財團法人國際合作發展基金會'''（簡稱'''國合會'''，英文譯稱：'''International Cooperation and Development Fund'''，縮寫：'''ICDF '''），是[[中華民國外交部|中華民國外交部]]成立的一個財團法人機構，總部設於[[台灣|台灣]][[台北|台北]]，主要是為協助友好或[[開發中國家|開發中國家]]發展、增進之間經濟關係而成立，也提供協助遭受[[天然災害|天然災害]]的國家或國際[[難民|難民]]，透過[[投資|投資]][[融資|融資]]、技術合作、[[人道援助|人道援助]]及國際教育訓練等方式執行，協助友好或開發中國家進行不同領域的發展計畫<ref>{{cite web |author1=王照坤|title=友邦人才不足 我助培訓盼創多贏|url=https://www.rti.org.tw/news/view/id/2008823|publisher=中央廣播電台|accessdate=2019-01-19}}</ref><ref>{{cite web |author1=李怡欣|title=助友邦改善公衛醫療 國合會展現軟實力|url=http://www.epochtimes.com/b5/18/12/19/n10920205.htm|publisher=大紀元|accessdate=2018-12-19}}</ref>，包括[[環境|環境]]、[[公衛|公衛]][[醫療|醫療]]、[[農業|農業]]、[[教育|教育]]及資[[通訊|通訊]]等<ref name="國際發展合作的概念與實務">{{cite book |author1=陳正忠 |title=國際發展合作的概念與實務(二版)|date=2007-03 |publisher=智勝文化事業有限公司 |isbn=978-957-28648-4-5|accessdate=2018-10-09}}</ref>。藉由提升人力素質、強化組織能力建構，援外的專業度，提供技術專業建議。強調計畫的縱深度，落實技術轉移，有效運用人力資源，以在不同國家複製成功經驗<ref name="國際發展合作的概念與實務"></ref>。

== 歷史沿革 ==
*1959年12月28日，中華民國政府派遣農技團赴[[越南|越南]]協助發展農業，為首次對外援助<ref>{{cite book |author1=蘇怡仲、蘇于修|title=天地行腳：台灣技術人員的無國界貢獻 |date=2009-12 |publisher=財團法人國際合作發展基金會 |isbn=978-957-28648-7-6|accessdate=2018-10-09}}</ref>。
*1961年政府成立「先鋒案執行小組」專責派遣農耕隊前往[[非洲|非洲]]國家。
*1962年擴大該組織為「中非技術合作委員會」。
*1972年「中非技術合作委員會」與「外交部海外技術合作委員會」合併為「海外技術合作委員會」（海外會），專責我國對外技術援助工作，協助友好發展中國家農業發展。
*1989年因國內經濟快速成長，成立「海外經濟合作發展基金管理委員會」（海合會），對友好發展中國家提供各種經濟援助。
*嗣後基於援外業務日益專業，為整合援外資源，經[[行政院|行政院]]提請[[立法院|立法院]]審議，於1995年12月19日三讀通過「財團法人國際合作發展基金會設置條例」<ref>{{cite web |title=國際合作發展基金會設置條例 |url=https://law.moj.gov.tw/LawClass/LawAll.aspx?PCode=E0000012 |publisher=法務部全國法規資料庫|accessdate=2018-10-09}}</ref>。
*1996年1月15日[[總統|總統]]明令頒布生效後，「國際合作發展基金會」於當年7月1日正式成立，海合會及海外會則分別裁撤<ref name="國際發展合作的概念與實務"/>。

== 合作計畫 ==
國合會秉持全球關懷精神，針對夥伴國家實際需求，擬定核心業務計畫；計畫一經評估可行，即派遣專業人員以實際行動參與協助。於[[非洲|非洲]]、[[中南美洲|中南美洲]]、[[加勒比海|加勒比海]]、[[亞太|亞太]]及亞西等30個合作國家執行技術合作業務<ref name=cooperative>{{cite web |title=合作計畫總覽|url=http://www.icdf.org.tw/ct.asp?xItem=335&CtNode=29779&mp=1 |publisher=財團法人國際合作發展基金會|accessdate=2019-01-17}}</ref>。
