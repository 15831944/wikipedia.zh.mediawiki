'''國際會館站'''（{{jpn|j=国際会館駅|hg=こくさいかいかんえき|rm=Kokusai-Kaikan eki|f=y}}）是一個位於[[日本|日本]][[京都府|京都府]][[京都市|京都市]][[左京區|左京區]]{{link-ja|岩倉 (京都市)|岩倉 (京都市)|岩倉}}，屬於[[京都市營地下鐵|京都市營地下鐵]][[烏丸線|烏丸線]]的[[終點站|終點站]]。車站編號是'''K01'''。此站獲選為[[近畿車站百選|近畿車站百選]]<ref>{{cite web|url=http://wwwtb.mlit.go.jp/kinki/koutsu/tetsudo/tetu100_3.pdf|title=近畿の駅百選 あの駅この駅すてきな駅25駅|publisher=近畿運輸局|accessdate=2014-10-11|format=pdf}}</ref>。

== 歷史 ==
* 1997年（[[平成|平成]]9年）6月3日 - [[京都市營地下鐵烏丸線|京都市營地下鐵烏丸線]]由[[北山站_(京都府)|北山站]]延長至此站<ref name=sone04>{{Cite book|和書|author=[[曽根悟|曽根悟]]（監修） |title=週刊 歴史でめぐる鉄道全路線 公営鉄道・私鉄 |editor=朝日新聞出版分冊百科編集部（編集） |date=2011-04-03 |publisher=[[朝日新聞出版|朝日新聞出版]] |series=週刊朝日百科 |volume=4号 京福電気鉄道・叡山電鉄・嵯峨野観光鉄道・京都市交通局 |pages=22-23}}</ref>。

== 車站結構 ==
[[島式月台|島式月台]]1面2線的[[地底車站|地底車站]]。閘口只設1個。設有對應[[輪椅|輪椅]]的升降機與多用途廁所等[[無障礙設施|無障礙設施]]。
