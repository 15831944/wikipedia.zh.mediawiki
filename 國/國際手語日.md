'''國際手語日'''（[[英文|英文]]：{{lang|en|International Day of Sign Languages}}）是通過[[聯合國大會|聯合國大會]]的決議草案而設立的國際性日子，目的在於推廣[[手語|手語]]，第一屆為2018年，而日期為每年的9月23日。

== 決議草案 ==
=== 提交 ===
最初是由{{link-en|世界聾人聯合會|World Federation of the Deaf}}透過各會員所屬國家的駐聯合國代表團向[[联合国大会#附屬委員會|聯合國大會第三委員會]]提交決議草案，初次決議草案「A/C.3/72/L.36」是由成員國[[安提瓜和巴布達|安提瓜和巴布達]]、[[澳大利亞|澳大利亞]]、[[貝里斯|貝里斯]]以及[[日本|日本]]於2017年10月30日向第三委員會提交<ref>{{cite web |title=成員國提交至聯合國大會第三委員會的決議草案 |url=https://undocs.org/ch/A/C.3/72/L.36 |date=2017-10-30 |publisher=[[聯合國大會|聯合國大會]] |accessdate=2018-10-01 |language=zh-hans}}</ref>；而後於同年11月14日再提交訂正決議草案「A/C.3/72/L.36/Rev.1」，同時提案成員國增加[[阿根廷|阿根廷]]、[[亞美尼亞|亞美尼亞]]、[[巴西|巴西]]、[[智利|智利]]、[[薩爾瓦多|薩爾瓦多]]、[[危地馬拉|危地馬拉]]、[[以色列|以色列]]、[[黎巴嫩|黎巴嫩]]、[[利比里亞|利比里亞]]、[[毛里裘斯|毛里裘斯]]、[[摩洛哥|摩洛哥]]、[[巴拿馬|巴拿馬]]、[[斯里蘭卡|斯里蘭卡]]、[[多哥|多哥]]以及[[委內瑞拉玻利瓦爾共和國|委內瑞拉玻利瓦爾共和國]]<ref>{{cite web |title=成員國提交至聯合國大會第三委員會的訂正決議草案 |url=https://undocs.org/ch/A/C.3/72/L.36/Rev.1 |date=2017-11-14 |publisher=[[聯合國大會|聯合國大會]] |accessdate=2018-10-01 |language=zh-hans}}</ref>。

=== 審議 ===
第三委員會於2017年11月16日審議早前提交的決議草案，通過「A/C.3/72/L.36/Rev.1」決議草案；之後，[[阿富汗|阿富汗]]、[[安哥拉|安哥拉]]、[[奧地利|奧地利]]、[[孟加拉國|孟加拉國]]等共79位成員國加入成為提案國，總共有98位提案成員國<ref>{{cite web |title=聯合國大會第三委員會報告 |url=https://undocs.org/zh/A/72/439 |date=2017-12-01 |publisher=[[聯合國大會|聯合國大會]] |accessdate=2018-10-01 |language=zh-hans |archive-url=https://web.archive.org/web/20171229045052/http://undocs.org/zh/A/72/439 |archive-date=2017-12-29 |dead-url=no }}</ref>。

== 大會全體決議 ==
在2017年12月19日的聯合國大會第七十二屆會議第七十三次全體會議上，表決通過決議草案「A/C.3/72/L.36/Rev.1」並訂立於2018年開始的每年9月23日為「國際手語日」<ref>{{cite web |title=聯合國大會第七十二屆會議第七十三次全體會議決議通過文件 |url=http://undocs.org/zh/A/RES/72/161 |date=2017-12-19 |publisher=[[聯合國大會|聯合國大會]] |accessdate=2018-10-01 |language=zh-hans}}</ref>。

== 設立目的 ==
世界聾人聯合會設立「國際手語日」的目的在於保存手語以及聽障人士的權利，而選擇9月23日是因為此日同時是世界聾人聯合會於1951年的成立日<ref>{{cite web |script-title=en:United Nations declared 23 September as International Day of Sign Languages |trans-title=聯合國通過並訂立每年9月23日為國際手語日 |url=https://wfdeaf.org/news/un-23-sept-as-international-day-of-sign-languages/ |publisher={{link-en|世界聾人聯合會|World Federation of the Deaf}} |archiveurl=https://web.archive.org/web/20181001185207/https://wfdeaf.org/news/un-23-sept-as-international-day-of-sign-languages/ |archivedate=2018-10-01 |deadurl=no |accessdate=2018-10-01 |language=en }}</ref>。

認識或學習手語可以使聽障人士在學校、醫院以及法院等公眾地方打破交流障礙，並使聽障人士及兒童都能夠利用手語來接受教育；世界各地的聽障人士到訪公眾場合時均會受到障礙，南非一名男子前往醫院檢查[[人類免疫缺陷病毒|HIV]]時，醫生除了在紙張上寫上呈陽性以及要求他離開外，並沒有得到任何支援；在面臨衝突、失去家園、或其他需人道救援的情況時，聽障人士遇到的障礙將會增加並得不到所需的支援<ref>{{cite web |script-title=en:Sign Language Key to Deaf People’s Rights |trans-title=手語是聽障人士的權利 |url=https://www.hrw.org/news/2018/09/23/sign-language-key-deaf-peoples-rights |publisher=[[人權觀察|人權觀察]] |date=2018-09-23 |archiveurl=https://web.archive.org/web/20181002103534/https://www.hrw.org/news/2018/09/23/sign-language-key-deaf-peoples-rights |archivedate=2018-10-02 |deadurl=no |accessdate=2018-10-02 |language=en}}</ref><ref>{{cite web |script-title=en:A Call for Inclusion on International Day of Sign Languages |trans-title=包容手語的國際日子 |url=https://www.hrw.org/news/2018/09/23/call-inclusion-international-day-sign-languages |publisher=[[人權觀察|人權觀察]] |date=2018-09-23 |archiveurl=https://web.archive.org/web/20181002110250/https://www.hrw.org/news/2018/09/23/call-inclusion-international-day-sign-languages |archivedate=2018-10-02 |deadurl=no |accessdate=2018-10-02 |language=en}}</ref>。

而手語亦可以與口語一樣是可以共存，手語可以令聽障人士參與、貢獻社會盡一份力，對聽障人士而言是至關重要<ref>{{cite web |title=學用手語，人人參與！ |url=https://www.mda.org.mo/index.php/component/content/article/26-mda/mda-deaf-news/719-719 |publisher=澳門聾人協會 |date=2018-09-27 |archiveurl=https://web.archive.org/web/20181002122913/https://www.mda.org.mo/index.php/component/content/article/26-mda/mda-deaf-news/719-719 |archivedate=2018-10-02 |deadurl=no |accessdate=2018-10-02 |language=zh-hant }}</ref>。

== 各地活動 ==
世界各地均有「國際手語日」的相關活動，如世界聾人協會亞洲區秘書處聯同日本大使館以及{{link-en|日本財團|Nippon Foundation}}於2018年9月23日舉行活動推廣手語、澳門聾人協會於2018年9月23日舉行「2018國際聾人週及國際手語日」等，讓更多人認識手語以及聽障人士使用手語的權利<ref>{{cite web |script-title=en:International Day of Sign Languages Commemorative Event |trans-title=國際手語日紀念活動 |url=https://www.youtube.com/watch?v=dq5xDR9l25g |publisher=世界聾人協會亞洲區秘書處 |date=2018-09-23 |accessdate=2018-10-02 |language=en }}</ref><ref>{{cite web |title=「2018國際聾人週及國際手語日」活動開幕禮 |url=https://www.mda.org.mo/index.php/component/content/article/26-mda/mda-deaf-news/715-715 |publisher=澳門聾人協會 |date=2018-09-21 |archiveurl=https://archive.today/20181002123856/https://www.mda.org.mo/index.php/component/content/article/26-mda/mda-deaf-news/715-715 |archivedate=2018-10-02 |deadurl=no |accessdate=2018-10-02 |language=zh-hant }}</ref>。
