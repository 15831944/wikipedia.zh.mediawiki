'''国后岛'''（{{jpn|j=国後島|hg=くなしりとう|rm=Kunashiritō|f=y}}）<ref>{{cite web|title=萨哈林州自然环境|url=http://www.crc.mofcom.gov.cn/article/wangshangtuijie/200408/42535_1.html|website=中俄投资促进网|accessdate=2017-04-08|date=2004-08-26}}</ref>是[[俄罗斯|俄罗斯]][[远东联邦管区|远东联邦管区]][[萨哈林州|萨哈林州]][[千島群島|千島群島]]最南端的一座[[島嶼|島嶼]]，俄羅斯與日本兩國同時宣稱擁有此島主權（也就是所謂[[北方四島爭議|北方四島爭議]]），但實際上是由俄羅斯治理，並在岛上还驻有常备部队[[俄罗斯第18机枪炮兵师|俄罗斯第18机枪炮兵师]]。该岛無論俄文還是日文名稱都是源自此地[[原住民|原住民]]語言[[阿伊努語|阿伊努語]]中的「kunne-sir」（{{lang|ja|クンネ・シリ}}，意指「黑島」）或「kina-sir」（{{lang|ja|キナ・シリ}}，意指「草島」） <ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/4150.pdf|title=アイヌ語地名リスト|accessdate=2017-08-16|author=北海道庁|date=|publisher=北海道庁}}</ref>。

國後島的地理位置在[[北海道岛|北海道岛]]东北，島西与[[知床半岛|知床半岛]]上的[[羅臼町|羅臼町]]隔[[根室海峡|根室海峡]]相望，島東北為[[國後水道|國後水道]]。

日本方面在行政上於該島設置有[[北海道|北海道]]所管[[國後郡|國後郡]]，下轄分別管轄該島西南半部的[[泊村_(北海道根室振興局)|泊村]]及東北半部的[[留夜別村|留夜別村]]兩個村級行政區。但因日本並未實際治理該島，因此此一郡兩村目前都是虛設行政單位。

== 地理 ==
东北－西南延伸123公里，宽4-30公里，面積為1498.56[[平方公里|平方公里]]，[[南千岛群岛|南千岛群岛]]中面積僅次於[[擇捉島|擇捉島]]。地形陡峻，最高峰为火山[[爷爷岳|爷爷岳]]，海拔1822米，1973年7月曾大喷发。2006年全岛人口6801人，2018年人口為8531人。国后岛上的最大城鎮為[[南庫里爾斯克|南庫里爾斯克]]（過去在日治時期為[[國後郡|國後郡]][[泊村_(北海道根室支廳)|泊村]][[南庫里爾斯克|古釜布]]），最北部城镇[[近布内|近布内]]（{{lang-ru|Отрада}}）也有少量人口分布。地質上為千島火山帶之最南端。

2017年12月，日本政府「地震調查委員會」發布地震預測<ref>{{cite web|url=https://news.ebc.net.tw/news.php?nid=91044|title=遊日注意！北海道恐爆規模9強震 日官方：非常緊迫|publisher=ebc|accessdate=2017-12-20|date=2017-12-19|language =zh-tw}}</ref>，預測該區可能會發生8.8規模強震<ref>{{cite web|url=https://news.ftv.com.tw/news/detail/2017C19I15M1|title=日研究：北海道千島海溝 恐現8.8規模強震|publisher=ftv|accessdate=2017-12-20|date=2017-12-19|language =zh-tw}}</ref>。

== 歷史 ==
[[第二次世界大戰|第二次世界大戰]]結束後的1945年8月底9月初，[[蘇聯|蘇聯]]根据[[雅尔塔协定|雅尔塔协定]]，佔領含此岛在内的[[南千岛群島|南千岛群島]]。<ref>新聞速報【中央社】,"中俄國後島養海參 要日吃鱉？"{{cite web |url=http://news.chinatimes.com/world/11050404/132011021501394.html |title=存档副本 |accessdate=2012-10-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20110218045220/http://news.chinatimes.com/world/11050404/132011021501394.html |archivedate=2011-02-18 }},[[中國時報|中國時報]],2011-02-15.</ref>

===人口退減===
1990年代由於經濟狀況不佳，人口曾經減少。1994年10月4日北海道東方發生M<sub>w</sub>8.2地震，[[南庫里爾斯克|南庫里爾斯克]]受到三公尺高的海嘯襲擊。該次地震約使三分之一的島民離開国后岛。
