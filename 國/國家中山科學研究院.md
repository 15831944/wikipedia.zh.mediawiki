'''國家中山科學研究院'''（簡稱：'''中山科學研究院'''、'''中科院'''，英文簡稱：NCSIST），為[[中華民國|中華民國]]國防科技研發機構，屬於[[中華民國國防部|國防部]]監督之[[行政法人|行政法人]]。主要目標為提升國防科技能力、建立自主國防工業、拓展國防及軍民通用技術<ref>{{Cite web |url =http://law.moj.gov.tw/LawClass/LawContent.aspx?PCODE=F0000083 |title = 國家中山科學研究院設置條例 |author =[[中華民國國防部|中華民國國防部]] |date =2014-01-29  |publisher = [[中華民國法務部|中華民國法務部]] |language = zh-tw |accessdate = 2017-06-10 |quote =}}</ref>。

== 歷史 ==
===沿革===
*1965年4月9日：[[中華民國國防部|中華民國國防部]]成立「石門科學研究院籌備處」。
*1965年11月12日，命名為「中山科學研究院籌備處」<ref>{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=[[國史館|國史館]] |year=2012 |isbn=978-986-03-3586-6 }}</ref>。
*1969年7月1日，正式成立「中山科學研究院」。
*2004年3月1日，改隸[[國防部軍備局|國防部軍備局]]，更名為「國防部軍備局中山科學研究院」。
*2014年4月16日，改制為[[行政法人|行政法人]]，更名為「國家中山科學研究院」。
