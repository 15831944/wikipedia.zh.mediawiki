[[欧盟|欧盟]]的'''國際求救電話'''號碼是112。[[美国|美国]]、[[加拿大|加拿大]]的求救电话是[[緊急救助電話|911]]。[[中国大陆|中国大陆]]、[[日本|日本]]、[[台灣|台灣]]的求救电话是[[緊急救助電話|110]]或[[緊急救助電話|119]]。英聯邦國家／地區和前英聯邦的地區、[[香港|香港]]及[[澳門|澳門]]的求救號碼是[[999_(緊急電話號碼)|999]]。

绝大多数使用[[GSM|GSM]]的[[移动电话|手機網路]]的系統，发生紧急情况时，只要在手機是有電力、且可收到任何一個移動電話網路訊號的狀況下，就算手機沒有插入[[用户身份模块|移動電話卡]]或被鎖定，仍可免費直撥「112」發出求救訊號，語音應答系統就會為求援者轉接至國家的緊急救助電話中心。

這種緊急求救的功能使全球GSM移動電話網路和手機的安全更多了一項保障，并广泛使用于欧盟地区。在其他國家，112會直接接通每個國家的正確求救電話。

== 中国大陆 ==
中国大陆的紧急号码为110（匪警）、119（火警）、120（急救）和122（交通故障），在手机无信号时拨打可能会无法接通，并将听到和下文相同的录音；北京地区还可拨打999接入[[北京市红十字会紧急救援中心|北京市红十字会紧急救援中心]]<ref>{{Cite web|url=http://www.beijing999.com.cn/infoshow.aspx?id=304|title=999急救中心北京市红十字会紧急救援中心简介|accessdate=2018-08-24|work=www.beijing999.com.cn}}</ref>；而112和911，无论有没有信号，固定电话和移动电话拨打后听到的均为录音。

在[[中国移动|中国移动]]网络下拨打112或911，以及无SIM卡时拨打110、119、120或122时，聽到的錄音内容是：「你好，匪警请拨110，火警请拨119，急救中心请拨120，交通故障请拨122。For Police dial 110, to report the Fire dial 119, for an Ambulance dial 120, to report the Traffic Accident dial 122.」；在北京以外地区拨打999，听到的自动语音为：「您的号码有误，请查证后再拨。The number is incorrect. Please check and dial again.」在重複第二次錄音內容之後手機便會自動掛線。

在[[中国联通|中国联通]]网络下拨打112、911或999（北京以外），以及无SIM卡时拨打110、119、120或122时，聽到的錄音内容是：「你好，匪警请拨110，火警请拨119，急救中心请拨120，交通故障请拨122。For Police dial 110, to report the Fire dial 119, for an Ambulance dial 120, to report the Traffic Accident dial 122.」在重複第二次錄音內容之後手機便會自動掛線。

== 香港 ==
在香港境內用手機撥打112將會轉駁至[[香港警務處|香港警務處]]的999[[指揮及控制中心|指揮及控制中心]]，流動電話網絡訊號微弱的[[香港郊野公園|郊野公園]]或鄉郊地方均可使用此緊急電話號碼。

== 中華民國 ==
在中華民國，早期將112用作電話障礙申告用。引進GSM後，為與GSM標準一致，因此將電話障礙申告納入市內電話的服務專線號碼123之內，手機撥打112便與國際一樣用作緊急求救電話。接通112後需依語音指示，按"0"以轉接到當地警察局（110）或按"9"以轉接到當地消防局（119）。

如手機具有SIM卡而能正常通話，也可以直接撥110或119：
*119 當地消防局。火災、急難救助等。
*110 當地警察局。治安事件、交通事故等報警或求援。
