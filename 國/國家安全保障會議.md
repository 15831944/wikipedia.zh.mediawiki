'''國家安全保障會議'''（{{jpn|j=国家安全保障会議|hg=こっかあんぜんほしょうかいぎ}}、{{lang-en|National Security Council}}<ref>[http://www.mod.go.jp/e/about/organization/chart_a.html About Ministry Organization Chart] 防衛省公式サイト</ref>、縮寫：{{lang|en|'''NSC'''}}）是[[日本行政機關|日本行政機關]]之一，隸屬於[[內閣_(日本)|內閣]]。依據{{link-ja|國家安全保障會議設置法|国家安全保障会議設置法}}，本會議負責審議國家安全重要事項與處理重大[[緊急情況|緊急情況]]，類似於[[美國國家安全會議|美國國家安全會議]]（NSC）。

[[主任大臣|主任大臣]]與[[議長|議長]]由[[內閣總理大臣|內閣總理大臣]]擔任。內閣總理大臣與部分[[國務大臣|國務大臣]]分別組成四大臣會議、九大臣會議、緊急大臣會議等三組會議。

== 概要 ==
國家安全保障會議設於內閣，是審議國家安全保障重要事項的機關（國家安全保障會議設置法1條）、協助首相擬定決策與政治決定<ref name=Tsuite>{{Cite web |date= |url=http://www.kantei.go.jp/jp/singi/ka_yusiki/dai6/siryou1.pdf |title=「国家安全保障会議」について（「国家安全保障会議の創設に関する有識者会議」説明資料） |publisher=内閣官房 国家安全保障会議設置準備室 |accessdate=2013-12-24}}</ref>。國家安全保障會議的組織由會議與事務組織組成。

國家安全保障會議的中樞是首相、官房長官、外相、防衛相組成的「'''四大臣會議'''」（{{jpn|j=四大臣会合}}）。該會議每月舉行2次，決定安全相關政策與擬定對外政策基本方向<ref name="Tsuite"/>。必要時舉行同前身[[#國防會議與安全保障會議|安全保障會議]]相同組成的「'''九大臣會議'''」」（{{jpn|j=九大臣会合}}），以多方面的觀點審議國防方針與重大緊急情況<ref name="Tsuite"/>。與會者除了4大臣外還加入副總理、總務大臣、財務大臣、經產大臣、國交大臣、國家公安委員長。緊急事態發生時，總理、官房長官以及總理指定之大臣召開「'''緊急事態大臣會議'''」（{{jpn|j=緊急事態大臣会合}}）<ref name="Tsuite"/>。此會議除大臣外，在總理大臣許可下還可加入統合幕僚長等相關人士<ref name="Tsuite"/>。

=== 諮詢事項 ===
內閣總理大臣應就以下事項向國家安全保障會議諮詢。但若發生武力攻撃或生存危機、重要影響及重大緊急情況時，特別是緊急情況的應對處理，由國家安全保障會議向內閣總理大臣提出建議（同法2條）。
* 國防基本方針
* {{link-ja|防衛計畫大綱|防衛計画の大綱}}
* 防衛計畫相關產業的調整計畫大綱
* 武力攻撃事態等<ref group="注">武力攻撃事態與武力攻撃預測事態</ref>或生存危機事態的對應方針
* 武力攻撃事態等或生存危機事態的對應重要事項
* 重要影響事態的對應重要事項
* 自衛隊法第3條第2項第2號的自衛隊活動重要事項
* 其他國防重要事項
* 國家安全保障相關外交政策及防衛政策基本方針與相關重要事項
* 其他國家安全保障相關外交政策及防衛政策基本方針與其政策之重要事項

=== 會議成員 ===
[[File:Abe_Shinzo_2012_02.jpg|thumb]]]]
家安全保障會議新設四大臣會議與緊急事態大臣會議。九大臣會議則是延續前身的安全保障會議。依據國家安全保障會議設置法3條規定，會議由議長與議員組成。
* 議長：[[內閣總理大臣|內閣總理大臣]]
:※內閣總理大臣空缺時由{{link-ja|內閣總理大臣臨時代理|内閣総理大臣臨時代理}}執行職務。

:※必要時，議長可要求其他[[國務大臣|國務大臣]]針對特定議案出席會議。另外，{{link-ja|統合幕僚長|統合幕僚長}}等[[自衛隊|自衛隊]]相關人士也可出席會議表達意見。但由於屬於列席者，無法參與表決。
* 幹事：為議長、議員的協助者。由內閣任命相關行政機關職員擔任，非正職，人數在10人以內。

=== 國家安全保障局 ===
[[File:Shotaro_Yachi_at_the_Pentagon_Jan_2014.jpg|thumb]]]]
'''國家安全保障局'''（{{jpn|j=国家安全保障局|hg=こっかあんぜんほしょうきょく}}、{{lang-en|National Security Secretariat}}<ref>[http://japan.kantei.go.jp/tyoukanpress/201512/3_p.html 内閣官房長官記者会見 2015年12月3日（木曜日） 英語版]</ref>、簡稱：{{lang|en|'''NSS'''}}）是隸屬於內閣官房，協助國家安全保障會議的事務局。國家安全保障局負責省廳間協調、緊急狀況的政策提議、中長期外交安保政策立案、向外務省、防衛省、警察廳、公安調査廳、經濟產業省、國土交通省、[[內閣情報調査室|內閣情報調査室]]等各省廳與情報體系要求情報、各省庁都有義務向國家安全局報告<ref name="Tsuite"/>。

國家安全保障局長常與{{link-ja|內閣危機管理監|内閣危機管理監}}合作<ref name="Tsuite"/>。本職位由[[內閣|內閣]]應[[內閣總理大臣|內閣總理大臣]]要求進行任免。

2014年（平成26年）1月7日，國家安全保障局以67名體制成立。初代局長是曾任外務事務次官、政府代表、{{link-ja|內閣官房參事|内閣官房参与}}的[[谷內正太郎|谷內正太郎]]，同時兼任{{link-ja|內閣顧問|内閣顧問|內閣特別顧問}}<ref name=Mainichi0107>{{Cite news|title=国家安全保障局:発足、初代局長・谷内氏に辞令|newspaper=毎日新聞|date=2014-01-07|author=|url=http://mainichi.jp/select/news/20140107k0000e010207000c.html|accessdate=2014-01-07|deadurl=yes|archiveurl=https://web.archive.org/web/20140107065034/http://mainichi.jp/select/news/20140107k0000e010207000c.html|archivedate=2014-01-07}}</ref>。局長下轄防衛省與外務省出身、由{{link-ja|内閣官房副長官補|内閣官房副長官補}}兼任的2名局次長，以及同省出身的3名[[審議官|審議官]]。局內分為6班，包含負責綜合事務與國家安全保障會議事務的「總括・調整班」、負責[[美國|美國]]、[[歐洲|歐洲]]各國、[[ASEAN|ASEAN]]等的「政策第1班」、負責[[東北亞|東北亞]]與[[俄羅斯|俄羅斯]]的「政策第2班」、負責[[中東|中東]]、[[非洲|非洲]]、中南美的「政策第3班」、負責防衛計畫大綱與國家安全保障戰略等中長期安全保障政策的「戰略企劃班」、負責機密情報相關省廳等政府內聯繫的「情報班」<ref>[http://www3.nhk.or.jp/news/html/20140107/k10014323371000.html 首相 国家安全保障局の意義を強調] {{webarchive|url=https://web.archive.org/web/20140111013319/http://www3.nhk.or.jp/news/html/20140107/k10014323371000.html |date=2014-01-11 }} NHKニュース 2013年1月7日</ref>。由於與內閣情報調査室合作密切，通常情報班的班長多由警察廳出身的內閣情報調査室成員擔任<ref name=sankei1114>{{Cite news |title=NSCの組織編成全容判明　部門長は防衛省3　外務2、警察1　内調とも連携 |newspaper=産経新聞 |date=2013-11-09 |author= |url=http://sankei.jp.msn.com/politics/news/131109/plc13110910540002-n2.htm |accessdate=2013-11-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20131112171407/http://sankei.jp.msn.com/politics/news/131109/plc13110910540002-n2.htm |archivedate=2013-11-12 }}</ref>。

初代局長谷內正太郎曾任外務事務次官，有助於國家安全保障局的外交與[[外務省|外務省]]一體化，以外務省的別動隊與一般公家機關難以接觸的對象進行聯繫<ref>{{cite news|title=別働隊超えた“第1外務省”　NSC事務局　プーチン氏側近と会談　中国にも接触　ソウルで意見交換　隠密外交フル稼働|newspaper=産経新聞|date=2014-06-14|url=http://sankei.jp.msn.com/politics/news/140614/plc14061411080016-n1.htm|accessdate=2014-06-14|deadurl=yes|archiveurl=https://web.archive.org/web/20140614060824/http://sankei.jp.msn.com/politics/news/140614/plc14061411080016-n1.htm|archivedate=2014-06-14}}</ref>。

國家安全保障局的所在地是東京都千代田區永田町2丁目4-12（[[內閣府|內閣府]]廳舍別館）。該大樓是1971年（昭和46年）建設的民間大樓，依照日本政府耐震基準是無法確保人命安全的「3類」，因此被認為是災害時危機管理的問題點之一。對此，日本政府決定將國家安全保障局移往新廳舍。新廳舍建設預定地是[[內閣官房|內閣官房]]與內閣府入駐的{{link-ja|中央合同廳舍第8號館|中央合同庁舎第8号館}}東側（東京都千代田區永田町1丁目4）<ref>{{cite news|title=政府、新合同庁舎を建設へ　ＮＳＣ事務局を移設|publisher=産経ニュース|url=http://www.sankei.com/politics/news/151022/plt1510220003-n1.html|date=2015-10-22|accessdate=2017-04-22|archiveurl=https://web.archive.org/web/20151022140752/http://www.sankei.com/politics/news/151022/plt1510220003-n1.html|archivedate=2015年10月22日}}</ref><ref>{{cite news|title=菅長官激怒、すくむ経産省　報道対応、自ら突然の規制|publisher=朝日新聞デジタル|url=http://www.asahi.com/articles/ASK4F35ZFK4FUTFK002.html|date=2017-04-22|accessdate=2017-04-22|archiveurl=https://web.archive.org/web/20170422003623/http://www.asahi.com/articles/ASK4F35ZFK4FUTFK002.html|archivedate=2017年4月22日}}</ref>。

== 設立過程 ==
=== 第1次安倍內閣嘗試創立 ===
2006年，[[第1次安倍內閣|第1次安倍內閣]]提議創立'''國家安全保障會議'''（日本版NSC）以替代既有的安全保障會議た。

做為起點，時任[[內閣總理大臣|內閣總理大臣]][[安倍晉三|安倍晉三]]召開國家安全保障官邸機能強化會議。會議成員包括議長代理[[小池百合子|小池百合子]]（{{link-ja|內閣總理大臣補佐官|内閣総理大臣補佐官}}（安全保障擔當））、議員[[鹽崎恭久|鹽崎恭久]]（[[內閣官房長官|內閣官房長官]]）、[[岡崎久彥|岡崎久彥]]（前駐泰國大使）、{{link-ja|小川和久|小川和久}}、[[森本敏|森本敏]]（[[拓殖大學|拓殖大學]]教授）、{{link-ja|柳井俊二|柳井俊二}}（前駐美大使）、[[北岡伸一|北岡伸一]]（東京大學教授）、{{link-ja|佐佐淳行|佐々淳行}}（前內閣安全保障室長）、{{link-ja|佐藤謙|佐藤謙}}（前{{link-ja|防衛事務次官|防衛事務次官}}）、[[鹽川正十郎|鹽川正十郎]]（前官房長官）、{{link-ja|先崎一|先崎一}}（前統合幕僚長）。

會議從2007年2月開始，每兩周舉行一次。安倍內閣在第166回國會向[[日本眾議院|日本眾議院]]提案安全保障會議設置法等修改案（安保會議設置法修正案）。此修正案包括將「安全保障會議」改名為「國家安全保障會議」、將審議事項擴大為國家安全相關事項，允許安排專門會議，會議設立事務局等。

=== 福田康夫內閣放棄設立 ===
但在166回國會召開後兩周，安倍晉三因[[潰瘍性結腸炎|潰瘍性結腸炎]]辭去總理大臣。繼任總理是國家觀點相異的[[福田康夫|福田康夫]]，加上民主黨等在野黨取得參議院過半数，法案前景不甚樂觀。2007年12月24日，[[福田康夫內閣|福田康夫內閣]]以「現有的安全保障會議具有足夠機能」為理由，放棄設立國家安全保障會議。

另一方面，[[自由民主黨_(日本)|自由民主黨]]的防衛省改革小委員會（[[濱田靖一|濱田靖一]]委員長）仍持續提出國家安全保障會議（日本版NSC）的設立<ref>{{cite web |url=http://www.jimin.jp/jimin/seisaku/2008/seisaku-008.html |title=存档副本 |accessdate=2015-01-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20090326132123/http://www.jimin.jp/jimin/seisaku/2008/seisaku-008.html |archivedate=2009-03-26 }}</ref>，並加入自民黨的防衛大綱提案之中。

[[第45屆日本眾議院議員總選舉|第45屆日本眾議院議員總選舉]]，自民黨大敗。2009年9月[[民主黨_(日本)|民主黨]]上台。民主黨在2010年11月24日由黨的外交防衛調査會提議設立國家安全保障室（NSO），之後外交防衛調査會也提出成立國家安全保障會議（日本版NSC），但並沒有具體行動。

== 國防會議與安全保障會議 ==
[[File:Yasuhiro_Nakasone_in_Andrews_cropped.jpg|thumb]]]]
=== 歷史 ===
* 1954年（昭和29年）7月1日：規定內閣設置國防會議的{{link-ja|防衛廳設置法|防衛庁設置法}}生效。但其第43條規定「國防會議組成與其他相關必要事項由其他法律訂定。」，無法立即成立。
* 1956年（昭和31年）7月2日：實施「國防會議組成法」，內閣設置'''國防會議'''。同時總理府設置國防會議事務局。
* 1957年（昭和32年）8月1日：依內閣法等修正案（昭和32年法律第158號），國防會議事務局成為國防會議直屬事務局。
* 1986年（昭和61年）7月1日：廢止國防會議，依安全保障會議設置法設置'''安全保障會議'''。
* 2007年（平成19年）：提出將安全保障會議重組為國家安全保障會議的安全保障會議設置法修正案，翌年廢止。

=== 安全保障會議成員 ===
成員與國家安全保障會議的九大臣會議相同。
* 議長：[[內閣總理大臣|內閣總理大臣]]
:※必要時，議長可要求其他[[國務大臣|國務大臣]]針對特定議案出席會議。另外，{{link-ja|統合幕僚長|統合幕僚長}}等[[自衛隊|自衛隊]]相關人士関係者也可出席會議表達意見。但由於屬於列席者，無法參與表決。
* 議員：{{link-ja|內閣法|内閣法}}第九條規定之第一順位指定大臣（[[副總理_(日本)|副總理]]）、[[總務大臣|總務大臣]]、[[外務大臣_(日本)|外務大臣]]、[[財務大臣|財務大臣]]、[[經濟產業大臣|經濟產業大臣]]、[[國土交通大臣|國土交通大臣]]、[[防衛大臣|防衛大臣]]、[[內閣官房長官|內閣官房長官]]、[[國家公安委員會委員長|國家公安委員會委員長]]
* 幹事：為議長、議員的協助者。由內閣任命相關行政機關職員擔任，非正職，人數在10人以內。
