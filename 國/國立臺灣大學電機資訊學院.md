'''國立臺灣大學電機資訊學院'''（'''College of Electrical Engineering and Computer Science, National Taiwan University'''），原隸屬於工學院下，1997年8月成立臺大電機學院，2000年8月1日更名國立臺灣大學電機資訊學院。其下分設電機學群及資訊學群, 電機及資訊兩個學系, 分別位居大學入學招生第二類組的第一和第二志願。電機學群有電機工程學系暨研究所，光電工程學研究所，電信工程學研究所，電子工程學研究所，資訊學群包含資訊工程學系暨研究所，資訊網路與多媒體研究所，以及跨學群的生醫電子與資訊學研究所<ref>{{cite web |url=http://www.eecs.ntu.edu.tw/intro/super_pages.php?ID=intro1&Sn=1|title=國立臺灣大學電機資訊學院 |accessdate=2012-12-08}}</ref>。
相關單位有資訊電子研究中心, 物聯網研究中心，奈米機電系統研究中心，綠色電能研究中心。

==系所==
===電機工程學系暨研究所===
電機工程學系前身為1943年5月成立的「臺北帝國大學工學部電氣工學科」，1945年11月改制為國立臺灣大學工學院電機工程學系<ref name='EE'>{{cite web |url=http://www.ee.ntu.edu.tw/intro.php|title=國立臺灣大學電機工程學系|accessdate=2012-12-11}}</ref>。<br />
目前，電機學群的系所架構為一系五所，內含十一個教學研究分組，其中電機所含自動控制組、電力系統與電力電子組、計算機科學組；光電所含光電組；電信所含電波組、通訊與信號處理組、資料科學與智慧網路組；電子所含奈米電子組、積體電路與系統組、電子設計自動化組；生醫電資所含醫工組<ref>{{cite web |url=https://web.ee.ntu.edu.tw/research1.php|title=國立臺灣大學電機工程學系 研究領域發展|accessdate=2017-04-22}}</ref>。<br />
電機所包含「自動控制」、「電力系統與電力電子」、「計算機科學」三組。自動控制組研究領域包含：線性、非線性及離散事件動態系統、適應、強健、智慧等控制理論、精密伺服控制在國防及工業上之應用、[[機器人學|機器人學]]與[[自動化工程|自動化工程]]、[[遙測|遙測]]及全球定位技術、新興系統之智慧型自動化與控制。電力系統與電力電子組研究領域包含：[[電力系統|電力系統]]、電機控制、電力電子、[[智慧電網|智慧電網]]。計算機科學組研究領域：平行與向量編譯程式、網際網路與多媒體系統、計算機容錯、[[計算機結構|計算機結構]]、計算理論、資料庫與資訊勘測、行動計算、[[網路安全|網路安全]]與[[密碼學|密碼學]]<ref>{{cite web |url=http://www.ee.ntu.edu.tw/graduate/index.php?option=com_content&view=article&id=48&Itemid=192|title=國立臺灣大學電機工程學研究所|accessdate=2017-04-22}}</ref>。

===電信工程學研究所===
1997年8月電信工程學研究所成立，設置「電波」及「通訊與信號處理」兩組。電波組主要研究領域為[[天線|天線]]、[[電磁理論|電磁理論]]、[[微波|微波]]、毫米波電路、系統封裝及電磁相容等技術。「通信及訊號處理組」主要研究領域為[[通訊|通訊]]、[[信號處理|信號處理]]、[[光通訊|光通訊]]及多媒體網路。2016年增設「資料科學與智慧網路組」，主要研究領域為[[機器學習|機器學習]]、[[大數據|大數據]]分析、認知神經機器人學、資料隱私與訊息檢索、[[物聯網|物聯網]]、[[軟體定義網路|軟體定義網路]]、與網路功能虛擬化技術等<ref>{{cite web|url=https://comm.ntu.edu.tw/AboutGICE.php|title=國立臺灣大學電信工程學研究所|accessdate=2018-01-25}}</ref>。

===光電工程學研究所===

光電所前身是電機所光電組，1992年8月成立招收碩士班學生，1995年8月增設博士班。 
光電所研究領域包括：[[光纖通訊|光纖通訊]]、光纖光學、[[積體光學|積體光學]]、[[半導體雷射|半導體雷射]]及[[光偵測器|光偵測器]]、半導體光電現象、光電半導體材料及元件、光電子技術、雷射技術與光電應用、[[非線性光學|非線性光學]]、光電量測技術、光學設計、光電的生物醫學應用等<ref>{{cite web |url=http://gipo.ntu.edu.tw/p1about.php|title=國立臺灣大學光電工程學研究所 |accessdate=2012-12-09}}</ref>。

===電子工程學研究所===
2001年電子工程學研究所奉准成立。電子所分為「[[數位電路|數位積體電路與系統]]」、「[[類比電路|類比積體電路與系統]]」、「奈米電子」、「[[電子設計自動化|電子設計自動化]]」四組，研究重點以積體電路系統晶片設計及奈米電子元件設計與製程技術兩方面為主<ref>{{cite web |url=http://www.giee.ntu.edu.tw/portal/intro_history.php|title=國立臺灣大學電子工程學研究所 |accessdate=2012-12-09}}</ref>。

===生醫電子與資訊學研究所===
生醫電子與資訊學研究所結合電機與資訊領域，進行生物醫學之研究及教學。2006年8月，生醫電資所開始招收博士班，碩士班也於2007年8月開始招生。涵蓋範圍有生醫材料、生醫力學、生醫電子、生醫資訊、生醫影像、生醫光電以及臨床工程等主要領域<ref>{{cite web |url=http://www.bebi.ntu.edu.tw/web/about/about.jsp?no=CP1474856525375|title=國立臺灣大學生醫電子與資訊學研究所 |accessdate=2017-04-22}}</ref>。

===資訊工程學系暨研究所===
1977年資訊工程學系成立，1981年成立碩士班，1984年成立博士班。重點研究領域可分為計算機理論、[[軟體|計算機軟體]]、[[計算機系統|計算機系統]]、[[電腦網路|計算機網路]]、智慧系統、[[多媒體|多媒體]]系統、生醫資訊。<ref>{{cite web |url=https://www.csie.ntu.edu.tw/intro/super_pages.php?ID=intro&Sn=1|title=國立臺灣大學資訊工程學系暨研究所 |accessdate=2017-04-22}}</ref>。

===資訊網路與多媒體研究所===
1991年創立通訊與多媒體實驗室，2000年資工系加入電機資訊學院後，由電機系及資工系教授組成網路與多媒體研究所籌劃小組，2003年資訊網路與多媒體研究所成立。主要研究領域有多媒體技術與系統、影像視訊處理與分析、資訊網路與行動通訊、網際網路及社群網路、[[嵌入式系統|嵌入式系統]]、普及計算與優質生活應用。<ref>{{cite web |url=http://www.inm.ntu.edu.tw/intro/super_pages2.php?ID=intro1&Sn=1|title=國立臺灣大學資訊網路與多媒體研究所 |accessdate=2017-04-22}}</ref>

==系館==
1945年臺大電機系改制成立之初，當時地點設於現今之新生大樓位址，原為木造的日本式建築，並與土木系共同使用。1955工學院大樓落成後（今土木系系館）遷入電機系大部分實驗室。當時電機系大部份是在此棟建築的1、2、3樓西半邊，另一部份則在機械館的1樓使用3個房間<ref name='EE'></ref>。而今，電機系主要使用的系館有電機一館、電機二館、博理館、明達館。

===電機一館===
[[File:電機一館.JPG|thumb]]
1960年代，在當時的系主任多方奔走及系友捐款之下，電機系終於獲得臺大校方的認同與支持，於1968年1月由臺大校長[[錢思亮|錢思亮]]主持電機一館動土典禮。1969年6月，電機一館落成啟用，是當時難得一見、臺大第一棟的中央空調建築，同時也是臺大首度經由募集捐款所完成的建築<ref name='EE'></ref>。




===電機二館===
[[File:電機二館.JPG|thumb]]
由於電機系人員與研究設備的擴充，遂於1976年5月提出興建電機二館之規劃。1982年，教育部撥款，1983年電機二館第一期（東半部）工程順利動土施工，1985年8月，電機二館東半部落成，並於1986年1月遷入使用。1993年8月，電機二館二期工程（西半部）完工。其中一樓東側約五百坪用地，自一期工程完工後即由工學院圖書館使用，至1998年11月工學院圖書館遷入臺大新[[國立臺灣大學圖書館|總圖書館]]後，方從工學院交予電機學院<ref name='EE'></ref>。

===博理館（電機三館）===
博理館為電機系系友[[廣達電腦|廣達電腦]]公司董事長[[林百里|林百里]]捐贈，是一棟地下一層，地上七層之鋼筋混凝土建築，以丁掛磚、斬石子、格子樑、鋁玻璃帷幕牆、鋁花板為建材<ref>{{cite web |url=http://www.hoy.com.tw/ugC_ShowroomItem_Detail.asp?hidShowKindID=1&hidShowTypeID=0&hidShowCatID=0&hidShowID=30&hidBuildingType=07|title=國立臺灣大學電機資訊大樓 |accessdate=2017-04-22}}</ref>。
根據陳維昭校長回憶錄所載，大樓命名來自於捐贈人林百里名字「百里」的[[廣東話|廣東話]]諧音。2002年11月15日，臺灣大學創校74週年校慶餐會中，林百里與[[陳維昭|陳維昭]]校長共同簽約完成博理館捐贈儀式。2003年1月8日為大樓新建工程動土典禮，由林百里董事長、陳維昭校長、時任許博文院長共同主持動土奠基典禮。2003年10月20日舉行上樑典禮。2004年6月5日，林百里系友捐贈的電機系第三棟教學研究大樓—博理館舉行落成典禮。
博理館大樓，地下一層，地上七層，並有一間可容納200多人之階梯式會議廳。地下1樓為博理藝廊、讀書中心(系K)、娛樂室(系康)、系學會辦公室等。1樓主要為教學教室，2樓為重點與卓越計畫使用，3樓為院務辦公之用、4至6樓是實驗室、教師之研究室與討論室，7樓為電機系與廣達電腦公司共同合作計畫之用<ref name='EE'></ref>。

===明達館（電機四館）===
[[File:明達館.JPG|thumb]]
明達館為[[明基友達集團|明基友達集團]][[李焜耀|李焜耀]]董事長捐贈，捐贈金額為三億元。是一棟地上7層，地下1層之鋼筋混凝土建築。外形上擷取文學院的立面造型，並採用迴廊、拱門、十三溝面磚等素材；而高樓層則以格狀板和玻璃做為立面，格狀板具備遮陽之功能，呼應臺大建築景觀，也同時具備現代建築之機能考量。
明達館的正式簽約儀式於2004年9月29日舉行，並在2005年5月17日開工動土，2007年1月11日明達館落成啟用。「明達館」的意涵為「明理達義，明德達信」在建築的一角有[[孫中山|孫中山]]先生的此字帖書法<ref name='EE'></ref>。

明達館的地下1樓為停車場，1樓為公共空間，包括餐廳以及展示空間，2樓與3樓主要為教學與實驗空間，4樓為與明基及友達之產學合作空間，5至7樓為教師研究室與實驗室。

明達館前身是一般民宅，約在1989年11月，校地校地完成徵收，此處不再是私人產業。而在興築明達館前，基地上有間「臺大福華」自助餐廳<ref>{{cite web|url=http://gallery.lib.ntu.edu.tw/archives/482|title=點描明達館的前世今生|accessdate=2012-12-11|deadurl=yes|archiveurl=https://web.archive.org/web/20120621030250/http://gallery.lib.ntu.edu.tw/archives/482|archivedate=2012-06-21}}</ref>。

===德田館===
而資訊工程學系在1977年成立之初沒有自己的教室，1982年8月系館(今[[計算機及資訊網路中心|計算機及資訊網路中心]])建成遷入，1993年新系館完工(今德田館北側)，8月遷入。2003年，華宇電腦李森田董事長夫婦捐贈的資訊館二期工程(德田館)動工，並於2004年9月29日落成啟用<ref>{{cite web |url=http://www.csie.ntu.edu.tw/intro/super_pages.php?ID=intro&Sn=5|title=國立臺灣大學資訊工程學系暨研究所 |accessdate=2012-12-11}}</ref>。

==知名校友==
===學術界===
[[方復|方復]]：1951年電機系畢業，[[中央研究院|中央研究院]]院士、[[美國國家工程院|美國國家工程院]]院士、Fellow of APS、IEEE、Franklin Institute等。曾於IBM T. J. Watson Research Center從事[[半導體|半導體]]物理及元件之研究，曾獲得[[美國物理學會|美國物理學會]]（APS）巴克萊（Oliver E. Buckley）固態物理獎、IEEE沙若諾夫（David Saronov）獎、[[富蘭克林|富蘭克林]]學院金質獎章、德國宏保（Alexander von Humboldt）基金會研究獎。

[[傅京孫|傅京孫]]：1953年電機系畢業，美國[[普渡大學|普渡大學]]教授。[[電氣電子工程師學會|電氣電子工程師學會]]（[[IEEE|IEEE]]）會士、[[美國國家工程院|美國國家工程院]]院士、[[中央研究院|中央研究院]]院士，首屆國際[[模式識別|模式識別]]大會（International Conference Pattern Recognition，縮寫ICPR）主席、[[IEEE|IEEE]][[機器智能|機器智能]]與[[模式識別|模式識別]]委員會首任主席，[[模式識別|模式識別]]與[[機器智能|機器智能]]領域的先驅之一，被譽為「[[模式識別|模式識別]]之父」。

[[鄭文燦|鄭文燦]]：1987年電機系肄業轉系至社會系，臺大國家發展研究所碩士，曾任行政院新聞局局長、行政院發言人、海基會副秘書長、民進黨中常委、民進黨中執委、民進黨文宣部主任、桃園縣議員和野百合學運決策小組召集人。現任桃園市市長。

[[林宗男|林宗男]]：1989年電機系畢業，美國[[普林斯頓大學|普林斯頓大學]]電機工程學系的碩士及博士。[[財團法人資訊工業策進會|財團法人資訊工業策進會]]資安科技研究所所長、[[消費者文教基金會|消費者文教基金會]]副祕書長，國立臺灣大學電機工程學系教授。

[[林婉容|林婉容]]：1986年電機系畢業，福音詩歌創作歌手，1999年成立「小羊詩歌」網站並出版首張福音詩歌集《鼓舞》。
