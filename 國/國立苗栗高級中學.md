'''國立苗栗高級中學'''，簡稱'''苗栗高中'''、'''國立苗中'''，舊稱'''省立苗中'''，位於[[臺灣|臺灣]][[苗栗縣|苗栗縣]][[苗栗市|苗栗市]]，與[[苗栗高商|苗栗高商]]及[[建臺中學|建臺中學]]三校相鄰，是苗栗縣最早成立的國立高中。該校於103學年及104學年連續兩年，獲得[[教育部|教育部]]遴選為全國10所精進標竿高中之一。

== 校名演進 ==
*1941年4月，「苗栗家政女子學校」成立，翌年改為「苗栗農業實踐女學校」。
*1946年4月，國民黨政府接收台灣，改制「新竹縣立苗栗初級中學」。
*1949年9月，試辦高中部。
*1950年10月，苗栗設縣，改隸為「苗栗縣立苗栗初級中學」。
*1951年8月，改制完全中學，更名「苗栗縣立苗栗中學」。
*1952年8月，改隸省政府，為「台灣省立苗栗中學」。
*1962年8月，苗栗縣先行試辦「[[省辦高中，縣市辦初中|省辦高中，縣市辦初中]]」，初中部停止招生。
*1970年8月，改制為「臺灣省立苗栗高級中學」。
*1971年，增設空中補習學校。
*1972年，增設商科。
*1973年8月，商科撥交臺灣省立苗栗高級商業職業學校（即為現今的[[苗栗高商|國立苗栗高商]]）。
*2000年，改隸教育部，更名為「國立苗栗高級中學」。

== 學生自治組織 ==
===班聯會===
「班聯會」是苗栗高中的一個類屬社團的組織，但有別於其它單一性質的社團，班聯會負責的事務也廣泛許多，所以又稱「學生自治會」(MLSH student conucil)，透過民主的程序經由全校學生投票選出主席，正副主席及儲備幹部為全校性幹部，組成班聯會，組織內除正副主席外共分秘書、活動、公關、文書、美宣、總務、資訊七組。
每學期班聯會與由全校班級各選出一位班級代表，定期與班聯會幹部召開「班級聯合代表大會」討論學校學生事宜。

現在班聯會是第22屆（2018-2019），現任主席為莊諺麒，於2018年4月27日經選舉當選。

第23屆（2019-2020）候任主席為林子渝，於2019年4月19日經選舉當選。

== 校園建築 ==
*知行樓
:知行樓除了包含一樓行政用途的教官室、學務處、保健中心外，二至三樓為三年級學生教室，四至五樓則是二年級學生教室。
*北大樓
:北大樓的興建時間較知行樓晚，設備上也較為新穎，緊鄰著位於苗栗高中東側的正門，同時也是苗栗高中側門、司令台坐落之處。北大樓一樓為教務處、會客室、總務處與員生合作社所在；二樓除了校長室以外尚有會計室、輔導室、秘書室、輔導課教室一間以及會議室兩間；三樓為各科教師之辦公室；四至五樓則為一年級學生教室；六樓為美術科教室，同時還有一間美術科老師辦公室，此外苗栗高中之升旗台亦在此；七樓則有生活科技實驗室、音樂教室、客語數位教室、家政教室以及電腦教室。
*圖書館
:於2011年5月8日修建完畢，坐落於苗栗高中西南側。
*學生活動中心
:學生活動中心地下一樓為游泳池，一樓則是籃球場，旁邊有體育組辦公室，二樓為韻律教室所在，三樓即是禮堂，平常作羽球場之用。
*科學樓
:一至四樓有自然實驗室。

==籃球隊==
苗栗高中籃球隊在77學年度在崇尚進攻的[[林增福|林增福]]教練帶領下獲得首屆HBL冠軍，除了81學年度無緣八強外，78學年度至80學年度和82學年度至84學年度都是八強，目前由前國手[[賴國弘|賴國弘]]教練接掌兵符。

之後沉寂多年後，105學年度再度挑戰甲級，外卡戰59：74不敵[[漢英高中|漢英高中]]結束今年HBL賽季。

106學年度再度挑戰甲級，外卡戰73：94不敵[[宜蘭高中|宜蘭高中]]結束今年HBL賽季。
