'''國立臺灣海洋大學'''，簡稱'''[[臺灣海大|臺灣海大]]'''、'''[[臺海大|臺海大]]'''或'''[[海大|海大]]'''，是位於[[中華民國|中華民國]][[臺灣|臺灣]][[基隆市|基隆市]]的[[國立大學|國立大學]]，在[[教育部|教育部]]「[[高教深耕計畫|高教深耕計畫]]」裡參與全球鏈結卓越研究中心，近年連續獲得「[[邁向頂尖大學計畫|邁向頂尖大學計畫]]」補助，且該校亦為唯一每年（自2005年計劃開始起）榮獲「[[獎勵大學教學卓越計畫|獎勵大學教學卓越計畫]]」補助的國立大學，同時也是「[[臺北聯合大學系統|臺北聯合大學系統]]」成員之一。是[[臺灣|臺灣]]在[[海洋|海洋]]相關領域之最高學府，國際多項大學評比中是全世界海洋大學中排名第一。位於[[基隆市|基隆市]][[中正區_(基隆市)|中正區]]八尺門海濱，佔地三十餘公頃，倚山面海，與[[和平島|和平島]]及[[碧砂漁港|碧砂漁港]]等景點相臨。現有7個[[學院_(大學)|學院]]、15個[[學系|學系]]、11個獨立研究所、26個碩士班及19個博士班，另設有[[學士後|學士後]]商船及輪機學士學位學程。

== 簡介 ==
國立臺灣海洋大學隸屬[[教育部|教育部]]高等教育司，為[[臺灣|臺灣]]擁有海洋領域系所最為完整的大學，並為[[臺灣|臺灣]]此領域的最高學府與研究重鎮<ref>[http://www.ntou.edu.tw/1about_man.php 《海大首頁 > 治校理念》，國立臺灣海洋大學官網] {{webarchive|url=https://web.archive.org/web/20081011224553/http://www.ntou.edu.tw/1about_man.php |date=2008-10-11 }}</ref>，對[[臺灣|臺灣]][[航運|航運]]產業及海洋科技高級人才的培育貢獻卓著，[[臺灣|臺灣]]相關領域高階人才多畢業於此，該校專注於[[海事|海事]]、[[航運|航運]]、[[海洋法|海洋法]]、[[水產養殖|水產養殖]]、水產生技、[[海洋生物|海洋生物]]、漁業科學、[[食品科學|食品科學]]、造船工程、[[輪機工程|輪機工程]]、[[導航|導航]]、[[海洋科學|海洋科學]]、海洋工程與航海技術及所有海洋相關領域，並代表[[臺灣|臺灣]]的最高水準<ref>[http://ua.twaea.org.tw/school/final/940815%E5%8F%B0%E7%81%A3%E6%B5%B7%E6%B4%8B%E5%A4%A7%E5%AD%B8%E8%A9%95%E9%91%91%E5%A0%B1%E5%91%8A.pdf 《國立臺灣海洋大學評鑑報告》，社團法人臺灣評鑑協會，2005年8月15日]</ref>。目前設有海運暨管理學院、生命科學院、工學院、海洋科學與資源學院、電機資訊學院、人文社會學院以及海洋法律與政策學院等7個學院，下設15個學系，11個獨立研究所，26個碩士班，19個博士班。共有83個社團，11個運動代表隊。教師博士學位比91%。

由於研究成果斐然（請參見[[#科研及學術成就|科研及學術成就]]）且海洋立國的意識成為政府的口號、願景並獲民眾的認同，近年國家直接經費補助大幅增加。2007年起為參與[[中華民國教育部|教育部]]「[[繁星計畫|繁星計畫]]」25所大學之一。自2005年起每年皆獲得[[教育部|教育部]]之「[[獎勵大學教學卓越計畫|獎勵大學教學卓越計畫]]」補助。2006年起獲得教育部「[[發展國際一流大學及頂尖研究中心計劃|發展國際一流大學及頂尖研究中心計劃]]」（五年五百億）第一階段補助經費1.4億元，在獲補助之全國17校中金額排名第13；2008年起獲得此計劃第二階段補助經費每年0.9億元，在獲補助之全國15校中金額排名第12。2009年海大推廣產學合作獲教育部三年2700萬補助。2011年名列2011-2012年，世界前400名頂尖大學。2014年加入[[臺北聯合大學系統|臺北聯合大學系統]]。

由於部分學校升格後之校名與國立臺灣海洋大學校名近似，特別是簡稱，常常令人混淆。此自2017年5月3日起，國立臺灣海洋大學正式由經濟部智慧財產局取得「國立臺灣海洋大學」、「台海大」、「臺海大」、「海大」及「校徽」圖之專用商標權利，限制日後其他大專院校不得再使用該簡稱<ref>[https://www.tipo.gov.tw/mp.asp?mp=1 經濟部智慧財產局]</ref>。

== 校區 ==
校區可分三部分，分別為'''濱海校區'''、'''龍崗校區'''、'''祥豐校區'''，除祥豐校區及部分龍崗校區外，其餘部分校地係自1957年起開始分階段填海而形成，校地在陸續擴增下, 至1984年方形成現今校地之雛形。全校倚山傍海環境絕美、景色秀麗、規劃維護優良，可在校內高樓觀賞一望無際的太平洋海景；而濱海校區北端（小艇碼頭東側綠地及工學院後方）海堤欣賞海景夕陽令人流連忘返，是臺灣少數幾間臨海之大學校院，亦為北部新人婚紗拍攝取鏡及旅遊賞景的著名勝地。
*'''濱海校區'''大門口有一海大的象徵——海鷗雕像，為著名藝術家[[朱銘|朱銘]]為海大所作，濱海校門外即為[[東海|東海]]，在校區操場即可觀賞東海夕照，夜晚點點漁火，美不勝收。每年四、五月間，龍崗校區更有滿山遍野的[[螢火蟲|螢火蟲]]，為臺灣北部賞螢火蟲的重要景點之一，且是臺灣北部生態多樣性數一數二的蝴蝶棲地之一。
*'''龍崗校區'''步道的最高點為[[槓子寮山|槓子寮山]]（海拔163公尺，二等三角點#1168基石），其上有列屬國家三級古蹟的[[槓子寮砲臺|槓子寮砲臺]]，與基隆港週邊各要地之[[二沙灣|二沙灣]]、[[獅球嶺|獅球嶺]]、[[白米甕|白米甕]]等砲臺形成犄角之勢，護衛[[基隆港|基隆港]]，此處可俯望校區、[[基隆港|基隆港]]、[[基隆嶼|基隆嶼]]、[[碧砂漁港|碧砂漁港]]、美景盡收眼底<ref>[https://web.archive.org/web/20071123214253/http://www.ntou.edu.tw/include2/map.php 海洋大學全球資訊網  校園導覽]</ref>。在面對碧砂漁港的絕壁上分布著「東亞最美麗的百合花」之稱但已屬嚴重瀕臨絕滅之保育類植物「[[豔紅鹿子百合|豔紅鹿子百合]]」的原生群落<ref>[https://web.archive.org/web/20110401205727/http://econgis.forest.gov.tw/rareplant/species.asp?id=t00122 《豔紅鹿子百合》--台灣的稀有及瀕危植物資料庫__農委會自然資源與生態資料庫]</ref>。
由於校區所在之基隆地區在過去為多雨的天氣型態，為因應此狀況，校區之建築規劃採人性化考量，在建物大樓間以風雨走廊、天橋、地下道等方式相連接，在校內不用雨具可通行全校八成以上的建物及大樓。

另外還有觀音校區，位在桃園市觀音區，也正在籌備中。[http://news.ltn.com.tw/news/life/breakingnews/2110840 ]

== 歷史 ==
1953年春，[[鄧傳楷|鄧傳楷]]為臺灣省教育廳廳長，時臺省百廢待興，目睹若干縣政府所在地尚無省立專科學校。乃於5月主持籌創[[臺灣省立海事專科學校|臺灣省立海事專科學校]]，設於基隆北寧路八尺門之濱。
*1953年，臺灣省立海事專科學校創校於基隆市，設有駕駛科、輪機科及漁撈科。
*1964年改制為臺灣省立海洋學院。
*1979年改隸為國立臺灣海洋學院。
*1989年升格為'''國立臺灣海洋大學'''。
*2005年獲得國家五年五百億計劃經費，為全國前15所頂尖大學之一。
*2008年再度獲得國家五年五百億計劃第二階段經費，為全國前14所頂尖大學之一。
*2011年名列2011-2012年，世界前400名頂尖大學。教育部自2011年起至2015年止，推動「[[邁向頂尖大學計畫|邁向頂尖大學計畫]]」(五年五百億)，海大為17所頂尖大學之一。
*2014年增設海洋觀光管理學士學位學程、海洋法政學士學位學程、光電與材料科技學士學位學程。加入「[[臺北聯合大學系統|臺北聯合大學系統]]」。
*2015年2月，原人文社會科學院之海洋法律研究所、海洋法政學士學位學程及原海運暨管理學院之海洋觀光管理學士學位學程，新組海洋法律與政策學院。
*2016年，增設海洋文創設計產業學士學位學程，隸屬人文社會科學院。
*2018年2月，原海洋法律與政策學院之海洋觀光管理學士學位學程，改隸屬人文社會科學院。[[教育部|教育部]]「[[高教深耕計畫|高教深耕計畫]]」裡參與全球鏈結卓越研究中心。

==學校特殊節慶==
===海大螢火蟲季===
始於2000年，每年四到五月間，龍崗校區的山間小溪步道會有大量螢火蟲群聚。於此期間校方及課指組的海大生態解說群會與相關社團合力舉辦一系列生態展覽、解說及導覽活動。
在校生以及校外人士皆能自由參加，報名方式有分為網路報名以及現場報名兩種。

===海大棉花糖節===
起源於1999年，日期通常是每年的五月底。當天下午男同學可藉由送棉花糖，進入平時門禁森嚴的女生宿舍，向心儀的對象告白，讓男生有機會對朝思暮想的夢中情人表達自己的心意。收到棉花糖必須要等到吃完的時候才會發現傳情的意義，棉花糖棒子的顏色可傳遞對於不同人不同的情感及意義。棉花糖情語－紅色代表熱戀，紫色代表苦戀，綠色代表祝福，白色代表友情，藍色代表歐趴(ALL PASS)，黑色代表著愛就宅在一起（不過黑色是以訛傳訛，比較像是惡搞）。

除了棉花糖傳情之外，主辦單位（校方及社團）會推出了一系列活動，有熱鬧的園遊會、音樂性社團表演等，希望藉由熱情表演，傳達綿綿情感。另外，在校園設置愛情祈願樹，讓同學將寫好願望的愛情卡掛在樹上，祈求愛情的來臨。同時也可以透過點歌服務活動，由吉他社同學為大家獻唱情歌，讓同學把自己不敢說的話藉由歌曲表達給心儀的對象。

==國際交流／姊妹校==
努力推動及擴展與國際姊妹校之學術交流，支援系所辦理各項國際學術交流活動並積極鼓勵本校學生赴海外姊妹校進行短期研修，使學生接受不同國家文化的洗禮，感受不一樣的教育文化，以提昇其國際視野與未來就業競爭力。
