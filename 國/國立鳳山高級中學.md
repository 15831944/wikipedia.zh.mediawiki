'''國立鳳山高級中學'''，簡稱'''鳳山高中'''、'''鳳中、南高二中'''或'''省鳳'''、'''省鳳中'''（在[[精省|精省]]之前為'''台灣省立鳳山高級中學'''，屬歷史舊稱，改制國立之前為'''省鳳高中'''，再更早之前'''省立鳳中'''），位於[[台灣|台灣]][[高雄市|高雄市]][[鳳山區|鳳山區]]，佔地近7[[公頃|公頃]]，與[[高雄市政府|鳳山行政中心]]（舊[[高雄縣政府|高雄縣政府]]）為鄰，早期以近3成的高留級淘汰率著稱，故曾被戲稱為"鳳中當鋪"、"鳳中五專"。

鳳山高中為[[中華民國教育部|中華民國教育部]]最早實驗高中[[男女合班|男女合班]]的中學，每年招收18班新生。

鄰近[[中華民國陸軍軍官學校|中華民國陸軍軍官學校]]、[[國軍802高雄總醫院|國軍802高雄總醫院]]、[[台鐵|台鐵]][[鳳山車站_(臺鐵)|鳳山車站]]、[[正義/澄清車站|正義/澄清車站]]、[[高雄市公車|高雄市公車總站（高雄市政府交通局公車服務中心）]]、[[捷運|捷運]][[鳳山西站|鳳山西站]]及[[衛武營站|衛武營站]]和[[衛武營國家藝術文化中心|衛武營國家藝術文化中心]]。

== 校史 ==
*創立於日治時期，1939年[[高雄市立林園高級中學|林園庄立林園農業專修學校]]擴充學區，由[[大寮區|大寮]]、[[林園區|林園]]及[[林園區|三庄]]設立，且合併始設'''鳳山園藝專修學校'''為該校前身。
*1944年4月，改稱為'''鳳山女子農業學校'''，專收女生。
*1945年，戰後校名易為'''高雄縣立鳳山初級中學'''，兼收男女生。
*1953年，高中部招生，改制為'''高雄縣立鳳山中學'''。
*1954年8月自大陸播遷來台之私立誠正中學由高雄縣教育局接管後，改名為高雄縣立鳳山中學。因此奉准升格為'''[[臺灣省|臺灣省]]立鳳山中學'''，[[初中|初中部]]及高中併行招生。
*1970年8月，配合政府「[[九年國教|九年國教]]」政策，及「[[省辦高中縣市辦初中|省辦高中、縣辦國中]]」的政策，接收原高雄縣立鳳山中學高中部之學生後，停招初中部，更名為'''臺灣省立鳳山高級中學'''。
*2000年配合政府精簡省府組織，更名為'''國立鳳山高級中學'''。

== 教學特色 ==
===資優班===
:自1990年起奉准成立數理資優班，而後於1995年再奉准成立英文資優班。

===第二外語課===
:高一各班開設[[日語|日語]]或[[德語|德語]]課程，105 (2016) 學年度新增[[越南語|越南語]]，106 (2017) 學年度新增[[韓語|韓語]]。

===教育旅行===
:自2005年起組成教育旅行團，由校長領隊到日本高校進行參訪，與外國學生作文化交流。

===交換學生===
:自2006年起由[[國際扶輪|國際扶輪]]安排外國學生到該校學習中文。<ref name="classmate from abroad">{{cite web | title=外籍生讀鳳中好幸福！ 有學伴還有專師指導 - 生活 | website=自由時報電子報 | date=2017-09-08 | url=http://news.ltn.com.tw/news/life/breakingnews/2187793 | language=zh | access-date=2018-01-07}}</ref>

===段考===
:原每學年度之第一學期四次段考，第二學期則進行三次段考，105學年度後改為上、下2學期皆行3次段考。

== 教學設備 ==
*'''電腦教室'''：3間。
*'''語言教室'''：具語言教學電腦廣播教學系統，可進行團體播放與個別指導。
*'''社會視聽館'''：一樓為128席絨布座椅座位之多媒體視聽教學與小型演講聽，單槍投射120吋螢幕。四樓2間各50席座多媒體教學教室。
*'''自然視聽館'''：一樓為數理資源教室，供師生作專題研究與科學展覽實驗。三樓90席座位多媒體視聽教學教室，四樓地球科學實驗教室。
*'''藝術大樓'''：美術科、音樂科專屬教室各兩間。演藝廳120席座位，可進行多媒體教學與學藝活動。
*'''理化實驗大樓'''：[[物理|物理]]實驗室、[[化學|化學]]實驗室、[[生物|生物]]實驗室各2間。
*'''工藝家政大樓'''：一、二樓為工藝科專屬教室，有[[動力機械|動力機械]]、[[金工|金工]]、[[木工|木工]]、[[製圖|繪圖]]等教室與[[簡報|簡報]]教學室。三樓為二間[[家政|家政]]教室，其中一間有全套[[烹飪|烹飪]]教學器具。
*'''軍護大樓'''：全棟三樓，一樓為學生社團辦公室，二樓為[[軍訓教官_(中華民國)|教官]]教學教室，三樓為護理多媒體視聽教學教室。
*'''圖書館'''：圖書館位於校園東邊，為東西座向，為獨棟三樓建築物。全館總面積533坪（1760平方公尺），一樓閱覽室有288席座位；二樓有辦公室、出納台、閱覽區、書庫閱報區、期刊區、參考區、視聽區等，採開架式電腦管理；三樓K書中心有314席座位，專供高三學生自習讀書。本館目前藏書四萬二千餘冊，期刊一百五十餘種，報紙十種，有12部資料查詢電腦供師生使用，並裝置無線網路。
*'''校園美化'''：前庭育樂亭花圃；中庭左右兩區花圃，有荼蘼花架與鯉魚噴水池；藝術大樓前有奇石園，圖書館前有拱橋蓮池，操場有綠樹成排環繞。
*'''校園e化''' : 全校皆社友[[無線網路|無線網路]]，每間教室裝置[[數位投影機|單槍投影機]]以及[[液晶電視|液晶電視]]，均可實施多媒體視訊教學。
*'''教室人性化''' : 每間教室裝置2台分離式冷氣、教學擴音設備、電動板擦機、投影機與電動式布幕(於107學年度第一學期新設置)。

==== 綜合教學大樓(新建校舍) ====
[[File:New_building_under_construction_of_FSSH.jpg|thumb]]
位於[[蔣中正|蔣公]]銅像左手邊，一棟地上五層，地下一層的建築，於2018年中旬建成。這是鳳中2016年起過去二十五年來第一次獲得[[中華民國中央政府|中央政府]]如此大規模補助的重大建設。

2016年，鳳中獲得[[中華民國教育部|中華民國教育部]]國民及學前教育署1億1900萬元補助，建設地下一層、地上五層，供各學科特色課程使用的專科教室，本案工期近兩年，預計於民國106年12月完工落成。<ref name="new campus">[http://www.fssh.khc.edu.tw/pic.asp?ItemID=587 國立鳳山高中-新地標「綜合教學大樓」開工動土典禮]</ref>建築師於本案的設計理念是「人性化的空間規劃，結合在地化的景觀設計」。建築師巧妙的利用建築結構所形成的剩餘空間，打造休憩平台，讓同學在課餘有更多集會、交誼的場所；配合建案環境地貌，引進自然光的照射、空氣的流動、減少人造光、冷氣用電，實質的達成「節能綠建築」。<ref name="new campus"/>本案委託營建署南區工程處做專案管理，藉由營建署專案施工經驗，從設計初期便將施工時可能遭遇的問題一一解決，以「善用公帑為基礎，營造舒適學習環境為目標」，期許本案「如期、如質」完工。<ref name="new campus"/> <!--公立機關因職務所需發布之新聞稿，屬於著作權豁免之範圍內，特此聲明。-->

===鳳中管樂暨校友管樂團===
{{main article|國立鳳山高中校友管樂團}}
[[File:鳳山高中管樂社暨校友管樂團2016暑期音樂會.jpg|thumb]]（2016暑期音樂會）。]]
鳳山高中校友管樂團成員大多來自[[國立鳳山高級中學|國立鳳山高級中學]]管樂社的畢業校友。

管樂社為校內活躍的社團之一，前身為該校軍樂隊，成立時間已不可考。平日負責學校典禮配樂以及升旗勤務，代表學校參加全國學生音樂比賽，均獲得佳績。自1990年起，管樂社每年暑假皆於鄰近之[[高雄縣|高雄縣]]婦幼青少年館舉行暑期社區音樂會，社團畢業校友於此時返回高中母校聚會並幫忙協助演出。此外管樂社每年寒假皆會舉行社團露營（地點大多於[[臺南市|臺南市]][[走馬瀨觀光休閒農場|走馬瀨觀光休閒農場]]），社團已畢業之學長姐此時都會前往參加指導。管樂社寒暑假的活動構成了校友管樂團成立的契機。

經過多年努力，在創團人朱松偉先生號召下，鳳山高中校友管樂團於2003年七月成立，由朱松偉先生擔任首任團長與首任總幹事，是年八月由陳瑋琮老師指揮，假[[高雄縣|高雄縣]]勞工育樂中心舉行首次公演。2004年七月登記立案於[[高雄縣|高雄縣]]文化局之下，並在2011年元月因[[高雄市縣合併|高雄市縣合併]]改立案於高雄市文化局。

== 校服 ==

鳳山高中現行制服不論男女，上衣皆為淺鵝黃色，夏季為國民領短袖上衣，冬季為西裝領長袖襯衫式上衣。制服上並無校徽，僅於口袋上方繡有校名。男生夏季，搭配卡其色長褲，早期上衣下擺均應紮入褲內，後於90學年度後，可置於褲腰外。冬季亦搭配卡其色長褲，將上衣置於褲腰外，不強制繫腰帶，外套為艾森豪外套（實際上穿的人不多）。女生制服上衣與男生相同，夏季為黑色百折裙，冬季為黑色長褲，但目前女生夏季可自由選擇著長褲或裙子，且女生制服不論夏、冬季皆不需紮入褲腰內，亦無腰帶。女生冬季制服外套為黑色西裝外套。

運動服方面，1990年（[[民國|民國]]79學年度）以前為暗紅色領白色棉質上衣（夏季短袖，冬季長袖），上衣口袋有校名縮寫字樣，暗紅色長褲。1990年（79[[學年度|學年度]]）之後新生改使用紅白色圓領短袖棉質上衣，藍色長褲，冬季並無長袖，僅有藍色運動外套，上印有黃色校徽，校徽上方繡有學號，同時自1992年（81學年度）起，另有一套類似、惟以鵝黃色為底的運動服，與白色底各學年交替使用，至1998年左右始全部改用白色底。因舊式運動服冬季缺乏保暖性，且外套之樣式與[[岡山農工|國立岡山農工]]極為相似，故自94學年度起，將運動服改為新款樣式。夏季上衣維持舊款，改搭配黑底三白線的運動短褲。冬季上衣為淺藍色長袖上衣，搭配深藍色，淺藍直條長褲。外套改為深色運動風衣外套，於袖子兩側亦有與長褲相同之淺藍色長條。

早期[[軍訓|軍訓課程]]必須有軍訓服，一套典型的軍訓服包括大盤帽、卡其上衣（夏長冬短）、卡其褲（女生為卡其窄裙）以及白襪黑皮鞋，皮鞋可選購，而學號則以藍字縫繡。1990年之後取消大盤帽，1994年（83學年度）後取消卡其軍訓服，改以制服實施軍訓教育，惟保留白襪黑皮鞋。至1995年（84學年度）之後始全面以制服實施。

== 書包 ==

書包為綠色棉質帆布斜背式傳統書包，上漆白字（校名），目前書包上字樣為“鳳山高中” (由右而左書寫)，改制國立之前為“省鳳高中”，再更早之前為“省立鳳中”。

== 校歌 ==
<blockquote>''<big>我們是民族的戰士</big>''</blockquote><blockquote>''<big>我們是革命的洪流</big>''</blockquote><blockquote>''<big>跟著時代前進　趕上時代前頭</big>''</blockquote><blockquote>''<big>鳳山復興基地　學府日起新猷</big>''</blockquote><blockquote>''<big>手腦並用創造　文武合一貽謀</big>''</blockquote><blockquote>''<big>恢復文化　光復神州</big>''</blockquote><blockquote>''<big>大家努力　狀志必酬</big>''</blockquote>

由於[[台灣日治時期|日治時期]]校歌詞曲資料佚失，所以現今的鳳中校歌創作於1952年，是由[[趙崑祥|趙崑祥]]作曲，[[譚地大|譚地大]]作詞，在當時全國上下充滿[[反攻大陸|反攻大陸]]口號的情況下，加上鳳中位處於[[軍事院校|軍事院校]]林立與軍區密布的區域（鳳中附近有[[衛武營|衛武營]]、[[中華民國陸軍軍官學校|中華民國陸軍軍官學校]]、[[國軍802高雄總醫院|國軍802高雄總醫院]]等重要軍事單位），因此在時代與地理氛圍影響下，鳳中校歌有著一般普通高中校歌中罕見之軍歌式的詞曲風格。之後鳳中曾一度存在更換校歌之提議，但考量歷屆校友反對，最終決定不予更換。

== 學生自治組織 ==
=== 班聯會 ===
此會為鳳山高中重要的社團組織。成立於1991年，隸屬於學務處訓育組。主要負責各項學生自治事務之推動及各項活動之舉辦及籌畫工作，例如運動會、校慶、聖誕晚會等活動。領袖為班聯會會長，其下設有副會長及活動、公關、總務、美工等各組，分別負責專門之事務，組長由會長提名，組員則由組長徵選。各班有選舉班代表，由班代表（而非班長）參與會議事務。班聯會會長於每年5月底到6月中舉行選舉。選舉時，正副會長需聯名登記參選，選舉權人投票圈選時是以一組候選人為圈選單位。 此選舉被選舉人為該校1年級學生，投票人為1、2年級學生，3年級學生無投票權。

然而，三年級無投票權的辦法與《高級中等教育法》第五十三條之旨意有抵觸疑慮，援引如下：

''高級中等學校應輔導學生成立由全校學生選舉產生之學生會及其他相關自治組織，並提供其必要協助，以增進學生在校學習效果及自治能力。''

''學生為前項學生會當然會員。''

=== 班長班代大會 ===
此大會為學校與學生間例行之溝通形式之一，每學期固定於學期初及學期末召開兩次常會，其召集單位為校方為學務處，學生方面為[[班聯會|班聯會]]。開會時間一般為中午用餐及午休時間，由各班所選出之班長及班代，代表班級共同出席，聽取學校相關行政單位事務說明，並代表學生向學校提出各種意見及建議。
