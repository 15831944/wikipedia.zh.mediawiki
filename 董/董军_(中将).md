'''董军'''，男，[[中国人民解放军海军中将|中国人民解放军海军中将]]。

1979年参加[[中国人民解放军|中国人民解放军]]，曾任[[中国人民解放军海军|中国人民解放军海军]]司令部军训部部长、[[北海舰队|北海舰队]]副参谋长、海军92269部队司令员。2013年，任[[东海舰队|东海舰队]]副司令员<ref>{{Cite web |url =  http://www.fmprc.gov.cn/ce/cemx/chn/sgxx/t1191820.htm|title =邱小琪大使会见出席墨西哥独立庆典中方代表团  |author =  |date = 2014-09-16 |publisher = 中华人民共和国驻墨西哥大使馆 |language =  |accessdate = 2019-01-13 |quote =  }}</ref>。2014年12月转任海军副参谋长<ref>{{cite web |author1=岳菲菲 |title=解放军多位将领岗位调整 |url=http://politics.people.com.cn/n/2015/0104/c1001-26318074.html |website=人民网 |accessdate=2019-01-13 |date=2015-01-04}}</ref>。2017年1月，升任[[南部战区|南部战区]]副司令员<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1648939 |title = 海军原副参谋长董军少将出任南部战区副司令员 |author =  |date = 2017-03-27 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。

2012年7月，晋升少将军衔<ref>{{cite web |title=海军举行晋衔仪式 3人晋升中将，6人晋升少将军衔 |url=http://bbs.tiexue.net/post2_6020452_1.html |website=铁血网 |accessdate=2019-01-13 |date=2012-08-04}}</ref>。2018年7月，晋升中将军衔<ref>{{cite web |title=习近平在视察南部战区时强调  加快推进战区指挥能力建设  坚决完成担负的使命任务 |url=http://tv.cctv.com/2018/10/26/VIDEc9uTm2ILnEpjsXRLzVpy181026.shtml |website=新闻联播 |accessdate=2019-01-13 |date=2018-10-26}}</ref>。
