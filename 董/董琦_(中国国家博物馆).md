'''董琦'''（{{bd|1952年|10月||}}）生于[[上海|上海]]，[[中华人民共和国|中华人民共和国]]考古学家，[[中国国家博物馆|中国国家博物馆]]副馆长<ref name=dongqi>{{Cite web |url =http://www.cctv.com/science/special/C15704/20060515/102849.shtml  |title =  董琦|author =  |date = 2006-05-15 |publisher = 央视国际 |language =  |accessdate =  |quote =  }}</ref><ref name=bb>{{Cite web |url = http://news.artxun.com/dongzuo-1597-7982596.shtml |title =董琦  |author =  |date =  2011-03-28 |publisher = 博宝资讯网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1976年自[[北京大学|北京大学]]历史系考古专业毕业。1992年获北京大学考古系历史学博士学位。是[[中国共产党|中国共产党]]党员，研究馆员，[[中国考古学会|中国考古学会]]会员。1976年到[[中国历史博物馆|中国历史博物馆]]工作，历任[[垣曲商城|垣曲商城]]考古队领队，中国历史博物馆展览部主任，馆刊编辑委员会、文物征集鉴定委员会、职称评审委员会委员，[[中国国家博物馆|中国国家博物馆]]展览部主任。2001年6月获国务院[[政府特殊津贴|政府特殊津贴]]。2003年9月至2013年任中国国家博物馆副馆长。2004年3月任[[中华人民共和国文化部|文化部]]文物博物专业高级职称评审委员会委员。后兼文化部创新奖评审委员会委员、[[中央民族大学|中央民族大学]]硕士研究生导师、北京[[郑和下西洋|郑和下西洋]]研究会副理事长等职务<ref name=dongqi/><ref name=bb/>。

董琦长期从事考古田野发掘以及夏商周考古学研究，先后参加[[山西|山西]][[夏县|夏县]][[东下冯遗址|东下冯遗址]]、[[河南|河南]][[登封|登封]][[王城岗遗址|王城岗遗址]]、[[宁夏|宁夏]][[海原|海原]][[菜园遗址|菜园遗址]]、[[山西|山西]]垣曲商城遗址的发掘工作。其中，东下冯遗址、王城岗遗址都是国家“[[六五计划|六五]]”社会科学重点研究项目。董琦参与执笔的《登封王城岗与阳城》大型发掘报告获“[[夏鼐|夏鼐]]考古学基金”奖及[[河南省|河南省]]社会科学研究一等奖。董琦参与了《中国古代科技文物展》和《香港国宝展》的筹展工作<ref name=bb/>。

==著作==
董琦主要著作有：
*《虞夏时期的中原》<ref name=bb/>

董琦的学术论文有30多篇，主要有：
*《论我国史前时期的城堡》（《北方文物》1988年4期）
*《中国先秦城市发展史概述》（《中原文物》1995年1期）
*《虞夏时期的社会发展阶段》（《中原文物》1996年3期）
*《夏代的中原》（载《夏文化研究论集》，中华书局1996年出版）
*《王城岗城堡遗址分析》（《文物》1984年11期） 
*《再析《偃师商城与夏商文化分界》的研究脉络》 
*《析《偃师商城与夏文化分界》的研究脉络》 
*《王城岗城堡遗址再分析》（中国历史文物）<ref name=bb/>

董琦参与编撰出版了《中国通史陈列》、《中国古代科技文物展》、《天工开物》等图录<ref name=bb/>。
