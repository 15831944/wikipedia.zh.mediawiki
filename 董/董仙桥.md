'''董仙桥'''（{{bd|1895年|2月|1990年|3月22日}}）原名'''董海云'''，[[吉林省|吉林省]][[榆树县|榆树县]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，中共[[佳木斯市|佳木斯市]]委书记、市长，[[吉林省高级人民法院|吉林省高级人民法院]]副院长，[[吉林省政协|吉林省政协]]常委、副秘书长<ref name=yushu>{{Cite web |url = http://ysws.yushulife.com/index.php?c=article&id=250 |title = 董仙桥 |author =  |date =2015-11-11  |publisher = 榆树文史 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1895年农历二月初九日，董仙桥生于榆树县大坡西郝家屯一个贫困家庭。 1906年入[[私塾|私塾]]学习。1909年入本乡官办小学学习。1914年考入吉林省立第一师范学校，后因家贫而辍学。因在校期间品学兼优，被推荐到榆树县文华中学当教员。1923年7月，考入吉林法政专门学校。1924年秋，转入[[北京|北京]][[中国大学_(北平)|中国大学]]法律系。1927年毕业后，应聘到吉林省立第五中学任教，1930年任该校代理校长。在此期间，因营救遭当局逮捕的学生而被撤职<ref name=yushu/>。当时他因言论而被怀疑为共产党。此前在1928年夏，董仙桥的朋友李恩举全家迁居佳木斯，李恩举应聘到西门小学教书并任校长。被撤职的董仙桥乃携全家到佳木斯投奔李恩举<ref name=jms>{{Cite web |url = http://www.jms.gov.cn/html/index/content/2017/10/11508573085514.html |title =董仙桥  |author =  |date = 2017-04-12 |publisher = 中国佳木斯 |language =  |accessdate =  |quote =  }}</ref>。随后董仙桥到佳木斯桦川中学任教<ref name=yushu/>。

1931年[[九一八|九一八]]事变后，[[日军|日军]]侵占中国[[东三省|东三省]]。1932年农历四月，[[日本|日本]]军舰开进佳木斯，当时正任佳木斯女子师范教员的董仙桥和其他教员宣传抗日救亡<ref name=jms/>。

1933年6月，[[中共河北省委|中共河北省委]]特派员[[苏梅|苏梅]]和党员[[李向之|李向之]]来到佳木斯，秘密建立中共地下组织，组织抗日义勇军。董仙桥和妻子李淑云经这两人介绍，加入[[中国共产党|中国共产党]]<ref name=yushu/>。1933年10月，董仙桥在佳木斯加入中国共产党，并任佳木斯第一个中共基层组织“西门外小组”的组长。西门外小组成立后积极发展党员，后发展到11人<ref name=jms/>。1934年春，在西门外小组的基础上成立佳木斯西门外支部，董仙桥任支部书记<ref name=yushu/><ref name=jms/>。当时支部设在董仙桥家里，成为地下党组织的秘密联络点。1936年10月，中共下江特委决定将西门外支部改成佳木斯特别支部，董仙桥负责宣传工作<ref name=jms/><ref name=lzw>{{Cite web |url = http://www.zglz.gov.cn/jmsweb/renwu1.htm?id=jed5k90dv2 |title =人物  |author =  |date =  |publisher = 中国龙志网 |language =  |accessdate =2018-03-20  |quote =  }}</ref>。

1937年1月，佳木斯特别支部、桦川中学党支部合并，成立中共佳木斯市委员会，董仙桥任市委书记。市委在他领导下，党员发展至60多人，先后成立6个支部<ref name=yushu/><ref name=jms/><ref name=lzw/>。他还掩护和输送中共地下党员加入[[东北抗日联军|东北抗日联军]]。[[八女投江|八女投江]]中的队长[[冷云|冷云]]就是董仙桥的学生，经董仙桥安排参加了抗联。为给抗联提供情报，他将中共地下党员派到敌人内部，掌握敌人动向。根据中共北满临时省委批示，他为抗联三军、六军筹集军需物资。仅1937年就筹集服装600余件，油印机4台和大批药品、宣传品等<ref name=yushu/>。

1938年春，[[满洲国|满洲国]]日伪警察和特务机关加紧搜捕抗日人员。3月15日，日伪军队、警察、宪兵、特务出动千余人实施大逮捕，史称“三·一五”事件，三江地区的中共地下党组织被全部破坏。同日拂晓，董仙桥和30多名中共党员、抗日救国会会员等被捕。董仙桥被捕后宁死不屈、拒不招供，后被押到哈尔滨高等法院审判。1939年4月，哈尔滨高等法院以颠覆满洲国的罪行判处董仙桥[[有期徒刑|有期徒刑]]20年，关押在哈尔滨市道里监狱。1942年6月，转押到满洲国[[新京|新京]]（今[[长春|长春]]）。在狱中，董仙桥秘密成立了中共狱中党支部，发展党员，与敌斗争。1945年8月，日本在投降前准备枪杀董仙桥等大批政治犯，但[[苏联红军|苏联红军]]突然轰炸新京的监狱，董仙桥等人死里逃生<ref name=yushu/>。

1945年[[日本投降|日本投降]]，董仙桥出狱。随后回到佳木斯市，任[[合江省|合江省]]政府秘书长兼教育厅长。1945年底，中共合江省工委派他担任中共领导下的佳木斯市政府首任市长（副市长为[[孙西林|孙西林]]，不久即遇刺身亡），并任市委常委<ref name=lzw/><ref name=yushu/>。他担任市长直到1948年5月<ref name=jms/>。任内面对日伪留下的乱摊子，他发动群众建立基层政权，组织私营工商业者恢复生产，打击特务活动，安定社会秩序<ref name=yushu/>。

[[中华人民共和国|中华人民共和国]]成立后，董仙桥历任合江省人民法院副院长，[[松江省|松江省]]人民法院副院长，[[最高人民法院|最高人民法院]]东北分院办公室主任，吉林省高级人民法院副院长，吉林省政协副秘书长（1956年任），第二至四届吉林省政协常委<ref name=lzw/><ref name=yushu/>。1956年经[[中共吉林省委|中共吉林省委]]安排参加[[民革|民革]]，成为中共和民革的“交叉党员”，历任吉林省民革第一至三届常务委员，1980年当选吉林省民革第四届副主任委员<ref name=yushu/>。

“[[文化大革命|文化大革命]]”期间，董仙桥受到批判和迫害。[[粉碎四人帮|粉碎“四人帮”]]后，董仙桥获得平反，重返工作岗位。当时他年过八旬，已不能每天上班，但坚持每天听广播，看报纸，读文件。1980年代，董仙桥抱病多次应邀到佳木斯市，参加回顾佳木斯党的斗争历史的座谈会<ref name=yushu/>。

1990年3月22日，董仙桥病逝，享年95岁<ref name=yushu/>。
