'''董仲舒墓'''是[[西汉|西汉]]大儒[[董仲舒|董仲舒]]的墓葬，位于[[陕西省|陕西省]][[西安市|西安市]][[碑林区|碑林区]][[和平门_(西安)|和平门]]内[[下马陵|下马陵]]（亦称虾蟆陵<ref> {{Cite web |url= http://news.sina.com.cn/c/2010-07-06/022017759794s.shtml |title= 昔日武帝下马 如今车来车往(图) |author= 冯强 |date= 2010年7月6日 |publisher= 华商网  |language= zh-hans |accessdate= 2013年2月21日 |quote=  }}</ref>）。董仲舒墓占地约4.5亩，现存圆丘形封土，底径6米，高2.5米，墓前立墓碑一通，墓后数米是一座六角凉亭，并有近代建祠堂三间。<ref name=beilin> {{Cite web |url= http://www.wmbeilin.com/wch/ShowInfo.asp?InfoID=266 |title= 董仲舒墓的保护和管理工作汇报 |author=  |date= 2011年10月11日 |publisher= 文明碑林网 |language= zh-hans |accessdate= 2013年2月21日 |quote=  }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

据记载[[汉武帝|汉武帝]]出于对董仲舒的尊敬，每次经过董仲舒墓，都要下马，民间就称此为下马陵。<ref>{{Cite web |url= http://sn.ifeng.com/zhuanti/detail_2012_12/08/470541_3.shtml |title= 探寻下马陵 |author=  |date= 2012年12月8日 |publisher= 凤凰网 |language= zh-hans | accessdate= 2013年2月21日 |quote=  }}</ref><ref name=xawb>{{Cite web |url= http://epaper.xiancn.com/xawb/html/2011-06/19/content_45219.htm |title= 汉代大儒董仲舒的精神墓地 西安下马陵 |author= 薛雯 |date= 2011年6月19日 |publisher= 《[[西安晚报|西安晚报]]》 |language= zh-hans |accessdate= 2013年2月22日 |quote=  }}</ref>唐人[[韦述|韦述]]的《两京记》中对此也有记载：“汉武帝至墓前下马，故曰下马陵”。

[[陕西省人民委员会|陕西省人民委员会]]于1956年8月6日将董仲舒祠墓公布为陕西省第一批重点文物保护单位。<ref name=hsw>{{Cite web |url= http://news.hsw.cn/system/2012/03/31/051289312.shtml |title= 董仲舒墓深藏干休所大院 不对外开放 |author= 张丹、李琦 |date= 2012年3月31日 |publisher= 华商网 |language= zh-hans |accessdate= 2013年2月21日 |quote=  }}</ref>

==保护情况==
1949年以前董仲舒的后裔曾在董墓中发现汉简，因不认识文字又复埋简于墓中，而陵园长期被[[胡宗南|胡宗南]]作为官邸。<ref name=hsw/>1987年拨款修葺，1989年增建了陵园门楼。<ref name=hsw/>

因为董仲舒墓位于[[兰州军区|兰州军区]]西安和平路干休所内，正殿长期被作为老干部活动中心使用，后被出租做摄影室，所以不对游人开放。<ref name=hsw/><ref> {{Cite web |url= http://www.chinanews.com/cul/2012/01-29/3625429.shtml |title= 西安碑林将保护开发董仲舒墓 整合儒学文化资源 |author= 周倩 |date= 2012年1月29日 |publisher= 中国新闻网 |language= zh-hans |accessdate= 2013年2月21日 |quote=  }}</ref>

==关于墓葬位置的争议==
尽管文物保护部门认定和平门内为董仲舒墓在所在，但对于董仲舒墓的位置一直都存在不同意见。

有观点认为董仲舒墓位于现[[西安交通大学|西安交通大学]]校园内，以[[唐长安城|唐长安城]][[长乐坊|长乐坊]]的[[蛤蟆陵|蛤蟆陵]]为董仲舒墓址。唐人韦述的《两京记》，唐元和年间[[李肇|李肇]]的《唐国史补》，北宋时期的[[宋敏求|宋敏求]]的《[[长安志|长安志]]》，元代[[骆天骧|骆天骧]]的《类编长安志》，明代《[[大明一统志|大明一统志]]》中均持此种观点。

亦有观点认为董仲舒墓址应在陕西[[兴平县|兴平县]][[汉茂陵|茂陵]]随葬，北宋时期《太平寰宇记》，《兴平县志》持此观点。根据《汉书·董仲舒传》记载：董仲舒“年老，以寿终于家。家徙茂陵。”<ref> {{Cite web |url= http://www.chinanews.com/cul/2012/11-28/4365565.shtml |title= 汉武帝茂陵传说：墓穴可坐看长安 为防盗全用沙子   |author=  |date= 2012年11月28日 |publisher= 西安日报 |language= zh-hans |  accessdate= 2013年2月22日 |quote=  }}</ref>近代以来，考证认为董仲舒随葬于茂陵的观点开始增多。<ref name=xawb/><ref name=paper>{{Cite journal zh |author = 孙民柱   |date = |year = 2000年|month = |title = 董仲舒墓址辨惑 |journal = 中国历史地理论丛 |volume = 3 | issue = |pages = |publisher = |location = |issn =|id = |url = http://www.cnki.com.cn/Article/CJFDTotal-ZGLD200003014.htm |language = zh-hans|format = |  accessdate = |laysummary = |laysource = |laydate = |quote = }}</ref>

认为董仲舒墓应位于和平门的说法是自明代以来的官方观点。<ref name=beilin/><ref name=paper/>1607年的《关中陵墓志》，乾陵年间的《陕西通志》和嘉庆年间的《咸宁县志》都认为董仲舒墓址位于胭脂坡（现和平门内）。但是据学者考证，现在和平门内的董仲舒墓是在明代嘉靖年间，陕西巡按都御史赵廷锡将城外“蝦蟆陵”处董子祠移建于西安城内今和平门内西侧路北，并在祠后起冢为坟，称作董仲舒墓。<ref name=xawb/>康熙六年，咸宁知县[[黄家鼎|黄家鼎]]在此重建祠堂三间，并在大门外立石，上书“下马陵”三字，门额刻“董子祠”三字。<ref name=xawb/><ref name=paper/>

==传说==
传说[[明朝|明朝]]初年，[[朱樉|朱樉]]为秦王驻守关中，在重建长安城时，下令把“下马陵”拒之城外。长安城建好后，“下马陵”不知怎么搞的仍在城内。朱樉十分恼怒，杀了监工不说，又下令把建好的南城墙拆掉，在“下马陵”以北重新修建。3年后，工程告竣，当朱樉再次巡查时，“下马陵”还在城里。如此反复，“下马陵”始终在城内。因为朱樉屡次北移南城墙，使长安城的南大街越来越短。一直到现在，南大街成了四条大街中最短的一条大街。<ref> {{Cite web |url= http://sx.sina.com.cn/news/m/2012-10-19/162324187.html |title= 西安城墙南北短东西长的传说 |author=  |date= 2012年10月19日16 |publisher= 新浪陕西 |language= zh-hans |accessdate= 2013年2月21日 |quote=  }}</ref>
