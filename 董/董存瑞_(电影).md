《'''董存瑞'''》，1955年由郭维执导，张良、杨启天、张莹、周凋、任颐等主演的战争片。

== 剧情 ==
[[抗日战争|抗日战争]]之际，[[董存瑞|董存瑞]]和好友[[郅顺义|郅顺义]]因为帮助在[[大日本帝国|大日本帝国]]的扫荡中牺牲的区党委书记王平交党费而被破格允许参军，因为他们年龄不达标。抗日战争胜利后，[[中国共产党|中国共产党]]和[[中国国民党|中国国民党]]为争夺中国大陆政权爆发了[[国共内战|国共内战]]，在[[河北省|河北省]][[承德市|承德市]][[隆化县|隆化县]]，国共双方展开争夺战，董存瑞接受中共高层命令，爆炸碉楼，他用手举着炸药包和碉楼同归于尽。

== 花絮 ==
导演[[郭维|郭维]]虽然拍摄了红色电影《董存瑞》，但是在1957年依然被打成[[右派分子|右派分子]]。

《董存瑞》导演郭维在《大众电影》访谈时说：“‘英勇献身’是推测出来的，没人亲见，因后来在桥底发现了媳妇缝给董的袜底才猜想是他牺牲了。”<ref>{{cite web |url=http://yantai.dzwww.com/zszl/201306/t20130624_8550356.htm |title= 故纸堆：导演谈董存瑞电影|accessdate=2013 |publisher= 大众网|work= |language =zh-cn}}</ref>

2013年，[[中国共产党中央委员会宣传部|中国共产党中央委员会宣传部]]、[[中华人民共和国教育部|中华人民共和国教育部]]、[[中国共青团|中国共青团]]团中央向中国大陆青少年推荐100种优秀图书、100部优秀影视片，该片入选。<ref>{{cite web |url=http://news.xwh.cn/news/system/2013/07/19/010374242.shtml |title=百部向青少年推荐影视片 长影8部电影入选 |accessdate=2013-07-19 |publisher=新文化网 |work= |language=zh-cn }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
