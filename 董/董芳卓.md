'''董芳卓'''（{{bd|1985年|1月23日}}）原名'''董方卓'''，出生於[[遼寧省|遼寧省]][[大连市|大连市]]，[[中国|中国]][[足球|足球]][[运动员|运动员]]，曾效力於[[英格蘭超級足球聯賽|英格蘭超級足球聯賽]][[曼聯足球會|曼聯足球會]]。

== 早期生涯 ==
董芳卓职业生涯起于[[中国足球乙级联赛|中国足球乙级联赛]]的[[哈爾濱毅騰足球俱樂部|大连毅腾]]少年隊。2000年時，[[大连实德足球俱乐部|大连实德]]從[[哈爾濱毅騰足球俱樂部|大连毅腾]]中收購了董芳卓，成為大連實德四隊的球員之一，同年的全國錦標賽中，他助球隊奪得了冠軍。

2001年，董芳卓參與了[[沙特阿拉伯|沙特阿拉伯]]的阿布杜拉王子盃。其後的1年，他晉升至大連實德二隊，首度於甲B聯賽上陣。2003年，他在A3聯賽中代表大連實德參賽，兩度後備上陣。隨後的[[亞洲聯賽冠軍盃|亞洲聯賽冠軍盃]]中，董芳卓在第二場分組賽中攻入他在俱乐部的第一球。

== 曼聯 ==
2004年1月12日，[[英超|英超]]俱乐部[[曼聯|曼聯]]以3,500,000[[英镑|英镑]]的转会费（首付1,000,000[[英镑|英镑]]）簽入董芳卓。從而他是首位[[東亞|東亞]]球員加盟曼聯，也是首位中國球員加盟[[G-14|G-14]]創始成員球隊。但因工作證等問題后租借至[[比利时|比利时]]的[[皇家安特卫普足球俱乐部|安特卫普]]，身批9号战袍。他在2005至2006球季中以上陣28場，攻入18球的成績，成为比利时乙组联赛神射手。在球季未開始的暑假，他返回曼聯及參與2005曼联亚洲之旅，与[[香港足球代表隊|香港隊]]的比賽上，董芳卓身穿43号球衣攻進为曼联效力以来的首個进球，協助曼联以2-0战胜对手。2006年暑假的南非之旅中，他再一次代表曼聯上陣多場友誼賽，取得幾個入球。

他在2006年12月15日曼聯終於成功為董芳卓取得工作證後，於年尾返回曼聯<ref>[http://news.bbc.co.uk/sport2/hi/football/teams/m/man_utd/6184803.stm 曼聯為董方卓取得工作證]</ref>。隨後在2007年1月17日，董芳卓與曼聯再次簽訂合約，為期4年，直至2010年約滿，並且穿起21號球衣，惟於2008/2009赛季，董芳卓的球衣號碼被收回，轉予[[拉斐尔·达·席尔瓦|拉斐尔·达·席尔瓦]]。<ref>{{Cite web |url=http://www.manutd.com/default.sps?pagegid=%7BB4CEE8FA-9A47-47BC-B069-3F7A2F35DB70%7D&newsid=395623 |title=Dong signs Reds deal |access-date=2017-07-20 |archive-url=https://web.archive.org/web/20170325202658/http://www.manutd.com/default.sps?pagegid=%7BB4CEE8FA-9A47-47BC-B069-3F7A2F35DB70%7D&newsid=395623 |archive-date=2017-03-25 |dead-url=yes }}</ref><ref>{{Cite web |url=http://www.manutd.com/default.sps?pagegid=%7BF9E570E6-407E-44BC-800F-4A3110258114%7D&newsid=6616127 |title=Squad numbers registered |access-date=2017-07-20 |archive-url=https://web.archive.org/web/20170325202702/http://www.manutd.com/default.sps?pagegid=%7BF9E570E6-407E-44BC-800F-4A3110258114%7D&newsid=6616127 |archive-date=2017-03-25 |dead-url=yes }}</ref>

2007年3月13日，曼聯參與一場為慶祝[[歐盟|歐盟]]成立以及曼聯參加歐洲賽50周年的表演賽，對手為[[歐洲明星隊|歐洲明星隊]]，比賽至第72分鐘，董芳卓替补[[阿兰·史密斯|阿兰·史密斯]]出场，首次代表曼聯在[[奧脫福球場|奧脫福球場]]亮相，最終曼聯以4-3擊敗對手。5月9日，董方卓在对阵[[切尔西足球俱乐部|車路士]]的[[英超联赛|英超联赛]]中首次為已經奪得聯賽錦標的曼联和[[奧利·根拿·蘇斯克查|奧利·根拿·蘇斯克查]]首发出场。8月4日，董芳卓在對陣英乙球隊[[彼德堡聯足球會|彼德堡]]時，攻入1球並獻上1個助攻，結束重返曼聯後多個月的入球荒。9月26日，在聯賽盃對戰[[考文垂足球俱乐部|高雲地利城]]中為曼聯在奧脫福球場上陣全場賽事。同年的12月12日，董芳卓在[[歐洲聯賽冠軍盃|歐洲聯賽冠軍盃]]對陣[[羅馬足球俱樂部|羅馬]]賽事中後備替補[[韋恩·魯尼|韋恩·魯尼]]出場，成為繼[[孫祥|孫祥]]以後第二位參與歐洲聯賽冠軍盃賽事的[[中國國家足球隊|中國球員]]<ref>[http://sports.sohu.com/20071213/n254051287.shtml 董方卓成中国欧冠第二人 与队友精妙配合显默契]</ref>。在2007年至2008年季度，董芳卓僅得三次於正式比賽上陣的機會，其餘時間皆於預備組渡過。他曾經被[[布力般流浪|布力般流浪]]後衛鏟傷前在3場預備組比賽中取得4個進球；他的賽季被迫提早完結，成為離開曼聯的關鍵因素<ref>{{Cite web |url=http://www.manutd.com/default.sps?pagegid=%7BF9E570E6%2D407E%2D44BC%2D800F%2D4A3110258114%7D&newsid=6617020 |title=Reds confirm Dong exit |access-date=2008-10-19 |archive-url=https://web.archive.org/web/20081003211202/http://www.manutd.com/default.sps?pagegid=%7BF9E570E6%2D407E%2D44BC%2D800F%2D4A3110258114%7D&newsid=6617020 |archive-date=2008-10-03 |dead-url=yes }}</ref><ref>[http://soccer.sina.com.hk/cgi-bin/so/nw/show.cgi/6/2/1/1524731/1.html 解析董方卓如何被廢 前程就這樣被私心與黑腳折斷]</ref><ref>[http://sports.sina.com.cn/g/2008-10-17/11254014248.shtml 留洋军团08难逃命中大劫 郑智前已5人重伤2将梦碎]</ref><ref>[http://www.higoal.com/pub/csl/2008/1017/92678.htm 留洋军团难以逾越高山 伤病导致结束征程]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://sports.hsw.cn/2008-10/17/content_10342335.htm 马德兴：伤病成留洋最大死穴 郑智恐遭石油清洗] {{Wayback|url=http://sports.hsw.cn/2008-10/17/content_10342335.htm |date=20140304083939 }}</ref>。

== 离开曼联 ==
在[[2008年夏季奧林匹克運動會|2008年北京奧運]]中，他在對[[新西蘭|新西蘭]]的賽事中攻下了中國於男子足球項目中的首個入球<ref>[http://2008.sohu.com/20080807/n258668579.shtml 董方卓创纪录入男足奥运首球 中国1-1新西兰]</ref>。

在2008年8月27日，董芳卓因為被提前解約而重返[[大连实德|大连实德]]<ref>{{en icon}}[http://www.manutd.com/default.sps?pagegid={F9E570E6-407E-44BC-800F-4A3110258114} {{lang|en|Reds confirm Dong exit}}] {{Wayback|url=http://www.manutd.com/default.sps?pagegid=%7BF9E570E6-407E-44BC-800F-4A3110258114%7D |date=20170325202913 }}</ref>。他在[[中超|中超]]两个赛季出场26次，惟未取得进球。

2010年2月19日，[[波兰|波兰]]球队[[華沙萊吉亞足球俱樂部|華沙萊吉亞]]官方确认签入董芳卓<ref>[http://sports.sina.com.cn/g/2010-02-20/03134845558.shtml 波甲豪门高官确认将签董方卓 1.5+2合同揽前曼联前锋]</ref>。8月4日，上场机会寥寥无几的董芳卓被球队解约<ref>[http://news.goalhi.com/201008/74116.html 官方消息：董芳卓离开莱吉亚]</ref>。8月19日，董方卓加盟[[葡超|葡超]]升班马[[波蒂莫嫩塞|波蒂莫嫩塞]]<ref>[http://sports.sina.com.cn/g/2010-08-19/17255157157.shtml 葡超升班马证实董方卓到来 引用新浪视频介绍中国前锋]</ref>，惟表现平平，仅在葡萄牙杯中取得一个进球，而联赛三次登场则颗粒无收。

2011年2月，董芳卓在[[塞爾維亞足球超級聯賽|塞爾維亞足球超級聯賽]]升班马[[斯洛博達足球俱樂部|斯沃伊诺自由]]试训。同年3月，董芳卓转会加盟[[亚美尼亚足球超级联赛|亚美尼亚足球超级联赛]]球队[[米卡足球俱乐部|米卡]]。3月19日，董芳卓在他的首次亮相就取得进球，帮助米卡队客场3比0击败对手<ref>[http://sports.sina.com.cn/g/2011-03-28/18245509244.shtml 董方卓加盟亚美尼亚球队 首秀补时第3分钟进处子球]</ref>。这也是他近4年零3个月以來的首个联赛进球。

[[File:Dong_Fangzhuo_In_Hunan_Billows.JPG|thumb]]
2012年1月，董芳卓回国加盟[[中甲|中甲]]球队[[湖南湘涛|湖南湘涛]]，他与球队签订两年合同。

2014年1月，董芳卓宣布，新赛季他将加盟[[中甲|中甲]]升班马[[河北中基|河北中基]]，结束在[[湖南湘涛|湖南湘涛]]两年的时光。<ref>[http://sports.163.com/14/0126/20/9JHTEO2100051C8L.html#p=9J714HF84FFD0005 董方卓宣布加盟河北中基 湘涛锋线同投中甲新军]</ref>4月份，被爆身材已经严重走样。<ref>[http://sports.sina.com.cn/b/2014-04-14/03137117926.shtml 董方卓避回应身材发福囧闻 中基:来的时候他就这样]</ref>

与河北中基的一年合同到期后，董方卓离开了职业联赛。2016年1月3日，他代表业余球队表美罗蒂俱乐部参加了2016长沙县足协贺岁杯足球赛。赛后采访中，董方卓表示自己已经因为伤病而退役，参加贺岁杯前已经有几个月没有踢球。<ref>{{cite news|title=董方卓现身长沙业余比赛 亲承因伤病已退役数月|url=http://sports.163.com/16/0105/11/BCIGIHN300051C8U.html|accessdate=2016-04-29|agency=人民网|date=2016-01-05}}</ref>2016年4月，他又加入梅州的一支业余球队征战五人制足球比赛。<ref>{{cite news|title=网传董方卓加入地方业余队 将参加5人制比赛(图)|url=http://sports.ifeng.com/a/20160422/48554627_0.shtml|accessdate=2016-04-29|agency=凤凰体育|date=2016-04-22}}</ref>

== 改名 ==
2007年7月底，董芳卓签名时将自己的名字写成了由“董方卓”改为了“董芳卓”，据了解，这是因为一位专门研究签名的字体专家给他的提议，因为这样“写起来更加好看一些”。<ref>[http://sports.sina.com.cn/g/2007-08-06/11003085521.shtml 董方卓将自己签名改成"董芳卓" "芳名"源自专家建议]</ref><ref>[http://www.oushinet.com/news/29853.htm 董方卓改名小芳走好运]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 不实报道 ==
某些无良记者在媒体上刊登了多则关于董方卓的不实报道，其中包括董方卓花7万英镑购买奥迪R8汽车，在训练中偷懒，耍大牌甚至顶撞佛格森，不积极学习英语，语言能力差，平时沉迷于网络游戏以及弗格森批评董方卓在场上跑动不积极等，后来都被证实为胡编乱造的无稽之谈，甚至涉嫌恶意造谣。
董方卓曾表示：谁说我买了奥迪R8，我都想去告这些记者！
佛格森从未发表过关于董方卓训练消极以及态度不端正的言论，均为无良记者凭空捏造。
董方卓良好的英语水平可以在一则接受亚美尼亚记者赛季末采访的视频中清楚的体现。
