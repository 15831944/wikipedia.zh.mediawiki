'''董万瑞'''（{{bd|1941年|3月|2017年|2月9日|catIdx=D}}），[[山西省|山西省]][[翼城县|翼城县]]中卫乡董家坡人，[[中国人民解放军中将|中国人民解放军中将]]<ref name=qq/>。

==生平==
早年参加[[中国人民解放军|中国人民解放军]]。曾任步兵某师师长。1993年任[[中国人民解放军陆军第三十一集团军|陆军第三十一集团军]]军长。1996年任[[南京军区|南京军区]]副司令员，并兼任南京军区国防动员委员会常务副主任、南京军区党委常委。1997年晋升中将军衔<ref name=qq>{{Cite web |url = http://news.qq.com/a/20170210/032676.htm?winzoom=1 |title = 98抗洪前线，那位眼含热泪致敬抗洪战士的将军走了 |author =  |date = 2017-02-10 |publisher =腾讯新闻  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.sina.com.cn/c/268736.html |title =  南京军区召开国防动员委员会会议|author =  |date =2001-06-04  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。他是[[第十届全国人大代表|第十届全国人大代表]]，任内曾提出《[[中华人民共和国国防动员法|中华人民共和国国防动员法]]》立法提案<ref name=qq/>。

1998年8月至9月，在[[九八抗洪|九八抗洪]]中，董万瑞奉南京军区命令，担任南京军区九江抗洪前线总指挥，是取得九江抗洪胜利的主要功臣之一。董万瑞的儿子董三榕时任陆军第三十一集团军某部少尉排长，当时与父亲在同一个大堤上抗洪抢险，被传为佳话<ref name=qq/><ref>{{Cite web |url = http://news.sohu.com/20081021/n260149016_1.shtml |title = 回顾98抗洪历程：惊心动魄的岁月 |author =  |date = 2008-10-21 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref name=wy/>。

董万瑞还是军事题材电视剧《[[DA师|DA师]]》的总顾问。

2017年2月9日，董万瑞在[[南京|南京]]病逝，享年75岁<ref name=wy>{{Cite web |url =http://news.163.com/17/0210/18/CCUCOI2I00018AOR.html  |title = 98抗洪前线，那位眼含热泪致敬抗洪战士的将军董万瑞走了 |author =  |date = 2017-02-10 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.ifeng.com/a/20170210/50675904_0.shtml  |title = 98抗洪前线，那位眼含热泪的将军走了 |author =  |date = 2017-02-10 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref>{{cite news|title=董万瑞同志逝世|url=http://cpc.people.com.cn/n1/2017/0404/c87393-29187570.html|accessdate=2017-04-12|agency=人民网}}</ref>。
