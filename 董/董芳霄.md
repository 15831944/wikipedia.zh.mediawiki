'''董芳霄'''（{{bd|1986年|1月23日|||catIdx=Dong,董}}），（实际出生年份为1983年<ref>[http://sports.163.com/10/0430/15/65HE6NO800051CAQ.html 董芳霄妈妈:女儿就是83年出生 这就是中国国情]</ref>）女，[[中国|中国]][[河北省|河北省]][[唐山市|唐山市]]人。<ref>[http://www.ccvic.com/tiyu/zonghetiyu/20100429/145384.shtml 董芳霄个人简介及体操战绩]</ref>，前中国体操运动员，強項是自由體操，目前在[[新西蘭|新西蘭]]擔任體操教練。

== 运动员经历 ==

董芳霄4岁起进入唐山一所体操幼儿园接受体操训练。1994年开始接受专业训练， 1996年进入中国国家体操队，1999年天津世锦赛和2000年[[2000年夏季奥林匹克运动会|悉尼奥运会]]上代表中国获得了女子体操团体 两枚铜牌。
2001年[[2001年夏季世界大學生運動會|世界大学生运动会]]团体金牌和[[自由体操|自由操]]的金牌，2001年中国第9届[[中华人民共和国第九届运动会|全运会]]团体金牌和自由体操金牌。2002年因[[股骨头坏死|股骨头坏死]]退役<ref name="南周">{{cite news|url=http://www.infzm.com/content/45187|author=叶伟民|title=为国争光的董芳霄|date=2010-05-20|publisher=南方周末|language=zh|accessdate=2010-05-21}}</ref>。

退役后董芳霄一直处于失业状态，2004年进入北京第二外国语大学法文系进修<ref name="南周" />。

== 裁判员经历 ==
2008年[[北京奥运会|北京奥运会]]时担任跳马项目技术官员。
== 家庭 ==
董芳霄与新西兰籍丈夫特雷现住[[新西兰|新西兰]][[北島_(紐西蘭)|北岛]][[怀卡托|怀卡托]]地区的[[亨特利|亨特利]]，董已经获得新西兰永久居民权。

== 年龄疑云 ==
国际奥委会宣布，由于年龄造假，褫奪中国体操女队2000年悉尼奥运会团体铜牌。国际奥委会还取消了体操运动员董芳霄悉尼奥运会的所有比赛成绩。董芳霄當時申報年齡為17岁，但国际体操联合会调查发现她当时只有14岁，远远低于最低参赛年龄规定的16岁。此前国际体联取消了董芳霄1999年-2000年所有重大国际比赛中获得的成绩。她的队友杨云因证据不充分，被处以警告处罚。国际奥委会责令中国奥委会尽快将女子体操团体铜牌归还，以便能够重新颁发给当时排在第四位的美国队。<ref>[http://sports.ynet.com/view.jsp?oid=65188781 北青网新闻：董芳霄年龄造假遭严惩 国际大赛成绩全清零]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://news.xinhuanet.com/sports/2010-04/29/c_1263244.htm 中國體操女隊年齡造假遭嚴懲 雪梨奧運銅牌被剝奪] {{webarchive|url=https://web.archive.org/web/20100502230900/http://news.xinhuanet.com/sports/2010-04/29/c_1263244.htm |date=2010-05-02 }}</ref>
