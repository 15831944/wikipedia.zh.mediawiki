'''塗師村'''是[[中華民國|中華民國]][[嘉義縣|嘉義縣]][[六腳鄉|六腳鄉]]轄下的行政區，位於嘉義縣東部。東接[[雙涵村|雙涵村]]、南接[[蒜頭村|蒜頭村]]、[[溪厝村|溪厝村]]、西接[[潭墘村|潭墘村]]、北接[[三義村|三義村]]，全村分為五個主要聚落：大塗師、檳榔宅、竹圍仔、三姓王和北圳。

面積約1.59平方公里，行政區包含12個鄰，270戶，人口數為732人。區內的[[九玄宮|九玄宮]]、[[福安宮|福安宮]]、龍鳳宮、[[龍鳳井|龍鳳井]]、文昌祠為該村景點<ref name=":0">{{Cite web|url=https://lcss.epa.gov.tw/LcssViewPage/Responsive/CommunityDetail.aspx?Id=B2B2A1C9718EAD0212B6909167252421|title=低碳永續家園資訊網-嘉義縣六腳鄉塗師村簡介|accessdate=2017-12-17|author=|date=|publisher=}}</ref><ref name=":1">{{Cite web|url=http://lioujiao.cyhg.gov.tw/cp.aspx?n=29D1A6CC2883568E&s=B1BC717B1C9DAA93|title=嘉義縣六腳鄉公所-塗師村|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

== 地理 ==
坐落於嘉南平原上嘉義縣六腳鄉，東接雙涵村、南接蒜頭村、溪厝村、西接潭墘村、北接三義村，全村分為五個主要聚落；大塗師、檳榔宅、竹圍仔、三姓王和北圳<ref name=":0" /><ref name=":1" />。

== 聚落歷史 ==
塗師村開拓年代可追溯至明朝年間，最早稱為「土獅子」。村內主要有五個聚落，以大塗師聚落為最大，北圳聚落是侯姓為主的宗族聚落。

早期居民普遍以務農維生，主要種植花生、稻米、玉米、蔬菜等作物，村民雖多以農耕為主，但另有竹編技藝作為家家庭共同的副業。竹編的主要原料為[[刺竹|刺竹]]、[[桂竹|桂竹]]，製成之成品如如竹簍。竹簍等竹編品大多批發至[[朴子市|朴子]]、[[北港鎮_(台灣)|北港]]、[[四湖鄉|四湖]]、[[元長鄉|元長]]、[[水林鄉|水林]]等各地商店銷售。

現在為高齡化農村型社區，農業區域占全村90%以上。近年農業轉向精緻農業<ref name=":2">{{Cite web|url=https://baacho.weebly.com/-2039922570276652592523621-226152410721271223232039923478336742814928304.html|title=嘉義北圳侯家莊|accessdate=2017-12-17|author=|date=|publisher=}}</ref><ref name=":0" />。

== 文化資源 ==
塗師村內有三座宮廟:九玄宮、福安宮、龍鳳宮。
* 九玄宮：主祀神明為九天玄女，陪祀神明為土地公和註生娘娘，寺廟性質為道教民間寺廟

* 福安宮：主祀神明土地公。建立於1909年，日治時期明治42年，為道教民間寺廟。

* 龍鳳宮: 主祀神明有觀世音菩薩、廣澤尊王、九天玄女、天上聖母，陪祀神明為土地公。於1980年創立，主祀神明由[[福建鳳山寺|福建鳳山寺]]請回。為一道教民間寺廟<ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/lioujiau/1008008-JXG|title=文化資源地理資訊系統-九玄宮|accessdate=2017-12-17|author=|date=|publisher=}}</ref><ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/lioujiau/1008008-FAG|title=文化資源地理資訊系統-福安宮|accessdate=2017-12-17|author=|date=|publisher=}}</ref><ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/lioujiau/1008008-LFG|title=文化資源地理資訊系統-龍鳳宮|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

== 景點 ==
塗師村內有兩個主要景點：
* 龍鳳井：為一口建立三百多年的古井，整座井由俗稱「[[壓船石|壓船石]]」的紅磚塊堆疊而成。當地人世代相傳，只要將這口被觀音佛祖下旨封為「龍鳳井」的古井中乾淨的井水拿來插香祭拜，井水會散發濃濃藥香，病人喝下，有助改善病情，此井坐落於塗師村北圳聚落的中心，侯氏古厝北方。

* 文昌祠：又稱聖人公祠，位於塗師村北圳聚落東北邊，供奉一位於光緒年間受禮聘教育北圳聚落學子，卻因為過度辛勞不幸英年早逝的林姓老師。林姓老師移回故鄉[[新港鄉|新港鄉]][[溪北村_(新港鄉)|溪北村]]安葬，當地人為感念老師的功勞，遂蓋一間面向溪北方向的小祠，牌位書寫“聖人先師”供奉。於民國50年(1961年)原址修復重建<ref>{{Cite web|url=http://www.forgemind.net/phpbb/viewtopic.php?f=24&t=9629|title=準建築人手札網站|accessdate=2017-12-17|author=|date=|publisher=}}</ref><ref name=":2" />。

== 其他特色(ex特殊產業、相關文學影視作品、出身名人) ==
產業:
* [[竹編藝術|竹編藝術]]: 早期村民雖多以農耕為主，但另有竹編技藝作為家家庭共同的副業，但隨著塑膠產品的普及，此一傳統技藝已漸漸沒落、失傳。 
出身名人：
* [[劉啟光|劉啟光]](1905~1968)：字明遠，本名侯朝宗。日治明治38年(1905年)出生於大塗師聚落。抗日名人，經常發表公開演說反對日本財閥剝削農民。民國16年(1927年)，被逮捕禁錮，出獄後，仍以不屈不撓、繼續發表反日言論。在日本計畫逮捕之際，得朴子西醫師[[林瑞西|林瑞西]]及摯友[[李天生|李天生]]等人掩護與資助下逃亡中國。返國後易名劉啟光，擔任新竹縣長和[[華南銀行|華南銀行]]董事長，於[[二二八事件|二二八事件]]中協助國民政府[[清鄉|清鄉]]。晚年失意於政壇，自號了然居士。民國57年(1968年)因心臟病發逝世，享年64歲<ref>{{Cite web|url=http://www.shtes.cyc.edu.tw/old/modules/tadnews/index.php?nsn=93|title=嘉義縣六腳鄉蒜頭國民小學-塗師村|accessdate=2017-12-17|author=|date=|publisher=}}</ref><ref>{{Cite web|url=https://zh.wikipedia.org/wiki/%E5%8A%89%E5%95%9F%E5%85%89|title=維基百科-劉啟光|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。
