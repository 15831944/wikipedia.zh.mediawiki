'''宏安集團有限公司'''，（'''Wang On Group Limited'''，{{hkex|1222}}），簡稱'''宏安集團'''，創立於1987年<ref>{{cite web|url=http://www.wangon.com/new/b5_corporate4.asp|title=企業資訊|publisher=宏安集團|date=2016年|deadurl=yes|archiveurl=https://web.archive.org/web/20160911012852/http://www.wangon.com/new/b5_corporate4.asp|archivedate=2016-09-11}}{{zh-hant}}</ref> ，是[[香港上市公司|香港上市公司]][[位元堂|位元堂]]的[[控股公司|控股公司]]，持有位元堂28.2%股權，亦為[[盧森堡大藥廠|盧森堡大藥廠]]（即[[珮夫人|珮夫人]][[咳藥水|咳藥水]][[品牌|品牌]]的持有人）<ref>{{cite web |url=http://www.wangon.com/buz_waiyuentong_c.html |title=存档副本 |accessdate=2008-12-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20081210041051/http://www.wangon.com/buz_waiyuentong_c.html |archivedate=2008-12-10 }}</ref>。另外宏安集團是[[萬有街市|萬有街市]]的經營者，其[[外判|外判]][[承包|承包]]：[[荃灣|荃灣]][[麗城花園|麗城花園]]、[[馬鞍山|馬鞍山]][[頌安邨|頌安邨]]、[[東涌|東涌]][[富東邨|富東邨]]等多處[[街市|街市]]經營權。<ref>{{cite web |url=http://www.wangon.com/buz_allmartportfo_c.html |title=存档副本 |accessdate=2008-12-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20081210041043/http://www.wangon.com/buz_allmartportfo_c.html |archivedate=2008-12-10 }}</ref><ref>{{cite web |url=http://www.wangon.com/buz_propdevelopment_c.html |title=存档副本 |accessdate=2008-12-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20090105185835/http://www.wangon.com/buz_propdevelopment_c.html |archivedate=2009-01-05 }}</ref>宏安亦持有[[天水圍|天水圍]][[俊宏軒|俊宏軒]]商場、西營盤[[薈溱|薈溱]]及[[油塘|油塘]][[四山街|四山街]]13及15號工業地皮等。

2012年5月31日宏安集團成功收購深水埗營盤街140-142號舊樓，地盤合共佔地約4624方呎，以地積比率九倍計，料重建樓面約41616方呎，預計將發展與旗下紅磡薈點相若類型的住宅。

2013年1月29日土地註冊處資料顯示，宏安（01222）旗下旺角彌敦道724至726號銀座式商廈8樓、11樓及12樓分別以6600萬、5610萬及5610萬元易手。

宏安於2010年以3.2億元向科達地產主席湯君明購入漢英大廈，然後改建為一幢25層高銀座式商廈，並拆售其中7至25樓，合共16層樓面，每層面積約2200方呎，預期以餐廳、零售等作主題，並於2014年落成。

2014年6月25日宏安集團夥拍錦華實業，成功投得沙田市地段第599號地皮。<ref>[https://hk.finance.yahoo.com/news/%E5%9C%B0%E7%9A%AE%E9%96%8B%E6%A8%99-%E9%A6%AC%E9%9E%8D%E5%B1%B1%E9%A6%AC%E9%8C%A6%E8%A1%97-%E9%99%90%E9%87%8F-%E4%BD%8F%E5%AE%85%E5%9C%B0%E7%94%B1%E5%AE%8F%E5%AE%89%E4%BB%A57-04%E5%84%84%E5%A5%AA%E5%BE%97-092459537.html 《地皮開標》馬鞍山馬錦街「限量」住宅地由宏安以7.04億奪得] infocast，2014年6月25日</ref>8月13日，雙方再次合作，成功投得沙田市地段第598號地皮。<ref> [http://www.etnet.com.hk/www/tc/news/categorized_news_detail.php?newsid=ETN240813996&page=1&category=latest 【本港樓市】億京39億奪荃灣地遜預期，宏安4.3億奪馬鞍山地] 經濟通，2014年8月13日</ref>

2015年1月7日，宏安集團成功投得沙田市地段第587號地皮。<ref>[http://cn.reuters.com/article/cnBizNews/idCNL3S0UM30K20150107?sp=true 香港房地产：宏安集团以7.73亿港元中标沙田一幅住宅土地] 路透社，2015年1月7日</ref>

2017年6月14日[[旭輝控股|旭輝控股]]以約6.645億港元，向[[宏安地產|宏安地產]]收購[[油塘|油塘]][[四山街|四山街]]13及15號50%權益，項目土地面積41,080平方呎，可建樓面面積約27.2萬平方呎。

2017年6月22日[[宏安地產|宏安地產]]以轉讓公司形式元向[[亞洲聯合基建控股|亞洲聯合基建控股]]90%和[[俊和發展集團|俊和發展集團]]10%購入馬鞍山白石耀沙路地皮，即沙田市地段第601號的所有股權，佔地面積252954平方呎，地積比率約1.53倍，預計總樓面面積可達約387,500平方呎，雙方都未有披露成交金額，市場估計應接近29億元，樓面地價約7,500元。

2017年9月12日[[宏安地產|宏安地產]]以24.4125億元向[[碧桂園|碧桂園]]旗下全資附屬公司[[恒宙國際|恒宙國際]]出售馬鞍山白石耀沙路地皮，即沙田市地段第601號的60%股權，交易完成後，宏安地產將仍持有地皮40%權益。

2018年4月11日，[[宏安地產|宏安地產]]以8.673億元成功投得青衣寮肚路與亨美街交界(青衣市地段第192號)用地，地皮面積約為14376平方呎，以最高可建樓面102145方呎計算，每呎樓價地價為8492元，中標者須自費興建公共交通總站<ref>[http://www.etnet.com.hk/mobile/tc/news/categorized_news_detail.php?category=latest&newsid=ETN280411917 【本港樓市】宏安地產以８﹒６７億元奪青衣住宅地皮] 經濟通，2018年4月11日</ref>
