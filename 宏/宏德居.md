'''宏德居'''（'''Walton Estate'''），是[[香港|香港]][[居者有其屋|居者有其屋]]屋苑，由凱聯有限公司發展，及由香港房屋委員會負責監管整個發展過程及推售，由關善明建築師樓有限公司設計，位於香港[[柴灣|柴灣]][[柴灣道|柴灣道]]343號，鄰近永利中心，[[樂軒臺|樂軒臺]]及[[怡翠苑|怡翠苑]]。在1982年落成，屬於居者有其屋第3期甲，為本港第3個私人機構參建居屋計劃，亦是香港無需[[補地價|補地價]]買賣的居屋之一<ref>{{Cite web |url=http://valuers.hk/property-valuation-hos/1523.html |title=毋需補地價的居屋 |access-date=2013-04-12 |archive-url=https://web.archive.org/web/20140222022320/http://valuers.hk/property-valuation-hos/1523.html |archive-date=2014-02-22 |dead-url=yes }}</ref>。

== 物業資料 ==
* 出售期數：第3期甲
* 落成年份：1982
* 樓宇類型：私人機構參建居屋計劃
* 發展商：香港房屋委員會、凱聯建築
* 樓宇座數：4
* 樓宇名稱：[[德福樓|德福樓]]、[[德禧樓|德禧樓]]、[[德祿樓|德祿樓]]、[[德壽樓|德壽樓]]
* 單位數目：760
* 單位建築面積（平方米）：43-52
* 單位實用面積（平方米）：36-43
* 首次推出售價（$）：165,300-232,600
* 入伙日期：21/08/1982
* 管理公司：創毅物業服務顧問有限公司(7/2016起)* 物業設施：中西餐飲食肆、兒童遊樂場、停車場、商場、游泳池、銀行

== 組成 ==
宏德居由4座樓宇組成，外牆以深啡色為主，每層有8個建築面積466至599平方呎單位組成，全部為2房2廳間隔單位，因物業為1982年前的私人機構參建[[居屋|居屋]]計劃，宏德居單位轉讓並不需要向[[香港房屋委員會|香港房屋委員會]]申請補[[地價|地價]]。

== 早期物業環境 ==
早於80年代，因宏德居為區內最新物業，加上物業本身坐落柴灣旺區，因而發展出很多商業活動在宏德居附近，宏德居通往德祿樓，德福樓之間通道因而成為小販擺賣之地方，亦因此成為區內數一數二之露天街市，當時市政局為解決非法擺賣之問題，在宏德居近德壽樓附近興建柴灣臨時街市，可惜使用率奇低。早期中國國貨有限公司亦位於宏德居商場2樓（前身為[[香港創價學會|香港創價學會]]，於2012年改為百佳超級市場）至90年代，政府決定在宏德居附近，柴灣臨時街市對面興建[[柴灣市政大廈|柴灣市政大廈]]，內設永久街市，以解決近20年來非法擺賣之問題，柴灣臨時街市拆卸改建公園，因此宏德居附近環境衛生得以改善。
