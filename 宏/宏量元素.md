'''-{zh-hans:常;zh-hant:宏}-量元素'''，又称'''-{zh-hans:宏; zh-hant:常}-量元素'''、'''大量元素'''，指在体内含量丰富的元素。

==定义==
宏量元素指在体内含量占[[生物|生物]]體總[[質量|質量]]0.01%以上的[[化学元素|化学元素]]，計有[[碳|碳]]、[[氫|氫]]、[[氧|氧]]、[[氮|氮]]、[[磷|磷]]、[[硫|硫]]、[[氯|氯]]、[[鉀|鉀]]、[[鈉|鈉]]、[[鈣|鈣]]和[[鎂|鎂]]，這些元素在人體中的含量均在0.04%－62.8%之間，這11種元素共占[[人|人]]體總質量的99.97%。

在天然的條件下，[[地球|地球]]上或多或少地可以找到90多種元素，根據目前掌握的情況，多數科學家比較一致的看法，認為生命必需的元素共有28種，在28種[[生命元素|生命元素]]中，按體內含量的高低可分為宏量元素和[[微量元素|微量元素]]。

==各种宏量元素在人体内的作用==
[[File:Linolenic-acid-3D-vdW.png|250px]][[α-亞麻酸|α-亞麻酸]]的分子含有其中三个宏量元素：碳、氢、氧。]]
人体内拥有11种宏量元素，其中四种组成了身体的大部分：[[碳|碳]]、[[氫|氫]]、[[氧|氧]]和[[氮|氮]]。

===碳===
{{see also|有机碳化合物}}
碳是在体内最丰富的四种元素之一。作为[[生命|生命]]的必须元素，碳和其它物质（一般是其它生命主要元素）组成了所有的[[有机化合物|有机化合物]]。碳之所以可以组成有机化合物的骨架，是因为[[碳原子|碳原子]]的[[共价键|共价键]]可以与其它碳原子连接成为长碳链。<ref>{{Cite web |url = http://chem188.cn/Article/ShowArticle.asp?ArticleID=5905 |title = 有机物中碳原子的成键特点 |date = 2013-03-17 |publisher = 快乐化学网 |accessdate = 2014-07-20 |archive-url = https://web.archive.org/web/20140728104734/http://chem188.cn/Article/ShowArticle.asp?ArticleID=5905 |archive-date = 2014-07-28 |dead-url = yes }}</ref>

===氫===
[[File:Ethane-A-3D-balls.png|150px]][[乙烷|乙烷]]的分子]]
[[File:Ethanol-3D-balls.png|150px]]的分子比乙烷多了一个[[氧|氧]]原子]]
氢是另外一中生命要素。有机化合物和[[水分子|水分子]]都包括至少一个氢原子，其中[[水|水]]在人体内占了大约70％。水可以帮助排毒并辅助代谢，水还可以保护器官，使其正常运作。<ref>{{Cite web |url = http://www.chinadmd.com/file/os6upuawrprceievwrxxxpwt_1.html |title = 人体水 |publisher = 中华文本库 |accessdate = 2014-07-20 |archive-url = https://web.archive.org/web/20150923202808/http://www.chinadmd.com/file/os6upuawrprceievwrxxxpwt_1.html |archive-date = 2015-09-23 |dead-url = yes }}</ref>氢原子可以和碳原子接触形成[[共价键|共价键]]，共价键使氢和碳原子组成[[碳氢化合物|碳氢化合物]]，一种由氢和碳组成的有机化合物。碳氢化合物可以经过一些[[化学反应|化学反应]]衍生成为其它有机物。

===氧===
[[氧|氧]]在大多数有机物中都存在，它也是组成水的元素之一。在一些有机物（例如[[甲酸|甲酸]]）内，氧原子和其它原子组合成为[[含氧酸|含氧酸]]，其中有一些是[[有机酸|有机酸]]。<ref>{{Cite web |url = http://www.organicchem.com/show.asp?id=2411 |title = 含氧酸 |date = 2010-08-20 |publisher = 有机化学网 |accessdate = 2014-07-20 }}</ref>含氧的酸性[[羧基|羧基]]可以帮助[[脂肪酸|脂肪酸]]和[[甘油|甘油]][[脱水缩合|脱水]]结合成为[[三酸甘油酯|三酸甘油酯]]，一种储存多余能量的方式。[[人体|人体]]内基本上所有有机化合物都含有氧原子（碳氢化合物没有氧原子，但是[[人体|人体]]内只有微量[[益生菌|肠道细菌]][[代谢|代谢]]后的[[甲烷|甲烷]]）。

===氮===
[[File:Glucagon.png|100px]]，一种简单的[[肽|肽]]，是由含氮的氨基酸组成的]]
虽说氮是[[大气层|大气层]]中含量最丰富的[[气体|气体]]，但是人体无法利用[[氮气|氮气]]。人体内的有机氮来源于一些[[固氮生物|细菌]]经过[[固氮作用|固氮作用]]形成的[[氨|氨]]。这些氨溶解于水后经过植物根部以[[铵|铵]]的形式被吸收。氮和其它元素组成了[[生物鹼|生物碱]]、[[氨基酸|氨基酸]]和[[碱基|核碱基]]。氨基酸可以组成[[蛋白质|蛋白质]]，而核碱基可以于[[磷酸|磷酸]]和[[脱氧核糖|脱氧核糖]]或[[核糖|核糖]]组成[[DNA|DNA]]和[[RNA|RNA]]。

===磷===
[[File:Phospholipid_TvanBrussel.jpg|200px]]是组成细胞膜的主要成分。]]
[[磷|磷]]是一种以[[磷酸|磷酸]]的形式由[[植物|植物]]的[[根|根]]吸收。磷酸在人体内十分重要，磷酸和其它化合物组成了[[生物|生物]]必须的[[DNA|DNA]]（储存[[基因|基因]]）、[[RNA|RNA]]（使DNA的基因转化为[[蛋白质|蛋白质]]）、[[三磷酸腺苷|三磷酸腺苷]]（储存和制造[[能量|能量]]）、[[磷酸钙|磷酸钙]]（组成[[骨骼|骨骼]]和[[牙齿|牙齿]]）和[[磷脂|磷脂]]（组成[[细胞膜|细胞膜]]）。磷的存在和生活息息相关，例如围绕[[神經細胞|神经细胞]][[轴突|轴突]]的[[髓鞘|髓鞘]]（由[[鞘磷脂|鞘磷脂]]组成）可以使[[神经冲动|神经冲动]]得以传播进入[[器官|器官]]（例如[[肌肉|肌肉]]和[[腺体|腺体]]）。<ref>[http://www.term.gov.cn/pages/homepage/result2.jsp?id=333797&subid=10001798&subject=%E8%84%82%E8%B4%A8&subsys=%E7%94%9F%E7%89%A9%E5%8C%96%E5%AD%A6%E4%B8%8E%E5%88%86%E5%AD%90%E7%94%9F%E7%89%A9%E5%AD%A6]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>


===硫===
半胱氨酸、[[甲硫氨酸|甲硫氨酸]]和牛磺酸等一些含有氨基的[[有机酸|有机酸]]里面有[[硫|硫原子]]。而牛磺酸可以抑制神经冲动，用于控制[[痉挛|痉挛]]，而含硫的[[氨基酸|氨基酸]]组成了[[蛋白质|蛋白质]]和[[生命|生命]]重要的一部分：甲硫氨酸可以增进食欲，标识蛋白质的形成，并携带人体需要的硫和[[甲基|甲基]]；半胱氨酸可以帮助解毒和修复[[放射线|放射线]]损害。

===氯===
血液和細胞內液電解質主要成分，也是胃酸的主要成分。

===鉀===
血液和細胞內液電解質主要成分。

===鈉===
血液和細胞外液電解質主要成分。

===鈣===
骨和牙齒的結構; 作用在細胞信號傳導，代謝，組織維持中。

===鎂===
骨頭結構中重要的角色。
