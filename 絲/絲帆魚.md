'''絲帆魚'''（[[学名|学名]]：{{lang|la|''Nematistius pectoralis''}}）是'''丝帆鱼科'''（{{lang|la|Nematistiidae}}）及'''絲帆魚屬'''唯一的[[物種|物種]]{{r|FishBase1|FishBase2|FishBase3}}，属于[[辐鳍鱼纲|辐鳍鱼纲]][[鲹形目|鲹形目]]{{r|Deepfin|PLoS}}。

== 特徵 ==
體色偏銀灰色，背部具有黑帶，硬棘向後延伸看似雞冠。可達120公分，重45公斤。卵生，雌雄異體。{{r|FishBase3}}

== 分布 ==
[[太平洋|太平洋]]東側沿岸，從[[下加利福尼亞半島|下加利福尼亞]]到[[秘鲁|祕魯]]。{{r|FishBase3}}

== 經濟利用 ==
是一種釣魚比賽常用魚種，亦是[[美国|美國]]西岸海釣著名魚種。{{r|FishBase3}}
