'''親愛精誠'''是[[黄埔军校|黄埔军校]]校训<ref>[http://www.huangpu.org.cn/lzp/201209/t20120922_3115218.html 黄埔军校校训--親愛精誠, 黄埔军校同学会网站]</ref>，也是[[政治大學|政治大學]]自1927年成立於南京時制定並沿用至今的校訓。在中国大陆，[[广州市第六中学|广州市第六中学]]亦以此作为校训。

==历史==
黄埔军校于1924年6月16日第一期學生開學典禮時，[[孙中山|孙中山]]审定“親愛精誠”作为校训。<ref>[http://news.163.com/11/1104/15/7I1BIMU900014AED.html “親愛精誠”镌刻校训，《羊城晚报》2011-11-04]</ref>1925年[[元旦|元旦]]黄埔军校校长[[蒋介石|蒋介石]]訓話：「親愛」是要所有的革命同志能「相親相愛」，「精」是「精益求精」，「誠」是「誠心誠意」<ref>{{cite web |language=zh-tw |title=中華民國陸軍軍官學校 校訓 |url=http://www.cma.edu.tw/ac_motto.php |location=高雄 |publisher=中華民國陸軍軍官學校 |date=2012-12-25 |accessdate=2012-12-27}}</ref>。“親愛精誠”四字中，“精”字少一點，“愛”字多一點，隱喻“少一點精明，多一點愛”。

1924年第一版[[黄埔军校校歌|黄埔军校校歌]]《陆军军官学校校歌》“莘莘学子，亲爱精诚”；<ref name="huangpu3">{{Cite web |language=zh-cn |title=黃埔校歌 |url=http://www.huangpu.org.cn/hpjx/201206/t20120613_2739396.html|location=北京 |publisher=黃埔軍校同學會 |accessdate=2016-11-06}}</ref>1926年版《中央军事政治学校校歌》“亲爱精诚，继续永守”。現同時為[[廣州市第六中學|廣州市第六中學]]校歌<ref>{{cite web |language=zh-cn |title=黄埔军校校歌 |url=http://www.hoplite.cn/Templates/hpjxwx0042.htm |location=北京 |publisher=中國黃埔軍校網 |accessdate=2014-06-25}}</ref><ref>{{cite web |language=zh-tw |title=校歌勒石 |url=http://www.cma.edu.tw/ac_song.php |location=高雄 |publisher=中華民國陸軍軍官學校 |date=2010-01-02 |accessdate=2013-03-05}}</ref><ref>{{cite web |language=zh-tw |title=黄埔軍校創校 |url=http://webarchive.ncl.edu.tw/archive/disk3/06/72/24/79/41/200806243042/20090505/web/cma.edu.tw/ch/cma_1_7.html |location=高雄 |publisher=中華民國陸軍軍官學校 |accessdate=2013-03-05}}</ref>
