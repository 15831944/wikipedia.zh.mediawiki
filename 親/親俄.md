'''親俄'''指在[[政治|政治]]、[[經濟|經濟]]、[[社會|社會]]、[[文化|文化]]各層面上對[[俄羅斯|俄羅斯]]抱持親近、欣賞、喜愛甚至認同的行為。對應詞為[[反俄|反俄]]。

== 概述 ==
對俄羅斯、俄羅斯文化等俄羅斯的因素在整體上持正面的看法。在政治上十分重視同俄羅斯的關係，認為俄羅斯對國際問題起著十分重要的作用（例如[[中華人民共和國|中華人民共和國]]）。

== 在西方世界中的親俄情緒 ==
在西方國家中的親俄情緒可能基於民眾受到大眾文化影響而產生的刻板印象（例如俄羅斯人待客友善等）等因素。美國作家羅伯特·亞歷山大曾說：“我喜歡俄羅斯的戲劇，它情感的本質。他們不懼怕愛情，不懼怕痛苦，不懾於浮誇和衝動。”<ref>[http://www.rusoffagency.com/authors/alexander_r/kitchen_boy/kitchen_boy_reading_guide.htm An Interview with Robert Alexander]</ref>

2004年10月，[[蓋洛普|蓋洛普組織]]公佈了一份調查結果<ref name=HelsSan>''[[Helsingin_Sanomat|Helsingin Sanomat]]'', October 11, 2004, [http://www.hs.fi/english/article/International+poll+Anti-Russian+sentiment+runs+very+strong+in+Finland/1076154202275 International poll: Anti-Russian sentiment runs very strong in Finland. Only [[Kosovo|Kosovo]] has more negative attitude]</ref>，統計出在[[西歐|西歐]]的居民中大約有20%的民眾對俄羅斯抱有正面的看法，其中“最正面”的看法來自[[冰島|冰島]]、[[希臘|希臘]]、[[英國|英國]]。在其他國家及地區的調查中，[[芬蘭|芬蘭]]、[[土耳其|土耳其]]、[[日本|日本]]為9%，[[立陶宛|立陶宛]]為38%，[[拉脫維亞|拉脫維亞]]為36%，[[愛沙尼亞|愛沙尼亞]]為34%。另外，愛沙尼亞尤其是拉脫維亞有大量的少數民族，這很可能會影響到調查結果。

{{main|烏克蘭親俄羅斯武裝衝突 (2014年－2015年)}}
[[烏克蘭|烏克蘭]]國內各地區的親俄比例存在很大差異。

烏克蘭的東部和南部的親俄人群較為龐大，而西部的民眾則多對俄羅斯抱有厭惡情緒<ref>[http://sankei.jp.msn.com/world/news/140301/erp14030108480008-n2.htm {{lang|ja|緊迫のクリミア・ルポ　「ロシア編入」求める住民　「権利守るのはプーチンだけ」}}] {{webarchive|url=https://web.archive.org/web/20140407082609/http://sankei.jp.msn.com/world/news/140301/erp14030108480008-n2.htm |date=2014-04-07 }}</ref><ref name=sinro>[http://sankei.jp.msn.com/world/news/140226/erp14022619320004-n1.htm {{lang|ja|大統領選　親欧派が優位　革命かクーデターか　ウクライナ分裂危機}}] {{webarchive|url=https://web.archive.org/web/20140907204157/http://sankei.jp.msn.com/world/news/140226/erp14022619320004-n1.htm |date=2014-09-07 }}</ref>。烏克蘭政府高層則傾向[[西方國家|西方國家]]，引起了東部親俄勢力的不滿。<ref name=sinro></ref>

從2014年開始，烏克蘭親俄派以[[烏克蘭親歐盟示威運動|烏克蘭親歐盟示威運動]]為契機，在烏克蘭東部的[[頓巴斯|頓巴斯]]地區成立了[[新俄羅斯|新俄羅斯]]，並和烏克蘭國內的親西方派及烏克蘭中央政府展開紛爭。

=== 東方各國 ===
==== 中國 ====
中國從[[國民政府|國民政府]][[第一次国共合作|時期]]開始就和當時的蘇聯保持着親密的關係。舊日本軍暴走奪權及侵略中國，是國民政府親俄所刺激出來的；蔣介石當時雖然[[清黨|清黨]][[剿匪|剿匪]]，但實際上則放任國民政府被共諜滲透，在戰略上也成為替蘇聯對抗日本的人肉盾牌<ref>[https://read01.com/0dgKkR.html 劉仲敬：抗戰打斷「黃金十年」是國民黨發明的神話][https://read01.com/NoeD3J.html 劉仲敬：蔣介石在日本人眼中是「蘇聯代理人」]</ref>。而在国民政府尚未直接管辖的新疆地区，[[由蘇聯支持的新疆伊斯蘭叛亂|苏联政府持续策动新疆叛乱]]，其影响持续至今。当时，[[新疆省_(中華民國)|新疆]]是[[俄罗斯人|俄罗斯人]]（包括苏联侨民）在中国的主要聚居地。1940年，由苏联政府大力支持下，新疆当地反叛、[[东突厥斯坦独立运动|分裂势力]]成功的发动[[三區叛亂|三区叛乱]]。俄罗斯人虽已归化中华民国，仍效忠于苏联，支持叛乱。

1920年代，蘇聯通過[[第三國際|第三國際]]建立[[中國共產黨|中國共產黨]]，並指示其與國民政府合作進行[[北伐戰爭|北伐戰爭]]和[[抗日戰爭|抗日戰爭]]，是為兩次[[國共合作|國共合作]]。在此期間兩黨亦爆發[[第一次國共內戰|第一次國共內戰]]，中國共產黨在[[中東路事件|中東路事件]]中站在蘇聯一邊，提出“[[武裝保衛蘇聯|武裝保衛蘇聯]]”的口號。1931年11月7日，中国共产党成立了受苏联控制的假缓冲国——[[中华苏维埃共和国|中华苏维埃共和国]]，定都[[瑞金|瑞金]]，是为[[中华人民共和国|中华人民共和国]]的前身。[[抗戰|抗戰]]期間，位於中國[[中国东北地区|東北地區]]的共產黨抗日武裝——[[東北抗日聯軍|東北抗日聯軍]]撤退至蘇聯境內後，被改編為蘇軍遠東方面軍第88獨立步兵旅。[[八月风暴|1945年8月，苏军进入中国东北地区彻底击败侵华日军]]，宣告了抗战胜利和[[第二次世界大戰|第二次世界大戰]]的全面结束。1946年其他蘇軍部隊撤出東北地區後，[[蘇聯海軍|蘇聯海軍]]則一直留在位於[[遼東半島|遼東半島]]的[[中国人民解放军海军旅顺基地|旅順海軍基地]]，直接影响中国历史走向。蘇聯除协助中國共產黨攻占东北地区外，亦帮助其取得[[第二次国共内战|第二次国共内战]]的胜利。在此期間[[北塔山事件|國民政府與亞洲第一個社會主義國家——蒙古人民共和國在北塔山發生軍事衝突]]，中國共產黨亦站在蒙古和蘇聯一邊。

1949年2月，[[斯大林|斯大林]]表示可随时应中国共产党的要求将苏联海军撤出[[中国人民解放军海军旅顺基地|旅顺海军基地]]。对此[[毛泽东|毛泽东]]表示反对：“从旅顺撤军的问题必须等到中国粉碎了反动势力，把人民动员起来，没收了外国资本，并在苏联的帮助下把国家治理得井然有序时再来考虑。等我们强大起来的时候，我们要签订类似苏波条约那样的互助条约。”1949年10月[[中華人民共和國|中華人民共和國]]成立后即宣佈與蘇聯結盟，兩國于1950年初簽訂《[[中蘇友好同盟互助條約|中蘇友好同盟互助條約]]》。苏联政府對中華人民共和國在軍事、经济等方面給予大力扶持，并支持中華人民共和國出兵[[朝鮮戰爭|朝鮮半島]]和[[越南戰爭|印度支那]]對抗[[美軍|美軍]]。苏联和东欧各国在1950年代初援助的[[156项重点工程|156项重点工程]]，更帮助中华人民共和国初步建立起自己的工业体系，促成[[中华人民共和国工业史|中国初步完成国家工业化]]。在两国关系的蜜月期，中华人民共和国政府在外交上奉行[[一边倒|一边倒]]政策。[[苏南冲突|蘇南交惡]]後，中華人民共和國選擇站在蘇聯一邊，並支持蘇聯出兵鎮壓[[匈牙利十月事件|匈牙利十月事件]]，力捧蘇聯在國際共產主義運動中為首的地位。

從中華人民共和國建國至1960年代初[[中蘇交惡|中蘇交惡]]為止，中華人民共和國實際上是蘇聯最大的[[衛星國|衛星國]]。早在建国前的1949年6月，[[毛泽东|毛泽东]]就曾经表示：“[[莫斯科|莫斯科]]是国际共产主义运动的最高统帅部，而中国共产党则只是一个方面军的司令部。根据局部利益服从整体利益的原则，我们中共服从苏联的决定。即使没有了[[第三国际|第三国际]]，中共也没有参加[[共产党和工人党情报局|共产党情报局]]，我们也必须这样做。如果中共与苏联方面发生争论，我们中共在说明我们的意见后，准备服从并坚决执行苏联的决定。”而建国使用的“人民共和国”这一国号本身就意味着把一切所谓“非人民”排斥在共和政体之外。这就是二战后包括中国在内的一切苏联卫星国──“[[蒙古人民共和国|蒙古人民共和国]]”，“[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]”，“[[波兰人民共和国|波兰人民共和国]]”，“[[匈牙利人民共和国|匈牙利人民共和国]]”，“[[阿尔巴尼亚人民共和国|阿尔巴尼亚人民共和国]]”，“[[保加利亚人民共和国|保加利亚人民共和国]]”等国的立国根据。无一例外，这些国家的国号都带有“人民”二字。这事实上是苏俄胎记。中华人民共和国政府和军队各部门均有苏联顾问把持，中国领土上也设有苏联海军基地。中华人民共和国政府在当时的各项外交政策方面亦与苏联完全保持一致，甚至1954年[[五四宪法|第一部《中华人民共和国宪法》]]的制定也是在苏联的帮助下完成的。这些都是明显的苏联卫星国特征。因此中國共產黨也被一些亲美仇俄者称为“[[黃俄羅斯計劃|黃俄]]”。

在對華經濟和民生援助方面，蘇聯在中國成立了兩國合营的中蘇石油公司、中蘇金属公司、中蘇造船公司和中蘇民用航空公司。蘇聯駐華經濟總顧問團的專家組負責對每一個生產部門進行指導。在對華軍事援助方面，根據《[[中蘇友好同盟互助條約|中蘇友好同盟互助條約]]》中的軍事條款，蘇聯獲得了繼續使用旅順海軍基地的權利，蘇聯海軍[[俄羅斯太平洋艦隊|太平洋艦隊]]第5分艦隊被部署在此地，幫助中華人民共和國保衛領海安全。此地一度成為蘇聯最大的海外軍事基地，駐軍人數曾達到一萬余人，一直保留至1956年蘇聯單方面宣佈撤出。在此期间的1953年2月22日，[[周恩来|周恩来]]曾率中华人民共和国政府代表团慰问驻扎该地的苏联海军官兵。1954年蘇聯政府提出撤軍的問題後，中華人民共和國政府曾一度反對蘇聯海軍撤離該地區，后因蘇聯方面堅持撤軍而作罷，但仍保留了蘇軍在未來必要的情況下重返該地區的條款。[[中國人民解放軍|中國人民解放軍]]亦獲得了大量蘇軍的一線裝備 ，其正規化建設完全依靠蘇聯駐華軍事總顧問團的指導進行，在解放軍團級以上單位和[[中國人民武裝警察部隊|公安軍]]總隊以上單位均設有蘇聯顧問進行指導工作。軍事工業部門在蘇聯顧問的幫助下也逐步掌握了製造大量蘇式裝備的能力。1957年8月2日，中国、苏联、朝鲜军队在中苏朝边境沿海地区举行了以陆军为主、海军舰队和空降兵协同的歼灭敌军濒海集团的方面军进攻战役实地演习，同时也进行了城市防御战役演习、防空演习和以封锁海峡为背景的岛屿防御战役演习。此外，蘇聯顧問和專家也為中國發展核武器提供了初步的幫助。

1950年代起，应苏联政府的要求，在中华人民共和国境内的苏联侨民除自动离开外，[[1950年代遣返苏联侨民|中华人民共和国政府开始遣返苏联侨民回国]]。[[新疆苏联侨民问题|新疆苏联侨民问题]]对新疆社会的直接影响持续至1960年代中期。少数遗留者，在[[中華人民共和國國籍法#历史背景|中华人民共和国政府不承认双重国籍后]]，被归为[[中国俄罗斯族|俄罗斯族]]。

1960年代初，两国因国家利益、意识形态的原因，关系迅速恶化，是为[[中苏交恶|中苏交恶]]。此后，两国长期敌对状态持续至[[苏联解体|苏联解体]]前夕。1960年代末[[中苏边界冲突|中苏边界冲突]]中的[[珍宝岛事件|珍宝岛事件]]将两国推向全面战争边缘。[[1979年中越战争|1979年中越战争]]发生于[[中国南方|中国南方]]边境，同时，中华人民共和国将在遥距战场数千公里、位于[[中国北方|北方]]边境的苏联视为防备重点。1980年代，[[中苏关系#关系缓和|中苏关系逐渐缓和]]。1989年[[六四事件|六四事件]]后，中华人民共和国外交环境恶化，[[对于六四事件的反应|西方国家普遍与中华人民共和国决裂]]。作为[[东方阵营|东方阵营]]领袖，苏联未参与其中。

1990年代初期蘇聯解体後，最主要的继承者俄罗斯仍是世界性大国，但国际影响力已逊于苏联。进入1990年代后期，[[中俄關係|中俄兩國關係愈加緊密]]。政府层面的经济、军事合作频繁。

[[2014年克里米亞危機|2014年克里米亞危機]]及之后的[[顿巴斯战争|乌克兰东部武装冲突]]爆发后，[[西方国家|西方国家]]反对俄罗斯吞并克里米亚并为此对俄罗斯实行经济制裁。中华人民共和国政府对事件中各方虽未有明确支持，但明确反对西方国家制裁俄罗斯，并对俄罗斯提供经济支持。2016年菲律宾和越南等国针对中国发起[[南海仲裁案|南海仲裁案]]，俄罗斯政府支持中华人民共和国政府对仲裁不予承认和不予接受的立场，并于当年派出海军同中国在南海水域举行了代号“[[海上联合-2016|海上联合-2016]]”的联合军演。持续至今的[[叙利亚内战|叙利亚内战]]是俄罗斯与美国直接参与的角力场。中华人民共和国政府、军队虽未直接参与其中，但中华人民共和国政府与俄罗斯共同支持叙利亚主政的[[阿萨德家族|阿萨德政权]]，并多次在联合国安理会上对有关制裁阿萨德政权的提案中投下反对票或弃权票。2018年俄羅斯更邀請中國參加在[[遠東地區|遠東地區]]舉行的“東方-18”大型軍演，兩國政治軍事關係的親密程度達到頂峰。

由于中国政府的亲俄传统，[[中俄关系|两国关系]]总体稳定和密切。大部分中国民众对俄罗斯的正面印象基本停留在1950年代的中苏蜜月期。虽然民间交流自[[中苏交恶|中苏交恶]]后便较为缺乏，但随着[[苏联解体|苏联解体]]后两国关系不断升温，中国民众对俄罗斯的关注和了解也逐步增多。两国在教育领域的合作亦逐步加深。2017年9月，两国合办的中俄联合大学在[[深圳|深圳]]正式开学。

==== 北韓 ====
[[北韓|北韓]]從[[金日成|金日成]]時代就開始和當時的蘇聯保持著極為親密的關係。蘇聯幫助北韓建國，並在軍事上大力扶持。[[中蘇交惡|中蘇交惡]]後，北韓選擇站在蘇聯一邊，兩國關係十分緊密。

[[金正恩|金正恩]]上台後，兩國關係愈加緊密。<ref>[http://www.news24.jp/articles/2014/11/18/10263576.html {{lang|ja|金第１書記の最側近、露訪問で関係強化へ 2014年（平成26年）11月18日}}]</ref>

==== 蒙古 ====
曾經的[[蒙古人民共和國|蒙古人民共和國]]是亞洲第一個[[社會主義國家|社會主義國家]]。當時蘇聯積極協助[[外蒙古獨立|外蒙古獨立]]、推動[[俄語|俄語]]和[[西里爾字母|西里爾字母]]在蒙古國內的發展，抹滅蒙古文字的地​​位。 [[中蘇交惡|中蘇交惡]]後，蒙古選擇站在蘇聯一邊，兩國關係十分緊密。
