《'''親親我的寶貝'''》（{{lang-ja|いのちの器}}）是[[上原希美子|上原希美子]]創作的漫畫，於1991年起在漫畫雜誌《{{link-ja|for Mrs.|フォアミセス}}》（[[秋田書店|秋田書店]]）上連載<ref>[https://kotobank.jp/word/%E3%81%84%E3%81%AE%E3%81%A1%E3%81%AE%E5%99%A8-695071 いのちの器]</ref>，至2018年底日文單行本已出版至第77集；共300話。本作1998年改拍為真人版電視劇<ref>[https://www.fujitv.co.jp/b_hp/inoti/ いのちの器 番組紹介]</ref>。

== 故事簡介 ==
婦產科醫生有吉響子繼承了養父在[[安曇野市|安曇野]]鄉間的婦產科醫院，歷經與丈夫離異又復合，種種人生悲喜，也在守護產婦與嬰兒的工作中，看見人性的光輝。

== 主要角色 ==
;有吉響子
:本作主角，婦產科醫生。母親千尋難產去世，其後接生響子的山野醫生收養了響子。
:養父去世後與丈夫晃從東京回到鄉下繼承家業。
;有吉晃
:響子的丈夫，與響子回到鄉下後難以適應而有了外遇，與外遇對象汐崎房江生有一女優子，並與響子離婚。
:爾後帶著優子回來與響子復合。
;有吉優子
:晃和汐崎房江所生的長女，由響子親自接生。10歲時得知房江才是自己生母。
:高中畢業後考上後母響子念過的名門T大醫學部，成為一名醫生。
;有吉雅志
:有吉家長男，山野醫生親生兒子鷹志之子，鷹志夫婦去世後，由響子夫婦收養。
:與花園和香結婚後，繼承了花園診所。
;有吉桃子
:有吉家次女，由響子在新幹線上接生，生母幸子產後送醫時去世。
:由於桃子是幸子和姊夫偷情所生，幸子之姊美子對桃子不甚友善，於是響子收養了桃子。
:國中時得知身世，但心中所認的父母依然是有吉夫婦，後來與美子和解。
;有吉穗高
:有吉家次男，響子與晃的第一個親生子。
:留學歸國後，與友人一起創業。
;有吉五郎
:有吉家三男。響子與晃的第二個親生子。
;有吉春菜
:有吉家老么。生母春海難產死亡，其後被有吉家收養。
