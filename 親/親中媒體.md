== 香港 ==
在香港，親中媒體主要分為兩種，一類是直接中資背景的媒體，如《[[大公報|大公報]]》、《[[文匯報_(香港)|文匯報]]》<ref>{{Cite web |url=http://www.voacantonese.com/content/article-20111230h-k-based-beijing-envoy-criticizes-university-of-h-k-poll-136423953/936635.html |title=親中媒體祭出陰謀論 中聯辦官員批港大調查 |access-date=2013-08-01 |archive-url=https://web.archive.org/web/20140502010057/http://www.voacantonese.com/content/article-20111230h-k-based-beijing-envoy-criticizes-university-of-h-k-poll-136423953/936635.html |archive-date=2014-05-02 |dead-url=no }}</ref>、《[[香港商報|香港商報]]》及[[鳳凰衛視|鳳凰衛視]]等，強調「愛國愛港」的政治及中國官方立場。部分媒體的股權後來被香港親中商人、[[中資|中資]]商人或大陆商人收購，如[[星島日報|星島日報]]、[[成報|成報]]、[[電視廣播有限公司|電視廣播有限公司]]等。

此外亦有傾向支持[[香港特別行政區政府|香港特別行政區政府]]及偏向[[建制派_(香港)|建制派]]、較比大眾接受的主流媒體，如[[星島新聞集團|星島新聞集團]]、[[東方報業集團|東方報業集團]]、[[電視廣播有限公司|電視廣播有限公司]]（TVB）及旗下[[無綫新聞|無綫新聞]]等。此類型媒體部份生意來往或其持有者與中國內地有關聯，或其政治立場因為商業利益主動向中共靠攏。<ref>{{Cite web |url=http://epochweekly.com/b5/222/9285.htm |title=晾晾多家港媒老闆的政治面 |access-date=2013-08-01 |archive-url=https://web.archive.org/web/20130911015417/http://epochweekly.com/b5/222/9285.htm |archive-date=2013-09-11 |dead-url=no }}</ref>

== 困境 ==
通常，中國媒體在走出國門的的時候，也會遭遇西方國家的限制，例如在國際記者會時會限制提問時間及問題數目，原因是這些親中傳媒會積極提問，以阻止其他傳媒機構發言及提問的機會，特別是當報導內容涉及中國的時候。

=== 媒體網站 ===
{{See also|防火长城}}
香港、台灣及境外眾多親中傳媒，即使其親中立場，但由於其報導反對聲音相對中國大陸媒體持平，其官方網站及報章依然被中國干擾或完全封鎖。如中聯辦旗下[[大公報|大公報]]、[[香港文匯報|香港文匯報]]、[[香港商報|香港商報]]的網站是少數「解封」的港台新聞網站。
