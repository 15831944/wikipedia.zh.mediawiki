'''虹桥经济技术开发区'''，简称'''虹桥开发区'''，是[[国家级经济技术开发区|国家级经济技术开发区]]。位于[[上海市|上海市]][[长宁区|长宁区]]，东起[[中山西路|中山西路]]，西至[[古北路|古北路]]，北临[[仙霞路|仙霞路]]，南至[[虹桥路|虹桥路]]，规划用地0.652平方公里<ref>{{cite web|title=国务院关于上海市建立闵行和虹桥两个经济技术开发区的批复_政府信息公开专栏|url=http://www.gov.cn/zhengce/content/2012-12/31/content_6046.htm|website=www.gov.cn}}</ref>，名字取自[[虹桥镇_(上海市)|虹桥乡]]（当地原为虹桥乡马家角）。1979年开始规划，1983年启动建设，是面积最小的[[国家级经济技术开发区|国家级经济技术开发区]]，也是中华人民共和国境内唯一辟有[[领事馆|领事馆]]区的国家级开发区。上海地产闵虹（集团）有限公司下属的上海虹桥经济技术开发区联合发展有限公司受托统一负责虹桥开发区的开发建设和经营管理。<ref>{{cite web|title=上海虹桥经济技术开发区 - 投资上海|url=http://www.investment.gov.cn/aqff/20170406/57725.html|website=www.investment.gov.cn}}</ref>

== 历史 ==
{{wikisource|国务院关于上海市建立闵行和虹桥两个经济技术开发区的批复}}
1982年，[[上海市人民政府|上海市人民政府]]决定辟建以对外活动为主要特征的虹桥开发区，并决定采用合资企业统一开发、经营的模式实施。1983年，上海市闵行虹桥开发公司成立。1984年，上海市人民政府人民政府批设立虹桥开发区，位于[[虹桥镇_(上海市)|虹桥乡]]北部，占地0.652平方公里。1985年2月14日，上海虹桥联合发展公司成立并登记注册，由闵行虹桥开发公司和其他香港资本合资成立<ref>{{cite web|title=国家企业信用信息公示系统|url=http://www.gsxt.gov.cn/corp-query-entprise-info-primaryinfo-subpubsys-PROVINCENODENUM310000310000000081985021400013.html?entType=6110&subsite=100000|website=www.gsxt.gov.cn|accessdate=2018-04-23}}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1984年9月，[[上海县|上海县]][[漕河泾街道|漕河泾]]、[[龙华街道_(上海市)|龙华镇]]、[[北新泾镇|北新泾镇]]及龙华、[[梅陇镇|梅陇]]、[[虹桥镇_(上海市)|虹桥]]、[[新泾镇|新泾]]、[[北桥镇|北桥]]5乡177个生产队共49.98平方公里划属上海市区<ref name="shxz-jjbh">{{cite web|title=上海县志 >> 第一篇建置 >> （二）县域 >>  境域变化|url=http://www.shtong.gov.cn/newsite/node2/node4/node2250/shanghai/node53607/node53612/node53614/userobject1ai50766.html|website=上海通网站|accessdate=2017-07-31}}</ref>。

1986年，正式被[[中华人民共和国国务院|中华人民共和国国务院]]批准，成立国家级虹桥经济技术开发区<ref name="hqkfq">{{cite web|title=长宁区志 >> 第四编虹桥开发区、古北新区|url=http://www.shtong.gov.cn/newsite/node2/node4/node2249/changning/node15342/index.html|website=上海通网站}}</ref>，即现在[[长宁区|长宁区]][[虹桥街道_(上海市)|虹桥街道]]的大部分辖区。

2014年，上海虹桥经济技术开发区联合发展有限公司和[[闵行开发区|闵行开发区]]的开发实体上海闵行联合发展有限公司进行了资产重组，成立了上级控股公司，即上海地产闵虹（集团）有限公司<ref>{{cite web |title=国家级闵行、虹桥开发区成立30周年纪念 |url=http://www.smudc.com/news/news_pop.asp?id=1638 |website=www.smudc.com |accessdate=2018-07-17}}</ref>。
