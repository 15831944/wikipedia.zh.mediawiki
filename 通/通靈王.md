《'''通靈王'''》（{{lang-ja|シャーマンキング|''Shaman King''}}，台譯'''通靈-{}-童子'''，港譯'''通靈-{}-王'''）是一部以人類與[[靈魂|靈]]的關係，和其所發展出的冒險為基礎的日本[[漫畫|漫畫]]作品，作者为[[武井宏之|武井宏之]]。漫畫連載於1998年到2004年間在《[[週刊少年Jump|週刊少年Jump]]》上發行，後期因故而提前結束連載。直到2008年時漫畫再版重新發行『完全版』，並增加由武井宏之重新繪製的完整故事結局。

動畫版由漫畫改編，於2001年至2002年在[[東京電視台|東京電視台]]播放，之後由[[台灣|台灣]]、[[香港|香港]]、[[新加坡|新加坡]]等地分別取得代理翻譯權。

== 故事開端 ==
徘徊於世間的亡者之魂，在大地森林中生息的精靈，以及崇高的[[神|神佛]]們，他們被人類稱作為「靈」。
在人類的世界裡，有少部分的人能夠與他們自由的交流，並且能夠隨意操控某種常人無所可及的力量，這些人便通稱為「通靈人」。

羅侯星和計都星是[[秘教|秘教]]中象徵「破滅」的神祕之星。每500年會一起接近[[地球|地球]]，相傳會為地球帶來大災難。
每當此時，就會出現一位救世主，將500年一次的「破壞」轉為重大的「再生」。
「通靈人大賽」便是為此而每500年舉行一次，勝利者可得到代表一切的偉大精靈「The Great Spirit」的祝福，而成為「通靈王」。

== 故事簡介 ==
主角為13歲的通靈人麻倉葉，為了實現「能夠過著無憂無慮的生活」之願望和與未婚妻恐山安娜的約定，從老家[[出雲國|出雲]]出發到[[東京|東京]]修行。
越接近比賽開始之時，開始有來自世界各地的通靈人，懷抱各自的夢想聚集到日本東京參加東京通靈人預賽。
葉在這段期間內為了存活並變的更強，接受各式各樣的考驗，碰到各式各樣的夥伴與敵人，心靈與力量隨之成長茁壯。直到正式比賽開始，要跟隨帕契族出發到[[美國|美國]]時，遇到了一位自稱「未來王」的人，名叫葉王，野心是創造一個「只有通靈人的世界」，將所有的人類毀滅。而此時麻倉家為了阻止祖先傳下來的罪孽，將一切實情與秘笈「超．占事略決」交予安娜，要她前往通靈人大賽，並將這背負的使命傳達給葉。

然而樂天的葉，選擇了另一種做法。

== 用語 ==
===基本===
;通靈人 (シャーマン)
:具有靈能力、可與靈交流的人類的總稱。關於普世意義的薩滿，請見[[薩滿教|薩滿教]]。

;巫力
:通靈人的能力值，即是日本漫畫中常見的「強度數值化」。當要進行通靈術：如附身合體或超靈體（OVER-SOUL）等，都會消耗巫力，是一種精神力量。而每一個靈體均需要一定程度的巫力，如通靈者並無足夠巫力而強行使用超靈體，精神將會崩潰並失常。巫力在休憩後會自行回復，但每名通靈者的巫力都有一定的極限，其極限在修煉、[[瀕死|瀕死]]或[[復活|復活]]時均會提升。

;靈
:靈魂的總稱。包括人類的靈、動植物的靈、自然靈等各種不同種類。靈有等級之分：一般靈體→精靈、天使等級(一般靈死後六百年加上經過修練有機會達到，天使則是此等級中最接近神靈者)→神靈級(夏馬修,如來,明王...等),神靈等級中最高等則是精靈王，代表整顆星球的靈魂，五大精靈(火,雷,雨,風,土)則是由精靈王衍生出來，分別代表五大元素最強的靈。

;附身合體
:使靈體依附在通靈人身上的技能。通靈人在附身合體之後，能獲得該靈體在生前的某一些能力或知識，性格也會受到影響。

;超靈體（OVER-SOUL）
:通靈人將靈體依附在物體上並具體化進行攻擊，簡稱「O.S」，如麻倉葉將阿彌陀丸附在名刀春雨上，而靈體附身後一般均可具體化。

;通靈王大戰
:五百年一度由帕契族舉辦、由通靈人參賽，決定通靈王的對戰。參與戰鬥的通靈人至少必須要能使用超靈體。

;通靈王
:通靈王大戰的優勝者，是持有偉大靈魂(神)的通靈人，掌管森羅萬象的地球之王。根據本作的設定，兩千五百年前的[[釋迦摩尼|釋迦摩尼]]和兩千年前的[[耶穌基督|耶穌基督]]都是通靈王。

===帕契族===
;帕契族
:[[美洲|美洲]]的一支原住民族，遵從偉大靈魂的指示、舉辦通靈王大戰。具有和現代科學匹敵的傳統工藝技術。

===麻倉家===
;春雨
:阿彌陀丸生前愛用的刀，由喪助鍛造。曾斷過兩次。

;超‧占事略決
:記錄了一千年前由大[[陰陽師|陰陽師]]麻倉葉王所開發出來的巫術的書物。

;1080
:恐山安娜持有的靈珠，有1080顆珠子。

===其它===
;二段媒介
:以兩個媒介做成的超靈體，如叶将[[布都御魂|布都御魂]]之剑与附在春雨的阿彌陀丸合体的剑之精灵。

;巨大超靈體
:將超靈體巨大化。葉、道蓮、轟隆轟隆、龍、法斯特和巧克力愛情進入超占事略决時學到。（只在动画版出现）

;甲缚式超靈體
:巫力達到某種程度，包缚在身体的超灵体，融合攻擊和防禦功能於一體。在漫画版中叶等人於故事後期習得。

負責'''山谷'''地形的十祭司，也是負責瑞瑟格的祭司。持有靈是[[蜘蛛|蜘蛛]]。擅長[[空戰|空戰]]，但是其實是用[[蜘蛛網|蜘蛛網]]在空中移動，戰鬥時會在四周佈下蜘蛛網以偵測對手。擁有極強的精神力，即使連續遭受梅登的[[刑具|刑具]]攻擊也能繼續戰鬥，但是最後為瑞瑟格之火葬所敗。他和列尼姆也是負責催眠席巴的十祭司。

*'''列尼姆'''
負責'''[[雨林|雨林]]'''地形的十祭司，持有靈是[[變色龍|變色龍]]。他和布隆也是負責催眠席巴的十祭司，被小黑炭踢死（死得最窩囊的一個）。

*'''馬庫拿'''
負責火山地形的十祭司，持有靈是[[貓頭鷹|貓頭鷹]]·馬庫拿潛望鏡。是葉王的手下，藉著持有靈的能力能在黑暗中狙殺敵人。敗於蓮的電子BANG。

*'''尼克羅姆'''
: {{配音2|JP|[[伊藤健太郎|伊藤健太郎]]}}
負責洞穴地形的十祭司，前十祭司古洛姆之弟，持有靈是[[蠍子|蠍子]]，後來取得兄長之持有靈舞動之蟲，視蓮為殺親仇人，最後釋懷敗給蓮。

*'''塔利姆'''
負責海地形的十祭司，持有靈是綠色種子（植物之靈）。葉和葉王曾到他的咖啡店「豆」消費過。最後敗給巧克力愛情。

*'''拉瑟福'''
負責宇宙地形的十祭司，持有靈是[[外星人|外星人]]。他長的相當像外星人其實是因為他的甲縛式靈體「來訪者」。真面目是一個小[[蘿莉|蘿莉]]。葉王認為她的持有靈相當有趣。雖然沒有死，但被葉打敗。

=== 其他 ===
*'''沙堤'''（サチ）（配音：[[小松由佳_(声优)|小松由佳]]）
*:1963年12月31日生，血型A型。在漫畫中與葉王，以及X-Laws並列三大勢力的組織：「'''犍陀羅'''」的領袖，以拯救眾生為目標，暗中幫助葉等人，試圖阻止葉王成為通靈王，讓葉、蓮、瑞瑟格、轟隆轟隆、巧克力愛情到地獄修練。僅在漫畫版，以及[[PS2|PS2]]格鬥遊戲「通靈王－發奮丘溫泉」中出場，在作者的另一部作品《佛ZONE》（台譯：《[[來自天外|來自天外]]》）當中則以她的童年為故事主軸。

=== 五大精靈 ===

隱藏於'''G.S'''最深處，也是'''地獄'''社群，一般的靈魂在這裡很快就會變得不正常，而且有各種被稱為'''守護者'''的王所守護。（與金、木、水、火、土等[[五行|五行]]屬性不同。）

*'''靈魂之火'''
五百年前被葉王（轉生為帕契族時）奪走的五大精靈，在漫畫版最後，葉王將與精靈王結合，使精靈王成為他的持有靈，於是將它丟給葉他們，並說：『我不需要他了。』。代表為五人之一'''瑞瑟格·戴佐'''的所屬精靈。

*'''靈魂之雨'''
只在漫畫版中出現，被想將五大精靈賜予五位戰士的沙堤部下'''矢薙'''奪走。代表為五人之一'''轟隆轟隆'''的所屬精靈。

*'''靈魂之雷'''
只在漫畫版中出現，被想將五大精靈賜予五位戰士的沙堤部下'''寂尊'''奪走。代表為五人之一'''道蓮'''的所屬精靈。

*'''靈魂之風'''
只在漫畫版中出現，被想將五大精靈赐予五位戰士的沙堤部下'''卡度'''奪走。代表為五人之一'''巧克力愛情'''的所屬精靈。

*'''靈魂之土'''
只在漫畫版中出現，被想將五大精靈賜予五位戰士的'''沙堤'''親身挑戰閻羅大王後所得。代表為五人之一'''葉'''的所屬精靈。
