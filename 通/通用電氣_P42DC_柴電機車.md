通用電氣 P42DC (又稱''GENESIS Series I'')柴電機車，是[[通用電氣|通用電氣公司]]的P40DC柴電機車的後繼型式 <ref>P42 Builders Plate http://www.flickr.com/photos/76498245@N05/6871547365/in/photostream</ref> 。與其他P40DC系列柴電機車一樣，其尺寸較小，能行駛於美鐵所有的路線。引擎輸出為 {{convert|4250|hp|lk=in|sigfig=3}}（轉速為 1047 rpm），驅動客車集中供電發電機但無輸出電力時為 {{convert|3550|hp|sigfig=3}}（轉速為 900 rpm）。不過當客車集中供電發電機以800千瓦全力輸出時，本型機車的牽引力僅有 {{convert|2525|hp|sigfig=3}} 。

本型車最高時速為 {{convert|110|mph|abbr=on|sigfig=3}}，但在[[維亞鐵路|維亞鐵路]]路線最高時速限制為{{convert|100|mph|abbr=on|sigfig=3}}<ref>[http://www.viarail.ca/en/about-via-rail/our-fleet/locomotives-p42dc P42 - Via Rail]</ref> 。起動時牽引力為{{convert|280.25|kN|lbf|abbr=on|sigfig=3}} 。<ref>[https://web.archive.org/web/20020413053714/http://www.getransportation.com/general/apps/specsheets/Amtrak_Genesis_1/Genesis_1.htm P42DC spec sheet  via the Web Archive]</ref>

P42DC柴電機車主要用於美鐵公司東北走廊以外路線的長途高速列車，以及加拿大[[維亞鐵路|維亞鐵路]]公司的魁北克－溫莎線；於該線運行時速最高為 {{convert|160|km/h|mph|0|abbr=on}} 。
