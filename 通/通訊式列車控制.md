'''基于无线通信的列车控制技术'''（{{lang-en|'''CBTC'''，Communication-based Train Control}}），是一种{{le|鐵路信號系統|Railway signalling}}，利用[[電信|電信]]溝通列車和軌道設備以達到交通管理和設施控制。相較傳統的信號系統，現今的通訊式列車控制系統已能更準確追蹤列車位置。

依據[[電機電子工程師學會|電機電子工程師學會]]1474號標準的定義，'''通訊式列車控制'''（CBTC）系統是「連續、獨立於[[軌道電路|軌道電路]]、利用高解析列車位置判定的[[列車自動控制系統|列車自動控制系統]]；連續、大容量、雙向的列車對路端數據通訊；並且裝載於列車和路端的[[中央處理器|處理器]]有能力嵌入[[列車自動保護系統|列車自動保護]]（ATP）功能，和非必須的[[列車自動運行裝置|列車自動運行]]（ATO）與[[列車自動監督|列車自動監督]]（ATS）功能」。<ref name="IEEE1474">1474.1-1999 - IEEE Standard for Communications-Based Train Control (CBTC) Performance and Functional Requirements.[https://ieeexplore.ieee.org/document/815310] (January 14, 2019).</ref>


CBTC(Communication Based Train Control)系统是一个安全的，具有高可靠性、高稳定性的基于无线的列车自动控制系统，现较广泛的应用于城市轨道交通运输中。它最大的特点是可以无线通信，由列车-地面间周期传递列车位置信息和地面-列车间传递移动授权来实现功能。基于通信的列车控制系统（CBTC）包含两种类型 一种是基于感应环线的型CBTC， 一种是基于无线的CBTC。

== 由来 ==
无线电技术飞速发展后，人类开始孜孜不倦地研发一种基于无线通信的列车自动控制系统。实现移动闭塞，减少列车运行间隔，进而在不增加[[硬件|硬件]]投资的前提下，提高系统运能。研究人员期望这种系统能够减少[[铁路|铁路]]轨旁[[信号|信号]]线缆的铺设，并同时期望减少线缆的日常维护工作（实际引入轨旁通信设备，带来相应的轨旁维护工作量）以降低列车实行[[自动控制|自动控制]]的成本。

== 特性 ==
CBTC相比传统的铁路信号系统有着诸多特性，比如：
* 不须繁杂的[[电缆|电缆]]，转而以无线通信系统代替，减少电缆铺设及维护成本。
* 可以实现车辆与控制中心的双向通信，大幅度提高了列车区间通过能力。
* 信息传输流量大、效率高、速度快，容易实现移动自动闭塞系统。
* 容易适应各种车型、不同车速、不同运量、不同牵引方式的列车，兼容性强。
* 可以将信息分类传输，集中发送和集中处理，提高调度中心工作效率。

== 应用 ==
CBTC可以使用的双向无线通信系统种类很多，例如[[欧洲|欧洲]]使用的是GSM-R系统，美国使用扩频通信等其他多种无线通信系统，[[中国大陆|中国大陆]]使用无线自由波、[[波导管|波导管]]、[[漏波电缆|漏波电缆]]或三种互相组合的地车信息传输方式。目前应用CBTC系统的有[[美国|美国]]的[[纽约地铁|纽约地铁]]、[[台灣|台灣]]的[[台北捷運|台北捷運]][[文湖線|文湖線]]等，中国大陆也有部分城市轨道交通使用了CBTC系统，如[[武汉地铁|武汉地铁]][[武汉地铁1号线|1号线]]，[[上海轨道交通|上海轨道交通]]的[[上海轨道交通8号线|8号线]]，[[北京地铁|北京地铁]]（除[[北京地铁5号线|5号线]]、[[北京地铁13号线|13号线]]、[[北京地铁八通线|八通线]]）、[[广州地铁|广州地铁]]（除1、2、8号线）及[[成都地铁|成都地铁]]等。其中，[[北京地铁房山线|北京地铁房山线]]的顺利开通标志中國北京交控科技成为继[[德国|德国]][[西门子|西门子]]、[[法国|法国]][[阿尔斯通|阿尔斯通]]、[[加拿大|加拿大]][[庞巴迪|庞巴迪]]后第四个成功掌握CBTC核心技术并顺利开通应用实际工程的廠商，实现了全生命周期性价比最高的目标，比引进系统低20%左右<ref>[http://edu.sina.com.cn/l/2011-01-26/1447199310.shtml 《北京交通大学自主创新CBTC系统“十年磨一剑”》，新浪教育，2011年01月26日]</ref>。

== 功能 ==
通訊式列車控制可以含有不同层次的轨道自动化技术，包含以下几种主要功能：
* [[列车自动保护系统|列车自动保护]]（ATP，Automatic Train Protection）
* [[列车自动运行系统|列车自动运行]]（ATO，Automatic Train Operation）
* [[列車自動監督系統|列車自動監督]]（ATS，Automatic Train Supervision）
另外，个别厂商也将[[数据通信系統|数据通信系統]]（DCS，Data Communication System）独立作为子系统。
