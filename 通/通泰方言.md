'''通泰方言'''（[[南通市|通州]]-[[泰州|泰州]]方言），属于[[江淮官話|江淮官話]]。大致分佈在[[江蘇|江蘇]]省[[長江|長江]]以北，[[京杭運河|京杭運河]]以東的濱海區域。佔地面積15000余平方公里，使用人口1137万（2004年）。

== 内部分片 ==

按照[[南京大學|南京大學]][[顧黔|顧黔]]的意見，通泰方言可以劃為東、中、西三區。

'''東區'''（'''南通小片'''），包括[[南通|南通]]和[[通州|通州]]的西南部，區別于中區的特點是古[[假攝|假攝]]讀法同[[吳語|吳語]]，有兩類韻母-o,-a，流攝分裂為-ei,-ieu。

'''中區'''（'''[[如皋小片|如皋小片]]'''），包括[[如臯|如臯]]、[[如東|如東]]、[[泰興|泰興]]、[[扬中|扬中]]東部、[[東台|東台]]東部、[[海安|海安]]東部、[[靖江|靖江]]西南部，區別西區的特點是有[[疑母|疑母]]ng，能分辨[[鼻音_(辅音)|鼻音]]n和[[邊音|邊音]]l，流攝分裂為-ei,-ieu。

'''西區'''（'''泰州小片'''）包括[[泰州|泰州]]、[[姜堰|姜堰]]、海安西部、東台西部、[[大豐|大豐]]、[[興化|興化]]、[[江都|江都]]東部，通泰特徵最不明顯，特點是不分[[鼻音_(辅音)|鼻音]]n和[[邊音|邊音]]l，除“我”讀[[疑母|疑母]]ng外沒有[[聲母|聲母]]ng，流攝仍讀-eu。

== 語音特點 ==

通泰方言最大的特點是古[[全濁聲母|全濁聲母]]字今逢[[塞音|塞音]]、[[塞擦音|塞擦音]]不論平仄一律讀[[送氣清音|送氣清音]]，如“地”thi，“啤”phi。根據這一項特點，即可以把'''通泰方言'''和[[江淮方言|江淮方言]]（主要是[[淮东片|淮东片]]）以及[[吳方言|吳方言]]分別開。

通泰方言[[聲調|聲調]]有6~7個，相比于只有5聲調（[[陰平|陰平]]、[[陽平|陽平]]、[[陰上|陰上]]、[[陰去|陰去]]、[[陰入|陰入]]）的[[江淮方言|江淮方言]]，多出了[[陽入|陽入]]（和[[陽去|陽去]]）。古[[濁入|濁入]]字今歸[[陽入|陽入]]調，古全濁上、[[濁去|濁去]]字在7聲調的地區今讀[[陽去|陽去]]調，在6聲調的地區則整體歸入[[陰平|陰平]]。能區分陰陽去、陰陽入也是通泰方言區別于江淮方言的特徵。

通泰方言除東區外的各點，古[[假攝|假攝]]字的[[主元音|主元音]]只有一類，即a。而相鄰的江淮方言和吳語均已發生分化，吳語一般有兩類韻母即-o,-a，淮揚方言也有兩類即-a,-iI。

通泰方言多數點有豐富的[[支微入魚|支微入魚]]現象，例如“推”[[白讀|白讀]]thy，“雷”[[白讀|白讀]]ly。這種現象不見于江淮方言，吳語雖有，但數量遠不及通泰方言之多。

通泰方言的[[咸山攝|咸山攝]]韻母分化為三類，分法和吳語、贛語相同，異于西部的江淮方言。粗略的說，即[[咸攝|咸攝]]談韻的[[牙音|牙音]]、[[喉音|喉音]]字韻母和覃韻合為一類，而与談韻的[[舌音|舌音]]、[[齒音|齒音]]字有別。同樣，[[山攝|山攝]]寒韻的[[牙音|牙音]]、[[喉音|喉音]]字韻母和桓韻合為一類，而与寒韻的[[舌音|舌音]]、[[齒音|齒音]]字有別。

在聲調上，通泰方言[[調值|調值]]陰高陽低的模式不復存在。[[陰平|陰平]]普遍低於[[陽平|陽平]]，除[[如東|如東]]以外各點，[[陰入|陰入]]皆低於[[陽入|陽入]]。但是，陽入後接鼻音、邊音、濁擦音、零聲母字時，調值為21，低於陰入，例如：毒藥、臘月、熟人、十五、十一、逆毛。

== 詞彙句法特點 ==

通泰方言[[詞彙|詞彙]]上的過渡色彩較濃。南方諸方言的“落雨”、“落雪”，泰如方言中、東區仍說“落雨”、“落雪”，西區則“落雨＊下雨”、“落雪＊下雪”兩可，而北方話和淮揚方言只説“下雨”、“下雪”。類似于此的還有“尋＊找”。

考訂通泰方言的[[本字|本字]]，發現与南方相同者甚多，[[基本詞彙|基本詞彙]]的南方成分也相當多。例如，淮揚方言的“我們”“你們”“他們”，通泰方言說“我倈”“你倈”“他倈”；淮揚方言的“沒niau”，通泰方言說“不曾”；淮揚方言的“今個”“明個”，通泰方言說“今朝”“明朝”，等等。

[[句法|句法]]上，有3種[[句式|句式]]值得注意：
# [[反復疑問句|反復疑問句]]，通泰方言以“格VP”句式爲主，“VP不VP”句式較少見。淮揚方言大致兩者皆可，北方話一般只有后一種。
# [[雙賓語句|雙賓語句]]，通泰方言[[直接賓語|直接賓語]]和[[間接賓語|間接賓語]]的位置接近南方諸方言。例如北京話的“給我一本書”，泰如方言、吳語等說“把本書我”。
# [[比較句|比較句]]，通泰方言存在這樣一種句式，即[[比較短語|比較短語]]位于中心形容詞后。例如“你高似他”，北方話一般説“你比他高”。

== 發展現狀 ==

通泰方言區甚爲狹小。1949年后國家大力“[[推普|推普]]”，使得通泰方言以向作爲“民族[[共同語|共同語]]”的[[普通話|普通話]]靠攏。

具體來説，就是一批字產生近似江淮方言讀法的[[文讀音|文讀音]]，在和原有的[[白讀音|白讀音]]長期共存、抗衡中地位從弱勢逐漸變為優勢，直到最後將原有[[白讀音|白讀音]]消滅。例如，[[全濁聲母|全濁聲母]]字逢[[塞音|塞音]]、[[塞擦音|塞擦音]]，[[仄聲|仄聲]]讀[[送氣清音|送氣清音]]是通泰方言區別于江淮方言其他分区的最大特點。而在[[文讀音|文讀音]]中，仄聲就讀[[不送氣清音|不送氣清音]]，和江淮方言其他分区以及整個北方話一致。大致來説，通泰方言西區產生的文讀較多，東區較少。
