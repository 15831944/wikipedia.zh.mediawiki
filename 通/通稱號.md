'''通稱號'''（[[日文|日文]]：'''通称号'''、{{lang|ja|つうしょうごう}}、'''符'''）為[[大日本帝國陸軍|舊日本陸軍]]於[[太平洋戰争|太平洋戰争]]期間部隊名稱使用的一種代號。

== 概說 ==
'''通稱號'''為賦予[[師團|師團]]、[[旅|獨立混成旅團]]以上擁有獨立作戰能力的部隊一或兩字的漢字作為代號，其隷下部隊則以數字區別。此處漢字稱為'''兵團文字符'''，個別的數字則為'''通稱番號'''。以上兩者合稱為'''通稱號'''，為根據《陸軍部隊戰時通稱號規定（陸機密第143號）》所訂定的。另外還有'''秘匿名'''、'''秘匿號'''、'''通稱符'''等說法。和[[大日本帝國陸軍|舊日本陸軍]]編制不同的[[大日本帝國海軍|舊日本海軍]]則沒有使用通稱號。

== 制定歷史 ==
平時於本土擁有的常設師團，因為司令部與駐紮地的位置都已確定，因此沒有使用代號的必要，只有戰時才開始編制的部隊為了避免洩露情資而隱藏正式名稱。至[[九一八事變|九一八事變]]為止[[大日本帝國陸軍|舊日本陸軍]]並沒有特別隱藏部隊名稱，直到1937年9月1日''《動員部隊の称呼名に関する件》''（關於動員部隊稱呼名）發布後，才開始隱藏外地部隊的名稱。一開始是以部隊指揮官的姓氏取名為"木多部隊"、"赤松部隊"之類，但隨著戰局逐漸激烈，指揮官的異動與戰死也越來越常見。

1940年9月10日《昭和16年度陸軍動員計畫令細則》便規定賦予[[師團|師團]]、[[旅|獨立混成旅團]]以上擁有獨立作戰能力的部隊一或兩字的漢字作為'''兵團文字符'''，其隷下部隊則擁有3到5碼的數字作為'''通稱番號'''。

=== 例 ===
1938年以維持中國戰線治安而編成的三單位制師團[[第23師團|第23師團]]於[[諾門罕戰役|諾門罕戰役]]時的秘匿名為指揮官[[小松原道太郎|小松原道太郎]][[中將|中將]]的姓而來的'''小松原兵團'''，而[[太平洋戰爭|太平洋戰爭]]時則稱為'''旭兵團'''。

1939年10月在[[滿洲國|滿洲]]編成的[[第24師團|第24師團]]，1941年開始的兵團文字符為'''山'''，其隷下[[步兵第22聯隊|步兵第22聯隊]]則擁有'''3474'''這個通稱番號。
也就是說，[[第24師團|第24師團]]以通稱號表示為'''山兵團'''，而[[步兵第22聯隊|步兵第22聯隊]]則為'''山第3474部隊'''。（請參考表格）

== 兵團文字符與兵團 ==
此通稱號不只是日本本土或外地，就算部隊以外的諸機構也擁有。[[太平洋戰爭|太平洋戰爭]]末期隨著兵團數量激增，一個字的代號不夠用的關係也有增加至兩個字的狀況，比如1945年編成的[[第134師團|第134師團]]為'''真心'''、[[第138師團|第138師團]]為'''不動'''。

本來通稱號只是為了隱藏部隊正式的名稱，兵團文字符中時常以常設師團為中心用編成地與兵種類推而成，也因此成為另一種的暱稱。[[皇居|宮城]]警備任務的[[近衛師團|近衛師團]]為'''宮'''，[[東京|東京]]的[[第1師團|第1師團]]為'''玉'''，[[大阪|大阪]]的[[第4師團|第4師團]]為'''淀'''，[[廣島|廣島]]的[[第5師團|第5師團]]為'''鯉'''，[[北海道|北海道]]的[[第7師團|第7師團]]為'''熊'''，[[戰車第4師團|戰車第4師團]]為'''鋼'''，[[第3飛行師團|第3飛行師團]]為'''隼'''等等，從中可以見到其關聯性。

制定通稱號的規則為根據1945年4月20日制定的《陸軍部隊戰時通稱號規定（陸機密第143號）》更改。

== 日本本土 ==
為了在新聞報導等等地方防諜，[[軍|軍]]以上稱為'''集團'''，[[師團|師團]]、[[旅團|旅團]]為'''兵團'''，[[聯隊|聯隊]]、[[大隊|大隊]]為'''部隊'''、[[中隊|中隊]]、[[小隊|小隊]]為'''隊'''。士兵的家人對前線通信、電報的狀況也是根據通稱號，終戰後搜尋未歸還兵的其中一個方式，便是其家族根據通稱號向[[厚生省|厚生省]]（現[[厚生勞動省|厚生勞動省]]）查詢其原部隊。
