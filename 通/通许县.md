'''通许县'''是[[中华人民共和国|中华人民共和国]][[河南省|河南省]][[开封市|开封市]]下属的一个县, 位于[[中原经济区|中原经济区]]建设的核心区之内。目前下辖6个镇、6个乡<ref name="gov">{{cite web |url=http://www.kaifeng.gov.cn/zk_xxgk/gkzn.jsp?id=1227073184097 |title=存档副本 |accessdate=2012-09-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20131110224141/http://www.kaifeng.gov.cn/zk_xxgk/gkzn.jsp?id=1227073184097 |archivedate=2013-11-10 }}</ref>：[[城关镇|城关镇]]、[[朱砂镇|朱砂镇]]、[[玉皇庙镇|玉皇庙镇]]、[[四所楼镇|四所楼镇]]、[[长智镇|长智镇]]和[[竖岗镇|竖岗镇]]，[[邸阁乡|邸阁乡]]、[[练城乡|练城乡]]、[[冯庄乡|冯庄乡]]、[[厉庄乡|厉庄乡]]、[[大岗李乡|大岗李乡]]和[[孙营乡|孙营乡]]。全县共有309个[[行政村|行政村]]，611个[[自然村|自然村]]。

截至[[2011|2011]]年全县总人口为60万人<ref name="gov"/>。民族[[汉族|汉族]]居多，占99.2%，[[回族|回族]]和其他少数民族占0.8%。县人民政府驻[[城关镇|城关镇]]。2011年，全县[[GDP|GDP]]为142.7亿[[人民币|人民币]]<ref>{{cite web |url=http://www.kaifeng.gov.cn/html/5e0aed881df6354d011e00121b520012/2012040111103116.html |title=存档副本 |accessdate=2012-09-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20120521141743/http://www.kaifeng.gov.cn/html/5e0aed881df6354d011e00121b520012/2012040111103116.html |archivedate=2012-05-21 }}</ref> 。

== 历史沿革 ==

通许县[[春秋|春秋]]时期为[[许国|许国]]地，[[战国|战国]]时期属[[魏国|魏国]][[大梁|大梁]]郡。[[北宋|北宋]][[建隆|建隆]]元年（960年）始置通许镇，属[[扶沟县|扶沟县]]。[[宋太祖|宋太祖]]赵匡胤下诏疏浚[[蔡河|蔡河]]。自[[京师|京师]]至通许镇，沿河设置闸门，按时开闸，调节水量，[[漕运|漕运]]畅通。取自[[汴京|汴京]]直通许国故地之义，故名通许。[[咸平|咸平]]五年（1002年）置[[咸平县|咸平县]]，以年号为名。[[金|金]][[大定|大定]]二十九年（1189年），因其与[[咸平府|咸平府]]（今[[辽宁省|辽宁省]][[开原县|开原县]]）同名，易名为通许县，属[[南京路|南京路]][[开封府|开封府]]。[[元|元]]属[[河南江北中书省|河南江北中书省]][[汴梁路|汴梁路]]总管府。[[明|明]][[清|清]]属[[河南省|河南省]][[开封府|开封府]]。[[中华民国|中华民国]]初属[[豫东道|豫东道]]；[[废道|废道]]后，属[[河南省第一行政督察专员公署|河南省第一行政督察专员公署]]。

[[中华人民共和国|中华人民共和国]]建立后，属[[陈留专区|陈留专区]]；1952年归[[郑州专区|郑州专区]]，1954年改属[[开封专区|开封专区]]；1958年9月，通许县并入[[尉氏县|尉氏县]]，改为通许镇；1962年3月，恢复通许县建置，仍属[[开封专区|开封专区]]；1983年划归[[开封市|开封市]]迄今。

== 自然地理 ==

通许县地处[[豫东平原|豫东平原]]，全境介于[[北纬|北纬]]34°15'-34°34'，[[东经|东经]]114°18'-114°38'之间，县境东西宽27.7[[公里|公里]]，南北长34.8公里，总面积767平方公里<ref name="gov"/>。东距[[黄海|黄海]]450公里，临[[商丘市|商丘市]]；西望[[嵩山|嵩山]][[少林寺|少林寺]]，距[[省会|省会]][[郑州市|郑州市]]100公里；南距[[周口市|周口市]]90公里，北距七朝古都[[开封市|开封市]]45公里。

通许县境历受[[黄河|黄河]]泛滥冲积的影响，成为自[[孟津|孟津]]向东黄河南泛冲积而成的[[黄淮平原|黄淮平原]]的一部分。地势西高东低，北高南低，由西北向东南微倾斜。地面[[海拔|海拔]]在57.6至66.5米之间，相对高差在10米左右，地面[[坡度|坡度]]较小，一般为1/1000至1/5000。

通许县[[地质构造|地质构造]]属[[中朝准地台|中朝准地台]][[华北坳陷|华北坳陷]]南部的[[通许隆起|通许隆起]]，处于[[中国|中国]]巨型[[秦岭－昆仑纬向构造体系|秦岭－昆仑纬向构造体系]]与[[新华夏第二沉降带|新华夏第二沉降带]]、[[华北坳陷|华北坳陷]]复合交接部位。[[沉积层|沉积层]]为厚1000至5000米的[[湖相红色岩系|湖相红色岩系]]（第三系）和河流冲积、洪积相（第四系）地层。[[地质构造|地质构造]]形迹大多隐伏在巨厚的[[沉积层|沉积层]]下，地表形迹不明显，地质构造较为单一。

通许县属[[暖温带大陆性季风|暖温带大陆性季风]]型[[气候|气候]]，四季分明，光照充足，气候温和，雨量较少。年均[[气温|气温]]在14.24-14.50度之间，[[无霜期|无霜期]]为213-215天，年均日照为2500个时数，年均[[降雨量|降雨量]]775[[毫米|毫米]]。

== 交通 ==

通许县区域位置优越，西距[[新郑国际机场|新郑国际机场]]、[[郑州航空港经济综合实验区|郑州航空港经济综合实验区]]（[[郑州新郑综合保税区|郑州新郑综合保税区]]）60公里，北距[[中原经济区|中原经济区]]之核心区[[郑汴新区|郑汴新区]]45公里；[[新密|新密]]至[[商丘|商丘]]铁路穿境而过，长达46公里，连接[[京广|京广]]、[[京九|京九]]铁路；规划中衡潢铁路途径地；境内高速公路纵横交错，遍布全县，[[兰南高速|兰南高速]]（豫高速S83）和[[大广高速|大广高速]]交汇于县城东北3公里处，[[郑民高速|郑民高速]]（豫高速S82）与建设中的商丘至登封高速公路穿境而过，距[[新郑国际机场|新郑国际机场]]仅有半小时车程，3至5小时能到达周边各[[省会|省会]]城市；境内另有S218、S325、S326等[[省道|省道]]3条，县域内全面实现了村村通[[柏油|柏油]]公路，全县公路通车总[[里程|里程]]达1423公里<ref>[http://roll.sohu.com/20111020/n322755656.shtml</ref>，形成了以[[高速公路|高速公路]]、[[铁路|铁路]]、[[省道|省道]]为框架，以县乡村道路为脉络的现代交通网络体系。
