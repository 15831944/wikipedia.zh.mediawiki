'''通勤鐵路'''、或称'''城市铁路'''、'''市域铁路'''、'''市郊铁路'''等，在中國對於地鐵制式的大站距線路则又稱為「'''[[快軌|快軌]]'''」（{{lang-en|Rapid Transit}}），是一種主要提供[[城市|城市]][[市中心|市中心]]與[[郊區|郊區]]之間往來[[通勤|通勤]]的[[鐵路運輸|鐵路運輸]]服務。

20世紀末以來，很多地方的城市都著重發展通勤铁路，一方面希望減輕市區內的路面交通壓力，以及減少[[污染|污染]]，及對[[汽車|汽車]]和[[石油|石油]]的依賴；另一方面亦希望通勤铁路能帶動途經地區發展成為都會的[[住宅區|住宅區]]。<ref name="建構臺灣都會區域整合性通勤鐵路之芻議">{{cite web| url=http://www.cepd.gov.tw/m1.aspx?sNo=0013032| title=建構臺灣都會區域整合性通勤鐵路之芻議| accessdate=2012-09-30}}{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref><ref name="一小时都市服务圈 北京拟开通1000公里市郊铁路">{{cite web | url=http://www.chinanews.com/gn/2011/10-15/3390717.shtml | title=一小时都市服务圈 北京拟开通1000公里市郊铁路 | accessdate=2012-09-30}}</ref><ref name="建设通勤铁路网,构筑上海交通圈">{{cite web | url=http://d.wanfangdata.com.cn/Periodical_shcsgh200801012.aspx | title=建设通勤铁路网,构筑上海交通圈 | accessdate=2012-09-30}}</ref>

== 定義 ==
===通勤铁路（市郊铁路）===
'''通勤铁路'''（{{lang-en|Commuter rail}}），或称'''市郊铁路'''（{{lang-de|S-Bahn}}），主要服務需要每日通勤的[[學生|學生]]與[[上班族|上班族]]。在全球大部分的[[城市群|城市群]]，每天都有大量的[[乘客|乘客]]使用通勤铁路上下班和上下學。<ref name="建構臺灣都會區域整合性通勤鐵路之芻議"/><ref name="一小时都市服务圈 北京拟开通1000公里市郊铁路"/>

對於通勤動車組的定義，不同地區均有不同的定義。例如在英國，即使不是使用橫向座椅的列車，也可被看作通勤動車組。不過如以日本的標準來看，大部分西方國家的「通勤動車組」則只可被看成近郊型列車。但以日本標準來看，香港[[東鐵綫|東鐵綫]]列車則可被看成「近通混雜」型，有時亦皆可被看成近郊或通勤型。情況有點像[[JR東日本|JR東日本]][[JR東日本E217系電聯車|E217系]]及[[JR西日本|JR西日本]][[JR西日本207系電車|207系]]。不過 E217 系為近郊型，而 207 系則機件配備和列車性能介乎於近郊及通勤，但擁有通勤式車體設計而被定為通勤型。後來，自[[JR東日本E231系電車|E231系]]的出現，鑑於其分成通勤和近郊型的特點，後來JR也出現了一種新的電車種類：[[一般型電車|一般型電車]]。而且，有些地區的通勤帶非常廣闊，就連近郊鐵路都成了不少人的通勤交通（例東京為中心的[[湘南新宿線|湘南新宿線]]，京阪神為中心的[[JR京都線|JR京都]]、[[JR神戶線|神戶線]]）。所以，現時鐵路界仍未有一套公認且標準的定義。

===与其他类型轨道交通的区别===

*'''[[地铁|地铁]]'''和'''[[城市軌道交通|城市軌道交通]]'''

地铁一般是服务单一市区、服务范围由市中心向外不超过20公里半径，列车班次频繁，有高架或地下之独立轨道。通勤列车往往与干线铁路在轨道、科技和适用法律上都相同或相近。但两者的区分并非绝对，例如[[利物浦|利物浦]]的[[默西塞德铁路|默西塞德铁路]]拥有独立的地下及地面轨道，担负的功能与城市地铁已无二异，但法律上仍属于[[英国全国铁路|英国全国铁路]]网，但在某一城市同时拥有地铁及通勤铁路的情况下才相对容易，如纽约的[[纽约地铁|纽约地铁]]和[[长岛铁路|长岛铁路]]，或者巴黎的[[巴黎地铁|巴黎地铁]]及[[RER|RER]]。

*'''[[S-Bahn|S-Bahn]]'''

在德国，S-Bahn是许多地方都有的列车服务，其中主要以两种形式出现。
#在[[柏林|柏林]]及[[汉堡|汉堡]]地区S-Bahn的服务接近于地铁水平，虽然当地已有{{tsl|de|U-Bahn}}（德语“地铁”的意思）。这些列车拥有专用轨道，车站大多都在人口高密度地区，站间距较短，全天都有高密度的班次，也基本没有平交路口。在柏林和汉堡，S-Bahn和U-Bahn最大的区别是火车站里面S-Bahn的月台是跟其他火车平行的。类似的系统在哥本哈根和维也纳也有。在汉堡和哥本哈根，也有柴油驱动的列车沿着S-Bahn线路继续向更远的地方运行。在柏林，主城以外的高密度地区也有以S-Bahn运行的环线。
#另外一种在[[鲁尔地区|鲁尔地区]]常见的S-Bahn，是没有独立轨道且线路更长的服务，列车也会在城际间运行，其运行方式更接近于[[区域铁路|区域铁路]]。该地区很多城市的建成区已经连绵在了一起。在慕尼黑和法兰克福也是如此。

总而言之，S-Bahn的概念已经超过了普通的通勤铁路的范畴，需要根据情况来区别对待。

*'''[[区域铁路|区域铁路]]'''

区域铁路一般运行在同一区域的中小型城镇之间，而并非如城际列车那样在各区域主要城市之间运行。区域铁路的停站更加频密，旨在为本地社区提供服务，为较长距离的列车喂给客流。类似的服务有如英国铁路的Regional Railways，法国的[[法国省际列车|法国省际列车]]，德国的[[德铁区域运输|德铁区域运输]]，以及韩国的[[統一號|統一號]]。

美国的铁路客运并不发达，所以缺少此类服务，在使用该词时往往将之与通勤铁路混用。

*'''[[城际铁路|城际铁路]]'''
城际铁路指那些距离较远的客运铁路服务，跨越各区域链接主要城市。停靠时往往只停靠大站。车资较其他类型的铁路客运服务更贵。城际铁路列車以座位為主，並設有[[行李|行李]]擺放的設施，部分車廂還有洗手間、餐車等。

*'''[[高速铁路|高速铁路]]'''
随着高速铁路的诞生，长途运行的时间缩短了，使得部分人士得以从更远的距离乘坐高速列车进行通勤，这在日本的[[新干线|新干线]]沿线比较常见。这些线路属不属于通勤铁路还有争议，但因为高速铁路往往是为长途乘坐而设计，与一般意义上的通勤铁路存在差异。在日本、德国、瑞士等地，能实现高速铁路通勤的一个要素在于主要车站紧邻[[中心商务区|中心商务区]]。

== 特點 ==
[[File:Connex_Train_Melbourne.jpg|thumb]]
大部分的通勤铁路都是[[重型鐵路系統|重鐵]]，使用跟[[国铁|国铁]]相同或相近的标准<ref name=cstl>{{cite book|author=孙章 何宗华 徐金祥|title=城市轨道交通概论|publisher=中国铁道出版社|year=2000|isbn=9787113038311}}</ref>。

跟[[輕軌運輸|輕鐵]]或[[地铁|地铁]]相比，通勤铁路主要分別是：<ref name="建構臺灣都會區域整合性通勤鐵路之芻議2">{{cite web| url=http://www.cepd.gov.tw/dn.aspx?uid=7916| title=建構臺灣都會區域整合性通勤鐵路之芻議| accessdate=2013-02-13| deadurl=yes| archiveurl=https://web.archive.org/web/20140108060714/http://www.cepd.gov.tw/dn.aspx?uid=7916| archivedate=2014-01-08}}</ref>

* 使用的車輛體積較大
* 多數班次較稀疏
* 有特定的抵發時間表（列車在指定的時刻開車，而不是每隔一段時間開出）
* 服務地區的人口密度一般較低，通常是由市郊到市中心
* 跟城市間的市際列車，或貨車使用同一[[鐵軌|路軌]]

因為通勤铁路可以使用原有的市際列車或貨運鐵路的路軌，建造成本得以大為降低。但亦有部分通勤铁路為避免與其它列車爭路而造成誤點，而特意建造獨立的路軌。

通勤铁路以城市公共交通客流为服务对象，短途、通勤是其最大特点，一般不承担货物运输。按照服务范围的大小，可将其分为两个部分：[[市郊铁路|市郊铁路]]和[[城市快速铁路|城市快速铁路]]。而市郊铁路是通勤铁路的主要形式<ref name=cstl/>。

通勤铁路通常採用當地的既有铁路线的[[軌距|軌距]]。例如在英、美為 1435 毫米（4 呎 8½ 吋）[[標準軌距|標準軌距]]，在[[日本|日本]]、[[臺灣|臺灣]]及[[瑞士|瑞士]]為 1067 毫米[[窄軌|窄軌]]。

在某些城市，通勤铁路跟一般城市軌道交通系統（或地鐵）之間的分別變得模糊。通勤铁路列車到了市中心會在高架或地下的路軌上行駛，到了市郊路段則回降至地面。例子有[[西班牙|西班牙]][[馬德里|馬德里]]的Cercanías，[[法國|法國]][[巴黎|巴黎]]的[[區域快鐵|RER]]，[[澳洲|澳洲]][[悉尼|悉尼]]及[[墨爾本|墨爾本]]的近郊鐵路。[[香港|香港]]的[[港鐵|港鐵]][[東鐵綫|東鐵綫]]提供的服務班次及密度接近一般城市軌道交通系統，而車輛亦以站立為主，但它仍然與[[城際鐵路|城際鐵路]]使用同一路軌。日本[[東京|東京]]的[[JR山手線|JR山手線]]列車則運行於環狀的獨立路軌，營運路線站距與列車班距同樣接近於城市軌道交通系統，並且具有和其他長途鐵路線相互接駁的功能。[[中國|中國]][[北京|北京]]的[[北京地铁13号线|地鐵十三號線]]是由郊區中一些較少使用的市際及貨運路軌改裝而成，但營運模式跟城市軌道交通系統差不多。

某些[[歐洲|歐洲]]國家由於城市間的距離短，使到市際列車服務及通勤铁路的分野亦變得模糊。例如[[荷蘭|荷蘭]]及[[比利時|比利時]]的市際列車運載有很多通勤上班人士，它們使用的車輛、行車距離及班次亦接近其它國家的通勤铁路。

在[[北美洲|北美洲]]，通勤铁路鐵路通常是由政府或半政府機構營運。

== 列車 ==
{{main|通勤型電力動車組}}
通勤铁路的列車多數都會設法增加載客量，但因為行走距離有時會比[[地鐵|地下鐵路]]等長，所以一般都不會令乘客感覺太過擠迫。大部份的通勤铁路列車都不會有餐車等屬於長途車的裝備。通勤铁路一般的行走距離為15至180[[公里|公里]]，運行速度由每小時55至175公里不等。[[鐵路車輛#客車|車廂]]可能是單層或雙層，常見的單層車廂的載客量約80至110人，雙層的約145至170人。

=== 車輛 ===
由於通勤铁路站距較短，通常使用加減速性能優秀的[[柴聯車|柴油動車組]]或[[電聯車|電力動車組]]以縮短行車時間。有些地方亦會使用[[鐵路機車|機車]]牽引[[鐵路客車|車廂]]行走。使用機車牽引時，列車大多使用[[推拉操作|推拉操作]]，往一方向行駛時由機車在前拉；往另一方向則由機車在後推，由司機在另一端附有駕駛室的客車廂遙控機車。日本則以[[動車組|動車組]]為主流。

=== 马来西亚 ===
马来西亚共有4条通勤铁路，分为北中两区。其中中马区共有两条，分别为{{rint|my|1}}{{KLRT line|1}}和{{rint|my|2}}{{KLRT line|2}}。两条线路也被纳入吉隆坡首都圈（[[巴生谷|巴生谷]]）的[[巴生谷综合运输系统|综合运输系统]]，是[[吉隆坡|吉隆坡]][[轨道交通|轨道交通]]的一部分。{{rint|my|10}}{{KLRT line|10}}则是一条建造中的“机场”通勤铁路，于[[梳邦机场|梳邦机场]]终止。

而[[KTM通勤铁路北马区|KTM通勤铁路北马区]]的路线则是{{rint|my|pbl}}巴东勿刹线和{{rint|my|prl}}硝山线。
