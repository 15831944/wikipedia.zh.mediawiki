'''通州区'''是[[中华人民共和国|中国]][[江苏省|江苏省]][[南通市|南通市]]下辖的一个[[市辖区|市辖区]]。位于江苏省东南部，[[长江|长江]]入海口，东临[[黄海|黄海]]，南依[[长江|长江]]，因[[通州区_(北京市)|北通州]]今日改称「南通」，紀曉嵐乃戲謂「南通州，北通州，南北通州通南北」。境內有水陸碼頭，空運機場。通州是晩清[[状元|状元]][[張謇|張謇]]的故鄉。

== 历史 ==
通州最初为[[长江|长江]]口海域中几块相邻的沙洲，[[南北朝|南北朝]]中期露出水面，至[[唐朝|唐朝]]末年与大陆接壤。[[五代|五代]]时称静海镇，[[后周|后周]]显德五年（公元958年）设立通州，属[[扬州|扬州]]管辖，下辖静海县和海门县。[[宋朝|宋朝]]天圣元年（公元1023年），改名为崇州，又称崇川。明道二年（公元1033年）又仍改称通州。[[宋徽宗|宋徽宗]]政和七年（公元1117年）改设静海郡，不久又改名为通州。[[元朝|元朝]]至元十五年（公元1278年）升格为通州路，至元二十一年仍称通州，归扬州路管辖。[[明朝|明朝]]洪武元年（公元1368年）废除静海县，通州直接管辖静海以及海门县和崇明县，属扬州府。洪武九年，[[崇明|崇明]]县划归[[苏州|苏州]]府。[[清朝|清朝]]顺治二年（公元1645年）属江南省扬州府。[[康熙|康熙]]六年（公元1667年）起属江苏省扬州府。康熙十一年（公元1672年）因海门江岸坍塌入长江，改海门县为乡，并入通州。

[[中华民国|中华民国]]元年（1912年），由于要与直隶的[[通州区_(北京市)|通州]]（今属北京）区分，故因位于南方而改名为南通县，属江苏省。

1949年2月2日，解放军占领南通县，城区及港闸等地设置南通市，南通县政府移驻金沙镇，属南通专署管辖。1983年实行市管县体制，南通县属[[南通|南通]]市管辖。

1993年2月，南通县撤县改市，称通州市。

2009年3月31日经[[中国国务院|中国国务院]]批准，撤销通州市，设立南通市通州区<ref>[http://www.jiangsu.gov.cn/tmzf/szfxxgk/szfxxgkml/sgfxwj/mzyldbz/200904/P020090421344757110478.doc 省政府关于撤销通州市设立南通市通州区的通知]，江苏省政府网站，2009年4月22日造访。</ref>。

== 行政区划 ==
2002年，通州市辖23个建制镇，3个街道办事处，376个村，52个社区，5个农场，总面积1351平方公里，总人口128.13万人。

2007年2月16日，通州市调整部分镇行政区划<ref>南通市政府《关于调整通州市部分镇行政区划的通知》（通政发[2007]16号）</ref>，北兴桥镇、海晏镇并入三余镇，新联镇并入刘桥镇。此后，通州市共辖19个镇：金沙镇、西亭镇、二甲镇、东社镇、三余镇、十总镇、骑岸镇、五甲镇、石港镇、四安镇、刘桥镇、平潮镇、平东镇、五接镇、兴仁镇、兴东镇、张芝山镇、川姜镇、先锋镇。

2009年4月17日，国务院作出《关于同意江苏省调整南通市部分行政区划的批复》，江苏省人民政府下发了《关于撤销通州市设立南通市通州区的通知》。经国务院批准，撤销通州市，设立南通市通州区，以原通州市的行政区域为通州区的行政区域，区人民政府驻金沙镇。

2015年3月24日，根据江苏省人民政府《省政府关于调整南通市通州区部分行政区划的批复》（苏政复[2015]22号），通州区共辖4街道：金沙街道，金新街道，兴东街道，先锋街道；12个镇：平潮镇，五接镇，刘桥镇，石港镇，兴仁镇，张芝山镇，川姜镇，西亭镇，十总镇，东社镇，二甲镇，三余镇。

== 地理 ==
通州区地处长江三角洲，土地肥沃，高程一般在3.8～4.5米，近海最低处为2.2米。海岸长16.27公里，江岸长15公里。

通州地处北亚热带，寒暑变化显著，气候温和湿润，日照充裕，雨水丰沛。

== 经济 ==
2004年，全市完成地区生产总值200.4亿元，比上年增长15.8％，人均地区生产总值达到15656元。在全国百强县市社会经济综合发展指数测评中排名73位。2005年居全国百强县第34位。

通州区是中国出口商品综合生产基地之一，是优质粮棉油、多种蔬菜和蚕茧、[[薄荷|薄荷]]、[[灵芝|灵芝]]等特种经济作物的重要产地。56万多亩江海滩涂是盛产[[干贝|干贝]]、[[鲥鱼|鲥鱼]]、[[紫菜|紫菜]]等特种养殖业，也是全国唯一的泥蚶苗种基地。

通州已形成纺织、服装、机械、冶金、电子、建材、医药、食品、饮料等门类的工业体系，是全国纺织工业的重要基地之一。

== 方言 ==
通州地区方言比较复杂，西部地区主要使用[[南通话|南通话]]，属于[[通泰方言|通泰方言]][[南通小片|南通小片]]。东部使用三种吴语，第一种是[[通东话|通东话]]亦可称古常州语，属于[[吴语|吴语]][[太湖片|太湖片]][[毗陵小片|毗陵小片]]，分布在金沙以东地区，和启东海门的通东话连成一片。第二种是[[启海话|启海话]]，属于[[吴语|吴语]][[太湖片|太湖片]][[苏沪嘉小片|苏沪嘉小片]]，分布在通东话的南北两侧，分别和[[如东|如东]]东部的启海方言区、[[启东|启东]][[海门|海门]]的启海方言区连成一片。第三种是[[金沙话|金沙话]]，其归属在语言学上尚有争议，目前暂时归类为[[吴语|吴语]]，分布在金沙及其周边乡镇。
