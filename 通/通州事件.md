'''通州事件'''，又称'''通州大屠殺'''，日本侵华战争時期由日本扶植而成立并且受日本控制的[[冀東防共自治政府|冀東防共自治政府]]下属通州保安隊的中國人士兵于1937年7月29日轉向攻擊冀東防共自治政府之時进行的屠杀。

1937年[[卢沟桥事变|卢沟桥事变]]爆发后，华北中国军队和[[日军|日军]]发生全面交火。7月29日在[[北平|北平]]附近，駐守[[通州_(北京)|通州]]（今[[通州区_(北京市)|通州区]]）的通州保安隊叛變，對該地的日軍守備隊和特務機關發動攻擊，通州保安隊搗毀了日軍機關，俘虜了[[殷汝耕|殷汝耕]]（通州保安隊撤退时[[殷汝耕|殷汝耕]]逃走）。在此次事件，日方称通州保安隊更對居留當地的日本僑民進行襲擊，百餘名日本僑民（据称多數是老弱婦孺）被搶劫、[[強姦|強姦]]、凌辱和殺戮。根据[[远东国际军事法庭|远东国际军事法庭]]的日方證言，通州保安隊的中國人士兵殺死約235個日本官兵、僑民、顧問和日韓浪人。

== 背景 ==
1937年[[卢沟桥事变|卢沟桥事变]]爆发后，华北中国军队和日军发生全面交火。1937年7月27日，日军向驻通州附近的中国部队二十九军發出通牒並進行轟炸。駐紮在通州旧城南门外的[[通州|通州]]保安队未执行日军的部署，按兵不动。日军[[轟炸機|轟炸機]]因分不清楚[[冀察|冀察]]和[[冀東|冀東]]的分界線，誤炸了靠近寶通寺軍營的冀東保安隊幹部訓練所，導致發生數名保安隊員死亡的事件。接到誤炸的報告後，細木特務機關長直接前往[[冀東防共自治政府|冀東防共自治政府]]向[[殷汝耕|殷汝耕]]長官致歉，並親自前往現場視察，慰問死者家屬。次日28日，細木召集了保安隊教導總隊的幹部，解釋了誤炸的原因，並表示會處理善後。[[张庆餘|张庆餘]]等保安队领导人則早在1935年12月就與29軍長[[宋哲元|宋哲元]]見面，預謀起兵叛變，不再接受日军和亲日的[[冀東防共自治政府|冀東防共自治政府]]的领导，而投奔当时驻扎在附近的中国军队。

==经过==
7月27日在[[北平|北平]]附近，驻守[[通州_(北京)|通州]]（今[[通州区_(北京市)|通州区]]）的中國保安队因聽信南京電臺（中國[[國民政府|國民政府]]的官方電臺）對外宣傳了“[[盧溝橋|盧溝橋]]29軍大敗日軍，中國軍隊已經陸續奪回[[豐台|豐台]]和[[廊坊|廊坊]]，中央政府將陸續向華北派遣野戰軍，殲滅日軍指日可待。”的謠言，而事實正好相反。南京政府最後還宣稱“軍事會議決定，蔣委員長命令附近的29軍大舉進攻冀東，血洗偽都通州，並血祭漢奸殷汝耕的決議”的決定。聽了廣播後，通州的中國保安隊於是誤判情勢，於7月29日開始對該地的日軍守備隊和特務機關發動攻擊，當時，通州的日本留守部隊為藤尾小隊的40人，山田自動車中隊的50人，連[[憲兵|憲兵]]、兵站（從事補給）、兵器部等其他部門在內一共110人的薄弱兵力。[[張慶餘|張慶餘]]、[[張研田|張研田]]率領的兩支保安部隊看準日方部隊力量單薄的時候，開始對日本軍民進行襲擊。中國保安隊趁著夜色襲擊了長官公署並俘虜了[[殷汝耕|殷汝耕]]，中國保安隊的主力則攻擊日本軍守備隊。日方雖然只有極少的部隊，但是依然進行抵抗。日本守備隊除了[[輕機槍|輕機槍]]、[[步槍|步槍]]、[[手榴彈|手榴彈]]以外，沒有任何的重武器。守備隊冒著敵方的密集火力，死傷不斷增加的情況下開始撤退，日本僑民的住處和特務機關最後也挺不住。通州日本特務機關遭到了相當於一個中隊的保安隊的襲擊，終於全員陣亡，特務機關被全部消滅。日方称大量的日本僑民（多數是老弱婦孺）被搶劫、[[強姦|強姦]]、凌辱和殺戮。至於此事件的責任者以及日本僑民是否為「平民」仍存在爭議。

=== 死者 ===
目击事件的日本、美国记者认为日本人居民117人和朝鲜人居民106人遭奸杀。[[蒋中正|蒋中正]]二战后出版的《[[蒋总统秘录|蒋总统秘录]]》回忆过这一事件，但称日本人死亡104、朝鲜人死亡108，并认为被杀者全是军人，是日本夸大了事件<ref>{{cite web|url=http://www.voachinese.com/content/japan-comfort-woman-20160605/3362330.html|title=日本申遗“通州事件”抗中韩申遗“慰安妇”}}</ref>。

== 其他觀點 ==
有觀點認為：<ref>{{cite web|url=http://www.here4news.com/article/1037440|title=交通事故水準的「大屠殺」 -- 談談通州事件}}</ref> 
#日本的居留民不應該被認為是一般意義上的平民。無論是[[九一八事變|九一八事變]]還是[[一二八事變|一二八事變]]，日本的居留民都是持有武器，和日本軍隊共同作戰的。日本方面也承認當時的 “居留民”都持有槍支武器，事實上後來的華北僞軍很多都是日本“居留民”中的預備役軍人訓練的。手持武器作戰，又要求別人把自己當作平民對待，是一種採取[[雙重標準|雙重標準]]的行為。
#通州事件中，施暴的並非中國政府所屬的軍隊。張慶余、張硯田所部，正是日軍入侵中國過程中建立並指揮的僞軍。此事件中日本人未能適當管理下屬偽軍，因此最終的責任者應是日本人自身。

== 结果及影响 ==
這次的攻擊導致“[[冀东防共自治政府|冀东防共自治政府]]”威信受創。但是由于通州保安队没有得到华北的29军[[宋哲元|宋哲元]]部的配合，因此沒有能夠改變華北戰局。

通州事件之後，冀东政府和日本政府雙方一起驗屍，冀东防共自治政府最後向日方賠償了約120万日元。<ref>{{cite web|url=http://www.geocities.jp/yu77799/tuushuu/tuushuu1.html|title=「通州事件」への視点 - ゆうのページ|language=ja}} </ref>

近年来有历史学者开始介入研究“通州屠杀事件”与4个半月之后发生的“[[南京大屠杀|南京大屠杀]]”之间是否存在相关影响和因果联系。
