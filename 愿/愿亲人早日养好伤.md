== 创作背景 ==
[[File:1964-09_1964年_红嫂.jpg|thumb]]
1961年，作家[[刘知侠|刘知侠]]根据1943年[[沂南县|沂南县]]聋哑妇女[[明德英|明德英]]用[[乳汁|乳汁]]拯救重伤的13岁[[八路军|八路军]]伤员的真实事例，创作了短篇小说《红嫂》，后来被改编成[[京剧|京剧]]<ref>{{cite web |author1=姚老庚 |title=令人动容的沂蒙红嫂明德英故事 |url=http://www.crt.com.cn/news2007/news/YLGRSMZZZXSLFHZHBBGS/2018/3/1837957196H1A050DDDDD4C1JGDJE.html |website=中红网}}</ref>。1964年，全国京剧现代戏观摩演出大会在[[北京|北京]]举行<ref>{{cite web |author1=仝志强 |author2=王珏 |author3=郝际钰 |title=忆当年，“红嫂”进京 |url=http://www.people.com.cn/GB/wenhua/25806/2967459.html |website=人民网}}</ref>，淄博京剧团的《红嫂》是[[山东省|山东省]]选派的作品之一，获得优秀奖。汇演结束后，《红嫂》等剧目专程到[[北戴河|北戴河]]为[[毛泽东|毛泽东]]主席等[[党和国家领导人|党和国家领导人]]演出，毛泽东看后对其大加称赞，并指示将其拍成电影<ref>{{cite web |author1=滕久昕 |title=毛主席曾评现代京剧《红嫂》：玲珑剔透 |url=http://www.chinanews.com/cul/2013/11-04/5458248.shtml |website=中国新闻网-北京青年报}}</ref>。

1971年初，[[芭蕾舞剧|芭蕾舞剧]]《[[沂蒙颂|沂蒙颂]]》创作组成立，其后6个月间，曾先后六次到山东[[沂蒙山区|沂蒙山区]]采风，在沂南县岸堤、横河等地体验生活。1972年初，《沂蒙颂》在北京[[天桥剧场|天桥剧场]]举行试验性公演，1973年5月16日，精编版《沂蒙颂》正式公演，1975年，精编版《沂蒙颂》由[[八一电影制片厂|八一电影制片厂]]拍摄为电影<ref>{{cite web |title=追寻芭蕾舞剧《沂蒙颂》背后的真情故事 |url=http://www.chinanews.com/cul/2011/06-13/3107704.shtml |website=中国新闻网-新华日报}}</ref>。
