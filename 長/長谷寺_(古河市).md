'''長谷寺'''（又稱為'''長谷觀音'''，日語：はせかんのん），是位於日本[[茨城縣|茨城縣]][[古河市|古河市]]的[[真言宗豐山派|真言宗豐山派]]佛寺，正式的[[山號|山號]]為明觀山；院號為觀音院；寺號為長谷寺<ref name="koho">{{Cite web |url=http://www.city.ibaraki-koga.lg.jp/cmsfiles/contents/0000000/283/17.pdf |title=広報「古河」No.22（2007年7月1日）　文化財の窓「長谷観音」 |access-date=2018-09-07 |archive-url=https://web.archive.org/web/20140104205426/http://www.city.ibaraki-koga.lg.jp/cmsfiles/contents/0000000/283/17.pdf |archive-date=2014-01-04 |dead-url=yes }}</ref>。是與[[古河公方|古河公方]]有關的寺院。

== 歷史 ==

[[战国时代_(日本)|戰國時代]]的[[明應|明應]]2年（[[1493年|1493年]]），{{tsl|ja|足利成氏}}中的初代[[古河公方|古河公方]]為了除厄古河城的鬼門，從[[鎌倉|鎌倉]]的[[長谷寺_(鎌倉市)|長谷寺]]恭請[[十一面觀音|十一面觀世音菩薩]]立像，並建立堂舍為此寺的起源。在[[江戶時代|江戶時代]]，也成為歷代古河城主的祈願所。在[[文政|文政]]13年（[[1830年|1830年]]）左右寫成的《古河志》裡，與[[大和國|大和]]（[[奈良县|奈良县]]）和鎌倉的長谷觀音一起並列以「三長谷」稱呼介紹。在[[明治|明治]]初年（[[1868年|1868年]]）被廢寺，八間四面的本堂也被破壞，到[[大正|大正]]3年（[[1914年|1914年]]），才由有志的供養者於原地復興。但是，同時十一面觀音被移往市内横山町的德星寺<ref>『古河市史 民俗編』 古河市、825-826頁、1983年</ref><ref>『古河市史 資料 別巻』 古河市、275-276頁、1973年　（『古河志』所収部）</ref>。

被認為是日本三大長谷觀音之一的原因，是因為有口述傳說大河、鎌倉、古河的長谷觀音都是從同一棵[[樟树|樟树]]所雕刻出來的<ref>[http://www.hasekannon.org/syokai/ 古河長谷観音ホームページ　長谷観音の歴史]</ref>。

== 文化資產 ==

* 木造 十一面觀世音菩薩立像： 其由來根據前述。像高為206.34公分，寄木造、玉眼。屬於古河市指定文化資產（雕刻）<ref>『古河市の文化財』 20頁</ref>。

== 葛飾坂東觀音靈場 ==

長谷寺（長谷觀音）也是[[葛飾坂東觀音靈場|葛飾坂東觀音靈場]]的番外禮所。其御詠歌為「無論什麼時候 發誓努力拯救三界眾生 長谷觀音 （日語：いつの世も　三国の衆生を　すくわんと　ちかいもかたき　長谷観音）」<ref>鈴木印刷所 制作・印刷・製本　『葛飾坂東札所めぐり公認ガイドブック　葛飾坂東観音御開帳』　平成26年（2014年）、49頁（長谷寺）</ref>。

葛飾坂東觀音靈場的各寺院都有其「御詠歌」，據說參拜時唱誦著御詠歌，與繞唱經文有同樣的功德<ref name="guide11">鈴木印刷所 制作・印刷・製本　『葛飾坂東札所めぐり公認ガイドブック　葛飾坂東観音御開帳』　平成26年（2014年）、11頁（葛飾坂東観音御開帳のあらまし）</ref>。
