'''長榮海運股份有限公司'''（{{Lang-en|'''Evergreen Marine Corporation'''}}，{{lse|EGMD}} 、{{twse|2603}}），簡稱'''長榮海運'''或'''長榮，'''是一家[[中華民國|中華民國]]籍的[[海運|海運]]公司，為[[長榮集團|長榮集團]]核心事業之一，也是臺灣第一大海運公司。總部位於桃園市[[蘆竹區|蘆竹區]][[南崁|南崁]]，是一家全球性集裝箱貨運航運公司。其主要航線遍及全球，並擁有超過150艘集裝箱船。該企業是全球第五的之航運公司，僅次於[[快桅|快桅]]、[[地中海航運公司|地中海航運公司]]、[[法國達飛海運集團|法國達飛海運集團]]及[[中遠集團|中遠海集裝箱海運]]。同時，亦在臺灣與[[陽明海運|陽明海運]]、[[萬海海運|萬海海運]]並列為「臺灣三大航運公司」。其業務包括船運、集裝箱和港口管理、工程和房地產開發。子公司和下屬部門包括立榮海運股份有限公司（臺灣）、Evergreen Marine (UK) Ltd.（英國）和航運公司Italia Marittima S.p.A.（意大利）。 2007年，Hatsu Marine Limited（英國）、Italia Maritima和長榮合併為單一的“Evergreen Line”。

長榮的貨櫃大部分都是綠色的，並在兩側放置白色字母“Evergreen”。 立榮的貨櫃樣式亦大致相同，但是底色為亮橙色。長榮的冷藏貨櫃之顏色配置則為綠字白底。

== 簡介 ==
*1968年9月1日，長榮海運由[[張榮發|張榮發]]創立。
*1975年，受惠於全球[[能源危機|能源危機]]，長榮海運開闢[[遠東|遠東]]╱[[美國東岸|美國東岸]]全貨櫃定期航線，首開臺灣全貨櫃船隊之先例。
*1984年，長榮海運首先開闢了環球東西雙向全貨櫃定期航綫，以全自動貨櫃船，配合電腦資訊聯機系統，連結亞、歐、美3大洲之運送服務網路，提供運輸服務。
[[File:Evergreen_Marine_Building.jpg|thumb]]
[[File:Evergreen_shipping_container.jpg|thumb]]
*1990年代初期曾爲全球排名第一的貨櫃[[集裝箱|集裝箱]][[運輸|運輸]]公司。 
*1999年8月，長榮海運於[[桃園市|桃園市]][[蘆竹區|蘆竹區]]自行斥資興建「長榮企業大樓」，9至10樓設置「長榮船員訓練中心」。
*2002年，長榮海運以二條鐘擺航綫取代了經營18年的環球東西雙向航綫。 
*2014年3月：長榮海運加入由[[中遠集團|中遠]]、川崎汽船、[[陽明海運|陽明]]以及[[韓進海運|韓進]]四家所形成的CKYH聯盟後，組建成新的CKYHE聯盟<ref>張佩芬、呂淑美，[http://www.chinatimes.com/newspapers/20140221000136-260204 長榮海運 加入CKYHE聯盟]，《中時電子報》，2014-02-21</ref>。
*發展至今，已共經營約165艘全[[貨櫃輪|貨櫃輪]]，不論船隊規模或貨櫃承載量皆位居全球領先地位。
*而後擴大經營範圍，先後成立[[長榮航空|長榮航空]]、[[立榮航空|立榮航空]]、[[長榮國際|長榮國際]]、[[長榮國際儲運|長榮國際儲運]]、[[長榮航太科技|長榮航太科技]]、[[長榮保全|長榮保全]]、[[長榮空廚|長榮空廚]]、[[長榮交響樂團|長榮交響樂團]]及[[長榮桂冠酒店|長榮桂冠酒店]]等，並且併購原[[義大利|義大利]]國營「義郵輪船」公司（目前更名為意大利海運公司）。其全體事業機構一般總稱爲[[長榮集團|長榮集團]]。

==海運據點與航線==
*長榮海運網路遍及全球114多個[[國家|國家]]，服務據點315餘處。
*所經營的遠、近洋全貨櫃定期航綫：[[亞洲|亞洲]]—[[北美|北美]]航綫 / 亞洲—[[加勒比海|加勒比海]]地區；亞洲—[[歐洲|歐洲]]航綫 / 亞洲—[[地中海|地中海]]；歐洲—[[美國|美國]]東岸[[大西洋|大西洋]]；亞洲—[[澳洲|澳洲]] / 亞洲—[[模里西斯|模里西斯]]、[[南非|南非]]、[[南美|南美]]；亞洲區域航綫 / 亞洲—[[中東|中東]]、[[紅海|紅海]] / 亞洲—[[印度|印度]]次大陸地區。除了主要[[航綫|航綫]]外，亦開闢了區域性[[接駁船|接駁船]]的服務網，如加勒比海及印度次大陸等地區。

== 聯營夥伴 ==
長榮海運公司與臺灣第二大航商[[陽明海運|陽明海運]]與香港航商[[東方海外|東方海外]]聯合開闢連接臺灣/香港/胡志明港之快速航線（Taiwan/Hong Kong/Ho Chi Minh Express Service，簡稱THX），以提供貨主每週雙班臺灣/香港至胡志明港直航運送服務。
