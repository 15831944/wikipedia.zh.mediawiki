== 型態 ==
長喙殼菌屬(Ophiostoma)具有長喙，子囊在成熟後不會進行噴發而是進行分解，黏稠狀的孢子會聚集在頂端，此屬之大多種類為異絲型，即此類真菌需要不同配對型配對才能產生有性繁殖之[[子囊孢子|子囊孢子]] 。<gallery>
File:Изображение 037 htl.jpg|黏稠的孢子聚集在頂端，圖中水滴狀的物體就是他的孢子
File:Ophiostoma ulmi coremium (1).jpg
</gallery>

具有Pesotum、Sporothrix、Leptographium三種分生孢子型態，此三種型態皆為濕性孢子，濕性孢子能夠助於其附著在昆蟲上面進行傳播。

== 與小蠹蟲的關係 ==
此分類群的多數種類真菌能藉由樹皮甲蟲（小蠹蟲，''Scolytidae''）傳播，雌性甲蟲尋找死亡或削弱的榆木，挖掘樹皮和木材，造成取食痕，雌蟲在主要的取食痕產卵，幼蟲在小的取食痕末端化蛹羽化，讓成蟲攜帶孢子進行傳播，真菌會配合幼蟲羽化時，產生大量的分生孢子，讓孢子著在其之上，並傳染給下一個寄主。而健康榆樹的感染發生在甲蟲以葉腋和幼枝為食時。一些孢子掉入樹木的[[木质部|木質部]]，此時真菌將透過酵母型迅速萌芽繁衍。

== Ophiostoma與Ceratocystis的区別 ==
{{NotMandarinSect}}

Ophiostoma與[[小囊菌目|小囊菌目]]的Ceratocystis屬的有性繁殖型態相近，同樣具有長喙與濕性孢子。但兩種真菌在諸多地方也有相異之處。
