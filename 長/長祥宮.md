'''長祥宮'''又稱'''五穀先帝廟'''，位於[[新屋區|新屋區]][[九斗|九斗里]]，佔地共1甲7分，是當地重要的信仰中心。民間信仰認為[[神農|五穀神農]]是新屋的守護神，護佑當地魚米物產豐收。因新屋是[[桃園市|桃園]]最大的米倉，故又有魚米之鄉的美名。長祥宮每逢假日會有許多香客前來進香祭拜，是新屋地區重要的信仰中心之一 <ref>{{cite web | url = http://easy.sina.com.tw/scenic/article_contentid-1144.html | title = 新屋鄉五穀先帝廟 - 長祥宮 | work = Tranews | accessdate = 2018-06-04 | deadurl = no | archiveurl = https://web.archive.org/web/20170914151146/http://easy.sina.com.tw/scenic/article_contentid-1144.html | archivedate = 2017-09-14 | df =  }}</ref>。

==全台最大神農大帝青銅像==
新屋區長祥宮，有一座手持稻穗的神農大帝像，是目前全[[台灣|台灣]]最高的神農大帝青銅像，於[[中華民國|民國]]82年（西元1993年）鑄造完成。信眾們共籌募四千萬元，以青銅六十五噸、鍍鋅鐵七十五噸，打造了這座高二十公尺的神農大青銅像。加上底座三層樓高的「新屋老人會館」，整座神農大帝青銅像總高度達到三十六公尺<ref>{{cite web| url =http://www.merit-times.com.tw/NewsPage2.aspx?unid=261932| title =神農帝今聖誕 新屋長祥宮神農帝 全台最大青銅像| work =人間福報| location =桃園| date =2012-05-16| accessdate =2018-06-04| deadurl =no| archiveurl =https://web.archive.org/web/20180608095319/http://www.merit-times.com.tw/NewsPage2.aspx?unid=261932| archivedate =2018-06-08| df =}}</ref>。

==歷史==
據傳[[清朝|清朝]][[乾隆|乾隆]]時期，有一位信士揹著一尊[[神農|五穀神農]]像，恭奉在埔頂莊彭進安的住家。西元1902年9月7日，信士[[范姜姓|范姜]]捐贈九斗村現在的地址建置神農像。安置神農像的舊廟原先是以土磚興造，歷經80年風吹雨打，廟體早已破舊不堪。村民與地方仕紳經過商議後，決定翻新舊廟，以表對五穀神農的崇敬。在長祥宮重建委員大力的倡導與信士們慷慨解囊下，於民國69年（西元1980年）農歷4月初4日子時動工分段興建，除主體建築外，另於廟旁空地增建公園，讓香客及附近的居民可以在公園活動休憩<ref name=‘cloud’>{{cite web | url = http://cloud.hakka.gov.tw/Details?p=66129 | title = 新屋長祥宮 | work = 臺灣客庄文化數位典藏系統 | date = 2016-10-20 | accessdate = 2018-06-04 | deadurl = no | archiveurl = https://web.archive.org/web/20180608095318/http://cloud.hakka.gov.tw/Details?p=66129 | archivedate = 2018-06-08 | df =  }}</ref>。

民國72年（西元1983年）12月10日丑時，舉辦玉皇大帝登龕大典。民國74年（西元1985年）天運乙丑年11月20日起，舉行5天的慶成福醮活動<ref name = ‘cloud’/>。

== 主祀 ==
長祥宮正殿供奉五殼神農大帝、[[媽祖|天上聖母]]、[[註生娘娘|註生娘娘]]、[[關羽|關聖帝君]]等。二殿供奉[[玉皇上帝|玉皇大帝]]、南北二斗星君、[[三官大帝|三官大帝]]、[[文昌帝君|文昌君]]等神祇<ref>{{ cite web | url = http://www.xinwu.tycg.gov.tw/home.jsp?id=16&parentpath=0,1,8,15 | title = 寺廟 - 長祥宮 | work = 桃園市新屋區公所 | date = 2014-11-11 | accessdate = 2018-06-04 | deadurl = no | archiveurl = https://web.archive.org/web/20180608095318/http://www.xinwu.tycg.gov.tw/home.jsp?id=16&parentpath=0,1,8,15 | archivedate = 2018-06-08 | df =  }}</ref>。

== 景觀設施 ==
長祥宮內進門右邊有一個小吃部，左邊有廁所，宮內有一座公園，公園內有三座[[亭|涼亭]]供香客、民眾休息；還有一座[[池塘|水池]]，裡面有養魚。
