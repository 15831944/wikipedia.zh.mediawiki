'''長瀨麻美'''（{{lang-ja|長瀬 麻美}}；{{lang-en|Mami Nagase}}，{{bd|1992年|6月6日}}），日本[[青森縣|青森縣]]出身的[[AV女優|AV女優]]及[[寫真偶像|寫真偶像]]。前「[[Aquas_Entertainment|Aquas Entertainment]]」、「Lupinus Production」、「J's Promotion」及「Studio Delight」旗下藝人，目前所屬事務所為「Diaz Group」，兼擔任「[[Max-A|Max-A]]」專屬女優。

== 簡歷 ==
長瀨麻美在[[青森縣|青森縣]]中學畢業後往[[東京_(消歧義)|東京]]發展<ref>{{Cite web |url=https://news.dmm.co.jp/article/901328/ |title=長瀬麻美「AVならもっと自己表現できるかなと転身をあっさり決意しました（笑）」（前編） |publisher={{tsl|ja|FANZA||FANZA}} |date=2014-12-17 |accessdate=2018-12-1 |language=ja}}</ref>，於2009年至2012年1月31日間曾以「水谷彩也加」名義，擔任寫真偶像<ref>{{Cite web |url=https://news.mynavi.jp/article/20130329-a257/ |title=Hカップバストのド迫力! 長瀬麻美、シースルー衣装でバストが立体的に |publisher={{tsl|en|Mynavi||Mynavi}} |date=2013-3-29 |accessdate=2018-12-1 |language=ja}}</ref>，其胞妹[[水谷瑠奈_(寫真偶像)|水谷瑠奈]]亦從事相同行業。2012年4月1日，她才用現時藝名「長瀨麻美」加入藝能事務所「Lupinus Production」，並曾參加由日本報章《[[東京體育|東京體育]]》舉辦的《Miss 東spo 2014》<ref>{{Cite web |url=https://news.mynavi.jp/article/20131206-a545/ |title=ミス東スポ2014ファイナリストの長瀬麻美、過激水着で限界ギリギリ露出 |publisher=Mynavi |date=2013-12-6 |accessdate=2018-12-1 |language=ja}}</ref>，但最後落敗。

2014年初，長瀨麻美與新公司「J's Promotion」簽約，同年7月底又轉投另一事務所「Studio Delight」，9月則選擇出道成為「[[Alice_Japan|Alice Japan]]」旗下專屬女優，開始拍攝成人影片<ref>{{Cite web |url=http://orientaldaily.on.cc/cnt/adult/20140829/00290_004.html |title=色日放送：長瀨麻美最強胸器 |publisher=[[東方日報_(香港)|東方日報]] |date=2014-8-29 |accessdate=2018-12-1}}</ref>，且憑著其豐滿上圍而受到歡迎<ref>{{Cite web |url=http://orientaldaily.on.cc/cnt/adult/20140702/00290_002.html |title=模影模蹤：長瀨麻美女王駕到！ |publisher=東方日報 |date=2014-7-2 |accessdate=2018-12-1}}</ref><ref>{{Cite web |url=https://www.ettoday.net/dalemon/post/32950 |title=吉村卓欽點「五大酥麻」女優　初戰三上悠亞：那餘韻忘不了 |publisher=[[ETtoday新聞雲|ETtoday新聞雲]] |date=2018-1-15 |accessdate=2018-12-1}}</ref><ref>{{Cite web |url=http://www.dansen-web.com/article/detail/1138657/ |title=【イベント情報】ニコ生「ハートの部屋(仮)」キリ番40回に長瀬麻美を迎えブリしゃぶ鍋でお祝い＜麻雅庵のセクシーアイドル覗きアナ＞ |publisher=[[東京體育|東京體育]] |date=2018-10-20 |accessdate=2018-12-1}}</ref>。
