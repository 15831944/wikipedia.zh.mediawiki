'''長島冰茶'''（{{lang-en|Long Island Iced Tea}}）雖取名冰茶，卻是在沒有使用[[紅茶|紅茶]]的情況下，調製出具有紅茶色澤與口味的[[雞尾酒|雞尾酒]]。調製的主要成份為[[伏特加|伏特加]]、[[龍舌蘭|龍舌蘭]]、[[琴酒|琴酒]]，酒精濃度大約30%。<ref>[http://www.xiachufang.com/recipe/1072625/]，长岛冰茶。</ref>

起源之一的說法為1920年代[[美國|美國]][[禁酒令|禁酒令]]期間，酒保將烈酒與可樂混成一杯看似茶的飲品，來鑽法令漏洞；起源之二是據說在1972年，由長島橡樹灘客棧（Oak Beach Inn）的酒保發明了這種以四種基酒混製出來的飲料。

== 特色 ==
味道微辣而帶有[[可樂|可樂]]與紅茶的氣味，適合餐後飲用。

因為口味香甜，蓋過濃烈的酒精味，使人不知覺的飲用過量而酒醉，因此又稱為失身酒。

== 調製方法 ==
* 一份[[伏特加|伏特加]]
* 一份[[琴酒|琴酒]]
* 一份[[蘭姆酒|蘭姆酒]]
* 一份[[龍舌蘭酒|龍舌蘭酒]]
* 一份[[橙皮酒|橙皮酒]]
* 半份[[酸甜汁|酸甜汁]]或檸檬汁
* 一片檸檬
* 可樂

將所有材料放入[[可林杯|可林杯]]中，然後添滿可樂以產生類似紅茶的顏色，最後在杯緣用檸檬片裝飾。

== 變種 ==
{{commonscat|Long Island Iced Tea}}
* 長島冰茶第二代 - 多加一份[[波本威士忌|波本酒]]。
* 長島冰茶第三代 - 在第二代裡多加一份[[白蘭地|白蘭地]]。
* 長堤冰茶 - 用[[小紅莓|小紅莓]]汁代替可樂。
* 加州冰茶 - 用[[阿玛雷托|杏仁酒]]代替龍舌蘭與橙皮酒，並用等量的[[小紅莓|小紅莓]]汁與[[鳳梨|鳳梨]]汁取代可樂添滿。
* 夏威夷冰茶 - 用[[香博树莓力娇酒|香博树莓力娇酒]][[覆盆子酒|覆盆子酒]]取代龍舌蘭與橙皮酒，並用[[雪碧|雪碧]][[汽水|汽水]]取代可樂。
* 邁阿密冰茶 - 用{{link-en|蜜瓜酒|Midori (liqueur)|蜜瓜酒}}與桃味蒸餾酒（peach schnapps）取代龍舌蘭與橙皮酒，而用柳橙汁代替可樂。
* 加勒比冰茶 - 用牙買加黑蘭姆酒（dark jamaican rum）取代龍舌蘭與伏特加。
* 德州冰茶 - 用白蘭地取代琴酒。
* 東京紅茶 - 用一份{{link-en|蜜瓜酒|Midori (liqueur)|蜜瓜酒}}取代可樂，又稱「三哩島」。
* 比佛利山冰茶 - 用香檳替代可樂。
* 觸電冰茶 - 用波本酒取代龍舌蘭。
* 阿拉斯加冰茶 - 用四份[[库拉索酒|库拉索酒]]與四份[[酸甜汁|酸甜汁]]取代龍舌蘭酒，檸檬蘇打水取代可樂。
* 廣島冰茶 - 用生命之水取代伏特加、用[[百加得151|百加得151]]取代蘭姆（皆為同酒型但酒精濃度為兩倍）
