'''長沙灣天主教英文中學'''（{{lang-en|Cheung Sha Wan Catholic Secondary School}}，簡稱CSWCSS或長天／舊稱長記）創辦於1970年，屬[[香港天主教教區|香港天主教教區]]轄下一間(第一組別中-尾)政府資助的[[香港英文授課中學|英文文法中學]]，亦是香港114間以英語為主要教學語言的學校之一。<ref name=":2">{{Cite web|url=https://www.schooland.hk/ss/cswcss|title=長沙灣天主教英文中學 Cheung Sha Wan Catholic Secondary School 長沙灣天主教中學, 長天|accessdate=2017-08-27|work=www.schooland.hk|language=zh-hk}}</ref>

== 辦學宗旨<ref name=":2" /> ==
秉承天主教辦學宗旨，以德、智、體、群、美、靈六育並重，「全人發展」為基本教育方針，致力發展關愛文化，教導學生認識福音的真理，並在生活上活出基督的精神。在德育方面遵照校訓「自強不息」的信念，致力培育學生奮發自強、修身不懈、積極的人生觀及培養服務社會的精神。

== 學術 ==
在學術方面，該校重視兩文三語的訓練，學生在公開試的成績一向出眾，入大學率高達8成<ref>{{Cite news|url=http://www.sundaykiss.com/%E5%90%8D%E6%A0%A1%E6%89%8B%E5%86%8A/%E4%B8%AD%E5%AD%B8/%E9%95%B7%E6%B2%99%E7%81%A3%E5%A4%A9%E4%B8%BB%E6%95%99%E8%8B%B1%E6%96%87%E4%B8%AD%E5%AD%B8-%E6%95%99%E5%8D%80%E8%8B%B1%E4%B8%AD-%E9%AB%94%E8%82%B2%E6%9C%80%E5%BC%B7/|title=長沙灣天主教英文中學 教區英中 體育最強 - 中學 - Sundaykiss|author=|date=2015-01-03|work=|newspaper=Sundaykiss|accessdate=2017-08-27|language=en-US}}</ref>。課外活動多元化，在學界運動比賽、文化藝術、學術比賽及服務社會多方面均有出色表現。

該校初中的科目包括：中國歷史、中國語文、倫理、公民及道德教育、普通話、體育、數學、綜合科學、英國語文、視覺藝術、通識教育、閱讀、電腦及科技、音樂；高中的科目包括：中國語文、體育、倫理、中國歷史、體育（文憑課程）、宗教教育、視覺藝術、通識教育、英國語文、數學、數學延伸單元、化學、物理、生物、地理、歷史、經濟、資訊及通訊科技、企業會計與財務概論。<ref name=":2" />

==四社==
每名學生於中一入學時，均會根據班別而被編入其中一個學社。
*聖路加社（St. Luke's House）
*聖馬爾谷社（St. Mark's House）
*聖若望社（St. John's House）
*聖瑪竇社（St. Matthew 's House）

而每年分數最低的一個社可優先選擇明年屬於自己社的中一班別，大多由精英班（1b，1c）先選，由此可看出去年四社排名。因此，中一入學時同班同學皆為同一個社（1E班除外）。

== 運動 ==

===長跑文化===
該校擁有一條長285米跑道圍繞校園，有體育課堂跑校圈的練習，而每年體育課考試則是環校跑，於每年11及5月進行。

該校的另一年度活動是越野賽。每年秋季，該校會連同學生會安排全體學生於[[金山郊野公園|金山郊野公園]]參與越野賽。賽道主要為繞水塘的山路進行，賽程約6公里。優勝學生可獲推薦進入越野賽校隊參與集訓，及代表學校參與各項長跑比賽。2012年11月23日，學校首次因大雨而取消比賽；2018年11月2日，因熱帶風暴[[颱風玉兔_(2018年)|玉兔]]吹襲懸掛[[一號戒備信號|一號戒備訊號]]而將比賽延期至2019年1月4日。

該校亦多次在港九區第一組學界越野賽及多個長跑比賽中取得佳績。

===排球===
自2004年起，該校多次獲得港九區第一組學界總冠軍。於2010年，分別取港九區第一組學界總冠軍以及精英賽冠軍。該校三度獲得學界「全港學界精英排球比賽」冠軍<ref>{{Cite news|url=http://sportsroad.hk/archives/100806|title=【學界排球精英賽】長天復仇挫筲官  勇奪精英賽三連霸|newspaper=體路 Sportsroad|accessdate=2017-08-27}}</ref>。

===香港天主教教區中學聯校運動會===
該校曾於兩年一度的[[香港天主教教區中學聯校運動會|香港天主教教區中學聯校運動會]]的第四屆（1976年）至第十九屆（2012年）連續16屆獲得男子全場總冠軍。

及後在第二十屆不幸落敗，其後重回勝軌。

== STEM 發展 ==
在2015-2016學年，該校獲政府選為八間資訊科技增潤計劃學校之一，獲得七百萬港元推動STEM發展<ref>{{Cite web|url=https://www.eitp.gov.hk/tc/it-class-cswcss|title=資訊科技班 - 資訊科技增潤計劃 - 長沙灣天主教英文中學|accessdate=2017-08-27|author=|date=|work=www.eitp.gov.hk|publisher=|language=zh-hant}}</ref>。

該校學生在顧問老師[[屈凱添|屈凱添]]的帶領下，在2016青少年機械人世界香港區公開賽囊括8個獎項，更包辦足球賽的冠亞季軍<ref>{{Cite web|url=http://hk.apple.nextmedia.com/news/art/20160703/19679517|title=中小學機械人足球賽「長天」包辦三甲|accessdate=2017-08-27|work=Apple Daily 蘋果日報}}</ref>。
