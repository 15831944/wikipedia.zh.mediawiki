'''長絲𩷶'''（[[學名|學名]]：''Pangasius sanitwongsei''）是[[𩷶屬|𩷶屬]]的一種底棲[[魚類|魚類]]。<ref name=Fishbase>{{FishBase species|genus=Pangasius|species=sanitwongsei|year=2007|month=3}}</ref> 俗稱'''成吉思汗 (魚)'''、'''成吉思汗魚'''，分佈于[[湄南河|湄南河]]及[[湄公河|湄公河]]。

== 特徵 ==
長絲𩷶的身體上有[[黑色素細胞|黑色素細胞]]，[[頭部|頭部]]闊而扁平，沒有鰓鬚。它們的[[腹部|腹部]]呈銀色及彎曲，背部呈深褐色。背鰭、胸鰭及臀鰭都是深灰色的，第一鰭條幼長成絲狀，可長達300公分，重300公斤，世界上最大的[[鯰魚|鯰魚]]之一。

== 行為 ==
長絲𩷶主要吃[[甲殼類|甲殼類]]及[[魚類|魚類]]。它們是會遷徙的。它們一般會在[[季風|季風]]季節前產卵。<ref name=Fishbase />

== 與人類關係 ==
捕獵長絲巨鯰往往是與[[宗教|宗教]]禮儀有關，也有將它們作為食物的。也有飼養做為觀賞魚的。<ref name=Fishbase />
