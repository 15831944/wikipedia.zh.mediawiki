'''长沙市第十一中学'''，原名“妙高峰簡易師範”、“妙高峰中學”，位於[[湖南省|湖南省]][[長沙市|長沙市]][[雨花區|雨花區]]城南中路，毗鄰[[天心閣|天心閣]]，是一所公立全日制普通高级中学，亦是众多湖南省省级示范性普通高级中学中的一所。<ref name="1a">{{Cite web |url=http://www.11edu.com/newsdetail.aspx?typeid=314&id=1863 |title=学校简介1_学校介绍_长沙市第十一中学 |accessdate=2018-08-05 |work=www.11edu.com}}</ref>

== 歷史 ==
*1906年，學校在長沙市境內的妙高峰[[城南書院|城南書院]]創立，定名“妙高峰簡易師範”。<ref>{{Cite web |url=http://www.sohu.com/a/232779071_390660 |title=揭秘，长沙这些百年历史学校，你不了解一下？ |accessdate=2018-08-05 |date=2018-05-24 |work=www.sohu.com}}</ref>
*1914年，學校更名為“妙高峰中學”。
*1953年，學校更名為現時之名，即“長沙市第十一中學”。<ref name="1c">{{Cite web |url=http://www.11edu.com/newsdetail.aspx?typeid=314&id=2595 |title=学校简介2_学校介绍_长沙市第十一中学 |accessdate=2018-08-05 |work=www.11edu.com}}</ref>
*1954年，學校搬遷至現址長沙市城南中路。<ref name="1a" />

*[[曾滌|曾滌]]：[[中国人民解放军|中国人民解放军]]将领、中国人民解放军开国少将。
