'''長崎繞道'''（{{jpn|j=長崎バイパス|hg=ながさきバイパス|rm=Nagasaki By-Pass}}）是起於[[長崎縣|長崎縣]][[諫早市|諫早市]]多良見町，至同縣[[長崎市|長崎市]]的[[國道34號|國道34號]][[繞道|繞道]]，同時是[[西日本高速公路|西日本高速公路]]以[[收費道路|一般收費道路]]型態管理的[[快速道路|快速道路]]。本公路於[[長崎多良見交流道|長崎多良見交流道]]與[[長崎自動車道|長崎自動車道]]連接，另外在川平交流道起分歧為昭和町方向出口與西山町方向出口。屬於[[全國路線網|全國路線網]]的一環。

高速公路路線編號被編為 {{Ja Exp Route Sign|E96}} <ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/list/index.html|title=高速公路編號列表|accessdate=2017年2月26日|publisher=國土交通省}}</ref><ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/map/index.html|title=高速公路編號路線圖|accessdate=2017年2月27日|publisher=國土交通省}}</ref>。

從諫早市多良見町道[[川平交流道|川平收費站]]間的路段為全線4車道，但存在著許多急彎。全區間可使用[[ETC|ETC]]支付通行費。

== 概要 ==
* 起點：長崎縣諫早市多良見町市布
* 終點：長崎縣長崎市昭和二丁目、長崎縣長崎市西山四丁目
* 全長：15.1km
* 車道數：4車道（起點 - 川平）、[[2車道高速公路|暫定2車線]]（川平 - 昭和、川平 - 西山）
* 通行費徵收期限：[[2050年|2050年]][[8月15日|8月15日]]
* 速限：60km/h

== 交流道 ==
* 交流道欄背景色<span style="color:#BFB">■</span>區間是已通車區間，設施欄背景色<span style="color:#CCC">■</span>區間是未通車區間與未使用設施，未通車區間名稱全部是暫稱。
* 為了收費區名稱的方便上，多良見 - 昭和為A區間（因從川平分歧的關係，多指川平 - 昭和這一段）、川平 - 西山稱為B區間（通稱為「西山繞道」）。另外，[[川平收費道路|川平收費道路]]通稱為C區間。需要注意的是，從多良見方向進入到終點的川平之前都無法下交流道（古賀市布交流道、間之瀨交流道皆為僅能出入川平方向的半交流道）。

== 歷史 ==
* [[1967年|1967年]][[11月17日|11月17日]]：部分通車（昭和線完工），為九州第一條收費快速道路。當初為暫定2車道，後拓寬為4車道。
* [[1991年|1991年]][[3月27日|3月27日]]：二期工程完成（西山線完工）。
* [[2006年|2006年]][[10月31日|10月31日]]：試辦ETC通勤折扣制度。[[2007年|2007年]]度決定延長。但不適用轉乘川平收費道路者（自2008年10月後已適用）<ref>{{Cite web|url=http://corp.w-nexco.co.jp/corporate/release/kyushu/h18/1016/|title=長崎繞道導入ETC通勤折扣|accessdate=2006年10月16日|publisher=西日本高速公路}}</ref>。
* [[2010年|2010年]][[6月28日|6月28日]] 長崎多良見 - 川平間被指定為實行通行費全車種免費化社會實驗的對象區間。但是長崎繞道連接的[[川平收費道路|川平收費道路]]並非免費化實驗之對象。
