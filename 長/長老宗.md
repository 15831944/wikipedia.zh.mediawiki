'''長老宗'''（{{lang-en|'''Presbyterianism'''}}），其[[教会|教会]]称'''长老教会'''（{{lang-en|'''Presbyterian church'''}}，簡稱'''長老會'''），是[[西方基督教|西方基督教]][[新教|新教]][[喀爾文宗|喀爾文宗]]的一個[[基督教教派列表|流派]]，源自[[16世纪|十六世紀]]的[[蘇格蘭宗教改革|苏格兰改革]]。 长老教会持守[[加爾文主義|加尔文主义]]，尤其是[[苏格兰长老会|苏格兰长老会]]基本完全延续着[[讓·喀爾文|讓·喀爾文]]（John Calvin，1509年7月10日－1564年5月27日）及其学生的教义。20世纪，长老会对[[普世教會合一運動|普世教会合一运动]]有相当的参与。

== 词源 ==
這個教派的名字是源自[[希腊语|希腊语]]詞「{{lang|ru|πρεσβύτερος}}」（Presbyteros），就是“[[长老|长老]]”之意。

== 起源 ==
長老會的協會起源可以追蹤到[[蘇格蘭宗教改革|蘇格蘭宗教改革]]，長老教會於1560年由[[讓·喀爾文|讓·喀爾文]]的學生[[約翰·諾克斯|约翰·諾克斯]]（John Knox，1514年－1572年11月24日）在蘇格蘭進行宗教改革時正式建立。愛丁堡的[[圣吉尔斯大教堂|聖吉爾斯大教堂]]有時被認為是世界長老會的母會。

== 分布 ==
長老教會主要分布於[[臺灣|台灣]]、[[日本|日本]]、[[大韩民国|南韓]]、[[英国|英國]]、[[美国|美國]]、[[加拿大|加拿大]]、[[澳洲|澳洲]]、[[新西兰|紐西蘭]]、[[荷兰|荷蘭]]、[[法国|法國]]、[[巴西|巴西]]、[[新加坡|新加坡]]、[[越南|越南]]、[[马来西亚|馬來西亞]]、[[印度尼西亚|印尼]]、[[印度|印度]]等地。

== 歷史背景 ==
蘇格蘭人[[約翰·諾克斯|約翰·諾克斯]]（1505年—1572年）在[[日内瓦|日內瓦]]學習了[[加爾文主義|加爾文主義]]，1560年返回到[[蘇格蘭|蘇格蘭]]后帶領[[蘇格蘭教會|蘇格蘭教會]]接受改革。

== 教義 ==
有一些長老派的傳統採取了1647年創作的《[[韋斯敏斯德信仰宣言|韋斯敏斯德信仰宣言]]》當他們唯一的信條，作為本會牧師受訓練的教條標準。
