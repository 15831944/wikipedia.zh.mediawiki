{{vgname|理查德·伯恩斯拉力赛|en=Richard Burns Rally|official-cn=理查德·伯恩斯拉力赛|简称'''RBR'''}}是一款由Warthog Games开发，SCi发行的模拟竞速游戏。本作拥有[[世界拉力锦标赛|世界拉力锦标赛]]冠军[[理查德·伯恩斯|理查德·伯恩斯]]的授权及协助。
== 摘要 ==
本作拥有8辆获得授权的赛车和36条官方认可的赛道<ref name=sina />。最为人称道的是它的[[物理引擎|物理引擎]]<ref>{{Cite web |url = http://www.yystv.cn/p/3664 |title = 这可能是最棒的拉力赛车游戏 |author = Kong |date = 2018-08-01 |publisher = 游研社 |language = zh-cn |accessdate = 2018-08-04 }}</ref>。

本作被许多玩家评价为最真实与最难上手的模拟竞速游戏之一。
