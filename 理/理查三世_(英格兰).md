'''理查三世'''（{{lang-en|'''Richard III'''}}，{{bd|1452年|10月2日|1485年|8月22日|catIdx=R}}），[[英格蘭|英格蘭]]國王，1483年到1485年在位，他是[[爱德华四世_(英格兰)|愛德華四世]]之弟，封[[格洛斯特公爵|格洛斯特公爵]]，同時也是[[約克王朝|約克王朝]]的最後一任國王。他在[[博斯沃思原野战役|博斯沃思原野战役]]的敗戰，為[[玫瑰戰爭|玫瑰戰爭]]與[[金雀花王朝|金雀花王朝]]畫上句點。其兄愛德華四世過世後，理查曾短暫以護國公的身分替愛德華四世的兒子─[[愛德華五世|愛德華五世]]攝政，隨後將愛德華五世與他的手足，[[第一代约克公爵什鲁斯伯里的理查|舒茲伯利的理查]]（即第一任[[約克公爵|約克公爵]]）送進[[倫敦塔|倫敦塔]]後奪權成功，並於1483年7月6日被加冕為英格蘭王。

歷史曾發生過兩場針對理查三世的大規模叛亂行動：
第一樁發生於1483年，由強烈反對愛德華四世的人士及支持理查三世稱王的[[亨利·斯塔福德，第二代白金漢公爵|亨利·斯塔福德，第二代白金漢公爵]]領導。這次事變很快就被理查三世平定，而白金漢公爵則在賽利斯伯利地區的牛頭旅館附近被斬首。第二樁發生於1485年，領導者是[[亨利七世_(英格兰)|亨利·都鐸，第二任理奇蒙伯爵]]與其叔父[[賈斯伯·都鐸|賈斯伯·都鐸]]。叛軍成功登陸英格蘭，並於博斯沃思原野战役擊敗了理查三世。理查三世就此成為金雀花王朝的最後一任國王與最後一個死在戰鬥中的英格蘭國王。

== 童年 ==

於1452年生於弗德林黑城堡的理查是[[理查·金雀花，第三代約克公爵|理查·金雀花，第三代約克公爵]]與西西莉納維爾八個子女中的么子，是約克公爵夫婦四個活到成年的後代之一。理查大部分的童年時光都在溫絲莉谷的[[米德漢|米德漢]]城堡裡度過，由他的表兄[[理查·內維爾，第十六代沃威克伯爵|理查·內維爾，第十六代沃威克伯爵]]監護下成長。（此人因為在玫瑰戰爭裡的卓著的影響力，在歷史上被稱為「造王者」。）

在他的父親與兄長[[愛德蒙，拉特蘭伯爵|愛德蒙，拉特蘭伯爵]]於[[韋克菲爾德戰役|韋克菲爾德戰役]]中戰死後，還是個小男孩的理查便被沃威克伯爵納入了保護。他在沃威克的莊園裡與法蘭西斯·洛維爾（即後來的洛維爾子爵）成為至交，並相當穩固。這個家庭裡的另外一個小孩子是沃威克的女兒，[[安妮·內維爾|安妮·內維爾]]，這名女性後來嫁給了理查。

== 愛德華四世統治期間 ==

在愛德華四世治理英格蘭的期間，理查不但展現他的忠誠，也表現出做為一名軍事統帥的才華。他被賜予位於北英格蘭的廣大土地與[[格洛斯特公爵|格洛斯特公爵]]的頭銜，並被指定為北方總督。這一切讓理查成為英格蘭最有權勢的貴族，以及愛德華四世忠心的左右手；諷刺的是，家族裡另一個成功活到成年的哥哥，[[喬治·金雀花，第一代克拉倫斯公爵|喬治·金雀花]]（即後來的第一代[[克雷倫斯公爵|克雷倫斯公爵]]）卻在1478年因為叛國罪而被愛德華四世處死。

理查掌控北英格蘭直到愛德華四世駕崩。在此地，尤其是[[約克市|約克市]]，他享有廣泛民眾的愛戴。他的施政不僅公正且慨然援助大學，對教堂也有相當的貢獻。1482年的時候由蘇格蘭人手中重新奪下特威德河畔的貝里克鎮這件事也對他的名聲貢獻良多。

== 登上王位的經過 ==

愛德華四世駕崩於1483年4月9日，他12歲的兒子愛德華五世與9歲的理查則是第一與第二順位的王位繼承人。但理查三世卻逮捕了愛德華五世的監護人[[安東尼·伍德維爾，第二代瑞佛斯伯爵|安東尼·伍德維爾，第二代瑞佛斯伯爵]]（愛德華四世的妻舅）與其它顧問，將他們送往位於約克郡的龐特福萊克堡，據傳以企圖刺殺愛德華五世的罪名將這些人處死。之後，他將愛德華與他的幼弟理查送進倫敦塔。

1483年6月22日，理查三世的一名代表在聖保羅大教堂外宣讀了一份聲明，首次宣稱：基於愛德華四世與伊莉莎白·伍德維爾的婚姻不合法的緣故，先王的兩個兒子自動失去了王位繼承權。因此，英國皇位的真正繼承人應當是理查，而非愛德華五世。因為某名主教作證，在愛德華四世迎娶[[伊莉莎白·伍德維爾|伊莉莎白·伍德維爾]]之前，他曾為愛德華與[[伊蓮娜·巴特勒|伊蓮娜·巴特勒]]女士證婚；基於這名女子在伊麗莎白·伍德維爾嫁給愛德華四世時依然在世。理查三世這則指稱愛德華四世與伊莉莎白·伍德維爾間的婚姻無效的宣言隨即被議會認同，並訂立了[[王權法案|王權法案]]以正式宣佈兩個小王子失去繼承權，英王頭銜則屬於理查三世。

1483年7月6日，理查三世在[[西敏寺|西敏寺]]加冕成為英格蘭王。

== 關於殺害先王遺孤的傳聞的討論 ==

英國史學家長期以來，針對理查三世殺死先王遺孤（即他十二歲的親侄子愛德華五世與其弟[[什魯斯伯里|理查]]）的討論，各個時代都有人提出疑問或翻案討論。

在文獻上，首次出現理查三世殺侄的傳說是在約克王朝覆滅後廿年後的事，且執行的凶手已經身亡；正式流傳下來的版本，是由[[湯瑪斯·摩爾|托馬斯·莫爾]]所著的《理查三世傳》。論者以為，在[[亨利七世_(英格兰)|亨利·都鐸]]的年代，有必要消滅前朝合法性地位的方式，便是對其施以污名化的手法。目前尚無當代的第一手資料可以證明這件事，但摩爾的著作太過深入人心，以致現在理查三世的簡介中依然背負「篡位者」與「殺侄」的罪名。

關於理查三世殺侄傳說的通俗資料，可以參考小說家[[約瑟芬·鐵伊|約瑟芬·鐵伊]]的作品《[[時間的女兒|時間的女兒]]》。

== [[博斯沃思原野戰役|波斯沃之役]]與駕崩 ==

1485年8月22日，理查與亨利·都鐸麾下的[[蘭開斯特王朝|蘭開斯特]]勢力於波斯沃平原交鋒。即便理查的兵力一開始以二比一的數量佔了優勢，但湯瑪斯·史坦利（即第一任德卑伯爵）、威廉·史坦利爵士以及亨利·柏西（即第四任諾桑伯蘭伯爵）卻陣前倒戈，投向了亨利·都鐸。這嚴重削弱了理查三世的軍力，並對波斯沃之役的結果有著顯著的影響。除此之外，理查三世的相當親近的夥伴─諾弗克公爵約翰·霍華的陣亡也大大地打擊了理查與其部下的士氣。有些記載指出理查在波斯沃之役裡相當驍勇善戰，不但將著名的勇士─約翰·錢尼爵士打下馬來，且殺了亨利·都鐸的掌旗手─威廉·布蘭登爵士。在他被圍攻身亡前還幾乎殺到亨利本人身旁。自古相傳他的遺言是：「叛國、叛國、叛國、叛國、叛國。」

亨利·都鐸之後繼承了理查的王位，成為[[亨利七世_(英格兰)|英王亨利七世]]後，隨即以迎娶約克家族的女繼承人─愛德華四世之女伊莉莎白為手段鞏固了自己的寶座。

== 繼承人 ==

在約克家族於[[蒂克斯伯里战役|圖克斯伯利之戰]]擊敗了蘭開斯特家族，取得了關鍵勝利後，理查便於1472年7月12日與沃威克伯爵的小女兒[[安妮·内维尔|安妮·内维尔]]步入禮堂。安妮的第一任丈夫是亨利六世的兒子[[威斯敏斯特的愛德華|威斯敏斯特的愛德華]]（死於1471年）。

理查與安妮育有一子，名為[[愛德華·金雀花|愛德華·金雀花]]（也被稱為米德漢的愛德華，1473年─1484年4月9日），但這個王子在被立為威爾斯王子後沒多久就過世了。理查也有數個私生子，其中包括：被亨利七世處決的[[格洛斯特的約翰|格洛斯特的約翰]]（1470年-1491年）、以及一個叫做凱瑟琳，後來嫁給[[威廉·赫柏特|威廉·赫柏特]]（第二任潘布魯克伯爵）的女兒（卒於1487年）。這兩人的生母被認為有可能是一個名字曾經出現在家族紀錄上的女性─[[凱瑟琳·霍特|凱瑟琳·霍特]]。這兩個小孩都比理查三世長命，但似乎都沒有留下後代。不過，神秘的理查·金雀花（又稱伊斯維爾的理查）以及一個被稱為「建築大師理查」的人倒是有可能也是理查三世的子嗣。 

在理查三世最後一次對上蘭開斯特勢力時，他是個沒有合法繼承人的鰥夫。在他的兒子過世後，理查三世一開始是將自己的姪兒[[第十七代沃里克伯爵爱德华·金雀花|沃威克伯爵愛德華]]（他的哥哥[[第一代克拉伦斯公爵乔治·金雀花|克雷倫斯公爵喬治]]之子）立為繼承人。但在他的皇后安妮過世後，理查改立了他姐姐─約克的伊莉莎白，蘇福克公爵夫人的兒子，約翰·迪拉波爾（林肯伯爵）為王位繼承人。

== 遺澤 ==

理查三世在波斯沃之役的死最終導致了始於1154年[[亨利二世|亨利二世]]的[[金雀花王朝|金雀花王朝]]的滅亡，這個王朝統治了英格蘭331年。最後一個姓金雀花的男性─[[爱德华·金雀花，第十七代沃里克伯爵|爱德华·金雀花，第十七代沃里克伯爵]]（理查三世的哥哥克雷倫斯公爵之子）則在1499年被亨利七世處死。

理查於1485年所建立的'''北方議會'''極大地改善了北英格蘭的情況，在此之前，該區的平民沒有任何獨立於倫敦外的實質經濟活動。這個議會是後來英國內閣職位'''北方事務部部長'''的前身。

== 遺體與考古發現 ==

[[File:The_King_In_The_Car_Park_-_Page_15_-_Figure_12.png|thumb]]

博斯沃思原野戰役結束後，理查三世赤裸的屍體被遊街示眾，之後被埋葬於[[列斯特郡|列斯特郡]]的聖方濟教堂前。根據記載，一名教士領走他的身體後草草下葬，沒有棺木也沒有裹屍布。傳聞埋葬理查三世的教堂，在1538年英王[[亨利八世|亨利八世]]頒布解散修道院法令時，被夷為平地，此後理查三世的遺體下落不明。有傳聞指出在他埋骨的修院解散後，他的屍體被扔進附近的索爾河。

2012年9月，[[萊斯特大學|萊斯特大學]]的考古研究小組，於英國列斯特郡一個停車場底下，挖掘教堂遺址時，在原為古代教堂的墓園地點，發現一副遺骨。骨質考古學家認為這副遺骨來自一名30歲的男子，有嚴重脊椎側彎。這副骸骨身上有十處傷口，頭骨被刀削去一部份，另一處刀傷砍穿頭顱，為致命傷；脊椎骨間發現一個箭頭穿刺。[[放射性碳定年法|碳-14同位素定年法]]確定死者死亡時間在1455年至1540年之間。其[[DNA|DNA]]比對，與一名理查三世姐姐安妮的第17代後裔，一名移居[[加拿大|加拿大]]的木匠男子伊布森相符。萊斯特大學考古研究小組，根據這些證據，認為它可能是理查的遺體。在理查三世遺骨出土地的列斯特大教堂則表示，將以莊嚴尊重的儀式重新將理查三世下葬在該教堂<ref>[http://www.bbc.co.uk/ukchina/simp/uk_life/2012/09/120912_life_king_richard_bones.shtml 英停车场掘出骸骨或属理查三世]，2012年9月12日[[BBC|BBC]]中文網</ref><ref>[http://www.cna.com.tw/News/aEDU/201302040329-1.aspx 英停車場骸骨證實為理查三世]，2013年2月4日[[中央社|中央社]]</ref><ref>{{Cite web|url=http://news.chinatimes.com/world/50407232/112013020500150.html|language=zh-tw|author=江靜玲|title=停車場骨骸確為英王理查三世|publisher=中國時報|date=2013-02-05|accessdate=2013-02-05|deadurl=yes|archiveurl=https://web.archive.org/web/20130205142055/http://news.chinatimes.com/world/50407232/112013020500150.html|archivedate=2013-02-05}}</ref><ref>[http://www.bbc.co.uk/zhongwen/trad/uk/2015/03/150322_richard_iii_burial 英國理查三世遺骸遷往萊斯特大教堂安葬]，2015年3月22日BBC中文網</ref>。

2013年2月，[[萊斯特大學|萊斯特大學]]的考古研究小組發表DNA比對結果，進一步證實停車場骨骸確為英王理查三世。<ref>[http://news.singtao.ca/vancouver/2013-02-05/headline1360059680d4333002.html 英「暴君」葬身停車場DNA驗屍證身分]</ref>

2015年3月26日，骸骨出土後，理查三世出身地約克和埋骨處萊斯特為爭取成為其安息地而打官司，最後判決萊斯特勝訴。 
理查三世的骸骨放進後人伊布森為他打造的棺木裡，由馬車拉著靈柩從萊斯特大學出發，前往他當年戰死的沙場，民眾穿上古代兵服和鎧甲並鳴放21響禮砲致意，沿途群眾帶著象徵他出身地的白玫瑰相送，最後在萊斯特大教堂停靈，棺木上放了他當年王冠復刻版，讓民眾瞻仰。

== 爭議與名聲 ==

之前被視為理查三世相關的許多「事實」都已經被現代史學家所質疑。舉例來說，理查三世在都鐸王朝人士的筆下被描寫成肢障者，這在當時被視為是邪惡人格的象徵。傳聞中理查三世萎縮的手臂、跛腳與駝背，這個說法可能是來自於[[托马斯·莫尔|湯瑪斯·摩爾]]所留下來的存疑史料。這份資料給[[威廉·莎士比亚|莎士比亞]]留下了極深的印象，而且長久以來被當作權威的歷史紀錄。從理查三世遺骸可以發現他確有嚴重的[[脊椎側彎|脊椎側彎]]，但手臂並沒有萎縮。這推翻了過去對理查三世道德方面所做出的指控與及針對他的外貌所做出的毀謗。

「理查三世會」成立於20世紀，自創立起便收集了許多以理查三世生平與統治為主題的研究資料。這個社團的目標可以由其贊助人，也就是當前的理查親王與格洛斯特公爵的一段話總結：「……理查三世會的主旨與力量來自於相信事實比謊言來的更有力量─一個在經過這麼多世紀以後依然重要的信仰。這證明了我們將文明視為一種像名譽一樣深奧又脆弱，但卻又值得為其所戰的事物。」

理查三世會的美國分會對所有在理查三世案件相關的疑犯作出了它們自己的判斷，其內容可以在線上圖書館"Whodunit?"裡可以找到。

為了挽回這個英格蘭王的名譽，「理查三世之友會」同樣在20世紀創立。該社團創立於約克市這個在1485年理查身亡時曾公開宣告「仁慈治理我們的先王理查……被可悲地謀殺，全城致上深刻的哀悼。」。
