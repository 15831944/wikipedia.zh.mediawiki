'''理查德·威廉·海因里希·阿贝格'''（{{bd|1869年|1月9日|1910年|4月3日}}），德国化学家。他所提出的[[阿贝格规则|阿贝格规则]]，即某元素最高化合价和最低化合价之差为8启发了[[吉尔伯特·路易斯|吉尔伯特·路易斯]]的研究，被认为是价键理论的先驱<ref name="CBD">{{cite book|title = [[Chambers_Biographical_Dictionary|Chambers Biographical Dictionary]]|isbn = 0-5501-8022-2|pages = 3}}</ref>。阿贝格是热气球飞行的爱好者，也因热气球事故而早逝。

==生平与学术贡献==
阿贝格生于普鲁士王国的[[但泽|但泽]]（今波兰格坦斯克），弟弟威廉·阿贝格后来曾担任普鲁士内务部国务秘书。从柏林威廉高中毕业后，阿贝格进入[[基尔大学|基尔大学]]和[[图宾根大学|图宾根大学]]学习有机化学，后来在柏林大学教授[[奥古斯特·威廉·冯·霍夫曼|奥古斯特·威廉·冯·霍夫曼]]的指导下进行有机化学研究。1891年7月19日，阿贝格获得[[柏林大学|柏林大学]]博士学位。毕业一年后，他前往[[莱比锡|莱比锡]]大学，在[[弗里德里希·威廉·奥斯特瓦尔德|弗里德里希·威廉·奥斯特瓦尔德]]指导下开始物理化学研究。1894年他在[[哥廷根大学|哥廷根大学]]担任物理化学教授[[沃尔特·能斯特|沃尔特·能斯特]]的助手，后来还曾到[[斯德哥尔摩大学|斯德哥尔摩大学]]协助过[[斯凡特·阿伦尼乌斯|斯凡特·阿伦尼乌斯]]进行物理化学研究。

在物理化学研究中，阿贝格先研究了冰点下降理论，冰的[[介电常数|介电常数]]，水溶液的[[渗透压|渗透压]]等。他发现某些原子喜欢结合成分子存在而某些不喜欢，从而提出了价电子对化学反应的重要作用和全排满的电子层有利于元素稳定。他发现元素最高化合价和最低化合价之差为8即阿贝格规则。他还研究了氧化电势等<ref name=EB-2>{{cite encyclopedia|year = 2010 |title = Abegg, Richard Wilhelm Heinrich|encyclopedia = Encyclopædia Britannica Ultimate Reference Suite |publisher = Encyclopædia Britannica|location = Chicago}}</ref>。1897年他成为[[布雷斯劳大学|布雷斯劳大学]]的化学教授。两年后，阿贝格成为[[弗罗茨瓦夫技术大学|弗罗茨瓦夫技术大学]]的化学系主任。一年后担任化学教授。1909年他成为正教授，和同事 Guido Bodländer一起提出了[[电子亲和能|电子亲和能]]这一无机化学中的基础概念。从1901年他就担任《电化学学报》的编辑，直到1910年在热气球事故中去世。<ref name=JCS />

在求学和研究生涯中，阿贝格也完成了自己的军事义务。1891年他成为预备役军官。1895年他和莱恩·西蒙结婚，西蒙是一位热气球爱好者。1900年阿贝格成为第九轻骑兵团预备队上尉，这一年他第一次乘坐热气球，目的是进行军事侦查。之后热气球旅行成为阿贝格和他的妻子的爱好，他也在之后的气球旅行中拍摄了大量照片，有多项科学发现，但都没有发表。<ref name="JCS">{{cite journal|last1 = Hills|first1 = Walter|title = Richard Abegg|journal = Journal of the Chemical Society, Transactions|year = 1911|volume = 99|pages = 599–602|doi = 10.1039/CT9119900599|url = http://pubs.rsc.org/en/Content/ArticleLanding/1911/CT/CT9119900599|accessdate = 4 September 2010|format = PDF}}</ref>
