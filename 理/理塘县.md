'''理塘县'''（{{bo|z=ལི་ཐང་རྫོང|w=li thang rdzong|p=Litang Zong}}）是[[中国|中国]][[四川省|四川省]][[甘孜藏族自治州|甘孜藏族自治州]]辖县，位于甘孜州西南部，舊名'''理化縣'''，屬[[西康省|西康省]]，在[[雅江縣|雅江縣]]西二百八十里，人口约4.75万（2001年），[[海拔|海拔]]4014米。境內漠番雜處，有藏傳佛教大寺，為川藏間往來要道，土地平闊，人煙稠密，商業稱盛。

理塘诞生过许多[[藏族|藏族]]名人，包括[[格桑嘉措|格桑嘉措]]、[[楚臣嘉措|楚臣嘉措]]、[[外蒙古|外蒙古]]的[[哲布尊丹巴|哲布尊丹巴]]、[[拉卜楞寺|拉卜楞寺]][[第五世嘉木样活佛|第五世嘉木样活佛]]等。

主要寺院有[[长青春科尔寺|长青春科尔寺]]等。理塘作為地名出現在六世[[達賴喇嘛|達賴喇嘛]][[倉央嘉措|倉央嘉措]]的情詩中。

== 沿革<ref>段木干主編.《中外地名大辭典》（1至5冊）.人文出版社,民國70年06月.第3359頁</ref> ==

* 元至元九年（1272年）置[[李唐州|李唐州]]，后设奔不儿亦思刚招讨使司（萬戶府）；元二十五年（1288年）設錢糧總管府。
* 明置里塘宣撫司，後為扎兀東思麻千戶所；明末清初為固始汗屬地；
* 清康熙四十八年（1709年）設正副營官，屬青海岱慶和碩齊部。康熙五十八年（1719年），大兵經裹塘，青海遣<u>達瓦藍占巴</u>據此阻師，誘斬之，附近裏塘之<u>瓦述毛丫</u>，<u>瓦述祟喜</u>各土司，皆呈戶口納糧，後內附[[四川省|四川省]]；雍正七年（1729年）置裏塘正副宣撫司，屬打箭爐廳；乾隆五十七年(1792年)設糧務委員。光緒三十二年（1906年）置[[順化縣|順化縣]]；三十四年(1908年)奏准置[[裏化廳|裏化廳]]，轄稻壩（稻城）、定鄉（鄉城）、順化（理塘）等縣。宣統三年(1911年)升裹化府。
*民國二年（1913年），易'''裏'''為'''理'''，置理化縣，屬川邊特別行政區，十四年（1925年），改屬西康西康道；國民政府成立，廢道，直屬西康省政府。
* 1951年12月14日，更名為理塘縣。<ref>丹珠昂奔 周潤年 莫福山 李雙劍主編.《藏族大辭典》.甘肅人民出版社,2003年02月第1版.第1079頁</ref>

== 地理 ==
理塘县属高原型副极地气候，日夜温差大，日照终年充足。冬寒冷干燥，夏凉而湿润；降雨集中在6月至9月。1月平均气温−5.8℃，7月平均气温10.6℃，年平均气温 3.25℃。

== 行政区划 ==
理塘县辖1个镇、23个乡：
* 镇：高城镇。
* 乡：君坝乡、哈依乡、觉吾乡、莫坝乡、亚火乡、绒坝乡、呷柯乡、奔戈乡、村戈乡、禾尼乡、曲登乡、喇嘛垭乡、章纳乡、上木拉乡、下木拉乡、中木拉乡、濯桑乡、甲洼乡、藏坝乡、格木乡、拉波乡、麦洼乡、德巫乡。
