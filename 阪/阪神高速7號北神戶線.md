'''阪神高速7號北神戶線'''（{{lang|ja|阪神高速7号北神戸線／はんしんこうそく7ごうきたこうべせん}}，{{lang|en|Route 7 Kita-Kobe Line}}）是[[日本|日本]][[阪神高速道路|阪神高速道路]]路線，從[[兵庫縣|兵庫縣]][[神戶市|神戶市]][[西區_(神戶市)|西區]]的[[第二神明道路|第二神明道路]][[伊川谷系統交流道|伊川谷系統交流道]]至兵庫縣[[西宮市|西宮市]][[中國自動車道|中國自動車道]]（[[西宮山口系統交流道|西宮山口系統交流道]]）。{{Fact|若從[[名神高速公路|名神高速公路]]、[[北攝|北攝]]、[[寶塚市|寶塚]]前往[[姫路市|姫路]]、[[德島市|德島]]方向時，行駛本道路是經過[[收費道路|收費道路]]之中最便宜和最快速的路線。另外，若從姫路、德島往神戶市中心方向時，若要避開第二神明道路[[須磨交流道|須磨交流道]]附近與[[阪神高速3號神戶線|3號神戶線]]上的[[交通堵塞|塞車]]時，可以利用本道路改道行駛[[阪神高速31號神戶山手線|31號神戶山手線]]與[[阪神高速32號新神戶隧道|32號新神戶隧道]]。}}

== 路線名 ==
* 兵庫縣道高速北神戶線
* 神戶市道高速道路北神戶線

== 道路規格 ==
* 道路區分 : 第2種第2級
* 設計速度 : 70[[km/h|km/h]]
* [[車道|車道]]數 : 雙向4車道
* 出入口 : C規格、設計速度40km/h
※警察廳修正「高速公路等最高速度規定基準」（{{lang|ja|高速道路等における最高速度規制基準}}）後，自2010年末部分路段的最高速度變更為70km/h、80km/h（請參考後述的「車道與速限」）<ref>[http://www.hanshin-exp.co.jp/drivers/driver/speed/kitakobe.html 阪神高速道路7號北神戶線最高速度限制之再評價] {{webarchive|url=https://web.archive.org/web/20130313142320/http://www.hanshin-exp.co.jp/drivers/driver/speed/kitakobe.html |date=2013-03-13 }}</ref>。

== 歷史 ==
* [[1985年|1985年]][[8月10日|8月10日]] : 伊川谷系統交流道 - 前開出入口間通車
* [[1986年|1986年]][[4月25日|4月25日]] : 前開出入口 - 布施畑西出入口間通車
* [[1990年|1990年]][[7月2日|7月2日]] : 布施畑西出入口 - 箕谷臨時出入口間通車。（箕谷臨時出入口現在改為連接神戶市道長田箕谷線日之峰2丁目十字路口）
* [[1994年|1994年]][[4月20日|4月20日]] : 箕谷出入口通車。與新神戶隧道收費道路連接
* [[1998年|1998年]][[4月2日|4月2日]] : 箕谷出入口 - 有馬口出入口間、有馬口系統交流道 - 柳谷系統交流道間通車
* [[2003年|2003年]][[4月28日|4月28日]] : 有馬口出入口 - 西宮山口系統交流道間通車，同時'''全線通車'''
* [[2010年|2010年]][[12月2日|12月2日]] : 伊川谷系統交流道 - 布施畑西出入口間、箕谷出入口-西宮山口南出入口間、有馬口系統交流道 - 柳谷系統交流道間的最高速度改為70km/h，布施畑西出入口-箕谷出入口間的最高速度提升為80km
* [[2012年|2012年]][[4月2日|4月2日]] : 有馬口、西宮山口南（西行）、西宮山口南（東行）、西宮山口東等4座收費站引進'''自動收費機'''（{{lang|ja|自動収受機}}）並且開始運用<ref>[http://www.hanshin-exp.co.jp/drivers/ryoukin/topics/2012-03-jidousyujyuki.html 7號北神戶線等4收費站引進自動收費機。]</ref>
