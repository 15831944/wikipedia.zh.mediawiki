'''阪神、淡路大震災'''（{{jpn|j=阪神・淡路大震災|hg=はんしん・あわじだいしんさい|rm=Hanshin Awaji dai shinsai|f=y}}），中文多稱為'''阪神大地震'''、'''神戶大地震'''或'''關西大地震'''，是1995年1月17日發生於[[日本|日本]][[關西地方|關西地方]]規模為[[日本气象厅地震规模|日本气象厅地震规模]]7.3的[[地震|地震]]災害，因受災範圍以[[兵庫縣|兵庫縣]]的[[神戶市|神戶市]]、[[淡路島|淡路島]]、以及神戶至[[大阪市|大阪]]間的[[都市|都市]]為主而得名。

== 概要 ==
阪神大地震的發生時間為1995年1月17日上午5時46分52秒（[[日本標準時間|日本標準時間]]，即[[UTC+9|UTC+9]]），[[震央|震央]]在[[淡路島|淡路島]]北部的[[明石海峽|明石海峽]]海域（[[北緯|北緯]]34度35.9分、[[東經|東經]]135度2.1分），[[震源|震源]]深度為16公里，地震規模為[[日本气象厅地震规模|日本气象厅地震规模]]7.3<ref group="注釋">原测得震级为M7.2，日本气象厅改变震级计算方法后，震级修正为M7.3。</ref>（[[矩震级|矩震级]]6.9）。該地震是由淡路島的[[野島斷層|野島斷層]]地殼活動引起，屬於上下震動型的強烈地震。

此次地震是[[日本|日本]]自1923年[[關東大地震|關東大地震]]以來規模最大的[[都市|都市]][[直下型地震|直下型地震]]。由於神戶是日本首屈一指的大城市，人口密集（當時人口約105萬人），地震又在清晨發生，因此造成相當多傷亡。官方統計有6,434人死亡，43,792人受傷，房屋受創而必須住到組合屋的有32萬人。

阪神大地震在日本地震史上具有重要的意義，它直接引起了日本對於地震科學、都市建築防震、交通防震的重視。當時一般日本學者認為關西地區不可能有大地震發生，導致該地區缺乏足夠的防範措施和救災系統{{fact|time=6-3-2011}}；特別是神戶周圍有相當多交通要道都通過隧道或高架橋，在地震時隧道受損嚴重，影響了搜救速度。神戶市區更因[[瓦斯|瓦斯]]外洩、木造房屋密集、引起快速的連鎖性大火；如神戶市[[長田區|長田區]]，全部的木造房屋都付之一炬。

另外，此次地震也对日本政坛造成了一定的冲击。由于时任首相[[村山富市|村山富市]]在应对灾情的处理上反应过于迟缓，导致民意支持率大跌。不久之后社会党便接连于各种选举中败北，而曾经一度失去政权的[[日本自民党|日本自民党]]再度回归政坛核心。

== 命名 ==
在1月17日地震發生當日，[[氣象廳_(日本)|日本氣象廳]]根據命名規定的基準，將'''本次地震'''命名為「'''平成7年（1995年）兵庫縣南部地震'''（The South Hyogo prefecture Earthquake in 1995）」。

各媒体对本次'''地震及所带来的灾害'''采用不同的名称。[[毎日新聞|毎日新聞]]首先使用「'''阪神大震災'''」的名稱，其他的傳媒也跟進使用。另外還有不同的命名，如[[朝日新聞|朝日新聞]]和[[日刊體育|日刊體育]]是「'''關西大震災'''」<ref>1995年（平成7年）1月18日各紙朝刊</ref>，[[讀賣電視台|讀賣電視台]]則称「'''關西大地震'''」。

之後，政府方面認為有必要為此一大規模災害的日後復原工作取一個統一的名稱，因為淡路島地區的受災情況嚴重，在2月14日的內閣會議上口頭上將災害稱為「'''阪神・淡路大震災'''」。2月24日時，制定了一個5年內有效的法令：「'''阪神・淡路大震災復興的基本方針及組織相關法律'''（平成7年法律第12号）」（即日施行）。正式使用「'''阪神・淡路大震災'''」的名稱。

雖然此名稱提到「阪神」和「淡路」，但是沒有提到緊鄰[[神戶市|神戶市]]、且同樣受災嚴重的的[[明石市|明石市]]，導致出現了這個名稱是有包含[[大阪市|大阪]]、神戶兩市之間的區域、亦或是僅指兩市的爭議，而大阪市的受災並不嚴重，也使得該震灾命名的準確度受到批評。

== 災情 ==
當時阪神附近的斷層未被察覺，且該斷層與野島斷層有相連，以至於防震準備不足的[[關西地方|關西地方]]受到強烈傷害。尤其其郊區多為不耐震的木造房屋，因此大部分死傷皆發生在郊區。

* 避難人數（最大時） : 316,678人
* 房屋損害 : 全坏104,906棟、半坏144,274棟、全半坏合計249,180棟（約46萬戶）、部分損坏390,506棟
* 火災損害 : 住家全部燒毀7,036棟、燒毀損壞（非住家、住家）合計7,574棟、受災住戶8,969戶
* 其他損害 : 道路7,245處、橋梁330處、河川774處、山崩347處
* 該損總額 : 約10兆[[日元|日元]]
<ref>死者、住家被害、火災被害、その他被害の数値は消防庁発表「[http://www.fdma.go.jp/bn/阪神・淡路大震災（確定報）.pdf 阪神・淡路大震災について（確定報）]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}」（2006/05/19）より</ref>

== 其他 ==

[[國家地理頻道|國家地理頻道]]災難調查節目《[[重返危機現場|重返危機現場]]》（{{lang-en|''Seconds From Disaster''}}）第二季曾以該起地震製作過一檔節目，並命名為神戶大地震，以神戶當地的受災為製作重點，同時該集並未提及到大阪。

因阪神一帶為[[西日本|西日本]]的工商業重鎮及重要的海港之一，大地震發生後讓亞洲多國股市受影響而下跌。[[英國|英國]][[霸菱銀行|霸菱銀行]]的投資交易員[[尼克·李森|尼克·李森]]（Nick Leeson）操作的[[衍生性金融商品|衍生性金融商品]]投機蒙受巨大損失，且無法補救，使得該銀行損失14億美元而倒閉，最後以1[[英鎊|英鎊]]的象徵價格賣給荷蘭的[[ING集團|ING集團]]，成為該集團成員之一。
