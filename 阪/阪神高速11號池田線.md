'''阪神高速11號池田線'''（{{lang|ja|阪神高速11号池田線、はんしんこうそく11ごういけだせん}}、{{lang|en|Route 11 Ikeda Line}}）是從[[大阪府|大阪府]][[大阪市|大阪市]]的環狀線分歧（土佐堀附近）至[[池田市|池田市]]池田木部出入口的[[阪神高速道路|阪神高速道路]]路線。因為本路線聯絡[[大阪國際機場|大阪國際機場]]與[[名神高速公路|名神高速公路]]、[[中國自動車道|中國自動車道]]相當便利，利用車輛相當多導致經常[[交通堵塞|塞車]]。但是在[[近畿自動車道|近畿自動車道]]、[[阪和自動車道|阪和自動車道]]與[[第二京阪道路|第二京阪道路]]全面通車之後多少影響了交通量，近年有減少的傾向。另外，在[[關西國際機場|關西國際機場]]開港前一直被稱為'''機場線'''（{{lang|ja|空港線}}）。

== 路線名 ==
* （與阪神高速1號環狀線合稱）大阪府道高速大阪池田線
* 兵庫縣道高速大阪池田線

== 歷史 ==
* [[1965年|1965年]][[12月12日|12月12日]] : 1号環狀線[[土佐堀出口|土佐堀]] - 梅田通車。
* [[1967年|1967年]][[8月29日|8月29日]] : 福島 - 豐中北通車。
* 1967年[[10月26日|10月26日]] : 中之島 - 福島通車。
* [[1969年|1969年]][[2月1日|2月1日]] : 豐中北 - 大阪機場通車。開始被稱為'''機場線'''。
* [[1970年|1970年]][[3月14日|3月14日]] : 大阪機場 - 池田通車。
* [[1989年|1989年]] : 導入路線編號制，改稱為'''池田線'''。
* [[1998年|1998年]][[4月2日|4月2日]] : 螢池西町 - 池田木部通車，'''全線通車'''。

八月第三個星期六舉辦的豬名川煙火大會開幕時，為了防止煙火關係所造成的各種事故，木部方面分歧〜池田木部間會暫時禁止通行。

==== 交通堵塞 ====
除了[[星期六|星期六]]、[[星期日|星期日]]、[[日本公眾假期|假日]]之外，南下的塚本出入口、北上的福島出入口幾乎每天都會發生起於這兩處的交通堵塞。特別是福島的交通堵塞（包括周六早上）經常透過環状線波及[[阪神高速15号堺線|15号堺線]]、[[阪神高速14号松原線|14号松原線]]、[[阪神高速13号東大阪線|13号東大阪線]]、[[阪神高速12号守口線|12号守口線]]等其他路線。
