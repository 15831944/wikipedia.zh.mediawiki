'''容积率'''（{{Lang-en|floor area ratio, plot ratio}}），-{zh-cn:香港称'''地积比率''';zh-hk:大陸、台灣稱'''容積率'''; zh-tw:香港稱'''地積比率''';}-，是指总[[建筑面积|建筑面积]]{{efn|建筑面积一般来说大体等于各层楼板的面积总和，但是由于实际建筑总会有架空、悬挑、阳台、斜屋面等各种情况出现，因此实际上建筑面积的计算非常复杂，各国都会有专门的建筑规范对建筑面积的计算进行详细的规定。}}与建筑用地面积的比值。例如，在1万[[平方米|平方米]]的土地上，有3000平方米的建筑总面积，其容积率为0.3。由于[[高层建筑|高层建筑]]的存在，容积率数值可以大于1。

== 容積管制 ==
容积率是[[城市规划|城市规划]]中的一个重要技术指标，它间接反映了单位土地上所承载的各种人为功能的使用量，即[[土地|土地]]的开发强度。[[人口|人口]]众多的地区往往建筑容积率较高。容积率越高表示土地的利用率也就越高，但是对周边的[[城市基础设施|城市基础设施]]的压力也越大，因此不可能无限制的提高容积率，所以世界大部分地区对于[[城市|城市]]的建筑容积率均有管制。

== 獎勵容積 ==
世界很多国家都有容积率奖励制度。若开发者能在其开发土地中为公众提供便利而让出部分利益，例如将底层架空作为公共活动空间，[[政府|政府]]规划部门就会提高該土地的容積率上限，幅度根据法规、[[城市总体规划|城市总体规划]]和项目具体情况而定。这也是政府和民间的良性互动。

== 容積移轉 ==
為了換取土地利用的最大價值，世界上部分地區允許容積率在土地間移轉，稱為容積移轉。容積移轉主要可分為兩種情形：
# 一是原屬一宗土地之可建築容積，一部份或全部移轉至另一宗可建築土地建築使用；
# 二是受限制發展地區（如古蹟）的容積一部或全部移轉至另一宗可建築土地建築使用。

目前台灣都市地區已經實施容積移轉制度，但數筆土地之間的容積移轉須在同一計畫區內方可為之。[[香港|香港]]也實施相似制度，以保護具有特別價值的地點及[[建築物|建築物]]。
