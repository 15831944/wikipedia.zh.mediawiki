'''容肇祖'''（{{bd|1897年|12月1日|1994年|1月23日|}}），字元胎，[[广东|广东]][[东莞|东莞]]人氏，[[容庚|容庚]]的三弟，近现代著名的古典哲学研究家、民俗学家和历史学家。<ref name="齐鲁出版">{{cite web |url=http://www.qlss.com.cn/News/5/20/NewsDetail_164_1.html|title=容肇祖学记及其与齐鲁出版之缘|accessdate=|work=|publisher=齐鲁书社}}</ref>生前任中国社会科学院哲学研究所研究员，硕士生导师，中国社会科学院研究生院兼职教授，国务院古籍整理出版小组顾问，中国民俗学会副会长，中国哲学史学会顾问。<ref>{{cite web |url=http://www.philosophy.ac.cn/wsgk/ygxz/201507/t20150710_2588715.shtml|title=容肇祖|accessdate=2015-07-10|work=|publisher=中国社会科学院哲学研究所}}</ref>。他的著作《明代思想史》被学界誉为“里程碑式著作，断代哲学史的典范”。<ref>{{cite web |url=http://dg99.mzj.dg.gov.cn/business/htmlfiles/dgzzyd/s39008/201310/692175.htm|title=影响中国的东莞人：著名哲学史家容肇祖|accessdate=2013-10-30|work=|publisher=东莞市民政局}}</ref>


==家世==

祖父容鹤龄为同治二年进士，不愿做官，掌教东莞龙溪书院数十年。父亲容作恭，是[[广雅书院|广雅书院]]学生，[[清朝|清朝]]的拔贡。<ref name="齐鲁出版" />

==经历==
容肇祖于1897年出生在广东省东莞县。他于1917年毕业于东莞中学，次年考进[[广东高等师范学校|广东高等师范学校]]英语部学习，1921年毕业。1922年，他考入北京大学哲学系，先是沉醉于学习新兴的心理学，后受[[胡适|胡适]]的影响，逐渐转向思想史研究。其时胡适开设的“中国哲学史”课程颇具影响，[[傅斯年|傅斯年]]、[[顾颉刚|顾颉刚]]以其系统而有条理，甚为认可。<ref name="研究传承">{{cite web |url=http://www.cssn.cn/zgs/zgs_sb/201509/t20150930_2481910.shtml|title=思想史研究传承：胡适与容肇祖|accessdate=2015-09-30|work=|publisher=中国社会科学报}}</ref>1925年参加北大民俗调查会组织的对妙峰山进香者的调查。这是国内第一次有目的、有计划、有组织的民俗调查，故称为开中国民俗运动之先河。
    
1926年他于北大毕业后，任厦门大学国文系讲师兼国学研究院编辑。

1927年2月回广州，当时[[鲁迅|鲁迅]]任中山大学教务长，在厦大时容方认识鲁迅，并在魏晋思想研究方面得鲁迅之惠。不久，由[[林语堂|林语堂]]写信给鲁迅，[[顾颉刚|顾颉刚]]也写信给中大文学院长[[傅斯年|傅斯年]]，容肇祖得以到中山大学预科教国文兼文学院哲学系的中国哲学史。鲁迅赠容《汉代文学史》讲义。随后容先生任预科国文教授兼国文科主任，并与任中山大学历史系教授兼系主任的[[顾颉刚|顾颉刚]]以及[[钟敬文|钟敬文]]等人发起成立了中山大学民俗学会。顾颉刚在中大主持创办了三种杂志：<ref name="齐鲁出版" />
* 《中山大学语言历史学研究所周刊》
* 《民间文艺周刊》
* 《中山大学图书馆周刊》，
钟敬文与容肇祖主要负责《民间文艺》（后改为《民俗》周刊），而以容肇祖负责的期数最多，并推动《民俗》进入其繁盛期。

1928年任民俗学会第一任主席，并主编《民俗周刊》、出版民俗丛书。1930年至1933年，他先后任中山大学、岭南大学副教授。1934年任辅仁大学国文、历史、音学副教授兼北京大学哲学系讲师。1937年到1945年，容肇祖先后任教于北大、清华、南开三所大学组成的长沙临时大学、昆明西南联合大学及[[香港中文大学|香港中文大学]]、[[岭南大学|岭南大学]]、中山大学。1946年，容肇祖三进北京大学，任哲学系教授。

1952年，他被调到北京市文教委员会文物组任研究员，从事古文物整理、鉴定等工作。1956年，容肇祖到中国科学院哲学社会科学部哲学研究所任研究员。

[[文革|文革]]期间，容被“抄家三次”，并先后被下放到河南息县和明港等地劳动。以后虽于1973年调回北京，但他几十年所积累的资料及“专著和散文，特别是未付印的手稿（《李贽传，附李贽著作考》和《清代思想史》等），丧失殆尽”。<ref name="齐鲁出版" />于1986年12月离休。1994年在北京病逝，享年97岁。
