'''荒子觀音'''（日語：あらこかんのん）是位於日本[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[中川區|中川區]]荒子町的[[天台宗|天台宗]][[佛寺|佛寺]]。

也稱為「'''荒子觀音寺'''」，正式名稱為「'''淨海山圓龍（円竜）院觀音寺'''」，也常被親切稱為「'''荒子的觀音桑（荒子の観音さん）'''」。屬於[[尾張四觀音|尾張四觀音]]之一。祭祀本尊為[[聖觀音|聖觀音]]（33年一開扉的秘佛）。

== 沿革 ==
根據寺傳，在[[天平_(聖武天皇)|天平]]元年（[[729年|729年]]），由泰澄草創，在天平13年（[[741年|741年]]），泰澄的弟子僧人自性整頓堂宇。泰澄是加賀（[[石川縣|石川縣]]）白山的開祖，是傳說人物，以上這些草創傳承的事蹟不確定包含了多少史實。

創建當時是在中川區高畑町（現在的西側），經過幾度興衰而到現在的場所。

* [[天文_(後奈良天皇)|天文]]5年（[[1536年|1536年]]）：多寶塔再建（名古屋市内現存最老的建築物・國家[[重要文化財|重要文化財]]）
* [[天正_(日本)|天正]]4年（[[1576年|1576年]]）：[[前田利家|前田利家]]重建本堂。同時也捐贈奉納自己的盔甲。

: 利家出生於荒子的土豪之家，據說在他前往封地[[北陸地方|北陸]]之前，在佛寺附近建構了荒子城。也就是說該寺是他的[[菩提寺_(日本)|菩提寺]]。

* [[延寶|延寶]][[貞享|貞享]]年間　：[[圆空|圆空]]曾數度拜訪該寺，留下了山門的[[金剛力士|仁王像]]和超過1,200尊的木雕佛像（圓空佛）。
* 大正5年（1926年）：現在的山門完工<ref>[http://www.city.nagoya.jp/jutakutoshi/page/0000038216.html 名古屋市ホームページ（都市景観重要建築物指定物件・観音寺）]</ref>。

== 荒子觀音寺的圓空佛 ==
關於圓空佛的研究始於荒子觀音寺，一般認為是在荒子観音寺結束，因為荒子觀音寺擁有各式各樣的圓空佛。荒子観音寺在[[2014年|2014年]]時盤點，現存約有1255尊的圓空佛，若包含被移出的11尊，確認共有1266尊{{sfn|小島梯次|2014|p=51}}。2014年時的盤點，確認日本全國現存的圓空佛有5374尊，實際上有4分之1以上屬於荒子觀音寺{{sfn|小島梯次|2014|p=123}}。根據荒子觀音寺十八世・金精法印的『淨海雜記』紀載，圓空在延寶4年（[[1676年|1676年]]）曾滯留於荒子觀音寺，從樣式層面來看，荒子觀音寺的圓空佛是延寶4年前後的造像{{sfn|小島梯次|2014|p=51}}。

荒子觀音寺的本堂安放有[[釋迦牟尼佛|釋迦牟尼佛]]、大黒天二像，山門則安放有仁王像二尊，仁王像以最大的圓空佛著名{{sfn|小島梯次|2014|p=51}}。1979年，長谷川公茂發表了關於荒子觀音寺的仁王像是圓空在水中漂浮雕刻的推論<ref>長谷川公茂「編年口絵写真解説」『円空研究・別巻2』</ref>。從技術面被棚田家司山田匠琳<ref>棚田家司・山田匠琳「技法から知る円空」『美術手帖』2013年</ref>、小島梯次等人否定{{sfn|小島梯次|2014|p=52}}。從圓空之後進行同樣造像活動的僧人木喰來看，[[柳宗悅|柳宗悅]]以[[宮崎縣|宮崎縣]][[西都市|西都市]]認為木喰五智館木喰佛，也是將佛像漂浮在水面上雕刻，立松和平認為木喰和圓空於荒子観音寺水上雕像的方式一樣進行造像。另一方面，小島梯次認為沒有關於圓空在荒子觀音寺水上雕像的記録、傳承，而否定了這樣的說法{{sfn|小島梯次|2014|p=53}}。

除此之外的圓空佛集中在客殿，其中三尊護法神以圓空的代表作而知名，也被加上圓空的[[和歌|和歌]]{{sfn|小島梯次|2014|pp=53-54}}。

另外，以不整齊的零碎板切雕成的目、鼻、口，被稱為「木端佛（日語：こっぱぶつ）」的群像共31尊。[[1972年|1972年]]（昭和47年），在境内六角堂的木箱中發現了1024尊的圓空佛{{sfn|小島梯次|2014|p=55}}，包含了千面菩薩諸像，在平成18年的調査有千面菩薩1005尊，另有混雜發黑的像，確認有4尊被移座{{sfn|小島梯次|2014|p=55}}。根據『淨海雜記』紀載，圓空是先以[[日本扁柏|檜木]]的大木雕刻仁王像，再用剩餘材料雕刻其他佛像，接著用平木材和零碎的木材製作木端佛和千面菩薩{{sfn|小島梯次|2014|p=57}}，逗子中除了佛像也被放入了木屑{{sfn|小島梯次|2014|p=57}}。

荒子觀音寺的圓空佛在每月第2個星期六下午1時至4時會公開，圓空佛的公開日也同時會舉辦雕刻圓空佛的體驗教室<ref>[http://www.geocities.jp/kotupa_enku/] - 円空仏彫刻・木端の会 </ref>。
