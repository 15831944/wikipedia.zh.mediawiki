'''荒川弘'''（{{lang-ja|あらかわ ひろむ}}，{{bd|1973年|5月8日|||Hiromu, Arakawa}}<ref>《創》2006年6月號36頁「26歲出道（{{lang|ja|26歳くらいでデビュー}}）」的發言。</ref><ref name="amg">{{Cite web |url=http://www.amgakuin.co.jp/contents/?p=1190&author=4&cat=8 |script-title= ja:荒川弘 マンガ家 - プロからのメッセージ |work={{lang|ja|アミューズメントメディア総合学院　-　東京校}} |accessdate=2010年6月24日 }}</ref>）是[[日本|日本]]的[[女性|女性]]<ref>{{Cite web |url=http://www.asahi.com/culture/entertainment/dvd/news/TKY200507250335.html |script-title=ja:原作者・荒川弘に聞く　映画「鋼の錬金術師」 |work=Asahi |date=2005年7月25日 |accessdate=2010年6月24日 |language=ja}}</ref>[[漫画家|漫画家]]，出身於[[北海道|北海道]]<ref name="amg" />。

本名為'''荒川弘美'''，筆名荒川弘是直接去掉本名“美”字而來<ref name="GS">{{Cite web |url=http://www.gangansearch.com/kikaku/interview/arakawa.php |script-title=ja:作家インタビュー > 荒川弘先生 |work=GanganSearch |accessdate=2014年9月8日 |title={title} |archive-url=https://web.archive.org/web/20140222221701/http://www.gangansearch.com/kikaku/interview/arakawa.php |archive-date=2014-02-22 |dead-url=yes }}</ref>。另有一猜測為，使用筆名「荒川弘」的原因是編輯認為用女性名字的漫畫家畫[[少年漫畫|少年漫畫]]會不及[[男性|男性]]名字的漫畫家受歡迎，故去掉了“美”字，取一個比較中性的筆名。

== 經歷 ==
荒川弘出生在北海道[[幕別町|幕別町]]一帶的[[酪農|酪農]]家庭，為家中五姐弟的老四。[[幼稚園|幼稚園]]時期，荒川弘受到[[田河水泡|田河水泡]]漫畫《{{lang|ja|のらくろ}}》影響，開始對漫畫產生興趣<ref name="Itmedia">{{Cite web |url=http://ebook.itmedia.co.jp/ebook/articles/1202/24/news041.html |script-title=ja:荒川弘の『百姓貴族』からほとばしるフロンティアスピリット |work=It Media |author=西尾泰三 |date=2012年2月24日 |accessdate=2015年6月22日}}</ref>；而小學期間，[[高橋留美子|高橋留美子]]漫畫《[[福星小子|福星小子]]》的美形畫風也讓她特別印象深刻。<ref name="Itmedia"/>

在青春期間荒川弘就讀於[[農業|農業]]相關的[[北海道帶廣農業高等學校|北海道帶廣農業高等學校]]，於高校期間裡荒川弘參加校內[[空手道|空手道]]社團、並有獲得空手道黑帶的資格。<ref>《[[Animedia|Animedia]]》2009年5月號</ref>高校畢業後，荒川弘未繼續升學，而是回到老家幫忙家中酪農事務，並在此期間抽空參加漫畫相關的投稿活動，如：以[[筆名|筆名]]{{nihongo|「艾德蒙荒川」|エドモンド荒川}}在[[新聲社|新聲社]]的《[[GAMEST|GAMEST]]》和《[[Comic_GAMEST|Comic GAMEST]]》發表插圖或[[四格漫畫|四格漫畫]]，1998年則以筆名「稱霸全國」（ZENKOKUSEIHA）創作一篇以《[[新世紀福音戰士|新世紀福音戰士]]》為題材的[[同人誌|同人誌]]《EVA DX》。<ref>{{Cite web |url=http://mandarake.co.jp/information/2013/04/20/21cmp101/index.html |script-title=ja:5F ”コンプレックス5周年” 荒川弘先生のエヴァ…襲来 |work=Mandarake |accessdate=2015年6月22日}}</ref>

1999年，荒川弘以[[獸人|獸人]]為題材的奇幻短篇漫畫《STRAY DOG》投稿獲得第9屆[[艾尼克斯|艾尼克斯]]21世紀漫畫大獎後，<ref name="GS"/>離開家鄉前往[[東京|東京]]找尋成為漫畫家的機會。來到東京的荒川弘起初過著一邊打著像是保全、便當工廠的雜工，另外一邊投稿的生活。<ref>{{cite book |title=百姓貴族3 |date=2014年7月10日 |publisher=[[東立出版社|東立出版社]] |author=荒川弘 |isbn=978-986-348-519-3 |page=67頁 }}</ref>在此期間曾在創作《[[咕嚕咕嚕魔法陣|咕嚕咕嚕魔法陣]]》的漫畫家[[衛藤浩幸|衛藤浩幸]]下擔任助手，<ref name="amg"/>並有在2000年發表一篇以幻想的[[上海市|上海市]]為背景、打擊邪惡妖怪為題材的短篇《上海妖魔鬼怪》。

2001年夏季，荒川弘在雜誌《[[月刊少年GANGAN|月刊少年GANGAN]]》上發表奇幻少年漫畫《[[鋼之鍊金術師|鋼之鍊金術師]]》後開始獲得關注。此作品在往後9年連載完畢前期間曾改編成[[電視動畫|電視動畫]]、[[OVA|OVA]]，並讓荒川弘多次以此拿下各項大獎。<ref>{{Cite web |url=http://www.oricon.co.jp/prof/artist/329494/ |script-title=ja:荒川弘のプロフィール |work=[[Oricon|Oricon]] |accessdate=2014年9月8日}}</ref>

在《鋼之鍊金術師》完結後，荒川弘於2011年在《[[週刊少年Sunday|週刊少年Sunday]]》上推出酪農題材漫畫《[[銀之匙_Silver_Spoon|銀之匙 Silver Spoon]]》。另外也有與同樣喜愛[[三國時代|三國時代]]歷史、曾共事過的助手[[杜康潤|杜康潤]]共同合著以該時期為題材的四格漫畫《三国志魂》；<ref>{{Cite web |url=http://www.gamer.ne.jp/news/201202140016/ |script-title=ja:漫画家・荒川弘氏が描く三国志4コマ漫画「三国志魂」上・下巻が3月に発売決定 |work=Gamer |date=2012年2月14日 |accessdate=2014年9月8日 |language=ja}}</ref>或是以自身過去從事的酪農工作經驗、於家鄉發生之實際趣事的隨筆漫畫《[[百姓貴族|百姓貴族]]》等創作發表。<ref>{{Cite web |url=http://books.rakuten.co.jp/event/book/interview/arakawa-h/ |script-title=ja:著者インタビュー > 荒川弘先生 |work=Rakuten |accessdate=2014年9月8日}}</ref>

== 其它 ==
個人自畫像為戴著眼鏡的乳牛，採用契機為因出生自酪農家庭、以及本身為[[金牛座|金牛座]]的因素。<ref>{{cite journal |year=2006年 |journal=創 |volume=6 |pages=35頁}}</ref><ref>{{cite journal |year=2004年 |journal={{lang|ja|ぱふ}} |volume=9 |pages=11頁}}</ref>由于其自画像为乳牛，而且是岁数较大的女性，所以也被部分人称其为「牛姨」或「牛媽」。<ref>{{cite web|title=牛姨“人体炼成”成功！漫画家荒川弘40岁诞下第三子 178动漫频道|url=http://acg.178.com/201402/186193633572.html|access-date=2016-03-01|archive-url=https://web.archive.org/web/20160307094510/http://acg.178.com/201402/186193633572.html|archive-date=2016-03-07|dead-url=yes}}</ref>

為已婚人士，依序在2007、2011及2014年共生下三名孩子。<ref>{{Cite web|url=http://www.rbbtoday.com/article/2014/02/12/116793.html |script-title=ja:『鋼の錬金術師』作者・荒川弘が第3子出産報告……“超人ぶり”に驚きの声|work=RbbToday |date=2014年2月12日 |accessdate=2014年9月8日}}</ref><ref>{{Cite web|url=http://websunday.net/backstage/arakawa/|script-title=ja:まんが家BACKSTAGE 荒川弘 Vol.7|publisher=[[周刊少年Sunday|周刊少年Sunday]] |date=2014年2月14日 |accessdate=2014年9月8日}}</ref>

在故鄉的家訓為{{nihongo|「不幹活的人沒飯吃」|働かざるもの食うべからず}}。

個人最喜歡的地方及顏色分別為山與天空藍。喜歡的國家是[[日本|日本]]、其次是[[中國|中國]]。

尊敬的漫畫家為[[昭和|昭和]]時期的[[田河水泡|田河水泡]]，<ref name="GS"/>另外也是[[內藤泰弘|內藤泰弘]]的漫畫《[[槍神Trigun|槍神Trigun]]》的粉絲，曾在槍神劇場版上映期間寄插圖加油打氣。在喜愛作家方面有[[山田章博|山田章博]]<ref name="GS"/>以及小說《[[奇諾之旅|奇諾之旅]]》，而她本人承認自己沒有看過任何一本被認為是喜歡[[奇幻|奇幻作品]]的人必看的小說，例如《[[魔戒|魔戒]]》、《[[哈利波特|哈利波特]]》等。
