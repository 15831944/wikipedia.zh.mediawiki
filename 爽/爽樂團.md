'''爽樂團'''（英文：ShuangBand），是一個臺灣樂團，成立於2011年底。該樂團前身為「台灣爽樂團」，於2018年2月28日改為現名， 成員為主唱紅中、團長/鍵盤手小樂、[[貝斯|貝斯]]手禮義、[[吉他|吉他]]手維軒<ref>{{cite web |author1=何宗翰 |title=海祭專輯之二》爽樂團呼叫夢想 過去鳥事《都只是過程》 |url=http://news.ltn.com.tw/news/culture/paper/1216583 |website=自由電子報 |accessdate=2018-12-25}}</ref>。該樂團三人都是中壢人，另外一人也在當地生活多年，曾參加多場比賽，亦曾獲得一些獎項<ref>{{cite web |title=爽樂團發片怕沒人聽 CD拿去當杯墊吧 |url=https://tw.news.yahoo.com/%E7%88%BD%E6%A8%82%E5%9C%98%E7%99%BC%E7%89%87%E6%80%95%E6%B2%92%E4%BA%BA%E8%81%BD-cd%E6%8B%BF%E5%8E%BB%E7%95%B6%E6%9D%AF%E5%A2%8A%E5%90%A7-065724882.html |website=Yahoo奇摩 |accessdate=2018-12-25}}</ref>。
