《'''毕业制作'''》（{{Jpn|j=卒業制作|rm=Sotsugyo Seisaku}}）是[[日本|日本]][[摇滚乐队|摇滚乐队]][[SHISHAMO|SHISHAMO]]在高校时期自主制作的[[原創專輯|原创专辑]]，于2013年1月23日首次发行，后又在2013年11月13日新专辑《[[SHISHAMO_(专辑)|SHISHAMO]]》发行同日再次发行。

== 简介 ==
2010年在{{Link-ja|川崎综合科学高等学校|川崎総合科学高等学校}}轻音部结成的SHISHAMO，在2012年10月限量推出了第一张实体化单曲唱片《[[作业没做完|作业没做完]]》，因专业乐评人的赞赏而逐渐获得了关注度和知名度。此后在2012年12月宣布将要发行本张专辑的消息。这张专辑是SHISHAMO的第二张唱片，首次在日本全国范围发行<ref name=":0">{{Cite web|url=https://www.musicman-net.com/artist/21196|title=現役女子高生バンドSHISHAMO、初アルバム「卒業制作」1/23発売|accessdate=2018-07-12|author=|date=2012-12-26|work=Musicman-NET|publisher=|language=ja}}</ref>。

这张专辑发售时值SHISHAMO的三名成员即将毕业之时，因此这张专辑也被作为高校生活的总结<ref name=":0" />，收录了共8首歌曲。专辑发售后SHISHAMO举办了以专辑发售为主题的自身首次全国性的巡回，在日本各地一共进行11场的演唱会<ref>{{Cite web|url=https://rockinon.com/news/detail/77551|title=現役女子高生バンドSHISHAMO、『卒業制作』リリース日に卒業旅行（全国ツアー）を発表|accessdate=2018-07-12|author=|date=2013-01-24|last=|work=rockin'on|publisher=|language=ja}}</ref>，SHISHAMO自己称之为“毕业旅行”。

== 发行 ==
《毕业制作》于2013年1月23日首次发行。在出道专辑《[[SHISHAMO_(专辑)|SHISHAMO]]》发行的当天（2013年11月13日）再次发行<ref>{{Cite web|url=https://artist.cdjournal.com/d/-/4113090361|title=SHISHAMO / 卒業制作 [再発]|accessdate=2018-07-12|author=|date=|work=CDJournal|publisher=|language=ja}}</ref>。
* 首次发行（商品编号：DGRC-004）
* 再次发行（商品编号：XQFQ-1400）
**再次发行时增加了歌词册的内容，除了乐队三人的一些趣事外，还附赠了《做点心》这首歌的乐谱。
