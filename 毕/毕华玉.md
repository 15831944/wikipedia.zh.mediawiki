'''毕华玉'''（{{bd|1960年||2013年|3月26日}}），[[云南省|云南省]][[昆明市|昆明市]][[石林彝族自治县|石林彝族自治县]][[西街口镇|西街口镇]]人，著名[[毕摩|毕摩]]，《[[阿诗玛|阿诗玛]]》的[[国家级非物质文化遗产|国家级非物质文化遗产]]项目传承人，曾参编《彝汉词典》。

==生平==

毕华玉出身于毕摩世家，为第六代毕摩<ref>{{cite web |url=http://www.chinesefolklore.org.cn/web/index.php?NewsID=11255 |title=毕华玉：一位毕摩的使命 |author=张雁群 |date=2007-12-02 |website=www.chinesefolklore.org.cn |publisher=中国民俗学会 |access-date=2018-04-19 |quote=}}</ref>。34岁时进入石林彝族自治县民宗局文史研究室，从事彝族撒尼彝文典籍的收集整理工作，参与石林彝文古籍丛书的整理出版、彝族文字教授等工作，先后参与整理、出版了《祭祀》、《密枝祭词》、《地明古歌》等5套彝文古籍<ref>{{cite web |url=http://www.ynich.cn/view-ml-13110-806.html |title=阿诗玛——毕华玉 |author= |date=2010-03-26 |website=www.ynich.cn |publisher=云南省非物质文化遗产保护中心 |access-date=2018-04-19 |quote=}}</ref>。毕华玉通晓彝文和撒尼人习俗，能主持《尼姆》、《占卜》、《祭祀》、《叫魂》等60多种民俗礼仪<ref name="黄平山"></ref>。2007年6月5日，经[[中华人民共和国文化部|中华人民共和国文化部]]确定，毕华玉作为毕摩《阿诗玛》代表与作为民间《阿诗玛》代表的[[王玉芳|王玉芳]]一同列入《阿诗玛》的[[国家级非物质文化遗产|国家级非物质文化遗产]]项目传承人名单<ref name="黄平山">{{cite web |url=http://www.yizuren.com/people/yryj/hn/21973.html |title=沉痛悼念毕华玉大师 |author=黄平山 |date=2013-03-29 |website=www.yizuren.com |publisher=彝族人网 |access-date=2018-04-19 |quote=}}</ref>。与王玉芳善于演唱《阿诗玛》不同的是，毕华玉注重彝族传统文化的研究和整理<ref>{{cite web |url=http://cn.globalgeopark.org/gyzt/one/shilin/xw/4965.htm |title=石林阿诗玛传习馆7月亮相 |author=贾薇 |date=2008-06-17 |website=cn.globalgeopark.org |publisher=[[中国地质科学院|中国地质科学院]] |access-date=2018-04-19 |quote=}}</ref>。2013年3月26日，毕华玉在石林逝世，享年60岁<ref name="黄平山"></ref>。
