'''毕节市'''是[[中华人民共和国|中华人民共和国]][[贵州省|贵州省]]下辖的[[地级市|地级市]]，位于贵州省西北部，[[乌江|乌江]]上游，[[黔|黔]]、[[滇|滇]]、[[川|川]]三省交界[[乌蒙山|乌蒙山]]区。东与[[遵义|遵义]]、[[贵阳|贵阳]]两市相接，南连[[安顺|安顺]]、[[六盘水|六盘水]]两市，西南与[[云南省|云南省]][[曲靖市|曲靖市]]接壤，北与云南省[[昭通市|昭通市]]及[[四川省|四川省]][[泸州市|泸州市]]相邻。全市总面积26,849平方公里，人口660.61万，是贵州省人口最多的[[地级行政区|地级行政区]]。

== 名称典故 ==
“毕节”是[[彝语|彝语]]“比跻”的谐音，乃[[彝族|彝族]]部族名。后来亦有不同的说法。清爱必达《黔南识略》卷二十六：毕节县“近城一水俨若壕堑，昔人谓之毕川，邑之得名以此。或云，元初设驿，时逢除夕，故名曰毕节。”清光绪《毕节县志》卷二：“毕节筑城以十二月晦日始成，因名毕节。”

== 历史沿革 ==
[[周朝|周朝]]时为[[蜀国|蜀国]]东南境，[[秦朝|秦朝]]置[[巴郡|巴郡]]。[[汉朝|汉朝]]属[[益州|益州]]之[[牂牁|牂牁]]、[[犍为|犍为]]两郡所辖，[[蜀汉|蜀汉]]分属牂牁、[[朱提郡|朱提郡]]。[[西晋|西晋]]分益州置[[宁州|宁州]]，毕节市分属牂牁、朱提、建宁三郡。[[南北朝|南北朝]]之宋、齐、梁均为宁州所辖。[[隋朝|隋朝]]为[[爨氏|爨氏]]之卢鹿部。[[唐朝|唐朝]]先属[[昆明国|昆明国]]，后为[[黔州都督府|黔州都督府]]属地。[[五代|五代]]时分属[[郝州|郝州]]、[[晖州|晖州]]、[[龚州|龚州]]、[[宝州|宝州]]、[[禄州|禄州]]。

[[宋朝|宋朝]]时，毕节东部属罗氏鬼国、乌撒、毗那等部。[[元朝|元朝]]为亦溪不薛宣慰司、乌撒乌蒙宣慰司所治。[[明朝|明朝]]分属水西宣慰司、乌撒军民府、[[永宁宣抚司|永宁宣抚司]]和乌撒、毕节、赤水、永宁四卫。

[[清|清]][[康熙|康熙]]五年（1666年）置大定（今大方）、黔西、平远（今织金）三府，改四川乌撒土府为[[威宁府|威宁府]]；二十二年（1683年）降黔西、平远二府为州，隶属[[大定府|大定府]]；二十六年（1687年）降大定府为州，隶属威宁府，领大定、平远、威宁三州和毕节、永宁二县。[[雍正|雍正]]七年（1729年）升大定州为府领黔西、平远、威宁三州和毕节县、水城厅。

[[中华民国|中华民国]]成立后，境内所有县划归[[贵西道|贵西道]]管辖；民国24年（1935年），贵州省政府设毕节第四行政督察专员公署，辖毕节、[[大定|大定]]、[[黔西|黔西]]、[[威宁|威宁]]、[[水城县|水城县]]；民国30年（1941年）在原建制析设[[纳雍|纳雍]]、[[金沙縣|金沙]]二县，织金县改隶第四行政督察区；民国31年（1942年）析威宁县东北部置[[赫章县|赫章县]]，至此贵州第四行政督察区共辖毕节、大定、黔西、金沙、织金、纳雍、水城、威宁、赫章9县。

[[中华人民共和国|中华人民共和国]]成立后，成立贵州省人民政府毕节区行政督察专员公署，辖民国末第四行政督察区所属各县。1955年，威宁县改建为[[威宁彝族回族苗族自治县|威宁彝族回族苗族自治县]]。1958年，[[大定县|大定县]]更名为[[大方县|大方县]]。1970年，[[水城县|水城县]]并入[[水城特区|水城特区]]，划归[[六盘水地区|六盘水地区]]管辖，同年毕节专区更名[[毕节地区|毕节地区]]，辖毕节、大方、黔西、金沙、织金、纳雍、赫章县和威宁彝族回族苗族自治县。

1994年，毕节撤县建县级市。2007年，成立贵州[[百里杜鹃风景名胜区|百里杜鹃风景名胜区]]，为正县级派出机构。

2011年10月撤销毕节地区和县级毕节市，设立地级毕节市，原县级毕节市改为[[七星关区|七星关区]]<ref>[http://cpc.people.com.cn/GB/64093/64387/16226005.html 国务院批准毕节撤地设市]，中国共产党新闻网，2011年11月12日。</ref>。

2012年,毕节发生[[2012年贵州毕节儿童死亡事件|留守儿童死亡事件]]。2015年，毕节再次发生[[2015年贵州毕节儿童死亡事件|留守儿童死亡事件]]。

== 地理 ==
毕节市地处贵州省西北部，东部和南部与本省[[遵义市|遵义市]]、[[贵阳市|贵阳市]]、[[安顺市|安顺市]]、[[六盘水市|六盘水市]]接壤，西部和北部与[[云南省|云南省]][[昭通市|昭通市]]、[[四川省|四川省]][[瀘州市|瀘州市]]毗邻。位于川、滇、黔三省结合部，东经105°36′——106°43′，北纬26°21′——27°46′之间。境内大部属[[乌蒙山|乌蒙山]]区，以[[喀斯特地形|喀斯特地形]]和高山丘陵为主。[[乌江|乌江]]、[[北盘江|北盘江]]、[[赤水河|赤水河]]皆发源于此；还有[[六冲河|六冲河]]、[[二道河|二道河]]、[[白水河|白水河]]、[[牛栏江|牛栏江]]、[[可渡河|可渡河]]、[[白甫河|白甫河]]（白布河）等河流。平均海拔1600米，最高点为赫章县与威宁县交界的小韭菜坪，海拔2900.6米。

==经济==
===毕节试验区===
毕节是贵州省最贫困落后的地区之一，直到1987年，全区人均工农业总产值仅为288.9元，农民人均收入只184元，人均粮食不足200公斤，未解决温饱人口达300余万，农村贫困人口占其总人口的53.68%，人口自增率21.29‰，森林覆盖率仅为8.53%。1988年6月，时任贵州省委书记的[[胡锦涛|胡锦涛]]亲自倡导创建毕节“开发扶贫、生态建设”试验区。着手解決生态恶化、水土流失、人口膨胀、粮食短缺等问题。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为6536370人<ref>{{cite book |author=毕节地区统计局|title=《毕节地区2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次人口普查|第五次人口普查]]相比，十年共增加208899人，增长了3.3%。年平均递增0.33%。其中，男性人口为3400195人，占52.02%；女性人口为3136175人，占47.98%。总人口性别比（以女性为100）为108.42。0－14岁人口为2029934人，占31.06%；15－64岁人口为4018583人，占61.48%；65岁及以上人口为487853人，占7.46%。城镇的人口为1711222人，占26.18%；乡村人口为4825148人，占73.82%。

2012年底，毕节市户籍人口超过850万，常住人口650万。

=== 民族 ===
全区常住人口中，[[汉族|汉族]]人口为4824015人，占73.80%；各[[少数民族|少数民族]]人口为1712355人，占26.20%。

== 交通 ==
目前主要以{{中国国道名|321}}、{{中国国道名|326}}贯通市内外交通网络。毕节市是全省唯一不通铁路的地级市，已建成的几段铁路均为“擦边”、“过境”性质<ref>[http://www.gzbj.cn/news/bjsyq/2013/0417/content_12083.shtml 毕节铁路建设迎来发展春天] {{Wayback|url=http://www.gzbj.cn/news/bjsyq/2013/0417/content_12083.shtml |date=20130424042757 }}</ref>。兴建中的[[成贵客运专线|成贵客运专线]]将经过毕节市区，预计2019年建成通车。同时在建的还有毕（节）水（城）兴（义）铁路毕节至水城段、昭通－威宁－毕节－金沙－遵义等铁路<ref>[http://news.gog.com.cn/system/2010/12/23/010979615.shtml “三纵三横”铁路规划使毕节通江达海] {{webarchive|url=https://web.archive.org/web/20130502095338/http://news.gog.com.cn/system/2010/12/23/010979615.shtml |date=2013-05-02 }}</ref>。

== 历史名人 ==
*[[奢香|奢香]]（1361—1396）女，彝族，名舍兹，[[四川|四川]][[古蔺|古蔺]]人。明代贵州[[彝族|彝族]]政治家。
*[[李世杰|李世杰]]（1716—1794）字汉三，一字云岩，黔西县人。父[[李植|李植]]，官两淮富安场盐大使。
*[[丁宝桢|丁宝桢]]（1820—1886）字稚璜，织金县人。[[清朝|清朝]]政治人物，[[洋务运动|洋务运动]]重要成员，官至四川总督。
*[[柏格理|柏格理]]（Samuel Pollard，1864—1915）[[英国|英国]][[卫理公会|卫理公会]][[传教士|传教士]]，滇东北老[[苗文|苗文]]的创建者之一。
*[[余若瑔|余若瑔]]（1870—1934）字达父，彝族，毕节县人。民国时期有名的彝族诗人、学者和法律学家。
