'''毕京京'''（{{bd|1954年|||}}）祖籍[[山东省|山东省]][[桓台县|桓台县]]<ref>{{Cite web |url = http://www.htzx.gov.cn/show.asp?id=1116 |title =当代桓台在外名人风采录  |author =  |date = 2011-12-06 |publisher = 中国人民政治协商会议山东省桓台县委员会 |language =  |accessdate =  |quote =  }}</ref>，生于[[北京市|北京市]]，[[中国人民解放军中将|中国人民解放军中将]]。

==生平==
1968年底，毕京京、[[任志强|任志强]]等北京[[知青|知青]]来到今[[陕西省|陕西省]]延安县冯庄公社郭庄大队（今[[延安市|延安市]][[宝塔区|宝塔区]]郭庄村）插队。当时毕京京年仅14岁，是到延安插队的年龄最小的知青<ref>{{Cite web|url =http://www.sznews.com/culture/content/2010-07/08/content_4734265_4.htm|title =任志强母亲文革中为护子女手持菜刀与红卫兵对峙|author =|date =2010-07-08|publisher =深圳新闻网|language =|accessdate =|quote =}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref>{{Cite web |url =http://money.163.com/10/0709/10/6B53PPGG00253B0H_3.html  |title = 任志强：高官之子被喻“国策顾问”（三） |author =  |date = 2010-07-09 |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。1970年，毕京京入伍，历任战士、班长、团政治处组织股股长，师政治部副主任，[[中国人民解放军国防大学|中国人民解放军国防大学]]政治部组织部部长、校办公室主任、基本系政委、政治部副主任等职务。2006年，任国防大学马克思主义教研部主任。同时他还兼任国防大学中国特色社会主义理论体系研究中心领导小组副组长，国防大学国防经济研究中心副理事长<ref>{{Cite web |url =http://news.163.com/12/0131/17/7P45G1NB00014JB6.html  |title = 国防大学马克思主义教研部主任毕京京做客人民网 |author =  |date =  2012-01-31|publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。2012年12月，任国防大学副校长。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。

2005年7月，晋升少将军衔。2014年7月，晋升中将军衔。
