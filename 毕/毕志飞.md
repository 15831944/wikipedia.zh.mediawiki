'''毕志飞'''（{{bd|1979年|2月10日|}}），北京实传创文化传媒有限公司总经理<ref name="weibo">{{cite web |title=时光网Mtime 新浪微博2019年4月2日信息 |url=https://www.weibo.com/1663735900/HnO9CqrPg?filter=hot&root_comment_id=0&type=comment#_rnd1554213313053 |author=时光网Mtime 新浪微博 |date=2019-04-02}}</ref>。

== 生平 ==
2001年毕业于[[北方工业大学|北方工业大学]]的工业设计专业，后又以第一名的成绩考上了[[北京电影学院|北京电影学院]]表演系的硕士。2007年考上北京大学。<ref>{{cite web |author1=Vista看天下 |title=毕志飞与“史上最烂片导演”称号的斗争 |url=http://www.kaixian.tv/gd/2018/0212/310868.html |website=新浪网 |accessdate=2018-09-24}}</ref>因自称筹备了12年的电影《[[纯洁心灵·逐梦演艺圈|纯洁心灵·逐梦演艺圈]]》饱受恶评，并起诉[[豆瓣网|豆瓣网]]而被熟知。

== 争议事件 ==
=== 起诉豆瓣 ===
《纯洁心灵·逐梦演艺圈》剛一上映就在豆瓣獲得大量差評。2017年5月30日，毕志飞发布公开信，表示不满自己的电影在豆瓣只有2分。<ref name="fh">{{cite web |title=国产片《纯洁心灵》上映5天撤档 曾痛批豆瓣评分低 |url=http://ent.ifeng.com/a/20170927/42987949_0.shtml |website=凤凰网 |accessdate=2018-09-24}}</ref>豆瓣在公开信发布第二天在官方[[微博|微博]]发布了一则起诉状，称已将毕志飞和其公司告上法庭，控诉其诽谤。<ref>{{cite news |author1=袁文幻 |author2=顾岳 |title=史上最低评分，我不服 |url=http://zqb.cyol.com/html/2018-06/06/nw.D110000zgqnb_20180606_1-10.htm |accessdate=2018-09-23 |work=中国青年报}}</ref>9月26日，上映五天的《纯洁心灵·逐梦演艺圈》被撤档，毕志飞表示撤档是由于负评太多。<ref name="fh" />

2018年1月22日，《逐梦演艺圈》出品方（北京实传创文化传媒有限公司）对豆瓣运营方（北京豆网科技有限公司）提出民事起诉，控告豆瓣没有尽其作为平台管理者的管理义务，导致其网站上持续出现对于此电影的恶意、不实评论，构成误导，有损实传创公司的合法权益；实传创公司又指，豆瓣在《逐梦演艺圈》上映当天把电影评分锁定为最低分。毕志飞发微博表示只索赔1元。此案于8月22日在北京朝阳法院开庭审理。<ref>{{cite web |title=《纯洁心灵》遭遇史上最低评分 毕志飞起诉豆瓣 |url=http://ent.sina.com.cn/m/c/2018-01-22/doc-ifyquptv8715229.shtml |website=新浪网 |accessdate=2018-09-24}}</ref><ref>{{cite news |title=《逐梦演艺圈》起诉豆瓣 要求为恶评泛滥道歉 |url=http://epaper.dyrbao.com/yaowen/1702.html |accessdate=2018-09-24 |work=大粤日报 |date=2018-08-22}}</ref>

=== 博士论文引批评 ===
毕志飞于2011年博士毕业，撰写论文《艺术与商业融合的不懈探索——马丁·斯科西斯导演研究》。在处女作获得恶评后，又有网友剖析毕志飞的博士论文，发现存在参考文献不规范、大量引用教材、错误随处可见等问题。<ref>{{cite web |author1=张丰 |title=那些“心灵不纯洁”的博士论文，公众可以打差评吗 |url=http://view.news.qq.com/original/intouchtoday/n4037.html |website=腾讯评论 |accessdate=2018-09-29}}</ref>
