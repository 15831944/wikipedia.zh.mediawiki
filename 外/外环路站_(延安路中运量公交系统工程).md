'''外环路站'''位于[[上海市|上海市]][[闵行区|闵行区]][[虹桥镇_(上海市)|虹桥镇]]北部，[[延安西路_(上海)|延安西路]]南支与[[沪青平公路|沪青平公路]]、[[环西一大道|环西一大道]]路口东侧中央隔离带，临近龙柏新村，为[[上海公交71路|上海公交71路]]及其区间车共用的地面[[岛式月台|岛式车站]]。<ref>{{cite web|url=http://www.changning.sh.cn/art/2016/8/23/art_3990_369220.html|title=公告公示 延安路中运量公交线长宁设十二个站点|accessdate=2017-07-10|language=en|website=上海长宁区门户网站}}</ref>

== 车站構造 ==
该站点开工于2016年6月25日<ref name="kg">{{cite web | title = 延安路中运量公交系统工程开工 | publisher = 上海市交通委员会 | date = 2016-06-27 | url = http://www.jt.sh.cn/jtw/xwzx/n218/u1ai51190.html | accessdate = 2016-11-30 | language = zh | archive-url = https://web.archive.org/web/20161010223253/http://www.jt.sh.cn/jtw/xwzx/n218/u1ai51190.html | archive-date = 2016-10-10 | dead-url = yes }}</ref>，于2017年2月1日清晨正式投入使用<ref name="syy">{{cite news|title=上海延安路中运量公交开通试运营|url=http://www.thepaper.cn/newsDetail_forward_1610252|publisher=澎湃新闻|date=2017-02-01}}</ref>。

[[File:延安路中运量系统工程：在建外环路站及其天桥.jpg|thumb]]
外环路站为[[岛式月台|岛式月台]]，为2面2线地面车站组成，站台设有半高屏蔽门。西侧有天桥引桥，并设有两台[[电梯|电梯]]由站台通往天桥，南北天桥出口处附近还各有一台电梯。这座天桥也是整个中运量系统中唯一的新建天桥。<ref>{{cite web|title=延安路中运量公交 新建天桥吊装完成 A09-上海新闻-新闻晨报|url=http://newspaper.jfdaily.com/xwcb/html/2016-10/27/content_224374.htm|website=newspaper.jfdaily.com|publisher=新闻报社|accessdate=2017-07-10|date=2016-10-27}}</ref>

== 周边设施 ==
* 站点南面：[[龙柏新村|龙柏新村]]、[[沪渝高速|沪渝高速]]
* 站点北面：[[上海动物园|上海动物园]]临时枢纽站
* 站点西面：机场新村
* 站点上方：[[沪青平立交|沪青平立交]]

== 公交换乘 ==
附近有3个常规公交站点：
* 延安西路环西一大道：[[上海公交91路|91路]]、196路、519路、739路、748路、911路、941路、1207路
* 沪青平公路环西一大道：[[上海公交91路|91路]]、196路、519路、709路、739路、911路
* 上海动物园临时枢纽站：721路、709路、328夜宵

== 车站出口 ==
这个车站有两个出入口。
* 南出入口在延安路南侧，直通延安西路环西一大道（市区方向）公交站点。
* 北出入口在延安路北侧公园内。
