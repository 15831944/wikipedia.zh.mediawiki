'''外星生物創造論'''、'''外星人創造論'''或'''外星人複製論'''，是一套[[創造論|創造論]][[假說|假說]]，指[[地球|地球]]及地球[[生物|生物]]或[[文明|文明]]是由[[古代太空人|古代太空人]]所[[創造|創造]]出來。這些假說除了常出現在[[科幻小說|科幻小說]]和[[新興宗教|新興宗教]]外，亦有科學家研究這種假說的可行性。大眾電視廣播媒體亦有節目探討這種假說，如電視節目[https://web.archive.org/web/20141127004722/http://news.cts.com.tw/cna/entertain/201103/201103090686955.html 遠古外星人][http://www.youtube.com/watch?v=iSxr6O2f6rk 第二集：眾神與外星人]。

== 相信外星生物創造論的宗教 ==
一些新兴[[宗教|宗教]]相信地球文明是由[[古代太空人|古代太空人]]創造出來。較著名的有[[雷爾教派|雷爾教派]]和[[天堂之門|天堂之門]]。一些外星生物創造論者稱「星際聯邦宇宙大艦隊指揮官」[[阿斯塔|阿斯塔]]正幫助地球人作心靈成長。{{fact|date=2016-01-28}}

===雷爾運動===
[[雷爾運動|雷爾運動]]認為“造物主”其實是另一個[[星球|星球]]的[[科學家|科學家]]，利用[[基因工程|基因工程]][[創造|創造]]人類，而“造物主”的人類也是用此種方法造物，並且認為若能解決目前的人類問題，將來也能利用[[科技|科技]]創造生命，成為下一個有[[智慧|智慧]][[生命|生命]]的造物主。

== 科學界的看法與角度 ==
{{main|泛種論#引導性泛種論}}
从科学角度而言，不能全盘否定外星生物创造论的价值，合理的质疑是科学进步的动力。

从科学的角度而言，外星生物确实有可能创造新的物种。就拿人类来说，人类目前已经运用[[转基因|转基因]]、[[複製|複製]]等[[生物工程技术|生物工程技术]]创造了不少自然界原本不存在的生物。从理论上来说，人类有可能用生物工程的方法创造全新的[[物种|物种]]，因此对这些全新的物种而言，人类就成为“造物主”。同样的道理，地球上的人类也有可能是外星生物创造的。

在[[科學|科學]]世界，[[外星生物|外星生物]]的題材長期引起關注，雖然一些科學家假設外星人創造[[地球|地球]][[文明|文明]]<ref>
[http://www.newscientist.com/channel/fundamentals/mg18825271.600;jsessionid=AHDPKBENOFAN 尋找智慧生物] {{webarchive|url=https://web.archive.org/web/20070930061927/http://www.newscientist.com/channel/fundamentals/mg18825271.600;jsessionid=AHDPKBENOFAN |date=2007-09-30 }} 新科學人雜誌</ref>，但目前主流研究因受限政治因素及學術資源等現實條件，多仍集中探討[[外太空|外太空]]環境是否有條件讓生物形成。

迄今，{{fact|外星生物創造論並無任何公認的[[證據|證據]]支持|time=2018-12-28}}。但憑諸如古文明的巨大遺蹟超越認知的古代技術連結荒誔不經的情節，令[[傳說|傳說]]長久在坊間流傳。
