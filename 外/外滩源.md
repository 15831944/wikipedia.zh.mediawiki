'''外滩源'''是[[上海|上海]]一块[[上海历史|历史]]悠久的区域，位于[[上海外滩|上海外滩]]北端，[[苏州河|苏州河]]与[[黄浦江|黄浦江]]的交汇处，是近代上海政治、文化、西学、金融、商业的重要发源地之一<ref name="外滩源">[http://wtysh.com/about.php?type=1 外滩源故事] {{webarchive|url=https://web.archive.org/web/20140203033337/http://wtysh.com/about.php?type=1 |date=2014-02-03 }} 上海外滩源.</ref>，并保留了众多历史悠久的建筑，现在正在实施综合改造计划以让它重放光彩<ref name="上海政务">{{Cite web |title = 洛克·外滩源将艺术文化历史交融汇合 |url = http://shzw.eastday.com/shzw/G/20130303/u1ai100184.html |publisher = 上海政务 | date = 2013-03-03 |accessdate = 2014-01-28 }}</ref>。其具体位置为苏州河以南、黄浦江以西、[[四川路_(上海)|四川中路]]以东、[[滇池路|滇池路]]以北，占地面积为22.9万平方米<ref name="新浪">{{Cite web |title = 上海洛克菲勒集团外滩源项目即将正式启动 |url = http://sh.house.sina.com.cn/news/2008-04-05/111062010.html |publisher = 新浪网 |date = 2008-04-05 |accessdate = 2014-01-28 |deadurl = yes |archiveurl = https://web.archive.org/web/20140202203904/http://sh.house.sina.com.cn/news/2008-04-05/111062010.html |archivedate = 2014-02-02 }}</ref>。

外滩源拥有外滩地区最早的一批建筑<ref name="中国日报">{{Cite web |title = 8年精心打造 上海外滩源显露迷人风采 |url = http://www.chinadaily.com.cn/dfpd/expo2010/2010-04/30/content_9799242.htm |publisher = [[中国日报|中国日报]] |date = 2010-04-30 |accessdate = 2014-01-28 }}</ref>。1848年，[[英国驻上海总领事馆|英国驻上海总领事馆]]最先在这里开始建造。之后，这里又集中了许多的[[非营利组织|非营利组织]]，如[[亚洲文会|亚洲文会]]北中国支会等。所以，外滩源的“源”并不是说外滩源是外滩的源头，而是指所处于外滩源的历史建筑的年代要早于外滩的意思<ref name="生活周刊">{{Cite web |title = 外滩源 大隐于市的另一面 |url = http://shzw.eastday.com/shzw/G/20130303/u1ai100184.html |publisher = [[生活周刊|生活周刊]] |accessdate = 2014-01-28 }}</ref>。外滩源区域内的一期开发项目已有不少已开业或即将开业，如[[洛克·外滩源|洛克·外滩源]]、[[上海半岛酒店|上海半岛酒店]]、[[外滩源33|外滩源33]]项目、[[益丰·外滩源|益丰·外滩源]]等。

== 开发进度 ==
外滩源历史悠久，然而过去，这片区域虽然离外滩十分接近，却一直没有得到非常完善的开发<ref name="生活周刊" />。直到2002年12月，外滩源项目正式立项，共分三期，并委托[[新黄浦集团|新黄浦集团]]进行拆迁和开发<ref name="新浪" />。一期位于中山东一路以西、苏州河以南、虎丘路以东、滇池路以北。外滩源项目以“重现风貌、重塑功能”为原则，在对该区域进行保护的同时开展开发。2004年5月，美国洛克菲勒国际集团与新黄浦集团签约成为项目的总体开发商<ref name="新浪" />。2005年，在[[百仕達控股|百仕達控股]]加入后，三方合资成立了上海洛克菲勒集团外滩源综合开发有限公司<ref name="新浪" />。2010年，外滩源一期的组成部分相继开业，当年3月18日半岛酒店开业<ref name="中国日报" />。

== 一期 ==
一期位于中山东一路以西、苏州河以南、虎丘路以东、滇池路以北，占地总面积9.7公倾，共包括外滩源33项目、益丰·外滩源、洛克·外滩源、半岛酒店和市政及环境景观配套五个项目<ref name="外滩源" />。目前，半岛酒店与益丰·外滩源已经开张营业，洛克·外滩源老建筑部分已开始营业。市政管线及道路改造也已完工<ref name="外滩源" />。

=== 外滩源33项目 ===
外滩源33项目由作为项目核心的[[原英国驻沪总领事馆|原英国驻沪总领事馆]]的两栋历史建筑和原[[教会公寓|教会公寓]]（今3号楼）、原[[新天安堂|新天安堂]]（今4号楼）、原划船俱乐部（今5号楼）以及公共绿地、亲水平台和地下空间等共同组成，集会展、会务、文化、沙龙等为一身<ref name="外滩源" />。项目总占地面积约2.7万平方米。原英国驻沪总领事馆的主楼（今1号楼），也就是外滩现存最早的建筑，现在被改造成了外滩源壹号，提供餐饮、会务、艺术展览等服务。还开设有外滩源壹号金融家俱乐部以及壹号酒窖。原领事馆二号楼现租给[[百达翡丽|百达翡丽]]作为其[[远东|远东]]旗舰店——“百达翡丽源邸”<ref name="外滩源" />。

原教会公寓由香港永力管理有限公司使用，为女性养生会所。新天安堂曾是[[罗素|罗素]]于1920年在上海的演讲之地，现在被改建成了一个展览会馆，曾举办众多商业及文化活动，如2012年上海双年展。同时，这里也成为了众多新人拍摄婚纱照的绝佳场所之一。5号楼的原划船俱乐部曾是上海英国侨民的体育俱乐部，现在是日本LD&K公司旗下SHANGHAI ROSE酒吧的所在地<ref name="外滩源" />。

=== 洛克·外滩源 ===
“洛克·外滩源”处于[[圆明园路|圆明园路]]、[[虎丘路|虎丘路]]、[[北京东路_(上海)|北京东路]]、[[南苏州路|南苏州路]]所包围的区域，占地面积1.9万平方米<ref name="上海政务" />，建筑面积9.4万平方米。项目将改建11幢历史建筑和兴建6幢现代建筑，其中6号楼将在保留原有[[美丰洋行|美丰洋行]]东、南两个立面的基础上翻建而成<ref>[http://wenku.baidu.com/view/fe353a4559eef8c75fbfb3aa.html?re=view 沈淳-外滩源美丰洋行局部外墙结构加固与保护方案]</ref><ref>[http://www.rockbundlife.com/chinese/v.asp?id=120 Rockbund Life] {{webarchive|url=https://web.archive.org/web/20140221202309/http://www.rockbundlife.com/chinese/v.asp?id=120 |date=2014-02-21 }}洛克·外滩源.</ref>。建成后将作为高档写字楼、奢侈品商场、高档餐厅、酒店式公寓等使用，成为上海最高端奢华的区域之一。区域内的[[上海亚洲文会大楼|原亚洲文会大楼]]已改建成[[上海外滩美术馆|上海外滩美术馆]]已于2012年5月开业，成为了洛克·外滩源最早亮相的一处。6栋现代建筑预计在2014年年中竣工<ref name="上海政务" />。

=== 益丰·外滩源 ===
“益丰·外滩源”位于国内现存最长的巴洛克式建筑——原益丰洋行大楼内，在北京东路上，建筑面积29025.5平方米，共5层<ref>[http://www.insiteasset.com/project/yi-feng/ The Galleria·益丰·外滩源] {{webarchive|url=https://web.archive.org/web/20140201203816/http://www.insiteasset.com/project/yi-feng/ |date=2014-02-01 }} 盈石资产管理有限公司.</ref>。拥有奢侈品商场、[[米其林餐厅|米其林餐厅]]、私人会所等多个设施。

== 二期 ==
二期包含虎丘路西、四川中路东、南苏州路南、北京东路北和北京东路南、圆明园路西、滇池路北、四川中路东的两个地块，占地5.2[[公顷|公顷]]<ref name="新浪" />。目前已开展规划、动迁等前期工作<ref>[http://www.shbiz.com.cn/Item/64693.aspx 外滩源二期项目正式启动] {{webarchive|url=https://web.archive.org/web/20140222043616/http://www.shbiz.com.cn/Item/64693.aspx |date=2014-02-22 }} [[上海商报|上海商报]].</ref>；但由于种种原因，至今仍未有实质性进展。

== 历史建筑 ==
外滩源区域内拥有大量历史建筑。目前，已列入[[上海市优秀历史建筑|上海市优秀历史建筑]]共有14座<ref name="上海档案信息网">{{Cite web |title = “外滩源”的过去和现在 |url = http://www.archives.sh.cn/shjy/scbq/201203/t20120313_5686.html |publisher = 上海档案信息网 | date = 2009-02-20 |accessdate = 2014-02-08 }}</ref>。
