{{Vgname|外圍世界|en=The Outer Worlds}}是一款由[[黑曜石娛樂|黑曜石娛樂]]開發、{{tsl|en|Private Division|Private Division}}發行的動作角色扮演遊戲。本作預定2019年內推出，並會有[[Microsoft_Windows|Microsoft Windows]]、[[PlayStation_4|PlayStation 4]]及[[Xbox_One|Xbox One]]版本。

== 玩法 ==
在遊戲的早期階段，玩家可以創建自己的角色並解鎖一艘太空船，而這艘太空船則充當著遊戲的中央樞紐，向玩家提供快速旅行、倉庫等功能，同時亦是玩家與其伙伴的家<ref name="Mejia@181207">{{Cite web |url = https://www.shacknews.com/article/108855/the-outer-worlds-preview-flaws-in-the-system |title = The Outer Worlds preview: Flaws in the system |author = Ozzie Mejia |date = 2018-12-07 |publisher = Shack News |language = en |accessdate = 2018-12-16 |archive-url = https://web.archive.org/web/20181209124421/https://www.shacknews.com/article/108855/the-outer-worlds-preview-flaws-in-the-system |archive-date = 2018-12-09 |dead-url = no }}</ref>。玩家可以在遊戲內遇到並招募不同的非玩家角色作為同伴，而每個角色都有自己的個人任務和故事<ref>{{Cite web |url = https://www.shacknews.com/article/108958/the-outer-worlds-interview-character-building-companions-branding-and-more |title = The Outer Worlds interview: Character building, companions, branding, and more |author = Ozzie Mejia |date = 2018-12-10 |publisher = Shack News |language = en |accessdate = 2018-12-16 |archive-url = https://web.archive.org/web/20181211013016/https://www.shacknews.com/article/108958/the-outer-worlds-interview-character-building-companions-branding-and-more |archive-date = 2018-12-11 |dead-url = no }}</ref>。每個同伴都具有個人技能和特殊攻擊，他們也可以發展自己的技能專長<ref name="Fextralife@181210">{{Cite web |url = https://fextralife.com/the-outer-worlds-everything-we-know-about-fallouts-spiritual-successor/ |title = The Outer Worlds: Everything We Know about Fallout’s Spiritual Successor |author = Fextralife |date = 2018-12-10 |language = en |accessdate = 2018-12-16}}</ref>。在探索時，玩家可以攜帶兩個同伴，而在戰鬥中他們更起到輔助作用<ref name="Fextralife@181210"/>。另外，玩家可以在[[對話樹|對話樹]]中作出許多不同的決定，而這些決定將會影響遊戲分支故事的走向<ref name="Tyrrel@181207">{{Cite web |url = https://www.ign.com/articles/2018/12/07/the-outer-worlds-is-bringing-fun-back-to-science-fiction |title = The Outer Worlds Is Bringing Fun Back to Science Fiction |author = Brandin Tyrrel |date = 2018-12-07 |publisher = IGN |language = en |accessdate = 2018-12-16 |archive-url = https://web.archive.org/web/20181209123649/https://www.ign.com/articles/2018/12/07/the-outer-worlds-is-bringing-fun-back-to-science-fiction |archive-date = 2018-12-09 |dead-url = no }}</ref>。玩家亦可以透過不同的方式來應對非玩家角色，例如是扮演英雄或者瘋子等<ref name="Tyrrel@181207"/>。

在戰鬥時，玩家可以使用不同種類的武器應戰，例如近戰武器和槍械，而槍械的彈藥類型則分別有輕型、中型和重型三種。玩家可以透過改造槍械來增加元素損害<ref name="Fenlon@181207">{{Cite web |url = https://www.pcgamer.com/obsidians-the-outer-worlds-blends-firefly-and-fallout-into-a-bold-open-ended-sci-fi-rpg/ |title = Obsidian's The Outer Worlds blends Firefly and Fallout into a bold, open-ended sci-fi RPG |author = Wes Fenlon |date = 2018-12-07 |publisher = PC Gamer |language = en |accessdate = 2018-12-16 |archive-url = https://web.archive.org/web/20181216093312/https://www.pcgamer.com/obsidians-the-outer-worlds-blends-firefly-and-fallout-into-a-bold-open-ended-sci-fi-rpg/ |archive-date = 2018-12-16 |dead-url = no }}</ref>。玩家還可以利用隱身或使用社交技能說服，以避免戰鬥的發生。隨著遊戲推進，玩家會獲得經驗值，可供玩家升級及解鎖自己或同伴的技能<ref name="Tyrrel@181207"/>。除了戰鬥技能外，玩家亦可以學習技術技能。技術技能包括三大範疇：科學、醫學及工程，而投資這些技能能令玩家解鎖額外能力並且改善戰鬥效率<ref name="Tyrrel@181207"/>。遊戲還設有隊長技能機制，當玩家與同伴出行時便會發動，同伴分別會獲得提高戰鬥力或降低傷害的增益效果<ref name="Tyrrel@181207"/>。另外，遊戲設有「缺陷」系統，這是指當遊戲中發生某些事情後，玩家可以選擇的角色特徵<ref name="Fenlon@181207"/>。當玩家受到傷害或遇到某些挫折時，「缺陷」便會出現，其中包括害怕黑暗、火焰等<ref name="Mejia@181207"/><ref>{{Cite web |url = https://www.gamespot.com/articles/the-outer-worlds-looks-a-lot-like-fallout-but-that/1100-6463757/ |title = The Outer Worlds Looks A Lot Like Fallout, But That's Only Half The Story |author = Peter Brown |date = 2018-12-07 |publisher = GameSpot |language = en |accessdate = 2018-12-16 |archive-url = https://web.archive.org/web/20181209124434/https://www.gamespot.com/articles/the-outer-worlds-looks-a-lot-like-fallout-but-that/1100-6463757/ |archive-date = 2018-12-09 |dead-url = no }}</ref>。這些「缺陷」是永久的，玩家可以擁有多達三個「缺陷」，但相對地玩家可以獲得額外的獎勵<ref name="Fenlon@181207"/>。

== 劇情 ==
故事設定在未來世界，那時候的大型企業已經開始在外星球上進行改造和殖民。其中，玩家所乘坐的太空船因錯誤而到達銀河系邊緣的殖民地。當玩家從冷凍睡眠中醒來，卻發現大多數乘客仍處於休眠狀態，其後玩家需要前往附近的殖民地，並會遇到各種不同的勢力與派系<ref name="Fenlon@181207"/><ref>{{Cite web |url = https://gnn.gamer.com.tw/0/172250.html |title = Obsidian 曝光新作《外圍世界 The Outer Worlds》 由《異塵餘生》原創作者製作 |author = 巴哈姆特電玩資訊站 |date = 2018-12-07 |language = zh |accessdate = 2018-12-16}}</ref>。

== 開發 ==
本作由[[黑曜石娛樂|黑曜石娛樂]]開發，並由[[Take-Two_Interactive|Take-Two Interactive]]旗下的發行品牌{{le|Private Division}}發行<ref>{{Cite web |url = https://www.rockpapershotgun.com/2018/12/07/obsidian-entertainment-announce-the-outer-wilds/ |title = Obsidian Entertainment announce The Outer Worlds |author = Alice O'Connor |date = 2018-12-07 |publisher = [[Rock,_Paper,_Shotgun|Rock, Paper, Shotgun]] |language = en |accessdate = 2018-12-16 |archive-url = https://web.archive.org/web/20181213165642/https://www.rockpapershotgun.com/2018/12/07/obsidian-entertainment-announce-the-outer-wilds/ |archive-date = 2018-12-13 |dead-url = no }}</ref>。儘管在本作發佈時，黑曜石娛樂已被[[微軟|微軟]]收購，但這個項目在此之前一直處於開發階段，而且Take-Two Interactive亦在微軟收購前獲得發行權<ref name="Hall@181207">{{Cite web |url = https://www.polygon.com/2018/12/7/18130594/the-outer-worlds-obsidian-microsoft |title = The Outer Worlds isn’t a Microsoft game, even though it’s buying Obsidian |author = Charlie Hall |date = 2018-12-07 |publisher = Polygon |language = en |accessdate = 2018-12-16 |archive-url = https://web.archive.org/web/20181217072434/https://www.polygon.com/2018/12/7/18130594/the-outer-worlds-obsidian-microsoft |archive-date = 2018-12-17 |dead-url = no }}</ref>。

《[[異塵餘生系列|異塵餘生]]》系列的創作者{{tsl|en|Tim Cain|蒂姆·坎}}（{{lang-en|Tim Cain}}）及{{tsl|en|Leonard Boyarsky|倫納德·博亞雷斯基}}（{{lang-en|Leonard Boyarsky}}）擔任本作的總監<ref name="Fenlon@181207"/><ref>{{Cite web |url = https://nichegamer.com/2017/12/14/new-obsidian-rpg-coming-via-original-fallout-creators-tim-cain-and-leonard-boyarsky/ |title = New Obsidian RPG Coming via Original Fallout Creators Tim Cain and Leonard Boyarsky |author = Brandon Orselli |date = 2017-12-14 |publisher = Niche Gamer |language = en |accessdate = 2018-12-16 |archive-url = https://web.archive.org/web/20180725062845/http://nichegamer.com/2017/12/14/new-obsidian-rpg-coming-via-original-fallout-creators-tim-cain-and-leonard-boyarsky/ |archive-date = 2018-07-25 |dead-url = no }}</ref>。兩位遊戲總監描述本作為「蒂姆的愚蠢及博亞雷斯基的黑暗病態的組合」，並希望在創作遊戲的語氣和敘事時尋求愚蠢和戲劇之間的平衡<ref name="Fenlon@181207"/>。在遊戲開發時，開發人員曾考慮加入一些浪漫的選擇，但最終被刪去了<ref name="Hall@181207"/>。
