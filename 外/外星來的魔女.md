《'''外星來的魔女'''》（{{lang-ja|星のウイッチ}}）是[[日本|日本]][[漫畫家|漫畫家]][[原悠衣|原悠衣]]的漫畫作品，由[[Mag_Garden|Mag Garden]]發行的〈[[月刊Comic_BLADE|月刊Comic BLADE]]〉於2006年3月號到2007年6月號之間連載<ref>{{cite web|url=http://www.amazon.co.jp/dp/B000E5MJAK|title=月刊 COMIC BLADE (コミックブレイド) 2006年 03月号|publisher=[[Amazon.co.jp|Amazon.co.jp]]|accessdate=2015年8月12日|language=ja}}</ref><ref>{{cite web|url=http://www.amazon.co.jp/dp/B000PAAJNS|title=月刊 COMIC BLADE (コミックブレイド) 2007年 06月号|publisher=Amazon.co.jp|accessdate=2015年8月12日|language=ja}}</ref>，[[單行本|單行本]]全3卷。

== 故事簡介 ==
這個世界有兩個星球，分別是「普拉涅特」和只存在於傳說中的「阿爾特密斯」。從小就對阿爾特密斯的存在深信不疑的日月，有一天突然被下了死亡詛咒！為了解除詛咒，日月終於來到了阿爾特密斯，但卻再也回不去了！

== 登場人物 ==
; {{nihongo|日月|カヅキ}}
:男主角。父親曾告訴有阿爾特密斯的存在故事，並且相信。有天遇見了奇異服裝的魔女，阿爾特密斯確實的存在。

; {{nihongo|瑪莉|マリー}}
:女主角。魔女一族唯一存活百年的瑪莉，來到普拉涅特爲了追黑色惡魔的少女，後來遇見了日月，回到了阿爾特密斯想繼續熱心幫助，但大家稱瑪莉為幽靈的魔女。似乎也藏著不可告知的秘密!

; {{nihongo|薇歐拉|ヴィオラ}}
:瑪莉來到普拉涅特時，追逐黑色惡魔少女。而來到普拉涅特時，對著日月說著「救救那個人..」，薇毆拉目的究竟是?

; {{nihongo|亞斯塔|アスター}}
:魔導士，百年前魔導士與魔女族發生戰爭，公主柯蒂莉雅死後，打算復仇毀滅阿爾特密斯星。與瑪莉有著很深的關係，瑪莉拒絕亞斯塔的作為，亞斯塔最後只好奪走瑪莉的魔力。

; {{nihongo|柯蒂莉雅|コーデリア}}
:與瑪莉和亞斯特有著很深的關係，戰爭死去。但是與薇歐拉有著相似的關連。

; {{nihongo|潔西卡|ジェシカ}}
:老公亞歷克失去記憶，日月和瑪莉的幫助下，亞歷克恢復了記憶。

; {{nihongo|歐茲|オズ}}
:自稱小丑歐茲，作為一個旁觀者看著世界的過去和未來，總是攜帶著雨傘，真實身分是傳說中阿爾特密斯的大大大魔女。一直以來守護著星球。
