台灣將外籍勞工分為兩種，「藍領」（又稱外籍勞工）與「白領」（又稱外國專業人員）。根據目前的法規，外勞的薪資等[[勞動條件|勞動條件]]受《[[就業服務法|就業服務法]]》與《[[勞動基準法_(中華民國)|勞動基準法]]》規範。其中就業服務法規定聘僱外國人工作必須先保障保障國民工作權，因此雇主必須先申請許可才能雇用外國人，「不得妨礙本國人之就業機會、勞動條件、國民經濟發展及社會安定」。就業服務法也要求注意並且通報政府外國人的健康情形、行蹤等等。外國人如果有居留證，則適用[[全民健保|全民健保]]。2006年1月起，[[台灣桃園國際機場|台灣桃園國際機場]]的[[外勞服務站|外勞服務站]]啟用，有越語，泰語，印尼語，英語四國語言電話專線作為申訴管道，減少不願登機、不當遣返等情形。目前藍領外勞在台工作年限為12年，每 3 年不須強制出境。<ref>[http://www.sea.com.tw/news_detail.php?sn=2642 即日外勞展延送件或尚未審核案件均適用新規定]外勞通訊社2012-02-01</ref>2018年初，「藍領」與「白領」外國人士合計已近71萬（均不含中國籍）。

;藍領外勞
1989年之前，中華民國中央政府並不開放[[低技術|低技術]]國際移工（外勞）進入國內勞動市場，並且嚴格管制其他類別的外勞。但是由於產業結構逐漸變化與調整，產業界對引進外勞的需求越來越高；而勞動市場上也存在著許多「非法外勞」。1989年政府推動[[十四項建設|十四項建設]]，勞力短缺，於是以「專案方式」正式引進第一批低技術外勞。1992年，通過《[[就業服務法|就業服務法]]》後，其中設有外籍勞工專章，允許民間產業引進外勞，並且逐次放寬引進的行業種類。當時對外勞政策的爭議主要有三項，一是外勞是否會影響國民工作權，二是外勞在國內的基本人權如何保障，三是[[外勞仲介制度|外勞仲介制度]]的高額仲介費造成剝削。{{ref|histo}}之後台灣的外勞法規逐年放寬，2012年2月，藍領外勞在台工作年限由9年延長至12年，取消展延規定，聘僱許可期間由2年修正為3年。<ref>[http://www.sea.com.tw/news_detail.php?sn=2631 立院三讀通過 外勞工作年限延長至12年]外勞通訊社2012-01-19</ref>9月通過80歲以上且巴氏量表60分者可申請外籍家庭看護工照護，並刪除千人以上製造業申請外籍勞工比率，應刪減1％至3％之規定。<ref>[http://www.sea.com.tw/news_detail.php?sn=2720 80歲巴氏量表放寬至60分 9月實施]外勞通訊社2012-05-09</ref><ref>[http://www.sea.com.tw/news_detail.php?sn=2825 80歲可申請外勞 刪除千人遞減率]外勞通訊社2012-09-06</ref>2013年，外籍看護工逃跑等待期由原本6個月降至3個月；另外除外籍看護工之外所有業別外勞由原本查獲後才可遞補，放寬至依規定通報滿6個月仍未查獲就可申請遞補。<ref>[http://www.sea.com.tw/news_detail.php?sn=3206 養護機構或製造工逃逸已滿6個月]外勞通訊社2013-12-27</ref><ref>[http://www.sea.com.tw/news_detail.php?sn=3169 立院初審通過 外勞逃逸雇主遞補鬆綁]外勞通訊社2013-10-30</ref><ref>[http://www.peopo.org/news/2012 在臺灣工作外國人數，破55萬人。我們該怎麼辦呢?]</ref>2015年，外籍家庭看護工經專業訓練或自立學習，有特殊表現，符合主管機關所定資格，在臺工作年限可延至14年。<ref>[http://www.sea.com.tw/news_detail.php?sn=3664 外籍家庭看護工最長年限14年 10/9生效]外勞通訊社2015-10-09</ref>2016年取消外籍勞工工作期滿須出國1日的規定，並同時新增外勞於聘僱許可期間得請假返國，雇主應予同意規定。<ref>[http://www.sea.com.tw/news_detail.php?sn=4254 立院三讀 取消外勞3年聘滿出國一日]外勞通訊社2016-10-21</ref>

至2017年5月，台灣外勞人數（不含外籍專業人士）約為647,915人。按國籍分，以[[印尼|印尼]]籍最多（252,406），[[越南|越南]]籍次之（193,436），[[菲律賓|菲律賓]]籍第三（142,406），[[泰國|泰國]]籍第四（59,666）<ref name="外勞人數按國籍及性別分">{{cite web|url=http://www.mol.gov.tw/statistics/2452/|title=統計表-外籍工作者-產業及社福外籍勞工人數－按國籍分|publisher=行政院勞動部勞力發展署|accessdate=2017-07-11}}</ref>。人數以[[桃園市|桃園市]]105,623人居冠、其次是[[臺中市|臺中市]]92,723人和[[新北市|新北市]]91,079人。<ref name="外勞人數按地區分">{{cite web|url=http://www.mol.gov.tw/statistics/2452/|title=統計表-地區別資料-產業及社福外籍勞工人數－按地區及國籍分|publisher=行政院勞動部勞力發展署|accessdate=2017-07-11}}</ref>

外勞也漸漸改灣了台灣的社會。[[新北市|新北市]][[中和區|中和區]]的區公所每年都會舉辦與當地民眾及泰籍勞工、外籍配偶同樂的[[潑水節|潑水節]]活動；另外如[[台北市|台北市]][[台北地下街|台北地下街]]、[[桃園市|桃園市]][[中壢區|中壢區]]與台中市[[第一廣場|第一廣場]]等地，經常看見許多專門服務東南亞外勞的商家，而[[中壢車站|中壢車站]]與[[桃園車站|桃園車站]]的告示牌上有書寫[[泰文|泰文]]、[[越南文|越南文]]、[[印尼文|印尼文]]、[[他加祿語|他加祿語]]等東南亞語言。政府另有委託廣播電台製播外勞節目，多半於週日播出。台灣民間與政府出版多種東南亞文字刊物，例如世新大學出版的五份東南亞月刊[[四方報|四方報]]、台北市政府出版的[[外勞e通訊|外勞e通訊]]、高雄市政府出版的[[南洋好姊妹季刊|南洋好姊妹季刊]]。

2005年8月在[[高雄捷運|高雄捷運]]移工宿舍發生外勞因為抗議生活條件太差的[[群體事件|集體暴動]]事件（參見[[高雄捷運外勞弊案|高雄捷運外勞弊案]]）。

台灣有時以凍結外勞作為外交手段，例如2011年2月由於[[菲律賓|菲律賓]]政府將[[中華民國|中華民國]]籍嫌犯遣送到中國大陸，外交部宣布將嚴格審查菲勞來台工作簽證，勞委會也提高申請菲勞案件的審查層級，短暫凍結菲籍外籍勞工入台工作<ref>[http://www.sea.com.tw/news_detail.php?sn=2160 即日起菲勞初招、重招等案延長審核]外勞通訊社2011-02-08</ref>。2013年[[廣大興28號事件|2013年菲律賓公務船攻擊臺灣漁船事件]]後臺灣政府採取反制措施，終止[[菲律賓|菲律賓]]外勞申請來臺<ref>[http://www.sea.com.tw/news_detail.php?sn=3049 制裁菲律賓 即日起凍結菲勞申請案]外勞通訊社2013-05-15</ref>。3個月後，[[菲律賓|菲律賓]]官方代表致歉，台灣於是解除對菲律賓的項制裁措施<ref>[http://www.sea.com.tw/news_detail.php?sn=3091 最快晚上宣布 菲勞解凍擬8/9公告]外勞通訊社2013-08-08</ref>。

;白領外勞
2018年7月底，台灣白領外勞有效聘僱許可人數約為3萬人，主要來自日本、美國、印度、馬來西亞等地<ref>{{cite web|url=http://statdb.mol.gov.tw/html/mon/212120.htm|title=外國專業人員－按核准機關及國籍分|publisher=行政院勞動部勞力發展署|accessdate=2018-08-27}}</ref>。男性約2萬3千人，女性約7千人，大多數從事的專門性技術性工作（1萬8千人），其次是學校和補習班的教師工作，有4千餘人<ref>行政院勞委會職訓局[http://statdb.mol.gov.tw/html/mon/212100.htm 外國人專業人員：按性別及申請類別分]</ref>。台灣白領外勞人數最多的縣市是[[台北市|台北市]]，有1萬1,916人。其次是[[新北市|新北市]]，有2,760人。第三是[[台中市|台中市]]，有2,493人。

== 問題點 ==
=== 爭議事件 ===
*由於來台的東南亞外勞經常受到資方的不當管控與剝削，除了薪資偏低、生活條件差以外，還有限制外勞活動範圍、並限制外勞使用撥打返鄉[[國際電話|國際電話]]的[[電話卡|預付卡]]（俗稱「外勞卡」），造成外勞的不平之鳴。此一因素也導致了2005年[[高雄捷運|高雄捷運]]在高雄縣岡山鎮(今[[高雄市|高雄市]][[岡山區|岡山區]])爆發的[[高雄捷運外勞弊案|高雄捷運外勞弊案]]，並造成中華民國政府主管機關部分首長引咎辭職。
*另外還有大陸漁工（大多來自[[福建省|福建省]]）或外籍漁工（以[[印尼|印尼]]居多）的相關問題經常於台灣出現，尤其是漁工殺害台籍船長的海上喋血案<ref>{{Cite web |url = http://news.ltn.com.tw/news/society/paper/706992 |title=特宏興喋血案／船長抓欄杆求生 遭印尼漁工踹下海|accessdate=2013-08-21 |author=地方中心／宜蘭報導 |date=2013-08-21 |work= |publisher=自由時報|}}</ref>、漁船的失聯案等等<ref>{{Cite web |url = http://www.chinatimes.com/newspapers/20130728000265-260102 |title=近年漁船失聯 清一色僱用印尼漁工|accessdate=2013-07-28 |author= |date=2013-07-28 |work= |publisher=中時電子報|}}</ref>。其實過去還曾有台籍船長殺害漁工的案例<ref>{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20121122/34657205/ |title=海上閻王殺15人 冷血船長拋屍入海 囚13年回台|accessdate=2012-11-22 |author= |date=2012-11-22 |work= |publisher=蘋果日報|}}</ref><ref>{{Cite news|url=https://www.twreporter.org/a/far-sea-fishing-tempo-report|title=【調查報導追蹤】《Tempo》：海上奴工的數目，遠比台灣官方數字多 - 報導者 The Reporter|accessdate=2017-03-14|language=zh-TW}}</ref>。
