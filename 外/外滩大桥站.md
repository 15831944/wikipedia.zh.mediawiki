'''外滩大桥站'''，建设时曾用名'''桃渡路站'''，是[[浙江省|浙江省]][[宁波市|宁波市]]一座地下轨道交通车站，属于[[宁波轨道交通2号线|宁波轨道交通2号线]]。车站于2015年9月26日启用<ref name="main">{{cite web|url=http://news.cnnb.com.cn/system/2010/12/23/006790463.shtml|title=2号线车站设计人性化而有宁波味 效果图抢先看|author=陈锦源|publisher=中国宁波网|date=2010-12-23|accessdate=2011-01-21}}</ref>。规划中的7号线也将到达外滩大桥站。

== 位置 ==
外滩大桥站位于[[江北区_(宁波市)|江北区]]大庆南路、惊驾路路口，沿大庆南路方向。西南侧为宁波来福士广场，北侧为江北区中心小学<ref name="main" />。

== 车站形制 ==
外滩大桥站长188.9米，标准段宽17.8米，深15.73米，设4个出口，2组风亭，运营初期开放3个出入口。地下一层为站厅，二层为[[岛式车站|岛式车站]]<ref name="main" />。

== 车站装饰 ==
[[File:Waitan_Bridge_Station,_2015-10-01_07.JPG|thumb]]
外滩大桥站为2号线文化站之一。车站文化墙主题为“北岸笛韵”。文化墙通过宁波[[江北岸|江北岸]]（老外滩）的河、船、城等建筑剪影表现老外滩的历史，同时利用透光表现古今融合的寓意<ref>{{cite web|url=http://www.nbmetro.com/news_dt.php?info/961|title=2号线一期6大车站文化墙亮相|publisher=宁波轨道交通集团|date=2015-09-23|accessdate=2015-09-27}}</ref>。

== 交通接驳 ==
外滩大桥站接驳槐树路站、外滩大桥西站、大庆南路站和宁大附属医院站。
