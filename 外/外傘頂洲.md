== 歷史 ==
[[File:Map_of_Taiwan_1901.jpg|270px]]
外傘頂洲自然造陸運動形成於日治前不久，1901年[[台灣總督府|台灣總督府]]發行由[[禮密臣|禮密臣]]編輯，日人T. Obanawa描繪的福爾摩沙圖，在東石鄉西側外海己出現三個小圓圈，位置明顯在北港溪出海口之南，這是今日外傘頂汕之原型，在地理上應屬嘉義縣。原位於台灣雲林縣[[臺西鄉|臺西鄉]]外海，在20世紀初才開始受地方政府注意。1914年[[台灣日治時期|日治時代]]，日本人在此首次設立燈塔<ref>{{cite news|title=外傘頂洲 雲縣釘門牌 漂進嘉縣|url=http://www.libertytimes.com.tw/2007/new/nov/26/today-so3.html|publisher=自由時報|author=林國賢|date=2007-11-26||language=zh-tw}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，並命名為佐佐木島。惟沙洲地基不穩，六年後即倒塌。1947年3月26日改名為'''海豐島'''。不論在日治、[[臺灣戰後時期|戰後]]初期、1988年等時期，其管轄政府都曾實施植林固沙的計劃，但均告失敗，仍無法阻止漂沙移動。在1984年資料中，[[潮汐|退潮]]時外傘頂洲面積約1000餘[[公頃|公頃]]，漲潮時約200餘公頃，南北長10餘公里，此後不斷縮小<ref>{{cite news|title=外傘頂洲 小檔案|url=http://www.libertytimes.com.tw/2007/new/nov/26/today-so4.htm|publisher=自由時報|author=嘉義縣政府提供、蔡民一整理|date=2007-11-26|language=zh-tw|deadurl=yes|archiveurl=https://web.archive.org/web/20071127152246/http://www.libertytimes.com.tw/2007/new/nov/26/today-so4.htm|archivedate=2007-11-27}}</ref>。

== 變遷&機制 ==
[[File:Waisanding-Sandbank_1956-2000.svg|270px]]
外傘頂洲受到[[波浪|波浪]]、[[沿岸流|沿岸流]]及[[東北季風|東北季風]]影響，平均每年往西南方向漂移60至70公尺，被稱為「移動的國土」<ref>{{cite news
|title=沙洲流失 外傘頂洲流浪到嘉義
|date=2007-06-01
|publisher=聯合晚報
|author=陳永順
|language=zh-tw
}}</ref>。

早期[[濁水溪|濁水溪]]主流藉[[北港溪|北港溪]]、[[西螺溪|西螺溪]]注入海中，大量泥沙[[河流作用|沈積]]在河口附近海域，經過長年累月的波浪和[[海流|海流]]作用，外傘頂洲可能因此形成；1911年日人開始濁水溪整治，截斷北港溪等支流，主流改由西螺溪入海後，北港溪自成水系，輸砂量也因此銳減，每年減少約10萬立方公尺，旺盛的沉積作用不再，外傘頂洲開始有侵蝕後退的趨勢。[[國立中山大學|國立中山大學]]海洋科學院長[[陳陽益|陳陽益]]指出，外傘頂洲從1904年至1995年，歷經沙洲地形的幼年、青壯年，現已邁入老年階段，沙源無法充分補充，加上侵蝕作用影響，沙洲上的堆沙陸續被沖刷至東側較低窪之[[潟湖|潟湖]]，目前外傘頂洲往西南偏移速度逐漸增大，面積也日漸縮減且被波浪及潮水切割成幾部分。

中山大學研究資料顯示，以20年為單位，在1962年至1982年、1982年至2002年間，外傘頂洲分別被侵蝕流失520公頃、547公頃的沙洲，現存者僅100餘公頃；此外，1982年沙洲上2公尺以上的沙體還有5200萬立方公尺，但在強烈東北季風作用下，2000年以後幾乎刮除殆盡，只剩平坦沙洲。學者研判，如不設法保固。預計在2028年，「移動的國土」可能將成為「消失的國土」<ref>{{cite news

=== 面積逐漸變小 ===
1994年至1998年間，外傘頂洲面積實為遞增，平均約每年增加0.766平方公里。但自1999年起面積逐漸變小，平均約每年減少0.378平方公里<ref name=陳蔚瑋>[https://ir.nctu.edu.tw/bitstream/11536/40387/1/681201.pdf 陳蔚瑋《衛星影像的灘線辨識及其應用至灘線變遷之研究》國立交通大學土木工程研究所，2010年] {{zh-tw icon}}</ref>。 外傘頂洲在高潮線以上之乾砂洲面積，1984年約2.05平方公里，在2003年則縮減至0.5平方公里<ref name=郭禮安>[http://www.tsoe.org.tw/downloads/thesis/2003G9.pdf 郭禮安《外傘頂洲保護措施進行數值分析與水工模型試驗之研究》第25屆海洋工程研討會論文集 國立台灣海洋大學，2003年10月] {{zh-tw icon}}</ref>。1994年至2015年之灘線與面積觀測發現，外傘頂洲海側灘線在北端以每年30.5公尺，南端以每年119公尺之速度往內陸內縮，被侵蝕速率近年有上升的趨勢，以2009至2015年觀測之沙洲陸地被侵蝕的速率推估，2060年外傘頂洲陸地將完全低於平均潮位，淪為潛沒沙洲<ref name=張憲國B>[http://www.csprs.org.tw/Temp/2.106A1201.pdf  張憲國《應用衛星影像的水線辨識於外傘頂洲的灘線變遷》航測及遙測學刊第22卷第4期，2017年12月] {{zh-tw icon}}</ref>。

=== 朝逆時針方向偏移 ===
外傘頂洲呈逆時針方向偏移，每年偏轉角度約0.2度，呈向東南延伸的趨勢，每年南移平均約50至60公尺<ref name=郭禮安/>。

=== 向臺灣本島靠近 ===
外傘頂洲上游段逐漸變寬，而下游段漸漸變窄，兩端皆逐漸向臺灣本島靠近。以目前變化率推算，外傘頂洲北側將於2060年與臺灣本島相連，南側則將於2174年會與本島連接<ref name=陳蔚瑋/>。

=== 濁水溪砂石量影響外傘頂洲變遷 ===
自濁水溪整治計畫實施，北港溪輸沙量大減，使得濁水溪南移漂沙僅達箔子寮漁港，統汕洲與箔子寮汕的縮減明顯且大幅南移。自1993年之後濁水溪砂石量減採，其4年後外傘頂洲乾砂洲面積即有增大趨勢，濁水溪砂石開採對外傘頂洲之變遷具顯著的影響。研究顯示濁水溪流域懸浮載輸砂量之變化，其後8年內會影響到外傘頂洲面積的變化<ref name=張憲國>[http://www.tsoe.org.tw/downloads/thesis/2005D1.pdf  張憲國《以衛星影像探討外傘頂洲的海灘變遷》第27屆海洋工程研討會論文集、 國立中興大學 2005年12月] {{zh-tw icon}}</ref>。

=== 夏天淤積冬天侵蝕 ===
外傘頂洲的沙灘具季節性變化，夏天（5～9月）會淤積而冬天（10～隔年4月）會受到侵蝕。但是若北港溪在該年有足夠沙源流至外海，則南向水流會造成該年外傘頂洲之沙灘不減反增<ref name=張憲國B/>。

==建築==
外傘頂洲上現有燈塔一座及漁民陳姓老夫妻自行搭建之「竹篙寮」，2012年已搬離。
===塭港堆燈塔===
{{main|塭港堆燈塔}}
塭港堆燈塔又稱外傘頂洲燈塔，不同於一般水泥結構燈塔，採鋼架結構。所處海域沙洲甚多，[[船舶|船舶]]稍有不慎即易擱淺，故而設立。1914年首建，高度15公尺，裝用五等[[電石氣燈|電石氣閃光燈]]，每2秒一閃，[[光力|光力]]800燭光，1920年傾倒，隔年重建，塔形燈器與原塔同，後因沙洲移動再次流失。 1957年另建20.7公尺高方形鋼架塔，初用五等電石氣燈，光力800燭光，後於1965年改裝直流電燈，光力增至2600燭光，每3秒一閃。 1974年因沙洲移動，並為改善目標，移至附近新址，改建為方錐形鋼架塔，換用四等直流電燈，光力增至3500燭光。 1977年改進設備，光力加強為10000燭光，塔高32.4公尺，燈高35.5公尺，[[公稱光程|公稱光程]]（nominal range）14.3浬。另設有多向金屬雷達反射器<ref>{{cite web
|url=http://wshnt.kuas.edu.tw/sundayasp2/g4/new_page_9.htm
|title=塭港堆燈塔
|author=黃文祥
|accessdate=2007-11-27
|language=zh-tw
}}</ref>。

自1914年首建以來，塭港堆燈塔因沙洲漂移、海水浸蝕等因素而多次傾斜倒塌，前後歷經七次重建，是全臺灣燈塔中，重建次數最多者<ref>{{cite news|title=外傘頂洲沙灘｜首頁＞景點＞景點快搜|url=https://swcoast-nsa.travel/zh-tw/attraction/details/309|publisher=|author=雲嘉南濱海國家風景區官方網站|date=2018-07-04|accessdate=2019-01-23|language=zh-tw}}</ref>。

== 經濟 ==
外傘頂洲與嘉義縣海岸之間的水域被稱為'''東石潟湖'''，因外傘頂洲的屏障形成平靜的廣大淺水[[海域|海域]]，目前是台灣主要[[蚵仔|蚵仔]]（牡蠣）[[水產養殖|養殖]]區之一。據嘉義縣政府統計，嘉義沿海蚵仔年產量約7000多[[公噸|公噸]]，約佔全臺灣產量50%，產值[[新臺幣|新臺幣]]9億1000多萬元，且全臺灣90%的蚵又都在東石[[食品加工|加工]]取肉，東石鄉民小至小學生、大至70-80歲老人都會剖蚵，為地方重要經濟來源<ref>{{cite news|title=台朔將協助嘉義沿海蚵殼處理問題可望獲得重大突破|url=http://www.cyhg.gov.tw/news1.asp?news_id=5280|publisher=嘉義縣政府新聞室縣政行銷課|date=2005-08-23|accessdate=2007-11-28|language=zh-tw|deadurl=yes|archiveurl=https://web.archive.org/web/20051201184418/http://www.cyhg.gov.tw/news1.asp?news_id=5280|archivedate=2005-12-01}}</ref>。此外，外傘頂洲的觀光產業也很發達，可從[[嘉義縣|嘉義縣]][[東石鄉|東石鄉]][[東石漁人碼頭|東石漁人碼頭]]乘漁筏前往；據嘉義縣政府統計，截至2015年，每年約有10萬人次到外傘頂洲觀光<ref>{{cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20150608/625080/|title=外傘頂洲將成消失國土 每年湧進10萬遊客|agency=蘋果日報|date=2015-06-08|language=zh-tw}}</ref>。

1990年代開始，有人建議將東石潟湖完全填平成[[海埔新生地|海埔新生地]]，因為最大可填平300[[平方公里|平方公里]]的土地；但由於擔心對當地生態造成破壞，加上當地偏僻遠離市區，經濟效益仍待商榷，故未實施。

== 爭議 ==
外傘頂洲南面是內海淺灘，有利於貝類養殖，也導致了嘉義東石、雲林口湖兩地漁民漁產權之爭。

1978年11月30日，嘉義縣長[[凃德錡|凃德錡]]曾率員利用假日前往外傘頂洲實地勘察以利行政劃分之商議<ref>{{cite journal|title=外傘頂洲管轄權問題|author=劉兩生|date=1979年5月|journal=嘉義文獻|volume=10|pages=40-48}}</ref>；多年來隨著沙洲南移，兩縣管轄權之爭愈趨激烈。2001年，雲林縣長[[張榮味|張榮味]]任內，曾指派副縣長[[高孟定|高孟定]]率員至外傘頂洲釘上「雲林縣口湖鄉蚶寮126之201號」門牌宣示管轄權<ref>{{cite news|url=http://news.ltn.com.tw/news/local/paper/612047|title=〈中部〉《外傘頂洲》竹篙寮門牌失蹤 雲林將重掛|author=廖淑玲|date=2012-09-03|agency=自由時報|language=zh-tw}}</ref>。

2006年，嘉義縣府以「沙洲已經漂流到東石外海14公里」為由，向中央政府提出將其從雲林改劃由嘉義管轄，未獲同意。2007年6月，嘉義縣長[[陳明文|陳明文]]率縣府、議會人員及學者前往外傘頂洲會勘，並解釋此行與管轄權歸屬無關，為配合中山大學海洋科學院研究，透過評估分析外傘頂洲固沙保育及養殖海籍圖等現況，由學者專家從學術面，探討國土復育可能性。此舉引發了雲林縣議員的緊張，在議會上公開要求雲林縣府採積極作為，對此，雲林縣長蘇治芬表示，外傘頂洲現已漂至嘉義縣外海，雲林縣府的行政管轄權有實際上的行使困難，要如何防止沙洲消失，涉及跨縣市問題，中央部會應出面整合。[[中華民國內政部|內政部]]則回應，正依《地方制度法》檢討修正行政區劃法草案，外傘頂洲的管轄問題，將等行政區劃法完成立法，再行妥善規劃<ref>{{cite news|title=地址雲林縣 漂到嘉義縣 管轄起爭議|date=2007-06-01|publisher=聯合晚報|author=陳永順、魯永明、黃宣翰}}</ref>。經[[監察院|監察院]]2015年做成的報告指出，2012年行政院雖將涉及外傘頂洲區劃問題的《行政區劃法》草案送交立法院審議，但直至2015年仍未進入逐條審查<ref>{{cite web|url=http://www.cy.gov.tw/sp.asp?xdUrl=./di/message/message_1.asp&ctNode=903&msg_id=5335|title=國內海岸疑不當設置消波塊及海堤 監察院促請經濟部、內政部檢討改進|work=中華民國監察院|date=2015-09-02|language=zh-tw}}</ref>。

== 事故 ==
1955年2月20日，21名東石鄉型厝村漁民搭乘大竹筏到外傘頂洲海域捕魚，21人均不明原因[[失蹤|失蹤]]；翌年11月20日，共乘一艘竹筏的4名漁民再傳失蹤。事故發生後，村民集體乘竹筏出海找尋，並未找到任何遺物，事故究竟如何發生、是否有人生還等細節均成謎。有10名漁民的家屬，50餘年來仍抱著親人生還的希望，直至2007年11月25日，終決定向村幹事蔡信茂、村長顏銘源求助，希望辦理[[死亡|死亡]]宣告、除戶事宜。蔡文欽胞弟蔡文雄於事故中失蹤，他表示，當年其弟十八歲，尚未娶妻，迄今政府所發放敬老津貼名冊上都有其弟名字，徒增傷感，才決定辦理死亡宣告<ref>{{cite news|title=外傘頂洲海難 辦死亡宣告|author=蔡民一|publisher=自由時報|url=http://www.libertytimes.com.tw/2007/new/nov/26/today-so1.htm|date=2007-11-26|language=zh-tw|deadurl=yes|archiveurl=https://web.archive.org/web/20071127152226/http://www.libertytimes.com.tw/2007/new/nov/26/today-so1.htm|archivedate=2007-11-27}}</ref>。
