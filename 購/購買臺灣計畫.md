'''購買台灣計畫'''是由[[美國|美國]]駐日公使[[湯森·哈里斯|湯森·哈里斯]]1854年向[[美國國務院|美國國務院]]提出，主張美國應該向[[清廷|清廷]]購買[[台灣|台灣]]，來鞏固在[[亞洲|亞洲]]的[[商業|商業]]利益，但因當時美國國內[[黑奴|黑奴]]問題日益嚴重，且美國也無意干預台灣事務，此案最終遭到否決。

== 舊帝國主義下的台灣 ==
自15世紀以來，就有日本[[海盜|海盜]]及中國商人在此貿易，直到[[歐洲人|歐洲人]]東來後，因其位置符合當時西方國家作為在亞洲[[貿易|貿易]]及傳教的據點，[[福爾摩沙|福爾摩沙]]遂成為競爭的焦點，台灣先後受到[[荷蘭|荷蘭]]及[[西班牙|西班牙]]的統治。直到1683年，台灣正式納入清廷版圖後，台灣的國際競爭時期暫時告一段落。

== 新帝國主義下的台灣 ==
19世紀以後，西方國家歷經[[工業革命|工業革命]]的洗禮後，挾持著船堅炮利來到亞洲做貿易，當時清廷閉關自守，而遂引發两次[[鸦片战争|鴉片戰爭]]。戰爭期間，英軍曾派遣軍艦進犯[[基隆|基隆]]，但最後仍被守軍擊退（詳見[[大安之役|大安之役]]）。

爾後，清廷因與英、法兩國之間爆發[[英法聯軍之役|英法聯軍之役]]所簽訂的《[[天津條約|天津條約]]》，台灣必須開放[[基隆市|雞籠]]（今基隆市）、[[淡水區|滬尾]]（今淡水區）、[[安平區|安平]]（今[[台南市|台南市]]）、[[高雄市|打狗]]（今高雄）四個港口作為[[通商口岸|通商口岸]]，西方勢力再度伸入台灣。

1860年，台灣正式開港通商後，[[樟腦|樟腦]]的產量出口量大幅增加，所產生的鉅額的利潤，引起了西方[[列強|列強]]的注意。

== 背景 ==
十九世紀初，[[美國|美國]]對華貿易日益增加，來華商務船隻亦日益增多，經過[[台灣海峽|台灣海峽]]（當時西方稱為「[[福爾摩沙海峽|福爾摩沙海峽]]」）常常發生[[風災|風災]]觸礁之事，致使美方多次向清廷抗議，美商[[奈伊|奈伊]]（Gideon Nye）向美國駐華大使[[伯駕|伯駕]]建議控制台灣才能確保航行安全，而且台灣亦可作為航行補給站，此一建議[[伯駕|伯駕]]在贊同下，美國[[東方艦隊|東方艦隊]]司令[[培里|培里]]命赴台調查[[雞籠|雞籠]]煤炭以及在台失蹤沈船失蹤的船員，調查之後認為煤炭樣本品質優良、價廉物美<ref>{{Cite web|title=【民報】【專文】 另位Harris催促美國合併台灣|url=https://www.peoplenews.tw/news/76a123d7-0643-45ca-8cd8-60c656a7708f|work=www.peoplenews.tw|accessdate=2019-02-08|language=zh-TW}}</ref>，台灣在軍事戰略及貿易航線上均對美國意義重大，於是與[[伯駕|伯駕]]一同向[[美國國務院|美國國務院]]建議採取行動，但美國考慮會造影響在華商業利益因而作罷。

== 湯森·哈里斯購買台灣計畫 ==
美國駐日大使[[湯森·哈里斯|湯森·哈里斯]]提出具體的計畫並於1854年向[[美國國務院|美國國務院]]提案。計畫中指出台灣先後有[[荷蘭|荷蘭]]、[[西班牙|西班牙]]、[[鄭成功|鄭成功]]、[[清廷|清廷]]統治，認為轉移其[[所有權|所有權]]是有根據，而且在[[地理位置|地理位置]]上不但可作為通往東方補給站，亦是美國對華貿易的必經門戶，並指出台灣內部政治及社會的弱點，認為消極治理加上官吏[[貪汙|貪汙]]嚴重而招致民怨，且清廷僅能有效[[統治|統治]]西半部地區，能帶給清廷[[稅收|稅收]]也不多，認為以上幾點是美國應該有機會購買台灣的理由。

== 結果 ==
美國雖然對台灣深感興趣，但礙於國內黑奴造成的問題，也顧及會引起周圍其他列強抗議而影響其在華的利益考量下而否決。
