'''榎梓'''（えのき あづさ、1988年3月16日<ref>{{Cite web |url=http://gree.jp/kataoka_azusa/introduction |title=片岡梓 公式ブログ プロフィール - GREE |accessdate=2012-10-08}}</ref> - ），出生於[[日本|日本]][[千葉縣|千葉縣]]，[[日本|日本]][[女性|女性]][[配音員|配音員]]。本名・前藝名：'''片岡梓'''（片岡あづさ）。一直以來都是[[自由工作者|自由工作者]]<ref name="freelance">{{Cite web|date=|url= http://ameblo.jp/adusa-kinoko/entry-11522183737.html|title= ☆★ご報告★☆|work=|author= 片岡あづさの繁殖日記〜胞子振り撒いちゃうゾッ☆〜|publisher=[[サイバーエージェント|サイバーエージェント]]|accessdate= 2013-05-01}}</ref>。於2013年8月1日，成為{{link-ja|砂岡事務所|砂岡事務所}}<ref name="office">{{Cite web|date=|url= http://ameblo.jp/enoki-adusa/entry-11584251896.html|title= 片岡あづさ改め！|work=|author= 胞子ふりまき隊。|publisher=サイバーエージェント|accessdate= 2013-08-01}}</ref>所属。

== 來歷 ==
14歳開始入了配音員培訓所上配音課程。

[[波麗佳音|波麗佳音]]主催的藝能Audition「VS Audition 2003」裡，決賽入選，值得讚揚。

2004年中的電視動畫『[[型仔湊女記|型仔湊女記]]』，配演步的朋友並出道。

== 人物 ==
暱稱（[[綽號|綽號]]），是「あづ子」、「あづちゃん」、「あづあづ」、「あづっち」「オマンキー（[[屄|屄]]）」。

== 插曲 ==
在各種雜誌、網站等，片岡梓的「づ」字被寫成「ず」字的「片岡あずさ」錯誤標記的事件引起。
