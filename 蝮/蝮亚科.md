'''蝮亚科'''（学名''Crotalinae''）是[[蝰蛇科|蝰蛇科]]的一个亚科，统称'''蝮蛇'''，其主要特征是在眼与鼻孔之间具颊窝。除少數為[[卵生|卵生]]外，絕大部分品種爲[[卵胎生|卵胎生]]。

== 其代表蛇岛蝮简介 ==
详见：[[蛇島蝮|蛇島蝮]]<br />
其代表为蛇岛蝮，灰褐色，是唯一一种不处于濒危状态的毒蛇。分布于[[蛇島|蛇島]]。

== 分佈 ==
中低海拔樹林樹腐葉裡及近水邊潮溼地，常於樹根部及石塊邊盤成一團，人往往因休息坐於樹石被咬；在台灣稱為[[百步蛇|百步蛇]]，中國稱為[[七步蛇|七步蛇]]、[[五步蛇|五步蛇]]。

== 食性 ==
食量大，食性廣，以鳥為主食，兼食蛙、鼠、蜥蜴、和小型无毒蛇，如[[游蛇科|游蛇科]]的成員和[[裸鱗蛇|裸鱗蛇]]、[[針尾蛇|針尾蛇]]和[[盲蛇|盲蛇]]等。

== 性格 ==
性格较温顺，攻击性比[[眼镜蛇|眼镜蛇]]低，但比蝰亚科的成员攻击性高。毒牙较短，属于原始和进步之间的蝰蛇科成员。
