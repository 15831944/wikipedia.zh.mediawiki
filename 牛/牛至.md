'''牛至'''（[[學名|學名]]：''{{lang|la|Origanum vulgare}}''）（英文：{{lang|en|oregano}}、{{lang|en|wild marjoram}}）又名'''滇香薷'''、'''奧勒岡葉'''、'''披薩草'''、'''香芹酚'''、'''野馬鬱蘭'''，是[[唇形科|唇形科]][[牛至属|牛至属]]中的一種植物。

== 形态 ==
[[多年生植物|多年生]][[草本|草本]]植物，高达60厘米，全株被有微柔毛，有芳香；方形茎；卵形或矩圆状卵形的叶子对生，有腺点和柔毛；夏季开紫红色至白色的唇形花，成伞房状圆锥花序；卵圆形小[[坚果|坚果]]。

== 分布 ==
原生於歐洲[[地中海|地中海]]沿岸地區；生于山坡草地、路旁。栽培需要排水良好的[[土壤|土壤]]及充足的日照。

== 用途 ==
牛至全草可提取芳香油，也作藥用，可紆緩傷風感冒病徵，但味道極不好喝<ref>《真情康翠瑩》，2018-04-11 13:55，am1430</ref>；做[[烹調|烹調]]用時，常與[[番茄|番茄]]、[[乳酪|乳酪]]搭配；牛至與[[羅勒|羅勒]]是給予[[義大利菜|義大利菜]]獨特香味的兩大用料。因時常撒在[[披薩餅|披薩餅]]上，所以又名披薩草。
