'''牛頭天王'''（ごずてんのう）是日本[[神佛習合|神佛習合]]的神。[[京都|京都]][[東山區_(日本)|東山]][[祇園|祇園]]、[[播磨國|播磨國]][[廣峰山|廣峰山]]鎮座之神，和[[蘇民將來|蘇民將來]]傳說的'''[[武塔神|武塔天神]]'''視為同一神。[[印度|印度]][[释迦牟尼|釋迦牟尼]]的誕生地[[祇園精舍|祇園精舍]]的守護神，稱作'''祇園神'''的[[祇園信仰|祇園信仰]]之神。在[[陰陽道|陰陽道]]和'''天道神'''視為同一神。神佛習合中和[[藥師如來|藥師如來]]、[[素戔嗚尊|素戔嗚尊]]習合。從現在稱作[[八坂神社|八坂神社]]的感神院祇園社[[勸請|勸請]]，在全國的'''祇園社'''、'''天王社'''祭祀。

== 概要 ==
牛頭天王是京都祇園社（現[[八坂神社|八坂神社]]）的祭神。

牛頭天王是[[平安京|平安京]]祇園社的祭神，所以也稱作祇園[[天神|天神]]，從[[平安時代|平安時代]]作為行疫神受到信仰<ref name=uda>菟田「牛頭天王」（2004）</ref>。[[平安時代|平安時代]]末期，民眾受傳說影響為了[[瘟神|疫病神]]退散，出動[[花笠|花笠]]、[[山鉾|山鉾]]在市中巡遊鎮祭，就是[[祇園祭|祇園祭]]的起源<ref name="gion">真弓編『祇園信仰事典』（2002）</ref>。此傳說裡，印度祇園精舍的守護神牛頭天王某一夜外出至女人家的途中，由於天色昏暗想借宿人家一晚，他先跟富裕的巨旦將來求助，拒於門外，反而是巨旦的弟弟、貧窮的蘇民將來讓他到家裡住宿一晚。<ref name=":0">{{Cite book|title = 京都原來如此：千年古都的風俗、習慣、飲食文化解謎|last = 博學堅持俱樂部|first =|publisher = 遠足文化事業股份有限公司|year = 2015|isbn = 9789869189620|location =|pages = 第212到213頁}}</ref>八年之後，某天牛頭天王又突然出現在蘇民家中，要他製作一個茅輪掛在門簷上。當天晚上起，瘟疫蔓延，只有遵照牛頭天王指示的蘇民一天逃過死劫，最後蘇民與其子孫開創了新國家。<ref name=":0" />

牛頭天王之子[[八王子權現|八王子權現]]、[[眷屬|眷屬]]也被視為掌管疫病之神<ref name="gion"/>。

=== 神佛分離 ===
[[明治|明治]]政府的[[神佛分離|神佛分離]]政策下，以牛頭天王為祭神的祇園社、天王社等神社，祭神全部改成素戔嗚尊，感神院祇園社也遭到廢寺，改為八坂神社。

== 祭祀牛頭天王的寺院 ==
現在也有少數祭祀牛頭天王的寺院逃過廢佛毀釋而繼續存在。
