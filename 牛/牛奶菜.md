'''牛奶菜'''（[[学名|学名]]：{{lang|la|''Marsdenia sinensis''}}）是[[夹竹桃科|夹竹桃科]][[牛奶菜属|牛奶菜属]]的植物，是中国的特有植物。它分布在[[中国大陆|中国大陆]]的[[四川|四川]]、[[浙江|浙江]]、[[广东|广东]]、[[江西|江西]]、[[湖南|湖南]]、[[湖北|湖北]]、[[广西|广西]]、[[福建|福建]]等地，生长于海拔300米的地区，多生长在山谷疏林中，目前尚未由人工引种栽培。

== 别名 ==
三百银、婆婆针钱包（浙江）
