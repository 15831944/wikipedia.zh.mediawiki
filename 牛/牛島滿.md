'''牛島滿'''{{BD|1887年|7月31日|1945年|6月22日|catIdx=Ushijima Mitsuru}}，是[[第二次世界大戰|第二次世界大戰]]結束前[[沖繩島戰役|沖繩島戰役]]島上[[日軍|日軍]][[第32軍_(日本陸軍)|第32軍]][[指揮官|指揮官]]。戰前日軍精銳[[第9師團|第9師團]]2萬兵力自[[琉球|琉球]]被抽調至[[台灣|台灣]]，導致牛島僅餘12萬餘兵力，[[美軍|美軍]]則以18萬兵力登島進攻，但牛島仍運用有限的兵力據險死戰。雖然牛島最終被登陸美軍擊敗而[[切腹|切腹]]自盡，但此役使美軍付出高昂的代價。而此役日軍所發動的神風攻擊和玉碎戰，被認為是美國後來決定對日本使用[[原子彈|原子彈]]的因素。

美軍總傷亡人數大約是65,000人，其中12,513人死亡或失蹤。美軍陣亡的人數比[[硫磺島戰役|硫磺島戰役]]和[[瓜達爾卡納爾島戰役|瓜達爾卡納爾島戰役]]的總死亡人數多了兩倍以上，此數字還不包含幾千名在日後間接死亡（由傷口和其他原因導致）的軍人。攻佔沖繩島的美軍司令[[小西蒙·玻利瓦尔·巴克纳|巴克納]]中將也在此戰遭到日軍炮擊陣亡，成為第二次世界大戰美軍戰歿最高階將領。而日軍除了約7,000人被俘虜外，其餘約100,000名守軍則盡數陣亡，付出了比美軍高十倍的陣亡代價。

== 初年 ==
牛島滿是[[九州|九州]]南端[[鹿兒島縣|鹿兒島縣]][[鹿兒島市|鹿兒島市]]人。1937年[[日本侵華戰爭|日本侵華戰爭]]開始，他在[[中國|中國]]晉升為[[少將|少將]][[旅團長|旅團長]]。1939年起他晉升為[[中將|中將]]並擔任日軍精銳部隊[[第11師團|第11師團]][[師團長|師團長]]，在華中地區及[[緬甸|緬甸]]大戰盟軍數場。1941年起他被派回日本擔任[[後備役|後備役]]軍官動員訓練學院院長。1943年他晉升為日軍陸軍官校校長到1944年。之後前往琉球擔任戰事指揮官；並因戰敗自殺。

== 年譜 ==
*1908年（明治41年）5月 - [[日本陆军士官学校|陸軍士官學校]]畢業（20期）。
**12月 - 升[[少尉|少尉]]。[[近衛歩兵第4聯隊|近衛歩兵第4聯隊]]付。
*1911年（明治44年）12月 - 升[[中尉|中尉]]。
*1916年（大正5年）5月 - [[日本陸軍大學校|陸軍大學校]]畢業（28期）。
*1918年（大正7年）8月 - [[西伯利亞出兵|西伯利亞出兵]]（～1920年8月）。西伯利亞派遣軍野戰交通部參謀。
**12月 - 升[[大尉|大尉]]。
*1919年（大正8年）4月 - [[近衛步兵第4聯隊|近衛步兵第4聯隊]]中隊長。
*1920年（大正9年）8月 - 陸軍步兵學校教官。
*1924年（大正13年）3月 - 升[[少佐|少佐]]。[[步兵第43聯隊|步兵第43聯隊]]大隊長。
*1925年（大正14年）4月 - [[步兵第45聯隊|步兵第45聯隊]]付（配屬[[鹿兒島縣立鶴丸高等學校|第一鹿兒島中學校]]）。
*1928年（昭和3年）3月 - 歩兵第23連隊附。
**8月 - 升[[中佐|中佐]]。
*1930年（昭和5年）8月 - [[下關要塞|下關要塞]]參謀。
*1932年（昭和7年）8月8日 - 升[[大佐|大佐]]。[[陸軍戸山學學校|陸軍戸山學學校]]教育部長。
*1933年（昭和8年）3月18日 - [[陸軍省|陸軍省]]高級副官。
*1936年（昭和11年）3月28日 - [[步兵第1聯隊|步兵第1聯隊]]長。
*1937年（昭和12年）3月1日 - 升[[少將|少將]]。歩兵第36旅團長。
**12月5日 - [[陸軍預科士官學校|陸軍預科士官學校]]幹事。
*1939年（昭和14年）3月9日 - 陸軍預科士官學校校長。陸軍戸山學校校長（兼任）。
**8月1日 - 升[[中将|中将]]。
**12月1日 - [[第11师团|第11师团]]長。
*1941年（昭和16年）10月15日 - [[陸軍公主嶺學校|陸軍公主嶺學校]]校長。
*1942年（昭和17年）8月8日 - 陸軍士官學校校長。
*1944年（昭和19年）9月26日 - [[第32軍_(日本陸軍)|第32軍]]司令官。
*1945年（昭和20年）5月22日 - 決定南部撤退。
**6月23日 - 清晨、於摩文仁丘洞窟内與[[長勇|長勇]]參謀長自決（同日為[[沖繩縣|沖繩縣]][[慰靈日|慰靈日]]。據說在22日自決）。同日特別晉升為陸軍[[大將|大將]]。
