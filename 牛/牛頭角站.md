'''牛頭角站'''（{{lang-en|'''Ngau Tau Kok Station'''}}）是[[港鐵|港鐵]][[觀塘綫|觀塘綫]]的一個架空車站，位於[[九龍東|九龍東]][[觀塘區|觀塘區]][[觀塘道|觀塘道]]及[[牛頭角道|牛頭角道]]之間，鄰近[[觀塘商貿區|觀塘商貿區]]，於1979年10月1日啟用。車站以淺綠色為主要色調。

===洗手間===
由於牛頭角站周邊200米範圍內並無設置公眾[[廁所|洗手間]]，故此[[香港鐵路有限公司|港鐵公司]]於[[2008年|2008年]]1月決定在該站近觀塘道橋底的地面興建洗手間。洗手間已於2011年1月完成工程並隨即開放予乘客使用。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Ngau Tau Kok Station 2017 10 part2.jpg | caption1 = 1號月台（往調景嶺方向）
}}
由於牛頭角站接近觀塘道及啟褔道往紅磡、土瓜灣和尖沙咀方向的行車天橋，故此月台不設有樓梯通往路軌，而需要途徑車站機房進出路軌，因此高架橋和行車天橋之間平行的一段設有鐵網和[[鐵絲網|鐵絲網]]（刀片刺網），以防止逃票的乘客由行車天橋進入本站月台。

雖然車站月台大部分是直的，但是1號月台（[[調景嶺站|調景嶺]]方向）的車頭部分稍微向內彎曲，因此該處的空隙跟[[觀塘站|觀塘站]]的接近。

該站是一天中最多不載客列車經過的車站，因為除了平日繁忙時間後大量西行列車經此站2號月台返回[[港鐵車廠#九龍灣車廠|九龍灣車廠]]外，還有清晨及深夜時段部分西行班次會於觀塘站開始或停止服務，有關列車將不會服務此車站。

===出入口===
牛頭角站大堂設有A和B兩個出入口，而B出口前方則為行人隧道，貫通牛頭角商貿區和花園大廈一帶住宅區，港鐵亦為行人隧道各出入口分別編上B1至B6的出口編號。此行人隧道由港鐵管理，但該六個隧道出口並不屬於車站範圍，因此每晚港鐵收車後，行人隧道也會無間斷開放予公眾使用。

==鄰接車站==
牛頭角站是觀塘綫的中途站，乘客可在此乘坐往[[黃埔站_(香港)|黃埔]]或[[調景嶺站|調景嶺]]列車，往返[[九龍|九龍]]及[[新界|新界]]等地。

==利用狀況==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Ngau Tau Kok Station 2014 03 part4.JPG | caption1 = 由月台往大堂的扶手電梯
| image2 = NTK-MTR-train.jpg | caption2 = 一列停靠於車站的[[K-Train|K-Train]]列車
}}
牛頭角站啟用初期，除了為鄰近住宅區的乘客提供服務外，鄰近工廠林立的工業區亦是乘客一大來源。基於90年代香港工業式微，牛頭角站在非繁忙時間人流量並不高。直至[[觀塘商貿區|觀塘商貿區]]的[[創紀之城|創紀之城]]建成後，周邊亦興建更多商業樓宇，導致人流大幅增長，然而車站大堂面積較小，且沒有空間擴建，導致於早上及傍晚成為嚴重擠迫的「重災區」。未來更多的商業樓宇落成，預計使用量將會繼續上升。

==意外事件==
[[File:Ngau_Tau_Kok_Station_Escalator_accident_201902.jpg|220px]]
2019年2月28日早上9時許，車站一條由月台往大堂的扶手電梯發生意外，扶手梯旁板有鐵片突然彈出，9名乘客腳部被鎅傷。事後港鐵已即時停止扶手梯運作和圍封該處，同時有警員及港鐵人員維持秩序。有乘客均對此感到擔心。<ref>{{cite news |title=港鐵牛頭角站扶手電梯意外9傷者全出院　受傷乘客：只刮傷小腿 |url=https://topick.hket.com/article/2282135/%E6%B8%AF%E9%90%B5%E7%89%9B%E9%A0%AD%E8%A7%92%E7%AB%99%E6%89%B6%E6%89%8B%E9%9B%BB%E6%A2%AF%E6%84%8F%E5%A4%969%E5%82%B7%E8%80%85%E5%85%A8%E5%87%BA%E9%99%A2%E3%80%80%E5%8F%97%E5%82%B7%E4%B9%98%E5%AE%A2%EF%BC%9A%E5%8F%AA%E5%88%AE%E5%82%B7%E5%B0%8F%E8%85%BF |accessdate=2019-03-02 |work=香港經濟日報 |date=2019-02-28}}</ref>

===早期系統===
{{see also|香港地鐵}}
根據1970年的《集體運輸計劃總報告書》，彩虹↔觀塘為第4期工程，並為「早期系統」中優先發展的首4期工程之一，故此車站當年曾被考慮興建。

===修正早期系統===
{{see also|1=觀塘綫|l1=修正早期系統}}
1975年，「修正早期系統」動工，1979年落成啟用，除了小規模的翻新外，牛頭角站至今沒有過大的變動，一直為[[觀塘綫|觀塘綫]]的中途站。

===扶手電梯預留位===
往調景嶺方向車頭及往黃埔方向車尾的位置同時預留空間方便日後加設多一條扶手電梯往返大堂及月台，但1979年10月1日啟用自今，未有任何加裝工程。

===月台閘門===
現時牛頭角站所有月台已完成裝上[[月台閘門|月台閘門]]，並已投入運作。該月台閘門與現時[[欣澳站|欣澳站]]及[[迪士尼站|迪士尼站]]所採用的月台閘門為同一設計，只有現時[[月台幕門|月台幕門]]的一半，約高1.5米。整個計劃於平日深夜終止服務的時間加建。車站與[[九龍灣站|九龍灣站]]及[[荃灣綫|荃灣綫]][[葵芳站|葵芳站]]、[[葵興站|葵興站]]屬於第二期進行月台閘門加裝工程，已於2011年8月完工，本站所有月台的月台閘門加裝工程經已完工。

===車站改善工程 - 現代化工程===
港鐵為配合現代化設計，於2012年6月起把舊有的灰黑色[[紙皮石|馬賽克磚]]的牆身和柱身，改成白色及淺綠色人造纖維板。目前已完成所有月台支柱翻新工程，並正進行牆壁翻新工程。
