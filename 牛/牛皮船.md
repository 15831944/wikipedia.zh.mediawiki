'''牛皮船'''，是[[西藏|西藏]][[青藏高原|青藏高原]][[藏族|藏族]]人的水上運輸工具。<ref name="青藏高原－衣食住行">[http://www.itpcas.cas.cn/kxcb/kpwz/200811/t20081108_2034943.html 中國科學院科學傳播局〈青藏高原－衣食住行〉]</ref>

== 歷史 ==
牛皮船的歷史源遠流長。根據文獻記載，它的歷史可追溯至[[唐朝|唐朝]]。當時西藏青藏高原[[東女國|東女國]]的人以牛皮船來渡河。{{noteTag|資料來自[[後晉|後晉]][[劉昫|劉昫]]《舊唐書‧東女國傳》卷197載：「其王所居名康延川，中有弱水南流，用牛皮船以渡。」}}而[[清代|清代]]史學家[[趙翼|趙翼]]在其著作《陔餘叢考‧牛皮船》亦指出「是牛皮為船，由來久矣，皆出於番俗也。」這反映了牛皮船在古代已經是外族人使用的一種交通工具，而且歷史悠久，非一時的產物。由於[[中國|中國]][[四川|四川]][[大金川|大金川]]江自北向南的終年河流流量大，加上河面寬闊，要興建橋樑和架索都具有困難。若使用木船時，其體積龐大笨重，必須有固定的口岸才能停泊。就此不能夠作為該青藏高原的交通工具。相反，牛皮船輕便靈活，船工可把它任意搬運。所以適用於航行交通之用。這比較上配合河流流量大的地理形勢。

== 船隻結構 ==
牛皮船以木架作龍骨。<ref name="藏族">[http://www.gov.cn/test/2006-04/11/content_250767.htm 國家民委網站〈藏族〉，2006年4月11日]</ref>船身長3米多，高l米、寬2米左右，重30至40公斤。一般而言，工人需要至少兩張口徑六尺左右的牛皮、加上藤條和樹脂膠縫製牛皮船。船工完成工序後會在船裏塗上一層菜籽油，從而令牛皮船更為柔韌耐用。至於載客量方面，每隻牛皮船由一個人攏渡，最多可盛載5至10人左右。<ref>{{web reference|URL=http://mag.cnyes.com/Content/20090526/d4f15191f79140fb9a956f02bab476a3_1.shtml|title=西藏絕版牛皮船製作工藝|work=西藏旅遊 第104期|year=2009年|date=5月26日}}</ref>{{noteTag|資料出自[[清代|清代]][[李心衡|李心衡]]《金川鎖記》中對牛皮船容量的描述：「用極堅樹枝作骨，蒙似牛革，形圓如 。一人持漿，中可坐四五人，順流而下，疾於奔馬，頃刻達百里。」。}}<ref name="試論我國行旅民俗">[http://hk.crntt.com/crn-webapp/cbspub/secDetail.jsp?bookid=19963&secid=20046 中國評論學術出版社〈試論我國行旅民俗〉]</ref>

== 傳說 ==
有言東方國的女官[[高霸|高霸]]在黎明乘坐牛皮船從格爾丹寺沿河而下。她到達噶延時正值太陽初升，便向女王報告路上考察的情况。女王聽後讚賞高霸行動快捷，並下令多造牛皮船以充實軍備。牛皮船之後成為女王十分喜愛的交通工具。在随後的千百年內，東女國的後裔依靠牛皮船往來大金川口岸。現今在金川江面上可以看到[[藏族|藏族]]人乘坐牛皮船捕魚。而漂流探險者也以它漂流大金川。{{noteTag|見北宋李昉《太平廣記》卷481 蠻夷二「東女國」段。}}
