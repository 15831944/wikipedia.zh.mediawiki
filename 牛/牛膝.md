'''牛膝'''（[[学名|学名]]：{{lang|la|''Achyranthes bidentata''}}）为[[苋科|苋科]][[牛膝属|牛膝属]]的植物。

== 形态 ==
多年生[[草本|草本]]；圆柱形根；茎有棱角，节部膝状膨大；椭圆形至披针形的叶子对生；夏秋开绿色小花，穗状花序。

== 分布 ==
分布于[[非洲|非洲]]、[[俄罗斯|俄罗斯]]、[[越南|越南]]、[[印度|印度]]、[[马来西亚|马来西亚]]、[[菲律宾|菲律宾]]、[[朝鲜|朝鲜]]以及[[中国大陆|中国大陆]]除东北外的各地等，生长于海拔200米至1,750米的地区，常生长在山坡林下，目前尚未由人工引种栽培。

== 别名 ==
百倍、牛茎、脚斯蹬、铁牛膝、杜牛膝、怀牛膝、怀夕、真夕、怀膝、土牛膝、淮牛膝、红牛膝、牛磕膝等。
== 用药部位 ==
根

== 性味 ==
性平，味苦、甘、酸。歸肝、腎經。

== 异名 ==

* Achyranthes bidentata Bl. var. japonica Miq.
* Achyranthes japonica (Miq.) Nakai
* Achyranthes ogatai Yamamoto
