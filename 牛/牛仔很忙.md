《'''牛仔很忙'''》是周杰伦2007年专辑我很忙的主打歌，由黄俊郎填词，是一首美国西部乡村民谣风格的歌曲，体现了周杰伦曲风上的变化 <ref name="niu">{{cite web |url= http://ent.ifeng.com/music/review/200711/1105_39_285064.shtml|title= 《牛仔很忙》：改变就等于进步？|accessdate=2009-07-18 |work= [[凤凰网|凤凰网]]|publisher= [[凤凰卫视|凤凰卫视]]|date= 2007-11-05}}</ref>。这首歌在专辑推出后保持了一周华语歌曲排行榜首地位<ref>{{cite web |url= http://ent.cctv.com/20071114/100263.shtml|title= 周杰伦新歌《牛仔很忙》占据华语歌曲排行榜首|accessdate=2009-07-18 |work= [[央视国际|央视国际]]|publisher= [[CCTV|CCTV]]|date= 2007-11-14}}</ref> <ref name="yue">{{cite web |url= http://ent.cctv.com/20071122/103992.shtml|title= 乐评人批周杰伦新专辑 《我很忙》粗制太平庸 |accessdate=2009-07-18 |work= [[央视国际|央视国际]]|publisher= [[CCTV|CCTV]]|date= 2007-11-22}}</ref>，并在2008年度TVB8金曲榜颁奖典礼上获得最佳音乐录音带演绎奖。

== 批评 ==
在专辑推出后，遭到众多乐评人和歌迷批评 <ref name="niu"></ref> <ref name="yue"></ref>，尤其是作为主打歌的牛仔很忙。乐评人批评概念先行，音乐不突出<ref name="yue"></ref>，用周杰伦擅长的R&B唱腔来演绎乡村民谣，并不是一组好的搭配<ref name="yue"></ref>。而本歌的歌词，并非方文山所写，也被指幼稚、像是小学生作文，周杰伦不适合这样的歌词<ref name="yue"></ref>，周杰伦回应称，《牛仔很忙》本来是写给小朋友的<ref>{{cite web |url= http://news.xinhuanet.com/tai_gang_ao/2007-11/30/content_7172497.htm|title= 新专集被批"史上最差" 周董发狠:都是屁话|accessdate=2009-07-18 |work= 新华网|date= 2007-11-30}}</ref>。

此外，这首歌的旋律则被质疑抄袭美国动画片《小红帽》中的配乐，一些网友则指其与《大风车》中一首儿歌的旋律开头、韩国卡通“Pororo”中一首儿歌很相似，而一些乐评人和官方工作人员则否认这一说法<ref>{{cite web|url= http://www.sn.xinhuanet.com/2007-12/14/content_11946301.htm|title= 周杰伦:《牛仔很忙》被质疑抄袭美国儿歌|accessdate= 2009-07-18|work= 新华网|date= 2007-12-14|deadurl= yes|archiveurl= https://web.archive.org/web/20160304122415/http://www.sn.xinhuanet.com/2007-12/14/content_11946301.htm|archivedate= 2016-03-04}}</ref>。
力挺：越听越好听
在某网站的音乐频道，记者看到，在周董新歌《牛仔很忙》首播评价投票栏中，2071位参与者有69.29%的人还是觉得新歌“非常好听”，相比“不好听”所占的10.04%，还是占据了大多数。一位年轻歌迷告诉记者：“刚开始听还觉得一般，但听了几遍后觉得越听越好听，到后来不知不觉就随着  牛仔很忙
旋律抖动身体，很轻快，现在情歌太泛滥了，反倒是乡村风让人耳目一新，说不定会引领新的潮流。”
另一位资深“粉丝”则评价：“周董这首新歌的聪明，就在于虽然是非主流的美国乡村风，但歌曲内容唱的却都是周杰伦自己的生活现状和亲身感受，让歌迷听着不熟悉的曲风，却还能感受到杰伦亲切的味道，可爱，还很调皮。”对于部分听众担心周董新歌转变太大，一位乐评人表示：“周杰伦很聪明，他找到了自己的饱和点，在R&B曲风泛滥成灾的时候，主动跳脱出来。”
《牛仔很忙》同美国早年歌手Paul Clayton《If She'S Gone》的曲子相近。

== 使用 ==
2014年的喜剧片《[[采访_(电影)|采访]]》（Interview）在片中曾使用本曲作为影片的背景音乐，2016年上海彩虹室内合唱团的合唱曲目《[[张士超你到底把我家钥匙放哪里了|张士超你到底把我家钥匙放哪里了]]》后段也引用了本曲副歌的旋律。
