'''牛鬼蛇神'''原是[[道教|道教]]、[[佛教|佛教]][[術語|術語]]，说的是[[阴司|阴司]]的[[鬼卒|鬼卒]]、神人等，后成为固定成语，比喻邪恶丑陋之物。

在[[中国|中国]][[文化大革命|文化大革命]]中成为专用术语，指要被打倒冲击的人的统称。“牛鬼蛇神”的称谓取代了[[康生|康生]]所提出的“[[黑帮_(文革)|黑帮]]”的称谓。<ref>郝斌. 《老来忆“牛棚”》//《南方周末》2013年4月11日，版E27</ref>

== 佛教 ==
「牛」頭「鬼」差，參見[[牛頭馬面|牛頭馬面]]。《[[五苦章句經|五苦章句經]]》：「鬼將名阿傍，牛頭人手，兩足牛蹄，力壯排山，持鋼鐵叉。」<ref>{{cite web |url = http://tripitaka.cbeta.org/zh-cn/T17n0741_001 |title = 五苦章句經 | publisher=中華電子佛典協會}}</ref>

「蛇神」指[[天龍八部|天龍八部]]中的「[[摩睺羅伽|摩睺羅伽]]」－大蟒蛇神，職責是守衛佛法。<ref>{{cite web |url = http://www.emsfj.com/ptjl/ShowArticle.asp?ArticleID=2960 |
title = 佛教趣談：揭秘牛鬼蛇神的真實來歷 |publisher = 峨眉山佛教協會 }}
</ref>

== 文化大革命 ==
[[毛泽东|毛泽东]]在其文章、讲话中，曾习用此语：“最近一个时期，有一些牛鬼蛇神被搬上舞台了。”<ref>[[毛泽东|毛泽东]]《在中国共产党全国宣传工作会议上的讲话》（1955年3月），见《[[毛泽东选集|毛泽东选集]]》[[毛澤東選集第五卷|第五卷]]，p. 416.</ref>这是指传统戏曲中的鬼戏。后来借指“[[右派|右派]]”，如“让牛鬼蛇神都出来闹一闹”、“这不叫诱敌深入，叫自投罗网”。1960年代则进一步泛指敌对势力：“让地、富、反、坏、牛鬼蛇神一齐跑了出来，而我们的干部则不闻不问，有许多人甚至敌我不分……少则几年、十几年，多则几十年，就不可避免地要出现全国性的反革命复辟……”。（1963年）“天下大乱，达到天下大治。过七八年又来一次。牛鬼蛇神自己跳出来。”<ref> 《毛泽东给[[江青|江青]]的一封信》（1966年） </ref> 语义有对象和宽窄的变化，但都是作为负面社会形象出现的指称符号。

[[陈伯达|陈伯达]]于1966年5月31日接管《[[人民日报|人民日报]]》后，口授、修订的第一篇社论（6月1日），就是阐明“文革”目标的《[[橫掃一切牛鬼蛇神|橫掃一切牛鬼蛇神]]》。文中该词被用于指称“盘据在思想文化阵地上”的“[[资产阶级|资产阶级]]的‘[[专家|专家]]’、‘[[学者|学者]]’、‘权威’、‘祖爷’”，社论指控这些“大大小小的[[三家村|三家村]]’反党反社会主义黑线”，做“[[资本主义|资本主义]]复辟的梦，……制造复辟舆论……争夺群众，争夺年青一代和将来一代”。这篇社论经[[中央人民广播电台|中央人民广播电台]]播出，全国各地主要报刊全文刊载，该词便广为传播，流行于文革全过程。

由于该词本来词义宽泛，未经严格定义，被借作为政治概念，弹性很大，先后经历了许多演变：从最初的[[黑帮|黑帮]]、[[反动学术权威|反动学术权威]]，很快发展到指称地、富、反、坏、右等[[黑五类|黑五类]]，后来主要指[[走资派|走资派]]、叛徒、特务等。凡是历次政治运动被打入另册的“反党、反社会主义、反毛泽东思想”分子都是。最后蔓延到只要是[[造反派|造反派]]、当权者不喜欢或认定应打击的对象，无论你的出身好不好，已否做过历史结论，还是说错了一句话，行为作派与众不同……都可以定为“牛鬼蛇神”。它成为了一张无所不包的天罗地网。对这些人的处理，也象他们罪名的确立一样，随意性较大。因为没有法定的政策界限。轻的被贴[[大字报|大字报]]、[[批斗|批斗]]、审查，剃[[阴阳头|阴阳头]]，限制人身自由，关进牛棚；重的被迫到[[五七干校|五七干校]][[劳动改造|劳动改造]]，遣送农村落户，注销城市[[户口|户口]]，甚至家属亲友也受株连。

据统计，文革中的[[冤假错案|冤假错案]]达三百多万件，被投入牛鬼蛇神这张天罗地网里的涉及人数逾千万{{NoteTag|参见“[[一小撮|一小撮]]”裡对文革整肃人数及其百分比的讨论，和出处的注。}}。进入统计的，都是立过案并有结论的，还有部分未立案、未记入档案的，因此实际人数应当还要多些。
