《'''靜默之川'''》（{{jpn|j=最期の川|hg=さいごのかわ|rm=Saigonokawa|f=y}}）為日本[[男子音樂組合|男子音樂組合]][[化學超男子|化學超男子]]的第22張單曲，於2007年10月24日發行。歌曲為由[[秋元康|秋元康]]創作的小說《{{Link-ja|象的背影|象の背中}}》改編的同名電影主題曲，由{{Link-ja|井上能徵|井上ヨシマサ}}作曲，並由秋元本人填詞與製作。《靜默之川》是一首使用鋼琴、弦樂等樂器伴奏的抒情歌曲，歌詞內容反映了小說《象的背影》主人公臨終前的心聲。

歌曲獲得了主流樂評整體正面的評價，包括讚揚歌曲的旋律多變以及歌詞的意義深遠。歌曲發行後實體銷量突破5萬張，較前作銷量大幅上升。[[日本唱片協會|日本唱片協會]]於2007年11月認證歌曲的全曲下載為金唱片，代表歌曲取得超過100,000首手機全曲下載量。

《靜默之川》的音樂錄像帶由山口保幸導演，並請來演員[[大杉漣|大杉漣]]與{{Link-ja|松原汐織}}演出。化學超男子於許多場合現場表演了歌曲，包括於電影《象的背影》首映會、「[[音樂戰士_MUSIC_FIGHTER|音樂戰士 MUSIC FIGHTER]]」，以及演唱會「CHEMISTRY 2008 TOUR "Face to Face" BUDOKAN FINAL」與「10th Anniversary Tour -neon- at [[埼玉超級競技場|埼玉超級競技場]] 2011.07.10」等。

== 發行背景 ==
[[File:Yasushi_Akimoto.jpg|150px]]，同時為小說《{{Link-ja|象的背影|象の背中}}》作者。]]
《靜默之川》為[[秋元康|秋元康]]原作改編電影《{{Link-ja|象的背影|象の背中}}》的主題曲。當電影主題曲還未決定的時候，秋元便已經希望透過自己填詞以表達小說的世界觀，並請求作曲家{{Link-ja|井上能徵|井上ヨシマサ}}創作出以痛苦的男人為題材的抒情曲<ref name="sanspo">{{cite news |language = ja |title = 切なくも力強く…ＣＨＥＭＩＳＴＲＹ、映画「象の背中」主題歌熱唱 |department=芸能 |publisher = スポーツニュース |date = 2007-08-17 |accessdate = 2007-08-20}}</ref>。秋元收到井上所寫的旋律後腦海便即時浮現了副歌的歌詞<ref name="sanspo"/>。其後秋元親自向化學超男子提出演唱歌曲的請求，認為他們是演唱歌曲的不二之選<ref name="sanspo"/>。組合收到歌曲時曾經對於歌詞內容感到困惑，不肯定自己能否把「站在死亡邊緣上的男人所留下的最後話語」真實地表現出來<ref name="livedoor"/>。二人其後特意在錄音前閱讀了原作小說並觀看了電影的片段，以更加深刻理解歌詞的世界<ref name="livedoor">{{cite news |language = ja |title = CHEMISTRY、秋元康・原作の映画主題歌 |url=http://news.livedoor.com/article/detail/3271184/ |publisher = livedoor |date = 2007-08-17 |accessdate = 2017-09-17}}</ref>。

2007年8月17日，化學超男子官方網站公佈了新單曲《靜默之川》的發行計劃<ref>{{cite web|url=http://www.chemistryclub.net/pc/top.jsp|archiveurl=https://web.archive.org/web/20071017031326/http://www.chemistryclub.net/pc/top.jsp|title=chemistry club|publisher=chemistry official web site|accessdate=2017-08-31|language=ja|archivedate=2007-10-17|deadurl=yes}}</ref>。使用歌曲片段的電影預告則於公佈消息的翌日開始在全國電影院上映<ref name="livedoor"/>。歌曲其後於2007年10月27日正式發行，而電影則於作品發行三天後於全國上映<ref name="livedoor"/>。

== 詞曲 ==
《靜默之川》是一首慢板抒情曲，歌曲全長6分19秒<ref name="yodo">{{cite web|url=http://www.yodobashi.com/product/100000009000472774/ |script-title=ja: 最期の川 通販【全品無料配達】 |language=ja  |publisher=[[友都八喜|ヨドバシ]] |accessdate=2017-08-28}}</ref>。歌曲以[[B大調|B大調]]創作<ref name="DL">{{cite web|url=https://www.dlmarket.jp/products/detail/309173 |script-title=ja: 最期の川 / CHEMISTRY (ピアノ＆ヴォーカル譜)　【楽譜・スコア】|language=ja  |publisher=DL Market |accessdate=2017-09-17}}</ref>，[[拍號|拍號]]為每分鐘72拍<ref name="DL"/>。化學超男子在歌曲中的音域範圍為G<sub>2</sub>到F<sub>5</sub>（包括歌詞以外的部份）<ref>{{cite web|url=http://www.music-key.com/chemistry/ |script-title=ja: 音域データ - CHEMISTRY |language=ja  |publisher=音域データ |accessdate=2017-09-17}}</ref>。歌曲使用的樂器包括[[鋼琴|鋼琴]]、[[弦樂|弦樂]]與人聲等<ref name="CDJournal">{{cite web|url=http://artist.cdjournal.com/d/-/4107090168 |script-title=ja: CHEMISTRY - 最期の川 |language=ja  |publisher=CDJournal |accessdate=2017-08-28}}</ref>。歌詞以「知道自己死期的男人向深愛的人所傳達的最後心意」為主題<ref name="natalie">{{cite web|url=http://natalie.mu/music/news/3062 |script-title=ja: [CHEMISTRY] 秋元康原作映画の主題歌を担当 |language=ja  |publisher=ナタリー |date=2007-08-17 |accessdate=2017-09-17}}</ref>，歌詞內容反映了小說《象的背影》主人公臨終前的心聲<ref name="livedoor"/>。秋元其後以「希望能與重要的人一起聆聽的一首歌」評論歌曲<ref name="natalie"/>。

== 專業評價 ==
《靜默之川》獲得樂評家整體正面的評價。《CDJournal》讚揚以臨終前的男人的心情與遺言作主題的歌詞意義深遠，而具說服力的聲樂亦準確把歌詞主題表達出來，以及欣賞歌曲主要以鋼琴與弦樂作為伴奏的做法<ref name="CDJournal"/>。《excite》亦有相似的評價，以「把「生與死、以至真實的愛」恢宏的主題認真地唱出來的感動抒情歌曲」形容歌曲<ref>{{cite web |url=http://www.excite.co.jp:80/music/disc/4562104044329/ |archiveurl=https://web.archive.org/web/20080413235638/http://www.excite.co.jp/music/disc/4562104044329/ |script-title=ja: 最期の川 / CHEMISTRY |publisher=excite |accessdate=2017-08-28 |language=ja |archivedate=2008-04-13 |deadurl=yes }}</ref>。

《hotexpress》的元井庸介讚揚組合二人把歌詞的感情以值得自豪的出色表達出來，並同時認為歌曲的旋律多變：「即使單是追求旋律而聽，便已經能不可思議地讓人反覆想聽<ref>{{cite web |url=http://www.hotexpress.co.jp:80/review/1524/ |archiveurl=https://web.archive.org/web/20071225015610/http://www.hotexpress.co.jp/review/1524/ |script-title=ja: レびューリスト |publisher=hotexpress |accessdate=2017-08-28 |language=ja |archivedate=2007-12-25 |deadurl=yes }}</ref>。」《聆聽日本》的森杜男讚揚歌曲確實是首感動的抒情曲，認為能啟發聽眾思考「真實的愛與幸福的意思」，並以歌曲結尾的歌詞「打從心底感恩感謝」總結對歌曲的讚賞<ref>{{cite web |url=http://listen.jp:80/store/diskreview_21362.htm |archiveurl=https://web.archive.org/web/20071108230803/http://listen.jp/store/diskreview_21362.htm |script-title=ja: 新作レビュー - CHEMISTRY / 最期の川 |publisher=listen.jp |accessdate=2018-06-11 |language=ja |archivedate=2007年11月8日 |deadurl=yes |df= }}</ref>。《選擇性聆聽》的桂格讚揚二人在歌曲裡的歌聲，但認為歌曲的長度稍為過長<ref>{{cite web|url=http://selective-hearing.com/chemistry-face-to-face/ |title= Chemistry - Face To Face |language=en  |publisher=Selective Hearing |accessdate=2017-08-28}}</ref>。

== 商業成績 ==
《靜默之川》於商業上取得成功。在實體唱片方面，歌曲首週以20,167張的銷量空降[[公信榜|公信榜]]第4位<ref>{{cite web|url=http://www.oricon.co.jp/rank/js/w/|archiveurl=https://web.archive.org/web/20071031180609/http://www.oricon.co.jp/rank/js/w/ |script-title=ja: 2007年11月05日付 ランキング シングル  週間ランキング|publisher=[[Oricon|Oricon]]|accessdate=2017-08-28 |language=ja|archivedate=2007-10-31}}</ref>，比起前作《{{Link-ja|This Night}}》首週第13位的排名大幅提升<ref>{{cite web|url=http://www.oricon.co.jp/rank/js/w/|archiveurl=https://web.archive.org/web/20070809223329/http://www.oricon.co.jp/rank/js/w/ |script-title=ja: 2007年8月13日付 ランキング シングル  週間ランキング|publisher=[[Oricon|Oricon]]|accessdate=2017-08-28 |language=ja|archivedate=2007-08-09}}</ref>，同時也是組合自2006年的《{{Link-ja|約定的地方|約束の場所 (CHEMISTRYの曲)}}》以來再次打進週榜前5位的單曲<ref>{{cite web|url=http://www.oricon.co.jp/prof/266851/products/single/ |script-title=ja: CHEMISTRYのシングル作品 |language=ja  |publisher=Oricon |accessdate=2017-08-28}}</ref>。歌曲其後以41,987張的銷量登上2007年11月月榜第16位<ref>{{cite web|url=http://www.oricon.co.jp/rank/js/m/|archiveurl=https://web.archive.org/web/20071213112837/http://www.oricon.co.jp/rank/js/m/|script-title=ja: 2007年11月度 ランキング シングル  月間ランキング|publisher=[[Oricon|Oricon]]|accessdate=2017-08-28|language=ja|archivedate=2007-12-13|deadurl=yes}}</ref>，以及以47,653張的銷量登上2007年度年榜第168位<ref>{{cite web|url=http://ranking.oricon.co.jp/index.asp |script-title=ja: オリコンランキング情報サービス「you大樹」|publisher=you大樹|subscription=yes |accessdate=2017-08-28 |language=ja}}</ref>。歌曲累積銷量突破5萬張，比起前作的銷量大幅回升，也是繼《約定的地方》後組合銷量再次突破5萬張的單曲<ref>{{cite web|url=http://nendai-ryuukou.com/artist/094.html |script-title=ja: CHEMISTRY CDシングル売上枚数一覧 |publisher=年代流行 |accessdate=2017-08-28 |language=ja}}</ref>。

在線上下載方面，《靜默之川》於2007年11月獲[[日本唱片協會|日本唱片協會]]認證全曲下載為金唱片<ref>{{cite web|script-title=ja: 一般社団法人 日本レコード協会 |language=ja |url=http://www.riaj.or.jp/f/data/cert/gd.html |publisher=[[日本唱片協會|日本レコード協会]] |accessdate=2017-08-28 }}在下拉菜單中選擇「2007年11月」。</ref>，代表歌曲取得超過100,000首手機全曲下載量。《靜默之川》是組合繼《約定的地方》後再次獲得下載認證的歌曲<ref name="RIAJ">{{cite web|script-title=ja: 一般社団法人 日本レコード協会 |language=ja |url=http://www.riaj.or.jp/f/data/cert/hs_search.html |publisher=日本レコード協会 |accessdate=2017-08-28 }}於歌手欄輸入「{{lang|ja|CHEMISTRY}}」搜尋。</ref>，也是組合第三首獲得下載認證的歌曲（包括與[[Crystal_Kay|Crystal Kay]]合作名義的《{{Link-ja|Two As One|Two As One}}》）<ref name="RIAJ"/>。

== 音樂錄像帶 ==
《靜默之川》的音樂錄像帶由山口保幸導演<ref>{{cite web|url=http://www.spaceshowertv.com/search/detail.cgi?mu=0064054&ch=2 |script-title=ja: CHEMISTRY 最期の川 ミュージックビデオサーチ  |publisher=スペースシャワーTV |accessdate=2017-09-17 |language=ja}}</ref>，請來演員[[大杉漣|大杉漣]]與{{Link-ja|松原汐織}}演出<ref>{{cite web|url=http://www.lespros.co.jp/talent/models/siori_matsubara/#tabs19 |script-title=ja: 松原汐織 モデル レプロエンタテインメント |publisher=レプロエンタテインメント |accessdate=2017-08-31 |language=ja}}</ref>，並以故事形式拍攝。錄像帶以女兒抬望天空作開始，其後女兒開始回想起與當時不久人世的父親一同旅行的回憶，當中穿插著化學超男子成員演唱的場景。回憶最後以父親成功橫過河流作終結，而錄影帶的最後一幕則以二人的合照結束。

2017年2月14日，化學超男子新開設的[[YouTube|YouTube]]頻道公開了《靜默之川》與其他32首歌曲的短版音樂錄像帶<ref>{{cite web|url=http://www.oricon.co.jp/news/2085943/ |script-title=ja: CHEMISTRY、YouTube公式チャンネル開設 MV33本&5年半ぶり新曲も |publisher=Oricon |date=2017-02-14 |accessdate=2017-08-31 |language=ja}}</ref>。歌曲的音樂錄像帶至今錄得超過22萬次觀看次數<ref>{{cite web|url=https://www.youtube.com/watch?v=Pk17rkaqz2U |script-title=ja: CHEMISTRY 『最期の川』MV |publisher=CHEMISTRY Official YouTube Channel |accessdate=2018-06-11 |language=ja}}</ref>。
