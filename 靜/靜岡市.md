'''静冈市'''（{{jpn|j=静岡市|hg=しずおかし|rm=Shizuoka shi|f=y}}）是一座位於[[日本|日本]]中部[[東海地方|東海地方]][[静冈县|静冈县]]中部的城市，也是静冈县[[縣廳所在地|縣廳所在地]]，並在2005年4月1日成为[[政令指定都市|政令指定都市]]<ref>{{cite web |url=http://www.soumu.go.jp/main_sosiki/jichi_gyousei/bunken/shitei_toshi-ichiran.html |script-title=ja:指定都市一覧 |accessdate=2014-09-03|publisher={{lang|ja|総務省}} |language=ja}}</ref>。靜岡市有人口約70万人，是靜岡縣內面積和人口第二大都市（均位列[[濱松市|濱松市]]之後）<ref>{{cite web |url=http://toukei.pref.shizuoka.jp/jinkoushugyouhan/data/02-030/documents/2608jk.xls |script-title=ja:平成26年8月市町別推計人口 |accessdate=2014-09-03|publisher={{lang|ja|静岡県}} |language=ja}}</ref>。靜岡市的轄區面積超過1,411平方公里，是日本面積第五大都市（位於[[高山市|高山市]]、[[濱松市|濱松市]]、[[日光市|日光市]]、[[北見市|北見市]]之後），亦是都道府縣廳所在地中面積最大的城市<ref>{{cite web |url=http://www.gsi.go.jp/common/000077734.pdf |script-title=ja:市区町村別面積の順位 大小各20位 |accessdate=2014-09-03|publisher={{lang|ja|国土地理院}} |language=ja}}</ref>。靜岡市下設[[葵區|葵區]]、[[駿河區|駿河區]]、[[清水區_(日本)|清水區]]三個區。現在的靜岡市是在2003年4月1日由舊静冈市和[[清水市|清水市]]兩座都市合併誕生的<ref>{{cite web|url=http://www.city.shizuoka.jp/deps/kikaku/sskeii2.html|script-title=ja:「静岡市・清水市合併協議会」における協議の経緯（年表）|accessdate=2014-09-03|publisher={{lang|ja|静岡市}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。舊靜岡市是一座以[[駿府城|駿府城]]為核心的城下町，而清水市則是一座港口都市。兩座城市之間有著鮮明的差異且中間有[[日本平|日本平]]相隔，使得靜岡市成為一座具有多樣特色的都市。

== 历史 ==
=== 古代時期 ===
自[[繩文時代|繩文時代]]開始，靜岡市就已經開始有人居住。現在靜岡市內的[[登呂遺址|登呂遺址]]即為[[彌生時代|彌生時代]]時的村落和農田遺跡<ref>{{cite web|url=http://www.city.shizuoka.jp/deps/kyoikusoumu/torohaku_sub2.htm|script-title=ja:登呂遺跡について|accessdate=2014-09-04|publisher={{lang|ja|静岡市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20120121042831/http://www.city.shizuoka.jp/deps/kyoikusoumu/torohaku_sub2.htm|archivedate=2012-01-21}}</ref>。在奈良時代，[[駿河國|駿河國]]的國府設在現在的靜岡市。1338年，{{link-ja|今川範國|今川範国}}被[[足利尊氏|足利尊氏]]封為駿河守護，從此現靜岡市一帶地區成為[[今川氏|今川氏]]的大本營。今川氏在靜岡模仿[[京都|京都]]修建城市，使得靜岡市在現在保留有京都名稱相同的地名和町名。當時由許多文化人逃離戰亂的京都來到靜岡，使得靜岡文化繁榮，有「[[小京都|小京都]]」之稱<ref>{{cite web |url=http://www2.tokai.or.jp/marco/10-sizu/10-siz-1.html |script-title=ja:この街のかたちを考える|accessdate=2014-09-04|publisher={{lang|ja|今川時代再発見}} |language=ja}}</ref>。但在1560年[[桶狹間之戰|桶狹間之戰]]後，今川氏迅速衰亡。1570年，[[武田信玄|武田信玄]]攻下靜岡，修建江尻城，靜岡市一度被武田氏統治。而舊清水市一帶在這一時期是武田氏的水軍基地。不過在1582年，武田氏又在[[織田信長|織田信長]]、[[德川家康|德川家康]]勢力的攻擊下滅亡。武田氏的領地改由德川家康領有。德川家康在靜岡修築[[駿府城|駿府城]]<ref name="Castle">{{cite web|url=http://www.shizuoka-cvb.or.jp/oogosho400/study/03_01.htm|script-title=ja:大御所の城「駿府城」とは?|accessdate=2014-09-04|publisher={{lang|ja|公益財団法人 静岡観光コンベンション協会}}|language=ja|title=存档副本|archive-url=https://web.archive.org/web/20150407101630/http://www.shizuoka-cvb.or.jp/oogosho400/study/03_01.htm|archive-date=2015-04-07|dead-url=yes}}</ref>，使得靜岡成為德川家康的發跡地。在德川家康創建[[江戶幕府|江戶幕府]]並將將軍一職讓給[[德川秀忠|德川秀忠]]之後，家康自[[江戶|江戶]]移居至駿府，並在這裡展開[[大御所|大御所]]政治，靜岡一度成為日本實際上的政治中心。江戶時代的靜岡的人口大約有15,000人，是茶葉、木材、紙等物資的集散地，亦是當時日本規模較大的城下町<ref name="Chishi">{{cite book |author={{lang|ja|藤田佳久・田林明}}|script-title=ja:《日本の地誌7　中部圏》 |year=2007 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-16767-2|language=ja}}</ref>{{rp|235}}。德川幕府還開闢了清水湊，當時的清水湊也是江戶和大阪之間重要的物資中轉基地<ref>{{cite web|url=http://www.shimizu.pa.cbr.mlit.go.jp/kiwameru/04_01roots.html|script-title=ja:港のルーツをたどる|accessdate=2014-09-04|publisher={{lang|ja|国土交通省中部地方整備局 清水港湾事務所}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20130717030719/http://www.shimizu.pa.cbr.mlit.go.jp/kiwameru/04_01roots.html|archivedate=2013-07-17}}</ref>。

=== 舊靜岡市 ===
在[[明治維新|明治維新]]之後，靜岡的名稱由江戶時代的「府中」（在日文中和「不忠」同音）改為靜岡。1889年4月1日，靜岡城下的74町和[[安倍郡|安倍郡]]的50町、有渡郡南安東村的部分地區統合，實施[[市制_(日本制度)|市制]]，靜岡市正式成立<ref name="Heisei24">{{cite web |url=http://www.city.shizuoka.jp/000119844.pdf |script-title=ja:平成２４年度 静岡市区政概要 |accessdate=2014-09-04 |publisher={{lang|ja|静岡市}} |language=ja }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。當時的靜岡市面積為4.36平方公里，人口有37,681人，是日本最初的31個市之一。隨著鐵路[[東海道本線|東海道本線]]的開通，靜岡市逐步確立了其靜岡縣內中樞都市的地位。此後靜岡市亦經歷過多次市町村合併，市域面積逐漸擴大。1940年的靜岡大火和二戰中的空襲使得靜岡受到嚴重打擊，但戰後靜岡迅速復興。1969年，隨著市附近的6個村被併入靜岡市，靜岡市的人口超過40萬人。1996年，靜岡市被指定為[[中核市|中核市]]。
=== 舊清水市 ===
而在舊清水市方面。清水的歷史和港口息息相關，1899年，[[清水港|清水港]]被指定為貿易港，開始和外國進行貿易<ref>{{cite web |url=http://www.portofshimizu.com/%E6%B8%85%E6%B0%B4%E6%B8%AF%E3%81%AE%E6%A6%82%E8%A6%81/%E6%B8%85%E6%B0%B4%E6%B8%AF%E3%81%AE%E6%AD%A9%E3%81%BF-%E6%AD%B4%E5%8F%B2/ |script-title=ja:清水港の歩み|accessdate=2014-09-04|publisher={{lang|ja|清水港}} |language=ja}}</ref>。在昭和中期之前，清水的三保一帶不適合農耕，有不少民眾都選擇移民，以至三保曾有「美國村」之稱。1924年2月11日，清水町和入江町、不二見村、三保村合併為清水市。當時清水市面積25.34平方公里，人口有43,295人，是日本第101個市<ref>{{cite web|url=http://www.city.shizuoka.jp/deps/simizu/rekishinennpyou.html|script-title=ja:年表|accessdate=2014-09-04|publisher={{lang|ja|清水区}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。和靜岡市相同，清水市亦在二戰中遭遇美軍的空襲。1974年7月7日，清水市內發生洪水，市區受災嚴重<ref>{{cite web|url=http://www.city.shizuoka.jp/deps/simizu/dekigoto.html|script-title=ja:歴史　できごと|accessdate=2014-09-04|publisher={{lang|ja|清水区}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。但另一方面，清水亦成長為日本重要的港口，加上[[日本平|日本平]]、[[三保之松原|三保之松原]]等優美的風景，使得清水成為知名的觀光都市。隨著人口的增加，清水市亦在2001年4月1日升為特例市。
=== 兩市合併 ===
自1950年代開始，就有合併靜岡市和清水市的呼聲。為促進兩市的廣域性開發，1965年，｢静清廣域行政協議會｣成立<ref name="Merge">{{cite web|url=http://www.city.shizuoka.jp/deps/kikaku/ssshiryou.html|script-title=ja:静岡市・清水市合併資料集|accessdate=2014-09-04|publisher={{lang|ja|静岡市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140909193621/http://www.city.shizuoka.jp/deps/kikaku/ssshiryou.html|archivedate=2014-09-09}}</ref>。1998年，兩市組建了「静岡市・清水市合併協議會」<ref>{{cite web|url=http://www.city.shizuoka.jp/deps/kikaku/sskeii1.html|script-title=ja:「静岡市・清水市合併協議会」設置までの経緯（年表）|accessdate=2014-09-04|publisher={{lang|ja|静岡市}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，靜岡清水兩市合併開始實質議論階段。2002年，静岡市・清水市合併協議會通過兩市合併的決定<ref>{{cite web|url=http://www.city.shizuoka.jp/deps/kikaku/sskeii2.html|script-title=ja:「静岡市・清水市合併協議会」における協議の経緯（年表）|accessdate=2014-09-04|publisher={{lang|ja|静岡市}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在靜岡縣議會通過決議承認兩市合併之後，2003年4月1日，靜岡市和清水市正式合併，新靜岡市誕生<ref>{{cite web|url=http://www.city.shizuoka.jp/deps/kikaku/sskeii3.html|script-title=ja:静岡市・清水市廃置分合に係る法定手続と新市誕生に向けた経緯（年表）|accessdate=2014-09-04|publisher={{lang|ja|静岡市}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在2005年4月1日，靜岡市成為日本第14個[[政令指定都市|政令指定都市]]。成為政令指定都市之後，靜岡市分為三個區，其中駿河區和葵區相當於舊靜岡市區域，而清水區則相當於舊清水市區域。在2006年和2008年，[[庵原郡|庵原郡]]的{{link-ja|蒲原町|蒲原町}}<ref>{{cite web|url=http://www.city.shizuoka.jp/deps/kikaku/skshiryou.html|script-title=ja:静岡市・蒲原町合併資料集|accessdate=2014-09-04|publisher={{lang|ja|静岡市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140821010409/http://www.city.shizuoka.jp/deps/kikaku/skshiryou.html|archivedate=2014-08-21}}</ref>和[[由比町|由比町]]<ref>{{cite web|url=http://www.city.shizuoka.jp/deps/kikaku/syshiryou.html|script-title=ja:静岡市・由比町合併資料集|accessdate=2014-09-04|publisher={{lang|ja|静岡市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20130313105517/http://www.city.shizuoka.jp/deps/kikaku/syshiryou.html|archivedate=2013-03-13}}</ref>又相繼被編入靜岡市，使得靜岡市的面積進一步擴大。

== 地理 ==
[[File:Shizuoka_city_center_area_Aerial_photograph.1988.jpg|thumb]]
靜岡市位於靜岡縣的中部，幾乎位處[[東京特別區|東京]]和[[名古屋市|名古屋]]的中間點上。靜岡市市域範圍橫跨[[駿河灣|駿河灣]]至[[赤石山脈|赤石山脈]]，面積十分廣大。市的東端位於富士川河口右岸，西端位於百俁澤之頭附近，南端位於大崩海岸，北端位於{{link-ja|間之岳|間ノ岳}}，亦是靜岡縣的最北端。在2003年4月至2005年2月之間，靜岡市曾經是日本面積最大的都市。即使現在，靜岡市仍然是日本面積第五大城市和面積最大的都道府縣廳所在地城市。據國土地理院的調查，靜岡市的面積達1,411.93平方公里<ref>{{Cite web|url=http://www.gsi.go.jp/KOKUJYOHO/MENCHO-title.htm|script-title=ja:全国都道府県市区町村別面積調|author={{lang|ja|国土地理院}}|accessdate=2014-09-03}}</ref>。靜岡市三面環山，南面則是[[駿河灣|駿河灣]]。靜岡市的大部分地區都是無法居住的山林。靜岡市山地面積佔總面積的82%，市域北部有間之岳、鹽見岳、赤石岳、前聖岳等海拔高度超過3,000米的山峰<ref>{{cite book |author={{lang|ja|平岡昭利・野間晴雄}} |script-title=ja:《地図で読む百年　中部Ⅰ》 |year=2000 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=978-4772250474}}{{ja}}</ref>{{rp|67-72}}。市內98%的人口集中在面積不足10%的平原地區，市區也高度集中在南部的狹小平原地區。在統計上來說，靜岡市雖然是靜岡縣內人口和面積第二大都市，但[[日本人口集中地區|人口集中地區]]的面積為98.8平方公里，超過濱松市，位居靜岡縣內首位<ref>{{cite web |url=https://www2.pref.shizuoka.jp/all/file_download1050.nsf/6E6419EDC33BA4C8492573E50030C332/$FILE/syohidoko18.pdf |script-title=ja:平成18年度静岡県の消費動向 |accessdate=2014-09-03 |publisher={{lang|ja|静岡県商工労働部商業まちづくり室}} |language=ja }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。據2010年的人口普查，靜岡市的人口重心位於葵區古莊5丁目19附近<ref>{{Cite web|url=http://www.stat.go.jp/data/kokusei/topics/topi61.htm|script-title=ja:我が国の人口重心 －平成22年国勢調査結果から－|accessdate=2014-09-03|publisher={{lang|ja|総務省統計局ホームページ}} |language=ja}}</ref>。

=== 氣候 ===
靜岡市屬於[[太平洋側氣候|太平洋側氣候]]區。靜岡市的春季開始於3月。日本春季的象徵—[[櫻花|櫻花]]在靜岡市多於3月底開放。在靜岡梅雨季節後，靜岡市進入炎熱的夏季。7月和8月時，靜岡市的平均氣溫超過25度。夏秋之際吹襲靜岡市的颱風亦為靜岡市帶來豐沛降水。靜岡市的市區地區是日本溫差較少的都市，即使是冬季也較為溫暖，亦很少降雪，使得冬季的靜岡市氣候溫暖乾燥。但靜岡市北部的山麓地區屬於內陸性氣候，在冬季降雪頗多。靜岡市的山區地區也是靜岡縣內降水量最多的地區，年平均降水量超過3,000毫米<ref>{{Cite web|url=http://www.jma-net.go.jp/shizuoka/tokusei.html|script-title=ja:静岡県の気象特性|accessdate=2014-09-04|publisher={{lang|ja|静岡地方気象台}} |language=ja}}</ref>。

== 政治 ==
在國會選舉方面，靜岡市葵區和駿河區的大部分地區以及清水區的小部分地區被劃入日本眾議院選舉的[[靜岡縣第1區|靜岡縣第1區]]，而清水區的大部分地區和葵區及駿河區的小部分地區與[[富士宮市|富士宮市]]以及[[富士市|富士市]]的部分地區共同組成了[[靜岡縣第4區|靜岡縣第4區]]。就歷史上的選舉結果來看，靜岡縣1區呈現[[自由民主黨_(日本)|自民黨]]和[[民主黨_(日本)|民主黨]]互有勝負的局面。而在靜岡縣4區，自民黨則有較為明顯的優勢，民主黨的候選者只在2009年當選過<ref>{{cite web |url=http://www.pref.shizuoka.jp/senkan/shugiin2009/kai/kai-shou/kai-4ku.html|script-title=ja:衆議院小選挙区　開票結果（静岡県第4区）|accessdate=2014-09-04|publisher={{lang|ja|静岡県選挙管理委員会}} |language=ja}}</ref>。現任靜岡1區的議員是[[上川陽子|上川陽子]]<ref>{{cite web |url=http://www.pref.shizuoka.jp/senkan/shugiin2012/kai/kai-shou/kai-1ku.html|script-title=ja:衆議院小選挙区　開票結果（静岡県第1区）|accessdate=2014-09-04|publisher={{lang|ja|静岡県選挙管理委員会}} |language=ja}}</ref>，靜岡4區的議員則是[[望月義夫|望月義夫]]<ref>{{cite web |url=http://www.pref.shizuoka.jp/senkan/shugiin2012/kai/kai-shou/kai-4ku.html|script-title=ja:衆議院小選挙区　開票結果（静岡県第4区）|accessdate=2014-09-04|publisher={{lang|ja|静岡県選挙管理委員会}} |language=ja}}</ref>。兩人都所屬於自民黨<ref>{{cite web |url=http://www.yomiuri.co.jp/election/shugiin/2017/kaihyou/ya22.html|script-title=ja:【小選挙区】静岡県 |accessdate=2017-10-24|publisher={{lang|ja|読売新聞}} |language=ja}}</ref>。	

而在市政方面，合併之後的首任靜岡市長是{{link-ja|小島善吉|小嶋善吉}}。他自1994年當選舊靜岡市的市長之後，在2003年合併之後實現連任，繼續就任新靜岡市的市長。現任靜岡市長是{{link-ja|田邊信宏|田辺信宏}}。他在2011年4月的市長選舉中首次當選<ref>{{cite web |url=http://www.city.shizuoka.jp/000115731.pdf|script-title=ja:静岡県議会議員選挙静岡市長選挙選挙結果調|accessdate=2014-09-04|publisher={{lang|ja|静岡市}} |language=ja}}</ref>，並在2015年、2019年順利連任<ref>{{cite web |url=https://go2senkyo.com/local/senkyo/18503|script-title=ja:静岡市長選挙 |accessdate=2019-04-11|publisher={{lang|ja|選挙ドットコム}} |language=ja}}</ref>。靜岡市議會共有議員48人，其中自民黨是議會最大會派，有議員20人。其他在靜岡市議會擁有議席的政黨包括[[公明黨|公明黨]]、[[日本共產黨|日本共產黨]]、[[日本維新會|日本維新會]]等政黨<ref>{{cite web |url=http://www.city.shizuoka.jp/000115731.pdf|script-title=ja:議員の名簿／会派別名簿|accessdate=2014-09-04|publisher={{lang|ja|静岡市}} |language=ja}}</ref>。

=== 工業 ===
靜岡市內的大型工廠主要集中在舊清水市地區，而舊靜岡市地區的工業企業則大多規模較小<ref>{{cite web |url=http://www.city.shizuoka.jp/000009671.xls|script-title=ja:概況|accessdate=2014-09-04|publisher={{lang|ja|静岡市}} |language=ja}}</ref>。清水港地區在二戰時期興建了眾多軍需工廠，使得戰後清水的重工業頗為發達。清水地區的主要產業是木材、造船、石化、食品、鋁加工產業。{{link-ja|日本輕金屬|日本軽金属}}的清水工廠是日本最重要的鋁製品工廠之一<ref>{{cite web |url=http://www.nikkeikin.co.jp/company/seizou/shimizu-map.html|script-title=ja:清水工場|accessdate=2014-09-04|publisher={{lang|ja|日本軽金属}} |language=ja}}</ref>。舊靜岡市地區則是傳統產業和地場產業較為發達，其中又以雛具、[[漆器|漆器]]<ref>{{cite web|url=http://www.shizuoka-kougei.jp/015.html|script-title=ja:駿河漆器|accessdate=2014-09-04|publisher={{lang|ja|静岡県郷土工芸品振興会}}|language=ja|title=存档副本|archive-url=https://web.archive.org/web/20141013202218/http://www.shizuoka-kougei.jp/015.html|archive-date=2014-10-13|dead-url=yes}}</ref>、[[塑料模型|塑料模型]]<ref>{{cite web|url=http://r25.yahoo.co.jp/fushigi/rxr_detail/?id=20081204-90005739-r25|script-title=ja:全国売上シェア90％以上！“プラモの聖地”静岡の謎に迫る|accessdate=2014-09-04|publisher={{lang|ja|web R25}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20160304200042/http://r25.yahoo.co.jp/fushigi/rxr_detail/?id=20081204-90005739-r25|archivedate=2016-03-04}}</ref>產業最為知名。靜岡市現在是日本雛具生產量最大的都市，產量佔日本全國的90%以上<ref>{{cite web|url=http://www.shizuoka-kougei.jp/003.html|script-title=ja:駿河雛具|accessdate=2014-09-04|publisher={{lang|ja|静岡県郷土工芸品振興会}}|language=ja|title=存档副本|archive-url=https://web.archive.org/web/20141013200416/http://www.shizuoka-kougei.jp/003.html|archive-date=2014-10-13|dead-url=yes}}</ref>。
=== 服務業 ===
靜岡市是一座行政都市和商業都市，這使得服務業成為靜岡市最重要的產業。在金融業方面，靜岡縣的三家地方銀行中，[[靜岡銀行|靜岡銀行]]<ref>{{cite web |url=http://www.shizuokabank.co.jp/companyinfo/profile/gaiyo_history.html|script-title=ja:会社概要|accessdate=2014-09-04|publisher={{lang|ja|静岡銀行}} |language=ja}}</ref>和{{link-ja|清水銀行|清水銀行}}<ref>{{cite web |url=http://www.shimizubank.co.jp/aboutus/information/index.html#a_2|script-title=ja:清水銀行のご案内|accessdate=2014-09-04|publisher={{lang|ja|清水銀行}} |language=ja}}</ref>的總部都是設在靜岡市，顯示靜岡市擁有靜岡縣內金融中心的地位。在商業方面，[[靜岡站|靜岡站]]附近地區是靜岡市內最繁華的商業地區，[[松坂屋|松坂屋]]、[[伊勢丹|伊勢丹]]、[[丸井|丸井]]等百貨店就位於這一地區。車站附近的{{link-ja|兩替町|両替町}}聚集了眾多餐館酒家，是靜岡縣內最大的歡樂街。而[[清水站|清水站]]附近地區則是舊清水市地區最重要的商業區。日本重要物流企業{{link-ja|鈴與|鈴与}}的總部也位於靜岡市清水區。

== 文化 ==
在日本戰國時期，靜岡文化繁榮，駿府文化有「戰國三大文化」之稱。今日的靜岡也是一座文化興盛的都市。市內圖書館、博物館、美術館眾多，具代表性的有[[靜岡縣立美術館|靜岡縣立美術館]]<ref>{{cite web |url=http://www.spmoa.shizuoka.shizuoka.jp/japanese/outline/enkaku/|script-title=ja:沿革|accessdate=2014-09-04|publisher={{lang|ja|静岡県立美術館}}|language=ja}}</ref>、駿府博物館等。靜岡的飲食文化亦具有特色，特別是[[關東煮|關東煮]]的人均消費量位居日本前列<ref>{{cite web |url=http://shizuoka.mytabi.net/shizuoka/archives/shizuoka-oden.php|script-title=ja:静岡おでん|accessdate=2014-09-04|publisher={{lang|ja|my旅しずおか}}|language=ja}}</ref>。靜岡市還是著名漫畫動畫作品[[櫻桃小丸子|櫻桃小丸子]]的故事舞台，作者[[櫻桃子|櫻桃子]]亦出身於靜岡市清水區。
=== 媒體 ===

靜岡縣內全部電視台的總部均位於靜岡市內。現在總部設在靜岡市內的電視台包括[[NHK靜岡放送局|NHK靜岡放送局]]、[[靜岡第一電視台|靜岡第一電視台]]、[[靜岡朝日電視台|靜岡朝日電視台]]、[[靜岡放送|靜岡放送]]、[[靜岡電視台|靜岡電視台]]。靜岡市內的廣播電台則有靜岡放送、{{link-ja|靜岡FM放送|静岡エフエム放送}}、{{link-ja|CityFM靜岡|シティエフエム静岡}}、{{link-ja|FM清水|エフエムしみず}}。[[靜岡新聞|靜岡新聞]]<ref>{{cite web |url=http://www.at-s.com/group/|script-title=ja:グループ会社・関連団体一覧|accessdate=2014-09-04|publisher={{lang|ja|静岡新聞}}|language=ja}}</ref>是一家總部位於靜岡市的地方報紙，總發行量約有70萬，是日本較為大型的一份地方報紙。
=== 體育 ===
靜岡市的舊清水市地區有「足球王國」「日本的巴西」之稱，是日本國內[[足球|足球]]文化最早開始流行的地區<ref>{{cite web|url=http://www002.tokai.or.jp/s-footballogy/hp2/treatise/treatise_chapter2.htm|script-title=ja:第二章	地域社会のスポーツの発展～清水市の事例|accessdate=2014-09-04|publisher={{lang|ja|社会サッカー学のすすめ}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140610163739/http://www002.tokai.or.jp/s-footballogy/hp2/treatise/treatise_chapter2.htm|archivedate=2014-06-10}}</ref>。日本足球界元老級人物[[三浦知良|三浦知良]]亦出身於靜岡市。清水地區的各級學校在日本青少年足球賽事中一直有優秀的成績。雖然隨著足球在日本其他地區也得到普及，清水的球隊在各類賽事中的成績已不像以往那樣突出，但清水仍然是日本足球氛圍最強的地區之一。現在[[J聯賽|J聯賽]]球隊[[清水心跳|清水心跳]]的主場位於靜岡市。[[棒球|棒球]]在靜岡市亦很受歡迎。{{link-ja|靜岡縣立靜岡高等學校|静岡県立静岡高等学校}}曾奪得[[全國高等學校野球選手權大會|全國高等學校野球選手權大會]]的冠軍。

== 教育 ==
在高等教育方面，靜岡市內有多所大學，其中最知名的是[[靜岡大學|靜岡大學]]。靜岡大學設置於1949年<ref name="University">{{cite web |url=http://www.shizuoka.ac.jp/outline/profile/about/index.html |script-title=ja:静岡大学とは|accessdate=2014-09-04|publisher={{lang|ja|静岡大学}} |language=ja}}</ref>，是一座國立大學，亦是靜岡縣內教育研究水準最高的大學之一。靜岡大學分為靜岡校區和濱松校區兩個校區，其中靜岡校區位於靜岡市駿河區，亦是大學的本部所在地<ref name="University" />。除了靜岡大學之外，靜岡市的主要高等教育機構還有[[靜岡縣立大學|靜岡縣立大學]]、{{link-ja|常葉大學|常葉大学}}等大學。靜岡市內一共有33座高中，其中20座是公立高中，13座是私立高中；初中則共有54座，其中43座為市立<ref>{{cite web |url=http://www.city.shizuoka.jp/000168461.pdf |script-title=ja:市立中学校一覧|accessdate=2014-09-04|publisher={{lang|ja|静岡市}} |language=ja}}</ref>，11座為私立。小學則共有90所，其中87所為市立<ref>{{cite web |url=http://www.city.shizuoka.jp/000168460.pdf |script-title=ja:市立小学校一覧|accessdate=2014-09-04|publisher={{lang|ja|静岡市}} |language=ja}}</ref>，3所為私立。

== 觀光 ==
靜岡市擁有眾多觀光景點，是日本著名的觀光都市。在2008年，有超過2,500萬遊客到訪靜岡市，其中有超過148萬人在靜岡市內住宿<ref>{{cite web |url=http://www.city.shizuoka.jp/000097979.pdf |script-title=ja:第２章 静岡市の観光の現状と課題 |accessdate=2014-09-04 |publisher={{lang|ja|静岡市}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20150323023622/http://www.city.shizuoka.jp/000097979.pdf |archivedate=2015-03-23 }}</ref>。靜岡市雖然不直接和[[富士山|富士山]]相鄰，但市內有多處景點適合遠眺富士山。在富士山於2013年列入[[世界文化遺產|世界文化遺產]]時，靜岡市內的[[三保之松原|三保之松原]]亦作為富士山景觀的一部分而被列入世界文化遺產<ref>{{cite web |url=http://hellonavi.jp/shizuoka/shimizu_cv/shisetsu/1357.html |script-title=ja:三保松原（羽衣の松）|accessdate=2014-09-04|publisher={{lang|ja|静岡県観光協会}} |language=ja}}</ref>。三保之松原亦是[[日本三大列表|新日本三景]]和日本三大松原之一。此外位於駿河區和清水區交界處的[[日本平|日本平]]也是遠眺富士山的知名景點，在1959年被日本政府指定為名勝<ref>{{cite web |url=http://kunishitei.bunka.go.jp/bsys/maindetails.asp |script-title=ja:日本平|accessdate=2014-09-04|publisher={{lang|ja|国指定文化財等データベース}} |language=ja}}</ref>。在人文景觀方面，靜岡市有{{link-ja|鐵舟寺|鉄舟寺}}、{{link-ja|龍華寺 (靜岡市)|龍華寺 (静岡市)|龍華寺}}、{{link-ja|草薙神社|草薙神社}}、[[靜岡淺間神社|靜岡淺間神社]]等歷史悠久的寺廟神社。[[駿府城|駿府城]]是日本具代表性的城堡建築<ref>{{cite web|url=http://www.city.shizuoka.jp/deps/bunkazai/bunkazai_tyousa_sunpujyou_index.htm|script-title=ja:駿府城跡|accessdate=2014-09-04|publisher={{lang|ja|静岡市}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140912035306/http://www.city.shizuoka.jp/deps/bunkazai/bunkazai_tyousa_sunpujyou_index.htm|archivedate=2014-09-12}}</ref>，位列[[日本100名城|日本100名城]]之內，亦是賞櫻的聖地。

== 交通 ==
[[File:Shizuoka_Station_(North_exit).jpg|thumb]]]]
靜岡市位處[[東海道|東海道]]上，自古就是日本重要的交通樞紐。在鐵路方面，靜岡市內最主要的鐵路幹線是[[東海道本線|東海道本線]]和[[高速鐵路|高速鐵路]]的[[東海道新幹線|東海道新幹線]]，兩條路線均由[[東海旅客鐵道|東海旅客鐵道]]（JR東海）運營。東海道本線橫貫靜岡市區，沿途經過清水和靜岡等靜岡市內的主要車站。[[靜岡站|靜岡站]]是駿河區和葵區最主要的鐵路車站。東海道新幹線所有的[[木靈號列車|木靈號列車]]和部分[[光號列車|光號列車]]在靜岡站停車。每個小時有兩班木靈號列車和一班光號列車在靜岡站停車。[[清水站_(靜岡縣)|清水站]]則是清水區最重要的鐵路車站。除了JR東海之外，靜岡市內還有{{link-ja|靜岡鐵道|静岡鉄道}}和[[大井川鐵道|大井川鐵道]]兩家公司經營鐵路客運業務。靜岡鐵道經營有連接[[新靜岡站|新靜岡站]]和{{link-ja|新清水站|新清水駅}}的[[靜岡鐵道靜岡清水線|靜岡清水線]]，是靜岡市民重要的交通手段。大井川鐵道的{{link-ja|大井川鐵道井川線|大井川鐵道井川線|井川線}}途徑靜岡市北部山區<ref>{{cite web |url=http://www.oigawa-railway.co.jp/abt_top.html |script-title=ja:南アルプスあぷとライン|accessdate=2014-09-04|publisher={{lang|ja|大井川鐵道}} |language=ja}}</ref>，沿途人煙稀少，現在主要乘客多是觀光客。

[[File:Satta_yukei.jpg|thumb]]（山側）和[[東名高速公路|東名高速公路]]（海側）]]

靜岡市內有多條公路經過。較為重要的有[[國道1號_(日本)|國道1號]]、[[國道52號|國道52號]]、{{link-ja|國道150號|国道150号}}等。在高速公路方面，[[新東名高速公路|新東名高速公路]]和[[東名高速公路|東名高速公路]]均經過靜岡市。此外自清水區出發至[[長野縣|長野縣]][[小諸市|小諸市]]的[[中部横断自動車道|中部横断自動車道]]亦以靜岡市為起點。靜岡市內有多條公車運行，並且亦有自靜岡出發，前往東京、[[名古屋市|名古屋]]、[[近畿地方|近畿地方]]等較遠地區的高速巴士。在海運方面，[[清水港|清水港]]被指定為國際據點港灣，是日本的重要港口。清水港的主要出口產品是摩托車、汽車零件和工業機械等產品，主要進口產品則是工業原料、液化天然氣和金槍魚等。清水港憑藉其可眺望富士山的優美景色而被評為日本三大美港之一<ref>{{cite web |url=http://www.shimizu.pa.cbr.mlit.go.jp/nyumon/sugoi.html |script-title=ja:清水港のココがスゴい！ |accessdate=2014-09-04 |publisher={{lang|ja|国土交通省中部地方整備局 清水港湾事務所}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20150225235932/http://www.shimizu.pa.cbr.mlit.go.jp/nyumon/sugoi.html |archivedate=2015-02-25 }}</ref>。現在清水港除了是座工業港之外，亦有眾多郵輪停靠。靜岡市內並無機場，距離靜岡市最近的機場是[[靜岡機場|靜岡機場]]，自靜岡市出發乘坐汽車可在約40分內到達<ref>{{cite web |url=http://www.mtfuji-shizuokaairport.jp/access/index.html |script-title=ja:交通アクセス|accessdate=2014-09-04|publisher={{lang|ja|静岡空港}} |language=ja}}</ref>。
