'''米卡埃尔·"米克"·尼尔松'''（{{lang-sv|Mikael "Micke" Nilsson}}，{{bd|1978年|6月24日|catIdx=Nilsson, Mikael}}），是一名[[瑞典|瑞典]]足球运动员，司职[[后卫_(足球)|边后卫]]或[[中场|中场]]，现效力于[[丹麦|丹麦]]球队[[邦比足球會|布隆德比]]。

== 俱乐部生涯 ==
尼尔松1997年在家乡球队[[奥韦斯霍尔姆体育俱乐部|奥韦斯霍尔姆]]开始自己的职业足球生涯。两年后，他转会到丹麦球队[[奥胡斯霍尔纳足球俱乐部|奥胡斯霍尔纳]]。仅过了一个赛季，他便于2000年再次转会，加盟[[哈尔姆斯塔德足球俱乐部|哈尔姆斯塔德]]，重新回到[[瑞典足球超级联赛|瑞典联赛]]。在哈尔姆斯塔德的首个赛季，尼尔松大部分时间都在板凳上度过。但后来他终于成为队中的主力球员。

2004年，尼尔松转会[[英超|英超]][[南安普顿足球俱乐部|南安普顿]]。<ref>{{cite web|url=http://www.sr.se/cgi-bin/radiosporten/nyheter/artikel.asp?Artikel=438527 |title=Mikael Nilsson till Southampton |publisher=瑞典广播电台 |date=2004年7月1日 |accessdate=2009年8月7日 |language=sv |deadurl=yes |archiveurl=https://web.archive.org/web/20090530075535/http://www.sr.se/cgi-bin/radiosporten/nyheter/artikel.asp?Artikel=438527 |archivedate=2009年5月30日 }}</ref>在南安普顿，他未能争得主力位置，再加上修咸頓降班英冠。次年便转会至[[希腊|希腊]]的[[帕纳辛奈科斯足球俱乐部|帕纳辛奈科斯]]。

2009年4月8日，尼尔松宣布自己将在2008－2009赛季结束后加盟丹麦的[[邦比足球會|布隆德比]]。他与布隆德比签订了为期三年的合同，自2009年7月1日起生效。<ref>{{cite web |url=http://www.transfermarkt.de/de/news/26602/mikael-nilsson-zu-broendby-if.html |title=Mikael Nilsson zu Bröndby IF |publisher=transfermarkt.de |date=2009年4月8日 |accessdate=2009年8月7日 |language=de |deadurl=yes |archiveurl=https://web.archive.org/web/20090411122011/http://transfermarkt.de/de/news/26602/mikael-nilsson-zu-broendby-if.html |archivedate=2009年4月11日 }}</ref>

尼尔松被瑞典球迷称为“迷你”（MiNi）。这个昵称取自尼尔松名和姓的首两个字母。

== 国家队生涯 ==
尼尔松2002年11月20日首次代表[[瑞典国家足球队|瑞典队]]出场，参加了与[[捷克国家足球队|捷克队]]的比赛。在这场处子秀中，他打进两球，帮助瑞典队与对手战成3:3。<ref>{{cite web |url=http://svenskfotboll.se/arkiv/tidigare/2002/11/herr-malkalas-i-arets-sista-landskamp/ |title=Herr: Målkalas i årets sista landskamp |publisher=瑞典足球协会 |date=2002年11月20日 |accessdate=2009年8月7日 |language=sv |archiveurl=https://web.archive.org/web/20110807103515/http://svenskfotboll.se/arkiv/tidigare/2002/11/herr-malkalas-i-arets-sista-landskamp/ |archivedate=2011年8月7日 |deadurl=yes }}</ref>尼尔松已经代表瑞典队参加过[[2004年欧洲足球锦标赛|2004年欧洲足球锦标赛]]、[[2006年世界杯|2006年世界杯]]和[[2008年欧洲足球锦标赛|2008年欧洲足球锦标赛]]，<ref name="hbk">{{cite web|url=http://www.hbk.se/extra/pod/?action=pod_show&id=354&module_instance=1 |title=Mikael Nilsson |publisher=[[哈尔姆斯塔德足球俱乐部|哈尔姆斯塔德足球俱乐部]] |date= |accessdate=2009年8月7日 |language=sv |deadurl=yes |archiveurl=https://web.archive.org/web/20071031111705/http://www.hbk.se/extra/pod/?action=pod_show&id=354&module_instance=1 |archivedate=2007年10月31日 }}</ref>通常司职后卫。

事实上，尼尔松出道时的位置是中场。但在国家队中，一旦球队缺少后卫，尼尔松便后撤到后防线上。在[[2010年世界杯外围赛_(欧洲区)|2010年世界杯预选赛]]上，瑞典队改打[[足球阵式|3-5-2阵型]]，其中尼尔松的位置则被安排到中场。

== 荣誉 ==
尼尔松跟随哈尔姆斯塔德获得了瑞典足球超级联赛的冠军（2000年）和亚军（2004年）各一次。<ref name="hbk"/>
