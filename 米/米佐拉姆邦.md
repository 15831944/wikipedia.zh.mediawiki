'''米佐拉姆邦'''（[[印地語|印地語]]：'''{{lang|hi|मिज़ोरम}}'''，[[拉丁字母|拉丁字母]]轉寫：{{Unicode|Mizoram}}）是位於[[印度東北部|印度東北部]]的一個邦。於2001年人口普查中邦人口約為890,000。面积21081km²。米佐拉姆邦的[[識字率|識字率]]高達89%，於印度各邦中僅次於[[喀拉拉邦|喀拉拉邦]]。

== 民族 ==
米佐拉姆邦的主要民族是[[米佐人|米佐族]]（[[:en:Mizo_people|Mizo people]]），使用[[米佐语|米佐语]]。米佐族分為很多的亞族，其中最大的是[[:en:Lushai|Lushai]]（卢赛部落），幾乎佔了全邦三分之二的人口。其他亞族包括有[[:en:Ralte|Ralte]]、[[:en:Hmar|Hmar]]、[[:en:Paite|Paihte]]、[[:en:Poi|Poi]]、[[:en:Mara|Mara]]及[[:en:Pawi|Pawi]]，此外還由不屬米佐族，源自[[:en:Arakanese|Arakanese]]的[[:en:Chakma|Chakma]]族（[[查克瑪人|查克瑪人]]）。

== 语言 ==
该邦的官方语言为[[米佐语|米佐语]]和[[英语|英语]]。

== 宗教 ==
全邦約87%的人口为[[基督徒|基督徒]]，是印度三个以基督教為主的邦之一。少数人信仰佛教或印度教。

几乎所有的米佐族人都信奉[[基督教|基督教]]，大多屬于[[長老宗|長老宗]]或[[浸信宗|浸信宗]]。另外少数民族查克瑪族信奉混合了[[印度教|印度教]]和萬物有靈成份的[[南传佛教|南传佛教]]。

1980年代，一位本地研究者宣佈米佐人是「[[以色列人|以色列人]][[失踪的十支派|失落的支派]]」，其後大約有5,000名米佐人和[[庫奇人|庫奇人]]（與米佐人相關的一個民族）在近年歸信了[[猶太教|猶太教]]。然而邦內最具影響力、差不多包括了75万米佐族人口的基督教會认为這種說法是無稽之談。

然而於2005年4月1日，[[以色列|以色列]]的[[瑟法底犹太人|瑟法底犹太人]]（Sephardic Jews）首席[[拉比|拉比]]Shlomo Amar宣佈確認此猶太教社群為可信的以色列失落支派，並同時派遣一小隊拉比到印度為該社群舉行[[犹太教正统派|犹太教正统派]]（Orthodox Judaism）的正式歸信儀式。根據以色列的[[回歸法|回歸法]]，此批自稱屬[[瑪拿西|瑪拿西]]支派的米佐猶太人將可移居以色列。雖然該社群的男性[[基因|基因]]（見[[Y染色體|Y染色體]]）找不到與[[猶太人|猶太人]]的任何聯繫，但在女性的基因中卻找到明顯的中東人特徵，這正符合認定猶太人的常規：母親是猶太人則子女就是猶太人。
