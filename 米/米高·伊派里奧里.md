'''米高·伊派里奧里'''（'''Michael Imperioli'''，{{bd|1966年|3月26日}}）是一名美國演員、編劇和導演，他最出名是在電視劇《[[人在江湖_(美國電視劇)|人在江湖]]》中飾演{{link-en|Christopher Moltisanti|Christopher Moltisanti}}<ref>{{Cite web |url = https://www.biography.com/people/michael-imperioli-204401 |title = Michael Imperioli Biography |publisher = Biography.com |language = en}}</ref>，為他在2004年贏得了{{link-en|黃金時段艾美獎劇情類最佳男配角|Primetime Emmy Award for Outstanding Supporting Actor in a Drama Series}}。他也在電視劇《[[律政風雲|律政風雲]]》中飾演[[紐約市警察局|紐約市警察局]]探員{{link-en|Nick Falco|Nick Falco}}。他在2009年編劇和執導他第一部電影《''{{link-en|The Hungry Ghosts|The Hungry Ghosts}}''》<ref>{{Cite web |url = https://www.moviemaker.com/archives/moviemaking/directing/articles-directing/michael-imperioli-the-hungry-ghosts-20101004/ |title = Michael Imperioli Feeds The Hungry Ghosts by Dante A. Ciampaglia |author = Daniel Pyne |date = 2010-10-04 |publisher = MovieMaker |language = en}}</ref>。
