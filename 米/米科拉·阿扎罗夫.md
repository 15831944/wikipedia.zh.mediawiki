'''米科拉·阿扎罗夫'''（{{lang-uk|Микола Янович Азаров}}，{{lang-ru|Николай Янович Азаров}}，{{bd|1947年|12月17日|||Азаров}}），[[乌克兰|乌克兰]][[政治家|政治家]]。前任[[乌克兰总理|乌克兰总理]]。2004年12月至2005年1月被[[乌克兰总统|乌克兰总统]][[列昂尼德·库奇马|列昂尼德·库奇马]]任命为代总理。[[维克多·亚努科维奇|维克多·亚努科维奇]]在2010年成为总统之后，即被议会选举成为总理。2014年1月28日，阿扎罗夫辞去乌克兰总理职务。

== 早期生活 ==
阿扎羅夫出生於蘇聯時代的[[卡盧加|卡盧加]]，畢業於[[莫斯科國立大學|莫斯科國立大學]]，取得[[地質學|地質學]]及[[礦物學|礦物學]]博士学位。他自1984年移居[[烏克蘭|烏克蘭]]。

== 政治生涯 ==
在[[維克多·亞努科維奇|維克多·亞努科維奇]]擔任總理的兩次任期下，阿扎羅夫擔任財相掌管烏國經濟，不過他似乎並不擅於使用[[烏克蘭語|烏克蘭語]]。

在[[烏克蘭總統|烏克蘭總統]][[庫奇馬|庫奇馬]]任內，他主張與[[俄羅斯|俄羅斯]]、[[白俄羅斯|白俄羅斯]]、[[哈薩克斯坦|哈薩克斯坦]]合作，成立單一經濟區，但此舉被批評阻礙乌克兰融合[[世貿|世貿]]及[[歐洲|歐洲]]經濟。

2014年1月28日辞去乌克兰总理职务，他表示希望自己的辭職能讓烏克蘭衝突各方達成妥協方案并結束衝突。<ref>{{cite news| title=烏克蘭總理阿扎羅夫申請辭職 望烏和平結束衝突| url = http://news.xinhuanet.com/world/2014-01/28/c_126075946.htm| work=新华网|language=zh-hk|accessdate=2014年1月28日}}</ref>同日总统亚努科维奇接受辞职申请，拟任[[谢尔盖·阿尔布佐夫|谢尔盖·阿尔布佐夫]]担任代总理。<ref>{{cite news| title=谢尔盖阿尔布佐夫将担任总理| url = http://podrobnosti.ua/podrobnosti/2014/01/28/955707.html| work=http://podrobnosti.ua/|language=ru|accessdate=2014年1月28日}}</ref>
