'''米高·「米奇」·達菲'''（{{lang-en|'''Michael "Mickey" Duffy'''}}，本名'''威廉·米高·庫斯克'''（'''William Michael Cusick'''），{{bd|1888年||1931年|8月30日}}）也被稱為'''約翰·墨菲'''（'''John Murphy'''）和'''喬治·麥克尤恩'''（'''George McEwen'''），是一名在[[禁酒時期|禁酒令]]期間的[[波蘭人|波蘭裔]][[美國|美國]]黑幫和{{link-en|馬克斯·賀芙|Max Hoff (mobster)}}的競爭對手。他成為費城最有名和最強大的啤酒私酒販之一。<ref name="Mickey">{{Cite web |url = http://gamma.library.temple.edu/exhibits/exhibits/show/potable-power--delaware-valley/duffy |title = Mickey Duffy (1888-1931) |publisher = Temple University Library |language = en}}</ref><ref name="Boardwalk Empire">{{Cite web |url = http://www.denofgeek.com/us/tv/boardwalk-empire/205447/boardwalk-empire-season-5-the-real-mickey-duffy-aka-mickey-doyle |title = Boardwalk Empire Season 5: The Real Mickey Duffy aka Mickey Doyle |author = Tony Sokol |date = 2014-10-19 |publisher = Den Of Geek |language = en}}</ref>

== 生涯 ==
本名是威廉·米高·庫斯克，賓夕凡尼亞州費城{{link-en|Grays Ferry, Philadelphia|Grays Ferry, Philadelphia|Grays Ferry}}的波蘭移民<ref name="camden" />，他改了他的名字，以適應在[[費城|費城]]的[[愛爾蘭|愛爾蘭]]幫派。達菲在年輕時參與輕微偷竊和其他輕罪，在青少年期間犯了更嚴重的罪行，包括於禁酒令期間在進行非法私酒之前進行[[搶劫|武裝搶劫]]和{{link-en|卡車劫持|Truck hijacking|劫持}}。<ref name="Mickey" />

1919年5月，達菲因意圖襲擊和毆打殺人而被捕，在費城的[[東州教養所|東州教養所]]中服役兩年零十一個月。出獄後，禁酒令變成法律，而[[有組織犯罪|有組織犯罪]]集團開始[[走私|走私]]、製造和銷售非法酒精。他出獄後不久便與Edith Craig結婚。<ref name="camden" />

=== 勢力崛起 ===
到1920年代初，達菲已經成為在{{link-en|德拉瓦山谷|Delaware Valley}}最佔支配地位的私酒販之一，擁有在費城、[[肯頓_(新澤西州)|肯頓]]和南澤西的啤酒廠。他的合夥人包括前競爭對手Max Hassel、Harry Green、James Richardson、Charles Bodine和Nicholas Delmore，雖然他在整個十年本來打算頻繁的與對手鬥爭，例如賀芙和Bailey兄弟。<ref name="Mickey" />

在此期間，達菲擴展到合法的生意，包括擁有幾個著名的俱樂部，包括1924年在費城的23街和栗樹街（Chestnut streets）的Perkin和時髦的Club Cadix。<ref name="Mickey" /><ref name="Boardwalk Empire" />他在舊的麗思卡爾頓酒店（Ritz-Carlton hotel）經營他的私酒和{{link-en|彩票賭博|Numbers game}}生意。達菲曾在1927年2月25日晚間離開Club Cadix時被Francis Bailey和Peter Ford槍殺了三次。他的保鏢John Bricker被殺，而俱樂部的看門人Earl Brown也受傷。 這次槍擊事件是在費城地下世界中使用[[湯普森衝鋒鎗|湯普森衝鋒鎗]]的第一個事例。達菲在費城的{{link-en|卓克索大學醫學院|Drexel University College of Medicine|赫尼曼醫院}}接受治療，並返回他的私酒生意。<ref name="camden" />

在1920年代中期，達菲的暴力方式使他與以[[雷丁_(賓夕法尼亞州)|雷丁]]作基地的私酒販Max Hassel衝突。與[[瓦克西·戈登|瓦克西·戈登]]一起，Hassel控制了在賓夕凡尼亞州和新澤西北部的一些啤酒廠。好鬥的達菲用他的方法進入有利可圖的澤西州地盤，迫使Hassel把一間啤酒廠交給他。<ref name="camden" />

達菲從啤酒和{{link-en|彩票賭博|Numbers game}}生意中獲得了巨大的利潤，到了1930年，他在賓夕凡尼亞州的{{link-en|佩恩懷恩 (賓夕凡尼亞州)|Penn Wynne, Pennsylvania|佩恩懷恩}}為自己和他的妻子Edith建了一個官邸。達菲的家在第77街對面佩恩懷恩的城市線一側。 由McWilliams和Maloney以地中海別墅風格建造，結構有白色與綠色薩堤爾（satyrs）在兩側和黑色棕櫚樹畫在正面上。<ref name="camden" />

一名[[禁酒局|禁酒局]]探員[http://www.odmp.org/officer/4840-special-agent-john-g-finiello John Finiello]於1930年9月19日突擊搜查在新澤西州{{link-en|伊麗莎白小鎮 (新澤西州)|Elizabeth Township, New Jersey|伊麗莎白小鎮}}其中一間達菲的啤酒廠時被殺，當地政府便開始打擊達菲的犯罪業務，造成他的伙伴之間的一些敵意，包括他的保鏢和司機Joseph Beatty。<ref name="Mickey" /><ref name="Boardwalk Empire" />

== 逝世 ==
達菲住在[[大西洋城|大西洋城]]的大使酒店（Ambassador Hotel）時候，於1931年8月31日被不知名的襲擊者開槍殺死。雖然案件仍未解決，但當時當局懷疑被指控的襲擊者可能是對達菲有所不滿的合夥人，並且密謀奪取達菲的私酒業務的控制權。<ref name="Mickey" />

在幾個月內，據稱涉及的兩個人Samuel E. Grossman和Albert Skale，在1931年12月在瓦茨街（Watts Street）和{{link-en|吉拉德大道|Girard Avenue}}被槍殺，開始了在費城地下世界的不同幫派之間的暴力浪潮。<ref name="Mickey" />

達菲的葬禮是一個大事。數以千計的人湧向公墓，但警察封鎖線將他們留在入口外。達菲的朋友和家人需要特別通行證才可進入。<ref name="camden" />

== 流行文化 ==
