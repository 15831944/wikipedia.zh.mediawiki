'''米南加保酋長'''（{{lang|ms|Undang}}，或通稱為'''酋長'''）是[[馬來西亞|馬來西亞]][[森美蘭|森美蘭]][[米南加保人|米南加保部族]]的[[世襲|世襲]][[貴族|貴族]]或地方領袖。這個名稱相信是源自馬來語辭彙「法律」（{{lang|ms|undang-undang}}）。

在17世紀以前，移居至馬來半島的米南加保人會從部族中選出[[彭古魯|彭古魯]]（部族領袖）。至18世紀初期，[[雙溪烏絨|雙溪烏絨]]、[[日叻務|日叻務]]、[[柔河|柔河]]和[[林茂|林茂]]領袖的權勢凌駕於其他部族領袖之上，這四位部族領袖被稱為「酋長」（{{lang|ms|Undang}}）。

==繼承和稱號==
四位酋長的法定地位被[[馬來西亞憲法|聯邦憲法]]第181條款承認，酋長是經由米南加保的[[母系制度|母系制度]]和傳統習俗（{{lang|ms|Adat Perpatih}}），從領地（{{lang|ms|Luak}}）的貴族中選出。

* 雙溪烏絨酋長是由{{lang|ms|Waris Hulu}}家族和{{lang|ms|Waris Hilir}}家族選出，世襲的稱號是{{lang|ms|Dato' Kelana Petra Seri Jaya}}。<ref>{{cite web |title=Sungai Ujong |url=http://www.royalark.net/Malaysia/sungai.htm |work=Royal Ark |accessdate=2011-06-06}}</ref>

* 日叻務酋長是由四個家族選出，分別是{{lang|ms|Waris Jelebu}}、{{lang|ms|Waris Ulu Jelebu}}、{{lang|ms|Waris Sarin}}和{{lang|ms|Waris Kemin}}家族，世襲的稱號是{{lang|ms|Dato' Mendelika Mentri Akhir ul-Zaman}}。<ref>{{cite web |title=Jelebu |url=http://www.royalark.net/Malaysia/jelebu.htm |work=Royal Ark |accessdate=2011-06-06}}</ref>

* 柔河酋長是由來自相同母系家族的兄弟或長女的長子繼承{{lang|ms|Dato' Johan Pahlawan Lela Perkasa Setiawan}}的稱號。<ref>{{cite web |title=Johol |url=http://www.royalark.net/Malaysia/johol.htm |work=Royal Ark |accessdate=2011-06-06}}</ref>

* 林茂酋長由兩個家族輪流担任，分別是{{lang|ms|Waris Jakun}}（繼承{{lang|ms|Dato' Lela Maharaja}}的稱號）和{{lang|ms|Waris Jawa}}（繼承{{lang|ms|Dato' Sedia di-Raja}}的稱號），兩個家族都是由來自相同母系家族的兄弟或長女的長子繼承。<ref>{{cite web |title=Rembau |url=http://www.royalark.net/Malaysia/rembau.htm |work=Royal Ark |accessdate=2011-06-06}}</ref>

「酋長」的[[配偶|配偶]]（[[元配|元配]]）稱為「督潘」（{{lang|ms|To' Puan}}）。

==酋長的義務==
如今，米南加保酋長衹有一些象徵性的權力，他們的義務是出席州立法議會的開幕儀式，以及在現任[[最高統治者|最高統治者]]駕崩時，從候選的四位王子（{{lang|ms|Putera Yang Empat}}）中挑選適任的繼承者。

四位王子的稱號是：
* {{lang|ms|Tunku Besar Seri Menanti}}
* {{lang|ms|Tunku Laksamana}}
* {{lang|ms|Tunku Musa Serting}}
* {{lang|ms|Tunku Panglima Besar}}
