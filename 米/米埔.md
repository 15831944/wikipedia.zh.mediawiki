'''米埔內-{后}-海灣拉姆薩爾國際重要濕地'''（{{lang-en|'''Mai Po Inner Deep Bay Ramsar Site'''}}），在當地稱為'''米埔自然護理區'''（'''{{lang|en|Mai Po Nature Reserve}}'''），是一片位於[[香港|香港]][[新界|新界]][[元朗區|元朗區]]米埔一帶的[[濕地|濕地]]。米埔以[[候鳥|候鳥]]、自然護理區和[[紅樹林|紅樹林]]而聞名。於冬季，濕地有不少來自[[中國北部|中國北部]]及[[西伯利亞|西伯利亞]]的候鳥在此過冬。

護理區範圍包括[[大榔基|大榔基]]、[[石山|石山]]和[[尖鼻咀|尖鼻咀]]一帶的濕地，當地亦是[[深圳河|深圳河]]、[[山貝河|山貝河]]及[[天水圍渠|天水圍渠]]的[[河口|河口]]，佔地約1,540[[公頃|公頃]]<ref name="米埔濕地">{{cite web| author=漁農自然護理署 |title=米埔拉姆薩爾濕地一般資料 |url = https://www.afcd.gov.hk/tc_chi/conservation/con_wet/con_wet_look/con_wet_look_gen/con_wet_look_gen.html |accessdate=2014年9月11日 |date=2014年8月14日}}</ref>。米埔自然護理區本身是一個禁區，[[邊界道路|邊界道路]]以外更屬於[[香港邊境禁區|邊境禁區]]，該地可說是雙重禁區<ref name="邊境禁區界線">{{cite web| author=香港警務處 |title=對公眾人士作出的通告-最新邊境禁區界線 |url = http://www.police.gov.hk/ppp_tc/11_useful_info/licences/remind.html |accessdate=2016年5月13日 |date=2016年5月}}</ref>。

米埔濕地被紀錄於《[[國際重要濕地名錄|國際重要濕地名錄]]》中。

==歷史==
[[File:Mai_Po_Marshes_Plaque.jpg|thumb]]
[[File:Mai_Po_Marshes_Gei_wai_view1_2016.jpg|thumb]]
[[File:Mai_Po_Marshes_Gei_wai_boardwalk_2016.jpg|thumb]]
米埔一帶從前是一個養殖[[蝦|蝦]]隻的地方，在濕地中使用[[基圍|基圍]]進行養殖。1950年[[韓戰|韓戰]]爆發，當時的[[香港殖民地時期#香港政府|香港政府]]將香港跟中國大陸接壤的土地劃為[[香港邊境禁區|邊境禁區]]，當中包括米埔濕地。這限制了其發展之餘，米埔濕地的自然生態環境一直都並無得到任何形式的保護。直到[[英國|英國]]於1976年加入《[[濕地公約|濕地公約]]》（拉姆薩爾公約），其條款於1979年延伸至當時為[[殖民地|殖民地]]的[[香港英治時期|香港]]，同時香港政府將米埔濕地列為「[[具特殊科學價值地點|具特殊科學價值地點]]」，予以保護。1984年香港政府將米埔濕地交予[[世界自然基金会|世界自然基金香港分會]]管理，並藉此推行環境保護及教育工作。1995年，米埔濕地正式根據國際濕地公約列為國際重要濕地。

后海灣濕地位於香港西北，是生物多樣性最豐富之一的亞洲濕地。濕地佔地2,700公頃，為多種物種提供棲身之所，更是遷徙性候鳥的天堂。

世界自然基金會香港分會自1983年管理米埔自然保護區（佔后海灣380公頃土地），一直在保護區內進行研究和增設教育設施，同時推出多項保育工作維持濕地。保護區極具生物多樣性，擁有豐富的動植物種，遍佈6種濕地，包括魚塘、基圍、潮間帶泥灘、紅樹林、蘆葦叢及淡水池塘。世界自然基金會將竭盡所能管理米埔濕地，保護這片具重要生態價值的地方。

1990年代開始，由於[[深圳|深圳]][[后海灣|-{后}-海灣]]沿岸的工業發展，加上深方在對岸（今後海企業總部基地及[[深圳灣口岸|深圳灣口岸]]）進行填海工程，米埔濕地的生態或會受到損害。不過到了1990年代後期，由於[[羅湖區|羅湖區]]的地產物業極之興旺，發展商家亦意識到毗鄰米埔濕地的[[深圳灣濕地|深圳灣濕地]]所提供的大自然景觀有助提升物業價值，因此都積極保護當地的生態。

1995年9月4日，內后海灣1,500公頃的土地獲《拉姆薩爾公約》劃為「國際重要濕地」。 

劃為[[拉姆薩爾濕地|拉姆薩爾濕地]]，讓香港尚存最大面積的濕地得以保存。根據拉姆薩爾濕地劃分的管理區，米埔自然保護區屬生物多樣性管理區，為區內的野生生物帶來莫大裨益，同時在教育、休閒及保育等活動方面，為本地社群作出貢獻。

==水文==
米埔濕地是元朗盆地內一個天然淺水河口濕地，平均水深約2.9米，潮差中位值為1.4米。內后海灣是香港與深圳水流和沉積物的匯流處，潮間帶泥灘由沉積物沖積而成，土質主要為幼細淤泥<ref name="米埔濕地" />。

香港屬[[亞熱帶季風氣候|亞熱帶季風氣候]]，每年均受季候風影響，降雨主要集中在4月至9月。潮間帶水域的鹽度隨著季節變化，趨勢十分明顯。在晚春至夏季期間，由於雨水增加，水的鹽度會大幅下降，有時甚至幾乎變成淡水，夏季將結束時，鹽度則會回升，到了秋冬/初春乾燥季節則達到全年最高水平。

這片濕地有助紓減新界西北部的水浸問題，而紅樹林則有穩定后海灣海岸的價值。

==特色==
[[File:Mai_Po_Nature_Reserve_Gei_wai_pano_201611.jpg|center]]全景]]

米埔濕地每年有超過60,000隻的水鳥過冬，候鳥品種包括[[勺咀鷸|勺咀鷸]]、[[小青腳鷸|小青腳鷸]]、[[半蹼鷸|半蹼鷸]]、[[灰尾鷸|灰尾鷸]]、[[黑嘴鷗|黑嘴鷗]]及[[黑臉琵鷺|黑臉琵鷺]]。其中米埔濕地錄得黑臉琵鷺的數目更佔全球總數的25%。這些候鳥主要在[[春季|春季]]和[[秋季|秋季]]出現，有達30,000隻禽鳥同一時間棲身於米埔濕地的泥灘，以作長途遷徙旅程的中途歇息之處。

此外，米埔濕地也有許多野生生物在此定居，包括超過400種[[昆蟲|昆蟲]]、300種[[雀鳥|雀鳥]]、18种[[哺乳动物|哺乳动物]]、21种[[爬行动物|爬行动物]]、90種[[海洋|海洋]][[無脊椎動物|無脊椎動物]]及50種[[蝴蝶|蝴蝶]]。而米埔濕地的潮間帶[[紅樹林|紅樹林]]適應當地海岸生長，並提供落葉給水生動物維生，而面積更為廣東地區第1、[[中國|中國]]第6。

[[世界自然基金會|世界自然基金會]]香港分會在2015年5月至2016年12月，於米埔及[[后海灣|后海灣]][[拉姆薩爾濕地|拉姆薩爾濕地]]內進行[[生物多樣性普查|生物多樣性普查]]，發現逾2,050種生物，包括416種[[雀鳥|雀鳥]]、33種[[哺乳類|哺乳類]]、8種[[兩棲類動物|兩棲類動物]]、22種[[爬行類動物|爬行類動物]]、54種魚、11種蝦 、40種蟹 、逾100種[[蜘蛛|蜘蛛]]、155種[[蜂|蜂]]類、51種[[蜻蜓|蜻蜓]]及[[豆娘|豆娘]]、316種[[蛾|蛾]]、逾15種[[蟻|蟻]]、逾400種[[鞘翅目|鞘翅目]]([[甲蟲|甲蟲]])以及323種植物。當中更發現一種懷疑是全球新品種的[[異足蛛屬|異足蛛屬]][[蜘蛛|蜘蛛]]，有待專家進一步核實<ref>{{cite web |url=http://www.wwf.org.hk/news/press_release/?17860/Press-Release-WWFs-Discovering-Biodiversity-in-Hong-Kong-Wetlands-project-raises-the-number-of-species-in-Mai-Po-to-2050| title= WWF「香港濕地生物多樣性普查」全面更新米埔物種名錄至逾2,050種 新資料助規劃未來棲息地管理及保育工作 | publisher=[[世界自然基金會|世界自然基金會]]香港分會|date=2017年2月13日 }}</ref><ref>{{cite web |url=http://news.mingpao.com/pns/dailynews/web_tc/article/20170212/s00002/1486835421320 | title= 濕地普查發現逾2050物種 米埔錄逾百種蜘蛛 疑現全球新種 | publisher=[[明報|明報]]|date=2017年2月12日 }}</ref><ref>{{cite web |url=http://hk.on.cc/hk/bkn/cnt/news/20170211/bkn-20170211200149287-0211_00822_001.html | title= 米埔蜘蛛暴增至過百種　驚現疑似全球新品種 | publisher=[[東方日報_(香港)|東方日報 (香港)]]|date=2017年2月11日 }}</ref><ref>{{cite web |url=http://hk.apple.nextmedia.com/realtime/news/20170211/56291787 | title= 米埔「居民」唔止得雀！ 水獺豹貓仲有疑似新品種蜘蛛  | publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]]|date=2017年2月11日 }}</ref><ref>{{cite web |url=https://www.hk01.com/%E6%B8%AF%E8%81%9E/71011/-%E5%A4%9A%E5%9C%96-%E7%B1%B3%E5%9F%94%E6%99%AE%E6%9F%A5%E7%99%BC%E7%8F%BE%E7%96%91%E6%96%B0%E5%93%81%E7%A8%AE%E8%9C%98%E8%9B%9B-%E6%A3%AE%E6%9E%97%E7%89%A9%E7%A8%AE%E5%A2%9E%E9%A1%AF%E7%A4%BA%E6%BF%95%E5%9C%B0%E9%96%8B%E5%A7%8B%E8%80%81%E5%8C%96 | title= 【多圖】米埔普查發現疑新品種蜘蛛　森林物種增顯示濕地開始老化  | publisher=[[香港01|香港01]]|date=2017年2月11日 }}</ref>。

而米埔濕地的中心地區則保留了24個傳統基圍蝦塘，位於潮間區的淺水地帶。基圍蝦塘現時仍維持營運，成為[[華南地區|華南地區]]僅存的同類作業。每年4月至10月是基圍蝦塘的作業期，蝦苗來自由后海灣沖入基圍的小蝦，以蝦塘內的[[浮游生物|浮游生物]]維生。到了每年11月至3月期間，基圍蝦塘會被輪流放乾，露出水面的泥濘及大量魚類，成為各種以捕食魚類為生的鳥類（包括[[蒼鷺|蒼鷺]]、[[白鷺|白鷺]]和[[黑臉琵鷺|黑臉琵鷺]]）的覓食和棲息地點。而這種獨特的養蝦方式有利於米埔濕地的生態價值，對米埔濕地的濕地資源有[[生生不息|生生不息]]的作用，有利於該處的[[可持續發展|可持續發展]]，並充份利用米埔濕地的天然生產能力，讓人與濕地的各種生物和諧共處。

==米埔觀鳥賽==
米埔觀鳥賽在每年的3月舉行，是一個國際性的觀鳥賽事。參賽者為世界各地的觀鳥高手，是世界知名的頂級觀鳥賽事之一。

==參觀方法==
根據《野生動物保護條例》（第170章），米埔自然保護區是一個「限制進入或處於其內的地區」，以便對保護區內野生動植物的干擾減至最低。參觀者需要持有由漁農自然護理署署長發出的有效「進入米埔沼澤許可證」方可進入米埔自然保護區範圍。世界自然基金會香港分會可為申請人申請通行證，申請手續一般需時四星期，每張個人通行證的行政費為港幣100元<ref>{{cite web |author=世界自然基金會香港分會 |title=專家參觀 |url = http://www.wwf.org.hk/getinvolved/gomaipo/spetour/|accessdate=2014年9月11日}}</ref>。

米埔濕地每年約有40,000參觀人次，其中包括約10,000名學生。學生參觀項目於逢星期一至星期五舉行，費用由[[香港教育局|教育局]]全數資助，而每年的中小學校參觀活動接近400個。於逢星期六、日及公眾假期（農曆新年除外），米埔濕地則開放給公眾人士參觀，世界自然基金會香港分會並會安排自然導賞員作講解，一個三小時的導賞活動收費每位70港元（包括代辨[[邊境禁區通行證|禁區通行證]]）<ref>{{cite web |author=世界自然基金會香港分會 |title=公眾導賞團 |url=http://online.wwf.org.hk/booking/tc/info.html?type=PT&st=Public#1 |accessdate=2014年9月11日 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

參觀名額是相當有限，尤其是於候鳥出沒的高峰季節。

== 交通 ==
米埔濕地並沒有任何[[公共交通|公共交通]]工具直達。最接近的公共交通工具是行經[[青山公路|青山公路]]的[[九龍巴士76K線|九龍巴士76K線]]，來往[[元朗|元朗]][[朗屏邨|朗屏邨]]至[[上水|上水]]，但乘客於米埔村外的巴士站下車後要再沿[[担竿洲路|-{担}-竿洲路]]步行前往。

前往米埔濕地的人士亦可於[[西鐵綫|西鐵綫]][[元朗站_(西鐵綫)|元朗站]]、[[錦上路站|錦上路站]]或[[東鐵綫|東鐵綫]][[上水站|上水站]]轉乘[[的士|的士]]前往<ref>《[http://www.wwf.org.hk/chi/maipo/publicvisit/visitmap.php 世界自然基金會網頁：如何抵達米埔自然保護區？] {{webarchive|url=https://web.archive.org/web/20070808161250/http://www.wwf.org.hk/chi/maipo/publicvisit/visitmap.php |date=2007-08-08 }}》</ref>。
