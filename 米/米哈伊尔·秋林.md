'''米哈伊尔·弗拉基斯拉沃维奇·秋林'''（{{lang-ru|Михаил Владиславович Тюрин}}，{{bd|1960年|3月2日|}}）是[[俄罗斯|俄罗斯]][[宇航员|宇航员]]。

==早年==
1984年毕业于[[莫斯科航空学院|莫斯科航空学院]]，获得工程学学位。毕业后在[[科罗廖夫能源火箭航天集团|科罗廖夫能源火箭航天集团]]担任工程师，兴趣点是[[动力学|动力学]]、[[弹道学|弹道学]]和软件开发。<ref>{{cite web |title=MIKHAIL TYURIN |url=https://web.archive.org/web/20160801034339/http://www.jsc.nasa.gov/Bios/htmlbios/tyurin.html |publisher=NASA |accessdate=2019-04-12 |language=en |date=2014}}</ref>

==宇航员生涯==
1994年，他被选中开始宇航员训练，后成为远征3航空工程师预定人选，也是[[远征1|远征1]]任务备用成员之一。
===远征3===
2001年8月10日乘坐[[STS-105|STS-105]]前往国际空间站，执行远征3任务。10月8日，他走出[[国际空间站|国际空间站]]进行[[舱外活动|舱外活动]]。这是秋林的第一次、也是俄罗斯太空史上宇航员的第100次太空行走<ref>{{cite web |title=俄罗斯宇航员完成第100次太空行走 |url=http://it.sohu.com/09/59/article15445909.shtml |publisher=[[北京青年报|北京青年报]] |accessdate=2019-04-12 |archiveurl=https://web.archive.org/web/20021217204149/http://it.sohu.com/09/59/article15445909.shtml |archivedate=2002-12-17 |date=2001-10-10}}</ref>。

===远征14===
2006年11月23日，秋林和迈克尔·洛佩斯穿着“海鹰M”宇航服，成功完成了太空行走任务。他们对[[星辰号服务舱|星辰号服务舱]]的中子望远镜进行了维修，同时拆除了WAL2[[无线通讯|无线通讯]]系统的[[天线|天线]]并将其安装在星辰号上面<ref>{{cite web |title=国际空间站成员米哈伊尔·秋林和迈克尔·洛佩斯－阿莱格里亚成功完成了进入太空行走的计划 |url=http://sputniknews.cn/russia/2006112341589029/ |publisher=[[俄罗斯卫星通讯社|俄罗斯卫星通讯社]] |accessdate=2019-04-12 |date=2006-11-23}}</ref>。[[北京时间|北京时间]]23日上午9点，秋林在国际空间站外打出了一杆太空[[高尔夫球|高尔夫球]]<ref>{{cite web |title=俄罗斯宇航员打出第一杆太空高尔夫球 |url=http://news.sina.com.cn/w/2006-11-23/103910577449s.shtml |publisher=新浪新闻 |accessdate=2019-04-12 |archiveurl=https://web.archive.org/web/20190412054641/http://news.sina.com.cn/w/2006-11-23/103910577449s.shtml |archivedate=2019-04-12}}</ref>。
===远征38/39===
2013年11月7日，他和美国宇航员里克·马斯特拉吉奥和日本宇航员[[若田光一|若田光一]]乘坐“联盟TMA-11M”号太空飞船，将[[索契冬奥会|索契冬奥会]]的奥运圣火首次带上国际空间站<ref>{{cite web |title=俄美日宇航员临行合影 将首度传递奥运圣火至太空 |url=http://www.chinadaily.com.cn/hqzx/2013-11/06/content_17085944.htm |publisher=[[中国日报|中国日报]] |accessdate=2019-04-12 |archiveurl=https://web.archive.org/web/20170908224048/http://www.chinadaily.com.cn/hqzx/2013-11/06/content_17085944.htm |archivedate=2017-09-08 |date=2013-11-06}}</ref>。

2014年5月14日，哈萨克斯坦[[卡拉干達州|卡拉干達州]]机场举行了国际空间站“联盟号”飞船宇航员安全返回欢迎仪式<ref>{{cite web |title=2014年5月20日—26日，中俄“海上联合—2014”军事演习实兵演练成功举行 |url=http://www.chinatoday.com.cn/ctchinese/picturechina/article/2014-06/03/content_622130.htm |publisher=中国日报 |accessdate=2019-04-12 |archiveurl=https://web.archive.org/web/20170923120136/http://www.chinatoday.com.cn/ctchinese/picturechina/article/2014-06/03/content_622130.htm |archivedate=2017-09-23 |date=2014-06-03}}</ref>。
