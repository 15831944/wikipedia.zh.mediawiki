'''米林县'''（{{bo|z=སྨན་གླིང་རྫོང་།|w=sman gling rdzong|p=Mainling Zong}}；[[博嘎尔方言|博嘎尔語]]譯：{{Lang|adl|Miling Sên}}）是[[中华人民共和国|中华人民共和国]][[西藏自治区|西藏自治区]][[林芝市|林芝市]]下属的一个[[县级行政区|县]]。[[藏语|藏语]]中意思是“药洲”。

米林县面积9507平方公里，[[邮政编码|邮政编码]]860500，县政府驻米林镇。該縣南部實際被[[印度|印度]]佔領，邊界問題仍未解決。{{Fact|time=2009-02-07T08:14:53+00:00}}<!-- 本模板調用由 {{subst:Fact/auto}} 自動產生 -->

== 历史沿革 ==
1954年设立塔工总管，拉岗宗归塔工总管。1959年8月，以则拉岗宗的西南部设立米林县，县政府驻地米林村，隶属林芝专区管辖。1961年5月，米林县迁驻东多村，1964年，撤销林芝公署，米林县划归[[拉萨市|拉萨市]]。1983年，恢复林芝地区，米林县划归林芝地区。2015年，隶属林芝市管辖。

== 地理 ==
米林县位于[[西藏|西藏]]东南部，[[雅鲁藏布江|雅鲁藏布江]]中游河谷地带，在[[念青唐古拉山|念青唐古拉山脉]]与[[喜马拉雅山脉|喜马拉雅山脉]]之间。全县平均[[海拔|海拔]]3700米。

== 教育 ==

* 小学：米林县中心小学，多卡中心小学，南伊乡小学，里龙乡小学，卧龙镇小学，扎绕乡小学，派镇小学，丹娘乡小学，羌纳乡小学

== 经济 ==
