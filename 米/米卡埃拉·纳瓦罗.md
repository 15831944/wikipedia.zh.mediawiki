'''米卡埃拉·纳瓦罗·加尔松'''（{{lang-es|Micaela Navarro Garzón}}，{{bd|1956年|9月2日|}}）是[[安达卢西亚|安达卢西亚]]出身的[[西班牙工人社会党|西班牙工人社会党]]女性[[政治人物|政治人物]]，也是倡导[[性别|性别]]平等，反对性别暴力的[[女权主义|女权主义]]者。<ref name=lainfor/>2014年至2016年担任西班牙工人社会党主席。2016年1月就任西班牙众议院第二副议长<ref>{{cite web|title=Micaela Navarro, elegida vicepresidenta segunda del Congreso de los Diputados|url=http://www.ideal.es/jaen/jaen/201601/13/micaela-navarro-elegida-vicepresidenta-20160113131512.html|publisher=Ideal|date=2016-01-13|accessdate=2016-02-13}}</ref>。

==经历==
出生于安达卢西亚[[哈恩省|哈恩省]][[安杜哈尔|安杜哈尔]]。未接受过大学教育。是家庭主妇，也是2个女儿的母亲。<ref name=lainfor>{{cite web |title=Micaela Navarro, feminista y sin carrera, será la nueva presidenta del PSOE |url=https://www.lainformacion.com/espana/micaela-navarro-feminista-y-sin-carrera-sera-la-nueva-presidenta-del-psoe_8ssF5nDjgqfyVCxjiMUlQ2/ |publisher=LA INFORMACIÓN |accessdate=2019-02-25 |language=es |date=2014-07-26}}</ref>

1994年加入西班牙工人社会党[[哈恩省|哈恩省]]分部执行委员会，1996年至2000年是西班牙参议院哈恩选区议员<ref>{{cite web |title=Navarro Garzón, Micaela |url=http://www.psoe.es/transparencia/transparencia-cargos/ficha/micaela-navarro-garzon.Ufx9u6bWzptte2syzL3b2A/ |publisher=PSOE |accessdate=2019-02-25 |archiveurl=https://web.archive.org/web/20170610143301/http://www.psoe.es/transparencia/transparencia-cargos/ficha/micaela-navarro-garzon.Ufx9u6bWzptte2syzL3b2A/ |archivedate=2017-06-10 |language=es}}</ref>。2004年4月25日当选安达卢西亚政府平等与社会福利部部长<ref>{{cite web |author=Ministerio de Trabajo y Asuntos Sociales |title=BOE-A-2006-3116 |url=https://www.boe.es/diario_boe/txt.php?id=BOE-A-2006-3116 |publisher=[[西班牙国家官方公报|BOE]] |accessdate=2019-02-25 |pages=7180-7182 |language=es |date=2006-02-22}}</ref>。2008年至2016年担任[[安达卢西亚自治区政府|安达卢西亚自治区政府]]议会哈恩选区[[民意代表|民意代表]]<ref>{{cite web |title=la Junta de Andalucía |url=https://web.archive.org/web/20081217034226/http://andaluciajunta.es/especiales/aj-index-legviii.html |website=andaluciajunta.es |accessdate=2019-02-25 |language=es |date=2008-12-17}}</ref>。

2014年7月被任命为西班牙工人社会党主席<ref>{{cite web |author1=Anabel Díez |author2=Lourdes Lucio |title=Micaela Navarro será la nueva presidenta federal del PSOE |url=https://elpais.com/politica/2014/07/25/actualidad/1406314158_050188.html |publisher=[[国家报|El País]] |accessdate=2019-02-25 |language=es |date=2014-07-25}}</ref>，成为该党自成立以来的首位女性主席<ref>{{cite web |title=Micaela Navarro: "El PSOE no tiene que parecerse a nadie" |url=https://www.huffingtonpost.es/2014/07/26/micaela-navarro_n_5623748.html?utm_hp_ref=es-micaela-navarro |publisher=El[[HuffPost|HuffPost]] |accessdate=2019-02-25 |language=es |date=2014-07-26}}</ref>。
