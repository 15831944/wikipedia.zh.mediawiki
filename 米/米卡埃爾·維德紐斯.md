'''烏爾夫·米卡埃爾·維德紐斯'''（{{lang-sv|Ulf Michael Widenius}}，{{bd|1962年|3月3日}}），常暱稱作'''蒙提'''（Monty），芬蘭[[程式設計師|程式設計師]]與企業家，[[開放原始碼|開放原始碼]][[數據庫|數據庫]][[MySQL|MySQL]]的主要設計者，同時也是[[MySQL_AB|MySQL AB]]公司的創始成員和現任[[MariaDB|MariaDB]]的[[首席技術官|首席技術官]]。

== 生平 ==
=== 早年生活 ===
維德紐斯在1962年3月3日出生於[[赫爾辛基|赫爾辛基]]，為[[芬蘭瑞典族|芬蘭瑞典族]]。在從[[赫爾辛基理工大學|赫爾辛基理工大學]]中途輟學後，他於1981年開始為Tapio Laakso [[有限公司|Oy]]工作。到了1985年，維德紐斯與艾倫·拉爾森（Allan Larsson）一起在[[瑞典|瑞典]]創辦一間名叫TCX DataKonsult [[Aktiebolag|AB]]的[[資料倉儲|資料倉儲]]公司{{r|management}}。1995年，他開始與{{link-en|大衛·阿克馬克|David Axmark}}一同編寫[[MySQL|MySQL]]數據庫的第一個版本，並於隔年公開。2003年，維德紐斯獲頒芬蘭軟件年度[[創業|創業]]家獎<ref>{{cite web | publisher = Finnish Software Entrepreneurs Association | title = Supercellin tiimi on vuoden ohjelmistoyrittäjä | language = fi | accessdate = 2017-04-24 | url = https://ohjelmistoyrittajat.fi/fi/supercellin-tiimi-vuoden-ohjelmistoyritt%C3%A4j%C3%A4 | archive-url = https://web.archive.org/web/20160305234532/http://www.ohjelmistoyrittajat.fi/fi/supercellin-tiimi-vuoden-ohjelmistoyritt%C3%A4j%C3%A4 | archive-date = 2016-03-05 | dead-url = no }}</ref>。

在[[MySQL_AB|MySQL AB]]於2008年被[[昇陽電腦|昇陽電腦]]收購前<ref>{{cite web |publisher = [[MySQL_AB|MySQL AB]] |work = News & events |title = Sun to Acquire MySQL |url = http://www.mysql.com/news-and-events/sun-to-acquire-mysql.html |accessdate = 2008-09-05 |deadurl = yes |archiveurl = https://web.archive.org/web/20080905194020/http://www.mysql.com/news-and-events/sun-to-acquire-mysql.html |archivedate = 2008-09-05 }}</ref>，他一直都是MySQL AB的[[首席技術官|首席技術官]]，同時也是MySQL開發的主要推動者之一<ref>{{cite web |first = Kaj |last = Arnö |title = Monty: The First MySQL Fellow |date = Apr 2007 |publisher = Kaj Arnö |url = http://kajsql.kajarno.com/2007/04/monty-first-mysql-fellow.html |accessdate = 2012-10-23 |archive-url = https://web.archive.org/web/20131227035332/http://kajsql.kajarno.com/2007/04/monty-first-mysql-fellow.html |archive-date = 2013-12-27 |dead-url = no }}</ref>。

=== 昇陽收購MySQL ===
維德紐斯於2008年1月將MySQL AB出售予昇陽，MySQL AB在2008年的資本收益約為1660萬歐元（總收入達1680萬歐元），使他在那年成為芬蘭十大最高收入者之一<ref name="iltasanomat">{{cite web |work = Ilta-Sanomat |title = Suomen 100 suurituloisinta 2008 |url = http://www.iltasanomat.fi/uutiset/kotimaa/uutinen.asp?id=1748083 |date = 2009-11-02 |language = fi |location = [[Finland|FI]] |accessdate = 2009-11-06 |archive-url = https://www.webcitation.org/66sRmKP7p?url=http://www.iltasanomat.fi/kotimaa/Suomen%20100%20suurituloisinta%202008/art-1288338449921.html |archive-date = 2012-04-12 |dead-url = no }}</ref><ref name="hs">{{cite web |work = Helsingin Sanomat |title = Verotiedot 2008 |language = fi |location = Helsinki, FI |url = http://www.hs.fi/verotiedot/list.do?personPattern=widenius&year=2008&submit=%A0Hae+tiedot%A0 |accessdate = 2009-11-06 }}</ref>。2009年2月5日，維德紐斯宣佈離開昇陽，以開創自己的公司<ref>{{cite web |publisher = Monty's blog |title = Time to move on |url = http://monty-says.blogspot.com/2009/02/time-to-move-on.html |accessdate = 2009-02-05 |archive-url = https://www.webcitation.org/66sRoDBX1?url=http://monty-says.blogspot.com/2009/02/time-to-move-on.html |archive-date = 2012-04-12 |dead-url = no }}</ref>。

=== 離開昇陽之後 ===
2009年12月12日，維德紐斯請求MySQL客戶[[遊說集團|遊說]][[歐盟執行委員會|歐盟執行委員會]]（EC）展開關於[[甲骨文公司|甲骨文]]收購昇陽的辯論，原因是其擔心前者會獨攬MySQL<ref>{{cite web |publisher = Monty's blog |title = Help saving MySQL |url = http://monty-says.blogspot.com/2009/12/help-saving-mysql.html |accessdate = 2009-12-12 |archive-url = https://www.webcitation.org/66sRoxkR1?url=http://monty-says.blogspot.com/2009/12/help-saving-mysql.html |archive-date = 2012-04-12 |dead-url = no }}</ref>，此舉導致了一個名叫「拯救MySQL」的在線請願活動<ref>{{cite web |title=Help saving MySQL |url=http://monty-says.blogspot.com/2009/12/help-saving-mysql.html |last=Michael |first=Widenius |date=2009-12-12 |work=Monty Says |publisher=Google |accessdate=2012-09-16 |archive-url=https://www.webcitation.org/66sRoxkR1?url=http://monty-says.blogspot.com/2009/12/help-saving-mysql.html |archive-date=2012-04-12 |dead-url=no }}</ref>。

離開昇陽之後，維德紐斯創立Monty Program AB並開發出MySQL的分支——以其最小的女兒瑪莉亞（Maria）命名的[[MariaDB|MariaDB]]<ref>{{cite web |url = https://kb.askmonty.org/en/why-is-the-project-called-mariadb/ |title = Why is the project called MariaDB? |work = AskMonty |type = knowledgebase |accessdate = 2013-02-07 |archive-url = https://web.archive.org/web/20130604042605/https://kb.askmonty.org/en/why-is-the-project-called-mariadb/ |archive-date = 2013-06-04 |dead-url = no }}</ref>。它同時也包含數個由公司自己和社群開發的[[補丁|補丁]]和[[插件|插件]]，其中一個為從Maria重命名，以避免與MariaDB混淆的亞莉亞（Aria）{{link-en|資料庫引擎|storage engine}}。Monty Program AB接著與SkySQL合併，後來改名為MariaDB公司（MariaDB Corporation）。維德紐斯還是MariaDB基金會的[[首席技術官|首席技術官]]，該基金會是一個負責推廣、維護和發展MariaDB代碼庫、社群和[[商業生態系統|生態系統]]的非營利組織<ref name="ceo_mf">{{cite web | url=https://mariadb.com/about/leadership | title=Leadership | publisher=mariadb.com | accessdate=2014-12-08 | archive-url=https://web.archive.org/web/20141008023547/https://mariadb.com/about/leadership | archive-date=2014-10-08 | dead-url=no }}</ref>。

開放數據庫聯盟（ODBA）<ref>{{Citation |url = http://odba.org/ |title = ODBA |publisher = Open database alliance |type = official [[World_Wide_Web|World Wide Web]] site |access-date = 2012-10-15 |archive-url = https://web.archive.org/web/20121020052549/http://odba.org/ |archive-date = 2012-10-20 |dead-url = no }}.</ref>是一個於2009年由Monty Program和{{link-en|Percona|Percona}}的組織。據其第一次公告所指：「開放數據庫聯盟將由一群公司合作，共同為MariaDB提供軟件、支援和服務。MariaDB是一個企業級，由社區開發的MySQL分支」<ref>{{cite web |publisher = Open Database Alliance |title = Welcome |url = http://odba.org/news/welcome_to_the_odba |accessdate = 2012-10-18 |deadurl = yes |archiveurl = https://web.archive.org/web/20120920033119/http://odba.org/news/welcome_to_the_odba |archivedate = 2012-09-20 }}</ref><ref name="mf">{{cite web | url=https://mariadb.org/en/foundation/ | title=About the MariaDB Foundation | publisher=mariadb.org | accessdate=2014-12-08 | archive-url=https://web.archive.org/web/20150214002251/https://mariadb.org/en/foundation/ | archive-date=2015-02-14 | dead-url=no }}</ref>。

== 個人生活 ==
維德紐斯與其第二任妻子安娜（Anna）、女兒瑪莉亞（Maria）一起在[[赫爾辛基|赫爾辛基]]居住；他的前妻生育有女儿米（{{lang|sv|My}}，MySQL得名于她）<ref>{{cite web |url = http://dev.mysql.com/doc/refman/5.1/en/history.html |title = History of MySQL |work = MySQL |edition = 5.1 |type = Reference Manual |publisher = MySQL |accessdate = 2011-08-26 |archive-url = https://web.archive.org/web/20110902052729/http://dev.mysql.com/doc/refman/5.1/en/history.html |archive-date = 2011-09-02 |dead-url = no }}</ref>和兒子馬克斯（Max）。
