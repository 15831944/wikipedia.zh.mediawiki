'''米歇尔·奥斯汀·莫兰德'''（{{lang-en|Mitchell Austin Moreland}}，{{bd|1985年|9月6日|||Moreland, Mitch}}）是效力于[[美国职棒大联盟|美国职棒大联盟]][[波士顿红袜|波士顿红袜]]的一名[[一垒手|一垒手]]。之前曾效力于[[德克萨斯游骑兵_(棒球)|德州游骑兵]]。

==业余生涯==
莫兰德出生和成长在[[密西西比州|密西西比州]]的[[艾默里_(密西西比州)|艾默里]]，曾在艾默里高中就读，高中时期莫兰德参加[[棒球|棒球]]、[[美式足球|美式足球]]和[[足球|足球]]三项运动。当时他和{{link-en|德里克·诺里斯|Derek Norris}}是队友。高四那年莫兰德发挥出色，投球成绩为7胜1败、防御率0.53，打击率为4成56，整个赛季仅被三振3次。高中毕业后，莫兰德进入[[密西西比州立大学|密西西比州立大学]]。大三那年，他打出了10支全垒打、3成43打击率的成绩，同时作为投手还投出了2胜0败、防御率3.46的成绩<ref>{{cite web|url=http://www.hailstate.com/ViewArticle.dbml?ATCLID=944206|title=Mitch Moreland - 2007 Baseball Roster - Mississippi State Athletics|publisher=|accessdate=2018-08-14}}</ref>。

==职业生涯==

===小联盟生涯===
[[File:Mitch_Moreland.jpg|thumb]]的莫兰德]]
莫兰德在[[2007年美国职棒大联盟选秀|2007年美国职棒大联盟选秀]]中第17轮被[[德克萨斯游骑兵_(棒球)|德州游骑兵]]挑中。和球队签约后，莫兰德从短1A的[[斯波坎印地安人|斯波坎印地安人]]起步，这个赛季他打出2成59打击率、2支全垒打、15分打点的成绩。

{{by|2008年}}莫兰德进入1A的[[克林顿伐木王|克林顿伐木王]]，整个赛季他的打击率达到3成26，并有8支全垒打和59分打点。他还入选了该赛季[[中西部联盟|中西部联盟]]的全明星赛阵容和季后赛全明星阵容<ref>{{cite web |url=http://www.oursportscentral.com/services/releases/?id=3708529 |title=Moreland, Gac Named Post-Season All-Stars |author= |date=2008年8月26日 |work=OurSports Central |publisher=oursportscentral.com |accessdate=2018年8月16日}}</ref><ref>{{cite web |url=http://www.clintonherald.com/sports/local_story_240114559.html?keyword=secondarystory |title=LumberKings top Chiefs in home finale |first=Marcus |last=Jacobs |date=2008年8月27日 |work=The Clinton Herald |accessdate=2010年1月18日 |deadurl=yes |archiveurl=https://archive.is/20120722035642/http://www.clintonherald.com/sports/local_story_240114559.html?keyword=secondarystory |archivedate=2012年7月22日 }}</ref>。

{{by|2009年}}莫兰德分别效力于小联盟的两支球队。他先在高阶1A的[[贝克斯菲尔德火焰|贝克斯菲尔德火焰]]打出了3成41打击率、8支全垒打和26分打点，随后升入2A的[[佛利斯柯义勇骑兵|佛利斯柯义勇骑兵]]，打出了3成26打击率、8支全垒打和59分打点的成绩<ref>{{cite web |url=http://texas.rangers.mlb.com/news/article.jsp?ymd=20091029&content_id=7576930&vkey=news_tex&fext=.jsp&c_id=tex |title=Moreland keeps it simple by hitting |first=Rush |last=Olson |date=October 29, 2009年10月29日 |publisher=[[mlb.com|mlb.com]] |accessdate=2010年1月18日 |archive-url=https://web.archive.org/web/20110605164711/http://texas.rangers.mlb.com/news/article.jsp?ymd=20091029&content_id=7576930&vkey=news_tex&fext=.jsp&c_id=tex |archive-date=2011-06-05 |dead-url=yes }}</ref>。赛季结束后他还参加了[[亚利桑那秋季联盟|亚利桑那秋季联盟]]的比赛。这个赛季他被球队评为“汤姆·格里夫小联盟年度最佳球员”<ref>{{cite web |url=http://mlb.mlb.com/news/article.jsp?ymd=20100107&content_id=7888804&vkey=news_mlb&fext=.jsp&c_id=mlb |title=Rangers announce Minors award winners |author=T.
