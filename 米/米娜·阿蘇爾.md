'''米娜·阿蘇爾'''（{{lang-en|Mila Azul}}，{{bd|1997年|12月1日}}），本名「Mila Dabalos」，烏克蘭女性成人模特兒及攝像頭女郎。

== 簡歷 ==
米娜·阿蘇爾從小在[[烏克蘭|烏克蘭]]西面一個小鎮長大和就學，後來移居首都[[基輔|基輔]]生活<ref name=MA1>{{Cite web |url=http://droyc-studios.com/mila-azul/ |title=DROYC STUDIOS - ABOUT MILA AZUL（Interview + Photography） |work=DROYC STUDIOS |accessdate=2019-03-13 |language=en }}</ref>，首份工作是替[[討債公司|討債公司]]收回債項，但始終不能接受那種專橫的討債方式，故只入職6個月便選擇離開<ref name=MA1/>。2016年，她為了賺外快以學習[[吉他|吉他]]和編寫歌曲，遂嘗試參加首次[[成人模特兒|成人模特兒]]試鏡，表現令[[裸體攝影|裸體攝影]]師很滿意，繼而再參與拍攝 Hegre.com 創辦人[[彼得·赫格雷|彼得·赫格雷]]（Petter Hegre）的作品，結果那組照片曝光後意外吸引許多同行者注意，更獲他們爭相邀約合作<ref name=MA1/><ref>{{Cite web |url=https://www.playboy.pl/dziewczyny/4196/mila-azul-slodka-ale-niegrzeczna |title=Mila Azul: słodka, ale niegrzeczna |work=[[波蘭|波蘭]][[花花公子|花花公子]]（網上版） |date=2018-04-13 |accessdate=2019-03-13 |language=pl }}</ref>，故此順勢加入該行業。

米娜·阿蘇爾因樣貌甜美且身材突出，2017年曾擔任電腦軟件「iStripper」桌面舞者之一<ref>{{Cite web |url=https://www.xbiz.com/news/220554/istripper-offers-monthly-talent-contest-names-mila-azul-talent-of-the-month |title=iStripper Offers Monthly Talent Contest, Names Mila Azul 'Talent of the Month' |work=[[XBIZ|XBIZ]] |date=2017-06-02 |accessdate=2019-03-13 |language=en }}</ref>，又以 [[MyFreeCams.com|MyFreeCams.com]] [[攝像頭|攝像頭]]女郎身份，拍攝過逾百部[[自慰|自慰]]影片而受到不少網民歡迎<ref name=MA1/><ref>{{Cite web |url=https://www.menclub.hk/girls/11268 |title=「烏克」麗麗 歐美 AV 女優清泉 Mila Azul |work=[[MenClub|MenClub]] |date=2017-07-29 |accessdate=2019-03-13 }}</ref><ref>{{Cite web |url=https://orientaldaily.on.cc/cnt/adult/20180511/mobile/odn-20180511-0511_00290_001.html |title=Mila Azul 沉醉自摸挑逗 |work=[[東方日報_(香港)|東方日報]] |date=2018-05-11 |accessdate=2019-03-13 }}</ref>；也曾成為2017年全球最大色情影音網站 [[Pornhub|Pornhub]] 用戶使用調查報告 ——「[[台灣|台灣]]地區最常搜尋的女演員」<ref>{{Cite web |url=https://www.ettoday.net/news/20180120/1097014.htm |title=AV 網站曝性癖　台人最愛烏克蘭「天然粉乳妹 Mila Azul」 |work=[[ETtoday新聞雲|ETtoday新聞雲]] |date=2018-01-20 |accessdate=2019-03-13 }}</ref>，超越[[日本|日本]][[AV女優|AV女優]] [[JULIA_(AV女優)|JULIA]] 及[[波多野結衣|波多野結衣]]<ref>{{Cite web |url=https://www.setn.com/e/news.aspx?newsid=339555 |title=AV 網站數據出爐！台灣人最愛女優　烏克蘭嫩模力退波多野 |work=[[三立新聞網|三立新聞網]] |date=2018-01-20 |accessdate=2019-03-13 }}</ref>。
