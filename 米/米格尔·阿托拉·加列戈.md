'''米格尔·阿托拉·加列戈'''<ref>{{cite book |editor=辛华 |title=西班牙语姓名译名手册 |date=1973 |publisher=[[商务印书馆|商务印书馆]] |location=北京 |edition=修订本}}</ref>（{{lang-es|Miguel Artola Gallego}}，{{bd|1923年|7月12日||}}）是西班牙[[历史学家|历史学家]]和[[作家|作家]]。他出名于对{{tsl|es|Liberalismo español|西班牙自由主义}}的研究。<ref>{{cite web |title=Memoria de historiador |publisher=El País|url=https://elpais.com/cultura/2013/05/09/actualidad/1368093831_343963.html|archiveurl=https://web.archive.org/web/20181220074758/https://elpais.com/cultura/2013/05/09/actualidad/1368093831_343963.html |archivedate=2018-12-20 |language=es |date=2013-05-11}}</ref>

==经历==
加列戈曾在马德里中央大学（现[[马德里康普顿斯大学|马德里康普顿斯大学]]）学习[[文学|文学]]和[[哲学|哲学]]。1960年，他成为[[萨拉曼卡大学|萨拉曼卡大学]]西班牙史教授。<ref>{{cite web |title=Miguel Artola Gallego |url=https://www.biografiasyvidas.com/biografia/a/artola.htm |publisher=Biografías y Vidas |accessdate=2018-12-20 |language=es}}</ref>之后他主要进行对西班牙[[西班牙民主转型|旧政权]]向自由[[资产阶级|资产阶级]]社会过渡的研究，并出版了一系列著作。1981年，他当选为西班牙皇家历史学院院士。<ref>{{cite web |title=Miguel Artola Galleg |url=http://www.rah.es/d-miguel-artola-gallego/texto-procedente-del-diccionario-biografico-espanol-d-miguel-artola-gallego/ |publisher=Real Academia de la Historia |accessdate=2018-12-20 |language=es}}</ref>1991年获得[[阿斯图里亚斯亲王奖|阿斯图里亚斯亲王奖]]。<ref>{{cite web |title=MIGUEL ARTOLA GALLEGO |url=http://www.fpa.es/en/princess-of-asturias-awards/laureates/1991-miguel-artola-gallego.html?texto=trayectoria&especifica=0 |publisher=FUNDACIÓN PRINCESA DE ASTURIAS |accessdate=2018-12-20 |language=es}}</ref>
