'''米莱娜·帕夫洛维奇-巴里利'''（{{lang-sr-Latn|Milena Pavlović-Barili}}，{{lang-sr-cyr|Милена Павловић-Барили}}；{{bd|1909年|11月5日|1945年|3月6日}}）是[[塞爾維亞族|塞爾維亞]][[绘画|画家]]和[[诗人|诗人]]，也是知名的[[現代主義|現代主義]]女性[[艺术家|艺术家]]之一。<ref>{{cite web|url=http://stazenezele.rs/milena-pavlovic-barili/|title=Milena Pavlović Barili|last=Vukovic|first=Sofija|publisher=stazenezele|language=sr|accessdate=2016-01-02|deadurl=yes|archiveurl=https://web.archive.org/web/20150224032819/http://stazenezele.rs/milena-pavlovic-barili/|archivedate=2015-02-24}}</ref>

==生平==
父亲是[[意大利|意大利]][[作曲家|作曲家]]布鲁诺·巴里利<ref>{{cite web |title=Bruno Barilli (1880-1952) |url=http://www.rodoni.ch/busoni/wolfbusoni/brunobarilli.html |publisher=rodoni |accessdate=2019-03-19 |language=it}}</ref>；母亲是塞尔维亚王国[[卡拉喬傑維奇王朝|卡拉喬傑維奇王朝]]的后代，曾在[[慕尼黑|慕尼黑]]的一所音乐学院学习钢琴和演唱。父母于1905年11月在慕尼黑相识。<ref>{{cite book|author=Јанковић, Оливера|year=2001|title=Милена Павловић Барили|location=Београд|publisher=Topy|isbn=978-86-335-0098-2| |language=sr|page=15-25}}</ref>

米莱娜出生在[[波扎雷瓦茨|波扎雷瓦茨]]，1922–1926年在[[塞尔维亚|塞尔维亚]][[贝尔格莱德|贝尔格莱德]]皇家艺术学院学习，1926年在[[慕尼黑美术学院|慕尼黑美术学院]]学习，师从[[弗朗茨·冯·施图克|弗朗茨·冯·施图克]]。<ref name=o>{{cite web |title=O Mileni |url=http://www.fondacijamilenindom.com/o-mileni/ |publisher=Fondacija Milenin Dom |accessdate=2019-03-19 |archiveurl=https://web.archive.org/web/20171213011645/http://www.fondacijamilenindom.com/o-mileni/ |language=sr |archivedate=2017-12-13}}</ref>1928年夏离开[[慕尼黑|慕尼黑]]，与母亲一起前往巴黎，参观了[[卢浮宫|卢浮宫]]。<ref>{{cite web |author=Milisav Milenković |title=Рушила баријере |url=http://www.novosti.rs/%D0%B4%D0%BE%D0%B4%D0%B0%D1%82%D0%BD%D0%B8_%D1%81%D0%B0%D0%B4%D1%80%D0%B6%D0%B0%D1%98.524.html:279749-%D0%A0%D1%83%D1%88%D0%B8%D0%BB%D0%B0-%D0%B1%D0%B0%D1%80%D0%B8%D1%98%D0%B5%D1%80%D0%B5 |publisher=Компанија Новости |accessdate=2019-03-19 |language=sr |date=2009-07-08}}</ref>

1930年开始，在[[西班牙|西班牙]]、[[罗马|罗马]]、[[法国|法国]]、[[伦敦|伦敦]]等地旅游，受[[让·谷克多|让·谷克多]]、[[安德烈·布勒東|安德烈·布勒東]]和[[乔治·德·基里科|乔治·德·基里科]]等大师的影响。1939年[[第二次世界大战|第二次世界大战]]爆发前夕，从[[勒阿弗尔|勒阿弗尔]]搬到了[[纽约|纽约]]。1940年，受[[萨尔瓦多·达利|萨尔瓦多·达利]]参展画商的邀请，在纽约举办了个人画展。<ref name=serb>{{cite web|author=Dejan Djorić|url=http://www.nacionalnarevija.com/en/tekstovi/Br%2016/Barili.html |title=Driven Away to Succeed. Milena Pavlović-Barili. (1909-1945), Truly Serbian, Truly International|publisher=Национална Ревија - Serbia National Review|year=2009|accessdate=2011-11-20|language=en}}</ref>

1943年5月，在[[华盛顿|华盛顿]]的一场画展中遇到了一位年轻的[[美国|美国]]飞行员士官，同年12月结婚。1944年夏，在[[新婚旅行|新婚旅行]]的一次郊游途中不幸从马背坠地，[[背部|背部]]摔伤。<ref>{{cite book|author=Mazzola, Adele|year=2009|title=Aquae passeris: o Mileni Pavlović Barili|location=Beograd|publisher=HISPERIAedu|isbn=978-86-7956-021-6|page=169-172|language=sr}}</ref>

1945年3月由于[[心脏病|心脏病]]发作，猝死在纽约的公寓里。1947年遗体被转移到罗马，1949年8月5日埋葬在[[泰斯塔西奥|泰斯塔西奥]]公墓。<ref name=o/>
==遗产==
在去世后，她的一些画作在意大利[[罗马|罗马]]、美国[[纽约|纽约]]的博物馆以及塞尔维亚[[貝爾格勒當代藝術博物館|貝爾格勒當代藝術博物館]]和家乡[[波扎雷瓦茨|波扎雷瓦茨]]的展览馆展出。<ref>{{cite web|url=http://www.serbia.com/about-serbia/famous-serbs/famous-artists/milena-pavlovic-barilli/|title=Milena Pavlović Barilli, she painted the world|publisher=Serbia.com|accessdate=2016-01-02|language=en}}</ref>

她一生还用[[意大利语|意大利语]]、[[西班牙语|西班牙语]]、[[法语|法语]]和[[塞尔维亚语|塞尔维亚语]]4种语言写了60首诗，其中一些发表在各国报刊杂志上。<ref>{{cite web |title=Милена Павловић Барили: Звезда уметничког номадизма |url=https://patriot.rs/milena-pavlovic-barili-zvezda-umetnickog-nomadizma/ |publisher=Патриот |language=sr |date=2017-01-31}}</ref>
