'''米日古丽·图尔荪'''（{{lang-ug|مېھرىگۈل تۇرسۇن}}，1989年—），[[维吾尔族|维吾尔族]]，公民权力奖的获得者。她自称曾在“[[新疆再教育营|新疆再教育营]]”受到酷刑，并自称一个儿子在中国当局监管期间神秘死亡。<ref>{{cite web |url=https://www.rfa.org/english/news/uyghur/detentions-11012018100304.html|title=Interview: ‘I Did Not Believe I Would Leave Prison in China Alive’| work=www.rfa.org| accessdate=2018-11-01}}</ref>中国外交部详细反驳了米日古丽的说法。<ref name="华" />

==在美国作出的证词==
2018年11月26日，米日古丽·图尔荪在华盛顿特区的{{le|国家记者俱乐部 (美国)|National Press Club (United States)|国家记者俱乐部}}作证。她声称这些营地的被拘留者遭到殴打、挨饿、电击和脱衣搜查。 她说：“我的手被他们打出血，每次电击时，我的整个身体都会剧烈摇晃，我能感受到静脉的疼痛，我想我宁愿死也不愿经历这种折磨，并恳求他们杀了我”。<ref>{{cite web |url=https://www.telegraph.co.uk/news/2018/11/28/begged-kill-uighur-woman-describes-torture-us-politicians/|title='I begged them to kill me', Uighur woman describes torture to US politicians| work=www.telegraph.co.uk| accessdate=2018-11-26}}</ref><ref>{{cite web |url=https://www.japantimes.co.jp/news/2018/11/27/asia-pacific/uighur-woman-describes-torture-drugging-degrading-treatment-chinese-detention-camp/#.XBNeWtIza70|title=Uighur woman describes torture, drugging and degrading treatment in Chinese detention camp| work=www.japantimes.co.jp| accessdate=2018-11-27}}</ref><ref>{{cite web |url=https://www.washingtontimes.com/news/2018/nov/26/woman-tells-of-torture-beatings-in-chinese-detenti/|title=Woman describes torture, beatings in Chinese detention camp| work=www.washingtontimes.com| accessdate=2018-11-26}}</ref><ref>{{cite web |url=https://www.smh.com.au/world/asia/china-will-retaliate-to-any-us-sanction-over-muslim-uighurs-envoy-20181128-p50iu7.html|title=China will retaliate to any US sanction over Muslim Uighurs: envoy| work=www.smh.com.au| accessdate=2018-11-28}}</ref>

2018年11月28日，米日古丽·图尔荪通过翻译向[[国会及行政当局中国委员会|国会及行政当局中国委员会]]就她在三次拘留中遭受的悲惨经历作证。她说：“大约有60人被关在430平方英尺的牢房里，在晚上，要10到15名女性站起来，我们其他人才能侧躺着睡觉，然后我们每2个小时就轮换一次。有些人一年多都没洗过澡。<ref>{{cite web |url=https://www.independent.co.uk/news/world/asia/uighur-muslim-china-mihrigul-tursun-torture-reeducation-camps-a8656396.html|title=Muslim woman describes torture and beatings in China detention camp: ‘I begged them to kill me’| work=www.independent.co.uk| accessdate=2018-11-28}}</ref><ref>{{cite web |url=http://time.com/5467628/china-uighur-congress-torture/|title='I Begged Them to Kill Me.' ==中国外交部的反驳==
中国外交部发言人称米日古丽的说法为“一个撒谎者编造的虚假故事”。根据外交部的说法，米日古丽从来没有被收监，也从来没有被职业技能教育培训中心收教。2017年4月21日至5月10日，米日古丽因涉嫌煽动民族仇恨和民族歧视被新疆且末县公安局刑事拘留。除了这20天以外，米日古丽在中国期间是完全自由的，而且多次出境。<ref name="华" />

外交部也反驳了米日古丽的一个儿子在乌鲁木齐儿童医院死亡的说法。根据外交部的说法，2016年，米日古丽的一个儿子木俄子由米日古丽本人及其家人送至新疆乌鲁木齐市儿童医院住院治疗三次。2018年4月22日，米日古丽和丈夫携木俄子和米日古丽的女儿持埃及护照离开中国。米日古丽的另一个儿子2016年1月由米日古丽携带离开中国赴土耳其。<ref name="华" />

==中国官媒的反应==
2018年11月27日，中国官媒《[[环球时报|环球时报]]》发表评论文章驳斥，“中国人一眼就看出这个女子在撒谎”，并认为她是要借此获得留在美国的难民资格，又或收受他人好处。<ref>{{cite web |url=http://www.globaltimes.cn/content/1129284.shtml|title=Bias stops West seeing real Xinjiang| work=www.globaltimes.cn| accessdate=2018-11-27}}</ref><ref>{{cite web |url=https://www.cincainews.com/s/1698019/20181128-china-state-media-bias-stops-west-seeing-real-xinjiang|title=维族女揭新疆再教育营惨况 华官媒：一看就知道是撒谎| work=www.cincainews.com| accessdate=2018-11-28}}</ref>

2018年12月3日，中国官方媒体《环球时报》发布了一篇批评米日古丽·图尔荪的文章。文章写道：“西方媒体爱她，好像他们中了大奖一样，最终抓住证人的证词攻击中国，同时对她所说的所有漏洞都如此无动于衷。 一些外国人花钱买米日古丽·图尔荪的故事，因为他们对中国产生了严重的误解，这种误解源于无知。<ref>{{cite web |url=http://www.globaltimes.cn/content/1130164.shtml|title=Have Western media given up duty of objective reporting?| work=www.globaltimes.cn| accessdate=2018-11-03}}</ref>
