'''米尼阿久尔餐厅旧址'''，是[[全國重點文物保護單位|全國重點文物保護單位]][[哈尔滨犹太人活动旧址群|哈尔滨犹太人活动旧址群]]的一部分，亦是[[哈爾濱市保護建築|哈爾濱市保護建築]]之一，位於中國[[黑龙江省|黑龙江省]][[哈爾濱市|哈爾濱市]][[道里區|道里區]][[中央大街|中央大街]]85號。米尼阿久爾咖啡茶食店在1920年代下旬設立，其中「米尼阿久爾」是[[俄語|俄語]]「精美藝術品」的意思，主要提供俄式[[糖果糕點|糖果糕點]]和[[咖啡|咖啡]]。它在1939年與另一商號合併為維多利亞西餐館，其後轉手並易名為紫羅蘭西餐館。後來，紫羅蘭西餐館倒閉，舊址建立了哈爾濱攝影社，後來又改為時裝店。在建築風格方面，米尼阿久尔餐厅旧址採用[[新藝術運動|新藝術運動]]風格，並可能是新藝術運動建築的終結。

== 歷史 ==
在米尼阿久尔餐厅開辦的時代，[[哈尔滨|哈尔滨]]很盛行西洋文化風尚，中国人流行学習做[[西餐|西餐]]<ref name="新晚报2014"/>。1920年代下旬（一說1926年，一說1928年），猶太人伊曼紐爾·阿那托維奇·卡茨（{{lang|en|Emanuel Anatolevich Katz}}）在[[中央大街|中央大街]]58号建立了米尼阿久尔咖啡茶食店，其中「米尼阿久尔」是[[俄語|俄語]]「{{lang|ru|Миниатюр}}」音譯，即「精美艺术品」的意思<ref name="新晚报2014"/><ref name="新晚报2012"/><ref name="黑龙江新闻网"/>。它是一所[[西餐廳|西餐廳]]，主要提供以[[莫斯科|莫斯科]]製法來製作的莫斯科式[[糖果糕點|糖果糕點]]和[[咖啡|咖啡]]，也有西餐、[[中餐|中餐]]和各种饮料，並設独特的[[早餐|早餐]]和[[晚餐|晚餐]]餐點<ref name="新晚报2014"/>。餐廳聘用的厨师都是中国名厨，包括厨师李帮庆和孟宪廷，还有合稱「四大义」的著名厨师王洪义、杨洪义、尤洪义、朱凤义<ref name="新晚报2012"/>。

因應米尼阿久尔的成功，中國本土的大罗香糖果工厂开始細心留意细品糖果的製作，又注意店面装潢是否美观，從而與俄國的糖果糕點抗衡<ref name="哈尔滨市档案局 "/>。

1938年後，佔領[[中國東北|中國東北]]的[[滿州國|滿州國]]（由[[大日本帝國|大日本帝國]]控制）实施[[配给制|配给制]]，物资匮乏，哈尔滨饮食业进入寒冬，西餐厅门庭冷落<ref name="新晚报2014"/>。1939年，[[白俄羅斯人|白俄羅斯人]]布列斯把米尼阿久尔咖啡茶食店與一所中國人经营的商號（一說為[[西义顺|義順合]]，一說為「义气商号」）合并，組成维多利亚西餐馆<ref name="新晚报2014"/><ref name="新晚报2012"/>。其後，维多利亚西餐馆轉由中国人卜大生经营，易名為紫罗兰西餐馆，仍然是哈尔滨著名的西餐厅<ref name="新晚报2012"/>。

在[[文化大革命|文化大革命]]期間，餐厅柱头上的裸体女性雕塑受到人為破壞，已被砸掉<ref name="新晚报2014"/>。

紫罗兰西餐馆倒閉後，旧址建立了[[哈尔滨摄影社|哈尔滨摄影社]]，後來又改為一家时装店<ref name="新晚报2014"/><ref name="新晚报2012"/>。

== 建築特色 ==
米尼阿久尔餐厅旧址樓高兩層，為砖木结构，採用[[新艺术运动|新艺术运动]]的建筑风格，是1920年代罕有的新艺术运动建筑，並可能是新艺术运动建筑的终结<ref name="凤凰网黑龙江"/>。建築物外觀呈灰色，[[立面|正立面]]有曲线和圆环装饰；顶部有高大的[[女儿墙|女儿墙]]，女儿墙以铁艺玉石实砖垛相连而成，同樣有曲线装饰，高大得像多蓋了一层<ref name="黑龙江新闻网"/>。一樓採用了宽大的玻璃窗，二樓則有圓形和方形的额窗，構成对比效果<ref name="黑龙江新闻网"/>。整體而言，建筑物打破传统建筑的严谨构图，同時有华贵质感和曲线表现力，符合新艺术运动建筑风格的典型特点<ref name="黑龙江新闻网"/>。

== 保護 ==
米尼阿久尔餐厅旧址屬於[[哈爾濱市保護建築|哈爾濱市保護建築]]的第一类<ref name="凤凰网黑龙江"/>。2007年，米尼阿久尔餐厅旧址列入[[哈爾濱市文物保護單位|哈爾濱市文物保護單位]]，旧址南侧过道劃為重点保护区，东侧人行道則屬於一般保护区<ref name="哈尔滨市人民政府"/>。2013年，[[哈爾濱猶太人活動舊址群|哈爾濱猶太人活動舊址群]]列入[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]中的[[第七批全国重点文物保护单位中的近现代重要史迹及代表性建筑|近现代重要史迹及代表性建筑]]，米尼阿久尔餐厅旧址是其中一部分<ref name="黑龙江文化厅"/>。哈尔滨中央大街上的历史建筑（包括餐厅旧址在內）一直受到良好的保护，格局從來沒有重大变化<ref name="黑龙江新闻网"/>。
