'''米多比鎮久'''（[[永祿|永祿]]7年(1564年)一說[[天文|天文]]19年?(1550年？)－[[寬永|寬永]]10年(1633年9月11日)），[[日本|日本]][[戰國_(日本)|戰國時代]]之[[筑前國|筑前國]][[糟屋郡|糟屋郡]]的[[國人|國人]]領主，[[米多比直知|米多比直知]]（大學助、彈正忠鎮家）之子，幼名米王丸、五郎次郎，通稱新蔵人、三左衛門，官稱丹波守，異名為虎丹波。擔任[[立花氏|立花氏]]的[[家老|家老]]後受賜姓立花。妻為[[立花道雪|立花道雪]]繼女・安武吉子以及[[吉弘統幸|吉弘統幸]]之女。

== 出身 ==
原本是筑前國糟屋郡米多比山城的小領主，祖先是[[宣化天皇|宣化天皇]]的末裔[[丹治氏|丹治氏]]，後來略稱"丹"姓。和薦野氏為同族，因分支居住於米多比山因而改姓米多比。
鎮久的祖父米多比實元(次郎左衛門、主計頭、越中守)於[[弘治|弘治]]3年（1557年）7月，參加[[大友家|大友家]]討伐[[秋月文種|秋月文種]]、[[筑紫惟門|筑紫惟門]]之戰鬥，於是役戰死。

== 筑前的良臣 ==
[[永祿|永祿]]11年（1568年）原大友家臣的[[立花鑑載|立花鑑載]]起兵反叛，[[大友氏|大友氏]]的重臣，[[戶次鑑連|戶次鑑連]]奉命鎮壓後，鑑連被主君[[大友宗麟|大友宗麟]]任命繼承立花家為立花山城主，改名[[立花道雪|立花道雪]]。
鎮久其父米多比直知和薦野宗鎮由於在之前不響應立花鑑載的反叛計劃而在立花山城遭其謀殺，鎮久便在薦野宗鎮之子[[薦野增時|薦野增時]]的內應行動下配合道雪討伐立花鑑載，並一同成為了道雪的寄騎與力眾。往後因為鎮久為人剛勇果斷且恥為人後而成為道雪賞識的武將，並受大友宗麟(大友義鎮)受賜「鎮」字而取名為鎮久，與薦野增時一同成為輔佐道雪，成為立花雙翼中的副翼(副將)。
此外，鎮久尤善書道，其筆墨被誇讚為「有如傾國姿色之美女」。<ref>『旧柳川藩志』第十八章・人物・第七節書家列傳 P.763</ref>

== 立花式的忠誠 ==
自鎮久娶得吉子之後被賜姓立花，通稱「立花三左衛門」並官稱「丹波守」；被冠以立花姓的鎮久，當然不會做出任何背叛主家的事。就在[[天正|天正]]13年（1585年），道雪和紹運率軍參加筑後遠征之時，鎮久和[[薦野增時|薦野增時]]、[[十時連貞|十時連貞]]一同和少主[[立花宗茂|立花宗茂]]守在立花山城，而勢力龐大的筑前國人[[秋月種實|秋月種實]]則看準道雪和紹運不在，率領8千軍力想偷襲奪下立花山城，並且早先安排好內應，讓立花家臣中的[[櫻井治部|櫻井治部]]、[[櫻井中務|櫻井中務]]兄弟臨陣倒戈。

就在櫻井兄弟商量如何倒戈之時，不巧被鎮久耳聞聽到，櫻井兄弟爲怕事跡敗露，極力勸誘鎮久參加倒戈的行動，鎮久爲怕打草驚蛇，先假裝鎮重答應倒戈，隨後避開櫻井兄弟耳目到達宗茂處報告此事，宗茂聽後命令鎮久快速封鎖各處連結城外的關口，不一會宗茂便以驚人的手段平定此內應之事，使[[秋月種實|秋月種實]]計劃失敗，鎮久無疑立下一大忠功。

隨後鎮久擔任游擊隊率百餘人實行少主宗茂的夜襲計畫第一步，便是由鎮久率騎兵往秋月家的古處山城進攻，誘使秋月種實誤以為立花全軍進攻己方居城而分兵追擊鎮久，這時[[立花宗茂|立花宗茂]]和[[薦野增時|薦野增時]]對秋月軍本陣放火使秋月軍大亂，鎮久又甩開追兵和[[十時連貞|十時連貞]]配合宗茂及增時夾攻秋月軍本陣，使的秋月種實無功而返，鎮久此戰又立大功，展現了立花家一員應有的忠誠和勇猛。

== 大虎、小虎 ==
九州征伐過後，立花家領受筑後柳川13萬2千石的領地，鎮久則分封鷹尾城3千5百石的俸祿，為立花家五大重臣之一。

接著，鎮久以立花軍備隊大將的身分參加兩度朝鮮戰役(文祿・慶長之役)。於[[慶長|慶長]]2年([[1597年|1597年]])9月13日，立花軍於朝鮮唐島(巨濟島松真浦)駐扎，夜晚時分，鎮久巡邏海峽之時巧見一處虎洞，洞口還有一隻幼虎，鎮久爲免生亂以一隻短鐵砲將其射殺，隔日向主君宗茂報告過後，宗茂以‘子虎既然已死，那麼親生的大虎必定會回來報仇，請不要鬆懈了’為忠告給鎮久，果然鎮久再赴同一處，發現一隻大虎於洞口正因哀而怒，鎮久此次則換了把大隻的鐵砲將其射殺。後年鎮久則因為這殺虎的事件而把這二支愛用的鐵砲取名為「大虎」、「小虎」、自身也因此而得異名「虎丹波」二支鐵砲以及虎齒如今仍存在後代家中。

== 忠孝難兩全 ==
鎮久於朝鮮戰地征戰多年，隨主公宗茂參戰[[碧蹄館之戰|碧蹄館之戰]]<ref>『舊柳川藩志』第十八章・人物・第十六節 柳川偉人小伝（三）P.867 記載鎮久於碧蹄館之戰中奮勇驍烈連斬數十名敵人，激戰之程度使得自己的軍旗甚至染滿了敵人的血，至今此軍旗仍存在後代家中。</ref>等等大戰，之後也奮力於關原時期的立花軍相關戰事當中，從舊主[[立花道雪|立花道雪]]至[[立花宗茂|立花宗茂]]時代，立下的忠功絕不是少。

[[關原之戰|關原之戰]]後立花家被迫改易，鎮久相比其他立花家臣轉仕大名或者跟隨主公宗茂，選擇的卻是扶養道雪遺孀寶樹院以及遺女，立花家女當主的[[立花誾千代|立花誾千代]]。
鎮久因此依[[加藤清正|加藤清正]]的安排受領3千石於肥後腹赤村的一處陣屋敷中日日行孝於寶樹院，並且幫忙誾千代的日常事務。

然而誾千代卻於[[慶長|慶長]]7年（1602年）因為日日鬱悶和衣食不均而先於母親寶樹院逝去，寶樹院則於（1616年）年與世長辭，此後鎮久則因悲傷驅使而歸於隱居生活中。

== 丹波，晚年喪子痛 ==
最終立花家回歸柳川大名後，鎮久在宗茂再三請求下又仕官，任大番頭領1千石。
鎮久有三子，長子[[米多比鎮信|米多比鎮信]]（立花主馬、民部少輔、三左衛門）、次子[[小田部鎮教|小田部鎮教]](安太夫、右馬助、土佐守)以及三子[[米多比茂成|米多比茂成]]（采女、七兵衛、源太左衛門）。

次子鎮教因為立花家重臣[[小田部統房|小田部統房]]無嗣而過繼給統房，鎮信以及茂成則作為立花家回歸柳川後的重臣和側近活躍著，然而鎮信卻在[[寬永|寬永]]元年（1624年）年於柳川突然病死，而三子茂成也在[[寬永|寬永]]三年（1626年）於江戶為宗茂側近期間病死，接連的喪子之痛使鎮久悲傷不堪，終於[[寬永|寬永]]十年（1633年）也隨之逝去，享年69歲，法名：空源院西譽榮鎮大居士(一說法號為清竜院)。羿月其妻吉子也隨之逝去。

不過鎮久的噩運卻並無因為自己的逝去而結束，繼承米多比氏立花家的鎮信之子鎮實（米多比鎮實、三左衛門）也就是鎮久的嫡孫，雖然在鎮信死後擔任立花家大組頭之一，卻也在[[寬永|寬永]]十四年（1637年）年爆發的島原之亂，於12月20日隨主君立花忠茂於一次進攻之中擔任先鋒，雖力戰斬殺數人卻遭到敵方猛烈的銃彈亂射中彈戰死，年僅25歲。

此後米多比家由鎮實之子米多比鎮俊(三左衛門、丹波)、茂成之子米多比茂辰(出雲、七兵衛)之家系延續下去。

== 付記 ==
鎮久與其米多比一族於柳川城的堀割發展也有所貢獻，於柳川城南東側所建造的「米多比堤」如今被指定為福岡縣柳川市的文化財而被保存下來。
