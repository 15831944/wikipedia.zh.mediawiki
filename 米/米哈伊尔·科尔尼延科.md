'''米哈伊尔·鲍里索维奇·科尔尼延科'''（{{lang-ru|Михаи́л Бори́сович Корние́нко}}，{{bd|1960年|4月15日|}}）是[[俄罗斯|俄罗斯]]飞行工程师，已退役[[宇航员|宇航员]]。

==早年==
出生在[[萨马拉州|萨马拉州]][[塞茲蘭|塞茲蘭]]。父亲鲍里斯·科尔尼延科（Борис Корниенко）是一名军队[[飞行员|飞行员]]，在1965年10月[[Mi-6直昇機|Mi-6直昇機]]事故中坠毁身亡<ref>{{cite web |url=http://www.nasa.gov/mission_pages/station/expeditions/expedition23/kornienko_interview_prt.htm |title=Preflight Interview: Mikhail Kornienko |date=23 March 2010 |publisher=NASA}}</ref>。早年生活在[[车里雅宾斯克州|车里雅宾斯克州]]的[[南烏拉爾斯克|南烏拉爾斯克]]，1977年毕业于[[車里雅賓斯克|車里雅賓斯克]]第15中学。1978年至1980年在空降部队服役。1981年至1987年在[[莫斯科航空学院|莫斯科航空学院]]学习，并获得了[[工程学|工程学]]学位。1980年至1986年在莫斯科执法部门工作。1986年开始在机械工程设计局担任测试[[工程师|工程师]]。1991年至1995年为一些商业公司工作。1995年4月开始在[[科罗廖夫能源火箭航天集团|科罗廖夫能源火箭航天集团]]担任工程师职务。<ref name=nasa>{{cite web |title=Biographical Data:Mikhail Borisovich Kornienko |url=https://web.archive.org/web/20180620192515/https://www.jsc.nasa.gov/Bios/htmlbios/kornienko.html |publisher=[[NASA|NASA]] |accessdate=2019-04-13 |language=en |date=2018}}</ref>

==宇航员生涯==
1998年2月，他成为未来宇航员的人选，开始在[[加加林宇航员培训中心|加加林宇航员培训中心]]接受基础训练。1999年获得测试宇航员资格。2008年8月成为[[远征23|远征23]]航空工程师预定人选，接受高级培训。<ref name=nasa/>

哈萨克斯坦时间2010年4月2日，[[联盟-FG|联盟-FG]]型运载火箭在[[拜科努尔航天发射场|拜科努尔航天发射场]]成功将载有科尔尼延科、[[亚历山大·斯克沃尔佐夫|亚历山大·斯克沃尔佐夫]]以及美国宇航员[[特雷西·考德威尔·戴森|特雷西·考德威尔·戴森]]的[[联盟TMA-18|联盟TMA-18]]号飞船送入太空近地轨道。<ref>{{cite web |author=中国新闻网 |title=俄联盟号载人飞船发射成功 3名宇航员上太空 |url=http://news.sohu.com/20100402/n271285998.shtml |publisher=[[搜狐|搜狐]] |archiveurl=https://web.archive.org/web/20190413064950/http://news.sohu.com/20100402/n271285998.shtml |archivedate=2019-04-13 |date=2010-04-02}}</ref>

[[莫斯科时间|莫斯科时间]]2010年7月27日8时11分（[[北京时间|北京时间]]12时11分），他和[[费奥多尔·尤尔奇欣|费奥多尔·尤尔奇欣]]穿着俄制“奥兰-MK”舱外宇航服，打开[[码头号对接舱|码头号对接舱]]舱门，开始了太空行走，整个过程共持续6小时43分。他们主要完成了两项任务：一是从[[星辰号服务舱|星辰号服务舱]]外部更换了一个[[摄像头|摄像头]]，用于监测欧洲自动货运飞船与国际空间站俄罗斯舱段的对接过程；二是分别在[[黎明号实验舱|黎明号实验舱]]、星辰号服务舱和[[曙光號功能貨艙|曙光號功能貨艙]]之间铺设并接通了[[电缆|电缆]]，这标志着黎明号小型实验舱将正式投入使用。<ref>{{cite web |author=聂云鹏 |title=国际空间站俄宇航员成功完成太空行走 |url=http://news.sciencenet.cn/htmlnews/2010/7/235248.shtm |publisher=[[新华社|新华社]] |archiveurl=https://web.archive.org/web/20190413062353/http://news.sciencenet.cn/htmlnews/2010/7/235248.shtm |archivedate=2019-04-13 |date=2010-07-29}}</ref>

2010年9月24日，由于[[联盟号宇宙飞船|联盟号宇宙飞船]]与国际空间站分离失败，科尔尼延科、斯克沃尔佐夫以及戴森的返航计划被迫推迟至25日。[[俄羅斯航太|俄羅斯航太]]官员表示故障原因为连接飞船与空间站的插销未成功开启，这类[[故障|故障]]尚属首次发生。<ref>{{cite web |title=飞船与空间站分离失败 俄3宇航员返航计划推迟 |url=http://www.chinanews.com/gj/2010/09-25/2552253.shtml |publisher=[[中国新闻网|中国新闻网]] |archiveurl=https://web.archive.org/web/20190413063753/http://www.chinanews.com/gj/2010/09-25/2552253.shtml |archivedate=2019-04-13 |date=2010-09-25}}</ref>
===远征43/44/45/46===
2015年3月，他和斯科特·凯利以及根纳季·帕达尔卡乘坐[[联盟号宇宙飞船|联盟号宇宙飞船]]从哈萨克斯坦发射升空，抵达空间站。<ref>{{cite web |title=美俄宇航员开启创纪录的太空之旅 |url=https://www.voachinese.com/a/us-russia-space-20150328/2698599.html |publisher=[[美国之音|美国之音]] |accessdate=2019-04-13 |date=2015-03-29}}</ref>他将和[[美国|美国]]宇航员[[斯科特·凯利|斯科特·凯利]]在国际空间站上度过约一年的时间，以便更好地了解太空生活对人体的长期影响。<ref>{{cite web |title=看到流星赶快许愿？有可能只是宇航员的排泄物 |url=http://www.chinadaily.com.cn/micro-reading/interface_yidian/2015-09-17/14193469.html |publisher=[[中国日报|中国日报]] |archiveurl=https://web.archive.org/web/20190413061609/http://www.chinadaily.com.cn/micro-reading/interface_yidian/2015-09-17/14193469.html |archivedate=2019-04-13 |date=2015-09-17}}</ref>

2016年1月1日，他和[[尤里·马连琴科|尤里·马连琴科]]、[[谢尔盖·沃尔科夫|谢尔盖·沃尔科夫]]在国际空间站用[[中文|中文]]问候中国人民新年好。<ref>{{cite web |title=俄罗斯宇航员太空用中文问候中国人民：新年好 |url=http://www.xinhuanet.com//world/2016-01/01/c_128588496.htm |publisher=[[新华网|新华网]] |archiveurl=https://web.archive.org/web/20190413061105/http://www.xinhuanet.com//world/2016-01/01/c_128588496.htm |archivedate=2019-04-13 |date=2016-01-01}}</ref>

莫斯科时间2015年8月10日17时17分（北京时间22时17分），他和[[根纳季·帕达尔卡|根纳季·帕达尔卡]]走出国际空间站，在进行了约6小时的舱外任务后顺利返回[[码头号对接舱|码头号对接舱]]。<ref>{{cite web |author1=李圆 |title=俄罗斯宇航员顺利完成出舱任务 |url=http://pic.people.com.cn/n/2015/0811/c1016-27445326.html |publisher=[[人民网|人民网]] |archiveurl=https://web.archive.org/web/20190413061931/http://pic.people.com.cn/n/2015/0811/c1016-27445326.html |archivedate=2019-04-13 |date=2015-08-11}}</ref>

莫斯科时间2016年3月2日4时左右（北京时间9时左右），他和斯科特·凯利以及沃尔科夫乘坐联盟TMA-18M飞船与国际空间站分离。经过数小时飞行后，飞船返回舱于莫斯科时间7时25分（北京时间12时25分）在[[哈萨克斯坦|哈萨克斯坦]]南部的[[杰兹卡兹甘|杰兹卡兹甘]]地区成功着陆。<ref>{{cite web |author=张继业 |title=国际空间站３名宇航员安全返回地面 |url=http://world.chinadaily.com.cn/guoji/2016-03/02/content_23712053.htm |publisher=新华社 |archiveurl=https://web.archive.org/web/20190413063132/http://world.chinadaily.com.cn/guoji/2016-03/02/content_23712053.htm |archivedate=2019-04-13 |date=2016-03-02}}</ref>
