'''米尔顿·弗里德曼自由奖'''（{{lang-en|'''Milton Friedman Prize for Advancing Liberty'''}}）是[[美国|美国]][[自由意志主義|自由意志主義]]智庫[[卡托研究所|卡托研究所]]頒發的奖项，奖项以[[美国|美国]][[诺贝尔经济学奖|诺贝尔经济学奖]]获得者[[米尔顿·弗里德曼|米尔顿·弗里德曼]]命名。<ref name="fhsa " >{{cite web |url= http://finance.ifeng.com/news/people/20120329/5827856.shtml|title=茅于轼获2012弗里德曼自由奖 潘石屹任志强等祝贺 |publisher=凤凰网 |accessdate= 2012年|work= |language =zh-cn}}</ref><ref name="albbw " >{{cite web |url= http://info.china.alibaba.com/news/detail/v0-d1025353915.html|title=茅于轼获弗里德曼自由奖 |publisher=阿里巴巴 |accessdate= 2012年|work= |language =zh-cn}}</ref>

== 介绍 ==
米尔顿·弗里德曼自由奖创建于2002年，每2年一次，审核会一共9人，奖金25万美元，提名者面向全球，主要颁发给“推动个人自由”的人士。<ref name="albbw " />
