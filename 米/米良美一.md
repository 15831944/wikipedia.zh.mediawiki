'''米良美一'''（{{lang-ja|米良 美一}}，{{bd|1971年|5月21日|||Mera Yoshikazu}}）是一名出生于[[宮崎縣|宮崎縣]][[西都市|西都市]]的[[日本|日本]]男性聲樂家。擅於以[[假聲男高音|假聲男高音]]演唱，個人的音域範圍為3個半8度[[音階|音階]]。<ref>{{cite web|url=http://www.bach-cantatas.com/Bio/Mera-Yoshikazu.htm |title=Yoshikazu Mera (Counter-tenor) |work=bach-cantatas.com |accessdate=2012年11月14日 }} </ref>[[人類身高|身高]]142.7cm<ref>[http://doraku.asahi.com/hito/interview/html/100115.html 米良美一さん（1/3）- インタビュー - ] {{webarchive|url=https://web.archive.org/web/20120716035607/http://doraku.asahi.com/hito/interview/html/100115.html |date=2012-07-16 }} </ref>。

== 經歷 ==
=== 小時 ===
米良美一出生於[[西都市|西都市]]。因先天患有[[成骨不全症|成骨不全症]]疾病問題，小時的米良美一常飽受發生骨折的困擾。而受此症狀影響，之後成年的米良美一身高也僅142公分左右。<ref>{{cite web |url=http://doraku.asahi.com/hito/interview/html/100115.html |script-title=ja:クラシックにこだわらず歌で人を喜ばせる原点に戻った 第百三十二回 米良美一さん (1) |work=DORAKU |accessdate=2012年11月14日 |deadurl=yes |archiveurl=https://web.archive.org/web/20120716035607/http://doraku.asahi.com/hito/interview/html/100115.html |archivedate=2012年7月16日 }} </ref>

在小時米良美一便喜歡上唱歌，但在就讀西都市立三財小學時，曾因成骨不全症的關係而唱歌姿勢怪異的模樣被一些孩子們給[[欺凌|欺凌]]。7歲過後為了養病因素，米良美一改在宮崎縣立赤江養護學校裡住宿。當時他在養護學校迷上了歌手[[松田聖子|松田聖子]]，並受[[日本歌謠曲|日本歌謠曲]]、[[演歌|演歌]]、[[民謠|民謠]]等類型音樂所影響。<ref>{{cite web |url=http://doraku.asahi.com/hito/interview/html/100115_02.html |script-title=ja:クラシックにこだわらず歌で人を喜ばせる原点に戻った 第百三十二回 米良美一さん (2) |work=DORAKU |accessdate=2012年11月14日 |deadurl=yes |archiveurl=https://web.archive.org/web/20130728174814/http://doraku.asahi.com/hito/interview/html/100115_02.html |archivedate=2013年7月28日 }} </ref>

=== 青年期 ===
青年時的米良美一仍持續居住在養護學校，而米良美一為了排解寂寞，開始有在學校浴室裡吟唱的習慣。之後學校的護理長注意到米良美一的行為後，開始鼓勵他學習用風琴演唱。數年後米良美一決定走上音樂家之路，在不顧父親反對下進入了[[洗足學園音樂大學|洗足學園音樂大學]]。

=== 成年期 ===
在洗足學園音樂大學就讀的米良美一專攻將[[男高音|男高音]]轉換為女聲般的[[假聲男高音|假聲男高音]]領域，之後在1994年從大學畢業後曾獲得當年第8屆{{nihongo|古典樂大賽|古楽コンクール}}的大獎、以及隔年在奏樂堂的{{nihongo|日本歌曲大賽|日本歌曲コンクール}}第3名。

=== 職業歌手 ===
1996年米良美一加入了唱片公司[[King_Records|King Records]]，並在同年發表個人第一張專輯{{nihongo|《母親之歌 日本歌曲集》|母の唄 日本歌曲集}}。之後當時正製作動畫《[[魔法公主|魔法公主]]》的動畫導演[[宮崎駿|宮崎駿]]在收音機上聽到米良美一演唱的歌曲引起了興趣，便邀請他為《魔法公主》演唱[[魔法公主_(單曲)|同名的片尾曲]]。<ref>{{Cite book |script-title=ja:もののけ姫 ロマンアルバム |location=日本 |publisher=[[德間書店|德間書店]] |date=1997年11月1日 |pages=66頁  |ISBN=978-4-19-720026-9 |accessdate=2012年11月14日 }}</ref>替《魔法公主》演唱後的米良美一開始獲得關注，除在1998年的[[日本金唱片大獎|日本金唱片大獎]]獲得最佳古典音樂歌手的榮譽外，也在同年的[[1998年冬季帕拉林匹克運動會|冬季帕拉林匹克運動會]]的開幕式演唱，<ref>{{cite web|url=http://www.shinmai.co.jp/olympic/para/kaikai.htm |script-title=ja:長野パラリンピック開会式の主な内容 |work=信濃毎日新聞 |accessdate=2012年11月14日 }} </ref>隨後則於1999年推出個人第一張原創專輯《I'll be there》。

在2003年前米良美一曾一度因為壓力而無法表現出過去的美聲，在走出低潮後開始朝向日本海外演出，並數次與[[艾迪塔·葛貝洛娃|艾迪塔·葛貝洛娃]]同台共演。除持續進行演唱的活動外，2007年米良美一也發行了自傳性質的書籍《{{lang|ja|天使の声〜生きながら生まれ変わる}}》，並多次參加地方巡迴演講、以自身過去的經歷作為鼓舞大眾的題材。
