'''米亚·法罗'''（{{lang|en|Mia Farrow}}，全名{{lang|en|Maria de Lourdes Villiers-Farrow}}，{{bd|1945年|2月9日|catIdx=Farrow, Mia}}）是[[美國|美國]]女演員，演出超過四十部影片並獲得[[金球獎_(影視獎項)|金球獎]]、[[英國電影和電視藝術學院|英國電影和電視藝術學院]]最佳女演員等獎項。同時她也身兼[[聯合國兒童基金會|聯合國兒童基金會]]親善大使的身份。

== 早期 ==
法羅出生於[[加州|加州]][[洛杉磯|洛杉磯]]，父親為[[澳洲|澳洲]]籍導演約翰·法羅（John Farrow），母親為愛爾蘭籍知名演員莫琳·歐莎莉文（Maureen O'Sullivan）。其兩位妹妹Tisa Farrow與Prudence Farrow亦為女演員。米亞從小在[[南加州|南加州]][[比佛利山|比佛利山]]成長，因父母拍片的關係，而有機會隨他們到處旅行。1947年米亞與母親和妹妹拍攝了一部短片，此為她初次於螢幕上露面。1950年，她在[[美國政府|美國政府]][[冷戰|冷戰]]宣傳片Duck and Cover中演出。

== 職業生涯 ==
法羅參為選秀為取得電影《[[音乐之声_(电影)|真善美]]》中長女Liesl Von Trapp的角色（最終由Charmian Carr獲得此角色）。1960年代，她由擔任配角而開始其職業演員生涯。爾後她在黃金時段的肥皂劇《冷暖人間》的演出而成為知名女演員。她與[[法蘭克·辛納屈|法蘭克·辛納屈]]因此片而相識並結婚。1968年，在[[羅曼·波蘭斯基|羅曼·波蘭斯基]]導演的《[[失嬰記|失嬰記]]》中，米亞首度擔任女主角。此片為米亞法羅深具義意的演出並因此而獲得金球獎新進女演員獎（此獎項在1983年後就不再頒發）。因《失嬰記》的成功，米亞躍為一線女演員。

== 支援人權 ==
米亚·法罗為[[聯合國兒童基金會|聯合國兒童基金會]]當[[義工|義工]]，為《[[芝加哥論壇報|芝加哥論壇報]]》寫[[文章|文章]]，愛發表[[國際|國際]]、[[政治|政治]]、[[人道主義|人道主義]]的個人見解。米亚·法罗是[[人權|人權]]運動活躍人士，她關注[[蘇丹|蘇丹]][[種族屠殺|種族屠殺]]問題。近來她於個人網站[http://www.miafarrow.org/]詳細明列如何參與反對達佛種族著殺，並貼出她個人探訪達佛、查德及中非共和國的照片和親身體驗。2007年，米亚·法罗曾經到過[[查德|查德]]、[[盧安達|盧安達]]、[[亞美尼亞|亞美尼亞]]、[[波士尼亞|波士尼亞]]、[[德國|德國]]、[[柬埔寨|柬埔寨]]等地作特殊項目[[旅遊|旅遊]]，她的言論一直受到不少輿論的壓力。

2007年3月28日，米亚·法罗在[[美国|美国]]《[[华尔街日报|华尔街日报]]》上刊文，稱北京奥运会是“种族屠杀奥运会”，批評担任北京奥运会艺术顾问的美国导演[[斯蒂芬·斯皮尔伯格|斯蒂芬·斯皮尔伯格]]。2008年，由於西藏[[314事件|314事件]]，斯皮尔伯格在當地左派工會威脅罷工的壓力下宣布辞去[[北京奥运会|北京奥运会]]艺术顾问一职。[[香港外國記者協會|香港外國記者協會]]透露，米亚·法罗計劃在2008年5月2日於奧運[[2008年夏季奧林匹克運動會聖火傳遞|聖火]]在香港傳送當日訪問[[香港|香港]]，發表人權演講。但她在入境期間遭到[[香港政府|香港政府]]官員的留難，此後未再獲准進入中國。她在2014年公開支持香港的[[雨傘革命|雨傘革命]]。
