'''米哈游网络科技'''（全称'''上海米哈游网络科技股份有限公司'''，简称'''米哈游'''或'''miHoYo'''）是一家成立于2012年2月13日总部位于[[中华人民共和国|中华人民共和国]][[上海|上海]]的一家[[ACG|ACG]][[动作游戏|动作游戏]]公司。它以开发和管理了[[动作游戏|动作游戏]][[崩坏系列|崩坏学院]]而闻名。

==公司简介==
米哈游主要由三位热爱技术和国产[[动漫|动漫]]文化的[[上海交通大学|上海交通大学]]学生创立。公司成立于2012年，业务主要集中在国产动漫文化下的移动游戏、[[漫画|漫画]]等领域。

2011年，米哈游创始团队以独立开发者的身份推出了游戏《FlyMe2theMoon》，并仅发布于[[iOS|iOS]]平台。

2012年米哈游公司正式注册成立，上线了移动游戏《崩坏学园》（已停止运营）

2014年，推出[[崩坏学园2|《崩坏学园2》]]。同年，《崩坏3》开始立项研发。

2016年10月，[[崩坏3|《崩坏3》]]正式上线。

2018年11月，《崩坏4》开始立项研发。（具网友透露，《崩坏4》的人物是以《崩坏3》中的圣痕开发）

自2017年起，米哈游开始向[[中国证监会|中国证监会]]申报[[首次公开募股|首次公开募股]]，并于2018年1月收到反馈意见<ref>{{Cite web |url=http://www.xinhuanet.com/fortune/2018-04/24/c_129857635.htm |title=游戏公司IPO热情不减 8家游戏公司现身最新排队名单-新华网 |first=刘 |last=绪尧|date=2018-04-24|accessdate=2018-04-28 |work=新华网|language=zh-CN}}</ref>。

该公司曾因未取得许可证而进行网络游戏经营被文化执法部门处罚<ref name=caijing>{{Cite web |url=http://stock.caijing.com.cn/20180404/4431244.shtml |title=一款游戏独撑业绩，涉黄、无证经营遭罚，米哈游IPO要“崩坏”？_资本观察_证券频道首页_财经网 - CAIJING.COM.CN |accessdate=2018-04-28 |language=zh-CN}}</ref>。

==处罚==
2014年7月22日，米哈游因为未取得《网络文化经营许可证》从事网络游戏上网经营活动并诱导网络游戏用户投入网络游戏虚拟货币后以随机抽取的偶然方式获得网络游戏产品和服务的行为处以没收违法所得人民币16,408元和罚款人民币2万元的行政处罚<ref name=caijing></ref>。

2016年1月25日，因为公司以随机抽取的偶然方式，诱导网络游戏用户采取投入网络游戏虚拟货币方式获取网络游戏产品以及未对网络游戏用户使用有效身份证件进行实名注册的违法行为予以责令改正，并从轻处以罚款人民币2万元<ref name=caijing></ref>。
