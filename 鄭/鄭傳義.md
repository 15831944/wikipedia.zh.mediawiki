'''鄭傳義'''（{{bd|1958年|12月4日|2018年|12月7日|}}），[[中华民国|中华民国]]企業家。

== 经历 ==

鄭傳義畢業於[[東海大學|東海大學]]國貿系，於1984年進入[[長榮集團|長榮集團]]，2013年擔任[[長榮航空|長榮航空]]總經理，2016年3月自長榮航空退休，並於2016年7月受邀擔任[[航發會|航發會]]於[[中華航空|中華航空]]之法人董事。鄭傳義亦是[[星宇航空|星宇航空]]創辦人兼董事長[[張國煒|張國煒]]在長榮時期密切的合作夥伴。2018年5月10日，鄭傳義接任星宇航空副董事長<ref>{{cite web |title=鄭傳義辭華航董事   張國煒證實：來星宇當副董事長{{!}}自由財經 |url=http://ec.ltn.com.tw/article/breakingnews/2419191 |accessdate=2018-07-17}}</ref><ref>{{cite web |title=星宇航空副董事長 鄭傳義正式接任 {{!}} 即時 {{!}} 經濟日報 |url=https://money.udn.com/money/story/5641/3136895 |website=經濟日報 |accessdate=2018-07-17}}</ref>。

2018年12月7日，因[[肺癌|肺癌]]病逝於[[新北市|新北市]][[新店區|新店區]][[慈濟醫院|慈濟醫院]]，享壽60歲。<ref>[https://money.udn.com/money/story/5612/3524696 一起打天下 鄭傳義是張國煒最得力的助手]</ref>
