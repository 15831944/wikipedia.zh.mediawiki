'''鄭志剛'''博士{{JP}}（{{bd|1979年|11月}}），富商鄭氏家族之第三代接班人，為[[周大福集團|周大福]](控股)有限公司創辦人、全球華人十大富豪[[鄭裕彤|鄭裕彤]]的長孫及[[鄭家純_(企業家)|鄭家純]]之長子<ref>{{Cite web|url = http://www.telegraph.co.uk/finance/newsbysector/retailandconsumer/6366827/Adrian-Cheng-the-Asian-retail-heir-who-is-aiming-to-change-the-way-we-shop.html|title = Adrian Cheng: the Asian retail heir who is aiming to change the way we shop|accessdate = 2015-07-27|author = |date = |publisher = }}</ref>。他是現任[[新世界發展|新世界發展有限公司]]的執行副主席。在其管理的公司包括新世界發展（00017.HK )、[[新世界中國地產|新世界中國地產]]、[[新世界百貨_(中國)|新世界百貨]]中國（00825.HK )、[[周大福集團|周大福]]（01929.HK）以及[[國際娛樂|國際娛樂]]（01009.HK）<ref>{{Cite web|url = http://www.ft.com/intl/cms/s/0/424863ca-d33a-11e2-95d4-00144feab7de.html#slide0|title = Adrian Cheng adds shine to Chow Tai Fook brand|accessdate = 2015-07-27|author = |date = |publisher = }}</ref>。鄭志剛熱愛藝術，其藝術先鋒及專家形象在國際藝術圈裡享負盛名。他於2008年創立[[K11_(香港)|K11]]品牌<ref>{{Cite web|url = http://www.scmp.com/lifestyle/arts-culture/article/1733513/k11-founder-adrian-cheng-bringing-six-exhibitions-art-basel|title = How New World heir Adrian Cheng combines his passion for art with entrepreneurship|accessdate = 2015-07-27|author = |date = |publisher = }}</ref>，開創先河成立全球首個「購物藝術館」，成功結合藝術×商業模式。2012年，他被《[[財富_(雜誌)|財富]]》雜誌評為「2012 年全球 40 位 40 歲以下商界精英」之一<ref>{{Cite web|url = http://fortune.com/40-under-40/2012/adrian-cheng-38|title = 40 UNDER 40 2012 – Adrian Cheng|accessdate = 2015-07-27|author = |date = |publisher = }}</ref>，為榜上唯一香港人<ref>{{Cite news|url = http://www.scmp.com/news/hong-kong/article/1063837/adrian-cheng-sole-hongkonger-fortunes-under-40-list|title = Adrian Cheng is sole Hongkonger on Fortune’s under 40 list|last = |first = |date = |work = |accessdate = }}</ref>。

作為文化企業家，他不僅創造了K11，更創辦[[非營利組織|非牟利]]藝術基金會：K11 Art Foundation (「KAF」)<ref>{{Cite web|url = http://www.k11artfoundation.org/en/about-us/|title = About Chairman – K11 Art Foundation Official website|accessdate = 2015-07-27|author = |date = |publisher = }}</ref>，致力栽培具潛質的中國藝術人才及提供國際性的創意平臺以展示他們的作品至更廣泛的受眾，包括與巴黎PALAIS DE TOKYO（[[東京宮|東京宮]]）及[[中央美术学院|中央美術學院]]美術館開展三年合作計劃<ref>{{Cite web|url = http://www.ft.com/cms/s/0/f5dc36ee-5851-11e4-a31b-00144feab7de.html|title = Arts interview: Adrian Cheng|accessdate = 2015-07-27|author = |date = |publisher = }}</ref>。入選 2014年[[藝術觀察|Art Review]]雜誌 「最具影響力100人」<ref>{{Cite web|url = http://artreview.com/power_100/adrian_cheng/|title = Adrian Cheng – Art Review|accessdate = 2015-07-27|author = |date = |publisher = }}</ref>的鄭志剛，是公認為現今當代藝術界舉足輕重的人物。

== 商業發展 ==
鄭志剛博士畢業於[[哈佛大学|哈佛大學]]，於2003年取得東亞研究文學士（榮譽）學位。自哈佛畢業後，他往京都留學一年， 修讀[[日本藝術|日本藝術]]和文化。他在[[日本|日本]]的研究學習給他提供了一個嶄新的角度來徹底改變未來亞洲藝術世界<ref>{{Cite news|url = http://www.forbes.com/sites/alexandreerrera/2014/08/16/the-three-layers-a-new-era-of-collectors-in-china/|title = The Three Layers: A New Era of Collectors In China|last = |first = |date = |work = |accessdate = }}</ref>。他於2014年獲得[[薩凡納藝術設計學院|薩凡納藝術與設計學院]]（ SCAD ）的[[人文學科|人文學科]]榮譽博士學位，以及[[嶺南大學_(香港)|嶺南大學]]的榮譽院士<ref>{{Cite web|url = http://commons.ln.edu.hk/honorary_fellowship/2014/honorary_fellows/2/|title = Honorary Fellowship Award 2014|accessdate = 2015-07-27|author = |date = |publisher = }}</ref>。他相信歷史人文應當學習成基礎，而企業管理知識則可以從實踐中吸收。

大學畢業後，他曾在[[高盛|高盛]]、[[瑞士信贷集團|瑞信]]和[[瑞银集团|瑞銀]]當投資銀行家，並獲取企業金融的相關經驗，為他在新世界所作出的貢獻奠下基礎。在2014年，他帶領集團打破地產營業額紀錄，比上一年提高達50%。在鄭志剛主導下，集團於2015年4月與[[阿布扎比|阿布扎比]]投資局協定成立新合營企業，各占五成權益，並藉此重組旗下酒店業務，將3間香港酒店─香港[[君悅酒店|君悅酒店]]、[[萬麗海景酒店|萬麗海景酒店]]和[[尖沙咀|尖沙嘴]][[凱悅酒店|凱悅酒店]]注入合營公司，雙方將共同營運相關酒店業務。3間酒店出售並轉讓予合營企業的總代價為185億元。該等交易收取的款項有效顯示[[新世界集團|新世界集團]]重要資產的真正價值，同時大幅增加現金流，有助投資未來項目，擴充[[土地儲備|土地儲備]]及用作其他用途。

== 藝術成就 ==
[[鄭志剛|鄭志剛]]博士於2010年創辦非牟利藝術基金會K11 Art Foundation (「KAF」)<ref>{{Cite web|url = http://www.k11artfoundation.org/en/about-us/|title = About Chairman – K11 Art Foundation Official website|accessdate = 2015-07-27|author = |date = |publisher = }}</ref>， 致力栽培具潛質的中國藝術人才及提供國際性的創意平臺以展示他們的作品至更廣泛的受眾。他在2014年三月受訪於ARTnews雜誌時提及：『新銳藝術家有很高的潛質，但卻不廣為人知，原因是中國在這方面被拍賣市場和商業市場牽頭。我做的不是畫廊。作品我都不賣的。K11 Art Foundation 是一個註冊於香港的非牟利藝術組織，但我們面向整個[[大中華地區|大中華區]]，包括香港和[[臺灣|臺灣]]。』

在他統籌下，K11 Art Foundation與PALAIS DE TOKYO（[[東京宮|東京宮]]）在位於[[K11_(上海)|上海K11]]購物藝術中心B3層的chi K11藝術空間，共同簽署從2014年起的三年合作計劃，值此加強中法兩國之間的藝術交流和合作<ref>{{Cite news|url = http://www.forbes.com/sites/alexandreerrera/2014/01/21/the-five-myths-about-chinese-contemporary-art/|title = The Three Layers: A New Era of Collectors In China|last = |first = |date = |work = |accessdate = }}</ref>。

他前瞻性地創辦K11品牌，融合藝術、人文、自然，讓其成為一種極具創新意義的零售商業模式。他渴望創造屬於大眾的藝術 ，讓大眾和下一代能學習欣賞藝術。2014年，他把法國[[印象派|印象派]]大師[[克洛德·莫奈|莫奈]]的作品首次帶到中國大陸，展覽於上海K11購物藝術中心<ref>{{Cite web|url = http://www.chinadailyasia.com/life/2014-05/09/content_15134269.html|title = Shanghai artist brings Monet to the mall|accessdate = 2015-07-27|author = |date = |publisher = |deadurl = yes|archiveurl = https://web.archive.org/web/20160305133533/http://www.chinadailyasia.com/life/2014-05/09/content_15134269.html|archivedate = 2016-03-05}}</ref>。此次展覽，開拓了中國藝術家、藝術學生和廣泛觀眾對莫奈及印象派全新的詮釋和價值，及對中國[[當代藝術|當代藝術]]發展的思考。

== 法國藝術與文學軍官勳章 ==
鄭志剛於2017年12月9日，獲法國政府頒發法國藝術與文學軍官勳章，以表揚他在中法文化交流，及為中國當代藝術作出的貢獻。法國政府文化部於1957設立榮譽勳章至今，鄭志剛是香港中最年輕的一名軍官勳章授勳者。<ref>{{Cite web|url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20171211/20240704|title=鄭志剛獲頒法國勳章|accessdate=11-12-2017|author=|date=|publisher=}}</ref>
