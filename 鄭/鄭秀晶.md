== 早年生活與教育程度 ==
鄭秀晶於1994年10月24日在[[美國|美國]][[加利福尼亞州|加利福尼亞州]][[舊金山|舊金山]]出生，父親是一名拳擊選手，母親是一名體操運動員，姐姐為韓國女子團體[[少女時代|少女時代]]前成員[[鄭秀妍|鄭秀妍]]。2000年，鄭秀晶在韓國與媽媽、姐姐逛[[百貨公司|百貨公司]]的時候被[[SM娛樂|SM娛樂]]星探發掘，當時只有6歲的她，因為年齡太小，直至2006年才正式進入公司訓練。

鄭秀晶分別持有[[美國護照|美國護照]]和[[韓國護照|韓國護照]]，早年曾以英文本名Chrystal Soo Jung生活{{r|鄭秀晶英文本名為'''Chrystal Soo Jung'''之來源-01}}{{r|鄭秀晶英文本名為'''Chrystal Soo Jung'''之來源-02}}{{r|鄭秀晶英文本名為'''Chrystal Soo Jung'''之來源-03}}，後來[[韓國|韓國]]於2011年實施新的《國籍法修訂案》<ref>{{Cite web |url = http://www.epochtimes.com/b5/10/12/24/n3121954.htm |title = 韓國明年起施行雙重國籍法 | date = 2010-12-23 |publisher = 大紀元 |language = zh |accessdate = 2014-11-01 }}</ref>使鄭秀晶擁有[[雙重國籍|雙重國籍]]的身份{{r|鄭秀晶於韓國護照上的英文「성(姓)／Surname」和「이름(名)／Given names」為'''Jung Soo-jung'''之來源-01}}。

鄭秀晶回到韓國後一直就讀{{tsl|ko|한국켄트외국인학교|韓國肯特外語學校}}，之後轉學[[翰林演藝藝術高等學校|翰林演藝藝術高等學校]]。2012年，鄭秀晶被[[成均館大學|成均館大學]]和[[漢陽大學|漢陽大學]]同時錄取，與父母商議後最終選擇成均館大學。2013年2月7日，在首爾翰林藝術高中的畢業典禮上獲學校頒發成就獎。2013年2月27日，鄭秀晶參加開學典禮正式入讀[[成均館大學|成均館大學]]成為表演藝術學系的學生<ref>{{Cite web |url = http://mwave.interest.me/enewsworld/article/28610#gnb_header_language |title = f(x) Krystal Performs at Her High School Graduation | author=Hani Park | date=2013-02-07 | publisher=CJ E&M enewsWorld | language=en }}</ref>，入學時的成績更是該學系的第一名。

== 職業生涯 ==
===2009年－2010年===
* 鄭秀晶是韓國女子音樂團體[[f(x)_(組合)|f(x)]]的成員之一，成員包括中國籍[[宋茜|Victoria]]、美籍华裔[[劉逸雲|Amber]]和韓國籍[[朴善英|Luna]]、[[崔真理|Sulli]]。2009年9月5日以首張數位單曲《LA chA TA》正式出道。

* 2010年，《[[出發夢之隊2|出發夢之隊2]]》是韓國[[KBS|KBS]]電視台的運動型綜藝節目。3月21日，鄭秀晶在Couple特輯中與[[珉豪|珉豪]]獲得第一名，並以1.8m打破女子跳高紀錄。6月6日的女子跳高特輯，再次獲得第一名，更以1.95m打破自己所保持的跳高紀錄。
* 2010年7月，鄭秀晶出演家庭情景喜劇《[[越看越可爱|越看越可爱]]》，飾演郑秀晶，是一名性格开朗、古靈精怪的高中生，在劇中的著名台詞是：“正着念是鄭秀晶，倒着念也是鄭秀晶”和“哇哦”，並憑此劇榮獲MBC演藝大賞喜劇新人獎。

===2011年－2013年===
* 2011年，鄭秀晶接受出演韓國首個以花樣滑冰為主題的真人秀節目《[[金妍儿的Kiss_&_Cry|金妍儿的Kiss & Cry]]》。鄭秀晶與前花樣滑冰國家代表選手李東勳拍擋參賽，他們以高難度的託舉動作和出色的滑行技巧獲得評審和觀眾的注目。擔任評審的韓國著名職業花樣滑冰選手[[金妍兒|金妍兒]]更以「像真正的滑冰選手」形容鄭秀晶，使鄭秀晶獲得高度評價。節目進行期間，鄭秀晶曾因狀態不佳而沒有進食，比賽結束後感到無力暈倒。總決賽以Carmen獲得55.2高分並獲勝，於8月14日與[[金妍兒|金妍兒]]參加「All that Skate Summer 2011」花樣滑冰表演。

* 2011年9月，鄭秀晶出演[[文化廣播_(韓國)|MBC]]情景喜劇《[[Highkick3短腿的反擊|Highkick3短腿的反擊]]》播出，飾演性格單純、大大咧咧的高中生安秀晶，更帶動了「buing buing」撒嬌的潮流。

* 2013年，鄭秀晶得到在迷你連續劇演出的機會，參演由當紅人氣演員[[朴信惠|朴信惠]]、[[李敏鎬|李敏鎬]]及[[金宇彬|金宇彬]]主演的[[SBS_(韓國)|SBS]][[SBS水木迷你連續劇|水木迷你連續劇]]《[[繼承者們|繼承者們]]》，在劇中飾演李寶娜一角，扮演一名燦爛、富有的高中學生，性格刁蠻，但率真可愛，是個討人喜歡的女孩。在整個拍攝過程，鄭秀晶在合作明星和工作人員中相當受歡迎<ref>{{Cite web |url =http://www.allkpop.com/article/2013/11/minhyuk-says-he-cant-help-but-smile-due-to-krystals-cuteness |title =CNBLUE's Minhyuk says he can't help but smile due to Krystal's cuteness in preview for 'Cheongdamdong 111' |author = |date =2013-11-28 |publisher =allkpop.com |language =en |accessdate = |quote = }}</ref><ref>{{Cite web |url =http://www.allkpop.com/article/2014/01/kim-woo-bin-talks-about-the-women-of-heirs |title =Kim Woo Bin talks about the women of 'Heirs' |author = |date =2014-01-07 |publisher =allkpop.com |language =en |accessdate = |quote = }}</ref>。製作商亦讚揚鄭秀晶在片場能很投入李寶娜一角，大家都相當驚訝<ref>{{Cite web |url =http://mwave.interest.me/enewsworld/tw/article/46183 |title =Krystal戲裡戲外皆多金　飾Mega娛樂公司繼承人到位 |author =陳玉恬 |date =2013-09-16 |publisher =CJ E&M enewsWorld |language =zh }}</ref>。由鄭秀晶飾演的李寶娜與飾演尹燦榮的[[CNBLUE|CNBLUE]][[姜敏赫|姜敏赫]]這對情侶角色更在第二屆《Annual DramaFever Awards》獲得「最佳情侶獎」<ref>{{Cite web |url =http://www.dramafever.com/zh-cn/news/kang-min-hyuk-and-krystal-jung-win-best-couple-at-the-2013-dramafever-awards-/?pl=1&pc=DFSale25&utm_source=pjx&utm_medium=&utm_term=&utm_content=&utm_campaign=cmpjx&PJX=21181 |title =Kang Min Hyuk and Krystal Jung win Best Couple at the 2013 DramaFever Awards |author =CoCo Kdrama |date =2014-05-02 |publisher =dramafever |language =en |accessdate = |quote = }}</ref>。
