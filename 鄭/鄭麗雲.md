'''鄭麗雲'''（{{lang-en|Leigh Wen}}，{{bd|1959年|||}}），是一名旅美艺术家，生于[[台湾|台湾]][[台北|台北]]<ref>{{cite web | url = http://www.asiaartcenter.org/asia/portfolio/「宇宙洪荒」-郑丽云个展/?lang=zh-hans | title = 「宇宙洪荒」- 郑丽云个展 | work = 亚洲艺术中心 | accessdate = 2018-06-12}}</ref>。鄭麗雲在[[美國|美國]][[紐約市|紐約市]][[雀儿喜_(纽约)|雀兒喜]]<ref name = 'fine art'>{{cite web | url = http://leighwenfineart.com/about/ |title = 工作室簡介 | work = 鄭麗雲工作室 | accessdate = 2018-07-07}}</ref>和台灣陽明山都成立有工作室<ref>{{cite web | url = https://tw.asiatatler.com/life/鄭麗雲的線性創作語彙 |title = 鄭麗雲的線性創作語彙 | author = Cathy Chiang | date = 2016-09-12 | accessdate = 2018-07-07}}</ref>。

鄭麗雲是美國聯邦政府的合約畫家，2002年至2012年曾多次擔任美國國務院藝術大使<ref name = 'caini'>{{cite web | url = http://chinigallery.com/index.php?lang=tw&do=artist_introduction&artid=53 | title = 鄭麗雲簡介 | work = 采泥藝術| accessdate = 2018-07-04}}</ref>。鄭麗雲曾參與超過十四個國際展覽作品，同時她的作品曾展覽於16處[[美國大使館|美國大使館]]、總領事館及世界各政府機構，包括[[波蘭|波蘭]]、[[丹麥|丹麥]]、[[約旦|約旦]]、[[菲律賓|菲律賓]]、[[新加坡|新加坡]]、[[馬來西亞|馬來西亞]]、包貝多斯、[[博茨瓦納|博茨瓦納]]、[[賴比瑞亞|賴比瑞亞]]等歐、亞、非洲的美國領事館。其中，在包貝多斯美國大使館和[[美國在台協會|美國在台協會]]展出的作品中有8件已被美國聯邦政府收購，成為美國永久收藏<ref name = 'xuexue'/>。

== 人生經歷 ==
鄭麗雲幼時家境貧穷，父母曾在窯場工作，母親的工作是在瓷器上畫畫，這是鄭麗雲接觸繪畫的開端。鄭麗雲高中時考上了台北的[[臺北市私立再興高級中學|再興中學]]，但由於喜愛藝術，鄭麗雲後來進入了[[国立台湾艺术大学|國立藝專]]（今國立台灣藝術大學）。鄭麗雲在大學期間成績優秀，1980年被評為亞洲十大傑出才藝青年，1981年獲得美術學士（B.A.）學位<ref name = 'yuan'>{{cite web | url = http://125.227.255.111/yuan/web/113/pdf/P24-113.pdf | title = 鄭麗雲的藝術人生 | author = 白雪蘭 | accessdate = 2018-06-13}}</ref><ref name = 'up'>{{cite web | url = http://www.upmedia.mg/news_info.php?SerialNo=3323 |title = 逆流而上 不被埋沒的天份 - 鄭麗雲 | author = 許元龍 | date = 2016-08-31 | accessdate = 2018-07-03}}</ref>。

從國立藝專畢業之後，為了追求更開闊的創作觀，鄭麗雲1983年來到[[美國|美國]]留學。1984年鄭麗雲獲得[[華盛頓州立大學|華盛頓州立大學]]美術學士學位（B.F.A），1985年獲得[[紐約州立大學|紐約州立大學]]藝術碩士學位（M.A），1994年获得紐約州立大學的美術碩士學位（M.F.A）<ref name = 'yuan'/><ref>{{cite web | url = http://www.mingshanart.com/-leigh-li-yun-wen/ | work = 名山藝術 | title = 鄭麗雲簡介 | accessdate = 2018-07-03}}</ref>。

1996年鄭麗雲首次在[[台灣|台灣]]舉辦個人畫展。自從來美國留學之後，鄭麗雲就長期住在美國，但1997年父親去世之後，為了陪伴母親，鄭麗雲開始常回台灣探親，同時也慢慢接觸台灣藝術界。1999年到2005年間鄭麗雲在台灣和[[西班牙|西班牙]]兩地創作，尤其在2002年時鄭麗雲到西班牙北邊的卡特濃那藝術村創作一系列以「大地」為主題的作品。2005年，鄭麗雲應美國第一夫人[[劳拉·威尔士·布什|劳拉·布希]]（Laura Bush）之邀參訪白宮。2006年，鄭麗雲被美國政府派往非洲包茲瓦納擔任文化大使，考察當地文化藝術。<ref name = 'yuan'/>

2011年鄭麗雲在[[美國|美國]][[紐約市|紐約市]][[雀儿喜_(纽约)|雀兒喜]]成立工作室Leigh Wen Fine Art<ref name = 'fine art'/>。2013年鄭麗雲返回台灣，在陽明山成立工作室<ref>{{cite web | url = http://smagtw.org/smagtw_2015LeighWen.html | title = 物、語、花、女人——鄭麗雲個人展 | work = 現代畫廊 | accessdate = 2018-07-07}}</ref>。鄭麗雲在陽明山的家中種有花木，花卉同也是她創作的靈感來源。同年鄭麗雲舉辦以花為主題、名為「似花非花」的藝術展覽，向藝術史上的女性藝術家致敬<ref name = 'yuan'/><ref name = 'liuxian'>{{cite web | url = http://www.mingshanart.com/leigh-critics-6/ | author = 白雪蘭 | title = 流線風華——2016鄭麗雲個展於名山藝術 | work = 名山藝術 | accessdate = 2018-07-07}}</ref>。

== 地氣火風 ==

鄭麗雲的作品中多以地、氣、火、風為主題，這源於她常被「造物者大自然的力量震懾，並受東方哲學敬天愛地的思想影響」<ref name = 'liuxian'/>。
