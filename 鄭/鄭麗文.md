'''鄭麗文'''（1969年11月12日－）[[中國國民黨籍|中國國民黨籍]]政治人物，祖籍[[雲南省|雲南省]]，曾任[[行政院發言人|行政院發言人]]、[[立法委員|立法委員]]、[[中國國民黨中央委员会|中國國民黨中央委员会]]政策委员会副執行長、[[中國國民黨|中國國民黨]]發言人、[[中國國民黨中央委員會文化傳播委員會|中國國民黨中央委員會文化傳播委員會]]主任委員、[[國民大會|國民大會]]代表。

== 生平 ==
=== 民主進步黨時期 ===
鄭麗文在就讀[[國立臺灣大學|國立臺灣大學]]法律學系時，正趕上台灣的民主浪潮，鄭麗文開始投入學運，曾參與[[野百合學運|野百合學運]]、擔任國立臺灣大學大學論壇社社長、新生代國家安全論壇召集人等，積極參與[[民主進步黨|民主進步黨]]領導的街頭遊行活動。大學畢業後，加入民主進步黨，鄭麗文出任[[台灣人權促進會|台灣人權促進會]]秘書長。1996年代表民主進步黨當選第三屆[[國民大會|國民大會]]代表，參與了當時的修憲活動，1999年任民主進步黨國民大會黨團副總召集人。

2000年國民大會虛級化後，鄭麗文擔任民主進步黨青年部副主任，後期與[[陳水扁|陳水扁]]領導的民主進步黨理念發生落差，鄭開始在媒體上批評民主進步黨，並辭去青年部副主任一職，與民主進步黨主流漸行漸遠。

2002年[[立法委員|立法委員]][[李慶安|李慶安]]指控[[行政院衛生署|衛生署]]長[[涂醒哲|涂醒哲]]上KTV對男子鄭可榮「舔耳」引發爭論，涂醒哲堅持說他並沒有去KTV，民進黨內也宣佈站在涂醒哲這邊，不准黨員亂發言。但鄭麗文當時竟仍在和親民黨立委孫大千主持的 CALL IN 節目中說了對涂醒哲的一些負面揣測，引起民進黨不滿。過幾天後事件大逆轉，親民黨的李慶安出來道歉說自己認錯人，反突顯鄭麗文的處境相當尷尬，民進黨也在10月8日通過決議'''開除'''鄭麗文的黨籍<ref>{{Cite web |url = http://www.newtaiwan.com.tw/bulletinview.jsp?bulletinid=48740 |title = 只要鄭麗文能認錯罪應不至於需要開除黨籍 |agency = 台灣日報 |date = 2002-10-10 |publisher = 新台灣新聞週刊|accessdate = 2016-06-27 |quote =  }}{{dead link}}</ref>。而事後鄭麗文則一再堅稱是自己主動退黨，她說「當時覺得民進黨已偏離了原先的政治路線，不符合本人最初理想，加上對黨派作風不滿意，因此選擇了退出。」<ref>{{Cite web |url =http://news.sohu.com/20050429/n225399181.shtml |title = 国民党发言人郑丽文：我们积极回应总书记邀请 |agency = 新民晚报 |date = 2005-04-29 |publisher = 搜狐新闻|accessdate = 2016-06-27 |quote =  }}</ref>

=== 中國國民黨时期 ===
2002年与[[民主進步黨|民主進步黨]]决裂後，鄭麗文一度沒有任何[[公職|公職]]，但最終還是復出，對政治活動躍躍欲試。隨著對民主進步黨政府的批判加強，越來越多人認為鄭麗文與前民主進步黨婦女部主任[[陳文茜|陳文茜]]一樣轉投[[泛藍|泛藍]]陣營。2004年3月27日，泛藍陣營在[[凱達格蘭大道|凱達格蘭大道]]上舉行大規模集會抗議聲稱[[2004年中華民國總統大選|總統大選]]不公，鄭麗文與農運詩人[[詹澈|詹澈]]共同擔任主持，被視為正式挺藍。

2004年底鄭麗文投入[[2004年中華民國立法委員選舉|2004年中華民國立法委員選舉]]，披上「婚紗」吸引[[媒體|媒體]]目光，並說要「嫁給[[高雄市|高雄市]]」，於該市第二選區參選，最終以24,780票宣告落敗。立法委員選舉過後，由於出身學運與民主進步黨，選舉經驗豐富加上外型出色，在[[中國國民黨主席|中國國民黨主席]][[連戰|連戰]]的邀請下，在2005年正式加入[[中国国民党|中国国民党]]，並擔任[[中國國民黨中央委員會文化傳播委員會|中國國民黨中央委員會文化傳播委員會]]副主任委員與發言人等黨職。2005年中國國民黨主席選舉後，升任為國民黨文傳會主任委員。

鄭麗文在學運和民主進步黨時間累積的選務經驗及活動主持等能力，受到中國國民黨重用，於同年的[[2005年中華民國地方公職人員選舉|2005年中華民國地方公職人員選舉]]，不論是主持遊行或造勢活動，鄭伶俐的口齒讓人印象深刻，成功地拉抬[[中國國民黨|中國國民黨]]選情。

2005年，當選[[國民大會|國民大會]]代表。2006年7月鄭辭去文傳會主委，轉任中國國民黨中央委員會政策委員會副執行長。

[[2008年中華民國立法委員選舉|2008年中華民國立法委員選舉]]中，由中國國民黨提名為不分區立法委員候選人，排第十八名，終於當選。[[2012年中華民國立法委員選舉|2012年中華民國立法委員選舉]]中國國民黨徵召，參與[[臺中市第七選舉區|臺中市第七選舉區]][[立法委員|立法委員]]選舉，惟仍以8,637票的敗選。

2012年10月20日，鄭麗文接任[[行政院發言人|行政院發言人]]。<ref>{{cite news|last=蔡和穎|url=http://www.cna.com.tw/News/FirstNews/201210200016.aspx|title=政院發言人 內定鄭麗文|work=中央通訊社|date=2012-10-20 }}</ref>。2014年2月6日，鄭麗文請辭發言人，由[[孫立群|孫立群]]接任。

2014年4月，接任[[TVBS|TVBS]][[麗文正經話|麗文正經話]]主持人。

2018年8月，出任[[中國國民黨|中國國民黨]]義務副秘書長。2018年11月24日，[[2018年中華民國地方公職人員選舉|九合一選舉]]結束後，鄭麗文即離任國民黨副秘書長職務。

== 家庭生活 ==
大學時期認識大她一歲的學長[[駱武昌|駱武昌]]（曾任民進黨台北市黨部主委），兩人自學運時期即結為伴侶，於2011年10月登記結婚。
