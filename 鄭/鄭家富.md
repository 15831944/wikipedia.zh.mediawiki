'''鄭家富'''（{{lang-en|'''Andrew Cheng Kar-foo'''}}，{{bd|1960年|4月28日|catIdx=Z鄭}}），[[香港|香港]]出生，籍貫[[廣東|廣東]][[潮州|潮州]]，執業[[律師|律師]]，前[[香港立法會|香港立法會]]議員，前[[香港數碼電台|香港數碼電台]]節目主持。曾任1999年 - 2011年大埔中選區民選區議員。在[[香港數碼廣播停播風波|香港數碼廣播停播風波]]後轉投網路電台[[D100|D100]]，並曾主持節目[[十級自由Phone|十級自由Phone]]及[[風波裡的茶杯|風波裡的茶杯]]。

==簡歷==
鄭家富中學就讀於[[慈幼英文學校|慈幼英文學校]]，其後曾就讀於[[香港浸會大學|浸會書院]]、[[澳洲|澳洲]][[新南威爾士大學|新南威爾士大學]]、[[悉尼大學|悉尼大學]]及[[香港大學|香港大學]]，取得[[碩士|碩士]]學位。

鄭家富最初加入論政團體[[匯點|匯點]]，隨後合併成為[[民主黨_(香港)|民主黨]]成員。1995年，當屆[[1995年香港立法局選舉|立法局選舉]]新設立九個新功能界別（即[[新九組|新九組]]），鄭家富順利從金融保險地產及商業服務界得以當選為立法局議員，直至1997年「落車」。1998年後，一直自新界東選區直選加入立法會，直至2012年。

[[區議會|區議會]]方面，鄭家富初時為[[南區_(香港)|南區]]區議員，1999年起空降到[[大埔區議會|大埔區議會]]之大埔中競選，一直連任至2011年。[[2011年香港區議會選舉|2011年香港區議會選舉]]中，鄭家富並沒有報名，放棄連任。

===反對政改方案而退出民主黨===
2010年6月，立法會審議[[2012年香港政治制度改革|2012政改方案]]期間，鄭家富因就香港政治形勢評估及爭取民主政制的策略，與民主黨主流派意見呈嚴重分歧，故拒絕隨民主黨立場對政改方案投贊成票，並且在2010年6月23日於立法會會議上就議案辯論發言時，即場宣佈退出民主黨，其後並向外表示，將以獨立民主派成員身分繼續爭取民主，不會加入其他泛民政黨。<ref>{{cite web |last=星島日報 |url=http://www.singtao.com/yesterday/loc/0624ao01.html|title=《鄭家富 民主黨 各行各路》|publisher=香港：《星島日報網頁》港聞要聞版面（正體中文）|date=2010-06-24 |accessdate=2010-06-24}}</ref><ref>{{cite web |last=明報|url=http://news.mingpao.com/20100624/gac1.htm|title=《「忠義兩難全」 鄭家富退黨 何俊仁指難挽留：要講的都講了》|publisher=香港：《明報新聞網》要聞版面（正體中文）|date=2010-06-24 |accessdate=2010-06-24}}</ref>

在[[2012年香港政治制度改革|政改]]議案被通過後翌日的2010年6月26日，接受[[香港商業電台|商台]]節目《[[政經星期六|政經星期六]]》訪問，指民主派因政改一役四分五裂，呼籲民主黨在真普選問題上未有突破前，對於獲邀上京、授勳及發還回鄉證等問題小心處理，並謂政治上無永遠的朋友和敵人，希望自己可以做民主派的「中間人」，拉攏民主黨和其他民主派人士繼續合作。<ref>{{cite web |last=星島日報 |url=http://www.singtao.com/breakingnews/20100626a104135.asp|title=《鄭家富望做「中間人」》|publisher=香港：《星島日報網頁》港聞要聞版面（正體中文）|date=2010-06-26 |accessdate=2010-06-26}}</ref><ref>{{cite web|last=明報 |url=http://hk.news.yahoo.com/article/100626/4/iv8m.html|title=《授勳回鄉證上京問題 鄭家富囑民主黨小心》|publisher=香港：《明報》，轉引自《Yahoo! 新聞網站》港聞版面（正體中文）|date=2010-06-27|accessdate=2010-06-29}}</ref>

===2012年不競選立法會===
離開民主黨後，鄭家富和政改後退出民主黨的前成員為主的[[新民主同盟|新民主同盟]]緊密合作，不少海報均例如和擔任其立法會議員助理的沙田區議員[[丘文俊|丘文俊]]一同進行地區工作。但由於就[[五區公投|五區公投]]之立場，鄭家富和[[新民主同盟|新民主同盟]]之成員看法不盡相同，所以鄭家富未有加入[[新民主同盟|新民主同盟]]。<ref>[http://www.youtube.com/watch?v=of2QkEYv6h8 鄭家富(民主黨會大後)教協門外會見傳媒 ]</ref>

「拉布」完結後，鄭家富其後宣佈不會參與2012年香港立法會選舉，正式結束其議員生涯。2012年7月17日，鄭家富最後一次在議會發言，搏斥[[工聯會|工聯會]][[王國興|王國興]]的「拉布」浪費議會時間之說，大罵：「功能組別已經浪費了香港人很多的­時間！」<ref>[https://www.youtube.com/watch?v=QIp5HsJ72dY 〈鄭家富議員十七年立法會生涯的最後一次發言〉]，根據《進出口條例》就批准《2012年進出口(登記)(修訂)規例》而動議的擬議決議案，2012年7月17號晚，23:49pm。</ref>

[[2012年香港立法會選舉|2012年香港立法會選舉]]中，鄭家富「交波」支持[[新民主同盟|新民主同盟]][[范國威|范國威]]，還支持除了民主黨以外的民主派候選人，海報上印有他的支持海報，包括[[社民連|社民連]]的[[梁國雄|梁國雄]]、[[人民力量|人民力量]]的[[陳志全|陳志全]]、[[公民黨|公民黨]]的[[湯家驊|湯家驊]]及[[工黨_(香港)|工黨]]的[[張超雄|張超雄]]，最終成功配票，五人均勝出當選。范國威奪取了新界東最後一席，接替其在新界東原有議席。

===2016年復出參選立法會===
2015年中，一度傳出鄭家富打算復出參選立法會。<ref>[http://hk.apple.nextmedia.com/news/art/20160215/19491474 〈擬9月再戰立會新東選舉 鄭家富：我仲有團火 〉]，《蘋果日報》即時新聞，2016年2月15日。</ref>

2016年3月，鄭家富曾一度宣佈放棄再次參選，支持年輕新世代參政，甚至公開呼籲[[梁國雄|梁國雄]]也不要參選，他又說擔心自己出選會影響[[范國威|范國威]]、[[陳志全|陳志全]]選情，故決定放棄出選，支持[[范國威|范國威]]、[[陳志全|陳志全]]連任。<ref>[http://hk.apple.nextmedia.com/realtime/news/20160322/54898314 〈力保范國威同陳志全　鄭家富放棄新東復出 〉]，《蘋果日報》即時新聞，2016年3月22日。</ref>

但到了2016年7月12日，鄭家富改變主意宣佈將會復出參選[[2016年香港立法會選舉|2016年香港立法會選舉]]。<ref>{{cite web|title=【立會選舉】鄭家富再戰新東　無新同盟樁腳　范國威：摸不着頭腦|url=http://www.hk01.com/%E7%AB%8B%E6%B3%95%E6%9C%83%E9%81%B8%E8%88%89/31072/-%E7%AB%8B%E6%9C%83%E9%81%B8%E8%88%89-%E9%84%AD%E5%AE%B6%E5%AF%8C%E5%86%8D%E6%88%B0%E6%96%B0%E6%9D%B1-%E7%84%A1%E6%96%B0%E5%90%8C%E7%9B%9F%E6%A8%81%E8%85%B3-%E8%8C%83%E5%9C%8B%E5%A8%81-%E6%91%B8%E4%B8%8D%E7%9D%80%E9%A0%AD%E8%85%A6|publisher=香港01|accessdate=2016-07-12|date=2016-07-12}}</ref>9月4日，選舉當日[[新民主同盟|新民主同盟]]的3名地區樁腳區議員替他助選，包括[[陳兆陽|陳兆陽]]、[[趙柱幫|趙柱幫]]及[[丘文俊|丘文俊]]，丘文俊是他的前議員助理，鄭家富最終以全區第15高票的17,892票與[[范國威|范國威]]一同落選。<ref>[http://hk.apple.nextmedia.com/news/art/20160905/19760602 〈范國威 鄭家富 上演師徒決裂 〉]《蘋果日報》 2016年9月5日</ref>

== 投身電台時事評論 ==
2012年中，鄭家富加盟[[香港數碼電台|香港數碼電台]]成為電台節目主持，同年年底轉投網台[[D100|D100]]繼續主持節目[[十級自由Phone|十級自由Phone]]及[[風波裡的茶杯|風波裡的茶杯]]。

== 個人生活 ==
鄭家富是[[天主教徒|天主教徒]]，已婚，育有兩女；其妻子陳桂英為前民主黨成員兼前立法會議員[[黃成智|黃成智]]的表妹。<ref>[http://www.youtube.com/watch?v=qOM_EzCWT7w 人民力量 (第六十二集): 鄭家富與大嚿談退出民主黨的原因 2012.5.23 (1/2)]</ref>

鄭家富曾經超速危險駕駛，後來認罪及罰錢了事。<ref>[http://news.sina.com.hk/cgi-bin/nw/show.cgi/2/1/1/1065422/1.html 不涉意外事故 輕微控罪可書面認罪]</ref>

鄭家富亦經營一間婚禮證婚公司<ref>[http://www.centurywedding.com.hk/aboutus.html 世紀見証]</ref>，同時為一間律師事務所之合夥人<ref>{{cite web|title=周啟邦律師事務所 |url=http://www.kbchau.com.hk/en_partners.html }}</ref>。2012年10月，鄭家富成立「法正網絡」，提供義務法律諮詢服務。

鄭家富在2011年7月出書《政海一聲笑》，書版稅全數撥捐香港防癌會。
