'''鄭正鈐'''（{{bd|1969年|5月31日|catIdx=Jheng鄭}}），[[台灣|台灣]][[政治人物|政治人物]]，[[中國國民黨|中國國民黨]]籍，現任[[新竹市議員|新竹市議員]]、[[中國國民黨中央委員會|國民黨中央委員]]、[[中國國民黨中央委員會組織發展委員會|國民黨中央組織發展委員會]]副主任委員，以及國民黨次級團體[[五六七大聯盟|五六七大聯盟]]成員。

推動[[新竹市立棒球場|新竹市立棒球場]]重建案，但拒絕原地重建，認為應該另外尋覓合適地點興建符合國際標準全新球場，以免危害交通、阻礙城市發展、延誤新竹市棒球運動的未來。堅決反對核災食品進入台灣。

==選舉==
===[[2004年中華民國立法委員選舉|2004年中華民國立法委員選舉]]===

==「新竹共和國」提案 ==
2003年新竹市議會上，國民黨籍議員鄭正鈐建議新竹市長[[林政則|林政則]]建立「新竹共和國」，舉行獨立[[公民投票|公投]]。而市長林政則也相當認真的回答質詢，當場指示民政局好好研商獨立的可能。
新竹市每年上繳156億稅收給國庫，然而中央補助下來的經費卻不到五分之一，也因此鄭正鈐議員認為如果新竹市獨立成共和國，光靠竹科園區，國民生產毛額將會提升到8萬美元，還可以跟中華民國建交，成為中華民國最有錢的邦交國<ref>{{Cite news|url = http://news.tvbs.com.tw/old-news.html?nid=394353|title = 竹市議員提「新竹獨立」市長：很心動|last = |first = |date = |work = |accessdate = }}</ref>。
