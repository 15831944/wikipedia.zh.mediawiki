'''鄭泳舜'''{{MH}}（{{lang-en|Vincent Cheng}}，1979年7月18日<ref>{{cite web|url=https://m.facebook.com/chowhoding/photos/a.489140861241665.1073741830.429790780510007/839721669516914/?type=3|title=周浩鼎立法會議員Facebook專頁|date=2017年7月18日|publisher=[[周浩鼎|周浩鼎]]}}</ref>-），祖籍[[潮陽|潮陽]]，[[香港|香港]][[政治人物|政治人物]]，現任[[香港立法會議員|香港立法會議員]]<ref>{{cite web|url=https://www.hk01.com/港聞/167248/-立法會補選-富二代-鄭泳舜-憑甚麼打敗姚松炎創歷史-|title=【立法會補選】「富二代」鄭泳舜　憑甚麼打敗姚松炎創歷史？|date=2018年3月12日|publisher=[[香港01|香港01]]}}</ref>，[[民建聯|民建聯]]黨員，[[深水埗|深水埗]][[南昌北_(選區)|南昌北]]選區（F03）區議員<ref>{{zh-hant}}{{cite news|url=http://www.districtcouncils.gov.hk/ssp/tc/2012_2015/member_details.html#chengws|title=區議會成員資料|publisher=[[深水埗區議會|深水埗區議會]]|accessdate=2017-09-09|deadurl=yes|archiveurl=https://web.archive.org/web/20150430073213/http://www.districtcouncils.gov.hk/ssp/tc/2012_2015/member_details.html#chengws|archivedate=2015年4月30日|df=}}</ref>，曾任[[青年民建聯|青年民建聯]]副主席，[[鄭錦鐘|鄭錦鐘]]兒子，其妻子是前香港小姐及前[[TVB|無綫]]女藝人[[胡杏兒|胡杏兒]]的大姐胡可兒。

== 簡歷 ==
鄭泳舜少時居於[[深水埗|深水埗]][[南山邨|南山邨]]，中學時期搬入[[屯門|屯門]]，在[[佛教沈香林紀念中學|佛教沈香林紀念中學]]讀至中四往[[紐西蘭|紐西蘭]]升學<ref>{{web reference|URL=https://hk.news.appledaily.com/local/realtime/article/20170717/56963539|title=【八方相當年】鄭泳舜好「打」得 同鬼仔化干戈為玉帛|work=香港蘋果日報|year=2017年|date=7月17日}}</ref>，後來在當地的[[奧克蘭大學|奧克蘭大學]]修讀[[統計學|統計學]]。鄭泳舜在畢業後於2003年回到香港，適逢[[SARS事件|SARS事件]]，當時其首份工作乃廣告設計公司上班，月入8,500港元，及後轉職至商場擔任租務及管理。<ref>{{zh-hant}}{{cite web|url=http://hk.apple.nextmedia.com/news/art/20170718/20093367|title=八方相當年：被鬼仔吐口水　鄭泳舜靠打波化解|date=2017年7月18日|publisher=《[[蘋果日報_(香港)|蘋果日報]]》|accessdate=2017-09-09}}</ref>

===區議會選舉===
2005年，鄭泳舜涉足政界，開始進行地區工作，翌年加入[[建制派|建制派]]政黨[[民建聯|民建聯]]。因父親鄭錦鐘乃知名商人，鄭泳舜曾被指為[[富二代|富二代]]，與傳統建制派的基層背景不同，因而被視為黨內的重點裁培對象。

[[2007年香港區議會選舉|2007年香港區議會選舉]]，鄭泳舜被代表[[民建聯|民建聯]]出戰[[南昌北_(選區)|南昌北]]選區，以挑戰競選連任的[[民主陣線|民主陣線]]暨[[神州青年服務社|神州青年服務社]]成員[[梁漢華|梁漢華]]，當時[[民協|民協]]同時派出原[[深水埗區議會|深水埗區議會]]副主席兼[[南昌東|南昌東]]選區區議員[[梁欐|梁欐]]出戰，使本區成為[[泛民主派|泛民主派]]撞區的例子之一。這次兩名泛民互撞，源於[[西九龍填海區|西九龍填海區]]（俗稱西九四小龍）及[[歌和老街|歌和老街]]近[[筆架山|筆架山]]地區人口增加，導致選區改劃現象，兩名在任區議員皆不想放棄原有的部份選區所置<ref>{{cite web |url=http://www.eac.gov.hk/pdf/distco/maps/dc2003f.pdf |title=2003年區議會選舉 - 選區分界概要 |publisher=[[香港選舉管理委員會|香港選舉管理委員會]] |date=2013-01-05 |accessdate=2013-01-05 |language=zh-hant }}</ref>。最終，在漁人得利下，鄭泳舜以當區唯一建制派候選人的身份取得912票，相比梁欐的898票和梁漢華624票，在漁人得利的情況下取代兩位區議員的位置，但泛民建制得票比例仍有六十和四十之間。他以27歲之齡成為區议员。

[[2011年香港區議會選舉|2011年香港區議會選舉]]，鄭泳舜於同區競選逐連任，今次對手除了[[民協|民協]]的[[梁欐|梁欐]]外，[[人民力量|人民力量]]亦以「票債票償」名義派出[[盧蔚然|盧蔚然]]參選<ref>{{cite web |url=http://202.55.1.84/news/11/10/16/GW-1414685.htm |deadurl=yes |accessdate=2017-09-09 |language=zh-hant }}{{dead link|date=2017-11 |bot=InternetArchiveBot }}</ref>，以狙擊民協支持[[2012年香港政治制度改革|2012年香港政治制度改革]]的決定。結果，鄭泳舜票數增長至1485票，超過梁欐的1133票及盧蔚然的296票。民建聯在資源優勢及現任區議員網絡下，令鄭泳舜以過半數得票連任<ref>{{cite web |url=http://www.eac.gov.hk/pdf/distco/ch/2011dc_report/2011dcereport_appendix5.pdf|title=2001區議會選舉報告書　附錄四 二零一一年區議會選舉結果|accessdate=2017-09-09|language =zh-hant }}</ref>。

[[2015年香港區議會選舉|2015年香港區議會選舉]]，鄭泳舜再次角逐連任，這次的選區範圍回復為[[大埔道|大埔道]]、[[南昌街|南昌街]]、[[長沙灣道|長沙灣道]]、[[欽州街|欽州街]]所包括的樓宇為主，即1985年—2007年所採用的選區範圍，同時南昌東選區重新設立。這次，[[民協|民協]]未有再派人挑戰，而[[人民力量|人民力量]]則派出參與[[旺角鳩嗚團|旺角鳩嗚團]]的成員[[陳永輝|陳永輝]]參選，成為民主派的候選人。結果鄭泳舜在繼續票數增長下以1461票多於陳永輝的976票成功連任<ref>{{cite news |url=http://the-sun.on.cc/cnt/news/20101207/00410_041.html |title=SUN世代：富家子情陷深水埗|publisher=《[[太陽報_(香港)|太陽報]]》|accessdate=2017-09-09 |language=zh-hant }}</ref>。

===立法會選舉===
鄭泳舜過往曾參與兩屆立法會選舉，皆代表[[民建聯|民建聯]]出戰。因其本身是[[深水埗區議會|深水埗區議會]][[區議員|區議員]]的關係，加上形象不差，故成為地區樁腳，被納入出選名單之中。
[[2008年香港立法會選舉|2008年香港立法會選舉]]為鄭泳舜首次代表出選，成為[[李慧琼|李慧琼]]名單的候選人之一；[[2012年香港立法會選舉|2012年香港立法會選舉]]第二次參選，則成為[[蔣麗芸|蔣麗芸]]名單的候選人之一<ref>{{cite news|url=http://www.elections.gov.hk/legco2012/pdf/introd/LC2_05.pdf|title=2012年立法會選舉 - 候選人簡介|accessdate=2017-09-09 |language =zh-hant }}</ref>。

2017年，鄭泳舜於民建聯黨內遴選中擊敗[[油尖旺區議會|油尖旺區議會]]主席兼[[葉國忠|葉國忠]]之子[[葉傲冬|葉傲冬]]，代表建制派披甲出戰因[[游蕙禎|游蕙禎]]被法庭裁定宣誓無效而取消議員資格衍生的[[2018年3月香港立法會補選|2018年3月香港立法會補選]]。自2017年6月以來，其宣傳品及曝光率次數大幅度出現在九龍西，並以「堅守信念，舜有希望」作為競選口號<ref>{{cite news |url=http://hk.apple.nextmedia.com/realtime/news/20170614/56824789 |title=【九西補選】鄭泳舜K.O.葉傲冬 勝出民建聯內部遴選|date=2017年6月14日|publisher=《[[蘋果日報_(香港)|蘋果日報]]》|accessdate=2017-09-09 |language=zh-hant }}</ref>。3月12日凌晨，經重新點票後，鄭泳舜以107,479票，以2,419票的差距擊敗[[姚松炎|姚松炎]]，勝出九龍西議席，成功當選，成為自回歸以來立法會地方選區補選之中，第一位勝出的建制派代表。<ref>{{cite web|url=https://www.hk01.com/港聞/167269/-立法會補選-鄭泳舜意外擊敗姚松炎-今次是-逆境戰-|title=【立法會補選】鄭泳舜意外擊敗姚松炎：今次是「逆境戰」|date=2018年3月12日|publisher=[[香港01|香港01]]}}</ref>

== 家庭 ==
鄭泳舜的父親鄭錦鐘是前[[保良局|保良局]]副主席，先後經營服裝生意及電腦配件分銷致富，包括創立上市公司[[偉仕控股有限公司|偉仕控股有限公司]]（{{hkex2|0856}}）。鄭錦鐘身兼多項公職，並曾為[[中央政策組|中央政策組]]非全職顧問，曾先後獲頒授[[榮譽獎章|榮譽獎章]]（B.H.）及[[銅紫荊星章|銅紫荊星章]]（BBS）。
