'''鄭進文'''（{{bd|1958年|12月27日|2017年|3月26日|catIdx=Z}}），筆名'''鄭問'''，生於[[臺灣|臺灣]][[桃園市|桃園縣]][[大溪區|大溪鎮]]（今桃園市大溪區），[[漫畫家|漫畫家]]。[[新北市私立復興高級商工職業學校|復興商工]]雕塑科畢業。1990年在[[日本|日本]]發表作品，造成話題。2017年3月26日因為[[心肌梗塞|心肌梗塞]]過世，享年58歲<ref>[http://www.cna.com.tw/news/firstnews/201703285015-1.aspx 中央社　鄭問58歲病逝 阿鼻劍作者傲視漫畫界20年]</ref>。

== 簡歷 ==
早年曾在數家設計公司任職，後來自行成立[[室內設計|室內設計]]公司。1984年在《[[時報周刊|時報周刊]]》上發表第一篇漫畫作品《戰士黑豹》，開始他的漫畫創作生涯。獲得好評後又發表了《鬥神》及以《[[史記|史記]]》中的刺客列傳為題材的水墨手繪漫畫《刺客列傳》。

自1989年2月22日與作家[[郝明義|馬利]]合作，於《[[星期漫畫|星期漫畫]]》上連載《阿鼻劍》，1989年遇日本[[講談社|講談社]]旗下雜誌之漫畫周刊(Morning)及「午宴喜劇」月刊(Afterhoon)總編栗原良幸來臺挖掘漫畫家，鄭問成為被邀稿的名單之一。栗原表示：「鄭問畫風和日本畫家很不一樣」，後歷經籌備，1990年3月起，在日本發表《東周英雄傳》。

1991年每年一度[[日本漫畫家協會獎|日本漫畫家協會獎]]頒予「優秀賞」，他成為此獎項二十年來第一位非日籍得獎者（為當年多位得獎者之一）。另亦在日本發行以鄭問為名的《鄭問之三國志》，並由日本著名作家[[神涼介|神涼介]]撰寫劇本合作，因此也是台灣首位登上日本電玩的台灣漫畫家。

2000年與[[臺灣|臺灣]]的[[布袋戲|布袋戲]]世家[[黃氏|黃氏]]建立的[[霹靂國際多媒體|霹靂國際多媒體]]及[[香港|香港]][[黃玉郎|黃玉郎]]先生所創的[[玉皇朝集團|玉皇朝集團]]合作，推出以[[霹靂布袋戲|霹靂布袋戲]]系列故事改編的《漫畫大霹靂》，不過後來因工作量太大不堪負荷，畫完第一部完結篇後，退出其製作。休養一段時間後，2002年與香港漫畫家[[馬榮成|馬榮成]]合作推出第二本全彩手繪漫畫《風雲外傳：天下無雙》。

2013年12月30日，25歲愛子[[鄭宇書|鄭宇書]]於友人[[陳志豪|陳志豪]]位於台北市萬華區的家中，用槍頂住後腦自殺<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20170328/1085890/ 蘋果日報　「亞洲至寶」過世！生前慟愛子超離奇身亡]</ref>。

2017年3月26日凌晨03點10分，在家中突因[[心肌梗塞|心肌梗塞]]不適送醫，急送往台北[[慈濟醫院|慈濟醫院]]仍宣告不治，享年58歲<ref>[http://news.ltn.com.tw/news/life/breakingnews/2018453 自由時報　台灣知名漫畫家鄭問過世 享年58歲]</ref>。

== 畫風 ==
鄭問作畫向以嚴謹見稱，風格詭異明麗，獨成一派，常將日常生活的物件用於創作，例如[[牙刷|牙刷]]、[[菜瓜布|菜瓜布]]，甚至是[[塑膠袋|塑膠袋]]。他擅長歷史、武俠、神怪題材，並將中國[[水墨|水墨]]技法與西方繪畫技巧融合於作品中，筆觸間更隱現出狂放灑脫的豪情俠意。連一些配角的小人物都有誇張的表情，呈現出非常豐富的變化。

他曾經獨自畫出[[家用遊戲機|家用遊戲]]《[[鄭問之三國志|鄭問之三國志]]》與[[線上遊戲|線上遊戲]]《鐵血三國志》中的所有三國歷史人物，不禁讓人歎為觀止。日本漫畫家[[池上遼一|池上遼一]]是鄭問相當欣賞的一位偶像，早期的畫風也深受其影響。

== 逸事 ==
*鄭問雖然鮮少與外人來往，但曾為了幫一位罹患[[癌症|癌症]]的漫畫家籌款治療，借攤位義賣眾漫畫家的作品，每天親自顧攤，並無限量提供作品義賣。
*阿鼻劍因1991年星期漫畫月刊於第八十期停刊，故連載大受影響，加以鄭問又忙於其他作品，遂告中斷。
*日本NHK為拍攝孔子傳，邀請鄭問繪畫原稿，鄭問因而數度進出[[國立故宮博物院|國立故宮博物院]]及[[大英博物館|大英博物館]]蒐集相關資料。
*鄭問作畫時間甚長，工作壓力相當大，他紓解壓力的方法，就是作畫時不停地嚼[[檳榔|檳榔]]。
