'''蒋树声'''（{{bd|1940年|4月|catIdx=J}}），[[江苏|江苏]][[无锡|无锡]]人，中国物理学家，原[[南京大学|南京大学]]校长。2003年4月加入[[中国民主同盟|中国民主同盟]]，2005年至2012年间曾任[[民盟|民盟]]中央主席，第十一届[[全国人大常委会副委员长|全国人大常委会副委员长]]。

== 主要经历 ==
蒋树声1963年7月毕业于[[南京大学物理学系|南京大学物理学系]]，获学士学位。毕业后，留校工作四十多年；曾历任[[南京大学固体微结构实验室|南京大学固体微结构实验室]]副主任，[[南京大学|南京大学]]物理系主任，副校长等职；1997年4月，出任[[南京大学|南京大学]]校长。在[[南京大学|南大]]工作期间，蒋树声先后在[[英国|英国]]、[[澳大利亚|澳大利亚]]、[[意大利|意大利]]学习工作五年多；其间，分别于2002年和2003年获[[美国|美国]][[约翰·霍普金斯大学|约翰·霍普金斯大学]]和[[英国|英国]][[布里斯托尔大学|布里斯托尔大学]]的名誉博士学位。

2005年12月，在[[民盟|民盟]]九届四中全会上，刚入盟两年的蒋树声接替[[丁石孙|丁石孙]]出任民盟第九届中央主席。2007年12月，在民盟十届一中全会上，蒋树声连任了民盟中央主席的职务。2008年3月在[[第十一屆全國人民代表大會第一次會議|第十一屆全國人民代表大會第一次會議]]上當選為[[第十一屆全國人大常委會|第十一屆全國人大常委會]]副委員長。

== 学术活动 ==
自1985年以来，蒋树声先后承担了[[国家自然科学基金|国家自然科学基金]]、[[中华人民共和国教育部|中华人民共和国教育部]]的多项重点科研项目。

== 荣誉 ==
曾获[[国家教委|国家教委]]科技进步二等奖、江苏省科技进步一等奖、[[国家自然科学奖|国家自然科学奖]]三等奖等奖项。

他还曾获得“[[国家有突出贡献的中青年专家|国家有突出贡献的中青年专家]]”等[[荣誉称号|荣誉称号]]。
