'''蒋尊玉'''{{BD|1957年|6月|||catIdx=J}}，[[江苏省|江苏省]][[丰县|丰县]]人，毕业于[[中国人民解放军西安政治学校|中国人民解放军西安政治学校]][[哲学|哲学]]专业，在职研究生学历，[[中国共产党|中国共产党]]党员（至2015年4月7日）。

曾担任[[中国共产党深圳市委员会|中共深圳市委]]常委、市[[政法委|政法委]]书记。2014年10月13日下午，因涉嫌严重违纪问题被中共[[广东省|广东省]][[中国共产党纪律检查机关|纪律检查委员会]]带走调查。<ref>{{cite web|url=http://city.ifeng.com/a/20141024/414356_0.shtml|title=深圳政法委书记蒋尊玉正在接受组织调查|accessdate=2014|publisher=凤凰网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20141024174837/http://city.ifeng.com/a/20141024/414356_0.shtml|archivedate=2014-10-24}}</ref><ref>{{cite web |url=http://news.163.com/14/1024/21/A9BONOI3000146BE.html |title= 深圳市委常委政法委书记蒋尊玉接受组织调查|accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://news.china.com/domestic/945/20141024/18894644.html |title= 深圳政法委书记蒋尊玉落马 或涉大运会工程腐败|accessdate=2014 |publisher=中华网 |work= |language=zh-hans}}</ref> 2015年3月，经[[广东省人民检察院列表|广东省人民检察院]]指定管辖，广州市人民检察院决定，依法对蒋尊玉涉嫌受贿犯罪立案侦查并采取强制措施<ref>{{cite web|url=http://money.163.com/15/0320/18/AL61DD8U00253B0H.html#from=money_index|publisher=网易|accessdate=2015-03-20|title=深圳政法委原书记被立案侦查 曾为家人买42套房}}</ref>。据媒体报道，其家族插手[[深圳大运会|深圳大运会]]所有工程<ref>{{cite web |url=http://news.163.com/14/1202/02/ACE60GIF0001124J.html |title=深圳政法委书记落马 家族插手大运会所有工程 |accessdate=2014 |publisher= 网易|work= |language=zh-hans}}</ref><ref>{{cite web |url=http://news.sina.com.cn/c/2014-12-02/002031230383.shtml |title= 蒋尊玉落马背后：家族插手深圳大运会所有工程|accessdate=2014 |publisher=新浪网 |work= |language=zh-hans}}</ref>，为家人先后买入了42套住房，存款和投资股票银行资金有两亿多元<ref>{{cite web|url=http://money.163.com/15/0320/18/AL61DD8U00253B0H.html#from=money_index|publisher=网易|accessdate=2015-03-20|title=深圳政法委原书记被立案侦查 曾为家人买42套房}}</ref>。2015年4月7日被开除党籍，2017年8月28日被判处无期徒刑。

==生平==
* 1957年，蒋尊玉出生在[[江苏省|江苏省]][[丰县|丰县]]。
* 1976年2月，他参加工作，1978年7月，他加入[[中国共产党|中国共产党]]。
* 1976年到1983年，他一直在基建工程兵部队工作。
* 1983年到1989年，他在深圳市机电设备安装公司工作。
* 1989年到1992年，蒋尊玉在深圳机场建设领导小组办公室工作。
* 1992年到2001年，他在深圳市规划国土局房地产市场处工作。
* 2001年到2009年，蒋尊玉进入深圳市人民政府。
* 2009年到2013年，蒋尊玉在深圳市[[龙岗区|龙岗区]]工作，曾担任区委书记、区人大常委会主任、党组书记、区人大常委会主任。
* 2010年5月，担任中共深圳市委常委，兼任龙岗区委书记。
* 2013年1月，担任市政法委书记。
* 2014年10月13日下午，蒋尊玉被广东省纪委带走调查。
* 2014年10月24日，广东省纪委监察厅网站宣布蒋尊玉因涉嫌严重违纪问题接受调查。
* 2015年4月7日，蒋尊玉因违反廉洁自律规定，收受礼金、参与[[赌博|赌博]]、与他人[[通奸|通奸]]、未如实向组织报告个人有关事项、在工程项目等方面为他人谋取利益、收受巨额贿赂，已被广东省纪委开除党籍并开除公职<ref>{{cite web|url=http://news.sina.com.cn/o/2015-04-08/075931692135.shtml|publisher=新浪网|accessdate=2015年4月8日|title=蒋尊玉被双开 曾与他人通奸}}</ref>。
* 2017年8月28日，广州中院对蒋尊玉受贿案公开宣判，以受贿罪判处蒋尊玉无期徒刑，剥夺政治权利终身，并处没收个人全部财产；受贿所得财物予以没收、上缴国库<ref>{{cite web|url=http://news.ifeng.com/a/20170829/51795312_0.shtml|publisher=咩事 |accessdate=2017-08-29|title=曾被中纪委机关报痛斥的广东高官一审宣判：无期徒刑！}}</ref>。但蒋尊玉不服判决结果，当即口头提出上诉<ref>{{cite web|url=http://china.caixin.com/2017-08-28/101136601.html|publisher=财新网  |accessdate=2017-08-28|title=深圳政法委原书记一审获无期 }}</ref>。
