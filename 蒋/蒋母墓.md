'''蒋母墓'''是[[蒋中正|蒋中正]]母亲[[王采玉|王采玉]]的墓葬，位于[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[奉化区|奉化区]][[溪口镇_(宁波市)|溪口镇]]的白岩山鱼鳞岙。墓葬始建于1921年11月，两年后新建名为“慈庵”的墓庐和“蒋母墓道”石坊，形成总长668米的墓道。1949年[[蒋中正|蒋中正]]离开中国大陆后，蒋母墓受到中华人民共和国政府的保护和修复，2006年成为[[全国重点文物保护单位|全国重点文物保护单位]]<ref name="nbwb-detail">{{cite web|url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=830&type=2|title=蒋氏故居|accessdate=2017-09-02|date=2006-06-22|publisher=宁波市文物保护管理所|archiveurl=https://web.archive.org/web/20180222043927/http://www.nbwb.net/pd_wwbh/info.aspx?Id=830&type=2|archivedate=2018-02-22|deadurl=no}}</ref>。

== 历史 ==
1913年，蒋中正之父[[蔣肇聰|蔣肇聰]]去世时，与元配徐氏和续娶孙氏合葬于溪口桃坑山。此时，蒋中正之母王采玉即告知蒋中正，将来自己不与蒋父合葬。1921年春，王采玉病重，6月14日去世，去世前再度要求另择墓地<ref name="唐瑞福">{{Cite book|title=浙江文史资料选辑（第二十三辑）|author=唐瑞福|first=|author2=汪日章|publisher=浙江人民出版社|year=1982|isbn=|editor-last=中国人民政治协商会议浙江省委员会文史资料研究委员会|location=杭州|pages=|chapter=蒋介石的故乡|unified=11103·78}}</ref>{{Rp|46}}。当年11月，王采玉下葬<ref name="nbwb-detail" />，[[陳果夫|陈果夫]]、[[戴季陶|戴季陶]]等要员到场，[[孙中山|孙中山]]、[[胡汉民|胡汉民]]、[[张静江|张静江]]等政治人物为蒋母墓题字，编有《哀思集》。1923年蒋母六十[[冥诞|冥诞]]，蒋中正在墓道上修建墓庐数间，题名为“慈庵”，供奉曾祖以下历代牌位，另建墓道牌坊一座<ref name="唐瑞福" />{{Rp|47}}。每年回乡扫墓时，蒋中正多会在慈庵居住<ref name="王舜祁">{{cite book|title=宁波文物古迹保护纪实|author=王舜祁|publisher=宁波出版社|accessdate=2017-09-02|year=2000|isbn=9-787-80602-418-8|chapter=蒋氏故居保护和开放|chapterurl=http://www.nbzx.gov.cn/art/2015/9/18/art_4055_690833.html|archiveurl=https://web.archive.org/web/20180222043756/http://www.nbzx.gov.cn/art/2015/9/18/art_4055_690833.html|archivedate=2018-02-22|deadurl=no}}</ref>，[[西安事变|西安事变]]后及离开中国大陆前往台湾前的1949年1月至4月，蒋中正也曾在此下榻<ref name="王凡">{{Cite journal|title=奉化溪口遗迹中的蒋家故事（下）|author=王凡|url=|journal=人物|issue=12|year=2010}}</ref>。[[中国人民解放军|中国人民解放军]]占领溪口后，对蒋母墓实施保护，1956年[[中共中央|中共中央]]透过[[章士钊|章士钊]]向蒋中正传话，称“奉化之墓庐依然，溪口之花草无恙”。[[文化大革命|文化大革命]]期间的1968年，慈庵被闯入的红卫兵炸毁<ref>{{Cite journal|title=两岸“密使”曹聚仁给毛泽东与蒋介石牵线搭桥|author=李振广|url=http://paper.people.com.cn/hqrw/html/2008-07/01/content_60565.htm|journal=环球人物|issue=13|year=2008|access-date=2018-07-22|archive-url=https://web.archive.org/web/20151112064234/http://paper.people.com.cn/hqrw/html/2008-07/01/content_60565.htm|archive-date=2015-11-12|dead-url=no}}</ref>，1979年修复，1984年对外开放参观<ref name="nbwb-detail" />。此后，[[蔣孝勇|蔣孝勇]]、[[蒋孝严|蒋孝严]]等蒋氏后裔曾到蒋母墓祭祖，至2008年底达30余次<ref name="奉化市志2008">{{cite book|title=奉化市志（1989-2008）|publisher=浙江人民出版社|year=2016|isbn=978-7-21307-5476}}</ref>{{Rp|1013-1014}}。

== 布局 ==
[[File:Layout_of_Tomb_of_Chiang_Kai-shek's_Mother.svg|缩略图]]
蒋母墓位于溪口镇白岩山鱼鳞岙，为蒋中正聘请奉化[[风水师|风水师]]勘验确定。白岩山为[[雪窦山|雪窦山]]余脉，传此山形似[[弥勒佛|弥勒佛]]，而墓穴则位于弥勒佛的肚脐上。初建时，为不压坏龙穴，仅修建了一座普通坟茔，墓庄等建筑为后来增建<ref name="唐瑞福" />{{Rp|46-47}}。墓道总长668米，由石牌坊、下轿亭、孝子亭、慈庵、方圆池、仰慈亭和墓冢组成<ref name="裘国松">{{Cite book|title=溪口谈助——蒋氏故里文化品读|author=裘国松|publisher=宁波出版社|year=2015|isbn=978-7-55262-2089|location=宁波}}</ref>{{Rp|34}}。路面原为鹅卵石铺就，后改为石板路<ref name="nbwb-detail" />。石牌坊为三开间，横额“蒋母墓道”四字为孙中山题写<ref name="唐瑞福" />{{Rp|47}}。距石牌坊约250米为下轿亭，俗称白凉亭，为跨路亭<ref name="nbwb-detail" />，传蒋中正至此均下轿步行。下轿亭以上有八角亭，民间称“孝子亭”，为蒋中正念及去世的母亲裹小脚而建的休息场所<ref name="裘国松" />{{Rp|34}}。

墓庐位于墓冢下，建于1923年，头门刻“墓庐”，二门刻“慈庵”二字，由[[谭延闿|谭延闿]]题写<ref name="王舜祁" />。墓庐共有三座，均为平房。主屋五开间，正中立碑，正反面分别写有孙中山撰《祭蒋母文》和蒋中正撰《先妣王太夫人事略》，左右壁有蒋中正《哭母文》和[[中國國民黨中央執行委員會|中國國民黨中央執行委員會]]《慰劳蒋总司令文》，此外有从蒋中正曾祖蒋祈增到蒋中正弟弟蒋瑞青的牌位<ref name="裘国松" />{{Rp|34}}。东两间为蒋中正和宋美龄的居所，西两间为客房。另有作为厨房和管理用房的附属平屋六间。文化大革命期间，墓庐被红卫兵毁坏，后复建<ref name="王舜祁" />。原藏有[[汪精卫|汪精卫]]作志，[[胡汉民|胡汉民]]作铭，[[沈尹默|沈尹默]]书《蒋太夫人墓志铭》。由于汪精卫投靠日本，[[抗日战争|抗日战争]]结束后，墓志铭被蒋中正埋入土中，后在墓庐复建中被发现，但并未存留，沈尹默书法原件位于藏于台北[[中正纪念堂|中正纪念堂]]<ref name="唐瑞福" />[[中正纪念堂|{{Rp]]<ref name="裘国松" />{{Rp|35-36}}。

蒋母墓冢位于墓道尽头，坐南朝北。墓圈为石砌，黄土封顶，前有墓碑，碑文“蒋母之墓”为孙中山书写<ref name="裘国松" />{{Rp|34}}，上有横额写有“壼范足式”，意为“女中模范，完美榜样”，同为孙中山书写。墓碑旁楹联为“祸及贤慈，当日梗顽悔已晚；愧为逆子，终身沉痛恨靡涯”，署名“不孝子周泰”，为蒋中正拟，[[张静江|张静江]]书写。墓前拜台有“凤凰翠竹”浮雕，与蒋母出生地溪口葛竹村暗合<ref name="杨韵华">{{Cite journal|title=溪口蒋氏遗迹考察纪行|author=杨韵华|url=|journal=武汉文博|issue=3|doi=|others=|year=2008|volume=|page=|pmid=|author2=翁沁锦}}</ref>。蒋母墓西下有仰慈亭，为后世兴建。此处为蒋中正为自己选定的墓地之一，意在“长伴母灵，侍候永久”<ref name="裘国松" />{{Rp|34}}。

== 保护和展示 ==
{{external media|width=250px|image1=[http://www.nbwb.net/UploadFile/images/2018012213572929.jpg 蒋母墓保护区划图]，原始内容[https://web.archive.org/web/20180711164547/http://www.nbwb.net/UploadFile/images/2018012213572929.jpg 存档]于2018年7月11日}}
1949年，蒋中正离开中国大陆后，中国人民解放军占领溪口，浙江省军管会主任[[谭震林|谭震林]]曾派专人看管蒋母墓。1979年，中共中央拨款，修复了包括蒋母墓在内的蒋氏家族四座陵墓。此次修复还重建了文化大革命期间被拆毁的墓庐<ref name="王舜祁" /><ref name="奉化市志2008" />{{Rp|1007}}。1984年12月31日，蒋母墓对外开放参观。2005年，蒋母墓与[[武山庙|武山庙]]一起被列入[[浙江省文物保护单位|浙江省文物保护单位]]<ref>{{Cite web|url=http://culture.zjol.com.cn/05culture/system/2006/08/03/007785967.shtml|title=武山庙（含蒋母墓）|accessdate=2018-08-14|author=|date=2006-05-18|publisher=浙江文物网|archive-url=https://web.archive.org/web/20090318104448/http://culture.zjol.com.cn/05culture/system/2006/08/03/007785967.shtml|archive-date=2009-03-18|dead-url=no}}</ref>。2006年[[第六批全国重点文物保护单位|第六批全国重点文物保护单位]]公布时，蒋母墓和[[文昌阁_(溪口)|文昌阁]]、[[蒋氏宗祠_(溪口)|蒋氏宗祠]]等建筑以“[[溪口镇建筑群|溪口镇建筑群]]”的名义并入[[第四批全国重点文物保护单位|第四批全国重点文物保护单位]][[蒋氏故居|蒋氏故居]]<ref name="nbwb-detail" />。
