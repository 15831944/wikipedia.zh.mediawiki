'''蒋定之'''（{{bd|1954年|9月|catIdx=J蒋}}），[[江苏|江苏]][[溧阳|溧阳]]人。男，汉族。1975年1月参加工作，1978年7月加入[[中国共产党|中国共产党]]。[[南京大学|南京大学]]哲学系专科毕业，[[中共中央党校|中共中央党校]]函授研究生学历，[[南京理工大学|南京理工大学]]环境工程学硕士。[[中国共产党第十八届中央委员|中国共产党第十八届中央委员]]。曾任[[中共|中共]][[海南|海南]][[省委書記|省委副书记]]、[[省长|省长]]，[[江苏省人大常委会|江苏省人大常委会]]副主任、党组书记，江苏省政协主席、党组书记等职务。

== 从政经历 ==
蒋定之是[[江苏|江苏]][[溧阳|溧阳]]人，他从中共基层做起，1975年1月在家乡溧阳县任埭头公社通讯报道员。

1977年3月—1979年1月在镇江地区第二师范学校学习，之后在[[江苏|江苏]][[镇江|镇江]]地区教育局、地委组织部工作。

1983年9月，进入[[南京大学|南京大学]]哲学系干部专修科学习。

1985年7月，回到[[镇江|镇江]]，历任市委组织部副科长、科长、副部长，市委副秘书长。

1994年10月，任下辖县级中共[[扬中市|扬中市]]市委书记。

1996年11月，任中共江苏省委组织部副部长。

2000年8月，任中共[[无锡市|无锡市]]委书记。

2001年11月，任[[中共江苏省委|中共江苏省委]]常委、无锡市委书记，晋升副部级。

2003年2月，任中共江苏省委常委、常务副省长、省政府党组副书记。2003年11月起兼任江苏省行政学院院长<ref>{{cite web|accessdate=2017-02-09|url=http://www.thepaper.cn/newsDetail_forward_1614880|title=蒋定之当选为江苏省政协主席，阎立增选为省政协副主席|publisher=澎湃新闻}}</ref>。

2005年11月，任[[中国银行业监督管理委员会|中国银行业监督管理委员会]]副主席、党委副书记、国务院三峡工程建设委员会委员。

2010年12月，任[[中共海南省委|中共海南省委]]常委、常务副省长、省政府党组副书记。

2011年7月，任中共海南省委副书记、代理省长<ref>[http://news.163.com/11/0830/09/7CMOJTIH0001124J.html 蒋定之夏宝龙分别任海南浙江代理省长]</ref>（2012年2月，正式当选省长<ref>{{cite web|title=蒋定之当选海南省省长|url=http://politics.people.com.cn/GB/41223/17096474.html|publisher=人民网|accessdate=2012-02-13}}</ref>）<ref>{{cite web|accessdate=2014-06-12|url=http://district.ce.cn/newarea/sddy/201204/28/t20120428_23283781_1.shtml|title=罗保铭当选海南省委书记 蒋定之、李宪生为副书记|publisher=中国经济网}}</ref>。

2014年12月，被免去海南省委副书记，辞去省长职务<ref>{{cite web|title=刘赐贵任海南省委副书记 蒋定之不再担任|url=http://news.ifeng.com/a/20141230/42832342_0.shtml|publisher=凤凰网|accessdate=2014-12-30}}</ref>。

2015年1月，任[[江苏省人大常委会|江苏省人大常委会]]党组书记<ref>{{cite web|accessdate=2015-01-06|url=http://www.hq.xinhuanet.com/news/2015-01/06/c_1113886748.htm|title=江苏省委决定蒋定之任江苏人大常委会党组书记|publisher=新华网|deadurl=yes|archiveurl=https://web.archive.org/web/20150124070855/http://www.hq.xinhuanet.com/news/2015-01/06/c_1113886748.htm|archivedate=2015-01-24}}</ref>。

2015年2月1日，补选为[[江苏省人大常委会|江苏省人大常委会]]副主任<ref>{{cite web|title=蒋定之史和平补选为江苏省人大常委会副主任|url=http://politics.people.com.cn/n/2015/0201/c1001-26486588.html|accessdate=2015-02-01|publisher=人民网}}</ref>。

2017年2月9日，当选为[[江苏省|江苏省]]政协主席<ref>{{cite web|url=http://district.ce.cn/newarea/sddy/201702/09/t20170209_20074152.shtml | language= | title=蒋定之当选江苏省政协主席 张连珍不再担任(简历) |publisher=中国经济网 |date=2017年2月9日 |accessdate= }}</ref><ref>{{cite web|accessdate=2017-02-09|url=http://www.thepaper.cn/newsDetail_forward_1615018|title=解读：从政府到人大再到政协，63岁蒋定之再度履新令人关注|publisher=澎湃新闻}}</ref>。

2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。2018年1月29日，卸任江苏省政协主席、党组书记职务，由江苏省委副书记[[黄莉新|黄莉新]]接任。

== 因病退居二线 ==
[[多維新聞網|多維新聞網]]2014年1月報導，[[中國共產黨第十八屆中央委員會第三次全體會議|中國共產黨第十八屆中央委員會第三次全體會議]]結束後，中國各省市區今年1月開始密集召开省常委會議，蔣定之卻意外連續缺席[[海南省委|海南省委]]6屆5次全會等重頭會議。蔣定之出席海南省委常委會後便再無公開露面，已經超過40天，還缺席省委全會與「政府工作意見徵求座談會」等重要会议。有北京知情人士對[[明鏡出版社|明鏡新聞網]]說，海南省省長蔣定之現在[[南京|南京]]治病，並非如外傳涉及周永康案被查<ref>{{Cite web |url=http://www.mingjingnews.com/MIB/news/news.aspx?ID=N000016355 |title=明鏡新聞網-海南：省長治病，副省長待捕 |access-date=2014-01-18 |archive-url=https://web.archive.org/web/20140201132645/http://www.mingjingnews.com/MIB/news/news.aspx?ID=N000016355 |archive-date=2014-02-01 |dead-url=yes }}</ref>。

2014年12月30日，60岁的蒋定之因为耳疾的原因辞去海南省省长，由[[国家海洋局|国家海洋局]]局长[[刘赐贵|刘赐贵]]接替。蒋定之被时任江苏省委常委、组织部长[[王炯|王炯]]接回家乡江苏任职，提前退居二线。
