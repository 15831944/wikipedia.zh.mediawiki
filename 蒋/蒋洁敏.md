'''蒋洁敏'''（{{bd|1955年|10月4日||catIdx=J蒋}}），[[山东|山东]][[阳信|阳信]]人。毕业于[[山东大学|山东大学]]。他是中共第十七届中央候补委员，第十八届中央委员。第九届、十届全国人大代表。他是中共十八大后首位被查的时任中共中央委员。

== 官场生涯 ==
1976年8月加入[[中国共产党|中国共产党]]，[[中央党校|中央党校]]函授学院省部级干部[[在职研究生|在职研究生]]班[[政治学|政治学]]专业毕业，历任[[山东胜利石油管理局|山东胜利石油管理局]]副局长，[[青海石油管理局|青海石油管理局]]局长，[[中国石油天然气集团公司|中国石油天然气集团公司]]总经理助理，[[中国石油天然气股份有限公司|中国石油天然气股份有限公司]]董事、副总裁，[[青海省人民政府|青海省人民政府]]副省长，[[中共青海省委|中共青海省委]]常委、省委副书记，[[中国石油天然气集团公司|中国石油天然气集团公司]]副总经理、总经理，[[中国石油天然气股份有限公司|中国石油天然气股份有限公司]]副董事长、总裁、董事长等职。<ref>[http://www.apdnews.com/news/33431.html 港媒：内地反腐升级 蒋洁敏落马] {{webarchive|url=https://web.archive.org/web/20130904071737/http://www.apdnews.com/news/33431.html |date=2013-09-04 }}，亚太日报，2013年9月2日</ref>

2011年4月起出任[[中国石油集团|中国石油集团]]董事长<ref>{{cite web |accessdate=2012-05-01 |url = http://news.sohu.com/20111107/n324725730.shtml |title = 中石油集团建立董事会 蒋洁敏将出任董事长 |publisher=[[搜狐|搜狐]]}}</ref>，直至2013年3月出任[[国务院国有资产监督管理委员会|国务院国有资产监督管理委员会]]主任<ref>{{cite web |accessdate=2013-03-19 |url=http://news.sina.com.hk/news/20130319/-9-2922183/1.html |title=中石油蔣潔敏掌國資委 |publisher=[[新浪網|新浪網]] |deadurl=yes |archiveurl=https://web.archive.org/web/20150402091552/http://news.sina.com.hk/news/20130319/-9-2922183/1.html |archivedate=2015-04-02 }}</ref>，接替出任[[国务委员|国务委员]]的前国务院国资委主任王勇。但仅仅半年之后，据大陆官方媒体2013年9月1日消息：国务院国资委主任、党委副书记蒋洁敏因涉嫌严重违纪，已经由[[中共中央纪委|中共中央纪委]]调查<ref>[http://www.apdnews.com/news/33422.html 中国国资委主任蒋洁敏被中纪委调查] {{webarchive|url=https://web.archive.org/web/20131227040541/http://www.apdnews.com/news/33422.html |date=2013-12-27 }}，[[亚太日报|亚太日报]]，2013年9月2日</ref><ref>{{cite web |accessdate=2013-09-01 |url = http://www.chinanews.com/fz/2013/09-01/5230108.shtml |title = 国资委主任蒋洁敏涉嫌严重违纪正接受调查 |publisher = [[中新網|中新網]]}}</ref>。

2014年6月30日，[[中共中央政治局|中共中央政治局]]会议决定，将蒋洁敏开除党籍，移送司法机关处理<ref>[http://finance.ifeng.com/a/20140630/12630126_0.shtml 蒋洁敏严重违纪违法被开除党籍]，凤凰网财经，2014年6月30日</ref>。2015年3月19日，蒋洁敏涉嫌受贿、巨额财产来源不明、国有公司人员滥用职权一案由湖北省人民检察院[[汉江|汉江]]分院向湖北省汉江中级人民法院提起[[公诉|公诉]]。<ref>{{cite web |url=http://legal.gmw.cn/2015-03/19/content_15153172.htm |language= |title=最高检：李春城、蒋洁敏被依法提起公诉 |publisher=光明网 |date=2015-03-19 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150321012744/http://legal.gmw.cn/2015-03/19/content_15153172.htm |archivedate=2015-03-21 }}</ref>

2015年4月13日，蒋洁敏一案开庭审理，检察院认定其收受14个单位和个人财物累计1403．9073万元，以及有1482．6174万元的巨额财产不能说明来源。2004年至2008年受周永康之托，利用职权为他人获取非法利益。他对这些指控供认不讳，法院将择期宣判。<ref>{{cite web |url=http://news.163.com/15/0413/19/AN3S8FOT00014JB5.html |language= |title=蒋洁敏案一审休庭 法庭将依法择期宣判 |publisher=新华网 |date=2015-04-13 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150416034657/http://news.163.com/15/0413/19/AN3S8FOT00014JB5.html |archivedate=2015-04-16 }}</ref>

2015年10月12日，湖北省汉江中级人民法院一审公开宣判国务院国有资产监督管理委员会原主任蒋洁敏受贿、巨额财产来源不明、国有公司人员滥用职权案。法庭认定被告人蒋洁敏犯受贿罪判处有期徒刑十二年，并处没收个人财产人民币一百万元；犯巨额财产来源不明罪判处有期徒刑五年；犯国有公司人员滥用职权罪判处有期徒刑五年，决定执行有期徒刑十六年，并处没收个人财产人民币一百万元。同时，对蒋洁敏受贿、巨额财产来源不明的财物和滥用职权使他人获取的非法利益予以追缴，上缴国库。蒋洁敏当庭表示服判，不上诉。法庭经审理查明，2004年至2013年，被告人蒋洁敏利用职务便利，为他人在项目建设、职务调整等事项上谋取利益，收受贿赂折合人民币1403.9073万元；截至2013年8月31日，蒋洁敏个人和家庭财产、支出明显超过其个人和家庭合法收入，对于差额部分中的1476.6174万元的巨额财产不能说明来源；2004年至2008年，蒋洁敏在周永康指使或授意下，违反规定，为他人开展经营活动提供帮助，致使国家利益遭受特别重大损失。汉江中级人民法院认为，被告人蒋洁敏的行为已构成受贿罪、巨额财产来源不明罪和国有公司人员滥用职权罪，依法应数罪并罚。到案后，蒋洁敏能够主动交代侦查机关尚未掌握的受贿事实，构成自首，还能够如实供述国有公司人员滥用职权事实，认罪悔罪，积极退缴全部赃款赃物，依法可予从轻处罚。据上，法庭依法作出上述判决。<ref>{{cite news|accessdate=2015-10-12|url = http://news.sina.com.cn/c/nd/2015-10-12/doc-ifxirmpy1530659.shtml |title = 国资委原主任蒋洁敏一审被判处有期徒刑16年 |publisher = 新浪网}}</ref>

== 南堡油田骗局 ==
{{main|南堡油田}}
南堡油田，是隶属于[[中国石油天然气集团公司|中国石油天然气集团公司]]、位于曹妃甸区的[[油田|油田]]。2007年5月3日，中石油高调宣称冀东南堡油田油气总储量为10.2亿吨，探明储量4.05亿吨，约为30亿桶，媒体称之为“40多年来中国石油勘探最激动人心的发现”。但数月之后，被证实其实际储量远小于这一数据，该油田经济可采储量仅为6.35亿桶。<ref>{{cite web | url=http://www.21cbh.com/HTML/2010-5-28/5NMDAwMDE3OTU5NQ.html | title=中石油冀东南堡油田储量或少于10亿吨预期 | publisher=21世纪网 | accessdate=2016-07-04 | deadurl=yes | archiveurl=https://web.archive.org/web/20100608040021/http://www.21cbh.com/HTML/2010-5-28/5NMDAwMDE3OTU5NQ.html | archivedate=2010-06-08 }}</ref>2010年5月，[[路透社|路透社]]报道，冀东南堡油田储量低于预期。<ref>{{cite web | url=http://cn.reuters.com/article/CNAnalysesNews/idCNCHINA-2379920100527 | title=冀东大油田储量低于预期 中国仍将依赖石油进口 | publisher=路透社 | accessdate=2016-07-04}}</ref>此后，关于冀东南堡油田的报道越来越少。2013年，蒋洁敏被双规后，南堡油田被认为是其在中石油任内的“注水政绩”。<ref>{{cite web | url=http://news.qq.com/a/20130917/006891.htm | title=蒋洁敏境外运油重庆建厂 为薄熙来政绩做文章 | publisher=腾讯网 | accessdate=2016-07-04}}</ref>南堡油田的储量骗局令曹妃甸区陷入困局<ref>{{cite news|title=南堡油田造假蔣潔敏屬禍首|url=http://hk.apple.nextmedia.com/realtime/7015342/20131008/51794468|accessdate=2016-07-04|agency=香港苹果日报}}</ref>。

== 贱卖油田 ==
蒋洁敏曾指示中石油管理层向国资委谎称[[辽河油田|辽河油田]]无油可采，需要废弃。蒋洁敏协助周永康之子[[周滨|周滨]]和吴兵（周家的白手套）的公司以1000万人民币收购辽河油田。周永康家族通过此油田赚取了数十亿暴利<ref name="jmyt">{{cite web|url=http://orientaldaily.on.cc/cnt/china_world/20131017/00178_029.html|title=蔣潔敏認出術助買油田 周永康子三年賺40億|publisher=东方日报|date=2013年10月17日| language =zh-hk }}</ref>。后因油田职工的上访，国资委派人审计调查，发现辽河油田还有当时价值3000亿元的石油储备<ref name="jmyt"/>。

辽河油田于2009年下半年勘探发现高升井区有新油出现，但周永康家族却秘而不宣。经香港联合能源集团顾问黄炎（原中石油总经理）勾兑，该公司参与合作开发新采油区。辽河油田从账面上得到香港联合能源集团1000万人民币。但是香港联合能源集团董事长张宏伟私底下在香港付给原中央政法委书记周永康的儿子周滨8亿人民币。张宏伟曾在上海雇佣黑社会暴打维权律师严义明，主犯被周永康命令上海市公安局长[[张学兵|张学兵]]以证据不足为名释放。<ref>{{cite web|url=http://dailynews.sina.com/gb/chn/chnnews/ausdaily/20131213/01575265458.html|title=周永康父子贱卖辽河油田事发 被确认软禁|publisher=新浪网|date=2013年12月13日| language =zh-hans }}</ref>
