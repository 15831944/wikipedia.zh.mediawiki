'''蒋耀平'''（{{bd|1952年|9月||}}）[[江苏|江苏]][[武进|武进]]人，[[中华人民共和国|中华人民共和国]]官员。

==生平==
蒋耀平1974年7月加入[[中国共产党|中国共产党]]。[[广西民族学院|广西民族学院]]外语系毕业，[[挪威|挪威]]管理学院信息管理硕士，[[哈尔滨工业大学|哈尔滨工业大学]]管理科学与工程博士。教授级高级工程师。历任[[南宁市|南宁市]]电信局副局长、局长，[[广西壮族自治区|广西壮族自治区]]邮电管理局副局长、局长，[[国家信息化工作领导小组|国家信息化工作领导小组]]计算机网络与信息安全管理办公室主任，[[中华人民共和国信息产业部|中华人民共和国信息产业部]]政策法规司司长。2004年4月任信息产业部副部长、[[党组|党组]]成员。2008年3月任[[中华人民共和国商务部|中华人民共和国商务部]]副部长、党组成员<ref>{{Cite web |url =http://jiangyaoping.mofcom.gov.cn/aarticle/resume/200804/20080405505747.html  |title = 蒋耀平简历 |author =  |date =2008-04-30  |publisher =中华人民共和国商务部  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.ce.cn/xwzx/gnsz/gdxw/201112/08/t20111208_22902598.shtml|title = 资料:商务部副部长蒋耀平简历 |author =  |date = 2011-12-08 |publisher =  中国经济网|language =  |accessdate =  |quote =  }}</ref>。2013年6月，免去蒋耀平的商务部副部长、党组成员职务<ref>{{Cite web |url =http://bgt.mofcom.gov.cn/article/c/d/201306/20130600163163.shtml  |title = 中共中央国务院任免商务部领导人员 |author =  |date = 2013-06-14 |publisher =  中华人民共和国商务部|language =  |accessdate =  |quote =  }}</ref>。2013年4月起，任[[海峡两岸关系协会|海峡两岸关系协会]]副会长。

他是[[第十二届全国政协委员|第十二届全国政协委员]]。
