'''蒋光慈'''（{{bd|1901年|9月11日|1931年|8月31日}}），学名'''如恒'''、后改'''宣恒'''，自号'''侠生'''、'''侠增'''，笔名'''光赤'''、'''光慈'''、'''华希理'''、'''维素'''、'''华维素'''、'''魏克特'''、'''敦夫'''、'''陈情'''等。[[安徽|安徽]][[六安|六安]][[霍邱县|霍邱]]白塔畈（今属金寨）人。小说作家。

== 生平 ==
蒋光慈的祖辈原籍是河南人，家里贫穷，从祖父起流落到安徽，靠抬轿为生。父亲蒋从甫，开一杂货铺，后自学诗文，被当地一塾馆聘为教师。蒋光慈十一岁，到志诚小学读书，受国文教员[[詹谷堂|詹谷堂]]影响甚大。蒋光慈参加了詹谷堂组织的读书会，阅读了大量进步书刊。1916年夏，蒋光慈完成高等小学学业，并考入河南省固始中学。他不满校长嫌贫爱富的做法，于是召集学校里较为贫困的同学打了校长，因而被学校开除学籍，返回家乡。1917年夏，入[[安徽省立第五中学|安徽省立第五中学]]学习<ref>{{Cite book|title=文学的鉴赏与评论上 方铭文集|last=[[方铭|方铭]]著|first=|publisher=合肥：[[安徽大学出版社|安徽大学出版社]]|year=2014.03|isbn=978-7-5664-0684-2|location=|pages=227-230}}</ref>。次年，与[[李宗邺|李宗邺]]、[[钱杏邨|钱杏邨]]、[[李克农|李克农]]等在五中成立安社，信仰[[无政府主义|无政府主义]]，发表反对军阀、列强、私有制文章。[[五四运动|五四运动]]爆发后，改名光赤，歌颂[[十月革命|十月革命]]，是芜湖学生运动领袖之一。1920年春，入[[上海共产主义小组|上海社会主义青年团]]所办外国语学校学习，并加入社会主义青年团。次年，赴苏联[[莫斯科东方大学|莫斯科东方大学]]学习。1922年，加入[[中国共产党|中国共产党]]。1924年回国，先在河南固始开展建党活动，后到[[上海大学|上海大学]]任教。1925年，去北京[[中共北方区委|中共北方区委]]工作。[[第一次国共合作|第一次国共合作]]破裂后，根据[[瞿秋白|瞿秋白]]指示，于1928年初与[[阿英|阿英]]等组织[[太阳社|太阳社]]，创办[[春野书店|春野书店]]，主编《[[太阳月刊|太阳月刊]]》杂志<ref>{{Cite book|title=安徽历史名人词典 下|last=《安徽历史名人词典》编辑委员会编|first=|publisher=合肥：[[安徽教育出版社|安徽教育出版社]]|year=2008.05|isbn=978-7-5336-4867-1|location=|pages=1126}}</ref>。

1930年，左联成立，蒋光慈被选为候补常委，负责主编机关刊物《拓荒者》。积极提倡革命文学，并致力于文学创作。同年秋，因和“左联”领导发生争执，递交退党书，遂被开除出党。

1931年8月31日，病逝于[[上海同仁医院|上海同仁医院]]<ref>{{Cite book|title=上海名人名事名物大观|last=[[熊月之|熊月之]]主编|first=|publisher=上海：[[上海人民出版社|上海人民出版社]]|year=2005.01|isbn=7-208-05281-6|location=|pages=270}}</ref>。

== 作品 ==
著有诗集《新梦》、《哀中国》、《哭诉》，短篇小说集《鸭绿江上》，中篇小说《少年漂泊者》、《短裤党》、《野祭》、《菊芬》、《最后的微笑》<ref>{{Cite book|title=共产党人理想之歌 第1册|last=[[戚积广|戚积广]]选编|first=长春：[[时代文艺出版社|时代文艺出版社]]|publisher=2009.07|year=978-7-5387-0337-5|isbn=|location=|pages=70}}</ref>，长篇小说《咆哮了的土地》、《丽莎的哀怨》、《冲出云围的月亮》，译著有苏联名著《一周间》。有《蒋光慈文集》行世<ref>{{Cite book|title=你必须知道的2500个文学常识|last=文学常识编委会编|first=|publisher=重庆：[[重庆大学出版社|重庆大学出版社]]|year=2012.03|isbn=978-7-5624-6209-5|location=|pages=151}}</ref>。

其小说多以大革命为背景，采用“革命加恋爱”的主题，表现历史转变关头革命青年的苦闷、悲愤和奋起抗争的精神世界<ref>{{Cite book|title=历史文化常识全知道|last=[[斗南|斗南]]主编|first=|publisher=北京：[[中国华侨出版社|中国华侨出版社]]|year=2015.01|isbn=978-7-5113-4439-7|location=|pages=421}}</ref><ref>{{Cite book|title=你一定要懂的文学知识|last=[[王贵水|王贵水]]编著|first=|publisher=北京：[[北京工业大学出版社|北京工业大学出版社]]|year=2015.02|isbn=978-7-5639-4181-0|location=|pages=97}}</ref>。
