'''花蓮港車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[花蓮市|花蓮市]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[花蓮臨港線|花蓮臨港線]]的[[鐵路車站|鐵路車站]]，為不辦客運之[[貨運車站|貨運車站]]。

第一代的花蓮港車站其實並非今日的花蓮港車站，而是已經拆除成為市區道路用地的[[花蓮舊站|花蓮舊站]]。

== 車站構造 ==
*貨運車站，無月台，共四股道。

== 利用狀況 ==
* 目前為[[三等站_(台鐵)|三等站]]，僅有貨運列車停靠。
* [[臺東線拓寬計畫|臺東線拓寬計畫]]前的[[臺東線|臺東線]]由此站發車，在[[花蓮高中|花蓮高中]]附近接送學生至[[花蓮車站|花蓮舊站]]。

== 車站週邊 ==
*[[花蓮港|花蓮港]]
*[[內政部警政署|內政部警政署]]花蓮港務警察總隊
*[[財政部關務署|財政部關務署]][[基隆關|基隆關]]花蓮分關
*美崙海濱公園
*花蓮漁市
*花蓮港景觀橋
*85度C海岸店

==歷史==
*1939年9月20日：設站，初名'''東花蓮港驛'''。
*[[第二次世界大戰|第二次世界大戰]]期間，遭[[美軍|美軍]]炸毀。
*1949年6月24日：車站重建。
*1948年1月1日：更名為'''花蓮港車站'''之現名。
*1982年12月：改建為目前的站體。
