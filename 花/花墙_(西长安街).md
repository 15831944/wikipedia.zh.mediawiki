'''花墙'''，位于[[北京市|北京市]][[西城区|西城区]][[西长安街|西长安街]]南侧，[[新华门|新华门]]对面。

==简介==
[[新华门|新华门]]在[[清朝|清朝]]时为宝月楼，建成于1758年。其正南方原先是[[清真普宁寺|回回营清真寺]]，建成于1764年。传说中，宝月楼、回回营清真寺都与[[香妃|香妃]]有关<ref name=xiangfei>{{Cite web |url =  http://finance.ifeng.com/city/msls/20090609/767252.shtml|title = 东安福胡同：一座与香妃有关的清真寺 |author =  |date = 2009-06-09 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref name=zuobie/>。

[[中华民国|中华民国]]成立后，[[中华民国临时政府_(袁世凯)|袁世凯政府]]接管了清室的西苑三海，1913年将中海和南海辟为[[中华民国大总统府|大总统府]]，从此中海和南海合称“[[中南海|中南海]]”<ref name=zuobie/>。1913年，由内务总长兼京都市政公所督办[[朱启钤|朱启钤]]主持，将宝月楼辟为中南海正门，称“新华门”。将新华门前的一段西长安街命名为“府前街”，将中南海西侧新辟道路命名为“[[府右街|府右街]]”（“府”均指大总统府）。此后，又将[[六部口|六部口]]以南的排水沟上改建的道路命名为“新华街”（今[[北新华街|北新华街]]、[[南新华街|南新华街]]），还在新华门对面的“府前街”南侧砌筑一道西洋式花墙，以遮挡背后杂乱破旧的平房，改善大总统府门前的观瞻<ref>{{Cite web |url =http://www.takefoto.cn/viewnews-868753.html  |title =北京中南海新华门开始维修 崭新面貌值得期待  |author =  |date =2016-08-03  |publisher = 北晚新视觉 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.sohu.com/61/50/news148075061.shtml |title = 20世纪初改造了北京城的人--京城名人故居与轶情 |author =  |date = 2002-03-06 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。由朱启钤主持砌筑的这道西洋式花墙长约200米，与新华门之间的比例很协调，设计上突出了中南海<ref name=zuobie/>。同时，此次朱启钤主持的工程中还拆除了回回营清真寺位于北面的门楼式的正门，用花墙封堵了清真寺通往北面的通路<ref name=xiangfei/>。

2009年4月1日，[[西长安街|西长安街]]道路拓宽拆迁工程进入施工阶段。该项目位于西长安街新华门南侧，东到[[石碑胡同|石碑胡同]]，西到[[中共中央组织部|中共中央组织部]]办公楼，北到西长安街南沿线，南到[[西安福胡同|西安福胡同]]、[[东安福胡同|东安福胡同]]西段、东安福胡同东段南延50米。拆迁范围内共有单位42户，居民778户。由北京市西城区人民政府成立拆迁指挥部，主导拆迁。拆迁完成后，该路段由双向8车道拓宽为双向10车道，交通获得改善。在此次道路拓宽拆迁工程中，将花墙拆除后南移，按照原样重建<ref name=zuobie>{{Cite web |url = http://news.163.com/09/0420/15/57BQI3T800011SM9.html |title = 作别老长安街 |author =  |date =2009-04-20  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。
