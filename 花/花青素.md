'''花青素
'''（{{lang-en|'''anthocyanidin'''}}）或稱'''花色素'''，[[化學式|化學式]]為C<sub>15</sub>H<sub>11</sub>O<sub>6</sub>，是一種[[水溶性|水溶性]]的[[生物色素|植物色素]]，存在于[[液泡|液泡]]内的细胞液中。其与[[糖类|糖类]]物质以[[糖苷键|糖苷键]]结合之后即为[[花色苷|花色苷]]，與花的顏色、葉變紅等有關，是一種天然的[[抗氧化劑|抗氧化劑]]。

花青素的顏色會隨着身處環境的[[酸鹼值|酸鹼值]]而有所變化，從酸性環境的紅色到紫色、再到鹼性環境下的藍色。因此，花青素亦有用於[[試紙|試紙]]。可吸收光能，但與[[光合作用|光合作用]]無關。

== 語源 ==
「花青素」這個名稱源自其希臘語名稱的直譯：
* {{polytonic|''ἀνθός''}}(''anthos'') = 花；
* {{polytonic|''κυανός''}}(''kyanos'') = 青色的。

== 用途 ==
花青素可作[[食物添加劑|食物添加劑]]使用，代號是[[E編碼|E]]163。

== 常见的花色素分子 ==

常见的花色素分子有六种，分别是：
* 矢车菊素(cyanidin)
* 天竺葵素(pelargonidin)
* 芍药色素(peonidin)
* 飞燕草素(delphinidin)
* 牵牛花素(petunidin)
* 锦葵色素(malvidin)

花色素不稳定，自然界中花色素多以结合糖甘配基的方式存在，后者被称作花色苷。
== 近期研究 ==
2004年12月，[[美國|美國]][[密歇根州立大學|密歇根州立大學]]在《[[美國化學學會|美國化學學會]]》（''American Chemical Society''）雜誌上刊登研究文章，指花青素有助提高身體的[[胰島素|胰島素]]產量達50%。不過，研究尚在起步階段，還需更多研究。
