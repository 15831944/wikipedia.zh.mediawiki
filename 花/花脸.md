'''花脸'''，或稱'''淨'''，是[[中國|中國]][[戲劇|戲劇]]的一種程式化的角色行當，一般扮演性格剛烈、暴躁甚至陰險兇惡的男性角色，要勾畫臉譜，因此俗稱「花臉」。

== 类别 ==
各個劇種有所不同，但比較一致的大概分為如下幾種：
* 架子花臉（副淨）：也叫二花臉，重做工念白，一般扮演地位較高，性格暴躁的男性角色，如：[[曹操|曹操]]、[[張飛|張飛]]、竇爾墩等。
* 銅錘花臉（正淨）：即大花臉，重唱功，一般扮演性格剛強的男性角色，如：[[包拯|包公]]、《[[二進宮|二進宮]]》的[[徐彦昭|徐彦昭]]等。
* 三塊瓦（武丑）：一般扮演地位低下、性格特殊、有武功的男性角色，重做工武打，如：《[[水滸傳|水滸傳]]》中的[[時遷|時遷]]等。

== 昆曲净角 ==

== 越剧净角 ==
[[越剧|越剧]]里的净多称作[[大面|大面]]，俗称大花脸，多为奸恶角色，有时亦为耿直角色。如《[[追鱼|追鱼]]》中的[[包拯|包公]]。
