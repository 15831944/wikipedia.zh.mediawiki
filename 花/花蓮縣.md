'''花蓮縣'''（{{lang-nan-tw|Hoa-lian-koān / Hoa-liân-koān}}；{{lang-hak|Fâ-lièn-yen}}），[[中華民國|中華民國]][[臺灣省|臺灣省]]的[[縣_(中華民國)|縣]]，位於[[臺灣本島|臺灣本島]][[東臺灣|東部]]，[[北回歸線|北回歸線]]通過其南部。西臨高聳的[[中央山脈|中央山脈]]，東臨[[太平洋|太平洋]]，周邊與[[宜蘭縣|宜蘭縣]]、[[台中市|台中市]]、[[南投縣|南投縣]]、[[高雄市|高雄市]]、[[台東縣|台東縣]]相鄰，是臺灣面積最大的[[縣_(中華民國)|縣級行政區]]。

人口主要聚集於[[花東縱谷|花東縱谷]]與[[海岸山脈|海岸山脈]]最北端的沿海地帶，其以自然[[觀光|觀光]]資源著名，境內有[[太魯閣國家公園|太魯閣國家公園]]、[[玉山國家公園|玉山國家公園]]。

縣旗以太陽的故鄉為設計理念，太陽在海面昇起的意象，象徵太平洋的曙光；每個光點代表各族群色彩：[[阿美族|阿美族]]、[[太魯閣族|太魯閣族]]、[[布農族|布農族]]、[[噶瑪蘭族|噶瑪蘭族]]、[[撒奇萊雅族|撒奇萊雅族]]、[[賽德克族|賽德克族]]與[[漢族|漢族]]，各大族群融合。

== 地理 ==
花蓮縣在地理形狀上，呈現長條狀，西邊是高聳的臺灣本島[[中央山脈|中央山脈]]，東臨[[太平洋|太平洋]]（[[菲律賓海|菲律賓海]]）

花蓮縣的南北長約137.5[[公里|公里]]，東西寬約43公-{里}-。

其面積雖為全臺灣各縣市之冠，但為多山地形，平原僅有全面積的百分之10。多數人口集中在平原一帶，其中有半數人口居住於市區。花蓮縣的平原地帶並非直接面向海洋，而是夾在東側[[海岸山脈|海岸山脈]]與西側的中央山脈之間。寬度僅有3到9公里，地理上屬於縱谷平原的河川[[沖積扇|沖積扇]]地形，稱為[[花東縱谷|花東縱谷]]或花東走廊。

花蓮縣東側的海岸山脈由於生成時間較晚（約400萬年前形成），高度低於西側的中央山脈相當多，最高的[[新港山|新港山]]海拔為1682公尺。海岸山脈地質上多為軟性的[[沉積岩|沉積岩]]、易受風化。目前受到地殼運動影響、海岸山脈1年上升3公分。

中央山脈在花蓮縣內占有相當大的比率，其中高於3000公尺以上，列入「[[臺灣百岳|臺灣百岳]]」的山峰，在花蓮縣就有43座，最著名的有[[南湖大山|南湖大山]]、[[奇萊山|奇萊山]]、[[秀姑巒山|秀姑巒山]]等。

花蓮縣內主要的河川多為西到東向，由北到南依序有[[和平溪|和平溪]]（過去稱為大濁水溪）、[[立霧溪|立霧溪]]、[[花蓮溪|花蓮溪]]和[[秀姑巒溪|秀姑巒溪]]。

[[File:Taiwan_2009_CingShui_Cliffs_on_SuHua_Highway_FRD_6762_Pano_Extracted.jpg|thumb]]
花蓮東岸的海岸線長有124公-{里}-，由於有多條河流出海，造成種類眾多的海岸景觀，其中以斷崖峭壁為最險峻的景觀，特別是和平溪口南邊的[[清水斷崖|清水斷崖]]，高1000公尺以上，是蘇花公路上的知名景點，也常因雨造成崩塌而道路中斷。

此外，在花蓮縣南方的[[石門海岸|石門海岸]]區，屬於堅硬的[[礁岩|礁岩]]岸區，有相當多[[海蝕|海蝕]]的景觀，再南方一點還有[[石梯坪|石梯坪]]，此地的岩盤如石階般伸入海中，亦有許多海蝕溝、海蝕崖、海蝕平台、隆起珊瑚礁等，其中[[壺穴|壺穴]]景觀堪稱臺灣第一。

== 歷史 ==
花蓮古稱'''[[奇萊|奇萊]]'''，據《花蓮縣志》記載，[[西班牙|西班牙]]人在1622年到這裡採取砂金，並將花蓮地區稱為'''[[哆囉滿省區|多羅滿]]'''（西班牙語：Turoboan），是[[臺灣原住民|臺灣原住民]]的天下。而漢人至花蓮開墾是[[咸丰_(年号)|咸豐]]元年（1851年），相傳來此開墾的[[閩南人|閩南人]]，見到溪水奔海，而與波瀾交會，作迴旋狀，故稱之為[[洄瀾|洄瀾]]，後改為[[閩南語|閩南語]]音近的花蓮。

相較臺灣其他地方，花蓮的開發較遲。在[[台灣日治時期|日治時期]][[昭和|昭和]]7年（1932年）臨海道路（今[[蘇花公路|蘇花公路]]之前身）通車前，往來花蓮通常需要乘船、而有「陸上離島」之稱。直至[[北迴線|北迴鐵路]]通車後，花蓮的交通終獲大幅度改善。

行政制度方面，[[明治|明治]]42年（1909年），花蓮地區脫離臺東，成為[[花蓮港廳|花蓮港廳]]，獨立設治至今。[[臺灣戰後時期|中華民國接管台灣後]]，{{noteTag|或稱接收台灣、[[臺灣光復|光復台灣]]、[[台灣地位未定論|佔領台灣]]}}於1946年1月9日設立花蓮縣。1950年10月15日成為第一個依據《[[中華民國憲法|中華民國憲法]]》實行[[地方分權|地方自治]]的縣市<ref>{{cite ==文學==
在[[文學史|文學史]]及其論述方面，目前還沒有一本「花蓮區域文學」、「花蓮文學史」之類的專書出版，比較相關的書籍有：文訊雜誌社出版的「花東地區文學會議」論文集、《第二屆花蓮文學研討會論文集》、《第三屆花蓮文學研討會》；花蓮地區原住民族的傳說故事、戰後文學發展、地誌書寫、文學獎等議題，已有人研究，博碩士論文有林淑湄〈花蓮地區詩歌研究(1945-1989)〉、黃嘉眉〈花蓮地區[[撒奇萊雅族|撒奇萊雅族]]傳說故事研究〉梁文祥〈花蓮海神信仰及傳說故事─以大陳人、[[阿美族|阿美族]]人、[[噶瑪蘭族|噶瑪蘭]]人為例〉、王衍智〈旅人與移民者的花蓮書寫(2001~2010)〉、馬翊航〈虛實對照，城鄉融涉─論花蓮文學中的地方意識與市/街書寫〉、張宏誠〈台灣解嚴後縣市文學獎研究(1987~2010)〉等；期刊論文則有黃憲作〈花蓮地區的傳統文學〉、文訊雜誌社所出版的「花東地區文學會議」論文集、[[王惠珍_(學者)|王惠珍]]〈地誌書寫港市想像—[[龍瑛宗|龍瑛宗]]的花蓮文學〉、[[王浩威|王浩威]]〈地方文學與地方認同：以花蓮文學為例〉等。

在[[民間文學|民間文學]]的編採方面，花蓮縣文化局在2009年出版《花蓮客家民間文學集》、吳淑姿總編輯的《花蓮縣民間文學集》、李進益總編輯的《花蓮縣民間文學集(二)》；原住民族的文學方面，目前有：林道生編著的《台灣原住民族口傳文學選集》、《花蓮原住民音樂(一)：[[布農族|布農族]]篇》、《花蓮原住民音樂(三)：[[泰雅族|泰雅]]篇》。

推廣及贊助文學活動的基金會有「花蓮縣文化基金會」<ref>台灣文學館出版《2007 台灣文學年鑑》的「文學活動基金會」名錄。</ref>；縣內可能沒有全國性與地方性文學社團<ref>台灣文學館出版《2007 台灣文學年鑑》的「文學團體」名錄。</ref>。

花蓮縣文化局舉辦的文學活動有：
*舉辦「[[花蓮文學獎|花蓮文學獎]]」徵文活動及其得獎作品集的出版
*舉行5屆的「花蓮文學研討會」，並且出版研討會論文集。
*舉辦書寫花蓮之類的徵文活動，並且出版作品集，目前有：吳淑姿總編輯的《書寫花蓮》與《花蓮故事》、吳進書主編的《愛上花蓮的理由》與《給花蓮的一首詩》。

在花蓮縣出生及後來歸入花蓮縣的作家、[[文學評論|文學評論]]者，有[[楊牧_(台灣)|楊牧]]、[[李崗|李崗]]、[[李潼|李潼]]、[[顏崑陽|顏崑陽]]、[[馬得倫|馬得倫]]（筆名：[[馬蔚|馬蔚]]）、[[張開基（台灣）|張開基]]、[[陳克華|陳克華]]、[[陳雨航|陳雨航]]、[[陳義芝|陳義芝]]、[[陳慧文|陳慧文]]、[[孫昭懿|孫昭懿]]（[[童童|童童]]）、[[鄭開來|鄭開來]]、[[藍國揚|藍國揚]]、[[羅元輔|羅元輔]]（[[羅葉|羅葉]]）等<ref>國立台灣文學館，「2007台灣作家作品目錄」搜尋系統。</ref>、[[劉富士|劉富士]]、[[葉日松|葉日松]]、[[陳黎|陳黎]]、[[黃錦城|黃錦城]]、[[陳贊昕|陳贊昕]]、[[林蒼鬱|林蒼鬱]]、[[林豐明|林豐明]]、[[吳瑩|吳瑩]]、[[王浩威|王浩威]]、[[張子樟|張子樟]]、[[高自芬|高自芬]]、[[凌性傑|凌性傑]]、[[吳岱穎|吳岱穎]]、[[廖鴻基|廖鴻基]]等人。

== 政治 ==
花蓮縣位於[[中華民國|中華民國]][[臺灣|臺灣]]東部，別稱後山，乃全國面積最大的縣份，以自然觀光資源著名。由於商業活動並不頻繁，居民多是原居民和軍公教人員，政治表現上也較為保守，藍綠基本盤約為七比三。花蓮縣地方選舉史上除第一屆縣長楊仲鯨為[[黨外_(臺灣)|黨外人士]]外，歷屆公職人員選舉一直由[[中國國民黨|中國國民黨]]長期執政，直到1992年的立法委員選舉，前[[民主進步黨|民進黨]]主席黃信介參選花蓮市區域立委，以近三萬票當選立法委員後，泛綠才突破泛藍在花蓮的一黨優勢之局面。
