{{日本作品|花之天使的夏日戀歌|フローラル・フローラブ}}是[[SAGA_PLANETS|SAGA PLANETS]]在2016年7月29日<ref>{{Cite web|url=https://www.game-style.jp/special/201606/21/00flo_sp.php|title=Game-Style「花之天使的夏日戀歌」特集|publisher=Game-Style|date=2016-06-21|accessdate=2018-05-12|language=ja}}</ref>發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]。2018年8月23日由Entergram發售[[PlayStation_4|PlayStation 4]]與[[PlayStation_Vita|PlayStation Vita]]版本<ref>{{Cite web|url=https://www.gamer.ne.jp/news/201804260115/|title=PS4/PS Vita「フローラル・フローラブ」が8月23日に発売―天使のようなヒロインたちとの恋愛を楽しもう！|publisher=Gamer|date=2018-04-26|accessdate=2018-05-12|language=ja}}</ref>。

== 故事簡介 ==
齊須柾鷹所就讀的高中「聖加百列學園」有著被譽為「四大天使」的四名美少女，而其中的一位女孩美鳩夏乃對柾鷹抱持著好感，並每天向他示愛。之後柾鷹不斷被捲入跟四大天使有關的事件，並和她們的關係變得親密<ref>{{Cite web|url=https://www.game-style.jp/soft/201606/15/01flo_sp.php|title=記念すべき20作目となるSAGA PLANETSの新作は、天使の翼が見える主人公と天使のような美少女たちが織り成す学園ラブストーリー！ 『フローラル・フローラブ』7月29日発売！|publisher=Game-Style|date=2016-06-15|accessdate=2018-05-12|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|齊須柾鷹|斉須 柾鷹（さいす まさたか）}}
: 作品男主角，聖加百列學園二年級生。年幼時曾遇見真正的天使，並得到了能辨別他人的善惡的能力。
; {{nihongo|美鳩夏乃|美鳩 夏乃（みはと かの）|聲：宙白青<ref>{{cite web|url=http://sagaplanets.product.co.jp/works/floflo/chara01.html|title=美鳩 夏乃|publisher=[[SAGA_PLANETS|SAGA PLANETS]]|accessdate=2018年6月16日|language=ja}}</ref><ref name="CHARACTER">{{cite web|url=http://www.entergram.co.jp/floflo/character.html|title=CHARACTER|publisher=Entergram|accessdate=2018年6月16日|language=ja}}</ref>}}
: 名家的大小姐，聖加百列學園二年級生，和柾鷹是青梅竹馬，對柾鷹抱持好感，並且每天對柾鷹展開熱烈追求。不擅長讀書，喜歡看漫畫。
; {{nihongo|朱鷺坂七緒|朱鷺坂 七緒（ときさか ななお）|聲：{{lang|ja|くすはらゆい}}<ref>{{cite web|url=http://sagaplanets.product.co.jp/works/floflo/chara02.html|title=朱鷺坂 七緒|publisher=SAGA PLANETS|accessdate=2018年6月16日|language=ja}}</ref><ref name="CHARACTER"/>}}
: 黑髮少女，聖加百列學園二年級生，柾鷹的同班同學，和夏乃是好朋友。性格冷淡，當對眼前的事物沒興趣時便會開始玩手機。
; {{nihongo|阿德爾海特·馮·貝格施特拉瑟|アーデルハイト・フォン・ベルクシュトラーセ|聲：秋野花<ref>{{cite web|url=http://sagaplanets.product.co.jp/works/floflo/chara03.html|title=アーデルハイト・フォン・ベルクシュトラーセ|publisher=SAGA PLANETS|accessdate=2018年6月16日|language=ja}}</ref><ref name="CHARACTER"/>}}
: 中歐的小國貝格施特拉瑟公國的公主，由歐洲前來日本留學，就讀聖加百列學園一年級。有著如同天使般的美聲。性格慵懶且喜歡睡覺。
; {{nihongo|椿姬戀羽|椿姫 こはね（つばき こはね）|聲：大堂茉莉<ref>{{cite web|url=http://sagaplanets.product.co.jp/works/floflo/chara04.html|title=椿姫 こはね|publisher=SAGA PLANETS|accessdate=2018年6月16日|language=ja}}</ref><ref name="CHARACTER"/>}}
: 聖加百列學園三年級生，柾鷹的學姐。由修道院撫養長大。飼養一隻名為摩西的鼯鼠。

=== 其他角色 ===
; {{nihongo|齊須莉玖|斉須 莉玖（さいす りく）|聲：小鳥居夕花<ref>{{cite web|url=http://sagaplanets.product.co.jp/works/floflo/chara05.html|title=斉須 莉玖|publisher=SAGA PLANETS|accessdate=2018年6月16日|language=ja}}</ref><ref name="CHARACTER"/>}}
