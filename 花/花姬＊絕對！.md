{{日本作品|花姬＊絕對！|ハナヒメ＊アブソリュート！}}是mirai在2016年8月27日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]。<ref>{{cite web |title=ハナヒメ＊アブソリュート！ 初回限定版 |url=http://www.getchu.com/soft.phtml?id=822244 |website=Getchu.com |accessdate=2019-04-11 |language=ja}}</ref>

==故事==
2025年NoRN公司舉辦花姬們與星王子們對戰[[社群網路遊戲|社群網路遊戲]]「Crown Hearts」的年度活動「神冠大戰」，愛内虎春參加劍花姬猫屋敷芽亞和Polina主持「神冠大戰[[VR|VR]]」開幕式。但是在儀式進行途中卻突然出現遊戲的怪獸並且與芽亞發生戰鬥，就在芽亞陷入苦戰時虎春在Polina的引導成為負責領導指揮花姬們的國王。花姬們擊倒怪獸開幕式結束後，虎春等人卻發現原本應該是虛擬遊戲角色的Polina會是真實存在的人。

==開發==
《[[閃耀十字軍|閃耀十字軍]]》的製作團隊代表{{lang|ja|かんなぎれい}}於2012年4月離開{{lang|ja|ぺんしる}}公司的Lillian後，於同年5月來到{{lang|ja|ホークアイ}}公司成立mirai。<ref>{{cite web |title=ご報告です。 |url=http://blog.livedoor.jp/kannagi_hp/archives/52024092.html |website=ライブドアブログ |language=ja |date=2012年12月29日}}</ref>2014年7月30日開放官方網站發表作品情報<ref>{{cite web |title=かんなぎれい新ブランド「mirai」が誕生 ―第1弾作品『ハナヒメ＊アブソリュート』公式サイト公開 |url=http://moedigi.com/game/2014/07/mirai-soft-hanahime/ |website=moedigi |language=ja |date=2014年7月30日}}</ref>，遊戲發售日期定為2016年5月27日。<ref>{{cite web |title=『ハナヒメ＊アブソリュート！』の発売日が5月27日に決定！初回特典・予約特典情報も公開 |url=http://moedigi.com/game/2015/12/hanahime-release-date/ |website=moedigi |language=ja |date=2015年12月11日}}</ref>後來由於負責開發戰鬥部分的外包公司進度延遲，導致發售日期延期至8月27日。<ref>{{cite web |title=重要なお知らせ |url=http://www.mirai-soft.net/info.php |website=mirai |language=ja |date=2016年5月18日}}</ref>另外原本是配角的鳳杏子由於在人氣投票中獲得第一名，官方在2016年8月4日公開追加製作她的路線劇情<ref>{{cite web |title=クルくるスタッフ完全新作『ハナヒメ＊アブソリュート！』がマスターアップ。鳳杏子ルート追加決定 |url=http://moedigi.com/game/2016/08/hanahime-master-up/ |website=moedigi |language=ja |date=2016年8月4日}}</ref>。

==評價==
