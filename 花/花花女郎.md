《'''花花女郎'''》（{{lang-en|'''''Playgirl'''''}}）是一本美國雜誌，內容介紹一般的有趣文章、生活風格和名人新聞，另加半裸和全裸的男人。在1970年代和1980年代，本雜誌以月刊出版，以[[女性|女性]]為主要市場，不過在很少有[[男同性戀者|男同性戀]]色情雜誌的時期，卻擁有一個重要的男同性戀讀者群。<ref>{{Cite web |url = http://www.esquire.com/entertainment/a55592/playgirl-magazine-history/?src=nl&mag=esq&list=nl_enl_news&date=062517 |title = A PENIS ON EVERY PAGE: THE RISE AND FALL OF PLAYGIRL |author = Matthew Rettenmund |date = 2017-06-25 |publisher = Esquire.com |language = en}}</ref>

該雜誌由Douglas Lambert在[[女權主義|女權主義]]運動的高峰時期為了對色情男性雜誌如《[[花花公子|花花公子]]》和《[[閣樓_(雜誌)|閣樓]]》作回應而於1973年創立的。1977年，Lambert將《花花女郎》賣給了接任出版人的Ira Ritter。該雜誌涵蓋了流產、平等權利和夾雜男人的性感照片等問題，並為婦女的[[性革命|性革命]]發揮了關鍵作用。<ref>{{Cite web |url = http://www.nytimes.com/2008/11/16/fashion/16playgirl.html?_r=1&scp=1&sq=playgirl&st=cse |title = They Couldn't Get Past the ‘Mimbos’ |author = Cara BuckleyUCKLEY |date = 2008-11-14 |publisher = The New York Times |language = en}}</ref>從2009年3月至2010年2月，《花花女郎》只出現線上版。該雜誌自2010年3月號開始有時回歸為印刷書。最後一期印刷是2016年冬季。截至2016年，該雜誌據信只有約3000位訂閱者。<ref>{{Cite web |url = http://splinternews.com/how-did-playgirl-magazine-go-from-feminist-force-to-fla-1793856910 |title = How did 'Playgirl' magazine go from feminist force to flaccid failure? |author = Cleo Stiller |date = 2016-05-19 |publisher = fusion.net |language = en}}</ref>
