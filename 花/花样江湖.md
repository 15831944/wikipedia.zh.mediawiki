《'''花样江湖'''》，古装剧之一，导演高飞，主演沈梦辰、大张伟、M.I.C男团。<ref>{{cite web |url= http://tech.huanqiu.com/news/2014-08/5099267.html|title=芒果TV打造马栏山出品《花样江湖》 |accessdate=2014 |publisher=环球网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://yule.sohu.com/20140728/n402808414.shtml |title= 综艺古装剧《花样江湖》开机 大张伟沈梦辰加盟|accessdate=2014 |publisher=搜狐网 |work= |language=zh-hans}}</ref>湖南卫视将该剧定位为“综艺古装剧”和“花样美少年剧”。<ref>{{cite web|url=http://news.xinhuanet.com/ent/2014-08/21/c_126900028.htm|title=芒果TV首部定制剧《花样江湖》顺利杀青|accessdate=2014|publisher=新华网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20141006102346/http://news.xinhuanet.com/ent/2014-08/21/c_126900028.htm|archivedate=2014-10-06}}</ref><ref>{{cite web |url=http://ent.ifeng.com/a/20140821/40257632_0.shtml |title= 芒果TV首部定制剧《花样江湖》 顺利杀青|accessdate= 2014|publisher=凤凰网 |work= |language=zh-hans}}</ref>2015年1月28日在芒果TV網站中播出。

==剧情==
该剧讲述一个即将因缺少顾客而关门的酒店，“司马习习”、“钱小宝”、“苏玉凤”、“柳如夏”等到来之后，生意盈门，故事连篇。
