{{日本作品|花咲Workspring！|花咲ワークスプリング!}}是[[SAGA_PLANETS|SAGA PLANETS]]在2015年3月27日<ref>{{Cite web|url=https://www.game-style.jp/special/backnumber/201502/26/00sak_sp.php|title=Game-Style「花咲Workspring！」特集|publisher=Game-Style|date=2015-02-26|accessdate=2018-04-20|language=ja}}</ref>發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]。2017年4月27日由Entergram發售[[PlayStation_Vita|PlayStation Vita]]版本<ref>{{Cite web|url=http://www.4gamer.net/games/369/G036915/20170119050/|script-title=ja:PS Vita用恋愛アドベンチャー「花咲ワークスプリング！」が4月27日に発売決定。「幽霊部」の部長とヒロイン達の恋模様が描かれる|publisher=4Gamer|date=2017-01-19|accessdate=2018-04-20|language=ja}}</ref>。2017年7月7日由{{lang|ja|萌えAPP}}發售[[Android|Android]]和[[iOS|iOS]]版<ref>{{Cite web|url=https://www.gamer.ne.jp/news/201707070079/|script-title=ja:一癖も二癖もある少年少女たちの青春物語！「花咲ワークスプリング！」がiOS/Android向けに配信開始|publisher=Gamer|date=2017-07-07|accessdate=2018-04-20|language=ja}}</ref>。

== 故事簡介 ==
一直過著懶散生活的花咲遊真，某天在學校發現了「幽靈社」的社團教室，並被學姐玖音彩乃任命為社長。之後遊真為了不讓人數不足的幽靈社被廢社而開始尋找社員<ref>{{Cite web|url=https://www.game-style.jp/soft/201502/19/01sak_sp.php|script-title=ja:変わり者ばかりの“幽霊部員”たちと、くだらなくもかけがえのない青春の日々を謳歌しよう！ 『花咲ワークスプリング！』3月27日発売！|publisher=Game-Style|date=2015-02-19|accessdate=2018-04-20|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|花咲遊真|花咲 遊真（はなさき ゆうま）}}
: 作品男主角，鈴丘學園二年級生。一直都過著懶散生活。
; {{nihongo|不知火祈|不知火 祈（しらぬい いのり）|聲：[[遙空|遙空]]<ref name="Character">{{cite web|url=http://sagaplanets.product.co.jp/works/hanasaki/character.html|title=Character|publisher=[[SAGA_PLANETS|SAGA PLANETS]]|accessdate=2018年6月16日|language=ja}}</ref>}}
: 鈴丘學園二年級女學生，目前正在體驗入學，但幾乎沒有在教室上課。個性冷淡、言語刻薄，因而有「鈴丘的雪女」之稱。
; {{nihongo|玖音彩乃|玖音 彩乃（くおん あやの）|聲：{{lang|ja|みとことみ}}<ref name="Character"/>}}
: 鈴丘學園三年級女學生，是學園中留級最多年的人。小時候就被稱為天才，考試都名列前茅，在校內相當有名。
; {{nihongo|琴吹光|琴吹 ヒカリ（ことぶき ひかり）|聲：{{lang|ja|[[遠野そよぎ|遠野そよぎ]]}}<ref name="Character"/>}}
: 鈴丘學園一年級女學生，遊真的學妹，對遊真一見鍾情，告白後被遊真拒絕，但仍對遊真抱持好感，知道遊真加入幽靈社後積極參加幽靈社的活動。
: 擅長各種家事，和遊真的妹妹乃乃香是上同一個料理教室的好友。
; {{nihongo|空森若葉|空森 若葉（そらもり わかば）|聲：[[澤澤砂羽|澤澤砂羽]]<ref name="Character"/>}}
: 鈴丘學園二年級女學生，遊真的同班同學，與由真和翔太是好友。分類委員會的成員，一直想要遊真加入分類委員會。

=== 其他角色 ===
; {{nihongo|上月柑南|上月 柑南（こうつき かんな）|聲：[[海原艾利娜|海原艾利娜]]<ref name="Character"/>}}
: 可攻略角色，鈴丘學園二年級女學生，遊真的同班同學。在若葉家工作的女僕，並擔任若葉的隨從。
; {{nihongo|花咲乃乃香|花咲 ののか（はなさき ののか）|聲：{{lang|ja|塚本リタ}}<ref name="Character"/>}}
: 可攻略角色，遊真的妹妹，與遊真並沒有血緣關係。喜歡照顧人且擅長家事，經常被學校的男生告白但都拒絕。
; {{nihongo|山下久美|山下 久美（やました くみ）|聲：[[松田理沙|松田理沙]]<ref name="Character"/>}}
: 遊真的班導師，因為身材嬌小經常被認成學生，擅長飲酒。
; {{nihongo|芥川翔太|芥川 翔太（あくたがわ しょうた）|聲：銀二郎<ref name="Character"/>}}
: 鈴丘學園二年級男學生，遊真的同班同學與好友。經常被女同學告白，但本人因為喜歡成熟女性而全部拒絕。
; {{nihongo|權田橋晉作|権田橋 晋作（ごんたばし しんさく）|聲：{{lang|ja|梅咲チャーリー}}<ref name="Character"/>}}
: 分類委員會的委員長、文武雙全且有禮貌的優等生。
; {{nihongo|柚木真子|柚木 真子（ゆずき まこ）|聲：{{lang|ja|百瀨ぽこ}}}}
: 鈴丘學園二年級女學生，遊真的同班同學，在多個社團擔任幫手。實際上是分類委員會的委員，負責秘密調查各個社團的不正當行為。
