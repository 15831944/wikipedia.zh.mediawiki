{{Vgname|花与乙女的祝福|ja=花と乙女に祝福を}}是[[WillPlus|WillPlus]]旗下的男性向游戏品牌ensemble在2009年5月29日发售的第1部[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=http://www.getchu.com/soft.phtml?id=650905|title=花と乙女に祝福を 通常版|publisher=Getchu.com|accessdate=2015年11月6日|language=ja}}</ref>。2011年2月18日由[[未来数位|未来数位]]发售[[正体中文|正体中文]]版。续作《花與乙女的祝福 ~皇家花束~》（{{lang|ja|花と乙女に祝福を ロイヤルブーケ}}）在2010年1月29日发售<ref>{{cite web|url=http://www.getchu.com/soft.phtml?id=669830|title=花と乙女に祝福を ロイヤルブーケ 通常版|publisher=Getchu.com|accessdate=2015年11月6日|language=ja}}</ref>，2012年2月6日發售正體中文版。

2010年7月8日由[[Alchemist|Alchemist]]发售PS2移植版《{{lang|ja|花と乙女に祝福を 〜春風の贈り物〜}}》，PS2版增加了「獅堂沙織」「朝霧七緒」两个角色。<ref>{{cite web |url=http://www.amazon.co.jp/dp/B003E37Z5I/|title=花と乙女に祝福を ~春風の贈り物~(通常版)|publisher=[[Amazon.co.jp|Amazon.co.jp]]|accessdate=2015年11月6日|language=ja}}</ref>2011年10月27日发售PSP移植版《{{lang|ja|花と乙女に祝福を 春風の贈り物 portable}}》。<ref>{{cite web |url=http://www.amazon.co.jp/dp/B0050MLQK6/|title=花と乙女に祝福を 春風の贈り物 portable (初回限定版)|publisher=[[Amazon.co.jp|Amazon.co.jp]]|accessdate=2015年11月6日|language=ja}}</ref>

== 简介 ==
主人公月丘彰为了让病弱的妹妹不会留级，决定代替她去圣露比娜丝学园上学。
