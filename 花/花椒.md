'''花椒'''，又称'''秦椒'''、'''川椒'''或'''山椒'''，指的是[[芸香科|芸香科]]'''花椒属'''（[[学名|学名]]：''{{lang|la|Zanthoxylum}}''）的[[灌木|灌木]]或[[乔木|乔木]]以及其果实。《[[神农本草经|神农本草经]]》中记载：“始产于秦。”

花椒形状球形，椒皮外表红楬色，曬乾後呈黑色。有龜裂纹，顶端开裂。内含种子一粒，圆形，有光泽。花椒含有[[檸檬烯|檸檬烯]]、[[香葉醇|香葉醇]]、[[異茴香醚|異茴香醚]]、[[花椒油烯|花椒油烯]]、[[水芹香烯|水芹香烯]]、[[香草醇|香草醇]]等等挥发性物质。具有独特浓烈香气。

花椒按大小分为'''大椒'''，（大椒又称大红袍、狮子头），其果粒大，色艳红或紫红，内皮呈淡黄色。'''小椒'''（小椒又称小黄金，色红，粒小，味麻，香味次于大椒。），按采收季节又分为'''秋椒'''和'''伏椒'''。

== 形态特征 ==
[[File:Zanthoxylum_acanthopodium_var._timbor_-_Kunming_Botanical_Garden_-_DSC03185.JPG|thumb]]的''[[Zanthoxylum_acanthopodium|Zanthoxylum acanthopodium]]'' var. ''timbor'']]
[[File:Zanthoxylum_americanum.jpg|thumb]]''的果實]]
[[File:Zanthoxylum_rhetsa_bark.JPG|thumb]]''的莖]]
花椒树属落叶[[灌木|灌木]]，高3-7m，茎干通常有增大皮刺；枝灰色或褐灰色，有细小的皮孔及略斜向上生的皮刺；当年生小枝被短柔毛。奇数羽状复叶，叶轴边缘有狭翅；小叶5-11个，纸质，卵形或卵状长圆形，无柄或近无柄，长1.5-7cm，宽1-3cm，先端尖或微凹，基部近圆形，边缘有细锯齿，表面中脉基部两侧常被一簇褐色长柔毛，无针刺。聚伞圆锥花序顶生，花被片4-8个；雄花[[雄蕊|雄蕊]]5-7个，雌花[[心皮|心皮]]3-4个，稀6-7个，子房无柄。果球形，通常2-3个，红色或紫红色，密生疣状凸起的油点。花期3-5月，果期7-9月。喜光，适宜温暖湿润及土层深厚肥沃壤土、沙壤土，萌蘖性强，耐寒，耐旱，抗病能力强。不耐涝，短期积水可致死亡。

== 功效 ==
中醫認為花椒果性味辛、温，有小毒，可使血管扩张，从而起到降低[[血压|血压]]的作用<ref>[[孙思邈|孙思邈]]的《千金食治》中说：“蜀椒，味辛、大热，有毒。主邪气，温中下气，留饮宿”。</ref>；服花椒水能去除[[寄生虫|寄生虫]]；去腥、芳香健胃、温中散寒、除湿止痛、杀虫解毒、促進食欲的功效<ref>《[[本草纲目|本草纲目]]》中记载：“治上气，咳嗽吐逆疝瘕，风湿寒痹。下气杀虫，利五脏，去老血。”《中华人民共和国药典》中规範花椒具有“温中止痛，杀虫止痒”的药效。</ref>。

== 用途 ==
花椒在[[中药|中药]]上也有一定的作用。作[[中醫|中醫]][[食療|食療]]，亦可作防护刺篱。花椒果皮可作为[[调味料|调味料]]，并可提取芳香油，又可入药，种子可食用，又可加工制作肥皂。作為调味料，花椒果及種子可除各种肉类的腥气；促进唾液分泌，增加食欲，《[[诗经|诗经]]·周颂》描述说道：“有飶其香，邦家之光。有椒其馨，胡考之宁”。花椒亦可浸制酒，[[李嘉佑|李嘉佑]]有《夜闻江南人家赛神》诗：“雨过风清洲渚闲，椒浆醉尽迎神还”，[[崔寔|崔寔]]《四民·月令》记载：“过腊一日，谓之小岁，释贺君亲，进椒酒，椒是玉衡星精，服之能老”。

中国[[川菜|川菜]]的传统烹饪方法中，大量使用花椒。随着川菜的流行，花椒的种植面积与产量也大幅度提高。甚至花椒的英语名称是Sichuan Peppercorn或Sichuanese Peppercorn（即“四川椒子”）。中国北方传统的家常调料中，花椒与[[八角|大料]]是必备的炖肉料。

花椒用途廣泛，果皮可作為調味料，並可提取芳香精油，又可入藥，種子可食用，也能加工製作肥皂。最常使用於調味與入藥。

== 文化 ==
花椒树结实累累，是子孫繁衍的象徵，故《[[詩經|詩經]]·唐风》稱：“椒聊之實，藩衍盈升。”<ref>[[闻一多|闻一多]]：《风诗类钞》</ref>。又[[班固|班固]]《西都賦》載“后宮則有掖庭椒房，后妃之室”，意思是皇帝的妻妾用花椒泥涂墙壁，謂之[[椒房|椒房]]<ref>《漢書·官仪》记载，“皇后以椒涂壁和椒房，取其温也。”《汉书·车千秋传》：“ 江充先治甘泉宫人，转至未央椒房。” [[颜师古|颜师古]]注：“ 椒房，殿名，皇后所居也。”[[李善|李善]]注引《三辅黄图》：“长乐宫有椒房殿。”《[[北史|北史]]·周纪下·高祖武帝》：“椒房丹地，有众如云，本由嗜欲之情，非关风化之义。”[[张居正|张居正]]《论外戚封爵疏》：“今使椒房之属与有大勋劳之人并享茅土，非所以昭有功、劝有德也。”</ref>，希望皇子們能像花椒树一样旺盛。
