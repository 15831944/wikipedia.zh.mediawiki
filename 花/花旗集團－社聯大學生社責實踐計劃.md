'''花旗集團－社聯大學生社責實踐計劃'''（{{lang-en|'''Citi-HKCSS Community Intern Program，CIP'''}}）是[[花旗集團|花旗集團]]和[[香港社會服務聯會|香港社會服務聯會]]合辦<ref>{{cite web |language=zh-hk |url=https://skypost.ulifestyle.com.hk/article/2103015/%E8%8A%B1%E6%97%97%E4%BC%8D%E7%87%95%E5%84%80+%E6%96%99%E6%B8%AF%E9%8A%80%E4%B8%8B%E5%8D%8A%E5%B9%B4%E6%88%96%E5%8A%A0%E6%81%AF | title=花旗伍燕儀 料港銀下半年或加息 |publisher=[[晴報|晴報]] |date=2018-06-27 |accessdate=2018-07-02}}</ref>，專為[[香港|香港]]就讀大學的工商管理學生，在非牟利機構透過實習發揮所長，並加強學生對企業社會責任的意識，培育成為更關心社會的商業人才。計劃由2000年開始實行，每名實習生需進行為期4至8星期實習，合共160小時，負責籌款、品牌建立、營運及社區服務等工作<ref>{{cite web |language=zh-hk |url=https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/51895/%E5%95%86%E7%A7%91%E7%94%9F%E9%81%B8%E6%93%87%E7%A4%BE%E4%BC%81%E5%AF%A6%E7%BF%92-%E5%AD%B8%E7%BF%92%E5%85%B1%E4%BA%AB%E7%B6%93%E6%BF%9F | title=商科生選擇社企實習　學習共享經濟 |publisher=[[香港01|香港01]] |date=2016-11-01 |accessdate=2018-07-02}}</ref><ref>{{cite web |language=zh-hk |url=https://paper.hket.com/article/1440234/T%E6%81%A4%E5%88%86%E4%BA%AB%E6%9B%B4%E7%94%9F%E4%BA%BA%E5%A3%AB%E5%BF%83%E8%B7%AF%E6%AD%B7%E7%A8%8B%20%E4%B8%AD%E5%A4%A7%E7%94%9F%E6%9C%9B%E6%8E%A8%E5%8B%95%E7%A4%BE%E6%9C%83%E5%85%B1%E8%9E%8D | title=T恤分享更生人士心路歷程 中大生望推動社會共融 |publisher=[[香港經濟日報|香港經濟日報]] |date=2016-06-07 |accessdate=2018-07-02}}</ref><ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2013/05/04/AY1305040006.htm | title=花旗夥社聯培訓60大學生 |publisher=[[文匯報|文匯報]] |date=2013-05-04 |accessdate=2018-07-02}}</ref><ref>{{cite web |language=zh-hk |url=https://skypost.ulifestyle.com.hk/article/1940969/%E8%8A%B1%E6%97%97%E6%8E%A8%E5%8B%95%E7%A4%BE%E6%9C%83%E6%9C%8D%E5%8B%99%2079%E5%A4%A7%E5%AD%B8%E7%94%9F%E7%A4%BE%E4%BC%81%E5%AF%A6%E7%BF%92 | title=花旗推動社會服務 79大學生社企實習 |publisher=[[晴報|晴報]] |date=2017-11-06 |accessdate=2018-07-02}}</ref>。

== 計劃目標 == 
* 提升香港就讀大學工商管理學生對社區的認識<ref>{{cite web |language=zh-hk |url=http://orientaldaily.on.cc/cnt/finance/20141008/00271_002.html | title=絲絲講場：花旗幫學生搵工加分 |publisher=[[東方日報_(香港)|東方日報 (香港)]] |date=2014-10-08 |accessdate=2018-07-02}}</ref>
* 為學生提供實習機會，讓他們將課堂所學，應用於協助非牟利機構，以實踐機構的目標<ref>{{cite web |language=zh-hk |url=https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/203505/%E7%A4%BE%E8%81%AF%E5%A4%A7%E5%AD%B8%E7%94%9F%E7%A4%BE%E8%B2%AC%E5%AF%A6%E8%B8%90%E8%A8%88%E5%8A%83-%E5%9F%B9%E9%A4%8A%E6%9C%AC%E5%9C%B0%E5%A4%A7%E5%AD%B8%E7%94%9F%E7%A4%BE%E6%9C%83%E4%BC%81%E6%A5%AD%E8%B2%AC%E4%BB%BB | title=社聯大學生社責實踐計劃　培養本地大學生社會企業責任 |publisher=[[香港01|香港01]] |date=2018-06-26 |accessdate=2018-07-02}}</ref>
* 為非牟利機構及未來工商管理人才建立聯繫，擴闊學生以非牟利服務界作為就業取向的視野

== 實習情況 ==
在2018年中，獲得「超凡卓越大獎」的樹仁大學企業管治的學生黃汶靜<ref>{{cite web |language=zh-hk |url=https://www.am730.com.hk/news/%E6%96%B0%E8%81%9E/%E5%A4%A7%E5%AD%B8%E7%94%9F%E7%A4%BE%E8%B2%AC%E5%AF%A6%E8%B8%90%E8%A8%88%E5%8A%83-%E8%88%89%E8%A1%8C%E5%98%89%E8%A8%B1%E7%A6%AE-102346 | title=大學生社責實踐計劃 舉行嘉許禮 |publisher=[[am730|am730]] |date=2017-11-03 |accessdate=2018-07-02}}</ref>，當時她被派往聾人機構「[[龍耳|龍耳]]」，負責製作手語電視新聞，其間不時要與聾人合作<ref>{{cite web |language=zh-hk |url=https://news.mingpao.com/pns/dailynews/web_tc/article/20180627/s00011/1530037928276 | title=社聯邀商科生社企實習 |publisher=[[明報|明報]] |date=2018-06-27 |accessdate=2018-07-02}}</ref>。她指出和聾人的溝通，最重要的是自己踏出第一步，不要害怕跟他們接觸。她表示，除了用手語溝通，文字也是一種方法，雙方可以互相遷就。該機構中心主任馮福平表示，實習生在機構可以以自身知識幫助處理日常工作，更能培養出關懷社會的年輕人，對社會有深遠影響<ref>{{cite web |language=zh-hk |url=https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/203660/%E5%95%86%E7%A7%91%E7%94%9F%E6%A3%84%E5%AF%AB%E5%AD%97%E6%A8%93%E5%AF%A6%E7%BF%92%E5%88%B0%E7%A4%BE%E4%BC%81%E5%8A%A9%E8%81%BE%E4%BA%BA-%E8%BA%AB%E5%BF%83%E5%8A%9B%E8%A1%8C%E6%8E%A5%E8%A7%B8%E5%BC%B1%E5%8B%A2%E7%A4%BE%E7%BE%A4 | title=商科生棄寫字樓實習到社企助聾人　身心力行接觸弱勢社群 |publisher=[[香港01|香港01]] |date=2018-06-26 |accessdate=2018-07-02}}</ref>。
