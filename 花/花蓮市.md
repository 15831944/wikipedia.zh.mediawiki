'''花蓮市'''（（{{lang-nan-tw|Hua-lian-chhī}}、[[撒奇萊雅語|撒奇萊雅語]]：'''Nabakuwan'''；[[阿美語|阿美語]]：'''Kalinko'''），位於[[臺灣|臺灣]][[花蓮縣|花蓮縣]]東部北段，北接[[新城鄉_(台灣)|新城鄉]]，南連[[吉安鄉|吉安鄉]]，西鄰[[秀林鄉|秀林鄉]]，東臨[[太平洋|太平洋]]（[[菲律賓海|菲律賓海]]），為全國人口第二密集的[[縣轄市|縣轄市]]（次於新竹縣[[竹北市|竹北市]]）<ref>{{cite web|url=http://www.ris.gov.tw/zh_TW/346|title=人口統計資料|publisher=內政部戶政司|date=2016-07-11}}</ref>。值得一提的是，花蓮市和[[新城鄉_(台灣)|新城鄉]]面積經四捨五入取小數點後四位後仍完全一樣(29.4095km<sup>2</sup>)，兩者同列為全花蓮縣最小的鄉鎮，與[[宜蘭市|宜蘭市]]之面積四捨五入取第2位亦同。

同時亦為花蓮縣政府之[[縣治|所在地]]，也是本縣及[[東臺灣|東台灣]]的[[政治|政治]]、[[經濟|經濟]]、[[交通|交通]]、[[文化|文化]]中心之一。管理花蓮、臺東的[[行政院東部聯合服務中心|行政院東部聯合服務中心]]亦設在本市。

== 歷史 ==
[[File:Karenko_1_7500_1945.jpg|thumb]]繪製的花蓮市地圖]]
花蓮古稱「奇萊」，是阿美族稱其為「Sakiraya」取「kiray」的音譯<ref>撒奇萊雅族（Sakizaya）--花蓮奇萊平原的原住民[https://www.apc.gov.tw/portal/getfile?source=79ADDDD9195DB0E52610217BBF0B058FA9DAB2A97BBE1DD069833B4B7E17A57DB13762A3A354BE8F15E5012E113EEE51FD8F7D1A5BBDAB3202D91740DAA23761&filename=A277D91567D8E9414EFD9B133971C1F36D525A284ED61971CB90CB17197FE07B8E388C4D6B0E5673D740DD5412B95526E7DCFF54540F2661D0636733C6861689]頁2，2018-02-21查詢.原住民族委員會</ref>，1909年花蓮地區脫離[[臺東廳|台東]]獨立設治。本市前身為'''鯉浪港'''，位於[[美崙溪|美崙溪]]口。亦稱'''新港'''，此名係與[[花蓮溪|花蓮溪]]口的花蓮港舊港（南濱仔）相對。

1920年台灣地方改制，成立'''[[花蓮港街|花蓮港街]]'''，隸屬[[花蓮港廳|花蓮港廳]]花蓮港支廳管轄，1937年東部實施郡制，改隸於花蓮港廳[[花蓮郡|花蓮郡]]，1940年實施市制，改為'''[[花蓮港市|花蓮港市]]'''，戰後，1946年改為'''花蓮市'''至今。

== 地理 ==
;水文
[[美崙溪|美崙溪]]舊稱「砂婆礑」，流程全長19.58公里，集水區面積72.54平方公里，流貫花蓮市，水源來自沙婆礑山，位於花蓮縣[[秀林鄉|秀林鄉]]水源村，海拔1,120公尺，花蓮市為美崙溪沖積扇平原，其[[沖積扇|沖積扇]]扇端逼近[[吉安鄉|吉安]]、花蓮市、嘉里一線，美崙溪最後於[[花蓮港|花蓮港]]南側注入[[太平洋|太平洋]]<ref>{{cite web|language=zh-tw|url=http://www.hualien.gov.tw/project/informationinfo.aspx?v=1E8A23C624915FE8|title=美崙溪流域簡介|author=|publisher=花蓮市公所|date=2013-08-20|accessdate=2014-07-03|deadurl=yes|archiveurl=https://web.archive.org/web/20140714222219/http://www.hualien.gov.tw/project/informationinfo.aspx?v=1E8A23C624915FE8|archivedate=2014-07-14}}</ref>。

== 語言 ==
花蓮市通行[[中華民國國語|中華民國國語]]、[[台語|台語]]、[[台灣客家語|台灣客家語]]，原住民族的通行語為[[阿美語|阿美語]]、[[德路固語|太魯閣語]]、[[撒奇萊雅語|撒奇萊雅語]]<ref>[http://news.ltn.com.tw/news/politics/breakingnews/2220153 55公所 原民「地方通行語」公告了]</ref>。
== 經濟 ==
;商業
花蓮市區最熱鬧的地段莫過於中正路、中華路、中山路的「金三角商圈」，另結合週遭的溝仔尾（最早發跡的商業中心地）、一心街、光復街、大禹街、舊鐵道商圈而形成的商業區域。知名連鎖速食餐廳、飲料店與咖啡廳如[[麥當勞|麥當勞]]、[[肯德基|肯德基]]、[[達美樂|達美樂]]、[[必勝客|必勝客]]、[[摩斯漢堡|摩斯漢堡]]、[[SUBWAY|SUBWAY]]、[[拿坡里|拿坡里]]、[[胖老爹|胖老爹]]、[[星巴克|星巴克]]、[[怡客咖啡|怡客咖啡]]、[[争鮮|争鮮]]、[[三商巧福|三商巧福]]、[[清心福全|清心福全]]、[[85度C|85度C]]、[[壹番屋|壹番屋]]、[[八方雲集|八方雲集]]等均設有營業據點，[[遠東百貨|遠東百貨]]新館於2009開業，位於和平路及林森路旁，為東台灣最大、品牌最齊全的百貨公司，內部設有[[愛買|愛買]]花蓮店、[[無印良品|無印良品]]、[[UNIQLO|UNIQLO]]、[[統一多拿滋|Mister Donut]]、[[義式屋古拉爵|義式屋古拉爵]]、[[H&M|H&M]]、[[誠品生活|誠品生活]]等專櫃。而中大型專門店有在林森路的[[B&Q|特力屋]]，亦有[[全國電子|全國電子]]、[[燦坤|燦坤3C]]、[[屈臣氏|屈臣氏]]、[[寶雅生活館|寶雅生活館]]、[[康是美|康是美]]、[[全聯福利中心|全聯福利中心]]、[[光南大批發|光南大批發]]、[[義美食品|義美食品]]、生活百貨世界、政大書城、[[九乘九文具專家|九乘九文具專家]]、商務旅館等集中在市區周遭與這三條路段上。另外服飾店、流行飾品店、運動用品店、連鎖服飾店、書店、銀行、名產店等都聚集於「三中地段」週遭，而[[好樂迪|好樂迪KTV]]、[[秀泰影城|秀泰影城]]、商務飯店、汽車旅館、電動遊樂場、World Gym健身工廠、大魯閣保齡球館則聚集在火車站附近的國聯商圈。無疑為東部商業最繁盛的城市，也是東部地區居民逛街購物的主要地點之一，商況尚稱熱絡。

花蓮市與[[沖繩縣|沖繩縣]][[八重山群島|八重山群島]]的[[石垣市|石垣市]]、[[與那國町|與那國町]]和[[竹富町|竹富町]]簽署「台灣東部、沖繩八重山諸島觀光經濟圈」共同宣言，希望實現「國境交流特區」的構想。
