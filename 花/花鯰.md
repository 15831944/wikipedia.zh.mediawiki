'''花鯰'''（[[学名|学名]]：''Silurus asotus''）俗稱'''鯰'''、'''鯰魚'''、'''鯰仔'''、'''溪鯰'''、'''江鯰'''、'''黑龍江鯰'''，是[[鯰科|鯰科]][[鯰屬|鯰屬]]的一種魚類。花鯰就是常說的'''鯰魚'''，但鯰魚這個名稱也可以通稱所有[[鮎形目|鮎形目]]魚類。蓋因中國古代將現今的鯰魚稱作[[鮎形目|鮎]]，日本古代將[[香魚|香魚]]也稱作「鮎」，因此日本創造出'''鯰'''這個字來代表花鯰<ref>{{Cite news|url=http://blog.xuite.net/ban.ryu/tureduregusa/63072222-%E9%AE%8E%E3%80%81%E9%AF%B0|title=鮎、鯰|last=(ban.ryu)|first=ban|newspaper=隨意窩 Xuite日誌|accessdate=2018-08-16}}</ref><ref>{{Cite web|url=http://www.books.com.tw/products/0010554118|title=你不可不知的日本飲食史|accessdate=2018-08-16|author=|date=|work=博客來|publisher=|pages=34}}</ref>。中国俗称的鲶鱼有花鯰和[[大口鯰|大口鯰]]（南方鯰）。

== 身體特徵 ==
花鯰最大可以長到60公分，但一般多是以30-50之間的長度較為常見。花鯰因為身上佈滿著花紋，因此被稱為花鯰；跟其他[[鯰形目|鯰形目]]魚類一樣沒有鱗片，具黏膜，有2對用來感應的[[觸鬚|觸鬚]]，尾小呈截形，與腹鰭相連，背鰭退化成脂鰭狀。體色多變，從茶褐色到黑色均有。

== 生活習性 ==
花鯰棲息的環境多變，從水流平緩的沼澤、池塘、埤塘、湖泊、溝渠、[[水庫|水庫]]到湍急的溪流、江河，都有花鯰棲息。花鯰生性活躍，善於跳躍，多半在夜間活動，以[[魚類|魚類]]、[[蝦類|蝦類]]、[[昆蟲|昆蟲]]等活體為食，常守候在石縫或水草中，等獵物靠近後再发起突袭。

== 分佈 ==
花鯰分布于從[[中國|中國]]东北到[[越南|越南]][[紅河|紅河]]流域的广大地区及[[日本|日本]]、[[台灣|台灣]]。<ref>"P. asotus (Linnaeus, 1758) - амурский сом"[http://www.sevin.ru/vertebrates/index.html?Fishes/189.html],2011/07.{{ru icon}}</ref>
