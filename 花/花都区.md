'''花都区'''是[[中国|中国]][[广东省|广东省]][[广州市|广州市]]的一个[[市辖区|市辖区]]，位于广州市北部。区政府驻'''[[花城街道|花城街道]]'''。是[[广州市|广州市]]北上至[[粤北|粤北]]、[[湖南|湖南]]以至内地北方的最主要通道。

== 歷史 ==
花都区原称花县、花都市。汉朝属[[番禺|番禺]]管辖；隋朝属[[南海县|南海县]]辖；宋朝以后分属番禺、南海县辖。清康熙二十五年（1686年）建花县，属广州府。民国3年（1914年），花县属粤海道。民国9年，由广东省直辖。民国25年，归省第一行政督察区。抗战爆发后，隶西江行署。民国32年，复归第一行政督察区，次年改属省政府直属督察区。

1949年10月13日，花县解放，屬中共北江专区，同年12月隶属珠江专区。1952年和1956年先后改属粤北行政区和佛山专区。1958年11月花县改名广北县，次年3月复名花县。1960年4月20日划为广州市属县。1993年6月18日，中华人民共和国国务院批准撤县设市，定名花都市（县级），由广东省直辖，委托广州市代管，以原花县的行政区划为花都市的行政区划。8月1日，花都各机关单位更名并启用新印章。

2000年5月21日，中华人民共和国国务院批准撤销花都市，设立广州市花都区，将原花都市的行政区划为花都区的行政区划。7月10日，举行撤市设区挂牌仪式。同年12月25日，中共花都区委和花都区人民政府机关由新华街建设路1号迁至新都大道（今迎宾大道）的新办公楼办公。

== 地理 ==
花都区位于广东省中北部，[[珠三角|珠三角]]北缘位置，广州市北部。境内地势较为平坦，全境地势北高南低。境内有[[广花平原|广花平原]]，是珠江三角洲平原一部分，區西北面和東北面為[[丘陵|丘陵]]地带。市内重要饮用水源[[流溪河|流溪河]]干流由东至西流經區南部邊緣。东側与[[从化区|从化区]]相通，西側与[[佛山市|佛山市]][[三水区|三水区]]连接。南側紧邻[[白云区|白云区]]，北側与清远市[[清城区|清城区]]相连。

== 行政区划 ==
1993年撤县设市后，花都市有新华、花山、花东、狮岭、赤坭、炭步、北兴、梯面、芙蓉9个镇。下辖1个街道办事处，17个居民委员会，181个村民委员会，1746个村民小组。1994年，撤销花东镇可爱村，设立珠湖村、元岗村。1996年，增设新华镇培英、富华、宝华、凤华、金华5个居委会。1998年增设新华镇松园居委会。1999年8月19日，广东省人民政府批准成立花都市花侨镇（保留华侨农场牌子）。至2000年，花都市有10个镇，183个村委会，31个居委会，1个街道办事处，1911个村民小组。2002年9月26日起，[[白雲區|白雲區]]雅瑶镇划归花都区管辖。

2014年1月8日，花都區委區政府宣佈撤銷雅瑤鎮，拆分新華街道以及獅嶺鎮南部，將三鎮街轄區重新拆分改造為新華街道、新雅街道、秀全街道、花城街道，使花都行政区划正式从“一街七镇”变成“四街六镇”。

花都区现辖四个街道办事处和六个镇，其中区委、区政府所在地位於[[花城街道|花城街道]]。

== 交通 ==
=== 航空運輸 === 
[[广州白云国际机场|广州白云国际机场]]坐落在'''花都区'''[[新华街道|新雅街道]]、[[花山镇|花山镇]]、[[花东镇|花东镇]]和[[白云区|白云区]][[人和镇|人和镇]]交界处，首期工程年旅客输送量为2500万人次，货运吞吐量为100万吨，最大规模可达每年8000万人次客运和300万吨货运。

=== 鐵路運輸 ===
[[广州北站|广州北站]]位於花都區新華街道。[[京广铁路|京广铁路]]、[[武广客运专线|武广客运专线]]、[[廣清城際軌道交通|廣清城際軌道交通]]及[[广佛环线|广佛环线]]縱貫全區。

=== 道路 ===
[[106国道|106国道]]、[[107国道|107国道]]、114省道、118省道、381省道、[[京港澳高速公路|京港澳高速公路]]、[[广州机场高速公路|广州机场高速公路]]、[[广清高速公路|广清高速公路]]、[[大广高速公路|大广高速公路]]、[[广州绕城高速公路|广州绕城高速公路]]、[[珠三角环线高速公路|珠三角环线高速公路]]及[[广乐高速公路|广乐高速公路]]貫穿全區。

=== 公共交通及客運 ===
花都區內主要客運站包括花都客运站，以營運广州城区及區內各鎮的客运班车為主；花都汽车总站，以營運外围城市的客运班车及區內公交线路為主；广州北站汽车站，以營運珠三角一帶城市及白雲機場的客运班车為主。

公共交通線路方面，现时，区內有40多条公交线路行驶区内各镇街。2010年亚运后，花都区除梯面镇外已实现乡镇中心区与白云区、广州城区或人和地铁站的公交线路联系。而在亚运后，花都区进一步实现推行羊城通第16次六折的公交及老人和学生乘车优惠政策，令花都区更好融入广州市。

== 旅遊 ==
花都區內有省级的[[王子山森林公园|王子山森林公园]]、省级旅游度假胜地—[[洪秀全故居|洪秀全故居]]和纪念馆、芙蓉度假村、花山香草世界、玖珑湖欧洲小镇、资政大夫祠、盘古王庙、圆玄道观等主要景点。其中洪秀全故居是全国重点文物保护单位。
