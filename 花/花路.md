== 爭議 ==
2018年3月19日，據韓國媒體《亞洲經濟》報導顯示，該報導談及《花路》發行後的隔天，T.O.P目前服兵役中的[[首爾|首爾]]龍山廳便接到首爾地方兵務廳詢問，目前正以社會服務要員服役中的T.O.P是否違反兼職及營利活動的相關規定。<ref name="auto1">{{cite web |url=http://view.asiae.co.kr/news/view.htm?idxno=2018031910064869209 |title='군인신분' 빅뱅 탑 음원 발매 논란…용산구 '겸직금지 위반 여부 확인중' |language=ko |publisher=亞洲經濟 |date=2018-03-19 |accessdate=2018-03-21 |deadurl=no |archiveurl=https://web.archive.org/web/20180322081700/http://view.asiae.co.kr/news/view.htm?idxno=2018031910064869209 |archivedate=2018-03-22 }}</ref>

根據韓國兵務廳「社會服務要員服務管理條例」第28條，社會服務要員無法兼職，且T.O.P現行的軍人身分也不得進行營利活動<ref name="auto1"/>。對此，T.O.P表示，該曲是於2015年至2016年錄製完畢的歌曲，對於《花路》的音源會被YG娛樂發行一事並不知情<ref>{{cite web |url=http://hk.on.cc/hk/bkn/cnt/entertainment/20180319/bkn-20180319120132793-0319_00862_001.html |title=GD訓練時整親手腳？T.O.P出新歌疑違規 |language=zh |publisher=東網巨星 |date=2018-03-19 |accessdate=2018-03-21 |deadurl=no |archiveurl=https://web.archive.org/web/20180322081918/http://hk.on.cc/hk/bkn/cnt/entertainment/20180319/bkn-20180319120132793-0319_00862_001.html |archivedate=2018-03-22 }}</ref>，而YG娛樂事發後澄清「這首歌曲是在T.O.P入伍前錄製一事眾所皆知，且該事實早已向媒體公開告知大眾，無法理解為何會有此問題」，其經紀公司代表[[梁鉉錫|梁鉉錫]]則於個人[[社群網站|社群網站]][[Instagram|Instagram]]上傳一部影片並附言寫道「#BIGBANG#花路#FLOWERROAD#2016年12月13日#VLIVE#2年前錄製的歌曲#PERFECT_ALLKILL#時隔2年盛開的花路」，而該影片為2016年12月13日BIGBANG五名成員於V LIVE的直播片段。影片中成員們談及「未公開的歌曲，未來有機會的話會讓大家聽」，證實了《花路》是T.O.P服兵役前便已錄製完畢的歌曲。<ref>{{Cite web|url=https://idol001.com/news/bigbang/detail/5aaf84697a1173a35d8b549e/|title=[BigBang][新闻]180319 【官方立场】《花路》2年前已录音完毕 不知为何引争议——IDOL新闻|accessdate=2018-03-21|author=|date=|work=idol001.com|publisher=|language=zh-Hans|deadurl=no|archiveurl=https://web.archive.org/web/20180331141843/https://idol001.com/news/bigbang/detail/5aaf84697a1173a35d8b549e/|archivedate=2018-03-31}}</ref>

而首爾龍山廳的相關人士透露，已向T.O.P要求確認其所屬經紀公司YG娛樂的合約，並表示這是為了確認合約中的契約時間，以及音源的持有權相關內容。<ref name="auto1"/>

2018年4月5日，《{{tsl|en|No Cut News|No Cut News}}》獨家報導了一則新聞，該媒體表示首爾地方兵務廳已交接了相關事件，指出該名社會服務要員徵召前製作的音源不是兼職審批的對象，因為不做和音源相關的活動所以不能說是違反了禁止兼職的規定。而首爾龍山廳相關人員也回應道「首爾地方兵務廳在3月27日以回信告知他們，表示通過確認程序後T.O.P未違反禁止兼職的規定」，因此以T.O.P為首的所有BIGBANG成員擺脫了《花路》音源的爭議。<ref>{{cite web |url=http://www.nocutnews.co.kr/news/4950204 |title=[단독] "탑, 규정위반 아냐"…빅뱅 '꽃길' 음원 논란 종결 |publisher=No Cut News |date=2018-04-05 |accessdate=2018-04-06 }}</ref>
