'''花園町站'''（{{jpn|j=花園町駅|hg=はなぞのちょうえき|rm=Hanazonochō eki|f=y}}）是一個位於[[日本|日本]][[大阪府|大阪府]][[大阪市|大阪市]][[西成區|西成區]]旭一丁目，屬於[[大阪市高速電氣軌道|大阪市高速電氣軌道]]（大阪地下鐵）的[[鐵路車站|鐵路車站]]。車站編號是'''Y17'''。

== 歷史 ==
* 1942年（[[昭和|昭和]]17年）5月10日 - 3號線（現在的四橋線）大國町－此站間開通時作為終點站開業。
* 1956年（昭和31年）6月1日 - 3號線延伸至岸里，成為中間站。
* 1974年（昭和49年） - 設置連結上下線月台的聯絡通道。
* 2018年（平成30年）4月1日 - [[大阪市交通局|大阪市交通局]]的民營化，成為[[大阪市高速電氣軌道|大阪市高速電氣軌道]]（大阪地下鐵）的車站。

== 車站結構 ==
[[File:Hanazonocho-home.jpg|thumb]]
對向式2面2線月台的[[地底車站|地底車站]]。閘口、月台位於地下1樓。閘口在兩個月台的南北，合計設有4個，月台之間互相在中程設置聯絡通道。
