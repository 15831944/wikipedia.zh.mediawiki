'''花壇車站'''位於[[台灣|台灣]][[彰化縣|彰化縣]][[花壇鄉|花壇鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
花壇車站站內設有[[島式月台|島式]]、[[岸式月台|岸式月台]]各一座，共四條路線。四條路線中僅有三條設有月台，最靠近東北側站房的是供停放車輛用的側線，無月台設置。

== 營運狀況 ==
*花壇車站為臺鐵旗下的[[臺灣鐵路管理局車站等級#簡易站|乙種簡易站]]，僅有[[臺鐵區間車|區間車]]停靠。站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[icash|icash 2.0]]及[[HappyCash|HappyCash]]付費。

== 歷史 ==
*1905年3月26日：設「茄苳腳驛」。
*1920年10月1日：改為「花壇」。
*1971年4月30日：站房改建完成。
*1993年12月1日：由三等站降為甲種簡易站，並指定由[[彰化車站|彰化站]]管理。
*2015年6月30日：啟用多卡通刷卡機。
*2017年9月16日：由甲種簡易站調整為乙種簡易站。
