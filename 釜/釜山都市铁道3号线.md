'''釜山都市铁道3号线'''（{{kor|諺=부산 도시철도 3호선|漢=釜山 都市鐵道 3號線|羅=Busan Doshicheoldo 3 hoseon}}）是位于[[韩国|韩国]][[釜山|釜山]]的一条地铁线路，线路自[[水营区|水营区]][[水营站_(釜山交通公社)|水营站]]至[[江西区_(釜山)|江西区]][[大渚站|大渚站]]，全长18.3公里，属于[[釜山都市铁道|釜山都市铁道]]，由[[釜山交通公社|釜山交通公社]]經營。在1997年11月動工，于2005年11月28日全线开通运营。

== 路线资料 ==
*总长：18.3km
*轨间距离：1435mm
*站数：17
*复线区间：全线
*电化区间：全线（直流1500V接触网供电）
*闭塞方式：ATC
*最高时速：80km/h

== 历史 ==
*1997年11月 - 开始动工修建。
*2005年11月28日 - 全线开通。
