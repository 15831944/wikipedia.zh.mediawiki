'''釜石自動車道'''（{{jpn|j=釜石自動車道|hg=かまいしじどうしゃどう|rm=Kamaishi jidōshadō|f=y}}）是從[[岩手縣|岩手縣]][[花卷市|花卷市]][[花卷系統交流道|花卷系統交流道]]到岩手縣[[釜石市|釜石市]]釜石系統交流道的[[高速公路|高速公路]]。略稱為'''釜石道'''（{{lang|ja|かまいしどう、KAMAISHI EXPWY}}）。

高速公路編號則是與[[秋田自動車道|秋田自動車道]]（[[北上系統交流道|北上系統交流道]] - [[河邊系統交流道|河邊系統交流道]]間）被分配為{{Ja Exp Route Sign|E46}}<ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/list/index.html|title=高速道路ナンバリング一覧|accessdate=2017年2月26日|publisher=国土交通省}}</ref>。

== 交流道 ==
* 全區間皆在[[岩手縣|岩手縣]]。
* 全線未設有附加油站的服務區、停車區。
* 交流道編號欄的背景色為<span style="color: #BFB;">■</span>表示為已通車區間。 設施欄背景色{{Color|#CCC|■}}為未通車區間或未啟用設施，未通車區間之名稱為暫稱。
* 巴士停留所的記號○/●為運用中，◆為休息中的設施。無印者未設巴士站。

* 東和交流道 - 遠野交流道 : '''[[新直轄方式|新直轄方式]]區間'''
* 遠野交流道 - 遠野住田交流道 : '''[[國道283號|一般國道283號]] 遠野道路'''
* 遠野住田交流道 - 釜石西交流道 : '''[[國道283號|一般國道283號]][[仙人峠道路|仙人峠道路]]'''
* 釜石西交流道 - 釜石系統交流道 : '''[[國道283號|一般國道283號]] 釜石道路'''

== 歷史 ==
* [[2002年|2002年]][[11月7日|11月7日]] - 花卷系統交流道～東和交流道間（11.4 公里）通車<ref name="iwatenippo20021107_1">『東和 - 花巻間が開通 東北横断自動車道釜石花巻線 太平洋側に向け初』 - 岩手日報2002年11月7日夕刊 1面</ref>。
* [[2005年|2005年]][[10月1日|10月1日]] - 隨著[[日本道路公團|日本道路公團]][[民營化|民營化]]，由[[東日本高速公路|東日本高速道路株式會社]]繼承。
* [[2007年|2007年]][[3月18日|3月18日]] - 遠野住田交流道～釜石西交流道間（18.4 公里）通車<ref name="kamaishi20070313-1">{{Cite web |url= http://www.thr.mlit.go.jp/bumon/kisya/kisyah/13153_kisya_preview.html|title= 一般国道283号「仙人峠道路」3月18日全線開通|accessdate= 2015-08-31|date= 2007-02-02|publisher= 三陸国道事務所・岩手県県土整備部}}</ref>。
* 2007年[[6月1日|6月1日]] - 東和收費站廢除，移設花卷機場主線收費站<ref name="iwatenippo20070531_1">『花巻空港本線料金所を新設 自動車道・あすから』 - 岩手日報2007年5月31日朝刊 23面</ref>。
* [[2008年|2008年]][[3月16日|3月16日]] - 瀧觀洞交流道啟用<ref name="kamaishi20080316-1">{{Cite web |url= http://www.thr.mlit.go.jp/bumon/kisya/kisyah/17360_kisya_preview.html|title= 一般国道283号 仙人峠道路「滝観洞IC」3月16日開通！|accessdate= 2015-08-31|date= 2008-03-11|publisher= 三陸国道事務所}}</ref>。
* [[2010年|2010年]][[2月2日|2月2日]] - 花卷系統交流道～東和交流道間被指定為[[高速公路免費化|高速公路免費化]][[社會實驗|社會實驗]]的對象區間。
* [[2011年|2011年]][[11月21日|11月21日]] - 2011年第3次補正預算成立，遠野交流道～遠野住田交流道間、釜石西交流道～釜石系統交流道間新規事業化<ref name="kamaishi20111121-1" />。
* [[2012年|2012年]][[10月15日|10月15日]] - 遠野住田交流道～釜石西交流道間 速限由60 km/h提升至70 km/h<ref name="kamaishi20121015-1">{{Cite web |url= http://www.thr.mlit.go.jp/bumon/kisya/kisyah/images/43214_1.pdf|title= 三陸国道事務所 10月15日より仙人峠道路の規制速度が70km/hになります|accessdate= 2015-08-31|date= 2012-10-10|format= PDF|page = 1|publisher= 岩手県警察本部・三陸国道事務所}}</ref>。
* 2012年[[11月25日|11月25日]] - 東和交流道 ～- 宮守交流道間（23.7 公里）通車<ref name="kamaishi20121125-1">{{PDFlink|[http://www.thr.mlit.go.jp/bumon/kisya/kisyah/images/43350_1.pdf 東北横断自動車道釜石秋田線の宮守〜東和間が平成24年11月25日(日)に開通します]|1366 [[キビバイト|KiB]]}}. 岩手河川国道事務所 p 1. (2012年10月29日). 2015年8月31日閲覧。</ref>。
* [[2015年|2015年]][[12月5日|12月5日]] - 宮守交流道～遠野交流道間（9.0 公里）通車<ref>{{Cite web|url=http://www.thr.mlit.go.jp/bumon/kisya/kisyah/images/58670_1.pdf|title=復興支援道路【東北横断自動車道釜石秋田線】遠野〜宮守 12月5日(土)に開通します 〜遠野インターへのアクセス県道も同時に開通します〜|accessdate=2015-10-14|date=2015-10-14|publisher=国土交通省東北地方整備局 岩手河川国道事務所|format=PDF}}</ref>。
* [[2018年|2018年]]度 - 遠野交流道～遠野住田交流道間（11.0 公里）、釜石系統交流道～釜石西交流道間（6.0 公里）預定通車<ref name="tohoku2015" />。
