'''釜山文化广播'''（[[朝鲜语|朝鲜语]]：부산문화방송，[[朝鲜汉字|朝鲜汉字]]：{{lang|ko|釜山文化放送}}；[[英语|英语]]：Busan Munhwa Broadcasting Corporation）是位于[[韩国|韩国]][[釜山|釜山]]的[[文化广播_(韩国)|文化广播]]（MBC）的放送局，亦是[[文化广播_(韩国)|文化广播]]最初创建的位置。

== 所在地 ==
613-728[[釜山|釜山]][[水营区|水营区]]民乐洞316-2

== 沿革 ==
*1959年4月15日 - 以'''文化广播'''的名义开局，开始播音（呼号'''HLKU'''，功率1kW）。
*1965年3月5日 - 改为现在名称。
*1969年1月1日 - 功率增至20kW。
*1970年1月24日 - 电视节目开通（视频输出功率2kW，声音输出功率500W）。
*1970年4月15日 - FM广播开通（88.9MHz，1kW）。
*1972年3月1日 - FM广播实现立体声传送。
*1980年12月22日 - 开通彩色电视节目。
*1982年7月1日 - FM广播功率增加至5kW。
*1983年8月1日 - 电视输出功率增强（视频输出功率10kW，声音输出功率2kW）。
*1984年11月2日 - 与[[日本|日本]][[九州朝日放送|九州朝日放送]]结为友谊关系。
*1988年2月1日 - MBC标准FM开通。
*1994年5月21日 - 与[[中国|中国]][[天津电视台|天津电视台]]结为友谊关系。
*1997年3月31日 - 搬入新的办公室。
*2003年8月4日 - 与[[中国|中国]][[西安电视台|西安电视台]]结为友谊关系。
*2004年7月30日 - 开始播放数字电视节目。
*2007年8月31日 - 开始播放[[DMB|DMB]]。
*2012年10月9日 - 模拟电视停播。
