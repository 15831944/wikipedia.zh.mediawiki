}}'''繆浩昌'''（{{lang-pt|Mau Hou Cheong}}，{{bd|1980年|9月27日}}），藝名'''6號'''，生於[[澳門|澳門]]，香港歌手及樂隊成員，乃香港著名樂隊[[RubberBand|RubberBand]]之主音。除樂隊事業外，在近年亦嘗試身兼演員，並在[[ViuTV|ViuTV]]的電視劇中擔鋼演出。太太為樂隊填詞人[[Tim_Lui|Tim Lui]]。

==簡介==
繆浩昌出生於澳門，兒時居於澳門木屋區，7-8歲隨家人移民至香港<ref name="hk.o_住板間房">{{Cite web |title=住板間房大 6號@RB初到香港唔適應 |trans-title= |author= |work=on.cc東網 |date= |access-date=4 November 2018 |url= https://hk.on.cc/hk/bkn/cnt/entertainment/20180818/bkn-20180818055017671-0818_00862_001.html |language=zh |quote=}}</ref>，於中學就讀[[伊利沙伯中學|伊利沙伯中學]]，2002年在[[香港中文大學|香港中文大學]]新聞及傳播學系畢業，之後應邀[[香港電台|香港電台]]電視部助導，2004年與四位志同道合的隊友（包括阿正、阿偉、泥鯭和已於2011年離隊的藝琛）成立RubberBand，到2008年正式打入樂壇。憑藉溫柔卻有張力的聲線與同年出道的另一焦點樂隊[[Mr.|Mr.]]以強勁風格的主音[[布志綸|布志綸]]Alan鬥得難分難解，兩支樂壇新勢力的良性競爭令沉寂多時的香港樂隊文化得以復活，並在其後多年開始壟斷樂壇樂隊獎項。

2012年，6號與拍拖超過10年的女友兼RubberBand御用填詞人[[Tim_Lui|Tim Lui]]結婚<ref name="hk.e_「因為話">{{Cite web |title=「因為話今年世界末日」RubberBand主音趕結婚 |author= |work=Apple Daily 蘋果日報 |date= |access-date=24 September 2018 |url= https://hk.entertainment.appledaily.com/entertainment/daily/article/20121112/18064154 |language= |quote=}}</ref>。

2016年，RubberBand為以更大自由度發展成為獨立樂隊，而繆浩昌亦獲得ViuTV的青睞，獲得主演電視劇的機會。首先在短劇《車房·耶穌·師傅仔》中便擔正任男主角，其後亦以客串身份首次參演由老婆Tim Lui執導的電視劇《[[未來還未來|未來還未來]]》<ref name="hk01_【未來還">{{Cite web |title=【未來還未來】RB 主音6號孖老婆執導新劇：係好機會迫自己去試 |trans-title= |author=鄺鈺瑩 |work=香港01 |date= |access-date=24 September 2018 |url= https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/119755/%E6%9C%AA%E4%BE%86%E9%82%84%E6%9C%AA%E4%BE%86-rb-%E4%B8%BB%E9%9F%B36%E8%99%9F%E5%AD%96%E8%80%81%E5%A9%86%E5%9F%B7%E5%B0%8E%E6%96%B0%E5%8A%87-%E4%BF%82%E5%A5%BD%E6%A9%9F%E6%9C%83%E8%BF%AB%E8%87%AA%E5%B7%B1%E5%8E%BB%E8%A9%A6 |language=zh |quote=}}</ref>。2018年終於首次成為劇集男主角，在《[[身後事務所|身後事務所]]》中領銜主演<ref name="hk.e_【另一種">{{Cite web |title=【另一種體會】拍《身後事》有深刻反思 6號：最緊要在生嘅人點去生活 |author= |work=Apple Daily 蘋果日報 |date= |access-date=24 September 2018 |url= https://hk.entertainment.appledaily.com/enews/realtime/article/20180825/58604095 |language= |quote=}}</ref>。
