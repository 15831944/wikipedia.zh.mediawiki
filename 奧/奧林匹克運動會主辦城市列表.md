第一屆現代奧林匹克運動會於1896年在[[希臘|希臘]][[雅典|雅典]]舉辦，至今有22個城市成功舉辦30屆[[夏季奧林匹克運動會|夏季奧林匹克運動會]]、17個城市成功主辦21屆[[冬季奧林匹克運動會|冬季奧林匹克運動會]]；當中，[[1916年夏季奧林匹克運動會|1916年]]、[[1940年夏季奧林匹克運動會|1940年]]及[[1944年夏季奧林匹克運動會|1944年]]的三屆夏季奧林匹克運動會以及[[1940年冬季奧林匹克運動會|1940年]]及[[1944年冬季奧林匹克運動會|1944年]]連續兩屆的冬季奧林匹克運動會由於两次世界大战而被迫取消。除此之外，雅典在1906年主辦的[[1906年夏季奧林匹克運動會|屆間運動會]]並沒有得到[[國際奧林匹克委員會|國際奧林匹克委員會]]承認。

目前[[日本|日本]][[東京都|東京]]已被[[國際奧委會|國際奧委會]]選為[[2020年夏季奧林匹克運動會|2020年]]夏季奧林匹克運動會的主辦城市，[[法國|法國]][[巴黎|巴黎]]選為[[2024年夏季奧林匹克運動會|2024年]]夏季奧林匹克運動會主辦城市，[[美國|美國]][[洛杉磯|洛杉磯]]選為[[2028年夏季奧林匹克運動會|2028年]]夏季奧林匹克運動會主辦城市，而[[南韓|南韓]][[平昌郡|平昌]]和[[中國|中國]][[北京|北京]]則被被選為[[2018年冬季奧林匹克運動會|2018年]]及[[2022年冬季奧林匹克運動會|2022年]]冬季奧林匹克運動會的主辦城市。同时，[[2020年冬季青年奥林匹克运动会|2020年冬季青年奥林匹克运动会]]和[[2018年夏季青年奥林匹克运动会|2018年夏季青年奥林匹克运动会]]将分别在[[瑞士|瑞士]][[洛桑|洛桑]]和[[阿根廷|阿根廷]][[布宜诺斯艾利斯|布宜诺斯艾利斯]]举行。國際奧委會於2018年10月8日決定[[2022年夏季青年奥林匹克运动会|2022年]]夏季青年奧林匹克運動會的主辦城市為[[塞內加爾|塞內加爾]][[達喀爾|達喀爾]]。

== 主辦城市產生過程 ==
[[國際奧林匹克委員會|國際奧林匹克委員會]]於1999年的第110屆國際奧林匹克委員會全體會議上通過新的奧運會主辦城市選舉程序，整個申辦過程需時約兩年<ref>{{en}}{{cite web | title=Host City |publisher=[[國際奧委會|國際奧委會]] |accessdate=2010年7月2日 |url=http://www.olympic.org/en/content/Olympic-Games/Host-City/ }}</ref>。通常在該屆奧運舉辦的7年前會投票確定主辦城市（由1998年的長野冬季奧運會開始，例如2009年的國際奧委會全體委員大會投票選出2016年奧運的主辦城市）。

首先，有意主辦奧運會的城市必須在國際奧委會所訂的限期前提交有意申請書及問卷調查；同時，有意主辦的城市必須在同時提交申辦會徽以及向國際奧委會發表10分鐘的簡報以介紹他們的管理團隊；另一方面，國際奧委會也會向有意主辦的城市審查有關在[[住宿|住宿]]、[[交通|交通]]、[[保安|保安]]等基本要求。國際奧委會執行委員會在初次遴選過程後列出符合可主辦奧運會規格的城市為申請城市（{{lang|en|applicant city}}）；有關申請城市通過國際奧委會執行委員會審查達到適當資格後，國際奧委會執行委員會會批准有關申請城市正式成為候選城市（{{lang|en|candidate city}}），以授權有關城市進行下一步申辦程序<ref>{{en}}{{cite web |title=How Olympic host cities are chosen |publisher=Helium |last=Kalmus |first=Sage |accessdate=2010年7月9日 |url=http://www.helium.com/items/1018357-how-olympic-host-cities-are-chosen |deadurl=yes |archiveurl=https://web.archive.org/web/20080814015216/http://www.helium.com/items/1018357-how-olympic-host-cities-are-chosen |archivedate=2008年8月14日 }}</ref>。

候選城市需在翌年向國際奧委會評審委員會提交一份詳細報告并準備為該委員會成員在計劃舉辦的選址等地進行實地考察。除非國際奧委會在兩個月內會選出主辦城市，否則評審委員會會就各城市的考察向國際奧委會執行委員會提交評審報告。而國際奧委會委員{{efn|1=候選城市所在国的國際奧委會委員沒有投票權<ref>{{en}}{{cite web | title=What is the Process for Determining the Location of the Olympic Games? |publisher=wiseGEEK |accessdate=2010年7月9日 |url=http://www.wisegeek.com/what-is-the-process-for-determining-the-location-of-the-olympic-games.htm }}</ref>。}}以[[多數制|多輪絕對多數制]]在國際奧委會全體委員大會上以不記名投票方式選出得到超過半數選票的候選城市成為主辦城市（{{lang|en|host city}}）<ref>{{en}}{{cite web |title=How Olympic host cities are chosen |publisher=Helium |last=Kalmus |first=Sage |accessdate=2010年7月9日 |url=http://www.helium.com/items/1018357-how-olympic-host-cities-are-chosen?page=2 |deadurl=yes |archiveurl=https://archive.is/20130426115549/http://www.helium.com/items/1018357-how-olympic-host-cities-are-chosen?page=2 |archivedate=2013年4月26日 }}</ref>。由於國際奧委會主席不會投票，因此若投票至最後一輪，兩個候選城市票數相同時，國際奧委會主席則以自己的一票來決定主辦權归屬。

當選出所屬奧運會的主辦城市後，國際奧委會隨即在當日和該城市簽訂《主辦城市合約書》；合同書具有[[法律|法律]]約束力的責任外，主辦城市為其主辦奧運會而設的組織委員會必須履行《[[奧林匹克憲章|奧林匹克憲章]]》條文外，也要遵從國際奧委會的指示來主辦奧運會。

=== 大洲 ===
奧林匹克運動會曾在[[歐洲|歐洲]]國家舉辦29次，而其第30次已於2014年舉辦；[[北美洲|北美洲]]則舉辦過12次，而單是美國已佔四分之三。位於[[東亞|東亞]]的[[中華人民共和國|中]][[日本|日]][[大韓民國|韓]]三國以及位于[[東南亞|東南亞]]的[[新加坡|新加坡]]為[[亞洲|亞洲]]舉辦過7次奧林匹克運動會，而[[南亞|南亞]]、[[中亞|中亞]]及[[西亞|西亞]]均未曾舉辦奧林匹克運動會。[[2016年夏季奧林匹克運動會|2016年夏季奧運會]]於[[巴西|巴西]][[里約熱內盧|里約熱內盧]]舉辦，這也是[[南美洲|南美洲]]國家首次主辦奧運會。[[大洋洲|大洋洲]]也因為[[澳洲|澳洲]]曾主辦的兩屆奧林匹克運動會而舉辦過2次。而之前一直未曾舉辦過奧運會的[[非洲|非洲]]也因為[[2022年夏季青年奧林匹克運動會|2022年夏季青奧會]]將於[[塞內加爾|塞內加爾]][[達喀爾|達喀爾]]舉辦，得以首次迎來該項盛事。

=== 緯度 ===

南半球已成功舉辦三屆夏季奧運，分別為1956年（南半球夏季）、2000年（南半球春季）、2016年的奧運（南半球冬季）。同時也有一次青年奧運，為2018年10月1日至12日（南半球春季）。

由因為是南北半球的季節顛倒的緣故，故舉辦期間通常會往南半球春季或夏季的9到12月傾向，不會是北半球大部分的7或8月。不過2016年的奧運在8月舉行，成爲首次完全在主辦國冬季期間舉辦的夏季奧運會。

南半球未曾舉辦冬季奧運，但是[[智利|智利]][[圣地亚哥_(智利)|圣地亚哥]]（2026年，成功獲得將成為舉辦地緯度最低）、[[阿根廷|阿根廷]][[巴塔哥尼亞|巴塔哥尼亞]]（2026年）是申辦的大熱門，但是因氣候因素須把時間改為7或8月（南半球冬季）才可。

=== 氣候 ===

因為夏季過於炎熱，除了[[新加坡|新加坡]]，未曾在低緯度地點或熱帶沙漠地區舉辦奧運，即使成功也需要在10到12月舉行（如6次[[亞運會|亞運會]]就是如此），形成「夏季賽事在冬季舉辦」的現象。

=== 宗教 ===
* 目前舉辦奧林匹克運動會的國家中，絕大多數都是以[[基督宗教|基督宗教]]（[[基督新教|基督新教]]、[[東正教|東正教]]、[[羅馬天主教|羅馬天主教]]等）為信仰的歐洲和美洲國家為主
* 1964年的[[1964年夏季奧林匹克運動會|東京奧運會]]是首次由[[神道教|神道教]]與[[佛教|佛教]]信仰為主的日本所舉辦，在那之後的1988年、2008年的夏季奧運以及预定举行的2020年夏季奥运，2010年、2014年的青年奥运和1972年、1998年、2018年以及預定舉行的2022年冬季奧運，也都是以佛教、[[道教|道教]]、[[儒教|儒教]]、[[中國民間信仰|中國民間信仰]]、神道教等信仰為主的中日韓新四國所主辦
* 而以[[伊斯蘭教|伊斯蘭信仰]]為主的國家中至今仍沒有一國舉辦過任何夏季奧運和冬季奧運（[[土耳其|土耳其]]、[[卡達|卡達]]、[[亞塞拜然|亞塞拜然]]都曾考慮申辦，但土耳其[[伊斯坦堡|伊斯坦堡]]在2020年最後一輪敗給[[東京|東京]]，其他國家因為政治因素或保守，不接受[[以色列|以色列]]參賽未曾申辦）。
* 另外一方面以[[印度教|印度教]]信仰為主的南亞大陸和信奉[[猶太教|猶太教]]的以色列至今也沒有主辦過夏季奧運和冬季奧運。（同時也未曾有申辦奧運的想法）
