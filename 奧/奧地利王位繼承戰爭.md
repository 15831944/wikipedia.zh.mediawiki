'''[[奧地利王位繼承戰爭|奧地利王位繼承戰爭]]'''（{{lang-de|Österreichischer Erbfolgekrieg}}、1740年—1748年），或稱'''奧地利領地繼承權戰爭'''，起因為[[奧地利|奧地利]][[奧地利哈布斯堡王朝|哈布斯堡王朝]]絕嗣，[[歐洲|歐洲]]兩大陣營為爭奪奧地利王位<sup>[[#注释|1]]</sup>，並在奧地利獲取利益而引發的戰爭。當中包括了第一次西里西亞戰爭和第二次西里西亞戰爭。

== 烽煙四起 ==
[[File:Kaiserin_Maria_Theresia_(HRR).jpg|left]]]]
1740年10月20日[[神聖羅馬帝國|神聖羅馬帝國]]皇帝[[查理六世_(神圣罗马帝国)|卡爾六世]]逝世，並無遺下男性後嗣，而根據卡爾六世於1713年所頒佈的《[[1713年國事詔書|國事遺詔]]》，其長女[[玛丽娅·特蕾西娅|玛丽娅·特蕾西娅]]有權承襲其奧地利大公之位，而玛丽娅·特蕾西娅的夫婿[[弗朗茨一世_(神圣罗马帝国)|弗兰茨·斯蒂芬]]則可承襲其神聖羅馬帝國帝位。可是[[法國|法國]]、[[西班牙|西班牙]]、[[普魯士|普魯士]]、[[巴伐利亞|巴伐利亞]]、[[薩克森|薩克森]]、[[撒丁|撒丁]]、[[皮埃蒙特|皮埃蒙特]]、[[那不勒斯王國|那不勒斯王國]]並不承認《1713年國事遺詔》，而[[德意志|德意志]]的三大諸侯國--[[普魯士|普魯士]]、[[巴伐利亞|巴伐利亞]]和[[薩克森|薩克森]]更極力協助[[玛丽娅·特蕾西娅|玛丽娅·特蕾西娅]]的堂姐夫巴伐利亞[[選帝侯|選帝侯]][[查理七世_(神圣罗马帝国)|卡爾·阿尔布雷希特]]登上神聖羅馬帝國皇位，即[[查理七世_(神圣罗马帝国)|卡爾七世]]。至此戰爭已不可避免，因此反奧陣營裡最積極的一方[[普魯士|普魯士]]在1740年12月16日正式向[[奧地利|奧地利]]宣戰。[[奧地利|奧地利]]則聯合[[英國|英國]]、[[波希米亞|波希米亞]]、[[匈牙利|匈牙利]]、[[荷蘭|荷蘭]]、[[西里西亞|西里西亞]]及[[俄國|俄國]]，抵抗反奧陣營的入侵。以兩次[[西里西亞戰爭|西里西亞戰爭]]為骨幹，長達8年的[[奧地利王位繼承戰爭|奧地利王位繼承戰爭]]正式爆發。

== 第一次西里西亞戰爭 ==
=== 普軍出擊 ===
戰事爆發初期，奧軍在各處均處於不利位置。1740年12月，[[腓特烈大帝|腓特烈大帝]]和[[施維林|施維林]]元帥率領八萬普軍南下突擊[[西里西亞|西里西亞]]，短期之內毫無防禦準備的奧地利駐軍被擊潰，首府[[布雷斯劳|布雷斯劳]]陷落，只有[[尼斯堡|尼斯堡]]等幾個孤立堡壘仍然堅守。但是[[圍城戰|圍城戰]]向非普軍所長，腓特烈率兩萬普軍屯兵尼斯堡堅城之下，準備長期圍困。1741年4月，奧地利元帥[[奈伯格|奈伯格]]（Neipperg）領兵近兩萬，突然從西里西亞東面的[[摩拉維亞|摩拉維亞]]境內發起進攻，來救尼斯堡，由此在[[莫爾維茨村|莫爾維茨村]]引發[[莫爾維茨會戰|莫爾維茨會戰]]。普王腓特烈親率2.16萬人與奈伯格將軍統率的1.9萬奧軍在此處決戰。普軍在[[步兵|步兵]]與[[火炮|火炮]]上佔優，而奧軍則在[[騎兵|騎兵]]上佔優，雙方激烈交戰，最後奧軍因不敵普軍的猛烈炮火，被逼撤退。
[[File:Charles_VII,_Holy_Roman_Emperor.PNG|thumb]]]]

=== 各國參戰 ===
在[[莫爾維茨會戰|莫爾維茨會戰]]後，[[普魯士|普魯士]]與[[法國|法國]]、[[巴伐利亞|巴伐利亞]]、[[薩克森|薩克森]]和[[西班牙|西班牙]]結盟進攻[[奧地利|奧地利]]，以加強攻勢。1742年1月，奧軍主動進攻[[巴伐利亞|巴伐利亞]]，以逼使巴伐利亞退出戰爭，並從而進逼[[波希米亞|波希米亞]]。普王[[腓特烈|腓特烈]]立即親率普、法、薩克森三國聯軍共3.4萬人進逼至奧地利邊境，在解除巴伐利亞與波希米亞的危機後，普軍調兵北向。[[格林|格林]]的[[卡爾親王|卡爾親王]]則率3萬奧軍拊其側背，從[[摩拉維亞|摩拉維亞]]方向抄後路向腓特烈進攻，於是1742年5月17日爆發[[查圖西茨會戰|查圖西茨會戰]]。經過一輪激戰後，雙方互有勝負，但[[奧地利|奧地利]]的傷亡較大（奧軍損失6330人，普軍損失4800人）。因此，奧軍被逼撤退。

=== 被逼停戰 ===
1742年6月初，奧地利無法再戰，被迫向普魯士求和。並在該年7月28日與普魯士在簽訂《[[1742年柏林条约|柏林和約]]》，以西里西亞及格拉茨割予普魯士為代價，兩國停戰，至此，第一次西里西亞戰爭宣告結束。

== 奧軍反攻 ==
奧地利並不甘心於這種屈辱的議和條約，因此便與英國、[[漢諾威|漢諾威]]、[[黑森|黑森]]、荷蘭結盟進攻反奧陣營除普魯士外的其他盟國。戰事重啟，至1742年底，波希米亞和巴伐利亞已被奧軍奪取。1743年2月8日，奧地利軍在[[特勞恩伯爵|特勞恩伯爵]]元帥的統率下，在義大利與[[莫特馬爾|莫特馬爾]]統率的西班牙軍隊決戰，結果雙方均損失慘重。1743年6月，以英軍及奧軍為主幹的聯軍約4萬人，由英王[[喬治二世|喬治二世]]與奧地利的[[奈伯格將軍|奈伯格將軍]]統率，向[[阿沙芬堡|阿沙芬堡]]進軍，與由[[阿德里安·毛瑞斯·德·诺阿耶|諾阿耶]][[元帥|元帥]]統率的法軍約2萬6千人在[[德廷根戰役|德廷根戰役]]中戰敗。法軍不敵，被逼撤退。同時間，奧地利再與薩克森結盟，其軍勢變得更強。1744年夏，奧軍攻至法國的[[阿爾薩斯|阿爾薩斯]]，並同時進逼義大利的[[那不勒斯王國|那不勒斯王國]]。

== 第二次西里西亞戰爭 ==
[[File:Martin_van_Meytens_006.jpg|thumb]]]]
普魯士不願見到任何一方獲得全勝，尤其害怕奧地利獲勝之後，會回頭追討西里西亞，因此於1744年重新加入戰爭。8月17日，普魯士再度施展突襲攻勢，這次普魯士兵分三路：施維林元帥一路16,000人，[[利奧波德二世|利奧波德二世]]15,000人，自率主力4萬人以[[齊騰|齊騰]]的1,300[[驃騎兵|驃騎兵]]為[[先鋒|先鋒]]，三路大軍採向心攻擊姿態，從西里西亞出發，大敗[[薩克森|薩克森]]（上次戰爭的盟友，現在倒戈） ，再指向西里西亞正南方波希米亞的首都[[布拉格|布拉格]]。1744年9月16日攻佔布拉格，繼續以62,000人的主力向[[維也納|維也納]]進軍，第二次西里西亞戰爭正式揭幕。與上次一樣，奧軍因普軍的突然襲擊而在最初的數個月被其大敗。奧軍只得轉打消耗戰，奧地利軍[[總司令|總司令]]卡爾親王率5萬人回援，採取堅壁清野的方針派兵偷襲普軍的補給線，以截斷其彈藥及糧食供應，使腓特烈在波希米亞境內遭遇後勤補給困難，威脅腓特烈後路並成功消滅普軍近1.2萬人，約佔其軍隊總數的五分之一。

=== 以退為進 ===
當腓特烈緩慢回撤至布拉格附近的時候，曾經想引誘卡爾出來決戰，但是10月份已經天寒地凍，查理親王又會合了薩克森盟軍，兵力增加至10萬人，於是腓特烈承認戰略上已無利可圖，準備緩慢撤回西里西亞。但是他這次又犯了一個錯誤：過於貪圖在波希米亞得到的土地，捨不得撤走分散各個要塞的守軍。結果普魯士[[野戰|野戰]]軍一撤，這些[[據點|據點]]被奧軍逐一拔除，總共白白損失了3,000人。當普軍分路渡過易北河撤回西里西亞境內的時候，因為傷病和逃散，普軍一仗未打就損失了一半力量。經過這次戰略挫折，腓特烈認識到，因為普魯士本身人口有限，軍中很多是[[德意志|德意志]]其他邦國的[[雇傭兵|雇傭兵]]，這樣的軍隊，必須要有鐵的紀律才能凝聚成戰鬥力，而且絕不能分散使用。只准集中，不准分散，這是腓特烈後來一直記取的一個教訓。1744年11月26日，普軍因欠缺彈藥及糧食供應，只得退出布拉格，撤至西里西亞，轉為戰略防禦，儘管戰略上受了一次挫折，但是腓特烈在戰場上並沒有失敗，他將計就計，裝做向西里西亞首府佈雷斯勞撤退，引誘卡爾親王走出波希米亞和摩拉維亞的山地，追擊到西里西亞的平原上，以便以一次乾淨俐落的戰役徹底擊敗奧地利。1745年初，奧軍擊敗法軍和巴伐利亞軍，巴伐利亚被迫于4月22日签订[[富森条约|富森条约]]退出战争。該年5月，奥军攻至西里西亞。5月26日，普軍約5.9萬人在西里西亞的[[霍亨弗里德堡|霍亨弗里德堡]]埋伏。6月4日清晨，[[霍亨弗里德堡戰役|霍亨弗里德堡戰役]]爆發，普軍再度施展其拿手好戲，突襲奧地利與薩克森聯軍，使得奧薩聯軍損失慘重，被逼撤退。

=== 再度議和 ===
1745年9月30日，奧薩聯軍在[[索爾戰役|索爾戰役]]再度與普軍決戰，但又被普軍擊敗。奧地利經過兩次西里西亞戰爭，四次大戰役全部失敗，但是瑪麗婭·特蕾西婭大公的抵抗決心仍然堅韌不拔，1745年年底，奧地利準備集中主力從波希米亞出發，撇開北面西里西亞的腓特烈，向西北方向，越過薩克森，直接攻擊西方的[[勃蘭登堡|勃蘭登堡]]本土，爆發[[凱撒斯多夫會戰|凱撒斯多夫會戰]]。這次敗仗徹底打消了奧地利和薩克森與普魯士為敵的信心，為免損失加劇，奧地利、薩克森在衡量得失後只得在該年聖誕日與普魯士簽訂《[[德雷斯頓和約|德雷斯頓和約]]》。奧地利以西里西亞全境割予普魯士為代價，換取普魯士承認瑪利亞‧特蕾西亞為奧地利大公，及其夫婿[[弗朗茨一世_(神圣罗马帝国)|弗兰茨·斯蒂芬]]為「神聖羅馬帝國」皇帝<sup>[[#注释|2]]</sup>，退出奧地利王位繼承戰，第二次西里西亞戰爭宣告結束。 此後，奧地利再與俄國結盟，並與法軍在奧屬荷蘭、義大利北部及海上進行戰鬥，互有勝負。1746年，法國由於知悉俄國援軍即將抵達奧屬荷蘭，決定與英、奧議和，於是簽訂《[[第二亚琛和约|亞琛和約]]》<sup>[[#注释|3]]</sup>，和約確定普魯士在西里西亞的領土主權；並讓西班牙與[[薩丁|薩丁]]得到奧地利在義大利的部份領地；法國則放棄其在[[印度|印度]]的[[殖民地|殖民地]]；統治神聖羅馬帝國與奧地利的[[哈布斯堡-洛林王朝|哈布斯堡-洛林王朝]]正式建立，奧地利王位繼承戰爭至此完全結束。

== 影響 ==
普魯士的陸軍在這場戰爭裡充份地表現了其卓越的素质，並且在戰後獲得大片土地，從此成為德意志地區除奧地利外最強的邦國。

這場戰爭的決戰數量較此前的[[三十年戰爭|三十年戰爭]]、[[第二次北方战争|第二次北方战争]]、[[大北方戰爭|大北方戰爭]]與[[西班牙王位繼承戰爭|西班牙王位繼承戰爭]]為少，改為以大量的消耗戰消耗敵軍的戰鬥力，影響了18世紀的大部份戰爭的戰略，使得整個18世紀的戰爭均著重消耗戰，直至[[拿破侖戰爭|拿破侖戰爭]]時才出現改變。

[[線式戰術|線式戰術]]在這場戰爭裡開始顯得有心無力，普軍對其進行了改革，發明了「[[斜線式戰術|斜線式戰術]]」的新型線式戰術，使得攻擊力大增，並成為日後戰爭的主要戰術之一。

== 注释 ==
1. '''奧地利王位繼承戰爭'''這譯名為以往中國初接觸外國時，因不了解外國的實際國情而產生的誤譯，正確譯名該為'''奧地利領地繼承權戰爭'''，但因為約定俗成的關係，本文沿襲此名。而因為當時[[奧地利統治者列表|奧地利大公]]同時亦為[[神聖羅馬帝國|神聖羅馬帝國]]皇帝，所以爭奪奧地利的王位(是民間相傳的謬誤，該為奧地利的大公，但因約定俗成，本文沿襲此名)及領土亦即是爭奪奧地利所控制的神聖羅馬帝國帝位及領土。

2. 因為神聖羅馬帝國皇帝之位不能由女性繼承，因此奧地利大公與神聖羅馬帝國皇帝並非同一人。此時奧地利大公為玛丽娅·特蕾西娅，而[[神圣罗马帝国皇帝列表|神聖羅馬帝國皇帝]]則是其夫婿[[弗朗茨一世_(神圣罗马帝国)|弗朗茨一世]]。

3.《[[第二亚琛和约|亞琛和約]]》共有兩條，文中指的是《第二亞琛和約》。
