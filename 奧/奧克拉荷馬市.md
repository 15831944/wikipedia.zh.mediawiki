'''奧克拉荷馬市'''（[[英語|英語]]：'''{{lang|en|Oklahoma City}}'''）是[[美國|美國]][[奧克拉荷馬州|奧克拉荷馬州]]的[[首府|首府]]、第一大城市，[[奧克拉荷馬縣|奧克拉荷馬縣]]縣治。2005年7月1日人口532,517。<ref>http://www.mongabay.com/igapo/2005_world_city_populations/2005_city_population_03.html</ref>奧克拉荷馬市位於奧克拉荷馬州的中部。面積621.2平方英里（1,608.8平方千米），其中水域面積14.2 sq mi（36.7 km²）。奧克拉荷馬市建立在1889年。1907年奧克拉荷馬州加入美國聯邦，奧克拉荷馬市代替了[[加斯里_(俄克拉何马州)|加斯里]]（Guthrie）成為奧克拉荷馬州的首府，所以很多人把資本從加斯里移動去向奧克拉荷馬市投資，而且在1928年發現了油田，所以奧克拉荷馬市就開始興旺起來。

== 历史 ==
俄克拉何马城是在1889年4月22日安顿下来的。当时这个地区叫“没有规划的土地”（unassigned land），在经历“The Oklahoma Land Run”事件之后，这片土地正式开放给别人开发。有大概一万人在一天之内在当今的俄克拉何马城闹市创造了一个帐篷城市。

在俄克拉何马在1907年加入联邦之后，俄克拉何马州政府由加斯里搬到俄克拉何马。

== 地理与气候 ==
奥克拉荷马市属[[副热带湿润气候|副热带湿润气候]]，四季分明，降水充足而5、6月降雨量最多。冬季温和，时而偏向温暖，日最高气温低于或等于{{convert|0|C}}的平均日数为8.3天，日最低气温低于或等于{{convert|0|C}}的平均日数为71天，低于或等于{{convert|-10|C}}的平均日数为6.3天；夏季酷暑漫长，日高温达{{convert|30|C}}的日数年均有100天，达{{convert|100|F|disp=flip}}的有10.4天。<ref name = "NWS Norman, OK"/>最冷月（1月）均温{{convert|39.2|F|1|disp=flip}}，极端最低气温{{convert|−17|F|0|disp=flip}}（1899年2月12日）。<ref name = "NWS Norman, OK"/>最热月（7月）均温{{convert|83.0|F|1|disp=flip}}，极端最高气温{{convert|113|F|0|disp=flip}}（1936年8月11日与2012年8月3日）。<ref name = "NWS Norman, OK"/>无霜期平均为221天（3月30日至11月5日）；可测量降雪平均期为12月13日至2月15日。<ref name = "NWS Norman, OK"/>年均降水量约{{convert|36.5|in|sigfig=3|disp=flip}}；年极端最少降水量为{{convert|15.74|in|disp=flip}}（1901年），最多为{{convert|56.95|in|disp=flip}}（2007年）。<ref name = "NWS Norman, OK"/>年均降雪量为{{convert|7.6|in|cm|disp=flip}}；冬季极端最少降雪量微少以致难以测量（1931–32年与1934–35年），1947–48年的降雪量最多，积累降雪量为{{convert|25.2|in|cm|disp=flip}}。<ref name = "NWS Norman, OK"/>
