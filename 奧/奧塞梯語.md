'''奧塞梯語'''（奧塞梯語：Ирон ӕвзаг, ''Iron ævzag'' 或 Иронау, ''Ironauу''，{{lang-ru|Осетинский язык}}）屬[[印度-伊朗语族|印度-伊朗语族]][[伊朗語支|伊朗語支]]，是一種通行於[[俄羅斯|俄羅斯]]及[[格鲁吉亚|格鲁吉亚]]邊界、[[高加索|高加索]]山區的一種語言。目前使用人口大約有50萬，其中60%居住在俄羅斯的[[北奧塞梯|北奧塞梯]]，15%居住在格魯吉亞的[[南奧塞梯|南奧塞梯]]。

奧塞梯語與[[Tati|Tati]]波斯語及[[Talyshi|Talyshi]]都是伊朗語支的主要語言，它在[[高加索|高加索]]地區有相當數量的使用人口。<!-- It is descended from the language of the [[Alans|Alans]], a group within the nomadic [[Sarmatians|Sarmatians]].  It is believed to be the only surviving descendant of a Sarmatian language. -->

相傳奧塞梯語起源于[[薩爾馬提亞語|薩爾馬提亞語]]。

== 字母 ==
[[image:Oseta_kartvela_skribo.jpg|225px]]來書寫的奧塞梯語之民間文學書籍，於1940年在[[南奥塞梯|南奥塞梯]]出版]]

奧塞梯語總共有35個[[音素|音素]]：26個[[辅音|辅音]]、7個[[元音|元音]]和2個[[雙元音|雙元音]]。古代，奧塞梯語曾采用[[希臘字母|希臘字母]]拼寫，现代则使用一套改良过的[[西里爾字母|西里爾字母]]。不過奧塞梯語在从[[土耳其獨立|土耳其獨立]]到[[斯大林|斯大林]]在蘇聯執政这段时间内採用[[土耳其语字母|土耳其语字母]]來書寫。

奧塞梯語西里爾字母的特点是字母的分工比土耳其字母更精细。其中常用的元音字母「[[ӕ|ӕ]]/Ӕ」（[[æ|æ]] 的西里爾版本）更是奧塞梯語所獨有。

== 方言 ==
奧塞梯語有兩种主要方言：[[伊龍方言|伊龍方言]](''[[:en:Iron_dialect|Iron]]'')及[[季戈爾方言|季戈爾方言]](''[[:en:Digor_dialect|Digor]]'')。伊龍方言使用人口达六分之五。季戈爾方言在12世紀以前曾是主要方言，但之後被伊龍方言取代。此外，还有第三大方言[[傑西克方言|傑西克方言]](''[[:en:Jassic_dialect|Jassic]]'')，曾在[[匈牙利|匈牙利]]境内使用。

奧塞梯語的書面語以伊龍方言為基础。
