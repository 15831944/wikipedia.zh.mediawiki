'''奧蘭治-拿騷勳章'''（{{lang-nl|Orde van Oranje-Nassau}} {{IPA-nl|ˈɔrdə vɑn oːˌrɑɲə ˈnɑsʌu}}）是[[荷蘭王國|荷蘭王國]][[荷蘭授勳及嘉獎制度|授勳及嘉獎制度]]中的一項民、軍事級[[騎士團|騎士勳章]]，由[[攝政|攝政太后]][[埃瑪公主_(瓦爾德克-皮爾蒙特)|埃瑪]]和[[威廉明娜_(荷蘭)|威廉明娜女王]]於1892年4月4日所創立，授予「每位為社會贏得特殊價值的人士」，表揚獲勳者以其獨特方式開創生涯，值得社會的讚賞和認可<ref>{{cite web |url = https://lintjes.nl/onderscheidingen/de-orde-van-oranje-nassau |title = De Orde van Oranje-Nassau |trans-title = 奧蘭治-拿騷勳章 |website = kanselarij.nl |accessdate = 2018-07-06 |language = nl}}</ref>。奧蘭治-拿騷勳章中較低級別者，可類比[[英國授勳及嘉獎制度|英國授勳及嘉獎制度]]中[[大英帝國勳章|大英帝國勳章]]的地位，但在荷蘭，除了[[威廉軍事勳章|威廉軍事勳章]]獲勳者之外，一般不會頒給頭銜、前綴或後綴稱謂（學術銜稱除外）。

== 歷史 ==
1841年，[[荷蘭國王|尼德蘭國王]]和[[盧森堡大公|盧森堡大公]][[威廉二世_(尼德蘭)|威廉二世]]創立了[[橡樹冠勳章|橡樹冠勳章]]。雖然嚴格來說，橡樹冠勳章並不是荷蘭的勳章，但它確實時常被授予[[荷蘭人|荷蘭人民]]。在[[威廉三世_(荷蘭)|威廉三世]]過世後，盧森堡王國根據王朝協約，成為了[[拿騷王朝|拿騷王朝]]的另一個分支領土。因此，荷蘭出現了繼[[威廉軍事勳章|威廉軍事勳章]]和[[荷蘭獅子勳章|荷蘭獅子勳章]]之後的第三級勳章的需求，方能頒授皇室榮譽予他國外交官與市民階級。

在[[第二次世界大戰|第二次世界大戰]]期間，奧蘭治-拿騷勳章被用於授予荷蘭和其他國家軍隊的軍官，以表揚他們協助從[[納粹德國|納粹德國]]的佔領中解放荷蘭，以及解放在太平洋地區的前荷蘭殖民地的功績。在現代，奧蘭治-拿騷勳章仍然是荷蘭最常使用的民、軍事級勳章，它的位階亦僅次於荷蘭獅子勳章。通常每年在君主的官方生日（目前為4月27日）進行授勳儀式，至今約共有三千五百位獲勳者。奧蘭治-拿騷勳章也被授予其他國家的王子、部長、政要和外交官等。

歷經近三十年的討論，1994年荷蘭對其授勳及嘉獎制度進行了大幅度的調整。透過修法，荷蘭將勳級與獲勳者的社會地位與階級脫鉤，以求建立一個更為民主的榮譽體制。自此以後，原則上所有荷蘭公民不論地位與階級，都能被授予奧蘭治-拿騷勳章。在這項變革之前，奧蘭治-拿騷勳章下設了五個級別以及額外的榮譽獎章（金、銀、銅牌）；其中，榮譽獎章只作為勳章的附加榮譽，並沒有正式被納入勳章榮譽制度中。1996年，榮譽獎章被廢止，以奧蘭治-拿騷勳章的「員佐勳章」取代。員佐勳章只頒予荷蘭公民。

== 級別 ==
[[File:Oranje-Nassau_Officier,_Vlaascho.jpg|thumb]]
[[File:Orange-Nassau_Grand_Cross.jpg|thumb]]
奧蘭治-拿騷勳章分為民事和軍事兩類，前者在徽章上以[[月桂|桂冠]]作為標誌，後者則是在徽章和星章上刻有交叉的劍。

荷蘭的[[荷蘭國王|國王]]或[[攝政王后|王后]]是奧蘭治-拿騷勳章的總司令。

=== 1996年起至今 ===
1996年起，除了分為民事和軍事兩類，奧蘭治-拿騷勳章共設有六個級別：
# 騎士大十字勳章
# 大官佐勳章
# 司令勳章
# 官佐勳章
# 騎士勳章
# 員佐勳章

== 獲勳者 ==
每年約有四千五百人獲頒奧蘭治-拿騷員佐勳章，同時約有三千位既有獲勳者逝世。<ref>獲勳者名單見分類：[[:Category:奧蘭治-拿騷勳章|奧蘭治-拿騷勳章]]；[[:Category:奧蘭治-拿騷騎士大十字勳章|騎士大十字勳章]]；[[:Category:奧蘭治-拿騷大官佐勳章|大官佐勳章]]；[[:Category:奧蘭治-拿騷司令勳章|司令勳章]]；[[:Category:奧蘭治-拿騷官佐勳章|官佐勳章]]；[[:Category:奧蘭治-拿騷騎士勳章|騎士勳章]]及[[:Category:奧蘭治-拿騷員佐勳章|員佐勳章]]。</ref>
