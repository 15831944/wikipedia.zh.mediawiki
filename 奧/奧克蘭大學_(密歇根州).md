'''奧克蘭大學'''（'''Oakland University'''，縮寫：'''OU'''）是位於[[美國|美國]][[密歇根州|密歇根州]][[奧克蘭縣_(密歇根州)|奧克蘭縣]][[奧本山|奧本山]]和[[羅切斯特山_(密歇根州)|羅切斯特山]]的一所[[公立大學|公立]][[研究型大學|研究型大學]]。

==历史==
成立於1957年。<ref name="Oakland University Home Page">{{cite web|url=http://www.oakland.edu/|title=Home - Oakland University|publisher=oakland.edu}}</ref><ref>{{cite web |url=http://www2.oakland.edu/map/ |title=Oakland University – Campus Map |deadurl=yes |archiveurl=https://web.archive.org/web/20081203200811/http://www2.oakland.edu/map/ |archivedate=2008-12-03 }}</ref> 它是奧克蘭縣唯一的公立研究型大學。<ref name=aboutOU>{{cite web |url=http://www4.oakland.edu/?id=1611&sid=18 |title=About the Community }}</ref>

該大學最初隸屬[[密歇根州立大學|密歇根州立大學]]，稱密歇根州立大學奧克蘭分校（Michigan State University–Oakland），1963年改現名。
