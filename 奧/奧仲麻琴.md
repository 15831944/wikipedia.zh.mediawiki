'''奧仲麻琴'''（{{bd|1993年|11月18日|||catIdx=Okunaka Makoto}}），出身自[[埼玉縣|埼玉縣]]，隸屬於[[PLATINUM_PRODUCTION|PLATINUM PRODUCTION]]旗下的女演員。亦是[[日本|日本]]女子組-{}-合[[PASSPO☆|PASSPO☆]]的前成員之一

== 經歷 ==
奧仲麻琴在中三時被現時的經紀公司發掘<ref>{{cite web|url=http://trendy.nikkeibp.co.jp/article/column/20120829/1042729/ |language=ja |title=<span lang="ja">【NEXT BREAKERS】奥仲麻琴～仮面ライダーで演技に初挑戦</span> |publisher=日經BP社 |date=2012-09-04 |accessdate=2013-07-30}}</ref>，於2009年跟另外9名女孩子組成偶像組合[[PASSPO☆|PASSPO☆]]<ref>{{cite web|url=http://www.oricon.co.jp/news/music/84334/full/ |language=ja |title=<span lang="ja">CA風10人組ユニット・ぱすぽ☆がメジャーデビュー　憧れは少女時代</span> |publisher=[[Oricon|Oricon]] |date=2011-01-30 |accessdate=2013-07-30}}</ref>。PASSPO☆的主流出道單曲《[[少女飛行|少女飛行]]》（2011年5月發行）在發行首星期獲得[[Oricon公信榜|Oricon公信榜]]週榜冠軍，為該榜首次有女子組合的出道單曲取得冠軍<ref>{{cite web|url=http://www.oricon.co.jp/news/rankmusic/87473/full/ |language=ja |title=<span lang="ja">CA風アイドル・ぱすぽ☆、女性グループ初の初登場首位デビュー</span> |publisher=[[Oricon|Oricon]] |date=2011-05-10 |accessdate=2013-07-30}}</ref>。

奧仲由2011年起開始有個人寫真的工作，同年11月發行了自己首本個人寫真集《{{lang|ja|RUN RUN まこと}}》<ref name="sinkan111105">{{cite web|url=http://www.sinkan.jp/news/index_2311.html |language=ja |title=<span lang="ja">「自信がつきました」　ぱすぽ☆奥仲麻琴が初写真集</span> |publisher=sinkan.jp |date=2011-11-05 |accessdate=2013-07-30}}</ref>。這本寫真集由{{link-ja|川島小鳥|川島小鳥}}拍攝，拍攝為期四個月，以「17歲，青春的大爆發」為主題<ref name="sinkan111105" />。2013年4月，發行了第二本寫真集《{{lang|ja|かわいくてマコトにすいません！}}》，當中刊載了同年1月下旬在[[關島|關島]]拍攝的泳裝寫真<ref>{{cite web|url=http://news.livedoor.com/article/detail/7611807/ |language=ja |title=<span lang="ja">「仮面ライダーウィザード」のヒロイン・奥仲麻琴、水着写真集を発売</span> |publisher=livedoor |date=2013-04-21 |accessdate=2013-07-30}}</ref>。

2012年，奧仲獲起用為[[朝日電視台|朝日電視台]]特攝劇《[[假面騎士Wizard|假面騎士Wizard]]》的女主角「曆」（{{lang|ja|コヨミ}}），初次出演[[電視劇|電視劇]]<ref name="oricon120628">{{cite web|url=http://deview.co.jp/News?am_article_id=2013641 |language=ja |title=<span lang="ja">アイドルグループ・ぱすぽ☆の“まこっちゃん”こと奥仲麻琴が『仮面ライダーウィザード』ヒロインに！</span> |publisher=[[Oricon|Oricon]] |date=2012-06-28 |accessdate=2013-07-30}}</ref>。曆是一名身份神秘的少女，在劇中是支援主角的角色<ref name="oricon120628" />。在訪問中被問及參加《Wizard》選角會的契機時，表示「在看[[宮崎葵|宮崎葵]]的《[[只是愛着你|只是愛着你]]》時，對演戲的興趣湧了起來」<ref name="famitsu">{{Cite journal|title={{lang|ja|仮面ライダーウィザード}} |journal={{lang|ja|オトナファミHEROES 仮面ライダー&スーパー戦隊公式ガイド}} |page=12|publisher=[[enterbrain|enterbrain]]|date=2012年8月}}</ref>。

== 個人 ==
暱稱是「{{lang|ja|まこっちゃん}}」（Mako-cchan），組合中的個人代表色是粉紅色。曾自稱「永遠的十七歲」（{{lang|ja|永遠のセブンティーン}}），在2012年的演唱會上慶生時亦說「過了第三次的十七歲」<ref>{{cite web|url=http://news.walkerplus.com/article/34784/ |language=ja |title=<span lang="ja">ぱすぽ☆がアルバム購入者に無料ライブ！ 誕生日を迎えた“3回目の17歳”奥仲麻琴に厳しいツッコミ!!</span> |publisher=Walkerplus |date=2012-11-19 |accessdate=2013-07-30}}</ref>。家中有兩個姐姐<ref>{{cite web|url=http://ameblo.jp/11181029/entry-11454767554.html |language=ja |title=<span lang="ja">うれしい日</span> |publisher=個人官方部落格 |date=2013-01-22 |accessdate=2013-07-30}}</ref>。奥仲是前[[AKB48|AKB48]]成員[[前田敦子|前田敦子]]的支持者，不過只是喜歡前田個人而已<ref name="famitsu" />。
