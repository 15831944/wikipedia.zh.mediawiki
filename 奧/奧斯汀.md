'''奧斯汀'''（{{lang-en|'''Austin'''}}）是[[美國|美國]][[德克薩斯州|德克薩斯州]]的[[首府|首府]]、[[特拉維斯縣|特拉維斯縣]]縣治，也是[[德克薩斯州立大學奧斯汀分校|德州大學]]的所在，以及眾多[[高科技|高科技]][[企業|企業]]所在的高科技城，又因瀕臨德州丘陵地形之起點，故有「硅丘」（{{lang|en|Silicon Hills}}）之稱，以與美國西部傳統電子重鎮[[矽谷|矽谷]]相呼應。

== 地理 ==
奧斯汀位於德州中部[[丘陵地帶|丘陵地帶]]（Hill Country），在三大主要城市（[[休斯敦|休斯敦]]、[[達拉斯|達拉斯]]和[[圣安东尼奥_(得克萨斯州)|圣安东尼奥]]）之間，交通十分方便。

奥斯汀属[[副热带湿润气候|副热带湿润气候]]，冬无严寒，降水充足而集中在5、6、和10月。冬季温和，时而偏向温暖，日最高气温低于或等于{{convert|5|C}}的平均日数为4.3天，日最低气温低于或等于{{convert|0|C}}的平均日数为13天，低于或等于{{convert|-5|C}}的平均日数为1.9天；夏季酷暑漫长，日高温达{{convert|30|C}}的日数年均有150天，达{{convert|100|F|disp=flip}}的有18天；日低温达{{convert|25|C}}的年均日数为19天。<ref name = "NWS New Braunfels, TX"/>最冷月（1月）均温{{convert|51.5|F|1|disp=flip}}，极端最低气温{{convert|−2|F|0|disp=flip}}（1949年1月31日）。<ref name = "NWS New Braunfels, TX"/>最热月（8月）均温{{convert|85.8|F|1|disp=flip}}，极端最高气温{{convert|112|F|0|disp=flip}}（2000年9月5日与2011年8月28日）。<ref name = "NWS New Braunfels, TX"/>无霜期平均为293天（2月16日至12月5日）。<ref name = "NWS New Braunfels, TX"/>年均降水量约{{convert|34.3|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|11.42|in|disp=flip}}（1954年），{{convert|64.68|in|disp=flip}}（1919年）。<ref name = "NWS New Braunfels, TX"/>年均降雪量为{{convert|0.6|in|cm|disp=flip}}；许多冬季的积累降雪量微少以致难以测量或许无雪，1937–38年的降雪量最多，积累降雪量为{{convert|11.0|in|cm|disp=flip}}。<ref name = "NWS New Braunfels, TX"/>

== 歷史 ==
=== 建城 ===

=== 硅山之稱 ===
1983年，[[德克薩斯州立大學奧斯汀分校|德州大學]]、奧斯汀市政府、德州州政府和美國一些[[高科技企業|高科技企業]]聯手發展在奧斯汀成立了Microelectronics and Computer Technology Consortium（MCC），這是奧斯汀的第一家高科技企業。由於MCC是一個聯營企業，美國各個高科技企業都會派人來參與，各大公司的行政主管等漸漸認識到奧斯汀是一個非常適合居住的地方（見「城市規劃·與其他城市比較」一段），也具有非常好的高科技發展潛力，於是各大公司紛紛來奧斯汀設立分部，其中包括有[[IBM|IBM]]、[[思科|思科]]、[[英特爾|英特爾]]、[[AMD|AMD]]、[[摩托羅拉|摩托羅拉]]、[[飛思卡爾|飛思卡爾]]、[[昇陽電腦|昇陽電腦]]、[[三星電子|三星]]等。與此同時，當時還就讀德州大學的[[戴爾|戴爾]]創始人[[邁克爾·戴爾|邁克爾·戴爾]]（Michael Dell）開始自行組裝並售賣其所組裝的[[電腦|電腦]]，漸漸發展成戴爾企業，至今其總部仍然在奧斯汀市。

由於有越來越多的高科技企業搬來奧斯汀，奧斯汀漸漸獲得了「硅山」（Silicon Hill）之稱，與[[加州|加州]]的硅谷相呼應。

== 城市規劃 ==
奧斯汀居民獨鍾的小城風味，也因此，即使其規模現已名列全美前二十大城，但都市規劃及整體風格，並不像美國其大城擁塞，反而略顯零星。

科羅拉多河（Colorado River）流經奧斯汀市中心。由於奧斯汀為德州的首府，州議會、州長官邸、及州政府各單位皆散布在市中心，因此之故，法律、行政及金融等為奧斯汀市中心主要行業。此外，因應政商差旅，尚有旅店、餐飲等商業機構。

奧斯汀以「世界現場音樂之都」（Live Music Capital of the World）聞名。最著名為[[奧斯汀第六街|第六街]]（Sixth Street），街上各類酒吧和其他夜生活場所，入夜後大多有各類樂團現場駐唱。

州議會大樓以北便是[[德克薩斯州立大學奧斯汀分校|德州大學]]校園。此大學學生五萬餘人、教職員兩萬左右，在奧斯汀人口比例中，佔有相當的份量。

高科技企業主要集中在城市周邊。較著名廠商如[[戴爾|戴爾]]、[[惠普|惠普]]、[[摩托羅拉|摩托羅拉]]、[[三星電子|三星]]、[[蘋果公司|蘋果]]、[[IBM|IBM]]、[[ARM|ARM]]等，都在此設廠從事研發生產。

=== 人口分布 ===
中國留學生主要住在德州大學的已婚家庭宿舍、校園及其附近（校園以北的海德公園地區〔Hyde Park〕、紅河路〔Red River〕，西校園〔West Campus〕）、大西部地區（Far West）和科羅拉多河南岸（Riverside）地區。在高科技企業工作的華人主要居住在離他們公司近的地方，北部的一般住在奧斯汀北部的[[衛星城市|衛星城市]]，如Round Rock、Pflugerville、Cedar Park等，南部的主要住在奧斯汀西南部的新興住宅區。科羅拉多河南岸多[[墨西哥|墨西哥]]移民；奧斯汀東部地區多[[非裔美國人|非裔美國人]]。

德州大學的留學生數目以[[南韓|南韓]]居首，中國和[[印度|印度]]其次；但是若把中國、港澳及世界各地區与國家（如[[台灣|台灣]]、[[新加坡|新加坡]]等）的華裔留學生都算進去的話，華裔留學生還是最多的。在奧斯汀北部North Lamar與Airport Blvd交界附近有韓國城，North Lamar與Braker Lane交界處有[[唐人街|中國城]]。

=== 交通 ===
空中交通方面，為因應不斷膨脹的人口，奧斯汀拆除了原有在市中心的國內機場Robert Mueller Municipal Airport（RMMA），而啓用全美第一個由軍用機場改設營運的國際[[機場|機場]]Austin-Bergstrom International Airport (ABIA)。目前[[國際航班|國際航班]]有直接飛往[[墨西哥|墨西哥]]的[[航線|航線]]，大部份[[國際航線|國際航線]]還是經由[[休士頓|休士頓]]或[[達拉斯|達拉斯]]等地轉機。而國際機場設立的最大功用，在於設置[[海關|海關]]於此城，以便利進出口的商業單位。

陸上交通方面，南北貫穿奧斯汀的[[州際高速公路|州際高速公路]]I-35，從市中心東部經過；由於從南自[[墨西哥|墨西哥]]為起點，經過大城[[聖安東尼奧|聖安東尼奧]]，此線為全美交通最頻繁的一條[[州際公路|州際公路]]之一。沿著I-35開，往北能到[[達拉斯|達拉斯]]（車程約三小時）、[[福和市|福和市]]（沃斯堡，車程約三小時）、[[阿靈頓|阿靈頓]]（車程約四小時），以致離開德州去[[奧克拉荷馬州|奧克拉荷馬州]]等地；往南能到[[聖安東尼奧|聖安東尼奧]]（車程約一個半小時）、Corpus Christi（車程約五小時）、向南越過邊境進入墨西哥。

除了I-35以外，另外還有三條主要高速公路穿過奧斯汀，分別是南北向的US-183，和東西向的US-290、TX-71。沿著US-290往東開，或者TX-71往東接I-10往東都可以到[[休士頓|休士頓]]（車程約三小時）。在奧斯汀市內，與I-35平行，在城的西邊還有一條州轄高速公路Loop-1。

為了舒緩交通問題，聯邦政府已通過預算建設數條通連的[[收費公路|收費公路]]，以及計劃興建[[環城公路|環城公路]]，讓穿州過省的貨車可以繞開市中心而行。

首府大眾運輸（[http://www.capmetro.org/ 官方网站]）經營本地公共交通服務，包括公共汽車、德州大學校車和為老年人和殘疾人服務的特種交通服務，以及一个進行中的關於修建到市西北的鐵路的計畫。公共汽車路線在市區和郊區的轉乘中心較密集。部分市區的公共汽車免费，大部分公共汽車对德州大學學生和老年人、殘疾人免费。同時，為協助降低空氣污染，在2006年9月底及之前，當空氣指數達警戒值而由德州環境保護單位宣佈為臭氧保護日（Ozone Day）時，首府大眾運輸提供免費公共交通服務，以鼓勵大眾使用公共運輸，減少廢氣排放。

=== 與其他城市比較 ===
* [[福布斯|福布斯]]根據[[單身人士|單身人士]]佔[[人口比例|人口比例]]、[[夜生活|夜生活]]、[[文化|文化]]氣息、單身人士[[生活費|生活費]]、[[就業|就業]]情況等一系列因素推選出奧斯汀為美國最適合單身人士居住的城市。（詳見http://www.forbes.com/2003/06/04/singleland.html）
* Centrum的一項研究根據居民的[[心理健康|心理健康]]素質、[[生理健康|生理健康]]素質、對[[生活方式|生活方式]]的追求、[[營養|營養]]、[[体育运动|体育运动]]等一系列因素推選出奧斯汀為美國十大最健康城市之一。（詳見http://www.bestplaces.net/docs/studies/healthy.aspx）
* 奧斯汀環境優美，[[植被|植被]]豐富，市政府對於保護[[自然環境|自然環境]]不遺餘力。全市建有數條數公里長的[[綠色地帶|綠色地帶]]，最長的Barton Creek Greenbelt及Town Lake Trail全長分別是11.6公里和10.1公里，再加上許多[[公園|公園]]（如[[麥金萊瀑布州立公園|麥金萊瀑布州立公園]]），為市民[[長跑|長跑]]、[[遠足|遠足]]、[[越野自行車|越野自行車]]、[[划船|划船]]、[[釣魚|釣魚]]、[[攝影|攝影]]、[[休息|休息]]等提供了優美的[[大自然|大自然]]場所。奧斯汀已經連續十二年是[[美國樹城|美國樹城]]（Tree City USA）了。此外，由市政府辦的[[電力|電力]]供應正在努力減少對[[石油|石油]]的倚賴，大量從德州西部購入[[風力發電|風力發電]]，如今奧斯汀已經在全美國以[[可持續發展能源|可持續發展能源]]發電量中躍居首位了。
