'''奥普拉·盖尔·温弗里'''（{{lang-en|Oprah Gail Winfrey}}，{{bd|1954年|1月29日|||Winfrey, Oprah}}），生于[[美国|美国]][[密西西比州|密西西比州]]，美国电视[[脱口秀|脱口秀]]主持人、制作人、投资家、慈善家及演員，美国最具影响力的非洲裔名人之一，[[时代百大人物|时代百大人物]]。

她是入选[[时代百大人物|时代百大人物]]次数最多者，总共9次。2005年[[美國在線|美國在線]]舉辦票選活動—《[[最偉大的美國人|最偉大的美國人]]》，她被選為美國最偉大的人物中的第九位。

== 早年 ==
歐普拉生于[[密西西比州|密西西比州]][[科修斯科|科修斯科]]，是女仆沃内塔·李（Vernita Lee）和一个士兵弗农·溫芙瑞（Vernon Winfrey）的孩子。她的出生证明上写的是奥帕（Orpah），本来是取自《[[圣经|圣经]]·[[旧约全书|旧约全书]]·[[路得记|路得记]]》一卷裡面的一位[[摩阿布族|摩阿布族]]女人的名字，但是邻居们在读写她的名字时把其中的R和P调换了位置。最终奥普拉成了广泛接受的名字。由于母亲不到20岁，未婚就怀了她，加上双亲的工作收入都相当微薄、奥普拉的童年过得相当清寒。温芙瑞的童年亦很痛苦，她更多次被性虐待。當她14歲時，她意外懷孕，但她的兒子早產，出生不久後死亡。
在17歲時，歐普拉贏得了黑人小姐田納西選美冠軍。

== 職業生涯 ==
歐普拉19岁开始其广播生涯。她是[[纳什维尔|纳什维尔]]WTVF-TV最年轻的新闻主播和第一个非裔女主播。其后转到[[巴尔的摩|巴尔的摩]]WJZ-TV成为「[[六点钟|六点钟]]」（新闻）的主播之一。然后又被招募至理查德·舍尔（Richard Sher）麾下成为WJZ的地方脱口秀节目[[人们在说|人们在说]]（People Are Talking）的主持人之一。

1985年歐普拉瑞移居[[芝加哥|芝加哥]]取代别人成为[[WLS-TV|WLS-TV]]的一个低收视率的半小时的晨间脱口秀[[AM芝加哥|AM芝加哥]]（AM Chicago）的主持人。节目在歐普拉的主持下在不到一年的时间里就取得了很大的成功，并重新命名为[[奥普拉·温芙瑞秀|奥普拉·温芙瑞秀]]，并扩充时长为一小时在全国首播。最初节目遵循传统[[脱口秀|脱口秀]]形式。在1990年代中期，形式变得更加严肃，选题更加偏重温芙瑞认为对女人至为重要并产生重要结果的问题。歐普拉也开始做了大量的[[慈善|慈善]]工作，她的节目对一些饱受贫穷折磨的人和意外事件的受害者做了特写。

歐普拉通常对媒体公开讨论她生活的各方面，包括一些不那么愉快的事情，例如受到[[性骚扰|性骚扰]]的童年及因而导致的[[毒品|毒品]]问题。1990年，她同母异父的妹妹Patricia Lee-Lloyd透露歐普拉曾于14岁怀孕并产下一个死婴。歐普拉的体重波动使她成了[[减肥|减肥]]的精神领袖。在1990年代末，歐普拉把她的[[书友会|书友会]]介绍到了电视上，不管她推荐什么，那本书立刻成为畅销书，这已经成为她的影响力的明证。

在1996年4月16日与[[霍华德·莱曼|霍华德·莱曼]]（Howard Lyman）讨论[[疯牛病|疯牛病]]的节目中，歐普拉喊道：「这让我在想再吃一个汉堡时冷静下来！」1998年[[德克萨斯州|德克萨斯州]]的养牛人起诉了她的节目和莱曼「无端诽谤易腐食品」及「[[行业歧视|行业歧视]]」称温芙瑞的评论导致牛价下跌，造成牛肉生产者损失了1200万美元。两个月后的2月26日，座无虚席的德克萨斯畜牧业乡间法庭宣判温芙瑞言行并未恶意，因而无罪，无需为损失负责。事后歐普拉收到了来自Rosie O'Donnell的明信片：「祝贺妳，妳战胜了牛肉！」

2007年獲得[[霍華德大學|霍華德大學]]榮譽博士學位。

== 事業版圖 ==
[[File:OprahEstate.jpg|thumb]]
歐普拉创办了[https://web.archive.org/web/20050404010919/http://www.oprah.com/uyl/oan_landing.jhtml 天使网络]（The Angel Network），一个每年募集成百万美元善款的组织. 她出版了自己的杂志“O, 奥普拉杂志”和“O居家”, 并与别人一起成立了女人有線电视网“[[氧气|氧气]]”。她是“哈博出品”（Harpo，即反写的奥普拉）的董事长, 出版包括[[托妮·莫里森|托妮·莫里森]]的小说“宠儿”（Beloved）改编的电视剧等。歐普拉也涉足演艺事业，最引人注目的是在電影《[[紫色姐妹花_(電影)|紫色姊妹花]]》（The Color Purple，她因此获得奥斯卡提名）和她自己出品的《[[宠儿|宠儿]]》中的演出。她也是一个出版物作者， 2002年[[艾美奖|艾美奖]]第一届[[鲍勃·霍普|鲍勃·霍普]]人道奖获得者。歐普拉主要在[[伊利诺斯州|伊利诺斯州]]的芝加哥活动但是在[[加利福尼亚|加利福尼亚]]的蒙特斯托也有一处住宅。据说她最近在[[毛伊岛|毛伊岛]]也置办了产业。
[[File:Oprah_Winfrey_2010.jpg|thumb]]获颁[[肯尼迪中心荣誉奖|肯尼迪中心荣誉奖]]]]
歐普拉最近签约将把节目延伸到2010-2011年度。到那时她的节目已经播出25年了。她也计划每年主持140场节目，到最后时再回到现有的数字上，130场。

根据2004年的[[福布斯|福布斯]]杂志称，奥普拉·温芙瑞据信身家超过10亿美金。奥普拉最近住在她的42英亩的加州蒙特斯托（Montecito）的住宅里。据说奥普拉出席前户主的聚会时爱上了这个房子，结果花掉了她5000万美元，虽然人家不是准备卖的。她在[[新泽西州|新泽西州]]的拉瓦莱特（Lavalette）也有房产。

2011年1月1日，自創電視台{{link-en|OWN|Oprah Winfrey Network (U.S. TV channel)}}（奧花·雲費電視網）正式開播。

== 评論 ==
奥普拉因其涉及“拥抱自由主义”遭到了保守派的批评。一个批评是，从1981到2002年担任家庭类杂志“女士居家通讯”（Ladies' Home Journal）的主编Myrna Blyth在其新书“纺纱姐妹”（Spin Sisters）中撰文：媒体女人怎样推销不幸——和自由主义——给美国女人，“媒体精英女人涉嫌向女人们推销不幸及吹捧错误建议”。

她也因为她的书友会及采访风格收到了许多电视小品节目的嘲笑。
