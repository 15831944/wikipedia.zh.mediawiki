'''奧什'''（{{lang-ky|Ош}}、{{lang-uz|O‘sh}}）是[[吉爾吉斯斯坦|吉爾吉斯斯坦]]第二大城市，亦是[[奧什州|奧什州]]的首府，位於吉爾吉斯南部，在费尔干纳盆地东南端，阿克布拉河出山口附近。人口約有50萬。

== 地名 ==

该城在汉代被称为“贰师”<ref>有争议</ref>，城名在《[[西域图志|西域图志]]》中写作“鄂什”，在《[[西域水道记|西域水道记]]》中写作“乌什”，还有卧什、鄂斯等写法。此外，中国民族史学家[[刘义棠|刘义棠]]认为，《元史》中提到的“[[倭赤|倭赤]]”，即如今的奥什。<ref>{{cite journal |author=胡振华 |year=2001 |month= |title=关于吉尔吉斯斯坦古城地名"奥什" |journal=中央民族大学学报：哲学社会科学版 |volume=28 |issue=3 |pages= |id= |url= |accessdate=2016年1月3日 |quote= }}</ref>



== 概况 ==

它是吉爾吉斯“第二個首都”，也是奧什州的行政中心。奧什市是一個滿有生氣的城市：它有[[中亞|中亞]]地區最大及最活躍的戶外市集。奧什市的人口成份並非單一，有[[塔吉克族|塔吉克族]]、[[吉尔吉斯人|吉尔吉斯人]]、[[乌兹别克族|乌兹别克族]]人。1990年就曾發生過很嚴重的種族衝突。

雖然奧什市是一個古城，但打從前[[蘇聯|蘇聯]]建立之時，就已經是一個[[工業|工業]]城市。不過城內大部份的工業都在蘇聯解體之後倒閉，現在慢慢恢復元氣。在[[费尔干纳谷地|费尔干纳谷地]]，[[吉尔吉斯斯坦|吉尔吉斯斯坦]]和[[乌兹别克斯坦|乌兹别克斯坦]]的边境划分相当复杂。在许多情况下，认为划定的边界分开了有着长久历史联系的土地和城镇，这使得[[贸易|贸易]]和经济发展困难。在这种情况下，奥什也失去了与其历史上控制的大部分地区的联系。

通过每天来往于首都[[比什凯克|比什凯克]]和奥什之间的班机，吉尔吉斯斯坦南北部分得以联系。另外，两城市间的长途艰险的道路，也得到了现代化，交通因此便利了许多。

== 历史 ==

古稱[[貳師|貳師]]。漢武帝聽聞大宛產良馬，名為[[汗血馬|汗血馬]]，遺使前往購買，然而使者被殺，錢物被搶，因此太初元年（前104年），封[[李廣利|李廣利]]為「貳師將軍」，率兵數萬進攻貳師城，以取得汗血馬。只是路途遙遠，又多高山大漠，而所過諸城皆閉不給食，結果進攻不下，士卒逃、死者十之六七，被迫撤兵。

[[帖木尔|帖木尔]]的后代，[[印度|印度]][[莫卧儿帝国|莫卧儿帝国]]的创立者[[巴卑尔|巴卑尔]]出生在[[费尔干纳谷地|费尔干纳谷地]]中离奥什不远的[[安集延|安集延]]（位于今天的[[乌兹别克斯坦|乌兹别克斯坦]]境内）。从那里开始，他走上了征服[[德里苏丹国|德里苏丹国]]，并开创[[莫卧儿帝国|莫卧儿帝国]]的征程。
