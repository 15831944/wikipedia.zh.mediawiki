'''奧蘭多·赫南德茲·貝德羅梭'''（{{lang-es|Orlando Hernández Pedroso}}，{{bd|1965年|10月11日}}），外號'''公爵'''，出生於[[古巴|古巴]][[比亞克拉拉省|比亞克拉拉省]]，後歸於[[波多黎各|波多黎各]]籍，是一名[[美國職棒大聯盟|美國職棒大聯盟]]右投[[先發投手|先發投手]]。赫南德茲的球速約138-151[[公里|公里]]（86-94[[哩|哩]]），拿手球路為[[四縫線快速球|四縫線快速球]]、[[曲球|曲球]]、[[滑球|滑球]]和[[變速球|變速球]]。他在1998年至2000年擔任紐約洋基隊[[先發投手|先發投手]]時攀上生涯巔峰，當時洋基隊連續3年奪下[[世界大賽|世界大賽]]冠軍。2005年改效力於芝加哥白襪隊時，該隊拿下當年[[世界大賽|世界大賽]]的冠軍。現為[[自由球員|自由球員]]。

赫南德茲誇張的高抬腿投球姿勢，是他著名的招牌動作。現效力於[[華盛頓國民隊|華盛頓國民隊]]的[[利邦·赫南德茲|利邦·赫南德茲]]（{{lang|es|Liván Hernández}}）是他同父異母的弟弟。

== 古巴歲月 ==
赫南德茲在古巴國家聯賽中為[[哈瓦那工業隊|哈瓦那工業隊]]效力，並助該隊贏得1992年及1996年的冠軍，他也在選拔賽中代表[[哈瓦那|哈瓦那]]的球隊出場，其他同隊成員還包括[[西伍達·哈巴納|西伍達·哈巴納]]及[[哈巴尼羅|哈巴尼羅]]。他在國家聯賽10年的生涯紀錄是126勝47敗、[[防禦率|防禦率]]3.05。國家聯賽及選拔賽生涯勝率0.728，是該聯盟紀錄保持人。

赫南德茲當時也是[[古巴棒球代表隊|古巴棒球代表隊]]的固定班底，在[[1992年夏季奧林匹克運動會|1992年夏季奧林匹克運動會]]比賽裡贏得了[[金牌|金牌]]。

1995年9月奧蘭多·赫南德茲的弟弟利邦·赫南德茲從古巴叛逃之後，1996年7月奧蘭多·赫南德茲便遭到古巴國家安全局扣留並審問；3個月後他遭到禁賽。1997年[[聖誕節|聖誕節]]赫南德茲從一個叫做卡巴里恩的小鎮搭船逃離古巴，[[美國海岸防衛隊|美國海岸防衛隊]]在[[巴哈馬|巴哈馬]]海域攔截了赫南德茲、他的夥伴諾瑞斯·波西、棒球選手艾波托·赫南德茲（無親戚關係）和另外5名人員。隨後整船的成員被遣送至巴哈馬管轄的自由港，以非法移民的罪名被監禁於短期收容所，預計如其他類似案件般最終將被遣返古巴。 

然而，透過運動經紀人馬克·古巴斯以及古巴裔美國人國家基金會（CANF）代表們的奔走，當時的司法總長珍妮特·雷諾基於其若被遣返回古巴將遭迫害以及其傑出運動員的身分，最終給予赫南德茲與波西特別待遇，也就是所謂的「人道假釋」，使他們得以進入[[美國|美國]]。

但是赫南德茲回絕了這項特許，而選擇進入[[波多黎各|波多黎各]]的[[精神療養院|精神療養院]]。因為若他當時立刻成為美國公民，他只能參加一般的棒球選秀，並只能與選中他的球隊談判；但是若身為非美國公民，他可以用[[自由球員|自由球員]]的身份來談判。2個月後，赫南德茲拿著[[紐約洋基隊|紐約洋基隊]]安排的簽證進入[[美國|美國]]，與該隊簽下4年660萬[[美金|美金]]的合約。

== 征戰大聯盟 ==
1999年是身為洋基一員的赫南德茲最風光的一年，他共拿下了17勝9敗的戰績，生涯最高的157次[[三振|三振]]以及214.1局的投球局數，在球季結束後他獲選為[[1999年美國聯盟冠軍賽|1999年美國聯盟冠軍賽]]的[[最有價值球員|最有價值球員]]。

2005年為[[芝加哥白襪隊|芝加哥白襪隊]]效力的赫南德茲在美聯分區系列賽對上[[波士頓紅襪隊|波士頓紅襪隊]]的第3戰第6局裡，投出了名留青史的好球。赫南德茲在滿壘無人出局時上場擔任[[中繼投手|中繼投手]]，他在三振了[[強尼·戴蒙|強尼·戴蒙]]後又連續造成2個高飛球出局數且1分未失，讓[[芝加哥白襪隊|芝加哥白襪隊]]能夠保住贏面，在季後賽橫掃[[波士頓紅襪隊|波士頓紅襪隊]]。2005年球季結束後他與[[中繼投手|中繼投手]][[路易斯·費茲卡諾|路易斯·費茲卡諾]]以及前景極度看好的的外野手[[克利斯·楊|克利斯·楊]]一起被交易到[[亞利桑那響尾蛇隊|亞利桑那響尾蛇隊]]。2006年5月24日他又被換到[[紐約大都會|紐約大都會]]。

赫南德茲轉戰[[國家聯盟|國家聯盟]]的首個球季創下了他自己在進攻方面的難得紀錄。2006年7月29日擊出生涯首支2[[打點|打點]][[安打|安打]]；接著在2006年8月20日完成生涯的首次[[盜壘|盜壘]]。

赫南德茲轉到[[紐約大都會隊|紐約大都會隊]]後投球狀況漸入佳境。20場先發9勝7敗4.09防禦率，幫助大都會拿下國聯東區冠軍。他9月亮眼的投球表現（2勝2敗2.01防禦率）使他贏得國聯分區系列戰首戰先發投手的殊榮，但是賽前一天做短跑練習時扭傷了小腿肌肉而被迫退出季後賽。他在2006年11月14日與大都會隊續約。

== 謊報年齡風波 ==
1998年赫南德茲與洋基簽約時宣布他生於1969年。1999年[[八卦|八卦]]雜誌''The Smoking Gun''挖出他在[[古巴|古巴]]的[[離婚|離婚]]文件，證實他其實生於1965年，也牽扯出他與前妻之間的小孩撫養官司。目前大聯盟官方網站仍標注其出生年為1969年，但[[ESPN|ESPN]]等其他棒球專業網站都已改為1965年。
