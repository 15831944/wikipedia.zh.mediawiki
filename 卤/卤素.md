'''卤素（halogen）'''是[[元素周期表|元素周期表]]上的第ⅦA族[[元素|元素]]（[[IUPAC|IUPAC]]新规定：17族），包括[[氟|氟]]（F）、[[氯|氯]]（Cl）、[[溴|溴]]（Br）、[[碘|碘]]（I）、[[砈|-{zh-hans:砹; zh-hant:砈;}-]]（At）和{{缺字|tian2|1|Ts}}（Ts）。

== 卤素的命名 ==
由于卤素可以和很多[[金属|金属]]形成盐类，因此[[英文|英文]][[卤素|卤素]]（halogen）来源于[[希腊語|希腊語]]''halos''（盐）和''gennan''（形成）两个词。在[[中文|中文]]裡，'''卤'''的原意是盐碱地的意思。所有暂时发现的[[卤素|卤素]][[英文|英文]]名称都以ine做结尾。

== 卤素的物理、化学特性 ==
通常来说，液体卤素分子的沸点均要高于它们所对应的烃链。这主要是由于卤素分子比烷链更加[[电极化|电极化]]，而分子的电极化增加了分子之间的连接力（正电极与负电极的相互吸引），这使我们需要对液体提供更多的能量才能使其[[蒸发|蒸发]]。

位于元素周期表右方的卤族元素是典型的非金属。卤素的电子构型均为ns<sup>2</sup>np<sup>5</sup>，它们获取一个电子以达到稳定结构的趋势极强烈。所以[[化学性质|化学性质]]很活泼，在自然状态下不能以[[单质|单质]]存在，一般[[化合价|化合价]]为-1价，即卤离子（X<sup>−</sup>）的形式。

== 卤素的无机化学反应 ==
[[氯|氯]]的存在范围最广，其餘鹵素按照[[氟|氟]]、[[溴|溴]]、[[碘|碘]]、[[砈|-{zh-hans:砹; zh-hant:砈;}-]]、{{缺字|tian2|1|Ts}}的顺序减少。卤素的[[单质|单质]]都是[[双原子分子|双原子分子]]，都有很强的挥发性，[[熔点|熔点]]和[[沸点|沸点]]随[[原子序|原子序]]的增大而增加。标准状况下，[[氟|氟]]、[[氯|氯]]是[[气体|气体]]、[[溴|溴]]是[[液体|液体]]，[[碘|碘]]和[[砈|-{zh-hans:砹; zh-hant:砈;}-]]是[[固体|固体]]。

=== 卤素的氧化性 ===
卤素单质都有氧化性，氧化性从氟到-{zh-hans:砹; zh-hant:砈;}-依次降低。碘单质氧化性比较弱，[[三价铁离子|三价铁离子]]可以把碘离子氧化为碘。卤素能与部分金属、非金属单质直接化合。卤素与水也能发生氧化还原反应，方程式为：
* 2X<sub>2</sub> + 2H<sub>2</sub>O → 4H<sup>+</sup> + 4X<sup>-</sup> + O<sub>2</sub>
氟与水反应剧烈，氯在光照下缓慢与水发生该反应，碘不发生这个反应。

=== 卤素的歧化反应 ===
卤素单质在碱中容易歧化，方程式为：
* X<sub>2</sub> + 2OH<sup>-</sup>（冷）→ X<sup>-</sup> + XO<sup>-</sup> + H<sub>2</sub>O
* 3X<sub>2</sub> + 6OH<sup>−</sup>（热）→ 5X<sup>−</sup> + XO<sub>3</sub><sup>−</sup> + 3H<sub>2</sub>O
但在酸性条件下，其逆反应很容易进行：
* 5X<sup>−</sup> + XO<sub>3</sub><sup>−</sup> + 6H<sup>+</sup> → 3X<sub>2</sub> + 3H<sub>2</sub>O
这一反应是制取溴和碘单质流程中的最后一步。

=== 卤素的氢化物 ===
卤素的氢化物叫卤化氢，为[[共价化合物|共价化合物]]；而其溶液叫氢卤酸，因为它们在水中都以[[离子|离子]]形式存在，且都是[[酸|酸]]。[[氢氟酸|氢氟酸]]（pKa=3.20）一般看成是弱酸。[[氢氯酸|氢氯酸]]（即[[盐酸|盐酸]]）、[[氢溴酸|氢溴酸]]、[[氢碘酸|氢碘酸]]都是典型的强酸，酸性从HCl到HI依次增强，它们的pKa均为负数。至於[[氫砈酸|-{zh-hans:氢砹酸; zh-hant:氫砈酸;}-]]則具備氫鹵酸中最強的酸性，但它極易分解為氫與-{zh-hans:砹; zh-hant:砈;}-單質。<ref>Fairbrother, Peter, [http://sci.tech-archive.net/Archive/sci.chem/2008-09/msg00105.html "Re: Is hydroastitic acid possible?"], accessed July 3, 2009.</ref>

=== 卤素的含氧酸 ===
卤素（除氟外，氟只有-1价）可以显示多种价态，正价态一般都体现在它们的含氧酸根中：
* +1: HXO（[[次卤酸|次卤酸]]）
* +3: HXO<sub>2</sub>（[[亚卤酸|亚卤酸]]）
* +5: HXO<sub>3</sub>（[[卤酸|卤酸]]）
* +7: HXO<sub>4</sub>（[[過卤酸|過卤酸]]）
卤素的[[含氧酸|含氧酸]]均有[[氧化性|氧化性]]，同一种元素中，次卤酸的氧化性最强。

卤素的含氧酸多数-{}-只存在于溶液中，而少数盐是以固态存在的，如[[碘酸盐|碘酸盐]]和[[高碘酸盐|高碘酸盐]]。HXO（X=Cl、Br、I）、HIO<sub>3</sub>和HXO<sub>4</sub>（X=Cl、Br、I）分子在气相中十分稳定，可用质谱和其他方法研究。卤素存在的含氧酸见下表<ref>《无机化学》丛书。第六卷.2.6.4卤素含氧酸及其盐.P</ref><sub>290-291</sub>。

=== 卤素的其他无机化学性质 ===
卤素的[[氧化物|氧化物]]都是[[酸酐|酸酐]]。像[[二氧化氯|二氧化氯]]（ClO<sub>2</sub>）这样的偶氧化态氧化物是混酐。

只由两种不同的卤素形成的化合物叫做[[互卤化物|互卤化物]]，其中显[[电正性|电正性]]的一种[[元素|元素]]呈现正氧化态，氧化态为奇数。这是由于卤素的价电子数是奇数，周围以奇数个其它卤原子与之成键比较稳定（如IF<sub>7</sub>）。互卤化物都能水解。

== 卤素的有机化学反应 ==
在[[有机化学|有机化学]]中，卤族元素经常作为决定[[有机化合物|有机化合物]]化学性质的[[官能团|官能团]]存在。常用X表示。如R-X是含鹵素原子的烴類。

卤素的[[物理特性|物理特性]]和[[化学特性|化学特性]]明显区分于与它对应的[[烃链|烃链]]的主要原因，在于卤素原子（如F、Cl、Br、I）与[[碳|碳]]原子的连接，即C-X的连接，明显不同于[[烃链|烃链]]C-H连接。

* 由于卤素原子通常具有较大的负电性，所以C-X连接比C-H连接更加电极化，但仍然是[[共价键|共价键]]。

* 由于卤素原子相较于碳原子，通常体积和质量较大，所以C-X连接的[[偶极子矩|偶极子矩]]（Dipole Moment）和[[连接能量|连接能量]]（Bonding Energy）远大于C-H，这些导致了C-X的连接力（Bonding strength）远小于C-H连接。

* 卤素原子脆弱的p轨道（Orbital）与碳原子稳定的sp<sup>3</sup>轨道相连接，这也大大降低了C-X连接的稳定性。

卤素最常见的有机化学反应为[[亲核性取代反应|亲核性取代反应]]（nucleophilic substitution）。

通常的化学式如：

Nu:<sup>−</sup> + R-X → R-Nu + X<sup>−</sup>

“Nu:<sup>−</sup>”在这里代表亲核负离子，离子的亲核性越强，则[[产率|产率]]和化学反应的速度越可观。

“X”在这里代表卤素原子，如F、Cl、Br、I，若X<sup>−</sup>所对应的酸（即HX）为强酸，那么[[产率|产率]]和反应的速度将非常可观，如果若X<sup>−</sup>所对应的酸为弱酸，则[[产率|产率]]和反应的速度均会下降。

== 有機卤化物的制成 ==
=== 通过[[加成反应|加成反应]] ===
可通过[[加成反应|加成反应]]，在一个未饱和烃链加入卤素，此为最简单的方式，如：

CH<sub>3</sub>-CH<sub>2</sub>-CH=CH<sub>2</sub> + HBr → CH<sub>3</sub>-CH<sub>2</sub>-CH（Br）−CH<sub>3</sub>

不需要[[催化剂|催化剂]]的情况下，产率90%以上。

=== Karasch方式 ===

如果希望将Br加在[[烃链|烃链]]第一个碳原子上，可以使用Karasch的方式：
                    
CH<sub>3</sub>-CH<sub>2</sub>-CH=CH<sub>2</sub> + HBr → CH<sub>3</sub>-CH<sub>2</sub>-CH<sub>2</sub>-CH<sub>2</sub>-Br + H<sub>2</sub>O

催化剂：HO-OH

产率90%以上。

=== 由[[苯|苯]]合成 ===
由[[苯|苯]]合成卤化物则必须要通过催化剂，如：
             
:[[File:Benzene_halogen.PNG|400px]]

催化剂：FeX<sub>3</sub>或者AlX<sub>3</sub>（X代表氟、氯、溴、碘）

产率较高。

=== 由[[醇|醇]]合成 ===
由[[醇|醇]]合成卤化物，必须用好的[[亲核试剂|亲核试剂]]，[[强酸|强酸]]作为[[催化剂|催化剂]]以提高产率和速度：
CH<sub>3</sub>-CH<sub>2</sub>-CH<sub>2</sub>-CH<sub>2</sub>-OH + Br<sup>−</sup> ←→ CH<sub>3</sub>-CH<sub>2</sub>-CH<sub>2</sub>-CH<sub>2</sub>-Br + H<sub>2</sub>O

催化剂：H<sup>+</sup>

'''注意'''此反应为[[双向反应|双向反应]]，故[[产率|产率]]和[[速度|速度]]有限。
