'''卤化氢'''即[[卤素|卤素]]原子（F、Cl、Br、I、At）与[[氢|氢]]原子（H）结合形成的[[共价化合物|共价化合物]]。它们之间的[[共价键|共价键]]极性较强，在水中易[[电离|电离]]。除氟化氢外，卤化氢都是[[强电解质|强电解质]]。卤化氢包括[[氟化氢|氟化氢]]、[[氯化氢|氯化氢]]、[[溴化氢|溴化氢]]、[[碘化氢|碘化氢]]、[[砹化氢|砹化氢]]。

== 化学性质 ==
=== 相似性/共性 ===
还原性：例如2HBr + Cl<sub>2</sub> = HCl + Br<sub>2</sub>

氧化性：例如Mg + 2HCl = MgCl<sub>2</sub> + H<sub>2</sub>↑

酸性：例如Fe(OH)<sub>2</sub> + 2HCl = FeCl<sub>2</sub> + 2H<sub>2</sub>O

=== 递变性/差异 ===
还原性： HF < HCl < HBr < HI

酸性：HF（中强酸）<HCl（强酸）<HBr（强酸）<HI（无氧酸最强酸）

热稳定性：HF > HCl > HBr > HI

== 用途 ==
*[[氢氟酸|氢氟酸]]（[[氟化氢|HF]]水溶液）:用于在玻璃上绘画、刻字。
*[[氢氯酸|氢氯酸]](盐酸)（[[氯化氢|HCl]]水溶液）:重要化工产品。用于金属表面除锈、制造药物(如盐酸麻黄素、氯化锌)等；人体胃液中含有氢氯酸。
