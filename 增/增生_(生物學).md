'''細胞增生'''（{{lang-en|Hyperplasia}}），是指由於細胞數量增加（{{le|細胞增殖|cell proliferation}}）而造成[[组织_(生物学)|組織]]的體積質量增加<ref>{{Cite web|title = Hyperplasia: MedlinePlus Medical Encyclopedia|url = https://www.nlm.nih.gov/medlineplus/ency/article/003441.htm|website = www.nlm.nih.gov|accessdate = 2015-05-30}}</ref>，並且有些情況下會導致[[器官|器官]]變大。細胞增生有時會與良性的[[贅生物|贅生物]]或是[[良性肿瘤|良性肿瘤]]混淆<ref name="Sembulingam">{{Cite book|title = Essentials of Medical Physiology|url = https://books.google.com/books?id=tdODr5fpxEAC&pg=PA20 |publisher = JP Medical Ltd|date = 2012-09-01|isbn = 9789350259368|first = K.|last = Sembulingam|first2 = Prema|last2 = Sembulingam}}</ref>。

細胞增生是細胞對外在刺激常見的腫瘤前期反應<ref>{{Cite book|title = Pathologic Basis of Veterinary Disease|url = https://books.google.com/books?id=eLHQAgAAQBAJ&pg=PA289 |publisher = Elsevier Health Sciences|date = 2013-12-11|isbn = 9780323291729|first = James F.|last = Zachary|first2 = M. Donald|last2 = McGavin}}</ref> ，在顯微鏡底下，增生細胞與正常細胞無異，但是數量變多，有時候增生細胞的體積也可能變大（[[肥大|細胞肥大]]）<ref>{{Cite book|title = Pathophysiology: Functional Alterations in Human Health|url = https://books.google.com/books?id=hRoRff4PH6wC&pg=PA17 |publisher = Lippincott Williams & Wilkins|date = 2007|isbn = 9780781762502|first = Carie Ann|last = Braun|first2 = Cindy Miller|last2 = Anderson}}</ref>。細胞增生與[[肥大|細胞肥大]]的主要差異在於細胞受到外在刺激之後的反應方式，細胞增生是受到刺激之後細胞數量增加，而[[肥大|細胞肥大]]則是細胞的體積變大<ref>{{Cite book|title = Essentials of Rubin's Pathology|url = https://books.google.com/books?id=7HdzBBhtxycC&pg=PA2 |publisher = Lippincott Williams & Wilkins|date = 2009|isbn = 9780781773249|first = Emanuel|last = Rubin|first2 = Howard M.|last2 = Reisner}}</ref>。

==症狀==
細胞增生是一種對特定刺激的正常[[生理|生理]]反應，增生的細胞也仍然受到[[細胞週期#真核細胞週期的調控|細胞生長週期調控]]<ref name="Sembulingam"/>；然而，如果出現過多的[[荷爾蒙|荷爾蒙]]或是[[生長因子|生長因子]]的刺激，細胞增生也會成為病理反應。異常的細胞增生和正常的細胞增生同樣受到[[生長因子|生長因子]]控制，並且在移除刺激後停止生長<ref name=":0">{{Cite book|title=Robbins Basic Pathology|last=Kumar, V., Abbas, A., Aster, J.|first= |publisher=Elsevier |year=2013 |isbn=9780808924326 |location=Philadelphia, USA |pages=4}}</ref>，這一點和[[癌症|癌症]]及良性腫瘤發展過程中的[[贅生物|贅生物]]不同，不同點在於[[癌变|癌变]]細胞不但異常生長，而且對於正常生長因子刺激沒有反應<ref>{{Cite book|title = Holland Frei Cancer Medicine Eight|url = https://books.google.com/books?id=R0FbhLsWHBEC&pg=PA26 |publisher = PMPH-USA|date = 2010|isbn = 9781607950141|first = Waun Ki|last = Hong|first2 = William N.|last2 = Hait}}</ref> 。即使如此，異常的細胞增生可以是進一步發生癌變的起點<ref name=":0" />。

== 原因 ==
許多原因都可能導致細胞增生，包括為了補償皮膚損傷而發生的表皮基底層增生、慢性發炎反應、荷爾蒙失調、或是因為[[創傷|創傷]]及其他疾病而發生的器官補償生長<ref>{{Cite book|title = Essentials of Pathophysiology: Concepts of Altered Health States|url = https://books.google.com/books?id=FFg88IaReBwC&pg=PA775 |publisher = Lippincott Williams & Wilkins|date = 2011|isbn = 9781582557243|first = Carol|last = Porth}}</ref> 。細胞增生可能無害且發生在特定組織，例如乳腺的泌乳細胞在懷孕之後發生細胞增生，以因應分娩後的哺乳需求<ref>{{Cite book|title = Breast Surgical Techniques and Interdisciplinary Management|url = https://books.google.com/books?id=ICfEsoKSwmsC&pg=PA188 |publisher = Springer Science & Business Media|date = 2011-01-15|isbn = 9781441960764|first = Frederick|last = Dirbas|first2 = Carol|last2 = Scott-Conner}}</ref>。

[[胰岛素样生长因子1|胰岛素样生长因子1]]（IGF）在人體上最主要的功能是促進細胞增生，也就是細胞分裂<ref>{{Cite book|title = Advanced Neuromuscular Exercise Physiology|url = https://books.google.com/books?id=tTaLY-JpsJsC&pg=PA146 |publisher = Human Kinetics|isbn = 9781450412278|first = Phillip|last = Gardiner}}</ref>，相對地，[[肥大|細胞肥大]]則只是細胞體積增加，例如骨骼肌細胞在[[負重訓練|重量訓練]]及使用[[同化類固醇|類固醇]]之後產生的變化<ref>{{Cite book|title = The Young Athlete: A Sports Doctor's Complete Guide for Parents|url = https://books.google.com/books?id=56QJzCNEjzkC&pg=PT126 |publisher = Little, Brown|date = 2009-10-31|isbn = 9780316086738|first = Jordan D.|last = Metzl|first2 = Carol|last2 = Shookhoff}}</ref>。使用IGF後，會導致細胞增生而真正地增加組織中肌肉細胞的數量<ref>{{Cite book|title = The Endocrine System|url = https://books.google.com/books?id=JemcAAAAQBAJ&pg=PA45 |publisher = Britannica Educational Publishing|date = 2011-11-01|isbn = 9781615307319|first = Britannica Educational|last = Publishing}}</ref>，之後施加重量訓練，無論是否使用類固醇，都可以使這些新產生的細胞在大小及力量上更成熟。有些理論認為某些運動的力量輸出訓練可以導致細胞增生，因此增加肌肉纖維的數量而不是僅增加單一肌肉纖維的體積<ref>{{Cite book|title = Therapeutic Exercise: Foundations and Techniques|url = https://books.google.com/books?id=udY-AAAAQBAJ&pg=PA169 |publisher = F.A. Davis|date = 2012-10-08|isbn = 9780803638976|first = Carolyn|last = Kisner|first2 = Lynn Allen|last2 = Colby}}</ref>。

==診斷==
以子宮內膜細胞增生為例，通常是透過[[巴氏涂片检查|巴氏涂片检查]]來確認診斷，或是在子宮頸檢查時同時取得子宮內膜組織切片<ref name="book">{{Cite book|title = Primary Care: The Art and Science of Advanced Practice Nursing|url = https://books.google.com/books?id=RR1hAQAAQBAJ&pg=PA709 |publisher = F.A. Davis|date = 2011-04-06|isbn = 9780803626478|first = Lynne M.|last = Dunphy|first2 = Jill E.|last2 = Winland-Brown}}</ref>。至於{{le|庫興氏病|Cushing's disease}}則可以檢查患者唾液的[[可體松|可體松]]濃度是否升高<ref>{{Cite book|title = Cushing's Syndrome: New Insights for the Healthcare Professional: 2013 Edition: ScholarlyBrief|url = https://books.google.com/books?id=OcKC0lXZn58C&pg=PA9 |publisher = ScholarlyEditions|date = 2013-07-22|isbn = 9781481651837}}</ref>。

===種類===
[[File:Hemihyper.jpg|thumb]]
細胞增生有些更常見的臨床形式，或是有些情況導致細胞增生：
*'''[[良性前列腺增生症|良性前列腺增生症]]'''—也被稱作攝護腺肥大<ref name="cite">{{Cite web|title = Prostate Enlargement: Benign Prostatic Hyperplasia|url = http://www.niddk.nih.gov/health-information/health-topics/urologic-disease/benign-prostatic-hyperplasia-bph/Pages/facts.aspx|website = www.niddk.nih.gov|accessdate = 2015-05-30}}</ref>。
*'''{{le|庫興氏病|Cushing's disease}}'''—因為患者體內[[促腎上腺皮質激素|促腎上腺皮質激素]]濃度升高，導致腎上腺皮質發生異常的細胞增生<ref>{{Cite web|title = Cushing disease: MedlinePlus Medical Encyclopedia|url = https://www.nlm.nih.gov/medlineplus/ency/article/000348.htm|website = www.nlm.nih.gov|accessdate = 2015-05-30}}</ref>。
*'''[[先天性腎上腺增生症|先天性腎上腺增生症]]'''—腎上腺的遺傳疾病<ref>{{Cite web|title = Congenital adrenal hyperplasia: MedlinePlus Medical Encyclopedia|url = https://www.nlm.nih.gov/medlineplus/ency/article/000411.htm|website = www.nlm.nih.gov|accessdate = 2015-05-30}}</ref>。
*'''[[子宮內膜增生症|子宮內膜增生症]]'''—子宮內膜發生細胞增生，是[[多囊卵巢綜合症|多囊卵巢綜合症]]的症狀之一，當[[雌激素|雌激素]]長時間對子宮內膜刺激，且子宮內膜缺乏[[黃體素|黃體素]]的保護時，就會導致子宮內膜增生。如果沒有接受適當的治療，可能會進一步發展成[[子宮內膜癌|子宮內膜癌]]<ref>{{Cite web|title = Endometrial Hyperplasia - ACOG|url = http://www.acog.org/Patients/FAQs/Endometrial-Hyperplasia|website = www.acog.org|accessdate = 2015-05-30}}</ref>。從子宮內膜增生症發展成子宮內膜癌是異常細胞增生可以導致癌變的典型範例<ref name=":0" /> 。
*'''{{le|半身細胞增生|hemihyperplasia}}'''—只有半邊身體發生細胞增生，有時會導致肢體長度不同<ref>{{Cite web|title = Hemihyperplasia - Glossary Entry|url = http://ghr.nlm.nih.gov/glossary=hemihyperplasia|website = Genetics Home Reference|accessdate = 2015-05-30}}</ref>。
*'''[[乳腺生長|乳房細胞增生]]'''—乳房細胞增生包括正常[[乳腺管|乳腺管]]增生以及{{le|非典型乳腺管增生|Atypical ductal hyperplasia}}，其中後者有較高的機率進展為乳癌<ref name="cite2">{{Cite book|title = Diagnostic Problems in Breast Pathology|url = https://books.google.com/books?id=W2BzkJwU21kC&pg=PA33 |publisher = Elsevier Health Sciences|date = 2009|isbn = 1416026126|first = Frederick C.|last = Koerner}}</ref> 。
*'''{{le|血管內膜增生|Intimal hyperplasia}}'''—血管內膜增生是血管受損之後的正常生理反應，也是器官移植失敗的原因之一，尤其是人工靜脈移植；血管內膜增生也是{{le|動脈內膜切除術|endarterectomy}}的併發症之一<ref>{{Cite journal|title = Analysis of arterial intimal hyperplasia: review and hypothesis|journal = Theoretical Biology & Medical Modelling|date = 2007-10-31|issn = 1742-4682|pmc = 2169223|pmid = 17974015|pages = 41|volume = 4|doi = 10.1186/1742-4682-4-41|first = Vladimir M|last = Subbotin}}</ref>。
*'''{{le|表皮細胞增生|Focal epithelial hyperplasia}}'''—這是在口腔黏膜組織（在喉腔偶爾也會發生）的疣狀增生，是受到某些特定型別的[[人類乳突病毒|人類乳突病毒]]感染引起。這個類型尚未發現進展為癌症的病例<ref>{{Cite book|title = Essentials of Oral Pathology|url = https://books.google.com/books?id=fDRtoGmVGQQC&pg=PA9 |publisher = JP Medical Ltd|date = 2011|isbn = 9789350252147|first = Swapan Kumar|last = Purkait}}</ref>。
*'''{{le|皮脂腺增生|Sebaceous hyperplasia}}'''—在這個種類中，患者的皮脂腺增生而變大，因此在皮膚上出現小而黃的突起，好發於臉部，這種疾病不具有傳染性也不危險<ref>{{Cite book|title = Skin Diseases in the Elderly: A Color Handbook|url = https://books.google.com/books?id=fPMoxcB2XDwC&pg=PA85 |publisher = CRC Press|date = 2011-10-01|isbn = 9781840766158|first = Colby Craig|last = Evans|first2 = Whitney A.|last2 = High}}</ref>。
*'''[[肝臟|肝臟]]代償性增生'''—肝臟受損之後，[[肝細胞|肝細胞]]大量分裂增殖，由新產生的細胞恢復肝臟功能到基本水準。肝臟即使有75%受損或是被切除，仍然可以在肝細胞分裂複製後恢復，這也是臨床上[[肝臟#肝臟移植|肝臟活體移植]]可行的原因<ref>{{Cite book|title = Robbins & Cotran Pathologic Basis of Disease|url = https://books.google.com/books?id=5NbsAwAAQBAJ&pg=PA36 |publisher = Elsevier Health Sciences|date = 2014-09-05|isbn = 9780323296359|first = Vinay|last = Kumar|first2 = Abul K.|last2 = Abbas|first3 = Jon C.|last3 = Aster}}</ref>。

==治療==
細胞增生的治療會依據每種疾病的不同採用不同的治療方法。例如[[攝護腺肥大|攝護腺肥大]]的患者可以合併使用{{le|甲型交感神經受體阻斷劑|Alpha-1 blocker}}以及{{le|5α還原酵素抑制劑|5α-Reductase inhibitor}}治療<ref>{{cite web|title=Benign Prostatic Hypertrophy Treatment & Management: Approach Considerations, Combination Therapy, Minimally Invasive Treatment|url=http://emedicine.medscape.com/article/437359-treatment#d10|website=Medscape|publisher=eMedicine|accessdate=3 September 2016}}</ref>。
