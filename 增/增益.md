'''增益'''在[[電子學|電子學]]上，通常為一個系統的[[訊號|訊號]]輸出與訊號輸入的[[比率|比率]]。如5倍的增益，即是指系統令[[電壓|電壓]]或[[功率|功率]]增加了5倍。增益主要應用於[[放大電路|放大電路]]中。

==對數單位與分貝==
電子學上常使用[[對數|對數]]單位量度增益，並以'''貝'''（bel）作為單位：

: Gain = log<sub>10</sub>(''P''<sub>2</sub>/''P''<sub>1</sub>) bel

其中P1與P2分別為輸入及輸出的功率。

由於增益的數值通常都很大，因此一般都使用[[分貝|分貝]]（dB，貝的10分之1）來表示：

: Gain = 10×log<sub>10</sub>(''P''<sub>2</sub>/''P''<sub>1</sub>) dB

一个类似的单位使用[[自然对数|自然对数]]，称为[[奈培|奈培]]。

當增益以電壓而非功率計算時，使用焦耳定律 （Joule's law，''P''=''V''<sup>2</sup>/''R''），其公式為：

* 增益 =10×log<sub>10</sub> ((''V''<sub>2</sub><sup>2</sup>/''R'') /(''V''<sub>1</sub><sup>2</sup>/''R'') )dB
=10×log<sub>10</sub>((''V''<sub>2</sub>/''V''<sub>1</sub>)<sup>2</sup>)dB

=20×log<sub>10</sub>(''V''<sub>2</sub>/''V''<sub>1</sub>) dB

此公式只有在負載[[阻抗|阻抗]]相等（[[阻抗匹配|阻抗匹配]]）時才為正確。在不少現代電子設備中，由於輸出阻抗較低，而輸入阻抗則較高，令負載可被忽略而不明顯地影響計算結果。

===舉例===
如一個[[放大器|放大器]]輸出1[[伏特|伏特]]至1[[歐姆|歐姆]]的負載，提供的輸出功率為1[[瓦特|瓦特]]。如放大器被調節至輸出10伏特至同一負載，它提供的輸出功率則為100瓦特（''P''=''V''<sup>2</sup>/''R''）。因此：

電壓增益 = 10/1 = 10倍
<br />
P = V²/R
功率增益 = (10²/R) / (1²/R) = 100倍


根據定義，其增益為 10Log100 = 20 dB

如果增益为1或0 dB，即输出电压等于输入电压，这个增益也称为'''单位增益'''。
