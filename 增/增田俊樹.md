'''增田俊樹'''（{{lang|ja|増田俊樹}}，{{lang|ja|ますだ としき}}、{{bd|1990年|3月8日|catIdx=Masuda Toshiki}}）日本的聲優、演員。廣島縣出身。

擔任主要或重要角色的作品有《[[遊戲王ZEXAL|遊戲王ZEXAL]]》（神代凌牙）、《[[武士弗拉明戈|佛朗明哥武士]]》（羽佐間正義）、《[[美男高校地球防衛部LOVE！|美男高校地球防衛部LOVE！]]》（藏王立）、《[[山田君與7人魔女|山田君與7人魔女]]》（宮村虎之介）、《[[IDOLiSH7|IDOLiSH7]]》(和泉一織)等。

日本另有一位姓名完全相同的增田俊樹，職業為演員、電影導演。<ref>{{Cite web|url=http://studio-casa.main.jp/t.masuda/|title=首頁|work=演員 電影導演 増田俊樹 Official web site|publisher=增田俊樹|accessdate=2016年10月3日}}</ref>

== 略歷 ==
自小就想當演員，高中時因看了“[[Code_Geass_反叛的魯路修|Code Geass 反叛的魯路修]]”對動畫產生興趣，“[[天元突破紅蓮螺巖|天元突破紅蓮螺巖]]”造就了想要成為聲優的契機。

2008年從廣島前往東京announce學院就讀放送聲優科。

2009年還在求學時，就通過了“[[網球王子舞台劇|網球王子舞台劇]]”的試鏡，以幸村精市這個角色出道。

2010年9月，因為之前就抱持著想成為聲優的夢想，故轉至事務所的聲優部門，10月開始在『A&G ARTIST ZONE 2h』擔任主持助手，由此開始聲優事業活動。

2011年10月，在[[文化放送|文化放送]]的“細谷佳正・増田俊樹的全力男子”中擔任主持人。次年2月，因節目而組成的「Max Boys」組合發行了CD。

2013年在動畫中初次擔當主角，飾演“[[武士弗拉明戈|武士弗拉明戈]]”的羽佐間正義一角。

2018年7月31日，在推特上宣布離開所屬事務所[[SPACE_CRAFT|SPACE CRAFT]]並刪除官方推特帳號<ref>{{Cite news|url=https://www.oricon.co.jp/news/2116693/full/|title=声優・増田俊樹、事務所退社を発表　ツイッターアカウントは7月31日で閉鎖|newspaper=ORICON NEWS|accessdate=2018-08-01|language=ja}}</ref>。

2018年8月1日，開設個人推特帳號<ref>{{Cite web|url=https://twitter.com/ToshikiMasuda38|title=増田俊樹 (@ToshikiMasuda38)|accessdate=2018-08-01|author=|date=|work=twitter.com|publisher=|language=en}}</ref>。

2018年11月1日，在推特上宣布加入事務所トイズファクトリー<ref>{{Cite web|url=https://twitter.com/ToshikiMasuda38/status/1057874926760210432|title=増田俊樹(@ToshikiMasuda38)在推特上的推文}}</ref>，並宣布將於2019年春天發行首張個人CD。 
<ref>{{Cite web|url=https://twitter.com/ToshikiMasuda38/status/1057876015031103488|title=増田俊樹(@ToshikiMasuda38)在推特上的推文}}</ref>

於2019年3月6日發行首張個人EP「This One」。

== 人物 ==
興趣是遊戲、料理、邊走邊吃。特技是遊戲、游泳。家庭成員有父親、母親、弟弟。

喜歡的食物是[[拉麵|拉麵]]、[[義大利麵|義大利麵]]之類的麵類。討厭的食物是生的蔬菜。以前也不喜歡[[芥茉|芥茉]]和[[番茄汁|番茄汁]]、但現在已經能喝番茄汁和蔬菜汁了。小黃瓜到現在還是無法喜歡。

因為喜歡貓，會在部落格上傳路邊所見到的貓的照片。喜歡的漫畫有[[荒川弘|荒川弘]]的『[[鋼之鍊金術師|鋼之鍊金術師]]』和[[鈴木央|鈴木央]]的作品等等。

原本想成為聲優，後來受到專門學校時代關係良好的前輩影響，也對舞台抱有興趣。

和在radio節目中共同出演的[[細谷佳正|細谷佳正]]有廣島出身、B型、『[[網球王子|網球王子]]』的出演者（細谷是動畫、增田是音樂劇）等共通點。

声優業的老師是[[速水獎|速水獎]]。在聲優出道前就受他指導、在速水的企劃（[[Super_Stylish_Doctors_Story|S.S.D.S]]等等）出演。

曾在推特上表示過喜歡神靈類的東西和參訪能量景點(パワースポット)。
