'''黃旭熙'''（{{lang-en|Wong Yuk Hei, Lucas}}，{{bd|1999年|1月25日|catIdx=W黃}}) 藝名： '''Lucas'''（{{lang-ko|루카스}}），为在[[中國大陸|中國大陸]]及[[韓國|韓國]]发展的男偶像艺人，Rapper。2018年以韓國[[男子組合|男子組合]][[NCT|NCT]]出道；2019年以中國大陸男子組合[[威神V|威神V]]出道。中泰混血兒，是第一位在SM娛樂出道的香港人。

==人物簡介==
1999年1月25日生於[[香港|香港]]。中泰混血兒，父親是香港人，母親則是泰國人，家裏有一名弟弟。在前往韓國當練習生前，曾於[[東華三院邱金元中學|東華三院邱金元中學]]就讀。2015年，黃旭熙參加了[[SM娛樂|SM娛樂]]的全球選秀（Global Audition），當場就被工作人員選中，從而成為[[SM娛樂|SM娛樂]]旗下的歌手練習生。<ref>{{Cite web |url = https://www.hk01.com/韓迷/82790/sm娛樂首位香港藝人lucas黃旭熙-最愛exid-hani金所炫極有異性緣 |title = SM娛樂首位香港藝人Lucas黃旭熙  |date = 2017-04-06 |language = zh-hk |accessdate = 2019-03-18 }}</ref> 

===[[SM_ROOKIES|SM ROOKIES]]時期===
*2017年4月6日，以艺名Lucas通过[[SM娛樂|SM娛樂]]预备明星组合[[SM_ROOKIES|SM ROOKIES]]，正式公开。<ref>{{Cite web |url = https://www.hk01.com/韓迷/82790/sm娛樂首位香港藝人lucas黃旭熙-最愛exid-hani金所炫極有異性緣 |title = SM娛樂首位香港藝人Lucas黃旭熙  |date = 2017-04-06 |language = zh-hk |accessdate = 2019-03-18 }}</ref> 
*2017年4月7日，參演[[NCT|NCT]]組合成員[[Ten_(歌手)|Ten]]《夢中夢》的歌曲MV。

===加入[[NCT|NCT]]出道===
*2018年1月30日，[[SM娛樂|SM娛樂]]發布預告片《NCT 2018 Yearbook》，正式加入[[NCT|NCT]]大型男子組合。<ref>{{Cite web |url = https://hk.ulifestyle.com.hk/topic/detail/204667/第4位港產韓星出道-長腿混血小鮮肉lucas做exo師弟 |title = 第4位港產韓星出道 長腿混血小鮮肉Lucas做EXO師弟 |date = 2018-02-01 |language = zh-hk |accessdate = 2019-03-18 }}</ref>
*2018年2月18日，隨子團體[[NCT_U|NCT U]]發布《BOSS》MV，並於22日在[[Mnet|Mnet]]音樂節目《[[M_Countdown|M Countdown]]》中表演初舞台，正式出道。
*2018年3月14日，隨組合[[NCT|NCT]]發行首張正規專輯《[[NCT_2018_Empathy|NCT 2018 Empathy]]》。
*2018年3月21日，出演MBC every1綜藝節目《[[一周的偶像|一周的偶像]]》，這是黃旭熙出道後首次參演綜藝節目。
*2019年1月17日，加入組合的中國分隊[[威神V|威神V]]推出首張數碼專輯《The Vision》，正式在中國出道。<ref>{{Cite web |url = http://ent.sina.com.cn/y/yrihan/2018-12-31/doc-ihqhqcis1883203.shtml |title = 男子偶像团体威神V确认一月出道 SM公司制作 |date = 2018-12-31 |language = zh |accessdate = 2019-03-18 }}</ref>
*2019年2月11日，宣布加盟[[浙江衛視|浙江衛視]]《[[奔跑吧|奔跑吧]]》，成為該節目第三季全新MC。<ref>{{Cite web |url = https://k.sina.com.cn/article_6896791707_19b14b09b00100h1cv.html |title = 威神V黃旭熙正式加入浙江衛視《奔跑吧》mc陣容 |date = 2019-02-11 |language = zh |accessdate = 2019-03-18 }}</ref>
