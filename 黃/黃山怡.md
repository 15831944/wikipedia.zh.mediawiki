'''黃山怡'''（{{lang-en|'''Kandy Wong Shan Yee'''}}，{{bd|1987年|10月16日<ref>電視廣播有限公司：[http://artiste.tvb.com/main/artistedetail/kandywong 女藝人 '''黃山怡'''基本資料]</ref><ref>{{cite news|url=http://news.mingpao.com/pns/dailynews/web_tc/article/20161120/s00016/1479578048833|title=告別小女孩 糖妹想放棄糖妹|agency=明報|date=2016-11-20}}</ref>||}}），藝名'''糖妹'''，香港歌手、演員及電視主持，是香港二人流行音樂組合[[糖兄妹|糖兄妹]]的主音歌手。現時為[[無綫電視|無綫電視]]基本藝人合約女藝人。

== 簡歷 ==
黃山怡原名'''黃麗怡'''（{{lang-en|Candy Wong Lai Yee}}），[[香港|香港]]出生，籍貫[[廣東省|廣東]]，她曾在網絡用原名「黃麗怡」上載自己的歌曲《熱戀，之後》<ref>{{Youtube|v-p26QFjN4Q|糖兄妹 熱戀之後MV}}</ref>。

糖妹自高中開始不斷參加各類歌唱比賽和參加樂隊演出，畢業後曾任職藝人助理、和音伴唱、唱片公關等工作，并經朋友介紹認識當時的[[吉他手|吉他手]]和樂隊主音[[潘雲峰|潘雲峰]]（糖兄）。2007年二人決定組成組合「[[糖兄妹|糖兄妹]]」，及後成為[[旺角|旺角]][[朗豪坊|朗豪坊]]商場每星期的駐場表演單位，持續至2010年12月。2008年，在一次在朗豪坊的演出中，[[維高文化|維高文化]]及[[蘇打綠|蘇打綠]]經理人監製林暐哲賞識其歌曲和表現，同年簽約維高文化。及後獲機會參與[[香港|香港]]和[[台灣|台灣]]的大型音樂節，包括台灣《[[春天吶喊|春天吶喊音樂節]]》。2009年糖兄妹在台灣錄製了第一張唱片《[[我最愛糖兄妹|我最愛糖兄妹]]》。

2012年，糖妹與[[無綫電視|無綫電視]]簽訂三年藝員合約。而近年為[[無綫電視|無綫電視]]監製[[陳維冠|陳維冠]]的常用演員。

2013年，她在[[微博|微博]]宣布自己助養了一個男孩。<ref>[http://weibo.com/p/1005051712349461/weibo?profile_ftype=1&key_word=%E5%85%92%E5%AD%90&is_search=1#_0 糖妹微博]</ref>

== 轶闻 ==
黄山怡的樣貌一直被指酷似香港著名女歌手[[陳慧嫻|陳慧嫻]]，加上經常表演及翻唱陳慧嫻的經典歌曲，故被傳媒冠以綽號「翻版[[陳慧嫻|陳慧嫻]]」<ref name="orientaldaily1210">{{cite web|url=http://orientaldaily.on.cc/cnt/entertainment/20111210/00282_038.html|title=糖妹人矮志氣高|date=2011-12-10|publisher=東方日報}}</ref>，出道初期曾經多次被認錯為陳慧嫻<ref>[http://www.orientalsunday.hk/2011/09/02/%e8%ae%80%e8%80%85%e5%a0%b1%e6%96%99%ef%bc%9a%e4%b8%8a%e5%80%8b%e6%9c%88%e6%88%91%e5%96%ba%e9%8a%85%e9%91%bc%e7%81%a3%e5%90%8c%e5%80%8b%e6%9c%8b%e5%8f%8b%e9%a3%9fdinner%e8%aa%a4%e5%b0%87%e7%b3%96/ 讀者報料：上個月我喺銅鑼灣同個朋友食dinner,誤將糖妹認錯咗係陳慧嫻…..]</ref>。

2010年8月22日糖妹在[[新浪微博|新浪微博]]上傳了一張和陳慧嫻《亞細亞巡迴演唱會佛山站》宣傳廣告的合照<ref>[http://weibo.com/1712349461/Bh0lnYfnrB 宣傳廣告的合照]</ref>。另外，有很多人認為「93[[紐約嫻情|紐約嫻情]]」音樂特輯，1987年「[[貪貪貪|貪貪貪]]」MV及[[歡樂天地|歡樂天地]]廣告<ref>{{土豆網|h2wrBCIxybg|被指為非常相似的1987年歡樂天地廣告}}</ref>中的陳慧嫻和現時的糖妹非常相似。連糖妹也笑言自己與慧嫻似「失散姊妹」<ref>[http://the-sun.on.cc/cnt/entertainment/20111023/00470_085.html 樂瞳糖妹愛情0分] 2011-10-23</ref>，要靠面上的癦來分辨<ref>[http://orientaldaily.on.cc/cnt/entertainment/20120716/00282_013.html?pubdate=20120716 糖妹自嘲唔靚似哈比人] 東方日報 2012-07-16</ref>。後糖妹在2011年[[歡樂滿東華|歡樂滿東華]]活動時以陳慧嫻《[[反叛|反叛]]》大碟造型獻唱《[[跳舞街|跳舞街]]》<ref>{{土豆網|Gt9zFthTxnY|東華2011 跳舞街}}</ref>，聲線相似程度令很多人嘖嘖稱奇<ref>{{土豆網|AXjVlVvamMI|寶麗金十五週年演唱會1986 跳舞街}}</ref><ref>{{土豆網|tZ3_bT3Bau8|在灣仔華潤大廈錄製的日本節目}}</ref>。

2012年糖妹在[[玩轉三周1/2|玩轉三周1/2]]之玩轉金曲頒獎典禮以陳慧嫻《[[變變變|變變變]]》大碟造型獻唱《[[貪貪貪|貪貪貪]]》<ref>[http://v.youku.com/v_show/id_XNDM3MjQwODky.html 糖妹玩轉三周1/2之玩轉金曲頒獎典禮]</ref>。2013年糖妹在無綫劇《[[女人俱樂部|女人俱樂部]]》戴上及肩短假髮飾演1980年代學生，被傳媒指極像1980年代的陳慧嫻<ref>[http://ent.sina.com.hk/news/-6-1003564/1.html 糖妹清純學生Look激似陳慧嫻] {{webarchive|url=https://web.archive.org/web/20131110040613/http://ent.sina.com.hk/news/-6-1003564/1.html |date=2013-11-10 }}</ref>。2014年6月19日糖妹出現「[[Back_to_Priscilla_30th_Anniversary_Concert_2014|Back to Priscilla 30th Anniversary Concert 2014]]」和陳慧嫻握手<ref>[http://hk.apple.nextmedia.com/enews/realtime/20140620/52606181 人生何處不相逢……當慧嫻遇上少女版慧嫻]</ref><ref>[http://instagram.com/p/pgA6oik4yS/?modal=true 被傳媒指為"世紀一握"的視頻]</ref>與在後台合照<ref>{{Cite web |url=https://www.facebook.com/sugarclubkandy/photos/pb.1393551037534441.-2207520000.1403588443./1517559961800214/ |title=糖妹Kandy facebook專頁帖文 |quote=終於比我見倒真人喇！ 我等左五年喇 慧嫻姐姐好好人呀！ |date=2014-06-20}}</ref>。

2015年糖妹在電視節目《[[Sunday靚聲王|Sunday靚聲王]]》第3集裡唱出陳慧嫻的著名情歌《[[夜機|夜機]]》，第22集唱出陳1984年的大熱歌曲《[[玻璃窗的愛|玻璃窗的愛]]》，同集糖妹與全場歌手合唱《[[千千闕歌|千千闕歌]]》。11月25日糖妹在[[instagram|instagram]]上傳了一張《陳慧嫻Priscilla-ism演唱會》被網民改圖宣傳廣告<ref>[https://www.facebook.com/sugarclubkandy/photos/a.1393925457496999.1073741828.1393551037534441/1732620273627514/?type=3 《陳慧嫻Priscilla-ism演唱會》宣傳廣告改圖]</ref>，陳慧嫻亦轉載了糖妹的instagram<ref>[https://www.facebook.com/PriscillaChanWaiHan/photos/a.437364713024319.1073741827.437359673024823/934001946693924/?type=3&theater 陳慧嫻轉載]</ref>，但糖妹要拍攝無綫劇《[[幕後玩家|幕後玩家]]》，無法在《陳慧嫻Priscilla-ism演唱會》出現。2016年糖妹被上海東方衛視邀請上《[[隱藏的歌手|隱藏的歌手]]》節目模仿陳慧嫻，本來有機會與陳慧嫻同台合唱，但糖妹因工作撞期出現不到節目<ref>{{Cite news |url=https://hk.entertainment.appledaily.com/enews/realtime/article/20160716/55367893 |title=前男友狂追十優港姐 懶理舊愛是非 糖妹：不如講吓我近況  |date=2016-07-16 |work=蘋果日報}}</ref>。

在2017年7月播出的其中一集《[[流行經典50年|流行經典50年]]》，糖妹翻唱[[鄭秀文|鄭秀文]]歌曲《如何掉眼淚》<ref>流行经典50年 - 糖妹 - 《如何掉眼淚》（原唱：鄭秀文）（[https://www.bilibili.com/video/av12535125/ 哔哩哔哩]、[http://www.iqiyi.com/w_19rv9ptlyd.html 爱奇艺]）</ref>，唱功表現獲不少網民讚賞，大獲好評<ref>{{Cite web |url=https://bka.mpweekly.com/focus/local/20170717-69796 |title=【別忘記她是歌手】糖妹唱《如何掉眼淚》 網民大讚眼前一亮 |date=2017-07-17 |publisher=明周娛樂}}</ref><ref>{{Cite web |url=http://www.bastillepost.com/hongkong/article/2232691-%E7%BF%BB%E5%94%B1%E7%8D%B2%E5%A5%BD%E8%A9%95-%E7%B3%96%E5%A6%B9%E9%96%8B%E5%BF%83%E5%B0%8D%E9%9F%B3%E6%A8%82%E4%BA%8B%E6%A5%AD%E7%9A%84%E9%BC%93%E5%8B%B5 |title=翻唱獲好評 糖妹開心:對音樂事業的鼓勵 |date=2017-07-20 |publisher=巴士的報 www.bastillepost.com}}</ref>。糖妹在訪問中稱對於翻唱被大眾稱讚感到開心，透露該曲是自己喜歡的經典歌曲，當被TVB邀請上該節目唱歌時，便馬上挑了這首曲了<ref>[https://www.youtube.com/watch?v=58CwqLAbUCw&t=109s 拒絕停留初嘗改變 糖妹學會聆聽自己]</ref>。
