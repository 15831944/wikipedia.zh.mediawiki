'''黃口攀蜥'''（{{Lang-xm|Diploderma polygonatum xanthostomum}}）又被稱為'''琉球龙蜥台北亚种'''<ref><!--本來源有些數據已過時-->{{cite web |title = 琉球龙蜥台北亚种 |url = http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=4400 |accessdate = 2018-04-03 |author = 中国科学院微生物研究所 |website = 中國動物物種編目數據 |publisher = 中国科学院动物研究所 |archiveurl = https://web.archive.org/web/20160305081533/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=4400 |archivedate = 2016-03-05 |deadurl = yes}}</ref>，俗名'''肚定'''（台語）、'''竹虎'''<ref name="ntu">{{cite web |title = 黃口攀蜥 |url = http://archive.zo.ntu.edu.tw/rept_index.asp?rept_id=R0112 |website = 臺灣大學數位動物博物館 |author = [[國立台灣大學|國立台灣大學]]生態學與演化生物學研究所 |archiveurl = https://web.archive.org/web/20180402164043/http://archive.zo.ntu.edu.tw/rept_index.asp?rept_id=R0112 |archivedate = 2018-04-02 |deadurl = yes |access-date = 2018-04-02 }}</ref>，為[[台灣|台灣]]特有亞種[[蜥蜴|蜥蜴]]。
==描述==
其特徵為口內側及舌頭為黃色，體型為台灣目前攀蜥屬中最小的，體長不會大於23公分。<ref name = "ntu"/>體色多變，體背以褐色及綠色為主，體側有由菱斑連貫之黃綠色縱帶，體色會隨所處環境改變。雄性體型較大，喉部有較明顯之橘色斑塊及鬛鱗。<ref name="taieol-72196"/>

外型與斯文豪氏攀蜥十分類似.但是黃口攀蜥嘴部有一道黃白色條紋可以區別。

==習性==
食物為昆蟲及一些小型無脊椎動物。<ref name = "ntu"/>生殖期以夏天為主，一窩可產4-6顆卵。尾巴不會自割。

==棲地==
黃口攀蜥的[[棲地|棲地]]分佈於台灣中部以北、海拔1000公尺以下的山區。<ref name = "ntu"/>
