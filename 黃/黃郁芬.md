'''黃郁芬'''（{{bd|1990年|3月17日|||catIdx=黃}}）（{{lang-en|Huang Yu-Fen}}），[[臺灣|臺灣]][[時代力量|時代力量]]籍黨員女性[[政治人物|政治人物]]，[[臺南市|臺南市]]人，現任[[臺北市議員|臺北市議員]]，立法委員[[林昶佐|林昶佐]]國會辦公室助理<ref>{{cite news |title=北市士北區議員激戰 時力推太陽花發言人參選 |url=https://www.mirrormedia.mg/story/20180414inv001/ |agency=鏡週刊 |date=2018-04-14}}</ref>，[[國立清華大學|國立清華大學]]社會學研究所研究生，[[黑色島國青年陣線|黑色島國青年陣線]]創始成員之一。因擔任[[太陽花學運|太陽花學運]]前期發言人，一些網友比喻她為「臺灣小[[柴玲|柴玲]]」<ref>{{Cite web |url = http://news.cts.com.tw/cts/general/201403/201403201394360.html |title = 夜占立法院 "黑色島國"一戰成名 |author = 蔡尚樺和宋佾璋 |date = 2014年3月20日 |publisher = [[華視新聞|華視新聞]] |language = zh-hant |accessdate = 2014年3月28日 |quote =  }}</ref>。從2013年至2014年期間，已參加數十場臺灣社會運動。2018年5月11日，代表時代力量參選臺北市議員。2018年11月，以13213票與[[林穎孟|林穎孟]]及[[林亮君|林亮君]]一同當選時代力量臺北市議員。

== 生平 ==
=== 社會運動 ===
黃郁芬其爺爺和奶奶是[[臺南市|臺南市]][[後壁區|後壁區]]的農民，2008年時考取了[[國立清華大學|國立清華大學]]人文社會學系。之後黃郁芬表示在就讀[[國立清華大學|國立清華大學]]時，由於[[姚人多|姚人多]]在必修課「現代國家與社會的形成」提到：「今天我們這些學生能夠在清大上課，不是因為我們自己，而是那些不能進來清大的人，因為全台灣納稅人繳的稅金，才能讓你們在清大享有這些資源。<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20140319/363117/ |title = 318佔領立院　黃郁芬盼喚醒民眾 |author = 施昂強和邱中岳 |date = 2014年3月19日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2014年3月28日 |quote =  }}</ref>」使得她開始嘗試以客觀的角度看待生活每件事，進而發現生活中許多理所當然的事情都是來自社會各處所累積出來的，而她也開始思考學生能夠從事那些工作幫助社會<ref>{{Cite web |url = http://www.stormmediagroup.com/opencms/news/detail/91349711-af1a-11e3-975a-ef2804cba5a1/?uuid=91349711-af1a-11e3-975a-ef2804cba5a1 |title = 318人民佔領立院行動 清大教授姚人多啟蒙 |author = 蔡慧貞和姚惠珍 |date = 2014年3月19日 |publisher = [[風傳媒|風傳媒]] |language = zh-hant |accessdate = 2014年3月28日 |quote =  |archive-url = https://web.archive.org/web/20140328101533/http://www.stormmediagroup.com/opencms/news/detail/91349711-af1a-11e3-975a-ef2804cba5a1/?uuid=91349711-af1a-11e3-975a-ef2804cba5a1 |archive-date = 2014年3月28日 |dead-url = yes }}</ref>。

之後黃郁芬先是參與了[[國立清華大學|國立清華大學]]舉辦的學生運動，之後陸陸續續參與了反對[[旺旺中時併購中嘉案|旺旺中時併購中嘉案]]與[[台灣壹傳媒賣盤案|台灣壹傳媒賣盤案]]引發的[[反媒體壟斷運動|反媒體壟斷運動]]、以及因為[[大埔事件|大埔事件]]而發起的佔領[[中華民國內政部|中華民國內政部]]行動<ref>{{Cite web |url = http://www.lihpao.com/?action-viewnews-itemid-132513 |title = 占領內政部 拆政府運動不停歇 |author = 郭琇真 |date = 2013年8月19日 |publisher = 《[[台灣立報|台灣立報]]》 |language = zh-hant |accessdate = 2014年3月26日 |quote =  }}</ref>，但是她在接受採訪時也提到其父母一開始並沒有完全支持其參與社會運動<ref>{{Cite web |url = http://news.ebc.net.tw/apps/newsList.aspx?id=1395488898 |title = 緊握主導江揆麥克風！ 　黃郁芬展強勢 |author =  |date = 2014年3月22日 |publisher = [[東森新聞台|東森新聞台]] |language = zh-hant |accessdate = 2014年3月28日 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20140328121653/http://news.ebc.net.tw/apps/newsList.aspx?id=1395488898 |archivedate = 2014年3月28日 }}</ref>。2012年自人文社會學系畢業後黃郁芬先是考取[[國立清華大學|國立清華大學]]台灣文學研究所，隔年則是考取[[國立清華大學|國立清華大學]]社會學研究所。而在2013年7月31日《兩岸服務貿易協議》舉辦公聽會時，她則擔任了由[[國立清華大學|國立清華大學]]、[[國立臺灣大學|國立臺灣大學]]、[[輔仁大學|輔仁大學]]、[[國立臺北教育大學|國立臺北教育大學]]、[[國立臺北藝術大學|國立臺北藝術大學]]、[[世新大學|世新大學]]、[[中原大學|中原大學]]等學校共同號召的「佔領立院，奪回未來─青年反服貿行動」之發起人<ref>{{Cite web |url = http://blog.twimi.net/2013/07/blog-post_5083.html |title = 占領立法院 奪回未來-青年反服貿行動 |author = 蔡慧貞和姚惠珍 |date = 2013年7月31日 |publisher = 獨立媒體 |language = zh-hant |accessdate = 2014年3月28日 |quote =  }}</ref>。同年8月[[魏揚|魏揚]]發起組織臺灣學生運動團體[[黑色島國青年陣線|黑色島國青年陣線]]時，黃郁芬也成為[[黑色島國青年陣線|黑色島國青年陣線]]的創始成員之一，並且在往後兩年參加數十場社會運動。
