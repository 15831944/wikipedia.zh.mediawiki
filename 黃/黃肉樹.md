'''小梗木薑子'''（[[学名|学名]]：{{lang|la|'''''Litsea hypophaea'''''}}）为[[樟科|樟科]][[木薑子屬|木薑子屬]]下的一个种，由於其木材的顏色而有'''黃肉樹'''、'''小梗黃肉楠'''、'''鐵屎楠'''等別名。<ref name = "劉業經">劉業經 等. 台灣樹木誌. 台灣: 國立中興大學農學院出版委員會. 1994: 117-119 </ref>　為[[臺灣|臺灣]]特有種<ref>特有生物研究保育中心 台灣野生植物資料庫</ref>，屬於中型常綠[[喬木|喬木]]，分布在全臺海拔1000m以下的低中海拔山區。<ref name = "黃增泉">黃增泉等（1984年）：《台灣植物誌》，第二版，第471頁，台灣：國科會。</ref>　
[[File:Litsea_hypophaea_Hayata.jpg|thumb]]
== 形態特徵 ==
=== 莖 ===
枝條直而細，成熟後木質化，無毛，樹皮褐色，表面帶有些許突起。嫩枝淡綠色，密佈細小的微毛，照光會呈現淡黃色[[金屬光澤|金屬光澤]]。<ref name = "黃增泉"/>
=== 葉 ===
葉互生，倒卵形或披針形，長 5-9 公分，寬 2-3 公分，先端鈍，基部楔形。<ref name = "黃增泉"/>近軸面顏色深於遠軸面，有光澤，羽狀脈，中肋與一級側脈為黃色，在葉背突出，長有些許的絨毛，側脈至葉緣前弧曲，在近基部側脈平行，為本種辨識特徵之一。葉芽剛發出時具絨毛，發展為幼葉時多往遠軸面反捲，呈披針形，黃綠色中帶紅色色澤，密佈細小的微毛，照光呈現淡黃色金屬光澤。葉柄短，小於1cm，整體葉部多叢生在枝條末端。<ref name = "劉業經"/>
=== 花 ===
單性花，[[雌雄異株|雌雄異株]]，春季開花。雌雄均為繖形花序，[[花梗|花梗]]短，約3mm，簇生在葉腋，[[苞片|苞片]]圓鈍，對生組成總苞，外側披毛。
雌花偏白色，[[雄蕊|雄蕊]]退化，[[柱頭|柱頭]]明顯，在頂端分岔，呈現白色，披有短毛。
雄花淡黃色，[[花被片|花被片]]兩輪，6片，[[雌蕊|雌蕊]]退化、不明顯，[[雄蕊|雄蕊]]明顯且四輪，[[花絲|花絲]]具毛，[[花藥|花藥]]4室。<ref>昆明植物研究所. Litsea hypophaea Hayata. 《中國高等植物資料庫全庫》. 中國科學院微生物研究所.</ref>

=== 果實 ===
[[果實|果實]]橢圓形[[核果|核果]]，表面無毛，未成熟時為綠色，有白色星狀斑，成熟後轉為紫黑色。果實基部具杯斗，外型圓錐狀，薄質，如雞尾酒杯。<ref>劉業經 等. 台灣樹木誌. 台灣: 國立中興大學農學院出版委員會. 1994: 117-119 </ref>　

== 生態 ==
分佈在全臺1000m以下低中海拔的[[闊葉林|闊葉林]]中，多生長在乾燥的向陽山坡地。在夏轉秋9-10月時開花，冬至春為果期。
哺乳類、山鳥等野生動物會取食小梗木薑子的果實。<ref name = "林務局">林務局台灣樹木解說</ref>

== 民生用途 ==
小梗木薑子的木材質地緻密、堅韌，多作為小型器具用材、器具柄，或供作薪材。<ref name = "林務局"/>
