'''黃家駒'''（'''Wong Ka Kui'''，{{bd|1962年|6月10日|1993年|6月30日|}}），[[香港|香港]]殿堂級音樂家、[[搖滾樂|搖滾樂]]歌手、詞曲創作家及唱片監製，華語樂壇公認的一代巨星及搖滾樂傳奇人物。

<ref>{{cite web|url=http://app1.rthk.org.hk/php/tvarchivecatalog/episode.php?progid=556&tvcat=2|title=不死傳奇 2007/2008}}</ref>在其搖滾樂隊[[Beyond|Beyond]]中擔任作曲、填詞、節奏吉他和主唱，從而成為樂隊的靈魂人物。<ref> [http://www.mingpaoweekly.com/eternalstars/wongkakui/index.php 黃家駒]特輯資料來源：《明報周刊》及《明日風尚》</ref>他的吉他技術精湛，極具音樂才華，<ref>[[文匯報|文匯報]]第十五版，1993年7月1日</ref>當中主音歌手和節奏結他手的身份最廣為人知。身為樂隊領袖，其擅長創作，樂隊大部分的歌曲是由他創作，也因其作品大幅度提高了樂隊的知名度。他以獨特的沙啞聲線、巧妙的尾音處理見稱。

Beyond在1983至1993年发表的大部分作品均为黄家驹主唱及作曲，其創作《大地》、《真的爱你》、《Amani》、《光辉岁月》、《长城》、《海阔天空》等歌曲家傳戶曉，歌詞涉及不少社會議題的元素，與當時香港以愛情為主的流行歌曲不一樣，他的歌迷除了香港本土外，其餘遍布[[兩岸三地|兩岸三地]]及海外華人地區；當中《海闊天空》、《長城》、《光輝歲月》等更成為香港[[遊行示威|遊行示威]]當中經常播放的歌曲，甚至在廣州、澳門、台灣、馬來西亞的社會運動亦有示威者高唱。1993年發表的《海闊天空》更被認為代表香港人近年對社會的無力感與迷思；2008年5月，《海闊天空》歌曲重新填上新的國語歌詞改編成《承諾》，並於「[[演藝界512關愛行動|演藝界512關愛行動]]」由[[兩岸三地|兩岸三地]]群星演唱，作為[[汶川大地震|汶川大地震]]賑災的主題曲，更顯黃家駒的大愛精神。

== 生活與事業 ==

=== 早年生活 ===
1962年6月10日生於[[英屬香港|英屬香港]]，祖籍[[廣東|廣東]][[台山市|台山]]。自小與家人居住在[[九龍|九龍]][[深水埗區|深水埗區]][[蘇屋邨|蘇屋邨]]。曾就讀香港四邑商工總會新會商會學校{{noteTag|資料出自Facebook公開群組「李鄭屋村 (七層高世代)」於2017年7月16日發出的貼文以及有關的回應內容。貼文回應提及黃家駒與黃家強在大坑東的香港四邑商工總會新會商會學校讀小學。}}及博允英文中學（昔日位於[[佐敦|佐敦]][[南京街_(香港)|南京街]]30-36號<ref>{{cite news|url=http://www.uwants.com/viewthread.php?tid=5653508&page=34 |title=博允英文中學招生啟示 |publisher=[[華僑日報|華僑日報]]|date=1974年4月25日}}{{zh-hant}}</ref>，已重建為朗逸酒店）。他在初中時受朋友影響愛上[[歐洲|歐洲]]和[[美國|美國]]的[[流行音樂|流行音樂]]，偶像是[[大衛·鮑伊|大衛·鮑伊]]（David Bowie）。

大約在14、15歲那年，無意之中撿得鄰居搬家遺留下的一把舊吉他，<ref name="repeat">Beyond，〈Scene 3 昔日舞曲〉 出自《心內心外》，1988年</ref>從此與這樂器結下不解之緣，其後更經常使用[[Fender_Stratocaster|Fender Stratocaster]]電吉他。

中五畢業後，曾任職辦公室助理、鋁窗、五金、空調、水電工程、電視台佈景員等工作，也曾加入樂隊鼓手[[葉世榮|葉世榮]]任職的德高保險有限公司作保險推銷員。<ref name="repeat"/>

=== 參與成立Beyond ===
1980年，黃家駒經琴行老闆介紹認識[[葉世榮|葉世榮]]，彼此音樂興趣相近，故聯同兩位朋友[[鄧煒謙|鄧煒謙]]及李榮潮組成樂隊，這支樂隊也是[[Beyond|Beyond]]的雛型。Beyond這個名字是第一代主音吉他手鄧煒謙所改，意思是「超越」，葉世榮曾說：「取名Beyond不是要超越別人，而是要超越自己。」1983年，Beyond正式組成；經過幾次人事變動後，1983年底，黃家駒的弟弟[[黃家強|黃家強]]加入成為低音吉他手；1985年，主音吉他手[[黃貫中|黃貫中]]加入。同年，Beyond在[[堅道|堅道]]的[[明愛中心|明愛中心]]自資舉辦《永遠等待演唱會》，翌年再推出盒帶《[[再見理想|再見理想]]》，之後在專業經理人[[陳健添|陳健添]]（Leslie Chan）的協助下，從[[地下音樂|地下音樂]]正式進入[[商品化|商業]]流行樂壇。

經過多年努力，Beyond成為香港最舉足輕重的樂隊。1988年，終於憑專輯《[[秘密警察_(專輯)|秘密警察]]》內《大地》一曲奪得[[十大勁歌金曲|十大勁歌金曲]]，為樂隊獲得首個電子傳媒獎項；1989年，由黃家駒作曲，[[梁美薇|小美]]填詞，以歌頌母親為題的《真的愛你》大熱，成為家傳戶曉的頌母歌曲，更奪得[[十大勁歌金曲|十大勁歌金曲]]及[[十大中文金曲|十大中文金曲]]兩大獎項；1990年，由黃家駒作曲及填詞，以歌頌當時剛出獄的[[南非|南非]]黑人民權領袖[[曼德拉|曼德拉]]為題材的《光輝歲月》大熱，奪得十大勁歌金曲獎，黃家駒亦憑此曲奪得「最佳填詞獎」。

當時Beyond發表的作品大部分均由Beyond自己包辦作曲、填詞及演唱（但部分歌曲交由填詞人填寫，如劉卓輝），除了經樂隊發表作品，黃家駒偶爾也為其他歌手作曲，曾經合作的歌手包括：[[許冠傑|許冠傑]]、[[譚詠麟|譚詠麟]]、[[麥潔文|麥潔文]]、[[彭健新|彭健新]]等。後來，Beyond後期所屬唱片公司（華納）的高層抱著「好作品應留給自己，樂迷要聽Beyond，就應買Beyond的唱片」的思想，反對Beyond把自己作品交給別人演唱。<ref>鄺國強、陳頌紅，〈光輝歲月－黃家駒的最後專訪〉，出自《壹週刊》174期特刊。1993年7月9日：第7頁</ref>

黃家駒是Beyond樂隊的靈魂人物，不時包辦樂隊歌曲的作曲及填詞。他的性格敢言，不時發表大膽的言論，例如曾說過：「'''香港沒有樂壇，只有娛樂圈。'''」批評當時香港娛樂圈趨勢，令香港樂壇沒有真正獨立自主的空間，因而有時令娛樂圈中人不滿。然而其果斷、健談的個性，也令他成為樂隊的發言人及領導人，帶領Beyond事業闖進多個高峰。

=== 日本發展時期 ===
1991年，Beyond於[[香港體育館|香港體育館]]舉辦五場《生命接觸演唱會》，成為繼[[太極樂隊|太極樂隊]]後第二隊於香港體育館舉行演唱會的搖滾樂隊。由於對香港樂壇感到失望，加上希望樂隊能衝出香港，故在九十年代初期Beyond決定發展東南亞、台灣、中國大陸、日本等亞洲市場。此時，日本經理人公司對Beyond感到興趣，與他們簽約，帶Beyond往日本發展。同年年尾，Beyond轉投[[華納唱片_(香港)|華納唱片]]，開始Beyond的華納時代。黃家駒表示，他們希望有更大自由度去玩音樂，以擺脫香港樂壇的束縛及題材限制。

1992年，Beyond開始長居日本，接下大大小小的工作，其中仍需參與Beyond最不喜歡的遊戲節目（黃家駒後來更於參與錄製遊戲節目時意外喪命）。Beyond起初以為日本比香港有更大的自由度去玩音樂，誰知依然要裝「鄰家男孩」去得到日本樂迷的關注。雖然他們在日本的生活孤獨，但在那裡能學習得到跟香港不一樣的音樂，這成為了Beyond在日本玩音樂最大的推動力。此時的Beyond，在日本的知名度已逐漸上升，同時亦吸納到一班日本歌迷。

期間，黃家駒抽空回港主演1992年寫實電影《[[籠民|籠民]]》；該片在各個頒獎禮中屢獲殊榮，包括[[1993年香港电影金像奖|第12届香港金像奖]]最佳影片獎。

=== 意外逝世 ===
1993年6月24日凌晨1時（日本時間），為宣傳即將發行的日語唱片，樂隊應邀到[[富士電視台|富士電視台]]位於[[河田町|河田町]]的4號錄影室拍攝遊戲節目《{{link-ja|小内小南的 想做什麼 就做什麼|ウッチャンナンチャンのやるならやらねば!}}》，遊戲進行了15分鐘便發生意外，黄家駒在狹窄並沾滿水渍的台上奔跑時不慎滑倒，強大衝力使身旁3塊佈景板的固定裝置脫落，佈景板鬆開，黃家駒與身旁的節目主持[[內村光良|內村光良]]一同翻落2.7米的台下<ref>[http://bagua.ifensi.com/article-168306.html 黄家驹從2.7米高台上墜下 遗憾告别人世] {{webarchive|url=https://web.archive.org/web/20110713003446/http://bagua.ifensi.com/article-168306.html |date=2011-07-13}}</ref>。黃家駒頭部朝下摔落，後腦<ref>「家駒遺體曾注射防腐劑」，[[文匯報|文匯報]]第十五版，1993年7月6日</ref>首先着地，隨即[[昏迷|昏迷]]。內村光良則[[胸部|胸部]]着地，僅受輕傷。鑑於黃家駒頭部嚴重受創，院方不敢輕舉妄動，因此並未施以手術清走腦內[[瘀血|瘀血]]。

6月26日晚上，樂迷聚集[[香港商業電台|香港商業電台]]停車場空地舉行祝禱會，祈求黃家駒盡快康復。留醫期間，Beyond的日本經理人公司請了一名叫辛勇的中國[[氣功|氣功]]師為他「發功」治療<ref>[http://www.youtube.com/watch?v=k14yKcNwu-0&feature=related 家駒出事新聞]，[[亞洲電視|亞洲電視]]及[[電視廣播有限公司|無綫電視]]，1993年6月（第56秒開始講述氣功師醫治家駒）</ref><ref>「北京氣功師應召赴日 為黃家駒發功驅瘀血 希望配合日本腦科醫生施手術」，[[文匯報|文匯報]]第二十八版，1993年6月27日</ref>，又處方[[中藥|中藥]]給他服食，惟病況依然無起色（注：早前致電聯絡[[嘉禾電影|嘉禾電影]]公司的[[鄒文懷|鄒文懷]]、[[蔡瀾|蔡瀾]]和[[成龍|成龍]]，希望能找到1985年成龍拍攝電影《龍兄虎弟》中受傷時幫成龍治療腦部的兩位[[南斯拉夫|南斯拉夫]]（今[[塞爾維亞|塞爾維亞]]）的世界級腦科醫生，可惜因當時發生南斯拉夫解體內戰而無法前來日本協助治療黃家駒的腦部），留醫六天後於日本時間1993年6月30日下午4時15分在東京女子醫科大學醫院[[逝世|逝世]]，終年31歲，院方公佈[[死因|死因]]為急性腦膜下血腫、頭蓋骨骨折、腦挫傷及急性腦腫脹<ref>[http://www.youtube.com/watch?v=UuAlNZKeOV4 黃家駒逝世新聞發佈會]，[[富士電視台|富士電視台]]，1993年6月30日</ref>。《小内小南的 想做什麼 就做什麼》也因黃家駒的逝世宣告停播。

7月2日，黃家駒的遺體運回香港，7月4日於[[北角|北角]][[香港殯儀館|香港殯儀館]]設靈。同日晚上香港商業電台於[[紅磡|紅磡]][[高山劇場|高山劇場]]舉行「永遠懷念家駒」悼念音樂會，全場坐滿2600人，飲泣聲此起彼落<ref>[http://www.youtube.com/watch?v=s8OzRj70Pgs&feature=related 靈通天地綫（第二部份）]，[[電視廣播有限公司|無綫電視]]，1993年7月5日</ref>。翌日中午大殮，靈車駛出殯儀館時，聚集在行人路上的數千名年輕歌迷終按捺不住情緒，衝出馬路並撲向靈車<ref>[http://www.youtube.com/watch?v=ZD5LOg3A3NY&feature=related 新聞報道]，[[亞洲電視|亞洲電視]]及[[無綫電視|無綫電視]]，1993年7月5日</ref>，場面一度混亂。遺體最後下葬於[[將軍澳華人永遠墳場|將軍澳華人永遠墳場]]，近500名歌迷到場送別<ref>[http://www.youtube.com/watch?v=XFRkFhrIlVs&feature=related 靈通天地綫]，[[電視廣播有限公司|無綫電視]]，1993年7月5日</ref>，而[[九龍塘|九龍塘]][[省善真堂|省善真堂]]亦安放了其靈位。

== 影響 ==
黃家駒的音樂作品，對香港、台灣、中國大陸、东南亚、日本及世界其他華人地區一代又一代年輕樂迷仍然有著深遠的影響。Beyond其他三名成員繼承其遺志，以樂隊名義推出多張專輯，即使樂隊2005年宣佈解散後，仍然繼續創作音樂。

2008年香港《[[東方新地|東方新地]]》雜誌，一篇《誰是歌神》的文章中稱黃家駒為真正的歌神。文章中的其中一段講述：
{{Cquote|可是世上若沒有黃家駒，就沒有Beyond，因為他是樂隊的創辦者及領導人；也沒有那許多許多經典，因為極大部分歌詞由他所作；當然也沒有香港搖滾樂傳奇，你可能會說若無Beyond可能也有另一樂隊彈出，但根據家駒逝世後的十五年史實證明，沒有。

黃家駒在『詞、曲、唱、奏』四方面俱創意澎湃，他的旋律驚世駭俗，他的吉他新銳激情，他的唱腔自成一格，他的歌詞別開蹊徑……你或許會說有些文法不通、不押韻，但他厲害的地方就在於不用押韻都可以這麼好聽，這是以氣馭歌的強烈風格。<ref>{{cite web|url=http://www.gznf.net/thread-56631-1-1.html|title=谁是歌神？！[转]-娱乐-有话讲-羊城网-互联网+粤文化 -|first=|last=Gznf.net|work=www.gznf.net}}</ref><ref>{{cite web|url=http://blog.yahoo.com/AgnesinWonderland/articles/35765|title=Yahoo|work=Yahoo}}</ref>}}

縱使黃家駒已逝世多年，但未被大眾遺忘，直至現在每年他的生忌和死忌，仍然有大批來自世界各地的樂迷前往[[將軍澳華人永遠墳場|將軍澳華人永遠墳場]]拜祭。

黃家駒創作的《海闊天空》、《光輝歲月》、《Amani》、《大地》、《長城》、《抗戰二十年》等與社會題材的金曲，歌詞內容中不少表達追求自由的理想，如在《光輝歲月》（風雨中抱緊自由）、《海闊天空》（原諒我這一生不羈放縱愛自由）、《Amani》（穿梭天際只想覓自由）等。

2014年9月29日，香港發生[[雨傘運動|雨傘運動]]時，大量群眾高唱《海闊天空》，使此曲成為民運主題曲之一。<ref>[http://hk.apple.nextmedia.com/news/art/20140930/18884413 萬人高歌　旺角海闊天空] 蘋果日報 (香港)2014-09-30</ref>

== 評價 ==
===香港===
著名填詞人[[劉卓輝|劉卓輝]]對黃家駒所發表「香港没有乐坛，只有娱乐圈」一句留下深刻印象，認為這是黃家駒與別不同之處。劉解釋「懂得說這句說話的人並不多，即使懂得說或心裏有同樣想法的人，也沒有膽量表達。黃家駒並非嘩眾取寵，而是在心裏壓抑一段很長時間，禁不住才說」。

===臺灣===
有「華語流行樂教父」之稱的台灣歌手[[羅大佑|羅大佑]]，就曾說過香港沒有真正的音樂人，除了黃家駒。羅大佑亦曾在《黃家駒為什麼會死》這本書中寫到 ：「不會再出一個黃家駒了，這樣的人降臨人世本來就是奇蹟，上帝讓黃家駒下凡，但是凡人沒有珍惜他，反而謾罵他，詛咒他，結果，上帝把黃家駒收回了。上帝不會再派一個音樂天使下凡的。」

== 感情生活 ==
黃家駒的感情生活是樂迷所關心、而他又不想透露的。不過，他可樂意分享一些年代久遠的初戀故事。

黃家駒在10歲那年初次對異性發生興趣，她跟他同住一屋邨，是個上海小姑娘，「我不敢向她表示，便砌一些模型，托隔壁的女孩轉交給她，又常常蹲在她家門口玩。後來，我托這位中間人問她喜不喜歡我，得來的答案是她喜歡了我的『仇人』，自此以後，她對我不瞅不睬，令我很不開心。三年後，有朋友告訴我，隔壁的女孩原來暗戀我，還在上海小妹面前說我壞話，難怪她不跟我說話了。」

到了16歲，黃家駒真正初嘗情滋味，初戀情人是他的同學，「我每天早上在巴士站都見到她，後來托朋友介紹認識，約她去游泳、看電影，漸漸發展至單獨約會。」這段情維持了四年，「是我提出分手的，因為我只愛玩音樂，吊兒郎當，自問很難給她甚麼保障和安全感。她本來不願意分手，但後來認識了別的男孩，分手不到半年就嫁人了。我們的理想與目標不同，她追求的是家庭，我滿足不了她。」放下四年情，總是有點不捨與無奈，但他愈來愈忙，談戀愛變得是一件「奢侈」的事，「我沒有時間認真投入一段感情，只好拍散拖。」

在黃家駒的喪禮上，有三位女性至為矚目——除了在靈堂表現失常的[[林楚麒|林楚麒]]、頭披黑紗的簡小姐（黃家駒的前任女友）外，還有據說是黃家駒的最後一任女友——Rita。Rita是Beyond經理人公司[[Amuse|Amuse]]的員工，與Beyond已相處了年多日子。對於這個「家駒女友」的身分，她卻持絕對否認的態度。但經過年多的共處，他們確實建立了一份互相信任的情誼，而Rita本人又極之欣賞Beyond，所以私底下與家駒是非常好的朋友。<ref> [http://www.mingpaoweekly.com/eternalstars/wongkakui/index.php 我們的永恆巨星特輯 - 黃家駒]特輯資料來源：《明報周刊》及《明日風尚》</ref>

== 專輯 ==
主唱作品以[[Beyond|Beyond]]名義推出。最後參與創作之作品，收錄於粵語專輯《[[樂與怒|樂與怒]]》及國語專輯《[[海闊天空_(Beyond專輯)|海闊天空]]》。

黃家駒去世後，Beyond所屬的唱片公司——[[華納唱片_(香港)|華納唱片]]及[[滾石唱片|滾石唱片]]先後推出致敬專輯，包括：

* 《遙望黃家駒不死音樂精神 - 特別紀念集92-93》（[[華納唱片_(香港)|華納唱片]]，1993年） - 收錄了寫予蔡興麟的歌曲《為了你，為了我》的黃家駒演唱版本；

* 2010年，由[[華納唱片_(香港)|華納]]、[[華星唱片|華星]]呈獻 《獅子山下的傳奇》（5CD + 「明報周刊」 巨星紀念集），收錄連同黃家駒在內的五位傳奇巨星（其餘四位分別是[[梅艷芳|梅艷芳]]、[[張國榮|張國榮]]、[[陳百強|陳百強]]和[[羅文|羅文]]）共90首名曲及隨碟附《[[明報周刊|明報周刊]]》 巨星紀念集（144頁），結集歷年相片、記載巨星片段，讓香港[[粵語流行曲|粵語流行曲]]成為[[歷史|歷史]]和[[文化|文化]]的一部分。
