'''黃韻玲'''（[[英文|英文]]名：'''Kay Huang'''，{{bd|1964年|10月1日|catIdx=Huang黃}}）生于[[台湾|台湾]][[宜兰|宜兰]]，资深[[台湾|台湾]][[音樂|音樂]]制作人、[[女歌手|女歌手]]、[[女演員|女演員]]，素有「音樂精靈」之美譽。

== 生涯简介 ==

黄韵玲生于1964年，籍贯[[台灣|台灣]][[宜蘭縣|宜蘭縣]]。

1979年，黃韻玲以14歲之姿報名參加第三屆[[金韻獎|金韻獎]]歌唱比賽，與[[許景淳|許景淳]]、黃珊珊、張瑞薰合組重唱組合「四小合唱團」，並獲得優勝。

1980年代，台灣流行樂壇興起「新音樂運動」，黃韻玲即為其中一份子。簽約[[滾石唱片|滾石唱片]]後，1986年，她推出自己的首張個人專輯《憂傷男孩》，即獲不俗的迴響。黃韻玲不僅自己能寫詞作曲，編曲、[[樂器|樂器]]演奏、[[和聲|和聲]]甚至歌曲製作亦是樣樣精通，才華洋溢，因此有「音樂精靈」之美譽。

1990年代起，除了自己的唱片外，黃韻玲倾注了更多的精力在其他歌手唱片的制作上，同属[[滚石唱片|滚石唱片]]的[[潘越雲|潘越雲]]、[[趙傳|趙傳]]、[[周華健|周華健]]、[[辛曉琪|辛曉琪]]等歌手的唱片中都收录了不少她的作品，而她亦參與製作羅文的絕響作品《我冷傲，但我是暖流》原版。

2007年起，在台灣著名電視歌唱選秀節目《[[超級星光大道|超級星光大道]]》擔任評審更為人熟知，評語風格屬於循循善誘型，頗得眾多選手信賴，參賽選手們尊稱為「小玲老師」。

2013年以〈回味〉單曲獲得[[第24屆金曲獎|第24屆金曲獎]]『最佳作曲人獎』，2016年以《初熟之物》專輯獲得[[第7屆金音獎|第7屆金音獎]]『最佳創作歌手獎』。

2017年擔任第28屆金曲獎評審團主席

2018年擔任中華音樂人交流協會年度十大專輯兼單曲評審團主席

2018年擔任湖南衛視中国首档美声偶像男团竞演养成类真人秀节目《聲入人心》（Super-Vocal）的聲樂指導

== 个人生活 ==
