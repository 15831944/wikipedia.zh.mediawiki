'''黃稚峰'''（{{bd|1993年|5月22日}}）<ref>{{cite web |author1=徐正揚 |title=中職》填補詹子賢空缺 兄弟黃稚峰生涯首次上一軍 |url=http://sports.ltn.com.tw/news/breakingnews/2526536 |website=自由電子報 |accessdate=2018-08-30}}</ref> <ref>{{cite web |author1=Shota |title=第九輪沒有比較差-中信兄弟黃稚峰 |url=https://www.sportsv.net/articles/55001 |website=運動視界 |accessdate=2018-08-30}}</ref><ref>{{cite web |author1=王真魚 |title=中職／黃稚峰、許基宏開轟　兄弟二軍9比1大勝桃猿二軍 |url=https://sports.ettoday.net/news/1198982 |website=ETtoday新聞雲 |accessdate=2018-08-30}}</ref>，為[[台灣|台灣]][[棒球|棒球]][[選手|選手]]，目前效力於[[中華職棒|中華職棒]][[中信兄弟|中信兄弟]]，守備位置為[[外野手|外野手]]。於[[中華職棒歷年選秀會#職棒二十七年（2016年）|2016年季中選秀]]中被[[中信兄弟|中信兄弟]]以第九輪第二十六順位選進<ref>[http://tsna.com.tw/tw/news/show.php?num=4022&page=503&type=1%E5%88%A5%E7%9A%84%E6%96%B0%E8%81%9E%E8%AA%AA%E4%B8%8D%E8%A1%8C 中信兄弟季中選秀　10位好手全達成共識]，TSNA，20160810</ref>。
