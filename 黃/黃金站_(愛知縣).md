'''黄金站'''（{{jpn|j=黄金駅|hg=こがねえき|rm=Kogane eki|f=y}}）是[[日本|日本]][[近畿日本鐵道|近畿日本铁道（近铁）]][[名古屋線|名古屋线]]上的车站，位于[[爱知县|爱知县]][[名古屋市|名古屋市]][[中村区|中村区]]黄金通8丁目。车站编号为E03。

== 历史 ==

* [[1938年|1938年]]（[[昭和|昭和]]13年）[[6月26日|6月26日]]：[[关西急行电铁|关西急行电铁]]桑名站——关急名古屋站（现近铁名古屋站）区间开通，本站作为中间站开始营业<ref name="sone02">{{Cite book|title=週刊 歴史でめぐる鉄道全路線 大手私鉄|author=[[曽根悟|曽根悟]]（監修）|date=2010-08-22|publisher=[[朝日新聞出版|朝日新聞出版]]|isbn=978-4-02-340132-7|series=週刊朝日百科|volume=2号 近畿日本鉄道 1|pages=18-23|和書}}</ref>。
* [[1940年|1940年]]（昭和15年）[[1月1日|1月1日]]：关西急行电铁与[[参宫急行电铁|参宫急行电铁]]合并，本站由合并后的参宫急行电铁管理。
* [[1941年|1941年]]（昭和16年）[[3月15日|3月15日]]：[[大阪电气轨道|大阪电气轨道]]与参宫急行电铁合并，成立新的[[关西急行铁道|关西急行铁道]]。本站所属的线路改称“名古屋线”<ref>{{Cite book|title=近畿日本鉄道 100年のあゆみ|author=近畿日本鉄道株式会社|date=2010-12|publisher=近畿日本鉄道|page=156|id={{全国書誌番号|21906373}}|和書}}</ref>。
* [[1944年|1944年]]（昭和19年）[[6月1日|6月1日]]：关西急行铁道与南海铁道（现在的[[南海电气铁道|南海电气铁道]]的前身）合并，本站由合并后的[[近畿日本铁道|近畿日本铁道]]管理。
* [[1945年|1945年]]（昭和20年）6月1日：暂停营业<ref name="chizu29">{{Cite book|title=[[日本鉄道旅行地図帳|日本鉄道旅行地図帳]]|author=[[今尾恵介|今尾恵介]]（監修）|publisher=[[新潮社|新潮社]]|year=2008|isbn=978-4-10-790026-5|volume=8 関西1|page=29|和書}}</ref>。
* [[1947年|1947年]]（昭和22年）[[3月1日|3月1日]]：恢复营业。
* [[2007年|2007年]]（[[平成|平成]]19年）[[4月1日|4月1日]]：导入IC卡“[[PiTaPa|PiTaPa]]”系统<ref>{{Cite press release|title=平成19年4月1日から、近鉄主要路線でICカードの利用が可能になります|publisher=近畿日本鉄道|date=2007-01-30|url=http://www.kintetsu.jp/news/files/iccard20070130.pdf|format=pdf|accessdate=2016-03-14}}</ref>。
* [[2014年|2014年]]（平成26年）[[12月21日|12月21日]]：无人化。

== 车站构造 ==
本站为[[地面車站|地面站]]，设有2台2线的[[侧式站台|侧式站台]]，均为3节编组电车的长度。站房和检票口位于2站台一侧，通过地道连接对面的1站台。2站台上设有厕所。

本站是[[近鐵名古屋站|近铁名古屋站]]管辖的无人站，设有[[自动检票机|自动检票机]]和自动结算机，支持[[PiTaPa|PiTaPa]]、[[ICOCA|ICOCA]]等IC卡，也可办理回数券卡和IC卡的充值业务。

== 车站周边 ==
本站与名古屋高速万场线的高架桥“黄金跨线桥”交叉。

; 公共设施

* 偕行会城西医院（原名古屋市立西部医疗中心城西医院）
* 名古屋市立丰国中学
* 爱知社区中心

; 其他

* 东海旅客铁道（JR东海）名古屋车辆区、名古屋工厂
* 日本货物铁道（JR货物）名古屋车辆所
* 笹岛信号场
* 丰丸产业
* 大丰建设名古屋分公司

; 周边道路

* 名古屋市道名古屋环状线
* 爱知县道115号津岛七宝名古屋线
