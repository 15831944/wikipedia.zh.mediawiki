'''黃錦星'''{{GBSHK}}{{JP}}（{{lang|en|'''Wong Kam-sing'''}}，{{bd|1963年|}}），籍貫[[廣東|廣東]][[順德區|順德]]<ref name="順德">[http://hkm.appledaily.com/detail.php?guid=57189363&category_guid=6996647&category=instant&issue=20170910 {{dead link|date=2018年5月}}</ref>，[[香港|香港]][[建築師|建築師]]，現任[[環境局局長|環境局局長]]<ref>{{cite news| url=http://www.apdnews.com/news/26856.html| title=黄锦星：香港不扩展堆填区或引致垃圾危机| newspaper=[[亚太日报|亚太日报]]| date=2013-05-27}}</ref><ref>{{cite news|title=中央任命香港特區第四屆政府主要官員|url=http://www.news.gov.hk/tc/categories/admin/html/2012/06/20120628_095212.shtml|newspaper=香港政府新聞網|date=2012-06-28}}</ref>，曾經擔任[[香港綠色建築議會|香港綠色建築議會]]副主席。

== 施政主张 ==
在任環境局局長期間，先後推出多份施政藍圖，以定下應對空氣污染、廢物資源管理、能源、自然保育及氣候變化的施政方向目標及時間表，藍圖包括：
*2013年3月，《香港清新空氣藍圖》<ref>{{cite web| url=http://www.enb.gov.hk/sites/default/files/New_Air_Plan_tc.pdf| title=《香港清新空氣藍圖》| publisher=[[環境局|環境局]]| date=2013-03}}</ref>；
*2013年5月，《香港資源循環藍圖2013-2022》<ref>{{cite web| url=http://www.enb.gov.hk/sites/default/files/WastePlan-C.pdf| title=《香港資源循環藍圖2013-2022》}}</ref> ；
*2014年2月，《香港廚餘及園林廢物計劃 2014-2022》<ref>{{cite web| url=http://www.enb.gov.hk/sites/default/files/pdf/FoodWastePolicyChi.pdf| title=《香港廚餘及園林廢物計劃 2014-2022》}}</ref>；
*2015年5月，《香港都市節能藍圖2015-2025+》<ref>{{cite web| url=http://www.enb.gov.hk/tc/energysavingplan.html| title=《香港都市節能藍圖2015-2025+》}}</ref>；
*2015年12月，《香港氣候變化報告2015》<ref>{{cite web| url=http://www.enb.gov.hk/sites/default/files/pdf/ClimateChangeChi.pdf| title=《香港氣候變化報告2015》}}</ref>；
*2016年12月，《香港生物多樣性策略及行動計劃 (2016－21）》<ref>{{cite web| url=http://www.afcd.gov.hk/tc_chi/conservation/Con_hkbsap/con_hkbsap.html| title=《香港生物多樣性策略及行動計劃 (2016－21）》| publisher=[[漁農自然護理署|漁農自然護理署]]}}</ref> ；
*2017年1月，《香港氣候行動藍圖2030+》 <ref>{{cite web| url=https://www.climateready.gov.hk/files/report/tc/HK_Climate_Action_Plan_2030+_booklet_Chin.pdf| title=《香港氣候行動藍圖2030+》}}</ref>。

針對空氣污染的挑戰，黃錦星一上任，就落實實施新空氣質素指標<ref>{{cite web|url=http://www.epd.gov.hk/epd/tc_chi/environmentinhk/air/air_quality_objectives/air_quality_objectives.html|title=空氣質素指標}}</ref>，並推出[[香港空氣質素健康指數|空氣質素健康指數]]（AQHI）<ref>{{cite web|url=http://www.aqhi.gov.hk/tc.html|title=空氣質素健康指數}}</ref>以配合新空氣質素指標。於2013年，提出以超過一百億元淘汰老舊柴油商業車，以改善空氣質素。<ref>{{cite news|url=http://www.singpao.com/xw/yw/201210/t20121019_396026.html|title=改善空氣 雷厲風行 老舊柴油車 被「判死刑」|newspaper=[[成報|成報]]|date=2012-10-19|deadurl=yes|archiveurl=https://web.archive.org/web/20150213123135/http://www.singpao.com/xw/yw/201210/t20121019_396026.html|archivedate=2015-02-13}}</ref>

廢物管理方面，提出五方面策略，包括：
# 以源頭減廢、污者自付的原則，計劃以立法推展廢物按量收費政策，再配合「購物膠袋收費」<ref>{{cite web|url=http://www.epd.gov.hk/epd/psb_charging/tc/index.html|title=全面推行塑膠購物袋收費計劃|publisher=[[環境保護署_(香港)|環境保護署]]}}</ref>、「廢電器電子產品」<ref>{{cite news|url=https://hk.news.yahoo.com/%E6%94%BF%E5%BA%9C%E5%BB%BA%E8%AD%B0%E6%92%A5%E4%BA%94%E5%84%84%E5%B1%AF%E9%96%80%E5%BB%BA%E5%BB%A2%E9%9B%BB%E5%99%A8%E9%9B%BB%E5%AD%90%E7%94%A2%E5%93%81%E5%9B%9E%E6%94%B6%E8%A8%AD%E6%96%BD-051600123.html|title=政府建議撥五億屯門建廢電器電子產品回收設施|publisher=[[商業電台|商業電台]]}}</ref>及「飲品玻璃樽」<ref>{{cite web|url=http://www.epd.gov.hk/epd/bottles_consult/tc/|title=《飲品玻璃樽生產者計劃公眾諮詢》|publisher=[[環境保護署_(香港)|環境保護署]]}}</ref>等生產者責任計劃；
# 分類回收，繼二十年前開展的「藍廢紙、黃金屬、啡塑膠」運動後，再推出乾淨回收運動<ref>{{youtube|Jfp0K4dZnis|有手尾 沖洗乾淨至回收}}</ref>，並提供手機應用程式「咪嘥嘢」<ref>{{cite web|url=http://www.gov.hk/tc/about/govdirectory/mobileapps/WASTELESS.htm|title=咪嘥嘢手機應用呈式|publisher=政府一站通|deadurl=yes|archiveurl=https://web.archive.org/web/20150213113127/http://www.gov.hk/tc/about/govdirectory/mobileapps/WASTELESS.htm|archivedate=2015年2月13日|df=}}</ref>，展示全港超過7,000個回收點；同時構建「綠在區區」，於全港十八區逐步打造社區環保站網絡<ref>{{cite web|url=http://www.info.gov.hk/gia/general/201411/30/P201411280922.htm|title=綠在區區項目最新進展|publisher=[[政府新聞署|政府新聞署]]}}</ref>，深化減廢回收；另推出十億「回收基金」<ref>{{cite news| url=https://hk.news.yahoo.com/%E9%BB%83%E9%8C%A6%E6%98%9F%E7%9B%BC%E5%BF%AB%E5%AF%A9%E5%9B%9E%E6%94%B6%E5%9F%BA%E9%87%91%E5%8A%A9%E6%BA%90%E9%A0%AD%E6%B8%9B%E5%BB%A2-023300921.html| title=黃錦星盼快審回收基金助源頭減廢| newspaper=[[星島日報|星島日報]]}}</ref>支持回收業界；
# 「惜食香港」運動<ref>{{cite web| url=http://www.foodwisehk.gov.hk/zh-hk/index.php|title=惜食香港專頁|publisher=[[環境保護署_(香港)|環境保護署]]}}</ref>，以[[大嘥鬼|大嘥鬼]]<ref>{{facebook|bigwaster.hk|大嘥鬼}}</ref>形象，推動全民從源頭減少食物浪費，並建立Facebook及Instagram 平台呼籲惜物惜福，亦透過環境及自然保育基金資助非政府組織推展剩食捐贈等<ref>{{cite web|url=http://www.ecf.gov.hk/tc/application/index.html|title=環境及自然保育基金|publisher=[[環境保護署_(香港)|環境保護署]]}}</ref>；
# 轉廢為能，計劃興建多座廚餘處理廠<ref>{{cite news|url=http://www.am730.com.hk/article-195160|title=籌建6有機資源回收中心|newspaper=[[am730|am730]]}}</ref>，轉廚餘為電能或其他可再生能源；而已規劃十多年的綜合廢物處理設施<ref>{{cite web|url=http://www.legco.gov.hk/database/chinese/data_ea/ea-integrated-waste-management-facilities.htm|title=個別政策事宜資料庫|publisher=[[香港立法會|香港立法會]]}}</ref>，亦轉化一般棄置物為能源，提供具規模的都市可再生能源；
# 衞生堆填，加強堆填區管理，並資助改裝垃圾車，裨益全港。<ref>{{cite news|url=http://news.sina.com.hk/news/20131026/-2-3099897/1.html|title=環境局申撥2000萬 資助垃圾車加尾蓋|publisher=新浪香港|deadurl=yes|archiveurl=https://web.archive.org/web/20150213113616/http://news.sina.com.hk/news/20131026/-2-3099897/1.html|archivedate=2015-02-13}}</ref>在2012至2017年間，先後通過三條與減廢相關的「生產者責任制」，包括擴大購物膠袋收費、飲品玻璃樽、廢電器及電子產品(四電一腦)。並且，為《都市固體廢物收費計劃》提出立法框架。<ref>{{cite news|url=http://www.info.gov.hk/gia/general/201703/20/P2017032000447.htm|title=都市固體廢物收費實施安排}}</ref>

自然保育方面， 2013年獲立法會通過條例，把[[大浪西灣|大浪西灣]]（西灣）、[[金山郊野公園|金山]]及[[圓墩郊野公園|圓墩]]的郊野公園「不包括的土地」得以納入各自的郊野公園範圍，增加約38公頃郊野公園面積。<ref>{{cite web|url=http://www.afcd.gov.hk/tc_chi/publications/publications_press/pr1847.html|title=大浪西灣、金山及圓墩的「不包括土地」將納入郊野公園範圍|publisher=漁農自然護理署}}</ref>於2015年的施政報告中提出，將與非政府組織研究採取合適和靈活的措施，結合民間力量，加強保護郊野。2016年，環境局與漁農自然護理署完成本港首份城市級的《生物多樣性策略及行動計劃》<ref>{{cite web|url=http://www.afcd.gov.hk/tc_chi/conservation/con_bsap/con_bsap.html|title=生物多樣性|publisher=[[漁農自然護理署|漁農自然護理署]]|deadurl=yes|archiveurl=https://web.archive.org/web/20150213124454/http://www.afcd.gov.hk/tc_chi/conservation/con_bsap/con_bsap.html|archivedate=2015-02-13}}</ref>另外，擾攘近四十年，被譽為「山林濕地 蜻蜓天堂」的極高生態價值私人土地- 沙羅洞，政府終於2017年6月宣佈，原則上同意以非原址換地計劃方式保育。<ref>{{cite news|url=http://www.info.gov.hk/gia/general/201706/15/P2017061500673.htm|title=以非原址換地方式長遠保育沙羅洞 (新聞稿)}}</ref>

節能綠建方面，環境局協調各政策局及部門，檢討能源及綠色建築政策，應對氣候變化。<ref>{{cite news|url=http://www.news.gov.hk/tc/record/html/2015/01/20150126_210900.shtml|title=黃錦星：從五方面改善環境|publisher=政府新聞網}}</ref>為落實「[[巴黎協定|巴黎協定]]」，香港政府已訂下目標在二○三○年把碳強度由二○○五年水平減少65%至70%，估算成效相等於由近年人均排碳量6.2公噸/年，減至2030年的3.3至3.8公噸。除了推出《香港都市節能藍圖2015-2025+》定立新節能目標：以2005年為基年，於2025年之前達致能源強度減少40%的目標外，<ref>{{cite news|url=http://www.info.gov.hk/gia/general/201512/13/P201512130460.htm|title=香港特區政府歡迎聯合國氣候大會達成新協議}}</ref> 在2017年4月，與兩電簽訂新的《[[利潤管制協議|利潤管制協議]]》，引入「上網電價」，鼓勵社區分佈式可再生能源。<ref>{{cite web|url=http://www.info.gov.hk/gia/general/201704/25/P2017042500756.htm|title=政府與兩家電力公司簽署新《管制計劃協議》}}</ref>另外，預留二億元為政府建築物、場地和設施裝置可再生能源設施，以推動可再生能源的發展。

2017年，黃錦星接受專訪時表示，將會振興本地綠色建築，從而達到今年初公布的《香港氣候行動藍圖2030+》（下稱藍圖）目標，即在2030年將本港絕對碳排放量由2005年水平降低26%至36%。「香港減碳關鍵在於推動綠色建築，因為香港六成碳排放和建築物相關，交通只佔約17%。但最大挑戰是要如何令市民大眾重視氣候問題<ref>[https://bkb.mpweekly.com/cu0004/20171025-56182 【綠色建築】黃錦星決戰民間：香港綠色建築何去何從？]</ref>。

== 个人经历 ==
黃錦星畢業於[[中華基督教會基慈小學|中華基督教會基慈小學]]<ref>{{web reference|URL=http://orientaldaily.on.cc/cnt/news/20131111/00176_087.html|title=中環出更：巧遇小學同窗 黃錦星喜合照|work=東方日報|year=2013年|date=11月11日}}</ref>，先後於[[聖文德書院|聖文德書院]]及[[保良局總理聯誼會第一中學|保良局總理聯誼會第一中學]]（今[[保良局第一張永慶中學|保良局第一張永慶中學]]）就讀，與商務及經濟發展局長[[邱騰華|邱騰華]]在同一所中學就讀預科班，於1988年在[[香港大學|香港大學]]建築系畢業，於1990年代於[[加拿大|加拿大]][[卑詩大學研究院|卑詩大學研究院]]進修可持續建築環境。<ref name="mingpao">{{cite news|title=黃錦星遊埠不忘環保|url=http://hk.news.yahoo.com/%E9%BB%83%E9%8C%A6%E6%98%9F%E9%81%8A%E5%9F%A0%E4%B8%8D%E5%BF%98%E7%92%B0%E4%BF%9D-213128094.html|newspaper=[[明報|明報]]|date=2012-06-15}}</ref>。

== 家庭生活 ==
黃錦星之妻[[梁素雲|梁素雲]]亦為建築師，曾經擔任[[香港建築師學會|香港建築師學會]]本地事務部主席<ref>{{cite web|title=關於我們－組織架構|url=http://www.hkia.net/hk/AboutUs/AboutUs_03_01.htm|publisher=[[香港建築師學會|香港建築師學會]]|accessdate=2012-09-12}}</ref><ref>{{cite news|title=學者出身 環境局局長熱門人選 黃錦星政治待歷練|url=http://news.hkheadline.com/dailynews/content_hk/2012/05/17/191078.asp|accessdate=2012-09-12|newspaper=[[頭條日報|頭條日報]]|date=2012-05-17}}</ref>，育有一女。
