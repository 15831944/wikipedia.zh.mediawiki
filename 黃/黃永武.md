'''黃永武'''（{{bd|1936年|2月9日|||catIdx=Huang黃}}），[[浙江省|浙江省]][[嘉善縣|嘉善縣]]人。文學博士、教授、兼[[作家|作家]]。曾任[[中華民國|中華民國]][[國立中興大學|國立中興大學]]、[[成功大學|成功大學]]文學院院長，中華民國古典文學研究會創會會長。所著《[[中國詩學|中國詩學]]》四冊，曾榮獲民國69年(1980年)第五屆 [[國家文藝獎|國家文藝獎]]。後以散文馳名，所著《[[愛廬小品|愛廬小品]]》四冊，再獲第十八屆 [http://www.ettoday.com/events/neca/plenty/index-a03-03a.htm 國家文藝獎]。後又以《[[生活美學|生活美學]]》四冊，《黃永武隨筆》上下冊，名重中外文壇。文皆雋永深刻，哲思渾成，餘味無窮。

==簡歷==
《[[愛廬談心事|愛廬談心事]]》一書中述及黃永武先生前半生的履歷。<ref>{{cite book|author1=黃永武|title=愛廬談心事|date=1995|publisher=三民書局|location=台灣|isbn=957-14-2210-X}}</ref>生平工作簡歷及各著作出版年月，見於《黃永武解周易》<ref>{{cite book|author1=黃永武|title=黃永武解周易|date=2011|publisher=新文豐出版公司|location=台灣|isbn=978-957-17-2152-1|pages=1593-1598}}</ref>、《好句在天涯--我怎樣寫散文》<ref>{{cite book|author1=黃永武|title=好句在天涯--我怎樣寫散文|date=2012|publisher=三民書局|location=台灣|isbn=978-957-145639-3}}</ref>、及《我心萬古心--我怎樣做學問》<ref>{{cite book|author1=黃永武|title=我心萬古心--我怎樣做學問|date=2014|publisher=三民書局|location=台灣|isbn=978-957-14-5908-0}}</ref>之附錄<黃永武著作年曆簡表>。

===早年生活===
1936年出生後，次年抗戰爆發，黃永武隨著從事抗日工作的父親黃麟書先生，多次出入淪陷區及大後方。1946年抗戰勝利，由屯溪回[[上海|上海]]讀小學五年級。童年種種可喜可愕的經歷，在《不識河山痛--我怎樣度童年》一書中詳述。

1950年12月自滬至[[香港|香港]]，1951年3月抵[[台灣|台灣]]。入[[台南一中|台南一中]]補習學校插班初中三年級，次年9月考入[[國立臺南大學|台南師範學校]]，畢業後在小學任教三年。當時已著有《呢喃集》與《心期》。

===文學研究生涯===
1958年入[[東吳大學|東吳大學]]中文系，1962年入[[國立臺灣師範大學|國立臺灣師範大學]]國文研究所。

1964年獲碩士，論文《[[形聲多兼會意考|形聲多兼會意考]]》由中華書局出版。1969年8月商務印書館發行其《[[字句鍛鍊法|字句鍛鍊法]]》初版本。

1970年11月黃永武先生獲國家文學博士，博士論文《[[許慎之經學|許慎之經學]]》亦由臺灣中華書局出版。

1971年黃永武教授任國立高雄師範大學國文系主任兼教務長。同年七月由三民書局出版《[[詩心|詩心]]》。

1976年出版《[[中國詩學·設計篇·鑑賞篇|中國詩學·設計篇·鑑賞篇]]》，1977年出版《[[中國詩學·考據篇|中國詩學·考據篇]]》，其間並編《[[杜詩叢刊|杜詩叢刊]]》及《[[杜甫詩集四十種索引|杜甫詩集四十種索引]]》。

1977年黃永武教授任[[國立中興大學文學院|國立中興大學文學院]]院長。1979年出版《[[中國詩學·思想篇|中國詩學·思想篇]]》，於次年2月獲文藝理論類國家文藝獎。是年創立中國古典文學研究會，推選為創會理事長。

1981年8月編成《[[敦煌寶藏|敦煌寶藏]]》第一輯十冊。1983年文學院長任滿，赴美國[[康乃爾大學|康乃爾大學]]任訪問教授。同年9月與[[張高評|張高評]]合著之《[[唐詩三百首鑑賞|唐詩三百首鑑賞]]》出版。1984年由美返國，出版《[[詩與美|詩與美]]》。

1985年出版遊記散文集《[[載愛飛行|載愛飛行]]》、文化考古論集《[[珍珠船|珍珠船]]》、及詩詞賞析集《[[抒情詩葉|抒情詩葉]]》。是年8月，黃永武教授往國立成功大學任文學院院長，並兼新創之歷史語言研究所所長。是年12月，《[[敦煌寶藏|敦煌寶藏]]》一百四十冊編成，為世界各國大學圖書館所珍藏。

1986年《敦煌遺書最新目錄》出版。是年將《[[字句鍛鍊法|字句鍛鍊法]]》增訂，由洪範書店印行。1987年出版《[[敦煌的唐詩|敦煌的唐詩]]》，部份被譯成日文，編入《講座敦煌》。是年又出版了《[[讀書與賞詩|讀書與賞詩]]》。

1988年黃永武教授返台北，在[[台北市立師範學院|台北市立師範學院（市北師）]]任教，於[[陽明山國家公園|陽明山]]後山之金山農場，置一工作坊，以[[陶渊明|陶淵明]]吾亦愛吾廬詩意，命名為愛廬。先生同時在[[中央日報|中央日報]]闢《愛廬小品》專欄，中華日報闢《[[海角讀書|海角讀書]]》專欄，[[新生報|新生報]]闢《[[詩香谷|詩香谷]]》專欄，致力散文寫作。

1989年10月九歌出版之《[[詩林散步|詩林散步]]》，乃黃永武由學術散文轉向知性散文小品大規模呈現前夕之作。

1990年黃永武教授出版《[[愛廬小品|愛廬小品]]》，分靈性、生活、勵志、讀書四冊，於1993年獲散文類國家文藝獎。1992年出版《[[詩香谷|詩香谷]]》第一集、第二集。1993年出版《[[愛廬談文學|愛廬談文學]]》<ref>{{cite web|url=http://www.sanmin.com.tw/Product/index/000092061|title = 愛廬談文學|author = 黃永武}}</ref>。1995年出版《[[愛廬談心事|愛廬談心事]]》<ref>{{cite web|url=http://www.sanmin.com.tw/Product/index/000092307|title = 愛廬談心事|author = 黃永武}}</ref>。1997年出版《[[生活美學|生活美學]]》，分天趣、諧趣、情趣、理趣四冊。

1996年黃永武教授自市北師退休<ref>{{Cite web|url = http://news.secr.ncku.edu.tw/files/16-1054-91316.php?Lang=zh-tw|title = 成大藝文沙龍　黃永武談散文 - 台灣時報|accessdate = 2016-02-17|author = 林東良|date = |publisher = |archive-url = https://web.archive.org/web/20160225105648/http://news.secr.ncku.edu.tw/files/16-1054-91316.php?Lang=zh-tw#|archive-date = 2016-02-25|dead-url = yes}}</ref>，至東吳大學任教兩年。1998年起不再任教職。同年出版《[[愛廬談諺詩|愛廬談諺詩]]》<ref>{{cite web|url=http://www.sanmin.com.tw/Product/index/000092965|title = 愛廬談諺詩|author = 黃永武}}</ref>、《[[詩與情|詩與情]]》<ref>{{cite web|url=http://www.sanmin.com.tw/Product/index/000093003|title = 詩與情|author = 黃永武}}</ref>。2000年出版《[[我看外星人|我看外星人]]》，有憑有據，不同於科幻。2001年出版《[[山居功課|山居功課]]》。2002年洪範書店出版《[[字句鍛鍊法|字句鍛鍊法]]》新增版本。而於中央日報所闢之散文專欄，直寫至2006年，名為《[[林下小記|林下小記]]》，文章已廣在網路傳誦。

2008年起，黃永武教授又撰就《新增本中國詩學》四冊，繼續開拓內涵，煥煥煌煌，由高雄巨流公司重新面世，徵引極廣，賞析尤精，巧思妙解，引人入勝。是年9月，洪範書局出版《黃永武隨筆》上下兩冊，將近十年的精品分輯為〈寫景〉、〈記學〉、〈說理〉、〈關情〉四輯。

2010年11月臺灣南華大學舉辦「黃永武先生學術研討會」，評述黃永武學術及散文者，計有黃慶萱、張高評<ref>{{cite web|url=http://nhuir.nhu.edu.tw/retrieve/18958/4023001304.pdf|title = 黃永武先生的學術成就|author = 張高評}}</ref>、陳章錫、[[林慶彰|林慶彰]]<ref>{{cite web|url=http://libwri.nhu.edu.tw:8081/Ejournal/4023001302.pdf|title = 黃永武先生編纂叢書的貢獻|author = 林慶彰}}</ref>、陳室如<ref>{{cite web|url=http://libwri.nhu.edu.tw:8081/Ejournal/4023001204.pdf|title = 從飛行到隨筆－黃永武遊記作品研究|author = 陳室如}}</ref>、王偉勇、王祥穎、張瑞芬、[[鄭定國|鄭定國]]<ref>{{cite web|url=http://libwri.nhu.edu.tw:8081/Ejournal/4023001306.pdf|title = 全面入境—談黃永武《中國詩學》鑑賞篇|author = 鄭定國}}</ref>、許建崑<ref>{{cite web|url=https://books.google.ca/books?id=12REDwAAQBAJ&pg=PT257&lpg=PT257&dq=%E8%A8%B1%E5%BB%BA%E5%B4%91+%E8%90%AC%E5%8D%B7%E6%A8%93+%E9%BB%83%E6%B0%B8%E6%AD%A6&source=bl&ots=aPmq4kXWM_&sig=ACfU3U1Z25MXLZrgL6yegyWiBY70cWOPvg&hl=en&sa=X&ved=2ahUKEwimr863zKjhAhXHuZ4KHSz1CswQ6AEwAHoECAgQAQ|title = 移情、借景與越位:當代作家作品論集|author = 許建崑}}</ref>、李添富<ref>{{cite web|url=https://www.books.com.tw/products/0010735692|title = 添富論學集 (黃永武先生「形聲多兼會意考」申議)|author = 李添富}}</ref>、林聰明<ref>{{cite web|url=http://nhuir.nhu.edu.tw/retrieve/18962/4023001402.pdf|title = 論黃永武教授斠理敦煌唐詩的貢獻|author = 林聰明}}</ref>等十二位教授。

2011年11月國立成功大學舉辦「文學家系列國際學術研討會」，評述黃永武散文者，計有李瑞騰、朱嘉雯<ref>{{cite web|url=http://blog.udn.com/fguedu/11523075|title = 勇闖天涯—黃永武的渡海經驗與治學精神|author = 朱嘉雯}}</ref>、林淑貞<ref>{{cite web|url=http://www.studentbook.com.tw/goods.php?id=4124|title = 圖像敘事與多元文本(菊花心事與生活理趣──黃永武散文書寫向度的轉折與特色)|author = 林淑貞}}</ref>等三位教授。李瑞騰館長評曰﹕「出之以潔淨暢達之文筆﹐創造當代學者散文之典範。」朱嘉雯教授評曰：「黃永武的治學散文筆調感性與理性兼具，而博學的根柢實為其文學的基礎。」林淑貞教授評曰：「揭示厚實真誠的隨興感會，含納古典詩文之後所吐哺出來的光華，以表現出練達人生的理趣為主，與生命特質相印合。」

2011年12月黃永武教授又撰成《黃永武解周易》，由新文豐出版，上下兩鉅冊，逾百萬字，不但校出易經原文所存的誤字，又整理出易經一貫之卦象，並將新出土的各本周易異文作詳析，引領後學跨出一大步。

2012年4月黃永武所著《好句在天涯--我怎樣寫散文》<ref>{{cite web|url=http://www.sanmin.com.tw/Product/index/002988554|title = 好句在天涯--我怎樣寫散文|author = 黃永武}}</ref>由三民書店發行，乃作者文心的自我剖解，坦誠詳盡，奇趣橫生，金針度人，境域與其他作家大不相同。

2012年9月，《[[愛廬小品|愛廬小品]]》四冊簡體字本由大陸漓江出版社在上海發行。10月《新增本中國詩學》四冊簡體字本，由中國步印文化新世界出版社在北京發行。中國當當網及京東網站佳評如瀑雷濤雪，喧騰一時。有讀者見書中所展示的中華文化之美，如見久別重逢的慈祥父母，歡欣起舞，感動不已。

2014年6月，黃永武為酬答友朋及讀者的熱情鼓勵，公開其從事學術工程的成功經驗，將親赴學術深淵中尋覓驪龍頷珠的秘辛過程奉獻出來，將許多實證趣例，以散文隨筆的形式表現，寫成《我心萬古心--我怎樣做學問》<ref>{{cite web|url=http://www.sanmin.com.tw/Product/index/004520557|title = 我心萬古心--我怎樣做學問|author = 黃永武}}</ref>一書，由三民書店發行。12月《[[詩林散步|詩林散步]]》增訂再版，由九歌出版社重排問世。

2016年撰就《不識河山痛—我怎樣度童年》。

2017年4月浙江大學出版黃永武著《敦煌文獻與文學叢考》，由鄭阿財教授編集黃永武所發表有關敦煌學之文章，諸如卷目標定、唐詩校勘、敦煌趣談等近四十篇。鄭阿財在序文中說：「黃永武《敦煌寶藏》編成，快速提昇寫卷的閱讀環境，大大改善了敦煌學的研究條件，使研究風氣大開，促使敦煌學逐漸跳脫了抓吃式、尋寶式等個別寫卷的研究方式，而步入專題式的系統整理與研究。」又說：「黃永武對敦煌唐詩的探究，開創多元的校勘方法，推論闡發詳盡，結論新穎，多所發明，展現獨特的研究風華。」<ref>{{cite book|author1=鄭阿財|title=敦煌文獻與文學叢考之序文:黃永武先生與敦煌學及其敦煌唐詩研究之成就|date=2017|publisher=浙江大學出版社}}</ref>

2018年以來寫還原成語真相的文章，計有調皮搗蛋<ref>{{cite web|url=https://www.worldjournal.com/5638492/article-%E8%AA%BF%E4%BB%80%E9%BA%BC%E7%9A%AE%EF%BC%9F%E6%90%97%E4%BB%80%E9%BA%BC%E8%9B%8B%EF%BC%9F/|title = 調什麼皮？搗什麼蛋？北美世界日報副刊|accessdate = 2018-07-04|author = 黃永武}}</ref>、爭風吃醋<ref>{{cite web|url=https://www.worldjournal.com/5846601/article-%E7%88%AD%E4%BB%80%E9%BA%BC%E9%A2%A8%EF%BC%9F%E5%90%83%E4%BB%80%E9%BA%BC%E9%86%8B%EF%BC%9F/|title = 爭什麼風？吃什麼醋？北美世界日報副刊|accessdate = 2018-09-07|author = 黃永武}}</ref>、亂七八糟<ref>{{cite web|url=https://www.worldjournal.com/5915149/article-%E4%B8%83%E6%98%AF%E4%BA%82%EF%BC%9F%E5%85%AB%E6%98%AF%E7%B3%9F%EF%BC%9F/|title =七是亂？八是糟？ 北美世界日報副刊|accessdate = 2018-10-15|author = 黃永武}}</ref>、馬馬虎虎<ref>{{cite web|url=https://www.worldjournal.com/5998086/article-%E6%8E%A2%E7%B4%A2%E9%A6%AC%E9%A6%AC%E8%99%8E%E8%99%8E/|title =探索馬馬虎虎 北美世界日報副刊|accessdate = 2018-12-06|author = 黃永武}}</ref>、狗皮倒竈<ref>{{cite web|url=https://www.worldjournal.com/6097137/article-%E7%8B%97%E7%9A%AE%E5%80%92%E7%81%B6%E7%9A%84%E7%94%B1%E4%BE%86/?ref=%E8%97%9D%E6%96%87_%E4%B8%96%E7%95%8C%E5%89%AF%E5%88%8A|title = 狗皮倒灶的由來 北美世界日報副刊|accessdate = 2019-02-01|author = 黃永武}}</ref>等五篇。追本探源，頗多創見。

2019年3月臺灣中華書局再版黃永武博士論文《[[許慎之經學|許慎之經學]]》<ref>{{cite web|url=http://www.chunghwabook.com.tw/product_d.php?lang=tw&tb=7&id=3306&cid=44|title = 許慎之經學|author = 黃永武}}</ref>。本書曾有陳章錫教授評論其客觀成就及學術意義<ref>{{cite web|url=http://libwri.nhu.edu.tw:8081/Ejournal/4023001305.pdf|title = 黃永武的經學主張與貢獻|author = 陳章錫}}</ref>。四月三民書局新排再版《[[詩心|詩心]]》<ref>{{cite web|url=http://www.sanmin.com.tw/Product/index/007147775|title = 詩心|author = 黃永武}}</ref>。

==文學風格==
黃永武教授著作豐富，學問淵博，情深心細，才氣縱橫。他的筆下總帶著歷史的厚度與哲思的深度，成為獨樹一格的優雅風貌。用詞遣字既有古風，又帶新意，比一般柔情感性之敘事散文，更添了知性與哲理。能將古典詩詞之美、中國文化鮮為人知之精與妙、及明清散文家之閑情與智慧，甚至古今中外的奇聞逸事、俊語諺談，盡皆融會貫通、深入淺出、淋漓盡致地呈現出來。宏觀時，家國天下﹐皆所情繫﹐氣實志定﹐詞鋒勁壯。微觀時，其筆觸將蟲鳥木石乃至繩瓶枕椅等瑣屑的題材，透過才筆的點化，像透過三稜鏡般即刻都化作七彩繽紛的虹霓。

正如逢甲大學[[張瑞芬|張瑞芬]]教授所述，黃永武先生的小品散文，「乃是秉持中國古代傳統文人哲理與美學意境，渾厚天成的。永遠有驚奇帶給他的讀者們。」又說﹕「黃永武真正成了始於學問卻不囿於學問﹐展現自我風格的台灣當代散文名家之一。」<ref>{{Cite journal|url = https://tln.nmtl.gov.tw/ch/m2/nmtl_w1_m2_c_8.aspx?k=%E9%BB%83%E6%B0%B8%E6%AD%A6&id=4860|title = 珍珠滿船──黃永武的散文成就|journal = 南華大學黃永武先生學術研討會論文集|issue = |doi = |others = |year = 2010|volume = |page = |pmid = |author = 張瑞芬}}</ref>台灣師範大學陳室如教授則說﹕「黃永武的現代散文有著極為特殊的學者風格﹐在傳統認知框架與新鮮體驗的交會之下﹐悠遊於異國山水與浩瀚知識中﹐沉澱出更為深層的人生智慧。」<ref>{{cite journal|author1=陳室如|title=從飛行到隨筆--黃永武旅行散文研究|journal=2010年黃永武先生學術研討會論文集|date=2010}}</ref>成功大學王偉勇教授則說﹕「黃永武將學者考據的工夫﹐運用在散文中﹐形成他的特色﹐在現代散文作家中﹐可謂絕無僅有。」<ref>{{cite journal|author1=王偉勇|title=言而有據--談「考據」在黃永武教授散文中的應用|journal=2010年黃永武先生學術研討會論文集|date=2010}}</ref>各家的評述﹐可以概見黃永武散文有多方面的特色。至於其學術成就﹐評者尤眾﹐不及備載。黃永武致力發揚中華文化、會通中西的異同、傳承文人風範，屢建學術奇功。慧光照處，爆發驚人的創作活力，不愧為當代鉅匠，既是大師，也是大手筆。
