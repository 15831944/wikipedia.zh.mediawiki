黃亮勛 ({{bd|1985年|6月26日||}})，出生於[[台灣|台灣]][[雲林|雲林]]，現居[[台北|台北]]，為[[霹靂布袋戲|霹靂布袋戲]]第五代接班人<ref>[http://www.chinatimes.com/newspapers/20131216000360-260102 《台大高材生 霹靂第5代接班人黃亮勛露頭角》，唐玉麟，中國時報]</ref>。

== 身世背景 ==
黃亮勛出生於[[布袋戲|布袋戲]]世家，為[[霹靂布袋戲|霹靂布袋戲]]第五代接班人<ref>{{Cite web |url=http://www.wealth.com.tw/article_in.aspx?nid=1866 |title=《黃家第五代 要讓霹靂不只是霹靂 根植傳統又激盪創新　玩布袋戲玩到上櫃》，游筱燕，財訊 |access-date=2017-03-27 |archive-url=https://web.archive.org/web/20170328015959/http://www.wealth.com.tw/article_in.aspx?nid=1866 |archive-date=2017-03-28 |dead-url=yes }}</ref>。其曾祖父為台灣[[布袋戲|布袋戲]]國寶級大師[[黃海岱|黃海岱]]，《[[雲州大儒俠|雲州大儒俠]]》創造者[[黃俊雄|黃俊雄]]是其祖父 <ref>[http://tccanet.org.tw/article-753.html#.WNiFo2-GNhE 《奇人黃亮勛 霹靂五代轟動江湖》，陳幸芬，兩岸文創誌，台灣文化創意產業聯盟協會]</ref>，父親為[[霹靂布袋戲|霹靂布袋戲]]創辦人[[黃強華|黃強華]]。現任[[霹靂國際多媒體|霹靂國際多媒體]]股份有限公司副總經理，大畫電影文化股份有限公司總經理。

== 學經歷 ==
黃亮勛曾就讀[[台灣大學|台灣大學]]生化科技系、[[政治大學|政治大學]]科技管理研究所、[[臺大|臺大]]-[[復旦|復旦]][[EMBA|EMBA]]專班與[[北京電影學院|北京電影學院]]影視金融商學院。求學時期熱衷[[樂團|樂團]]，喜愛[[冰島|冰島]][[後搖滾|後搖滾]]音樂，曾組[[獨立樂團|獨立樂團]]並擔任[[吉他手|吉他手]]，擅長詞曲創作與編曲，在藝術創作上亦頗具天賦。退伍後進入[[霹靂集團|霹靂集團]]擔任行銷企劃，曾參與《[[梁祝西湖蝶夢|梁祝西湖蝶夢]]》<ref>[http://home.pili.com.tw/news/2013030113414557177/ 《霹靂原創偶動畫【梁祝西湖蝶夢】》，霹靂網]</ref>劇本的編劇。

== 代表作品 ==

=== [[東離劍遊紀|東離劍遊紀]] ===
《[[東離劍遊紀|東離劍遊紀]]》是一部由[[Thunderbolt_Fantasy_Project|Thunderbolt Fantasy Project]]製作，[[臺灣|台灣]]和[[日本|日本]]雙方合作的[[奇幻|奇幻]][[武俠|武俠]][[布袋戲|布袋戲]][[電視|電視]]劇集。本作是[[布袋戲|布袋戲]]首次[[台灣|台灣]]與[[日本|日本]]跨界合作的作品，由黃亮勛擔任[[製片|製片]]<ref>[http://www.techbang.com/posts/44613-interview-with-pili-international-multimedia-vice-president-huang-liangxun-talk-about-east-travelled-sword-discipline-and-the-different-ways-how-traditional-puppet-making 《專訪霹靂多媒體副總黃亮勛，《東離劍遊紀》與虛淵玄如何顛覆霹靂布袋戲？》，T客邦，2016/07/14]</ref>，[[動畫|動畫]]製作統籌與總監製，[[虛淵玄|虛淵玄]]擔任[[編劇|編劇]]與[[監製|監製]]，主題曲還找來日本知名歌手[[西川貴教|西川貴教]]演唱、製作；日文版本則由日本[[聲優|聲優]]團隊以日語配音，霹靂的傳統「人物出場詩號」則保留台語發音<ref>[https://www.mirrormedia.mg/story/20161216ent008/ 《攻進日本市場　霹靂打造台灣漫威》，劉慧茹，鏡傳媒</ref>。

   2016年7月上映後，《[[東離劍遊紀|東離劍遊紀]]》居高不下的收視率驗證了作品的成功。根據統計，《[[東離劍遊紀|東離劍遊紀]]》在[[日本|日本]]七月新節目收視率排名第3；在[[中國|中國]]的[[视频弹幕网站|影片分享網站]][[bilibili|bilibili]]上，《[[東離劍遊紀|東離劍遊紀]]》也是熱度排名第4的新節目，首集就吸引了超過100萬名粉絲觀看。<ref>[http://game.zerotimes.net/8325/%E6%9D%B1%E9%9B%A2%E5%8A%8D%E9%81%8A%E7%B4%80%E6%94%9C%E6%89%8B%E8%99%9B%E6%B7%B5%E7%8E%84-%E9%A6%96%E9%9B%86%E8%A7%80%E7%9C%8B%E7%A0%B4%E7%99%BE%E8%90%AC/ 《“東離劍遊紀”攜手虛淵玄 首集觀看破百萬》，遊戲大爆炸，2016/07/19]</ref>，此為少數台灣本土品牌進軍國際舞台成功的案例，並引起歐美動漫迷廣泛迴響討論。

=== [[奇人密碼－古羅布之謎|奇人密碼－古羅布之謎]] ===
    [[霹靂布袋戲|霹靂布袋戲]]第五代接班人，七年級生的黃亮勛在全球首部[[台灣|台灣]]產製的[[3D|3D]]偶動漫[[電影|電影]]《[[奇人密碼－古羅布之謎|奇人密碼－古羅布之謎]]》的製作中擔任編劇一職<ref>[http://ent.ltn.com.tw/news/paper/853638 《黃亮勛遭老爸設計 霹靂解密邁入3D》，封以恩，自由時報]</ref>，同時也負責大畫後製的藝術監製。《[[奇人密碼－古羅布之謎|奇人密碼－古羅布之謎]]》耗時3年半打造，全片結合實景與人偶操作，加上全[[3D|3D]][[攝影|攝影]]以及[[動畫|動畫]][[特效|特效]]等元素，將[[布袋戲|布袋戲]]提升至媲美[[好萊塢|好萊塢]]的[[動畫|動畫]]水準<ref>[http://www.chinatimes.com/realtimenews/20150130002169-260410 莊丙農，《《科技》霹靂「奇人密碼：古羅布之謎」，2月11日上映》，中時電子報，2015/01/30]</ref>。

    《[[奇人密碼－古羅布之謎|奇人密碼－古羅布之謎]]》全片以3D真偶拍攝結合電腦特效，並突破過去純台語配音，開創「依人依地」的多口配音<ref>[http://www.playmusic.tw/column_info.php?id=3065&type=mf_movie 《動畫製作搞很大 角色多口配音 霹靂第五代黃亮勛為接班暖身》，PlayMusic音樂網]</ref>，此作試圖在電影環境裡創造一個新的境界，找到一個屬於現代人的新[[布袋戲|布袋戲]]，並藉由這樣的作品，讓布袋戲從傳統中走出來。這是黃亮勛當初創作的信念。身為創作者，黃亮勛認為文化是循序漸進，緩慢改變的過程。現代藝術應該要從本土文化中尋找根基。[[台灣|台灣]]在文化長河中兼容並蓄 : 繼承過去的脈絡，同時檢視當下，不斷的堆疊、演進、改變、融合，不拘泥於某一形式，只要能感動人，就是價值之所在。
