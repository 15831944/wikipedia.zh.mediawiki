'''黃埔站'''（{{lang-en|'''Whampoa Station'''}}）是[[港鐵|港鐵]][[觀塘綫|觀塘綫]]一個[[鐵路站|鐵路站]]，位於[[香港|香港]][[九龍城區|九龍城區]][[紅磡|紅磡]][[黃埔花園|黃埔花園]]<ref name="MTRC KTE 2004">[http://www.mtr-kwuntonglineextension.hk/pdf/public-consultation/consultation-meeting/public-forums/20090822_kte_forum_skh_fung_kei_millennium_priminary_school.pdf 觀塘綫延綫] {{webarchive|url=https://web.archive.org/web/20131227020816/http://www.mtr-kwuntonglineextension.hk/pdf/public-consultation/consultation-meeting/public-forums/20090822_kte_forum_skh_fung_kei_millennium_priminary_school.pdf |date=2013-12-27 }}，2008年3月12日，[[香港鐵路有限公司|地鐵公司]]</ref><ref>[http://orientaldaily.on.cc/cnt/news/20090708/00176_037.html 港鐵黃埔站擬設德安街] 2009-07-08 [[東方日報_(香港)|東方日報 (香港)]]</ref>，主要服務[[黃埔新邨|黃埔新邨]]及[[黃埔花園|黃埔花園]]一帶，較位於[[紅磡灣|紅磡灣]]的[[紅磡站|紅磡站]]更接近紅磡<ref>[http://hk.apple.nextmedia.com/news/art/20110519/15267143 觀塘延線服務紅磡] 2011-05-19 [[蘋果日報_(香港)|蘋果日報 (香港)]]</ref>，於2016年10月23日起取代[[油麻地站|油麻地站]]舊終點站，並成為[[觀塘綫|觀塘綫]]的下行終點站。車站所在的位置，鄰近紅磡昔日的一座大型船塢「[[黃埔船塢|黃埔船塢]]」，該船塢先後於1970及80年代經城都地產及[[長江實業地產|長江實業地產]]改建為大型住宅屋苑[[黃埔新邨|黃埔新邨]]及[[黃埔花園|黃埔花園]]，車站遂命名為'''黃埔站'''。

==簡介==
黃埔站是繼[[寶琳站|寶琳站]]、[[迪士尼站_(香港)|迪士尼站]]及[[博覽館站|博覽館站]]後，第四個只設1個[[月台|月台]]的重型鐵路車站<ref>不計算[[香港輕鐵|輕鐵]]車站</ref>，亦是首個單月台的地底車站；但由於單月台設計不足以容納所有[[觀塘綫|觀塘綫]]的班次，所以觀塘綫在星期一至五（公眾假期除外）繁忙時間（07:00至09:30，16:30至19:00）、星期六（07:15至20:00）、星期日及公眾假期（08:40至20:00）的列車服務，以「1+1」（輪流）的方式，一班以黃埔站為終點站，另一班在[[何文田站|何文田站]]終止服務並使用調頭隧道折返何文田載客，類似[[青衣站|青衣站]]繁忙時間的列車服務。因此，黃埔站在上述時間的班次是4-5分鐘一班，而不是[[觀塘綫|觀塘綫]]其它車站的2-3分鐘一班；其餘時間，所有班次均以黃埔站為終點站。

==車站設計==
===樓層===
黃埔站設有4層，地面為車站出入口，另外於地底設有兩個車站大堂（東大堂及西大堂）及1個側式月台供乘客上落<ref name="MTRC KTE 2004" />。

===大堂===
黃埔站設有兩個不相連的大堂，分別為東大堂和西大堂，東大堂同時接駁黃埔花園第二期及第五期地庫商場<ref name="meeting">[http://www.mtr-kwuntonglineextension.hk/en/pdf/public-consultation/community-liaison-group/kte-wha-clg-meeting_1st_19-5-2011.pdf 觀塘綫延綫社區聯絡小組第一次會議（黃埔段）] {{webarchive|url=https://web.archive.org/web/20140331113231/http://www.mtr-kwuntonglineextension.hk/en/pdf/public-consultation/community-liaison-group/kte-wha-clg-meeting_1st_19-5-2011.pdf |date=2014-03-31 }}，港鐵公司，2011年5月</ref>，即C1及D1出口。兩大堂步行距離大約4分鐘。值得一提的是，兩個大堂之中，只有東大堂設有客務中心，故乘客在西大堂遇到票務問題時只能利用自助客務機處理。有高齡居民擔心出現問題時職員未能及時處理，非常不便<ref>[http://hk.on.cc/hk/bkn/cnt/news/20161022/bkn-20161022150706672-1022_00822_001.html 萬人試用黃埔站 客轟視像客務機複雜唔識用]，東方報業集團網站，2016年10月23日</ref><ref>[http://www.hk01.com/%E6%B8%AF%E8%81%9E/49940/-%E8%A7%80%E5%A1%98%E5%BB%B6%E7%B7%9A%E9%80%9A%E8%BB%8A-%E9%BB%83%E5%9F%94%E7%AB%99%E8%A5%BF%E5%A4%A7%E5%A0%82%E4%B8%8D%E8%A8%AD%E5%AE%A2%E5%8B%99%E4%B8%AD%E5%BF%83-%E6%A9%9F%E5%99%A8-%E9%81%99%E6%8E%A7-%E9%95%B7%E8%80%85%E6%86%82%E9%9B%A3%E6%B1%82%E5%8A%A9 近300人參觀港鐵黃埔站 居民批不便：如果出唔到閘都唔知點算]，香港01，2016年10月22日</ref>。

====車站設施====
西大堂近B出口設[[恒生銀行|恒生銀行]]自動櫃員機、[[7-11便利店|7-11便利店]]、[[美心西餅|美心西餅]]及[[爭鮮外帶壽司|爭鮮外帶壽司]]，東大堂近D出口設[[恒生銀行|恒生銀行]]自動櫃員機及[[7-11便利店|7-11便利店]]。

====閘機====
共有18部閘機，全部為翼閘，其中4部為闊閘機，東、西大堂各7部窄閘機、2部闊閘機。

東大堂近D出口5部閘機（59,11-14）固定為出閘，其餘閘機則可按需要而改變方向。東大堂乘客要從近C出口的閘機入閘。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Whampoa Station 2017 09 part1.jpg | caption1 = 1號月台（往調景嶺方向）（西面）
| image2 = Whampoa Station 2017 09 part2.jpg | caption2 = 1號月台（往調景嶺方向）（東面）
}}
黃埔站<ref name="MTRC KTE 2004" />由於車站選址四周均為大廈樁柱又滿佈公共設施喉管電線，能夠施工的空間不多，故只設1個[[側式月台|側式月台]]供乘客上落之用，情況類似現時[[寶琳站|寶琳站]]、[[迪士尼站|迪士尼站]]和[[博覽館站|博覽館站]]的單月台設計，乘客下車後，列車便會改為相反的方向，供乘客進入列車前往[[調景嶺站|調景嶺]]方向。但寶琳站與黃埔站的差別在於列車{{Link-ja|留置線|留置線}}的位置，寶琳站的位於寶琳站月台旁，但延線的留置線則位於何文田站的的1號月台往調景嶺方向後方，也就是代表何文田站至黃埔站的路段是[[單線鐵路|單線雙程]]的路段<ref>[https://www.facebook.com/MTRiders/posts/1831224950444631:0 相片Photo]</ref>。

每當特殊的情況之下，部份觀塘綫的港鐵列車在清客完畢後，將會於車站暫停服務，屆時月台將會以廣播告知乘客。

當此站出現故障時，觀塘綫下行列車只能以何文田站作為臨時的終點站。屆時，前往此站的乘客將須要轉乘其他交通工具或港鐵提供的緊急接駁巴士前往。

黃埔站設有6個出入口<ref>{{cite web |url=http://ps.hket.com/content/1434327/%E8%A7%80%E5%A1%98%E5%BB%B6%E7%B6%AB%E6%96%99%E5%B9%B4%E5%BA%95%E9%80%9A%E8%BB%8A%20%E5%8D%80%E5%85%A7%E6%A8%93%E5%83%B9%E5%8B%A2%E5%BF%85%E8%B5%B7%E9%A3%9B |title=觀塘延綫料年底通車 區內樓價勢必起飛 |publisher=[[香港經濟日報|香港經濟日報]] |date=2016年5月28日}}{{zh-hant}}</ref>，分別設於[[民兆街|民兆街]]、[[紅磡道|紅磡道]]、[[黃埔花園|黃埔花園]]（[[船景街|船景街]]及[[德安街|德安街]]），其中有2個為連接商場地庫的出入口，分別為船景街黃埔新天地時尚坊（C1）及德安街黃埔花園第五期AEON Style（D1）。<ref name="new_plan">[http://www.mtr.com.hk/chi/projects/images/KCDC_Paper_final_kte.pdf 九龍城區區議會觀塘綫延綫區議會諮詢文件]，[[港鐵公司|港鐵公司]]，2009年8月</ref><ref>{{cite web|author=[[香港鐵路有限公司|香港鐵路有限公司]]|url=http://www.mtr-kwuntonglineextension.hk/pdf/publications/tc/leaflet/S0212_Whampoa.pdf|title=黃埔站小冊子|date=2015年7月|language=zh-hant|deadurl=yes|archiveurl=https://web.archive.org/web/20160201142849/http://www.mtr-kwuntonglineextension.hk/pdf/publications/tc/leaflet/S0212_Whampoa.pdf|archivedate=2016-02-01}}</ref>由於部分出入口連接商場，開放時間跟隨商場開放時間，比其他出口開放時間短。此外，連接電梯至D1的出入口需經六級樓梯才能到達，對嬰兒車及傷殘人士十分不便。

==車費優惠==
[[紅磡|紅磡]][[家維邨|家維邨]]商場設有適用於此站的[[港鐵特惠站|港鐵特惠站]]，為使用成人[[八達通|八達通]]的乘客提供港幣2元的車費回贈<ref>[http://www.mtr.com.hk/archive/corporate/en/press_release/PR-16-093-C.pdf 紅磡家維邨商場增設「港鐵特惠站」　提供$2車費折扣優惠]，2016年10月21日，[[港鐵公司|港鐵公司]]</ref>。

==利用狀況==
黃埔站位於[[香港|香港]][[紅磡|紅磡]]。由於土瓜灣一帶暫沒有鐵路途經，因此該區有不少乘客乘坐小巴到該站轉乘港鐵。另外，該站鄰近[[黃埔天地|黃埔天地]]、[[黃埔花園|黃埔花園]]、[[黃埔新邨|黃埔新邨]]、[[海濱南岸|海濱南岸]]、[[海逸豪園|海逸豪園]]等多座[[住宅|住宅]]、[[商場|商場]]和[[百貨公司|百貨公司]]，亦有在紅磡工業區的上班族使用該站（當中有不少利用午饍時間到[[家維村|家維村]][[港鐵特惠站|港鐵特惠站]]拍取車費優惠），因此每天有不少乘客進出車站，大部份使用該站出入閘的人流都是居住於黃埔及紅磡的居民、上班人士、中小學學生、食客、娛樂和購物人士。另外，在地鐵通車前市民來往黃埔和香港島、油麻地、旺角、深水埗等地，均需取道[[漆咸道北|漆咸道北]]、[[加士居道|加士居道]]、[[紅磡海底隧道|紅磡海底隧道]]等幹道，在繁忙時間飽受這些道路塞車之苦，加上巴士路綫需繞經紅磡灣，令車程增加不少，自地鐵通車後改乘地鐵節省不少車程，令不少市民已改為乘搭地鐵出入黃埔一帶。途經深水埗、旺角、油麻地的九巴212綫更因大量乘客流失而取消。

==歷史==
{{main|旅客捷運系統|沙中綫|觀塘綫延綫}}
2002年9月，[[九廣鐵路公司|九廣鐵路公司]]為擴展[[沙田至中環綫|沙田至中環綫]]的服務範圍至黃埔一帶，建議在黃埔建造[[旅客捷運系統|自動捷運系統]]，連接九鐵[[紅磡站|紅磡站]]，以便乘客在該站轉乘[[東鐵綫|東鐵綫]]、[[西鐵綫|西鐵綫]]和[[沙田至中環綫|沙田至中環綫]]<ref name="THB SCL KTE AnnexF">[http://www.legco.gov.hk/yr07-08/chinese/panels/tp/tp_rdp/papers/tp_rdp-thbtcr10101699-c.pdf 沙田至中環線及地鐵觀塘線延線]（立法會THB(T)CR 10/1016/99號文件）附件F，2008年3月11日，運輸及房屋局</ref>。同年12月，[[香港鐵路有限公司|地鐵有限公司]]亦建議延長地鐵[[觀塘綫|觀塘綫]]，由[[油麻地站|油麻地站]]至黃埔站，中途設[[何文田站|何文田站]]，以服務黃埔地區<ref name="THB SCL KTE AnnexF" />。2003年11月，政府請求地鐵公司進一步檢討觀塘線延線方案，地鐵公司並在翌年2月向政府提交經修訂後的建議<ref name="THB SCL KTE AnnexF" />。

2005年7月，九鐵公司和地鐵公司向政府提交沙中線的[[兩鐵合併|兩鐵合併]]方案，當中提議以地鐵的觀塘綫延線，取代九鐵2004年最終方案擬稿中的黃埔自動捷運系統。該方案在2007年7月由[[運輸及房屋局|運輸及房屋局]]向[[香港立法會|立法會]]提交<ref name="THB SCL KTE AnnexF" />。2007年12月，兩鐵合併，九鐵和地鐵網絡合併為[[港鐵|港鐵]]。

2008年3月，[[香港行政會議|行政會議]]批准[[港鐵公司|港鐵公司]]對觀塘綫延線進行進一步規劃及設計<ref>[http://www.legco.gov.hk/yr07-08/chinese/panels/tp/tp_rdp/papers/tp_rdp-thbtcr10101699-c.pdf 沙田至中環線及地鐵觀塘線延線]（立法會THB(T)CR 10/1016/99號文件），2008年3月11日，運輸及房屋局</ref><ref>[http://www.info.gov.hk/gia/general/200803/11/P200803110154.htm 沙中線及觀塘延線開展進一步規劃及設計（附短片）]，2008年3月11日，香港政府新聞公報</ref>。

勘探工作已經在2010年3月起陸續展開。工程已於2010年11月29日獲批，並於2011年6月2日動工。

港鐵在2016年10月22日（星期六）開放車站予公眾參觀，吸引近一萬名市民到場參觀<ref>[http://www.mtr.com.hk/archive/corporate/en/press_release/PR-16-094-C.pdf 黃埔站開放日　迎接10月23日觀塘綫延伸開通]，2016年10月22日，[[港鐵公司|港鐵公司]]</ref>。黃埔站於2016年10月23日隨觀塘綫延綫通車而正式啟用。
