'''黃貞瑜'''{{BD|1982年|2月23日||}}<ref name="選舉公報">{{cite web|title=臺灣省嘉義縣大林鎮第十七屆鎮長選舉選舉公報|url=http://103bulletin.cec.gov.tw/103/%E5%98%89%E7%BE%A9%E7%B8%A3/%E9%84%89%E9%8E%AE%E5%B8%82%E9%95%B7/%E5%A4%A7%E6%9E%97%E9%8E%AE%E9%8E%AE%E9%95%B7.pdf|publisher=嘉義縣選舉委員會|archiveurl=https://web.archive.org/web/20150413145751/http://103bulletin.cec.gov.tw/103/%E5%98%89%E7%BE%A9%E7%B8%A3/%E9%84%89%E9%8E%AE%E5%B8%82%E9%95%B7/%E5%A4%A7%E6%9E%97%E9%8E%AE%E9%8E%AE%E9%95%B7.pdf|archivedate=2015-04-13}}</ref>，[[中華民國|中華民國]]政治人物，出生於[[嘉義縣|嘉義縣]]，畢業於世新大學口傳系、[[國立中正大學|國立中正大學]]政治學研究所<ref name="選舉公報"/>。[[民主進步黨|民主進步黨]]籍，曾任嘉義縣[[大林鎮_(臺灣)|大林鎮]]代理鎮長，第16、17屆嘉義縣[[大林鎮_(臺灣)|大林鎮]]鎮長，曾為台灣最年輕女鎮長。歌手[[黃思婷|黃思婷]]的姑姑。現任[[嘉義縣|嘉義縣]]農會總幹事。

== 政治生涯 ==
2006年協助母親李秀美參與[[大林鎮_(台灣)|大林鎮]]選舉，選後至[[溪口鄉_(台灣)|溪口鄉]]擔任機要總務課員。2008年擔任大林鎮鎮長的母親[[李秀美_(嘉義縣)|李秀美]]因選舉送茶葉賄選官司遭[[台灣高等法院|台灣高等法院]]判決有期徒刑三個月定讞受停職處分，時民進黨籍縣長陳明文指派李秀美女兒黃貞瑜為代理鎮長。2009年12月當選第16屆大林鎮鎮長。

== 重要政績 ==
# 2010年於北向大林入口處，創設「大林My Darling」景觀地標入口意象，將「Dalin」（大林英文地名）與「Darling」（親愛的）的諧音趣味巧妙表現大林。此地標吸引許多結婚新人來拍婚紗照。
# 2010年、2013年兩度獲選環保署「營造優質環保示範區」全國前五大示範級鄉鎮，使大林鎮成為環保署標竿示範地區。
#2012年獲經濟部補助「大林鎮蘭花文化創意產業發展計畫」，高度行銷大林「米蘭小鎮‧My Darling」。
#2014年獲選環保署「營造友善城鄉環境」拔尖級推動單位，為營造My Darling慢城友善城鄉環境示範區。
