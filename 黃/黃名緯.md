'''黃名緯'''（{{lang-en|'''Ng Ming Wei'''}}，{{bd|1994年|11月20日||}}）是[[新加坡|新加坡]]男子[[跆拳道|跆拳道]]運動員。他於2017年[[英聯邦跆拳道錦標賽|英聯邦跆拳道錦標賽]]在男子58公斤以下獲得[[金牌|金牌]]，成為第一位新加坡人在[[英聯邦跆拳道錦標賽|英聯邦跆拳道錦標賽]]的對打項目獲得金牌。<ref>{{cite web |url=http://www.taekwondodata.com/ming-wei-ng.akhq.html|title=Ng Ming Wei Taekwondo Data |publisher=Taekwondo Data|accessdate=2018-11-12}}</ref>他也在[[2015的東南亞運會|2015的東南亞運會]]男子54公斤以下獲得[[銅牌|銅牌]]。<ref>{{cite web |url=https://www.myactivesg.com/team-singapore/athletes/n/ng-ming-wei|title=NG Ming Wei ActiveSG Profile |publisher=[[Sport_Singapore|Sport Singapore]]|accessdate=2018-11-11}}</ref><ref>{{cite web |url=https://en.wikipedia.org/wiki/Taekwondo_at_the_2015_Southeast_Asian_Games|title=Taekwondo at the 2015 Southeast Asian Games |publisher=[[Wikipedia|Wikipedia]]|accessdate=2018-11-12}}</ref> 截至2019年1月，他在[[Instagram|Instagram]]上共收集了47,500名粉絲，使他成為[[新加坡|新加坡]]最受關注的運動員之一。<ref>{{Cite web|url=https://www.straitstimes.com/sport/ng-banking-on-insta-fame|title=National taekwondo exponent Ng Ming Wei banking on Insta-Fame|first=|date=2019-01-22|website=The Straits Times|language=en|archive-url=|archive-date=|dead-url=|access-date=2019-01-27}}</ref>
