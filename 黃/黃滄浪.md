'''黃滄浪'''（{{Bd|1916年|3月14日|1986年|8月28日|catIdx=H黃}}<ref name=":0">{{Cite web|url=http://webcache.googleusercontent.com/search?q=cache:qAdEx4E0pNIJ:rulin.org.tw/erlin201/14%25E5%258D%2581%25E5%259B%259B%25E3%2580%2581%25E5%258F%25B0%25E7%2581%25A3%25E6%259F%2594%25E9%2581%2593%25E4%25B9%258B%25E7%2588%25B6%25E2%2594%2580%25E2%2594%2580%25E5%258D%2581%25E6%25AE%25B5%25E5%25B8%25AB%25E7%25AF%2584%25E9%25BB%2583%25E6%25BB%2584%25E6%25B5%25AA.pdf+&cd=12&hl=zh-TW&ct=clnk&gl=tw|title=參、人物篇|accessdate=2017-11-01|work=webcache.googleusercontent.com}}</ref>），[[號|號]]'''武峰'''，是一位出身彰化[[埤頭|埤頭]]芙朝地區的[[柔道|柔道]][[教育家|教育家]]。

== 生平 ==

=== 早年生活 ===
[[大正|大正]]5年（1916年）3月14日，黃滄浪[[出生|出生]]於[[臺中州|臺中州]][[北斗郡|北斗郡]][[埤頭庄|埤頭-{庄}-]]；他的[[父親|父親]]黃振興是一位[[農夫|農夫]]，他的[[母親|母親]]是吳李氏，他的雙親皆曾獲選為模範父/母親。<ref name=":0" />

[[昭和|昭和]]5年（1930年），黃滄浪背負著父親的期望前往[[日本|日本]]習[[醫學|醫]]，先入柏原中學校就讀；在校期間，因[[柔道|柔道]]非常盛行且黃滄浪[[體格|體格]]強壯又對該項技藝感[[興趣|興趣]]，他決定參加校內「寒中柔道特別訓練班」，並於結訓時受其指導[[教師|教師]]北川艇二珍重。 在學期間[[假期|假期]]返鄉時，黃滄浪曾前往北斗武德殿觀摩[[臺灣總督府警察|警察]]柔道訓練場，並親眼目睹警察對[[教練|教練]]岡田氏非常恭敬；當時，黃滄浪對此感到[[羨慕|羨慕]]並因此立志成為柔道[[教官|教官]]。 18歲時，黃滄浪在柏原中學校取得柔道初段，隔年又晉升二段。<ref name=":0" />

昭和11年（1936年），黃滄浪[[畢業|畢業]]於自柏原中學校並經[[競賽|競賽]]考入位在[[京都市|京都市]]的{{Link-ja|大日本武德會武道専門學校|大日本武徳会武道専門学校 (旧制)}}<ref>{{Cite web|url=http://webcache.googleusercontent.com/search?q=cache:HAs4je6HhaYJ:compete.camel.ntupes.edu.tw/ezcatfiles/b016/img/img/1346/270138679.pdf+&cd=57&hl=zh-TW&ct=clnk&gl=tw|title=日據時代臺灣柔道運動的傳入與發展 (1895～1945)|accessdate=2017-11-01|last=USER|work=webcache.googleusercontent.com}}{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，成為{{Link-ja|磯貝一|磯貝一}}、{{Link-ja|田畑昇太郎|田畑昇太郎}}、{{Link-ja|栗原民雄|栗原民雄}}等柔道家的[[弟子|弟子]]。<ref name=":0" /><ref name=":1">{{Cite book|url=https://books.google.com.tw/books/about/%E9%BB%83%E6%BB%84%E6%B5%AA.html?id=y8BYQwAACAAJ&redir_esc=y|title=黃滄浪: 臺灣柔道之父|last=(體育)|first=黃武雄|date=2007|publisher=臺灣身體文化學會|year=|isbn=9789868089938|location=|pages=|language=zh|chapter=}}</ref><ref name=":2">{{Cite book|url=https://books.google.com.tw/books?id=O1LqMQEACAAJ&dq=%E8%87%BA%E7%81%A3%E7%99%BE%E5%B9%B4%E9%AB%94%E8%82%B2%E4%BA%BA%E7%89%A9%E8%AA%8C&hl=zh-TW&sa=X&ved=0ahUKEwjH58Tpr5zXAhXGGpQKHbyvAfAQ6AEIJDAA|title=臺灣百年體育人物誌|last=|first=|date=2006|publisher=臺灣身體文化學會|year=|isbn=9789868089914|location=|pages=|language=zh|chapter=}}</ref>

=== 傳藝生涯 ===
昭和12年（1937年），黃滄浪晉升柔道四段鍊士。昭和14年（1939年），黃滄浪晉升柔道五段；當時，他曾獲選參加[[裕仁天皇|裕仁天皇]]御前柔道表演賽。昭和15年（1940年），黃滄浪自大日本武德會武道専門學校畢業。昭和18年（1943年），黃滄浪成為柔道七段達士，同時從事柔道[[教學|教學]]。此間，黃滄浪曾任教於{{Link-ja|香川縣高松第一中學校|香川県高松第一中学校}}，並為該校計劃培訓柔道隊，多次率隊參加地方性及全國性柔道比賽。<ref name=":0" />

1946年，黃滄浪帶領[[家人|家人]]返鄉，並任教於[[國立彰化高級中學|臺灣省立彰化中學]]。此後，黃滄浪為推廣柔道運動，曾數度於[[臺灣省運動會|臺灣省運動會]]表演「柔道綜合制敵法」。

[[民國紀年|民國]]四十四年（1955年），黃滄浪正式受聘擔任[[臺灣省政府|臺灣省政府]][[內政部警政署刑事警察局|警務處刑事警察總隊]]、[[中央警官學校|中央警官學校]]暨[[臺灣警察學校|臺灣省警察學校]]柔道[[教官|教官]]。<ref name=":0" />

1957年（民國四十六年）至1983年（民國七十二年）間，黃滄浪先後在[[三軍大學|三軍大學]]、[[憲兵學校|憲兵學校]]、[[中正理工學院|中正理工學院]]、[[國防醫學院|國防醫學院]]、[[政工幹校|政工幹校]]、[[國立臺灣大學|國立臺灣大學]]、[[國立臺灣師範大學|臺灣省立師範大學]]、[[中國文化學院|中國文化學院]]等校擔任教師，又自行開設「武峰柔道館」傳授其技藝。<ref name=":0" />

民國四十七年（1958年），黃滄浪擔任[[教練|教練]]帶領[[中華隊|中華隊]]柔道選手前往[[東京|東京]]參加第三屆[[亞洲運動會|亞洲運動會]]，並在返國後獲[[中華民國教育部|中華民國教育部]]表彰為優良教練。

民國四十九年（1960年），黃滄浪晉升柔道八段。<ref name=":0" />

民國五十六年（1967年）8月，黃滄浪前往美國猶他州[[鹽湖城|鹽湖城]]參加由世界柔道協會所主辦的國際裁判考選，並以[[中華民國國民|中華民國國民]]身份獲得世界裁判資格；此後，他曾在[[1964年夏季奧林匹克運動會|1964年夏季奧林匹克運動會]]、[[1972年夏季奧林匹克運動會|1972年夏季奧林匹克運動會]]及世界柔道錦標賽等賽事擔任[[裁判_(體育)|裁判]]。<ref name=":0" />

民國六十八年（1979年），黃滄浪晉升柔道九段。

民國七十五年（1986年），黃滄浪過世，並隨後被[[安葬|安葬]]於臺北縣[[金山區_(新北市)|金山鄉]]；當時，中華民國柔道協會於其家祭後褒揚並追贈其為柔道十段。<ref name=":0" />

此外，黃滄浪也喜愛練習[[書法|書法]]，並將其與武術練習結合。<ref name=":0" /><ref name=":1" /><ref name=":2" />

== 家庭 ==
黃滄浪在1938年與[[劍道|劍道]]家三品節子（[[漢名|漢名]]黃淑華）[[結婚|結婚]]，婚後兩人育有3子1女：分別是長子黃有終、長女黃碧珍、次子黃有正、三男黃有三。<ref name=":0" />

== 軼事 ==
* 黃滄浪曾被臺灣警察界人士取以「[[台灣地區|台灣地區]][[胸圍|胸圍]]最寬大的[[男人|男人]]」的[[別名|別名]]。<ref name=":0" /><ref name=":1" />

== 語錄 ==
* 「學習柔道不是爭強鬥狠的工具，而是全身的綜合運動，更是集[[智慧|智慧]]、[[精神|精神]]、[[注意力|注意力]]、堅[[忍_(佛教)|忍]]於一的[[訓練|訓練]]。」<ref name=":0" />
