'''黃教安'''（{{朝|諺=황교안|漢=黃敎安}}，{{bd|1957年|4月15日|||catIdx=C黃敎安}}），現任[[自由韓國黨|自由韓國黨]]代表<ref>[https://m-cn.yna.co.kr/view/ACK20190227005700881? 韓前總理黃教安當選最大在野黨黨首]</ref>，[[大韓民國|大韓民國]]政治人物，曾任[[大韓民國總統|代理總統]]、[[大韓民國|國務總理]]。

== 簡介 ==
1957年4月15日[[首爾|首爾]]出生，{{link-ko|光成中学校|광성중학교 (서울)}}、{{link-ko|京畿高等學校|경기고등학교|京畿高中}}、[[成均館大學|成均館大學]]畢業，1981年司法考試合格，[[司法研修院|司法研修院]]第13期畢業。<ref>[http://japanese.joins.com/article/755/170755.html 韓國法務長官「『わが民族同士』従北会員名簿で捜査]</ref>

1992年起擔任地方檢察官。2003年因偵辦[[大韓民國國家情報院|国家情報院]]非法監聽案成名，在本案中逮捕国家情報院課長。<ref>[http://japanese.joins.com/article/j_article.php?aid=38372 通信傍受資料流出-国情院職員ら３人を逮捕]</ref>

2009年1月升任昌原檢察長，2009年9月升大邱高等檢察長，2011年1月升釜山高等檢察長，創下司法界最快升遷過程引起社會議論<ref>[http://japanese.yonhapnews.co.kr/pgm/9810000000.html?cid=AJP20150521001400882 韓国新首相内定の黄教安氏　安定した法務行政に定評]</ref>，不久後退休轉任律師。

== 法務部長 ==
2013年3月11日，經由總統[[朴槿惠|朴槿惠]]提名擔任[[大韓民國法務部|法務部部長]]。

在法務部長任內，他對於[[親北|親北]][[朝鮮民族主義|朝鮮民族主義]]的[[極左派|極左派]]政黨[[統合進步黨|統合進步黨]]採取批判態度。由於統合進步黨議員[[李石基|李石基]]的親北言論，韓國政府向憲法法院提議解散統合進步黨的決定。黃教安表示：“統合進步黨在建黨目的與政黨活動層面上，都屬於‘違憲政党’，因而向憲法法院提交要求解散該黨的審判申請書”。除提出解散該黨的申請外，政府同時還申請判決該党所屬國會議員全部喪失議員資格，並針對該黨提交了停止政黨活動的臨時處理申請。<ref>[http://zh.apdnews.com/asia/asia/seouljianwen/144441.html 韓國為何容不下“親朝黨”？] 2014-12-22</ref>2014年11月在憲法法院與統合進步黨代表、前總統參選人[[李正姬|李正姬]]舌戰辯論4小時。2014年12月19日，憲法法院判決，统合進步黨爲非法叛亂組織並且必須解散，成為韓國民主化以來首個被解散的政黨。<ref>[http://www.epochtimes.com/b5/14/12/19/n4322107.htm 韓國解散親朝政黨 韓憲政史上首例] 2014-12-19 大紀元</ref>

== 國務總理 ==
在經過數次總理提名失敗，還有[[李完九|李完九]]因醜聞僅兩個月就辭職，2015年6月12日獲總統[[朴槿惠|朴槿惠]]提名爲[[韓國總理列表|國務總理]]候選人<ref>[http://news.chosun.com/site/data/html_dir/2015/06/12/2015061202792.html ［속보］황교안 인사청문보고서 與 단독 채택…野 퇴장] [[朝鮮日報|朝鮮日報]] 2015-06-12</ref>，2015年6月18日正式擔任總理一職<ref>{{cite news|title=韓國新任總理任命案獲國會通過 黃敎安任總理 |url=http://www.chinadaily.com.cn/micro-reading/dzh/2015-06-18/content_13860543.html |publisher=中國新聞網 |date=2015年6月18日 |accessdate=2015年6月18日 }}</ref>。

2016年11月2日，由於政府爆發[[2016年韓國政治醜聞|崔順實干政事件]]請辭<ref name=yonhap20161102>{{Cite news|url=http://japanese.yonhapnews.co.kr/Politics2/2016/11/02/0900000000AJP20161102004700882.HTML| title=朴大統領　内政の第一線退く＝新首相が「もう一人の大統領」に|agency=[[韩联社|韩联社]]|date=2016-11-02|accessdate=2016-11-02}}</ref>，但隨後因新總理[[金秉准|金秉准]]提名被撤回而繼續任職。<ref>{{Cite news|url=http://www.worldjournal.com/4522796/article-%E6%9C%B4%E6%A7%BF%E6%83%A0%E6%92%A4%E5%9B%9E%E7%B8%BD%E7%90%86%E6%8F%90%E5%90%8D-%E4%BF%83%E5%9C%8B%E6%9C%83%E6%8E%A8%E8%88%89%E6%96%B0%E4%BA%BA%E9%81%B8/|title=朴槿惠撤回總理提名 促國會推舉新人選|newspaper=[[世界新聞網|世界新聞網]]|date=2016-11-08|accessdate=2016-11-09|deadurl=yes|archiveurl=https://web.archive.org/web/20161109223626/http://www.worldjournal.com/4522796/article-%E6%9C%B4%E6%A7%BF%E6%83%A0%E6%92%A4%E5%9B%9E%E7%B8%BD%E7%90%86%E6%8F%90%E5%90%8D-%E4%BF%83%E5%9C%8B%E6%9C%83%E6%8E%A8%E8%88%89%E6%96%B0%E4%BA%BA%E9%81%B8/|archivedate=2016-11-09}}</ref>

== 代總統 ==
2016年12月9日，国会表决通过[[朴槿惠弹劾案|朴槿惠弹劾案]]，朴槿惠被国会停职，由國務總理黄教安代理总统，主持國務會議。<ref>{{cite news|url=http://www.sankei.com/world/news/161209/wor1612090031-n1.html|title=強大な権限は黄教安首相に　朴氏は大統領府で生活、給与も？|publisher=[[產經新聞|產經新聞]]|language=ja|date=2016-12-09|accessdate=2016-12-09}}</ref>

2017年3月10日，[[朴槿惠|朴槿惠]]被[[韓國憲法法庭|韓國憲法法庭]]正式彈劾解職，黃教安持續代理總統一職。

2017年3月15日，被視為執政黨[[自由韓國黨|自由韓國黨]]熱門總統候選人的黃教安表示「深思熟慮後，認為在當前的形勢下，應做好代理總統之責，不得有半點疏忽，為了治國和進行公正選舉，自己不適合參選總統」，且宣布下屆總統大選之日於5月9日。
