'''黄琬'''（{{bd|141年||192年|catIdx=H}}），字'''子琰'''，[[江夏郡|江夏]]安陆（今[[湖北|湖北]][[安陆|安陆]]北）人，[[东汉|东汉]]末年大臣。曾因[[党锢之祸|党锢之祸]]被禁锢二十几年，官拜[[豫州|豫州]]牧时，政绩显著。后与[[王允|王允]]谋诛[[董卓|董卓]]，董卓部下[[李傕|李傕]]、[[郭汜|郭汜]]攻破[[长安|长安]]后，死于狱中。

== 早年 ==
黄琬曾祖父為[[黃香|黃香]]，祖父为[[黄琼|黄琼]]，姑母为[[刘焉_(益州牧)|刘焉]]妻。幼年丧父，[[建和_(东汉)|建和]]元年（147年）[[正月|正月]]，京城[[洛阳|洛阳]]发生[[日食|日食]]，黄琼把情况报告给[[汉桓帝|汉桓帝]]。皇太后问黄琼太阳被遮了多少。黄琼思考答案但不清楚日食情况。黄琬当时7岁，提醒他道：“为什么不说太阳剩下的部分就像刚出来的月亮？”黄琼恍然大悟，立刻用他的话去报告。

后来黄琼为[[司徒|司徒]]时，黄琬以公孙拜[[童子郎|童子郎]]，但以疾病为由不去就任，得以名扬京师。[[司空|司空]][[盛允|盛允]]有疾，黄琼派黄琬去慰问，恰逢江夏有贼子作乱，因为黄琼为江夏人，盛允讽刺道：“江夏大邦，而蛮多士少”，黄琬反击道：“蛮夷猾夏，责在司空”，并拂衣而去。

== 官场生涯 ==
黄琬官拜[[五官中郎将|五官中郎将]]，并与当时的[[光禄勋|光禄勋]][[陈蕃|陈蕃]]成为朋友，当时东汉旧制要[[察举|察举]]三署郎，以才德优异者举为[[茂才|茂才]]，但权富子弟往往得举，而穷苦人士却被排除在外，于是黄琬与陈蕃同心协力，不论贫富启用人才、显用志士。因此[[平原郡|平原郡]]刘醇、[[河东郡_(中国)|河东郡]]朱山、[[蜀郡|蜀郡]]殷参等人以才行得举，但是黄、陈同时得罪了权贵。

当时的御史中丞为[[王畅|王畅]]，王畅身边的权贵就陷害黄琬、陈蕃、[[侍御史|侍御史]][[刁韪|刁韪]]等人结交朋党。于是陈蕃官职被免，黄琬、刁韪遭到禁锢。

黄琬一直被禁锢到[[光和|光和]]末年（184年），[[太尉|太尉]][[杨赐|杨赐]]上书荐黄琬有拨乱之才，于是征为[[议郎|议郎]]，并提拔为[[青州|青州]]刺史和[[侍中|侍中]]。[[中平|中平]]初年（184年），为[[右扶风|右扶风]]，征拜为[[将作大匠|将作大匠]]、[[少府|少府]]、[[太仆|太仆]]。官拜[[豫州|豫州]]牧时，境内盗贼猖獗，黄琬将其征讨平定，于是声威大震。政绩显扬天下，于是被封[[关内侯|关内侯]]。

== 董卓之乱 ==
到了[[董卓|董卓]]当权朝廷之时，黄琬以名臣被征为[[司徒|司徒]]，迁[[太仆|太仆]]，封[[阳泉|阳泉]][[乡侯|乡侯]]。董卓欲迁都长安，但遭到黄琬与司徒[[杨彪|杨彪]]的反对。黄琬驳斥董卓：“昔时[[周公|周公]]以[[洛邑|洛邑]]为首都安定了[[姬姓|姬姓]]，[[光武帝|光武帝]]以[[洛阳|洛阳]]为首都兴隆了[[汉朝|汉朝]]，这是天之所启，神之所安。大业既定，岂能妄加迁动，辜负了四海之期望？”当时人人都畏惧董卓一暴怒，黄琬必定被杀害，于是都劝谏黄琬。黄琬对他们说：“昔日[[白公胜|白公胜]]作乱于[[楚国|楚国]]，[[屈庐|屈庐]]冒着利刃向前；[[崔杼|崔杼]]杀[[齐庄公_(光)|齐庄公]]于[[齐国|齐国]]，[[晏婴|晏婴]]不畏惧他的盟誓。我即使无德，但真心钦慕古人之气节。”董卓警戒他是名门望族，不敢加害他，于是黄琬被罢免官职。后来与杨彪同拜[[光禄大夫|光禄大夫]]，及迁到西都长安，转[[司隶|司隶]][[校尉|校尉]]，并与[[司徒|司徒]][[王允|王允]]共同密谋诛杀董卓。最终[[李傕|李傕]]、[[郭汜|郭汜]]攻破长安，黄琬被收捕入狱，并死于狱中，时年五十二岁。

== 相关人物 ==
黄琬妻来氏，有弟[[来敏|来敏]]。汉末来氏姐弟避难荆州，黄琬的表侄[[刘璋|刘璋]]将他们接到蜀中。

在[[历史小说|历史小说]]《[[三国演义|三国演义]]》中，黄琬有一子[[黄奎|黄奎]]，与[[马腾|马腾]]一同密谋诛杀[[曹操|曹操]]，后事败被满门抄斩。黄奎为[[虚构人物|虚构人物]]，史书并无记载。
