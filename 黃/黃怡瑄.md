== 演藝生涯 ==

=== 出道以前 ===

從小就喜好音樂的黃怡瑄，擅長大提琴、鋼琴、直笛與吉他等樂器，國小到國中讀的都是音樂班。在就讀[[台南女中|台南女中]]時期，常參加各類歌唱比賽，並代表全體畢業生上台演唱畢業歌曲。於[[高雄醫學大學|高雄醫學大學]]主修心理學期間，曾參加[[台灣電視公司|台灣電視公司]]的選秀節目《[[我要當歌手|我要當歌手]]》與《[[金牌麥克風|金牌麥克風]]》<ref>{{Cite web |url =https://www.ettoday.net/news/20141202/432128.htm?t=善耕365／你還不知道的6位「人美心也美」公益女孩 |title = 善耕365／你還不知道的6位「人美心也美」公益女孩 |date = 2014年12月2日 |publisher = 東森新聞雲 |language = zh-tw |accessdate = 2017年11月4日}}</ref><ref>{{Cite web |url=http://koobii.com.tw/pl/koobii人氣嚴選23【高醫醫社─黃怡瑄】用甜美嗓音征服/|title = Koobii人氣嚴選23【高醫醫社─黃怡瑄】用甜美嗓音征服你的心 |date = 2014年3月16日 |publisher = koobii 高校誌|language = zh-tw |accessdate = 2017年11月4日}}</ref>。黃怡瑄的祖母曾因為她選擇踏入歌唱演藝事業而對她不諒解，經寫信給祖母溝通後，方獲支持。<ref name="pk">{{cite news|title=TUD出道鬧革命 斷祖孫關係真慘|url=http://www.appledaily.com.tw/appledaily/article/entertainment/20170925/37792402/|accessdate=2017-10-28|publisher=蘋果日報|date=2017-09-25}}</ref><ref>{{Cite web |url =https://news.tvbs.com.tw/entertainment/793972 |title = 新生女團TUD 　期望醜小鴨變天鵝 |date = 2017-10-20 |language = zh-tw |accessdate = 2017年11月4日}}</ref>

=== 團長兼主唱 ===

黃怡瑄在[[TUD|TUD]]團內除擔任主唱外，亦身兼團長一職，如其他團員一樣，她也出身於網路知名節目《校花點點名》<ref>{{Cite web |url =http://ent.ltn.com.tw/news/breakingnews/1972157 |title = 醫大校花愛「大便」 阿本崩潰喊怪胎 |date = 2017年9月24日 |publisher = 自由時報 |language = zh-tw |accessdate = 2017年11月4日}}</ref><ref>{{Cite News |url =https://news.tvbs.com.tw/entertainment/802393 |title =最正校花萌系怪獸舞教學　TUD臥虎藏龍個個明星臉 |publisher=TVBS|date = 2017-10-30 |language = zh-tw |accessdate = 2017年11月4日}}</ref><ref>{{Cite web |url =http://star.ettoday.net/news/1026487?t=全台最正5位校花長這樣！　｢沒碰過爸爸以外男體」 |title = 全台最正5位校花長這樣！　｢沒碰過爸爸以外男體」 |date = 2017年10月6日 |publisher = 東森新聞雲 |language = zh-tw |accessdate = 2017年11月4日}}</ref>。TUD成軍後，歷經一年的籌畫與培訓，於2017年9月29日發行首張數位EP《Monster》<ref>{{Cite web |url =https://www.nownews.com/news/20170924/2612930 |title = 「校花女團」TUD學歷嚇嚇叫　　團員卻被家人斷金援 |date = 2017-10-20 |language = zh-tw |accessdate = 2017年11月4日}}</ref><ref>{{Cite web |url =http://www.chinatimes.com/realtimenews/20170924002613-260404 |title =女團遭家人斷金援 拚出道練就一身「金鐘罩 |publisher=中國時報|date = 2017-09-24 |language = zh-tw |accessdate = 2017年11月4日}}</ref>。在《Monster》EP第三主打歌《不遠》的MV中，黃怡瑄親自擔任大提琴與鋼琴的演奏<ref>[https://www.youtube.com/watch?v=o1lKmdiWFV8  TUD《不遠》 Official MV HD]</ref>。
