'''黃家維'''（英文名：{{lang|en|Wong Ka Wai, Anthony}}，{{bd|1981年||||catIdx=Huang黃}}），是已故[[民建聯|民建聯]]主席[[香港立法會|立法會]]議員[[馬力_(立法會議員)|馬力]]的議員助理、民建聯政制事務副發言人、[[青年民建聯|青年民建聯]]副主席、[[東區區議會|東區區議會]]環境及衛生事務委員會增選委員。

== 簡歷 ==
黃家維2003年獲[[香港大學|香港大學]]頒授社會科學學士，主修政治及公共行政學。

他曾參與[[2007年香港區議會選舉|2007年香港區議會選舉]]競逐東區樂康選區議席，挑戰競逐連任的前立法局議員、選區內居屋[[樂翠臺|樂翠臺]]業主立案法團秘書，[[社會民主連線|社會民主連線]][[曾健成|曾健成]]，以1454對1912票落敗。

== 馬力立法會議員助理 ==
黃家維於2004年向當時剛勝出立法會香港島[[2004年香港立法會選舉|地區直選]]的民建聯主席馬力自薦當其議員助理，並要求到其出生的地方柴灣服務，並於2006年1月14日於[[柴灣|柴灣]][[高威閣|高威閣]]開設地區服務處。<ref> [http://paper.wenweipo.com/2007/08/11/HK0708110019.htm''徒兒灑淚 難忘恩師提攜, 文匯報, 2007年8月11日''] </ref>

== 政制事務副發言人 ==
黃家維除作為馬力議員助理於港島東區服務居民外，亦於2006年獲民建聯任為政制事務副發言人。2007年7月6日與[[譚耀宗|譚耀宗]]、[[蘇錦樑|蘇錦樑]]就香港政制發展闡述民建聯的立場。建議採取先易後難的方法，先於2017年推行[[行政長官|行政長官]]普選。<ref> [http://www.dab.org.hk/tr/main.jsp?content=article-content.jsp&articleId=5543&categoryId=2285  ''民建聯對未來政制發展的建議''] {{webarchive|url=https://web.archive.org/web/20070929093222/http://www.dab.org.hk/tr/main.jsp?content=article-content.jsp&articleId=5543&categoryId=2285 |date=2007-09-29 }} </ref>
