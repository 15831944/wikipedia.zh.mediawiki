'''黃惠蘭'''，（英文名︰Wong Wai Lan），[[香港|香港]]出生，是香港聾人機構「[[龍耳|龍耳]]」的手語主任<ref>{{Cite news|language=zh-hk |url=https://www.police.gov.hk/offbeat/1101/chi/6126.html|title=西區警區訓練人員與聾啞人士溝通|publisher=警聲|date=2017-12-06 至19 第1101期|accessdate=2018-05-30}}</ref><ref>{{Cite news|language=zh-hk |url=https://jmc.hksyu.edu/shuo/%E7%A4%BE%E4%BC%81%E6%8E%A8%E6%89%8B%E8%AA%9E%E9%A4%90%E5%BB%B3%E6%95%99%E9%A3%9F%E5%AE%A2%E3%80%8C%E9%80%A0%E6%89%8B%E3%80%8D/|title=社企推手語餐廳教食客「造手」|publisher=說在線|date=2015-10-09|accessdate=2018-05-30}}</ref>，亦是[[聾人粵劇團|聾人粵劇團]]的成員。現她符合《[[香港手語翻譯員名單|香港手語翻譯員名單]]》基本條件的手語翻譯員。

== 聾人粵劇團 ==
在2014年，黃惠蘭某次興起，將自己的[[粵劇|粵劇]]造型相片放在社交平台，卻引起不少聾人的興趣。他們表示，很希望能學習粵劇，可惜一直沒有機會。於是，在黃惠蘭發起下，聾人粵劇團就此成立<ref>{{Cite news|language=zh-hk |url=https://www.thestandnews.com/society/%E8%81%BE%E4%BA%BA%E9%83%BD%E6%9C%89%E7%B2%B5%E5%8A%87%E5%9C%98/|title=聾人都有粵劇團|publisher=[[立場新聞|立場新聞]]|date=2017-03-04|accessdate=2018-05-30}}</ref><ref>{{Cite news|language=zh-hk |url=http://www.metrodaily.hk/metro_news/%E6%96%B0%E8%81%9E%E5%B0%88%E9%A1%8C-%E8%97%9D%E8%A1%93%E7%84%A1%E5%88%86%E5%82%B7%E8%88%87%E5%81%A5-%E8%81%BE%E4%BA%BA%E9%83%BD%E6%9C%89%E7%B2%B5%E5%8A%87%E5%9C%98/|title=新聞專題 藝術無分傷與健 聾人都有粵劇團|publisher=[[都市日報|都市日報]]|date=2017-03-03|accessdate=2018-05-30}}</ref>。

她指出，此粵劇團會在表演現場安排即時的[[手語傳譯|手語傳譯]]、樂隊和粵曲演唱者，當響起絲竹聲及演唱聲，台上的聾演員們會配合手語傳譯的即時提示，演出相應的劇情。演員們表演前要將[[粵劇|粵劇]]的内容消化、記熟情節、同時演繹特别編製的手語動作及做出劇目內所需功架<ref>{{Cite news|language=zh-hk |url=http://hk.on.cc/hk/bkn/cnt/news/20160723/bkn-20160723083003694-0723_00822_001.html|title=東網透視：看得見的掌聲　聾人粵劇無聲有情|publisher=[[東網電視|東網電視]]|date=2016-07-23|accessdate=2018-05-30}}</ref>。

她續說，從來沒有人將粵劇手語化，此劇團是全世界第一個聾人粵劇。她無意間發現很多學生喜歡粵劇，鑼鼓聲大，聾人可感受到聲音，但始終無辦法好好的欣賞。於是，她找來了懂手語的粵劇老師，組成了聾人粵劇團，除了打破社會對聾人的成見外，更一償不少聾人的心願<ref>{{Cite news|language=zh-hk |url=https://www.hk01.com/%E7%A4%BE%E5%8D%80%E5%B0%88%E9%A1%8C/36093/%E5%85%A8%E6%B8%AF%E9%A6%96%E5%80%8B%E8%81%BD%E9%9A%9C%E4%BA%BA%E5%A3%AB%E7%B2%B5%E5%8A%87%E5%9C%98-%E5%94%AF%E4%B8%80%E7%94%B7%E5%9C%98%E5%93%A1-%E8%A6%81%E4%B8%96%E4%BA%BA%E7%9F%A5%E9%81%93%E8%81%BE%E4%BA%BA%E9%83%BD%E5%8F%AF%E5%81%9A%E7%B2%B5%E5%8A%87|title=全港首個聽障人士粵劇團　唯一男團員：要世人知道聾人都可做粵劇|publisher=[[香港01|香港01]]|date=2016-08-09|accessdate=2018-05-30}}</ref><ref>{{Cite news|language=zh-hk |url=http://hd.stheadline.com/news/daily/hk/486834/|title=首創聾人粵劇 聽障演員「動手」做大戲|publisher=[[頭條日報|頭條日報]]|date=2016-08-26|accessdate=2018-05-30}}</ref>。

== 手語旁述世界盃 ==
[[2014年國際足協世界盃|2014年國際足協世界盃]]，在[[奧海城|奧海城]]直播的世界盃賽事中，有15場首次有手語翻譯。黃惠蘭指出用[[手語旁述世界盃|手語旁述世界盃]]有困難，因九成的足球術語沒有相應的手語<ref>{{Cite news|language=zh-hk |url=https://hk.news.appledaily.com/local/daily/article/20140529/18735960|title=蘋果生活：義工自創插水、詐糊手勢 手語講波　聾人過癮睇世盃|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]]|date=2014-05-29|accessdate=2018-05-30}}</ref>。面對創新的世界盃旁述，她指足球術語要細心找出最合適的手語來代替<ref>{{Cite news|language=zh-hk |url=https://hk.sports.appledaily.com/sports/daily/article/20140521/18726923|title=話題：手語世盃無聲仿有聲|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]]|date=2014-05-21|accessdate=2018-05-30}}</ref>。對於香港聾人的支援不足，她表示曾多次向[[平等機會委員會|平等機會委員會]]反映問題<ref>{{Cite news|language=zh-hk |url=https://hk.sports.appledaily.com/sports/daily/article/20140521/18726935|title=港府支援聽障不足|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]]|date=2014-05-21|accessdate=2018-05-30}}</ref>。

== 電視的手語傳譯 ==
香港公共廣播的手語傳譯支援不足，黃惠蘭指大部分電視節目，包括新聞報道缺乏手語傳譯，就算有節目提供手語傳譯，但手語框佔螢幕的比例不足十二分之一，讓聾人觀看時感到吃力<ref>{{Cite news|language=zh-hk |url=http://std.stheadline.com/daily/news-content.php?id=216648&target=2|title=首個手語頻道「龍耳」電視啟播|publisher=[[星島日報|星島日報]]|date=2014-12-05|accessdate=2018-05-30}}</ref><ref>{{Cite news|language=zh-hk |url=https://jmc.hksyu.edu/shuo/%E6%A9%9F%E6%A7%8B%E6%8E%A8%E6%B8%AF%E9%A6%96%E9%96%93%E6%89%8B%E8%AA%9E%E9%9B%BB%E8%A6%96%E5%8F%B0/|title=機構推港首間手語電視台|publisher=說在線|date=2014-12-05|accessdate=2018-05-30}}</ref>。
