'''黃子菲'''（{{lang|en|'''Ava, Wong Chi Fei'''}}，{{bd|1992年|4月28日|catIdx=Huang黃子菲}}）[[香港|香港]]藝人，模特兒，主演《夜。上嗨》(未上演)，有翻版[[張柏芝|張柏芝]]少年版之稱。她在[[微博|微博]]、[[Instagram|Instagram]]人氣有多達30萬人，黃子菲有被記者稱為「微博小花」,「IG女神」。

現為[[寰宇|寰宇]]旗下藝人，模特兒公司為[https://web.archive.org/web/20090923182739/http://www.uniquemodel.com.hk/ Unique Model Limited]。於2006年參加YES!全港校花校草比賽，與First Cast簽約。後來被[[Starj_&_Snazz娛樂|Starj & Snazz娛樂]]公司經理人賞識簽約成為旗下藝人（已經完約）。

==創業==
據黃子菲於2016年初透露，她在尖沙咀[[厚福街|厚福街]]The Studio開了一所韓國入口時裝店，並親力親為地到韓國入貨，和自任模特兒為時裝拍造型照，於其Instagram 發怖宣傳。<ref name = ''S1''>香港新城數碼音樂台《想創你個心》電台節目訪問，2015年12月12日播出。</ref>

另外，黃子菲在其父親於[[佐敦|佐敦]]區經營一所獲得[[澳門|澳門]][[特許經營權|特許經營權]]的火鍋店，負責業務宣傳的工作。<ref name = ''S1''/>

==個人生活==
黃子菲在2012年1月20日下嫁大律師黃添偉，當時黃子菲19歲，丈夫比她年長約10年。在2016年，黃子菲於個人instagram上承認已和黃添偉離婚，並表示雙方已於2014年分開。<ref>{{cite news |language = 中文 |author = 陳寶 |url = http://orientaldaily.on.cc/cnt/entertainment/20120106/00282_079.html |title = 黃子菲嫁大律師 繼續工作 |publisher = 東方日報  |date = 2012-01-06 |accessdate = 2016-03-05}}</ref>
