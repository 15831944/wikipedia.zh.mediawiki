'''黃巾軍人物'''為[[後漢|後漢]][[中平|中平]]年間參與[[黃巾之亂|黃巾之亂]]的[[黃巾軍|黃巾軍]]人物，包含正史《[[三國志|三國志]]》所記載的史實人物及《[[三國演義|三國演義]]》中的[[虛構人物|虛構人物]]。

== 正史人物 ==
=== 領導人物 ===
==== 張角 ====
'''張角'''（？－184年），鉅鹿人（今[[河北|河北]][[平鄉|平鄉]]），黃巾軍的領導人，在[[黃巾之亂|黃巾之亂]]中逝世。自稱「天公將軍」。

==== 張寶 ====
'''張寶'''（？－184年），鉅鹿人（今[[河北|河北]][[平鄉|平鄉]]），黃巾軍的領導人，[[張角|張角]]二弟。[[中平|中平]]元年（184年）隨張角[[起事|起事]]，被[[太平道|太平道]]徒眾稱為「地公將軍」。其弟[[張梁|張梁]]在廣宗（今[[河北|河北]][[威縣|威縣]]）戰死，張寶被屬下[[嚴政|嚴政]]刺殺。

==== 張梁 ====
'''張梁'''（？－184年），鉅鹿人（今[[河北|河北]][[平鄉|平鄉]]），黃巾軍的領導人，張角與張寶之弟。[[中平|中平]]元年（184年）隨兄[[起事|起事]]，被[[太平道|太平道]]徒眾稱為「人公將軍」。遭到朝廷所派左中郎將[[皇甫嵩|皇甫嵩]]進攻時，張梁率三萬大軍在[[廣宗|廣宗]]（今[[河北|河北]][[威縣|威縣]]）進行反擊。後因疏忽，遭到漢軍夜襲，兵敗身亡，本部士兵皆被[[坑殺|坑殺]]。至此，黃巾起義基本宣告失敗。

=== 其他人物 ===
==== 馬元義 ====
'''馬元義'''（？－184年），黃巾軍的組織者之一。光和六年（183年），馬元義奉黃巾軍首領張角之命，籌畫起事的準備工作。馬元義聚集荊州、揚州信徒數萬人於鄴城，並收買中常侍封諝、徐奉等為內應，準備於次年（甲子年）三月初五（甲子日）內外同時起事。由於張角弟子濟南人唐周的告密，馬元義於中平元年（184年）正月被捕，車裂於洛陽，同時被殺的還有教眾近千人。由於馬元義的被殺，迫使張角提前於二月起兵發動叛亂。

==== 唐周 ====
'''唐周'''（？－？)，[[濟南|濟南]]人，[[黃巾軍|黃巾軍]]天公將軍[[張角|張角]]的弟子。[[光和|光和]]六年（183年），[[馬元義|馬元義]]奉黃巾軍首領張角之命，籌劃起事的準備工作。馬元義聚集[[荊州_(古代)|荊州]]、[[揚州_(古代)|揚州]]信徒數萬人於[[鄴城|鄴城]]，並收買[[中常侍|中常侍]][[封諝|封諝]]、[[徐奉|徐奉]]等為內應，準備於次年（甲子年）三月初五（甲子日）內外同時起事。由於[[張角|張角]]弟子濟南人唐周的告密，馬元義於於[[中平|中平]]元年（184年）正月被捕，[[車裂|車裂]]於[[洛陽|洛陽]]，同時被殺的還有教眾近千人。由於馬元義的被殺，迫使張角提前於二月起兵發動叛亂。在《三國演義》作'''唐州'''。

=== 南陽人物 ===
==== 張曼成 ====
'''張曼成'''（？－184年），[[南陽|南陽]]黃巾軍首領。[[光和|光和]]七年[[二月|二月]]（184年），太平教[[大方|大方]]（總指揮官）[[馬元義|馬元義]]事敗被捕於[[洛陽|洛陽]]車裂，張曼成聞訊立即率眾在南陽起兵。[[三月|三月]]庚子日擊斬南陽[[郡守|郡守]][[褚貢|褚貢]]後，屯兵[[宛縣|宛縣]]（南陽郡政府所在地）城下百餘日；六月，遭新任南陽郡長[[秦頡|秦頡]]反攻擊斬。

==== 趙弘 ====
'''趙弘'''（？－184年），南陽黃巾渠帥張曼成麾下，張曼成被南陽太守秦頡擊斃後，被擁立為首領。[[漢靈帝|漢靈帝]][[光和|光和]]七年（西元184年）六月，南陽黃巾軍首領[[張曼成|張曼成]]於宛縣遭東漢政府南陽郡長[[秦頡|秦頡]]反攻戰死。南陽黃巾軍餘眾擁立趙弘當元帥，再振聲勢，擁眾十餘萬，攻拔宛縣作為根據地。時東漢政府鎮賊[[中郎將|中郎將]][[朱儁|朱儁]]與[[荊州|荊州]][[刺史|刺史]][[徐璆|徐璆]]等率軍聯合包圍宛縣數月，於八月在朱儁軍團攻勢下被擊斬。

==== 韓忠 ====
'''韓忠'''（？－184年），南陽黃巾渠帥張曼成麾下，張曼成被南陽太守[[秦頡|秦頡]]擊斃後，繼續擁立趙弘為帥，但趙弘隨即被朱儁再次剿滅。韓忠率領余部佔據宛城（今河南南陽），與朱儁相持，後被朱儁擊敗而投降，但卻被南陽太守秦頡所殺

==== 孫夏 ====
'''孫夏'''（？－184年），《三國演義》作'''孫仲'''。南陽黃巾渠帥張曼成麾下，繼韓忠為南陽黃巾首領。因主力被殲，危城不守，餘部於十一月西移鄂（今河南南陽市東北）之精山（今河南南陽市西北），朱儁率軍追擊，孫夏等萬餘人戰死。

=== 青州黃巾 ===
==== 張饒 ====
張饒等群輩二十萬眾從冀州還，孔融逆擊，為張饒所敗。

==== [[管亥|管亥]] ====
管亥圍困孔融。孔融遣東萊太史慈求救於平原相劉備。

==== 徐和 ====
濟南黃巾軍首領，起義後進攻濟南，為曹操部將夏侯淵、臧霸、呂虔擊敗，被殺。

==== 司馬俱 ====
樂安黃巾軍首領，起義後進攻樂安，為曹操部將夏侯淵所破。

=== 白波賊 ===
==== 郭泰 ====
黃巾餘賊郭泰起於西河白波谷，轉寇太原，遂破河東，百姓流轉三輔，號為「白波賊」，眾十餘萬。董卓遣中郎將牛輔擊之，不能卻。<ref>《後漢書》作者范曄父名泰，故改郭泰為郭太。</ref>

==== [[楊奉|楊奉]] ====
李傕將楊奉本白波賊帥。

==== [[韓暹|韓暹]] ====
獻帝自長安東歸，右賢王去卑與白波賊帥韓暹等侍衞天子。

==== 胡才 ====
[[楊奉|楊奉]]戰友。[[董卓|董卓]]死後由李傕等人掌權，[[兴平_(东汉)|興平]]二年（195年），楊奉領兵叛逃李傕後天子令楊奉眾人護送其回[[洛陽|洛陽]]。郭汜聯合李傕率領追兵追到[[弘農|弘農]]的[[曹陽|曹陽]]。楊奉緊急招喚[[河東|河東]]故友[[白波|白波]]帥[[韓暹|韓暹]]、胡才、[[李樂|李樂]]抵抗李傕等人。後楊奉戰敗，與天子連夜乘船逃走到河東[[大陽|大陽]]縣。天子為[[張楊_(東漢)|張楊]]迎接，但仍然為楊奉等人脅持，升胡才為[[征東將軍|征東將軍]]。[[建安_(东汉)|建安]]元年（196年），胡才、[[李樂|李樂]]和[[董承|董承]]等人就汉献帝是否回洛阳而相互攻击，為天子所阻。胡才後來与[[李樂|李樂]]一起留在河東郡，胡才為怨家所害。

==== 李樂 ====
[[楊奉|楊奉]]戰友。[[董卓|董卓]]死後由李傕等人掌權，[[興平_(東漢)|興平]]二年（195年），楊奉領兵叛逃李傕後天子令楊奉眾人護送其回[[洛陽|洛陽]]。郭汜聯合李傕率領追兵追到[[弘農|弘農]]的[[曹陽|曹陽]]。楊奉緊急招喚[[河東|河東]]故友[[白波|白波]]帥[[韓暹|韓暹]]、[[胡才|胡才]]、李樂抵抗李傕等人，李樂主要負責護衛天子。後楊奉戰敗，與天子連夜乘船逃走到[[河東郡_(中國)|河東]][[大陽縣|大陽縣]]。天子為[[張楊_(東漢)|張楊]]迎接，但仍然為楊奉等人脅持。护卫的[[董承|董承]]、[[张杨|张杨]]和汉献帝想回洛阳，李樂、[[楊奉|楊奉]]、胡才等拒绝，想把汉献帝留在河东，导致[[董承|董承]]和[[李樂|李樂]]等人相互攻击，在汉献帝劝和下止战。李樂等服从汉献帝的指令护送汉献帝前往洛阳。途中李乐反悔，诈称路上会遇到匈奴，汉献帝不从，但最终并没有遇到匈奴。李乐害怕汉献帝责备，于是留在河东，不再护送汉献帝<ref>《后汉纪》卷29：时董承、张阳欲天子还洛阳，杨奉、李乐不欲。尚书上官洪言还洛之议，李乐辅洪。由是诸将错乱，更相疑贰。董承奔野王，韩暹屯闻喜。胡才、杨奉之坞乡，欲攻韩暹，上使人喻止之。夏五月丙寅，遣使至杨奉、李乐、韩暹营，求送至洛阳，奉等从诏。六月乙未，车驾幸闻喜。杨奉、胡才悔令乘舆去，乃与李乐议，欲还大驾，诈言「当游渑池东，以避匈奴」，上不从。庚子，车驾从北道出，傍山而东，无匈奴寇。李乐愧其言，惧而辞还。</ref>。李樂後來留在河東郡病死。而在《[[三國演義|三國演義]]》中，李樂後來則轉而與李傕、郭汜勾結，詐稱李傕、郭汜，追上車駕，意圖劫駕，與楊奉部將即後來的曹魏名將[[徐晃|徐晃]]單挑時僅一回合便被劈死。

=== 豫州黃巾 ===
==== 波才 ====
'''波才'''（？－184年），黃巾軍將領，[[中平|中平]]元年(184年)在[[潁川|潁川]]（今河南[[禹州|禹州]]）起事。據史書記載，波才曾率領黃巾軍大敗右中郎將[[朱儁|朱儁]]，並圍左中郎將[[皇甫嵩|皇甫嵩]]於長社（今河南[[長葛|長葛]]西），後來被皇甫嵩用火攻所破，追擊波才於[[陽翟|陽翟]]（今禹州）。

==== 彭脫 ====
'''彭脫'''（？－184年），黃巾軍渠帥。[[皇甫嵩|皇甫嵩]]擊破[[波才|波才]]於[[長社|長社]]，並與[[朱儁|朱儁]]乘勝追討[[汝南|汝南]]、[[陳國|陳國]]之黃巾賊，追[[波才|波才]]於[[陽翟|陽翟]]，擊破彭脫於[[西華|西華]]。剩下的黃巾賊皆潰散、投降，三郡悉數平定。

==== [[何儀|何儀]] ====
黃巾軍將領。黃巾之亂後，一直佔據汝南與潁川一帶。據《三國志》記載，何儀與劉辟、黃邵、何曼等人，各擁有數萬軍隊，先回應袁術，後來依靠孫堅。建安元年（196年）二月，曹操帶兵攻擊，殺死劉辟與黃邵，何儀率眾投降。

==== 黃邵 ====
'''黃邵'''（？－196年？或197年？），黃巾軍將領。黃巾之亂後，一直佔據汝南與潁川一帶。據《三國志》記載，何儀與劉辟、黃邵、何曼等人，各擁有數萬軍隊，後被曹操帶兵擊殺。

==== [[何曼|何曼]] ====
黃巾軍將領。建安元年，汝南、潁川黃巾何儀、劉辟、黃邵及曼等，眾各數萬，初應袁術，又附孫堅。二月，曹操進軍討破之，斬辟、邵等，儀及其眾皆降。在《三國演義》裡，何曼自稱「截天夜叉」，被曹洪用拖刀計所殺。

==== [[刘辟_(东汉)|劉辟]] ====
黃巾軍將領。曹公與袁紹相拒於官渡，汝南黃巾劉辟等叛曹公應紹。紹遣先主將兵與辟等略許下。

==== 龔都 ====
《三國志》中記載為'''共都'''。黃巾軍將領，其部活動在汝南（治今河南上蔡西南）一帶。建安六年（201）袁紹遣先主將本兵復至汝南，與都等合，眾數千人。曹操遣蔡陽來攻，為他所破。後曹操親自南征，劉備投奔劉表，其部散去。

==== 吳霸 ====
黃巾軍將領，為[[李通_(三国)|李通]]所敗，歸降李通。

=== 益州黃巾 ===
==== 馬相 ====
荆州南阳(今河南南阳)人，[[益州|益州]]民變首領。[[中平|中平]]五年（公元188年），馬相與[[趙袛|趙袛]]等人在益州[[緜竹|緜竹]]縣起兵，在兩天內成功號召自稱「黃巾」；擊斬東漢朝廷益州刺史（益州牧）[[郤儉|郤儉]]，進攻[[巴郡|巴郡]]、[[犍為|犍為]]。不足一月連破三郡，擁眾數萬人，遂自稱皇帝。後遭東漢政府益州從事（益州州府參謀官）[[賈龍|賈龍]]所率官民等反擊，堅持數日被逐。

==== 趙祗 ====
以馬相為首。

[[會稽|會稽]]黃巾賊帥，被郡吏[[留贊|留贊]]斬殺。

[[吳郡|吳郡]]黃巾餘黨，[[建安|建安]]七年(西元202年)，陳敗和萬秉在吳郡作亂，後被吳郡太守[[朱治|朱治]]平定。

==== 陳寶 ====
[[吳郡|吳郡]]黃巾賊，欲強姦[[許升|許升]]之妻[[呂榮|呂榮]]，呂榮不從，憤而殺之。<ref>《太平御覽》卷53引《列女後傳》：呂榮者，吳郡許升之妻也。升爲賊所害，榮貞固自守，黃巾賊陳寶欲幹穢之，榮執節不聽，寶遂殺榮。是日疾風暴雨，雷電晦冥，寶恐懼叩頭謝，葬之而去。糜府君聞榮高行，遣主簿祭之，又出錢助縣爲冢于嘉興郭裏墟北，名曰義婦阪。</ref>

== 虛構人物 ==
=== 程遠志 ===
《三國演義》人物，以[[黃巾|黃巾]]賊將的身份登場。[[光和|光和]]七年（西元184年），黃巾軍起義，程遠志與副將[[鄧茂|鄧茂]]統兵五萬進犯涿郡。[[劉焉|劉焉]]令[[鄒靖|鄒靖]]與[[劉備|劉備]]兄弟三人，前去迎戰。兩軍在大興山相對。最後副將[[鄧茂|鄧茂]]被張飛以丈八蛇矛給刺死，本人也被關羽以青龍偃月刀給劈為兩半。

在正史中，有提及幽州廣陽黃巾軍殺死太守[[劉衛|劉衛]]及刺史[[郭勳_(東漢)|郭勳]]，但未明言統率此黃巾軍的將領是誰。

=== 鄧茂 ===
《三國演義》人物，以[[黃巾|黃巾]]賊將的身份登場。[[光和|光和]]七年（西元184年），黃巾軍起義，與主將[[程遠志|程遠志]]統兵五萬進犯幽州涿郡。在大興山與劉備等幽州軍相對。與張飛交戰，只一個回合就被張飛殺死，是張飛的得意兵器[[丈八蛇矛|丈八蛇矛]]的第一個犧牲者。

=== 高昇 ===
《三國演義》人物，[[黃巾|黃巾]]將領。[[光和|光和]]七年（184年）黃巾之亂時，與主將[[張寶|張寶]]屯兵[[曲陽|曲陽]]。[[朱儁|朱儁]]派[[劉備|劉備]]作為先鋒迎擊張寶，張寶派副將高昇迎敵，劉備派[[張飛|張飛]]迎敵。張飛與高昇交戰數回合，刺死了高昇<ref name="張飛">{{Cite wikisource | title = 三國演義 | wslink = 三國演義/第002回 | wslanguage = zh | last = 羅 | first = 贯中 | authorlink = s:author:羅貫中 | quote = 是時曹操自跟皇甫嵩討張梁，大戰於曲陽。這裡朱儁進攻張寶。張寶引賊眾八九萬，屯於山後。儁令玄德為其先鋒，與賊對敵。張寶遣副將高昇出馬搦戰。玄德使張飛擊之。飛縱馬挺矛，與昇交戰，不數合，刺昇落馬。}}</ref>。

=== [[杜遠|杜遠]] ===
《三國演義》人物，黃巾軍餘黨，聚眾成為匪首。他將關羽所護送的劉備夫人們綁架到自己的地盤，後被同夥廖化所殺。

=== 卞喜 ===
《三國演義》人物。根據《三國演義》第27回記載，卞喜為[[曹操|曹操]]部下[[汜水關|汜水關]]守將。[[關羽|關羽]]護送[[甘夫人|甘]]、[[麋夫人|麋]]二嫂投奔[[劉備|劉備]]，經過第三關汜水關。他知關羽勇猛難敵，假意在[[鎮國寺|鎮國寺]]內設宴款待，欲借機謀害，但[[寺廟|寺廟]]裡的[[僧人|僧人]][[普淨|普淨]]帶關羽去喝茶，示意關羽有人要殺他，陰謀被關羽識破，將他殺死。卞喜是關羽在[[過五關斬六將|過五關斬六將]]中被斬的第四名將領。

=== [[周倉|周倉]] ===
《三國演義》人物，原黃巾軍之武將，後跟隨關羽，忠心不二；在聽說關羽兵敗被殺後，自刎而死。

=== 裴元紹 ===
《三國演義》人物，原黃巾軍之武將。黃巾起義失敗之後，與周倉一同率領殘部在山中落草當山賊。西元200年，在關羽欲返劉備旗下，在突破曹操的五道關卡後路過其落草之地，與周倉一同向關羽要求能以期成為關羽家臣。但此時僅周倉同行，其他弟兄則於山中等待。不久後，因其欲奪偶然路過的趙雲之馬，反遭討伐戰敗身死。
