'''黃錦輝'''畢業於[[香港大學|香港大學]]生物系<ref>{{cite web |url=http://cepa.ajc.org.hk/interview_WongKF.html |title=存档副本 |accessdate=2013-11-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20100815014557/http://cepa.ajc.org.hk/interview_WongKF.html |archivedate=2010-08-15 }}</ref>，是[[新思維_(香港)|新思維]]司庫，亞太區壽險總會前會長<ref>[http://hk.apple.nextmedia.com/news/art/20101008/14531204]</ref>。在2003年及2004年度曾任香港人壽保險從業員協會會長<ref>[http://www.cuhk.edu.hk/cpr/pressrelease/030901c.htm]</ref>。他亦是[[2004年香港立法會選舉|2004年香港立法會選舉]]港島區的候選人之一，但結果以區內最低票數落敗。<ref>{{cite web |url=http://www.elections.gov.hk/elections/legco2004/tc_chi/results/rs_gc_hk.html |title=存档副本 |accessdate=2011-08-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20110805121008/http://www.elections.gov.hk/elections/legco2004/tc_chi/results/rs_gc_hk.html |archivedate=2011-08-05 }}</ref>。
