'''黃幡神'''（{{lang|ja|おうはんじん}}或{{lang|ja|おうばんしん}}）為[[日本|日本]]民間信仰、[[九曜|九曜]]之一的[[神祇|神祇]]「[[羅睺|羅睺]]」，一般在[[聚落|聚落]]村莊中心、村內外的叉路或三叉口以石碑的型態立祀，屬於[[道祖神|道祖神]]的一種。

== 概要與解說 ==
雖然現在黃幡神以[[道祖神|道祖神]]之型態成為日本民間信仰，但來源乃[[印度神話|印度神話]]裡的蛇神[[羅睺|羅睺]]。傳說當羅睺吞食日月時便發生[[日蝕|日蝕]]、[[月蝕|月蝕]]，[[佛教|佛教]]傳入日本後[[神佛習合|神佛習合]]，便與引起日蝕的[[素戔嗚尊|素戔嗚尊]]相互習合<ref>日本神話裡，行為怪誕荒亂的素戔嗚尊嚇得姐神[[天照大神|天照大神]]躲入[[天岩戶|天岩戶]]，世界頓時日月無光。</ref>。再者，習合的結果導致該神明與[[密宗|密宗]]具有深切關係，一般視之為[[摩利支天|摩利支天王]]。

此外，黃幡神也是司掌方位吉凶的[[八將神|八將神]]（{{lang|ja|はっしょうじん}}）之一，別名為「萬物之墓之方」和「兵亂之神」<ref>參看{{ja}}[http://www.ndl.go.jp/koyomi/kotoba/02_eto03.html 国立国会図書館：暦の中のことば方位神] {{webarchive|url=https://web.archive.org/web/20130122170940/http://www.ndl.go.jp/koyomi/kotoba/02_eto03.html |date=2013-01-22 }}。</ref>。在此方位若是動土等事項則為凶，與武藝相關之事物則為吉。

=== 種類 ===
雖然黃幡神神像外觀酷似[[道祖神|道祖神]]，也能區分成下述的種類：
# 文字黃幡神：石碑上以[[漢字|漢字]]或[[梵語|梵語]]刻有「羅睺」、「黃幡」等相關字句。
# 蛇頭黃幡神：以[[不動明王|不動明王]]憤怒貌顯現，頭部則為九頭蛇，其原型來自羅睺。
# 日月黃幡神：在羅睺、黃幡神的上下左右刻上[[太陽|太陽]]、[[月亮|月亮]]等文字或圖案，此種神像乃是[[素戔嗚尊|素戔嗚尊]]相互習合下的產物。
# 蛇形黃幡神：神像型態以[[蛇|蛇]]顯現，有人說原型來自羅睺，也有人認為其半身源自[[計都|計都]]。
# 混合黃幡神：前述種類混合在一起的黃幡神神像。

== 其他 ==
[[中國|中國]][[四大名著|四大名著]]《[[水滸傳|水滸傳]]》裡有個角色名曰「卓萬里」，人稱黃幡神，乃方蠟麾下頭領。官拜潤州統制官，後被孔明、孔亮所擒。
