'''黃子綝'''（{{bd|1986年||2015年|1月4日}}），生於臺灣[[彰化縣|彰化縣]][[員林市|員林鎮]]（今員林市），15歲時經檢查發現罹患{{link-en|多發性神經纖維瘤|Neurofibromatosis}}第二型，此疾使她失去[[聽力|聽力]]和{{link-en|平衡感|Sense of balance}}，又因多次手術使顏面神經失去功能，因此她不能言語、沒有表情且肢體癱瘓，必須藉行動[[輔具|輔具]]移動、以手語或鍵盤輸入文字來與人溝通。她專注於[[繪畫|繪畫]]與文學創作，繪有多幅作品，就讀[[南華大學_(臺灣)|南華大學]]文學系期間曾二度舉辦個人畫展，並於2013年獲得[[總統教育獎_(中華民國)|總統教育獎]]。2015年1月4日，因吞嚥[[早餐|早餐]]造成氣管堵塞而逝世，得年28歲。

== 生平 ==
=== 早年 ===
黃子綝出生於彰化縣員林市[[百果山|百果山]]出水巷，下有一弟一妹。自小學起因做著處理水果的打工以賺取零用錢<ref name = "書 6-7"/>。雖然父母經常爭吵、且長年酗酒的父親經常對她[[家庭暴力|施暴]]，但她並不在意，常藉著爬樹轉換心情<ref name = "書 10-11"/>，她自嘲這段經歷將自己培養成「一個野性又大膽的女生」<ref name = "書 243"/>。在黃子綝就讀國中一年級時，父親因[[肝癌|肝癌]]病逝。黃子綝開始學習繪畫，夢想成為一名漫畫家，並開始研究[[占星術|占星術]]和命理學，此時的她也意識到自己的聽力正逐漸退化<ref name="中央總統"/><ref name = "書 76.92"/>。國中二年級時，經醫師診斷發現腦部長出多顆腫瘤，罹患和母親相同的{{link-en|多發性神經纖維瘤|Neurofibromatosis}}，始被認定為殘障人士<ref name = "書 100"/>。國中畢業後，黃子綝先後進入五專和高職，並以成為營養師為志向<ref name = "書 140"/>，但不久後因病休學，病情也逐漸惡化。19歲時，母親因多發性神經纖維瘤逝世<ref name = "書 174"/>。而黃子綝的纖維瘤也致使她完全失去聽力，逐漸失去言語能力，四肢癱軟無力、不良於行<ref name="中時"/>，更削弱她的視力<ref name = "書 167"/>。至2014年為止，黃子綝總共進行六次手術<ref name="中央"/>，這些手術使得她的顏面神經失去功能，無法做出[[面部表情|表情]]<ref name="中時"/>。

=== 大學生涯 ===
2010年起，黃子綝進入[[南華大學_(臺灣)|南華大學]]文學系就讀<ref name = "書 189"/>，校方安排同學擔任其聽打員，將課堂內容轉換為文字，降低其學習過程所受到的阻礙，亦協助她與師生們交流。在黃子綝完全失去說話和獨立行走的能力後，仍經常以手語或[[即時通訊|即時通訊]]軟體留言鼓勵學弟妹<ref name="南華追思"/>，並開始以輪椅代步，在二年級下學期起使用電動輪椅，她自嘲自己如同[[變形金剛_(虛構生物)|變形金剛]]<ref name = "書 336"/>。即使身坐輪椅，她也並不缺席校外教學，且每週至彰化員林精進畫技<ref name="中央總統"/>。這些努力也讓她於2013年獲贈[[總統教育獎_(中華民國)|總統教育獎]]表揚<ref name="中央總統"/><ref name="總統"/>。黃子綝的精神也感動了大林慈濟醫院公共傳播室主任于劍興，讓他為黃子綝撰寫傳記《當聲音去流浪》。傳記於2014年付梓，黃子綝在同年於嘉義縣[[佛教慈濟醫療財團法人|大林慈濟醫院]]、[[佛教慈濟醫療財團法人|台中慈濟醫院]]舉辦畫展及新書發表會，同年11月至南華大學續展，並將部份所得捐作公益<ref name="中央"/><ref name="ET"/>。2014年6月，黃子綝自南華大學畢業，同時獲頒勤奮向學獎<ref name="中央畢典"/>。在畢業後選擇在校輔修傳播系課程，並準備文學所碩士班考試。

=== 逝世 ===
2015年1月4日，黃子綝於學生宿舍吃早餐時，因吞嚥不慎造成氣管堵塞而[[窒息|窒息]]。由於時逢元旦連續假期，宿舍內沒有其它學生，她的室友於下午三點返回宿舍並發現異狀，但黃子綝在送醫後仍宣告不治，得年28歲<ref name="自由訃聞"/><ref name="中時訃聞"/>。逝世後，胞弟將她生前的畫作帶回彰化老家展出<ref name="自由訃聞"/><ref name="蘋果訃聞"/>，而南華大學於同月13日為其舉辦追思會，並展出她的三十多幅畫作、詩詞和其它作品以茲悼念。<ref name="南華追思"/><ref name="中央訃聞"/><ref name="中時訃聞"/>

== 作品 ==
黃子綝使用右手作畫，在多次手術導致右手無力後改以左手作畫<ref name = "書 290/>。她不喜歡畫人像<ref name = "書 213/>，作品多為動物和風景。平均每二、三週能繪製一幅新作品<ref name="中時"/>，她透過販售畫作支應自己的學費與生活費<ref name="中央總統"/>，亦將部份所得捐出回饋社會<ref name="ET"/>。黃子綝表示自己的原點是日本漫畫《[[美少女戰士|美少女戰士]]》<ref name = "書 290/>，她也喜歡[[宮崎駿|宮崎駿]]和[[水木茂|水木茂]]的作品。<ref name = "書 317/>
