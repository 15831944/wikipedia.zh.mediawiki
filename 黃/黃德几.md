'''黃德-{几}-'''（{{lang|en|'''Dickie Wong'''}}，{{bd|1976年|2月24日<ref name=DW3>[http://invest.hket.com/article/963882/%E8%B2%A1%E9%87%91%E5%9C%88%E5%96%9C%E4%BA%8B%E5%A4%9A%20%E5%8F%88%E7%B5%90%E5%A9%9A%E5%8F%88%E7%94%9F%E4%BB%94 財金圈喜事多 又結婚又生仔]，[[香港經濟日報|香港經濟日報]]，2009年5月29日</ref><ref>[https://www.instagram.com/p/BfkXy-DFTWT/ 今年生日踏正12點，第一位同我講生日快樂嘅，就係我呢位最好嘅朋友喇······]，黃德-{几}-（wongdickie） - Instagram，2018年2月24日</ref>}}），香港股評人。

== 簡歷 ==
黃德-{几}-曾就讀[[番禺會所華仁小學|番禺會所華仁小學]]，後升讀加拿大私立學校 Mentor College，1999年畢業於[[約克大學_(加拿大)|約克大學]]經濟學系學士課程<ref name=DW>[http://orientaldaily.on.cc/cnt/finance/20090814/00271_003.html 絲絲講場：黃德几疑似「黎天王」]，[[東方日報_(香港)|東方日報]]，2009年8月14日</ref>。大學期間，他曾參加《[[全球華人新秀歌唱大賽|全球華人新秀歌唱大賽]]》，且獲得首5名<ref name=DW/>，原有機會成為歌手，卻選擇於證券行業發展<ref name=DW/><ref name=DW2>[https://www.enlightenfish.com.hk/bookstore/finance-and-investment/%E6%8A%95%E8%B3%87%E5%BE%97%E6%A9%9F-detail 投資理財  投資得機 - 亮光文化]</ref><ref>[https://archive.today/20180502083926/http://ckmy-karis.blogspot.nl/2010/05/blog-post_30.html 曾貪勝不知輸墮谷底 黃德几教訓中成長 學懂避險]，[[商報|商報]]，2010年5月30日</ref>。

黃德-{几}-現時擔任企業培訓及投資課程導師，亦為金利豐證劵研究部執行董事、證券商協會會董、香港股票分析師協會理事、中華廠商聯合會專業諮詢服務計劃顧問、中華總商會青年委員會委員、高智商人士組織 Mensa 會員及傲翔善慈基金名譽理事，定期在電視台、電台、網上媒體主持財經節目<ref name=DW2/>，並於報章雜誌撰寫專欄<ref>[http://www.metrodaily.hk/metro_columns/%E9%BB%83%E5%BE%B7%E5%87%A0%E9%9A%A8%E3%80%8C%E5%87%A0%E3%80%8D%E6%87%89%E8%AE%8A%E6%B4%9E%E6%82%89%E8%88%87%E5%85%88%E6%A9%9F/ 黃德几##隨「几」應變##洞悉與先機]，[[都市日報|都市日報]]，2015年2月24日</ref>和教授理財心得<ref>[http://hk.on.cc/hk/bkn/cnt/finance/20161002/bkn-20161002161352303-1002_00842_001.html 我要做財演：學做財演至緊要 Talk 得]，[[ON.CC|東網]]，2016年10月2日</ref>。

== 家庭生活 ==
黃德-{几}-於2009年與女朋友 Eva 結婚<ref name=DW3/><ref>[http://orientaldaily.on.cc/cnt/finance/20100622/00271_001.html 絲絲講場：黃德几12月榮升老竇]，東方日報，2010年6月22日</ref>，兩人育有一子（Arlend）一女（Aika）<ref>[https://hk.finance.appledaily.com/finance/daily/article/20150903/19280380 黃德几：1歲就學何謂錢]，[[蘋果日報_(香港)|蘋果日報]]，2015年9月3日</ref>。
