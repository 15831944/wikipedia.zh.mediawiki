'''黃宏成台灣阿成世界偉人財神總統'''{{BD|1968年|3月20日}}，[[臺灣|臺灣]][[桃園市|桃園]]人，[[作家|作家]]、行動藝術工作者。

黃氏出生名'''黃宏成'''，歷經多次改名後以現名15字居全臺姓名最長者<ref>{{Cite news|url=https://www.ettoday.net/news/20150524/510888.htm|title=「台灣阿成」又改名了！新名字共15字　簡稱「黃總統」 {{!}} ETtoday新聞雲|last=https://www.facebook.com/ETtoday|newspaper=ETtoday新聞雲|accessdate=2018-11-21|language=zh-TW|archive-url=https://web.archive.org/web/20181122051705/https://www.ettoday.net/news/20150524/510888.htm|archive-date=2018-11-22|dead-url=no}}</ref><ref>{{Cite news|url=http://news.ltn.com.tw/news/life/paper/825757|title=全台姓名最長13字 最短僅1字 - 生活 - 自由時報電子報|accessdate=2018-11-21|archive-url=https://web.archive.org/web/20181122005759/http://news.ltn.com.tw/news/life/paper/825757|archive-date=2018-11-22|dead-url=no}}</ref>，投身政治前做過直銷、法律業務，二度參加律師資格考試皆落榜，2000年代後期以三輪車騎遍全臺灣，蒐集並親吻臺灣地區各鄉鎮市區[[泥土|泥土]]拼成「寶島台灣圖」而名噪一時。2010年代轉而投身政治，參加[[嘉義縣縣長|嘉義縣縣長]]、嘉義市區域立委及[[嘉義市市長|嘉義市市長]]選舉皆未當選，但其競選風格特殊，已經引發輿論熱議。

== 早年生涯 ==
黃宏成台灣阿成世界偉人財神總統在1968年（中華民國57年）3月20日生於今[[桃園市|桃園市]]<ref name=":0">嘉義市選舉委員會 (編).[http://ebulletin.cec.gov.tw/%E9%81%B8%E8%88%89/%E7%AB%8B%E6%B3%95%E5%A7%94%E5%93%A1/105%E5%B9%B4%E7%AC%AC9%E5%B1%86/01%E5%8D%80%E5%9F%9F/20%E5%98%89%E7%BE%A9%E5%B8%82/%E5%98%89%E7%BE%A9%E5%B8%82%E7%AB%8B%E5%A7%94%E9%81%B8%E8%88%89.pdf 第９屆立法委員選舉選舉公報] {{Wayback|url=http://ebulletin.cec.gov.tw/%E9%81%B8%E8%88%89/%E7%AB%8B%E6%B3%95%E5%A7%94%E5%93%A1/105%E5%B9%B4%E7%AC%AC9%E5%B1%86/01%E5%8D%80%E5%9F%9F/20%E5%98%89%E7%BE%A9%E5%B8%82/%E5%98%89%E7%BE%A9%E5%B8%82%E7%AB%8B%E5%A7%94%E9%81%B8%E8%88%89.pdf |date=20160122051916 }}.</ref>，畢業於[[桃園市中壢區內壢國民小學|桃園縣中壢市內壢國民小學]]、[[桃園市私立復旦高級中學|桃園縣私立復旦高級中學]]、[[東吳大學_(臺灣)|東吳大學]]法律系，曾在[[美国|美国]][[維吉尼亞大學|維吉尼亞大學]]拿破崙希爾基金會進修<ref name=":0" />。企管碩士、企業管理專業人<ref name=":1">{{Cite news|title=他說愛台灣 改名黃宏成台灣阿成|author=劉愛生|date=2010年5月29日|work=聯合報}}</ref>。他也擔任過中國青年創業協會總會十七屆公關副主委、台北復旦校友會監事<ref name=":0" />。曾居[[桃園市|桃園市]][[中壢區|中壢區]]<ref name=":2">{{Cite news|title=蒐集泥土 送給下任總統 台灣阿成 三輪車2年凸全台灣|author=林倖妃|date=2006年8月2日|work=[[中國時報|中國時報]]|language=zh-tw}}</ref>。他以寫書、[[演講|演講]]、上[[廣播電台|廣播電台]]等管道宣揚「積極成功學」為業<ref name=":3">{{Cite news|title=「台灣阿成」凸完台灣 下一步環球|author=周麗蘭|date=2008年12月31日|work=中國時報|language=zh-tw}}</ref>。

黃氏自2006年起，騎著三輪車造訪臺灣地區各鄉鎮市區，除了販賣自己著作之外<ref name=":3" />，也親吻該地區泥土並蒐集起來<ref name=":2" />，2008年初作成高1公尺的「寶島台灣圖」<ref name=":3" />。其出發點為[[雲林縣私立義峰高級中學|雲林縣私立義峰高級中學]]<ref name=":4">{{Cite news|title=黃宏成回復旦 師生親吻土地|author=鄭國樑|date=2006年11月3日|work=聯合報|language=zh-tw}}</ref>。他拜訪[[桃園市|桃園市]][[龜山區|龜山區]]時發生車禍，旅途受阻，而他僅要求肇事者——一名大學生——一起親吻當地泥土而和解<ref name=":4" />。

2009年4月18日，在[[嘉義市|嘉義市]]開[[記者會|記者會]]，說計畫以37年收集全世界195國泥土製作[[世界地圖|世界地圖]]做為2045年10月24日[[聯合國|聯合國]]100周年賀禮，目前已訪[[美國|美國]]、[[日本|日本]]、[[瑞典|瑞典]]、[[新加坡|新加坡]]、[[中華人民共和國|中華人民共和國]]等國家<ref>{{Cite web |url=http://www.appledaily.com.tw/appledaily/article/headline/20090419/31559508/ |title=台灣阿成發願 37年吻遍世界 |access-date=2013-01-20 |archive-url=https://web.archive.org/web/20150203094227/http://www.appledaily.com.tw/appledaily/article/headline/20090419/31559508/ |archive-date=2015-02-03 |dead-url=no }}</ref>。2010年，他開始籌備三輛車環遊中國大陸31省100大城市之旅，希望藉此促進[[台灣海峽兩岸關係|台灣海峽兩岸關係]]。

2010年5月至[[戶政事務所|戶政事務所]]申請更名'''黃宏成台灣阿成'''，仍姓'''黃'''，名'''宏成台灣阿成'''，為其第一次改名<ref name=":1" />。同年8月，在嘉義市成立台灣阿成工作室分部，販售「台灣阿成狀元糕」，營利所得作為其親吻全世界土地之用<ref>{{Cite news|title=台灣阿成 成立分部|date=2010年8月28日}}</ref>。

2013年10月再改名'''黃宏成台灣阿成世界偉人'''，為其第二次改名<ref>{{Cite news|title=2次改名 姓名11字 黃宏成是世界偉人 身分證有寫|author=魯永明|date=2013年10月03日|work=聯合報|author2=謝恩得}}</ref>。

== 投入政治 ==
黃宏成台灣阿成世界偉人財神總統沒有背景勢力、政黨支持，歷次選舉的競選經費由多來自其著作出版，不設立競選總部，也不插旗幟<ref name=":5">{{Cite news|url=https://udn.com/news/story/7326/3024951|title=除檯面上幾位大咖 嘉義市長參選人還有他|author=王慧瑛|date=2018-03-11|work=[[聯合報|聯合報]]|accessdate=2018-12-15|deadurl=no|language=zh-tw|archiveurl=https://web.archive.org/web/20181215111134/https://udn.com/news/story/7326/3024951|archivedate=2018-12-15|quote=以財神總統自居的黃宏成……曾選過國民大會代表、嘉義縣長、嘉義市立委等，……有3本著作，堅信思想是世界上最偉大的力量，透過寫書、賣書傳遞理念，也靠出版品掙生活費、選舉經費，2016年選立委那次，同為候選人的翁壽良掏出30萬向黃宏成買書，其中20萬繳保證金，另10萬做文宣支出。幾次選舉都沒設競選總部、沒插旗幟……「我一定要成功，最終目標是選總統，主要政見是興建海峽隧道，從嘉義布袋接澎湖、廈門」黃宏成說，他沒背景、沒政黨……。}}</ref>。其競選風格在[[臺灣|臺灣]]獨樹一幟，頗令選民熱議<ref>{{Cite web|url=https://udn.com/news/story/10958/3488269|title=政見會講英文被消音 「台灣阿成」喊告|accessdate=2018-12-15|author=王慧瑛|date=2018-11-18|publisher=|language=zh-tw|newspaper=聯合報|archive-url=https://web.archive.org/web/20181213192954/https://udn.com/news/story/10958/3488269|archive-date=2018-12-13|dead-url=no}}</ref>。曾經參選第4屆[[國民大會|國民大會]]代表<ref name=":0" />，但該次選舉因修憲而取消。

=== 2014年嘉義縣縣長選舉 ===
2014年登記參選[[嘉義縣|嘉義縣]]縣長<ref>{{cite news|url=http://www.ettoday.net/news/20140903/396820.htm|title=「台灣阿成」姓名長達11字　嘉縣委員會：選票可能改字體|date=2014-09-03|accessdate=2014-11-28|archive-url=https://web.archive.org/web/20141202045211/http://www.ettoday.net/news/20140903/396820.htm|archive-date=2014-12-02|dead-url=no}}</ref>，並向[[嘉義縣選舉委員會|嘉義縣選舉委員會]]要求將他的自製[[國旗|國旗]]「偉大中華台灣神聖民主共和國旗」刊登在[[選舉公報|選舉公報]]上但被拒絕。2014年9月19日，帶著[[臺灣嘉義地方法院|臺灣嘉義地方法院]][[行政訴訟|行政訴訟]]裁定書，到[[高雄高等行政法院|高雄高等行政法院]]控告[[中華民國中央選舉委員會|中央選舉委員會]]與嘉義縣選舉委員會[[違憲|違憲]]及剝奪[[言論自由|言論自由]]<ref>{{Cite web |url=http://www.chinatimes.com/realtimenews/20140919003989-260407 |title=選舉公報不給PO圖 台灣阿成告選委會 |access-date=2014-09-19 |archive-url=https://web.archive.org/web/20141110061553/http://www.chinatimes.com/realtimenews/20140919003989-260407 |archive-date=2014-11-10 |dead-url=no }}</ref>。2014年11月29日，選舉結果揭曉，黃氏以8,639票、2.82%得票率低票落選。後續因不甘落選，向[[嘉義地方法院|嘉義地方法院]]提起選舉無效之訴，表示不讓他刊登自製國旗屬於違憲，法官以圖片不得刊登為由駁回。<ref>{{Cite web |url=http://news.ltn.com.tw/news/society/breakingnews/1258246 |title=自創國旗選舉公報未刊 提選舉無效被駁 |access-date=2016-02-03 |archive-url=https://web.archive.org/web/20160202135325/http://news.ltn.com.tw/news/society/breakingnews/1258246 |archive-date=2016-02-02 |dead-url=no }}</ref>{{2014年中華民國縣市長選舉/嘉義縣}}

=== 2016年[[嘉義市選舉區|嘉義市選舉區]]立法委員選舉 ===
2015年5月23日第三次改名，新姓名'''黃宏成台灣阿成世界偉人財神總統'''成為[[中華民國內政部|中華民國內政部]]公布的最長的名字，同時公開宣布擬參選中華民國總統<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/1327043|title=「可以簡稱黃總統」 全名15字破紀錄|author=余雪蘭|date=2015-05-24|work=[[自由時報|自由時報]]即時新聞|accessdate=2015-05-24|archive-url=https://web.archive.org/web/20150524061728/http://news.ltn.com.tw/news/life/breakingnews/1327043|archive-date=2015-05-24|dead-url=no}}</ref>。同年11月卻登記參選[[嘉義市選舉區|嘉義市選舉區]]的區域[[2016年中華民國立法委員選舉|立法委員]]。該次競選經費有部分來自同選區候選人[[翁壽良|翁壽良]]花費新台幣30萬元購買其著作以示支持。在政見發表會上裝扮為[[財神|財神]]，並清唱[[拜年|拜年]]歌謠、食用[[火雞肉飯|雞肉飯]]。最後以2000餘票落選。

{{2016年中華民國立法委員選舉結果/嘉義市選舉區}}

對於自己的三次更名，他認為可以彰顯台灣的民主法治、重視言論自由之精神，並指出「中國就不可能有這種名字出現」。受限於《姓名條例》規定，他不能再改名；但如果法令允許，他仍希望改名成「黃宏成台灣阿成世界偉人財神總統一定已經當選」<ref>{{cite news|url=https://tw.appledaily.com/new/realtime/20181209/1480746/|title=比林務局長更狂 他改名總統成台灣最長的男人|author=李宗祐|date=2018-12-09|publisher=蘋果日報|accessdate=2018-12-10|language=zh-tw}}</ref>。

2016年1月16日投票日當天找不到其[[中華民國國民身分證|中華民國國民身分證]]而無法投票，雖然最後在[[嘉義縣|嘉義縣]][[朴子市|朴子市]]自家倉庫找回身分證，但仍然來不及投票<ref>{{Cite news|url=http://udn.com/news/story/3/1445889|title=台灣阿成身分證遺失 求大眾協尋|author=林伯驊|date=2016-01-16|work=[[聯合報|聯合報]]即時新聞|accessdate=2016-01-16|archive-url=https://web.archive.org/web/20160117143118/http://udn.com/news/story/3/1445889|archive-date=2016-01-17|dead-url=no}}</ref>。
