'''黃迪揚'''（英文：Huang Di-Yang，{{bd|1986年|7月21日|catIdx=Huang黃迪揚}}） ，[[台灣|台灣]][[高雄市|高雄市]]出身，[[台北藝術大學|台北藝術大學]][[戲劇|戲劇]]系畢業，主修表演，為[[台灣|台灣]][[男演員|男演員]]<ref>{{Cite web |url = http://www.weichunggo.com/artist.php |title = 金星娛樂藝人列表 |author = 金星娛樂 }}</ref>。2004年從高雄北上讀書，學習戲劇演出後參與過許多劇本演出<ref>{{cite news |author = 104 |url = http://www.104.com.tw/jb/area/media/article/detail/id/167121573302876467/cat/64 |title = 【職人分享】黃迪揚：只給自己一條單程路選擇 |publisher = 104 |date = 2014-09-09 |deadurl = yes |archiveurl = https://web.archive.org/web/20141123064725/http://www.104.com.tw/jb/area/media/article/detail/id/167121573302876467/cat/64 |archivedate = 2014-11-23 }}</ref>。早年活躍於[[舞台劇|舞台劇]]「[[瘋狂電視台|瘋狂電視台]]」，在[[戲劇|戲劇]]「[[飯糰之家|飯糰之家]]」中扮演[[邱一傑|邱一傑]]一角而嶄露頭角<ref>{{cite news |author = 台灣電視公司 |url = http://www.ttv.com.tw/drama10/RiceFamily/behind.asp?PID=13 |title = 飯糰之家 人物介紹 |publisher = 台灣電視公司 |date = 2010-05-10 }}</ref>。2016年以[[戲劇|戲劇]]「[[植劇場-天黑請閉眼|植劇場-天黑請閉眼]]」中的警察[[陳亞正|陳亞正]]一角受到注目。

＂小時候藏考卷不讓媽媽知道是我認識表演的開始。後來書桌成了我拍戰爭片、英雄片的場域，範圍漸漸擴大到房間直到現在的舞台。＂
