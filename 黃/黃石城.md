'''黃石城'''（{{bd|1935年|8月27日|||}}），[[臺灣|臺灣]][[彰化縣|彰化縣]][[大村鄉_(台灣)|大村鄉]]人。[[大村國小|大村國小]]畢業，[[永靖高工|永靖農工]]畢業，[[東吳大學_(台灣)|東吳大學]]法律系畢業，曾任[[彰化縣|彰化縣]]第9屆、第10屆縣長、[[行政院|行政院]][[政务委员_(中华民国)|政务委员]]、[[總統府國策顧問|總統府國策顧問]]、[[中央選舉委員會|中央選舉委員會]]主委（1994年7月30日－1995年6月16日兼任，1999年12月17日－2000年5月20日兼任，2000年5月20日－2004年6月16日專任），現擔任[[財團法人台灣傳統基金會|財團法人台灣傳統基金會]]董事長<ref>[http://www.heritage.tw 財團法人台灣傳統基金會網站]</ref>、環宇國際文化教育基金會[[董事長|董事長]]、[[健保免費連線|健保免費連線]]諮議。

== 家庭 ==
黃石城之妻[[陳照娥|陳照娥]]女士曾任[[國大代表|國大代表]]、[[台灣省政府|台灣省政府]]政務委員，現擔任世界和平婦女會台灣總會理事長。

黃石城之子[[黃偉峰|黃偉峰]]專長領域是當代西洋政治理論、英美政府與政治、[[歐洲聯盟|歐洲聯盟]]政治、選舉研究及政治社會學，為[[美國|美國]][[麻省理工學院|麻省理工學院]]碩士、[[英國|英國]][[牛津大學|牛津大學]]政治學博士。曾擔任[[國立台灣大學|國立台灣大學]][[國家發展研究所|國家發展研究所]]兼任副教授、[[行政院大陸委員會|行政院大陸委員會]]副主委、[[駐美國台北經濟文化代表處|駐美國台北經濟文化代表處]]副代表。現任[[中央研究院|中央研究院]]歐美研究所副研究員。

黃石城於2004年力挺女兒[[黃文玲|黃文玲]]參選彰化縣[[立法委員|立法委員]]，黃文玲最後以14,896票落選。2011年10月11日，黃文玲獲[[台灣團結聯盟|台灣團結聯盟]]提名為[[2012年中華民國立法委員選舉|2012年中華民國立法委員選舉]][[2012年中華民國立法委員選舉#全國不分區及僑居國外國民選舉|全國不分區]]立委排名第二位並當選。

== 批判 ==
2007年10月23日，黃石城接受《[[中國時報|中國時報]]》專訪時說，人民選出[[執政黨|執政黨]]是要來替人民解決問題、不是來解釋問題，「國家治理不好，拿稅金的人當然要負責！」他說，台灣現在面臨價值混亂虛無的危機，國家搞得一團糟，執政黨[[民主進步黨|民主進步黨]]卻老是將責任推給[[在野黨|在野黨]]阻撓、不配合，「但稅金是你收的，錢是你用的，資源是你在分配，責任當然要你負，問題當然要你解決，這道理天公地道」；而[[中國國民黨|中國國民黨]]的改革能量似乎在[[辛亥革命|辛亥革命]]後就消耗殆盡，現在已經完全沒有改革的戰鬥力，對於民主進步黨的無能完全束手無策、完全崩潰，讓人也難有期待。他說，現在台灣的[[民主選舉|民主選舉]]根本是不斷的[[黑箱作業|黑箱作業]]、是幌子，說穿了都是騙人的，不配叫做「深化民主」；台灣人民不斷被[[泛藍|泛藍]][[泛綠|泛綠]]綁架，以前藍營執政搞[[專制|專制]]，現在綠營[[陳水扁政府|陳水扁政府]]執政同樣搞專制，被綁架的永遠是台灣人民。他說，現在檯面上的所謂[[第三勢力|第三勢力]]政治人物，包括：曾任[[行政院院長|行政院院長]]的前[[中華民國空軍|中華民國空軍]][[一級上將|一級上將]][[唐飛|唐飛]]，或[[第三社會黨|第三社會黨]]、[[台灣農民黨|台灣農民黨]]，不是曾在綠營失勢、就是在藍營失勢，沒有一點說服力，他不認為能成什麼氣候；但是台灣真正的中道力量還是應該出面制衡藍綠兩黨在政壇形成的[[霸權|霸權]]，才是對台灣人民有利。他同時批評，[[行政院新聞局|行政院新聞局]]局長[[謝志偉_(台灣)|謝志偉]]沒有身為廣義[[公務員|公務員]]的自覺，「每天像在作秀表演，沒有一點樣子」，「這種人當官，跟請[[動物園|動物園]]的[[猴子|猴子]]來表演當官有什麼不一樣？實在很悲哀。」<ref>[[何博文|何博文]]、蕭旭岑.黃：以選舉搞黑箱 台灣民主騙人. [[中國時報|中國時報]].2007-10-24</ref>

2013年[[九月政爭|九月政爭]]爆發後，78歲的黃石城加入「929罵英九怒火聯盟」，呼籲929罵英九怒火聯盟用公民力量換掉[[馬英九政府|馬英九政府]]這個與民為敵的「五流政府」。他表示，他一生未加入黨派、未上街遊行，但實在看不下去馬政府不顧[[失業率|失業率]]攀升卻硬推油電雙漲，服貿及[[核四|核四]]等政策處處與人民作對、不把人民看在眼裡，政府沒照顧好人民，人民才會主動嗆聲。黃石城表示，[[國民黨全代會|國民黨全代會]]延期，代表政府害怕人民的力量。他痛批，國民黨因公民要包圍，就以全代會延期逃避，是糟蹋、耍弄人民、製造更大對立，非常不負責任，顯示他們還停留在威權時代的心態，若再讓馬執政，台灣會完蛋。黃反諷馬政府：「幾十萬人民都到面前，要告訴你那裡做不好，是聽人民聲音的大好機會，才知道他們的苦痛。卻迴避閃躲，還說要下鄉long stay，這不是作秀？還是去吃香喝辣？」他強調，民主政治應是國家、人民的利益高於政黨利益，如今卻是政黨利益凌駕於人民、國家利益之上，失去民主政治意義、搞「幫派」治國，呼籲全民拒絕分贓利益、「吃天吃地吃百姓」的國民黨。他認為，台灣當前問題只能靠公民運動解決，「人民才是皇帝，政府不要搞錯對象」。<ref>王品昇.[http://www.ettoday.net/news/20130924/273971.htm 黑衫軍嗆馬大遊行！黃石城：要讓政府知道人民才是皇帝].[[ETtoday|ETtoday]].2013-09-24</ref><ref>謝文華.[http://www.libertytimes.com.tw/2013/new/sep/25/today-fo4.htm 929照常嗆馬 黑衫軍凱道燃怒火].[[自由時報|自由時報]].2013-09-25 {{Wayback|url=http://www.libertytimes.com.tw/2013/new/sep/25/today-fo4.htm |date=20131009000920 }}</ref>

2014年12月27日，黃石城與媒體餐敘時說，總統選舉與立法委員選舉合併舉行，不但不公平，而且容易產生弊端：總統於5月就職，原先總統選舉是在3月，如果為了與立法委員選舉合併而提早到1月，空窗期太長，「中間會有變化，這是[[風險|風險]]問題」；另一個問題是，總統選舉的選民範圍大、[[買票|買票]]不知從何買起，但立法委員選舉會有[[樁腳|樁腳]]，易生弊端；合併選舉讓選舉品質變差，如果選到不好的人，代價是幾千、幾萬倍；支持合併選舉者說合併選舉可節省選舉經費，但他主張「真的要省就是不要辦，選舉不是經費的問題」<ref>{{cite news|author=呂欣憓|url=http://www.cna.com.tw/news/aipl/201412270267-1.aspx|title=黃石城：總統立委選舉應分開|work=|publisher=[[中央通訊社|中央通訊社]]|pages=|date=2014-12-27|accessdate=2014-12-27}}</ref>。

2015年9月2日，針對前總統[[李登輝|李登輝]]在日本[[PHP研究所|PHP研究所]]《Voice》月刊2015年9月號所說台灣與日本在[[第二次世界大戰|第二次世界大戰]]時期「同屬一國」的言論，黃石城說，當時李登輝是[[台籍日本兵|台籍日本兵]]，本來就不是為[[中華民族|中華民族]][[抗日|抗日]]；以這個[[邏輯|邏輯]]推論，李登輝沒參加抗日，因此不清楚當時的情形，發表上列言論時也沒有弄清楚當時的情況；李登輝不應對此「一概而論」，台灣社會更不要因此被誤導而忽略真正的史實。黃石城說，[[八卦山之役|八卦山之役]]是[[乙未戰爭|乙未戰爭]]中最大的正面會戰，造成臺灣[[義軍_(台灣客家人)|客家民軍]]領袖[[吳湯興|吳湯興]]和[[浙江|浙江]]人[[黑旗軍|黑旗軍]][[吳彭年_(清朝)|吳彭年]]等人[[陣亡|陣亡]]，他希望後人正視歷史，因此他在1983年完成[[八卦山抗日烈士紀念碑公園|八卦山抗日烈士紀念碑公園]]，碑文詳細記載這段血淚史。針對[[臺灣海峽兩岸關係|臺灣海峽兩岸關係]]，黃石城表示，[[抗日戰爭|抗日戰爭]]是兩岸共同經歷之事，本來就應不分你我、不分誰抗日，一起慶祝這偉大的時刻，「怎麼可以輕易[[去中國化|去中國化]]」；台灣社會太過民主，導致彼此缺乏包容性，「有智慧才能解決[[兩岸問題|兩岸問題]]」<ref>{{Cite web|author=張郁琦|url=http://www.chinatimes.com/newspapers/20150903001085-260309|title=黃石城 籲後輩勿忘抗日歷史|publisher=[[旺報|旺報]]|date=2015-09-03|accessdate=2015-09-04}}</ref>。

== 對中共的批判 ==
2013年6月，時任臺灣傳統基金會董事長的黃石城在[[新唐人电视台|新唐人电视台]]「新唐人穿越时空互动大展」表示，倫理道德是做人的基本，沒有這些就像動物園的動物一樣，所以不管怎麽富有、官位多大都沒有用。他舉例，[[中國共產黨|中國共產黨]]在[[文化大革命|文化大革命]]中完全摧毀[[中華文化|中華文化]]，中共對[[法轮功|法轮功]]的迫害最主要是因“沒有中華文化”而做出“完全沒有符合人性”的行為，“如果一個國家權力大的官位都落在沒有道德的人身上，這個國家一定會腐化、墮落、沈淪，最後會完蛋”<ref>{{Cite web|author=鍾元、徐翠玲|url=http://www.epochtimes.com/b5/13/6/13/n3892671.htm|title=新唐人《穿越時空》展 正統文化與高科技互動|publisher=[[大紀元時報|大紀元時報]]|date=2013-06-13|accessdate=2015-09-04}}</ref>。
