'''黃國強'''，[[香港|香港]]人，資深廣告、電視及電影導演。

== 生平 ==
[[香港浸會大學|香港浸會大學]]社會學系畢業, [[香港中文大學|香港中文大學]]哲學文學碩士, 於1989年加入無綫，任職節目部高級編導期間，參與多部熱門劇集製作，包括：《[[搜神傳|搜神傳]]》（2007）、《[[巾幗梟雄之義海豪情|巾幗梟雄之義海豪情]]》（2010）、《[[天與地_(無綫電視劇)|天與地]]》（2011）、《[[法證先鋒III|法證先鋒III]]》（2011）等。2012年過檔港視，擔任港視開台之作《[[選戰|選戰]]》總導演，劇集引起全城哄動，收視每日平均高達56萬人。2016年，與游達志、黃國輝合作執導電影《鬼網》，於2017年10月香港上映。剛完成由FOX投資的華語迷你劇《香港華爾街》。
