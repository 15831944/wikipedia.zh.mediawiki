'''黃典權'''（{{bd|1927年|10月16日|1992年|7月18日}}），字'''衡五'''，歷史學者，是臺灣臺南市文獻委員會創會委員之一{{r|生平}}{{r|吳}}{{r|丁}}。曾擔任臺南市文廟管理委員、臺南縣文獻委員、臺灣史蹟源流會指導委員、內政部古蹟評定委員、文化建設委員會古蹟管理維護及考評工作小組顧問等職{{r|丁}}。黃典權主持《臺南文化》編務工作近20年，也帶領學生從事史蹟踏查，對於臺南市文獻史料的建立與保存有卓越貢獻{{r|吳}}{{r|劉}}。他一生的讀書札記卡片，以「四角號碼」統記歸分，其中光是[[明鄭時代|明鄭時代]]人物相關者，數量便將近有一萬四千多片{{r|吳}}。黃典權所發現之文物，會依其價值來向相關機構諮請收藏，如鄉土文物會送給臺南市歷史館、國史相關文物送給國立歷史博物館，又有一些會送給成大歷史系文物館{{r|丁}}。

黃典權寫文章有諸多筆名，有'''衡五'''、'''惠兒'''、'''陳惠兒'''、'''南史'''、'''啼魂'''、'''啼痕'''、'''憨園'''、'''赤嵌樓主'''等等{{r|丁}}。此外在主編《臺南文化》期間，上頭以「本會」名義發表的文章，大多出自黃典權之手{{r|丁}}。其文章除刊載於《臺南文化》，也刊載於《臺灣風物》、《南瀛文獻》、《臺北文獻》、《臺灣省文獻》、《高雄市文獻》、《臺北文物》、《孔孟月刊》、《大義雜誌》、《暢流》、《藝文誌》、《[[中央日報|中央日報]]》、《[[中華日報|中華日報]]》等期刊報章上{{r|丁}}。此外他也參與過《臺南市志稿》、《臺南市志》、《屏東縣志》等志書的編纂{{r|丁}}。

== 生平 ==
黃典權於民國16年（1927年）10月6日出生於中華民國福建省龍溪縣{{NoteTag|當時黃典權祖宅的位置是龍溪縣南薰鎮布觀音35號，接近[[南山寺_(漳州)|南山寺]]（今漳州市薌城區）{{r|丁}}。}}，民國20年（1931年）進入省立龍溪中學附屬小學就讀{{r|生平}}。民國26年（1937年）考入龍溪中學初中部，因[[中國抗日戰爭|中日戰爭]]，隨學校迫遷到閩西長汀縣{{r|生平}}{{r|丁}}。民國29年（1940年），考取省立長汀中學高中部{{r|生平}}。民國32年（1943年），考入此時遷到[[長汀|長汀]]的國立[[廈門大學|廈門大學]]歷史系{{r|生平}}。就學期間深受谷霽光、葉國慶、歐陽琛等教授影響{{r|丁}}。民國36年（1947年）自廈門大學歷史系畢業{{r|生平}}，當時的畢業論文是《明末福建漳泉之海盜》{{r|吳}}。大學畢業後，應龍溪中學校長謝新周之邀，前往教授高一國文與高三兩班外國史{{r|生平}}。

二次大戰後，民國38年（1949年）8月黃典權為尋找三兄黃典錡而到臺灣臺南{{NoteTag|黃典錡因其父要求乘船前往臺南購物，結果後來船東把輪船賣掉逃逸而無法離臺{{r|丁}}。}}，不久兩人便重逢{{r|生平}}{{r|丁}}。隔年（1950年）於臺南女子商業學校（今[[國立家齊高中|國立家齊高中]]前身）任教，不久兼任教務主任{{r|生平}}。當時的[[臺南市市長|臺南市長]]卓高煊設立「臺灣史料編輯委員會」，黃典權獲聘為委員{{r|生平}}。民國40年（1951年），「臺灣史料編輯委員會」改為「臺南市文獻委員會」{{NoteTag|該年4月正式成立{{r|丁}}。}}，黃典權為創立時的委員之一{{NoteTag|創立時的主任委員是市長葉廷珪，副主委是教育科長謝新周，委員除黃典權外，還有林勇、林朝均、許子文、[[石暘睢|石暘睢]]、林咏榮、[[許丙丁|許丙丁]]、高祥瑞、高崇煦、莊松林、江家錦、連景初、賴建銘、[[黃天橫|黃天橫]]{{r|生平}}。總務組長是謝碧連，整理組長是[[韓石麟|韓石麟]]，編纂組長是林咏榮，總幹事是丘同德{{r|生平}}。}}{{r|生平}}。該年（1951年）10月，期刊《臺南文化》{{NoteTag|臺南縣市合併後，與臺南縣的期刊《南瀛文獻》合併成新期刊《臺南文獻》。}}發行，黃典權於創刊號上撰寫〈寧靖王〉一文{{r|生平}}。民國42年（1953年）6月1日，黃典權受聘擔任臺南市文獻委員專任編纂組長，負責《臺南文化》編務，同時主修《臺南市志稿》{{r|生平}}。他與石暘睢、莊松林、林咏榮、連景初被推選為「志目研擬小組」成員，並由黃典權負責起草{{r|生平}}。由於當時文獻會位在[[赤嵌樓|赤嵌樓]]文昌閣內，黃典權以「赤嵌樓主」作為筆名{{r|生平}}。

民國44年（1955年）7月24日，黃典權與洪玲花結婚{{r|生平}}。民國45、46年（1956、1957年），國立歷史博物館館長包遵彭聘黃典權為研究委員{{r|丁}}。民國53年（1964年），臺南進士[[施瓊芳|施瓊芳]]、[[施士洁|施士-{洁}-]]後人因經濟問題找上黃典權，希望能收購先人的手稿{{r|劉}}。黃典權原先打算婉拒，後來為保存文稿而向[[許丙丁|許丙丁]]借錢收購{{r|劉}}。民國54年（1965年）11月，受聘在臺南私立崑山工業專科學校（今[[崑山科技大學|崑山科技大學]]）擔任講師，兼校長室秘書{{r|生平}}。之後升任副教授，仍兼任秘書，同時擔任臺南市文獻委員與編纂組長{{r|生平}}。此外該年（1965年）臺灣銀行經濟研究室出版黃典權整理施瓊芳、施士-{洁}-作品而成的《後蘇龕合集》{{r|劉}}。民國59年（1970年），受聘為教育部文化局「祭孔工作小組」顧問{{r|丁}}。民國60年（1971年）春，黃典權赴[[國立成功大學|國立成功大學]]歷史系演講，當時系主任吳緝華有意聘請黃典權，但當時並沒有師資員額的缺而未成{{r|生平}}。而該年（1971年）夏，日本漢學家{{link-ja|森克己|森克己}}、{{link-ja|酒井忠夫|酒井忠夫}}率團來臺，在到臺南時曾抽空前往黃典權家拜訪{{r|丁}}。到了民國62年（1973年），成大歷史系系主任陳捷先請黃典權兼課，教授「臺灣省志」一科{{r|生平}}。黃典權在成大歷史系推動「史蹟勘考小組」，並主籌編印刊物《史蹟勘考》{{r|生平}}。成大歷史系在黃典權協助下，與[[國立歷史博物館|國立歷史博物館]]、省立臺南社會教育館（今[[國立臺南生活美學館|國立臺南生活美學館]]）、[[臺南市政府|臺南市政府]]締結建教合作{{r|生平}}。此外成大歷史系在陳捷先與黃典權推展下，獲得哈佛燕京學社贊助{{r|生平}}{{r|丁}}。

民國64年（1975年）春，黃典權與臺南社教館（當時設在今[[臺南公會堂|臺南公會堂]]）館長書道弘商訂[[南門碑林|南門碑林]]拓製計劃，選何培夫、李永麟、黃啟迪、顏文魁、吳書樂、楊師宏等學生參與，歷時兩個月{{r|丁}}。而在該年（1975年）6月5日，在哈佛燕京社贊助下黃典權率成大歷史系學生前往澎湖拓碑，6月7日返臺{{r|丁}}。同年8月，成大歷史系系主任陳捷先聘請黃典權成為專任副教授，教授「明史」課程，同時還有外系數班的「中國通史」{{r|生平}}。而先前的「臺灣省志」課程，也繼續教授{{r|生平}}。民國65年（1976年）11月11日，成大歷史系「文物館」成立，展覽的藏品中便有黃典權帶學生至各處{{NoteTag|範圍據說北到彰化，南到高屏{{r|生平}}。}}實地查勘的成果{{r|生平}}。該年（1976年）12月23日，之前南門碑林的拓圖在社教館展出五天，之後成果臺臺南市鄭五在民國68年（1979年）10月出版{{r|丁}}。民國68年（1979年），黃典權參與《臺南市志》的纂修{{r|丁}}。

民國71年（1982年）3月，黃典權遭遇重大車禍，肝腎內臟已有受損{{r|生平}}。民國73年（1984年）9月，英國劍橋大學[[李約瑟|李約瑟]]和其助理[[魯桂珍|魯桂珍]]應[[陳立夫|陳立夫]]之邀來臺，並在28日來到臺南入住赤嵌大飯店{{r|丁}}。當時臺南市長蘇南成請黃典權陪同李約瑟等人參訪，同時並請他在薦舉一人陪同{{r|丁}}。而後黃典權便找上同事丁煌，原先丁煌以恐難以勝任為由拒絕，但黃典權表示李約瑟此行到臺南打算訪問道士陳榮盛，而丁煌研究道教多年，若不肯同行恐怕沒有可對話答詢者，於是丁煌遂同意同行{{r|丁}}。李約瑟與魯桂珍在臺南除拜訪陳榮盛宅外，並前往[[延平郡王祠|延平郡王祠]]與[[赤嵌樓|赤嵌樓]]參觀{{r|丁}}。

民國78年（1989年）4月5日，黃典權前往香港出席[[香港大學|香港大學]]亞洲研究中心「亞太地區地方歷史國際會議」{{r|生平}}。而其兄長黃典誠當時受[[香港中文大學|香港中文大學]]之邀，從廈門來參與港大中文系召開的「[[章太炎|章太炎]]、[[黃侃|黃侃]]學術研討會」，兩人因而在分開多年之後重逢{{r|生平}}{{r|丁}}。民國79年（1990年）11月，在成大歷史系梁華璜（系主任）、黃典權、丁煌，高雄師大王賢德、張守真及日本酒井忠夫等諸位教授多次聯繫商議下，訂下「臺灣南部寺廟調查暨研究計劃書」，有臺日多位學者參與{{r|丁}}。

民國80年（1991年），黃典權健康狀況欠佳，入住省立臺南醫院（今[[衛生福利部臺南醫院|衛生福利部臺南醫院]]），但仍從事教職{{r|生平}}。民國81年（1992年）7月16日晚上，黃典權在住家咳血，妻子與長男隨即僱車將他送往成大醫院急診，後於7月18日凌晨12時5分因肝硬化引起的併發症去世{{r|生平}}{{r|丁}}。

== 身後 ==
成大歷史系教授丁煌在黃典權去世整理其遺著目錄，初稿刊於1992年12月15日發行的《中國歷史學會會訊》第44期{{r|丁}}。而後寫有〈黃典權教授與臺灣史研究〉、〈文獻委員黃典權教授傳略及其治學業績〉等文記述其生平與治學貢獻{{r|丁}}。

2014年，在成大歷史系教授蕭瓊瑞教授協助，並經黃典權妻子洪玲花同意，將黃典權存放在臺南土城[[正統鹿耳門聖母廟|正統鹿耳門聖母廟]]的研究資源捐給臺南市政府文化局{{r|吳}}。同年年底，曾與黃典權共事的成大歷史系教授丁煌，以及黃典權學生何培夫教授參與黃典權教授紀念專輯的編寫{{r|吳}}。

2017年4月，黃典權家屬將黃典權收藏的《喆園吟草》、《後蘇龕草》、《文魁吳尚霑筆跡詩集》文稿捐給[[國立臺灣文學館|國立臺灣文學館]]{{r|劉}}。

== 爭議 ==
* 臺南[[延平郡王祠|延平郡王祠]]前日式[[鳥居|鳥居]]改建的石造牌坊上，原為前國防部長[[白崇禧|白崇禧]]題字的對聯「孤臣秉孤忠，浩氣磅礡留萬古，正人扶正義，莫教成敗論英雄」。然而在1964年重建時，遭黃典權更改為「孤臣秉孤忠五馬奔江留取汗青垂宇宙，正人扶正義七鯤拓土莫將成敗論英雄」。<ref>{{Cite web|url=https://nchdb.boch.gov.tw/assets/overview/historicalBuilding/20100817000001|title=延平郡王祠 -|accessdate=|author=文化部文化資產局--國家文化資產網|date=|publisher=}}</ref>造成外界誤會為白崇禧對聯原文為如此，現今臺南市文獻委員會已於2006年在牌坊旁立碑說明此事。

== 子女 ==
*黃聯乙（長子）：1956年出生，[[國立中興大學|國立中興大學]]地政學系畢業，從商{{r|生平}}。
*黃平蘭（長女）：1958年出生，崑山工專（今[[崑山科技大學|崑山科技大學]]）化工科畢業{{r|生平}}。與邱江泉結婚，育有二子（大瑋、柏豪）一女（佩瑩）{{r|生平}}。
*黃平芸（次女）：1960年出生，國立藝術專科學校（今[[國立臺灣藝術大學|國立臺灣藝術大學]]）美工科畢業，後與童清峰{{NoteTag|國立藝專廣播電視科畢業，婚後到美國密蘇里州攻讀新聞碩士，取得學位後返臺{{r|生平}}。後任《[[天下雜誌|天下雜誌]]》編輯{{r|生平}}。}}結婚{{r|生平}}。
*黃平蒂（三女）：1964年出生，[[逢甲大學|逢甲大學]]畢業，後與蕭天露結婚，有一女（冠瑜）{{r|生平}}。
