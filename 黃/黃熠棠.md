'''黃熠棠'''（遊戲代號：'''Maple'''、'''楓棠歲月'''，{{bd|1997年|10月10日}}）是一名台灣《[[英雄聯盟|英雄聯盟]]》職業選手，目前於[[閃電狼|閃電狼]]職業戰隊擔任中路選手。<ref>{{Cite web|url = http://www.ironforum.com.tw/subjdetail/12/2|title = 《名人對談》AsSen x Maple：「天才小釣手」楓棠歲月！|date = 2014年2月7日|publisher = 鉄人論壇|access-date = 2017年12月14日|archive-url = https://web.archive.org/web/20171215000110/http://www.ironforum.com.tw/subjdetail/12/2|archive-date = 2017年12月15日|dead-url = yes}}</ref><ref>{{Cite web |url = http://www.flashwolves.com/TeamsMemberIntro.aspx?player=13|title = 閃電狼官方網站選手介紹 }}</ref>
剛加入時因為未滿17歲，而不能參加比賽。<ref>{{Cite web |url = http://www.chinatimes.com/realtimenews/20140206003465-260412|title = 從小接觸電玩 電競新秀出頭天|date = 2014年2月6日 |publisher = 中時電子報 }}</ref>
