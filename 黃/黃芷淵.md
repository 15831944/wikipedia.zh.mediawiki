'''黃芷淵'''（英語：Wong Tsz Yuen，19??年4月20日-），全國港澳研究會會員、[[鳳凰衛視|鳳凰衛視]]高級[[記者|記者]]、主持人、香港問題特約評論員、[[專欄作家|專欄作家]]、粵港澳大灣區青年協會媒體顧問。畢業於[[香港中文大學|香港中文大學]]藝術系，獲[[香港浸會大學|香港浸會大學]]媒體管理碩士學位。<ref name="intro" >{{cite web |url= http://phtv.ifeng.com/album/interview/detail_2011_11/12/10608979_1.shtml |title=鳳凰衛視黃芷淵簡介 |accessdate=2011 |publisher=鳳凰網 }}</ref> 曾赴美國專修藝術、傳媒。2006年獲選為「香港十大傑出學生」。<ref name="best" >{{cite web |url= http://www.yukz.com/hkstar/06/060807ap_student.html |title=2006年全港「十大傑出學生」選舉 |accessdate=2006 }}</ref> 
2010年參選[[香港新聞台|香港新聞台]]「未來主播全球招兵戰」獲得冠軍，加入媒體行業，任職[[新聞主播|新聞主播]]及記者。<ref name="Mad" >{{cite web |url= https://www.youtube.com/watch?v=xj8zWZh6Lf8 |title=《未來主播》優勝者黃芷淵 |accessdate=2010 |publisher=HKBNnews }}</ref> 
2011年加入鳳凰衛視，負責粵語及普通話雙語新聞報導，曾多次前往世界多國，擔任重大新聞事件及大型國際會議的特派記者。<ref name="intro" />

曾出版《我們在現場---從香港出發》，並與同[[張愛玲|張愛玲]]並稱為「南玲北梅」的當代著名作家梅娘共同出版《與青春同行》《[https://baike.baidu.com/item/邂逅相遇：梅娘、芷渊、茵渊书礼/5959156 邂逅相遇]》《大作家與小畫家》等作品。<ref name="intro" />

== 經歷 ==
黃芷淵出生於香港，讀了十三年女校<ref name="girl" >{{cite web |url= http://www.whb.cn/zhuzhan/bihui/20180107/115251.html |title=在香港，我讀了十三年女校 }}</ref> ，畢業於[[聖士提反女子中學|聖士提反女子中學]]，在校為學生長及學生會核心成員，曾擔任校園電視主席、校報校刊委員會編委、美術學會副主席等。<br>

2006年，黃芷淵獲選為「香港十大傑出學生」<ref name="best" />，入讀香港中文大學，主修藝術系、副修政治與行政學系。曾赴美國專修藝術、傳媒。後獲香港浸會大學媒體管理碩士學位。<ref name="Jenny" /><br>

=== 職業生涯 ===
2010年參選香港新聞台「未來主播全球招兵戰」獲得冠軍，任職新聞主播及記者，曾擔任主播、天氣主播、節目主持人、新聞記者、專題記者等，工作內容集外聯、採編、播報等於一體。<ref name="Mad" /><br>

2011年轉職鳳凰衛視，負責粵語及普通話雙語新聞報導。曾多次赴[[日本|日本]]、[[韓國|韓國]]、[[新加坡|新加坡]]、[[馬來西亞|馬來西亞]]、[[泰國|泰國]]、[[俄羅斯|俄羅斯]]、[[哈薩克斯坦|哈薩克斯坦]]、[[埃及|埃及]]等採訪報導，擔任重大新聞事件及大型國際會議的特派記者，並擔任多間媒體的專欄作家及香港問題特約評論員。<ref name="Jenny" >{{cite web |url= http://phtv.ifeng.com/a/20160912/44450585_0.shtml |title=鳳凰衛視黃芷淵英文簡介 |accessdate=2016 |publisher=鳳凰衛視 }}</ref><br>

黃芷淵曾參與的重大新聞採訪報導包括：[[香港行政長官選舉|香港行政長官選舉]]、[[香港回歸|香港回歸]]二十週年系列報導、[[上海世博會|上海世博會]]、歷史性[[习马会|習馬會]]、[[兩會|全國兩會]]、[[中共十九大|中共十九大]]會議、中國國家領導人外事訪問、[[俄羅斯總統|俄羅斯總統大選]]、泰國政變、緬甸政權交接、[[東協峰会|東盟國家領導人會議]]、金磚國家領導人會議、G20國家領導人會議、上合組織領導人峰會、[[博鰲亞洲論壇|博鰲亞洲論壇]]等。<ref name="Jenny" /><br>

曾多次擔任大型活動主持及司儀，包括：香港學界國慶文藝晚會、香港政商界國慶文藝晚會、《尋根追夢，同心同行》青年論壇、十九大後香港新定位論壇、香港書展名作家系列講座、香港書展兒童慈善閱讀活動等。<ref name="Jenny" /><br>

曾專訪的著名人物包括：泰國前總理[[英拉|英拉]]、泰國前副總理[[素貼·特素班|素貼·特素班]]、中國外交部部長[[王毅|王毅]]、中國外交部前部長[[李肇星|李肇星]]、港澳辦前常務副主任[[陳佐洱|陳佐洱]]、[[關島|關島]]總督[[埃迪·卡尔沃|埃迪·卡爾沃]]等。<ref name="Jenny" /><br>

* 2016年出版'''《我們在現場---從香港出發》'''。這本書在黃芷淵加入鳳凰衛視五年之時出版，記錄了她在新聞現場採訪的點點滴滴，還有在第一線與她並肩作戰的攝影師們不為人知的內心自白，以及他們在鏡頭背後的故事。<ref>[http://paper.wenweipo.com/2017/09/04/BK1709040002.htm 【書評】鏡頭背後，我們在現場]. 韓晞汶. [2017].</ref>
