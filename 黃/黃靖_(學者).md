'''黄靖'''（{{bd|1956年|||}}）祖籍[[山东省|山东省]][[掖县|掖县]]（今[[莱州市|莱州市]]），[[美国|美国]]籍华人，美国政治学家。

==生平==
1956年，黄靖出生于一个中国军人家庭，父母作为军医曾参加[[抗美援朝战争|抗美援朝战争]]和[[援越抗美|援越抗美]]。[[1972年尼克松访华|1972年尼克松访华]]，使正在[[云南省|云南省]]偏僻山区[[上山下乡|上山下乡]]的[[知青|知青]]黄靖对美国产生了浓厚兴趣<ref name=jueqi>{{Cite web |url =http://www.ce.cn/xwzx/gjss/gdxw/200705/15/t20070515_11352189.shtml  |title = 中国崛起难免触及美国利益 学者:都要学会包容 |author =  |date = 2007-05-15 |publisher = 中国经济网 |language =  |accessdate =  |quote =  }}</ref>。后来黄靖进入[[四川大学|四川大学]]外文系英语专业学习，成为英语专业78级毕业生。此后他获得[[复旦大学|复旦大学]]历史学硕士学位。1986年进入[[美国|美国]][[哈佛大学|哈佛大学]]学习7年，后获得哈佛大学政治学博士学位。在哈佛大学7年的学习期间，他曾卖书、卖肉制品、当保安、当神经病院经理、从事社会调查<ref name=shui>{{Cite web |url = http://www.zaobao.com/realtime/singapore/story20170804-784581 |title = 谁是黄靖？ |author =  |date =2017-08-04  |publisher =  联合早报网|language =  |accessdate =  |quote =  }}</ref><ref name=jueqi/>。1993年至1994年，任哈佛大学讲师。1994年至2002年，任[[犹他州立大学|犹他州立大学]]政治学教授和亚洲研究中心主任。1998年在美国获终身教授职位。2002年至2003年，任[[斯坦福大学|斯坦福大学]]雪伦斯顿研究员。2004年至2008年，在美国智库[[布鲁金斯学会|布鲁金斯学会]]（Brookings Institution）任资深研究员<ref name=shui/>。他是布鲁金斯学会第一位华裔高级研究员，主要从事国际安全研究<ref name=jueqi/>。

2008年左右，黄靖来到[[新加坡|新加坡]]，此后担任[[新加坡国立大学|新加坡国立大学]]李光耀公共政策学院特聘讲座教授、李氏基金会讲座教授，及亚洲与全球化研究所所长等职务。他在新加坡国立大学李光耀公共政策学院发起主持中印二轨安全问题对话、中印媒体高层对话、中-日-美三方二轨对话、亚洲环境政策对话、中-印-日三国能源政策对话、[[亚洲|亚洲]]-[[欧盟|欧盟]]能源安全对话，还开发了[[西伯利亚|西伯利亚]]/[[远东|远东]]中的多边国际合作等项目，并任中日青年领袖论坛导师<ref name=shui/>。

2017年8月4日，[[新加坡内政部|新加坡内政部]]发表声明说，[[新加坡移民与关卡局|新加坡移民与关卡局]]8月4日已根据移民相关法令，取消了黄靖和妻子杨秀萍（两人均为美国公民）的出入境证件，在确定黄靖曾从事有损新加坡国家利益的活动后，根据移民法令相关条款将黄靖列为不受欢迎的移民，禁止黄靖入境新加坡。内政部的声明还称，黄靖在知情的情况下同他国情报机构及情报人员接触，与他们合作影响[[新加坡政府|新加坡政府]]的外交政策及新加坡本地舆论。声明并未说明黄靖是为哪个国家工作。同日，新加坡国立大学回应称，黄靖在新加坡国立大学的教职即日起吊销并停薪，校方将同新加坡内政部合作处理此事<ref>{{Cite web |url = http://www.guancha.cn/Neighbors/2017_08_04_421473.shtml |title = 旅居新加坡美籍学者黄靖被指为“外国代理人” 遭永久禁止入境 |author =  |date = 2017-08-04 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>。2017年8月4日，[[香港|香港]]《[[南华早报|南华早报]]》报道称，目前仍在新加坡的黄靖当天表示，新加坡政府已就相关文告通知其本人，但他否认新加坡政府的所有指控，黄靖说：“说我是试图影响本地舆论的外国代理人，这是胡说，为什么不直接指出是哪个国家……我的家和家人都在新加坡，如果他们有证据，应该把我送上法庭。”黄靖表示将向律师及美国驻新加坡大使馆寻求帮助<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1752840 |title =学者黄靖被指“外国代理人”遭永久逐出新加坡，否认所有指控  |author =  |date =2017-08-05  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
