'''黃拓榮'''（1910年－？），[[臺東縣|臺東縣]][[客家人|客家人]]，祖籍[[屏東縣|屏東縣]]。台灣教師、政治人物。[[中國國民黨|中國國民黨]]籍，曾擔任台東縣議會議長、第三、四屆[[縣長|縣長]]。在[[台灣地方派系|地方派系]]屬「黃派」，被視為「黃派」創始人。

== 早年生平 ==
黃拓榮祖籍[[屏東縣|屏東縣]]，與[[吳金玉|吳金玉]]有姻親關係，吳金玉為黃拓榮姑丈。<ref>[http://pr.ntnu.edu.tw/newspaper/index.php?mode=data&id=4553 客家身影 饒慶鈴 博士女議長第一人]，臺灣師範大學秘書處公共事務中心。</ref>黃拓榮生於[[明治|明治]]四十三年（1910年）。早年就讀[[臺東公學校|臺東公學校]]高等科，後升入[[臺南師範學校|臺南師範學校]]講習科，畢業後執教。[[昭和|昭和]]八年（1933年），帶職保送[[臺北帝國大學|臺北帝國大學]]（今[[國立臺灣大學|國立臺灣大學]]）農學院農業教員講習班攻讀農藝。結業後返鄉從事農業教育。{{r|簡|page=323}}

二戰結束後，黃拓榮被派為首任里壠鎮（今[[關山鎮|關山鎮]]）長，並加入[[中國國民黨|中國國民黨]]。{{r|簡|page=323}}

民國三十五年（1946年）冬，黃拓榮出任臺東縣農會理事長，隔年因病辭去。後出任臺東縣農會理事主席、臺東縣警民協會理事長等職。並曾入[[革命實踐研究院|革命實踐研究院]]第十六期及革命實踐研究聯戰第八期結業。{{r|簡|page=323}}

== 政途 ==
1952年，黃拓榮被選為臺東縣議會副議長。1953年，在[[吳金玉|吳金玉]]出任縣長後，空出的議長之位由黃拓榮獲國民黨支持競選，雖然黃拓榮與吳金玉同為[[客家|客家]]人，然而「吳派」卻暗挺[[福佬|福佬]]籍[[許添枝|許添枝]]，雖然在國民黨縣黨部施壓下，迫使許添枝公開放棄競選。然在在投票時，出人意料由許添枝獲得三分之二票數當選議長，{{r|政|page=340}}本應由許添枝宣誓就職。然而最後在國民黨利用休會的五分鐘空檔，軟硬兼施運作下使許添枝宣布辭去議長，傳出膾炙人口的「許添枝大哭辭退議長」、「許添枝痛哭辭退議長」一幕趣劇。{{r|湖|page=59}}{{r|政|page=340}}但由於黃拓榮是在難堪的立場下出任議長，除了對吳金玉恨之入骨外，{{r|政|page=340}}也開啟了「吳派」及「黃派」的惡鬥。{{r|東|page=268}}之後黃拓榮連任第三屆議長。

深知自己在「吳派」絕無出路下，黃拓榮利用主持議會期間，大力培養自己的政治勢力，成為「黃派」的開山始祖。{{r|政|page=340}}黃拓榮網羅地方上具影響力的人士，如[[王錫五|王錫五]]、[[賴萬成|賴萬成]]、[[洪掛|洪掛]]等人，終於在[[1957年中華民國縣市長選舉#臺東縣|第三屆縣長選舉]]時，獲國民黨提名出馬角逐縣長一職。然而其間遭到「吳派」推出的黃忠阻撓，最終仍由黃拓榮當選縣長。

[[1960年中華民國縣市長選舉#臺東縣|第四屆縣長選舉]]，黃拓榮獲國民黨提名角逐連任，又遇上「吳派」暗挺[[林德村|林德村]]與之對抗。最終黃拓榮仍連任成功，然而兩次選舉中他所付出的代價，幾乎使他瀕臨破產的邊緣。{{r|政|page=342}}

黃拓榮在縣長任內，對吳金玉的建設計畫一律予以否定。吳金玉任內曾興建加路蘭漁港的（[[富岡漁港|富岡漁港]]）受到災害損害。黃拓榮不予重視，反而花費三千萬大力投資[[大武漁港|大武漁港]]互別苗頭。在意氣用事下，也導致兩頭落空，浪費了不少公帑。{{r|政|page=342}}{{r|東|page=268}}

縣長卸任後，黃拓榮被擢升任輔導會副祕書長。{{r|政|page=343}}

== 家庭與個人生活 ==
黃拓榮的結髮原配，隱居關山。黃拓榮在政壇上與蔣彩鳳公開同居，並未與元配離婚。黃拓榮在當選縣長後，未依慣例由夫人出任臺東縣婦聯會主任委員，而是推蔣彩鳳出任婦聯會主委。鳩佔鵲巢曾引起很多人不滿，縣議會亦為此大張口伐過，但黃拓榮仍安之泰然。{{r|政|page=343}}
