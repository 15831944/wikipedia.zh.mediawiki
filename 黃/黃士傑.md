'''黃士傑'''（Aja Huang；{{bd|1978年|||}}），台灣人，[[國立臺灣師範大學|國立臺灣師範大學]]資訊工程研究所博士<ref>{{cite web|url=http://pr.ntnu.edu.tw/news/index.php?mode=data&id=16487|title=黃士傑：臺師大是我開發 AlphaGo 的搖籃|date=2017年1月5日|accessdate=2017年6月18日|publisher=[[国立台湾师范大学|国立台湾师范大学]]网站}}</ref>，現為英國[[DeepMind|DeepMind]]工程師，為人工智慧[[AlphaGo|AlphaGo]]的重要推手。

== 生平 ==
[[臺北市立成功高級中學|臺北市立成功高級中學]]畢業。[[國立交通大學|國立交通大學]]資訊科學系畢業，而後進入[[國立臺灣師範大學|國立臺灣師範大學]]資訊工程研究所取得碩士、博士，其博士班時期開發的Erica圍棋程式<ref>{{cite web|url=https://chessprogramming.wikispaces.com/Shih-Chieh+Huang|title=Shih-Chieh Huang|language=En|date=|accessdate=2017年6月18日}}</ref>，曾於2010[[奧林匹亞電腦遊戲程式競賽|奧林匹亞電腦遊戲程式競賽]]得到圍棋冠軍，畢業後加入DeepMind，為AlphaGo的重要推手<ref name="Nature2016">{{cite web |url=https://www.nature.com/nature/journal/v529/n7587/full/nature16961.html|title=Mastering the game of Go with deep neural networks and tree search|publisher=《[[自然_(期刊)|自然]]》|date=2016年1月28日|accessdate=2017年6月18日|language=En}}</ref>。

== 圍棋棋力 ==
曾於1999年獲得全國大專盃圍棋錦標賽段位乙組冠軍<ref>{{cite web|url=http://www.haifong.org/?page_id=1598|title=1999全國大專盃圍棋錦標賽|publisher=[[海峰棋院|海峰棋院]]|date=|accessdate=2017年6月18日}}</ref>，現為台灣業餘六段，常於網上下棋。

== 學術研究 ==
主要研究方向為電腦圍棋，其碩士論文為《電腦圍棋打劫的策略》<ref>{{Cite web|url=http://handle.ncl.edu.tw/11296/ndltd/25964842550487285246 |title=碩士論文《電腦圍棋打劫的策略》|publisher=[[中華民國國家圖書館|中華民國國家圖書館]]|author=黃士傑|date=2003年|accessdate=2017年6月19日}}</ref>，博士論文為《應用於電腦圍棋之蒙地卡羅樹搜尋法的新啟發式演算法》<ref>{{cite web|url=http://pr.ntnu.edu.tw/news/index.php?mode=data&id=15734|title=資工所博士黃士傑擔任GoogleAlphaGo幕後推手 擊敗南韓棋王李世石|publisher=[[国立台湾师范大学|国立台湾师范大学]]网站|date=2016年3月10日|accessdate=2017年6月18日}}</ref>，加入DeepMind後，將AlphaGo成果整理，2016年1月發表於《[[自然_(期刊)|自然]]》杂志，題目名為Mastering the Game of Go with Deep Neural Networks and Tree Search，黄士杰是该论文的两位第一作者之一<ref name="Nature2016"/>。
