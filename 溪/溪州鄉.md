'''溪州鄉'''位於[[臺灣|臺灣]][[彰化縣|彰化縣]]最南端，南隔[[濁水溪|濁水溪]]與[[雲林縣|雲林縣]]相望，位居[[濁水溪沖積扇|濁水溪沖積扇]]之上，地勢低平。溪州鄉面積居全縣第三大（僅次於[[二林鎮|二林鎮]]、[[芳苑鄉|芳苑鄉]]），人口密度則居全縣倒數第四。

== 歷史 ==
<gallery>

</gallery>溪州為彰化縣南端之鄉鎮，過去原為濁水溪沖積河床、浮覆地，故名為「溪州」。過去溪州鄉舊眉村一帶，曾有平埔族「[[巴布薩族|巴布薩]]」在此生活（眉理社）。清朝時期屬於[[東螺西堡|東螺西堡]]。日本統治台灣後，實施市街庄制，改稱為溪州庄，屬台中州北斗郡所轄，設庄役場於今日的溪州村，溪州之所以成為日後的鄉名，至此確立。

=== 莿仔埤圳 ===
{{main|莿仔埤圳}}
莿仔埤圳是溪州鄉主要的灌溉水源，一開始是在乾隆年間由鹿港人陳四芳集資修築，後來經由彰化蔡壽星、埤頭許烏美、溪州葉惠清相繼修建。日本時期之後，日本人公布「公共埤圳規則」，將葉惠清管理之莿仔埤圳收歸官有，並於1909年重新整建完畢，是為台灣第一條官設水圳。日本時代水圳，連同支線總長84公里，灌溉3983公頃之土地。

戰後再度修建，灌溉面積高達8597公頃。

==== 中科四期抗爭 ====
2011年，為了因應下游二林[[中部科學工業園區|中部科學園區]]第四期的引水工程，國科會（現科技部）欲興建一條專管，從上游[[集集攔河堰|集集攔河堰]]引濁水溪溪水，每日6.6萬公噸，直接送到二林園區（參考資料：中部科學工業園區第四期二林園區開發計畫<ref>{{Cite web|url = http://zh.scribd.com/doc/63778436/%E4%B8%AD%E9%83%A8%E7%A7%91%E5%AD%B8%E5%B7%A5%E6%A5%AD%E5%9C%92%E5%8D%80%E7%AC%AC%E5%9B%9B%E6%9C%9F-%E4%BA%8C%E6%9E%97%E5%9C%92%E5%8D%80-%E9%96%8B%E7%99%BC%E8%A8%88%E7%95%AB-%E7%94%A8%E6%B0%B4%E8%A8%88%E7%95%AB%E6%9B%B8|title = 中部科學工業園區第四期（二林園區）開發用水計畫書|accessdate = |author = |date = |publisher = }}</ref>）。
[[File:中科四期搶水工程示意圖.jpeg|thumb]]
此事因廠商至溪州水源頭大庄村埋設管路，遭居民發現，居民群起抗爭。溪州農民認為十年前[[集集攔河堰|集集攔河堰]]完工之後，每天有30公噸水源送至雲林麥寮[[第六套輕油裂解廠|六輕]]，莿仔埤圳的水源就開始短缺，彰化農田水利會時常必須以「供四停六」的方式調節、調度水源。若是中科四期專管再將6.6萬公噸的水源送至二林，這樣溪州鄉的水源勢必更加短缺，因此溪州鄉民拒絕讓引水工程在溪州施工（參考資料：環資中心報導<ref>{{Cite news|url = http://e-info.org.tw/node/69411|title = 搶濁水、排毒水，中科四期上、中、下游吃夠夠|last = |first = |date = |work = |accessdate = }}</ref>）。

根據《立報》2011/8/22報導，這個工程主要爭議可分為：
# 引水工程之沉砂池超過10公頃（位於溪州鄉境內），尚未通過環評。
# 工程目的與主觀機關不明卻已發包給偉盟工程。
# 彰南地區嚴重缺水，農業及民生用水必須大量抽取地下水，已有地層下陷之虞。（資料來源<ref>{{Cite news|url = http://www.lihpao.com/?action-viewnews-itemid-110101|title = 中科工程無主 硬是違法搶水|last = |first = |date = |work = |accessdate = }}</ref>）
後經溪州在地作家[[吳晟|吳晟]]、[[台灣農村陣線|台灣農村陣線]]等團體、各界民眾聲援之下。2012年初，當時新就任的國科會主委[[朱敬一|朱敬一]]承諾重新檢討園區規劃（資料來源<ref>{{Cite news|url =http://pnn.pts.org.tw/main/2012/03/16/%E5%8F%8D%E4%B8%AD%E7%A7%91%E6%90%B6%E6%B0%B4/|title =中科四期停工檢討 農民要求廢止徵收、搶水|last =|first =|date =|work =|accessdate =|archive-url =https://web.archive.org/web/20160304193559/http://pnn.pts.org.tw/main/2012/03/16/%E5%8F%8D%E4%B8%AD%E7%A7%91%E6%90%B6%E6%B0%B4/|archive-date =2016-03-04|dead-url =yes}}</ref>）。但引水工程仍持續進行，2012年5月10日，施工廠商於溪州鄉東州村一帶準備動工的時候，作家吳音寧於怪手前靜坐阻擋，隨後鄉親並共同靜坐表達抗議（資料來源<ref>{{Cite news|url = http://www.newsmarket.com.tw/blog/6272/|title = 【公民寫手】靜坐在怪手之前─守護我們的母親之河|last = |first = |date = |work = |accessdate = }}</ref>）。經過將近百日的靜坐之後，行政院同意從莿仔埤圳埤頭段（12.5K處）取水，結束長達一年半的反引水工程抗爭。

== 地理 ==
本位於彰化縣最南邊的溪州鄉，南臨濁水溪與雲林縣[[西螺鎮|西螺鎮]]、[[莿桐鄉|莿桐鄉]]相望，東鄰二水鄉與田中鎮，北鄰北斗鎮、西接埤頭鄉與竹塘鄉，全鄉面積約75.831平方公里，南北長8.18公里，東西寬13.6公里。本鄉主要聯外交通為南北向的[[台1線|台一線]]及國道一號南下交流道，南方由[[西螺大橋|西螺大橋]]及溪州大橋連接雲林縣西螺鎮。

本鄉全境皆為濁水溪沖積扇扇頂位置，由北而南依序是[[東螺溪|東螺溪]]（舊濁水溪）、西螺溪（今日濁水溪）、[[虎尾溪|虎尾溪]]、[[北港溪|北港溪]]，至日本時代東螺溪是主流，本鄉正位在東西螺溪間（舊濁水溪與今日濁水溪）的浮覆地，目前分流之處位於溪州鄉的大庄村。因是溪中之沙洲，故名「溪洲」，後來才改為「溪州」。

== 經濟 ==
溪州鄉產業是以農業為主，過去製糖曾盛極一時，1970台糖搬遷之後逐漸沒落。今天溪州農業是以稻米、蔬果、苗木為主，其中以水田稻作為最大宗，水果以芭樂為多、苗木則以[[馬拉巴栗|馬拉巴栗]]較為有名。另外溪州也有[[正新輪胎|正新輪胎]]、允強不銹鋼等大型工廠。

=== 糖業 ===
溪州糖廠，舊稱溪州製糖所，位於台中州北斗郡溪州庄，為現今彰化縣溪州庄。1909年，日本政府要求當時台灣的富商[[林本源家族|板橋林家]]，在台灣總督府鼓勵糖業的慫恿下，設立林本源製糖合名會社，並於溪州庄設立新式製糖工場。1913年時，林本源製糖合名會社增資，改名為[[林本源家族|林本源]]製糖株式會社。1925年溪州製糖所爆發[[二林事件|二林蔗農事件]]，造成極大紛爭；加上板橋林家內部不和，1927年時，林本源製糖株式會社併入鹽水港製糖株式會社，將製糖工場改稱為溪州製糖所，並提升壓榨能力至2700公噸。

戰後，屬台灣糖業公司第四區分公司，改名溪州糖廠；雖因遭受轟炸攻擊廠區受損，但復舊工程相當順利，於1945年12月即復工製糖。1950年改行總廠制，屬台中總廠。1954年，彰化糖廠與溪州糖廠併入[[溪湖糖厂|溪湖糖廠]]，製糖工廠亦隨之停閉，製糖設備於1959年遷至善化糖廠。

1955年時台灣總公司遷至溪州糖廠廠區辦公，廠區內曾經有過著名的台糖招待所「拾翠樓」。可惜總公司於1970年才遷回台北後，該區土地經多年閒置，建築全數遭到拆除，僅留一小片圍牆，遺址現作為溪州森林公園使用（資料引用來源：[http://map.net.tw/taisugar/?dir-item=%E6%BA%AA%E5%B7%9E%E7%B3%96%E5%BB%A0 台灣製糖工場百年文史地圖]）。依據台糖員工回憶，總公司之所以將辦公室遷自溪州，是為了避免台北遭戰火波及，故「疏散」至溪州。於是當時總經理[[楊繼曾|楊繼曾]]，吩咐協理張季熙主辦遷移事宜。

目前溪州森林公園仍然保存有「順風牌47號」內燃機車，即[[臺灣糖業鐵路|五分車]]火車頭，1956年開始運轉。

今日溪州糖業盛況已不如當年，生食甘蔗種植面積約24公頃左右。

=== 稻米 ===
溪州鄉一期稻作面積約1800公頃，二期稻作約1700公頃，稻作面積為彰化縣第二名，僅次於二林鎮<ref>{{Cite web|url=http://agr.afa.gov.tw/afa/afa_frame.jsp|title=農委會農糧署農情調查網|accessdate=2016-07-21|author=|date=|publisher=}}</ref>，為台灣重要糧倉之一。

詩人[[吳晟|吳晟]]形容：「吾鄉鄉民多數從事農業，耕作田地灌溉用水，引自濁水溪渠道。濁水溪水流最大特質，是從高山峻嶺的上游，挾帶不斷崩解的鐵板沙奔騰而下，灌溉用水流進農田，泥沙逐漸沉積，年復一年，積成豐厚肥沃的黑色土壤。黑色土壤的廣大農田，以種植水稻為主，這就是『頂港有名聲、下港有出名』非常聞名的『濁水米』，又香又Q的『祕訣』。」（引用自吳晟〈大圳悲嘆〉：自由時報 2011/8/7）<ref>{{Cite news|url = http://news.ltn.com.tw/news/supplement/paper/514517|title = 大圳悲嘆|last = 吳|first = 晟|date = |work = |accessdate = }}</ref>

彰南平原自從八堡圳、莿仔埤圳圳完成之後，形成大片肥沃的水田，溪州鄉成為稻米的重要產區，目前每期稻作約有1800公頃左右。近年養生風氣盛行，溪州糧商推出「黑米」，為秈稻（在來米）與糯稻雜交的品種，果皮為紫黑色，富涵花青素，而糙米狀態營養成分較白米高，因此受到重視健康的消費者歡迎，打響「溪州黑米」的品牌<ref>{{Cite news|url = http://news.ltn.com.tw/news/local/paper/822437|title = 〈中部〉紫黑米養生夯 彰化溪州種植面積倍增|last = |first = |date = |work = |accessdate = }}</ref>。

另外在2012年護水抗爭之後，溪州農民組成[http://water-farmer.com.tw/ 「溪州尚水」友善農產平台]，董事長為農民謝寶元。2013年莿仔埤圳產業文化協會進行「水田濕地復育計畫」，並與特有生物中心的研究團隊合作進行生態調查，以友善生態、友善農民的方式，每分地以合理價格向農友契約收購，期望能帶動溪州友善稻米的產業。

=== 彰南產業園區 ===
行政院環保署於1998年9月7日通過「彰化人纖專業工業區開發計畫」環評，彰化人纖於該年獲准報編為工業區，由彰化縣政府主管（時任縣長為[[阮剛猛|阮剛猛]]）。但此工業區因經濟不景氣等因素一直未能完成開發及招商。2010年彰化縣政府復於人纖工業區開始了新的「彰化縣彰南產業園區」開發計畫。

彰南產業園區位於彰化縣南端，溪州鄉之台糖水尾農場，中山高速公路北斗交流道西南方約6公里處，溪州焚化爐旁。約100公頃之土地。主要目的事業為橡膠業，如正新、建大等企業。溪州鄉水尾村附近居民得知後，組成自救會反對。2014年九合一選舉時，民進黨彰化縣長候選人[[魏明谷|魏明谷]]及民進黨主席[[蔡英文|蔡英文]]來到水尾村震威宮拜票，表示應堅持土地正義，不要造成農地的災難<ref>{{Cite news|url=http://news.ltn.com.tw/news/politics/breakingnews/1128729|title=蔡英文談彰南產業園區 強調土地正義 - 政治 - 自由時報電子報|accessdate=2016-12-06}}</ref>。

魏明谷當選彰化縣長之後，經過四個月的思考，終於提出「彰南農業、彰北工業」，取消彰南產業園區計畫<ref>{{Cite news|url=https://www.thenewslens.com/article/15171|title=捏一把冷汗...魏明谷宣布：彰南產業園區撤案 - The News Lens 關鍵評論網|last=Yang|date=2015-04-10|newspaper=The News Lens 關鍵評論網|accessdate=2016-12-06|language=zh-TW}}</ref>，考慮往農業生態園區方向發展。但目前工業用地遲未解編，當時縣府表示程序正在進行中<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/1825388|title=彰南產業園區工業地解編 歷時1年仍原地踏步 - 生活 - 自由時報電子報|accessdate=2016-12-06}}</ref>。

2017年8月，彰化縣政府預計將園區轉型成「彰南高科技農業產業園區」，目前正在籌備中。<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/2163824|title=賠錢轉型！彰南產業園區輪胎王國夢碎 開發溫室蔬果定調|author=|date=|work=|accessdate=}}</ref>
