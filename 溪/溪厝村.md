'''溪厝村'''是[[中華民國|中華民國]][[臺灣|臺灣]][[嘉義縣|嘉義縣]][[六腳鄉|六腳鄉]]下的行政村，比鄰朴子溪北岸，因日治時期河川氾濫而多次移居而成。曾為日治時期早期抵抗日軍佔領的集結地之一。轄下有12個鄰、約265戶，人口約有691人。

== 地理 ==
位於[[嘉南平原|嘉南平原]][[朴子溪|朴子溪]]北側，村莊範圍瀕臨河岸，得地名為「溪墘厝」。今日溪厝村區域以南臨朴子溪、北邊以中溝排水線為範圍。 日治時期因朴子溪曾於1918年、1933年有大規模氾濫，不適居住，瀕臨河岸的「十八戶仔」聚落居民，與其他朴子溪畔居民移住比鄰的[[朴子市|朴子市]]「新吉庄」(日治時期地名，原意為閩南語的『新(sin)結(kiat)庄(tsng)』，今日朴子市[[德興里|德興里]]範圍)。 [[嘉南大圳|嘉南大圳]]於日治時期(1895-1943年))末期落成後，易受水患波及的居民又經搬遷至目前較遠離河邊的嘉50縣道旁，形成目前的聚落樣貌<ref>{{Cite web|url=http://gn.moi.gov.tw/geonames/index.aspx|title=地名資訊服務網|accessdate=2017-07-05|author=|date=|publisher=}}</ref>。

== 聚落歷史 ==
聚落沿朴子溪建立，於1651年([[荷據時期|荷據時期]])由[[福建省|福建省]][[泉州府|泉州府]]南安的侯定一族於此墾殖，在1851年(咸豐1年)侯姓家族籌建庄廟(德龍宮)的紀錄。詹姓居民則集中於今日溪厝村主要的街區。

甲午戰後(1895)以經營糖廍為業的[[侯西庚|侯西庚]](1858-1904)，組織溪厝村附近民眾抵抗日軍接收。主要作戰區域為今日[[布袋鎮|布袋鎮]]布袋嘴區域及[[東石鄉_(台灣)|東石鄉]]，因居民善用海邊累積蚵殼堆作為戰鬥掩護，稱為「蚵殼城之役」(1895年10月12日至14日)。戰役失敗後地方居民潰逃，侯西庚流亡福建省[[厦门市|廈門]]，晚年回鄉居住。其子侯江也受日治時期政治所累，需佯裝癲狂以免被日治時期逮捕<ref>{{Cite web|url=http://old.tncsec.gov.tw/b_native_stage/index_view.php?act=home&c03=41&a01=0302&c04=3&num=1811|title=臺南生活美學館─人文景觀─奇人妙事|accessdate=2017-07-05|author=|date=|publisher=}}</ref>。

== 教育 ==
本區域未曾有過官方設立的教育設施，目前為蒜頭國民小學及六嘉國民中學的學區範圍內。

== 文化資源 ==
本地以民間信仰的文化資源為主：
* 慈聖宮<ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/|title=文化資源地理資料庫─慈聖宮|accessdate=2017-07-05|author=|date=|publisher=}}</ref>：主神為[[天上聖母|天上聖母]]。建立於1999年。
* 德龍宮<ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/|title=文化資源地理資料庫─德龍宮|accessdate=2017-07-05|author=|date=|publisher=}}</ref>：主神為[[刑府千歲|刑府千歲]]。建立於1851年(咸豐1年)，1907年(明治40年)、1964年曾部分修葺。
* 龍旨宮<ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/|title=文化資源地理資料庫─龍旨宮|accessdate=2017-07-05|author=|date=|publisher=}}</ref>：主神為[[關聖帝君|關聖帝君]]。建立於1931年(昭和6年)，為新港鄉南港村[[新港水仙宮|水仙宮]]的分靈廟宇。
