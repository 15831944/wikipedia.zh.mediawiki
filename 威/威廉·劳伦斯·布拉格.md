'''威廉·劳伦斯·布拉格'''爵士，[[名譽勳位|CH]]，[[OBE|OBE]]，[[軍功十字勳章|MC]]，[[皇家學會|FRS]]（{{lang-en|Sir '''William Lawrence Bragg'''}}，{{bd|1890年|3月31日|1971年|7月1日|catIdx=B}}），出生於[[澳洲|澳洲]]的[[物理学家|物理学家]]，他擁有澳洲和[[英國|英國]]雙重國籍，因為發現了關於X射線衍射的[[布拉格定律|布拉格定律]]，1915年与其父[[威廉·亨利·布拉格|威廉·亨利·布拉格]]一同获得[[诺贝尔物理学奖|诺贝尔物理学奖]]。<ref name="nobel_1915">{{cite web |url=http://nobelprize.org/nobel_prizes/physics/laureates/1915/index.html |title=The Nobel Prize in Physics 1915 |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date= |accessdate=2008年12月26日 |language=en }}</ref>

== 成长经历 ==
劳伦斯·布拉格1890年出生于[[澳大利亚|澳大利亚]][[阿德莱德|阿德莱德]]。<ref name="nobel_bio">{{cite web |url=http://nobelprize.org/nobel_prizes/physics/laureates/1915/wl-bragg-bio.html |title=Biography |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date= |accessdate=2008年12月26日 |language=en }}</ref>当时他的父亲威廉·布拉格在[[阿德莱德大学|阿德莱德大学]]当[[数学|数学]][[物理|物理]]教授。劳伦斯·布拉格年少时便已表现出对科学和数学的兴趣。5岁时，刚上学不久的他有一次从[[三轮车|三轮车]]上掉下，摔伤了手臂。他父亲此前不久刚好了解到[[威廉·伦琴|威廉·伦琴]]对[[X射线|X射线]]的最新研究，于是他使用X射线对劳伦斯·布拉格的手臂进行检查。这成为澳大利亚有记载的首次在外科医学中使用X射线。<ref name="cam">{{cite web |url=http://www-outreach.phy.cam.ac.uk/camphy/physicists/bragg_prelim.htm |title=Cambridge Physicists - William Lawrence Bragg |publisher=[[剑桥大学|剑桥大学]]物理系 |date= |accessdate=2008年12月26日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20070721155224/http://www-outreach.phy.cam.ac.uk/camphy/physicists/bragg_prelim.htm |archivedate=2007年7月21日 }}</ref>

劳伦斯·布拉格是一名有才华的学生。他先在[[圣彼得学院|圣彼得学院]]上学，1904年年仅14岁时进入阿德莱德大学，学习数学、[[化学|化学]]和物理，1908年毕业。同年，他父亲接受了[[利兹大学|利兹大学]]提供的教授职位，全家搬回英国。1909年秋，劳伦斯·布拉格尽管因患[[肺炎|肺炎]]而在病床上参加考试，仍获得数学[[奖学金|奖学金]]，入读[[剑桥大学|剑桥大学]][[三一学院_(剑桥)|三一学院]]。完成数学方面的学习后，他转入物理学习，于1911年毕业。<ref name="nobel_bio"/><ref name="cam"/>

== 学术经历 ==
=== X射线结晶学 ===
劳伦斯·布拉格最著名的成就是对X射线[[衍射|衍射]]的研究，以及据此提出的[[布拉格定律|布拉格定律]]。布拉格定律给出了受到[[电磁辐射|电磁辐射]]和[[粒子|粒子]]波照射时，[[晶体|晶体]]内[[原子|原子]]平面间隔与在该平面上产生最强反射的入射角之间的关系，可以用于测定[[波长|波长]]和晶体的点阵间隔。<ref name="law">{{cite web |url=http://tw.britannica.com/MiniSite/Article/id00005974.html |title=布拉格定律 |publisher=[[大英百科全書|大英百科全書]]線上繁體中文版 |date= |accessdate=2008年12月26日 |language=zh }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>这个定律是1912年劳伦斯·布拉格研究生第一年时发现的。其间，他与在利兹大学研究X射线[[分光计|分光计]]的父亲进行了讨论。由于是父子间的合作，有人认为这项研究始于父亲，这让劳伦斯·布拉格感到难过。<ref name="cam"/>父子两人对X射线衍射的研究为后来[[DNA|DNA]][[双螺旋形|双螺旋结构]]的发现奠定了基础。<ref name="aus">{{cite web |url=http://www.guangzhou.china.embassy.gov.au/gzhochinese/Dest12.html |title=12位澳大利亚诺贝尔奖得主 |publisher=[[澳大利亚|澳大利亚]]驻[[广州|广州]][[总领事馆|总领事馆]] |date= |accessdate=2008年12月26日 |language=zh }}</ref>

他的研究在两次[[世界大战|世界大战]]期间都被迫中断，战争时期他为政府研究[[声波|声波]]测距方法。1919年至1937年，他在[[曼彻斯特维多利亚大学|曼彻斯特维多利亚大学]]担任物理教授。1937年[[卢瑟福|卢瑟福]]去世后，劳伦斯·布拉格接替他成为剑桥大学[[卡文迪许实验室|卡文迪许实验室]]主任。二战结束后，他将实验室分为若干个研究小组。他认为“由6到12名[[科学家|科学家]]带领一些助手组成的团队是最理想的研究单元”。<ref name="cam"/>

=== 蛋白质 ===
1948年，劳伦斯·布拉格开始对[[蛋白质|蛋白质]]结构产生兴趣，并且在部分方面负责创建一个使用物理方法解决[[生物|生物]]问题的研究小组。1953年DNA双螺旋结构的发现过程中，他尽管没有直接参与，但也做出了贡献，在他的支持下，[[弗朗西斯·克里克|弗朗西斯·克里克]]和[[詹姆斯·沃森|詹姆斯·沃森]]在卡文迪许实验室开展这方面的研究。而他40年前做的X射线研究也在这一过程中得到应用。同一时期，[[马克斯·佩鲁茨|马克斯·佩鲁茨]]在卡文迪许实验室开展[[血红蛋白|血红蛋白]]研究，后来他也因这项研究获得1962年[[诺贝尔化学奖|诺贝尔化学奖]]。经劳伦斯·布拉格提名，克里克、沃森以及[[莫里斯·威尔金斯|莫里斯·威尔金斯]]成为1962年[[诺贝尔生理学或医学奖|诺贝尔生理学或医学奖]]的候选，并最终获奖。<ref name="cam"/>

=== 皇家研究所 ===
1953年，劳伦斯·布拉格成为[[皇家研究所|皇家研究所]]富勒里安化学教授。他提出，皇家研究所应当提供一些公众服务，并建议开展一系列演示实验，供中小学生们观摩。这一想法得到积极回应，到1965年，平均每年有两万名学生观摩了这一系列的实验讲座。他在皇家研究所工作到1966年9月后退休。<ref name="cam"/>

== 荣誉 ==
1915年，因“开展用X射线分析晶体结构的研究”，劳伦斯·布拉格和他的父亲威廉·亨利·布拉格一起获得诺贝尔物理学奖。<ref name="nobel_1915"/>当时他年仅25岁，成为历史上最年轻的诺贝尔奖获得者，这个纪录直到2014年才由獲得[[諾貝爾和平獎|諾貝爾和平獎]]的[[馬拉拉·優素福扎伊|馬拉拉]]打破，但布拉格仍為史上最年輕的物理学奖得主。此外，他还先后获得[[马泰乌奇奖章|马泰乌奇奖章]]（1915年）<ref>{{cite web |url=http://www.accademiaxl.it/finalita_prscientifici_med_matteucci_eng.php |title=Matteucci Medal |publisher=意大利科学院 |date= |accessdate=2008年12月26日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20080401080739/http://www.accademiaxl.it/finalita_prscientifici_med_matteucci_eng.php |archivedate=2008年4月1日 }}</ref>、[[休斯奖章|休斯奖章]]（1931年）<ref>{{cite web |url=http://royalsociety.org/page.asp?id=1770 |title=Hughes archive winners 1989 - 1902 |publisher=[[皇家学会|皇家学会]] |date= |accessdate=2008年12月26日 |language=en }}</ref>、[[皇家奖章|皇家奖章]]（1946年）<ref>{{cite web |url=http://royalsociety.org/page.asp?id=1752 |title=Royal archive winners 1949 - 1900 |publisher=[[皇家学会|皇家学会]] |date= |accessdate=2008年12月26日 |language=en }}</ref>和[[科普利奖章|科普利奖章]]（1966年）<ref>{{cite web |url=http://royalsociety.org/page.asp?id=1742 |title=Copley archive winners 1989 - 1900 |publisher=[[皇家学会|皇家学会]] |date= |accessdate=2008年12月26日 |language=en }}</ref>。

1921年，他当选为[[皇家学会|皇家学会]]院士。<ref name="physica">{{cite web |url=http://www.physica.cn/about/015.htm |title=亨利·布拉格，劳伦斯·布拉格 |publisher=physica.cn |date= |accessdate=2008年12月26日 |language=zh |archive-url=https://web.archive.org/web/20081011012751/http://www.physica.cn/about/015.htm |archive-date=2008年10月11日 |dead-url=yes }}</ref>他还是[[美国|美国]]、[[法国|法国]]、[[瑞典|瑞典]]、[[中国|中国]]、[[荷兰|荷兰]]和[[比利时|比利时]]等国科学院的名誉院士或外籍院士。<ref name="nobel_bio"/>

他先后被[[英国王室|英国王室]]授予[[大英帝国勋章|大英帝国勋章]]（1918年）、[[军功十字勋章|军功十字勋章]]（1918年）和[[名誉勋位|名誉勋位]]（1967年）。1941年被封为[[爵士|爵士]]。<ref name="physica"/>

1965年12月，他应诺贝尔基金会之邀，在斯德哥尔摩介绍了其研究领域过去50年提的发展状况，这也是历史上第一个“诺贝尔讲座”。<ref name="nobel_bio"/>

自1992年起，澳大利亚物理学会设立一个全国年度最佳物理博士论文奖项，向最佳论文的作者颁发“布拉格金牌”（The Bragg Gold Medal for Excellence in Physics）。这枚奖牌的的命名是为了纪念布拉格父子，奖牌正面图案为劳伦斯·布拉格肖像。<ref name="gold">{{cite web |url=http://www.aip.org.au/content/bragg |title=The Bragg Gold Medal for Excellence in Physics |author= |publisher=澳大利亚物理学会 |date= |accessdate=2008年12月26日 |language=en }}</ref>

== 个人生活 ==
劳伦斯·布拉格1921年与艾丽丝·霍普金森结婚。他们育有四个子女。<ref name="nobel_bio"/>劳伦斯·布拉格的爱好包括[[绘画|绘画]]、[[文学|文学]]和[[园艺|园艺]]。<ref name="cam"/>
