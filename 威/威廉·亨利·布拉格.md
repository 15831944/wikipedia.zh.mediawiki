'''威廉·亨利·布拉格'''爵士，[[功績勳章_(英國)|OM]]，[[KBE|KBE]]，[[皇家學會院士|FRS]]（{{lang-en|Sir '''William Henry Bragg'''}}，{{bd|1862年|7月2日|1942年|3月10日|catIdx=B}}），[[英国|英国]][[物理学家|物理学家]]、[[化学家|化学家]]，1915年与其子[[威廉·劳伦斯·布拉格|威廉·劳伦斯·布拉格]]一同获得[[诺贝尔物理学奖|诺贝尔物理学奖]]。<ref name="nobel_1915">{{cite web |url=http://nobelprize.org/nobel_prizes/physics/laureates/1915/index.html |title=The Nobel Prize in Physics 1915 |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date= |accessdate=2008年12月23日 |language=en }}</ref>

== 早期 ==
威廉·布拉格1862年出生于英国[[威格顿|威格顿]]。他先后在Market Harborough[[文法学校|文法学校]]和[[马恩岛|马恩岛]]的威廉国王学院学习，完成大学之前的教育。<ref name="nobel_bio">{{cite web |url=http://nobelprize.org/nobel_prizes/physics/laureates/1915/wh-bragg-bio.html |title=Biography |publisher=[[诺贝尔基金会|诺贝尔基金会]] |date= |accessdate=2008年12月23日 |language=en }}</ref>1881年，他获得[[奖学金|奖学金]]入读[[剑桥大学|剑桥大学]][[三一学院_(剑桥)|三一学院]]，在著名教师[[爱德华·约翰·劳思|爱德华·约翰·劳思]]的指导下学习[[数学|数学]]。<ref name="nobel_bio"/>1884年6月，他在优等生数学考试中名列第一部分第三，1885年初进入第二部分学习，同年有一段时间在[[卡文迪许实验室|卡文迪许实验室]]学习[[物理|物理]]。<ref name="physica">{{cite web |url=http://www.physica.cn/about/015.htm |title=亨利·布拉格，劳伦斯·布拉格 |publisher=physica.cn |date= |accessdate=2008年12月23日 |language=zh |archive-url=https://web.archive.org/web/20081011012751/http://www.physica.cn/about/015.htm |archive-date=2008-10-11 |dead-url=yes }}</ref><ref name="gutenberg">{{cite web |url=http://gutenberg.net.au/dictbiog/0-dict-biogBr-By.html#bragg1 |title=BRAGG, SIR WILLIAM HENRY (1862-1942) |publisher=Dictionary of Australian Biography |date= |accessdate=2008年12月23日 |language=en }}</ref><ref name="adb">{{cite web |url=http://www.adb.online.anu.edu.au/biogs/A070396b.htm |title=Bragg, Sir William Henry (1862 - 1942) |author=S. G. Tomlin |publisher=Australian Dictionary of Biography, Online Edition |date= |accessdate=2008年12月23日 |language=en }}</ref>

== 阿德莱德大学时期 ==
1885年，威廉·布拉格被[[澳大利亚|澳大利亚]][[阿德莱德大学|阿德莱德大学]]聘为数学物理[[教授|教授]]，于1886年初正式上任。此前他的物理知识并不多，在[[阿德莱德|阿德莱德]]他才大量学习物理知识，但真正涉及到重要研究已经是40岁之后了。<ref name="gutenberg"/>1904年，在[[但尼丁|但尼丁]]召开的一次澳大拉西亚科学促进会的会议上，他担任所在小组的主席，并发表了论文''Some Recent Advances in the Theory of the Ionization of Gases''（《气体电离理论的新发展》）。后来他在这篇论文的基础上，继续开展研究，于1912年出版了他的第一本著作''Studies in Radioactivity''（《放射能研究》）。<ref name="gutenberg"/>1904年那次会议后不久，他得到一些溴化镭，并进行相关研究，当年年底在''Philosophical Magazine''（《哲学杂志》）上发表了关于[[镭|镭]][[射线|射线]]的研究论文。<ref name="gutenberg"/>1907年，他当选英国[[皇家学会|皇家学会]]会士。<ref name="nobel_bio"/>1908年底，他从阿德莱德大学辞职。他在这所大学的23年间，见证了其学生数的数倍增长，对其理学院的发展也尽到了最大的贡献。<ref name="gutenberg"/>

== 利兹大学时期 ==
1909年，威廉·布拉格到[[利兹大学|利兹大学]]担任卡文迪许物理教授。他在这里继续[[X射线|X射线]]研究，并大获成功。他发明了X射线[[分光计|分光计]]，并与他的儿子[[威廉·劳伦斯·布拉格|威廉·劳伦斯·布拉格]]创立了用X射线分析[[晶体结构|晶体结构]]的新学术领域。这项技术的应用为稍后[[DNA|DNA]]双螺旋结构的发现奠定了基础。<ref name="aus">{{cite web |url=http://www.guangzhou.china.embassy.gov.au/gzhochinese/Dest12.html |title=12位澳大利亚诺贝尔奖得主 |publisher=[[澳大利亚|澳大利亚]]驻[[广州|广州]][[总领事馆|总领事馆]] |date= |accessdate=2008年12月23日 |language=zh }}</ref>正是由于这项成就，1915年父子两人一同被授予诺贝尔物理学奖。<ref name="nobel_1915"/>

== 伦敦大学学院时期 ==
1915年，威廉·布拉格被[[伦敦大学学院|伦敦大学学院]]聘为奎恩物理教授，但受[[第一次世界大战|第一次世界大战]]的影响，他直到战争结束后才开始工作。战争期间，他主要为英国政府服务，进行[[潜艇|潜艇]]探测的研究。1918年，他回到[[伦敦|伦敦]]，担任海军司令部的顾问。恢复在大学的工作后，他主要从事的研究仍然是晶体结构分析。<ref name="gutenberg"/>

== 皇家研究所时期 ==
1923年起，他成为[[皇家研究所|皇家研究所]]的富勒里安化学教授和戴维·法拉第研究实验室（Davy Faraday Research Laboratory）的主任。在他的领导下，实验室发表了大量有价值的论文。<ref name="gutenberg"/>1935年，他当选为英国皇家学会的会长。<ref name="nobel_bio"/>

== 荣誉 ==
除外诺贝尔物理学奖之外，威廉·布拉格还先后获得过[[马泰乌奇奖章|马泰乌奇奖章]]（1915年）<ref>{{cite web |url=http://www.accademiaxl.it/finalita_prscientifici_med_matteucci_eng.php |title=Matteucci Medal |publisher=意大利科学院 |date= |accessdate=2008年12月23日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20080401080739/http://www.accademiaxl.it/finalita_prscientifici_med_matteucci_eng.php |archivedate=2008年4月1日 }}</ref>、[[拉姆福德奖章|拉姆福德奖章]]（1916年）<ref>{{cite web |url=http://royalsociety.org/page.asp?id=1747 |title=Rumford archive winners 1988 - 1900 |publisher=[[皇家学会|皇家学会]] |date= |accessdate=2008年12月23日 |language=en }}</ref>和[[科普利奖章|科普利奖章]]（1930年）<ref>{{cite web |url=http://royalsociety.org/page.asp?id=1742 |title=Copley archive winners 1989 - 1900 |publisher=[[皇家学会|皇家学会]] |date= |accessdate=2008年12月23日 |language=en }}</ref>。他还先后被英国王室授予司令勋章（C.B.E.，1917年）、爵级司令勋章（K.B.E.，1920年）和[[功绩勋章|功绩勋章]]（O.M.，1931年）。<ref name="nobel_bio"/>

他上过的文法学校和威廉国王学院都有以他名字命名的建筑，作为对这位杰出毕业生的纪念。自1992年起，澳大利亚物理学会设立一个全国年度最佳物理博士论文奖项，向最佳论文的作者颁发“布拉格金牌”（The Bragg Gold Medal for Excellence in Physics），这枚奖牌的的命名是为了纪念布拉格父子。<ref name="gold">{{cite web |url=http://www.aip.org.au/content/bragg |title=The Bragg Gold Medal for Excellence in Physics |author= |publisher=澳大利亚物理学会 |date= |accessdate=2008年12月23日 |language=en }}</ref>
