'''威廉一世'''（{{lang-de|'''Wilhelm I'''}}，{{bd|1797年|3月22日|1888年|3月9日|catIdx=W}}），全名'''威廉·腓特烈·路德维希'''（{{lang|de|Wilhelm Friedrich Ludwig}}），[[普鲁士|普鲁士]]国王（1861年1月2日—1888年3月9日），1871年1月18日就任[[德意志帝国|德意志帝国]]第一任皇帝。他死後，因為[[德意志統一|德意志統一]]，被其孫[[威廉二世_(德國)|威廉二世]]尊為[[大帝|大帝]]，號稱「威廉大帝」。

== 早期生活和军事生涯 ==
[[File:Franz_Krüger_001.jpg|thumb]]骑马并行，1836年]]

威廉一世是普鲁士国王[[腓特烈·威廉三世|腓特烈·威廉三世]]的第二子，母亲为路易斯王后。由于没有预料到会登上王位，威廉親王并没有接受多少教育。1814年2月入伍参与反[[拿破仑|拿破仑]]战争，服役报告称他是一名勇敢的士兵。1815年后，他也成为一个有魅力的[[外交家|外交家]]。1848年他成功地粉碎了针对其兄长[[腓特烈·威廉四世|腓特烈·威廉四世]]的政变，得到“霰弹亲王”的绰号。1857年，无子女的腓特烈·威廉四世[[中风|中风]]，身体局部瘫痪，然后精神失常无法料理国事，1858年10月威廉親王出任摄政。

威廉一世於生涯中對普魯士实施軍事改革，普魯士許多的軍事作風都源自於他，老毛奇被任命掌管參謀本部也要歸功於威廉一世。
若無威廉一世的軍事改革，俾斯麥所實施的數場戰爭在軍事層面上或許都無法進展順利。

== 国王和皇帝 ==
1861年1月2日，腓特烈·威廉四世逝世，威廉親王登基為普魯士威廉一世。他登位後，國王和[[議會|議會]]仍然不和。他比他兄長較少干預政治，一般被認為是一個政治中立的人。但他只不過是尋找一個謹慎的解決方法，去解決鬥爭：他任命保守派的[[奧托·馮·俾斯麥|奧托·馮·俾斯麥]]為首相。根據普魯士[[憲法|憲法]]，首相只需對听从國王，並不需要搭理議會。虽然俾斯麥说他和威廉的工作關係是一個臣子對他的長官盡忠，实际上俾斯麥拿着管理內政和外交的实权。数次与威廉不同意见，俾斯麥揚言辭職，威廉只好同意俾斯麥的做法。

普军在[[普法戰爭|普法戰爭]]取胜，1871年1月18日，威廉在巴黎[[凡爾賽宮|凡爾賽宮]]称帝，亦即'''德意志皇帝（Deutscher Kaiser）'''。儀式過後，[[北德意志邦聯|北德意志邦聯]] (1867-1871) 轉改為[[德意志帝國|德意志帝國]] ("Deutsches Reich", 1871-1918) 。德意志帝國是一個资本主义國家；皇帝是君主們的[[國家元首|國家元首]]和[[總統|總統]]。君主包括[[巴伐利亞|巴伐利亞]]、[[符騰堡|符騰堡]]与[[薩克森|薩克森]]國王；[[巴登|巴登]]和[[黑森|黑森]][[大公|大公]]；[[漢堡|漢堡]]、[[呂貝克|呂貝克]]和[[不萊梅|不萊梅]][[參議院|參議院]]等。威廉不情願地接受“德意志皇帝”的名號。他曾提議“'''德國皇帝（Kaiser von Deutschland）'''”的名號，但明显地君主們不會接受。
[[File:Wernerprokla.jpg|thumb]]
在俾斯麥的回憶錄裏，俾斯麥认为威廉是一個傳統、謙恭、絕對有禮的紳士，是名副其實的普魯士軍官。他有些判斷偶爾會被“婦人之仁”所影響。

1878年5月11日，叛亂者马克斯·霍德尔（Max Hödel）在[[柏林|柏林]]企圖行刺皇帝威廉一世，但失敗。同年6月2日，卡尔·诺比林格（Karl Nobiling）试图行刺，伤了威廉之后自尽。這成為[[清除社會主義法|清除社會主義法]]於1878年10月21日設立的原因。法例由俾斯麥的政府提出，並受到[[德国国会|国会]]的大力支持。法例目的是打擊[[社會主義者|社會主義者]]和[[工人階級|工人階級]]的行動，并剝奪了[[德國社會民主黨|德國社會民主黨]]的合法地位。它禁止所有黨組織、工人大眾組織、社會主義者和工人階級的刊物，并可用以裁決或充公社會主義者的文學作品，但又給予社會民主主義者賠償。法例每二、三年擴大一次。虽然惩罚严重，社會民主黨影響继续扩大。基於大量工人階級行動的壓力，法例於1890年10月1日被廢除。

== 家庭 ==
1829年1月11日，威廉一世在柏林与表侄女[[萨克森-魏玛-艾森纳赫|萨克森-魏玛-艾森纳赫]]大公[[卡尔·弗里德里希_(萨克森-魏玛-艾森纳赫)|卡尔·弗里德里希]]的二女儿[[奥古斯塔_(萨克森-魏玛)|玛丽·路易丝·'''奥古斯塔'''·卡特琳]]公主（Marie Luise ''Auguste'' Katharine，1811年9月30日—1890年1月7日）结婚，有一子一女：
* 子[[腓特烈三世_(德国)|'''腓特烈'''·威廉·尼库劳斯·卡尔]]（Friedrich Wilhelm Nikolaus Karl，1831年10月18日—1888年6月15日），1888年任德国皇帝兼普鲁士国王，称腓特烈三世。
* 女[[路易丝公主_(德国)|路易丝·玛丽·伊莉莎白]]（Luise Marie Elisabeth，1838年12月3日—1923年4月23日），1856年与[[巴登统治者列表|巴登大公]][[弗里德里希一世_(巴登)|弗里德里希一世]]结婚，有二子一女。
