'''威尔士'''的行政區劃分两個層級，分別是'''主要地區'''（principal area）和'''社區'''（community）。

[[File:Wales_Administrative_Map_2009.png|威爾斯現時行政區劃]]

==主要地區==

主要地區為單一層級（single-tier）的行政區，似[[單一管理區|單一管理區]]，政府有原先縣和區2級行政區的權限。

該層級的行政區共有22個行政區，依種類可分'''縣'''（county）、'''都市縣'''（city and county）、'''縣級市'''（city）和'''縣級鎮'''（county borough）。

==行政區劃歷史==
[[Image:Wales_Administration_Map_1947.png|1889至1974年威爾斯行政區劃]]
===1889至1974年===
基於'''1888年地方政府法案'''（Local Government Act 1888），自1889年到1974年3月31日，威爾斯的縣級行政區被分成'''行政縣'''（administrative county）和'''縣級鎮'''（county borough）2類，共有17個縣級行政區。而行政縣下1級行政區為'''鎮'''（borough）、'''市區'''（urban district）和'''鄉區'''（rural district）。

共有13個。分別是：
# 安格爾西（[[:en:Anglesey|Anglesey]]）
# 布雷克諾（[[:en:Brecknockshire|Brecknockshire]]）
# 錫爾迪金（[[:en:Cardiganshire|Cardiganshire]]）
# 卡馬森（[[:en:Carmarthenshire|Carmarthenshire]]）
# 卡那封（[[:en:Caernarfonshire|Caernarfonshire]]；原名Carnarvonshire，1926年7月1日改名）
# 登比（[[:en:Denbighshire|Denbighshire]]）
# 弗林特（[[:en:Flintshire|Flintshire]]）

# 蒙茅斯（[[:en:Monmouthshire|Monmouthshire]]；直至1974年，其被當作威爾斯屬縣的地位都是模糊不明的）
# [[蒙哥馬利#英國|蒙哥馬利]]（[[:en:Montgomeryshire|Montgomeryshire]]）
# 彭布羅克（[[:en:Pembrokeshire|Pembrokeshire]]）
# 拉德诺（[[:en:Radnorshire|Radnorshire]]）

====縣級鎮====
共有'''4'''個。分別是：
# [[卡迪夫|卡迪夫]]（[[:en:Cardiff|Cardiff]]，1889年設）
# [[斯旺西|斯旺西]]（[[:en:Swansea|Swansea]]，1889年設）
# [[紐波特|紐波特]]（[[:en:Newport|Newport]]，1891年自Monmouthshire分出升格）
# 梅瑟蒂德菲爾（[[:en:Merthyr_Tydfil|Merthyr Tydfil]]，1908年自Glamorgan分出升格）

===1974至1996年===
基於'''1972年地方政府法案'''（Local Government Act 1972），自1974年4月1日到1996年3月31日，威尔士原先的縣級鎮被撤併，且縣級行政區重新組建成新的'''縣'''（county）。

===1996年至今===

基於'''1994年（威爾斯）地方政府法案'''（ Local Government (Wales) Act 1994），自1996年4月1日至今，威尔士原先的縣和區2級區劃被合併，行政區重新組建成新的'''主要地區'''，為單一層級的行政區，政府有原先縣和區2級行政區的權限。此行政區劃一直維持到現今。

主要地區依種類可分為'''縣'''（county）、'''都市縣'''（city and county）、'''縣級市'''（city）和'''縣級鎮'''（county borough），共有22個行政區。

2002年，[[紐波特|紐波特]]（Newport）被授予[[城市地位|城市地位]]，自縣級鎮改制縣級市。
