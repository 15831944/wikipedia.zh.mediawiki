'''威靈頓鳳凰足球俱樂部'''（[[英文|英文]]：'''Wellington Phoenix FC'''）是[[紐西蘭|紐西蘭]][[威靈頓|威靈頓]]的一支職業[[足球|足球]]隊。該隊於2007年加盟[[澳洲甲組足球聯賽|澳洲甲組足球聯賽]]（A-League），取代來自[[奧克蘭|奧克蘭]]的[[新西蘭騎士隊|新西蘭騎士隊]]。

== 歷史 ==

新西蘭騎士隊在A-League作賽的兩個球季中表現差劣，而澳洲足球協會亦於2006-2007年球季尾段決定褫奪騎士隊在A-League中的參賽資格。騎士隊在球季餘下的五星期内由紐西蘭足球協會管理，而騎士隊在2007年1月21日與[[珀斯光榮隊|珀斯光榮隊]]在球季最後一場比賽中交鋒後，亦正式解散。

澳洲足協之後給予紐西蘭足協一個臨時A-League牌照，讓紐西蘭足協在國内物色一支球隊在2007-2008年球季加盟A-League。另一方面，一個來自[[昆士蘭州|昆士蘭州]][[湯斯維爾|湯斯維爾]]，名為「澳洲熱帶足球」（Tropical Football Australia）的財團亦表示有意取代騎士隊在A-League中的席位，並已開始準備申請手續。<ref>[http://tvnz.co.nz/view/page/410965/1023055 Shock! A-League decision delayed]，[[紐西蘭電視台|紐西蘭電視台]]，2007年3月14日</ref>惟澳洲足協已表明希望聯賽中繼續有一支來自紐西蘭的球隊，而澳洲熱帶足球亦放棄角逐騎士隊的席位，改為在次年向A-League申請成立一支擴展球隊。

在申請期截止之際，惠靈頓一名地產商人Terry Serepisos宣佈提供新球隊所需之申請費用，而威靈頓的新球隊亦於2007年3月19日被確定加盟A-League。<ref>{{cite web

== 主場 ==
[[File:Westpac_Trust_stadium_viewed_from_Wadestown.jpg|thumb]]

鳳凰隊的主場是位於[[威靈頓|威靈頓]]的[[西太平洋銀行體育場|西太平洋銀行體育場]]，可容納34500名觀衆，加上臨時座位則可以容納約40000人。球場於1998年動工，2000年開幕，離威靈頓市中心以北約1公里，臨近市内主要交通設施（如威靈頓火車站）。

== 洲際賽事資格爭議 ==
鳳凰隊無法參加[[亞足聯冠軍聯賽|亞足聯冠軍聯賽]]及[[大洋洲聯賽冠軍盃|大洋洲聯賽冠軍盃]]。

[[亞洲足球協會|亞洲足球協會]]因鳳凰隊是來自[[新西蘭|新西蘭]]，而不確認鳳凰隊為亞足協認可澳洲隊伍；[[大洋洲足球協會|大洋洲足球協會]]卻因鳳凰隊球員皆註冊於[[澳大利亞足球協會|澳大利亞足球協會]]（亞足協成員），而認定鳳凰隊無資格參與大洋洲聯賽冠軍盃。
