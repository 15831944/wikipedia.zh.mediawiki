'''維根'''（{{lang-en|Wigan}}），[[英國|英國]][[英格蘭|英格蘭]][[英格蘭西北|西北]]區域[[大曼徹斯特|大曼徹斯特]]郡的鎮，曾歸入[[蘭開夏|蘭開夏]]郡。據2001年人口統計，威根人口有81,203人，屬小鎮式規模。

== 歷史 ==
威根早在十二世紀已有人居住。18世紀正值[[工業革命|工業革命]]年代，威根乃英國的工業城鎮，其中第一家亨氏茄汁黃豆罐頭工廠正在於此，同時也有大量紡織廠在此設立。二次大戰後隨著英國製造業式微，威根地位逐漸下降。

== 地理 ==
由於位處[[英格蘭|英格蘭]]中部，威根地理上與西面的曼徹斯特、[[利物浦|利物浦]]等等相當接近。不少火車、巴士線都連接至曼徹斯特，同時還有運河連接到曼徹斯特。事實上按行政規劃，目前的威根是由曼徹斯特市管轄。

== 體育 ==
[[File:JJB_Wigan.jpg|thumb]]及欖球隊[[威根勇士|威根勇士]]共用[[JJB球場|JJB球場]]。]]
作為小鎮的威根，體育運動卻十分有名。欖球隊[[威根勇士|威根勇士]]是一支相當成功的欖球勁旅，而足球隊[[威根竞技足球俱乐部|威根竞技]]則至2005年升班至[[英格蘭超級聯賽|英格蘭超級聯賽]]，開始為全世界所知，兩隊共用[[JJB球場|JJB球場]]。
