'''威廉·法蘭西斯·哈格蒂四世'''<ref name=cankao/>（{{lang|en|William Francis Hagerty IV }}，{{bd|1959年|8月||}}）[[美国|美国]]商人、美国驻日本大使。

==生平==
哈格蒂曾作为共和党[[乔治·赫伯特·沃克·布什|老布什]]政府的职员，在通商、财政、防卫、信息等领域为[[白宫|白宫]]提供意见。他还和[[2012年美国大选|2012年美国大选]]共和党候选人[[威拉德·米特·罗姆尼|罗姆尼]]关系密切。他曾在[[日本|日本]][[东京|东京]]工作3年。<ref name=hqw/>

2016年7月，哈格蒂加入特朗普的竞选团队，主要负责特朗普过渡事务的策划及执行，后主管特朗普政府班子的人事提名事务。2017年1月5日，美国当选总统[[唐纳德·特朗普|唐纳德·特朗普]]提名出身商界的哈格蒂任美国驻日本大使。分析人士认为，该提名反映了特朗普计划将经济问题放在美日关系中的首要位置。<ref name=cankao>{{Cite web |url = http://www.cankaoxiaoxi.com/ym/20170109/1591841.shtml |title =  1月9日译名发布： 威廉·哈格蒂|author =  |date =2017-01-09  |publisher =[[参考消息|参考消息]]网  |language =  |accessdate =  |quote =  }}</ref><ref name=hqw>{{Cite web |url = http://world.huanqiu.com/exclusive/2017-01/9907066.html |title = 特朗普将提名威廉·哈格蒂出任新驻日大使 |author =  |date = 2017-01-05 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref>
