'''威廉·菲爾希納'''（{{lang-de|Wilhelm Filchner}}）（{{bd|1877年|9月13日|1957年|5月7日}}）是一位[[德國|德國]][[探險家|探險家]]。

== 早期探險生涯 ==
菲爾希納出生於[[德意志帝國|德國]][[拜羅伊特|拜羅伊特]]，在[[慕尼黑|慕尼黑]]成長，15歳時加入[[巴伐利亞王國|巴伐利亞王國]]陸軍。21歲時，菲爾希納第一次參與探險，目的地是[[俄國|俄國]]。兩年後，他獨自一人騎著馬匹穿越[[帕米爾高原|帕米爾高原]]，途經[[奧什|奧什]]、[[穆爾加布|穆爾加布]]、[[瓦罕走廊|瓦罕走廊]]和[[塔什庫爾干|塔什庫爾干]]，隨後返國。1903年至1905年，他率領探險隊前往[[西藏|西藏]]。

== 南極洲探險 ==
從西藏歸國後，菲爾希納被賦與探查[[南極洲|南極洲]]的任務，並前往[[斯匹次卑爾根島|斯匹次卑爾根島]]進行極地探險訓練。1911年5月4日，他與其他隊員們搭乘「德國」號前往[[南極|南極]]。此次探險進入[[威德爾海|威德爾海]]，並發現[[盧依波德海岸|盧依波德海岸]]與[[龍尼-菲爾希納冰架|龍尼-菲爾希納冰架]]。他們比80年前[[詹姆斯·威德爾|詹姆斯·威德爾]]所探查的海域還要更加深入許多。<ref name="howgego">

探險隊原本打算在[[冰架|冰架]]上建立基地度過嚴冬，但因冰層裂解而失敗，所以他們被困在滿是浮冰的海面上，隨波逐流。1912年9月，「德國」號從威德爾海脫困。在受困期間，菲爾希納嘗試找尋由[[班哲明·莫雷爾|班哲明·莫雷爾]]於1823年記錄的[[新南格陵蘭島|新南格陵蘭島]]，但一無所獲。因此他認為莫雷爾是看到了[[蜃景|蜃景]]。<ref>{{Cite web|title= Wilhelm Filchner 1877–1957|url=http://www.south-pole.com/p0000103.htm|publisher=South-pole.com|accessdate=December 20, 2008}}</ref>

== 晚年 ==
歸國後，菲爾希納並沒有再前往南極探險，而多是參與[[亞洲|亞洲]]的探查活動。1926年到1928年，自費前往[[青海湖|青海湖]]探險。1934年到1937年，率領德國政府出資的探險隊前往西藏。1939年，尼泊爾地理調查。

1937年，[[希特勒|希特勒]]授與菲爾希納[[德國國家藝術與科學獎|德國國家藝術與科學獎]]，以表彰他在探險活動的貢獻。

[[二戰|二戰]]期間，他在[[印度|印度]]被拘留。直到1940年9月13日，他都被拘留於[[巴特那|巴特那]]當地一所簡陋醫院。1941年，被關在普拉達的假釋營中。1941年9月到1946年11月，被關在[[薩塔拉|薩塔拉]]的假釋營中。戰爭結束後，菲爾希納定居於印度[[浦那|浦那]]，之後返回歐洲居住。

1957年，菲爾希納於[[瑞士|瑞士]][[蘇黎世|蘇黎世]]過世，享年79歲。
