'''威海市'''是[[中华人民共和国|中华人民共和国]][[山东省|山东省]][[地级市|地级市]]，地处[[山东半岛|山东半岛]]最东端，北东南三面濒临[[黄海|黄海]]，属温带季风气候，是港口及旅游城市，2017年人口普查数为282.56万人。
<br />

== 历史 ==
{{main|威海市歷史}}
威海历史悠久，东夷人最早生活的地区，[[新石器时代|新石器时代]]中期，境内就有人类聚居。[[夏朝|夏]]、[[商朝|商]]、[[周朝|周]]三代，今市境为[[东夷|东夷]]之地，属[[青州_(九州)|青州]]，有莱夷聚居。[[西漢|西漢]]置[[不夜縣|不夜縣]]，属[[東萊郡|東萊郡]]。[[北齐|北齐]][[天统_(北齐)|天统]]四年（568年），境内设[[文登县_(北齐)|文登县]]，今市境尽属之。此后，朝代更替，但隶属文登县未变。

[[明朝|明]][[洪武|洪武]]三十一年（1398年），为防[[倭寇|倭寇]]侵扰，设'''威海卫'''，取“威震海疆”之意，“威海”即由此而来。1898年，[[英租威海卫|英租威海卫]]，并于1900年设威海卫行政长官署。1930年10月，中国收回除刘公岛外的威海卫，置威海卫行政区，直属国民政府行政院。刘公岛至1940年由英国退还中华民国。[[威海卫旗帜|威海卫旗帜]]（图一）是威海卫在1903年至1930年期间作为英属殖民地的重要标志。[[File:British_Weihaiwei_flag.svg|thumb]]威海市是中国第一支现代海军的摇篮，也是1894－1895年[[甲午战争|甲午战争]]时[[北洋海军|北洋海军]]的一个重要军事基地。1895年，日军从[[荣成市|荣成]]登陆，攻占威海卫。1898年（[[光绪|光绪]]二十四年）7月1日，[[滿清|滿清]]政府与英国签订了《[[订租威海卫专条|订租威海卫专条]]》。英国强租32年后，在1930年归还中国，設為省級威海衛行政區，準用[[直轄市|直轄市]]法規。抗戰時期[[汪精衛政權|汪精衛政權]]之[[华北政务委员会|華北政務委員會]]將其改為縣級特別區，由[[烟台市|烟台]]管辖。抗戰勝利後由中共接收，改稱威海衛市，隸於山東省膠東行政公署治下。

[[中華人民共和國|中華人民共和國]]成立後，原隶属于[[烟台|烟台]]地区下属的县级市，1950年5月，成立文登专区，管辖威海、[[荣成|荣成]]、[[文登县_(北齐)|文登]]、[[昆嵛镇|昆嵛]]、[[乳山|乳山]]、[[海阳县_(清朝)|海阳]]、[[牟平|牟平]]、[[福山|福山]]等8县。1956年2月，文登专区撤销，今威海地域改属[[莱阳专区|莱阳专区]]。1958年10月，莱阳专区改为烟台专区，1967年2月，烟台专区更名为烟台地区，今威海地域属之。

1983年8月撤销[[烟台地区|烟台地区]]，设烟台市（[[地级市|地级]]），同时威海市改为省辖[[县级市|县级市]]，由烟台市代管，文登、[[荣成|荣成]]、[[乳山|乳山]]3县归烟台市管辖。

1987年6月15日，威海市升为[[地级市|地级市]]，原县级威海市设为[[环翠区|环翠区]]。烟台市的荣成、文登、乳山3县划归威海市管辖。1988年10月24日，文登县改为省辖县级市，由威海市代管；11月，荣成县改为省辖县级市，由威海市代管。1993年7月，乳山县改为省辖县级市，由威海市代管。

2014年初，文登市撤市划区，成为威海市区级行政单位。3月18日，山东省威海市文登区正式挂牌成立。按照国务院《批复》（国函〔2014〕13号）和省政府《通知》（鲁政字〔2014〕48号）精神，威海市此次部分区划调整的主要内容是撤销文登市设立威海市文登区，以原文登市的行政区域（不含汪疃镇、 蔄山镇）为文登区的行政区域。同时，将原文登市的汪疃镇、蔄山镇从行政区域上划归威海市环翠区，仍由临港区管辖。<ref name=新华网山东频道>{{cite|title=山东省威海市调整部分行政区划 文登撤市设区|url=http://www.sd.xinhuanet.com/news/2014-03/19/c_119831818.htm|newspaper=新华网山东频道|date=19 March 2014|deadurl=yes|archiveurl=https://web.archive.org/web/20140421064033/http://www.sd.xinhuanet.com/news/2014-03/19/c_119831818.htm|archivedate=2014年4月21日}}</ref>

== 地理 ==
威海市地处北纬36°41′ - 37°35′、东经121°11′ - 122°42′，作为[[山东半岛|山东半岛]]的最东端三面环海，北东南三面濒临[[黄海|黄海]]，北与[[辽东半岛|辽东半岛]]相对，东及东南与[[朝鲜半岛|朝鲜半岛]]和[[琉球群岛|琉球群岛]]隔海相望，西与[[烟台|烟台]]市接壤。东西最大横距 135公里，南北最大纵距 81公里，总面积 5698平方公里，其中市区面积 769平方公里。其中，低山占土地总面积的 15.77%，丘陵占  52.38% ，平原占 27.56%，岛屿占 0.28%，滩涂占 4.01%。海岸线长 985.9公里，<ref name=地理位置>{{cite|title=威海概况-地理位置|url=http://www.weihai.gov.cn/col/col17049/index.html|publisher=威海市人民政府|date=2017-03-18|deadurl=no|archiveurl=https://web.archive.org/web/20170201120244/http://www.weihai.gov.cn/col/col17049/index.html|archivedate=2017-02-01}}</ref>海岸类型属于港湾海岸，海岸线曲折，岬湾交错，多港湾、岛屿。

=== 气候===
威海市属北[[温带季风气候|温带季风型大陆性气候]]，四季变化及季风进退都很明显，气温变化和缓，降水较为充足。因受海洋调节作用，表现出春冷、夏凉、秋暧、冬温，昼夜温差小、无霜期长、大风多和湿度大等[[海洋性气候|海洋性气候]]特点<ref>{{Cite web|url=http://sd.weather.com.cn/sdqh/gsqhtd/11/86241.shtml|title=威海市气候特点|accessdate=2019年1月22日|author=|date=|publisher=中国天气网山东站}}</ref>。年平均气温约12.2℃，气温通常在-4°C至29°C之间变化，极少低于-8°C或高于33°C。一月最冷，平均气温−1.4℃；八月最热，平均气温25.1℃。年均降水量687mm，主要集中在夏季，尤其是7月和8月。年均日照时数2538小时<ref>{{Cite web|url=https://zh.climate-data.org/亚洲/中国/山东/威海-2584/|title=气候 威海|accessdate=2019年1月23日|author=|date=|publisher=}}</ref>。

3-5月是春季，较同纬度的内陆相比气温低2-3°C且回升慢，推迟约半个月，大风多，降水少，大约占全年总降水量的14%，易发春旱。

6-8月为夏季，受海洋影响，温度比同纬度内陆相比低1-4°C。由于夏季东南季风的关系，降水多，湿度大，降水占全年59%。

9-11月是秋季，温度与同纬度内陆相比偏高1-2°C。秋季总降水量占全年的23%，集中在9月份。10月份起，多秋高气爽的晴天。11月中旬起，冷空气开始加强，气温迅速下降。

12-2月为冬季，受蒙古高气压影响，西北季风致使气候干燥寒冷，冬季温度比同纬度内陆相比偏高约2°C，且昼夜温差小。12月下旬起，受海洋影响，阴天和阵雪天气增多，雨雪量比内陆同纬度地区多三倍以上<ref>{{Cite web|url=https://zh.weatherspark.com/y/138514/中国、威海的全年平均天气|title=威海的平均天气|accessdate=2019年1月23日|author=|date=|publisher=Weather Spark}}</ref>。

* 市辖区：[[环翠区|环翠区]]、[[文登区|文登区]]

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为280.48万人<ref>{{cite book |author=威海市统计局|title=《威海市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加了20.78万人。增长8.00%，年平均增长0.77%。其中，男性为141.32万人，占总人口的50.39%；女性为139.16万人，占总人口的49.61%，总人口性别比（以女性为100）为101.55。0－14岁的人口为28.28万人，占10.08%；15－64岁的人口为218.36万人，占77.85%；65岁及以上的人口为33.84万人，占12.07%。

威海市[[少数民族|少数民族]]常住人口24433人。对近年公安户政部门的人口数据进行分析比对，实有少数民族暂住人口约2万人，共计有少数民族人口约4.5万人，占全市总人口的1.6%。

== 旅游 ==
[[File:Weihai2.jpg|thumb]]
[[File:Weihai_City.jpg|thumb]]
威海旅游资源丰富，有海岛海岸、城市园林、历史遗蹟、民俗风情等10多种类型。拥有国家级[[5A级风景名胜区|5A级风景名胜区]]1处，[[4A级风景名胜区|4A级风景名胜区]]8处，[[3A级风景名胜区|3A级风景名胜区]]6处，[[2A级风景名胜区|2A级风景名胜区]]1处；省级旅游度假区4处，工农业旅游示范点18处，游览景区（点）80多处。境内近千公里的海岸线上，有中国近代第一支海军的诞生地[[刘公岛|刘公岛]]、[[秦始皇|秦始皇]]东巡过的“东方好望角”天尽头、中国道教全真派发祥地圣经山、凝聚中日韩三国人民友谊的赤山法华院、亚洲最大的天鹅栖息地天鹅湖、大东胜境铁槎山、天下第一滩银滩、母爱圣地大乳山等众多自然、人文景观。

== 文化 ==
*威海是中国[[高尔夫|高尔夫]]运动的发源地，中国第一块高尔夫球场于1902年诞生于刘公岛。
*中国[[现代足球|现代足球]]运动起源地，经由[[英国|英国]]殖民者传入威海。
*威海拥有中国最好的自然环境，1990年威海被授予[[国家级卫生城市|国家级卫生城市]]，是中国第一个获此殊荣的城市。
*威海是全国著名的[[温泉之乡|温泉之乡]]，自古泡汤文化盛行，山东省内18座天然泉眼威海拥有其中的9座。
*中国第一支近代海军诞生在威海的[[刘公岛|刘公岛]]。
*[[昆嵛山|昆嵛山]]是[[道教|道教]][[全真派|全真派]]的发祥地。
*成山头被认为是古时[[日神|日神]]所居之地，[[姜子牙|姜子牙]]，[[秦始皇|秦始皇]]，[[汉武帝|汉武帝]]，[[华国锋|华国锋]]，[[胡耀邦|胡耀邦]]等众多古今中外名人曾到此观光朝拜。
*这里是中国沿海距[[韩国|韩国]]最近的城市，1990年，威海率先开通了中国至韩国的第一条海上航线。

== 荣誉 ==
威海市被评为中国投资硬环境40优城市以及全国综合经济实力50强城市。1984年成为第一批中国[[沿海开放城市|沿海开放城市]]。1996年由建设部评为[[国家园林城市|国家园林城市]]。1997年10月13日被国家环保部授予[[国家环境保护模范城市|国家环境保护模范城市]]。1999年1月成为第一批[[中国优秀旅游城市|中国优秀旅游城市]]。2003年10月6日成为国内第十一个获得[[联合国人居奖|联合国人居奖]]的中国城市（第一個為1990年，唐山市人民政府的震后重建项目）。2009年5月7日被评选为[[国家森林城市|国家森林城市]]。2015年2月28日成为第四届[[全国文明城市|全国文明城市]]。
