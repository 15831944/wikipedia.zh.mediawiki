'''横断山脉'''（英文：Hengduan Mountain Range）是[[中国|中国]]西南部的著名山脉，有'''广义'''和'''狭义'''的区分。广义的横断山脉位于[[青藏高原|青藏高原]]东南部（介于北纬22°～32°05′，东经97°～103°之間），为[[四川省|四川省]]西部、[[云南省|云南省]]西北部和[[西藏自治区|西藏自治区]]东部南北向山脉的总称，是青藏高原的边缘山系。它东起[[邛崃山|邛崃山]]，西抵[[伯舒拉岭|伯舒拉岭]]-[[高黎贡山|高黎贡山]]，北达[[昌都|昌都]]、[[甘孜|甘孜]]至[[马尔康|马尔康]]一线，南抵中[[緬甸|缅]]边境的山区，面积60余万平方公里，是中国最长、最宽和最典型的南北向山系。狭义的横断山脉指三江并流地区的四条山脉，即[[沙鲁里山|沙鲁里山]]、[[芒康山|芒康山]]-[[云岭|云岭]]、[[他念他翁山|他念他翁山]]-[[怒山|怒山]]及伯舒拉岭-高黎贡山。

== 名称 ==
因其山高谷深，山川并列，横断了东西向的山地和交通孔道，所以称为“横断山脉”。

== 组成 ==
横断山区并列的山川，自东至西依次是[[邛崃山|邛崃山]]、[[大渡河|大渡河]]、[[大雪山|大雪山]]、[[雅砻江|雅砻江]]、[[沙鲁里山|沙鲁里山]]、[[金沙江|金沙江]]、[[芒康山|芒康山]]（[[宁静山|宁静山]]）-[[云岭|云岭]]、[[澜沧江|澜沧江]]、[[他念他翁山|他念他翁山]]-[[怒山|怒山]]、[[怒江|怒江]]和[[伯舒拉岭|伯舒拉岭]]-[[高黎贡山|高黎贡山]]等。其间还有一些小的支脉，如邛崃山和大雪山之间还有[[夹金山|夹金山]]，大雪山向南延伸成[[锦屏山|锦屏山]]，云岭有支脉[[玉龙山|玉龙山]]（[[玉龙雪山|玉龙雪山]]）等等。

== 地质地貌 ==
横断山脉由[[喜马拉雅运动|喜马拉雅运动]]时期[[歐亞板块|歐亞板块]]与[[印澳板块|印澳板块]]碰撞，形成[[褶皱|褶皱]]山脉，并形成一系列[[断陷盆地|断陷盆地]]。大雪山主峰[[贡嘎山|贡嘎山]]海拔7556米，为横断山脉最高峰。[[金沙江|金沙江]]、[[澜沧江|澜沧江]]和[[怒江|怒江]]，两岸陡峻，江面狭窄，属典型的“V”字型[[深切峡|深切峡]]谷。山崩、滑坡、泥石流和地震频繁，是中国主要[[地震带|地震带]]之一。

== 资源概况 ==
横断山脉是中国重要的[[有色金属|有色金属]]矿产地。金沙江、澜沧江和怒江蕴藏多达百种有色金属矿。[[攀枝花|攀枝花]]地区是中国[[铁矿|铁矿]]储量很大的地区之一。

横断山脉是中国主要水能资源分布区。如金沙江以枯水位计算，干流落差达3000余米，包括支流在内，水能蕴藏量近1亿千瓦。
