'''横峰县'''位于[[中国|中国]][[江西省|江西省]]东北部，隶属于[[上饶市|上饶市]]。

== 历史沿革 ==
[[明|明]][[嘉靖|嘉靖]]39年（公元1560年）由[[弋阳|弋阳]]析横峰寨置'''兴安县'''。因与广西兴安县同名，故于1914年改为横峰县，因县北横峰山得名。

==行政区划==
现辖2镇6乡1办事处：
*[[镇|镇]]：[[岑阳镇|岑阳镇]]、[[葛源镇|葛源镇]] 
*[[乡|乡]]：龙门畈乡、青板乡、司铺乡、莲荷乡、姚家乡、港边乡
*办事处：红桥垦殖场

== 地理 ==
北部为[[怀玉山|怀玉山]]区，中部和南部为丘陵，有“七山半水二分田，半分道路与庄园”之称。

== 交通 ==
横峰县距上饶市35公里，距南昌210公里，距上海624公里。
*铁路：[[浙赣铁路|浙赣铁路]]、[[橫南铁路|橫南铁路]]
*公路：[[320国道|320国道]]、[[沪昆高速公路|沪昆高速公路]]

== 经济 ==
现有农业、锁业、纺器业、铜材加工业、服装加工业等五大支柱产业，建有兴安工业园。

2001年被列为[[国家扶贫重点县|国家扶贫重点县]]。

== 自然资源 ==
[[稀有金属|稀有金属]][[钽|钽]][[铌|铌]]储量位居“亚洲第一”。
原煤储量约4000万吨。

== 风景名胜 ==
*[[仙垄山|仙垄山]]，位于城东，为[[丹霞地貌|丹霞地貌]]景观
*[[葛源镇|葛源镇]]，[[中国历史文化名镇|中国历史文化名镇]]，原[[赣东北革命根据地|赣东北革命根据地]]中心
