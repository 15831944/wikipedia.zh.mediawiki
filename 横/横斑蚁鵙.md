'''横斑蚁鵙'''（[[学名|学名]]：{{lang|la|''Thamnophilus doliatus''}}）是[[蚁鵙科|蚁鵙科]][[蚁鵙属|蚁鵙属]]的一种。生活于[[新热带界|新热带界]]中，可在[[墨西哥|墨西哥]]的[[塔毛利帕斯州|塔毛利帕斯州]]、整个[[中美洲|中美洲]]、[[特立尼达和多巴哥|特立尼达和多巴哥]]、[[南美洲|南美洲]][[安地斯山脉|安地斯山脉]]以东大部分地区，[[阿根廷|阿根廷]]北部、[[玻利维亚|玻利维亚]]和[[巴拉圭|巴拉圭]]发现。另外，在美国[[得克萨斯州|得克萨斯州]]南部亦有观察报告。<ref>{{cite web |url=http://www.texasbirds.org/tbrc/ar2007.html |title=存档副本 |accessdate=2009-08-21 |0=2009-12-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20091208133322/http://texasbirds.org/tbrc/ar2007.html |archivedate=2009-12-08 }}</ref>横斑蚁鵙的栖息地为干旱和湿润的[[疏林|疏林]]（包括[[花园|花园]]和[[公园|公园]]）。横斑蚁鵙分布广泛，是蚁鵙中最为常见的一种。

== 描述 ==
[[Image:Thamnophilus_doliatus_female.jpg|thumb]]
[[File:Thamnophilus_doliatus_-Panama-8.jpg|left]]
[[File:Thamnophilus_doliatus_-Goias,_Brazil-8b.jpg|left]]的未成年雌性个体]]
典型的横斑蚁鵙身长约16.5厘米，体重25克。雄性个体全身布满黑白交替的斑纹，并有着白底黑羽的羽冠（如图中所示）。雌性个体外表棕色，有着栗色的羽冠，头部两侧和颈部缀有黑色条纹，下半身为较深的米黄色。 

[[亚类|亚类]]的差异主要表现为体色的明暗、条纹的多少和下半身的色调。例如，多巴哥亚种（学名：{{lang|la|''T. d. tobagensis''}}）的雄性个体要比[[指名亚种|指名亚种]]的下身更白，而雌性则要比指名亚种更黑。最为独特的亚种是生活在[[巴西|巴西]]东北部的{{link-en|卡廷加|Caatinga}}亚种（学名：{{lang|la|''T. d. capistratus''}}），其雄性的羽冠为纯黑色（而不是在羽冠基部带有白色条纹），雌性喉部覆盖有条纹，腹部亦见依稀的条纹。此外，该亚种还是唯一有着枣红色（而非黄色）[[虹膜|虹膜]]的亚种。最近有意见认为，该亚种应被考虑为另一独立种群“卡廷加横斑蚁鵙”。
