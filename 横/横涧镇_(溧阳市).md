'''横涧镇'''，是{{PRC admin/fullname|32/04/81/104/000|3}}下辖的一个已撤销[[乡镇|乡镇]]级行政单位，位于溧阳市东南端。2007年并入[[戴埠镇|戴埠镇]]。
==历史==
清代在惠得區设橫澗鎮，中华民国初撤区建市、乡，横涧集镇属惠隨鄉（乡公所驻[[戴埠镇|戴埠]]，下设74图）。1929年改市乡制为区－乡二级制，建橫澗鄉，属戴埠區（1935年改称第六區），1947年改属丁山橋區（区署驻丁山桥）。

1949年建立橫澗鄉（属县辖戴埠區），1956年并入同官乡，次年改为横涧乡。1958年至1983年期间改称横涧[[人民公社|公社]]。

2000年初横涧撤乡建镇，2007年撤销横涧镇并入[[戴埠镇|戴埠镇]]<ref>{{cite web |title=省政府关于同意调整溧阳市部分镇行政区划的批复 |url=http://www.jiangsu.gov.cn/art/2007/3/17/art_46143_2543681.html |website=江苏省政府网 |publisher=江苏省人民政府 |accessdate=2018-08-20 |date=2007-03-17}}</ref>。

== 行政区划 ==
撤销前横涧镇下辖以下地区：

横涧社区、松岭村、河洛村、南渚村、深溪岕村、横涧村、黄岗岭村、金山里村、同官村、徐家玕村和李家园村。
