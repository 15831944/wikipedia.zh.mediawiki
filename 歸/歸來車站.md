'''歸來車站'''位於[[屏東縣|屏東縣]][[屏東市|屏東市]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[屏東線|屏東線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]兩座。
*配合[[高雄潮州鐵路捷運化計畫|高雄潮州鐵路捷運化計畫]]，本站已改建為高架車站。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，僅停靠[[臺鐵區間車|區間車]]
*於2013年6月25日切換為高架路段。

== 歷史 ==
*「'''歸來'''」地名的由來，據記載於[[清朝|清朝]]時稱為「傀儡蕃」，據《台灣雜記》所陳述，「傀儡蕃內深林茂竹，行數日不見日色，路徑錯雜，傀儡蕃常伏於此，截人取頭而去」（「傀儡蕃」，台語「「傀儡仔」指[[原住民|原住民]]也），可見此地昔日是危險的蕃地，經常有人被「[[出草|出草]]」。恰好「{{lang|ja|カイライ}}」（傀儡）的日文發音與「{{lang|ja|キライ}}」（歸來）相似，故改稱「歸來」，也有希望出外人都能平安歸來之意。
*1956年5月1日：設簡易站，並指定由[[屏東車站|屏東站]]管理。
*1982年8月1日：由簡易站降為招呼站。
*2013年6月25日：高架路線通車，高架新車站營運。
*2015年8月23日：高架雙軌電氣化通車。
*2015年10月15日：啟用多卡通刷卡機。
