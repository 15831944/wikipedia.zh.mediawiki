'''风云二号H气象卫星'''（{{lang-en|FY-2H}}），又名'''风云二号H星'''或'''风云二号09星'''，是一颗中国的气象卫星。2017年11月15日02时35分在[[西昌卫星发射中心|西昌卫星发射中心]]以[[长征三号甲运载火箭|长征三号甲运载火箭]]发射并随后成功入轨<ref>{{Cite web |url=http://www.xinhuanet.com/politics/2018-06/05/c_1122941929.htm |title=风云二号H星成功发射 服务“一带一路”沿线国家 |accessdate=2018-06-06 |date=2018-06-05}}</ref>。

== 简介 ==
风云二号H星是风云二号第03批业务卫星，也是风云二号系列最后一颗卫星<ref name="xinhua_five_questions">{{Cite web |url=http://www.xinhuanet.com/video/sjxw/2018-06/06/c_129887022.htm |title=五问风云二号H星 |accessdate=2018-06-06 |author=新华网 |date=2018-06-06}}</ref>。相对于G星，H星共有15项技术状态得到改进提升，设计寿命4年<ref name="xinhua_five_questions" />。

卫星发射后，在东经86.5°进行在轨测试，但正式投入使用后会定在东经79°，目的是服务更多“[[一带一路|一带一路]]”沿线国家<ref>{{Cite web |url=http://tech.ifeng.com/a/20180606/45015045_0.shtml |title=刚刚，风云二号H星成功发射！年近九旬的总设计师亲临一线 |accessdate=2018-06-06 |date=2018-06-05}}</ref>。
