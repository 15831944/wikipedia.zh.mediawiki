'''风云三号D气象卫星'''，又名'''风云三号D星'''或'''风云三号04星'''，是一颗中国的气象卫星。2017年11月15日02时35分由[[长征四号丙运载火箭|长征四号丙运载火箭]]在[[太原卫星发射中心|太原卫星发射中心]]发射并随后成功入轨<ref>{{Cite web |url=http://scitech.people.com.cn/n1/2017/1115/c1007-29646312.html |title=风云三号04星成功发射:天气预报将更精准 |accessdate=2018-06-02 |date=2017-11-15}}</ref>。高风云三号D星是中国第二代极轨气象卫星，也是首颗设置红边谱段的光学成像卫星。

风云三号D星将接替八年前开始工作的[[风云三号B星|风云三号B星]]的观测任务<ref name="cma">{{Cite web |url=http://www.cma.gov.cn/2011xwzx/2011xqxxw/2011xqxyw/201711/t20171115_456065.html |title=“风云三号”D星成功发射 可更早获知未来天气状况 服务生态文明建设和“一带一路” |accessdate=2018-06-02 |date=2017-11-15}}</ref>。

== 仪器设备 ==
风云三号D星共装载了10台遥感探测仪器，分别为电离层光度计（IPM）、微波温度计Ⅱ型（MWTS-Ⅱ）、中分辨率光谱成像仪Ⅱ型（MERSI-Ⅱ）、红外高光谱大气成像仪（HIRAS）、广角极光成像仪（WAI-Ⅰ）、微波成像仪（MWRI）、微波湿度计Ⅱ型（MWHS-Ⅱ）、空间环境检测器（SEM）、GNSS掩星大气探测仪（GNOS）、高光谱温室气体监测仪（GAS）<ref name="cma" />。
