
'''風水師'''是具備[[風水|風水]]知識，受人或国家委託斷定風水好壞，必要時並予以修改的一種[[職業|職業]]。通常風水師也兼具[[卜卦|卜卦]]、[[看相|看相]]、[[擇日|擇日]]等学问，而某些[[道士|道士]]、[[巫師|巫師]]、[[廟祝|廟祝]]、[[泥水匠|泥水匠]]、[[中醫師|中醫師]]等亦可能偶以風水營生。專職的風水師亦稱'''堪輿師或地师'''，而具有传承和高明堪舆学问的真师被称为'''地仙'''，民間相信風水者常尊稱其為'''風水先生'''，有時則俗稱為'''陰陽先生'''。

专业的風水師一般具有[[室內設計|室內設計]]、[[建筑结构|建筑结构]]、[[环境科学|环境科学]]和[[天文学|天文学]]等的知識，因為不懂相关知识的風水師傅會被質疑其传承和水平。風水師的收費並無[[市價|市價]]，大部分风水师会参考服务内容的复杂性、以及委托者的地位而定价，有者甚至只收[[紅包|紅包]]一封。

== 工作 ==
現代的風水師主要工作內容有：
*選擇築宅或墓葬的良好地點－相地审气理。
*判斷建築設計是否符合風水原則。
*調整室內外的结构布局以營造更佳的風水環境。
*配合真太阳时选择黄道吉课办理事情。

==主要道具==
# 准确无误的罗盘（罗经）
# 细小的拉长线（方便拉出正确的旺点位置）
# 米袋、30寸高的椅子或罗盘专用架（摆放罗盘量度时使用）
# 立极尺（方便处理建筑图测的准确位置）
# 其他如各类笔，尺、拉尺、原子泥、针，甚至墨斗等都会视情况而使用

== 著名風水師 ==
古代著名风水师有：东晋的[[郭璞|郭璞]]，唐代的[[杨筠松|杨筠松]]，以及他的弟子[[曾文迪|曾文迪]]、[[刘江东|刘江东]]，[[司马头陀|司马头陀]]，[[廖金精|廖金精]]等人，宋代的[[吴景峦|吴景峦]]、[[赖布衣|赖布衣]]，明代的[[蒋大鸿|蒋大鸿]]，清代的[[章仲山|章仲山]]、[[沈竹礽|沈竹礽]]。

[[臺灣|臺灣]]清代也有三大風水名師，[[北臺灣|北臺灣]]的[[林琅|林琅]]、[[南臺灣|南臺灣]]的[[林半仙|林鎮]]，還有[[中臺灣|中臺灣]]的蝨母仙[[范金郎|-{范}-金郎]]。
