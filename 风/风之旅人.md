{{vgname|风之旅人|en=Journey|又译“'''旅途'''”；在中国大陆，为与开发公司之前的作品相对应，有时也译作“'''旅'''”或“'''行'''”}}，是一款[[冒险游戏|冒险]]类[[独立游戏|独立游戏]]，由[[thatgamecompany|thatgamecompany]]开发，[[索尼电脑娱乐|索尼电脑娱乐]]通过数字下载方式在[[PlayStation_Network|PlayStation Network]]发行，首发于[[PlayStation_3|PlayStation 3]]平台。游戏于2013年3月在香港、台湾、北美、欧洲和日本等地相继发售。本作由华人游戏制作人[[陈星汉|陈星汉]]负责制作，是第三款thatgamecompany与索尼电脑娱乐签约发行的游戏，同时也是两者所签合同中最后一款游戏。2013年8月，囊括了《风之旅人》、前两款thatgamecompany开发的游戏《[[流_(游戏)|流]]》（浮游世界）、《[[花_(游戏)|花]]》、外加3款游戏的额外内容的实体合集《风之旅人 典藏版》（{{lang|en|''Journey Collector's Edition''}}）发售。本作之后移植到[[PlayStation_4|PlayStation 4]]平台上，交由Tricky Pixels工作室负责游戏的移植工作，于2015年7月发售。<ref>{{cite web |url=http://www.gameinformer.com/b/news/archive/2015/04/18/journey-getting-physical-release-for-playstation-4-arrives-this-summer.aspx |author=Kyle Hilliard |title=Journey Getting Physical Release For PlayStation 4, Arrives This Summer |publisher=GameInforner |date=2015-04-18 |accessdate=2015-04-19 |deadurl=no |archiveurl=https://web.archive.org/web/20150419120009/http://www.gameinformer.com/b/news/archive/2015/04/18/journey-getting-physical-release-for-playstation-4-arrives-this-summer.aspx |archivedate=2015-04-19 }}</ref><ref>{{cite web |url=http://www.ign.com/articles/2015/07/23/journey-ps4-review |author=Marty Sliva |title=Journey PS4 Review |publisher=IGN |date=2015-07-23 |accessdate=2015-07-30 |deadurl=no |archiveurl=https://web.archive.org/web/20150729012107/http://www.ign.com/articles/2015/07/23/journey-ps4-review |archivedate=2015-07-29 }}</ref>2018年12月，游戏宣布将通过Epic Store登陆[[Windows|Windows]]平台，并交由{{le|Annapurna Interactive|Annapurna Interactive}}发行，发售日期待定。<ref>{{cite web |url=https://gematsu.com/2018/12/journey-coming-to-pc |title=Journey coming to PC |publisher=[[gematsu|gematsu]] |date=2018-12-07 |accessdate=2018-12-07}}</ref>

在《风之旅人》游戏过程中，玩家将控制一名穿着长袍的角色，在广袤的沙漠中向远方山峰的顶端前行。在途中，游戏者可以随机结识遇到另一名相同旅程的玩家；虽然两人可以在旅途中相互帮助，却无法像其他游戏那样通过语音或文字交流，并且无法看到对方的名字。玩家间的唯一交流途径是没有词句的吟唱，这种吟唱会对游戏世界产生影响，使玩家可以通-{}-过关卡继续游戏。开发团队期冀通过游戏，让玩家感到自己在世界中的渺小和孤独，继而与陌生人结为同伴，以此使玩家真正产生兴趣和其他玩家社交，建立情感纽带。<ref name="zhihu chen's answer">{{cite web|url=http://www.zhihu.com/question/20125406/answer/14067659 |author=陈星汉|title=陈星汉答：设计《旅程》（Journey）这款游戏时，内心中追求的主题是什么？ |publisher= 知乎 |accessdate=2012-12-21}}</ref>本作音乐由[[奥斯丁·温特里|奥斯丁·温特里]]作曲；本作的音乐会随玩家行动而产生相应变化。游戏通过音乐树立独立的主题，以此对应与阐释游戏故事所带来的情感。

《风之旅人》获得了业界的高度好评。游戏编辑和评论对游戏的视听艺术大加赞扬，同时还称赞和陌生人游玩时建立的陪伴感，能为玩家带来触动心弦的情感体验。游戏获[[IGN|IGN]]等多家游戏媒体授予的“2012年度游戏”荣誉。<ref name="Journey IGN GOTY">{{cite web |url=http://www.ign.com/videos/2012/12/21/igns-2012-game-of-the-year-video |title=Journey GOTY |publisher="[[IGN|IGN]]" |accessdate=2012-03-19 |deadurl=no |archiveurl=https://web.archive.org/web/20121223082232/http://www.ign.com/videos/2012/12/21/igns-2012-game-of-the-year-video |archivedate=2012-12-23 }}</ref>而游戏的音乐原声在[[第55屆葛萊美獎|第55屆葛萊美獎]]上获“最佳视觉媒体配乐专辑”提名。<ref>{{cite web |url=http://www.grammy.com/nominees |title=55th Annual GRAMMY Awards Nominees |deadurl=yes |archiveurl=https://www.webcitation.org/63bRXv6ZA?url=http://www.grammy.com/nominees |archivedate=2011-12-01 }}</ref>

== 游戏概况 ==

[[File:Journey-PS3-Screenshot.jpg|left]]

玩家在《风之旅人》中扮演一名身披长袍者，从一片广袤的沙漠里开始游戏旅途。在序章的简单介绍过后，玩家会看到所控角色站在沙丘上，很远处浮现出一座高峰，<ref name="GIreview" />本作的最终目的便是前往该山峰。这段旅途划分为了若干个不同区域的关卡，玩家只有通过它们方能到达目的地。游戏默认采用[[第三人称视角游戏|第三人称]]式追尾视角，玩家可操作[[DualShock_3|DualShock 3]]的[[模拟控制摇杆|类比摇杆]]，或倾斜手柄内置六轴平衡感应器，来改变镜头视角。<ref name="GSreview" />制作者刻意简化了游戏操作，因此跳跃、帮助同伴时的吟唱、发出简单交流用音符等只需一键完成。角色所吟唱的时间长短和音量仅与按键有关，而音符的曲调则要与背景音乐合拍。<ref name="IGNreview" />这些操作并没有单独的教学关卡，而是在游戏开始之初形象化呈现。在《风之旅人》中，除了制作人员名单和标题以外，很少使用文字或语音传递信息。 <ref name="GIreview" />

玩家所操控的角色全身裹着一件深红色的长袍，面戴黑面具，只能看到眼睛。角色头部后面飘着一条长围巾，在接触关卡中发光的符印后玩家的围巾会加长，同时围巾上面的符文也会增多，以延长玩家角色浮空和跳跃的时间和距离。浮空需要消耗围巾上代表能量的发光符文，之后符文光亮会慢慢自动恢复，也可由同伴吟唱音符来快速恢复，或在关卡中接触飘动的布条碎片来恢复。<ref name="EGreview" />游戏场景中还散布着更大的破旧长形布条，玩家可以在其附近吟唱，将其转化为与角色所穿的长袍相似的深红色印花布条。这也是游戏的解迷要素之一，转化这些大布条可以对游戏世界产生不同的影响，例如释放布条碎片，形成桥梁，或者使玩家漂浮。玩家还需以此打开关底的大门继续旅途。<ref name="IGNreview" />在路途中，玩家还会遇到像风筝一样的布製飞行生物，有些会为玩家领路或伴随在玩家左右。到了游戏后期，还会出现充满敌意的岩石怪物，如果角色不小心接触到它们，围巾就会被扯去一部分，玩家的能力因之下降。<ref name="GSreview" />

《风之旅人》的大多数游戏时间中，玩家会在服务器上与其他玩家随机配对搭档前行，每一关都可能是一位新的搭档，也可能整个旅途都是同一个人相伴。不同于其他合作类游戏或网络游戏，两名玩家无法像主流游戏那样，通过语音或文字交流互动，而且游戏过程中无法知道对方的名字。玩家之间的互动虽然不多，但都直接明了。玩家角色之间的身体接触可以使对方的长袍上的符文发光，同时也为对方补充能量。此外，玩家还可以通过吟唱来引起对方注意，此举亦能为对方补充能量。在游戏时，玩家需要与对方配合相互帮助来解密闯关，玩家之间无法相互阻挡或阻碍对方，这是游戏物理引擎特别为避免玩家相互陷害而设定的。<ref name="GadioInterview">[https://itunes.apple.com/bb/podcast/zhuan-fang-chen-xing-han-ii/id420660752?i=178919739&mt=2 GadioPro Vol.43专访陈星汉-《Journey》] {{webarchive|url=https://web.archive.org/web/20150317105422/https://itunes.apple.com/bb/podcast/zhuan-fang-chen-xing-han-ii/id420660752?i=178919739&mt=2 |date=2015-03-17 }}.机核网.2011-09-02.[2014-04-30]</ref>在游戏的最后，当游戏制作人员名单结束后，玩家可以看到那位与自己一同经历了这场旅途的玩家的[[PlayStation_Network|PlayStation Network]] ID。

在《风之旅人》中，制作组还加入了之前他们所开发游戏中的元素作为[[彩蛋_(視覺)|彩蛋]]，比如在沙漠深处的某个地方有出自《[[花_(游戏)|花]]》的花；<ref>[http://www.ign.com/wikis/journey/Hidden_Desert_Flower Hidden Desert Flower] {{webarchive|url=https://web.archive.org/web/20140430020630/http://www.ign.com/wikis/journey/Hidden_Desert_Flower |date=2014-04-30 }}.IGN.[2014-04-30]</ref>还有在遗迹深处出自《[[流_(遊戲)|流]]》的浮游生物。

== 故事设定 ==
《风之旅人》的故事主要由游戏中的遗迹壁画以及预渲染的过场动画来表达。游戏伊始，玩家所操纵的人物站在一片看似一望无际的沙漠中的沙丘上。在远方隐约矗立着一座高耸的山峰，峰顶被一道发光的裂缝一分为二。随着玩家们步向这座高峰，他们发现了一个曾经繁荣的文明残存下来的遗迹，但现在已经被风沙侵蚀许久。整个遗迹的每个关底区域，都分布着先驱旅者们安息于此的墓碑，点亮这些墓碑会点亮位于墓碑群中央的大石碑，并且会唤醒一位身穿发光白色长袍的高大角色，它会以类似古代壁画的形式告诉玩家这个文明的兴衰变化，也会展示出玩家们所经历的旅程。

随着玩家与同伴在前往高峰的旅途上逐步深入，玩家会进入位于山脚下的一座庞大复杂的城市遗址。该遗址便是通往峰顶的必经之路，接下来旅者们就要爬上顶峰。向山巅跋涉的过程十分艰难，气候严寒，大雪封山，带有能量的围巾也被凛冽的寒风吹散。虽然距离峰顶的那道发光的缝隙仍然路途遥远，但是旅者与同伴终于不支倒在了雪地中。此时，几位像在之前关底出现过的白袍角色出现在玩家的面前，他们给予了玩家全新的力量，让玩家向山的顶端飞去。最终在顶峰上，玩家遇到了之前的同伴，两人一起步入那道发光的缝隙，之后游戏的画面被光芒染成白色。至此，游戏的旅程就暂告一段落，不过在游戏开发人员名单的出现的时候，画面的背景中，有一颗流星从峰尖的那个缝隙中迸发而出。它随着人员名单从玩家之前所经历的所有场景中穿过；在这些场景中，还可以看到某个旅者也在前往山顶的道路上前行。最终，这颗流星落到了游戏开始时的那座沙丘的后面，并给出玩家重新开始游戏的选项。

《风之旅人》是[[thatgamecompany|thatgamecompany]]与索尼电脑娱乐签订的第三款也是最后一款游戏，前两款分别是《[[流_(游戏)|流]]》和《[[花_(游戏)|花]]》。在《花》发行之后，本作于2009年正式开始制作。共有18个人参加了本作的开发过程，公司联合创始人[[陈星汉|陈星汉]]担任本作的创意总监，尼克·克拉克担任本作的设计总监，马特·纳瓦担任本作的艺术总监；<ref>Nava, Matthew. (2013) ''The Art of Journey'', pp.182-183, 2nd ed., LA: BULUCAVAS</ref>而另一位公司创始人[[凯莉·圣迪亚戈|凯莉·圣迪亚戈]]并不在开发人员之列，因为其作为公司主席需要将更多的精力投入到管理方面。前[[EA|EA]]旗下的[[Maxis|Maxis]]工作室制作人{{le|罗宾·汉妮卡|Robin Hunicke}}加入开发团队并担任本作的制作人，以填补凯莉的空位。<ref name="GSint1"/>

在本作开始制作的时候，索尼方面预计本作的开发周期会在一年左右，不过《风之旅人》最后完工时距离开始制作已经过去了3年。<ref name="IGNpm"/>游戏延期在thatgamecompany的意料之中，汉妮卡曾提到，开发团队认为在一年内开发完游戏根本“不现实”。<ref name="GSpm"/>但是开发最终耗时还是比团队预计得长，原因出在团队难以舍弃游戏中的创意和保持团队有效的沟通。<ref name="GSpm"/>在开发的整个过程中，thatgamecompany参与其中的开发人员从最早的7位发展到了最后的18位。<ref name="IGNpm"/><ref name="TGCteam"/>在本作的开发周期的第二年年末，索尼给的延长期限即将用完，但是游戏并没有如团队所望唤起玩家的那份情感。索尼方面同意再延长一年的时间来改进游戏，不过最后花费的时间还是超过了这一期限。<ref name="DICEspeech"/>

项目带来了巨大压力，导致团队没有时间和资金来完成其愿景，这又使得开发团队感到更大的压力，还引发了本作在设计上的争论。最后开发成员减少了在项目上的加班赶工时间，以防出现[[倦怠|倦怠]]，但这也使得进度更为落后，甚至有风险在游戏接近完成时资金耗尽。在2013年第16届{{le|D.I.C.E峰会|D.I.C.E. Summit}}的年度颁奖礼上，陈星汉承认在开发的最后几个月里，公司确实已经走在了破产的边缘，以至于一些员工降低了工资甚至是无薪在工作。<ref name="DICEspeech"/>汉妮卡形容，最终能完成游戏开发，靠的是学会减少争端，学会放弃那些没办法做进游戏里的想法，以及“对他人好一些”。<ref name="IGNpm"/>

《风之旅人》的立意是旨在使玩家感到自身的“渺小”，并让玩家对其周围事物环境产生一种敬畏感。<ref name="GSPOTpreview2"/>虽然本作开始制作于2009年，不过据创意总监[[陈星汉|陈星汉]]介绍，他在大学时就有了关于《风之旅人》的雏形想法。<ref>[http://www.gdcvault.com/play/1017700/Designing Designing Journey] {{webarchive|url=https://web.archive.org/web/20150109232311/http://www.gdcvault.com/play/1017700/Designing |date=2015-01-09 }}.游戏开发者大会.2013.[2015-01-10].</ref>游戏最基本的概念是由陈星汉提出，他希望制作一款游戏能跳脱当前大部分电子游戏中所包含的“典型的失败/杀敌/获胜的套路”。<ref name="VGpreview"/>团队曾制作了一个测试原型游戏，在游戏中玩家需要试着把一只缠着另一个玩家的巨大怪兽引开。但是后来开发团队意识到玩家会很容易各自为战而无视对方的存在，而且原型游戏是在二维平面进行的，这个创意想法很难直接搬到三维的游戏世界里，所以最终这个原型和其他的二维原型被束之高阁。<ref name="VGpreview"/><ref>[http://gamerboom.com/archives/81509 《风之旅人》设计师分享团队开发游戏的过程] {{webarchive|url=https://web.archive.org/web/20150111061132/http://gamerboom.com/archives/81509 |date=2015-01-11 }}.游戏邦.2014-02-10.[2015-01-11].</ref>

开发组将本作设计成像一座“[[日本庭園|日式庭园]]”，他们试图将游戏中所有格格不入的游戏元素移除，给玩家带来开发组所希望游戏能唤起的情绪。<ref name="GSChenInt"/>这种极简风格的目的在于能使玩家直观感觉到游戏展现在其眼前，这样他们会在没有直接的指示下展开探索，感受惊喜。《风之旅人》的剧情线设计遵从了美国作家[[约瑟夫·坎贝尔|约瑟夫·坎贝尔]]的神学叙事理论和[[英雄旅程|英雄旅程]]概念，这一理念被陈星汉视为整个游戏的架构，<ref name="UCGinterview">{{cite book |author=一刀 |coauthors=陈星汉 |chapter=中国力量 |title= 《英雄旅程 伴你同行：再访TGC创意总监陈星汉--重生路上的砥砺前行》 |publisher= [[游戏机实用技术|游戏机实用技术]] |issn= 1008-0600 |pages=第8至11页 |language= zh-hans |location= 中国深圳 | accessdate=2015-01-16|date=2012年4月}}</ref>提升并加强了玩家之间在共同游戏时的情感纽带。<ref name="GSTGCInt"/>陈星汉在他D.I.C.E峰会上的演讲时曾提到：在25名试玩者中有3名在测试游戏时哭了出来。<ref name="DICEspeech"/>

《风之旅人》作为一款多人在线游戏，其多人游戏部分的设计理念是让玩家间的合作更加方便，而非让玩家互相对抗，但又不会强制双方必须合作进行游戏。<ref name="GSChenInt"/>其目的是希望玩家双方通过在游戏中的一同探索来感受到一种和他人的关系，而不是像其他游戏那样通过交谈或进行打斗来与其他玩家产生交集。<ref name="GSPOTpreview2"/>为了达到这个目的，开发人员的计划是“创造的游戏能让玩家感觉到与其他玩家是有关系的，并在其中展现出人性里积极的那一面”。<ref name="GSChenInt"/>为此，开发人员在游戏引擎上做出改动，取消了玩家所控制的角色间的物理碰撞效果，因为他们发现即使去有意引导玩家在游戏里的行为，还是会出现有人陷害另外一个人的情况，甚至是开发人员自己也会去做类似这样的事。<ref name="GadioInterview"/>在开发过程中，开发组还意识到让玩家保持对对方的关注并非易事，因为在游戏时，玩家很容易被很多其他游戏元素干扰而转移注意力，由此“无视”掉游戏中的同伴。<ref name="GSChenInt"/>游戏中，玩家只能通过简单的吟唱来实现互动，这来自于开发人员的考量。他们认为在游戏中加入文字或语音交流功能以及显示对方的用户名很有可能会让玩家产生偏见。<ref name="GSBellInt"/>

《风之旅人》在2011年6月底向北美地区的PlayStation Plus用户提供封闭Beta测试的邀请，7月时在欧洲地区用户提供Beta测试资格。<ref>[http://www.eurogamer.net/articles/2011-06-27-how-to-get-in-the-journey-beta How to get in the Journey beta] {{webarchive|url=https://web.archive.org/web/20150226194333/http://www.eurogamer.net/articles/2011-06-27-how-to-get-in-the-journey-beta |date=2015-02-26 }}.Eurogame.2011-06-27.[2015-02-27].</ref>2012年3月13日起，《风之旅人》在各地区的PlayStation Network商店陆续上架，而PlayStation Network的付费会员在3月8日就可以提前下载购买。<ref name="JourneyRelease"/>之后索尼在其PlayStation 3的网络服务[[PlayStation_Home|PlayStation Home]]上特地开放了一个以《风之旅人》为主题的区域。<ref name="PShome"/>《风之旅人》的零售版——《风之旅人 典藏版》于2012年8月28日发售在PlayStation 3平台上，这个版本中除了本身的《风之旅人》以外，还包含了[[thatgamecompany|thatgamecompany]]之前的作品《流》和《花》以及相关的游戏资料、游戏原声以及若干款thatgamecompany开发的迷你游戏。<ref name="CollectEd"/>2012年9月，索尼和thatgamecompany合作出版了《风之旅人》的艺术设定集——《The Art of Journey》，本书由《风之旅人》的艺术总监马特·纳瓦主编，书中包含了从游戏早期的概念设定到游戏最终的画面。<ref name="ArtOfJourney"/>

2013年4月1日，《风之旅人》的开发团队在自己的Youtube频道上上传了将为《风之旅人》开发名为“火箭弹死亡比赛”（Rocket Death Match）的[[追加下载内容|追加下载内容]]（DLC）内容，后证实为[[愚人节|愚人节]]玩笑。<ref>{{cite web |url=http://www.youtube.com/watch?v=MqXHDdwI2ug |title=Official Journey PS3 Rocket Death Match DLC Teaser |publisher=[[Youtube|Youtube]] |date=2013-04-01 |accessdate=2014-04-22 |deadurl=no |archiveurl=https://web.archive.org/web/20140523172859/http://www.youtube.com/watch?v=MqXHDdwI2ug |archivedate=2014-05-23 }}</ref>2014年8月12日，陈星汉在[[索尼電腦娛樂聖塔莫尼卡工作室|索尼電腦娛樂聖塔莫尼卡工作室]]的官方网站上宣布《风之旅人》将在2014年内登陆[[PlayStation_4|PlayStation 4]]；这是继《流》（浮游世界）和《花》之后，thatgamecompany的游戏作品再一次移植到PlayStation 4平台上。PS4版本的《风之旅人》由Tricky Pixels工作室负责游戏的移植工作。<ref name="PS4"/>不过游戏并没有在2014年如期发售，而该工作室通过其[[推特|推特]]帐号与玩家互动时表示游戏还在开发中，不到开发结束不会放出消息，不过不会让玩家等太久。<ref>[http://customstoday.com.pk/tricky-pixels-to-release-ps4-version-of-journey-soon/ Tricky Pixels to release PS4 version of Journey soon] {{webarchive|url=https://web.archive.org/web/20150226212710/http://customstoday.com.pk/tricky-pixels-to-release-ps4-version-of-journey-soon/ |date=2015-02-26 }}.CustomsToday.2015-01-12.[2015-02-27].</ref>

《风之旅人》的游戏音乐由曾与[[thatgamecompany|thatgamecompany]]合作过《[[流_(遊戲)|流]]》的美国作曲家[[奥斯丁·温特里|奥斯丁·温特里]]操刀制作。温特里在制作配乐的过程中还与音效设计师史蒂夫·约翰逊以及游戏的程序团队进行紧密合作，这让游戏中的音乐可以动态地与玩家的行为和由附近物体造成的声效相合，让玩家感觉到音乐像是“实时演奏的”。<ref name="TSAint"/>约翰逊觉得，如果游戏音乐没有对玩家的行为有所回应，而只是循环播放几段音乐将会“十分遗憾”，因此他想令其音乐既有所变化、又能保有编曲的情感起伏。在游戏开发之初，陈星汉与温特里会面，并向温特里表述了他对这个项目的愿景。当天晚上，温特里就编曲并录制了一份以大提琴为主的游戏原声样曲。此后的三年时间里，他都在为这个游戏的音乐原声殚精竭虑，实验了很多也放弃了很多点子。<ref name="PARint"/>

有许多电子游戏在每个角色或场景上都会使用不同主题的音乐曲目，温特里则选择基于同一个主题来创作游戏中所有的曲目，以此从音乐上构建出游戏角色和他们的旅程，像大提琴独奏是专门代表玩家的角色。他形容为本作创作的音乐像“一组大型大提琴[[协奏曲|协奏曲]]，玩家担任的是独奏，而其他乐器代表的是玩家周遭的世界”。<ref name="TSAint"/><ref name="IGint"/>在游戏正式开始之前，大提琴的独奏“使玩家浸入电子声效的海洋中”，拉开了游戏的序幕，之后才有游戏角色的出现；而之后渐渐加入整个乐团的旋律，反映出玩家将要踏上前往山峰的旅途。<ref name="GUARint"/>游戏本身的艺术风格是基于几种不同的文化而形成的，温特里在游戏音乐中所做的是尝试从乐曲中移除明显的文化特征，并尽可能地隐藏文化上的特质，使游戏中的音乐“普世化”。<ref name="TSAint"/>美籍华裔大提琴演奏家[[郭婷娜|郭婷娜]]负责游戏原声中的大提琴演奏，她和温特里也因为这次合作成为了好友。<ref name="IGint"/>原声配乐中所有非电子乐器演奏的曲目均由一现场交响乐团完成。<ref name="PARint"/>

《风之旅人》的游戏音乐被[[電玩交響音樂會|電玩交響音樂會]]选用，在2012年开始在電玩交響音樂會上作为演出节目之一，由音乐制作人奥斯丁·温特里登台指挥。<ref>{{cite web |url=http://austinwintory.com/news/video-games-live/ |title=Video Games Live! |publisher=Austin Wintory |date=2012-05-27 |accessdate=2015-01-10 |deadurl=no |archiveurl=https://web.archive.org/web/20150109234220/http://austinwintory.com/news/video-games-live/ |archivedate=2015-01-09 }}</ref>

=== 原声大碟 ===
《风之旅人》的原声大碟于2012年4月10日在苹果[[iTunes|iTunes]]网络商店和索尼PlayStation Store网络商店上架。<ref name="TGCalbum"/>专辑收录了所有配乐中“最重要”的曲目，由温特里重新编曲，
与游戏中有所区别，他认为音乐在游戏中的效果和单独拿出来听是不一样的感受。<ref name="TSAint"/>专辑收录了18首游戏原声曲目，总时长超过了58分钟。曾为《[[潜龙谍影4_爱国者之枪|潜龙谍影4 爱国者之枪]]》演唱过主题曲的美国女歌手{{le|丽丝贝思·斯科特|Lisbeth Scott}}也为本作献唱了结尾曲《I Was Born for This》。

在发售后，这张原声大碟在超过20个国家的iTunes原声音乐榜上登上过前十名，而首周超过4,000张的销量也使它登上了[[公告牌二百强专辑榜|公告牌专辑榜TOP200]]的第116位，这也是当时所有游戏原声专辑中第二高的打榜成绩。<ref name="BBsales"/>在2012年10月9日，游戏原声的实体版专辑由Sumthing Else Music Works推出。<ref name="Palbum"/>此后，温特里在音乐流网站[[Bandcamp|Bandcamp]]上还推出了名为《Journey Bonus Bundle》的下载版游戏原声专辑，在这张专辑中包括了《风之旅人》的不同主题的配乐和来自《[[流_(遊戲)|流]]》的音乐。<ref name="JBBalbum"/>这张专辑在2013年6月19日于Bandcamp上开始免费提供。<ref name="Bandcamp"/>而在2014年5月1日，本原声专辑的钢琴演奏版《Journey: Transfiguration》在Bandcamp上发售数字版以及实体版。这张专辑作为電玩交響音樂會在众筹网站[[Kickstarter|Kickstarter]]上的目标计划之一公布，并筹款成功。在这张专辑中，《I Was Born for This》的演唱者由丽丝贝思·斯科特变为了劳拉·伊特维亚。她还负责为整张专辑重新编曲。<ref name="GMOtrans"/>

《风之旅人》获得了业界的高度评价，并荣获大量奖项。在发售前的2011年[[E3游戏展|E3游戏展]]上，游戏获得[[1UP.com|1UP.com]]、[[GameSpy|GameSpy]]和[[GameTrailers|GameTrailers]]颁发的“最佳下载游戏”奖，<ref name="TGCawards" />游戏制作人[[宫本茂|宫本茂]]试玩后称赞游戏“非常美丽”。<ref>{{cite web|url=http://www.weibo.com/1834812694/eBQDW0Bpmp8?mod=weibotime |title=@Jenova陈星汉：今天E3迟到，意外的发现宫本茂在索尼的展台玩≪Journey≫。他说＂非常美丽＂我不知如何回答TAT |publisher=新浪微博 |date=2011-06-10 |accessdate=2014-03-26 }}</ref>本作发售后，就在当年年末开始的各种游戏评奖中收获颇丰。《风之旅人》在第16届{{le|D.I.C.E峰会|D.I.C.E. Summit}}年度颁奖礼上（原[[互动艺术与科学学会|互动艺术与科学学会]]互动成就奖），赢得包括“年度游戏”、“杰出游戏创新”、“年度休闲游戏”、“杰出游戏指导成就”、“杰出艺术指导成就”、“杰出在线游戏成就”、“杰出原创音乐作曲成就”以及“杰出音效设计成就”在内的8项大奖，成为当晚最大的赢家；此外它还获得了“年度下载游戏”、“杰出游戏工程成就”和“杰出故事成就”等奖项提名。<ref name="DICEawards" /><ref name="DICEnoms" />在[[英国电影和电视艺术学院|英国电影和电视艺术学院]]主办的[[英國學院遊戲獎|英国学院电子游戏奖]]上，《风之旅人》摘得“艺术成就”、“音响成就”、“游戏设计”、“在线多人游戏”和“原创音乐”等5项大奖，并获“最佳游戏”、“游戏创新”和“游戏故事”等奖项提名。<ref name="BAFTAnom" /><ref name="skynews" />本作在{{le|斯派克游戏大奖|Spike Video Game Awards}}的颁奖礼上，赢得“最佳PlayStation 3游戏”、“最佳独立游戏”、“最佳游戏音乐”等奖项<ref name="VGAps3" /><ref name="VGAindie" /><ref name="VGAmusic" />；并获得了“年度游戏”、“最佳下载游戏”、“最佳画面”以及《I Was Born For This》所获得的“最佳歌曲”等奖项提名。<ref name="VGAgoty" /><ref name="VGAdownload" /><ref name="VGAgraphics" /><ref name="VGAsong" />在2013年3月由[[游戏开发者大会|游戏开发者大会]]举行的[[游戏开发者选择奖|游戏开发者选择奖]]上，《风之旅人》赢得了“最佳音响”、“最佳游戏设计”、“最佳视觉艺术”、“最佳下载游戏”、“创新奖”和“年度游戏”等6项大奖。<ref name="variety" />游戏还被[[IGN|IGN]]和[[GameSpot|GameSpot]]等著名游戏媒体评为2012年“年度游戏”。<ref name="IGNgoty" /><ref name="GSgoty" />此外，游戏还获得IGN年度评选的“最佳音乐”奖以及“最佳画面”的提名。<ref name="IGNmusic" /><ref name="IGNgraphics" />游戏被GameSpot选为“最佳PlayStation Network游戏”<ref name="GSpsn" />除游戏本身外，《风之旅人》的游戏音乐原声专辑在2013年[[第55屆葛萊美獎|第55屆葛萊美獎]]上获得了“最佳影视媒体作品配乐专辑”的提名，这是首次有游戏音乐获得该奖项提名。<ref name="Grammy" />2013年的{{le|第40届安妮奖|40th Annie Awards}}把“最佳电子游戏动画”颁给了《风之旅人》。<ref name="annie" />

《风之旅人》获得了业界高度赞誉，游戏编辑和评论对游戏的视听艺术大加赞扬，同时还称赞和陌生人游玩时建立的陪伴感，能为玩家带来触动心弦的情感体验。[[IGN|IGN]]的瑞恩·克莱门茨形容本作是“这个世代里最美的游戏”，并称“每个瞬间都像一幅画作”。<ref name="IGNreview" />[[GameSpot|GameSpot]]的简·道格拉斯有着相似的赞许，称《风之旅人》“美丽得无以复加”，并称赞游戏世界的视觉多样性，以及对沙漠的细腻刻画。<ref name="GSreview" />而[[Game_Informer|Game Informer]]的马特·米勒赞扬了游戏中砂砾和各种生物的动画，说本作是视觉上的震撼。<ref name="GIreview" />《[[游戏机实用技术|游戏机实用技术]]》在其游戏年鉴里评价本作除了探讨人与自然的关系，还引入了像“[[禅|禅]]”和“[[轮回|轮回]]”等极具东方色彩的哲学思想，加深了游戏的内涵。<ref name="UCGreview2">{{cite journal|author=胜负师|journal=游戏机实用技术|issue=316–317|title=2012年下载游戏年鉴|page=60|year=2012}}</ref>

评论普遍称赞了游戏带来的情感体验，特别是与其他玩家一起游戏时。[[Eurogamer|Eurogamer]]的克里斯蒂安·唐兰将这种体验形容为一种超越言喻的“无教派的宗教体验”，再加上其他的玩家，成为了一种对玩家而言的“朝圣之旅”。<ref name="EGreview" />《[[Edge_(杂志)|Edge]]》的一位撰稿人写到：游戏中情感的铺垫与宣泄不时迸发出惊人的能量。《[[连线|连线]]》的帕特里克·肖同时也说道游戏令他感到“各种各样的情绪……奇妙的、恐惧的、甚至是悲伤的。” Game Informer的马特·米勒说在他三次打通关本作的时间里，“每一次通关都没有失败，有些时刻……成功地让我起了鸡皮疙瘩，并且这些片段在之后几周的时间里都会在我的脑海中萦绕。”<ref name="EDGEreview" /><ref name="WIREDreview" />《[[PlayStation官方杂志－英国|英国PlayStation官方杂志]]》的乔尔·格雷戈里称赞《风之旅人》的故事是开放给玩家们自己去解读的。<ref name="OPMUKreview" />Eurogamer的克里斯蒂安·唐兰形容加入无名的同伴这一创意十分出彩，简直是“神来之笔”，《Edge》称其成就了“更加精彩绝伦的体验”。<ref name="EGreview" /><ref name="EDGEreview" />

对于《风之旅人》的少量批评以游戏长度和节奏为主。IGN的瑞恩·克莱门茨指出，并非所有玩家都喜欢节奏“谨慎压抑”且流程较短的游戏；《Edge》的评测也给出相似评论。<ref name="IGNreview" /><ref name="EDGEreview" />Game Informer的马特·米勒认为，本作缺乏多元的游戏元素。《连线》的帕特里克·肖也对游戏的数小时的流程失望，但GameSpot的简·道格拉斯和《[[游戏机实用技术|游戏机实用技术]]》的评论认为游戏长度恰到好处。<ref name="GIreview" /><ref name="GSreview" /><ref name="UCGreview" /><ref name="WIREDreview" />Game Informer的马特·米勒在他的总结里写道，本作可以好比“一场音乐演唱会、一部精心执导的影片，抑或是一部期待已久的著作”；<ref name="GIreview" />而IGN的瑞恩·克莱门茨总结称：“《风之旅人》的通关会带来一段难以忘怀的回忆。”<ref name="IGNreview" />

《风之旅人》在获得好口碑的同时也获得了良好的商业成绩。在PlayStation 3版游戏发售后，它马上打破了PlayStation Network的销售记录，成为当时北美和欧洲地区的最畅销的[[PlayStation_Store|PlayStation Store]]游戏。<ref name="SONYsales" />而PlayStation 4版在发售后也获得了当月欧美地区PlayStation Network的销售冠军。<ref>{{cite web |url=http://www.gamespot.com/articles/best-selling-ps4-ps3-and-ps-vita-games-on-psn-for-/1100-6429587/ |title=Best-Selling PS4, PS3, and PS Vita Games on PSN for July 2015 Revealed |publisher=[[GameSpot|GameSpot]] |date=2015-08-07 |accessdate=2015-08-10 |deadurl=no |archiveurl=https://web.archive.org/web/20150829034215/http://www.gamespot.com/articles/best-selling-ps4-ps3-and-ps-vita-games-on-psn-for-/1100-6429587/ |archivedate=2015-08-29 }}</ref>

《风之旅人》还对其他游戏产生了影响。於在2013年4月9日发售的独立游戏《[[Guacamelee!|Guacamelee!]]》中，有若干向其他电子游戏致敬的场景，其中包括了以《风之旅人》为主题的场景。<ref>{{cite web |url=http://www.ign.com/wikis/guacamelee/Easter_Eggs_and_References |title=Guacamelee! Easter Eggs and References |publisher=[[IGN|IGN]] |date=2013-04-01 |accessdate=2014-04-22 |deadurl=no |archiveurl=https://web.archive.org/web/20140503210117/http://www.ign.com/wikis/guacamelee/Easter_Eggs_and_References |archivedate=2014-05-03 }}</ref>而另一款在2015年发售的独立游戏《{{le|超时空战队 究极版|Super Time Force}}》在其[[PlayStation_4|PlayStation 4]]及[[PlayStation_Vita|PlayStation Vita]]版本中包含有与索尼旗下相关的独占游戏内容，其中就有《风之旅人》中旅人一角作为游戏中的角色登场。<ref>{{cite web |url=http://blog.us.playstation.com/2015/04/16/super-time-force-ultra-journey-the-order-characters-revealed/ |title=Super Time Force Ultra: Journey & The Order Characters Revealed |publisher=PlayStation Blog |date=2015-04-18 |accessdate=2015-04-19 |deadurl=no |archiveurl=https://web.archive.org/web/20150418201323/http://blog.us.playstation.com/2015/04/16/super-time-force-ultra-journey-the-order-characters-revealed/ |archivedate=2015-04-18 }}</ref>
