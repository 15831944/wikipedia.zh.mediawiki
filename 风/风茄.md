'''风茄'''（[[学名|学名]]：''{{lang|la|Mandragora}}''，{{lang-en|Mandrake}}），又名'''毒參茄'''、'''曼德拉草'''，是[[茄参属|茄参属]]植物的通称，属[[茄目|茄目]][[茄科|茄科]]<ref>含致命的茄属植物和天仙子。</ref>，其根部外型类似人的樣子。长期用于[[巫术|巫术]]仪式，包括今天的[[威卡教|威卡教]]。

其根部分叉并长出地表一个褶皱、易碎的卵圆形物，叶为锯齿状，6到16英寸长，类似烟草。春天从颈部伸出许多下垂的花梗，开淡绿色花，将近2英寸宽，果实为球形、多汁的橙色或红色浆果，类似小型的番茄，次年春天成熟。风茄的每一部分都有毒性。它的原产地是[[欧洲|欧洲]]南部和中部、[[地中海|地中海]]周围地区，以及[[科西嘉岛|科西嘉岛]]。

被阿拉伯人稱為luffâh或beid el-jinn（「妖魔的蛋」）。在小說《[[哈利波特-消失的密室|哈利波特-消失的密室]]》中被稱為魔蘋果。

== 旧约圣经 ==
根据[[创世纪30章|创世纪30章]]的记载，[[雅各|雅各]]和[[利亚|利亚]]的长子[[流便|流便]]在田间发现了风茄。雅各的第二位妻子、利亚的妹妹[[拉结|拉结]]，由于不能生育，渴望从姐姐手中得到风茄，提出将该夜与丈夫同床的权利作为交换。此前利亚在生下四个儿子后，已经停止生育，但在此后利亚又再度怀孕，连续生下2个儿子和1个女儿。犹太人传统上认为风茄能帮助不孕妇女怀孕。

风茄在[[希伯来语|希伯来语]]中称为{{rtl-lang|he|דודאים}}（dûwôdãym），意为“爱的植物”<ref>見：[[創世記|創世記]]30章14节（“爱的植物”）和[[雅歌|雅歌]]7章13节（“风茄放香”）</ref>。在某些亚洲文化中，相信它具有帮助受孕的作用。其他许多植物也被认为有此功效，例如[[香蕉|香蕉]]、[[百合|百合]]、[[柑橘屬植物|柑橘屬植物]]和[[无花果|无花果]]，但是其功用并无令人满意的证据。

== 传说中的风茄 ==
{{Section OR}}
[[File:Mandragora_Tacuinum_Sanitatis.jpg|thumb]]
根据传说，风茄根大，呈人形，因此被认为具有很强大的魔力。美洲的风茄是盾叶鬼臼根茎，与欧洲的并不是同一种植物。在[[中世纪|中世纪]]，风茄价格相当昂贵。水手们常买一小片根以保佑航海平安。把小片的风茄根放在大门上，可以防止火灾和失窃；放在枕头下有利睡眠；放在床垫下，尤其是婴儿的床垫下可以防止做噩梦。完整的风茄草根价格更为昂贵，根部越像人形，价格就越高。

由於风茄是野生植物，为防止他人采集，卖风茄的人就四处散布谣言，说直接采集风茄会带来麻烦。风茄的根呈人形，人们传说它是个具有灵性的活物，若把它从土里采出，就像把人从家里抢走一样。因此风茄被拔出时会发出一种剧烈的惨叫声，惨叫声会令采集者精神失常甚至丧命。采集风茄有一种特殊的方法，需要猎号、长绳、饥饿的狗和一块肉。据说风茄只生长在绞刑架下，靠死尸滴下的脂油或血液灌溉成长。因此在月圆之夜，魔法师会用绳子将草牢牢套住，绳子的另一端绑在狗的项圈上，然后魔法师退到安全的地方把肉扔出，狗因饥饿而迅速扑向肉，连着的绳子就会把草从土里拉出来。因此，魔法师要大声吹号盖过草的惨叫。狗听到惨叫声立即毙命，而魔法师则顺利地获得了完整的风茄。

酷似人形的风茄价格要高出一般风茄的几倍，很多魔法师在草的生长过程中会对草根进行修饰。他们找到小风茄，小心挖出检验之后，用刀雕刻得更像人形，如果有必要甚至会刻出人脸的形状，然后把草又重新植入土中。过上一个月左右，再次把草重新挖出来检查，然后再次进行修饰，植入土中。这样，等到草长成后挖出时，似乎纯天然就长得极像人形，可以售得高价。也正因为如此，风茄会被做成雌株与雄株。

另一个传说是，风茄能在黑暗中闪闪发光。植物学家多琳·瓦里安特曾解释说，这是因为它的叶子能吸引萤火虫。萤火虫盘踞在叶片上，就像叶子自己能发光一样。他还说，风茄草有很强的催眠作用，在过去的手术中常被当做麻醉药使用。风茄原产自地中海地区，在15世纪或更早被引入英国。大小与风茄极为相似、有肉质根、白色或黑色灌木植物的[[泻根|泻根]]会被认为是它的优良替代品。

== 在文学中 ==
* 在[[圣经|圣经]]创世纪30章14节，利亚给了拉结一些风茄，以换取当夜与丈夫雅各同床的权利。
<blockquote>
::::“割麦子的时候，流便往田里去， 
::::寻见风茄，
::::拿来给他母亲利亚。 
::::拉结对利亚说， 
::::请你把你儿子的风茄给我些。"
</blockquote>

雅歌7章13节

<blockquote>
::::“风茄放香，
::::在我们的门口有各样新陈佳美的果子。
::::我的良人，
::::这都是我为你存留的。”
</blockquote>
