'''风筝'''，亦称'''风琴'''、'''纸[[鹞形|鹞]]'''、'''[[鹞形|鹞]]子'''、'''纸[[鸢|鸢]]'''，[[閩南語|閩南語]]稱'''風吹'''（[[白話字|白話字]]：Hong-chhoe）。古代稱之爲'''“鷂”'''，北方謂之'''“鳶”'''。风筝是一种比[[空气|空气]]重的，能够借助[[风|风]]力在空中漂浮的制品。晚唐，人们在纸鸢上加哨子，其鸣如筝如琴，故称“风筝”或“风琴”。

现代以风筝作为统称，包括没有哨子的纸鸢。

== 沿革 ==
[[File:Chinese_dragon_kite_(Berkeley,_California_-_2000).jpg|200px]]”，主要以龙头蜈蚣风筝为主，也是潍坊风筝的一大特色。]]
风筝起源于[[中国|中国]]，中国风筝有悠久的历史。最早的風箏並不是以紙製作，而是木制的。傳説中第一個風箏為“[[魯班|魯班]]”製作，稱之爲“木鳶”，《[[渚宫旧事|渚宫旧事]]》记载[[魯班|魯班]]「尝为木鸢，乘之以窥宋城」。而以《[[呂氏春秋|吕氏春秋·爱类]]》記載“公输般为高云梯，欲以攻宋。墨子闻之，自鲁往，裂裳裹足，日夜不休，十日十夜而至于郢，见荆王曰：‘臣北方之鄙人也，闻大王将攻宋，信有之乎？’”説明[[魯班|魯班]]當時造的是“高雲梯”而不是木鳶<ref>見《呂氏春秋·愛類·五曰·二》</ref>。

《[[韩非子|韩非子]]》記載中木鳶的發明者不是[[魯班|魯班]]而是墨子[[墨子|墨翟]]。[[墨子|墨翟]]（公元前478-392年），為[[春秋战国|春秋战国]]时的哲人，[[諸子百家|諸子百家]]中[[墨家|墨家]]的代表人物。《[[韓非子|韩非子·外储说]]》記載墨翟曾「费时三年，以木制木鸢，飞升天空」。墨子在鲁山（今[[山东|山东]][[潍坊|潍坊]]境内），「為木鳶，三年而成，蜚一日而敗」<ref>見《韓非子·外儲說左上·十一》</ref>。意思是说墨子曾就研究试制了三年，以木板制成了一只木鸟，將其放飛並使其成功飛起，但只飞了一天就坏了。墨子制造的「木鸢」（或「木鹞」）就是世界上最早的风筝，距今已有二千四百多年。另《[[康熙字典|康熙字典]]》中釋義的木鳶發明者也是“墨翟”<ref>見《康熙字典》“鳶”字釋義</ref>。而《墨子》中記載[[魯班|魯班]]發明的“木鵲”比墨子的“木鳶”更加精巧：“公輸子削竹木以為鵲，成而飛之，三日不下”<ref>見《墨子·卷十三·魯問·二十一》</ref>。后[[東漢|東漢]][[蔡倫|蔡倫]]發明造紙朮之後，才開始出現紙做的風箏，稱爲“紙鳶”。

据说[[汉朝|汉朝]]大将[[韩信|韩信]]曾利用风筝进行测量。[[梁武帝|梁武帝]]时曾利用风筝传[[信|信]]，但未成功。<ref>見《獨異志》[[:s:獨異志#卷中|卷中]]，原文：「梁武帝大清三年，侯景反，圍臺城，遠近不通。簡文與太子大器為計，縛鳶飛空，告急於外。侯景謀臣謂景曰：『此必厭勝術，不然即事達人。』令左右射之。及墮，皆化為禽鳥飛去，不知所在。」</ref>[[南北朝|南北朝]]有人背着风筝从高处跳下而没有跌死。[[唐朝|唐朝]]的[[张丕|张丕]]被围困时曾利用风筝传信求救兵，書有“三日不解，臨名士且為悅食”之言<ref>見《[[新唐書|新唐書]]》[[:s:新唐書/卷210|列傳第135藩鎮魏博]]，原文：「悅自將兵數萬繼進，又使朝光攻臨洺將張伾。伾固守，食且盡，……有詔河東馬燧、河陽李芃與昭義軍救伾。三節度次狗、明二山間，未進。伾急，以紙為風鳶，高百餘丈，過悅營上，悅使善射者射之，不能及。燧營噪迎之，得書言『三日不解，臨洺士且為悅食。』……」</ref>，取得了成功。从唐朝开始，风筝逐渐变成[[玩具|玩具]]。到了晚唐，风筝上已有用丝条或[[竹笛|竹笛]]作成的响器，风吹声鸣，因而有了“风筝”的名字<ref>（唐）高承《事物纪原》：“纸鸢其制不一，上可悬灯，又以竹为弦，吹之有声如筝，故又曰风筝”</ref>。也有人说“风筝”这名字起源于[[五代|五代]]，从[[李邺|李邺]]用纸糊风筝，并在它上面装竹笛开始{{Fact|time=2007-10-04T13:52:31Z}}。

[[清朝|清朝]][[道光|道光]]年间，[[郭麟吟|郭麟吟]]清明的一首竹枝词描绘道：“一百四日小寒食，冶游争上白浪河，纸鸢儿子秋千女，乱比新来春燕多。”[[郑板桥|郑板桥]]有诗曰：“纸花如雪满天飞，娇女秋千打四围，飞彩罗裙风摆动，好将蝴蝶斗春归。”，风筝艺术亦达到鼎盛阶段。

== 分类 ==
[[中国|中国]][[传统|传统]]的风筝一般分为以下三大类：

'''软翅风筝'''：主体骨架多数做成[[浮雕式|浮雕式]]，骨架有单层、双层和多层，[[升力片|升力片]]（翅）由一根主翅条构成，翅膀的后半部是软性的，没有主条依附。

'''硬翅风筝'''：[[骨架|骨架]]由上下两根[[竹条|竹条]]做成，两侧边缘高，中间略凹，翅的端部向后倾，使风能从两翅的端部逸出。

'''板子风筝'''：即平面型风筝，升力片是主体，无凸出部分，风筝四边有竹条文撑，是少年儿童最喜爱的一种。

其他主要类型还包括'''串子类'''、'''立体类'''、'''桶行形类'''、'''自由类'''（其特点是运用现代新技术和外国风筝之长）等。

== 产地 ==
[[开封|开封]]、北京、天津、[[潍坊|潍坊]]、[[南通|南通]]、[[阳江|阳江]]并称中国六大传统风筝产地，而全国最大的风筝制造地在[[山东|山东]]的[[潍坊|潍坊]]，被称为“世界风筝之都”。

== 制法 ==
首先将竹篾浸水，令竹篾软身，再用刀将竹篾破开，约三分之一粗度，然后修半形，因为稍后要将竹篾贴在马拉纸上，如果太粗，竹篾会拉破纸张，同时竹篾太粗，纱纸条就贴不稳。将修好的竹篾裁成两条长短适当的长度，约为16英寸及23英寸。下一步，就将马拉纸裁成一个四方，长约24英寸，马拉纸是一种非常粗糙的纸张，最适合作风筝之用。到此时就可以将竹篾贴在纸上，但要记着，将长长的竹篾，用纱纸扎在短的三份之一，然后慢慢屈曲，直至长竹篾两端触到纸的对角之上将它贴好。最后一步就将风筝的尾巴贴在风筝的下方，校好线与风筝的角度后，就可起放。

风筝的尾巴是平衡风筝的主要工具，当风筝乘风而上之时，如果一方较重，风筝就会偏向这方，而尾巴最好比较长，因为越长就会有一个重量令风筝头部升起，使全身受风，平衡了斜的一方。

风筝的丝线可以用牛皮线、棉线、玻璃线等线辘可分圆线辘及排辘将线扎在风筝上，要成一斜角。

== 西傳 ==
公元[[十三世纪|十三世纪]][[意大利|意大利]]旅行家[[马可·波罗|马可·波罗]]将风筝传入欧洲。

== 題詠 ==
<poem>
*風箏（唐•鮑溶）
何響與天通，瑤箏掛望中。
彩弦非觸指，錦瑟忽聞風。
鴈柱虛連勢，鸞歌且墜空。
夜和霜擊磬，晴引鳳歸桐。
幽咽誰生怨，清泠自匪躬。
秦姬收寶匣，搔首不成功。
*風箏歌（宋•田錫）　
白蘋洲暖春風生，畫樓檻上銀箏鳴。
鏗鏘節奏急復慢，空中一部天樂聲。
三十六宮深窈窕，繡楣藻井光相照。
十三弦上千般聲，朝靄微吟暮煙嘯。
夜來親向月中聞，繁音錯節何紛紜。
碎如鸞鈴與珂珮，巫山隊仗迎湘君。
晚來金屋愁微雨，風細箏聲不全舉。
依希嬪妾怕人知，啾啾切切私相語。
洪纖斷續何所拘，鳳凰著對飛鸞孤。
梧桐枝邊泊未穩，琅玕島上鳴相呼。
有時主奏俄中絕，宮商鬥頓如刀截。
杏花露重鴛鴦寒，空見如霜滿庭月。
有時半日全無風，一一暮天樓閣紅。
唯聞鳥雀啄弦上，暖珠寒玉何玲瓏。
清音朝朝與暮暮，誤聲不管周郎顧。
祗嫌雅鄭交奏時，寶鐸丁冬闇相妬。
*九日旗峰戲詠風箏（明•鄧雲霄）
誰繪珍禽剪綵箋？化成鸞鶴奏鈞天。
抑揚線索騰雙翮，泛弄宮商撫一弦。
瑤瑟聽來移別調，玉簫隨去挾飛仙。
繁聲莫遣驚南雁，歲晏音書正待傳。
*小孫放風箏，志勗（淸•鄭用錫）
昂藏意氣入雲煙，喜放風箏到九天。
要識扶搖能直上，全憑一線手中牽。
</poem>
