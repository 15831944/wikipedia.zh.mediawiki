'''貓科'''包括[[獅子|獅子]]、[[老虎|老虎]]和[[豹|豹]]等動物，是[[食肉目|食肉目]]的9個科中最為[[肉食性|肉食性]]的[[哺乳動物|哺乳動物]]。第一種的貓科動物出現於[[漸新世|漸新世]]，人們更熟悉的[[家貓|家貓]]和人類在一万年前開始有關連。家貓的野生種親戚[[斑貓|野貓]]，仍然生存在[[歐洲|歐洲]]、[[非洲|非洲]]和[[亞洲|亞洲]]東部等地，雖然[[棲地破壞|棲地破壞]]限制了其居住範圍。

其他貓科動物中知名的成員有包括如[[獅子|獅子]]、[[老虎|老虎]]、[[豹|豹]]和[[美洲豹|美洲豹]]等[[豹屬|大貓]]，以及其他如[[美洲獅|美洲獅]]、[[獵豹|獵豹]]、[[猞猁|猞猁]]、[[獰貓|獰貓]]和[[短尾貓|短尾貓]]等[[貓亚科|貓亚科]]动物。已滅絕的[[劍齒虎亞科|劍齒虎亞科]]為真的貓科動物，而其他如[[袋劍虎|袋劍齒虎]]和[[獵貓科|獵貓]]等相似的動物則不是。

== 演化 ==

已知現今世上有38種貓科動物，牠們都源自一個共同的祖先。這些物種源自於[[亞洲|亞洲]]，且經由[[陸橋|陸橋]]散布至各[[洲|洲]]去。根據發表在《[[科學_(期刊)|科學]]》上，由[[美國國家癌症研究所|美國國家癌症研究所]]的Warren Johnson和Stephen O'Brien對[[粒線體基因|粒線體基因]]和[[細胞核基因|細胞核基因]]所做的研究，確定了貓的祖先在[[白令陸橋|白令陸橋]]和[[巴拿馬地峽|巴拿馬地峽]]至少10次洲與洲的雙向[[遷徙|遷徙]]之間，演化出8個主要的世系。其中，[[豹屬|豹屬]]是最古老的，而[[貓屬|貓屬]]則是最年輕的。他們估計大約百分之60的現存物種是在最近的100萬年內演化出來的<ref name =Mott>{{cite web| last = Mott| first = Maryann| title = Cats Climb New family Tree| publisher = National Geographic News| date = January 11, 2006| url = http://news.nationalgeographic.com/news/2006/01/0111_060111_cat_evolution.html| accessdate =  2006-07-15}}</ref>。大多數的貓科動物有18或19個[[染色體倍性|染色體倍性]]。新世界（中南美）中的貓有18個染色體倍性，這可能是導因於兩個較小的[[染色體|染色體]]結合成了一個較大的染色體之故<ref name = robinson>{{cite book| last = Vella | first = Carolyn| authorlink =  | coauthors = ''et al.'' | title = Robinson's Genetics for Cat Breeders and Veterinarians, 4th ed.| publisher = Butterworh-Heinemann | date = 2002 | location = Oxford| id = ISBN 978-0-7506-4069-5 }}</ref>。

在此發現之前，[[生物學家|生物學家]]大多不能由[[化石|化石]]的記錄中建立出貓的分類樹，因為不同物種的化石看起來都很相似，主要是尺寸上的不同。

貓科動物最親近的親戚被認為是[[麝貓|麝貓]]、[[鬣狗科|鬣狗]]和[[獴科|獴]]。所有的貓科動物都有一種基因異常，使得牠們嘗不到甜味，亦無法消化大分子醣，故成為純肉食動物。<ref name=PLOS>{{cite web| last = Xia| first = Li| coauthors = et.al.| title = Pseudogenization of a Sweet-Receptor Gene Accounts for Cats' Indifference toward Sugar| publisher = Public Library of Science| date = July 2005| url = http://genetics.plosjournals.org/perlserv/?request=get-document&doi=10.1371/journal.pgen.0010003| accessdate = 2006-07-15| deadurl = yes| archiveurl = https://web.archive.org/web/20060403162459/http://genetics.plosjournals.org/perlserv/?request=get-document&doi=10.1371%2Fjournal.pgen.0010003| archivedate = 2006-04-03}}</ref>

== 貓科動物的化石 ==
{{Portal box|生物學|貓}}
{{Commonscat|Felidae}}
{{Wikispecies|Felidae}}
已知最古老的真正貓科動物（[[始貓|始貓]]）生存在[[漸新世|漸新世]]和[[始新世|始新世]]的時期。在始新世時，牠演化出了[[假貓|假貓]]。假貓被認為是現存的兩個亞科和已滅絕的[[劍齒虎亞科|劍齒虎亞科]]等的最古老共同祖先。較以馬刀齒貓為知的這一群動物於[[更新世|更新世]]後期開始滅絕，其包括了劍齒虎、[[短劍劍齒虎屬|短劍劍齒虎]]、[[恐貓屬|恐貓]]和[[似劍齒虎屬|似劍齒虎]]。
