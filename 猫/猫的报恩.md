《'''貓的報恩'''》（{{lang-ja|猫の恩返し}}，中國大陸、臺灣、新加坡譯作《貓-{的}-報恩》，港澳地區譯作《貓-{之}-報恩》）是[[吉卜力工作室|吉卜力工作室]]2002年推出的一部[[动画|动画]]电影，片名叫做《貓的報恩》的原因就是因為故事裡的主角小春救了一隻貓王國的王子，貓王國的貓國王為了報答她而來的。故事改編自漫畫家[[柊葵|柊葵]]的漫畫。於2002年第6回[[文部省|文部省]][[文化廳媒體藝術祭|文化廳媒體藝術祭]]獲選動畫部門優秀獎。

與吉卜力另一部短篇動畫《[[Ghiblies_episode2|Ghiblies episode2]]》，為同時在戲院上映的作品。<ref>{{lang|ja|スタジオジブリの軌跡 1984-2011}}：131頁</ref>
片尾曲〈[[幻化成風|幻化成風]]〉被[[馬來西亞|馬來西亞]]女[[歌手|歌手]][[梁靜茹|梁靜茹]]翻唱成為〈[[親親|小手拉大手]]〉。

== 故事大綱 ==
內容描述一位平凡且善良的高中女生小春，在一次放學途中用棍網球球把救了一隻貓，意外開啟了一趟奇幻冒險經歷。小春救的是貓王國的王子，「貓之國度」的貓國王為了報答她，不斷送她禮物（甚至還有老鼠），並邀請小春到貓王國作客，貓國王希望她嫁給貓王子。无助的小春向「貓之事務所」的{{nihongo|貓男爵|バロン}}求助，遇到肥貓{{nihongo|胖胖|ムタ}}、烏鴉{{nihongo|多多|トト}}。突然，小春被一大群貓強行帶往貓國，貓男爵和胖胖、多多緊追進入貓國。待在舒服的貓國中，小春漸漸地長出貓鬍鬚和貓耳朵，小春快變成真正的貓了。在貓王的阻撓之下，小春、貓男爵和胖胖拼命的奔往迷宮出口，想让小春在天亮之前重回人類世界，否则将永远做猫。

== 角色介紹 ==
*'''吉岡春'''（{{lang|ja|吉岡ハル}}）
:本片主角，小名叫「小春」個性行事皆平凡且正直和善良的女高中生，有時會因糊塗而誤事。
:某日在路上解救一隻差點被車子撞上的貓後，就面臨到貓咪們為了答謝；而要將她作為貓國王子妻子的麻煩事件。
:之後在不知如何是好情況下聽到了一個神秘的聲音，要她去貓咪事務所尋找貓男爵協助。
:經歷了不同事情後喜歡上了貓男爵, 在末段向他表白, 而男爵也回答說他也喜歡她。（在之後廣美告知小春她一直暗戀的男生町田君和他的女朋友分了手, 小春卻説:"現在已經沒關係了~", 可見小春是真的喜歡上男爵。）

*'''貓男爵'''（{{lang|ja|バロン}}）
:是一位「貓咪事務所」的所長，本名為「'''佛貝魯‧馮‧吉金肯'''」（{{lang|ja|フンベルト・フォン・ジッキンゲン、Humbert von Gikkingen}}）。他是替遇上要被帶去貓王國的小春解決問題，長相帥氣，有著沉穩的紳士個性又很聰明，擁有[[騎士|騎士]]精神。一隻陶瓷貓，傳說中他非常難對付。
:在事務所裡有掛著一幅在《心之谷》故事中與貓男爵為一對的女貓露意絲的畫像。

*'''胖胖'''（{{lang|ja|ムタ}}）
:貓男爵的朋友，本名為「'''雷納多‧蒙'''」（{{lang|ja|ルナルド・ムーン、Renaldo Moon}}）。外型巨大肥胖的白貓。喜歡吃，嘴巴說的雖有點惡劣但個性不壞。其實他是被貓王國通緝的重大要犯，因為他把貓王國的魚吃的一點都不剩，此事件也是貓王國史上最惡名昭彰的事件。
:牠平時的稱呼「姆達」（{{lang|ja|ムタ}} 、故事譯『胖胖』）常被小春誤唸成「豬」（{{lang|ja|ブタ}}）。

*'''多多'''（{{lang|ja|トト}}）
:貓男爵的朋友，為一座烏鴉雕像，是隻黑色的烏鴉，常常和胖胖鬥嘴。

*'''月牙'''（{{lang|ja|ルーン}}）
:貓王國的王子。為了尋找送給戀人小雪的禮物來到人間社會，在路上要被車撞到的危急時刻被小春所救，而他父親貓國王曾說要迎娶小春，但最後月牙決定跟小雪結婚。

*'''小雪'''（{{lang|ja|ユキ}}）
:貓王國的侍女，一隻漂亮的白色母貓，和貓王國的王子相戀。小時候受過小春的幫助，而且叫小春去貓咪事務所的聲音就是她。

*'''貓國王'''（{{lang|ja|猫王}}）
:貓王國的統治者，隨意行事、任性的君王，曾經說要他兒子月牙要迎娶小春。在敗給貓男爵之後決定退位。

*'''拿多利'''（{{lang|ja|ナトリ}}）
:貓國王的秘書之一，性格拘謹、常常替貓王打定計劃。最後貓國王決定退位時也跟著同進退。

*'''拿多露'''（{{lang|ja|ナトル}}）
:貓國王的秘書之一，性格總是悠哉無慮的母貓。

*'''吉岡直子'''（{{lang|ja|吉岡直子}}）
:小春的媽媽，從事布料設計的工作。

*'''廣美'''（{{lang|ja|ひろみ}}）
:小春的朋友，有著開朗的性格，喜歡一個叫柘植的男學生。
:原作漫畫作者柊葵的單行本作品《'''{{lang|ja|耳をすませば～幸せな時間}}'''》<small>（臺灣中譯本漫畫名：-{心之谷}-）</small>裡頭另收錄的一則短篇《'''{{lang|ja|桔梗の咲く頃}}'''》<small>（桔梗花開之際）</small>中，也有一名外形相同、同樣叫作廣美、喜歡柘植的開朗女學生角色。<ref>-{心之谷}-收錄短篇：桔梗花開之際</ref>

== 製作 ==
=== 契機 ===
起初於1999年吉卜力工作室收到某間遊樂園的請託，希望能製作一篇跟貓有關的短篇動畫。
當時宮崎駿指定要讓動畫中出現《[[心之谷|心之谷]]》裡的三樣東西──「地球屋」、「貓男爵」以及胖貓「Moon」；並特別請求《心之谷》原作漫畫作者[[柊葵|柊葵]]替這件企畫案構思一篇漫畫，事後柊葵便完成了漫畫作品《貓男爵》（{{lang|ja|バロン - 猫の男爵}}）<ref name="dvd">DVD收錄短片：「{{lang|ja|猫の恩返し誕生物語}}」的訪談 </ref>，並將作品風格定位為「像是《心之谷》中女主角月島{{lang|ja|雫}}所撰寫的故事」。<ref>{{lang|ja|路地のむこうは猫の国 ─ 猫の恩返し&ギブリーズ徹底ガイド}}：38頁 </ref>

但之後當時提出要求的遊樂園取消了這件案子，宮崎駿因此退而將原本的企畫做為讓吉卜力工作室裡年輕動畫師的測試，讓合適的人選將此改編成45分左右的錄影帶動畫。事後挑中的人選為先前在長篇動畫《[[隔壁的山田君|隔壁的山田君]]》中表現優異的[[森田宏幸|森田宏幸]]，<ref>{{lang|ja|路地のむこうは猫の国 ─ 猫の恩返し&ギブリーズ徹底ガイド}}：43頁 </ref>而森田便在製作短片《[[可羅的大散步|可羅的大散步]]》時；接下了此提案。<ref>{{lang|ja|猫の恩返し ロマンアルバム}}：59頁</ref>

後續[[鈴木敏夫|鈴木敏夫]]看了森田宏幸畫完的劇情分鏡時、認為作品內容有搬上大銀幕的可行性，便進一步提議讓森田宏幸做為此動畫的導演；讓《貓的報恩》成為了吉卜力工作室另一部的電影動畫。<ref name="dvd">DVD收錄短片：「{{lang|ja|猫の恩返し誕生物語}}」的訪談 </ref>

起初作品的名稱為{{lang|ja|《ルナワーズの猫》}}。<ref>{{lang|ja|猫の恩返し ロマンアルバム}}：67頁</ref>劇情描述著一位名叫{{nihongo|「春」|ハル}}的少女為了尋找她下落不明的戀人「{{lang|ja|ジオ}}」，而請求貓男爵及胖胖協助。之後她們搭著飛行艇來到了一處名叫{{lang|ja|「ルナワーズ」}}的街道，發現那兒有著住著貓王的{{nihongo|「巴斯泰托」|ブバスティス}}神殿。而貓王有著企圖將全部人類變化成貓、佔領人間社會的計畫，之後貓男爵及胖胖便在內部有如立體迷宮的巴斯泰托神殿中設法擊敗貓王，以阻止牠的野心。<ref>{{lang|ja|猫の恩返し ロマンアルバム}}：65～67頁</ref>

之後劇情大綱修改為動畫版相似的內容，作品名稱也更改為{{nihongo|《貓男爵》|バロン─猫の男爵}}。故事中大白貓胖胖出沒的十字街為柊葵居住的[[函館市|函館市]]其中一帶的地名，但場景造型並非來自於當地實景，而是以懷舊風格的建物為主。<ref>{{lang|ja|猫の恩返し ロマンアルバム}}：69頁</ref>

== 其它 ==
=== 動畫[[彩蛋_(視覺)|彩蛋]] ===
*小春與廣美一開始走在街上時，街道對面有間寫著「森田針灸院」（此片導演姓氏）的建物。<ref>{{lang|ja|猫の恩返し ロマンアルバム}}：92頁</ref>
*貓王在王宮內的房間查看前來貓王國的小春和胖胖時，視聽器材旁邊有盒寫著{{nihongo|「種樹的貓」|木を植えた猫}}的錄影帶。原片名為加拿大動畫導演Frédéric Back改編自法國作家[[讓·紀沃諾|讓·紀沃諾]]的著作{{nihongo|《[[種樹的男人|種樹的男人]]》|木を植えた男}}同名動畫。<ref>{{lang|ja|猫の恩返し ロマンアルバム}}：55頁</ref>（後續動畫《種樹的男人》的重製版DVD由吉卜力工作室發行）

=== 軼事 ===
*作曲家[[野見祐二|野見祐二]]在處理貓王出巡到小春家段落的配樂時，原先欲搭配以埃及風格為主的配樂；但試不出好結果，之後採用[[日本雅樂|日本雅樂]]的樂器（[[笙_(日本)|笙]]、[[篳篥|篳篥]]等）才得以解決。<ref>{{lang|ja|猫の恩返し ロマンアルバム}}：78頁</ref>
*鈴木敏夫因中途才加入《貓的報恩》的製作群，在未完全熟悉作品架構的情況下；被他認為是在此之前處理預告片最困難的一次。<ref>{{lang|ja|映畫道樂}}：160～161頁</ref>
*著名馬來西亞歌手[[梁靜茹|梁靜茹]]的歌曲《小手拉大手》，由[[陳綺貞|陳綺貞]]作詞，改編自主題曲《幻化成風》。
