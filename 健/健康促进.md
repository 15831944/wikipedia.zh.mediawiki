根据[[世界卫生组织|世界卫生组织]]（WHO）的定义，'''健康促进'''是指促使人们提高與改善健康状态的过程<ref>{{cite web |url=http://www.euro.who.int/AboutWHO/Policy/20010827_2 |title=存档副本 |accessdate=2008-01-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20070730040837/http://www.euro.who.int/AboutWHO/Policy/20010827_2 |archivedate=2007-07-30 }}</ref>。在美国一般采用狭义的定义，即指帮助人们改变其生活习惯以达到理想健康状态的一门科学与艺术。<ref>http://www.healthpromotionjournal.com/]</ref>

== 渥太華五大憲章 ==
1986年WHO在加拿大渥太华召开的第一届国际健康促进大会上通过了[[:en:Ottawa_Charter|渥太华宣言]]，其中包括健康促进的基本原则<ref>{{cite web |url=http://www.euro.who.int/AboutWHO/Policy/20010827_2 |title=存档副本 |accessdate=2008-01-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20070730040837/http://www.euro.who.int/AboutWHO/Policy/20010827_2 |archivedate=2007-07-30 }}</ref>:
* 健康政策宣導落實''(政策)''
* 创造有利环境''(環境)''
* 强化社区行动''(社區)''
* 发展个人技能''(個人技能)''
* 重新调整卫生服务''(服務)''

== 健康促进的制定与实施 ==
健康促进计划的制定与实施与其他的规划一样，需要理论和实际相结合。理论是前人在经验和教训之上总结出来的。但是没有一个理论可以适应现实中所有情况。根据具体的情况，遵循一定的程序，将各种理论结合，在做出具体的微调是制定一个健康促进计划的标准模式。
===醫院健促===
*醫院的健康促進，有針對[[醫師|醫師]]、[[護理人員|護理人員]]進行健康介入，以預防員工壓力過大的情形發生。<ref>[http://www.hpa.gov.tw/Bhpnet/Web/HealthTopic/TopicArticle.aspx?No=201205300013&parentid=201110140002 國內健康促進醫院介紹--案例八 員工心理健康管理計畫]財團法人[[彰化基督教醫院|彰化基督教醫院]]</ref>
**改善方向以組織[[環境|環境]]改善、個人[[壓力調適|壓力調適]]為主
**'''介入措施：'''問卷、個案追蹤比率、輔導諮商服務。

== 健康促进的评价 ==
为了给实施类似的计划提供借鉴，以便今后的计划能够达到更好的健康促进效果。包括本计划的实施者，以及其他人可以吸取本计划的经验和教训，需要对已经进行的计划进行评价和总结。对进行的健康促进计划进行评价和总结也是给与计划利益相关人信息反馈的需要，让计划的利益相关人：政府、非政府组织、社区相关人员、以及捐款人了解健康促进计划进行的情况。
