'''健樂士'''（{{lang-it|'''Geox'''}}），是一個[[意大利|意大利]]休閒鞋履生產及零售商，產品應用了防水物料科技。<ref>[http://www.info.gov.hk/gia/general/brandhk/0407002.htm 意大利著名鞋店在港開設專門店]《香港政府新聞稿》2004年4月7日</ref>

==公司歷史==
健樂士由馬里奧·波萊加托（Mario Polegato）於1995年創立<ref>{{cite web |author=Digital PR |url=http://www.geox.biz/chi_storia_1.asp |title=portale di informazione aziendale del Gruppo Geox |publisher=Geox.biz |date= |accessdate=2011-09-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20110904014246/http://www.geox.biz/chi_storia_1.asp |archivedate=2011-09-04 }}</ref>，品牌名稱由[[希臘文|希臘文]]土地（Geo）與代表科技的「x」所組成。

波萊加托於1952年在[[意大利|意大利]]東北部城市[[特雷維索|特雷維索]]附近出生<ref>{{cite web |author=Digital PR |url=http://www.geox.biz/chi_mmp.asp |title=portale di informazione aziendale del Gruppo Geox |publisher=Geox.biz |date= |accessdate=2011-09-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20110902052153/http://www.geox.biz/chi_mmp.asp |archivedate=2011-09-02 }}</ref>，原先經營家族的[[葡萄酒|葡萄酒]]釀製[[家族生意|生意]]，有一次到雷諾出席行業會議時經過洛基山脈，走路太焗腳時得到靈感，想到於鞋底鑽挖小孔藉以幫助腳掌散熱，經過短暫的測試階段後，決意創立自己的商標。
