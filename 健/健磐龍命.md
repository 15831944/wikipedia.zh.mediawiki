'''健磐龍命'''（{{lang-ja|タケイワタツノミコト}}）為[[日本神話|日本神話]]中的[[神祇|神祇]]，也是[[阿蘇神社|阿蘇神社]]（位於[[熊本縣|熊本縣]][[阿蘇市|阿蘇市]]）的主祭神。傳說祂是[[神武天皇|神武天皇]]之子[[神八井耳命|神八井耳命]]的[[兒子|兒子]]，一般被視為[[阿蘇山|阿蘇山]]之神。

== 概要 ==
傳說健磐龍命乃神武天皇之[[孫|孫]]，被派遣前往平定[[日向國|日向]]（今[[宮崎縣|宮崎縣]]），接著從高千穗前往阿蘇。途經高森之草部（{{ja}}くさかべ），寄居在[[伯父|伯父]][[日子八井命|日子八井命]]（又名草部吉見神）家中。由於祂和草部吉見神共同努力開拓阿蘇，後者遂將其[[女兒|女]]阿蘇都比咩命（{{ja}}アソツヒメノミコト）許配給祂。

健磐龍命前往阿蘇時，從外輪山向西眺望，見到山上有面積廣大的[[火山口湖|火口湖]]，於是他使勁踢破山壁讓大量湖水流出，成為灌溉附近農地的水源。由於山壁有二層，健磐龍命使勁踢了兩次才成功，故踢破山壁之處稱作「二重峠」；而祂站立處稱為「立野」，音同「{{ja}}立てぬ」；湖水向西奔流時跟著沖出數隻[[鹿|鹿]]，故為後來的[[數鹿流瀑布|數鹿流瀑布]]（{{ja}}数鹿流ヶ滝）。後來祂教導當地人種植[[稻|稻]]米，並將收割後的米堆成一座小山，成為現在標高954[[公尺|公尺]]的米塚山。

傳說祂的家臣鬼八具有驚人怪力，因健磐龍命愛好[[日本弓術|弓術]]，鬼八負責撿拾射出的箭矢。健磐龍命總是坐在阿蘇中央火山口丘群的往生岳，朝著「的石」之地射箭。有時射偏的箭就交由鬼八撿拾，祂來來回回99次，終於到了第1百次時覺得疲累厭倦，就隨便地將箭矢夾在腳趾間丟回給健磐龍命。此舉觸怒了健磐龍命，他生氣地砍下鬼八的[[頭|頭顱]]，想不到頭顱竟與身體再度連接，所以健磐龍命將鬼八碎屍萬段，後者才終於死亡。後來阿蘇地區的農作物受[[霜|霜害]]所苦，大家開始謠傳這是鬼八的怨靈作祟。為了撫慰鬼八的冤魂，設立霜宮神社，並舉行「火焚神事」（{{ja}}火焚き神事）祭典，從8月19日的「乙女入神事」起連續59日燃火，至10月16日的「乙女上げ神事」才將火種撲滅。燃燒火焰的目的乃是為了讓鬼八感到溫暖，不會因寒冷而作怪（霜害）<ref>參見{{ja}}[http://www.pref.kumamoto.jp/site/arinomama/sinjisimomiya.html 霜宮の火焚き神事（しもみやのひたきしんじ）- 熊本県庁阿蘇市] {{webarchive|url=https://web.archive.org/web/20120421221258/http://www.pref.kumamoto.jp/site/arinomama/sinjisimomiya.html |date=2012-04-21 }}。</ref>。

== 解說 ==
自古以來，健磐龍命就是航海安全的守護神，現在也作為交通安全、除災、姻緣、學問之神。

== 信仰 ==
日本祭祀健磐龍命的[[神社|神社]]主要有下列：
* 阿蘇神社（[[熊本縣|熊本縣]][[阿蘇市|阿蘇市]]）
* 中畑神社（[[宮崎縣|宮崎縣]][[西臼杵郡|西臼杵郡]]高千穗町）：或稱「仲畑神社」。
