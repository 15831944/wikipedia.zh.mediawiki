'''闽南金三角'''指[[中國|中國]][[福建省|福建省]][[閩南|閩南]]沿海的[[厦门|厦门]]、[[泉州|泉州]]和[[漳州|漳州]]三个地级市及所辖县区。人口1700餘萬人，主要講[[閩南語|閩南語]][[泉漳片|泉漳片]]，[[財政|財政]]收入約占[[福建省|福建省]]的52%、[[進出口|進出口]]總約占[[福建省|福建省]]的70%、經濟生產量([[GDP|GDP]])佔福建省的4成，是[[中國大陸|中國大陸]]在經濟[[改革開放|改革開放]]後[[經濟|經濟]]較發達的地區之一，是[[大陆|大陆]]台商投资较密集的区域。此一名稱因此主要是用於經濟事務，常用以與[[長江三角洲|長江三角洲]]和[[珠江三角洲|珠江三角洲]]等經濟發達地區作比較<ref>{{Cite web |url = http://www.time-weekly.com/story/2014-02-20/132293.html|title = 闽南金三角 谁的同城化  }}</ref>。其中[[厦门|厦门]]依靠[[經濟特区|經濟特区]]、[[港口|港口]]成為大陸[[东南沿海|东南沿海]]重要[[港口|港口]]；[[泉州|泉州]]依靠发达的[[制造业|制造业]]，[[物流|物流]]行业蓬勃发展：而[[漳州|漳州]]正从以[[农业|农业]]為主，向[[工业|工业]]化转型<ref>{{Cite web |url = http://www.chinanews.com/df/2015/09-18/7532362.shtml |title = 福建漳州欲打造闽南“金三角”物流坚强一极  }}</ref>。

==由來==
闽南金三角是中国[[沿海经济开放区|沿海经济开放区]]之一，沿海经济开放区包括[[长江三角洲|长江三角洲]]、[[珠江三角洲|珠江三角洲]]、闽南三角地区、[[辽东半岛|辽东半岛]]（[[大连|大连]]）、[[山东半岛|山东半岛]]、[[环渤海地区|环渤海地区]]等<ref>{{Cite web |url = http://www.china10k.com/trad/history/6/64/64j/64j06/64j0613.htm |title = 沿海經濟開放區  }}</ref>。

==產業==
閩南金三角的主要產業為：[[成衣|成衣]]、[[计算机|计算机]]、[[电子器件|电子器件]]、[[建材|建材]]、[[石化|石化]]。2010年，閩三角[[工业|工业]]总产值為1兆1879亿[[人民幣|人民幣]]，占[[福建省|福建省]]总量的54%，工业增加值為3371.62亿[[人民幣|人民幣]]，占[[福建省|福建省]]的52.7%。

2015年，閩三角将形成8個主要[[價值鏈|产业链]]、13個[[聚集經濟效應|產業群聚]]。
*8個主要[[产业|产业]]链分别是[[泉州|泉州]][[石化|石化]]、[[泉州|泉州]]体育用品、[[泉州|泉州]][[纺织|纺织]]服装、[[厦门|厦门]][[计算机|计算机]]及[[通信|通信]]设备、[[厦门|厦门]]工程[[机械|机械]]、[[厦门|厦门]][[光电|光电]]产业、[[漳州|漳州]][[金属|金属]]及深加工、[[漳州|漳州]][[古雷石化|古雷石化]]。
*13個[[聚集經濟效應|產業群聚]]是[[泉州|泉州]][[纺织|纺织]][[服装|服装]]产业集群、[[泉州|泉州]][[体育用品|体育用品]]产业集群、[[厦门|厦门]][[光电|光电]]产业集群、[[厦门|厦门]][[计算机|计算机]]及通信产业集群、[[湄洲湾|湄洲湾]]南岸[[石化|石化]]产业基地、漳州[[古雷|古雷]]石化产业基地、漳州[[金属|金属]]及深加工产业基地、[[泉州|泉州]]机械装备制造产业集群、[[泉州|泉州]]休闲[[食品|食品]]产业集群、[[泉州|泉州]][[建陶|建陶]]及[[水暖器材|水暖器材]]产业集群、[[漳州|漳州]][[光伏玻璃|光伏玻璃]]及新材料产业基地、漳州[[农产品|农产品]][[加工|加工]]产业集群、[[厦门|厦门]][[汽车|汽车]]及工程[[机械|机械]]产业集群<ref>{{Cite web |url = http://zhangzhou.house.163.com/news2/120907/1/1009657-1.shtml |title = 厦漳泉将打造8条特色产业链建7个产值千亿产业群 |date = 2012年9月7日 |publisher = [[海峡导报|海峡导报]] |language = zh-hans |accessdate = 2012年9月7日 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20150701024504/http://zhangzhou.house.163.com/news2/120907/1/1009657-1.shtml |archivedate = 2015年7月1日 }}</ref>。
==同城化建設==
===厦漳城际轨道===
[[漳州市|漳州市]][[市政府|市政府]]規劃將在廈漳之間建设3条轨道线网：城际轨道2条（R1线、R3线）、城市轨道1条（M1）。

*城际R1线：自[[厦门|厦门]][[海沧区|海沧区]]沿翁角路进入[[台商|台商]]投资区，沿角江路布设穿[[福厦铁路|福厦铁路]]、跨[[沈海高速|沈海高速]]和[[九龙江|九龙江]]进入[[漳州市|漳州市]]主城区，最后止于[[南靖车站|南靖车站]]
*城际R3线，自[[厦门火车站|厦门火车站]]出发，利用在建[[港尾铁路|港尾铁路]]，经由漳州规划滨海新城，到[[龙海市|龙海市]]，从圆山大道进入圆山新城，经龙江路，延伸到北环城路与R1线衔接。 
*城鐵R3线：从[[漳州市|漳州市]]主城区接出，先后途经[[芗城|芗城]]、[[龙文|龙文]]、圆山新城、[[龙海市|龙海市]]、漳州开发区，下穿[[厦门湾|厦门湾]]海域，直达[[厦门岛|厦门岛]]。全长约72公里，投资约142亿[[人民幣|人民幣]]，站点16个，其中漳州段全长约65公里。线路设计时速120km/H，50分钟内可从[[漳州市|漳州市]]区到达[[厦门岛|厦门岛]]内<ref>{{Cite web |url = http://news.ifeng.com/a/20140802/41415860_0.shtml|title = 厦漳城际轨道进入勘察设计阶段 漳州到厦门岛内或只需50分钟  |date =  2014年8月2日 |publisher = [[海峡都市报|海峡都市报]] |language = zh-hans |accessdate = 2014年8月2日 |quote =  }}</ref>。

===厦漳同城大道===
[[厦漳同城大道|厦漳同城大道]]起点在漳州台商投资区，设[[海翔大道|海翔大道]]、角嵩路、港中路3个对接点与[[厦门|厦门]]相接，全长26 公里，主道双向6至8车道，设计速度80km/h。厦漳同城大道分为[[台商|台商]]投资区段和圆山段，起点在漳州台商投资区官路村附近，终点与规划的铁路[[漳州|漳州]]站站前路对接<ref>{{Cite web |url = http://news.xmnn.cn/a/xmxw/201402/t20140212_3708899.htm|title = 厦漳同城大道全线开工 最快明年底建成(图) }}</ref>。
