'''闽江学院'''，简称'''闽院'''，位于[[中國|中國]][[福建省|福建省]][[福州市|福州市]]，2002年经教育部批准，由原福州师范高等专科学校和原闽江职业大学合并组建的综合性本科大学。正努力建设成为一所多科性应用教学型本科大学。

== 历史沿革 ==
闽江学院前身之一的'''福州师范高等专科学校'''创办于1958年，1961年撤并，1978年5月复办，1981年7月经[[国务院|国务院]]批准为三年制师范[[专科学校|专科学校]]。

闽江学院的另一个前身'''闽江职业大学'''则是1984年成立的一所培养专业技术人才的全日制高等职业[[专科学校|专科学校]]。1990—1996年，时任福州市委书记[[习近平|习近平]]兼任闽江职业大学校长。
[[File:Minjiang_University_gymnasium,_October_2018.jpg|thumb]]
2002年3月21日教育部根据《普通高等学校设置暂行条例》和全国高等学校设置评议委员会的评议结果，批准同意在福州师范高等专科学校与闽江职业大学合并基础上建立闽江学院，同时撤销原两所学校的建制<ref>{{cite news| first=| last=| coauthors=| title=闽江学院正式获准开办| date=2002-03-28| publisher=TOM.COM| url=http://news.tom.com/Archive/2002/3/28-85499.html| work=中国新闻社| pages=| accessdate=2007-12-31| language=zh-hans| deadurl=yes| archiveurl=https://web.archive.org/web/20031227100527/http://news.tom.com/Archive/2002/3/28-85499.html| archivedate=2003-12-27}}</ref>。

2010年6月15日，成立新华都商学院。

2018年11月11日，闽江学院出台新规，[[2018年闽江学院外卖风波|禁止外卖]]进入校园，学生领取外卖须在学校门口的取餐点领取。与此同时，闽江学院还对部分课程安排以及食堂营业时间稍作调整。但由于学生、外卖商家的不满以及舆论的非议，校方放弃了这条禁令。<ref>{{cite web |title=闽江学院官方微博内容 |url=https://m.weibo.cn/5627467890/4305313522440828 |website=闽江学院官方微博 |accessdate=2018-11-12}}</ref><ref name="甲">{{cite news |title=福州闽江学院“禁止外卖” 商家发“免费午餐”抗议 |url=http://www.bjnews.com.cn/edu/2018/11/12/520656.html |accessdate=2018-11-12 |work=新京报 |date=2018-11-12}}</ref>

== 校园环境 ==
现有福州闽侯上街（大学城本部）、工业路、长乐路、洪塘4个校区，现已建有行政大楼、教学实验楼、教学大楼、艺术大楼、学术交流中心、学生公寓等建筑群，拥有图书馆和实验室等公共资源体系。建有广场、文体活动中心、艺术长廊、闽籍院士长廊、校园石刻等景观。

·闽台合作项目分为“4+0”（会去台湾地区合作院校就读一小段时间）和“3+1”（前3年在闽院就读，最后1年在台湾地区合作院校就读）两种模式，学生在闽院就读期间，部分课程将会使用到台湾地区高校教材。

== 附属中学 ==
闽江学院附属中学创建于1987年，原名福州市第二十八中学，福州师范高等专科学校附属中学，是福州市直属完全中学、省二级达标学校、闽江学院教科研实验基地。学校地处六一中路和群众东路交界处，校园面积52多亩，建筑总面积18322平方米，绿化面积6000平方米。现有初、高中46个教学班，2157多名学生和165位教职员工。

== 争议 ==
{{main|2018年闽江学院外卖风波}}
2018年11月11日，闽江学院开始禁止外卖进入校园，引发了学生以及周边商家的反弹。由于点餐、取餐不方便，学生大量前往食堂就餐，导致了食堂普遍出现拥挤现象。尤其是在11月12日中午，闽江学院第二食堂就餐人数爆满，大量学生在食堂门口排队等候。部分学校周边的外卖商家为排队等候的学生提供免费午餐，呼吁学生抵制学校制定的“霸王条款”，但遭到校方驱逐，并且被送至派出所拘留。随后，校方工作人员就此事接受了[[新京报|新京报]]、[[梨视频|梨视频]]等媒体的采访，表示学校并未禁止学生通过网络点餐，只是禁止外卖车辆进入校园。<ref name="甲"/>11月12日晚，部分学生为表达对学校禁止外卖的不满，在宿舍楼集体喊楼示威。
