《'''梵諦岡奇蹟調查官'''》（{{lang-ja|バチカン奇跡調査官}}）是一部由[[藤木稟|藤木稟]]創作的小說作品，插畫由{{tsl|ja|THORES柴本|THORES柴本}}負責。小說由KADOKAWA的角川恐怖文庫出版，漫畫版於月刊Comic Gene上連載。

== 概要 ==
起初本作是以單行本的形式於[[角川書店|角川書店]]出版，其後在{{tsl|ja|角川ホラー文庫|角川恐怖文庫}}再版。

於2016年宣佈本作動畫化的消息<ref>{{Cite web |url = http://www.animatetimes.com/news/details.php?id=1479193150 |title = 天才神父コンビの人気ミステリ『バチカン奇跡調査官』がアニメ化企画進行中！　シリーズ累計は95万部突破 |work = animate Times |date = 2016-11-15 |language = ja |accessdate = 2017-02-21}}</ref>，於2017年7月7日起由[[WOWOW|WOWOW]]以及[[東京都會電視台|東京都會電視台]]等電視台播放<ref>{{Cite web |url = http://www.animatetimes.com/news/details.php?id=1487576993 |title = 天才神父コンビの事件簿『バチカン奇跡調査官』が、2017年夏放送開始予定！　監督は、米たにヨシトモ氏に |work = animate Times |date = 2017-02-20 |language = ja |accessdate = 2017-02-21}}</ref>。

== 故事簡介 ==
在[[梵蒂岡|梵蒂岡]]內，有一個專門受理全球的「神蹟申請」並負責檢證其真偽的機關——「{{nihongo|'''聖座'''|聖徒の座}}」，而隸屬聖座的神父則會被稱作奇蹟調查官。

天才科學家'''平賀·約瑟夫·庚'''及暗號解讀專家'''羅貝多·尼可拉斯'''均是聖座的奇蹟調查官，他們經常調查在世界各地所發生的神蹟。但在調查期間，卻逐漸揭露一些被隱藏的事件及陰謀。
