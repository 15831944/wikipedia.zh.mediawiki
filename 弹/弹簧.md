'''彈簧'''是一種存儲機械能的彈性物體。彈簧通常由[[彈簧鋼|彈簧鋼]]製成。有許多彈簧設計。在日常使用中，該術語通常指的是[[彈簧|彈簧]]。利用它的弹性可以控制机件的运动、缓和冲击或震动、储蓄能量、测量力的大小等。广泛用于机器、仪表中。

當常規彈簧沒有剛度變化特性時，從其靜止位置被壓縮或拉伸時，它施加與它的長度變化成正比的相反力（此近似分解為較大的撓度）。彈簧的“速率”或“彈簧常數”是它施加的力的變化，除以彈簧的偏轉變化。 也就是說，它是力與偏轉曲線的梯度。延伸或壓縮彈簧的速率以力的單位除以距離表示。[[扭轉彈簧|扭轉彈簧]]，當它圍繞其軸扭轉一定角度時扭轉，它產生與角度成比例的[[扭矩|扭矩]]。 扭轉彈簧的速率以扭矩除以角度為單位，例如 [[牛頓米|牛頓米]] / [[弧度|弧度]]或[[英呎磅力|英呎磅力]] / [[度|度]]。彈簧率的倒數是順應性，即：如果彈簧的速率為 10 N/mm，則其彈性為 0.1 mm/N。並聯彈簧的剛度（或速率）是相加的，彈簧串聯的順應性也是如此。

彈簧由各種彈性材料製成，最常見的是[[彈簧鋼|彈簧鋼]]。小彈簧可以由預硬化的坯料捲繞，而較大的彈簧由[[退火|退火]]鋼製成並在製造後硬化。一些[[有色金屬|有色金屬]]也用於[[磷青銅|磷青銅]]和[[鈦|鈦]]用於需要耐腐蝕性的部件和[[鈹銅|鈹銅]]用於承載電流的彈簧（因為它的電阻低）。

== 歷史 ==

== 定義和特徵 ==
該對象具有稱為彈性的屬性，即使通過施加力而使其變形也會嘗試返回其原始狀態{{Sfn|日本ばね学会（編）|2008|p=1}}。彈簧的廣義上的定義，這彈性的性質的利用為主要目的的總稱。{{Sfn|渡辺・武田|1989|p=3}}。
彈簧所需的特性大致分為：
* 具有彈性
* 能量積累和釋放
* 具有特定頻率的

這3個舉出的特性，這些是“彈簧的三大特點”之稱<ref>{{Harvnb|蒲|2008|p=46}} / {{Harvnb|「ばねの歴史」編纂ワーキンググループ（編）|2012|pp=2–4}} / {{Cite web |url=http://www.nhkspg.co.jp/museum/02.html |title=ばねの基礎知識 |work=ばねの話 |publisher=日本発条株式会社 |accessdate=2016-12-29}}</ref>。除了稱為彈簧的零件和物體之外，還提供這三個特徵，但可以說使用這些特徵可以特別好地利用彈簧{{Sfn|蒲|2008|p=46}}。其他也彈簧的基本性質和工作的分法{{efn|例えば、{{Harvnb|日本ばね学会（編）|2008|pp=1–5}}、{{Harvnb|ニッパツ・日本発条株式会社（編）|1998|p=3}}、{{Harvnb|渡辺・武田|1989|pp=8–10}}。}}、在這裡，我們按照這三大類來解釋彈簧的基本特徵。

=== 恢復力 ===
[[File:Modes_deformation_cropped.svg|thumb]]
[[File:Three_kinds_of_load-deflection_curve.svg|thumb]]
當施加力時彈簧變形並且在移除力時具有恢復其原始形狀的特性{{Sfn|蒲|2008|p=46}}。彈簧的基本要求是，即使通過這種方式施力使其變形，也具有試圖恢復原狀的性質，這是必要條件。{{Sfnm|小玉|1985|1p=16|蒲|2008|2p=46}}。能夠回到原來的形狀, 被稱為 "恢復力", 恢復力的存在被列為彈簧的一大特徵<ref>{{Harvnb|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=3}} / {{Harvnb|蒲|2008|p=46}} / {{Cite web |url=http://www.nhkspg.co.jp/museum/02.html |title=ばねの基礎知識 |work=ばねの話 |publisher=日本発条株式会社 |accessdate=2016-12-29}}</ref>。

恢復力是指物質的“[[彈性|彈性]]”，並且在移除力時恢復到其原始形狀的變形稱為“彈性變形”{{Sfnm|「ばねの歴史」編纂ワーキンググループ（編）|2012|1p=3|村上|1994|2p=11}}。然而，當力（更精確地，[[應力|應力]]）施加了超過該材料的限制，所以留下了（更確切地說，[[應變_(物理學)|應變]]）甚至不包括力變形{{Sfn|村上|1994|p=11}}。該特性稱為“[[塑性變形|塑性變形]]”，由於塑性的性質而不能恢復到原始狀態的變形稱為“塑性變形”。{{Sfnm|蒲|2008|1p=42|村上|1994|2p=11}}。彈性變形中變形保持的最大應力稱為“彈性極限”<ref>{{cite book |和書 |author=大路清嗣・中井善一 |title=材料強度 |publisher=コロナ社 |year=2010 |edition=第1版 |isbn=978-4-339-04039-5 |pages=40-41}}</ref>。由於彈簧是回到原來的前提,使用的塑性變形是用在受力範圍內適用于彈簧的理想,一般不超過彈性極限{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=2}}。

[[File:Hookes-law-springs_2.png|thumb]]
彈簧的變形和變形量為“彎曲”{{Sfnm|小玉|1985|1p=14|「ばねの歴史」編纂ワーキンググループ（編）|2012|2p=3}}。有兩種類型的[[計量單位|計量單位]]的偏轉，[[位移|位移]]（長度變化）和[[轉動|旋轉角度]]（扭轉角度和彎曲角度的變化）{{Sfn|「JIS B 0103」|2015|p=15}}。在壓縮螺旋彈簧中，利用長度變化的事實，偏轉單位由位移表示{{Sfn|ばね技術研究会（編）|2001|p=1}}。棒のねじり角度が変化することを利用する[[トーションバー|トーションバー]]では、たわみの単位は回転角（ねじり角）である{{Sfn|「JIS B 0103」|2015|p=15}}。對應於物理偏轉量，可以想到幾種引起偏轉的載荷。如果是位移，則是[[結構荷重|結構荷重]]（純[[力|力]]），如果是扭轉角，則考慮[[力矩|扭轉力矩]]{{Sfn|日本ばね学会（編）|2008|pp=1–2}}。一些實際的彈簧會引起復雜的偏轉以及位移和旋轉變形{{Sfn|蒲|2008|p=50}}。



彈簧的基本特性和功能之一是這種載荷和撓度具有一定的關係{{Sfn|日本ばね学会（編）|2008|p=1}}。由彈簧指示的載荷和撓度之間的關係稱為“彈簧特性”，“載荷 - 撓度特性”，“載荷特性”等{{Sfnm|「JIS B 0103」|2015|1p=13|ニッパツ・日本発条株式会社（編）|1998|2pp=5–6}}。彈簧最常用的彈簧特性通常是線性的。偏轉並線性負載[[比例|比例]]，它增加和在10中的彈簧減小公斤的重量掛在彈簧1[[厘米|厘米]]的伸長率，並且使得懸浮延伸20kg的重量2厘米{{Sfn|蒲|2008|p=51}}。這種關係也被稱為“[[胡克定律|胡克定律]]”{{Sfn|山田|2010|p=9}}。對於作為線性特性的彈簧，負載和偏轉之間的關係由下式表示。

:<math>P = k \delta</math>

這裡、''P'' 在有負載（力）、''δ'' 偏轉（位移）。''k'' 這是 ''P'' 和 ''δ'' “[[彈簧常數|彈簧常數]]”在“功率常數”的比例常數中，單位為[[力|力]] / [[長度|長度]]{{Sfn|小玉|1985|p=14}}。例如，彈簧常數為10 [[kgf|kgf]] / [[公分|公分]]意味著必須懸掛10公斤的重量才能產生1厘米的偏差。{{Sfn|蒲|2008|p=51}}。說到實際產品，大型汽車和鐵路車輛[[懸掛|懸掛]]的彈簧需要大的彈簧常數，與[[床|床]]和[[沙發|沙發]]的彈簧相比，彈簧常數小是必需的{{Sfnm|小玉|1985|1pp=14–15|蒲|2008|2p=51}}。

當負載是扭矩 ''T'' 時，偏轉是扭轉角 ''θ'' 當

:<math>T = k \theta</math>

這個公式。在這種情況下的''k''“力矩”/“角度”的單位、''k'' 可以稱為“旋轉彈簧常數”等，以將其與正常彈簧常數區分開{{Sfn|渡辺・武田|1989|p=8}}。

還有一個彈簧，其載荷和撓度不成比例，這種關係稱為[[非線性|非線性]]{{Sfn|山田|2010|p=45}}。在的非線性特性的彈簧，例如，彈簧10[[千克|千克]]的懸掛時延伸1厘米的重量，但這樣的懸浮20公斤1.2厘米只伸長的重量{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=3}}。此外，當施加和移除負載時，負載和偏轉之間的關係是不同的，並且存在負載 - 偏轉曲線繪製[[遲滯現象|遲滯現象]]的彈簧{{Sfn|日本ばね学会（編）|2008|p=2}}。作為具有非線性特徵的彈簧的一個例子，提到了[[碟型墊片|碟型墊片]]和壓縮螺旋彈簧中的特殊彈簧{{Sfn|山田|2010|p=45}}。

=== 能量累積和釋放 ===
[[File:Carina_Christiansen,_2012.jpg|thumb]]
[[File:復元力と弾性エネルギー.svg|thumb]]
當彈簧變形時，能量以[[彈性能量|彈性能量]]的形式存儲在彈簧中{{Sfn|門田|2006|p=164}}。如果釋放儲存的能量，可以使彈簧機械[[功|機械功]]{{Sfn|小玉|1985|p=19}}。這種“能量積累和釋放”功能被列為彈簧的第二個主要特徵<ref>{{Harvnb|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=3}} / {{Harvnb|蒲|2008|p=46}} / {{Cite web |url=http://www.nhkspg.co.jp/museum/02.html |title=ばねの基礎知識 |work=ばねの話 |publisher=日本発条株式会社 |accessdate=2016-12-29}}</ref>。例如，通過[[弓|弓]]釋放[[箭頭|箭頭]]利用這種能量積累和釋放{{Sfn|蒲|2008|p=47}}。在主發條時鐘，手錶在移動時釋放[[發條|發條]]中存儲的能量{{Sfn|マコーレイ|2011|p=79}}。與弓形相比，在主發條手錶的情況下，它在釋放時逐漸使用彈性能量{{Sfn|蒲|2008|p=47}}。在用於汽車的[[懸掛裝置|懸掛裝置]]的彈簧的情況下，彈簧受到從路面傳遞的衝擊，並且通過將衝擊力改變為彈簧的彈性能量來緩沖沖擊力。{{Sfn|蒲|2008|p=48}}。

儲存在彈簧的彈性能量等於使所述彈性變形由負載所做的工作{{Sfn|日本機械学会（編）|2007|p=1084}}。在載荷-撓度圖中，由曲線和水平軸包圍的區域對應於彈性能量{{Sfn|日本ばね学会（編）|2008|p=2}}。不限於線性特徵，負載 ''P'' 撓度 ''δ'' 當它是一般[[函數|函數]]時、 ''P''(''δ'') [[積分|積分]]和彈性能量 ''U'' 如下{{Sfn|日本ばね学会（編）|2008|p=2}}。

:<math>U = \int P(\delta) d\delta</math>

如果它是線性特徵的彈簧，因為封閉區域是三角形

:<math>U = \frac{P\delta}{2} =\frac{k \delta^2}{2}=\frac{P^2}{2k}</math>

是彈性能量{{Sfn|村上|1994|pp=24–25}}。負載施加在彈簧上 ''P'' 是相同的，彈簧常數 ''k'' 吸收的能量越小 ''U'' 可以增加{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=2}}。當彈簧用於減輕鐵路車輛的[[耦合器|耦合器]]或[[緩衝裝置|緩衝裝置]]的碰撞時，吸收的能量越大，越有利{{Sfn|小玉|1985|p=19}}。

在負載-偏轉曲線繪製磁滯迴線的非線性特性彈簧的情況下，與環繞的部分的面積相對應的能量被摩擦等消耗。{{Sfn|小玉|1985|pp=19–20}}。通過該滯後消耗彈性能量作為[[衰減振動|衰減振動]]，從衝擊鬆弛的觀點出發，環路所包圍的面積越大，越有利。{{Sfnm|日本ばね学会（編）|2008|1p=165|小玉|1985|2p=20}}。

===固有频率===
[[File:Federpendel-mod.gif|thumb]]
[[File:Harmonic_oscillator_gain.svg|thumb]]
將砝碼懸掛在天花板上，將砝碼向下拉，釋放力。 然後重量以恆定[[頻率|頻率]]上下振盪{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=4}}。這個恆定頻率稱為“[[簡正模|固有頻率]]”{{Sfn|蒲|2008|p=48}}。在由線性特徵的彈簧，[[質點|質點]]（重量）和基礎（天花板）組成的一個自由度系統中，固有頻率為

:<math>f_n = \frac{1}{2\pi} \sqrt{\frac{k}{m}}</math>

該{{Sfn|末岡ら|2002|p=18}}。''m'' 重量是[[質量|質量]]、''k'' 彈簧常數、''π'' 是[[圓周率|圓周率]]、''f<sub>n</sub>'' 是固有頻率{{Sfn|末岡ら|2002|p=18}}。具有這樣的固有頻率是彈簧的大特徵中的第三個<ref>{{Harvnb|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=4}} / {{Harvnb|蒲|2008|p=46}} / {{Cite web |url=http://www.nhkspg.co.jp/museum/02.html |title=ばねの基礎知識 |work=ばねの話 |publisher=日本発条株式会社 |accessdate=2016-12-29}}</ref>。上在等式中、''k'' 越大 ''f<sub>n</sub>'' 變得更大、''k'' 越小 ''f<sub>n</sub>''變得更小。一般來說，彈簧越硬，固有頻率越高，彈簧越軟，固有頻率越小{{Sfn|蒲|2008|p=49}}。

固有頻率實際上與[[振動|振動]]的任何問題有關，當考慮振動問題時，固有頻率也被稱為最重要的物理量{{Sfn|末岡ら|2002|p=18}}。特別是，當尺寸和方向周期性波動的力被加到質點或支撐彈簧的基礎本身周期性擺動時，當來自外部的這種頻率與固有頻率一致時，“[[共振|共振]]“被稱為”振動“猛烈地發生{{Sfn|下郷・田島|2002|pp=46–47, 57–58}}。也有積極使用共振的機器和工具，但通常有必要避免共振{{Sfn|蒲|2008|p=49}}。當發生共振時，機器的操作可能變得不穩定，可能導致故障，最壞的可能導致故障事故{{Sfn|末岡ら|2002|pp=26–27}}。因此，需要設計機器和結構，以便從外部改變固有頻率和頻率{{Sfn|蒲|2008|p=49}}。

同時，通過利用彈簧的固有振動的特性，可以減輕振動的傳遞{{Sfnm|門田|2006|1p=162|ばね技術研究会（編）|1998|2p=79}}。當固有頻率遠小於來自外部的頻率時，振動不太可能傳遞到彈簧支撐的質點{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=4}}。通過使用它，可以減小由彈簧支撐的物體的振動{{Sfn|KYB株式会社（編）|2013|p=80}}。作為使振動不太可能傳遞的一般標準，希望使固有頻率為來自外部的頻率的1/3或更小{{Sfn|日本ばね学会（編）|2008|p=5}}。例如，在鐵路車輛中，彈簧常數可以比金屬彈簧[[空氣彈簧|空氣彈簧]]小，以提高乘坐舒適性<ref name="宮本">{{Cite book |和書 |author=宮本昌幸 |title=図解・鉄道の科学 |edition=初版 |year=2006 |publisher=講談社 |series=ブルーバックス |isbn=4-06-257520-5 |pages=28-30}}</ref>。

== 類型 ==
彈簧種類繁多。 有各種分類方法，沒有決定性的東西{{Sfn|日本ばね学会（編）|2008|pp=5–8}}。在下文中，將主要說明按類型和材料類型的類型，並且將觸及其他分類。

=== 根據負載力的施加方式進行分類 ===
* '''拉伸/拉伸彈簧'''：彈簧設計用於拉伸載荷，因此彈簧在載荷施加時伸展。
* '''壓縮彈簧'''：設計用於在壓縮載荷下運行，因此彈簧在施加載荷時會變短。
* '''扭力彈簧'''：不同於上述類型，其中負載是軸向力，施加到扭轉彈簧的負載是扭矩或扭力，彈簧的末端旋轉通過 施加載荷時的角度。
* '''恆力彈簧'''：支撐載荷在整個撓曲周期內保持不變。<ref>[http://www.pipingtech.com/products/constant-springs.htm Constant Springs] Piping Technology and Products, (retrieved March 2012)</ref>
* '''可變彈簧'''：線圈對負載的阻力在壓縮過程中變化。<ref>[http://www.pipingtech.com/products/variable-springs.htm Variable Spring Supports] Piping Technology and Products, (retrieved March 2012)</ref>
* '''可變剛度彈簧'''：線圈對負載的阻力可以動態地改變，例如通過控制系統，這些彈簧的某些類型也改變它們的長度，從而也提供致動能力。<ref>{{cite journal|url=https://patents.google.com/patent/WO2017077541A9/en|title=Springs with dynamically variable stiffness and actuation capability|author=|date=3 November 2016|website=google.com|accessdate=20 March 2018}}</ref>

=== 根據它們的形狀進行分類 ===
* '''扁平彈簧'''：這種類型由扁平[[彈簧鋼|彈簧鋼]]製成。
* '''機加工彈簧'''：這種類型的彈簧是通過加工棒料和車床銑削操作而不是捲繞操作來製造的。由於它是機加工的，彈簧可以包括除彈性元件之外的特徵。機加工彈簧可以在典型的壓縮/伸展，扭轉等負載情況下製造。
* '''蛇形彈簧'''：一種粗鋸的粗線 - 經常用於現代室內裝潢/家具。

=== 最常見的彈簧類型 ===
* '''懸臂彈簧'''：僅在一端固定的彈簧。
* '''線圈彈簧'''或''''螺旋彈簧'''：彈簧（通過繞圓柱繞線製成）有兩種類型：
** “張力”或“拉伸彈簧”：設計為在負載下變長。它們的轉動（環）通常在卸載位置接觸，並且它們在每端具有鉤，眼或一些其他連接方式。
** “壓縮彈簧”：設計為在裝載時變短。它們的轉彎（環）在卸載位置沒有接觸，它們不需要連接點。
** “空心管彈簧”：可以是拉伸彈簧或壓縮彈簧。中空管填充有油以及改變管內流體靜壓力如膜或微型活塞等硬化或放鬆彈簧的裝置，例如隔膜或微型活塞等，以使彈簧變硬或鬆弛，例如隔膜或微型活塞等，以使彈簧變硬或鬆弛，或者，管道的橫截面選擇的形狀是當管道受到扭轉變形時它會改變其面積-橫截面積的變化轉化為管道內部容積的變化以及可以進出彈簧的油流量由閥控制，從而控制剛度。中空管的彈簧還有許多其他設計，它們可以改變任何所需頻率的剛度，通過倍數改變剛度或者除了彈簧質量之外還像線性致動器一樣移動。    

* '''錐形彈簧'''：錐形壓縮螺旋彈簧，使得在壓縮時線圈不會相互擠壓，從而允許更長的行程。
* '''髮細彈簧'''或'''平衡彈簧'''一種精緻的螺旋彈簧，用於[[手錶|手錶]]，[[電流計|電流計]]和電源必須攜帶到部分旋轉裝置（如方向盤）而不妨礙旋轉的地方。
* '''板簧'''：用於車輛[[懸吊系統|懸吊系統]]，電氣開關和弓箭的平板彈簧。
* '''V型彈簧'''：用於古董槍械機制。也是門鎖彈簧，用於古董門閂機構。<ref>{{cite web|url=https://www.springmasters.com/sp/door-lock-springs.html|title=Door Lock Springs|author=|date=|website=www.springmasters.com|accessdate=20 March 2018}}</ref>

=== 其他類型包括 ===
* '''[[碟型墊片|碟型墊片]]'''或叫'''碟形彈簧'''：一種圓盤形彈簧，通常用於向螺栓施加張力。
* '''氣彈簧'''：壓縮氣體體積
* '''Ideal Spring'''：物理學中使用的概念彈簧，它沒有重量、質量或阻尼損耗。彈簧施加的力與彈簧從其鬆弛位置伸展或壓縮的距離成正比。<ref>{{Cite web|url = http://www.physics.buffalo.edu/phy101ab/Section_B/Lect21/Lect21.pdf|title = Ideal Spring and Simple Harmonic Motion|date = |access-date = 2016-01-11|website = |publisher = |last = |first = }}</ref>
* '''[[發條彈簧|發條彈簧]]'''：螺旋形帶狀彈簧，用作發條機構中的動力存儲器：手錶，鐘錶，音樂盒，發條玩具和機械手電筒
* '''恒壓彈簧'''：橫截面略微凹的薄金屬帶。當捲繞時，它採用扁平的橫截面，但當展開時，它恢復到原來的曲線，從而在整個位移過程中產生恒定的力，並“抵消”任何重新纏繞的趨勢。最常見的應用是[[捲尺|捲尺]]。<ref>{{Cite book|last=Samuel|first= Andrew|author2=Weir, John  |title=Introduction to engineering design: modelling, synthesis and problem solving strategies|publisher=Butterworth|location=Oxford, England|year=1999|edition=2|page=134|isbn=0-7506-4282-3}}</ref>
* '''漸進式螺旋彈簧'''：一種具有可變速率的螺旋彈簧，通常通過具有不等的節距來實現，使得當彈簧被壓縮時，一個或多個線圈靠在其相鄰的位置上。
* '''[[橡皮筋|橡皮筋]]'''：拉伸彈簧，通過拉伸材料儲存能量。
* '''彈簧墊圈'''：用於沿[[緊固件|緊固件]]的軸施加恆定的拉力。
* '''扭力彈簧'''：任何彈簧設計扭曲而不是壓縮或延伸。<ref>{{Cite book|url=https://books.google.com/books?id=eMqVygLDSa0C&pg=PA577|title=Technical Drawing|last=Goetsch|first=David L.|date=2005|publisher=Cengage Learning|isbn=1401857604|language=en}}</ref>用於扭杆車輛懸架系統。
* '''波形彈簧'''：許多波形彈簧，墊圈和膨脹器中的任何一種，包括線性彈簧 - 所有這些都通常由扁平線或圓盤製成，根據工業術語，通常通過模壓，“波紋化”成波浪狀規則圖案 導致曲線狀的裂片。還存在圓形線波形彈簧。

=== 材料別 ===
有各種材料產生彈簧恢復力{{Sfn|日本ばね学会（編）|2008|pp=5–6}}。原則上，所有具有彈性的材料都可以是彈簧材料{{Sfn|日本機械学会（編）|2007|p=1042}}。當按材料分類時，它大致分為金屬彈簧和非金屬彈簧，並且如下分類為一個示例{{Sfn|日本ばね学会（編）|2008|pp=5–6}}。
* [[金屬|金屬]]彈簧
** [[鋼鐵|鋼鐵]]彈簧（[[碳鋼|碳鋼]]、[[合金鋼|合金鋼]]、[[不銹鋼|不銹鋼]]等）
** [[有色金屬|有色金屬]]彈簧（[[銅合金|銅合金]]、[[鎳合金|鎳合金]]、[[鈦合金|鈦合金]]等）
* [[非金屬|非金屬]]彈簧
** [[高分子|高分子]]彈簧（天然橡膠、[[塑膠|塑膠]]、[[纖維強化塑膠|纖維強化塑膠]]等）
** [[無機化合物|無機化合物]]彈簧（[[陶瓷材料|陶瓷]]）
** [[流體|流體]]彈簧（[[空氣|空氣]]、[[非活性氣體|惰性氣體]]等）

==== 金屬彈簧 ====
[[File:Trampoliinin_jousia.jpg|thumb]]）的拉伸螺旋彈簧]]
當彈簧材料分為金屬和非金屬時，除特殊情況外，通常使用金屬彈簧{{Sfn|山田|2010|p=21}}。雖然成本低，但金屬彈簧的一個優點是它可以承受很大的力並且可以確保大的撓曲量{{Sfn|山田|2010|p=21}}。在金屬材料中，[[鋼|鋼]]材料由於其高強度和多功能性而特別廣泛地使用{{Sfn|日本ばね学会（編）|2008|p=20}}。彈簧用的鋼材是「[[彈簧鋼|彈簧鋼]]」也由名稱已知的，它是為了增加彈性極限增強材料中的比典型的鋼的碳濃度<ref>{{Cite web |url=https://kotobank.jp/word/%E3%81%B0%E3%81%AD%E9%8B%BC-115819#E6.97.A5.E6.9C.AC.E5.A4.A7.E7.99.BE.E7.A7.91.E5.85.A8.E6.9B.B8.28.E3.83.8B.E3.83.83.E3.83.9D.E3.83.8B.E3.82.AB.29 |title= ばね鋼 日本大百科全書(ニッポニカ)の解説 |author=小学館 |work=コトバンク |publisher=朝日新聞社/VOYAGE GROUP |accessdate=2017-01-02}} / {{cite book |和書 |editor=日本熱処理技術協会 |title=熱処理ガイドブック |publisher=大河出版 |year=2013 |edition=4版 |isbn=978-4-88661-811-5 |page=10}}</ref>{{efn|ばね鋼とは、後述の熱間成形用のばね用鋼材のみを指す場合もある<ref>{{Harvnb|ニッパツ・日本発条株式会社（編）|1998|p=8}} / {{cite book |和書 |author=山方三郎 |title=図解入門　よくわかる最新熱処理技術の基本と仕組み |publisher=秀和システム |year=2010 |edition=第2版 |isbn=978-4-7980-2573-5 |page=42}}</ref>。}}。彈簧鋼大致分為[[冷成型|冷成型]]和[[熱成型|熱成型]]<ref>{{Cite web |url=https://kotobank.jp/word/%E3%81%B0%E3%81%AD%E9%8B%BC-115819#E6.97.A5.E6.9C.AC.E5.A4.A7.E7.99.BE.E7.A7.91.E5.85.A8.E6.9B.B8.28.E3.83.8B.E3.83.83.E3.83.9D.E3.83.8B.E3.82.AB.29 |title= ばね鋼 日本大百科全書(ニッポニカ)の解説 |author=小学館 |work=コトバンク |publisher=朝日新聞社/VOYAGE GROUP |accessdate=2017-01-02}}</ref>。冷成形適合於通過在室溫狀態下將材料加工成彈簧形狀來形成相對小的彈簧{{Sfnm|小玉|1985|1pp=37–38|ばね技術研究会（編）|2000|2p=90}}。熱成型適合於通過在加熱到高溫的狀態下將材料加工成彈簧形狀來形成相對大的彈簧{{Sfnm|蒲|2008|1p=84|ばね技術研究会（編）|2000|2p=90}}。作為彈簧鋼的種類，使用以碳為主要添加元素的[[碳鋼|碳鋼]]，或者特別添加碳以外的元素的[[合金鋼|合金鋼]]{{Sfn|ばね技術研究会（編）|2000|pp=34, 110}}。[[不銹鋼|不銹鋼]]、[[耐腐蝕性|耐腐蝕性]]和[[耐熱性|耐熱性]]優異的材料用作其他鋼材{{Sfn|Oberg et al.|2012|p=306}}。

[[銅合金|銅合金]]、[[黃銅|黃銅]]、[[磷青銅|磷青銅]]、[[西白|西白]]、[[鈹銅|鈹銅]]等材料常用作彈簧所用的有色金屬材料{{Sfn|ニッパツ・日本発条株式会社（編）|1998|pp=14–16}}。利用銅合金[[導電率|導電率]]的優點，用[[連接器|連接器]]等減少電阻和發熱{{Sfn|ばね技術研究会（編）|2000|p=188}}。除了耐腐蝕性和非磁性之外，還存在與鋼材相比成本高的缺點{{Sfnm|小玉|1985|1p=48|Oberg et al.|2012|2p=307}}。

作為另一種有色金屬材料，[[鎳合金|鎳合金]]具有優異的耐腐蝕性，耐熱性和[[耐寒性|耐寒性]]也可用作彈簧材料{{Sfn|Oberg et al.|2012|p=308}}。特別是[[英高鎳合金|英高鎳合金]]在鎳合金中很常見{{Sfn|Oberg et al.|2012|p=308}}。鎳合金材料用作彈簧，用於高於400 ℃的高溫區域{{Sfn|蒲|2008|p=92}}。[[鈦合金|鈦合金]]也用於彈簧作為與鋼相比可以製造得更輕的材料{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|pp=235–236}}。與鋼相比，鈦合金的[[彈性模量|彈性模量]]和[[比重|比重]]更小，因此可以減輕彈簧的重量{{Sfn|日本ばね学会（編）|2008|p=112}}。另一方面，存在成本高的缺點{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=236}}。

==== 非金屬彈簧 ====
[[File:Pryžová_pružina_tlačná.jpg|thumb]]
[[File:Lufe_schnitt.png|thumb]]
當希望獲得用金屬材料無法實現的功能和性能時，非金屬材料用作彈簧材料{{Sfn|山田|2010|p=21}}。[[塑料|塑料]]和[[橡膠|橡膠]]等[[聚合物材料|聚合物材料]]）也用作彈簧材料。 利用橡膠彈性的彈簧特別稱為“[[橡膠彈簧|橡膠彈簧]]”{{Sfn|「JIS B 0103」|2015|p=3}}。橡膠的彈性是非線性的，並且只有在[[應變|應變]]小的範圍內才可以認為是線性的{{Sfn|日本ばね学会（編）|2008|p=594}}。具體材料包括通用[[天然橡膠|天然橡膠]]，高耐候性[[氯丁橡膠|氯丁橡膠]]，良好的減振特性[[丁基橡膠|丁基橡膠]]等{{Sfn|ばね技術研究会（編）|2000|pp=243–245}}。與金屬彈簧相比，它具有可以根據方向自由調節彈簧常數的優點，在橡膠的內部摩擦變形時產生阻尼力。{{Sfn|KYB株式会社（編）|2013|p=89}}。它廣泛用作車輛和工業機械的“防振橡膠”{{Sfn|日本ばね学会（編）|2008|p=593}}。另一方面，還有一個缺點是性能在高溫和低溫下趨於惡化，[[蠕變|蠕變]]很可能在長期的重負荷下發生{{Sfn|門田|2016|p=80}}。此外，由於不能清楚地計算橡膠彈簧的性能，因此需要在近似範圍內進行計算{{Sfn|Spotts et al.|2004|pp=301–302}}。

塑料材料也用於彈簧。與金屬彈簧相比，塑料彈簧具有重量輕，不腐蝕，易於成型等優點{{Sfn|ばね技術研究会（編）|2000|p=237}}。另一方面，存在如橡膠那樣可能發生蠕變的缺點，並且強度和彈性模量小於鋼{{Sfnm|ばね技術研究会（編）|2000|1p=237|ばね技術研究会（編）|1998|2p=49}}。在塑料材料中，[[工程塑料|工程塑料]]在彈簧中很常見{{Sfn|日本ばね学会（編）|2008|p=119}}。例如，由[[聚醚醚酮|聚醚醚酮]]（PEEK）等製成的螺旋彈簧用於需要耐化學性的地方<ref>{{Harvnb|門田|2016|p=102}} / {{Cite web2 |url=http://www.plabane.co.jp/page/case_study |title=プラばねの活用事例 |publisher=株式会社プラばね |accessdate=2017-02-18}}</ref>。

為了克服塑料的低強度，含有增強纖維的[[纖維強化塑膠|纖維強化塑膠]]（[[FRP|FRP]]）也用作彈簧材料{{Sfn|小玉|1985|p=51}}。有兩種FRP用作彈簧材料：[[玻璃纖維強化塑膠|玻璃纖維強化塑膠]]（GFRP）和[[碳纖維強化聚合物|碳纖維強化聚合物]]（CFRP）{{Sfn|小玉|1985|pp=51–52}}。取決於增強纖維的取向，FRP具有強度和彈性模量根據施加力的方向而不同的特性。<ref>{{Cite web2 |url=http://www.me.sophia.ac.jp/~suemasu/files/composite.pdf |title=複合材料の力学と破壊について |author=末益博志 |accessdate=2017-02-18}} / {{Harvnb|ばね技術研究会（編）|2000|p=252}}</ref>。因此，為了優化彈簧常數並利用FRP所具有的高強度，有必要設計具有適當方向的彈簧。{{Sfn|ばね技術研究会（編）|2000|p=252}}。雖然重量減輕GFRP製成板簧它曾經被投入實際使用的汽車底盤系統，但缺點的是成本高和回收困難{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|pp=236–237}}。CFRP也用作板簧作為代表性示例{{Sfn|ばね技術研究会（編）|2000|p=250}}。與其他材料相比，CFRP具有[[比強度|比強度]]和比彈性模量特別優異，以及高[[金屬疲勞|金屬疲勞]]等優點{{Sfn|ばね技術研究会（編）|2000|p=250}}。利用這些優點，已經嘗試將CFRP彈簧應用於其他材料不可能的應用{{Sfn|ばね技術研究会（編）|1998|p=57}}。

[[陶瓷|陶瓷]]無機材料也用作彈簧。 它具有耐熱性，即使在700°C至1000°C的高溫下也能實際使用，現有金屬彈簧無法處理{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=187}}。陶瓷是[[脆性材料|脆性材料]]，即使小的缺陷也被破壞，強度變化很大，因此以前認為它不適合作為彈簧材料{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=185}}。隨著隨後的製造技術的進步，高強度陶瓷誕生並且實際上可用作彈簧{{Sfn|ばね技術研究会（編）|2000|p=255}}。實際使用例子，氮化矽被用來在高溫下彈簧夾具中所使用{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|pp=253–254}}。

還有一個利用氣體和液體[[流體|流體]]的彈簧，特別是利用空氣彈性的彈簧稱為“[[空氣彈簧|空氣彈簧]]”{{Sfn|「JIS B 0103」|2015|p=3}}。在恆定溫度下，氣體的體積與壓力成反比[[波義耳定律|波義耳定律]]是創造空氣彈簧彈性的基本原則{{Sfn|蒲|2008|p=98}}。彈簧高度，可接收的負載，彈簧常數可以獨立設置，通過提供[[絞り|絞り]]可以產生阻尼力，通過提供調節閥，彈簧高度可以保持恆定，它有優勢{{Sfn|日本機械学会（編）|2005|p=135}}。特別是，由於第一個優點，在相同條件下彈簧常數可以比金屬彈簧小，並且當用作車輛的懸架裝置時，可以提高乘坐舒適性{{Sfnm|渡辺・武田|1989|1p=69|KYB株式会社（編）|2013|2p=95}}。根據形狀，可大致分為兩種“形状によって”、“ベローズ形”{{Sfn|日本ばね学会（編）|2008|p=592}}。作為缺點，與金屬彈簧相比，結構複雜，還需要除空氣彈簧之外的附屬裝置，並且成本高{{Sfn|門田|2016|p=78}}。

還有一個彈簧使用惰性氣體，如[[氬|氬]]和[[氦|氦]]，而不是空氣，這種彈簧稱為“[氣彈簧]”{{Sfn|「JIS B 0103」|2015|p=3}}。彈簧特性設定的自由度高，可以節省空間，可以發揮大的負荷{{Sfn|日本ばね学会（編）|2008|p=596}}。另一方面，工作溫度存在限制，存在氣體洩漏的擔憂{{Sfn|日本ばね学会（編）|2008|p=596}}。

==== 磁性彈簧 ====
雖然它不使用彈性，但也有一個稱為“磁彈簧”的彈簧，它使用磁鐵的[[磁力|磁力]]作為恢復力{{Sfn|ばね技術研究会（編）|1998|p=77}}。當使磁鐵的相同磁極靠近它時產生排斥力時，它可以用作在壓縮方向上具有恢復力的彈簧{{Sfn|日本ばね学会（編）|2008|p=607}}。當磁鐵的相對磁極彼此面對時，由於當磁鐵在橫向方向上移位時產生吸力，它可以用作在橫向上具有恢復力的彈簧{{Sfn|日本ばね学会（編）|2008|p=607}}。有一個優點是可以避免物體之間的接觸並且沒有質量，因此不會發生下面描述的波動{{Sfn|ばね技術研究会（編）|1998|p=77}}。

=== 其他分類 ===
除了上述基本形狀/材料類型之外，彈簧也從以下觀點分類：
* 負載形式
:根據彈簧接收的負載的類型（類型）進行分類。 有承受軸向壓縮載荷的“壓縮彈簧”，承受軸向拉伸載荷的“拉伸彈簧”，繞軸承受扭矩的“扭轉彈簧”{{Sfn|「JIS B 0103」|2015|p=2}}。

* 應力狀態
:承保接受場合，絮狀行為狀態初始分類。 由於實際應力狀態是各種應力的複雜組合，因此主要根據其接收的程度進行分類。 例如，板簧主要應用於受到彎曲應力的彈簧，壓縮螺旋彈簧主要應用於受到扭轉應力的彈簧，環形彈簧主要應用於受拉伸/壓縮應力的彈簧{{Sfn|日本ばね学会（編）|2008|p=7}}。

* 彈簧特性
:基於負載和彈簧偏轉之間的關係（彈簧特性）進行分類。 [[線性|線性]]特性，沒有[[滯後|滯後]]特性的[[非線性|非線性]]和具有滯後的非線性特性。 例如，扭杆用於線性特徵彈簧，錐形螺旋彈簧（一種壓縮螺旋彈簧）用於非滯後的非線性特性，而板簧用於具有滯後的非線性特性{{Sfn|日本ばね学会（編）|2008|p=8}}。

* 材質形狀
:由作為彈簧材料的材料形狀分類。 它們可大致分為使用板狀材料（板材），棒狀材料（桿材料）或使用線性材料（線材）的彈簧的彈簧。 例如，螺旋彈簧用於使用板材的彈簧，螺旋彈簧使用桿材料或線材施加到彈簧上{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=5}}。

== 受力與變形 ==
'''弹簧'''的受力与变形符合[[虎克定律|虎克定律]]，即受力與形變量需成正比，见下式:
:* <math>F=-kx</math>
::* <math>F</math>=弹簧所产生的力单位为[[牛顿|牛顿]]，简称[[牛|牛]]
::* <math>k</math>=彈簧常數，又称倔强系数或劲度系数，单位为[[牛顿/米|牛顿/米]]，简称[[牛/米|牛/米]]
::* <math>x</math>=弹簧的变形量或壓縮量，单位为[[米|米]]
::* 此外弹簧的種類有許多，如常見的有[[拉簧|拉簧]]、[[扭簧|扭簧]]、[[發條彈簧|發條彈簧]]、[[鋼片彈簧|鋼片彈簧]]和[[空氣彈簧|空氣彈簧]]等等。

==扭力彈簧-彈簧常數公式==

彈簧常數 :以 <math>k</math> 表示,當彈簧被扭轉時,每增加1°扭轉角之負荷 (kgf*mm). 

:<math>K=\frac{E*d^4}{1167*D_m*pi*N+389*R}</math> (kgf*mm).
 
:<math>E</math>= 線材之剛性模數
::琴鋼線<math>E</math>=21000 
::不銹鋼線<math>E</math>=19400 
::磷青銅線<math>E</math>=11200 
::黃銅線<math>E</math>=11200 

:<math>d </math>= 線徑
:<math>D_o</math>=OD =外徑
:<math>D_i</math>= ID = 內徑
:<math>D_m</math>=MD =中心徑 =<math>D_o-d</math>
:<math>N</math> = 總圈數
:<math>R</math> =負荷作用之力臂
:<math>pi

</math> = 3.1416　

==拉伸彈簧-彈簧常數公式==

彈簧常數 :以<math>k</math>表示。當彈簧被拉長時，每增加1mm行程之負荷(kgf/​mm). 

:<math>K=\frac{G*d^4}{8*D_m^3*N_c} </math> kgf/mm).

:<math>G</math>= 線材之剪切模數
::琴鋼線<math>G</math>=8000
::不銹鋼線 <math>G</math>=7300 
::磷青銅線<math>G</math>=4500 
::黃銅線<math>G</math>=3500 
:<math>d </math>= 線徑
:<math>D_o</math>=OD =外徑
:<math>D_i</math>= ID = 內徑
:<math>D_m</math>=MD =中心徑 =<math>D_o-d</math>
:<math>N</math> = 總圈數
:<math>N_c</math> =有效圈數=<math>N-2</math>

==定力彈簧/恆力彈簧==
扭力計算:
:<math>F=\frac{E*b*h^3}{6.6*d_n^2}</math>

:<math>E </math>彈性模數
::琴鋼線<math>E =21000 </math>,不銹鋼線<math>E =19400 </math>,磷青銅線 <math>E=11200</math> ,黃銅線<math>E=11200 </math>
:<math>d_n</math> 彈簧自由狀態下的內圈直徑(輸入端)
:<math>b</math> 截面寬度
:<math>h</math> 截面厚度

==定扭力彈簧/定扭矩彈簧/恆扭矩彈簧==

扭矩計算:

:<math>F=\frac{E*b*h^3}{6.6*d_n^2}</math>
:<math>T=\frac{(E*b*h^3d_1)*(\frac{1}{d_n}+\frac{1}{d_1})^2}{12}</math>

:<math>E</math>——彈性模數，琴鋼線 <math>E</math>=21000 ,不銹鋼線 <math>E</math>=19400 ,磷青銅線 <math>E</math>=11200 ,黃銅線 <math>E</math>=11200 
:<math>d_n</math>—— 彈簧自由狀態下的內圈直徑(輸入端)
:<math>d_1</math>—— 彈簧的心軸直徑 (輸出端)
:<math>b</math>—— 截面寬度
:<math>h</math>—— 截面厚度

彈簧總長度 : 
:<math>L=\pi*N(D_o+N_t)+10D_o</math>
這個公式考慮了啟動的直徑，形成圈數（<math>N</math>），材料厚度（<math>t</math>）還有輸出軸直徑（<math>D_o</math>）

== 設計與製造 ==
=== 基本設計 ===
在設計彈簧時最重要的是澄清它的用途{{Sfn|蒲|2008|p=58}}。與其他[[機器|機器]][[構件|構件]]一樣，有必要為彈簧設計適合於使用目的的性能{{Sfn|日本ばね学会（編）|2008|p=14}}。具體地，可以引用以下功能作為由彈簧實現的功能：{{Sfnm|ばね技術研究会（編）|2000|1p=2|蒲|2008|2p=59|ニッパツ・日本発条株式会社（編）|1998|3p=3}}
* 在卸載時利用彈性返回原始位置和形狀
* 彈性地握住物體
* 隔振/放鬆
* 創造振動和使用
* 放鬆衝擊力
* 儲存和釋放能量
* 負載測量和調節

除了滿足該功能的要求之外，彈簧的設計還需要以下內容：{{Sfn|日本ばね学会（編）|2008|p=315}}
* 適合空間限制
* 不會發生永久變形和破壞
* 在使用期限內具有足夠的強度
* 在使用環境中具有足夠的強度
* 重量輕
* 緊湊
* 易於製造
* 價格便宜

彈簧的採購方法，存在從所銷售的標準品中選出的場合，規格品沒有的東西的個別製作的情况。{{Sfn|門田|2016|p=110}}。由於彈簧的應用是多種多樣的，除了緊固彈簧之外，它通常是逐個設計的{{Sfnm|ニッパツ・日本発条株式会社（編）|1998|1p=5|日本ばね学会（編）|2008|2p=437}}。因此，在彈簧的設計中，從標準產品中選擇的方法並不像螺栓和軸承那樣多的機械元件。{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=5}}。

==== 經典理論公式和有限元法 ====
彈簧的設計的時候，載荷變形的關係和發生的應力的計算方法，在材料力學的經典理論式使用方法和數值分析的有限要素法使用的方法。{{Sfnm|ニッパツ・日本発条株式会社（編）|1998|1pp=16–17|日本ばね学会（編）|2008|2pp=133–136|「ばねの歴史」編纂ワーキンググループ（編）|2012|3pp=157–160}}。在經典理論中，計算公式通常以代數表達式的形式給出，並且很容易用計算器等計算{{Sfn|日本ばね学会（編）|2008|p=133}}。此外，如是否影響多少形狀多少變化一旦特性，因素和結果之間的關係，可以清楚地理解{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=160}}。

另一方面，經典理論有一些假設來推導計算公式，只有使用接近這些假設的範圍才能預期表達式的精確度。{{Sfn|ニッパツ・日本発条株式会社（編）|1998|pp=16–17}}。例如，如果確定形狀和材料特性的數值，則可以通過以下基本方程計算一般壓縮螺旋彈簧的彈簧常數“k”。<ref>{{cite book|和書 |author=日本工業標準調査会 |title=JIS B 2704-1 コイルばね－第１部：圧縮及び引張コイルばね基本計算方法 |year=2009 |page=3}}</ref>。
:<math>k=\frac{Gd^4}{8N_a D^3}</math>

這裡，''G''是材料屬性的值， ''d'', ''N<sub>a</sub>'', ''D'' 以及每個尺寸值。 然而，該公式假設三個假設，即負載在線圈中心的直線上，俯仰角的影響（螺旋的傾斜度）很小且可以忽略不計，並且僅考慮扭矩，並且應用範圍有限{{Sfn|日本ばね学会（編）|2008|p=178}}。在一些實際設計中，還必須在超出這些假設的範圍內使用它{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|pp=155–156}}。

在一個FEM中，通過將彈簧的形狀除以稱為元素的小區域在計算機上創建模型，並給出解決方案{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=17}}。也可以計算具有特殊形狀的彈簧，這些彈簧對適用的彈簧形狀幾乎沒有限制，也沒有建立代數形式的公式{{Sfn|日本ばね学会（編）|2008|p=133}}。可以進行更接近實際產品的計算{{Sfn|蒲|2008|p=73}}。但是，在改變形狀時，每次都需要改變模型，有必要重複工作，使其收斂到最佳設計{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=160}}。與經典理論公式相比，它通常是耗時且昂貴的{{Sfn|蒲|2008|p=73}}。在設計中，通常考慮經典理論方程和FEM的優缺點，並單獨使用它們{{Sfn|日本ばね学会（編）|2008|p=136}}。

====振動問題====
[[File:Half_car_dynamic_model.svg|thumb]]
如果使用彈簧的目的是振動的鬆弛，則在某些情況下可能需要使振動[[阻尼|阻尼]]遠離彈簧的機械元件。 阻尼代表[[油阻尼器|油阻尼器]]等作為阻尼的機械元件，通過將物體的振動能轉換為[[熱能|熱能]]並消散它<ref>{{cite book |和書 |editor=日本機械学会 |title=振動のダンピング技術 |series=新技術融合シリーズ：第4巻 |publisher=養賢堂 |year=1998 |edition=第1版 |isbn=4-8425-9816-6 |pages=pp. 8, 140}}</ref>。雖然彈簧本身俱有像橡膠彈簧一樣的阻尼，但是一般的金屬螺旋彈簧會產生少量阻尼，因此需要單獨使用阻尼器<ref>{{cite book |和書 |editor=日本機械学会 |title=機械工学便覧　基礎編　''α''2　機械力学 |publisher=丸善 |year=2004 |edition=初版 |isbn=4-88898-116-7 |page=187}}</ref>。通過阻尼，可以避免由彈簧支撐的物體隨著自由振動而搖動{{Sfn|日本機械学会（編）|2007|p=380}}。為了更強烈地抑制振動，除了彈簧/阻尼器之外還可以設置致動器{{Sfn|末岡ら|2002|pp=96–97}}。車輛等的主動懸架的示例<ref>{{cite book |和書 |editor=日本機械学会 |title=車両システムのダイナミックスと制御 |series=新技術融合シリーズ：第5巻 |publisher=養賢堂 |year=2008 |edition=OD版第1版 |isbn=978-4-8425-9901-4 |pages=pp. 152–154}}</ref>。

在處理振動問題等時，將目標機制視為[[數學模型|數學模型]]，並將其視為由單個元素組成的系統（[[物理系統|物理系統]]）{{Sfn|下郷・田島|2002|p=4}}。基本振動模型由三個元素組成：慣性元件，恢復元件和阻尼元件<ref name="基礎振動工学">{{cite book |和書 |author=横山隆・日野順市・芳村敏夫 |title=基礎振動工学 |publisher=共立出版 |year=2015 |edition=第2版 |isbn=978-4-320-08211-3 |pages=16-18}}</ref>。典型的修復元件是彈簧<ref name="基礎振動工学"/>。彈簧的載荷 - 如果它是能夠確定的偏移特性，可以賦予其特性作為振動模型的元件{{Sfn|日本ばね学会（編）|2008|p=134}}。但是，振動模型上模型化的話，實際的彈簧始終理想化的東西，需要特別注意。{{Sfn|日本ばね学会（編）|2008|p=134}}。振動模型上的彈簧被視為沒有質量，但實際結合的彈簧具有質量{{Sfn|日本ばね学会（編）|2008|p=134}}。實際的彈簧本身也是一個振動系統{{Sfn|日本ばね学会（編）|2008|p=202}}。因此，彈簧本身也會振動，並且振動中也存在自然振動{{Sfn|日本ばね学会（編）|2008|p=202}}。當彈簧本身的固有頻率與外部頻率重合時發生共振{{Sfn|山田|2010|p=60}}。特別是對於以高頻膨脹和收縮的壓縮螺旋彈簧而言，這成為一個問題{{Sfn|山田|2010|p=60}}。當發生滑脫，彈簧不能跟隨機構的運動，導致系統變得不穩定或導致彈簧斷裂{{Sfnm|山田|2010|1p=60|ニッパツ・日本発条株式会社（編）|1998|2p=115}}。當滑脫是一個問題時，通過增加彈簧本身的固有頻率來採取對策{{Sfn|ばね技術研究会（編）|2001|p=34}}。

==== 強度 ====
[[File:0_ressort_rompu_par_fatigue.jpg|thumb]]
在一般的機械設計中，重要的是具有足夠的強度以使其不會破裂並且彈簧是相同的{{Sfn|門田|2016|p=111}}。與其他[[機械元件|機械元件]]相比，設計中的一個特殊點是它需要通過變形進行偏轉{{Sfn|門田|2016|p=110}}。用其他機械元件進行強度評估，但通常在評估變形量之前不需要{{Sfn|日本ばね学会（編）|2008|p=133}}。另一個設計特徵是使彈簧的使用範圍在如上所述的[[彈性變形|彈性變形]]範圍內{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=16}}。這也可以說是彈簧設計的“絕對要求”{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=16}}。如果它超過材料的彈性極限，則不能正常地實現彈簧的功能{{Sfn|小玉|1985|p=28}}。在彈簧強度方面特別重要的是“疲勞”和“永久變形”{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=17}}。

疲勞是由於在物體上重複施加載荷而在物體中產生[[裂縫|裂縫]]並且發生破壞的現象{{Sfn|日本機械学会（編）|2007|p=1109}}。這種重複負載稱為“動態負載”{{Sfn|日本ばね学会（編）|2008|p=328}}。用於繼續接收振動的車輛的懸架系統的彈簧的示例接收這樣的負載{{Sfn|蒲|2008|p=54}}。疲勞強度是材質、形狀、負荷形式、使用溫度，許多因素如環境影響{{Sfn|日本機械学会（編）|2007|p=1109}}。由於彈簧通常以受到重複載荷的形式使用，因此從設計的角度來研究疲勞強度也很重要{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=18}}。通常，如果重複施加的負載數量是1000萬次，則設計使彈簧不會疲勞{{Sfn|蒲|2008|p=55}}。取決於彈簧的應用，存在足以承受比其更少的次數或為了承受更多次的情況{{Sfnm|蒲|2008|1p=55|「ばねの歴史」編纂ワーキンググループ（編）|2012|2pp=230–231}}。

凹陷不僅低於屈服應力載荷, 甚至長期, 而且在材料中逐漸發生塑性變形, 造成永久性彎曲彈簧現象。{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=21}}。當負載繼續保持幾乎恆定時，也會發生凹陷{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=21}}。這種負載也稱為“靜態負載”{{Sfn|日本ばね学会（編）|2008|p=328}}。凹陷主要是由一種叫做蠕變的現象引起的{{Sfn|日本ばね学会（編）|2008|p=329}}。例如，在用於汽車的懸架系統的彈簧中，由於跟部引起的車輛高度的變化成為問題。 特別是在高溫區域，容易發生斷裂，因此要考慮在高溫區域使用的彈簧，例如將產生的應力抑製到較低水平，採用高度抗下垂的材料{{Sfnm|小玉|1985|1p=33|ばね技術研究会（編）|2000|2pp=4–6}}。雖然在高於450°C的高溫區域出現風化現象，但在400°C以下區域出現下垂的機制仍然不清楚<ref name="へたり">{{Cite journal|和書 |author =圧縮コイルばねのへたりに関する研究委員会 |year =2014 |title =圧縮コイルばねのへたりに関する研究委員会報告 |journal =ばね論文集 |volume =2014 |issue =59 |page =48 |publisher =日本ばね学会  |doi =10.5346/trbane.2014.47}}</ref>。

=== 製造業的基本項目 ===
彈簧的製造過程根據類型而變化。 以下是金屬彈簧製造的一般描述。

[[File:Shot_peening.png|thumb]]
在金屬彈簧的情況下，主要通過塑性加工來執行從桿狀或板狀材料到預定彈簧形狀的成形{{Sfn|門田|2016|p=136}}。彎曲或滾動材料並將其加工成所需形狀{{Sfn|門田|2016|p=136}}。金屬彈簧的塑性成形大致分為冷成型和熱成型{{Sfn|日本ばね学会（編）|2008|p=437}}。如上所述，通過在室溫狀態下將材料加工成彈簧形狀，在相對小的彈簧上進行冷成形{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=28}}。通過在加熱到高溫的狀態下將材料加工成彈簧形狀，在相對大的彈簧上進行熱成形{{Sfn|ニッパツ・日本発条株式会社（編）|1998|p=28}}。

在金屬彈簧的情況下，在成型後施加[[熱處理|熱處理]]。 如果是鋼的熱成形彈簧（如板簧，竹筍彈簧，螺旋彈簧等），在成形後立即淬火[[淬火|淬火]]然後[[回火|回火]]{{Sfnm|日本ばね学会（編）|2008|1pp=503–504, 514–515|蒲|2008|2p=104}}。通過硬化和回火，可以將其製成堅硬且持久的材料{{Sfn|門田|2016|p=106}}。在形成鋼冷成形彈簧（薄板簧，螺旋彈簧，碟形彈簧等）之後進行熱處理時，進行[[退火|退火]]以去除淬火或回火的[[殘餘應力|殘餘應力]]{{Sfn|日本ばね学会（編）|2008|pp=522, 546, 548}}。在有色金屬材料的情況下，施加[[析出硬化|析出硬化]]並且同樣增加強度{{Sfn|日本ばね学会（編）|2008|p=458}}。

在熱處理後的許多情況下表面改質下完成{{Sfn|門田|2016|p=136}}。表面改質是一種在彈簧表面高速錘擊無數硬顆粒的過程，將彈簧表面壓縮[[殘餘應力|殘餘應力]]以提高疲勞強度{{Sfn|日本ばね学会（編）|2008|p=468}}。在表面改質或熱處理之後，在許多情況下執行稱為“預先設定”或“設定”的過程，該過程施加大於最大設計負荷的負荷。{{Sfn|門田|2016|p=136}}。セッチングを行うことでへたりに対する耐性を向上させることができる{{Sfn|門田|2016|p=136}}。熱成形螺旋彈簧等有時進行“熱處理”，其中在高溫狀態下進行設定同時進行回火{{Sfn|ばね技術研究会（編）|2001|p=162}}。因此能夠增加由熱定形淬火抗流掛性{{Sfn|日本ばね学会（編）|2008|p=479}}。在最後的過程中，如果需要，通過電鍍或塗漆進行表面處理{{Sfn|門田|2016|p=150}}。

在塑料彈簧的情況下，用於彈簧的大多數塑料是[[热塑性塑料|热塑性塑料]]，因此它是通過[[注塑成型|注塑成型]]成型的{{Sfn|ばね技術研究会（編）|1998|p=50}}。將熔融材料壓入模具中並冷卻並固化以生產{{Sfn|ばね技術研究会（編）|1998|p=50}}。在作為橡膠彈簧之一的防振橡膠的情況下，我們製造和捏合原材料，並通過[[硫化|硫化]]將它們粘合到金屬配件上{{Sfn|ばね技術研究会（編）|2000|p=246}}。

== 應用實例 ==
利用彈簧的特性和功能，彈簧被廣泛應用於各個領域{{Sfn|日本ばね学会（編）|2008|p=9}}。從熟悉的電器到大型機器和結構，從老式設備到現代設備，彈簧的使用很普遍{{Sfn|ばね技術研究会（編）|2000|p=i}}。

有些[[玩具|玩具]]使用彈簧的各種特性{{Sfn|門田|2016|p=12}}。驚奇的盒子是一個經典的玩具，當打開蓋子時玩偶等彈出恢復力的彈出玩具{{Sfn|門田|2016|p=12}}。[[音樂盒|音樂盒]]旋渦彈簧，作為動力，發出聲音。{{Sfn|ばね技術研究会（編）|1998|pp=166}}。這是如何使用彈簧將能量作為彈性能量積累並逐漸釋放彈簧的一個例子{{Sfn|日本ばね学会（編）|2008|p=9}}。玩具模型汽車的發條也是螺旋彈簧運行的動力源{{Sfn|蒲|2008|p=136}}{{Sfn|門田|2016|p=12}}。

=== 車輛 ===
[[File:Nockenwelle2,_closeup.jpg|thumb]]
一輛車使用了2,000到3,000個彈簧，汽車和彈簧之間的關係很強{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=9}}。汽車[[內燃機|內燃機]]中使用的典型類型是根據凸輪軸的凸輪形狀移動進氣門和排氣門的彈簧，稱為“氣門彈簧”或“閥門彈簧”{{Sfnm|蒲|2008|1p=122|ばね技術研究会（編）|1998|2p=80}}。在約120℃下在油即使上述拉伸1億次要求不疲勞斷裂，並且進一步總是需要小型化和輕量化{{Sfn|日本ばね学会（編）|2008|p=9}}。在整個彈簧中，閥門彈簧可以說是在最惡劣的環境中使用的彈簧{{Sfn|蒲|2008|p=122}}為了滿足使用條件，形狀和線間距截面形狀進行了特殊設計{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=231}}。關於材料，[[抗拉強度|抗拉強度]]超過2000 [[帕斯卡|帕斯卡]]的鋼絲已經標準化並用作閥門彈簧材料，並且是“最廣泛使用的批量生產的材料高品質的彈簧{{Sfnm|ばね技術研究会（編）|2000|1pp=116, 118|「ばねの歴史」編纂ワーキンググループ（編）|2012|2p=232}}。

各種彈簧也用於車輛的懸架，該車輛在保持車輪的同時支撐車身並減輕來自路面的衝擊。{{Sfn|KYB株式会社（編）|2013|pp=11–12}}。最廣泛使用的懸架彈簧是壓縮螺旋彈簧，重量輕的小型轎車的很多因為使用的{{Sfnm|KYB株式会社（編）|2013|1p=83|新星出版社編集部（編）|2009|2p=134}}。層壓板簧是重型和重型的，但由於它們的高承載能力，它被用於[[貨車|貨車]]，[[公共汽車|公共汽車]]，[[越野車輛|越野車輛]]等。{{Sfnm|KYB株式会社（編）|2013|1p=80|新星出版社編集部（編）|2009|2p=134}}。儘管空氣彈簧具有諸如高度調節和乘坐舒適性之類的優點，但它們昂貴且用於公共汽車和豪華汽車{{Sfnm|KYB株式会社（編）|2013|1pp=95–96|新星出版社編集部（編）|2009|2p=134}}。扭杆是一級方程式賽車的主懸架彈簧{{Sfn|蒲|2008|p=124}}。為了抑制側傾擺動車體，從小型車到大型卡車的扭杆使用範圍很廣。{{Sfn|日本ばね学会（編）|2008|p=245}}。

[[File:ČD_842_bogie_1.jpg|thumb]]
[[鐵路車輛|鐵路車輛]]的懸架系統由兩種彈簧組成，即枕形彈簧和軸向彈簧{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=249}}。枕形彈簧是存在於車身和鐵路車輛之間的彈簧，主要使用空氣彈簧{{Sfn|門田|2016|p=24}}。通過使用空氣彈簧，可以在獲得軟彈簧常數的同時保持車身的高度<ref name="宮本"/>。軸簧是位於轉向架和輪對之間的彈簧, 主要使用螺旋彈簧{{Sfn|門田|2016|p=24}}。

除了懸掛系統，列車的[[集電弓|集電弓]]，而且通過氣動，通過彈簧得到靠在集電弓接觸導電<ref>{{Cite book|和書 |author=近藤圭一郎 |title=鉄道車両技術入門 |edition=初版 |date=2013-07-20 |publisher=オーム社 |isbn=978-4-274-21383-0 |page=78}}</ref>。在年齡較大的有軌車車輪彈簧用於緩衝耦合{{Sfn|小玉|1985|p=161}}。[[レール|レール]]を[[枕木|枕木]]に固定するためにも、板ばねや線ばねが使われている<ref>{{Cite book|和書 |author= 上浦正樹・小野田滋・須長誠 |year= 2000 |title= 鉄道工学 |publisher= 森北出版 |edition=初版 |pages= 68-70 |isbn= 978-4627484719}}</ref>。

對於其他車輛，在建築車輛的推土機的懸架中裝有彈簧，以便將輪胎給予履帶，但即使施加異常力，它也可以釋放它{{Sfn|ばね技術研究会（編）|1998|p=110}}。 該彈簧稱為“反沖彈簧”，主要使用螺旋彈簧{{Sfn|ばね技術研究会（編）|1998|p=110}}。一些复進簧的也是巨大的壓縮螺旋彈簧超出人類的脊柱{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=248}}。

=== 電氣和電子設備 ===
[[File:Power_strip_inside_with_plug.jpg|thumb]]
[[File:Hard_disk_head_on_platter.jpg|thumb]]驅動器的磁頭（左臂尖端）]]
彈簧還用於電氣設備和電子設備。 在某些情況下，彈簧本身是電路的一部分，在這種應用中，使用具有良好導電性的銅合金彈簧{{Sfn|門田|2016|p=100}}。銅薄板簧安裝在出口用於獲取電力，這個薄板簧與插頭電連接並保持插頭{{Sfn|蒲|2008|p=130}}。這不容易地移除插頭，並且也可以用適度的力拉出插頭{{Sfn|門田|2016|p=26}}。在電路、電子電路中的[[繼電器|繼電器]]和開關中，也有各種各樣的電力接觸{{Sfn|ばね技術研究会（編）|1998|pp=69–70}}。筆記型電腦和手機等電子機器類，高度的輕量化和小型化被要求，它們中的繼電器開關、連接器等中使用薄鋼板，也同樣輕量化和小型化被要求，作為結果懸架裝置用彈簧普通的高强度的彈簧也會用{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=244}}。

照明和遙控等開關也使用彈簧進行操作{{Sfn|蒲|2008|p=129}}。假設沒有彈簧，當開關緩慢按下時，電觸點也會緩慢接近並接觸，這樣電弧很可能在觸點之間長時間發生，導致損壞<ref name="配線器具"/>。通過使用彈簧，即使緩慢按下開關，也可以立即接觸端子<ref name="配線器具">{{Cite book|和書 |author=高橋秀憲 |year= 2008 |title= 配線器具入門―安全な設計・施工・取扱いのポイント |publisher=オーム社 |edition=初版 |pages=141-146 |isbn= 978-4-274-50180-7}}</ref>。使用壓縮螺旋彈簧或橡膠的機構，以及連接端子本身是板簧的機構<ref>{{Harvnb|蒲|2008|p=129}} / {{Cite book|和書 |author=高橋秀憲 |year= 2008 |title= 配線器具入門―安全な設計・施工・取扱いのポイント |publisher=オーム社 |edition=初版 |pages=141-146 |isbn= 978-4-274-50180-7}}</ref>。

在電腦的示例中，彈簧被包含在執行操作的[[鍵盤|鍵盤]]中。 在較舊的鍵盤上，金屬螺旋彈簧構建在每個鍵下方，用於推回鍵{{Sfnm|門田|2016|1p=26|蒲|2008|2p=132}}。還有一種用橡膠的恢復力推回鍵盤的方法，截至2008年，這種類型的鍵盤是主流{{Sfnm|門田|2016|1p=26|蒲|2008|2p=132}}。在存儲設備的[[硬碟|硬碟]]驅動器中，稱為磁頭的組件在磁盤上移動以向盤讀取和寫入信息<ref>{{Cite web |url=http://www.nhkspg.co.jp/special/odoroki_archives/20061028.html |title=会話も弾む？オドロキの数字（1）  |publisher=日本発条株式会社 |accessdate=2017-01-21}}</ref>。在這種情況下，薄板彈簧稱為懸浮液給出了一個恆定的負載到磁頭，磁頭有助於維持在幾十納米的磁盤上的位置{{Sfn|「ばねの歴史」編纂ワーキンググループ（編）|2012|p=247}}。

=== 結構 ===
[[File:Base_isolators_under_the_Utah_State_Capitol.jpg|thumb]]
彈簧也用於建築和土木工程領域的結構本身。 為了保護建築物免受地震的影響，將建築物和基礎分開，並在其間安裝彈簧和阻尼器稱為地震隔離結構{{Sfn|斉藤|2008|pp=39,56–64}}。線圈彈簧也用於地震隔離結構中，但通常使用層壓橡膠，其中金屬板和橡膠層疊成層{{Sfn|門田|2016|p=28}}。體操比賽的百合地板也被鋪滿了的彈簧支撐著{{Sfn|蒲|2008|p=139}}。根據這個，地板競技高難度的空翻技巧成為可能。<ref>{{Cite web |url=https://kotobank.jp/word/%E3%82%86%E3%81%8B(%E4%BD%93%E6%93%8D%E7%AB%B6%E6%8A%80)-1603447 |title=ゆか 日本大百科全書(ニッポニカ)の解説 |author=小学館 |work=コトバンク |publisher=朝日新聞社/VOYAGE GROUP |accessdate=2017-01-05}}</ref>。層壓橡膠等甚至結合在橋的軸承中，由此缺少橋的上部結構的移動{{Sfn|ばね技術研究会（編）|1998|p=124}}。

除了基礎隔離結構之外，還有一種減振結構可以保護建築物免受震動{{Sfn|斉藤|2008|p=39}}。在減振結構中，提供稱為動態減振器的機構，以通過彈簧和減震器將重物附著到建築物的上部。{{Sfn|斉藤|2008|pp=53–54}}。與地震隔離結構不同，由於可以減少由強風引起的振動，因此尤其對於高層建築物而言需要阻尼結構<ref>{{Cite book |和書 |author= 大成建設「超高層ビル」研究プロジェクトチーム |year= 2009 |title= 超高層ビルの"なぜ"を科学する |edition= 初版 |publisher= アーク出版 |isbn= 978-4-86059-076-5 |pages= 46–47}}</ref>。例如，在[[台北101|台北101]]大樓中為了因應高空強風及颱風吹拂造成的搖晃，在87至92樓掛置一個重達661.02公噸、直徑5.5公尺的調諧質塊阻尼器（又稱「調質阻尼器」）。
