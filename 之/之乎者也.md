《'''之乎者也'''》，是[[羅大佑|羅大佑]]在1982年所發行的首張創作專輯<ref name="資料庫">[http://www.luodayou.net/big5/db/history.htm 羅大佑音樂創作歷程] (2007年3月10日引用)</ref>。

== 過程 ==
=== 投入創作（1974年－1980年）===
羅大佑還沒有發行這張專輯之前，早就發表這張專輯的一些歌曲。1974年（民國63年），羅大佑在讀[[中國醫藥學院|中國醫藥學院]]二年級時發表兩首歌曲，而且歌詞都是兩位文學家所寫的，分別是[[徐志摩|徐志摩]]的《歌》和[[余光中|余光中]]的《鄉愁四韻》；三年後，《歌》被選為《閃亮的日子》的電影插曲之一<ref name="資料庫"/>。1976年，羅大佑發表《錯誤》、《蒲公英》這兩首創作歌曲，其中《錯誤》的歌詞是羅大佑改寫於[[鄭愁予|鄭愁予]]的同名散文<ref name="資料庫"/>。1979年，羅大佑花費30分鐘完成《戀曲1980》，這首歌是為中國醫藥學院的護理系的學姊而創作的<ref name="追夢羅大佑">[http://lotayu.org/modules/news/article.php?storyid=215 歌的故事 :《戀曲1980》的女主角] {{Wayback|url=http://lotayu.org/modules/news/article.php?storyid=215 |date=20070219175012 }} (2007年3月10日引用)</ref>，同時發表《搖籃曲》、《童年》這兩首歌曲<ref name="資料庫"/>。1980年，羅大佑幫[[張艾嘉|張艾嘉]]創作《光陰的故事》這首歌曲，這時也發表《將進酒》和《鹿港小鎮》這兩首歌曲<ref name="資料庫"/>；其中羅大佑上台北見習，當時台灣的[[十大建設|十大建設]]已經完成，台北的農村景觀迅速地消失；他從台中上來時，有感覺到失去純樸之味，所以創作了《鹿港小鎮》<ref name="追夢羅大佑2">[http://lotayu.org/modules/news/article.php?storyid=226 歌的故事:從摩托車修理工到鹿港小鎮] {{Wayback|url=http://lotayu.org/modules/news/article.php?storyid=226 |date=20070219214945 }} (2007年3月10日引用)</ref>。

=== 籌備專輯（1980年－1982年）===
後來羅大佑從中國醫藥學院畢業，就跟張艾嘉建立「果實音樂製作有限公司」（GOSH），執行籌製個人首張專輯<ref name="資料庫"/><ref name="追夢羅大佑3">[http://lotayu.org/modules/news/article.php?storyid=258 歷史報道:羅大佑和黃霑] {{Wayback|url=http://lotayu.org/modules/news/article.php?storyid=258 |date=20070219215207 }} (2007年3月10日引用)</ref>；在錄製專輯過程中，羅大佑曾為了要解決音樂母帶交給日本的處理費用，向自己的父親借了一些錢<ref name="追夢羅大佑7">[http://lotayu.org/modules/news/article.php?storyid=64 歷史的紀錄——述評羅大佑的一張重要專輯] {{webarchive|url=https://web.archive.org/web/20070928013259/http://lotayu.org/modules/news/article.php?storyid=64 |date=2007-09-28 }} (2007年7月18日引用)</ref>。一年之後，羅大佑拿著自己的音樂母帶，四處尋找願意發行的[[唱片公司|唱片公司]]；接二連三被很多唱片公司拒絕之後，剛成立不久的[[滾石唱片|滾石唱片]]接受羅大佑的音樂母帶<ref name="追夢羅大佑4">[http://lotayu.org/modules/news/article.php?storyid=69 羅大佑作品全集:之乎者也] {{Wayback|url=http://lotayu.org/modules/news/article.php?storyid=69 |date=20070928013310 }} (2007年3月10日引用)</ref>；也在這段期間，羅大佑完成了《之乎者也》的創作<ref name="資料庫"/>。

=== 發行專輯 ===
1982年（民國71年）4月21日，羅大佑發行首張創作專輯《之乎者也》，就造成一股風波，那時共賣了十五萬張<ref name="追夢羅大佑5">[http://lotayu.org/modules/news/article.php?storyid=151 最新消息:羅大佑：永遠不站在既得利益那邊] {{webarchive|url=https://web.archive.org/web/20070928013457/http://lotayu.org/modules/news/article.php?storyid=151 |date=2007-09-28 }} (2007年3月10日引用)</ref>。後來《台灣流行音樂百張經典專輯》一書，將《之乎者也》選為1975年9月到1993年1月的第一名專輯<ref name="MENU">[http://home.kimo.com.tw/tomotoching/new_page_88.htm#a 各類排行榜整理] {{webarchive|url=https://web.archive.org/web/20031221183430/http://home.kimo.com.tw/tomotoching/new_page_88.htm |date=2003-12-21 }} (2007年3月11日引用)</ref>。
