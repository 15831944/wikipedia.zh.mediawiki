'''慕绥新'''（{{bd|1943年|10月|2002年|3月2日|catIdx=M慕}}），[[籍贯|籍贯]][[陕西省|陕西省]][[吴堡县|吴堡县]]<ref name="a1"/>，生于[[绥德县|绥德县]]<ref name="a2"/>。原[[辽宁省|辽宁省]]副省长、[[沈阳市|沈阳市]]市长。因犯[[受贿罪|受贿罪]]和[[巨额财产来源不明罪|巨额财产来源不明罪]]，被[[大连市中级人民法院|大连市中级人民法院]]于2001年10月10日判处[[死刑|死刑]]，[[缓刑|缓期]]二年执行，剥夺政治权利终身。2002年3月2日因[[癌症|癌症]]去世。

== 生平 ==
出生于陕西省绥德县。父母是中共政权的成员，慕绥新幼年时与父母一同南北转战。9岁时，随家人定居于[[长春|长春]]。1970年毕业于[[清华大学|清华大学]]电子工程系电机制造专业<ref name="a2"/>。从冶金部三冶总公司的工人干起，至三冶总公司代常委书记。期间，于1978年加入中国共产党<ref name="a2">{{Cite web |url=http://www.zslz.com/news/show.asp?newsid=759 |title=慕绥新悲剧人生：把有价值的东西毁灭给人看（2002第二期案例剖析） |accessdate=2015-04-16 |author= |date=2003-01-23 |work= |publisher=舟山廉政网 |language=Zh-hans }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1984年起，任[[海城县_(顺治)|海城县]]县委副书记、副县长、[[海城市|海城市]]市长。1987年，任[[鞍山市|鞍山市]]市长助理。1988年起，进入[[辽宁省人民政府|辽宁省人民政府]]任职，历任[[辽宁省体改委|辽宁省体改委]]副主任、[[劳动局|劳动局]]副局长、党组副书记。1990年起，任劳动局局长、建设厅厅长、省长助理。1996年，任辽宁省副省长、省委政治委副书记、建设厅厅长。6月，卸任建设厅厅长。12月起，兼任中共沈阳市委副书记。1997年2月，又兼任沈阳市长。8月，任职为中共沈阳市委副书记、市长<ref name="a1">{{Cite web |url=http://news.sohu.com/80/64/news143986480.shtml |title=背景介绍:慕绥新简历(图)|accessdate=2015-04-16 |author= |date=2001-08-16 |work= |publisher=搜狐网，来源：南方网-南方都市报|language=Zh-hans }}</ref>。

== 案发 ==
{{see also|慕马案}}
1999年7月，沈阳市副市长[[马向东|马向东]]在澳门豪赌，而被“[[双规|双规]]”，同年11月被逮捕<ref>{{cite news|url=http://news.sina.com.cn/china/1999-11-25/35277.html|title=沈阳市副市长马向东涉嫌贪污被依法逮捕|publisher=[[中国新闻社|中国新闻社]]|date=1999-11-25|accessdate=2018-07-17}}</ref>。2000年10月18日，[[中共中央纪委|中共中央纪委]]决定成立“10·18”[[专案组|专案组]]。同年12月，慕绥新以身体不佳为由请求辞去沈阳市市长职务，沈阳市人大常委会接受了这一请求<ref>{{cite news|url=http://www.chinanews.com/2000-12-22/26/62953.html|title=图文快讯:沈阳市人大常委会接受慕绥新辞去市长职务|publisher=[[中国新闻社|中国新闻社]]|date=2000-12-22|accessdate=2018-07-17}}</ref><ref>{{cite news|url=http://www.people.com.cn/GB/channel4/27/20001222/358962.html|title=涉嫌黑帮案 沈阳市长辞职|publisher=[[江南时报|江南时报]]|date=2000-12-22|accessdate=2018-07-17}}</ref>。

此前，他和贾桂娥的两个女儿已相继离开中国。2000年1月<ref name="a2"/>，[[前妻|前妻]]贾桂娥准备飞往[[美国|美国]]。在中央纪委的“攻势”下，贾桂娥上缴多本护照，及签证、机票和钱物。2001年2月<ref name="a3"/>，专案组对慕绥新现任妻子平晓芳“双规”。3月22日，专案组对慕绥新进行“双规”。慕绥新此前已于2000年被查出患有肺癌<ref>{{cite web|url=http://news.eastday.com/epublish/gb/paper148/20020821/class014800018/hwz748365.htm|title="两面市长"慕绥新"红到黑"蜕变的心路历程|publisher=新民周刊|date=2002-08-21|accessdate=2018-07-17}}</ref>，2002年3月2日在大连医科大学附属第二医院病故。

案发后，长女慕洋的[[公婆|公婆]]李琦、罗桂芝夫妇转移了两千万元的[[赃款|赃款]]。[[中华人民共和国公安部|公安部]]对李琦夫妇发出[[B级通缉令|B级通缉令]]。2002年9月25日，两人在[[浙江省|浙江省]][[金华市|金华市]]被警方抓获<ref>{{Cite web |url=http://www.hangzhou.com.cn/20020127/ca153679.htm |title=慕绥新亲家金华落网|accessdate=2015-04-16 |author=记者：杨波 |date=2002-09-27 |work= |publisher=杭州网，来源：每日商报|language=Zh-hans }}</ref>。
