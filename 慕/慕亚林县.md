'''慕亚林县'''（{{lang-ms|Daerah Muallim}}），是[[马来西亚|马来西亚]][[霹雳州|霹雳州]]东南部的一个县，其面积为934平方公里，2016年人口为69,639。<ref name="perak.gov.my">{{cite web|url=http://pdtmuallim.perak.gov.my/profil/profile-jabatan/pentadbiran-mukim|title=Laman Web Pejabat Daerah Dan Tanah - Pentadbiran Mukim|website=pdtmuallim.perak.gov.my}}</ref>该县北临[[马登巴冷县|马登巴冷县]]，西临[[峇眼拿督县|峇眼拿督县]]，南以安南河和[[雪兰莪州|雪兰莪州]][[乌鲁雪兰莪县|乌鲁雪兰莪县]]为界，东临[[彭亨州|彭亨州]][[劳勿县|劳勿县]]。慕亚林县的县府和最大城市为{{tsl|ms|Tanjung Malim|丹绒马林}}，其他地区包括{{tsl|ms|Slim River|仕林河}}、{{tsl|ms|Behrang|美冷}}、{{tsl|ms|Trolak|地罗叻}}、{{tsl|ms|Bandar raya Proton|宝腾城}}、{{tsl|ms|Hulu Bernam|乌鲁安南}}等。慕亚林县的主要经济收入源自油棕园丘、农业及工业<ref name="auto">{{cite news|url=http://www.enanyang.my/news/20160111/%E3%80%90%E7%8B%AC%E5%AE%B6%E3%80%91%E6%8F%90%E4%BE%9B%E5%B0%B1%E4%B8%9A%E6%9C%BA%E4%BC%9A-%E5%90%B8%E5%BC%95%E4%BA%BA%E6%89%8D%E5%9B%9E%E6%B5%81br-%E6%85%95%E4%BA%9A%E6%9E%97%E7%9D%80/|title=【独家】提供就业机会 吸引人才回流 慕亚林着重土地发展|publisher=[[e南洋|e南洋]]}}</ref>。

==历史==
慕亚林县的前身是隶属[[马登巴冷县|马登巴冷县]]下的仕林河副县，自2014年起便开始准备升格，并在2015年10月1日升格为拥有两个巫金（区）的新县份，占原本马登巴冷县的32%土地面积<ref>{{cite news|url=http://www.sinchew.com.my/node/1008489|title=赞比里：苏丹赐名慕亚林．丹绒马林升县|date=2014-11-02|publisher=[[星洲网|星洲网]]}}</ref><ref name="auto"/>。然而慕亚林县直到2016年1月11日才由霹雳州苏丹纳兹林沙于丹绒马林县议会正式颁布升格，成为霹雳州第11个县<ref>{{cite news|title=Muallim is now Perak's 11th district|url=http://www.nst.com.my/news/2016/01/121608/muallim-now-peraks-11th-district|agency=The Star|publisher=New Straits Times Publication|date=2016-01-11}}</ref><ref>{{cite news|title=Muallim rasmi menjadi daerah ke-11 di Perak|url=http://www.bharian.com.my/node/113170|publisher=Berita Harian|date=2016-01-11}}</ref>。

==名称来源==
本县由霹雳州苏丹纳兹林沙赐名，其中“慕亚林”是阿拉伯语，并有“教师”或“亚林人”（Orang Alim）的意思，是依据本县作为霹州知识分子出生地的历史（即自1922年于此设立的{{tsl|ms|Universiti Pendidikan Sultan Idris|苏丹依德利斯师范大学}}）而命名<ref name="auto"/>。 

== 行政区划 ==
慕亚林县下辖二个[[巫金|巫金]]（区级行政区），分别是乌鲁安南和仕林，其中乌鲁安南被雪兰莪州内的乌鲁安南分隔为东西两个部分，其中乌鲁安南西是该县的[[外飞地|外飞地]]<ref name="perak.gov.my"/>。

==发展==
[[File:Slim_River.JPG|thumb]]
慕亚林县和下霹雳、马登巴冷以及峇眼拿督县是州政府所推行的南霹雳经济特区计划（South Perak）进行区域，这项计划将推动霹雳南部4县的发展，同时提高当地居民的社会经济水平及减少贫富悬殊。<ref name="auto"/>

随着丹绒马林、普腾汽车城及美冷的数项发展计划发展计划带动下，使当地人口逐步上升并迈向城市化，因此需要分划新县来缩小行政管理范围，同时使丹绒马林能更集中发展。<ref name="auto"/>

该县也设有霹雳州内重要的工业区，其中包括宝腾城和拟建中的青年城（Youth City）。<ref name="auto"/>

==教育==
虽然慕亚林县已成立，但该县的学校主要仍由'''马登巴冷县教育局'''管辖。目前县内共有19间国民小学、5间华文小学、7间淡米尔小学、6间国民中学、1间华文中学、1间宗教中学、及1间技术职业中学。<ref>{{cite web |title=PPD Batang Padang - JPN Perak |url=http://jpnperak.moe.gov.my/ppdbatangpadang/}}</ref>
