'''慕容雪村'''（{{bd|1974年|||CatIdx=M慕}}）<ref>{{cite web |url=http://cn.nytimes.com/opinion/20140319/c19murong/ |title=东莞扫黄暴露央视信任危机 |accessdate=2014年3月19日 |publisher=纽约时报 |author=慕容雪村 |language =zh-cn}}</ref>，本名'''郝群'''，[[山东|山东]][[平度|平度]]人，[[中国|中国]]当代著名[[网络作家|网络作家]]，畢業于[[中國政法大學|中國政法大學]]。主要作品有：长篇小说《成都，今夜请将我遗忘》、《天堂向左，深圳往右》、《伊甸樱桃》，散文小说集：《遗忘在光阴之外》，其作品被译为英文、德文、法文、越南文、日文、韩文等多国文字。最新小说《原谅我红尘颠倒》，他是中国大陆广为人知的公共知识分子。<ref name="fish " >{{cite web |url=http://news.ifeng.com/society/2/detail_2012_09/18/17703295_0.shtml |title=慕容雪村：我是被逼成公知的 |accessdate=2012年9月18日 |publisher=凤凰网 |work= 何三畏|language =zh-cn}}</ref>

== 生平 ==
1974年，慕容雪村出生在[[中国|中国]][[山东省|山东省]][[平度市|平度市]]，14岁迁居[[吉林省|吉林省]][[白山市|白山市]]，慕容雪村曾在[[中国政法大学|中国政法大学]]读书，1996年大学毕业后，去了[[四川省|四川省]][[成都市|成都市]]，2001年写出处女作《成都，今夜请将我遗忘》。<ref name=" wyis" >{{cite web |url=http://news.163.com/10/0422/09/64S5TL4A00014AEE.html |title=慕容雪村卧底传销23天 |accessdate=2010-04-22 |publisher=网易 |work= |language =zh-cn}}</ref>

2009年12月29日，他深入[[江西|江西]][[上饶|上饶]]传销团伙23天，写出《中国，少了一味药》，以亲身经历解析传销骗局，试图将传销者拉出火坑，但是现实是几多无奈。<ref name="man " >{{cite web |url=http://news.ifeng.com/gundong/detail_2012_07/23/16219526_0.shtml?_from_ralated |title=慕容雪村：作家不应该是喇叭 |accessdate=2012年7月23日 |publisher=凤凰网 |work= |language =zh-cn}}</ref><ref name=" wyis" />

2010年12月，慕容雪村获得了[[人民文学奖|人民文学奖]]的“特别行动奖”。

2013年5月11日慕容雪村的新浪微博因以「散播謠言」為由被封鎖<ref>[http://www.guardian.co.uk/world/2013/may/15/china-microblogging-accounts-closed China tries to rein in microbloggers]</ref>，不久被註銷，其腾讯、网易、獀狐的全部中国互联网微博帐户均被不明原因注销，在香港接受BBC中文网专访談起此事。<ref>{{cite web |url=http://dzh.mop.com/shzt/20130518/0/zlg7S8I2c141f8FS.shtml |title=存档副本 |accessdate=2013-05-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20130708215620/http://dzh.mop.com/shzt/20130518/0/zlg7S8I2c141f8FS.shtml |archivedate=2013-07-08 }}</ref>

== 风格 ==
慕容雪村以非虚构性小说而著名<ref name="man " />，慕容雪村在被《[[人物周刊|人物周刊]]》采访时揭示自己的文字风格深受[[拉美|拉美]][[魔幻现实主义|魔幻现实主义]]作家[[加夫列尔·加西亚·马尔克斯|加西亚·马尔克斯]]、英国作家[[格雷厄姆·格林|格雷厄姆·格林]]、美国作家[[欧内斯特·海明威|欧内斯特·海明威]]、[[自由主义|自由主义]]作家[[胡适|胡适]]的影响。<ref name="fish " />
