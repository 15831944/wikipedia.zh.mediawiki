[[丹斯里|丹斯里]]'''慕尤丁·莫哈末雅欣'''（{{lang-ms|Tan Sri Dato' Haji Muhyidin bin Haji Mohammad Yassin}}，{{bd|1947年|5月15日|catIdx=M}}）是位[[马来西亚|马来西亚]]政治人物，也是现任[[马来西亚内政部|内政部长]]和{{pc|巴莪}}区国会议员。他曾于2009年至2015年擔任[[副首相|副首相]]及[[巫來由人統一組織|巫來由人統一組織]]（[[巫統|巫統]]）署理主席，在这主导[[國民陣線|国阵]]的成员党直至2016年。

1978年，他当选巴莪国会议员，先后被委任联邦直辖区副部长及贸工副部长。身为巫统[[柔佛州|柔佛]]主席，他于1986年至1995年担任[[柔佛州务大臣|柔佛州务大臣]]。

1995年他回掌联邦事务，被委任青年及体育部长，1999年大选后担任国内贸易及消费者事务部长，2000年成为巫统副主席。在[[阿都拉·巴达威|阿都拉]]掌政期间慕尤丁担任农业部长（2004年－2008年）和国际贸易及工业部长（2008年－2009年）。

2008年，他竞选并在巫统署理主席的职位中获胜，于2009年受[[纳吉·阿都拉萨|纳吉]]委任为副首相兼教育部长。就职教育部长期间，他结束了英语教数理政策。当在野党要挑战他表态是否“马来西亚人优先”，他表示了争议性的“马来人优先”。2015年7月纳吉进行内阁改组，他被除名。2016年6月他被巫统开除党籍。之后与[[马哈迪·莫哈末|马哈迪]]成立[[土著团结党|土著团结党]]，出任党主席至今。<ref>{{Cite news|url=http://www.enanyang.my/news/20160909/%E5%A4%A7%E9%A9%AC%E5%9C%9F%E8%91%97%E5%9B%A2%E7%BB%93%E5%85%9Abr-%E6%B3%A8%E5%86%8C%E6%AD%A3%E5%BC%8F%E8%8E%B7%E6%89%B9/|title=大马土著团结党 注册正式获批|last=快而准|first=e南洋商报 -|newspaper=e南洋商报Nanyang Siang Pau {{!}} 马来西亚新闻真实报道|accessdate=2017-02-13|language=en-US}}</ref>

==早期生活==
慕尤丁在[[柔佛|柔佛]][[麻坡县|麻坡县]][[麻坡|麻坡]]出生，其父亲莫哈末雅欣是[[武吉斯人|武吉斯人]]。莫哈末雅欣是在麻坡香妃城的伊斯兰学者，母亲哈嘉卡蒂嘉是[[爪哇族|爪哇人]]<ref>{{cite web|url=http://thestar.com.my/news/story.asp?file=/2010/10/24/nation/7289147&sec=nation|title=Archives|work=thestar.com.my|deadurl=yes|archiveurl=https://web.archive.org/web/20111011044125/http://thestar.com.my/news/story.asp?file=%2F2010%2F10%2F24%2Fnation%2F7289147&sec=nation|archivedate=2011-10-11}}</ref>。

小学于马哈拉尼国小接受教育，毕业于依斯迈国中及[[麻坡高级中学|麻坡高级中学]]。之后在[[马来亚大学|马来亚大学]]考获经济及马来人研究学士学位（1971年）。完成学业后的慕尤丁，曾在柔佛州的公共部门担任培训及奖学金助理秘书。

== 政治生涯 ==
1971年慕尤丁加入巫统，成为巴莪区部党员，开始其政治生涯。1976年他当选巴莪区部青年团长及秘书，之后成为巫统柔佛州青年团长至1987年。

慕尤丁在巫统全国青年团执委里占有一席位。1984年，他取代奥斯曼沙阿成为巴莪区部主席，在柔佛州巫统党内地位步步高升。从担任柔佛州行政议员，他最终成为柔佛州巫统主席及柔佛州务大臣。

慕尤丁在1978年竞选并当选巴莪区国会议员至1982年，担任外交部政务次长；之后升任至副部长，先后服务于联邦直辖区部及贸工部。1986年大选时，他中选武吉士南邦州议席，并于8月13日成为柔佛州务大臣至1995年5月6日。1995年大选回归竞选巴莪国会议席。

他在内阁期间先后服务各种不同部门。1995年出任青年及体育部部长，1999年被调任至国内贸易及消费人事务部部长。2004年出任农业及农基工业部部长，2008年调至国际贸易及工业部部长。

1984年，慕尤丁参选巫统最高理事但落选，之后受委巫统柔佛州联委会主席及受委最高理事成员。1990年11月参选副主席职位但再次落选。1993年11月慕尤丁再次参选副主席，这次成功当选。1996年党选捍卫副主席职位失败。2000年党选再次当选副主席，直至2008年党选成功再上一层楼，接替纳吉成为党主席后留下的署理主席职。

== 2009年巫统大会及党选 ==
慕尤丁批评阿都拉的权利转移计划“太久”，有舆论认为慕尤丁不直接表态地向阿都拉“逼宫”辞去首相退位。2008年全国大选期间，慕尤丁原打算继续保留在巫统里的职位，直到大选成绩差强人意，他才想要改变。

在2009年巫统党选期间，巫统高层进行改组，慕尤丁被赋予最高提名竞选署理主席。同时，他受到马六甲首席部长拿督斯里[[莫哈末.阿里.魯斯塔姆|莫哈末.阿里.魯斯塔姆]]（简称阿里）及当时的乡村及区域发展部部长[[莫哈末.塔伊卜|莫哈末.塔伊卜]]的挑战。根据民意调查，希望慕尤丁中选的人，远远超过另外两个对手。不过在巫统党员中，阿里的支持率超越了慕尤丁。不料，阿里因为涉及贿选被巫统纪律局禁选署理主席，使到慕尤丁胜券在握（因为塔伊卜不被看好中选）。最后，慕尤丁以1575票击败莫哈末.塔伊卜916票，正式當选为署理主席，2009年4月9日纳吉委任他成为第十任副首相兼教育部长。

== 副首相 ==
2009年4月9日纳吉取代阿都拉成为首相后公布内阁阵容，慕尤丁受委为副首相。

身为教育部长，他宣布政府中小学恢复使用马来语教数理，废除马哈迪任期时推行的英语教数理政策。<ref>[http://www.themalaysianinsider.com/index.php/malaysia/32910-muhyiddin-defends-ppsmi-decision "Muhyiddin defends PPSMI decision"] {{webarchive|url=https://web.archive.org/web/20090825153840/http://www2.themalaysianinsider.com/index.php/malaysia/32910-muhyiddin-defends-ppsmi-decision |date=2009-08-25 }}, ''{{tsl|ms|The Malaysian Insider|}}'', 21 July 2009.</ref>

在接受英国新闻台访问时，记者问到关于纳吉的一个马来西亚理念有什么看法时，他回答[[马来人至上|马来人优先]]才是马来西亚人<ref>{{YouTube|_rxiJSs7aYE|Muhyiddin: I'm Malay first}} 31 March 2010</ref>。<ref>{{cite web|url=http://www.themalaysianinsider.com/malaysia/article/muhyiddin-all-can-form-own-perkasa/|title=Muhyiddin: All can form own ‘Perkasa’|work=themalaysianinsider.com|deadurl=yes|archiveurl=https://web.archive.org/web/20100716180649/http://www.themalaysianinsider.com/malaysia/article/muhyiddin-all-can-form-own-perkasa|archivedate=2010-07-16}}</ref><ref>[http://thestar.com.my/news/story.asp?file=/2010/4/2/nation/5978760&sec=nation "PM defends Muhyiddin’s ‘Malay first’ statement"] {{webarchive|url=https://web.archive.org/web/20100620173205/http://thestar.com.my/news/story.asp?file=%2F2010%2F4%2F2%2Fnation%2F5978760&sec=nation |date=2010-06-20 }}, ''The Star'', 2 April 2010.</ref>

2015年7月28日因为公开批评[[一個馬來西亞發展有限公司|一马公司丑闻]]而被首相[[纳吉·阿都拉萨|纳吉]]的内阁大改组被除名，其位置被时任的内政部部长[[阿末扎希|阿末扎希]]兼任。纳吉声称，撤换副手慕尤丁是艰难的抉择，并认为慕尤丁不应该在公开场合发表意见不同的见解。<ref>{{Cite news|url=http://www.orientaldaily.com.my/nation/gn45635010490765|title=各官職「大風吹」　納吉續任財長職|last=马来西亚东方日报|first=东方网 -|newspaper=东方网 - 马来西亚东方日报|accessdate=2017-02-13|language=zh}}</ref> 但慕尤丁还是巫统署理主席，继续批评巫统，最后2016年6月被党最高理事会决议开除党籍。慕尤丁表示不曾背叛党，还是会继续发声。

== 土著团结党 ==
2016年8月，慕尤丁与[[马哈迪|马哈迪]]注册新政党，党名为[[土著团结党|土著团结党]]。慕尤丁为该党主席，而马哈迪及儿子[[慕克里|慕克里]]分别担任名誉主席和副主席。新党主要向土著开放，非土著虽可以入党但不能参与党选及投票<ref>{{citeweb|url=http://www.straitstimes.com/asia/se-asia/muhyiddin-registers-mahathirs-new-party|title=Muhyiddin registers Mahathir's new party|publisher=Strait Times|date=10 August 2016|accessdate=27 September 2016}}</ref>。马哈迪曾表示如果下一场大选[[希望联盟|希望联盟]]胜出，而[[安华|安华]]又无法参选，该党将会建议委任慕尤丁担任[[马来西亚首相|首相]]一职<ref>{{citeweb|url=http://www.themalaymailonline.com/malaysia/article/dr-m-tips-muhyiddin-to-be-pm-if-opposition-wins|publisher=Malay Mail|date=27 September 2016|accessdate=27 September 2016|title=Dr M tips Muhyiddin to be PM if opposition wins}}</ref>。2018年5月21日，慕尤丁正式被新任首相[[马哈迪·莫哈末|马哈迪]]委任为[[马来西亚内政部长|内政部长]]。
