'''調景嶺公共圖書館'''（{{lang-en|'''Tiu Keng Leng Public Library'''}}）是[[香港|香港]][[西貢區|西貢區]]的一座[[圖書館|圖書館]]，屬[[康樂及文化事務署|康樂及文化事務署]][[香港公共圖書館|香港公共圖書館]]轄下的[[分區圖書館|分區圖書館]]，該圖書館[[總樓面面積|樓面面積]]2,900平方米，位於[[將軍澳|將軍澳]][[調景嶺|調景嶺]][[翠嶺路|翠嶺路]]4號（毗鄰[[香港知專設計學院|香港知專設計學院]]），2015年7月9日啟用，主要為[[將軍澳市中心|將軍澳市中心]]的居民服務。<ref>[http://www.info.gov.hk/gia/general/201507/06/P201507060417.htm 調景嶺公共圖書館周四啟用　提供多元化圖書館服務（附圖）  2015-07-06]</ref>為將軍澳第二個公共圖書館。

== 設計 ==
圖書館由[[呂元祥建築師事務所|呂元祥建築師事務所]]設計，風格以簡約設計為主，以白色和灰色作主調，並透過中庭及玻璃外牆來擴大空間感及採光，將外面的景色融入館內。中庭的假天花亦掛上白色、綠色和燈色的吊燈。

而兒童圖書館的設計充滿繽紛色彩，以大自然為主題，並設大量座位及活動空間。而書架的指示牌設計為動物圖案，感覺具趣味性。

== 設施 ==
圖書館樓高3層，佔地超過2,900多平方米，地下設兒童圖書館、自助借書及續借服務、自修室和推廣活動室；1樓設成人圖書館及報刊閲覽室、和電腦資訊中心；2樓設成人圖書館、多媒體資訊系統、電腦資訊中心及咖啡閣。學生自修室和報刊閱覽室已分別於2015年3月和6月啟用。<ref>[http://www.news.gov.hk/tc/city_life/html/2015/07/20150706_130156.shtml 調景嶺公共圖書館周四啟用 香港政府新聞網 2015-07-06]</ref>天台為綠化樓層，暫不開放。

== 保安 ==
雖然圖書館設計創新，唯該圖書館規則與香港所有圖書館一樣，禁止公眾進行拍攝及錄影，而此館與[[屏山天水圍公共圖書館|屏山天水圍公共圖書館]]一樣，對禁止攝影的管理較一般圖書館更嚴。場內正門入囗、兒童圖書館每個角落、閲讀座位區以至窗邊的樓梯亦貼上多張「勿在館內拍攝」告示及派遣保安作為監視，以提防有人拍攝及錄影。圖書館只亦不容許普通市民進行攝影申請，可見管理者的作風十分保守。

== 獎項 ==
項目獲得綠建環評鉑金評級，同時獲選為「環保建築大獎 2014」及「世界建築節大獎2014」的入圍項目。<ref>[http://www.archsd.gov.hk/archsd/html/report2015/tc/greening-and-landscaping.html 資源運用及管理 / 綠化園境  建築署 - 可持續發展報告2015]</ref>
