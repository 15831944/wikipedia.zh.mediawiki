{{日本作品|調教咖啡廳|ブレンド・S}}是[[日本|日本]][[漫畫家|漫畫家]][[中山幸|中山幸]]的[[四格漫畫|四格漫畫]]作品，於[[芳文社|芳文社]]雜誌《[[Manga_Time_Kirara_Carat|Manga Time Kirara Carat]]》2014年3月號开始連載<ref>{{cite web|url=http://www.dokidokivisual.com/magazine/carat/book/index.php?mid=361|title=Manga Time Kirara Carat 2014年3月號|publisher=[[芳文社|芳文社]]|accessdate=2017年3月16日|language=ja}}</ref>。已出版5冊[[單行本|單行本]]，台灣[[尖端出版|尖端出版]]已出版4冊。2016年12月27日在《Manga Time Kirara Carat》宣佈推出電視動畫，2017年10月7日起於[[東京都會電視台|東京都會電視台]]、[[AT-X|AT-X]]、[[日本BS放送|日本BS放送]]等電視台播出。

== 故事簡介 ==
以咖啡廳「Stile」為舞台，講述着店員之間的交流，新人店員莓香來到了有著傲嬌、妹系等各種「屬性」的咖啡廳，而且被店長指派的是「虐待狂」的S屬性！？努力工作的同時，也剛好來的客人M屬性很多，意外的也讓「虐待狂」才能漸漸開花……充滿歡樂的打工由此展開！

== 登場角色 ==
=== 咖啡廳「STILE」 ===
==== 侍應 ====
; {{nihongo|櫻之宮莓香|桜ノ宮 苺香（さくらのみや まいか）||聲：[[和氣杏未|和氣杏未]]<ref name="character">{{cite web|url=http://blend-s.jp/character/|title=CHARACTER|work=電視動畫「調教咖啡廳」官方網站|language=ja}}</ref>（日本）；[[林美秀_(配音員)|林美秀]]（台灣）}}
: 年齢：16歲／生日：4月4日／星座：[[白羊座|白羊座]]／身高：156cm／擔當：[[抖S|抖S]]角色
: 黑色（動畫中偏紫色）長髮，平时会梳成双长辫，因為髮絲很細，所以常常受靜電影響而打結。性格認真且禮貌，做事雖然很努力，不過也因為笨拙而常失敗。
: 雖然本人沒有惡意，但經常因為不擅長與人交流而感到尷尬，一旦認真，緊張時眼神會變可怕，又天然地說了一些「抖S」的話容易導致他人誤解，莓香對此也感到煩惱。但是，提諾對包含這樣的性格一見鍾情，因此邀請她來STILE咖啡廳擔任抖S角色。
: 生活在充滿和服、和食的日本傳統風格的家庭環境，反而對和式感到厭煩。5歲時見到父親邀請的外國人來家裡而迷上了外國，以憧憬的海外留學為目標打工著。
: 本人有[[腹黑|腹黑]]兼女王傾向，不過本人毫無自覺。雖然是幺妹，但是其家中的權勢似乎超越兄姊。

; {{nihongo|日向夏帆|日向 夏帆（ひなた かほ）||聲：[[鬼頭明里|鬼頭明里]]{{r|character}}（日本）；[[許淑嬪|許淑嬪]]（台灣）}}
: 年齢：17歲／生日：8月12日／星座：[[獅子座|獅子座]]／身高：165cm／擔當：[[傲嬌|傲嬌]]角色
: 金髮[[雙馬尾|雙馬尾]]，[[巨乳|巨乳]]。STILE咖啡廳的大堂負責人之一，同時也是STILE咖啡廳最受歡迎的女服務生。
: 愛好是精通遊戲，所有種類都非常擅長，但是經常因為在遊戲廳看到高難度的遊戲，不自覺地想去挑戰導致錢都花光而苦惱着。另外因為沉迷遊戲而在現實中朋友很少。學習非常差勁，測驗經常不及格，因此麻冬常常幫助夏帆溫習功課。
: 與秋月紅葉同一時期到STILE咖啡廳打工，會以某種方式關心對方的存在。原來的擔當角色並不是傲嬌，但在看到紅葉的真傲嬌性格後，便決定努力擔當此角色直到現在。

; {{nihongo|星川麻冬|星川 麻冬（ほしかわ まふゆ）||聲：[[春野杏|春野杏]]{{r|character}}（日本）；[[林沛笭|林沛笭]]（台灣）}}
: 年齢：20歲／生日：12月23日／星座：[[摩羯座|摩羯座]]／身高：135cm／擔當：[[妹妹|妹妹]]角色
: 茶色短髮。雖然是大學生，但屬於矮小且平胸的幼女體型，有個比自己高的弟弟。夢想成為小學老師或幼稚園褓姆。
: 經常教訓偷懶的提諾。平時面無表情且安靜，工作時會暗示自己變得開朗和孩子氣，很受小孩子的喜愛，在小孩子稱呼她「姐姐」時會感到非常開心。非常喜愛兒童電視節目，每周都會觀看。
: 手工靈巧，會把自製的小物品送給顧客。在後期開始與STILE咖啡廳的一名叫「{{nihongo|伊藤|イトウ|聲：[[武內駿輔|武內駿輔]]}}」的高個子男性熟客以朋友關係保持聯絡。

; {{nihongo|天野美雨|天野 美雨（あまの みう）||聲：[[種崎敦美|種崎敦美]]{{r|character}}（日本）；（台灣）}}
: 年齢：22歲／生日：6月9日／星座：[[雙子座|雙子座]]／擔當：[[姐姐|姐姐]]角色
: 粉紅色雙麻花辮髮型。筆名「{{nihongo|花園文件|花園フォルダ}}」，[[同人|同人]]創作社團的主催者，以創作包含百合、[[BL_(和製英語)|BL]]、NL題材的[[十八禁|十八禁]]作品為主。平時有戴眼鏡。自稱是神崎日照的粉絲。
: 起初在STILE咖啡廳遺漏了自己的作品，之後找回後覺得STILE咖啡廳眾人的特色很適合作為創作素材積累而選擇在STILE咖啡廳打工，平時也會忙於同人創作趕稿而申請休假。喜歡觀察人群，覺得莓香與提諾的關係很有「萌系」要素，便仿照兩人繪畫出同人誌。

; {{nihongo|神崎日照|神崎 ひでり（かんざき ひでり）||聲：[[德井青空|德井青空]]{{r|character}}（日本）；（台灣）}}
: 年齢：16歲／生日：7月10日／星座：[[巨蟹座|巨蟹座]]／擔當：[[偶像_(日本)|偶像]]角色
: [[偽娘|偽娘]]。白色長髮。農家出身，却十分厌恶农村生活，為能夠讓將自己可愛的一面獲得世人的認同加入STILE咖啡廳，夢想是成為可愛偶像的偽娘店員，身著自己修改的制服。對自己的可愛外貌非常有信心。雖然性格有些傲慢，說話刻薄，但也有男子漢氣概的一面。

==== 廚房 ====
; {{nihongo|提諾|ディーノ|Dino|聲：[[前野智昭|前野智昭]]{{r|character}}（日本）；[[蔣鐵城|蔣鐵城]]（台灣）}}
: 年齢：26歲／生日：5月5日／星座：[[金牛座|金牛座]]／身高：188cm／擔當：店長兼廚房
: [[意大利|意大利]]出身，[[金髮|金髮]]。一位非常喜歡美少女動畫與人物的[[御宅族|御宅族]]，其在咖啡廳二樓的住所裡經常填滿如手辦等各種各樣的御宅商品，經常看深夜動畫而第二天太累在工作上打瞌睡，也因此常常被星川麻冬收拾。有著一興奮就會流鼻血的怪癖。
: 偶然在火車站見到莓香一見鍾情，之後莓香經過STILE咖啡廳時再次見到後發現其滿足作為「抖S」的條件而招聘其加入，一涉及到莓香的事就會得意忘形或者興奮到流鼻血，一旦被莓香抖S時，都會露出「抖M」的體質。雖說是店長卻經常不被尊重。

; {{nihongo|秋月紅葉|秋月 紅葉（あきづき こうよう）||聲：[[鈴木達央|鈴木達央]]{{r|character}}（日本）；[[何志威|何志威]]（台灣）}}
: 年齢：21歲／生日：11月28日／星座：[[射手座|射手座]]／身高：170cm／擔當：廚房
: 髮色是草色。[[三白眼|三白眼]]。STILE咖啡廳的大堂負責人之一。因抱有「更容易獲得厨师执照」的理由而與日向夏帆同一時期到STILE咖啡廳打工，由於店長提諾相當偷懶，廚房工作一直由他負責。和夏帆是同期員工，倆人亦有著共同的興趣，會以某種方式關心對方的存在。
: [[百合_(藝術類型)|百合]]愛好者，喜歡觀察女性顾客，對女孩子之間的互動完全沒有抵抗力，覺得很神秘並且充滿可能性，希望店鋪能多些女性顧客，但是自己對現實中的女性交談時則膽小且沒耐心。天生具備傲嬌屬性。雖然是男性員工而平時也沒有接客，但似乎也有著少量的粉絲。
