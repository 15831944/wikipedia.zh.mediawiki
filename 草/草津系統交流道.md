'''草津系統交流道'''（{{lang|ja|くさつジャンクション}}）是位於[[滋賀縣|滋賀縣]][[草津市|草津市]]的[[名神高速公路|名神高速公路]]及[[新名神高速公路|新名神高速公路]]大津連絡路之[[系統交流道|系統交流道]]。附近設置了[[草津田上交流道|草津田上交流道]]。

== 連接道路 ==
*[[名神高速道路|名神高速道路]]
*[[新名神高速道路|新名神高速道路]]大津連絡路（以此系統交流道為起點）

== 历史 ==
*1963年7月16日 - [[名神高速道路|名神高速道路]][[栗東IC|栗東IC]]至[[尼崎IC|尼崎IC]]開通。
*2005年3月19日 - 名神高速道路支線草津JCT至[[草津田上IC|草津田上IC]]開通，此系統交流道也同時啟用。
*2008年2月23日 - [[新名神高速道路|新名神高速道路]]大津連絡路全線開通，草津JCT至草津田上IC路段編入新名神高速道路大津連絡路。

== 與草津PA關係 ==
由於草津PA與此系統交流道很接近，所以使用此系統交流道前往不同方向或需進入草津PA時，便必須留意路牌指示：
;大津連絡路→草津PA
在進入草津JCT後，在第一個分支點靠左，然後在第二個分支點再靠左，進入草津PA，在第二個分支點靠右會進入名神高速道路本線京都方向。
;名神高速道路米原方向→（草津PA）→大津連絡路
如果不需進入草津PA，沿名神高速道路本線米原方向至草津JCT，在分支點靠左，便進入大津連絡路。<br>
如果需進入草津PA，在離開草津PA後，在分支點靠右，便進入大津連絡路，在分支點靠左會進入名神高速道路本線米原方向。
;名神高速道路京都方向→草津PA
在進入草津JCT後，在第一個分支點靠左，然後在第二個分支點靠右，進入草津PA，在第二個分支點靠左會進入大津連絡路。
