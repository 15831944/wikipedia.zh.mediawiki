'''草榴社区'''是一个[[繁体中文|繁体中文]]網路[[色情|色情]][[网络论坛|论坛]]，建立于2006年11月。社区在2007年4月正式取得固定[[域名|域名]]及地址，论坛[[服务器|服务器]]架设於[[美国|美国]][[科罗拉多州|科罗拉多州]]。目前該论坛具会员数超过20万。由于涉及成人内容，为中國大陸“[[扫黄打非·净网2014|扫黄打非]]”的封杀对象<ref>[http://news.sohu.com/20100531/n272463947.shtml 扫黄打非办四个月收到14万条举报，低俗标准模糊]，瞭望东方周刊， 2010年5月</ref><ref>[http://jiangxi.jxnews.com.cn/system/2009/08/21/011187113.shtml 打击网络淫秽犯罪陷计量之困 专家呼吁完善相关立法]，大江网-新法制报，2009.08.21</ref>。

由于涉及成人内容及敏感内容，草榴一直是中国政府的封杀对象，“草榴社区”的中国国内维护人员，在2009年4月被法院以传播淫秽物品牟利罪判处有期徒刑3年，缓刑3年，并处罚金5000元。但是“草榴社区”的运行并没有终止，在不断被封杀的同时不断更换地址，并通过各种渠道及时向会员更新。

== 历史 ==
社区2006年11月创立时，实行开放注册制，任何人均可免费注册。自2007年1月29日后，社区改為实行邀请注册，须输入由現会员用贡献值购买的[[邀请码|邀请码]]方可注冊。2011年6月，其机房发生大火，部分服务器在大火中损毁，部分资源及会员资料丢失，同月6日，社区开放注册两天，两天内注册会员超过13万，並因而再度关闭开放注册。现在草榴社区仍需邀请码才能注册，在很多技术讨论区的帖子中会有老会员以发“福利”的形式发放邀请码，但基本都很快被抢光。

2015年5月17日始，草榴社区因为数据丢失无法正常访问。2015年5月18日，草榴社区出現消息指，称因为[[網絡防火長城|網絡防火長城]]攻击可能会导致网站永久关闭。<ref>[http://tech.ifeng.com/a/20150519/41085210_0.shtml 草榴将永久关闭 网友感慨“做好人真难”]，[[凤凰网|凤凰网]]，2015年05月19日</ref>但随后被官方辟谣，称这条消息是谣传，网站仅仅是在整理数据。

== 规则宗旨 ==
*社区宗旨：凡年满18岁人士，在遵守版规前提下，并由版主认可，才可自由发布言论，畅所欲言。
*社区特色：会员以“1024”回复主帖是草榴社区的特色，这是由于社区限制等级较低的用户每隔1024秒仅可回复一次，因此回帖表示其对该帖的支持。以“1024”或“+1024”回复帖子的方式在其他论坛同样流行起来，起初含有暗指自己活跃于草榴的意思，后来则演变为诙谐表达赞同的通用回帖方式。

== 文化影响 ==
雖然是成人论坛，但是社群也经常对社会事件进行讨论。在中国大陆[[2011年甬台温铁路列车追尾事故|2011年甬台温铁路列车追尾事故]]发生后，草榴社区网友大批上传事发一日照片，事件还被《[[澳洲日报|澳洲日报]]》报道。<ref>[http://dailynews.sina.com/bg/chn/chnnews/ausdaily/20110727/18112636891.html 報導溫州動車意外事故色情網站竟比官方詳細]，澳洲日报，2011年07月27日</ref>
