'''帝王陵墓'''，或稱'''帝王陵園'''，简称“'''皇陵'''”或“'''帝陵'''”，是指[[君主|君主]]的[[墳墓|墳墓]]。陵墓中深深渗透着[[礼法|礼法]]的内容，[[天子|天子]]死后的埋葬地称“陵”，[[诸侯|诸侯]]称“封”，[[大夫|大夫]]称“坟”。坟丘的大小、衣襟[[棺椁|棺椁]]、墓葬仪式都是按等级严加规定的。

== 東亞帝王陵墓 ==
{{ see also|中國君主陵墓列表}}
東亞在丧葬方面有“[[视死如生|视死如生]]”、“事死如事生”（也有人写作“侍死如侍生”）的传统，生前，帝王有奢华的享受，死后，亦有如此。帝王通常在登基后，即开始为自己营造陵墓。帝王的陵墓通常规模宏大，花费巨大，即便是以节俭著称的帝王也不例外。
===中國===
[[西汉|西汉]]的[[汉文帝|文帝]]，开创[[文景之治|文景之治]]。《[[史记|史记]]》称文帝在建造[[霸陵|霸陵]]时，皆用瓦器，不得以金银铜锡为饰，不治坟，想要节俭，不烦扰民众<ref>《[[史记|史记]] 孝文本纪第十》孝文帝从代来，即位二十三年，宫室苑囿狗马服御无所增益......治霸陵皆以瓦器，不得以金银铜锡为饰，不治坟，欲为省，毋烦民。</ref>。实际上霸陵的修建，耗费了全国三分之一的[[财政收入|财政收入]]<ref>{{Cite web |url=http://hsb.hsw.cn/gb/newsdzb/2002-11/07/2002-11-07-11zhxw3.htm |title=汉文帝霸陵如今成荒冢 |accessdate=2013-05-27 |author=记者：雨石，实习记者：豫章|date=2002-11-07  |work= |publisher=华商网|language=Zh-hans }}</ref>。

1956年至1957年期间，尽管有[[郑振铎|郑振铎]]、[[夏鼐|夏鼐]]等人的反对，但在时任北京市副市长[[吴晗|吴晗]]的极力支持下，[[中华人民共和国国务院|中华人民共和国国务院]]决定对[[明定陵|明定陵]]进行发掘。但十数年后，即在[[文化大革命|文化大革命]]中遭到严重损害。不主动挖掘帝王陵墓，亦成为中华人民共和国政府的政策<ref>{{Cite web |url=http://news.sina.com.cn/o/2007-01-04/140510924414s.shtml |title=帝陵 挖还是不挖 |accessdate=2013-05-30 |author= |date=2007年1月4日 |work= |publisher=新浪网，来源：大众网-农村大众，据《南方周末》|language=Zh-hans }}</ref> 。

=== 特有建築 ===

*石[[牌坊|牌坊]]：明[[十三陵|十三陵]]陵区最南端的建筑，在大宫门南二里许，建于[[嘉靖|嘉靖]]十九年（1540年），石牌坊5间6柱11楼，高14米，宽28.86米，是现今国内保存下来、最精美的石牌坊。牌坊的夹杆石，四面有[[浮雕|浮雕]]，刻有八对狮子滚绣球和十六条云龙图案。
*[[圣德神功碑|圣德神功碑]]楼：记述皇帝生平功绩的主要建筑，俗称大碑楼。楼内圣德神功碑上刻有为皇帝歌功颂德的文字。
*[[谥号碑|谥号碑]]亭：又称作神道碑亭、小碑亭或小碑楼，是放置[[谥号|谥号]]碑的亭式建筑。
*[[神道_(道路)|神道]]：陵墓前的一条大道。为陵墓的附属建筑，是进入陵区的主要通道。
*[[石象生|石象生]]：指在神道两侧放置的石人石兽。分别排列着十二对石兽，两坐两立，依次为[[狮子|狮子]]（象征威武）、[[獬豸|獬豸]]（象征公正）、[[骆驼|骆驼]]（象征运输）、[[象|象]]（象征吉祥太平）、[[麒麟|麒麟]]（象征吉祥太平）、[[马|马]]（象征征战），其中骆驼、象、马又分别为各地运输工具，故又象征疆域辽阔；石人十二尊，其中武臣（象征侍卫将军）、文臣（象征近身文臣）、勋臣（象征有功勋的文武百官）各四尊。陵墓前的石人又称翁仲，对称立有两座，分别为文官和武官形象。关于翁仲的形象有一种说法是来源于秦朝有一位名叫[[阮翁仲|阮翁仲]]的大力士，传说他身长一丈三尺，力大无比,曾驻守临洮，征服匈奴有功。他死后，秦始皇特制翁仲铜像立于咸阳宫司马门外。匈奴人来咸阳，见到铜人，竟为是活着的阮翁仲而逃遁。从此，人们便把宫阙或陵墓前的铜人、石人称为翁仲。陵墓前的石兽也有由来。石兽放置在墓前是从汉代[[霍去病|霍去病]]墓开始的。后来的历代帝王的修建自己的陵墓时也都沿用石人石兽的陵前装饰，所以现在的唐陵、宋陵、明陵、清陵几乎都在陵墓前陈列仪仗队式的石人石兽。
*[[享殿|享殿]]
*[[宝城|宝城]]：指环绕在宝顶四周的封闭城垣建筑。
*[[宝顶|宝顶]]：指位于地宫上方的圆丘状[[封土|封土]]。
*[[地宫|地宫]]：安放死者[[棺椁|棺椁]]的地下建筑。
