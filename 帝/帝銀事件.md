'''帝银事件'''，是发生于1948年1月26日[[东京都|东京都]][[丰岛区|丰岛区]]帝国银行（后来的[[三井银行|三井银行]]，現時為[[三井住友銀行|三井住友銀行]]）内的一起银行抢劫案，案中造成12人死亡。这是一起发生在日本戰後[[同盟國軍事占領日本|同盟軍占領]]期间的案件，虽然其後抓获了罪犯并判了[[死刑|死刑]]，但案件仍然有诸多疑点，被較普遍認為是一[[冤假錯案|冤假錯案]]。

== 事件经过 ==
1948年1月26日下午3时，在东京都丰岛区帝国银行椎名町支店快要关门的时候，突然进来了一个约45岁、穿着[[西服|西服]]带着红袖章的男子。他一邊向店员散布[[名片|名片]]一边寻找支店店长，因为店长腹痛所以由代理支店长接待。男子的名片上写着“[[厚生省|厚生省]]厚生部某医学博士”，他声称附近有四人感染了[[痢疾|痢疾]]，其中一人曾来过这家银行，他接到同盟军司令部的指示前来分发预防药，消毒班稍后也会过来。他宣称药物共分两种，第一种药物和牙齿的[[珐琅质|珐琅质]]接触时会有疼痛感，因此要直接吞下。第二种药物则要在一分钟后再服用。在他率先示范过后，包括一名职员的小孩在内的16人都照着吩咐喝下药水，据幸存者描述喝药时非常痛苦，这种感觉就像是不能喝[[酒|酒]]的人突然喝烈酒一样。一分钟后服用了第二种药之后有人提出要去漱口，中年男子同意，但是在走向[[厨房|厨房]]的过程中店员一个接一个的晕倒。意识模糊的职员村田正子爬到了道路上，向两名女[[学生|学生]]求助，后者报告了附近的警署。16人中有11人当场死亡，一人在送往[[医院|医院]]的途中断气。<ref name=lj>{{Cite web |url=http://gonta13-lj-hp.web.infoseek.co.jp/newpage8.htm |title=帝銀事件 |access-date=2010-12-11 |archive-url=https://web.archive.org/web/20110116215103/http://gonta13-lj-hp.web.infoseek.co.jp/newpage8.htm |archive-date=2011-01-16 |dead-url=yes }}</ref><ref>[http://showa.mainichi.jp/news/1948/01/post-16d9.html 帝銀事件]</ref>

== 调查 ==
尸检结果顯示，所谓的第一种药物其实是剧毒的[[氰化物|氰化物]]。总计有16万3410[[日元|日元]]现金和一张1万7450日元的[[支票|支票]]被拿走。而现场也没有发现犯人当日分发的名片，推测是事后被犯人拿走了。根据幸存者的回忆绘制出了中年男子的肖像，那张1万7千日元的支票在第二天于[[安田银行|安田银行]]板桥支店被成功兑换。支票的背书上地址、姓名分别写着“板橋三の三六六一”、“後藤丰治”，调查表明这是一個不存在的地址和人名。根据安田银行代理支店店长的回忆，兑换支票的人身高约5尺3寸，肩膀圆厚，说话不带口音。因为犯人使用氰化物作为犯罪工具，[[731部队|731部队]]的相关人员成为了重点怀疑对象。經過大规模的排查後发现，案发前曾发生过两起类似的案件。<ref name="tk">[http://www.tokyo-kurenaidan.com/teigin.htm 帝銀事件を歩く]</ref>

* 安田银行荏原支店：1947年10月14日下午3点左右，一名中年男子来到安田银行荏原支店，向职员分发厚生技官医学博士'''松井蔚'''的名片。然后声称曾有[[痢疾|痢疾]]患者来过这家银行，应占领军要求大家服用预防药物。然后像帝国银行事件中那样，让大家依次服用两种药物，所不同的是职员们喝下药后没有出现异常状况。调查發現，确实有松井蔚这个人，而且的确是厚生省的[[医生|医生]]。<ref name="tk"/>

* 三菱银行中井支店：1948年1月19日下午关门的时候，有一个中年男子用类似的手段声称要让大家服下药物，并分发了厚生省技官医学博士山口二郎名片。但是因为声称痢疾患者来过这里存钱而受到了店员们的质疑，于是此人只是在地上撒了点透明的药水就离开了。这一次的名片是伪造的假名片。<ref name="tk"/>

警察们在仙台找到了松井蔚博士，博士本人有不在场的证明。根据博士的回忆，他印制了100张名片，其中有6张还没有发出去。搜查班结合博士的记忆对剩下的94张进行了搜索，结果找回62张，证明24张丢失，另外8张因不知去向而成为重点排查对象。最终，[[北海道|北海道]][[小樽|小樽]]的56岁[[蛋彩画|蛋彩画]]画家[[平泽贞通|平泽贞通]]成为了怀疑对象。他曾在一艘渡船上同松井蔚博士交换过名片，但是以钱包被盗为理由未能拿出名片。1948年8月21日，警方在小樽市逮捕了他。

== 争议与审判 ==
幸存者看過平泽的[[相片|相片]]后未得出确定的答复。平泽在刚遭到逮捕时否认控罪，但卻於9月23日作出招认的供词，10月12日被起诉。12月20日，平泽在东京地方裁判所判决时又进行了翻供，1950年7月24日地方裁判所一审判决死刑，1955年4月7日向最高法院[[上诉|上诉]]遭到拒绝，5月7日被确定为死刑。在之后的岁月里，平泽不断进行上诉并3次试图[[自杀|自杀]]，但都自杀未遂。一直到1987年5月10日，已屆95歲的平泽才在八王子医疗监狱内因病去世，此时距离死刑判决已过去30年。<ref name=lj/>

对平泽的[[审判|审判]]存在诸多争议，例如警官[[平塚八兵衛|平塚八兵衛]]在办案过程中刑讯逼供，平泽本人也患有[[科尔萨科夫综合征|科尔萨科夫综合征]]的后遗症，他的不在场证明是当时处于家中，因为证人都是直系亲属因而没有效用。平泽也没有使用氰化物的知识、经验和获取氰化物的途径。有些人认为平泽的被捕只不过是为了转移对731部队的注意，直到他死后仍有人在争取为其翻案。<ref>[http://www.tanken.com/teigin.html 帝銀事件について]</ref>
