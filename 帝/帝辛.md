'''帝辛'''<ref>[[武庚|武庚]]給父親上諡號</ref>，子姓，諱'''受'''或'''受德'''{{noteTag|1= 據《[[中華通史|中華通史]]》第一冊，[[章嶔|章嶔]]著}}，是[[商朝|商朝]]最后一代的君主。[[夏商周断代工程|夏商周断代工程]]認為他在前1075年至前1046年在位。

帝辛非常重视[[农业|农业]]和社会生产力的发展，使国力强盛。在位期间，继续对东南方的[[人方|人方]]（[[东夷|东夷]]）用兵，最后将其击败，将商朝势力扩展到[[江淮|江淮]]一带。然而，这场长久的战争导致了国内空虚。后来，[[周朝|周國]]联合西方各[[方国|方国]]一起讨伐商朝。在[[牧野之战|牧野之战]]中，商军溃败，帝辛[[自焚|自焚]]而死，商朝灭亡<ref>{{cite web|title=商纣王帝辛
|url=http://www.todayonhistory.com/people/201506/9629.html}}</ref>。死后，其子[[武庚|武庚]][[谥|谥]]他为'''帝辛'''。[[周武王|周武王]]在其生前稱呼他是“'''商王受'''”。帝辛死後，[[周武王|周武王]]便給他一個惡谥之名'''纣'''，因此帝辛以'''纣王'''一名为后世人所熟知。

== 生平 ==
=== 幼年 ===
帝辛高大威猛，力大無窮，自幼聪敏过人、反應敏捷。《[[荀子|荀子]]·非相篇》说帝辛“长巨姣美，天下之杰也；筋力超劲，百人之敌也。” 《[[史记|史记]]·[[殷本纪|殷本纪]]》也说“帝纣资辨捷疾，闻见甚敏，材力过人，手格猛兽”。

=== 继位 ===
帝辛继位后，定都于沫
（前1082年），后改[[沬邑|沬邑]]为[[朝歌|朝歌]]（今[[河南|河南]][[淇县|淇县]]）。他重视农桑，社会生产力发展，商朝国力逐渐强盛。在位期间继续对[[人方|人方]]（周人称之为[[东夷|东夷]]）、[[林方|林方]]、[[虎方|虎方]]等用兵，在经历了长期的战争之后最终取得了胜利，并将商朝势力扩展到[[江淮|江淮]]一带。然而，对人方的战争导致了国内空虚，最终导致了商朝的灭亡。<ref>《左传·昭公十一年》：“桀克有缗以丧其国，纣克东夷而陨其身。”</ref>

=== [[周武王|周武王]]灭商 ===
公元前1046年，[[周武王|周武王]]联合西方11个小国会师[[孟津|孟津]]，对[[商朝|商朝]]发起进攻。商军主力在东南，无法征调，只得将战俘临时武装起来进行抵抗。而此时帝辛已经受到了众多贵族和巫师等的集体反对，军心涣散，周武王以在距離[[朝歌|朝歌]]（今[[河南|河南]][[淇縣|淇縣]]）七十-{里}-外的牧野与商軍交战，商軍倒戈而前军攻击后军。商军因而大败，此战就是历史上的[[牧野之战|牧野之战]]。周军随即攻至朝歌。帝辛登上鹿台，穿上宝玉衣，投于火中自焚而死，商朝灭亡。<ref>{{Wikisource history|史記|003|帝辛|quote=甲子日，纣兵败。纣走入（朝歌），登鹿台，衣其宝玉衣，赴火而死}}《逸周书·世俘》曰：“甲子夕，纣取天智玉琰五，环身以自焚”。</ref>死後頭顱被周武王斬下，懸在白旗，兩位妻妾的頭顱懸在紅旗上，被送入周太廟祭拜周人祖先。<ref>《史記》〈周本紀〉「(周武王)遂入，至紂死所。武王自射之，三發而後下車，以輕劍擊之，以黃鉞斬紂頭，縣大白之旗。已而至紂之嬖妾二女，二女皆經自殺。武王又射三發，擊以劍，斬以玄鉞，縣其頭小白之旗。」</ref><ref>《逸周書》〈世俘解〉：「武王在祀，大師負商王紂縣首白旂，妻二首赤旂，乃以先馘，入燎于周廟。」</ref>

帝辛死后，葬于[[淇水|淇水]]之滨。

周武王征服商朝之后，改封其子[[武庚|武庚]]为诸侯，令其统治朝歌一带，又派三个弟弟[[管叔|管叔]]、[[蔡叔|蔡叔]]、[[霍叔|霍叔]]监视其举动。

== 考古研究 ==
考古学者认为，[[殷墟遗址|殷墟遗址]]的1567号墓是帝辛的墓。<ref>[http://www.xiancn.com/gb/wbpaper/2007-07/16/content_1251496.htm 规模宏大的安阳殷墟商代王陵(图)]</ref>

[[周朝|周朝]]灭商以后，纣王与[[夏桀|夏桀]]一起被当作是暴君的典型，不过近年以来有不少学者对此提出怀疑。

[[毛澤東|毛澤東]]认为“其實紂王是個很有本事、能文能武的人。他經營東南，把[[東夷|東夷]]和[[中原|中原]]的統一鞏固起來，在歷史上是有功的。紂王伐[[徐州|徐州]]之夷，打了勝仗，但損失很大，俘虜太多，消化不了，[[周武王|周武王]]乘虛進攻，大批[[俘虜|俘虜]]倒戈，結果[[商朝|商朝]]亡了國。”“商紂王是很有本領的人，[[周武王|周武王]]把他說得很壞。他的[[俘虜|俘虜]]政策做得不大好，所以以後失敗了。”

近代[[史學家|史學家]][[顧頡剛|顧頡剛]]也曾綜合考證，發現紂的70多條罪狀都是[[周朝|周朝]]以後逐漸增加，劇情也逐漸強烈，[[戰國|戰國]]時期增加了20項、[[西漢|西漢]]增加21項、[[東晉|東晉]]增加13項，至於[[比干|比干]]之死[[夏商周斷代工程|夏商周斷代工程]]的深入研究發現與史記上紀載有出入的證據逐漸增多，現存[[比干|比干]]墓寺廟中碑文有顯示其比紂王還晚死十多年，不可能是紂王所殺，此一證據不能說是鐵證，但已經構成懷疑探討點，而所謂比干諫言而死是到[[春秋時期|春秋時期]]才有紀載，已相隔甚遠，<ref>[http://www.hunantv.com/v/5/1137/f/258573.html 湖南衛視-商纣王的历史真面目]</ref>且諸多文獻表明隨著歷史推演紂王的描述越來越殘暴，新增的情節也越多，[[炮烙之刑|炮烙之刑]]詳細紀載為[[夏桀|夏桀]]發明，後不知怎麼也成了紂王，不能排除後人醜化的可能。到了戰國時期又有了[[屈原|屈原]]的說法比干其實是投水而死。

===史书中对其暴虐事迹的记载===
《[[史記|史記]]》记载，帝辛“知足以拒諫，言足以飾非”；又說他“好酒淫樂，嬖于婦人{{noteTag|1= 意思是宠信[[妲己|妲己]]，建立[[酒池肉林|酒池肉林]]}}，以酒為池，懸肉為林，使男女裸相逐其間，為長夜之飲”；為人凶殘成性；殺害忠臣义士，如其叔父[[比干|比干]]；囚禁异己，如[[西伯昌|西伯昌]]被幽禁羑里（今河南[[湯陰|湯陰]]）七年之久。他為了觀察胎兒，竟殘忍地讓人剖開孕婦的肚子。他想知道冬天光腳過河的農夫為什麼不怕冷，竟叫人砍掉他的雙腳。{{noteTag|1=《[[呂氏春秋|呂氏春秋]]．貴直論．過理》：“糟丘酒池，肉圃為格，雕柱而桔諸侯，不適也。刑鬼侯之女而取其環，截涉者脛而視其髓，殺梅伯而遺文王其醢，不適也。文王貌受，以告諸侯。作為琁室，築為頃宮，剖孕婦而觀其化，殺比干而視其心，不適也。” }}《[[战国策|战国策]]》记载“昔者鬼侯、鄂侯、文王，纣之三公也。鬼侯有子而好，故入之于纣，纣以为恶，醢鬼侯；鄂侯争之急，辨之疾，故脯侯；文王闻之，喟然而叹，故拘之于牖里之库百日，而欲令之死。”

《[[韩非子|韩非子]]·喻老》亦有記载：“昔者纣为象箸而箕子怖。”[[司马迁|司马迁]]在《[[史记|史记]]·宋微子世家》亦云“纣为象箸，[[箕子|箕子]]叹曰：彼为象箸，必为玉杯；为玉杯，则必思远方珍怪之物而御之。舆马宫室之渐自此始，不可振也”。

《[[帝王世纪|帝王世纪]]》记载，帝辛在位期间，西伯的嫡長子[[伯邑考|伯邑考]]在殷商做人質，當時負責替帝辛駕車。但某日，帝辛派人烹殺了伯邑考，並將他做成了肉羹賜給[[周文王|西伯昌]]，並說：「聖人應當不會吃自己兒子做成的肉羹。」[[周文王|西伯昌]]最後還是吃了肉羹。帝辛評價說：「誰說西伯昌是聖人？吃了自己兒子做成的肉羹尚且不自知。」<ref>《史記正義·殷本紀》：帝王世紀云。「囚文王，文王之長子曰伯邑考質於殷，為紂御，紂烹為羹，賜文王，曰『聖人當不食其子羹』。文王食之。紂曰『誰謂西伯聖者？食其子羹尚不知也』。」</ref>

在众多文献中，帝辛是一个实行炮烙之刑，滥杀无辜，开设[[酒池肉林|酒池肉林]]，穷奢极侈的昏暴之君。他与[[夏桀|夏桀]]并论，在中国一直是暴君的代名词。这一段历史，後人加上[[想像|想像]]，寫成了《[[封神演义|封神演义]]》。成語有“[[助紂為虐|助紂為虐]]”{{noteTag|1= [[祖君彥|祖君彥]]《為李密檄洛州文》：“達等助紂為虐，嬰城自固。”}}。

===对暴虐记载的质疑===
在《[[尚书|尚书]]·[[牧誓|牧誓]]》中，[[周武王|周武王]]只列举了帝辛的四条罪状，分别是听[[妇人|妇人]]言、不认真[[祭祀|祭祀]]、不重用[[亲戚|亲戚]]、任用逃犯做官，然而“听妇人言”、“不重用亲戚”與“任用逃犯做官”在商代社会并不罕見，周武王有此一說可能是因當時周已演化出了完整的[[禮樂制度|禮樂制度]]，而對商朝的作為表示懷疑。

[[商朝|商朝]]时女性地位很高，女性可以拥有封地，还可以担任辅政[[大臣|大臣]]或[[将军|将军]]；到了[[西周|西周]]时，女性地位就大大下降了，不仅不能拥有[[封地|封地]]，也不能参与国事。“商纣王与[[苏妲己|苏妲己]]淫乱失国”的故事其实反映了殷人男女平等的历史。商朝时用人不重视[[家庭|家庭]]身份，[[王族|王族]]成员并没有太多被委以[[特权|特权]]；而[[西周|西周]]时期的[[等级制度|等级制度]]比商代要森严很多，实行[[宗法制|宗法制]]下绝对的[[世官制|世官制]]，在周人看来，不任用亲戚为官是严重罪过；商代政治组织松散，人口流动性强，诸侯中常有人逃往他国，其他国家的[[奴隶|奴隶]]、[[农奴|农奴]]逃往商国，就获得自由，一部分贤能之人甚至被任用为官员；殷墟[[甲骨文|甲骨文]]中有许多“小臣x”之类的人名，大多是出身卑微之人；周代政治组织严密，从[[周文王|周文王]]开始就实行“有亡荒阅”在全天下搜捕逃亡者，周人才指责帝辛“遗其王父母弟不用，乃维四方之多罪逋逃是崇是长”。<ref>杨宝成《殷墟文化研究》</ref><ref>王晖《商周文化比较研究》</ref>

[[殷墟|殷墟]]考古来看，商朝末期的[[人祭|人祭]]和[[人殉|人殉]]都很少，甚至没有[[西周|西周]]中早期严重。从各种文物中，看不出帝辛有太多暴行。西周初年时所宣扬的帝辛的罪状，除了《尚书·牧誓》中的那四条，也就只有[[酗酒|酗酒]]、相信自己有命在天了。到了后来，西周统治者和商王朝的叛徒继续指责帝辛，并造出了“纣”这个字。[[春秋战国|春秋战国]]时期的诸子不了解历史真相，为了向世人宣扬“恶有恶报”，便又给纣王增添了不少新的罪状，例如作[[炮烙之刑|炮烙之刑]]、把人剁成肉酱、剖割孕妇之胎等。[[汉代|汉代]]以后的统治者继续把纣王作为反面形象宣传。纣王的“荒淫残暴”实际上是千年积毁的结果<ref>顾颉刚《纣恶七十事的发生次第》</ref>。

[[子貢|子貢]]曾说过：“紂之不善，不如是之甚也。是以君子惡居下流，天下之惡皆歸焉。”{{noteTag|1=《[[论语|论语]]·子张》}}認為帝辛許多的罪過並未發生，是被後人所加的。南宋学者[[罗泌|罗泌]]在他的著作《[[路史|路史]]》中，也认为商纣王的许多罪行并不真实。

在《[[韩非子|韩非子]]》中提到，[[费仲|费仲]]劝谏帝辛杀掉西伯姬昌，但帝辛却认为“夫仁义者，上所以劝下也，今昌好仁义，诛之不可”（仁義的人，君主可以用來勸導臣民。[[姬昌|姬昌]]就是這種人，不應該殺。）没有采用费仲的建议<ref>《韩非子·外储说左上》</ref>。

根据《左传》等文献以及殷墟[[甲骨文|甲骨文]]，帝辛征服了淮夷 ，派[[蜚廉|飞廉]]出使北方，对中国古代的统一与各族文化的交流和发展有过一定贡献。《诗经·商颂·玄鸟》：“武王靡不胜。”《诗经·商颂·长发》：“武王载旆，有虔秉钺。”经郭沫若等学者考证，这里的“武王”是指帝辛，他在生前曾深受百姓的爱戴，是殷民心目中的“武王”<ref>郭沫若《殷契粹编》</ref>。河南淇县[[朝歌|朝歌]]镇有后人修建的纣王之墓，当地人民均以帝辛为荣。

== 相关文物 ==
帝辛时期的[[青铜器|青铜器]]有很多，例如现存于[[北京故宫|北京故宫]]的二祀邲其卣、四祀邲其卣、六祀邲其卣分别是帝辛二年、四年、六年时铸造的青铜器。这三件器物都提到了邲其这个人物，据铭文记载，他协助帝辛进行了祭祀，帝辛赏赐了他贝币。除此之外还有作册般青铜鼋、版方鼎、宰虎角等等。从当时的[[金文|金文]]中可以知道作册般为帝辛时期的史官。

== 後世創作 ==
纣王也是明代文學小說《[[封神演义|封神演义]]》中的一个登场人物，在该小说中也称他为“寿王”。

== 影视错误 ==
{{citation needed|“纣”是周朝统治者给的[[谥号|谥号]]，影视作品在他活着的时候称他为“纣王”，但当时帝辛还没有死，不可能用周朝统治者给他的“谥”。}}

根据《[[逸周书|逸周书]]·谥法解》可知，谥法起源于[[周公|周公]]旦（“维周公旦、太公望，开嗣王业，建功于牧之野，终将葬，乃制谥”）。也有人认为谥法起源于更晚些时候（[[周厉王|周厉王]]时期）。总之，在商纣王帝辛时期，[[谥号|谥号]]还未流行。另外，《[[牧誓|牧誓]]》中记载的[[周武王|周武王]][[牧野之战|牧野之战]]时发表的[[檄文|檄文]]，里面称呼帝辛为“商王受”<ref>{{cite web|title=毛澤東點評十八大帝王：知識分子沒出息（圖）|url=http://tw.aboluowang.com/2015/0922/617315.html}}</ref>。
