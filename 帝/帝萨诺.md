'''帝萨诺'''（{{lang-it|Disaronno Originale}}）是一款[[阿玛雷托|阿玛雷托]][[力娇酒|力娇酒]]，具有独特的苦甜杏仁的味道。 Disaronno声称其原创的秘密配方自1525年来从未改变过<ref name="Disaronno Story">[http://www.disaronno.com/home.aspx Disaronno] {{webarchive|url=https://web.archive.org/web/20091216105044/http://www.disaronno.com/home.aspx |date=2009-12-16 }}. Retrieved January 1, 2007. Home → Heritage → Page 4: ''The "Originale" Story''. (A direct link is not available due to the [[Adobe_Flash|Adobe Flash]]-based interface.)</ref>，并声称Luini传说是其自己的独特历史。

制造商介绍帝萨诺使用“杏仁油”与“无水酒精”和“焦糖”酿造，其成分还包括17种精挑细选的草药和水果萃取物。琥珀色的酒液被装在一个[[穆拉诺岛|穆拉诺]]工匠设计的长方形玻璃酒瓶中。

Disaronno目前仍然在[[意大利|意大利]]Saronno生产，其品牌由Illva Saronno持有，但已经远销海内外。

== 历史 ==
制造商声称Disaronno的起源就是著名的Luini传说。虽然这则故事的真实性有待考证，但至少它在Saronno本地文化中占据有一席之地。

1525年，Saronno一家教堂委托[[列奥纳多·达·芬奇|列奥纳多·达·芬奇]]的学生{{tsl|en|Bernardino Luini|伯纳迪诺·卢伊尼}}在他们的避难所绘制壁画<ref name="Bernardino Luini">it.wikipedia.org: {{tsl|it|Bernardino Luini|}}. Retrieved January 1, 2007.</ref>。由于教堂是献给[[馬利亞_(耶穌的母親)|圣母玛丽亚]]的，因此卢伊尼绘画的主题必须是[[聖母瑪利亞_(繪畫)|圣母]]，所以他需要一个模特。他选择了一个年轻的寡妇，并最终与她坠入爱河（根据多数版本的故事）。女人激发了画家的灵感，并且出于感激和爱戴，她希望给他一份礼物。但是女人并没有高超的技艺，所以她简单的将杏仁浸泡在[[白兰地|白兰地]]中，并将得到的酒液献给卢伊尼。<ref name="FoodNetwork">foodnetwork.ca. [http://www.foodnetwork.ca/guides/a-brief-history-of-amaretto/8076/story.html "A Brief History of Amaretto"]. Shaw Media Inc. Retrieved December 30, 2011.</ref><ref name="Disaronno Legend">[http://www.disaronno.com/home.aspx Disaronno] {{webarchive|url=https://web.archive.org/web/20091216105044/http://www.disaronno.com/home.aspx |date=2009-12-16 }}. Retrieved January 1, 2007. Home → Heritage → Page 2: ''The Legend''. (A direct link is not available due to the [[Adobe_Flash|Adobe Flash]]-based interface.)</ref>

最初这款力娇酒被称作“Amaretto di Saronno”，意思是来自Saronno的杏仁酒，由于市场营销的原因其名字演变成了“Disaronno Originale”。

== 饮用 ==
帝萨诺可以就冰块饮用，或混合可乐、果汁饮用，当然也可以用来调制鸡尾酒。
