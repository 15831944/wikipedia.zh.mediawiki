'''人格主義勤勞革命黨'''（{{vie|v=Cần lao Nhân vị Cách Mạng Ðảng|hn=勤勞人為革命黨}} 或 {{lang|vi|Đảng Cần lao Nhân vị / 黨勤勞人為}}），簡稱'''勤勞黨'''，是[[越南共和國|越南共和國]]的一個政黨，於1950年代初期由總統[[吳廷琰|吳廷琰]]和其胞弟政治顧問[[吳廷瑈|吳廷瑈]]建立。<ref>{{Cite book|title = 南越第一共和國興亡史：越南戰爭序曲|last = 唐|first = 向宇|publisher = 獨立作家|year = 2014|isbn = 9865729393|location = |pages = 176}}</ref>

勤勞黨基於社團組織和秘密網路作為有效工具，在給吳廷琰的權力創建政治基礎中扮演了相當重要的角色，並幫助吳廷琰控制南越的所有政治活動。<ref name=":0">{{Cite book|title = Nationalist in the Viet Nam Wars: Memoirs of a Victim Turned Soldier|last = Cong Luan|first = Nguyen|publisher = Indiana University Press|year = 2012|isbn = 9780253005489|location = |pages = 156}}</ref>該黨的表面上教條是基於吳廷瑈的{{link-en|人性尊嚴論|Person Dignity Theory}}以及{{Link-en|伊曼紐爾·穆尼耶|Emmanuel_Mounier}}的{{link-en|人格主義|Personalism}}思想。

吳廷琰在[[1963年南越政變|1963年南越政變]]中被推翻，隨後該黨被解散。1965年，其後繼政黨{{link-vi|越南人社革命黨|Việt Nam Nhân xã Cách mạng Đảng}}成立。
