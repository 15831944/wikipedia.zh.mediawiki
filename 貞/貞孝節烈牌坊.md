'''貞孝節烈牌坊'''是[[中國|中國]]傳統[[貞節旌表|貞節旌表]]制度下所建立的一種[[建築物|建築物]]，為[[牌坊|牌坊]]的一種。

== 建筑 ==
節孝牌坊的建築，必須經由固定的貞節孝旌表制度，由官方許可才能建築，建築形式各地區略有不同，但一般為[[牌樓|牌樓]]形式，少數則為[[墓碑|墓碑]]形式。上面常用的四大字為「欽旌節孝」等，其下可能記載節婦相資料事蹟，或兩側附相關[[對聯|對聯]]詩句。

== 历史 ==
现今发现中国历史上最后一批贞节牌坊包括衡阳县柳树村祖堂组的廖黄氏贞节牌坊，建于1921年，牌坊“节励松筠”石匾为时任大总统[[徐世昌|徐世昌]]所题<ref>{{Cite news|url=http://news.163.com/15/1019/05/B693M0HT00014Q4P.html|title=衡阳发现 最后一批贞节牌坊|last=陈|first=鸿飞|date=2015年10月19日|work=湖南日报|accessdate=}}</ref>。

== 分类 ==
贞节牌坊分为贞节坊、贞烈坊、节孝坊、贞寿坊等<ref name=":0">{{Cite news|url=http://mn.sina.com.cn/news/m/2013-11-03/171462875.html|title=厦门老牌坊:被遗忘的贞节牌坊|last=黄|first=绍坚|date=2013年11月4日|work=厦门晚报|accessdate=}}</ref>。

如厦门市境内现存的11座贞节牌坊，有贞节坊1座、节孝坊7座、贞寿坊3座<ref name=":0" />。[[File:江门市白沙祠内的贞节牌坊.JPG|thumb]]

== 意义 ==
節孝牌坊相較於其他的貞節旌表形式，將旌表榮譽在視覺空間上具體化，對於被旌表婦女及其家族而言具有更大的意義，也因此在中國近代反傳統禮教的思想下，節孝牌坊四字成為傳統禮教的重要代稱和象徵之一。

* 教授郑岩、汪悦进认为，贞节牌坊多为高门大户炫耀显赫地位的工具，并未对守节妇女真正进行关怀<ref>{{Cite news|url=http://www.jiemian.com/article/1166671.html|title=牌坊的修建和那些终究缺席的无名贞妇|last=汪|first=悦进|date=2017年3月13日|work=界面新闻|accessdate=}}</ref>。
