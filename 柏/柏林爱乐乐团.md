'''柏林爱乐乐团'''（{{lang-de|'''Berliner Philharmoniker'''}}，簡稱'''柏林愛樂'''，舊名{{lang|de|'''Berliner Philharmonisches Orchester'''}}），[[德国|德国]][[管弦樂團|管弦樂團]]，成立于1882年。普遍认为柏林爱乐是世界级的管弦乐团。

== 乐团管理 ==
柏林爱乐实行[[民主|民主]]管理。乐团的首席[[指挥|指挥]]需要通过乐团成员秘密投票，并且获得多数赞成的情况下才被认可成為首席[[指挥|指挥]]。

柏林爱乐原来隶属于[[柏林市政府|柏林市政府]]的科学、研究与文化部，舊名為柏林爱乐管弦乐团（或譯作柏林爱乐交響乐团，德文: Berliner Philharmonische Orchester，簡稱：BPO）。但在2002年1月1日，[[柏林市议会|柏林市议会]]通过并执行了关于柏林爱乐地位转变的新[[法律|法律]]，並改用現今的名稱。在新[[法律|法律]]之下，「爱乐基金会」成立，这标志着乐团有着更大的自主性。[[基金|基金]]会的理事会由柏林爱乐的[[团长|团长]]、艺术总监（即首席指挥）以及两名[[乐师|乐师]]组成，负责制定音乐会节目计划、音乐厅的经营、音乐普及[[教育|教育]]、乐团成员的利益保护等等。一切决定都必须取得这4名理事会成员的一致同意。在理事会之上设有一个以监督理事会活动的监事会，其成员由[[柏林市政府|柏林市政府]]文化部的负责人、市议会议员、乐师代表、乐团学院代表、爱乐管弦乐团之友协会代表组成。按基金会的[[章程|章程]]规定：「基金会不是一个以赚钱为首要目标的机构，而要以公众利益为目标」。

== 财政 ==
柏林爱乐的老家—[[柏林爱乐大厅|柏林爱乐大厅]]有着大小两个[[音乐厅|音乐厅]]，其房屋、地皮产权归[[柏林市政府|柏林市政府]]，而使用权则归乐团。乐团自己的音乐会占用率约为1/3，其他时间会出租给另外的乐队使用，以收取租金。柏林市政府给予爱乐基金会的补贴目前每年为1400万[[欧元|欧元]]，是乐团支出预算的一半，也正好相当于乐团的全部员工的工资。另一半需要基金会通过[[音乐会|音乐会]]收入、录制[[唱片|唱片]]、接受捐赠等方式获得,其中,国外巡回演出是乐团最重要的收入来源。

== 义务 ==
[[富特文格勒|富特文格勒]]为柏林爱乐确立了一项传统，每年都要举行大约25场大众音乐会、6场民族音乐会和其他室内乐演奏会等等义务演出，作为回报国家和柏林市政当局向乐团提供的财政资助。后来又确定，柏林爱乐乐团每年举办的交响音乐会不得少于120场。

指挥[[阿巴多|阿巴多]]会邀请那些残疾，智力障碍的儿童去参观乐团的排练，试图通过音乐改善他们的心智。

自[[西蒙·拉特尔|西蒙·拉特尔]]爵士上任以来，为了唤起青少年对[[古典音乐|古典音乐]]的兴趣，普及音乐知识，柏林爱乐启动了一个名为“未来”的校园行动。具体是乐团的乐手和拉特会到[[柏林|柏林]]的各个学校为学生解释、示范、演奏作品。此外，柏林某些学校乐队会定期在拉特指导下排练。

== 大事紀 ==
1867年，音乐家[[本雅明·比尔泽|本雅明·比尔泽]]（Benjamin Bilse）在柏林成立了一支冠以他自己名字的乐队，很快乐团就拥有乐手70人。这支乐队是柏林爱乐乐团的前身。

1887年，54名乐手因为对工资合同不满，离开了乐团。他们在演出经纪人[[赫尔曼·沃尔夫|赫尔曼·沃尔夫]]的倡议下，与其他一些业余音乐家组建了另一支乐团，名为爱乐乐团。这一事件标志着柏林爱乐乐团正式成立。乐团最初的指挥是[[弗朗茨·维尔纳|弗朗茨·维尔纳]]。但他很快就离开了乐团。弗朗茨·维尔纳并没有给初生的柏林爱乐带来很大的影响。

后来的指挥是[[布拉姆斯|布拉姆斯]]的好友[[约瑟夫·约阿希姆|约瑟夫·约阿希姆]]。但他的演奏曲目仅限于[[布拉姆斯|布拉姆斯]]等少数的几位德奥作曲家的范围内。乐团未见得有很大的进步。

再后来接掌柏林爱乐的人乃是[[漢斯·馮·彪羅|漢斯·馮·彪羅]]。他是赫赫有名的[[李斯特|李斯特]]之女[[柯西瑪|柯西瑪]]的前夫，以鋼琴家出身，由於[[華格納|華格納]]與[[柯西瑪|柯西瑪]]外遇，離婚後失意的他開始轉型，朝[[指揮|指揮]]之路專心發展。他的音樂造詣深厚，在5年时间內就把柏林愛樂提升至一流樂團的新境界，並獲得團員們的衷心敬愛。他後來被視為柏林愛樂的奠基者。与此同时，著名的作曲家如[[布拉姆斯|布拉姆斯]]、[[古斯塔夫·馬勒|马勒]]、[[葛利格|葛利格]]、[[柴可夫斯基|柴可夫斯基]]、[[理察·史特勞斯|理察·史特勞斯]]以及[[普菲茨纳|普菲茨纳]]也是乐团的客席指挥。

1895年，[[尼基什|尼基什]]担任乐团的首席指挥。在他的手中，柏林爱乐成为世界顶级乐团，演出曲目大大拓宽。

1923年，[[尼基什|尼基什]]逝世，乐团的首席指挥由36岁的[[福特萬格勒|福特萬格勒]]担任。[[福特萬格勒|福特萬格勒]]带领乐团做了不少巡回演出，使得乐团的霸主地位进一步稳固。

1944年，在[[二战|二战]]的硝烟中[[柏林爱乐大厅|柏林爱乐大厅]]被炮火摧毁，乐团在教堂的场合继续演出。

1945年4月15日，柏林被攻陷前的最后一场音乐会。

1945年5月26日，指挥[[里奧·波查德|里奧·波查德]]就在废墟中指挥乐队举行了战争结束后的首场音乐会。

1945年，[[罗马尼亚|罗马尼亚]]指挥[[谢尔盖·切利比达奇|切利比达奇]]成为乐团首席指挥。

1947年，[[福特萬格勒|福特萬格勒]]重返柏林爱乐。

1952年，[[福特萬格勒|福特萬格勒]]再次就任音乐总监。

1954年，[[福特萬格勒|福特萬格勒]]逝世。但临终时的[[福特萬格勒|福特萬格勒]]并不愿意让[[卡拉扬|卡拉扬]]接任。

1955年，[[卡拉扬|卡拉扬]]成为柏林爱乐首席指挥。乐团人数达120人，成为一个可演奏所有规模作品的演出团体。

1956年，[[卡拉扬|卡拉扬]]成为乐团终身首席指挥。这是音乐史上最成功的乐队指挥组合。通过巡回演出，音乐节演出和灌录唱片，乐团的国际声誉达到最高峰。两者合作的34年间，录制唱片无数。

1959年，在[[卡拉扬|卡拉扬]]带领下，乐团为[[德国唱片公司|德国唱片公司]]录音（[[理查·斯特劳斯|理查·斯特劳斯]]的《英雄生涯》）。搭成了铁三角关系。

1963年10月15日，柏林爱乐大厅重新开幕。同年，[[卡拉扬|卡拉扬]]指挥乐团灌录[[贝多芬|贝多芬]]交响曲全集。

1967年－1970年，乐团在[[卡拉扬|卡拉扬]]的指挥下录制[[瓦格纳|瓦格纳]][[尼伯龙根的指环|尼伯龙根的指环]]全剧。

80年代，[[卡拉扬|卡拉扬]]和乐团矛盾爆发，两者合作减少。

1989年，[[卡拉扬|卡拉扬]]去世，由意大利指挥家[[阿巴多|阿巴多]]出任乐团首席指挥。

从2002年至今，经由乐团成员选举，决定由[[英国|英国]]指挥家[[西蒙·拉特|西蒙·拉特]][[爵士|爵士]]接掌乐团。拉特爵士致力于发展年轻的乐手。在他的手下，柏林爱乐的平均年龄最小。他还尝试在柏林爱乐和[[维也纳爱乐管弦乐团|维也纳爱乐管弦乐团]]（维也纳爱乐）之间建立一种合作关系。为此他还举行过一场特别的音乐会，音乐会上他指挥柏林爱乐和维也纳爱乐合奏。

2005年11月，西蒙·拉特率领乐团访問中國。在[[北京|北京]]、[[上海|上海]]、[[台北|台北]]和[[香港|香港]]四地演出。

2011年11月，西蒙·拉特再次率領樂團來到中国進行2場演出。此演出在中国及日本等國家實況轉播，並在數位音樂廳對全世界訂戶同步播映，規模極其盛大。

2015年6月22日，[[基里爾·佩特連科|基里爾·佩特連科]]經選舉獲得大多數團員支持，將繼任成為樂團首席指揮與柏林愛樂基金會藝術總監，西蒙·拉特則會於2018年8月卸任離開樂團。
