'''柏寒'''（{{bd|1955年||2012年|2月19日|catIdx=B柏}}）[[江苏|江苏]][[扬州|扬州]]人，中国知名演员，曾改名为“柏含”<ref>{{cite web|url=http://www.hunantv.com/c/20120221/0915515194.html|title=柏寒：生活中并非麻辣婆婆 40岁才经历初恋|publisher=金鹰网 来源：新民晚报|author=编辑：罗伊久|date=2012-02-21 |accessdate=2012-02-29 |language=Zh-hans}}</ref>。2012年因为[[神经性内分泌肿瘤|神经性内分泌肿瘤]]治疗无效而逝世。<ref name="xinws " >{{cite web |url=http://www.chinanews.com/yl/2012/02-22/3689947.shtml |title=柏寒生前最爱阅读佛书 谈婆媳之道：自己要管住嘴 |accessdate= 2012年2月23日|work=|publisher=中国新闻网 |language=zh-hans}}</ref>

== 生平 ==
柏寒的父亲是[[公安大学|公安大学]]教员，母亲是[[法院|法院]]记录员。<ref name="shs " >{{cite web |url=http://yule.sohu.com/20120222/n335487839.shtml |title=柏寒与乔布斯罹患同样病症 遗体捐献给医疗事业 |accessdate= 2012年2月22日|work=|publisher=搜狐网 |language=zh-hans}}</ref>[[反右|反右]]和[[文革|文革]]彻底毁了这个家庭，柏寒两岁时，父亲被打成右派，下放到农场参加劳动，母亲下放到工厂喷油漆而患上精神病，当她15时，母亲自杀。<ref name="shs " /><ref name="wyis " >{{cite web |url=http://img1.cache.netease.com/ent/2012/2/21/201202211547051109f.jpg |title= 海清曝柏寒老师遗言：至少让我有尊严的死|accessdate= 2012-02-21|work=|publisher=网易 |language=zh-hans}}</ref> 1978年，文革停止，父亲回到了自己的身边，却因为积劳成疾患上食道癌而死。<ref name="shs " /> 早年的柏寒因为出身不好而二十几次考取[[文工团|文工团]]都没有被录取。<ref name="xinws " /> 直到1978年，22岁的她才得以被[[中央实验话剧院|中央实验话剧院]]录取。<ref name="xinws " /> 

2012年2月19日，柏寒因为神经性内分泌肿瘤而逝世。死前在微博发言：“爱她的人们不悲伤，她走得很安详；爱她的人们不难过，生死本就无常。”<ref name="xinws " /><ref name="shs " /><ref name="wyis " /> 对家人的遗言说：“让我有尊严的死”。<ref name="xinws " /><ref name="shs " /><ref name="wyis " />

== 感情生活 ==
25岁的柏寒组建了第一个家庭，生了一个孩子，最后离了婚。<ref name="xinws " />

1996年5月，柏寒和[[北京电影学院|北京电影学院]][[导演|导演]][[韩小磊|韩小磊]]结婚，2003年3月19日，韩小磊逝世两人的婚姻又告一段落。<ref name="xinws " /> 柏寒一直对外宣布这才是她的[[初恋|初恋]]。<ref name="xinws " /> 儿子[[韩青|韩青]]亦毕业于[[中央戏剧学院|中央戏剧学院]]表演系，就职于[[中国铁路文工团|中国铁路文工团]]）

== 生活爱好 ==
柏寒的思想很[[现代化|现代化]]，她爱开[[吉普车|吉普车]]，喜欢[[上网|上网]]，嗜好[[游泳|游泳]]，偶尔练练[[瑜伽|瑜伽]]。<ref name="xinws " /> 她对媒体称自己从来不看娱乐类书籍，一心只读[[佛经|佛经]]。<ref name="xinws " />
