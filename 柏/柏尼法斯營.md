'''柏尼法斯營'''（{{lang-en|Camp Bonifas}}；{{lang-ko|캠프 보니파스}}）是[[聯合國軍|聯合國軍]]設於[[朝鮮半島|朝鮮半島]]上的[[軍事基地|軍事基地]]，位在[[南北韓非軍事區|南北韓非軍事區]]南部邊界以南後方400米處<ref>{{Cite web |url= http://www.globalsecurity.org/military/facility/camp-bonifas.htm |title= Camp Bonifas Camp Liberty Bell / Camp Bonifas East |author=  |date=  |publisher= [[:en:GlobalSecurity.org|GlobalSecurity.org]] |language= en |accessdate= 2011年2月11日}}</ref>。而只要再往北2,400米持續前進，便可抵達位在[[板門店|板門店]]的[[共同警備區|共同警備區]]，橫越此處的[[三八線|三八線]]是[[大韓民國|大韓民國]]（南韓）及[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]（北韓）的[[邊界|邊界]]。

柏尼法斯營駐有聯合國軍警備營（{{lang|en|United Nations Command Security Battalion}}），該單位的主要任務是監控、並約束南北韓雙方遵守[[朝鮮停戰協定|朝鮮停戰協定]]的規定。[[大韓民國國軍|大韓民國國軍]]及[[駐韓美軍|駐韓美軍]]士兵負責非軍事區和週邊地區的導覽。營地內也有一家禮品店，主要販賣與非軍事區和共同警備區相關的[[伴手禮|伴手禮]]。

該營地的前身最早命名為小鷹營（{{lang|en|Camp Kitty Hawk}}）。到了1986年8月18日，為了紀念1976年[[板門店事件|板門店事件]]中遭[[朝鮮人民軍|朝鮮人民軍]]士兵殺害的[[美國陸軍|美國陸軍]][[:en:Captain_(United_States)|上尉]]亞瑟·柏尼法斯（{{lang|en|Arthur Bonifas}}），而改軍事基地名稱為柏尼法斯營。在營地那裡，還有一個[[標準桿|標準桿]]的「[[高爾夫球場|高爾夫球場]]」，其中有一部份也已使用了[[:en:AstroTurf|AstroTurf]]草地綠化。但值得注意的是，這高爾夫球場的三面由[[地雷|地雷]]區包圍著<ref name = "stat"/>。《[[運動畫刊|運動畫刊]]》把這座高爾夫球場稱呼為「最危險的高爾夫球場球洞」，並有報告說至少已經有一次因誤擊而導致地雷引爆<ref name = "stat"/>。1998年，凱文·沙利文於《[[華盛頓郵報|華盛頓郵報]]》中稱柏尼法斯營為「僅距非軍事區南部440碼遠，由三道[[:en:Barbed_tape|帶刺鐵絲網]]包圍著建築小聚落」，並說如果不是周遭的雷區和士兵，這裡「看起來像是一座大型的[[美國童軍|童子軍]]營地」<ref name = "stat">{{Cite web |url= http://www.stat.ualberta.ca/people/schmu/panmunjom.htm |title= CBorderline Absurdity A Fun-Filled Tour of the Korean DMZ |author= Kevin Sullivan |date= 1998-01-11 |publisher= 《[[華盛頓郵報|華盛頓郵報]]》 |language= en |accessdate= 2011-02-11}}</ref>。
