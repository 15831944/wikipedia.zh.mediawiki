'''交通和地圖符號'''是一個[[Unicode區段|Unicode區段]]，其中包含了與交通運輸和地圖有關的圖標。這個區段的創立主要是為了與日本電話公司的[[Shift_JIS|Shift JIS]]表情符號實現兼容，並將[[Wingdings|Wingdings]]和[[Wingdings_2|Wingdings 2]]的字符集編入正式編碼。
== 區段 ==
{{Unicode chart Transport and Map Symbols}}

== 繪文字 ==
此區段包含了94個[[繪文字|繪文字]]，他們分別位於:
U+1F680–U+1F6C5、U+1F6CB–U+1F6D2、U+1F6E0–U+1F6E5、U+1F6E9、U+1F6EB–U+1F6EC、U+1F6F0 與 U+1F6F3–U+1F6F8。<ref name="UTR51">{{Cite web|url=http://unicode.org/reports/tr51/| date=2017-05-18 |title=UTR #51: Unicode Emoji|publisher=Unicode Consortium}}</ref><ref name="EmojiData">{{Cite web|url=http://unicode.org/Public/emoji/latest/emoji-data.txt| date=2017-03-27 |title=UCD: Emoji Data for UTR #51|publisher=Unicode Consortium}}</ref>

此外，此區段還包含了46個{{tsl|en|Variant form (Unicode)\變體型式 (Unicode)|標準變體}}以決定為以下23個文字顯示為繪文字式樣（U+FE0F VS16）或文本式樣（U+FE0E VS15）：
U+1F687、U+1F68D、U+1F691、U+1F694、U+1F698、U+1F6AD、U+1F6B2、U+1F6B9–U+1F6BA、U+1F6BC、U+1F6CB、U+1F6CD–U+1F6CF、U+1F6E0–U+1F6E5, U+1F6E9、U+1F6F0 與 U+1F6F3。<ref>{{Cite web|url=http://unicode.org/Public/emoji/latest/emoji-variation-sequences.txt|title=UTS #51 Emoji Variation Sequences | publisher=The Unicode Consortium}}</ref>
所有這些基本字符的默認是設定都是顯示為文本。

== 差異 ==
此區塊中有六個符號用來表示人，或是人的身體部位。
