'''交通部公路總局高雄市區監理所'''為[[中華民國|中華民國]][[交通部公路總局|交通部公路總局]]所屬機關，掌理[[高雄市|高雄市]][[楠梓區|楠梓區]]、[[左營區|左營區]]、[[鼓山區|鼓山區]]、[[三民區|三民區]]、[[苓雅區|苓雅區]]、[[前金區|前金區]]、[[新興區|新興區]]、[[鹽埕區|鹽埕區]]、[[前鎮區|前鎮區]]、[[旗津區|旗津區]]、[[小港區|小港區]]等11個行政區之公路監理業務，包括車輛檢驗、駕駛人考驗、牌照管理、駕照管理、汽車運輸業督導、交通安全稽查、代徵汽車燃料使用費及辦理道路交通安全講習等事項。

== 沿革 ==
1979年7月1日，[[高雄市|高雄市]]改制為[[直轄市|直轄市]]，[[高雄市政府|高雄市政府]]成立[[高雄市政府建設局|建設局]]接管[[台灣省公路局|台灣省公路局]]有關高雄市之公路工程、[[公路運輸|公路運輸]]及公路監理業務；1980年6月20日，[[行政院|行政院]]「台交字第7015號函」核定，本年7月1日增設[[高雄市監理處|高雄市監理處]]，借用[[高雄市公車處|高雄市公車處]]場地辦公；1985年9月27日，高雄市監理處搬遷至高雄市楠梓區[[德民路|德民路]]71號；2003年1月1日，高雄市政府成立[[高雄市政府交通局|交通局]]，高雄市監理處改隸交通局。

2010年12月25日，[[高雄縣|高雄縣]]、高雄市奉行政院核定合併改制為直轄市「高雄市」，改制後公路監理業務交由中央政府統籌管理。2012年1月1日，高雄市監理處改隸交通部公路總局，更名為「交通部公路總局高雄市區監理所」。

== 業務範圍 ==
高雄市區監理所服務轄區為高雄市楠梓區、左營區、鼓山區、三民區、新興區、苓雅區、前金區、鹽埕區、前鎮區、小港區及旗津區，主要業務職掌為辦理車輛檢驗、駕駛人考驗、牌照管理、駕駛執照管理、交通稽查、交通安全教育、汽車運輸業督導及汽車燃料使用費等有關事項。2015年6月止，轄管汽/機車158萬輛、駕駛人189萬人、汽車代檢廠28家、駕訓班10家、運輸業960家，徵收汽車燃料使用費新臺幣30億1仟萬元。

== 使命與願景 ==
* 以「提供便捷優質的公路監理服務」為使命。
* 以「親切、便民、行動、效力」為理念。
* 以「成為最受信賴的政府機關之一」為願景。
