'''交通部中央氣象局地震震度分級'''是[[臺灣|臺灣]]使用的[[地震震度|地震震度]]分級，由[[交通部中央氣象局|交通部中央氣象局]]制定，以地動[[加速度|加速度]]（[[伽|gal]]，<math>\mathrm{1gal=1cm/s^2}</math>）來區分，分為0-{}-級至7-{}-級，共8個等級。<ref name="eq_35">{{Cite web |url = https://scweb.cwb.gov.tw/zh-TW/Guidance/FAQdetail/36 |title = 35. 何謂震度? |work = 地震百問 |publisher = 中央氣象局地震測報中心 |accessdate = 2019-04-09}}</ref>

== 歷史 ==
臺灣的地震觀測始於[[台灣日治時期|日治時期]]，因此2000年8月之前的震度分級是沿用日本1936至1948年所使用的[[日本氣象廳震度等級|震度分級]]，將地震震度分成0-{}-級至6-{}-級，共7個等級，震度6-{}-級為地動加速度在250gal以上。<ref>{{Cite web |url = https://twgeoref.moeacgs.gov.tw/GipOpenWeb/wSite/ct?mp=105&ctNode=1233&xItem=140853 |author = 吳逸民 |title = 震度Intensity |work = 地質知識服務網地質百科 |publisher = 經濟部中央地質調查所 |accessdate = 2019-04-09}}</ref>1999年[[921大地震|921大地震]]時交通部中央氣象局曾記錄到地動加速度超過980gal的資料，由於測得震度6-{}-級的地區廣闊，不利於受災情況之研判，交通部中央氣象局在2000年8月公告增加一個震度分級，以地動加速度400gal作為震度6-{}-級的上限，400gal以上定為震度7-{}-級。<ref name="cwb_news">{{cite press release |url = http://www.rel.org.tw/921/rec_doc/doc/g018/g018-2/09-890808%E6%96%B0%E8%81%9E%E7%A8%BF-%E5%A2%9E%E4%BF%AE%E4%B8%AD%E5%A4%AE%E6%B0%A3%E8%B1%A1%E5%B1%80%E9%9C%87%E5%BA%A6%E5%88%86%E7%B4%9A.doc |title = 增修中央氣象局震度分級－新增震度七級劇震 |date = 2000-08-08 |publisher = 交通部中央氣象局 |format= DOC |accessdate = 2019-04-09}}</ref>

==差異==
交通部中央氣象局地震震度分級是由地動加速度換算而得，[[日本氣象廳震度等級|日本氣象廳震度等級]]則是由加速度值與地震動持續時間計算而得。<ref name="eq_35" />
