'''交趾陶藝術節'''，是[[臺灣|臺灣]][[嘉義市|嘉義市]]政府所舉辦的文化節慶活動，已從1997年起舉辦第一屆至今，約於每年的11－12月舉辦為期數週的一系列活動<ref>{{cite book |author1=王薏瑄 |editor=賴萬鎮 |title=文化的軌跡-嘉義市文化中心八週年成果專輯 |date=2000 |url=http://lib.fjsh.cy.edu.tw/detdata.php?pagerows=20&orderby=BRN&pageno=1&qrow=19&ti=%B9%C5%B8q%A5%AB |accessdate=2018-11-21|language =zh-hant}}</ref>。

== 活動源起 ==
1997年，當時的[[中華民國文化部|文建會]]舉辦的各縣市文化中心文藝季（全國文藝季）中<ref>{{cite web |author1=文化部 |title=文化部成立沿革 |url=https://www.moc.gov.tw/content_246.html |accessdate=2018-11-21|archive-url=https://web.archive.org/web/20181019124736/https://www.moc.gov.tw/content_246.html|archivedate=2018-10-19|language =zh-hant}}</ref>，嘉義市政府以「嘉義交趾節」為主題舉辦相關文藝活動，至此開始每年一度的嘉義交趾陶藝術節<ref>{{cite news |author1=洪美秀 |title=嘉義交趾陶藝術連接傳統與現代 |url=https://www.ct.org.tw/1150410 |accessdate=2018-11-21 |publisher=基督教論壇報- 全球華人新聞網 |date=1997-04-27 |archive-url=https://web.archive.org/web/20181121072530/https://www.ct.org.tw/1150410 |archivedate=2018-11-21 |language=zh-hant |dead-url=yes }}</ref><ref>{{cite web |title=嘉義市交趾陶館 |url=http://superspace.moc.gov.tw/hall/local_culture_page.aspx?oid=9da25b40-7641-4a2a-b669-f474fec6a90a |publisher=文化部 |accessdate=2018-11-21|archive-url=https://web.archive.org/web/20180223212223/http://superspace.moc.gov.tw/hall/local_culture_page.aspx?oid=9da25b40-7641-4a2a-b669-f474fec6a90a|archivedate=2018-02-23|language =zh-hant}}</ref>。
