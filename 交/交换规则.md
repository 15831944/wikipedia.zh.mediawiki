'''交换规则'''是一个[[连珠|连珠]]的开局规则。它是[[国际连珠联盟|国际连珠联盟]]（RIF）在80年代末确定的官方规则。从1989年到1995年，它在世界连珠锦标赛中被使用。<ref>{{Cite web|url=https://web.archive.org/web/20100826045920/http://www.renjusha.net/database/oldrule.htm|title=ルール（対局規定）の変遷史|accessdate=2018-12-11|author=|date=|publisher=}}</ref><ref name=":0">{{Cite web|url=http://www.renju.net/study/rifrules.php|title=RIF rules - RenjuNet|accessdate=2018-12-11|author=|date=|work=www.renju.net|publisher=}}</ref>

== 规则介绍 ==
交换规则的详细开局流程如下：

* 假黑方在棋盘中心落下第1手棋。
* 假白方在棋盘中心的3×3方格内落下第2手棋。
* 假黑方在棋盘中心的5×5方格内落下第3手棋。此时，26种[[连珠开局|连珠开局]]之一被确定。
* 假白方选择自己与对方的棋子颜色（即：白方有权交换）。
* 交换后的白方在棋盘的空点处落下第4手。
* 黑方为第5手棋选择两个打点。这两个打点不能对称。白方在两个打点中，选择一个点，作为最终的第5手。然后，白棋落下第6手棋。<br />
* 当第5手棋选择完成后，开局规则结束。

在开局阶段，不允许棋手使用pass权利。<ref name=":0" /><ref>{{Cite web|url=http://www.renju.net/media/showrule.php?rule=1|title=RIF rule|accessdate=2018-12-11|author=|date=|work=www.renju.net|publisher=}}</ref>

== 简要介绍 ==
这个规则使得大多数棋手使用斜止开局作出斜开口主导，因此在1996年，被[[RIF规则|RIF规则]]所取代。交换规则和[[RIF规则|RIF规则]]唯一的区别在于，第2手棋由哪名棋手所指定。交换规则对假白方较有利，而RIF规则对假黑方较有利。<ref>{{Cite web|url=http://www.renju.net/media/showrule.php?rule=5|title=Yamaguchi rule|accessdate=2018-12-11|author=|date=|work=www.renju.net|publisher=}}</ref>

== 相关比赛 ==
在1995年以前，交换规则是[[世界连珠锦标赛|世界连珠锦标赛]]的正式规则。<ref>{{Cite web|url=http://renju.net/media/wc.php|title=Renju World Championships - RenjuNet|accessdate=2018-12-11|author=|date=|work=renju.net|publisher=}}</ref>
