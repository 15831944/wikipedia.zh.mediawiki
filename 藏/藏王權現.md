'''蔵王権現'''（ざおうごんげん）是[[日本|日本]]獨特的[[修驗道|修驗道]]的本尊。正式名稱為[[金剛蔵王権現|金剛蔵王権現]]（こんごうざおうごんげん）或稱[[金剛蔵王菩薩|金剛蔵王菩薩]]（こんごうざおうぼさつ）。並非源起於印度，而是日本獨有的佛、位於[[奈良県|奈良県]][[吉野町|吉野町]]的[[金峯山寺|金峯山寺]]本堂（蔵王堂）的本尊較為人知。「金剛蔵王」是究極不滅真理的體現、亦有掌管所有一切之王的含意[2]。[[權現|權現]]是指「在日本神佛習合的演化，透過日本神明的名諱進而展現佛教神佛姿態」之意。蔵王権現亦包含了佛、菩薩、諸尊、諸天善神、天神地祇等所有的力量[3]。

== 緣起 ==
根據傳承，藏王權現是[[役小角|役小角]]在[[吉野山|吉野]]的[[金峰山|金峯山]]修行中感得。[[釋迦如來|釋迦如來]]、[[千手千眼觀世音菩薩|千手觀音]]、[[彌勒菩薩|彌勒菩薩]]三尊之合體，現在[[吉野山|吉野山]]的藏王堂將並立的3尊藏王權現作為本尊祭祀。
在神佛習合的教化下與 [[安閑天皇|安閑天皇]]（廣國押武金日）為同一神格。因此在明治時代的[[神佛分離|神佛判然令]]頒布時，除本山（金峯山寺）所供奉為藏王權現以外，其餘的神社所祭祀的大多為[[安閑天皇|安閑天皇]]。
此外在[[神道|神道]]中、藏王權現又與[[大己貴命|大己貴命]]、[[少彦名命|少彦名命]]、[[國常立尊|國常立尊]]、[[日本武尊|日本武尊]]、[[金山毘古命|金山毘古命]]等[[神佛習合|習合]]並視為一同。因此大多祭祀藏王權現的神社亦供奉著上述五尊的神明。

== 神像 ==
藏王權現的形象類似密教的[[明王|明王]]像，一般是激烈的忿怒相、怒髪衝天、右手和右脚抬高，右手持[[金剛杵|三鈷杵]]、左手在腰部結「劍印」。

== 歷史 ==
「金剛藏王」的名稱最早出現在[[平安時代|平安時代]]的記錄。

[[神佛習合|神佛習合]]的教說中，和[[安閑天皇|安閑天皇]]（廣國押建'''金'''日命）被視為同一神格，[[明治時代|明治時代]]的[[神佛分離|神佛分離]]之際，祭祀藏王權現的神社的祭神大多改成安閑天皇。

[[神道|神道]]之中，藏王權現和[[大己貴命|大己貴命]]、[[少彦名命|少彦名命]]、[[國常立尊|國常立尊]]、[[日本武尊|日本武尊]]、[[金山毘古神|金山毘古命]]等習合，被視為同一神。所以，祭祀藏王權現的神社，主要以上記的5組神作為[[祭神|祭神]]。

另外，[[宮城縣|宮城縣]]和[[山形縣|山形縣]]縣境的[[日本百名山|日本百名山]]之一的[[藏王連峰|藏王連峰]]（藏王山），是古稱刈田嶺、不忘山的-{[[山岳信仰|山岳信仰]]}-和[[歌枕|歌枕]]之[[山|山]]。[[白鳳時代|白鳳時代]]（7世紀後半），從吉野勸請藏王權現，平安時代，作為修驗道的修行場被稱作藏王山。
