'''藏南镇'''是[[中华人民共和国|中国]][[山东省|山东省]][[青岛市|青岛市]][[黄岛区|黄岛区]]下辖的一个镇。

== 行政区划 ==

藏南镇下辖横河川村、上丁家洼村、大马家疃村、小马家疃村、潘家庄村、刘家庄村、茉旺村、乜家庄村、瓦屋村、唐家庄村、小岭子村、孙家屯村、丁家皂户村、下丁家洼村、崖下村、崖上村、西南地村、河西屯村、高戈庄村、曾家官庄村、长阡沟村、胡家庄村、臧家村、丁家松园村、贺吉沟村、王家官庄村、丁家官庄村、于家官庄村、西陡崖村、东陡崖村、河崖村、范家洼村、韩家溜村、岭西头村、岭南头村、刘卜疃村、大柳家庄村、后沟村、赵家沟村、马栏村、七亩地村、黄家营村、六合桥村。<ref>{{Cite web|url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2015/37/02/11/370211109.html|title=统计用区划和城乡划分代码/山东省/青岛市/黄岛区/藏南镇|accessdate=2017-04-23|author=国家统计局|date=2015-09-30|publisher=}}</ref>
== 人口 ==

2010年[[中国第六次人口普查|中国第六次人口普查]]时，藏南镇共有人口26388人。共有家庭9529户，平均每户2.76人。14岁以下的少年儿童共3695人，占总人口14.00%；15-64岁人口共18467人，占总人口69.98%；65岁及以上的老年人共4226人，占总人口的16.01%。男性共计13296人，占总人口50.38%；女性共计13092，占总人口49.61%。本地居住的人口中，拥有本地户籍的人口为24958人，占94.58%。<ref name="人口普查2010">{{cite web| url =http://www.yearbookchina.com/navipage-n2013030152000017.html| title =《中国2010年人口普查分乡、镇、街道资料》| author =中国国务院人口普查办公室、中国国家统计局人口就业统计司等| year =2010| website =统计年鉴分享平台| page =表15 山东省乡、镇、街道人口| format =收费| access-date =2018-07-19| archive-url =https://web.archive.org/web/20180719084356/http://www.yearbookchina.com/navipage-n2013030152000017.html| archive-date =2018-07-19| dead-url =no}}</ref>
