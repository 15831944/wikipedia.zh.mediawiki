《'''疯狂的外星人'''》是一部由[[宁浩|宁浩]]导演、根据[[刘慈欣|刘慈欣]]《[[乡村教师_(小说)|乡村教师]]》改编的中國科幻喜劇片，[[黄渤|黄渤]]、[[沈腾|沈腾]]等演员出演<ref name="douban">{{cite web |title=疯狂的外星人 (豆瓣) |url=https://movie.douban.com/subject/25986662/ |website=豆瓣 |accessdate=2018-06-23}}</ref>。主要讲述了动物园饲养员耿浩阴差阳错地救起了一个外星生物、历尽艰辛送其回家的故事。除了宁浩电影一贯独有的[[黑色幽默|黑色幽默]]风格，这也是其第一次拍摄科幻题材<ref>{{cite web |author1=宁浩 |title=如何评价宁浩的电影《疯狂的外星人》？ |url=https://www.zhihu.com/question/305188717/answer/590465085 |website=知乎 |accessdate=2019-02-08}}</ref>。

这是宁浩继《[[疯狂的石头|疯狂的石头]]》（2006年）、《[[疯狂的赛车|疯狂的赛车]]》（2009年）之后“疯狂系列”的第三部作品，也是[[黄渤|黄渤]]在宁浩电影中第三次出演名叫“耿浩”的角色{{noteTag|此前[[黄渤|黄渤]]在《[[疯狂的赛车|疯狂的赛车]]》和《[[心花路放|心花路放]]》中的角色都叫“耿浩”，但角色本身并无关联。}}。电影于2017年7月开拍，同年12月杀青<ref>{{cite web |author1=Mtime时光网 |title=宁浩《疯狂的外星人》杀青首曝剧照 |url=http://news.mtime.com/2017/12/11/1576361.html |accessdate=2018-06-23}}</ref>，2019年2月5日（大年初一）在中国大陆上映。

==北美撤档==
原本电影将在美国上映，但却传出遭到撤档，原因推测可能由于片中讽刺美国的剧情没有通过审查，而此举也引起美国言论自由真伪的争论。
