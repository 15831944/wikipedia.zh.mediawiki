'''介休市'''是[[中华人民共和国|中华人民共和国]][[山西省|山西省]]的一个[[县级市|县级市]]，由[[晋中市|晋中市]]代管。介休是為紀念[[晋文公|晋文公]]的臣子[[介之推|介之推]]亡於境内的绵山（介山）上而得名。置縣已有二千五百年以上的歷史。

== 历史 ==
[[春秋时期|春秋]][[晋顷公|晋顷公]]十二年（前514年）置邬县。[[秦朝|秦]]置界休县。新莽改为界美。[[东汉|东汉]]复名界休。[[西晋|西晋]]改介休县，以[[春秋时期|春秋]]时[[晋文公|晋文公]]的臣子[[介之推|介之推]]死亡在其境内的绵山（介山）上而得名。[[十六国|十六国]]时期邬县、介休县废。[[北魏|北魏]][[太和_(北魏孝文帝)|太和]]八年（484年）置介休，十九年复置邬县。[[北齐|北齐]]天保年间并入永安县。[[北周|北周]][[宣政|宣政]]元年（578年）恢复。[[大成_(宇文赟)|大成]]元年（579年）改名平昌县。隋[[开皇|开皇]]十年（590年）析置[[灵石县|灵石县]]，十八年复名介休县。其后先后隶属[[太原郡|太原郡]]、[[西河郡|西河郡]]、[[汾州|汾州]]、[[汾州府|汾州府]]等。1992年撤县设市。

== 地理 ==
=== 区位 ===
介休地处[[太原盆地|太原盆地]]和[[太岳山|太岳山]]的结合部，太原盆地西南端，太岳山北侧，[[汾河|汾河]]南畔。

=== 气候 ===
介休属半干旱大陆性气候，四季分明，季风气候显著。冬寒冷，夏季闷热，春秋较短。最冷月（1月）平均气温−4.4℃，最热月（7月）平均气温23.9℃，年平均气温10.6℃，降水量455毫米。

== 经济 ==
农业主产[[小麦|小麦]]、[[玉米|玉米]]、[[高粱|高粱]]等。境内[[煤炭|煤炭]]储量丰富，是中国重要的焦煤产地，有汾西矿业公司等重要企业。依托于煤炭，介休近年来[[炼焦|炼焦]]产业也发展迅速，但亦带来不少环境污染。

== 物产 ==
*矿产有煤炭、铝钒土、石膏、粘土、石英矿、硫铁、石灰岩、紫砂、蛭石、紫木节、水晶、硝石、磷矿、石棉等。
*野生动物有土豹、林麝、青羊、穿山甲、马雕、狼、狐、獾、山猪等。
*中草药材有猪苓、地骨皮、香加皮、荆芩、山桃仁、酸枣仁、麻黄、黄芩、板蓝根、紫苏、椿根皮、绵贝、元胡、大叶三七、荔枝草、野山参等。
*土特产主要有亨乐牌黄酒、鸑鷟香、贯馅糖、绵山牌陈醋。
