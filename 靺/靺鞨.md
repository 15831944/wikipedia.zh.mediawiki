'''靺鞨'''{{NoteTag|{{标音 |字 = 靺鞨 |拼音 = Mòhé（音同“莫合”）或 Mòjié（音同“勿吉”）}}}}，是活跃于[[中国古代|中国古代]][[中国东北地区|东北地区]]一个[[民族|民族]]，[[南北朝|南北朝]]之前稱'''[[勿吉|勿吉]]。'''靺鞨的一个主体部落[[粟末靺鞨|粟末靺鞨]]于[[唐朝|唐代]]建立了[[渤海国|渤海国]]。

《[[旧唐书|旧唐书]]》卷一九九下《靺鞨传》记载：“靺鞨，盖肃愼之地，后魏谓之勿吉，在京师东北六千余里。东至于海，西接[[突厥汗国|突厥]]，南界[[高句丽|高丽]]，北邻[[室韦|室韦]]。其国凡为数十部，各有酋帅，或附于高丽，或臣于突厥。”《[[新唐书|新唐书]]》的记载略同。

隋代靺鞨大体以吉林松花江流域为中心，分布在东至今俄罗斯东部[[濱海邊疆州|滨海边疆州]]，北至[[黑龙江|黑龙江]]、[[乌苏里江|乌苏里江]]的广大地区。

历史上的靺鞨有数十个部落，主要的部落有7个——[[伯咄部|伯咄部]]、[[安车骨部|安车骨部]]、拂涅部、号室部、[[白山靺鞨|白山部]]、[[黑水靺鞨|黑水部]]、[[粟末靺鞨|粟末部]]，各部相距二三百里，史称“[[靺鞨七部|靺鞨七部]]”。<ref name=":0">{{Cite book |title = 《渤海简史》|author = 王承礼 |publisher = 黑龙江人民出版社 |year = |ISBN =}}</ref>

== 歷史 ==
靺鞨在[[周朝|周代]]、[[秦朝|秦朝]]到[[西汉|西汉]]时称为[[肃慎|肃慎]]，[[东汉|东汉]]至[[曹魏|魏]][[晋朝|晋]]时称为[[挹娄|挹娄]]，[[南北朝|南北朝]]时称[[勿吉|勿吉]]，各部分布在今[[长白山|长白山]]以北，[[松花江|松花江]]、[[黑龙江|黑龙江]]和[[乌苏里江|乌苏里江]]的广大地区，东临[[日本海|日本海]]。<ref name=":0" />

漢以來臣屬[[扶餘|扶餘國]]（挹婁是在[[黃初|黃初]]中脫離扶餘），自[[北魏|北魏]][[延兴|延兴]]五年（475年）勿吉遣使到[[北魏|北魏]]朝贡后，与中原关系日益紧密，并逐渐兴盛起来。包括[[東扶餘|扶餘]]、[[高句丽|高句丽]]、[[百济|百济]]在內五十余国遣使向北魏朝贡。478年，勿吉人曾朝贡北魏，要求准许其和百济配合，南北夹攻高句丽。北魏朝廷以三方都是自己的藩属，令彼等“宜共和顺，勿相侵扰”。勿吉听从北魏朝廷的谕令，停止对高句丽的进攻。

公元493年（北魏[[太和|太和]]十七年），勿吉灭亡邻近的[[東扶餘|扶餘]]，领土扩展到[[伊通河|伊通河]]流域[[松辽平原|松辽平原]]的中心，为东北一支强大势力。

高句丽灭亡后，唐统治了高句丽之前在东北控制的土地，部分粟末靺鞨人、白山靺鞨人和高句丽人被移居至营州（今辽宁省[[朝阳市|朝阳市]]）一带。后唐代设营州上都督府与饶乐都督府，掌管东北边疆民族事务。<ref name=":0" />

公元696年（[[武则天|武则天]]万岁通天元年），营州都督[[赵文翙|赵文翙]]侵侮契丹部落，激起契丹、靺鞨各族不满。契丹酋长[[大贺窟哥|大贺窟哥]]之孙[[松漠都督府|松漠都督府]]都督[[李尽忠|李尽忠]]和内兄[[孙万荣|孙万荣]]叛唐，杀营州都督赵翙，占据营州。徙居营州一带的靺鞨人、高句丽人也参加了李尽忠的反唐起义。武则天为了分化反唐队伍，封靺鞨酋长[[乞四比羽|乞四比羽]]为许国公，封粟末靺鞨部落首领[[乞乞仲象|乞乞仲象]]为震国公，赦免其反唐行为。乞四比羽拒绝接受。后乞乞仲象与乞四比羽率领靺鞨人与高句丽人，趁唐王朝营州被攻占、无暇顾及，且靺鞨故地统治空虚的时机，率军离开营州，东渡[[辽河|辽水]]（辽河），返回故地。武则天命令契丹降将[[李楷固|李楷固]]率兵追击，斩杀乞四比羽，途中乞乞仲象去世。<ref name=":0" />

乞乞仲象之子[[大祚荣|大祚荣]]，为人骁勇，善于用兵，他团结了[[契丹人|契丹人]]与高句丽人，将李楷固战胜于[[天门岭之战|天门岭]]，李楷固失败而归。<ref name=":0" />

公元698年（聖历元年）突厥入侵唐王朝，占据了[[妫州|妫州]]、[[檀州_(隋朝)|檀州]]、[[定州市|定州]]、[[赵州_(河北)|赵州]]，契丹叛唐，重新依附于突厥，由此唐帝国中原通往东北地区的道路被阻断。大祚荣趁机率领靺鞨人与高句丽人向东进军，回到了太白山（[[长白山|长白山]]）东北的坡奥娄河（[[牡丹江|牡丹江]]）上游一带，这一带原为挹娄故地。大祚荣在东牟山下筑城为居，建立靺鞨人最初的都城“旧国”（今[[吉林省|吉林省]][[敦化市|敦化市]][[敖东城|敖东城]]）。<ref name=":0" />

同年，大祚荣自立为“震国王”，仍称“靺鞨”。<ref name=":0" />

公元713年，[[唐玄宗|唐玄宗]]赐封大祚荣为“[[渤海郡王|渤海郡王]]”，大祚荣遂将国名“靺鞨”更改为“渤海”，以忽汗州（号“上京龙泉府”，今[[黑龙江省|黑龙江省]][[宁安市|宁安市]]西南[[东京城镇|东京城]]）为上京，史称“[[渤海国|渤海国]]”。<ref name=":0" />

== 族情 ==
靺鞨各部发展不平衡。南迁的[[白山靺鞨|白山]]、[[粟末靺鞨|粟末]]、[[伯咄靺鞨|伯咄]]、[[安车骨靺鞨|安车骨]]四部较留守肃慎故地的[[拂涅靺鞨|拂涅]]（來自[[沸流|沸流]]）、[[号室靺鞨|号室]]、[[黑水靺鞨|黑水]]三部先进。同时兩大分支在風俗文化等方面已存在明顯差異。

粟末部在最南，较先进，居[[松花江|粟末水]]（今第二松花江）流域，常附属[[高句丽|高句丽]]。[[隋炀帝|隋炀帝]]即位初，首领[[突地稽|突地稽]]率部千余户降，移居[[营州|营州]]（今[[辽宁|辽宁]][[朝阳|朝阳]]）。其后，由原高句丽贵族的[[粟末靺鞨|粟末靺鞨]]人为主体、联合高句丽族建立的[[渤海国|渤海]]政权强盛，除了[[黑水靺鞨|黑水部]]其余靺鞨皆附属于渤海。曾经有靺鞨到[[出羽国|出羽国]]与[[北海道|北海道]]。

[[黑水靺鞨|黑水部]]在最北，农业经济发展较慢，分十六部，以勇健著称，常与高句丽和渤海征战。唐[[开元|开元]]十年（722年），黑水部酋[[倪属利稽|倪属利稽]]入朝，[[唐玄宗|唐玄宗]]任为[[勃利州|勃利州]]（今[[俄国|俄国]][[伯力|伯力]]）刺史。后在其境置黑水军，又于其最大部落内置[[黑水都督府|黑水都督府]]，仍以首领为都督。其余各部隶都督府，设州，首领为州刺史，唐派长史监领之。十六年，唐赐其都督姓李，兼黑水经略使，隶幽州都督。後來黑水靺鞨改稱為「'''女真'''」。

靺鞨以农业经济为主，多粟、麦、穄，善养猪，富者多至数百口，亦从事狩猎。各部首领称“'''大莫弗瞒咄'''”（大酋長，莫弗即是滿州語的瑪法，長老的意義），不相统属。其俗多穴居，妇女服布裙，男子衣猪皮。

役屬高句麗的是白山部與一部分粟末，明永樂年間[[奴兒干都司|奴兒干都司]]頭人[[阿哈出|阿哈出]]是胡里改人，挹婁後人。

== 轶事 ==

[[法国|法国]]学者[[伯希和|伯希和]]一度認為此部落是[[东罗马帝国|东罗马帝国]]史料中的木乞里部，包庇了大量[[柔然|柔然]]人。俄羅斯科學家說他們的面部非常扁平，接近东亚人種的極限，他們對[[烏爾奇人|烏爾奇人]]與[[那乃人|那乃人]]形成影響最大。最近他們的是[[黄种人|黄种人]]西伯利亞型：如[[尼夫赫人|尼夫赫人]]，[[科里亞克族|科里亞克族]]（特別是古亞細亞人種[[尤卡吉爾人|尤卡吉爾人]]）。勿吉與肅慎、挹娄是同種異族，婚喪習俗不同，而不是同一民族在不同時代的不同稱呼{{cn}}。

== 沃沮与勿吉 ==

中国歷史學家、考古學家[[金毓黻|金毓黻]]先生主张“靺鞨，当亦为沃沮、勿吉之音转”，即勿吉与靺鞨音同，是不同时期的不同汉字标音。[[金毓黻|金毓黻]]先生认为朝鲜半岛北部的沃沮即[[勿吉|勿吉]]，又分北沃沮（黑水等部）、东沃沮（白山等部），[[东女真|东女真]]当即古代[[东沃沮|东沃沮]]的后裔。此指东女真即[[曷懒甸|曷懒甸]]女真，为唐以前沃沮的后裔。此观点认为朝鲜半岛北部的沃沮即[[勿吉|勿吉]]，為女真人的族源之一。

真正的通古斯{{what}}應是勿吉、靺鞨{{cn}}，該名稱被認為語今日之[[烏德蓋族|烏德蓋族]]（Udege）之族名同源，而該族又處於南北通古斯之交界，以"勿吉"一名的古音/mudged//vudged/及"靺鞨"/modged/（唐初本寫作靺羯/ㄐㄧㄝˊ/，後改作靺鞨。故本應讀作/ㄇㄛˋㄐㄧㄝˊ/，而非今日之官方讀音/ㄇㄛˋㄏㄜˊ/）來看，通古斯民族應於隋唐前遷徙至今日的黑龍江流域並且定居，並自此開始分化出南北通古斯（見胡增益著"鄂倫春語研究"）。

在有关女真族起源的说法中，韩国学者[[李丙焘|李丙焘]]和日本学者[[三上次男|三上次男]]主张渤海的强制移置说。三上次男以为黑水一带诸部族是[[渤海武王|渤海武王]]（[[大武艺|大武艺]]，719-737年在位）末期的被征服人民，以后黑水部被强制移民到渤海南京南海府（朝鲜民主主义人民共和国[[清津市|清津市]]，一说在[[咸兴|咸兴]]）附近，随着[[新罗|新罗]]后期国势的衰微，逐渐往南迁移。如此，则在黑水部真移置于渤海南京附近以前，后来[[曷懒甸|曷懒甸]]女真所据的原[[汉四郡|汉四郡]]地区，可说是一片无人地带。后来[[高丽太祖|高丽太祖]]的开拓地区也是女真尚未占据的原[[汉四郡|汉四郡]]地区。
