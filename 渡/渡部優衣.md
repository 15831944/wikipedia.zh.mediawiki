'''渡部優衣'''（{{bd|1988年|12月4日|catIdx=Watanabe Yui}}）是[[日本|日本]]的女性[[聲優|聲優]]、[[藝人|藝人]]、[[歌手|歌手]]，[[dandelion|dandelion]]所屬。[[大阪府|大阪府]][[箕面市|箕面市]]出身。身高167cm。血型A型。

== 經歷 ==
2008年進入[[JTB_Entertainment_Academy|JTB Entertainment Academy]]（東京校第2期）。2009年8月加入女性偶像團體「{{nihongo|星之乙女歌劇團|星のオトメ歌劇団}}」。

2010年4月自養成所畢業並加入[[JTB娛樂|JTB娛樂]]。同年10月擔任[[東京晴空塔|東京晴空塔]]吉祥物角色「{{nihongo|小索拉卡拉|ソラカラちゃん}}」的聲音演出。

2016年8月轉屬[[dandelion|dandelion]]。

== 人物 ==
為主場位於大阪的[[日本職棒|日本職棒]]球隊[[阪神虎|阪神虎]]的球迷，興趣是「考慮阪神虎的打線順序」。和[[東京養樂多燕子|東京養樂多燕子]]球迷的[[松嵜麗|松嵜麗]]是會彼此互稱「燕！」、「阪神！」的好友<ref>{{Cite web |url=http://www.gamer.ne.jp/news/201409260003|title=「アイドルマスター ワンフォーオール」DLC配信記念スペシャルインタビュー第3回：新庄と桧山に恋した少女が夢に賭けた青春！ 横山奈緒役・渡部優衣さんに聞きました。|publisher=Gamer|accessdate=2015-05-18}}</ref>。

與同個事務所的[[山口立花子|山口立花子]]交情很好<ref>{{Cite web|url=http://news.mynavi.jp/news/2014/11/25/498/|title=新人声優・渡部優衣インタビュー＆グラビア 【新人声優図鑑】|publisher=[[マイナビ|マイナビ]]ニュース|accessdate=2015-05-18|deadurl=yes|archiveurl=https://web.archive.org/web/20150519064511/http://news.mynavi.jp/news/2014/11/25/498/|archivedate=2015-05-19}}</ref>。
