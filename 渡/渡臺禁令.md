'''渡臺禁令'''，是指[[清朝|清朝]]於1684年派遣[[施琅|施琅]]擊敗[[明鄭時期|鄭氏政權]]並將[[臺灣|臺灣]]納入版圖後所公佈的移民政策，即「不許[[偷渡|偷渡]]來臺、不許携眷來臺、不許[[廣東人|粵民]]來臺」。<ref>[http://www.taiwanus.net/history/3/39.htm 滿清的禁止移民政策]</ref>
{{台灣歷史}}

== 背景 ==
1683年，[[清|清]][[康熙帝|康熙帝]]派[[施琅|施琅]]領兵攻打[[臺灣島|臺灣島]]，[[鄭克塽|鄭克塽]]隨後投降，[[明鄭|明鄭]]政權覆亡。

然而，康熙用兵於臺灣，僅是為了掃除前朝反對勢力，並非有意治理臺灣；當時，康熙曾表示：「臺灣僅彈丸之地，得之無所加，不得無所損」，[[朝廷|朝廷]]中亦有不少大臣反對將臺灣併入帝國版圖。<ref>{{Cite web|url=http://www.shs.edu.tw/works/essay/2014/03/2014032822392219.pdf|title=渡臺禁令|accessdate=2016-11-30|author=林思綺,陳婷均|date=|publisher=}}</ref>

經施琅屢次上奏（如在〈[[恭陳臺灣棄留疏|恭陳臺灣棄留疏]]〉中提到將臺灣併入版圖並不會增加中央的財政及兵力負擔，此外該地又更可作為東南沿海各省的屏障），康熙帝終於接受其建議，並於1684年正式將臺灣納入版圖，隸屬[[福建省|福建省]][[臺廈道|臺廈道]][[臺灣府|臺灣府]]。

但是，[[臺灣清治時期|清朝治理臺灣]]初期仍多採取被動防制政策，並對[[臺灣人|臺灣人]]非常不信任，制定各種嚴厲的禁令與規定，如：依施琅之建議頒布〈渡臺禁令〉限制[[漢人|漢人]]前往臺灣、實行[[班兵制度|班兵制度]]以及禁止各地建築（土）城等。<ref>{{Cite web|url=http://distance.shu.edu.tw/taiwan/ch09/CH09_SEC02.HTM|title=第9章第二節　清初移民三禁|accessdate=2016-11-30|work=distance.shu.edu.tw}}</ref><ref>李筱峰.戴寶村 主持.《[[臺灣史望春風|臺灣史望春風]]》.中華電視公司.2005</ref>

== 條文內容 ==
壹、欲渡船臺灣者，先給原籍地方[[照單|照單]]，經分巡[[臺廈兵備道|臺廈兵備道]]稽查，依[[臺灣府海防補盜同知|臺灣海防同知]]審驗批准，潛渡者嚴處。

貳、渡臺者不准攜帶妻兒家眷，業經渡臺者，亦不得招致。（此條在1732年一度廢止，後又重新發布）

参、[[粵|粵]]地屢為[[海盜|海盜]]淵藪，以積習未脫，禁其民渡臺。<ref>{{Cite book|url=https://books.google.com.tw/books?id=7OkYCgAAQBAJ&pg=PA48&lpg=PA48&dq=%E6%B8%A1%E8%87%BA%E7%A6%81%E4%BB%A4&source=bl&ots=iDZ1RTDNK_&sig=LZ3AzFG8qV8m83n8wY6My6bmBt4&hl=zh-TW&sa=X&ved=0ahUKEwjG5da6_M_QAhUCW5QKHXfgAQQ4KBDoAQg_MAc#v=onepage&q=%E6%B8%A1%E8%87%BA%E7%A6%81%E4%BB%A4&f=false|title=追尋臺灣法律的足跡: 事件百選與法律史研究|last=王泰升|last2=薛化元|last3=黃世杰|date=2015-07-17|publisher=台灣五南圖書出版股份有限公司|isbn=9789571178615|language=zh-TW}}</ref>

== 影響沿革 ==
其中第二條禁令造成[[臺灣人口|臺灣人口]]之[[性別比|男女比例]]嚴重失衡，也促成某些移民向[[原住民|原住民]]（大多為[[平埔族|平埔族]]族群）婦女求婚的意願，並藉此獲得其於[[母系制度|母系社會]]系統中所繼承之財產（如土地），當地亦因此產生[[諺語|諺語]]「有[[唐山_(消歧义)|唐山]]公，無唐山媽」<ref name=":0">{{Cite web|url=http://wsp.ks.edu.tw/~twn/adage.htm|title=台灣歷史與台灣諺語（總篇）|accessdate=2016-11-30|author=黃招榮|date=|publisher=}}</ref>。某些無法娶妻，或[[髮妻|髮妻]]未能得子，又希望有後代繼承其[[香火|香火]]者，往往以[[過繼|過繼]]、收養子等方式因應此一問題，因此產生「大量[[收養|螟蛉子]]」之現象。<ref>{{Cite book|url=https://books.google.com.tw/books?id=3XoHzQPG0pEC&pg=PA17&lpg=PA17&dq=%E3%80%8C%E6%B8%A1%E8%87%BA%E7%A6%81%E4%BB%A4%E3%80%8D%E7%9A%84%E9%A0%92%E8%A1%8C&source=bl&ots=4e58NF2mZK&sig=pE-uR-wZw65KXft_rtVI1zycJc4&hl=zh-TW&sa=X&ved=0ahUKEwivy-Kd8s_QAhWEi5QKHYfNDIwQ6AEIRzAJ#v=onepage&q=%E3%80%8C%E6%B8%A1%E8%87%BA%E7%A6%81%E4%BB%A4%E3%80%8D%E7%9A%84%E9%A0%92%E8%A1%8C&f=false|title=女真後裔在臺灣: 粘氏宗族與彰化福興地區的發展|date=2005-01-01|publisher=五南圖書出版股份有限公司|isbn=9789867332448|language=zh-TW}}</ref>，平埔族重視女兒的風俗融入，也造成臺灣「[[抽豬母租|抽豬母租]]」的風氣盛行。此外，大量[[羅漢腳|羅漢腳]]（男性[[遊民|遊民]]）亦因此禁令產生，成為當地社會動亂（如[[分類械鬥|械鬥]]）之源頭。

而第三條禁令之產生係因[[粵東人|粵東人]]在[[中國大陸|中國大陸]]屢次幫助[[明鄭王朝|鄭軍]]對抗清軍；在施琅過世後，該禁令即告解除，過去認為此令造成[[臺灣客家人|臺灣客家人]]在[[臺灣人口|人口]]數上不及[[閩南人|閩南]]籍者，或遭同化成[[福佬客|福佬客]]。<ref>{{Cite web|url=http://163.32.177.8/hakka/h4.htm|title=客家語言|accessdate=2016-11-30|work=163.32.177.8}}</ref><ref>{{Cite book|url=https://books.google.com.tw/books?id=cTc-iVts9oUC&pg=PA77&lpg=PA77&dq=%E6%B8%A1%E8%87%BA%E7%A6%81%E4%BB%A4&source=bl&ots=GTHLzI7wJu&sig=PfM84wHdIShU7t339GQrrqVB-fQ&hl=zh-TW&sa=X&ved=0ahUKEwjbm6n--8_QAhXJk5QKHSnyBUw4HhDoAQhDMAk#v=onepage&q=%E6%B8%A1%E8%87%BA%E7%A6%81%E4%BB%A4&f=false|title=戰後臺灣的歷史學硏究, 1945-2000: 臺灣史|date=2004-01-01|publisher=國立臺灣大學出版中心|isbn=9789570176193|language=zh-TW}}</ref>{{or|但此說目前因合理性不足而備受質疑。|time=2018-03-26}}

前述兩條禁令之執行雖時寬時嚴，卻始終沒有取消。<ref>{{Cite book|url=https://books.google.com.tw/books?id=xtlNAgAAQBAJ&pg=PA444&lpg=PA444&dq=%E6%B8%A1%E8%87%BA%E7%A6%81%E4%BB%A4&source=bl&ots=oBzV0Ubaga&sig=hSXAhYI91s6YlXwAqynh4oZidV8&hl=zh-TW&sa=X&ved=0ahUKEwig4daP-c_QAhUKmZQKHd3vD0o4FBDoAQhQMAk#v=onepage&q=%E6%B8%A1%E8%87%BA%E7%A6%81%E4%BB%A4&f=false|title=盛清社會與揚州研究|last=馮明珠等|first=|date=2011-11-09|publisher=遠流出版|year=|isbn=9789573268918|location=|pages=|language=zh-TW}}</ref>

1875年，[[沈葆楨|沈葆楨]]受命出任[[沈葆楨#臺灣海防欽差大臣|臺灣海防欽差大臣]]，經考察後認為必須解除過去限制漢人的渡臺禁令以落實「[[開山撫番|開山撫蕃]]」政策，否則臺灣之土地將無法充分開發；同年2月，沈葆楨上奏請求解除關於臺灣之各種禁令，並獲得朝廷准許。

臺灣於清治初期之出入港口亦因此只開放[[福建|福建]][[廈門|廈門]]與[[臺南|臺南]][[鹿耳門|鹿耳門]]對渡；後因[[北臺灣|北臺灣]]逐漸開發，才分別於1784年和1788年開放[[鹿港鎮|鹿港]]及[[八里坌|八里坌]]為通商口岸，當地亦因此產生俗諺「[[一府二鹿三艋舺|一府二鹿三艋舺]]」。<ref name=":0" /><ref>{{Cite web|url=http://library.taiwanschoolnet.org/cyberfair2003/C0316360079/web/iohw/doc17.htm|title=【府城舊事】‧清領時期|accessdate=2016-11-30|work=library.taiwanschoolnet.org}}</ref>
