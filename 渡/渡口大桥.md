'''渡口大桥'''或称'''渡口桥'''，是[[中華人民共和國|中華人民共和國]][[四川省|四川省]][[攀枝花市|攀枝花市]][[东区_(攀枝花市)|东区]]的一座公路桥，跨越[[金沙江|金沙江]]，连接北岸[[向阳村街道|向阳村]]和南岸[[大渡口街道|大渡口]]。

此处先后曾有三座名为渡口桥的跨金沙江桥梁，现时可分别称为'''渡口吊桥'''、'''老渡口大桥'''和'''新渡口大桥'''，其中老渡口大桥已于2002年被拆除，其余两座桥仍在使用中。今日人们所称之渡口桥或渡口大桥，若无特别说明，皆指2005年建成的新渡口大桥。

== 建桥起因 ==
攀枝花市始建于1965年，初名渡口市，[[市辖区|市辖区]]在建市前分属于[[云南省|云南省]][[永仁县|永仁县]]和[[华坪县|华坪县]]以及[[四川省|四川省]][[盐边县|盐边县]]和[[会理县|会理县]]，渡口桥处的金沙江曾是两省的界江。金沙江沿岸多为农业地区，起初两岸交往不多，往来依靠渡船。1960年代国家[[三线建设|三线建设]]新建项目攀枝花钢铁基地开工建设后，金沙江成为公路交通运输的最大障碍，解决过江问题成为燃眉之急。

== 渡口吊桥 ==
[[File:Dukou_Suspension_Bridge.jpg|thumb]]
为适应攀枝花基地建设时间紧、速度快的需要，确保物资过江及建设队伍的集结，渡口总指挥部决定，在渡口大桥建成以前，先修建一座简易跨江人行吊桥，以解决过江问题。

1964年9月，四川省交通厅勘察设计院对现场进行勘察。原设计为1孔跨径180米无加劲桁人行吊桥，上下游主索中距4米，桥面净宽3.8米，除人行外，可通行2吨以下汽车。由于使用密封异形钢丝绳，原设计圆形锚锭桩卡子锚紧固方法无法卡固，1964年底决定改锚锭坑设计为套简锚固。改变设计后，尚有增加载重的潜力，总指挥部研究后决定于1965年1月重新设计，改变吊杆及桥面系的设计，达到通过8吨汽车的载重量，由四川省交通厅公路工程局第一工程处一队承担此桥施工。1964年10月21日正式开工挖基，仅用4个月零20天的时间，于1965年3月11日正式通车。

渡口吊桥完成主要工程数量为：开挖塔基及锚锭坑石方1590立方米，浇筑[[混凝土|混凝土]]及[[钢筋混凝土|钢筋混凝土]]1248立方米，浆砌石料300立方米，架设桥面180米，修筑桥头[[引道|引道]]305米，全桥实耗主材料为直径30.5及直径40.5毫米密封异形钢丝绳59.84吨，型钢12.84吨，钢筋19.7吨，钢铁铸锻件16.67吨，水泥432吨，原木609立方米，锌522公斤，由于此桥为攀枝花基地建设重点突击任务，采取边设计边施工的办法，投资控制数为92万元，后来两次进行修改设计，改人行为单车行，因而增加套简拉杆，增大桥面系部分构件尺寸，实际开支为99.61万元。<ref name="桥梁网百科">{{cite web| title =攀枝花渡口大桥| publisher =中国桥梁网百科| date =| url =http://www.cnbridge.cn/baike/index.php?doc-view-4381.html| accessdate =| language =| deadurl =yes| archiveurl =https://web.archive.org/web/20160305114659/http://www.cnbridge.cn/baike/index.php?doc-view-4381.html| archivedate =2016-03-05}}</ref>

渡口吊桥通车两年时间，昼夜承受着南来北往的汽车及过往的行人，汽车排队过桥有时多达300余辆，少则30至50辆。由于长期超负载使用，大大缩短了使用寿命，出现了桥北锚锭坑开裂、桥面木材腐烂现象。1966年12月渡口大桥建成通车，吊桥胜利完成历史使命，便拆除了吊桥两头桥面，封闭禁行。1981年，[[攀钢|攀钢]]将此桥改作输送煤气管道专用桥，移交市煤气公司管理。<ref name="第三章桥梁和渡口">{{cite web| title =攀枝花市志 第八篇交通 第三章桥梁和渡口| publisher =www.panzhihua.gov.cn| date =1994年7月| url =http://www.panzhihua.gov.cn/zjpzh/pzhsz/dbpjt/891.shtml| accessdate =| language =| deadurl =yes| archiveurl =https://web.archive.org/web/20140202114424/http://www.panzhihua.gov.cn/zjpzh/pzhsz/dbpjt/891.shtml| archivedate =2014-02-02}}</ref>

== 老渡口大桥 ==
[[File:绘有渡口大桥的盘子.jpg|thumb]]
渡口大桥于1966年12月建成通车，位于渡口吊桥上游约600米处，连接大渡口和攀钢厂区。桥梁代号为3002桥，是攀枝花钢铁基地继[[雅江桥|雅江桥]]（3001桥）之后第二座非临时性的跨江大桥，也是城区第一座正式桥梁。

大桥全长304.5米，设计载重标准为汽—18、拖—80，桥面净宽8米，两侧人行道各宽1.5米，沥青混凝土挂铁栏杆，安装水银路灯46盏。大桥正桥为单孔净跨180米等高悬链折线钢板箱肋无铰拱，钢箱拱主孔拱肋采用高强度螺栓连接；引桥北岸为二孔15米，南岸三孔15米，采用梳型钢板伸缩缝；由交通部第二公路勘察设计院设计，交通部第二公路局第五工程处（后改为渡口市桥梁工程处）施工，1965年7月动工建设，1966年12月竣工通车，是中国60年代第一座跨径最大的钢拱公路桥。<ref name = "第三章桥梁和渡口"/>

== 新渡口大桥 ==
[[File:渡口.jpg|thumb]]
1997年，攀枝花桥梁养护部门在对渡口大桥进行钢结构[[喷砂|喷砂]]除锈时发现，与以往所检测的数据对比，桥梁主要构件螺栓断裂严重、钢纵梁变形偏移且与轴线偏移量较大。在进行病害处理的同时，向市交通局提交了报告，市交通局立即成立了专家组，对该桥整治提出了方案。2000年6月1日，攀枝花市对渡口大桥进行了限载通行措施。<ref>{{cite web| title =跨越——攀枝花市公路桥梁管养工作纪实| publisher =攀枝花日报| date =2007年12月25日| url =http://pzhrb.newssc.org/html/2007-12/25/content_30052.htm| accessdate =| language =}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>2002年2月6日，攀枝花市根据省交通厅公路局文件要求，正式关闭了渡口大桥，并决定在拆除该桥后在原址建一座新渡口大桥。为攀枝花钢铁基地建设立下大功的渡口大桥，安全运行35年之后拆除，在大桥原址建设的新渡口大桥于2002年12月29日开工。

新渡口大桥计划投资4100万元，桥型为上承式[[钢筋混凝土|钢筋砼]]箱形拱桥，主孔净跨170米，桥面全宽26.5米，设计荷载等级为汽—20、拖—120，桥梁总长为385.88米，双向4车道。新桥于2005年3月3日建成通车。<ref>{{cite web| title =攀枝花市新渡口大桥通车| publisher =四川交通在线| date =2005年3月9日| url =http://www.scjtonline.cn/Show_News.asp?NewsId=495| accessdate =| language =}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>新渡口大桥建设将交通功能与美化、亮化及桥南、桥北广场建设进行整体构思，成为攀枝花市的又一城市景观。新桥保留了旧桥在外形上的特征，新桥建好后，犹如旧桥再现。<ref name = "桥梁网百科"/>

== 纪念意义 ==
由于老渡口大桥是攀枝花市城区第一座正式的跨江大桥，几乎与攀枝花市同岁，全程见证了攀枝花的开发建设历史，且桥名与攀枝花市的原名渡口市同名，是城市的精神象征，在攀枝花市民心中具有崇高地位。亦因此，市政府拆除老桥的决定多年来一直受到批评。而此事也引致民众更强烈地反对当前老[[密地大桥|密地大桥]]（3003桥）的拆除决议。
