'''转录'''（{{Lang-en|Transcription}}）是在[[RNA聚合酶|RNA聚合酶]]的催化下，遗传信息由[[DNA|DNA]]复制到[[RNA|RNA]]（尤其是[[mRNA|mRNA]]）的过程。作为[[蛋白质生物合成|蛋白质生物合成]]的第一步，转录是合成[[mRNA|mRNA]]以及[[非編碼RNA|非編碼RNA]]（[[tRNA|tRNA]]、[[rRNA|rRNA]]等）的途径。

[[真核生物|真核生物]]合成蛋白质的转录过程以特定的单链[[DNA|DNA]]片段作为模板，[[RNA聚合酶|RNA聚合酶]]作为[[催化剂|催化剂]]，合成[[前mRNA|前mRNA]]，前mRNA经进一步加工后转为成熟mRNA。转录时，DNA分子的双链打开（是否需要[[DNA解旋酶|DNA解旋酶]]仍存在争议），在RNA聚合酶的作用下，游离的4种[[核糖核苷酸|核糖核苷酸]]按照[[碱基互补配对原则|碱基互补配对原则]]结合到DNA单链上，并在RNA聚合酶的作用下形成单链mRNA分子。

转录成RNA分子的DNA片段称为转录单元，编码至少一个基因。如果转录的基因编码蛋白质，则会产生[[信使RNA|信使RNA]]（mRNA），这个mRNA又在[[翻译_(生物学)|翻译]]过程中作为合成蛋白质的模板。基因还可能编码[[非编码RNA|非编码RNA]]，例如[[小分子RNA|小分子RNA]]，[[核糖体RNA|核糖体RNA]]（rRNA），[[转运RNA|转运RNA]] （tRNA）或有催化作用的RNA分子[[核酶|核酶]]。

== 主要步骤 ==
转录通常按以下步骤进行：
# RNA聚合酶与一种或多种[[通用转录因子|通用转录因子]]一起结合[[启动子|启动子DNA]]。
# RNA聚合酶产生一个{{le|转录泡|Transcription bubble}}，并通过打开互补DNA核苷酸之间的[[氢键|氢键]]分离DNA双螺旋的两条链。
# RNA聚合酶催化聚合核糖核苷酸（与模板DNA链的脱氧核糖核苷酸互补）。
# 在RNA聚合酶的作用下形成RNA糖—磷酸骨架，进而形成RNA链。
# RNA—DNA螺旋的氢键断裂，释放新合成的RNA链。
# 如果细胞有[[细胞核|核]]，RNA可以进一步被处理。 这可能包括[[聚腺苷酸化|聚腺苷酸化]]，[[5'端帽|加帽]]和[[RNA剪接|剪接]]等。
# RNA可以保留在核内或通过[[核孔|核孔]]复合物离开核进入[[细胞质|细胞质]]。

具体来讲，转录可分为启动、延伸、终止三个阶段<ref name="nature">{{Cite web | url = https://www.nature.com/scitable/topicpage/dna-transcription-426 | title = DNA Transcription | language = en}}</ref>。
=== 启动 ===
转录开始于[[RNA聚合酶|RNA聚合酶]]与[[通用转录因子|通用转录因子]]共同结合到模板DNA的[[启动子|启动子序列]]上，形成RNA聚合酶－启动子闭合复合物。

然后，RNA聚合酶在通用转录因子的协助下，暴露出大约14个[[碱基对|碱基对]]，形成RNA聚合酶－启动子开放复合物。在开放复合物中，启动子DNA部分解开为单链。 暴露的单链DNA被称为“转录泡”。接着，RNA聚合酶选择转录泡中的转录起始位点，结合起始[[NTP|NTP]]和与转录起始位点序列互补的延伸NTP（也可能是短RNA[[引物|引物]]和延伸NTP），催化[[磷酸二酯键|磷酸二酯键]]的形成，产生起始RNA产物。

原核生物的RNA聚合酶全酶由两个α[[亚基|亚基]]、一个β亚基、一个β′亚基和一个σ亚基组成，其中[[σ因子|σ因子]]协助RNA聚合酶识别并结合启动子。<ref name="bio-med">{{Cite web|url=http://bio.med.stu.edu.cn/news/edit/UploadFile/2012112111331161.ppt|title=RNA的生物合成（转录）}}</ref>很多基因起始位点的上游拥有[[普里布诺盒|普里布诺盒]]基因序列（TATAAT），上游约35个碱基处拥有TTGCCA共有序列，约40~60个碱基处拥有一片富含A、T碱基的区域，有助于加快转录速度。<ref name="nature"></ref>

真核生物的转录起始上游区段比原核生物多样化，转录起始时，RNA聚合酶不直接结合模板，其起始过程比原核生物复杂。<ref name="bio-med"></ref>

转录的启动还受到另外的蛋白质影响，如[[信号转导及转录激活蛋白|转录激活蛋白]]和{{le|阻遏蛋白|Repressor}}。

前两个NTP缩合成[[磷酸二酯键|3′—5′磷酸二酯键]]后，RNA聚合酶脱离启动子，启动阶段结束，进入延伸阶段。

=== 延伸 ===
[[File:simple_transcription_elongation1.svg|thumb]]
转录启动后，DNA双螺旋链解开，其中一条链作为模板链。随着转录的进行，RNA聚合酶在DNA模板链上移动，根据[[碱基互补配对原则|碱基互补配对原则]]合成RNA单链。转录的[[方向性_(分子生物学)|方向]]为3′→5′（对于DNA模板链）或5′→3′（对于合成的RNA链或DNA编码链）。

细菌在37°C（98.6°F）下以42~54个核苷酸每秒的速度转录，而真核生物的转录速度大约是22-25个核苷酸每秒。<ref name="nature"></ref>
=== 终止 ===
{{Main|终止子}}
转录终止于DNA非编码序列末端附近的[[终止子|终止子]]处。

[[细菌|细菌]]转录的终止有两种方式：内源性终止（也称内在型终止，本体转录终止或ρ-非依赖性终止）和ρ-依赖性终止。在内源性终止中，基因末端的{{le|反向重复序列|Inverted repeat}}使新转录的RNA序列折叠成[[茎环|发夹环]]，从而使RNA聚合酶脱落，转录产物被释放。<ref>{{Cite web | url = https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3622210/#!po=11.2069 | title = Bacterial Transcription Terminators: The RNA 3′-End Chronicles | language = en}}</ref>ρ-依赖性终止子利用可在转录时主动展开DNA—RNA复合物的ρ因子，释放新合成的RNA。<ref name="nature"></ref>

[[真核生物|真核生物]]转录过程的终止方式取决于其所使用的聚合酶的不同。RNA聚合酶I的终止机制与细菌的ρ-依赖性终止类似；而RNA聚合酶III的终止与细菌的内源性终止极为相似。有研究表明，多个[[胸腺嘧啶|胸腺嘧啶]]的终止信号引起RNA聚合酶III的失活，从而使延伸终止并将酶转运至最近的RNA二级结构，进而促进酶的释放。RNA聚合酶III和细菌的转录终止机制之间的相似性表明，这种依赖[[发夹环|发夹环]]的终止可以追溯到多亚基RNA聚合酶的共同祖先。<ref>{{Cite web | url = https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3760304/ | title = Mechanism of Eukaryotic RNA polymerase III transcription termination | language = en}}</ref>RNA聚合酶II的转录可以在非编码序列的末端后持续数百甚至数千个核苷酸，它的终止更为复杂，涉及到新合成RNA的切割以及[[多腺苷酸化|多腺苷酸化]]的过程。

==抑制剂==
[[抗生素|抗生素]]可作为转录的抑制剂，包括[[病原细菌|抗细菌药]]与[[抗真菌药|抗真菌药]]。以利福平为例，其通过结合DNA依赖性[[RNA聚合酶|RNA聚合酶]]的β-亚基来抑制{{le|细菌转录|Bacterial transcription}}；而8-羟基喹啉是{{le|真菌转录|Eukaryotic transcription}}的抑制剂；{{le|组蛋白甲基化作用|Histone methylation}}也可能会对转录有抑制作用。

== 历史 ==
[[方斯華·賈克柏|方斯華·賈克柏]]（François Jacob）和[[賈克·莫諾|賈克·莫諾]]（Jacques Lucien Monod）首先假设一种让遗传物质成为蛋白质的分子。 [[塞韦罗·奥乔亚|塞韦罗·奥乔亚]]（Severo Ochoa）于1959年获得[[诺贝尔生理学或医学奖|诺贝尔生理学或医学奖]]，开发一种用{{le|多核苷酸磷酸化酶|Polynucleotide phosphorylase}}体外([[In_vitro|In vitro]])合成RNA的方法，该方法可用于破解[[遗传密码|遗传密码]]。 由RNA聚合酶合成RNA是由几个实验室在1965年以前在体外([[In_vitro|In vitro]])建立的; 然而，由这些酶合成的RNA具有表明存在正确终止转录所需的额外因子的特性{{citation needed|date=January 2011}}。

1972年，Walter Fiers成为第一个真正证明终止酶存在的人。

[[罗杰·科恩伯格|罗杰·科恩伯格]]（Roger D. Kornberg）因其研究{{le|真核生物转录|Eukaryotic transcription}}的分子基础而获得2006年[[诺贝尔化学奖|诺贝尔化学奖]]<ref>{{cite web|url=http://nobelprize.org/nobel_prizes/chemistry/laureates/2006/|title=Chemistry 2006|work=Nobel Foundation|accessdate=March 29, 2007}}</ref>。
