'''转运核糖核酸'''（{{lang|en|Transfer RNA}}），又称'''傳送核糖核酸'''、'''转移核糖核酸'''，通常简称為'''tRNA'''，是一種由76-90個[[核苷酸|核苷酸]]所組成的[[RNA|RNA]]<ref name=sharp1985>{{cite journal|last1=Sharp|first1=Stephen J|last2=Schaack|first2=Jerome|last3=Cooley|first3=Lynn|last4=Burke|first4=Deborah J|last5=Soll|first5=Dieter|title=Structure and Transcription of Eukaryotic tRNA Genes|journal=CRC Critical Reviews in Biochemistry|date=1985|volume=19|issue=2|pages=107–144|doi=10.3109/10409238509082541|pmid=3905254}}</ref>，其3'端可以在[[氨酰-tRNA合成酶|氨酰-tRNA合成酶]]催化之下，接附特定種類的[[氨基酸|氨基酸]]。[[轉譯|轉譯]]的過程中，tRNA可藉由自身的[[反密碼子|反密碼子]]識別[[mRNA|mRNA]]上的[[密碼子|密碼子]]，將該密碼子對應的氨基酸轉運至[[核糖體|核糖體]]合成中的[[多肽|多肽鏈]]上。每個tRNA分子理論上只能與一種[[氨基酸|氨基酸]]接附，但是[[遺傳密碼|遺傳密碼]]有简并性（degeneracy），使得有多於一個以上的tRNA可以跟一種[[氨基酸|氨基酸]]接附。

== tRNA的发现 ==
1955年Zamecnik认为标记的ATP可能参与RNA的生物合成。于是他将<sup>14</sup>C标记的ATP与微粒体（Microsome）和细胞抽提液的可溶性部分一起保温后，发现RNA居然也被标记了。他有点怀疑。可是，当他将<sup>14</sup>C标记的氨基酸与微粒体和可溶性部分在同样条件下保温后，他惊奇地发现，与RNA合成无关的<sup>14</sup>C氨基酸也标记了RNA，而且更意想不到的是<sup>14</sup>C标记的RNA不是核糖体的大分子RNA，而是可溶性部分中的小分子RNA。进一步，仅将可溶性部分与<sup>14</sup>C标记的氨基酸和ATP一起保温，则这种<sup>14</sup>C标记的氨基酸仍能与其中的小分子RNA结合。因此，这种可溶性部分中的小分子RNA被称为称sRNA（soluble RNA）。1956年Watson曾访问Zamecnik实验室，并对他们说，1955年Crick已经提出过“适配子”的设想。后来，这种 sRNA被命名为tRNA。

== tRNA的結構 ==
[[File:TRNA-Phe_yeast_en.svg|thumb]]
[[File:TRNA-Phe_yeast_1ehz.png|thumb]]
tRNA為74~95個[[鹼基|鹼基]]的小片段RNA鏈，會折疊成苜蓿葉狀的[[核酸二級結構|核酸二級結構]]，呈三叶草形，它由氨基酸臂、二氢尿嘧啶环、反密码环、额外环和TΨC环五部分组成。

tRNA有一级结构（5'到3'的核苷酸方向），二级结构（通常显示为三叶草结构）和三级结构（所有的tRNA具有类似L-形的三维结构，允许它们与[[核糖体|核糖体]]的P、A位点结合）。

=== 特色 ===
# [[方向性#5'端|5'端]][[磷酸|磷酸]]。
# [[受体臂|受体臂]]（accept stem，也被稱作''amino acid stem''）是一個7個鹼基長的臂，其中包含5'端，與有3'端[[羟基|羟基]]（[[OH|OH]]，能結合胺基酸於其上）的3'端。受体臂有可能含有非Watson-Crick所發現的鹼基對。
# [[CCA尾|CCA尾]]（CCA tail）是tRNA分子3'端的CCA序列，在翻译时，帮助酶识别tRNA。
# [[D臂|D臂]]（D arm）是在一个环（D loop）的端部4个碱基的臂，通常含有[[二氢尿嘧啶|二氢尿嘧啶]]（dihydrouridine）。
# [[反密码子臂|反密码子臂]]（anticodon arm）有5个碱基，包括[[反密码子|反密码子]]（anticodon）。每一tRNA包括一个特异的三联反密码子序列，能够与氨基酸的一个或者多个密码子匹配。例如[[赖氨酸|赖氨酸]]（lysine）的密码子之一是AAA，相应的tRNA的反密码子可能是UUU（一些反密码子可以与多于一个的密码子匹配被称为“摆动”）。
# [[T臂|T臂]]（T arm）是5个碱基的茎，包括序列TψC。
# [[修饰碱基|修饰碱基]]（Modified bases）是tRNA中的一些不常见的碱基，如腺嘌呤、鸟嘌呤、胞嘧啶和尿嘧啶的修饰形式。

=== 氨酰化 ===
{{see also|氨酰tRNA}}
[[氨酰化|氨酰化]]（Aminoacylation）是添加一个氨酰基团到化合物的过程。 

在[[氨酰tRNA合成酶|氨酰tRNA合成酶]]（aminoacyl tRNA synthetase）的作用下，tRNA与特异的氨基酸进行[[氨酰化反应|氨酰化反应]]（aminoacylated）。对于一种氨基酸而言，尽管可能有多种 tRNA和多种反密码子，但是通常只有一种氨酰tRNA合成酶。合成酶对合适的tRNA的识别不仅仅是反密码子，受体臂也起了显著的作用<ref name="pmid7692438">{{cite journal |vauthors = Schimmel P, Giegé R, Moras D, Yokoyama S |title = An operational RNA code for amino acids and possible relationship to genetic code |journal = Proc. Natl. Acad. Sci. U.S.A. |volume = 90 |issue = 19 |pages = 8763–8 |year = 1993 |pmid = 7692438 |pmc = 47440 |doi =10.1073/pnas.90.19.8763|bibcode = 1993PNAS...90.8763S}}</ref>。

反应：
# 氨基酸 + [[三磷酸腺苷|ATP]] →氨基酰- AMP + [[焦磷酸盐|PPi]]
# 氨基酰-AMP + tRNA →氨基酰 - tRNA + [[單磷酸腺苷|AMP]]

某些生物可能缺少一种或多种氨酰基tRNA合成酶。 这导致通过化学相关的氨基酸被氨酰化的tRNA，并且通过使用一种或多种酶，tRNA被修饰为正确的被氨酰化。
