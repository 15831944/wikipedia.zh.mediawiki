'''平均寿命'''、'''生命期望'''或'''預期壽命'''（{{lang-en|life expectancy}}），指生物群体中衡量单一生命存活平均长度的统计量。預期壽命最常用的測量方法是自出生起算（{{lang-en|Life expectancy at birth}}，LEB），也常以不同性別、不同年齡去做統計。

== 历史 ==

[[19世纪|19世纪]]初，全球各地平均寿命不超过40岁，一半人口在成年前[[夭折|夭折]]。1800年之前，只有欧洲少数精英阶层的人均寿命超过40岁。1820年左右，[[西欧|西欧]]平均寿命是36岁，[[日本|日本]]是34岁。[[中国贵族|中国贵族]]的人均寿命由1800年代初的37岁降至1830年后的32岁<ref name="a1">{{Cite web |url=http://news.163.com/16/1107/10/C58UHNM0000187UE.html |title="长寿时代":19世纪以来各国人均寿命的增长 |accessdate=2018-01-03 |author= |date=2016-11-07 |work= |publisher=网易|language=Zh-hans }}</ref>。

欧洲国家进入[[工业化|工业化]]早期，人均寿命曾有所下降<ref name="a1"/>，其后有大幅提高。西欧[[发达国家|发达国家]]自19世纪早期开始快速提高，至1950年时达到65岁。与西欧类似，[[北欧|北欧]]、[[北美|北美]]、[[澳新|澳新]]的平均寿命在1950年代时接近70岁。[[南欧|南欧]]、[[东欧|东欧]]则在20世纪初由40岁提高到50岁，于1970年代时追赶上西欧。[[二战|二战]]结束后，大部分[[发展中国家|发展中国家]]的人均寿命随着[[死亡率|死亡率]]下降而增加。1950年时，发展中国家平均寿命为40岁，1965年，提高至50岁。其中在1970年代中期，非洲平均寿命47岁，亚洲为57岁，拉丁美洲达为62岁。1990年代，发达国家平均寿命达到73岁，发展中国家平均寿命57岁，但发展中国家平均寿命状况不一，并不均衡<ref name="a2">{{cite journal en|quotes =

== 影响因素 ==

经济、文化、[[医疗|医疗]]是多种影响因素。[[婴儿死亡率|婴儿死亡率]]是影响平均寿命的直接因素之一<ref name="a2"/>。
