'''预测性维护'''（'''PdM'''）技术用于确定在役[[设备|设备]]状况，以预测何时需要做{{le|维护|Maintenance (technical)}}。这比日常或定期进行的[[预防性维护|预防性维护]]能节约成本，因为只在确有必要时才做维护。 
 
预测性维护的实现通常包含下述组成部分：<ref>{{Cite book|url=https://www.researchgate.net/publication/322900854_Fault_Class_Prediction_in_Unsupervised_Learning_using_Model-Based_Clustering_Approach?channel=doi&linkId=5a74cec40f7e9b41dbce3114&showFulltext=true|title=Fault Class Prediction in Unsupervised Learning using Model-Based Clustering Approach|last=Amruthnath|first=Nagdev|last2=Gupta|first2=Tarun|date=2018-02-02|doi=10.13140/rg.2.2.22085.14563}}</ref> 
*数据收集与预处理
*早期故障检测
*故障检测
*失败预测
*维护计划
*资源优化

预测性维护有助于改善生产率、达成[[及時制度|即时化生产]]。<ref>{{Cite book|url=https://www.researchgate.net/publication/322869981_A_Research_Study_on_Unsupervised_Machine_Learning_Algorithms_for_Fault_Detection_in_Predictive_Maintenance?channel=doi&linkId=5a738a1ca6fdcc53fe1474b5&showFulltext=true|title=A Research Study on Unsupervised Machine Learning Algorithms for Fault Detection in Predictive Maintenance|last=Amruthnath|first=Nagdev|last2=Gupta|first2=Tarun|date=2018-02-01|doi=10.13140/rg.2.2.28822.24648}}</ref>

== 概述 ==
预测性维护首先要做定期（离线）或连续（在线）的设备[[状态监测|状态监测]]，以确定在设备发生故障前的成本效益最优的维护时机。<ref>{{Cite web|url=http://learnoilanalysis.com/oil-analysis-saves-you-money-but-how-much/|title=How much does predictive maintenance save you money?{{!}} Learn Oil Analysis|website=learnoilanalysis.com|language=en|access-date=2017-12-03|archive-url=https://web.archive.org/web/20171003234014/http://learnoilanalysis.com/oil-analysis-saves-you-money-but-how-much/#|archive-date=2017-10-03|dead-url=yes}}</ref>. 

"预测"部分是指预测设备状态的未来趋势这个目标。用{{le|统计过程控制|statistical process control}}原理来确认未来合适的维护时机。
