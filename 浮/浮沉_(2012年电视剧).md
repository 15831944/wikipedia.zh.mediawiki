== 剧情 ==
《浮沉》为职场情感和商战片，讲述了以主人公乔莉升职为销售员替赛思中国公司跑业务，卷入国企、私企和外企的黑幕纠纷之中的故事。根据崔曼莉小说《浮沉》改编，但是包括原著作者本人在内，大多数看过原著的读者观众都不认同改编效果。<ref>{{cite web |url= http://ent.ifeng.com/tv/news/mainland/detail_2012_06/16/15347442_1.shtml|title=《浮沉》30日开播 “黄小仙”恋上“大老王” |accessdate=2012年6月16日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

== 花絮 ==
该剧改编自[[崔曼莉|崔曼莉]][[长篇小说|长篇小说]]《浮沉》。

该剧的演员是《[[失恋33天|失恋33天]]》的原班人马，剧中，白百何和张嘉译上演了一出“[[萌叔|萌叔]]和[[萝莉|萝莉]][[老少配|忘年恋]]”的故事。<ref name="fish " >{{cite web |url=http://ent.qq.com/a/20120617/000114.htm? |title= 《浮沉》上海亮相 张嘉译：孙俪宋佳真没法说|accessdate=2012年6月17日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

该剧的主题是国企改制、官商勾结关系、职场斗争。<ref name="fish " />

该剧被誉为“升级版的《[[杜拉拉升职记|杜拉拉升职记]]》”。<ref>{{cite web |url=http://yule.sohu.com/20120613/n345532004.shtml |title= 《浮沉》：一部升级版《杜拉拉升职记》
|accessdate= 2012年6月13日|publisher=搜狐网 |work= |language =zh-cn}}</ref>
