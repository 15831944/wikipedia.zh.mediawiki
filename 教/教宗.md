'''教宗'''（{{lang-la|Papa}}），或譯'''教-{}-皇'''<ref>[[艾儒略|艾儒略]]，《[[:s:職方外紀_(四庫全書本)/卷2|職方外紀·卷二]]》：「教皇即居於此，以代天主在世。」</ref><ref>[[魏源|魏源]]，《[[:s:海國圖志/卷041|海國圖志·卷四十一]]》：「意大里亞為天主教之宗國，稱曰教皇。凡西洋各國王即位，必得教皇札付而後立。」</ref>（[[華語圈|華語圈]]外國家如[[韓國|韓國]]<ref>{{cite journal |title=교황 |url = https://ko.wikipedia.org/w/index.php?title=%EA%B5%90%ED%99%A9&oldid=18806916|date=2017-05-25 |journal=위키백과, 우리 모두의 백과사전 |language=ko }}</ref>、[[日本|日本]]<ref>{{cite journal |title = 教皇 |url = https://ja.wikipedia.org/w/index.php?title=%E6%95%99%E7%9A%87&oldid=64122470 |date=2017-05-15 |journal=Wikipedia |language=ja }}</ref>、[[越南|越南]]等目前仍用教皇為譯名<ref>《[[现代汉语词典|现代汉语词典]]》“教皇”条</ref><ref>{{cite news |title = 罗马教皇祝贺中国新年：中国一直是伟大的国家 |url = http://world.people.com.cn/n1/2016/0203/c1002-28107474.html|date=2016-02-03 |author=人民网国际频道}}</ref><ref>{{cite web|title=教育部國語辭典簡編本 「教皇」條|url=http://dict.concised.moe.edu.tw/cgi-bin/jbdic/gsweb.cgi?o=djbdic&searchid=Z00000022541|}}</ref>），是[[天主教羅馬教區|羅馬]][[主教|主教]]，同時為普世[[天主教會|天主教會]]領袖與[[梵蒂岡城國|梵蒂岡城國]][[国家元首|国家元首]]，傳統上被認為是[[西門彼得|聖伯多祿]][[宗徒|宗徒]]之位的[[使徒统绪|後繼者]]。現任（第266任）教宗[[方濟各_(教宗)|方濟各]]於2013年3月13日當選<ref>[http://zht.radiovaticana.va/news/2017/03/13/%E6%95%99%E5%AE%97%E6%96%B9%E6%BF%9F%E5%90%84%E7%95%B6%E9%81%B8%E5%9B%9B%E9%80%B1%E5%B9%B4%EF%BC%9A%E7%9C%9F%E6%AD%A3%E7%9A%84%E6%94%B9%E9%9D%A9%E5%A7%8B%E6%96%BC%E2%80%9C%E5%BF%83%E9%9D%88%E9%9D%A9%E6%96%B0%E2%80%9D/1298438 教宗方濟各當選四週年：真正的改革始於“心靈革新”]，[[梵蒂岡廣播電台|梵蒂岡廣播電台]]，2017年3月13日</ref>。

== 辞源 ==
教宗的[[拉丁文|拉丁文]]原詞「{{lang|la|Papa}}」源自[[希腊文|希腊文]]的「{{lang|el|πάπας}}」，原意為「[[父亲|父亲]]」<ref>{{cite web |url = https://ahdictionary.com/word/search.html?q=pope |title=American Heritage Dictionary of the English Language |publisher=Houghton Mifflin Harcourt |date= |accessdate=2017-03-16}}</ref>，起初用于对任何[[司铎|司铎]]或[[主教|主教]]親暱的称呼。4世纪或5世纪时，「{{lang|el|πάππας}}」在[[西罗马帝国|西罗马帝国]]仍频繁地用于称呼主教，尽管它已经开始逐渐转变成现代的用途，即局限于称呼罗马主教，而在[[东罗马帝国|东罗马帝国]]，尤其是在[[希腊|希腊]]和[[俄罗斯|俄罗斯]]，司铎仍然称为「{{lang|el|πάπας}}」。

== 沿革 ==
從《[[宗徒大事錄|宗徒大事錄]]》时期開始，[[使徒|使徒]]們在[[猶太會堂|猶太會堂]]講道，但大部分基督徒聚會是分散在各個家庭中進行，甚至當有迫害時也會在地下墳墓中進行，因此聚會地點乃相當分散，漸漸的這些家庭聚會以大城市為中心形成一個個[[教區|教區]]的概念，在教區裡面他們就設立[[主教|主教]]，主教或[[教父|教父]]這個詞在中文裡面也被翻譯為監督或是[[長老|長老]]，聖經中多次提到使徒召集各地方長老與祂們說話，這些主教在接受使徒的教導後，便將這些教導帶回各教區，這些長老繼承使徒成為各地方教會的宗教領袖。<ref name="初代的教會">蘇文峰，[http://www.smyrnasrci.org/wp-content/uploads/4-Church-History/ChurchHistory_L01_ForPDF.pdf 教會歷史第一課：初代的教會（上）]</ref><ref name="初代教會職務">[http://www.radiovaticana.va/proxy/cinesebig5/churchistory/storiaconcis/1storia13.html 初世紀教會職務的形成與演變]，梵蒂岡廣播電台，天主教會史淺談上十三</ref>

[[File:Tiara.png|180px]]，象征教宗之训诲、圣化、治理三项神权]]

而主教之所以在社會上擁有很大的權力也有其歷史因素。初代教會由於聖經及使徒書信多半由人工抄寫在羊皮上，流傳並不普及，一般信徒難以直接閱讀，加上當時基督徒多數不識字，無法了解教義，因此各種的「異端」教義容易流傳。特別是[[諾斯底主義|諾斯底主義]]的流行使得長老們在試圖消滅這些「異端」的過程中，加強了這些主教的權力。
其次，由於初代幾位具影響力的[[教父|教父]]（也就是教會領袖，特別是初代教會的神學家）十分看重與強調主教制度，無形中提高了主教的地位。例如[[安條克的依納爵|伊格納修]]、[[愛任紐|愛任紐]]、[[居普良|居普良]]等，其中居普良提倡主教制度最為出力，他說，一切的主教都是平等的，背叛主教就是背叛教會<ref name="主教制與傳統">[http://www.youtube.com/watch?v=pwv_tdBvwDg&list=PLAE44332EC8D0D527 教會歷史主教制與傳統蔡麗貞]</ref>。到了國教時期，羅馬帝國形成[[五大牧首|五個主要主教]]制度，即羅馬主教、[[亚历山大牧首|亞歷山大港主教]]、[[安条克牧首|安提阿主教]]、[[君士坦丁堡普世牧首|君士坦丁堡主教]]、以及[[耶路撒冷牧首|耶路撒冷主教]]，五個主教明爭暗鬥，都希望能成為主教之首，直到[[伊斯蘭教|伊斯蘭教]]興起後，耶路撒冷、安提阿和亞歷山大港教會都因[[穆斯林的征服|穆斯林勢力入侵]]而衰落，剩下羅馬主教和君士坦丁堡主教分別成為西東方教會重要領袖，而羅馬主教在政權分裂的西部地區因數次與入侵蠻族成功的談判，其社會重要性更遠超過政權，成為西部有政教實權的人物<ref name="宣教歷史">莊祖鯤，[http://www.youtube.com/watch?v=vjBvxx1dzVs&list=PLsNJO0nl_IhRrKGSFdDq_SCWmKvqWaBpR 宣教歷史六：中古世紀論]</ref>。

羅馬主教權高於其他地區主教，有以下八個原因：
*羅馬是羅馬帝國的首都
*相傳使徒保祿與伯多祿都是在羅馬殉道，因此羅馬的宗教地位就更高
*羅馬城曾在[[尼祿|尼祿]]時期被放火燒城，作為屠殺基督徒的藉口。[[保罗_(使徒)|保祿]]與[[伯多祿|伯多祿]]在此次迫害中殉道，羅馬城受到帝國最大的迫害，因此受到基督徒欽佩
*羅馬教會曾慷慨救濟災荒，特別是[[耶路撒冷|耶路撒冷]]饑荒
*330年，罗马帝國首都遷到[[拜占庭|拜占庭]]後，[[羅馬|羅馬]][[主教|主教]]就成了西方教會的領袖
*公元4-5世紀，當[[東方教會|東方教會]]在爭論神學問題時常會找羅馬[[主教|主教]]來主持公道，因為相較於東方教會受希臘文化影響，傾向神祕主義，較為感性，羅馬所代表的西方教會受[[羅馬法|羅馬法]]的影響，重視用邏輯推理，嚴謹的方式解讀聖經，所以東方教會常找西方教會的羅馬主教來做判定
*羅馬教會人才輩出有許多傑出神學家
*基於對{{Bible|瑪竇福音|16|18-19}}的解釋，他們認為[[聖伯多祿|聖伯多祿]]是羅馬教會的第一任主教，所以羅馬主教繼聖伯多祿之後擁有主教的正統。<ref name="主教制與傳統" />

在整個[[中世紀|中世紀]]，由於《[[聖經|聖經]]》的不普及，對《聖經》的解釋都由各地方主教擔任，主教的權威甚至高於《聖經》的權威，同時基督教信仰也在這時期控制整個[[西方世界|西方世界]]，羅馬教會有了教廷的稱呼，羅馬主教也有了教宗的稱呼，其影響力及影響範圍甚至超過政權，政權的執政者也都受教宗的加冕，教宗可以從各地教會獲得捐稅，掌握神職人員的任用以及教會司法權。

一直到13世紀下半叶，英法兩國的皇室貴族，因自身利益不願受外來勢力影響，無法容忍羅馬教會干涉各國政治，公元14-15世紀，英、法在神職人員任用及教會司法權上便歸到[[君主|君主]]手上，但仍向羅馬教廷納稅。中古時期，由於神職人員的腐敗，教廷勢力漸漸低落，公元1300年左右，義大利和[[法國|法國]]彼此爭奪對教會的影響力，教廷一度遷移到當時為教宗領地但臨近法國的[[亞威農|亞威農]]（現今為法國領地），形成七任教宗都在亞維農治事的[[亞維農教廷|亞維農時期]]。之後，甚至出現同時有兩個教廷與兩位教宗的局面，公元1409年，巴黎大學的教授們提議召開比薩會議，會議中決議將兩位教宗都撤職，另立一位新教宗，然而此次會議法義的兩位教宗都沒有出席，各國仍自擁各自認定的教宗，因此比薩會議後形成了有三個教宗的局面。一直到公元1415年的[[康士坦斯大公會議|康士坦斯大公會議]]，才正式取消了三個教宗的職位，消除了教會的分裂，也把教宗的權力限制在立法團體——即大公會議，但仍無法解決教廷腐敗的問題。在公元1517年，在[[馬丁路德|馬丁路德]]的[[宗教改革|宗教改革]]中，因為[[馬丁路德|馬丁路德]]在數次與大公教會的辯論中，始終堅持聖經權威高過主教權威的論點<ref name="馬丁路德（2）">[http://www.youtube.com/watch?v=0bYsUbwLdTQ&list=PLAE44332EC8D0D527 教會歷史馬丁路德（2）蔡麗貞]</ref>，持此觀點的[[新教|新教]]便由大公教會分裂出來，新教也稱為改革宗、更正教、抗議宗或廣泛稱為基督教會。原來的大公教會，一般也稱為天主教，便仍保存原來的教廷與教宗制度，而新教由於馬丁路德人人皆祭司的觀點，強調每個個人都可以藉由讀聖經、禱告直接明白上帝的旨意，不需透過主教或神職人員做為媒介<ref name="馬丁路德（2）" />，因此產生了非主教制的教會體系，但依然有維持主教制的新教教會，如[[聖公宗|聖公宗]]。

== 頭銜 ==
教宗的完整頭銜是：'''羅馬主教、基督之代表、宗徒長之繼承人、普世教會最高教長、意大利首席主教、羅馬教省總主教及都主教、梵蒂岡城邦元首及天主眾僕之僕'''（{{lang-la|Episcopus Romanus, Vicarius Christi, Successor principis apostolorum, Pontifex Maximus, Primatus Italiae, Archiepiscopus ac metropolitanus provinciae ecclesiasticae Romanae, Princeps sui iuris civitatis Vaticanae, Servus Servorum Dei}}），但很少使用。天主教會認為教宗為初任羅馬主教，即[[十二門徒|十二門徒]]之一[[伯多祿|伯多祿]]的後繼，因此一開始稱呼他為「伯多祿之代表」，直到11世紀教宗革命（又称[[格里高利改革|格里高利改革]]）時才改稱為目前的「基督之代表」。

2006年《宗座年鑑》刪除了「西方[[宗主教|宗主教]]」（{{lang|la|Patriarcha Occidentis}}）的稱呼。它的上一次修訂在1969年，當時教宗[[保祿六世|保祿六世]]把「天主眾僕之僕」加於名號之中，並刪除了「光榮在位」的語句<ref name="公教報">[http://kkp.catholic.org.hk/gl/gl3243.htm 教宗除西方宗主教名號 教廷稱有利基督徒合一]，[[公教報|公教報]]，第3243期，2006年4月16日</ref>。

由於教宗在[[中世紀|中世紀]]時擁有超乎西欧世俗君王的精神地位，曾被[[明朝天主教|明末天主教]]传教士译为“'''教化皇'''”<ref>張春申，[http://archive.hsscol.org.hk/Archive/periodical/ct/CT055/CT055B.htm 〈利瑪竇對我國教會的啟示： 文化工作的教會幅度〉]《神學論集》55期，（[[台北|台北]]：[[輔仁大學|輔仁大學]]，1983）pp. 37-42</ref>，继而影响周边使用[[汉字|汉字]]的国家。
另外在[[西方世界|西方世界]]，教宗有時也會被敬稱為「聖父」（{{lang|en|Holy Father}}），其意為「神聖的父親」，與[[三位一體|三位一體]]所指的[[聖父|聖父]]不同<ref>Michael，[http://www.catholicworld.info/sameroot/C.htm#C2 「聖父」和「神父」之名不對嗎？]同根生，2004年1月16日</ref>。

== 職權 ==
教宗被[[天主教會|天主教會]]認為是[[耶穌|耶穌]]在當今[[世界|世界]]的代表，其主要任務即是領導耶穌建立的[[教會|教會]]，並將信仰[[傳福音|傳佈]]到世界各個角落。教宗的{{link-en|教務職權|Ecclesiastical jurisdiction}}被稱為「[[聖座|聖座]]」，是以教宗為中心構成的教務領導團體，除了負責管理羅馬的天主教會，還具有代表與領導整個天主教會的功能，現今亦為[[國際法|國際法]]承認的[[主權|主權]]實體，可與世界各國締結[[外交關係|外交關係]]。

除了[[宗教|宗教]]职务之外，教宗也是[[梵蒂冈|梵蒂岡城國]]的[[国家元首|国家元首]]，該國由[[聖座|聖座]]管治。梵蒂冈整个被[[意大利|意大利]]首都[[罗马|罗马]]包围，在1870年之前，教宗统治的区域一度拓展到整个意大利中部，其[[领土|领土]]——即[[教廷国|教廷国]]，被称为「圣伯多祿的遗产」。尽管授予教宗对这些[[领土|领土]]的统辖权的文件，也就是所谓的《[[君士坦丁御赐教产谕|君士坦丁御赐教产谕]]》，在15世纪被证明只是[[偽造文書|偽造文書]]，但教宗对教廷国的统治还是延续至1870年[[意大利统一|意大利统一]]，而[[意大利王國|意大利王國]]和[[羅馬教會|羅馬教會]]之间最终的政治协议一直到1929年的《[[拉特兰条约|拉特兰条约]]》才得以实现。1948年通過的《[[意大利共和國|意大利共和國]]憲法》，再次確定該條約的合法性和有效性。

== 任職方式 ==
{{Main|教宗选举}}

教宗是由[[樞機|樞機]]組成的[[樞機團|樞機團]]召開[[教宗选举|秘密會議]]互選產生的，一旦當選就是終身職，但中途可[[教宗辭職|辭職]]。前任教宗逝世或辭職後，至繼任教宗選出前的職位空缺期，被稱為「[[宗座從缺|宗座從缺]]」。

現今的[[教宗选举|教宗选举]]制度是在1268年—1271年的三年空位期之後，於1274年由{{link-en|里昂第二次大公會議|Second Council of Lyon}}採用的，包括樞機團在選舉時必須與外界隔離的規定；而教宗選舉在1492年起固定於梵蒂岡的[[西斯廷禮拜堂|西斯廷禮拜堂]]舉行迄今。

== 近況 ==
[[若望·保禄二世|若望·保禄二世]]于1978年58岁时被选为教宗，也是自[[哈德良六世|哈德良六世]]（1522年—1523年間在位）以来首位被选为教宗的非[[意大利|意大利]]人。若望·保禄二世於2005年4月2日逝世後，於4月2日到18日期間產生职位[[宗座從缺|宗座空缺]]，4月18日开始的[[教宗选举|教宗选举]]，用了不足1天即選出[[德国|德国]]籍[[枢机|枢机]][[约瑟夫·拉青格|约瑟夫·拉青格]]為新任教宗，聖號為[[本篤十六世|本篤十六世]]。

[[聖座|聖座]]发言人于2013年2月11日宣布，時年85岁的教宗[[本笃十六世|本笃十六世]] （Pope Benedict XVI）因為健康因素，在2月28日晚上8点（歐洲中部時間）正式退位，成为西元1415年以来，首位[[教宗辭職|主动请辞的教宗]]。<ref>[http://www.voachinese.com/content/pope-20130211/1601089.html 教皇本笃十六世即将退位"]</ref>

2013年3月13日，[[梵蒂岡|梵蒂岡]]宣佈[[阿根廷|阿根廷]]的[[枢机|枢机]]豪赫·馬利歐·貝果格利歐當選教宗，[[尊號|尊號]]為[[方濟各_(教宗)|方濟各]] （Pope Francis）。他是[[額我略三世|額我略三世]]以後一千二百年來，第一位不是[[歐洲|歐洲]]出身的教宗；亦是[[基督教史|天主教史]]上第一位來自[[美洲|美洲]]與[[南半球|南半球]]的教宗，以及首位[[耶穌會|耶穌會]]出身的教宗。
