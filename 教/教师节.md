'''教師節'''广义上讲是一種感謝[[教師|老師]]教導的[[節日|節日]]，旨在肯定教師为教育事業所做的贡献與努力。不同國家和组织制定的「教師節」時間有所不同。

== 世界教師日 ==
主條目：[[世界教师日|世界教师日]]

1966年10月5日[[聯合國教科文組織|聯合國教科文組織]]與[[國際勞工組織|國際勞工組織]]聯合主辦了關於教師地位的政府間特別會議並通過了《關於教師地位的建議書》。<ref name=":0">{{Cite journal|title=復蘇始於教師——世界教師日來源及其2010年聯合致辭|author=熊建輝|url=http://www.wei.moe.edu.cn|journal=世界教育信息|issue=2011.11|doi=|others=|year=2011|volume=1|page=14-15|pmid=}}</ref>

1994年兩組織共同作出決定，把每年的10月5日定為世界教師日。<ref name=":0" />
==各地教師節==
===亞洲===
====中華民國====
{{Wikisource|纪念日及节日实施办法}}

教師節最早是[[中华民国大陆时期|中华民国大陆时期]]由[[邰爽秋|邰爽秋]]、[[程其保|程其保]]在[[1931年|1931年]]於[[國立中央大學_(南京)|中央大學]]提出，其發表要求“改善教師待遇，保障教師工作和增進教師修養”的宣言，發表要求並議定6月6日為教師節，也稱雙六節。該教師節沒有被政府所承認<ref>{{Cite web |url=http://www.gsedu.cn/qunyg/zhonggjjj/index.html |title=中國教師節 |accessdate=2017-09-10 |author=甘肅省教育廳師範教育處 |date=2015. |publisher=甘肅省教育廳師範教育處 |archive-url=https://web.archive.org/web/20150323012044/http://www.gsedu.cn/qunyg/zhonggjjj/index.html |archive-date=2015-03-23 |dead-url=yes }}</ref>。

1939年，中华民国教育部設立農曆8月27日為教師節，並頒布了《教師節紀念暫時辦法》。但由於戰爭原因並未推行。<ref>{{Cite web |url = https://mp.weixin.qq.com/s?__biz=MjM5NDg5MjExMQ==&mid=2650756896&idx=1&sn=9a574861cbb2aecc832e4eb65c77d83d |title=檔案君 教師節的來歷|accessdate=2017-09-10|author=檔案君 |date=2017-09-09| website = 中國青年網 }}</ref>

中華民國政府迁台后，于[[1952年|1952年]]頒訂至聖先師誕辰及教師節改為[[9月28日|9月28日]]直至今日，該日又稱「[[孔子誕辰日|孔子誕辰紀念日]]」{{NoteTag|[[1931年|1931年]]發起建立時日期原為[[6月6日|6月6日]]，[[1932年|1932年]]被[[政府|政府]]教育部確定為全國教師節；[[1939年|1939年]][[教育部|教育部]]改定[[孔子誕辰日|孔子誕辰日]][[八月廿七|八月廿七]]為教師節。後經[[陰曆|曆數]]及考紀專家將孔子誕辰換算為陽曆應為[[9月28日|9月28日]]，頒行至今。}}。每年各地[[孔廟|孔廟]]都會舉行隆重的[[釋奠佾舞|釋奠典禮]]。在教師節周間，全台各級教育行政單位均會舉辦「慶祝教師節大會」表揚資深及優良教師及職員，每年的「[[師鐸獎|師鐸獎]]」、「教育奉獻獎」亦在此時頒發。<ref name="各級教育行政機關學校教育團體慶祝孔子誕辰及教師節辦法">《[http://law.moj.gov.tw/LawClass/LawHistory.aspx?PCode=H0150008 各級教育行政機關學校教育團體慶祝孔子誕辰及教師節辦法]》，本辦法乃[[中華民國教育部|中華民國教育部]]於1952年9月2日訂定發布，至1997年1月15日公佈廢止。共有六條條文。其宗旨如第1條：「各級教育行政機關各級學校及教育團體應舉行慶祝會，以樹立尊師重道風氣。」</ref>

==== 中華人民共和國 ====
[[中華人民共和國|中華人民共和國]]成立後，教師節定為每年的6月6日，於[[1951年|1951年]]，中國共產黨本著「教師是工人隊伍一員的原則」改為與國際勞動節一天的5月1日，1957年之後由於文革的影響，教師的地位極大下降。教師節名存實亡。<ref>{{Cite web|url=http://www.sohu.com/a/190964381_613209|title=除了放假，原來教師節還有那麼多故事……|accessdate=2017-09-10|date=2017-09-10|work=www.sohu.com}}</ref>

1982年4月，[[張承先|張承先]]和[[方明|方明]]共同提交《關於恢復「教師節」的請示報告》於中共中央，建議以[[馬克思|馬克思]]誕辰日5月5日為教師節。<ref>{{Cite web|url=http://news.china.com.cn/rollnews/education/live/2014-09/10/content_28668888.htm|title=教師節設立內幕:習仲勳曾圈閱"恢復教師節的請示"_中國網教育 中國網|accessdate=2017-09-10|author=|date=|work=news.china.com.cn|publisher=}}</ref>

[[1985年|1985年]]時任總理[[趙紫陽|趙紫陽]]提出建立教師節的議案，經過人民代表大會通過，定為秋季新學年之初的[[9月10日|9月10日]]<ref>[http://news.xinhuanet.com/misc/2008-09/05/content_9801822.htm 新華網：教師節的由來]</ref>。該教師節沒有法定假期<ref>{{Cite web|url=http://www.gov.cn/zwgk/2013-12/11/content_2546227.htm|title=國務院關於修改《全國年節及紀念日放假辦法》決定|accessdate=2017-09-10|work=www.gov.cn}}</ref>，但是部分學校，特別是小學在教師節會放假休息半天至一天不等。每一年的教師節，中國教育部都會設定一個主題，如2018年的主題為「弘扬高尚师德，潜心立德树人」<ref>{{Cite web|url=http://www.moe.gov.cn/jyb_xwfb/xw_fbh/moe_2069/xwfbh_2018n/xwfb_20180831/sfcl/201808/t20180831_346712.html|title=2018年教师节主题及主要活动安排有关情况介绍|author=教育部教师工作司 |work=中华人民共和国教育部}}</ref>。

有部分人士倡議將[[孔子|孔子]]誕辰日作為教師節。<ref name="新华网">{{cite news|url=http://www.edu.cn/20060306/3176621.shtml|title=政协委员建议教师节应改在孔子诞辰9月28日|author=记者：邬焕庆、周玮|date=2006年3月6日|publisher=中国教育可科研计算机网|quote=部分全国政协委员近日向国家有关部门建议：应该把我国教师节从每年的9月10日改在孔子诞辰纪念日，即公历9月28日。|access-date=2009年9月6日|archive-url=https://web.archive.org/web/20090922005708/http://www.edu.cn/20060306/3176621.shtml|archive-date=2009年9月22日|dead-url=yes}}</ref>2013年，国务院法制辦公佈的《教育法律一揽子修订草案（征求意见稿）》中更提到拟规定“每年9月28日为教师节”，是官方首次对教师节改在孔子诞辰纪念日的正面回应。<ref>[http://www.mnw.cn/quanzhou/edu/792716.html 网传教师节改期孔子诞辰日 建议：解决教师现实问题是关键]</ref>

====={{HKG}}=====
香港於[[1997年|1997年]][[香港回歸|主權移交]]前，教師節原依照[[中華民國政府|中華民國政府]]的習慣定於[[孔子誕辰|孔子誕辰]]，即每年的[[9月28日|9月28日]]；主權移交後則隨[[中華人民共和國|中華人民共和國]]改為每年的[[9月10日|9月10日]]。<ref name="香港教师节">{{cite web|url=http://www.eol.cn/zong_he_3381/20130910/t20130910_1014504.shtml|title=教师节香港兴送卡台湾兴祭孔|accessdate=2013-09-10|year=2013|work=中国教育网|publisher=中国教育网|language=中文}}</ref>

====={{MAC}}=====
澳门的教師節跟隨[[中華人民共和國|中華人民共和國]]為每年的[[9月10日|9月10日]]。<ref>{{cite news|url=http://www.chinanews.com/ga/kjww/news/2008/02-01/1153515.shtml|title=澳门拟将教师节列法定假 教学人员制1日起咨询|date=2008年2月1日|publisher=中国新闻网}}</ref>

==== {{KOR}} ====
韩国教師節是[[5月15日|5月15日]]，為首創[[諺文|諺文]]的[[世宗大王|世宗大王]]陽曆生日。在這天，學生會送[[康乃馨|康乃馨]]給老師，並和學生一起度過歡愉的一天。

==== {{蒙古}} ====
蒙古国教师节为[[2月|2月]]第一个[[周末|周末]]。

==== {{SIN}} ====
新加坡教师节为9月的第一个星期五。这天[[新加坡|新加坡]]所有的学校放假一天。同时，一般在教师节前一天在学校庆祝。

==== {{MAL}} ====
马来西亚教师节為[[5月16日|5月16日]]。但華文獨立中學則跟隨[[中華民國|中華民國]]定於每年的[[9月28日|9月28日]]慶祝教師節。因各華文獨立中學的行政運作有別，訂定教師節的日期均有不同，並非所有華文獨立中學都以9月28日定為教師節，且大多數華文獨中于5月份或6月份左右舉行各校教師節慶祝活動，以紀念各校教師的付出與辛勞。

==== {{VNM}} ====
越南教師節是每年[[11月20日|11月20日]]。這日[[越南|越南]]的所有學校都放一天假，學生向老師獻花。

==== {{IND}} ====
印度教師節是[[9月5日|9月5日]]，同時還紀念印度第二位總統[[萨瓦帕利·拉达克里希南|萨瓦帕利·拉达克里希南]]爵士，他也是一位[[教育家|教育家]]。在傳統上，印度教師節的這天，學校教書的工作是交給較高年級的學生負責，讓老師能夠休假。

==== {{THA}} ====
泰國教師節在[[1月16日|1月16日]]。

==== {{阿富汗}} ====
阿富汗的教师节在每年10月。这天学校会放假，学生和老师相聚在学校庆祝这一天。学生会准备传统食品，点心，音乐和给老师的礼物。

===歐洲===

==== {{ALB}} ====
[[阿尔巴尼亚|阿尔巴尼亚]]教師節是每年的[[3月7日|3月7日]]，正好在[[母親節|母親節]]的前一天。在教師節這天，阿爾巴尼亞放假一天。

==== {{CZE}} ====
捷克教師節是一個假日，定於[[3月28日|3月28日]]，這一天是[[約翰·阿摩司·夸美紐斯|約翰·阿摩司·夸美紐斯]]的生日。學童會在教師節這天送花給他們的老師。

==== {{RUS}} ====
俄罗斯教師節是10月5日，不过在1965年到1993年间则把教师节定为10月第一个星期日，直到1994年改为10月5日。按照傳統，這一天許多中學、職業院校及高等院校的學生將以各種形式向老師祝賀節日：贈送鮮花、組織晚會、表演戲劇等等。此外俄羅斯主要的電視頻道還將播放獻給老師的電影和節日晚會。

===={{HUN}} ====
匈牙利教师节是6月第一个星期日。

==== {{罗马尼亚}} ====
罗马尼亚的教师节是6月5日。

==== {{ESP}} ====
西班牙的教师节是11月27日。

==== {{POR}} ====
葡萄牙的教师节是10月5日。

===大洋洲===
===={{AUS}}====
澳大利亚教师节是10月的最后一个星期五。

=== 北美 ===

==== {{USA}} ====
美国教師節是5月的第1個完整周（始自周日）的星期二，是個放假的節日。
* [[麻薩諸塞州|麻薩諸塞州]]也曾採用[[9月11日|9月11日]]為教師節。

=== 拉丁美洲 ===
拉美教師節一般為[[9月11日|9月11日]]。

==== {{PAN}} ====
這個節日是在[[1943年|1943年]][[巴拿馬|巴拿馬]]所舉行的[[泛美教育會議|泛美教育會議]]上所制定的。

==== {{ARG}} ====
這天也是[[阿根廷|阿根廷]]教育家[[多明戈·福斯蒂諾·薩米恩托|多明戈·福斯蒂諾·薩米恩托]]的逝世紀念日。


許多[[拉丁美洲|拉丁美洲]]國家也會根據自己的國家的歷史，設立教師節。
==== {{BRA}} ====
在巴西，教師節是每年的[[10月15日|10月15日]]。

==== {{MEX}} ====
墨西哥的教師節始於[[1917年|1917年]]的9月，由國家議會頒布[[5月15日|5月15日]]為教師節，並於[[1918年|1918年]]首次慶祝。

==== {{GUA}} ====
瓜地馬拉的教師節為每年的[[6月25日|6月25日]]，紀念一位名为玛利亚·金吉拉（Maria Chinchilla）的老師於對抗政府的衝突中逝世。

==== {{HON}} ====
宏都拉斯的教師節為每年的[[9月17日|9月17日]]，紀念何塞·特立尼达·雷耶斯（José Trinidad Reyes）。

==== {{SLV}} ====
薩爾瓦多的教師節為每年的[[6月22日|6月22日]]。

==== {{CRC}} ====
哥斯达黎加的教师节为每年的[[11月22日|11月22日]]，纪念該国最伟大的教育家莫洛·费尔南迪斯·阿库尼亚（[[Mauro_Fernández_Acuña|Mauro Fernández Acuña]]）（11/22/1843）。


除此之外，全球有100多個國家直接使用世界教師日作為教師節。
