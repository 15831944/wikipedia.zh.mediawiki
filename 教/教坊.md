'''教坊'''是由[[唐|唐]]到[[清|清]]，管理[[宮廷|宮廷]]中演出[[音樂|音樂]]、[[舞蹈|舞蹈]]及[[戲劇|戲劇]]的組織。由於教坊中的女藝人為[[舞優|舞優]]，也被稱作[[宮妓|宮妓]]，是廣義的[[妓女|妓女]]中的一種，因此到宋元以後，民間的[[妓院|妓院]]，尤其是[[女樂|女樂]]演出的場所，有時亦被稱作教坊。

== 唐代的教坊 ==

教坊最早在[[唐高祖|唐高祖]][[武德|武德]]年間（618年－626年）出現，是為'''內教坊'''，歸屬於[[太常寺|太常寺]]管理。[[武后|武后]]年間（684年－704年）一度改稱'''雲韶府'''。[[唐中宗|唐中宗]]時（705年）又恢復舊稱，到開元二年（714年）又增設外教坊四處，二在[[洛陽|洛陽]]，二在[[長安|長安]]，並使教坊直接由宮廷派教坊使、教坊副使領導，而不受太常寺管理。

唐代的教坊的業務為管理負責在宮庭中演出[[歌舞|歌舞]]、[[散樂|散樂]]等的男女藝人。這些藝人依其地位及技藝高低分成不同等級。如：一般的女藝人稱「官人」，較高級女藝人的稱作「內人」或「前頭人」，平民人家進入的女性要學習各種樂器包括[[琵琶|琵琶]]、[[箏|箏]]、[[箜篌|箜篌]]等樂器，被稱作[[搊彈家|搊彈家]]。

相較於[[太樂署|太樂署]]、[[鼓吹署|鼓吹署]]等組織以演奏儀式用的[[雅樂|雅樂]]為主，但也演奏[[燕樂|燕樂]]，教坊所所演奏的是較出自民間的娛樂性音樂。

== 宋代的教坊 ==

[[北宋|北宋]]初年於960年即在[[汴京|汴京]]設置了教坊，分為四部，之後宋平定南方各地，又在各地招來許多新的樂工，[[靖康|靖康]]二年（1127年），[[金|金]]人攻陷汴京，造成樂器和樂書的散失和教坊的結束。
[[南宋|南宋]]初由於在戰亂中，未恢復教坊組織，但[[紹興|紹興]]14年（1144年），才再成立，其後紹興31年（1161年），金人再度侵宋，教坊再廢。之後雖有類似名稱的單位掌音樂事務，但已沒有樂工，而是需演出時臨時招募人員排練演出，這也反映了當時民間藝人的興盛。

宋代的教坊的作用與唐類似，以演演宴樂之用的殕樂為主。在北宋時，分成[[大曲部|大曲部]]、[[法曲部|法曲部]]、[[龜茲部|龜茲部]]、[[鼓笛部|鼓笛部]]四部，分掌不同樂種的教習。至南宋時，更進一步按樂器及表演種類分成十三部色，包括[[篳篥|篳篥]]部、[[大鼓|大鼓]]部、[[杖鼓|杖鼓]]部、[[拍版|拍版]]色、[[笛|笛]]色、[[琵琶|琵琶]]色、[[箏|箏]]色、[[方響|方響]]色、[[笙|笙]]色、[[舞旋|舞旋]]色、歌板色、[[雜劇|雜劇]]色、[[參軍|參軍]]色，各設部頭或色長。

== 遼金元明清的教坊 ==

[[遼|遼]]、[[金|金]]亦仿漢人之制設立教坊，以供宴樂演出之用。元代則在[[大都|大都]]設有教坊司，掌管及訓練藝人演出音樂及戲劇。明代則在[[洪武|洪武]]年間（1358年-1398年），在[[南京|南京]]設立教坊司，隸屬於[[禮部|禮部]]，主管乐舞和戏曲。清初亦沿明代設教坊司，[[順治|順治]]十六年（1659年），廢除教坊司中的女樂，而以男性內監代替。至[[雍正|雍正]]七年（1729年），將教坊司改為[[和聲署|和聲署]]。

==朝鮮的教坊==
朝鮮半島於[[高麗|高麗]]開始出現[[妓生|妓生]]，訓練妓生的場所也稱為教坊，也是她們奉客表演的地方。
== 教坊對中國音樂的影響 ==

教坊尤其在唐宋時期，作為國家力量支持的藝術組織，吸收並訓練了大批的藝人，一方面將民間音樂帶入宮庭，另一方面也對於藝人藝術水準的提昇有所貢獻，而在朝代衰落時，常有不少藝人又從教坊流入民間，也將原本宮庭的音樂傳入民間，促成民間音樂的進一步發展。
