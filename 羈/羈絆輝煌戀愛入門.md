==故事==

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|三城刀輝|三城 刀輝（さんじょう とうき）}}
:本作的男主角，叢雲學園技巧科2年級學生。刀匠名门三城家的后代，折纸锻造有刀匠级水平，但是至今没有打造过一把折纸，只是经常捣鼓机械，享受着悠哉的学园生活。性格开朗，处事老练。无法放下困扰的人不管，老好人一个，喜欢关心人，照顾他人得心应手，
; {{nihongo|上和泉櫻夜|上和泉 桜夜（かみいずみ さや）|聲：{{lang|ja|蓬かすみ}}}}
:身高：160cm，三圍：B91（D）/W58/H89，生日：1月16日，血型：A型，恋爱：贤妻良母
:叢雲學園武藝科2年級學生。群马出身，在奶奶要求下转入丛云学园。拥有着速度型选手的天赋，在刀上灌注了信念和觉悟。一开始并不适应刃道，在刀辉指导下成长，在与紫苑的模拟战中领悟了自己的刃道，击败了她获得了第一次刃道比赛的胜利。
; {{nihongo|朱雀院椿|朱雀院 椿（すざくいん つばき）|聲：[[卯衣|卯衣]]}}
:身高：155cm，三圍：B81（B）/W54/H82，生日：12月10日，血型：A型，恋爱：年上妻子
:叢雲學園武藝科3年級學生。丛云演武祭二连冠，学园最强的武艺者。自称刀辉的姐姐，十分关心刀辉。认为学生时代的男女交往是不纯洁的，对H的事情没有抵抗力，经常面红耳赤。被“朱雀院”之名束缚着，一心只考虑着“胜利”，总是在学园训练室锻炼自己。对希娅对自己刃道战斗“无聊”的评价很气愤，和樱夜因刀辉做过一场，对紫苑这个后辈很满意，经常与千石葵针锋相对。
; {{nihongo|藍原紫苑|藍原 しおん（あいはら）|聲：藤崎紗矢香}}
:身高：146cm，三圍：B85（E）/W53/H83，生日：8月9日，血型：O型，恋爱：爱管闲事
:叢雲學園武藝科1年級學生。因传家灵刀邂逅刀辉，在他的教导下找到了方向。精神感应系数极高，因此经常损坏折纸，经常被技巧科学生说教。有成为力量型选手的潜质。
; {{nihongo|弗莉希娅·神速|フリージア・ゴッドスピード|聲：{{lang|ja|ヒマリ}}}}
:身高：143cm，三圍：B79（A）/W51/H81，生日：10月8日，血型：AB型，恋爱：好奇宝宝
:叢雲學園武藝科1年級學生，國外槍器製造商「神速」的大小姐。被刀辉极高的折纸锻造技术和丰富的刃道知识折服，立志成为他的弟子。

=== 其他角色 ===
; {{nihongo|绫濑|綾瀬（あやせ）|聲：[[小倉結衣|小倉結衣]]}}
:寄宿在鐵鏡的女神。三城家的守护神，喜食清汤薯片。
; {{nihongo|千石葵|千石 葵（せんごくあおい ）|聲：麻黒ほん}}
:叢雲學園武藝科3年級學生。两次在演武祭上输给椿。每次看到椿时都针锋相对，吵架是日常现象。经常在椿面前性骚扰刀辉，以此作乐。
; {{nihongo|银杏田萌生菜|銀杏田 萌生菜（いちょうだ めいな） |聲：綾瀬あかり}}
:叢雲學園武藝科2年級學生。紫苑在道场时的前辈。曾和刀辉组队参加丛云演武祭，因此在班里和他熟识。被喊绰号时会大发雷霆。
; {{nihongo|朱雀院都子|朱雀院 都子（すざくいん みやこ）|聲：野々村纱夜}}
:朱雀院家的三女兒，四姐妹中刃道实力No.1。侍奉弗莉希娅的女仆，告诉她刃道的元凶。
; {{nihongo|佐佐木巫琴|佐々木 巫琴（ささき みこと）|聲：野上結生}}
:刀輝的班級老師兼校医。半妖。空闲时喜欢抽电子烟来消磨时光。总是懒洋洋的样子，讨厌惹上麻烦。不少学生喜欢这位散发着沉着气氛的成熟教师。
; {{nihongo|东雲雲雀|東雲 ひばり（しののめ ひばり） |聲：橘美月}}
:刀輝的同學。丛云学园后勤科2年纪学生。喜欢百合，偶尔会冒出关西口音。
; {{nihongo|凭代君|ヨリシロくん|聲：七々瀬輪}}
:電子模擬系統的虛擬角色，正式名為「平賀電算靈式試行機YSR-04E」。运用了全息投影技术打造。具有显示情报确认窗口的功能，还可以投影出灵式武器和对战选手，来进行比试。使用凭代君进行模拟战之后，对战数据会自动上传云端来供学园所有人阅览。经常使用式神的虚拟形象，也能随时变换为已登录的对战选手的身姿。
