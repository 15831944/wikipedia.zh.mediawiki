'''旗津區'''是[[中華民國|中華民國]][[高雄市|高雄市]]的一個[[市轄區|市轄區]]，位於市區西南部的'''旗津島'''上，與[[鼓山區|鼓山區]]、[[前鎮區|前鎮區]]及[[小港區|小港區]]隔[[高雄港|高雄港]]相望。位於[[南海|南海]]的[[東沙群島|東沙群島]]及[[南沙群島|南沙群島]]的[[太平島|太平島]]及[[中洲礁|中洲島]]亦劃歸該區轄管。

原本旗津島是一個[[沙洲|沙洲]]半島，南部與台灣本島相連，1967年因高雄港第二港口的興建而被截斷<ref>{{cite web |url=http://cijin.kcg.gov.tw/history.php |title=存档副本 |accessdate=2013-01-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20111014053553/http://cijin.kcg.gov.tw/history.php |archivedate=2011-10-14 }}</ref>。連接旗津和前鎮的[[高雄港過港隧道|高雄港過港隧道]]是臺灣唯一的海底隧道，以[[沉管式隧道|沉箱式]]工法興建，1984年完工通車。同樣位於旗津的[[中洲污水處理廠|中洲污水處理廠]]是高雄市第一座[[污水處理廠|污水處理廠]]，1987年完工啟用，每天可以處理約75萬公噸的污水，仍是高雄市容量最大的污水處理廠。

== 歷史 ==
旗津地區在[[台灣清治時期|台灣清治時期]]至[[台灣日治時期|日治]]，為[[大竹里|大竹里]][[旗後街|旗後街]]、烏松-{庄}-、中洲-{庄}-的轄區。

1673年，[[旗後天后宮|旗後媽祖宮]]建廟。[[台灣清治時期|滿清]]於1862年在此設立旗後海關分關，旗後正式開港通商，萬帆聚集，成為高雄的首善之區

[[臺灣日治時期|臺灣日治時期]]，1898年，[[日本|日本]]在此設立高雄第一所學校－打狗公學校（今旗津國小）。1909年設打狗區役場於旗後，但在1920年設立[[高雄街|高雄街]]後，區役場被遷往[[哨船頭|哨船頭]]。1925年，[[日本|日本]]取「旗鼓堂皇，津粱永固」之義，改地名為「旗津」。

[[二次大戰|二次大戰]]後，[[國民政府|國民政府]]合併[[旗後町|旗後町]]、[[平和町|平和町]]、[[綠町_(高雄市)|綠町]]，改設高雄市旗津區。

旗津島原本是和南端[[小港區|小港區]]相連的[[半島|半島]]，但是由於[[高雄港|高雄港]]第二港口的擴建，旗津和小港之間遂崛以水道分割。1979年高雄市升格為[[直轄市|直轄市]]，並將[[東沙島|東沙島]]及南沙[[太平島|太平島]]劃歸旗津區代管。1984年5月，[[高雄港過港隧道|高雄港過港隧道]]通車，使旗津的對外交通大為改善。

== 宗教信仰文化 ==
[[File:Chiang_Kai-shek_Temple.JPG|thumb]]
===佛教、道教和臺灣民間信仰===
*[[旗津天后宮|旗津天后宮]]：高雄第一間媽祖廟，亦是旗津當地的重要信仰，主要供奉[[媽祖|媽祖]]，是漁民徐阿華以及其他漁民所建
*[[旗津臨水宮|旗津臨水宮]]：旗津當地的重要信仰之一，主要供奉[[臨水夫人|臨水夫人]]
*[[北極紫菱殿|北極紫菱殿]]：是復興里的重要信仰，主要供奉[[北極玄天上帝|北極玄天上帝]]
*[[旗津廣濟宮|廣濟宮]]：是旗津中洲里的重要信仰
*[[紫竹寺|紫竹寺]]：主要供奉[[觀音菩薩|觀音菩薩]]
*[[福壽宮|福壽宮]]：是旗津區中華里一帶的重要信仰，主要供奉[[五府千歲|五府千歲]]
*[[蔣公感恩堂|蔣公感恩堂]]：是北汕里、實踐里（[[大陳新村|大陳新村]]）的重要信仰。內供奉[[蔣介石|蔣介石]]、[[觀音佛祖|觀音佛祖]]、[[三官大帝|三官大帝]]
*[[蔣公報恩觀|蔣公報恩觀]]：位於實踐里，內有蔣公銅像供人景仰，以及[[三官大帝|三官大帝]]、[[張志和|漁師大神]]、[[阮弼真君|阮弼真君]]、[[孚佑帝君|孚佑帝君]]、[[福德正神|福德正神]]
*[[旗津鳳山寺|旗津鳳山寺]]：是旗津區中興里（舊七柱）居民重要信仰，主要供奉[[廣澤尊王|廣澤尊王]]
*[[福興宮_(高雄市)|福興宮]]：位於中興里，近[[高雄港過港隧道|高雄港過港隧道]]，內部主要供奉[[土地公|土地公]]

===基督宗教===
*[[台灣基督長老教會|台灣基督長老教會]]旗後教會：台灣首間[[基督新教|基督新教]][[長老宗|長老宗]]禮拜堂
*[[旗津海星聖母堂|旗津海星聖母堂]]：當地唯一的[[天主教|天主教]]堂

== 旗津漁港 ==
[[File:Cijin_Fishing_Port_in_Kaohsiung,_Taiwan.JPG|thumb]]
旗津漁港位於旗津區中段，鄰高雄港主航道及風車公園，為本市近海漁業中心，海風涼爽宜人，海產豐富新鮮，港邊賞船眺望85大樓，從不同的角度欣賞高雄港灣城市建築的天際線，更令人心情愉悅；海風、海產與賞船實為旗津漁港最大特色。
