'''旗幟'''或者'''旌旗'''泛指懸掛在桿上的布。旗上面通常繪有圖案，具有標誌性，用於傳達訊息或分類。時至今日，旗幟也用於宣傳或純粹的裝飾用。按其形状旗帜分为仿古旗、广告刀旗、锦旗、常规旗帜等。國家、地區、軍队等通常都有自己的旗幟。現代大部分[[國旗|國旗]]或區旗一般是長方形的，也有個別國旗使用[[瑞士国旗|正方形]]或[[尼泊尔国旗|不規則五角形]]。研究旗幟的學問稱做[[旗幟學|旗幟學]]。

== 国旗 ==
[[File:Dannebrog_3.jpg|thumb]]

[[国旗|国旗]]是一个国家的象征与标志，悬挂着的国旗就代表了国家的[[主权|主权]]。国旗经常被政府机构悬挂，但通常也可被本国的公民使用。不同的国旗有着不同的样式，包括不同的长宽比例，甚至不同的形状。
公用和私人建筑物（例如学校和法院）经常悬挂国旗。在某些国家，国旗在民用建筑上只在几个规定的国旗悬挂日悬挂。国旗通常分为陆上旗和海洋旗两种，各有三种类型，但大多数国家在这几种类型上采用相同的设计。

== 陆上用旗 ==
在陆地上，通常分为[[民用旗|民用旗]]，[[政府旗|政府旗]]和[[军旗|军旗]]三种。[[政府旗|政府旗]]通常限制用于政府机关，而[[民用旗|民用旗]]可以被任何人使用。[[军旗|军旗]]通常被陆海空三军所使用。

大多数国家（例如[[美国|美国]]和[[英国|英国]]）三种陆上旗完全相同。而一些国家，特别是在[[拉丁美洲|拉丁美洲]]，政府用旗和民用旗不尽相同。民用旗通常是简化后的[[政府旗|政府旗]]，例如去掉[[国徽|国徽]]等。

一些国家采用和国旗图案不同的旗帜作为军旗。在[[北欧|北欧]]国家的国旗的燕尾版本经常作为军旗和船旗，也有时用作国家元首旗帜。[[菲律宾|菲律宾]]没有军旗，法律上规定在战争状态悬挂时将红色置于上方，而不是平时的蓝色在上方。

== 海上用旗 ==
船隻上使用以辨别所屬國籍的旗幟叫做[[船旗|船旗]]（Ensign）。和陸地上類似的是，船旗也具有三種類型：[[民用旗|民船旗]]，[[政府旗|政府船旗]]和[[軍艦旗|軍艦旗]]。

一些國家的船旗和國旗完全相同。然而另外一些國家，例如英國和日本使用特别的旗幟作為船旗。大多数國家没有更為具體的船旗，但英國例外。[[紅船旗|紅船旗]]作為民用旗，[[白船旗|白船旗]]作為海軍旗，[[藍船旗|藍船旗]]為政府所屬非軍用船隻使用。對於軍艦而言，另需懸掛[[艦首旗|艦首旗]](Jack)，通常與國旗圖案相關。

== 地區性旗幟 ==
大多數國家的行政區都有自己的旗幟（省旗、州旗、郡旗、縣旗、鄉旗、村旗、鎮旗、區旗），上面通常有行政區徽章、名稱。

== 企业旗帜 ==
[[File:House_flags_1900.jpg|thumb]]
[[企业旗帜|企业旗帜]]是企业的象征，代表一个企业。旗面一般是企业的象征颜色（如红色、蓝色、白色和绿色）。旗上面通常有公司名称或者公司标识，悬挂时一般放在企业门口，与国旗同时悬挂，并与国旗同时悬挂时通常低于国旗。企业旗帜蕴涵企业的气质，彰显企业的文化理念，体现企业的精神风貌，展示企业的个性特色，对于一所企业具有十分重要的意义。

== 校旗 ==
[[校旗|校旗]]，泛指学校旗帜，上面通常标有学校名，是一所学校的象征与标志。学校旗通常涵盖学校识别系统的基本元素：校名、校徽、标准字、标准色等。学校旗蕴涵学校的学术气质，彰显学校的文化理念，体现学校的精神风貌，展示学校的个性特色，对于一所学校具有十分重要的意义。
