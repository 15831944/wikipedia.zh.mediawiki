== 故事簡介 ==
故事圍繞一群以御刀祓除「荒魂」的神社巫女「刀使」。她們一邊提高個人技能，一邊在就讀全國五處完全中學。她們隸屬警察組織的特別祭祀機動隊，驅除稱為「荒魂」的神秘敵對生物，履行守護大眾的職責。

; 胎動篇（第1話 - 第12話）
: '''衛藤可奈美'''在御前比賽遇上'''十條姬和'''。姬和突然襲擊刀劍類管理局局長'''折神紫'''，獲可奈美協助，得以逃離。
: 隨事態發展，她們與'''柳瀨舞衣'''、'''糸見沙耶香'''、'''益子薰'''、'''古波藏愛蓮'''會合，與組織「舞草」合作，打倒被荒魂附體的折神紫。
:<!--留空一行-->
; 波瀾篇（第13話 - 第24話）
: 折神紫被打倒後，關東經常出現荒魂。可奈美與其他刀使四出戰鬥以祓除荒魂，但有神秘人物奪去「NORO」，其中1人是獅童真希，主角們因此被捲入一連串事件。

=== 折神家親衛隊 ===
; {{nihongo|折神紫|折神 紫（おりがみ ゆかり）|聲：[[瀨戶麻沙美|瀨戶麻沙美]]}}
: 折神家家主，警視廳刀劍類管理局局長。是立於全部刀使頂點的存在。
: 武器為雙刀，御刀是大包平和童子切安綱，流派為[[二天一流|二天一流]]。
: 在二十年前的相模灣岸大災厄中被任命為特務隊隊長，討伐了造成巨大損害的大荒魂，成為救國英雄。
: 其力量仍未見衰退，巋然不動地保有最強刀使的寶座。
: 『鐮倉事件』後重傷帶職休養，發覺其生理年齡停留在二十年前。
: 動畫第二十話，為保護市杵島姬突圍，單人擊倒綾小路刀使阻擊部隊十餘人。
: 『年關災厄事件』中再度重傷，帶職告病休養。

; {{nihongo|獅童真希|獅童 真希（しどう まき）|聲：[[內山夕實|內山夕實]]}}
: 折神家親衛隊第一席。第一人稱是「僕」，流派：神道無念流，御刀是薄綠（吼丸）。右手有綁著繃帶的特徵，在刀使們較量劍術的大賽中達成前回、前前回優勝的連霸，並加入了親衛隊。
: 擁有基於信念的正直的強大，手中之劍的每一擊都鋒銳而沉重。
:主要任務是折神紫的護衛以及荒魂討伐的作戰指揮，自己並不會經常出動，一旦拔出御刀便會立即討伐荒魂。
: 不論在異性還是同性間的人氣都很高。

; {{nihongo|此花壽壽花|此花 寿々花（このはな すずか）|聲：[[市道真央|M·A·O]]}}
: 折神家親衛隊第二席。流派：鞍馬流，御刀是九字兼定。京都名門此花家的大小姐，言談舉止十分優雅。
: 擁有在劍術大賽中連續兩次亞軍的實力。
: 但對於在兩次的決賽中都將自己擊破的真希懷有競爭意識。
: 擁有敏銳的洞察力和高超的指揮能力，經常負責直接的作戰指揮，此外在紫擔任政治職務時也會同行。

; {{nihongo|皋月夜見|皐月 夜見（さつき よみ）|聲：[[淵上舞_(聲優)|淵上舞]]}}
: 折神家親衛隊第三席。在親衛隊當中也是經常守候在折神紫身邊的一位，流派：深甚流，御刀是水神切兼光。
: 不怎麼表露出感情，也不會進行超出必要程度的對話。
: 與擁有光輝實績的真希和壽壽花不同，至今為止並沒有什麼醒目的實績，實力不明。
: 目前只有以自殘的方式召喚大量荒魂出來攻擊。
: 是有著重重謎團的刀使。
: 事實上，她沒有被御刀認同，直到3年前被雪那注入荒魂後才被認同。
: 在動畫第二十話被此花壽壽花迅移近身砍中背部後掉下高樓，生死不明。
: 『年關災厄事件』中，擊殺圍攻鎌府校長高津雪那的眾荒魂，力竭後死去。

; {{nihongo|燕結芽|燕 結芽（つばくろ ゆめ）|聲：[[水瀨祈|水瀨祈]]}}
: 折神家親衛隊第四席。親衛隊中最年輕，也是頂尖的劍術高手。流派：[[天然理心流|天然理心流]]，御刀是笑面青江。
: 是實力甚至凌駕於大賽二連霸的真希之上的天才刀使，除了對陣折神紫至今未有敗績。擁有出眾的才能，但性格卻是和年齡相符的孩子氣，一旦打開了開關就沒人能阻止她。
: 因此周圍對待她也極為慎重，不常對她下達出擊許可。本人對此感到相當不滿。
: 是一位小五跳級念國一的跳級生，由於健康因素，曾經長期住院，病重時從折神紫手上得到荒魂的力量得以延續生命，並加入了親衛隊，但堅持不肯在戰鬥中運用荒魂的力量。只想在戰鬥中留下活過的證明，希望有人記得她。
: 在動畫第十一集『鐮倉事件』中遭薰與愛蓮兩人車輪戰，勝而消耗體力死去。是首個在動畫中死去的角色，結芽的死給真希、壽壽花帶來了巨大的影響。
: 在手遊版「刀使之巫女 刻印一閃的燈火」第2部 第11章『隠世』篇中，獅童真希與此花壽壽花一行人前往隠世，取回御刀「南無藥師景光」，回程時遭遇隠世中喪失生前記憶的燕結芽，但還是一直追尋其他刀使並與刀使對戰，最後瀨戶內智惠不得已將取回的「南無藥師景光」給獅童真希制止燕結芽，讓燕結芽回復記憶，隨後將她帶回現世，並在現世中復活。時間點是在高津雪那逼相樂結月選出NORO疫苗的適合者。
