{{vgname|刀剑神域 －虚空断章－|ja=ソードアート・オンライン -ホロウ・フラグメント-|en=Sword Art Online: Hollow Fragment|aka-cn=刀剑神域 －虚空碎片－}}是一款根据[[輕小說|輕小說]]《[[刀劍神域|刀劍神域]]》改编的[[PlayStation_Vita|PlayStation Vita]]游戏，为该系列的第二款游戏。游戏由[[Aquria|Aquria]]开发，[[万代南梦宫游戏|万代南梦宫游戏]]发行。本游戏于2014年4月23日在日本地区，5月29日在台湾发行中文版<ref>{{Cite web |url = https://gnn.gamer.com.tw/8/96488.html |title = PS Vita《刀劍神域 ―虛空斷章―》繁體中文版 公布限定版及特典情報 |author = KEN |date = 2015-10-25 |publisher = 巴哈姆特 |language = zh-tw |accessdate = 2018-04-15 }}</ref> ，10月在北美和欧洲地区发行。<ref name="gamefaqs">{{cite web | url = http://www.gamefaqs.com/vita/734608-sword-art-online-hollow-fragment/data| title = Release information | publisher = [[GameFAQs|GameFAQs]]|language = en| date = | accessdate = 2015-10-23}}</ref>

2015年在[[PlayStation_4|PlayStation 4]]推出一个包含虚空断章重制版和[[刀剑神域_－失落之歌－|失落之歌]]导演剪辑版。本作在重制版被命名为《'''刀剑神域 Re：－虚空断章－'''》，重制版除了包含原版游戏的所有内容外，还追加了新剧情、新[[頭目_(角色類型)|BOSS]]、支持自定义游戏角色性别、新的多人模式并提升人物等级上限<ref>{{Cite web |url = http://ps4.tgbus.com/news/201508/20150824151432.shtml |title = 《刀剑神域》最新PV Re虚空断章可单独购买 |author = 天楼神威 |date = 2015-08-24 |publisher = 电玩巴士 |language = zh-cn |accessdate = 2018-04-15 }}</ref>。预定于2018年3月23日在STEAM推出《Re：－虚空断章－》的PC版<ref>{{Cite web |url = http://www.ali213.net/news/html/2018-2/347771.html |title = 《刀剑神域：虚空断章》PC版发售日公布 预订PC《刀剑神域：夺命凶弹》可免费获赠游戏 |author = cloud |date = 2018-02-23 |publisher = 游侠网 |language = zh-cn |accessdate = 2018-04-15 }}</ref>。

== 游戏玩法 ==
玩家可以选择100多个角色来陪伴玩家完成任务。<ref>{{cite web|url=http://www.siliconera.com/2014/02/24/can-befriend-100-characters-sword-art-online-hollow-fragment/|title=You Can Befriend Over 100 Characters In Sword Art Online: Hollow Fragment|publisher=Siliconera|date=2014-02-24|accessdate=2018-04-15}}</ref>和前作[[刀剑神域_－无限瞬间－|无限瞬间]]一样，本作引入许多[[大型多人在线角色扮演游戏|大型多人在线角色扮演游戏]]中的元素。<ref name="gameinformer">{{cite web|url=http://www.gameinformer.com/b/news/archive/2014/05/01/duo-of-sword-art-online-games-coming-to-north-america.aspx|title=Duo Of Sword Art Online Games Coming To North America|publisher=[[Game_Informer|Game Informer]]|date=2014-05-01|accessdate=2017-04-15|archive-url=https://web.archive.org/web/20170415202732/http://www.gameinformer.com/b/news/archive/2014/05/01/duo-of-sword-art-online-games-coming-to-north-america.aspx|archive-date=2017-04-15|dead-url=yes}}</ref>游戏包含[[戀愛模擬遊戲|恋爱元素]]，并支持最多四人的多人模式<ref>{{cite web|url=http://www.escapistmagazine.com/news/view/133910-Sword-Art-Online-Hollow-Fragment-Arrives-in-North-America-this-Summer|title=Sword Art Online: Hollow Fragment Arrives in North America this Summer|publisher=[[The_Escapist_(magazine)|The Escapist]]|language = en|date=2014-04-22|accessdate=2018-04-15}}</ref>。
