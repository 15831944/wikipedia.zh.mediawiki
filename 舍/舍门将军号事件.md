'''舍门将军号事件'''（英语：'''General Sherman Incident'''  韩语：'''제너럴셔먼호 사건''' ）1866年发生的美国武装商船与朝鲜军队冲突事件。朝鲜民主主义人民共和国将这一事件作为朝鲜近代史的开端。

== 背景 ==
美国希望加大对东方的贸易，而[[朝鲜王朝|朝鲜王朝]]则采取[[闭关锁国|闭关锁国]]政策。1845年美国[[纽约州|纽约州]]议员、众议院海军委员会主席普拉特（Z. Pratt）就向众议院提交过《开放朝鲜的议案》；1853年1月一艘美国船只驶入朝鲜东莱府的龙堂浦。

== 过程 ==

[[美国|美国]]在华商人普雷斯顿（W B Preston）得到[[英国|英国]]驻[[天津|天津]]密迪士商会（Meadows & Co）的支持，将美式的两帆船（Schooner）''舍门将军号''（''General Sherman''）装备为武装商船，去朝鲜贸易。船长及驾驶员均为[[美国人|美国人]]，水手多为[[中国人|中国人]]，还有3名[[马来人|马来人]]。李八行为笔谈翻译，英国人何各斯（George Hogarth）为[[朝鲜语|朝鲜语]]翻译，[[英国圣公会|英国圣公会]][[传教士|传教士]]托马斯（Rev Robert Thomas，中文名[[崔兰轩|崔兰轩]]）乘该船赴朝鲜传教。

1866年8月8日，从中国芝罘（[[烟台|烟台]]）出发，8月18日（阴历七月九日）该船到达[[大同江|大同江]]口，溯江而上，至[[黄海道|黄海道]][[黄州|黄州]]，黄州牧使丁大植、翻译李容肃等登船，禁止贸易，但该船人员不予理睬。8月22日（阴历七月十三日）到达[[平壤|平壤]]府。[[平安道|平安道]]中军[[李玄益|李玄益]]不准其贸易和传教。命立即退去。平壤地方政府4次给予他们白[[米|米]]、[[牛肉|牛肉]]、[[鸡蛋|鸡蛋]]、柴薪等，让他们原地待命，等待中央政府指令。但舍门将军号继续溯江而上，崔兰轩等人上岸侦察，准备盗掘当地王陵。李玄益驾船追击，遭船员囚禁。舍门将军号以1000石白米、大量金银和人参作为撤退条件。平壤府庶尹[[申泰鼎|申泰鼎]]要求释放李玄益被拒，引发冲突。平安道观察使[[朴珪寿|朴珪寿]]督战。因江水过浅，舍门将军号无法施展，于9月2日（阴历七月二十四日）被数百艘装满茅草、浇足了油的火船焚毁，船员全部死亡。<ref>[http://book.ifeng.com/section.php?book_id=494&id=34117]</ref>一名叫朴春权的退伍军人只身救出了李玄益。<ref>《朝鲜通史》（下卷），19—21页，朝鲜民主主义人民共和国科学院历史研究所。</ref>崔兰轩爬上岸，被打死。朝鲜在这次事件中共有7人死亡，5人受伤。

== 事后 ==
1871年[[美国|美国]]对此进行了报复（[[辛未洋扰|辛未洋扰]]）。舍门将军号上的两门大炮，现在分别陈列于朝鲜国立中央历史博物馆和民族解放斗争博物馆中。金日成在自傳《[[與世紀同行|與世紀同行]]》称曾祖父金膺禹参加了焚毁舍门将军号，1986年朝鲜政府将这一事蹟刻在大同江边的石碑上。2006年朝鲜政府发行舍门将军号事件纪念[[邮票|邮票]]。<ref name="website">[http://www.kcna.co.jp/item/2006/200609/news09/02.htm#11 New Stamps Issued] {{webarchive|url=https://web.archive.org/web/20141012092409/http://www.kcna.co.jp/item/2006/200609/news09/02.htm |date=2014-10-12 }}</ref>

1968年1月22日美国间谍船[[普韋布洛號通用環境研究艦|普韋布洛號通用環境研究艦]]在[[元山|元山]]外海距海岸约12海里处遭朝鲜一艘猎潜艇、四艘鱼雷快艇追擊、包圍、俘获。1998年普韦布洛号被拖平壤大同江畔舍门将军号事件发生地。<ref>http://www.globalsecurity.org/military/ops/sherman.htm</ref>

== 争议 ==
1960年代开始，[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]历史学家称[[金日成|金日成]]的[[曾祖父|曾祖父]][[金膺禹|金膺禹]]参加了这次冲突。但国际上无法找到有关证据。
