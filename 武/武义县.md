'''武义县'''是[[中國|中國]][[浙江省|浙江省]][[金华市|金华市]]下辖的一个[[县|县]]，素有“温泉之城、萤石之乡”的美称。

== 历史 ==
[[春秋时期|春秋时期]]属[[越国|越国]]，战国后期属[[楚国|楚国]]。

[[秦朝|秦朝]]至[[东汉|东汉]]属乌伤县，三国至隋属永康县。

[[唐朝|唐]][[天授|天授]]二年（691年），始设武义县，属[[婺州|婺州]]。其后婺州改婺州路、宁越府、金华府，武义县属之。

明[[景泰|景泰]]三年（1452年），析[[麗水縣_(處州)|丽水县]]置[[宣平縣_(浙江)|宣平县]]，属处州府。

1958年5月，宣平县大部并入武义县。
1958年10月，武义县并入永康县。1961年12月恢复武义县建制。<ref>{{Cite web|title=建制沿革|publisher=武义县人民政府网站|url=http://www.zjwy.gov.cn/wy/zjwy/lsyx/|deadurl=yes|archiveurl=https://web.archive.org/web/20120428220753/http://www.zjwy.gov.cn/wy/zjwy/lsyx/|archivedate=2012-04-28}}</ref>

==行政区划==
下辖3个街道、8个镇、7个乡，14个社区、1个居民区、544个行政村：
*街道办事处：[[白洋街道|白洋街道]]、[[壶山街道|壶山街道]]、[[熟溪街道|熟溪街道]]。
*镇：[[履坦镇|履坦镇]]、[[桐琴镇|桐琴镇]]、[[泉溪镇|泉溪镇]]、[[王宅镇|王宅镇]]、[[桃溪镇|桃溪镇]]、[[柳城畲族镇|柳城畲族镇]]、[[新宅镇|新宅镇]]、[[茭道镇|茭道镇]]。
*乡：[[大田乡|大田乡]]、[[白姆乡|白姆乡]]、[[坦洪乡|坦洪乡]]、[[西联乡|西联乡]]、[[三港乡|三港乡]]、[[大溪口乡|大溪口乡]]、[[俞源乡|俞源乡]]。

== 地理 ==
武义县南部、西部和北部三面环山。海拔1560.2米的[[牛头山_(武义)|牛头山]]，为[[金华市|金华市]]第一高峰。

==交通==
[[File:Wuyi_huochezhan_20120219.jpg|thumb]]]]
*铁路：[[金温铁路|金温铁路]][[武义站|武义站]]距金华32公里，每天运行列车14对。
*公路：[[长深高速公路|长深高速公路]]、[[330国道|330国道]]、44省道穿境而过。
*航空：武义距义乌民航机场约45分钟车程，距杭州、上海国际机场约3－4小时车程。

== 经济 ==
经济以工业、[[农业|农业]]为主，目前為[[山稻|山稻]]推廣試驗基地。

== 人口 ==
2007年年末全县户籍总人口333327人。

居民以[[汉族|汉族]]为主，土著少数民族有[[畲族|畲族]]，另外有[[苗族|苗族]]、[[布依族|布依族]]、[[侗族|侗族]]等少数民族。

全县共有316个姓，其中万人以上大姓按人数排列依次为徐、陈、王、朱、李、吴、何、周、张。<ref>{{Cite web|title=人口民族|publisher=武义县人民政府网站|url=http://www.zjwy.gov.cn/wy/zjwy/rkmz/|deadurl=yes|archiveurl=https://web.archive.org/web/20120428220759/http://www.zjwy.gov.cn/wy/zjwy/rkmz/|archivedate=2012-04-28}}</ref>
