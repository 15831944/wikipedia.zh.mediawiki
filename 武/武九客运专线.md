'''武九客运专线'''是中国大陆一条铁路[[客运专线|客运专线]]，因连接[[湖北省|湖北省]][[武汉市|武汉市]]和[[江西省|江西省]][[九江市|九江市]]而得名。该线路江西段和湖北段先后于2013年底、2014年底动工，2017年9月21日全线通车<ref>[http://www.hb.xinhuanet.com/2015-03/10/c_1114593060.htm 武九客专大冶至阳新段开建 2017年通车] {{webarchive|url=https://web.archive.org/web/20150402150738/http://www.hb.xinhuanet.com/2015-03/10/c_1114593060.htm |date=2015-04-02 }}</ref><ref>[http://jiangxi.jxnews.com.cn/system/2013/10/11/012704677.shtml 武九客专江西段初步设计获批]</ref><ref>{{cite news|url=http://ctdsb.cnhubei.com/html/ctjb/20170920/ctjb3174583.html|title=武九客专明日全线开通运营 26对长途动车车票陆续发售 武汉2小时到南昌|publisher=[[楚天金报|楚天金报]]|date=2017-09-20|accessdate=2017-09-21}}</ref>。

== 概况 ==
[[武汉站|武汉]]至[[大冶北站|大冶北]]段与[[武黄城际铁路|武黄城际铁路]]共线，于[[大冶北站|大冶北站]]出岔为新建线路，終到[[九江站|九江站]]。铁路为[[国铁|国铁]]Ⅰ级[[双线铁路|双线]][[电气化铁路|电气化铁路]]，设计时速为250公里，該鐵路東南方向將與[[昌九城際鐵路|昌九城際鐵路]]及[[昌福鐵路|昌福鐵路]]连接，形成一条连接中部地區、东南沿海地區的国家大通道；東與[[九景衢鐵路|九景衢鐵路]]連接，形成一条連接中部和东部的快速通道<ref>{{cite web | url=http://www.hubei.gov.cn/zwgk/rdgz/rdgzqb/201706/t20170607_1003536.shtml|title=武九客专湖北段6月中旬通车 武汉到南昌2小时|publisher=湖北日报| date= 2017-06-09 | accessdate= 2017-06-09}}</ref>。

武九客运专线全線通車後，武漢至九江將由現在的4小時縮減為1小時，武漢至南昌將由現在的2小時40分縮減為1.5小時<ref>{{cite web| url=http://www.moc.gov.cn/st2010/hubei/hb_zhengwudt/hb_shizhousm/201703/t20170313_2175490.html| title=【武汉】武九客专6月开通 武汉到南昌最快仅需一个半小时| publisher=湖北省交通運輸廳| date=2017-03-13| accessdate=2017-06-09}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 车站 ==
武九客运专线在[[武汉站|武汉]]至[[大冶北站|大冶北]]段与[[武黄城际铁路|武黄城际铁路]]共线，设[[武汉站|武汉站]]、[[武汉南线路所|武汉南线路所]]、[[肖马杨线路所|肖马杨线路所]]、[[花山线路所|花山线路所]]、[[花山南站|花山南站]]、[[左岭站|左岭站]]、[[葛店南站|葛店南站]]、[[华容南站|华容南站]]、[[鄂州站|鄂州站]]、[[鄂州东站|鄂州东站]]、[[花湖站|花湖站]]、[[黄石北站|黄石北站]]、[[大冶北站|大冶北站]]<ref>[http://hb.sina.com.cn/news/j/2014-04-30/0834163958.html 武黄城铁10个站点确定 大冶北至阳新段年内开工]</ref>，[[大冶北站|大冶北]]至[[九江站|九江]]段另设[[白沙铺站|白沙铺站]]、[[阳新站|阳新站]]、[[枫林站|枫林站]]、[[瑞昌西站|瑞昌西站]]、[[柴桑站|柴桑站]]、[[庐山站|庐山站]]、[[九江站|九江站]]。可与[[武汉地铁|武汉地铁]][[武汉地铁4号线|4号线]]在[[武汉火车站_(武汉地铁)|武汉火车站]]实现换乘。

武九客运专线最早于2008年底提出<ref>[http://jiangxi.jxnews.com.cn/system/2008/11/01/002873100.shtml 江西省掀起新一轮铁路建设高潮]</ref>；当时计划于2012年完工<ref>{{Cite web |url=http://jiangxi.jxnews.com.cn/system/2008/12/29/010027771.shtml |title=铁道部与江西省签署加快推进铁路建设意见 |access-date=2013-11-09 |archive-url=https://web.archive.org/web/20131109154354/http://jiangxi.jxnews.com.cn/system/2008/12/29/010027771.shtml |archive-date=2013-11-09 |dead-url=yes }}</ref>。

2009年底[[武广客运专线|武广客运专线]]通车，有报道指出对江西的影响时同时指出武九客运专线时速为300公里以上<ref>[http://www.jxnews.com.cn/xxrb/system/2009/12/24/011274359.shtml 武广高铁开通 影响江西几何]</ref>。

2010年初武九客运专线的规划时速为350公里，其中江西段长58.8公里<ref>[http://jiangxi.jxnews.com.cn/system/2010/02/25/011315076.shtml 武九客运专线时速将达350公里]</ref>，并计划在2010年开建<ref>[http://jiangxi.jxnews.com.cn/system/2010/03/05/011320984.shtml 武九客运专线今年将开建]</ref>。6月17日确认在湖北武九客运专线将与[[武黄城际铁路|武黄城际铁路]]衔接，而远期武汉至黄石段新建双线铁路与武黄城际铁路分开运行<ref>[http://news.cnhubei.com/todaynews/dcw/hsyw/201006/t1248850.shtml 武黄城际铁路建设正酣，武九客运专线又将启动 届时黄石至京广沪渝都只需4小时]</ref><ref>[http://jj.jxnews.com.cn/system/2010/06/25/011414442.shtml 武汉至九江铁路客运专线2013年将全线建成]</ref>。2010年10月19日湖北省环境保护厅印发《新建大冶北至阳新铁路工程环评第一次公示》<ref>[http://www.hbepb.gov.cn/wsbs/gsgg/hpgs/hpdwhp/201010/t20101019_36103.html 新建大冶北至阳新铁路工程环评第一次公示]</ref>。

2010年有报道称该线路有望在2011年开建<ref>[http://jj.jxnews.com.cn/system/2010/11/25/011527085.shtml 九景衢铁路、武九客运专线有望明年开工建设]</ref>，然而到了2012年仍然停留在拟开建阶段，计划在2013年开建，此时设计近期时速200公里以上，远期时速达350公里，全长198公里<ref>[http://jiangxi.jxnews.com.cn/system/2012/10/30/012157499.shtml 武九客运专线力争明年开工 南昌北京将实现高铁直达[图<nowiki>]</nowiki>]</ref>。2012年10月19日到22日铁道部相关人员到庐山站、瑞昌南站场址现场踏勘<ref>[http://jj.jxnews.com.cn/system/2012/10/30/012158475.shtml 武九客运专线瑞昌至九江段可研评审会在武汉召开]</ref>。
