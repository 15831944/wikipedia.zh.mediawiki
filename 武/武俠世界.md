《'''武俠世界'''》是武俠世界出版社的[[武俠小說|武俠小說]]雜誌，由[[新報_(香港)|新報]]創辦人[[羅斌|羅斌]]於1959年4月1日創辦，環球／武林出版社出版，《武俠世界》為週刊，逢星期一出版。雜誌除發行[[香港|香港]]、[[澳門|澳門]]和[[台灣|台灣]]以外還遍及[[亞太|亞太]][[歐美|歐美]]等國<ref>{{cite web|author1=郭雨南|title=作家學者談文論武|url=http://www.takungpao.com.hk/culture/text/2016/0722/10870.html|website=大公網|accessdate=2017年7月6日|date=2016年7月22日}}</ref>。2002年羅斌出售香港旗下業務，由當時雜誌主編[[沈西城|沈西城]]接手營運<ref>{{cite news|title=【文化籽】沈西城憶《新報》　羅斌從不炒人魷魚|url=http://hk.apple.nextmedia.com/supplement/special/art/20150717/19221818|accessdate=2017年7月3日|agency=蘋果日報|date=2015年7月17日}}</ref>。

== 歷史 ==
1950年代武俠出事業版蓬勃發展，三大武俠雜誌先後登場。[[新報|新報]]創辦人羅斌於1959年4月1日推出《武俠世界》，首期連載蹄風的《鐵掌雄風》和金鋒的《虎俠擒龍》；明報創辦人[[金庸|金庸]]則於1960年1月11日推出《[[武俠與歷史|武俠與歷史]]》<ref>{{cite news|url=http://www.rthk.hk/tv/dtt31/programme/sciencewithyou2017/episode/425743|title=代 Cult2：八十前 撲報攤買雜誌最難忘|date=2010年8月31日|accessdate=2017年7月3日|agency=蘋果日報}}</ref>，首期連載金庸自己大作《飛狐外傳》；一時瑜亮，鬥得難分難解。其後《姊妹》雜誌於1970年1月25日出版《[[武俠春秋|武俠春秋]]》<ref>{{cite web|author1=謝詩琪|title=《武俠世界》：難離難捨的雜誌情懷|url=http://media.stu.edu.cn/hkbookfair/?p=2560|website=汕头大学新闻学院|accessdate=2017年7月3日|date=2016年7月27日}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，首期連載古龍的《蕭十一郎》。至此三雄鼎立，各自精彩。

武俠作家的作品先在《武俠世界》雜誌上連載，然後結集出版[[單行本|單行本]]。不少武俠名家[[石沖|石沖]]、[[蹄風|蹄風]]、[[張夢還|張夢還]]、金童（[[臥龍生|臥龍生]]）、[[倪匡|倪匡]]、[[黃易_(香港)|黃易]]都在雜誌上刊載過小說<ref name="連民安吳貴龍2016">{{cite book|author1=連民安|author2=吳貴龍|title=星光大道──五六十年代香港影壇風貌|url=http://books.google.com/books?id=fLzsDQAAQBAJ&pg=PT133|date=2016年7月11日|publisher=中華書局(香港)有限公司|isbn=978-988-8420-06-3|pages=133–134}}</ref>

《武俠世界》高峰時期銷售量為20,000本，但因應電子傳媒的普及，紙媒漸漸失去市場，現在銷售量維持2,000本<ref>{{Cite web |url = http://www.rthk.hk/tv/dtt31/programme/knowledgezone2016/episode/375185 |title = 漫遊百科 2016 第二十一集 沈西城：武俠世界一甲子 |author = 香港電台 |date = 2016年11月12日 |language = zh-hant |accessdate = 2017年7月4日 }}</ref>。

2019年1月15日起《武俠世界》宣佈停刊，網絡潮流衝擊，該刊編輯部決定停刊紙本，並表示會研究出版網媒的可能性，希望日後可在網上刊登目前尚未刊完的小說<ref>{{cite news |title=【世界書訊】香港︰60年歷史《武俠世界》停刊 |url=http://std.stheadline.com/daily/article/detail/1952773-%E6%95%99%E8%82%B2-%E3%80%90%E4%B8%96%E7%95%8C%E6%9B%B8%E8%A8%8A%E3%80%91%E9%A6%99%E6%B8%AF%EF%B8%B060%E5%B9%B4%E6%AD%B7%E5%8F%B2%E3%80%8A%E6%AD%A6%E4%BF%A0%E4%B8%96%E7%95%8C%E3%80%8B%E5%81%9C%E5%88%8A |accessdate=2019-04-22 |work=星島日報 |date=2019-01-28}}</ref>。
