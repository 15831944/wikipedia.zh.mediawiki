== 剧情 ==
该剧讲述武松在景阳冈打虎之后回家乡成了打虎英雄，遇到哥哥武大郎被西门庆和嫂嫂潘金莲毒害，武松新替哥哥报仇雪恨，杀害西门庆、潘金莲、王婆，被官府判刑流放，发配[[孟州|孟州]]，后来结识鲁智深，加入梁山泊，成为江湖豪杰。

== 花絮 ==
该剧中潘金莲加入了导演的同情元素，潘金莲出轨属于不能满足[[性生活|性生活]]，大胆追求[[爱情|爱情]]，敢爱敢恨，时尚元素是催泪、虐心。<ref>{{cite web |url=http://www.dzwww.com/yule/yulezhuanti/mtcbg/201307/t20130724_8711098.htm |title=孙耀琦多剧登暑期档 《武松》两大卫视将播 |accessdate=2013年7月27日 |publisher=大众网 |work= |language=zh-cn }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web|url= http://www.dzwww.com/yule/yulezhuanti/mtcbg/201307/t20130718_8686917.htm|title= 《武松》卫视将播 孙耀琦版潘金莲“情路虐心”|accessdate= 2013年7月27日|publisher= 大众网|work= |language= zh-cn|archive-url= https://web.archive.org/web/20130721062025/http://www.dzwww.com/yule/yulezhuanti/mtcbg/201307/t20130718_8686917.htm|archive-date= 2013-07-21|dead-url= yes}}</ref>

该版武松被誉为《[[赛德克·巴莱|赛德克·巴莱]]》，[[金像奖|金像奖]]影后[[惠英红|惠英红]]出演王婆。<ref>{{cite web |url=http://ent.qq.com/a/20130718/012648.htm |title=《武松》收视直逼破五 明星阵容重铸武侠传奇 |accessdate=2013年7月27日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>
