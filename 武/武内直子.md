'''武内直子'''（{{bd|1967年|3月15日|||Takeuchi, Naoko}}），小名為「'''直子公主'''」。在[[日本|日本]][[山梨縣|山梨縣]]甲府市出生，[[日本|日本]]女性[[漫畫家|漫畫家]]，代表漫畫作為《[[美少女戰士|美少女戰士]]》，在1999年與[[冨樫義博|冨樫義博]]結婚，改名為'''冨樫直子'''。曾就讀於甲府市立北中學、山梨縣立甲府第一高中。畢業於共立藥科大學藥學系（現[[慶應義塾大學|慶應義塾大學]]藥學系）。持有藥劑師及臨床檢查技師的資格。大學畢業後的半年間，曾在慶應義塾大學醫院中央檢查部工作過。1986年以作品《[[Love_Call|Love Call]]》入選第2回《[[Nakayoshi|Nakayoshi]]》新人漫畫賞而出道。

== 重要記事 ==
*1986年武內直子的作品《[[Love_Call|Love Call]]》獲得第二屆《[[Nakayoshi|Nakayoshi]]》新人漫畫賞。
*1993年武內直子的作品《[[美少女戰士|美少女戰士]]》獲得了第十七屆「[[講談社|講談社]]」漫畫賞的大獎。
*1997年12月24日武內直子與冨樫義博共同出版一本[[同人誌|同人誌]]。冨樫義博畫了武內直子的作品《[[美少女戰士|美少女戰士]]》，武內直子也畫了一些平常不畫的東西，並且書中還有武內直子與冨樫義博的唇印，附註是冨樫王子與直子公主的聖誕禮物（自此時起二人的八卦謠言滿天飛）。
*1999年1月武內直子終於在眾人的祝福下與冨樫義博步入禮堂。當時的兩位司儀是[[三石琴乃|三石琴乃]]（《[[美少女戰士|美少女戰士]]》[[月野兔|月野兔]]的聲優）和[[佐佐木望|佐佐木望]]（《[[幽遊白書|幽遊白書]]》[[浦飯幽助|浦飯幽助]]的聲優）。
*2000年11月冨樫夫婦生下一個兒子。

== 人物介绍 ==
武內直子原本是做医药行业的[[药剂师|药剂师]]，在读大学的时候就已经是《Nakayoshi》的漫画新人奖的人选而正式进入漫画界。初中时最得意的学科是[[理科|理科]]，[[美术|美术]]在那时也非常得手，因为武內直子从小就很喜欢画画，高中时代参加过美术社团与天文社团。实际上武內直子出生在一个有錢的家庭，因为她的家庭是经营珠宝店的，所以在她创作的作品中大多都是讲述有钱人的故事。

武內直子从小读到大的[[漫画杂志|漫画杂志]]就是《Nakayoshi》，第一次读到《Nakayoshi》是一次刚进小学得了感冒時，家人为了安慰她才买了这本杂志送她，后来阅读漫画上瘾就开始假装生病，她以为就能得到很多漫画。之后从初中到高中，每本[[少女漫画|少女漫画]]她都会买，一个月大约要买20多本。最喜欢[[松本零士|松本零士]]漫画中的少女形象以及[[上原贵子|上原贵子]]漫画中华丽的画风和有趣的对白。

武內直子开始认真创作漫画是在高中的时候，她最爱用的画材是0.05[[铅笔|铅笔]]、[[马克笔|马克笔]]、[[彩色光漆|彩色光漆]]、[[彩色墨水|彩色墨水]]以及用起来很顺手的[[彩色粉笔|彩色粉笔]]，武内直子在用彩色墨水时总担心印刷出来的色调和原稿的差别很大，所以每次在画彩色画稿时总是薄薄的上色。上沾水笔线对武内直子来说是很痛苦的，画不好的地方总是喜欢用[[白颜料|白颜料]]来修正、掩盖，她本人也喜欢闪闪发光的效果。除了画漫画外，她还为自己作品（特别是《美少女战士》）的游戏与音乐CD写剧本，作词，设计衍生的角色形象等。当然她还画过[[小说家|小说家]]-[[小泉真理绘|小泉真理绘]]所著作小说的插图。直到2006年正式淡出漫画界

== 工作日程 ==
以下为武內直子一个月的工作日程安排情况：
*1—4日、《るんるん》漫画杂志的封面以及KC单行本的设计与装饰。
*5—6日、时不时的给彩色画稿上色。
*7—9日、阅读相关的资料与书籍，取材来构思作品。
*10—16日、全心投入到画《Nakayoshi》即将连载的画稿中。
*17—20日、在家里休息睡觉、时不时跟朋友出去逛街、聊天。
*21—30日、主要专心彩稿的着色工作等。
