'''武汉轨道交通16号线'''（又称'''武汉地铁16号线'''，英语：'''Wuhan Metro Line 16'''）是[[武汉地铁|武汉地铁]]规划中的城郊地铁线路之一，线路代表色是芙蓉红。本线连接[[武汉经济技术开发区|武汉经济技术开发区]]与[[汉南区|汉南区]]，南起汉南区纱帽周家湾，北至[[国博中心南站|国博中心南站]]，沿纱帽大道、马影河大道、汉洪高速、军山大道、沌口路敷设，穿越了马影河、大军山、小军山、四环线、分蓄洪区、东荆河、三环线等走行。全线长约32.1公里，其中高架段长约26.2km，地下段约5.52km。设11座站点，其中高架站8座，地下站3座，在国博中心南站可换乘[[武汉地铁6号线|地铁6号线]]、[[武汉地铁11号线|11号线]]和[[武汉地铁12号线|12号线]]，在[[老关村站|老关村站]]可换乘[[武汉地铁6号线|地铁6号线]]，16号线一期取消了之前规划的与10号线换乘的万家湖站。此外，16号线二期规划走向从周家河至通航产业园站，全长3.7公里，设站1座。作为武汉西南端的主干线，地铁16号线沿途经过汉阳共建区、汉阳纸厂、万家湖旧城改造片区，跨东荆河、汉洪高速后进入智慧生态城、过大军山、港口物流园后进入汉南新城，远期延伸至通航产业园区。已于2017年开始建设，预计2019年投入运营。<ref name="地铁16号线进入全面建设阶段 年内将完成一半土建预埋工程"></ref><ref>{{Cite web |url =http://news.cnhubei.com/xw/wuhan/201702/t3791561.shtml|title =武汉地铁16号线一期11座站点公布 设两座换乘车站|author =郭徽|date =2017-02-25|publisher =楚天都市报}}</ref><ref>{{Cite web |url =http://news.cnhubei.com/xw/wuhan/201701/t3769396.shtml|title =地铁16号线有望今年开工 2019年汉南进入地铁时代|author =张希为, 周明德|date =2017-01-07|publisher =武汉晚报}}</ref><ref>{{Cite web |title =16号线一期工程设计总体总包及单项设计招标公告|url =http://www.whrt.gov.cn/public_forward.aspx??url=public_content_page.aspx?newid=0B531B062A41B253482580D100016B4F|date = 2017年2月24日|publisher =武汉地铁集团有限公司 }}</ref>
