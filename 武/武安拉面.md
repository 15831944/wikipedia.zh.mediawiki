'''武安拉面'''是河北[[邯郸|邯郸]][[武安|武安]]具有邯郸市本地特色的[[小吃|面食小吃]]。与其他的[[邯郸菜系|邯郸菜系]]，构成了邯郸主要有[[邯郸小吃|十大名小吃]]<ref>{{Cite web |url=http://travel.hebnews.cn/zt/2012/2012-09/29/content_2868168.htm |title=闻名遐迩 邯郸最有名的十大地道小吃 |access-date=2017-08-20 |archive-url=https://web.archive.org/web/20130221210750/http://travel.hebnews.cn/zt/2012/2012-09/29/content_2868168.htm |archive-date=2013-02-21 |dead-url=yes }}</ref><ref>[http://hebei.news.163.com/13/0925/14/99KIL4S50279068A.html 十一精彩旅游推荐之邯郸“最好吃的美食”]</ref>。

== 起源 ==
武安拉面起始于[[清末|清末]][[民初|民初]]，由当时的[[武安|武安]]本地的一对兄弟二人创制，后来，一人留在武安本地经营，一人去了[[山西|山西]]。
== 作法 ==
武安拉面主要讲究的是[[和麵|和麵]]的手法，首先和麵时，必须要在水里加少许的[[食盐|食盐]]，并且还要搅拌均匀，做到一边到水，一边和麵，一直做到让麵的软硬度适中为好；其次，在和好的麵上面盖一湿布，这样的目的是让麵醒十多分钟；最后就是绕麵、让麵的本身起到光滑、软硬适中的作用，再用擀麵杖擀成圆饼形状，用刀切条，散开，根据个人的喜好将麵切的或窄或宽，切好后把麵条拉开下锅即可<ref>{{cite web|title=武安拉面|url=http://hebei.ifeng.com/travel/meishi/detail_2014_07/17/2611970_0.shtml|website=凤凰网|accessdate=2017-08-20}}</ref>。
