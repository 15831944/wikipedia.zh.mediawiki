'''武汉商学院'''创建于1985年，原名'''武汉商业服务学院'''，是[[湖北省人民政府|湖北省人民政府]][[主管|主管]]、[[武汉市人民政府|武汉市人民政府]][[主办|主办]]、[[独立设置|独立设置]]的[[公办|公办]][[普通本科院校|普通本科院校]]。学院[[占地面积|占地面积]]1026亩，[[建筑面积|建筑面积]]32万平方米，位于[[武汉经济技术开发区|武汉经济技术开发区]][[薛峰堡|薛峰堡]]，招生面向全国31个[[省|省]]、[[市|市]]、[[自治区|自治区]]。<ref name="武汉商学院学校简介">{{cite web| publisher =| title =武汉商学院简介，| url =http://www.wbu.edu.cn/s/22/t/95/p/1/c/4/list.htm| author =| date =1| accessdate =2013-04-18| deadurl =yes| archiveurl =https://web.archive.org/web/20160929233126/http://www.wbu.edu.cn/s/22/t/95/p/1/c/4/list.htm| archivedate =2016年9月29日| df =}}</ref>

== 历史沿革 ==
*1963年创建[[武汉市服务学校|武汉市服务学校]]。 
*1971年[[武汉市商业学校|武汉市商业学校]]和[[武汉市服务学校|武汉市服务学校]]合并，以[[武汉市商业学校|武汉市商业学校]]为校名；
*1973年二商校迁至[[汉阳区|汉阳区]][[墨水湖|墨水湖]]路校址；
*1985年经[[武汉市政府|武汉市政府]]批准，[[教育部|教育部]]备案，以[[二商校|二商校]]为基础组建[[武汉商业服务学院|武汉商业服务学院]]；
*2001年学院整体搬迁至[[武汉经济技术开发区|武汉经济技术开发区]]薛峰堡新校址；
*2004年经[[武汉市人民政府|武汉市人民政府]]报请[[湖北省人民政府|湖北省人民政府]]批准，[[武汉市经济干部管理学院|武汉市经济干部管理学院]]、[[武汉职工财经学院|武汉职工财经学院]]、[[武汉市职工大学|武汉市职工大学]]并入'''武汉商业服务学院'''，组建新的[[武汉商业服务学院|武汉商业服务学院]]；
*2013年4月18日，教育部批准在[[武汉商业服务学院|武汉商业服务学院]]基础上建立本科层次的'''武汉商学院'''。<ref name="教育部关于同意在武汉商业服务学院基础上建立武汉商学院的函">{{cite web | url=http://www.moe.gov.cn/publicfiles/business/htmlfiles/moe/s181/201305/151842.html | title=教育部关于同意在武汉商业服务学院基础上建立武汉商学院的函 | accessdate=18 04, 2013}}</ref>

== 学院简介 ==
'''武汉商学院'''位于[[武汉经济技术开发区|武汉经济技术开发区]][[薛峰堡|薛峰堡]]，学院设有[[工商管理学院|工商管理学院]]、[[商贸物流学院|商贸物流学院]]、[[旅游与酒店管理学院|旅游与酒店管理学院]]、[[体育与马术学院|体育与马术学院]]、[[烹饪与食品工程学院|烹饪与食品工程学院]]、[[机电工程与汽车服务学院|机电工程与汽车服务学院]]、[[信息工程系|信息工程系]]、[[应用艺术系|应用艺术系]]、[[应用外语系|应用外语系]]、[[思想政治理论课部|思想政治理论课部]]、[[继续教育学院|继续教育学院]]等11个教学院系部，另设有53个专科专业及专业方向，其中，[[制冷与空调技术|制冷与空调技术]]、[[动漫设计与制作|动漫设计与制作]]、[[摄影摄像技术|摄影摄像技术]]等专业在同类院校和社会中具有一定影响，率先在全国开办了[[赛马|赛马]]专业。

== 师资力量 ==
学校现有[[专任教师|专任教师]]600余人，其中，[[研究生|研究生]]学历教师占47%，[[副高|副高]]以上[[专业技术|专业技术]]职务教师占39%。教师中有享受[[国务院|国务院]]和省市政府[[津贴|津贴]]专家6人、“武汉市有[[突出贡献|突出贡献]]中[[青年专家|青年专家]]”2人、“[[武汉市学科带头人|武汉市学科带头人]]”15人、[[国家级行业大师名师|国家级行业大师名师]]25人、[[省级教学团队|省级教学团队]]3个、[[楚天技能名师|楚天技能名师]]8名及一批省市[[优秀教师|优秀教师]]。
