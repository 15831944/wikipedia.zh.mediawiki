'''武昌区'''是[[中华人民共和國|中国]][[湖北省|湖北省]][[武汉市|武汉市]]的一个市辖区，人口为108.6万（2017年常住户籍人口），是原[[武昌市|武昌市]]的主体部分。当地人的习惯将[[武汉市|武汉市]][[长江|长江]]以南的市区部分统称[[武昌_(消歧义)|武昌]]，包括'''武昌区'''、[[青山区_(武汉市)|青山区]]、[[洪山区|洪山区]]三个[[行政区划|行政区划]]和[[东湖新技术开发区|东湖新技术开发区]]。武昌区也是[[中华人民共和国|中华人民共和国]]湖北省委、[[湖北省人民政府|湖北省人民政府]]、湖北省人大以及湖北省政协等省级国家机关驻地。

== 历史 ==
'''武昌'''是一座历史文化名城，建城历史可以追溯到1800年前的[[三国|三国]]时期。“武昌”一词，发源于[[东汉|东汉]]末年，当时的武昌所指地理范围是今天的[[鄂州|鄂州]]，而当时的[[夏口|夏口]]城是今天的武昌。时值[[汉|汉]][[丞相|丞相]][[曹操|曹操]]率军讨伐东南的地方诸侯[[孙权|孙权]]。双方在[[长江|长江]]中游两岸形成气氛十分紧张的军事对峙。[[孙权|孙权]]为抵抗[[曹操|曹操]]的军事打击，在[[长江|长江]][[江汉平原|江汉平原]]段的南岸严密布兵，其后的战争，史称[[赤壁之战|赤壁之战]]。其中今[[武汉市|武汉市]]和[[鄂州市|鄂州市]]交接处，因驻扎有大型军事基地而逐渐商贾聚集，形成集市；当地因武而昌，故时名武昌。还有一种说法，是孙权给武昌命名，意为“武运昌盛”。武昌与鄂州这两个名称在历史上多次交换，到[[元朝|元朝]]开始定为今天所见的地名。

武昌形成之后，因其逐渐繁华，加之地理位置特殊，其在[[楚|楚]]地的地位也逐渐上升。到[[元|元]]、[[明|明]]两朝，武昌的地位开始赶超[[荆州|荆州]]，成为[[湖北|湖北]]的省级政府办公所在地。至[[明朝|明朝]]末年，[[长江|长江]]对岸的[[汉口|汉口]]在商业上取得了极大发展，开始形成武昌、[[汉阳_(武汉三镇)|汉阳]]、[[汉口|汉口]]“三镇鼎立”的局面。[[清朝|清朝]]在此设有武昌府，是[[湖北|湖北]][[巡抚|巡抚]]和[[湖广|湖广]]总督（后来是[[两湖|两湖]]总督）的办公机构所在地。

1911年10月10日，[[同盟会|同盟会]]领导的起義在武昌城爆发，史稱[[武昌起义|武昌起义]]，打响了[[辛亥革命|辛亥革命]]第一枪。[[中国|中国]]最后一个[[封建社会|封建王朝]][[清王朝|清王朝]]在此次[[辛亥革命|辛亥革命]]中土崩瓦解，[[中国|中国]]2000多年来的[[君主制度|君主制度]]至此终结。

1927年1月[[中华民国|中华民国]][[国民政府|国民政府]]建立并实施[[北伐|北伐]]、攻占武昌后，将武昌、[[汉阳|汉阳]]、[[汉口|汉口]]三大重镇合并为一个行政区划，即[[武汉市|武汉市]]。

[[File:Chinese-Army-Wuhan-Flag-1911-1928_dots18.svg|thumb]]，[[武昌起义|武昌起义]]后[[中华民国|中华民国]][[湖北军政府|湖北军政府]]宣告成立时的旗帜]]
1949年4月17日，[[解放军|解放军]]进驻武昌。[[中华人民共和国|中华人民共和国]]成立后，将[[武汉市|武汉市]]依定为[[湖北省|湖北省]]的[[省会|省会]]，[[湖北省|湖北省]]省委、省政府及省级政府部门都设置在武昌区。

== 地理 ==
'''武昌区'''位于[[武汉市|武汉市]]的中东部，东、南与[[洪山区|洪山区]]接壤，西与[[江岸区|江岸区]]、[[江汉区|江汉区]]、[[硚口区|硚口区]]毗邻，北与[[青山区_(武汉市)|青山区]]相接。长江位于武昌区西岸。地形以平原为主，有蛇山、凤凰山、洪山、小洪山、[[珞珈山|珞珈山]]等残丘，区内有[[沙湖_(武汉)|沙湖]]、[[武汉东湖|东湖]]、[[晒湖|晒湖]]等湖泊塘堰。

== 人口 ==
据2006年人口普查，武昌区户籍总人口100.54万，人口密度为12412人/km²，人口出生率6.71 ‰，自然增长率2.7‰，符合政策生育率 99.24 %，人口出生统计准确率100 %，流动人口育龄妇女《婚育证明》查验率93％，出生人口性别比108<ref>[http://www.wuchang.gov.cn/publish/wcqzfw/2006-11/607/18533.html 我区今年全面完成人口和计划生育责任目标]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}，武昌区政府官方网站，2006年11月7日</ref>。

== 教育 ==
[[File:WHU-wenge.jpg|thumb]]行政大楼的早期照片]]

武昌区是[[武汉市|武汉市]]的教育大区，初等教育、中等教育普及率很高，有132所中小学校，中小学生12万余人。该区的中等教育在[[湖北省|湖北省]]名列第一，有[[华中师范大学附属第一中学|华中师范大学附属第一中学]]、[[湖北省武昌实验中学|湖北省武昌实验中学]]、[[武汉中学|武汉中学]]、[[武汉市第十四中学|武汉市第十四中学]]、[[湖北省水果湖高级中学|湖北省水果湖高级中学]]等知名的省重点中学。该地区有[[武汉大学|武汉大学]]、[[华中科技大学|华中科技大学]]、[[武汉理工大学|武汉理工大学]]、[[中南财经政法大学|中南财经政法大学]]等高校。武昌区也是重要的科研基地，有[[中科院武汉分院|中科院武汉分院]]等市级以上科研机构48所。

== 卫生 ==
位于武昌区的[[武汉大学人民医院|武汉大学人民医院]]、[[武汉大学中南医院|武汉大学中南医院]]、[[广州军区武汉总医院|广州军区武汉总医院]]、[[武汉市第三医院|武汉市第三医院]]等综合医院和[[武汉大学口腔医院|武汉大学口腔医院]]等专科医院是[[华中地区|华中地区]]知名医院。

== 经济 ==
武昌商贸、金融业发达，一直是长江沿岸重要商品集散地，有[[中商广场|中商广场]]、[[亚贸广场|亚贸广场]]等一批大中型国有商业零售企业，国际零售业巨头[[家乐福|家乐福]]、[[麦德龙|麦德龙]]、[[百安居|百安居]]已在武昌落户。
