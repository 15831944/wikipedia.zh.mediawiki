'''武夷山风景名胜区'''位于中国[[福建省|福建省]][[武夷山市|武夷山市]]中部，行政区域属[[武夷街道|武夷街道]]和[[星村镇|星村镇]]，设立于1979年，1982年被列为第一批[[国家重点风景名胜区|国家重点风景名胜区]]<ref>{{cite book |title=福建省志·武夷山志 |author=福建省地方志编纂委员会 |publisher=方志出版社 |year=2004 |isbn=7-80192-188-7 |pages=127 }}</ref>，是[[武夷山_(世界遗产)|“武夷山”世界遗产]]的重要组成部分。也是中國少數以觀光成功拉動低排放經濟的城市。

武夷山不是一座独立的山峰，而是一片低山丘陵，属于典型的[[丹霞地貌|丹霞地貌]]，发源于[[黄岗山|黄岗山]]的九曲溪由西向东，历九曲十八弯，横贯这一地区，自古即以“丹山碧水”著称于世，被誉为“奇秀甲东南”。

早在[[青铜时代|青铜时代]]，这里就是闽越先民的活动地带，至今还保存有[[武夷山崖墓群|武夷山崖墓群]]。[[西汉|西汉]]时期，[[汉武帝|汉武帝]]就曾派遣使者在大王峰南麓设坛祭祀山神[[武夷君|武夷君]]，武夷山由此而得名。[[唐玄宗|唐玄宗]][[天宝|天宝]]年间封武夷山为“天下名山”。唐末五代时把武夷山列为天下三十六洞天的“第十六洞升真元化之天”。[[南宋|南宋]]时期，[[朱熹|朱熹]]在五曲隐屏峰下设武夷精舍，讲学数十年，将理学发扬光大。宋代遇林亭一带是[[建窑|建窑]]的主要生产地之一，现在仍存有[[遇林亭窑址|遇林亭窑址]]。元代在四曲一带设有御茶园，进贡[[武夷岩茶|武夷岩茶]]，至近代，[[大红袍|大红袍]]更成为茶中极品。

武夷山古时即被人们称其具备“三三六六”之胜，三三得九，即九曲溪；六六三十六，指三十六座奇峰。另外还有七十二洞、九十九岩、一百零八景等称谓。现代的风景名胜区共分为六大景区：九曲溪景区、武夷宫-大王峰景区、云窝-天游景区、虎啸岩-一线天景区、水帘洞-大红袍景区、遇林亭-莲花峰景区。<ref>{{cite book |title=福建省志·武夷山志 |pages=127 }}</ref>

自1979年建制至2008年，武夷山风景名胜区共接待海内外游客2430万人次，其中境外游客191.1万人次。<ref>{{cite web |title=武夷山风景名胜区建制30年接待游客逾2400万人次 |author=许雪毅 |url=http://www.xinhuanet.com/chinanews/2009-12/06/content_18423042.htm |publisher=新华网 }}</ref>2011年武夷山风景名胜区接待中外游客711万人次，并以旅游总收入122.6亿元列全国5A旅游景区第一位。<ref>{{cite web |title=武夷山去年旅游总收入破120亿列全国5A景区之首 |author=熊慎端 |url=http://www.gd.chinanews.com/2012/2012-02-09/2/177827.shtml |publisher=中国新闻网 }}</ref>

==九曲溪景区==
九曲溪发源于[[黄岗山|黄岗山]]南麓，全长62.8千米，流域面积534平方千米。自星村由西向东，历九曲十八弯，横贯景区，在武夷宫附近汇入[[崇阳溪|崇阳溪]]。古人游览九曲溪，多从武夷宫逆流而上，以武夷宫附近记为一曲，玉女峰下为二曲，直至星村为九曲。从星村至武夷宫，长约9.5千米，直线距离仅5千米，曲率达1.9<ref>{{cite book |title=福建省志·武夷山志 |pages=128 }}</ref>。沿途水清见底，山回溪转，移舟换景，是武夷山的精华所在。以朱熹的《九曲棹歌》为代表，历代文人对九曲溪歌咏不断。

由九曲顺流而下，沿途可见的景点有：<ref>{{cite book |title=福建省志·武夷山志 |pages=128-145 }}</ref>
* 九曲（星村—嶂岩）：白云岩、三教峰、磨盘石等
* 八曲（嶂岩—芙蓉滩）：烟际岩、双乳峰、鼓楼岩、涵翠岩、紫芝峰、象鼻岩、骆驼峰等
* 七曲（芙蓉滩—獭控滩）：北廊岩、天壶峰、三层峰、铸钱岩、东华岩、猴藏岩、城高岩、太姥岩等
* 六曲（獭控滩—老鸦滩）：响声岩、梧岩、晒布岩、天游峰、隐屏峰等
* 五曲（老鸦滩—古锥滩）：更衣台、天柱峰、晚对峰、仙迹岩、丹炉岩、茶灶石等
* 四曲（古锥滩—雷磕滩）：玉华峰、金谷岩、希真岩、小九曲、题诗岩、大藏峰、卧龙潭等
* 三曲（雷磕滩—水乐滩）：小藏峰、宴仙岩、仙钓台、车钱岩、升日峰、上升峰、仙游岩等
* 二曲（水乐滩—浴香潭）：仙馆岩、仙榜岩、玉女峰、妆镜台等
* 一曲（浴香潭—武夷宫）：铁板嶂、儒巾石、水光石、月岩、大王峰、幔亭峰、狮子峰等

==武夷宫—大王峰景区==
武夷宫始建于[[唐朝|唐]][[天宝|天宝]]年间，称天宝殿。[[五代|五代]]闽王[[王审知|王审知]]改称武夷观，祀武夷君。南唐改称会仙观。北宋绍圣二年（1095年）改称冲佑观，时为全国九大道观之一。元明清诸朝先后称万年宫、武夷宫。<ref>{{cite book |title=福建省志·武夷山志 |pages=147 }}</ref>

大王峰为武夷三十六名峰之首，上大下小，陡峭异常。据说此处为古时闽越先民酋长安放船棺之地，故而得名。<ref>{{cite book |title=福建省志·武夷山志 |pages=153 }}</ref>

==云窝—天游景区==
云窝-天游景区在九曲溪中段（五曲、六曲）北岸，有隐屏峰、云窝、天游峰、桃源洞等景点，与九曲溪景区同列为两大游客最多的景区。<ref>{{cite book |title=福建省志·武夷山志 |pages=158 }}</ref>

云窝在隐屏峰、天游峰及九曲溪三者之间，因清晨常有云雾弥漫，故而得名。北宋[[游酢|游酢]]在此筑舍，名为水云寮。北宋乡贤江贽又在此筑叔圭精舍。明代兵部侍郎[[陈省|陈省]]罢官后在叔圭精舍遗址上筑幼溪草庐，并留下大量摩崖石刻。<ref>{{cite book |title=福建省志·武夷山志 |pages=159-160 }}</ref>

隐屏峰是武夷三十六峰之一，南宋淳熙十年（1183年）朱熹在峰南建武夷精舍，2000年重建。[[刘子羽神道碑|刘子羽神道碑]]现已移入武夷精舍保存。<ref>{{cite book |title=福建省志·武夷山志 |pages=137 }}</ref>

天游峰和仙掌峰俱为武夷三十六峰之一，两者连为一体，顶部以胡麻涧相隔。胡麻涧保存有三十多方历代摩崖石刻。<ref>{{cite book |title=福建省志·武夷山志 |pages=167 }}</ref>

桃源洞又称小桃源，由九曲溪六曲北岸经松鼠涧北行，至豁然开朗的谷地即为桃源洞。谷内有20世纪90年代重建的开源堂道观和高16米的老子造像。<ref>{{cite book |title=福建省志·武夷山志 |pages=168-169 }}</ref>

桃源洞有小路通往三仰峰。三仰峰为武夷三十六峰之一，是风景名胜区内的最高峰，海拔717.7米。<ref>{{cite book |title=福建省志·武夷山志 |pages=169 }}</ref>

==虎啸岩—一线天景区==
虎啸岩和一线天景区在风景名胜区南部，古时多由二曲上岸进入虎啸岩，再到一线天，今有景区公路可直达两地。虎啸岩为武夷三十六名峰之一，四壁陡峭。山南有天成禅寺。<ref>{{cite book |title=福建省志·武夷山志 |pages=171 }}</ref>

灵岩为武夷三十六名峰之一，岩腹有三个岩洞：灵洞、伏羲洞、风洞，三洞有一条巨大的岩罅相通，长178米，最狭处仅0.3米，故称“一线天”。<ref>{{cite book |title=福建省志·武夷山志 |pages=173 }}</ref>

==水帘洞—大红袍景区==
水帘洞和大红袍景区在风景名胜区北部。水帘洞是一块巨大的内凹形岩壁，岩顶有瀑布飞流而下，岩壁内凹处有木构三贤祠，祀刘子翚、刘甫和朱熹三人。<ref>{{cite book |title=福建省志·武夷山志 |pages=175-176 }}</ref>

九龙窠是一条东西向的深长峡谷，[[武夷岩茶|武夷岩茶]]名品[[大红袍|大红袍]]的原生地就在九龙窠内。大红袍名丛共有六株，生长在岩壁上。九龙窠峡谷北侧高地上有天心永乐禅寺，已被列为[[武夷山市文物保护单位|武夷山市文物保护单位]]。

从水帘洞经章堂涧、慧苑寺、鹰嘴岩、流香涧可到达九龙窠。章堂涧南侧有丹霞嶂，为武夷三十六峰之一，山腰处有岩洞，洞内有清代崖居遗构，俗称“天车架”<ref>{{cite book |title=福建省志·武夷山志 |pages=177 }}</ref>。丹霞嶂崖居已被列为[[福建省文物保护单位|福建省文物保护单位]]。鹰嘴岩亦为武夷三十六峰之一，岩顶斜尖，酷似鹰嘴，故而得名。慧苑寺也是武夷山市文物保护单位。

==遇林亭—莲花峰景区==
遇林亭-莲花峰景区在风景名胜区的西北部，主要有遇林亭窑址、白岩崖墓、莲花峰、佛应岩、佛国岩、弥陀岩、龙峰、碧石岩等景点。<ref>{{cite book |title=福建省志·武夷山志 |pages=183-186 }}</ref>
