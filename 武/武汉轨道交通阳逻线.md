'''武汉轨道交通阳逻线'''（又称'''武汉轨道交通21号线'''，英语：'''Wuhan Metro Yangluo Line'''或'''Wuhan Metro Line 21'''）是[[武汉|武汉]]首条郊区地铁线路，连接[[武汉市|武汉市]][[新洲区|新洲区]][[阳逻|阳逻]]和[[汉口|汉口]]市区，沿途经过[[江岸区|江岸区]]、[[黄陂区|黄陂区]]、[[新洲区|新洲区]]。<ref>[http://dx.cnhubei.com/dongxiang/831768517568368640 武汉地铁阳逻线将提前1年通车 7号线一期将延期开通]2017-02-15 湖北日报</ref><ref>[http://www.hb.xinhuanet.com/2014-12/28/c_1113800036.htm 武汉地铁21和27号线昨同日开建 连黄陂新洲通江夏] {{webarchive|url=https://web.archive.org/web/20150924025456/http://www.hb.xinhuanet.com/2014-12/28/c_1113800036.htm |date=2015-09-24 }}</ref>

== 工程介绍 ==
===一期工程===
一期工程线路全长35km，其中[[地下线|地下线]]及暗埋段9.33km，敞开段及地面线0.282km，[[高架|高架]]线25.40km。设站16座，其中地下站6座，位于[[江岸区|江岸区]]；高架站10座，位于[[黄陂区|黄陂区]]和[[新洲区|新洲区]]。6座地下站分别是[[后湖大道站|后湖大道站]]、[[百步亭花园路站|百步亭花园路站]]、[[新荣客运站|新荣客运站]]、[[幸福湾站|幸福湾站]]、[[朱家河站|朱家河站]]、谌家矶站；10座高架站分别是青龙站、高车站、武湖站、沙口站、军民村站、武生站、平江路站、阳逻开发区站、施岗站、金台站。阳逻线在武湖泵站河以东、汉施公路以南地块设武湖停车场，在新洲倒水河西侧、汉施公路以北设倒水河车辆段。控制中心设于武湖综合体（与武湖站合建），为地铁10号线、18号线、阳逻线、22号线4条线共用。全线新建2个主变电站，分别位于武湖停车场和施岗站附近。

===未来发展===
阳逻线东西两端，未来还将分别延伸到[[中一路站|中一路站]]（对接地铁8号线，并与之换乘）和[[邾城|邾城]]方向。<ref>[http://news.cnhubei.com/xw/wuhan/201412/t3136378.shtml 新洲江夏同日开建地铁 21、27号线2018年底通车(图)]2014-12-28 荆楚网</ref>

==影响==
===换乘1号线方案暂缓===
[[新荣站|新荣站]]原计划为阳逻线与[[武汉轨道交通1号线|1号线]]的[[换乘站|换乘站]]，但由于阳逻线车站南面的商业综合体正在施工，换乘通道无法建设，阳逻线与1号线的换乘方案搁浅，并且阳逻线的新荣站已更名为[[新荣客运站|新荣客运站]]，与1号线的新荣站以示区别。<ref>[http://liuyan.cjn.cn/threads/content?tid=12309 1号线与21号线新荣站如何换乘]武汉城市留言板 2017-05-31</ref><ref name="6条新地铁线 73座车站有了站名">[http://hb.sina.com.cn/news/zwzl/2017-06-13/detail-ifyfzfyz3599527.shtml?from=hb_ydph 张晟. 6条新地铁线 73座车站有了站名<nowiki>[N]</nowiki>. 长江日报, 2017年6月13日(4).]</ref>此为[[武汉地铁|武汉地铁]]线网中首个两线相交而无法换乘的案例。现今，乘客搭乘阳逻线可在[[后湖大道站|后湖大道站]]换乘[[武汉轨道交通3号线|3号线]]前往市区其他地方，亦可在[[新荣客运站|新荣客运站]]出站转乘1号线。待阳逻线车站南面的商业综合体建成后，将开通一个换乘通道，与1号线新荣站实现站内换乘，预计2021年落成。<ref>{{Cite web |url =http://news.cjn.cn/sywh/201712/t3126489.htm|title =新地铁有多处便民尖板眼 新增868个“P+R”停车位 |author = 黎先才, 袁永华|date = 2017-12-27 |publisher = 楚天都市报 |language = 中文}}</ref>

===线路名称变更===
轨道交通阳逻线曾经历过S1线、21号线等多次名称变化，易造成市民和相关职能部门的混淆和误会。按照武汉市委市政府要求，为提高市民辨识度，21号线工程项目名称已易名为阳逻线。<ref>{{Cite web |url = http://liuyan.cjn.cn/threads/content?tid=52193|title =对武汉21号线正式名称的建议分析|author =武汉地铁集团 |date = 2017-10-13 |publisher = 武汉城市留言板 |language = 中文 |accessdate = 2017-10-15}}</ref>因此，阳逻线成为武汉首条以汉字命名的地铁线路。

阳逻线列车为[[A型车|A型车]]，在运营初期，阳逻线列车将采用4节编组，远期根据客流的增长，列车会扩编至6节编组，届时最大载客量为2340人。列车最高运营时速为100公里/小时，4节车厢均有动力，两节列车是全动力，另外两节则是半动力。车内顶部“移植”了[[武汉地铁6号线|地铁6号线]][[汉正街站|汉正街站]]的蓝天白云设计方案，让乘客感觉即便身处地下，抬头也可见蓝天白云。阳逻线列车车载电子显示屏也远远大于现有的地铁车载电子显示屏，达到了24寸，其他线路显示屏多为19-20寸。本线共配备25列列车，其中15列在武汉生产、10列在株洲生产。<ref>[http://cjrb.cjn.cn/html/2017-07/20/content_5625542.htm 张晟. 年底载蓝天白云穿越长江新城<nowiki>[N]</nowiki>. 长江日报, 2017年7月20日(2).]</ref>
