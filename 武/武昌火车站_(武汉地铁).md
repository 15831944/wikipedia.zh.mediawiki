{{武昌火车站}}
'''武昌火车站'''是[[武汉地铁|武汉轨道交通]][[武汉轨道交通4号线|4号线]]、[[武汉轨道交通7号线|7号线]]的[[换乘车站|换乘车站]]，位于[[武汉市|武汉市]][[武昌区|武昌区]]。<ref name="武汉61座地铁站名已定">{{cite web |url = http://news.cnhubei.com/xw/wuhan/201208/t2187330.shtml |title = 武汉地铁4号线实现全线动工 串起三大火车站 |accessdate=2012-03-23}}</ref>

== 車站 ==
轨道交通4、7号线于武昌站西广场设站，轨道交通11、12号线于武昌站东广场设站。其中4、7号线车站通道换乘，11、12号线车站通道换乘，形成四线两两换乘格局。武昌站东、西广场目前被国铁站场分隔，同时也阻隔了轨道交通4、7号线与轨道交通11、12号线的站内换乘路径。近期换乘可通过东、西广场既有的社会联络通道进行站外换乘。远期武昌火车站周边整体规划如有调整，将结合规划情况，考虑改造国铁站场地下空间，将4、7号线与11、12号线车站地下连通，实现站内换乘。<ref>范理纯. 武汉地铁武昌火车站换乘站设计方案研究[J]. 城市建设理论研究: 电子版, 2014(8).</ref><ref>[http://news.cnhubei.com/xw/wuhan/201701/t3775593.shtml 7号线武昌火车站建设如火如荼 220余名工人过年不回家]2017-01-18 武汉晨报</ref>
