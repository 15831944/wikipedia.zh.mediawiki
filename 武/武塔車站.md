'''武塔車站'''位於[[臺灣|臺灣]][[宜蘭縣|宜蘭縣]][[南澳鄉|南澳鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[北迴線|北迴線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]兩座。

== 利用狀況 ==
*[[武塔車站|武塔]]～[[漢本車站|漢本]]間的13.2公里路程，為[[北迴線|北迴線]]中的最遠站距。
*目前降為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，由[[南澳車站|南澳站]]管理。
*僅有[[臺鐵區間車|區間車]]停靠。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

==公車資訊==

== 歷史 ==
*1980年2月1日：[[北迴線|北迴線]]正式營業時設站。
*2001年6月1日：由三等站降為甲種簡易站，並指定由[[南澳車站|南澳站]]管理。
*2003年6月13日：由甲種簡易站降為招呼站。
*2016年3月1日：啟用多卡通刷卡機。
