'''武吉免登'''（{{lang-ms|'''Bukit Bintang'''}}，亦稱'''星光大道'''或'''金三角'''）是[[吉隆坡|吉隆坡]]主要的商業和娛樂場所集中地。區內集合了多家大型購物中心與國際飯店，包括了:[[金河廣場|金河廣場]]（Sungei Wang Plaza）、[[武吉免登購物中心|武吉免登購物中心]]（BB Plaza）、[[樂天廣場|樂天廣場]]（Lot 10）、[[成功時代廣場|成功時代廣場]]（Berjaya Times Square）、[[劉蝶廣場|劉蝶廣場]]（Low Yat Plaza）、[[升禧廣場|升禧廣場]]（Starhill Gallery）、[[華氏88廣場|華氏88廣場]]（Fahrenheit 88）等，以及沿著武吉免登路（Jalan Bukit Bintang）及其巷子內的數百間店鋪。

== 地理位置 ==
'''武吉免登'''位於[[吉隆坡|吉隆坡]]市中心，現在所指'''武吉免登'''主要是由[[燕美路|燕美路]]（Jalan Imbi）、[[蘇丹依斯邁路|蘇丹依斯邁路]]（Jalan Sultan Ismail）、[[武吉免登路|武吉免登路]]（Jalan Bukit Bintang）和[[拉惹朱蘭路|拉惹朱蘭路]]（Jalan Raja Chulan）所圍成的區域。<ref name="enanyang.my">http://www.enanyang.my/news/20160925/武吉免登重建添魅力/</ref>

== 歷史 ==
武吉免登的馬來名稱為「Bukit Bintang」，「Bukit」的[[中文|中文]]意思為山，而「Bintang」的意思為'''星星'''，所以'''武吉免登'''在中文可直譯為“星星之丘”。這個名稱的來源於座落在現在的[[金河廣場|金河廣場]]和[[武吉免登購物中心|武吉免登購物中心]]原址的中華遊藝場。在戰後時期，它曾經因為這裡的夜總會、商店、電影院和露天市集而成為[[吉隆坡|吉隆坡]]最時髦的地區。在70年代由於其他地區的娛樂場所兴起，中華遊藝場因此沒落而倒閉。在1977，當時全馬最大的購物中心[[金河廣場|金河廣場]]的營業，武吉免登才重拾往日的光輝，進而成為全馬最主要也是最著名的購物商圈。<ref name="enanyang.my"/>

97至98年間，因為高檔的[[升禧廣場|升禧廣場]]開幕，[[吉隆坡|吉隆坡]]市政府將樂天廣場至[[升禧廣場|升禧廣場]]的武吉免登規劃為'''星光大道'''（Bintang Walk）。在馬路旁設置許多的露天咖啡亭和酒吧，行成一條連接多條購物中心的行人步道。<ref name="enanyang.my"/>

2003年，隨著[[吉隆坡單軌|吉隆坡單軌]]的開始營運和全馬面積最大及世界排名第7的[[成功時代廣場|成功時代廣場]]開幕，進而帶動更多的人潮往武吉免登流去，使該地成為[[吉隆坡|吉隆坡]]最主要的旅遊區。在[[911事件|911事件]]發生後，許多的[[中東|中東]]遊客開始到馬旅遊，政府為了吸引更多的[[中東|中東]]游客，而在[[蘇丹依斯邁路|蘇丹依斯邁路]]旁，設置了一個小型的阿拉伯公園。<ref name="enanyang.my"/>

2007年，[[柏威年廣場|柏威年廣場]]商場的開幕，使武吉免登成為一個全面型的購物商圈，外國品牌進駐大馬市場的首選地區。<ref name="enanyang.my"/>

2012年，隨著[[城中城－武吉免登空橋系統|城中城－武吉免登空橋系統]]的啟用，馬來西亞觀光局與武吉免登和[[城中城|城中城]]（KLCC）的13個購物廣場、酒店等單位，在2012年2月一同成立武吉免登－城中城旅遊協會（BBKLCC），希望兩個商圈能合二為一，將[[吉隆坡|吉隆坡]]打造成為區域的購物樞紐。2017年，前[[半山芭监狱|半山芭监狱]]被改造为武吉免登城市中心（BBCC）－一座综合发展项目，使武吉免登在未来将迎来新面貌。<ref name="enanyang.my"/>

== 政治 ==
武吉免登同時也是馬來西亞下議院的一個選區，屬吉隆坡[[聯邦直轄區|聯邦直轄區]]（P120），現時國席議員為[[希望聯盟|希望聯盟]][[民主行動黨|民主行動黨]]的[[方貴倫|方貴倫]]<ref>{{cite web|url=https://dapmalaysia.org/cn/about-us/elected-representatives/parliment/|title=国会议员 <!-- parliment -->|publisher=[[民行党|行动党]]}}</ref>。

== 治安 ==
武吉免登因人潮眾多，且擁有許多的小巷和老舊大樓，加上外勞大量進駐在附近的住宅區或店屋樓上，這裡仍有不少的色情行業藏匿在其中。在2014年10月9日的凌晨，武吉免登路太陽大廈的夜總會懷疑是黑幫尋仇，遭手榴彈襲擊，造成14人受傷，1人死亡的爆炸事件<ref>{{cite news|url=http://news.sinchew.com.my/node/390612|title=武吉免登爆炸案．代客泊车员伤重毙命|date=2014-10-09|publisher=[[星洲网|星洲网]]}}</ref> 。
