'''武藏中原站'''（{{jpn|j=武蔵中原駅|hg=むさしなかはらえき|rm=Musashi-Nakahara eki|f=y}}）是一个位于[[神奈川县|神奈川县]][[川崎市|川崎市]][[中原区_(日本)|中原区]]上小田中六丁目，属于[[东日本旅客铁道|东日本旅客铁道]]（JR东日本）[[南武線|南武線]]的[[铁路车站|铁路车站]]。

== 历史 ==
* [[1927年|1927年]]（[[昭和|昭和]]2年）[[3月9日|3月9日]]：伴随南武铁道線[[川崎站|川崎站]] - [[登户站|登户站]]間开通开始营业。
* [[1944年|1944年]]（昭和19年）[[4月1日|4月1日]]：南武铁道線国有化，成为[[日本国有鉄道|国铁]]南武線的车站。
* [[1961年|1961年]]（昭和47年）[[1月16日|1月16日]]：停止办理客运业务。
* [[1987年|1987年]]（昭和62年）4月1日：[[国铁分割民营化|国铁分割民营化]]，成为东日本旅客铁道的车站。
* [[1990年|1990年]]（[[平成|平成]]2年）12月20日：随着南武线该段高架化，车站向武藏小杉站方向移动0.2公里。
* [[2001年|2001年]]（平成13年）[[11月18日|11月18日]]：IC卡[[Suica|Suica]]启用。
* [[2005年|2005年]]（平成17年）[[10月2日|10月2日]]：车站自動放送由東海道型变更为ATOS III期型。
* [[2011年|2011年]]（平成23年）[[4月9日|4月9日]]：南武線快速电车开始运营。该站为快速电车停车站。

== 车站结构 ==
该站为拥有2面4线岛式月台的高架车站。外侧线路主要用于客运，内侧线路主要用于车辆进出中原检车区和快速电车的待避线。

== 使用情况 ==
2016年度1日平均上车人次为'''34,909人'''。在南武線内的车站中排名第8位，JR东日本的车站中排名第123位。该站的平均上车人次在南武線的非换乘站中仅次于[[武藏新城站|武藏新城站]]，比属于换乘站的[[稻田堤站|稻田堤站]]要多。
