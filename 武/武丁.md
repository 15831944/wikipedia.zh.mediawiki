'''武丁'''（约前1250年-前1192年在位），子姓，根据《[[今本竹书纪年|今本竹书纪年]]》名昭，是中国商朝第23位国王，共在位59年，“武丁”是他逝世后获得的[[諡号|諡号]]，殷末又追谥庙号为“高宗”（但在[[殷墟|殷墟]]卜辞中没有该庙号）。他是商王敛（[[小乙_(商朝)|小乙]]）之子，商王旬（[[盘庚|盘庚]]）、商王颂（[[小辛|小辛]]）的侄子。

== 在位年數 ==
現今流傳文獻記載的武丁在位年數有3種說法：

*在位五十五年，《[[史記|史記]]·魯世家》引據〈毋逸〉-{云}-。
*在位五十九年，《[[尚書|尚書]]·無逸篇》，《今本[[竹書紀年|竹書紀年]]》、《[[帝王世紀|帝王世紀]]》、《[[皇極經世|皇極經世]]》、《[[資治通鑑外紀|資治通鑑外紀]]》、《[[資治通鑑前編|資治通鑑前編]]》、《[[通志|通志]]》、《[[文獻通考|文獻通考]]》均同《尚書》。
*在位百年，漢《[[熹平石經|熹平石經]]》、《論衡·氣壽篇》。

== 生平 ==

小乙时期，殷商国运已经衰微。武丁即位后，他跟随老臣师般（[[甘盘|甘盘]]）学习排布兵阵，他还亲自提拔了侯雀（[[傅说|傅说]]）、望乘、沚瞂、藉臣、戉、[[禽匕|禽匕]]（小臣毕）、西吏旨、仓侯虎、侯告等一大批臣子，又让[[妇妌|妇妌]]帮助发展生产力，[[妇好|妇好]]、[[妇癸|妇癸]]等女将统帅军队，使得商朝政治清明。

[[大彭国|大彭国]]、[[豕韦国|豕韦国]]曾经是[[商汤|商汤]]时的[[方伯|方伯]]，它们逐渐发展壮大，便不向商王朝进贡，武丁便起兵南征，一举攻灭了这两个国家<ref>《国语·郑语》：大彭、豕韦，商灭之。 甲骨文中也有相应记载</ref>。荆楚地区挑起叛乱，武丁亲自率军深入其境，荡平了叛乱以后，又认真地安抚、教导当地人民<ref>《诗经·商颂·殷武》</ref>。

维护国家统一的同时，武丁还大大开拓了商朝的疆土。武丁先后讨伐了土方、贡方、基方、苟方、人方、鬼方<ref>《易卦爻辞》上说：“高宗(武丁)伐鬼方，三年克之。”甲骨文中有“鬼方易”的记载</ref>等游牧民族，总共征服了81个敌国和叛乱不朝的方国。武丁又派小臣毕率领众多罪犯到这些新的领土上开垦荒地，大大发展了商朝的经济。衰落的商朝重新振兴，史称“[[武丁中兴|武丁中兴]]”或“武丁盛世”。

武丁从大邑商（洹北商城）迁都到洹水南岸的中商，就是现在[[殷墟|殷墟]]的主体，发展了殷墟时代。武丁时期出土的文物很多，[[甲骨文|甲骨文]]中有70%都写成于武丁时期，殷墟中超过四分之三的[[人牲|人牲]]都集中在武丁时期，这时开始出现铜、锡、铅三元合金，琢玉工艺也得到很大发展。

== 高宗肜日 ==

武丁祭祀[[商汤|商汤]]的一天，有一只野鸡在鼎耳上鸣叫，武丁认为这是不祥之兆而惊恐。長子[[祖己|祖己]]認為：“要先宽解君王的心，然后纠正他祭祀的事。”于是向武丁上奏：“上天监视下民，赞美他们合宜行事。上天赐给人的年寿有长有短，并不是上天使人夭折，而是有些人自己断绝自己的性命。有些人有不好的品德，有不顺从天意的罪过。上天已经发出命令纠正他们不好的品德，您说：‘要怎么样呢？’啊！先王继承帝位被百姓敬重，无非都是老天的后代，在祭祀的时候，近亲中的祭品不要过于丰厚啦！”武丁听了以后，便改革了祭祀制度，减少了祭祀的供品数量，避免了奢侈<ref>《尚书·高宗肜日》</ref>。

== 评价 ==

殷人将其尊为'''高宗'''，后世的人又称他是“殷之大仁”。在中国的传统文献《尚书》、《逸周书》、《诗经》、《史记》、《国语》、《吕氏春秋》等书中，无一不对他有很高的评价，认为他是一个虚怀若谷、宽厚仁慈、为民服务的优秀政治家。武丁是[[先秦|先秦]]时期最有作为的君主。但是，武丁也杀过大量战俘，[[殷墟|殷墟]]中70%以上的[[人牲|人牲]]都发生在武丁时期。
