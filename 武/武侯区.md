'''武侯区'''是[[中华人民共和国|中华人民共和国]][[四川省|四川省]][[成都市|成都市]]的市辖区，同时也是成都市中心[[五城区|五城区]]之一，设立于1990年。武侯区因其辖区内的[[成都武侯祠|武侯祠]]（[[诸葛亮|诸葛亮]]）而得名，是[[以人名命名的中国地名列表|以人名命名的中华人民共和国县级以上行政区之一]]，[[汶川大地震|2008年大地震]]期间武侯祠曾作为群众临时避难场所。2014年武侯区地区生产总值为739.32亿元，经济增长以[[第三产业|第三产业]]为主。

武侯区地理坐标[[北纬|北纬]]30°34′—30°39′与[[东经|东经]]103°56′—104°05′之间，东西长约13千米，南北宽约10千米，全区面积76.56平方公里；人口108.3万人（2010年），成都中心城区中人口密度最大。武侯区以北毗邻[[青羊区|青羊区]]，南与[[双流区|双流区]]和成都成都高新区接壤，以西隔河相望[[锦江区|锦江区]]。境内最著名的为武侯祠，其主体建筑是在清朝时期的重建，祠堂最早历史追溯至公元223年。在明朝与纪念[[刘备|刘备]]的“汉昭烈庙”合并一起，形成唯一的一座君臣合庙。作为一知名旅游景点的[[锦里古街|锦里古街]]也位于武侯区，毗邻武侯祠。

== 历史 ==
成都所在的[[成都平原|成都平原]]养蚕缫丝历史悠久，在[[汉代|汉代]]时开始织棉以其所出的[[蜀锦|蜀锦]]最为有名。位于武侯区的簇桥被认为是[[南方丝绸之路|南方丝绸之路]]的起点之一和四川丝绸交易的一个中心，簇桥之名也来自于蚕丝交易，在唐宋时被称作[[茧|茧]]桥，后以喂蚕的食物（统称作簇）命名沿用至今<ref>{{cite news|author1=郑光路|title=清末南方丝绸之路起点在簇桥|url=http://www.cdrb.com.cn/html/2014-07/26/content_2088488.htm|accessdate=2016-01-19|agency=成都日报|publisher=四川新闻网|date=2014-07-26|deadurl=yes|archiveurl=https://web.archive.org/web/20160125201051/http://www.cdrb.com.cn/html/2014-07/26/content_2088488.htm|archivedate=2016-01-25}}</ref>。簇桥也是自汉一直到[[清代|清代]]都是长盛不衰的商业重镇{{cfn|区志|y=2011|p=4}}。[[秦昭王|秦昭王]]在位时（公元公元前256～前251年），蜀守[[李冰|李冰]]在今武侯区北的老南门修建了跨越[[锦江_(岷江支流)|锦江]]的[[长星桥|长星桥]]，成为连接城南的重要陆路通道，促进了当地商贸的发展。此桥后又名万里桥、老南门大桥<ref name="one">{{cite web|author1=武侯区地方志编纂委员会办公室|title=认识武侯 > 历史文化>桥名文化|url=http://www.cdwh.gov.cn/index.php/?cid=1491|website=武侯区人民政府网站|accessdate=2016-01-19|deadurl=yes|archiveurl=https://web.archive.org/web/20160125041615/http://www.cdwh.gov.cn/index.php/?cid=1491|archivedate=2016年1月25日|df=}}</ref>。在秦代武侯这一区域隶属蜀郡成都县。汉代属益州蜀郡成都县。

[[蜀汉|蜀汉]][[章武|章武]]三年（公元223年）[[刘备|刘备]]逝世，安葬在修建在成都南郊（武侯区境内）的[[汉惠陵|惠陵]]，根据当时的制度，还修建了祭祀刘备的祠庙即汉昭烈庙。在南北朝时期（公元5世纪左右）人们将位于成都少城的祭祀诸葛亮的祠堂迁入惠陵、汉昭烈庙旁，形成今武侯祠的前身<ref>{{cite news|author1=吴云霞|title=昭烈庙为何又称武侯祠|url=http://www.cdrb.com.cn/html/2015-05/16/content_2201770.htm|accessdate=2016-01-18|agency=成都日报|publisher=四川新闻网|date=2015-05-16|deadurl=yes|archiveurl=https://web.archive.org/web/20160818083847/http://www.cdrb.com.cn/html/2015-05/16/content_2201770.htm|archivedate=2016-08-18}}</ref>。[[明|明]][[洪武|洪武]]二十三至二十四年蜀王[[朱椿|朱椿]]以强化自身权力为由将专门祭祀诸葛亮的武侯祠与汉昭烈墓和惠陵合并，形成三者合一局面。

成都市中心的区划原为东城区、西城区和金牛区，1990年9月国务院批准后调整了成都市中心的区划，以境内武侯祠而得名的武侯区随之诞生。

== 地理 ==
武侯区地势平坦，平均海拔502.5米，由西北向东南倾斜。境内平原由沙漫滩、[[冲积扇|冲积扇]]、扇状平原等平原组成。全区地质结构主要为第四纪松散堆积物。全区土地面积在2005年为75.36平方公里。武侯区气候属于中亚热带季风性湿润气候，年平均气温在16°C至18.6°C间。七月为当地最热月，平均气温在24.9°C至26.1°C间，极端日高温在24°C至38.7°C间。

*浆洗街街道、望江路街道、玉林街道、跳伞塔街道、火车南站街道、双楠街道、晋阳街道、红牌楼街道、机投桥街道、金花桥街道、簇锦街道、簇桥街道、华兴街道。
*由成都高新区托管：芳草街道、肖家河街道、石羊场街道、桂溪街道

==著名景点==
武侯区得名于武侯祠，是文化悠久，并且富有现代气息的地区。区内名胜古迹众多：
望江楼公园（薛涛井、望江楼）、武侯祠、锦里。另外，还有江安河生态公园、映月公园、江安河郊野公园、南郊公园、丽都公园等公园。参考资料来源<ref>{{Cite web|url=http://www.cdwh.gov.cn/index.php/?cid=351|title=旅游景点|author=成都市武侯区人民政府门户网站|deadurl=yes|archiveurl=https://web.archive.org/web/20131219103349/http://www.cdwh.gov.cn/index.php/?cid=351|archivedate=2013年12月19日|df=}}</ref>

==经济建设==
2012年功能区竣工项目48个，建筑面积290多万平方米，建成面积达860多万平方米。人民南路科技商务区：新增保利中心等6个竣工项目，建筑面积达68万多平方米；红牌楼商圈：新增莱蒙置地广场等13个竣工项目，建筑面积达99万多平方米；红牌楼广场二期等21个项目加快建设，建筑面积达234万平方米。武侯新城：新增外贸鞋城等29个竣工项目，建筑面积达125万多平方米。环城生态区：初步完成区域控规调整，积极推进陆坝新型社区建设。参考资料来源<ref>{{Cite web|url=http://www.cdwh.gov.cn/index.php/?cid=348|title=经济建设|author=成都市武侯区人民政府门户网站|deadurl=yes|archiveurl=https://web.archive.org/web/20131219123451/http://www.cdwh.gov.cn/index.php/?cid=348|archivedate=2013年12月19日|df=}}</ref>


==社会事业 ==
积极实施“阳光圆梦”工程，区内没有一个孩子因贫困而失学。建成全国首个政府主导、社会参与的善工家园助残中心，为残疾人提供“量体裁衣”式个性化服务；在城区街道和流浪乞讨人员相对集中的社区建立救助管理工作站；在13个街道建立社区居家养老服务机构，积极探索社区低龄老年志愿者为高龄老人服务的居家养老模式。参考资料来源<ref>{{Cite web|url=http://www.cdwh.gov.cn/index.php/?cid=350|title=社会事业|author=成都市武侯区人民政府门户网站|deadurl=yes|archiveurl=https://web.archive.org/web/20131219124139/http://www.cdwh.gov.cn/index.php/?cid=350|archivedate=2013年12月19日|df=}}</ref>

==民俗民风 ==
武侯区有着它独具特色和魅力的民风民俗，其中社会民俗有成都大庙会，游喜神方等；生活民俗有衣着习俗，饮食习俗等；生产民俗有农谚等；经济民俗有商业规俗等。参考资料来源<ref>{{Cite web|url=http://www.cdwh.gov.cn/index.php/?cid=1490|title=民风民俗|author=成都市武侯区人民政府门户网站|deadurl=yes|archiveurl=https://web.archive.org/web/20131219122019/http://www.cdwh.gov.cn/index.php/?cid=1490|archivedate=2013年12月19日|df=}}</ref>

==历史文化 ==
武侯区位于北纬30°34′31″～30°39′49″，东经103°56′45″～104°05′33″，地形呈蝶状，平均海拔高度500多米，土地肥沃。辖区内属亚热带温润季风气候，四季分明，春季气温回升快，但冷空气活动频繁，易出现倒春寒。  
武侯区内历史文化悠久，尤其以三国蜀汉文化，建筑文化有著名的武侯祠，望江楼古建筑群等。参考资料来源<ref>{{Cite web|url=http://www.cdwh.gov.cn/index.php/?cid=1491|title=历史文化|author=成都市武侯区人民政府门户网站|deadurl=yes|archiveurl=https://web.archive.org/web/20160125041615/http://www.cdwh.gov.cn/index.php/?cid=1491|archivedate=2016年1月25日|df=}}</ref>
