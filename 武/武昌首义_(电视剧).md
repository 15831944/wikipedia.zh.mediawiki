《'''武昌首义'''》，中国大陆电视剧，导演[[李云亮|李云亮]]，主演[[李成儒|李成儒]]、[[潘琼|潘琼]]、[[徐光明|徐光明]]、[[张国民|张国民]]。

== 剧情 ==
该剧讲述1911年的10月9日到12月31日[[辛亥革命|辛亥革命]]推翻[[清朝|清朝]]清政府的过程。<ref>{{cite web |url=http://news.hexun.com/2011-10-02/133924064.html |title=电视剧《武昌首义》今晚播出 纪实风格浓烈 |accessdate= 2011年10月2日|publisher=和讯网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.sina.com.cn/c/cul/2006-08-31/10529900249s.shtml |title=电视剧《武昌首义》将与观众见面 |accessdate=  2006年8月31日|publisher= 新浪网|work= |language =zh-cn}}</ref>

== 花絮 ==
该剧是“纪念辛亥革命95周年”献礼剧。<ref>{{cite web |url=http://news.sina.com.cn/o/2010-01-21/062716967585s.shtml |title= 年内开拍“武昌首义”电视剧|accessdate=2010年1月21日 |publisher=新浪网 |work= |language =zh-cn}}</ref><ref>{{cite web |url= http://ent.sina.com.cn/x/2006-08-31/08011224547.html|title= 革命历史题材电视剧《武昌首义》将与观众见面|accessdate=2006年8月31日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

该剧中将详细展现“彭刘杨三烈士就义、床底拖出的‘大总统’、光复武汉三镇、刘家庙之战、黄兴拜将、火烧汉口”等史实。<ref>{{cite web |url=http://www.cnhubei.com/200608/ca1170467.htm |title=电视连续剧《武昌首义》央视下月播出 |accessdate=2006-09-29 |publisher=荆楚网 |work= |language =zh-cn}}</ref>

剧组原本打算邀请[[傅彪|傅彪]]出演黎元洪，但是傅彪患病，换成李成儒出演，李成儒出演的黎元洪带有“[[黑色幽默|黑色幽默]]”。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2005-12-21/0220935737.html |title=《武昌首义》浙江杀青 |accessdate= 2006年3月18日|publisher=新浪网 |work= |language =zh-cn}}</ref>
