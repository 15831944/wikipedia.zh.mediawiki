'''武来岸'''（{{lang-ms|Broga}}）是[[馬來西亞|馬來西亞]][[雪蘭莪州|雪蘭莪州]][[乌鲁冷岳县|乌鲁冷岳县]]的一个小镇，位于[[雪兰莪州|雪]][[森美兰州|森]]边界。

==历史==
武来岸新村早已是一个拥有150年历史的老镇，村里的石拿督庙（擁有140年历史）就是其明证。在1825年至1850年之间，英国陆续派人来武来岸地区探测锡米。之后英国人在1851年用铁船在拿督坳以铁船（Kampung Kapal）进行采锡的活动。在1870年，因武来岸原属森美兰州辖地，森州以及雪州的统治者为了解决边界纠纷签下了署协议。协议里清楚的列明；森州的武来岸、[[士毛月|士毛月]]和峇玲珑（Beranang）归雪州，而雪州的芦骨（Lukut）则归森州。根据历史记载，到了1920年，武来岸河繁忙的活动，逐渐形成了武来岸这个以采锡为主的小镇。武来岸的经济也因20世纪初橡胶业的冒起，而被带动了起来，但1941年日军侵占[[马来亚|马来亚]]之后便在雪州[[加影|加影]][[蕉赖|蕉赖]]和森州知知港进行大屠杀。战后，英殖民政府在1950年利用集中营方式把散居在周围的华民迁移集中在武来岸，并建立了武来岸新村，其中迁移人口达 993人。1954年，武来岸新村华小正式创办，为村内唯一一所华文学校。<ref>{{cite web|url=https://www.utar.edu.my/dssc/file/Broga.pdf|format=PDF|title=武来岸}}</ref>

== 行政 ==
武来岸是[[华人新村|华人新村]]中少数跨州的新村。位于雪州一侧的主城称为武来岸打仑新村，属于[[士毛月|士毛月]][[巫金|巫金]]；而位于森州的则称为武来岸新村，属于[[良冷|良冷]]。<ref>{{cite web|url=http://www.sinchew.com.my/node/1525492|title=武来岸村．山水鱼闻名|date=2014-11-20|publisher=}}</ref>武来岸政府诊所和区足球场位于雪州，而警察局和新村主要区域则位于森州。不过，除了大选投票及书信地址之间的差别外，当地村民都不分雪、森州之别。{{cn}}

== 人口 ==
武来岸主要族群为[[马来西亚华人|华族]]，而这里附近也有一小部分是原住民（Orang Asli）。

=== 宗教信仰 ===
这里的华人多数不是[[佛教|佛教]]就是[[道教|道教]]，原住民则为无宗教信仰，这里有一间[[道教|道教]]庙——石拿督庙，位于森美兰州一侧。{{cn}}

== 经济 ==
武来岸是一个以[[农业|农业]]和[[橡胶|橡胶]]为主的小镇，虽然森雪很多地方都已改种[[油棕|油棕]]树，但当地还是保留并继续栽种橡胶。{{cn}}

== 教育 ==
这里只有一间小学，即为武来岸新村华小，属[[森州|森州]]教育局管辖。中学方面则需前往[[良冷|良冷]]就读。
