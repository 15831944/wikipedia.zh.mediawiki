'''武藏小金井站'''（{{jpn|j=武蔵小金井駅|hg=むさしこがねいえき|rm=Musashi-koganei eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[小金井市|小金井市]]本町六丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[中央本線|中央本線]]的[[鐵路車站|鐵路車站]]。

包括此站的路段在運行系統上顯示為「[[中央線快速|中央線]]」。運行形態的詳細資料參見該條目。

==車站構造==
[[島式月台|島式月台]]2面4線的[[高架車站|高架車站]]。閘口共2個。

== 使用情況 ==
2016年度1日平均'''上車'''人次為'''61,035人'''，JR東日本管內車站當中排行第75位。然而，此數字比中央快速線東京-高尾之間不轉乘路線車站當中的[[三鷹站|三鷹站]]還多，中央快速線內24個車站當中排行第14位。

== 歷史 ==
* 1924年（[[大正|大正]]13年）4月4日 - 觀櫻假乘降場開業。 
* 1926年（大正15年）1月15日 - 停車場昇格為'''武藏小金井站'''。旅客營業開始。 
* 1959年（[[昭和|昭和]]34年）9月1日 - 構內的武藏小金井電車區完成。 
* 1987年（昭和62年）4月1日－[[國鐵分割民營化|國鐵分割民營化]]，本站由JR東日本接手管理。
* 2001年（[[平成|平成]]13年）11月18日－智能卡系統[[Suica|Suica]]正式投入服務。
* 2007年（平成19年）11月25日 - 武藏小金井電車區改稱[[豐田車輛中心|豐田車輛中心]][[武藏小金井派出所|武藏小金井派出所]]。
