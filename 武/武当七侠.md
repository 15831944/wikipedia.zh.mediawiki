'''武当七侠'''是[[金庸武侠小说|金庸武侠小说]]《[[倚天屠龙记|倚天屠龙记]]》中的人物。

== 概述 ==
在《[[倚天屠龍記|倚天屠龍記]]》中，武當七俠是指[[武當派|武當派]]祖師[[張三丰|張三丰]]的七位弟子，於武當享有崇高的地位，他們分別是：
*大弟子[[宋远桥|宋远桥]]：穩重踏實、沖淡謙和、恂恂儒雅，頗有君子風範，處事公道，江湖聲望地位極高。宋远桥有一子[[宋青书|宋青书]]，宋青书后来成为武当派的叛徒，并亲手杀死了七师叔莫声谷。
*二弟子[[俞莲舟|俞莲舟]]：為人深沉，喜怒不形於色，沉默寡言。武功為七人中第一。
*三弟子[[俞岱巖|俞岱巖]]：被西域金剛門的阿三以大力金剛指折斷四肢，成為廢人。
*四弟子[[张松溪|张松溪]]：足智多謀。
*五弟子[[張翠山|張翠山]]：[[張無忌|張無忌]]之父，外號「銀鉤鐵劃」，位列武當七俠第五，擅長寫[[書法|書法]]。
*六弟子[[殷梨亭|殷梨亭]]：性格懦弱，長相帶有稚氣，較晚熟，在第一版倚天屠龙记中，被称为殷利亨。他曾经与峨嵋派[[纪晓芙|纪晓芙]]订有婚约，后来又和纪晓芙的女儿[[杨不悔|杨不悔]]结婚。
*七弟子[[莫声谷|莫声谷]]：滿臉濃鬚，較早熟，被武当叛徒宋青书和[[成昆|成昆]]的徒弟[[陈友谅|陈友谅]]所杀。

== 史實 ==
武當派張三丰的七位弟子史上確有其人，唯六弟子殷梨亭原名應為「殷利亨」（取《[[易經|易經]]》「元亨利貞」之意）。
