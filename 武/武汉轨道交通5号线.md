'''武汉轨道交通5号线'''（又称'''武汉地铁5号线'''，英语：'''Wuhan Metro Line 5'''）是[[武汉地铁|武汉地铁]]的一条在建线路。全线均位于武昌地区且平行贴近[[长江|长江]]，线路南起南三环站，北至武汉火车站，途径[[洪山区|洪山区]]、[[武昌区|武昌区]]及[[青山区_(武汉市)|青山区]]。<ref>[http://www.hb.xinhuanet.com/2016-08/03/c_1119325546.htm 武汉地铁5号线本月开工 2020年年底通车]2016-8-3 长江日报</ref>

==线路简介==
轨道交通5号线南起南三环，主要沿烽胜路、白沙洲大道、复兴路、黄鹤楼南路、和平大道、工人村路、冶金大道、站前路敷设至武汉火车站，南北向贯穿武昌三区，是服务于武昌镇内沿江发展的重要骨架线路。5号线全长34.83km，共设车站25座，高架区间位于武昌南部的白沙洲和青菱地区，因为该处为[[岩溶地貌|岩溶区]]，地质条件恶劣，所以从最初计划的地下线路改为高架线路，全线工程才得以顺利推进。本线设车辆段和停车场各一处：其中工人村车辆段位于线路北端工人村东侧，与都市工业园站接轨，工人村车辆段承担5号线车辆的定修、周月修及临修任务，5号线列车的厂架修由7号线[[野芷湖车辆段|野芷湖车辆段]]承担；青菱停车场位于线路南端南三环站北侧，与南三环站接轨。控制中心设置在铁机路，与地铁4、8号线共用。5号线还是武汉地铁首条全自动驾驶线路，采用A型车，6节编组，DC1500V第三轨受电。全线预计2020年12开通，预计投资280.2223亿元。<ref>[http://www.hb.xinhuanet.com/2016-08/03/c_1119325546.htm 武汉地铁5号线本月开工 预计2020年底通车]2016-08-03 长江日报</ref>

==影响==
===填补[[青山区_(武汉市)|青山区]]轨道交通空白===
地铁4号线一期于2013年12月开通，4号线一期走向为武昌站至武汉站，其中岳家嘴-武汉站区间沿着团结大道往北至武汉火车站，青山区的交通大动脉和平大道与友谊大道与之平行。但是4号线没有一个车站设置在青山区，而是在洪山区，与青山打了个“擦边球”，所以4号线距离青山区人口腹地较远，并不能有效帮助青山区居民出行。

在早期规划中，5号线沿着和平大道贯穿青山区，但是由于5号线并非过江线路，且与4号线部分平行，所以5号线一直被搁置，如此下去青山区将在很长一段时间内无法摆脱没有地铁的困境。在人大代表和市民的呼吁下，政府决定将5号线的建设计划提前2至3年，以解决青山区53万人的出行<ref>[http://hb.sina.com.cn/news/sh/2014-01-07/0906135418.html 武汉地铁5号线有望提前开工 明年定具体开工时间]2014年1月7日 荆楚网-湖北日报</ref><ref name="武汉地铁5号线被排在2017年以后开工 网友不满">{{cite web | url=http://hb.qq.com/a/20100809/000506.htm | title=武汉地铁5号线被排在2017年以后开工 网友不满 | accessdate=2012-07-31}}</ref>。

===江盛路站、江民路站合并===
轨道交通5号线江盛路站原计划位于电信大楼（中国电信白沙洲营业厅）所在地，建造该站需要将电信大楼拆除，由于拆迁电信大楼面临难度大、成本高、周期长等问题，而轨道交通5号线建设工期不能拖延，地铁集团于是退而求其次，通过调整5号线车站选址，避让该电信大楼，大致方案为将江盛路站北移、江民路站南移合并成一座新车站（站名暂定为江民路站），并由高架改为地下，新车站位于城南新居小区。同时，本次调整还顺带更改江国路站选址，该站被南移100余米，意在缩小江国路站至青菱站的站间距。<ref>{{Cite web |url = http://liuyan.cjn.cn/threads/content?tid=42690 |title = 地铁五号线目前的方案有哪些? |author = 武汉地铁集团 |date = 2017-09-20 |publisher = 武汉城市留言板 |language = 中文 |accessdate = 2017-10-15}}</ref>

===线路色更改===
5号线之前的线路色是“扬子蓝”，与江对岸1号线的“地铁蓝”遥相呼应。后来考虑到5号线经过武昌古城——[[武昌起义|武昌起义]]的发源地，为突显武汉首义文化的特色，遂将5号线的线路色更改为首义红。<ref>{{Cite web |url =http://liuyan.cjn.cn/threads/content?tid=54545|title = 请问5号线的标识色更换了吗？ |author = 武汉地铁集团  |date = 2017-10-24|publisher = 武汉城市留言板 |language = 中文}}</ref>
