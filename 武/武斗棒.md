'''武斗棒'''{{NoteTag|{{lang|ja|ゲバルト棒}}在汉语里并没有正式的译名。但很多提到此物的翻译文献常将其翻译为'''武斗棒'''<ref name ="to">[https://www.marxists.org/chinese/reference-books/mia-chinese-japanesetroskism2008.htm 盐川喜信（托洛茨基研究所所长）特别采访]</ref><ref>[http://www.rylww.com/paper/sort10/down-3300.html 日本人的思想构造[日语翻译中文]日本人の意識構造] - [http://www.rylww.com 日语论文网] 本页面中更可以看到直接的中日对照翻译</ref>}}（{{jpn|j='''ゲバルト棒'''|hg=ゲバルトぼう|rm=Gebaruto Bou}}，常简称{{lang|ja|'''ゲバ棒'''}}）是日本[[安保斗争|安保斗争]]时左翼人士使用的棍状简易武器。

== 概说 ==
武斗棒因安保斗争而出现。在安保斗争早期，赤手空拳的示威学生面对全副武装的警方{{link-ja|機動隊|機動隊|机动队员}}时吃了很大的亏。<ref name ="to"/>一些学生因此将示威标语牌的把手拆掉，做为和警方机动队对抗的武器。<ref name ="ka">[http://www.z930.com/1968_6.html ゲバ棒] - [http://www.z930.com/index.html 1968年全共闘だった時代]</ref>参与示威的学生将这种简易棍棒起名为「{{lang|ja|ゲバルト棒}}」。{{lang|ja|ゲバルト}}来自德语词汇{{lang|de|Gewalt}}，意为暴力。据说当时的学生喜欢用罕见的外来语来命名事物。这个名字很快流行开来。<ref name ="ka"/>武斗棒一般来自拆卸的公共设施，如门框、长椅等，因此材料和尺寸较不统一。但一般为日本常见的方形木棍（{{lang|ja|角材}}），也有少量铁和铝棍。长度最长的可以达到2米。后来因为需求量大，也有学生组织在木材店专门订制。武斗棒除了用来和机动队搏斗，也可以用来悬挂旗帜。有些武斗棒的前部会安装[[钉子|钉子]]，以提高杀伤力。

武斗棒是非常简陋的武器。他很不结实，在强烈的冲击下很容易弯折或折断。<ref name ="ka"/>因此基本上是一次性使用的武器。但其效果也很明显——被攻击者完全有可能在打击下受到严重的伤害。<ref>《敬重与惜别》 p32</ref>因此对于缺少武器的学生来说仍然是实用的武器。

武斗棒在{{link-ja|羽田事件|羽田事件|第一次羽田斗争}}中首次出场，随后成为学生运动反抗当局或内斗<ref name ="ka"/>的主要武器之一。赞助左翼组织武斗棒在当时也成为支持安保运动的一种方式。今日，武斗棒、[[燃烧瓶|燃烧瓶]]（{{lang|ja|火炎瓶}}）和写有组织名称的[[盔|安全盔]]（{{lang|ja|安全ヘルメット}}）已成为[[昭和|昭和]]时期学生运动的标志。
