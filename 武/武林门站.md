'''武林门站'''是[[杭州地铁2号线|杭州地铁2号线]]的一个车站，位于[[环城西路|环城西路]]和[[体育场路_(杭州)|体育场路]]交叉口以南，[[武林门|武林门]]附近。<ref>{{cite web |url=http://www.zj.xinhuanet.com/newscenter/focus/2013-09/06/c_117250126.htm |language= |title=杭州地铁2号线武林门站要"南移" 沈塘桥站也有微调 |publisher=都市快报 |date=2013-09-06 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20131219100352/http://www.zj.xinhuanet.com/newscenter/focus/2013-09/06/c_117250126.htm |archivedate=2013-12-19 }}</ref>2号线车站于2017年7月3日开通运营，未来的[[杭州地铁3号线|3号线]]也会经过此站，成为两线的换乘站。
[[File:20170703武林门站与大字壁.jpg|thumb]]
== 结构 ==
*2号线西北段武林门站位于环城西路，南至桃花港桥，北至体育场路口，为地下两层双柱三跨箱型框架结构标准站。车站总长190.2米，标准段净宽21.7米，标准段深17.5米,共设置4个出入口，3组风亭。

*3号线的武林门站位于环城西路和体育场路交叉口的西侧，沿体育场路东西向布置，与2号线成通道换乘。为地下三层岛式车站。车站标准段为双柱三跨箱形框架结构，车站站台宽14米，主体长178m，宽21.7～25.6m，总建筑面积15477平米，主体建筑面积为12516平米，附属面积2961平米，车站共设置3个出入口，1个安全疏散口及2组风亭。<ref>{{cite web |title=杭州地铁3号线一期工程土建施工SG3-7标段土建施工SG3-7标段(古荡站、松木场站、武林门站、古荡站～玉古路站区间、玉古路站～松木场站区间、松木场站～武林门站区间、武林门站～武林广场站区间招标公告 |url=https://www.bidcenter.com.cn/newscontent-46313152-1.html |website=采招网}}</ref>
