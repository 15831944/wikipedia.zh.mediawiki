'''武昌南机务段'''（简称：武局南段）位于[[武汉市|武汉市]][[洪山区|洪山区]]狮子山街武南铁路新村168号，设在[[京广铁路|京广铁路]]上行线和京广铁路下行线及[[武咸城际铁路|武咸城际铁路]]之间，南邻三环线，西接[[武昌南站|武昌南站]]，是[[武汉铁路局|武汉铁路局]]下辖三个机务段之一，担当着424对高铁、客、货车牵引任务<ref>{{cite web|title=【全国铁路文明单位简介】武昌南机务段|url=http://www.whrailway.cn/html/2015/lujuxinwen_0403/41600.html|publisher=武汉铁路局|date=2015-04-03}}</ref>（此前负责江岸西至城陵矶间234公里货物列车运输及武昌至信阳间232公里旅客列车牵引任务<ref>{{Cite book|title=郑州铁路局企业名录|last=郑州铁路局史志编纂委员会办公室编|first=|publisher=北京：中国铁道出版社|year=1994年6月|isbn=7-113-01804-1|location=|pages=34}}</ref>）。下辖武客技整备场和汉口机务折返段<ref>{{cite news|title=依托科技创新 提升检修质量|url=http://wap.china-railway.com.cn/cpyfw_813/tlkj/201707/t20170718_65666.html|publisher=中国铁路总公司|date=2017-07-21}}</ref>。

== 历史 ==
[[File:武昌南机务段大门.jpg|缩略图]]
1976年，当时的武汉铁路局开始筹备成立武昌南机务段<ref>{{Cite book|title=武汉市志 1980-2000 第4卷|last=武汉市地方志编纂委员会编|first=|publisher=武汉：武汉出版社|year=2006年11月|isbn=7-5430-3559-6|location=|pages=16}}</ref>。1980年5月，武昌南机务段正式成立。

2004年11月，原武昌机务段整建制并入武昌南机务段<ref>{{Cite book|title=郑州铁路局年鉴 2005|last=郑州铁路局史志编纂委员会办公室编|first=|publisher=郑州铁路局史志编纂委员会|year=2005年12月|isbn=|location=|pages=37}}</ref><ref>{{cite news|title=武昌机务段整体搬迁|url=http://www.wuchang.gov.cn/wcqzfzz/xwzx/jx/823254/index.html|publisher=武昌区人民政府|date=2005-11-14|deadurl=yes|archiveurl=https://web.archive.org/web/20171024043056/http://www.wuchang.gov.cn/wcqzfzz/xwzx/jx/823254/index.html|archivedate=2017年10月24日|df=}}</ref>。

2006年2月18日起，原[[江岸机务段|江岸机务段]]下辖的汉口机务折返段改由武昌南机务段管理<ref>{{cite news|title=武昌南机务段客运机车交路扩容|url=http://www.whrailway.cn/html/2006/lujuxinwen_0306/2078.html|publisher=武汉铁路局|date=2006-03-06}}</ref>。

== 规模与功能 ==
[[File:武昌南机务段电力检修库.jpg|缩略图]]
武昌南机务段主要负责武汉局大部分客车和武九线部分货车牵引工作以及武汉枢纽长江以南地区调车工作，也担当部分动车组机务乘务工作。<ref>{{Cite book|title=武汉铁路分局年鉴 1998|last=武汉铁路分局年鉴编纂委员会编；田新法主编|first=|publisher=北京：中国铁道出版社|year=1998年12月|isbn=9787113024482|location=|pages=234}}</ref><ref>{{Cite web |url =http://news.cnhubei.com/hbrb/hbrbsglk/hbrb03/200801/t191985.shtml|title =动车组：司机预演春运|author =邹汉青，刘卫兵，罗强|date = 2008-01-02|publisher = 湖北日报|language = 中文}}</ref><ref>{{Cite web |url = http://news.cnhubei.com/ctjb/ctjbsgk/ctjb09/200810/t453389.shtml|title = 图文：出行旅客多火车检修忙|author =戴辉，周钢|date = 2008-10-03|publisher =楚天金报|language = 中文}}</ref><ref>{{Cite web |url = http://news.cnhubei.com/gdxw/201008/t1367229.shtml|title =武昌南机务段做好防暑降温工作为职工送清凉|author = 陈启亮|date =2010-08-16|publisher =荆楚网|language =中文 }}</ref>

武昌南机务段内设有内燃整备场、电力整备场、镟轮库、架修库等设施，担当本段配属和局管内部分内燃机车的大、中、小、辅修和电力机车中、小、辅修及整备任务。

===武客技整备场===
武客技整备场又名武昌折返段，位于[[武昌站|武昌站]]东南侧。武客技整备场主要承担武昌站到发、换挂普速客运机车以及[[武昌站|武昌站]]调车机车的日常整备任务，日整备机车130台次；涉及[[襄阳机务段|襄阳]]、[[北京机务段|北京]]、[[西安机务段|西安]]、[[广州机务段|广州]]、[[郑州机务段|郑州]]等8个机务段9种机型<ref>{{cite web|title=武昌南机务段新型机车行检作业门禁系统建成并正式启用|url=http://www.whrailway.cn/html/2016/lujuxinwen_0108/44263.html|publisher=武汉铁路局|date=2016-01-08}}</ref>。
