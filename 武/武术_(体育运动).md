'''武术'''，作为一种体育项目，发源于中国，目前主要流行于亚洲地区，是[[中华人民共和国全国运动会|中华人民共和国全国运动会]]、[[中华民国全国运动会|中华民国全国运动会]]、[[亚洲运动会|亚洲运动会]]、[[东亚运动会|东亚运动会]]、[[东南亚运动会|东南亚运动会]]和[[南亚运动会|南亚运动会]]等比赛的正式项目，是[[世界运动会|世界运动会]]的邀请项目，但不是[[奥林匹克运动会|奥林匹克运动会]]的比赛项目。

== 武术与大型赛事 ==
*1933年，[[中华民国第五届全国运动会|中华民国第五届全国运动会]]在[[南京|南京]]举行，武术以国术的名称首次设立。
*1959年，[[中华人民共和国第一届全运会|中华人民共和国第一届全运会]]在[[北京|北京]]举行，比赛项目包括武术，此后各届比赛一直设立该项目。
*1990年，[[第十一屆亞洲運動會|北京亚运会]]首次设立武术项目，此后一直为[[亚运会|亚运会]]正式项目。
*1993年，[[第一届东亚运动会|第一届东亚运动会]]在[[上海|上海]]举行，比赛项目包括武术，此后各届比赛一直设立该项目。
*1991年，[[第16届东南亚运动会|第16届东南亚运动会]]开始设立武术项目。
*2004年，[[第9届南亚运动会|第9届南亚运动会]]开始设立武术项目。
*2009年，[[世界运动会|世界运动会]]将增设武术项目，作为邀请项目。

== 管理机构 ==
[[国际武术联合会|国际武术联合会]]负责国际范围内的武术比赛组织和管理规则。

== 项目 ==
武术比赛分[[套路|套路]]和[[散手|散手]]两种类型，前者类似于艺术体操，后者类似于自由搏擊。
