'''武汉保利广场'''（Wuhan Poly Plaza），曾称为'''武汉保利文化广场'''，位於[[武汉市|武汉市]][[武昌区|武昌区]][[中南路|中南路]]与民主路交汇处，洪山广场的西侧，与洪山体育馆隔街相望。主建筑地上高度至楼顶221米<ref name="中建三局武昌保利文化广场主体结构验收顺利通过">{{cite web | url=http://www.chrd.com.cn/News/2221.htm | title=中建三局武昌保利文化广场主体结构验收顺利通过 | accessdate=2012-04-07}}</ref><ref name="保利文化广场主体结构验收顺利通过">{{cite web| url=http://www.cscec3b.com/readinfo.asp?id=2&title=63147&date1=2011-5-30%208:25:36| title=保利文化广场主体结构验收顺利通过| accessdate=2012-04-07| deadurl=yes| archiveurl=https://web.archive.org/web/20131019082802/http://www.cscec3b.com/readinfo.asp?id=2&title=63147&date1=2011-5-30%208:25:36| archivedate=2013年10月19日| df=}}</ref>，无天线，总建筑面积14.9万平方米，是一座具有开放式公共空间的综合商业广场。

== 概况 ==
保利广场地处[[湖北省|湖北省]]行政文化中心，距离湖北省政府、中共湖北省委、[[地方各级人民代表大会|湖北人大]]、[[中国人民政治协商会议湖北省委员会|湖北政协]]、[[中国人民银行|中国人民银行]]武汉分行、[[武汉铁路局|武汉铁路局]]等行政机构仅数分钟车程甚至步行距离。由[[中国保利集团|中国保利集团]]投资，[[中建三局|中建三局]]承建，2009年2月开建，2012年建成。保利广场由美国[[SOM|SOM]]公司设计，总建筑面积14万平方米，主楼（A座）46层，副楼（B座）20层，于空中连体，造型似一把椅子。其中主楼地上1至5层为商业区，6、7层为餐饮区，8层为影院层。<ref name="武昌新地标保利文化广场动工">{{cite web | url=http://news.cnhubei.com/mtjj/200902/t586795.shtml | title=武昌新地标保利文化广场动工 | accessdate=2012-04-07}}</ref>

[[香港特别行政区政府驻武汉经济贸易办事处|香港特别行政区政府驻武汉经济贸易办事处]]曾设于A座18层<ref name="香港特區政府駐武漢經濟貿易辦事處">{{cite web | url=http://www.info.gov.hk/gia/general/201404/01/P201404010552.htm | title=香港特區政府駐武漢經濟貿易辦事處 | accessdate=2015-03-12}}</ref>，后转入长期办公地址汉口[[武汉新世界国贸大厦|新世界国贸大厦]]。<ref name="香港政府駐武漢經濟貿易辦事處舉行成立儀式（附圖）">{{cite web | url=http://www.info.gov.hk/gia/general/201504/08/P201504080911.htm | title=香港政府駐武漢經濟貿易辦事處舉行成立儀式（附圖） | accessdate=2015-11-09}}</ref>
