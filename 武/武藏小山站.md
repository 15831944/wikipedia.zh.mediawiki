'''武藏小山站'''（{{jpn|j=武蔵小山駅|hg=むさしこやまえき|rm=Musashi-koyama eki|f=y}}）是一個位於[[東京都|東京都]][[品川區|品川區]][[小山_(品川區)|小山]]三丁目，屬於[[東京急行電鐵|東京急行電鐵]][[目黑線|目黑線]]的[[鐵路車站|鐵路車站]]<ref name="tokyu">{{Cite web |url=http://www.tokyu.co.jp/railway/station/info/?id=24 |title=武蔵小山駅｜各駅情報 |publisher=東京急行電鉄 |accessdate=2015-01-23}}</ref>。[[車站編號|車站編號]]是'''MG03'''。

==歷史==
* 1923年（[[大正|大正]]12年）3月11日 - 此站啟用，當時稱為'''小山站'''<ref name="jtb87">[[#jtb|東急の駅]]、p.87。</ref>。當時還設置[[島式月台|島式月台]]1面2線貨物月台<ref name="jtb87"/>。
* 1924年（大正13年）4月20日 - 由於在[[東北本線|東北本線]]已有相同名稱的[[小山站|小山站]]，改名為'''武藏小山站'''<ref>{{Cite web |url=http://dl.ndl.go.jp/info:ndljp/pid/2955648/5 |title=「地方鉄道駅名改称」『官報』1924年4月26日}} </ref>。
* 1925年（大正14年）11月 - 站房改建<ref name="jtb87"/>。
* 1937年（[[昭和|昭和]]12年）2月 - 撤除貨物線<ref name="jtb87"/>。
* 1974年（昭和49年）6月 - 設置自動驗票閘門。
* 2000年（[[平成|平成]]12年）8月6日 - 東急[[目蒲線|目蒲線]]分拆成[[目黑線|目黑線]]及[[東急多摩川線|東急多摩川線]]，車站改屬[[目黑線|目黑線]]。
* 2001年（[[平成|平成]]13年）6月22日 - 月台門啟用<ref name="rail fan2003-02p7"/>。
* 2006年（平成18年）7月2日 - 車站搬至新建成之地下車站<ref name="news06-09">{{Cite press release |url=http://www.tokyu.co.jp/file/060424.pdf |title=目蒲線（現：目黒線）目黒駅付近〜洗足駅付近間立体交差事業　7月2日（日）、目黒線不動前駅〜洗足駅間の約2.4Kmを地下化し、16カ所の踏切を解消します。9月下旬（予定）に目黒線で急行運転を開始|publisher=東京急行電鉄 |date=2006-04-24 |accessdate=2015-01-23 |format=pdf}}</ref>。
** 從以前的地面1面2線改為地下2面4線。
** 原計畫是[[側式月台|側式月台]]2面2線，中央2線有通過線。由於此計畫的武藏小山站不停靠急行，在地方民眾的請願以及商店街與居民的資助下，改成島式月台2面4線構造。
* 2009年（平成21年）5月 - 站前廣場完成。
* 2010年（平成22年）9月17日 - 車站大樓啟用<ref name="ekibiru"/>。

== 站名由來 ==
站名取自於本地地名。地名一說是取自近鄰的{{link-ja|小山八幡神社|小山八幡神社|'''小山'''八幡神社}}<ref name="jtb87"/>，另一說是附近有小山而得名。

開業當時為「小山站」，為了避免與[[東北本線|東北本線]][[小山站|小山站]]混淆（但兩站念法不同），套上舊國名「[[武藏國|武藏]]」為「武藏小山站」<ref name="jtb87"/>。

==車站結構==
本站為[[島式月台|島式月台]]2面4線的[[地下車站|地下車站]]<ref name="tokyu map">{{Cite web|url=http://www.tokyu.co.jp/railway/station/yardmap/?id=24|title=武蔵小山駅｜駅構内図|publisher=東京急行電鉄|accessdate=2015-01-23|deadurl=yes|archiveurl=https://web.archive.org/web/20150420111030/http://www.tokyu.co.jp/railway/station/yardmap/?id=24|archivedate=2015-04-20}}</ref>。目黑側設有單渡線。

目黑線在2006年9月25日起開始運行急行列車。此時本站下行全日、上行早晚尖峰時段實施[[緩急接續|緩急接續]]。午間的上行列車不實施緩急接續。

2008年6月22日，目黑線延伸至[[日吉站_(神奈川縣)|日吉]]，目黑線增加急行班次並取消午間的目黑折返各站停車班次，本站上下線全日實施緩急接續。

[[月台門|月台門]]與[[西小山站|西小山站]]一樣從地面站房時代就已設置<ref>目蒲線改為目黑線時尚未設置。</ref><ref name="rail fan2003-02p7">{{Cite journal|author = 鉄道友の会東京支部東急部会 |date = 2003-02-01 |title = 2001年度 東急総決算 |journal = RAIL FAN |issue = 2 |volume = 50 |publisher = 鉄道友の会 |pages = 7}}</ref>。

2010年9月17日，車站大樓開幕<ref name="ekibiru">「[http://www.tokyu.co.jp/contents_index/guide/news/100726.html 9月17日 季節の変化豊かな屋上庭園を併設した武蔵小山駅直結の新しい駅ビルがオープンします] {{webarchive|url=https://web.archive.org/web/20100808023513/http://www.tokyu.co.jp/contents_index/guide/news/100726.html |date=2010-08-08 }}」東京急行電鉄公式サイトニュースリリース、2010年7月26日</ref>。
