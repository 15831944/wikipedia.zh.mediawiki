'''武姓'''是一個[[中國|中國]][[姓氏|姓氏]]，在《[[百家姓|百家姓]]》中排名第250位。此姓氏在[[越南|越南]]亦有分佈，約佔人口3.9%。

== 起源 ==
* 出自[[姬姓|姬姓]]。《[[元和姓纂|元和姓纂]]》记载[[周平王|周平王]]少子（后被武则天追尊为[[周睿祖|周睿祖]]）出生后手掌中便有一片特殊的纹路，形状就像个“武”字，后来他的子孙以武为氏。<ref>《元和姓纂》：周平王少子生而有文在手，曰武，遂以为氏。</ref>
* [[汉代|汉代]]的“武班碑”指出：“殷王武丁之后”。
*《世本》指出：“夏臣武罗，其后子孙也称武氏”。
* 《风俗通义》所载：“春秋时宋武公之后以谥为氏”。
* 又据《[[风俗通义|风俗通义]]》所载，东汉有武强侯[[王梁_(漢朝)|王梁]]，封地在今河北省[[武强县|武强县]]，其后代因封地武强简为武氏。
* 出自以武字开头的复姓 - 武安氏和武疆氏。
* 出自[[唐代|唐代]]的冒姓或被赐姓为武姓。

== 郡望 ==
