1928年7月，[[共产国际|共产国际]]召开第六次代表大会，[[布哈林|布哈林]]在会上作了《国际形势和共产国际的任务》报告，提出了“第三时期”理论。“第三时期”又被称为革命或战争时期，这一理论完全是以苏俄的利益为导向制定的。当时[[斯大林|斯大林]]担心以英国为首的资本主义国家正准备对苏战争，因而共产国际就号召各国共产党拥护苏联，反对对苏进攻，并将进攻苏联的战争转化为推翻各国资本主义统治的革命。<ref>{{Cite web |url=http://paper.people.com.cn/rmlt/html/2012-06/11/content_1078067.htm?div=-1 |title=存档副本 |access-date=2015-04-30 |archive-url=https://web.archive.org/web/20160304064613/http://paper.people.com.cn/rmlt/html/2012-06/11/content_1078067.htm?div=-1 |archive-date=2016-03-04 |dead-url=yes }}</ref>

1929年[[中东路事件|中东路事件]]爆发后，[[共产国际|共产国际]]远东局明确要求中共中央要提出'''武装保卫苏联'''的[[口号|口号]]，并组织大规模的反对国民党和拥护苏联的群众示威。对此，中共领导人瞿秋白、李立三、向忠发、周恩来做出了积极的响应。他们召开政治局会议，决定在8月1日“反帝日”举行示威，而且争取发动上海工人总罢工。<ref>{{Cite web|url = http://yuanchuang.kandian.360.cn/api/html/3700369594.php?dtid=3700497301&did=1101344458|title = 军迷俱乐部第第217期|accessdate = 2015-07-19|author = |date = |publisher = |deadurl = yes|archiveurl = https://web.archive.org/web/20150719004032/http://yuanchuang.kandian.360.cn/api/html/3700369594.php?dtid=3700497301&did=1101344458|archivedate = 2015-07-19}}</ref>

对于中共中央的做法，陈独秀专门致信中共中央提出批评，主张在这个时候片面宣传“拥护苏联”“于我们不利”，绝不能简单地认为“广大群众都认同苏联是中国解放的朋友”<ref>（《陈独秀给中共中央的信》，1929年7月28日，8月11日）</ref>。[[陈独秀|陈独秀]]等人因此被开除党籍。

中东路事件中，中苏双方的武装冲突在1929年12月即已结束。但中国共产党对“武装保卫苏联”的宣传并未结束，持续数年<ref>{{Cite web |url = http://ucwap.ifeng.com/mil/milh/milhisphoto/news?aid=32325191&mid=9GXyXk&v=2 |title = 广元市利州区发现巨幅红军石刻标语 |accessdate = 2013-06-06 |author = |date = 2012-03-01 |work = |publisher = 凤凰网，来源：利州灾后重建网 |language = Zh-hans }}</ref>。

九一八事变爆发后，1931年9月20日，中国共产党发表的《中国共产党为日本帝国主义强暴占领东三省事件宣言》就包含“武装保卫苏联”的内容。同日中共中央代总书记[[王明|王明]]也发表了“武装保卫苏联”的讲话。
