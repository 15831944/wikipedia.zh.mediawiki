'''武山庙'''是中国[[浙江省|浙江省]][[宁波市|宁波市]][[奉化区|奉化区]]的一座庙宇，位于[[溪口镇_(宁波市)|溪口镇]]武山脚下。武山庙至迟建始于明初，清[[乾隆|乾隆]]、[[嘉庆|嘉庆]]年间曾进行大修，形成典型的清中叶江南庙宇风格。庙宇最初为溪口多族共祀，后逐渐成为蒋氏家庙。[[蒋中正|蒋中正]]、[[蒋经国|蒋经国]]父子曾多次至庙中祭祀并出资维修<ref name="王舜祁">{{cite book|title=宁波文物古迹保护纪实|author=周金康|first=|publisher=宁波出版社|accessdate=2018-07-24|year=2000|isbn=9-787-80602-418-8|location=|pages=|chapter=溪口武山庙博物馆建立|chapterurl=http://www.nbzx.gov.cn/art/2015/9/18/art_4055_690797.html|archiveurl=https://web.archive.org/web/20180708143539/http://www.nbzx.gov.cn/art/2015/9/18/art_4055_690797.html|archivedate=2018-07-08|deadurl=no}}</ref>。2006年，武山庙成为[[全国重点文物保护单位|全国重点文物保护单位]]<ref name="nbwb-detail">{{cite web|url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=830&type=2|title=蒋氏故居|accessdate=2017-09-02|date=2006-06-22|publisher=宁波市文物保护管理所|archiveurl=https://web.archive.org/web/20180222043927/http://www.nbwb.net/pd_wwbh/info.aspx?Id=830&type=2|archivedate=2018-02-22|deadurl=no}}</ref>。

== 历史 ==
武山庙确切的始建年代已不可考，根据庙中碑文可知，至迟在明洪武年间，武山庙已经存在。庙宇最初为溪口镇上任、宋、单、张、蒋五姓共同祭祀，清代以后蒋姓成为镇上第一大姓，社首即由蒋姓担任<ref name="王舜祁" />。清乾隆五十五年（1790年）及嘉庆元年（1796年）曾修葺并扩建廊庑和山门<ref>{{Cite book|title=溪口胜景|author=王舜祁|author2=胡元福|publisher=奉化市政协文史委员会|year=2000}}</ref>。清末，蒋中正祖父[[蒋斯千|蒋斯千]]、父亲[[蒋肇聪|蒋肇聪]]均曾担任庙董。蒋肇聪担任庙董期间，五族再次对武山庙进行扩建，形成当下的规模<ref name="裘国松">{{Cite book|title=溪口谈助——蒋氏故里文化品读|author=裘国松|publisher=宁波出版社|year=2015|isbn=978-7-55262-2089|location=宁波}}</ref>{{Rp|5}}。

中华民国成立后，武山庙内兴办武山小学，蒋经国曾于1916年在此学习，蒋中正也曾捐资修庙并多次在此祭拜<ref name="裘国松" />{{Rp|5}}。1936年12月，[[西安事变|西安事变]]的消息传到溪口，正在武山庙看戏的蒋中正兄长[[蒋介卿|蒋介卿]]受惊中风，三天后去世。次年，蒋中正元配夫人[[毛福梅|毛福梅]]曾经到武山庙求签，求得“秀才出门，状元归家”签，随后被称为“秀才”的蒋介卿出殡，几天后蒋经国携苏联妻子[[蔣方良|蒋方良]]回到溪口补办中式婚礼，武山庙因而被蒋家视为灵验<ref name="唐瑞福">{{Cite book|title=浙江文史资料选辑（第二十三辑）|author=唐瑞福|first=|author2=汪日章|publisher=浙江人民出版社|year=1982|editor-last=中国人民政治协商会议浙江省委员会文史资料研究委员会|location=杭州|pages=|chapter=蒋介石的故乡|unified=11103·78}}</ref>{{Rp|58,59}}。1949年1月，在[[第二次国共内战|第二次国共内战]]中面临溃败的蒋中正下野，在武山庙抽到下签，并以“武山庙的菩萨是灵的”回应随从的劝慰<ref name="王凡">{{Cite journal|title=奉化溪口遗迹中的蒋家故事（下）|author=王凡|journal=人物|issue=12|year=2010}}</ref>。

中华人民共和国成立后，武山庙曾作为粮站仓库使用，建筑遭到破坏。1983年，溪口镇筹建的博物馆选址武山庙，1986年起对庙宇进行大修以恢复其本来面貌，1988年对外开放<ref name="王舜祁" />。2010年溪口博物馆搬迁至新馆后，武山庙成为“梦回溪口”武山庙庙会表演场地<ref>{{Cite web|url=http://zfxxgk.fh.gov.cn/auto80/auto143/201412/t20141229_144236.html|title=“梦回溪口”武山庙庙会活动全面开启|accessdate=2018-07-25|author=溪口旅游集团|date=2014-12-29|publisher=宁波市奉化区政府信息公开|archive-url=https://web.archive.org/web/20180727162521/http://zfxxgk.fh.gov.cn/auto80/auto143/201412/t20141229_144236.html|archive-date=2018-07-27|dead-url=no}}</ref>。

== 建筑 ==
武山庙位于溪口武山脚下，坐西北朝东南，为四合院建筑。建筑均为硬山顶，分为前后两进，大殿为抬梁结合穿斗式建筑，天井有戏台，左右有廊庑<ref name="nbwb-detail" />。

== 保护与利用 ==
{{external media|width=250px|image1=[http://www.nbwb.net/UploadFile/images/2018012213515867.jpg 摩诃殿、丰镐房、蒋氏宗祠、文昌阁遗址、武山庙保护区划图]，原始内容[https://web.archive.org/web/20180711164544/http://www.nbwb.net/UploadFile/images/2018012213515867.jpg 存档]于2018年7月11日}}蒋氏父子迁往台湾后，武山庙曾一度成为粮站仓库所在地，偏房则被溪口卫生院占据，庙宇面临坍塌的危险。1983年，奉化县文管局等机构提出将武山庙改为博物馆的设想。1985年及1987年，浙江省文物局和国家文物局先后对武山庙修缮工程拨款计18万元人民币。1986年2月底，由临海县古建筑工程公司主持对武山庙进行维修，实施了增修戏台八面结顶藻井、更换正殿木柱为浮雕龙柱等修缮工程。1988年春节，溪口博物馆正式对外开放，成为宁波市境内首家建制博物馆<ref name="王舜祁" />。2005年，武山庙与[[蒋母墓|蒋母墓]]一起被列入[[浙江省文物保护单位|浙江省文物保护单位]]<ref>{{Cite web|url=http://culture.zjol.com.cn/05culture/system/2006/08/03/007785967.shtml|title=武山庙（含蒋母墓）|accessdate=2018-08-14|author=|date=2006-05-18|publisher=浙江文物网|archive-url=https://web.archive.org/web/20090318104448/http://culture.zjol.com.cn/05culture/system/2006/08/03/007785967.shtml|archive-date=2009-03-18|dead-url=no}}</ref>。2006年[[第六批全国重点文物保护单位|第六批全国重点文物保护单位]]公布时，武山庙和[[文昌阁_(溪口)|文昌阁]]、[[蒋氏宗祠_(溪口)|蒋氏宗祠]]等建筑以“[[溪口镇建筑群|溪口镇建筑群]]”的名义并入[[第四批全国重点文物保护单位|第四批全国重点文物保护单位]][[蒋氏故居|蒋氏故居]]<ref name="nbwb-detail" />。溪口博物馆搬迁至[[玉泰盐铺|玉泰盐铺]]西侧后，武山庙成为“梦回溪口”庙会表演场地，并于2016年被列为浙江省不可移动文物保护利用优秀案例<ref>{{Cite web|url=http://www.sach.gov.cn/art/2016/7/19/art_722_132453.html|title=浙江省公布不可移动文物保护利用优秀案例|accessdate=2018-07-27|author=|date=2016-07-19|publisher=国家文物局|archive-url=https://web.archive.org/web/20180727162520/http://www.sach.gov.cn/art/2016/7/19/art_722_132453.html|archive-date=2018-07-27|dead-url=no}}</ref>。
