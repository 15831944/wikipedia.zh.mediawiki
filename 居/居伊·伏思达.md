'''居伊·伏思达'''（'''Guy Verhofstadt'''，{{bd|1953年|4月11日|}}），又译为'''居伊·费尔霍夫施塔特'''、'''盖·维霍夫斯塔'''，前任[[比利时首相|比利时首相]]，现任欧洲议会[[争取欧洲自由人士和民主人士联盟党团|自由民主党]]党团主席。

== 生平 ==
1953年生于[[登德尔蒙德|登德尔蒙德]]市。1992年11月，[[荷语自由进步党|荷语自由进步党]]改为[[荷语自由民主党|荷语自由民主党]]后，当选荷语自由民主党主席。1999年7月12日当选为[[比利时|比利时]]首相。他是[[联邦欧洲|联邦欧洲]]的支持者，“[[欧洲合众国|欧洲合众国]]”的倡导者<ref>{{Cite web |url=http://civ.ce.cn/main/todaynews/200907/17/t20090717_19558200.shtml |title=对话比利时前首相伏思达:他胸怀"欧洲合众国"梦想 中国经济网 |access-date=2009-07-17 |archive-url=https://web.archive.org/web/20140517160001/http://civ.ce.cn/main/todaynews/200907/17/t20090717_19558200.shtml |archive-date=2014-05-17 |dead-url=yes }}</ref>，所著《走向欧洲合众国》。

伏思达的父亲曾担任荷语自由进步党主席威利·德克莱尔的法律顾问。1972至1974年伏思达在[[根特|根特]]学习法律的时期担任荷语自由学生联合会（Flemish Liberal Student's Union）主席，1982年，年仅29岁的伏思达成为荷语自由进步党主席。1999年开始成为比利时首相，直到2008年辞职为止连任三届。

== 首相任期 ==
他在1999至2008年担任三屆[[比利時首相|比利時首相]]，是比利时数十年来首位並非来自社会党及基督教民主党的首相。任內在2003年通過[[比利時同性婚姻|比利時同性婚姻]]合法化。

== 辞职以后 ==
2009年成為欧洲由民主党团主席，積極在[[欧洲议会|欧洲议会]]发挥作用。
