'''岡本裕一朗'''（{{jpn|j=岡本 裕一朗|hg=おかもと ゆういちろう|rm=Okamoto Yuuichirou}}，{{bd|1954年||}}），日本[[哲学|哲学]]、倫理學者，現任[[玉川大學|玉川大學]]教授<ref>{{cite web |title=未来のビジネスのために「哲学」を始めよう！ 《いま世界の哲学者が考えていること》著者・岡本裕一朗に訊く |url=https://wired.jp/2017/05/20/interview-yuichiro-okamoto/ |website=wired.jp |accessdate=2018-08-19}}</ref><ref>{{cite web |title=哲学者・岡本裕一朗は「テック時代における人間のありかた」を、いかに定義するか？ |url=http://icf.academyhills.com/interview/special_yuichiro_okamoto.php |website=academyhills.com |accessdate=2018-08-19 |archive-url=https://web.archive.org/web/20180819114356/http://icf.academyhills.com/interview/special_yuichiro_okamoto.php |archive-date=2018-08-19 |dead-url=yes }}</ref>。

==經歷==
1954年（[[昭和|昭和]]29年）出生於福岡縣<ref>[https://hillslife.jp/innovation/2017/10/05/technology-wants-philosophy/ ヒルズライフ HILLS LIFE｜哲学者・岡本裕一朗は「テック時代における人間のありかた」を、いかに定義するか？]</ref>，1978年畢業於[[山口大學|山口大學]]文理学部哲学、倫理学科，1984年[[九州大学|九州大学]]大学院文学研究科博士課程滿期退學。1997年以《[[精神現象學|精神現象學]]的體系構想-[[阿里士多德|阿里士多德]]哲學的重建》（「精神の現象学」の体系構想 -アリストテレス哲学の再建）取得九大文学博士。其後先後擔任九州大学文学部助手、[[東和大學|東和大學]]工学部講師、助教授、教授、玉川大学文学部教授。<ref>《思考実験》著者紹介、[http://jglobal.jst.go.jp/detail.php?JGLOBAL_ID=200901038973672261&q=%E5%B2%A1%E6%9C%AC%E8%A3%95%E4%B8%80%E6%9C%97&t=1 研究者情報]</ref>
