'''岡山神社'''是[[臺灣日治時期|臺灣日治時期]]位在[[高雄州|高雄州]][[岡山郡|岡山郡]]（今[[高雄市|高雄市]][[岡山區|岡山區]]）的[[神社|神社]]。

== 歷史 ==
[[File:岡神神社水手缽.jpg|thumb]]
岡山神社於昭和8年（1933年）因紀念〈國民精神作興詔書〉頒布十週年而籌建{{r|楊}}，選址於[[高雄州|高雄州]][[岡山郡|岡山郡]][[岡山街|岡山街]]前峯{{r|金2}}（今[[高雄市|高雄市]][[岡山區|岡山區]]中山公園（即岡山公園）、[[岡山壽天宮|岡山壽天宮]]）{{r|岡}}。昭和9年（1934年）6月16日申請建立許可，昭和10年（1935年）3月7日獲准，3月14日地鎮（4月動土興建{{r|岡}}{{r|楊}}），9月17日上棟，12月9日鎮座<ref>{{Cite book | title = 五甲尾公學校 岡山神社造營記念寫真帖，岡山神社創立概況| url = https://get.google.com/albumarchive/114456741671868994545/album/AF1QipP9IZOnYJSiZ2O81Qi8JNgT8NXNLxZGlF0V8ajN/AF1QipMH2wd82CZncUSEJ45nW1_KscB_gsBii3q4HOfq | language = ja}}</ref>（{{r|岡}}{{r|楊}}也記載了於9月17日上棟，12月9日落成的訊息）。而當時為了慶祝神社落成，岡山郡役所要求底下7個街庄表演節目並予以評比，獲得第1名的是配合郡役所要求表演「打倒迷信」的[[湖內庄|湖內庄]]，而第2名是表演「田寮之歌」的[[田寮庄|田寮庄]]{{r|岡}}。

又神社[[近代社格制度#諸社|社格]]為郷社（昭和17年（1942年）1月24日列格）{{r|岡}}{{r|金2}}，並為高雄州四個鄉社（指岡山、鳳山、東港、潮州）之一{{r|楊}}。祭祀[[北白川宮能久親王|北白川宮能久親王]]、[[開拓三神|開拓三神]]（[[國魂|大國魂命]]、[[大國主神|大己貴命]]、[[少彥名命|少彥名命]]）、[[天照大神|天照大神]]與[[明治天皇|明治天皇]]{{r|岡}}{{r|金2}}。例祭日11月15日{{r|楊}}{{r|金2}}。又於昭和13年（1938年）11月時岡山郡獸肉組合與岡山楠梓專賣品關係業者曾獻納了一對銅馬{{r|楊}}。

戰後神社廢止。昭和12年（1937年）時（原位於今維仁里維仁路上，被政府以消除水患、開闢市容整修街道之理由而{{r|壽}}）拆除的[[岡山壽天宮|岡山壽天宮]]於民國38年（1949年）重建於神社原址上（今壽天里公園路40號{{r|壽}}），原神社狛犬則充做廟前石獅{{r|岡}}。

== 現況 ==
岡山神社原址現為中山公園（岡山公園）與岡山壽天宮{{r|岡}}。（石造）[[石獅子|狛犬]]1對充做（壽天宮）廟前石獅，而在公園內也保留了改為牌樓的[[鳥居|鳥居]]{{r|岡}}。其中鳥居指原岡山神社一之鳥居，戰後曾去除原鳥居的混凝土製"笠木"（上樑）並予以加工作為中山公園入口牌樓<ref>{{Cite web |url = https://www.google.com.tw/maps/@22.7965763,120.2931603,3a,22.3y,281.76h,90.8t/data=!3m6!1e1!3m4!1sx4wp7G8rdWqvun8zz_Vocw!2e0!7i13312!8i6656 |title = 街景中的原中山公園牌樓 |date = 2012-06 |publisher = google地圖 |language=zh-TW}}</ref>，現已修復完成並漆成紅色。

此外在[[岡山壽天宮|岡山壽天宮]]的玉皇殿1樓內仍保留了原神社的2台神輿{{r|金1}}<ref>{{Cite web |url = https://sites.google.com/site/gsstktw/gsstk02/07 |title = 天照大小神轎 |publisher = 岡山壽天宮全球資訊網 |language=zh-TW}}</ref>，而在金爐旁也堆置了原神社的手水缽<ref>{{Cite web |url = https://sites.google.com/site/gsstktw/gsstk02/03 |title = 岡山神社手水舍水槽|publisher = 岡山壽天宮全球資訊網 |language=zh-TW}}</ref>和石燈籠構件數枚（包含石燈籠構件"中台"、"火袋"、"寶珠"各1枚以及昭和12年（1937年）由岡山郡守飯島稔奉獻的大石燈籠（同款現存於[[高雄神社|高雄神社]]與[[嘉義神社|嘉義神社]]）構件"竿"2枚<ref>{{Cite web |url = https://sites.google.com/site/gsstktw/gsstk02/04 |title = 岡山神社石燈籠 |publisher = 岡山壽天宮全球資訊網 |language=zh-TW}}</ref>）{{r|金1}}。在[[國立岡山高級農工職業學校|國立岡山高級農工職業學校]]正門右側(校園內)走道兩旁也堆置有石燈籠構件"笠"（約11枚，倒置，部分置於花盆下<ref>{{Cite web |url = https://www.google.com.tw/maps/@22.803311,120.2899721,3a,15y,194.21h,84t/data=!3m6!1e1!3m4!1sobdhRzvAI87hs-_9QNk1jA!2e0!7i13312!8i6656?hl=zh-TW |title = 街景中的岡山高農內的石燈籠"笠"|date = 2017-02 |publisher = google地圖 |language=zh-TW}}</ref>）{{r|金1}}。而2012年時岡山公園內的中國城茶館前仍堆置有石燈籠構件"寶珠"數枚，但隨公園整建房屋拆除現已不知去向<ref>{{Cite web |url = http://blog.xuite.net/d0963064845/twblog1/123679959-2012%2F6%2F25%E9%AB%98%E9%9B%84%E6%B4%B2%E5%B2%A1%E5%B1%B1%E9%83%A1%E5%B2%A1%E5%B1%B1%E7%A5%9E%E7%A4%BE(%E4%BB%8A%E5%B2%A1%E5%B1%B1%E5%A3%BD%E5%A4%A9%E5%AE%AE) |title = 高雄洲岡山郡岡山神社（今岡山壽天宮）|author = 黃家榮 |date = 2012-06-25 |publisher = 隨意窩 |language=zh-TW}}</ref>。
