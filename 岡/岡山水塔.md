'''岡山水塔'''位於[[高雄市|高雄市]][[岡山區|岡山區]]的岡山淨水廠內，於2004年3月2日公告為歷史建築{{r|文}}，並為[[中華民國各縣市歷史建築十景列表#高雄市|高雄縣歷史建築十景]]與[[臺灣歷史建築百景列表|臺灣歷史建築百景]]（25名）之一{{r|史}}，是岡山地區的地標{{r|高}}。

== 沿革 ==
岡山水塔是[[臺灣日治時期|臺灣日治時期]]於1925年設置的岡山上水道系統在1937年11月20日至隔年4月20日之間擴建的設施之一{{r|岡|page=2-10}}，是在擴大供水範圍後做為加壓配水之用{{r|岡|page=2-11}}。

二次大戰後，為了配合岡山鎮的人口增長，臺灣省公共工程局分別在1957年與1962年進行岡山淨水廠第一、二期擴建工程，其中在第一期過程中將慢濾池改為快濾池，並興建抽水機室取代水塔加壓功能，於是岡山水塔改作為快濾池的高加反沖砂系統{{r|岡|page=2-13}}。然而後來由於[[阿公店水庫|阿公店水庫]]的淤積以及[[阿公店溪|阿公店溪]]上游養豬戶的增加導致供水不足與水的濁度提升{{r|岡|page=2-15}}，雖然在1990年試圖以生物處理設備解決汙染問題，但此一設備在1991年啟用的第二年便因為細菌被泥巴封住弄死而廢掉，岡山淨水廠更於1992年5月25日宣告永久停用，內部設施遭到拆除或改建，而岡山水塔也從此閒置至今{{r|岡|page=2-16}}。

== 建築特色 ==
水塔為鋼筋混擬土建築，高約30.9公尺，內徑5.4公尺、水深3公尺，滿水位標高25.4公尺，有效容量為113立方公尺，水槽內設有水位計{{r|岡|page=2-10}}。底部的唧筒室裝有兩臺15匹馬力的的電動機，頂部設有避雷針{{r|岡|page=2-10}}。目前附屬的鐵管樓梯雖有鏽蝕脫落現象，但建築主體仍保持完整{{r|史}}。
