'''岡田結實'''（{{bd|2000年|4月15日|||catIdx=Okada Yui}}）,[[日本|日本]][[女演員|女演員]]、[[模特兒|模特兒]]、電視藝人，出身自[[大阪府|大阪府]][[大阪市|大阪市]]<ref name="incent" />，隸屬於[[奧斯卡傳播|奧斯卡傳播]]旗下。<ref>[http://www.hochi.co.jp/entertainment/20160901-OHT1T50197.html ますおか岡田長女・結実、初司会！日テレ系「誰だって波瀾爆笑」１０・２から…小林麻耶は卒業、2016年9月2日] {{webarchive|url=https://web.archive.org/web/20170325031308/http://www.hochi.co.jp/entertainment/20160901-OHT1T50197.html |date=2017年3月25日 }} [[報知體育|報知體育]]</ref>

父親是[[搞笑藝人|搞笑藝人]][[岡田圭右|岡田圭右]]（[[增田岡田|增田岡田]]）<ref>[http://www.rbbtoday.com/article/2014/04/03/118491.html ますだおかだ岡田圭右の娘はモデル! 「家ではネクラ」と父の意外な一面明かす] RBB TODAY、2014年4月3日</ref>，母親是前搞笑藝人[[上嶋祐佳|上嶋祐佳]]，兄長是演員[[岡田隆之介|岡田隆之介]]。

== 略歴 ==
2001年，作為兒童模特兒進入藝能界。<ref>{{Cite news|url= http://www.sponichi.co.jp/entertainment/news/2016/06/03/kiji/K20160603012710280.html |title=岡田結実、父以上？の大物感 夢は女優も…バラエティー引っ張りだこ |publisher=スポニチアネックス |date=2016-06-03 |accessdate=2016-07-16}}</ref>

2010年開始的4年間，在NHK教育頻道的《[[天才兒童MAX|天才兒童MAX]]》作為[[てれび戦士|てれび戦士]]出演。2012年，在該節目內挑戰「一分鐘內用單腳穿上襪子並左右對齊的數量」，成功穿上11隻襪並獲得[[健力士世界紀錄|健力士世界紀錄]]認證。

2011年開始2年間，擔任時尚雜誌《[[Nico☆Puchi|Nico☆Puchi]]》的專屬模特兒。

2014年，雜誌《[[Pichi_Lemon|Pichi Lemon]]》的第22屆海選，從約10000人當中脫穎而出成為大獎4人之一<ref>{{Cite journal |和書 |author= |title=新ピチ㋲グランプリ発表!! |year=2014 |publisher=学研プラス |journal=ピチレモン |issue=2014年7月号 |page=4}}</ref>，之後1年間擔任該雜誌專屬模特兒。

2016年，從原經紀公司[[INCENT|INCENT]]<ref name="incent">{{Cite web|url=http://www.incent.jp/junior/model/okada/index.html |title=岡田結実 |publisher=インセント |archiveurl=https://web.archive.org/web/20141129145818/http://www.incent.jp/junior/model/okada/index.html |archivedate=2014-11-29 |accessdate=2016-02-25 |deadurl=yes }}</ref>移籍至[[奧斯卡傳播|奧斯卡傳播]]。<ref>{{Cite web |date=2016-02-21 |url=http://ameblo.jp/okada-yui-official/entry-12131195338.html |title=報告 |publisher=岡田結実オフィシャルブログ「今日も明日もゆい日和」 |accessdate=2016-02-25}}</ref>

2016年至2017年年末年始電視出演排行榜女性首位。<ref>{{Cite web |date=2017-01-06 |url=http://www.sponichi.co.jp/entertainment/news/2017/01/06/kiji/20170106s00041000066000c.html |title=年末年始ＴＶ出演ランク　嵐が３年連続ＴＯＰ３独占！岡田結実が初女王 |publisher=[[日刊體育|日刊體育]] }}</ref>

在2017年2月上映的電影《[[傷痕累累的惡魔|傷痕累累的惡魔]]》作為演員出道。<ref>{{cite news|url=http://www.oricon.co.jp/news/2075420/full/|title=ますおか岡田の娘・結実が役者デビュー「精一杯努力します」|newspaper=ORICON STYLE|date=2016-07-21|accessdate=2016-07-21}}</ref>17歲生日當天的2017年4月15日，同時發行兩部[[寫真集|寫真集]]（男性粉絲向的《{{lang|ja|きゅーじゅーはちぱーせんとのゆい}}》和女性粉絲向的《hungry!!!》）。<ref name="sanspo20170311">{{Cite news|url= http://www.sanspo.com/geino/news/20170311/geo17031105030012-n1.html |title= 父もワォ！？岡田結実、誕生日に「出し尽くした」初写真集×２発売 |newspaper= SANSPO.COM |publisher= 産経デジタル |date= 2017-03-11 |accessdate= 2017-03-11 }}</ref>
