'''桿石目'''（[[學名|學名]]：{{lang|la|'''Bactritida'''}}），又名'''桿棱石目'''，是[[頭足綱|頭足綱]]下已[[滅絕|滅絕]]的一個[[目_(生物)|目]]。由[[直角石目|直角石目]]於[[泥盆紀|泥盆紀]]演化而來，持續生存到[[三疊紀|三疊紀]]後滅絕。<ref name="Fossilworks" />牠們被認為是從[[鸚鵡螺亞綱|鸚鵡螺類]]演化成[[菊石|菊石]]的過渡類型，之後演化成菊石亞綱和[[蛸亞綱|蛸亞綱]]（[[章魚|章魚]]、[[魷魚|魷魚]]、[[墨魚|墨魚]]和[[箭石|箭石]]等）。桿石類在分類學上有著不同的定位，有些學者將桿石目歸類為鸚鵡螺亞綱，有些則將其歸入菊石亞綱，還有學者將桿石類獨立成一個亞綱──'''桿石亞綱'''（[[學名|學名]]：{{lang|la|'''Bactritoidea'''}}），也就是非鸚鵡螺，也非菊石的頭足類。<ref name="Bactrites">{{Cite web |url = http://liusir.pixnet.net/blog/post/248634458-bactrites-%E8%8F%8A%E7%9F%B3%E4%B9%8B%E7%A5%96---%E6%A1%BF%E7%9F%B3-%28%E5%BE%B7%E5%9C%8B%29 |title = Bactrites 菊石之祖 - 桿石 (德國) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-06-13 }}</ref>

== 特徵 ==
桿石類的殼形介於[[直角石|直角石]]式殼和[[弓角石|弓角石]]式殼之間，牠們在水中可能是頭向下垂直生活的，如同某些直角石類如{{tsl|en|Michelinoceras|米契林角石}}和''[[Buttsoceras|Buttsoceras]]''等。從其殼形可推測牠們應該不是活躍的游泳者，而是隱藏在[[海藻|海藻]]中的潛伏突擊者。<ref name="klug" >Klug, C. and Korn, D. 2004. The origin of ammonoid locomotion. Acta Palaeontologica Polonica 49 (2): 235–242
