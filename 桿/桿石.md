'''桿石'''（[[學名|學名]]：{{lang|la|'''''Bactrites'''''}}），又名'''桿棱石'''，是一[[屬|屬]]已[[滅絕|滅絕]]的[[頭足類|頭足類]]。為[[桿石科|桿石科]]的模式屬。牠們生存在早[[泥盆紀|泥盆紀]]至晚[[二疊紀|二疊紀]]的[[海洋|海洋]]中，被認為是從[[鸚鵡螺亞綱|鸚鵡螺類]]演化成[[菊石|菊石]]的過渡類型，牠們具有一些異於一般鸚鵡螺類，反倒近似於菊石類的特徵，例如小型且產量多的[[卵|卵]]、體管位於[[腹|腹]]側等。有些學者將桿石歸類為鸚鵡螺類，有些則將其歸入菊石類，也有學者主張將其獨自歸成一類，也就是非鸚鵡螺，也非菊石的頭足類。<ref name="Bactrites">{{Cite web |url = http://liusir.pixnet.net/blog/post/248634458-bactrites-%E8%8F%8A%E7%9F%B3%E4%B9%8B%E7%A5%96---%E6%A1%BF%E7%9F%B3-%28%E5%BE%B7%E5%9C%8B%29 |title = Bactrites 菊石之祖 - 桿石 (德國) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-06-13 }}</ref>牠們的化石被發現於[[美國|美國]]、[[英國|英國]]和[[哈薩克|哈薩克]]等地。<ref name="Fossilworks" />

== 特徵 ==
桿石有著細長桿狀的外殼，體管位於腹側。牠們在水中可能是頭向下垂直生活的，從其殼形可推測牠們應該不是活躍的游泳者，而是隱藏在[[海藻|海藻]]中的潛伏突擊者。<ref name="klug" >Klug, C. and Korn, D. 2004. The origin of ammonoid locomotion. Acta Palaeontologica Polonica 49 (2): 235–242
