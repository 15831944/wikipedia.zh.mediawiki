'''昌赣客运专线'''是一条建设中的客运专线铁路，北起南昌铁路枢纽和京九铁路在[[横岗站|横岗站]]接轨，南至赣州市设[[赣州西站|赣州西站]]；铁路于2014年开工，將於2019年底通车。根据发改委的批复该线路旅客列车设计行车速度：250公里/小时，基础设施预留进一步提速条件。最小曲线半径：困难地段4000米，地形较好地段可适当放宽。

== 概况 ==

== 历史 ==
=== 前期准备 ===
该线路最早于2008年8月提出<ref>[http://jndsb.jxnews.com.cn/system/2008/08/11/002817297.shtml 南昌至赣州城际铁路敲定]</ref>，当时多称为'''昌吉赣城际铁路'''，计划在2009年开展前期工作<ref>[http://www.jxnews.com.cn/jxrb/system/2008/11/01/002873094.shtml 江西省掀起新一轮铁路建设高潮--江西日报]</ref>，并将在南昌和[[昌九城际铁路|昌九城际铁路]]对接<ref>[http://gs.jxnews.com.cn/system/2008/12/31/010029452.shtml 2020年江西铁路里程将增至6000公里]</ref>。

2009年初计划在年底开建于2012年通车，设计时速为200公里/小时<ref>[http://jiangxi.jxnews.com.cn/system/2009/01/20/011009994.shtml 昌吉赣城际铁路年底前有望开建]</ref>；然而2009年底仍然未开建，而此时[[武广客运专线|武广客运专线]]通车，有报道指出对江西的影响时同时指出昌吉赣城际时速为300公里以上<ref>[http://www.jxnews.com.cn/xxrb/system/2009/12/24/011274359.shtml 武广高铁开通 影响江西几何]</ref>。

2010年2月原[[铁道部|铁道部]]人员查看线路走向，规划线路沿[[京九线|京九线]]既有线至[[泰和|泰和]]后绕过[[兴国|兴国]]从[[万安|万安]]到[[赣州|赣州]]<ref>[http://jxja.jxnews.com.cn/system/2010/02/02/011302141.shtml 铁道部专家在吉安察看昌吉赣客专线走向]</ref>；2010年9月昌吉赣和赣深被一同初步列入国家“十二五”铁路规划<ref>[http://jxgz.jxnews.com.cn/system/2010/09/04/011470929.shtml 昌吉赣、赣深客列两辆专线列入规划 赣州将迎来高铁时代]</ref>，昌吉赣、赣深计划将于2013年通车，时速为200～250公里<ref>[http://www.jxnews.com.cn/xxrb/system/2010/09/08/011473369.shtml 昌吉赣客专2013年通车]</ref>；根据2010年底的报道将于2011年动工<ref>[http://jj.jxnews.com.cn/system/2010/11/25/011527085.shtml 九景衢铁路、武九客运专线有望明年开工建设]</ref>。

2011年5月出台“[[十二五|十二五]]”正式规划只将昌吉赣纳入<ref>[http://www.tjtdxy.cn/trans.aspx?id=3094 铁路十二五规划全文和高清图] {{webarchive|url=https://web.archive.org/web/20121107145036/http://www.tjtdxy.cn/trans.aspx?id=3094 |date=2012-11-07 }}</ref>，同月江西省联合广东省共同向铁道部行文要求将赣深客专也纳入“十二五”<ref>[http://jiangxi.jxnews.com.cn/system/2011/05/06/011654689.shtml 南昌将有直达深圳“高铁”]</ref>；2011年底又有报道表示将于2013年动工<ref>[http://jiangxi.jxnews.com.cn/system/2011/12/12/011844144.shtml 昌吉赣客列专线预计2013年开工]</ref>。

2012年11月16日铁道部召开了昌吉赣客运专线预可研审查会，计划时速超过300公里<ref>[http://jxgz.jxnews.com.cn/system/2012/11/22/012186166.shtml 昌吉赣客专进行站点初步选址 力争明年开工建设]</ref>；11月18日铁道部人员在吉安对线路走向和车站位置进行实地考察，拟新建万安站和吉安西站<ref>[http://www.jxnews.com.cn/xxrb/system/2012/11/19/012181137.shtml 昌吉赣专线拟在吉安建两个新车站]</ref>；2012年底表示将计划在2013年开工<ref>[http://jndsb.jxnews.com.cn/system/2012/12/31/012233657.shtml 明年我省力争开建昌吉赣客专]</ref>。

== 走向方案 ==
该线路早期规划经过[[万安县|万安县]]并且不经过[[兴国县|兴国县]]，但是随后的探测对两条走向都进行了探测，并考虑走折中路线<ref>[http://www.jxnews.com.cn/xxrb/system/2013/06/05/012454026.shtml 昌吉赣客运专线会从谁家门前过]</ref>；2014年6月的详细环评显示昌赣客运专线在泰和并站后将偏西南下至万安县县城东北侧约9公里的[[枧头镇|枧头镇]][[建村村|建村村]]附近设[[万安站|万安站]]，随后转为偏南向东至兴国县县城西侧约8公里的[[埠头乡|埠头乡]][[蕉田村|蕉田村]]附近设[[兴国西站|兴国西站]]；根据环评后网友贴出的图片，在泰和到赣州之间一共有6种走向设计<ref>{{Cite web |url=http://bbs.jxnews.com.cn/forum.php?mod=viewthread&tid=2305451 |title=兴国弯请大家不要责备铁四院 |access-date=2014-10-11 |archive-url=https://web.archive.org/web/20141017203100/http://bbs.jxnews.com.cn/forum.php?mod=viewthread&tid=2305451 |archive-date=2014-10-17 |dead-url=yes }}</ref><ref>[http://www.doc88.com/p-8139297587003.html 昌吉赣客专可研总说明书]</ref>。

== 轶事 ==
动工后江西省铁路建设办公室特意向[[铁四院|铁四院]]发来感谢函<ref>[http://www.crfsdi.com.cn/sysWeb/NewsCenterDetailsPlatform.aspx?ArticleID=2066 昌吉赣客专开工 铁四院收获江西省铁办情真意切感谢函] {{Wayback|url=http://www.crfsdi.com.cn/sysWeb/NewsCenterDetailsPlatform.aspx?ArticleID=2066 |date=20150121124259 }}</ref>。

发改委的批文对时速使用的是“旅客列车设计行车速度：250公里/小时”而同天公布的连云港至镇江铁路为“速度目标值：250公里/小时”<ref>[http://www.ndrc.gov.cn/zcfb/zcfbtz/201411/t20141105_647423.html 国家发展改革委关于新建连云港至镇江铁路 可行性研究报告的批复]</ref>，使得批文公布后时速是250公里/小时还是350公里/小时一直成为该工程的热点话题<ref>[http://mt.sohu.com/20150510/n412768340.shtml 昌赣高铁350公里/小时毋庸置疑]</ref>。
