'''昌平城墙'''为历代昌平县城、昌平州城城墙，自西汉置县后数易其址，均位于今[[北京市|北京市]][[昌平区|昌平区]]境内。

== 历史 ==
昌平为西汉置县，最早与[[军都县|军都县]]同属[[上谷郡|上谷郡]]，自西汉至北魏初其城址位于今址东南、[[温榆河|温榆河]]北岸（今[[沙河地区|沙河镇]]辛力屯村一带），北魏初裁撤昌平县入军都县，东魏[[天平_(东魏)|天平]]年间复置昌平县，属[[平昌郡_(北魏)|平昌郡]]，并将县治改设在始建于战国末期的原'''军都县故城'''，位于今址西南（今[[马池口镇|马池口镇]]土城村一带，原军都县则改在今[[兴寿镇|兴寿镇]]东、西新城村一带新建，称军都县新城）<ref>清·吴履福等修，缪荃孙等纂，《光绪昌平州志·卷三·土地记》：昌平故城，《一统志》在今昌平州东南，汉置县，……后魏初省。……此汉晋至魏初昌平当在今沙河店迆东上下东郭二村之西，村名东郭，盖因城而得名也。军都故城，《一统志》在昌平州西十七里，汉置县，后魏置东燕州及昌平县于此，《魏书·地形志》东燕州天平中置，寄治幽州军都城，《水经注》在居庸关南、㶟余水北，当是双塔故城，今之故城村也。军都新城，《寰宇记》在幽州西北九十五里，本汉军都县，后魏移军都于今县东北二十里，旧志今州东四十里有军都邨，亦曰故县址，当是今新城村北之土城。白浮图城，旧志今名旧县，在州城西八里。按《元史》皇庆二年冬十月辛巳徙昌平县于新店，元初昌平县治白浮图城，见宋渤梁公祠碑记，是由白浮图城徙之，也不知何年复自新店徙回，俟考。</ref><ref>{{cite web | url=http://js.ifeng.com/humanity/detail_2015_06/09/3986323_0.shtml| title=《昌平军都城始建于战国末期 曾出土一堆堆的马骨》| publisher=凤凰网 | date=2015-06-09 | accessdate=2017-12-09}}</ref>。约在唐代[[建中_(唐朝)|建中]]年间昌平县治又自军都故城迁至距今址西八里的旧县村一带，建城名'''白浮图城'''<ref>清·麻兆庆，《昌平外志·卷一·沿革》：昌平故城考：案《括地志》居庸关在幽州昌平县西北四十里，以关东南四十里求唐昌平城，即军都故城是也。《元和郡县志》补曰昌平城在军都关南即此城。明陵大红门北有名担尘殿者，为朱孝廉蕡家别业，光绪初掘得宋俨墓志铭，建中四年四月葬幽州昌平县东北十里武安乡，是此时已迁旧县矣。建中三年朱滔称冀王，滔家太尉乡，以朱怀珪墓考之，虎谷东之太平庄也，南距旧县八里，似昌平北迁当在此时。辽金昌平以义冢幢记考之亦旧县也。元徙新店，至正二年徙回，明景泰三年徙永安城今州治。</ref>。元代[[皇庆|皇庆]]二年（1313年）一度再迁县治至其西南六里的新店，约在[[至正|至正]]年间仍迁回原址<ref>《昌平外志·卷一·沿革》：《元史·地理志》案皇庆二年冬十月辛巳，徙昌平县治于新店（旧县西南六里），新志云未知何年徙回。考皇庆徙后，泰定二年置谏议书院于昌平县，祀唐刘蕡，此乃治新店之昌平县也，刘司户祠纪略曰至正间又建祠旧县东，此乃县于至正时徙回旧县，故祠随之迁而又建也。马房辉重修儒学记至正二年正月祭圣庙，顾盼阙坏，捐资重修，此又县徙回之一证也。</ref>。

为加强对[[明十三陵|皇陵]]的守备，[[明代宗|明代宗]][[景泰_(年号)|景泰]]元年正月辛巳（1450年1月18日）下令于[[天寿山|天寿山]]南修筑护陵城，名'''永安城'''，内驻[[長陵_(明)|长陵]]、[[獻陵_(明)|献陵]]、[[景陵_(明)|景陵]]三卫官军，城周十二里（实一千四百九十二丈九尺八寸）<ref>《明英宗睿皇帝实录·卷之一百八十七·废帝郕戾王附录第五》：景泰元年春正月辛巳，命于天寿山之南筑城，周围十二里，以居长陵、献陵、景陵三卫官军，并移昌平县治于内。</ref>，高二丈一尺，近正方形，东、西、南三面均设瓮城，城内设十字街。景泰二年十月己卯（1451年11月7日）昌平县治、儒学、仓库等自旧县村迁至永安城（州志载为景泰三年迁址），以永安城兼作护陵城与昌平县城<ref>《明英宗睿皇帝实录·卷之二百九·废帝郕戾王附录第二十七》：景泰二年冬十月己卯，徙昌平县治并儒学仓库等衙门于长陵卫新筑土城之内。</ref>。[[万历|万历]]元年（1573年）陵卫增多，又在旧城南与之相连新筑一城，城周八十四丈五尺（不含旧城南墙），高一丈七尺。[[崇祯|崇祯]]九年（1636年）兵部侍郎[[张元佐|张元佐]]将旧城南墙拆除，用其砖石修补东城门楼，两城遂合为一城，城周共十里二十四步，长方形，护城河深、宽均约两丈。清康熙十四年（1675年）重筑城墙，增高至三丈，护城河深八尺、宽三丈<ref>《光绪昌平州志·卷三·土地记》：今州城：州故永安城，明景泰元年筑，东、西、南三门俱重门瓮城，内外各有层楼，周一千四百九十二丈九尺八寸，高二丈一尺，徙长、献、景三陵卫于内，以护陵寝，三年并昌平治徙焉。天顺三年天寿山守备廖镛奏建谯楼于城中，万历元年因陵卫续设益多，于城南复筑一城连之，墙减旧城四尺，只有南门瓮城，箭楼亦微小，东、西、南三面共八十四丈五尺，后屡经营造，两城皆甃以砖，至崇祯九年，兵部侍郎张元佐拆旧城南面砖石修补东城门楼，城遂合而为一，周一十里二十四步，池深广各二丈许。国朝康熙十四年重筑新旧城，均高三丈，池深八尺、阔三丈。</ref>。

自1954年起开始拆除城墙，至1986年东、西、南、北四条环路建成后原城墙已拆除殆尽<ref name="早期城址">{{cite web| url=http://e.bjcpn.com/epaper/2015/5/27/news7379_1.html| title=《昌平城之变迁》| publisher=昌平报| date=2015-05-27| accessdate=2017-12-09| deadurl=yes| archiveurl=https://web.archive.org/web/20171210015612/http://e.bjcpn.com/epaper/2015/5/27/news7379_1.html| archivedate=2017年12月10日| df=}}</ref>。
== 现状 ==
[[File:Former_Changping_Drum_Tower_in_1930s.jpg|thumb]]
西汉至北魏初的昌平城位于今沙河镇辛力屯村一带，已无迹可寻，其东侧有上下东郭村，即因城而得名，1985年曾在辛力屯村附近发现汉墓，或与城址有关<ref>{{cite web | url=http://news.xinhuanet.com/local/2015-11/10/c_128410420.htm|title=《东廓，城乎？村乎？》| publisher=新华网 |date=2015-11-10 | accessdate=2017-12-09}}</ref>。东魏至唐代的昌平城（军都故城）位于今城区西南8公里、马池口镇土城村一带，俗称“土城”，为长方形，南北0.75公里，东西0.5公里，仅存部分夯土城墙，2003年公布为昌平区文物保护单位<ref name="文物">{{cite web | url=http://www.bjchp.gov.cn/cpwhw/tabid/5651/InfoID/390390/frtid/5538/Default.aspx|title=《昌平区区级文物保护单位》| publisher=北京市昌平区文化委员会 |date=2016-09-14 | accessdate=2017-12-09}}</ref>。唐代至明初的昌平城（白浮图城）位于今[[城南街道_(北京市)|城南街道]]旧县村一带，原城墙材料在永安城的后续修葺过程中被逐渐拆取，至清代已不见旧城痕迹<ref>{{cite web | url=http://www.thepaper.cn/newsDetail_forward_1592182|title=《上海书评︱龙虎台前暑气深——罗新走向金莲川》| publisher=澎湃新闻 |date=2017-01-04 | accessdate=2017-12-09}}</ref>，村南尚存植于唐代的[[国槐|国槐]]，其后有观音庵，为村内曾有的15座祠庙中仅存的一座，其它如纪念[[狄仁杰|狄仁杰]]的狄梁公祠等今已不存<ref name="早期城址" />。

明景泰之后的昌平城（永安城）位于今[[城北街道_(北京市)|城北街道]]城区中心，原城墙已不存，其框架被建于1978年-1986年的东、西、南、北四条环路代替（其中西环路、北环路建于原护城河上，东环路建于原城墙基上，南环路建于原城墙外<ref name="早期城址" />），原旧城南城墙则位于今政府街南侧，今鼓楼东、西、南、北街即原旧城内的十字街，此外城内仍保留了大量与驻兵相关的胡同，如城南的康陵卫胡同、城中的长陵卫胡同、城北的永陵卫胡同，以及草场胡同、营房胡同、东、西盔甲胡同、大将军胡同、帅府墙胡同等，另一些胡同则与昔日的庙宇有关，如文庙胡同、红庙胡同、七圣庙胡同、龙王庙胡同、铁佛庵胡同等。原东门外有松园广里许，遍植松桧，无一杂树，以便向陵区内移植补缺，今仅存松园路地名<ref>{{cite web | url=http://bj.people.com.cn/n/2015/0901/c233080-26197229.html| title=《再说昌平永安：古城肌理看胡同》| publisher=人民网 | date=2015-09-01| accessdate=2017-09-18}}</ref><ref>清·于敏中，《日下旧闻考·卷一百三十七·京畿昌平州四》：原松园在昌平州东门外，延袤里许皆松桧，无一杂树，以备陵树缺移栽也，二百年来禁人樵采，遂成大林。（《燕都游览志》）</ref>。

原鼓楼东、西、南、北街交叉处曾建有谯楼（亦名鼓楼），明天顺三年（1459年）由天寿山守备[[廖镛|廖镛]]奏建，为城内的重要地标，1971年因阻碍交通被拆除。此外城内的主要建筑包括霸昌道署（其址今为昌平区法院，已无存）、昌平县署（其址今为昌平区政府，已无存）、[[昌平文庙|文庙]]（其址今为昌平二中，大成殿于1995年因扩建校舍被拆除，现仅存两棵柏树）<ref>{{cite web | url=http://news.ifeng.com/a/20150526/43834909_0.shtml| title=《永安城：护卫皇陵做先锋》| publisher=凤凰网 | date=2015-05-26 | accessdate=2017-09-18}}</ref>、燕平书院（其址今为昌平区检察院，已无存）、[[昌平城隍庙|城隍庙]]（仅存后殿，位于今昌平邮政局院内，为昌平区文物保护单位，2014年解体大修<ref>{{cite web | url=http://www.chinanews.com/cul/2014/04-12/6057047.shtml|title=《北京昌平城隍庙被拆后重建 市民称太折腾》| publisher=中国新闻网 |date=2014-04-12 | accessdate=2017-12-09}}</ref>）和[[昌平关帝庙|关帝庙]]（现存前、后殿各三间及[[抱厦|抱厦]]一间，位于昌平一中内，为昌平区文物保护单位，2013年大修）<ref name="文物" />等。
