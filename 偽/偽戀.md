《'''偽戀'''》（{{lang-ja|ニセコイ}}）是由[[古味直志|古味直志]]所作的一部[[校園漫畫|校園漫畫]]，於《[[週刊少年Jump|週刊少年Jump]]》2011年48號至2016年36・37合併號期間中連載。

== 劇情 ==
{{main|偽戀角色列表}}

日本[[黑道|黑道]]「'''集英組'''」第二代的高中生'''一條樂'''，雖然暗戀著女同學'''小野寺小咲'''，卻為了阻止黑幫鬥爭，而與另一個黑幫大小姐'''桐崎千棘'''假裝成一對戀人，而這齣暴走喜劇在加入了「[[警視總監|警視總監]]」的女兒'''橘萬里花'''及殺手'''鶇誠士郎'''之後，更是一發不可收拾…。

另以外傳《魔法甜點師小咲！》，原作中的小咲、千棘和萬里花等人變身為魔法少女，以對抗邪惡組織的姿態在作品中登場。

== 發展 == 
於[[集英社|集英社]]《[[週刊少年Jump|週刊少年Jump]]》2011年48號開始連載<ref>{{cite news|title=ジャンプのラブコメ連載「ニセコイ」TVアニメ化決定|url=http://natalie.mu/comic/news/91836|publisher=natalie|date=2013-06-03|accessdate=2014-11-27}}</ref>，台灣由[[東立出版社|東立出版社]]的《[[寶島少年|寶島少年]]》連載<ref>{{cite news|title=正統派（？）校園戀愛喜劇《偽戀》本季動畫好評播映中！第9集限定版 閃光登場|url=http://www.animen.com.tw/Page/News/NewsContent.aspx?ID=NS14030124|publisher=Animen|date=2014-03-21|accessdate=2014-11-27|deadurl=yes|archiveurl=https://web.archive.org/web/20141204200827/http://www.animen.com.tw/Page/News/NewsContent.aspx?ID=NS14030124|archivedate=2014-12-04}}</ref>。本作是古味直志在{{Nihongo|《[[雙藝士|雙藝士]]》|ダブルアーツ}}後推出的連載<ref>{{cite news|title=ジャンプNEXT！に古味直志＆古舘春一の新作掲載|url=http://natalie.mu/comic/news/43170|publisher=Natalie|date=2011-01-08|accessdate=2014-11-27|language=ja}}</ref>，是高中生男女主角的戀愛漫畫<ref>{{cite news|title=ダブルアーツの古味直志、新連載はラブコメ「ニセコイ」|url=http://natalie.mu/comic/news/59171|publisher=Natalie|date=2011-11-07|accessdate=2014-11-27|language=ja}}</ref>。

2015年，其連載話數超越《[[草莓100%|草莓100%]]》167話的紀錄，成為在《週刊少年Jump》上最長壽的戀愛喜劇作品。2016年8月完結<ref>{{cite news|script-title=ja:「ニセコイ」4年9カ月の連載に幕、10月のジャンプGIGAに古味直志 新作|url=http://natalie.mu/comic/news/197401|publisher=Natalie.mu|date=2016年8月8日|accessdate=2016年8月9日}}</ref>，共229話。

2013年5月，集英社公佈本作改編為電視動畫的消息，動畫將由SHAFT公司制作，於2014年1月開始播放<ref>{{Cite web|url=http://cartoon.southcn.com/a/2013-07/04/content_72570239.htm |title=新房昭之将负责执导动画《伪恋》 |date=2013-07-04 |work=南方动漫资讯 }}</ref><ref>{{Cite web |url=http://comic.qq.com/a/20130823/008557.htm |title=TV动画《伪恋》将于2014年1月开播 |work=腾讯动漫 |date=2013-08-23 }}</ref>。2014年10月宣布製作第2期<ref>{{cite news|title=漫畫家「古味直志」校園戀愛代表作《偽戀》將推出電視動畫第二期、千棘的母親可望登場！|url=http://www.animen.com.tw/Page/News/NewsContent.aspx?ID=NS14100043|publisher=Animen|date=2014-10-06|accessdate=2014-11-27|deadurl=yes|archiveurl=https://web.archive.org/web/20141204200524/http://www.animen.com.tw/Page/News/NewsContent.aspx?ID=NS14100043|archivedate=2014-12-04}}</ref><ref>{{cite news|title=Nisekoi: False Love Manga Gets 2nd TV Anime Season|url=http://www.animenewsnetwork.com/news/2014-10-01/nisekoi-false-love-manga-gets-2nd-tv-anime-season/.79421|publisher=動畫新聞網|date=2014-10-01|accessdate=2014-11-27|language=en}}</ref><ref>{{cite news|title=［ニセコイ］テレビアニメ第2期が制作決定　「ジャンプ」の人気ラブコメ|url=http://news.mynavi.jp/news/2014/10/06/002/|publisher=2014-10-06|date=|accessdate=2014-11-27|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20141006014850/http://news.mynavi.jp/news/2014/10/06/002/|archivedate=2014-10-06}}</ref>，隨後確定於2015年4月10日開始播放。中間亦兩度與漫畫[[單行本|單行本]]綑綁發售[[OVA|OVA]]。

== 轶事 ==
其中一名狂热读者，編輯部將其匿名为“千叶县的Y先生（{{lang|ja|千葉県のYさん}}）”，十分热爱本作角色橘万里花，多次在作品相关的投票或排行中给予支持。
*2013年第一次人气排行投票中，Y先生為万里花手写寄投了1500明信片票，超過了萬里花得票數的一半，最终万里花人气排行第4位。<ref name="acgmoon">{{cite web|title=伪恋结局《千叶县的Y桑终于攻略万里花》真爱竟然打破了次元壁… {{!}} 寂月神社|url=https://www.acgmoon.com/37218.html}}</ref>
*2014年1月13日，手游《伪恋:真假物语（{{lang|ja|ニセコイ マジコレ!?}}）》上线，首日万里花角色排行上首位，贡献分数最多的玩家是来自于千叶县的山本（YAMAMOTO），被认为是Y先生的支持。<ref name="acgmoon"/><ref>[https://news.gamme.com.tw/887577 撐不過一年《偽戀手機遊戲》宣布收攤 千葉縣的Y桑崩潰中？]</ref>
*2014年2月10日第二次人气排行投票中，万里花以4518票人气排行第2位，而且也激发了其他角色的爱好者的大量灌票，导致排行总投票数急剧上升，因此作品编辑部没再公布个人灌票数，以防效仿。另外Y先生也意外登上本次排行榜。<ref name="acgmoon"/><ref name="comicyu">{{cite web|title=《伪恋》第二回人气投票 投票读者“千叶县Y桑”上榜_国际_漫域_中国动漫综合门户|url=http://news.comicyu.com/2014/02/19/136003.html}}</ref>
*2015年3月30日的剧情投票中，万里花的剧情支持为1204票，其中Y先生投出超過800票。由于本次投票必须购买《[[周刊少年JUMP|周刊少年JUMP]]》并寄送内附的投票信件才能参加投票，可推知Y先生为此投入了大量财力。<ref name="acgmoon"/><ref>[https://news.gamme.com.tw/1119537 這次採用應募券投票制 Y桑終於沒辦法再灌票了…？]</ref>
*2016年10月，萬里花聲優[[阿澄佳奈|阿澄佳奈]]在廣播劇朗讀Y先生寄給她的感謝信，感謝她在獲得人氣優勝時為萬里花提出的短篇構想（番外篇「公主大人」）。<ref>[http://forum.gamme.com.tw/thread-86914-1-1.html 《偽戀》萬里花聲優阿澄佳奈廣播節目中朗讀 Y 桑寄來的感謝信]</ref><ref>{{Cite web |url=http://acg.178.com/201610/270862913698.html |title=打破次元之壁的宅男：扒一扒橘万里花真爱粉Y桑的奇迹物语！ |access-date=2017-06-26 |archive-url=https://web.archive.org/web/20171009045526/http://acg.178.com/201610/270862913698.html |archive-date=2017-10-09 |dead-url=yes }}</ref>
*2015年、2016年、2017年3月3日万里花生日时，Y先生寄給作品编辑部送给万里花的生日礼物（花束、月球土地權利書、玻璃鞋附鑰匙、繪本等等）。<ref name="acgmoon"/><ref>[https://twitter.com/nisekoi_k 『ニセコイ』公式アカウント]</ref>
由于其狂热程度，在作品连载杂志的介绍中广泛流传，最终在本作完结时，万里花表示最后的相亲对象是来自千叶县，被认为是向Y先生致敬。<ref name="acgmoon"/><ref name="jiangsu.china">{{cite web|title=打破次元壁！你见过粉丝攻略动漫女主角的吗 - 热点资讯 - 中国网•东海资讯|url=http://jiangsu.china.com.cn/html/ent/hot/6833545_1.html|access-date=2016-08-22|archive-url=https://web.archive.org/web/20160822154001/http://jiangsu.china.com.cn/html/ent/hot/6833545_1.html|archive-date=2016-08-22|dead-url=yes}}</ref>
