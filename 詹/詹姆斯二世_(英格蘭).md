'''詹姆斯二世'''（[[蘇格蘭|蘇格蘭]]稱為'''詹姆斯七世'''，{{bd|1633年|10月14日|1701年|9月16日|CatIdx=J}}），1685年到1688年間[[英格兰|英格兰]]、[[苏格兰|苏格兰]]和[[爱尔兰|爱尔兰]]的[[国王|国王]]，也是最后一位信奉[[天主教|天主教]]的[[英国君主|英国国王]]。其[[宗教|宗教]]政策和专权都遭到臣民反对，最终在[[光荣革命|光荣革命]]中被剥夺王位。其天主教儿子[[詹姆斯·弗朗西斯·爱德华·斯图亚特|詹姆斯·弗朗西斯·爱德华·斯图亚特]]在他被迫退位后也没有获得王位，王位落到了其[[新教|新教]]女儿[[玛丽二世_(英格兰)|玛丽二世]]和女婿[[威廉三世_(奥兰治)|威廉三世]]手中。以其子为合法国王的人则被称为[[詹姆斯党|詹姆斯派]]。

詹姆斯二世退位后受到法国国王[[路易十四|路易十四]]的保护。一直到他死后，路易、他的儿子詹姆斯·弗朗西斯·爱德华和孙子[[查尔斯·爱德华·斯图亚特|查尔斯·爱德华·斯图亚特]]还继续策划恢复詹姆斯派的王位，但最后也没有成功。

詹姆斯二世的另一个比较出名的封号是[[约克公爵|约克公爵]]。他的兄弟[[查理二世_(英格兰)|查理二世]]将[[北美洲|北美洲]]的一个殖民地以他的封号命名为[[纽约|纽约]]。

== 早年生活 ==
詹姆斯二世是[[查理一世_(英格蘭)|查理一世]]和他的王后[[法兰西的亨利埃塔·玛丽亚|亨莉雅妲·瑪利亞]]的第二个儿子。1644年他被封为[[约克公爵|约克公爵]]。

在[[英国内战|英国内战]]时，他在[[保皇黨|保皇黨]]的基地[[牛津|牛津]]暫避。1646年牛津被[[克伦威尔|克伦威尔]]攻克后他被监禁在圣詹姆斯宫。1648年他逃离詹姆斯宫逃到[[荷兰|荷兰]][[海牙|海牙]]。1649年查理一世被共和党人处死后，保皇党人立詹姆斯的哥哥为[[查理二世_(英格兰)|查理二世]]。但查理二世被[[奥利弗·克伦威尔|奥利弗·克伦威尔]]击败后被迫逃亡。

詹姆斯逃亡法国，他加入了[[法國軍事|法国军队]]，1656年查理与英国的敌人[[西班牙|西班牙]]同盟后他又加入了[[西班牙军队|西班牙军队]]。他的上级军官对他的评价都非常好。

1660年克伦威尔死后查理二世重返英国，詹姆斯随同他一起回到英国。虽然他在王位继承的顺序上居第一位，但当时他真的继承王位的可能不大，因为查理当时还是一个年轻人，还有生下繼承人的可能。1660年9月，詹姆斯与查理最重要大臣——[[第一代克拉伦登伯爵爱德华·海德|克拉倫登]]的女儿[[安妮·海德|安妮·海德]]结婚。

詹姆斯被任命为[[海軍大臣|海軍大臣]]（Lord High Admiral），在[[第二次英荷战争|第二次英荷战争]]和[[第三次英荷战争|第三次英荷战争]]中他是[[英國皇家海軍|英國皇家海军]]最高指挥官，雖然他作戰勇敢負責，但英軍仍在兩次[[英荷戰爭|英荷戰爭]]中落居下風，草草收場。1664年荷兰殖民地[[纽约|新阿姆斯特丹]]被攻克后被改名为[[纽约|纽约]]（新约克），作为对他的成果的纪念。他也是从事[[奴隶贸易|奴隶贸易]]的[[皇家非洲公司|皇家非洲公司]]的首领。

== 宗教 ==
1668年或1669年詹姆斯转化为罗马天主教徒。他在国会内的敌人通过了一部[[立誓法|立誓法]]。在这部法令中任何在英国接受军民官职的人必須發誓接受[[圣公会|圣公会]]的信仰（并诅咒一些天主教的仪式和信仰）。詹姆斯拒绝发誓，因此他放弃了最高海军大臣的职务。

查理反对詹姆斯皈依天主教，他下令詹姆斯的子女必须按新教培养長大。1671年詹姆斯的第一位夫人去世，查理容许詹姆斯于1673年娶了一个天主教徒为妻——{{le|摩德纳的玛丽|Mary of Modena}}。但英国人对天主教不信任，他们认为新的约克公爵夫人是[[教宗|教宗]]的间谍。

为了取得新教徒的欢心，1677年詹姆斯让他的女儿玛丽嫁给一个新教徒（后来的[[威廉三世_(英格兰)|威廉三世]]）。虽然如此，新教徒还是怕一个天主教徒会成为国王。尤其是當[[凱薩琳王后|查理二世的王后]]無法懷孕後，这个恐惧逐渐上升为歇斯底里的情緒。1678年，一个圣公会的[[教士|教士]]造谣说有一个杀查理而立詹姆斯为国王的“教宗的阴谋”，並指控詹姆斯等天主教徒涉入其中。[[清教徒|清教徒]]所支持的[[輝格黨_(英國)|輝格黨]]人士，瘋狂地捕殺了三十多人，並要求廢除詹姆斯的王位繼承權，使查理二世的王權降到最低點。為了躲開政治大風暴，1679年詹姆斯明智地离开了[[英格蘭|英格蘭]]，暫居[[西屬尼德蘭|西屬尼德蘭]]的[[布鲁塞尔|布鲁塞尔]]。1680年查理任命他为[[苏格兰|苏格兰]]最高委员长，他遷至[[爱丁堡|爱丁堡]]的[[荷里路德宫|荷里路德宫]]定居。

在英格蘭，许多有权势的[[国会议员|國會議員]]企图取消詹姆斯的继承权。有人甚至愿意让查理的新教徒私生子[[第一代蒙茅斯公爵詹姆斯·斯科特|蒙默斯公爵]]登上王位。1679年，当取消詹姆斯王位的法令几乎要通过国会时，查理宣布解散国会（这个法令是英国两党制的开始，[[輝格黨_(英國)|英国辉格党]]是支持这个法令的党派，而[[托利党|托利党]]是反对这个法令的党派），1680年和1681年召开的议会也因同一原因被解散。

1681年的国会被解散后没有再召开国会，民心逐漸轉向而支持查理國王，查理因此在1682年打敗輝格黨，重建[[君主專制|君主專制]]。詹姆斯得以于1682年重返英国，之後成为[[托利党|托利党]]的準領袖，查理于1684年任命他为[[海軍大臣_(英國)|海军大臣]]。

== 统治 ==
1685年查理二世没有合法子女就死了。在他临死前他还皈依了天主教，他的弟弟詹姆斯二世继位。1685年4月23日他在[[西敏寺|西敏寺]]加冕。一开始反对他的人很少。甚至一些保守的新教徒也支持他。1685年5月召开的新议会（以[[托利党|托利黨]]為主）似乎也对詹姆斯很友好，还授予他很高的終身收入，進一步強化[[君主專制|絕對君主制]]。

但詹姆斯刚刚登基，查理的私生子[[第一代蒙默斯公爵詹姆斯·斯科特|蒙默斯公爵]]就开始造反，称他自己是合法的国王。这次造反很快就被詹姆斯的心腹將領[[約翰·邱吉爾|約翰·邱吉爾]]給平定，蒙默斯公爵在[[伦敦塔|伦敦塔]]被处死，但这次事件使詹姆斯开始对他的臣民产疑心。他下令严惩造反者，这个过度的措施使得公众认为他们的国王是个暴君。为了保护自己，詹姆斯决定建立一支强大的军队——三萬[[常備軍|常備軍]]，而军队中的一些部队则是由天主教徒组成的，这使得他又与托利黨国会产生了矛盾。1685年11月他解散国会。在詹姆斯统治期间国会未再召开。

1686年宗教问题越来越严重。一个由詹姆斯树立的法官团宣布国王可以下令取消立誓法的规定。詹姆斯借机让天主教徒占据王国最高的职位。他在伦敦接见教宗的代表，这是从这是自[[玛丽一世_(英格兰)|玛丽一世]]以来第一次教宗代表来到英国。詹姆斯的告解師是一个[[耶稣会|耶稣会]]神父，他尤其被新教徒看作敌人，这些措施使得詹姆斯失去了本来支持他的同盟者。

此后詹姆斯又解除了反对天主教的伦敦主教的职务，其他一些新教官员也被他解除。1687年他下令废除所有迫害[[天主教|天主教]]徒和其他宗教信徒的法令（不清楚的是他废除了這些法令究竟是為了争取这些人的支持，还是為了确保[[宗教自由|宗教自由]]是正確的）。1687年他再次解散[[议会|议会]]，后来又改革政府来削弱贵族的力量。

他允许天主教徒占据[[牛津大学|牛津大学]]的两个最大的学院的重要职务，这也使得许多新教徒反对他。他还任命了四个天主教的主教为政府官员。

== 光荣革命 ==
{{main|光榮革命}}
1688年4月詹姆斯重申废除对[[天主教|天主教]]徒和[[异教|异教]]徒的迫害，并下令[[新教|新教]][[牧師|牧師]]必须在[[教堂|教堂]]裡宣读这个法令。当[[坎特伯雷|坎特伯雷]]的[[大主教|大主教]]和其他六个[[主教|主教]]上诉，希望国王對宗教政策三思时，他们被控告且遭逮捕，但后来被判无罪。1688年6月10日詹姆斯的儿子出生，这就加大了建立一个[[舊教|舊教]]王朝的危险了（有人称实际上詹姆斯的儿子是死胎，但别人换了一个活胎，不过这个说法没有任何根据），因此一些非常有影响的新教徒开始与詹姆斯的女婿威廉三世谈判。威廉三世在[[荷蘭共和國|荷兰共和國]]与[[天主教|天主教]]的[[法国君主列表|法国国王]][[路易十四|路易十四]]作战，被看作是新教的希望。

1688年6月30日一群新教贵族要求威廉带军进入英国。9月时威廉将要进攻英国已经非常明显，但詹姆斯拒绝了法王路易十四为他提供军队的建议，因为他怕这样更加引起英国人的反对。他相信他的军队足以抵抗威廉。但他错了。1688年11月5日威廉登陆后所有的新教军官都叛变了，詹姆斯自己的女儿安妮也参加了入侵军并带走了许多王室的支持者。12月11日詹姆斯将王国的玺印扔进[[泰晤士河|泰晤士河]]并企图逃往法国，但在[[肯特郡|肯特]]被捕。威廉不希望詹姆斯成为一个殉教者，因此于12月23日故意让他逃跑了。路易十四收留了詹姆斯，给了他一座宫殿和很高的收入。

詹姆斯离开王国时没有议会召开，虽然一般议会由君主召开，但威廉召集了一次“惯例议会”（惯例议会一般在王位空缺的时候召开，比如内战后将王位授予查理二世的就是一个惯例议会）。1689年2月12日惯例议会申明詹姆斯于12月11日逃离伦敦时就已经自动放弃王位了，因此目前王位空缺（会议没有将王位授予詹姆斯的儿子）。会议决定立詹姆斯的女儿玛丽为女王，她与她的丈夫威廉三世共同统治。同年4月11日[[蘇格蘭議會|苏格兰議会]]通过了同样的决议。

威廉和玛丽上台后首先通过了一个法令宣布惯例议会的决定生效，威廉和玛丽正式成为君主，詹姆斯放弃了王位。此外这个法令还申明詹姆斯滥用权利：他放弃了立誓法，七位主教因上述情况被捕，建立了一支常规军和施加酷刑。最后这个法令还写明了王位的继承人顺序：首先是威廉和玛丽的孩子（假如他们有孩子的话），然后是安妮公主和她的孩子，最后是威廉假如再娶的话所生育的孩子。

== 晚年 ==
1689年3月，在法國[[海軍上將|海軍上將]][[圖爾維爾|圖爾維爾]]的護航下，詹姆斯带领一支[[法军|法军]]在爱尔兰登陆。[[爱尔兰议会|爱尔兰议会]]没有随从[[英国议会|英国议会]]，而是宣布詹姆斯依然是[[爱尔兰君主列表|爱尔兰国王]]。但1690年7月1日詹姆斯在[[波尼战役|波尼战役]]中战败。他立刻逃亡[[法国|法国]]。由于他如此的懦弱，很多追随者也都放弃了他，以致在爱尔兰获得了“屎詹姆斯”（''Séamus á Chaca''）的称号。

詹姆斯此后住在法国[[圣日耳曼昂莱|圣日耳曼昂莱]]的王宫里，1696年有人企图[[暗杀事件列表|刺杀]][[威廉三世_(英格兰)|威廉三世]]让他重返王位，但没有成功。同年[[路易十四|路易十四]]推舉他为[[波兰国王|波兰国王]]。但詹姆斯拒绝了，因为他怕再接受[[波兰|波兰]]王位后，就與[[英国國王|英国國王]]寶座絕緣了。这使得路易十四很不高兴，此后路易也不支持詹姆斯了。1697年路易十四甚至与威廉签署了一个条约正式中止了双方的敌对。此后詹姆斯至死都似一个严格的忏悔者一般，也葬在他一直居住的[[圣日耳曼昂莱|圣日耳曼昂莱]]。

== 后事 ==
1694年[[玛丽二世_(英格兰)|玛丽二世]]去世，1702年[[威廉三世_(英格兰)|威廉三世]]逝世后詹姆斯年轻的女儿[[安妮_(英国女王)|安妮]]成为女王。在1701年公布的法令中申明：假如在1689年的法令中所规定的继承人全部无子死亡后，王位由一个德国的远亲漢諾威女选帝侯[[索菲_(汉诺威选侯夫人)|蘇菲亞]]和她的新教后代继承。因此1714年安妮死后王位由苏菲亚的儿子[[乔治一世_(大不列颠)|乔治一世]]继承。

詹姆斯的儿子在乔治一世继位后在苏格兰进行过一次暴动，但没有多少结果。他1766年死后他的两个儿子还试图重返王位。他的二子是一个枢机主教，无子，因此他死后实际上詹姆斯的家族就后继无人了。虽然如此至今一直有人认为詹姆斯是英国王位正统，他们宣称詹姆斯的一些远亲依然有继承权。目前这个王族系列的继承人应该是巴伐利亚公爵弗朗茨，虽然巴伐利亚家族没有人要求英国的王位，但依然有人称弗朗茨为“弗朗西斯二世”。
