'''小詹姆斯·約瑟夫·巴爾傑'''（{{lang-en|James Joseph Bulger, Jr.}}，{{bd|1929年|9月3日|2018年|10月30日|catIdx=Bulger, James Joseph}}），綽號'''白毛·巴爾傑'''（{{lang-en|Whitey Bulger}}），生於[[美國|美國]][[麻塞諸塞州|麻塞諸塞州]]多切斯特（Dorchester），美國幫派分子與罪犯，在[[波士頓|波士頓]]組織了以愛爾蘭裔成員為主的[[冬山幫|冬山幫]]（Winter Hill Gang），成為南波士頓勢力最大的幫派份子。

他被指控犯下多起犯罪，包括賭博、販毒、殺人、搶劫銀行與暴力犯罪。他曾與[[愛爾蘭共和軍|愛爾蘭共和軍]]合作，走私軍火。其弟{{le|威廉·巴爾傑|William Bulger}}，是麻塞諸塞州的前參議員。

==生平==
從1975年開始，他長期擔任[[美國聯邦調查局|美國聯邦調查局]]的[[線民|線民]]，FBI為了獲取情報，長期故意忽視與放任他進行犯罪。

1997年開始，新英格蘭地區的新聞媒體報導了巴爾傑的犯罪事件，引起政府的尷尬，FBI開始主動對他進行調查。

1994年12月23日巴爾傑逃離FBI的監視，從此失蹤了16年。他列名在FBI十大要犯名單之中，懸賞金僅次於賓拉登。

2012年6月遭到逮捕，2013年8月12日時則遭到陪審團裁定其謀殺、敲詐、洗錢、販毒以及非法藏械等31項罪名成立，其中包括有涉嫌參與11起謀殺案。<ref>{{Cite web |url= http://boston.com/metrodesk/2013/08/12/bulgerverdict/1Kj1TAeB6WgVH00lFmufgL/story.html |title= Whitey Bulger, notorious Boston gangster, convicted in sweeping racketeering case; jury finds he participated in 11 murders |author= Shelley Murphy、Milton J. Valencia和Martin Finucane |date= 2013年8月12日 |publisher= 《[[波士頓環球報|波士頓環球報]]》 |language= en |accessdate= 2013年8月14日 |quote=  }}</ref>

== 逝世 ==
2018年10月29日，巴爾傑從[[奧克拉荷馬州|奧克拉荷馬州]][[俄克拉荷馬城|俄克拉荷馬城]]的{{link-en|聯邦轉移中心 (俄克拉荷馬城)|Federal Transfer Center, Oklahoma City|聯邦轉移中心}}轉移到西維珍尼亞州的{{link-en|黑澤爾頓美國監獄|United States Penitentiary, Hazelton}}。10月30日上午8點20分，89歲的巴爾傑在監獄中被發現昏迷。巴爾傑坐在輪椅上，被多名囚犯用一個襪子包住的掛鎖和一把刀子毆打致死，他的眼睛差點被挖出來，舌頭幾乎被切斷了。<ref>{{Cite web |url = https://www.nbcnews.com/news/crime-courts/notorious-mob-boss-whitey-bulger-found-dead-prison-n926266 |title = James 'Whitey' Bulger dead: Notorious Boston mob boss killed inside West Virginia prison |date = 2018-10-31 |publisher = NBC News |language = en}}</ref>這是40天內發生了第三件兇殺案。<ref>{{Cite web |url = https://www.wtae.com/article/notorious-crime-boss-whitey-bulger-has-been-killed-in-prison/24439089 |title = Notorious crime boss Whitey Bulger killed in prison |date = 2018-10-30 |publisher = WTAE |language = en}}</ref>在最近的黑澤爾頓監獄死亡事件發生前幾天，懲教人員已經向國會發出警告，稱設施極其人手不足。位於麻薩諸塞州的[[美國黑手黨|黑手黨]]殺手Fotios "Freddy" Geas是策劃殺害巴爾傑的主要嫌疑人，而他沒有對他的職責提出異議<ref>{{Cite web |url = https://nypost.com/2018/10/31/meet-the-mob-hitman-suspected-of-killing-whitey-bulger/ |title = Meet the mob hitman suspected of killing Whitey Bulger |author = Lia Eustachewich |date = 2018-10-31 |publisher = New York Post |language = en}}</ref>。51歲的Geas和他的兄弟因為他們在幾件暴力犯罪事件中於2011年被判處終身監禁，其中包括2003年殺害{{link-en|吉諾維斯犯罪家族|Genovese crime family}}的老大{{link-en|阿道弗·布魯諾|Adolfo Bruno}}，他在麻薩諸塞州[[春田市_(麻薩諸塞州)|春田市]]的停車場被槍殺。<ref>{{Cite web |url = https://www.nbcnews.com/news/crime-courts/officials-whitey-bulger-killed-lock-sock-mafia-hitman-eyed-n929581 |title = 'Whitey' Bulger killed with a 'lock-in-the-sock,' mafia hitman eyed, officials say |author = Tom Winter, Andrew Blankstein, Rich Schapiro |date = 2018-11-01 |publisher = NBC News |language = en}}</ref>

2018年11月8日，在南波士頓的聖莫尼卡–聖奧古斯丁教堂（Saint Monica – Saint Augustine Church）為巴爾傑舉行[[安魂彌撒|葬禮彌撒]]。包括他的弟弟、前麻薩諸塞州參議院議長威廉·巴爾傑和Catherine Greig等的家庭成員都有出席。<ref>{{Cite web |url = https://www.bostonglobe.com/metro/2018/11/08/funeral-mass-held-for-james-whitey-bulger-south-boston/39OXzbTkWTb8oc313MxOSJ/story.html |title = Funeral Mass held for James 'Whitey' Bulger in South Boston |author = Emily Sweeney, Shelley Murphy |date = 2018-11-08 |publisher = The Boston Globe |language = en}}</ref>

巴爾傑被埋葬在波士頓近鄰{{link-en|西羅克斯|West Roxbury}}的{{link-en|聖約瑟夫公墓 (西羅克斯，麻薩諸塞州)|St. Joseph Cemetery (West Roxbury, Massachusetts)|聖約瑟夫公墓}}，刻有他父母名字的巴爾傑家族墓碑下。<ref>{{Cite web |url = https://web.archive.org/web/20181111015004/https://www.bostonglobe.com/metro/2018/11/08/photos-whitey-bulger-funeral-south-boston/nbplQBeVOuCagf8TC6sveP/story.html |title = Photos: 'Whitey' Bulger's funeral in South Boston |date = 2018-11-08 |publisher = The Boston Globe |language = en}}</ref>

== 流行文化 ==
[[馬田·史高西斯|馬田·史高西斯]]的2006年電影《[[無間道風雲|無間道風雲]]》中的角色Frank Costello（[[積·尼高遜|積·尼高遜]] 飾演）是大概根據巴爾傑而創作出來的，但是電影情節是改編自2002年香港電影《[[無間道|無間道]]》。

2015年電影《[[黑勢力_(2015年電影)|黑勢力]]》，描敍了他的犯罪生平。由[[強尼·戴普|強尼·戴普]]飾演巴爾傑，電影改編自迪克·莱尔（Dick Lehr）和杰拉德·-{zh-hans:奥尼尔;zh-hant:奧尼爾;zh-tw:歐尼爾;}-（Gerard O'Neill）的2001年非虛構書籍《''Black Mass: The True Story of an Unholy Alliance Between the FBI and the Irish Mob''》。<ref>{{Cite web |url = https://www.bostonmagazine.com/arts-entertainment/2013/06/20/who-should-play-whitey-bulger-black-mass/ |title = Who Should Play Whitey Bulger in Black Mass? |author = Bryanna Cappadona |date = 2013-06-20 |publisher = Boston |language = en}}</ref>
