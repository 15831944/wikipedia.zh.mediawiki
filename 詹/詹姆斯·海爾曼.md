'''詹姆斯·M·海爾曼'''（{{bd|1980年}}，James M. Heilman，又稱「Doc James」）是一位加拿大的[[急診科醫生|急診科醫生]]、[[维基人|維基人]]，是倡議應當提升{{le|維基百科衛生保健相關內容|Health information on Wikipedia}}的行動主義者。他持續的推動鼓勵其他和他一樣的醫療從業人員一起來貢獻維基百科這個史上最大的線上百科全書。

他是一位活躍貢獻[[Wikipedia:医学专题|維基百科:医学专题]]的維基編輯、也曾於2010年到2013年間擔任[[维基媒体基金会|維基媒體加拿大分會]]的會長，他同時創立了維基醫學專題基金會並擔任過會長<ref name="ebola">{{Template:Cite web|url = 海爾曼是[[不列顛哥倫比亞大學|不列顛哥倫比亞大學]]急診部門的臨床指導醫師<ref name="UBC">{{Template:Cite web|url = http://emergency.med.ubc.ca/person/james-heilman/|title = James Heilman, MD, CCFP-EM|publisher = [[University_of_British_Columbia|University of British Columbia]]|accessdate = March 19, 2014}}</ref> ，同時也在他居住的[[克兰布鲁克|克兰布鲁克]]擔任{{Le|東庫特尼地區醫院|Regional_District_of_East_Kootenay}}的急診科醫師<ref name="digest">{{Template:Cite web|url = http://www.readersdigest.ca/magazine/true-stories/is-google-making-us-sick-0?page=0,3|title = Is Google Making Us Sick?|work = [[Reader's_Digest|Reader's Digest]]|date = September 2013|accessdate = January 9, 2014|last = Laidlaw|first = Katherine}}</ref><ref name="who">{{Template:Cite journal|last = Fleck|first = Fiona|last2 = |first2 = |date = January 1, 2013|title = Online encyclopedia provides free health info for all.|url = http://www.who.int/bulletin/volumes/91/1/13-030113.pdf|journal = Bulletin of the World Health Organization|publisher = World Health Organization|volume = 91|issue = 1|pages = 8–9|doi = 10.2471/BLT.13.030113|pmid = 23397345|accessdate = January 9, 2014|pmc = 3537258}}</ref>。

== 童年與學生時期 ==
海爾曼在[[薩斯喀徹溫|薩斯喀徹溫]]出生、成長<ref name="bot">{{Template:Cite web|url = https://wikimediafoundation.org/wiki/Board_of_Trustees|title = Board of Trustees|publisher = Wikimedia Foundation|accessdate = 20 August 2015}}</ref> 。他2000年畢業於[[萨斯喀彻温大学|萨斯喀彻温大学]]的解剖學[[理學士|理學士]]，而後他在2003年繼續拿到了[[醫學博士|醫學博士]]學位<ref name="who">{{Template:Cite journal|last = Fleck|first = Fiona|last2 = |first2 = |date = January 1, 2013|title = Online encyclopedia provides free health info for all.|url = http://www.who.int/bulletin/volumes/91/1/13-030113.pdf|journal = Bulletin of the World Health Organization|publisher = World Health Organization|volume = 91|issue = 1|pages = 8–9|doi = 10.2471/BLT.13.030113|pmid = 23397345|accessdate = January 9, 2014|pmc = 3537258}}</ref> 。他後來在[[不列颠哥伦比亚|不列颠哥伦比亚]]完成了他的住院醫師階段<ref name="bot">{{Template:Cite web|url = https://wikimediafoundation.org/wiki/Board_of_Trustees|title = Board of Trustees|publisher = Wikimedia Foundation|accessdate = 20 August 2015}}</ref>。

== 行醫 ==
海爾曼先是在穆斯喬聯合醫院工作到2010年，這是一所在薩斯喀徹溫[[穆斯乔|穆斯乔]]的醫院。在這之後，他開始了在{{Tsl|en|Regional_District_of_East_Kootenay|東庫特尼地區}}醫院的工作<ref name="who">{{Template:Cite journal|last = Fleck|first = Fiona|last2 = |first2 = |date = January 1, 2013|title = Online encyclopedia provides free health info for all.|url = http://www.who.int/bulletin/volumes/91/1/13-030113.pdf|journal = Bulletin of the World Health Organization|publisher = World Health Organization|volume = 91|issue = 1|pages = 8–9|doi = 10.2471/BLT.13.030113|pmid = 23397345|accessdate = January 9, 2014|pmc = 3537258}}</ref><ref name="white">{{Template:Cite news|last = White|first = Patrick|title = Rorschach and Wikipedia: The battle of the inkblots|url = http://www.theglobeandmail.com/news/national/rorschach-and-wikipedia-the-battle-of-the-inkblots/article4280876/|accessdate = January 12, 2014|newspaper = [[The_Globe_and_Mail|The Globe and Mail]]|date = July 29, 2009}}</ref>。2012年10月，他被指派為東庫特尼地區醫院的[[急診科|急診科]]部門主管<ref name="who">{{Template:Cite journal|last = Fleck|first = Fiona|last2 = |first2 = |date = January 1, 2013|title = Online encyclopedia provides free health info for all.|url = http://www.who.int/bulletin/volumes/91/1/13-030113.pdf|journal = Bulletin of the World Health Organization|publisher = World Health Organization|volume = 91|issue = 1|pages = 8–9|doi = 10.2471/BLT.13.030113|pmid = 23397345|accessdate = January 9, 2014|pmc = 3537258}}</ref>。

== 維基百科的編輯與倡議 ==
[[File:Q&A_with_James_Heilman_on_teaching_with_Wikipedia.webm|right]]進行編輯維基百科的演講，觀眾提問時的照片。]]

海爾曼在2008年開始從醫學相關的維基百科條目開始編輯，他從那時起便一直鼓勵他的醫師同仁一起參與維基的醫學內容編輯<ref name="who">{{Template:Cite journal|last = Fleck|first = Fiona|last2 = |first2 = |date = January 1, 2013|title = Online encyclopedia provides free health info for all.|url = http://www.who.int/bulletin/volumes/91/1/13-030113.pdf|journal = Bulletin of the World Health Organization|publisher = World Health Organization|volume = 91|issue = 1|pages = 8–9|doi = 10.2471/BLT.13.030113|pmid = 23397345|accessdate = January 9, 2014|pmc = 3537258}}</ref>。他在一個很漫長的值班夜裡，開始對編輯維基百科產生興趣。當他查閱[[肥胖症|肥胖症]]的條目時發現了不少的錯誤。「我發現我其實可以修正這些錯誤。我進行了大量的編輯並且大幅提升了條目的品質。我好像從此就無法自拔地迷上了。」他在2011年告訴{{le|漢彌爾頓觀察報|Hamilton_Spectator}}<ref name="Spec">{{Template:Cite news|last = Mcneil|first = Mark|title = Wikipedia makes a house call to Mac|url = http://www.thespec.com/news-story/2205263-wikipedia-makes-a-house-call-to-mac/|accessdate = January 12, 2014|newspaper = The Spec|date = October 4, 2011}}</ref>。

海爾曼參與了一項維基醫學專題基金會與[[無國界譯者|無國界譯者]]共同合作的前導計畫，致力於改善英文維基百科的醫學內容條目並將重要的內容翻譯到其他語言中<ref name="cohen2012">{{Template:Cite news|last = Cohen|first = Noam|title = Book That Plagiarized From Wikipedia Is Pulled From Market|url = http://mediadecoder.blogs.nytimes.com/2012/06/12/book-that-plagiarized-from-wikipedia-is-pulled-from-market/|accessdate = January 14, 2014|newspaper = New York Times|date = June 12, 2012}}</ref><ref>{{Template:Cite journal|last = Teigen|first = Sarah|title = Medical translations for minority languages|url = http://translatorswithoutborders.org/sites/all/themes/twb/images/news/Teigen-MedTrnsltnsMinorityLang-OctNov131.pdf|accessdate = January 12, 2014|newspaper = Multilingual|date = October–November 2012|via = TranslatorsWithoutBorders.org}}</ref><ref>{{Template:Cite news|last = Yeung|first = Lien|title = Wikipedia's medical errors and one doctor's fight to correct them|url = http://www.cbc.ca/news/canada/british-columbia/wikipedia-s-medical-errors-and-one-doctor-s-fight-to-correct-them-1.2743268|publisher = [[CBC_News|CBC News]]|date = August 21, 2014}}</ref>。維基醫學專題基金會也跟[[加利福尼亚大学旧金山分校|加利福尼亚大学旧金山分校]]合作，透過提供學生學分來招募科學文獻的編輯志工並提升醫學相關的維基百科品質<ref>{{Template:Cite web|url = http://www.dailydot.com/lifestyle/med-school-wikipedia-edit/|title = Doctors prescribe better editors for Wikipedia—themselves|work = The Daily Dot|date = 1 October 2013|accessdate = 6 April 2015|author = Sankin, Aaron}}</ref>。在2014年，維基醫學專題基金會也與{{le|考克蘭實證醫學資料庫|Cochrane_Collaboration}}成為夥伴，提升維基百科上資料的正確性與可靠性。有關這種合作，海爾曼提到：「維基百科運作的方式是所有內容完全以所列的參考資料為準，若用來寫維基百科的參考資料是最好的，有機會讓維基百科中有最好的資訊。」<ref>{{Template:Cite web|url = http://www.nationaljournal.com/health-care/wikipedia-is-a-massively-popular-yet-untested-doctor-20140220|title = Wikipedia Is a Massively Popular (Yet Untested) Doctor|work = [[National_Journal|National Journal]]|date = 20 February 2014|accessdate = 21 September 2014|author = Ritger, Clara}}</ref>

海爾曼在2014年[[維基媒體國際會議|維基媒體國際會議]]提到，有93%的醫學系學生使用維基百科，因此他認為「修正維基百科上的內容」對每個關心醫療保健的人而言，都是現今重要的工作<ref>{{Template:Cite web|url = http://www.timeshighereducation.co.uk/news/wikimania-student-medics-get-credit-for-webside-manner/2015117.article|title = Wikimania: student medics get credit for webside manner|work = Times Higher Education|date = 14 August 2014|accessdate = 12 January 2015|author = Reisz, Matthew}}</ref>。

海爾曼在2015年6月選為[[维基媒体基金会|维基媒体基金会]]董事.<ref name="board">{{Template:Cite web|url = http://blog.wikimedia.org/2015/06/05/board-election-results/|title = Wikimedia Foundation Board election results are in|work = Wikimedia blog|date = 5 June 2015|accessdate = 11 June 2015|author = Varnum, Gregory}}</ref>。在2015年12月，董事會除去了海爾曼的董事職位，此一決定造成維基社群之間很大的爭議<ref name="removal-resolution">{{cite web |url= https://wikimediafoundation.org/wiki/Resolution:James_Heilman_Removal |title = Resolution:James Heilman Removal |work = Wikimedia Foundation Board of Trustees |date = 28 December 2015 |accessdate = 29 December 2015}}</ref><ref>{{cite web|url=https://lists.wikimedia.org/pipermail/wikimedia-l/2015-December/080479.html|title=[Wikimedia-l] Announcement about changes to the Board|author=|date=|work=wikimedia.org|accessdate=January 7, 2016}}</ref><ref name=Lih>{{cite news|last1=Lih|first1=Andrew|title=Wikipedia just turned 15 years old. Will it survive 15 more?|url=https://www.washingtonpost.com/news/the-intersect/wp/2016/01/15/wikipedia-just-turned-15-years-old-will-it-survive-15-more/|accessdate=16 January 2016|work=[[華盛頓郵報|The Washington Post]]|date=2016-01-15|language=en-US}}</ref>。在除去了海爾曼董事職位後，基金会發表一個聲明，認為海爾曼已失去基金会的信任。海爾曼自己提到：「在前幾週基金会曾經給他自行辭職的的選項，不過身為社群選出的董事，我認為我的任務是來自選我的社群，因此拒絕辭職，我認為這様的舉動是辜负选我的人。」<ref>{{cite web | url=http://www.theregister.co.uk/2016/01/12/wikimedia_dumps_elected_trustee/ | title=Wikimedia Foundation bins community-elected trustee | work=The Register | date=2016-01-12 | accessdate=2016-01-27| author=Orlowski, Andrew}}</ref>

== 研究著作 ==
2014年5月之前，海爾曼和[[多倫多大學|多倫多大學]]的Samir Grover進行一個研究，指定醫學院的學生用維基百科或是醫學教科書進行測試，確認哪一個比較準確<ref>{{Template:Cite web|url = http://www.theatlantic.com/health/archive/2014/05/can-wikipedia-ever-be-a-definitive-medical-text/361822/|title = Can Wikipedia Ever Be a Definitive Medical Text?|work = The Atlantic|date = 7 May 2014|accessdate = 30 March 2015|author = Beck, Julie}}</ref>。海爾曼也和[[微软|微软]]一起進行研究，發現在伊波拉病毒影響最大的三個國家中，維基百科是最多人使用的疾病資訊來源<ref>{{Template:Cite web|url = http://www.cmaj.ca/content/early/2015/02/02/cmaj.109-4982|title = WikiProject Medicine Making Progress|work = CMAJ|date = 27 January 2015|accessdate = 30 January 2015|author = Murray, Terry}}</ref>。在2015年，海爾曼和Andrew West發表一個研究，指出2008年至2013年之間，關注醫學條目編輯的維基人減少了40%<ref name="pinsker">{{Template:Cite web|url = http://www.theatlantic.com/business/archive/2015/08/wikipedia-editors-for-pay/393926/|title = The Covert World of People Trying to Edit Wikipedia—for Pay|work = The Atlantic|date = 11 August 2015|accessdate = 11 August 2015|author = Pinsker, Joe}}</ref>。這些結論及以其他維基百科上醫學內容產出及被使用的詳細分析，都在2015年的{{le|醫學網際網路|Journal of Medical Internet Research}}期刊中發表<ref>{{Template:Cite journal|last1 = Heilman|first1 = JM|last2 = West|first2 = AG|title = Wikipedia and medicine: quantifying readership, editors, and the significance of natural language.|journal = Journal of medical Internet research|date = 4 March 2015|volume = 17|issue = 3|pages = e62|pmid = 25739399|url = http://www.jmir.org/2015/3/e62/}}</ref>。
