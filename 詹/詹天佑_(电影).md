《'''詹天佑'''》，中国大陆电影，导演[[孙道临|孙道临]]、[[姚寿康|姚寿康]]，主演[[冯淳超|冯淳超]]、[[孔祥玉|孔祥玉]]、[[高明|高明]]、[[卢奇|卢奇]]、[[林连昆|林连昆]]、[[戴兆安|戴兆安]]、[[惠娟艳|惠娟艳]]。

== 剧情 ==
1904年（[[大清|大清]][[光绪|光绪]]三十年），[[清政府|清政府]]决心修筑[[京城|京城]]到[[张家口|张家口]]的[[铁路|铁路]]，[[大英帝国|大英帝国]]和[[沙皇|沙皇]][[俄国|俄国]]为了争夺修筑权利益互相向清政府施压，导致清政府骑虎难下，最后计划破产，便自行修建。清政府旋即在招聘科技人才，[[耶鲁大学|耶鲁大学]]毕业生詹天佑最终被招聘，并且成功设计了京张铁路。孙中山等革命人士建立民国之后，提出“全国铁路网”，詹天佑再度出山，建立了贯穿南北的铁路。<ref name="kill " >{{cite web |url= http://ent.sina.com.cn/m/c/2001-09-26/58453.html|title=孙道临了却夙愿 电影《詹天佑》首映受好评 |accessdate=  2001年9月26日|publisher=新浪网 |archiveurl=http://ent.sina.com.cn/m/c/2001-09-26/58453.html |archivedate= 2013年1月2日|work= |language=zh-cn}}</ref>

==花絮==
该片从剧本写作到拍摄完毕，一共花费了3年，历尽艰辛。<ref name="tiger " >{{cite web |url=http://news.sina.com.cn/o/2005-03-16/14285375709s.shtml |title=孙道临不是“詹天佑”(图) |accessdate= 2005年3月16日|publisher=新浪网 |archiveurl= http://news.sina.com.cn/o/2005-03-16/14285375709s.shtml|archivedate=2013年1月2日 |work= |language=zh-cn}}</ref>

詹天佑的名言“生命有长短、命运有沉升。”是该片的结束语。<ref name="tiger " />

1963年，[[北京电影学院|北京电影学院]]导演[[凌子风|凌子风]]计划拍摄《詹天佑》，并且邀请此片导演孙道临出演詹天佑，但是由于[[文化大革命|文化大革命]]的到来而被迫终止。<ref name="kill " /><ref name="bee " >{{cite web |url=http://www.people.com.cn/GB/wenyu/64/127/20010924/567951.html |title= 电影《詹天佑》国庆主打|accessdate= 	2001年9月24日 |publisher=人民网 |archiveurl=http://www.people.com.cn/GB/wenyu/64/127/20010924/567951.html |archivedate=2013年1月2日 |work= |language=zh-cn}}</ref>

该片获得了中共领导高层的肯定，[[江泽民|江泽民]]、[[丁关根|丁关根]]、[[徐光春|徐光春]]、[[傅志寰|傅志寰]]都对该片评价甚高。<ref>{{cite web |url=http://www.people.com.cn/GB/wenyu/64/127/20010922/567409.html |title= 江泽民称赞电影《詹天佑》弘扬了爱国主义精神|accessdate=  2001年9月22日|publisher= 人民网|archiveurl=http://www.people.com.cn/GB/wenyu/64/127/20010922/567409.html |archivedate=2013年1月2日 |work= |language=zh-cn}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/c/2001-09-23/58162.html |title=刻画民族志士振兴中华 江泽民高度评价《詹天佑》 |accessdate=2001年9月23日 |publisher=新浪网 |archiveurl=http://ent.sina.com.cn/m/c/2001-09-23/58162.html |archivedate=2013年1月2日 |work= |language=zh-cn}}</ref>
