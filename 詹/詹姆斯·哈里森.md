== 早年生活 ==
詹姆斯·哈里森出生于1936年12月27日。<ref name=aoty>{{cite web |url=http://www.australianoftheyear.org.au/honour-roll/?view=fullView&recipientID=459 |title=James Harrison OAM |publisher=australianoftheyear.org.au |accessdate=2014-06-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20140611190222/http://www.australianoftheyear.org.au/honour-roll/?view=fullView&recipientID=459 |archivedate=2014年6月11日 |df= }}</ref> 14岁的时候，哈里森接受了一次大手术，而在手术中他被输入了{{convert|13|l||lk=on}}的血液。<ref name="DailyMail" /> 术后他在床上休息了三个月，而就在他休息的这段时间里做出了一个决定。他决定从18岁时开始献血，因为18岁是澳大利亚规定的献血最小年龄。<ref name="DailyMail" />

== 献血生涯 ==

== 立场观点 ==
2007年，詹姆斯·哈里森批评澳大利亚政府向外国公司开放本国的血浆捐献。哈里森认为开放血浆捐献会打击本国献血者的积极性。该项开放政策源于澳大利亚与美国的自由贸易协定。<ref name="TheAustralian">{{cite web|url=http://www.theaustralian.com.au/news/opinion/james-harrison-fta-threatens-blood-donor-system/story-e6frg6zo-1111112853758|title=James Harrison: FTA threatens blood donor system|date=2007-01-19|work=[[The_Australian|The Australian]]|accessdate=2010-04-03|location=[[Australia|Australia]]}}</ref>

== 荣誉奖励 ==
詹姆斯·哈里森在1999年6月7日被授予[[澳大利亞勳章|澳大利亚勋位奖章]]。<ref>{{cite web|title=Award Extract - Australian Honours Search Facility|url=https://honours.pmc.gov.au/honours/awards/878145|publisher=澳大利亚总理内阁部|accessdate=2018-05-13}}</ref> 2011年，詹姆斯·哈里森被提名为“年度澳大利亚人奖”新南威尔士州地方英雄。<ref name=aoty/>
