'''詹順貴'''（{{bd|1963年|catIdx=Z}}），[[台灣|台灣]][[臺中市|台中]]人，台灣[[環境法|環境法]][[律師|律師]]出身，曾任[[台灣動物社會研究會|台灣動物社會研究會]]理事長、[[民間司法改革基金會|民間司法改革基金會]]執行委員等職<ref>{{Cite web |url= http://www.libertytimes.com.tw/2012/new/dec/16/today-taipei6-3.htm | author=劉力仁|language=zh-tw ||title=《小檔案》|date= 2012-12-16 |publisher=[[自由時報|自由時報]] }}</ref>。2016年3月出任[[行政院環境保護署|行政院環境保護署]]副署長，兩年多來屢傳環評爭議，後於2018年9月遞出辭呈，遲至同年10月8日確定請辭。

==簡介==
===早年===
詹順貴的祖父是[[佃農|佃農]]、父親是工人，小時候因為父親工作關係而住在[[高雄縣|高雄縣]]，小學五、六年級就讀[[高雄縣|高雄縣]][[橋頭區|橋頭鄉]][[高雄市橋頭區甲圍國民小學|甲圍國民小學]]，導師是台灣本土文學大師[[葉石濤|葉石濤]]。高中時就讀[[台中一中|台中一中]]。1988年，詹順貴一退伍就考上[[中國國際商業銀行|中國國際商業銀行]]，擔任四年法務人員。後來決定離開銀行業，1992年考上並擔任律師<ref name="《真情人物》詹順貴">{{cite web|language=zh-tw|url=http://www.libertytimes.com.tw/2012/new/dec/16/today-taipei6.htm|title=《真情人物》詹順貴 用訴訟伸張環境正義|author=劉力仁|publisher=自由時報|date=2012-12-16|accessdate=2013-08-22}}</ref>。
===投入環保運動===
1990年代，台灣興起[[高爾夫球場|高爾夫球場]]興建熱潮，環保團體四處抗爭，他在幕後協助環保團體分析法令，開始踏入環保運動<ref>[https://web.archive.org/web/20151119112847/http://news.ltn.com.tw/news/life/paper/417599 法律是最好的武器　自由時報　2010-08-08] {{webarchive|url=https://web.archive.org/web/20151119112847/http://news.ltn.com.tw/news/life/paper/417599 |date=2015-11-19 }}</ref>。2005年曾擔任[[行政院環境保護署|行政院環境保護署]]環境影響評估審查委員會委員。近年來他長期替環保團體提供法律協助，並且義務幫弱勢團體打官司，曾參與2003年[[雲林縣|雲林]][[林內鄉|林內]][[焚化爐|焚化爐]]、[[新店區|新店]][[安坑地區|安坑]]垃圾掩埋場、[[新竹縣|新竹]][[橫山鄉_(臺灣)|橫山]]垃圾掩埋場、[[美麗灣度假村爭議|台東杉原美麗灣度假村]]、[[北纜案|北纜案]]、[[中部科學園區|中部科學園區]]三期及四期開發案訴訟、雲林麥寮[[六輕|六輕]]大火、[[大埔徵地事件|大埔徵地事件]]、[[反台南鐵路東移自救會|反台南鐵路東移自救會]]<ref>{{Cite web |url=http://newtalk.tw/news/2014/05/12/47197.html| author=林雨佑|language=zh-tw ||title=南鐵自救會北上陳情 籲鐵路東移重啓環評|date= 2014-05-12|publisher=[[新頭殼|新頭殼]]}}</ref><ref>{{Cite web |url=http://www.cna.com.tw/news/asoc/201405120230-1.aspx| author=黃意涵|language=zh-tw ||title=反南鐵東移 自救會表達訴求|date= 2014-05-12|publisher=[[中央通訊社|中央通訊社]]}}</ref>等訴訟案，協助環保團體將環保議題從街頭運動帶進法庭。此外，詹順貴也投入《環境基本法》、《[[國土計畫法|國土計畫法]]》、《環境損害賠償法》、《[[海岸法|海岸法]]》等法案及民間版草案之研擬。除了環境議題之外，詹順貴亦長期關注[[智能障礙|智能障礙]]者，曾經擔任[[心路社會福利基金會|心路社會福利基金會]]、[[中華民國智障者家長總會|中華民國智障者家長總會]]義務律師長達十年。另外他也關心[[台灣原住民|台灣原住民]]議題，曾參與[[新竹縣|新竹縣]][[司馬庫斯|司馬庫斯]]風倒[[櫸木|櫸木]]事件訴訟案<ref name="《真情人物》詹順貴"/>。

2013年8月3日，詹順貴與[[台灣蠻野心足生態協會|台灣蠻野心足生態協會]]秘書長[[林子淩|林子淩]]在[[新北市|新北市]]新店戶政事務所[[登記結婚|登記結婚]]，兩人都是[[再婚|再婚]]，證婚人是[[立法委員|立法委員]][[林淑芬|林淑芬]]與[[國立政治大學|國立政治大學]]教授[[徐世榮|徐世榮]]<ref>{{Cite web |url= | author=張文馨|language=zh-tw ||title=濕地法過了 「我們結婚吧！」|date= 2013-08-03|publisher=[[聯合晚報|聯合晚報]]}}</ref>。2014年8月3日，詹順貴與林子淩成立[[惜根台灣協會|惜根台灣協會]]。

2014年9月29日，詹順貴在其[[部落格|部落格]]與[[風傳媒|風傳媒]]發文，以[[臺南市區鐵路地下化計畫|臺南市區鐵路地下化計畫]]等三個事件三問[[民主進步黨主席|民主進步黨主席]][[蔡英文|蔡英文]]，諷刺[[民主進步黨|民主進步黨]]已經與[[中國國民黨|中國國民黨]]毫無差異<ref>[http://thomas0126.blogspot.tw/2014/09/blog-post_29.html 民進黨異於國民黨幾希？]，2014年9月29日，詹順貴的部落格</ref><ref>[http://www.storm.mg/article/23663 三問蔡英文─民進黨異於國民黨幾希？]，2014年9月29日，風傳媒</ref>。

===參與政府運作===
2014年12月8日，[[台北市市長|台北市市長]]當選人[[柯文哲|柯文哲]]拜會[[宜蘭縣縣長|宜蘭縣縣長]][[林聰賢|林聰賢]]，表達支持[[北宜直線鐵路|北宜直線鐵路]]走最短路線，也就是經過[[翡翠水庫|翡翠水庫]][[集水區|集水區]]的路徑；身兼柯文哲市政顧問的詹順貴表示，柯文哲發言沒有徵詢市政顧問團的意見，若柯文哲對影響弱勢權益或環境議題再隨意發言，他會辭去市政顧問，不再為柯文哲的發言背書。<ref>[http://www.ettoday.net/news/20141209/436406.htm 不滿柯文哲北宜直鐵主張　詹順貴揚言辭市政顧問]</ref>2014年12月18日，詹順貴因認為柯文哲宣布[[臺北翡翠水庫管理局|臺北翡翠水庫管理局]]局長[[劉銘龍|劉銘龍]]接任[[臺北市政府環境保護局|臺北市政府環境保護局]]局長的行為不尊重「環保局局長遴選委員會」，宣布辭去市政顧問一職<ref>{{Cite news|url=http://www.ettoday.net/news/20141219/441105.htm|title=柯P找劉銘龍接環保局長 詹順貴怒辭市政顧問|language=zh-tw|publisher=[[ETtoday新聞雲|ETtoday新聞雲]]|author=政治中心|date=2014-12-19|accessdate=2014-12-21}}</ref>。

2015年8月25日，[[台灣綠黨|台灣綠黨]]召開記者會，宣布2016立法委員選舉將由召集人之一[[李根政|李根政]]、環境律師詹順貴、[[台灣伴侶權益推動聯盟|台灣伴侶權益推動聯盟]]執行長[[許秀雯|許秀雯]]及[[中華電信工會|中華電信工會]]秘書長[[張麗芬|張麗芬]]擔任全國不分區立法委員候選人。

;擔任環保署副署長
2016年3月24日，詹順貴進入[[蔡英文政府|蔡英文政府]]組織的[[林全內閣|林全內閣]]，擔任[[行政院環境保護署|行政院環境保護署]]副署長<ref>{{cite news|url=http://udn.com/news/story/9485/1671113|author=何孟奎|title=新聞眼／社運派入閣 與社會對話 |newspaper=[[聯合報|聯合報]]|date=2016-05-04|accessdate=2016-05-04}}</ref>。

2016年9月1至2日，[[高雄市政府|高雄市政府]]連續2天執行[[高雄市|高雄市]][[三民區|三民區]]十全果菜市場牴觸戶拆遷<ref>{{cite news|title=十全市場拆遷 阿嬤哭癱|url=http://www.appledaily.com.tw/appledaily/article/headline/20160903/37369626|publisher=台灣蘋果日報|date=2016-09-03|author=湯寶隆、吳世龍|accessdate=2016-09-28}}</ref>。2016年9月4日，詹順貴於[[facebook|facebook]]聲援[[台灣人權促進會|台灣人權促進會]]且具名參加「請高雄市政府堅守人權與環境正義，以民主程序重新檢視爭議」網路[[連署|連署]]，他說，[[威權統治|威權統治]]遺毒之一是任何土地開發計畫都是「上面大官說了算」，就算他贊同[[高雄市市長|高雄市市長]][[陳菊|陳菊]]的拆遷理由與發展圖像，他也完全無法支持這個拆遷過程。2016年9月15日，民進黨[[台北市議員|台北市議員]][[梁文傑|梁文傑]]批評，詹順貴是雙面人、故意破壞民進黨形象；詹順貴回批「當官是否就不能有社會良心？梁議員是否仍有[[家天下|家天下]]觀念？就留給社會公評」，梁文傑再批「民進黨請你來當官，是請你來一同打拚，不是請你來是非不分捅自己人的」<ref>{{cite news|url=http://www.upmedia.mg/news_info.php?SerialNo=4197|title=詹順貴連署高雄果菜市場重檢遭轟 與梁文傑隔空交火|publisher=[[上報|上報]]|author=塗鵬揚|date=2016-09-16|accessdate=2016-09-30}}</ref><ref>{{cite news|url=http://udn.com/news/story/1/1965168|title=詹順貴批陳菊權威遺毒 遭轟「捅自己人」|publisher=聯合報即時新聞|author=丘采薇|date=2016-09-16|accessdate=2016-09-28}}</ref>。

2017年1月，詹順貴在《[[新新聞|新新聞]]》的專訪中說，享有[[公權力|公權力]]就不可以像過去「高舉著崇高的理想就一直往前衝」，必須穩健改革；[[行政院院長|行政院院長]][[林全|林全]]邀請他入閣，林全是明理之人，讓環保不再每次都成為角力下的犧牲品；他進入體制之後工作忙，他與公民團體的互動一定會變少，但還是有保持聯繫，至少有消息上的交流<ref>{{cite web|author=李又如|title=詹順貴：跟以前做社運一樣勇往直前|url=http://www.new7.com.tw/NewsView.aspx?i=TXT20170125145225JQP|publisher=《新新聞》第1560、1561期合刊號|language=zh-tw|date=2017-01-26|accessdate=2017-02-12|pages=54}}</ref>。

2017年6月6日，環保署召開環境影響評估標準及《環境影響評估法施行細則》修正案公聽會，數十個環保團體在會前質疑部分條文放寬環評限制、有利財團開發，因此與詹順貴發生口角；於是會議才開始五分鐘，詹順貴宣布會議取消後離場，約11點30分重返現場道歉<ref>{{cite news|author=許詩愷|title=詹順貴與環團因修法對峙 五分鐘便離席散會|url=https://www.civilmedia.tw/archives/64697|publisher=[[公民行動影音紀錄資料庫|公民行動影音紀錄資料庫]]|date=2017-06-06|accessdate=2017-06-10}}</ref>。

2018年3月15日，詹順貴在[[深澳發電廠|深澳燃煤電廠]]擴建案環評大會上擔任審查主席，大會經近4小時激烈討論後表決，在贊成與反對票數相同下，昔日力挺禁用生煤的詹順貴投下關鍵贊同票，以1票之差讓擴建案順利通過；在現場的環團錯愕不已，表示會用盡方法持續把關。<ref>[https://tw.news.yahoo.com/%E7%92%B0%E4%BF%9D%E9%AC%A5%E5%A3%AB-%E8%AE%8A%E7%87%83%E7%85%A4%E6%8E%A8%E6%89%8B-%E6%B7%B1%E6%BE%B3%E9%9B%BB%E5%BB%A0-1%E7%A5%A8%E9%81%8E%E9%97%9C-215006151.html 詹順貴身為環保署副署長，昔日禁用生煤，卻在2018/03/15投票贊成 深澳燃煤電廠擴建。]</ref>蔡英文政府因深澳燃煤電廠與[[第三天然氣接收站|第三天然氣接收站]]環評案屢傳爭議，詹順貴最終於同年10月8日請辭環保署副署長。
