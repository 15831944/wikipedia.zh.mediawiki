'''理查·詹姆斯·哈特'''（'''Richard James Hart'''，本名'''詹姆斯·文森佐·卡彭'''（'''James Vincenzo Capone'''），{{bd|1892年|3月28日|1952年|10月1日}}），出生於[[意大利|意大利]]的[[安格里|安格里]]，是1920年代和1930年代[[芝加哥|芝加哥]]最著名的黑幫[[艾爾·卡彭|艾爾·卡彭]]最年長的哥哥。

文森佐·卡彭在16歲時離家出走，從[[布魯克林|布魯克林]]搬到[[內布拉斯加州|內布拉斯加州]]，表面上是加入了馬戲團。他努力甩掉布魯克林口音，並試圖掩蓋他的[[意大利裔美國人|意大利血統]]。他在[[第一次世界大戰|第一次世界大戰]]中入伍[[美國陸軍|陸軍]]，在[[法國|法國]]服役並獲得任命為[[中尉|中尉]]。<ref name=TruTV>{{Cite web |url = http://www.trutv.com/library/crime/gangsters_outlaws/mob_bosses/capone/hart_10.html |title = Al Capone: Chicago's Most Infamous Mob Boss |author = Marilyn Bardsley |publisher = Crime Library |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20121010212233/http://www.trutv.com/library/crime/gangsters_outlaws/mob_bosses/capone/hart_10.html |archivedate = 2012-10-10 }}</ref>

戰後，卡彭合法地改名為理查·詹姆斯·哈特（Richard James Hart），部分是為了紀念他最喜歡的牛仔電影明星{{link-en|威廉·S·哈特|William S. Hart}}。他於1919年結婚，很快便成為一名聯邦[[禁酒令|禁酒]]探員，在內布拉斯加州的{{link-en|荷馬 (內布拉斯加州)|Homer, Nebraska|荷馬}}成家。經過一連串成功針對私酒販的襲擊後，他獲得了「雙槍」哈特的綽號。1920年代中期，報社記者了解到他與芝加哥黑幫的親屬關係。

1926年，哈特成為[[印第安事務局|印第安事務局]]的特別探員。他被指派到[[南達科他州|南達科他州]]的{{link-en|夏延河印第安保留區|Cheyenne River Indian Reservation}}。在那裡，他曾經負責保護[[卡爾文·柯立芝|卡爾文·柯立芝]]總統及其家人訪問[[布拉克山|布拉克山]]。他後來被轉移到[[華盛頓州|華盛頓州]]的{{link-en|史坡堪人|Spokane people|史坡堪印第安保留區}}。他被歸功於在該地區逮捕了至少20名被通緝的兇手，而且追捕印第安違法者和追捕[[私酒|私酒]]（moonshine）。他曾在[[愛達荷州|愛達荷州]]{{link-en|普拉默 (愛達荷州)|Plummer, Idaho|普拉默}}的{{link-en|科達倫保留區|Coeur d'Alene Reservation}}擔任執法官。<ref name=TruTV />

哈特在1931年返回荷馬做[[禁酒局|禁酒探員]]。隨著兩年後廢除禁酒令，他成了一名[[太平紳士|太平紳士]]。他於1952年在內布拉斯加州的荷馬因[[心肌梗塞|心臟病發作]]而去世，享年60歲。<ref>{{Cite web |url = https://www.findagrave.com/cgi-bin/fg.cgi?page=gr&GSln=Hart&GSfn=Richard&GSby=1892&GSbyrel=in&GSdy=1952&GSdyrel=in&GSob=n&GRid=66613217&df=all& |title = Richard James Hart |publisher = findagrave.com |language = en}}</ref>
