'''詹姆斯·S·A·科里'''（{{lang-en|'''James S. A. Corey'''}}）是合作者{{link-en|丹尼爾·亞伯拉罕|Daniel Abraham (author)}}與泰·弗蘭克（Ty Franck）所使用的筆名。第一個和最後的名字是分別取自亞伯拉罕和弗蘭克的中間名，S·A·則是亞伯拉罕的女兒姓名的首字母。這名字也模仿了許多1970年代的科幻小說作家。<ref>{{Cite web |url = http://www.barnesandnoble.com/blog/sci-fi-fantasy/the-evolution-of-james-s-a-coreys-space-epic-the-expanse/ |title = Evolution of a Space Epic: James S.A. Corey's ''The Expanse'' |author = Andrew Liptak |date = 2015-05-27 |publisher = Barnes & Noble.com |language = en}}</ref>在[[德國|德國]]，他們的著作以'''詹姆斯·科里'''的名字出版，省略了中間名縮寫。<ref>{{Cite web |url = https://www.amazon.de/Leviathan-erwacht-Roman-Expanse-Serie-Band/dp/3453529316 |title = Leviathan erwacht |publisher = Amazon.de}}</ref>

== 生涯 ==
他們用該名字寫作「[[蒼穹浩瀚_(小說系列)|蒼穹浩瀚系列]]」的第一本科幻小說《{{link-en|利維坦覺醒|Leviathan Wakes}}》（2011年）。《利維坦覺醒》被提名2012年[[雨果獎最佳長篇小說獎|雨果獎最佳長篇小說獎]]和2012年{{link-en|軌跡獎最佳科幻小說獎|Locus Award for Best Science Fiction Novel}}。作者於每年推出該系列中的其他小說：《''{{link-en|Caliban's War|Caliban's War}}''》（2012年）、《''{{link-en|Abaddon's Gate|Abaddon's Gate}}''》（2013年）和《''{{link-en|Cibola Burn|Cibola Burn}}''》（2014年）。《''Abaddon's Gate''》獲得{{link-en|軌跡獎最佳科幻小說獎|Locus Award for Best Science Fiction Novel}}。{{link-en|Orbit Books|Orbit Books}}與科里簽約創作「蒼穹浩瀚系列」中的額外小說至總共九本。系列的第五本《''{{link-en|Nemesis Games|Nemesis Games}}''》，於2015年6月2日出版。<ref>{{Cite web |url = https://www.amazon.com/Nemesis-Games-Expanse-James-Corey/dp/0316217581 |title = Nemesis Games (The Expanse) |publisher = amazon.com |language = en}}</ref>

近年，他們為系列中每本長篇小說之間出版短篇作品。第一個短篇故事名為《''The Butcher of Anderson Station: A Story of The Expanse''》，於2011年10月以電子書形式出版。接着是69頁的短篇小說《''Gods of Risk''》，於2012年9月以電子書形式出版。2012年11月出版短篇故事《"Drive"》，收錄在選集《''Edge of Infinity''》中<ref>{{Cite web |url = http://www.goodreads.com/book/show/13547291-edge-of-infinity |title = Edge of Infinity (Infinity Project #2) |publisher = Goodreads |language = en}}</ref>。另一本短篇小說《''The Churn''》，於2014年4月29日出版。所有作品都設定在「蒼穹浩瀚系列」中。

他們也寫作過一本[[星球大戰|星球大戰]]小說《''Honor Among Thieves''》，在2014年由Random House出版，以及一個與《蒼穹浩瀚》沒有關聯的短篇故事《"A Man Without Honor"》，包含在選集《''{{link-en|Old Mars|Old Mars}}''》，由[[喬治·R·R·馬丁|喬治·R·R·馬丁]]編輯。<ref>{{Cite web |url = http://www.georgerrmartin.com/grrm_book/old-mars/ |title = OLD MARS |publisher = georgerrmartin.com |language = en}}</ref>
