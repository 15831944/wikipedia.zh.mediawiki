'''詹姆斯·布坎南'''（{{lang|en|James Buchanan}}，{{bd|1791年|4月23日|1868年|6月1日}}），[[美国|美国]]第15任[[美國總統|總統]]。在任期內最受批評的是他無法調和南北之間的衝突，身后亦因此臭名昭著。

== 早年生涯 ==
詹姆斯·布坎南於1791年4月23日出生於美國[[賓夕法尼亞州|賓夕法尼亞州]]默塞尔斯堡附近的科文盖普，父母親來自[[愛爾蘭|愛爾蘭]]，布坎南是家中的長子。

1807年，布坎南就讀於[[狄金森学院|狄金森学院]]，攻讀法律。1810年又到[[蘭卡士特市|蘭卡士特市]]跟隨詹姆斯·霍金斯學習法律。1812年，布坎南正式成為律師。

1814年，布坎南成為蘭卡士特郡的郡議員；1815年當選為賓州州議員，擔任議員期間他曾大力批評麥迪遜總統、國會和聯邦銀行，使他得罪了許多人。兩屆任期後，布坎南在父親影響下競選國會議員，但最終落選。

1819年，布坎南曾與安妮·柯爾曼訂婚，但因為女方父親反對和人們謠傳布坎南已有女友，這項婚約遂告吹。後來女方服藥過量而死亡，從此布坎南也終身未婚，成為[[美國歷史|美國歷史]]上唯一一位單身的總統。後人懷疑因布坎南有同性戀的傾向，而導致安妮悲憤而死。

1820年，布坎南代表聯邦黨當選眾議員。其後布坎南轉投民主黨，被傑克遜總統任命為駐俄公使，在任期間他積極拓展[[美俄關係|美俄關係]]。1833年布坎南辭去駐俄大使之職，回國競選參議員，但不幸落敗。1834年，布坎南被遞補為參議員，並擔任此職達十一年之久。[[詹姆斯·波爾克|詹姆斯·波爾克]]當選總統後，布坎南出任國務卿，期間[[德克薩斯共和國|德克薩斯共和國]]併入美國，墨西哥政府極為不滿，[[美墨戰爭|美墨戰爭]]遂爆發，以美國取得大勝收場。1848年，布坎南卸任國務卿。

== 总统任内 ==
1856年，布坎南擊敗時任總統皮爾斯，獲得民主黨提名，並當選為第十五任美國總統。

由於當時女性意識高漲，女性要求與男性享有同等受教育權利，故布坎南通過了一些保障[[男女平等|男女平等]]的法案。1857年，由於一家人壽保險公司突然宣佈破產，觸發各地銀行的擠兌風潮，全國陷入經濟恐慌，但布坎南政府卻不採取任何行動，使其政府聲望受損。

雖然布坎南有為南北和平作出努力，但南北衝突畢竟在他任內激化了。當1860年林肯當選總統而布坎南尚未卸任時，[[南卡羅來納州|南卡羅來納州]]首先退出聯邦，隔一個月又有六個州退出。布坎南對這些州的脫離仍採用懷柔政策，導致全體閣員反對而集體呈辭。當布坎南離開白宮時，民主黨已一分為二，由道格拉斯領導的北方民主黨支持維護聯邦統一，南方民主黨則脫離聯邦，十一個允許[[奴隸制|奴隸制]]的州在布坎南卸任僅一個月即發動叛亂，[[美國內戰|美國內戰]]爆發。布坎南未能調解南北衝突，導致他的聲望臭名昭著。

== 卸任之後 ==
內戰在布坎南退休後的兩個月內爆發。他支持聯邦，給以前的同事說，“對[[桑特堡|桑特堡]]的攻擊是由邦聯各州開啟的戰爭，沒有其他方法了，我們的部分只能積極去面對。”他還寫了一封信給他的賓夕法尼亞州民主黨人，敦促他們“加入已經在戰場上，成千上萬的勇敢和愛國志願者”。

但是，布坎南在剩餘的歲月，公眾指責他引發內戰，其中甚至提到了一些為“布坎南的戰爭”為自己辯護。他開始每天接受憤怒和恐嚇信，有幅留下的布坎南的肖像眼睛著墨紅色，脖子畫一個絞索和在他的額頭上寫著“叛徒”。參議院提出譴責的決議（沒有通過），有報紙指責他與南部邦聯勾結。他的前內閣成員，其中五名仍在林肯政府任職，卻拒絕公開為布坎南辯護。

面對不斷的攻擊，布坎南終於開始捍衛自己，在1862年10月刊登自己和溫菲爾德·斯科特的換文在報紙《[[國家情報|國家情報]]》上。他很快就開始寫他最充分的公開辯護，用回憶錄形式在1866年發表《叛亂前夕的布坎南政府》一書。

布坎南1868年5月患感冒，由於年事已高病情很快惡化。他死於1868年6月1日，因呼吸衰竭在77歲時在惠特蘭的家中去世，被安葬在蘭開斯特伍德沃德山公墓。

== 其他資料 ==
1999年，“有綫-衛星公共事務電視網”（CSPAN）對歷史學者進行調查，發現歷史學家們認為[[亞伯拉罕·林肯|亞伯拉罕·林肯]]、[[乔治·华盛顿|乔治·华盛顿]]和[[富蘭克林·羅斯福|富蘭克林·羅斯福]]是美國最出色的三位總統，贊許度遠超其他人；而[[威廉·亨利·哈里森|威廉·亨利·哈里森]]、[[安德鲁·约翰逊|安德鲁·约翰逊]]、[[福蘭克林·皮爾斯|福蘭克林·皮爾斯]]和詹姆斯·布坎南是最失敗的總統。2000年，《[[華盛頓郵報|華盛頓郵報]]》也做了一次調查，結果顯示受調查者認為只有[[華盛頓|華盛頓]]、[[林肯|林肯]]和[[富蘭克林·德拉諾·羅斯福|富蘭克林·德拉諾·羅斯福]]這三位才能稱作“偉大”的總統，而[[安德鲁·约翰逊|安德鲁·约翰逊]]、[[福蘭克林·皮爾斯|福蘭克林·皮爾斯]]、[[沃倫·蓋瑪利爾·哈定|沃倫·蓋瑪利爾·哈定]]和布坎南能稱作“失敗者”。
