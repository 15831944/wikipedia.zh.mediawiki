'''伏見稻荷大社'''（{{lang-ja|伏見稲荷大社}}）是一位於[[日本|日本]][[京都市|京都市]][[伏見區|伏見區]]內的[[神社|神社]]，是遍及日本全國各地約四萬多所的[[稻荷神|稻荷神]]社之總本社，以境內所擁有的「千本鳥居」聞名。

== 参拜 ==
伏見稻荷大社位於[[稻荷山|稻荷山]]的山麓，在傳統上整個稻荷山的範圍都被視為是神域（聖地）的範圍。伏見稻荷大社主要是祀奉以[[宇迦之御魂神|宇迦之御魂大神]]為首的諸位[[稻荷神|稻荷神]]，自古以來就是[[農業|農業]]與[[商業|商業]]的神明，除此之外也配祀包括[[佐田彥大神|佐田彥大神]]、[[大宮能賣大神|大宮能賣大神]]、[[田中大神|田中大神]]與[[四大神|四大神]]等其他的神明。由於每年都有大量的香客前來神社祭拜求取農作豐收、生意興隆、交通安全，使得該神社成為京都地區香火最盛的神社之一。另外，起源於[[江戶時代|江戶時代]]的習俗，前來此地許願的人們往往會捐款在神社境內豎立一座[[鳥居|鳥居]]來表達對神明的敬意，使得伏見稻荷大社的範圍內豎有數量驚人的大小鳥居，而以「千本鳥居」之名聞名日本全國乃至於海外。捐款豎立鳥居的單位包含個人、公司行號乃至於各地的商會組織，目前現存的鳥居，最早可以追溯到[[明治|明治]]年間。

在神社的分類上，伏見稻荷大社屬於[[式內社|式內社]]（被[[延喜式神名帳|延喜式神名帳]]所收錄的神社）中的[[名神大社|名神大社]]（祭祀知名神明、自古以來就非常靈驗的神社）等級，為[[二十二社|二十二社]]之一，舊時神社社格屬[[官幣大社|官幣大社]]等級。

== 歷史 ==
相傳伏見稻荷大社是在[[奈良時代|奈良時代]]的711年（[[和銅|和銅]]4年）2月19日時，由[[伊侶具秦公|伊侶具秦公]]奉命在伊奈利山（稻荷山）三峰設置，用以祭祀[[日本秦氏|秦氏]]世代以來的守護神（[[氏神|氏神]]）也是農耕之神的稻荷神，並將神社命名為「伊奈利社」（「伊奈利是日文中「稻生」的發音直接轉譯）。

伏見稻荷大社的[[社家|社家]]（世襲神社內神職人員一職的家族）曾經出過多位學者，其中又以江戶時代中期的[[國學_(日本)|國學者]][[荷田春滿|荷田春滿]]最為知名。今日伏見稻荷大社境內可以見到被特別保存下來的荷田春滿舊宅，在一旁則可以見到原本是為了祭祀荷田春滿所設、目前已獨立而出的[[東丸神社|東丸神社]]。東丸神社前的小徑旁的牆上可以見到大量的[[繪馬|繪馬]]，是長年來來到此地求取考運的學子為了祈福而懸掛。<ref>{{Cite news|url=http://www.nippon.com/hk/views/b03204/|title=外國人為何會覺得伏見稻荷神社「很酷」？|newspaper=nippon.com|accessdate=2017-01-10}}</ref>

== 建築 ==
伏見稻荷大社最初的本殿已於[[應仁之亂|應仁之亂]]時燒毀，今日所見到的本殿為[[明應|明應]]8年（1499年）時重建的版本，目前被日本政府指定為[[國家重要文化財產|國家重要文化財產]]。

除了主殿建築之外，伏見稻荷大社境內最受注目的建物莫過於滿佈整座山頭的各式大小鳥居。除此之外，神社的範圍之內也可以見到許多[[狐狸|狐狸]]（稻荷神的使者）的雕像，與非常特殊的狐狸造型繪馬。

== 交通 ==
在許多前往伏見稻荷大社的交通方式之中，鐵路是最為便捷的一種。隸屬於[[JR西日本|JR西日本]][[奈良線|奈良線]]沿線的[[稻荷車站|稻荷車站]]，與伏見稻荷大社的入口僅有一街之隔，是來此參訪的香客們主要的進出管道，而車站月台的遮雨棚樑柱也特別漆成像鳥居的橘紅色。另外[[京阪電鐵|京阪電鐵]]亦設有伏見稻荷車站，離大社入口約步行5分鐘。
另市內公車則有從京都站開出的南5及105號線兩個選擇，下車後走7分鐘即可到達入口。

== 流行文化中的曝光 ==
* 以京都的[[藝妓|藝妓]]生活作為主題的美國電影《[[藝伎回憶錄|藝伎回憶錄]]》（Memoirs of a Geisha）中，曾出現過以伏見稻荷大社的千本鳥居作為背景的場景。
* 日本動畫《[[名偵探柯南|名偵探柯南]]》劇場版《[[名偵探柯南：迷宮的十字路|迷宮的十字路口]]》，片尾曲曾出現伏見稻荷大社的千本鳥居作為背景。
* 日本動畫《名偵探柯南》的旅遊特別篇《[[明治維新|明治維新]]神秘之旅》中出現過。
* 漫畫及動畫作品《[[狐仙的戀愛入門|狐仙的戀愛入門]]》中的「伊奈里神社」即以此為原型。
* 《[[假面骑士Fourze|假面骑士Fourze]]》 第32集也是在此地取景。
*小說《歡迎光臨陰陽屋》中的主角在書中也是在稻荷神社中找到，可以化身為狐狸。
*日本动畫《[[此花亭奇譚|此花亭绮譚]]》第12集出現以千本鳥居作为背景的場景。
