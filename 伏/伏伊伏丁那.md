'''伏伊伏丁那自治省'''（[[塞尔维亚语|塞尔维亚语]]：Аутономна Покрајина Војводина/Autonomna Pokrajina Vojvodina；[[匈牙利语|匈牙利语]]：Vajdaság Autonóm Tartomány；[[斯洛伐克语|斯洛伐克语]]：Autonómna pokrajina Vojvodina；[[罗马尼亚语|罗马尼亚语]]：Provincia Autonomă Voivodina；[[克罗地亚语|克罗地亚语]]：Autonomna Pokrajina Vojvodina；[[卢森尼亚语|卢森尼亚语]]：Автономна Покраїна Войводина）是[[塞尔维亚|塞尔维亚]]北部的一个[[自治省|自治省]]。伏伊伏丁那民族构成复杂，其中[[塞尔维亚人|塞尔维亚人]]为主体民族，[[馬扎爾人|馬扎爾人]]為第二大民族。伏伊伏丁那有六种官方语言。最大都市是[[诺维萨德|诺维萨德]]。

== 历史 ==
远古时期伏伊伏丁那的居民有[[伊利里亚人|伊利里亚人]]、[[色雷斯人|色雷斯人]]、[[凯尔特人|凯尔特人]]等。后来该地成为[[罗马帝国|罗马帝国]]的一部分。当时伏伊伏丁那属于[[潘诺尼亚行省|潘诺尼亚行省]]，境内的[[斯雷姆斯卡米特罗维察|斯雷姆斯卡米特罗维察]]当时是该地区的重镇，有六名[[罗马皇帝|罗马皇帝]]在该地或附近出生。

[[匈人|匈人]]於395年将罗马人逐出潘诺尼亚。后来[[匈奴人|匈奴人]]、[[拜占庭帝国|拜占庭帝国]]、[[保加尔人|保加尔人]]、[[匈牙利人|匈牙利人]]相继统治伏伊伏丁那。1526年，[[奥斯曼帝国|奥斯曼帝国]]军队北上欧洲，在[[第一次摩哈赤战役|第一次摩哈赤战役]]中击败匈牙利，匈牙利[[亚盖隆王朝|亚盖隆王朝]]随之崩溃，匈牙利大部分地区被奥斯曼帝国兼并。

[[奥地利|奥地利]]的[[哈布斯堡王朝|哈布斯堡王朝]]於18世纪初接管伏伊伏丁那。奥地利皇帝给予当地塞尔维亚人选择公爵的权利，该地因此被塞尔维亚人称为“公爵领地”（Vojvodina），即“伏伊伏丁那”名称的来源。

在哈布斯堡的统治下，来自王朝其他地区的移民纷纷移居伏伊伏丁那，造成了该地的民族构成的多样性。1867年奥地利正式改为[[奥匈帝国|奥匈帝国]]（[[奥地利|奥地利]]与[[匈牙利|匈牙利]]），此时伏伊伏丁那属于[[匈牙利王国|匈牙利王国]]管辖。

[[File:Danube_banovina_(zh).png|thumb]]
[[第一次世界大战|第一次世界大战]]中，[[奥匈帝国|奥匈帝国]]战败，旋即崩溃。伏伊伏丁那加入了[[塞尔维亚王国|塞尔维亚王国]]，该国后来又加入了[[南斯拉夫王国|塞尔维亚人、克罗地亚人和斯洛文尼亚人王国]]的一部分。王国于1929年更名為[[南斯拉夫|南斯拉夫]]。[[第二次世界大战|第二次世界大战]]期间伏伊伏丁那大体并入了[[轴心国|轴心国]]成员的[[匈牙利王国_(1920年—1946年)|匈牙利]]和[[克罗地亚独立国|克罗地亚独立国]]，战后又重新成为由[[铁托|铁托]]领导的[[南斯拉夫社会主义联邦共和国|南斯拉夫社会主义联邦共和国]]的一部分。

南斯拉夫于1974年实行的新宪法给予了伏伊伏丁那自治省的地位，被称为[[伏伊伏丁那社会主义自治省|伏伊伏丁那社会主义自治省]]。但在塞尔维亚在[[米洛舍维奇|米洛舍维奇]]的领导下于1990年取消了该自治省的许多权利。但与[[科索沃|科索沃]]不同，伏伊伏丁那局势始终比较平静，没有爆发大规模战争。

[[米洛舍维奇|米洛舍维奇]]倒台后，伏伊伏丁那的各少数民族大体拥护新政府。2002年该自治省的自治权得到了恢复，2004年采用了新省旗。

2008年，科索沃宣布独立，引发国际关注。另外引起注意的是，伏伊伏丁那也出现独立声浪，但[[塞尔维亚共和国|塞尔维亚]]对科索沃独立持强烈反对态度，伏伊伏丁那的独立诉求恐难以获得正面答覆。2009年，伏伊伏丁那通过了新的自治宪法。

== 地理 ==
[[File:Map_of_Serbia_(Vojvodina).PNG|thumb]]
[[File:_Vojvodina_ethnic2002.png|thumb]]
伏伊伏丁那位于[[塞尔维亚共和国|塞尔维亚]]北部地区。伏伊伏丁那被[[多瑙河|多瑙河]]与[[蒂萨河|蒂萨河]]一分为三，其中西北部分为[[巴奇卡|巴奇卡]]地区，东部为[[巴纳特|巴纳特]]地区，西南部为[[斯雷姆|斯雷姆]]地区。

== 政治 ==
伏伊伏丁那有多个地方性政党，其中包括：伏伊伏丁那社会民主联盟、伏伊伏丁那改革党、伏伊伏丁那联合党团、伏伊伏丁那运动、伏伊伏丁那社会主义者联盟。

伏伊伏丁那自治省首長现为Bojan Pajtic（民主党），[[伏伊伏丁那自治省议会|伏伊伏丁那自治省议会]]议长为Bojan Kostres（伏伊伏丁那社会民主联盟）。
