'''伏努科沃國際机场'''（{{lang-ru|Международный аэропорт Внуково}}；{{Airport codes|VKO|UUWW}}），位於[[俄羅斯|俄羅斯]]首都[[莫斯科|莫斯科]]市中心西南28公里處，是一個雙跑道國際機場。它是繼[[多莫杰多沃国际机场|多莫杰多沃国际机场]]及[[谢列梅捷沃国际机场|谢列梅捷沃国际机场]]之後的俄羅斯第三繁忙的機場，也是服務莫斯科地區的三個主要機場之一。在2010年機場的旅客量達到946萬人次，比2009年增長了22.4%<ref name=Stats>{{Cite web|url=http://www.vnukovo.ru/media/about_airport/aci/01F9050E-EF9E-4D74-87DD-06AD8696C197/2010/dec_2010.pdf|title=ACI Europe Airport Traffic Reports|publisher=Airports Council International（ACI）|date=2010-12|language=en|deadurl=yes|archiveurl=https://web.archive.org/web/20111230110233/http://www.vnukovo.ru/media/about_airport/aci/01F9050E-EF9E-4D74-87DD-06AD8696C197/2010/dec_2010.pdf|archivedate=2011-12-30}}</ref>。

在[[第二次世界大戰|第二次世界大戰]]期間，伏努科沃機場曾被作為軍事用途，不過戰後就一直是民用機場。

== 歷史 ==
[[Image:Reagans_land_in_Moscow_1988.jpg|left]][[罗纳德·里根|里根]]在伏努科沃機場（1988年）]]

伏努科沃國際机场是莫斯科仍然運作的機場中歷史最悠久的。於1937年批准興建，1941年落成。在[[苏德战争|苏德战争]]中它是俄軍的空軍基地；戰後，開始了民航客運服務。

1956年9月15日，[[圖-104|圖-104]]噴氣式飛機的首航從伏努科沃機場起飛，經[[鄂木斯克|鄂木斯克]]飛往[[伊爾庫茨克|伊爾庫茨克]]。

1957年11月4日，一架載有[[格奥尔基·乔治乌-德治|格奥尔基·乔治乌-德治]]、[[基伏·斯托伊卡|基伏·斯托伊卡]]、[[尼古拉·齐奥塞斯库|尼古拉·齐奥塞斯库]]等[[罗马尼亚共产党|罗马尼亚工人党]]領導人的飛機在伏努科沃機場失事。事故中時任[[罗马尼亚社会主义共和国|罗马尼亚社会主义共和国]]外交部長身亡。

[[伊尔-18|伊尔-18]]的首航（莫斯科－[[阿拉木圖|阿拉木圖]]，1956年4月20日）和[[圖-114|圖-114]]的首航（莫斯科－[[伯力|伯力]]，1961年4月2日）也都是從伏努科沃機場起飛的。

1980年，因舉辦[[1980年奧運會|1980年奧運會]]而對機場進行了擴建。

1993年Vnukovo Airport成為[[合股公司|合股公司]].

[[File:Vnukovo_airport_under_renovation_aerial_view.JPG|thumb]]
從2003年起，隨著所有權從聯邦政府轉移到莫斯科市政府，伏努科沃國際機場為迎合戰略發展而開始了一系列的大規模重建：
* 2004年4月：新的航站樓B落成。目前該航站樓仍有國際航班，之後將轉為只服務國內航線。該航站樓總面積為8萬平方米，每年最大客運量為4百萬人次。
* 2005年8月：伏努科沃機場連接[[基輔站|基輔站]]的機場快線通車。
* 2010年12月：新的航站樓A落成。

== 機場簡介 ==
[[Image:Aeroport-vnukovo-terminal.jpg|thumb]]

在莫斯科的三個機場中，伏努科沃是海拔最高的（204米），因此在大霧天它經常是作為備降機場。<ref>{{Cite web|url=http://www.vnukovo.ru/eng/about/competitive_edge/|title=Airport's Competitive Advantages|publisher=伏努科沃機場官網|accessdate=2012-04-15|language=en}}</ref>

機場有兩條相交的跑道，長度分別是3000米和3060米。每條跑道寬為60米，兩邊各有10米的安全路肩。兩條跑道合共可容每小時60架次的班機起降。

機場有兩個客運航站樓（A和B），一個私人飛機與包機航站樓，一個貨運航站樓，以及60個停機坪。

機場有4000名員工，每小時可處理多達3000名乘客。2008年機場的客運量達到將近8百萬人次，比2007年增長16.7%。<ref name=Stats/>

機場也有一個VIP大廳，許多來訪的政要和名人都曾使用過。俄羅斯總統也使用伏努科沃的VIP設施。

機場邊上也有一個[[图波列夫公司|图波列夫]]飛機維修車間，許多大型檢修和改造工程會在這裡進行。

== 扩建计划 ==
機場即將進行一項擴建工程，預計於2015年完工。屆時伏努科沃國際機場將進一步提升競爭力，提供更廣泛的客運和貨運服務，並成為一個重要的國際空港。

新落成的航站樓A將擁有25萬平方米的空間，能夠處理每小時7800人次的客運量，每年的最大客運量可達1千8百萬至2千萬人次。<ref>{{Cite web|url=http://www.vnukovo.ru/eng/about/outlook/terminal/|title=Terminal A Information|publisher=伏努科沃機場官網|accessdate=2012-04-16|language=en}}</ref>這令各個使用本機場的航空公司能夠擴容並大大地改善旅客服務質量和舒適度。航站樓採用不規則伸展結構從現有的國內旅客航站樓延伸出來，朝著伏努科沃居民區方向延伸，並將連接周邊的空地以便作將來的擴展。

隨著新的航站樓的建設，目前最老的航站樓（建於1941年）於2005年11月起開始了逐步拆除。建於1970年的國內航班第二航站樓目前仍繼續運作，直到翻新工程的後階段會被新的航站樓所取代。

擴建工程也會把目前的V型雙跑道（3000米和3060米）延長至3800米，並會把目前的[[仪表着陆系统|仪表着陆系统]]由CAT II升級至CAT III。另外還要建一個全新的控制[[塔台|塔台]]，並把滑行道作相應地延伸。此外貨運航站樓和多層停車場也會擴建。

== 交通 ==
[[Image:Aeroport-vnukovo-station.jpg|thumb]]
'''[[機場快線_(俄羅斯)|伏努科沃機場快線]]'''是一條高速鐵路線，連接了伏努科沃機場和莫斯科市中心的鐵路[[基輔站|基輔站]]，單程需時約35分鐘。

== 航空公司，航站樓和目的地 ==
機場有以下航站樓：
* 航站樓A同時服務國內和國際航線。
* 航站樓B供國際航班使用，將來的主要服務對象為[[包機|包機]]和[[廉價航空公司|廉價國際航班]]。
* 航站樓D主要服務來自[[北高加索|北高加索]]地區的航班。
