{{俄罗斯历史}}
'''伏爾加保加利亞'''，又名'''伏爾加卡馬河保加利亞'''，從7世紀維持至13世紀。範圍是現在的[[楚瓦什共和国|楚瓦什共和国]]與[[鞑靼斯坦共和国|鞑靼斯坦共和国]]，現在俄羅斯的[[楚瓦什人|楚瓦什人]]與[[韃靼人|韃靼人]]是他們的後裔。尤其是喀山韃靼人，被稱為'''新保加爾人'''。

== 起源 ==
第一手關於[[保加尔人|保加爾人]]的資料相當少，多數來自俄羅斯文件與伊斯兰地理學家。據信最早居住該地區的是[[烏拉爾語系|烏拉爾語系]]的[[芬蘭|芬蘭]]-烏戈爾人與[[匈人|匈人]]的其中一部庫特利格爾。

保加爾人據信起源于[[帕米爾高原|帕米爾高原]]，俄羅斯學者認為他們就是中國史書所記載的[[烏揭|烏揭]]，但也有一些中國學者認為他們是白狄之後（即白色皮膚的外人）。保加爾人的一個主要部落'''乌基杜爾'''人其實就是呼揭人，公元372年保加爾人与[[匈人|匈人]]（HUNS）一同遷徙到伏爾加河西部的草原。

保加爾人是在公元372年的時候遷入东欧的，這個時間相當于中國[[东晋|东晋]]王朝的晚期。經過現代人的考證，保加爾人起源于中亞和阿尔泰地區。因為史籍記載的非常不完整所以他們的起源並不是很清楚。最早提到保加爾人這個名詞的是約丹尼斯的《[[哥特史|哥特史]]》，但更早的時候[[托勒密|托勒密]]、馬爾西林努斯曾提到過“Rha”，亦即保加爾人的聚居区。20世紀初期的學者們根據一些古羅馬或东罗马的語焉不詳的記載認為保加爾人可能與[[突厥人|突厥人]]的遠祖有關。還有一些學者認為他們是'''帕爾米拉人'''的後代，因為羅馬時期的帕爾米拉人曾短暫的統治過這個地區。

但是，現在這個理論被完全否定。現代人根據考古證據發現保加爾人與[[匈人|匈人]]有著千絲萬縷的聯繫。他們屬于[[西突厥|西突厥]]語族中匈奴種的(on-ogur)十箭部落，他們是特裏格斯人和烏提格爾人的近親。在公元372年前後，他們由于所處中亞地區的寒冷，所以一部分保加爾人遷移到了[[裏海|裏海]]與[[黑海|黑海]]之間的廣闊土地（[[欽察|钦察]]）。在這裏他們安定下來，並開始進入了部落联盟的時期，這些人分成了幾個軍事貴族集團進行聯盟統治，與此同時開始出現了崇拜[[太陽神|太陽神]]和[[戰神|戰神]]的宗教組織。另一個不里阿爾（耳）人實際上是保加爾人的一個分支。

保加爾人在635年於[[黑海|黑海]]以北的[[亞速海|亞速海]]沿岸建立了[[大保加利亚_(中世纪)|老大保加利亞]]，由'''庫弗拉特汗'''統治。庫弗拉特死後，汗國一分為二：一部分在[[巴颜|巴顏]]率領下前往伏爾加河的森林區。另一些保加尔部落，在'''[[阿斯巴鲁赫|阿斯巴魯赫]]'''率領下继续向西遷移。最终在[[多瑙河|多瑙河]]下游，现在被称为保加利亚的[[默西亞_(羅馬行省)|默西亚]]定居下来。他们在那里与早先到达那里[[斯拉夫人|斯拉夫人]]融合或被同化，采用[[南斯拉夫|南斯拉夫]]語和[[正教會|东正教]]信仰。

大多數學者同意，保加爾人在[[670年|670年]]受到[[可萨人|可薩汗国]]攻擊，一支保加爾人向北遷徙，汗國是在9世紀正式立國，首都在[[保加尔市|保加爾市]]，即現在的[[喀山|喀山]]。多數學者有懷疑，但该国可以断言的独立是从可薩汗國在965年被[[基辅罗斯|基辅罗斯]]攻擊開始。

== 崛起 ==
大部分人口操[[突厥语族|突厥語]]，現在的[[楚瓦什人|楚瓦什人]]與喀山韃靼人是保加爾人後裔。或多或少外加[[芬兰-乌戈尔语族|芬兰-乌戈尔語族]]和突厥语的人口，包括[[欽察|欽察]]人，另一些是[[馬札兒人|馬札兒人]](匈牙利人)。

保加爾汗國接受[[伊斯蘭教|伊斯蘭教]]是在10世紀，[[白益王朝|伊斯兰哈里發]][[穆克台迪爾|穆克台迪爾]]派出使者[[伊本·法德蘭|伊本·法德蘭]]前往保加爾汗國。并把法官和[[伊斯蘭教法|沙里亞法]]教师給予伏尔加保加利亚，以及帮助建立一个炮台和一座清真寺。[[薩滿教|薩滿教]]与其他宗教持续实行。

保加爾人控制伏尔加河中游，在其过程中，国家控制许多在欧洲和亚洲之间的贸易路线。首都保加爾市([[喀山|喀山]])是一个欣欣向荣的城市，擁有與伊斯兰世界最大的中心巴格達不相上下的规模和财富。贸易伙伴包括海盗，[[維京人|維京人]]和[[薩莫耶德語族|薩莫耶德語族]]在北。[[巴格达|巴格达]]和[[君士坦丁堡|君士坦丁堡]]在南方，从西欧向中在东部地区。其他主要城市包括bilär，suvar，qaşan 和cükätaw （juketau）。现代城市喀山和[[苏瓦尔|苏瓦尔]]分别成為伏尔加保加利亚的边境要塞。

一些伏尔加保加利亚城市迄今仍未找到，但他们都提到。其中一些城市毀在蒙古人入侵。

[[基辅罗斯|基辅罗斯]]向保加爾构成唯一有形的军事威胁。在11世纪，该国遭受基辅罗斯侵袭行动。在12和13世纪之交，基辅罗斯大公[[弗拉基米尔|弗拉基米尔]]急于扩张自己的东部边界，系统地侵袭保加爾城市。保加爾人受斯拉夫侵袭，汗國不得不將其首都從[[喀山|喀山]]搬遷東方。

== 衰落 ==
1223年9月，在[[薩馬拉|薩馬拉]]附近，[[成吉思汗|成吉思汗]]的大将[[哲別|哲别]]和[[速不台|速不台]]攻击和杀掠保加爾汗國，在河濱被汗國打敗，史稱[[薩馬拉河之戰|薩馬拉河之戰]]。後來蒙古第二次西征，在1236年征服汗國，成為[[蒙古|蒙古]][[金帳汗國|金帳汗國]]一部分（[[巴什科尔托斯坦共和国|巴什基尔]]兀鲁思）。1430年起成為[[喀山汗國|喀山汗國]]一部分。历史记录蒙古入侵杀掠大量保加尔人，大多數人向北逃亡，成为今天楚瓦什人与鞑靼人。
