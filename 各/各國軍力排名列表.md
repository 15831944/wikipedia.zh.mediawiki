== 統一後的朝鮮半島 ==
2017年環球軍力排行榜列出了[[朝鮮半島統一問題|統一後的朝鮮半島]]的軍力報告，根據估計，統一後的朝鮮半島軍力將達到全球第6名的水準，甚至比鄰近的[[日本|日本]]略勝一籌<ref>{{Cite web |url = https://www.globalfirepower.com/unified-korea-military-strength.asp#powerindex |title = Korea (Unified) Military Strength |accessdate = 2018/02/02 }}</ref>。
