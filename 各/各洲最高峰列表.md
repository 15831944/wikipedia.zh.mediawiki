== [[亚洲|亚洲]] ==
* [[珠穆朗玛峰|珠穆朗玛峰]]：[[海拔|海拔]]8,848米，位於[[中國|中國]]、[[尼泊爾|尼泊爾]]交界的[[喜馬拉雅山脈|喜馬拉雅山脈]]，也是[[世界|世界]]最高峰。

== [[欧洲|欧洲]] ==
* [[厄尔布鲁士峰|厄尔布鲁士峰]]：海拔5,642米，位於[[俄羅斯|俄羅斯]]的[[高加索山脈|高加索山脈]]，經勘測及衛星定位，已鑑定厄尔布鲁士峰峰頂位於俄羅斯的歐洲界線境內，故厄尔布鲁士峰被認為是歐洲第一高峰<ref>[https://www.cia.gov/library/publications/the-world-factbook/geos/rs.html 美國中央情報局－俄羅斯]</ref>。
另有因爭議而提出白朗峰：
* [[白朗峰|白朗峰]]：海拔4,808米，位於[[法國|法國]]、[[義大利|義大利]]交界的[[阿爾卑斯山脈|阿爾卑斯山脈]]。

== [[非洲|非洲]] ==
* [[吉力馬扎羅山|吉力馬扎羅山]]：海拔5,893米，位於[[坦桑尼亞|坦桑尼亞]]，也是非洲最高的[[火山|火山]]。

== [[北美洲|北美洲]] ==
* [[迪纳利|迪纳利山]]：海拔6,194米，位於[[美國|美國]][[阿拉斯加州|阿拉斯加州]]的[[阿拉斯加山脈|阿拉斯加山脈]]。

=== [[中美洲|中美洲]] ===
有時因北美洲特分出中美洲而提出中美洲最高峰：
* [[奧里薩巴山|奧里薩巴山]]：海拔5,636米，位於[[墨西哥|墨西哥]]，也是中美洲最高的火山。

== [[南美洲|南美洲]] ==
* [[阿空加瓜山|阿空加瓜山]]：海拔6,961米，位於[[阿根廷|阿根廷]]的[[安地斯山脈|安地斯山脈]]，也是世界最高的火山、死火山、[[南半球|南半球]]最高峰。

== [[大洋洲|大洋洲]] ==
* [[查亞峰|查亞峰]]：海拔4,884米，位於[[印度尼西亞|印度尼西亞]]的[[新幾內亞島|新幾內亞島]]西側的[[蘇迪曼山脈|蘇迪曼山脈]]，也是世界最高的[[島|島]]上山峰。
由於新幾內亞島雖然地理上列為大洋洲，但以國家區分的話，印尼被劃分在亞洲，所以另有版本島上東側的威廉山：
* [[威廉山|威廉山]]：海拔4,509米，位於[[巴布亞紐幾內亞|巴布亞紐幾內亞]]的新幾內亞島東側的[[俾斯麥山脈|俾斯麥山脈]]。

=== [[澳洲|澳洲]] ===
有時因大洋洲特分出澳洲而提出澳洲最高峰：
* [[科修斯科山|科修斯科山]]：海拔2,228米，位於[[澳大利亞|澳大利亞]]的[[大分水嶺|大分水嶺]][[雪山山脈_(澳大利亞)|雪山山脈]]。

== [[南极洲|南极洲]] ==
* [[文森山|文森山]]：海拔4,892米，位於[[南極半島|南極半島]]的[[艾爾斯渥茲山脈|艾爾斯渥茲山脈]]。

== 七大洲八大高峰<ref>{{Cite web|url=http://hk.on.cc/hk/bkn/cnt/news/20180610/bkn-20180610191643407-0610_00822_001.html|title=最年輕港人攀八大高峰|accessdate=|author=|date=|publisher=}}</ref> ==
基於不同計算方法及七大洲邊界沒有一致說法，先後有不同人士列出「[[七大頂峰|七大頂峰]]」或「[[八大頂峰|八大頂峰]]」的不同版本。

* [[美国|美国]]人{{link-en|理查·貝斯|Richard Bass|理查·貝斯}}（Richard Bass）的版本包括：[[珠穆朗玛峰|珠穆朗玛峰]]，[[阿空加瓜山|阿空加瓜山]]，[[迪纳利|迪纳利山]]，[[吉力馬扎羅山|吉力馬扎羅山]]，[[文森山|文森山]]，[[厄尔布鲁士峰|厄尔布鲁士峰]]，[[科修斯科山|科修斯科山]]
* 意大利人[[Reinhold_Messner|Reinhold Messner]]的版本包括：[[珠穆朗玛峰|珠穆朗玛峰]]，[[阿空加瓜山|阿空加瓜山]]，[[迪纳利|迪纳利山]]，[[吉力馬扎羅山|吉力馬扎羅山]]，[[文森山|文森山]]，[[厄尔布鲁士峰|厄尔布鲁士峰]]，[[查亞峰|查亞峰]]

亦有人把以上八個頂峰合稱「[[八大頂峰|七大洲八大頂峰]]」。
