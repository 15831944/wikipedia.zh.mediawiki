为解决城市东部环境恶化问题，满足城市扩容发展的需要，2004年8月于[[中国|中国]][[陕西省|陕西省]][[西安市|西安市]]城区东部建立'''浐灞生态区'''，规划面积129[[平方公里|平方公里]]（集中治理区89平方公里），为[[国家级生态示范区|国家级生态示范区]]之一，<ref>[http://www.xachanba.com.cn/front/cbgk/cbgkDetail.jsp?artId=5624&nodeId=606 生态区简介] {{webarchive|url=https://web.archive.org/web/20120422140019/http://www.xachanba.com.cn/front/cbgk/cbgkDetail.jsp?artId=5624&nodeId=606 |date=2012-04-22 }}</ref> 是西安市统筹城乡、推动生态经济发展的重点区域。

==名称来源==
“浐灞”这一名称来源于“[[长安八水|长安八水]]”中的[[浐水|浐水]]水系和[[灞水|灞水]]水系。

== 地理区位 ==
浐灞生态区位于西安城区东部，主体位于西安市[[灞桥区|灞桥区]]和[[未央区|未央区]]，距离市中心10公里。北到渭河，南到绕城高速，包括浐灞河两河四岸的南北向带状区域。内外与西安纺织新区，[[西安国际港务区|西安国际港务区]]毗邻。

== 发展历程 ==
2004年9月9日，“西安市浐灞河综合治理开发建设管理委员会”挂牌成立，为西安浐灞生态区前身。
 
2005年1月，浐灞河管委会第一个重点项目——广运潭生态景观区项目启动。
 
2007年11月，西安浐灞生态区成为[[欧亚经济论坛|欧亚经济论坛]]永久性会址。<ref>[http://finance.people.com.cn/GB/71364/6493881.html 西安被确定“欧亚经济论坛”永久会址 ]</ref>

2011年7月，经国家[[环保部|环保部]]批准成为国家级生态区。<ref>[http://www.sxtvs.com/content/2011-07/18/content_4895964.htm?COLLCC=821476062& 西安浐灞生态区成为西部首个国家级生态区]</ref>
 
2011年4月至10月，[[2011年西安世界园艺博览会|2011年西安世界园艺博览会]]在浐灞举办。<ref>{{Cite web |url=http://www.forestry.gov.cn/portal/main/s/195/content-476449.html%20 |title=西安世界园艺博览会开幕 |access-date=2014-05-04 |archive-url=https://web.archive.org/web/20150924014613/http://www.forestry.gov.cn/portal/main/s/195/content-476449.html |archive-date=2015-09-24 |dead-url=yes }}</ref>

2017年，浐灞生态区成为[[中国（陕西）自由贸易试验区|中国（陕西）自由贸易试验区]]的一部分<ref>{{cite web| url=http://www.xaftz.gov.cn/Pages/IntroductionDetaile.aspx?id=228| title=中国（陕西）自贸试验区西安核心区| publisher=西安自贸区| date=2017-03-29| accessdate=2017-05-04}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。
