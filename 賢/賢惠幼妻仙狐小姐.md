{{日本作品|賢惠幼妻仙狐小姐|世話やきキツネの仙狐さん}}是日本[[漫畫家|漫畫家]]{{lang|ja|リムコロ}}創作的[[日本漫畫|漫畫]]作品，於漫畫雜誌《[[Newtype_(雜誌)|Comic Newtype]]》開始連載，並由[[KADOKAWA|KADOKAWA]]出版。2018年12月正式宣佈推出電視動畫作品，2019年4月10日開播<ref>{{cite web|script-title=ja:「世話やきキツネの仙狐さん」TVアニメ化、800歳のキツネ娘に甘やかされる|url=https://natalie.mu/comic/news/310531|website=[[Natalie_(網站)|Natalie]]|accessdate=2019年1月13日|language=ja|date=2018年12月3日}}</ref>。

== 故事簡介 ==
每天加班搭著末班車回家、身心俱疲的上班族中野，某日回到家發現作為神明使者的仙狐來到家中，實際上已經800歲卻看似幼女的仙狐，為了能夠消解中野疲勞而發揮了壓倒性的包容力讓他撒嬌，展開了一段充滿療癒感的喜劇故事。

== 登場角色 ==
; {{nihongo|仙狐|仙狐（せんこ）|聲：[[和氣杏未|和氣杏未]]<ref name="character">{{cite web|url=http://senkosan.com/character/|title=CHARACTER|work=TV動畫「賢惠幼妻仙狐小姐」官方網站|accessdate=2019年4月9日|language=ja}}</ref>}}
: 外表是幼女，實際年齡為八百歲的金毛狐狸，下來凡間“拯救”快被工作壓得喘不過氣的中野。除了做家事外，讓中野能夠盡情的撒嬌也是日常任務之一，還會給中野“特殊的服務”。在狐狸“總本山”有著崇高的地位。自稱與中野的關係是“妻子兼母親”。
; {{nihongo|中野|中野（なかの）|聲：[[諏訪部順一|諏訪部順一]]<ref name="character"/>}}
: 中年上班族，為了事業為了工作天天賣著肝，往往必須乘坐末班車才能回到家，甚至直接睡在公司裡。向仙狐小姐要求“特殊服務”是每天的例行公事之一。家裡囤積了一堆遊戲卻沒時間玩，在與白的遊戲對戰中輸給了白的千里眼。是白的專屬椅子。
; {{nihongo|高圓寺安子|高円寺安子（こうえんじ やすこ）|聲：[[佐倉綾音|佐倉綾音]]<ref name="character"/>}}
: 住在中野隔壁的漫畫家，一開始無法忍受中野與仙狐的嬉鬧聲而踹門抗議，在知道仙狐會定時提供飯點時，秉持著“會給我飯吃的人都不是壞人”的精神決定不與他們倆計較。
; {{nihongo|白|シロ|聲：[[内田真禮|内田真禮]]<ref name="character"/>}}
: 純白的狐娘，多次拜訪中野家，與高圓寺安子是好朋友，第一次拜訪時被中野強烈的慾望嚇走了。對於吃的有異常的執著，對於砂糖有絕對的信任，認為不管何種食物，加了砂糖都會很好吃（曾經在泡麵裡添加砂糖）。曾未經告知“借用”夜空的海灘而遭受“小小的懲罰”。
; {{nihongo|夜空|夜空（そら）|聲：[[喜多村英梨|喜多村英梨]]}}
