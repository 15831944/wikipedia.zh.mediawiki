《'''逆战'''》是由[[琳琅天上|琳琅天上]]开发，并由腾讯运营的一款使用[[虚幻3引擎|虚幻3引擎]]开发的近未来风格[[第一人称射击游戏|第一人称射击游戏]]，采用[[网络游戏免费模式|游戏免费、道具收费]]的盈利模式运营。该游戏于2011年5月开始技术测试，当时的名称为《火力突击》，<ref name="游戏实测">{{Cite web |url = http://www.gamersky.com/hardware/201509/658898_10.shtml |title = 10大游戏实测 电竞玩家GTX 950与GTX 960显卡该选谁？ |author = 马振华 |date = 2015年9月4日 |publisher = 游民星空 |language = zh-cn }}</ref>2012年1月12日，腾讯与[[英皇电影|英皇电影]]合作，并与英皇电影所制作的同名电影联合推出新版本，该游戏亦改为现名。<ref>{{Cite web |url = http://games.qq.com/a/20120113/000152.htm |title = 携手周杰伦谢霆锋 腾讯大作更名逆战 |date = 2012年1月13日 |publisher = 腾讯游戏 |language = zh-cn }}</ref>2013年12月12日，该游戏正式公测。<ref>{{Cite web |url = http://games.qq.com/a/20131130/005741.htm#p=1 |title = 张杰国际范MV首发《逆战》宣布12月12日公测 |author = QQ军团 |date = 2013年11月30日 |publisher = 腾讯网 |language = zh-cn }}</ref>目前该游戏已推出7部资料片。<ref name="游戏实测" />
