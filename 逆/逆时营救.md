《'''逆时营救'''》（{{lang-en|''Reset''}}），是一部于[[2017年电影列表|2017年]]上映的科幻动作电影。由[[杨幂|杨幂]]、[[霍建华|霍建华]]、[[金士杰|金士杰]]、[[刘畅_(男演员)|刘畅]]、[[张艺瀚|张艺瀚]]领衔主演，<ref>{{Cite web |url = http://www.sohu.com/a/143351663_114733 |title = 《逆时营救》曝杨幂PK霍建华预告 杨幂挑战自己 |author =  |date = 2017-05-25 |publisher = 搜狐 |language = zh-cn |accessdate = 2017-06-03 }}</ref><ref>{{Cite web |url = http://www.sohu.com/a/141523587_114733 |title = 《逆时营救》曝霍建华特辑 变绑匪“狠虐”杨幂 |author =  |date = 2017-05-18 |publisher = 搜狐 |language = zh-cn |accessdate = 2017-06-03 }}</ref> 2017年6月29日于[[中国大陆|中国大陆]]上映。<ref>{{Cite web |url = http://media.people.com.cn/n1/2017/0531/c14677-29308423.html |title = 《逆时营救》三个"杨幂"同台 与霍建华时隔7年再合作 |author = 赵光霞、宋心蕊 |date = 2017-05-31 |publisher = 人民网 |language = zh-cn |accessdate = 2017-06-03 }}</ref> 


== 剧情大纲 ==
物理研究所高级研究员夏天（[[楊冪|楊冪]] 飾），单亲妈妈，儿子豆豆（[[張藝瀚|張藝瀚]] 飾），被綁匪崔琥（[[霍建華|霍建華]] 飾）綁架并将其杀害，为了救活孩子，夏天拼了命去利用時光科技逆转时间1小时50分钟前拯救自己的儿子。（因为夏天每次回到过去的时间点是不同的，所以本片不是时间循环类的影片。<ref>{{Cite web |url = http://www.1905.com/news/20170701/1197359.shtml |title = 说《逆时营救》烂，是因为你们文科生不懂物理 |author = Kiki |date = 2017-07-01 |publisher = 1905电影网 |language = zh-cn |accessdate = 2017-11-19 }}</ref> ）
