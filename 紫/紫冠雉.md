'''紫冠雉'''（''Penelope purpurascens''）是一種[[鳳冠雉|鳳冠雉]]。牠們分佈在[[墨西哥|墨西哥]]南部及[[尤卡坦半島|尤卡坦半島]]至[[厄瓜多爾|厄瓜多爾]]西部及[[委內瑞拉|委內瑞拉]]北部達[[海拔|海拔]]1850米的地方。

== 特徵 ==
紫冠雉是大型的[[鳥類|鳥類]]，一般長86厘米及重1.7公斤。牠們的外觀像[[火雞|火雞]]，頭細小而有冠，腳長而有力，尾巴闊而長。牠們主要呈深褐色，[[頸部|頸部]]及[[胸部|胸部]]有白色斑點。[[臀部|臀部]]及[[腹部|腹部]]都呈赤褐色。眼眶沒有[[羽毛|羽毛]]，[[皮膚|皮膚]]呈藍灰色，[[喉嚨|喉嚨]]有紅色的垂肉。雌雞及雄雞相似，小雞則有黑色斑紋及赭色斑點。

== 行為 ==
紫冠雉棲於[[森林|森林]]中。牠們的鳥巢是築於樹上或樹樁上，以樹枝組成及[[葉子|葉子]]圍邊。雌雞每次會生2-3隻蛋，蛋表面粗糙及呈白色。雌雞會獨自槃蛋。

紫冠雉是群居的，很多時都是成雙成對或一群6-12隻居住。牠們會在樹枝上走動，尋找[[果實|果實]]及[[葉子|葉子]]作食物。

== 保育 ==
紫冠雉較偏遠的分佈地因伐林及獵殺而大幅縮小。
