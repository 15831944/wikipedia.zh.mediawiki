'''紫金山天文台'''（全称'''中国科学院紫金山天文台'''，简称'''紫台'''，{{lang-en|Purple Mountain Observatory, CAS}}）是[[中国科学院|中国科学院]]在[[空间科学|空间科学]]领域的一所重要研究机构。前身为1928年成立的[[中华民国|中华民国]]'''国立中央研究院天文研究所'''和由天文研究所建于1934年的'''国立紫金山天文台'''。1950年5月20日国立中央研究院天文研究所更名为中国科学院紫金山天文台。紫金山天文台是中国自己建立的第一个现代天文学研究机构，是中国近现代天文学的发祥地，被誉为“中国现代天文学的摇篮”。

国立紫金山天文台旧址位于[[南京|南京]][[玄武区|玄武区]][[紫金山|紫金山]]海拔267米的第三峰天堡峰上，由于[[雾霾|灰霾]]和[[光污染|光污染]]原因于20世纪80年代末改为科普教育基地，1996年被列为[[第四批全国重点文物保护单位|第四批全国重点文物保护单位]]。

== 历史沿革 ==
[[File:National_PMO_Old_Site.jpg|thumb]]
自[[南朝_(中国朝代)|南朝]][[刘宋|刘宋]]初年开始，[[紫金山|紫金山]]北麓余脉的[[北极阁|鸡笼山]]建立了“日观台”，又称“司天台”，宋元明时期日观台经过多次扩建成为钦天台并添置了[[简仪|简仪]]、[[浑天仪|浑天仪]]等多样的仪器<ref>{{cite book|title=百里秦淮话沧桑|editor= 南京市政协文史（学习）委员会|pages=181|isbn=9787806149898}}</ref>，此后随着明清时期政治中心的北移钦天台逐渐荒废<ref>{{cite book|title=南京明清建筑|editor=杨新华, 卢海鸣 |year=2001|pages=|isbn=9787305036699}}</ref>。

1913年，从比利时学成归来的时任[[中央观象台|中央观象台]]台长[[高鲁_(天文学家)|高鲁]]首次提出在紫金山的最高峰筹集天文台。1927年[[民国政府|民国政府]]在南京建立，高鲁向主持[[国立中央研究院|国立中央研究院]]工作的[[蔡元培|蔡元培]]提出兴建天文台，同年4月国立紫金山天文台正式开始筹建<ref name="history1">{{Cite web |url=http://www.sohu.com/a/164076872_349947 |title=中国现代天文学的摇篮 |accessdate=2018-07-19 |author=南京发布 |date=2017-08-12 |publisher=搜狐历史 |archiveurl=https://web.archive.org/web/20180720001021/http://www.sohu.com/a/164076872_349947 |archivedate=2018-07-20 |deadurl=no }}</ref>。1928年2月国立中央研究院天文研究所成立，高鲁担任首任所长，并负责为研究所选址和添置设备。1929年高鲁历任后，[[余青松|余青松]]接任所长，他与建筑大师[[杨廷宝|杨廷宝]]一起完成了对紫金山天文台的勘察选址和设计工作<ref name="history1"/><ref>{{Cite journal |title=中国现代天文事业的奠基者余青松 |author=丁蔚 |url=http://xuewen.cnki.net/CJFD-TWAH701.019.html |journal=天文爱好者 |issue=10 |year=1997 |volume=1 |accessdate=2013-12-16 |archiveurl=https://web.archive.org/web/20131217045728/http://xuewen.cnki.net/CJFD-TWAH701.019.html |archivedate=2013-12-17 |deadurl=no }}</ref>。1934年9月1日紫金山天文台正式竣工，紫金山天文台成为中国第一座现代天文台，竣工后的紫金山天文台也成为国立中央研究院天文研究所的办公驻地<ref name="move">{{Cite web |url=http://qclz.youth.cn/gw/xwtt/201209/t20120901_2402750_2.htm |title=青春励志故事张钰哲：追星不移 |accessdate=2018-07-20 |author=朱琪红 |date=2012-09-01 |publisher=中国青年网 |archiveurl=https://web.archive.org/web/20180720001023/http://qclz.youth.cn/gw/xwtt/201209/t20120901_2402750_2.htm |archivedate=2018-07-20 |deadurl=no }}</ref>。

1949年中国科学院成立后，次年4月6日在南京接管了包括国立中央研究院天文研究所在内的5个研究所<ref>{{Cite web |url=http://www.cas.cn/jzzky/ysss/bns/200909/t20090928_2528957.shtml |title=中科院编年史1949 |accessdate=2018-07-20 |author=中科院 |date=2009-09-28 |archiveurl=https://web.archive.org/web/20141024022750/http://www.cas.cn/jzzky/ysss/bns/200909/t20090928_2528957.shtml |archivedate=2014-10-24 |deadurl=no }}</ref>。1950年，根据中科院研究机构的改组方案中央研究院天文研究所更名为中国科学院紫金山天文台<ref>{{Cite web |url=http://www.cas.cn/jzzky/ysss/bns/200909/t20090928_2529002.shtml |title=中科院编年史1950 |accessdate=2018-07-20 |author=中科院 |date=2009-09-28 |archiveurl=https://web.archive.org/web/20141004223559/http://www.cas.cn/jzzky/ysss/bns/200909/t20090928_2529002.shtml |archivedate=2014-10-04 |deadurl=no }}</ref>。1950年5月20日张钰哲被[[中华人民共和国政务院|政务院]]委任为中科院紫金山天文台台长<ref>{{Cite web |url=http://www.pmo.ac.cn/cxwh/zyzys/ |title=纪念张钰哲院士逝世三十周年 |accessdate=2018-07-20 |author=中科院 |archiveurl=https://web.archive.org/web/20180720001022/http://www.pmo.ac.cn/cxwh/zyzys/ |archivedate=2018-07-20 |deadurl=no }}</ref><ref name="Intro1">{{Cite web |url=http://www.pmo.cas.cn/ztjj/ |title=紫台简介 |accessdate=2018-07-19 |author=中科院紫金山天文台 |archiveurl=https://web.archive.org/web/20171225083143/http://www.pmo.cas.cn/ztjj/ |archivedate=2017-12-25 |deadurl=no }}</ref>。1938年至1946年紫金山天文台因[[抗日战争|抗日战争]]随国立中央研究院天文研究所西迁至[[昆明|昆明]]东郊的[[凤凰山_(昆明)|凤凰山]]，抗战结束后天文研究所迁回南京时在凤凰山留下一个工作成为了[[云南天文台|云南天文台]]的前身<ref>{{Cite web |url=http://www.ynao.cas.cn/jggk/jgjj/ |title=云南天文台简介 |accessdate=2018-07-20 |author=中科院云南天文台 |date=2015-11-13 |archiveurl=https://web.archive.org/web/20171225110540/http://www.ynao.cas.cn/jggk/jgjj/ |archivedate=2017-12-25 |deadurl=no }}</ref>。

1949年后，紫金山天文台曾成立中国科学院紫金山天文台、地球物理所上海联合工作站，直属管理徐家汇观象台、佘山观象台，后来发展成为[[上海天文台|上海天文台]]。相继于50年代成立的天津纬度站、北京授时台则在1962年发展为[[国家天文台|国家天文台]]的前身北京天文台<ref name="Intro2">{{Cite web |url=http://www.pmo.cas.cn/cxwh/whhd/201205/P020120821580470473026.pdf |title=紫金山天文台概况2012 |accessdate=2018-07-19 |author=中科院紫金山天文台 |date=2012-01-01 |archiveurl=https://web.archive.org/web/20130722094116/http://www.pmo.cas.cn/cxwh/whhd/201205/P020120821580470473026.pdf |archivedate=2013-07-22 |deadurl=no }}</ref>。紫金山天文台主办的《天文学报》等刊物亦在1966年至1974年期间停刊<ref>{{Cite web |url=http://www.twxb.org/index.php/index/index/index/23503 |title=天文学报过刊阅览 |accessdate=2018-07-20 |author=《天文学报》编辑部 |archiveurl=https://web.archive.org/web/20171227141816/http://www.twxb.org/index.php/index/index/index/23503 |archivedate=2017-12-27 |deadurl=no }}</ref>。紫金山天文台在文革后才得到逐步恢复。

由于80年代末起，南京夜间[[光污染|光污染]]严重，紫金山天文台的晚间专业观测的功能受到严重影响，此外受到灰霾的影响天文台需要更为频繁的花费大量经费擦拭镜头，又考虑到紫金山天文台的主要观测工作，已由青海德令哈等外地优良观测台址承担，紫金山的天文观测项目逐步减少，除仅保留一些太阳黑子、太阳射电、太阳光谱等白天进行的观测项目，主要以开放游客参观，担当天文博物馆的功能
<ref name="museum">{{Cite news |url=http://dz.xdkb.net/html/2013-03/23/content_253271.htm |title=紫金山天文台 “改行”做博物馆 |author=胡玉梅 |date=2013-03-23 |work=现代快报 |accessdate=2018-07-19 |page=A02 |archiveurl=https://web.archive.org/web/20180720001022/http://dz.xdkb.net/html/2013-03/23/content_253271.htm |archivedate=2018-07-20 |deadurl=no }}</ref>。1996年，[[中华人民共和国国务院|国务院]]将紫金山天文台以国立紫金山天文台旧址名义列入第四批全国重点文物保护单位<ref name="Heritage">{{Cite web |url=http://www.sach.gov.cn/col/col1647/index.html |title=国务院关于公布第四批全国重点文物保护单位的通知(国发[1996]47号) |accessdate=2018-07-19 |author=国家文物局 |date=1996-11-20 |archiveurl=https://web.archive.org/web/20141008064915/http://www.sach.gov.cn/col/col1647/index.html |archivedate=2014-10-08 |deadurl=no }}</ref>。

== 机构设置 ==
=== 研究部 ===
紫金山天文台以天体物理和天体力学为主要研究方向，共设置了四个研究部<ref>{{Cite web |url=http://www.pmo.cas.cn/jgsz/yjsz/ |title=研究室组综合概况 |accessdate=2018-07-19 |author=中科院紫金山天文台 |archiveurl=https://web.archive.org/web/20171219100014/http://www.pmo.cas.cn/jgsz/yjsz/ |archivedate=2017-12-19 |deadurl=no }}</ref>：
* 暗物质和空间天文研究部：成立于1976年，前身是空间天文实验部，主要在空间天文领域开展研究，重点研究暗物质粒子空间探测技术<ref>{{Cite web |url=http://www.pmo.cas.cn/jgsz/yjsz/awzhkjtwyjb/ |title=暗物质和空间天文研究部 |accessdate=2018-07-19 |author=中科院紫金山天文台 |archiveurl=https://web.archive.org/web/20150510102442/http://www.pmo.cas.cn/jgsz/yjsz/awzhkjtwyjb/ |archivedate=2015-05-10 |deadurl=no }}</ref>。
* 南极天文和射电天文研究部：成立于2006年，主要在射电天文观测和南极地区的天文观测领域开展研究<ref>{{Cite web |url=http://www.pmo.cas.cn/jgsz/yjsz/njtthsdtwyjb/ |title=南极天文和射电天文研究部 |accessdate=2018-07-19 |author=中科院紫金山天文台 |archiveurl=https://web.archive.org/web/20160621052747/http://pmo.cas.cn/jgsz/yjsz/njtthsdtwyjb/ |archivedate=2016-06-21 |deadurl=no }}</ref>。
* 应用天体力学和空间目标与碎片研究部：成立于1957年，前身是中科院人造卫星观测研究中心，主要研究方向为碎片观测和航天器精密定轨<ref>{{Cite web |url=http://www.pmo.cas.cn/jgsz/yjsz/yyttlxykjmbyspyjb/ |title=应用天体力学和空间目标与碎片研究部 |accessdate=2018-07-19 |author=中科院紫金山天文台 |archiveurl=https://web.archive.org/web/20150510095623/http://www.pmo.cas.cn/jgsz/yjsz/yyttlxykjmbyspyjb/ |archivedate=2015-05-10 |deadurl=no }}</ref>。
* 行星科学和深空探测研究部：主要研究领域为深空探测和天文历法<ref>{{Cite web |url=http://www.pmo.cas.cn/jgsz/yjsz/xxkxhsktcyjb/ |title=行星科学和深空探测研究部 |accessdate=2018-07-19 |author=中科院紫金山天文台 |archiveurl=https://web.archive.org/web/20171217085712/http://www.pmo.cas.cn/jgsz/yjsz/xxkxhsktcyjb/ |archivedate=2017-12-17 |deadurl=no }}</ref>。
===实验室===
紫金山天文台有4个中国科学院重点实验室<ref>{{Cite web |url=http://www.bfse.cas.cn/sysjcss/yzdsys/ |title=院重点实验室 |accessdate=2018-07-19 |author=中科院前沿科学与教育局 |archiveurl=https://web.archive.org/web/20171227002301/http://www.bfse.cas.cn/sysjcss/yzdsys/ |archivedate=2017-12-27 |deadurl=no }}</ref>：
* 中科院空间目标与碎片观测重点实验室：隶属于应用天体力学和空间目标与碎片研究部，从事碎片观测和航天器精密定轨研究。
* 中科院行星科学与深空探测重点实验室：隶属于应用天体力学和空间目标与碎片研究部，与中科院上海天文台共建，从事行星深空探测领域的研究。
* 中科院暗物质与空间天文重点实验室：隶属于暗物质和空间天文研究部，开展暗物质粒子空间探测领域的研究。
* 中科院射电天文重点实验室：隶属于南极天文和射电天文研究部，开展射电天文观测领域的研究。

=== 观测站 ===
紫金山天文台辖有6个野外观测台站分别是<ref>{{Cite web |url=http://www.pmo.cas.cn/jgsz/gctz/ |title=观测台站 |accessdate=2018-07-19 |author=中科院紫金山天文台 |archiveurl=https://web.archive.org/web/20171124083205/http://www.pmo.cas.cn/jgsz/gctz/ |archivedate=2017-11-24 |deadurl=no }}</ref>：
====青海德令哈射电天文观测站====
青海德令哈射电天文观测站，简称青海观测站，始建于20世纪80年代初，位于[[青海省|青海省]]第三大城市[[德令哈|德令哈]]，海拔3200米，东经97°33'，北纬37°22'。该站运行有中国最先进的13.7米口径毫米波波段射电望远镜。
====江苏盱眙天体力学观测站====
{{main|紫金山天文台盱眙观测站}}
江苏盱眙天体力学观测站，简称盱眙观测站，位于[[江苏省|江苏省]][[盱眙县|盱眙县]][[铁山寺国家森林保护区|铁山寺]][[跑马山|跑马山]]，东经118"28′，北纬32°44′，海拔高度180米。该站装备了我国口径最大的施密特望远镜——口径105/120厘米的近地天体探测望远镜和口径65/73厘米水平式空间碎片探测望远镜。
==== 江苏赣榆太阳观测站 ====
天文台赣榆太阳观测站，简称赣榆观测站，位于[[江苏省|江苏省]][[赣榆县|赣榆县]]境内，是江苏境内几个日照时间最长的地方之一，多数年份日照时间超过2200小时，装备有有一架口径为26公分的太阳精细结构望远镜。
==== 黑龙江洪河观测站 ====
黑龙江洪河观测站，简称洪河观测站，位于[[黑龙江省|黑龙江省]]东部[[三江平原|三江平原]]地区的洪河农场内，装备了4台光电观测设备，包括1套光电望远镜阵、1台精密跟踪望远镜和2台联动望远镜。
==== 山东青岛观象台 ====
{{main|青岛观象台}}
山东青岛观象台，简称青岛观象台，位于[[山东省|山东省]][[青岛市|青岛市]] ，1898年由德国人创建，在开展太阳研究方面较为突出。

==== 云南姚安观测站 ====
云南姚安观测站，简称姚安观测站，位于[[云南省|云南省]][[楚雄彝族自治州|楚雄彝族自治州]][[姚安县|姚安县]]西湖岭，该地区日照时间长观测条件好，装备有4个3层直径6米的天文圆顶。

==学术工作与成就==
=== 学术刊物 ===
紫金山天文台负责多种学术刊物和书籍的编辑、出版工作，包括<ref name="Intro1"/>：
* 《天文学报》（季刊）
* 《Chinese Astronomy and Astrophysics》（英文）

==== 小行星 ====
1955年至1983年间，在紫金山天文台共发现了147颗小行星。1983年后由于光污染等原因造成观测障碍，该台未再发现任何小行星。<ref name="museum"/>
