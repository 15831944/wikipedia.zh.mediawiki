'''紫金县'''（传统外文：{{lang|pinyin|'''Tzekim'''}}）位于[[中国|中国]][[广东省|广东省]]中部、[[东江|东江]]中游东岸，属于[[河源市|河源市]]管辖，东边与广东省[[五华县|五华县]]相邻，南边与广东省[[惠东县|惠东县]]、[[陆河县|陆河县]]相邻，西边与广东省[[博罗县|博罗县]]、[[惠州市|惠州市]][[惠城区|惠城区]]相邻，北边与广东省[[东源县|东源县]]相邻。

== 行政区划 ==
2005年5月辖20个镇：'''[[紫城|紫城]]'''、[[中坝|中坝]]、'''[[龙窝|龙窝]]'''、[[九和|九和]]、[[上义|上义]]、'''[[蓝塘|蓝塘]]'''、[[凤安|凤安]]、[[义容|义容]]、'''[[古竹|古竹]]'''、'''[[临江|临江]]'''、[[柏埔|柏埔]]、[[黄塘|黄塘]]、[[敬梓|敬梓]]、[[乌石|乌石]]、[[水墩|水墩]]、[[南岭|南岭]]、[[苏区|苏区]]、[[瓦溪|瓦溪]]、[[附城|附城]]、[[好义|好义]]。县政府驻地在[[紫城|紫城]]。（'''粗体'''为中心镇<ref>[http://www.heyuan.gov.cn/T/2009-12-23/Article_31853_1.shtml 河源市行政区划一览表] {{Wayback|url=http://www.heyuan.gov.cn/T/2009-12-23/Article_31853_1.shtml |date=20110114131841 }}，河源市政府门户网站，2009-12-11，2010年7月11日查阅</ref>）

== 历史 ==
[[明朝|明朝]][[隆庆|隆庆]]三年正月辛未（1569年2月12日）置永安县。

民国3年（1914年），改名为紫金县，因县城的紫金山而得名。

== 民族 ==
紫金县以[[汉族|汉族]][[客家人|客家人]]为主。

== 语言 ==
全县通用[[客家话|客家话]]，属纯客家县。

== 经济 ==
2007年，全县的[[地区生产总值|地区生产总值]]为46.7亿元，工农业生产总值为55.21亿元。

== 旅游 ==
比较著名的景点有：紫金观、庙子石南母寺、 越王石、道姑岩、武顿山、白溪水库风景区、九和温泉（御临门温泉度假村）、天字嶂。

== 交通 ==
全县通车总里程1960公里。
