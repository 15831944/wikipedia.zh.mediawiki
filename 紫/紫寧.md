'''張紫寧'''（{{bd|1996年|3月9日|}}），藝名'''紫寧'''，英文名'''Winnie'''，为麥銳媒樂旗下的團體[[Mera|Mera]]成員之一，團內擔任主唱。目前為女團[[火箭少女101|火箭少女101]]成員，因[[騰訊視頻|騰訊視頻]]將簽訂兩年割裂式合約，在組合兩年存續期中將不會以原公司名義進行活動。2018年8月9日，麥銳媒樂聲明退出。騰訊下屬海南周天娛樂於9日及14日兩度發表聲明，稱該行為無法律依據及合同依據，不予接受。 2018年8月17日，麥銳娛樂、騰訊下屬海南周天娛樂發佈聯合聲明，宣佈紫寧歸隊。

==簡介==
===性格===

外表像只精灵内心却黄金般稳定的灵动系少女，外冷内热，孝顺、坚韧又独立，对音乐有着特别的偏执。是一位唱歌、吉他和作曲全能型音乐才女，双眼大又带着十足的灵性，特长之一就是睫毛特长，在认真唱歌的时候会化身实力小姐姐，而在嬉戏玩乐的时候又会瞬间化身吵吵闹闹的小精灵。

科班出身，有良好稳定的唱功，在组合里发挥着不可小觑的作用。浑身散发仙气，外型条件优越，像个懂事乖巧的少女，内心却蕴藏着无限的演艺潜力，在出道预告短片中，通过她的精彩演绎，搭配轻松自在的旋律和动人舒服的编曲，完美的展现出了少女追求自我，勇于挑战的面貌。

== 經歷 ==
===出道前===
2017年7月19日，主演泰国曼谷城市先导片《Hello 梦想》MV公开，在片中饰演一位画家。

===2017年:MERA出道===
*8月3日，[[Mera|Mera]]组合发行首张迷你专辑第一首主打曲《Hello 梦想》
*8月16日，參加[[Mera|Mera]]組合在北京举行出道发布会，正式发行第一张迷你专辑《天生》，正式出道。
*8月19日，出演[[CCTV|CCTV]]音乐节目《[[全球中文音乐榜上榜|全球中文音乐榜上榜]]》，是為张紫宁的电视首秀。
*10月，受邀参加春夏巴黎时装周。
*11月23日，参加组合举办的出道100天见面会，并推出第四首单曲《陪在我身边》，该曲是一首以抒情性的旋律线以及大众性的hook为基础的歌曲。
*11月29日，随组合出席亚洲音乐盛典颁奖典礼，获得年度荣誉新人组合奖。

*12月22日，随组合发行第五首单曲《冬日的奇迹》，并参与该曲中文歌词的创作。

===2018:参加《创造101》===
2018年4月21日起，以练习生的身份参加由[[腾讯视频|腾讯视频]]出品的女团竞演节目《[[创造101|创造101]]》，并随节目发行同名主题曲《创造101》。

*6月23日，《[[创造101|创造101]]》收官，[[火箭少女101|火箭少女101]]正式成团。以第7名的成绩加入组合，并于现场演唱成团曲《rocket girls》。

===火箭少女101===
====2018====
*6月24日，组合受邀出演湖南卫视《毕业歌会》节目，这是[[火箭少女101|火箭少女101]]组合的成团首秀。
*7月6日，组合受邀出演《奔跑吧嘉年华》，演唱歌曲《创造101》。
*7月26日，火箭少女101发行为电影《[[西虹市首富|西虹市首富]]》演唱的插曲《卡路里》，并发行MV。
*8月9日，宣布退出组合[[火箭少女101|火箭少女101]]。<ref>{{cite web |title=美岐宣仪紫宁退出火箭少女 原经纪方:工作超负荷{{!}}孟美岐{{!}}吴宣仪{{!}}紫宁_新浪娱乐_新浪网 |url=http://ent.sina.com.cn/y/yneidi/2018-08-09/doc-ihhnunsp9659094.shtml |website=ent.sina.com.cn |date=2018-08-09}}</ref>
*8月17日，麥銳娛樂证实将回归[[火箭少女101|火箭少女101]]。<ref>{{cite web |title=乐华麦锐腾讯发布联合声明：三人将回归火箭少女{{!}}火箭少女101{{!}}乐华娱乐{{!}}麦锐娱乐_新浪娱乐_新浪网 |url=http://ent.sina.com.cn/y/yneidi/2018-08-17/doc-ihhvciiw4263489.shtml |website=ent.sina.com.cn |date=2018-08-17}}</ref>
*12月31日，組合受邀出席湖南衛視「快樂中國2018-2019跨年演唱會」，演唱歌曲《月亮警察》、《卡路里》、《LIGHT》。<ref>{{cite web |title=网曝四大卫视跨年晚会阵容：跑男7子现身浙江，火箭少女助阵湖南 |url=http://www.sohu.com/a/282417560_114941 |website=www.sohu.com}}</ref>

*1月12日，組合在中國上海市梅賽德斯奔馳文化中心舉辦第一場演唱會。
