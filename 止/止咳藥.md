{{Medical}}
'''止咳药'''是指治疗[[咳嗽|咳嗽]]的[[药品|药品]]，通常为[[药片|片状]]（亦有含化的）、[[溶液|溶液]]和[[混悬液|混悬液]]。

== 种类 ==
;药片
复方甘草片是[[中国大陆|中国大陆]]生产的一种含有[[吗啡|吗啡]]成分的止咳药，2005年以前在中国大陆几乎是常用药品。起初药店不用提供任何身份证明即可购得，直到2004年列入处方药以后，使用者慢慢减少，更多人则选择了[[非处方药|非处方药]]。

;溶液
[[File:Professorin_yskanlaake.jpg|thumb]]
部分止咳水含有[[可待因|可待因]]或[[麻黄碱|麻黄碱]]的过量服用会使人产生欣快感，故其被滥用者稱為'''搖頭水'''<ref>{{cite news|url=http://www.sznews.com/news/content/2013-07/11/content_8303235.htm|title=止咳水成瘾调查：被挥霍的青春是社会之殇|work=深圳新闻网|date=2013-07-11|accessdate=2014-08-16}}</ref>），泛指用於止咳的所有[[藥水|藥水]]，包括[[醫生|醫生]]或[[藥房|藥房]]提供給[[病人|病人]]使用的藥水。虽然遵循[[医嘱|医嘱]]使用有治疗[[咳嗽|咳嗽]]的作用，但是如果使用過多止咳水而造成攝入止咳水内高劑量的[[可待因|可待因]]或[[鴉片|鴉片]]，會令用者感覺興奮，及致[[上癮|上癮]]甚至犯罪<ref>{{cite news|url=http://epaper.syd.com.cn/sywb/html/2013-12/18/content_966035.htm|title=败家子取款机前抢劫只为筹“嗑药”钱|work=沈阳晚报|date=2013-12-18|accessdate=2014-08-16|archive-url=https://web.archive.org/web/20140819085510/http://epaper.syd.com.cn/sywb/html/2013-12/18/content_966035.htm|archive-date=2014-08-19|dead-url=yes}}</ref>，而由醫生處方則為一湯匙，適量可大大減少咳嗽機會。<ref>{{Cite web|url=http://www.youth.com.tw/db/epaper/es002004/eb2702.htm|title=為何感冒藥讓人「很感冒」？|accessdate=2016-12-14|work=www.youth.com.tw}}</ref>

;混悬液

一些止咳药采用混悬液的方式制作，通常饮用之前需要摇匀。例如[[右美沙芬|右美沙芬]]混悬液。

== 副作用 ==
* [[呼吸|呼吸]]壓抑
* [[口渴|口渴]]
* [[便秘|便秘]]
* [[食慾不振|食慾不振]] 
* [[皮膚|皮膚]]瘙痒、长[[皮疹|皮疹]]
* [[蛀牙|蛀牙]]、[[代謝症候群|代謝症候群]]（許多止止咳水含糖，使用後不注重牙齒清潔、過量使用則會有此副作用）
* [[精神错乱|精神错乱]]
* [[物质依赖|物质依赖]]（一些止咳水是鴉片類藥物，過量使用會成癮，依建議量使用通常成瘾的可能性比较小。倘若使用鴉片類藥物來治疗咳嗽，则其所需的劑量遠低於成癮劑量）
== 娱乐滥用 ==
一些含有[[罂粟|罂粟]]及其提取物如[[鸦片|鸦片]]、[[吗啡|吗啡]]、[[可待因|可待因]]的止咳药若大量服用会造成严重的[[物质依赖|物质依赖]]。2014年，沈阳一名男子为购买复方磷酸可待因溶液（商品名“[[立健亭|立健亭]]”，黑话俗称“大力”）而抢劫，因服用此止咳药水之后语无伦次而[[網路爆紅|網路爆紅]]<ref>{{cite news|url=http://liaoning.lnd.com.cn/htm/2014-01/29/content_3450226.htm|title=“大力哥”一夜暴富后生活乱了套|work=辽沈晚报|date=2014-01-29|accessdate=2014-08-16|archive-url=https://web.archive.org/web/20140819102823/http://liaoning.lnd.com.cn/htm/2014-01/29/content_3450226.htm|archive-date=2014-08-19|dead-url=yes}}</ref>。
