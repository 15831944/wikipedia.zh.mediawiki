'''箇舊市'''是[[中華人民共和國|中国]][[雲南省|雲南省]][[紅河哈尼族彝族自治州|紅河哈尼族彝族自治州]]的一個[[縣級市|縣級市]]，面积1587平方公里，人口45.33万。當地以產[[錫|錫]]著名，開採錫礦的歷史有約2000年。箇舊市是中国最大的产锡基地，被稱作「中國錫都」。箇舊市同时是世界上最早的产锡基地。

== 历史 ==
清光绪十一年（1885年），设箇舊厅，建立衙署，专管矿务。民国二年（1913年），箇舊被列为云南省一等大县。

1951年1月撤县建市。1958年，箇舊市隶属红河州，州府迁箇舊市，同时撤销开远、蒙自县归个箇舊辖。后两县于1961年划出。

1988年，经国务院批准，箇舊为云南省计划单列市，行使地州级经济管理权。

2003年，红河州府迁[[蒙自市|蒙自市]]。

2012年，箇舊市依然为红河州最繁荣城市，在云南省中经济发展较为发达。

现在箇舊市正在计划联合周边[[开远市|开远市]]与[[蒙自市|蒙自市]]打造云南第二大城市箇开蒙城市群。

== 行政区划 ==
箇舊市辖1個街道、7個镇、2個乡：
* 城区街道
* 锡城镇、[[沙甸镇|沙甸镇]]、鸡街镇、大屯镇、老厂镇、卡房镇、蔓耗镇
* 贾沙乡、保和乡

== 地理 ==
座落在东经102°54'~103°24'，北纬23°01'~23°36'位置剛好位於[[北回歸線|北回歸線]]上，最高海拔2740米，最低海拔150米，市區海拔1688米。

=== 气候 ===
年平均氣溫16.4℃，最冷月（1月）:10.1℃，最热月（7月）：20.5℃，被称为“最适合人类居住的城市之一”。
