'''西部衝上斷層山地'''位於[[雪山山脈|雪山山脈]]、[[玉山山脈|玉山山脈]]之西，為大致數百公尺至2,000公尺左右之山地，其地質屬於[[西部麓山帶|西部麓山帶]]，為新第三紀之[[砂岩|砂岩]]與[[頁岩|頁岩]]之互層，有別於以[[板岩|板岩]]為主的[[脊梁山脈西翼地質帶|脊梁山脈西翼地質帶]]。[[林朝棨|林朝棨]]將其稱之為「西部衝上斷層山地」。<ref>林朝棨（民46）。臺灣地形，臺灣省通志稿，卷一，第1冊。南投市：臺灣省文獻委員會。</ref>北起鼻頭角，南至鳳鼻頭，全長約330公里。以[[濁水溪|濁水溪]]為界，分為南北兩段，北段稱[[加里山山脈|加里山山脈]]（加裡山山脈），南段為[[阿里山山脈|阿里山山脈]]。<ref name="multiple">楊貴三、沈淑敏（民99）。台灣全志，卷二，土地志，地形篇。南投市：台灣文獻館。</ref>

== 範圍 ==
西部衝上斷層山地北起[[鼻頭角|鼻頭角]]，南至[[鳳鼻頭|鳳鼻頭]]，全長約330公里。以[[濁水溪|濁水溪]]為界，分為南北兩段，北段稱[[加里山山脈|加里山山脈]]（加裡山山脈），南段為[[阿里山山脈|阿里山山脈]]，亦有將其合稱阿里山山脈者。本山地與東側之[[雪山山脈|雪山山脈]]、[[玉山山脈|玉山山脈]]，以[[屈尺斷層|屈尺斷層]]、[[水長流斷層|水長流斷層]]、[[水裡坑斷層|水裡坑斷層]]、[[陳有蘭溪斷層|陳有蘭溪斷層]]、[[塔塔加斷層|塔塔加斷層]]（也稱[[大塔口斷層|大塔口斷層]]）為界。本山地之西境為[[新莊斷層|新莊斷層]]、[[大平地斷層|大平地斷層]]、[[三義斷層|三義斷層]]、[[車籠埔斷層|車籠埔斷層]]、[[大尖山斷層|大尖山斷層]]、[[觸口斷層|觸口斷層]]，與西側之[[林口台地|林口台地]]、[[桃園台地|桃園台地]]、[[竹東丘陵|竹東丘陵]]、[[苗栗丘陵|苗栗丘陵]]、[[后里台地|后里台地]]、[[台中盆地|台中盆地]]、[[斗六丘陵|斗六丘陵]]、[[嘉義丘陵|嘉義丘陵]]、[[新化丘陵|新化丘陵]]為鄰。<ref name="multiple">楊貴三、沈淑敏（民99）。台灣全志，卷二，土地志，地形篇。南投市：台灣文獻館。</ref>

== 地質 ==
西部衝上斷層山地在地質上主要為新第三紀之[[砂岩|砂岩]]與[[頁岩|頁岩]]之互層，此山地有許多縱向[[斷層|斷層]]，將此山地縱切成數個斷層片而形成[[覆瓦構造|覆瓦構造]]，且此山地平行逆斷層群斷層面，有愈東傾斜角愈大之傾向。<ref name="multiple">楊貴三、沈淑敏（民99）。台灣全志，卷二，土地志，地形篇。南投市：台灣文獻館。</ref>

== 地形 ==
[[層階地形|層階地形]]為此山地之主要地形，由軟硬岩互層（通常是砂頁岩互層）經差別風化後，形成[[單面山|單面山]]與[[豬背嶺|豬背嶺]]。<ref name="multiple">楊貴三、沈淑敏（民99）。台灣全志，卷二，土地志，地形篇。南投市：台灣文獻館。</ref>
