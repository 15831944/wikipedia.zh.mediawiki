'''西樣似站'''（{{jpn|j=西様似駅|hg=にしさまにえき|rm=Nishi-samani eki|f=y}}）位於[[北海道|北海道]][[樣似郡|樣似郡]][[樣似町|樣似町]]西町，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|ニサ}}'''。

== 車站構造 ==
* [[側式月台|側式月台]]1面1線的地面站。以前是島式月台，也是森林鐵道的起點。
* 站房是由[[守車|守車]]改造而來。

== 車站周邊 ==
* {{Link-ja|國道336號|国道336号}}
* 樣似海水浴場
* 親子岩
* 觀音山公園
* JR北海道巴士「西町」站牌

== 歷史 ==
* 1937年8月10日 - 設立。
* 1977年2月1日 - 廢止貨物和行李處理的業務。
* 日期不詳 - 車站簡易委託化。
* 1987年4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
* 日期不詳 - 車站無人化。
