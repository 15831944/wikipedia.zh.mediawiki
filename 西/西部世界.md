{{作品名稱|name=''Westworld''|hk=西部世界|tw=西方極樂園|equiv-cn=hk}}是一部於[[HBO|HBO]]播出的美國[[科幻|科幻]][[西部片|西部]]電視劇，由[[強納森·諾蘭|強納森·諾蘭]]和妻子[[麗莎·喬伊|麗莎·喬伊]]共同創作，改編自美國小說家[[麥可·克萊頓|麥可·克萊頓]]導演和編劇的1973年科幻電影《[[鑽石宮_(1973年電影)|鑽石宮]]》，以及較少程度的1976年續集《[[未来世界_(1976年电影)|未來世界]]》。這是第二次根據兩部電影改編的電視劇，第一次是一部短暫的1980年劇集《{{link-en|西部之外|Beyond Westworld}}》。[[強納森·諾蘭|強納森·諾蘭]]、[[麗莎·喬伊|麗莎·喬伊]]、[[J·J·亞柏拉罕|J·J·亞柏拉罕]]與[[布萊恩·柏克|布萊恩·柏克]]共同擔任執行製片人，乔納森擔任第一集與季末最終集的導演。該劇於2016年10月2日首播，由10集組成<ref>{{cite web|url=http://deadline.com/2015/08/westworld-hbo-anthony-hopkins-ed-harris-1201496303/|title=‘Westworld’ First Trailer: HBO Teases Series With Anthony Hopkins, Ed Harris|publisher=Deadline.com|first=Jeremy|last=Gerard|date=2015-08-09|accessdate=2015-08-10}}</ref><ref name="Release-Month-Reveaed">{{cite news|title=#Westworld is coming this Fall on @HBO.|url=http://twitter.com/WestworldHBO/status/735908712901169152|accessdate=2016-05-26|publisher=''[[Twitter|Twitter]]''|date=2016-05-26}}</ref>。

2016年11月，HBO續訂第二季共10集，<ref>{{Cite web |url = http://deadline.com/2016/11/westworld-divorce-insecure-renewed-season-2-hbo-1201853994/ |title = 'Westworld', 'Divorce' & 'Insecure' Renewed For Season 2 By HBO |author = Nellie Andreeva |date = 2016-11-14 |publisher = Deadline Hollywood |language = en}}</ref>於2018年4月22日首播。2018年5月1日，HBO宣布續訂第3季。<ref>{{Cite web |url=http://deadline.com/2018/05/westworld-renewed-season-3-hbo-1202380588/|title=‘Westworld’ Renewed For Season 3 By HBO|publisher=Deadline Hollywood|author=Denise Petski|date=2018-05-01|accessdate=2018-05-01}}</ref>

== 劇情 ==
故事設定在未來世界，在一個龐大的高科技成人主題樂園中，有著模擬真人的機器「接待員」能讓遊客享盡情享受性欲、暴力等慾望的放縱，主要敘述被稱為「西部世界」的未來主題公園。它提供給遊客殺戮與性慾的滿足。

但是在这世界下，各种暗流-{zh-hans:涌;zh-hant:湧;}-动。部分机器人出现自我觉醒，发现了自己只是作为故事角色的存在，并且想摆脱乐园对其的控制；乐园的管理层害怕乐园的创造者控制着乐园的一切而试图夺其控制权，而乐园创造者则不会善罢甘休并且探寻其伙伴创造者曾经留下的谜团；而用買下乐园的一名神秘男子试图重新发现当年旅程所留下的謎團。

== 反響 ==
該劇贏得了正面好評。於[[爛番茄|爛番茄]]整合的評論中，第1季持有88%的新鮮度、8.13/10分（78名評論家），評論家共識：「有著受譽的高水準品質原作，精彩又令人上癮的《西部世界》平衡了智慧及迷人戲劇所衝撞出的絕對瘋狂。」<ref>{{cite web |url=https://www.rottentomatoes.com/tv/westworld/s01/ |title=Westworld: Season 1|publisher=[[爛番茄|爛番茄]] |accessdate=2016-09-26}}</ref>於[[Metacritic|Metacritic]]整合的評論中，第1季獲得74分（滿分100分；43位評論家），屬普遍的好評。<ref>{{cite web |url=http://www.metacritic.com/tv/westworld |title=Westworld: Season 1 Reviews|publisher=[[Metacritic|Metacritic]] |accessdate=2016-09-29}}</ref>〈[[Collider_(網站)|Collider]]〉評比第一季為“2016年前25大最佳電視節目”中之第15名。<ref>{{cite web|url=http://collider.com/best-tv-shows-2016#westworld|title=The 25 Best TV Shows of 2016, Ranked|publisher=[[Collider_(網站)|Collider]]|date=2016-12-30|accessdate=2017-01-01}}</ref>

第2季依舊獲得多數影評家的好評，於爛番茄整合的評論中，該季持有95%的新鮮度、7.84/10分（38名評論家），評論家共識：「建立在《西部世界》實驗性的第1季之上，深入挖掘AI人性的一面而不失其時髦又血腥的優點。」<ref>{{cite web |url=https://www.rottentomatoes.com/tv/westworld/s02/|title=Westworld: Season 2|publisher=爛番茄|accessdate=2018-04-22}}</ref>於Metacritic整合的評論中，該季季獲得75分（滿分100分；26位評論家），屬普遍的好評。<ref>{{cite web |url=http://www.metacritic.com/tv/westworld/season-2|title=Westworld: Season 2 Reviews|publisher=Metacritic|accessdate=2018-04-22}}</ref>
