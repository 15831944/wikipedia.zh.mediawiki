'''西奈半島'''（[[阿拉伯语|阿拉伯语]]：{{lang|ar|شبه جزيرة سيناء|}}；[[英文|英文]]:Sinai Peninsula）是一块[[亚洲|亚洲]]西端，北接[[地中海|地中海]]南鄰[[红海|红海]]的三角形[[半岛|半岛]]。其西部边界是[[苏伊士运河|苏伊士运河]]，东北部边界为[[以色列|以色列]]和[[埃及|埃及]]国界。西奈半島数千年以來一直具有重要战略意義。

== 地理 ==
西奈半島面積為61,000平方公里，位于[[西亚|西亚]]和[[中东|中东]]，而[[埃及|埃及]]的主要国土位於[[北非|北非]]。

== 現況 ==
西奈半島現在全部屬於[[埃及|埃及]]，地形上約為南高北低；行政區劃上，分為[[北西奈省|北西奈]]與[[南西奈省|南西奈]]兩省。人口約36萬，居民主要為世代以放牧羊及駱駝為生的[[貝都因人|貝都因人]]，還有西方農耕的[[埃及人|埃及人]]，兩者人口約相等。

== 相关事件 ==
* 據《[[出埃及记|出埃及记]]》，[[耶和华|耶和华]]在西奈订立律法之约。
* 以色列人在約公元前1250年出埃及後曾在此流落四十年。
* 1956年埃及取回[[苏伊士运河|苏伊士运河]]的主權後，[[英国|英国]]、[[法国|法国]]、[[以色列|以色列]]三国的结盟企图再次霸占西奈半岛与苏伊士运河，见：[[第二次中东战争|第二次中东战争]]
* 以色列曾在1967年[[六日战争|六日战争]]中占领西奈半岛，后於1982年4月25日大部分归还，直至1989年歸還[[塔巴_(埃及)|塔巴]]後，西奈半島完全由埃及管理。
* 2015年10月31日，一架[[俄羅斯|俄羅斯]][[科加雷姆航空|科加雷姆航空]][[科加雷姆航空9268號班機空難|9268號班機]]在西奈半島被炸毀，機上224人全數罹難。<ref>{{Cite web |url = https://tw.news.yahoo.com/俄客機墜毀-空巴重大失事再添-筆-043503820.html |title = 俄客機墜毀 空巴重大失事再添一筆 |publisher = 法新社 |language = zh-tw |accessdate = 2015年11月1日 }}</ref>
