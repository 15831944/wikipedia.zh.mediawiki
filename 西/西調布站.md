'''西調布站'''（{{jpn|j=西調布駅|hg=にしちょうふえき|rm=Nishi-chōfu eki|f=y}}）是一個位於[[東京都|東京都]][[調布市|調布市]]上石原一丁目，屬於[[京王電鐵|京王電鐵]][[京王線|京王線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KO19'''。

== 年表 ==
* 1916年9月1日，作為京王電氣軌道的車站啟用，最初名為'''上石原站'''。
* 1944年5月31日，京王與[[東京急行電鐵|東京急行電鐵]]合併後，上石原站作為京王線的車站運行。
* 1948年6月1日，東急與京王帝都電鐵分拆，上石原站成為京王的車站。
* 1959年6月1日，改稱'''西調布站'''。
* 2011年1月16日，開始暫時採用[[跨站式站房|跨站式站房]]，並且加裝升降機。
* 2013年2月22日，[[快速列車_(日本)|快速]]開始在西調布站停站，同日以'''KO19'''作為編號。

== 車站構造 ==
為[[相對式月台|相對式月台]]2面2線的[[地面車站|地面車站]]。跨站式站房於2011年1月16日啟用。付費區大廳與各月台間設有電梯與電扶梯，北口、南口與付費區外大廳設有電梯，南口設有電扶梯。

舊站房位於東端上行月台側。

== 車站周邊 ==
北口是在古時屬{{link-ja|舊甲州街道|旧甲州街道}}上石原宿（[[甲州街道#甲州街道三十八宿一覧|布田五宿]]之一）。車站與道路間有西調布站前通商店會及西調布一番街等[[商店街|商店街]]。南口則有{{link-ja|MINDS農業協同組合|マインズ農業協同組合|JA MINDS}}西調布支店等所在的西調布南口商店街。

月台西端與[[中央自動車道|中央自動車道]]交會，鄰近[[調布IC|調布交流道]]。

== 站名由來 ==
車站開設時的站名「'''上石原'''」來自於當地地名。後隨著都市發展，改為現在的「'''西調布'''」。
