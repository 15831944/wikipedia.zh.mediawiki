'''西宮伸一'''（{{bd|1952年||2012年|9月16日|catIdx=Nishimiya Shin'ichi}}）[[日本|日本]][[外交官|外交官]]，曾任[[外務省|外務省]][[外務審議官|外務審議官]]。

西宮的父親曾擔任[[贊比亞|贊比亞]]大使。西宮[[东京都立青山高等学校|东京都立青山高等学校]]畢業後，进入[[東京大學|東京大學]]法學部，1976年进入日本外务省工作。

西宮原本被安排於2012年10月接替[[丹羽宇一郎|丹羽宇一郎]]担任[[日本駐華大使|日本駐華大使]]一職。9月13日上午8時46分左右（[[日本時間|日本時間]]）在[[東京都|東京都]][[澀谷區|澀谷區]]的自宅附近路邊倒地昏迷，被送往醫院搶救<ref>[http://tchina.kyodonews.jp/news/2012/09/37523.html 候任日本駐華大使西宮伸一住院]，共同社2012年9月13日</ref>，至同月16日早上於東京都一家醫院內病逝，終年60歲<ref>[https://web.archive.org/web/20120918214707/http://www3.nhk.or.jp/news/html/20120916/t10015057891000.html 西宮伸一 中国大使が死去]，[[NHK|NHK]]2012年9月16日</ref>。

== 外交生涯 ==
1976年（昭和51年），进入日本外务省工作，曾先后担任北美局北美第2课长、[[日本駐俄羅斯大使館|日本駐俄羅斯大使館]]参事官、大臣官房参事官兼亚洲及大洋洲局，日本驻華公使和特命全权大使。<ref name=" fish" >{{cite web |url=http://www.chinanews.com/gj/2012/09-11/4174584.shtml |title=日本任命西宮伸一為新駐華大使 |accessdate=2012年9月11日 |publisher=中新网 |work= |language =zh-cn}}</ref>

2007年，担任北美局局长。<ref name=" fish" />

2010年2月，担任驻纽约总领事；同年12月，担任外务省的經濟事務外务审议官。<ref name=" fish" />

2012年8月，日本《[[共同社|共同社]]》发布新闻，日本政府擬起用西宫伸一接替丹羽宇一郎担任日本国驻中华人民共和国大使。<ref>{{cite web |url=http://tchina.kyodonews.jp/news/2012/08/35912.html |title=日本政府擬起用西宮伸一出任駐華大使 |accessdate= 2012-08-20|publisher=共同社 |work= |language =zh-cn}}</ref>

2012年9月11日，日本政府召开內閣會議決定任命西宫伸一接替[[丹羽宇一郎|丹羽宇一郎]]担任[[日本駐華大使|日本国驻中华人民共和国大使]]，準備10月上旬到[[北京|北京]]履新，然而直至逝世尚未履新。
