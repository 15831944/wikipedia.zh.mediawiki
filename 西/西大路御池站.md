'''西大路御池站'''（{{jpn|j=西大路御池駅|hg=にしおおじおいけえき|rm=Nishi-Ōji-Oike eki|f=y}}），是一個位於[[日本|日本]][[京都府|京都府]][[京都市|京都市]][[中京區|中京區]]西之京東中合町<ref>{{Cite web |url=http://www2.city.kyoto.lg.jp/kotsu/tikadia/hyperdia/menu0138.htm |title=ハイパー地下鉄ダイヤ：京都市営地下鉄 東西線 西大路御池駅 |accessdate=2016-03-22 |publisher=京都市交通局}}</ref>，屬於[[京都市營地下鐵|京都市營地下鐵]][[東西線_(京都市營地下鐵)|東西線]]的[[鐵路車站|鐵路車站]]。

設置於西大路御池交差點正下方。[[車站編號|車站編號]]是'''T16'''<ref name="map">[http://www.city.kyoto.lg.jp/kotsu/cmsfiles/contents/0000008/8995/metro_map20080116.gif 京都市営地下鉄路線図] - 京都市情報館 交通局</ref>。

== 歷史 ==
* 2008年（[[平成|平成]]20年）1月16日 - [[京都市營地下鐵東西線|京都市營地下鐵東西線]]的[[二條站|二條站]]延伸開通至[[太秦天神川站|太秦天神川站]]而開業<ref>{{Cite book|和書|author=[[曽根悟|曽根悟]]（監修） |title=週刊 歴史でめぐる鉄道全路線 公営鉄道・私鉄 |editor=朝日新聞出版分冊百科編集部（編集） |date=2011-04-03 |publisher=[[朝日新聞出版|朝日新聞出版]] |series=週刊朝日百科 |volume=4号 京福電気鉄道・叡山電鉄・嵯峨野観光鉄道・京都市交通局 |pages=22-23}}</ref>。

== 車站結構 ==
月台是島式1面2線，設有[[月台閘門|月台閘門]]。[[閘口|閘口]]只設1個<ref name="kounaizu">外部リンク（駅施設）の構内図より。</ref>。

東西線車站設有車站顏色，此站的車站顏色是向日葵色<ref name="kensetsu kiroku">「京都市営地下鉄東西線建設記録 第3巻 二条・太秦天神川間」2008年発行</ref>。
