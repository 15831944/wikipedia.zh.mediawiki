'''西南棋王赛'''，一般被简称为“'''西南王'''”，是中国棋院主办的一项[[中国|中国]]国内区域性[[围棋|围棋]]比赛，主辦單位為[[中国棋院|中国棋院]]、成都棋院，協辦單位是紅塔集團。2002年開始。

== 参赛棋手规定 ==
除主办方、赞助方特别邀请的棋手外，[[雲南省|雲南省]]、[[貴州省|貴州省]]、[[重慶市|重慶市]]、[[四川省|四川省]]等四省市的中国围棋联赛参赛队伍中的中国棋手方可报名参加。

== 比赛规则 ==
采用中国围棋规则，黑贴3又3/4子；每步棋30秒，每方有10次1分钟的保留时间。

== 奖金 ==
冠军16万人民币，亚军8万人民币（税前）。棋手对局费每局5000元（税后）。
