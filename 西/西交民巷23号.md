'''西交民巷23号'''是[[北京市|北京市]][[西城区|西城区]][[西交民巷|西交民巷]]23号的[[全国人大|全国人大]]机关的办公区。

==简介==
西交民巷23号有被称作“23号院”的全国人大办公楼。此处原是[[中华人民共和国铁道部|中华人民共和国铁道部]]一所设计院的旧楼，1980年代末，全国人大支付一定资金后，接手该楼使用<ref name=lw>{{Cite web |url = http://www.china.com.cn/chinese/law/605162.htm  |title = 全国人大打造新办公地点的工作暂时停止 |author =  |date =2004-07-08  |publisher = 瞭望东方周刊 |language =  |accessdate =  |quote =  }}</ref>。前门西大街1号的[[全国人大机关办公楼|全国人大机关办公楼]]2010年建成启用前，全国人大多数专门委员会、全国人大常委会工作委员会和机关单位集中在西交民巷23号院办公<ref name=lvcm>{{Cite web |url =http://www.npc.gov.cn/npc/zgrdzz/2012-10/16/content_1739492.htm  |title =从人民大会堂到西交民巷23号院  |author =吕聪敏  |date =  2012-10-16 |publisher = 中国人大网  |language =  |accessdate =  |quote =  }}</ref>。

2010年全国人大机关办公楼建成后，多数单位已迁至全国人大机关办公楼办公，仅留下《[[中国人大_(杂志)|中国人大]]》杂志社等单位仍在西交民巷23号办公。2013年12月19日，在《中国人大》杂志创刊20周年前夕，[[中共中央政治局常委|中共中央政治局常委]]、[[全国人大常委会委员长|全国人大常委会委员长]][[张德江|张德江]]来到《中国人大》杂志社调研，看望干部职工，并召开人大新闻宣传工作座谈会<ref>{{Cite web |url =http://www.npc.gov.cn/npc/xinwen/tpbd/2013-12/20/content_1817892.htm  |title =张德江到中国人大杂志社调研  |author =  |date =  2013-12-20|publisher =  中国人大网|language =  |accessdate =  |quote =  }}</ref>。2015年12月18日，在全国人大常委会委员长张德江视察《中国人大》杂志社工作两周年前夕，由《中国人大》杂志社开通的首个专业服务全国人大及地方各级人大工作的[[微信|微信]]公众号“西交民巷23号”正式上线，全国人大常委会副秘书长[[窦树华|窦树华]]及全国人大机关有关部门负责人参加了开通仪式<ref>{{Cite web |url = http://www.npc.gov.cn/npc/xinwen/2015-12/21/content_1955135.htm |title =“西交民巷23号”微信公众号开通  |author =  |date =2015-12-19  |publisher =中国人大网  |language =  |accessdate =  |quote =  }}</ref>。西交民巷23号坐北朝南，大门上方悬挂有[[中华人民共和国国徽|中华人民共和国国徽]]。
