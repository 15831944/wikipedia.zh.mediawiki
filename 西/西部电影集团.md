'''西部电影集团'''，简称'''西影集团'''，位于[[陕西省|陕西省]][[西安市|西安市]]，是[[陕西省人民政府国有资产监督管理委会员|陕西省人民政府国有资产监督管理委会员]]监管的省属企业<ref>{{Cite web |url=http://www.sxgz.gov.cn/newstyle/pub_newsshow.asp?id=2114252&chid=100057|title=西安市与省属企业投资合作座谈会成功举办|accessdate=2018-11-15 |author= |date=2018-10-24 |work= |publisher=陕西省人民政府国有资产监督管理委会员网站|language=Zh-hans }}</ref>。

集团前身为'''西安电影制片厂'''，1956年筹建，1958年8月23日成立<ref>{{Cite web |url=http://ent.qq.com/a/20081031/000237.htm|title=西影集团五十年成就辉煌 电影之歌传唱半世纪|accessdate=2018-11-15 |author= |date=2008-10-31 |work=责任编辑：lilylin |publisher=腾讯网|language=Zh-hans }}</ref>。曾经走出过中国的[[中国大陆电影|第五代导演]]代表人物[[吴天明|吴天明]]、[[张艺谋|张艺谋]]、[[陈凯歌|陈凯歌]]和[[顾长卫|顾长卫]]。2009年内正式成立为西影集团。
