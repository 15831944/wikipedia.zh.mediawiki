'''西荻窪站'''（{{jpn|j=西荻窪駅|hg=にしおぎくぼえき|rm=Nishi-ogikubo eki|f=y}}）位於[[日本|日本]][[東京都|東京都]][[杉並區|杉並區]][[西荻南|西荻南]]三丁目，是[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[中央本線|中央本線]]的[[鐵路車站|鐵路車站]]。中央線快速電車的[[車站編號|車站編號]]是'''JC 10'''，中央·總武線各站停車是'''JB 03'''。杉並區最西端車站。

作為運行系統有行走{{link-ja|急行線|急行線}}的[[中央線快速|中央線快速電車]]與行走{{link-ja|急行線|急行線|緩行線}}的[[中央·總武緩行線|中央·總武線各站停車]]2個系統停靠。但是中央線快速電車只限平日繁忙時段停靠（參見「[[中央線快速#快速停車站有關討論|中央線快速#快速停車站有關討論]]」）。另外，中央本線內依據[[特定都區市內|特定都區市內]]，本站屬於「[[特定都區市內#設定區域一覧|東京都區內]]」與{{link-ja|東日本旅客鐵道東京支社|東日本旅客鉄道東京支社|東京支社}}管內的最西端車站。

== 車站結構 ==
[[中央線快速|快速線]]、[[中央、總武緩行線|中央、總武線]]分別為1座[[島式月台|島式月台]]，共2面4線的[[高架車站|高架車站]]。

本站共有2座島式月台，由快速線、緩行線分別使用1座。周末及公眾假期時由於快速線列車全日通過本站，3、4號月台亦同時會被封閉。而平日快速線列車雖然在中野以西路段為各站皆停，但方向幕仍顯示列車種類為「快速」，避免與以三鷹為終點的緩行線混淆。

== 車站周邊 ==
車站周邊的道路與站前廣場十分狹窄。形成[[西荻窪|西荻窪]]的獨特氣氛。

[[古董|古董]]、古書店多。商店街設於車站的北與南，北口那邊比較繁榮。

== 历史 ==
*1922年7月15日 - 開業。
*1969年4月6日 - 荻窪站～三鷹站間高架雙複線化完成。
:中央線（各站停車）、[[營團地下鐵|營團地下鐵]]（現[[東京地下鐵|東京地下鐵]]）[[東西線_(東京地下鐵)|東西線]]的乘入運行區間延長至荻窪站～三鷹站（日曜、祝日為快速通過）
*1987年4月1日 - [[國鐵分割民營化|國鐵分割民營化]]時由東日本旅客鐵道繼承
*1994年12月3日 - 時刻表改正後土曜日為休日時間化。（與高圓寺、阿佐谷同樣於土曜日快速列車通過不停站。）
