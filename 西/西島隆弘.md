'''西島隆弘'''（{{jpn|j=西島 隆弘|hg=にしじま たかひろ}}，1986年9月30日），[[日本|日本]]男[[藝人|藝人]]，[[AAA_(團體)|AAA]]主唱，經紀公司為[[愛貝克思管理|愛貝克思管理]]。身高170公分，[[北海道|北海道]][[札幌市|札幌市]]出身，青森山田高等學校畢業。

==來歷==
*[[北海道|北海道]]札幌工業高等學校
*代表札幌Actors Studio的在2003年結成的5人組男子音樂團體「[[japs|japs]]」的初代成員之一。
**在2003年9月japs發行出道單曲「Precious Love／不停止」(CD+DVD)。
*2003年AVEX舉辦的甄選會合格。
*2004年2月參加了Actors Studio的｢BACKEN RECORD vol.13｣的出演後、到[[東京|東京]]。在這個live，獨唱了[[EXILE|EXILE]]的歌曲。
*2005年擔任NTT docomo[[北海道|北海道]]「i　Project」的Image character。
*2005年擔任[[鈴木亞美|鈴木亞美]]的「Delightful」、「Hopeful」、「Eventful」歌曲的伴舞。
*2006年8月18日TBS電視台[[戀愛觀察室|戀愛觀察室]]節目企劃裡和女演員[[金子SAYAKA|金子SAYAKA]]約會。
*2006年11月15日發行的AAA 11th單曲「[[口香糖(AAA)|口香糖]]」和同成員[[浦田直也|浦田直也]]只有2人一起唱。
*2007年1月17日出演舞台劇「LOVE　LETTERS」、和[[大澤AKANE|大澤AKANE]]完成了共演。
*2007年4月2日開始在[[東京電視台|東京電視台]]連續劇「[[美味學院|美味學院]]」和同成員[[與真司郎|與真司郎]]一起演出。
*2007年4月3日在「音時間」和同成員[[宇野實彩子|宇野實彩子]]一起擔任MC。
*2007年9月5日發行的[[奥村初音|奥村初音]]的出道單曲「戀、花火」的短篇電影和[[谷村美月|谷村美月]]一起演出。(全編[[大阪|大阪]]收錄)
*2007年11月2日在[[NHK|NHK]]教育台放送的「王子的lesson street dance」演出。
*2008年初夏預定放映的[[園子溫|園子溫]]導演的電影「[[愛的曝光|愛的曝光]]」，第一次演出電影和主演。
*2008年6月27日發行個人寫真集。
*2008年在「Harold 和 mode」出演。
*2009年4月2日在[[NHK|NHK]]連續劇「[[Ghost_Friends|Ghost Friends]]」(幽靈朋友)與[[福田沙紀|福田沙紀]]主演
*2010年1月23日電影「再見 總有一天」上映，由小說改編電影，戲中與西岛秀俊是父子關係。

== 人物 ==
* 6歲已經開始跳舞及唱歌。
* 在2005年的一次練習中跌倒，導致左臂骨折。2006年10月取出左臂上的螺栓。
* 會彈吉他，單曲{{lang|ja|夢ノカケラ}}的音樂錄影帶和2008年巡迴演唱會的現場版《口香糖》都有展現。
* 與藝人[[藤谷太輔|藤谷太輔]] ([[Kis-My-Ft2|Kis-My-Ft2]])、[[大東駿介|大東駿介]]、[[三浦翔平|三浦翔平]]、[[相葉裕樹|相葉裕樹]]的關係良好。
* 早年受柚子樂隊的主唱[[北川悠仁|北川悠仁]]影響最深，2014年同時出席且並排出現在日本唱片大賞的時候被主持人[[枡田繪理奈|枡田繪理奈]]介紹為「這是兩個柚子麽……（ゆずのお二人ですとか…）」，Twitter的愛好者也留言表示「兩人十分相似」「被兩人相似的笑容治癒了」「兩人的合影超可愛」。<ref>{{cite web |url=http://news.livedoor.com/lite/article_detail/9630900/ |title = AAA西島隆弘&北川悠仁が「ゆずのお二人」と紹介される珍事|accessdate=2014-12-31|publisher= [[ライブドアニュース|ライブドアニュース]] }}</ref>
*最喜歡吃的食物是刺身，不喜歡吃黃瓜。
*西島隆弘沒有在任何[[SNS|SNS]]社交網站開設個人主頁。AAA的隊友[[日高光啟|日高光啟]]表示：「因為西島是機械盲。例如發送訊息到他電腦，他卻不懂打開的方法，最後還是我到西島家幫他打開<ref>{{Cite web |url = https://twitter.com/SkyHidaka/status/24364364450504704 |title =luv0930nissy: にっしーがツイッターやらない理由分かりますか? |author=[[日高光啟|日高光啟]]|date = 2011-01-10|publisher = Twitter |language = ja|accessdate =2015-05-06 |quote= 機械オンチだからね。西島のPCにデータを送っても、開封の仕方がわからなかったらしく、結局西島の家に行って俺が開封したくらいです。彼はバーナーが出る度「ウイルスだ！」と怯えます。 |archiveurl= http://favotter.net/status.php?id=24364364450504704|archivedate=2013-06-16}}</ref>。」
*西島隆弘的工作組在2015年4月辦了twitter的工作賬戶，跟隨者至2018年已有40萬人https://twitter.com/nissystaff<nowiki/>（Nissy的twitter網址）
