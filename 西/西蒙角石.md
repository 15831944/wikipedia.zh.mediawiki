'''西蒙角石'''（[[學名|學名]]：{{lang|la|'''''Cimomia'''''}}），又名'''西蒙鸚鵡螺'''，是一[[屬|屬]]已[[滅絕|滅絕]]的[[鸚鵡螺亞綱|鸚鵡螺類]]。生存在[[白堊紀|白堊紀]]至[[漸新世|漸新世]]的[[海洋|海洋]]中。<ref name="Fossilworks">{{Cite web |url = http://fossilworks.org/bridge.pl?a=taxonInfo&taxon_no=13277 |title = Fossilworks: Cimomia |language = en |accessdate = 2018-05-16 }}</ref>牠們的外殼側面具有[[S|S]]型彎曲的縫合線。<ref name="Pseudaganides">{{Cite web |url = http://liusir.pixnet.net/blog/post/248633462-cimomia-%E8%A5%BF%E8%92%99%E8%A7%92%E7%9F%B3-%28%E6%BE%B3%E6%B4%B2%29 |title = Cimomia 西蒙角石 (澳洲) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-05-16 }}</ref>
