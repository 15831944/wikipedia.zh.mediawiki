'''西印度洋矛尾魚'''（[[学名|学名]]：{{lang|la|''Latimeria chalumnae''}}），又稱'''東非矛尾魚'''，是[[腔棘魚目|腔棘魚目]][[矛尾魚屬|矛尾魚屬]]的其中一[[種|種]]。原以為[[總鰭魚|總鰭魚]]已經全面[[滅絕|滅絕]]，但1938年一名南非漁民捕魚時竟發現了活體，後又多次在同一海域成功捕獲，故被稱為“[[活化石|活化石]]”。目前主要分佈于[[印度洋|印度洋]]西南沿海。矛尾魚一般生活在約400米深的海水中。{{r|washingtonpost}}

2000年，該[[物種|物種]]已被[[世界自然保护联盟|世界自然保护联盟]]評定為[[極危|極危]]。{{r|IUCN}}

== 捕食方式 ==
[[肉食性|肉食性]]，以衝刺方式捕食，專吃[[烏賊|烏賊]]和小型[[魚類|魚類]]。
== 壽命 ==
根據其內[[耳石|耳石]]年輪估計，壽命為約80歲。

== 外形特征 ==
长度可达2米，重量高达90公斤。 矛尾鱼体呈长梭形，躯体粗，头大，口宽，牙齿锐利。颅骨具特殊的颅间关节。在头下下颌间有1对很大的喉板。因而体表粗糙，体后部和鳍基部鳞较小。
