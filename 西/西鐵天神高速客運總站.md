'''西鐵天神高速客運總站'''（[[日語|日語]]：{{lang|ja|西鉄天神高速バスターミナル}}），位於[[日本|日本]][[福岡縣|福岡縣]][[福岡市|福岡市]][[中央區_(福岡市)|中央區]]，是[[西日本鐵道|西日本鐵道]]設置的[[巴士總站|客運總站]]。

== 概要 ==
車站在[[西鐵福岡（天神）車站|西鐵福岡（天神）車站]]的直上，SOLARIA總站大樓3樓。到發了許多連結九州，四国和本州主要城市長途客運路線。
車站裡有10個月台，東邊的上車月臺符號了0-6號，西邊的有3個下車月臺<ref>[[国土交通省|国土交通省]]自動車交通局 {{PDFlink|[http://www.mlit.go.jp/jidosha/sesaku/jigyo/sonota/bus_term/ichiranhyou.pdf 一般バスターミナル現況]}}</ref>。

== 櫃臺 ==
有[[西鐵巴士|西鐵巴士]]售票櫃臺，早晨和深夜沒有辦理。
另外，自動售票機可以買車票（單程，往返)。

== 月臺 ==
[[File:Nishitetsu_Tenjin_Expressway_Bus_Terminal_Information_Map.svg|缩略图]]
東邊有7面上車月臺。月臺如下所示正每隔目的地被分開。
福岡縣內巴士大致從1和2月臺開車，福岡縣外巴士從3-6月臺開車。
夜行高速巴士原則上從6月臺發車。
在加班車的時候，有上車月臺被改為的情況。
