'''西湖畫派'''指的是[[民国时期|民國時代]]在[[杭州国立艺术院|杭州國立藝專]]（今中國美術學院）六位有師承關係的藝術家，分別是[[林风眠|林風眠]]（1900－1991）、[[吳大羽|吳大羽]]（1903－1984）、[[方幹民|方幹民]]（1906－1984）、[[吳冠中|吳冠中]]（1919－2010）、[[朱德群|朱德群]]（1920－2014）、[[趙無極|趙無極]]（1921－2013）。<ref name="左">{{cite book |author1=刘钺 |title=左杭州 右巴黎-西湖画派六人集论 |publisher=当代中国出版社 |isbn=9787515401409}}</ref>

== 簡介 ==
林風眠於1928年年創立[[国立西湖艺术院|國立西湖藝術學院]]（現為[[中国美术学院|中國美術學院]]），任校長兼教授。在[[国立西湖艺术院|西湖藝專]]，[[吳大羽|吳大羽]]任西畫系主任，[[方幹民|方幹民]]則曾擔任教務長及油畫系教授。 

1935至1936年間，趙無極、朱德群和吳冠中進入[[国立西湖艺术院|國立西湖藝術學院]]就讀，先後師從吳大羽、方幹民和[[潘天寿|潘天壽]]等老師。

[[林风眠|林風眠]]、[[吳大羽|吳大羽]]、[[方幹民|方幹民]]、[[吳冠中|吳冠中]]、[[朱德群|朱德群]]、[[趙無極|趙無極]]六人都在調和中西藝術的道路上作出了重要的貢獻。<ref>{{Cite web|title=温岭日报|url=http://wldaily.zjol.com.cn/html/2013-11/08/content_1267854.htm?div=1|work=wldaily.zjol.com.cn|accessdate=2019-04-04}}</ref>

== 展覽 ==
1945年，林風眠、吳大羽、方幹民和趙無極舉辦並參加重慶「[[現代繪畫聯展|現代繪畫聯展]]」，展出作品83件，提倡「現代中國繪畫藝術與現代世界藝術合流」，其他參展者包括[[關良|關良]]、[[丁衍庸|丁衍庸]]、[[龐薰琴|龐薰琴]]等人，多為杭州藝專教授。

1946年，林風眠、吳大羽、方幹民、朱德群和趙無極參加了重慶「獨立美展」，除原「現代繪畫聯展」成員外，另加入朱德群 、[[胡善餘|胡善餘]]、[[李可染|李可染]]等人。<ref name=":0">{{Cite book|title=左杭州 右巴黎-西湖画派六人集论|last=刘|first=钺|publisher=当代中国出版社|year=2012|date=7-1|isbn=9787515401409|location=|pages=}}</ref>

== 評價 ==

* 王承昊在《傳統之根——試論趙無極、 朱德群、 吳冠中的中國藝術精神》一文中寫到：「[[趙無極|趙無極]]、[[朱德群|朱德群]]、吳冠中......先後進入杭州國立藝專學習時，正值學校的鼎盛時期。校長[[林風眠|林風眠]]以[[蔡元培|蔡元培]]「思想自由，兼容並包」的辦學宗旨廣納賢才，這裡既有像[[潘天寿|潘天壽]]這樣傳統功力深厚的國畫大師，也有像[[吳大羽|吳大羽]]、[[方幹民|方幹民]]這樣深通西方現代繪畫藝理的著名畫家，這使得他們在校的學習能在一個容納中西、開闊視野的學術環境下，吸收中西文化的精髓......林風眠、潘天壽、吳大羽、方幹民等藝術教育的形式與思想更深深影響了當時的朱德群、 趙無極、吳冠中。探尋西方藝術之源成為三人共同的夢想。」<ref>{{Cite journal|title=传统之根 ——试论赵无极、 朱德群、 吴冠中的中国艺术精神|author=王承昊|url=|journal=China Academic Journal Electronic Publishing House|issue=|doi=|others=|year=|volume=|page=100-102|pmid=}}</ref>
*莊華嶽先生回憶道，當年杭州藝專的教授中，[[林风眠|林風眠]]、[[吳大羽|吳大羽]]、[[方幹民|方幹民]]是最有才氣的三位，他們三人都是留學法國，林風眠與吳大羽多繼承了[[表現主義|表現主義]]的藝術觀，而方幹民帶回的則是[[立体主义|立體主義]]的。<ref name=":0" />
