'''西安地铁'''（全称“西安市城市轨道交通系统”），是[[中华人民共和国|中国]][[陕西省|陕西省]][[西安市|西安市]]的[[城市轨道交通系统|城市轨道交通系统]]，始建于2006年9月29日；第一条线路于2011年9月16日开通试营运，使西安市成为[[中国西北地区|中国西北地区]]首座营运地铁的城市，同时也是[[中国大陆|中国大陆]]第十个营运地铁的城市。现已投入运营[[西安地铁1号线|1号线]]、[[西安地铁2号线|2号线]]、[[西安地铁3号线|3号线]]、[[西安地铁4号线|4号线]]，共94座车站，运营里程126.55公里。西安地铁日均客流量超过200万人次，<ref>{{Cite web |url=http://www.taihainet.com/news/xmnews/szjj/2014-09-22/1312755_2.html|title= 厦门地铁预计2018年建成通车}}</ref>最高客流出现在2019年3月8日（妇女节），达312.10万人次。目前，1号线二期、5号线一期、5号线二期、6号线一期、6号线二期和9号线一期（临潼线）以及编入西安地铁网络的西安北站至西安咸阳国际机场的城际铁路等正在施工建设，8号线、14号线等正进行前期准备工作。根据2012年8月重新编制的《西安市城市轨道交通线网规划（修编）》，西安市政府将轨道交通线网的远期规划，增加到15条线路，总长586.7公里。<ref name="xam"/>西安地铁是大中华地区第一个也是目前唯一一个给每一座车站都设计独立标识的轨道交通系统。

== 建设简史 ==
*1970年代，西安市城防工程指挥部要求拆掉[[西安城墙|城墙]]，效仿北京修地下铁，并向上级提交了申请报告，计划获得军管的陕西省委的批准。后因缺乏资金导致计划搁浅。<ref name=chinaweekly>{{Cite web |url= http://www.chinaweekly.cn/bencandy.php?fid=63&id=6052 |title= 幸运儿西安城墙 |author= 刘畅 |date= 2012-05-01 |publisher= 中国周刊 |language= zh-hans |accessdate= 2014年5月3日 |quote=  }}{{Dead link|time=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>
*1980年代初，西安市开始策划、研究发展城市快速轨道交通。<ref>[http://news.hsw.cn/node_3656.htm 西安地铁大事记]</ref>
*1987年，西安市地铁交通研究小组成立。<ref>[http://www.xiancn.com/zt/xametro.html 西安进入地铁时代]</ref>
*1994年，在《西安城市总体规划（1995—2010年）》中，首次正式提出兴建4条城市轨道交通线路，长度73.17公里。
*1999年，《西安城市总体规划（1995—2010年）》得到[[中华人民共和国国务院|国务院]]批复意见。
*2004年2月，西安市向[[国家发改委|国家发改委]]上报《西安市城市快速轨道交通建设规划》。
*2006年6月，经过一年多的评估、调研、审核，国家发改委将《规划》上报国务院。
*2006年9月13日，西安地铁项目正式获得国家批准。
*2006年9月29日，首条线路——[[西安地铁2号线|西安地铁2号线]]试验段开工建设。
*2011年9月16日，2号线一期工程（[[北客站站|北客站]]至[[會展中心站_(西安)|會展中心站]]）开通试运营。
*2013年9月15日，1号线一期工程（[[後衞寨站|後衞寨站]]至[[紡織城站|紡織城站]]）开通试运营，西安地铁十字骨架初步形成，迎来换乘时代。<ref>{{Cite web |url= http://news.hsw.cn/system/2013/09/15/051756646.shtml |title= 西安地铁一号线9月15日10时通车 全程5元刷卡7折 |author=  |date= 2013-09-15 |publisher= 华商网 |language= zh-hans |accessdate= 2013年9月15日 |quote=  }}</ref>
*2014年6月16日，2号线南延线（[[會展中心站_(西安)|會展中心站]]至[[韋曲南站|韋曲南站]]）四站四区间开通。<ref>{{Cite web |url= http://news.cnwest.com/content/2014-06/16/content_11261910.htm |title= 西安地铁二号线南段今天通车 长安区进入地铁时代 |date= 2014-06-16 |publisher= 西部网 |language= zh-hans }}</ref>
*2016年11月8日，3号线一期工程（[[魚化寨站|魚化寨站]]至[[保稅區站_(西安)|保稅區站]]）开通试运营。<ref>{{Cite web |url= http://news.xinhuanet.com/politics/2016-11/08/c_129356034.htm |title= 西安地铁三号线今日开通运营 |date= 2016-11-08 |publisher= 新華网 |language= zh-hans }}</ref>
*2018年12月26日，4号线（[[航天新城站|航天新城站]]至[[北客站（北广场）站|北客站（北广场）站]]）开通试运营，其中[[火车站站|火车站站]]因国铁[[西安站|西安站]]改造暂缓开通。<ref>{{Cite web |url= http://news.hsw.cn/system/2018/1226/1051675.shtml |title= 西安地铁四号线上午10时开通试运营 线路最高票价8元 |date= 2018-12-26 |publisher= 华商网 |language= zh-hans }}</ref>

== 票务 ==

西安地铁采用“里程计价制”。票价为起步价2元6（含6公里,以下类同）公里，每增加1元，可继续乘坐4、4、6、6、8、8公里。即0-6公里2元；6-10公里3元；10-14公里4元；14-20公里5元；20-26公里6元；26公里以上每增加8公里增加1元<ref name=xametro>
{{Cite web |url= http://www.xianrail.com/html/ditiexinwen/20161028/2476.html |title= 西安地铁最新票制票价确定 - 西安地铁官方网站 |author= 西安市物价局 |date= 2016-10-28 |publisher= 西安地铁官方网站 |language= zh-hans |accessdate= 2016-11-16 |quote=  }}</ref>。采用非接触式IC卡车票，分为单程票和定值票。

;单程票
*无折扣
*在自动售票机内选择目的车站及票数后，投入纸币5元、10元或硬币1元，即可出票和找零
*单程票为IC卡，只需要在入站闸机处感应便可进入付费区；出站时只需将车票塞入出站闸机的插票口即可
*购买后2小时未进站，车票自动作废；120分钟未出站须缴纳线网最高票价5元。

;定值票
*需在地铁客服中心人工购买
*购买50元定值票预付人民币38元；购买100元定值票预付人民币60元
*进站、出站均需在闸机处感应
*定值票内余额使用完后，可换取一张线网内最高票额的单程票，且无2小时内进站时限。


*自2015年1月1日起，使用[[长安通|长安通]]普通卡可享受9折优惠，长安通学生卡可享受5折优惠
*使用长安通老年卡可在非高峰时段免费乘车（高峰时段：指周一至周五7:30-9:30、17:30-19:30）。<ref name=sxgov>
{{Cite web |url= http://www.shaanxi.gov.cn/0/1/9/39/186684.htm |title= 明年起乘地铁长安通恢复9折优惠 |author= 陕西日报 |date= 2014-12-28 |publisher= 陕西省人民政府门户网站 |language= zh-hans |accessdate= 2014-12-28 |quote=  }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
*2017年12月30日起，持外地[[全国交通一卡通互联互通|交通一卡通]]也可以在西安地铁乘车，但是不享受刷长安通的9折优惠。
*2018年1月1日，西安地铁、西安城市管理局和蚂蚁金服三方联合宣布，只需在西安地铁App或者在“支付宝-付钱-乘车码”里领取“西安电子地铁卡”，即可在地铁闸机上扫手机二维码乘车。
*2018年12月1日，西安地铁线网闸机改造工作全面完成，全线各车站所有闸机均可实现手机支付宝、微信扫码快速过闸功能。

=== 西安地铁1号线（森林公园 - 后卫寨）===  
*区间：西起西咸新区的[[森林公园站_(咸阳市)|沣河森林公园]]，沿世纪大道敷设，到达后卫寨，与一期贯通运营。
*规划:  线路全长约为6.1公里，共设车站4座
*进展：2014年11月，开始主体施工；2015年11月西咸新区段开工建设，计划2019年建成通车。

=== 西安地铁5号线（纺织城火车站 - 交大创新港）=== 
*区间：东起[[西康铁路|西康铁路]]的纺织城火车站，西南前行跨[[浐河|浐河]]，绕曲江新区，在[[青龙寺|青龙寺]]站换乘3号线，过南二环，在[[西安交通大学|西安交通大学]]南门折向西，沿友谊路直行，经[[李家村|李家村]]站换乘四号线，[[南稍门|南稍门]]站换乘2号线，在[[西北工业大学|西北工业大学]]与6号线换乘，而后由劳动南路西北前行至昆明路，跨西三环至西窑头，再经[[阿房宫|阿房宫]]到达和平村。再向西敷设，经镐京遗址、过沣河，联通西安市与西咸新区沣东新城和沣西新城，终到于交大创新港。
*线路一期全长25.2公里，其中1.4公里为高架线，其余为地下线，设车站21座；2015年12月动工，计划2020年建成。纺织城火车站即未来的西安东站，受西安东站站位未定等原因影响，纺织城火车站站不与其它站点同时开通。
*线路二期全长19.9公里，其中12.2公里为高架线，其余为地下线，设车站13座；2017年2月动工，计划2020年与一期同时建成。2018年，在建设过程中于秦皇大道东侧东马坊村西侧发现了楚汉争霸时代章邯封地首都废丘遗址，为保护历史遗迹，5号线二期线路进行一定程度偏移以避开遗址区，部分已建好高架段桥墩被拆除，但预计不影响2020年底通车的工期要求。

*6号线一期工程全长20.1公里，共设车站15座，全地下线；已於2016年3月28日开工，计划2020年建成开通试运营。<ref>[http://www.sn.xinhuanet.com/news1/2016-03/29/c_1118470646.htm 西安地铁六号线开工 计划2020年通车]新華網</ref>受西安南站站位未定等原因影响，南客站站不与其它站点同时开通。

=== 机场城际（机场西 - 北客站）===  
*区间：西起[[西安咸阳国际机场|西安咸阳国际机场]]的[[机场西|机场西]]，沿渭河北岸敷设，到达[[北客站|北客站]]，与[[西安地铁4号线|4号线]]同台换乘。一说与14号线（北客站-贺韶村）段贯通运营。 
*规划:  线路全长约为29.28公里，共设车站10座，设计时速100km/h，从北客站到机场运行时间将缩短至27分钟以内。
*进展：2014年7月，开始主体施工；计划于2019年建成通车。机场站（T5）受西安咸阳国际机场扩建影响，不与其它站点同时开通。
*备注：西安北至机场为城际轨道项目，向东与[[西安地铁14号线|地铁14号线]]贯通运营，终点预留向西延伸条件。

== 法律法规 ==
2011年5月25日，西安市第十四届人大常委会第三十次会议通过《西安市城市轨道交通条例》。该条例自2011年9月1日起施行。西安成为中国第一个“地铁未通，法规先行”的城市。<ref>{{Cite web |url= http://www.chinametro.net/Content/DisplayNews.aspx?ID=27489 |title= 西安：地铁未通法规先行 |author= 台建林 |date= 2012-05-15 |publisher= 《法制日报》 |language= zh-hans |accessdate= 2012-08-04 |quote=  }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2018年10月30日，西安市第十六届人大常委会第十五次会议通过新修订的《西安市城市轨道交通条例》。该条例自2019年3月1日起施行。西安市将加强地铁建设时的树木保护，并对一些违法行为设定了明确的处罚标准。

条例中规定地铁内禁止饮食。

== 重大事故及事件 ==
===施工时发生的事故===
2009年8月2日上午9时，西安地铁1号线8标段洒金桥车站附近建设工地发生严重塌方，两名施工人员不幸被埋。这两名被埋人员分别是，53岁[[四川|四川]]籍的甘祥宾和22岁的[[云南|云南]]籍的涂千军；两人分别在被埋三小时后救出现场，但终因抢救无效死亡。这是西安地铁开工建设以来首次出现人员伤亡的事故。<ref>
{{Cite web |url= http://zqb.cyol.com/content/2009-08/04/content_2786886.htm |title= 西安地铁施工塌方原因正在调查 “军令状”为何挡不住事故频发 |author= 孙海华 |date=  2009-08-04 　 |publisher= 《[[中国青年报|中国青年报]]》 |language= zh-hans |accessdate= 2012-08-05 |quote=  }}</ref>

2013年5月6日凌晨，西安地铁3号线通化门至胡家庙区间始发井左北隧道开挖至8米进深，突然出现隧道顶部塌方。塌方发生后，两名作业人员经营救已被送往医院救治，其中一人骨折一人轻伤，均无生命危险；有两名作业人员未受伤，<ref>{{Cite web |url= http://sx.sina.com.cn/news/b/2013-05-06/085064542.html |title= 西安地铁3号线施工塌方 4名工人被困正在营救 |author=  |date= 2013年5月6日 |publisher= 新浪陕西/西部网 |language= zh-hans |accessdate= 2013年5月6日 |quote=  }}</ref>汉中籍的王国生、杨全军、刘继红、周建军与西安籍的司斌五人虽经全力抢救但不幸遇难。<ref>{{Cite web |url= http://www.xametro.gov.cn/xwshow.asp?id=2211 |title= 地铁3号线一施工现场塌方 省市有关部门全力组织救人 |author=  |date= 2013-05-07 |publisher= 西安地铁官方网站 |language= zh-hans |accessdate= 2013年5月7日 |quote=  }}</ref><ref>{{Cite web |url= http://xian.qq.com/a/20130507/000362.htm |title= 地铁3号线塌方遇难人员名单公布 最小仅22岁 |author=  |date= 2013年5月7日 |publisher= 西安网/腾讯西安 |language= zh-hans |accessdate= 2013年5月7日 |quote=  }}</ref>

===3号线使用问题电缆事件===
{{main|西安地铁3号线使用问题电缆事件}}
2017年3月，一名自稱為[[奧凱電纜|奧凱電纜]]公司員工的網友在網路論壇發文，稱西安地鐵3號線因全線所用電纜的橫截面積小於標稱面積，而有過熱引發火災的危險。3月15日，涉事電纜的生產商奧凱電纜公司在網上發佈消息稱該網帖為造謠，公司方面已經報警。公司在西安地鐵建設招投標過程中無任何違法情況，提供的電纜經過質監局多次抽查均為合格產品<ref>{{cite news|title=质检总局“特急”通知各地彻查问题电缆|url=http://epaper.ynet.com/html/2017-03/24/content_243270.htm?div=-1|publisher=北京青年报|date=2017-03-24|accessdate=2017-03-24|deadurl=yes|archiveurl=https://web.archive.org/web/20170324092155/http://epaper.ynet.com/html/2017-03/24/content_243270.htm?div=-1|archivedate=2017年3月24日|df=}}</ref>。但經記者調查，奧凱電纜公司曾在2015年11月9日、2016年1月25日及2016年2月22日被西安市質監局以「生產銷售不符合國家標準的電力電纜，銷售偽造產品質量證明的電力電纜，以及銷售不符合國家標準的電力電纜」的理由處以行政處罰<ref name=thepaper1641151>{{cite news|title=西安地铁三号线电缆被指偷工减料，当地政府已成立联合调查组|url=http://www.thepaper.cn/newsDetail_forward_1641151|publisher=澎湃新聞|date=2017-03-17|accessdate=2017-03-22}}</ref>。

事发后，西安市人民政府表示已责成市监察局、质监局、安监局、公安局组成联合调查组立即开始调查核查复测检验，抽样送国家指定权威机构检测，检测结果将第一时间向社会公布<ref name=sina2>{{cite news|title=网传西安地铁电缆由小作坊生产 已成立调查组|url=http://finance.sina.com.cn/china/dfjj/2017-03-17/doc-ifycnpvh4744696.shtml|publisher=新华网|date=2017-03-17|accessdate=2017-03-24}}</ref>。3月20日，初步檢驗結果出爐：經過[[國家電線電纜監督檢驗中心|國家電線電纜監督檢驗中心]]檢測，5份樣品全部不合格。西安市方面表示，雖然不合格電纜不影響行車安全，但地鐵將在最短時間內對不合格電纜進行更換<ref name=thepaper1643706>{{cite news|title=西安：问题电缆不影响地铁3号线运行，涉事公司8人被控制|url=http://www.thepaper.cn/www/v3/jsp/newsDetail_forward_1643706|publisher=澎湃新聞|date=2017-03-20|accessdate=2017-03-22}}</ref>。同日中共西安市委常委、市纪委书记[[杨鑫|杨鑫]]表示，奧凱電纜公司的8名相關人員已經被警方控制，生產場所與帳目也都已經被封存<ref name=thepaper1643706/>。

===被曝工程承包受贿事件===
在使用问题电缆事件后的2017年4月，有地铁公司员工在网络举报，西安地铁资源开发有限公司负责人王志强涉工程承包受贿，违规将西安地铁十余个地铁出口和用地，承包给个人公司陕西合力拓科技工贸有限公司。随后西安地铁公司表示已将网贴举报内容向市纪委反映<ref>{{cite web|title=西安地铁又有人被举报受贿 官方:已向纪委反映|url=http://cnews.chinadaily.com.cn/2017-04/19/content_28996086.htm|publisher=环球网 |date=2017-04-19|accessdate=2017-04-19}}</ref>。
