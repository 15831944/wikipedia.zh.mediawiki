'''西新站'''（{{jpn|j=西新駅|hg=にしじんえき|rm=Nishijin eki|f=y}}）是一個位於[[日本|日本]][[福岡縣|福岡縣]][[福岡市|福岡市]][[早良區|早良區]]{{link-ja|西新|西新}}2丁目，屬於[[福岡市地下鐵|福岡市地下鐵]][[機場線_(福岡市地下鐵)|機場線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是K04。

== 歷史 ==
* 1981年（昭和56年）7月26日 - 福岡市地下鐵的西新站開業。

== 車站結構 ==
[[島式月台|島式月台]]1面2線的[[地底車站|地底車站]]。[[姪濱站|姪濱站]]方設有渡線，日間作為此站到發的[[箱崎線|箱崎線]]直通列車折返。

== 使用情況 ==
2016年（平成28年）度的1日平均'''上車'''人次為'''23,628人'''<ref>{{PDFlink|[https://subway.city.fukuoka.lg.jp/subway/about/pdf/subway_05.pdf 駅別乗車人員の推移（年度別1日平均）]}} - 福岡市交通局</ref>。直至2012年度為止次於[[天神站|天神站]]、[[博多站|博多站]]排行第3位，2013年度起被[[福岡機場站|福岡機場站]]超越跌至第4位。
