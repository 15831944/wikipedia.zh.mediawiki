'''西岸車站'''（{{jpn|j=西岸駅|hg=にしぎしえき|rm=Nishigishi eki|f=y}}）是一位於[[日本|日本]][[石川縣|石川縣]][[七尾市|七尾市]]，由地方[[第三部門|第三部門]][[鐵路業者|鐵路業者]][[能登鐵道|能登鐵道]]（{{lang|ja|のと鉄道}}）所使用的[[鐵路車站|鐵路車站]]。西岸車站被暱稱為小牧風車站。

== 車站構造 ==
[[對向式月台|對向式月台]]2面2線設有交會設備的[[地面車站|地面車站]]。站舍位於上行月台側。沒有跨線天橋，來往站舍與對向月台需跨過近七尾一側的站內平交道。此外，沒有設置[[自動售票機|自動售票機]]。

== 歷史 ==
* 1932年（[[昭和|昭和]]7年）8月27日 - 配合[[日本國鐵|日本國鐵]]七尾線[[能登中島車站|能登中島車站]] - [[穴水車站|穴水車站]]間延伸路段通車而啟用。
* 1960年（昭和35年）4月1日 - 裁撤貨運服務。
* 1972年（昭和47年）3月15日 - 不再派駐站員，成為無人車站。
* 1987年（昭和62年）4月1日 - 配合[[日本國鐵分割民營化|日本國鐵分割民營化]]，劃歸成為西日本旅客鐵道管轄的車站。
* 1991年（[[平成|平成]]3年）9月1日 - 七尾線[[和倉温泉車站|和倉温泉車站]] - [[輪島車站|輪島車站]]間的路段轉換經營權，成為能登鐵道的車站。

== 大眾文化 ==
[[File:YunosagiStation.JPG|thumb]]

2011年（[[平成|平成]]23年）4月播出的動畫《[[花開物語|花開物語]]》（{{lang|ja|花咲くいろは}}），動畫中虛構的「湯乃鷺站」取景自西岸站，製作公司[[PA_works|PA works]]與能登鐵道合作，於月台上設立「湯乃鷺站」站名牌，因此大受歡迎，成為[[聖地巡禮|聖地巡禮]]的熱門地點之一<ref>{{Cite web|date=|url=http://www.incl.ne.jp/ntr/index.html|title=新着情報|work=|author=のと鉄道公式サイト|publisher=incl|accessdate=2011-04-29|deadurl=yes|archiveurl=https://web.archive.org/web/20110504015048/http://www.incl.ne.jp/ntr/index.html|archivedate=2011-05-04}}</ref><ref>{{Cite web|date=2011-04-23|url=http://hanasakuiroha.seesaa.net/archives/20110423-1.html|title=リアル湯乃鷺駅だ!|work=|author=花咲くいろは公式ブログ|publisher=花咲くいろは公式サイト|accessdate=2011-04-29}}</ref><ref>{{Cite web|date=2011-04-28|url=http://hanasakuiroha.seesaa.net/article/198147127.html |title=GW花いろ情報|work=|author=花咲くいろは公式ブログ|publisher=花咲くいろは公式サイト|accessdate=2011-04-29}}</ref>。牌子於動畫結束半年內仍會存在<ref>{{Cite web|url=http://hanasakuiroha.seesaa.net/archives/201104-1.html|title=湯乃鷺駅誕生!|work=|author=花咲くいろは公式ブログ|publisher=|date=2011-04-30|accessdate=2011-08-02}}</ref>。
