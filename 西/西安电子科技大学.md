'''西安电子科技大学'''（{{lang-en|'''Xidian University'''}}<ref>{{Cite web |url = http://dag.xidian.edu.cn/info/1041/1880.htm |title = 梁继民：从“西电”到“XIDIAN”-西安电子科技大学-档案馆 |accessdate=2017-05-11 |work=dag.xidian.edu.cn }}</ref>）是[[中华人民共和国教育部|中华人民共和国教育部]]与[[工业和信息化部|工业和信息化部]]共建的一所[[中华人民共和国教育部直属高等学校列表|教育部直属]][[普通高等学校|普通高等学校]]，国家首批“[[211工程|211工程]]”和[[优势学科创新平台项目|优势学科创新平台项目]]重点建设高校，“[[111计划|111计划]]”重点建设高校，入选“[[2011计划|2011计划]]”，是中国最早的2所国防工业重点军校之一（[[哈军工|哈军工]]、'''西军电'''并称），是中国电子信息领域科学研究和人才培养的核心基地。

== 概况 ==
截止2017年8月，学校共有在校全日制研究生10000余人，其中硕士生9000余人，博士生近1700人，各类在校生30000余人，设有研究生院。设有通信工程学院等18个学院，2个国家一级重点学科（覆盖6个二级学科）、1个国家二级重点学科，34个省部级重点学科，27个博士学位授权点（覆盖了理学、工学、军事学3个学科门类），63个硕士学位授权点（覆盖了文学、法学、教育学、管理学、理学、工学、军事学7个学科门类），对本科专业的覆盖率已达95%以上。现有4个国家级重点实验室、6个教育部重点实验室和16个省部级重点实验室。建有7个博士后工作流动站，可在“通信与信息系统”等24个专业学科接收博士后研究人员。2012年教育部全国第三轮一级学科评估结果中，“信息与通信工程”学科全国排名第二，“电子科学与技术”学科全国排名第四。[http://mobile.xidian.edu.cn/mobile/gk1/xxjj.htm]

== 沿革 ==
;晋冀鲁豫军区通信学校
*1931年1月28日，[[红一方面军|红一方面军]]总司令[[朱德|朱德]]、总政治委员[[毛泽东|毛泽东]]签发命令，红一方面军总部立即创办“'''无线电训练班'''”。1931年2月，“无线电训练班”在[[江西省|江西省]][[宁都县|宁都县]]小布村正式成立并开课<ref name=aixin/>。
*1931年底，“无线电训练班”遵令更名“'''中央革命军事委员会无线电学校'''”，下设报务、机务两个专业<ref name=aixin>{{Cite web |url = http://www.tongyuanfoundation.com/tongyuanfoundation/vip_doc/477204.html### |title = 关于通信工程学院 |author =  |date =  |publisher = 南京通院爱心发展基金会 |language =  |accessdate = 2017-10-05 |quote =  }}</ref>。学校位于[[江西省|江西省]][[瑞金县|瑞金县]]，是[[中国共产党|中国共产党]]创建的第一所工程技术学校<ref name=lishi>{{Cite web |url = http://www.xidian.edu.cn/xxgk/hhls.htm |title =辉煌历史|author =  |date = 2017-06-09 |publisher =西安电子科技大学    |language =  |accessdate =  |quote =  }}</ref>。
*1933年6月，更名“'''中国工农红军通信学校'''”，并在报务、机务两个专业外，增设电话、司号旗语班<ref name=aixin/>。
*1934年8月，中国工农红军通信学校随红一方面军开始[[长征|长征]]。长征途中，学校被称作“'''中央红军通信教导大队'''”，但保留学校建制，并利用战斗间隙继续教学<ref name=aixin/>。
*1935年12月，长征到达[[陕北|陕北]]后的中国工农红军通信学校与陕北红军无线电报务训练班合并，更名“'''中央军委无线电通信学校'''”，位于[[陕西省|陕西省]][[延安|延安]]<ref name=lishi/><ref name=aixin/>。后来[[新疆|新疆]][[迪化|迪化]]的无线电训练班并入<ref name=aixin/>。
*1942年，更名“'''电信工程专门学校'''”<ref name=aixin/>。
*1945年[[日本投降|日本投降]]后，各[[解放区|解放区]]很快扩大，电信工程专门学校派出大量人员赴前线工作，学校规模随之缩小。留校教员、学员在校长[[吴泽光|吴泽光]]率领下，随[[中共中央|中共中央]]、军委三局迁至[[河北省|河北省]]，1945年10月在[[河北省|河北省]][[武安县|武安县]]创建“'''[[晋冀鲁豫军区|晋冀鲁豫军区]]通信学校'''”，直属中央军委<ref name=aixin/><ref name=lishi/>。

;晋冀鲁豫军区电讯工程专科学校
*1947年，创建'''晋冀鲁豫军区电讯工程专科学校'''，驻[[河北省|河北省]][[阜平县|阜平县]]<ref name=lishi/>。

;中央军委三局电讯队
*1948年，创建'''中央军委三局电讯队'''，驻[[河北省|河北省]][[平山县|平山县]]<ref name=lishi/>。

;中央军委气象队
*1944年，创建'''中央军委气象队'''，后驻[[河北省|河北省]][[平山县|平山县]]<ref name=lishi/>。

;晋察冀军区电讯工程专科学校
*1941年3月，成立晋察冀军区司令部三科（即通信科）材料股无线电研究组。到1941年底，各分区调至无线电研究组的人员陆续到达，共11人：组长张宾，副组长林爽，成员有刘兆生、常家明、潘家晋、张中甫、赵干青、史铁夫、刘坚、崔青吉、雷振武等9人<ref name=jinchaji>{{Cite web |url = http://dag.xidian.edu.cn/info/1040/1794_3.htm |title =第三部分 晋察冀军区电讯工程专科学校时期 忆晋察冀军区无线电研究组  |author = 林爽 |date = 2015-05-11 |publisher = 西安电子科技大学档案馆 |language =  |accessdate =  |quote =  }}</ref>。
*1941年秋季反“扫荡”结束后，[[太平洋战争|太平洋战争]]爆发。部分国际友人在[[中国共产党|中国共产党]]帮助下逃离日占区[[北平|北平]]、[[天津|天津]]抵达[[晋察冀边区|晋察冀边区]]，其中有[[燕京大学|燕京大学]]物理系主任[[班威廉|班威廉]]教授及夫人，经济系讲师[[林迈可|林迈可]]及夫人。经[[聂荣臻|聂荣臻]]司令员恳留，班威廉、林迈可成为无线电研究组的导师<ref name=jinchaji/>。
*1942年3月，除研究组原全体成员11人外，还增加了陪班威廉夫妇、林迈可夫妇而从平西到晋察冀的王士光，以及晋察冀军区无线电训练班教务主任韩克树，这时研究组共13人<ref name=jinchaji/>。
*1943年春，晋察鲁豫又派5人参加学习：韩黎、吴本毅、吴立中、张芝荪、刘一青。另有金鑫来的时间比上述5人略早。此时研究组正式改称高级班<ref name=jinchaji/>。
*1943年反“扫荡”后期，在班威廉夫妇要求下，聂荣臻司令员派部队护送他们赴延安。后他们经[[重庆|重庆]]离开中国<ref name=jinchaji/>。
*1944年4月，无线电工程课讲完，林迈可回国。此后研究组成员分到各地区、各部门工作。1946年从[[张家口|张家口]]撤退时，一批高级知识分子从[[国统区|国统区]]到[[解放区|解放区]]参加革命，其中无线电工程及通讯人才先后调到无线电训练大队工作<ref name=jinchaji/>。
*1947年，将无线电训练大队改为'''晋察冀军区电讯工程专科学校'''<ref name=jinchaji/>。

;中国人民解放军军事电信工程学院
*随着[[第二次国共内战|第二次国共内战]]发展，[[晋察冀边区|晋察冀边区]]和[[晋冀鲁豫边区|晋冀鲁豫边区]]连成一片，[[华北人民政府|华北人民政府]]和[[华北军区|华北军区]]成立。1948年，中共中央决定，将晋冀鲁豫军区通信学校、晋冀鲁豫军区电讯工程专科学校、晋察冀军区电讯工程专科学校合并为“'''华北军区电讯工程专科学校'''”，任命军委三局局长[[王诤|王诤]]兼任校长和政委，学校仍直属中央军委<ref name=aixin/>。
* 1948年5月，晋冀鲁豫军区通信学校、晋冀鲁豫军区电讯工程专科学校、晋察冀军区电讯工程专科学校、中央军委气象队、中央军委三局电讯队合并组建华北军区电讯工程专科学校。校址位于[[河北省|河北省]][[获鹿县|获鹿县]]<ref name=qianzhi/><ref name=lishi/>。
* 1949年，[[中央工委青年训练班|中央工委青年训练班]]并入。
* 1949年3月31日，中央军委通令：“拟即举办一所机要通信干部学校，……并附设高级研究机构”。军委副主席[[周恩来|周恩来]]亲自指定[[第四野战军|第四野战军]]副参谋长[[曹祥仁|曹祥仁]]任校长，同时周恩来接见曹祥仁并做如下指示：第一，这所学校规模要大，要以华北电讯工程专科学校为基础。第二，立即选择新校址，中央大力支持。第三，以中央军委名义，你们派人到已解放的各大城市招生。同时新学校定名为“中国人民革命军事委员会工程学校”（简称“军委工校”），直属中央军委领导，曹祥仁任校长兼政委<ref name=qianzhi>{{Cite web |url =http://dag.xidian.edu.cn/info/1042/1571.htm  |title =  我校迁址西安的前前后后|author = 刘嘉相 |date = 2015-04-30  |publisher =西安电子科技大学-档案馆  |language =  |accessdate =  |quote =  }}</ref>。
* 1949年10月，根据[[毛泽东|毛泽东]]建议，中央军委命令，以华北电讯工程专科学校为基础，改建'''[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]工程学校'''（通称'''张家口军委工程学校'''），校址位于河北省[[张家口市|张家口市]]<ref name=lishi/><ref name=aixin/>。1949年11月，中央人民政府人民革命军事委员会工程学校建成。该校又称“'''中央军委机要通信干部学校'''”，简称“军委工校”。军委工校第一部便是通信工程专业部<ref name=aixin/>。1949年12月27日，正式举行开学典礼<ref name=qianzhi/>。
* 中央人民政府革命军事委员会工程学校原中央工委青年训练班部分再次分出，建立[[南京机要学校|南京机要学校]]。
* 1952年，[[大连工学院|大连工学院]]电讯系、[[西南军区通信学校|西南军区通信学校]]工程系并入。
* 1952年，[[一五計畫_(中國)|中国第一个五年计划]]开始实行，全国高校进行调整<ref name=aixin/>。1952年5月，中央军委决定，将军委工校第一部扩建为'''中国人民解放军通信工程学院'''<ref name=lishi/><ref name=aixin/>。1952年6月1日，中国人民解放军通信工程学院正式成立<ref name=aixin/>。
* 1953年，院长[[王诤|王诤]]得知[[北京农业大学|北京农业大学]]要扩大，准备外迁。1953年12月3日，王诤向代总参谋长[[聂荣臻|聂荣臻]]提交报告，要求将该院从张家口迁到北京罗道庄的北京农业大学校址。1954年3月15日，聂荣臻致函[[北京市人民政府|北京市人民政府]]，要求将北京农业大学旧校址拨给该院，北京市市长[[彭真|彭真]]批示同意。1954年4月15日，[[中央人民政府高等教育部|中央人民政府高等教育部]]批复同意该院建在罗道庄，不久总参谋长[[黄克诚|黄克诚]]也批复同意该院迁到罗道庄。但随后上级决定该院不迁北京而迁[[陕西省|陕西省]][[西安市|西安市]]<ref name=qianzhi/>。
* 1955年1月，更名'''中国人民解放军军事通信工程学院'''<ref name=lishi/>。
* 1955年8月，更名'''中国人民解放军通信学院'''<ref name=lishi/>。
* 1956年3月31日，[[中国人民解放军总参谋部|中国人民解放军总参谋部]]批准建立“中国人民解放军通信兵学院西安办事处”，立刻投入在西安市筹建中国人民解放军通信兵学院的工作<ref name=qianzhi/>。
* 1958年1月，中国人民解放军通信学院更名'''中国人民解放军通信兵学院'''<ref name=lishi/>。1958年暑假，从河北省张家口市迁往陕西省西安市。1958年9月1日，正式在西安市的新校址开学<ref name=qianzhi/>。
* 1959年，被列为[[中国全国重点大学|全国重点大学]]之一。
* 1960年1月，更名'''中国人民解放军军事电信工程学院'''（由此开始被称为'''西军电'''），校址仍位于[[陕西省|陕西省]][[西安市|西安市]]<ref name=sanqinyou>{{Cite web |url =http://www.sanqinyou.com/shaanxi/info/138615015872915_4.html   |title =  建国后的陕西部队（4）|author =  |date =2013-08-06  |publisher = 三秦游网 |language =  |accessdate =  |quote =  }}</ref><ref name=lishi/><ref name=aixin/>。
