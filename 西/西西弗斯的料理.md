'''西西弗斯的料理'''，2018年奇幻美食类题材搜狐自制网络剧。该剧由[[搜狐视频|搜狐视频]]、[[北京新光影业|北京新光影业]]出品，导演[[大鬼|大鬼]]执导，[[陈若轩|陈若轩]]、[[李梦颖|李梦颖]]联袂主演。2016年10月21日于苏州开机<ref>{{Cite web |url=https://www.yunchou.com/project/5194 |title=搜狐视频自制网剧《西西弗斯的料理》|accessdate=2017-12-03 |author= |date= |work= |publisher=云筹|language=Zh-hans }}</ref>，同年12月5日杀青<ref>{{Cite web |url=http://yule.sohu.com/20161205/n474971243.shtml |title=《西西弗斯的料理》杀青 李梦颖演绎萌系铲屎官|accessdate=2017-12-03 |author= |date=2016-12-05 |work=李治政 UK005 |publisher=搜狐娱乐|language=Zh-hans }}</ref>。原计划2017年于搜狐视频独家播出<ref>{{Cite web |url=http://yule.sohu.com/20161108/n472618644.shtml |title=搜狐视频2017新战略：新“视界观” 造“全娱乐”|accessdate=2017-12-03 |author= |date=2016-11-08 |work=李治政 UK005 |publisher=搜狐娱乐|language=Zh-hans }}</ref>，但截至2017年12月也未播出，亦无定档的消息。
