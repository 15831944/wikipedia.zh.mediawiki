'''西城都荟'''（{{lang|en|Metropolitan Plaza}}），位於[[中华人民共和国|中国]][[广州市|广州市]][[荔湾区|荔湾区]][[黄沙大道|黄沙大道]]8号，是一个大型综合购物商场，由[[和记黄埔地产|和记黄埔地产]]（广州荔湾）有限公司发展。

== 简介 ==
西城都荟佔地约71,000平方米，总建筑面积88,726[[平方米|平方米]]，商场共4层（地下1层和地上3层），毗邻[[广州地铁|广州地铁]][[广州地铁1号线|1号线]]及[[广州地铁6号线|6号线]][[黄沙站|黄沙站]]、[[上下九步行街|上下九步行街]]和[[沙面岛|沙面岛]]，商场上方是住宅项目逸翠湾，由建筑公司许李严设计（亚洲）有限公司设计，和记黄埔地产（广州荔湾）有限公司发展，该公司是[[香港|香港]]和记黄埔地产有限公司与[[长江实业|长江实业（中国）有限公司]]联营公司之下属公司。

香港[[娛藝|UA院線]]已在西城都荟开設其在廣州的第一间分店。UA影城约占地5500平方米，有6间戏院，共提供950个豪华座位，并将引入在香港的Cityline网上购票服务。<ref>{{cite news|url=http://paper.wenweipo.com/2006/07/06/ME0607060008.htm|title=西城都薈引入UA院線|publisher=文汇报|date=2006-07-06|accessdate=2011-04-07}}</ref>(註:以開業日期計，[[KK_Mall|KK Mall]]的UA影城才是中國大陸第一間分店)

== 历史 ==
西城都荟是广州地铁开始兴建时，[[广州市人民政府|广州市人民政府]]在香港招商的首批地铁上盖综合商住项目。有报道指，和黄买下该[[地皮|地皮]]已经超过11年。2004年传出动工消息，公司宣布住宅与商业部分均可能在2007年落成。<ref>{{cite news|url=http://www.topbiz360.com/html/newscenter/businessproperty/20080711/32971.html|title=广州：太古汇、西城都荟今年工期仍有延迟可能 |publisher=第一财经日报|date=2008-07-11|accessdate=2011-04-07}}</ref>

2006年9月，西城都荟宣布引入[[东南亚|东南亚]]零售巨头力宝集团旗下的[[乐宾百货|乐宾百货]]，原计划在2007年下半年开业。但由于地基施工缓慢无法交付使用物业，乐宾百货开业时间先是延期两次后，双方最后解除合作。 <ref>{{cite news|url=http://bb.focus.cn/news/2008-09-09/529712.html|title=西城都荟密洽新主力 百盛有望重返广州市场 |publisher=第一财经日报|date=2008-09-09 |accessdate=2011-04-07}}</ref>商場其後引入摩登百貨，開設摩登新天地，但其後已結業，原址被拆成多間商舖<ref name="zyz"/>。

和黃2013年8月29日發公告，旗下Viewin擬以13.92億元，出售Rhine Rise股份，另以約2.42億元作價轉讓Rhine Rise貸款；長實旗下長江中國物業，則以16.34億元出售BEL股份。Rhine Rise及BEL各持有雅和50%股權，而雅和持有廣州西城都薈廣場及位於其地庫一層與二層之所有停車場、老年活動中心。和黃預期，交易將帶來約4.9億元收益。西城都薈廣場的買家是獨立第三方離岸公司GCREF Acquisitions 22 Limited。<ref>{{cite news|url=http://finance.sina.com.hk/news/-40-6123556/1.html|title=長和系賣廣州商場套30億|publisher=明報|date=2013-08-30}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

2017年4月9日，[[領展房地產投資信託基金|領展房地產投資信託基金]]以人民幣40.65億元向[[基匯資本|基匯資本]]和摩根士丹利房地產基金組成的財團收購西城都薈。西城都薈當時的物業租用率約為94.1%。<ref>{{cite news|url=http://hk.apple.nextmedia.com/financeestate/art/20170410/19985354|title=領展斥46億購廣州商場|publisher=蘋果日報|date=2017-04-10|accessdate=2017-04-14}}</ref>

== 商場結構 ==
商場樓高四層，由於商場設計時打算引入百貨公司，因此商場分為東、西兩部分，其中地庫被[[廣州地鐵1號綫|廣州地鐵1號綫]][[黃沙站|黃沙站]]站廳相隔<ref name="zyz">追影族 - [http://jpoon9394.blogspot.com/2017/04/gz-metropolitanplaza.html 西城都薈]</ref>。商場地面設有露天廣場。

== 交通 ==
西城都薈位於[[廣州地鐵|廣州地鐵]][[廣州地鐵1號綫|1號綫]][[廣州地鐵6號綫|6號綫]][[黃沙站|黃沙站]]上蓋，商場亦附設一個公交總站。
