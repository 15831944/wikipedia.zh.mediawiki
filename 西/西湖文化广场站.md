'''西湖文化广场站'''是[[杭州地铁1号线|杭州地铁1号线]]设立的一个站点，以及未来与[[杭州地铁3号线|3号线]]和[[杭州地铁中轴快线|中轴快线]]的换乘站，位于[[西湖文化广场|西湖文化广场]]（核心商业区）东北侧，[[中山北路|中山北路]]地下，北至文晖路路口、南至朝晖路以北180米处，北接[[打铁关站|打铁关站]]、南接[[武林广场站|武林广场站]]。

==历史==
*2009年4月10日，车站开工<ref>{{cite web |title=不知道中山北路封道许多车主昨天“被迫”绕路|url=http://www.hangzhou.com.cn/20090413/ca1692853.htm|website=杭州网}}</ref>
*2012年11月24日，车站随1号线一起开通。

== 车站构造 ==
{{multiple images|width=200|align=right|direction=vertical
|image1=West Lake Cultural Square Station, 2014-10-12 02.JPG|caption1=车站站厅
|image2=20160522西湖文化广场站台与扶梯.jpg|caption2=地下三层站台与扶梯
}}
本站是地铁1号线的控制站点之一，为地下三层上下平行双岛式'''换乘车站'''（由于3号线尚未启用，因此1号线区域和3号线区域暂使用隔墙分割，故西湖文化广场站虽然实际为双层岛式站台，但目前可视作双层侧式站台），站台宽13米，全长272.08米，标准段宽21.8米～24.3米，埋深23.8米～24.4米。中轴快线为地下五层岛式站台。

==未来发展==
[[杭州地铁中轴快线|杭州地铁中轴快线]]将经过本站（工程名杭州地铁3号线一期工程潮王路站～西湖文化广场站区间地下连接工程）。中轴快线西湖文化广场站位于文晖路与中山北路交叉口东侧，沿文晖路东西向布置，与既有1、3号线车站采用通道换乘，采用明挖顺做法（地下1~3层）+盖挖法（地下4/5层）施工。为地下五层双柱三跨箱型框架结构，主体长171.1m（内衬墙外包），标准段宽度24.3m，基坑深度约33m。<ref>{{cite web |title=杭州地铁3号线一期工程潮王路站～西湖文化广场站区间地下连接工程土建施工潮王路站-西湖文化广场站区间地下连接工程土建施工E3300000001000701083001招标文件公示 |url=new.zmctc.com/zjgcjy/infodetail/?infoid=af535f15-2b4f-49c9-8d91-c2bd738abd01 |website=浙江省公共资源交易中心电子招投标交易平台}}</ref>
