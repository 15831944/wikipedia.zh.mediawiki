'''西昌学院'''（{{lang-en|'''Xichang College'''}}），位于[[中国|中国]][[四川省|四川省]][[西昌|西昌]]市，是一所综合性省属普通高等本科院校。学校的前身是1939年成立的国立西康技艺专科学校，后更名为西昌农业专科学校，2003年经教育部批准与西昌师范高等专科学校、凉山大学、凉山教育学院合并组建成西昌学院。

== 历史沿革 ==
1939年由北洋工学院内迁西昌创建的国立西康技艺专科学校，1978年更名为西昌农业专科学校，并于2003年与西昌师范高等专科学校，凉山大学、凉山教育学院合并，改名为西昌学院。<ref>{{cite web|title=西昌学院历史|url=http://www.xcc.sc.cn/xcxy/xygk/lsyg/index.html|publisher=西昌学院}}</ref>

== 现状 ==
学院位于旅游城市西昌市，占地1636亩，分东、南、北三大校区（东校区已经没有设置专业）。校舍建筑面积38万平方米，固定资产总值7.8亿元，教学科研仪器设备总值2.14亿元。图书馆总面积3.3万平方米，图书203万册，电子图书120万册。学校现有教职工970人，专任教师791人，教授116人、副教授245人；硕士、博士研究生550人。学校设置16个二级学院和1个思想政治理论课教学部，开设64个本科专业和15个专科专业，涵盖了农学、工学、管理学、理学、教育学、文学、艺术学、经济学、法学等九大学科门类；面向全国19个省、自治区、直辖市招生，现有全日制普通在校生17100余人，2015年学校被列为四川省首批整体转型试点学校。<ref>{{cite web|title=四川省4所高校试点本科院校整体转型|url=http://sichuan.scol.com.cn/dwzw/201512/54178112.html|publisher=四川在线}}</ref><ref>{{cite web|title=西昌学院现状|url=http://www.xcc.sc.cn/xcxy/xygk/xqzl/index.html|publisher=西昌学院}}</ref>

== 校区介绍 ==
'''北校区'''
:原西昌农业高等专科学校校区，位于安宁镇马坪坝村。校本部，位置比较偏僻。
'''南校区'''
:原西昌师范高等专科学校校区，位于海滨中路24号，面临邛海，背靠泸山森林公园，风景最好。
'''东校区'''
:原凉山大学校区，现已拆迁，目前未设置专业。
