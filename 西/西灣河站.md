'''西灣河站'''（{{lang-en|'''Sai Wan Ho Station'''}}）是[[港鐵|港鐵]][[港島綫|港島綫]]的一個[[鐵路車站|鐵路車站]]，位於[[香港|香港]][[東區_(香港)|東區]][[西灣河|西灣河]][[筲箕灣道|筲箕灣道]]地底，車站於1985年5月31日啟用，車站大堂由金門/Kier/Lilley聯營公司負責興建，車站隧道由[[西松建設|西松建設]]負責興建。

==車站設計==
西灣河站為一個複雜的建築，有5層地庫，上方為住宅、文娛中心、體育館、街市及熟食中心，於1984-1991年分期落成啓用。西灣河站共分5層，大堂層為車站控制室、員工設施及商店；下方有機房及月台各兩層，並有結構為上方住宅及文娛中心提供支承。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Sai Wan Ho Station 2019 01 part4.jpg | caption1 = 大堂（收費區內）
| image2 = Sai Wan Ho Station 2019 01 part3.jpg | caption2 = 大堂（收費區外）
| image3 = Sai Wan Ho Station Escalator 2014.jpg | caption3 = 由大堂至月台的扶手電梯
}}

西灣河站大堂位於[[西灣河文娛中心|西灣河文娛中心]]的地底。大堂內設有[[升降機|升降機]]連接大堂及月台，並有10條[[電動扶梯|扶手電梯]]連接出入口、大堂及月台。為節約能源，該10條扶手電梯中會有5條於非繁忙時間期間暫停使用。

西灣河站的車站商店數目不多，祇有4間商店，包括[[便利商店|便利店]]、[[麵包糕餅店|麵包糕餅店]]等。此外，該站設有[[自動櫃員機|自動櫃員機]]讓乘客提款，並有[[自動售賣機|自動售賣機]]供乘客購買飲料<ref name="shops"/>。

==利用狀況==
西灣河站鄰近多個政府部門、[[東區法院大樓|東區法院大樓]]、[[死因裁判法庭|死因裁判法庭]]、屋苑、[[逸濤灣|逸濤灣]]、[[嘉亨灣|嘉亨灣]]、[[香港電影資料館|香港電影資料館]]、著名食街及海天廣場等。尤其是本站鄰近[[蘇豪東|蘇豪東]]，有多達20多家食店。因此，每天有不少乘客進出此站。

==歷史==
===選址===
在1970年代設計港島綫的走線時，西灣河站原本定線在較近山腰的[[西灣河街|西灣河街]]地底，不過最終定位在筲箕灣道現時的位置，以配合1980年代初東區的大型填海工程。車站於1985年5月31日啟用。
===翻新月台===
2008年，此站月台牆壁下的[[紙皮石|紙皮石]]進行了重鋪工程。
===加裝升降機===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Sai Wan Ho Station 2017 part5.jpg | caption1 = 地面升降機
}}
為方便有需要的人士進出車站，港鐵公司於西灣河站近閘內升降機，安裝一部閘外[[升降機|升降機]]連接大堂至聖十字徑，工程已於2011年11月展開，於2015年12月14日落成啟用<ref>{{cite web|url=http://www.mtr.com.hk/archive/corporate/en/press_release/PR-15-113-C.pdf | title=港鐵西灣河站新站外升降機令出入更方便 |publisher=[[港鐵|港鐵]] |date=2015年12月14日}}{{zh-hant}}</ref>。
