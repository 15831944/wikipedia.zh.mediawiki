'''西林寺'''又名'''西琳寺'''，为位于[[中国|中国]][[江西省|江西省]][[九江市|九江市]][[庐山|庐山]]西北麓、[[东林寺|东林寺]]之西数百步的一座佛寺，创建于东晋[[太元_(东晋)|太元]]二年（377），由太府卿陶范为[[慧永|慧永]]禅师所立，宋太宗赐额太平兴国乾明禅寺，为庐山北山第一寺，也是庐山三大名寺（西林寺、东林寺、[[大林寺|大林寺]]）之一。其建筑屡毁屡建，今除千佛塔外均为近年重建，中轴线上依次为三间四柱牌坊、山门三间、天王殿五间、大雄宝殿五间与千佛塔。

'''千佛塔'''又称'''砖浮屠'''，为唐玄宗敕建，明崇祯四年（1631）重修<ref>民国·吴宗慈《庐山志·卷二·山川胜迹》：“东林寺西为西林寺，寺在白鹤乡庐山麓，晋太元二年太府卿陶范为慧永创建第一寺，初，敕赐太平兴国乾明禅寺。隋开皇间重修，唐宋尤盛，元壬辰兵燬。明洪武十四年复建，崇祯四年王鸣玉僧一如重修千佛宝塔。清顺治十一年古嶷开法西林，遂为禅寺，乾隆戊戌年寺圮，僧际远捐修佛像一新，咸丰四年燬，十一年僧麓松主席此地，重修殿宇，塑新佛像，丛林复兴。（同治化志）砖浮屠（即千佛塔）在西林寺后，唐元宗敕建，明给谏王鸣玉重修，自为记，后被火中空，外状崔巍如故。（查慎行游记、同治化志）。”</ref>，2005年-2006年采用深层沉井排土纠倾法进行纠倾维修，并修复塔坪、塔心室楼板与楼梯等<ref>{{cite web |url=http://www.chinanews.com/news/2006/2006-03-20/8/705119.shtml|title=《具有1100多年历史的庐山千佛塔纠倾维修工程结束》|accessdate=2017-08-31|work=|publisher=中国新闻网}}</ref><ref>{{cite web |url=http://www.gmw.cn/01gmrb/2006-02/24/content_379044.htm|title=《江西九江抢救千年古塔》|accessdate=2017-08-31|work=|publisher=光明日报}}</ref>。今塔为六角七层的[[楼阁式塔|楼阁式]]砖塔，高约46米，各层南北相对辟门，其余各面辟佛龛，南面每层门上皆有题额，自下而上分别为千佛塔、羽室才、金刚幢、灵就来、无上法、聪雨花和无明藏<ref>{{cite web |url=http://www.jiujiang.gov.cn/zjjj/sjjj/zlwk/jjsz/201506/t20150624_1304677.htm|title=《九江市志·风景名胜卷》|accessdate=2017-08-30|work=|publisher=九江市政府}}</ref>。
