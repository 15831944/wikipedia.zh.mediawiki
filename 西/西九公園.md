'''西九公園'''（[[英文|英文]]：'''West Kowloon Park'''）是[[香港|香港]]一座正在興建的[[海濱公園|海濱公園]]，位於[[九龍|九龍]][[西九文化區|西九文化區]]西南部（前身為[[西九龍海濱長廊|西九龍海濱長廊]]），佔地23[[公頃|公頃]]，工程於2016年7月啟動，部份公園及海濱長廊已於 2017 年 12月起陸續對外開放，其餘部份亦將於2018年年中竣工。

== 願景 ==
「締造令[[香港|香港]]引以為傲的優質[[綠化|綠化]]戶外空間。」

== 使命 ==
「成為啟發、推動和鼓勵文化活動的[[海濱公園|海濱公園]]。」

== 設計 ==
西九公園的設計概念是建基於[[西九文化區|西九文化區]]「[[城市中的公園|城市中的公園]]」之主題，至2013年，[[監察西九文化區計劃推行情況聯合小組委員會|監察西九文化區計劃推行情況聯合小組委員會]]提議將西九公園當中的園境設計簡約化，理由為西九文化區之用地為[[填海|填海]]得來，某些品種的樹木未必適合在該處栽種，加上西九公園位處多風且無屏蔽地帶，日後的園林護理成本將相當高昂，因此於2014年公布中的設計方案，刪除了種植5,000棟樹木之原設計，更改為大型草坪<ref>[http://www.ol.mingpao.com/cfm/style5.cfm?File=20140725/sta34/vsb1.txt 西九聞問切﹕西九公園樹林不見了] 《明報》 2014年7月25日</ref>。

西九公園由[[劉榮廣|劉榮廣]][[伍振民|伍振民]]建築師事務所（香港）有限公司帶領[[園境師|園境師]]West 8（荷蘭）及傲林國際（ACLA）（香港）所組成的團隊負責進行概念設計。<ref name="wk140722">[http://www.westkowloon.hk/tc/newsroom/news/west-kowloon-cultural-district-authority-unveils-concept-design-for-the-park/ 西九文化區管理局公布公園概念設計]，西九文化區管理局，2014年7月22日</ref>各場館以至海濱長廊均由一條（暫定名稱）文化大道（Cultural Boulevard）串連，並且與貫穿西九文化區心臟地帶的[[林蔭大道|林蔭大道]]連接。

== 設施 ==
西九公園為戶外音樂、舞蹈、劇場表演、藝術展覽以及其他文化活動提供演出場地，由[[M+|M+]]所策劃的大型戶外雕塑及裝置展覽亦融會其中。

*小型藝術展館（Arts Pavilion）：位於西九公園東北部的高地上，從該處可以欣賞[[維多利亞港|維多利亞港]]海景，鄰近[[M+博物館|M+博物館]]及[[藝術廣場|藝術廣場]]。
*自由空間：位於文化大道兩旁
**黑盒劇場（Black Box）：其中黑盒劇場處於中央，可以設置站位或者座位，最多容納900人。
**戶外舞台 （Outdoor Stage）
*草坪：
**中型草坪：位於西九公園南部鄰近海邊，能夠最多容納5,000人的站位。
**大型草坪：位於西九公園西面，為多用途活動場地，包括舉辦戶外節目、[[古典音樂|古典音樂]]及[[流行音樂|流行音樂]]會，能夠容納逾10,000人的站位。
*海濱活動廣場
*[[海濱長廊|海濱長廊]]：採用雙層設計，能夠從多角度飽覽[[維多利亞港|維多利亞港]]的景色。
*[[香港單車徑|單車徑]]：可以通往西九公園各處<ref>[http://std.stheadline.com/yesterday/loc/0719ao05.html 西九公園南海濱擬雙層設計] 《星島日報》 2014年7月19日</ref>。
*康樂、休憩及其他公共設施
**自助單車租賃服務系統
*零售及餐飲設施
*船隻靠泊設施

==管理附例==
為了既鼓勵公眾在共享空間內進行各式各樣的活動，又尊重和平衡不同使用者的需要。[[西九文化區管理局|西九文化區管理局]]委託[[香港大學|香港大學]]文化政策研究中心總監許焯權教授及獨立研究機構「政策二十一」於2014年5月9日至6月29日舉辦「西九公園，人人共享──如何管理將來的西九公園」公眾意見調查，透過網上問卷、7場專題小組論壇及分析從[[自由野|自由野]]及[[大角咀廟會|大角咀廟會]]兩個活動中所收集得來的意見進行，祈求就如何以創新模式營運和管理西九公園以符合公眾期望，收集公眾的意見和想法<ref>[http://www.westkowloon.hk/tc/the-district/architecture-facilities/park/chapter/public-survey/ 西九公園    人人共享] {{Wayback|url=http://www.westkowloon.hk/tc/the-district/architecture-facilities/park/chapter/public-survey/ |date=20141219042713 }}</ref>。同年8月12日，西九文化區管理局匯報初步調查結果，共接收3,302份問卷，發現公眾普遍期管理模式偏向彈性。

有別於香港一般公共公園及花園，西九公園直接由[[西九文化區管理局|西九文化區管理局]]管理，而非[[康樂文化事務署|康樂文化事務署]]。[[西九文化區管理局|西九文化區管理局]]委任了顧問探討[[英國|英國]]、[[加拿大|加拿大]]及[[澳大利亞|澳大利亞]]等與[[香港|香港]]同樣實施[[普通法|普通法]]的國家之公園附例，並且參考了[[香港海洋公園|香港海洋公園]]及康樂文化事務署之公園附例。

初步建議以[[康樂文化事務署|康樂文化事務署]]的模式管理西九公園，以《西九文化區(公眾休憩用地)附例》禁止放[[風箏|風箏]]、[[熱氣球|熱氣球]]及[[模型飛機|模型飛機]]等可能為公眾帶來危險的行為<ref>[http://std.stheadline.com/yesterday/loc/0915ao06.html 西九公園禁放風箏模型機] 《星島日報》 2014年9月15日</ref>，詳情於2015年年初提交予[[立法會|立法會]]審議。唯最後2016年5月22日所議決的條例比康文署公園更嚴格。包括舉行活動要預先獲局方書面批准，禁止「派發或張貼任何招貼、標語牌或告示」，局方更有權用武力「移走」使用者。藝術家批評有關條例阻礙文化發展，與西九當年「人人共享」的目的背道而馳。<ref>[http://www.hk01.com/%E7%A4%BE%E5%8D%80/26286/%E6%96%99%E8%A5%BF%E4%B9%9D%E5%85%AC%E5%85%B1%E7%A9%BA%E9%96%93%E6%A2%9D%E4%BE%8B%E8%BC%95%E6%98%93%E7%8D%B2%E6%89%B9-%E9%99%B3%E6%B7%91%E8%8E%8A-%E5%94%94%E6%98%8E%E7%99%BD%E9%BB%9E%E8%A7%A3%E5%92%81%E8%B6%95%E9%80%9A%E9%81%8E 料西九公共空間條例輕易獲批　陳淑莊︰唔明白點解咁趕通過 香港01 2016-06-15]</ref>

== 歷史 ==
[[File:Mobile_M+_Inflation_temporary_Sculpture_Park_201305.jpg|thumb]]當代戶外雕塑展覽。]]

為了未來西九文化區內的主要場地的落成作好準備，並且鼓勵公眾參與藝術活動，西九文化區管理局於2012年10月接收[[西九龍海濱長廊|西九龍海濱長廊]]後，至2014年在未動工的西九公園選址多次舉辦大型公眾活動及節日慶典，包括[[西九大戲棚|西九大戲棚]]、[[M+進行：充氣！|M+進行：充氣！]]、[[Clockenflap|Clockenflap]]香港音樂及藝術節、BloHK Party及香港最大型戶外藝術節之一的[[自由野|自由野]]，合共逾30次活動，至2014年7月吸引逾66萬名訪客參與。2014年4月，西九文化區管理局亦在此處引入香港首個自助單車租賃服務系統──[[悠遊西九|悠遊西九]]。　

2014年2月，[[西九文化區管理局|西九文化區管理局]]諉托[[劉榮廣|劉榮廣]][[伍振民|伍振民]]建築師事務所、園境師 West 8（[[荷蘭|荷蘭]]）及傲林國際（ACLA）（[[香港|香港]]）支援的設計團隊，負責西九公園的設計及管理建造工程<ref>[http://www.arts-news.net/articles/2014/08/13/%E8%8D%89%E5%9C%B0%E3%80%81%E8%97%9D%E8%A1%93%E3%80%81%E8%B8%8F%E5%96%AE%E8%BB%8A%EF%BC%9A%E5%85%AC%E7%9C%BE%E6%9C%9F%E6%9C%9B%E8%A5%BF%E4%B9%9D%E5%85%AC%E5%9C%92%E6%9B%B4%E8%87%AA%E7%94%B1 草地、藝術、踏單車：公眾期望西九公園更自由]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }} 藝術推廣新聞頻道 2014年8月13日</ref>。同年7月22日，西九文化區管理局連同設計團隊公布西九公園的設計概念，以確定多個位於西九公園內的文化藝術場地的位置。團隊表示，西九公園將為公眾提供一隅寧謐的空間，在[[市中心|市中心]]開創全新的[[綠化帶|綠化帶]]<ref>[http://www.mingpaocanada.com/Van/htm/News/20140723/HK-gzc1_r.htm 西九公園 有大草地可搞騷] 《明報》 2014年7月23日</ref>。西九文化區管理局行政總裁[[連納智|連納智]]則期望，西九公園可以為香港締造一個以文化藝術為核心、充滿活力及富啟發性的公共空間<ref>[http://paper.wenweipo.com/2014/07/23/HK1407230042.htm 西九公園設計文化大道串連] 《文匯報》 2014年7月23日</ref>。<ref>[http://www.westkowloon.hk/tc/newsroom/news/west-kowloon-cultural-district-authority-unveils-concept-design-for-the-park/ 西九文化區管理局公布公園概念設計]</ref>
