'''西单大悦城'''，位于[[北京市|北京市]][[西城区|西城区]][[西单北大街|西单北大街]]甲131号，是[[中粮集团|中粮集团]]旗下的大型综合性商场。

==简介==
2007年12月28日，西单大悦城开业。这是[[中粮集团|中粮集团]]商业地产的首个项目，理念为“源于[[西单|西单]]、高于西单、引领西单”。西单大悦城定位为城市综合体商业模式。开业后最初五年多，销售业绩平均每年递增40%以上，单日最高人流量22万人次、单日最高销售额将近3000万元<ref>{{Cite web |url = http://www.xdjoycity.com/XiDanJoyCity/index.jhtml |title =西单大悦城  |author =  |date =  |publisher = 西单大悦城 |language =  |accessdate =2018-02-12  |quote =  }}</ref>。但是2015年出现销售与客流增长趋缓。2016年，西单大悦城对50%的品牌换血，同时对优秀品牌提出“独有定制”的概念。2017年继续调整，建设了国际化妆品专区，并推出查特花园（Chat Garden）、样街（Young Street）两个主题街区<ref name=ifeng>{{Cite web |url = http://finance.ifeng.com/a/20180207/15973747_0.shtml |title =西单大悦城2017年销售额超41亿元  |author =  |date = 2018-02-07 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。
 
西单大悦城共有13个楼层，每层都有自己的主题<ref name=ifeng/>。第十层是[[首都电影院|首都电影院]]西单大悦城店，2008年开业<ref>{{Cite web |url = http://news.sina.com.cn/c/2017-12-28/doc-ifyqcsft7724116.shtml |title =首都电影院庆祝开业八十周年 陈凯歌忆儿时往事  |author =  |date = 2017-12-28 |publisher =新浪 |language =  |accessdate =  |quote =  }}</ref>。

2018年2月11日在西单大悦城内，[[河南省|河南省]][[西华县|西华县]]人朱某某（男，35岁）持械行凶，致13人（3男10女）受伤，其中一名女性因抢救无效死亡。朱某某当场被警方抓获<ref>{{Cite web |url =http://news.ifeng.com/a/20180211/55977571_0.shtml  |title =西单大悦城行凶案后，北京20万人连夜巡逻防控  |author =  |date = 2018-02-11 |publisher =凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
