'''西爾維奧·佩蒂羅西國際機場'''（{{lang-es|Aeropuerto Internacional de Asunción Silvio Pettirossi}}；{{Airport codes|ASU|SGAS}}）是一座位於[[巴拉圭|巴拉圭]][[亞松森|亞松森]]市郊[[盧克|盧克]]的[[國際機場|國際機場]]，服務巴拉圭首都[[亞松森|亞松森]]，同時為巴拉圭最主要的機場之一，為該國提供國際航班。機場是以該國飛行家[[西爾維奧·佩蒂羅西|西爾維奧·佩蒂羅西]]而命名。此機場前稱為斯特羅斯納總統國際機場，這是以該國前總統[[阿爾弗雷多·斯特羅斯納|阿爾弗雷多·斯特羅斯納]]而命名。

==客運大樓==
機場客運大樓被分成國內與國際航班兩處，按计划國內航班使用5和6號閘口，而國際航班則使用1至4號閘口。但是目前只有一班國內航班，而使用國際航班閘口。因此，目前國內航班閘口被用作貨運、私人飛機與包機用途。

==事故與意外==
*1945年12月3日：[[美國空軍|美國空軍]]一架[[C-47運輸機|道格拉斯C-47B-5-DK]]（註冊編號：43-48602）由亞松森飛往[[蒙特維多|蒙特維多]]途中，在[[阿根廷|阿根廷]]卡洛斯佩萊格里尼東南方16公里墜毀，機上14人罹難<ref>{{Cite web | url=http://aviation-safety.net/database/record.php?id=19451203-1 | title=Accident description 43-48602 | publisher=Aviation Safety Network | accessdate=2011年7月21日}}</ref>。
*1955年6月16日：[[巴西泛空航空|巴西泛空航空]]一架[[洛克希德星座|洛克希德 L-049/149 星座]]（註冊編號：PP-PDJ）由里約熱內盧/加利昂經聖保羅與亞松森往布宜諾斯艾利斯—埃塞薩，在飛機快將降落在此機場時，撞倒12米高的樹木，使部分機機翼折斷，飛機墜毀和起火。機上有24名乘客與機組人員，當中16人罹難。
*1980年8月27日：TAM巴拉圭航空一架[[C-47運輸機|道格拉斯 C-47B]]（註冊編號：FAP2016）在此機場降落時墜毀。此飛機原本是由此機場前往Ayolas，但飛機起飛後不久，發現飛機引擎失效並決定折返機場。事故中一人死亡<ref name=ASN270880>{{cite web|url=http://aviation-safety.net/database/record.php?id=19800827-0 |title=FAP2016 Accident description |publisher=Aviation Safety Network |accessdate=2010年7月24日}}</ref>。
