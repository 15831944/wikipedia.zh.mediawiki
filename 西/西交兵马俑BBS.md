'''西交兵马俑BBS'''是[[西安交通大学|西安交通大学]]的官方BBS，[[中国教育和科研计算机网|中国教育网]]内较早的BBS之一，也是较有影响的BBS之一。下设开发技术、电脑应用、学术科学、社会科学、文学艺术、知性感性、体育运动、休闲音乐、游戏娱乐、兄弟院校、乡音乡情、校务信息、俱乐部区、本站系统、交通大学和新闻信息十六个大的讨论区。

== 大事记 ==
* 2012年3月15日，新认证系统上线，受此影响，每个交大邮箱（包括 stu.xjtu.edu.cn、mail.xjtu.edu.cn以及idp.xjtu6.edu.cn）将可用于不超过4个兵马俑ID的实名制验证。
* 2011年12月初，该站无法访问，查明硬盘再度受损，至2011年12月19日正常恢复。<ref>{{Cite web|url=http://bbs.xjtu.edu.cn/BMY/con?B=sysop&F=M.1324019827.A|title=[公告]BBS维护停机|author=lanboy|date=2011-12-16|publisher=}}</ref>{{Dead link}}
* 顺应社交网站的潮流，该站站务组3月开通[[新浪微博|新浪微博]]官方主页，5月开通[[人人网|人人网]]公共主页。站务组负责将站内热门话题编辑整理后转播到社交网站，并吸引回访。
* 2011年3月10日晚22点左右，该站无法访问，后恢复，系服务器硬盘故障。
* 2010年12月初，部分用户发现的生命力出现调整。部分用户自动变成666生命力，部分变成365的生命力。12月6日，站务组发布公告，正式宣布调整用户生命力的计算方法。注册时间2年（730）天一下的用户，生命力算法不变。注册时间2年至5年（1825天）内，生命力固定为365，5年至8年（2920天），生命力固定为666，8年以上生命力固定在888。注册满2年的用户生命力不再下降。部分用户抱怨称此举将造成注册满两年的用户自杀后无法释放注册邮箱，只能通过挂活；但实际上，自杀功能与id死亡释放邮箱没有任何关系，在此次修改以前，用户也无法通过自杀达到释放注册邮箱的目的。
* 2010年11月30日，兵马俑 BBS 第十四届纪委选举开始。本次选举中不再区分权重。首席纪委 THEKid 开启民意调查，即娱乐性质的投票方式，每日跟踪参选用户的支持率。
* 2010年11月，yyxx、binzhen、lengthlee、SevenEleven被选举为新任站长。Lene、WesternSnow、Coldstar、xiaolei、Wcrow、chunhui、Vanalan不再担任站长职务。dianfeng成为继任站务站长，Lene加入顾问团。

==三月通告==
该站关于禁止校外IP访问的通告中指出“接到本站上级主管部门命令，本站将在两会期间对访问策略进行调整”。从2005年3月5日起，“我校BBS只开通本校校园网范围访问。因此取消 bbs.xanet.edu.cn的IP和功能变数名称。在防火墻上加上外网不能访问的规则”。“本站不主张校内用户随意提供到本站的穿梭。此次访问策略调整待本站上级主管部门另行通知后解除”。本公告同在同一时间其他各大BBS的通告不同，没有说是因为“系统维护”原因进行这个调整。据此，有人断定，其他BBS，如[[水木清华|水木清华]]等，进行调整的真正原因可能不是因为系统维护，而是这里说的“上级主管部门命令”。

2005年3月17日，该站在此发出通告，指出：“接到上级主管部门通知”，“在前次调整访问策略的基础上”，“完全关闭校外访问，具体恢复时间等待上级通知”。同时希望“校外网友备份好自己的数据，以免造成更大的损失”。在3月17日左右，大部分中国大陆的高校BBS都有类似通告。此次高校BBS的调整据说是[[中華人民共和國教育部|中国大陆教育部长]][[周济_(1946年)|周济]]的主张。尽管大部分学生很不理解，很多人以各种形式抗议，到目前为止，没有消息表明政府改变决策。

===代码及文档===
兵马俑BBS源代码遵循[[GNU|GNU]] [[GPL|GPL]]发布，任何用户均可以下载用于非商业用途。代码可以通过[[Subversion|SVN]]签出，官方的代码仓库为 svn co http://bbs.xjtu.edu.cn/svn/src_1.8。[[Google|Google]] [[Google_Code|Code]]上有一份同步镜像。<ref>{{Cite web|url=http://code.google.com/p/bmybbs|title=BMY BBS 项目管理}}</ref>

正在使用的这套源代码使用C语言开发，大量使用了 [[POSIX|POSIX]] 及其他开源库，可以在 [[Linux|Linux]]/[[Cygwin|Cygwin]] 等环境下使用 [[GCC|GCC]] 等工具编译运行。兵马俑BBS的编译安装可以参考YTHT系BBS的文档，兵马俑程序组整理了安装指南《BMYBBS权威安装文档》、《nju09如何使用debian自带apahce2.2》以及《BMYBBS 简易安装方法》。<ref>{{Cite web|url=http://ironblood.net/2011/12/04/easy-installation-for-bmybbs/|author=IronBlood|date=2011-12-04|title=BMYBBS 简易安装方法}}</ref>

技术站长 interma 曾整理过一个系列的技术文章''from Bmy Code to Linux Programming''<ref>{{Cite web|url=http://olds.blogcn.com/wp-content/uploads/46/4644/2007/11/bc2lp.html|author=InterMa|date=2007-11|title=from Bmy Code to Linux Programming|deadurl=yes|archiveurl=https://web.archive.org/web/20160306045414/http://olds.blogcn.com/wp-content/uploads/46/4644/2007/11/bc2lp.html|archivedate=2016年3月6日|df=}}</ref>，便于新人熟悉兵马俑代码以及Linux下的程序开发。

*内存 16GB EEC
*存储 [[SAN|SAN]] 1000G
*网络 双1000Mbps
