'''西伯利亞'''（{{lang-rus|Сибирь|Sibir}}）是[[俄羅斯|俄羅斯]]及[[蒙古国|蒙古国]]北部的一片非常大的地域，佔有整個[[北亞|北亞]]，面積約1,276萬[[平方公里|平方公里]]，範圍西至[[烏拉山脈|烏拉山脈]]、東至[[太平洋|太平洋]]；北至[[北冰洋|北冰洋]]，南至[[中亚|哈薩克]]的中北部，以及[[蒙古国|蒙古国]]和[[中國|中國]]的邊境<ref>{{cite web |url = http://encycl.yandex.ru/dict/bse/article/00070/76500.htm?text=%D0%A1%D0%B8%D0%B1%D0%B8%D1%80%D1%8C |title = Great Soviet Encyclopedia |language = ru |publisher = Encycl.yandex.ru |date =  |accessdate = 2010-05-15 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。整個地域屬於[[俄羅斯|俄羅斯]]，並且占据了其75%的領土。

== 歷史 ==
{{Main|西伯利亚历史}}
早在石器時代以前，西伯利亞已經有人居住。當時西伯利亞的氣候比現在溫和得多，而且當時的水位比較低。現時的[[白令海峽|白令海峽]]，當年有一條陸橋連接西伯利亞和[[阿拉斯加|阿拉斯加]]。現時[[美洲原住民|美洲原住民]]的大多數人口都是透過這條陸橋從西伯利亞來到美洲的。

西伯利亞亦是古代不少強悍民族的搖籃。[[匈奴|匈奴]]、[[鮮卑|鮮卑]]、[[突厥人|突厥]]、[[契丹|契丹]]、[[蒙古|蒙古]]及[[女真|女真]]等等各種民族都是從西伯利亞崛起的。

[[東斯拉夫人|東斯拉夫人]]前往西伯利亞旅行可追溯至11、12世紀，當時[[諾夫哥羅德|諾夫哥羅德]]人與西伯利亞獵人建立貿易關係。最遠到達[[葉尼塞河|葉尼塞河]]，這是基輔羅斯歷史書往年記事說的。

至於西伯利亞的現代發展始於16世紀後期。最初從西邊入侵西伯利亞的不是俄國正規軍，而是[[斯特羅加諾夫|斯特羅加諾夫]]家族[[僱傭軍|僱傭軍]]。1558年，因為[[烏拉山脈|烏拉山脈]]以西的毛皮日漸減少，[[沙皇|沙皇]][[伊凡四世|伊凡雷帝]]授權斯特羅加諾夫家族率[[哥薩克|哥薩克]]在[[烏拉山脈|烏拉山脈]]以東開發商棧，攫取當地豐富的毛皮資源。1572年，在西伯利亞中部的半遊牧國家[[失必兒汗國|西伯利亞汗國]]拒絕用毛皮向俄國朝貢，這成了俄國進攻西伯利亞的理由。還有一種說法是這次征服的起因是當地俄羅斯的居民點經常受到原住民的騷擾。哥薩克領袖[[葉爾馬克·齊莫菲葉維奇|葉爾馬克·齊莫菲葉維奇]]在1578年（[[明朝|明]][[萬曆|萬曆]]六年）10月開始東侵。幾年內，葉瑪克等征服了[[庫臣汗國|庫臣汗國]]，平定西部西伯利亞。其後葉爾馬克在進攻西伯利亞汗國時陣亡，西伯利亞汗國則於1598年滅亡。之後俄國人繼續東進，1636年（明[[崇禎|崇禎]]九年）到達[[鄂霍次克海|鄂霍次克海]]，征服了西伯利亞全境。這個地區成為了俄國人的領土，俄羅斯亦至此成為世界上最大的國家。

當俄國的勢力接近[[清朝|清朝]]時，便發生了軍事衝突。1652年（清[[順治|順治]]九年），俄人東入[[黑龍江|黑龍江]]，「駐防[[寧古塔|寧古塔]][[章京|章京]]海色率所部擊之，戰於烏扎拉村」。這是中俄之間第一場戰鬥。1657年，沙俄派正規軍在尼布楚河與石勒喀河合流處建立了[[雅克萨|雅克薩]]城與[[尼布楚|尼布楚]]城。之後中俄之間發生多次外交和軍事上的衝突。1685年，清[[康熙帝|康熙帝]]在平定「[[三藩之亂|三藩之亂]]」後，派將軍[[彭春|彭春]]5月22日從[[璦琿|璦琿]]起兵五千人，5月25日攻入雅克薩。之後清軍撤軍而俄軍捲土重來。1686年清軍再攻雅克薩。1689年9月7日《[[尼布楚條約|尼布楚條約]]》正式簽字，条约以[[外兴安岭|外兴安岭]]和[[额尔古纳河|额尔古纳河]]为界划分两国边界。晚清衰败以后，沙皇俄罗斯强迫中国清政府签订1858年瑷珲条约和1860年的中俄北京条约，割让黑龙江北、乌苏里江东、库页岛共100多万平方公里的土地给俄罗斯，中国还失去了日本海和鄂霍茨克海的海洋资源和出海口。

1917年[[俄國|俄國]]發生[[十月革命|十月革命]]之後，[[白軍|白軍]]在西伯利亞獨立，成立[[遠東共和國|遠東共和國]]。然而隨著布爾什維克勢力日增，俄國遠東共和國在1922年改組成為[[遠東蘇維埃共和國|遠東蘇維埃共和國]]，並於同年被蘇聯合併。1944年，[[唐努烏梁海|唐努烏梁海]]被併入蘇聯，成為圖瓦共和國。

西伯利亞由於其荒涼，歷史上一直是[[俄罗斯帝国|沙俄]]和[[蘇聯|蘇聯]]政府[[流放|流放]]罪犯的地方。

== 自然地理 ==
=== 名称来源 ===
西伯利亞此名稱可能來自古[[突厥语族|突厥语]]，意思就是「寧靜的土地」。[[西伯利亞韃靼人|西伯利亚鞑靼人]]以此为国家名稱建立[[西伯利亚汗国|西伯利亚汗国]]，后來被[[莫斯科大公国|莫斯科大公国]]征服，西伯利亚此名詞进入了[[俄羅斯文|俄羅斯文]]。也有說法為「[[鮮卑|鮮卑]]利亞」，來自[[鮮卑|鮮卑]]。亦有一個解釋，西伯利亞被解作[[昔班|昔班]]的地方，因為他的封地位於西伯利亞。又有一说，[[俄罗斯沙皇国|沙俄]]以[[锡伯族|锡伯族]]的族名创造北亚的新地名。而在[[中國|中國]]古地圖上，西伯利亞被稱為'''羅荒野'''。

=== 地形與資源 ===
西伯利亞有非常豐富的天然資源，包括[[礦物|礦物]]資源、大量[[油田|油田]]、茂盛的森林及位於西伯利亞西南部的大草原。西伯利亞大致可以分成[[西西伯利亞平原|西西伯利亞平原]]及[[中西伯利亞高原|中西伯利亞高原]]兩部份。這廣大面積的土地大多數都是[[沼澤|沼澤]]，並不適合耕種；其他非沼澤的地方，以一年的一大半被冰遮蔽了的土地，以[[泰加林|泰加森林]]的針葉樹為中心的森林帶很廣闊。但是近年由於開發的過度，使環境破壞成為問題。極北地區被稱為[[凍原|凍原帶]]，終年被[[永久凍土|永久凍土]]所遮蔽。

在西伯利亞有即將滅絕的[[東北虎|西伯利亞虎]]，即東北虎。它們主要在泰加森林生活。這個森林亦是很多野生生物生活的地方。

=== 人口 ===
西伯利亞的人口密度只有每平方公里3人。然而當地的居民大多數都是[[俄羅斯族|俄羅斯族]]及[[乌克兰人|烏克蘭人]]等外來民族。俄羅斯族源自數千年前[[東歐|東歐]]的[[斯拉夫人|斯拉夫人]]。西伯利亞最早的居民是[[通古斯族|通古斯族群]]（[[肃慎人|肃慎人]]）[[蒙古族|蒙古人]]和[[突厥人|突厥人]]的部落族群，如[[布里亞特人|布里亞特人]]、[[圖瓦人|圖瓦人]]和[[雅庫特人|雅庫特人]]。他們的後裔現在仍然生活在那裡。其他少數民族包括: [[通古斯人|通古斯人]]、[[楚科奇族|楚科奇族]]、[[科里亞克族|科里亞克族]]及[[尤卡吉爾人|尤卡吉爾人]]。

大約70%的西伯利亞人都是住在城市的，住在小型的單位。而大多數住在郊外地區的人，生活較為簡單，並居住在大型的木屋。[[新西伯利亞|新西伯利亞]]是西伯利亞最大的城市，人口達150萬人。[[托博爾斯克|托博爾斯克]]、[[托木斯克|托木斯克]]、[[伊爾庫茨克|伊爾庫茨克]]及[[鄂木斯克|鄂木斯克]]是較舊的城市。
