'''西北大学礼堂'''是[[中华民国|中华民国]]时期[[东北大学|东北大学]]礼堂旧址，位于[[陕西省|陕西省]][[西安市|西安市]][[碑林区|碑林区]][[西北大学_(中国)|西北大学]]太白校区内。现为[[陕西省文物保护单位|陕西省文物保护单位]]。<ref>{{Cite web |url= http://www.wenwu.gov.cn/contents/230/9394.html |title= 第五批陕西省文物保护单位名单（254处） |author=  |date= 2011年3月10日 |publisher= 陕西省文物局 |language= zh-hans |accessdate= 2013年2月5日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20130318191829/http://www.wenwu.gov.cn/contents/230/9394.html |archivedate= 2013年3月18日 |df=  }}</ref>

“[[九一八事变|九一八事变]]”后，东北大学被迫内迁，先至北平，后随[[张学良|张学良]]迁到西安。到达西安后的东北大学落脚在现在的西北大学校园，大礼堂就是在此时建造的。张学良还为东北大学在西安的校舍奠基题写了“沈阳设校，经始维艰。自九一八，惨遭摧残。流离燕市，转徙长安。勖尔多士，复我河山。”并立碑纪念，后石碑于内战中毁于胡宗南部队，<ref>{{Cite web |url= http://gaojiao.jyb.cn/xwbj/200906/t20090622_284917.html |title= 西北大学大礼堂 |author=  |date= 2009年6月22日 |publisher= 中国教育新闻网－中国教育报 |language= zh-hans |accessdate= 2013年2月5日 |quote=  }}</ref>现在的石碑是学校于1992年根据原碑的拓片重立的。<ref>{{Cite web |url= http://news.nwu.edu.cn/content.php?id=4144&cid=1 |title= 大礼堂 |author= 熊晓芬 |date= 2007年5月22日 |publisher= 西北大学报 |language= zh-hans |accessdate= 2013年2月5日 |quote=  }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>礼堂周围的两排平房同样被保留下来。
