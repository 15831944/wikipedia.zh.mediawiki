'''西门口站'''，全称'''西门口(天一阁·月湖景区)站'''，是[[浙江省|浙江省]][[宁波市|宁波市]]一座地下轨道交通车站，是[[宁波轨道交通|宁波轨道交通]][[宁波轨道交通1号线|1号线]]车站。车站位于[[宁波市|宁波市]][[海曙区|海曙区]]，于2014年5月30日开通运营<ref name="main">宁波市发展和改革委员会, 关于宁波市轨道交通1号线一期工程初步设计的复函, 甬发改投资函〔2009〕118号</ref>。

== 车站形制 ==
西门口站位于[[海曙区|海曙区]]望京路、长春路与中山西路交叉路口的西侧，沿中山西路设置。车站为地下二层岛式车站。净长186.6米，净宽17.3米，地下建筑面积9385平方米<ref name="main" />。

== 装饰 ==
[[File:西门口站文化墙1.JPG|thumb]]
西门口站是宁波轨道交通1号线的6个文化站之一。车站的文化主题为“故城书韵”，取材于附近的古代藏书楼[[天一阁|天一阁]]。车站文化墙的背景为天一阁，装饰由书卷和水乡景致组成。明代藏书家，天一阁创始人[[范钦|范钦]]和清代[[浙东学派|浙东学派]]代表人物[[黄宗羲|黄宗羲]]也出现在文化墙中<ref>{{cite web|url=http://news.cnnb.com.cn/system/2014/05/13/008061217.shtml|title=宁波轨道交通1号线6个站点文化墙惊艳亮相|publisher=中国宁波网|date=2014-05-13|accessdate=2014-05-13}}</ref>。

== 交通接驳 ==
西门口站设有一座公交首末站，位于西门口广场东北角的中实巷南，面积986平方米，能够作为2条公交线终点站<ref>{{cite web|url=http://news.cnnb.com.cn/system/2014/03/10/008007423.shtml|title=宁波轨道交通1号线（一期）配套公交接驳站项目开工|publisher=中国宁波网|date=2014-03-10|accessdate=2014-05-18}}</ref>。此外，车站附近还设有2个[[宁波公共自行车|公共自行车]]租赁点，安设有50辆自行车<ref>{{cite web|url=http://news.cnnb.com.cn/system/2014/04/04/008029106.shtml|title=宁波轨道交通1号线一期公交接驳方案发布|publisher=东南商报|date=2014-04-04|accessdate=2014-05-18}}</ref>。
