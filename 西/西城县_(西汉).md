'''西城县'''，中国古[[县|县]]名，在今[[陕西省|陕西省]][[安康市|安康市]]境。

《[[史记|史记]]<ref>《史记·卷五·秦本纪第五》（[[秦惠文王|秦惠文王]]更元）十三年，庶长章击楚於丹阳，虏其将屈匄，斩首八万；又攻楚汉中，取地六百里，置汉中郡。楚围雍氏……</ref>》称前312年，[[秦国|秦国]]取[[楚国|楚国]]汉中置[[漢中郡|汉中郡]]。安康市<ref name="a1">{{Cite web |url=http://www.ankang.gov.cn/Node-3107.html |title=历史沿革 |accessdate=2019-03-06 |author= |date=2018-04-10 |work= |publisher=安康市人民政府网站，内容来源：安康年鉴（2017）|language=Zh-hans }}</ref>和[[汉滨区|汉滨区]]政府编撰的地方年鉴<ref>{{Cite web |url=http://www.hanbin.gov.cn/Content-94260.html |title=汉滨区2017年年鉴（概况）》行政区划 |accessdate=2019-03-06 |author=王慧芳 |date=2018-04-13 |work=中共安康市汉滨区委、汉滨区人民政府主办，汉滨区档案史志局编纂 |publisher=安康市汉滨区人民政府网站|language=Zh-hans }}</ref>皆认为同时置西城县，亦指西城县为汉中郡[[治所|治所]]<ref name="a1"/>，未提及史料来源。[[王蘧常|王蘧常]]在《[[秦史_(王蘧常)|秦史]]》中考证{{noteTag|1={{cite journal en|quotes =

西汉时，西城县属汉中郡<ref>《[[汉书|汉书]]·卷二十八上·地理志第八上》零陵郡……汉中郡，秦置。莽曰新成。属益州。户十万一千五百七十，口三十万六百一十四。县十二：西城，旬阳……</ref>。北魏置[[东梁州|东梁州]]，西城县为州治。西魏以东梁州改置[[金州_(西魏)|金州]]，仍为治所。隋朝时，以金州为[[西城郡|西城郡]]。隋末废除西城县。[[义宁|义宁]]二年（618年），以[[金川县_(隋朝)|金川县]]复置<ref name="a2">《[[新唐书|新唐书]]·卷四十·志第三十》◎地理四……金州，汉阴郡，上。本西城郡，天宝元年曰安康郡，至德二载更名……县六：有府一，曰洪义。西城，上。本金川，义宁二年更名。有牛山。汉水有金。洵阳……</ref>。唐朝时，历属金州、[[安康郡|安康郡]]、[[汉阴郡|汉阴郡]]<ref>《[[旧唐书|旧唐书]]·卷三十九·志第十九》◎地理二……金州，隋西城郡，武德元年，改为金州，领洵阳、石泉、安康等县……天宝元年，改为安康郡。至德二年二月，改为汉南郡。乾元元年，复为金州。旧领县六……西城，州所理。汉西城县，属汉中郡。后魏置安康郡，寻改为东梁州。又以其地出金，改为金州。皆以西城为治所。隋末废。义宁二年，复置。洵阳……</ref>，为[[上县|上县]]<ref name="a2"/>。宋朝时，仍属金州，为[[下县|下县]]<ref>《[[宋史|宋史]]·卷八十五·志第三十八》地理一……京西路……金州，上，安康郡，乾德五年，改昭化军节度……县五：西城，下。洵阳……</ref>。元朝时废除。
