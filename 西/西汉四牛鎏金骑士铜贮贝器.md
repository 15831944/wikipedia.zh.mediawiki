'''西汉四牛鎏金骑士铜贮贝器'''{{efn|该器也被称为四牛骑士贮贝器、鎏金骑士贮贝器、四牛鎏金骑士铜贮贝器、青铜四牛鎏金骑士贮贝器等}}，中国[[西汉|西汉]]时期文物，1956年于[[云南|云南]][[晋宁|晋宁]][[石寨山古墓群|石寨山]]10号墓出土。该器器高50厘米，盖径25.3厘米，整体为束腰圆筒形，为典型的束腰圆筒形[[贮贝器|贮贝器]]类型，腰部两侧各饰有一个[[虎|虎]]形耳，器盖上饰有4头公牛和1名骑马的骑士，骑士通体鎏金。现藏于[[云南省博物馆|云南省博物馆]]。

== 出土 ==
[[云南省博物馆|云南省博物馆]]从1950年代开始在云南省全省范围内进行的科学考古发掘、试掘、调查<ref>{{cite web |title=云南省博物馆 |url=http://www.china.com.cn/book/zhuanti/1bowuguan/2008-11/11/content_16746569.htm |website=中国网 |accessdate=2018-07-14 |archive-url=https://web.archive.org/web/20180714135242/http://www.china.com.cn/book/zhuanti/1bowuguan/2008-11/11/content_16746569.htm |archive-date=2018-07-14 |dead-url=no }}</ref>。1955年，在[[云南省|云南省]][[昆明市|昆明市]][[晋宁县|晋宁县]]的石寨山西面坡上发掘了新石器时代遗址一处，获得大量陶器和石器。1955年至1960年间，先后四次在石寨山山麓发掘了[[春秋战国|春秋战国]]至[[东汉|东汉]]时期的墓葬48座，出土文物四千多件<ref>{{cite web |author1=白寿彝主编 |title=第八节 汉代少数族地区的考古资料 |website=中国通史}}</ref>。四牛骑士贮贝器于1956年在石寨山10号墓出土<ref>{{cite journal |author1=陆静杰 |title=四人缚牛祭祀扣饰的修复 |journal=中国文物保护技术协会学术年会 |date=2004 |pages=123-124}}</ref>，同墓中出土的文物还有虎牛搏斗扣饰、四蹲蛙铜鼓等<ref name="国宝档案" />。

== 形制 ==
[[File:Kunming_Oct_2007_031.jpg|缩略图]]
[[贮贝器|贮贝器]]是[[滇国|滇国]]特有的贮放[[贝币|贝币]]的青铜器，器上铸造出姿态各异的群像，具体而生动地表现了祭祀、战争、狩猎、祈年、播种等情形，是研究滇国社会历史的重要资料<ref>{{cite web |author1=陈成军 |title=古滇国的外汇 |url=http://www.chnmuseum.cn/tabid/438/InfoID/78565/frtid/285/Default.aspx |website=中国国家博物馆 |accessdate=2018-07-14 }}</ref>。

四牛鎏金骑士铜贮贝器为典型的束腰圆筒形贮贝器类型。腰部自然收束，两侧分别饰有一个作向上攀爬状的虎形耳。底部为三只兽足形矮足。器盖上的装饰分内外两层，外层雕铸四头[[公牛|公牛]]，呈[[逆时针方向|逆时针方向]]排列，牛角弯长，膘肥体壮，领肉隆起，为云南常见的[[封牛|封牛]]；里层立一个圆柱柄托盘，托盘上饰有一名骑马的骑士，马昂首翘尾并张口作嘶鸣状，骑士赤脚佩剑且全身鎏金<ref>{{cite web |author1=翟路 |title=战国　鎏金骑士贮贝器 |url=http://www.gg-art.com/profesional/index.php?detailid=711 |website=中华博物 |accessdate=2019-01-07}}</ref>。古滇国出土其他器物上常见有赤脚的人物形象，反映了滇国当时的穿着习惯。器盖上的四头牛代表了墓主人生前的财富状况，骑士形象则反映墓主人生前拥有较高的权力，专家推测出土该器的10号墓的墓主人生前应为滇国王室成员或奴隶主贵族<ref name="国宝档案">{{cite web |title=《国宝档案》 20121208 西汉 四牛鎏金骑士贮贝器 |url=http://tv.cntv.cn/video/C14121/a6d28ed9770c4ac19042ed612242638a |website=央视网 |accessdate=2019-01-07 |date=2012-12-08}}</ref><ref name="张蕊">{{cite web |author1=张蕊 |title=云南省三件国宝入选《国家宝藏》第二季 来看看它们稀奇在哪儿 |url=http://yn.yunnan.cn/system/2018/12/03/030131532.shtml |website=云南网 |accessdate=2019-01-07 |date=2018-12-03}}</ref>。
