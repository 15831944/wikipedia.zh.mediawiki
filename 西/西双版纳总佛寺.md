==历史==
1990年之前，[[傣语|傣语]]称该寺为“'''洼坝吉'''”，“洼”意为寺院，“坝”意为森林，“吉”是[[佛陀|佛陀]]度过25个[[雨安居|雨安居]]的[[祗陀太子|祗陀]]林[[给孤独园|给孤独园]]的简称，“洼坝吉”的意思是像祗陀林给孤独园一样的森林道场。相传该寺是自[[缅甸|缅甸]]前来弘法的两位[[巴利语系佛教|巴利语系佛教]]僧人于7世纪中期发心兴建，是西双版纳最早建立的巴利语系佛教寺院之一<ref name=kaig>{{Cite web |url =http://www.fjnet.com/jjdt/jjdtnr/201212/t20121212_201783_1.htm  |title =西双版纳总佛寺举行大雄宝殿落成开光庆典活动  |author =  |date =2012-12-12  |publisher = 佛教在线  |language =  |accessdate =  |quote =  }}</ref>。据说建于615年<ref name=liuyw>刘扬武，气势恢弘 清净庄严的西双版纳总佛寺，云南档案2014年第01期</ref>。傣文史籍《佛陀之教史话》载：“朱腊沙哈（西双版纳地区使用的历法）前23年（即公元615年），西双版纳地区建盖第一所佛寺‘洼坝吉’（即西双版纳总佛寺前身）。之后，各地的佛寺、佛塔才陆续建立。”<ref name=qiannian/>

13至14世纪，该寺被西双版纳宣慰确立为西双版纳最高等级佛寺，是发布全境性佛事活动、昭示批准高级僧职晋升、举办新任宣慰使和各勐土司宣誓仪式的场所，也是西双版纳的宣慰（西双版纳最高统治者“[[召片领|召片领]]”，傣语意为“土地之王”）和属下各勐土司头人拜佛、各地佛寺住持朝拜的场所<ref name=kaig/><ref name=liuyw/>。

西双版纳傣族几乎每个村寨都有佛寺。其佛寺分四个等级：最低一级是各村寨的佛寺；上一级是由4所以上村寨佛寺组成的中心佛寺“布萨堂佛寺”；再上一级是十二版纳各一个的“拉扎坦”总寺，负责管辖各中心佛寺；最高一级就是景洪的“拉扎坦大总寺”即该寺，统辖西双版纳的近六百座佛寺<ref name=liuyw/>。

该寺历史上多次损毁和修复，[[文革|文革]]中遭到彻底毁灭。[[改革开放|改革开放]]后，在[[云南省佛教协会|云南省佛教协会]]会长[[刀述仁|刀述仁]]居士倡导下，经[[中共云南省委|中共云南省委]]、[[云南省人民政府|云南省人民政府]]、中共西双版纳傣族自治州委、西双版纳傣族自治州人民政府和云南省佛教协会支持，由云南省佛教协会和西双版纳傣族自治州人民政府共同出资30万元人民币恢复重建，1989年2月14日动工，这是该寺第四次重建。1990年5月竣工。1990年5月6日至8日举行开光法会，全州佛教界人士及一万多名信徒参加。同时，西双版纳傣族自治州佛教协会移锡总佛寺，恢复了该寺在西双版纳佛教界的地位及社会属性。1990年6月18日，[[中国佛教协会|中国佛教协会]]会长[[赵朴初|赵朴初]]为洼坝吉佛寺更名为“'''西双版纳总佛寺'''”，并亲笔题写寺名<ref name=kaig/><ref name=qiannian>{{Cite web |url = http://www.bndaily.com/c/2012-12-23/9038.shtml |title = 千年古寺盛世重光 佛光普照祈福万民——西双版纳总佛寺前世今生掠影 |author =  |date = 2012-12-23 |publisher =西双版纳新闻网 |language =  |accessdate =  |quote =  }}</ref>。2001年11月12日，西双版纳总佛寺举行新建僧寮“聚缘楼”落成典礼<ref>凌海成，西双版纳总佛寺举行聚缘楼落成典礼，法音2001(12):43</ref>。

1989年总佛寺恢复重建后，仅有大殿和僧寮（为住宿、教学、办公一体化）<ref name=kaig/>。大殿面积偏小，且大殿屋顶多处漏雨<ref name=qiannian/>。在徐军居士发心捐献243万元人民币后，对西双版纳总佛寺重新规划修建，2009年2月20日举行了重建奠基，这是该寺第五次重建。新规划重建的总佛寺主体建筑由大殿、戒堂、阿戛牟尼舍利塔、钟楼、鼓楼、僧寮、长老寮、迎宾楼、斋堂以及高约7.9米的立佛等组成，各主体建筑都按[[傣族|傣族]]传统建筑风格修建。总佛寺占地面积约3000平方米，建筑面积约1000平方米。2012年11月竣工。2012年12月12日，总佛寺大雄宝殿、福顺楼、鼓楼落成开光庆典启动仪式举行<ref name=kaig/><ref name=qiannian/>。

重建后的西双版纳总佛寺是云南省佛教协会南传佛教工作委员会、西双版纳傣族自治州佛教协会、景洪市佛教协会、[[云南佛学院|云南佛学院]]西双版纳分院驻会和办公的场所<ref name=kaig/><ref name=qiannian/>。

2014年6月11日，万达西双版纳国际度假区举行西双版纳总佛寺禅修堂奠基仪式，邀请了当地佛教僧侣为奠基仪式做了洒净，为佛塔的开工培土。西双版纳总佛寺禅修堂位于万达西双版纳国际度假区内<ref>{{Cite web |url = http://www.wanda.cn/2014/2014latest_0611/28640.html |title =西双版纳总佛寺禅修堂隆重奠基  |author =  |date =2014-06-11  |publisher =  万达集团|language =  |accessdate =  |quote =  }}</ref>。

==建筑==
[[File:WatChienglarn.jpg|thumb]]
西双版纳总佛寺位于[[曼听公园|曼听公园]]附近，为一矩形大院，坐西南朝东北。主要建筑有<ref name=jianzhu>{{Cite web |url = http://fo.ifeng.com/a/20160218/41552180_0.shtml |title = 西双版纳总佛寺 |author =  |date =2018-03-09  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>：

*南门（大门）：位于坝吉路北侧，寺院正南的围墙中间，三个门洞，中门上方有“西双版纳总佛寺”匾额，匾额为傣文、汉文写成，上方为傣文，下方为赵朴初题写的“西双版纳总佛寺”。
*北门：位于戒堂西北，坐南朝北，三个门洞，无围墙连接，是点景建筑。
*大雄宝殿（傣语“雄罕”）：位于正中，坐西朝东<ref name=jianzhu/>。共两层，一层是钢筋混凝土粘贴水泥制佛寺传统雕花图案，二层采用西双版纳佛寺木建筑风格<ref name=qiannian/>。面阔五间，进深九间。大殿二层东门前是宇厦，下有台阶通向地面，殿门上方悬傣文和汉文“大雄宝殿”匾额。走上台阶进入殿内，红色圆形木柱分立殿厅两侧。殿厅内尽头有供佛像的上下两台基座。基座上台正中供奉一尊[[释迦牟尼|释迦牟尼]]金像。金像左右及前方供奉多尊高不足1米的佛像。基座下台供多尊佛像。殿厅下方为一层大厅，正中供奉一尊竹编佛，高约5米，是西双版纳现有最高的竹佛，2015年由[[法住禅林|法住禅林]]请缅甸高僧都波温香（又称威马拉波迪（Vimalabodhi）尊者）编制而成，这位缅甸尊者曾发愿要编108尊竹佛，总佛寺的这尊竹佛是他编成的第73尊。竹编佛左右两侧有多尊金身佛像<ref>{{Cite web |url =http://foxue.163.com/16/0114/10/BD9IVJGE03240LQR.html  |title = 西双版纳总佛寺 |author =  |date = 2016-01-14 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。大雄宝殿一层南北两侧各开三个门，南侧中门上方挂有汉文“爱国利民”匾额；东侧在台阶下南北各开一个门；西侧二层也有台阶通向地面，台阶南北两侧一层各开一门。总计一层开十个门（南北各三个，东西各两个），每个门上方各有一尊护法神像<!--（以下待查考），分别是：四面[[梵天|梵天]]（位于南侧东门），[[广目天王|广目天王]]（位于南侧中门），[[大自在天|大自在天]]（位于南侧西门）；[[迦楼罗|迦楼罗]]（位于北侧东门），[[增长天王|增长天王]]（位于北侧中门），[[夜叉|夜叉]]（位于北侧西门）；[[持国天王|持国天王]]（位于东侧南门），[[西古打阿满|西古打阿满]]（或称 “丢麻拉”<ref name=tian/>，功德证人，位于东侧北门）；[[罗摩}召朗玛|罗摩}召朗玛]]？（位于西侧北门），？（位于西侧南门）-->，各在一座2017年建成的银色小亭内。西双版纳地区傣族佛寺大殿顶部的塔状金属饰物，傣语称“萨帕”，代表天界，两端的鸱吻圈定了天界的范围；两侧屋脊上的火焰状瓦饰称“密打”，垂脊上的卷叶纹状瓦饰称“密来”，象征云彩；戗脊首端一般装有胸部镶有小圆玻璃镜子的孔雀或有翼的龙，脊上有狮子、鸭、鱼、人物等瓦饰（陶饰）；[[歇山顶|歇山顶]]的山尖下装有镂空的木制悬鱼，山尖屋檐下的板壁上绘有金水图案；以上装饰都象征着进入佛寺就进入了天上的佛国<ref>李靖寰，中国佛教艺术的一支奇葩--谈傣族的佛教造型艺术，云南艺术学院学报2001(4):80-91</ref>。
*立佛：大雄宝殿门前台阶下方正东有供台，台上供有金身佛像一尊。
*戒堂（傣语“波苏”，直译为莲花极顶亭）：位于大雄宝殿东北，平面近似十字形，坐北朝南。
*阿戛牟尼舍利塔：位于戒堂西南侧，为白色舍利塔，是为[[松溜·阿嘎牟尼|松溜·阿嘎牟尼]]（又译松列·阿嘎牟尼）而建，塔身的牌子上写有傣文和汉文的“松列阿嘎牟尼舍利塔”。
*钟楼：位于大雄宝殿东北，戒堂东南，坐西朝东，三层亭式建筑。门前有台阶直通二层。一层为基座，三层悬挂一口大钟。
*鼓楼：位于大雄宝殿东南，二层楼阁式建筑。门前有台阶直通二层。一层为基座。
*长老寮：位于寺院东北角，戒堂东北，两层，平面为曲尺形，曲尺拐角外侧朝向东北。正门开在内侧拐角处，面对西南方。长老寮西南与戒堂之间的庭院中央有一水池，水池中央为[[帕媚托拉尼|喃托腊尼]]（土地女神）<ref name=tian>田玉玲，傣族南传佛教艺术综论，西南学刊2010(00)</ref>。
*福顺楼：即僧寮（傣语“哄暖”），是总佛寺僧侣住房。位于大雄宝殿西北侧，三层，坐西朝东。
*迎宾楼
*斋堂：位于大雄宝殿以西，福顺楼南侧，两层，坐西朝东。

==活动==
;对外交流
西双版纳总佛寺与中国国内佛教界及[[东南亚|东南亚]]、[[南亚|南亚]]佛教国家间交流频繁。1990年2月17日，[[上海市佛教协会|上海市佛教协会]]会长、[[玉佛寺_(上海)|玉佛寺]]方丈[[释真禅|真禅]]率团访问西双版纳州佛教协会，这是西双版纳佛教恢复后首批内地佛教参访团。1990年12月20日至27日，[[全国政协副主席|全国政协副主席]]、中国佛教协会会长[[赵朴初|赵朴初]]在总佛寺主持召开中国南佛上座部佛教工作座谈会，并为西双版纳总佛寺题词。2002年4月23日，上海市佛教协会会长、玉佛寺方丈[[释觉醒|觉醒]]来访。2003年4月16日，[[福建省佛教协会|福建省佛教协会]]常务副会长、[[鼓山|鼓山]][[涌泉寺_(福州)|涌泉寺]]方丈[[释普法|普法]]来访。2007年11月14日，[[台湾|台湾]][[佛光山|佛光山]][[释星云|星云]]来访并种下一株[[菩提树|菩提树]]<ref name=qiannian/>。

1990年5月17日，根据中国佛教协会同[[泰国|泰国]]方面的协议，派送都罕听等10名[[沙弥|沙弥]]留学泰国帕布达般丹帕寺。1990年1月16日，泰国布达般丹帕寺阿章艮罕大长老率团访问西双版纳州佛教协会，并赠1尊佛像，这是[[改革开放|改革开放]]后泰国首个访问西双版纳的佛教团。1993年3月3日至4日，泰国[[诗琳通公主|诗琳通公主]]访问西双版纳，4日到访总佛寺并种一株菩提树。泰国皇家为庆祝[[普密蓬·阿杜德|普密蓬]]国王72岁寿辰，特向总佛寺御赐袈裟那衣，并捐26.2万元人民币建造总佛寺戒堂。1993年6月30日，[[泰国僧王|泰国僧王]][[颂绿·帕映纳旺讪旺|颂绿·帕映纳旺讪旺]]智护尊者到访总佛寺并种两株[[扇椰子|贝叶树]]。1998年10月29日，泰国王姐[[干拉雅妮·瓦塔娜|干拉雅妮·瓦塔娜]]公主到访总佛寺参加戒堂落成开光法会。2010年11月3日，[[世界佛教徒联谊会|世界佛教徒联谊会]]主席、泰国红十字会秘书长鹏·旺纳麦提一行奉泰国国王普密蓬派遣，到总佛寺捐一尊金身佛像并布施[[袈裟|袈裟]]，当时世界佛教徒联谊会为纪念泰国国王普密蓬80岁寿辰共铸19尊素可泰金身佛像，并将佛像分别供奉在19个国家和地区的重要场所，其中赠送中国5尊，分别供奉在[[北京|北京]][[灵光寺_(北京)|灵光寺]]、[[上海|上海]][[静安寺|静安寺]]、西双版纳总佛寺、[[陕西|陕西]][[法门寺|法门寺]]、[[大慈恩寺|大慈恩寺]]。2005年10月26日，总佛寺举行恭迎[[斯里兰卡|斯里兰卡]]菩提圣树法会。1998年10月31日，参加中、[[日本|日]]、[[韩国|韩]]佛教友好交流委员会北京会议的日本代表11人，在中国佛教协会副会长兼秘书长[[刀述仁|刀述仁]]陪同下访问总佛寺。2002年2月8日，[[老挝总理|老挝总理]][[本扬·沃拉吉|本扬·沃拉吉]]到访总佛寺<ref name=qiannian/>。2014年11月1日，[[泰国|泰国]]前总理[[英拉|英拉]]和[[他信|他信]]抵达西双版纳，在总佛寺拜佛<ref>{{Cite web|url = http://picture.youth.cn/qtdb/201411/t20141102_5964257.htm|title = 英拉和他信在西双版纳总佛寺拜佛|author = |date = 2014-11-02|publisher = 中国青年网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180309182830/http://picture.youth.cn/qtdb/201411/t20141102_5964257.htm|archive-date = 2018-03-09|dead-url = yes}}</ref>。

;佛事活动
1998年4月14日，西双版纳总佛寺举行首次浴佛迎新年法会，该活动此后成为总佛寺每年一度的迎新法会。2003年4月14日，浴佛迎新年法会被州政府纳入[[傣历|傣历]]新年节活动内容。2001年10月2日，总佛寺首次举办解夏托钵化缘活动<ref name=qiannian/>。

2004年1月29日至31日，总佛寺举行[[都龙庄|都龙庄]]比丘晋升“祜巴”仪式法会，来自[[缅甸|缅甸]]、[[老挝|老挝]]、[[泰国|泰国]]及中国国内的高僧和信众数万人参加<ref name=qiannian/>。都龙庄由此晋升为[[祜巴龙庄勐|祜巴龙庄勐]]。

2016年2月17日至19日，由[[云南省佛教协会|云南省佛教协会]]、[[中国社会科学院世界宗教研究所|中国社会科学院世界宗教研究所]]主办，西双版纳傣族自治州佛教协会、西双版纳总佛寺承办的首届南传佛教高峰论坛暨帕祜巴和帕松列升座庆典法会在西双版纳总佛寺举行。其中2月19日举行了升座庆典法会。此次升座的高僧共六位：祜巴龙庄勐升为帕松列、西滴看堂升为帕祜巴、玛哈香升为帕祜巴、都香达升为帕祜巴、都炳升为帕祜巴、都罕听升为帕祜巴<ref>{{Cite web |url = http://www.fjnet.com/jjdt/jjdtnr/201602/t20160218_238608.htm |title =西双版纳总佛寺举行中国南传佛教帕松列和帕祜巴升座庆典法会  |author =  |date = 2016-02-18 |publisher =  佛教在线|language =  |accessdate =  |quote =  }}</ref>。此次升座庆典是继1956年之后规模最大的一次。[[中国佛教协会|中国佛教协会]]副会长、云南省佛教协会副会长、西双版纳总佛寺住持祜巴龙庄勐由此晋升为[[帕松列龙庄勐|帕松列龙庄勐]]<ref>孙云霞、梁晓芬，荷如来慧命 弘南传佛教 记新升座的帕松列龙庄勐，中国宗教2016年04期</ref>。

2017年2月4日，总佛寺举行177位少年出家法会，这是[[改革开放|改革开放]]后，总佛寺首次举办大型少年出家活动<ref>{{Cite web |url = http://fo.ifeng.com/a/20170205/44539108_0.shtml |title = 场面震撼！西双版纳总佛寺举行177位少年出家法会 |author =  |date = 2017-02-05 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

;文化活动
2004年2月1日，“西双版纳康朗协会”在总佛寺成立，会员主要由曾在总佛寺出家后来还俗的“康朗”共23人组成。2007年4月13日，该协会换届并更名为“西双版纳保护傣族传统文化协会”，吸收僧人及其他所有热爱傣族文化的人员参加。2005年，在[[联合国教科文组织|联合国教科文组织]]支持下，保护西双版纳佛教寺院传统装饰艺术和建筑工艺的“基那利”项目在西双版纳实施<ref name=qiannian/>。

1998年3月5日，由西双版纳州佛教协会和[[中国科学院西双版纳热带植物园|中国科学院西双版纳热带植物园]]共同组建的西双版纳傣族自治州宗教植物文化协会获批准，同年6月16日在总佛寺挂牌成立<ref name=qiannian/>。

1996年，西双版纳总佛寺成立新星乐队，1997年4月发行首张西双版纳现代傣语歌曲专辑，开创了西双版纳现代傣语歌曲的先河。1998年7月新星乐队更名“傣家乐”乐队，2000年4月发行1张专辑。2002年，“傣家乐”乐队自总佛寺独立，更名为“盛太乐”乐队。2007年，部分人员从“盛太乐”乐队分出，成立“星王”乐队。2011年，又有部分人员从“盛太乐”乐队分出，成立“星光傣”乐队<ref name=qiannian/>。

;慈善事业
2003年7月26日，在总佛寺住持祜巴龙庄勐倡导下，西双版纳佛光之家成立。前身是[[联合国儿童基金会|联合国儿童基金会]]支持的“希望帮助”项目，项目结束后，发展为向公众普及[[艾滋病|艾滋病]]防治知识并救助艾滋病病毒感染者、病人及因艾滋病致孤致困人群、家庭的非盈利民间组织<ref name=qiannian/>。
