'''西九州自動車道'''（{{lang|ja|西九州自動車道、にしきゅうしゅうじどうしゃどう、NISHI-KYUSHU EXPRESSWAY}}）是從[[福岡縣|福岡縣]][[福岡市|福岡市]]經[[佐賀縣|佐賀縣]][[唐津市|唐津市]]、[[長崎縣|長崎縣]][[佐世保市|佐世保市]]，至佐賀縣[[武雄市|武雄市]]與[[長崎自動車道|長崎自動車道]]連接，總長約150 [[公里|公里]]的[[日本高速公路|高速公路]]（[[國土交通大臣指定高規格幹線道路（一般國道快速道路）|國土交通大臣指定高規格幹線道路（一般國道快速道路）]]）。全線被指定為[[#國道497號|國道497號]]。一般以'''西九州道'''（{{lang|ja|にしきゅうしゅうどう}}）略稱之。

高速公路路線編號被編為 {{Ja Exp Route Sign|E35}} <ref>[http://www.mlit.go.jp/road/sign/numbering/ 高速道路ナンバリング路線図(国土交通省)]</ref>。

== 國道497號 ==
{{Ja_Route_Sign|497}}
* 起點 ：福岡市
* 終點 ：武雄市（東川登町大字袴野字宮市10147番1武雄南交流道）
* 重要經過地：佐賀縣[[東松浦郡|東松浦郡]][[濱玉町|濱玉町]]、唐津市、伊萬里市、松浦市、長崎縣[[北松浦郡|北松浦郡]][[江迎町|江迎町]]、同郡[[佐佐町|佐佐町]]、佐世保市
* 距離 : 105.1 [[公里|公里]]<small>（福岡縣 6.7 公里、福岡市 8.1 公里、佐賀縣 42.8 公里、長崎縣 47.5 公里）</small><ref name="encho">{{Cite web|format=PDF|url=http://www.mlit.go.jp/road/ir/ir-data/tokei-nen/2016/pdf/d_genkyou26.pdf|title=表26 一般国道の路線別、都道府県別道路現況|page=28|work=道路統計年報2016|publisher=[[国土交通省|国土交通省]][[道路局|道路局]]|accessdate=2017-05-06}}</ref><ref group="注釋" name="encho">統計至2015年4月1日</ref>
* 共線距離 : 無
* 未啟用距離 : 23.5公里<small>（福岡縣 - 公里、福岡市 - 公里、佐賀縣 12.2 公里、長崎縣 11.3 公里）</small><ref name="encho"/><ref group="注釋" name="encho"/>
* 實際距離 : 81.5 公里<small>（福岡縣 6.7 公里、福岡市 8.1 公里、佐賀縣 30.6 公里、長崎縣 36.2 公里）</small><ref name="encho" /><ref group="注釋" name="encho" />

== 歷史 ==
* [[1983年|1983年]][[4月1日|4月1日]] : 當面活用區間之二丈濱玉道路通車。
* [[1988年|1988年]][[3月24日|3月24日]] : 佐世保大塔交流道 - 波佐見有田交流道間通車。
* 1988年度 : 唐津道路事業化。
* [[1989年|1989年]][[11月30日|11月30日]] : 波佐見有田交流道 - 武雄南交流道間通車。
* [[1990年|1990年]][[1月26日|1月26日]] : 武雄南交流道 - 武雄系統交流道間通車，與長崎自動車道連接。
* [[1992年|1992年]][[2月|2月]] : 唐津道路[[都市計畫|都市計畫決定]]。
* 1992年度 : 唐津道路用地徵收開始。
* [[1993年|1993年]][[3月26日|3月26日]] : 周船寺交流道 - 東交叉點間通車。
* [[1995年|1995年]]度 : 唐津道路開工。伊萬里道路事業化。
* [[1997年|1997年]]度 : 伊萬里松浦道路事業化。
* [[1998年|1998年]][[4月17日|4月17日]] : 佐世保港交流道 - 佐世保大塔交流道間通車。
* 1998年[[9月26日|9月26日]] : 拾六町交流道 - 周船寺交流道間通車。
* [[2001年|2001年]][[10月13日|10月13日]] : 福重系統交流道 - 拾六町交流道通車，與福岡高速1號線連接。
* [[2003年|2003年]][[11月|11月]] : 福重系統交流道 - 前原交流道間4車道化。
* [[2005年|2005年]][[6月14日|6月14日]] : 伊萬里松浦道路 長崎縣區間 (7.1 公里) 都市計畫決定。
* 2005年[[9月16日|9月16日]] : 伊萬里松浦道路 佐賀縣區間 (10.1 公里) 都市計畫決定。
* 2005年[[12月18日|12月18日]] : 濱玉交流道 - 唐津交流道間通車。
* [[2006年|2006年]][[12月25日|12月25日]] : 伊萬里道路都市計畫決定。
* [[2009年|2009年]][[12月12日|12月12日]] : 二丈鹿家交流道 - 濱玉交流道間通車。
* [[2010年|2010年]][[3月12日|3月12日]] : 22時起廢除佐世保港交流道出入口的收費站，改移轉至佐世保大塔交流道併設的收費站。
* 2010年[[3月20日|3月20日]] : 相浦中里交流道 - 佐世保港交流道間通車。
* 2010年[[6月28日|6月28日]] : 佐世保道路與武雄佐世保道路實施高速公路免費化社會實驗。
* [[2011年|2011年]][[2月26日|2月26日]] : 與福岡都市高速5號線連接。
* 2011年[[6月19日|6月19日]] : 本日起佐世保道路、武雄佐世保道路的免費化社會實驗暫時凍結。

* 道路幅員
** 一般部: 暫定11.25[[公尺|公尺]]（完成 23.5 公尺）

* 車道數: 4車道（[[2車道高速公路|暫定2車道]]）

* 總長: 8.3 km
* 規格: 第1種第3級
* 設計速度: 80 km/h（速限為70 km/h）
<!--* 道路幅員: ○○m-->
* 車線幅員: 3.5 公尺
* 車道數: 計畫4車道（暫定2車道）
2010年3月20日全線通車。2010年時為暫定2車道，道路中央設置了車道分離用的防撞反光桿。

: 連接佐賀縣武雄市與長崎縣佐世保市的快速道路。由西日本高速公路所管理之全國路線網的一般收費道路。
* 起点: 佐賀縣武雄市東川登町大字袴野（[[武雄南交流道|武雄南交流道]]）
* 終点: 長崎縣佐世保市大塔町（佐世保大塔交流道）
* 總長: 22.1 km
* 規格: 第1種第3級
* 設計速度: 80 km/h（速限為70 km/h。但是佐世保大塔交流道 - [[佐世保三川内交流道|佐世保三川内交流道]]間的4車道路段除外）
* 道路幅員: 20.5 m
* 車道數: 4車道（暫定2車道）
