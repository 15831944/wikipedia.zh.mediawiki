'''西螺大橋'''，[[台灣|台灣]]公路橋樑，在[[臺灣日據時期|日據時]]稱'''濁水溪大橋'''<ref name="yunlin">{{cite web |url=http://tour.yunlin.gov.tw/guest/index-1.asp?m=99&m1=10&m2=42&sid=30 |title=【西螺】永遠的地標-西螺大橋 |work=雲林縣文化旅遊網 |author= |publisher=雲林縣政府文化處 |accessdate=2015-11-12 |language=zh-tw |date=2009-02-10 }}</ref>，位於[[雲林縣|雲林縣]]與[[彰化縣|彰化縣]]之間，橫跨[[濁水溪|濁水溪]]下游，以[[華倫式桁架橋|華倫式桁架橋]]設計，連接起南端之[[西螺鎮|西螺鎮]]、北端之[[溪州鄉|溪州鄉]]二地的交通。1952年完工時，西螺大橋是僅次於[[美國|美國]][[舊金山|舊金山]][[金門大橋|金門大橋]]的世界第二大橋，也是當時全台灣最長的橋樑，被譽為「遠東第一大橋」。

==諸元==
*橋身全長：1939.03[[公尺|公尺]]。
*橋面寬：7.32公尺。
*橋墩：32座。
*橋孔：31個。
*橋身架構：華倫式設計，以鋼鐵桁架做為梁，橋墩主要構成為[[水泥|水泥]]。

== 歷史 ==
===興建背景===
濁水溪為[[中台灣|台灣中部]]重要分界河。由於河面廣闊，16世紀歐洲人所繪之台灣地圖，甚至誤將台灣分為北、南兩島。長期以來，兩岸人民往來時必須長期倚賴竹筏等交通工具，若遇上溪水暴漲，則嚴重影響兩地交通，因此一直有興建交通橋樑之提議。

在日治時期，跨越濁水溪下游的橋梁僅有[[縱貫線_(鐵路)|縱貫鐵路]]之[[台鐵濁水溪橋|濁水溪橋]]，未有公路陸橋跨過。旅客渡濁水溪則須轉乘鐵路或竹筏。

===興建歷程===
1937年（昭和12年），發包興建濁水溪大橋<ref name="yunlin"/>，主要的建設部分為橋墩，方法是先以鐵條綁出橢圓形空心板模，接著灌漿抽沙來加重重量使板模下沉，如此反覆作業，使每座橋墩的基樁高度約二層樓高，32座橋墩於1941年完成，之後因戰爭鋼材挪作他用而中止建設，成為未成道路。[[臺灣戰後時期|戰後]][[國民政府|國民政府]]遷台，在[[美援|美援]]的資助下於1952年5月29日再度開工。

===竣工通車===
西螺大橋於1952年12月25日完工，1953年1月28日正式通車，雲林、彰化兩縣居民為橋名引起爭執，後來是沿用美國總統[[杜魯門|杜魯門]]在國會使用英譯名，才稱為西螺大橋<ref>{{Cite web |url=http://digiarch.sinica.edu.tw/content/subject/resource_content.jsp?id=4764 |title=西螺大橋的故事 |author=蘭文里 |publisher=中央研究院 數位典藏計畫 |accessdate=2012-12-23}}</ref>。

完工當時是僅次於[[美國|美國]][[舊金山|舊金山]][[金門大橋|金門大橋]]的世界第二大橋，也是當時遠東第一大橋<ref>{{Cite web |url=http://subtpg.tpg.gov.tw/web-life/taiwan/9904/9904-04.html |title=唯一出現在紙鈔上的橋樑—西螺大橋 |author=王派仁 |work=臺灣月刊雙月電子報99年4月號 |publisher=臺灣省政府 |accessdate=2012-12-23}}</ref>，亦使日治時期即定線的縱貫道路全線通車。另外，橋上亦併設[[糖業鐵路|糖業鐵路]]，形成鐵公路同行現象，為[[南北平行預備線|南北平行預備線]]最晚完工的一段，唯1979年以妨礙交通為由拆除<ref>[http://www7.www.gov.tw/twsix/tlMain_age.php?years=2&start=18 走過台灣一甲子 1950～1959]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}（黑白照片）</ref>。

===近況===
1994年，[[中沙大橋|中沙大橋]]與西螺大橋間的[[溪州大橋_(濁水溪)|溪州大橋]]完工後，西螺大橋轉為供小型車、機車、自行車通行的便橋，2000年曾被提議因此橋老舊應予拆除。在雲林縣與彰化縣政府的努力下，使西螺大橋轉型為觀光大橋，2004年11月19日，彰化縣及雲林縣政府均將西螺大橋列入該縣的歷史建築。
大甲媽每年農曆三月前往新港遶境進出雲林必走西螺大橋，是少數特例開放大型車走西螺大橋的特例。[[2013雲林農業博覽會|2013雲林農業博覽會]]期間獲選為百大亮點。<ref>[http://expo.yunlin.gov.tw/?page_id=2248 百大亮點 - 2013雲林農業博覽會] {{webarchive|url=https://web.archive.org/web/20140822073845/http://expo.yunlin.gov.tw/?page_id=2248 |date=2014-08-22 }}</ref>
