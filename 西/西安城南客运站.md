'''西安城南客运站'''是位于[[西安|西安]]市南郊[[朱雀大街|朱雀大街]]南段的汽车客运站。西安城南客运站于1999年建成，2000年元月正式营业。<ref name=info/>新站于2012年8月30日投入使用，位于朱雀大街与西安南绕城高速交汇处，占地约187亩。<ref> {{Cite web |url= http://www.sanqindaily.com/News/20120824/204766.html |title= 西安新城南客运站8月30日试运营 |author= 姬娜、刘星、王缦、郭勋图、王东启 |date= 2012-08-24 |publisher= 《[[三秦都市报|三秦都市报]]》 |language=  |accessdate= 2012-08-24 |quote=  }}</ref>

城南客运站的省际班线辐射[[四川|四川]]、[[湖北|湖北]]、[[湖南|湖南]]等省；省内班线覆盖[[汉中|汉中]]各县及[[安康|安康]]、[[洛南|洛南]]部分地区；市内班线主要有[[户县|户县]]、焦岱、汤峪、[[翠华山|翠华山]]、[[太乙宮|太乙宮]]等。<ref name=info>{{Cite web|url=http://www.cnkyz.com/bzjj.asp|title=简介|author=|date=|publisher=|language=中文|accessdate=2012-07-25|quote=西安城南客运站位于西安市南郊朱雀大街南段，占地面积35亩。于1999年建成，2000年元月正式营业，现有员工80余人，企业总资产5300万元。|deadurl=yes|archiveurl=https://web.archive.org/web/20120728040356/http://www.cnkyz.com/bzjj.asp|archivedate=2012年7月28日|df=}} </ref>

==城南客运站新站==
西安城南客运站新站属于陕西省省综合枢纽示范工程。建筑包括售票大厅、西侧的长途车次候车室，以及东侧的中短途车次候车室，同时设置母婴候车室，位于候车大厅西厅东南角；行李寄存采用自动存包柜，使用人脸识别系统；同时开通网上售票与自动售票机、自动闸机验票系统，更加方便旅客购票、乘车。新站在主体大楼下建设有1000平方米大小的地下车库，可容纳200多辆车。<ref>{{Cite web |url= http://hsb.hsw.cn/2012-07/27/content_8380103.htm |title= 即将投用的新城南客运站 看看哪些新鲜服务带给你方便(图) |author= 王警 |date= 2012-07-27 |publisher= 《[[华商报_(陕西)|华商报]]》 |language= 中文 |accessdate= 2012-07-28 |quote=  }}</ref><ref name=bus>{{Cite web |url= http://news.hsw.cn/system/2012/08/30/051444104.shtml |title= 西安城南客运站今投入运营 7趟公交车可抵达新站 |author= 张乃千 |date= 2012-08-30 |publisher= 《[[华商报_(陕西)|华商报]]》 |language= zh-hans |accessdate= 2012-08-31 |quote=  }}</ref><ref>{{Cite web |url= http://news.hsw.cn/system/2012/08/30/051444100.shtml |title= 西安城南客运站今投入运营 今昔对比差异大 |author= 张乃千|date= 2012-08-30 |publisher= 《[[华商报_(陕西)|华商报]]》 |language= zh-hans |accessdate= 2012-08-31 |quote=  }}</ref>
