'''西湖鄉'''（[[臺灣客家語|臺灣客家語]][[四縣腔|四縣腔]]：xiˊ fuˇ hiongˊ），是[[臺灣|臺灣]][[苗栗縣|苗栗縣]]西部的一個[[鄉_(中華民國)|鄉]]，位於中臺灣，東鄰[[苗栗市|苗栗市]]，北鄰[[後龍鎮_(台灣)|後龍鎮]]，西鄰[[通霄鎮|通霄鎮]]，南接[[銅鑼鄉_(台灣)|銅鑼鄉]]。居民以客家族群為主，約佔全鄉9成左右<ref>{{cite web | url=http://ebooks.lib.ntu.edu.tw/1_file/hakka/78/51272219271.pdf | title=各鄉鎮客家人口統計 |}}</ref>

== 詞源 ==
西湖地區舊稱「'''四湖'''」，1920年設立-{[[四湖庄_(苗栗郡)|四湖庄]]}-，隸屬[[新竹州|新竹州]][[苗栗郡|苗栗郡]]管轄，戰後設鄉時，因與雲林[[四湖鄉|四湖鄉]]同名而改稱'''西湖鄉'''。從出海口開始算起，遇到的第一處，稱做「頭湖」，依次為[[二湖村_(西湖鄉)|二湖]]、三湖、[[四湖村_(西湖鄉)|四湖]]、[[五湖村_(西湖鄉)|五湖]]，但沒有六、七、八湖，而另有九湖，按數字順序連成一串，除了「頭湖」現在屬於西北側的[[後龍鎮_(台灣)|後龍鎮]]福寧里、九湖屬於東南側的[[銅鑼鄉_(台灣)|銅鑼鄉]]九湖村外，其他四個「湖」，都在西湖鄉內。

== 歷史事蹟 ==
1713年（[[清|清]][[康熙|康熙]]五十二年），[[廣東|廣東]][[陸豐|陸豐]]、[[五華|五華]]等粵籍墾民入墾二湖、公館地區，為西湖鄉漢人開發之始。當時二湖尚有[[隘曆埔社|隘曆埔社]][[平埔|平埔]]原住民，後因墾民日漸增多，原住民被迫退入苗栗縣東部山區。[[雍正|雍正]]年間，廣東[[梅縣|梅縣]]人古蘭祥也進入二湖地區開墾。1732年（[[雍正|雍正]]十年），清廷開放對[[台灣|台灣]]移民的海禁，粵籍移民大量湧入<ref name="皮皮遊苗栗">[http://content.edu.tw/local/mauli/chushin/4/8-2.htm 《西湖鄉歷史沿革》，鄉土教材皮皮遊苗栗網站]{{zh-tw}}</ref>。

1752年（[[乾隆|乾隆]]十七年），廣東陸豐人劉恩寬入墾[[四湖村_(西湖鄉)|四湖]]一帶。乾隆中葉，廣東[[惠州|惠州]]人吳友宗率族人開拓高埔地區。
乾隆末年，廣東[[鎮平|鎮平]]人徐國興、[[揭陽|揭陽]]人彭祥麟等二人開拓三湖、四湖地區。[[嘉慶|嘉慶]]年間，廣東梅縣人黎姓家族
開拓四湖一帶荒埔<ref> [http://203.64.185.208/~yhshiau/miaoli/seahu.htm 《苗栗采風：西湖鄉》]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}{{zh-tw}}</ref>。
到了[[道光|道光]]、[[同治|同治]]年間，今日西湖鄉一帶幾已開拓完畢<ref name="皮皮遊苗栗"/>。

== 地理形態 ==
苗栗縣西湖鄉主要為[[丘陵|丘陵]]地形，林木茂盛，雙邊有小山，綠野平疇，聚落廣布。[[西湖溪|西湖溪]]由南至北貫穿全鄉。西湖鄉內氣候適宜，氣候屬於[[亞熱帶|亞熱帶]]，溫和多雨，雨季主要在5月至9月。住民以[[客家|客家]]族群為主。相對都市空氣較新鮮，鄉內設有[[自行車|自行車]][[車道|車道]]供人們自由歇息和玩樂，並有許多觀光景點。西湖鄉主要經濟為農業，稻米文旦生產是西湖農業大宗。目前有很多[[西湖鄉_(台灣)#特產|特產]]在本鄉。
