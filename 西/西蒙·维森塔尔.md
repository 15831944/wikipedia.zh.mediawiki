'''西蒙·維森塔爾'''（{{lang-de|'''Simon Wiesenthal'''}}，{{bd|1908年|12月31日|2005年|9月20日|catIdx=W}}）是[[猶太|猶太裔]][[奧地利|奧地利籍]][[建築工程|建築工程師]]、[[猶太人大屠殺|猶太人大屠殺]]的倖存者，是著名的{{en-link|纳粹猎人|Nazi hunter}}。他一生致力追查[[納粹|納粹]]黨人和蒐證，把他們送上法庭，要他們為[[戰爭罪行|戰爭罪行]]和[[非人道罪行|非人道罪行]]負責。[[西蒙·維森塔爾中心|西蒙·維森塔爾中心]]為紀念他而設立。

== 早年生活和二戰 ==
維森塔爾出生於[[奧匈帝國|奧匈帝國]]的[[布恰奇|布恰奇]]（Buczacz，現屬[[烏克蘭|烏克蘭]]的[[利維夫州|利維夫州]]）一個猶太商人家庭。由於對猶太學生人數設限，他被拒入讀勒沃夫理工大學（現利維夫理工學院），最終1932年他畢業於[[布拉格捷克理工大學|布拉格工業大學]]。1936年他與齊拉·繆勒 (Cyla Müller)結婚。

[[二戰|二戰]]初期，維森塔爾居於[[波蘭|波蘭]]的勒沃夫。基於[[苏德互不侵犯条约|苏德互不侵犯条约]]，該地成為[[蘇聯|蘇聯]]的[[軍事佔領|軍事佔領區]]。他的繼父和弟弟遭蘇聯的[[秘密警察|秘密警察]][[NKVD|NKVD]]探員殺害。維森塔爾被迫結束公司，改於工廠工作。當[[納粹德國|納粹德國]]於1941年入侵蘇聯時，他與家人被拘捕。

他與妻子首先被囚禁於雅諾夫斯基 (Janowski)集中營，被迫修建[[鐵路|鐵路]]。波蘭的地下抗德組織營救了他的妻子，以換取他所繪製的鐵路交點分布圖。她的金色頭髮讓她得以隱藏猶太人的身份；雖然被迫在[[萊茵河地|萊茵河地]]工作，但逃過屠殺厄運。

1943年，納粹開始屠殺營內猶太人，維森塔爾逃離集中營，但於當年六月被捉回。經過兩次[[自殺|自殺]]不遂，他與營中僅餘的34名囚犯被押解至波蘭和德國不同的集中營，最後抵達奧地利的毛陶森 (Mauthausen)集中營。他被釋放時，已前後被囚禁於12個集中營，其中5個是進行屠殺的集中營，多次差點被殺害。在大屠殺中，他與妻子共失去了89名親人。

== 納粹獵人 ==
1945年5月5日，維森塔爾被[[美軍|美軍]]救出。美軍發現他時，身高6英尺，卻只有45公斤重。當他回復健康，他隨即為[[紐倫堡審訊|紐倫堡審訊]]蒐證。1947年，他與其餘30個志願者於奧地利[[林茨|林茨]]成立[[猶太人檔案中心|猶太人檔案中心]]，為日後的審訊蒐集資料。可惜，[[美國|美國]]和蘇聯對進一步審判戰爭罪行失去興趣，志願者紛紛離去。維森塔爾全職為受二戰影響的人提供協助，同時繼續蒐證工作。

1962年，[[阿道夫·艾希曼|阿道夫·艾希曼]]被處死後，維森塔爾重開猶太人檔案中心，繼續蒐證，把納粹戰犯送上法庭。其中最為人熟知的是協助成功拘捕卡爾·修伯爾鮑爾（Karl Silberbauer）。他是當時拘捕[[安妮·法蘭克|安妮·法蘭克]]的[[盖世太保|盖世太保]]軍官。根據修伯爾鮑爾的口供，推翻了《安妮的日記》是子虛烏有的指責。他成功找出另外9名在逃納粹戰犯，並以謀殺猶太人的罪名送往[[西德|西德]]的法庭受審。他協助拘捕[[弗蘭茨·施坦格爾|弗蘭茨·施坦格爾]]（Franz Stangl，[[特雷布林卡滅絕營|特雷布林卡滅絕營]]和[[索比布爾集中營|索比布爾集中營]]的指揮官）和[[赫爾明娜·布勞恩施泰因納爾|赫爾明娜·布勞恩施泰因納爾]]（Hermine Braunsteiner，曾任納粹集中營的女性軍官，戰後居於[[長島|長島]]。她曾下令虐待和殺害大量在[[邁丹尼克集中營|邁丹尼克集中營]]的小孩。）

== 西蒙·維森塔爾中心 ==
1977年, 一個猶太人大屠殺紀念組織命名為[[西蒙·維森塔爾中心|西蒙·維森塔爾中心]]。中心提倡人們關注[[反猶太主義|反猶太主義]]、監察[[新納粹|新納粹]]團體、於[[洛杉磯|洛杉磯]]和[[耶路撒冷|耶路撒冷]]舉辦展覽會、以及把納粹戰犯送上法庭受審。

== 奧國政治和晚年 ==
1970年代，維森塔爾涉足奧地利政壇，始於他指控由[[SPÖ|社會黨]]首相[[布鲁诺·克赖斯基|布鲁诺·克赖斯基]]新組成的內閣中有成員曾經是納粹黨人。本身是猶太人的[[克赖斯基|克赖斯基]]攻擊維森塔爾是「用石頭砸自己腳的人」（ ''Nestbeschmutzer''）。其實，奧地利要數十年時間才能接受自己在納粹罪行中扮演的角色，期間，維森塔爾常被忽略，甚至被揶揄。

多年來，維森塔爾接過多次死亡恐嚇。1982年，德國和奧地利的新納粹份子在他[[維也納|維也納]]的家外引爆炸彈。

維森塔爾90歲後仍常留在位於維也納的猶太人檔案中心的細小辦公室中工作。

[[2003年4月|2003年4月]]，維森塔爾宣布退休，聲稱他已找到他要找的屠殺者，又說：「我比他們活得久。如果仍有漏網之魚，他們已衰老得不能接受審判。我的工作已經完成。」根據維森塔爾，最後一個仍在生的奧地利主要戰犯是阿羅伊斯·布魯納爾（Alois Brunner），他是阿道夫·艾希曼的心腹，相信他藏身於[[敘利亞|敘利亞]]，受到[[巴沙尔·阿萨德|巴沙尔·阿萨德]]政權的保護。

維森塔爾的晚年在維也納渡過。陪伴他的妻子於2003年11月10日因病逝世，享年95歲。維森塔爾於2005年9月20日在睡夢中離世。

== 批評 ==
維森塔爾指非猶太人在大屠殺中的死亡人數達500萬，新納粹分子Peter Novick和Yehuda Bauer批評數字是維森塔爾捏造。{{ref|fab}}

另一名納粹獵人Tuviah Friedman批評維森塔爾，指他在利用艾希曼的事件中吹噓自己的能力和說謊，並在事件中獲得厚利。[https://web.archive.org/web/20051226133738/http://motlc.specialcol.wiesenthal.com/instdoc/d09c07/iss21z3.html],[https://web.archive.org/web/20051203223445/http://motlc.specialcol.wiesenthal.com/instdoc/d09c07/iss12z3.html],[https://web.archive.org/web/20051203223641/http://motlc.specialcol.wiesenthal.com/instdoc/d09c07/iss13z3.html],[https://web.archive.org/web/20051203223531/http://motlc.specialcol.wiesenthal.com/instdoc/d09c07/iss15z3.html],[https://web.archive.org/web/20051203223349/http://motlc.specialcol.wiesenthal.com/instdoc/d09c07/iss17z3.html]

[[美國特別調查處|美國特別調查處]]處長 Eli Rosenbaum 在他關於 Kurt Waldheim 事件的書 ''Betrayal: The Untold Story of the Kurt Waldheim Investigation and Coverup''中寫道：

:「總括來說，維森塔爾在重大的納粹戰犯案時，如 Mengele、Martin Bormann 和艾希曼案，表現出愚拙、誇大和自誇。」

他向維森塔爾的自傳作家形容維森塔爾為「天生的說謊者」。[http://hgs.oxfordjournals.org/cgi/reprint/11/2/256?maxtoshow=%20&HITS=10&hits=10&RESULTFORMAT=%20&fulltext=wiesenthal&andorexactfulltext=and&searchid=1120901548741_96&stored_search=%20&FIRSTINDEX=0&sortspec=relevance&resourcetype=1&journalcode=holgen]

== 媒體形象 ==
作家[[艾拉·萊文|艾拉·萊文]]以維森塔爾作為小說《巴西來的男孩們》角色以斯拉·利柏文的藍本。在弗雷德里克·福赛斯的小說《敖德萨档案》中，維森塔爾曾出現，向一名德國記者提供尋找納粹戰犯的線索。電影方面，[[班·金斯利|班·金斯利]]在《戰犯就在你身邊：西蒙·維森塔爾傳》一片中飾演主角西蒙·維森塔爾。
