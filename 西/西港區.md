'''西港區'''位於[[臺灣|臺灣]][[臺南市|臺南市]]西部，北接[[佳里區|佳里區]]，西連[[七股區|七股區]]，南鄰[[安定區_(台灣)|安定區]]、[[安南區|安南區]]，東北則鄰[[麻豆區|麻豆區]]。西港地處[[曾文溪|曾文溪]]北岸，海拔五公尺。昔為[[臺江內海|臺江內海]]內港，後來因臺江內海淤塞陸化，失去港口機能，轉而變成附近農產品交易中心。

== 地名由來 ==
西港區的名稱來自過去[[臺江內海|臺江內海]]港口之一的「西港仔港」與港口周邊的市街「西港仔街」，後人將「西港仔」簡化成「西港」{{r|縣|page=577}}{{r|刈香|page=220}}。至於「西港仔」一名的由來，則有位在「直加弄港」（今[[安定區_(台灣)|安定區]]）西邊或後營（今西港區後營里、營西里）西邊等說法{{r|縣|page=577}}。

== 歷史沿革 ==
西港區在過去大部分是臺江內海的一部分{{r|縣|page=577}}，剩下的陸地則可能是[[西拉雅族|西拉雅族]][[蕭壠社|蕭壠社]]的地盤{{r|刈香|page=220}}{{r|縣|page=577}}。明鄭時期，西港一地隸屬[[天興州|天興州]]（縣）[[永定里_(堡里)|永定里]]，據說鄭軍即在後營地區（後營里、營西里）屯墾{{r|刈香|page=220}}{{r|縣|page=577}}。

進入清朝時期後，永定里在康熙廿二年（1683年）之後改稱「安定里」，隸屬[[諸羅縣|諸羅縣]]{{r|刈香|page=220}}{{r|縣|page=577}}。康熙六十年（1721年）前後分設「安定里東保」和「安定里西保」，迨「[[朱一貴|朱一貴事件]]」時，因地方民番助戰有功而興建向忠亭，「安定里東保」乃改稱「向忠里」{{r|刈香|page=220}}{{r|縣|page=577、578}}。並設有碑文紀念，現存放於西港玉勅慶安宮廟旁。

道光三年（1823年）灣裡溪（[[曾文溪|曾文溪]]）在暴風雨後改道，將大量泥沙帶入臺江內海，使得臺江內海逐漸陸化{{r|縣|page=577}}。道光十五年（1835年）[[嘉義縣_(清朝)|嘉義縣]]與[[臺灣縣_(1684年-1887年)|臺灣縣]]的縣界調整，以改道後的「曾文溪」為新縣界{{r|縣|page=578}}。該年也設立了[[西港仔堡|西港仔保]]{{r|刈香|page=220}}。

進入日治時期後，西港仔保改為西港仔堡，在明治卅四年（1901年）隸屬於[[鹽水港廳|鹽水港廳]]，明治四十二年（1909年）行政區劃調整後改隸[[臺南廳|臺南廳]]{{r|刈香|page=220}}{{r|縣|page=578}}。大正九年（1920年）行政區劃改為州郡街庄制，今西港區一帶設'''-{[[西港庄|西港庄]]}-'''，屬[[臺南州|臺南州]][[北門郡|北門郡]]{{r|刈香|page=220}}{{r|縣|page=578}}。

二次大戰後，民國卅十五年（1946年）將西港庄改為西港鄉，隸屬臺南縣北門區{{r|刈香|page=220}}{{r|縣|page=578}}。「[[縣轄區_(中華民國)|縣轄區]]」裁撤後，直屬於縣。2010年12月25日因行政區劃改制，改為'''西港區'''。

== 行政區劃 ==
{{see also|西港庄}}
西港區前身「西港鄉」設立時，設有西港、新復、永樂、中東、南海、竹林、劉厝、八份、三樂、檨林、後營、營西、金砂等13個村{{r|縣|page=578}}。1978年，中東村併入南海村，三樂村與八份村合併成港東村{{r|縣|page=578}}。1988年，西港村分割出慶安村，總村數成為12村{{r|縣|page=578}}。2010年臺南縣市合併成直轄市後，原本的村改制為里。

== 旅遊 ==
*曾橋晚照：曾橋晚照就是西港大橋上夕陽落下的景像。而往昔與車輛奔行於西港大橋的[[南北平行預備線|南北平行預備線]][[臺灣糖業鐵路|臺糖小火車]]，亦隨著臺糖停止製糖而停行了。如今曾橋晚照的美麗，只留存於照片中了。
*[[西港慶安宮|西港慶安宮]]：西港區的大廟，又稱做「西港仔廟」{{r|刈香|page=221}}。該廟定期舉辦的[[西港香|西港香]]是[[南瀛五大香|南瀛五大香]]之一，三年一醮（丑、辰、未、戌年），號稱臺灣第一香。
