'''西南医院'''（'''中国人民解放军陆军军医大学第一附属医院'''、'''中国人民解放军陆军军医大学西南临床医学院'''），位于[[中国|中国]][[重庆市|重庆市]][[沙坪坝区|沙坪坝区]]高滩岩正街30号，隶属[[中国人民解放军陆军军医大学|中国人民解放军陆军军医大学]]<ref name=jeshao/>。

==简介==
西南医院的前身是'''国民政府中央医院'''，1929年建于[[南京|南京]]，1941年迁到[[重庆|重庆]]<ref name=jeshao>{{Cite web |url = http://www.xnyy.cn/news_view.aspx?id=4124 |title =医院介绍  |author =  |date = 2017-01-03 |publisher =西南医院  |language =  |accessdate =  |quote =  }}</ref>。1949年11月30日重庆解放，[[中国人民解放军|中国人民解放军]]接管中央医院。1950年10月15日，西南军区后勤卫生部秘字第139号令，决定正式将中央医院改编为'''西南医院'''。1950年12月3日举行命名典礼。刚命名的西南医院由张方贤、梁玉堂任副院长。1950年11月，归隶[[西南军政委员会|西南军政委员会]]卫生部领导<ref name=luyi>{{Cite web|url =http://jiangsu.china.com.cn/html/2016/cqnews_0202/3852941.html|title =泸医改名之争再曝历史证据 西南医科大学就在重庆|author =|date =2016-02-02|publisher =中国网|language =|accessdate =|quote =}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1953年5月26日，根据[[中共中央西南局|中共中央西南局]]的决定，西南军区后勤卫生部下达0017号指示，决定自1953年6月1日起将西南医院改隶中国人民解放军第七军医大学领导，西南第一护士学校附属西南医院领导不变。改隶中国人民解放军第七军医大学领导后，该医院沿称西南医院，相当于师级单位<ref name=luyi/>。

1955年6月，改称'''中国人民解放军第七军医大学第一附属医院'''（对外仍然称'''西南医院'''）。1956年8月，陆军第四十八医院并入第七军医大学，与第七军医大学第一附属医院合并，经调整改设为'''中国人民解放军第七军医大学附属内科医院'''和'''中国人民解放军第七军医大学附属外科医院'''。1958年1月，重新合并为'''中国人民解放军第七军医大学第一附属医院'''<ref name=luyi/>。第七军医大学更名为第三军医大学后，该医院随之更名为'''中国人民解放军第三军医大学第一附属医院'''。2017年，更名'''中国人民解放军陆军军医大学第一附属医院'''。

西南医院在烧伤、感染病、肝胆、微创外科、整形外科、医学检验、康复理疗等学科的综合实力位居[[中华人民共和国|中华人民共和国]]国内前列<ref name=xunli>{{Cite web |url = http://www.81.cn/jwgz/2017-06/04/content_7627632.htm |title = 2017军校巡礼第六站：第三军医大学（附报考指南） |author =  |date = 2017-06-04  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

2002年，通过英国皇家标准协会（BSI）的ISO9001质量管理体系认证。2007年，获评为全国“首批数字化试点示范医院”。2010年，通过国际SIDCER伦理认证。2012年，获评为首批研究型医院建设试点医院。2014年，获评为军队研究型医院建设示范单位<ref name=jeshao/>。
