'''《西游记后传》'''是导演[[曹榮_(演員)|曹荣]]以《[[西游记|西游记]]》的故事为背景，拍摄的一部30集古装神话电视连续剧。主要讲述唐僧师徒取经归来后，得知[[如来|如来]][[圆寂|圆寂]]，魔头无天借此机会想统治三界，最终被孙悟空舍身除掉的故事。本剧于2000年在中国各地电视台播出。

== 相关 ==
曹荣在本剧中自导自演，在这之前他担任了《[[西游记续集|西游记续集]]》的动作导演<ref name="general">{{cite news|url=http://www.eladies.com.cn/amusement/news/amusenews/2000-02-21/19971.shtml|title=曹荣自导自演《西游记后传》|date=2002-02-21|publisher=新浪|accessdate=2015-02-22|deadurl=yes|archiveurl=https://web.archive.org/web/20090914154319/http://www.eladies.com.cn/amusement/news/amusenews/2000-02-21/19971.shtml|archivedate=2009年9月14日|df=}}</ref>。

本剧在南京、云南、江苏等地播出后获得了较高收视率，但在北京的收视并不是太好<ref>{{cite news|url=http://ent.sina.com.cn/v/2001-07-28/51676.html|title=新版“美猴王”曹荣接受专访谈《西游记后传》|date=2001-07-28|publisher=新浪|accessdate=2015-02-22}}</ref>。

2015年，编剧钱雁秋以本剧故事改编而成的电视剧《[[石敢当之雄峙天东|石敢当之雄峙天东]]》播出。

== 评价 ==
导演曹荣表示“打斗场数和特技量比补集增加一倍”<ref name="general"/>而实际上，电视剧中几乎所有的打斗部分都要连续重复播放三遍（包括正序，反序播放）以上，一些特写镜头还要从几种不同的拍摄角度来展示。这一点使观众普遍不满，但是2016年acfun上已经有纯剧情向无鬼畜版本出现<ref>{{Cite web|url=http://www.acfun.cn/v/ac3001464|title=《西游记后传》无鬼畜版 纯剧情向 合集 - AcFun弹幕视频网 - 认真你就输啦 (?ω?)ノ- ( ゜- ゜)つロ|accessdate=2017-08-04|work=www.acfun.cn|language=zh}}</ref>。整个剧中颠覆了西游记的理念，从顽劣不堪、任性调皮到正义勇敢、舍己为人的孙悟空，从丑陋异常到英俊潇洒的猪八戒，从佛学理论到领略了武学真谛的唐三藏,从胆小弱势到后来担负重担,勇猛冲锋的沙悟净。此剧的片头曲《我欲成仙》和片尾曲《相思》广受好评。
