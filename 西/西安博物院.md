'''西安博物院'''是一座位于[[中华人民共和国|中华人民共和国]][[陕西省|陕西省]][[西安市|西安市]]友谊西路的[[博物院|博物院]]。西安博物院总投资2.2亿元，占地面积245亩，于2007年5月18日“国际博物馆日”正式对外开放。<ref name=about>{{Cite web |url= http://www.xabwy.com/SitePage.aspx?ID=145 |title= 西博介绍 |author=  |date=  |publisher= 西安博物院 |language= zh-hans |accessdate= 2012年12月4日 |quote=  }}</ref>整个博物院由博物馆区、唐[[小雁塔|小雁塔]][[荐福寺|荐福寺]]历史名胜区、园林游览区三部分组成。

2012年，西安博物院被[[国家文物局|国家文物局]]核定为第二批[[国家一级博物馆|国家一级博物馆]]<ref>{{Cite web |url = http://www.gov.cn/zwgk/2012-12/07/content_2285163.htm |title =关于发布第二批国家一级博物馆名单的通知（文物博函〔2012〕2237号）  |author =  |date =2012-12-07  |publisher = 中国政府网 |language =  |accessdate =  |quote =  }}</ref>。

==文物展馆==
文物展馆占地16000余平方米，陈列面积5000余平方米，展馆由[[中国工程院院士|中国工程院院士]][[张锦秋|张锦秋]]主持设计，馆藏13万件文物，其中国家三级以上的珍贵文物有14400多件。<ref name=about/>

==荐福寺==
{{main|荐福寺|小雁塔}}
荐福寺是唐代著名的皇家寺院，毁于唐末战乱，唯存寺内佛塔——小雁塔。<ref name=about/>[[小雁塔|小雁塔]]原名荐福寺塔，高约43.4米，底层边长11.38米，保持了唐代初修时的风貌，1961年被国务院公布为[[全国重点文物保护单位|全国重点文物保护单位]]。<ref name=about/>唐朝天文学家[[一行|一行]]曾在荐福寺随[[善无畏|善无畏]]和[[金刚智|金刚智]]修行。<ref>{{Cite web|url=http://epaper.xawb.com/xawb/html/2012-10/24/content_153859.htm|title=僧一行在长安|author=朱文杰|date=|publisher=《[[西安晚报|西安晚报]]》|language=zh-hans|accessdate=2012年12月4日|quote=}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>
