'''西直门交通枢纽'''位于[[北京市|北京市]][[西城区|西城区]]，[[西直门桥|西直门桥]]西北角，是一个综合性大型客运交通枢纽，也是北京市区主要交通枢纽之一<ref>{{cite news|url=http://ccnews.people.com.cn/GB/12643519.html|title=北京13大交通枢纽将在2013年全部建成|date=2010-09-06|accessdate=2012-09-05}}</ref>。枢纽主要服务于3条城市轨道交通（[[西直门站|西直门站]]）内部的换乘，同时引入国家铁路（[[北京北站|北京北站]]）、路面公交，以及[[出租车|出租车]]等其他交通方式。与西直门交通枢纽配套建设的写字楼与商业零售建筑称作'''西环广场'''<ref name="xinhuanet20050329">{{cite news|url=http://www.xinhuanet.com/chinanews/2005-03/29/content_3963031.htm|title=西直门交通枢纽西区建设完工 东区将作相应调整|date=2005-03-29|accessdate=2012-09-05}}</ref>，其主体为四座塔楼，其中三个较高的塔楼的立面呈椭圆形，为[[西直门|西直门]]地区的[[地标|地标建筑]]<ref name="cng">{{cite journal|author=朱文轶|journal=中国国家地理|issue=8|pages=95-105|title=北京城：50年巨变|date=2008年8月}}</ref>。

== 位置和布局 ==
西直门交通枢纽位于[[北京二环路|西二环路]]外，[[西直门外大街|西直门外大街]]北侧，西至[[高梁桥路|高梁桥路]]，北至[[南长河|南长河]]（此段为[[盖板河|盖板河]]），规划总用地面积5.99公顷，总建筑面积26.4万平方米，其中商业和办公区建筑面积18.2万平方米，交通建筑面积6.7万平方米。<ref name="WA">{{cite journal|title=北京西直门交通枢纽设计研究|author=李兴钢 苗茁|journal=世界建筑|year=2008|issue=8}}</ref>

交通枢纽分为东、中、西三个区域。西区主要为配套商业开发，亦即引言中所述的地标建筑——西环广场；中区为[[北京地铁13号线|地铁13号线]]车站；东区北部为国铁[[北京北站|北京北站]]，中部为地下换乘大厅和公交始发站，南部通过通道连接1980年代建成的[[北京地铁2号线|地铁2号线]]、[[北京地铁4号线|4号线]]车站。<ref>{{cite news|url=http://www.bj.xinhuanet.com/bjpd_sdzx/2006-08/25/content_7876528.htm|title=东直门交通枢纽七成客流将实现地下换乘|date=2006-08-25|accessdate=2012-09-05|deadurl=yes|archiveurl=https://web.archive.org/web/20060910195941/http://www.bj.xinhuanet.com/bjpd_sdzx/2006-08/25/content_7876528.htm|archivedate=2006-09-10}}</ref>

== 设计 ==
整个设计围绕东区的换乘厅展开，换乘大厅位于地下一层，中区三层的13号线站台通过一个连续的通道与换乘大厅直接连接，再通过换乘大厅连接2、4号线车站，以避免出站换乘；同时换乘大厅也与北京北站地下一层相连。由于枢纽内只有两路公交车从北京北站前经过，其他公交车停在高梁桥路西侧，所以在西区二层设置了换乘通道，以方便乘客由公交转乘地铁13号线<ref name="WA" />，但这种公交车站的布置方式仍然造成了换乘上的麻烦<ref>{{cite journal|url=http://www.cnki.com.cn/Article/CJFDTOTAL-CSWT200906008.htm|title=西直门地区交通换乘的主要问题|journal=城市问题|author=李强 隋洪鑫 王运静 张珩|year=2009|issue=6|accessdate=2012-09-05}}</ref>。

西区的四个塔楼中，最北端的一个是北京地铁13号线的控制中心，与13号线统一设计建造，而南部的三座则是写字楼，之间以[[裙房|裙房]]相连，用作商业开发。南部三座塔楼没有设置标准层，各层结构和平面设计均不相同，以实现椭圆形立面的效果。三个塔楼的南侧都设置了[[遮阳|遮阳]]用的[[百叶|百叶]]。<ref name="WA" />

== 建造 ==
[[File:Beijing_Building.JPG|thumb]]
西直门交通枢纽各部分的时间跨度很长。枢纽建筑于2001年开工，2002年9月地铁13号线车站就开始投入使用，西区的商业和写字楼于2005年9月交付使用<ref name="xinhuanet20050329" />，枢纽主体2008年5月竣工<ref name="WA" />，用于地铁换乘的通道也于同年6月开通<ref>{{cite news|url=http://sports.cctv.com/20080610/100759.shtml|title=西直门换乘大厅奥运前双向通行 换乘时间省一半|date=2008-06-10|accessdate=2012-09-05}}</ref>，但是地下混合换乘大厅直至2011年才开通运营<ref>{{cite web|url=http://www.bii.com.cn/705-1781-3970.aspx|title=西直门交通枢纽混合换乘大厅正式开通|date=2011-09-29|accessdate=2012-09-05}}</ref>。单独改造的北京北站则于2008年1月开工<ref>{{cite news|url=http://www.chinanews.com/cj/xfsh/news/2008/07-01/1297996.shtml|title=新北京北站计划在8月1日部分投入使用|date=2008-07-01|accessdate=2012-09-05}}</ref>，次年1月投入使用<ref>{{cite news|url=http://www.chinanews.com/cj/gncj/news/2009/01-16/1531198.shtml|title=北京北站新站房启用 发送旅客能力是老北站5倍|date=2009-01-16|accessdate=2012-09-05}}</ref>。
