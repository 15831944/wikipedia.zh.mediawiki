'''西安站'''位于[[中国|中国]][[陕西省|陕西省]][[西安市|西安市]][[新城区_(西安市)|新城区]]环城北路[[解放门_(西安)|解放门]]外，[[陇海铁路|陇海铁路]]1077.7公里处（站中心），始建于1934年（民国23年）12月，1935年6月启用，现为[[中国铁路西安局集团有限公司|中国铁路西安局集团有限公司]]管辖的[[特等站_(中铁)|特等站]]。经过铁路有[[陇海铁路|陇海铁路]]、[[西康铁路|西康铁路]]、[[宁西铁路|宁西铁路]]及[[侯西铁路|侯西铁路]]。

== 历史 ==
* 1934年12月：西安站始建。
* 1935年6月：西安站建成启用，當時为客货運[[一等站_(中鐵)|一等站]]。
* 1937年：西安站更名为[[长安站|长安车站]]。
* 1952年1月1日：西安站恢复原來名称。
* 1984年：西安站開始改建工程，加建主站房。
* 1985年12月31日：西安站改建工程竣工。
* 1988年8月1日：西安站由[[一等站_(中鐵)|一等站]]升为[[特等站_(中鐵)|特等站]]。
* 1990年10月1日：西安站东配楼售票厅及办公楼竣工。
* 2010年2月6日，为配合[[郑西高铁|郑西高铁]]开通，西安站首次开行高速动车组列车<ref>{{Cite web |url=http://news.qq.com/a/20100201/002034.htm|title= 郑西高铁车票西安首发 1小时内售出260张 |author= |date= 2010年2月1日 |publisher= 华商网 |language= zh-hans |accessdate=  |quote=  }}</ref>，但在2011年1月11日，由于[[西安北站|西安北站]]部分启用，郑西高铁所有列车均改为西安北站始发终到。
* 2013年12月28日，由于[[西宝客运专线|西宝客运专线]]开通，西安至宝鸡的动车组列车改为在西安北站办理客运业务，西安站不再开行动车组列车。<ref>{{Cite web|url=http://news.house365.com/gbk/xaestate/system/2013/12/30/023211408.html|title=14对西宝高铁每天公交化运营 比一比看咋去宝鸡最划算-西安365地产家居网|accessdate=2017-08-09|last=江苏三六五网络有限公司|work=news.house365.com}}</ref>
*2017年10月7日23时33分，西安站首次开行通过网友参与[[众筹|众筹]]所得的非图定列车（即开往{{站|榆林}}的K8188次列车）。该列车是中国大陆第一个众筹火车项目，共有122名网友参与了众筹。而返程班次K8187次则于10月8日开行<ref>{{Cite web |url=http://news.163.com/17/1010/01/D0BLN2PF000187VI.html|title= 西安开通全国首列“众筹火车” |author= |date= 2017-10-10 |publisher= 北京青年报 |language= zh-hans |accessdate=  |quote=  }}</ref>。

== 車站設施 ==
[[File:Xian_Station_Hall.jpg|缩略图]]
西安站建有候车站台6座，共有13条到发线。其中一至四站台为低站台，五、六站台为高站台。第六站台为09年新建，为动车组专用站台，现在该站台主要承担短编组的管内通勤和普客列车的到发。

== 使用情况 ==
西安站是[[陇海铁路|陇海铁路]]、[[西康铁路|西康铁路]]及[[宁西铁路|宁西铁路]]的[[特等站_(中鐵)|特等站]]。

== 未來發展 ==
西安站计划于2014秋季开始为期5年的扩建工程，扩建计划投资约八十多亿元人民币。<ref name=rebuild>{{Cite web |url= http://hsb.hsw.cn/2014-07/28/content_8544482.htm |title= 西安火车站改扩建 向北衔接大明宫丹凤门(图) |author=  |date= 2014年7月28日 |publisher= 华商网 |language= zh-hans |accessdate= 2014年7月28日 |quote=  }}</ref>扩建工程包括：
* 新建火车站北广场、北侧新建北站房、高架候车室，和[[大明宫|大明宫]][[丹凤门|丹凤门]]通过候车广场衔接在一起；
* 由6站台9股道扩建到9站台18股道；
* 扩建出站地下通道，延伸到新建的西安火车站北广场，还在车站的东侧再新建一条出站地下通道，贯穿车站新建的南北广场；<ref>{{Cite web |url= http://sx.sina.com.cn/news/m/2012-07-13/07293105.html |title= 西安火车站改扩建拟明年动工 2016年前后建成 |author= 张毅伟,焦阳 |date= 2012-07-13 |publisher= 《三秦都市报》 |language= zh-hans |accessdate= 2012.0724 |quote=  }}</ref>
* 西安东站改建为客车整备检修基地；
* 西安站至灞桥段增建双线，增强客车出入能力。
扩建完成后，西安站有效候车面积将由7000平方米提升到3万平方米以上。发送旅客从现在的日均7.7万人，提升到10万人左右。<ref name=rebuild/>
