'''西定高架道路'''，又稱'''西定快速道路'''，位於[[臺灣|臺灣]][[基隆市|基隆市]]，為基隆市內的[[高架道路|高架道路]]之一，全線於2006年9月20日通車<ref>{{cite web | url = http://udn.com/NEWS/DOMESTIC/DOM5/8953221.shtml  | title = 西定河高架道通車 出口打結  | date = 2006-09-21}}</ref>。

== 簡介 ==
本高架道路全長'''2.3公里'''，全線皆為'''雙向各1車道'''（中央無[[分隔島|分隔島]]，為雙黃實線），最高速限'''時速40公里'''。

起點位於[[中和路_(基隆市)|中和路]]，全線構築於[[西定河|西定河]]之上，採高架方式興建，於[[新西街_(基隆市)|新西街]]設置一[[平交路口|平交路口]]，終點位於[[成功路_(基隆市)|成功一路]]、[[中山路_(基隆市)|中山一路]]、[[安樂路|安樂路]]一段交叉路口。

完工通車後連接市區及[[內木山|內木山]]一帶新社區，紓解[[西定路|西定路]]一帶交通壅塞問題，促進[[安樂區|安樂區]]、[[中山區_(基隆市)|中山區]]繁榮發展。

== 注意事項 ==
本高架道路僅限小型汽車與[[大型重型機車|大型重型機車]]行駛，大型客車或貨車、一般機車、自行車及行人皆不可通行。
