'''西灣河'''（英語：Sai Wan Ho）是香港[[東區區議會|東區區議會]]下轄的[[選區|選區]]，代號'''C05'''，成立於1982年，現任區議員為[[工黨_(香港)|工黨]]副主席麥德正。

==範圍==
選區範圍為西灣河部分地方，例如[[欣景花園|欣景花園]]和[[逸濤灣|逸濤灣]]等等，與其相鄰的選區有[[太古城西|太古城西]]、[[鯉景灣_(選區)|鯉景灣]]、[[愛秩序灣_(選區)|愛秩序灣]]、[[康山_(選區)|康山]]、[[興東_(選區)|興東]]和[[下耀東|下耀東]]<ref>[http://www.elections.gov.hk/dc2011/maps/dc2011c.pdf]</ref>。

==沿革==
西灣河選區首次出現於[[1982年香港區議會選舉|1982年香港區議會選舉]]，由[[香港公民協會|香港公民協會]][[張建忠|張建忠]]和兩名無黨派人士趙志輝、吳炳新爭奪，最後[[張建忠|張建忠]]勝出，並於[[1985年香港區議會選舉|1985年香港區議會選舉]]擊敗[[李景華|李景華]]連任，但於[[1988年香港區議會選舉|1988年香港區議會選舉]]被[[傅元章|傅元章]]擊敗。

[[1991年區議會選舉|1991年區議會選舉]]，原於黃大仙區任委任議員的[[陳啟榮|陳啟榮]]，轉至該區參選。陳多年來擔任[[太安樓|太安樓]]業主立案法團的法律顧問，其政黨[[自民聯|自民聯]]在東區的勢力亦很大，但傅仍能以1,686票大勝陳的810票。

[[1994年區議會選舉|1994年區議會選舉]]，當時[[傅元章|傅元章]]改往[[鯉景灣_(選區)|鯉景灣]]參選。這次有三位無黨派人士江澤濠、姚蘭英和姚述法爭奪，最後江澤濠勝出。

[[1999年區議會選舉|1999年區議會選舉]]，當時人口估算約19,237人<ref>{{cite web |url=http://www.elections.gov.hk/elections/dcelect99/cg12.htm |title=存档副本 |accessdate=2012-04-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20111229025617/http://www.elections.gov.hk/elections/dcelect99/cg12.htm |archivedate=2011-12-29 }}</ref>，其中有8,304位是選民<ref>{{cite web |url=http://www.elections.gov.hk/elections/dcelect99/cg12.htm |title=存档副本 |accessdate=2012-04-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20111229025617/http://www.elections.gov.hk/elections/dcelect99/cg12.htm |archivedate=2011-12-29 }}</ref>。今次[[江澤濠|江澤濠]]自動當選。

[[2003年區議會選舉|2003年區議會選舉]]，當時人口估算約18,307人<ref>{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/summary/summary_e.html |title=存档副本 |accessdate=2012-04-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20111108185626/http://www.elections.gov.hk/elections/dc2003/tc_chi/summary/summary_e.html |archivedate=2011-11-08 }}</ref>，其中有7,944位是選民<ref>{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/eb/eb_num_e.html |title=存档副本 |accessdate=2012-04-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20110115012307/http://www.elections.gov.hk/elections/dc2003/tc_chi/eb/eb_num_e.html |archivedate=2011-01-15 }}</ref>。今次[[江澤濠|江澤濠]]自動當選。

[[2007年區議會選舉|2007年區議會選舉]]，當時人口估算約18,263人<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/summaryc.html |title=存档副本 |accessdate=2011-11-08 |0=2008-05-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20080501203936/http://www.elections.gov.hk/dc2007/chi/summaryc.html |archivedate=2008-05-01 }}</ref>，其中有8,519位是選民<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/num_electors_eastern.html |title=存档副本 |accessdate=2015-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20150621034533/http://www.elections.gov.hk/dc2007/chi/num_electors_eastern.html |archivedate=2015-06-21 }}</ref>，今次[[江澤濠|江澤濠]]自動當選。

[[2011年區議會選舉|2011年區議會選舉]]，當時人口估算約19,454人<ref>[http://www.elections.gov.hk/dc2011/chi/summaryc.html]</ref>，其中有8,707位是選民<ref>[http://www.elections.gov.hk/dc2011/chi/ebriefsc.html]</ref>，江澤濠與[[泛民區選聯盟|泛民區選聯盟]][[陳顯中|陳顯中]]爭奪，江澤濠以1,826票勝出，但陳顯中提出選舉呈請，指江澤濠違規於中國銀行的閘門張貼海報，令此區有機會重選，不過最後並沒有發生。

[[2015年區議會選舉|2015年區議會選舉]]，預計人口(截至2015年6月30日)約19,482人，2015年正式選民登記冊是8,691人，現任區議員江澤濠自1994年當選以來，該選區由回歸起首三屆皆是江澤濠自動當選。上屆江澤濠僅以三百多票之差險勝泛民區選聯盟推薦的陳顯中，今屆的對手則換上工黨的麥德正。何秀蘭自連任立法會後便把其地區辦事處搬至本選區，目的是為[[工黨|工黨]]開拓該區一帶票源，最終麥德正以266票之差勝出，結束江澤濠壟斷該區20年的局面。連同新當選的葉榮及譚駿賢，工黨於現任地區直選立法會議員所屬選區及地區辦事處皆有一席區議會議席。
