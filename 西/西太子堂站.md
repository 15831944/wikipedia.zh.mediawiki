'''西太子堂站'''（{{jpn|j=西太子堂駅|hg=にしたいしどうえき|rm=Nishi-taishidō eki|f=y}}）位於[[日本|日本]][[東京都|東京都]][[世田谷區|世田谷區]][[太子堂_(世田谷區)|太子堂]]四丁目，是[[東京急行電鐵|東京急行電鐵]][[世田谷線|世田谷線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''SG02'''。

== 歷史 ==
* 1925年（[[大正|大正]]14年）1月18日 - '''西山站'''開業。
* 1939年（[[昭和|昭和]]14年）10月16日 - 改稱'''西太子堂站'''。

== 車站構造 ==
對向[[側式月台|側式月台]]2面2線的[[地面車站|地面車站]]。包括此站在內，世田谷線內的所有中途站都是這種構造。月台設置了接近表示機。

== 利用狀況 ==
官方未公布本站的乘車人數。世田谷線全線的乘車人數請參照[[東急世田谷線#使用情況|東急世田谷線#使用情況]]。

== 車站周邊 ==
{{See also|太子堂 (世田谷區)|若林 (世田谷區)|三軒茶屋}}
北側鄰近太子堂五丁目、世田谷區立太子堂小學校，南側鄰近[[若林_(世田谷區)|若林]]一丁目。
三軒茶屋站因周邊進行再開發而往本站方向移動，使得兩站間的距離僅有300公尺。

== 巴士路線 ==
車站南側的太子堂四丁目交差點附近有'''西太子堂'''巴士站，[[東急巴士|東急巴士]]運行以下路線。
* <澀05> 澀谷站 → '''西太子堂''' → {{link-ja|東急巴士弦卷營業所|東急バス弦巻営業所|弦卷營業所}}
: ※ 僅往弦卷営業所方向有設站。
