《'''西厢记'''》（{{lang-en|'''Romance of the Western Chamber'''}}），播出時又稱《新西廂記》，中国大陆电视剧，改编自唐朝元稹《[[莺莺传|莺莺传]]》、元朝[[王实甫|王实甫]]《[[西厢记|西厢记]]》，导演[[路奇|路奇]]，主演周奇奇、张晓晨、邓家佳、孙坚、宋佳。2010年12月殺青，2013年1月30日於[[央視八套|央視八套]]首播。

==播放平台==
*2013年1月30日 {{CHN}} [[央視八套|央視八套]]
*2014年7月22日 {{CAN}} [[城市电视|城市电视]] 逢星期一至五西岸时间2:00 PM 及 8:30 PM

== 剧情 ==
{{See Also|西厢记}}
唐朝，书生张珙和相国千金莺莺在普救寺相遇之后，迅速热恋上对方，但是相国夫人不同意他们的亲事，经过红娘的协助，张珙最终娶得莺莺。

==花絮==
该剧的制作方是中国国际电视总公司、中国广播电影电视节目交易中心、西安曲江文化旅游有限公司、大唐芙蓉园景区管理公司、北京新世邦文化传媒有限公司。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2010-09-27/18483099634.shtml |title=资料：电视剧《西厢记》制作方的基本资料 |accessdate=2010 |publisher= 新浪网|archiveurl= http://ent.sina.com.cn/v/m/2010-09-27/18483099634.shtml|archivedate=2012年12月28日 |work= |language=zh-cn}}</ref>制作方表示将忠实于原著。<ref>{{cite web |url= http://ent.163.com/10/1127/02/6MFB4CTL00034JAU.html|title= 电视剧《西厢记》追求古典味 拍摄忠于原著|accessdate= 2010|publisher= 网易|archiveurl= http://ent.163.com/10/1127/02/6MFB4CTL00034JAU.html|archivedate=2012年12月28日 |work= |language=zh-cn}}</ref>

该剧拍摄于[[西安|西安]]大唐芙蓉园、[[陕西|陕西]]等地。<ref>{{cite web |url=http://dianshiju.cntv.cn/20101119/100215.shtml |title=《西厢记》张晓晨苦练古琴书法 张生风骨忠原著 |accessdate= 2010年11月19日|publisher=中国网络电视台 |archiveurl=http://dianshiju.cntv.cn/20101119/100215.shtml |archivedate=2012年12月28日 |work= |language=zh-cn}}</ref><ref>{{cite web |url=http://xian.qq.com/a/20100916/000347.htm |title=电视剧<西厢记>西安开机 宋佳苦等15年终圆梦 |accessdate= 2010|publisher=腾讯网 |archiveurl=http://xian.qq.com/a/20100916/000347.htm |archivedate=2012年12月28日 |work= |language=zh-cn}}</ref>

该剧开拍之初，[[吴尊|吴尊]]出演张珙的呼声非常高，但最终由张晓晨出演张珙。<ref>{{cite web |url=http://www.jsw.com.cn/site3/zjrb/html/2010-08/30/content_1496914.htm |title=电视剧《西厢记》开机在即 |accessdate=2010 |publisher= 镇江日报|archiveurl=http://www.jsw.com.cn/site3/zjrb/html/2010-08/30/content_1496914.htm |archivedate=2012年12月28日 |work= |language=zh-cn}}</ref>

该剧比以往“西厢记”题材电视不同的是增加了红娘的戏份。<ref>{{cite web |url=http://finance.qq.com/a/20100829/000502.htm |title= 电视剧《西厢记》将开拍|accessdate=2010 |publisher=腾讯网 |archiveurl=http://finance.qq.com/a/20100829/000502.htm |archivedate=2012年12月28日 |work= |language=zh-cn}}</ref>
