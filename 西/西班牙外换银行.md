'''毕尔巴鄂比斯开银行'''（{{lang-es|Banco Bilbao Vizcaya Argentaria}}，常称作'''西班牙外换银行'''或'''西班牙对外银行'''；缩写'''BBVA'''）是[[西班牙|西班牙]]的主要银行，成立于1999年，总行位于[[巴斯克自治区|巴斯克自治区]][[毕尔巴鄂|毕尔巴鄂]]。

== 概要 ==
1999年由畢爾包比斯卡亞銀行和Argentaria两家银行合并，名称因而变更为毕尔巴鄂比斯开银行（Banco Bilbao Vizcaya Argentaria，简称BBVA）。其中毕尔巴鄂比斯开银行本身几年前由三家银行合并而成的：毕尔巴鄂银行，比斯开银行和西班牙外换銀行。BBVA於[[2008–09賽季西班牙足球甲級聯賽|2008-09]]赛季開始贊助[[西班牙甲組足球聯賽|西班牙甲組足球聯賽]]<ref>{{cite web |title=La Liga rompe la hucha |url=https://www.hoy.es/deportes/futbol/liga-primera/liga-gasto-fichajes-20180719185148-ntrc.html |website=hoy.es |accessdate=2019-01-29 |archiveurl=https://web.archive.org/web/20180720225322/https://www.hoy.es/deportes/futbol/liga-primera/liga-gasto-fichajes-20180719185148-ntrc.html |archivedate=2018-07-20 |language=es |date=2018-07-19}}</ref>，持续了8个赛季，直到[[2015–16賽季西班牙足球甲級聯賽|2015-16]]赛季<ref>{{cite web |title=西甲变天！BBVA退出历史舞台 西甲迎新赞助商 |url=https://sports.qq.com/a/20160721/029869.htm |publisher=[[腾讯|腾讯]]体育 |accessdate=2019-01-29 |archiveurl=https://web.archive.org/web/20160725224103/http://sports.qq.com/a/20160721/029869.htm |archivedate=2016-07-25 |language=zh-cn |date=2016-07-21}}</ref>。而在2010年也開始贊助[[國家籃球協會|NBA]]<ref>{{Cite web|url=http://in.reuters.com/article/2010/09/13/idINIndia-51454220100913|title=BBVA becomes NBA sponsor|accessdate=2017-01-02|date=2010-09-13|publisher=reuters|language=en}}</ref>，直到2017年为止<ref>{{cite web |author=Esperanza Balaguer |title=BBVA finaliza su patrocinio con la NBA |url=http://www.bez.es/321037026/bbva-finaliza-patrocinio-bbva.html |website=bez@bez.es |archiveurl=https://web.archive.org/web/20170911161836/http://www.bez.es/321037026/bbva-finaliza-patrocinio-bbva.html |archivedate=2017-09-11 |language=es |date=2017-07-01}}</ref>。

2018年，西班牙对外银行成为全球首家使用[[区块链|区块链]]技术发放贷款的银行<ref>{{cite web |author=雷锋网 |title=全球首家！西班牙BBVA银行使用区块链技术发放贷款 |url=http://finance.sina.com.cn/stock/usstock/c/2018-04-27/doc-ifztkpip4513508.shtml |publisher=新浪财经 |archiveurl=https://web.archive.org/web/20180427181659/http://finance.sina.com.cn/stock/usstock/c/2018-04-27/doc-ifztkpip4513508.shtml |archivedate=2018-04-27 |language=zh-cn |date=2018-04-27}}</ref>。它还同时名列“2018西班牙最有价值的100大品牌”排行榜第3位<ref>{{cite web |title=2018西班牙最有价值的100大品牌排行榜 |url=http://www.sinotf.com/GB/News/1005/2018-04-27/zNMDAwMDMxMjMzNA.html |publisher=中国贸易金融网 |archiveurl=https://web.archive.org/web/20180430033029/http://www.sinotf.com/GB/News/1005/2018-04-27/zNMDAwMDMxMjMzNA.html |archivedate=2018-04-30 |language=zh-cn |date=2018-04-27}}</ref>，《[[财富_(杂志)|财富]]》杂志2018年世界500强第224位<ref>{{cite web |title=2018世界500强发布 |url=http://finance.sina.com.cn/stock/usstock/c/2018-07-19/doc-ihfnsvzc0314436.shtml |publisher=新浪财经 |archiveurl=https://web.archive.org/web/20180919132750/http://finance.sina.com.cn/stock/usstock/c/2018-07-19/doc-ihfnsvzc0314436.shtml|archivedate=2018-09-19 |language=zh-cn |date=2018-07-19}}</ref>。
