'''西藏往事'''（英文名：'''Once Upon a Time in Tibet'''），2010年10月16日上映于[[中国大陆|中国大陆]]的[[爱情片|爱情片]]。

== 剧情 ==
该片根据[[西藏|西藏]]作家[[扎西达娃|扎西达娃]]的小说《[[香巴拉的猫|香巴拉的猫]]》改编而成。

该片讲述了[[抗日战争|抗日战争]]期间，[[美军|美军]][[飛行员|飛行员]][[罗伯特|罗伯特]]和[[西藏|西藏]]女子雍措之间的“跨国恋”爱情故事。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2011-04-13/09513279964.shtml |title=《西藏往事》好评如潮 小宋佳点燃催泪弹 |accessdate= 2010年|work=|publisher=新浪网 |language = zh-cn}}</ref><ref name="fhs " />

== 幕后 ==
该片拍摄于4800米之上的西藏高原，拍摄得非常艰苦，显示了各班人马的敬业精神。<ref>{{cite web |url=http://data.ent.sina.com.cn/movie/moviedetail.php?id=13339&detail=background |title=西藏往事 |accessdate= 2010年|work=|publisher=新浪网 |language = zh-cn}}</ref><ref name="fhs " /> 导演[[戴玮|戴玮]]说：“我们从09年9月开始拍摄这部电影，在海拔5000多米的高原上经常一个镜头就拍好几个小时，期间很多工作人员都是带病工作，到今天能够上映，是我们很多人付出努力的结果。”<ref>{{cite web |url=http://ent.ifeng.com/movie/news/mainland/detail_2011_04/14/5736756_0.shtml |title= 《西藏往事》情动大影节 观众洒泪导演哽咽|accessdate= 2011年|work=|publisher=凤凰网 |language = zh-cn}}</ref>

因为该片在西藏拍摄，[[何润东|何润东]]自曝曾为了显示肮脏而不洗澡，“藏区的水非常珍贵，我们洗澡的热水只有很少很少，所以后来我干脆不洗了，那种自然脏的感觉更符合角色。反正最好让大家认不出来自己，这样我的角色就算塑造成功了。”<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2011-03-30/18553268693.shtml |title=《西藏往事》将映 何润东挑战马景涛咆哮功 |accessdate= 2010年|work=|publisher=凤凰网 |language = zh-cn}}</ref>

女主角扮演者[[宋佳|宋佳]]为了演好角色，还跟[[藏族|藏族]]学会了怎么放羊。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/p/2011-04-11/14563277747.shtml |title=小宋佳《西藏往事》演藏族女孩以假乱真 |accessdate= 2010年|work=|publisher=新浪网 |language = zh-cn}}</ref>

演员[[朱子岩|朱子岩]]为了学[[藏语|藏语]]，晒黑了自己，并且说：“那里有最干净的天空，还有最纯洁的心灵，我终于能够体会戴玮导演一定要拍西藏电影的心情。”<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2011-04-07/15523274931.shtml |title= 朱子岩为《西藏往事》学藏语 被误认为藏族姑娘|accessdate= 2011年|work=|publisher=新浪网 |language = zh-cn}}</ref>

== 评价 ==
影评人[[战台烽|战台烽]]：“《西藏往事》是[[戴玮继|戴玮继]]《[[冈拉梅朵|冈拉梅朵]]》后的全新突破，导演凭借对[[西藏|西藏]]的了解和熟悉，完美描绘出了藏区美丽的湖泊和山川，而且对藏民族风土人情的表现也很真实，细节上高度还原历史。”<ref name=" oes" >{{cite web |url=http://ent.ifeng.com/movie/news/mainland/detail_2011_04/02/5528125_0.shtml |title=《西藏往事》点映获好评 何润东小宋佳农奴造型颠覆 |accessdate= 2011年|work=|publisher=凤凰网 |language = zh-cn}}</ref>

影评人芳菲“在当今国内影坛这种自然清新的风格并不多见，并把影片形容为是一曲爱情、生命与自由的挽歌。”<ref name=" oes" />
