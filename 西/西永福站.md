'''西永福站'''（{{jpn|j=西永福駅|hg=にしえいふくえき|rm=Nishi-eifuku eki|f=y}}）是一個位於[[東京都|東京都]][[杉並區|杉並區]][[永福_(東京都)|永福]]三丁目，屬於[[京王電鐵|京王電鐵]][[井之頭線|井之頭線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''IN10'''。

== 歷史 ==
* 1933年（[[昭和|昭和]]8年）8月1日 - 隸屬於帝都電鐵的車站開始營運。
* 1940年（昭和15年）5月1日 - 與小田原急行鐵道合併，成為該公司帝都線的車站。
* 1942年（昭和17年）5月1日 - [[小田急電鐵|小田急電鐵]]與[[東京急行電鐵|東京急行電鐵]]（大東急）合併。
* 1948年（昭和23年）6月1日 - 從東急分離，成立京王帝都電鐵，成為該公司井之頭線的車站。
* 2008年（平成20年）3月25日 - 跨橋式站房完成。

== 車站構造 ==
設有[[島式月台|島式月台]]1面2線的[[地面車站|地面車站]]。

原車站閘口僅有南口一處，月台與站房之間以地下道連結。之後為了將車站改善為[[通用設計|無障礙空間]]而採用[[跨站式站房|跨站式站房]]，此外新開設了北口以及新裝設電梯、電扶梯，還開設了南北向的聯絡通道。2007年10月開始該善工程，同年12月16日完成跨橋式站房及車站閘口、開設北口（同時設有一般階梯、電梯及電扶梯）、站內電梯和電扶梯裝設作業並開放使用。2008年3月25日完成南口的一般階梯、電梯及電扶梯裝設作業並開放使用。北口因空間不足的問題，電扶梯僅設一座供上樓使用。改建以前，站內未設有[[公共廁所|洗手間]]，需使用站前廣場的杉並區公用廁所；新站房完工之後，於車站閘口內新設立了洗手間以及無障礙洗手間。

與鄰站[[永福町站|永福町站]]僅相距700m，另一端的鄰站[[濱田山站|濱田山站]]相距800m，從此站月台可輕易與鄰站月台相望。

== 車站周邊 ==
=== 北口 ===
* 西永福北口[[商店街|商店街]] - 平日朝夕之間，因作為「通勤通學道路」及「購物道路」的關係，此時段的商店街內禁止汽車通行。另外，與鄰站急行停車站的永福町站之間的距離短，急行電車與普通車通過的時間差非常短（平交道經常處於禁止通行）且商店街的道幅狹窄，即便是一般日間時段也有許多購物民眾及學生通行，車輛也難以進出於此。
