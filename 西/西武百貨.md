'''西武百貨'''，是[[日本|日本]][[崇光·西武|崇光·西武]]公司旗下零售品牌，總店位於[[東京都|東京都]][[豐島區|豐島區]]南[[池袋|池袋]]1丁目28番1號，隸屬於日本[[7&I控股|7&I控股]]，與[[崇光百貨|崇光百貨]]爲姐妹品牌，目前海外已無品牌授權及任何合資。

== 歷史 ==
1940年，[[西武鐵道|西武鐵道]]的前身[[武藏野鐵道|武藏野鐵道]]收購[[京濱急行電鐵|京濱電氣鐵道]]系列的菊屋百貨（京濱百貨）池袋支店，開設武藏野食糧店。[[第二次世界大戰|第二次世界大戰]]結束前，[[東京大空襲|東京大空襲]]令池袋一帶建築物消失。1947年被帝都百貨店管理、易名武藏野百貨，1949年再易名西武百貨店。1950年代前半期開始分期擴建，成為關東首家大規模的一站式百貨公司。

西武百貨曾創造不少零售品牌包括：[[Parco|Parco]]、[[Loft|Loft]]、[[無印良品|無印良品]]等，亦參與文化事業，興建如銀座Saison劇場、澁谷Parco劇場、Saison美術館、Parco出版等。

2000年代，由於日本[[泡沫經濟|泡沫經濟]]破滅，西武百貨需面對重組，2003年6月1日，十合（[[崇光百貨|崇光百貨]]）與西武百貨兩社共同組成[[控股公司|控股公司]]千禧零售集團（{{lang|ja|ミレニアムリテイリンググループ}}）。

2005年12月26日，7&I控股宣佈併購千禧零售集團（{{lang|ja|ミレニアムリテイリンググループ}}， Millennium Retailing )<ref>[[7&I控股#歷史|7&I控股#歷史]]</ref>。

歷經多次公司改組後，現今爲[[崇光·西武|崇光·西武]]公司旗下零售品牌，並隸屬於日本[[7-Eleven|7-Eleven]]母公司7&I控股。
== 香港西武 ==
[[File:HK_Queensway_Pacific_Place_SEIBU.JPG|thumb]]金鐘[[太古廣場|太古廣場]]的西武百貨]]
[[File:Langham_Place_Seibu_2008.jpg|thumb]][[砵蘭街|砵蘭街]]的[[朗豪坊|朗豪坊]]西武]]

日本西武原擁有[[香港|香港]]西武部份股權，首家香港西武於1989年於[[金鐘|金鐘]][[太古廣場|太古廣場]]開業，面積90,000平方呎。其後於1993年，日本西武把其股權出售予香港商人[[潘迪生|潘迪生]]的[[迪生創建|迪生創建]]，自此香港西武除可繼續使用「西武」名稱對外經營外，其與日本西武已沒有任何關係。

股權易手後香港西武仍然持續擴展，包括於1997年於[[銅鑼灣|銅鑼灣]][[皇室大廈|皇室大廈]]開設分店，面積30,000平方呎。唯由於租約問題，銅鑼灣分店於2006年5月結業。2004年11月於[[旺角|旺角]][[朗豪坊|朗豪坊]]開設[[九龍|九龍]]區首家分店。2006年底於[[尖沙咀|尖沙咀]][[九龍酒店|九龍酒店]]開設分店。

金鐘太古廣場西武和尖沙咀西武已於2011年6月及9月結束營業。而最後一間香港西武旺角店亦於2013年9月17日結業<ref>[http://hk.apple.nextmedia.com/news/art/20130918/18427912 23年西武從此就冇 改名轉賣化妝品迎合自由行]，蘋果日報，2013年9月18日</ref>。

== 中國大陸分店 ==
西武百貨在中國大陸曾由[[迪生創建|迪生創建]]擁有，曾於[[成都|成都]]、[[瀋陽|瀋陽]]、[[深圳|深圳]]以及[[長春|長春]]設有分店。最後一間分店，位於深圳新城市廣場店已於2012年9月30日結業<ref>「最後的西武」告別內地百貨生意難做星島日報B06財金解密</ref>。
