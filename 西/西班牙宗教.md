'''[[羅馬天主教|羅馬天主教]]'''是'''[[西班牙|西班牙]]'''最大的宗教，<ref name=CIS2018 />但實際上西班牙的世俗化情形很強。只有3％的西班牙人認為宗教是他們最重要的三個價值之一，甚至低於歐洲平均水平的5％。<ref name=EUROBAROMETER83 />[[1978年西班牙憲法|1978年西班牙憲法]]廢除了天主教作為官方的國教，同時承認它在西班牙社會中的角色。因此，西班牙沒有[[國教|國教]]，[[宗教自由|宗教自由]]受到法律保障。<ref name=es_CONSTITUTION>{{cite act |title=[[Constitution_of_Spain|Spanish Constitution]] |type=Constitution |number= |language=en |date=29 December 1978 |article=14, 16 & 27.3 |articletype=Sections |page= |url=http://www.congreso.es/portal/page/portal/Congreso/Congreso/Hist_Normas/Norm/const_espa_texto_ingles_0.pdf |accessdate=5 March 2018 |ref=}}</ref>

根據西班牙社會學研究中心在2018年7月的統計，66.3％的西班牙人表示是天主教信徒，2.5％為其他信仰的信徒（包括[[伊斯蘭教|伊斯蘭教]]、[[基督教新教|基督教新教]]、[[佛教|佛教]]等），28.5％為[[無神論者|無神論者]]或[[無宗教者|無宗教者]]。<ref name=CIS2018 />大多數西班牙人很少參加宗教活動，有宗教信仰者中，62.5％幾乎沒有參加過宗教活動，13.4％的人每年參加一次，每月參加幾次宗教活動有8.7％，大部分參加有12.4％，都有參加有1.9％。<ref name=CIS2018 />

儘管大多數西班牙人都是天主教信徒，但大多數人，尤其是年輕一代的人，都會忽視天主教會關於[[婚前性行為|婚前性行為]]，[[性向|性向]]，[[婚姻|婚姻]]或[[避孕|避孕]]等問題。<ref name="LeMonde">{{cite news |last = Morel |first = Sandrine |coauthors = |title = La sécularisation express des jeunes Espagnols |trans-title= The express secularization of the Spanish youth |work = |pages = |publisher = [[Le_Monde|Le Monde]] |date = 2017-02-01  | url = http://www.lemonde.fr/europe/article/2017/02/01/la-secularisation-express-des-jeunes-espagnols_5072504_3214.html | language = fr |accessdate = 2018-03-28}}</ref><ref name="cbn1">{{cite web|url=http://www.cbn.com/mundocristiano/elmundo/2011/July/Espana-es-el-pais-europeo-en-el-que-mas-retrocede-el-catolicismo/ |title=España experimenta retroceso en catolicismo - El Mundo - Mundo Cristiano|language=es |publisher=CBN.com |date=2011-07-07 |accessdate=2015-05-29}}</ref><ref name="DAWN">{{cite news |last = Tarvainen |first = Sinikka |coauthors = |title = Reforms anger Spanish church |work = |pages = |publisher = Dawn International |date = 2004-09-26 |url = http://www.dawn.com/2004/09/26/int5.htm |accessdate = 2008-03-21}}</ref><ref name="WWRN">{{cite news|last =|first =|coauthors =|title =Zapatero accused of rejecting religion|work =|pages =|publisher =Worldwide Religious News|date =2004-10-15|url =http://www.wwrn.org/article.php?idd=15453&sec=59&con=53|accessdate =2008-03-21|deadurl =yes|archiveurl =https://web.archive.org/web/20081023070458/http://www.wwrn.org/article.php?idd=15453&sec=59&con=53|archivedate =2008-10-23|df =}}</ref><ref name="NYT"/><ref name="CSMONITOR"/> [[教區|教區]][[神父|神父]]總數從1975年的24,300人減少到2018年的18,500人，平均年齡為65.5歲。<ref name=INFOCAT2018 /><ref name=ESTADISTICA2005 /><ref name=LV2016 />

== 信仰態度 ==
雖然羅馬天主教仍然是西班牙最大的宗教，但大多數西班牙人（特別是年輕人），選擇忽視道德，政治或性行為方面的天主教教義，並且不常參加[[彌撒|彌撒]]。<ref name="LeMonde" /><ref name="PBS">{{cite news |last = Abernethy |first = Bobby |coauthors = |title = Catholicism In Spain |work = |pages = |publisher = PBS |date = 2006-07-07 |url = https://www.pbs.org/wnet/religionandethics/week945/cover.html |accessdate = 2008-10-21}}</ref><ref name="NYTIMES">{{cite news |last = Sciolino |first = Elaine |coauthors = |title = Europeans Fast Falling Away From Church
|work = |pages = |publisher = New York Times |date = 2005-04-19 |url = https://www.nytimes.com/2005/04/19/international/worldspecial2/19europe.html?scp=3&sq=april%2019,%202005%20mass&st=cse |accessdate = 2008-10-21}}</ref> 無宗教者包括[[不可知論|不可知論]]和[[無神論|無神論]]，世俗化情形和[[西歐|西歐]]國家一樣。<ref name="NYT">{{cite news |last = Loewenberg |first = Samuel |coauthors = |title = As Spaniards Lose Their Religion, Church Leaders Struggle to Hold On |work = |pages = |publisher = New York Times |date = 2005-06-26 |url = https://www.nytimes.com/2005/06/26/weekinreview/26loew.html?_r=1&oref=slogin |accessdate = 2008-10-21}}</ref><ref name="CSMONITOR">{{cite news |last = Pingree |first = Geoff |coauthors = |title = Secular drive challenges Spain's Catholic identity |work = |pages = |publisher = Christian Science Monitor |date = 2004-10-01 |url = http://www.csmonitor.com/2004/1001/p07s02-woeu.htm |accessdate = 2008-10-21}}</ref><ref name="PBS"/><ref name="LAM">{{cite news |last = MacHarg |first = Kenneth D. |coauthors =  |title = Spain’s Awakening:Is revival around the corner for Spain? |work =  |pages =  |publisher = Latin American Mission |date =  |url = http://www.lam.org/news/article.php?id=230 |accessdate = 2008-10-21 |deadurl = yes |archiveurl = https://web.archive.org/web/20081023235229/http://www.lam.org/news/article.php?id=230 |archivedate = 2008-10-23 |df =  }}</ref><ref name="IFES">{{cite news|last =|first =|coauthors =|title =Spain|work =|pages =|publisher =International Fellowship of Evangelical Students|date =|url =http://www.ifesinteraction.org/interaction/app/eventpages/eventpage.asp?PageID=7|accessdate =2008-10-21|deadurl =yes|archiveurl =https://web.archive.org/web/20081023122250/http://www.ifesinteraction.org/interaction/app/eventpages/eventpage.asp?PageID=7|archivedate =2008-10-23|df =}}</ref>

西班牙的[[文化戰爭|文化戰爭]]與政治關係遠遠超過宗教，而與[[創造論|創造論]]等典型宗教相關問題避免在衝突中被討論。天主教會複興主義成效不彰。<ref name="NYTIMES"/><ref name="LAM"/>
根據歐盟民調處（Eurobarometer），只有3％的西班牙人認為宗教是他們三個最重要的價值之一，低於的歐洲平均水平5％。<ref name=EUROBAROMETER83>{{cite web|url=http://ec.europa.eu/commfrontoffice/publicopinion/archives/eb/eb83/eb83_citizen_en.pdf|title=Eurobarometer 83 - Values of Europeans / Spring 2015|page=56|date=May 2015|accessdate=7 March 2018}}</ref><ref name=EUROBAROMETER69>{{cite web|url=http://ec.europa.eu/public_opinion/archives/eb/eb69/eb69_values_en.pdf|title=Eurobarometer 69 - Values of Europeans |page=16 |date=November 2008|accessdate=7 March 2018}}</ref>

並根據2005年歐盟民調處調查：<ref name=EUROBAROMETER>{{cite web|url=http://ec.europa.eu/public_opinion/archives/ebs/ebs_225_report_en.pdf|title=Eurobarometer on Social Values, Science and technology 2005 | page= 9|accessdate=7 March 2018}}</ref>

*59％的西班牙人回答說“他們相信有一位上帝。”

*21％的人回答說“他們相信有某種精神或生命力”。

*18％的人回答說“他們不相信有任何精神，上帝或生命力。”

西班牙同性婚姻合法化的廣泛支持可以看出當代西班牙世俗性的證據，根據社會學研究中心2004年的一項研究，超過70％的西班牙人支持同性婚姻。<ref>[http://www.cis.es/ Centre for Sociological Research]</ref> 
2005年6月，一項法案以187票對147票通過，允許同性婚姻，使西班牙成為歐盟第三個允許[[同性伴侶|同性伴侶]]結婚的國家。

== 宗教社群 ==
===羅馬天主教===
{{main article|西班牙天主教}}
天主教為西班牙最大宗教，目前有3700萬受洗的天主教信徒，占總人口的90%。共有70個教區和總教區。

===基督新教===
{{main article|{{link-en|西班牙基督教新教|Protestantism in Spain}}}}
西班牙新教基督徒主要為近幾年的新移民所信奉，[[西班牙人|西班牙人]]所佔的比例很少，在新教福音派中，西班牙被視為福音派傳教士視為較難傳教的地區。<ref name="LAM"/>西班牙新教教會聲稱擁有約1,200,000名信徒。<ref>{{cite web|url=http://www.ferede.org/general.php?pag=estad#1|title=Información general - Datos estadísticos|trans-title=General information - Statistical data|language=es|publisher=Federation of Evangelical Religious Entities of Spain|deadurl=yes|archiveurl=https://web.archive.org/web/20110930083055/http://www.ferede.org/general.php?pag=estad|archivedate=September 30, 2011}}</ref><ref>{{cite web|url=http://wwrn.org/articles/27149/|title=Protestants call for 'equal treatment'|work=[[El_Pais|El Pais]]|date=December 10, 2007|accessdate=November 5, 2016}}</ref>

===猶太教===
{{main article|{{link-en|西班牙猶太人歷史|History of the Jews in Spain}}}}
西班牙的猶太人佔西班牙人口的比例不到1％，主要集中在[[巴塞羅那|巴塞羅那]]、[[馬德里|馬德里]]和[[莫夕亞|莫夕亞]]。<ref>{{cite web|url=http://www.eurojewcong.org/ejc/news.php?id_article=117|archive-url=https://web.archive.org/web/20060301113502/http://www.eurojewcong.org/ejc/news.php?id_article=117|dead-url=yes|archive-date=2006-03-01|title=European Jewish Congress - Spain|publisher=}}</ref>

===伊斯蘭教===
{{main article|西班牙伊斯蘭教}}
[[File:BasharatMosque1.jpg|left]]的Basharat清真寺為西班牙鑑於近代的第一座清真寺]]
近年來移民到西班牙的新移民，特別是在1990年代後的移民，多數是[[穆斯林|穆斯林]]和新教基督徒。如今，伊斯蘭教是西班牙第二大宗教，但信徒人數遠遠落後於羅馬天主教信徒和非宗教。2016年西班牙穆斯林占西班牙總人口的4％<ref name="politica.elpais.com"/>，絕大多數西班牙穆斯林來自[[摩洛哥|摩洛哥]]和其他非洲國家，約有78萬名穆斯林擁有西班牙國籍。<ref name="politica.elpais.com"/>

===少數宗教===
西班牙除了基督宗教各派，伊斯蘭教，猶太教和非宗教外，還有少數印度教徒，佛教徒，異教徒，道教和巴哈伊教徒。
==== 佛教 ====
{{main|{{link-en|歐洲佛教|Buddhism in Europe}}}}
佛教於20世紀末期在西班牙出現，根據估計，西班牙約有65,000名教信徒。然而，一些佛教教義，如[[輪迴|輪迴]]或[[業|業]]，通過[[新紀元運動|新紀元運動]]與主流文化部分融合。
[[File:Stupa_de_Benalmadena.jpg|thumb]]的{{link-en|Benalmádena 佛塔|Benalmádena Stupa}}在2003年落成，是歐洲最大的佛塔]]

==== 道教 ====
西班牙道教由中國大陸道士田成陽在2000年代引入，並創立了加泰羅尼亞道教協會（AsociacióndeToísmodeCataluña）和道教廟宇清靜宮（Templo de la Pureza y el Silencio）<ref>{{cite web |url=http://www.taoismo.es/primertemplotaoi.html |title=Archived copy |accessdate=2013-07-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20130612072212/http://taoismo.es/primertemplotaoi.html |archivedate=2013-06-12 |df= }}</ref>，2014年 另一家[[:ca:Temple_Taoista_de_Barcelona|道教寺廟]]於巴賽羅納建立。<ref>{{cite web |url=http://www.taoismo.es/primertemplotaoi.html |title=Archived copy |accessdate=2013-07-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20130612072212/http://taoismo.es/primertemplotaoi.html |archivedate=2013-06-12 |df= }}</ref><ref>{{cite web |url=http://www.taoismo.es/primertemplotaoi.html |title=Archived copy |accessdate=2013-07-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20130612072212/http://taoismo.es/primertemplotaoi.html |archivedate=2013-06-12 |df= }}</ref>
