'''西加里曼丹'''是[[印尼|印尼]]在[[婆羅洲|婆羅洲]]島[[加里曼丹|加里曼丹]]地區的四個省份之一。面積約146,807平方公里。居民中超過一半信仰為[[伊斯蘭教|伊斯蘭教]]、23.94%為[[天主教|天主教]]、12.38%為[[基督新教|基督新教]]，其餘信[[佛教|佛教]]，[[印度教|印度教]]和[[孔教|孔教]]。首府[[坤甸|坤甸]]（Pontianak）剛好位於[[赤道|赤道]]線上，故该省又称为“赤道上的土地”。2000年[[人口|人口]]約3,740,017，人口年增長率是1.53%，[[人口密度|人口密度]]為（每平方公里）26人。

== 地理 ==
=== 週圍地區 ===
* 北與[[馬來西亞|馬來西亞]]的[[砂拉越|砂拉越]]州接壤。
* 東北是[[東加里曼丹|東加里曼丹]]。
* 東南為[[中加里曼丹|中加里曼丹]]。
* 南瀕[[爪哇海|爪哇海]]。
* 西濱[[卡里馬塔海峽|卡里馬塔海峽]]。

=== 地形 ===
西加里曼丹大部份地區都是[[沼澤|沼澤]]，包括西部的沿海地帶。北部、東北部與東南部有山脈。

=== 城市 ===
有2個主要城市，包括[[首府|首府]][[坤甸|坤甸]]和[[山口洋市|山口洋市]]以及12個縣：[[道房縣|道房縣]], [[新当縣|新当縣]], [[三發縣|三發縣]], [[邊口洋縣|邊口洋縣]], [[喃吧哇縣|喃吧哇縣]], [[蘭打縣|蘭打縣]], [[大古布縣|大古布縣]], [[桑高縣|桑高縣]], [[昔加羅縣|昔加羅縣]], [[默拉维縣|默拉维縣]], [[上卡普阿斯縣|上卡普阿斯縣]], [[北加荣縣|北加荣縣]]。<ref>{{YouTube|Iw3eHOtGRF8|Kalimantan Barat 西加里曼丹 West Kalimantan}}</ref>

=== 河流 ===
# [[卡普阿斯河|卡普阿斯河]](Kapuas)，印度尼西亚境内最长的河流，全长约1038公里。
# 帕萬河(Pawan)
# 肯道阿河(Kendaua)
# 默拉維河(Melawi)

=== 山脈 ===
# 馬勒山脈(Muller)：位於東南部，東北-西南走向，此山脈是與中加里曼丹的天然分界。
# 斯赫瓦納山脈(Schwaner)：位於東南部，東北-西南走向，是馬勒山脈往西南方向的延續，也是與中加里曼丹的天然分界。
# 卡普阿斯胡盧山脈(Kapuas Hulu)：位於北部，西北-東南走向，在東北部與馬勒山脈會合。

== 气候 ==
西加里曼丹首府坤甸市正处于[[赤道|赤道线]]上，属[[热带气候|热带气候]]，日平均气温在22与32摄氏度之间，年均降雨量在3000到3900毫米之间，平均湿度为85.2%。

== 生物 ==
天然[[植物|植物]]有[[鐵樹|鐵樹]]、[[柚樹|柚樹]]、[[櫟樹|櫟樹]]、各種[[菌類|菌類]]、[[蕨類|蕨類]]和[[蘭花|蘭花]]。

== 經濟 ==
主要為[[農業|農業]]，糧食作物有[[水稻|水稻]]、[[玉蜀黍|玉蜀黍]]、[[木薯|木薯]]；經濟作物為[[橡膠|橡膠]]、[[油棕|油棕]]和[[椰子|椰子]]。

工業多為家庭式的小型工廠，種類有碾米、加工棕油、木雕、織布、編草蓆和籃子等。

== 交通 ==
主要靠河運、海運和空運，有少數的[[公路|公路]]連接省府[[坤甸|坤甸]]和其他地區。

== 政治 ==
印尼政府于2001年批准西加里曼丹省开始实施自治。后来该省三大民族达雅族、马来族、华族达成一项协议：西加里曼丹省的华人是当地原住民之一，该省的第二大城市——华人占60％的[[山口洋市|山口洋市]]，必须由华人担任市长一职；首府[[坤甸市|坤甸市]]，华人约占30％左右，则规定必须由华人担任副市长一职。截止2006年，西加里曼丹省已有一位华人当选为印尼国会议员，4位华人当选为省议员<ref>{{cite web | language =zh-cn | publisher=广东省人民政府外事办公室
