{{作品名稱|name=''Jonah Hex''|hk=疤面鏢客|tw=疤面鬼煞手|cn=西部英雄约拿·哈克斯}}是一部2010年6月18日上映于美国的战争片，由美国著名演员乔什·布洛林和梅根·福克斯主演，华纳兄弟发行。電影取材自美國漫畫出版公司[[DC漫畫|DC漫畫]]同名漫畫角色《[[約拿·海克斯|約拿·海克斯]]》改編。

== 剧情 ==
约拿·哈克斯是是[[印第安人|印第安人]]和[[白人|白人]]的[[混血儿|混血儿]]，早年，家中被特恩布尔暗算，一把火烧死了他的父亲和母亲，他得以逃生，为了报仇，他不停地寻找仇人昆汀·特恩布尔，后来被仇人抓住，用烧红的烙铁在脸上刻下印记，使得他的脸部毁容，喝水都会从脸部的洞口流出来。哈克斯从此做了一个[[赏金猎人|赏金猎人]]，专门负责找人、抓人、杀人。他和一个妓女赖丽非常要好，经常去陪她睡觉。

一日，美国联邦政府在赖丽家中找到了哈克斯，两人当时匆匆忙忙，没来的急穿戴衣服，只得束手就擒。美国政府请求哈克斯抓捕分裂联邦的势力特恩布尔，而这人正好是哈克斯的仇人，他同意了美国政府的请求，开始追捕特恩布尔，最终将其杀死，挽救了美国政府，也报了仇。

== 花絮 ==
电影根据同名[[漫画|漫画]]改编而成。<ref name="lion " >{{cite web |url= http://ent.qq.com/a/20100428/000392.htm?|title=《西部英雄约拿·哈克斯》电影海报曝光 |accessdate=2010年4月28日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

电影的海报使用暗蓝色为主色调，充满了美国西部苍凉粗犷的美感。<ref name="lion " />

此前，梅根·福克斯的电影一直都是以花瓶女郎的形象的出现，在这部电影中，她扮演的妓女将持枪战斗，改变了她以往的角色造型。<ref>{{cite web |url=http://ent.qq.com/a/20100428/000332.htm |title= 梅根·福克斯新片曝宣传海报 变身性感持枪女郎|accessdate=2010年4月28日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

乔什·布洛林扮演的哈克斯脸上的伤疤让人感到恐怖和吓人。<ref>{{cite web |url= http://news.mtime.com/2010/04/22/1430315.html|title=《西部英雄》主人公正面照 右脸疤痕有点吓人 |accessdate=2010-04-22  |publisher=时光网 |work= |language =zh-cn}}</ref>

==評價==
根據[[香港蘋果日報|香港蘋果日報]]2010年11月24日報導, 美國權威財經雜誌《[[福布斯|福布斯]]》就2010年上映美國電影製作成本及全球票房收入, 統計出10大最蝕本電影, 結果由[[美瑾·霍絲|美瑾·霍絲]]與[[佐斯·布連|佐斯·布連]]合演漫畫真人片《疤面鏢客》淪為蝕錢冠軍。成本4,800萬美元, 全球票房只得1,100萬美元, 只收回成本24％。由於票房太差, 香港代理發行電影商沒有安排電影於戲院上映, 只安排DVD發行。<ref>{{cite web |url= http://hk.apple.nextmedia.com/entertainment/art/20101124/14694338|title=《福布斯》10大票房毒藥美瑾爛片2010蝕本王 |accessdate=2010-11-24 }}</ref><ref>{{cite web |url=http://www.forbes.com/2010/11/22/brendan-fraser-megan-fox-harrison-ford-business-entertainment-box-office-turkeys.html |title=Hollywood's Biggest Turkeys Of 2010 |accessdate=2010年11月22日 |publisher=Forbes.com |work= |language=en |archiveurl=https://archive.is/20120918144300/http://www.forbes.com/2010/11/22/brendan-fraser-megan-fox-harrison-ford-business-entertainment-box-office-turkeys.html |archivedate=2012-09-18 |deadurl=yes }}</ref>
