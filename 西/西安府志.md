《西安府志》是一部记载[[清朝|清代]][[西安市|西安]]地方的[[府志|府志]]，内容为西安的历史、地理、风俗、人情等。<ref name=":0">{{Cite book|title=《圖文天下——明清舆地學要籍》|last=故宮博物院 編|first=|publisher=紫禁城出版社|year=2011年9月出版發行|isbn=978-7-5134-0164-7|location=|pages=106頁}}</ref>

== 纂修历史 ==
清[[乾隆|乾隆]]三十七年（[[1772年|1772年]]），[[毕沅|毕沅]]署理[[陝西等處承宣布政使司|陕西布政使]]，他主张“能以文学抜饰吏治，不特长吏与有荣施”，并大力提倡地方官纂修方志。他有感于[[长安|古都西安]]山水掌故之放矢，认为修《西安府志》十分必要。适逢著名学者[[嚴長明|严长明]]游历[[渭河平原|关中]]，与毕沅观点颇为相同。<ref name=":0" />

乾隆四十一年（[[1776年|1776年]]），毕沅上书请求纂修《西安府志》获得批准。在编纂过程中《西安府志》的每一项工作、每一个环节都由严长明亲自负责。[[知府|知府]][[舒其绅|舒其绅]]“复为参稽案牍，斟酌民言，俾<!-- 使得 -->一郡之掌故眉列掌示”。从设想、奏请到删削定稿，毕沅一直参与其中。《西安府志》初稿成时，“凡与[[关中平原|秦中]]文献关涉者，计得千五百种，发凡举例，类聚区分，文成数万，为门一十有五，分类五十有一，合成一百卷”。后经毕沅“亲加裁削，为《西安府志》八十卷”。<ref name=":0" />

乾隆四十四年（[[1779年|1779年]]），《西安府志》书成，次年付梓<!-- 书稿雕版印行 -->发行。<ref name=":0" />

== 目录分类 ==
《西安府志》分地理、名山、[[食货志|食货]]、[[科举|选举]]、[[古蹟|古迹]]、大事、艺文等八十卷，卷首一卷。<ref name=":1">{{Cite book|title=《西安府志》|last=舒其紳等 修，嚴長明等 纂|first=何炳武 總校點，董健橋 審校|publisher=三秦出版社|year=2011年10月|isbn=978-7-80736-905-9|location=|pages=目錄}}</ref>

在职次名序上，时任[[陕甘总督|陕甘总督]][[勒爾謹|勒尔谨]]为“鉴裁”首位，毕沅第二，西安知府舒其绅为“甄辑”第一，严长明为“编订”。<ref name=":1" />

== 后世评价 ==
《西安府志》是[[西安府|西安设府]]以来仅有的一部府志，与以前西安地方志相比，无论在搜荟群籍、决疑纠缪上，还是在体例结构、史学思想上，都具特色。<ref name=":0" />

《西安府志》资料详实，考据充分，结构精当，体例严谨，详略得当，经世致用。书成之后，舒其绅感叹曰：“今国家重修统志，方成数省，而[[四库全书馆|四库馆]]亦未竣事，是篇为诸郡嚆<!-- 嚆矢，先声、先例意。 -->也。”<ref name=":0" />

[[浦霖|浦霖]]评价《西安府志》“取材宏富，立意精严，方诸史传大都取则扶风<!-- 疾风 -->，而荟萃鉴裁，与[[宋朝|宋]][[欧阳修|欧阳氏]]之《[[新唐书|新唐书]]》相似，非近日搜刮家<!-- 史料收集者 -->之能事也。”<ref name=":0" />
