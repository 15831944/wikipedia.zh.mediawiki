'''西斯廷小堂'''（{{lang-la|Sacellum Sixtinum}}；{{lang-it|Cappella Sistina}} {{IPA-it|kapˈpɛlla siˈstiːna|}}；也稱為'''西斯廷禮拜堂'''）是一座位於[[梵蒂岡|梵蒂岡]][[宗座宮殿|宗座宮殿]]內的[[天主教|天主教]][[小堂|小堂]]，緊鄰[[聖伯多祿大殿|聖伯多祿大殿]]，以[[米開朗基羅|米開朗基羅]]所繪《[[創世紀_(壁畫)|創世紀]]》穹頂畫，及[[壁畫|壁畫]]《[[最後的審判_(壁畫)|最後的審判]]》而聞名。小堂中的壁畫在[[西斯都四世|西斯都四世]]治下的[[1482年|1482年]]完成，[[1483年|1483年]][[8月15日|8月15日]]西斯都四世在這裡舉行了第一場聖母升天節彌撒，並決定這裡將要供奉[[童貞瑪利亞|童貞瑪利亞]]<ref name="Pietrangeli1986">{{Harvnb|Pietrangeli|1986|p=28}}</ref><ref>{{Citation |last=Monfasani |first=John |year=1983 |title=A Description of the Sistine Chapel under Pope Sixtus IV |journal=Artibus et Historiae |volume=4 |issue=7 |pages=9–18 |issn=0391-9064 |url=http://www.artibusethistoriae.org/?menu=art&gdzie=artibusChapterResult&id=68 |doi=10.2307/1483178 |publisher=IRSA s.c. |postscript=. |jstor=1483178}}</ref>。此外，現在這裡也是[[教宗選舉|教宗選舉]]的舉行處。

== 歷史 ==
由教宗[[西斯都四世|西斯都四世]]發起創建，教堂名「西斯廷」即來源於教宗之名「西斯都」。教堂始建於1445年。1481年完工後，[[波提切利|波提切利]]等[[文藝復興|文藝復興]]初期畫家以[[耶穌基督|耶穌基督]]為題目創作了一批壁畫。後來根據[[儒略二世|儒略二世]]的命令由米開朗棋羅繪製天花板和剩下的壁畫。 牆上的壁畫取《舊約·聖經》為題材，正面是《最後的審判》，右側是《基督傳》，左側以《摩西傳》為主題。天花板的中央是《舊約·聖經》的創世紀，畫有預言者、巫女的像。最精彩是《[[創造亞當|創造亞當]]》、《原罪與流放樂園》、《諾亞方舟》。1483年，在此舉行過西斯都四世的聖體告別。起初穹頂繪有天空的圖案。教堂長40.25公尺，寬13.41公尺，高20.73公尺。依照[[列王紀|列王紀]]第6章所描述的[[耶路撒冷聖殿|所羅門王神殿]]比例（60：20：30）所建。

== 穹頂畫 ==
西斯都四世之侄[[儒略二世|儒略二世]]於1506年，命米開朗基羅繪製。對於喜歡自稱為雕刻家的米開朗基羅來說，雖非本願，但1508年開始設計創作此畫，從教堂內腳手架設置，到設計創作，均為單獨一人創作。1512年完成。穹頂中心描繪了[[舊約|舊約]]中創世紀的9個場景。場面宏大，氣勢恢宏，人物眾多。是米開朗基羅的代表作。

在米開朗基羅創作途中曾前來觀摩的拉斐爾深受此畫影響，他當時正在創作壁畫《[[雅典學院_(畫作)|雅典學院]]》。

畫面分成兩部分，繪有上帝創造世界的9個場景，即《劃分光暗》、《創造日月》、《劃分水陸》、《創造亞當》、《創造夏娃》、《逐出樂園》、《挪亞方舟》、《挪亞獻祭》、《挪亞醉酒》。房頂與牆壁連接處的弧面為另一部分，分別繪有耶穌的祖先等。

== 祭壇畫 ==
20年後[[克雷芒七世|克雷芒七世]]聘請[[米開朗基羅|米開朗基羅]]設計創作祭壇背面的[[最後的審判_(壁畫)|最後的審判]]。克雷芒七世死後，由其繼承者保祿三世接續。創作時間為1535年至1541年。以《[[瑪竇福音|瑪竇福音]]》中描述的“[[最後的審判|最後的審判]]”為題材的作品。大體上分為四個階層，從最上層開始，分別描繪有，天使群像，耶穌基督為中心的[[天堂|天堂]]，被拖入[[地獄|地獄]]的人群，地獄。米開朗基羅「地獄」部分的創作，靈感來源於[[但丁|但丁]]敘事詩《[[神曲|神曲]]》之地獄篇。米開朗基羅的作品給觀者帶來了震撼的效果，同時因為描繪了眾多裸體人像，被當時很多人牴觸，保祿三世死後，裸體的陰部被加繪上了遮羞布等。

== 修復 ==
西斯廷小堂作為一個宗教儀式舉行地，常年受蠟燭等薰染，穹頂畫和壁畫受到了很嚴重的損壞，近年來的修復工作使得穹頂畫和壁畫重現了繪製當時的鮮艷色彩，一部分的遮羞布也被除去，還原了當時的本來面目。
