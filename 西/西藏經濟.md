'''西藏經濟'''由自給自足的[[農業|農業]]為主。由於耕地有限與氣候高寒，牲畜飼養是青藏高原的主要活動。其中有山羊、綿羊，[[犛牛|犛牛]]，毛驢和馬匹。農業在一般河谷以[[青稞|青稞]]種植為主，在[[藏南|藏南]]農作物品種較多。在最近幾年已開始演變成一個具有多重結構的經濟，農業和[[第三產業|第三產業]]的發展並排。

==经济水平==

在2008年西藏國內生產總值為396億元人民幣。中國政府說，豁免{{来源请求|所有西藏的稅務}}和提供西藏地方政府[[财政|财政]]中90%的收入<ref name="a1"/>。批評者說，北京政府剝離西藏的資源，而忽略了[[藏族|藏族]]人民的福利。

據西藏自治區主席[[向巴平措|向巴平措]]説，西藏的經濟從2000年至2006年每年增長平均12%。在2006年人均國內生產總值達到10000元，在西藏歷史上的第一次。

在2008年首6個月，西藏因拉萨騷亂經濟衰退了一半。比2007年14.7%下跌一半。

== 地方政府财政与中央政府的發展努力 ==

在2010年代，[[西藏自治区人民政府|西藏自治区人民政府]]的收支平衡与中国大部分省级人民政府，特别是边疆地区类似，高度依赖中央政府提供的[[财政转移支付|财政转移支付]]来维持。2014年至2017年的数据显示，西藏自治区人民政府财政收入在两百亿元人民币左右，支出在一千亿元以上。数年来，两者之间的缺口维持在千亿元人民币的规模<ref name="a1">{{Cite web |url=http://cj.sina.com.cn/article/detail/1010236564/349470?column=realty&ch=9 |title=重磅：神奇的财政平衡术，31省财政转移支付全景图 |accessdate=2018-04-30 |author= |date=2017-08-07 |work= |publisher=新浪网|language=Zh-hans }}</ref>。

2010年1月18-20日，國務會議宣布對西藏地區的四川，雲南，甘肅和青海等藏族聚居的舉行大量的計劃，以改善該地區的發展。

[[中共中央政治局常務委員會|中共中央政治局常務委員會]]委員：胡錦濤，[[吳邦國|吳邦國]]，[[溫家寶|溫家寶]]，[[賈慶林|賈慶林]]，[[李長春|李長春]]，[[習近平|習近平]]，[[李克強|李克強]]，[[賀國強|賀國強]]，[[周永康|周永康]]出席了會議，標誌著中國高層領導人的承諾，西藏和藏族地區的發展。該計劃要求西藏農村收入提高到2020年對所有農村的藏族兒童免費教育的國家標準。中國政府自2001年以來，先後投資310億元。“西藏的國內生產總值預期，以達到在2009年437億元，在2000年增長170%達12.3%，超過過去九年間每年的增長。預計外界的觀察員增加在西藏的興趣超過來關心，導致藏族在2008年種族騷亂的民族主義。[[王力雄|王力雄]]也说西藏經濟與現代化靠外力維持。

== 各类产业 ==

===旅游业===

{{main|西藏旅游}}
近年來，特別是在2006年7月[[青藏鐵路|青藏鐵路]]完工之后，西藏旅遊業迅速擴大。在2006年，西藏接待了250萬遊客，其中包括15萬外國人。

2007年，這一數字上升到約4百萬人次，但在2008年，由于三月到六月期间旅遊區關閉，该年接待人数下降到2,246,400人。

2009年一月到六月间，有超过270万游客到访了西藏，为2008年同期人数的三倍，为当地提供了22.9亿元的收入。

2010年，西藏共接待了685万国内外游客，旅游收入为71.4亿元，占其GDP的14%。

2012年一月到十一月，西藏创纪录的接待了一千万游客，超过了2011年的869万。根据政府数据，有将近30万人受雇于该地区的旅行机构。

===傳統手工藝品===

快速發展帶來了傳統手工藝的復興，很多藏族人今天向遊客出售工藝品和文化產品。[[地毡|地毡]]與[[唐卡|唐卡]]是重要產品。
