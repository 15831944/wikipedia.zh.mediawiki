'''西孟加拉邦'''（{{lang-bn|পশ্চিম বঙ্গ}}；{{lang-ne|पश्चिम बङ्गाल}}；{{lang-en|West Bengal}}）位于印度东部，面积88,752平方公-{里}-，人口為九千一百萬（2011年）。

此邦位於[[孟加拉|孟加拉地區]]西部，东部毗连於[[孟加拉国|孟加拉国]]和印度[[阿萨姆邦|阿萨姆邦]]，西部與[[奥里萨邦|奥里萨邦]]、[[比哈尔邦|比哈尔邦]]以及[[贾坎德邦|贾坎德邦]]相壤，北傍[[不丹|不丹]]和[[锡金邦|锡金邦]]，南濒[[孟加拉湾|孟加拉湾]]。主要是[[孟加拉人|孟加拉人]]，邦官方语言是[[孟加拉语|孟加拉语]]。首府位于[[加尔各答|加尔各答]]，为印度第三大城市。

== 省名 ==
因原為英屬印度的[[孟加拉省|孟加拉省]]西部而得名。1999年，西孟加拉邦议会通过决议，将邦名更改为孟加拉邦，但未得印度国会的认可。

== 行政区划 ==
西孟加拉邦分为19个县。

== 经济 ==
西孟加拉邦土壤肥沃，全邦有四分之三的人口生活在[[农村|农村]]，盛产[[水稻|水稻]]和[[黄麻|黄麻]]。邦內主要工业有棉麻纺织、製茶、製糖、钢铁和造纸等，素以[[甜食|甜食]]聞名。著名地点有[[加尔各答|加尔各答]]、[[阿散索尔|阿散索尔]]、[[大吉岭|大吉岭]]、[[西里古里|西里古里]]。

== 社会 ==
2011年12月中旬，[[加尔各答|加尔各答]]以南约30公里处的松格兰布尔村庄周围地区的贫苦劳工，发生了饮用[[甲醇|甲醇]]酿造的[[假酒|假酒]]集体[[中毒|中毒]]事件，造成143人死亡<ref>{{cite web |url = http://www.zaobao.com/gj/gj111216_005.shtml |language = zh-hans |title = 印度143人饮假酒中毒身亡 |publisher=联合早报 |date=2011-12-16 |accessdate= }}</ref>。
