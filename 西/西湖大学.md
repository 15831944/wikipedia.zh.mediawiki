'''西湖大学'''是[[中华人民共和国|中国]]的一所社会力量举办、国家重点支持的非营利性的新型研究型大学，位于[[浙江省|浙江省]][[杭州市|杭州市]][[西湖区_(杭州市)|西湖区]]。该校成立于2018年10月20日，由[[杭州市西湖教育基金会|杭州市西湖教育基金会]]举办<ref>{{cite web|author=刘潇翰 |url=http://www.moe.gov.cn/srcsite/A03/s181/201804/t20180402_332130.html |title=教育部关于同意设立西湖大学的函 |publisher=教育部 |date=2018-03-20 |accessdate=2018-04-02}}</ref><ref name="成立">{{cite web|url=http://news.sciencenet.cn/htmlnews/2016/12/363133.shtm | language= | title=中国首所民办高等研究院成立 施一公任院长 |publisher=科学网 |date=2016-12-10 |accessdate= }}</ref>。由浙江省教育厅统筹管理和指导。

西湖大学从举办研究生教育起步，适时开展本科生教育，全日制在校生规模不超过5000人。<ref name=":0">{{Cite web|url=http://www.moe.gov.cn/srcsite/A03/s181/201804/t20180402_332130.html|title=教育部关于同意设立西湖大学的函 - 中华人民共和国教育部政府门户网站|accessdate=2018-10-30|author=|date=|last=|work=中华人民共和国教育部政府门户网站|publisher=}}</ref>

目前云栖校区已建成理学院、工学院和生命科学学院三大学院，汇集了一批世界一流科学家。可容纳120个独立实验室，供2000位科研及辅助人员研究学习。预计于2021年底交付使用的云谷校园可容纳300个独立实验室和3000名博士生。2022年开始小规模本科生的培养。

今后十年，学校将重点发展理学、医学、工学三个学科门类，推动跨学科交叉，深度创新。<ref>{{cite web |author1=谢盼盼 |author2=童笑雨 |author3=聂晨静 |title=西湖大学正式成立 将于2022年小规模培养本科生 |url=http://www.xinhuanet.com/politics/2018-10/20/c_1123588163.htm |website=新华网 |publisher=中国新闻网 |language=zh-cn|accessdate=2018-10-30}}</ref>

== 历史==
2015年3月11日，施一公、[[陈十一|陈十一]]、[[潘建伟|潘建伟]]、[[饶毅|饶毅]]、[[钱颖一|钱颖一]]、张辉、王坚七名学者，正式向国家提交《关于试点创建新型民办研究性的大学的建议》获得支持。2015年12月1日，[[浙江西湖高等研究院|浙江西湖高等研究院]]在杭州注册成立。<ref name="成立"/> 2016年12月10日，浙江西湖高等研究院成立大会在杭州举行，全国政协副主席[[韩启德|韩启德]]、浙江省代省长[[车俊|车俊]]等众人出席大会。<ref >{{cite web|url=http://www.wias.org.cn/index.php?a=detail&catid=370&id=8458&web=chinese | language=zh-cn | title=心怀高远 砥砺前行：浙江西湖高等研究院正式成立 |publisher=浙江西湖高等研究院 |date=2016-12-11 ||accessdate=2018-10-30 }}</ref>

浙江西湖高等研究院的创始捐赠人有[[王东辉|王东辉]]、[[王健林|王健林]]、[[邓锋|邓锋]]、[[葛航|葛航]]、[[黄昌华|黄昌华]]、[[马化腾|马化腾]]等人。设有理学研究所、前沿技术研究所、基础医学研究所和生物学研究所共计四所科研机构，开展科学研究和[[博士生|博士生]]培养。<ref name="成立"/>

2017年3月19日，西湖高等研究院首批学术人才和教职员工入驻位于杭州[[云栖小镇|云栖小镇]]约10万平米的园区，高研院园区正式启用。<ref >{{cite web| url = http://zjnews.zjol.com.cn/zjnews/hznews/201703/t20170319_3335229.shtml | language=zh-cn | title=西湖高研院园区启用 马化腾施一公现身启动仪式 |publisher=浙江在线 |date=2017年3月19日 || accessdate = 2018-10-30 }}</ref>

2018年2月14日，西湖大学获[[中华人民共和国教育部|教育部]]批准设立。校董会成员已经进入实质性提名和遴选阶段，在校长到任前，由西湖大学筹委会主任、筹办工作组组长、浙江西湖高等研究院院长施一公统筹负责大学日常管理事务<ref name=":0" /><ref>{{cite web| url = http://hznews.hangzhou.com.cn/chengshi/content/2018-04/02/content_6837999.htm| language=zh-cn | title=西湖大学正式获批设立 |publisher=浙江日报  |date=2018-04-02  || accessdate = 2018-10-30 }}</ref>。

2018年4月2日，教育部正式印发关于同意设立西湖大学的函<ref>{{Cite web|url=http://news.sina.com.cn/c/nd/2018-04-02/doc-ifysvnzy5236995.shtml|title=教育部:国家重点支持西湖大学 在校生规模不超5千|accessdate=2018-10-30|work=新浪新闻}}</ref>。

2018年4月3日，西湖大学主校区云谷校区校园建设工程正式启动，云谷校区位于西湖区紫金港科技城云谷区块，一期用地面积约1495亩，其中首期建设用地635亩，首期校舍建筑面积约42.2万平米，预计2021年底建成<ref>{{Cite web|url=http://www.wias.org.cn/chinese-detail-370-8938.html|title=西湖大学云谷校区校园建设工程正式启动|accessdate=2018-08-31|work=西湖大学}}</ref>。

2018年4月16日，召开创校校董会第一次会议。[[钱颖一|钱颖一]]当选校董会主席，[[杨振宁|杨振宁]]当选校董会名誉主席。施一公当选首任校长<ref>{{Cite web|url=https://www.thepaper.cn/newsDetail_forward_2079677|title=杨振宁担任西湖大学校董会名誉主席，施一公当选为首任校长|accessdate=2018-10-30|author=|date=|work=澎湃新闻|publisher=}}</ref>。

2018年8月24日，招收的120名2018级博士研究生入学，这是西湖大学二期学生，也是大学获得教育部正式批复以来招收的首届学生。

2018年10月20日，西湖大学正式宣告成立<ref>{{cite web|url=https://m.thepaper.cn/newsDetail_forward_2547451|title=西湖大学举行成立大会，校长施一公致辞：磨砺以须，倍道而进_教育家|accessdate=2018-10-30|website=澎湃新闻}}</ref>。

2018年11月5日，西湖大学、莱斯大学、纽约大学、南方科技大学等大学校长以及麻省理工学院、耶鲁大学等大学代表联合签署《2018西湖高等教育论坛联合宣言》（简称《联合宣言》）<ref>{{Cite web|url=http://ent.zjol.com.cn/jyck/201811/t20181105_8663771.shtml|title=西湖大学牵手世界名校发布《联合宣言》-人文频道-浙江在线|accessdate=2018-11-09|work=ent.zjol.com.cn}}</ref>。

2018年11月16日，“西湖大学-博智林智能机器人联合研究院”在佛山顺德和杭州揭牌<ref>{{Cite web|url=http://www.xinhuanet.com/house/2018-11-20/c_1123740662.htm|title=西湖大学与碧桂园成立智能机器人联合研究院 - 新华网|accessdate=2018-11-24|author=|date=2018-11-19|work=新华网|publisher=}}</ref>。

==概况==

=== 校内建筑 ===
云栖校区占地10万平方米，共7幢楼。

分别是生物学、基础医学、理学和前沿技术4个研究所，以及两幢学生公寓，行政楼被这些楼围在中间。公寓旁边还有一个小操场。<ref>{{Cite web|url=http://zjnews.zjol.com.cn/zjnews/hznews/201808/t20180810_7983982.shtml|title=第一次开学典礼在即 “小而精”的西湖大学来了 - 浙江在线|accessdate=2018-10-30|author=郑琳|date=2018-04-10|coauthors=蒋敏华|publisher=浙江在线}}</ref>

===学部、学院（系）===
*学院：生命科学学院（含生物学和基础医学）、理学院、工学院
*重点一级学科：理学、生命与健康、前沿技术
*建设中一级学科：生物学、生物医学工程、药学、物理学、数学、化学、基础医学、计算机科学与技术、电子科学与技术、光学工程、材料科学与工程、机械工程<ref>{{Cite web|url=http://gaokao.eol.cn/zhe_jiang/dongtai/201804/t20180408_1594029.shtml|title=西湖大学“新”在哪里？有情怀的科学家提出创办 —中国教育在线|accessdate=2018-10-28|author=新华网|date=2018-04-08|work=|publisher=中国教育在线|language=zh-cn}}</ref>。

===师资力量===
西湖大学根据学科发展需求与建设进度安排，分阶段、按年度面向全球招聘顶尖科学家和优秀的青年学者。

截至2018年4月2日，已完成6次全球招聘，拟聘任学术人才58人，其中，前期入职的教师已有7人入选第十四批国家“千人计划”青年项目。<ref>{{Cite web|url=http://paper.people.com.cn/rmrb/html/2018-04/02/nw.D110000renmrb_20180402_1-12.htm|title=西湖大学有啥不一样|accessdate=2018-10-30|last=第012版|work=paper.people.com.cn}}</ref>

预计到2026年西湖大学的师资队伍（教授、副教授、助理教授）规模达到300人。

== 办学定位 ==

* 高起点：以博士生培养为起点，适时开展本科生教育。
* 小而精：坚持发展“有限学科”。
* 研究型：主要开展基础性、前沿科学技术研究，着重培养拔尖创新人才。<ref>{{Cite web|url=http://www.wias.org.cn/chinese-detail-370-8935.html|title=西湖大学正式获教育部批准设立|accessdate=2018-04-17|work=西湖大学}}</ref>

== 探索创新 ==

====== 探索中国高等教育办学体制改革 ======
该校由杭州西湖教育基金会筹集社会资源，实行董事会领导下的校长负责制。<ref>{{Cite web|url=http://tech.sina.com.cn/i/2018-10-20/doc-ihmrasqs9174656.shtml|title=西湖大学成立 施一公:努力做中国高等教育改革探索者|accessdate=2018-11-09|work=tech.sina.com.cn}}</ref>

====== 探索建立新型校企关系 ======
秉承“大学和企业的根本目的都是是造福社会企业”的理念，实践这样的校企关系新模式。使得企业高精尖的科学家将能够在西湖大学里有兼职，大学教授可以很大程度上参与企业创新，但同时保证没有利益冲突。<ref>{{Cite web|url=http://news.sciencenet.cn/htmlnews/2018/11/419665.shtm|title=施一公：西湖大学将为探索新型校企关系做出“第一件事”—新闻—科学网|accessdate=2018-11-09|work=news.sciencenet.cn}}</ref>

====== 举办西湖高等教育论坛 ======
[[西湖高等教育论坛|西湖高等教育论坛]]是由西湖大学发起并举办的国际性论坛，计划自2018年起，每两年举办一届。

旨在鼓励学生交流，促进教职员工定期互访，推动跨学科联合研究，加强科研和学术合作，在各国法律框架内，鼓励学术自由。<ref>{{Cite web|url=http://hznews.hangzhou.com.cn/kejiao/content/2018-11/07/content_7094393.htm|title=西湖大学发起并举办首届西湖高等教育论坛 - 杭州网 - 杭州新闻中心|accessdate=2018-11-09|author=|date=|work=hznews.hangzhou.com.cn|publisher=}}</ref>
