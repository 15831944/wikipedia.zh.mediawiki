'''西表島'''（{{jpn|j=西表島|hg=いりおもてじま|rm=Iriomote-jima}}，{{ryu|r=西表|kn=イリムティ|rm=Irimuti}}）是[[日本|日本]][[琉球列島|琉球列島]][[八重山群島|八重山群島]]中面積最大的島嶼，也是日本[[沖繩縣|沖繩縣]]內僅次於[[沖繩島|沖繩島]]的第二大島。西表島的面積有289平方公里，島上人口約有2300人。行政區劃屬於日本沖繩縣[[八重山郡|八重山郡]][[竹富町|竹富町]]。西表島位於[[台灣|台灣]]東北部外海，距離台灣[[宜蘭縣|宜蘭]][[蘇澳港|蘇澳港]]僅約200公里。

== 地理環境 ==
[[File:Iriomote.jpg|thumb]]
西表島被稱為東洋的亞馬遜，處於[[亞熱帶|亞熱帶]]環境，年均溫為[[攝氏|攝氏]]23.4度，月均溫最高為七月的攝氏28.4度，最低為一月的攝氏17.6度。每年六月到九月常遭受[[颱風|颱風]]的侵襲。

島上90%土地為[[亞熱帶|亞熱帶]]原生[[森林|森林]]及[[紅樹林|紅樹林]][[沼澤|沼澤]]所覆蓋，其中80%是受到保護的國有土地，[[西表石垣國立公園|西表石垣國立公園]]佔全島34.3%。西表島平地較少，古見岳（{{lang|ja|こみだけ}}，Komidake）高470[[公尺|公尺]]為島上最高的山。

[[西表山貓|西表山貓]]為島上的特有動物，為[[夜行性|夜行性]]山貓，已瀕臨絕種，目前估計僅存約一百隻。

== 人文 ==
島上部分居民使用[[八重山方言|八重山方言]]。

東經123度45分6.789秒的子午線正好通過西表島西部，島上特地在該位置建立「竹富町子午線接觸館」，作為觀光景點。

== 歷史 ==
於二次戰前曾經爆發[[瘧疾|瘧疾]]流行，因此造成島上開發較晚，直到[[美國治理琉球時期|美國治理琉球時期]]消滅了瘧疾，島上才開始有較顯著的開發。

西表島上曾有煤礦開採。
