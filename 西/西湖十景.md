'''西湖十景'''指分布在[[杭州|杭州]][[西湖|西湖]]周围的十个有代表性的景点。通常是指[[苏堤春晓|苏堤春晓]]、[[曲院风荷|曲院风荷]]、[[平湖秋月|平湖秋月]]、[[断桥残雪|断桥残雪]]、[[花港观鱼|花港观鱼]]、[[南屏晚钟|南屏晚钟]]、[[双峰插云|双峰插云]]、[[雷峰夕照|雷峰夕照]]、[[三潭印月|三潭印月]]以及[[柳浪闻莺|柳浪闻莺]]这十个景点，其中许多又被改编成[[广东音乐|广东音乐]]。西湖十景最早称呼于[[南宋|南宋]]，后随朝代的替换，景点的名称，位置也有所更换。而到[[清朝|清朝]]时则增加到十八景，甚至是二十四景。但其中仍以南宋所传的十景最为著名。1985年，[[杭州|杭州]]市多家单位重新评选了西湖十景，并将新评选的西湖十景命名为“新西湖十景”。而南宋时的西湖十景，则相对的被称为“西湖老十景”。

== 南宋 ==
南宋时的十景位置基本靠近[[西湖|西湖]]或就在湖中。据[[南宋|南宋]][[祝穆|祝穆]]《[[方舆胜览|方舆胜览]]》、[[吴自牧|吴自牧]]《[[梦粱录|梦粱录]]》中记载。当时的西湖十景为：[[平湖秋月|平湖秋月]]、[[苏堤春晓|苏堤春晓]]、 [[断桥残雪|断桥残雪]]、[[雷峰塔|雷峰落照]]、[[南屏晚钟|南屏晚钟]]、[[曲院风荷|曲院风荷]]、[[花港观鱼|花港观鱼]]、[[柳浪闻莺|柳浪闻莺]]、[[三潭印月|三潭印月]]、[[两峰插云|两峰插云]]。西湖十景的称呼虽然起源于南宋，但在当时却并不是很公认。而十景的名称多出现在有关[[西湖|西湖]][[山水画|山水画]]的题名中，或是[[南宋|南宋]]后期的一些诗词的名目中。

== 元代 ==
[[元朝|元朝]]时的西湖十景有别于[[南宋|南宋]]。部分景点的位置已超出了[[西湖|西湖]]的范围。元代西湖十景分别是：六桥烟柳、九里云松、灵石樵歌、[[孤山_(西湖)|孤山霁雪]]、北关夜市、[[葛岭|葛岭朝暾]]、浙江秋涛、冷泉猿啸、两峰白云、西湖夜月。元朝初期，因为战争巨大的破坏，西湖十景一度相当冷落。但在元朝中后期时，又有所恢复。

==明代==

[[明|明]][[张岱|张岱]]作《[[西湖梦寻|西湖梦寻]]》中《西湖十景诗》沿用南宋的十景，另有冷泉猿啸、韬光观海、天竺香市、西溪探海、[[蕉石鸣琴|蕉石鸣琴]]、凤岭松涛、灵石樵歌、九里云松。

== 清代 ==
===康熙年间===
[[清朝|清朝]]时期，西湖十景延续[[南宋|南宋]]西湖十景的称呼。清[[康熙|康熙]]三十八年(1699年)，[[康熙|康熙]]南巡至杭，游览十景后，将“两峰插云”改为“双峰插云”“雷峰落照”(或称“雷峰夕照”)改为“雷峰西照”“南屏晚钟”改为“南屏晓钟”。康熙为十景题字后，浙江地方官吏先后将所书景名，刻石立碑，建亭恭护。于此，西湖十景从只书上有所记载，转为十景所在景点标志。

===雍正年间===
清[[雍正|雍正]]年间，[[浙江总督|浙江总督]][[李卫|李卫]]疏浚西湖，西湖十景转成为“西湖十八景”，十八景分别是湖山春社、功德崇坊、海霞西爽、梅林归鹤、鱼沼秋蓉、莲池松舍、[[宝石山|宝石凤亭]]、亭湾骑射、蕉石鸣琴、[[玉泉_(杭州)|玉泉鱼跃]]、凤岭松涛、湖心平眺、[[吴山|吴山大观]]、天竺香市、云栖梵径、韬光观海及西溪探梅。十八景和南宋时的十景几乎无一景类同。

===乾隆年间===
[[乾隆|乾隆]]年间，乾隆南巡[[杭州|杭州]]时，曾为延续的十景各赋诗一首，刻于景碑碑石阴面，这使[[西湖|西湖]]十景景名开始广为人知。于此，西湖十景被公认为西湖山水的代表。
清[[乾隆|乾隆]]后期，[[乾隆|乾隆]]南巡时在杭州的品题赋诗被列为了杭州二十四景。二十四景中，有十三景是取自[[雍正|雍正]]年间西湖十八景，二十四景分别为湖山春社、[[宝石山|宝石凤亭]]、玉带晴虹、[[吴山|吴山大观]]、梅林归鹤、湖心平眺、蕉石鸣琴、[[玉泉|玉泉鱼跃]]、凤岭松涛、天竺香市、韬光观海、云栖梵径、西溪探梅、小有天园、漪园湖亭、留余山居、篁岭卷阿、吟香别业、瑞石古洞、黄龙积翠、香台普观、澄观台、[[六和塔|六和塔]]、述古堂

== 1985年 ==
[[File:Hupao.jpg|thumb]]是西湖新十景其中一景。]]
1984年由《[[杭州日报|杭州日报]]》社、杭州市园林文物管理局、[[浙江电视台|浙江电视台]]、杭州市旅游总公司、《园林与名胜》杂志五家单位发起，历时8个多月，经5万余人评选出'''西湖新十景'''，于1985年9月22日正式公布，分别为[[云栖竹径|云栖竹径]]、[[满陇桂雨|满陇桂雨]]、[[虎跑梦泉|虎跑梦泉]]、[[龙井问茶|龙井问茶]]、[[九溪烟树|九溪烟树]]、[[吴山|吴山天风]]、[[阮墩环碧|阮墩环碧]]、[[黄龙吐翠|黄龙吐翠]]、[[玉皇山|玉皇飞云]]、[[宝石山|宝石流霞]]。新十景得到了许多肯定，但在影响上，仍不及南宋时期的西湖十景深远。<ref>{{cite web |title=20年前的“新西湖十景”让杭州西湖蒙羞了吗？ |publisher=杭州网 |url=http://www.hangzhou.com.cn/20040101/ca488679.htm |date=2004年7月30日 |access-date=2005-08-07 |archive-url=https://web.archive.org/web/20050404033635/http://www.hangzhou.com.cn/20040101/ca488679.htm# |archive-date=2005-04-04 |dead-url=yes }}</ref>

== 2007年 ==
2007年“三评西湖十景”，历时九个月，经33.86余万人参与投票，在10月27日第九届[[中国杭州西湖博览会|中国杭州西湖博览会]]开幕式时公布：[[灵隐寺|灵隐禅踪]]、[[六和塔|六和听涛]]、[[杭州岳王庙|岳墓栖霞]]、[[湖滨晴雨|湖滨晴雨]]、[[钱王祠|钱祠表忠]]、[[万松书院|万松书缘]]、[[杨公堤|杨堤景行]]、[[三台山|三台云水]]、[[梅家坞村|梅坞春早]]、[[北山路|北街梦寻]]。<ref>{{cite web |title=十处好风景 十个新名称 |author=傅一览, 麻剑辉 |publisher=青年时报 |url=http://www.qnsb.com/fzepaper/site1/qnsb/html/2007-10/28/content_134845.htm |date=2007年10月28日}}</ref>

== 2013年 ==
2013年，西湖十景作为文化景观，被[[中华人民共和国国务院|国务院]]整体列入第七批[[全国重点文物保护单位|全国重点文物保护单位]]名单。<ref>{{Cite web |url=http://www.sach.gov.cn/col/col2128/index.html |title=第七批全国重点文物保护单位名单 |date=2013年 |publisher=中华人民共和国国务院 |access-date=2014-08-26 |archive-url=https://web.archive.org/web/20140606142512/http://www.sach.gov.cn/col/col2128/index.html |archive-date=2014-06-06 |dead-url=yes }}</ref>
