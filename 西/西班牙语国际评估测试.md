'''西班牙语国际评估测试'''（{{lang-es|Servicio Internacional de Evaluación de la Lengua Española}}，'''SIELE'''）是针对不同年龄层段的应试人群开设的考试，适用于所有西班牙语使用者（包括西班牙语[[母语|母语]]使用者及[[外语|外语]]使用者）。
==历史==
经过4年多的准备，“西班牙语国际评估测试”在2016年3月[[波多黎各|波多黎各]]第七届西班牙语国际会议（CILE）上正式发布。测试的主办方是[[西班牙|西班牙]][[塞万提斯学院|塞万提斯学院]]、[[萨拉曼卡大学|萨拉曼卡大学]]、[[西班牙电信|西班牙电信]]、[[墨西哥国立自治大学|墨西哥国立自治大学]]和阿根廷[[布宜诺斯艾利斯大学|布宜诺斯艾利斯大学]]。<ref>{{cite web |first=J.|last=Ruiz Mantilla|title=Examínate de español en cualquier parte |url=https://elpais.com/cultura/2016/03/16/actualidad/1458149708_262795.html |publisher=[[国家报|EL País]] |location=[[圣胡安_(波多黎各)|San Juan]] |language=es |date=2016-03-17}}</ref>{{as of|2019}}，全球已有55个国家设有超过290个SIELE授权考试中心。<ref>{{cite web |title=¡Español para todos! SIELE ha llegado para quedarse |url=https://www.telefonicaeducaciondigital.com/-/-espanol-para-todos-siele-ha-llegado-para-quedarse |publisher=Telefónica |accessdate=2019-03-09 |language=es}}</ref>

2018年4月9日，75所大学和机构的代表出席了[[萨拉曼卡|萨拉曼卡]]SIELE第二次会议。会议强调了西班牙语在[[中国|中国]]、[[巴西|巴西]]和[[美国|美国]]的重要性。<ref>{{cite web |title=El SIELE da un paso más para su expansión por el mundo |url=https://www.lagacetadesalamanca.es/hemeroteca/siele-da-paso-expansion-mundo-FVGS234540 |publisher=Grupo Promotor Salmantino |accessdate=2019-03-09 |language=es}}</ref>
==考试概要==
该考试与[[托福|托福]]类似。<ref>{{cite web |title=SIELE, un examen universal de español, se aplicará ya el próximo curso |url=https://www.notimerica.com/cultura/noticia-siele-examen-universal-espanol-aplicara-ya-proximo-curso-20150701142749.html |publisher=Notimérica |accessdate=2019-03-09 |location=[[墨西哥城|México]] |language=es |date=2015-07-01}}</ref>整个考试过程均通过数字化手段完成，包括注册、报名、咨询和查询。考生可以通过[[电脑|电脑]]自主选择考试时间。考生将会在三周之内收到测试结果，并可以自行打印证书或测试报告。成绩有效期为五年。

完整的考试分为四个部分：阅读理解、听力、书面表达和口头表达与互动。总共考试时长为3小时，总分为1,000。
