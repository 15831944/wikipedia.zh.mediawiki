'''邹承鲁'''（{{bd|1923年|5月17日|2006年|11月23日|catIdx=Z邹}}），祖籍[[江苏省|江苏]][[无锡市|无锡]]，生于[[青岛市|青岛]]，中国[[生物化学|生物化学]]家，[[中国科学院院士|中国科学院院士]]，前[[中国科学院生物物理研究所|中国科学院生物物理研究所]]所长。

== 生平 ==
[[File:Zou_Chenglu_aboard_a_ship_to_England.jpg|thumb]]

邹承鲁1923年出生于青岛。1935年，就读于长沙雅礼中学。1941年，邹承鲁高中毕业于[[重庆南开中学|重庆南开中学]]，同年考入[[国立西南联合大学|西南联合大学]]化学系。1945年毕业后，他以第一名的优异成绩得到了[[庚子赔款|庚子赔款]]公费出国留学的机会。1946年，他进入了[[英国|英国]][[剑桥大学|剑桥大学]]，从师于著名生物化学家凯林教授，从事[[呼吸链酶|呼吸链酶]]研究。在研究生期间，他单独署名的论文在英国[[自然_(期刊)|《自然》杂志]]发表。<ref name=":0">{{Cite news|url=http://news.sina.com.cn/c/2006-11-30/114911663671.shtml|title=中国生化界泰斗邹承鲁：一个说真话的人走了|author=|date=|work=|accessdate=}}</ref>
1951年，他在获得了[[英国|英国]][[剑桥大学|剑桥大学]]生物化学[[博士|博士]]学位后立即回国，开始在中国科学院上海生理生化研究所工作。1958年，他与年轻的同事们参与了人工全合成[[牛胰岛素|牛胰岛素]]的工作，负责A、B链的拆分和重组，并于1965年取得成功。20世纪60年代，他建立了“邹氏公式”和“邹氏做图法”，后为国际上多种教科书详细介绍。1970年，邹承鲁调到北京，开始在中国科学院生物物理研究所工作，并发表了文革后中国科学家在《自然》上的第一篇论文。{{来源请求}}
1978年至1993年，他当选为[[全国政协|全国政协]]委员，1993年至1997年当选全国政协常委。1980年当选为中国科学院学部委员（后称院士）。{{来源请求}}
邹承鲁曾经担任国内外一些著名科学期刊的编委。1981年─1982年，他应聘担任[[美国|美国]][[哈佛大学|哈佛大学]]访问教授。1986年─1990年间，他曾分期担任[[美国国立健康研究所|美国国立健康研究所]][[高级研究员|高级研究员]]。邹承鲁还致力于越过政治障碍，促进海峡两岸科学界同行进行学术交流、参加学术组织和学术交流。2006年11月23日5时22分，邹承鲁因[[癌症|癌症]]医治无效在北京逝世。<ref name=":0" />

== 主要贡献 ==
* 世界首次[[人工合成结晶牛胰岛素|人工合成结晶牛胰岛素]]（多单位协作，邹承鲁为突出贡献者之一）
* [[蛋白质|蛋白质]]必需基团的化学修饰和[[酶|酶]]活性丧失的定量关系工作（公式及方法分别被称“[[邹氏公式|邹氏公式]]”、“[[邹氏作图法|邹氏作图法]]”）

== 捍卫真理 ==
邹承鲁本人一生坚决反对[[学术造假|学术造假]]、[[学术腐败|学术腐败]]，追求真理、实事求是。为了维护科学尊严，反对不正之风，他曾经在杂志和报刊上撰文40余篇。

邹承鲁对[[科学家|科学家]]应该是什么样的人有过一个说法，他认为：“一个真正的科学家，即便由于种种原因不能戳穿假话，或者无法说出真话，或者可以选择不说话，但至少要做到不说假话，不主动用假话去邀功请赏，去谋财害命。这是科学道德的最低的一条底线。”<ref>[http://www.guokr.com/article/439923/ 果壳网：国家自然科学奖一等奖都颁给过谁？ 2015-02-05]</ref>

===“核酸营养”事件===
[[珍奥核酸|珍奥核酸]]公司自从1990年代起，就在全国范围内推销他们的[[核酸|核酸]][[保健品|保健品]]。他们为了让人们相信产品的营养价值，还邀请了很多研究核酸的专业人士为他们做宣传。2000年上海工业微生物研究所的一位高级工程师以生化学会工业生化专业委员会副秘书长的名义在上海的一家报纸上发表介绍核酸营养的文章。邹承鲁发现后，立刻给这位学者写了信，告诫他不要用自己学会的头衔发表文章，为商家说话<ref>{{cite web|author=李虎军; 朱鹏程|url=http://learning.sohu.com/07/62/article214416207.shtml|title=邹承鲁：“核酸风波”不是学术之争|publisher=[[南方周末|南方周末]]|date=2001-08-24|accessdate=2018-07-21}}</ref>。邹承鲁还在媒体上发表意见，大意就是：第一，核酸到处都有，正常饮食中一定不会缺乏核酸，既然不缺乏，就不必要补充；第二，核酸吃得过多是有害的，尤其是对于痛风病人。
不过，众多学者的质疑还是抵不过珍奥核酸铺天盖地的广告。但是，邹承鲁没有想到退缩。直到晚年多病的时候，他还写文章揭露。他经常强调科学工作者应该遵循的道德底线：“一个真正的科学家，即便由于种种原因不能戳穿假话，或者无法说出真话，或者可以选择不说话，但至少要做到不说假话，不主动用假话去邀功请赏，去谋财害命。”<ref>《[http://www.cctv.com/news/science/20031013/100322_2.shtml 邹承鲁：痛击科学腐败]》，CCTV《[[面对面|面对面]]》</ref>

== 其他观点 ==
邹承鲁曾与海外学者[[饶毅|饶毅]]、[[中华人民共和国科技部|中华人民共和国科技部]]顾问[[鲁白|鲁白]]在英国著名期刊《[[自然_(期刊)|自然]]》雜誌上撰文，建议撤销科技部。邹承鲁也曾建议重建[[国立西南联合大学|国立西南联合大学]]，真正重塑[[大学精神|大学精神]]。

== 家庭 ==
[[File:Li_Siguang_family.jpg|thumb]]与邹承鲁的婚礼]]
