'''邹运明'''（{{bd|1962年|||}}）[[河北省|河北省]][[沧州市|沧州市]][[河间市|河间市]][[米各庄镇|米各庄镇]]希插村人，[[中国人民解放军少将|中国人民解放军少将]]<ref name=hbxw>{{Cite web |url =  http://hebei.hebnews.cn/2015-08/27/content_4995534.htm|title =  河北人的骄傲！看邹运明将军带领阅兵方队|author =  |date = 2015-08-27 |publisher = 河北新闻网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
邹运明是[[中国共产党|中国共产党]]党员。1979年进入[[河北科技大学|河北化工学院]]学习。1983年毕业后参军<ref name=hbxw/>。后毕业于[[中国人民解放军陆军步兵学院|中国人民解放军石家庄陆军学院]]，获军事学硕士学位。他还曾获古代文学硕士学位。历任陆军某集团军排长、指导员、师政治部组织科干事，集团军党委秘书、秘书处处长、干部处处长，第一一二师政治部主任，[[中国人民解放军陆军第三十八集团军|陆军第三十八集团军]]装甲第六师政委等职。2009年升任陆军第三十八集团军副政委。2011年，接替[[吴刚_(中将)|吴刚]]升任陆军第三十八集团军政委。2015年9月，在[[纪念中国人民抗日战争暨世界反法西斯战争胜利70周年大会|纪念中国人民抗日战争暨世界反法西斯战争胜利70周年大会]]中，邹运明作为新型坦克方队领队参加阅兵式，也成为56位将军领队中唯一的陆军集团军政委。2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，陆军第三十八集团军番号撤销，邹运明也成为该集团军末任政委。2017年，邹运明出任东部战区陆军副政委兼东部战区陆军党委政法委书记<ref name=hbxw/><ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1733341 |title =  原“万岁军”政委邹运明出任东部战区陆军副政委|author =  |date = 2017-07-14  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.163.com/15/0831/14/B2BT6MQU00014JB6.html |title = 走进“阅兵村”：将军领队邹运明 唯一集团军政委 |author =  |date =2015-08-31 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2010年7月，晋升为少将军衔<ref name=pp/>。
