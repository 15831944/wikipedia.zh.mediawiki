'''邹炎'''（{{bd|1927年|7月23日|1992年|11月6日}}），出生于[[山东省|山东省]][[福山縣_(登州)|福山县]][[芝水区|芝水区]]山后村，中国人民解放军空军将领。

== 生平 ==
早年随父[[闯关东|闯关东]]到[[哈尔滨|哈尔滨]]做生意，后担任汽车修理工，由于日军进犯，他跟着父亲回到山东老家。1945年12月，参加了中国共产党创办的山东大学青年训练班学习，参加过宣传队。1946年5月1日，加入[[中国共产党|中国共产党]]。同年6月初，山东省选派学员迅速赴东北工作，邹炎被选中，在龙口乘木船渡渤海湾，到辽宁省[[庄河县|庄河县]]登陆，然后再改乘汽车至安东。8月1日，分配到民主联军航空学校学习飞行。1948年4月，随着[[东北人民解放军航空学校|东北人民解放军航空学校]]培训规模的扩大，第2期飞行班开始训练。1949年8月毕业，编入航校飞行队<ref name=as />。

当时，上海地区连遭到国军空军轰炸，中华人民共和国政府商请苏联派空军部队协助保卫上海地区的安全。巴基斯基中将率部于1950年2、3月间到达上海、南京、徐州等地，担负防空任务。1950年5月，中央军委决定以各航校速成班毕业的飞行员为骨干组成[[中国人民解放军空军航空兵第一师|空军混成第4旅]]，于6月19日在南京宣告成立。7月25日，空军混成第4旅由徐州转驻[[上海龙华机场|上海龙华机场]]，后移驻[[上海大场机场|上海大场]]、[[虹桥机场|虹桥机场]]，在苏联顾问的帮助下进行改装[[米格-15战斗机|米格-15战斗机]]，邹炎成为中华人民共和国首批喷气式战斗机飞行员，并担任10团射击主任。10月17日，由华东军区司令员[[陈毅|陈毅]]主持，空军副司令员[[常乾坤|常乾坤]]、华东军区空军司令员[[聂凤智|聂凤智]]参加，接收了苏联巴基斯基中将所部的装备，包括38架米格-15飞机<ref>{{Cite book|title=浦江风云|last=[[秦义民|秦义民]]编|first=|publisher=上海：[[汉语大词典出版社|汉语大词典出版社]]|year=2001.04|isbn=7-5432-0528-9|location=|pages=258}}</ref>。1951年月27日，邹炎驾驶米格-15跟随编队组成搜索，遇到美军两架[[F-86战斗机|F-86战斗机]]，并成功击落一架战斗机。1952年2月5日，他再次击伤一架F-86战斗机；4月24日，击伤一架F-80战斗机<ref name=as>{{Cite book|title=飞歼空中敌：寻访空军飞行员战斗英雄|last=萧邦振，牛锐利等著|first=|publisher=北京：[[解放军出版社|解放军出版社]]|year=2008.01|isbn=978-7-5065-5504-3|location=|pages=200-208}}</ref>。1953年3月，经过休整的空四师重回朝鲜战场，转场[[大孤山机场|大孤山机场]]<ref>{{Cite book|title=中国人民解放军空军气象工作回忆录 上|last=空军司令部气象局编|first=|publisher=空军司令部气象局编|year=|isbn=|location=|pages=191-192}}</ref>，邹炎升任空4师10团团长<ref>{{Cite book|title=寻找牟敦康 追踪长眠于海底的志愿军飞行员|last=清秋子著|first=|publisher=北京：中国环境科学出版社|year=2011.11|isbn=978-7-5111-0778-7|location=|pages=235}}</ref>，率领八架米格-15比斯，与22架美军飞机在高空作战，美军被摧毁四架、志愿军被摧毁两架<ref name=as />。朝鲜战争中，立一等功<ref>{{Cite book|title=建国后军事行动全录-战争秘闻|last=邓礼峰著|first=|publisher=太原：山西人民出版社|year=1994.03|isbn=7-203-02298-6|location=|pages=571}}</ref>，获“二级英雄”称号<ref>{{Cite book|title=志愿军英雄谱|last=何焕昌主编；邹世勇等副主编；惠州市抗美援朝历史研究会，《志愿军英雄谱》编纂委员会编|first=|publisher=北京：[[中共党史出版社|中共党史出版社]]|year=2013.04|isbn=978-7-5098-2033-9|location=|pages=747}}</ref>。

1953年9月，他升任空4师副师长，分管飞行训练。1956年3月30日，空4师改称[[中国人民解放军空军航空兵第一师|中国人民解放军空军第1师]]，师辖第1、2、3团，邹炎担任空军第1师首任师长<ref>{{Cite book|title=戎马春秋：一个红军打旗兵的回忆|last=李世安著|first=|publisher=北京：[[军事科学出版社|军事科学出版社]]|year=1995.01|isbn=7-80021-828-7|location=|pages=245}}</ref>。1966年3月3日至1969年3月1日，担任[[中国人民解放军空军第二军|中国人民解放军空军第二军]]军长。1969年3月至1973年5月，担任[[中国人民解放军沈阳军区空军|中国人民解放军沈阳军区空军]]副司令员<ref>{{Cite book|title=共和国将帅全纪录 第5卷|last=李广智，杨瑞华，贺方兴主编|first=|publisher=[[党史研究出版社|党史研究出版社]]|year=2010.09|location=|pages=2198}}</ref>。1973年5月至1977年2月，担任[[中国人民解放军空军|中国人民解放军空军]]副司令员、[[中国共产党中国人民解放军空军委员会|中国共产党中国人民解放军空军委员会]]常务委员<ref name=gswshang>{{Cite web |url =http://www.hprc.org.cn/gsw/detail_zzslk.jsp?channelid=75034&record=3150  |title =  附卷二  中国人民解放军组织(1949.10—1997.9) 上部　过渡时期和社会主义建设时期(1949.10—1966.5) 第四章　中国人民解放军各军兵种 第二节　空军及空军党委|author =  |date =  |publisher = 中华人民共和国国史网 |language =  |accessdate =  2017-09-16 |quote =  }}</ref>。1984年4月，离职休养。1992年11月6日，在[[北京|北京]]逝世，享年65岁。
