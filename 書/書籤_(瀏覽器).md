'''書籤'''（'''Bookmark'''）是[[瀏覽器|瀏覽器]]中用以儲存[[網址|網址]]的一個設計，方便使用者不須以紙筆抄寫或記住網址即能迅速連結至網站。所有現代網頁瀏覽器都有書籤功能。書籤在[[Internet_Explorer|Internet Explorer]]中叫'''收藏夹'''（簡體中文）或'''我的最愛'''（繁體中文）。

== 歷史 ==
自1993年[[Mosaic|Mosaic]]瀏覽器推出以來，書籤已被納入到瀏覽器中<ref>{{cite web |title=New X-based information systems browser available |url=http://groups.google.com/group/comp.infosystems/browse_thread/thread/f39e59346ee6bc8d/563e36cb72f214d7?lnk=st&rnum=27#563e36cb72f214d7 |publisher=[[Marc_Andreessen|Marc Andreessen]] |date=1993-02-16}}</ref>。書籤功能在Mosaic和[[Opera電腦瀏覽器|Opera]]早期版本被稱為熱門名單（Hotlists）<ref>[http://www.w3.org/News/9305.html "May World-Wide Web News"] in 1993 by Tim Berners-Lee</ref>。其他早期的網頁瀏覽器，如[[ViolaWWW|ViolaWWW]]<ref>{{cite book|author1=James Gillies|author2=R. Cailliau|title=How the Web was born: the story of the World Wide Web|url=http://books.google.com/books?id=pIH-JijUNS0C&pg=PA213|year=2000|publisher=Oxford University Press|isbn=978-0-19-286207-5|pages=213–217}}</ref>和[[Cello_(瀏覽器)|Cello]]<ref>{{Cite book|first=Neil |last=Randall|title=Teach yourself the Internet: around the world in 21 days|url=http://books.google.com/?id=0CpQ97qrcBgC|accessdate=28 March 2010|year=1994|publisher=Sams|isbn=978-0-672-30519-1|pages=211–212}}</ref>也有書籤功能。

== 即時書籤 ==
[[Mozilla_Firefox|Mozilla Firefox]]在2004年推出了即時書籤<ref>[http://www.mozilla.org/press/mozilla-2004-11-09.html "Mozilla Foundation Releases the Highly Anticipated Mozilla Firefox 1.0 Web Browser"], press release on November 9, 2004</ref>，即時書籤的功能跟RSS[[聚合器|聚合器]]差不多。用戶可儲存RSS位址，每次按進該書籤的時候，就可以看到相關網頁的最新更新<ref>[http://moztw.org/docs/fx/live-bookmarks/ 什麼是「即時書籤」（Live Bookmarks）？]</ref>。
