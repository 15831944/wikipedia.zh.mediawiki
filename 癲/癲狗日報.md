《'''癲狗日報'''》（{{lang-en|'''Mad Dog Daily'''}}，{{開始結束|1996年|3月18日}}），1996年由[[黃毓民|黃毓民]]創辦的[[香港|香港]]私營中文報紙，該報有兩個圖標，其一是一隻戴上頸圈的[[老虎狗|老虎狗]]圖案，另一是黃毓民穿[[唐裝|唐裝]]衫、腳踏在一隻[[陶瓷|陶瓷]]老虎狗頭上的照片。癲狗日報以「[[反共|反共]]、反[[董建華|董（建華）]]」作賣點，也是當時其中一份短時間內停刊的香港中文報刊。2018年3月18日，癲狗日報推出網上版復刊<ref name = "龍藝祥">{{cite news|author1=龍藝祥|title=黃毓民創辦《顛狗日報》網上版將復刊|url=https://www.litenews.hk/%E9%BB%83%E6%AF%93%E6%B0%91%E5%89%B5%E8%BE%A6%E3%80%8A%E9%A1%9B%E7%8B%97%E6%97%A5%E5%A0%B1%E3%80%8B%E7%B6%B2%E4%B8%8A%E7%89%88%E5%B0%87%E5%BE%A9%E5%88%8A/|accessdate=2018-04-21|agency=輕新聞|date=2018-03-06}}</ref>。

== 簡介 ==
《癲狗日報》於1996年3月18日創刊，以挑戰[[中國共產黨|中國共產黨]]、保障[[言论自由|言论自由]]作為賣點，售價為五港元。特別自1997年[[香港主權移交|香港主權移交]]後，大部分香港傳媒基於自身利益而[[自我審查|自我審查]]，淡化涉及中國共產黨以及香港特區政府的負面消息，而《癲狗日報》為香港少數仍然會直接批評中華人民共和國政府、香港特區政府及[[香港建制派|香港建制派]]的報章。內容點評時事、世界大事，也有重點的[[賽馬|賽馬]]消息。但報紙銷量未如理想，其後改為純[[馬經|馬經]]，提供獨家賽馬[[貼士|貼士]]及內幕消息，做到「有錢齊齊搵」（有錢一起賺）的信念，賣點為[[黃毓民|黃毓民]]強大的人際網絡，隨時收到馬場內幕消息，管有一隊人馬專門負責「收風」，及建立了一個價值以數百萬計的賽馬資料庫，但銷量依然未見起色，最終停刊。當時另設「毓民贏馬熱線」，聲稱提供獨家贏馬堅料，保證必中，收費每六秒一港元，亦因生意慘淡而停辦。

1997年10月10日，黃毓民再創辦改為《癲狗周刊》，創刊號以[[中華民國國旗|中華民國國旗]]作封面，標題為「[[中國國民黨|國民黨]]不舉，黃毓民[[扯旗|扯旗]]」，並附送該旗幟乙面，售價改為本書20港元，比市面其他周刊貴了不少；不過黃毓民辯稱《癲狗周刊》物有所值，並指「睇完《癲狗周刊》，條氣都順啲，命都長啲」（看完《癲狗周刊》，血氣都順，命都長），1998年7月9日，《癲狗周刊》停刊<ref name = "龍藝祥"/>。

2018年3月18日，癲狗日報推出網上版復刊<ref name = "龍藝祥"/>。
