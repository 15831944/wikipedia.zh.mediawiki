'''斯帕-弗朗科尔尚赛道'''（[[法语|法语]]：'''Circuit de Spa-Francorchamps'''）是位于[[比利时|比利时]][[列日省|列日省]][[斯塔沃洛|斯塔沃洛]]市郊的赛车场。

# 公共汽车弯(Chicane de l'arrêt de bus).

== 赛道特性 ==
斯帕赛道是一条复合型中高速赛道。这里既有长直道，也有非常需要空气动力学表现的中高速弯角。一般来说，F1车队会选择中等的下压力调校。

== 艾尔罗格弯 ==
艾尔罗格弯是斯帕赛道最著名的弯角，也是整个F1世界最著名的弯角之一。两届世界冠军[[費蘭度·阿朗素|费尔南多·阿隆索]]称：“ 车手进弯前是一个下坡，在坡地突然转向，然后是一段极陡的上坡路段。在弯心车手根本看不见出弯点，也不知道车究竟会在何处着陆。艾尔罗格弯在计时圈中相当重要，因为若是在这个弯犯错，在弯后的长直道上你就会损失大把的时间。但更重要的是车手的身体感受，在弯心底部感受到的强烈的压力。这感觉非常奇怪，但也挺有趣的。”

== 斯帕之王 ==
[[迈克尔·舒马赫|迈克尔·舒马赫]]曾在这里六次夺冠。芬兰车手莱科宁在这里也有不俗表现，四次夺魁，并被冠以“斯帕之王”的美誉。在08年的一场经典雨战中，[[奇米·雷克南|奇米·雷克南]]与当时的积分榜领跑者领跑者[[刘易斯·汉密尔顿|刘易斯·汉密尔顿]]在最后几圈展开了激烈缠斗。芬兰车手[[奇米·雷克南|奇米·雷克南]]與英国车手[[刘易斯·汉密尔顿|刘易斯·汉密尔顿]]争抢彎道位置导至[[刘易斯·汉密尔顿|刘易斯·汉密尔顿]]的[[迈凯轮车队|迈凯轮车队]]赛车开出赛道并因此超过了[[奇米·雷克南|奇米·雷克南]]。尔后[[奇米·雷克南|奇米·雷克南]]尝试再超越[[刘易斯·汉密尔顿|刘易斯·汉密尔顿]]时赛车滑出赛道上墙。汉密尔顿赛后亦受到赛事干事处罚。
