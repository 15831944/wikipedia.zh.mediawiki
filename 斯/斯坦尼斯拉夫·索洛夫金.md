'''斯坦尼斯拉夫·亚历山德罗维奇·索洛夫金'''（{{lang-rus|Станислáв Алексáндрович Соло́вкин}}，1977年1月10日－）是一位[[俄罗斯|俄罗斯]][[记者|记者]]、[[导演|导演]]和[[制片人|制片人]]。<ref>http://www.knews.kg/analitika/60496_shelkovyiy_teleput/</ref><ref>http://www.trud.ru/article/15-06-2000/7507_rynok_rabov.html</ref>他最出名的作品有《[[:en:Last_Hero|最后的英雄]]》，《[[:ru:Жди_меня_(телепередача)|等着我]]》）等热门电视节目。<ref>{{cite web |url=http://lenta.ru/mideast/2004/04/08/ernst/ |title=存档副本 |accessdate=2015-04-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20150403162001/http://lenta.ru/mideast/2004/04/08/ernst/ |archivedate=2015-04-03 }}</ref><ref>http://www.regnum.ru/expnews/132737.html</ref><ref>http://www.vokrug.tv/article/show/Davno_ne_videlis_s_Olgoi_Arosevoi_31939/</ref> 

索洛夫金的两部作品《奴隶市场》和《斯大林之谜——一个传记版本》在被俄罗斯电视[[:en:TEFI|TEFI]]奖提名并进入前三名。<ref>{{cite web |url=http://www.tefi.ru/ru/tefi/tefi-vict1997/ |title=存档副本 |accessdate=2015-04-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20150629001927/http://www.tefi.ru/ru/tefi/tefi-vict1997/ |archivedate=2015-06-29 }}</ref>作为制片人，他还参与拍摄了著名的艾美奖获奖真人秀节目《[[极速前进|极速前进]]》（美国[[:en:CBS|CBS]]电视网）。<ref>http://www.imdb.com/name/nm5439694/</ref><ref>http://www.cbs.com/shows/amazing_race/about/</ref><ref>http://www.emmys.com/shows/amazing-race</ref>

==生平 ==

斯坦尼斯拉夫·索洛夫金于1977年1月10日在莫斯科出生。<ref>http://www.dp.ru/a/2003/07/07/V_gotovnosti_k_oblomu_nash/</ref><ref>http://lenizdat.ru/articles/130052/12.</ref><ref>http://www.labyrinth.ru/content/card.asp?cardid=33898</ref>

17岁时他就在《青年新闻》（[[第一頻道|ORT]]）节目得到了一份工作，在那里他首先担任记者职位，然后做了故事片制片人和导演。<ref>http://www.dp.ru/a/2003/07/07/V_gotovnosti_k_oblomu_nash/</ref><ref>http://www.labyrinth.ru/content/card.asp?cardid=33898</ref>

1996年他被莫斯科大学新闻系录取，他于2002年毕业。<ref>http://www.dp.ru/a/2003/07/07/V_gotovnosti_k_oblomu_nash/</ref><ref>http://www.labyrinth.ru/content/card.asp?cardid=33898</ref>

从1996年到1997年，他是新闻周报《新报》的通讯员。<ref>http://www.dp.ru/a/2003/07/07/V_gotovnosti_k_oblomu_nash/</ref><ref>http://lenizdat.ru/articles/130052/</ref>

1997年，索洛夫金开始为电视公司“VID”工作，<ref>http://www.dp.ru/a/2003/07/07/V_gotovnosti_k_oblomu_nash/</ref><ref>http://saint-petersburg.ru/m/spb/old/25188/</ref><ref>https://zonakz.net/articles/13557</ref><ref>http://lenizdat.ru/articles/130052/</ref>多年来他在其中为诸如《Vzglyad》、《等着我》、《迈出一步》、《另一种生活》、《发生方式》<ref>http://www.labyrinth.ru/content/card.asp?cardid=33898</ref>的节目还有许多其他节目担任作家、编剧、编辑、制片人兼导演。在这同时，他是纪录片《奴隶市场》的作者和导演，<ref>https://zonakz.net/articles/13557</ref><ref>http://www.trud.ru/article/15-06-2000/7507_rynok_rabov.html</ref><ref>http://www.regnum.ru/news/analitics/132737.html</ref>这是一部TEFI入围片。该片获得了一致好评。<ref>{{cite web |url=http://28.arhivesetey.ru/archive/46432495117/6/b849a6ec97841027ab2724f978dc8df3.php |title=存档副本 |accessdate=2015-04-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20150403125840/http://28.arhivesetey.ru/archive/46432495117/6/b849a6ec97841027ab2724f978dc8df3.php |archivedate=2015-04-03 }}</ref> ​​索洛夫金领导了一个电视摄制组，其拍摄了[[第二次车臣战争|第二次车臣战争]]事件。<ref>https://zonakz.net/articles/13557</ref><ref>http://www.regnum.ru/expnews/132737.html</ref> 

索洛夫金于21世纪初移居到以色列，在那里，他在俄语频道“以色列+ ”工作。<ref>http://www.dp.ru/a/2003/07/07/V_gotovnosti_k_oblomu_nash/</ref>后来，他在《彩色电视》杂志担任副主编并成为了《最后的英雄》节目的制片经理以及稍后的共同执行制片人，此节目是世界著名的《幸存者》特许节目的俄罗斯版。<ref>http://www.regnum.ru/news/analitics/132737.html</ref><ref>http://www.fontanka.ru/2003/07/02/50150/</ref> 

自2004年以来，他为多家制作俄罗斯电视节目的电视公司担任负责人。<ref>{{cite web |url=http://www.sem40.ru/lenta/news-dir/81562.html |title=存档副本 |accessdate=2015-04-02 |deadurl=yes |archiveurl=https://archive.is/20150314230844/http://www.sem40.ru/lenta/news-dir/81562.html |archivedate=2015-03-14 }}</ref>作为一个执行制片人，他参与创作了《罗曼诺夫爱与死之家》、《罗曼诺夫：末日》、《斯大林之谜——一个传记版本》 （TEFI全国大赛决赛入围节目）等电视节目。<ref>{{cite web |url=http://www.tefi.ru/ru/tefi/tefi-vict1997/ |title=存档副本 |accessdate=2015-04-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20150629001927/http://www.tefi.ru/ru/tefi/tefi-vict1997/ |archivedate=2015-06-29 }}</ref>他还制作了喜剧节目《第一频道喜剧》、《滑稽人物》、《Umora》、<ref>http://www.yuga.ru/news/53210/</ref><ref>{{cite web |url=http://www.dni.ru/news/showbiz/2005/7/14/66666.html |title=存档副本 |accessdate=2015-04-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20150403145428/http://www.dni.ru/news/showbiz/2005/7/14/66666.html |archivedate=2015-04-03 }}</ref>《滑稽图片》、真人秀《横冲直撞》和《塔》。<ref>http://mediananny.com/raznoe/22944/Voijna_formatov__Kto_pobedit_v_pryzhkah_s_vyshki___Pljusy_ili__Pervyij_kanal__Rossii_</ref> 

自2008年以来，他曾与CBS哥伦比亚广播公司电视网（美国）合作，他帮助该公司完成了《极速前进》的9季节目。<ref>http://www.imdb.com/name/nm5439694/</ref>该节目获得了多个艾美奖。<ref>http://www.cbs.com/shows/amazing_race/about/</ref>

在2014年，他与著名制片人弗拉基米尔·卡塔什科夫一起创办了电视和电影制作公司“翱翔制作”。<ref>http://www.knews.kg/analitika/60496_shelkovyiy_teleput/</ref><ref>{{cite web |url=http://soartv.tv/about.html |title=存档副本 |accessdate=2015-04-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20150404151318/http://soartv.tv/about.html |archivedate=2015-04-04 }}</ref><ref>http://viperson.ru/wind.php?ID=488372</ref><ref>http://www.kino-teatr.ru/kino/producer/ros/281102/bio/</ref><ref>http://www.imdb.com/name/nm5237599/</ref>

2015年索洛夫金在圣彼得堡推动了一集《[[英国疯狂汽车秀|英国疯狂汽车秀]]》的制作。<ref>http://www.imdb.com/name/nm5439694/</ref>
