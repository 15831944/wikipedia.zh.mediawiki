'''斯瓦尔巴'''（{{lang-no|Svalbard}}）或譯-{'''冷岸群}-島'''，是位於[[北極地區|北極地區]]的[[群島|群島]]，亦是[[挪威|挪威]]最北界的國土範圍。它坐落在[[歐洲|歐洲]]大陸北方，約佇立於挪威大陸與[[北極點|北極點]]兩者正中間。群島橫跨[[北緯|北緯]]74°到 81°、[[東經|東經]]10°到35°。[[斯匹茲卑爾根島|斯匹茲卑爾根島]]（或稱史匹茲卑爾根島）是群島中最大的島嶼，接著是[[東北地島|東北地島]]與[[厄齊島|厄齊島]]，故群岛前称為'''斯匹次卑尔根群岛'''（{{lang-nl|Spitbergen}}）（或稱史匹茲卑爾根群島）。

行政區劃上，斯瓦尔巴群島獨立於挪威本土的[[挪威行政區劃|19個郡]]，為一[[非建制地區|非建制地區]]。群島的最高行政首長為挪威政府委派的[[斯瓦尔巴总督|斯瓦尔巴总督]]。2002年，斯瓦巴群島首府[[朗伊爾城|朗伊爾城]]建立了一個[[朗伊爾社區議會|民選地區議會]]，其職能與挪威本土之市議會相當。其他的居住地、研究基地，則有研究中心[[新奧爾松|新奧爾松]]，採礦基地[[斯維格盧瓦|斯維格盧瓦]]，以及居民以[[俄羅斯人|俄羅斯人]]爲主的採礦聚落[[巴倫支堡|巴倫支堡]]。此外，斯瓦巴群島也是有常駐平民人口的人類[[聚居地|聚居地]]當中最北的。比斯瓦巴群島更北的人類聚居地無常住人口，只有輪替工作的研究人員。

在17至18世紀，此地被當作捕鲸站使用，但其後遭廢棄。20世紀初，開始有人開采斯瓦巴群島上的煤礦，並形成了數個永久聚居地。1920年簽訂的《[[斯瓦尔巴条约|斯瓦尔巴条约]]》確認了挪威對此地的[[主權|主權]]，而1925年通過的《斯瓦尔巴法令》則確立了此地作爲[[挪威王國|挪威王國]]一部分，且為[[自由貿易園區|自由貿易園區]]及[[非軍事區|非軍事區]]的身份。現時，該地只有一家挪威國營公司以及一家俄羅斯國營公司在此采煤。此外，因為[[斯瓦爾巴大學中心|斯瓦爾巴大學中心]]及[[斯瓦尔巴全球种子库|斯瓦尔巴全球种子库]]皆坐落於此，[[研究|研究]]和[[旅游業|旅游業]]成爲了當地的重要輔助工業。群島上所有聚居地並無道路連接，交通主要以[[雪地電單車|雪地電單車]]、船隻、以及[[航空器|航空器]]爲主。[[斯瓦爾巴朗伊爾機場|斯瓦爾巴朗伊爾機場]]為斯瓦巴唯一一個機場，也是當地主要對外門戶。

== 历史 ==
斯瓦尔巴群岛在12世纪由[[挪威|挪威]]人最早发现，但直到1596年，才被[[荷兰|荷兰]][[航海家|航海家]][[威廉·巴伦支|威廉·巴伦支]]命名为“斯瓦尔巴”，意为“寒冷的海岸”，因此也意譯為「-{冷岸群島}-」。1920年，《[[斯瓦尔巴条约|斯瓦尔巴条约]]》的签订，将整个群岛的主权划给挪威。根据条约规定，所有签字国的公民在斯瓦尔巴拥有平等的经商权；同时该群岛维持军备废除的状态，责成挪威保护岛上居民安全及独具特色的自然荒野地貌。

== 地理 ==
1920年簽訂的《[[斯瓦尔巴条约|斯瓦尔巴条约]]》<ref name=cia>{{cite web |url=https://www.cia.gov/library/publications/the-world-factbook/geos/sv.html  |title=Svalbard |work=[[World_Fact_Book|World Fact Book]] |publisher=[[Central_Intelligence_Agency|Central Intelligence Agency]] |date=15 January 2010 |accessdate=24 March 2010}}</ref>將斯瓦巴群島的疆域定為北緯74°到81°、東經10°到35°之間的所有島嶼、坻和礁。<ref name=npi>{{cite news |title=Svalbard |publisher=[[Norwegian_Polar_Institute|Norwegian Polar Institute]] |url=http://npweb.npolar.no/english/geografi/svalbard |accessdate=24 March 2010}}</ref><ref name=treaty>{{cite news |title=Svalbard Treaty |publisher=[[Wikisource|Wikisource]] |url=http://en.wikisource.org/wiki/Svalbard_Treaty |date=9 February 1920 |accessdate=24 March 2010}}</ref>斯瓦巴群島共佔地61,022平方公里，其中最大島[[斯匹茲卑爾根島|斯匹茲卑爾根島]]面積佔全群島一半以上。<ref name="islandsize">{{cite web |url=http://www.ssb.no/english/subjects/02/befsvalbard_en/tab-2009-10-22-01-en.html |title=Population in the settlements. Svalbard |publisher=[[Statistics_Norway|Statistics Norway]] |date=22 October 2009 |accessdate=24 March 2010 |archive-url=https://web.archive.org/web/20140201112128/http://www.ssb.no/english/subjects/02/befsvalbard_en/tab-2009-10-22-01-en.html |archive-date=2014-02-01 |dead-url=yes }}{{cbignore|bot=medic}}</ref>除了在[[熊島|熊島]]和[[希望島|希望島]]上的[[氣象站|氣象站]]外，所有人類聚居地皆位於最大島斯匹茲卑爾根島。<ref name=cia /> 業權方面，挪威政府擁有斯瓦巴100%的土地。

由於位於[[北極圈|北極圈]]内，此地[[冬季|冬季]]有84至128天的[[極夜|極夜]]，而[[夏季|夏季]]則有99至141天的[[极昼|极昼]]。斯瓦巴60%的领土被[[冰川|冰川]]覆盖，另外有30%為荒地，餘下10%則有[[植物|植物]]覆盖。矿藏有[[煤|煤]]、[[磷灰石|磷灰石]]、[[铁|铁]]、[[石油|石油]]和[[天然气|天然气]]等。沿海盛产[[海象|海象]]、[[海豹|海豹]]、[[北极狐|北极狐]]、[[鲸|鲸]]等。斯瓦尔巴65%的地区都被作为[[自然公园|自然公园]]保护，以维护其独特的动植物资源。也正是由于这个原因，整个地区包括将近5,000头[[北极熊|北极熊]]，超过了居民的数量。
斯瓦爾巴群島多山地，最高点[[海拔|海拔]]1,712米。沿岸多[[峡湾|峡湾]]。地处[[北极圈|北极圈]]内，气候寒冷。

== 氣候 ==
群島受到北大西洋暖流支配影響，夏季平均氣溫介於4°C（39°F）到6°C（43°F）之間，冬季一月氣溫則介於−16°C（3°F）到−12°C（10°F）之間，暖流在冬季影響更加明顯，本區明顯比同緯度的俄國與加拿大地區高了將近20°C（36°F）。其中山地比沿海平原夏季約溫暖2°C，冬季則寒冷3°C。由於群島南北橫跨北緯72度至81度的範圍，因此南北冬季平均溫差約5°C左右，夏季則為3°C。總體來說，東南部暖於西北部，群島東部比西部多降水，故有極地[[海洋性气候|海洋性氣候]]的特色。

== 经济 ==
斯瓦巴群島的[[经济|经济]]支柱为[[采矿业|采矿业]]及[[渔业|渔业]]，但[[旅遊業|旅遊業]]、勘探业、高等教育在20世纪末发展迅速。与挪威本土和[[扬马延|扬马延]]不同的是，斯瓦尔巴屬於[[自由经济区|自由经济区]]和[[非军事区|非军事区]]，不属于[[申根区|申根区]]和[[欧洲经济区|欧洲经济区]]<ref>{{cite web |url=http://www.lovdata.no/all/tl-19921127-109-0.html |title=Lov om gjennomføring i norsk rett av hoveddelen i avtale om Det europeiske økonomiske samarbeidsområde (EØS) m.v. (EØS-loven). |publisher=Lovdata |date=10 August 2007 |accessdate=4 March 2012 |archivedate=2012-03-04 |archiveurl=https://www.webcitation.org/65uDDjwpY?url=http://www.lovdata.no/all/tl-19921127-109-0.html |deadurl=no }}</ref>。旅客前往斯瓦爾巴，會在前一站（通常是挪威本土）離開申根區。
== 宗教 ==
群島大多數人口都隸屬於挪威路德宗教會，群島上的天主教信徒是由[[天主教奧斯陸教區|天主教奧斯陸教區]]服務的。
