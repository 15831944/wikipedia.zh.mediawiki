'''斯派克电子游戏大奖'''（{{lang-en|Spike Video Game Awards}}），又名 '''VGAs'''，最后颁发一年称'''VGX'''，是由[[Spike_(电视频道)|Spike TV]]在2003年到2013举办的一年一度的颁奖典礼<ref name="2013VGX">{{cite web|url=http://www.polygon.com/2013/11/15/5107808/spike-video-game-awards-vgx-2013-date|title=Spike Video Game Awards renamed VGX, set for Dec. 7|date=2013-11-15|author=Samit Sarkar|publisher=[[Polygon_(website)|Polygon]]|accessdate=2013-11-15}}</ref>，旨在表彰年度最佳的[[电子游戏|电子游戏]]。2014年由[[傑夫·吉斯利|傑夫·吉斯利]]創辦的[[遊戲大獎|遊戲大獎]]（The Game Awards）被視為是斯派克電子遊戲大獎的繼承者。
