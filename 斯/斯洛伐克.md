'''斯洛伐克共和国'''（{{lang-sk|Slovenská republika}}） 通稱'''斯洛伐克'''，是[[中欧|中欧]]的一个[[内陆国家|内陆国家]]，面积4.9万平方公里，约540万居民。西北邻[[捷克|捷克]]，北邻[[波兰|波兰]]，东邻[[乌克兰|乌克兰]]，南邻[[匈牙利|匈牙利]]，西南邻[[奥地利|奥地利]]，首都和最大城市为[[布拉迪斯拉发|布拉迪斯拉发]]。

斯洛伐克是一个[[议会民主制|议会民主制]]国家，官方语言为[[斯洛伐克语|斯洛伐克语]]。2004年3月29日加入[[北大西洋公约组织|北大西洋公约组织]]，从2004年5月1日起成为[[欧洲联盟|欧洲联盟]]的一员，在2007年12月21日成為[[申根公約|申根公約]]會員國。2009年1月1日起加入[[欧元区|欧元区]]。现在斯洛伐克被分类为中欧国家，但是在历史上很长一段时间属于东欧。

== 历史 ==
{{main|斯洛伐克歷史}}
最初在今天的斯洛伐克定居的是[[凯尔特人|凯尔特人]]，前5世纪开始他们被属于[[日耳曼人|日耳曼人]]的部落排挤，从1世纪到5世纪今天的斯洛伐克是罗马人和日耳曼人的边界，这里发生了多次边界战争。500年左右[[斯拉夫人|斯拉夫人]]的祖先来到这里。7世纪中斯洛伐克是当时[[萨莫王国|萨莫王国]]（''Samo''）的中心。8世纪时这里出现了[[尼特拉公国|尼特拉公国]]，833年它成为[[大摩拉维亚|大摩拉维亚]]的中心部分。10-11世纪斯洛伐克逐渐成为[[匈牙利王国|匈牙利王国]]的一部分。

1526年随匈牙利歸奥地利君主管辖，由此1867年成为[[奥匈帝国|奥匈帝国]]的一部分。16世纪和17世纪内当东南欧大部分地区被[[鄂图曼帝国|鄂图曼帝国]]占领时，匈牙利王国的实际组成部分只有今天的斯洛伐克。

[[第一次世界大战|第一次世界大战]]后，奥匈帝国瓦解，1918年斯洛伐克和捷克一起组成[[捷克斯洛伐克|捷克斯洛伐克]]，它也包括今天匈牙利的一部分，这些地区的主要居民是匈牙利人，同时今天匈牙利境内也有部分说斯洛伐克语的地区。但由于经济和政治上的纠纷，1920年代起，一些斯洛伐克的精英要求从捷克斯洛伐克中分离出去，最终导致了1939年[[斯洛伐克第一共和國|斯洛伐克第一共和國]]成立，得到纳粹德国的支持和保护。1945年，德国战败，苏联占领了斯洛伐克，交由捷克共产党控制。斯洛伐克共和国的首相[[约瑟夫·蒂索|约瑟夫·蒂索]]被审判后绞死，同时大批共和国的支持者流亡海外成为难民。对于这个只有6年寿命的斯洛伐克共和国，不同阵营的人看法完全相反：苏联社会主义阵营称其为“伪政权”。而不少斯洛伐克人认为它是合法政权，是被苏联的强权消灭了。

斯洛伐克被捷克合併后，实行[[社会主义|社会主义]]政策，1948年[[捷克斯洛伐克共产党|捷克斯洛伐克共产党]]开始一党执政。从1969年起，斯洛伐克成为[[捷克斯洛伐克社会主义共和国|捷克斯洛伐克社会主义共和国]]的一個[[加盟共和國|加盟共和國]]。

1989年，捷克斯洛伐克发生民主变革——[[天鵝絨革命|天鵝絨革命]]，实行多党议会民主制。之后，斯洛伐克的分离倾向日益明显，1992年通过全民公投，1993年1月1日斯洛伐克宣布脫離捷克斯洛伐克共和国，成为一个独立的国家，史稱「[[天鵝絨分離|天鵝絨分離]]」（Velvet Divorce）。

2003年5月17日至18日斯洛伐克举行全民公投，92.45%的公民支持加入欧洲联盟。420万选民中只有52.15%参加投票。2004年3月29日斯洛伐克加入[[北约|北约]]，2004年5月1日加入[[欧盟|欧盟]]。

2004年4月3日的总统选举中在首轮选举中，没有任何候選人获得50%的最低投票率，在第二轮选举中[[伊万·加什帕罗维奇|伊万·加什帕罗维奇]]以60%的选票获胜。在2004年举行的[[欧洲議會|欧洲議會]]的选举中只有16.66%的选民参加了投票。

2006年6月17日，新一届国会选举结果揭晓，[[方向党|方向党]]获得29.14%的选票，成为國会第一大党。[[斯洛伐克民主基督教同盟|斯洛伐克民主基督教同盟]]，获得18.35%的选票；接下来是[[斯洛伐克民族党|斯洛伐克民族党]]，获得11.73%的选票。<ref>{{cite news|url=http://news.xinhuanet.com/newscenter/2006-06/18/content_4712833.htm|title=斯洛伐克方向党在议会选举中获胜|publisher=新华网|language=zh-cn|date=2006年6月18日}}</ref>7月4日，斯洛伐克总统加什帕罗维奇正式任命方向党主席[[菲乔|菲乔]]为新政府总理。方向党与斯洛伐克民族党和[[争取民主斯洛伐克运动|争取民主斯洛伐克运动]]组建联合政府。<ref>{{cite news|url=http://news.xinhuanet.com/newscenter/2006-07/05/content_4796005.htm|title=菲乔出任斯洛伐克新政府总理|publisher=新华网|language=zh-cn|date=2006年7月5日}}</ref>

2009年3月21日的总统选举，在首轮选举中，争取连任的总统[[伊万·加什帕罗维奇|伊万·加什帕罗维奇]]获得46.7%的选票，领先于其他候选人。但没有任何候選人获得50%的最低投票率，在第二轮选举中[[伊万·加什帕罗维奇|伊万·加什帕罗维奇]]赢得55.53%的选票，击败对手拉迪乔娃（得票44.47%）成功连任。

2019年3月16日的总统选举第一轮投票中，查普托娃得票率40.57%，排名第一。由于13位候选人得票均未过半，她与排名第二、获得18.66%选票的[[马罗什·谢夫乔维奇|马罗什·谢夫乔维奇]]进入第二轮。<ref>{{cite news |title=斯洛伐克总统选举将举行第二轮角逐 |url=http://www.xinhuanet.com/world/2019-03/17/c_1124245334.htm |accessdate=2019-03-29 |work=新华网 |agency=新华社 |date=2019-03-17 |archive-url=https://web.archive.org/web/20190329175707/http://www.xinhuanet.com/world/2019-03/17/c_1124245334.htm |archive-date=2019-03-29 |dead-url=no }}</ref>2019年3月30日举行的总统选举第二轮投票中，查普托娃以58.40%的得票率当选[[斯洛伐克总统|斯洛伐克总统]]，成为斯洛伐克首位女性总统。她将在6月15日宣誓就职。<ref name="总统">{{Cite web |url = http://cn.rfi.fr/wire/20190331-政治素人赢斯洛伐克总统大选-专家与马克宏相似 |title = 政治素人赢斯洛伐克总统大选 专家：与马克宏相似 |agency=法新社 |publisher=法广 |date = 2019-03-31 |accessdate = 2019-03-31 }}</ref>

=== 名称由來 ===
[[File:Kosice_(Slovakia)_-_Main_Street_4.jpg|thumb]]
有证据称，该地区首次被称为斯洛伐克是在15世纪，而16世纪“斯洛伐克”成为常用称呼。然而5世纪在该处定居的斯拉夫民族，从9世纪开始自称为[[斯拉夫人|斯拉夫人]]，因此“斯洛伐克”之名有可能在15世纪前就已经被使用。

== 政治 ==
{{main|斯洛伐克政治}}
[[File:Bratislava,_Parlament,_Slovensko.jpg|thumb]]]]
[[File:Prezidentsky_palac.jpg|thumb]]
斯洛伐克实行多党[[议会民主制|议会民主制]]。[[斯洛伐克总统|总统]]是国家元首，由国民直接选举，任期五年，最多可连任两届。现任总统为[[安德烈·基斯卡|安德烈·基斯卡]]。

內閣負責人是总理，拥有最高的行政权，一般為议会中最大党或最大的政治联盟的黨魁，由总统任命。内阁的其他成员由总理提名，总统任命。

斯洛伐克的最高立法机关是一个一院制的议会──[[斯洛伐克共和国国家议会|斯洛伐克共和国国家议会]]（{{lang|sk|''Národná Rada Slovenskej Republiky''}}），共有150名议员，任期四年，由国民按比例选出。斯洛伐克的最高法院是宪法法院（{{lang|sk|''Ústavný súd''}}），决定涉及宪法的诉讼。它的13个大法官由总统在一份由议会提交的名单中任命。

== 人口 ==
{{main|斯洛伐克人口}}
斯洛伐克最大的民族是属于斯拉夫人的斯洛伐克人，占总人口的85.8%，最大的少数民族是[[匈牙利人|匈牙利人]]，占9.7%，主要集中在南部和东部。其他有[[吉普赛人|吉普赛人]]（他们人口增长最快）（1.7%）、捷克人（0.8%）等。[[犹太人|犹太人]]从战前的12万到今天只剩下约2300人。

斯洛伐克的宪法保障宗教自由，69%的公民信[[天主教|天主教]]，9%信[[新教|新教]]。

斯洛伐克的官方语言是[[斯洛伐克语|斯洛伐克语]]，属于[[斯拉夫语族|斯拉夫语族]]。在南部[[匈牙利语|匈牙利语]]也很常用。

斯洛伐克人非常喜欢[[石竹花|石竹花]]和[[玫瑰花|玫瑰花]]，认为它们是幸福的象征，并视其为“国花”。

== 经济 ==
{{main|斯洛伐克經濟}}
[[File:Bratislava_Central_Bank3.jpg|thumb]]
斯洛伐克成功地完成了从[[计划经济|计划经济]]到[[自由市场经济|自由市场经济]]的改革。祖林达政府在2001年起為稳定總體经济而进行结构改革。主要的企业私有化过程已经完成，几乎整个[[金融界|金融界]]现在在外资掌握下，外资投资不断扩大。尽管其主要出口市场于2000年后经济萧条，但斯洛伐克的经济发展超过预计。

2002年，斯洛伐克國民所得提高促进了国内需求的增高，克服了出口增长的减低，使得整个经济达到從1998年以来最高的成长率（4.4%）。2003年出口恢复，克服了国内消费的緊縮，使得国家经济成长4.2%。2001年的失业率达19.8%，祖林达政府2002年上台后采取的剧烈的改革使失业率降低，2004年6月降低為13.9%。2000年的通货膨胀率是12.0%，2002年降到3.3%，2003和2004年又有所增长。

2009年1月1日，斯洛伐克正式加入[[歐元區|歐元區]]，成為歐元區第16個國家。
