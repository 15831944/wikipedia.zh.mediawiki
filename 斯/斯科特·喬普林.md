'''斯科特·喬普林'''（{{lang-en|'''Scott Joplin'''}}，{{IPAc-en|ˈ|dʒ|ɒ|p|l|ɪ|n}}，{{bd|1868年||1917年|4月1日|catIdx=Joplin, Scott}}），[[美國|美國]][[非裔美国人|非裔]][[作曲家|作曲家]]、[[鋼琴家|鋼琴家]]，以其[[拉格泰姆|拉格泰姆]]作品而聞名於世，被譽為“拉格泰姆之王”<ref>{{Cite AV media notes| title = Maple Leaf Rag (2nd edition)| year = 1899| url = https://commons.wikimedia.org/wiki/File:Maple_Leaf_Rag_Title.jpg| last = Joplin| first = Scott| page = 1| type = sheet music cover| publisher = John Stark
}}</ref>。喬普林在其短暫的寫作生涯中共譜寫了44支傳統拉格泰姆樂曲，一支拉格泰姆[[芭蕾舞|芭蕾舞曲]]以及兩部歌劇。他的首部作品《楓葉拉格》（{{lang-en|''Maple Leaf Rag''}}）被譽爲[[拉格泰姆|拉格泰姆]]流派的典型作品，影響頗深<ref name=sketch33423>{{cite web|last=Berlin|first=Edward A.|title=Scott Joplin: Brief Biographical Sketch|url=http://www.edwardaberlin.com/scott_joplin__brief_biographical_sketch___33423.htm|publisher=Berlin|accessdate=April 9, 2013}}</ref>。

==背景==
喬普林于1867年下旬或1868年上旬（真實出生日期有爭議，其墓碑所記出生日期為1868年2月11日）出生於[[美國|美國]][[德克薩斯州|德克薩斯州]]東北部的一個音樂世家。他從小便在當地教師的幫助下學習音樂知識。之後，喬普林一家遷往[[德克薩斯州|德克薩斯州]][[特克薩卡納_(得克薩斯州)|特克薩卡納]]市。特克薩卡納是喬普林的成長地，還在該地成立了一個聲樂四重奏，並教授曼陀林和吉他。19世紀80年代後期，喬普林辭去鐵路工人的工作，致力於成爲巡迴音樂家，與樂隊在[[美國南部|美國南部]]舉辦巡迴演出。1893年，喬普林前往[[芝加哥|芝加哥]]，參加[[芝加哥哥倫布紀念博覽會|芝加哥哥倫布紀念博覽會]]，其在博覽會上的演奏使拉格泰姆這一流派在往後幾年迅速风靡美國全国。

1894年，喬普林遷往[[密蘇里州|密蘇里州]]錫代利亞市，以鋼琴教師謀生，其學生包括拉格泰姆作曲家亞瑟·馬歇爾（{{link-en|Arthur Marshall}}）斯科特·海登（{{link-en|Scott Hayden}}）與布倫·坎貝爾（{{link-en|Brun Campbell}}）。喬普林于1895年開始出版音樂，其中于1899年出版的作品《楓葉拉格》使其迅速成名。這支作品對後續的拉格泰姆作曲家的有著深遠的影響。這份作品還給喬普林帶來了穩定的生活收入來源，但喬普林之後便經常遇到經濟問題。

[[File:Scott_Joplin_House.jpg|thumb]]
1901年，喬普林遷往美國[[聖路易斯_(密蘇里州)|聖路易斯]]，繼續其作曲家生涯，並常在聖路易斯社區進行表演。1903年，他因拖欠賬款被政府强制沒收財產，其中包括他的第一部歌劇《榮譽的客人》（''{{link-en|A Guest of Honor}}''），該歌劇現已失傳。<ref name=sketch33423/>

1907年，喬普林再次喬家遷至[[紐約|紐約]]尋找新的歌劇製作人。在這一時期，他試圖擺脫音樂形式與名聲的限制，進行其他音樂的譜曲，但未獲在金錢方面上的成功。他的第二部歌劇《特里莫尼莎》（''{{link-en|Treemonisha}}''）並未獲得好評，在1915年的部分歌劇演出中反響不高。

==逝世==
1916年，喬普林被診斷身患[[梅毒|三級梅毒]]並陷入精神錯亂狀態。1917年1月，他被送往曼哈頓的一家精神病院，曼哈頓國家醫院<ref>[[#Berlin|Berlin (1994)]] p. 239.</ref><ref>{{cite news|url=http://www.time.com/time/magazine/article/0,9171,981443,00.html|title=American Schubert| date=September 19, 1994|accessdate=November 14, 2009|work=Time|first=Michael|last=Walsh}}</ref>。4月1日，喬普林病逝，享年49岁，病因為梅毒誘發的[[失智症|失智症]]。其尸體被埋葬在紐約皇后區東埃爾姆赫斯特的一個沒有任何標記的贫民墳墓中並持續了57年。1974年，喬普林的墓碑被重新標記。

==影響与遗产==
喬普林以及后续的拉格泰姆作曲家在其年代振興了美國流行音樂，通過創作振奮人心、释放情绪的舞蹈曲調，受到歐美裔美國人对非裔美國音樂风格的普遍赞誉。喬普林的逝世被廣泛認為是拉格泰姆退出主流音樂的標記。拉格泰姆這一音樂流派在其逝世后數年逐漸演變成{{link-en|大跨度鋼琴演奏|Stride (music)}}、[[爵士樂|爵士樂]]以及大樂團[[搖擺樂|搖擺樂]]。20世紀70年代初，美國音樂家約書亞·里夫金（{{link-en|Joshua Rifkin}}）在其《Scott Joplin: Piano Rags》專輯中演奏了喬普林的數支音樂作品。該專輯被認為是20世紀70年代初率先重新演繹喬普林音樂的作品，同時該專輯還在首年獲得百萬銷量。1973年，奧斯卡獲獎作品《[[老千計狀元才|老千計狀元才]]》也包含數支喬普林的作品（由[[馬文·漢里殊|馬文·漢里殊]]演奏），包括著名的《演藝人》（''{{link-en|The Entertainer}}''）。1972年，其歌劇《特里莫尼莎》(Treemonisha)终于獲得廣泛的讚譽。1976年，喬普林因“在二百週年之際為美國音樂作出貢獻”被追授[[普立茲獎|普立茲獎]]。<ref name="Pulitzer">{{cite web |url=http://www.pulitzer.org/bycat/Special+Awards+and+Citations |title=Special Awards and Citations |publisher=The Pulitzer Prizes |accessdate=March 14, 2009}}</ref>
