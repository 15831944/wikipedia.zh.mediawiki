'''斯堪尼亞N113型巴士'''（亦稱作'''紳佳N113型'''／'''世冠N113型'''，[[英文|英文]]：{{lang|en|'''Scania N113'''}}）是一款由[[瑞典|瑞典]][[斯堪尼亞|斯堪尼亞]]設計生產的後置引擎巴士底盤，用以取代[[斯堪尼亞N112型|斯堪尼亞N112型]]。於1988年推出的N113底盤尚未採用低地台設計，但就因應日益受到關注的[[環保|環保]]需要，而特別採用當時還不普及的環保[[柴油引擎|柴油引擎]]。

== 設計 ==
N113底盤採用後置引擎設計，可安裝單層或雙層巴士車身，除了因應不同長度而有兩軸及三軸的型號，還有單層鉸接巴士的版本。N113底盤有10.5米、11.3米、11.45米及12米的長度，其中11.45米及12米底盤採用三軸設計。N113採用斯堪尼亞DS11或DSC11的11公升柴油引擎，配用斯堪尼亞或[[福伊特|福伊特]]的波箱。

== 香港 ==
[[File:Gw3430@98A-G.jpg|thumb]]
[[九巴|九巴]]於1990年代初期開始測試環保引擎巴士，除了在兩輛[[丹尼士巨龍|空調巨龍11米]]安裝符合[[歐盟一期|歐盟一期]]的[[吉拿|吉拿]]LG1200引擎進行測試外，也計劃引進首批全新製造的環保引擎巴士，於是九巴在1993年向[[世冠汽車|世冠汽車]]購入兩輛裝配[[亞歷山大車身|亞歷山大車身]]的世冠N113型雙層巴士進行試驗。這兩輛新引入的N113獲編配車隊編號AS1（車牌：FU482）及AS2（車牌：FU2948）。這兩輛N113型空調雙層巴士的底盤是11.45米長的三軸版本，車尾引擎室的右側裝有達到歐盟一期排放規格的DS11-74柴油引擎，輸出馬力274匹，引擎室的左側為[[福伊特|福伊特]]提供的DIWA863全自動波箱。N113雙層巴士與同時期的[[富豪奧林比安|富豪奧林比安]]及[[丹尼士巨龍|丹尼士巨龍]]最大的分別是散熱水箱不是設於車頭，而是位於底盤中軸前方的右側，並以電力驅動散熱系統，大幅減低散熱風扇傳入車廂的噪音，令車廂格外寧靜。這兩輛樣版車於蘇格蘭裝配亞歷山大RH型車身後於1993年運抵香港，九巴當時以「進步環保巴士」宣傳這兩輛N113空調雙層巴士。N113的特色是行車噪音十分細小，巴士在身旁駛過時猶如風一樣<ref>[Scania N113 (AS1-AS22)]，書名：九巴車隊八十年(二)1981-2013，ISBN 9789881648303，尚線出版，第123頁</ref>。

九巴對這兩輛N113巴士的性能感到滿意，於是在1996年增訂20輛同型號巴士，但改用提升至[[歐盟二期|歐盟二期]]排放規格的DSC11-24柴油引擎。第二批N113的車身繼續採用亞歷山大的RH型，但改在[[九巴屯門車廠|九巴屯門車廠]]組裝車身。車廂的座位改為較寬闊的2+2安排，總載客量則因此而減少。第二批巴士的空調系統由[[日本電裝|日本電裝]]提供，雖然空調效能較高，可是空調系統發出音量明顯大於先前的兩輛樣辦車，令第二批N113的車廂環境不如首批寧靜。這批N113大部分時間都服務於將軍澳地區的路線，如{{KMB|98A}}及{{KMB|98D}}，以及行走市區的{{KMB|2A}}。

斯堪尼亞開發的N113雖然率先配備有環保引擎，但底盤的設計導致地台偏高是本車型作為市區巴士的明顯缺點。由於N113的散熱水箱位於車身後半部的地台下方，驅動軸又沒有採用下置式的U字形設計，導致巴士的地台難以降低。此外，九巴的N113因為要安裝空調系統，所以裝配車身後的N113雙層巴士高度達到4.5米，為香港法例對巴士全高的上限，也是香港巴士車款中最高身的型號。N113的地台高度較同期在香港使用的後置引擎雙層巴士為高，位於車頭的上客車門，梯級如同前置引擎的[[勝利二型|勝利二型]]共有兩級，比同樣裝配[[亞歷山大車身|亞歷山大車身]]的[[富豪奧林比安|富豪奧林比安]]高出一級，而且梯級高度較大，令N113的登車較為不便。因為散熱水箱位於底盤中間位置，所以下層所有座位都要升高一級。九巴於1997年引入[[丹尼士三叉戟三型|丹尼士三叉戟三型]]低地台雙層巴士後，香港的巴士服務逐漸走向低地台化，N113的高地台設計不符合潮流，加上N113的車價較[[富豪奧林比安|富豪奧林比安]]及[[丹尼士巨龍|丹尼士巨龍]]昂貴，因此九巴於1996年購入第二批共20輛N113雙層巴士後沒有再增購。

首輛退役的N113型雙層巴士是樣辦車AS2，於2010年3月11日晚上行駛118線期間發生機械故障，由於車齡已屆17年，因此九巴放棄進行修復，成為香港首輛退役的[[斯堪尼亞|斯堪尼亞]]雙層空調巴士；另一輛樣辦車AS1於2011年5月因為車齡屆滿18年而被停止載客服務，並於2011年6月20日退役。餘下20輛於1996年出牌的N113型巴士在服役後期全數轉交[[九巴荔枝角車廠|九巴荔枝角車廠]]管理，隨著行車年期屆滿，所有N113於2014年1月至8月期間相繼退役。最後一輛N113雙層巴士（AS21，GW4537）於2014年8月22日晚上行走翌日取消的{{KMB|87A}}後，N113型空調雙層巴士正式全數退役。

== 新加坡 ==
[[File:ScaniaN113-SBS.JPG|thumb]]
新加坡巴士公司（即現時的[[新捷運|新捷運]]）於1989年至1990年間購入200輛配[[亞歷山大車身廠|亞歷山大]]PS車身的N113CRB單層巴士，其中50輛（車牌SBS31L至SBS80U）為空調巴士，其餘150輛（車牌SBS3573U至SBS3722G）則不設空調設備。此款巴士與[[富豪B10M|富豪B10M]]及[[平治O405|平治O405]]是新加坡於1980年代後期至1990年代的主力單層巴士車型。自2000年起，部分N113CRB非空調巴士（車牌SBS3581X、SBS3628X及SBS3639P至SBS3722G）被加裝空調系統，為新加坡當時其中一款「普改冷」巴士。不設空調設備的款式，已全數於2008年12月8日退役並被拆毀；而設有空調設備的款式 （包括「普改冷」款式），亦已全數於2009年5月退役並被拆毀。新加坡於2007年分批引進1100輛[[斯堪尼亞K230UB|斯堪尼亞K230UB]]取代富豪B10M及斯堪尼亞N113單層巴士。

== 後繼產品 ==
N113型巴士的生產線於1998年終止。單層版本由N94系列接替，但為[[英國|英國]]市場而設的N113DRB雙層巴士一直無後繼車型，直到2002年[[斯堪尼亞|斯堪尼亞]]推出N94UD OmniDekka低地台雙層巴士。

另外，斯堪尼亞於2000年為香港市場推出[[斯堪尼亞K94UB|K94UB]]低地台雙層巴士，但只有一輛樣辦車出廠供應予[[城巴|城巴]]。
