'''斯昆石'''（{{lang|en|'''Stone of Scone'''}}），通称“命运石”或“加冕石”，乃[[苏格兰|苏格兰]]历代[[国王|国王]]加冕时使用的一块[[砂岩|砂岩]]。斯昆石色淡黄，呈长方形，长26英寸，宽16英寸，高11英寸，重336磅。据[[凯尔特人|凯尔特人]]传说，[[雅各|雅各]]看见[[天使|天使]]时（[[圣经|圣经]][[旧约|旧约]][[创世记|创世记]]二十八10~22），正是头枕此石，因此又名“雅各的枕头”、“雅格的支柱”或者“酋长石”。该石曾被保存在现已废弃的[[珀斯-金罗斯|珀斯郡]]的[[斯昆_(城镇)|斯昆修道院]]中，因而得名。

== 历史传统 ==

早在来自[[爱尔兰|爱尔兰]]的[[達爾里阿達王國|达尔瑞达]]王朝征服苏格兰之前，他们就已经有使用斯昆石作为加冕石的传统。随着他们的征服，这也变成了苏格兰的传统。我们可以肯定的是，大约847年的时候，[[肯尼思一世|肯尼思一世]]在其加冕典礼就坐在此石之上。之后的苏格兰国王们都遵循这一传统。斯昆石也因此成为了苏格兰的国家象征之一。四百多年间，这块石头始终没有离开[[斯昆_(城鎮)#斯昆修道院|斯昆修道院]]。

== 西敏寺 ==
[[File:Coronation_Chair_and_Stone_of_Scone._Anonymous_Engraver._Published_in_A_History_of_England_(1855).jpg|thumb]]
1296年，[[爱德华一世|爱德华一世]]将斯昆石作为战利品掳回[[英格兰|英格兰]]，安置在[[西敏寺|西敏寺]]英王加冕宝座“[[圣爱德华宝座|圣爱德华宝座]]”之下，象征英格兰和苏格兰统一在英王的主权下。不过，爱德华是否确实带走了真正的斯昆石，这一直都有争论。据传，[[斯昆|斯昆修道院]]的[[僧侣|僧侣]]们将真正的斯昆石藏在了[[泰河|泰河]]之中（一说埋在Dunsinane Hill）。这很可能是真实的，因为现有的这块斯昆石并不完全符合古人的描述。随着这些僧侣的逝去，没有人能再寻找到真相。

1328年，根据苏格兰和[[英格兰|英格兰]]签署的《{{link-en|爱丁堡-北安普敦条约|Treaty of Edinburgh–Northampton}}》，英王[[爱德华三世|爱德华三世]]同意将斯昆石送还苏格兰，不过，这一允诺从未兑现。

== 争夺和损坏 ==

1950年[[圣诞节|圣诞节]]，四名苏格兰学生(Ian Hamilton, Gavin Vernon, Kay Matheson和Alan Stuart)决意要从[[西敏寺|西敏寺]]将斯昆石盗取出来送还苏格兰。在行动中，他们失手将斯昆石摔成了两半。在[[肯特|肯特]]躲藏了几个星期后，他们成功的将斯昆石藏在一部借来的车後座，穿越了警方布置在苏格兰和英格兰之间的封锁线。学生们将斯昆石交给了[[格拉斯哥|格拉斯哥]]的一位资深政治家。这位政治家安排专家修复了石头，并严加收藏。英国政府展开了大规模搜寻，可是一无所获。

1951年4月11日，苏格兰人认为英国政府已经屈从于民意，于是举行了富有象征意义的庆典，将斯昆石供奉在[[安格斯|安格斯郡]]{{link-en|阿布罗斯修道院|Arbroath Abbey}}的祭坛上，希望交由[[苏格兰国教会|苏格兰国教会]]保管。 但是，[[伦敦|伦敦]]警方一旦知道了斯昆石的下落，就立即将其运回西敏寺。此后，一直有传言警方只拿到了复制品，英王宝座下面的不是真正的斯昆石。

== 回归苏格兰 ==

1996年，英国政府终于决定将斯昆石归还苏格兰，1996年11月15日，在两地边境上，[[英国内政部|英国内政部]]和{{link-en|蘇格蘭部|Office of the Secretary of State for Scotland}}的代表举行了一个交接仪式，之后，斯昆石被带回[[爱丁堡|爱丁堡]]城堡安放至今。由于爱丁堡城堡现在仍是英国驻苏格兰部队总司令部驻地，因此一些苏格兰人仍然希望斯昆石能被转移到更普通的地点安放。

虽然斯昆石现在已经回到苏格兰，但是英国政府仍然发布了法令，规定将来英王举行加冕典礼时，斯昆石仍然需要运回西敏寺，安放在圣爱德华宝座下。
