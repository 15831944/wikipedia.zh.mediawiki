'''斯皮罗·西奥多·阿格纽'''（{{lang-en|Spiro Theodore Agnew}}，{{bd|1918年|11月9日|1996年|9月17日|catidx=Agnew}}），[[美国|美国]]政治人物，曾任[[美國副總統|美國副總統]]。

== 早年经历 ==
斯皮罗·阿格纽生于[[马里兰州|马里兰州]][[巴尔的摩|巴尔的摩]]市。是希臘裔美國人。1937年进入[[约翰霍普金斯大学|约翰霍普金斯大学]]学习[[化学|化学]]，1941年至1945年間於[[美國陸軍|美國陸軍]]服役，以{{tsl|en|上尉||上尉}}階級退役。1967－1969年任[[马里兰州|马里兰州]]州长，1968年獲[[理查·尼克森|理查·尼克森]]提名為共和黨副總統候選人，並最終成功當選[[美国副总统|美国副总统]]，1972年和尼克森一同成功連任。

== 辭職 ==
1973年10月10日，因為[[水門事件|水門事件]]及其在马里兰州州長任內涉及貪污和洗錢等醜聞，在[[彈劾|彈劾]]威脅下導致他被迫辭職，為美國歷史上两个於任內辭職的副總統之一。
