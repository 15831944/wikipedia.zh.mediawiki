== 詞語釋意 ==
1973年8月23日，两名有前科的[[罪犯|罪犯]]{{link-en|簡-艾瑞克·歐爾森|Jan-Erik Olsson}}与{{link-en|克拉克·歐洛夫森|Clark Olofsson}}，抢劫[[瑞典|瑞典]][[斯德哥爾摩|斯德哥爾摩]]内位于{{link-en|諾瑪姆斯托格|Norrmalmstorg}}广场最大的一家信贷[[银行|银行]]，並挟持了四位银行职员。在与警察僵持了130个小时后，歹徒最終投降。然而这起事件发生后几个月，4名曾經遭受挟持的银行职员，仍然对绑架他们的人显露出怜悯的情感，表明並不痛恨歹徒，表達他們對歹徒不但沒有傷害他們卻對他們多加照顧的感激，並且對警察採取敵對的態度, 在案件發生後徙被挾持者不願意在法庭指出歹徙， 在挾持案件當中被挾持者還與歹徒成為朋友 <ref>Hubbard, 1986</ref><ref>McMains & Mullins, 1996</ref>。

这两名抢匪[[劫持|劫持]][[人质|人质]]达5日之久，在这期间他们[[威胁|威胁]]受俘者的[[性命|性命]]，但有时也表现出[[仁慈|仁慈]]的一面。在出人意表的心理错综转变下，4名人质抗拒[[瑞典政府|政府]]最终营救他们的努力。斯德哥爾摩症候群因为在斯德哥爾摩人质挟持事件中被发现而得名。

研究者发现到这种症候群的例子见诸于各种不同的经验中，从[[集中营|集中营]]的[[囚犯|囚犯]]、[[战俘|战俘]]与[[乱伦|乱伦]]的受害者，都可能发生斯德哥尔摩综合症。男女皆可能有此症狀，惟女性的比例比較高。

== 綁架中出現斯德哥爾摩症候群的人質特徵 ==
美國[[聯邦調查局|聯邦調查局]]的人質數據庫顯示，大约8%的人質表現出斯德哥爾摩症候群的症狀。<ref>G. Dwayne Fuselier, “Placing the Stockholm Syndrome in Perspective,” FBI Law Enforcement Bulletin, July 1999, 22-25.</ref>據心理學者的研究，情感上會依賴他人且容易受感動的人，若遇到類似的狀況，很容易產生斯德哥爾摩症候群。

出現斯德哥爾摩症候群的人質，通常有下列幾項特徵：
# 綁匪為了某種原因而綁架人質，並得到人質認同。
# 人質必須有真正感到[[綁匪|綁匪]]（加害者）威脅到自己的存活。
# 在遭挾持過程中，人質必須體會出綁匪（加害者）可能略施小惠的舉動。
# 除了綁匪的單一看法之外，人質必須與所有其他觀點隔離（通常得不到外界的訊息）。
# 人質必須相信，要[[脫逃|脫逃]]是不可能的。

而通常斯德哥爾摩症候群會經歷以下四大歷程：
# 恐懼：因為突如其來的[[脅迫|脅迫]]與[[威嚇|威嚇]]導致現況改變。
# 害怕：籠罩在不安的環境中，身心皆受威脅。
# 同情：和挾持者長期相處體谅到對方不得已行為，且並未受到『直接』傷害。
# 幫助：給予挾持者無形幫助如配合，不逃脫，[[安撫|安撫]]等；或有形幫助如協助逃脫，向法官說情，一起[[逃亡|逃亡]]等。

== 心理學的解釋 ==
[[心理分析學|心理分析學]]的看法，新生嬰兒會與最靠近的有力成人形成一種情緒依附，以最大化周邊成人讓他至少能生存（或成為理想父母）的可能，此症候群可能是由此發展而來。斯德哥爾摩症候群是角色認同[[心理防衛機制|防衛機制]]的重要範例。

== 音樂中的斯德哥爾摩症候群 ==
* 英國搖滾樂團 [[繆斯樂隊|MUSE]] 於2003年發行的專輯《[[赦免_(专辑)|Absolution]]》中收錄其中一首單曲《[[斯德哥尔摩综合症_(歌曲)|斯德哥尔摩综合症]]》（''{{link-en|Stockholm Syndrome|Stockholm Syndrome (song)}}''）。

* 香港歌手 [[陳奕迅|陳奕迅]] 於2013年發行的專輯《[[The_Key|The Key]]》中其中一首單曲《斯德哥爾摩情人》。

* 英國流行樂團 [[One_Direction|One Direction]] 於2014年發行的專輯《{{link-en|Four|Four (One Direction album)}}》中其中一首單曲（"Stockholm Syndrome"）。

*手機遊戲《造物法則》的主題曲《潘朵拉》於歌詞中提及。

== 電子遊戲中的斯德哥爾摩症候群 ==
* [[劫薪日2|劫薪日2]] 中角色的其中一個技能為斯德哥爾摩症候群
* [[俠盜獵車手V|俠盜獵車手V]]中的角色崔佛·菲利普，綁架墨西哥黑幫老大的老婆時，她就產生了此症候群
* [[夜光_(遊戲)|夜光]]中的男主角江皓辰被女主角黃以妤綁架時，如果正確的選擇選項，劇情中會對她產生了此症候群。
