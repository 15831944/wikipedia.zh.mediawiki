'''斯蒂芬妮·列尼达斯'''（{{lang-en|'''Stephanie Leonidas'''}}，{{bd|1984年|2月14日|||catIdx=L}}）出生于英国[[英格兰|英格兰]][[伦敦|伦敦]]，为英国著名女演员。她知名於電影《[[奇幻面具|奇幻面具]]》中飾演Helena，以及於[[科幻|科幻]]電視劇《[[地球異世界|地球異世界]]》中飾演Irisa<ref>{{Cite web |url = http://www.syfy.com/defiance/cast/irisa/3 |title = Defiance Cast: Irisa |publisher = Syfy |language = en}}</ref>。

== 生平 ==
1984年，列尼达斯出生在[[英国|英国]][[英格兰|英格兰]][[伦敦|伦敦]]，他的父亲来自希腊的塞浦路斯，家中有4个孩子，他的弟弟[[迪米特里·莱奥尼达斯|迪米特里·莱奥尼达斯]]、妹妹{{link-en|戈奧爾吉納·萊奧尼達斯|Georgina Leonidas}}都是演员，其中姐姐海伦是一个老师，她的母亲妈妈莱斯利在2001年死于癌症，父亲则在伦敦经营一家商店。
