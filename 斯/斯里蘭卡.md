'''斯里兰卡民主社会主义共和国'''（{{lang-si|ශ්‍රී ලංකා ප්‍රජාතාන්ත්‍රික සමාජවාදී ජනරජය}}；{{lang-ta|இலங்கை சனநாயக சோஷலிசக் குடியரசு}}；{{lang-en|Democratic Socialist Republic of Sri Lanka}}），通称'''斯里兰卡'''（{{lang-si|ශ්‍රී ලංකාව}}；{{lang-ta|இலங்கை}}；{{lang-en|Sri Lanka}}），1972年之前称'''锡兰'''，是一个位於[[南亞|南亞]]－[[印度次大陸|印度次大陸]]東南方外海的[[島國|島國]]，属于[[亚洲|亚洲]]。古代中国曾經稱之為[[已程不|已程不]]、[[獅子國|獅子國]]、師子國、[[僧伽羅|僧伽羅]]、[[楞伽島|楞伽島]]。

斯里兰卡是[[單一制|單一制]][[共和國|共和國]]，其政治首都位於[[斯里賈亞瓦德納普拉科特|斯里賈亞瓦德納普拉科特]]（简称为'''科特'''），经济首都位于[[科伦坡|科伦坡]]。

== 名称 ==
*斯里蘭卡梵语古名Simhalauipa，驯狮人，《[[汉书|汉书]]·地理志》称「已程不国」。《[[梁书|梁书]]》称「狮子国」。《大唐西域记》作「僧伽罗」，即梵语古名Simhalauipa的音译。斯里蘭卡的主要民族至今在漢語裏仍稱“僧伽罗人”，其語言稱“僧伽罗语”。

*斯里蘭卡古阿拉伯语Sirandib，[[宋代|宋代]]音译为“细兰”，[[明代|明代]]称“锡兰”。

*1815年起作爲皇家殖民地由[[英國|英國]]統治，正式名稱為“Ceylon”，漢譯“錫蘭”。此英語名稱來自[[葡萄牙語|葡萄牙語]]的“Ceilão”。1948年錫蘭獨立成爲“锡兰自治領”。

*1972年锡兰廢除[[君主制|君主制]]，改稱“斯里蘭卡共和國”。“斯里蘭卡”名稱來自僧伽罗语{{lang|si|ශ්‍රී ලංකා }}（{{lang|si-Latn|śrī laṃkā}}），讀音/{{IPA|ʃɾiːˈlaŋkaː}}/。在斯里蘭卡的另一官方語言[[泰米尔语|泰米尔语]]中則稱爲{{lang|si|இலங்கை}}（ilaṅkai），讀音/{{IPA|iˈlaŋɡai|IPA}}/。“斯里蘭卡”在僧伽羅語裡義爲“美好神聖的土地”，其中的“蘭卡”實際上就是佛經中的“楞伽”，《[[楞伽經|楞伽經]]》即講述佛陀到達楞伽島教化說法，故以地名爲經名。

*由於歷史原因，斯里蘭卡的衆多機構以及產品都仍以“錫蘭”為名。2011年斯里蘭卡政府宣佈將在所有政府可以控制的機構名稱中將“錫蘭”改爲“斯里蘭卡”。<ref>[http://www.nownews.com/2011/01/02/91-2678719.htm 斯里蘭卡除舊名「錫蘭」　紅茶例外]，今日傳媒BBC，2011-1-2</ref>

== 历史 ==
{{seealso|斯里蘭卡歷史}}
公元前5世紀，[[僧伽羅人|僧伽羅人]]從印度遷移到斯里蘭卡。

公元前247年，[[印度|印度]][[孔雀王朝|孔雀王朝]]的[[阿育王|阿育王]]派其子来岛，从此僧伽罗人摈弃[[婆罗门教|婆罗门教]]而改信[[佛教|佛教]]。

公元前2世纪前后，南印度的[[泰米尔人|泰米尔人]]开始迁入斯里兰卡。

311年左右，佛牙從印度傳入斯里蘭卡。

从公元5世纪直至16世纪，[[僧伽罗族|僧伽罗王国]]和[[泰米尔人|泰米尔王国]]间征战不断，直至1521年[[葡萄牙|葡萄牙]]船队在[[科伦坡|科伦坡]]附近登陸。

1656年5月12日，[[荷蘭|荷蘭]]軍隊攻克[[科伦坡|科伦坡]]。

1796年2月15日，英軍占領[[科伦坡|科伦坡]]，荷蘭人統治時期結束。

1802年英法兩國簽訂了[[亞眠條約|亞眠條約]]，斯里蘭卡被正式宣布為[[英國|英國]]的殖民地。

1948年2月4日，斯里蘭卡正式宣布从英国[[獨立|獨立]]，成為[[英聯邦|英聯邦]]的自治領，定國名為[[錫蘭|錫蘭]]。

1972年5月22日，改國名為斯里蘭卡共和國。

1978年8月16日，新[[憲法|憲法]]頒布，改國名為'''斯里蘭卡民主社會主義共和國'''。

1830年代，英国将印度南部的[[泰米尔人|泰米尔人]]大批迁至斯里兰卡，并扶持其占据了各個方面的主导地位，由此與当地的主要民族僧伽罗人结怨。斯里蘭卡獨立後，泰米尔人於1972年成立了[[猛虎组织|猛虎组织]]（1976年改稱“泰米尔—伊拉姆猛虎解放组织”），走上“獨立建国”的道路。

1983年，[[猛虎组织|猛虎组织]]與斯里兰卡政府开战，战火一度蔓延到科伦坡。

1987年，印度出兵協助斯政府清剿猛虎组织，迫使其签订停火协议。1990年印军撤离后，猛虎组织重新反攻並迅速控制北部广大地区，在[[贾夫纳半岛|贾夫纳半岛]]建立起“泰米尔政权”，此後雙方戰爭不斷，造成6萬餘人喪生。

在[[挪威|挪威]]等國斡旋下，2000年起雙方開始和談，2002年2月，猛虎组织与斯里兰卡政府在斯德哥尔摩签署了一份永久性的停火协议，斯里兰卡政府也于当年9月4日解除了对猛虎组织的禁令，使其成为合法组织。同年9月16日至18日，斯里兰卡政府和猛虎组织在[[泰国|泰国]]东南部春武里府的梭桃邑海军基地举行了首次和谈。6年来，尽管双方先后进行了8轮直接谈判，但他们之间的武装冲突一直不断，停火协议已名存实亡。

从2006年7月开始，政府军开始向猛虎组织控制区发动大规模军事进攻，在两年多的时间里收复了约1.5万平方公里的猛虎组织控制区。2008年3月，斯里兰卡政府指控猛虎组织领导人[[普拉巴卡兰|普拉巴卡兰]]犯有谋杀罪。

2009年1月2日，斯里兰卡总统拉贾帕克萨宣布，政府军当天攻占了反政府武装猛虎组织的大本营基利诺奇。1月7日，斯里兰卡政府决定重新禁止猛虎组织活动，这表明在政府军不断取得军事胜利的情况下，斯里兰卡政府已不再把猛虎组织作为谈判对手。1月25日，政府军攻入猛虎组织控制的最后一个主要城镇——东北部的穆莱蒂武。2月5日，政府军攻克猛虎组织最后一个海上武装基地。4月初，政府军攻占了猛虎组织在北部地区的最后一个主要据点普杜库迪伊鲁普，部分猛虎组织成员转移到位于斯里兰卡北部穆莱蒂武地区的约20平方公里的“安全区”内。4月20日，斯里兰卡政府军攻入猛虎组织最后据守的“安全区”。26日，斯里兰卡政府拒绝了猛虎组织当天的单方面停火声明。5月15日斯里兰卡总统马欣达·拉贾帕克萨已宣布，将在48小时内解救出猛虎组织控制区内的所有被困平民，并将收复所有被该组织控制的领土。5月16日斯里兰卡政府军收复斯里兰卡反政府武装泰米尔伊拉姆猛虎解放组织（猛虎组织）控制的最后一段海岸线。同日马欣达·拉贾帕克萨总统在约旦访问时说，斯里兰卡政府军已经击败泰米尔伊拉姆猛虎解放组织（猛虎组织）。5月17日猛虎组织承认与政府军长达25年的战争失败，宣布放下武器，结束与政府军的战斗。斯里兰卡政府方面则表示无法相信猛虎组织“放下武器”的声明，政府军将继续进攻猛虎组织控制的最后一片丛林，以收复“每一寸国土”。5月18日，斯里兰卡政府军在穆莱蒂武区击毙反政府武装泰米尔伊拉姆猛虎解放组织最高领导人普拉巴卡兰后，宣布[[斯里兰卡内战|斯里兰卡内战]]结束。

斯里蘭卡島是梨形，又与眼泪的形状相似，因此有“印度洋上的一滴眼泪”的雅称。整个岛屿位于[[印度洋|印度洋]]之中，東北邊是[[孟加拉灣|孟加拉灣]]；中部、南部是高原，多山地，北部和沿海是平原；有[[亚当峰|亚当峰]]。

斯里兰卡北部属[[热带草原气候|热带草原气候]]，南部属[[热带雨林气候|热带雨林气候]]，全年炎热；西部年降雨量2000－3000毫米，东北部较干燥，年降雨量约1000毫米。

== 政治 ==
{{seealso|斯里蘭卡政治}}

斯里蘭卡的[[斯里蘭卡總統|總統]]是[[國家元首|國家元首]]。總統任期6年。

斯里蘭卡的[[一院制|一院制]]議會有225人，普選，任期6年。

獨立之後的斯里蘭卡仍然是[[大英國協|大英國協]]成员。

== 經濟 ==
{{seealso|斯里蘭卡經濟}}
[[File:Sri_Lanka_Military_0274.jpg|缩略图]]戰機]]
斯里蘭卡的經濟以寶石出口和[[農業|農業]]為主，主要農產出口如[[稻米|稻米]]、[[橡膠|橡膠]]、[[椰子|椰子]]、[[咖啡|咖啡]]等許多[[熱帶|熱帶]]地區代表性的[[經濟作物|經濟作物]]；而該國最重要的出口產品是[[錫蘭紅茶|錫蘭紅茶]]，斯里蘭卡是世界三大產茶國之一，也因此國內經濟深受產茶情況的影響。斯里蘭卡的觀光資源也相當豐富，但自[[2004年印度洋大地震|2004年印度洋大地震]]所引起的[[大海嘯|大海嘯]]以來，該國的海岸線被嚴重破壞，觀光業也因此受到了一定程度的影響。

出於戰略考量[[中華人民共和國|中華人民共和國]]和斯里蘭卡長期有多種合作關係密切，中國企業投資建造的科倫坡南港國際集裝箱(貨櫃)碼頭於2013年啟用，更早的2008年斯里蘭卡就在距離首都科倫坡50公里處為中國投資者建立了一個佔地150公頃的經濟特區，中國企業在此可享受許多優惠，也開始洽談[[自由貿易協定|自由貿易協定]]。南部城市漢班托塔的工業園區也幾乎交由中國大陸的國企和民企來開發基礎建設，並開發酒店度假村。

== 人口及宗教 ==
{{seealso|斯里蘭卡人口|斯里蘭卡宗教}}

根據1981年的統計，斯里蘭卡人口約14,850,001人，平均每平方公里有226.3人，主要的有兩個民族[[僧伽羅|僧伽羅]]族約佔73.8%、[[泰米爾|泰米爾]]族佔18%（其中[[印度|印度]]移民佔5.1%）、[[摩爾人|摩爾人]]佔8.3%。77%屬於[[上座部佛教|上座部佛教]]徒、15%屬於[[印度教|印度教]]徒、7.5%屬於[[穆斯林|穆斯林]]及[[基督徒|基督徒]]。
