'''钟道然'''（{{bd|1990年|7月23日|catIdx=Z}}）[[北京|北京]][[海淀区|海淀区]]人，2009年入读于[[中国人民大学|中国人民大学]]经济学院，2012年出版书籍《[[我不原谅|我不原谅]]》。

== 简介 ==
钟道然早年就读于[[人大附中|人大附中]]，2009年就读于[[中国人民大学|中国人民大学]]经济学院。2012年，钟道然出版书籍《[[我不原谅|我不原谅]]》，希望改变[[中国教育|中国教育]]的现状，尽自己的力量推动中国教育的变革。<ref name="multiplermw123 " >{{cite web |url=http://culture.people.com.cn/GB/22219/17037010.html |title=易中天挺90后批中国教育 钟道然出版《我不原谅》 |accessdate=2012年2月7日 |work=|publisher=人民网 |language=zh-hans}}</ref>

== 作品 ==
《我不原谅》，[[三联书店|三联书店]]，ISBN 9787108039675<br />

== 思想观点 ==
=== 教育的真正内涵 ===
钟道然：“教育，应是像[[卢梭|卢梭]]阐释的那样，‘其目的，是让人成为天性所造就的人’，是像[[马斯洛|马斯洛]]说的那样，‘帮助人达到他能够达到的最佳状态。’中国的学生在学校能达到的最佳状态，就是在教室里看着老师发呆，盯着黑板狂抄笔记，然后晚上跳入题海绞尽脑汁苦熬。中国的学生，是背书做题造就的，是考试卷子造就的，是《五年模拟三年高考》造就的，是一道道题的标准答案造就的，但绝对不是天性造就的。”<ref name="multiplefhw956 " />

== 作品风格 ==
钟道然的书籍剖析了[[中国教育|中国教育]]现状，指出其缺乏独立精神，又分析了中国教育的行业性，指出中国教育的实质是“人才”制造业。钟道然深受[[柏拉图|柏拉图]]、[[卢梭|卢梭]]、[[梭罗|梭罗]]、[[王尔德|王尔德]]、[[马斯洛|马斯洛]]等[[西方|西方]][[思想家|思想家]]的影响，提倡中国学生应有而未得到的权利，号召青年学生改变现状，在精神上觉醒，也委婉地呼吁[[政府|政府]]对教育界进行改革，唤醒麻木的[[政府|政府]]和[[人民|人民]]。<ref>{{cite web |url=http://edu.ifeng.com/news/detail_2012_02/09/12406141_0.shtml |title= 钟道然《我不原谅》结束语：唤醒麻木|accessdate= 2012年2月9日|work=|publisher= 凤凰网|language= zh-hans}}</ref><ref name="multiplermw123 " /><ref name="multiplefhw956 " />
