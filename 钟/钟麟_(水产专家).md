'''钟麟'''（{{bd|1915年|10月29日|1996年|3月17日|catIdx=Zhong钟麟}}），[[广东|广东]][[南海县|南海县]]人，鱼类生理学家、鱼类人工繁殖专家，被誉为“家鱼人工繁殖之父”<ref name="prfri7">{{Cite web |url=http://www.prfri.ac.cn/main/pageShow/7 |title=主持家鱼人工繁殖攻关 |access-date=2016-03-18 |archive-url=https://web.archive.org/web/20160326121845/http://www.prfri.ac.cn/main/pageShow/7 |archive-date=2016-03-26 |dead-url=yes }}</ref><ref>[http://cwin.com/scpnews/view2.asp?id=7284 家魚人工繁殖之父鐘麟百年誕辰]</ref>，中国现代水产科学技术的开拓者之一<ref name="prfri5"/>。

1938年毕业于广东省立高级水产职业学校，1939年在[[香港|香港]]中国侨港渔民学校任教，而后随鱼类学家[[林书颜|林书颜]]先生进香港水产研究所当研究生，1941年冬到农林部广西（桂平）鱼类养殖场工作，开始从事鱼类人工受精工作。 1953年3月，作为广东省人民政府农林厅水产局水产研究所创办人之一，钟麟选定的开所第一个研究项目，就是[[家鱼|家鱼]]人工繁殖。1958年6月4日率先获得池养[[鲢|鲢]]、[[鳙鱼|鳙鱼]]人工繁殖成功，为[[渔业|渔业]]史上的创举，使中国的[[水产养殖业|水产养殖业]]从此进入一个崭新的时代。<ref name="prfri5">{{Cite web |url=http://www.prfri.ac.cn/main/pageShow/5 |title=钟麟先生生平要录 |access-date=2016-03-18 |archive-url=https://web.archive.org/web/20160326153744/http://www.prfri.ac.cn/main/pageShow/5 |archive-date=2016-03-26 |dead-url=yes }}</ref><ref name="prfri4">{{Cite web |url=http://www.prfri.ac.cn/main/pageShow/4 |title=珠江水产所创办人之一 |access-date=2016-03-18 |archive-url=https://web.archive.org/web/20160326132706/http://www.prfri.ac.cn/main/pageShow/4 |archive-date=2016-03-26 |dead-url=yes }}</ref>
