'''钟南山'''（{{bd|1936年|10月20日|catIdx=Z鐘}}），侨眷，[[福建|福建]][[厦门|厦门]]人，生于[[南京|南京]]，[[中国工程院|中国工程院]]院士、教授、[[博士|博士]]生导师，是[[呼吸道疾病|呼吸道疾病]]（特别是[[慢性支气管炎|慢性支气管炎]]与[[哮喘病|哮喘病]]）方面的专家，现任[[全国人大代表|全国人大代表]]、[[中国工程院院士|中国工程院院士]]、[[广州市科协主席|广州市科协主席]]、[[广州呼吸疾病研究所所长|广州呼吸疾病研究所所长]]、[[呼吸疾病国家重点实验室主任|呼吸疾病国家重点实验室主任]]。2003年[[非典型肺炎|非典型肺炎]]（SARS）肆虐时期[[中国大陆|中国大陆]]著名的非典专家，主持了中国[[广东省|广东省]]的非典防治工作。曾任[[广州医学院|广州医学院]]院长、广东省科协副主席等职<ref>{{Cite web |url = http://www.gird.cn/sklrd/Article-408.aspx |title = 钟南山 - 实验室主任 |accessdate = 2014-03-07}}</ref>。

==履历==
*1959年参加[[第一届全国运动会|第一届全国运动会]]，在预赛400米跨栏比赛中打破了全国纪录。<ref>{{Cite web|url=http://news.sina.com.cn/c/2003-06-16/02521173575.shtml|title=钟南山的妻子是女篮５号|accessdate=2019-01-18|work=news.sina.com.cn}}</ref>
*1960年毕业于[[北京医学院|北京医学院]]。
*1979年到[[英国|英国]]进修，在著名的[[伦敦|伦敦]][[聖巴多羅買醫院|聖巴多羅買醫院]]及[[爱丁堡大学|爱丁堡大学]]医学院进行研究，1981年获爱丁堡大学[[医学博士#英国及某些英联邦地区|医学博士]]学位。
*1984年被授予首批国家级有突出贡献专家称号。
*1985年后被指定为中央领导保健医生，受聘为[[世界卫生组织|世界卫生组织]]医学顾问、国际胸科协会特别会员、亚太分会理事。
*1992年至2002年担任广州医学院院长，并且被选为全国[[政治协商会议|政治协商会议]]委员。
*2002年被选为[[全国人大代表|全国人大代表]]。
*1996年当选中国工程院院士，是广东医疗卫生界获得这项最高学术地位的第一人。
*2005年4月当选为[[中华医学会|中华医学会]]第23任会长。

==SARS时期==
{{seealso|SARS事件}}
从2003年初，[[广东|广东]]等地开始爆发不明[[肺炎|肺炎]]的时候起，钟南山就在密切注视着疫情的发展情况。

2003年2月，钟南山否定了卫生部所属国家疾病预防控制中心关于“典型[[衣原体|衣原体]]是[[非典型肺炎|非典型肺炎]]病因”的观点，为广东卫生行政部门及时制定救治方案提供了决策论据。

钟南山领导的课题组提出了一套行之有效的救治方法，大大提高了广东地区“非典”危重病人的成功抢救率，降低了死亡率，且明显缩短了病人的治疗时间。凭着多年经验，钟南山采用面罩通气方式维持病人生命，再使用适当剂量的皮质激素遏止病人肺部纤维化。事实证明他的判断和尝试正确，[[世界卫生组织|世界卫生组织]]派出的专家组认为：以钟南山为首的广东专家摸索出来的治疗经验，对世界抗击非典型肺炎有一定的指导意义。

==关于抢劫案和收容制度的争议==
2006年5月8日钟南山在[[广州|广州]]被人抢走[[手提电脑|手提电脑]]，报案后引起广东省高层重视，警方积极侦办，十天以后将失窃电脑找回。之后钟南山建议恢复过去的流浪乞讨人员[[收容遣送|收容制度]]。这件事引起广泛争议。

== 养生经 ==
钟南山针对“四十岁前拿命博钱，四十岁后花钱买命”这一不健康生活方式发出提醒：工作做不好可再努力，身体却就像是空心[[玻璃球|玻璃球]]一样，一旦碎了就不能再恢复了。

==评价==
*钟南山为抗击SARS所做的一切为他赢得了“抗击非典第一功臣”的美誉。2003年6月，广东省举行“抗击非典表彰先进大会”，钟南山被授予惟一的特等功。他还荣获了2003年中国五一劳动奖章、“[[中国医学基金会|中国医学基金会]]华源医德风范奖” 2005年钟南山当选中华医学会第23届会长。2008年当选第十一届全国人大代表。
*2003年4月中国社会调查所做的一项电话[[民意调查|民意调查]]显示，在[[北京|北京]]、[[上海|上海]]、[[广州|广州]]、等地的1200位受访民众当中，有89%认为钟南山是一位英雄。
*2004年被评为“[[感动中国年度人物评选|感动中国2003年度]]”十大人物之一。
