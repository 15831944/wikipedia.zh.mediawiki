'''闹不机迷'''，意思为搞不明白，搞不懂。此词是位于内蒙古、河北、山西的交界处地区的方言。具体位于河北的北部地区；山西北部，中部和部分南部地区；内蒙的西北部地区。此为口语，非正式语。<ref>{{Cite web|url=https://baike.baidu.com/item/%E9%97%B9%E4%B8%8D%E6%9C%BA%E5%AF%86|title=闹不机密|accessdate=2019年3月29号|author=我的名字叫岳洁洁|date=2018年8月8号|publisher=闯丶天下}}</ref><ref>{{Cite web|url=https://www.zhihu.com/question/24408308/answer/56068770|title=你的方言里有哪些有趣的说法或者读音？|accessdate=2019年3月29日|author=老葛的生活|date=2015年7月26日|publisher=}}</ref>

== 释义 ==
意思为搞不明白，搞不懂。对事物或事情的不了解，或者无法清楚的描述事物或事情。
例句：“他这是做啥呢？”“我也闹不机迷。”<ref>{{Cite web|url=https://zhidao.baidu.com/question/295405622.html|title=山西人说不机密是什么意思啊~！|accessdate=2019年3月29日|author=大脸猫789|date=2012年3月12日|publisher=}}</ref>
      “你看看这是个啥东西了？”“闹不机迷这是个啥。”

== 用法 ==
这个词属于名词、代词。对于不懂的事物，或者不了解的事物都可以用“闹不机迷”形容。<ref>{{Cite web|url=https://baike.baidu.com/item/%E5%90%8D%E8%AF%8D|title=名词|accessdate=2019年3月29日|author=muavk|date=2018年8月7日|publisher=tcbxh2008}}</ref>

== 易混淆词 ==
“不机迷”这个词虽然与“闹不机迷”只有一字之差，但是意思却完全不一样。“不机迷”是形容人的头脑不清楚，思路不明确。有时还指人处于昏迷状态。<ref>{{Cite web|url=https://zhidao.baidu.com/question/295405622.html|title=山西人说不机密是什么意思啊~|accessdate=2019年3月29日|author=tnt1109|date=2012年3月15日|publisher=}}</ref>

== 注意事项 ==
这个词是部分地区的口语，并非正式语。若是在正规场合或官方场合使用过于口语化的词，会产生不必要的麻烦。
