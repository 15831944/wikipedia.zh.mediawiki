'''漕河泾开发区站'''位于[[上海|上海]][[徐汇区|徐汇区]][[虹梅路|虹梅路]][[漕寶路|漕寶路]]，为[[上海轨道交通9号线|上海轨道交通9号线]]的地下[[岛式站台|岛式车站]]，於2007年12月29日啟用。

== 公交换乘 ==
131、700、731、757、804、809、927、徐梅线

== 车站出口 ==
本站分为东西两个互不相同的站厅，1、2口位于东厅，3、4、5号口位与西厅

* 1号口：宜山路北侧，虹许路东
* 2号口：宜山路南侧，虹许路东
* 3号口：宜山路南侧，虹许路西（通往公交枢纽站）[[File:Lift.svg|File:Lift.svg]]
* 4、5号口：宜山路北侧，虹梅路西

注：[[File:Lift.svg|File:Lift.svg]]设有无障碍电梯

== 车站设施 ==
站台提供公用厕所及配备饮料自动售贩机

== 车站历史 ==
2016年1月30日-2月21日，本站停运封站，对楼梯扶梯实施整体改造，在其东西站厅各增加两组上下行楼梯，进一步提升该站站台至站厅的通行能力。<ref>{{cite web|url=http://www.shmetro.com/node49/201601/con114567.htm | language= | title=“补短板”  9号线漕河泾开发区站安全改造 |publisher=上海地铁官方网站 |date=2016-01-25 |accessdate= }}</ref>
