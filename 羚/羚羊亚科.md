'''羚羊亞科'''（學名Antilopinae）是屬於[[牛科|牛科]]以下的一個[[亞科|亞科]]。

== 概述 ==
羚羊亚科动物善于奔跑，有6属23种。寿命较短，一般不会超过20岁。

== 特征 ==
属中小型[[牛科|牛科]]动物，腿很细，善于奔跑，最快的速度可达时速90公里。

== 分布 ==
有三种[[原羚|原羚]]和[[鹅喉羚|鹅喉羚]]分布在中国，此外[[印度黑羚|印度黑羚]]、[[印度瞪羚|印度瞪羚]]分布在南亚，其余的都生活在非洲。
