'''契島'''（{{jpn|j=契島|hg=ちぎりしま|rm=Chigiri-shima|f=y}}）是[[廣島縣|廣島縣]][[豐田郡|豐田郡]][[大崎上島町|大崎上島町]]，[[瀨戶內海|瀨戶內海]]上[[藝予群島|藝予群島]]的一個[[島|島]]<ref name="tokyo2015"/>。

== 地理 ==
契島與廣島縣[[竹原市|竹原市]]隔海距離約4公里，面積為9萬平方公尺<ref name="asahi201705"/>。

全島幾乎為[[東邦鋅業|東邦鋅業]]所有，成為該公司工廠（契島製鍊所）<ref name="asahi201705"/>。日本40%以上的[[鉛|鉛]]是契島[[冶鍊|冶鍊]]而成<ref>http://www.toho-zinc.co.jp/business/metallurgy/lead/</ref>。截至2017年，日本國內僅有契島的工廠以鉛礦進行冶鍊<ref name="asahi201705"/>。

除大崎上島町營渡輪停泊處外、與公司有關者以外禁止進入<ref name="asahi201705"/>。近來卻在工廠、廢墟狂熱下成為僅次於[[端島|端島]]的「第二座軍艦島」<ref name="tokyo2015"/>，狂熱者未經許可進入廠區成為一大問題。<ref name="tokyo2015">[http://www.tokyo-sports.co.jp/entame/entertainment/355428/ 命知らずのマニアがこぞって上陸する「もう一つの軍艦島」] - 東京スポーツ・2015年1月13日</ref>。

除了七座工廠外，島上還有物流中心和職員住宅，目前2017年有30名作業員居住<ref name="asahi201705"/>。鉛由[[澳大利亞|澳大利亞]]與[[南美洲|南美洲]]等境外輸入礦石冶煉生產，2017年年間已出貨9萬噸<ref name="asahi201705"/>。主要作為汽車電池的原料<ref name="asahi201705">[http://www.asahi.com/articles/ASK522V2YK52UQIP007.html 海に浮かぶ鉛色、異形の塊　製錬所国内シェア１位の契島] 朝日新聞デジタル　2017年5月19日13時48分</ref>。

== 歷史 ==
1899年有深川礦山製鍊所於此煉[[銅|銅]]後，經營者屢次轉手，1950年東邦鋅業買下島嶼後今日擁有至今<ref name="asahi201705"/>。

== 交通 ==
契島的交通依靠航運，與[[竹原港|竹原港]]、[[白水港|白水港]]（[[大崎上島|大崎上島]]）間有定期航運服務。[[契島運輸|契島運輸]]提供竹原港─契島航線（航程20分鐘，平日時1日17班）與白水港─契島航線（航程20分鐘、1日7班）。另有大崎上島町營白水港─[[生野島|生野島]]─契島航線（航程30分鐘、1日6班）。
