'''螳螂'''（[[学名|学名]]：{{lang|la|Mantodea}}），因前肢发达有力呈镰刀状，又称'''刀螂'''、'''蜛蚥'''（音同“居斧”），是[[昆虫纲|昆虫纲]][[螳螂目|螳螂目]]昆蟲的通称，目前世界上约有1800～2000种，分布於世界各地。

== 特徵 ==
[[File:The_mantis_which_eats_a_Bee20080829.jpg|thumb]]
螳螂是一种中大型肉食性昆蟲。其体态特征如下：
* 前肢呈鐮刀形，長而有力，上有非常锋利的尖刺，捕食时能牢牢抓住獵物。
* 口器強而有力，能輕易咬破并咀嚼獵物。
* 消化系統強力，能够完全吞食消化獵物(包括昆蟲堅固的外骨骼)（这样的肉食性昆蟲属于少数）。螳螂食性挑剔，只捕食活的獵物。飼養螳螂时，如果捉不到活的昆蟲，可以用針線綁上生豬肉、牛肉、魚肉、雞肉等肉类在螳螂面前晃動，螳螂會把肉塊當成獵物捕食。螳螂跟其他昆蟲一樣，視力很差，所以看不清楚眼前其實是針線串著的肉塊。
* 體型修長，通常扁平。頭部呈三角形，可自由轉動。
* [[複眼|複眼]]突出，[[單眼|單眼]]通常有3個。觸角長、呈絲狀，由多个節构成。
* 前胸長，可自由轉動，有些个体往兩側擴展。
* 翅膀有4翼，覆蓋在腹部，某些种类，特別是雌蟲，翅膀變小或退化。翅脈複雜，前翅通常稍短於後翅，較粗厚。
* [[蹠節|蹠節]]通常有5節，末端2爪，有些个体股節和脛節外緣向外延伸。
* 腹部長，呈圓筒形。

螳螂是一种[[不完全變態|不完全變態]]的昆虫，各階段生活史習性相似。雄蟲飛行能力較佳，[[觸角|觸角]]較發達，腹部通常有8節，體型較小；雌蟲腹部則有6節，體型通常較大。
在黑暗中，螳螂眼睛通常呈現黑色。

== 人类认知 ==
[[File:Mantis.jpg|thumb]]

相较于其它昆虫，园丁们更喜欢螳螂，因为它们能消灭大部分有害的昆虫。因此，螳螂可以作為一種[[生物防治|生物防治]]手段在不使用[[农药|农药]]的[[有机园艺|有机园艺]]中使用，螳螂的卵鞘在一般的园艺商店均有出售。

绝大部分种类的螳螂行动较为缓慢，不像其它敏捷的昆虫(如[[蟑螂|蟑螂]])能快速行動。但它们拥有保护色，且有着出色的拟物形态，它们能够模仿叶子晃动的姿态前进，慢慢地接近獵物。一旦獵物到達可攻击的範圍内，螳螂就会極快地進攻。一般情況下，螳螂都是以伏擊的方式捕捉獵物。
[[File:Mantis_eating_a_Common_Bushbrown_(Mycalesis_perseus)_butterfly.jpg|thumb]]
螳螂通常以其他種類昆蟲為食，但有报導敘述螳螂也吃小型脊椎动物，例如： 较小的[[蜥蜴|蜥蜴]]、[[老鼠|老鼠]]、[[蛇|蛇]]、[[蛙|蛙]]、小型[[淡水魚|淡水魚]]或[[蜂鸟|蜂鸟]]之類的小型鳥類<ref>[http://www.birdwatchersdigest.com/site/backyardbirds/hummingbirds/mantis-hummer.aspx 鸟类观察家]</ref>。自然学家[https://en.wikipedia.org/wiki/Gerald_Durrell Gerald Durrell]在他的自传《茄子與碎肉》中记述了一场螳螂和[[壁虎|壁虎]]的大战。個別品種的螳螂，如[[蘭花螳螂|蘭花螳螂]]，由於外形與體色跟[[蘭花|蘭花]]相似，所以牠們通常躲在蘭花中伏擊獵物。也因此，牠們的獵物也以[[蝴蝶|蝴蝶]]及[[蜜蜂|蜜蜂]]等吸花粉的昆蟲居多。

为了获得卵子发育的足够营养，在螳螂交配后，部分雄性螳螂會被雌性螳螂吃掉。交配时,有的雄螳螂会自行拧断上半身,或被雌螳螂吃掉上半身,这样雄螳螂对交配的欲望会更加强烈,而下半身也还能维持交配姿态很长一段时间。不过大部分雄螳螂交配后都是直接飞起来逃走的，被直接吃掉的较少。

== 瀕危品種 ==
只有一种[[西班牙|西班牙]]种螳螂 ''Apteromantis aptera''，处于低灭绝风险或濒临威胁状态。

《爾雅翼-釋蟲二》說：“螗娘，世謂之天馬。蓋驤首奮臂，頸長而身輕，其行如飛，有馬之象。”而1108年的《[[证类本草|經史證類大觀本草]]》中详细记述了螳螂卵囊的结构，生命周期和结构甚至触须的作用。

18世纪，螳螂的生物学和形态学知识更加丰富。 [[Roesel_von_Rosenhof|Roesel von Rosenhof]]在他的著作《Insekten-Belustigungen》（昆虫娱乐）中精确绘制和描述了这种昆虫。

== 文化 ==
中国人用“螳臂挡车，自不量力”来比喻人过于高估自己的能力，而做出愚蠢的举动。成语“螳臂挡车”出自《庄子·人间世》：“汝不知夫螳螂乎？怒其臂以当车辙，不知其不胜任也，是其才之美者也。”故事来源：齐庄公出猎，有一虫举足将搏其轮。问其御曰：“此何虫也？”对曰：“此所谓螳螂者也。其为虫也，知进而不知却，不量力而轻敌。”庄公曰：“此为人，而必为天下勇武矣！”回车而避之，而勇士归之。

日本方面：假面騎士劍(港譯幪面超人劍騎)第二男主角相川始(森本亮治 飾)幪面超人卡里斯的變身者。人物設定的藍本是螳螂
