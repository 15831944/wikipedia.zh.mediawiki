'''合青高铁'''，又名“合肥至新沂铁路”，是[[安徽省|安徽省]]提出的连接[[江苏省|江苏省]][[新沂市|新沂市]]和安徽省[[合肥市|合肥市]]之间的一条[[高速铁路|高速铁路]]，并通过在建的[[徐连高速铁路|徐连高速铁路]]、[[青连铁路|青连铁路]]连接至[[山东省|山东省]][[青岛市|青岛市]]，目前在前期工作中，计划2018年开工<ref>{{Cite web|url=http://ah.people.com.cn/n2/2018/0129/c358428-31192306.html|title=合肥至新沂（青岛）高铁计划今年开工 可研报告已编制完成|accessdate=2018-01-29}}</ref>。

== 位置 ==
合肥至新沂铁路位于安徽省东北部和江苏省北部。线路起于[[合肥市|合肥市]]，从[[新合肥西站|新合肥西站]]引出，向北经[[定远县|定远]]、[[明光市|明光]]、[[五河县|五河]]、[[泗县|泗县]]、[[泗洪县|泗洪]]、[[宿迁市|宿迁]]后进入[[新沂市|新沂]]。在[[新沂南站|新沂南站]]接入在建的[[徐连客运专线|徐连客专]]后，再往东到[[连云港|连云港]]，然后再接入在建的[[青连铁路|青连铁路]]，最终到达[[青岛市|青岛]]。据悉，合肥至新沂段工程全长约330公里，其中安徽省境内约221公里，江苏境内约109公里。线路等级为客运专线，设计速度目标值350公里/小时。

== 開工相關 ==
近年来合肥高铁建设加速，“米”字形高铁网初步成形，但东北方向的最后一“撇”还没落笔。据省发改委相关负责人透露，[[中国铁路总公司|铁总]]要求合青高铁项目尽快上马开建。按照流程，此次环评公示后将进行环境影响报告书的编制，完成后再进行送审。与此同时，还将进行[[可行性研究|可行性研究]]评审，编制并上报可行性研究报告，批复通过后方可正式立项、开工。据悉，合肥至新沂铁路将于2018年开工建设。

== 優點与争议 ==
合青高铁建成后，合肥将进一步缩短和江苏中北部以及山东的时空距离。目前，合肥前往青岛虽然已开行动车，但需要经过[[济南|济南]]，路程相对较远，最快也要5个多小时。而这条高铁修好后，合肥到青岛之间将几乎呈直线距离，列车运行时间也将缩短至3个小时左右。同时，合肥前往[[日照市|日照]]，也将结束不通火车的历史，行程耗时大约在2.5个小时左右。

然而，目前[[徐州|徐州]]至[[连云港|连云港]]时速350的高速铁路也在建设，合肥完全可以途径[[合蚌客专|合蚌客专]]、[[京沪高铁|京沪高铁]]到达徐州后，转[[徐连高铁|徐连高铁]]、[[青连铁路|青连铁路]]到达日照、青岛，且无需绕行济南，线路也几乎是直线距离，列车运行时间不足3小时。所以本高铁规划被质疑与商合杭高铁一样，为了强行制造合肥米字型高铁枢纽而无视重复建设以及路网科学性，使国家主要出资来满足安徽省内需求，以及通过定远设站来迎合高层<ref>{{Cite web|url=https://web.archive.org/web/20161129205853/http://bbs.cnhubei.com/thread-3877541-1-1.html|title=合新高铁争议}}</ref>。
