'''合肥市文物保护单位'''是中国[[安徽省|安徽省]][[合肥市|合肥市]]的市级[[文物保护单位|文物保护单位]]，目前已公布四批，共26处<ref>{{cite web |url=http://edu.anhuinews.com/system/2011/03/22/003862383.shtml|title=《安徽大学主教学楼成市级文物保护单位》|accessdate=2012-06-11|work=|publisher=中安教育网}}</ref>，已升级至[[安徽省文物保护单位|安徽省文物保护单位]]和[[全国重点文物保护单位|全国重点文物保护单位]]的不再列入（序号以公布时为准）。

*第一批：1985年7月12日公布，共13处，其中4处已升级为安徽省文物保护单位，分别为[[庐州府城隍庙|庐州府城隍庙]]（其中戏楼列入第五批省保，其他部分第六批省保并入）、[[李鸿章故居|李鸿章故居]]、[[李鸿章墓|李鸿章享堂]]（合并为李鸿章故居及享堂）、三国新城（重新命名为[[三国新城遗址|三国新城遗址]]）。
*第二批：1986年公布，共1处（大孔祠堂），已升级为安徽省文物保护单位。
*第三批：2005年1月31日公布，共2处<ref>{{cite web|url=http://ah.anhuinews.com/system/2005/01/04/001094393.shtml|title=《包公墓和吴氏炮楼选为合肥第3批市级文物保护单位》|accessdate=2012-06-11|work=|publisher=中安在线|archive-url=https://web.archive.org/web/20160304083538/http://ah.anhuinews.com/system/2005/01/04/001094393.shtml|archive-date=2016-03-04|dead-url=yes}}</ref>。
*第四批：2011年3月3日公布，共18处<ref>{{cite web |url=http://zwgk.hefei.gov.cn/zwgk/public/spage.xp?doAction=view&indexno=002991856/2011-0013256|title=《关于公布第四批市级文物保护单位的通知》|accessdate=2012-06-11|work=|publisher=合肥市政府信息公开专栏}}</ref>，其中3处已升级为安徽省文物保护单位，分别为安徽大学龙河校区主教学楼（已重命名为[[安徽大学龙河校区|安徽大学龙河校区教学主楼]]）、安徽省地质博物馆大楼（已重命名为[[安徽省地质博物馆|安徽省地质博物馆陈列大楼]]）和[[安徽省图书馆|安徽省图书馆主楼]]，另有2处部分升级为安徽省文物保护单位。
