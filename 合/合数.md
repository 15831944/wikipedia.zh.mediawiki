'''合數'''（也稱為'''合成數'''）是除了1和其本身外具有其他正[[因數|因數]]的正[[整數|整數]]。依照定義，每一個大於[[1|1]]的整數若不是[[質數|質數]]，就會是合數。而[[0|0]]與[[1|1]]則被認為不是質數，也不是合數。例如，整數14是一個合數，因為它可以被分解成2 × 7。

== 性質 ==
* 所有大於2的偶數都是合數。最小的合數為4。
*大於或等於10的整數中，個位數為0、2、4、5、6、8者均為合數。
* 每一合數都可以以唯一形式被寫成質數的乘積。([[算術基本定理|算術基本定理]])
* 對任一大於5的合數''n''，<math>(n-1)! \,\,\, \equiv \,\, 0 \pmod{n}</math>。([[威爾遜定理|威爾遜定理]])

== 合數的類型 ==

分類合數的一種方法為計算其質因數的個數。一個有兩個質因數的合數稱為[[半質數|半質數]]，有三個質因數的合數則稱為[[楔形數|楔形數]]。在一些的應用中，亦可以將合數分為有奇數的質因數的合數及有偶數的質因數的合數。對於後者，

:<math>\mu(n) = (-1)^{2x} = 1\,</math> 

（其中μ為[[默比烏斯函數|默比烏斯函數]]且''x''為質因數個數的一半），而前者則為 

:<math>\mu(n) = (-1)^{2x + 1} = -1.\,</math> 

注意，對於質數，此函數會傳回 -1，且<math>\mu(1) = 1</math>。而對於有一個或多個重複質因數的數字''n''，<math>\mu(n) = 0</math>。

另一種分類合數的方法為計算其因數的個數。所有的合數都至少有三個因數。一質數的平方數，其因數有<math>\{1, p, p^2\}</math>。一數若有著比它小的整數都還多的因數，則稱此數為[[高合成數|高合成數]]。另外，完全平方數的因數個數為奇數個，而其他的合數則皆為偶數個。

合数也可分基本合数(有2和3因子的)，阴性合数(6N-1形)和阳性合数(6N+1形)三种。
