'''合略假名'''（{{jpn|j=合略仮名|hg=ごうりゃくがな|rm=Gōryakugana|f=h}}），是一種表示多個音節的複合[[假名|假名]]。

== 概要 ==
雖然合略假名在[[江戶時期|江戶時期]]之前常常被使用，但在明治33年（西元1900年），[[明治政府|明治政府]]明令將其廢止，而統一使用一個假名記錄一個音節。

在1876年由維也納皇家印刷局出版的綜合樣本信“Alfabete des gesammten Erdkreises”中，增加了79個字母縮寫的平假名和14個片假名的字母。

大部分的合略假名在[[Unicode|Unicode]]中並沒有被收錄。其中，「ヿ」和「ゟ」在2000年的[[JIS_X_0213|JIS X 0213]]被定義，而「𪜈」也在2009年的Unicode 5.2中被登錄在[[中日韓統一表意文字擴展區C|中日韓統一表意文字擴展區C]]。

== 片假名 ==
大部分的的複合假名由兩個假名的偏旁所組成，但例外如ヿ（コト）則是例外。

== 平假名 ==
平假名的縮寫假名由雜草的上部和下部組合組成，用於許多信件中。<ref name="kotoba-no-izumi">[http://dl.ndl.go.jp/info:ndljp/pid/862876 ことばの泉 : 日本大辞典 (21版)] P.3 落合直文 1904年</ref><ref name="daijiten-p42-p45">[http://dl.ndl.go.jp/info:ndljp/pid/864050 日本大文典. 第1編] P.42-45 落合直文 1894年</ref>。「まゐらせさうらふ」（まいらせそうろう）を1文字で表すような特殊な物もある。

== 類似的文字 ==
{{未翻譯術語}}
* 「シテ」の合略仮名とされる文字「[[File:Ligature_shite.gif|File:Ligature_shite.gif]]」<ref>伊木壽一『古文書学』新装版　慶應義塾大学出版会 [[1981年|1981年]]に、「トシテ 「片仮名の合字」」とある。
大石学『古文書解読辞典』改訂新版　東京堂出版 2000年に、「（と）シテ　合字」とある。<br/>
これらの類書のほか、古文書・古典文学等の翻刻本の凡例はほぼ片仮名の「合字」であると説明しており、この項目「合略仮名」の定義に合致する。
</ref>を「為」の省字として説明する文献があり<ref name="taikei-moji-one">[https://books.google.co.jp/books?id=iHWOQdV-dPYC&pg=frontcover 國語學大系 第七巻 文字(一)] P.302-303 福井久蔵 1939年</ref><ref name="honmatsu" />、[[Unicode|Unicode]] には文字情報基盤<ref>{{Cite web |title="MJ文字情報検索システム (MJ056854)" |url=http://mojikiban.ipa.go.jp/search/index.html#detail/MJ056854 |accessdate=2016-06-21}}</ref>を典拠に[[漢字|漢字]]として提案されている<ref>{{Cite web |title=Draft additional repertoire for ISO/IEC 10646:2016 (5th edition) CD.2 |date=2015-12-14 |accessdate=2016-06-29 |url=http://www.unicode.org/L2/L2015/15339-n4705.pdf |format=pdf |quote=2CF00 [[File:Ligature_shite.gif|File:Ligature_shite.gif]] JMJ-056854}}</ref>。
* 「なり（[[File:Ligature_nari.gif|File:Ligature_nari.gif]]）」は合略仮名ではなく「也」の草体（草体漢字）である<ref>児玉幸多『くずし字解読辞典〔普及版〕』東京堂出版 1993年 ほか。
田島毓堂「法華経為字和訓考—資料編（三）—」名古屋大学文学部研究論集 1990年。</ref>が、合略仮名に分類する文献もある<ref name="jisyu-yoran"/>。
* 「[[枡記号|〼]]」は「ます」と呼ぶが、これは計量に使用する枡を記号化した物であり、合略仮名とは異なる。
* 漢字の部首と仮名を組み合わせた漢字と構造を同じくする字が存在するが、これらは合略仮名とは異なる。
