'''合江省'''，是[[中華民國|中華民國]]時期新置的12省級行政區之一，為[[東北九省|東北九省]]之一。

== 名稱由來 ==
由於[[烏蘇里江|烏蘇里江]]與[[松花江|松花江]]皆在此與[[黑龍江|黑龍江]]會合，故命名為合江，簡稱'''合'''。

== 地理 ==
全省位於今[[黑龍江省|黑龍江省]]東北部的鐵力以東、穆棱及依蘭以北地區，面積為13萬5406.27平方公里。地形以[[三江平原|三江平原]]為主體，地勢低平。東部主要是[[溼地|溼地]]，人口聚集在西部。[[撫遠縣|撫遠縣]]的最東端[[烏蘇里江|烏蘇里江]]與[[黑龍江|黑龍江]]會合處是中國國土的極東點。東、北鄰[[蘇俄|蘇俄]]，西界[[黑龍江省_(中華民國)|黑龍江省]]、[[嫩江省|嫩江省]]，南接[[松江省|松江省]]。

== 管轄範圍 ==
轄境包括九一八事變前吉林省的東北部，黑龍江省東南部的綏濱、蘿北、湯原等地，相當於今[[黑龍江省|黑龍江省]]東北部的鐵力以東、穆棱及依蘭以北地區，全省面積為129,144平方公里<ref>傅林祥，鄭寶恆著《中國行政區劃通史 中華民國卷》為135,406.27平方公里。</ref>。東、北鄰[[蘇聯|蘇聯]]，西界[[黑龍江省_(中華民國)|黑龍江省]]、[[嫩江省|嫩江省]]，南接[[松江省|松江省]]。

== 人口 ==
人口民國三十六年（1947年）上半年統計資料為193萬6000人，同年下半年統計資料為184萬1100人。

== 歷史沿革 ==
來源於老[[吉林省|吉林省]]。满洲国时期称作[[三江省|三江省]]。民國三十四年（1945年）抗戰勝利後，國民政府分東北為九省二市（[[大連|大連]]、[[哈爾濱|哈爾濱]]，後來才加上[[瀋陽|瀋陽]]），並派任首長與省府委員。不過由於[[蘇聯紅軍|蘇聯紅軍]]佔領東北，阻撓國府接收，反而是來自[[關內|關內]]的[[共產黨|共產黨]]接收了该省。1945年10月25日，[[中共中央东北局|中共中央东北局]]派[[孙靖宇|孙靖宇]]、[[戴鸿宾|戴鸿宾]]率队赶赴，11月5日队伍进驻佳木斯市，成立了三江人民自治军和三江地区行政专员公署，孙靖宇任司令员兼专员，戴鸿宾任副司令兼副专员。11月17日，东北局派[[李延禄|李延禄]]、[[李范五|李范五]]等二十余人到达佳木斯市。11月17日，中共合江省工作委员会在佳木斯市宣告成立。李范五为书记（兼三江人民自治军政委），李范五、李延禄、[[刘英勇|刘英勇]]、[[彭施鲁|彭施鲁]]、孙靖宇为委员。11月21日，撤销三江地区行政专员公署，成立合江省政府，李延禄任主席，李范五任副主席。11月23日，佳木斯市政府成立，[[董仙桥|董仙桥]]为市长。12月18日，三江人民自治军改为合江人民自治军，成立了合江军区，[[方强|方强]]任军区司令员。1946年2月21日至25日，合江省第一届人民代表大会在佳木斯市召开，选举产生了合江省政府行政委员会，李延禄、李范五、方强等十九人为委员。李延禄为行政委员会主任委员兼省政府主席，李范五为行政委员会副主任委员兼省政府副主席。

中華民國合江省政府一度至哈爾濱待命前往；鑒於松、哈岌岌不保，共軍多集結合江省會佳木斯一帶，且為其後方重地，焉容政府人員進入，形式接收，亦非所允。中途知難而返，得保安全。民國三十五年（1946年）4月，蘇聯紅軍撤出哈爾濱，共黨各省、市政府也隨同蘇軍撤出哈爾濱。合江省主席[[吳瀚濤|吳瀚濤]]率領部分省府人員赴瀋陽建立合江省政府辦事處。

1946年6月，东北局与东北民主联军预判哈尔滨很可能会被国民政府军乘胜攻占，因此东北局与大批机关疏散到佳木斯，如：东北行政委员会留守处、[[东北大学|东北大学]]、[[东北军政大学|东北军政大学]]、东北日报社、东北新华广播电台、[[东北书店|东北书店]]、鲁艺文工团、[[东北电影制片厂|东北电影制片厂]]等。中共中央政治局委员[[张闻天|张闻天]]（化名张平之）和夫人[[刘英_(中国共产党党员)|刘英]]来到佳木斯市。6月20日，中共合江省工委改为中共合江省委，张闻天任省委书记兼合江军区政委，李范五任省委副书记。刘英任省委组织部长。

民國三十六年（1947年）6月5日，國民政府頒布[[東北新省區方案|東北新省區方案]]，公布東北九省的行政區域，以原[[滿州國|滿州國]]滿東安全省及[[三江省|三江省]]大部分區域置「合江省」。但自始自終[[中華民國政府|中華民國政府]]從未直接統治該地區，各項統計資料多為「[[滿洲國|滿洲國]]」紀錄或推估，而[[立法委員|立法委員]]與[[國大代表|國大代表]]為在南方的合江省人民選出。7月19日，國民政府派吳瀚濤為合江選舉事務所主任委員；國民政府同時派定其為[[國民大會代表|國民大會代表]]，[[立法院|立法院]][[立法委員|立法委員]]<ref name="民國史大事記">{{cite book|editor=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編|title=《中華民國史大事記》|location=[[北京|北京]]|publisher=[[中華書局|中華書局]]|date=2011年7月}}</ref>{{rp|8385-8386}}。

1948年6月，省委副书记[[张启龙|张启龙]]接任省委书记兼合江军区政委。

民國三十七年(1948年)年底[[遼瀋會戰|遼瀋會戰]]後，大部分地區為[[中共|中共]]所控。11月2日解放軍攻陷[[瀋陽|瀋陽]]，中華民國合江省政府機構徹底瓦解。

[[东北行政委员会|东北行政委员会]]于1949年4月21日发布的命令，合江省与松江省于5月11日合并，成立新的[[松江省|松江省]]。

== 行政區劃 ==
=== 省會 ===
民國三十六年（1947年）6月，國民政府公布東北九省的行政區域，合江省省會為[[佳木斯市|佳木斯市]]（今黑龍江省佳木斯市）。

=== 縣、市、設治局 ===
民國三十六年（1947年）6月，國民政府公布東北九省的行政區域，全省轄有1市、17縣，由於省境較小，不設行政督察區。由於本省因抗戰結束後，全境己被中共東北民主聯軍控制，故國民政府公佈的行政區劃並未得到執行。新設的佳木斯市，在滿州國時期已經存在，沿革過程，詳見[[三江省|三江省]]。
