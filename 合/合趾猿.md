'''合趾猿'''（[[學名|學名]]：'''''Symphalangus syndactylus'''''）或'''大長臂猿'''是[[靈長目|靈長目]]動物，為[[長臂猿|長臂猿]]當中體型最大的一種。合趾猿是其他長臂猿的大小的兩倍，達到1米的高度，重達14公斤。

== 生境 ==
合趾猿分佈於[[東南亞|東南亞]]的[[馬來西亞|馬來西亞]]及[[蘇門答臘|蘇門答臘]]，棲身於[[熱帶雨林|熱帶雨林]]以及高山[[森林|森林]]。  
 
== 習性 ==
合趾猿為樹棲性，四枝極為靈活。學名'''Hylobates'''解作「樹上的穴居者」，合趾猿利用「臂躍行動」，運用強壯的手臂，於樹木之間穿梭往來，在地上步行時，則會高舉雙臂以平衡身體。

合趾猿主要是在日間活動，過著以家庭為單位的小群體生活。無論雄性還是雌性，於[[喉嚨|喉嚨]]之下都有聲囊，鳴叫時聲囊會同時脹大。牠們以叫聲來宣示自己的存在及領域。叫聲之響亮遠至一公里外也可聽到。晚間牠們會於高樹的葉叢中睡覺。

合趾猿屬雜食性，以進食[[果實|果實]]、[[樹葉|樹葉]]、[[花蕾|花蕾]]及小動物為主。 

== 保育狀況 ==
合趾猿為瀕危物種，列入瀕危野生動植物種國際貿易公約附錄(I)。[[香港動植物公園|香港動植物公園]]現在饲養了一對合趾猿。
