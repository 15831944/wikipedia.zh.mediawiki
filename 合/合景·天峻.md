'''合景•天峻''' （'''Top of World'''）是[[广州|广州]]的一个私人屋苑，位于[[黄埔区|黄埔区]]九龙大道[[中新广州知识城|中新广州知识城]]南起步区，为[[广州地铁知识城线|广州地铁知识城线]][[旺村站|旺村站]]上盖项目。<ref name="合景泰富">{{cite news|language=zh-hans|url=http://www.kwgproperty.com/projects/project_view.aspx?itemID=126|title=广州•合景天峻|publisher=合景泰富地产控股有限公司|date=|accessdate=2014年9月30日}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 屋苑由[[合景泰富地产控股有限公司|合景泰富地产控股有限公司]]建设，分7期发展，占地超过19万平方米，于2014年3月28日动工，预计于2015年10月25日落成，为知识城的第一个住宅发展项目<ref name = "南方都市报(深圳)">{{Cite web |url = http://gz.house.163.com/14/0311/10/9N23CGLP00873L40.html |title = 中新知识城首盘来袭 |author = |date = 2014年3月11日 |publisher = 《[[南方都市报|南方都市报]](深圳)》 |language = zh-hans |accessdate =  2014年9月30日 |quote =  }}</ref>。 项目共建有2栋23层高和7栋32层高的大厦。<ref name = "房天下1">{{Cite web |url = http://hejingtianjun.fang.com/house/2812070626/housedetail.htm |title = 合景天峻基本信息 |author = |date = 2014年3月11日 |publisher = 房天下 |language = zh-hans |accessdate =  2014年9月30日 |quote =  }}</ref>。
