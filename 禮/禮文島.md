'''禮文島'''（{{jpn|j=礼文島|hg=れぶんとう|rm=Rebuntō|f=y}}）是位於[[日本|日本]][[北海道|北海道]]北部、[[稚內市|稚内]]以西方60公里處[[日本海|日本海]]上的一個島嶼。屬[[禮文郡|禮文郡]][[禮文町|禮文町]]管轄。地名的語源系來自於[[愛努語|愛努語]]中的「rep-un-sir/{{lang|ain|レプン・シリ}}」（意為海上的島）。禮文島位於[[利尻島|利尻島]]西北，隔[[禮文水道|禮文水道]]和利尻島相望。由於禮文島自海拔0米處就生長有200種類以上的[[高山植物|高山植物]]，因此又有'''花之浮島'''的別稱<ref name="park-rishiri">{{cite web |url=http://www.env.go.jp/park/rishiri/intro/outline.html |script-title=ja:利尻礼文サロベツ国立公園|accessdate=2014-07-28|publisher={{lang|ja|環境省}} |language=ja}}</ref>。2013年4月1日時，禮文島有人口2,760人。

== 氣候 ==
禮文島屬於亞寒帶濕潤氣候<ref name="park-rishiri"/>。由於禮文島屬海洋性氣候，因此即使在冬季最低溫度也不會低於-16℃。但同時受到[[里曼寒流|里曼寒流]]的影響，白天氣溫也較少上升。在1月至2月時，最高氣溫也只有約-3℃。2010年2月3日時，禮文島的最高氣溫曾只有-12.5℃，創下最高氣溫的新低記錄。

== 地理 ==
禮文島東西長約7.9公里，南北長約25.8公里，周長約72公里。在須古頓岬和金田岬中間是直径5公里，呈半圓形的船泊灣。最高地点是[[禮文岳|禮文岳]]（標高490米）<ref>{{cite web |url=http://www.rinya.maff.go.jp/hokkaido/soya/news/100409_2.html |script-title=ja:冬の礼文岳に登りました。|accessdate=2014-07-28|publisher={{lang|ja|北海道森林管理局}} |language=ja}}</ref>。

禮文島多海蝕崖，雖海拔不高但氣候寒冷，屬冰緣形地形，有[[高山植物|高山性植物]]群落。禮文島是座丘陵性地形的離島。島嶼北部和南部有標高100m左右，形成於[[第四紀|第四紀]][[更新世|更新世]]時期的海岸段丘。[[集落|集落]]多集中在東海岸和北部及南部，西海岸則有元地、宇遠内、西上泊、鐵府等集落。禮文島的東海岸和北部有香深、船泊等島上的中心集落，交通相對較為便捷。禮文島的東部有一條縱貫全島的公路，途徑島嶼的北部、東部和南部。但縱貫西海岸的道路是[[林道|林道]]，車輛無法通行。

=== 地形 ===
在禮文島上發現了自[[白堊紀|白堊紀]]至[[新第三紀|新第三紀]][[中新世|中新世]]在海底堆積的砂岩和泥岩，以及火山岩的地層。另外在地藏地蔵岩的附近發現了1億1150萬年前的[[菊石目|菊石目]]的[[化石|化石]]。島上[[新第三紀|新第三紀]][[中新世|中新世]]的地層則主要來自於海底火山的残骸，主要出現在須古頓岬的附近。白堊紀的地層主要出現在以禮文岳為中心的島嶼中部。另外桃岩是在約1300萬年前岩漿自海底噴出時形成，由這些證據來看，禮文島應該是在海底形成的<ref>{{cite web |url=http://www.geosites-hokkaido.org/class/ohg/ohg_rebun.html|script-title=ja:海底火山噴出物の島「礼文島」|accessdate=2014-07-28|publisher=Online Hokkaido Geopark |language=ja}}</ref>。

=== 自然 ===
西海岸地形險峻，加上全年都有自西吹來的強季節風，氣候也十分嚴峻，因此高大的樹木難以生長，植被較為荒涼，最多的植物是華箬竹。

禮文島各處都可看到[[安山岩|安山岩]]、[[玄武岩|玄武岩]]的柱状節理以岩脈的形式出現。另外禮文島還產出化石和燧石。
* 過去最高気温：31.0℃（1989年7月26日）船泊地區
* 過去最低気温：-19.4℃（1978年2月16日） 船泊地區
* 全年降水量：最多1247毫米（1999年）船泊地區、最少709毫米（1979年）船泊地區
* 日最大降水量:最多111毫米（2007年8月1日）幌泊地區
* 最大瞬間風速:29.9m/s（2010年9月29日）香深地區

=== 動植物 ===
==== 動物相 ====
禮文島上生存有[[日本歌鴝|日本歌鴝]]、[[紅喉歌鴝|紅喉歌鴝]]、[[日本樹鶯|日本樹鶯]]、[[蒼眉蝗鶯|蒼眉蝗鶯]]、[[北蝗鶯|北蝗鶯]]等鳥類，[[久種湖|久種湖]]附近有時可以看見[[牛背鷺|牛背鷺]]、[[大田鷸|大田鷸]]、[[小辮鴴|小辮鴴]]、[[戴勝|戴勝]]等鳥類。海岸地區生存有[[暗綠背鸕鶿|暗綠背鸕鶿]]、[[黑尾鷗|黑尾鷗]]、[[灰背鷗|灰背鷗]]。

在哺乳動物方面，島上沒有[[棕熊|棕熊]]、[[蝦夷鹿|蝦夷鹿]]、[[狐狸|狐狸]]<ref ==== 植物相 ====
島上的針葉樹大多是[[椴松|椴松]]，落葉闊葉樹則多是[[岳樺|岳樺]]、[[龍江柳|龍江柳]]，在低標高處也有[[偃松|偃松]]分佈。島上草原的主要植物則是[[千島笹|千島笹]]。

== 歷史 ==
禮文島形成在[[新生代|新生代]][[第三紀|第三紀]]或在其之前。在島上的香深井和船泊發現了先史時代的鄂霍次克文化期的遺跡，並且島上也發現了愛努時期的堡壘遺跡。1456年時，島上的香深井愛努部落和磯谷愛努部落在現在的桃岩附近交戰。

1685年時，禮文島成為[[松前藩|松前藩]]直轄的[[宗谷郡|宗谷場所]]的附屬場所。之後宗谷場所分裂，禮文、利尻、宗谷成為獨立場所。1846年時，[[陸奥國|陸奥國]]（青森）人首次在這裡開闢漁場。1885年時，小樽-利尻-禮文之間的航路開始通航。進入大正時代後，禮文島上開始開採煤礦。在1924年至1926年七級，為了驅除野鼠和養殖毛皮，禮文島自[[千島|千島]]群島[[新知島|新知島]]引入狐狸養殖，但狐狸卻成為棘球蚴病感染源。至1963年時，有約200人因[[棘球蚴病|棘球蚴病]]死亡。現在由於密獵者打獵的結果，狐狸在禮文島上已經滅絕，棘球蚴病在島上也已經被消滅，禮文島現在是棘球蚴病非汚染地區<ref>{{cite web |url=http://idsc.nih.go.jp/iasr/20/227/dj2275.html |script-title=ja:礼文島エキノコックス症の自然史|accessdate=2014-07-28|publisher={{lang|ja|[[国立感染症研究所|国立感染症研究所]]感染症情報センター}} |language=ja}}</ref>。

== 旅遊 ==
禮文島內整備有多條觀光路線，包括自最北部[[須古頓岬|須古頓岬]]出發，縱貫西岸的「愛和羅曼的8小時路線」和「環繞諸海岬路線<ref>{{cite web |url=http://idsc.nih.go.jp/iasr/20/227/dj2275.html |script-title=ja:散策路マップ|accessdate=2014-07-28|publisher={{lang|ja|礼文町}} |language=ja}}</ref>」、「花之路」等能觀賞高山植物的路線。由於氣候冷涼，島上自最高峰的[[禮文岳|禮文岳]]（490m）至島嶼西半部的地區分佈有[[禮文敦盛草|禮文敦盛草]]等約300種類的[[高山植物|高山植物]]，其中有多種植物只生存在禮文島<ref>{{cite web |url=http://www.env.go.jp/park/rishiri/guide/view.html |script-title=ja:みどころ紹介|accessdate=2014-07-28|publisher={{lang|ja|利尻礼文サロベツ国立公園}} |language=ja}}</ref>。全島均被劃入[[利尻禮文佐呂別國立公園|利尻禮文佐呂別國立公園]]。島內各漁港內的防波堤也是垂釣的聖地。

在2007年，禮文島上成功開採了溫泉（[[禮文島溫泉|禮文島溫泉]]）。近年在禮文島北部的船舶灣地區，全年都可觀察到野生[[海豹|海豹]]。此外有時也可以觀察到北海獅。

禮文島上有富士見丘滑雪場、久種湖畔滑雪場兩處滑雪場<ref>{{cite web |url=http://www.reikyoi.jp/index.php/%E7%A4%BE%E4%BC%9A%E6%95%99%E8%82%B2%E6%96%BD%E8%A8%AD/%E5%AF%8C%E5%A3%AB%E8%A6%8B%E3%83%B6%E4%B8%98%E3%82%B9%E3%82%AD%E3%83%BC%E5%A0%B4/ |script-title=ja:富士見ヶ丘スキー場・久種湖畔スキー場|accessdate=2014-07-28|publisher={{lang|ja|礼文町教育委員会}} |language=ja}}</ref>
。久種湖畔滑雪場是日本最北端的滑雪場。

日本最北端的湖泊[[久種湖|久種湖]]也位於禮文島。此外島上還有禮文瀑布，但是由於瀑布距離觀光路線較遠，因此較少有團體遊客到訪。

== 交通 ==
=== 航路 ===
禮文島上主要的交通據點是[[香深港|香深港]]。[[HeartLand渡船|HeartLand渡船]]運行有香深港和[[稚內港|稚內港]]、[[鴛泊港|鴛泊港]]、[[沓形港|沓形港]]之間的往返航線。前往[[稚內港|稚內港]]的渡輪需時2小時5分至2時間50分，每天有2至5次。前往[[鴛泊港|鴛泊港]]的渡輪需時40分，每日有1至2次。前往[[沓形港|沓形港]]的渡輪需時40分，每日運行2次2（僅5 - 9月運行）<ref>{{cite web|url=http://www.heartlandferry.jp/rishiri/index.html|script-title=ja:稚内～利尻島・礼文島|accessdate=2014-07-28|publisher=heartlandferry|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140728191450/http://www.heartlandferry.jp/rishiri/index.html|archivedate=2014-07-28}}</ref>。

=== 空路 ===
[[礼文機場|礼文機場]]是禮文島上唯一的機場，但自2003年（平成15年）4月之後已無定期航班飛行。自2009年4月開始，禮文機場暫停使用<ref>{{cite web |url=http://www.mlit.go.jp/report/press/cab05_hh_000012.html |script-title=ja:礼文空港の供用休止の許可について|accessdate=2014-07-28|publisher={{lang|ja|国土交通省}} |language=ja}}</ref>）。
