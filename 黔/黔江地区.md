'''黔江地区'''是[[四川省|四川省]]已撤销的[[地区_(中国行政区划)|地区]]（[[地级行政区|地级行政区]]），位于四川省东部，今属[[重庆市|重庆市]]，行署距离[[重庆主城区|重庆主城区]]425公里，是川东唯一的一个[[少数民族|少数民族]][[自治县|自治县]]为主的地区。

== 建制沿革 ==

*[[建安_(东汉)|建安]]之年（201），[[益州|益州]]牧[[刘璋|刘璋]]置'''丹兴县'''，为黔江建县之初。
*公元280年，晋实行州郡县三级制，丹兴废，其地并入涪陵、汉夏二县。
*[[北周|北周]]保定四年（564），在[[彭水|彭水]]置奉州（建德三年改名黔州），县为其属地。
*隋开皇五年（585）置'''石城县'''。
*唐武德元年（618）石城县改属'''黔州'''，天宝元年（742）改名黔江，属黔安郡（黔州）。
*清初，黔江县属重庆府。雍正十三年（1735）酉阳土司"[[改土归流|改土归流]]"后，置酉阳直隶州，废黔彭直隶万，复置黔江县，属酉阳直隶州。
*[[民国|民国]]元年（1912）废府，州、厅，黔江直属省。
*1949年11月25日，成立黔江县人民政府，隶属川东行政专区。1950年1月23日，置[[酉阳专区|酉阳专区]]，领酉阳、秀山、黔 江3县。1952年9月，[[酉阳专区|酉阳专区]]并入[[涪陵专区|涪陵专区]]，黔江随之属涪陵专区。1983年11月，经[[中华人民共和国国务院|中国国务院]]批准，撤销[[黔江县|黔江县]]，建立[[黔江土家族苗族自治县|黔江土家族苗族自治县]]。
*1988年5月18日，经中国国务院批准（从原涪陵地区10县中将酉秀黔彭石五个自治县划出单设）成立四川省黔江地区，辖[[石柱土家族自治县|石柱土家族自治县]]、[[彭水苗族土家族自治县|彭水苗族土家族自治县]]、[[黔江土家族苗族自治县|黔江土家族苗族自治县]]、[[酉阳土家族苗族自治县|酉阳土家族苗族自治县]]、[[秀山土家族苗族自治县|秀山土家族苗族自治县]]。黔江地区是[[中国西南地区|西南]]除[[西藏|西藏]]地区外，经济最贫困的地区。
*1997年重庆市升格为直辖市后，按照西南脱贫致富“城市带农村”的原则，纳入重庆市管理，同年3月17日经国务院批准，以扶贫为目的成立"重庆市黔江开发区"。2000年6月经国务院批准撤销原重庆市黔江开发区、黔江土家苗族自治县，设立"重庆市黔江区"，隶属重庆直辖市管理，并实际上为重庆市代管[[酉阳土家族苗族自治县|酉阳]]、[[秀山土家族苗族自治县|秀山]]、[[彭水苗族土家族自治县|彭水]]三县机关。

== 地理 ==
*黔江地区位于四川省的东南边缘，地处[[武陵山|武陵山]]腹地，东临[[湖北省|湖北省]]的[[咸丰县|咸丰县]]，北接湖北[[利川市|利川市]]，是川、鄂、湘、黔四省市的结合部。
*最高点[[灰千粱子|灰千粱子]]，[[海拔|海拔]]高度为1938.5米,最低点为黑溪河谷马斯口,海拔320米。

== 人口 ==
总人口100万人。民族有[[汉族|汉族]]、[[土家族|土家族]]、[[苗族|苗族]]、[[回族|回族]]等14个，其中，汉族占总人口的50%。[[人口密度|人口密度]]为150人/平方公里。

== 文化教育 ==
黔江城区共有中学5所：黔江中学、新华中学、民族中学、人民中学、平坝中学,小学3所:人民小学、黔江实验小学、民族小学，有一广播电视大学，职教中心。
黔江拥有一家广播电视集团和一家报业公司。即1991年成立，1992年7月18日正式开播的黔江电视台和1988年成立的武陵都市报。
