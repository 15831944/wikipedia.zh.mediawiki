'''黔江区'''位于[[重庆市|重庆市]]东南部，公路距离[[重庆主城区|重庆主城区]]425公里，是重庆市唯一的一个[[少数民族|少数民族]][[自治县|自治县]]改设的区。

== 建制沿革 ==
*[[建安|建安]]之年（201），[[益州|益州]]牧刘漳置'''丹兴县'''，为黔江建县之初。
*公元280年，晋实行州郡县三级制，丹兴废，其地并入涪陵、汉夏二县。
*[[北周|北周]]保定四年（564），在[[彭水|彭水]]置奉州（建德三年改名黔州），县为其属地。
*隋开皇五年（585）置'''石城县'''。
*唐武德元年（618）石城县改属'''黔州'''，天宝元年（742）改名黔江，属黔安郡（黔州）。
*清初，黔江县属重庆府。十三年（1735）酉阳土司"[[改土归流|改土归流]]"后，置酉阳直隶州，废黔彭直隶万，复置黔江县，属酉阳直隶州。
*[[民国|民国]]元年（1912）废府，州、厅，黔江直属省。
*1949年11月25日，成立黔江县人民政府，隶属[[川东行政专区|川东行政专区]]。1950年1月23日，置[[酉阳专区|酉阳专区]]，领[[酉阳|酉阳]]、[[秀山|秀山]]、[[黔江|黔江]]3县。1952年9月，酉阳专区并入[[涪陵专区|涪陵专区]]，黔江随之属涪陵专区。1983年11月，经[[国务院|国务院]]批准，撤销黔江县，建立[[黔江土家族苗族自治县|黔江土家族苗族自治县]]。
*1988年5月18日经国务院批准（从原涪陵地区10县中将酉秀黔彭石五个自治县划出单设）成立四川省黔江地区，辖[[石柱|石柱]]土家族自治县、[[彭水|彭水]]苗族土家族自治县、黔江土家苗族自治县、[[酉阳|酉阳]]土家苗族自治县、[[秀山|秀山]]土家苗族自治县。
*1997年重庆市升格为直辖市后，按照西南脱贫致富“城市带农村”的原则，纳入重庆市管理，同年3月17日经国务院批准，以扶贫为目的成立"重庆市黔江开发区"。2000年6月经国务院批准撤销原重庆市黔江开发区、黔江土家苗族自治县，设立"重庆市黔江区"，隶属重庆直辖市管理，并实际上为重庆市代管[[酉阳|酉阳]]、[[秀山|秀山]]、[[彭水|彭水]]三县机关。
*黔江区为全国农业标准化示范县 ，获得过“全市退耕还林工作先进区”、“最具风情民俗文化旅游目的地”、“全国农村饮水安全工程示范县”、“全国平安铁路示范区”等荣誉称号。黔江区是武陵山区的中心城市之一，为重庆市六大区域性中心城市之一。
*黔江区东临湖北省恩施土家族苗族自治州咸丰县，西接彭水苗族自治县，南连酉阳土家族苗族自治县县，北接湖北省恩施土家族苗族自治州利川市。至2014年，黔江区幅员面积2402平方公里，辖6个街道、12个镇、12个乡。2010年第六次人口普查，黔江区户籍人口53.6万人（常住人口44.5万人），其中以土家族、苗族为主的少数民族人口占总人口的73.3%。
*GDP 202.55亿（2015年）

== 地理 ==
*黔江区位于重庆市的东南边缘，地处[[武陵山|武陵山]]腹地，东临[[湖北|湖北]]省的[[咸丰|咸丰]]县，西界[[彭水|彭水]]县，南连[[酉阳|酉阳]]县，北接湖北[[利川|利川]]市，是渝、鄂、湘、黔四省市的结合部。
*辖区范围在东经108度28分至108度56分，北纬29度4分至29度52分之间。东西宽45公里，南北长90公里。全区幅员面积为2398.7平方公里。
*最高点[[灰千粱子|灰千粱子]]，[[海拔|海拔]]高度为1938.5米,最低点为黑溪河谷马斯口,海拔320米。

== 人口 ==
总人口48.15万人，其中非农业人口5.63万人。民族有[[汉族|汉族]]、[[土家族|土家族]]、[[苗族|苗族]]、[[回族|回族]]等14个，其中，少数民族人口占总人口的72.4%。[[人口密度|人口密度]]为199人/平方公里。

== 文化教育 ==
城区共有中学6所：黔江中学、新华中学、民族中学、实验中学、人民中学、平坝中学,小学3所:人民小学、黔江实验小学、民族小学，有一广播电视大学，重庆市黔江区民族职业教育中心，黔江旅游职业学院，黔江经贸职业学院
黔江拥有一家广播电视集团和一家报业公司。即1991年成立，1992年7月18日正式开播的黔江电视台和1988年成立的武陵都市报。
