'''甘肃省'''，简称'''甘'''或'''陇'''，是[[中华人民共和国|中国]]的一个[[中国一级行政区|省份]]，位於[[中国西北地区|中国西北地区]]、[[黄河|黄河]]上游，[[省会|省会]]为[[兰州市|兰州市]]。“'''甘肃'''”（{{Audio|Zh-Gansu.ogg|普通话发音}}）因[[甘州_(古代)|甘州]]（今[[张掖|张掖]]）与[[肃州_(隋朝)|肃州]]（今[[酒泉|酒泉]]）而得名。

== 历史 ==
{{Main|甘肃历史}}

甘肃为[[华夏文明|华夏文明]]的发祥地，天水的[[伏羲|伏羲]]文化历史久远，被称为伏羲故里。在甘肃秦安县发现的“大地湾文化遗址”被认为可能和上古时代的伏羲有关。陇东地区的庆阳一带是周人发祥地。礼县大堡子山有早期秦公大墓，秦人发祥地“西犬丘”就在礼县一带。甘肃在[[春秋时代|春秋时代]]东南部属于[[秦国|秦国]]，西部属于西戎，公元前688年在现在[[天水|天水]]一带秦国设置了邽县，前280年设置陇西、北地二郡。

[[汉代|汉代]]由于[[张骞|张骞]]通[[西域|西域]]，[[河西走廊|河西走廊]]成为“[[丝绸之路|丝绸之路]]”的必经要道，以及许多新的城镇兴起，[[汉武帝|汉武帝]]在现在甘肃一带将[[黄河|黄河]]以西[[河西四郡|分置四郡]]，黄河以东置五郡，统归[[涼州刺史部|凉州]]管辖。

[[五胡十六國|十六国时代]]有5个国家在甘肃境内建都。[[南北朝|南北朝]]时代属[[北魏|北魏]]，当时[[佛教|佛教]]从西域传入，在甘肃境内大兴石窟，目前最重要的有[[敦煌石窟|敦煌石窟]]（莫高窟千佛洞）、[[麦积山石窟|麦积山石窟]]、[[炳灵寺石窟|炳灵寺石窟]]等。

[[唐代|唐代]]重视通向西域的交通要道甘肃一带的作用，将其分置关内道、陇右道和山南道，共辖22州，其中在河西走廊最大的是甘州（现在[[张掖|张掖]]一带）和肃州（现在[[酒泉|酒泉]]一带）。[[五代十国|五代十国]]时期，甘肃倍受战乱，少数民族侵入，[[宋代|宋代]]时归属[[西夏|西夏]]。西夏將唐代的[[甘州_(古代)|甘]]、[[肃州_(隋朝)|肅]]兩州的首字合併，设置“[[甘肃军|甘肃军]]”，从此正式称为甘肃。[[成吉思汗|成吉思汗]]大军进攻西夏时，死于甘肃，遗体虽然运回[[蒙古|蒙古]]，现在甘肃兴隆山上有成吉思汗衣冠冢。

1261年，[[忽必烈|忽必烈]]在甘肅立[[西夏中興行省|西夏中興行省]]，治[[銀川|中興府]]。直至1286年，該省遷治[[甘州路|甘州路]]，並正式改名為甘肃行中书省。延至明代，甘肅行省被劃入[[陕西布政使司|陕西布政使司]]。至清康熙二年，分陕西省西部為[[鞏昌省|鞏昌省]]，五年改稱甘肅省，管辖整个西域。1884年[[慈禧太后|慈禧太后]]主政时将[[新疆省_(清朝)|新疆]]单独划出。1929年，[[中华民国政府|中华民国政府]]将[[青海省|青海]]、[[寧夏省_(中華民國)|宁夏]]单独置省。甘肃基本维持了目前的省境。1963年西北各省最后划定省界。

== 地理 ==
* 接邻省区：[[陕西|陕西]]、[[青海|青海]]、[[四川|四川]]、[[宁夏|宁夏]]、[[内蒙古|内蒙古]]、[[新疆|新疆]]
甘肃处于中国地理的中心，轮廓是细长形的，位于青藏、蒙新、黄土三大高原交汇处，其西北一端与[[蒙古国|蒙古国]]接壤。

东部是起伏较大的[[黄土高原|黄土高原]]，海拔1500－2000米，黄河及其支流[[渭河|渭河]]、[[洮河|洮河]]等流贯，水力资源丰富，[[秦岭|秦岭]]以南为[[白龙江|白龙江]]流域，气候温湿，植物茂密，年降水量500毫米左右；西部[[乌鞘岭|乌鞘岭]]以西，分別為南、北兩山對峙所形成的[[河西走廊|河西走廊]]，南山以[[祁连山|祁连山]]為主，山勢高大，多在三、四千公尺以上，北山則較為低矮，主要有[[龙首山|龙首山]]、[[合黎山|合黎山]]等，區域內的河川是內陸水系，多半是由南往北流，有[[疏勒河|疏勒河]]及[[弱水|弱水]]等，其中以[[弱水|弱水]]流域面積最大，气候干燥，年降水量150毫米左右，以半荒漠和荒漠为主，由于祁连山融雪灌溉，在山麓形成[[沖積扇|沖積扇]]，走廊中有河流的地方形成相连的[[绿洲|绿洲]]，是甘肃的粮仓，外面则是山地和[[沙漠|沙漠]]。

[[六盘山|六盘山]]以西植被稀少，以东降水丰富，植被繁茂。西南角和[[青藏高原|青藏高原]]接壤，是高山草原，水草丰美的天然牧场。

=== 民族和人口 ===
2017年，甘肃省有人口2626万人，出生率1.254%，死亡率0.652%，自然增长率0.602%。城镇化率46.39%[http://www.stats.gov.cn/tjsj/ndsj/2018/indexch.htm]。

甘肃[[历史|历史]]上就是以[[漢族|漢族]]為主，多民族混合的地方。自古甘肅就有眾多民族雜居，古代分佈在甘肅一帶的[[西夏|西夏]][[党項族|党項族]]後來融入了其他民族。少数民族总数为223万人，占总人口的8.69％。人数较多的少数民族依次是：[[回族|回族]]118万、[[东乡族|东乡族]]45万、[[藏族|藏族]]44万、[[土族|土族]]3万、[[满族|满族]]1.7万、[[蒙古族|蒙古族]]1.6万、[[保安族|保安族]]1.5万、[[裕固族|裕固族]]1.3万、[[撒拉族|撒拉族]]1.2万。
甘肃现有2个自治州：[[临夏回族自治州|临夏回族自治州]]和[[甘南藏族自治州|甘南藏族自治州]]；7个自治县：[[阿克塞|阿克塞]][[哈萨克族|哈萨克族]]自治县，[[肃北|肃北]][[蒙古族|蒙古族]]自治县，[[肃南|肃南]][[裕固族|裕固族]]自治县，[[天祝|天祝]][[藏族|藏族]]自治县，[[张家川|张家川]][[回族|回族]]自治县，[[东乡族自治县|东乡族自治县]]，[[积石山保安族东乡族撒拉族自治县|积石山保安族东乡族撒拉族自治县]]。

甘肃省[[河西|河西]]地区属于[[农业|农业]]发达地区，国家商品粮基地，模式以绿洲农业为主，甘肃[[粮食|粮食]]主要出口地区。相对日照丰富，水果质量非常高，著名的兰州[[白兰瓜|白兰瓜]]、秦安蜜桃远销全国多个城市以及[[静宁|静宁]]苹果在全国享有盛誉。东郭梨在成熟时须用木杆撑起枝条以防止压断树枝，此外尚有醉瓜（有酒香）、铁蛋瓜等多种瓜果。

[[河西走廊|河西走廊]]的[[张掖|张掖]]、[[武威|武威]]、[[酒泉|酒泉]]因位于[[丝绸之路|丝绸之路]]的要道之上，又有雪水浇灌之利，历史上农牧业及商贸发达，被称作“金张掖、银武威、玉酒泉”。

地方小吃有闻名全国的[[兰州牛肉面|兰州牛肉面]]、[[天水|天水]]呱呱、[[静宁|静宁]]烧鸡、静宁锅盔、靖远酸烂肉、武威面皮子、张掖牛肉小饭、山丹炒拨拉等。
地方特产有闻名全国的苦水玫瑰、静宁苹果、兰州百合、天梯山人参果、临泽小枣、兰州白兰瓜、民乐大蒜。

甘肃是新兴的[[工业|工业]]基地，是[[石油工业|石油工业]]、[[有色金属|有色金属]]、[[钢铁工业|钢铁工业]]和[[化工工业|化工工业]]的重要基地。是[[新疆|新疆]]石油的主要加工地点，“兰炼”和“兰化”是中国石化工业的奠基和摇篮。甘肃也是中国[[镍|镍]]、[[铝|铝]]的主要产区和加工基地。酒泉航天中心是中国四个航天基地中最早建设的。

风能资源总储量为2.37亿千瓦，居中国第五位<ref>{{cite web |url = http://news.xinhuanet.com/fortune//2009-06/13/content_11536634.htm |title = 甘肃有望10年内成为全国最大风能产业带 |website = 新华网 |date = 2009-06-13 |accessdate = 2010-01-05 |archive-url = https://web.archive.org/web/20090725083529/http://news.xinhuanet.com/fortune/2009-06/13/content_11536634.htm |archive-date = 2009-07-25 |dead-url = no }}</ref>，其中可开发利用的风能资源总量为2000万千瓦。<ref>{{cite web |url = http://www.gscn.com.cn/pub/gscircs/ziranziyuan/sqfengneng/2008/12/03/1228267061531.html |title = 甘肃省风能储量居全国第五 |website = 中国甘肃网 |date = 2008-12-03 |accessdate = 2010-01-05 |archive-url = https://web.archive.org/web/20110703012618/http://www.gscn.com.cn/pub/gscircs/ziranziyuan/sqfengneng/2008/12/03/1228267061531.html |archive-date = 2011-07-03 |dead-url = yes }}</ref>

甘肃[[畜牧业|畜牧业]]发达，著名品种有“河曲马”和“欧拉羊”。山地牧草稀少，仅能供耐受艰苦环境的[[驴|驴]]生长，形成了独特的驴类畜牧。

=== 旅游 ===

== 文化 ==
甘肃的地域文化为古老的秦陇文化的一部分，以黄河中上游的干支流河谷平原地带形成的典型旱作农耕文化为代表，曾在中国历史上产生深刻影响。

根據2012年的一項調查<ref name="CFPS2012"/>，只有約12％的甘肅人口屬於有組織的宗教，最大的群體是佛教徒，佔8.2％，其次是[[穆斯林|穆斯林]]占3.4％，[[新教|新教]]基督徒佔0.4％，[[公教|公教]]基督徒佔0.1％，甘肅[[基督徒|基督徒]]總數0.5％。大約88％的人口[[無宗教|無宗教]]或信仰[[佛教|佛教]]，[[儒教|儒教]]，[[道教|道教]]和[[中國民間信仰|中國民間信仰]]。

* [[道教|道教]]：可统计信徒21万多人，宗教活动场所212处。平凉[[崆峒山|崆峒山]]较著名。
* [[佛教|佛教]]：可统计信徒80万人，其中[[藏传佛教|藏传佛教]]45万人，主要为[[藏族|藏族]]、[[蒙古族|蒙古族]]、[[土族|土族]]、[[裕固族|裕固族]]<ref name="zj" />。汉传佛教35万多人。宗教活动场所682处。甘南的[[拉卜楞寺|拉卜楞寺]]是[[藏传佛教|藏传佛教]]黄教的六大宗主寺院之一。
* [[亚伯拉罕诸教|亚伯拉罕诸教]]：
** [[伊斯兰教|伊斯兰教]]：穆斯林160万多人，主要为[[回族|回族]]、[[东乡族|东乡族]]、[[撒拉族|撒拉族]]、[[保安族|保安族]]、[[哈萨克族|哈萨克族]]<ref name="zj">{{cite web |url = http://www.gansu.gov.cn/GsglItemQw.asp?id=14200 |title=甘肃省宗教|work=2008年甘肃年鉴 |date=2008-12-16 |accessdate=2010-01-05}}</ref>。宗教活动场所3731处。有“四大教派”（格的目、伊赫瓦尼、西道堂和色来非耶）、“四大[[门宦|门宦]]”（哲赫忍耶、虎夫耶、库布忍耶和尕德忍耶）的分别。其中伊赫瓦尼、西道堂、色来耶和哲赫忍耶等教派和门宦的传始人均在甘肃。
** [[新教|新教]]：可统计信徒6万人，宗教活动场所304处。系由15个教派联合而成：中华基督教会、内地会、宣道会、神召会、福音会、协同会、[[基督复临安息日会|基督复临安息日会]]、基督徒聚会处、浸信会、[[真耶稣教会|真耶稣教会]]等。
** [[天主教|天主教]]：可统计信徒3万多人，宗教活动场所83处。

== 教育 ==
{{See also|甘肃省高等学校列表}}

甘肃省拥有以[[兰州大学|兰州大学]]为首的众多高等教育机构。其中兰州大学位列“211工程”、“985工程”大学行列，入选国家“珠峰计划”、“2011计划”、“111计划”、“卓越法律人才教育培养计划”、“卓越农林人才教育培养计划”、“国家大学生创新性实验计划”、“海外高层次人才引进计划”，上海合作组织大学成员，设有研究生院，是一所综合性全国重点大学和中国西北地区重点教育枢纽。据美国《科学引文索引》（[[SCI|SCI]]）统计，兰州大学是中国在国际学术刊物上发表论文最多、引用率较高的大学之一。

== 交通 ==
1945年[[抗日战争|抗日战争]]结束前夕[[陇海铁路|陇海铁路]]通车至[[天水|天水]]。1952年天兰铁路建成通车，1956年[[兰新铁路|兰新铁路]]建成通车，1980年代从[[乌鲁木齐|乌鲁木齐]]至[[哈萨克斯坦|哈萨克斯坦]]的铁路通车，现在甘肃已经成为连接[[连云港|连云港]]到[[伦敦|伦敦]]的欧亚大陆桥的重要组成部分。目前，甘肃省主要的铁道路线有：陇海线、兰新线、包兰线、兰青线、干武线、敦煌线、寶蘭客運專線、兰新第二双线、兰渝线、天平线（建设中）、兰成线（建设中）。

[[高速公路|高速公路]]纵贯甘肃全境。甘肃省共有五座机场。位于[[兰州市|兰州市]]永登县中川镇的[[兰州中川机场|兰州中川机场]]，每周有飞往香港和新加坡的旅游包机。省内有109、212、213、215、227、309、310、312、316国道过境，西气东输管道过境，因甘肃的地理特征（狭长的一字型走廊），构成了交通、通讯、能源大通道。
