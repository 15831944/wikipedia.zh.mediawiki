'''甘氨酸'''（{{lang-en|Glycine}}，简写为'''Gly'''或'''G'''），即'''胺基乙酸'''，是20个[[蛋白氨基酸|蛋白氨基酸]]中[[分子量|分子量]]最小的一个。它是白色或浅黄色[[晶体|晶体]]，易溶于水，有[[甜|甜]]味。甘氨酸的[[侧键|侧键]]是一个[[氢|氢]][[原子|原子]]。甘氨酸的α[[碳|碳]]连接两个氢原子，故不是[[旋光异构|旋光异构体]]。由于甘氨酸的侧键非常小，它可以占据其它氨基酸无法占据的空间，比如作为[[胶原螺旋|胶原螺旋]]内的氨基酸。

在一些[[蛋白质|蛋白质]]中（比如[[细胞色素|细胞色素]]、[[肌红蛋白|肌红蛋白]]和[[血红蛋白|血红蛋白]]）它随着进化的演变变化相当小，因为假如一个比较大的氨基酸取代它的话整个蛋白质的结构就会变化。

大多数蛋白质只含少量甘氨酸，[[膠原蛋白|膠原蛋白]]是一个重要的例外，它含三分之一的甘氨酸。

== 特性 ==
甘氨酸的水溶性非常好，20[[℃|℃]]时每升水中可以溶解225克甘氨酸。

== 合成 ==
[[甲醛|甲醛]]、[[氰化氢|氰化氢]]和[[水|水]]可以合成甘氨酸：

<math>\mathrm{HCHO + HCN + H_2O \rightarrow H_2N{-}CH_2{-}COOH}</math>

[[File:Betain-Glycine.png|thumb]].]]

== 化学性质 ==
甘氨酸因为含有[[羧基|羧基]]，所以水溶液呈[[酸|酸]]性，可以用[[PH试纸|PH试纸]]测出来。它又能与碱发生中和反应，因此可以测出它的羧基数目。甘氨酸可以与氧气发生燃烧，生成[[氮气|氮气]]、[[二氧化碳|二氧化碳]]、[[水|水]]，它们都是氨基酸组成元素的最稳定氧化物。因为其中含有[[氮|氮]]，所以可以从甘氨酸中提取[[氨气|氨气]]。

甘氨酸常出现在肽链中的[[β折叠|β折叠]]。

== 生理作用 ==
在[[中枢神经系统|中枢神经系统]]，尤其是在[[脊椎|脊椎]]裡，甘氨酸是一个[[抑制性|抑制性]][[神经递质|神经递质]]。假如甘氨酸[[受体|受体]]被激活，[[氯|氯]]离子通过离子接受器进入神经细胞导致[[抑制性突触后电位|抑制性突触后电位]]。[[马钱子碱|马钱子碱]]是这些离子接受器的拮抗物。在鼠体内其[[LD50指标|LD50指标]]为0.96毫克/千克体重，死因是[[超兴奋性|超兴奋性]]。在中枢神经系统中甘氨酸与[[穀氨酸|穀氨酸]]同是激动剂。

生物可以在细胞内自己合成甘氨酸。

== 太空中 ==
1994年[[伊利诺大学|伊利诺大学]]的一个天文学家组称他们在太空中发现了甘氨酸分子。但后来的分析发现这个发现无法证实。2003年[[國立台灣師範大學|國立台灣師範大學]]的[[管一政|管一政]]與Steve Charnley使用[[射电望远镜|射电望远镜]]在三个[[星际物质|星际物质]]中发现了27条甘氨酸的[[光谱|光谱]]<ref>{{cite journal |author=Kuan YJ, Charnley SB, Huang HC, et al.|title=Interstellar glycine |journal=Astrophys J |volume=593 |issue=2 |pages=848–867 |year=2003 |doi=10.1086/375637|bibcode=2003ApJ...593..848K}}</ref>。电脑模拟和实验室裡的试验证明含简单有机分子的冰可能可以在[[紫外线|紫外线]]的照射下形成甘氨酸。

2004年10月的详细研究证明在所提到的三个星际物质中无法找到甘氨酸的迹象。

假如的确能够在太空中找到甘氨酸的话并不说明[[地球|地球]]以外的确有生命，但它说明在星际物质中可以形成氨基酸，它也可以作为[[有生源说|有生源说]]的一个间接支持。
