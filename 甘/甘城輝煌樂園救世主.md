{{日本作品|'''甘城輝煌樂園救世主'''|甘城ブリリアントパーク}}是[[賀東招二|賀東招二]]的[[日本|日本]][[輕小說|輕小說]]作品，由[[中島有華|中島有華]]負責插畫。刊載於《[[富士見Fantasia文庫|富士見Fantasia文庫]]》（[[富士見書房|富士見書房]]）。簡稱《甘輝》。

== 概要 ==
本作為[[賀東招二|賀東招二]]的《[[驚爆危機|驚爆危機]]》完結三年後的新系列。最初類型設為「戀愛喜劇」，之後賀東撤回<ref>賀東招二twitter [https://twitter.com/gatosyoji/status/281355121487654912]</ref>。

故事舞台是在東京西部[[架空|架空]]的都市甘城市，一座即將倒閉的主題公園「甘城輝煌樂園」，主要描寫主角自戀高中生可兒江西也拯救遊樂園與吉祥物打鬧的日常生活。

2014年2月8日於《[[月刊Dragon_Age|月刊Dragon Age]]》連載漫畫<ref>{{cite web|url=http://www.animenewsnetwork.com/news/2014-02-02/shoji-gatoh-amagi-brilliant-park-get-manga-in-dragon-age|title=Shoji Gatoh's ''Amagi Brilliant Park'' Gets Manga in ''Dragon Age''|publisher=[[Anime_News_Network|Anime News Network]]|date=2014年2月2日|accessdate=2014年10月8日|language=en}}</ref>，同年動畫化發表<ref name="anime1" />。

== 故事大綱 ==
某天在放學後的教室，'''可兒江西也'''被轉學生'''千斗五十鈴'''以[[毛瑟槍|毛瑟槍]]抵著[[太陽穴|太陽穴]]並要求與其[[約會|約會]]，心不甘情不願答應後，兩人便在星期日前往「'''甘城輝煌樂園'''」。這座遊樂園建立於[[泡沫經濟|泡沫經濟]]全盛期，近年假日來客人數不超過3千人，同時也是臭名遠播的最糟約會地點代名詞，現在最大的問題是未來三个月內來客人數如不滿50萬人就得關閉，此時遊樂園的公主'''拉媞琺'''委託西也重建遊樂園……

=== 主要人物 ===
;{{nihongo|可兒江西也|可児江 西也（かにえ せいや）|聲：[[內山昂輝|內山昂輝]]<ref name="cast">{{cite web|url=http://www.tbs.co.jp/anime/amaburi/staffcast/#cast|script-title=ja:キャスト|publisher=[[TBS電視台|TBS電視台]]|accessdate=2016年12月28日|language=ja}}</ref>（日本）；Adam Gibbs（美国）}}
:甘城高中一年級生（第二冊後為二年級）。頭腦聰明、運動神經優秀、長相帥氣的自戀美男子，在學校沒有半個朋友。
:被轉學生五十鈴強迫帶到甘城輝煌樂園，並受拉媞琺委托擔任甘城輝煌樂園的經理。起初原本打算拒絕，但在看到遊樂園的慘狀後，決定承擔拯救遊樂園的責任。有[[懼高症|懼高症]]。
:本來只打算擔任兩週的經理(在動畫中，本人直接承諾三個月)，在達成來客人數10萬人，化解遊樂園存亡危機後，繼續擔任遊樂園的經理。
:在被拉媞琺使用神託選中並藉由儀式賜予魔法之後，得到了讀取對方心思的能力，對同一個人只能使用一次。
:在小說版第六冊時發現自己已經能夠重覆讀取同一個人的心思，但重覆讀取心思越多次，就會逐漸失去被讀取的人相關的記憶。
:曾是人氣童星「{{nihongo|兒玉誠也|児玉誠也}}」，這件事是西也不太想觸碰的過去。
:雙親已經離婚，也從父方的姓氏「久武」改為母姓「可兒江」，但扶養權是在父親這邊，改姓是為了隱藏身為童星的過去。父親已再婚，還有一個沒有血緣關係的妹妹，曾四人一起生活過一段時間，但對與原本相處就很冷淡的父親沒什麼情感，也沒辦法適應新的母親與妹妹，後來選擇離開。現是與姑姑久武藍珠兩人一同生活。
:命名源自於[[肯伊·威斯特|肯伊·威斯特]]<ref name="mantan">{{cite web |url=http://mantan-web.jp/2014/08/01/20140731dog00m200059000c.html|title=ラノベ質問状 ： 「甘城ブリリアントパーク」　賀東招二が遊園地で着想　ヒップホップの影響も|publisher=[[每日新聞|每日新聞]]|date=August 1, 2014|accessdate=October 8, 2014|language=ja }}</ref>。

:第3學期轉入甘城高中的美少女轉學生，與西也同班，有著大大的雙眼、亞麻色長髮與巨乳，表情十分冷淡、沉靜。

:其魔法能夠從任何肌膚裸露處取出可填裝多種魔法彈的魔法毛瑟槍「Steinberger」，依據裝填子彈的不同可以引發不同效果。據本人所述，每次皆從裙襬處取出純粹是因為大腿部分的裸露面積較大方便拔槍。
:受到河童家系的祖先影響，本人言「一天沒洗三次澡就會有種快死掉的感覺」，也喜歡吃黃瓜。
:變成吉祥物型態時是一隻很可愛的粉紅色河童，拔出Steinberger的位子則是頭部的蓋子。五十鈴本人並不喜歡這個樣子。
:小說第二冊開始對西也抱持著微妙的好感，但本人內心刻意的否認。小說第四冊末以後，對西也的稱呼由「可兒江同學」改為「西也同學」。
:在動畫第14集中因為繆絲落枕而互換了身分，但是失誤連連。

:甘城輝煌樂園的支配者，雖然看似是只有14、15歲的少女，但其身體從十年前就沒有成長，實際年齡24歲。魔法國度「紅楓樂園」的公主。近衛隊五十鈴的工作對象。
:因為受到魔法師「欠缺魂之力」的詛咒影響，體弱多病且眼睛失明（動畫版中並無失明），並且身上還有「一年後身體的成長與記憶都會重來」的惡毒詛咒。
:唯有待在遊樂園才能存活，所以為了復興快倒閉的甘輝，使用神託選中了西也擔任經理。
:在動畫第14集中因為繆絲落枕，與千斗互換了身分，累積了不少工作造成失誤連連。
:命名源自於[[拉蒂法女皇|皇后·拉蒂法]]<ref name="mantan"/>。

=== 真實演員 ===
====甘輝三連星====
由鬆鬆餅、馬卡龍和堤拉米組成的{{nihongo|「甘輝[[三連星|三連星]]」|甘ブリ[[三羽烏|三羽ガラス]]}}。

;{{nihongo|鬆鬆餅|モッフル|Moffle|聲：[[川澄綾子|川澄綾子]]<ref name="cast"/>}}
:遊樂設施{{nihongo|「鬆鬆餅糖果屋」|モッフルのお菓子ハウス}}的[[吉祥物|吉祥物]]，糖果妖精，是吉祥物中的招牌角色，也是樂園幹部中的統轄者。
:血氣方剛，時常與來賓起爭執甚至動粗，是時常引發問題的吉祥物，但受到其他吉祥物的信任。
:與拉媞琺是舅舅與外甥女的關係（鬆鬆餅的姐姐是王妃），雖然本人並未在公開場合中透露，但是其他工作人員都知道這件事情。
:貴族中的軍人世家出身，隸屬紅楓樂園的特殊部隊，透過政治關係將職位提升到將軍，最後為了救受詛咒的拉媞琺而辭去軍職。
:高中時就讀{{nihongo|飄揚國防高中|ふわふわ国防高校}}，曾受過嚴苛訓練。
:全名{{nihongo|「鬆鬆餅·梅爾·摩西納斯」|モッフル・メル・モーセナス}}，雖然是甜點妖精，但並不喜歡甜食，反而喜歡可樂餅之類的食物。
:使用魔法APP拍攝出的人類姿態酷似演出《[[魔戒|-{zh-cn:指环王;zh-hk:魔戒;zh-tw:魔戒;}-]]》的[[維高·摩天臣|-{zh-cn:维戈·莫腾森;zh-hk:維高·摩天臣;zh-tw:維果·莫天森;}-]]。

:害怕那些喜歡對吉祥物SM的人，曾誤會西也是相關方面的人士而感到害怕。

;{{nihongo|馬卡龍|マカロン|Macaron|聲：[[白石涼子|白石涼子]]<ref name="cast"/>}}

:在遊樂園下班以後，時常與鬆鬆餅、堤拉米一起去居酒屋，也有吸菸的習慣。
:在紅楓樂園以暴力事件聞名的{{nihongo|震顫旋律高中|どきどきメロディ高校}}就讀時當過大哥，之後中途被退學。
:全名{{nihongo|「馬卡龍·馬可澤卡隆」|マカロン・マクゼカロン}}，日本名{{nihongo|「間桑一郎」|間桑 一郎}}；已經結婚，但目前處於離婚狀態，現在正為了女兒的贍養費而煩惱。
:使用魔法APP拍攝出的人類姿態是有點壞壞感覺的三十多歲音樂人美型男。
:害怕那些喜歡對吉祥物SM的人。

;{{nihongo|堤拉米|ティラミー|Tirami|聲：[[野中藍|野中藍]]<ref name="cast"/>}}

:本性好色並且喜歡開黃腔，對任何年齡層的女人都來者不拒，綽號{{nihongo|「51號性區」|性のエリア51}}，意指守備範圍跟[[背號|背號]]51的[[鈴木一朗|鈴木一朗]]有得拼。
:學生時代曾在園藝社栽種[[大麻|大麻]]，最後為了銷毀證據企圖放火燒毀，不料連學校也一併燒毀。
:因破解紅楓銀行金庫入獄留有前科，依{{nihongo|「毛茸茸計畫」|もこもこプログラム}}送到甘輝來社會服務。職業足球隊「Melody柴崎」的球迷。

== 用語 ==
; {{nihongo|甘城輝煌樂園|甘城ブリリアントパーク|Amagi Brilliant Park}}
: 本作主要舞台。東京西部．甘城市荒涼的遊樂園，特別是近年更受到許多負面的評價，假日進園人數不到3000人。
: 建立於1980年代的泡沫經濟全盛期，並原本預定在南區增建第二園區，但因為泡沫經濟破裂，資金週轉不靈，取消了此計畫。
: 當時的擴建計畫雖然作廢，仍然留下了被閉鎖的露營地與計畫實施前就蓋好的體育場。
: 遊樂園是由魔法國度{{nihongo|「紅楓樂園」|メープルランド|Maple Land}}出資的不動產行業「紅楓不動產」，還有在多摩地域具影響力的鐵路公司與甘城市出資成立的[[第三部門|第三部門]]——以廢止樂園為目標的「甘城企劃」共同營運。
: 過去兩陣營定下了數年間無法達成規定的來客數就得閉園的契約，故事開始時遊樂園面臨兩週內要有十萬人入場的困境（動畫改為三個月五十萬人）。
: 甘輝即是紅楓樂園的「頌葛尼」（日文「田園」之意），會凝聚客人心中的快樂與雀躍氣氛形成名為「魂之力」的結晶，這對魔法國度的人們是極為重要的能量，特別是受詛咒的拉媞琺更是需要魂之力，這也是紅楓不動產要維持遊樂園營運的主要原因。

; {{nihongo|真實演員|リアルキャスト|Real Cast}}
: 意指從魔法國度「紅楓樂園」來的工作人員。單以「演員」的情況下，意指地上界的工作人員。
: 甘輝的真實演員主要是拉媞琺、五十鈴、鬆鬆餅……等等，西也則是少數的「演員」。
: 作品中也有紅楓樂園以外的魔法國度，表示有很多真實演員存在。

; {{nihongo|斯坦伯格|シュタインベルガー|Steinberger}}<ref>[[普威爾|普威爾]]譯名，原作採英譯。</ref>
: 五十鈴隨身攜帶的毛瑟槍，是五十鈴家代代相傳的魔法武器，根據裝填的魔彈，能發揮出各種不同效果，而且前膛槍彈可以連續射擊。
: Steinberger是與五十鈴身心一體的魔法武器，能自由從任何部位的裸露肌膚拔出或收起槍，五十鈴大多從大腿拔出槍。
: 在美衣乃的詛咒事件中受惡靈的攻擊而摧毀，但其實五十鈴家裡的倉庫還有五十挺槍。

=== 與原作的差異 ===
* 原作鬆鬆餅被說壞話會生氣，動畫版把壞話改成{{nihongo|仿冒品|パクリ}}。
* 主角擔任代經理的期間，原作為2週，動畫版為3個月。
* 來客人數，原作為10萬人，動畫版為50萬人。
* {{nihongo|「水族館」|アクワーリオ|Acuario{{es}}}}遊樂設施更名為{{nihongo|「元素館」|エレメンタリオ|Elementario{{La icon}}}}。
* 拉媞琺失明設定無提及。
* 凡涉及敏感尺度均修正，例如具有性意味、美衣乃面試時大量失血畫面等未出現。
* 第11話於園區內體育館進行足球比賽的展開，原作為鄰近城市的體育館因火災而按照契約外借予地區球隊作[[日本職業足球聯盟|J聯盟]]開幕戰用，動畫版為爭取大量入場人數而主動贊助球隊到園區內體育館舉辦紀念足球比賽。
* 第12話在遊樂園達成來客人數邁入新的一年後，原作中拉媞琺的記憶被「重置」且完全忘記西也在前一年的活躍，動畫版中拉媞琺則因城堡庭園的大樹盛開，其所釋放之魂之力使詛咒得到緩解，成功保下這一年的記憶與成長。
