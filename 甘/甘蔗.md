'''甘蔗'''是[[禾本科|禾本科]]的[[单子叶植物|单子叶植物]]，为'''甘蔗属'''（[[学名|学名]]：{{lang|la|''Saccharum''}}）的总称。甘蔗是温带和热带农作物，是制造[[蔗糖|蔗糖]]的原料，且可提煉[[乙醇|乙醇]]作為[[生質能源|生質能源]]。全世界有一百多个国家出产甘蔗，最大的甘蔗生产国是[[巴西|巴西]]、[[印度|印度]]和[[中国|中国]]。中国最常见的食用甘蔗为中国竹蔗。

本属属于[[黍亚科|黍亚科]][[鬚芒草總族|鬚芒草總族]][[鬚芒草族|鬚芒草族]][[甘蔗亚族|甘蔗亚族]]。{{r|Soreng2015|Soreng2013}}

== 傳播歷史 ==
{{Refimprove section|time=2016-07-29T02:53:15+00:00}} 
[[File:1962-07_1962年_甘蔗制糖厂.jpg|thumb]]
甘蔗起源问题现在认识不一，有多种说法。一说起源于中国，中国华南、西南南部一带可能是甘蔗的原产地之一。其主要论据是：
#于公元前4世纪末，中国语系中早有“柘”或“薯柘”一词，后来“柘”字衍作“蔗”。
#甘蔗的命名，纯粹是按中国驯化植物的习惯来命名的，没有加上引进植物的“胡”、“番”、“洋”和外来地名，也不译音。
#甘蔗的近缘植物在中国分布很广，类型也相当多，甘蔗亚族中9个属植物都有。主要的甘蔗属割手密，北自秦岭  ，南至海南岛都有分布。尤其是华南、西南南部一带分布最多。
#澳大利亚的科学工作者分析了甘蔗的异构酶和黄酮类化合物，判断甘蔗物种间亲缘关系，准确的追溯到甘蔗的祖先是甘蔗属的[[割手密|割手密]]（學名：{{lang|la|''Saccharum Spontaneum''}}）和芒属的[[五节芒|五节芒]]（學名：{{lang|la|''Miscanthus Flordulus''}}）中国南部、西南部，是世界上甘蔗野生种割手密和五节芒的一个重要分布中心。
由此推论，中国华南、西南南部一带为甘蔗的原产地之一，甘蔗属的中国种（{{lang|la|''S.Singnse''}}），可以肯定是起源于中国。 

另一种说法為甘蔗原产地可能是[[新畿内亞|新畿内亞]]（[[新畿内亚野生蔗|新畿内亚野生蔗]]）或[[印度|印度]]，后来传播到南洋群岛，大约在周朝周宣王时传入中国南方。先秦时代的“柘”就是甘蔗，到了[[汉代|汉代]]才出现“蔗”字<ref>[[东汉|东汉]][[杨孚|杨孚]]《[[异物志|异物志]]》：“甘蔗，远近皆有。交趾所产，特好，本末无厚薄，其味甘”</ref>，“柘”和“蔗”的读音可能来自梵文sakara。10到13世纪（[[宋代|宋代]]），江南各省普遍种植甘蔗；中南半岛和南洋各地如[[真腊|真腊]]、[[占城|占城]]、[[三佛齐|三佛齐]]、[[苏吉丹|苏吉丹]]也普遍种甘蔗制糖。

[[亚历山大大帝|亚历山大大帝]]东征[[印度|印度]]时，部下一位将领曾说印度出产一种不需[[蜜蜂|蜜蜂]]就能产生[[蜜糖|蜜糖]]的草。公元6世纪[[伊朗|伊朗]][[萨珊王朝|萨珊王朝]]国王[[霍斯劳一世|霍斯劳一世]]（Khosrau I Anushirvan）将甘蔗引入伊朗种植。8到10世纪甘蔗的种植遍及[[伊拉克|伊拉克]]、[[埃及|埃及]]、[[西西里|西西里]]、[[伊比利亚半岛|伊比利亚半岛]]等地。后来葡萄牙和西班牙殖民者又把甘蔗带到了美洲。

== 臺灣甘蔗農業 ==
在[[台灣日據時期|日據時代]]到[[臺灣光復|光復]]初期，甘蔗為臺灣最重要的經濟作物之一。臺灣一年四季皆適合甘蔗生長，惟7-9月間常有颱風來襲，對甘蔗生長有不良影響。依據陳正祥氏的說法，臺灣最適合種植甘蔗的地區為[[雲林縣|雲林縣]][[虎尾鎮|虎尾鎮]]到[[屏東縣|屏東縣]]的[[平原|平原]]，因本區地形平坦，氣候良好。臺灣常見的經濟作物甘蔗有兩種：
* [[中國竹蔗|中國竹蔗]]（{{lang|la|''Saccharum sinense''}}）：即中文傳統上的甘蔗，俗稱「白甘蔗」<ref>[http://plant.tesri.gov.tw/plant100/WebPlantDetail.aspx?tno=628108030 甘蔗]，臺灣野生植物資料庫</ref>，外皮綠色，質地粗硬，不適合生吃，產量多，含糖量高。
* [[秀貴甘蔗|秀貴甘蔗]]（{{lang|la|''Saccharum officinarum''}}）：俗稱「紅甘蔗」<ref>[http://plant.tesri.gov.tw/plant100/WebPlantDetail.aspx?tno=628108020 秀貴甘蔗]，臺灣野生植物資料庫</ref>，皮墨紅色，莖肉富纖維質多汁液，清甜嫩脆，食而不膩。
另外還有：
* [[新幾內亞野生種甘蔗|新幾內亞野生種甘蔗]]：特徵為葉片亦為紫色，因糖分低，多不用來製糖，亦不食用。先民以其抗病力強，遂以訛傳訛的成為中藥材。在臺灣品種日漸退化，莖細長，反而不太像甘蔗，甚至於被誤認為茅草 。
* [[印度蔗|印度蔗]]（{{lang|la|''Saccharum barberi''}}）
* 野生甘蔗

== 甘蔗的營養 ==
[[File:Cut_sugarcane.jpg|thumb]]
《[[本草綱目|本草綱目]]》記載甘蔗能夠止咳化痰、利尿、養顏美容。<ref>《本草綱目》
甘蔗
释名 名竿蔗、遮。
气味 （蔗）甘、平、涩、无毒。</ref>主治：
#发热口干、小便赤涩。取甘蔗去皮，嚼汁咽下。饮浆亦可。
#反胃吐食。用甘蔗汁七升、生姜汁一升，和匀，每日细细饮服。
#干呕不息。有蔗汁温服半升，每日三次。加姜汁更好。
#虚热咳嗽，口干涕唾。用甘蔗汁一程式半、青粱米四合，煮粥吃。
#每日二次。极润心肺。
