'''甘拉亚妮·瓦塔娜'''（{{lang-th|กัลยาณิวัฒนา}}，{{IPA-th|kān.lā.jāː.ní.wát.tʰā.nāː}}，{{RTGS|Kanlayaniwatthana}}，{{bd|1923年|5月6日|2008年|1月2日}}）生于[[英国|英国]][[伦敦|伦敦]]，[[泰国|泰国]]语言学家，[[泰国国王|泰国国王]][[阿南塔·玛希敦|阿南塔]]和[[普密蓬·阿杜德|普密蓬]]的姐姐<ref name=lzh>{{Cite web |url = http://news.163.com/08/1115/16/4QQ97GKT0001121M.html |title = 泰国为王姐甘拉亚妮举行隆重皇家葬礼 |author =  |date = 2008-11-15 |publisher =网易 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
甘拉亚妮·瓦塔娜生于[[英国|英国]][[伦敦|伦敦]]，大学期间学习理科专业，同时艺术和语言造诣深厚。曾任[[泰国|泰国]]多所高校的客座教授，讲授外语及[[语言学|语言学]]，尤其是[[法语|法语]]。她是泰国法语教师协会创始人，1977年至1981年任该协会主席。甘拉亚妮·瓦塔娜经常出访，曾多次访问[[中国|中国]]，到过[[云南|云南]]、[[新疆|新疆]]等地<ref name=wy>{{Cite web |url = http://news.163.com/08/1115/15/4QQ64D4O000120GU.html |title = 泰国为王姐甘拉亚妮举行皇家隆重葬礼 |author =  |date = 2008-11-15 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2008年1月2日，甘拉亚妮·瓦塔娜在[[曼谷|曼谷]][[诗里拉吉医院|诗里拉吉医院]]病逝，享年84岁。同日开始持续100天的全国悼念活动。2008年11月14日晚，葬礼按泰国王家习俗在曼谷市中心的[[曼谷大皇宫|大王宫]]开始举行，仪式由[[普密蓬·阿杜德|普密蓬]]国王主持，王后[[诗丽吉王后|诗丽吉]]及其他王室成员和泰国各界人士出席。按照泰国政府安排，葬礼持续六天，其中遗体火化仪式于11月15日晚在曼谷[[王家田广场|王家田广场]]举行。仪式期间，泰国政府部门及学校等单位降半旗，全国暂停娱乐活动。11月15日早七时，泰国王储[[哇集拉隆功|哇集拉隆功]]、公主[[诗琳通公主|诗琳通]]等王室成员代表国王和王后到大王宫将遗体迎至灵车上，从大王宫西侧门移出，由人力车牵引灵车进王家田广场送入火葬亭。参加火葬礼的外国贵宾三千五百人（其中包括二百余个国家的外交使节），泰国民众数以万计。骨灰于16日晨收取，并在19日葬礼全部结束时供奉在大王宫内的玉佛寺<ref name=wy/><ref name=lzh/><ref>{{Cite web |url = http://news.163.com/08/1115/22/4QQRLPOB000120GU.html |title =  泰国为已故王姐瓦塔娜公主举行盛大葬礼|author =  |date = 2008-11-15 |publisher =  网易|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.ce.cn/xwzx/gjss/gdxw/200801/02/t20080102_14093644.shtml |title = 泰国王姐甘拉亚妮·瓦塔娜去世 |author =  |date =2008-01-02  |publisher = 中国经济网 |language =  |accessdate =  |quote =  }}</ref>。
