'''纪姓'''为[[中文姓氏|中文姓氏]]之一，在《[[百家姓|百家姓]]》中排名第122位。作为姓氏的“纪”读音应为“几（Jǐ）”，但现在一般都读作“季（Jì）”。

== 起源 ==
以国为姓，出自[[姜姓|姜姓]]。据《[[元和姓纂|元和姓纂]]》、《[[通志|通志]]·氏族略：以国为氏》载，[[西周|西周]]初年，为追念先圣先王的功德，封[[炎帝|炎帝]]的一个后代于纪（在今山东[[寿光市|寿光市]]东南），建立了[[纪国|纪国]]。[[春秋|春秋]]时，纪国被[[齐国|齐国]]所灭，纪国王族子孙就以国名为姓，世代相传姓纪。

纪姓另有一支由[[舒姓|舒姓]]改成。

== 郡望堂号 ==
望出天水（今属甘肃）、高阳（今河北保定一带）、平阳（今山西省临汾市）、襄平（今辽宁辽阳北——满族）。
　
== 分布 ==
纪姓现遍居全国各地，海外也有分布。据2006年数据，大陆纪姓按人口数排名157。一支于明末清初迁居台湾，据1978年统计，“纪”为台湾第71大姓。

名人 
*[[纪昌|纪昌]]：古代箭术家。
*[[纪信|纪信]]：为[[刘邦|刘邦]]起兵抗[[秦|秦]]时的部将。
*[[纪靈|纪靈]]：袁術的部下
*[[纪瞻|纪瞻]]：[[西晋|西晋]]末年会稽内史、官至骠骑将军。 
*[[纪少瑜|纪少瑜]]：[[南朝|南朝]][[梁國|梁國]]诗人。
*[[纪君祥|纪君祥]]：元代[[戏曲作家|戏曲作家]]。著有杂剧6种，现仅存《[[赵氏孤儿|赵氏孤儿]]》。
*[[纪晓岚|纪晓岚]]：清代乾隆、嘉庆年间朝廷重臣，著名学者。
*[[纪玉良|纪玉良]]：[[京剧|京剧]]老生演员。
*[[纪登奎|纪登奎]]：[[山西|山西]][[武乡|武乡]]人。曾任[[中华人民共和国|中华人民共和国]][[国务院|国务院]][[副总理|副总理]]。
*[[纪振|纪振]]：台湾台中縣海埔厝。台灣梧棲開港，擔任大肚中堡五十三庄大總理，又稱《[[開港總理|開港總理]]》
*[[纪政|纪政]]：[[台湾|台湾]][[短跑|短跑]][[运动员|运动员]]，[[1968年夏季奧林匹克運動會|1968年夏季奧林匹克運動會]]女子100公尺低欄銅牌得主。
*[[纪長|纪長]]：台湾雲林縣水林。於日治時期經營蔗糖產業，種植甘蔗並進行品種改量.其子紀經總後為甘蔗委員.<ref>台灣版高陽紀氏族譜</ref>
*[[紀淑如|紀淑如]]：台灣跆拳道運動員，[[2000年夏季奧林匹克運動會|2000年夏季奧林匹克運動會]]跆拳道女子58公斤級銅牌得主。
*[[紀曉君|紀曉君]]：台灣歌手，[[卑南族|卑南族]]。
