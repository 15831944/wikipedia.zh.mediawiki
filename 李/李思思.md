'''李思思'''（{{bd|1986年|11月4日|catIdx=L李}}），[[中华人民共和国|中国]][[吉林省|吉林省]][[长春市|长春市]]人，[[中国中央电视台|中国中央电视台]]主持人。2012年起主持央视春晚，成为史上最年轻的春晚主持人<ref>{{cite news|url=
http://news.163.com/13/0124/11/8LVTH4L300014JB6_9.html|title=央视春晚30年之最：刘晓庆最早李思思最年轻主持|publisher=网易|date=2013-01-24|accessdate=}}</ref>。

== 主要经历 ==
*1998年 - 2001年，长春外国语学校
*2001年 - 2004年，东北师范大学附属中学
*2004年 - 2008年，[[北京大学|北京大学]]艺术学院影视编导专业
*2005年，参加央视《[[挑战主持人|挑战主持人]]》节目，成为该节目史上第一位八期女擂主
*2006年，在央视《挑战主持人》节目全国选拔赛中获得季军
*2007年起，主持央视节目
*2008年，主持[[北京大学|北京大学]]校庆110周年庆典晚会
*2011年，主持央视春晚倒计八小时
*2012年 - 2014年，主持央视春晚及元宵晚会等

== 个人生活 ==
李思思的丈夫是她在读大三时认识的一位80后理工男。<ref>{{cite web |title=《星月私房话》20151224:李思思揭秘央视女主播成名背后 学舞蹈让我变成女汉子 |url=http://www.le.com/ptv/vplay/24263151.html |website=乐视视频 |accessdate=2019-02-21}}</ref>两人在2014年生下一子，名为“元宝”<ref>{{cite news |title=李思思产后快速恢复身材 儿子小名元宝儿 |url=http://ent.sina.com.cn/s/m/2015-02-11/doc-icczmvun6048373.shtml |accessdate=2019-02-21 |work=新浪 |date=2015-02-11}}</ref>；在2017年产下二胎儿子，名为“皮皮”。<ref>{{cite news |title=李思思二胎儿子首曝光 趴妈妈胸口安逸休息 |url=http://ent.ifeng.com/a/20170427/42914278_0.shtml |accessdate=2019-02-21 |work=凤凰网 |date=2017-04-27}}</ref>

== 争议 ==
2011年，[[湖南卫视|湖南卫视]]主持人[[张丹丹|张丹丹]]突然在自己微博中，评价CCTV3《欢乐英雄》包含李思思在内的三位女主持人：“没见过这么恶心人的主持人！长得真漂亮啊！” <ref>[http://www.mgtv.com/c/20110930/0903227572.html 张丹丹微博发飙“没见过这么恶心人的主持人] {{webarchive|url=https://web.archive.org/web/20160418232522/http://www.mgtv.com/c/20110930/0903227572.html |date=2016-04-18 }}</ref> 接着她又发了一条微博：“马上换台到央3，自己看，看得下去算你狠！” <ref>[http://gzdaily.dayoo.com/html/2011-09/29/content_1489040.htm 广州日报：管彤回应：言论自由，我们仨很稳定] {{webarchive|url=https://web.archive.org/web/20111002122914/http://gzdaily.dayoo.com/html/2011-09/29/content_1489040.htm |date=2011-10-02 }}</ref>，引起网友对CCTV主持人照本宣科，僵化的主持方式的讨论，以及对张丹丹微博的批评，原微博已经删除。
