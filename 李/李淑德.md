'''李淑德'''（{{bd|1929年||}}），專長[[小提琴|小提琴]]。曾擔任許多管弦樂團指揮，對台灣音樂教育貢獻良多，有｢台灣小提琴教母」的美譽。

== 生平 ==
[[File:李淑德故居.jpg|thumb]]
1929年出生于[[台灣|台灣]][[屏東|屏東]]，在家中排行老四。醫師父親李明家熱愛音樂，曾師從張福興學習[[小提琴|小提琴]]，母親也善於彈奏鋼琴
<ref name = 'portfolio'>{{cite web|title=台灣小提琴教母李淑德 | url=http://www.t-classical.com/portfolio-posts/lee-2| work=台灣古典音樂網 |accessdate=2017-12-20}}</ref>
。李淑德的父親在家中收藏了不少的樂器，逢年過節也常常聘請[[南管|南管]]樂團至家中演奏，並在屏東當地成立「曼陀鈴」樂團，李淑德從小便受家庭氛圍的影響對音樂產生興趣<ref name = 'musiciantw'>{{cite web|title=李淑德| url= http://musiciantw.ncfta.gov.tw/list.aspx?p=M017&c=&t=1| work=台湾音乐家名录 |accessdate=2017-12-20}}</ref>。1942李淑德年進入[[屏東女中|屏東女中]]就讀，为了能进入学生院团，李淑德開始學習小提琴<ref name = 'musiciantw'/>。學習過程中她一度愛上[[大提琴|大提琴]]的音色，但最後由於坐不住且嚮往小提琴的自由自在而選擇小提琴為終身志願。<ref name = 'yee'/>

1948年李淑德進考進[[臺灣省立師範學院|臺灣省立師範學院]]（今[[國立台灣師範大學|國立台灣師範大學]]），就讀於藝術系。二年級學校因「[[四六學潮|四六學潮]]」停課一個月，重新開課後許多同學選擇不繼續就學。因此李淑德決定轉系至音樂系，師從[[戴粹倫|戴粹倫]]學習小提琴，並成為了師大音樂系第一屆畢業生<ref name = 'musiciantw'/>。

1957年，李淑德赴[[美國|美國]]紐約州[[奈亞克教會學院|奈亞克教會學院]]就讀，三個月後成功申請獎學金，進入美國波士頓[[新英格蘭音樂院|新英格蘭音樂院]]就讀，申請時評審詢問李淑德未來的志向，李淑德因為自認起步比別人晚，不容易成為演奏家，於是回答：「我要當一位好老師」<ref name = 'yee'>{{cite web| title= 台灣音樂邁向國際的重要一人?台灣小提琴教母李淑德 | url=http://archives.lib.ntnu.edu.tw/c4/c4_5_47.jsp|work=國立台灣師範大學校史館 |}}</ref>。入學後她先後跟隨[[阿佛列‧克利波斯|阿佛列‧克利波斯]]（Alfred Krips）與[[露絲‧波賽特|露絲‧波賽特]]（Rufh Posselt）學習小提琴，並於1964年取得小提琴音樂碩士<ref name = 'musiciantw'/>。

1964年李淑德回台任教於[[臺南神學院|臺南神學院]]教授和聲學與試唱聽寫，並與[[楊瓊珍|楊瓊珍]]合作在台灣舉辦了四場演唱會。1965年獲母校國立台灣師範大學聘請任教，教學生涯教授了許多知名小提琴家，如[[林昭亮|林昭亮]]、[[胡乃元|胡乃元]]、[[蘇正途|蘇正途]]、[[陳沁紅|陳沁紅]]、[[簡名彥|簡名彥]]等<ref name = 'portfolio'/><ref name = 'musiciantw'/>。

1971年由李淑德擔任團長、教練兼樂團指揮的「[[中華少年管絃樂團|中華少年管絃樂團]]」成立，除了不時聘請客席指揮來指導樂團外，李淑德也在台南開設儲訓班訓練未來團員，並舉辦暑期集訓營。1976年「中華少年管絃樂團」以「中華民國華美青少年絃樂團」為名赴美巡迴演出，慶祝美國建國兩百年。1995年從國立台灣師範大學退休，至[[國立臺灣師範大學附屬高級中學|國立臺灣師範大學附屬高級中學]]音樂班擔任[[管絃樂團|管絃樂團]]指揮<ref name = 'musiciantw'/>。2009年舉辦八十大壽音樂會，曾經的學生共聚一堂，參與演出
<ref>{{cite web| title= 李淑德教授80大壽音樂會| url=http://www.arsformosa.com.tw/NDisplay.asp?Aid=515 |work=傳大藝術 | accessdate=2017-12-20}}</ref>。2016年，李淑德的學生鄭俊騰先生
