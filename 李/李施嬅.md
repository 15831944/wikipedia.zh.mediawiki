'''李施嬅'''（{{lang-en|'''Selena Lee Sze Wa'''}}，{{bd|1981年|2月12日|catIdx=L李}}），前名為'''李詩韻'''。<ref>{{citeweb|url= https://hk.entertainment.appledaily.com/enews/realtime/article/20150210/53421913|title= 【短片】【嫲嫲幫我改】李詩韻忽然改名叫「李施嬅」：覺得適合我嗎？|work= 蘋果日報|date= 2015-02-10}}</ref>出生於[[香港|香港]]，[[加拿大|加拿大]][[華人|華人]]，[[香港|香港]]女[[演員|演員]]，前[[無綫電視藝員列表|無綫電視經理人合約]]藝員。

==簡介==
李施嬅出生於[[香港|香港]][[富裕|富裕]]家庭，家中排行第三，其父親李光輝原本經營專門生產計算機的公司「雅達家電」。在7歲時隨家人移民到[[加拿大|加拿大]][[多倫多|多倫多]]，父親則長期在[[香港|香港]]工作，其後父母離婚，父親另娶，李施嬅有一對同父異母的弟妹。李施嬅跟母親和兄長居[[加拿大|加拿大]]，只能在暑假返[[香港|香港]]探父親，一年見一次面<ref>{{Cite web |publisher=蘋果日報 |title=準港姐李施嬅富爹疑做假帳 |url=http://tw.nextmedia.com/applenews/article/art_id/257981/IssueID/20030810 |date=2003年8月10日}}</ref>。父親的公司於2004年底被迫清盤，欠下一億巨債<ref>{{Cite web |publisher=壹週刊 |title=掟胡定欣 馬國明接收李施嬅 |url=http://hk.next.nextmedia.com/template/next/art_main.php?iss_id=1052&sec_id=1000847&art_id=13998774 |date=2010年5月6日}}</ref>，使李施嬅的生活立即大受影響，並與哥哥姐姐一起負起供養父母各自的家庭<ref>{{Cite web |publisher=香港文匯報 |title=形象清純緋聞少千金小姐決意賺錢養家 |url=http://paper.wenweipo.com/2008/10/13/EN0810130020.htm |date=2008年10月13日}}</ref>。
 
中學畢業後，李施嬅就讀於[[多倫多大學|多倫多大學]]主修[[會計|會計]]，曾是學校商科學生會[[副主席|副主席]]<ref>{{Cite web|publisher=新華網|title=爆料娛樂圈假學歷 "克萊登大學"明星校友現形記|url=http://news.xinhuanet.com/ent/2010-07/23/c_12363779.htm|date=2010年7月23日|deadurl=yes|archiveurl=https://web.archive.org/web/20120125083351/http://news.xinhuanet.com/ent/2010-07/23/c_12363779.htm|archivedate=2012年1月25日}}</ref>。

李施嬅於2003年回港參選[[香港小姐競選|香港小姐競選]]，因而綴學。李施嬅在準決賽以爆冷姿態奪得「最上鏡小姐」及「才藝小姐」獎項；爆冷奪獎後，李施嬅則被傳媒視為決賽三甲大熱門，最後大熱倒灶未能晉身前五名，更是首位五甲不入的最上鏡小姐<ref>{{Cite web |publisher=新浪娛樂 |title=最上鏡小姐李施嬅、修身美態獎得主李嘉怡訪談 |url=http://ent.sina.com.cn/s/h/2003-08-26/0754191171.html |date=2003年8月26日}}</ref>。
	 
當選後，李施嬅與[[香港|香港]][[無綫電視|無綫電視]]簽約，成為旗下經理人合約藝員。翌年拍攝首部電視劇《[[天涯俠醫|天涯俠醫]]》當配角，其後於多部電視劇中擔任配角，包括《[[阿旺新傳|阿旺新傳]]》、《[[佛山贊師父|佛山贊師父]]》、《[[最美麗的第七天|最美麗的第七天]]》、《[[太極_(電視劇)|太極]]》及《[[東山飄雨西關晴|東山飄雨西關晴]]》等電視劇。2005年在《[[老婆大人|老婆大人]]》飾演梁昕昕，漸露頭角，更因此而與同劇演員[[鄧健泓|鄧健泓]]交往，其後於2010年1月宣佈分手。

2005年至2008年分別參演《[[肥田囍事|肥田囍事]]》、《[[天機算|天機算]]》、《[[太極|太極]]》、《[[律政新人王II|律政新人王II]]》等電視劇。

2009年11月，李施嬅向無綫電視請大假返多倫多留學4個月，完成了10年前的商科學位課程<ref>{{Cite web|publisher=明報周刊|title=終圓母親心願 李施嬅花十年完成大學|url=http://www2.mingpaoweekly.com/contents/?id=7116|date=2010年7月18日|deadurl=yes|archiveurl=https://web.archive.org/web/20140228224916/http://www2.mingpaoweekly.com/contents/?id=7116|archivedate=2014年2月28日}}</ref>。其後，李施嬅投資過[[百萬|百萬]]開設「Technique」[[化妝|化妝]][[學校|學校]]，獨資當上老闆娘<ref>{{Cite web|publisher=明報周刊|title=為創業甘願延遲置業夢 李施嬅投資百萬開化妝學校|url=http://www2.mingpaoweekly.com/contents/?id=5872|date=2010年4月10日|deadurl=yes|archiveurl=https://web.archive.org/web/20140228225803/http://www2.mingpaoweekly.com/contents/?id=5872|archivedate=2014年2月28日}}</ref>，於2010年4月正式舉行開幕儀式，更不少圈中人友出席<ref>{{Cite web |publisher=東方日報 |title=李施嬅喊住做老闆 |url=http://orientaldaily.on.cc/cnt/entertainment/20100430/00282_017.html |date=2010年4月30日}}</ref><ref>{{Cite web|publisher=成報|title=馬國明胡定欣拒合照　李施嬅投資百萬開化妝學校|url=http://www.singpao.com/newsarticle.aspx?NewsID=89452&Lang=tc|date=2010年4月30日}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 李施嬅為學校考了ITEC國際美容文憑，<ref>{{Cite web |publisher=東方日報 |title=李施嬅job不停 壓力爆瘡 |url=http://orientaldaily.on.cc/cnt/entertainment/20100917/00282_038.html |date=2010年9月17日}}</ref>凡事親力親為，<ref>{{Cite web |publisher=東方日報 |title=李施嬅為慳皮撈埋清潔 |url=http://orientaldaily.on.cc/cnt/entertainment/20100519/00282_022.html |date=2010年5月19日}}</ref><ref>{{Cite web |publisher=3週刊 |title=留星語：李老闆 李施嬅 |url=http://www.powerfulnews.com.my/ctgtid.aspx?ctg=19&ctt=5307 |date=2008年10月15日 |deadurl=yes |archiveurl=https://web.archive.org/web/20111010110334/http://www.powerfulnews.com.my/ctgtid.aspx?ctg=19 |archivedate=2011年10月10日 }}</ref>。更因此直到該年11月暫停拍劇<ref>{{Cite web |publisher=頭條日報 |title=名人戰商界：不計成本 李施嬅 憑良心開化妝學校 |url=http://news.hkheadline.com/dailynews/headline_news_detail_columnist.asp?id=120209 |date=2010年8月20日}}</ref>。學校上軌道後，李施嬅再次重返無綫拍劇。因為受到更多限制，李施嬅已出售「Technique」學校股權<ref>[http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20120219&sec_id=462&subsec_id=830&art_id=16082964 李施嬅退出美容學校 2012年02月19日 蘋果日報]</ref>。
	
2010年10月參演[[梁材遠|梁材遠]]監製劇集《[[紫禁驚雷|紫禁驚雷]]》，飾演卓紫凝一角。2012年在劇集《[[大太監|大太監]]》演出索綽絡·婉（婉太嬪）一角，演技獲好評，更有大部分的網民留言支持其入圍《[[2012年萬千星輝頒獎典禮|2012年萬千星輝頒獎典禮]]》最佳女配角。

其後李施嬅在2014年於《[[寒山潛龍|寒山潛龍]]》首度分飾兩角演技獲好評、李施嬅於2015年在《[[風雲天地|風雲天地]]》演出[[汪明荃|汪明荃]]年輕版。於2016年在《[[純熟意外|純熟意外]]》中，更首度分飾三角，在劇中分别飾演男主角喬文傑（[[吳啟華|吳啟華]] 飾）的前世戀人月牙公主、女主角殷然（[[蔡思貝|蔡思貝]] 飾）的母親莊穎兒及凌若菲，演技獲好評。<ref>{{Cite web |publisher=劇評網|title=李施嬅《纯熟意外》一人出演三个角色|url=http://www.ontvb.com/show-16-7-12941.html |date=2016年6月5日}}</ref> 

2018年在劇集《[[天命|天命]]》飾演皇貴妃鈕祜祿·鈴兒。

2019年在劇集《[[荷里活有個大老千|-{荷里活}-有個大老千]]》飾演60年代的電影經理人瑪利蓮。

目前李施嬅已拍畢劇集《[[金宵大廈|金宵大廈]]》和《[[法證先鋒IV|法證先鋒IV]]》，分別飾演空姐Alex、60年代舞女Coco及法醫聞家希。

加入無綫後，由於人緣甚廣，因此結交了許多好朋友，其中與[[胡杏兒|胡杏兒]]、[[胡定欣|胡定欣]]、[[胡蓓蔚|胡蓓蔚]]、[[姚子羚|姚子羚]]及[[黃智雯|黃智雯]]組成「胡說八道會」<ref>[http://hk.on.cc/hk/bkn/cnt/entertainment/20170730/bkn-20170730104321202-0730_00862_001.html   「胡說八道會」澳洲Trip正式出發]</ref>。

2019年4月初與[[無綫電視|無綫電視]]結束合約。
