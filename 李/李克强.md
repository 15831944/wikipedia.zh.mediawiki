'''李克强'''（{{bd|1955年|7月1日|||Li, Keqiang}}），[[安徽|安徽]][[定远县|定远]][[吳圩鎮_(定遠縣)|吳圩鎮]]高埂村人，[[祖籍|祖籍]][[江西|江西]][[武寧|武寧]]<ref>[http://www.zhmodaoli.com/zh/lishiwenhua/muzonglianyi/wenhua_114.html 安徽定远九梓乡宗亲联谊记_中华磨刀李--永修县磨刀李文化研究会]</ref>，[[中国共产党|中国共产党]]和[[中华人民共和国|中华人民共和国]]主要[[党和国家领导人|领导人]]之一。[[北京大学|北京大学]]法学学士、经济学博士，[[在职研究生|在职研究生]]学历。1976年5月加入[[中国共产党|中国共产党]]，现任[[中华人民共和国国务院总理|中华人民共和国国务院总理]]、[[中国共产党国务院党组|党组书记]]。中共第十五届、十六届、十七届、十八届、十九届中央委员，十七届、十八届、十九届中央政治局委员、常委。第八届全国人大常委会委员。

李克强负责分管[[中央财经委员会|经济]]、[[中央外事工作委员会|外交]]、[[中央国家安全委员会|国家安全]]、[[中央全面深化改革委员会|深化改革]]、[[中央军民融合发展委员会|军民发展]]、[[中央网络安全和信息化委员会|网络信息]]、[[中央全面依法治国委员会|法治]]、[[中央教育工作领导小组|教育]]和[[中央审计委员会|审计]]等工作。此外，他还担任[[中央机构编制委员会|中央机构编制委员会]]主任、[[国家国防动员委员会|国家国防动员委员会]]主任、[[国家能源委员会|国家能源委员会]]主任等职务。

李克强在2017年10月召开的[[中国共产党第十九次全国代表大会|中共十九大]]上，与总书记[[习近平|习近平]]一起以全票当选[[中国共产党第十九届中央委员会委员列表|第十九届中央委员]]<ref>{{cite web |url = https://news.mingpao.com/pns/dailynews/web_tc/article/20171025/s00001/1508868284709 |title = 湘代表：习李全票当选 |quote=十九大代表陳永忠：「很少人得票特別低，'''习近平、李克强全票当选'''，絕大多數的委員都是高票當選。」 |website = 明報新聞網 |date=2017-10-25 }}</ref>，并于10月25日召开的[[中国共产党第十九届中央委员会第一次全体会议|中共十九届一中全会]]上连任中央政治局常委，开始其第三届政治局常委任期，排名继续维持在党内第二位。

==早年经历==
1955年7月1日生于[[安徽|安徽]][[合肥|合肥]]。1972年至1974年在[[合肥八中|合肥八中]]高中部学习。1974年至1976年为安徽省[[凤阳县|凤阳县]]大庙公社东陵大队[[知青|知青]]；1976年至1978年担任安徽省凤阳县大庙公社大庙大队党支部书记。

李克强是1977年中国恢复[[普通高等学校招生全国统一考试|高考]]后第一批[[北京大學|北京大學]][[北京大学法学院|法律系]]的学生，在北大上学期间，師從[[厉以宁|厲以寧]]等名家，曾任北大学生会常代会会长，毕业后在职获得北大[[经济学|经济学]][[博士|博士]]學位。<ref name="李克强足迹图集：北大法律系“黄埔一期”">{{cite web |url=http://news.ifeng.com/photo/special/zhonggongshibada/zuji/detail_2012_11/15/19179596_0.shtml#p=1 |title=李克强足迹图集：北大法律系“黄埔一期” |accessdate=2012-11-24}}</ref>李克强的[[外语|外语]]水平高，<ref name="李克强英语１">{{cite web |url=http://news.takungpao.com.hk/mainland/focus/2013-03/1494278.html |title=李克強英文出色 外國學生讚歎「Very cool」 |work=大公網 |date=2013-03-16}}</ref>能使用流利[[英语|英语]]与[[外国人|外国人]]交流。<ref name="李克强英语2">{{cite web |url=http://std.stheadline.com/daily/article/detail/1909333-中國綜合-李克強說英文為中國拉投資+會場響起熱烈掌聲 |title=李克強說英文為中國拉投資 會場響起熱烈掌聲 |work=星島日報 |date=2018-11-14}}</ref>李克强的夫人[[程虹|程虹]]是一名英语教授，据悉夫妇两人在家中会使用英语交谈。<ref name="李克强英语３">{{cite web |url=http://www.chinanews.com/gn/2014/05-05/6130718.shtml |title=李克强与埃塞总理英语交谈 夫人教授气质获网友好评 |work=中新网 |date=2014-05-05}}</ref>

== 步入政界 ==
李克強1976年5月加入[[中国共产党|中国共产党]]；1982年起先后担任北京大學團委書記、[[中國共青團|中國共青團]]中央書記處書記、[[全國青聯|全國青聯]]副主席。1993年至1998年任共青團中央書記處[[第一书记_(中国)|第一書記]]。1998年6月，任[[中共河南省委|中共河南省委]]副书记；1998年7月，任河南省副省长、代省长；1999年2月，任河南省省长；2002年12月，任中共河南省委书记、省长；2003年1月，任河南省委书记、河南省人大常委会的主任；2004年12月至2007年11月，任[[中共辽宁省委|中共辽宁省委]]书记、省人大常委会主任。

=== 主政河南===
1998年6月，李克强离开北京转任[[河南行政长官列表|河南省省长]]，开始了在河南长达7年的工作，期间河南省的经济总量跃居至全国第5位，居中西部省份之首。基于河南省城镇化落后，导致产业结构升级缓慢，现代农业发展困难重重的现实，2002年12月24日，李克强在河南提出加快工业化、城镇化、推进农业现代化，从而把更多的富余劳动力从土地上解放出来。2003年3月李克强提出“中原崛起”概念，在他的领导和推动下，协调发展大中小城市、规划建设[[郑东新区|郑东新区]]、谋划[[中原城市群|中原城市群]]、振兴小城镇等一系列举措相继展开。<ref>{{cite web |url=http://news.xinhuanet.com/politics/2013-01/29/c_114545497.htm|title=李克强开启河南城镇化发展新路子 |accessdate=2013年1月29日|publisher=新华网|work= |language=zh-hans}}</ref>

[[李长春|李长春]]任职河南期间时“[[河南血祸|血浆经济]]”对部分河南人民造成重大伤害，致使[[艾滋病|艾滋病]]疫情泛滥，<ref>{{cite web |url=http://www.voacantonese.com/content/china-next-pm-on-ngs-and-aids-prevention/1555194.html|title=李克强说防爱滋病民间组织作用不可替代|accessdate=2012-11-29 |publisher=美国之音|work= |language=zh-hans}}</ref>后來李克强担任[[中共河南省委|中共河南省委]]书记兼[[河南省人民代表大会|河南省人大]]常委会主任，组织力量开展了中国首次省级艾滋病情普查，组织省直部门对口帮扶重灾村，并确立了患者“四有一不”的权益机制，使疫情蔓延得以遏制。<ref>{{cite web |url=http://ah.sina.com.cn/news/wltx/2013-03-15/092442764_3.html|title=李克强:遏制了河南艾滋病的蔓延|accessdate=2013年3月15日|publisher=新浪网|work= |language=zh-hans}}</ref>

=== 转任辽宁===
2004年李克强出任[[中共辽宁省委|中共辽宁省委]]书记兼[[辽宁省人民代表大会|辽宁省人大]]常委会主任，解决了辽宁国有企业改革和大量人员“下岗”的难题。李克强鼓励非国营经济的发展，主导棚户区改造<ref>{{cite web |url=http://news.sina.com.cn/c/sd/2013-03-28/103626667580.shtml|title=李克强主政辽宁时曾翻垃圾判断居民生活境况|accessdate=2013年3月28日|publisher=环球人物杂志|work=|language=zh-hans}}</ref>。

== 政治局常委 ==
2007年10月在[[中共十七届一中全会|中共十七届一中全会]]上从中央委员直接晋升为[[中共中央政治局|中共中央政治局]]委员、常委，進入中國最高決策層。

2007年11月被任命为[[中华人民共和国国务院|国务院]]党组副书记，[[中共中央财经领导小组|中共中央财经领导小组]]成员，为2008年[[第十一届全国人大第一次会议|第十一届全国人大第一次会议]]中接任[[中华人民共和国国务院副总理|国务院副总理]]做准备。

2008年3月17日，李克强在十一屆全國人大一次會議第七次全體會議當選為[[中华人民共和国国务院副总理|国务院副总理]]（排名第一）。

李克强担任过中共第15屆至19屆中央委員，17屆至19屆中央政治局委員、常委。第8屆[[全國人大常委會|全國人大常委會]]委員。

2012年11月15日[[中共十八届一中全会|中共十八届一中全会]]上，李克强连任[[中国共产党|中国共产党]]第十八届[[中央政治局常务委员会|中央政治局常务委员会]]委员（排名第二，仅次于总书记[[习近平|习近平]]）。同年，美国《[[福布斯|福布斯]]》杂志[[2012年全球最具权力人物排行榜|2012年全球最具权力人物排行榜]]上，李克强名列13位，此后均入选“[[福布斯杂志全球最具影响力人物列表|全球最具权力人物排行榜]]”头20位。

2017年10月24日，李克强以全票当选[[中国共产党第十九届中央委员会委员列表|中共十九届中央委员]]。<ref>{{Cite web|url=https://news.mingpao.com/pns/dailynews/web_tc/article/20171025/s00001/1508868284709|title=湘代表：習李全票當選|quote=十九大代表陳永忠：「很少人得票特別低，'''習近平、李克強全票當選'''，絕大多數的委員都是高票當選。」|publisher=明報新聞網|date=2017-10-25}}</ref>10月25日的[[中共十九届一中全会|中共十九届一中全会]]上，李克强再次连任中共第十九届中央政治局常委。

2013年3月15日，[[第十二届全国人大第一次会议|第十二届全国人大第一次会议]]上，以2940票（得票率99.7%）当选[[中华人民共和国国务院总理|国务院总理]]，是[[中华人民共和国成立|中华人民共和国成立]]以来第七位总理。3月25日，李克强出任[[中央机构编制委员会|中央机构编制委员会]]主任。<ref>{{cite web|author=1689 |url=http://politics.people.com.cn/n/2013/0325/c1001-20908527.html |title=中央机构编制委员会领导调整 李克强任主任 |publisher=Politics.people.com.cn |date=2013-03-25 |accessdate=2017-05-23}}</ref>8月8日，李克强出任[[国家国防动员委员会|国家国防动员委员会]]主任。<ref>{{cite web|author=作者：来源：中国国防动员网发布时间：2013-08-08 18:46:42 |url=http://www.gfdy.gov.cn/org/2013-08/08/content_4961742.htm |title=国家国防动员委员会 |publisher=Gfdy.gov.cn |date=2013-08-08 |accessdate=2017-05-23}}</ref>

李克强分别在2013年12月和2014年1月的[[中共中央政治局|中共中央政治局]]会议上，出任新成立的[[中共中央全面深化改革领导小组|中共中央全面深化改革领导小组]]副组长和[[中共中央国家安全委员会|中共中央国家安全委员会]]副主席的职务。

2018年3月的[[第十三届全国人民代表大会|第十三届全国人民代表大会]]第一次全体会议上，李克强以2964票连任国务院总理。

===接手困境===
{{参见|李克强经济学|簡政放權}}
[[File:Li_Keqiang,_Chinese_and_foreign_press_conference.jpg|thumb]]后的[[国务院总理记者会|总理中外记者会]]]]
李克强从[[温家宝|温家宝]]手中接掌[[国务院总理|国务院总理]]一职时，他所承接的与温家宝主政十年已不可同日而语，生态问题、资源枯竭、内需不振、产业结构畸形、[[房地产|房地产]]泡沫超大、金融问题不断都是摆在李克强面前的难题<ref>{{cite web|url=http://www.voachinese.com/content/he-qing-lian-20141215/2558984.html |title=何清涟：总理大位纵易主 经济颓势也难挽 |language={{zh icon}} |publisher=Voachinese.com |date=2014-12-15 |accessdate=2017-05-23}}</ref>。另一方面，[[朱镕基|朱镕基]]时代三大改革的财富被彻底耗尽，使得李克强的改革之路将充满坎坷<ref>{{cite web|url=http://i.ifeng.com/news?m=1&aid=48181118|title= 朱镕基时代三大改革财富耗尽|publisher=凤凰卫视|date=2012-11-28| language =zh-hans }}</ref>。

李克强任内提出[[中国制造2025|中国制造2025]]、[[互联网+|互联网+]]、[[大众创业、万众创新|大众创业、万众创新]]、供给侧改革等政策，带动[[中华人民共和国经济|中国经济]]转型。

2018年7月6日，美国总统[[唐納·川普|特朗普]]正式对价值340亿美元的[[中國製造|中国商品]]加征25%[[关税|关税]]。李克强回应[[2018年中美贸易争端|中美贸易战]]时公开表示中国不会主动打[[贸易战|贸易战]]，但如果对方挑起贸易战，中方必将采取相应反制措施。李克强又强调贸易战不会有赢家，只能是损人害己，中国将坚定不移[[深化改革|深化改革]]、扩大开放<ref>{{Cite web|url=http://www.gov.cn/guowuyuan/2018-07/06/content_5304194.htm|title=李克强：打贸易战损人害己|date=2018-07-06|accessdate=2018-07-08|publisher=[[中国政府网|中国政府网]]}}</ref>。

===外交关系===
{{参见|李克强外交访问列表}}
