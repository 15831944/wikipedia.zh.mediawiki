'''李宴芳'''（{{Bd|1910年||1982年|}}），[[表字|字]]'''俊賢'''，[[法名|法名]]'''達明'''<ref>{{Cite web|url=http://www.buddhist-experience.org/welcome/article_detail/1145.html|title=《觀世音菩薩靈感事跡選》大小便阻塞不通，霍然痊癒 文/李宴芳|accessdate=2018-01-11|work=www.buddhist-experience.org}}</ref>，[[諢名|諢名]]「'''拿掉五分'''」或「'''A那美露'''」等，是一位出身臺灣苗栗[[銅鑼鄉_(台灣)|銅鑼]]的[[藝術教育|藝術教育]]家。他曾在[[新竹中學|新竹中學]]長年擔任美術教師，並於退休後投入於[[佛學|佛學]]研究。<ref name=":0">{{Cite web|url=http://www.hcccb.gov.tw/chinese/05tour/tour_f02.asp?titleId=405|title=人物誌|accessdate=2018-01-11|work=www.hcccb.gov.tw}}</ref>

== 生平 ==
1910年，李宴芳出生於[[日治臺灣|日治臺灣]]新竹廳苗栗支廳苗栗一堡[[銅鑼灣庄|銅鑼灣庄]]。1923年，他在畢業於銅鑼[[公學校|公學校]]（今苗栗縣銅鑼國民小學前身）後考入[[臺灣總督府臺北師範學校|臺灣總督府臺北師範學校]]就讀，並於1929年畢業於該校演習科。隨後，他先後在苗栗第一公學校（今[[苗栗縣苗栗市建功國民小學|苗栗縣苗栗市建功國民小學]]前身）、大湖公學校（今苗栗縣大湖國民小學前身）、後龍公學校、四湖國校等處擔任[[教員|教員]]、教頭、[[校長|校長]]等職位。<ref name=":1">{{Cite book|title=竹塹藝術家薪錄|last=蔡國川|first=|publisher=新竹市立文化中心|year=1994|isbn=|location=|pages=}}</ref><ref name=":2">{{Cite journal|title=我的父親－李宴芳|author=李彥毅|url=|journal=新竹中學校友會刊(6)|issue=6|doi=|others=|year=1992|volume=|page=|pmid=}}</ref>

[[戰後台灣|戰後]]，他辭去四湖國校校長<ref>{{Cite web|url=http://www.shihues.mlc.edu.tw/index.php?option=com_content&view=article&id=139&Itemid=1164|title=歷屆校長|accessdate=2018-01-11|last=User|first=Super|work=www.shihues.mlc.edu.tw|language=zh-tw}}</ref>職位，並應聘為私立[[建台中學|建台中學]]美術教師。1948年8月，他轉任[[臺灣省立新竹中學|臺灣省立新竹中學]]美術教師，並於後來兼任該校訓育組長，直至1970年<ref name=":3">{{Cite web|url=http://art.ncu.edu.tw/journal/words/%25E3%2580%258A%25E8%2597%259D%25E8%25A1%2593%25E5%25AD%25B8%25E7%25A0%2594%25E7%25A9%25B6%25E3%2580%258B%25E7%25AC%25AC%25E4%25B9%259D%25E6%259C%259F_%25E7%259B%259B%25E9%258E%25A7.pdf|title=試探日治時期與 1970 年代鄉土運動的巷弄美學 *|accessdate=2018-01-11|last=Administrator|work=webcache.googleusercontent.com|deadurl=yes|archiveurl=https://web.archive.org/web/20150122120521/http://art.ncu.edu.tw/journal/words/%E3%80%8A%E8%97%9D%E8%A1%93%E5%AD%B8%E7%A0%94%E7%A9%B6%E3%80%8B%E7%AC%AC%E4%B9%9D%E6%9C%9F_%E7%9B%9B%E9%8E%A7.pdf|archivedate=2015-01-22}}</ref>自願[[退休|退休]]為止。 <ref name=":1" /><ref name=":2" />

[[晚年|晚年]]，他潛心鑽研[[佛法|佛法]]，並譜撰[[佛教音樂|佛化歌曲]]，又將《釋迦牟尼畫傳》全部上色且製成[[幻燈片|幻燈片]]，攜至各地放映弘法。1976年，他同淨海法師、呂繩安將《佛陀畫傳》<ref>{{Cite web|url=http://buddhism.lib.ntu.edu.tw/DLMBS/author/authorinfo.jsp?ID=%2037085|title=佛學數位圖書館暨博物館 ::: 佛學著者權威資料庫|accessdate=2018-01-11|work=buddhism.lib.ntu.edu.tw}}</ref>在[[美國|美國]]出版。1973年春，他與同好[[捐款|捐款]]興建佛教無量壽圖書館，並親自擔任舘長。<ref name=":2" /><ref name=":1" />

== 藝術風格 ==
* 他的[[水彩畫|水彩畫]]完全採用不透明畫法，其各色[[顏料|顏料]]皆滲用[[白色|白色]]顏料調色。
* 其畫風濃厚鮮麗。
* 其寫生作品必按照實景繪製。<ref name=":0" />

== 教育觀點 ==
* 他認為美術教育的[[目的|目的]]並非在於訓練每一位學生成為畫家，而是在於培養學生獲得文化藝術[[氣質|氣質]]，以使其練習出[[審美|審美]]能力。
* 為培養學生的[[觀察|觀察]]力及[[創造力|創作力]]，他嚴格禁止學生模仿或[[臨摹|臨摹]]他人作品。
* 他在評審學生作品時一律依照「[[構圖|構圖]]」、「線條」、「[[顏色|顏色]]」、「[[明暗对照法|明暗]]」等項目考察。
* 他嚴格要求學生於畫架擺置、汲水倒水、[[圖釘|圖釘]]數量位置、課後收拾等事項，若有違反則以「拿掉五分」作[[懲罰|懲罰]]。
* 若遇學生觸犯[[辛志平|辛志平]]校長時期「考試[[作弊|作弊]]、[[打架鬥毆|打架]]、[[偷竊|偷竊]]者一律[[退學|退學]]」的規定時，他會好言相勸並私下另行處罰。
* 他在每年新竹縣[[運動會|運動會]]舉辦時總是盡情奔放的編唱歌曲，以期激發學生的[[精神|精神]]。<ref name=":0" />

== 軼事 ==
