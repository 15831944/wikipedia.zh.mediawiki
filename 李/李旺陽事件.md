'''李旺陽事件'''是指[[中華人民共和國|中國]][[湖南省|湖南省]][[六四民運|六四民運]]勞工[[維權|維權]]人士[[李旺陽|李旺陽]]在可疑情況下死亡事件。

李旺陽因支持八九民運囚禁累積計達22年。刑滿出獄後，2012年5月22日在朋友幫助下脫離了監視，接受香港[[有線新聞|有線新聞]]中國組記者[[林建誠|林建誠]]訪問相關六四民運。訪問在該年6月2日播出，當地[[公安部國內安全保衛局|公安部國內安全保衛局]]（「國保」）亦加強了對李旺陽的監控。6月6日，胞妹李旺玲到訪發現他已死亡，情況可疑，他也沒有留下遺書。當局6月8日進行了屍檢，報告的結論是自殺。6月9日李旺陽的遺體被當局沒有親人的許可下在邵陽市殯儀館強行火化。他的親戚和朋友質疑李旺陽是「[[被自殺|被自殺]]」，引起國內和外地關注。6月10日，成千上萬的人在香港上街加入參加遊行。6月中旬，因香港和其他的關注，北京下令刑事調查。湖南省委書記周強在2012年7月30日表示李旺陽死於自殺，證據確鑿；各界的人士仍然認為李旺陽的死重重疑點。

從李旺陽的死亡起，妹妹李旺玲和妹夫趙寶珠夫妻卻被嚴密監控。湖南省公安廳公佈李旺陽死因調查報告宣稱他是自殺後，李旺玲夫婦仍是失蹤狀態。記者繼續搜索他們的踪跡，最終九月初，他們被《明報》記者發現。他們接受採訪時堅持從來沒有同意屍檢，火化或自殺的判決，可是在當局的監督下回答問題他們要求「過去的事就不要再提了！」記者調查期間被當地公安在旅館扣留44小時，新聞界對香港記者受到威脅表示不滿。

'''李旺陽'''（1950年11月12日－2012年6月6日），[[中華人民共和國|中華人民共和國]][[湖南省|湖南省]][[邵陽市|邵陽市]][[大祥區|大祥區]]滑石新村人，1966年至1988年先後在邵陽市陶瓷廠、玻璃廠、水泥廠附屬加工廠當工人。李旺陽是[[中國獨立工會|中國獨立工會]]最早的活動分子，中國的工運領袖。李旺陽在1983年就組織了「邵陽市工人互助會」<ref>{{cite web |url=http://www.clb.org.hk/schi/content/前湖南邵阳工自联主席李旺阳开始绝食 |title=前湖南邵陽工自聯主席李旺陽開始絕食 |date=2001年9月20日 |publisher=中国劳工通讯}}</ref>。1989年李旺陽於[[邵陽|邵陽]]組織工人自治參與[[六四事件|六四事件]]，後因「[[反革命|反革命宣傳煽動罪]]」而被判入獄13年（1989年－2000年6月8日），至2001年再被判處[[顛覆國家政權罪|顛覆國家政權罪]]，囚禁10年（2001年5月6日─2011年5月5日），累積計達22年，是六四事件被囚禁時間最長的[[政治犯|政治犯]]之一<ref name="最長政治犯">{{cite news |url = http://cablenews.i-cable.com/webapps/news_video/index.php?news_id=385382|title = 李旺陽是六四後囚禁最長政治犯|publisher = 有線新聞|date = 2012-06-06|accessdate=2012-06-07}}</ref>。2012年6月4日獲[[全美中國學生學者自治聯合會|全美中國學生學者自治聯合會]]頒發自由精神獎<ref name="自由精神獎">{{cite news|url =http://www.rfa.org/mandarin/yataibaodao/64-06042012150248.html |title = 「全美學自聯」主舉紀念六四活動李旺陽獲「自由精神獎」|publisher = 自由亞洲電台|date = 2012-06-04|accessdate=2012-06-07}}</ref>。6月6日早凌晨4点，在[[湖南|湖南]][[邵陽市|邵陽市]]一間醫院被其親屬發現身亡，死因[[被自杀|成疑]]<ref name="自杀说法被质疑">{{cite news|url = http://www.bcc.com.tw/news/newsview.asp?cde=1803684|title = 湖南六四民运工运领袖身亡 自杀说法被质疑|publisher = 中國廣播公司|date = 2012-06-06|accessdate = 2012-06-07}}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，現葬於市內的大山嶺陵園。

=== 第一次入獄 ===
李旺陽是[[湖南|湖南]][[邵陽市|邵陽市]][[玻璃|玻璃]]廠工人，他在[[北京|北京]]時受到[[西單民主牆|西單民主牆]]的薰陶，對[[民主主義|民主主義]]產生共鳴，並於1983年成立工人互助會，創辦《[[資江民報|資江民報]]》。

在[[八九民運|八九民運]]中，他出任邵陽市工自聯主席，聲援學運。當民運被[[中國人民解放軍|中國人民解放軍]]鎮壓後，他舉辦了六四死難者追悼會，後被指「反革命份子」，犯上反革命組織罪，於6月9日被捕，原被判10年監禁，他在庭上抗辯：
{{quote|「[[遊行|遊行]][[示威|示威]]、[[言論自由|言論自由]]是[[中華人民共和國憲法|憲法]]賦予人民的權利，我既沒有罪，也沒有錯。…中國工人已經覺醒了！……這個[[中華人民共和國政府|政府]]已經走到了人民的對立面。」
}}

他最後被加刑至13年，初於邵陽市[[瀧溪監獄|瀧溪監獄]]（湖南省第六監獄）服刑，後改囚[[沅江監獄|沅江監獄]]（湖南省第一監獄），接著往[[岳陽勞改農場|岳陽勞改農場]]勞改<ref name="民運人士資料搜尋">{{cite news|url = http://www.alliance.org.hk/prison/?viewid=1077|title = 民運人士資料搜尋|publisher = 民運人士資料搜尋|date = 2011-09-01|accessdate = 2012-06-07}}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="六四廿三年何時解結--李旺陽專訪">{{cite news |url = |title = 六四廿三年何時解結--李旺陽專訪|publisher = 有線新聞|date = 2012-06-03|accessdate=2012-06-08}}</ref>。

他原在1996年6月因患多種病症而獲[[保外就醫|保外就醫]]，但[[鄧小平|鄧小平]]在1997年逝世，中國當局怕他以此機會號召平反六四，撤銷了他的保外就醫，於2月重囚沅江監獄。服刑期間，他堅持原則，拒絕當權者的說法，受盡[[酷刑|酷刑]]對待，疾病也得不到治療，[[絕食|絕食]]抗議但遭獄方撬掉左側[[牙齒|牙齒]]強迫進食。李旺陽在監獄的情況可由他接受[[香港有線|香港有線]]電視台訪問時可見一斑：「監獄裡面鐵匠打的那種土銬子，比手腕還小，銬不進去，用鉗子來使勁夾，等於是用鉗子在夾骨頭，他使勁的一鉗，我頭就發昏，眼睛就不看見了<ref name="民運人士李旺陽猝死">{{cite news|url=http://www.chinese.rfi.fr/中国/20120606-民运人士李旺阳突然死亡|title = 民運人士李旺陽猝死惹”被自殺”疑雲公安竟搶屍|publisher = 法國國際廣播電台|date = 2012-06-06|accessdate=2012-06-07}}</ref>。」他被囚於闊1米，長2米，高1.6米，有如棺材大小的囚牢不下二十次，囚牢只有一個取飯洞和一個排泄洞，沒有燈光和被褥，內部環境惡劣，虱子、蒼蠅滿佈，他只能或坐或臥，為時最短1個月，最長3個月<ref name="李旺陽困棺材倉企也不能 港今悼念頭七 獄中酷刑曝光">{{cite news|url=http://hk.news.yahoo.com/李旺陽困棺材倉企也不能-港今悼念頭七-獄中酷刑曝光-214744053.html|title = 李旺陽困棺材倉企也不能 港今悼念頭七 獄中酷刑曝光|publisher =《明報》|date = 2012-06-13|accessdate=2012-06-15|archiveurl=http://archive.is/4nWVI|archivedate=2013-01-05}}</ref>。受到如斯虐待，李旺陽於2000年已經患嚴重[[心臟病|心臟病]]、[[甲狀腺亢進|甲狀腺亢進]]、左眼[[失明|失明]]、頸椎及腰椎病、雙耳接近失聰，活動能力大減，因病重而減刑兩年，於該年6月8日出獄<ref name="民運人士資料搜尋"/><ref name="受盡22年酷刑">{{cite news |url = http://www.chinese.rfi.fr/中国/20120603-受尽22年酷刑六四工运人士李旺阳永不言悔|title = 受盡22年酷刑六四工運人士李旺陽永不言悔|publisher = 法國國際廣播電台|date = 2012-06-03| accessdate=2012-06-07}}</ref>。

=== 第二次入獄 ===
李旺陽出獄後，身體不能自理，由其妹妹李旺玲和妹夫趙寶珠夫妻照顧，他仍加入了[[中國民主黨|中國民主黨]]。民主黨人[[秦永敏|秦永敏]]亦呼籲海內外民運人士幫助貧病交迫的李旺陽，他在居住地受到嚴密監控，與他同時期放監的異見人士[[張京生|張京生]]打算探望李旺陽，在路途上被捕，並行政拘留十五日，湖南當局警告所有民運人士，誰探望李旺陽就抓捕誰。在此期間，邵陽市以找不到李旺陽檔案為由，拒發退休養老金，只發300元「低保」<ref name="湖南公民网络论坛">{{cite news |url = http://www.newcenturynews.com/Article/china/201109/20110924104940.html|title = 湖南公民网络论坛：正告邵阳地方政府，立即停止迫害李旺阳|publisher = 新世纪新闻网|date = 2011-09-24|accessdate=2012-06-07}}</ref>。李旺陽的家人亦遭受逼害，他的妹夫趙寶珠被無理解僱，因著他們是重點打擊對象，他們的居所被發展商[[強拆|強行清拆]]，不作賠償，使他們至今仍居無定所<ref name="李旺阳捐款">{{cite news |url = http://www.duping.net/XHC/show.php?bbs=11&post=1133187|title = 第一笔给苦难的中国民主党人李旺阳捐款$1050美元从纽约汇出|publisher = 獨立評論|date = 2011-05-04|accessdate=2012-06-07}}</ref>。

由於多年的虐待，他的身高由1.82米萎縮至1.73米，2001年1月5日向當地政府發信，揚言一週不送他去醫院，便向國際社會尋求人道援助，政府就範，但李旺陽一花光慈善組織捐助醫藥費後，醫院竟又停止治療。李旺陽認為中國政府用酷刑對待他，令至他身體殘障有不可推卸的責任，理應賠償損失，故聯合其他湖南民運人士絕食抗議，共絕食22日，他的妹妹李旺玲接受《[[美國之音|美國之音]]》、《[[自由亞洲電台|自由亞洲電台]]》訪問，被湖南當局判以三年勞改<ref name="结束绝食抗议">{{cite news|url = http://www.china-labour.org.hk/schi/node/1301965/printtitle = 前邵阳工自联领袖李旺阳结束绝食抗议|publisher = 中國勞工通訊|date = 2001-02-23|accessdate = 2012-06-07}}{{Dead link|date=2019年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。5月6日他於大祥醫院再次被刑事拘留，6月7日被捕，四天後被控[[煽動顛覆國家政權罪|煽動顛覆國家政權罪]]，9月11日，邵陽市中級法院將他判處10年徒刑，[[剝奪政治權利|剝奪政治權利]]4年，重囚沅江監獄<ref name="民運人士資料搜尋"/><ref name="籲國際重視">{{cite news |url = http://www.nownews.com/2001/02/04/168-354208.htm|title = 大陸工運領袖李旺陽絕食抗爭「中國人權」籲國際重視|publisher = 今日新聞網|date = 2001-02-04| accessdate=2012-06-07}}</ref>。

=== 釋放和死亡 ===
[[File:李旺阳之死—.jpg|thumb]]
[[File:李旺阳之死二.jpg|thumb]]
2011年5月5日，李旺陽刑滿出獄，他的病情惡化，雙耳已經完全失聰，只能抬回家中，其後轉往大祥醫院接受治療。妹妹李旺玲每天煮兩頓飯，跋涉七公里路去醫院照顧他<ref name="力抗中共　旺陽妹決不火化哥哥"/>。海內外民運人士捐款給他醫病，但銀行戶口卻無理被封，聯繫為李旺陽治療的醫生也因為[[公安|公安]]阻撓而不能成行<ref name="受訪後10公安監控">{{cite news |url = http://hk.apple.nextmedia.com/news/art/20120607/16404649|title = 受訪後10公安監控|publisher = 《蘋果日報》|date = 2012-06-07| accessdate=2012-06-07}}</ref>。

2012年5月22日，在李旺陽朋友幫助下，他脫離了監視，接受香港[[有線新聞台|有線新聞台]]中國組記者[[林建誠|林建誠]]訪問，由於失明失聰，只能靠手掌心寫字才明白，訪問中他表示，希望[[天安門母親運動|天安門母親運動]]的[[丁子霖|丁子霖]]堅持下去<ref name="砍頭我也不回頭">{{cite news |url = http://hk.apple.nextmedia.com/news/art/20120607/16404648|title = 因六四繫獄21年 飽受夾骨酷刑 失明失聰 李旺陽：砍頭我也不回頭|publisher =《蘋果日報》|date =2012-06-07 |accessdate=2012-06-07}}</ref>：
{{cquote|丁子霖教授，是一位偉大的母親，二十多年來她每一天都在為天安門事件的平反而呼號、呼籲、吶喊，不愧為天安門母親的稱號，我希望她能堅持到平反的那一天。六四事件必須平反，死難烈士的靈魂，應該得到安息。<ref name="最長政治犯"/>}}

談到自己因投身民運而被摧殘的身體，他表示不後悔：
{{cquote|國家興亡，匹夫有責，為了國家早日進入民主社會，為了中國早日實現多黨制，我就是砍頭，我也不回頭！<ref name="最長政治犯"/>}}

訪問在2012年6月2日播出，引起海外關注，當地[[公安部國內安全保衛局|公安部國內安全保衛局]]（「國保」）亦加強了對李旺陽的監控。6月4日，李旺陽獲全美中國學生學者自治聯合會頒發自由精神獎<ref name="自由精神獎"/>。

6月6日，李旺玲收到大祥區人民醫院通知李旺陽自殺，當她趕到時，發現他伏屍窗邊，頸項綁着白繩，白繩則綁着窗口，但他的雙腳着地、手搭在窗上，而房間遺物仍維持原狀，他也沒有留下遺書。李旺玲擁屍大哭，他的朋友只來得及拍了兩張半身屍體照片，來不及照全身照片，就被公安趕走。公安進行拍照後，便強行移走屍體<ref name="六四鐵漢">{{cite news|url=http://news.mingpao.com/20120607/gaa1.htm|title=六四鐵漢自縊疑點多  聾盲李旺陽醫院監控離奇死 記者：根本是暗殺 |work=《明報》|date=2012-06-07|accessdate=2012-06-07}}</ref>。

== 死後各界反應 ==
=== 李旺陽親友 ===
趙寶珠接受[[法國國際廣播電台|法國國際廣播電台]]訪問時，質疑李旺陽不是「自殺」，首先發現遺體的李旺陽家屬提供給傳媒的兩幅照片都顯示李旺陽的死亡超出常態，有大量疑點。李旺陽的家屬和朋友家屬在訪問中表示他沒有輕生念頭，有意醫治失聰的耳朵，而且他在獄中22年來堅強對抗當權者的虐待，從不屈​​服，所以他們判定李旺陽是被謀殺<ref name="民運人士李旺陽猝死"/>。

=== 香港政府高級官員 ===
[[食物及衛生局局長|食物及衛生局局長]][[周一嶽|周一嶽]]是首個評論李旺陽之死的[[香港政府高級官員|香港政府高級官員]]，他從[[醫學|醫學]]的角度推斷，表示李旺陽之死有不少疑點：「他是一個嚴重殘疾的人士，嚴重殘疾的人士要想自殺都不是那麼容易。」他又推斷：「如果任何人為了犧牲自己做一些驚天動地的事，他一定會留下一番他想說的話。」。

候任香港行政長官[[梁振英|梁振英]]表示對李旺陽之死有個人感受，但不作公開表態<ref name="梁振英重申不公開談李旺陽">{{cite news | url=http://hk.news.yahoo.com/%E6%A2%81%E6%8C%AF%E8%8B%B1%E9%87%8D%E7%94%B3%E4%B8%8D%E5%85%AC%E9%96%8B%E8%AB%87%E6%9D%8E%E6%97%BA%E9%99%BD-094207728.html| title=梁振英重申不公開談李旺陽 | work=《明報》 | date=2012-06-10 | accessdate=2012-06-11|archiveurl=http://archive.is/bz3QN|archivedate=2013-01-05}}</ref>，但他出席[[街工|街工]]論壇時，與同場人士一同為李旺陽默哀<ref name="響應呼籲為李旺陽默哀">{{cite news | url=http://paper.wenweipo.com/2012/06/17/HK1206170020.htm| title=響應呼籲為李旺陽默哀 | work=《文匯報》 | date=2012-06-17 | accessdate=2012-06-22}}</ref>，但在之後的一天表示自己只為死人默哀<ref name="梁振英拒認默哀 程翔斥聽命中共">{{cite news | url=http://www.ntdtv.com/xtr/b5/2012/06/19/a719411.html.-%E6%A2%81%E6%8C%AF%E8%8B%B1%E6%8B%92%E8%AA%8D%E9%BB%98%E5%93%80%20%E7%A8%8B%E7%BF%94%E6%96%A5%E8%81%BD%E5%91%BD%E4%B8%AD%E5%85%B1.html| title=梁振英拒認默哀 程翔斥聽命中共 | work=新唐人電台 | date=2012-06-19 | accessdate=2012-06-22}}</ref>。

[[保安局局長|保安局局長]][[李少光|李少光]]表示：「事件不離事實，終會水落石出。」他理解香港人對李旺陽之死的感受<ref name="周一嶽：李旺陽殘疾難自殺">{{cite news | url=http://inews.mingpao.com/htm/Inews/20120612/gb21444k.htm| title=周一嶽：李旺陽殘疾難自殺 | work=《明報》 | date=2012-06-14 | accessdate=2012-06-15}}</ref>。[[香港行政長官|香港行政長官]][[曾蔭權|曾蔭權]]在[[立法會|立法會]]問答大會中表示認同李旺陽之死有疑點，已經向中央政府反映<ref name="香港特首曾蔭權質疑李旺陽死因">{{cite news | url=http://www.bbc.co.uk/zhongwen/trad/chinese_news/2012/06/120614_hongkong_liwangyang.shtml| title=香港特首曾蔭權質疑李旺陽死因 | work=BBC中文網 | date=2012-06-14 | accessdate=2012-06-15}}</ref>。[[教育局|教育局]]局長[[孫明揚|孫明揚]]則表示，綁著李旺陽的繩索方向有問題，他的死有很多疑點<ref name="孫明揚：李旺陽事件疑點好多">{{cite news | url=http://www.rthk.org.hk/rthk/news/expressnews/20120622/news_20120622_55_849770.htm| title=孫明揚：李旺陽事件疑點好多 | work=香港電台 | date=2012-06-22 | accessdate=2012-06-22}}</ref>。

==後續相關事件==
最早發現李旺陽疑「自殺」的李旺陽好友[[朱承志|朱承志]]，因將在現場拍攝的圖片和視頻發布互聯網，質疑官方調查結果，於李旺陽遺體火化當天被警方帶走，要求簽署不再關注李旺陽死因的保證書。由於拒絕簽字，他被以「擾亂社會治安」的罪名行政拘留十日。6月18日，朱承志拘留期滿，當天再遭邵陽市公安当局以「問題還沒查清楚」為由拘留，關押在市看守所。7月25日被公安拘捕，目前羈押在邵陽市看守所。逮捕通知書上有邵陽市公安局印章，但並無簽署辦案人名字。當局仍然禁止他與家人見面及由家屬自聘律師。朱承志被以“煽動顛覆國家政權罪”逮捕。<ref>{{Cite web |url=http://news.sina.com.hk/news/1662/3/1/2732322/1.html |title=李旺陽好友「煽顛罪」被捕 |access-date=2012-08-24 |archive-url=https://archive.is/20130107180213/http://news.sina.com.hk/news/1662/3/1/2732322/1.html |archive-date=2013-01-07 |dead-url=yes }}</ref>

6月20日，維權人士肖勇，也因為關注李旺陽事件被判勞教一年半。

时任中共湖南省委書記的[[周強|周強]]7月底首次回應李旺陽事件，強調湖南沒有人質疑死因，李旺陽家屬接受「自殺結論」，周强说：“湖南公民中没有人对这个事件提出过任何质疑”。<ref>[http://news.sina.com.hk/news/1662/3/1/2732322/1.html周強：湖南無人質疑李旺陽死因]{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="fxlt "/>

香港《[[明報|明報]]》兩名記者9月初到邵陽訪問其失踪數週的妹妹李旺玲和妹夫趙寶珠。夫婦两人證實在李旺陽的死因調查過程中，從未參與驗屍報告、解剖報告等，也沒有在判定為「自殺」的驗屍報導上簽字同意。趙寶珠還說他沒有聽說過「聯合調查報告」的結論。報導引述過後趙寶珠在監視下回答記者有關事件的提問，神色緊張僅地反复說“過去的事就不要再提了”。<ref>[https://www.webcitation.org/6Ac9FJuFl?url=http://www.881903.com/Page/ZH-TW/newsdetail.aspx?ItemId=557731 港媒：李旺阳家人从未同意自杀论]. 英國廣播公司, 2012-09-12</ref> 《明報》記者調查期間被當地公安在旅館扣留44小時。受到嚴密監控，熟睡裡也多次被吵醒盤問。新聞行政人員協會主席趙應春說是非常過分及罕有，認為「情況嚴重並且很離譜」。<ref>[http://www.webcitation.org/6Ac9FJuFl 趙應春指湖南公安禁錮港記者非常過分]. 商業電台, 2012-09-12</ref>《明報》表示此行為「嚴重侵犯記者的人身安全和採訪自由，嚴重損害香港的新聞自由，本報強烈抗議並予以譴責」<ref>[https://www.webcitation.org/6Ads9q1Jv?url=http://news.mingpao.com/20120913/gaa2.htm 明報聲明]. 明報, 2012-09-13</ref> 工聯會新當選的立法會議員[[陳婉嫻|陳婉嫻]]表示死因有需要進一步調查；[[劉健儀|劉健儀]]再次去信國務院總理[[溫家寶|溫家寶]]，要求中央政府介入，確保新聞工作者採訪自由，把案件查到水落石出。<ref>[https://www.webcitation.org/6AdsaMVuu?url=http://news.mingpao.com/20120913/gak1.htm 劉健儀再函溫總 促重查死因]. 明報, 2012-09-13</ref>

《明報》兩名記者9月5日到邵陽訪問其失踪數週的妹妹李旺玲和妹夫趙寶珠之后，他夫妇被加大监控以及再次失踪，直到年前得以获得短暂自由，2013年春节过后又再次失踪（被监控在旅馆以及被旅游）直到2013年7月4日仍然了无音讯，由于关注李旺陽事件大概有几十上百人次的邵阳公民被监控在旅馆或家里，被喝茶，被旅游等。上述非法事件以李旺陽事件后的两个月内，9月5日之后到[[中共十八大|中共十八大]]开完会左右，從2013年5月底到6月10日左右三个时间段最为集中。2013年6月2日，两个衡阳网友黄勇华、魏贤力在邵阳祭拜铁汉李旺陽被拘留。
