'''李可染'''（{{bd|1907年||1989年|catIdx=Li,李}}），[[江蘇|江蘇]][[徐州|徐州]]人，中國當代著名畫家。以创作[[山水画|山水画]]为主。

== 創作概要 ==
李可染以[[山水畫|山水畫]]聞名於藝術界，中國寫意[[人物畫|人物畫]]亦有相當研究，其风格影响了颇多后来者，诸如[[刘大为|刘大为]]、[[陈玉先|陈玉先]]、[[冯远|冯远]]等人。李喜歡畫牛，也是[[書法家|書法家]]。

以下節錄1986年所舉辦的《李可染中國畫展》之前言，李可染總結其一生創作：
{{cquote|
我不依靠什麼天才，我是困而知之，我是一個苦學派。<br />假如我的作品有點成就的話，那是我深入學習傳統、深入觀察描寫對象、深入思考、深入實踐的結果。人離開大自然、離開傳統不可能有任何創造。 我常常問自己，我是在創作畫，還是在學畫、研究畫  ？結論：我是在學畫和研究畫，我現在已經70歲了。我一輩子都在學習和研究的過程中。……
}}

== 創作生平 ==

其山水畫代表作有《[[漓江勝境|漓江勝境]]》與《[[萬山红遍|萬山红遍]]》等。

: 1932年李可染25歲時，舉行第一次個人畫展。創作《[[鍾馗|鍾馗]]》，於1937年入選南京第二屆[[全國美展|全國美展]]。
: 1937年到1943年，對日抗戰爆發，創作抗戰作品，於[[武漢|武漢]]加入政府組織創作抗日愛國[[宣傳畫|宣傳畫]]。期間較受注意的宣傳作品如《是誰破壞了你快樂的家園！》……等，多存在報紙等大量複製品，其餘牆壁上所作僅有底片留存。<ref>[http://news3.xinhuanet.com/collection/2004-12/15/content_2335098.htm 邹佩珠女士将自己珍藏的32件李可染抗日宣传画的底片及大量复制品捐赠给沈阳[[张氏帅府博物馆|张氏帅府博物馆]]。] {{webarchive|url=https://web.archive.org/web/20160304193248/http://news3.xinhuanet.com/collection/2004-12/15/content_2335098.htm |date=2016-03-04 }}来源：北京娱乐信报</ref>

: 1939年武漢危急，李可染輾轉經長沙、貴陽取道入重慶。
: 1942年於[[重慶|重慶]]致力創作，參加[[當代畫家聯展|當代畫家聯展]]，發表《屈原》、《王羲之》、《風雨歸牧》、《牧童遙指杏花村》等，《牧童遙指杏花村》為[[徐悲鴻|徐悲鴻]]訂購。李可染在重慶居所靠近牛棚，此後有更多畫牛的主題。
: 1944年在重慶中蘇友好協會舉辦國畫個展。

: 2007年5月3日《萬山紅遍》在香港佳士得“中國近現代書畫”及“中國古代書畫”春季拍賣以逾三千五百萬港元高價成交，打破畫家作品最高成交價的世界紀錄。

: 2012年1月7日中國現代著名畫家李可染的紅色經典名作《革命聖地韶山》在南京經典拍賣有限公司的2011秋季藝術品拍賣會上，以6842.5萬元的高價成交

: 2012年5月12日中國現代著名畫家李可染的紅色經典名作《韶山》在中國嘉德2012春拍以1.242億元人民幣的天價成交，刷新了其作品拍賣紀錄

: 2012年6月3日中國現代著名畫家李可染的紅色經典名作《萬山紅遍》在北京保利國際拍賣有限公司2012春季拍賣會「中國近現代書畫夜場」上，以2.9億元（人民幣．下同）成交，刷新今年春拍成交紀錄，成為迄今中國藝術品拍賣史上最高價的藝術品
