'''李庆洲'''（{{bd|？|||}}）籍贯不详，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员。

==生平==
1980年代，李庆洲任[[国务院台湾事务办公室|中央对台工作领导小组办公室]]副主任。1988年参与筹建[[国务院台湾事务办公室|国务院台湾事务办公室]]<ref name=tang>{{Cite web |url = http://gov.eastday.com/node2/shds/n218/u1ai25504_K254.html |title =海峡两岸关系协会成立前后  |author = 唐树备 |date =  |publisher =中共党史研究2017(4):72-87  |language =  |accessdate =  |quote =  }}</ref>。1991年3月，[[中央对台工作领导小组|中央对台工作领导小组]]与国务院台湾事务办公室合并为[[中共中央台湾工作办公室|中共中央台湾工作办公室]]、[[国务院台湾事务办公室|国务院台湾事务办公室]]（[[一个机构两块牌子|一个机构两块牌子]]），李庆洲任办公室主任助理<ref>组委会代表、国台办主任助理李庆洲在“汉字文化学术研讨会”开幕式上的讲话，统一论坛1994(1):6-7</ref>。同时担任办公室发言人<ref>{{Cite web |url = http://www.huaxia.com/lasd/stfzzc/ztb/2003/06/474210.html |title =国务院台办发言人李庆洲就台湾当局通过"两岸关系条例"发表的谈话  |author =  |date = 2003-06-09 |publisher = 华夏经纬网 |language =  |accessdate =  |quote =  }}</ref>。

后来，李庆洲调任[[福建省人民政府|福建省人民政府]]省长助理、[[福建省人民政府外事办公室|福建省人民政府外事办公室]]主任。1998年5月，[[中共福建省委|中共福建省委]]调整中共福建省委外事工作领导小组成员，由省委副书记、省长[[贺国强|贺国强]]任组长，省委副书记[[何少川|何少川]]任副组长，小组成员有李庆洲（省长助理、省人民政府外事办公室主任）等人，领导小组办公室主任由李庆洲兼任<ref>{{Cite web |url =http://www.fjfao.gov.cn/ztzl/wsz/jgytt/201210/t20121031_1180158.htm  |title =涉外机构（中华人民共和国成立后）  |author =  |date = 2010-11-22 |publisher = 福建省人民政府外事办公室 |language =  |accessdate =  |quote =  }}</ref>。2005年7月8日，福建省人民政府任命[[宋克宁|宋克宁]]为福建省人民政府外事办公室主任，免去李庆洲的福建省人民政府外事办公室主任职务<ref>福建省人民政府关于宋克宁等职务任免的通知，福建省人民政府公报2005年23期</ref>。此后，历任福建省翻译工作者协会会长<ref>创设译界交流平台 书写翻译事业新篇，福建译协简讯2006年7月</ref>、[[厦门市|厦门市]]国际友好联络会会长<ref>{{Cite web |url = http://www.xaifc.org/a/jiaoliu/huodong/20131114/120.html |title =  李庆洲会长会见日本冲绳县日中友好协会代表团|author =  |date = 2013-11-14 |publisher = 厦门市国际友好联络会 |language =  |accessdate =  |quote =  }}</ref>等职务。
