《'''李冰传奇'''》，中国大陆的古装剧，导演沈燕林，主演刘奕君、王力可、鲍国安、王姬、马德钟。<ref>{{cite web |url=http://news.163.com/11/0520/06/74FR020E00014AED.html |title= 《传奇》热拍鲍国安王姬斗戏|accessdate=2011 |publisher=网易 |work= |language=zh-hans}}</ref>2011年7月杀青，2015年11月19日[[韩国|韩国]]CHING首播<ref>{{cite web |url=http://news.zum.com/articles/26604312?c=06 |title=도강언 건설의 미스터리? '치수의 神 이빙전기' 19일 첫방|date=2015-11-17 |publisher=http://news.zum.com/ |work= |language=ko}}</ref>。

==剧情==
该剧讲述[[春秋|春秋]][[战国|战国]]时期[[秦国|秦国]]李冰治理蜀郡[[都江堰|都江堰]]的历史故事。<ref>{{cite web |url= http://news.163.com/11/0105/04/6PK1IMPD00014AED.html|title=《传奇》将再现“李冰治水” |accessdate=2011 |publisher=网易 |work= |language=zh-hans}}</ref>
