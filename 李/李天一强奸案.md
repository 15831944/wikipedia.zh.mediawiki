'''李天一强奸案'''是指[[李双江|李双江]]之子[[李冠丰|李冠丰]]（旧名李天一）被曝其在2013年2月17日晚伙同狱友涉嫌轮奸的案件。

根据中國大陆相关法律<ref>[http://www.gov.cn/banshi/2005-05/26/content_982.htm 《中华人民共和国未成年人保护法》第42条] 中国政府门户网站</ref>对于未成年嫌疑人的规定，中國大陆媒体一般以“李某某”进行报道此事件。

== 事件 ==
2013年2月22日下午，[[北京市公安局|北京市公安局]]海淀分局就此事发布官方回应：“2013年2月19日，警方接到一女事主报警称，2月17日晚，她在[[海淀区|海淀区]]一酒吧内与李某等人喝酒后，被带至一宾馆内[[轮奸|轮奸]]。接警后，警方于2月20日将涉案人员李某等五人抓获，现五人因涉嫌强奸罪被[[刑事拘留|刑事拘留]]。”<ref>[http://cn.apdnews.com/news/22136.html 李双江之子李冠丰涉嫌轮奸案被刑拘]，亚太日报，2013年3月12日</ref>　据称李冠丰（李天一）已于2013年2月22日被北京市公安局海淀分局以涉嫌轮奸刑事拘留<ref>{{cite web|url=http://ent.sina.com.cn/s/m/2013-02-22/16553862131.shtml |title=传李冠丰轮奸被拘 律师:或判十年以上 |publisher=Ent.sina.com.cn |date= |accessdate=2013-02-22}}</ref>。

2013年2月17日晚，该案发生的第一个地点为北京市海淀区[[五道口站|五道口站]]附近成府路35号东源大厦B1楼GLOBAL[[酒吧|酒吧]]。午夜时分，该名杨姓女子与李冠丰、魏某等5人饮酒。凌晨3时36分左右，该女子与5人以及酒吧领班张光耀离开酒吧，至凌晨4时21分左右进入金鼎轩饭店吃夜宵。凌晨5点多，一行人欲进入海淀区的京国奥宾馆开房；20分钟后，一行人离开京国奥宾馆。5时58分，一行人进入湖北大厦酒店大堂。7时29分左右，一行人离开湖北大厦。

2013年2月19日，该女子向北京市海淀区警方报案。该女子报案时声称自己喝酒醉了之后，后来遭到5人的言语恐吓、暴力殴打，被强行带上小车到“湖北大厦”宾馆开房，其房间是550元一晚的高级标准间，该女子被5人轮奸<ref name=" fish" >{{cite web |url=http://news.163.com/13/0223/15/8ODJ7LJN00011229.html |title=李冠丰被抓细节:便衣围堵豪车 抓获3男1女 |accessdate= 2013-02-23|publisher= 网易|work= |language=zh-cn}}</ref><ref name="bull " >{{cite web |url= http://news.qq.com/a/20130223/000644.htm|title=李双江之子被抓细节：便衣围堵豪车 抓获3男1女 |accessdate=2013-02-23 |publisher=腾讯网 |work= |language=zh-cn}}</ref>。

警方接到报案之后，迅速采取行动调查取证逮捕犯罪嫌疑人<ref name=" fish" /><ref name="bull " />。2013年2月21日，北京市海淀区警方穿着便衣在顺景温泉酒店停车场围堵李冠丰等人的豪车，最终逮捕李冠丰等5名犯罪嫌疑人<ref name=" fish" /><ref name="bull " />。

== 质疑 ==
=== 年龄问题 ===
刑法专家张蕴章认为，假若李冠丰年龄是17岁，他将受到最高刑罚处罚是10年[[有期徒刑|有期徒刑]]；假若李冠丰是成年人，最低的刑法处罚是10年有期徒刑，情节严重可以判处[[无期徒刑|无期徒刑]]或者[[死刑|死刑]]。<ref name=" bee" >{{cite web |url=http://www.bbc.co.uk/zhongwen/simp/china/2013/03/130312_li_true_age.shtml |title=港媒质疑李双江之子李冠丰真实年龄 |accessdate=2013年3月12日 |publisher=BBC中文网 |work= |language=zh-cn}}</ref>

对于李冠丰的年龄，有网友在微博上提出质疑，因为网上传言2006年初李双江、梦鸽参加[[鲁豫|鲁豫]]《[[鲁豫有约|鲁豫有约]]》时说当时是14岁，2013年时李冠丰应已成年。但随后即有网友从《鲁豫有约》官方网站的视频纪录、舞台布置、李双江所着的07版军装等证据推断出该集节目播出的时间不可能是2006年、而是2011年，恰恰说明2013年李冠丰尚未成年。<ref>{{cite web |url=http://news.163.com/13/0224/11/8OFMPHTC00014JB5.html |title= 网友爆料李冠丰年龄造假待求证|accessdate=2013-02-23 |publisher=网易|work= |language=zh-cn}}</ref><ref>{{cite web |url=http://ent.ifeng.com/idolnews/special/lgf/content-6/detail_2013_02/24/22432926_0.shtml |title= 网友根据电视节目推测李冠丰年龄造假|accessdate=2013年2月24日 |publisher=凤凰网 |work= |language=zh-cn}}</ref>
