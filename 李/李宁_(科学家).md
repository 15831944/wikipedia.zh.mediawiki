'''李宁'''（{{bd|1962年|7月}}），[[江西|江西]][[南昌|南昌]]人。动物分子遗传育种学专家。[[中国农业大学|中国农业大学]]“[[长江学者奖励计划|长江学者奖励计划]]”特聘教授。1982年毕业于[[江西农业大学|江西农业大学]]牧医系。1992年获[[中国农业大学|中国农业大学]]与[[都柏林大学|都柏林大学]]联合培养博士学位。2007年当选为[[中国工程院|中国工程院]]农业学部院士<ref>{{cite web|url=http://www.cae.cn/cae/jsp/jump.jsp?oid=20111231115423640264270|title=李宁|accessdate=2015-12-25 |publisher=中国工程院 |language=zh-hans }}</ref>。主要研究方向为动物[[基因组学|基因组学]]、体细胞克隆动物及转基因动物及动物乳腺生物反应器。其岳父[[吴常信|吴常信]]亦是[[中国农业大学|中国农业大学]]教授，1995年当选为[[中国科学院|中国科学院]]生命科学和医学学部院士<ref>{{cite news|title=中国工程院院士李宁被查 其岳父：过不能抹杀功|url=http://news.sohu.com/20140828/n403860788.shtml|accessdate=2015-12-25|newspaper=搜狐新闻}}</ref>。

== 教育经历 ==
1978年7月，考入江西共产主义劳动大学总校（现[[江西农业大学|江西农业大学]]），就读牧医系畜牧兽医班。1982年7月，大学毕业。1985年9月，李宁考取了[[北京农业大学|北京农业大学]]（现[[中国农业大学|中国农业大学]]）畜牧系研究生。1988年7月，获动物遗传育种硕士学位。1988年9月至1990年7月，在爱尔兰[[都柏林圣三一大学|都柏林圣三一大学]]遗传系攻读博士，获都柏林大学与中国农业大学联合培养博士学位。1990年8月至1991年12月，在北京农业大学畜牧系学习，获分子遗传学博士学位。

== 工作经历 ==
1982年7月至1985年8月，江西农业大学牧医系任助教。1992年2月至1993年7月，在西德FAL小动物研究所从事博士后工作。1993年8月至1995年6月，北京农业大学生物学院任讲师、副教授。1995年7月至1995年9月，美国[[明尼苏达大学|明尼苏达大学]]任访问副教授。1995年10月至1996年1月，日本农业水产尖端技术研究所（{{lang|ja|農林水産先端技術研究所}}）任特邀研究员。1996年6月至1996年7月，德国慕尼黑、柏林、哥廷根大学访问学者。1997年5月至1997年6月，英国[[剑桥大学|剑桥大学]]、[[牛津大学|牛津大学]]任访问学者。2001年9月至2001年10月，美国加利福尼亚大学任访问学者。

== 奖项 ==
* 2000年，获国家科学技术进步二等奖。
* 2001年，获国家科学技术进步二等奖。
* 2003年，获国家技术发明奖二等奖。
* 2005年，获北京市科学技术一等奖。
* 2006年，获教育部自然科学奖一等奖。
* 2007年，当选中国工程院院士。

== 隔离审查 ==
2014年6月，因涉嫌将一个转基因项目的资金转移到自己的控股公司，涉及金额达千万元，现正接受组织调查<ref>{{cite news|title=工程院院士李宁被调查 反腐之风刮进科学界|url=http://news.sina.com.cn/c/2014-08-21/210630724144.shtml|accessdate=2015-12-25|newspaper=新浪新闻}}</ref>。根据相关规定，其院士资格被停止，[[中国工程院|中国工程院]]建院20多年来，该院尚无撤销院士称号的先例<ref>{{cite news|title=中国农大教授李宁被停止院士资格|url=http://politics.people.com.cn/n/2015/0118/c70731-26403806.html|accessdate=2015-12-25|newspaper=人民网}}</ref>。
李宁院士被关押已有近四年，却始终没有开庭审理，没有结论。
