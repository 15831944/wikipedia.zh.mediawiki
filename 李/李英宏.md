'''李英宏''' （{{bd|1986年|7月6日||}}），藝名'''DJ Didilong'''，[[臺灣|臺灣]][[饒舌|饒舌]]歌手，生於臺北市，畢業於[[國立台灣藝術大學|國立台灣藝術大學]]廣電系，2016年5月發行首張個人[[專輯|專輯]]《[[台北直直撞|台北直直撞]]》，並於9月30日在ATT Show Box舉辦第一次個人演唱會。<ref>{{Cite web|url = http://www.cna.com.tw/news/amov/201609180031-1.aspx|title = myMusic主辦 李英宏演唱會月底登場 |accessdate = 2016-09-18 |author = 中央社記者江明晏 |publisher = 中央社}}</ref>

== 簡介 ==
17歲加入當時最大的本土唱片公司[[豐華唱片|豐華唱片]]，成為[[張小燕|張小燕]]公司旗下一員， 曾經發行《[[大囍門|大囍門]]》同名專輯。剛出道時以快嘴饒舌著稱，在團體重組後沉澱了一段時間，後轉簽以饒舌音樂為主體的新興唱片公司[[顏社|顏社]]。 

在饒舌歌手[[蛋堡_(音樂人)|蛋堡]]的第二張專輯《[[Winter_Sweet|Winter Sweet]]》中，合作《夜聊（你說，我說）》一曲，在Youtube上擁有超過30萬的點閱率，陸續也推出了《[[古錐仔|古錐仔]]》《[[小心肝|小心肝]]》等充滿個人特色的音樂作品。2014年在饒舌團體LBC(喇叭嘴愛裝熟)的《從之前到現在》專輯中與他們合作三首跨刀作品：《保力達Break》《變了不多》、《我想和妳跳貼舞》。
