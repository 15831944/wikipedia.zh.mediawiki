'''李仁記'''（{{Bd|1919年||2001年|2月8日|catIdx=}}<ref name=":0">{{Cite news|url=http://mypaper.pchome.com.tw/lttoday/post/187071|title=尪姨李仁記辭世 平埔夜祭失舵手|author=陳俊昇|date=2001-02-09|work=|accessdate=}}</ref>），是一位臺灣臺南東山[[吉貝耍|吉貝耍]]的[[尪姨_(原住民族)|尪姨]]。

== 生平 ==
1919年，李仁記[[出生|出生]]於吉貝耍部落。<ref name=":1">{{Cite journal|title=西拉雅族  吉貝耍尪姨李仁記|author=國立臺灣歷史博物館|url=http://mocfile.moc.gov.tw/htmlfile/201704/e551b50f-c3d1-4ef5-8ea7-af5c8a83bd3c/watchtw_tw17/edm_c_p08.html|journal=觀‧臺灣|issue=17|doi=|others=|year=2013|volume=|page=|pmid=}}</ref>

1936年，李仁記被「[[阿立母|阿立母]]」選為其屬下。<ref name=":1" />

1937年起，阿立母連續[[顯靈|顯靈]]兩年多；期間，李仁記只靠[[喝水|喝水]]維持[[身體|身體]]正常運作。<ref name=":1" />

1940年，李仁記與駱易記[[結婚|結婚]]；不久，部落的尪姨過世，李仁記開始接手其工作。此後，李仁記在離家遠行前皆須「[[擲筶|擲筶]]」請示阿立母同意，並於每月[[朔日|初一]]及十五打掃大[[公廨_(原住民族)|公廨]]，為阿立母的壺瓶及向缸換水、換青，然後獻上[[檳榔|檳榔]]，再行「三向禮」。平日，李仁記也待在住處為[[信眾|信眾]]進行[[收驚|收驚]]儀式，或以「謝籃」奉請祀瓶（阿立母的[[化身|金身]]）前往各地為信眾「辦事」，以解決其疑難雜症。<ref>{{Cite news|url=http://blog.roodo.com/tust/archives/1712433.html|title=吉貝耍的阿嬤．尪姨李仁記@筆耕南瀛大地|newspaper=樂多日誌|accessdate=2017-11-03}}</ref><ref>{{Cite web|url=http://webcache.googleusercontent.com/search?q=cache:J7j-WF5jq6IJ:www.vivianwu.org.tw/02c.php?ser=32&ser1=6+&cd=1&hl=zh-TW&ct=clnk&gl=tw|title=財團法人吳舜文新聞獎助基金會|accessdate=2017-11-03|work=webcache.googleusercontent.com}}</ref>

1959年，李仁記在生下第八個[[子女|孩子]]後成為[[全職|全職]]尪姨。<ref name=":1" />

[[晚年|晚年]]，李仁記仍持續不休於吉貝耍主持[[阿立祖#.E5.A3.BA.E7.9A.84.E4.BF.A1.E4.BB.B0.E5.8F.A2.E7.B5.90|夜祭]]及哮海祭（[[嚎海祭|嚎海祭]]）等重要祭典。<ref name=":1" />

2001年2月8日16時，李仁記因[[心肌梗死|心臟麻痺]]過世。<ref name=":1" /><ref>{{Cite book|title=南瀛平埔誌|last=劉還月|first=|publisher=台南縣文化局|year=2005|isbn=9789570033588|location=|pages=}}</ref>

== 家庭 ==
李仁記與駱易記結婚並育有三子五女。<ref name=":0" />
