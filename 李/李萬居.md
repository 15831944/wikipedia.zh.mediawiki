'''李萬居'''（{{bd|1901年|7月22日|1966年|4月9日|catIdx=Li}}），字'''孟南'''，台灣[[雲林縣|雲林縣]][[口湖鄉|口湖鄉]]梧北村人，[[法國|法國]][[巴黎大學|巴黎大學]]畢業，[[中國青年黨|中國青年黨]]資深黨員。他是知名的報人與政治家，與[[郭雨新|郭雨新]]、[[許世賢|許世賢]]、[[郭國基|郭國基]]、[[吳三連|吳三連]]、[[李源棧|李源棧]]等人並稱[[臺灣省議會|臺灣省議會]]的「五龍一鳳」，有'''魯莽書生'''的稱號。兒子是曾任[[香港中文大學政治與行政學系|香港中文大學政治與行政學系]]系主任的[[李南雄|李南雄]]。

== 出身與中法留學 ==
李萬居出身於貧農家庭，九歲時喪父，母親又由於[[日本|日本]]保警的不時催租而自縊身亡，使得李萬居一生都有[[反日|反日]]的思想。李萬居早年受業漢塾，後前往中國大陸求學，就讀於[[上海國民大學|上海國民大學]]，後與[[鄧小平|鄧小平]]等人一同響應[[勤工儉學|勤工儉學]]的時代大潮，前往[[法國|法國]]留學。並且於在法國留學時加入了以[[民族主義|民族主義]]為號召的[[中國青年黨|中國青年黨]]，與[[曾琦|曾琦]]、[[李璜|李璜]]等人相熟。

李萬居畢業於法國[[巴黎大學|巴黎大學]]社會系後，首先於[[上海|上海]]任教於江南學院，後因為日中戰爭之故而轉往[[香港|香港]]，後又從[[廣東|廣東]]、[[廣西|廣西]]等道路前往[[重慶|重慶]]，投奔[[國民政府|國民政府]]。在重慶的四年間，李萬居在軍事委員會國際問題研究所主任，研究戰時日本的戰略與外交，並且常常發表相關社評；除此之外李萬居也協助組織台灣同盟會，為光復台灣進行組織。

== 參政與公論報社 ==
1945年9月，李萬居隨[[台灣省行政長官公署|台灣省行政長官公署]]赴台灣進行接收，他拒絕接收銀行，而出任《[[台灣新生報|台灣新生報]]》社長，後出任董事長；李萬居在[[二二八事件|二二八事件]]中被視為[[半唐山|半唐山]]，險成為受難者。二二八之後，李萬居1947年與[[陳其昌|陳其昌]]等自辦《[[公論報|公論報]]》，揭示「民主」、「自由」、「進步」的理念，使得公論報有「台灣《[[大公報|大公報]]》」的美稱。

李萬居除了是《公論報》負責人外，也從1946年起當選第一屆台灣省參議會議員；1953年台灣省議會成立後，又連續當選四屆台灣省議員，為[[中國青年黨|中國青年黨]]於雲林縣打下深厚基礎。而李萬居在議壇上言人所不敢言，在報紙上報導有關官方缺失，雖然讓李萬居成為與[[郭雨新|郭雨新]]、[[許世賢|許世賢]]、[[郭國基|郭國基]]、[[吳三連|吳三連]]、[[李源棧|李源棧]]譽為「五龍一鳳」的省議員，但也使得李萬居一直都是官方鎖定的對象，其於[[台北市|台北市]]康定路的居所甚至於被一把無名火燒毀。而各大官方單位也從不在《公論報》上購買廣告版面，《公論報》所需要的白報紙也一直長期缺乏。

中國青年黨是一個沒有實質反對黨功能的[[政治花瓶|政治花瓶]]政黨，李萬居一直有籌組一個實質反對黨的計劃，除出任「[[中國地方自治研究會|中國地方自治研究會]]」發起人之外，並且與[[雷震|雷震]]有所連繫；1960年雷震被捕後，當局便開始進行對李萬居的打擊，讓台北市議會議長[[張祥傳|張祥傳]]以藉著購買增資股權，以及當局司法手段的兩面手法，奪取《公論報》的經營權。李萬居雖然仍然有心辦一份給青年人的雜誌，但是因為[[糖尿病|糖尿病]]的舊疾復發與妻子的遽逝，而於1966年4月9日溘然長逝。

== 身後與歷史評價 ==
[[File:李萬居故居精神啟蒙紀念館.jpg|thumb]]
李萬居為定義上的[[半山仔|半山仔]]，在返台初期擔任官方媒體臺灣新生報負責人，與外省藉同僚一同支持臺人奴化論，且與國民黨特務密切來往，在二二八後對政府失望後投入[[民主政治|民主政治]]與[[言論自由|言論自由]]的鬥爭工作。擔任[[中國國民黨|中國國民黨]]的[[二二八事件|二二八事件]]善後委員期間，一面接受來自層峰的壓力猜忌，一面忍受來自民眾（對半山集團）的仇視唾棄，但他始終堅持這項艱鉅的工作，在自己也幾乎成為事件受難者的情況下，全力尋求將臺灣人損害降至最低的種種方法。<ref>《台灣新生報》 1947年3月4日台北發行之號外 p45 （《台灣新生報》1947.3.20:1，二欄題）p256 （《台灣新生報》1947.3.27:4，二欄題） p393 二二八事件臺灣本地新聞史料彙編(第一冊)</ref>

李氏宗族後世李文垻於[[民進黨|民進黨]]執政時取得[[政府|政府]]撥款，在[[雲林縣|雲林縣]][[口湖鄉|口湖鄉]]梧北村設立了「李萬居故居精神啟蒙紀念館」，供民眾參觀憑弔。該館成立時，[[副總統|副總統]][[呂秀蓮|呂秀蓮]]女士亦親自到場剪綵致意，[[2013年雲林農業博覽會|2013年雲林農業博覽會]]期間獲選為農博百大亮點。<ref>[http://expo.yunlin.gov.tw/?page_id=2248 百大亮點 - 2013雲林農業博覽會] {{webarchive|url=https://web.archive.org/web/20140822073845/http://expo.yunlin.gov.tw/?page_id=2248 |date=2014-08-22 }}</ref>

不過研究員[[許雪姬|許雪姬]]根據中研院台史所於2008年4月所購得的保密局臺灣站二二八史料指出「《臺灣新生報》社長李萬居派蔡水勝、林章／鐘（因資料中出現此二說法，且二字台語發音相近，故二字並呈）、蔡朝根、許家庭4人，打入報社中的『接收』委員，刺探[[阮朝日|阮朝日]]（《臺灣新生報》總經理）、[[吳金鍊|吳金鍊]]（《臺灣新生報》日文版總編、報社副總編）等人在事件中的動態。」其後，3月12日阮朝日遭國民黨情治人員帶走、處死，[[吳金鍊|吳金鍊]]也於報社被強行押走，至今下落不明。許雪姬以「令人不寒而慄」形容這批臥底史料所披露的訊息。因此過去對李萬居之正面評價，在這批史料問世後應嚴肅的重新審視。<ref>[http://www.ith.sinica.edu.tw/quarterly_download.php?name=05+%E8%A8%B1%E9%9B%AA%E5%A7%AC.pdf&filename=14310697562.pdf 許雪姬： 「保密局臺灣站二二八史料」的解讀與研究]，第13頁</ref>
