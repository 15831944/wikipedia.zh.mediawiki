'''李昕岳'''（{{lang-en|'''Sienna Li'''}}，{{bd|1990年|3月16日|||catIdx=L}}）[[北京人|北京人]]，毕业于[[中央戏剧学院|中央戏剧学院]]，中国大陆女演员、模特、主持人。<ref>{{cite web |url=http://ent.qq.com/a/20131108/000180.htm#p=1 |title=组图：女星李昕岳真空上阵走红毯 不惧走光 |accessdate=2013-11 |publisher= 腾讯网|work= |language=zh-hans}}</ref>

==生平==
1990年，李昕岳出生在[[北京市|北京市]]。她毕业于[[中央戏剧学院|中央戏剧学院]]表演预科班。2007年，她参加[[中国中央电视台|中国中央电视台]]第九届模特大赛荣膺冠军，之后转型当主持人，曾主持[[旅游卫视|旅游卫视]]《第一时尚》、《城市惠生活》、[[光线传媒|光线传媒]]《生活魔法师》。<ref>{{cite web |url=http://ent.sina.com.cn/s/m/2012-12-12/11523809665.shtml |title= 李昕岳杂志专访：嫩模也能有演技|accessdate=2012-12 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.163.com/13/0807/17/95MNKSUB00031GVS.html |title= 李昕岳谈从艺经历：每一个职业都会有收获|accessdate=2013-8 |publisher=网易 |work= |language=zh-hans}}</ref>
