'''李贞'''（{{bd|1908年|2月|1990年|3月11日|catIdx=L}}），[[湖南省|湖南省]][[浏阳县|浏阳县]]永和市镇[[小板桥乡|小板桥乡]]<ref name="a1"/>，[[中国人民解放军少将|中国人民解放军少将]]。[[中国人民解放军上将|上将]][[甘泗淇|甘泗淇]]之妻。第一、第二届[[全国妇联|全国妇联]]执委，第三、第四届全国妇联常委，[[第五届全国人大常委会|第五届全国人大常委会]]委员，[[中顾委|中顾委]]委员。

== 生平 ==

出身于农民家庭，六岁时成为[[童养媳|童养媳]]。随着[[國民大革命|大革命]]在1926年浪潮席卷而来，李贞开始参加社会活动。1927年初加入中国共产党，同年[[毛泽东|毛泽东]]领导[[秋收起义|秋收起义]]，攻打浏阳县城时，李参加部队。年末，参加[[王首道|王首道]]、[[张启龙|张启龙]]组织浏东游击队<ref name="a1"/>。其后，担任[[湘赣边区|湘赣边区]]红军妇女团政委。[[红二方面军|红二方面军]]政治部组织部副部长，参加[[长征|长征]]。后任[[八路军|八路军]]妇女学校校长。[[第二次国共内战|第二次国共内战]]期间，担任[[晋绥军区|晋绥军区]]政治部秘书长、[[西北军区|西北军区]]政治部秘书长<ref>{{cite news|title =李贞:上将丈夫少将妻 开国第一位女将军|url =http://news.qianlong.com/28874/2007/10/09/2502@4095257.htm|accessdate =2012-04-28|newspaper =千龙网|deadurl =yes|archiveurl =https://web.archive.org/web/20150709040528/http://news.qianlong.com/28874/2007/10/09/2502%404095257.htm|archivedate =2015-07-09}}</ref>。1951年，担任[[中国人民志愿军|中国人民志愿军]]政治部秘书长；1955年，授少将军衔，为中华人民共和国开国首位女少将<ref>{{cite news |title = 李贞：开国第一位女将军 |url = http://news.xinhuanet.com/politics/2007-10/09/content_6849601.htm |accessdate=2012-04-28 |newspaper=新华网}}</ref>。后担任[[中国人民解放军军事检察院|中国人民解放军军事检察院]]副检察长。

== 家庭 ==
李贞是家中的第六个女儿，六岁时，李贞成为古家的[[童养媳|童养媳]]。十六岁时，与丈夫古天顺正式成婚。1928年，因被[[國民政府|國民政府]][[通缉|通缉]]而被丈夫[[休书|休弃]]。1932年经组织批准，与[[张启龙|张启龙]]结婚，不久被迫离婚。1935年与[[甘泗淇|甘泗淇]]结婚。长征中，李贞早产生下两人的独子，最终孩子早夭。<ref name="a1">{{cite web |url = http://www.people.com.cn/GB/paper81/8389/789549.html|title=李贞：从童养媳到开国将军 |publisher=人民网，来源：《大地》(2003年第二十三期) |author=吴志菲 |date= |accessdate=2012-09-27 |language = zh-hans}}</ref>
