'''李凡荣'''，男，汉族，中共党员，1963年10月生于[[江西省|江西省]][[南昌市|南昌市]]<ref name="李凡荣同志简历">{{cite web | url=http://www.nea.gov.cn/2016-06/22/c_135457324.htm| title=李凡荣同志简历 |publisher=国家能源局|}}</ref>，1984年毕业于[[江汉石油学院|江汉石油学院]](现[[长江大学|长江大学]])采油工程专业，获工学学士学位<ref name="52岁中海油总公司副总经理李凡荣出任国家能源局副局长">{{cite web | url=http://www.thepaper.cn/newsDetail_forward_1480088| title=52岁中海油总公司副总经理李凡荣出任国家能源局副局长 |publisher=澎拜新闻|}}</ref><ref name="李凡荣、李仰哲任国家能源局副局长">{{cite web | url=http://finance.sina.com.cn/stock/t/2016-07-13/doc-ifxtwitr2101464.shtml| title=李凡荣、李仰哲任国家能源局副局长|publisher=新浪财经|}}</ref><ref name="国务院任命李凡荣为国家能源局副局长">{{cite web | url=http://energy.people.com.cn/n1/2016/0607/c71661-28418892.html| title=国务院任命李凡荣为国家能源局副局长 |publisher=人民网|}}</ref>；2003年7月获英国[[卡迪夫大学|卡迪夫大学]]商学院MBA专业硕士学位<ref name="国务院任命李仰哲、李凡荣为国家能源局副局长">{{cite web | url=http://news.ifeng.com/a/20160607/48932630_0.shtml| title=国务院任命李仰哲、李凡荣为国家能源局副局长 |publisher=凤凰资讯|}}</ref>。1984年，李凡荣进入[[中国海洋石油总公司|中国海洋石油总公司]]工作，2011年11月任中国海洋石油总公司首席执行官。 2016年5月起任国家能源局副局长、党组成员<ref name="国务院任命李仰哲、李凡荣为国家能源局副局长">{{cite web | url=http://news.ifeng.com/a/20160607/48932630_0.shtml| title=国务院任命李仰哲、李凡荣为国家能源局副局长 |publisher=凤凰资讯|}}</ref>。

==人物生平==
*1963年10月生于江西省南昌市；
*1984年毕业于[[长江大学|长江大学]]石油开发系采油工程专业，获[[工学学士|工学学士]]学位；
*1984年8月进入[[中国海洋石油总公司|中国海洋石油总公司]]南海东部石油公司；
*2010年4月，任中国海洋石油总公司副总经理、党组成员；
*2010年9月，兼任中国海洋石油有限公司总裁；
*2011年11月，兼任中国海洋石油有限公司首席执行官兼总裁；
*2016年5月起任国家能源局副局长、党组成员。
