'''李中华'''（{{bd|1961年|9月||}}）[[辽宁省|辽宁省]][[新宾县|新宾县]]人，[[中国人民解放军空军|中国人民解放军空军]]大校，空军试飞专家，“英雄试飞员”<ref name=lnrw/>。

==生平==
1983年7月自[[南京航空航天大学|南京航空航天大学]]毕业，获工学学士学位。毕业后招飞入伍，进入[[中国人民解放军空军第六飞行学院|中国人民解放军空军第六飞行学院]]学习，1985年7月毕业获[[军事学|军事学]]学士学位。1984年5月加入[[中国共产党|中国共产党]]<ref name="lnrw">{{cite web |url = http://www.lnlm.net/product/20140430115655.html |title = 李中华 |author =  |date =2014-04-30  |publisher = 辽宁人物 |language =  |accessdate =  |quote =  }}</ref>。作为中国人民解放军空军首批12位双学士飞行员之一，李中华提前晋升飞行中队长。1989年，李中华决定做仍在设计中的新型飞机的试飞员<ref name=rmw>{{Cite web |url =http://military.people.com.cn/GB/42967/5281872.html  |title = “中华的天空”―中国空军试飞勇士李中华 |author =  |date = 2007-01-15 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。此后先后在中国试飞员学校、[[俄罗斯|俄罗斯]]国家试飞员学校学习。历任副大队长、大队长、团参谋长、副团长<ref name=lnrw/>。他先后驾驶和试飞[[歼击机|歼击机]]、[[歼击轰炸机|歼击轰炸机]]、[[运输机|运输机]]等3个机种27个机型，安全飞行3000余小时；先后正确处置空中险情15起、空中重大险情5起；参与完成十多项重大科研试飞任务；在新技术验证试飞及新机鉴定试飞中填补两项国内空白；在国家及军队刊物发表20多篇文章；按照[[中国人民解放军总政治部|中国人民解放军总政治部]]要求撰写并发表了自传《蓝天亮剑》<ref name=lnrw/>。他还攻读了[[西北工业大学|西北工业大学]]航空专业硕士研究生<ref name=rmw/>。

截至2007年，李中华是中国仅有的3名国际试飞员之一，先后立一等功1次，二等功5次，三等功6次；国家航空工业部门先后给李中华记一等功4次，二等功5次，三等功6次；李中华获国家科学技术进步特等奖、二等奖各1次<ref name=shifei>{{Cite web |url =http://www.cnr.cn/military/tebie/sqd/dbfc/200710/t20071012_504593037.html  |title = 试飞英雄李中华——和平年代实践国家使命 |author =  |date = 2007-10-12 |publisher =  中国广播网|language =  |accessdate =  |quote =  }}</ref>。2006年，被评为全军首届“十大青年爱军精武标兵”之一。他是空军功勋飞行员，还曾获“中国航空航天月桂奖·英雄无畏奖”<ref name=rmw/>。

2006年3月25日，中共中央总书记、国家主席、中央军委主席[[胡锦涛|胡锦涛]]视察空军某部时，接见了空军某试飞团副团长李中华<ref name=shifei/>。2006年10月22日，李中华作为解放军代表，在[[人民大会堂|人民大会堂]]举行的全国纪念红军[[长征|长征]]胜利70周年大会的主席台上发言<ref name=rmw/>。

后来，他历任[[中国人民解放军空军装备部|中国人民解放军空军装备部]]科研订货部副部长、[[中国人民解放军成都军区空军|中国人民解放军成都军区空军]]装备部副部长，[[中国人民解放军空军指挥学院|中国人民解放军空军指挥学院]]训练部副部长<ref name=lnrw/>。

2017年7月，中央军委主席[[习近平|习近平]]签署命令，授予李中华等10位同志“[[八一勋章_(2017年)|八一勋章]]”<ref>{{cite web |url = http://news.ifeng.com/a/20170728/51522244_0.shtml |title = 景海鹏、程开甲、王刚等10人被授予“八一勋章” |author =  |date = 2017-07-28 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。2017年7月28日，中央军委颁授“八一勋章”和授予荣誉称号仪式在北京[[八一大楼|八一大楼]]举行，中共中央总书记、国家主席、中央军委主席习近平向李中华等10位“八一勋章”获得者颁授勋章和证书。这是“八一勋章”首次颁授。李中华同时获授“'''挑战极限、勇争第一的试飞英雄'''”[[荣誉称号|荣誉称号]]<ref>{{cite web |url = http://news.ifeng.com/a/20170728/51524452_0.shtml |title = 习近平向“八一勋章”获得者颁授勋章和证书 |author =  |date = 2017-07-28   |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
