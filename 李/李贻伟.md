'''李贻伟'''（{{bd|1965年|11月|}}），[[山东|山东]][[阳谷|阳谷]]人。男，汉族。1988年6月参加工作，1988年6月加入中国共产党。[[华南理工大学|华南理工大学]]轻化学造纸专业毕业。

==生平==
历任[[广东省|广东省]][[江门市|江门市]]江门造纸厂工程师，佛山市泰嵩纤维内底板制造有限公司制造部经理、副总经理、高级工程师，佛山市工艺美术公司经理、党委副书记、书记，佛山市工业投资管理有限公司总经理、党委副书记。

2000年12月，任佛山市三水市委副书记、市长。2003年1月，任佛山市[[三水区|三水区]]委副书记、区长。2003年8月，转任佛山市[[南海区|南海区]]委副书记、区长。2006年11月，任佛山南海区委书记。2006年12月，任佛山市委常委、南海区委书记、区人大常委会主任。2010年7月，任佛山市委副书记、市长。2011年8月，升任佛山市委书记。2012年1月，任佛山市委书记、市人大常委会主任。2014年5月，任广州市委副书记<ref>{{cite web |url = http://news.ifeng.com/a/20140519/40358644_0.shtml |title = 佛山市委书记李贻伟拟任广州市委副书记|date = 2014年5月19日|accessdate = 2017年7月20日|publisher = 凤凰资讯}}</ref>。2014年11月，被任命为广东省政府副秘书长<ref>{{cite web |url = http://district.ce.cn/newarea/sddy/201411/26/t20141126_3985843.shtml |title = 广州市委副书记李贻伟任广东省政府副秘书长|date = 2014年11月26日|accessdate = 2017年7月20日|publisher = 中国经济网综合}}</ref>
。

2018年6月后，任惠州市委书记
