'''李亞男'''（{{lang-en|'''Leanne Li'''}}，{{bd|1984年|11月25日|catIdx=L李}}），[[上海|上海]]出生；香港女艺人，2004年[[溫哥華華裔小姐|溫哥華華裔小姐]]及2005年[[國際華裔小姐競選|國際華裔小姐競選]]冠軍<ref name="auto">[http://news.xinhuanet.com/ent/2005-02/01/content_2534197.htm 國際華裔小姐出爐李亞男艷壓群芳摘冠] 新華網，2005年2月1日</ref>，前[[香港|香港]][[電視廣播有限公司|無線電視]]旗下經理人合約藝員，現為[[手工藝創作|手工藝創作]]旗下藝員。<ref>{{cite web|url=http://www.youtube.com/watch?v=1pqND1Vhtp8|title=YouTube|website=www.youtube.com}}</ref>丈夫為前無線男藝員[[王祖藍|王祖藍]]。

== 演藝經歷 ==
2004年參選[[溫哥華華裔小姐|溫哥華華裔小姐]]競選奪魁，隨後參選2005年[[國際華裔小姐競選|國際華裔小姐競選]]，並以大熱姿態奪得冠軍。<ref name="auto"/>同屆華姐包括[[陳法拉|陳法拉]]及[[徐子珊|徐子珊]]。隨後她與[[電視廣播有限公司|無綫電視]]簽約，成為旗下合約女藝員。同年8月加入《[[女人唔易做|女人唔易做]]》劇組，因此《女人唔易做》為其在無綫首部的電視作品。

李亞男加入無綫後，並不像同屆華姐[[陳法拉|陳法拉]]及[[徐子珊|徐子珊]]、前屆華姐冠軍[[鍾嘉欣|鍾嘉欣]]那樣被無綫力捧成為旗下當家花旦。相反，李亞男在多部劇集均以小配角或閒角身份出場，代表作少，出鏡率低，人氣不高。直到2008年的《[[師奶股神|師奶股神]]》及2009年的《[[烈火雄心3|烈火雄心3]]》，她的角色才有較重的戲份。憑藉在《[[烈火雄心3|烈火雄心3]]》與男主角[[鄭嘉穎|鄭嘉穎]]的深情吻戲開始為觀眾所留意。

自《烈火雄心3》開始，李亞男已成為監製[[王心慰|王心慰]]的御用演員，其餘御用演員包括[[李司棋|李司棋]]、[[鄭嘉穎|鄭嘉穎]]、[[吳卓羲|吳卓羲]]、[[唐詩詠|唐詩詠]]、[[陳展鵬|陳展鵬]]、[[田蕊妮|田蕊妮]]及[[阮兆祥|阮兆祥]]。在往後演出的劇集如《[[Only_You_只有您|Only You 只有您]]》、《[[真相_(無綫電視劇集)|真相]]》、《[[衝呀！瘦薪兵團|衝呀！瘦薪兵團]]》、《[[巨輪|巨輪]]》及《[[叛逃_(電視劇)|叛逃]]》都是王心慰所監製之電視作品。

2014年李亞男於王心慰監製的時裝懸疑電視劇《[[忠奸人_(電視劇)|忠奸人]]》飾演貪慕虛榮、內心陰沉的施嘉莉一角成功出位，被網民大贊演技有進步，同時被網民看好有望角逐《[[萬千星輝頒獎典禮2014|萬千星輝頒獎典禮2014]]》飛躍進步女藝員。最終她憑藉此劇在《[[TVB_馬來西亞星光薈萃頒獎典禮2014|TVB 馬來西亞星光薈萃頒獎典禮2014]]》中榮獲提名「我最喜愛TVB女配角」及在《[[萬千星輝頒獎典禮2014|萬千星輝頒獎典禮2014]]》榮獲提名「最佳女配角」。同年她在《[[老表，你好hea！|老表，你好hea！]]》的搞笑演出再次獲得網民及觀眾的一致好評。

2015年於台慶劇《[[東坡家事|東坡家事]]》出演刁蠻任性的壽安郡主，被網民大贊將此角色的性格演得活靈活現。

2016年在劇集《[[潮流教主|潮流教主]]》飾演一位退役模特兒兼模特兒公司經理人萬綺華（Martha）一角，由於演技精湛而被網民支持她應該力壓[[蔡思貝|蔡思貝]]於此劇擔正第一女主角。

2017年離開無綫，並簽約其丈夫[[王祖藍|王祖藍]]旗下的「手工藝創作」。

現時李亞男主要與[[王祖藍|王祖藍]]以夫妻檔登台及參加中國的真人秀，並在中國累積了不少人氣，微博粉絲高達200萬。

== 圈內好友及家庭 ==
*李亞男與[[鍾嘉欣|鍾嘉欣]]、[[王君馨|王君馨]]、[[苟芸慧|苟芸慧]]、[[岑麗香|岑麗香]]因同為[[基督|基督]][[新教徒|新教徒]]，數人自發組成祈禱小組「Amazing Esthers」，亦常有聚會，因基督教教義是反對[[婚前性行為|婚前性行為]]，而當時五人均未婚，故傳媒牽連在一起並戲稱為「TVB五大處女」<ref>{{cite web|url=http://hk.apple.nextmedia.com/realtime/videonews/20170115/56176929|title=香港自救|website=Apple Daily 蘋果日報}}</ref>。

*李亞男於2014年11月19日[[萬千星輝賀台慶|TVB邁向48周年台慶]]接受[[王祖藍|王祖藍]]求婚。2015年2月14日，李亞男與丈夫[[王祖藍|王祖藍]]於九龍城浸信會教堂舉行婚禮，這對“長短腿之戀”終於修成正果。正值當時王祖藍北上發展，令兩人婚後聚少離多，更試過因抑鬱及驚恐情緒影響而兩度入院<ref>{{cite web|url=https://hk.lifestyle.appledaily.com/nextplus/magazine/article/20171004/2_552558_0/%E6%9D%8E%E4%BA%9E%E7%94%B7%E9%A9%9A%E6%81%90%E7%97%87%E5%85%A5%E9%99%A2-%E7%8E%8B%E7%A5%96%E8%97%8D%E5%BF%83%E7%97%9B-%E6%88%91%E4%BB%A5%E7%82%BA%E4%BD%A2%E6%9C%83%E6%AD%BB|title=李亞男驚恐症入院 王祖藍心痛：我以為佢會死 - 2017-10-04 - 壹週刊 - 蘋果日報|website=Apple Daily 蘋果日報}}</ref>。但李亞男澄清自己未有因此而患上抑鬱症，王祖藍在接受《蘋果日報》獨家專訪時表示為了多陪伴妻子並造人而準備在明年所有工作完成後暫時停工。<ref>{{cite web|url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20170929/20167106|title=老婆被掉低空虛成病祖藍害李亞男患抑鬱|website=Apple Daily 蘋果日報}}</ref>

*2018年7月13日，李亞男在社交網站上載超聲波照，公布懷孕消息。<ref>[https://hk.ulifestyle.com.hk/topic/detail/205481/ 王祖藍李亞男結婚3年做爸媽]，港生活，2018-07-13</ref>
