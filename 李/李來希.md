'''李來希'''（1957年－），籍貫[[中華民國|中華民國]][[臺北市|臺北市]]，畢業於[[國立師大附中|國立師大附中]]、[[東吳大學|東吳大學]]法律系學士、[[威斯康星大学麦迪逊分校|威斯康辛大學麥迪遜分校]]法學[[碩士|碩士]]，現任[[勞動部|勞動部]]簡任第十二職等技監、社團法人[[中華民國全國公務人員協會|中華民國全國公務人員協會]]理事長、[[中華民國總統府|總統府]][[國家年金改革委員會|國家年金改革委員會]]委員<ref>[http://www.businesstoday.com.tw/article-content-80392-156285?page=3 年金改革委員會 他們的共識在哪？ 今周刊 2016/06/30] {{webarchive|url=https://web.archive.org/web/20160916091540/http://www.businesstoday.com.tw/article-content-80392-156285?page=3 |date=2016-09-16 }}</ref>，曾擔任[[行政院|行政院]][[勞工委員會|勞工委員會]]勞動條件處處長、綜合規劃處處長、勞動部[[參事|參事]]。

李來希曾參與籌組[[軍公教聯盟黨|軍公教聯盟黨]]，為其終身黨員。他時常領導現任與退休[[軍公教|軍公教]]抗議[[蔡英文政府|蔡英文政府]][[年金改革|年金改革]]<ref>{{cite news|author1=吳欣恬|title=上千軍公教上凱道 要求年金改革不溯及既往|url=http://news.ltn.com.tw/news/politics/breakingnews/1739422|accessdate=2016-08-18|agency=自由時報|date=2016-06-23}}</ref>，如「[[軍公教反污名要尊嚴九三大遊行|軍公教九三大遊行]]」[[示威|示威]]<ref>{{cite news|author1=周慧如、吳虹廷|title=不滿軍公教遭抹黑 監督年金聯盟號召10萬人9／3上凱道|url=http://www.chinatimes.com/realtimenews/20160811004364-260407|accessdate=2016-08-18|agency=中國時報|date=2016-08-11}}</ref>。

== 退休金改革 ==
=== 立場 ===
2016年11月19日，[[銓敘部|銓敘部]]對軍公教人員退休金設最低樓地板3萬2千元限制，公務人員代表李來希表示3萬2千元無法有尊嚴的活下去，軍公教退休金（最低樓地板）最少要5萬至6萬元之間<ref>[http://news.ltn.com.tw/news/politics/breakingnews/1891467 他嗆「退休公務員3萬2如何養老」網友：一堆22K怎麼活]，[[自由時報|自由時報]]，2016-11-19</ref><ref>[http://www.setn.com/News.aspx?NewsID=199796 軍公教退休金限制3萬2　李來希嗆：這怎麼活？]，[[三立新聞台|三立新聞台]]，2016-11-19</ref>。

2017年1月7日，[[國立中興大學|中興大學]]年金改革中區座談會中李來希表示，堅持信賴保護原則、反對溯及既往（銓敘部規畫全面廢除[[退休軍公教人員優惠存款|18%優存]]），維持退休金舊制與18%優存是基於法律的穩定性、制度的可預測性，不只是為了上一代、這一代，也是為了世世代代。[[公務人員退休撫卹基金|公務人員退休撫卹基金]]會破產的觀念是錯誤的，84年以前舊制沒有基金破產問題，只有基金準備不足等問題，並且質疑政府規畫砍退休軍公教18%是要軍公教去死嗎？<ref>[http://www.upmedia.mg/news_info.php?SerialNo=10286 李來希：政府砍退休老人18%，是直接要他去死嗎？]，[[上報|上報]]，2017-1-7</ref>全世界的社會保險都沒有破產的問題，只是基金準備存量不足，但可以因為參數、費率級給付率等改變，就可以適當處理<ref>[http://www.chinatimes.com/realtimenews/20170107003000-260407 談年金改革 李來希：支持改革 但反對亂改]，[[中國時報|中國時報]]，2017-1-7</ref>。

=== 議論 ===
2017年1月8日，年金改革分區會議第3場會前(8日下午)，8日早上[[公務革新力量聯盟|公務革新力量聯盟]]年輕公教人員召開記者會支持年金改革，公務革新力量聯盟發言人林于凱表示：年金改革失敗，45歲以下的年輕世代將是年金破產的受災戶。現在的年改方案是要年輕人多繳少領，改革方案必須溯及既往，才符合世代公平正義。
# 「改革必須溯及既往、一體適用」
# 「在職薪資愈高、所得替代率愈低」
# 「訂定天花板與地板條款」
# 「年金制度每5年精算微調(永續經營、動態調整)」
# 「警消、幼教、中小學教師等特殊族群退休年齡有特殊考量」
# 退輔基金監理委員會應納入各界代表
# 年金改革應有更多年輕公教代表
[[台灣風險與保險學會|台灣風險與保險學會]]理事長王儷玲表示：依照目前的制度，年輕族群是「保費繳最多、福利被砍最多」的一群。所得替代率超過70%的應該下降<ref>[http://newtalk.tw/news/view/2017-01-08/80815 年輕公教挺年金改革 籲溯及既往、一體適用]，[[新頭殼|新頭殼]]，2017-1-8</ref><ref>[http://www.cna.com.tw/news/firstnews/201701080055-1.aspx 年輕公教挺年金改革 籲世代平衡溯及既往]，[[中央通訊社|中央通訊社]]，2017-1-8</ref>。
