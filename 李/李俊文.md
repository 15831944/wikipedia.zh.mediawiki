'''李俊文'''（{{bd|1969年|}}<ref name="a1"/>），原[[山西省|山西省]][[太原市|太原市]][[小店区|小店区]]西温庄乡西攒村村民委员会主任、区[[人大代表|人大代表]]，中华人民共和国基层官员。

2003年至2011年12月间<ref name="a1">{{Cite web |url=http://www.ce.cn/xwzx/gnsz/gdxw/201212/20/t20121220_23960191.shtml|title=有4妻人大代表事件背后:分红不公引民怨被举报|accessdate=2017-06-16 |author= |date=2012-12-20 |work= |publisher=中国经济网，来源：东方网|language=Zh-hans }}</ref>为西攒村委会主任。李俊文有四个“[[老婆|老婆]]”，“合法生育”十个[[子女|子女]]。其中，第一位领取结婚证的妻子<ref name="a1"/>、第二、第三位情妇及九名子女的[[户口|户口]]都在西攒村<ref>{{Cite web |url=http://news.xinhuanet.com/2012-12/05/c_124048035.htm |title=山西一人大代表被指共有4老婆 10个小孩均上户口  |accessdate=2017-06-16 |author=任东杰 |date= |work= |publisher=新华网，来源：法治周末|language=Zh-hans }}</ref>。村民将李俊文的违反乱纪情况反映到[[信访局|信访局]]，一[[领导|领导]]竟回复“那是他的本事”<ref>[http://gx.people.com.cn/n/2012/1206/c179458-17821781.html 太原调查人大代表被指有4个老婆10个小孩事件] {{webarchive|url=https://web.archive.org/web/20121211003247/http://gx.people.com.cn/n/2012/1206/c179458-17821781.html |date=2012-12-11 }}</ref>。2012年12月5日，网络媒体曝光此事。6日，中共小店区委、小店区人民政府组织多部门召开会议<ref name="a1"/>。中共小店区区委宣传部在6日表示，李俊文的[[代表|代表]]资格已被暂停，[[公安机关|公安机关]]已对其采取[[取保候审|取保候审]]的刑事强制措施<ref>[http://news.ifeng.com/gundong/detail_2012_12/08/19971379_0.shtml “山西区人大代表有4妻10子”属实]</ref>。8日，李俊文被公安机关刑事拘留。9日，小店区人大常委会终止其代表资格<ref name="a1"/>。12月22日，检察机关已经逮捕李俊文，目前案件正经司法程序审理<ref>{{cite news|title=新疆公安局长“包养孪生姐妹”事件调查月余未果|url=http://news.163.com/13/0123/05/8LSMK1AQ00014JB6.html|accessdate=2013-01-23|newspaper=人民网}}</ref>。

2013年12月，山西省太原市小店区人民法院以犯[[故意伤害罪|故意伤害罪]]和[[伪造国家机关证件罪|伪造国家机关证件罪]]，一审判处李俊文[[有期徒刑|有期徒刑]]两年六个月。李俊文提起上诉，[[山西省太原市中级人民法院|山西省太原市中级人民法院]]维持原判。司法机关未追究李俊文[[重婚罪|重婚罪]]<ref>{{Cite web |url=http://news.ifeng.com/a/20140723/41272387_0.shtml |title=山西村主任与4女产10子被判刑 涉嫌重婚未受追究 |accessdate=2017-06-16 |author=任东杰 |date=2014-07-23 |work= |publisher=凤凰网，来源：法治周末|language=Zh-hans }}</ref>。
