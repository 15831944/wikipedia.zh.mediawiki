'''李鹏'''（{{bd|1928年|10月20日|catIdx=L,李鹏}}），原名'''李远芃'''，[[四川|四川]][[庆符|庆符]]（今属[[高县|高县]]）人，祖籍[[湖北|湖北]][[麻城|麻城]]{{r|bird|page=4}}，出生于[[上海法租界|上海法租界]]。毕业于[[苏联|苏联]]{{Link-en|莫斯科动力学院|Moscow Power Engineering Institute}}水力发电系，高级工程师，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]前[[党和国家领导人|主要领导人]]之一。1945年加入[[中国共产党|中国共产党]]，中共第十三至十五届[[中国共产党中央政治局常务委员会|中央政治局常委]]。曾先后任[[中华人民共和国电力工业部|电力工业部]]部长、[[中华人民共和国国家教育委员会|国家教育委员会]]主任、[[国务院副总理|国务院副总理]]、[[国务院总理|国务院总理]]和[[全国人大常委会委员长|全国人大常委会委员长]]等重要职务。

==个人经历==
===早年生涯===
1928年10月，李鹏以李远芃之名在[[上海|上海]][[上海法租界|法租界]]明德里十五号出生{{r|bird|page=3}}，属“远”字辈，“芃”字具有草木茂盛之意，父亲为[[李硕勋|李硕勋]]，母亲为[[赵君陶|赵君陶]]，二人皆加入共产党并从事于革命工作，其父亲更是中共领导人。1931年李硕勋因而身份被暴露，在海南海口被[[国民党|国民党]]處決{{r|李國強2|page=7}}。1939年受邓颖超之邀，李远芃由成都来到重庆，先后与[[吴玉章|吴玉章]]及其子[[吴震寰|吴震寰]]和[[周恩来|周恩来]]相识，并从吴震寰处第一次了解了水力发电的知识。

之后中共地下党送他到[[陶行知|陶行知]]创建的[[育才学校|育才学校]]读书。不久因国共关系变化，李远芃与[[边章五|边章五]]、[[李涛_(解放军)|李涛]]被送往延安{{r|bird|page=50-51}}。在途中，[[中共中央南方局|中共中央南方局]]青委书记[[蒋南翔|蒋南翔]]建议并帮他改名为李鹏{{r|bird|page=52}}。1946年7月，毕业后分配到[[张家口|张家口]]电业局工作{{r|bird|page=111}}。

1947年初，他离开[[晋察冀边区|晋察冀边区]]，经[[烟台|烟台]]前往[[大连|大连]]{{r|bird|page=119-123}}，又经[[朝鲜|朝鲜]]前往[[哈尔滨|哈尔滨]]，在哈尔滨油脂厂担任协理兼支部书记{{r|bird|page=第132页}}，开始成为基层领导{{r|李國強2|page=8}}。1948年至1955年在[[莫斯科|莫斯科]]留学，同去的还有中共高干及烈士子女20人（“[[4821苏修特务案|4821]]”）。李鹏自苏联[[莫斯科动力学院|莫斯科动力学院]]水力发电系毕业。其间，曾担任中国留苏学生总会主席{{r|李國強2|page=9}}。

=== 进入电力系统 ===
1954年李鹏毕业，恰逢1954年底[[中华人民共和国燃料工业部|燃料工业部]]第一副部长[[刘澜波|刘澜波]]率领中国电力代表团到苏联考察，在莫斯科让李鹏等毕业生随团活动并任翻译。李鹏也因此得到刘澜波赏识{{r|李國強3|page=50}}。李鹏回国后，1955年3月8日，刘澜波约李鹏讨论其工作，刘澜波表示李鹏可选择在燃料工业部任其秘书，或是下基层工作。李鹏明确表示愿下基层工作<ref name=cankao>{{Cite web |url = http://news.hexun.com/2014-07-06/166354696.html |title =李鹏任电力部长前曾遭人联名给中央写信反对  |author =  |date =2014-07-06  |publisher = 和讯 |language =  |accessdate =  |quote = 本文参考《李鹏回忆录（1928-1983）》一书 }}</ref><ref name=xia>{{Cite web|url =http://e.mzyfz.com/paper/paper_841_592.html|title =讲述特殊群体“4821”的故事（下）|author =|date =2014-07-24|publisher =民主与法制时报|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170314094748/http://e.mzyfz.com/paper/paper_841_592.html|archivedate =2017-03-14}}</ref>。1955年3月23日，李鹏接到通知，被分配到当时中国大陆最大的水电厂[[丰满发电站|丰满发电厂]]任见习厂长（厂长为李旭）<ref name=cankao/>{{r|李國強3|page=50}}。在见习期后，1955年12月10日，李鹏被任命为丰满发电厂副厂长兼总工程师<ref name=cankao/><ref name=jianli/>{{r|李國強3|page=51}}。

1957年，中共开展[[整风反右运动|整风运动]]，丰满发电厂党委和行政部门的人员对副厂长李鹏提出了批评意见。1959年在[[反右倾运动|反右倾运动]]中，李鹏被批评为执行“[[三面红旗|三面红旗]]”不力，差点被打成“右倾机会主义”分子，李鹏作了两次检查，丰满发电厂在给中共[[吉林市|吉林市]]委的报告中称其“检讨尚好，免予处分”<ref name=cankao/>。

1960年8月中旬，李鹏到[[青岛|青岛]]看望母亲时，表示想调离丰满发电厂。从青岛回丰满后不久，上级通知李鹏调任辽吉电业管理局副总工程师兼调度局局长。1961年5月，任东北电业管理局副总工程师兼调度局局长<ref name=cankao/>。1965年7月19日，国务院任命李鹏为当时中国最大的火力发电厂——[[阜新发电厂|阜新发电厂]]的厂长<ref name=cankao/><ref name=zinv>{{Cite web |url =  http://gd.people.com.cn/n/2014/0704/c123932-21580526.html |title = 人民日报：李鹏对子女要求严格 要求他们不替人传信 |author =  |date =2014-07-04  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>{{r|李國強2|page=10}}。东北电业管理局局长[[李平_(电力)|李平]]告诉李鹏，任命原因一是在[[四清运动|“四清”运动]]期间，李鹏一直接管阜新发电厂生产技术工作，如接任厂长便可使工作连续；二是为培养锻炼李鹏。李平还告诉李鹏，刘澜波和[[中华人民共和国水利电力部|水利电力部]]认为李鹏在丰满水电厂有了管理水电厂的经验，在东北电业管理局有了管理电网的经验，到阜新发电厂工作可增加管理火力发电厂的经验<ref name=cankao/>。在阜新发电厂，李鹏每天上午都深入生产一线。他和总工程师杨耀邦等人还领导了阜新发电厂首次9台机组满出力发电并获成功<ref name=zinv/>。

===首都电力===
1966年[[文化大革命|文化大革命]]爆发后，国务院总理[[周恩来|周恩来]]担心北京的供水、供电、煤气、交通问题，要刘澜波向北京供电局派一位得力干部。刘澜波推荐了李鹏，获周恩来批准。1966年，原本要从阜新发电厂调到东北电业管理局任副局长的李鹏被紧急调到北京，任北京供电局党委代理书记。此时正值“四清”运动后，原北京供电局党委书记、局长和几位副局长都在运动中发现有问题，正受隔离审查。刘澜波告诉李鹏，这次调动是经周恩来同意，将使他获得管理供电部门的经验<ref name=cankao/>。数年后，李鹏任北京供电局革命委员会主任<ref name=jianli>{{Cite web |url = http://www.china.com.cn/chinese/archive/128289.htm |title = 李鹏简历 |author =  |date =  |publisher =中国网  |language =  |accessdate = 2017-03-14 |quote =  }}</ref>。

调到北京供电局后，李鹏因第三批留学苏联的经历而被卷入[[4821苏修特务案|4821苏修特务案]]，被认为有“苏修特务”嫌疑<ref name=an>{{Cite web |url = http://news.163.com/09/0525/19/5A6BBUUF00011247.html |title = “4821”苏修特务案 |author =韩磊  |date = |publisher =炎黄春秋2009年第5期  |language =  |accessdate =  |quote =  }}</ref><ref name=rwzbw/>{{r|bird|page=370-374}}。阜新发电厂派人来北京，要将李鹏带回阜新批斗。但两次要人，都被北京供电局军代表张晓光挡回。一次张晓光在聊天中告诉李鹏，这批留苏干部被打成“苏修特嫌”（苏修特务嫌疑）的问题，周恩来已知道，“周总理说，你们第三批人没有这个问题”。但此案并未结束。后来，北京电业管理局革命委员会还向上级递交报告，建议将李鹏调出北京供电局，因为李鹏有“苏修特嫌”问题<ref name=rwzbw>{{Cite web|url =http://www.rwzbw.com/system/2014/07/22/010733585_01.shtml|title =胡耀邦为李鹏“苏修特务”案平反|author =|date =2014-07-22|publisher =人物周报网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170314153602/http://www.rwzbw.com/system/2014/07/22/010733585_01.shtml|archivedate =2017年3月14日|df =}}</ref>。

1970年6月，国务院总理周恩来在[[国务院小礼堂|国务院小礼堂]]接见水电会议代表，李鹏也在其内。周恩来在讨论中点名批评李鹏有“洋教条”和对工人管卡压的情况，李鹏承认错误并自我批评。会后，[[李德生|李德生]]、[[余秋里|余秋里]]、[[杨寿山_(巨鹿)|杨寿山]]等人留在会议室研究如何贯彻周恩来指示。李鹏也留下。李德生劝慰李鹏。余秋里告诉李鹏，周恩来批评意在让李鹏出来工作。1971年，北京电业管理局及其下属的北京供电局与北京电力建设局合并，成立北京电力工业局。原北京供电局革命委员会主任李鹏被任命为北京电力工业局革命委员会副主任、党委副书记<ref name=rwzbw/>。

1974年1月25日，中共中央和国务院直属机关“[[批林批孔|批林批孔]]”动员大会在[[首都体育馆|首都体育馆]]召开。随后北京电力工业局大楼内[[大字报|大字报]]铺天盖地，指责局党委对“批林批孔”不积极，还质问局党委为何不传达大会录音。主持日常工作的李鹏则以未收到通知为由，坚持不传达大会录音<ref name=zinv/>。

1970年代，京津唐电网电力供应紧张。李鹏牵头新建唐山陡河电厂，扩建北京高井电厂。扩建高井电厂时，1974年李鹏三天三夜坚守工地，第四天即1974年10月4日中午，李鹏在骑[[自行车|自行车]]回家路上在[[宣武门|宣武门]]附近被汽车撞伤，到[[首都医科大学宣武医院|宣武医院]]住院治疗，出院后休养至12月恢复工作<ref name=cankao/><ref>{{Cite web|url =http://www.cq.xinhuanet.com/2014-06/29/c_1111366892_4.htm|title =李鹏新书回忆爱情婚姻：与朱琳互称“大琳”“大鹏”|author =|date =2014-06-29|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170314065847/http://www.cq.xinhuanet.com/2014-06/29/c_1111366892_4.htm|archivedate =2017-03-14}}</ref>。1975年11月，北京电力工业局改称北京电业管理局<ref name=siwu>{{Cite web |url = http://news.ifeng.com/a/20140713/41136125_0.shtml |title = 李鹏去天安门悼念周恩来被调查 朱琳担忧致病 |author =  |date = 2014-07-13 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。李鹏任北京电业管理局局长、党组书记，任至1979年<ref name=jianli/>。

1976年1月8日，[[周恩来之死|周恩来逝世]]。1月9日，李鹏在北京电业管理局机关召开全体人员大会，发表了悼念讲话。此后李鹏等北京市干部接到[[中共北京市委|中共北京市委]]通知，到[[北京医院|北京医院]]小礼堂参加了遗体告别活动。1月12日至14日在[[北京市劳动人民文化宫|北京市劳动人民文化宫]]举行吊唁仪式，李鹏承担了保证供电安全的任务。4月4日[[清明节|清明节]]上午，因有保证天安门供电安全的责任，李鹏和北京供电局局长张绍贤、北京电业管理局计划处处长陈寿文乘车到天安门广场东南角的变电所检查工作。当天晚饭后，他又和妻子朱琳步行到广场，在清场抓人前离开从而未被抓<ref name=siwu/>。次日即发生[[四五运动|四五运动]]{{r|bird|page=418-420}}。4月5日下午，[[北京市公安局|北京市公安局]]派人来北京电业管理局调查，称清明节上午在广场东南角邮局前放火的男青年是乘北京电业管理局轿车去的，北京电业管理局人员则澄清是李鹏等三人乘车检查工作。几天后，[[邓颖超|邓颖超]]秘书[[张元_(邓颖超秘书)|张元]]电话告知朱琳，若李鹏不见了，朱琳须追问与李鹏最后在一起的是谁，是谁将李鹏带走。所幸后来李鹏并未被捕<ref name=siwu/>。

[[唐山大地震|唐山大地震]]后，李鹏担任唐山电力抢修队指挥部总指挥，参与抢险救灾工作。地震一个月后，唐山地区供电基本恢复<ref name=cankao/><ref name=zinv/>{{r|bird|page=424-431}}。

===调部任职===
1979年2月，中央决定撤销水利电力部，成立[[中华人民共和国电力工业部|电力工业部]]、[[中华人民共和国水利部|水利部]]。刘澜波任电力工业部部长、党组书记<ref name=cankao/>。刘澜波举荐李鹏任副部长{{r|李國強2|page=10}}。1979年4月7日，李鹏接到通知，任电力工业部副部长<ref name=cankao/>。他还任该部党组成员<ref name=jianli/>{{r|李國強2|page=10}}。1980年1月10日，经一年筹备，华北电业管理局成立大会召开，中央批准李鹏兼任华北电业管理局党组书记<ref name=cankao/>。

1980年6月22日，贵州乌江渡水电厂发生水淹厂房事故。[[贵州省|贵州省]]对事故处理有意见。李鹏亲自到现场处理事故，12月12日在乌江渡水电厂工地召开的事故调查会议上做了与原先贵州省水利学会事故调查报告内容相同的总结，获得与会者一致同意。李鹏还考察了贵州猫跳水河水电厂、清镇火电厂。同年12月下旬，李鹏在成都召开西南电业管理局筹备小组会议，四川省、贵州省、云南省电力局经请示各自省委同意后，一致同意成立西南电业管理局。李鹏此行成果颇丰<ref name=cankao/>。

1980年6月22日，[[中國共產黨中央書記處總書記|中共中央总书记]][[胡耀邦|胡耀邦]]约见电力工业部的[[李锡铭|李锡铭]]、李鹏谈话时，李鹏反映了“[[4821苏修特务案|4821]]”的情况。胡耀邦要李鹏写出21人名单，李鹏当场写出19人。1980年6月23日，胡耀邦向[[中共中央组织部|中共中央组织部]]批转该名单，要求考察了解名单上人员的情况。1981年1月17日，中共中央组织部将21人情况报给胡耀邦审阅。胡耀邦看后，对其中13人的使用作出批示，其中对李鹏的批示是：“我主张坚决提起来当部长。不提一些，不能转变风气。”经胡耀邦过问，中共中央组织部于1982年1月向胡耀邦及中共中央书记处写出报告，认为“所谓‘4821苏修特务案’ 纯属冤假错案”，宣布“平反”<ref name=an/><ref name=xia/>。

1980年12月31日，刘澜波邀李鹏到家里谈到，中央已决定刘澜波等老同志任国务院顾问，电力工业部部长由李鹏担任，李鹏则以资历尚浅表示推辞。1981年2月14日，刘澜波又邀李鹏到家里，说电力工业部有几位同志联名致信中央，以太年轻为由，不同意李鹏当部长。2月19日，刘澜波电话告知李鹏，中央已决定李鹏任部长。1981年3月2日，李鹏任电力工业部部长<ref name=cankao/>。他还担任该部党组书记{{r|bird|page=513-514}}。1981年7月2日，中央在怀仁堂举行中共省、市、自治区委员会书记座谈会讨论提拔中青年干部问题，李鹏参加，[[陈云|陈云]]在会上说：“李鹏是从苏联学习回来的，搞电30多年，这次提拔当部长还有争论。我看到的名单，第一次不是李鹏，第二次才是李鹏，是刘澜波同志力争的。”<ref name=cankao/>邓小平也曾赞赏刘澜波为“党内开明人士”。李鹏也因此次提拔而名声大震{{r|李國強3|page=52}}。

1981年9月，当时中国最大的水库[[龙羊峡水库|龙羊峡水库]]遭遇150年一遇特大洪水，水量5570立方米每秒，最高水位为2494.78米，一旦溃决将影响黄河下游各大坝、青海、甘肃、宁夏沿河地区安全。李鹏亲自去灾区指挥防汛，成功保住龙羊峡、刘家峡及下游地区与[[包兰铁路|包兰铁路]]{{r|bird|page=536-550}}。此次危难的成功解决，使李鹏享誉国内外{{r|李國強3|page=62}}。此事也促使次年国务院机构改革，水利部和电力工业部合并为[[中华人民共和国水利电力部|水利电力部]]，李鹏担任水利电力部副部长、党组副书记。1982年[[中共十二大|中共十二大]]上，李鹏当选为[[中国共产党第十二届中央委员会|中共中央委员]]{{r|李國強2|page=10}}。

===代总理、总理===
{{主条目|李鹏政府}}
1983年6月，李鹏出任[[国务院副总理|国务院副总理]]，主管能源、教育和交通，分管重点建设、电子科技、环境保护、重大设备制造等{{r|李國強2|page=12}}。1985年9月24日，他被增选为中共中央政治局委员和书记处书记{{r|李國強2|page=11}}。1986年，在李鹏的主导下，中国和英国、法国、美国签订协议建造[[大亚湾核电站|大亚湾核电站]]，这也是当时中国改革开放政策来，中外合资最大项目之一。同年9月，李鹏代表中国政府在多方协议下签字{{r|李國強2|page=10}}。

1987年，[[中国共产党第十三次全国代表大会|中共十三大]]、[[中国共产党第十三届中央委员会|中国共产党第十三届中央委员会]]召开决定，邓小平为主的中共元老进一步退隐，实现更广泛的新老交替。[[中共中央政治局常委|中共中央政治局常委]]因此大换血，邓小平、[[胡耀邦|胡耀邦]]、[[李先念|李先念]]、[[陈云|陈云]]均离开常委，晋升的是李鹏、[[乔石|乔石]]、[[胡启立|胡启立]]、[[姚依林|姚依林]]，其中李鹏位列四位新常委之首，排名仅次于连任的[[赵紫阳|赵紫阳]]。此前身兼中共中央总书记和国务院总理两大要职的赵紫阳辞去总理一职，并由相对年轻的李鹏担任[[中华人民共和国国务院代理总理|代总理]]。1988年4月，正式出任[[中华人民共和国国务院总理|中华人民共和国国务院总理]]。

=== 后续生涯 ===
{{Main|三峡工程|中国载人航天工程}}
[[File:Vladimir_Putin_with_Li_Peng-2.jpg|thumb]]]]
作为水电专家的李鹏在总理任内继续领导中国的水利工程，包括浩大的[[三峡工程|三峡工程]]。1992年3月，李鹏代表国务院向[[第七届全国人民代表大会|第七届全国人民代表大会]]提交三峡工程的议案，4月表决时竟有177票反对，664票弃权，更有25人未按表决器，为历史上绝无仅有。1993年1月，[[国务院三峡工程建设委员会|国务院三峡工程建设委员会]]成立，李鹏出任主任，负责监督整个三峡工程的建设。<ref>{{cite web|url=https://www.hk01.com/議事廳/181960/李鵬之子李小鵬現身三峽大壩-陪同習近平考察|title=李鵬之子李小鵬現身三峽大壩　陪同習近平考察|publisher=香港01|date=2018-04-25|quote=1992年，七屆中國人大第五次會議以1767票贊成、177票反對、664票棄權、25人未按表決器通過《關於興建長江三峽工程的決議》。這是迄今為止中國全國人大所通過的得票率最低的議案。}}</ref>

1998年3月，李鹏在两届总理任期结束后，按宪法规定[[废除干部领导职务终身制|不能再连任]]。李鵬在[[第九届全国人民代表大会|第九届全国人民代表大会]]上当选[[全国人民代表大会常务委员会委员长|全国人民代表大会常务委员会委员长]]，他亦在1997年9月的[[中国共产党第十五届中央委员会|中共十五届一中全会]]上连任中共中央政治局常委，党内地位维持第二位，仅次于时任总书记[[江泽民|江泽民]]。

=== 卸任 ===
2003年3月，74岁的李鹏卸任全国人大常委会委员长，离开政坛。李鹏退休后撰写多部个人回忆录。第一本《众志绘宏图：李鹏三峡日记》，后陆续发表《起步到发展：李鹏核电日记》、《电力要先行：李鹏电力日记》、《立法与监督：李鹏人大日记》、《市场与调控：李鹏经济日记》、《和平发展合作：李鹏外事日记》。

据报道，李鹏还写成三十万字的《关键时刻：[[李鹏六四日記|李鹏六四日記]]》内中披露了不少内幕，据信要为自己撇清历史责任，但因为“内容敏感”而不被中共高层批准出版<ref>{{cite web|url=http://www.yzzk.com/cfm/Content_Archive.cfm?Channel=ae&Path=372480111/13ae1.cfm|title=亞洲週刊-|website=亞洲週刊|accessdate=2018-10-27}}</ref>。2010年6月，一本据称是《关键时刻》的书籍由[[赵紫阳|赵紫阳]]秘书[[鲍彤|鲍彤]]之子[[鲍朴|鲍朴]]所创办的[[香港新世纪出版社|香港新世纪出版社]]出版。鲍朴通过中间渠道得到此书稿件，并认为其真实的可能性非常大<ref>{{cite web|url=http://www1.voanews.com/chinese/news/china/diary-lipeng-20100606-95722019.html|title=美国之音中文网 您可靠的信息来源|publisher=|accessdate=2018-10-27}}</ref>。
