'''李佐玉'''（{{bd|1912年||1992年|8月4日}}）[[江西省|江西省]][[兴国县|兴国县]]人，[[中国人民解放军开国少将|中国人民解放军开国少将]]。

==生平==
1928年参加赣西南农民暴动。1932年参加[[中国工农红军|中国工农红军]]，由[[中国共产主义青年团|中国共产主义青年团]]团员转为[[中国共产党|中国共产党]]党员。历任特派员、团政委、师政委、[[山西省军区|山西军区]]副政委<ref name=cpc/>。1955年被授予少将军衔<ref name=cpc/><ref>{{cite news|title=揭秘：中共首批开国将军的级别是怎样划分的？ |url=http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |accessdate=2012-06-15 |newspaper=凤凰网（转自摘自《党史博览》2010年第5期） |date=2010-06-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20120620091504/http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |archivedate=2012-06-20 }}</ref>。[[第一次国共内战|第一次国共内战]]时期，参加了[[中央苏区|中央苏区]]第四、五次反“围剿”斗争以及[[二万五千里长征|二万五千里长征]]；[[抗日战争|抗日战争]]、[[第二次国共内战|第二次国共内战]]期间，率部在[[内蒙古|内蒙古]][[大青山|大青山]]一带战斗；[[中华人民共和国|中华人民共和国]]成立后，为民兵、预备役工作的发展作出贡献<ref name=cpc>{{Cite web |url =http://dangshi.people.com.cn/GB/165617/165618/166491/167909/12091182.html  |title = 李佐玉 |author =  |date =  |publisher =  中国共产党新闻网|language =  |accessdate =2017-04-30  |quote =  }}</ref><ref>李佐玉同志逝世，解放军报1992年9月4日，第04版</ref>。

1992年8月4日，李佐玉在[[太原|太原]]病逝，享年80岁<ref name=cpc/>。
