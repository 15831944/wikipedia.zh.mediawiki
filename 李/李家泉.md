'''李家泉'''（{{bd|1927年|2月||}}）[[安徽|安徽]][[庐江|庐江]]人，[[中华人民共和国|中华人民共和国]][[台湾|台湾]]研究专家。

==生平==
李家泉是庐江县同大镇南闸村（邓渡）人。[[中国共产党|中国共产党]]党员。1949年参加工作后，长期在粮食系统工作，历任支前县兵站站长、县粮库主任、华东粮食局科员、[[中华人民共和国粮食部|中华人民共和国粮食部]]业务秘书。1959年，进入[[中共中央党校|中共中央高级党校]]理论班学习。1973年起，一直在从事台湾研究，历任[[中国社会科学院台湾研究所|中国社会科学院台湾研究所]]业务组长、研究室主任、副所长、研究员。自1980年代中期起，开始将其台湾研究的领域扩展至政治、社会等各方面。1990年代起，先后兼任[[全国台湾研究会|全国台湾研究会]]常务理事、北京台湾经济研究中心理事长、北京外国问题研究会高级顾问，[[北京大学|北京大学]]、[[北京联合大学|北京联合大学]]、[[国务院侨务办公室侨务干部学校|国务院侨务办公室侨务干部学校]]、[[中央社会主义学院|中央社会主义学院]]等院校的兼职教授，并且是国务院有关对台、涉台部门的专家组成员。曾参与《台湾问题与中国的统一》白皮书等的起草工作<ref name=lj>{{Cite web|url = http://renwen.ahljnews.com/2012/1019/853.shtml|title = 李家泉|author = |date = 2012-10-19|publisher = 庐江新闻网|language = |accessdate = |quote = }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url = http://news.ifeng.com/taiwan/special/tw2012/fangtan3.shtml |title =2012台湾大选访谈系列之三  |author =  |date =  |publisher =凤凰网  |language =  |accessdate = 2017-10-13 |quote =  }}</ref><ref name=jianli>{{Cite web |url =http://www.taiwan.cn/plzhx/zhjzhl/tyzhj/ljq/index.htm  |title = 李家泉教授简历 |author =  |date =  |publisher = 中国台湾网 |language =  |accessdate = 2017-10-13 |quote =  }}</ref>。
