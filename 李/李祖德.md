'''李祖德'''（'''Tsu-Der Lee'''，{{bd|1950年||||catIdx=TD李}}）生於[[台北市|台北市]]，畢業於[[臺北醫學大學|臺北醫學大學]]牙醫學系，由專業醫師轉為經營者，後又投身於創投業、當選北醫董事長、引進瑞士醫材產業等<ref>{{Cite web |url= http://store.gvm.com.tw/article_content_25197.html |title= 北醫大董事長李祖德的人生|author=高希均 |date= |publisher= 遠見雜誌2014年4月號 第334期 |language= |accessdate= |quote= }}</ref><ref>{{Cite web |url= http://www.cw.com.tw/article/article.action?id=5057441 |title= 要當領頭羊，也要當獨行狼|author=熊毅晰 |date= |publisher= 天下雜誌545期  |language= |accessdate= |quote= }}</ref>

==簡歷==
*1982年開設台灣首家連鎖式牙醫診所
*1992年擔任[[趙少康|趙少康]]台北縣(現[[新北市|新北市]])[[立法委員|立法委員]]之競選總幹事，協助創下地方選舉史上罕見的高票紀錄
*1995年受到創投教父[[徐大麟|徐大麟]]賞識，進入創投界。擔任漢鼎亞太中國區首席顧問
*1995年以北醫校友身分當選[[臺北醫學大學|臺北醫學大學]]董事
*1996年擔任[[河北省|河北省]][[唐山市|唐山市]]台商會創會會長
*1999年擔任漢鼎亞太中國區總經理，同年擢升為漢鼎公司副董事長迄今
*1999年擔任北京美大[[星巴克|星巴克]]咖啡有限公司董事長
*2006年獲頒北醫傑出校友企業經營獎<ref>{{Cite web |url= http://203.71.86.81/Personages/%E5%8C%97%E9%86%AB%E4%BA%BA%E9%A2%A8%E9%9B%B2%E9%8C%84/%E6%9D%8E%E7%A5%96%E5%BE%B7.aspx|title= 跨國際專業經理人-擅長資源整合北醫人|author= |date= |publisher= 北醫風雲錄|language= |accessdate= |quote= }}</ref>
*2007年接任財團法人[[臺北醫學大學|臺北醫學大學]]董事長<ref>{{Cite web |url= http://history.tmu.edu.tw/files/15-1005-3012,c664-1.php|title= 第十四屆~第十五屆 李祖德 任期：2007 ∼ 2014|author= |date= |publisher= 北醫風雲錄|language= |accessdate= |quote= }}</ref>
*2011年赴瑞士洽談Swissray Medical AG(全球第四大高階影像設備廠)及其企業全部股權併購案<ref>{{Cite web |url= http://www.chinatimes.com/newspapers/20140323000836-260301|title= 併購SWISSRAY 打開醫材市場大門|author=張雅雲 |date= |publisher= 旺報/台商周報|language= |accessdate= |quote= }}</ref>
*2013年擔任環瑞醫投資控股股份有限公司董事長，同年環瑞醫投資控股登錄興櫃，成為首家在台掛牌上市的瑞士品牌<ref>{{Cite web|url=http://www.wealth.com.tw/article_in.aspx?nid=650|title=整頓北醫18年　下個台灣生技的舵手？ 北醫推手李祖德　再戰生醫|author=洪綾襄|date=|publisher=財訊雙週刊 第426期|language=|accessdate=|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20160914152453/http://www.wealth.com.tw/article_in.aspx?nid=650|archivedate=2016年9月14日|df=}}</ref>
*2014年12月獲頒'''中華民國科技管理學會'''第16屆科技管理獎個人獎<ref>{{Cite web |url= http://www.csmot.org.tw/Content/264 |title= 第16屆－科技管理學會院士|author= |date= |publisher= 中華民國科技管理學會|language= |accessdate= |quote= }}</ref>
*2016年擔任鑽石生技投資股份有限公司副董事長
