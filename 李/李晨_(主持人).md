'''李晨'''{{BD|1982年|8月28日|||catIdx=L李}}，出生于[[上海市|上海市]]，中国男主持人，演员。

== 生平 ==
1982年，李晨出生在[[上海市|上海市]]。祖籍浙江宁波<ref name="Li hometown">{{cite web |url=http://hzdaily.hangzhou.com.cn/dskb/html/2007-09/07/content_360737.htm |title=“我的音乐地盘”群星演唱会胡可李晨携手主持 |accessdate=2014-06-28 |publisher=杭州日报 |work=徐宁 |date=2007-09-07 |language =zh-cn}}</ref>。

== 个人生活 ==
李晨的妻子名叫赵琼，2008年两人结婚。由于和男演员[[李晨_(演员)|李晨]]同名同姓，曾一度被传出与[[张馨予|张馨予]]的绯闻，导致众多新闻媒体出来解围，而妻子赵琼更是提防其出轨。<ref>{{cite web |url=http://www.chinanews.com/yl/2012/09-04/4156597.shtml |title=主持人李晨被误认为告白张馨予 老婆提醒其注意点 |accessdate= 2012年9月4日|publisher=中新网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/s/m/2012-09-03/00263729581.shtml |title=主持人李晨无奈被误认与张馨予婚外情 |accessdate= 2012年9月3日|publisher= 新浪网|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://www.chinanews.com/yl/2012/09-04/4156328.shtml |title=主持人李晨否认"表白"张馨予：名字相同真不是我 |accessdate=2012年9月4日 |publisher=中新网 |work= |language =zh-cn}}</ref>2014年，其子李晋毅出生。
