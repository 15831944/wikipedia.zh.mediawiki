'''李梓敬'''（{{lang-en|Dominic Lee Tsz-king}}，1984年1月22日－），[[香港|香港]]政商界人物。現為[[香港|香港]][[深水埗區議會|深水埗區議會]][[又一村|又一村]]選區區議員及[[自由黨_(香港)|自由黨]]成員。職業為電子零件企業的執行董事。已婚，育有兩子。

== 簡介 ==
李梓敬曾就讀[[喇沙小學|喇沙小學]]、[[拔萃小學|拔萃小學]]、[[拔萃男書院|拔萃男書院]]以及[[李寶椿聯合世界書院|李寶椿聯合世界書院]]，在[[萊斯大學|萊斯大學]]完成經濟學學士學位。李梓敬在大學時期已經開始創業，設立網上平台買賣教科書，並在畢業前將企業賣給雷曼兄弟前高層。<ref>{{cite web|url=https://www.prweb.com/releases/2008/09/prweb1280604.htm|title=Fortune 100 Businessman Nick Themelis on a New Mission}}</ref> <ref>{{cite web|url=http://www.leedominic.com/95/site-sold-to-former-fortune-100-executive-nick-themelis|title=李梓敬博客}}</ref> 之後，他再次建立不同的網絡銷售平台，售賣廚具<ref>{{cite web|url=http://dailynews.sina.com/bg/chn/chnpolitics/mingpao/20110806/14212663909.html|title=「小中産」遷出豪宅 為服務基層}}</ref>，寵物產品，嬰兒產品等等。<ref>{{cite web|url=http://www.leedominic.com/page/5|title=李梓敬博客}}</ref> 在2011年，李梓敬加入家族電子零件生意，負責網上銷售以及網絡營銷。<ref>{{cite web|url=http://nextplus.nextmedia.com/news/spot/20160819/419453|title=貼地中產？》父母擁5公司 李梓敬唔逼囝囝去Playgroup}}</ref>

政治方面，李梓敬大學時期曾為2004年[[美國|美國]][[民主黨_(美國)|民主黨]]總統候選人[[約翰·克里|約翰·克里]]助選，及為民主黨眾議員{{link-en|阿爾‧格林|Al Green (politician)}}，當見習生。回到香港後，在2009加入自由黨，2011成為自由黨青年團的主席。在[[2015年香港區議會選舉|2015年香港區議會選舉]]中，李梓敬在所居住的又一村選區參選，以2277票大比數擊敗[[社民連|社民連]]候選人周嘉發，當選深水埗區議員。<ref>{{cite web|url=https://www.elections.gov.hk/dc2015/chi/results_sham_shui_po.html?1528726963501|title=2015區議會選舉結果(深水埗)}}</ref>

在[[2016年香港立法會選舉|2016年香港立法會選舉]]中，李梓敬與[[田北俊|田北俊]]組織聯合名單，代表自由黨參選[[新界東選區|新界東選區]]，最終以20031票落選。

李梓敬現在於[[香港城市大学|香港城市大学]]攻讀法律博士課程。<ref>{{cite web|url=https://news.mingpao.com/pns/dailynews/web_tc/article/20170905/s00002/1504548346519|title=李梓敬讀「法律博士」 梁美芬教第一堂|publisher=明報|accessdate=2018-06-11|date=2017-09-05}}</ref>

== 政治立場 ==
李梓敬的政治立場是[[右翼|右翼]][[保守派|保守派]]。<ref>{{cite web|url=http://forum.hkej.com/node/105036|title=李梓敬：打造唯一右翼反濫用福利反同志活動|publisher=信報|date=2013-08-29}}</ref> 他反對[[同性婚姻|同性婚姻]]及[[性別認同|性別承認]]立法、要求為失業[[綜援|綜援]]年期設限<ref>{{cite web|url=http://podcast.rthk.hk/podcast/item_epi.php?pid=390&lang=zh-CN&id=27300|title=左右紅藍綠：改革失業綜援制度|date=2013-01-09}}</ref>、反對[[香港新移民|香港新移民]]可以享有社會褔利（包括綜援）<ref>{{cite web|url=https://hk.news.appledaily.com/local/daily/article/20131221/18560488|title=
隔牆有耳：終院一槌，論新移民綜援分兩極，自由黨李梓敬：非納稅人責任|date=2013-12-21}}</ref>、要求遣返濫用[[酷刑聲請|酷刑聲請]]的假難民。<ref>{{cite web|url=https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/57588/%E6%9D%8E%E6%A2%93%E6%95%AC-%E5%81%87%E9%9B%A3%E6%B0%91%E7%8A%AF%E7%BD%AA%E7%8E%87%E5%81%8F%E9%AB%98-%E6%87%89%E8%80%83%E6%85%AE%E8%A8%AD%E7%AB%8B%E7%A6%81%E9%96%89%E7%87%9F|title=李梓敬：假難民犯罪率偏高　應考慮設立禁閉營|date=2016-12-02}}</ref>

== 爭議 ==
李梓敬反對[[工會|工會]]介入工潮及勞資談判，在2013年[[葵涌貨櫃碼頭工潮|葵涌貨櫃碼頭工潮]]期間，他批評[[職工盟|職工盟]][[李卓人|李卓人]]介入工潮，使事件政治化。李梓敬在《城市論壇》上批評職工盟李卓人「騎劫」工運，爆出「工人想搞工運時，唔好畀工會介入」金句，成為一時「佳」話，有網民以「港人想爭取自由，唔好畀自由黨介入」回應。

李梓敬也強烈反對香港實行[[全民退休保障|全民退休保障]]制度，2014年4月29日，李梓敬出席立法會退休保障事宜小組委員會公聽會時，反對設立全民退休保障制度，表示全民退保的方：「我們的父母應該由我們去供養，不是要求其他人去供養。」他隨即被出席同一會議的關注長者權益大聯盟代表余婆婆鬧爆「仆街」，批評不應假設所有人都必然有能力供養父母，並「爆粗」表示：「你老竇老母就是人、人家老竇老母不是人嗎？你以為你生在香港就天生富貴？沒有我們這些人（長者）捱工廠、建樓和其他設施，你有這樣富貴？你在閪門（陰戶）爬出來時就這樣大的嗎？」余婆婆在立法會議事廳內「爆粗」，雖被主持會議的議員[[張國柱_(香港)|張國柱]]多次警告，但就博得部分出席人士掌聲，至於坐在婆婆前排的李梓敬，只能面露尷尬苦笑。他其後為在立法會公聽會玩手機致歉，卻表明在全民退保議題上面堅持立場<ref>[http://hk.apple.nextmedia.com/realtime/news/20140430/52431743 立會玩轉珠被指不尊重 李梓敬昨死撐今致歉] 香港蘋果日報</ref>。

== 言論 ==
2016年4月21日，他向[[平等機會委員會|平等機會委員會]]下戰書，反對平機會欲禁止酒吧舉行女士之夜活動，指平機會濫用[[性別歧視|性別歧視]]條例。同時，認為不應把營銷手法當作性別歧視，更因此而挑戰平機會舉行ladies night活動<ref>[http://hk.on.cc/hk/bkn/cnt/news/20160418/bkn-20160418114610007-0418_00822_001.html?eventsection=hk_news&eventid=402882b1540d90a301540dd4a29f079b Ladies Night被裁歧視　政黨轟周一嶽無恥] 香港東方日報</ref>。

2016年4月28日，在討論「如何在中國因素影響下，保持香港的自主」的環節中，與民主黨的區議員[[區諾軒|區諾軒]]對質，並提到民主黨支持2012年政改方案時與中聯辦對話一事，諷刺民主黨因選舉才與中央溝通<ref>[http://hk.on.cc/hk/bkn/cnt/news/20160428/bkn-20160428234823964-0428_00822_001.html 政Whats噏：李梓敬翻民主黨舊帳激嬲區諾軒] 香港東方日報</ref>。

2016年5月7日，隨着反香港假難民的聲音亦增多，自由黨在[[中環|中環]]舉行集會，再遊行至政府總部，有200人參與，他們沿途高叫「打倒假難民，促設禁閉營」<ref>[http://hk.on.cc/hk/bkn/cnt/news/20160507/bkn-20160507153546235-0507_00822_001.html 假難民禍患嚴重　200人遊行促設禁閉營] 香港東方日報</ref>。

2016年5月8日，在有線節目《SUNDAY有理講》節目中，[[青年新政|青年新政]]的[[梁頌恆|梁頌恆]]指因為他沒有簽署[[中英聯合聲明|中英聯合聲明]]，所以不需要理解和尊重[[香港特別行政區基本法|基本法]]。他即反問梁：「如果你是 (土生土長) [[美國公民|美國公民]]，乜你有份簽[[美國憲法|美國憲法]]架咩？」<ref>[https://www.facebook.com/leetszking/videos/1362071670484998/ 李梓敬facebook片段]</ref>

2016年5月22日，由於時值[[全國人大委員長|全國人大委員長]][[張德江|張德江]]訪港，並會與部分泛民[[香港立法會|香港立法會]]議員會面，在論壇的發問環節時，他向出席[[城市論壇|城市論壇]]的[[何秀蘭|何秀蘭]]提出質疑，認為部分泛民議員與領導人會面只會假抗爭，稱「泛民同中央溝通，我哋係樂見嘅。不過，如果你要溝通，就正正經經溝通；如果你要抗爭，就正正經經抗爭。好似何秀蘭呢啲政客，又要威又要戴頭盔，為咗抽水，戴條雨傘吊墜就當抗爭咗，做乜嘢啫？你以為咁做，[[張德江|張德江]]會驚咗你？你以為咁做，香港市民就會投票俾你？如果你咁都算抗爭嘅話，不如你下次著件黃色底衫當抗爭咗算把啦。」 <ref>[https://www.facebook.com/leetszking/videos/1371955699496595/ 李梓敬facebook片段]</ref>
