'''李榮安'''教授<ref>{{cite web |url=http://home.ied.edu.hk/~wolee/ |title=存档副本 |accessdate=2008-01-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20080102154336/http://home.ied.edu.hk/~wolee/ |archivedate=2008-01-02 }}</ref>{{MH}}
於1998年任職[[香港教育學院|香港教育學院]]，歷任教育基礎學院首任院長、教育政策與行政學系主任、社會科學系主任以及公民教育中心主任。2005年，李教授受聘於[[澳洲悉尼大學|澳洲悉尼大學]]，出任教育與社工學院教授及人文及社會科學學院國際發展總監。1998年加入教院前，他曾是香港大學教育學院副院長及比較教育研究中心首任總監。2003年，他獲香港政府頒授榮譽勳章，<ref>http://www.info.gov.hk/gia/general/200310/11/1011237.htm</ref>並獲中華教育藝術研究會與中國教育發展論壇向他授予「鑄魂育人」銅獎。<ref>http://paper.wenweipo.com/2007/02/09/HK0702090073.htm</ref>
2007年，他接替不獲續約的[[莫禮時|莫禮時]]出任[[香港教育學院|香港教育學院]]署理校長，直至2007年12月。其後，他繼續在香港教育學院擔任首席副校長與比較教育學講座教授。<ref>{{cite web |url=http://www.ied.edu.hk/about_hkied/html/management_c.html |title=存档副本 |accessdate=2008-01-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20080108050141/http://www.ied.edu.hk/about_hkied/html/management_c.html |archivedate=2008-01-08 }}</ref> 同年，他獲選編入南京師範大學德育學人庫。<ref>{{cite web |url=http://202.119.108.48/dyxr/Default.aspx?id='22' |title=存档副本 |accessdate=2008-08-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304210003/http://202.119.108.48/dyxr/Default.aspx?id='22' |archivedate=2016-03-04 }}</ref>2010年，他獲委任為新加坡國立教育學院教育研究院院長暨教育學教授。<ref>http://www.sginsight.com/xjp/index.php?id=3908</ref>同年，他獲選為世界比較教育聯合議會(World Council of Comparative Education Societies)會長。該會有40年歷史，以及40個來自世界各地的比較教育學會成員。他也是該會40年來首任華裔會長。<ref>http://teachersdreamfund.hk/基金-架構/評審委員/64-李榮安教授.html</ref>同年，他獲香港國際創價學會頒授香港國際創價學會獎。<ref>http://www.hksgi.org/cht/culture/seminar/</ref>2014年，李榮安獲委任香港公開大學副校長(行政與發展)及比較教育學講座教授。<ref>http://news.takungpao.com.hk/paper/q/2014/0909/2718469.html</ref>
