'''李鹏政府'''指时任[[中华人民共和国|中华人民共和国]][[国务院总理|国务院总理]]的[[李鹏|李鹏]]在1988年4月[[第七届全国人民代表大会|第七届全国人民代表大会]]任期开始至1998年3月[[第八届全国人民代表大会|第八届全国人民代表大会]]任期结束期间，领导的连续两届[[中华人民共和国国务院|中央人民政府]]包括[[第七届国务院|第七届国务院]]和[[第八届国务院|第八届国务院]]，其[[中华人民共和国国务院组成人员|国务院组成人员]]被外界称为'''李鹏内阁'''。

1988年4月12日举行的[[第七届全国人大第一次会议|第七届全国人大第一次会议]]上，[[国务院代总理|国务院代总理]]李鹏被正式任命接替升任[[中共中央总书记|中共中央总书记]]的[[赵紫阳|赵紫阳]]出任国务院总理，成为[[中华人民共和国|中华人民共和国]]的[[政府首脑|政府首脑]]<ref>{{Cite web |url= http://cn.theorychina.org/rdwx/dscdh/ |title= 第七届全国人民代表大会 |author= |date = |publisher= 理论中国（[[中央编译局|中央编译局]]主办） |language= zh}}</ref>。其後，李鹏根据《[[中华人民共和国宪法|中华人民共和国宪法]]》，向[[全国人民代表大会|全国人民代表大会]]提出[[中华人民共和国国务院副总理|国务院副总理]]、[[国务委员|国务委员]]及[[国务院组成部门|各组成部门]]负责人等其他国务院组成人员的人选，获全国人大批准後正式组成[[中华人民共和国国务院_(第七届)|其第一届内阁]]。1993年3月27日举行的[[第八届全国人大第一次会议|第八届全国人大第一次会议]]上，李鹏连任国务院总理，组成[[中华人民共和国国务院_(第八届)|其第二届内阁]]<ref>{{Cite web |url= http://news.xinhuanet.com/ziliao/2002-02/20/content_283417.htm |title= 第八届全国人大历次会议 |author = |date= 2002-02-20 |publisher= [[新华网|新华网]] |language= zh}}</ref>。
