'''李进'''{{BD|1949年|3月}}，[[江苏|江苏]][[武进|武进]]人，[[教授|教授]]，[[博士生导师|博士生导师]]，[[上海市|上海市]]第十届、第十一届[[政协委员|政协委员]]。现任[[上海杉达学院|上海杉达学院]]第五任校长、[[党委|党委]]副书记。
==任职==
2000年10月任[[上海东沪职业技术学院|上海东沪职业技术学院]]院长、[[党委|党委]]副书记，2001年11月任[[上海第二工业大学|上海第二工业大学]]副校长、2004年4月任党委书记，2006年1月任[[上海师范大学|上海师范大学]]校长、党委常委，2011年8月起{{R|BDI5IMVG00964KN4}}任[[上海杉达学院|上海杉达学院]]第五任校长、党委副书记。<ref name=shmbjy>{{cite web |title=李进副会长简介 |url=http://www.shmbjy.org/item-detail.aspx?NewsID=179&keywords=%BB%E1%B3%A4 |website=上海市民办教育协会 |accessdate=2018-06-26 |date=2012-07-04}}</ref><ref>{{cite web |title=上海师范大学李进同志先进事迹 |url=http://www.shmec.gov.cn/web/hdpt/hero_detail.php?id=35 |website=上海教育 |publisher=上海市教育委员会 |accessdate=2018-06-26}}</ref><ref>[http://www.sandau.edu.cn/xww/71/46/c1217a28998/page.htm 上海学校后勤：一位大学校长的心声——上海杉达学院李进校长访谈实录]， 上海市学校后勤协会 褚金凤，2018-03-19</ref>兼任中国高等教育学会产学研合作教育分会副会长、教育部全国高等院校设置委员会委员、教育部国家示范性高等职业院校建设工作协作委员会会长、全国高职高专校长联席会议名誉主席（原主席<ref>[http://www.ixueshu.com/document/ae6c70912fb156c1.html 热烈祝贺云南省高等教育社会高职高专教育研究会成立. 昆明冶金高等专科学校学报, 2005(5):2+85.]</ref>）、上海产学合作教育协会副会长兼秘书长，上海职教协会副会长。上海市社会科学界联合会副主席{{R|shmbjy}}，上海师范大学职业技术教育博士点学科带头人。<ref>*[https://www.shobserver.com/news/detail?id=52496 遇见未来的自己②|上海杉达学院校长李进：成就体面就业的“城市新人”]，上观新闻</ref>

==荣誉==
2016年1月17日，李进获选“2015年学生喜爱的大学校长”（[[中国高等教育学会|中国高等教育学会]]和[[中国青年报|中国青年报]]社主办）<ref>{{cite web |title=李进获选“2015年学生喜爱的大学校长” |url=http://www.shmbjy.org/item-detail.aspx?newsid=5844 |website=上海市民办教育协会 |accessdate=2018-06-26 |date=2016-01-19}}</ref><ref name=BDI5IMVG00964KN4>{{cite news |title=2015年“学生喜爱的大学校长”颁奖典礼暨论坛在京举行 |url=http://help.3g.163.com/16/0117/18/BDI5IMVG00964KN4.html |accessdate=2018-06-26 |work=中国青年报 |publisher=网易 |date=2016-01-17}}</ref>。
