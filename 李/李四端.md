'''李四端'''（{{bd|1956年|8月4日|catIdx=L李}}），[[台灣|台灣]]資深[[新聞工作者|新聞工作者]]。現任[[東森新聞台|東森新聞台]][[李四端的雲端世界|李四端的雲端世界]]節目主持人。同時於[[銘傳大學|銘傳大學]]傳播學院擔任講師&[[大雲文創|大雲文創]]《[[大雲有故事|大雲有故事]]》網路直播節目主持人。

== 簡歷 ==
*早年就讀[[國立政治大學|國立政治大學]][[新聞學|新聞學]]系，兼任該系實習報紙《柵美報導》[[記者|記者]]。
*國立政治大學新聞學系畢業後，李四端擔任《中央日報》記者。
*1983年，李四端經資深主播[[盛竹如|盛竹如]]等[[面試|面試]]，考進[[臺灣電視公司|臺灣電視公司]]（台視）[[台視新聞|新聞部]]，出任記者，並在短短幾年間升為採訪組組長，曾出任《台視新聞世界報導》等節目主播，其後更成為臺視的首席主播。
*1989年4月起，李四端以台視新聞部採訪組組長一職接任《台視晚間新聞》主播，在該年報導[[六四事件|六四事件]]而聲名大噪，也成為該台的收視保證。<ref>[[中華民國電視學會|中華民國電視學會]]電視年鑑編纂委員會 編纂，《[[中華民國電視年鑑|中華民國電視年鑑]]第六輯：民國七十七年至七十八年》，中華民國電視學會1990年6月30日-{出}-版，第18頁。</ref>
*1997年3月1日，時任台視新聞部經理的李四端因記者[[隋安德|隋安德]]在[[福建|福建]]被[[中華人民共和國人民警察|中華人民共和國人民警察]]拘捕一事請辭獲-{准}-，李四端請辭案自同年3月20日起生效，由副總經理[[顧安生|顧安生]]接任台視新聞部經理。<ref>中華民國電視學會電視年鑑編纂委員 編纂，《[[中華民國無線電視年鑑|中華民國無線電視年鑑]]第十輯：民國八十五年至八十六年》，中華民國電視學會1998年6月30日-{出}-版，第42至43頁。</ref>
*李四端轉戰[[無線衛星電視台|無線衛星電視台]]（[[TVBS|TVBS]]），出任TVBS董事長[[邱復生|邱復生]]特別助理與[[TVBS-N|TVBS-N]]《新聞百分百》節目主持人<ref>康添財 撰文，李四端的第二春在TVBS，《今周刊》第30期。</ref>。
*不久，李四端再投入無線新聞頻道的戰場，加盟[[中華電視公司|中華電視公司]]（華視），以華視[[總經理|總經理]]特別助理身份-{出}-任《華視晚間新聞》主播。期間李四端曾跨刀主持[[八大第一台|八大第一台]]談話性節目《四-{面}-八方》與[[超級電視台|超級電視台]]新聞節目《九點整新聞》，並一度取得亮麗成績。
*2004年7月26日，李四端與華視的合約到期，李四端辭去華視一切職位。在李四端舊部之一的TVBS新聞部經理[[潘祖蔭|潘祖蔭]]力邀下，李四端於2004年9月以「新聞總顧問」身份加盟TVBS，並在2006年初獲TVBS總經理[[李濤_(台灣)|李濤]]推薦升任副總經理。而李四端加盟TVBS亦曾引起不少震動：例如，TVBS主播[[蘇逸洪|蘇逸洪]]轉戰[[台視新聞|台視新聞]]，自己不諱言與李四端加盟TVBS有關。另外，原TVBS當家主播[[方念華|方念華]]後來亦離職轉投[[中天新聞台|中天新聞台]]，被指「一山不能藏二虎」；但她事後強調，自己緊隨李工作多年，轉職跟他加盟沒有關係，並表示將「永遠尊李四端為師」。
*2009年2月19日，李四端重返華視，擔任節目主持人。2009年10月26日，[[公共電視文化事業基金會|公共電視文化事業基金會]]（公視）推出李四端主持的[[青少年|青少年]]談話性節目《[[爸媽冏很大|爸媽冏很大]]》。
*2011年，加入[[東森新聞台|東森新聞台]]擔任《[[就是要新聞|就是要新聞]]》主持人；該節目於2012年改版為《[[李四端的雲端世界|李四端的雲端世界]]》，並擔任主持人。同時於銘傳大學傳播學院擔任兼課講師。
*2017年7月10日，李四端與李艷秋、詹慶齡、陳藹玲等人轉戰[[網路直播|網路直播]]，參與創立[[大雲文創|大雲文創]]，並擔任網路直播節目《[[大雲有故事|大雲有故事]]》主持人。
*2017年11月26日，李四端擔任[[WinTV|WinTV]]名人外景專訪節目《[[四端紅人會|四端紅人會]]》主持人。

== 特色 ==
李四端播報新聞風格與後起新聞主播最大差異在於，他不按照[[讀稿機|讀稿機]]唸稿，而會消化內容後「說新聞」，因此有時會因快速而吃螺絲；不過，正因他所扮演的不只是「讀稿機」的角色，也建立了新聞的信賴度，因而在擔任主播時期，收視率常居全國之冠。

== 同期競爭對手 ==
*[[沈春華|沈春華]]（前[[中視新聞|中視新聞]]主播兼任[[中視新聞台|中視新聞台]]總監）
*[[李艷秋_(主播)|李艷秋]]（前[[華視新聞|華視新聞]]主播，TVBS《[[新聞夜總會|新聞夜總會]]》播出近12年的主持人）
與在台視任職時的李四端三人，被並稱為1990年代初期[[老三台|老三台]]的代表性主播。

== 軼聞 ==
*就讀政大新聞系的在學期間某日，李四端和女友從國立政治大學搭公車去看電影，半路上公車發生碰撞意外，李四端立刻下車、現場採訪拍照，為當期的《柵美報導》跑了一條頭版大獨家；這篇報導事後還被《[[中央日報|中央日報]]》轉載，是李四端記者生涯的得意作品之一<ref>李郁寧、賴佳寧，與時代脈搏一起躍動：新聞系刊物變遷史，國立政治大學傳播學院1990年《薪傳》第94至99頁。</ref>。
