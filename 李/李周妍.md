'''李周妍'''<ref>[http://imgur.com/a/H0Kbg 李周妍簽名及經紀公司Better娛樂公布正名。] Better Ent {{ko}}</ref>（{{lang-ko|'''이주연'''}}，{{lang-en|'''Lee Ju-Yeon'''}}，{{Bd|1987年|3月19日|catIdx=L}}），未正名前音譯為'''李珠妍'''。[[韓國|韓國]]女子團體[[After_School|After School]]的第一期成員<ref>{{Cite web | url=http://www.ieasy.tv/news/detail/2313.html | title=After School公开最后四位成员宣传照 | date=2011年4月21日 | publisher=ieasy.tv | language=zh-cn }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，同時也是分隊[[After_School_Blue|After School Blue]]的隊長。2014年12月31日李周妍與經紀公司[[Pledis娛樂|Pledis娛樂]]合約到期後不再續約，就此退出[[After_School|After School]]單飛。

==經歷==
2002年李周妍被韓國[[Daum|Daum Cafe]]的五大臉讚網站推舉爲「臉讚」之一，因貌美受到關注後，進而獲得廠商青睞邀約拍廣告。2008年李周妍以[[孫淡妃|孫淡妃]]友人身分出席MBC《介紹明星的朋友》節目，亮眼的外型再次受到人們的矚目。<ref>{{Cite web |url= http://www.hkn24.com/news/articleView.html?idxno=21121|title= 孫淡妃的朋友 李周妍臉讚出生 |date= 2008年12月29日 |publisher= 韓國健康新聞 |language= ko }}</ref>

2009年1月17日李周妍以女子團體[[After_School|After School]]歌手身份出道，隔年即參與知名農村實境綜藝節目《青春不敗》第一季，是New G7固定成員之一。<ref>{{Cite web |url= http://www.asiae.co.kr/news/view.htm?sec=ent2&idxno=2010061823183638523 |title= 《青春不敗》加入新成員李周妍、Victoria、 Sori |date= 2010年6月19日 |publisher= 亞洲經濟日報 |language= ko }}</ref><ref>{{Cite web |url= http://article.joins.com/news/article/article.asp?total_id=4561357&ctg=1502 |title= 具荷拉和李周妍仍是關係最尷尬一對 |date= 2010年10月23日 |publisher= 中央日報 |language= ko }}</ref>接著與[[李太成|李太成]]出演音樂真人秀節目《那女人作詞，那男人作曲2》，節目安排男女主角一起去旅行尋求靈感，共同完成製作一首歌，拍攝期間曾到[[台灣|台灣]]取景，走訪[[平溪區|平溪]]、[[北投區|北投]]及[[淡水區|淡水]]。<ref>{{Cite web |url= http://history.n.yam.com/liontravel/entertain/20120828/20120828757512.html |title= 李泰成 攜手李周妍淡水譜戀曲 |date= 2012年8月28日 |publisher= 欣傳媒 |language= zh-tw }}</ref>

李周妍自己雖然是歌手身份，但演戲的願望一直沒有放棄，2010年開始挑戰電視劇，出演《[[笑吧！東海|笑吧！東海]]》，伴隨著與其他演員的共同努力，該劇的收視率一度達到44.8%，而李周妍在劇中的演技也逐漸被觀眾們認可。<ref>{{Citeweb |url= http://www.asiae.co.kr/news/view.htm?idxno=2011051319301717153 |title= 李周妍演出《笑吧！東海》的快樂時光 |date= 2011年5月13日 |publisher= 亞洲經濟日報 |language= ko }}</ref> 拍完《[[笑吧！東海|笑吧！東海]]》隔年接到《[[田禹治_(電視劇)|田禹治]]》邀約，作為首次嘗試古裝劇題材，極有特別意義。李周妍也表示拍攝《[[田禹治_(電視劇)|田禹治]]》得到非常寶貴的經驗，未來還想嘗試不同類型的角色。<ref>{{Cite web |url= http://www.asiae.co.kr/news/view.htm?idxno=2013022112525588944 |title= 李周妍專訪 作為演員身份的感想 |date= 2013年2月21日 |publisher= 亞洲經濟日報 |language= ko }}</ref><ref>{{Cite web |url= http://osen.mt.co.kr/article/G1109544499 |title= 專訪李周妍 角色扮演面對沉重的精神壓力 |date= 2013年2月19日 |publisher= OSEN |language= ko }}</ref><ref> {{Cite web |url= http://www.newsen.com/news_view.php?uid=201302151218400910 |title= 李周妍雖挨了批評但不逃避 |date= 2013年2月15日 |publisher= newsen |language= ko }}</ref>其後李周妍不僅接連拍了電視劇，更踏足電影及舞台劇演出，展現全方位發展的決心。

2014年12月31日李周妍與經紀公司[[Pledis娛樂|Pledis娛樂]]的專屬合約結束，決定不再續約並退出[[After_School|After School]]，隨後於2015年1月11日舉行畢業典禮和迷你粉絲見面會。<ref>{{Cite web |url= http://sports.donga.com/3/02/20141222/68708128/2 |title= 李周妍與經紀公司合約到期退出組合 |date= 2014年12月22日 |publisher= 東亞日報 |language= ko }}</ref>離開[[Pledis娛樂|Pledis娛樂]]的李周妍，2015年1月19日宣布加盟新東家[[Better娛樂|Better娛樂]]，以個人形式重新出發，朝著演戲等多種領域展開活動。<ref>{{Cite web |url= http://www.newsen.com/news_view.php?uid=201501190827411210 |title= 李周妍簽新東家Better娛樂 |date= 2015年1月19日 |publisher= newsen |language= ko }}</ref>
