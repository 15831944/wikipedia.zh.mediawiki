'''李景亮'''（{{bd|1988年|3月20日}}），中国新疆[[塔城地区|塔城地区]]人，中国[[综合格斗|综合格斗]]运动员，绰号“吸血魔”，效力于北京拳天下MMA綜合搏擊俱樂部(China Top Team)。现为[[终极格斗锦标赛|终极格斗锦标赛]]（UFC Ultimate Fighting Championship）次中量级选手。他是继[[张铁泉|张铁泉]]、[[居马别克|居马别克]]之后第三位进军UFC的中国选手。

== 职业生涯 ==
===早期生涯===
李景亮早年曾学习[[摔跤|摔跤]]和[[散打|散打]]，2007年加入新疆维吾尔自治区散打队，师从知名散打选手[[宝力高|宝力高]]。其后随宝力高转习综合格斗，并加入北京拳天下搏击俱乐部, 其後在[[巴西柔术|巴西柔術]]習得棕色帶。李景亮曾参加过英雄榜（Art of War Fighting Championship）8，15；终极勇士综合搏击大赛（UMAC Ultimate Martial Arts Combat）；[[武林传奇|武林传奇]]（Legend Fighting Championship）2，3，4，5，7，9，11<ref>{{cite web |title="THE LEECH" Jingliang Li |url=https://www.tapology.com/fightcenter/fighters/jing-liang-li-the-leech |website=.tapology |accessdate=2018-11-28}}</ref>。并2013年4月27日，在武林传奇 11中击败新西兰选手居莫（Luke Jumeau），获次中量级金腰带。<ref>{{cite web|url=http://www.legendfc.com/cs/fighters/40/Li-Jingliang/|title=李景亮|publisher=武林传奇|access-date=2014-12-20|archive-url=https://web.archive.org/web/20141220185416/http://www.legendfc.com/cs/fighters/40/Li-Jingliang/|archive-date=2014-12-20|dead-url=yes}}</ref><ref>{{cite web|url=http://blog.sina.com.cn/s/blog_684ad4220100uy77.html|title=专访李景亮：用实力说话|publisher=《功夫与博击》|date=2011-12-05}}</ref>

===UFC生涯===

== 争议 ==
=== 瘦肉精事件 ===
李景亮于2016年5月18日因反兴奋剂药检未通过（盐酸[[克伦特罗|克伦特罗]]呈阳性），8月23日被内华达州运动员委员会禁赛<ref>{{cite web |title=又是瘦肉精惹的祸？UFC中国选手李景亮药检未过 |url=http://sports.ifeng.com/a/20160824/49832589_0.shtml |website=凤凰体育 |accessdate=2018-11-28}}</ref><ref>{{cite web |author1=Marc Raimondi |title=Li Jingliang tests positive for clenbuterol, temporarily suspended by NAC, but not USADA |url=https://www.mmafighting.com/2016/8/23/12615934/li-jingliang-tests-positive-for-clenbuterol-temporarily-suspended-by |website=MMA Fighting |accessdate=2018-11-28}}</ref>。2016年8月26日就关于此事件发表声明，解释其原因为瘦肉精所导致<ref>{{cite web |title=UFC中国选手李景亮对瘦肉精事件发布声明 |url=http://www.sohu.com/a/113334186_440343 |website=搜狐体育 |accessdate=2018-11-28}}</ref>。2016年9月2日美国反兴奋剂协会发布声明表示由于李景亮体内被查出的盐酸克伦特罗含量很低，决定不对李景亮进行任何处罚<ref>{{cite web |title=美国反兴奋剂协会单方面决定不处罚李景亮 |url=http://sports.ifeng.com/a/20160911/49952681_0.shtml |website=凤凰体育 |accessdate=2018-11-28}}</ref><ref>{{cite web |author1=Marc Raimondi |title=USADA won’t sanction UFC’s Li Jingliang in clenbuterol case |url=https://www.mmafighting.com/2016/9/2/12774236/usada-won-t-sanction-ufc-s-li-jingliang-in-clenbuterol-case |website=MMA Fighting}}</ref>。9月13日，内华达州运动员委员会取消对李景亮的禁赛，并宣布将不对他进行任何处罚<ref>{{cite web |title=“吸血魔”李景亮免于处罚，将在UFC206出战查得 莱普里斯 |url=http://sports.ifeng.com/a/20161016/50108514_0.shtml |website=凤凰体育 |accessdate=2018-11-28}}</ref><ref>{{cite web |author1=Marc Raimondi |title=Li Jingliang clear of doping suspension from NAC, slated to fight Chad Laprise at UFC 206 |url=https://www.mmafighting.com/2016/10/14/13290768/li-jingliang-off-doping-suspension-from-nac-slated-to-fight-chad |website=MMA Fighting |accessdate=2018-11-28}}</ref>。

=== 插眼事件 ===
2018年2月11日，李景亮在UFC221对战杰克·马修斯的比赛中，一次插眼行为引起争议 <ref>{{cite web |title=Jake Matthews cops savage eye gouge from Li Jingliang at UFC 221 |url=https://www.sportingnews.com/au/mma/news/jake-matthews-eye-gouge-ufc-221-lin-jingliang-perth/1mb4shd68dpzl1990puwd4xrr3 |website=Sport news |accessdate=2018-11-28}}</ref>。其所在的拳天下俱乐部给出回应称：针对比赛中有所争议的“扣眼睛”动作，景亮当时的位置无法看到对手的脸在哪里，在缠斗中用力挣脱，并不存在故意犯规。杰克·马修斯也也在其社交网站上发表声明：“请忘记插眼的事关注这场比赛本身，我很尊敬李景亮”<ref>{{cite web |title=关于李景亮“插眼”，国外拳迷是这么评价的！ |url=http://www.sohu.com/a/222780636_589989 |website=搜狐体育 |accessdate=2018-11-28}}</ref>。
