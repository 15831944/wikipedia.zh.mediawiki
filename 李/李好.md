'''李好'''{{BD|1980年|10月12日|||catIdx= L李好}}出生于[[湖南省|湖南省]][[永州市|永州市]]，毕业于[[湖南大众传媒职业技术学院|湖南大众传媒职业技术学院]]，为[[湖南电视台|湖南电视台]]和[[江苏电视台|江苏电视台]]知名[[主持人|主持人]]，被誉为“全能李好”。<ref name=" ybw" />

== 个人生活 ==
李好的妻子[[郭晓敏|郭晓敏]]早年就读于[[重庆大学|重庆大学]][[美视电影学院|美视电影学院]]，曾是[[湖南电视台|湖南电视台]][[潇湘电影频道|潇湘电影频道]]的[[主持人|主持人]]、[[记者|记者]]，后来去[[美国|美国]]留学，毕业后为[[联合国|联合国]][[国际难民组织|国际难民组织]]义务工作，曾在[[美国|美国]]举行[[联合国东西财经论坛|联合国东西财经论坛]]，[[纽约|纽约]][[华人春晚|华人春晚]]等节目。<ref>{{cite web |url=http://www.hunantv.com/c/20120229/0936591970.html |title=李好与娇妻婚纱照曝光 女方曾是湖南卫视主持人 |accessdate= 2012年|work= |publisher=金鹰网 |language =zh-cn}}</ref><ref name=" zwq" >{{cite web |url=http://www.chinanews.com/yl/2012/02-29/3708026.shtml |title= 李好：在家里我都听我老婆的，洗脚水都是我给她倒|accessdate= 2012年|work= |publisher=中国新闻网 |language =zh-cn}}</ref><ref name="fhq " />

李好和妻子[[郭晓敏|郭晓敏]]的爱情到婚姻经过了漫长的8年，2011年，两人在[[美国|美国]][[拉斯维加斯|拉斯维加斯]]举办了婚礼。<ref>{{cite web |url=http://ent.ifeng.com/zz/detail_2011_09/14/9175612_0.shtml |title=结束8年爱情长跑 李好在美国拉斯维加斯结婚 |accessdate= 2011年|work= |publisher=凤凰网 |language =zh-cn}}</ref>
他说：“我们的感情经历过时间和地域的考验当年，我在[[重庆|重庆]]她在[[长沙|长沙]]；后来，我在[[南京|南京]]她在[[美国|美国]]。但我们都坚持下来了。即便工作上有摩擦也不会影响我们的感情。”<ref name="jyq " >{{cite web |url=http://www.hunantv.com/c/20120229/1657216047.html |title=李好郭晓敏夫妻双双当主持 |accessdate= 2012年|work= |publisher=金鹰网 |language =zh-cn}}</ref><ref name=" zwq" /><ref name="fhq " >{{cite web |url=http://ent.ifeng.com/idolnews/mainland/detail_2012_02/29/12864227_0.shtml |title= 李好携新婚妻子主持：台上是搭档 家里她是领导|accessdate= 2012年|work= |publisher=凤凰网 |language =zh-cn}}</ref>
