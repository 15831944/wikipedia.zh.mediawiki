'''李克'''（{{bd|1955年|11月||}}）籍贯[[山西|山西]][[阳泉|阳泉]]，[[中华人民共和国|中华人民共和国]]政府官员、法官<ref name=gzw/>。

==生平==
李克是[[中国共产党|中国共产党]]党员，硕士研究生。1970年12月参加工作。历任[[北京市海淀区人民法院|北京市海淀区人民法院]]院长，[[北京市第二中级人民法院|北京市第二中级人民法院]]院长，[[北京市高级人民法院|北京市高级人民法院]]副院长、党组副书记，[[最高人民法院|最高人民法院]]党组成员、[[最高人民法院政治部|政治部]]主任、[[中国共产党最高人民法院机关委员会|机关党委]]书记、国家二级大法官，[[最高人民法院审判委员会|最高人民法院审判委员会]]委员<ref name=gzw/>（2004年12月29日至2009年2月28日<ref name=ellsan>{{Cite web |url = http://www.npc.gov.cn/wxzl/wxzl/2004-02/09/content_327755.htm |title =最高人民法院（2003年－）  |author =  |date =  2004-02-09 |publisher = 中国人大网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.chinacourt.org/article/detail/2009/03/id/347413.shtml|title =  全国人民代表大会常务委员会免职名单|author =  |date = 2009-03-02  |publisher = 中国法院网 |language =  |accessdate =  |quote =  }}</ref>）。2009年3月4日，被任命为[[国有重点大型企业监事会|国有重点大型企业监事会]]主席（副部长级），分管第16办事处<ref name=gzw>{{Cite web |url = http://www.sasac.gov.cn/n1180/n20240/n13563062/n13566844/index.html |title =李克  |author =  |date =  |publisher = 国务院国有资产监督管理委员会 |language =  |accessdate = 2017-04-08 |quote =  }}</ref>。
