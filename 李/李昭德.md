'''李昭德'''（{{bd|？||697年|6月26日|catIdx=L李}}）<ref>[http://www.sinica.edu.tw/ftms-bin/kiwi1/luso.sh?lstype=2&dyna=%AD%F0&king=%AAZ%A6Z&reign=%AF%AB%A5%5C&yy=1&ycanzi=%20&mm=6&dd=3&dcanzi=%20 兩千年中西曆轉換<!-- Bot generated title -->]</ref>，[[雍州|雍州]][[长安|长安]]人，是武周[[武则天|武则天]]时代的高级官员和[[宰相|宰相]]。他以对武则天的强硬著称，他和他反对的酷吏[[来俊臣|来俊臣]]一起被处斩。
== 早年 ==
李昭德出自[[陇西李氏|陇西李氏]]，是北魏奉朝请、雍州大中正、宁朔将军、奉车都尉、新城永安二郡守[[李众庆|李众庆]]的玄孙，北周仪同三司、京兆郡大中正、雍州刺史、宕渠郡守、司藩大夫、光州刺史[[李亮_(北周)|李亮]]的曾孙，隋朝殿中侍御史、菊潭县令、侍御史、岐州渭滨县令、司隶刺史、朝请大夫、洛阳县令、唐朝通直散骑侍郎[[李伟节|李伟节]]的孙子，唐朝银青光禄大夫、守司刑太常伯[[李乾祐|李乾祐]]的儿子。李乾祐在[[唐太宗|唐太宗]][[贞观_(唐朝)|贞观]]年间为[[殿中侍御史|殿中侍御史]]，曾经劝谏太宗。其母去世时，唐太宗遣使就墓吊丧，旌表其门。李乾祐虽强直有器幹，而昵于小人，因为不谨慎而贬官。李昭德是李乾祐的[[庶子|庶子]]，强势干练有父风，年轻时通过科举[[明经|明经]]。他被提拔为[[御史中丞|御史中丞]]，[[唐睿宗|唐睿宗]][[永昌_(唐朝)|永昌]]元年（689年），他因事被贬为[[振州|振州]]陵水（今[[海南|海南]][[陵水|陵水]]）尉，不久李昭德被召回，当时增置夏官侍郎三员，和[[娄师德|娄师德]]、侯知一任夏官侍郎。<ref>《[[旧唐书|旧唐书]]·李昭德传》称693年李昭德回任并为宰相。见《旧唐书·卷87》{{cite web |url=http://ef.cdpa.nsysu.edu.tw/ccw/02/tan10.htm |title=存档副本 |accessdate=2007-10-11 |0=2007-10-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20071011103738/http://ef.cdpa.nsysu.edu.tw/ccw/02/tan10.htm |archivedate=2007-10-11 }}但根据《旧唐书》和《[[新唐书|新唐书]]》的则天本纪，明确记载692年被任命为宰相。见《旧唐书·卷6》{{cite web |url=http://ef.cdpa.nsysu.edu.tw/ccw/02/tan01.htm |title=存档副本 |accessdate=2007-11-26 |0=2007-12-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20071225082011/http://ef.cdpa.nsysu.edu.tw/ccw/02/tan01.htm |archivedate=2007-12-25 }}和《新唐书·卷4》 {{cite web |url=http://ef.cdpa.nsysu.edu.tw/ccw/02/ntan01.htm |title=存档副本 |accessdate=2007-11-26 |0=2007-10-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20071011061323/http://ef.cdpa.nsysu.edu.tw/ccw/02/ntan01.htm |archivedate=2007-10-11 }}</ref>

== 武则天时代 ==
=== 宰相 ===
690年，武则天取代儿子睿宗李旦，自为皇帝，建立[[武周|武周]]。692年，夏官侍郎李昭德上疏密奏武则天的侄子[[魏王|魏王]][[武承嗣|武承嗣]]为宰相，权力太大。武则天回应：“他是我的侄子，我信任他。”李昭德说：

{{quote|承嗣，陛下之侄，又是亲王，不宜更在机权，以惑众庶。且自古帝王，父子之间犹相篡夺，况在姑侄，岂得委权与之？脱若乘便，宝位宁可安乎？}}

武则天同意了。692年秋八月，她解除了武承嗣、[[武攸宁|武攸宁]]和[[杨执柔|杨执柔]]三个亲戚的相位。同时，李昭德被任命为鳳閣侍郎（鳳閣，[[中书省|中书省]]）、[[同鳳閣鸞臺平章事|同鳳閣鸞臺平章事]]。[[崔元综|崔元综]]、[[姚璹|姚璹]]、[[李元素_(亳州)|李元素]]、[[崔神基|崔神基]]与他同日为相。数日后[[王璿_(唐朝)|王璿]]为相。当时，凤阁舍人[[张嘉福|张嘉福]]令[[洛阳|洛阳]]人王慶之率轻薄恶少数百人诣阙上表，主张立武承嗣为太子，废黜皇嗣李旦。武则天并没有同意，但王慶之时常入宫请求皇帝。武则天命李昭德诘责王慶之，李昭德命人将王慶之打死，众轻薄恶少散去。李昭德回奏武则天：

{{quote|臣闻文武之道，布在方策，自古有侄为天子而为姑立庙乎？以亲亲言之，天皇（唐高宗），陛下夫也；皇嗣（睿宗李旦），陛下子也。当传之子孙为万世计。陛下承天皇顾托而有天下，又立承嗣，臣见天皇不来食矣。<ref>李昭德回避了[[新朝|新朝]]皇帝[[王莽|王莽]]为他姑母 [[汉朝|汉朝]]皇后[[王政君|王政君]]立庙之事，或许他认为这样说是明目张胆的表示武则天篡位。</ref>}}

武则天同意李昭德的建议，没有立武承嗣为太子。<ref>《[[資治通鑑|資治通鑑]]》记载此事发生在691年，但称李昭德的官职为鳳閣侍郎。但692年，李昭德才被任命为鳳閣侍郎。见《[[資治通鑑|資治通鑑]]》[[s:資治通鑑/卷204|卷204]].</ref>武承嗣知道后又向武则天诋毁李昭德，武则天说：“自我任昭德，每获高卧，是代我劳苦，非汝所及也。”

武则营建神都[[洛阳|洛阳]]，李昭德规创文昌臺（[[尚书省|尚书省]]）及定鼎門、上東門诸门，标置华壮。原来，[[隋炀帝|隋炀帝]]营造[[东都|东都]]洛阳，没有外城，只有低矮的围墙。李昭德此时开始营建东都外城。洛水有二桥，唐高宗[[上元|上元]]年间，司农卿韋機将一桥迁到長夏門外，洛阳人甚以为便，所以废利涉桥，所省之钱以万计。民利之，其一桥废，省巨万计。但是每年被洛水冲注，常劳废修缮。李昭德开始累石替代桥柱，锐其前来分水势，从此水无患。

李昭德为人耿直，有人从[[洛水|洛水]]中捞到一个白石头，上面有几个赤红点，进宫进献讨好皇帝。诸宰相诘责，这人说：“此石赤心，所以来进。”李昭德怒叱：“此石赤心，那洛水中其他石头岂不是全造反了？”李昭德常常参奏横行不法的[[酷吏|酷吏]]，来俊臣抛弃原妻而娶[[太原|太原]][[王庆诜|王庆诜]]之女，[[侯思止|侯思止]]也奏娶[[赵郡|赵郡]][[李自挹|李自挹]]之女，皇帝下敕命政事堂共商。李昭德拍手对诸宰相说：“大可笑！往年俊臣贼劫王庆诜女，已大辱国。今日此奴又请索李自挹女，无乃复辱国耶！”最后，侯思止私自贮存朝廷禁止民间使用的丝锦，李昭德亲自在朝堂杖杀[[侯思止|侯思止]]。

694年，李昭德[[检校|检校]][[內史|內史]]（鳳閣首长）。武则天派她的男宠和尚[[薛怀义|薛怀义]]率军攻打[[后突厥汗国|后突厥汗国]]，李昭德和另一个宰相[[苏味道|苏味道]]为薛怀义的行军长史，没有遇到突厥军而回军。李昭德和薛怀义不和，他常常被怀义鞭打，李昭德恐惧请罪。

李昭德为宰相，掌握权力，性格高傲，和同事不和。娄师德行动迟缓，李昭德上朝时得等着他，骂他为田舍夫。前鲁王府功曹参军丘愔、长上果毅鄧注上奏李昭德专权，武则天开始猜忌李昭德。694年秋[[九月廿一|九月廿一]]日，贬李昭德为南宾（今[[广西|广西]][[钦州|钦州]]）县尉，不久又减死流放。695年正月，[[周允元|周允元]]与司刑少卿皇甫文备上奏说内史[[豆卢钦望|豆卢钦望]]、同平章事[[韦巨源|韦巨源]]、[[杜景俭|杜景俭]]、苏味道、[[陆元方|陆元方]]依附李昭德，同被罢相。

=== 流放归来和被杀 ===
李昭德后被召回为[[監察御史|監察御史]]，司仆少卿[[来俊臣|来俊臣]]仗势贪求女色，通过诬告谋反，夺取官民妻妾，李昭德非常恨他，还多次羞辱秋官侍郎皇甫文備。697年，两人因此诬告李昭德谋反，李昭德被下狱。

来俊臣又诬告武氏诸王和[[太平公主|太平公主]]、李旦、庐陵王[[唐中宗|李哲]]谋反。[[河东|河东]]人卫遂忠告发来俊臣由此想窃取权力。697年6月26日（[[六月初三|六月初三]]日），李昭德和来俊臣被同日处死，当时的人都痛惜李昭德，对处死来俊臣拍手称快。[[唐中宗|唐中宗]][[神龙_(年号)|神龙]]二年（706年）赠李昭德左御史大夫。[[唐德宗|唐德宗]][[建中_(唐朝)|建中]]三年（782年），加赠[[司空|司空]]。

子李元紘，和后来的宰相[[李元纮|李元纮]]不是同一人。
