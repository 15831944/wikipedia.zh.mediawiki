'''李克农'''{{BD|1899年|9月15日|1962年|2月9日}}，前名'''震-{}-中'''，[[安徽|安徽]][[巢县|巢县]][[烔炀镇|烔炀镇]]（现安徽巢湖市烔炀镇）中李村人，[[中国人民解放军上将|中国人民解放军上将]]。

李克农早年在上海从事中共地下工作，被称为是“[[龙潭三杰|龙潭三杰]]”之一。在[[中央苏区|中央苏区]]任[[红一方面军|红一方面军]]政治保卫局局长，[[长征|长征]]到达陕北后，参与国共谈判。[[抗日战争|抗日战争]]时期，在[[国统区|国统区]]从事统战工作，后任[[中共中央社会部|中共中央社会部]]副部长。[[国共内战|国共内战]]时期，任[[中共中央社会部|中共中央社会部]]部长，长期负责情报工作。[[中华人民共和国|中华人民共和国]]成立后，任[[中华人民共和国外交部|中华人民共和国外交部]]副部长、[[中国人民解放军总参谋部|中国人民解放军副总参谋长]]、[[中共中央调查部|中共中央调查部]]部长等职。李克农是中国人民解放军开国上将中，唯一一位没有带兵打仗历史的人，被称为中共的“特工之王”。

==生平==
=== 早年生涯 ===
{{see also|钱壮飞|胡底|陈赓|顾顺章}}
1910年起先后就读于[[芜湖|芜湖]][[安徽公学|安徽公学]]、[[芜湖市第十一中学|圣雅阁书院]]（教会学校）。1919年任[[安庆|安庆]]《[[国民日报|国民日报]]》副刊编辑，后曾在安徽省政府、[[六安|六安]]县政府供职。<ref>{{Cite web |url=http://www.shopping518.com/c30260/w10217998.asp |title=李克农：活跃在特殊的外交战线上 |access-date=2014-11-21 |archive-url=https://web.archive.org/web/20141217180457/http://www.shopping518.com/c30260/w10217998.asp |archive-date=2014-12-17 |dead-url=yes }}</ref>

[[File:Li_Kenong.jpg|thumb]]
1925年，李克农在芜湖参与创办[[民生中学|民生中学]]<ref>{{cite web|url=http://www.ahage.net/mingren/14291.html|title=李克农与芜湖民生中学|accessdate=2010-02-26|publisher=安徽文化网|work=|language=zh-hans|archive-url=https://web.archive.org/web/20141129063100/http://www.ahage.net/mingren/14291.html|archive-date=2014-11-29|dead-url=yes}}</ref>，1926年冬加入[[中国共产党|中国共产党]]。1927年3月，任[[国民党|国民党]]芜湖县党部宣传部部长，开展地方群众工作。国共决裂后，辗转到上海，先后筹办中共组织领导的《[[铁甲车|铁甲车]]》报、《[[老百姓|老百姓]]》报。

1929年冬，李克农化名进入[[上海无线电管理局|上海无线电管理局]]，任广播新闻编辑，不久取得[[徐恩曾|徐恩曾]]信任，升任电务股股长，在[[中共中央特科|中共中央特科]]领导下从事秘密情报工作<ref>{{cite web |url=http://news.xinhuanet.com/mil/2014-04/22/c_126419460.htm|title=八大传奇红色特工：李克农没打过仗当上将|accessdate=2014-04-22|publisher=新华网|work= |language=zh-hans}}</ref>。1931年4月，中共中央政治局候补委员[[顾顺章|顾顺章]]在武汉被捕后投降国民党，这一消息被[[钱壮飞|钱壮飞]]获取。钱壮飞派其女婿将此事通知李克农，李克农再转报中央，使中共中央机关安全转移。<ref> {{cite book |author=Barnouin, Barbara and Yu Changgen.|title=Zhou Enlai: A Political Life.|year=2006|publisher= Chinese University of Hong Kong|location=Hong Kong|pages=45-47|isbn=962-996-280-2}}</ref>

1931年冬，李克农到[[中央苏区|中央苏区]]后，任[[江西省苏维埃政府|江西省苏维埃政府]]政治保卫分局执行部部长、[[中华苏维埃共和国|中华苏维埃共和国]]临时中央政府政治保卫局执行部部长，曾被选为中华苏维埃共和国中央候补执行委员。1932年任[[中国工农红军第一方面军|中国工农红军第一方面军]]政治保卫局局长、红军工作部部长。1934年10月参加[[长征|长征]]，负责进行敌情侦察和保卫中共中央机关安全的工作。李克农到[[陕北|陕北]]后，任[[中共中央联络局|中共中央联络局]]局长。

1936年春，李克农作为中共代表在[[洛川|洛川]]与[[张学良|张学良]]达成局部停战协定<ref>{{cite web|url=http://szb.northnews.cn/bfxb/html/2013-10/31/content_1060819.htm|title=西安事变前李克农曾与张学良秘密会谈|accessdate=2013-10-31|publisher=北方新报|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20141129114002/http://szb.northnews.cn/bfxb/html/2013-10/31/content_1060819.htm|archivedate=2014-11-29}}</ref>。[[西安事变|西安事变]]发生后，任中共代表团秘书长。协助周恩来到西安开展国共谈判。1937年1月，李克农代表红军，在潼关与[[顾祝同|顾祝同]]进行停止内战的谈判，后在西安组织领导红军办事处。

=== 抗日战争与第二次国共内战时期 ===
[[抗日战争|抗日战争]]爆发后，先后任[[八路军驻上海事处|八路军驻上海]]、[[八路军驻京办事处|南京办事处]]处长<ref>[http://www.huaxia.com/thjq/jsgoucheng/2014/09/4067090.html "八路军驻京办"藏身南京傅厚岗小巷 处长系李克农]</ref>、[[八路军总部|八路军总部]]秘书长、[[中共中央长江局|中共中央长江局]]秘书长。1938年5月，[[张国焘|张国焘]]背叛中共，来到武汉，被李克农扣住，直到中共中央决定让张国焘自选去处<ref>{{cite web |url=http://politics.people.com.cn/GB/101380/8209519.html|title=李克农：中共特工王|accessdate=2008-10-22|publisher=凤凰网 |work= |language=zh-hans}}</ref>。1938年11月，组建[[八路军驻桂林办事处|八路军驻桂林办事处]]，李克农任处长，开展[[统战|统战]]工作<ref>{{cite web |url=http://news.xinhuanet.com/politics/2010-12/09/c_12861781.htm|title=中共中央南方局派出机构 八路军桂林办事处成立前后|accessdate=2010-12-09|publisher=新华网 |work= |language=zh-hans}}</ref>。1941年1月[[皖南事变|皖南事变]]后，奉命撤回[[延安|延安]]，任[[中共中央社会部|中共中央社会部]]副部长。1942年兼任[[中共中央情报部|中共中央情报部]]副部长，参与领导秘密情报组织的建设工作。

1946年1月，李克农到北平[[军事调处执行部|军事调处执行部]]任中共方面秘书长，负责中共代表团的党务行政和情报工作。[[第二次国共内战|第二次国共内战]]爆发后，1947年3月，任[[中共中央后方委员会|中共中央后方委员会]]委员，在[[叶剑英|叶剑英]]领导下开展情报工作。1947年6月，李克农和[[康生|康生]]下令将[[王实味|王实味]]秘密处死<ref>{{cite web |url=http://news.163.com/13/1025/16/9C1V3PSR00014Q4P.html|title= 父亲李克农与潘汉年、王实味事件|accessdate=2013-10-25|publisher=网易|work= |language=zh-hans}}</ref>。1948年，任[[中共中央社会部|中共中央社会部]]部长和中央保密委员会主任委员。李克农长期负责中共的情报工作，被称为中共的“特工之王”<ref>{{cite web |url=http://news.ifeng.com/a/20141116/42483702_0.shtml|title= 巢湖岸走出的“特工之王”|accessdate=2014-11-16|publisher=凤凰网 |work= |language=zh-hans}}</ref>。

=== 中华人民共和国成立后 ===
{{see also|朝鲜战争}}
中华人民共和国成立后，任中共中央情报委员会书记、[[中央情报部|中央情报部]]部长、[[外交部|外交部]]副部长、[[中央军委总情报部|中央军委总情报部]]部长。1949年年底到1950年年初，李克农领导破获了国民党特工密谋刺杀[[毛泽东|毛泽东]]的一起案件<ref>{{cite web |url=http://news.ifeng.com/a/20141116/42483702_0.shtml|title=李克农人在大陆为何也能把毛人凤气晕死过去？|accessdate=2014-08-25|publisher=凤凰网 |work= |language=zh-hans}}</ref>。1951年7月，李克农任中共中央和中国政府的特派代表<ref name="欧阳青">{{cite book |author=[[欧阳青|欧阳青]] |title=《大授衔：1955共和国将帅授衔档案》|year=2011|publisher=长城出版社 |location=北京市西城区甘家口三里河路40号 |pages=319|isbn=9787548300588 |language = zh-cn}}</ref> ，赴朝鲜参加停战谈判，实际上是朝鲜停战谈判中方代表团的总指挥和最高负责人。<ref>{{cite web |url=http://news.xinhuanet.com/mil/2010-10/22/c_12690779_12.htm|title= 《解读抗美援朝战争》：重要人物解读 |accessdate=2010-10-22|publisher=新华网 |work= |language=zh-hans}}</ref>

1953年，李克农任[[中国人民解放军副总参谋长|中国人民解放军副总参谋长]]。1954年4月，出席[[日内瓦会议|日内瓦会议]]，协助[[周恩来|周恩来]]工作，并主持代表团的内外事务和联络工作<ref>{{cite web|url=http://szb.ch365.com.cn/dsck/html/2014-04/24/content_892014.htm|title=李克农与日内瓦会议|accessdate=2014-04-24|publisher=巢湖日报|work=|language=zh-hans}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1955年，李克农兼任[[中共中央调查部|中共中央调查部]]部长，同年被授予上将军衔，获一级[[八一勋章_(1955年)|八一勋章]]、一级[[独立自由勋章|独立自由勋章]]和一级[[解放勋章|解放勋章]]。李克农是[[中国共产党第八届中央委员会委员列表|中共第八届中央委员]]，[[中国人民政治协商会议第三届全国委员会委员名单|第三届全国政协常委]]。

1957年10月25日，李克农摔伤造成脑溢血。1962年，出席[[七千人大会|七千人大会]]，2月9日，因患[[脑软化|脑软化]]在[[北京|北京]]病逝。1989年9月14日，在李克农诞辰九十周年时，时任[[中华人民共和国主席|中华人民共和国主席]]的[[杨尚昆|杨尚昆]]称李克农为“久经考验的无产阶级革命家，杰出的社会活动家、外交家，忠诚的共产主义战士”。<ref>{{cite web|url=http://www.whsdaj.gov.cn/ArticleInfo.aspx?id=160|title=李克农|accessdate=2008-11-27|publisher=芜湖市档案局|work=|language=zh-hans|archive-url=https://web.archive.org/web/20141214004655/http://www.whsdaj.gov.cn/ArticleInfo.aspx?id=160|archive-date=2014-12-14|dead-url=yes}}</ref>

==家族==
妻子赵瑛（巢县人，原名赵彩英，别名赵曼，1961年去世），1917年结婚，育有三子、二女。

;儿女列表
*李宁，长女，原中央党校教授，生于1918年
*李冰，次女，中国医学科学院肿瘤医院原党委书记、副院长。生于1920年
*李治，长子，原中办局级领导
*李力，次子，原中央军委三局正师休干，总参通信部原副部长。生于1925年
*[[李伦|李伦]]，三子，原总后勤部副部长、中将

;孙辈：
*李凯城，1952年生，1969年入伍，曾任总参政治部政研室主任、总参政治部办公室副秘书长、师副政委等职，大校军衔，已退休，现百丈文化公司顾问。

大陆发生过多次冒充其子女诈骗钱财案，被骗对象甚至包括 原北京市委书记[[陈希同|陈希同]]和首钢董事长[[周北方|周北方]]等高官。案情之曲折，匪夷所思<ref>[http://news.xinhuanet.com/legal/2008-01/28/content_7510836_1.htm 人民法院报:“忽悠女王”凭高超骗技骗3800余万 周北方落入陷阱]</ref>。
