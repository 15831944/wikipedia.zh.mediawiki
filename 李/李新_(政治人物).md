'''李新'''{{BD|1953年|7月16日|2017年|9月28日|catIdx=Li李}}，[[中華民國|中華民國]][[政治人物|政治人物]]，生於[[臺灣|臺灣]][[臺北縣|臺北縣]]（今[[新北市|新北市]]），[[籍貫|籍貫]][[河南省|河南省]][[汝甯府|汝甯府]][[汝陽縣|汝陽縣]]（今[[駐馬店市|駐馬店市]][[汝南縣|汝南縣]]），[[國立中興大學法商學院|國立中興大學法商學院]]法律系學士（今[[國立臺北大學|國立臺北大學]]）、[[國立臺灣大學|國立臺灣大學]]政治研究所碩士<ref>[http://www.appledaily.com.tw/appledaily/article/headline/20170929/37796798/ 震驚 甫與女友分手 落寞赴前妻住處 議員李新墜樓亡].蘋果日報.2017-09-29</ref>。

出身[[中國國民黨|中國國民黨]]，1993年曾參與發起[[新黨|新黨]]，2001年改投[[親民黨|親民黨]]，2008年重投[[國民黨|國民黨]]，長期擔任[[臺北市議會|臺北市議會]]議員，一度擔任議會副議長。[[中華民國國語|國語]]、[[臺灣話|臺灣話]]皆流利<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20170929/1213198/ 【人物動畫】砲打馬英九洪秀柱　李新正直為黨].蘋果日報.2017-09-29</ref>，由於問政犀利，以直率敢言著稱，常受邀到[[政論節目|政論節目]]發表意見<ref>游昌樺、陳慧文、何純輝、謝育炘.[https://video.udn.com/news/758926 影／議員李新墜樓亡 身上未發現遺書].聯合影音網.2017-09-28</ref>。曾寫作《破風的車輪：翻轉的世代不翻轉》<ref name="忠">{{cite journal|title=政治巷仔內》忠貞的正藍軍　李新的從政路|url= https://www.nownews.com/news/20170929/2615680 |last=戴祺修|date=2017-09-28 |work=NOWnews 今日新聞}}</ref>。

2017年9月28日，李新在[[臺北市|臺北市]][[大安區_(台北市)|大安區]][[跳樓|跳樓]][[輕生|輕生]]，享壽64歲，輕生前被眾多媒體報導與郭新政分手，甚至被分手，檢、警正在調查李新跳樓的原因<ref>張君豪、陳以昇.[http://www.appledaily.com.tw/realtimenews/article/new/20170928/1212984/ 【影片曝光】李新墜樓前最後身影曝光　表情落寞心事重重].[[蘋果日報_(台灣)|蘋果日報]].2017-09-28</ref>。李新女友郭新政邀請資深新聞人[[盛竹如|盛竹如]]拍[[YouTube|YouTube]]影片，公開指控時任[[立委|立委]]的[[羅淑蕾|羅淑蕾]]設局害死李新<ref>{{youtube|oU2dcc25bbA|女友在李新忌日提告 控羅淑蕾設局害命－民視新聞}}.民視綜合頻道.2018-09-28</ref><ref>{{youtube|VYNG_5EG5ps|20181130 李新是怎麼摔死的 完整紀錄版（誰摔死了李新）}}.LH誰摔死了李新.2018-11-29</ref>。

==生平==
早年曾任[[立法委員|立法委員]][[郁慕明|郁慕明]]特別助理，並在[[國會|國會]]內組織「立法院助理[[工會|工會]]」，屬於[[中國國民黨|中國國民黨]][[新國民黨連線|新連線派]]<ref name="忠"/>。

1993年，新連線派創黨。李新追隨郁慕明，脫離[[中國國民黨|中國國民黨]]，成為[[新黨|新黨]]的創黨人之一<ref name="忠"/>。

1996年，當選[[國民大會|國民大會]]代表，時年四十三歲，走出幕後，正式從政<ref>{{Cite web |url=https://udn.com/news/story/11315/2727733 |title=後排的老大哥李新 質詢常奧援年輕人 |access-date=2017-09-29 |archive-url=https://web.archive.org/web/20170929091905/https://udn.com/news/story/11315/2727733 |archive-date=2017-09-29 |dead-url=yes }}</ref>。

1998年當選[[大安區_(台北市)|大安]][[文山區|文山區]][[臺北市議員|臺北市議員]]<ref name="忠"/>。

2001年在[[謝啟大|謝啟大]]的號召下，曾辭去議員，代表新黨在[[桃園縣|桃園縣]]角逐[[立法委員|立法委員]]落敗，同年轉投[[親民黨|親民黨]]，當選[[臺北市議會|臺北市議會]]議員，並當選第九屆市議會副議長（2002年至2006年），李新聰明、有謀略，被其他市議員封為「小[[諸葛亮|諸葛]]」<ref name="忠"/>。

2008年重投[[中國國民黨|中國國民黨]]<ref name="忠"/>。

2012年，李新提出有中國大陆籍[[配偶|配偶]]以[[輪椅|輪椅]]將[[榮民|榮民]]長者推到戶政機關登記結婚，隔天再推到銀行提領鉅款，第三天到地政事務所辦理房產轉移之事，但多數榮民長者的子女、家屬都不知情，直到該位榮民過世後爆發遺產的爭產風波才得知。這類中國大陆女子主動搭訕、專嫁領有[[終身俸|終身俸]]或有遺產的老榮民，以圖續領半俸並[[繼承|繼承]][[遺產|遺產]]的「職業收屍隊」騙婚詐財事件層出不窮，她們為求儘速達成目的，還以「夜夜求歡」、「過度進補」、「吸菸飲酒」等方式讓老榮民逐漸體弱，近乎[[謀殺|謀殺]]，其中很多人有多次結婚紀錄。為防止這群「職業收屍隊」一再得逞，他要求戶政事務所受理逾六十五歲長者登記結婚時，應一律通知直系血親（子女等）到場，對其配偶再婚兩次以上者，警政或社政機關應主動進行家戶拜訪以了解實情<ref>劉榮、邱紹雯.[http://news.ltn.com.tw/news/politics/paper/623332　中國女子「收屍隊」 專嫁榮民刮遺產].[[自由時報|自由時報]].2012-10-17</ref>
<ref>郭美瑜.[http://www.appledaily.com.tw/appledaily/article/headline/20121017/34578765/　防騙婚 老人結婚可通知子女].[[蘋果日報_(台灣)|蘋果日報 (台灣)]].2012-10-17</ref>。

2012年，[[臺北市長|臺北市長]][[郝龍斌|郝龍斌]]公開表示，支持入獄多年而健康不佳的[[陳水扁|陳水扁]][[保外就医|保外就醫]]，以促進社會和諧。李新出面痛批，如果郝龍斌不收回挺扁言論，將誓死反對郝角逐[[2016年中華民國總統選舉|2016總統]]大位，還要上街[[遊行|遊行]]向郝龍斌[[示威|示威]]<ref>邱俊福.[http://news.ltn.com.tw/news/society/breakingnews/2206808 台北市議員李新清晨驚傳墜樓身亡 警勘查中].[[自由時報|自由時報]].2017-09-28</ref>。

2014年3月29日，李新帶領警察眷屬、婦女團體、企業人士等在[[中正紀念堂|中正紀念堂]]前自由廣場舉辦「康乃馨擁抱太陽花」的「反[[太陽花學運|反服貿]]活動」，即[[康乃馨運動|康乃馨運動]]。發起人李新說，是因許多警眷反映，[[太陽花運動|太陽花運動]]讓他們長期見不到家人；整個活動主軸，就是要用和平、理性、柔性方式呼籲孩子們早點回家，從未規劃任何衝撞、抗爭的動作。雖有報導指出當時李新宣稱對[[服貿|服貿]]不甚瞭解，<ref>
[http://www.appledaily.com.tw/realtimenews/article/politics/20140328/369105/%E5%BA%B7%E4%B9%83%E9%A6%A8%E6%93%81%E6%8A%B1%E5%A4%AA%E9%99%BD%E8%8A%B1%E3%80%80%E5%8D%83%E4%BA%BA%E4%B8%8A%E8%A1%97%E7%B1%B2%E5%AD%B8%E7%94%9F%E5%9B%9E%E5%AE%B6 康乃馨擁抱太陽花　千人上街籲學生回家].蘋果日報.2014-03-28</ref>但現場亦發放央行總裁[[彭淮南|彭淮南]]所寫的「服貿懶人包」文宣供民眾了解。<ref>盧素梅.[http://www.chinatimes.com/newspapers/20140330000725-260301 康乃馨擁抱太陽花 孩子回家吧！].[[旺報|旺報]].2014-03-30</ref>

2014年7月，李新證實已[[離婚|離婚]]三年，李新稱將[[內湖|內湖]]的房屋、土地、[[新臺幣|新臺幣]]200萬元現金、價值約新臺幣600多萬元的[[債券|債券]]，價值約新臺幣100萬元的[[外幣|外幣]]，都給予前妻<ref>[https://ctee.com.tw/mobile/ViewCateNews.aspx?cateid=rwbd&newsid=162719 台北市議員李新，驚傳墜樓身亡].工商時報.2017-09-28 </ref>，並與前妻維持朋友關係<ref>陳重生.[https://newtalk.tw/news/view/2017-09-28/99042 人物》李新墜樓  國民黨失去一名改革派戰將].新頭殼newtalk.2017-09-28 </ref>。現任女友是相識十年的美體考究公司董事長郭新政，在美容界頗有名氣，是知名的商業女強人。因郭女捲入珠寶小開沈家民詐欺案，一家都長年支持李新的受害者，知名珠寶設計師嚴嘉慧，向李新陳情，希望他能出面協調，卻石沈大海，嚴女乃對郭新政等提起[[贓物|贓物]]罪、違反《[[:s:洗錢防制法|洗錢防制法]]》等告訴，嚴女並在李新陪同[[連勝文|連勝文]]掃街拜票時，激動大罵李新「王八蛋！」，她表示對被李新控告公然侮辱「並不後悔」。李新還為了郭新政的官司（後因假當票被依[[偽造文書|偽造文書]]判刑6個月）而與國民黨[[黨內同志|黨內同志]][[羅淑蕾|羅淑蕾]]翻臉<ref>[http://www.nextmag.com.tw/realtimenews/news/5995865 李新驚爆離婚　新女友曾捲入沈記小開珠寶詐欺案].[[壹週刊|壹週刊]].2014-07-30</ref><ref>張君豪.[http://www.appledaily.com.tw/realtimenews/article/new/20150312/573254/ 【獨家】掃街被罵王八蛋　李新控告女富商].[[蘋果日報_(台灣)|蘋果日報 (台灣)]].2015-03-12</ref><ref>[http://www.appledaily.com.tw/realtimenews/article/new/20170928/1212604/1/李新離婚認愛女友　李新離婚速認愛女友　為了她與羅淑蕾鬧翻].[[蘋果日報_(台灣)|蘋果日報 (台灣)]].2017-09-28</ref>。

2016年參選[[中國國民黨主席|中國國民黨主席]]，在選舉中批評[[中國國民黨黨產爭議|黨產爭議]]，[[中影案|中影案]]之中，國民黨將中影大樓賣了22億元，合約卻只寫8億元，剩下的14億元用「所謂的回算機制」帶過，沒有在黨的帳冊上出現，他具體主張「清查黨產、分配發放給黨員」。李新也砲轟總統[[馬英九|馬英九]]自我感覺良好，聽不進人民與黨內同志的聲音，執政八年失敗，引發民怨，是「國民黨大罪人」，呼籲馬英九下臺前[[下詔罪己|下詔罪己]]，真心誠意向人民道歉。李新也大嗆前[[黨魁|黨魁]][[洪秀柱|洪秀柱]]提出的「[[一中同表|一中同表]]」政策，根本是急統[[親共|親共]]政策，重傷國民黨，以致中間選民的選票，拱手讓給[[民進黨|民進黨]]的總統[[蔡英文|蔡英文]]。最終李新得票率5.45%，居第三，高於[[陳學聖|陳學聖]]，但還是由[[洪秀柱|洪秀柱]]當選。[[2017年中國國民黨主席選舉|次年的黨魁選舉]]，李新與多位議會同僚聯手支持[[吳敦義|吳敦義]]<ref>邱奕統.[http://news.ltn.com.tw/news/politics/breakingnews/1638386 李新爆：國民黨賣黨產22億只報8億].[[自由時報|自由時報]].2016-03-20</ref><ref>符芳碩.[https://newtalk.tw/news/view/2016-03-19/71312 李新：馬下台前應下詔罪己 藍才有未來].[[新頭殼|新頭殼]]. 2016-03-19</ref><ref>王家俊.[http://www.appledaily.com.tw/realtimenews/article/politics/20170928/1212732/ 【大砲性格】李新曾參選黨主席　痛罵馬英九是國民黨大罪人].[[蘋果日報_(台灣)|蘋果日報]].2017-09-28</ref>。

== 身後 ==
===自盡疑雲===
外界揣測李新的過世，是否是李新與女友[[郭新政|郭新政]]之間，出現感情、財務問題而導致<ref>邱俊福、錢利忠、陳恩惠、郭安家、周彥妤.[http://news.ltn.com.tw/news/politics/paper/1139364 北市議員李新 前妻住處墜樓亡].自由時報.2017-09-29</ref>。其友人指出，2017年7月時郭新政向李新提出分手，李新因此罹患[[重度憂鬱症|重度憂鬱症]]，而導致[[尋短|尋短]]<ref>[http://www.appledaily.com.tw/realtimenews/article/local/20170928/1212591/ 【李新墜樓亡】死前心情差　恐與新歡分居有關].[[蘋果日報_(台灣)|蘋果日報]].2017-09-28</ref>。議會同僚[[陳永德|陳永德]]表示，李新被封為「永遠的法規會召集人」，這次開議[[抽籤|抽籤]]時，李新不願再當召集人，引起議長[[吳碧珠|吳碧珠]]關心詢問，李新僅說身體狀況不佳<ref>張立勳、陳燕珩.[http://www.chinatimes.com/newspapers/20170929000431-260106 二段情 李新坦言婚姻不及格].中國時報.2017-09-29</ref>，且開議前李新曾吞了五、六十顆[[安眠藥|安眠藥]]輕生，但送醫搶救得宜，救回一命，不料李新依然發生憾事，陳永德對此痛心<ref>林媛玲.[https://tw.appledaily.com/new/realtime/20170928/1212680/ 【李新墜樓亡】同僚驚爆 李新日前曾吞安眠藥自殺 獲救].[[蘋果日報_(台灣)|蘋果日報]].2017-09-28</ref>。另一位[[議會|議會]]同事[[戴錫欽|戴錫欽]]受訪時表示，李新情變也發生兩個月了，如果要[[自害|自害]]，不必等到現在，可能是服用安眠藥之後[[夢遊|夢遊]]，導致墜樓意外<ref>邱明玉.[https://www.nownews.com/news/20170929/2616491 墜樓輕生？戴錫欽還原李新死亡真相].NOWnews 今日新聞.2017-09-29</ref>。李新辦公室表示，李新並非因感情、財務等原因身亡，而是因為問政操勞，導致嚴重[[失眠|失眠]]，才走向絕路，外界傳言皆屬憑空臆測<ref>張君豪.[http://www.appledaily.com.tw/realtimenews/article/new/20170928/1213112/ 【李新墜樓亡】服務處發聲明　強調「因失眠所苦」].蘋果日報.2017-09-28</ref>。

2018年李新生前的女友[[郭新政|郭新政]]出面表示，指李新的死因與感情無關，她澄清：「我們從來沒有分手，新哥怎麼摔死的？」而且李新死前希望她出國，避免臺灣仇家的攻擊。郭新政出資拍攝了[[類戲劇|類戲劇]]《誰摔死了李新》，共八集，八支影片在[[YOUTUBE|YOUTUBE]]上線兩日，點閱次數已達到數萬次，影片指控李新的死，乃受到[[國民黨|國民黨]]同黨同志[[羅淑蕾|羅淑蕾]]、[[黃昭順|黃昭順]]，及珠寶業者[[沈家民|沈家民]]、[[嚴嘉慧|嚴嘉慧]]等人的迫害。找來知名主播[[盛竹如|盛竹如]]旁白，涉及[[政治迫害|政治迫害]]、[[詐欺|詐欺]]與[[計中計|計中計]]，多家媒體稱內容直逼電影充滿[[謀殺|謀殺]]與[[官商勾結|官商勾結]]的知名電影《[[血觀音|血觀音]]》<ref>[https://www.setn.com/News.aspx?NewsID=435424 「誰摔死李新？」李新女友拍片緝凶　政治陰暗直逼血觀音！].三立新聞網.2018-09-28 </ref><ref>[https://tw.news.yahoo.com/%E8%AA%B0%E6%91%94%E6%AD%BB%E6%9D%8E%E6%96%B0-%E6%94%BF%E6%B2%BB%E8%BF%AB%E5%AE%B3%E7%9B%B4%E9%80%BC%E8%A1%80%E8%A7%80%E9%9F%B3-054506212.html 「誰摔死李新？」政治迫害直逼血觀音].三立新聞網.2018-09-28 </ref><ref>張鈜閔.[https://tw.appledaily.com/new/realtime/20180928/1437859/ 「誰摔死李新？」影片上架　網友驚呆：比血觀音還扯！].蘋果日報.2018-09-28 </ref>。
