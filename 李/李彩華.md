'''李彩華'''（{{lang-en|Rain Li Choi Wa}}，{{bd|1983年|11月11日|catIdx=L李}}），曾用藝名'''李彩樺'''，[[香港|香港]]女[[歌手|歌手]]和[[演員|演員]]。

== 簡歷 ==
李彩華有一名胞弟，父親畢業於[[英皇書院|英皇書院]]，是戰前望族，曾經營海味生意<ref>{{cite web |url=http://news.modia.com.hk/2017/07/26/32736/%E5%81%9A%E8%97%9D%E4%BA%BA%E4%B8%A6%E4%B8%8D%E9%A2%A8%E5%85%89%E3%80%80%E4%BB%96%E5%80%91%E6%9B%BE%E6%9B%BF%E5%AE%B6%E4%BA%BA%E9%82%84%E5%82%B5%E5%8F%97%E7%9B%A1%E8%8B%A6%E9%A0%AD/ |title=做藝人並不風光 他們曾替家人還債受盡苦頭 |last= |first= |date=2017-7-26 |website= |publisher=流動新聞 |access-date= 2019-3-13 |quote=}}</ref>。她於2000年在[[東華三院李潤田紀念中學|東華三院李潤田紀念中學]][[畢業|畢業]]後成為[[模特兒|模特兒]]及加入[[娛樂圈|娛樂圈]]，成為歌手，並且推出過多張唱片。後來李彩華專注於發展[[電影|電影]]及[[電視劇|電視劇]]。李彩華就讀中學時活躍於校際舞蹈節，是學校舞蹈組的成員。

2006年，改名為李彩樺<ref>[http://ent.ifeng.com/special/licaihua/news/200611/1116_273_33885.shtml   香港女星李彩华做客凤凰揭更名之谜]</ref>，並開始發展中國大陸[[電視劇|電視劇]]，參演[[中央電視台|中央電視台]]製作的《[[昭君出塞|昭君出塞]]》，飾演王昭君；又重拍《[[大旗英雄傳|大旗英雄傳]]》，飾演溫黛黛。2011年，李彩華於[[中國大陸|中國大陸]][[電視劇|電視劇]]《[[回家的诱惑|回家的诱惑]]》中飾演艾莉；以該劇打破了[[湖南卫视|湖南卫视]]自《[[大長今_(電視劇)|大長今]]》以来的收视记录。

她自曝未在中國大陸發展前，老板要求当其情人提供[[性服务|性服务]]，“当时他的确希望我提供工作以外的服务，被我拒绝了”，后被公司雪藏。把这件事向老板娘说了之后，就在她的协助下与公司解约了。<ref>{{cite web |url=http://ent.163.com/12/1121/08/8GQORHLN00032DGD.html |title=《凤游龙门》热拍 李彩桦：不接受任何潜规则 |accessdate= 2012|publisher= 网易|work= |language=zh-hans}}</ref>。

==家庭==
2018年11月11日，比她年長幾歲的黃先生（Eric Huang），已經相識多年，Eric去年11月11日在Rain生日當天（[[光棍節|光棍節]]）求婚，Eric預先佈置好場地，等她收工時即行動。<ref name="Marry"/>

2019年2月2日，二人領證結婚，稍後會在香港及大陸補辦婚宴，已計劃婚後生小朋友，她說：「老公都好鍾意小朋友，佢話越多越好。」<ref name="Marry"/>
