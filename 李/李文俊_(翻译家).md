'''李文俊'''{{BD|1930年||||catIdx=L李}}[[广东人|广东人]]，毕业于[[复旦大学|复旦大学]]，[[中国翻译协会|中国翻译协会]]副会长、[[中国作家协会|中国作家协会]]成员、[[中国社会科学院|中国社会科学院]]研究员。李文俊和妻子[[张佩芬|张佩芬]]都是翻译家。其主要成就是翻译[[美国|美国]]著名小说家[[威廉·福克纳|威廉·福克纳]]和女小说家[[弗朗西丝·霍奇森·伯内特|弗朗西丝·霍奇森·伯内特]]的小说以及[[加拿大|加拿大]]女小说家[[爱丽丝·门罗|爱丽丝·门罗]]的《逃离》。<ref name="dog" >{{cite book |author=威廉·福克纳、李文俊 |title=《喧哗与骚动》|year=2013-01-01 |publisher=漓江出版社 |location= |pages= |isbn=9787540761523 |language=zh-hans}}</ref><ref name="cats" >{{cite book |author=李文俊 |title=《威廉·福克纳》|year=2010-04-01 |publisher=人民文学出版社 |location=北京市 |pages= |isbn=9787020067084 |language=zh-hans}}</ref><ref name="cssat">{{cite web|url=http://beijingww.qianlong.com/1470/2013/07/01/553@194968.htm |title=李文俊、张佩芬：书不能看太多了，就看看古董 |accessdate=2013 |publisher=千龙网 |work= |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160304040936/http://beijingww.qianlong.com/1470/2013/07/01/553%40194968.htm |archivedate=2016-03-04 }}</ref><ref name="bee" >{{cite web |url=http://politics.caijing.com.cn/2013-10-20/113443731.html |title= 李文俊：“一条偏僻的路，但也给我走出来了”|accessdate=2013 |publisher=财经网 |work= |language=zh-hans}}</ref><ref name="oil" >{{cite web |url=http://culture.people.com.cn/n/2013/1028/c172318-23344831.html |title=门罗获奖并未让译者“发达”：当时只拿了18000元 |accessdate=2013 |publisher=人民网 |work= |language=zh-hans}}</ref>

==生平==
1930年，李文俊出生在[[广东省|广东省]][[中山市|中山市]]。<ref name="dogss" >{{cite web |url= http://mobile.163.com/13/0618/02/91KB2JQI001166V4.html|title= 李文俊 爱玛，这就是我！|accessdate=2013 |publisher=网易 |work= |language=zh-hans}}</ref>

1952年，李文俊从[[复旦大学|复旦大学]]新闻系毕业，毕业后担任《译文》和《世界文学》的编辑等职务，并且开始发表作品。<ref name="dog" />

1979年，李文俊加入[[中国作家协会|中国作家协会]]。<ref name="dog" />

==家庭==
李文俊的妻子[[张佩芬|张佩芬]]精通[[德文|德文]]，是著名德文文学翻译家。<ref name="cat" />
