'''李荆璞'''（{{bd|1908年|1月19日|2000年|11月3日|}}），生于黑龙江省[[宁安县|宁安县]]<ref name="a1">{{Cite web |url=http://dangshi.people.com.cn/GB/165617/165618/166491/167909/12091252.html |title=李荆璞 |accessdate=2017-11-04 |author= |date= |work= |publisher=中国共产党新闻网|language=Zh-hans }}</ref>（今[[宁安市|宁安市]]），[[中国人民解放军|中国人民解放军]]将领、[[中国人民解放军开国少将|中国人民解放军开国少将]]。

1931年，[[九一八事变|九一八事变]]后，东北各地出现各路[[东北义勇军|义勇军]]。李荆璞即是在此背景下，参加抗日活动，号“平南洋”<ref name="a2"/>，1933年5月<ref name="a1"/>（或说1935年<ref name="a2"/>）加入中国共产党。他于1934年间进入宁安工农义勇队，后来队中的[[胡德贵|胡德贵]]叛变，队长[[于洪仁|于洪仁]]被杀。李荆璞等十一名[[中国共产党|党]][[中国共青团|团]]员被抓获，但在押送时逃脱，并被[[姜健|姜健]]收纳，带去找[[周保中|周保中]]<ref name=ref1>{{cite book|author=权立|title=中国朝鲜族史研究（2）——纪念延边历史研究所成立30周年论文集|year=1994|publisher=延边大学出版社|url=http://book.chaoxing.com/ebook/detail_810220094e3ac1b64cbdfaa02198ce68e9d1d1447.html}}</ref>{{Rp|292}}。后任[[东北抗日联军|东北抗日联军]]第五军第一师师长。

1945年8月16日，[[苏联军队|苏联军队]]攻占[[牡丹江市|牡丹江市]]。李荆璞、[[金光侠|金光侠]]部随苏军进入牡丹江市。10月14日，成立牡丹市人民政府，李荆璞任市长<ref name="a2">{{Cite web |url=http://dangshi.people.com.cn/n1/2016/0303/c85037-28169463-4.html |title=东北根据地建立人民民主政权侧记 |accessdate=2017-11-04 |author=于佰春 |date=2016-03-03 |work= |publisher=中国共产党新闻网|language=Zh-hans }}</ref>。1952年，任[[热河省军区|热河省军区]]司令员，1956年任[[沈阳军区|沈阳军区]]第一文校校长。1955年，授予[[中国人民解放军少将|中国人民解放军少将]]<ref>{{cite news|title=揭秘：中共首批开国将军的级别是怎样划分的？ |url=http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |accessdate=2012-06-15 |newspaper=凤凰网（转自摘自《党史博览》2010年第5期） |date=2010-06-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20120620091504/http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |archivedate=2012-06-20 }}</ref>。1962年后，[[中华人民共和国国防部|国防部]][[第七研究院|第七研究院]]副院长等职<ref name="a1"/>。
