'''李东生'''（{{bd|1955年|12月|||catIdx=}}），[[山东省|山东]][[诸城市|诸城]]人。[[中国共产党|中国共产党]]和[[中华人民共和国|中华人民共和国]]高级官员。2013年底被[[中纪委|中纪委]]宣布调查、[[中组部|中组部]]证实免职前系[[中国共产党第十八届中央委员会委员|中国共产党第十八届中央委员会委员]]、[[中共中央政法委员会|中共中央政法委员会]]委员，[[中央防范和处理邪教问题领导小组|中央防范和处理邪教问题领导小组]]副组长、[[中央防范和处理邪教问题领导小组办公室|中央防范和处理邪教问题领导小组办公室]]（即[[610辦公室|610辦公室]]）主任，[[中华人民共和国公安部|中华人民共和国公安部]]副部长、党委副书记（[[正部长级|正部长级]]）。李东生是[[周永康|周永康]]的嫡系。<ref  name="dwds" />2015年8月21日，最高人民检察院发布消息称，李东生受贿一案由[[最高人民检察院|最高人民检察院]]侦查终结后移送天津市人民检察院第二分院审查起诉，天津市人民检察院第二分院已向[[天津市第二中级人民法院|天津市第二中级人民法院]]提起公诉。2016年1月12日，天津市第二中级人民法院对李东生受贿案作出一审判决，李东生被判处有期徒刑15年，没收非法所得100万元。<ref>{{cite web |url=http://news.youth.cn/sz/201601/t20160112_7516129.htm |language= |title=李东生受贿案一审宣判 被判处有期徒刑15年 |publisher=央视网 |date=2016-01-12 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20160113092505/http://news.youth.cn/sz/201601/t20160112_7516129.htm |archivedate=2016-01-13 }}</ref>

== 生平 ==
1955年12月，李东生出生。1973年4月，李东生参加工作<ref>[http://roll.eastday.com/c1/2013/1221/2680449562.html 李东生简历，东方网，2013-12-21] {{webarchive|url=https://web.archive.org/web/20131228003846/http://roll.eastday.com/c1/2013/1221/2680449562.html |date=2013-12-28 }}</ref>。1975年至1978年，李东生在[[复旦大学|复旦大学]]新闻系学习。

=== 中央电视台时期 ===
李东生毕业之后，来到[[中国中央电视台|中国中央电视台]]工作20余年，从[[摄影记者|摄影记者]]干起，历任中央电视台新闻部时政组副组长，政文部副主任，新闻采访部副主任、主任，新闻中心主任，副台长<ref name="ifeng">[http://news.ifeng.com/mainland/special/fanfu/content-3/detail_2013_12/21/32348897_0.shtml 李东生从央视摄影记者干起 曾推出《焦点访谈》，凤凰网，2013-12-21]</ref>。1993年创办《[[东方时空|东方时空]]》栏目，1994年，央视推出《[[焦点访谈|焦点访谈]]》栏目，时任中央电视台副台长的李东生主管该栏目，是该栏目的主要创意、组织、终审人之一<ref name="ifeng" /><ref name="xjb" />。

他曾经长期投身中国重大政治活动采访，曾参加1982年[[中共十二大|中共十二大]]，1987年[[中共十三大|中共十三大]]，1997年[[邓小平|邓小平]]逝世、[[香港回归|香港回归]]、[[中共十五大|中共十五大]]等新闻事件的报道<ref name="ifeng" />。

据[[中华人民共和国新闻出版总署|中华人民共和国新闻出版总署]]下属的[[韬奋基金会|韬奋基金会]]介绍，李东生曾经获得第二届[[韬奋新闻奖|韬奋新闻奖]]。

=== 广电总局、中宣部时期 ===
2000年，李东生出任[[国家广播电影电视总局|国家广播电影电视总局]]副局长。2002年5月，升任[[中共中央宣传部|中共中央宣传部]]副部长。2007年10月，李东生被任命为[[中共十七大|中共十七大]]新闻发言人<ref>[http://www.xinhuanet.com/zhibo/20071014/ 李东生主持十七大新闻发布会，新华网，2007-10-14]</ref>。2008年3月，出任[[第十一届全国人大常委会|第十一届全国人大常委会]]委员（任至2010年3月14日）、[[全国人大教科文卫委员会|全国人大教科文卫委员会]]委员<ref name="ifeng" /><ref name="zlk">[http://renwuku.news.ifeng.com/index/detail/100/lidongsheng 李东生简历，中国政要资料库，于2013-12-27查阅] {{webarchive|url=https://web.archive.org/web/20131227115749/http://renwuku.news.ifeng.com/index/detail/100/lidongsheng |date=2013-12-27 }}</ref>。

=== 610办公室、公安部时期 ===
{{See also|610办公室}}

经过周永康运作，<ref  name="dwds">{{cite web|url=http://china.dwnews.com/news/2013-12-22/59359863-all.html|title=揭秘李东生攀附周永康内幕|publisher=多维新闻网|date=2013-12-22}}</ref>2009年10月，李东生从宣传系统突然调任[[中华人民共和国公安部|公安部]]，任副部长（正部长级），被授予[[副总警监|副总警监]]警衔<ref name="ifeng" /><ref>{{cite news|title =原中宣部副部长李东生出任公安部党委成员|url =http://politics.people.com.cn/GB/41223/10278117.html|date =2009-10-28|accessdate =2012-06-10|newspaper =人民网|deadurl =yes|archiveurl =https://web.archive.org/web/20120123211417/http://politics.people.com.cn/GB/41223/10278117.html|archivedate =2012年1月23日|df =}}</ref>。2010年3月14日，[[第十一届全国人民代表大会第三次会议|第十一届全国人民代表大会第三次会议]]通过决定，接受李东生辞去第十一届全国人大常委会委员职务的请求<ref name="xjb">[http://news.ifeng.com/mainland/special/fanfu/content-3/detail_2013_12/21/32345518_0.shtml 公安部副部长李东生涉严重违纪违法被查，凤凰网，2013-12-21]</ref>。2010年8月，李东生兼任国家[[广州亚运会|亚运]]安保协调小组组长，主管[[广州亚运会|广州亚运会]]安保工作<ref name="zlk" />。2011年2月起，升任[[中华人民共和国公安部|中华人民共和国公安部]]副部长（正部长级）、党委副书记<ref name="ifeng" /><ref name="zlk" />。2013年时，其排名在部长[[郭声琨|郭声琨]]、常务副部长[[杨焕宁|杨焕宁]]之后，在九位副部长中排名第二。他还是[[中共中央政法委员会|中共中央政法委员会]]委员，[[中央防范和处理邪教问题领导小组|中央防范和处理邪教问题领导小组]]副组长、[[中央防范和处理邪教问题领导小组办公室|610办公室]]主任<ref name="xjb" />。2012年，李东生成为[[中国共产党第十八届中央委员会委员|中国共产党第十八届中央委员会委员]]<ref name="zlk" />。

李东生为官处事以“一脸正气，两袖清风”的形象为标准，但被外媒称为“中南海最大皮条客”。而明镜新闻报道“李东生靠投机取巧溜须拍马一路升迁，仅向[[周永康|周永康]]就‘贡献’出数位央视美女，并变成周的死忠爪牙”。<ref name="iask">{{cite news |title = 李东生涉嫌严重违纪违法接受组织调查 |url = http://www.iask.ca/news/china/2014/0223/251373.html |date=2014-02-23|accessdate = 2014-03-14 |newspaper = 家园新闻}}</ref>

=== 免职获刑 ===
2013年12月20日，[[中共中央纪律检查委员会|中共中央纪律检查委员会]]、[[中华人民共和国监察部|中华人民共和国监察部]]网站发布消息称：“中央防范和处理邪教问题领导小组副组长、办公室主任，公安部党委副书记、副部长李东生涉嫌严重违纪违法，目前正接受组织调查。”<ref name="zjw">{{cite news |title = 李东生涉嫌严重违纪违法接受组织调查 |url = http://www.mos.gov.cn/xwyw/201312/t20131220_15573.html |date=2013-12-20|accessdate = 2013-12-20 |newspaper = 中央纪律检查部网站}}</ref>李东生由此成为[[中共十八大|中共十八大]]之后第二位落马的[[中国共产党第十八届中央委员会委员|中国共产党第十八届中央委员会委员]]（第一位落马的是[[蒋洁敏|蒋洁敏]]）<ref name="ifeng" />。2013年12月25日，[[中共中央组织部|中共中央组织部]]有关负责人证实，“李东生涉嫌严重违纪，中央已决定免去其中央防范和处理邪教问题领导小组副组长、办公室主任，国务院防范和处理邪教问题办公室主任，公安部副部长、党委副书记职务，现正在按程序办理。”<ref>[http://news.xinhuanet.com/politics/2013-12/25/c_118709712.htm 中组部:中央已决定免去李东生领导职务，新华网，2013-12-25]</ref>

2014年2月24日，[[中国政府网|中国政府网]]发布来自[[人力资源和社会保障部|人力资源和社会保障部]]的关于国务院任免国家工作人员的公告信息，正式宣布“免去李东生的公安部副部长职务”<ref name="bj">{{cite web |url = http://bj.people.com.cn/n/2014/0225/c233086-20640908.html |title = 公安部副部长李东生昨日被正式公告免职 |publisher = 人民网 |date = 2014-02-25 |accessdate = 2014-03-15 |author = 记者 桂田田}}</ref>。同年6月30日，李东生涉嫌严重违纪违法被中共中央开除党籍并移送司法机关依法处理，经查李东生利用职务上的便利为他人谋取利益，索取、收受巨额贿赂<ref >{{cite web|url=http://news.xinhuanet.com/politics/2014-06/30/c_1111388268.htm | language= | title=中央防范和处理邪教问题领导小组原副组长、办公室主任，公安部原党委副书记、副部长李东生严重违纪违法被开除党籍 |publisher=新华网 |date=2014年6月30日 |accessdate= }}</ref>。

2014年6月30日，中共中央纪律检查委员会对[[610办公室|610办公室]]主任李东生严重违纪违法问题进行立案审查，发现李东生利用职务便利为他人谋取利益，索取、收受巨额贿赂，李东生的行为已构成严重违纪，其中受贿问题涉嫌违法犯罪。决定给予李东生开除党籍处分；由监察部报请国务院批准给予其行政开除处分；将其涉嫌犯罪问题及线索移送司法机关依法处理。<ref name="July">[http://www.afinance.cn/new/ffcl/201407/723112.html 李东生严重违纪违法被开除党籍 ， 第一金融网，2014-07-02]</ref>

2015年10月14日，天津市第二中级人民法院一审公开开庭审理李东生受贿案。检方指控，李东生利用其担任中国中央电视台副台长、中华人民共和国公安部党委委员、副书记、副部长、中共中央政法委员会委员等职务上的便利以及职权和地位形成的便利条件，为北京雷明顿广告发展中心某实际控制人、北京韩建集团有限公司董事长等11个单位和个人，在获取广告代理权、调动工作、案件处理、户口办理等事宜上提供帮助。检方还指控李东生直接或通过其亲属索取、非法收受上述单位或个人给予的财物共计折合人民币2198万余元，应以受贿罪追究李东生的刑事责任。该案审理后，法院将择期宣判<ref>{{cite web |title = 公安部原副部长李东生案择期宣判 涉受贿2198万|url = http://news.qq.com/a/20151014/033264.htm |publisher =新华网 |accessdate = 2015-10-14 }}</ref>。2016年1月12日，李东生被判处有期徒刑15年，没收非法所得100万元。
