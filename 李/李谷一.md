'''李谷一'''（{{bd|1944年|11月10日||||CatIdx=Li}}），原名'''李滇惠'''、'''李穀貽'''，1966年改現名。[[中國大陆|中國大陆]]著名女歌唱藝術家，[[湖南|湖南]][[長沙|長沙]]人。[[一級演員|一級演員]]，享受[[國務院特殊津貼|國務院特殊津貼]]。被[[中國文化部|中國文化部]]批准為優秀專家。[[中国共产党|中国共产党]]党员、[[中国民主同盟|中国民主同盟]]盟员<ref>{{Cite web|url=http://politics.people.com.cn/n1/2018/1126/c1001-30420269.html|title=关于改革开放杰出贡献拟表彰对象的公示|accessdate=2018-12-07|work=politics.people.com.cn}}</ref><ref>[http://ent.163.com/16/0509/11/BMKBOTT800031H2L.html 相声演员于谦已低调加入中国国民党革命委员会]</ref>。

李谷一的歌聲輕巧甜美，圓潤流暢，有著濃鬱的民族風格與韻味，富有極強的藝術魅力和感召力。在近30年的舞臺生涯中，以其獨特的[[演唱風格|演唱風格]]和技巧，對中國民族[[聲樂|聲樂]]和[[通俗音樂|通俗音樂]]的發展，起了極大的推動作用，並開創了一代歌風。 

她不僅有豐富的[[舞蹈|舞蹈]]和[[戲曲藝術|戲曲藝術]]功底，而且又學習了[[西洋傳統發聲法|西洋傳統發聲法]]，形成了中西合璧、獨樹一幟的現代[[新民歌|新民歌]]演唱風格，開創了一代歌風。在1980年代，她是中國廣大觀眾最熟悉、最喜愛的歌唱家。

1979年，演唱《鄉戀》引發軒然大波，這首大陸創作者原創的作品在中央電視台播出後，被戴上「低級」、「下流」的帽子，李谷一採用所謂「氣聲唱法」，被指責模仿[[鄧麗君|鄧麗君]]。《[[人民日報|人民日報]]》等媒體強力批判這首歌曲，李谷一因此蒙受巨大的壓力，「靡靡之音」也從這時起正式戴到了李谷一的頭上。不過隨著改革的進行，中國大陸政治開放的風氣日盛，她後來也被作為流行樂曲傳奇之一。

==生平==
===花鼓戲演員===
李穀一1944年11月10日出生於中國[[雲南省|雲南省]][[昆明市|昆明市]][[惠滇醫院|惠滇醫院]]，1961年畢業於[[湖南藝術學院|湖南藝術學院]]舞蹈專修科。

1961年至1974年，她逐步成為湖南省[[花鼓戲|花鼓戲]]劇院的主要演員之一。在十四年裏成功地塑造了二十多個不同時代，不同性格的年輕姑娘形象，其中主演《補鍋》一劇，於1964年和1965年獲湖南省和[[中南|中南]]五省[[戲劇|戲劇]]匯演優秀獎，拍成[[電影|電影]]，從此知名。她曾受到[[毛澤東|毛澤東]]、[[周恩來|周恩來]]等共產黨和國家的領導人的接見。

=== 文革 ===
1970年，李谷一因为花鼓戏《补锅》，被[[中国共产党|中国共产党]]视为“[[修正主义黑苗子|修正主义黑苗子]]”，随后他们抄了李谷一的家，把李谷一下放到瑶寨，[[劳动改造|劳动改造]]。<ref>{{cite web |url=http://eladies.sina.com.cn/qg/2012/0322/23061136241.shtml |title=女儿眼中的李谷一：一半坚强 一半脆弱 |accessdate= 2012年3月22日 |publisher= 新浪网|work= |language =zh-cn}}</ref>

===中央樂團演員===
1974年至1984年，調入[[中央樂團|中央樂團]]擔任[[獨唱|獨唱]]演員。開始時她曾在[[交響樂|交響樂]]《[[智取威虎山|智取威虎山]]》、[[鋼琴|鋼琴]][[伴唱|伴唱]]《[[紅燈記|紅燈記]]》中承擔主要角色。 後來李穀一改學西洋發聲法。1978年受到[[美國|美國]][[總統|總統]][[卡特|卡特]]的接見。1980年以一首《鄉戀》紅遍大江南北，被公認為中國第一位流行歌手。

[[文革|文革]]之後，她參加了《春天》等影片的拍攝工作。此後，進入演唱藝術的新盛時代，先後在國內各省市和國外參加了近兩千場次的演出、參加曆年的重大政治和節日演出活動、參加了[[中國中央電視臺|中國中央電視臺]]和許多省市臺的各種[[電視|電視]]、[[廣播|廣播]]節目播出。為一百多部電影、電視片配唱。共計演唱創作歌曲、電影電視歌曲三百餘首。 

經李穀一演唱的四十多首歌曲，在海內外成為知名歌曲，如《邊疆泉水清又純》、《妹妹找哥淚花流》、《潔白羽毛寄深情》、《心中的玫瑰》《知音》、《鄉戀》、《迎賓曲》、《絨花》等等，這些歌曲多次在中國全國性歌曲評比中獲獎，並獲第三界電影“[[百花獎|百花獎]]”特設的“最佳演唱獎”。錄制的個人專輯磁帶十餘版和唱片多張，銷量曾居國內榜首。1981年至1982年，兩次與[[美國|美國]][[紐約|紐約]][[交響樂|交響樂]][[指揮家|指揮家]][[戴維.吉爾伯|戴維.吉爾伯]]合作，演唱[[交響樂組曲|交響樂組曲]]，其中演唱的中國的《三江組曲》獲文化部二等獎。評論界認為，其演唱風格和技巧，對中國民族聲樂和通俗音樂和發展，有了新的突破，開創了一代歌風。 

=== 乡恋事件 ===
1980年，李谷一演唱《乡恋》，在中国大陆音乐界招致诸多攻击，[[中国共青团|中国共青团]][[团中央|团中央]]随后举行“15首歌评选”，[[王酩|王酩]]写的《小花》和李谷一唱的《乡恋》被当场点名批判，并且说“这是流氓喜欢的歌。”，在[[中国社会科学院|中国社会科学院]]的礼堂里，中共高层指责李谷一是“大陆的[[邓丽君|邓丽君]]”，封杀《乡恋》<ref>{{cite web |url=http://hn.rednet.cn/c/2008/04/21/1489290.htm |title=李谷一唱《相恋》被高官点名批评 |accessdate=  2008-04-21 |publisher= 红网|work= |language =zh-cn}}</ref><ref>{{cite web |url= http://lywb.lyd.com.cn/html/2008-08/01/content_430232.htm|title= 李谷一与“乡恋”被禁风波|accessdate= 2008年8月1日|publisher=洛阳晚报 |work= |language =zh-cn}}</ref>，李谷一决定找[[邓小平|邓小平]]评理，“这首歌到底是不是[[反动|反动]]的、[[黄色|黄色]]的？”但是没有遇到邓小平的机遇，随后，李谷一在[[南京市|南京市]]和[[上海市|上海市]]演唱这首歌，引发轰动，1983年在[[1983年中国中央电视台春节联欢晚会|第一届中国中央电视台春节联合晚会]]上解禁唱出，才得以正名，此歌曲也被评为内地流行歌曲的“开山之作”。<ref>{{cite web |url=http://lady.newssc.org/system/2012/06/17/013551049_02.shtml |title=李谷一从艺50年坎坷路 |accessdate=2012-06-17 |publisher=四川新闻网 |work= |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20131217224443/http://lady.newssc.org/system/2012/06/17/013551049_02.shtml |archivedate=2013-12-17 }}</ref><ref>{{cite web |url=http://roll.sohu.com/20120617/n345827676.shtml |title=李谷一从艺50年坎坷路 |accessdate= 2012年6月17日|publisher=搜狐网 |work= |language =zh-cn}}</ref>

===中國輕音樂團團長===
1982年，李穀一從[[中央樂團|中央樂團]]抽調出來，著手創建[[中國輕音樂團|中國輕音樂團]]的籌備工作。 

1984年至1985年，李穀一參加了大型[[音樂舞蹈史詩|音樂舞蹈史詩]]《[[中國革命之歌|中國革命之歌]]》的演出和同名電影的拍攝，獲表演一等獎。 

1986年中國輕音樂團正式成立，李穀一擔任團長。在[[中國輕音樂團|中國輕音樂團]]十多年中，李穀一籌劃演出了幾套[[輕音樂|輕音樂]]曲目，為中國音樂園地增添新的品種，並且努力為國家培養了一批優秀的乃至全國著名的歌手和演奏員。同時，帶領全團從[[北京|北京]]到全國五十多個省，地區演出千多場，並且參加了許多重大的政治和社會活動。例如，赴[[雲南|雲南]][[老山|老山]]前線慰問演出，為[[美國總統|美國總統]][[羅納德·裏根|裏根]]訪華演出等。受到[[文化部|文化部]]和地方政府、部隊的多次表彰。

在此期間，其演唱藝術再創高峰，《難忘今宵》、《故鄉是北京》、《前門情思大碗茶》、《我和我的祖國》、《劉海砍樵》等，在國內外頗具影響，為中國聲樂藝術拓展了新路。1989年獲[[廣電部|廣電部]][[中國唱片社|中國唱片社]]頒發的首屆“[[金唱片獎|金唱片獎]]”，1991年獲[[文化部|文化部]]“優秀演員獎”、“新曲目優秀獎”。 

曾受到[[鄧小平|鄧小平]]等中共和國家領導人的接見和贊揚。 

1993年，文化部、中央電視臺聯合拍攝和播出了[[藝術專題片|藝術專題片]]《緣分——我說李穀一》，概括了其艱辛探索的藝術生涯。 

===東方歌舞團===
1996年，李穀一調入[[東方歌舞團|東方歌舞團]]，擔任中共[[黨委書記|黨委書記]]、第一副團長。除完成繁忙的黨務工作和業務領導工作外，還完成了中央部委組織的“[[心連心藝術團|心連心藝術團]]”、“[[京九鐵路|京九鐵路]]慰問演出團”、“[[香港|香港]]回歸晚會”、“[[澳門|澳門]]回歸晚會”、“祖國頌”、“七·一晚會”、“[[南昆鐵路|南昆鐵路]]文化列車”以及[[東方歌舞團|東方歌舞團]]演出任務共計二百多場。 

幾年中，多次受到[[江澤民|江澤民]]等中共和國家領導人的接見。 

李穀一曾以中國[[藝術家|藝術家]]身份，十幾次出訪美國、[[法國|法國]]、[[荷蘭|荷蘭]]、[[日本|日本]]、[[新西蘭|新西蘭]]、[[澳大利亞|澳大利亞]]及香港、澳門演出，深受歡迎和好評。1985年在[[法國|法國]][[巴黎|巴黎]]和[[荷蘭|荷蘭]][[阿姆斯特丹|阿姆斯特丹]]、[[鹿特丹|鹿特丹]]等地舉辦了[[獨唱|獨唱]][[音樂會|音樂會]]，獲得極大成功，是[[中國大陸|中國大陸]]第一位歌唱家在這些國家舉辦的[[獨唱音樂會|獨唱音樂會]]。 

曾多次擔任日本、[[德國|德國]]、[[南斯拉夫|南斯拉夫]]、[[哈薩克斯坦|哈薩克斯坦]]、香港等國家和地區的流行樂壇的大賽和國內中央電視臺、文化部等政府以及民間舉辦的各類音樂比賽的評選工作。 

===近況===
最近調任[[中央民族樂團|中央民族樂團]][[藝術指導|藝術指導]]。 

近年來在國內多種報刊上，發表過有關[[演唱藝術|演唱藝術]]及[[聲樂技藝|聲樂技藝]]的論述文章。 

聘為[[沈陽音樂學院|沈陽音樂學院]]客座教授、[[湖南師範大學|湖南師範大學]]藝術系客座教授、[[湖南省女子大學|湖南省女子大學]]名譽校長等。 

被選為[[中國|中國]][[音樂家協會|音樂家協會]]副主席、中國[[輕音樂學會|輕音樂學會]]副主席、[[中國|中國]][[社會經濟文化交流協會|社會經濟文化交流協會]]副會長。是[[中國|中國]][[電視家協會|電視家協會]]會員、[[中國|中國]][[婦女基金會|婦女基金會]]理事、[[中國|中國]][[和平統一促進會|和平統一促進會]]理事。 

自1982年起至今，為[[中國人民政治協商會議|中國人民政治協商會議]]第六屆、第七屆、第八屆、第九屆全國委員。

==作品==
她唱過近百首歌曲，尤以《鄉戀》、《妹妹找哥淚花流》、《知音》、《難忘今宵》等最具傳唱性。 

其他知名歌曲還有《邊疆泉水清又純》、《潔白羽毛寄深情》、《心中的玫瑰》、《迎賓曲》、《絨花》等。

== 亲属 ==
李谷一的前夫是音乐家[[金铁霖|金铁霖]]。李谷一的现任丈夫肖卓能是[[中国人民解放军大将|中华人民共和国开国大将]][[肖劲光|肖劲光]]的儿子，李谷一和他育有一个女儿肖一。<ref>{{cite journal | language =zh-hans | author =学芝、谷雨 | title =《李谷一：歌声飘过五十年》| journal =《[[老年人_(杂志)|老年人]]》 | volume =第6期 | year = 2012年 | pages = 第38页 | issn =1007-2616 }}</ref><ref>{{cite web |url=http://news.sohu.com/20051227/n241150294.shtml |title=李谷一：湖北人是钢丝(组图)  |accessdate=2005 |publisher=搜狐网 |work= |language=zh-hans}}</ref>

==獲獎==

1988年，被具有權威性的[[美國傳記協會|美國傳記協會]]列入《世界傑出人物人名錄》。 

1996年，獲[[美國ABI協會|美國ABI協會]]頒發的“世界藝術家成就獎”的金獎。 

1999年，獲CCTV-MTV（中國中央電視臺與美國[[音樂電視網|音樂電視網]]（[[MTV|MTV]]）頒發的“終身成就獎”。 

2000年7月畢業[[中央黨校|中央黨校]][[導師制|導師制]][[研究生|研究生]]班。
