'''李殿勋'''（1967年11月—），[[河南|河南]][[上蔡县|上蔡县]]人，1991年7月参加工作，1994年6月加入中共，曾任[[重庆市|重庆市]]科委主任，现任重庆市副市长。

== 简历 ==
1989年取得[[华东师范大学|华东师范大学]]历史学学士学位，1991年取得[[西南政法大学|西南政法大学]]法学学士学位；

2000年12月，任[[重庆经济技术开发区|重庆经济技术开发区]]管委会副主任，开发区管委会财政局局长、国有资产公司董事长。

2003年2月，任重庆市人民政府法制办公室公室主任、书记，2003年9月兼任重庆仲裁委员会办公室主任（2004年4月至7月在[[中央党校|中央党校]]地厅级进修班学习）；

2009年3月，任中共[[开县|开县]]县委书记；

2013年2月，任中共重庆市[[南川区|南川区]]委书记；

2014年9月，任[[重庆市人民政府|重庆市人民政府]]副秘书长；

2015年2月，任重庆市科委主任。<ref>{{cite web |url=http://gbzl.people.com.cn/grzy.php?id=140300095 |language= |title=李殿勋现任重庆市科委主任 |publisher=人民网 |date= |accessdate=2017-09 |archive-url=https://web.archive.org/web/20170822015257/http://gbzl.people.com.cn/grzy.php?id=140300095 |archive-date=2017-08-22 |dead-url=yes }}</ref>

2018年1月，当选重庆市副市长。<ref>{{cite news|title=唐良智当选为重庆市人民政府市长|url=http://www.thepaper.cn/newsDetail_forward_1977808|accessdate=2018-01-31|work=华龙网}}</ref>
