'''李君豪'''{{BBS}}（'''Vincent Lee'''），廣東中山人，是[[李東海|李東海]]的兒子<ref name=thesunhk>{{cite web |url=http://the-sun.on.cc/cnt/entertainment/20100701/00470_017.html |title=李東海舉殯好友送別 |publisher=[[太陽報_(香港)|香港太陽報]] |date=2010-07-01 |language=zh-hant }}</ref>，亦是[[香港上市公司|香港上市公司]][[東泰集團|東泰集團]]的現任集團主席，1990年至2010年9月時為該集團之董事總經理<ref name=tungtai_cv>{{cite web |url=http://www.etungtai.com/vincentmlee1.html |title=李君豪 主席 |publisher=[[東泰集團|東泰集團]] |language=zh-hant }}</ref>，從2000年到現在一直為[[香港交易及結算所|香港交易及結算所]]的[[獨立非執行董事|獨立非執行董事]]<ref name=legco2012>{{cite web |url=http://www.elections.gov.hk/legco2012/pdf/introd/U_04_cn.pdf |title=參選政綱 |publisher=[[選舉事務處|選舉事務處]] |language=zh-hant }}</ref>。

李君豪曾於[[2011年香港選舉委員會界別分組選舉|2011年香港選舉委員會界別分組選舉]]參選[[功能組別|功能組別]]第一界別的[[2011年香港選舉委員會界別分組選舉#金融服務界|金融服務界]]<ref name=ecss2011>{{cite web |url=http://www.elections.gov.hk/ecss2011/pdf/intro_to_can/SU_013.pdf |title=李君豪參選政綱 |language=zh-hant }}</ref>，又在[[2012年香港立法會選舉|2012年香港立法會選舉]]參選<ref name=legco2012 />。他被認為是香港政壇親建制派的「梁粉」，在《[[陽光時務週刊|陽光時務週刊]]》引述[[劉夢熊|劉夢熊]]指控時任特首[[梁振英|梁振英]]「恩將仇報」事件中，否認梁振英曾在晚宴上指「建制派唐梁之爭是[[人民內部矛盾|人民內部矛盾]]而跟泛民主派是[[敵我矛盾|敵我矛盾]]」的說法<ref name=sharphk>{{cite news |url=http://www.sharpdaily.hk/realtime/fin/20130124/51238510/%E5%A4%A2%E7%86%8A%E5%87%BA%E6%B2%92-%E6%A2%81%E7%B2%89%E6%9D%8E%E5%90%9B%E8%B1%AA-%E5%A4%A2%E7%86%8A%E8%AC%9B%E5%A4%A7%E8%A9%B1 |title=【夢熊出沒】梁粉李君豪：夢熊講大話 |publisher=[[爽報_(香港)|香港爽報]] |date=2013-01-24 |accessdate=2013-01-25 |language=zh-hant }}</ref>。
