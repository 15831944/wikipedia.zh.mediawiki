作家'''[[李敖|李敖]]'''，自2000年代表[[新黨|新黨]]參選[[2000年中華民國總統大選|2000年中華民國總統大選]]，至2011年，宣布代表[[親民黨|親民黨]]於[[臺北市第八選舉區|臺北市第八選舉區]]投入[[2012年中華民國立法委員選舉|2012年中華民國立法委員選舉]]，共計參選3次，挑戰過[[總統|總統]]、[[立委|立委]]。

唯一一次在[[2004年|2004年]]，以[[無黨籍|無黨籍]]身分參選[[2004年中華民國立法委員選舉|2004年中華民國立法委員選舉]]，成功當選，成為[[中華民國立法院|中華民國立法院]][[2004年中華民國立法委員選舉|第6屆立委]]。

== 参选总统 ==
[[2000年中華民國總統大選|2000年中華民國總統大選]]，面臨泡沫化危機的[[新黨|新黨]]提出奇招，邀請並非新黨黨員的李敖代表新黨競選總統，以拉抬新黨的聲勢。<ref>[http://forums.chinatimes.com/report/vote2000/liao/88081904.htm 接受徵召 李敖決代表新黨參選總統]</ref>副總統候選人則由新黨立委[[馮滬祥|馮滬祥]]擔任。在[[两岸问题|两岸问题]]上，李支持[[中国统一|统一]]，主张两岸举行和平谈判，實行[[一国两制|一国两制]]。在選舉公報上，李敖填寫的學經歷資料只有兩行：

{{quote|國立台灣大學歷史系畢業。<br>[[勃起|勃起]]台灣、挺進大陸、[[威而鋼|威而鋼]]世界<ref>[http://forums.chinatimes.com/report/vote2000/main/89020318.htm 選舉公報 李敖學經歷欄很另類]</ref>}}

在選舉的最後階段，李敖呼籲支持者轉投票給宋楚瑜。<ref>[https://udn.com/news/story/6656/3038051 2000年兩人曾競選總統 李敖最後請支持者把票投宋楚瑜]</ref>選舉結果揭曉，宋楚瑜以三十多萬票差距敗給[[陳水扁|陳水扁]]，李敖自己的得票則僅剩16,782票，得票率0.13%，甚至不如[[無黨籍|無黨籍]]參選的[[許信良|許信良]]（得票率0.63%），成為[[中華民國|中華民國]][[總統|總統]][[直接选举|直選]]以來最低得票率。

李敖在立法委員任期內的主要活動是反对[[民主进步党|民主进步党]][[陳水扁政府|陳水扁政府]]时期[[行政院|行政院]]提出的[[三項軍購案|价值新台币6108亿元对美军购案]]。在国民党反对态度因美方压力而软化时，李敖与[[亲民党|亲民党]]合作推迟了军购案的进程。2005年3月17日，李敖在[[立法院程序委員會|立法院程序委员会]]质询[[中华民国国防部|国防部长]][[李傑_(臺灣)|李杰]]时，拿出狗鏈，直指台湾作为美国的看门狗却要自费购买武器，并亮出小刀，建议李杰「割掉[[𡳞脬|卵葩]]算了」。<ref>[https://udn.com/news/story/6656/3037658?from=udn-catelistnews_ch2 李敖大師短暫卻精采的政治路]</ref>

2006年10月24日，李敖为了抗议朝野各政党议程，以个人实际行动来阻挡当日程序进行。他先指出[[中国国民党主席|中国国民党主席]][[马英九|马英九]]内定要让军购案过关，再戴上[[V怪客|V怪客]]面具造型的[[防毒面罩|防毒面罩]]，表示不畏后果要求立法院程序委员会本次会议中止，希望在场委员离席；并出示[[电击棒|电击棒]]、[[防狼喷雾|防狼喷雾]]，对空喷射[[催淚彈|催淚彈]]，中断议事约二十分钟，導致許多立委吸入「[[催淚瓦斯|催淚瓦斯]]」，一身狼狽奪門而出。<ref>[http://www.appledaily.com.tw/appledaily/article/headline/20061025/2981192/ 狂喷瓦斯「李敖疯了」奇招阻軍購 王世堅痛罵].蘋果日報.2006-10-25</ref>11月10日，为了抗议军购案通过，李敖在总质询中秀出了自己26歲當[[預官|預官]]時所拍的的[[裸照|裸照]]大海報<ref>相关文章：[http://lis.ly.gov.tw/ttscgi/lgimg?@954801;0032;0038 院会纪录，《立法院公报》，第95卷48期]</ref>，由於照片露出[[阳具|阳具]]，是「全裸遛鳥」<ref>陳明旺、黃世宏(圖)[https://tw.appledaily.com/forum/daily/20061111/23023839 李敖秀裸照「好有自信」]. 蘋果日報.2006-11-11</ref>，嚇壞或笑壞在場立委及議事人員，也讓[[閣揆|閣揆]][[蘇貞昌|蘇貞昌]]在台上哭笑不得。<ref>[https://www.ettoday.net/news/20180318/1132822.htm 李敖「裸照」嚇壞議事人員　質詢風格獨特讓蘇貞昌哭笑不得].ETtoday.2018-03-18</ref>

2006年底，李敖還以参选[[2006年中华民国直辖市市长暨市议员选举|台北市长]]的方式宣扬其政治理念，但仅获7795票，得票率0.61%。

== 离开政坛 ==
2007年9月20日，李敖表示将投入[[2008年中华民国立法委员选举|2008年中华民国立法委员选举]]及组织政党。他将[[户籍|户籍]]由[[台北市|台北市]][[大安區_(臺北市)|大安区]]迁至[[文山區|文山区]]，准备参与文山区的立法委员选举，而同区的候选人包括中国国民党提名的[[赖士葆|赖士葆]]。9月30日，李敖以宣布成立政党「中国智慧党」的方式，提出十点政治理念，但不成功。两个月后，李敖发表新书《李敖议坛哀思录》，宣布告别政坛，专职写作。

== 再選立委 ==

2011年8月10日，李敖宣布代表[[親民黨|親民黨]]於[[臺北市第八選舉區|臺北市第八選舉區]]投入[[2012年中華民國立法委員選舉|2012年中華民國立法委員選舉]]，與[[中國國民黨|中國國民黨]]時任立法委員[[賴士葆|賴士葆]]對壘，聲稱特別選擇時任[[中華民國總統|中華民國總統]][[馬英九|馬英九]]的故鄉（[[文山區|文山區]]），希望藉由參選使特定媒體無法再封鎖其言論。此次選舉，李敖和2004年一樣，不進行傳統的競選活動。他對記者坦言此次當選機會渺茫，是「玩票性質的犧牲打」，主要目的是幫助親民黨越過政黨得票百分之五門檻，得到[[不分區立委|不分區立委]]席位。<ref>[http://www.bbc.com/zhongwen/trad/chinese_news/2012/01/120103_tw_election_lee_ao.shtml 李敖专访：玩票性质的牺牲打]</ref>

最終李敖以9,436得票數（得票率5.08%）慘敗給賴士葆的117,892得票數（得票率63.43%），但親民黨取得了不分區5.49%得票率，勉強到達不分區立委門檻。
