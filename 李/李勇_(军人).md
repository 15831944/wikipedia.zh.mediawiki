'''李勇'''（{{bd|1960年|7月||}}），[[北京市|北京市]]人，[[中国人民解放军少将|中国人民解放军空军少将]]。

毕业于[[中国人民解放军空军工程大学|空军工程大学]]。在空军期间，历任飞行中队长、大队长、某试训中心三团团长、空44师师长、空军飞行试验训练某基地司令员。2009年，出任[[空军指挥学院|空军指挥学院]]副院长，后改任[[兰州军区空军|兰州军区空军]]参谋长、副司令员等职务。2016年，任[[中部战区|中部战区]]副参谋长<ref>{{cite web |title=空军少将李勇晋升副战区级 曾任中部战区副参谋长 |url=http://news.ifeng.com/a/20180305/56467294_0.shtml |website=凤凰网 |accessdate=2018-12-01 |date=2018-03-05}}</ref>。2017年12月，升任[[中央军委联勤保障部队|中央军委联勤保障部队]]司令员<ref>{{cite web |title=【创刊号】讴歌新时代 筑梦新联勤 |url=https://mp.weixin.qq.com/s/Q1Ru_qwF9rvb71z59KObLg |website=联勤集结号（中央军委联勤保障部队官微） |accessdate=2019-02-21 |date=2019-02-19}}</ref><ref>{{cite web |author1=王俊 |title=副战区级空军少将李勇已担任联勤保障部队司令员 |url=https://www.thepaper.cn/newsDetail_forward_2695258 |website=澎湃新闻 |accessdate=2018-12-01 |date=2018-12-01}}</ref>。

李勇是[[第十三届全国人大代表|第十三届全国人大代表]]<ref>{{cite web |title=代表信息 |url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=200813281972 |website=中国人大网 |accessdate=2018-12-01}}</ref>。
