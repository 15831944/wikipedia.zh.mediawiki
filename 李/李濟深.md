'''李济深'''（{{bd|1885年||1959年|10月9日|catIdx=L李}}），[[广西|广西]][[苍梧|苍梧]]人，[[表字|字]]'''任潮'''<ref name="海">{{cite book|editor=辭海編輯委員會|title=《辭海》（1989年版）|year=1989|publisher=[[上海辭書出版社|上海辭書出版社]]}}</ref>{{rp|3319}}。[[祖籍|原籍]][[江苏|江苏]]。中华民国国民党军人，政治家。[[中华民国|中华民国]]陆军二级上将。曾于[[黄埔军校|黄埔军校]]任教，[[北伐|北伐]]期间支持[[蒋中正|蒋中正]]的[[四一二事件|四一二事件]]，之后又多次反蒋，曾于[[福建事变|福建事变]]时成立[[中华共和国人民革命政府|中华共和国人民革命政府]]任政府主席。[[中國抗日戰爭|中國抗日戰爭]]爆發後，響應中國共產黨一致抗日之號召，反對國民政府之反共政策<ref name="海"/>{{rp|3319}}。[[抗战|抗战]]胜利后脱离蒋中正之[[中国国民党|中国国民党]]。1948年發起成立[[中国国民党革命委员会|中国国民党革命委员会]]，任主席<ref name="海"/>{{rp|3319}}。[[中华人民共和国|中华人民共和国]]成立后，历任[[中央人民政府副主席|中央人民政府副主席]]、[[全国人大常委会副委员长|全国人大常委会副委员长]]、[[全国政协副主席|全国政协副主席]]<ref name="海"/>{{rp|3319}}。

== 教育至黃埔 ==
[[File:孫中山先生紀念碑奠基.JPG|thumb]]奠基，由李濟深所撰寫。]]
[[File:孫中山先生紀念碑碑文.jpg|thumb]]》，同樣由李濟深所撰寫。]]
李济深曾在[[梧州市|梧州市]]中学堂（现[[梧州市第一中学|梧州市第一中学]]）读书，是[[胡汉民|胡汉民]]的學生。后转学广东黄埔两广将弁学堂（與广东陆军中学堂合併，后更名为[[廣東陸軍速成學校|廣東陸軍速成學校]]），与[[邓铿|邓铿]]为同学。之后入读保定陆军军官学堂（[[保定陸軍軍官學校|保定陸軍軍官學校]]前身）。1912年[[辛亥革命|辛亥革命]]后，任广东北伐军第二十二师参谋长。北京陆军大学毕业<ref name="海"/>{{rp|3319}}，后留校任教。1920年应邓铿之邀，任[[陈炯明|陈炯明]]建国粤军第一师参谋长。邓铿被杀后，[[梁鸿楷|梁鸿楷]]继任。不久，李任粵军第一师代理师长<ref name="海"/>{{rp|3319}}。1922年冬参与策动粤军反戈驱逐[[陈炯明|陈炯明]]。1923年任梧州西江善后处长，曾动员[[李宗仁|李宗仁]]、[[黄绍竑|黄绍竑]]加入国民党；有「半个[[桂系|桂系]]」之称。1924年[[黃埔軍校|黃埔軍校]]建校，李济深任教练部少将主任。1925年[[国民政府|国民政府]]在广州成立，建国粤军第一师改編成[[国民革命军第四军|国民革命军第四军]]，李济深任第四军军长<ref name="海"/>{{rp|3319}}。第四军之下辖两个师及一个独立团。

== 北伐至四一二事件 ==
1926年，[[北伐|北伐]]开始，李济深兼任国民革命军总司令部参谋长<ref name="海"/>{{rp|3319}}，[[黄埔军校|黄埔军校]]副校长<ref name="海"/>{{rp|3319}}，总司令部留守主任，[[广东省政府|广东省政府]]主席等职。1927年4月[[宁汉分裂|宁汉分裂]]之际，李济深到[[上海|上海]]參加蒋中正在上海召开的会议，返回广州后展开反共，參加“[[广州四·一五事件|广州四·一五事件]]”<ref name="海"/>{{rp|3319}}。之后[[南京|南京]]國民政府成立，李任国民政府委员，军事委员会总参谋长，政治会议广州分会主席，第八路军总指挥（由第四军扩充）。同年8月共产党在[[南昌|南昌]]发动[[南昌起义|南昌起义]]，李派大军到[[潮州|潮州]]、[[梅州|梅州]]一帶镇压。

== 闽变至抗战 ==
1929年3月蔣中正和[[桂系|桂系]]爆发[[蒋桂战争|蒋桂战争]]。李济深当时手握广东军政大权，3月12日去南京斡旋蒋桂战争，3月21日蒋介石请他去赴宴，被扣留在汤山，並被永远开除党籍，监禁两年直至九一八事变后获释并恢复党籍。广东则为李济深原部下，拥蔣的[[陈济棠|陈济棠]]所据。

1933年與[[方鼎英|方鼎英]]、第十九路军[[陈铭枢|陈铭枢]]等組成「革命军事委员会」，提出拥胡反蔣，抗共抗日，再被南京永远开除党籍。同年11月與陳銘樞等在[[福州|福州]]成立抗日反蒋的「[[中华共和国|中华共和国]]人民革命政府」<ref name="海"/>{{rp|3319}}（称[[福建事变|福建事变]]），任主席兼[[中央军事委员会|军委]]主席<ref name="海"/>{{rp|3319}}。提出与中共合作。事败后，李出走香港。1935年7月25日，李济深与陈铭枢、蒋光鼐、蔡廷锴等原十九路军将领在香港建立了“中华民族革命同盟”，简称“大同盟”，李济深为主席。

1937年抗日戰爭爆發後，國民政府撤銷對李濟深的通缉，任李為軍事委員會委員，桂林行营主任，軍事参議院院長等職。1944年8月，乘湖南战局危急，暗通美国顾问[[史迪威|史迪威]]，密谋另立政府，迫蒋下野<ref>{{cite book|author=[[郭廷以|郭廷以]]|title=《近代中國史綱》|url=http://books.google.com/books?id=cNpdWZddi3kC&pg=703|year=1986|location=香港| publisher=中文大學出版社| isbn=978-962-201-352-0}}</ref><ref>{{cite book|author1=姜平|author2=罗克祥|title=《李济深传》|series=中国现代人物研究丛书|url=http://books.google.com/books?id=S0RNAAAAMAAJ|year=1993|location=北京|publisher=档案出版社|quote=1944年8月，在衡阳沦陷前后，李济深鉴于抗战形势十分严重，曾联络[[龙云|龙云]]、[[余汉谋|余汉谋]]、[[张发奎|张发奎]]、[[薛岳|薛岳]]等将领，并争取史迪威将军给予美国军事物资的援助，准备在西南地区独立坚持抗战。后因史迪威被美国总统召回美国，国内形势也发生了变化，这一计划未能实现。|pages=354-355頁|chapter=《李济深年谱》|chapterurl=http://book.ln.chaoxing.com/ebook/detail.jhtml?id=10290176&page=358}}</ref>。1945年，李被選為國民黨中央監察委員。

== 內戰至民革 ==
1947年2月，李济深前往香港，公开反蒋。1947年3月9日，李济深在[[香港|香港]]发表《对时局意见》，号召[[中国国民党|中国国民党]]內人士「站出来改正反动派错误」，再被中国国民党开除党籍。1947年5月份被永远开除出国民党。1948年1月，在[[香港|香港]]成立[[中国国民党革命委员会|中国国民党革命委员会]]，推举[[宋庆龄|宋庆龄]]为荣誉主席，李济深任中央委员会主席，[[何香凝|何香凝]]，[[冯玉祥|冯玉祥]]等原中国国民党党员被推为委员。同年5月，接任[[中国致公党|中国致公党]]中央主席，赴东北解放区。1949年1月22日，「發表時局聲明」，響應中國共產黨1948年5月1日提出召開「新政協」，「解決國是之主張」<ref>[[蔣經國|蔣經國]]《危急存亡之秋》，刊《風雨中的寧靜》，[[台北|台北]]，[[正中書局|正中書局]]，1988年，第140頁。</ref>。9月，到[[北平|北平]]参加中共发起第一次[[政协会议|政协会议]]。10月，任中华人民共和国[[中央人民政府|中央人民政府委员会]]副主席、全国政协副主席。之后在1954年任全国人大常委会副委员长、全国政协副主席。1959年10月9日因胃癌在[[北京|北京]]病逝，终年74岁。

== 子女后代 ==
李济深有子女多人，当中[[李沛文|李沛文]]为民革领导人，政协委员。李沛文1985年死后，由其弟[[李沛瑶|李沛瑶]]代替。李沛瑶后任民革主席，人大常委会副委員長，於1996年2月2日被其警卫战士张金龙潛入行竊時用刀杀害。
* 儿子[[李沛文|李沛文]]、李沛金、李沛钰、李沛琼、[[李沛瑶|李沛瑶]]、李沛钤
* 女儿李筱薇、李筱莲、李筱桐、李筱菊、李筱松
