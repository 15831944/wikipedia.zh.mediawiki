'''李文材'''（{{lang-ms|Lee Boon Chye}}；{{bd|1959年|9月26日}}），男，漢族，祖籍[[福建省|福建省]][[泉州市|泉州市]][[永春縣|永春縣]]。[[马来西亚|马来西亚]]政治人物，为[[公正党|公正党]]政治局委员兼[[霹雳州|霹雳州]]署理主席。他也是现任[[马来西亚卫生部|马来西亚卫生部]]副部长及{{pc|务边}}国会议员<ref>''[http://www.parlimen.gov.my/index.php?parti=All&p=All&Filter=Lihat&uweb=dr&modload=ahlidewan Senarai Ahli Dewan Rakyat]''. [[马来西亚国会|Parlimen Malaysia]], [2012-7-23]</ref><ref>{{citeweb|url=http://www.chinapress.com.my/?p=1289086|title=希盟政府完整内阁阵容·终于出炉|website=[[中国报|中国报]]|date=2018-07-02}}</ref>。

==简历==
2004年大选，李文材首次获委派为公正党务边国席候选人，虽然没有中选，但他依然在当地继续为务边选民服务。

2008年，在[[2008年马来西亚大选|全国大选]]中，马华当时的国会议员陈祖排退位让贤，由当时的马青署理总团长林熙隆，以新兵姿态为马华公会守城，对垒第二次在原区上阵的公正党候选人李文材。结果林熙隆只得2万2328票，李文材则有2万9694张票的支持，多数票7千368张。<ref>林熙隆務邊敗北‧7368票負李文材{{cite web |url=http://www.guangming.com.my/node/22981 |title=存档副本 |accessdate=2014-02-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20150924024909/http://www.guangming.com.my/node/22981 |archivedate=2015-09-24 }}</ref>在选民的支持下，李文材医生当选为务边区国会议员。 

[[File:Parlimen_Gopeng.JPG|缩略图]]
2013年，在第十三届全国大选，马华派出该党霹雳州联委会秘书拿督陈进明，为务边国会选区候选人<ref>李文材實力強 陳進明缺人和 反風猛吹 務邊苦戰{{cite web |url=http://www.chinapress.com.my/node/418425 |title=存档副本 |accessdate=2014-02-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20140303080441/http://www.chinapress.com.my/node/418425 |archivedate=2014-03-03 }}</ref>，李文材以1万5千309张多数票蝉联务边国会议员，多数票是308大选7千368张的双倍，他以4万7千558票打败得票3万2千249张的陈进明<ref>投票率高多數票增‧民聯務邊4比0全勝 [http://mykampung.sinchew.com.my/node/236555]</ref>。

== 背景与学历 ==
李文材医生来自[[柔佛州|柔佛州]][[昔加末县|昔加末县]]里的一个偏僻小新村－东猛新村，这新村只有70户人口和一间三间课室的微型华小，和多数村民一样、其父母也是靠割胶维生。新村的生活，让他切身体会新村所面对的问题、如缺乏就业机会、缺乏健康娱乐场所与社区中心，基本设施不足等等，加上每年新村拨款严重缺乏，导致新村发展停歇，50年外貌如一日、年青人外流和新村人口老化<ref>大选履历表 https://www.facebook.com/leeboonchye/photos/pb.184434428768.-2207520000.1393568990./10151574713818769/?type=3&permPage=1</ref>。中学时于昔华中学就读，1977年毕业。

李文材医生的学术表现优良。在1980年以优秀的高等教育文凭的成绩进入马来亚大学就读于医药学系。过后在国民大学和英国爱丁堡完成内科和心脏专科进修，在1994年定居于怡保执业至今。

==争议==
李文材曾在2018年8月20日表示医生和护士在巡房并讲解病人的情况时只能使用马来语或者英语为沟通语言。这是因为《新海峡时报星期刊》的一篇报导指出马来西亚医药界有一些“讲中文的医生”在巡房并向实习医生讲解病人情况时，仅使用[[马来西亚华语|华语]]或其他方言沟通，尤其是在华人人口多的市区医院，导致不谙中文的实习医生无法理解，影响培训。但他也澄清说，当医生与病人直接沟通时，可以使用另一种病人可以理解的语言。<ref>{{cite news|url=https://chinese.malaymail.com/chinese/malaysia/article/20180820-only-bm-and-english-on-ward-visits-from-now-doctors-told|title=李文材：医生巡房只能说国语或英语|date=2018-08-20|publisher=[[马来邮报|马来邮报]]}}</ref>
