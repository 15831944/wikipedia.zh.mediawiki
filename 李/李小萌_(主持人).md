== 简介 ==
1973年，李小萌出生于[[北京市|北京市]]，大学就读于[[中国传媒大学|中国传媒大学]]，大学毕业后分配到中国中央电视台工作。<ref>{{cite web |url=http://news.sina.com.cn/c/2009-02-26/152117294711.shtml |title= 央视主持人李小萌简介|publisher=新浪网 |accessdate=2009-02-26 |work= |language = }}</ref>

== 个人生活 ==
2012年1月13日，李小萌在北京生下一个女儿。<ref>{{cite web |url= http://ent.163.com/12/0129/10/7OU8V9IH00031H2L.html|title=央视主播李小萌低调生女 不愿透露丈夫身份 |publisher= 网易|accessdate= 2012-01-29|work= |language = }}</ref>
