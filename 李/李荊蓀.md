'''李荊蓀'''（{{Bd|1917年|2月6日|1988年|2月12日}}），是一位出身[[無錫|無錫]]、定居[[臺灣|臺灣]]的[[新聞從業員|新聞從業員]]及[[白色恐怖_(台灣)|白色恐怖]]受難者。他是[[新聞雜誌|新聞雜誌]]《新聞天地》共同[[創辦人|創辦人]]，曾任《[[中央日報|中央日報]]》駐南京及臺北[[總編輯|總編輯]]、[[中國廣播公司|中國廣播公司]]副[[總經理|總經理]]、《[[大華晚報|大華晚報]]》董事長、《[[中國時報|中國時報]]》[[主筆|主筆]]等職務。<ref>{{Cite book|url=https://books.google.com.tw/books?id=uTA2AQAAIAAJ&q=%E6%9D%8E%E8%8D%8A%E8%93%80%E6%A1%88%E5%8F%B2%E6%96%99%E5%BD%99%E7%B7%A8&dq=%E6%9D%8E%E8%8D%8A%E8%93%80%E6%A1%88%E5%8F%B2%E6%96%99%E5%BD%99%E7%B7%A8&hl=zh-TW&sa=X&ved=0ahUKEwiP4arm-vrYAhUCy7wKHbKiBTMQ6AEIJTAA|title=李荊蓀案史料彙編|last=王正華|first=|date=2008|publisher=國史館|year=|isbn=|location=|pages=|language=zh-TW|chapter=}}</ref><ref name=":0">{{Cite web|url=http://www.readingtimes.com.tw/readingtimes/ProductPage.aspx?gp=productdetail&cid=rtz00(SellItems)&id=Z00665&p=excerpt&exid=36743|title=李荊蓀先生事略|accessdate=2018-01-28|author=憑志翔|date=|work=www.readingtimes.com.tw|publisher=}}</ref>

== 生平 ==
1917年（農曆[[正月十五|正月十五]]）2月6日，李荊蓀出生於無錫；他的父親是家鄉[[徽州|徽州]]的李靜園，母親是家族世居無錫西村頭的馮氏，他是李家唯一的[[男孩|男孩]]。四歲時，他的父親逝世，母親於是帶著他遷至[[娘家|娘家]]依附[[外祖父|外祖父]]。此後，他受外祖父母及[[姨母|姨母]][[舅父|舅父]]教養，並在六歲時入無錫縣立第一[[小學|小學]]。畢業後，他突然罹患[[瘧疾|瘧疾]]，並因久病不癒而輟學[[在家自學|在家自修]]。<ref name=":0" />

十七歲時，他接獲同學樂毅自[[福州|福州]]寫信告知可前往同任《[[南方日報|南方日報]]》[[校對|校對]]工作，並開始進入[[新聞|新聞]]界工作；後來，在報社選拔人才下，他被保送重慶[[中央政治學校|中央政治學校]]新聞班就讀，並於1940年7月[[卒業|卒業]]。<ref name=":0" />此後，他曾撰寫文章表達對於[[梁寒操|梁寒操]]及[[蔣經國|蔣經國]]改革行動的支持，又曾因反對[[國民政府|國民政府]]制訂《[[出版法|出版法]]》，而遭有關當局斥責並關注。

後來，他在[[周至柔|周至柔]]及[[黃杰_(將軍)|黃杰]]出任[[臺灣省政府主席|臺灣省政府主席]]都曾被提名擔任[[臺灣省政府新聞處|臺灣省政府新聞處]]處長，但都遭國民黨中部分成員反對。稍後，周至柔又在出任[[國家安全會議_(中華民國)|國家安全會議]]主任委員時聘請他與[[丁中江|丁中江]]為其建設計畫委員。當時，周至柔與蔣經國及其各自人馬面臨競爭[[內閣|內閣]]職位；他在《大華晚報》每周發表的「星期雜感」內容因而成為特務當局關注對象。同時，在臺南新聞界工作的[[俞棘|俞棘]]因[[白色恐怖|白色恐怖]]案件被捕入獄，[[情治|情治]]當局也在整理其人[[社會關係|社會關係]]時發現俞氏早年曾在與李荊蓀為同事，又在俞氏所在搜查到李荊蓀於1948年在南京寫給俞氏的一封信件；當局於是暗示俞棘「立功贖罪」。隨後，俞棘一口咬定李荊蓀與其一同在福建參加過[[共產黨|共產黨]]相關[[讀書會|讀書會]]，並以當時拍攝的的一張合照作為[[證據|證據]]。<ref name=":1" />

1970年11月，李荊蓀經[[臺灣警備總司令部|臺灣警備總司令部]]約談後被[[逮捕|逮捕]]，又經日夜連續不停[[審訊|審訊]]，終於在身心難以支持下被迫[[供認|供認]]「加入共黨」；後來，雖然[[黃少谷_(政治)|黃少谷]]、周至柔等人寫信為其[[擔保|擔保]]其對於國民黨的[[忠誠|忠誠]]，但[[法務部調查局|調查局]]仍以其已供認不諱為由加以否定拒絕。在長達一年的審訊中，他終於編造完成自己受南京《[[新民報|新民報]]》女記者[[浦熙修|浦熙修]]指揮的[[故事|故事]]；當時，調查局的人員曾以死亡威脅其服從配合。<ref name=":1" />

1971年接受公開審判時，他在法庭裡遇見了[[妻子|妻子]]和新聞同業，並當場推翻先前所編造的供詞，疾聲高呼其所受之冤屈，又在與俞棘對質時對其說道：「[[良心|良心]]！良心！」；隨後，[[軍事法庭|軍事法庭]]當局宣判須加重其所受處分以[[懲罰|懲罰]]其不合作態度，並因而判以[[無期徒刑|無期徒刑]]。<ref name=":1" />

總統[[蔣中正|蔣中正]]逝世後，他被改判[[有期徒刑|有期徒刑]]十五年。1983年，[[成舍我|成舍我]]、[[程滄波|程滄波]]、胡健中、[[胡秋原|胡秋原]]、[[卜少夫|卜少夫]]等出身新聞界的[[立法委員|立法委員]]聯名上書總統蔣經國，他因而被由[[綠島人權文化園區|綠島]]移至[[台灣仁愛教育實驗所|台灣仁愛教育實驗所]]繼續服刑。<ref>{{Cite web|url=http://www.readingtimes.com.tw/readingtimes/ProductPage.aspx?gp=productdetail&cid=rtz00(SellItems)&id=Z00665&p=excerpt&exid=36744|title=急人之難，奮不顧身──卜少夫這個人|accessdate=2018-01-28|author=李荊蓀|date=|work=www.readingtimes.com.tw|publisher=}}</ref>

1985年11月17日，他被釋放出獄<ref name=":2">{{Cite web|url=http://www.readingtimes.com.tw/readingtimes/ProductPage.aspx?gp=productdetail&cid=rtz00(SellItems)&id=Z00665&p=excerpt&exid=36742|title=關於星期雜感|accessdate=2018-01-28|author=方佩倩|date=|work=www.readingtimes.com.tw|publisher=}}</ref>；不久，他應《中國時報》聘請擔任其主筆。

1988年2月12日，他因[[心肌梗塞|心肌梗塞]]逝世。<ref name=":1">{{Cite web|url=http://www.readingtimes.com.tw/readingtimes/ProductPage.aspx?gp=productdetail&cid=rtz00(SellItems)&id=Z00665&p=excerpt&exid=36745|title=千古奇冤|accessdate=2018-01-28|author=[[陸鏗|陸鏗]]|date=|work=www.readingtimes.com.tw|publisher=}}</ref>

== 私人生活 ==
他與方佩倩[[結婚|結婚]]並育有數名子女。<ref name=":2" />

== 語錄 ==
* 「於今遭此不白之冤，夫復可言，本人在[[失望|失望]]之餘，決放棄任何[[上訴|上訴]]機會，以示內心[[抗議|抗議]]。」<ref>{{Cite web|url=http://www.readingtimes.com.tw/readingtimes/ProductPage.aspx?gp=productdetail&cid=rtz00(SellItems)&id=Z00665&p=excerpt&exid=36746|title=「綠島報人」悄然走了──敬悼李荊蓀先生|accessdate=2018-01-28|author=[[司馬文武|司馬文武]]|date=1988-03|work=www.readingtimes.com.tw|publisher=}}</ref>

== 紀念 ==
* 1988年3月12日，成舍我、[[馬星野|馬星野]]、卜少夫、耿修業、楊乃藩、[[葉明勳|葉明勳]]、丁中江等數十人在[[臺北|臺北]]組成「李荊蓀先生治喪委員會」，以為其舉行[[追思会|追悼會]]。<ref name=":1" />
