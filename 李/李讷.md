'''李讷'''{{noteTag|1=毛泽东女儿叫李讷、李敏，取自《[[论语|论语]]》中的“君子欲讷于言而敏于行。”<ref name=" fish" /><ref name="毛泽东为家人取名字">{{cite journal zh |author=卢荣泰 |year=2008 |title=毛泽东为家人取名字 |journal=文史春秋 |issue= 12 |pages=47-50 |url=http://dangshi.people.com.cn/GB/85038/8990412.html }}</ref>。“讷”字在现代汉语中读音为“nè”，但在这里读作“nà”。1985年12月修订的《普通话异读词审音表》公布前，“讷”有nè、nà两种读音。上海辞书出版社编《辞海》中的“讷”字条目：[讷]（nè，旧读nà）。此外，据毛泽东贴身专职摄影师吕厚民所言，毛泽东本人在称呼李讷时亦是读nà。}} （{{bd|1940年|8月|catIdx=Li}}），出生于延安，籍貫[[湖南|湖南]][[湘潭|湘潭]]，[[毛泽东|毛泽东]]与[[江青|江青]]独生女儿。

== 生平 ==
=== 早年生涯 ===
1940年8月，李讷生于[[陝西|陝西]][[延安|延安]]，幼年深受父亲毛泽东的影响，喜好[[中国古典文学|中国古典文学]]和[[俄国文学|俄国文学]]。<ref name=" fish" >{{cite web |url=http://news.sohu.com/20060901/n245121342.shtml |title= “红色公主”李讷生平 |accessdate=2006年9月1日 |publisher= 搜狐网|archiveurl=http://news.sohu.com/20060901/n245121342.shtml |archivedate=2012年11月3日|work= |language =zh-cn}}</ref><ref name=" tiger" >{{cite web |url=http://www.chinanews.com/news/2004/2004-10-21/26/497147.shtml |title=《中华文摘》文章：李讷不平静的生活 |accessdate=2004年10月21日  |publisher=中国新闻网 |archiveurl=http://www.chinanews.com/news/2004/2004-10-21/26/497147.shtml |archivedate=2012年11月3日|work= |language =zh-cn}}</ref> 1949年，李讷跟着父母搬进北京居住，在育英小学读三年级。<ref name=" tiger" /> 1953年，李讷考入[[北京师范大学附属女子中学|北京师范大学附属女子中学]]，比姐姐李敏低一届。<ref name=" tiger" /> 1959年，李讷用优秀的成绩考取[[北京大学历史系|北京大学历史系]]，1966年毕业。<ref name=" tiger" /> 毕业后，被分配到《解放军报》当编辑，以“'''肖力'''”姓名。<ref name=" fish" /><ref name=" tiger" /> 27岁时，李讷就任《解放军报》总编辑。<ref name="fox " >{{cite web |url=http://news.xinhuanet.com/school/2005-12/07/content_3887173.htm |title= 毛主席之女李讷的四段爱情:不找高干子弟|accessdate=　2005年12月07日 |publisher= 新华网|archiveurl= http://news.xinhuanet.com/school/2005-12/07/content_3887173.htm|archivedate=2012年11月3日|work= |language =zh-cn}}</ref>

=== 文革生涯 ===
1967年1月13日，李讷组织“[[革命造反突击队|革命造反突击队]]”，贴出《解放军报向何处去》的[[大字报|大字报]]“揭批”[[胡痴|胡痴]]（其时任[[新华社|新华社]]代社长、全军文革小组成员）、[[宋琼|宋琼]]以及[[杨子才|杨子才]]等领导，左右了报社的运动方向。<ref>{{cite web |url=http://view.news.qq.com/a/20120802/000008.htm |title= 知情人讲述李讷在“文革”初期的活动|accessdate=2012年8月2日 |publisher=腾讯网 |archiveurl= http://view.news.qq.com/a/20120802/000008.htm|archivedate=2012年11月3日|work= |language =zh-cn}}</ref> 1月17日，[[林彪|林彪]]签署的《给解放军报社革命同志的一封信》（毛泽东批示“同意，这样答复好”），肯定这一行动“在报社内部点起了革命火焰”。先后担任军报版面组（一说“[[中央文革记者站|中央文革记者站]]”《[[快报_(文革)|快报]]》）组长、《解放军报》总编领导小组组长（相当于总编辑）。同年1月，由于她与毛泽东及江青的身份关系，上任[[中央文革小组|中央文革小组]]办事组负责人。1967年8月23日，肖力的第三张大字报出台：“反复辟，反保守，誓将革命进行到底！”1973年参加[[中国共产党第十次全国代表大会|中国共产党第十次全国代表大会]]。1974年至1975年先后任中共[[北京市|北京市]][[平谷县|平谷县]]委书记和北京市委副书记。

=== 晚年生涯 ===
1976年，文化大革命结束，李讷在政治上的职务被取消，曾一度赋闲，由中央办公厅分配到警卫局宿舍居住，两周探望一次母亲江青。<ref name=" fish" /><ref name=" tiger" /> 1984年江青保外就医时一度与其同住。1986年，李讷分配到到[[中共中央办公厅|中共中央办公厅]]秘书局工作，1990年代后退休。<ref name=" tiger" /> 李讷住在北京[[万寿路_(北京)|万寿路]]一处四室一厅的单元房。有一专用书房，喜读史书。[[书法|书法]]学其父、其母。除了参加家族祭奠、毛澤東銅像落成典禮、[[紅歌|紅歌]]會等活動以外，深居简出，省吃俭用，把自己多年积下的十万元人民币捐献给了[[南街村|南街村]]。2003年，李讷任[[中国人民政治协商会议第十届全国委员会委员名单|全国政协委员]]。

== 个人生活 ==
文革初期，李讷的母亲江青为其物色丈夫人选，[[浙江美术学院|浙江美术学院]]学生[[张永生|张永生]]为江青中意的人选，他是浙江省革命委员会副主任、红卫兵头目，经过[[张春桥|张春桥]]的试探，发现张永生为人独断专行、自行其是、目中无人，于是报告给江青，江青打消了把张永生选为女婿的意思。<ref name="fox " />

1970年，李讷到[[江西省|江西省]][[进贤县|进贤县]][[五·七干校|五·七干校]]劳动，爱上了中央办公厅北戴河管理处所属的内部招待所的服务员小徐，但是遭到母亲江青的反对，父亲毛泽东主张婚姻自由，两人就结了婚，生下儿子徐小宁。<ref name="fox " /> 毛泽东以一套《马列选集》相贺。但是两人隔阂加深之后，离婚。

1984年，毛泽东的卫士长[[李银桥|李银桥]]和李银桥妻子[[韩桂馨|韩桂馨]]做媒，李讷认识了王景清{{noteTag|1=王景清，1929年生，陕西[[神木县|神木县]]人，延安警卫团战士，曾任[[昆明军区|昆明军区]]怒江军分区参谋长 }} ，两人结婚。<ref name="fox " /> 而其子“徐小宁”也改名“王效芝”，由打工而经商。<ref name=" fish" /><ref>{{cite web|url=http://book.sina.com.cn/excerpt/sz/rw/2010-11-19/1703280159.shtml|title=李讷在毛泽东逝世江青入狱后艰难再婚|accessdate=2010年11月19日|publisher=新浪网|archiveurl=https://web.archive.org/web/20120313095034/http://book.sina.com.cn/excerpt/sz/rw/2010-11-19/1703280159.shtml|archivedate=2012年3月13日|work=|language=zh-cn|deadurl=yes}}</ref>
