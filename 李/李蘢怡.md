'''李蘢怡'''（{{lang-en|Tiffany Lee}}，{{bd|1982年|9月23日|catIdx=L李}}），原名'''李亮頤'''，別名'''李靜頤'''，又名'''李宜錚'''<ref>[http://hk.apple.nextmedia.com/entertainment/art/20121227/18114446 李蘢怡嘆雞煲扯煙]《蘋果日報》2012年12月27日</ref>，加拿大多倫多中學畢業<ref>{{cite web |url=http://data.yule.sohu.com/star/data/3/3469/ |title=存档副本 |accessdate=2010-05-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20130601190440/http://data.yule.sohu.com/star/data/3/3469/ |archivedate=2013-06-01 }} 李蘢怡 明星資料庫 搜狐</ref>，原為模特兒，後來簽約成為[[東方魅力|東方魅力]]旗下歌手，以拍攝[[電影|電影]]為主。

李宜錚早年就讀[[聖母小學|聖母小學]]和[[新法書院|新法書院]]，母親是仁愛堂前副主席鄭玉而。她的父母在她八歲之前已離婚，結果由祖母照顧她與胞弟成人，並在中學時帶他們到[[加拿大|加拿大]]繼續學業。<ref>{{cite web |url=http://www.xpweekly.com/common_emag/view_content.php?issue_no=572&pub_id=6&top_id=124&art_id=89376 |title=深情快語 Take2李蘢怡 |last= |first= |date=2009-7-27 |website= |publisher=快週刊 |access-date= 2019-3-20 |quote=}}</ref>在娛樂圈浮沉多年，李蘢怡曾投身設計師行業，現於[[倫敦|倫敦]]一所大學修讀三年期課程<ref>[http://www.weibo.com/1726453684/CfQPjEOnE 李蘢怡微博]，2015年5月1日</ref>。
