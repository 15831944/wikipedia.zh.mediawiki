'''李菁'''（英文名︰Li Ching）（{{bd|1981年|10月31日|2008年|3月1日|catIdx=L李}}），[[香港|香港]]出生，是一名已故聾人教師。她一歲半時因為發高燒，得不到適當的醫治，導致嚴重失聰。

==簡歷==
1999年的[[中學會考|中學會考]]，李菁獲得 2A3B 的成績，以交換生身份入讀美國高中學校，返港後被[[香港嶺南大學|香港嶺南大學]]社會科學系取錄，主修國際政治及經濟事務，亦曾代表香港以及美國學校贏得棋藝獎項<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/news/art/20130716/18336285 | title=狀元故事 | publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] | date= 2013-07-16 | accessdate=2013-07-16}}</ref>。

2004年8月，她畢業後，寄出逾二百封求職信，終於獲痙攣協會聘任文員。

2007年，[[香港中文大學|香港中文大學]]語言學及現代語言學系負責人看過她奮鬥向上的故事，主動邀請她當兼職教師，從事研究及協助弱聽小朋友<ref>{{cite web |language=zh-hk |url=https://topick.hket.com/article/2093241/%E6%99%82%E9%96%93%E6%B4%97%E4%B8%8D%E8%B5%B0%E5%B0%8D%E8%BC%95%E7%94%9F%E6%84%9B%E5%A5%B3%E7%9A%84%E7%89%BD%E6%8E%9B%E3%80%80%E5%9A%B4%E7%88%B6%E6%88%B0%E5%8B%9D%E6%8A%91%E9%AC%B1%E5%8A%A9%E4%BA%BA%EF%BC%9A%E8%AE%93%E5%A5%B3%E5%85%92%E3%80%8C%E6%B4%BB%E3%80%8D%E4%B8%8B%E5%8E%BB | title=時間洗不走對輕生愛女的牽掛　嚴父戰勝抑鬱助人：讓女兒「活」下去 | publisher=[[經濟日報|經濟日報]] | date= 2018-06-13 | accessdate=2018-06-13}}</ref>。

2007年10月，她和父親李光興透過[[宣明會|宣明會]]，開始助養內地一名貧困兒童<ref>{{cite web |language=zh-hk |url=http://the-sun.on.cc/channels/news/20080303/20080303022411_0000.html | title=失聰女教師自殺 早露端倪 「媽，可否畀我攬住你」 | publisher=[[太陽報_(香港)|太陽報 (香港)]] | date= 2008-03-03 | accessdate=2012-04-14}}</ref>。

2008年3月，她因長久在社會找不到工作和自己的價值，選擇輕生<ref>{{Cite news|language=zh-hk |url=http://paper.wenweipo.com/2008/03/02/YO0803020012.htm|title=失聰自強女教師 不堪壓力跳樓亡|publisher=[[大公報|大公報]]|date=2008-03-02|accessdate=2012-04-14}}</ref>。

李菁遭受歧視可在其《龍耳》的文章中看到<ref>{{cite web |language=zh-hk |url=https://hk.news.appledaily.com/local/daily/article/20080303/10819564 | title=女師自殺父延續愛心 擬成立基金協助失聰人士 | publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] | date= 2008-03-03 | accessdate=2012-04-14}}</ref>︰

==傳媒報導==
2002年7月，李菁就讀香港嶺南大學二年級時，得到獎學金贊助購買[[助聽器|助聽器]]，但有一次她跑步後遺失，向福利機構借一部助聽器，但戴那部助聽器時會覺得辛苦和頭暈。由於助聽器售價昂貴，無力支付一筆購買助聽器的費用，她的父親李光興盼獲善長慷慨捐助購買，應付她的學習需要<ref>{{cite web |language=zh-hk |url=https://hk.news.appledaily.com/local/daily/article/20020716/2714963 | title=弱聽大學生無力買助聽器 | publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] | date= 2002-07-16 | accessdate=2012-04-14}}</ref><ref>{{cite web |language=zh-hk |url=https://hk.news.appledaily.com/local/daily/article/20020807/2757896 | title=獲捐款購耳機弱聽大學生感謝善長 | publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] | date= 2002-08-07 | accessdate=2018-06-27}}</ref>。

2008年，李菁離世後，父親李光興首次瀏覽女兒生前的網誌，看到她的文章後，才知道她一直努力完成自己的期盼，但在社會中飽受歧視與挫折，令最終走上輕生的不歸路。為傳遞不盡的思念，排解難言的哀傷，李光興學習使用電腦，執筆延續女兒遺下的網誌，每天向她送上一封家書<ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2009/06/21/HK0906210001.htm | title=女亡驚覺少年苦 「眾人爸爸」網灑愛 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date= 2009-06-21|accessdate=2012-04-14}}</ref><ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2009/06/21/HK0906210002.htm | title=李光興給愛女的家書 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date= 2009-06-21|accessdate=2012-04-14}}</ref>。

2013年7月，失明及弱聽的[[香港中學文憑考試|香港中學文憑考試]]考生[[曾芷君|曾芷君]]，取得3科5**的成績<ref>{{cite web |language=zh-hk |url=http://news.stheadline.com/dailynews/content_hk/2013/07/17/246250.asp |title=唇讀女無懼挑戰 活出意義更勝物質 |publisher=[[頭條日報|頭條日報]] |date=2013-07-17 |accessdate=2013-07-17 |deadurl=bot: unknown |archiveurl=https://archive.today/20160218151131/http://news.stheadline.com/dailynews/content_hk/2013/07/17/246250.asp |archivedate=2016年2月18日 |df= }}</ref>，令社會關注殘障人士就業難的問題<ref>{{cite web |language=zh-hk |url=https://paper.hket.com/article/118187/99%E5%B9%B4%E8%81%BE%E4%BA%BA%E7%8B%80%E5%85%83%20%E5%A7%8A%E6%8F%AD%E8%87%AA%E6%AE%BA%E6%82%B2%E6%AD%8C | title=99年聾人狀元 姊揭自殺悲歌 寄逾百信無人請 姊籲僱主包容殘障者 |publisher=[[經濟日報|經濟日報]] | date=2013-07-17|accessdate=2013-07-17}}</ref>。李菁胞姐[[李鸝|李鸝]]表示殘障人士即使學歷高亦難找工作，顯示社會仍然歧視他們<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/news/art/20130717/18337100 | title=「龍耳」創辦人：社會歧視殘障生 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date= 2013-07-17|accessdate=2013-07-17}}</ref>。她亦表示有不少老闆開始願意聘請聾人及弱聽人士<ref>{{cite web |language=zh-hk |url=https://paper.hket.com/article/118425/%E8%81%BD%E9%9A%9C%E7%8B%80%E5%85%83%20%E6%BF%80%E8%B5%B7%E6%B8%AF%E4%BA%BA%E5%96%84%E5%BF%83 | title=聽障狀元 激起港人善心 |publisher=[[經濟日報|經濟日報]] | date=2013-07-18|accessdate=2013-07-20}}</ref>，但仍要繼續努力，爭取社會改變<ref>{{Cite news | language=zh-hk | url=http://hk.apple.nextmedia.com/news/art/20130716/18336285 | title=狀元故事 | publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] | date=2013-07-16 | accessdate=2013-07-16}}</ref>。

==成立慈善基金==
李菁離世後，有兩間大學分別為她設立慈善基金，去幫助傷殘人士，達致「傷健共融」的理念。
===李菁紀念基金===
{{main|李菁紀念基金}}
2008年3月，香港嶺南大學為了喚起社會對聾人及弱聽人士的關注，成立「[[李菁紀念基金|李菁紀念基金]]」<ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2008/09/10/ED0809100007.htm | title=嶺南大學 成立「李菁紀念基金」 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date= 2008-09-10|accessdate=2012-04-14}}</ref>，並決定將李菁生前的生活照及文章，結集成《[[凝望—君看雙眼色，不語似無愁|凝望—君看雙眼色，不語似無愁]]》一書<ref>{{cite web |language=zh-hk |url=http://news.sina.com/oth/mingpao/301-101-101-105/2008-09-04/16003274518.html | title=嶺大為自殺聽障教師出文集 |publisher=[[明報|明報]] |date= 2008-09-04|accessdate=2012-04-14}}</ref>。

===李菁聾健共融基金===
{{main|李菁聾健共融基金}}
2008年3月，李菁曾任職研究助理的[[香港中文大學手語及聾人研究中心|香港中文大學手語及聾人研究中心]]成立「[[李菁聾健共融基金|李菁聾健共融基金]]」，延續她樂於助人美意<ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2008/03/07/ED0803070005.htm | title=中大成立「李菁聾健共融基金」 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date= 2008-03-07|accessdate=2012-04-14}}</ref>。

==成立慈善機構==
2008年7月，她的胞姐[[李鸝|李鸝]]與[[邵日贊|邵日贊]]成立慈善機構「[[龍耳|龍耳]]」<ref>{{Cite news | language=zh-hk | url=http://hk.apple.nextmedia.com/supplement/culture/art/20130629/18315757 | title=綠是彩色：聽不到的需要 | publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] | date=2013-06-29 | accessdate=2013-06-29}}</ref><ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2012/06/04/FI1206040027.htm | title=小資料：龍耳成立宗旨 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date= 2012-06-04|accessdate=2012-06-06}}</ref>，協助聾人及弱聽人士融入社會。李菁的父親李光興是名譽顧問之一。

2012年4月，龍耳創辦人李鸝和邵日贊接受多個電視節目的訪問，講述成立龍耳的原因<ref>{{Cite news|language=zh-hk |url=https://www.youtube.com/watch?v=CZ3Ey4_nj88 | title=星期二檔案 他們都是高學歷 14 - 01 - 2014  | publisher=[[無綫新聞|無綫新聞]]|date=2014-01-14|accessdate=2012-04-16}}</ref>。

==《細聽我心聲》書刊==
{{main|細聽我心聲}}
由於李菁事件發生後，社會上越來越多人開始關注聾人及弱聽人士。有見及此，李菁聾健共融基金於2010年贊助《[[細聽我心聲|細聽我心聲]]》的徵文比賽。徵文比賽是由慈善機構「[[龍耳|龍耳]]」主辦，得到《[[国际狮子会#香港青年獅子會|國際獅子總會中國港澳 303 區青年獅子區會]]》的支持。比賽的目的是讓聾人及弱聽學生及在職人士藉著自身的經歷，表達聾人及弱聽人士的心聲，並藉此推廣[[融合教育|融合教育]]的理念。《細聽我心聲》為香港第一本由聾人及弱聽人士寫下他們過往求學及家長陪伴聾童的心路歷程之書刊<ref>{{cite web |language=zh-hk |url=http://www.silence.org.hk/file/listen.pdf | title=《細聽我心聲》徵文比賽得獎作品 |publisher=[[龍耳|龍耳]] |date= |accessdate=2012-04-14}}</ref>，一眾得獎聾童及家長期望各界可透過他們的作品多了解聾人。

自從《細聽我心聲》的書刊推出後，引起社會上很大的迴響，此書刊獲[[教育局_(香港)|教育局]][[常任秘書長|常任秘書長]]委任的一個委員會 － 《[[家庭與學校合作事宜委員會|家庭與學校合作事宜委員會]]》上載其網頁<ref>{{cite web|language=zh-hk|url=http://chsc.hk/section4_3_4.php?lang_id=2|title=網上家長錦囊|publisher=[[家庭與學校合作事宜委員會|家庭與學校合作事宜委員會]]|date=|accessdate=2012-04-14|deadurl=yes|archiveurl=https://web.archive.org/web/20111130214800/http://chsc.hk/section4_3_4.php?lang_id=2|archivedate=2011-11-30}}</ref>。

由於反應熱烈，「國際獅子總會中國港澳 303 區青年獅子區會」和「龍耳」於2011年再度合作，再次舉辦《聲空奇遇》徵文比賽<ref>{{cite web |language=zh-hk |url=http://www.leod303.org/projects/-sheng-kong-qi-yu-zheng-wen-bi-sai-1 |title=《聲空奇遇》徵文比賽得獎作品 |publisher=[[国际狮子会|國際獅子總會中國港澳 303 區青年獅子區會]] |date= |accessdate=2012-04-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20141129060705/http://www.leod303.org/projects/-sheng-kong-qi-yu-zheng-wen-bi-sai-1 |archivedate=2014-11-29 }}</ref>，希望藉著一系列的活動宣揚共融訊息，透過文字讓社會大眾了解聾人及弱聽人士的心聲，鼓勵社會大眾以積極正面的文字，勉勵聾人及弱聽人士活出積極及精彩的人生，為社會大眾與聾人及弱聽人士創造彼此溝通和了解的機會。

==【用心聽 聾健共融】通識教材套==
在2013年，龍耳製作【用心聽 聾健共融】通識教材套，希望透過結合聾人及弱聽人士生活經驗、社會機構的服務經驗及社會諸事百態以梳理出聾人及弱聽社群生活上所面對的種種問題，並透過資料的整理，讓學生能夠從中了解社群的特徵及生活文化，從而培養出公民視野，推動社會發展<ref>{{cite web |language=zh-hk |url=http://www.silence.org.hk/?p=2083 | title=【用心聽 聾健共融】通識教材套 歡迎下載 |publisher=龍耳 |date=2013-09-04 |accessdate=2013-09-04}}</ref>。

==出版《假如女兒沒有跳下去》書籍==
{{main|假如女兒沒有跳下去}}
