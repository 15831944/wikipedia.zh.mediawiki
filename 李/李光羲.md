'''李光羲'''{{BD|1929年||||catIdx=L李}}<ref>有媒体误作李光曦</ref>，[[天津市|天津市]]人，[[中华人民共和国|中华人民共和国]]男高音歌唱家。

==生平==
李光羲1929年生于天津一个普通职员家庭。17岁时，因父亲去世，李光羲从中学辍学接替父亲工作，成为天津[[开滦矿务局|开滦矿务局]]职员，养家糊口。在开滦矿务局工作期间，他利用业余时间参加演出，几年内几乎唱遍天津每个剧场，成为小有名气的业余歌手。1953年[[中华人民共和国国庆节|中华人民共和国国庆节]]前，他所在的合唱队在作为中苏友好协会天津分会办公地点的[[天津庆王府|天津庆王府]]内排练，在此他首次见到即将同台演出的王紫薇，后来二人结婚<ref name=wy>{{Cite web |url = http://news.163.com/14/0108/08/9I298NGJ00014AED.html |title = 相聚时刻永难忘 杯中盛满幸福酒 |author =  |date =2014-01-08  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

1956年，李光羲因主演中国第一部[[欧洲|欧洲]]古典歌剧《[[茶花女_(歌劇)|茶花女]]》而成名。1964年，在大型音乐舞蹈史诗《[[东方红_(音乐剧)|东方红]]》中演唱《松花江上》。[[文革|文革]]结束后的1970年代末，他演唱了《[[周恩来|周总理]]，您在哪里》、《祝酒歌》<ref name=wy/>。[[施光南|施光南]]的经典歌曲《祝酒歌》是在[[粉碎四人帮|粉碎“四人帮”]]后不久写成，并分别给[[罗天婵|罗天婵]]和[[中央歌剧院|中央歌剧院]]女中音歌唱家[[苏凤娟|苏凤娟]]试唱，李光羲无意中在同事苏凤娟处见到此歌，很快把它唱红，成为李光曦的保留曲目<ref name=xinmin>{{Cite web |url = http://xmwb.xinmin.cn/html/2013-01/20/content_19_7.htm |title =  博采众长成一格|author =  |date =2013-01-20  |publisher =新民晚报  |language =  |accessdate =  |quote =  }}</ref>。他还先后演唱了《鼓浪屿之波》、《北京颂歌》等经典歌曲。他历任第六至九届[[全国政协|全国政协]]委员，[[国家教委|国家教委]]美音教育促进会理事、[[中华人民共和国文化部|文化部]]技术职称考评委员剧院艺委会主任等职<ref name=wy/>。曾获文化部演出评比一等奖，首届金唱片奖，建国四十年优秀歌曲首唱奖，改革十年优秀演唱奖等。

== 作品 ==
李光羲演唱的代表曲目有：
*古典歌剧：《茶花女》、《叶甫根尼·奥涅金》、《货郎与小姐》
*民族歌剧：《[[刘胡兰_(歌剧)|刘胡兰]]》、《阿依古丽》
*歌曲：《祝酒歌》、《何日再相会》、《太阳出来喜洋洋》、《北京颂歌》、《牧马之歌》、《延安颂》、《红日照在草原上》、《周总理，您在哪里》、《鼓浪屿之波》、《远航》、《巍巍钟山迎朝阳》、《松花江上》
