'''李韫慧'''{{BD|1943年||||catIdx=L李}}出生于[[辽宁省|辽宁省]][[鞍山市|鞍山市]][[铁东区|铁东区]]，籍贯[[山东|山东]]，毕业于[[辽宁人民艺术剧院|辽宁人民艺术剧院]]，为知名配音演员，曾为《聪明的一休》中的一休配音。<ref name=" fish" /><ref name="man " /><ref>{{cite web |url=http://ent.163.com/12/0627/08/850A5AII00032DGD.html |title=配音演员李韫慧：为一休“代言”改变我一生 |accessdate= 2012-06-27|publisher=网易 |archiveurl=http://ent.163.com/12/0627/08/850A5AII00032DGD.html |archivedate= 2012年12月21日|work= |language=zh-cn}}</ref>

==生平==
1943年，李韫慧出生在[[辽宁省|辽宁省]][[鞍山市|鞍山市]][[铁东区|铁东区]]，他们家一共7个孩子，父亲是热爱京剧的高级技工。<ref name=" fish" >{{cite web |url= http://zh.cnr.cn/csyx/zhrw/200906/t20090601_505351468.html|title=李韫慧："一休哥"成阿婆 珠海发余热 |accessdate=  2009-06-01|publisher=中国广播网 |archiveurl= http://zh.cnr.cn/csyx/zhrw/200906/t20090601_505351468.html|archivedate=2012年12月21日 |work= |language=zh-cn}}</ref>李韫慧早年曾先后在鞍山市实验小学、鞍山市第二初级中学和鞍山市第一高级中学读书，1960年考入[[辽宁人民艺术剧院|辽宁人民艺术剧院]]，学习歌舞剧。1983年，李韫慧为《聪明的一休》中的一休配音。<ref name="man " >{{cite web |url=http://phtv.ifeng.com/program/lyyy/detail_2010_10/30/2950936_0.shtml |title= “一休”李韫慧：宋庆龄的精神激励我热爱事业 |accessdate=2010年10月30日 |publisher=凤凰网 |archiveurl= http://phtv.ifeng.com/program/lyyy/detail_2010_10/30/2950936_0.shtml|archivedate=2012年12月21日 |work= |language=zh-cn}}</ref>李韫慧加入了中国戏剧家协会、中国电视艺术家协会。她曾和赵本山、王刚和黄宏等同台合作演出。<ref name=" fish" />李韫慧现居广东省珠海市。<ref name=" fish" />
