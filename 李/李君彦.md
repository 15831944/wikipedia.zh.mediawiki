'''李君彦'''（{{bd|1913年||1999年|9月8日}}）[[河北|河北]][[定兴县|定兴县]]人，[[中国人民解放军开国少将|中国人民解放军开国少将]]，[[中国人民解放军海军|中国人民解放军海军]]顾问<ref name=rmw>{{Cite web |url =http://dangshi.people.com.cn/GB/165617/165618/166491/167909/12278183.html  |title =  李君彦|author =  |date =  |publisher =人民网  |language =  |accessdate = 2017-08-31 |quote =  }}</ref><ref name=shishi/>。

==生平==
1932年，加入[[中国共产主义青年团|中国共产主义青年团]]，1933年转入[[中国共产党|中国共产党]]。历任共青团北平市委宣传干事、河北省委宣传干事、江苏省委组织干事，共青团中央经济斗争部秘书、部长，共青团河北省委巡视员、北平市委书记、天津市委组织部长等职。1937年[[七七事变|七七事变]]后，他来到[[延安|延安]]，历任[[中国人民抗日军政大学|中国人民抗日军政大学]]政治教员，第二分校训练部政治教育科副科长、校部教育科科长兼[[晋察冀军区|晋察冀军区]]训练部编译科科长，晋察冀军区政治部宣传部教育科科长、宣传部长。[[第二次国共内战|第二次国共内战]]时期，历任[[张家口|张家口]]军事管制局局长、旅政治部主任、兵团政治部宣传部部长等职，随部队先后参加了[[察南战役|察南战役]]、[[绥东战役|绥东战役]]、[[应县战役|应县战役]]、[[保北战役|保北战役]]、[[平津战役|平津战役]]、[[太原战役|太原战役]]、[[兰州战役|兰州战役]]、[[宁夏战役|宁夏战役]]等战役<ref name=rmw/><ref name=shishi/>。

[[中华人民共和国|中华人民共和国]]成立后，他调任[[中国人民解放军海军政治工作部|中国人民解放军海军政治部]]宣传部部长、海军政治部副主任，海军政治部顾问、副主任，海军副政委兼政治部主任，海军顾问。1961年晋升少将军衔。他是[[中共十二大|中共十二大]]代表、[[中国共产党中央纪律检查委员会|中国共产党第十二届中央纪律检查委员会]]委员<ref name=rmw/><ref name=shishi/>。

1999年9月8日，李君彦在北京病逝，享年86岁<ref name=rmw/><ref name=shishi>李君彦同志逝世，人民日报1999年10月22日第四版</ref>。
