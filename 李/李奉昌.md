'''李奉昌'''（{{lang-ja|李奉昌}}，{{lang-ko|이봉창}}，{{bd|1900年|8月10日|1932年|10月10日|catIdx=L}}），生於[[漢城府|漢城府]]，大韓民國獨立運動支持者，1932年1月8日，爭取大韓民國獨立的組織[[韓人愛國團|韓人愛國團]]派李奉昌到日本[[東京|東京]]袭击正在[[閱兵|閱兵]]的[[昭和天皇|昭和天皇]]（是為[[櫻田門事件|櫻田門事件]]），最終[[手榴彈|手榴彈]]只傷及一名[[近衛師團|近衛兵]]，同年因[[大逆罪|大逆罪]]被判死刑。<ref name="森川哲郎1976">{{cite book|author=森川哲郎|title=朝鮮独立運動暗殺史|url=https://books.google.com/books?id=7sYfAAAAIAAJ|year=1976年|publisher=三一書房|pages=246-251,256-257|language=ja}}</ref><ref>{{Cite web |author= |date=1932-10-01|url=http://www.lib.kobe-u.ac.jp/das/jsp/ja/ContentViewM.jsp?METAID=10065459&TYPE=HTML_FILE&POS=1|title=桜田門事件犯人に死刑宣告・他人の煽動により大逆の愚を敢てす|trans_title=櫻田門事件犯人被判處死刑・煽動他人，我敢於反對愚蠢|work=大阪朝日新聞|publisher=神戸大学附属図書館デジタルアーカイブ |language=ja|accessdate=2018-10-01}}</ref>

== 死後 ==
李奉昌於1962年獲[[第二共和國_(大韓民國)|大韓民國第二共和國]]追授[[建國勳章|建國勳章]]，並於1992年發行紀念郵票。
