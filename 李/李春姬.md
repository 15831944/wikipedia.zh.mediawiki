'''李春姬'''（{{lang-ko|'''리춘희'''}}，{{bd|1943年|7月8日|catIdx=Ri, Chun-hee}}）[[朝鮮中央電視台|朝鮮中央電視台]][[新聞主播|新聞主播]]，也是[[朝鲜|朝鲜]]国宝级主播<ref>{{YouTube|4hCEsC8RVzA|親睹李春姬風采 民視獨家專訪}}</ref>，因其慷慨激昂的的播報語氣口吻而著名。2018年12月4日退休，結束47年的主播生涯<ref>{{Cite web |url=http://news.sina.com.cn/c/2018-12-05/doc-ihprknvt1987572.shtml |title=朝鲜75岁“咆哮式”女主播李春姬宣布退休}}</ref>。

==生平==
李春姬1943年出生於[[朝鮮日治時期|朝鮮日治時期]]的[[江原道_(北)|江原道]][[通川郡_(朝鮮)|通川郡]]的貧困勞動家庭，因為其家庭階級[[出身成分|出身成分]]關係，受到[[朝鮮民主主義人民共和國政府|朝鮮政府]]特意栽培，於[[平壤話劇電影大學|平壤話劇電影大學]]演員系畢業，自1971年2月進入朝鮮中央電視台擔任播音員，1974年起擔當主要播報員並一直受到朝鮮政府重用。

== 特点 ==
《[[朝鮮畫報|朝鮮畫報]]》介紹李春姬，說她“嗓音強勁有魄力，且號召力極強，擁有出眾的口才，每當發表聲明、講話時，能讓敵人們肝膽俱裂”等。西方媒體則以「富於情感的」（emotional）來形容李春姬的播報方式<ref>[http://tw.nextmedia.com/realtimenews/article/international/20111219/101659 消失2月 李春姬報金正日死訊]，台湾苹果日报</ref>。除了播報金日成、金正日[[訃聞|訃聞]]時著黑色[[朝鮮服|朝鮮服]]（[[韓服|韓服]]）外，李春姬大多著粉色系韓服播報新聞，因此她又有「粉紅女士」之稱<ref>[https://video.udn.com/news/748227 又見粉紅女士！北韓主播李春姬復出]</ref>。

==荣誉与特权==
李春姬曾經榮獲朝鮮播音員最高榮譽“人民播音員”稱號和“[[朝鲜民主主义人民共和国劳动英雄|劳动英雄]]”稱號。李春姬現定居[[平壤|平壤]]，拥有现代化的房舍和豪华轿车<ref>{{cite web|language =ko|url=http://news.chosun.com/site/data/html_dir/2008/04/16/2008041600088.html/2008%E5%B9%B44%E6%9C%8816%E6%97%A5%E4%BB%98|title=북성명 때마다 '전투적인 그녀' - 1등 인터넷뉴스 조선닷컴|publisher=朝鲜日报|date=2008年4月16日|accessdate=2010年6月17日}}</ref>，并可以在平壤最有名的休闲娱乐场所「[[蒼光院|蒼光院]]美容店」进行[[美容|美容]]美髮以及[[桑拿|桑拿]]。朝鲜最为新潮的服饰也都会低价提供给李春姬，并且引领着朝鲜的[[时尚|时尚]]潮流<ref>{{cite web|language =zh-hans|url=http://world.people.com.cn/GB/9516509.html|title=组图：揭密待遇丰厚的朝鲜电视主播|publisher=人民网转载新华网|date=2009年6月22日|accessdate=2010年6月17日}}</ref>。

== 私人生活 ==
《朝鲜画报》报道称“她（李春姬）的家位于首都平壤的一个美丽的地方，在那里她与丈夫和两个儿子，还有儿媳、孙女一同生活。现代式的住宅和高级轿车，都是国家送给她的礼物”<ref>[http://world.people.com.cn/GB/9516509.html 揭密待遇丰厚的朝鲜电视主播]，人民网</ref>，但并没有描述李春姬的具体住址。
