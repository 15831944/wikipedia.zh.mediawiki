'''李敖语妙天下'''是由[[新加坡|新加坡]][[盛亚卫视|盛亚卫视]]全球化媒体网络电视推出的一款文化讲坛栏目，由著名[[文学家|文学家]]、[[历史学家|历史学家]][[李敖|李敖]]主持，内容涉及古今中外的[[文化|文化]]、[[军事|军事]]、[[政治|政治]]、[[时事|时事]]等各个方面，主要讲谈[[中华文化|中华文化]]。<ref name="jyw " >{{cite web |url=http://ent.hunantv.com/z/20081227/151324.html |title= 李敖新节目开播 评饭岛爱：太俗气，不及格|accessdate=2008-12-27|work= |publisher=金鹰网 |language =zh-cn}}</ref><ref name=" shw" >{{cite web |url=http://news.sohu.com/20081226/n261431793.shtml |title=李敖称饭岛爱长相俗气不入流 嫌莫文蔚年纪大 |accessdate= 2008年12月26日|work= |publisher=搜狐网 |language =zh-cn}}</ref>

== 播出时间 ==
该栏目从2009年1月1日开始播出，在2009年6月14日结束播出，每周星期一到星期五的晚上9点播出，每一集长度为48分钟。<ref name="jyw " /><ref name=" shw" />

== 节目主持 ==
该栏目的节目主持人是著名[[文学家|文学家]]、[[历史学家|历史学家]][[李敖|李敖]]。<ref name="jyw " /><ref name=" shw" />
