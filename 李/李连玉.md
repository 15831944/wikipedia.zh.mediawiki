'''李连玉'''{{BD|1959年|12月|||catIdx=L}}[[江苏人|江苏人]]，[[中国共产党|中国共产党]]党员，毕业于[[南京理工大学|南京理工大学]]人文学院思想政治教育专业，曾担任[[徐州市|徐州市]]人民政府副市长、中共[[伊犁州|伊犁州]]委副书记。2014年9月月底，李连玉被免职，11月15日，李连玉接受[[中纪委|中纪委]]调查。2007年，李连玉当选中共十七大人大代表，是[[邳州市|邳州市]]1928年建县来第一个全国人大代表，由于此事，发生了当地万人欢迎李连玉归来的群众运动而被誉为“史上最牛市委书记”和“红地毯书记”。<ref>{{cite web |url= http://news.163.com/14/1115/20/AB4BAIVI00014SEH_2.html|title=徐州副市长李连玉落马 曾被称"史上最牛书记" |accessdate= 2014|publisher= 网易|work= |language=zh-hans}}</ref><ref>{{cite web|url=http://news.china.com/domestic/945/20141116/18968285.html|title=江苏"红毯书记"李连玉被查 数千群众曾夹道相迎|accessdate=2014|publisher=中华网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20150108225002/http://news.china.com/domestic/945/20141116/18968285.html|archivedate=2015-01-08}}</ref><ref>{{cite web |url=http://gb.cri.cn/42071/2014/11/15/2225s4767096.htm |title=江苏徐州副市长李连玉被查 曾因返乡受夹道迎接引热议 |accessdate= 2014|publisher=国际在线 |work= |language=zh-hans}}</ref>

==生平==
1959年，李连玉出生在[[江苏省|江苏省]][[丰县|丰县]]。1998年9月到2000年11月，李连玉就读于[[南京理工大学|南京理工大学]]人文学院思想政治教育专业。

1975年8月到1983年9月，李连玉担任徐州矿务局庞庄煤矿地质科文书，一共工作了8年。

1983年9月到1988年7月，李连玉调任[[丰县|丰县]]，曾担任城建局文书、秘书、团委副书记、人秘股副股长。

1988年7月到1995年5月，李连玉一直任职于中共丰县县委组织部，曾担任部务委员、组织科科长、组织部副部长、组织员办主任。

1995年5月到1997年12月，李连玉担任丰县[[大沙河镇|大沙河镇]]党委书记、大沙河果园总场主任。

1997年12月到2000年12月，李连玉陆续出任中共丰县县委常委、组织部长、丰县人民政府副县长、县政府党组成员。

2000年12月到2006年1月，李连玉在[[邳州市|邳州市]]工作，曾担任中共邳州市委副书记、邳州市人民政府代市长、邳州市人民政府党组书记，2002年12月转正中共邳州市委书记。

2006年1月到2014年9月，李连玉调任[[徐州市|徐州市]]工作，曾担任徐州市人民政府副市长、徐州市人民政府党组成员，主持了对口援疆伊犁州的建设工作。

2014年9月月底，李连玉被免职。2014年11月15日，李连玉接受[[中纪委|中纪委]]调查。<ref>{{cite web|url=http://zy.takungpao.com/2014/1107/170225.html|title=江苏“红地毯书记”李连玉落马|accessdate=2014|publisher=大公网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160303221031/http://zy.takungpao.com/2014/1107/170225.html|archivedate=2016-03-03}}</ref>
