'''李龙'''（{{kor|名=中|谚=리용}}，{{bd|1998年|1月23日||}}），[[中国朝鲜族|朝鲜族]]，出生于[[中国|中国]][[吉林省|吉林省]][[延边朝鲜族自治州|延边州]]，[[足球运动员|足球运动员]]。目前效力于[[延边富德|延边富德队]]，司职[[后卫_(足球)|后卫]]。

== 俱乐部生涯 ==
李龙于[[2017年中国足球超级联赛|2017年]]晋升至[[延边富德|延边富德队]]一线队，这象征着李龙职业足球生涯的开端<ref>{{cite web |url=http://sports.sohu.com/20170302/n482173114.shtml |title=2017中超联赛延边富德队完全名单 |publisher=[[搜狐网|搜狐新闻]] |date=2017-03-02 |accessdate=2017-07-09 |language=zh-hans |deadurl=no |archiveurl=https://web.archive.org/web/20170817145958/http://sports.sohu.com/20170302/n482173114.shtml |archivedate=2017-08-17 |df= }}</ref>。2017年4月29日，李龙在[[延边富德|延边富德队]]1:0主场战胜[[长春亚泰|长春亚泰队]]的比赛中以U23球员的身份首发登场，这是他职业足球生涯中的首秀，但在开场不到11分钟时就被换下场<ref>{{cite web |url=http://sports.sina.com.cn/china/j/2017-04-29/doc-ifyetwsm1180591.shtml |title=中超-金波小角度一剑封喉 延边1-0亚泰取7轮首胜 |publisher=[[新浪网|新浪体育]] |date=2017-04-29 |accessdate=2018-07-09 |language=zh-hans |deadurl=no |archiveurl=https://web.archive.org/web/20180709123447/http://sports.sina.com.cn/china/j/2017-04-29/doc-ifyetwsm1180591.shtml |archivedate=2018-07-09 |df= }}</ref>。

== 个人生活 ==
李龙的双胞胎兄弟李强也是一名足球运动员，目前效力于[[延边富德|延边富德预备队]]<ref>{{Cite news |url=http://www.sohu.com/a/123213500_500913 |title=新希望：延边18岁潜力新星爆发 或成下一个朴成！ |date=2017-01-02 |author=中超球评 |language=zh-hans |publisher=[[搜狐网|搜狐新闻]] |accessdate=2018-07-09 |deadurl=no |archiveurl=https://web.archive.org/web/20180709123441/http://www.sohu.com/a/123213500_500913 |archivedate=2018-07-09 |df= }}</ref>。
