'''李文科'''{{BD|1956年|7月}}，[[辽宁|辽宁]][[绥中|绥中]]人，1974年8月参加工作，1985年2月加入中国共产党，研究生学历，农学博士。

毕业于[[沈阳农学院|沈阳农学院]]（现[[沈阳农业大学|沈阳农业大学]]）研究生部栽培与耕作专业。历任辽宁省农业厅粮油处、办公室工作人员、辽宁省[[新民县|新民县]]农业局副局长、辽宁省土壤肥料总站副站长、站长、辽宁省农牧业厅粮油处处长、辽宁省[[铁岭县|铁岭县]]政府副县长兼中共铁岭新台子开发区党委书记、辽宁省农业厅副厅长、厅长、党组成员、党组书记。2000年任中共[[营口市|营口市]]委副书记、政府代市长（2001年转为正职，2003年至2004年挂职[[中国长江三峡工程开发总公司|中国长江三峡工程开发总公司]]工程建设部副主任），2004年任中共[[铁岭市|铁岭市]]委书记，2012年任辽宁省第十一届人民代表大会常务委员会副主任。他也是[[中共十七大|中共十七大]]代表以及第十届全国人大代表<ref>{{cite web |url=http://gbzl.people.com.cn/grzy.php?id=150100100 |title=中国党政领导干部资料库：李文科 |publisher=中国共产党新闻网 |date= |accessdate=2017年2月28日 |archive-url=https://web.archive.org/web/20170302025406/http://gbzl.people.com.cn/grzy.php?id=150100100 |archive-date=2017-03-02 |dead-url=yes }}</ref>。

2017年2月28日，因涉嫌严重违纪被组织调查<ref>{{cite web |url = http://www.ccdi.gov.cn/xwtt/201702/t20170228_94817.html |title = 辽宁省人大常委会副主任李文科涉嫌严重违纪接受组织审查 |accessdate = 2017年2月28日 |publisher = 中央纪委监察部网站}}</ref>，成为2017年辽宁两会结束、强调[[辽宁拉票贿选案|拉票贿选案]]影响后首位落马的副省级干部<ref>{{cite web |url = http://news.takungpao.com/mainland/topnews/2017-02/3425691.html |title = 辽宁人大副主任李文科涉嫌严重违纪 成该省第三位落马人大副主任 |accessdate = 2017年2月28日 |publisher = 大公网}}</ref>。

2017年5月10日，中央纪委监察部网站发布消息，辽宁省人大常委会原党组成员、副主任李文科严重违纪，经中共中央批准，给予开除党籍、开除公职处分，收缴其违纪所得，移交司法机关<ref>{{cite web |url = http://www.ccdi.gov.cn/xwtt/201705/t20170510_98985.html |title = 辽宁省人大常委会原党组成员、副主任李文科严重违纪被开除党籍和公职 |accessdate = 2017年5月10日 |publisher = 中央纪委监察部网站}}</ref>。

2018年9月13日，[[吉林省|吉林省]][[通化市|通化市]]中级人民法院一审宣判，以受贿罪、行贿罪两罪并罚，判处李文科有期徒刑十六年，并处罚金人民币三百万元。对李文科受贿所得财物及其孳息予以追缴，对其用于行贿犯罪的财物予以没收，上缴国库。李文科当庭表示服从判决，不上诉<ref>{{cite web |url = http://district.ce.cn/newarea/sddy/201809/13/t20180913_30296585.shtml|title = 辽宁人大常委会原副主任李文科受贿行贿获刑十六年 处罚三百万元 |accessdate = 2018-09-14 |publisher = 人民日报客户端}}</ref>。
