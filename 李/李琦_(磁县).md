'''李琦'''（{{bd|1918年|3月24日|2001年|4月17日}}）原名'''李尚杞'''，[[河北省|河北省]][[磁县|磁县]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员、政治人物<ref name=shishi>{{Cite web |url = http://www.people.com.cn/GB/paper39/3263/424337.html |title = 李琦同志逝世 |author =  |date = 2017-07-24 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref><ref name=deng>{{Cite web |url = http://cpc.people.com.cn/GB/69112/86369/87105/87211/5966333.html |title =致李琦  |author =[[邓颖超|邓颖超]]  |date =1981-07-28  |publisher = 邓颖超书信选集 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1936年3月，参加[[中华民族解放先锋队|中华民族解放先锋队]]。同年夏，加入[[中国共产主义青年团|少年中国共产党]]，11月转为[[中国共产党|中国共产党]]党员。1938年2月到1940年底，由中共党组织派到[[国民革命军|国民革命军]]新编第五军从事地下工作，任工委书记。1941年7月，任[[中共中央北方局|中共中央北方局]]宣传部编审科科长。1941年冬，调到[[八路军|八路军]]总部任[[彭德怀|彭德怀]]的秘书。1943年秋，回到[[延安|延安]]，先在[[中共中央党校|中央党校]]二部学习，后到[[中共中央书记处|中共中央书记处]]办公室担任秘书。1947年7月至1949年11月，历任中共太行区党委安阳地委书记、[[安阳市|安阳市]]军事管制委员会主任<ref name=shishi/>。

[[中华人民共和国|中华人民共和国]]成立后，1950年至1956年任[[周恩来|周恩来]]总理办公室副主任，1956年至1964年历任中共太原市委第一书记、[[中共山西省委|中共山西省委]]宣传部部长，1964年任[[中华人民共和国文化部|中华人民共和国文化部]]副部长兼[[中国人民对外友好协会|中国人民对外友好协会]]副会长。[[文化大革命|文化大革命]]中，被下放[[五七干校|五七干校]]劳动。1972年经周恩来过问回到北京。1973年夏起，历任国务院科教组负责人、[[中华人民共和国教育部|中华人民共和国教育部]]副部长和部[[党组|党组]]副书记、[[中国联合国教科文组织全国委员会|联合国教科文组织中国委员会]]主任。1979年4月，调任[[中共中央文献研究室|中共中央毛泽东主席著作编辑出版委员会办公室]]第一副主任。1980年5月，任[[中共中央文献研究室|中共中央文献研究室]]第一副主任。1982年4月，任中共中央文献研究室主任。1991年起，任中共中央文献研究室顾问。他是中共[[中共十二大|第十二]]、[[中共十三大|十三次全国代表大会]]代表，[[第五届全国政协|第五届全国政协]]常委，[[第六届全国人大常委会|第六]]、[[第七届全国人大常委会|七届全国人大常委会]]委员<ref name=shishi/>。

2001年4月17日，李琦在北京逝世，享年83岁。2001年4月26日，遗体在[[八宝山革命公墓|八宝山革命公墓]]火化<ref name=shishi/>。
