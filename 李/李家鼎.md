'''李家鼎'''（{{lang|en|'''Steve Lee Ka Ting'''}}，{{bd|1945年|11月1日|catIdx=Lee李家鼎}}），綽號「鼎爺」，出生在[[廣東省_(中華民國)|廣東省]][[廣州市_(中華民國)|廣州市]]，[[籍貫|籍貫]][[廣東|廣東]]。資深[[香港|香港]][[电视廣播有限公司|無綫電視]]（TVB）藝員（現為基本藝人合約），於[[香港|香港]][[無綫電視|無綫電視]]的處境喜劇《[[香港八一|香港八一]]》至《[[香港八六|香港八六]]》中飾演「銀禧小食店」伙計「李威」一角為人熟悉。現仍於無綫電視擔任演員，同時兼任無綫電視動作暨馬術指導，亦為資深馬術教練。李家鼎曾與前藝人[[施明|施明]]結婚，現已離婚，二人育有孿生子[[李泳漢|李泳漢]]及[[李泳豪|李泳豪]]，二子皆為藝人。

== 簡歷 ==
李家鼎生於廣州一個富裕家庭，兒時住[[西關|西關]][[泮塘|泮塘]]五約<ref>[http://programme.tvb.com/foodandtravel/theahistoricgrandpacookingshowsr30001/episode/26/#page-1 第26集 - 「蠔」氣花膠、香嫩豆胚、「雲腿冬瓜治」]，無綫電視節目《阿爺廚房》（第3輯） ，2019年1月21日首播</ref>，大宅有2,000呎。父親是一名廣州大地主，從事酒家、麵粉、醬油、絲綢生意，且是[[岭南大学_(广州)|廣州嶺南大學]]校董。外公當中醫師，姨媽則是一級婦產科醫生<ref>{{web reference|URL=http://hd.stheadline.com/ent/ent_content.asp?contid=44215&srctype=g|title=我有我的奧林匹克 李家鼎|work=頭條日報|year=2008年|date=11月20日}}</ref>。他雖為二房所出，卻深得家人疼愛。八歲時經歷[[土改|土改]]，所有家當被充公。李家鼎與家人逃難至香港，搬到[[何文田|何文田]]幾十呎的沙磚屋居住，因家貧被迫於小學四年級時輟學，開始其打工生涯。直至十八歲那年，李家鼎當暑期工期間認識了[[關正良|關正良]]<ref>{{ cite web |url=http://www.hkheadline.com/ent/ent_content.asp?contid=40494&srctype=g | title=名人專訪：李家鼎 世界從來都是文長武短 |publisher=頭條新聞|date=2008年9月12日}}</ref>教頭，拜其為師並開始習武，習過[[太極拳|太極拳]]、[[詠春拳|詠春拳]]。二十歲起擔當龍虎武師。其間有數間電影公司想聘其為男主角，但李家鼎為了養家而放棄薪酬較低的演員工作，繼續任武師，至今超過四十年。

李家鼎自二十歲出頭正式拜師學習馬術，曾參加多個國際賽，當中包括世界知名的「IBG國際商務馬術賽」及「FEI國際馬聯馬術障礙賽」等大賽，並奪冠而回，一生奪得超過二百個獎項。李家鼎騎術以狠辣精湛見稱，其拼搏及不服輸的比賽精神曾令對手聞風喪膽，洋人對手們都稱他「Crazy Steve」。李家鼎本欲參加[[奧運|奧運]]，但購買及訓練比賽用馬匹所費不菲，當時的馬術運動員又不會獲得官方或私人贊助，為了家庭和生活李家鼎只好放棄參賽，但至今仍擔任馬術指導工作，著名藝人[[陳慧琳|陳慧琳]]亦曾隨他學習騎術<ref>{{cite web |url=http://hk.epochtimes.com/7/4/25/43660.htm  | title=傷玉手留6吋疤痕陳慧琳4次墮馬  |publisher=大紀元時報 |date=2007年4月25日 |accessdate=2011年6月30日 }}</ref>。

1970年，李家鼎認識藝人施明，拍拖四年結婚，於1990年代離異。二人的孿生子李泳豪（後更名「李永豪」）及李泳漢（後更名「李永漢」），自小擔當童星，曾演出多部電影及電視劇，長大後同為演員（李泳豪目前仍然活躍於[[無綫|無綫]]，但李泳漢已經轉往政界）。70年代李家鼎加入[[無綫電視|無綫電視]]，1981年於處境喜劇《香港八一》至《香港八六》中飾演「李威」一角為人熟悉。多年來一直於無綫電視擔任演員及武術暨[[馬術|馬術]]指導，同時亦擔當電影演員及[[武術指導|武術指導]]工作。因外型嚴肅威怒，多出演反派或武術高手，行內人尊稱「鼎爺」，威武形象深入民心。<ref>{{cite web |url=http://www.hkheadline.com/ent/ent_content.asp?contid=44215&srctype=g&int_split_from=0  | title=我有我的奧林匹克 李家鼎 |publisher=頭條日報 |date=2008年11月20日 |accessdate=2011年6月30日 }}</ref>

2014年，他在《[[萬千星輝頒獎典禮2014|萬千星輝頒獎典禮2014]]》獲頒發「專業演員大獎」，肯定及嘉許他在演藝圈的貢獻和付出。

2016年，他主持的《[[阿爺廚房|阿爺廚房]]》與觀眾重溫50、60年代的家常菜煮法。播出後獲得媒體及網民一致好評，稱讚他烹飪手法精準，更令人對他武打演員的印象改觀。人氣上升的同時，大家亦開始關注他的私人生活。<ref>{{Cite web|url=http://news.mingpao.com/ins/instantnews/web_tc/article/20161222/s00007/1482407731045|title=【專訪】譚玉瑛爆李家鼎拍《阿爺廚房》壓力大到踎地|accessdate=2017-01-04|author=|date=|work=即時新聞 instant news|publisher=}}</ref><ref>{{Cite web|url=http://paper.wenweipo.com/2017/01/06/OT1701060007.htm|title=【演藝蝶影】《阿爺廚房》的意義|accessdate=2017-01-08|author=小蝶|date=|work=副刊|publisher=}}</ref><ref>{{cite web |url=http://hk.crntt.com/doc/1045/2/4/3/104524367.html?coluid=209&kindid=1290&docid=104524367&mdate=1229101552| title=71歲香港功夫明星李家鼎要再婚？|publisher=中評社 |date=2016年12月26日 |accessdate=2017年1月8日 }}</ref>

2017年7月，他挾著人氣在[[香港書展|香港書展]]期間推出首本個人食譜，更成為有關出版社當時最暢銷的書籍。<ref>{{cite web |url=http://paper.wenweipo.com/2017/07/25/HS1707250013.htm| title=《鼎爺廚房》最火 草日憑貓奪季|publisher=文匯報 |date=2017年7月25日 |accessdate=2017年8月1日 }}</ref>他亦與烘焙工房合作推出奶黃月餅。<ref>{{cite web |url=http://orientaldaily.on.cc/cnt/entertainment/20170727/00282_013.html| title=李家鼎出自家製月餅|publisher=東方日報|date=2017年7月27日 |accessdate=2017年8月1日 }}</ref> 9月，他在[[銅鑼灣|銅鑼灣]][[登龍街|登龍街]]開設私房菜餐廳。<ref>{{cite web |url=https://www.hk01.com/01%E6%95%99%E7%85%AE/108841/-%E9%98%BF%E7%88%BA%E5%BB%9A%E6%88%BF-%E9%BC%8E%E7%88%BA%E7%A7%81%E6%88%BF%E8%8F%9C9%E6%9C%88%E9%8A%85%E9%91%BC%E7%81%A3%E6%8F%92%E6%97%97-%E7%9B%B4%E6%93%8A%E9%98%BF%E7%88%BA%E5%88%80%E6%B3%95%E8%A3%BD%E7%87%88%E5%BD%B1%E5%8F%89%E7%87%92| title=鼎爺私房菜9月銅鑼灣插旗　直擊阿爺刀法製燈影叉燒|publisher=香港01|date=2017年8月1日 |accessdate=2017年8月1日 }}</ref>

2017年11月，李家鼎接受訪問時表示食肆正計劃開分店，非常忙碌與疲倦。亦透露所主持的烹飪節目《阿爺廚房》第三輯亦即將開拍，今次將會移師深山找食材挑戰農家菜和滿漢，百忙中的他亦不敵兩位美女訴求攻勢，收了羅霖和節目《我係小廚神》的導師兼評判朱庭萱為徒，親授烹飪技術<ref>[http://bka.mpweekly.com/interview/%E5%A8%9B%E6%A8%82123/20171125-91906 【獨家專訪】李家鼎收羅霖、朱庭萱為徒 食肆開分店不怕食客挑機]</ref>。
