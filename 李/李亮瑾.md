'''李亮瑾''' ( 英文名：'''Sunny Li'''，{{bd|1981年|6月25日}}），[[台灣|台灣]][[女藝人|女藝人]]<ref>{{cite news |language = zh-tw |url = http://www.setn.com/News.aspx?NewsID=112906/ |title = 從方思瑤到趙麗娟：李亮瑾演藝事業大豐收　戲裡戲外都拚命|publisher = 范庭瑄  |date = 2015-12-18 |accessdate = 2016-01-24}}</ref>。
畢業於[[高雄女中|高雄女中]]、[[國立台灣藝術大學|國立台灣藝術大學]]，因出演台劇而知名，2014年獲得新浪微博舉辦的「最受歡迎的台灣戲劇女演員」票選第一名，並與[[陳珮騏|陳珮騏]]共同獲得「最受歡迎台灣戲劇螢幕情侶」第一名<ref>{{cite news |language = zh-tw |url = http://www.appledaily.com.tw/realtimenews/article/new/20140915/470074/ |title = 《世間情》女女戀超人氣　奪強國螢幕情侶|publisher = 蘋果娛樂新聞  |date = 2014-09-15 |accessdate = 2016-01-09}}</ref>。

作品有《[[真情满天下|真情满天下]]》、《[[天下父母心|天下父母心]]》、《[[家和萬事興_(2010年電視劇)|家和萬事興]]》、《[[牽手_(2011年電視劇)|牽手]]》、《[[天下女人心|天下女人心]]》、《[[白蛇外傳|白蛇外傳]]》、《[[世間情|世間情]]》、《[[珍珠人生|珍珠人生]]》、《[[甘味人生|甘味人生]]》、《[[一家人|一家人]]》。

== 個人生活 ==
高雄人，是家中長女，17歲時父親過世<ref>{{cite news |language = zh-tw |url = http://star.ettoday.net/news/628429 |title = 李亮瑾17歲喪父「爸我過得很好」　新年願望全許給別人|publisher = ETtoday星光雲  |date = 2016-02-11 |accessdate = 2016-02-11}}</ref>。不畏困難從高雄北上半工半讀，畢業於台藝大，一開始出道當過六、七年臨時演員，曾一天只靠一顆饅頭夾蛋果腹，2014年拿出積蓄來幫生意失敗的弟弟還債，靠的一句：「我不能讓死去的爸爸丟臉」而努力撐下去<ref>{{cite news |language = zh-tw |url = http://star.ettoday.net/news/520487 |title = 《世間情》李亮瑾17歲喪父　拼命扛家計：不讓他丟臉 |publisher = ETtoday星光雲  |date = 2015-06-13 |accessdate = 2016-01-24}}</ref>。
