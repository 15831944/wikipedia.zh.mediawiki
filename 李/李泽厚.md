'''李泽厚'''（{{bd|1930年|6月13日|catIdx=L李}}）<ref>{{Cite web |url=http://m.sohu.com/a/148614689_222496/?pvid=000115_3w_a |title=《贺李泽厚先生八十七岁华诞》“六月十三日，李泽厚先生的生日。生于一九三〇年的这位湖南人，走进了八十七岁大寿。” |access-date=2019-02-07 |archive-url=https://web.archive.org/web/20171013013039/http://m.sohu.com/a/148614689_222496/?pvid=000115_3w_a |archive-date=2017-10-13 |dead-url=no }}</ref>，中国[[湖南|湖南]][[長沙|長沙]][[寧鄉|寧鄉]]人。[[哲学|哲学]]家、[[美学|美学]]家、[[中国思想史|中国思想史]]学家。曾担任[[中国社会科学院|中国社会科学院]]研究员、[[华东师范大学|华东师范大学]][[思勉人文高等研究院|思勉人文高等研究院]]、[[德国|德国]][[图宾根大学|图宾根大学]]、美国[[威斯康星大学|威斯康星大学]]、[[密歇根大学|密歇根大学]]、[[科罗拉多学院|科罗拉多学院]]、[[斯沃斯莫尔学院|斯沃斯莫尔学院]]客席教授、客席讲座教授，台北[[中央研究院|中央研究院]]客席讲座研究等职。曾當選為[[第七届全国人民代表大会|中華人民共和國第七屆全國人民代表]]，是第七屆全國人大教育科學文化衛生委員會的委員<ref>{{Cite web |url=http://www.people.com.cn/zgrdxw/zlk/rd/7jie/newfiles/a1080.html |title=七届全国人大专门委员会组成人员名单（１５６人） |access-date=2019-02-07 |archive-url=https://web.archive.org/web/20120511041249/http://www.people.com.cn/zgrdxw/zlk/rd/7jie/newfiles/a1080.html |archive-date=2012-05-11 |dead-url=no }}</ref>。

他在1988年当选{{le|巴黎国际哲学院|Collège international de philosophie}}院士，1998年获美国科罗拉多学院荣誉人文学博士学位<ref>{{cite web |url=http://finance.ifeng.com/opinion/zjgc/20100919/2637214.shtml |title=易中天对话李泽厚：中国往哪个方向走最危险 |publisher=[[凤凰网|凤凰网]] |date=2010-09-19 |accessdate=2012-04-01 |archive-url=https://web.archive.org/web/20111027225813/http://finance.ifeng.com/opinion/zjgc/20100919/2637214.shtml |archive-date=2011-10-27 |dead-url=no }}</ref>。

汉学家[[余英时|余英时]]曾评价他：“通过（他的）书籍，他使得一整代中国青年知识分子从[[共产主义|共产主义]]的[[意识形态|意识形态]]之中解放了出来<ref name="coloradocollege1">{{cite web | archiveurl=https://web.archive.org/web/20100530093301/http://www.coloradocollege.edu/Academics/Anniversary/Participants/Li.htm  |archivedate=20100530| url=http://www.coloradocollege.edu/Academics/Anniversary/Participants/Li.htm| deadurl=yes|author=Li Zehou| title=''The Confucian World''|website=[[科羅拉多學院|科羅拉多學院]]}}</ref>。」

==生平==
李泽厚1948年畢業於[[湖南省立第一師範|湖南省立第一師範]]。1954年毕业于[[北京大学|北京大学]]哲學系。在1955年的[[美学大讨论|美学大讨论]]中崭露头角，1950年代以重實踐、尚“人化”的“ 客觀性與社會性相統一”的[[美學觀|美學觀]]卓然成家。[[文革|文革]]以後在哲學、美學和思想史方面均有所建樹。出版了《批判哲學的批判》、《[[美的歷程|美的歷程]]》、《華夏美學》《中國古代、近代、現代思想史論》等作品。影響巨大。1989年[[六四事件|六四事件]]中，因批评[[中国政府|中国政府]]的处理方式，其作品被禁，本人受到中國政府和文人批判。1992年初獲准移居[[美國|美國]]，曾任教于美国[[科罗拉多学院|科罗拉多学院]]（[[科羅拉多學院|科羅拉多學院]]）。1999年退休，现居[[美国|美国]][[科罗拉多|科罗拉多]]。

==思想==
李泽厚宣揚[[儒家|儒家]][[主情論|主情論]]，以“[[告別革命|告別革命]]”說為轟轟烈烈的1980年代[[中国启蒙运动|啟蒙運動]]劃上了一個句號。李澤厚曾表示：“[[民族主義|民族主義]]，是很危險的事情。今天在世界任何地方都要反對民族主義。民族主義是最容易煽動民眾感情的一種主義。”<ref>{{Cite news| url=http://culture.people.com.cn/GB/3918041.html| title=专访李泽厚：中国哪方面都比西方差一大截 (4)| author=黄锫坚| newspaper=经济观察报| access-date=2019-02-07| archive-url=https://web.archive.org/web/20091008122225/http://culture.people.com.cn/GB/3918041.html| archive-date=2009-10-08| dead-url=no}}</ref>2005年12月06日但也有人認為李澤厚雖然反對民族主義，可沒有擺脫傳統“[[華夷之辨|華夷之辨]]”的思想。2006年在有关[[施琅|施琅]]问题的讨论中，他提出了[[李自成|李自成]]如果胜利了，要比少數民族建立的[[清朝|清朝]]强很多的观点。<ref>{{Cite web | url=https://www.rujiazg.com/article/965 | title=李泽厚再次就“施琅问题”答南都周刊记者问 |website=儒家网 | archiveurl=https://web.archive.org/web/20190207042634/https://www.rujiazg.com/article/965 | archivedate=2019-02-07|dead-url=no}}</ref><ref>{{Cite magazine|url=http://news.sina.com.cn/c/2006-05-09/15329811883.shtml|title=李澤厚批陳明：施琅絕對不是民族英雄|date=2006年5月9日|magazine=《[[南都周刊|南都周刊]]》|access-date=2019-02-07|archive-url=https://web.archive.org/web/20060614152504/http://news.sina.com.cn/c/2006-05-09/15329811883.shtml|archive-date=2006-06-14|dead-url=no}}</ref><ref>{{cite web| url=http://www.aisixiang.com/data/12143.html| title=李澤厚：關於施琅問題的訪談| website=愛思想| access-date=2019-02-07| archive-url=https://web.archive.org/web/20150325225426/http://www.aisixiang.com/data/12143.html| archive-date=2015-03-25| dead-url=no}}</ref>其全部哲學思想與美學體系曾受到知識界的強烈質疑（在香港中文大學、廈門大學、貴州大學和武漢一些高校屢遭當地學報列據反駁）。

==著作==
李泽厚著有《美的历程》、《美学四讲》、《华夏美学》（合称《[[美学三书|美学三书]]》）、《中国（古代、近代、现代）思想史论》、《批判哲学的批判》、《走我自己的路》、《李澤厚哲學美學文選》等。發表論文百餘篇，主要有《孔子再評價》、《漫述莊禪》、《啟蒙與救亡的雙重變奏》、《漫說西體中用》、《關於中國美學史的幾個問題》。香港[[天地图书|天地图书]]公司1995年出版《告别革命——二十世纪中国对谈录》，为李泽厚与[[刘再复|刘再复]]的对话录，由[[刘再复|刘再复]]执笔，在中国大陆与海外学界都产生巨大影响。该书于1999年由[[台北|台北]][[麦田出版公司|麦田出版公司]]编入“[[麦田人文|麦田人文]]”图书系列，[[王德威|王德威]]主编。
