== 生平 ==
1996年4月25日<ref>李冠丰父亲李双江 是山东省人 {{cite web |url=http://ent.xinmin.cn/2013/02/23/18790031_2.html |title=存档副本 |accessdate=2013-08-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20131102153556/http://ent.xinmin.cn/2013/02/23/18790031_2.html |archivedate=2013-11-02 }}</ref><ref name="love" >{{cite web |url=http://ent.163.com/13/0313/10/8PRCAJOE00031H2L.html |quote =当杜恩湖问道梦鸽李天一的真实年龄时，梦鸽一口回答道：“李天一今年17岁啊！”并透露李天一是1996年4月25日出生的。当记者提到别人的质疑李天一今年19岁时，梦鸽坚定的说：“那说的不正确。李天一确实是1996年4月25日出生。我生的孩子，出生时间会记得很准确的。” |title=李天一被曝已19岁 梦鸽确认儿子17岁尚未成年 |accessdate=2013年3月13日 |publisher=网易 |work= |language=zh-cn}}</ref>{{noteTag|1=“1996年4月25日”是李冠丰母亲梦鸽对新闻媒体曝光的李冠丰的年龄。}}<ref>网曝李双江十年前杂志旧文 证李某某犯案时已成年 {{cite web |url=http://www.hb.xinhuanet.com/2013-09/17/c_117395669.htm |title=存档副本 |accessdate=2013-11-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20131204034430/http://www.hb.xinhuanet.com/2013-09/17/c_117395669.htm |archivedate=2013-12-04 }}</ref>{{可疑}}，李冠丰出生在[[北京市|北京市]]。早年他曾陆续在北京[[海淀区|海淀区]][[中关村|中关村]]第三小学、[[中国人民大学附属中学|中国人民大学附属中学]]（未毕业）、[[美国|美国]]沙特克圣玛丽学院冰球学校读书。

4岁时，开始学习[[钢琴|钢琴]]，师从[[中央音乐学院|中央音乐学院]]钢琴教授韩剑明，5岁时，成为[[中华人民共和国|中华人民共和国]][[2008年北京奥运会|申奥大使]]。8岁时，开始学习[[书法|书法]]，师从[[清华大学|清华大学]]教授方-{志}-文，10岁时，加入[[中国少年冰球队|中国少年冰球队]]。<ref name="bull" >{{cite web |url= http://ent.qq.com/a/20110908/000297.htm|title= 组图：李双江儿子李天一 5岁就成最小申奥大使|accessdate=2011-09-08 |publisher=腾讯网 |work= |language=zh-cn}}</ref>

2011年9月，因[[李天一打人事件|无证驾驶宝马车并打人]]，被[[收容|收容教养]]1年，2012年9月获释，之后疑改名为李冠丰<ref name="tiger" />。

2013年3月，李某某因涉嫌[[强奸罪|强奸罪]]被检察机关批准逮捕，同年9月26日被北京市海淀区法院以强奸罪判处有期徒刑10年<ref>[http://www.apdnews.com/news/35220.html 李某某一审被判有期徒刑10年] {{webarchive|url=https://web.archive.org/web/20131125032943/http://www.apdnews.com/news/35220.html |date=2013-11-25 }}，亚太日报，2013年11月28日</ref><ref>{{cite web|title=李天一强奸罪名成立 获刑10年|url=http://news.163.com/13/0926/10/99MKR99D00014JB6.html|publisher=网易新闻|accessdate=2013-09-26}}</ref>。成年后已转入河北唐山某[[劳改|劳改]]农场服刑<ref>{{cite web|url=http://henan.china.com.cn/latest/2015/0824/757129_2.shtml|title=抗战胜利70周年特赦四类服刑罪犯 李天一会被特赦吗？|accessdate=2015-08-24|publisher=环球网|work=|language=zh-cn|archivedate=2016-04-28|archiveurl=https://web.archive.org/web/20160428172312/http://henan.china.com.cn/latest/2015/0824/757129_2.shtml|deadurl=yes}}</ref>。

== 犯罪记录 ==
{{Main|李天一打人事件|李冠豐性侵案}}

多家媒体报道：2011年1月至9月，李天一驾驶改装宝马车9个月违法32次，且全部未受到法律惩罚。<ref name="tiger" >{{cite web |url=http://legal.people.com.cn/n/2013/0226/c188502-20597870-2.html |title= 李天一曾驾车违法32次未受罚 收容1年后换豪车 |accessdate=2013年2月26日  |publisher=  人民网|work= |language=zh-cn}}</ref>

2011年9月6日晚，在海淀某小区门口，一对业主夫妻在开车拐入南门时，遭一辆无照[[宝马|宝马]]和一辆[[奥迪|奥迪]]司机[[李天一打人事件|殴打]]，并大喊“谁敢打[[110|110]]”。警方称：“经核实，宝马司机年龄15岁，无[[驾照|驾照]]，系李双江之子李天一”。李双江8日下午赴医院看望伤者并含泪道歉。伤者坚决拒绝私了，李天一因无证驾驶宝马车并打人，被[[收容|收容教养]]1年，2012年9月获释，之后疑改名为李冠丰<ref name="tiger" />。

2013年2月21日晚，李冠丰被指因涉及一宗[[李冠豐性侵案|轮奸案]]，被[[北京市公安局|北京市公安局]][[海淀|海淀]]分局[[刑事拘留|刑事拘留]]。

2013年3月7日，[[央視|央視]]新闻中心官方微博证实李天一因涉嫌[[强奸罪|强奸罪]]，已被[[检察机关|检察机关]]批准[[逮捕|逮捕]]<ref>[http://sn.people.com.cn/n/2013/0307/c190215-18263453.html 李天一涉嫌强奸罪 已被检察机关批捕(图)]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}，人民网</ref>。3月8日，[[新华网|新华网]]记者从北京市检方获悉：“李双江之子李某等人因涉嫌轮奸已被批捕，李某确定为[[未成年人|未成年人]]”<ref>{{cite news|title=检方确定李天一为未成年人|url=http://ent.163.com/13/0308/10/8PEI5CFA00031H2L.html|accessdate=2013-03-08|date=2013-03-08|newspaper=新华网}}</ref><ref>{{cite web|title=释疑:李天一为何以强奸罪而非轮奸批捕|url=http://ent.sina.com.cn/s/m/w/2013-03-08/00383872256.shtml|publisher=新浪娱乐|accessdate=2013-03-12}}</ref>。

=== 年龄 ===
刑法专家张蕴章认为，假若李冠丰年龄是17岁，他将受到最高刑罚处罚是10年有期徒刑；假若李冠丰是成年人，最低的刑法处罚是10年有期徒刑，情节严重可以判处无期徒刑或者死刑。<ref name="bee" >{{cite web |url=http://www.bbc.co.uk/zhongwen/simp/china/2013/03/130312_li_true_age.shtml |title=港媒质疑李双江之子李冠丰真实年龄 |accessdate=2013年3月12日 |publisher=BBC中文网 |work= |language=zh-cn}}</ref>

对于李冠丰的年龄，有网友在微博上提出质疑，因为2006年李双江、梦鸽、李冠丰参加[[鲁豫|鲁豫]]《[[鲁豫有约|鲁豫有约]]》时说当时是14岁，2013年必然已经19岁。不过也有网友指出该期[[鲁豫|鲁豫]]《[[鲁豫有约|鲁豫有约]]》的播出时间是2011年，因此质疑并不成立。<ref>{{cite web |url=http://news.163.com/13/0224/11/8OFMPHTC00014JB5.html |title= 网友爆料李冠丰年龄造假待求证|accessdate=2013-02-23 |publisher=网易|work= |language=zh-cn}}</ref><ref>{{cite web |url=http://ent.ifeng.com/idolnews/special/lgf/content-6/detail_2013_02/24/22432926_0.shtml |title= 网友根据电视节目推测李冠丰年龄造假|accessdate=2013年2月24日 |publisher=凤凰网 |work= |language=zh-cn}}</ref><ref>{{cite web |url= http://ent.qq.com/a/20130224/000122.htm|title= 网友根据电视节目推测李冠丰年龄造假|accessdate=2013年2月24日 |publisher=腾讯网 |work= |language=zh-cn}}</ref>{{R|love}}{{R|donglu}}

同样，在中国大陆发行的杂志《妇女生活（现代家长）》在2002年第07期第1页，有一篇署名为李双江的作者写的一篇文章，标题为《家教的关键是塑造孩子的性格》。文章中提到“我儿子叫李天一，小家伙圆圆的脑袋，胖乎乎的脸蛋，乌溜溜的黑眼珠，长得虎头虎脑的，很可爱。他聪明伶俐，今年才8岁，正上小学三年级，能背200多首唐诗，唱几十首歌曲，画也画得不错，被大家称为‘小天才’。”
