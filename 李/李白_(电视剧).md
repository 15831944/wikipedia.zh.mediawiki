《'''李白'''》，中国大陆[[唐朝|唐朝]]古装剧，导演邵警辉，主演陈建斌、曾黎、刘德凯、伊能静。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2010-03-01/04162884099.shtml |title=电视剧《李白》杀青 陈建斌展现诗仙一生(图) |accessdate= 2010|publisher=新浪网 |work= |language=zh-hans}}</ref>

==剧情==
{{See also|李白}}

该剧讲述了唐朝[[浪漫主义|浪漫主义]]诗人李白的人生旅程。

==制作==
该剧共投资3000万，剧组花费巨资搭建了[[华清池|华清池]]。<ref>{{cite web |url= http://ent.qq.com/a/20090515/000163.htm|title= 电视剧《李白》8月开机 陈建斌：不会打打杀杀|accessdate=2009 |publisher=腾讯网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.qq.com/a/20091030/000264_1.htm |title= 《李白》再爆精美剧照 华清池待伊能静大驾光临|accessdate= 2009|publisher=腾讯网 |work= |language=zh-hans}}</ref>

前中共中央政治局常委[[李长春|李长春]]曾就该剧说道：“目前，系统拍摄《李白》的电视剧在国内还是空白，尽管是拍一部电视剧，但要真实还原史实，拍成一部记录李白生平的权威电视剧。”<ref>{{cite web |url=http://culture.ycwb.com/2010-01/26/content_2412851.htm |title=三十集电视剧《李白》杀青 |accessdate=2010 |publisher=金羊网 |work= |language=zh-hans |archive-url=https://web.archive.org/web/20160306192011/http://culture.ycwb.com/2010-01/26/content_2412851.htm |archive-date=2016-03-06 |dead-url=yes }}</ref>
