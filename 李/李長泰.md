'''李长泰'''（{{bd|1862年||1922年|}}）[[字|字]]'''階平'''，[[直隷省|直隷省]][[順天府|順天府]][[武清县|武清县]]人，[[清|清末]][[中華民国|民初]]军事将领。

==生平==
李长泰毕业于[[天津武備学堂|天津武備学堂]]，1895年（[[光緒|光緒]]21年）加入[[新建陸軍|新建陸軍]]。此後。历任工程营管带、工兵参領、[[陆军第六镇|北洋第六鎮]]協統。<ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>

[[中華民国|中華民国]]成立後的1912年（[[民国紀元|民国]]元年），署理[[直隷省|直隷省]]大名鎮总兵。翌年，转任冀南鎮守使。1914年（民国3年）9月，任陆军第八師師長。1915年12月[[袁世凱称帝|袁世凱称帝]]，李长泰获授三等男爵。<ref name=xu/><ref name=liu>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

1917年（民国6年）7月，[[張勳復辟|張勳復辟]]之際，李长泰被[[段祺瑞|段祺瑞]]任命为[[討逆軍|討逆軍]]東路副司令，参加讨伐[[張勳|張勳]]。翌月，获任命为[[步軍統領衙門|步軍統領衙門]]統領。1919年（民国8年）5月辞職，此後寓居天津。<ref name=xu/><ref name=liu/>1919年5月21日，获[[北洋将军府|将军府]]授“泰威将军”。<ref>1919年5月22日第1184号. 《政府公报》第一四三册.</ref>

1922年（民国11年），李长泰逝世。享年61岁。<ref name=xu/>
