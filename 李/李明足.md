'''李明足'''，[[中華民國|中華民國]][[臺南|臺南]]人，[[國立臺東大學|國立臺東大學]][[兒童文學|兒童文學]]研究所碩士<ref>[http://ice.nttu.edu.tw/files/523-1011-21.php?Lang=zh-tw 國立臺東大學-兒童文學研究所的雜誌繪本棒棒糖第16期 2009.06收錄]</ref>，[[佛光大學|佛光大學]]中文與應用學系博士，知名兒童[[繪本|繪本]]作家、[[講師|講師]]<ref>[http://www.books.com.tw/products/0010710686 博客來-看見]</ref><ref>[http://www.rti.org.tw/program/focus/?recordId=3683 中央廣播電臺-臺灣好漾－李明足引領孩子進入文學世界]</ref>。著有 《看見》等知名繪本著作，看見這一本繪本躋進博客來全國前5名<ref>[http://www.chinatimes.com/newspapers/20160628000610-260107 林內媳生態繪本 博客來銷售讚 《看見》是博客來書城第一本躋身全國前5名銷售佳績的農村生態繪本。]</ref><ref>[http://www.kmdn.gov.tw/1117/1271/1272/189218/?cprint=pt 金門日報-毛毛蟲基金會講師李明足以繪本引領新移民親子進入閱讀的世界]</ref>。

== 得獎經歷 ==

*蘭陽文學獎

== 講師經歷 ==
*2014年與[[陳玉金|陳玉金]]、[[何華仁|何華仁]]、祝建太、[[邱承宗|邱承宗]]、[[孫心瑜|孫心瑜]]、[[黃郁欽|黃郁欽]]、[[陶樂蒂|陶樂蒂]]、陳維霖等知名作家一起擔任由[[宜蘭縣|宜蘭縣]]政府文化局舉辦的[[蘭陽繪本創作營|蘭陽繪本創作營]]講師。<ref>[http://www.epochtimes.com/b5/15/12/5/n4589229.htm/amp 蘭陽繪本創作營 新書發表]</ref>。
*2015年全國巡迴文藝營與[[李如青|李如青]]、[[林柏廷|林柏廷]]、[[孫心瑜|孫心瑜]]、[[祝建太|祝建太]]和[[謝鴻文|謝鴻文]]一起擔任繪本兒童文學組講師。
