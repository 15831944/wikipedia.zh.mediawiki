'''李宝荣'''（{{bd|1958年|12月|||catIdx=}}），[[陕西省|陕西省]][[洛川县|洛川县]]人<ref name=caixin/>，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中华人民共和国国务院副秘书长|国务院副秘书长]]。

==生平==
1976年11月参加工作，1984年9月加入中国共产党，大学本科学历，工商管理硕士<ref name=ggj/>。原在[[陕西省人民政府|陕西省人民政府]]工作，36岁转入[[中华全国供销合作总社|中华全国供销合作总社]]，历任秘书、国际合作部副部长等职<ref name=caixin>{{Cite web |url = http://china.caixin.com/2017-04-11/101076769.html |title = 国管局局长李宝荣任国务院副秘书长 |author =  |date = 2017-04-11 |publisher =财新网  |language =  |accessdate =  |quote =  }}</ref>。1998年9月到[[国务院机关事务管理局|国务院机关事务管理局]]（简称国管局）工作，历任人事司副司长、司长，财务管理司司长。2004年11月，任国管局党组成员兼财务管理司司长。2005年11月，任国管局党组成员兼机关党委书记。2007年2月，任国管局副局长、党组成员兼机关党委书记。2011年5月，任国管局副局长、党组成员。2015年10月，任国管局局长、党组书记<ref>{{cite news|title=李宝荣任国管局局长 焦焕成到龄卸任|url=http://district.ce.cn/newarea/sddy/201510/21/t20151021_6766131_1.shtml|date=2015-10-21|newspaper=中国经济网}}</ref><ref name=ggj/>。2017年3月，任国务院机关党组成员兼国管局局长、党组书记<ref name=ggj/>。2017年4月，任国务院副秘书长、机关党组成员，国管局局长、党组书记<ref name=ggj/><ref>{{cite news|title=国务院任免李宝荣、王永红、彭家瑞等职务|url=http://www.ce.cn/xwzx/gnsz/gdxw/201704/11/t20170411_21867785.shtml|accessdate=2017-04-11|newspaper=中国经济网}}</ref>。他是[[中共十八大|中共十八大]]、[[中共十九大|十九大]]代表<ref name=ggj>{{Cite web |url =http://www.ggj.gov.cn/zzjg/vldxx/200709/t20070924_21877.htm  |title =  局长、党组书记  李宝荣  |author =  |date =  |publisher =国家机关事务管理局  |language =  |accessdate =2018-02-16  |quote =  }}</ref>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|date=2018-01-26|last=|work=网易}}</ref>。
