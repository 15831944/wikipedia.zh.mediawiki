'''李瑞英'''（{{bd|1961年|7月16日||catIdx=Li,李}}），女，[[中华人民共和国|中国]][[北京|北京]]人，祖籍[[河南省|河南省]][[南乐县|南乐县]]。[[中国中央电视台|中国中央电视台]]新闻中心播音部副主任<ref>{{cite web |title = 李瑞英升官 央视新闻播报主持人全归她管（原载《华西都市报》） |publisher = 四川在线 |date = 2009-07-30 |url = http://www.scol.com.cn/ent/gatylkb/20090730/200973030754.htm |accessdate = 2009-07-30 |language = zh }}</ref>、新闻播音员、主持人。

== 生平 ==
1983年毕业于[[北京广播学院|北京广播学院]]（现中国传媒大学）。后在[[江苏电视台|江苏电视台]]主持新闻节目。1986年调入[[中国中央电视台|中国中央电视台]]主持《[[新闻联播|新闻联播]]》至今。历任播音指导、[[国家广播电影电视总局|国家广播电影电视总局]]高级职称评委会委员、中国广播电视播音学会]副会长、主持人协会副会长、[[全国政协|全国政协]]委员、[[全国青联|全国青联]]常委。

2014年5月28日，有媒体报道李瑞英与[[张宏民|张宏民]]退出《新闻联播》，后得到其本人证实<ref>{{cite news |url = http://ent.ifeng.com/a/20140528/40089166_0.shtml |title = 李瑞英证实与张宏民告别《新闻联播》 |publisher = 凤凰网 |accessdate = 2014-05-28}}</ref>。退休后，她负责了《[[新华字典|新华字典]]》官方手机应用版的单字播读工作<ref>{{cite news|title=60岁的《新华字典》推出了手机App 新闻联播播音员配音|url=http://tech.qq.com/a/20170612/006529.htm|publisher=澎湃新闻 |date=2017-06-12 |accessdate=2017-06-12 }}</ref>。
