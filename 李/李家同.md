'''李家同'''（{{bd|1939年|1月5日||}}）是[[台灣|台灣]]資訊科學家與作家。出生於[[上海市|上海市]]，其父[[李國瑊|李國瑊]]為經濟學家，為[[李鴻章|李鴻章]]長兄[[李瀚章|李瀚章]]之孫，李家同則為曾孫。

據說與李鴻章同日出生，故取名為「家同」。
李鴻章於國曆1823年2月15日出生，當天農曆為正月初五，而李家同的國曆生日為1939年1月5日。<ref>[http://hfu.edu.tw/~cl/lifewriting/book09.html 作家介紹—李家同].華梵大學中國文學系.</ref>曾任[[國立清華大學|國立清華大學]]代理校長、[[靜宜大學|靜宜大學]]與[[國立暨南國際大學|國立暨南國際大學]]校長、暨大資訊工程學系及資訊管理學系教授。於2008年5月31日卸下暨南大學教授職務退休。2009年獲[[中華民國總統府|中華民國總統府]]聘為無給職資政。目前在[[國立清華大學|國立清華大學]]兼任教職。2011年1月6日獲[[中華民國教育部|中華民國教育部]]頒予[[一等教育文化獎章|一等教育文化獎章]]。

== 專業貢獻 ==
李家同的研究領域為[[符號邏輯|符號邏輯]]、[[演算法|演算法]]設計分析、[[平行計算|平行計算]]方法、[[生物資訊|生物資訊]]等相關研究，並在這些領域發表超過一百篇以上之國際論文。近年則從事時事評論和英文學習書籍寫作。

== 軼事 ==
2018年5月7日，李家同於Facebook公開發表一篇標題為【我感到非常難過】之網誌，並附上總統府星際大戰紀念日(5/4)之活動照，認為「總統府是一個十分莊嚴的地方，不是胡鬧的地方」，

並問及「白金漢宮會有這種事嗎?克林姆林宮會有這種事嗎?白宮會有這種事嗎?」卻馬上被網友翻出日前白宮、白金漢宮之星際大戰紀念日活動舊照予以吐槽。

== 爭議言論 ==
* 2010年3月13日，李家同在《[[聯合報|聯合報]]》投書〈如果我被殺！〉一文，表達自己支持[[廢除死刑|廢除死刑]]的立場。李家同在文中宣稱自己如果被殺，也要寬恕殺他的人，也要為他祈禱。並宣稱社會應該注重「寬恕」的重要性。引發外界對非受害者家屬的第三者是否有資格提出寬恕的討論。
{{Quote box| align = center | quoted = true | bgcolor = #FFFFF0  | quote =
 |quote  = 如果我被殺，只要一息尚存，我一定會告訴殺我的人，我絕對寬恕他，也會為他祈禱，更會要求友人照顧他的家人，尤其要使他的孩子不再留在社會黑暗的角落。如果我無法說這些話，我也無所謂，因為我的太太和女兒絕不會痛恨殺我的人，她們一定會為他祈禱，也會照顧他的家人。不僅如此，我的學生不會有一個人希望他被處死刑。
 |source = –李家同，〈如果我被殺！〉首段，《聯合報》，2010年3月13日。}}李家同的言論遭到批評「寬恕」應屬於受害者自發「高尚的感情」，而非透過死刑廢除來勉強實現。<ref>{{Cite web|url = https://forum.ettoday.net/news/128689|title = 感情是不能勉強的，寬恕也一樣！（翁立揚）|date = 2012-11-08|publisher= ETtoday新聞雲|language=zh-tw}}</ref>此外「原諒」應屬於受害者家屬的權利，未歷經受害者傷痛的一般人，沒有權利以假想情況大言奢談原諒兇手。李家同個人的宗教家精神無法在死刑議題上說服社會大眾跟隨他，因為社會大眾是人不是宗教家。社會需要制度與法律來維繫，不能依賴個人的道德感來維持社會秩序。<ref>{{Cite web|url = https://tw.appledaily.com/headline/daily/20100318/32369473/|title = 原諒是受害者家屬的權利（徐元春）|date = 2010-03-18|publisher= 蘋果日報|language=zh-tw}}</ref>
* 2018年5月7日，李家同在臉書上發表一篇網誌針對[[總統府_(臺灣)|中華民國總統府]]在5月4日「星際大戰日」開放給星戰粉絲以《[[星際大戰|星際大戰]]》中的裝扮參訪的事件評論。李家同在標題開宗明義寫到「我感到非常難過」，並在文中稱「對我來講，總統府是一個十分莊嚴的地方，不是胡鬧的地方，所以當我看到以下的照片，我實在感到非常難過，怎麼會發生這種事？[[白金漢宮|白金漢宮]]會有這種事嗎？[[克林姆林宮|克林姆林宮]]會有這種事嗎？[[白宮|白宮]]會有這種事嗎？」。隨後遭網友直接在底下留言區貼上兩年前白宮同樣舉辦「星戰日」的新聞報導，時任美國總統[[歐巴馬|歐巴馬]]也共襄盛舉的照片。而英國白金漢宮也出現過《星際大戰》的角色扮演。知名警察球評[[石明謹|石明謹]]也在個人臉書貼文上諷刺「李家同：我感到非常難過/歐巴馬：你叫我？」。<ref>{{Cite web|url = https://www.nownews.com/news/20180508/2749792|title = 總統府辦星戰日「我感到非常難過」　李家同遭網友狂打臉|accessdate = 2018-05-08|publisher= nownews|language=zh-tw}}</ref><ref>{{Cite web|url = http://news.ltn.com.tw/news/life/breakingnews/2418417|title = 白宮會有嗎？ 李家同批總統府星戰日胡鬧 秒遭打臉|accessdate = 2018-05-08|publisher= 自由電子報|language=zh-tw}}</ref>
