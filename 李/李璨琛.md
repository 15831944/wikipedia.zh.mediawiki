'''李璨琛'''（{{lang-en|Sam Lee}}，{{bd|1975年|9月27日|catIdx=L李}}），又名'''李璨森'''，原名'''李瑋璁'''，香港電影演員。代表作品包括《[[香港製造|香港製造]]》、《[[特警新人類|特警新人類]]》、《[[狗咬狗_(2006年電影)|狗咬狗]]》等。並曾跟[[Ketchup|Ketchup]]合作；現時也是[[DJ|DJ]]，代號'''DJ Becareful'''。

== 背景 ==
李璨琛家有兩兄兩姊，早年在[[藍田_(香港)|藍田]]22座[[屋邨|屋邨]]長大，與[[敖嘉年|敖嘉年]]為樓上及樓下鄰居{{noteTag|資料來自無綫電視於2017年4月30日播出的節目《屋邨遊樂團》第8集 - 藍田邨。}}。他畢業於藍田循道小學（1987年上午校<ref>{{cite web |url=http://becarefullee.lofter.com/post/1d6a3d62_10995e02 |title=昨晚是「藍田循道小學一九八七年（上午校）畢業三十週年紀念暨謝師宴」一班三十年沒有見的老師和同學們都在一片愉快的氣氛中暢談了一個晚上實在是難得又開心 |last= |first= |date=2017-7-16 |website= |publisher=李璨琛Sam Lee |access-date= 2019-1-15|quote=}}</ref>；現為[[藍田循道小學|藍田循道小學]]）。升中時獲派位於[[香港島|香港島]]的[[慈幼英文學校|慈幼英文學校]]。但讀至中二時輟學，並且跟父親到[[油麻地果欄|油麻地果欄]]運送水果。及後曾向朋友的父親進修[[水電|水電]]課程。李璨琛在15歲時到[[西貢|西貢]]的村屋地盤當水電技工。{{noteTag|關於李璨琛加入娛樂圈前的生活與其家庭背景的資料出自2012年8月14日出版的《Face》周刊第273期專訪〈面對面 唔X係呀?!-李璨琛〉。}}直至1996年被電影導演[[陳果|陳果]]在街上發掘主演《[[香港製造|香港製造]]》，並憑這次演出獲得[[1998年香港電影金像獎|1998年香港電影金像獎]]的最佳新人獎。在2003年他與兒時玩滑板的好友Frankie建立自家品牌「SUBCREW」，經營男裝潮流及街頭衣著。其品牌在高峰期拓展至中國內地五個城市和東南亞國家，並設有專門店。

== 個人生活 ==
2012年跟拍拖九年的女朋友梁志瑩結婚。

2018年2月20日早上8時許，李璨琛於[[九龍城|九龍城]]駕著黑色座車沿[[太子道東|太子道東]]行駛期間被交通警截查，接受酒精呼氣測試後證實超標，涉嫌酒後駕駛被捕。
