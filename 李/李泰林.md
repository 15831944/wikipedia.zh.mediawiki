'''李泰林'''（{{lang-ko|'''이태임'''}}，{{lang-en|'''Lee Tae-Im'''}}，{{bd|1986年|10月5日}}），[[韓國|韓國]]引退[[女演員|女演員]]<ref>[http://csearch.naver.com/content/eprender.nhn?where=m&key=PeopleMobileDetailEndPage&pkid=1&q=이태임&os=158896&ro=%20 이태임 인물정보 : 네이버 통합검색]</ref>。2009年，她於SBS晨間劇《[[不要猶豫|不要猶豫]]》中首次擔任要角而備受關注，並且於年底獲得演技大賞中的新人賞。2018年3月19日，李泰林宣布退出演藝圈<ref name=quit>[https://star.ettoday.net/news/1133290 《有品位的她》女星突宣布引退！　曾飆罵後輩演員「垃圾女人」]</ref>，因其懷孕三個月並預計產子後結婚<ref>{{cite web|url=http://entertain.naver.com/ranking/read?oid=001&aid=0009973260|title=이태임, 임신 3개월…"출산후 결혼식…연예계 은퇴"|publisher=Naver entertain|accessdate=2018-03-21|date=2018-03-21|language = ko}}</ref><ref>{{cite web|url=http://entertain.naver.com/ranking/read?oid=108&aid=0002686826|title=[단독]이태임 임신 3개월, 출산 후 결혼..계약해지|publisher=Naver entertain|accessdate=2018-03-21|date=2018-03-21|language = ko}}</ref>，所屬經紀公司Management HENEM與其解約<ref>{{cite web|url=http://entertain.naver.com/ranking/read?oid=025&aid=0002806714|title=이태임, 임신 3개월…"출산 후 연상 사업가와 결혼…연예계 은퇴"|publisher=Naver entertain|accessdate=2018-03-21|date=2018-03-21|language = ko}}</ref>。

== 經歷 ==
李泰林於1986年10月5日在[[蔚山廣域市|蔚山廣域市]]出生，就讀過{{link-ko|聖光女子高等學校|성광여자고등학교}}，並畢業於[[漢陽大學|漢陽大學]]演劇映畫學科。

原在高中畢業後主修服裝設計，在2007年時因接拍攝了首支廣告而進入演藝圈，之後便開始參加電視劇的試鏡。其首部演出的電視劇為[[MBC週末連續劇|MBC週末連續劇]]《[[我人生的黃金期|我人生的黃金期]]》，她在劇中飾演男主角劉泰日（[[李鍾原|李鍾原]] 飾演）的驕縱的妹妹劉泰英。

2009年，李泰林在[[KBS大河連續劇|KBS大河連續劇]]《[[千秋太后_(電視劇)|千秋太后]]》中飾演金密花一角，同年10月，李泰林首度擔綱了主演，演出了[[SBS晨間連續劇|SBS晨間連續劇]]《[[不要猶豫|不要猶豫]]》。於劇中，她飾演向前男友一家人復仇的女主角張秀賢一角，而該劇平均收視13%，最高收視則獲得17.1%（AGB尼爾森）。而在該年底的[[SBS演技大賞|SBS演技大賞]]上，李泰林以此劇獲得了首座演技獎項－新星賞。

2010年6月，李泰林在[[KBS週末連續劇|KBS週末連續劇]]《[[嫁給我吧|嫁給我吧]]》中飾演介入男女主角婚姻的小三尹瑞瑩，而在該劇中，她身著泳衣的畫面引起了許多話題。

2011年，李泰林與[[嚴泰雄|嚴泰雄]]、[[周元_(韓國)|周元]]、[[鄭進永|鄭進永]]及[[成東日|成東日]]主演了首部電影《[[特別搜查本部|特別搜查本部]]》。

2013年，與[[梁東根|梁東根]]、[[朱相昱|朱相昱]]共同演出電影《[[懲戒者|懲戒者]]》，劇中她扮演男主角昌錫（梁東根 飾演）的未婚妻智希。而李泰林演出的兩部電影作品在票房上皆表現的不太理想，《特別搜查本部》在120萬名觀影人數的盈虧分界點上，僅獲不到110萬名的觀影人數，《懲戒者》更是僅動員了19萬名觀眾。

2014年，李泰林與[[李昭娟|李昭娟]]時隔6年繼《我人生的黃金期》後再次合作，演出了[[JTBC周末連續劇|JTBC周末連續劇]]《[[12年後的重逢：山蒜醬湯|12年後的重逢：山蒜醬湯]]》，劇中其飾演與女主角張國（李昭娟 飾演）及男主角柳俊秀（[[南宮民|南宮民]] 飾演）有著複雜三角戀的朱多海一角，然該劇播出未過半，就因收視低迷而被[[JTBC|JTBC]]從50集腰斬剩26集。同年6月，李泰林與[[李民基|李民基]]、[[朴星雄|朴星雄]]主演了電影《[[為了皇帝|為了皇帝]]》，劇中李泰林與李民基有著大膽的裸露床戲而引起討論，然而票房卻依舊失利。

2015年2月，李泰林參與MBC綜藝節目《同屬相的課外輔導》的錄影，後來因健康問題而退出，然此時便爆出了其在節目拍攝時辱罵[[金藝元|金藝元]]的影像，爾後此事被證實，李泰林因此受眾譴責。

起初大眾普遍支持金藝媛，然而就在3月底，兩人對話內容原委被公開，事實證明確實是因金藝媛並未說敬語導致事件的發生。事件雖經澄清，仍對李泰林形象重創，同時李泰林正參與[[SBS週末特別計劃劇|SBS週末特別企劃]]《[[我的心一閃一閃|我的心一閃一閃]]》的演出，亦導致該劇收視極差，迫得SBS將該劇從50集腰斬至30集（後再度腰斬剩26集），但在該劇播畢前，李泰林已因健康問題中途退出。李泰林在這期間曾表示因輿論壓力、健康問題及事業不順等原因，欲考慮退出演藝圈。

而在暫停了工作休息療養多月後，於10月底以有線台{{link-ko|Drama H|드라마 H}}首部自製電視劇《[[You_Will_Love_Me|You Will Love Me]]》復出，後在2017年9月5日SBS《正式演藝深夜》節目中首度談及曾因此事件重創演藝事業，因而一度萌生自殺念頭。

2018年3月19日，李泰林宣布引退，而李泰林所屬經紀公司向韓媒表示毫不知情<ref name=quit>[https://star.ettoday.net/news/1133290 《有品位的她》女星突宣布引退！　曾飆罵後輩演員「垃圾女人」]</ref>；而後因李泰林本身有三個月個身孕並預計產子後與圈外丈夫<ref group=註>職業為企業併購專家。</ref>結婚<ref>{{cite web|url=http://entertain.naver.com/ranking/read?oid=001&aid=0009973260|title=이태임, 임신 3개월…"출산후 결혼식…연예계 은퇴"|publisher=Naver entertain|accessdate=2018-03-21|date=2018-03-21|language = ko}}</ref><ref>{{cite web|url=http://entertain.naver.com/ranking/read?oid=108&aid=0002686826|title=[단독]이태임 임신 3개월, 출산 후 결혼..계약해지|publisher=Naver entertain|accessdate=2018-03-21|date=2018-03-21|language = ko}}</ref>，所屬經紀公司與其解約<ref>{{cite web|url=http://entertain.naver.com/ranking/read?oid=025&aid=0002806714|title=이태임, 임신 3개월…"출산 후 연상 사업가와 결혼…연예계 은퇴"|publisher=Naver entertain|accessdate=2018-03-21|date=2018-03-21|language = ko}}</ref>。長子於2018年9月10日出生。
