'''李宗霖'''（{{bd|1993年|9月20日|||catIdx=L李}}），[[香港|香港]]籍[[台湾|台湾]]男演員<ref>{{Cite web |url=http://www.mingweekly.com/entertainment/entertainmentnews/content-10011.html |title=李宗霖母胎單身 甘做炎亞綸備胎 |accessdate=2018-05-03 |author=李雨勳 |publisher=Mingweekly}}</ref><ref>{{Cite web |url=https://m.weibo.cn/status/E2N8Fu6xq#&gid=1&pid=1 |title=《校花的贴身高手》片尾演员表}}</ref>，出生於台湾高雄。父亲为香港人，曾为台北[[晶华酒店|晶华酒店]]的主厨，母亲为台湾人，是家庭主妇。目前是[[关晓彤|关晓彤]][[工作室|工作室]]旗下演员，主要在[[中國大陆|中國大陆]]发展。代表作為《[[智勝鮮師|智勝鮮師]]》和《[[校花的貼身高手|校花的貼身高手]]》目前正在參加愛奇藝自製選秀節目[[青春有你|青春有你]]。
