== 生平 ==
李克俭出演毛泽东始于1977年，已经出演影视剧50多部，被誉为演毛泽东“神形兼备”的演员。<ref>{{cite web |url=http://fj.sina.com.cn/news/video/d/2011-07-25/14099545.html |title= 《微博超级大》特型演员李克俭 称后半生演毛主席很幸福|accessdate= 2011年7月25日 |publisher=新浪网 |archiveurl=http://fj.sina.com.cn/news/video/d/2011-07-25/14099545.html |archivedate=2013年1月4日 |work= |language=zh-cn}}</ref><ref>{{cite web |url= http://www.fjsen.com/zhuanti/2011-07/08/content_5119871.htm|title=李克俭 |accessdate=2011 |publisher= 东南网|archiveurl=http://www.fjsen.com/zhuanti/2011-07/08/content_5119871.htm |archivedate=2013年1月4日 |work= |language=zh-cn}}</ref>
