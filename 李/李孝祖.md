'''李孝祖'''（{{bd|1987年|2月3日}}），[[台灣|台灣]][[音樂製作人|音樂製作人]]、[[吉他|吉他手]]。近年台灣獨立音樂圈最受矚目的新生代製作人。出道製作的幾張專輯：草東沒有派對《醜奴兒》、茄子蛋《卡通人物》、LEO37+SOSS《Be Well World》，風格路線各有不同，卻都是標竿等級的力作。<ref>{{cite web |author1=馬世芳 |title=7/25 耳朵借我：專訪製作人李孝祖 |url=https://medium.com/%E6%88%91%E6%98%AF%E9%A6%AC%E4%B8%96%E8%8A%B3/lee-xiao-tsu-f11012eae3d5 |accessdate=2018-07-25}}</ref>

其製作首張專輯的[[草東沒有派對|草東沒有派對]]、[[茄子蛋|茄子蛋]]，分別於2017年、2018年獲得[[第28屆金曲獎|第28屆金曲獎]]、[[第29屆金曲獎|第29屆金曲獎]]最佳新人獎。

== 職業音樂製作 ==
李孝祖退伍後，加入[[福茂唱片|福茂唱片]]<ref>{{cite web |author1=樂手巢 |title=用心生活，做出好音樂：好意思錄音室 |url=https://ysolife.com/how-is-it-studio/ |website=https://ysolife.com/ |publisher=樂手巢 |accessdate=04/19/2016}}</ref>，作為錄音助理，合作錄製過[[韋禮安|韋禮安]]等音樂人。其後頂下杉特（張懸吉他手）的錄音室，改建為好意思錄音室，專心致力於獨立音樂製作。2017年，以[[草東沒有派對|草東沒有派對]]《醜奴兒》專輯中的單曲〈大風吹〉，獲得[[第28屆金曲獎|第28屆金曲獎]]流行音樂作品類年度歌曲獎。2018年，以[[茄子蛋|茄子蛋]]《卡通人物》專輯，獲得[[第29屆金曲獎|第29屆金曲獎]]最佳台語專輯獎。
