'''李璐'''（{{bd|1992年||}}），笔名“林深之”，[[中国|中国]][[作家|作家]]，出生于[[山西|山西]][[柳林县|柳林县]]。

==简介==
李璐6岁时因意外导致脊髓神经损伤，造成双下肢瘫痪，从此与轮椅为伴。因身体残疾，李璐没有办法进入学校学习，直到2006年14岁才开始自学认字。2008年，李璐在网上以“璐教主”的笔名发表了自己的第一篇小说作品《水晶鞋D魔力》，受到不少读者的肯定<ref name="w">{{cite web |title=吕梁残疾女孩轮椅上自学拼音书和字典实现作家梦 |url=http://shanxi.sina.com.cn/news/report/2014-06-03/074359501.html |website=新浪山西 |accessdate=2019-01-18}}</ref>。此后李璐又发表了多个作品，2010年受邀担任网络文学杂志《倾城色》的文字总监。2012年成为吕梁市作家协会会员<ref name="e">{{cite web |title=轮椅女孩的微笑人生(组图)_网易新闻 |url=http://news.163.com/13/0426/04/8TC3FCO200014AED.html |website=网易新闻 |accessdate=2019-01-18}}</ref>。2013年获得“最美吕梁人”称号<ref>{{cite web |title=[山西] 吕梁举行首届“最美吕梁人”颁奖典礼(组图) |url=http://roll.sohu.com/20130324/n370066200.shtml |website=搜狐新闻 |accessdate=2019-01-18}}</ref>。2015年出版了小说《女孩，你要好好爱自己》<ref>{{cite web |title=她，坐在轮椅上追寻文学梦想 |url=http://www.sxrb.com/sxxww/xwpd/dsxw/5763015.shtml |website=山西新闻网 |accessdate=2019-01-18}}</ref>。
