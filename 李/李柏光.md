'''李柏光'''{{BD|1968年|10月|2018年|2月26日|catIdx=L李}}是一位[[人权律师|人权律师]]，目前為[[北京|北京]]「共信律师事务所」執業律師。1997年，李柏光從[[北京大學|北京大學]][[法律系|法律系]]博士畢業；1997年8月至1998年4月在[[海南大學|海南大學]]法學院當教師。在1999年到2001年期间翻译出版过《Self-help》《Character》《Duty》《Thrift》等品德教育书籍。李柏光在2006年初取保候審期滿，重新開始為受迫害的[[基督教|基督教]]家庭教會和其他權益受害者提供[[法律|法律]]諮詢，為此不斷被警方約談，行動自由受到限制。<ref name="維權網">{{cite web|url=http://wq.equalit.ie/?p=1779|title=李柏光，男，法律工作者|publisher=維權網|accessdate=2013-02-07|deadurl=yes|archiveurl=https://web.archive.org/web/20141224150809/http://wq.equalit.ie/?p=1779|archivedate=2014-12-24}}</ref>
2018年2月26日凌晨3点在[[南京市|南京市]][[中国人民解放军第八一医院|中国人民解放军第八一医院]]因肝癌驟逝，享年49歲<ref>[http://www.cna.com.tw/news/acn/201802260141-1.aspx 陸人權律師李柏光突過世 國際組織指不尋常]</ref>。

== 生平 ==
*1968年10月1日出生於[[湖南省|湖南省]][[郴州市|郴州市]][[嘉禾縣|嘉禾縣]]的小山村，為五子中的老么。<ref name="焦國標">{{cite web |url=http://www.newcenturynews.com/Article/gd/201009/20100912001457.html |title=焦國標：從屬血氣到屬聖靈——李柏光律師的維權道路|publisher=新世紀新聞網|accessdate=2013-02-07}}</ref>

*1987年考進[[湘潭大學|湘潭大學]][[哲學系|哲學系]]<ref name="焦國標" />，1997年從[[北京大學|北京大學]][[法律系|法律系]]博士畢業。<ref name="維權網" />
*2001年3月到2004年底，李柏光為全國十几个省市數萬失去土地的村民或庫區移民，如建福省福州市闽侯县和宁德地区福安市几个乡镇失去土地的農民以及河北唐山和秦皇島滿族移民、黑龙江省、吉林省、辽宁省、贵州省、新疆等省的农民和市民，提供法律諮詢服務，擔任他們的法律顧問，為他們撰寫罷免政府负责人和司法机关负责人動議書和公民聯署書。<ref name="維權網" />
*2003年7月開始，李柏光為[[山東省|山東省]][[臨沂市|臨沂市]][[沂南縣|沂南縣]]盲人、維權人士[[陳光誠|陳光誠]]先生提供法律諮詢服務。<ref name="維權網" />
*2004年12月14日，在去[[福建省|福建省]]寧德地區[[福安市|福安市]]為農民提供法律服務時，李柏光被[[福安市|福安市]]公安局以涉嫌詐騙罪名拘捕。與此同時，李柏光在[[北京|北京]]的住處遭到警方查抄，拿走包括電腦軟硬盤等大批物件。<ref name="維權網" />
*2005年1月21日，李柏光在福建福安被關押37天后，警方因无确凿证据指控该罪，以取保候審的名義将李柏光釋放。<ref name="維權網" />
*2005年2月9日，剛踏出福建福安看守所的李柏光，接手代理首起普通公民、[[黑龍江省|黑龍江省]]遜克農場女工劉傑起訴[[國務院|國務院]]案。<ref name="維權網" />
*2006年5月11日，當時的[[美國|美國]]總統[[小布希|小布希]]在[[白宮|白宮]]的家庭客廳裡，接見了基督徒人权活动人士李柏光、[[余杰|余杰]]和[[王怡|王怡]]。<ref>{{cite web|url=http://www.chinesepen.org/Article/bhdt/200605/Article_20060512162427.shtml|title=美國總統布什接見余杰 王怡|publisher=獨立中文筆會|accessdate=2013-02-07}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
*2008年6月17日，李柏光與張建紅、[[李和平|李和平]]、[[滕彪|滕彪]]、[[陳光誠|陳光誠]]、姚福信、胡石根等人，獲得[[美國國家民主基金會|美國國家民主基金會]]頒發獎章，以表彰他们對於[[中國|中國]][[人權|人權]]與宗教自由的推動。據[[美國之音|美國之音]]報導，這是美國國家民主基金會在1998年後，首次頒發民主獎章給7名中國維權人士。6月23日，当时的美国总统小布什在白宫地图室会见了李柏光、王天成和李和平三位中国人权活动人士<ref>{{cite web |url=http://news.sina.com/us/chinesedaily/104-103-102-102/2008-06-20/06353001323.html |title=七名中國維權及異議人士獲頒美民主獎章|publisher=北美新浪網|accessdate=2013-02-07}}</ref>
