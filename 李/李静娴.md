'''李静娴'''（{{bd|？|||}}）女，[[山东|山东]][[青岛|青岛]]人，[[中华人民共和国|中华人民共和国]]及[[比利时|比利时]]女中音歌唱家。

==生平==
李静娴是1980年代[[中国人民解放军海军政治工作部文工团|海政歌舞团]]著名女中音歌唱家，与[[吕文科|吕文科]]、[[卞小贞|卞小贞]]、[[苏小明|苏小明]]等齐名。她音域宽广，音质明亮，音色淳厚，演唱中个人特点十分鲜明。1983年她推出了个人专辑《暮归》，成为[[中华人民共和国|中华人民共和国]]较早推出个人专辑的歌手之一。在1986年开播的电视连续剧《[[西游记_(1982年电视剧)|西游记]]》中，李静娴一人演唱了两首插曲《走啊走》、《青青菩提树》，随着该电视剧的热播，这两首插曲也广泛流传。在1980年代末至1990年代初的出国潮中，李静娴出国，后成为[[比利时|比利时]][[皇家铸币局剧院|皇家铸币局剧院]]终身歌唱家<ref>{{Cite web |url = http://5sing.kugou.com/yc/1905877.html |title = 朋友（李静娴） |author =  |date =  |publisher = 5sing |language =  |accessdate =2017-04-17  |quote =  }}</ref><ref>{{Cite web |url =http://news.163.com/12/0117/13/7NVMGE8G00014AEE.html  |title = “亲情中华”与比利时侨胞同庆小年 欧洲之旅收官|author =  |date = 2012-01-17 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.fcjv.be/site/?p=760 |title =  比利时华人青年联合会`比利时青田同乡会主办 2015“水立方杯”中文歌曲大赛比利时赛区|author =  |date = 2015-07-03 |publisher = 比华青联 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://mil.sohu.com/20130812/n383926401.shtml |title =宋祖英升任海政文工团团长 或将退居幕后  |author =  |date = 2013-08-12 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
