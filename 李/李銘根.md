'''李銘根'''（{{lang-pt|José Lei}}；{{bd|1930年|10月30日}}）<ref>{{cite web|title=李銘根的個人資料|url=https://www.olympic.org/jose-lei|website=Olympic|accessdate=2018-05-24|deadurl=no|archiveurl=https://web.archive.org/web/20161007024545/https://www.olympic.org/jose-lei|archivedate=2016-10-07}}</ref>，是一名建築設計師，前[[香港建築師學會|香港建築師學會]]副會長及前[[香港建築署|香港建築署]]署長，於1991年退休<ref>{{cite news|title=西九交財團承辦 前建築署長反駁 「公僕也能搞好西九」|url=http://paper.hket.com/article/1334865/%E3%80%8C%E5%85%AC%E5%83%95%E4%B9%9F%E8%83%BD%E6%90%9E%E5%A5%BD%E8%A5%BF%E4%B9%9D%E3%80%8D|accessdate=2018-05-24|agency=香港經濟日報|date=2005-04-26|deadurl=no|archiveurl=https://web.archive.org/web/20180525093428/http://paper.hket.com/article/1334865/%E3%80%8C%E5%85%AC%E5%83%95%E4%B9%9F%E8%83%BD%E6%90%9E%E5%A5%BD%E8%A5%BF%E4%B9%9D%E3%80%8D|archivedate=2018-05-25}}</ref>。他曾以射擊運動員的身份代表香港參加[[1968年夏季奧林匹克運動會|1968年夏季奧林匹克運動會]]<ref name="HKG"/>。他亦是[[澳門|澳門]]和[[香港|香港]]一些地標建築物的設計者，其代表作有[[香港藝術館|香港藝術館]]、[[香港文化中心|香港文化中心]]、[[莉娜大廈|莉娜大廈]]等<ref name="MM02"/>。

==經歷與成就==
李銘根出生於澳門，是一名[[澳門土生葡人|澳門土生葡人]]，其父[[李珠|李珠]]（{{lang-pt|António Lei}}）是一名[[工程師|工程師]]，他於1949年設計了位於[[南灣_(澳門)|南灣]]的政府合署大樓（現[[初級法院大樓|舊法院大樓]]）。李銘根早年在他父親的辦公室學習，曾在香港當實習生，回到澳門後為[[澳葡政府|政府]]設計了一些葡式住宅、法院、醫院、學校和政府住宅。1954年他再次回到香港，先後為建築師[[范文照|范文照]]和[[港英政府|香港政府]]部門工作，後來考取了建築師資格<ref name="MM02">{{cite journal|author1=呂澤強|title=19世紀末20世紀初 澳門的建築設計（下）|journal=澳門雜誌|date=2018-01-19|volume=121|url=http://www.macauzine.net/?action-viewnews-itemid-1187-page-2|deadurl=no|archiveurl=https://web.archive.org/web/20180524092651/http://www.macauzine.net/?action-viewnews-itemid-1187-page-2|archivedate=2018-05-24}}</ref>。

李銘根早年從一些年輕建築師接觸了一些新的設計觀點與方法。他是部分香港重要建築物的設計者，其代表作有黃大仙綜合游泳池、[[摩士公園游泳池|摩士公園游泳池]]、[[香港文化中心|香港文化中心]]、[[香港藝術館|香港藝術館]]等<ref name="MM02"/>。他曾評價香港的建築水準已達到世界一流水平，又指[[香港會展中心|香港會展中心]]、[[新世界中心|新世界中心]]、[[滙豐總行大廈|滙豐總行大廈]]、[[中銀大廈_(香港)|中銀大廈]]都是世界一流的建築物。他亦指，由於[[香港經濟|香港經濟]]實力雄厚以及香港的建築師來自全球各地，因此香港很快就有外國式樣的建築，甚至比起來更先進<ref>{{cite book|author1=王彤|title=《香港這座橋》|date=1996|publisher=香港文學報社|isbn=9627567272|page=1996}}</ref>。

===運動員生涯===
李銘根是一名射擊運動員，曾代表香港參加[[1968年夏季奧林匹克運動會|1968年夏季奧林匹克運動會]]的和1974年[[大英國協運動會|大英國協運動會]]的[[射擊運動|射擊運動]]比賽，在[[1984年夏季奧林匹克運動會|1984年夏季奧林匹克運動會]]擔任香港射擊隊的隊經理，他亦參加了[[1966年亞洲運動會|1966年]]，[[1970年亞洲運動會|1970年]]和[[1974年亞洲運動會|1974年的亞運會]]<ref>{{cite web|title=Jose Lei|url=https://thecgf.com/results/athletes/50932|website=CGF|accessdate=2018-05-24|deadurl=no|archiveurl=https://web.archive.org/web/20180525093428/https://thecgf.com/results/athletes/50932|archivedate=2018-05-25}}</ref><ref name="HKG">{{cite web |url=http://www.hkolympic.org/images/products/hong%20kong%20delegation%20for%20the%20multi-sports%20games.pdf |title=Hong Kong Delegation |accessdate=2015-12-31 |work=Hong Kong Olympic Committee |deadurl=no |archiveurl=https://web.archive.org/web/20160304060836/http://www.hkolympic.org/images/products/hong%20kong%20delegation%20for%20the%20multi-sports%20games.pdf |archivedate=2016-03-04 }}</ref><ref name="SportsRef">{{cite web |url=https://www.sports-reference.com/olympics/athletes/le/jose-lei-1.html |title=José Lei |accessdate=2015-12-25 |work=Sports Reference |deadurl=no |archiveurl=https://web.archive.org/web/20160126134324/http://www.sports-reference.com/olympics/athletes/le/jose-lei-1.html |archivedate=2016-01-26 }}</ref><ref>{{cite book|editor=S.
