'''李匡筹'''（{{bd|？||895年|1月25日|catIdx=L}}？<ref name=ZZTJ259>[[:s:資治通鑑/卷259|《资治通鉴》卷二百五十九]]</ref><ref>895年1月25日是李匡筹弃卢龙试图逃奔义昌军节度使卢彦威辖区的日子；但卢彦威伏杀李匡筹。因此李匡筹之死是在1月25日当天还是稍后并不确切。</ref>），或作'''李匡俦'''，[[晚唐|晚唐]][[军阀|军阀]]，[[景福|景福]]二年（893年）推翻兄长[[李匡威|李匡威]]后成为[[卢龙节度使|卢龙节度使]]直至[[乾宁|乾宁]]元年（894年）末或二年（895年）初战败。

== 家世及接管卢龙 ==
李匡筹生年不详，是卢龙军部[[范阳|范阳]]人。父[[李全忠|李全忠]]于[[光启|光启]]元年（885年）推翻时任节度使[[李可举|李可举]]，自任节度使；二年（886年）李全忠死后，李匡筹兄李匡威接管卢龙，称[[留后|留后]]，<ref name=BT180>[[:s:舊唐書/卷180|《旧唐书》卷一百八十]]</ref><ref>[[:s:資治通鑑/卷256|《资治通鉴》卷二百五十六]]</ref>随后被任为节度使。<ref name=NBT212>[[:s:新唐書/卷212|《新唐书》卷二百一十二]]</ref>

李匡威当政时，李匡筹的表现记载不多，只知道他时任兵马留后、检校[[司徒|司徒]]，并娶了国色天香的张氏为妻。景福二年，[[河东节度使|河东节度使]][[李克用|李克用]]攻李匡威盟友[[成德节度使|成德节度使]][[王镕|王镕]]，李匡威准备率军往救。在家人送他出征的家宴上，李匡威酒醉，强奸了张氏，李匡筹因此怀恨。三月，李匡威军成功打退李克用后正从成德返回，李匡筹夺取军府，自称留后，下令以兵符召还士卒。李匡威军大部弃之而去投奔李匡筹，使李匡筹在李匡威没有进一步军事行动的情况下接管了卢龙。<ref name=BT180/><ref name=BT20-1>[[:s:旧唐书/卷20上|《旧唐书》卷二十上]]</ref><ref name=BT142>[[:s:旧唐书/卷142|《旧唐书》卷一百四十二]]</ref><ref name=NBT10>[[:s:新唐书/卷10|《新唐书》卷十]]</ref><ref name=NBT211>[[:s:新唐书/卷211|《新唐书》卷二百一十一]]</ref><ref name=NHFD39>[[:s:新五代史/卷39|《新五代史》卷三十九]]</ref>李匡筹素来暗懦，后李匡威评价李匡筹：“卢龙没有脱离我家的控制，我有什么好记恨的！只可惜李匡筹才短，不能保守，能到两年，就是幸事了。”<ref name=ZZTJ259/><ref name=NBT212/>

随后，王镕迎李匡威来成德，出于感激，事之如父（王镕时年只有19岁<ref>[[:s:資治通鑑/卷255|《资治通鉴》卷二百五十五]]</ref>），但四月，李匡威试图从王镕手中夺取成德，被忠于王镕的士兵所杀。<ref name=BT142/>王镕以礼敛葬李匡威，素服哭之，遣使告李匡筹。得知李匡威死讯，李匡筹虽然对兄长怀恨，却佯作大怒，上表[[唐昭宗|唐昭宗]]请求攻成德为其报仇，昭宗下诏不许，并下诏请[[宣武军节度使|宣武军节度使]][[朱全忠|朱全忠]]调停。当时幽州人多不认同李匡筹夺取军镇，同月，幽州将领[[刘仁恭|刘仁恭]]驻[[蔚州|蔚州]]，所部思归，刘仁恭率军攻李匡筹。李匡筹在[[居庸关|居庸关]]败刘仁恭，刘仁恭逃奔河东投李克用。<ref name=HFD135>[[:s:旧五代史/卷135|《旧五代史》卷一百三十五]]</ref>六月，李匡筹遣使檄责王镕其杀兄之罪，但幕客写的檄文多不称意，只有记室[[马郁|马郁]]起草的檄文条列事状，提出十个可疑点，文辞华美丰赡，马郁以此知名。<ref>[[:s:旧五代史/卷71|《旧五代史》卷七十一]]</ref>李匡筹不顾昭宗未授权，攻成德辖下[[乐寿|乐寿]]、[[武强|武强]]，声称为李匡威报仇。<ref name=BT180/><ref name=NBT212/>李克用也兵出缚马关，败成德军于平山，趁机进攻王镕外垒，王镕因失去幽州的帮助，只得乞求结盟才了事。<ref name=NBT211/><ref name=NHFD39/><ref>[[:s:旧五代史/卷54|《旧五代史》卷五十四]]</ref>朱全忠奉诏遣判官[[韦震|韦震]]出使幽州和解两镇。<ref name=BT20-1/>乾宁元年正月，李匡筹被任为检校[[太保|太保]]、幽州大都督府长史、卢龙军节度使。<ref name=ZZTJ259/><ref>[[:s:授李筹卢龙军节度使制|陆扆《授李筹卢龙军节度使制》]]</ref>

== 败亡 ==
刘仁恭到河东后，得李克用厚待。十月，他经由李克用谋主[[盖寓|盖寓]]请求李克用给他步骑一万人，可取卢龙。李克用同意了，但最初只给他数千人。刘仁恭初攻卢龙因此失败。李匡筹因此错判形势，自信可败河东军，十月，屡侵河东。李克用大怒，亲自从晋阳率军亲自率军大举进攻卢龙，<ref>[[:s:旧五代史/卷53|《旧五代史》卷五十三]]</ref>十一月，很快攻占[[武州|武州]]，围[[新州|新州]]。十二月，李匡筹派大将所领的援军步骑六七万在[[段庄|段庄]]被李克用所选精兵大败，斩首万余级，生擒将校三百人，用绳子绑着斩杀于城下。当夜，新州向李克用投降。李克用又破卢龙军于威塞，进军攻取[[妫州|妫州]]。李匡筹又发兵出居庸关，李克用派精骑当其前以疲之，遣养子步将[[李存审|李存审]]从另一条路出击夹击其背，幽州军大败，杀获万计，李存审率前军占领居庸关。<ref>[[:s:旧五代史/卷52|《旧五代史》卷五十二]]</ref><ref>[[:s:旧五代史/卷56|《旧五代史》卷五十六]]</ref><ref>[[:s:新五代史/卷25|《新五代史》卷二十五]]</ref>李克用又谋划幽州戍将[[高思继|高思继]]兄弟在孔领关有兵三千为后患，遣人招之，高思继等闻河东军为李匡威报仇，欣然从之，为河东军前锋。李匡筹听闻高思继兄弟都叛了，率族人弃幽州，奔邻镇义昌军部[[沧州|沧州]]。义昌军节度使[[卢彦威|卢彦威]]却贪图李匡筹携带的辎重和妓妾，派军伏击李匡筹，杀之，<ref name=BT20-1/><ref name=NBT10/><ref>《新五代史·王处直传》载李匡筹曾借道[[中山|中山]]以入京师，时任[[义武军节度使|义武军节度使]][[王处直|王处直]]伏甲于城外以备不虞。李匡筹离去后，甲士趁机入城围义武军行军司马李应之家，执杀李应之。但[[光化|光化]]三年（900年）王处直才成为义武军节度使，当时李匡筹早已去世，此记载疑误，或发生于王处直为节度使前。</ref>尽俘其众。幽州投降李克用，<ref name=ZZTJ259/>李克用让刘仁恭掌控卢龙。<ref name=NHFD39/><ref name=HFD135/><ref>[[:s:新唐书/卷218|《新唐书》卷二百一十八]]</ref><ref>[[:s:旧五代史/卷26|《旧五代史》卷二十六]]</ref><ref>[[:s:新五代史/卷4|《新五代史》卷四]]</ref><ref>[[:s:新五代史/卷48|《新五代史》卷四十八]]</ref>张夫人因哺乳未能逃脱，刘仁恭将她俘虏后献给李克用，为李克用所纳，后成为其宠姬。<ref name=BT180/><ref name=NBT212/><ref>[[:s:旧五代史/卷49|《旧五代史》卷四十九]]</ref>

== 评价 ==
*《旧唐书》
**史臣曰：如[[朱克融|朱克融]]、[[杨志诚|杨志诚]]、[[史元忠|史元忠]]、[[张公素|张公素]]、李可举、李全忠，以不仁得之，靡更曩志。或寻为篡夺，或仅传子孙，咸非令终，盖其宜也。
**赞曰：蝎石之野，气劲人豪。二百余载，自相尊高。载义（[[李载义|李载义]]）、仲武（[[张仲武|张仲武]]），亦多忠劳。余因篡得，不仁何逃？
