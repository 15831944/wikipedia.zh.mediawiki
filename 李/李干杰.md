'''李干杰'''（{{bd|1964年|11月||catIdx=L李}}），[[湖南省|湖南]][[望城区|望城]]人。1989年7月参加工作，1984年加入中国共产党。[[清华大学|清华大学]]工程物理系核反应堆工程专业毕业，核反应堆工程及安全专业硕士研究生学历，工学硕士，高级工程师。中共第十九届中央委员。

== 生平==

1989年7月，清华大学硕士研究生毕业。1989年7月，任[[国家核安全局|国家核安全局]]北京核安全中心助理工程师、工程师。1991年9月至1993年1月，在法国核安全与辐射防护研究院进修。1993年3月至1998年7月，任国家核安全局主任科员、副处长、处长。1998年1月至1999年1月在湖南省平江县挂职，任中共平江县委副书记。1999年1月，任{{Link-fr|中国驻法国大使馆|Ambassade de Chine en France}}科技一等秘书。2000年7月，任[[中华人民共和国环境保护部|国家环保总局]]核安全司正处级干部、副司长。2001年7月，任国家环保总局核安全中心主任（正局级）兼党委副书记。2002年7月，任国家环保总局核安全司司长（兼任核安全中心主任、党委副书记至2005年7月）。2006年12月，任国家环保总局副局长兼[[国家核安全局|国家核安全局]]局长。2008年3月，机构改革后任[[中华人民共和国环境保护部|环境保护部]]副部长兼[[国家核安全局|国家核安全局]]局长<ref>{{cite web |accessdate = 2013-05-01 |url = http://news.sina.com.cn/c/p/2007-01-12/142212021282.shtml |title = 周建李干杰出任国家环保总局副局长 |publisher = [[搜狐|搜狐]]}}</ref>。

2016年10月，任[[中国共产党河北省委员会|中共河北省委]]副书记<ref>{{cite web |accessdate = 2016-10-28 |url = http://www.chinanews.com/gn/2016/10-28/8046807.shtml |title = 李干杰任河北省委副书记 赵勇不再任河北省委副书记 |publisher = [[中国新闻网|中国新闻网]]}}</ref>。

2017年5月，任[[中华人民共和国环境保护部|环保部]]党组书记<ref>{{cite web |accessdate = 2017-05-31 |url = http://news.sina.com.cn/c/nd/2017-05-31/doc-ifyfuvpm6876745.shtml?wm=3049_0022&from=qudao|title = 李干杰任环保部党组书记 陈吉宁不再担任 |publisher = [[新浪网|新浪网]]}}</ref>。6月27日，經十二届全国人大常委会第二十八次会议表決，出任環保部部長<ref>{{Cite web|url=http://www.npc.gov.cn/npc/xinwen/2017-06/27/content_2024508.htm|title=全国人民代表大会常务委员会决定任免的名单（2017年6月27日第十二届全国人民代表大会常务委员会第二十八次会议通过）|accessdate=2017-09-06|work=中国人大网}}</ref>。

2018年3月，任新组建的[[中华人民共和国生态环境部|生态环境部]]部长。
