'''李榮浩'''（{{bd|1985年|7月11日|catIdx=L李}}），[[中华人民共和国|中國]]男[[歌手|歌手]]、[[安徽|安徽]][[蚌埠市|蚌埠市]]人，現居[[北京|北京]][[朝陽區_(北京市)|朝阳区]]。2005年曾以創作藝人身份簽約日本[[愛貝克思集團|愛貝克思集團]]屬下[[Avex_trax|Avex trax]]唱片公司，為公司旗下多位歌手打造歌曲。2013年9月發行個人首張專輯《模特》正式以歌手身份出道。2014年李榮浩憑藉《模特》專輯入圍[[第25屆金曲獎|第25屆金曲獎]]流行音樂類[[最佳國語專輯獎_(金曲獎)|最佳國語專輯獎]]、[[最佳作詞人獎_(金曲獎)|最佳作詞人獎]]、[[最佳專輯製作人獎_(金曲獎)|最佳專輯製作人獎]]、[[最佳國語男歌手獎_(金曲獎)|最佳國語男歌手獎]]、[[最佳新人獎_(金曲獎)|最佳新人獎]]5項大獎，新聞評論其為「最強黑馬」<ref>[http://www.setnews.net/E/News.aspx?PageGroupID=8&NewsID=24073&PageType=3 三立新聞]</ref>，並且最終奪下最佳新人獎項<ref>[http://www.cna.com.tw/news/firstnews/201406285009-1.aspx 金曲最佳新人：浩]，中央社，2014年6月28日</ref><ref>[http://www.cna.com.tw/news/firstnews/201406285003-1.aspx 金2014金曲獎完整得獎名單]，中央社，2014年6月28日</ref><ref>{{cite video|title=抓到了！杨丞琳住李荣浩北京家|url=https://www.youtube.com/watch?v=gysKmACkXfw|accessdate=2015-09-19|agency=苹果日报|date=2015-07-14}}</ref>。現任女友是[[台灣|台灣]]歌手[[楊丞琳|楊丞琳]]<ref>{{cite web |title=重現世紀之吻？楊丞琳陪李榮浩赴英開唱十指緊扣 |url=http://www.chinatimes.com/realtimenews/20180104005422-260404 |publisher=中時電子報 |date=2018-01-04}}</ref><ref>{{cite web |url=https://www.youtube.com/watch?v=BucrYxz-V3I |title=演唱會喊喜歡她李榮浩霸氣吻楊丞琳 |publisher=台視新聞 |date=2017-09-17}}</ref>。

==早年經歷（幕后制作人）==
李荣浩9岁开始通过听磁帶自学[[吉他|吉他]]，最後痴迷而荒废学业，初二便開始[[辍学|辍学]]；在父母的極力要求下每周从[[蚌埠|蚌埠]]坐火车去[[南京|南京]]学习低音提琴，學習了6-7年时间，师从演奏家[[尹富源|尹富源]]教授；他曾开设过吉他教学班，学生最多时达上百人；在南京组建过乐队，并获得[[江苏省|江苏省]]第五届流行音乐大赛一等奖；曾通过技校学习获得高考资格，一共经历过3次[[高考|高考]]，但最终都因成績太低落榜。

李荣浩经历过严重的意外车祸，在医院休养了1年多；出院后不久父亲因[[白血病|白血病]]离世，他带着母亲到[[北京|北京]]透过给其他人写歌、制作音乐以償還債務；2004年，接受人生中第一次专访，在音乐杂志《吉他平方》上发表吉他单曲《烦》；2005年接受音乐杂志《现代乐手》专访并发表单曲《DREAM》；同年9月担任《我的音乐梦想》系列吉他丛书主编，后正式成为吉他中国乐手；同年签约[[日本|日本]][[愛貝克思唱片|愛貝克思唱片]]，成为其独家创作艺人，这一年他卖出第一支作品《醉青楼》，被台灣歌手[[范逸臣|范逸臣]]演唱；2008年参与制作演員赵薇专辑《我们都是大导演》，并担任演員陈坤专辑《谜：ME》的专辑制作人；2011年开始担任公益项目行走的力量音乐总监。

==演藝經歷（歌手时期）==


2013年9月16日，发行首张专辑《模特》；9月25日，在北京举办“HAO·LIVE 李荣浩<模特>音享会”；12月23日，携原创作品《满座》参加《[[中国好歌曲|中国好歌曲]]》第一季海选，未获提名。

2014年3月24日，发行概念EP《作曲家》，同名单曲获香港歌手[[王菲|王菲]]推荐；5月20日，凭借专辑《模特》入围第25届台灣金曲奖最佳国语男歌手、最佳新人、最佳专辑制作人、最佳国语专辑、最佳作词人五项提名，并最终获得最佳新人奖，是首個拿到該獎的大陆人，实现了从制作人到歌手的转型。

9月4日，签约[[华纳唱片|华纳唱片]]；11月28日，发行第二张同名专辑《李荣浩》，在攻下五大排行榜和G-Music冠军的同时，也随即拿到[[台湾|台湾]]、[[香港|香港]]、[[新加坡|新加坡]]、[[马来西亚|马来西亚]]四地[[iTunes|iTunes]]专辑榜冠军；11月30日，在台北举办新歌演唱会；12月31日，在台北参加跨年演唱会。

2015年1月16日，参加《我是歌手第三季》第三期踢馆赛，最終踢馆失败，但對其关注度飙升；2月19日， 推出首张Live翻唱EP 《iTunes Session》， 是内地乐坛首位与iTunes合作推出自己的iTunes Session EP并获得全球范围推广的音乐人，也让其成为美国iTunes流行音乐热门专辑榜TOP120榜上唯一的华语歌手；4月4日，在香港举办新歌演唱会，香港演員、歌手[[古巨基|古巨基]]担任嘉宾；4月28日，发布为电影《何以笙箫默》创作的片尾曲《不将就》，该曲拿下iTunes台湾流行乐热门单曲榜No.1，台湾及香港[[KKBOX|KKBOX]]综合新歌榜、整体涵盖率冠军，KKBOX华语单曲日榜、周榜、月榜冠军等；5月18日，凭借歌曲《喜剧之王》入围第26届金曲奖最佳作曲人奖；5月到10月，举办8场天生李荣浩巡回演唱会；12月31日，参加[[江苏卫视|江苏卫视]]跨年演唱会。

2016年1月22日，发行第三张专辑《有理想》，该专辑收获亚洲近百张榜单冠军；5月20-21日，有理想世界巡回演唱会首站在北京乐视体育生态中心举办2场；6月15日，在上海迪士尼开园盛典演唱《心世界》；9月3日，发布为电影《从你的全世界路过》创作的主题曲《不说》；9月17日，参加上海理想音乐节演出；10月5日，参加上海简单生活节演出；12月30日，参加浙江卫视跨年演唱会。

2017年1月1日，担任国剧盛典表演嘉宾；1月28日，参演电影《[[乘风破浪_(2017年电影)|乘风破浪]]》上映，片中饰演反派地产商黄志强，并凭借该角色入围第20届[[上海电影节|上海电影节]]亚洲新人奖最佳男演员；2月25日，首次登上[[红磡体育馆|红磡体育馆]]开唱，[[容祖儿|容祖儿]]担任表演嘉宾；6月24日，受邀担任第28届金曲奖颁奖典礼演出嘉宾，致敬两岸三地摇滚经典；9月16日，首次在[[台北小巨蛋|台北小巨蛋]]开唱，[[楊丞琳|楊丞琳]]担任嘉宾，並首次公開承認其女友身份；11月17日，发行第四张专辑《嗯》，數位单曲在各音樂平台创下10亿次点击量；12月9日，在台北举办两场新歌演唱会。

2018年1月12日，主演电影《卧底巨星》上映，在片中饰演侦探“铁柱”，正式進軍演藝界；1月19日，加盟综艺《偶像练习生》 ，担任常驻音乐导师；4月21日，加盟综艺《无限歌谣季》，担任“无限唱作人”；6月3日，发行为电影《动物世界》创作的片尾曲《成长之重量》。10月17日，发行第五张专辑《耳朵》,多首歌曲占据了海内外多有华语排行榜的冠军宝座。此专辑也同时获得豆瓣2018年度音乐榜单的“年度最受关注专辑”。

2019年3月16日，李榮浩「年少有為」世界巡迴演唱會將於上海正式開跑，這也是繼「天生 李榮浩 亞洲巡迴演唱會」和「李榮浩 有理想 世界巡迴演唱會」之後，職業生涯的第三輪巡演，也是第五張專輯發行後眾多新歌首秀舞台，當然，前幾張專輯的經典曲目也將進行重新編曲再演繹，爭取讓歌迷們享受到一次全新的音樂之旅。
