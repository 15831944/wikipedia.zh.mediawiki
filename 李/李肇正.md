'''李肇正'''（{{bd|1954年||2003年}}），[[上海市|上海]][[嘉定区|嘉定]]人，[[中共|中共]]党员，[[中国作家协会|中国作家协会]]会员，中学语文[[教师|教师]]、[[小说家|小说家]]。<ref>{{cite web|url=http://www.chinawriter.com.cn/zxhy/member/2546.shtml|title=李肇正 (1954～2003)|publisher=中国作家网}}</ref>

== 生平 ==
1977年，毕业于[[安徽省|安徽省]][[和县|和县]]师范学校，任安徽和县第一中学语文教师。1983年，毕业于[[安徽师范大学|安徽师范大学]]中文系，分配至安徽和县幼儿师范学校任教。1993年调至上海，历任上海[[徐汇区|徐汇区]][[上海市中国中学|中国中学]]语文教师、[[上海市位育中学|位育中学]]语文教师。2003年3月，因心脏病突发去世，终年49岁。<ref>{{cite news|url=http://www.sohu.com/a/191058004_182884|title=“永远不说再见”－纪念教师、作家李肇正|date=2017-09-10|work=《小说月报》|publisher=搜狐网}}</ref>

李肇正从20世纪80年代开始发表作品，创作了[[长篇小说|长篇小说]]2部、[[短篇小说|短篇小说]]15篇、[[中篇小说|中篇小说]]58篇，共达300多万字。<ref>{{cite news|url=http://www.chinawriter.com.cn/2005/2005-06-08/15304.html|title=上海文学界纪念李肇正|date=2005-06-08|author=徐颖|work=新闻晨报|publisher=中国作家网}}</ref><ref>{{cite news|url=http://ent.sina.com.cn/x/2005-06-14/0829751237.html|title=十作家代故人签售|date=2005-06-14|work=劳动报|publisher=新浪网}}</ref>

== 作品 ==
* 长篇小说：《无言的结局》（1997年）、《躁动的城市》（1999年）。
* 中篇小说：《女工》（1995年）、《头等大事》（1997年）、《商人》（1998年）、《城市生活》（1998年）、《扭曲》（1999年）、《亭子间里的小姐》（1999年）、《祖宗》（2002年）、《永远不说再见》（2002年）、《傻女香香》（2003年）、《风和月在上海流淌》（2003年）等。
* 短篇小说：《阿胡的故事》等。

== 奖励 ==
* 1997年 《女工》获《小说月报》第七届百花奖中篇小说奖
* 2003年 《永远不说再见》获《小说月报》第十届百花奖中篇小说奖<ref>{{cite news|url=http://ent.sina.com.cn/2003-11-11/0816231026.html|title=《小说月报》百花奖颁发|date=2003-11-11|author=陈熙涵|work=东方网-文汇报|publisher=新浪网}}</ref>
* 2005年 《傻女香香》获《小说月报》第十一届百花奖中篇小说奖
