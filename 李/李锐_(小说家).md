'''李锐'''（{{bd|1950年|catIdx=L李}}），当代[[小说家|小说家]]。生于[[北京|北京]]，祖籍[[四川|四川]][[自貢|自貢]]。作品《厚土》获得过第八届全国优秀短篇小说奖，第十二届[[台湾|台湾]]《中国时报》文学奖。李锐是海外知名中国作家，他的作品被翻译为瑞典文、英文、法文、日文、德文、荷兰文等多种文字出版。<ref>{{cite web |url=http://www.chinawriter.com.cn/fwzj/writer/33.shtml|title=《中国作家网：李锐》|accessdate=|work=|publisher=中国作家网|language= zh-hans}}</ref>

==生平 ==
1966年，李锐畢業於楊閘中學。1969年一月父母雙亡後到[[山西|山西]]呂梁山區[[插隊|插隊]]落戶，先後做過六年農民，兩年半工人。1974年發表第一篇小說。1977年調入《山西文學》編輯部。1984年畢業於[[遼寧大學|遼寧大學]]中文系函授部。1988年轉為山西省作家協會專業作家。

1998年12月，李锐被当选为山西作家协会副主席。2003年10月辞去山西作协副主席职务，同时退出[[中国作家协会|中国作家协会]]，并且放弃中国作协会员资格，只保留山西省作協會員的資格。<ref>{{cite web |url=http://cul.sohu.com/20061205/n246806527.shtml|title=《洪峰退出作协 李锐称是“作家自由”》|accessdate=2006年12月5日|work=潘妤|publisher=搜狐网|language= zh-hans}}</ref>

2004年，法国图书沙龙，李锐、[[莫言|莫言]]、[[余华|余华]]三个作家获得“法兰西艺术与文学骑士勋章”。<ref>{{cite web|url=http://book.sina.com.cn/news/c/2004-04-15/3/59839.shtml|title=《“法兰西骑士”归来 莫言、李锐畅谈法图书沙龙》|accessdate=2004年4月15日|work=术术|publisher=新浪网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20040420125456/http://book.sina.com.cn/news/c/2004-04-15/3/59839.shtml|archivedate=2004年4月20日}}</ref>

2007年12月12日，李锐被[[香港公开大学|香港公开大学]]授予文学荣誉博士学位，他也是继莫言后第二位获此荣誉的中國作家。<ref>{{cite web|url=http://www.sxgov.cn/xwzx/sxxw/sxyw/513273.shtml|title=《山西省作家李锐被授予香港公开大学荣誉博士学位》|accessdate=2007-12-14|work=山西晚报|publisher=黄河新闻网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160304200637/http://www.sxgov.cn/xwzx/sxxw/sxyw/513273.shtml|archivedate=2016-03-04}}</ref>

==作品==
1974年发表第一篇小说，小说集《合坟》获得第8届全国优秀短篇小说奖。主要著作有《厚土》，长篇小说《旧址》、《无风之树》、《万里无云》、《银城故事》，随笔集《拒绝合唱》、《不是因为自信》等。

==影响和荣誉==
他在[[中國|中國]]普通读者中影响不大，在中國大陸以外影响較大。作品曾获[[台湾|台湾]]第12届《中国时报》文学奖，《[[亚洲周刊|亚洲周刊]]》[[20世纪中文小说100强|20世纪中文小说100强]]中《旧址》排名第45位。作品被译为[[瑞典|瑞典]]、英、法、日、德、荷兰等多种文字。[[诺贝尔文学奖|诺贝尔文学奖]]评委瑞典汉学家[[马悦然|马悦然]]甚至认为，李锐和[[北岛_(诗人)|北岛]]是最有希望获得诺贝尔奖的两位中国作家。

瑞典诺贝尔文学奖终审评委马悦然十分推崇山西作家李锐和[[曹乃谦|曹乃谦]]的作品，因此许多媒体和记者认为他们将是诺贝尔文学奖中国候选人的热门人选。<ref>{{cite web |url=http://news.163.com/08/0909/21/4LE99LEM00011229.html|title=《诺贝尔奖评委马悦然称中国人不应翻译本国作品》|accessdate=2008-09-09 |work=|publisher=网易|language= zh-hans}}</ref>马悦然评价李锐说：“李锐有他特殊的经验，特殊的经历，‘文革’10年他从北京到山西吕梁山的邸加河村插队，有6年的时间跟农民生活在一起，这是他永远不会忘记的。‘文革’给他的经验是他一辈子写作的主题，他不能离开这个主题，这个是他的使命，他要把这个经验传达给下一代人、下下一代人，让他们知道‘文革’是怎么回事情。有一个例外就是他写的《旧址》，这是应该读的书。《旧址》写他的家族1925年到1950年的历史，他的家族是做盐商的，他有点像俄国作家的大气魄，写得非常好。”<ref>{{cite web |url=http://cul.sohu.com/20071010/n252580088.shtml|title=《沈从文如果活着就肯定能得诺贝尔文学奖》|accessdate=2007年10月10日16|work=|publisher=搜狐网|language= zh-hans}}</ref>
