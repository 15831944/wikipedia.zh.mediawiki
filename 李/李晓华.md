'''李晓华'''（{{bd|1951年|1月18日|catIdx=L李}}），生于[[北京|北京]]，中国富豪。[[福布斯中国富豪榜|福布斯中国富豪榜]]中1998年和1999年第2名，2000年第11名，2001年第16名。[[胡润百富榜|胡润百富榜]]中，1999年第4位，2003年第23位，2007年第328位。

== 生平 ==
1968年下放到[[黑龙江生产建设兵团|黑龙江生产建设兵团]]。1976年回北京当锅炉工人。

80年代初在[[北戴河|北戴河]]卖饮料、开录像厅开始发迹。1985年到[[日本|日本]]通过代理[[章光101毛发再生精|章光101毛发再生精]]获得巨额财富。1988年在[[香港|香港]]创立[[华达国际投资集团公司|华达国际投资集团公司]]任董事局主席。

1994和1995年，被授予“全国百名优秀企业家”和“中国十佳优秀民营企业家”称号。1998年当选为[[全国政协委员|全国政协委员]]<ref>[http://business.sohu.com/90/53/article212315390.shtml 李晓华简历，搜狐财经频道]</ref><ref>[http://youth.cuc.edu.cn/html/gzdt/192131161.htm  商界巨子李晓华，传媒青年网] {{webarchive|url=https://web.archive.org/web/20071031084319/http://youth.cuc.edu.cn/html/gzdt/192131161.htm |date=2007-10-31 }}</ref>

== 家庭 ==
李晓华和[[李晓林|李晓林]]是兄弟。

女儿李响

== 影响 ==
1996年3月1日，由[[中国科学院|中国科学院]][[紫金山天文台|紫金山天文台]]发现并提名，[[国际小行星中心|国际小行星中心]]和[[国际小行星命名委员会|国际小行星命名委员会]]审议通过，“鉴于李晓华为世界慈善事业作出的巨大贡献”，将一颗国际永久编号为3556号，将国际编号3556号小行星命名为“李晓华星”。

1992年李晓华成为中国第一个拥有[[法拉利|法拉利]]轿车的人，车牌号是京A00001，他和那辆红色法拉利出现在天安门广场的照片曾经被悬挂在[[法国|法国]][[巴黎机场|巴黎机场]]和[[里昂广场|里昂广场]]，照片下面写着“一个来自红色资本主义的挑战者”。

2007年日本[[NHK|NHK]]纪录片《[[激流中国|激流中国]]·富人与农民工》中，李晓华作为主角之一，片中有他的生日宴会（出席者有全国政协副主席[[周铁农|周铁农]]等）、价值数百万元的新轿车和数千万元的豪华别墅、操纵股市获利等情节。李晓华再一次引起世人关注。
