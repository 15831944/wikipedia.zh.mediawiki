'''李兰茂'''（{{bd|1927年||2008年|8月19日}}）今[[河北省|河北省]][[任丘市|任丘市]]人，[[中国人民解放军|中国人民解放军]]中校，[[福州军区空军|福州军区空军]]副司令员<ref name=xww>{{Cite web |url = http://cpc.people.com.cn/GB/64093/87393/8389706.html |title =原福州军区空军副司令员李兰茂同志逝世  |author =  |date =  2008-11-23|publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref><ref name=hx>{{Cite web |url = http://www.huaxia.com/zt/rdzz/05-012/526253.html |title =  李兰茂(1927？ )|author =  |date = 2001-06-14 |publisher = 华夏经纬网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1945年加入[[中国共产党|中国共产党]]。1947年参加[[中国人民解放军|中国人民解放军]]。历任班长、文书、书记、宣传员。参加了[[保北战役|保北战役]]、[[平保战役|平保战役]]、[[平津战役|平津战役]]等战役战斗。1951年毕业于空军航空学校。后任空军飞行中队长。1952年参加[[抗美援朝战争|抗美援朝战争]]，任[[中国人民志愿军|中国人民志愿军]]空军副大队长、大队长，先后击落敌机四架，击伤一架，荣立一等功。1958年自[[苏联|苏联]]红旗空军学院毕业。此后历任空军团长、副师长、师长、[[中国人民解放军空军漳州指挥所|空军漳州指挥所]]主任、[[福州军区空军|福州军区空军]]副司令员<ref name=xww/><ref name=hx/>。

李兰茂1955年被授予大尉军衔，1964年晋升为中校军衔，曾获[[解放奖章|解放奖章]]、[[独立功勋荣誉章|独立功勋荣誉章]]<ref name=xww/>。

2008年8月19日，副兵团职离休干部李兰茂在[[济南|济南]]病逝，享年81岁<ref name=xww/>。
