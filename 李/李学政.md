'''李学政'''（1965年8月6日-）[[山东|山东]][[菏泽|菏泽]]人，[[中央军事委员会后勤保障部金盾影视中心|中央军委后勤保障部金盾影视中心]]主任。

==生平==
李学政是今菏泽开发区佃户屯人。1994年到1997年，在菏泽人民广播电台（现[[牡丹区|牡丹区]]人民广播电台）担任主持人。后到[[北京|北京]]发展，先后在多家媒体任职，最后进入中国人民解放军总后勤部金盾影视中心（2016年改为[[中央军事委员会后勤保障部金盾影视中心|中央军委后勤保障部金盾影视中心]]），任该中心成立后的第一任主任。此外，作为金盾影视中心主任的李学政还和[[莫言|莫言]]、[[周梅森|周梅森]]应邀担任最高人民检察院影视中心艺术总监和顾问<ref>{{Cite web |url =http://yule.sohu.com/20160511/n448817851.shtml  |title = 金盾影视中心助力《人民的名义》 |author =  |date = 2016-05-11 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.hezefc.com/news/26378.html  |title =《人民的名义》总监制总发行人 李学政先生做客菏泽文化城  |author =  |date = 2017-04-30 |publisher = 菏泽房产网 |language =  |accessdate =  |quote =  }}</ref>。
