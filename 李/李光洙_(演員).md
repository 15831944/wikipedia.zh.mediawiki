'''李光洙'''<ref>{{cite web | url=http://tenasia.hankyung.com/archives/3393 | title=광수│My name is… | publisher=10asia | date=2010-02-15 | accessdate=2013-07-08 | author=편집부 | language=ko}}</ref>（{{kor|k='''이광수'''|rm='''Lee Kwang Soo'''|hanja='''李光洙'''}}，{{bd|1985年|7月14日}}），[[韓國|韓國]]著名[[男演員|男演員]]、[[主持|主持]]和[[模特兒|模特兒]]，身高190公分。

曾是模特兒，於2008年以《KTF Show－二十歲篇》廣告出道；2009年拍攝[[文化廣播_(韓國)|MBC]][[MBC日日連續劇|日日劇]]《[[穿透屋頂的High_Kick|穿透屋頂的High Kick]]》後開始為人熟悉；2010年出演[[SBS株式會社|SBS]]綜藝節目《[[Running_Man|Running Man]]》後晉身為綜藝新星，在節目海外拍攝時獲得壓倒性人氣，被冠以「亞洲王子」的稱號<ref>{{cite web | url=http://hk.ent.yahoo.com/news/%E6%9D%8E%E5%85%89%E6%B4%99%E6%A6%AE%E7%99%BB-%E4%BA%9E%E6%B4%B2%E5%B0%8F%E7%8E%8B%E5%AD%90-%E4%BB%96%E9%AB%98%E6%BC%B2%E4%BA%BA%E6%B0%A3%E7%9A%84%E7%A7%98%E8%A8%A3%E6%98%AF-042812174.html | title=李光洙榮登 「亞洲王子」 他高漲人氣的秘訣是？ | publisher=Yahoo! 香港娛樂新聞 | date=2013-02-26 | accessdate=2013-07-08 | author=南又禎 (STAR N) | language=zh-hk}}</ref>，於電影界亦成為崛起的新勢力。现任女友为演员[[李善彬|李善彬]]<ref>[https://star.ettoday.net/news/1344481 李光洙認愛李善彬！ 官方證實：已經5個月了]</ref>。

==個人經歷==
===入行經過===
在高中一年級時和模特經紀公司簽約，開始參加時裝秀，後來進入東亞放送大學放送演藝科就學，漸漸對演技産生興趣，開始循此領域發展<ref name="multiple">{{cite web|title="우리 광수~　인터뷰했네?"|url=http://bookdb.co.kr/bdb/Interview.do?_method=InterviewDetail&sc.mreviewNo=19237|publisher=디시인사이드|author=권지현|date=2008-11-12|accessdate=2015-02-21|language=ko}}</ref>。大學期間，曾在一個兒童劇場工作，可是兒童劇的反響不佳，於是休學去服兵役。於2007年1月退伍，是陸軍上等兵，成為少數出道前已完成兵役的藝人<ref>{{cite web|title=한상의 세트 플레이 '좋은친구들' 지성.주지훈.이광수| url=http://gall.dcinside.com/board/view/?id=kwang&no=18364
|publisher=매거진M|author=김나헌|date=2014-07-11|accessdate=2015-02-21|language=ko}}</ref>。退伍後，決定從事演藝工作，可惜當時還沒有簽約經紀公司，於是拜托中介公司，最終於2008年透過廣告《KTF Show－二十歲篇》正式出道<ref name="multiple" />。2009年出演日日劇《[[穿透屋頂的High_Kick|穿透屋頂的High Kick]]》後開始為人熟悉。2010年成為[[SBS|SBS]]綜藝節目《[[Running_Man|Running Man]]》的主持人，節目在亞洲各地取景時，均受到粉絲們的熱烈歡迎，被冠以「亞洲王子」的稱號。雖然他本身是《[[Running_Man|Running Man]]》的固定班底，但在韓國或是海外節目都鮮少出現他的蹤影，曾被多位媒體透露綜藝感極佳的他被多個節目邀請擔任嘉賓或接下主持棒均被他拒絕<ref>[https://www.koreastardaily.com/tc/news/98324 推理綜藝《犯人就是你》陣容確定！劉在錫、李光洙、朴敏英、SUHO、金世正等7人出演！]</ref>。

===演藝經歷===
2008年以《KTF Show－二十歲篇》廣告出道 。並靠該廣告獲得2008 Mnet 20's Choice頒獎禮CF新星獎 和2009年亞洲模特頒獎典禮CF模特獎。
2009年拍攝MBC日日劇《穿透屋頂的High Kick》 後開始為人熟悉。
2010年固定出演了SBS週日綜藝《Running Man》。年末獲得SBS演藝大賞New Star賞。同年出演MBC電視劇《同伊》中的英達一角  。
2010年主演李俊益導演的《平壤城》  、《我妻子的一切》、《完美廣播》  、《我的小英雄》等電影作品。憑藉《我妻子的一切》獲得第33屆青龍電影節的最佳新人男演員提名。同樣《完美廣播》中的演出也獲得第48屆百想藝術大賞新人男演員提名 。

2011年出演《城市獵人》 中高奇俊一角。出演了電視劇《小伙子的蔬菜店》中南友鋒一角 。
2012年出演了KBS電視劇《善良的男人》，飾演樸載吉 ，還為動畫電影《海洋警察Marco》,《與恐龍同行3D》配音 。
2013年出演MBC電視劇《火之女神井兒》，飾演臨海君 。
2014年出演SBS水木劇《沒關係，是愛情啊》 ，飾演樸秀光。
2014年出演電影《好朋友們2014》 中民秀一角。並出演電影《德水里五兄弟》 中警察一角。
2014年8月，憑藉電視劇《沒關係，是愛情啊》中妥瑞症患者的演技入圍2014年韓國電視節男子優秀賞 。
10月21日成為2014年韓國-印尼電影節的宣傳大使 。 ,11月15日，在2014大田電視劇節獲得男子人氣賞 。 
2014年11月確定作為主演出演電影《陡然變異》 。
12月30日，獲得SBS演藝大賞娛樂部門優秀賞 ，12月31日獲得SBS演技大賞迷你劇部門特別演技賞。

2015年，確定出演中國明星假想戀愛真人秀《如果愛》 [42] 該節目將於7月2日播出。 7月16日，被委任為“2015韓國品牌和韓流商品博覽會”（KBEE）宣傳大使  。出演SBS電視台單幕劇《PUCK》，飾演完全沒有人情味的高利貸業主趙俊萬 。
2016年1月7日，榮獲“2015微博之夜”亞洲人氣藝人獎  。客串電視劇《花郎：The Beginning》 。 4月初拍攝韓國KBS電視台電視劇《心裡的聲音》 。客串TVn十週年特別企劃電視劇《My Dear Friends》飾演小兒子民浩。 11月4日，主演tvN喜劇《明星夥伴》（又名《Entourage》）飾演過氣演員車俊。 12月9日《心裡的聲音》首播。

2017年，與鄭有美、裴晟祐共演展現派出所警察生活的劇集《Live》 ，同時與成東日，權相宇出演電影《偵探2》 。 12月30日獲得SBS演藝大賞best couple 賞。
2018年參與錄製綜藝節目《犯人就是你》 ，與權相佑、成東日參與拍攝電影《偵探2》於6月13日上映。

===公眾評價===
雖然在Running man中背叛者、搞笑的綜藝形象深入民心，但現實中的他其實是個心地善良，沉穩內斂的人。經常被拍到在綜藝節目，片場和其他場合中替其他演員嘉賓披毛毯、拿重物等暖心小舉動。2014年[[tvN|tvN]]綜藝節目《[[名單公開|名單公開]]》公佈了李光洙不為人知的善行。2009年在[[京畿道|京畿道]][[楊平郡|楊平郡]]的一家超市外，遇到一對流著血的老夫婦和一名喝醉的中年男子，李光洙果斷報警，並積極配合警方工作進行事件相關的陳述。一年後，還為了這對老夫婦親自前往議政府法院作證。<ref>{{cite web | url=http://www.koreastardaily.com/tc/news/53757| title=《名單公開》李光洙於2009年救助被襲擊的老夫婦的善行得以公開| publisher=韓星網| date=2014-12-30| accessdate=2015-02-21 | author=希比| language=zh}}</ref>
[[李光洙|李光洙]]曾为帮助经济困难的儿童患者,非常低调地捐款1亿韩元。
在演藝圈中,李光洙是大家公認的「紳士男」，例如 [[金鐘國|金鐘國]]、[[李侑菲|李侑菲]]及[[鲁芬|鲁芬]]都曾經評價光洙是他們所見過的最善良謙虛的人。[[趙寅成|趙寅成]]說過像光洙這種朋友很稀少，要向他學習的地方很多。[[宋仲基|宋仲基]]曾表示光洙很有義氣，不管多累也會幫助朋友，也會處處體諒他人，為別人着想。[[朴寶英|朴寶英]]說他非常善良，即使痛苦時仍強顏歡笑不想別人擔心。[[朱智勳|朱智勳]]提到有一次拍戲的時候發現光洙光腳踩著滿地的玻璃渣，腳都出血了，但他為了不想拖累前輩NG，不顧疼痛繼續演戲，很讓人感動。很多合作過的拍檔如[[池城|池城]]、[[李聖經|李聖經]]、[[郑妮可|郑妮可]]、[[成東日|成東日]]、[[朴正民|朴正民]]等人都曾公開讚賞他的人品和對演戲的認真態度。

===家庭===
李光洙出自富裕的家庭，父亲李宗浩是韩国知名连锁家具品牌会长在韩国拥有超过15间大型工厂其规模不小，母亲则是一名家庭主妇，李光洙家中排行老大有一个小三岁的妹妹李智慧。
