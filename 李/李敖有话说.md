《'''李敖有话说'''》，由[[香港|香港]][[凤凰卫视中文台|凤凰卫视中文台]]制作的节目，主持人是著名[[文学家|文学家]]、[[历史学家|历史学家]][[李敖|李敖]]，主要内容涉及[[文化|文化]]、[[政治|政治]]、[[军事|军事]]和[[时事|时事]]，主轴为[[中华文化|中华文化]]。李敖的演播风格与学术性的理论研究相比较为平易，同时亦追求内容的学理性与权威性，力求雅俗共赏。

== 播出时间 ==
《李敖有话说》在2004年3月8日开始播出，2006年12月31日停止播出，播出时间为每星期一到星期五16点45分。2011年9月18日，李敖在鳳凰衛視中文台節目《[[文化大觀園|文化大觀園]]》说，他离开鳳凰衛視的原因是他第六次开始骂[[连战|连战]]的时候被封杀了，另一个原因是他骂连战使得[[刘长乐|刘长乐]]受到了某种程度的困扰，他觉得有愧于朋友，他离开而不破坏友情。<ref>{{cite web |url=http://phtv.ifeng.com/program/whdgy/detail_2011_09/19/9295312_0.shtml |title=李敖：告别《李敖有话说》 因我头上有把刀 |accessdate=2011年9月19日|work= |publisher=凤凰网 |language =zh-cn}}</ref>李敖表示，“决定在我的余生里面，再次维护[[中华文化|中华文化]]的尊严，去整理一些中国好的文字方面的东西”。

== 节目主持 ==
该栏目的节目主持人是著名文学家、历史学家李敖。李敖自谓：“一手包办、一言九鼎、一针见血，打破电视制作模式，以证据骂人、以口舌开心，闯出一片言论新天地”。<ref>{{cite web |url=http://media.people.com.cn/GB/22114/70684/75199/5110024.html |title=《李敖有话说》： 嘻笑怒骂，自成一格 |accessdate= 2006年11月30日|work= |publisher=人民网 |language =zh-cn}}</ref>

== 事件 ==
2004年，李敖在《李敖有話說》大談[[六四天安門事件|六四天安門事件]]，鳳凰衛視的[[編審|編審]]要求刪除；結果李敖[[一不做、二不休|一不做、二不休]]，連罵六集，從[[鄧小平|鄧小平]]到劉長樂全被他罵；結果劉長樂見狀，忍著氣下令「播了，不刪也不剪」，當日鳳凰衛視在最大的落地區[[廣東省|廣東省]]全天被[[蓋台|蓋台]]。劉長樂請[[陳文茜|陳文茜]]勸勸李敖，但陳文茜沒轉一個字、只把結果真相告訴李敖；第二天，《李敖有話說》錄影，李敖拐個彎，不談六四天安門事件，只談「[[毛語錄|毛語錄]]與[[毛主席|毛主席]]為什麼不一樣」。2005年9月24日，陳文茜說，這次事件為了鳳凰衛視在大陸難得的空間，也為了李敖日後在大陸的[[言論自由|言論自由]]，李敖「從獅子變成狐狸」<ref>{{cite web|author=陳文茜|url=http://www.appledaily.com.tw/appledaily/article/headline/20050924/2078262|title=大師變朋友|publisher=[[台灣蘋果日報|台灣蘋果日報]]|date=2005-09-24|accessdate=2015-01-04}}</ref>。
