'''李奧·沙普'''（{{lang-en|'''Leo Sharp'''}}，原名'''Leo Earl Sharp Sr.'''，{{bd|1924年|5月7日|2016年|12月12日}}），又名'''El Tata'''<ref>{{Cite web |url = https://www.nytimes.com/2014/06/15/magazine/the-sinaloa-cartels-90-year-old-drug-mule.html |title = There's a True Story Behind ‘The Mule’: The Sinaloa Cartel's 90-Year-Old Drug Mule |author = Sam Dolnick |date = 2014-06-11 |publisher = The New York Times |language = en}}</ref><ref>{{Cite web |url = https://www.businessinsider.com/the-story-of-detroit-urban-legend-leo-sharp-2014-6 |title = This 90-Year-Old Florist Fueled The Biggest Cocaine Smuggling Operation In Detroit's History |author = Michael B Kelley |date = 2014-06-11 |publisher = Business Insider |language = en}}</ref><ref>{{Cite web |url = https://www.thefix.com/content/ninety-year-old-leo-sharp-was-el-chapos-top-drug-mule |title = Ninety-Year-Old Leo Sharp Was El Chapo's Top Drug Mule |author = Paul Gaita |publisher = The Fix |language = en}}</ref>，是一名[[第二次世界大戰|第二次世界大戰]]的老兵、世界著名的[[園藝|園藝家]]<ref>{{Cite web |url = https://www.vice.com/en_us/article/znw3ax/a-drug-mule-might-kill-himself-on-his-90th-birthday |title = An 89-Year-Old Drug Mule Is Threatening to Kill Himself Rather Than Face Jail Time |author = Mike Pearl |date = 2014-05-02 |publisher = VICE |language = en}}</ref>，以及[[錫納羅亞販毒集團|錫納羅亞販毒集團]]分支的運毒販。<ref>{{Cite web |url = https://www.upi.com/Odd_News/2014/04/30/89-year-old-WWII-veteran-Leo-Sharp-hoping-to-avoid-jail-for-huge-cocaine-haul/7571398876809/ |title = 89-year-old WWII veteran Leo Sharp hoping to avoid jail for huge cocaine haul |author = Evan Bleier |date = 2014-04-30 |publisher = UPI.com |language = en}}</ref>

[[美國緝毒局|美國緝毒局]]特別探員Jeff Moore於2011年逮捕了沙普，並接受了《[[紐約時報|紐約時報]]》作家Sam Dolnick採訪關於世界上最老的毒驢（drug mule）的調查。

== 早年 ==
沙普出生於[[印第安納州|印第安納州]]的{{link-en|密芝根市|Michigan City, Indiana}}，並且在密芝根州的[[底特律|底特律]]長大。

== 服兵役和榮譽 ==
沙普打過[[第二次世界大戰|第二次世界大戰]]中的[[意大利戰役_(第二次世界大戰)|意大利戰役]]，而且因其服役獲得[[銅星勳章|銅星勳章]]。

== 逝世 ==
沙普在2016年12月因自然去世，享年92歲。他被埋葬在檀香山的{{link-en|太平洋國家紀念公墓|National Memorial Cemetery of the Pacific}}。

== 流行文化 ==
2018年電影《[[毒行俠|毒行俠]]》改編自李奧·沙普的生活，由[[奇連·伊士活|奇連·伊士活]]飾演Earl Stone（這角色基於沙普），以及[[畢列·谷巴|畢列·谷巴]]飾演Colin Bates（這角色基於[[美國緝毒局|美國緝毒局]]探員Jeff Moore）。<ref>{{Cite web |url = http://www.historyvshollywood.com/reelfaces/the-mule/ |title = THE MULE (2018) |publisher = HistoryvsHollywood |language = en}}</ref><ref>{{Cite web |url = http://www.tracking-board.com/clint-eastwood-circling-90-year-old-drug-courier-tale-the-mule-at-imperative-entertainment-exclusive/ |title = CLINT EASTWOOD CIRCLING 90-YEAR-OLD DRUG COURIER TALE “THE MULE” AT WARNER BROS., IMPERATIVE (EXCLUSIVE) |author = Jeff Sneider |date = 2018-01-30 |publisher = The Tracking Board |language = en}}</ref>
