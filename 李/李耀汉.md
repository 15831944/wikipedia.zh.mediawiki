'''李耀汉'''（{{bd|1878年||1942年|}}）[[字|字]]'''子雲'''，[[广東省|广東省]][[肇慶府|肇慶府]][[新興县|新興县]]人，[[中華民国|中華民国]]军事将领。[[護法战争|護法战争]]重要人物。<ref name=yi>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref name=er>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

==生平==
李耀汉早年出身[[緑林|緑林]]，清末投降官軍，任新興县的巡防营管带。[[辛亥革命|辛亥革命]]之際，他响应革命派。1914年（[[民国紀元|民国]]3年）3月，就任[[广東省_(中華民国)|广東]]肇陽羅鎮守使。後来，他任[[粤軍|粤軍]]第1師師長。<ref name=yi/><ref name=er/>

1917年（民国6年）8月，[[北京政府|北京政府]]的[[馮国璋|馮国璋]]任命他署理广東省省長。同年9月，广州非常国会选举他任籌餉督办。同年10月，他一度兼署理广東督軍。1918年（民国9月）9月，广州軍政府罢免其各职。1923年（民国12年），他任[[旧桂系|旧桂系]]的[[沈鴻英|沈鴻英]]手下的第5軍軍長，沈鴻英叛离[[孫文|孫文]]后在[[開平市|開平]]交战败北。此後，李耀漢率軍离开沈鸿英手下，在外流浪，后以故乡[[新興县|新興县]]为中心进行武装割据。<ref name=yi/><ref name=er/>

1943年（民国31年），李耀汉在新興县逝世。享年66岁。<ref name=yi/><ref name=er/>
