'''李周憲'''<ref>{{Cite web |url =http://ww1.sinaimg.cn/mw690/73217002jw1f5if9t196uj20sg0zv10k.jpg |title =MONSTA X 全員確認後的正名 | date = 2016-07-05}}</ref><ref>{{Cite web |url =http://www.weibo.com/1931571202/DDjAI4YdQ?type=comment#_rnd1481452395256 | title =MONSTA X 全員確認後的正名:詳細全文 | date = 2016-07-05}}</ref>（{{kor|k='''이주헌'''|rm=}}），1994年10月6日出生，藝名為'''JooHeon'''（{{kor|k='''주헌'''|rm=}}），是一名韓國男歌手、舞者。為[[Monsta_X|Monsta X]]七位成員之一，擔任主Rapper、形象擔當。'''NUBOYZ''' (Starship推出的小團體) 成員<ref>{{Cite web |url =https://www.last.fm/music/NUBOYZ |title =NUBOYZ | work = last.fm}}</ref>。

2015年自創撒嬌方式「咕咕嘎嘎」「噠蹦」，擁有很大的反轉魅力。

== 经历 ==

=== 出道前 ===
2014年11月参与[[STARSHIP娛樂|STARSHIP娱乐]]与有线电视[[Mnet|Mnet]]新偶像团真人选秀节目《[[NO.MERCY|NO.MERCY]]》，于2015年2月顺利获选，以[[Monsta_X|MONSTA X]]成功出道。
