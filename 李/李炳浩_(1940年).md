'''李炳浩'''（{{lang-ko|이병호}}，{{bd|1940年|10月4日}}）[[韓国|韓国]]政府官员，国家情报院院长。

== 学历 ==
*1959年，首尔景福高等學校毕业
*1963年，[[韓國陸軍官校|陆军官校]]19期文学士毕业
*1970年，陆军步兵学校毕业
*1972年，陆军装甲学校毕业
*1987年，美国[[乔治城大学|乔治城大学]]外交和行政学硕士毕业

== 经历 ==
* [[1967年|1967年]] [[越南战争|越南战争]]一年后参战。
* [[1968年|1968年]] 回国。
* [[1982年|1982年]] 陆军上校退役。
* [[1988年|1988年]] ~ [[1990年|1990年]] : 国家安全企划部国际局长。
* [[1990年|1990年]] ~ [[1992年|1992年]] : [[韓國駐美國大使館|驻美国大使館]]公使。
* [[1993年|1993年]] ~ [[1997年|1997年]] : 国家安全企划部第2次长。
* [[1997年|1997年]] ~ [[2000年|2000年]] : 驻[[马来西亚|马来西亚]]大使。
* [[2000年|2000年]] ~ [[2003年|2003年]] : 外交通商部[[無任所大使|本部大使]]。
* [[2003年|2003年]] ~ [[2015年|2015年]] : [[蔚山大学|蔚山大学]]国际学院聘任教授。
* [[2015年|2015年]] ~ [[2017年|2017年]] : 第12代[[大韓民国国家情報院|国家情報院長]]。
