'''李程'''（{{bd|766年||842年|}}？），字'''表臣'''，封'''彭原郡公'''，谥'''缪'''，[[唐朝|唐朝]]官员，[[唐敬宗|唐敬宗]]年间拜为[[宰相|宰相]]。

== 背景 ==
李程可能生于[[大历|大历]]元年（766年）。他是唐朝开国之君[[唐高祖|唐高祖]]堂弟襄邑王[[李神符|李神符]]的五世孙。<ref name=NBT131>[[:s:新唐書/卷131|《新唐书》卷一百三十一]]</ref>祖父[[李柏|李柏]]任太仆卿，父[[李鹔|李鹔]]，[[滁州|滁州]]刺史。<ref>《旧唐书》李程传没列出李程祖父的名字，称其父名[[李鷫伯|李鷫伯]]。见《旧唐书》卷一百六十七。《新唐书》宰相世系表提到了李程祖父的名字，进一步指出其父的名字是李鹔，对其父的名字含有和祖父李柏同音不同字的“伯”字，没有避讳，表示怀疑。见《新唐书》卷七十。{{cite web |url=http://www.sidneyluo.net/a/a17/070.htm |title=存档副本 |accessdate=2011-10-25 |0=2012-02-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20120218050017/http://www.sidneyluo.net/a/a17/070.htm |archivedate=2012-02-18 }}{{cite web |url=http://www.sidneyluo.net/a/a17/table/form15.htm |title=存档副本 |accessdate=2009-10-12 |0=2010-06-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20100620182703/http://sidneyluo.net/a/a17/table/form15.htm |archivedate=2010-06-20 }}</ref>

[[贞元_(唐朝)|贞元]]十二年（796年），李程中[[进士|进士]]，又通过宏辞科考试，作赋《日五色》，以语言警策拔俗为士人所推，后担任一些节度使的幕僚。调蓝田尉，独自宣判了积压十年的案子。二十年（804年）因京兆府考功第一，回京城[[长安|长安]]任监察御史，五月以承务郎监察御史里行身份参与祭奠御史中丞[[李汶|李汶]]，<ref>[[:s:祭李中丞文|柳宗元《祭李中丞文》]]</ref>十一月又奉召充[[翰林学士|翰林学士]]。<ref>[[:s:旧唐书/卷13|《旧唐书》卷一十三]]</ref>唐德宗季秋出猎，觉得冷，看着左右说：“九月穿衫，二月穿袍，不是顺时。朕欲改月，怎么样？”左右称好，唯独李程说：“[[唐玄宗|玄宗]]著《月令》，十月才穿裘，不可改。”德宗急忙停止。学士入官署，常观察日影判断时间，李程性懒，太阳过了八砖才到，时人号“八砖学士”。<ref name=NBT131/>二十一年（805年），德宗崩，子[[唐顺宗|唐顺宗]]继位，李程被顺宗信任的同僚[[王叔文|王叔文]]排挤出翰林院，三度迁为司勋员外郎。<ref name=BT167>[[:s:舊唐書/卷167|《旧唐书》卷一百六十七]]</ref>宦官[[俱文珍|刘贞亮]]等奏请立皇长子广陵王[[唐宪宗|李淳]]为[[皇太子|皇太子]]，勾当军国大事，顺宗同意，刘贞亮便召学士[[卫次公|卫次公]]、[[郑絪|郑絪]]、李程、[[王涯|王涯]]入金銮殿，草立储君诏。<ref>[[:s:旧唐书/卷184|《旧唐书》卷一百八十四]]</ref><ref>[[:s:新唐书/卷207|《新唐书》卷二百零七]]</ref><ref>[[:s:資治通鑑/卷236|《资治通鉴》卷二百三十六]]</ref>李淳即[[唐宪宗|唐宪宗]]。

== 唐宪宗、唐穆宗年间 ==
唐宪宗[[元和_(唐朝)|元和]]三年（808年）七月，知制诰，出为[[隋州|隋州]]刺史，<ref>《重修承旨学士壁记》</ref>因能治理，赐金紫服。五年后，除郎中。<ref>[[:s:除李程郎中制|白居易《除李程郎中制》]]</ref>元和中期，被[[西川节度使|西川节度使]][[李夷简|李夷简]]（813年—818年在任）辟为成都少尹、御史中丞、节度行军司马。<ref>[[:s:李程行军司马制|白居易《李程行军司马制》]]</ref>十年（815年）三月，被召回长安任兵部郎中，知制诰。<ref>[[:s:旧唐书/卷15|《旧唐书》卷一十五]]</ref>同年，宪宗伐自称[[淮西节度使|淮西节度使]]的军阀[[吴元济|吴元济]]，以[[宣武节度使|宣武节度使]][[韩弘|韩弘]]为淮西都统，<ref>[[:s:資治通鑑/卷239|《资治通鉴》卷二百三十九]]</ref>李程被派往[[汴州|汴州]]前线宣赐官告。<ref>[[:s:旧唐书/卷156|《旧唐书》卷一百五十六]]</ref>十一年（816年），他被任为[[中书舍人|中书舍人]]，不早于当年十一月权知京兆尹事。<ref>当年十一月京兆尹为[[柳公绰|柳公绰]]。</ref>十二年（817年），权知礼部贡举。同年，监察御史[[韦楚材|韦楚材]]被贬，因他与宰相[[裴度|裴度]]交好而裴度又与同僚宰相[[李逢吉|李逢吉]]不合，宪宗召给事中[[张贾|张贾]]及李程、御史中丞[[韦乾度|韦-{乾}-度]]、比部郎中兼侍御史知杂[[宋景|宋景]]及韦楚材、御史[[崔鄯|崔鄯]]讯问。十三年（818年）四月，李程拜礼部侍郎，左仆射[[郑余庆|郑余庆]]受任为礼乐详定使，奏刑部侍郎[[韩愈|韩愈]]与李程为副使。<ref>[[:s:旧唐书/卷158|《旧唐书》卷一百五十八]]</ref><ref>[[:s:新唐书/卷165|《新唐书》卷一百六十五]]</ref>六月，李程出为[[鄂州|鄂州]]刺史、[[鄂岳观察使|鄂岳观察使]]。<ref name=NBT131/><ref name=BT167/>十四年（819年）二月，[[白居易|白居易]]作诗《行次夏口先寄李大夫》。当年[[刘禹锡|刘禹锡]]也曾作《[[:s:为鄂州李大夫祭柳员外文|为鄂州李大夫祭柳员外文]]》祭奠[[柳宗元|柳宗元]]。十五年（820年）韩愈作《除官赴阙至江州寄鄂岳李大夫》。三月，[[唐穆宗|唐穆宗]]从李程所请，罢中州岁贡茶，减轻人力劳损。<ref name=CFYG>《册府元龟》</ref>

== 唐敬宗年间 ==
[[长庆|长庆]]元年（821年），刘禹锡又作《鄂渚留别李二十一表臣大夫》。二年十二月末，李程又被召回任吏部侍郎，<ref>据[[严耕望|严耕望]]《仆尚丞郎表》，且当月[[崔元略|崔元略]]已被任为新的鄂岳观察使。</ref>封'''渭源县男'''，食邑三百户。<ref name=NBT131/><ref name=BT167/>

四年（824年），唐穆宗崩，正月，其子唐敬宗继位不久，五月，李程以正议大夫、尚书吏部侍郎、[[上柱国|上柱国]]、渭源县开国男、食邑三百户、赐紫金鱼袋守本官，授[[同中书门下平章事|同中书门下平章事]]，为[[实质宰相|实质宰相]]。<ref name=NBT8>[[:s:新唐书/卷8|《新唐书》卷八]]</ref><ref>[[:s:授李程平章事制|:s:授李程平章事制]]</ref><ref>《册府元龟》作正月敬宗即位后事。</ref>当时，唐敬宗问李逢吉谁堪为宰相，李逢吉列举了一批官员，将李程列在首位，故被拜为宰相。当时敬宗年轻，想建造和重建宫室。李程指出敬宗新立，尚在穆宗丧期，展开这些工程不妥。在他的建议下，敬宗收集来的木石被用来修皇陵。六月，李程还和[[韦处厚|韦处厚]]建议对因和李逢吉不和被逐出京的高官裴度加礼，在他们建议下，敬宗加裴度宰相荣衔[[同平章事|同平章事]]。<ref name=ZZTJ243>[[:s:資治通鑑/卷243|《资治通鉴》卷二百四十三]]</ref>李程还奏请设置侍讲学士。李程善辩且多智算，能改变敬宗的心思。不久拜[[中书侍郎|中书侍郎]]，十二月进爵'''彭原郡公'''，<ref name=BT17-1>[[:s:旧唐书/卷17上|《旧唐书》卷一十七上]]</ref>仍居相位。<ref name=NBT131/><ref name=BT167/>

[[宝历_(唐朝)|宝历]]元年（825年）四月，敬宗受尊号，大赦天下。李程摄中书令、读册官，获赐一子出身。<ref>[[:s:受尊号赦文|:s:受尊号赦文]]</ref>同年，宰相李逢吉、[[窦易直|窦易直]]、李程在延英殿谈及鄠县令[[崔发|崔发]]拖拽中人获罪之事，奏称崔发虽犯大不敬，但崔发母是已故宰相[[韦贯之|韦贯之]]的姐姐，年已八十，自崔发下狱以来积忧成疾，请求恩宥崔发。敬宗这才知崔发有老母，立即遣中使送崔发回家并抚问崔母韦夫人。<ref name=BT171>[[:s:旧唐书/卷171|《旧唐书》卷一百七十一]]</ref>十月，摄太尉，充使，于孟冬时节荐献大圣祖于太清宫。<ref name=CFYG/>

同年，[[陈留|陈留]]人[[武昭|武昭]]被控恫吓要杀李逢吉。武昭先前曾任[[石州_(北周)|石州]]刺史，想寻求更好的任命，却被授以权力少很多的敬宗叔祖父袁王[[李绅_(袁王)|李绅]]府长史。李程与武昭交好，想给他一个更有用的职位；但李逢吉此时已不复和李程为友，反对，李程的族人水部郎中[[李仍叔|李仍叔]]告知武昭。一次武昭喝醉了，对朋友左金吾兵曹[[茅汇|茅汇]]说要杀了李逢吉。九月，事发，武昭被拘。李逢吉党羽前[[河阳节度使|河阳]]掌书记[[李仲言|李仲言]]对茅汇说，如果供出李程和武昭同谋，可以得生，反之必死。茅汇拒绝。调查结束后，武昭被杖杀，李仍叔贬[[道州|道州]]司马，李仲言流放[[象州_(古代)|象州]]，茅汇流放[[崖州|崖州]]。<ref name=BT167/><ref name=ZZTJ243/><ref>[[:s:旧唐书/卷169|《旧唐书》卷一百六十九]]</ref><ref>[[:s:新唐书/卷174|《新唐书》卷一百七十四]]</ref>

二年（826年），李程召同年进士[[郑州_(古代)|郑州]]刺史[[张仲方|张仲方]]为右谏议大夫。<ref name=BT171/><ref>[[:s:新唐书/卷126|《新唐书》卷一百二十六]]</ref>二月，李程摄中书令受册，于正衙宣诏。<ref name=CFYG/>九月，李程被罢相，尽管仍保有同中书门下平章事为荣衔，出任检校兵部尚书、[[太原|太原]]尹、北京留守、[[河东节度使|河东节度使]]。<ref name=NBT131/><ref name=BT167/><ref name=NBT8/><ref name=ZZTJ243/><ref name=BT17-1/>任上，表侍御史[[卢简辞|卢简辞]]为节度判官。<ref>[[:s:新唐书/卷177|《新唐书》卷一百七十七]]</ref>

== 唐文宗年间 ==
敬宗弟[[唐文宗|唐文宗]][[太和_(唐朝)|太和]]三年（829年）六月，奏得自称[[成德节度使|成德节度使]]的[[王廷凑|王廷凑]]书信，请纳[[景州_(隋朝)|景州]]；又奏魏博叛将[[亓志绍|亓志绍]]自缢。<ref name=ZZTJ244>[[:s:資治通鑑/卷244|《资治通鉴》卷二百四十四]]</ref>四年（830年）三月，李程被任为检校尚书左仆射、兼[[蒲州_(北周)|河中]]尹、河中晋绛慈隰等州节度使，仍保有同中书门下平章事衔。六年（832年），加检校[[司空|司空]]荣衔。七月，召回长安任[[左仆射|左仆射]]。此前，文宗刚下令，仆射上任，所有朝臣正式欢迎，四品及以下官员要向他下拜，早先拜为仆射的王涯和窦易直都享受过了这种盛大的待遇。李程认为不妥，请求文宗重新考虑。八月，御史中丞[[李漢_(唐朝)|李汉]]也认为这个仪式太严肃，但文宗仍命下拜仪式如此进行。<ref name=BT171/><ref name=BT17-2>[[:s:旧唐书/卷17下|《旧唐书》卷一十七下]]</ref><ref>[[:s:旧唐书/卷157|《旧唐书》卷一百五十七]]</ref><ref>《新唐书》也记载了这场争议，但文宗、李程、李汉的立场和《旧唐书》所述相反。鉴于《旧唐书》引用了文宗诏书原文，此版本可能更可靠。可比较《旧唐书》卷一百六十七和《新唐书》卷一百三十一。</ref>李程能干博学，为文宗所礼遇，文宗说他“高飞之翮，长者在前。卿朝廷羽翮也”。但尽管身居高位，却落下了放荡、不拘小节、过分滑稽的名声。<ref name=NBT131/><ref name=BT167/>

七年（833年）七月，李程与御史大夫[[郑覃|郑覃]]奉命同去尚书省疏理诸司囚徒。当月，他又一次被遣出长安，任检校司空、兼[[汴州|汴州]]刺史、[[宣武军节度使|宣武军节度使]]。<ref name=ZZTJ244/>八月，兼充[[亳州|亳州]]太清宫使，修葺因水灾损毁的太清宫。<ref name=CFYG/><ref>[[:s:令宣武军节度使李程兼充亳州太清宫使诏|:s:令宣武军节度使李程兼充亳州太清宫使诏]]</ref>九年（835年）六月，再任河中晋绛节度使，加检校[[司徒|司徒]]。[[开成|开成]]元年（836年）闰五月，召回长安任[[右仆射|右仆射]]，兼判太常卿事。为河东节度使[[李载义|李载义]]作纪功碑文。<ref>[[:s:新唐书/卷212|《新唐书》卷二百一十二]]</ref>十一月，兼判吏部尚书铨事。二年（837年）二月（一作三月），授检校司徒，出为[[襄州_(西魏)|襄州]]刺史、[[山南东道节度使|山南东道节度使]]。<ref name=BT17-2/>[[李商隐|李商隐]]《樊南文集补编》有《上汉南李相公状》。三年（838年），[[张祜|张祜]]作《戊午年感事书怀二百韵谨寄献太原[[裴度|裴令公]]淮南[[李德裕|李相公]]汉南李仆射宣武[[李绅|李尚书]]》。[[唐武宗|唐武宗]]继位后，[[会昌_(唐朝)|会昌]]元年（841年）二月，赐[[开府仪同三司|开府仪同三司]]、左卫上将军兼内谒者监[[仇士良|仇士良]]纪功碑，诏时任右仆射的李程作文。<ref>[[:s:旧唐书/卷18上|《旧唐书》卷一十八上]]</ref>当年，为东都留守。任内加[[张季戎|张季戎]]留守讨击使兼河阴盐铁留后。<ref>《张季戎墓志》</ref>李程卒年七十七，赠[[太保|太保]]，有司谥'''缪'''。<ref name=NBT131/><ref name=BT167/>而二年（842年）[[太子太傅|太子太傅]][[牛僧孺|牛僧孺]]被任为东都留守，李程或即在此前于任上去世。
