'''李铨'''（{{bd|1912年||1968年|}}）[[江西省|江西省]][[吉安县|吉安县]][[万福镇_(吉安县)|万福镇]]人，[[中国人民解放军开国少将|中国人民解放军开国少将]]<ref name=ji'an>{{Cite web |url = http://www.jianxian.gov.cn/html/2013/6-30/n36721466.html |title = 李铨 |author =  |date = 2013-06-30 |publisher = 中国吉安县 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1929年，李铨加入[[中国共产主义青年团|中国共产主义青年团]]。1930年转入[[中国共产党|中国共产党]]。1930年参加[[中国工农红军|中国工农红军]]。历任[[红二十军|红二十军]]一七三团连指导员，湘赣独立一师一团干事，[[红六军团|红六军团]]十七师四十九团政治处技术书记，没收委员会主任，特务团副政委、政委、十八师政治部主任、教导团政委，军团政治部宣传部部长，[[红二方面军|红二方面军]]随营学校政治部副主任。先后参加过[[湘赣苏区|湘赣苏区]]、[[湘鄂川黔苏区|湘鄂川黔苏区]]反“围剿”战斗，并参加[[长征|长征]]<ref name=ji'an/>。

[[抗日战争|抗日战争]]时期，李铨历任[[八路军|八路军]][[120师|一二〇师]][[359旅|三五九旅]]七一八团、七一七团政委、旅副政委。1944年冬，随同八路军南下支队南进至湘粤边地区。1945年[[日本投降|日本投降]]之后，向北返回中原地区。1946年，参加[[中原突围|中原突围]]，任三五九旅政委。后来转战来到[[陕甘宁边区|陕甘宁边区]]，任[[西北野战军|西北野战军]]第二纵队解放旅旅长兼政委，先后参加了[[晋西南战役|晋西南战役]]、[[汾孝战役|汾（阳）孝（义）战役]]、[[青化砭战役|青化砭战役]]、[[羊马河战役|羊马河战役]]、[[蟠龙战役|蟠龙战役]]、[[沙家店战役|沙家店战役]]、[[宜川战役|宜川战役]]、[[西府战役|西府战役]]等战役<ref name=ji'an/>。

[[中华人民共和国|中华人民共和国]]成立后，李铨起初任第二军五师政委。1949年12月至1951年，任第二十二兵团政治部主任。1953年至1954年10月，任第五军政委。还曾任中共伊犁区委书记。1954年10月至1956年8月，任[[中国人民解放军北疆军区|伊犁军区]]政委。1956年10月至1959年12月，任[[新疆军区|新疆军区]]副政委。后任[[中共新疆维吾尔自治区委|中共新疆维吾尔自治区委]]书记处书记<ref name=ji'an/>。1955年授予少将军衔，获二级八一勋章，二级独立自由勋章、一级解放勋章<ref>{{cite news|title=揭秘：中共首批开国将军的级别是怎样划分的？ |url=http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |accessdate=2012-06-15 |newspaper=凤凰网（转自摘自《党史博览》2010年第5期） |date=2010-06-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20120620091504/http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |archivedate=2012-06-20 }}</ref><ref name=ji'an/>。李铨是[[中国共产党第七次全国代表大会|中国共产党第七次全国代表大会]]代表<ref name=ji'an/>。

1968年，李铨在[[乌鲁木齐市|乌鲁木齐市]]病逝<ref name=ji'an/>。
