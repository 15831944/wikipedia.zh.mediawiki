'''李盛霖'''{{BD|1946年|11月13日||catIdx=L}}，男，[[漢族|漢族]]，[[江苏省|江蘇]][[南通市|南通]]人，曾任[[天津市人民政府|天津市人民政府]][[天津市市长列表|市长]]、[[中華人民共和國交通運輸部|中華人民共和國交通運輸部]]部長。

== 生平 ==
1965年7月至1970年8月，在镇江农业机械学院农业机械系农机专业学习。1970年8月至1972年10月，为天津拖拉机厂工具车间工人、计划员。1972年10月至1976年12月，任天津拖拉机厂团委干事、副书记、书记，工具车间党支部书记。1973年6月，加入中国共产党。1976年12月至1979年2月，任天津化机公司副经理。1979年2月至1980年11月，任天津市拖拉机工业公司技术科副科长、办公室副主任<ref>{{Cite book|title=中华人民共和国政府机构五十年|last=国家行政学院编著|first=|publisher=北京：党建读物出版社|year=2000年6月|isbn=7-80098-406-0|location=|pages=238}}</ref>。

1980年11月至1983年11月，任天津市政府办公厅计划处干部、三处负责人。1983年11月至1986年10月，任天津市政府副秘书长，1986年10月至1988年6月，任天津市纺织工业局局长、党委副书记，1988年6月至1991年10月，任天津市计委主任、中共天津市委计划工委副书记。1991年10月至1993年5月，任[[天津市|天津市]]副市长。1993年5月至1998年5月，任[[中共天津市委|中共天津市委]]副书记、天津市副市长、1998年5月至2002年12月，任[[中共天津市委|中共天津市委]]副书记、[[天津市市长|天津市市长]]<ref>{{Cite book|title=中华人民共和国年鉴 1999 总第19期 下|last=中华人民共和国年鉴编辑部编辑|first=|publisher=中华人民共和国年鉴社|year=2000|isbn=9771000964999|location=|pages=1459}}</ref>。2002年12月至2003年3月，任[[国家经济贸易委员会|国家经济贸易委员会]]副主任、党组副书记。2003年3月起，任[[国家发展和改革委员会|国家发展和改革委员会]]副主任<ref>{{Cite book|title=中国共产党历届中央委员大辞典 1921-2003|last=中共中央组织部，中共中央党史研究室编|first=|publisher=北京：中共党史出版社|year=2004年11月|isbn=7-80136-946-7|location=|pages=358}}</ref>。

2005年12月，担任交通部部长。2008年3月，任交通运输部部长<ref>{{Cite web|url=http://news.sina.com.cn/c/2012-08-31/122525075824.shtml|title=杨传堂被任命为交通运输部部长|accessdate=2017-07-22|work=新浪新闻}}</ref>。2012年8月2日，李盛霖不再担任交通部党组书记职务<ref>{{cite web |url = http://news.163.com/12/0802/14/87TLAC2R00014AED.html |title = 杨传堂担任交通部党组书记李盛霖不再担任 |accessdate = 2012-08-31}}</ref>。2012年11月22日，担任第十一届全国政协委员、经济委员会副主任。2013年3月5日，担任第十二届[[全国人大财政经济委员会|全国人大财政经济委员会]]主任委员<ref>{{Cite web|url=http://news.163.com/17/0425/11/CIS6QMPA00018AOQ.html|title=全国人大财经委主任李盛霖会见泰国国家立法议会代表团_网易新闻|accessdate=2017-07-22|last=网易|work=news.163.com}}</ref><ref>{{Cite web|url=http://renshi.people.com.cn/n/2013/0305/c139617-20685228.html|title=十二届全国人大财政经济委员会主任委员、副主任委员、委员名单--组织人事-人民网|accessdate=2017-07-22|last=103445|work=renshi.people.com.cn}}</ref>。

此外他还担任，中共第十五届、十六届、十七届中央委员。
