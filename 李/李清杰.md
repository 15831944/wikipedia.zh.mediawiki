'''李清杰'''（{{bd|1962年|12月||}}）[[山东|山东]][[成武|成武]]人，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
李清杰曾任[[中国人民解放军总后勤部卫生部|中国人民解放军总后勤部卫生部]]医疗管理局副局长、局长<ref name=huli/>。2012年至2015年，任中国人民解放军总后勤部卫生部副部长<ref name=huli/><ref>北京中医药大学与中国航天员科研训练中心签合作协议探索建立航天中医药学科，中医药导报2012(8):16-16</ref>。2015年，任中国人民解放军总后勤部卫生部部长<ref name=huli>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1326586  |title = 总后两将领履新：胡宜树任司令部参谋长，李清杰任卫生部部长 |author =  |date = 2015-04-30 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。2016年1月，出任新组建的[[中央军委后勤保障部卫生局|中央军委后勤保障部卫生局]]局长<ref>{{Cite web |url = http://news.163.com/16/0219/09/BG66OJM800014AED.html |title =  原总后卫生部部长李清杰出任军委后勤保障部卫生局局长|author =  |date =  2016-02-19|publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。2017年，升任[[中央军委后勤保障部|中央军委后勤保障部]]副部长。同时兼任[[全国爱国卫生运动委员会|全国爱国卫生运动委员会]]副主任<ref>{{Cite web |url = http://news.163.com/17/0526/11/CLC0IBKO000187VE.html |title = 军委后勤保障部卫生局局长李清杰升任军委后勤保障部副部长 |author =  |date =2017-05-26  |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。

2013年7月晋升少将军衔。
