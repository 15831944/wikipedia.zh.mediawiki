'''李浩杰'''（{{kor|名=中|谚=리호걸}}，{{bd|1993年|6月3日||}}），[[中国朝鲜族|朝鲜族]]，出生于[[中国|中国]][[吉林省|吉林省]][[延边朝鲜族自治州|延边州]][[延吉市|延吉市]]，目前是[[延边富德|延边富德队]]的球员，司职[[中场_(足球)|中场]]。

== 俱乐部生涯 ==
李浩杰于[[2012年中国足球甲级联赛|2012年]]被[[延边富德|延边富德队]]选拔至一线队，自此李浩杰开始了他的职业足球生涯<ref>{{cite web |url=http://sports.sina.com.cn/b/2012-03-17/00075984348.shtml |title=延边长白虎完全名单：马里国脚10号 韩国前锋11号 |publisher=新浪体育 |date=2012-03-17 |accessdate=2018-07-09 |language=zh-hans |deadurl=no |archiveurl=https://web.archive.org/web/20180709111412/http://sports.sina.com.cn/b/2012-03-17/00075984348.shtml |archivedate=2018-07-09 |df= }}</ref>。2012年6月26日，他在[[2012年中国足协杯|2012年中国足协杯]]的[[延边富德|延边富德队]]0:8负给[[大连实德|大连实德队]]的比赛中完成了成年队首秀。不过，他在这场比赛中的表现不佳，在第75分钟时因在[[禁区|禁区]]内对[[卡洛斯·阿德里亚诺·索萨·克鲁斯|阿德里亚诺]]犯规为对方创造了一粒[[点球|点球]]<ref>{{cite web |url=http://sports.sohu.com/20120626/n346576174.shtml |title=足协杯-阿德独进5球 实德8-0狂胜延边晋级第4轮 |publisher=搜狐体育 |date=2012-06-26 |accessdate=2018-07-09 |language=zh-hans |deadurl=no |archiveurl=https://web.archive.org/web/20150617195723/http://sports.sohu.com/20120626/n346576174.shtml |archivedate=2015-06-17 |df= }}</ref>。2013年4月20日，李浩杰在延边富德队0:0[[沈阳沈北|沈阳沈北队]]的比赛中完成了他在联赛中的首秀<ref>{{cite web |url=http://www.hybrb.com/show/?idx=30956 |title=全运小将融入长白虎战术体系 |publisher=延边新闻 |date=2013-04-23 |accessdate=2018-07-09 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160419051416/http://www.hybrb.com/show/?idx=30956 |archivedate=2016-04-19 |df= }}</ref>。2016年4月2日，李浩杰在延边富德队1:0胜于[[北京国安|北京国安队]]的比赛中完成了他在[[中国足球超级联赛|中超]]赛场上的首秀<ref>{{cite news |url=http://sports.sina.com.cn/china/j/2016-04-02/doc-ifxqxcnr5219090.shtml |title=河太均头球制胜 国安0-1延边两轮不胜 |publisher=新浪体育 |date=2016-04-02 |accessdate=2018-07-09 |language=zh-hans |deadurl=no |archiveurl=https://web.archive.org/web/20170222053448/http://sports.sina.com.cn/china/j/2016-04-02/doc-ifxqxcnr5219090.shtml |archivedate=2017-02-22 |df= }}</ref>。
