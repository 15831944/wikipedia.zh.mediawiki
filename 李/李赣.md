'''李赣'''，外号“'''李老八'''”，“'''电竞李伯清'''”，中国大陆[[英雄联盟|英雄联盟]]游戏主播。<ref name=eastday></ref>

==简介==
李赣学习成绩一般，高考考入[[西南石油大学|西南石油大学]]（大专）专科数控专业（后转为钻井专业），临近毕业没有拿到毕业证。肄业后的李赣加入[[平安保险|平安保险]]集团卖了两个月的保险，月薪8000元人民币；其父为军人，故有军警系统关系，成都交警招聘协警旋即考试加入。<ref name=":0" />

李赣最初在[[百度贴吧|百度贴吧]]黑WE吧中出名，<ref name="eastday">{{Cite web |url =http://sports.eastday.com/a/171214114851242000000-2.html|title =还记得曾在斗鱼风光无限的李赣吗？如今在外文台惨淡直播，乞讨为生}}</ref>其后制作“国服第一开发师”系列视频在[[Acfun|Acfun]]、[[bilibili|bilibili]]拥有人气，该系列视频以开发各种英雄的奇特玩法为主要内容。<ref name="gamedog">{{Cite web  |url = http://www.gamedog.cn/gonglue/20171130/2270356.html  |title = 斗鱼6324李哥成绝唱 李赣又双叒叕被封  |access-date = 2018-01-18  |archive-url = https://web.archive.org/web/20180119060643/http://www.gamedog.cn/gonglue/20171130/2270356.html  |archive-date = 2018-01-19  |dead-url = yes  }}</ref>

李赣最早在Acfun生放送进行直播，在在Acfun生放送转变成斗鱼TV后在[[斗鱼TV|斗鱼TV]]后进行游戏直播。<ref name="sjzcmw">{{Cite web |url = http://www.sjzcmw.com/news/star/24051.html  |title = 斗鱼李老八什么梗：斗鱼李老八是谁个人资料}}</ref>直播内容最初为分析各种英雄联盟游戏比赛带人下注，后又通过直播睡觉等事件走红。<ref name="gamedog" />在获得斗鱼TV给出的近300万人民币的年薪后，于2015年9月1日创立“抽象工作室”<ref name="sjzcmw" />，与于超、阿不多、[[孙笑川|孙笑川]]等人在直播间进行24小时直播。半年后，阿不多和于超离开抽象工作室。<ref name="eastday">{{Cite web |url =http://sports.eastday.com/a/171214114851242000000-2.html|title =还记得曾在斗鱼风光无限的李赣吗？如今在外文台惨淡直播，乞讨为生}}</ref>2017年9月，孙笑川离开抽象工作室。<ref name="eastday" />

2017年6月18日，李赣在直播中接到[[法轮功|法轮功]]宣传电话，接听十几秒后改为免提。在超级管理员<ref group="註">斗鱼TV的超级管理员是主播的管理员，拥有将直播间关闭和封禁的权限。</ref>到达直播间后，李赣拨打举报电话实名举报该宣传电话。几分钟后，李赣的直播间被永久封禁。<ref>[http://www.sjzcmw.com/news/sports/33512.html 李干接电话完整版视频：斗鱼6324被永封3分钟电话视频]</ref>被斗鱼永久封禁后，11月20日，李赣在[[Twitch_(網路)|Twitch]]中文台直播《[[绝地求生|绝地求生]]》。<ref name="te">{{Cite web|url =http://www.te5.com/news/2017/275695.html|title =LOL前斗鱼主播李赣微博称不再直播了|access-date =2018-01-18|archive-url =https://web.archive.org/web/20180119060617/http://www.te5.com/news/2017/275695.html|archive-date =2018-01-19|dead-url =yes}}</ref><ref name="gamedog" />

2017年11月25日，李赣就直播中接听法轮功宣传电话一事发表道歉声明，并解释是自己[[听力|听力]]问题而打开免提<ref name=te></ref>，并于11月29日声称不会再进行游戏直播。<ref name=gamedog></ref>

在陈羽凡吸毒事件后，斗鱼TV开展主播爱国教育培训，30多名主播参加此次培训，其中包括李赣。2018年8月21日，李赣在斗鱼TV重新开播。<ref name="eastday2">{{Cite web |url = http://sports.eastday.com/a/180824195153738000000.html|title =人人都说抽象话，无人再识李老八6324创始人李赣这次回来了}}</ref>

== 家庭情况 ==
李赣父亲原籍江西[[景德镇|景德镇]][[浮梁|浮梁]]，参军后长期居于[[成都|成都]][[新津|新津]]，退伍后担任新津县某兵工厂采购一职。其母亲为教师。<ref name=":0">{{Citation|title=李老八已婚丶 - Twitch|url=https://www.twitch.tv/liganss6324|accessdate=2018-03-12}}</ref>
