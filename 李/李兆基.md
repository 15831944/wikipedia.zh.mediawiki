'''李兆基'''博士{{大紫荊}}（{{lang-en|Lee Shau-kee}}，{{bd|1928年|2月20日|catIdx=L李}}）， 生于广东省顺德市， 人称“四叔”、“香港巴菲特”，[[香港|香港]]商人和慈善家。 曾为香港首富，在1996年至1997年是世界十大富豪，名列第四。現任 [[地產發展商|地產發展商]][[恆基兆業|恆基兆業]]及[[中華煤氣|中華煤氣]]的[[董事局主席|董事局主席]]，並且為[[新鴻基地產|新鴻基地產]]董事局副主席。

李兆基從6歲開始（即1934年），就被父親安排進家裡的商鋪頭學習做生意。母親本來有點擔心兒子年紀太小，不能應付那些年長的夥計，但李兆基沒過多久就成了父親的得力助手，而且聰明機智，擁有極好的心算能力。 在金鋪行業，有一句「打金偷金、打銀偷銀」的俗語。李兆基進入店鋪後，發現自己家的金鋪也存在這個問題。他想把這個消息告訴父親，讓父親採取措施，但鑄金匠又是奇缺的人才，得罪不起，因此而擔心鑄金匠離開後影響金鋪的生意。進退兩難中，他決定自己學習鑄金技術，不再受制於人。

1958年，李兆基與[[郭得勝|郭得勝]]和[[馮景禧|馮景禧]]，連同胡兆熾、黃少軒、戚宗煌、呂賢藻和郭錦濤創辦永業，合資經營地產。至1963年，李、郭和馮三人另組[[新鴻基地產|新鴻基地產]]。<ref>[http://www.mingpaonews.com/20070118/gaa4.htm 明報新聞: 創業伙伴離世 感觸夕陽近黃昏]</ref>1996年，李兆基的財富累積至127億美元，成為全球第四富有的人。

2007年7月1日，李兆基獲頒[[大紫荊勳章|大紫荊勳章]]。2015年，李兆基身家有1,950億港元，據《[[福布斯香港富豪榜|福布斯香港富豪榜]]》第二位。<ref>[https://web.archive.org/web/20070519002423/http://hk.news.yahoo.com/070118/12/2072d.html 李嘉誠位列大中華首席富豪]</ref>。
2018年，李兆基身家有2,566億港元，據《[[福布斯香港富豪榜|福布斯香港富豪榜]]》第二位<ref>[https://news.now.com/home/finance/player?newsId=250919 福布斯香港富豪榜】李嘉誠連續廿年高踞榜首]</ref>。李兆基在福布斯2019年全世界億萬富翁排行榜中名列第29位，資產達到301億美元<ref>{{Cite web |url =https://www.forbes.com/profile/lee-shau-kee/?list=billionaires#2df58f541083| title = #29 Lee Shau Kee | publisher = Forbes |language =en |accessdate = 2019-03-21 }}</ref>。

== 生平 ==
李兆基藉貫[[廣東省|廣東省]][[順德縣|順德縣]][[大良鎮|大良鎮]]，出生於1928年2月20日、農曆正月廿九<ref>[http://orientaldaily.on.cc/cnt/finance/20150318/00271_001.html 四叔賀壽化身白之戀人]</ref>，是父親李介甫與夫人陈鸾凤的第四个孩子。李家排行第一和第二的孩子在出生不久之后就夭折了，排行第三的孩子李兆麟就成了李家的长子。李兆基出生的家庭相當富裕，其父李介甫在二十年代中叶于故乡先后开设天宝荣金铺和永生银号，经营黄金、汇兑和外币买卖。

== 其他職位 ==

== 名下馬匹 ==
李兆基為[[香港賽馬會|香港賽馬會]]會員，與好友[[鄭裕彤|鄭裕彤]]及[[明嘉福|明嘉福]]（包括他的遺孀明李祝玲）共同擁有多匹以鑽字命名的競賽馬，包括已退役的鑽之寶、黃鑽石、幸運鑽石、紅鑽石。
