《'''李朝实录'''》，又稱《'''朝鮮王朝實錄'''》或《'''朝鮮封建王朝實錄'''》，是[[李氏朝鲜|李氏朝鲜]]《[[朝鮮太祖|太祖]]大王实录》到《[[朝鮮哲宗|哲宗]]大王实录》的一个统编，是李氏朝鲜（又称“朝鮮王朝”）472年的历代[[实录|实录]]。是第151號[[韓國國寶|韓國國寶]]。1990年代開始，韓國基金會將其翻譯成現代韓文並進行[[數位化|數位化]]<ref name="Foundation2013">{{cite book |author = The Korea Foundation |title = Korea Focus - April 2012 |url = http://www.koreafocus.or.kr |date = 30 March 2013 |publisher = i-ePUB, Inc. |ISBN = 978-89-86090-84-0 |pages = 83– |access-date = 2019-04-21 |archive-url = https://web.archive.org/web/20130514052057/http://koreafocus.or.kr/ |archive-date = 2013-05-14 |dead-url = yes }}</ref>。1997年10月，《李朝实录》登錄為[[世界紀錄遺產|世界紀錄遺產]]<ref name="朝鮮王朝實錄">{{cite web |title = 世界紀錄遺產 [朝鮮王朝實錄] |publisher = [[韓國觀光公社|韓國觀光公社]] |url = http://www.visitkorea.or.kr/cht/CU/CU_CH_3_4_5_2.jsp |accessdate = 2013-09-01 |archive-url = https://web.archive.org/web/20131203145908/http://www.visitkorea.or.kr/cht/CU/CU_CH_3_4_5_2.jsp |archive-date = 2013-12-03 |dead-url = yes }}</ref><ref>{{cite web |url = http://www.unesco.org/new/zh/communication-and-information/flagship-project-activities/memory-of-the-world/register/full-list-of-registered-heritage/registered-heritage-page-8/ |title = 世界记忆名录 S-T/朝鲜王朝实录 |publisher = [[联合国教科文组织|联合国教科文组织]] }}</ref>。韓國[[國史編纂委員會|國史編纂委員會]]在1993年完成《朝鮮王朝實錄》翻譯成韓文，2006年提供原文（[[漢文|漢文]]）和[[韓文|韓文]]本的網路閱覽服務，並預計在2033年完成英譯本<ref>{{cite web |title = 《朝鮮王朝實錄》，預計投入400億韓元到2033年完成英譯本 |url = http://chinese.joins.com/big5/article.do?method=detail&art_id=79072 |publisher =韓國《[[中央日報|中央日報]]》 |date = 2012-01-09 |author = 河賢玉 |accessdate = 2013-09-01 }}</ref>。

== 概要 ==
《李朝實錄》記載了由初代[[朝鮮太祖|太祖]]到[[朝鮮哲宗|哲宗]]的25代[[国王|国王]]、472年（1392年－1863年）间[[历史|历史]]事实的年月日順[[编年体|编年体]][[汉文|汉文]]记录；若連最後兩任君主的紀錄也包括在內的話，那包含總共27代君主、519年（1392年－1910年），共1893-{卷}-，888冊，總共約6400萬字。

朝鮮王朝前期，《李朝實錄》正本存放於[[漢陽|漢陽]]（今[[首爾|首爾]]）[[春秋馆|春秋馆]]，副本存放於[[忠州|忠州]]、[[星州|星州]]、[[全州|全州]]的史库。[[壬辰倭乱|壬辰倭乱]]中，除全州本之外，《李朝實錄》正本和其餘副本全部被毁。[[朝鮮宣祖|宣祖]]三十六年，依据全州本重新编修，[[印刷|印刷]]五部，分别存於春秋馆、[[摩尼山|摩尼山]]史库、[[太白山|太白山]]史库、[[妙香山|妙香山]]史库、[[韩国五台山|五台山]]史库。妙香山本後來移藏於[[赤裳山|赤裳山]]，摩尼山本後來移藏於[[鼎足山|鼎足山]]。1905年，春秋馆本移存於[[奎章阁|奎章阁]]。

1910年[[日韓合邦|日韓合邦]]後，奎章阁图书由日本[[朝鮮总督府|朝鮮总督府]]收集。1911年，太白山本和鼎足山本也移交朝鲜总督府，赤裳山本收藏於[[藏书阁|藏书阁]]，五台山本被“赠送”给[[东京帝国大学|东京帝国大学]]。1923年[[关东大地震|关东大地震]]，五台山本大部份被烧毁，剩餘74冊<ref>{{cite web |title =《朝鮮王朝實錄》首次露出廬山真面目 |url = http://tw.aboluowang.com/news/2006/0716/6399.html#.UiOAhX_rCAZ |publisher=阿波羅新聞 |author=金榮禎 |work=朝鮮日報 |date=2006-07-16 |accessdate=2013-09-01}}</ref>。1930年，朝鲜总督府将太白山本和鼎足山本送给[[京城帝国大学|京城帝国大学]]；[[日本投降|日本投降]]后，移交韩国，存放於[[首尔大学|首尔大学]]。[[朝鲜战争|朝鲜战争]]期间，鼎足山本被[[朝鲜人民军|朝鲜人民军]]缴获，移至[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]，现存放在[[金日成综合大学|金日成综合大学]]；太白山本则被[[韩国政府|韩国政府]]转移至[[釜山|釜山]]，至今仍存放於韩国[[國家記錄院|國家記錄院]]轄下的[[歷史檔案館|歷史檔案館]]。

現存的《李朝實錄》有三個版本：
* 鼎足山本有1181冊；
* 太白山本有848冊；
* 五台山本有27冊；
其他散逸本有21冊，合共2,077冊。

《李朝實錄》包括太祖、[[朝鮮定宗|定宗]]、[[朝鮮太宗|太宗]]直至哲宗的23部实录，以及[[燕山君|燕山君]]和[[光海君|光海君]]的兩部日记（体裁与实录相同）{{NoteTag|〈端宗大王實錄〉原名〈魯山君日記〉({{lang|ko|노산군일기}})，肅宗年間追宗後改稱〈端宗大王實錄〉({{lang|ko|단종대왕실록}})|name="danjong"}}。日本朝鲜总督府曾编纂了《高宗实录》和《纯宗实录》，记载[[朝鮮高宗|高宗]]和[[朝鮮純宗|純宗]]兩代所發生的事，但不為[[大韓民國|韓國]]及朝鮮民主主義人民共和國的[[歷史學家|歷史學家]]所承認。在韩国，不包含这些最后二朝的实录。

《李朝实录》关于朝鮮王朝25代君主472年的[[政治|政治]]、[[外交|外交]]、[[军事|军事]]、[[经济|经济]]方面的庞大[[史料|史料]]，被认为是研究[[李氏朝鲜|李氏朝鲜]]的基本史料<ref name="朝鮮王朝實錄" />。《李朝实录》中也包含不少有关[[中国歷史|中国歷史]]、[[女真歷史|女真歷史]]與[[日本歷史|日本歷史]]的史料，中国历史學家[[吴晗|吴晗]]曾经编纂《[[朝鲜李朝实录中的中国史料|朝鲜李朝实录中的中国史料]]》一書<ref>{{cite book |url = http://ishare.edu.sina.com.cn/f/4412344.html |title = 《朝鲜李朝实录中的中国史料》 |editor = 吴晗 |deadurl = yes |archiveurl = https://web.archive.org/web/20131203002456/http://ishare.edu.sina.com.cn/f/4412344.html |archivedate = 2013-12-03 }}</ref>。

1997年10月，《李朝实录》被[[联合国教科文组织|联合国教科文组织]]登记为[[世界纪录遗产|世界纪录遗产]]（此為韓國用詞，[[中國|中國]]稱為[[世界记忆项目|世界记忆项目]]）<ref>『조선왕조실록, 보존을 위한 기초 조사연구(1)』(송기중 신병주 박지선 이인성, 서울대학교출판부)</ref>，亦被列入[[韓國國寶|韓國國寶]]第151號。

== 出版 ==
1953年，《李朝实录》由[[日本学习院|日本学习院]]东洋文化研究所影印出版，共五十册，该版《李朝实录》未包含〈高宗实录〉和〈纯宗实录〉。

1959年，《李朝实录》中的〈高宗实录〉和〈纯宗实录〉由[[中国科学院|中国科学院]]和[[朝鲜科学院|朝鲜科学院]]合作根据“[[金柜秘本|金柜秘本]]”影印出版。
朝鲜文版《李朝实录》由[[朝鲜社会科学院|朝鲜社会科学院]]民族古典研究所翻译出版。

== 女真族問題 ==
在《李朝實錄》對外國或外族的記錄中，最值得一提的是，對女真族的記載最為詳細，比起《明實錄》或中國其他史書對女真族的記載還要完整。《李朝實錄》可說是一般研究女真歷史文化的重要史料。

== [[倭寇|倭寇]]問題==
在〈世宗實錄〉中，敘述「前朝之際，[[倭寇|倭寇]]興行，民不聊生。然倭人不過一二，而本国之民，反著倭服，成党作乱。」〈太宗實錄〉又寫「或時，反倭為寇。」紀錄寇邊的[[倭寇|倭寇]]，其組成實以朝鮮人為主。
