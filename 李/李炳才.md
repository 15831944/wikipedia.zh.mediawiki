'''李炳才'''（{{bd|1945年|1月|}}），[[江苏|江苏]][[宝应|宝应]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员。

==生平==
李炳才是[[中国共产党|中国共产党]]党员。1991年4月至1992年3月，任[[扬州市|扬州市]]人民政府副市长代市长。1992年3月至1993年3月，任扬州市人民政府市长<ref>{{Cite web |url =  http://www.jssdfz.gov.cn/book/zfz/ZFZ/D9/D1J3.HTM|title = 三、 省辖市人民政府、地区行署 |author =  |date =  |publisher = 江苏省地方志 |language =  |accessdate = 2017-10-12 |quote =  }}</ref>。1993年5月至1996年12月，任中共扬州市委书记<ref name=baoying/>。

1996年11月至2006年7月，历任[[中共中央台湾工作办公室|中共中央台湾工作办公室]]、[[国务院台湾事务办公室|国务院台湾事务办公室]]副主任，中共中央台湾工作办公室、国务院台湾事务办公室常务副主任。2000年7月任[[海峡两岸关系协会|海峡两岸关系协会]]常务副会长<ref name=baoying>{{Cite web |url =  http://www.byda.gov.cn/ggjy/Show.asp?ID=16|title = 时代精英 |author =  |date =  |publisher =宝应县档案网  |language =  |accessdate = 2017-10-12 |quote =  }}</ref>。2008年6月至2013年4月，任海峡两岸关系协会驻会副会长<ref name=erjj>{{Cite web |url =  http://news.ifeng.com/taiwan/special/chenjiangliuhui/content-3/detail_2010_12/17/3563925_0.shtml|title = 海峡两岸关系协会简介 |author =  |date = 2010-12-17 |publisher =  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.huaxia.com/tslj/rdqy/js/2012/05/2853238.html |title =  海协会副会长李炳才至昆山海峡商贸示范区考察|author =  |date =   2012-05-18|publisher = 华夏经纬网 |language =  |accessdate =  |quote =  }}</ref>。

2008年，当选第十一届全国政协委员<ref>{{cite news|title=第十一届全国政协委员名单|url=http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|accessdate=2013-04-09|newspaper=新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过)|date=2008年1月25日|deadurl=yes|archiveurl=https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|archivedate=2013年9月28日|df=}}</ref>，代表经济界，分入第三十六组。并担任第十一届全国政协港澳台侨委员会副主任<ref>{{cite news|title=全国政协信息-李炳才|url=http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0012170&wytype=wjwy|accessdate=2013-04-11|newspaper=全国政协委员会}}</ref>。
