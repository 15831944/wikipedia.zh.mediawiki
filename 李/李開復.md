'''李开复'''（{{bd|1961年|12月3日|catIdx=L李开复}}），生于[[臺灣|臺灣]][[臺北縣|臺北縣]][[中和區|中和市]]（現為[[新北市|新北市]][[中和區|中和區]]），擁有[[中華民國|中華民國]]國籍，是一位[[信息产业|信息产业]]的經理人、創業者和[[计算机科学|计算机科学]]的研究者。現任[[創新工場|創新工場]]的[[董事长|董事长]]兼[[執行長|執行長]]。曾在[[蘋果公司|蘋果]]、[[硅谷图形公司|SGI]]、[[微軟|微軟]]和[[Google|Google]]等多家IT公司擔當要職。他因2005年微软和谷歌的法律纠纷而成为公众的爭議焦点<ref>[http://edu.sina.com.cn/en/2013-06-18/152774858.shtml 李开复专访：苹果谷歌和Facebook]</ref><ref>[http://app.tech.ifeng.com/person/intro.php?name=%E6%9D%8E%E5%BC%80%E5%A4%8D 凤凰网科技-焦点人物] {{webarchive|url=https://web.archive.org/web/20121011190757/http://app.tech.ifeng.com/person/intro.php?name=%E6%9D%8E%E5%BC%80%E5%A4%8D |date=2012-10-11 }}</ref>。

== 生平 ==
李開復祖籍大陸四川省華陽縣，在臺灣省出生. 李开复的父親是中華民國[[第一屆立法委員|第一屆立法委員]][[李天民|李天民]]，曾任教於[[黃埔軍校|黃埔軍校]]成都分校以及[[國立政治大學|國立政治大學]]歷史系，李開复的母親王雅清為[[辽北省|遼北]]人。

中學時移民美國，獲得[[美國國籍|美國國籍]]。曾任職微軟公司、谷歌公司，1998年移居[[中国大陸|中国大陸]][[北京市|北京市]]。2011年，放棄美國國籍，只保留中華民國國籍。<ref>[http://www.chinatimes.com/newspapers/20130912000963-260301 20多個腫瘤纏身 李開復返台化療 - 中時電子報]</ref>

===早年===
1972年跟隨著哥哥至[[美國|美國]][[田納西州|田納西州]][[橡树岭_(田纳西州)|橡树岭]]就讀中學，1983年从[[哥伦比亚大学|哥伦比亚大学]]電腦科學系毕业，1988年获得[[卡内基梅隆大学|卡内基梅隆大学]]计算机系博士，當年被《[[商业周刊|商业周刊]]》授予“最重要科学创新奖”。<ref>{{cite web |url=http://www.5xue.com/modules/kaifulee/view.article.php/38/c0 |title=存档副本 |accessdate=2012-04-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20130202054038/http://www.5xue.com/modules/kaifulee/view.article.php/38/c0 |archivedate=2013-02-02 }} - 个人历史背景</ref>

90年代曾在[[苹果公司|苹果公司]]交互式多媒体部门工作，任副总裁，曾负责[[Apple_Pippin|Apple Pippin]]项目。后来转投[[硅谷图形公司|硅谷图形公司]]（SGI）公司工作，担任互联网部门副总裁兼总经理、Cosmo软件公司总裁，主要负责多平台、互联网三维图形和多媒体软件的研发工作。

===加入微軟===
1998年7月，李开复加入微软并在中国建立并领导微软中国研究院（现为[[微软亚洲研究院|微软亚洲研究院]]）。任微软公司自然交互式软件及服务部门副总裁，负责研发各种先进的技术和服务使得人机界面更加简便和自然。该部门负责开发的技术和产品包括[[语音|语音]]、[[自然语言|自然语言]]、全新的搜索和在线服务等技术。自然交互式软件及服务部门的使命就是要让所有这些技术能够更好地服务于微软的客户。在他的带领下，微软中国研究院以新一代多媒体、新一代用户界面和新一代信息处理技术为主要方向开展基础研究。后来被微软调回美国，於2000年担任微软自然交互式软件及服务部门副总裁<ref>{{cite news|url=http://tech.sina.com.cn/it/2004-06-30/1108381604.shtml|title=微软自然交互式软件及服务部门副总裁李开复简历|author=新浪科技|date=2004年6月30日|}}</ref>。2002年獲選為[[電機電子工程師學會|電機電子工程師學會]]會士。

===轉戰Google===
[[File:Google_music_search.jpg|thumb]]
2005年7月19日，谷歌宣布将在中国设立产品研发中心，李开复将负责其中国研发中心的运营，并担任谷歌全球副总裁和大中华区总裁<ref> [http://tech.sina.com.cn/focus/ms_kaifu/index.shtml 微软副总裁李开复闪电加盟谷歌] - 新浪网</ref>。当天，微软向美国华盛顿州地方法院提起诉讼，指控谷歌和前微软全球副总裁李开复违反了[[竞业禁止协议|竞业禁止协议]]。9月13日，美国金县高等法院就李开复违反竞业禁止协议案做出裁决：李开复可以立即为谷歌工作，但工作范围将受到限制。这一限制在2006年1月的庭审之前一直有效。12月22日，微软公司终止了对谷歌以及李开复的诉讼。微软公司、李开复以及谷歌公司就他们之间的诉讼，已经达成非公开协议来解决各项问题，各方对此协议都表示满意。<ref>[http://dotnet.csdn.net/n/20051223/31465.html 谷歌微软就李开复案握手言和] {{webarchive|url=https://web.archive.org/web/20051225225106/http://dotnet.csdn.net/n/20051223/31465.html |date=2005-12-25 }} - CSDN转计算机世界报</ref>

===创立「创新工场」===

从2000年起，李开复陆续发表了七封《给中国学生的信》<ref>《给中国学生的信》{{cite web |url=http://www.kaifulee.com/modules/kaifulee/view.article.php/37 |title=存档副本 |accessdate=2007-04-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20070606105018/http://www.kaifulee.com/modules/kaifulee/view.article.php/37 |archivedate=2007-06-06 }}{{cite web |url=http://www.kaifulee.com/modules/kaifulee/view.article.php/35 |title=存档副本 |accessdate=2007-04-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20070402031506/http://www.kaifulee.com/modules/kaifulee/view.article.php/35 |archivedate=2007-04-02 }}{{cite web |url=http://www.kaifulee.com/modules/kaifulee/view.article.php/34 |title=存档副本 |accessdate=2007-04-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20070524223954/http://www.kaifulee.com/modules/kaifulee/view.article.php/34 |archivedate=2007-05-24 }}{{cite web |url=http://www.kaifulee.com/modules/kaifulee/view.article.php/33 |title=存档副本 |accessdate=2007-04-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20070503105742/http://www.kaifulee.com/modules/kaifulee/view.article.php/33 |archivedate=2007-05-03 }}{{cite web |url=http://www.kaifulee.com/modules/kaifulee/view.article.php/32 |title=存档副本 |accessdate=2007-04-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20070402090300/http://www.kaifulee.com/modules/kaifulee/view.article.php/32 |archivedate=2007-04-02 }}{{cite web |url=http://www.kaifulee.com/modules/kaifulee/view.article.php/31 |title=存档副本 |accessdate=2007-04-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20070125081938/http://www.kaifulee.com/modules/kaifulee/view.article.php/31 |archivedate=2007-01-25 }}{{cite web |url=http://www.5xue.com/modules/wordpress/?p=1258 |title=存档副本 |accessdate=2009-11-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20090908003523/http://www.5xue.com/modules/wordpress/?p=1258 |archivedate=2009-09-08 }} - 我学网</ref>，其中分别谈到了做人要有诚信、如何从优秀到卓越、选择的智慧、新世纪的人才观等等。

[[创新工场|创新工场]]由李开复创办于2009年9月7日，是一所旨在帮助年轻人创业的天使投资公司。

===患病===
2013年9月5日22时17分，李开复在其[[新浪微博|新浪微博]]上发出一条微博：“世事无常，生命有限。原来，在癌症面前，人人平等。”<ref>[http://weibo.com/1197161814/A85WWyJQE 李开复的新浪微博]</ref> 随后，其认证资料为“创新工场联合创始人、新闻发言人”的同事王肇辉确认李开复被查出患有癌症，正在接受医生治疗<ref>[http://weibo.com/1197033464/A86g9qu4d 王肇辉的新浪微博]</ref>。而在9月6日清晨8时6分，李开復本人更新微博：“看到评论里这么多网友的问候和关心，真的很感动！我一直相信：每个人在病痛面前的姿态和选择决定了其未来精神世界的质量。虽然淋巴癌听起来并不乐观，也让家人和朋友们很担心。但生活就是这样：往往来得意外，但既然遭遇就应坦然面对。病痛也是生活的一部分，我会选择更加积极的心态来面对生活起伏”<ref>[http://weibo.com/1197161814/A89OrlCql 李开复的新浪微博]</ref>。证实自己所患为[[淋巴癌|淋巴癌]]，醫生稱其患病的原因是“承受压力太大”和“作息时间不规律”<ref>{{cite newspaper | url=http://news.sohu.com/20130907/n386085049.shtml | title=李开复患淋巴癌 称病因或为“熬夜多压力大” | publisher=[[新京報|新京報]] | date=2013-09-07}}</ref>。李开复患淋巴癌的事情在微博上引起广泛关注。9月9日晚，李开复往台灣治療<ref>{{cite newspaper | url=http://www.apdnews.com/news/34223.html | title=李开複全身20多处肿瘤已赴台治疗 | publisher=[[亚太日报|亚太日报]] | date=2013-09-11 | deadurl=yes | archiveurl=https://web.archive.org/web/20130918114101/http://www.apdnews.com/news/34223.html | archivedate=2013-09-18 }}</ref>。

==人工智能論==
李開復認為[[人工智慧|人工智慧]]註定顛覆世界，帶來前所未有的經濟失衡，甚至改變全球權力格局，且不可避免的會對[[就業|就業]]造成衝擊，超乎想像的職缺都可能被AI取代，改變世界經濟政治。而人工智慧是「強者更強」的產業，最終只有中美兩國有足夠人才和資金作對決。大部分人工智慧所創造的財富，會流入美國和中國。<ref>[https://www.limitlessiq.com/news/post/view/id/1129/ 李開復：AI創造的財富將流入中、美兩國]</ref>

而AI極可能造成大規模失業的解決方案，是社會創造出更多與「關愛服務」相關的工作，例如，陪伴老人就醫的志工、孤兒院教導員、戒酒互助社的義工，他認為這是唯一解藥。而經費來自政府靠AI獲得高額利潤公司，課更多稅。預判未來15年內AI將取代50%的工作，無論[[白領|白領]]、藍領都受衝擊；AI襲捲全球將引發[[失業|失業]]潮，並使貧富差距更擴大。<ref>[http://www.chinatimes.com/newspapers/20180827000455-260110 李開復談AI]</ref>教育體制也必須以創意培養為主，不能侷限在死背記憶的老路，在記憶方面沒人比得過AI。

人工智慧目前的發展路線是需要巨量的訊息樣本讓機器自我學習演化，<ref>[https://www.chinatimes.com/realtimenews/20180926003779-260408 佛里曼-中美大未來]</ref>中國又有比美國大的多的人口和政府能任意取樣個人資料，因此中國獲取超大數據並加以分析的能力超過美國，李表示「如果大數據是新石油，中國就是沙烏地阿拉伯」。

==家庭==
*父亲[[李天民|李天民]]（1909年10月14日-1993年6月24日）：生於四川；畢業於武漢[[中央軍事政治學校|中央軍事政治學校]]；大学本科畢業於日本[[早稻田大学|早稻田大学]]經濟學專業；中华民国第一届立法委员李天民，曾任教于黄埔军校成都分校以及[[国立政治大学|国立政治大学]]历史系；著有5本中國領導人傳記：周恩來、劉少奇、鄧小平、林彪及華國鋒。<ref>{{cite web|url=http://m.cwzg.cn/expose/201702/34066.html|title=别了，李开复——奇特“导师”不为人知的二三事|publisher=察網|date=2017年2月5日}}</ref>
*母亲[[王雅清|王雅清]]（1918年9月6日-）：辽北人，生于一个务农的家庭；为躲避日军侵略，12岁就从东北离家，随东北人成立的流亡中学漂泊异乡。在南国开始了自己的求学之路，并最终以优异的成绩毕业于上海体专。育有7个儿女（2儿5女）；长子出生于1935年，幼子李开复排行最小。
*姐姐[[李开敏|李开敏]]：台大社会工作系及通识教育中心讲师，諮商心理师，社工师。
*妻子[[谢先玲|谢先玲]]（？年4月9日-）：双方父母是世家；李开复父亲李天民和谢先玲父亲[[谢星曲|谢星曲]]是同事，谢星曲也是台湾的立法委员之一，两个爸爸在工作方面很熟络。1982年6月在美國讀大三的李開復回台灣度暑假期間，谢先玲和李开复的父亲共同的朋友冯伯伯介绍双方认识。1983年8月6日年仅21岁的李开复与谢先玲结婚，婚后谢先玲为全职家庭主妇。夫妇育有二女：李德宁與李德亭。
*长女[[李德宁|李德宁]]Jennifer Dening Lee（1991年12月16日-）：大学本科毕业于美国[[哥伦比亚大学|哥伦比亚大学]]心理学专业、[[帕森设计学院|帕森设计学院]]服装设计专业；現职服装设计师。
*小女[[李德亭|李德亭]]：愛好攝影，希望成為攝影師。<ref>{{Cite web |url = http://huaren.haiwainet.cn/n/2015/0307/c232657-28496039.html |title = 李开复癌症复出 悄临英国谈留学 |date = 2015-03-07 |publisher = 海外网|quote = 我的小女儿决定做摄影师。我给她分析，自动对焦技术一下可以拍1000张照片，回去再对焦，你们还有未来吗？业余者用Instragram发的照片已经有你的八成（功力），用单反的人拍的也到你的九成了，全民都会摄影。全部给她分析完，她说，爸爸，但这就是我的最爱，我认为我就是能做到机器不能做的。然后，我就支持她。 }}</ref>
