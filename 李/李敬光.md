'''李費蒙'''{{BD|1925年|4月25日|1997年|11月29日|catIdx=N}}本名'''李敬光'''，字費蒙，是[[台灣|台灣]]的[[漫畫家|漫畫家]]及[[小說家|小說家]]。出生於[[香港|香港]][[九龍|九龍]][[蒲崗村|蒲崗村]]，廣東番禺人。<ref>[http://blog.xuite.net/philipchiu99/hkblog/119210311-%E5%8F%B0%E7%81%A3%E6%BC%AB%E7%95%AB%E6%B3%B0%E6%96%97%E7%89%9B%E5%93%A5+-+%E6%9D%8E%E8%B2%BB%E8%92%99 台灣漫畫泰斗牛哥 - 李費蒙 @ 超哥網誌 :: 隨意窩 Xuite日誌]</ref>

==人物==
*牛哥擅長寫小說與畫漫畫，因出生於[[牛年|牛年]]，故取筆名為'''牛哥'''<ref>[http://blog.xuite.net/linmark.alumni/blog/162953515-%E7%89%9B%E5%93%A5%E6%BC%AB%E7%95%AB%E9%9B%86+%E5%9B%9E%E6%86%B6%E7%95%B6%E5%B9%B4+(Connie+Nieh+%E5%88%86%E4%BA%AB) 牛哥漫畫集 回憶當年 (Connie Nieh 分享) @ Linmark聯誼會與任江履昇女士清寒獎學金Blog歡迎您! :: 隨意窩 Xuite日誌]</ref>。他在年輕時曾為[[抗日|抗日]]當過[[間諜|間諜]]而被[[日本憲兵|日本憲兵]]逮捕，這項際遇也使得他日後在寫小說時有所幫助。牛哥的作品記錄了1950年代至1960年代的台灣社會，反映了當時的生活情況。李費蒙著名的[[漫畫|漫畫]]作品有《牛伯伯》、《牛小妹》，小說則有《職業兇手》、《賭國仇城》等；後者曾多次被改編搬上大銀幕，堪稱他最出名的代表作。
*牛哥在1949年結婚來台灣，但1955年結束了第一次婚姻。1956年，牛哥涉及[[鍾情|鍾情]]的[[妨害自由|妨害自由]]案件，入獄百天，出獄後自[[中國農村復興聯合委員會|中國農村復興聯合委員會]]離職。事件後，[[中國國民黨|中國國民黨]]台北市黨部去函各報，要求不要刊登牛哥的漫畫及小說，使牛哥作品一度遭各報封殺。<ref>[http://blog.xuite.net/philipchiu99/hkblog/119210311-%E5%8F%B0%E7%81%A3%E6%BC%AB%E7%95%AB%E6%B3%B0%E6%96%97%E7%89%9B%E5%93%A5+-+%E6%9D%8E%E8%B2%BB%E8%92%99 台灣漫畫泰斗牛哥 - 李費蒙 @ 超哥網誌 :: 隨意窩 Xuite日誌]</ref>1958年牛哥與馮娜妮（其父為[[馮庸|馮庸]]，與[[張學良|張學良]]是至交）結婚。1962年，牛哥開始收授徒弟，但並不收取學費，加入門下的徒弟包括[[趙寧|趙寧]]、[[林文義|林文義]]、[[洪義男|洪義男]]、林晉、小董、劉作泥、蔡東照、林世俊等人。1980年代時，牛哥發起「漫畫清潔運動」以抗議「漫畫審查制度」對於台灣漫畫家創作的打壓（盜版漫畫的審查明顯較寬鬆），他與政府機關、出版社開始了長達四年的訴訟官司。在這四年裡，他所有的創作都停止下來，但最終於1985年獲得了勝訴，使之後台灣新一代的漫畫家如[[朱德庸|朱德庸]]、[[敖幼祥|敖幼祥]]、[[鄭問|鄭問]]等可以出頭。
*1966年，牛哥開始負責《[[大華晚報|大華晚報]]》漫畫版的編輯工作，直到1987年該報停刊為止，他才放下這個編務工作。此漫畫版經營了二十年，對於台灣的漫畫發展具有貢獻。
*1980年，[[臺灣電視公司|臺灣電視公司]]將牛哥筆下漫畫人物搬上電視的[[綜藝節目|綜藝節目]]《[[牛伯伯俱樂部|牛伯伯俱樂部]]》不敵外界批評聲浪，播出五集以後正式停播，是台視僅次於1979年《百戰百勝》（播出三集以後停播）的第二個[[短命節目|短命]]綜藝節目。《牛伯伯俱樂部》是[[臺灣電視史|臺灣電視史]]上第一個將紙上人物搬上電視的節目，原始構想並非來自牛哥，但後來因[[權益|權益]]問題而由牛哥掛名製作人並且深入參與；牛哥給自己筆下漫畫人物設計造型、道具與服裝，但因沒有製作[[電視節目|電視節目]]的經驗，而且太著重於美工的表現，始終給人「用心用錯地方」的感覺，效果並不好<ref>{{cite news|url=|title=「牛伯伯俱樂部」不見了！ 播出僅五集，「牛伯伯」的命運便決定了，到底這是怎麼回事？|publisher=《電視綜合週刊》第214期|author=雨漣|date=1980-05-19|accessdate=2019-04-20|pages=118－120}}</ref>。
