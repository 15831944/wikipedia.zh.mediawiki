'''李仲三'''（{{bd|1881年||1969年|}}）名'''宗唐'''，[[陕西|陕西]][[潼关|潼关]]人，[[中华民国|中华民国]]及[[中华人民共和国|中华人民共和国]]政治人物。

==生平==
李仲三是[[清朝|清朝]]秀才。清末入[[新建陆军|新军]]一年，考入陕西师范学堂，后因参加学生运动而遭开除，革除秀才，当时他已加入[[中国同盟会|中国同盟会]]。1910年，受[[井勿幕|井勿幕]]派遣，到[[朝邑縣|朝邑]]联络[[刀客|刀客]]首领严孝全参加革命，参与[[大雁塔|大雁塔]]滴血盟誓。[[辛亥革命|辛亥革命]]爆发后，参加1911年10月22日西安起义，后任秦陇复汉军军政府东路安抚招讨使，同清军在[[陕西|陕西]][[蒲州|蒲州]]、[[潼关|潼关]]、[[阌乡|阌乡]]、[[灵宝|灵宝]]及[[山西|山西]]等地作战。后来卸去军职，到[[日本|日本]]两年，考察法政，又到[[留法勤工俭学|法国勤工俭学]]两年。1921年，任陕西[[靖国军|靖国军]]渭北水利委员会会长兼水利工程局局长，修建引[[泾河|泾]]灌溉工程。1924年，作为[[胡景翼|胡景翼]]的全权代表，联络[[冯玉祥|冯玉祥]]发动“[[北京政变|北京政变]]”。同年12月，任[[全国水利局|全国水利局]]副总裁<ref name=zgxa>{{Cite web |url =http://www.xa.gov.cn/websac/cat/975937.html  |title =  李仲三|author =  |date = 2014-04-10  |publisher = 中国西安 |language =  |accessdate =  |quote =  }}</ref>。1925年，任[[故宫博物院|故宫博物院]]临时董事会董事<ref name=tuandui>{{Cite web |url = http://www.dpm.org.cn/about/history_leader.html |title =  历任团队|author =  |date =  |publisher =故宫博物院  |language =  |accessdate = 2018-08-15 |quote =  }}</ref>。1925年，因“[[金佛郎案|金佛郎案]]”，领衔通电全中国声讨[[段祺瑞|段祺瑞]]，遭到通缉。[[北伐战争|北伐战争]]时，在[[广东|广东]]负责联络南北关系。1943年当选[[陕西省参议会|陕西省参议会]]议员，[[祝绍周|祝绍周]]主政陕西，将李仲三排挤出陕西省参议会，李仲三乃任西安市参议会议长<ref name=zgxa/>。他曾作为陕西代表参加过[[中国国民党|中国国民党]]在[[南京|南京]]、[[上海|上海]]、[[广州|广州]]召开的[[中国国民党全国代表大会|全国代表大会]]<ref name=wb>{{Cite web |url = http://epaper.xiancn.com/xawb/html/2011-10/23/content_58808.htm |title = 李仲三先生轶事|author =  |date =  2011-10-23|publisher =西安晚报  |language =  |accessdate =  |quote =  }}</ref>。[[中华人民共和国|中华人民共和国]]成立后，1952年加入西北历史文物研究会。1956年任西安市政协委员<ref name=zgxa/>。

1969年，李仲三在西安逝世，骨灰被其后人移葬到家乡<ref name=zgxa/><ref name=wb/>。
