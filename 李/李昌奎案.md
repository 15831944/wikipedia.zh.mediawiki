'''李昌奎案'''或称'''李昌奎故意杀人、强奸案'''，发生于2009年5月，[[云南省|云南省]][[昭通市|昭通市]][[巧家县|巧家县]][[茂租乡|茂租乡]]人李昌奎强奸、杀害同村女子王家飞，王家飞年仅三岁的弟弟王家红一同被杀。2010年7月，[[云南省昭通市中级人民法院|云南省昭通市中级人民法院]]一审判处李昌奎死刑立即执行。2011年3月，[[云南省高级人民法院|云南省高级人民法院]]（以下简称云南高院）二审（终审）改判李昌奎死缓。此案改判，是云南高院执行“少杀、慎杀”的典型<ref name="a3"/>，更企推动[[廢除死刑|废除死刑]]<ref name="a1">{{Cite web |url=https://news.qq.com/a/20110713/000001_2.htm |title=云南高院：“赛家鑫”案10年后将成标杆|accessdate=2018-11-28 |author=记者：刘子瑜，实习生：都力维 |date=2011-07-13|work= |publisher=腾讯网，来源：重庆晨报|language=Zh-hans }}</ref>。王家飞、王家红家属寻求云南省司法机关的司法救济无果，于6月在网络发帖救助，终引爆舆论<ref name="a2">{{Cite web |url=https://news.qq.com/a/20110803/000265.htm |title=李昌奎案办案法官：再审看似公正实则伤害法治|accessdate=2018-11-28 |author= |date=2011-08-03|work= |publisher=腾讯网，来源：四川新闻网|language=Zh-hans }}</ref>。云南高院副院长田成有表示，此案十年后将成为“一个标杆、一个典型<ref name="a1"/>”，更引舆论哗然。在重重压力下，云南高院于7月16日<ref name="a3">{{Cite web |url=http://news.ifeng.com/shendu/xsjzk/detail_2011_09/05/8942031_2.shtml |title=云南死刑变革进退维谷|accessdate=2018-11-28 |author=刘长 |date=2011-09-05|work= |publisher=凤凰网，来源：新世纪周刊|language=Zh-hans }}</ref>，宣布启动再审。8月22日，推翻二审（终审）判决，改判李昌奎死刑。9月29日，李昌奎在云南省昭通市被执行死刑。

== 二审争议 ==

云南高院指出二审改判不仅依据“少杀、慎杀”原则，主要认定李昌奎案属邻里纠纷，李又有自首情节，故云南高院认为“邻里纠纷引发的杀人案一般不判处死刑立即执行”。而“邻里纠纷引发的杀人案一般不判处死刑立即执行”的原则来源于1999年[[中华人民共和国最高人民法院|最高人民法院]]《全国法院维护农村稳定刑事审判工作座谈会纪要》{{efn|1999年《全国法院维护农村稳定刑事审判工作座谈会纪要》指出<ref>{{Cite web |url=http://dy.163.com/v2/article/detail/DPVTOA2K0519S96U.html |title=上海杀妻藏尸案判决死刑立即执行，为何南京吉星鹏杀妻案判死缓？|accessdate=2018-11-28 |author= |date=2018-08-24 |work= |publisher=网易|language=Zh-hans }}</ref>：“对于因婚姻家庭、邻里纠纷等民间矛盾激化引发的故意杀人犯罪，适用死刑一定要十分慎重，应当与发生在社会上的严重危害社会治安的其他故意杀人犯罪案件有所区别。对于被害人一方有明显过错或对矛盾激化负有直接责任，或者被告人有法定从轻处罚情节的，一般不应判处死刑立即执行。”}}。有学者指出，李昌奎案并不适用“邻里纠纷引发的杀人一般不判处死刑立即执行”规定。李昌奎和王家飞、王家红之间无直接纠纷。同时李昌奎与三岁幼儿王家红之间更不可能存在任何直接纠纷。他指出，云南高院的二审改判对“邻里纠纷”的引用，是“仅依据字面做了过于宽泛、教条化的理解。”<ref>{{cite journal en|quotes =

== 观点 ==

*[[国家检察官学院|国家检察官学院]]教授周洪认为，云南省高院的（二审）改判不合法理，也不符合中国当前的死刑政策。李昌奎案几乎没有什么可以对凶手不判处死刑立即执行的情节和条件，除非政府在一夜间突然宣布废除死刑<ref name="a2"/>。
*江西赣州检察院检察官杨涛撰文：“如果这么严重地杀害了两条生命的罪犯不判处死刑，那么，法院许多死刑判决都可以改判，特别是[[药家鑫|药家鑫]]{{efn|[[药家鑫案|药家鑫案]]中仅一人遇害。由于李昌奎犯案手段凶残，此案更被称为“赛家鑫案<ref name="a1"/>”。}}更是要在黄泉路上申冤了<ref name="a2"/>。”
