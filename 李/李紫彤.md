'''李紫彤'''是台灣的[[藝術家|藝術家]]、[[導演|導演]]，關注性別、政治、[[原住民|原住民]]、[[白色恐怖|白色恐怖]]、國族認同等議題。創作的作品以多媒體為主，結合網路、裝置、影像，也有多項參與式創作的藝術作品。<ref>{{cite web |title=色慾政治Ｘ數位皮囊──專訪藝術家鄭淑麗 |url=http://www.wmw.org.tw/tw/title/314}}</ref>

大學時期，李紫彤與朋友一同拍攝《擁抱》，獲得2010台大電影節徵影首獎。<ref>{{Cite journal|title=前往電影的路上on the way to cue. 喜歡上學的Vivian．李紫彤和她的夥伴們|author=李明璁|url=|journal=cue.電影生活誌|issue=第2期|doi=|others=|year=2010.7|volume=|page=70-75頁|pmid=}}</ref>拍攝場景之一取自暱稱[[洞洞館|洞洞館]]的台大建築，她們配合當年的建築物拆除計畫，拍下這部藝術電影。

2018年，由文化部贊助，與駐紐約臺北文化中心與紐約藝術機構Residency Unlimited (RU)合作辦理的「2018紐約Residency Unlimited創作發表計畫」，李紫彤與劉仁凱獲選為駐村藝術家，前往紐約四個月進行藝術創作，並於2018年9月以《形式的不可能性》(The Impossibility of Form)為題，進行創作成果展。<ref>{{cite web |title=公告「2018紐約Residency Unlimited創作發表計畫」甄選結果-中華民國文化部 |url=https://www.moc.gov.tw/information_250_80954.html}}</ref><ref>{{cite web |title=台灣藝術家展現紐約駐村創作成果 《形式的不可能性》九月十五日揭幕 -紐約臺北文化中心 |url=https://ny-ch.us.taiwan.culture.tw/information_106_88955.html}}</ref><ref>{{Cite web|title=The Impossibility of Form {{!}} artistsallianceinc|url=http://artistsallianceinc.org/cuchifritos-gallery-2/past-exhibitions/the-impossibility-of-form|accessdate=2019-03-31|language=en-US}}</ref>成果展作品《#迎靈者》(#GhostKeepers)，是一項{{le|參與式藝術|Participatory art}}，在世界各地尋找70年前，在不同國家發生戰後、[[白色恐怖|白色恐怖]]後等情況的[[受害者|受害者]]或加害者，與這些鬼魂對話，並開立這些鬼魂的社群媒體帳號，呈現他們死後如何面對自己國家，以及讓一般人去與其他鬼魂互動。<ref>{{cite web |title=2013-2018 作品 Lee Tzu-Tung 李紫彤 |url=https://www.tzutung.com/2013-2018}}</ref>

最新作品為《時差書寫》(Writing the Time Lag)，是一部自2014年開始拍攝的實驗民族誌，探討國族認同，預計於 2019 年完成。這段期間，她深入探訪各政治團體如[[台灣人公共事務會|台灣人公共事務會]]<ref>{{Cite web|title=台灣人公共事務會伊利諾卅分會舉辦2016年會|url=https://chicagochinesenews.wordpress.com/2016/11/03/%e5%8f%b0%e7%81%a3%e4%ba%ba%e5%85%ac%e5%85%b1%e4%ba%8b%e5%8b%99%e6%9c%83%e4%bc%8a%e5%88%a9%e8%ab%be%e5%8d%85%e5%88%86%e6%9c%83%e8%88%89%e8%be%a62016%e5%b9%b4%e6%9c%83/|work=芝加哥時報|date=2016-11-03|accessdate=2019-03-31|language=en|last=thechicagochinesenews}}</ref>、[[時代力量|時代力量]]、[[民主進步黨|民主進步黨]]等，並加入組織參與活動，親身觀察政治活動；同時也到太巴塱、港口等部落拍攝紀錄片。除了政治與原住民議題之外，本片也進行多項藝術實驗，包括打破電影圈的性別問題，因此《時差書寫》的劇組人員全部都是女性；以及讓受訪者也是拍攝者的{{le|參與式影片|Participatory video}}。<ref>{{cite web |title=Lee Tzu-Tung 李紫彤 Writing the Time Lag 時差書寫 (2018) |url=https://www.tzutung.com/writing-the-time-lag}}</ref>
