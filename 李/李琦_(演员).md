'''李琦'''（{{bd|1955年|2月17日||}}）生于[[山西|山西]][[大同|大同]]，[[中华人民共和国|中华人民共和国]]演员。

==生平==
李琦只上到小学四年级便辍学<ref>{{Cite web |url = http://epaper.hf365.com/jhcb/html/2017-02/22/content_271501.htm |title =“打死我也不说”的李琦这次畅所欲言了  |author =  |date = 2017-02-22 |publisher = 江淮晨报 |language =  |accessdate =  |quote =  }}</ref>。1970年，李琦考入[[陕西人民艺术剧院|陕西省人民艺术剧院]]，考官为[[朱龙广|朱龙广]]等人<ref name=gmw>{{Cite web |url = http://www.gmw.cn/01wzb/2005-09/08/content_301714.htm |title =“高传宝”细说舞台缘  |author =  |date = 2005-09-08 |publisher = 光明网|language =  |accessdate =  |quote =  }}</ref>。1979年，凭借在话剧《白居易在长安》中饰演的配角[[太监|太监]]获得中国戏剧“[[梅花奖|梅花奖]]”。1991年左右，来到[[北京|北京]]当自由演员。1995年，在[[中国中央电视台|中国中央电视台]]《[[综艺大观|综艺大观]]》第26期和[[郭达|郭达]]合作演出小品《打呼噜》，这是李琦首次上中国中央电视台的节目。后来，中国中央电视台导演陈平把李琦带进由他执导的栏目《日月星辰》文艺晚会，安排李琦和[[倪萍|倪萍]]合作演出一个小品。[[陈佩斯|陈佩斯]]在北京有线电视台开办《佩斯快乐岛》节目，找来李琦合作，让李琦当演员、制片。1997年，李琦在[[冯小刚|冯小刚]]导演的电影《[[甲方乙方|甲方乙方]]》中饰演胖厨师，由此出名。1998年，李琦入伍加入[[中国人民武装警察部队政治工作部文工团|武警文工团]]。此后，李琦先后与冯小刚、[[英达|英达]]等导演合作，接拍了《[[有话好好说|有话好好说]]》、《[[中国餐馆|中国餐馆]]》、《[[东北一家人|东北一家人]]》、《大话股神》等影视剧，成为知名演员。[[2001年中国中央电视台春节联欢晚会|2001年中国中央电视台春节联欢晚会]]上，李琦与[[刘金山|刘金山]]、[[尹相杰|尹相杰]]、[[刘惠_(相声演员)|刘惠]]合作演唱《[[康定情歌|康定情歌]]》<ref name=xiaopin>{{Cite web |url =http://news.sohu.com/20071024/n252819794.shtml  |title =李琦：我不是小品演员！  |author =  |date =  2007-10-24|publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.1905.com/mdb/star/3275/|title = 李琦 |author =  |date =  |publisher =  电影网|language =  |accessdate = 2017-04-18 |quote =  }}</ref><ref>{{Cite web |url =http://ent.ifeng.com/a/20160820/42670392_0.shtml  |title = 李琦曾出演《东北一家人 》 刘金山因《小兵张嘎》走红 |author =  |date = 2016-08-20 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
