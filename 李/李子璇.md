'''李子璇'''（{{bd|1995年|5月7日}})，出生於[[湖南省|湖南省]][[湘潭市|湘潭市]]，[[中國|中国]]流行音乐女歌手、舞者，[[覺醒東方|覺醒東方]]旗下藝人。2016年6月，參加[[东方卫视|東方衛視]]綜藝節目《[[加油！美少女|加油！美少女]]》，獲得了總決賽冠軍。2018年4月，參加[[腾讯视频|騰訊視頻]]青春成長類綜藝節目《[[創造101|創造101]]》，於第二期節目中獲得主題曲C位爲人熟知。6月23的總決賽中，獲得第12名，離出道位僅一步之遙。

== 經歷 ==

李子璇從4嵗開始學習舞蹈，母親是一名舞蹈教師，家教嚴厲。其曾在節目中坦言“（在學校）不讓我叫媽媽，讓我叫她張老師”。曾于以軍事化管理著稱的中國[[南方舞蹈學校|南方舞蹈學校]]學習舞蹈。17岁的李子璇曾在[[北京星舞團|北京星舞團]]做伴舞和舞蹈老師，而後放棄中央民族大學的保送資格，赴韓國做練習生<ref>{{cite web |url= http://news.cnr.cn/dj/20180711/t20180711_524297991.shtml|title=【奋斗的中国人】李子璇：用奋斗书写青春的可能性 |accessdate= 2018-07-11|publisher=央广网}}</ref>。曾在[[Jellyfish娛樂|Jellyfish娛樂]]與[[SALLY|劉些寧]]為同期練習生。

2016年6月11日，參加[[东方卫视|東方衛視]]勵志成長綜藝節目《[[加油！美少女|加油！美少女]]》。最終于8月27日的總決賽中，與其他5名選手一起获得“Diamond Girl”称号，並獲得總決賽冠軍。此後因沒有後續安排，她又重拾伴舞工作<ref>{{cite web |url= https://www.weibo.com/ttarticle/p/show?id=2309404248730910987539 |title= 关于 豆子 —李子璇 |accessdate= 2018-06-08|publisher= [[倪秋雲|倪秋雲]]}}</ref>，曾在國内各大電視臺為明星伴舞。12月，与歌手徐林合作发行單曲《oi》。

2018年4月，和[[覺醒東方|覺醒東方]]藝人[[倪秋雲|倪秋雲]]、[[林君怡|林君怡]]一起組成臨時組合DS98，參加[[腾讯视频|騰訊視頻]]青春成長類綜藝節目《[[創造101|創造101]]》並獲評A等級，於第二期節目中獲得主題曲中心位置（C位）爲人熟知。6月23的總決賽中，獲得第12名，離出道位僅一步之遙。
