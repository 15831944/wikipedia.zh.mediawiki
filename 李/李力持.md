'''李力持'''（{{lang|en|'''LEE，Lik Chi'''}}，{{bd|1961年|5月10日|catIdx=L李}}），被暱稱為「'''大力'''」，[[香港電影|香港電影]][[導演|導演]]、[[編劇|編劇]]、[[演員|演員]]。

==背景==
李力持小時候居住在[[九龍|九龍]][[土瓜灣|土瓜灣]][[山谷道邨|山谷道邨]]<ref>{{web reference|URL=http://paper.wenweipo.com/2016/04/18/CF1604180004.htm|title=淑梅足跡：大力的電影路|work=香港文匯報|year=2016年|date=4月18日}}</ref>12座。他是何文田海星學校第14屆畢業生（1967年至1973年）。1978年，李力持畢業於[[新亞中學|新亞中學]]，為該校第一屆校友，後於1980年在該校完成[[預科|預科]]課程。他曾於[[香港中學會考|香港中學會考]]中文科和數學科取得A級成績。預科畢業後當過倉務員。其後在1982年加入[[香港亞洲電視|香港亞洲電視]]任職助理編導，之後轉投[[無綫電視|無綫電視]]任編導<ref>{{cite web|url=http://www.hkswg.com/member_detail.asp?id=96|title=會員資訊|publisher=香港電影編劇家協會|deadurl=yes|archiveurl=https://web.archive.org/web/20120912072858/http://www.hkswg.com/member_detail.asp?id=96|archivedate=2012-09-12}}</ref>。他在無綫時的代表作有《[[蓋世豪俠|蓋世豪俠]]》和《[[他來自江湖|他來自江湖]]》，都是由[[周星馳|周星馳]]主演。因著與周星馳合作之成功，李力持開始被電影公司招攬，夥拍周星馳拍攝大量電影，包括《[[破壞之王|破壞之王]]》、《[[國產零零漆|國產零零漆]]》、《[[唐伯虎點秋香_(電影)|唐伯虎點秋香]]》、《[[食神_(電影)|食神]]》、《[[喜劇之王_(香港電影)|喜劇之王]]》、《[[少林足球|少林足球]]》等，曾是喜劇界「黃金組合」。<ref>[http://www.chinapress.com.my/node/363017 李力持 幽默背后 ] {{webarchive|url=https://web.archive.org/web/20131111101239/http://www.chinapress.com.my/node/363017 |date=2013-11-11 }} 中國報 19 Oct 2012</ref>

李力持自《[[少林足球|少林足球]]》後便沒有再與[[周星馳|周星馳]]合作，有傳是二人因拍攝手法鬧不和所致，亦有指因李力持能力不足，被周星馳棄用。<ref>李力持撐政改盡顯無知/ 王國興公然違反廣播條例 / 中共無意通過政改？〈蕭遙遊〉2015-04-27 d</ref>。此後，李力持所執導電影的質素每況愈下，不少香港觀眾均譏諷李力持為「爛片王」，其中《[[唐伯虎點秋香2|唐伯虎點秋香2]]》更被指拍攝超期超支，使電影公司感不滿，據資深傳媒工作者，著名電視、電影編劇蕭若元在網台透露，「早5、6年前，某人上大陸拍戲，以為自己係大導演。上到去先搞好所有劇本，日日係酒店房內，結果嚴重超支，最後比人摑了兩巴收場。」<ref>李力持撐政改盡顯無知/ 王國興公然違反廣播條例 / 中共無意通過政改？〈蕭遙遊〉2015-04-27 d  https://www.youtube.com/watch?v=1w-jb0koOKM  </ref>

近年李力持成為[[Umovie|Umovie]][[董事|董事]]及[[製作總監|製作總監]]，主力開班講課，教授短片製作。

於2010-2011年度微笑企業大獎頒獎典禮擔任微笑大使，頒獎予港澳地區多個於微笑服務表現優異的企業。亦於2015年度擔任微笑員工評審。<ref>[https://www.youtube.com/watch?v=urGZaqZDtOM 微笑企業大獎領獎典禮 2015]</ref>

2015年1月，根據[[肇慶市|肇慶市]]人民政府之網站上的公佈，李力持與前[[民主黨_(香港)|民主黨]][[葵青區|葵青區]]區議員[[周奕希|周奕希]]雙雙成為肇慶市增補[[政協委員|政協委員]]<ref>{{cite news|url=http://www.zhaoqing.gov.cn/jjzq2011/xwdt/zqyw/201501/t20150123_274318.html|title=市政協召開十屆十一次常委會議，決定2月4-5日召開市政協十屆四次會議|publisher=肇慶市人民政府官方網站|date=2015-01-23|accessdate=2015-08-20}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite news|url=http://cppcc.zhaoqing.gov.cn/wyfc_6660/201403/t20140312_233719.html|title=中國人民政治協商會議肇慶市第十屆委員會委員名單|publisher=政協肇慶市委員會官方網站|accessdate=2015-08-20|deadurl=yes|archiveurl=https://web.archive.org/web/20150912082937/http://cppcc.zhaoqing.gov.cn/wyfc_6660/201403/t20140312_233719.html|archivedate=2015-09-12}}</ref>。

== 電台主持 ==
主持方面，李力持於1997年進軍[[電台|電台]]，與[[吳君如|吳君如]]合作主持受歡迎節目《嘩嘩嘩！打到嚟》，同年奪得「最受歡迎電台節目主持人」第七位。亦曾經於[[香港商業電台|商業二台]]主持節目《你唔好去死》及《星期日朝你得我》。自1999年10月開始亦為[[香港電訊互動電視|香港電訊互動電視]]（ITV）監製並主持娛樂資訊節目《娛樂跨媒睇》（每星期製作一輯，每輯半小時）。

2004年，李力持開始於[[香港電台|香港電台]]與[[曾華倩|曾華倩]]合作主持《李真超人》，2006年與[[車淑梅|車淑梅]]合作主持港台節目《[[開心日報|開心日報]]》。

2008年，他與[[吳君如|吳君如]]和[[阮子健|阮子健]]在星期三在《嘩嘩嘩打到嚟！2008》以嘉賓主持出現。

== 報章雜誌專欄 ==
李力持的寫作層面亦涵蓋甚廣。曾為《[[太陽報|太陽報]]》副刊《唔系講笑》、《[[經濟日報|經濟日報]]》副刊《寫意空間──跨媒體創意》、《[[蘋果日報|蘋果日報]]》副刊《力持己見》及《[[成報|成報]]》副刊《力持己見》撰寫專欄，及每週為電腦IT資訊雜誌《電腦廣場》撰寫有關AV數碼產品文章。

現在李力持每天依然為《[[頭條日報|頭條日報]]》撰寫[[笑話|笑話]]，及為雙月刊攝影雜誌《JPG》撰寫專欄。

== 書籍出版 ==
出版方面，於2000年1月首次自資出版書籍《真係笑話》前篇及後篇，銷量已超越10萬本。其後于2001年7月成立「力出版有限公司」，出版不同類型的暢銷書籍，包括暢銷系列《吳君如減肥血淚史》及《[[推背圖|推背圖]]》。同時，《真係笑話》系列銷量衝破三十萬冊，成為香港有史以來最暢銷笑話系列叢書。除《真係笑話》外，李力持亦有嚴肅的散文作品，如《開心的牛》、《開心的接班人》及《逆境自強·截擊挫敗》[[VCD|VCD]]等。
