'''李陸史'''（{{lang-ko|이육사}}，{{bd|1904年|5月18日|1944年|1月|16日|}}）是[[韓國獨立運動|韓國抗日]][[诗人|诗人]]、活動家。原名為'''李源祿'''（{{lang|ko|이원록}}）、'''李源三'''（{{lang|ko|이원삼}}），[[祖籍|本貫]]是{{link-ko|眞寶李氏|진보_이씨}}。“李陸史”源於他在{{link-ko|大邱矯導所|대구교도소|大邱刑務所}}時的收監號“264”（[[朝鲜语|朝鲜语]]中“264”和“李陸史”都[[朝鮮漢字音|音读]]为“{{lang|ko|이육사}}”）。他被選為1995年7月的韓國[[本月文化人物|本月文化人物]]。
== 生平 ==
李陸史出生於[[庆尚北道|庆尚北道]][[安东市|安东市]]{{link-ko|陶山面|도산면_(안동시)}}原寸洞，是16世紀[[李氏朝鲜|朝鮮]][[程朱理学|朱子學者]][[李滉|李滉]]的第14代後人。<ref name="saigusa">{{cite web |url=http://www.han-lab.gr.jp/~cham/ajiwau/chap5/chap5.html|title=三枝寿勝の 「韓国文学を味わう」 第 V 章 近代朝鮮の詩人たち|language=ja|access-date=2017-11-29|quote=}}</ref>

1919年，15歲的李源祿在安東完成了基礎教育。<ref name="ansonjae">{{cite web |url=http://hompi.sogang.ac.kr/anthony/YiYukSa.html|title=Yi Yuk-sa: One hundred years|author=An Sonjae|year=2004|language=en|trans-title=李陸史:一百年|access-date=2017-11-29}}</ref>1920年，他隨家庭搬到[[大邱|大邱]]，在那當教師。1924年赴日本留學。<ref name="observer.yu.ac.kr">嶺南大學觀察{{cite web |url=http://observer.yu.ac.kr/news/quickViewArticleView.html?idxno=171 |title=存档副本 |accessdate=2013年5月20日 |deadurl=yes |archiveurl=https://web.archive.org/web/20140323135847/http://observer.yu.ac.kr/news/quickViewArticleView.html?idxno=171 |archivedate=2014年3月23日 |df= }}</ref>

1925年和兄弟一起參加韓國獨立運動組織[[義烈團|義烈團]]。<ref name="ansonjae" />1926年到北京的[[中國大學|中國大學]]學習社會科學，后到廣州[[中山大学|中山大学]]學習。1927年回到大邱，因為{{link-ko|張鎭弘|장진홍}}引發的朝鮮銀行大邱分行爆炸事件而受到[[連坐|連坐]]，進入大邱矯導所，被關押了18個月後，1929年5月才得到釋放。<ref name="ansonjae" />

1931年1月因為大邱檄文事件而被監禁，3月被釋放。

1932年進入朝鮮革命軍事政治学校學習，接受了軍事訓練，次年4月畢業。研究表明他可能在這個時期和[[鲁迅|鲁迅]]見過面。

1944年1月16日凌晨，李陸史在駐北京日本總領事館監獄中被殺害。
== 死後 ==
後來他的尸體被同志帶走火化，埋葬在[[首爾|首爾]]。<ref name="ansonjae" />1960年又重新葬回其故鄉安東，1968年在那立了紀念碑。

1946年其弟[[李源朝|李源朝]]為其出版了《陸史詩集》。

2004年，爲紀念他誕辰100周年和殉國60周年，安東市陶山面建立了“李陸史文學館”。<ref>Guidebook of Andong City https://www.msu.edu/~jeongeu4/DC2009/images/Andong%20Information.pdf</ref>

2011年，韓國[[文化廣播公司|文化廣播公司]]播出電視劇《絕頂》（{{lang|ko|절정}}，取名自李陸史最出名的詩作），[[金烔完|金烔完]]扮演以李陸史為原型的主人公。<ref>{{cite web |url=http://news.jkn.co.kr/articles/661889/20110811/%EA%B4%91%EB%B3%B5%EC%A0%88-%ED%8A%B9%EC%A7%91%EA%B7%B9-%EC%A0%88%EC%A0%95-%EC%9D%B4%EC%9C%A1%EC%82%AC-%EA%B9%80%EB%8F%99%EC%99%84-%EC%A0%88%EC%A0%95-%EC%9D%80-%EB%B8%94%EB%A1%9D%EB%B2%84%EC%8A%A4%ED%84%B0-%EC%98%81%ED%99%94%EB%B3%B4%EB%8B%A4-%EB%8D%94-%EB%B3%BC%EB%A7%8C%ED%95%9C-%EB%93%9C%EB%9D%BC%EB%A7%88.htm|title=[광복절 특집극 절정] 이육사 김동완, “‘절정’은 블록버스터 영화보다 더 볼만한 드라마”|year=2011-08-11|language=ko|access-date=2017-11-29}}</ref>
