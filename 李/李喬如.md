'''李喬如'''（Lee Chiao-Ju，{{bd|1957年|6月24日|||}}），[[臺灣|臺灣]][[政治人物|政治人物]]、[[民進黨|民進黨]]籍議員，出生於屏東縣，畢業於[[國立中山大學|國立中山大學]]修讀高階公共政策研究所碩士，1987年開始投身[[民主運動|民主運動]]<ref>{{Cite web |url = http://forums.chinatimes.com/report/vote2002/result/candidate/200108.htm  |title = 2002雙城爭霸  |author =  |date = 2001年8月 |publisher = 中國時報 |language = zh-hant |accessdate = 2015年10月14日 |quote = 民國七十六年參與民主政治運動，任職市議員、立法委員助理}}</ref>，曾經參與1993年[[高雄|高雄]][[三一四事件_(高雄)|三一四事件]]，至1994年12月25日開始連任第4、5、6、7屆[[高雄市議會|高雄市議會]][[議員|議員]]，2010年年[[2010年中華民國縣市改制直轄市|高雄縣市合併]]之後，繼續擔任鹽埕、旗津、鼓山區議員至今。

== 服務政見 ==
*國小兒童營養午餐免費並擴大至國中比照辦理
*爭取更多婦女就業保障權力及公里托育服務
*主張引進高科技龍頭產業並帶動相關下游產業
*廣建青年住宅提供低價、低利率租屋或購屋方案

== 爭議事件 ==
*2003年1月23日，[[高雄市議會|高雄市議會]]議長[[朱安雄|朱安雄]]夫婦涉嫌賄選，檢調單位傳喚李喬如等11名議員<ref>{{Cite web |url = http://old.ltn.com.tw/2003/new/jan/23/today-p8.htm  |title = 高市議長賄選案 六議員交保 |author = 鮑建信、戴上茹  |date = 2003年1月23日 |publisher = 自由時報 |language = zh-hant |accessdate = 2015年10月16日 |quote = 高雄市議長朱安雄夫婦涉嫌賄選案，檢調單位昨天傳喚李喬如等十一名議員，承辦檢察官蕭宇誠偵訊後，李喬如等六名議員各以卅萬元交保，另五名議員請假未露面。}}</ref>，9月9日法官宣判無罪<ref>{{Cite web |url = http://news.tvbs.com.tw/old-news.html?nid=402045 |title = 賄選判無罪　高議員李喬如激動哽咽 |author = 武紹隆 |date = 2003年9月9日 |publisher = 自由時報 |language = zh-hant |accessdate = 2015年10月16日 |quote = 高雄市議長賄選案，今天第三波宣判，前民進黨團總召李喬如被判無罪，她是到目前為止第一個判無罪的市議員。 }}</ref>。
*2012年5月24日，李喬如在市政質詢時，建議市政府應該針對超過30歲卻沒結婚生子的公務員，應給他們考績丙等<ref>{{Cite web |url = http://news.pts.org.tw/detail.php?NEENO=211106 |title = 綠議員言論 陳菊:封建思想難認同 |author = 王介村、孟昭權、許政俊 |date = 2012年5月24日 |publisher = 公共電視台 |language = zh-hant |accessdate = 2015年10月16日 |quote = 高雄市議會進行市政質詢，民進黨籍的議員李喬如，談到低生育率問題時，卻語出驚人，建議30歲公務員不婚不生，考績應該丙等。}}</ref>，另外30至58歲間不結婚不生子的人，也不能參選公職。市長陳菊回覆：「這款處罰不結婚不生子的思想，無法認同。」
*2013年5月30日，議員提案請市政府增加捕狗大隊人力、並建議夜間抓狗，因提案文件之案由寫著「以免影響市民安眠及成為城市之瘤<ref>{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20130530/35050948/  |title = 促捕狗 議員稱「滅城市之瘤」 |author = 涂建豐  |date = 2013年5月30日 |publisher =  |language = zh-hant |accessdate = 2015年10月16日 |quote = 議員李喬如最近提案建請市府增加捕狗大隊人力，調整為二十四小時作業，並建議夜間捕狗，以免流浪狗「影響市民安眠及成為城市之瘤。」動保團體反彈痛批李喬如沒同理心，其臉書也遭洗版。高雄市動保處認為，夜間視線不良，捉不到流浪狗，反而影響安寧。 }}</ref>」被動保團體痛批沒同理心。
*2013年12月13日，議員為籌設中的[[美術館國小|美術館國小]]校地請命，指摘鄰近的[[高雄市私立中華藝術學校|中華藝術學校]]不配合、還協助製作反對設校部分文宣，且校方10餘年來均未付費使用「藝片天廣場」，她要中華藝術學校「皮繃緊點<ref>{{Cite web |url = http://www.epochtimes.com/b5/13/12/6/n4028305.htm國小校地爭議--台議員槓中華藝校.html |title = 國小校地爭議 台議員槓中華藝校 |author = 程啟峰 |date = 2013年12月6日 |publisher = 大紀元日報 |language = zh-hant |accessdate = 2015年10月16日 |quote =李喬如並在議事廳內質詢時指控中華藝校因有附設雙語國小而反對美術館國小成立，校方還協助製作反對設校部分文宣，且校方10餘年來均未付費使用「藝片天廣場」，她要校方「皮繃緊點」。}}</ref>」。
*2014年11月14日，市議員候選人[[連立堅|連立堅]]公布4年前議長選舉時李喬如的投票影片暗指­她跑票<ref>{{Cite web |url = http://m.ftv.com.tw/newscontent.aspx?sno=2014B14U11M1 |title = 同志內鬨 連立堅.李喬如槓上法院 |author = |date = 2014年11月14日 |publisher = 民視新聞 |language = zh-hant |accessdate = 2015年10月16日 |quote =選戰倒數15天，民進黨高雄市議員選舉出現同志內鬨，而且越演越烈，­甚至摃上法庭！候選人連立堅公布4年前議長選舉時，同黨籍議員李喬如的投票影片，暗指­她跑票，李喬如告連立堅妨礙祕密罪，連立堅也反擊，下午到地檢署控告李喬如毀謗，雙方­箭拔弩張。}}</ref>，李喬如告連立堅妨礙祕密、連立堅也到地檢署控告李喬如毀謗。
