'''李子誦'''（{{lang-en|'''Lee Tze Chung'''}}，原名'''李頌'''，{{bd|1911年|5月21日|2012年|5月11日|catIdx=Li李子誦}}），[[廣東|廣東]][[順德|順德]]人，曾任五屆政協常務委員（包括首屆）及第七屆政協常委；1951年起於《[[文匯報_(香港)|文匯報]]》任職達38年，「[[六四事件|六四事件]]」爆發前夕，北京爆發學生運動，1989年5月21日，[[中华人民共和国政府|中国政府]]在北京實施[[戒嚴|戒嚴]]，李子誦等人於《文匯報》開天窗刊出「痛心疾首！」作為社論，兩個月後遭中国政府秋後算賬，李子誦等人被解除職務。

==生平==
李子誦16歲時父親離世，沒有小學畢業的李加入[[廣州|廣州]]新聞界，並在圖書館自學<ref>{{cite web |language=zh-hant| author=[[劉銳紹|劉銳紹]] |title=健哉李子誦社長 |page=AA04 |publisher=新報 |date=2007-04-25 }}</ref>；先後於《[[大公報|大公報]]》粵南版、[[韶關|韶關]]《[[建國日報_(韶關)|建國日報]]》、[[香港|香港]]《[[華僑日報|華僑日報]]》等新聞機構任職校對及編輯工作<ref name="oncc">{{cite web |language=zh-hant |url=http://orientaldaily.on.cc/cnt/news/20120513/00176_018.html | title=前文匯報社長李子誦逝世 |publisher=東方日報 |date=2012-05-13 }}</ref>。

1949年9月，李子誦出席[[中國人民政治協商會議第一屆全體會議|中國人民政治協商會議第一屆全體會議]]；他於兩年後出任香港《文匯報》總編輯，及至1978年出任社長一職<ref name="oncc" />。《文匯報》、《大公報》等三份報章於1951年轉載一篇《[[人民日報|人民日報]]》評論，[[港英政府|香港政府]]以「刊載煽動性文字」逮捕了李子誦、《大公報》社長[[費彝民|費彝民]]等人，稱為「三一事件」<ref>{{cite web |language=zh-hant| title=澳門新華社小插曲 |page=C15 |publisher=經濟日報 |date=2000-01-15  }}</ref>。於1959年起出任第三、四、五、六、七屆全國政協委員；而在出任等七屆全國政協委員時更是政協常委<ref name="resume">{{cite web |language=zh-hant |url=http://paper.wenweipo.com/2010/06/01/AY1006010001.htm | title=文匯同仁賀李子誦百歲榮壽 |page=A19  |publisher=文匯報 |date=2010-06-01 }}</ref>。

[[File:痛心疾首！.jpg|thumb]]]]
[[六四事件|六四事件]]前夕，李子誦與總編輯[[金堯如|金堯如]]相討並得到[[新華社香港分社|新華社香港分社]]副社長[[張浚生|張浚生]]同意下決定以開天窗「痛心疾首！」四個大字作為文匯報在5月21日的社論，以表達不滿中共當局[[六四事件#.E9.A0.92.E5.B8.83.E6.88.92.E5.9A.B4.E4.BB.A4|宣布戒嚴]]<ref name="bio">{{cite web |language=zh-hant |url=https://www.facebook.com/notes/毋忘六四/痛心疾首六四前決定文匯社論-開天窗-李子誦逝世/10150667377538078/ | title=六四前決定文匯社論「開天窗」李子誦逝世 |page=A04 |publisher=明報 |date=2012-05-12 }}</ref><ref>{{cite web |language=zh-hant| title=89年「開天窗」社論 文匯獲張浚生同意 |page=A10  |publisher=明報 |date=2011-09-03 }}</ref>；此外，李子誦以「有喜報喜，有憂報憂；有善必揚，有惡不隱」作為《文匯報》作為處理六四事件發生前後的新聞宗旨<ref name="Fu">{{cite web |language=zh-hant |url=https://hk.news.appledaily.com/local/daily/article/20090520/12779154 | title=《文匯報》社論開天窗剎那人性光輝閃現　夫子憶述兩個方案 |page=A22 |publisher=蘋果日報 |date=2009-05-20 }}</ref>；由於文匯報是[[中國共產黨|中國共產黨]]在香港的官方喉舌之一，社論發表後立即引起廣泛迴響外，也引領香港其他左派報章及中立報章譴責中共對學生運動的定性與戒嚴等處理手法<ref>{{cite web |language=zh-hant| title=文匯報開天窗更激進 |page=P10  |publisher=信報 |date=2007-06-27 }}</ref>；李更分別於6月4日及7月7日的記者會上痛哭地譴責中共鎮壓行為<ref name="bio" />。新華社香港分社副社長張浚生於7月13日親自到文匯報社以「停止留用李子誦」來解除李子誦的社長職務<ref name="Fu" /> 。副總編輯[[程翔|程翔]]、總編輯[[金堯如|金堯如]]、記者[[劉銳紹|劉銳紹]]亦先後被迫離開《文匯報》。

1991年，李子誦與程翔創辦《[[當代_(雜誌)|當代]]》時事周刊，惟於四年後因虧損而停辦<ref name="bio" />。香港資深傳媒人員聯誼會於2010年3月17日成立，他成為榮譽會長<ref>{{cite web |language=zh-hant |url=http://paper.wenweipo.com/2010/03/18/AY1003180003.htm | title=深傳媒人員聯誼會成立 |page=A12 |publisher=文匯報 |date=2010-03-18 }}</ref>。2011年被評選為[[2011感動香港|2011感動香港]]26位年度感動香港人物之一<ref>{{cite web |language=zh-hant| title=六四「開天窗」社長 李子誦良心贈亞視 |page=A19  |publisher=明報 |date=2011-09-12 }}</ref>。2012年5月11日早上在香港[[東華三院馮堯敬醫院|東華三院馮堯敬醫院]]以一百歲高齡離世<ref name="bio" />。

治喪委員會在2012年6月11日下午4時在北角香港殯儀館為李子誦設靈，翌日上午11時半舉行公祭及大殮，遺體其後往柴灣[[哥連臣角火葬場|哥連臣角火葬場]]火化。

==家庭==
李子誦育有三子一女：長子李淮濟、次子李邕復（已婚，其太太為張榮華）、三女李寸金（已婚，其丈夫是趙如莊，並育有一子趙崇安）、四兒李松興（已婚，其太太是陳潔冰）<ref name="resume" />。
