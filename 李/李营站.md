'''李营站'''是一个[[京九线|京九线]]上的铁路车站，位于[[北京市|北京市]][[大兴区|大兴区]][[黄村镇|黄村镇]]李营村，建于2001年，目前为[[五等站_(中国大陆)|五等站]]，邮政编码为102627。不办理旅客乘降。[[京沪高速铁路|京沪高速铁路]]在该站一带与京九线几乎平行，但并不在李营设站。

京九线在该站以北曾为单线铁路，直至2007年1月14日西黄复线铺通。<ref>{{cite news|url=http://news.sina.com.cn/c/2007-01-15/121211004759s.shtml|title=京九列车进出京 结束“单车道”历史|publisher=[[法制晚报|法制晚报]]|date=2007-01-15|accessdate=2015-04-04}}</ref>该站北侧亦有通往[[丰双铁路|丰双铁路]][[大红门站_(铁路)|大红门货运站]]的联络线（称“大李联络线”）<ref>{{cite journal|author=白宝英|coauthors=杨卫盛|title=利用铁路枢纽线路开行市郊列车的有关探讨|journal=铁道工程学报|url=http://xuewen.cnki.net/CJFD-TDGC201401002.html|year=2014|issue=01|issn=1006-2106|pages=10}}</ref>。

2017年7月，[[中国铁路设计集团|中国铁路设计集团]]发布北京至雄安铁路的环评公告，内称[[京雄铁路|京雄铁路]]起自北京李营站，线路全长100多公里，若能顺利通过环评，将可在2019年建成<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1742674 |title = 探访京雄铁路起点李营：南五环外的五等站，周边房价近6万 |author =  |date = 2017-07-26 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。
