'''李元甫'''{{BD|1872年||1941年||catIdx= L李元甫}}名'''树人'''，出生于[[湖南省|湖南省]][[湘乡市|湘乡市]]，毕业于[[湖南优级师范|湖南优级师范]]，为[[中华民国|中华民国]]著名[[教育家|教育家]]，培育了[[毛泽东|毛泽东]]、[[谭政|谭政]]、[[陈赓|陈赓]]、[[萧三|萧三]]、[[萧子升|萧子升]]等时代风云人物。<ref name="lsa ">{{cite web|url=http://www.hnfnu.edu.cn/mao116th/Content.aspx?nid=11783&page=1|title=发现毛泽东是“建国之才”的校长——毛泽东与李元甫先生|accessdate=2010年|work=|publisher=湖南历史网|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160304114313/http://www.hnfnu.edu.cn/mao116th/Content.aspx?nid=11783&page=1|archivedate=2016年3月4日|df=}}</ref>

== 生平 ==
李元甫儿时聪敏好学，20岁[[弱冠|弱冠]]之年考中了[[进士|进士]]，后来在[[湖南优级师范学堂|湖南优级师范学堂]]读大学。<ref name="lsa " /> 毕业后回到家乡[[东山学校|东山学校]]教书育人，培育了一批西式人才。<ref name="lsa " /> 李元甫在东山学校有自己的藏书楼，他曾特例准许毛泽东借阅书籍，毛泽东通过藏书楼的书籍了解到了西方世界的历史进程。<ref name="lsa " />

== 思想 ==
李元甫接受过西方先进的教育方式和科学技术，属于[[维新派|维新派]]，后来他担任[[东山学校|东山学校]]的校长，实行新政，开设了[[数学|数学]]、[[物理|物理]]、[[化学|化学]]、[[音乐|音乐]]、[[美术|美术]]等新课程，并且聘请留学生回到学校任职。<ref name="lsa " /><ref>{{cite web |url=http://dangshi.people.com.cn/GB/144956/8845735.html |title= 老师评语里的毛泽东|accessdate= 2009年|work= |publisher=中国共产党新闻网 |language =zh-cn}}</ref>

== 评价 ==
李元甫去世前，曾经遥望北方，作对联自挽：{{quote| 为国育才，应有一木支大厦<br /> 齐家教子，不及三葛在南阳。<ref name="lsa " />}}

1991年，湘乡市政府为李元甫树立碑文：{{quote| 
公讳树人，字元甫，本市共坪人，一八七二年生。少聪明才智颖勤于学，弱冠中试。会东山学堂，兴公以众望所归，受聘为堂长，延名师招俊士，庠风纯肃遐迩。一九一零年秋，毛泽东自[[湘潭|湘潭]]来校求学。后又助其入[[湘乡驻省中学|湘乡驻省中学]]深造，慧眼识英才，于公有焉。毛泽东不忘公扶掖之情，于大革命时与夫人[[杨开慧|杨开慧]]视公于[[武昌|武昌]][[旅鄂中学|旅鄂中学]]，屡叙师生之谊，东山校史遂亦增辉。[[民国|民国]]以来，公历任[[长沙|长沙]][[妙高峰中学|妙高峰中学]]学监主任，[[湘乡驻省中学|湘乡驻省中学]]及[[旅鄂中学|旅鄂中学]]校长，后出任[[宜章县|宜章县]]知事，复应全国县知事考试。既登榜，先后授任[[四川|四川]][[酉阳|酉阳]]、[[开县|开县]]、[[巫山县|巫山县]]知事及武昌堤工局长。一九三一年归里，一九四一年逝世，享年七十。越五十年，[[东山学校|东山学校]]及校友，谨勒其事于石，以昭后世，且为之赞曰：“育才宏猷，之东山俦。济民长泽，共涟水流。士林楷模，留芳千秋。<ref name="lsa " />
}}
