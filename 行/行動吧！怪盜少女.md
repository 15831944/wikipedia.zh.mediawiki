《'''行動吧！怪盜少女'''》（{{lang-ja|行くぜっ!怪盗少女}}）是[[桃色幸運草Z|桃色幸運草]]的出道[[單曲|單曲]]，於2010年5月5日由[[UNIVERSAL_J|UNIVERSAL J]]發售。2012年9月26日，此單曲以《行動吧！怪盜少女 〜Special Edition〜》之名重新發行，額外附送收錄《行動吧！怪盜少女》MV的DVD，同樣由UNIVERSAL J發行。<ref name="TOWER RECORDS" />

== 發行及宣傳 ==
單曲分為通常盤和[[限定版|初回限定]]盤A（{{lang|ja|[[早見朱莉|あかり]]}}盤）、B（{{lang|ja|[[佐佐木彩夏|あーりん]]}}盤）、C（{{lang|ja|[[有安杏果|ももか]]}}盤）、D（{{lang|ja|[[百田夏菜子|かなこ]]}}盤）、E（{{lang|ja|[[玉井詩織|しおり]]}}盤）、F（{{lang|ja|[[高城蕾妮|れに]]}}盤），共七種版本發行。每張初回限定盤以不同成員為封面，並以成員各自的[[桃色幸運草Z#成員|代表色]]為主色；收錄曲目則跟通常盤相同。

2010年3月3日，桃色幸運草於[[明治紀念館|明治紀念館]]舉辦公開簽約儀式，定下了「若未能達到『(身高-100)×0.8』的『[[日本偶像|偶像]]體重』，就不會簽約」的規則。會場上共有200名歌迷和記者。公開測量後，其中一名成員[[高城蕾妮|高城蕾妮]]超重0.8公斤。因此她們的合約變了「暫定合約」。她們在同一活動上說出該年的目標：「取得[[Oricon公信榜|Oricon]]第1名，在[[NHK紅白歌合戰|紅白]]登場，在[[日本武道館|武道館]]辦演唱會！然後舉辦一個給全部來到武道館的歌迷的[[握手會|握手會]]。」<ref>{{cite web|url=http://www.oricon.co.jp/news/confidence/73976/full/ |language=ja |title=<span lang="ja">ももクロ、公開体重測定でアイドルメジャーデビューが“仮契約”に</span> |publisher=[[Oricon|Oricon]] |date=2010-03-04 |accessdate=2013-03-14}}</ref>

== 歌曲 ==
桃色幸運草在《行動吧！怪盜少女》中自稱「怪盜少女」，歌曲甫開始就是以[[說唱|說唱]]風格介紹六名成員的名字。歌中主角宣示想得到「你的那顆心」（{{lang|ja|あなたのその心}}），以及夢想「用笑容和歌聲照亮世界」（{{lang|ja|笑顔と歌声で世界を照らし出せ}}）。<ref>{{cite web|url=http://www.hotexpress.co.jp/review/2619/ |language=ja |title=<span lang="ja">行くぜっ！怪盗少女</span> |publisher=hotexpress |date=2010-05-05 |accessdate=2013-03-14}}</ref>

== 歌詞變動 ==
歌曲開頭的歌詞會隨著成員人數變動，初期的歌詞為「レニ カナコ アカリ シオリ アヤカ モモカ」，[[早見朱莉|早見朱莉]]於2011年4月10日畢業後歌詞變更為「レニ カナコー シオリ アヤカ モモカ」，[[有安杏果|有安杏果]]於2018年1月21日畢業後歌詞變更為「レニ カナコー シオリ ササキアヤカ」。

== 編舞及音樂影片 ==
在《行動吧！怪盜少女》音樂影片（MV）的部份鏡頭，桃色幸運草穿上了[[女僕|女僕]]、[[護士|護士]]、[[女警|女警]]等服裝。<ref name="TOWER RECORDS" />MV中其中一個焦點在於[[百田夏菜子|百田夏菜子]]的「反蝦跳」（{{lang|ja|エビ反りジャンプ}}）——四肢擺後，身體向外曲跳起。<ref name="TOWER RECORDS" /><ref name="oricon1231" />這一跳亦是這首歌現場演出的重點。<ref name="oricon1231" /><ref>{{cite web|url=http://natalie.mu/music/news/54018 |language=ja |title=<span lang="ja">ももクロ東京タワーで「オレンジノート」アカペラ大合唱</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2011-07-31 |accessdate=2013-03-14}}</ref><ref>{{cite web|url=http://natalie.mu/music/gallery/show/news_id/58889/image_id/98804 |language=ja |title=<span lang="ja">ファッションショー！お悩み解決！ももクロ女祭り大成功</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2011-10-31 |accessdate=2013-03-14}}</ref>

== 銷售成績 ==
2010年5月3日，此單曲以約1.6萬的銷量登上[[Oricon公信榜|Oricon公信榜]]單曲日榜首名，翌日排名急降至第14名。<ref name="GREE">{{cite web |url=http://jp.news.gree.net/news/entry/525157 |language=ja |title=<span lang="ja">予約握手のコバンザメ商法!?　ももクロ、デイリー1位も翌日14位に急落</span> |publisher=GREE |date=2010-05-08 |accessdate=2013-03-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20130119062946/http://jp.news.gree.net/news/entry/525157 |archivedate=2013-01-19 }}</ref>其後此單曲取得該星期單曲周榜的第3名。<ref name="oricon" />兩年後重新發行的版本《行動吧！怪盜少女 〜Special Edition〜》則取得同一排行榜的第7名。<ref name="oricon re">{{cite web|url=http://www.oricon.co.jp/rank/js/w/2012-10-08/more/2/ |language=ja |title=<span lang="ja">2012年09月24日～2012年09月30日のCDシングル週間ランキング（2012年10月08日付）</span> |publisher=[[Oricon|Oricon]] |date=2012 |accessdate=2013-03-14}}</ref>

== 現場表演 ==
2012年12月31日，桃色幸運草Z初次登上[[NHK紅白歌合戰|NHK紅白歌合戰]]的舞台，演唱以「桃色紅白Z」（{{lang|ja|ももいろ紅白だZ!!}}）為題的組曲。該組曲由《行動吧！怪盜少女》和其當時最新的單曲《[[再會吧、那些愛憐的悲傷啊|再會吧、那些愛憐的悲傷啊]]》兩首歌曲組成。她們先穿上黑色或白色的長裙演唱《再會吧、那些愛憐的悲傷啊》，然後脫去身上的黑白長裙，露出裏面以五名成員各自的代表顏色（紅、黃、粉紅、綠、紫）為主色的裙子繼續演唱《行動吧！怪盜少女》。雖然當時[[早見朱莉|早見朱莉]]早於2011年4月離開組合，但歌中仍使用「{{lang|ja|We’re the ももいろクローバー　レニ　カナコ　アカリ　シオリ　アヤカ　モモカ}}」一段包含六人暱稱的歌詞。<ref name="oricon1231">{{cite web|url=http://www.oricon.co.jp/news/movie/2020272/full/ |language=ja |title=<span lang="ja">【紅白】ももクロ、4年越しの夢舞台で号泣　「あかりんに届いたと思う」</span> |publisher=[[Oricon|Oricon]] |date=2012-12-31 |accessdate=2013-03-13}}</ref>
