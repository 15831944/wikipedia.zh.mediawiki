{{日本作品|行星与共|プラネット·ウィズ}}，是由[[J.C.STAFF|J.C.STAFF]]原創製作、以[[機器人動畫|機器人]]為題材的[[日本|日本]][[電視動畫|電視動畫]]作品，2018年7月8日至9月23日首播<ref name="natalie20180323">{{Cite web|url=https://natalie.mu/comic/news/274827|script-title=ja:水上悟志の最新作がアニメ&マンガで始動！「この日をずっと待ってました」（動画あり / コメントあり）|work=[[Natalie_(網站)|Comic Natalie]]|publisher=Natasha|date=2018-03-23|accessdate=2018-04-03|language=ja}}</ref>。

原作者[[水上悟志|水上悟志]]自己表示，從開始企劃至今超過4年的時間，前後為這套作品總共繪製出1074張漫畫草圖，因此本作品的宣傳短片影像是用他畫的草圖所構成<ref name="natalie20180323" /><ref>{{Cite web|url=https://www.famitsu.tw/articles/2018/03/009500.html|title=原創機械人動畫《行星与共》2018夏季播映|work=[[Fami通|Fami通（台灣）]]|publisher=|date=2018-03-23|accessdate=2018-05-26|language=ja}}</ref>。

漫畫版從2018年4月起，由水上悟志親自繪製，開始在雜誌《[[Young_King_OURs|Young King OURs]]》6月號（[[少年畫報社|少年畫報社]]出版）進行連載<ref>{{Cite web|url=https://natalie.mu/comic/news/280233|script-title=ja:水上悟志の新連載「プラネット·ウィズ」始動、今夏TVアニメ放送作品のマンガ版|work=Comic Natalie|publisher=Natasha|date=2018-04-28|accessdate=2018-05-06|language=ja}}</ref>。

== 登場人物 ==
; {{nihongo|黑井宗矢|黒井 宗矢（くろい そうや）|聲：[[阿部敦|阿部敦]]<ref name="natalie20180323" />}}
: 失去記憶的少年，目前由銀子跟老師照顧，外星物體出現後開始選擇戰鬥，帶著面具被稱為星雲戰士，5年後成為星雲武官助理。

; {{nihongo|高天原望美|高天原 のぞみ（たかまがはら のぞみ）|聲：[[原田彩楓|原田彩楓]]<ref name="natalie20180323" />}}
: 宗矢的同班同學，因擔心宗矢總是很關心的問候，旦窮追不捨來到公園，宗矢知道望美很關心並說出自己的秘密，宗矢中途說“騙你的這種事你都相信”“望美知道事情後就說”“我就當是謊話吧”“很感謝宗矢保護這個鎮子”，喜歡宗矢，5年後成為星雲臨時心靈感應部隊員。

; {{nihongo|老師|先生（せんせい）|聲：[[小山力也|小山力也]]<ref name="staffcast">{{Cite web|url=http://planet-with.com/staff-cast/|title=製作人員&配音員|work=電視動畫《行星与共》官方網站|accessdate=2018-06-08|language=ja}}</ref>}}

; {{nihongo|白石小金(黃金)|白石 こがね（いらいし こがね）|聲：[[後藤沙緒里|後藤沙緒里]]<ref name="natalie20180323" />}}

; {{nihongo|虎居英雄|虎居 英雄（とらい ひでお）|聲：[[梅原裕一郎|梅原裕一郎]]<ref name="natalie20180323" />}}

; {{nihongo|因幡美羽|因幡 美羽（いなば みわ）|聲：[[大和田仁美|大和田仁美]]<ref name="natalie20180323" />}}，

; {{nihongo|根津屋正義|根津屋 正義（ねつや せいぎ）|聲：[[菅原慎介|菅原慎介]]<ref name="natalie20180323" />}}

; {{nihongo|鷹取紅華|鷹取 紅華（たかとり くれは）|聲：[[石上靜香|石上靜香]]<ref name="natalie20180323" />}}
