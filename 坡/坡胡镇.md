'''坡胡镇'''，是{{PRC admin/fullname|41/10/82/101/000|3}}下辖的一个[[乡镇|乡镇]]级行政单位。<ref>{{PRC admin/ref sgc|{{PRC admin/parent|41/10/82/101/000}}}}</ref>位于长葛市西部，距市区10千米。长（葛）禹（州）公路横穿镇境。<ref name="简介">{{cite web |url=http://www.xzqh.org/html/2009/0403/14950.html |language= |title=坡胡镇 |publisher=行政区划网 |date=2009-04-03 |accessdate= }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>镇政府驻地坡东村。该镇居民以[[胡姓|胡姓]]居多。

== 历史 ==
1954年建立坡胡乡；1958年改为公社；1984年改乡；'''1993年建镇'''。面积46.6平方千米，1997年人口为5.3万。<ref name= "简介"/>

== 地理位置 ==
南临[[石固镇|石固镇]]，北依[[后河镇|后河镇]]，西接[[禹州市|禹州市]][[古城镇|古城镇]]和[[无梁镇|无梁镇]]。

== 经济 ==
坡胡镇是著名的“农机配件之乡”、长葛市粮食高产区（4.4万亩耕地<ref name= "概况"/>）和第三产业基地。

长葛市汽车零配件专业园区2平方公里，位于西杨村彭花公路北，新庄坡村以南，东至苇园村，西至坡马村<ref name="概况">{{cite web |url=http://www.changge.gov.cn/cgcms/sites/phz/preview.jsp?ColumnID=696&TID=20101020091612046475682 |language= |title=坡胡镇简介 |publisher=坡胡镇网站 |date=2010-10-20 |accessdate= }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。著名企业有长葛市赛丹汽车配件制造有限公司（成立于2010年6月，注册资金500万元）[http://www.changge.gov.cn/cgcms/sites/phz/preview.jsp?ColumnID=1322&TID=20120405163840810302279]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}。
著名企业：长葛市旭日阳光机械制造有限公司（前身为长葛市阳光机械配件厂，网络关键字“阳光机械”就是介绍的这个著名企业）就位于这个专业园区内。

== 教育 ==
坡胡镇有22所[[小学|小学]]和四所[[初级中学|初级中学]]，初中分别为[[坡胡一中|坡胡一中]]（位于坡东村）、[[坡胡二中|坡胡二中]]（位于水么河村）<ref>{{cite web|url=http://post.8684.cn/o2443424_597.htm | language= | title=长葛市坡胡镇水么河二中邮编 |publisher=邮政编码 |date= |accessdate=2012年9月 }}</ref>、[[坡胡三中|坡胡三中]]（位于西杨村）和[[坡胡四中|坡胡四中]]（位于营张村）。
