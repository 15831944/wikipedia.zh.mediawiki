'''繪笛站'''（{{jpn|j=絵笛駅|hg=えふええき|rm=Efue eki|f=y}}）位於[[北海道|北海道]][[浦河郡|浦河郡]][[浦河町|浦河町]]字繪笛，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|フエ}}'''。站名來源於[[阿伊努語|阿伊努語]]的「epuy」（[[花蕾|花蕾]]）<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/1120P.pdf|title=アイヌ語地名リスト|accessdate=2017-08-16|author=北海道庁|date=|publisher=北海道庁}}</ref>，由來是附近有茂盛的[[蜂斗菜|蜂斗菜]]花莖（{{lang|ja|フキノトウ}}）。

== 車站構造 ==
[[側式月台|側式月台]]1面1線的地面站。沒有站房，只有一座小型的候車室。

== 車站周邊 ==
車站周圍多為牧場。
* 井寒台森林公園
* 繪笛川
* ポロイワ山

== 歷史 ==
* 1958年（[[昭和|昭和]]33年）7月15日 - 設立，只處理旅客業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
