'''黑羽快斗'''，是青山剛昌原作的日本漫画和動画作品《[[魔术快斗|魔术快斗]]》中的主角，也是日本漫画和動画作品《[[名侦探柯南|名侦探柯南]]》中的客串角色。黑羽快斗為第二代怪盗基德；在劇情中的八年前，其父親黑羽盗一為第一代怪盗基德。擅長使用魔術技法，主要道具是扑克牌枪。

== 命名來源 ==
全称“怪盗1412号”，因其国际罪犯代码为“1412”。因[[工藤優作|工藤優作]]將[[記者|記者]]寫的潦草的“1412”读作“KID.”（见TV 76、473），于是“怪盗基德”称呼诞生。
[[File:1412kid-conan.png|thumb]]

== 人物 ==
* 外号：'''怪盗基德'''（日文：怪盗キッド；罗马音：Kaitō Kiddo；英文：Kid.、Kid the phantom thief；TVB曾译'''怪盗少侠'''；台版漫畫譯為怪盜小子）
* 真实姓名：'''黑羽快斗'''（日文：くろばかいと；罗马音：Kuroba Kaito）来源：“黑”色+“怪盗”（“快斗(Kaito)”的日语发音和“怪盗(Kaitō)”相近）
* 化名：土井塔克树（日文：どいとうかつき），用假名写出来后可以重新組合為“怪盗KID”（日文：かいとうきっど），TV 132～133《魔术爱好者杀人事件》）、“红色鲱鱼”（“Red Herring”，即转移注意力的事物，剧集同上，为基德的网名）
* 爱好：魔术、冰淇淋、看报纸上与自己有关的新闻、恶作剧、小动物
* 弱点：鱼（《[[魔术快斗|魔术快斗]]》中设定，OVA 10有提及）、溜冰（《魔术快斗》Vol.3 File.2《名侦探登场！》，差点因此被中森警官抓住；不過卻很擅长滑雪）
* 智商：400（在《魔术快斗》Vol.3 File.4《红子的特快专递》中，白马探搜集到的基德数据分析其為运动全能、但是不會溜冰）

（注：有人认为400指的是魔术智商，因为现实中人类的IQ绝不可能达到400，但毕竟《魔术快斗》本身就带有魔幻色彩，再者原著中和青山本人从未提及所谓“魔术智商”的概念，官方资料也是如此；為尊重原著，以原作中提及的智商400为准。从原作可知，智商数据并非白马探检测基德头发得出，而是之前就搜集到的数据）

=== 第一代 ===
* 第一代「怪盗基德」：[[黑羽盗一|黑羽盗一]]（十八年前～八年前）
: [[黑羽快斗|黑羽快斗]]的父親，一位受人敬仰的世界級[[魔术师|魔术师]]，十八年前第一次以怪盗基德面目在[[巴黎|巴黎]]开始活动，八年前被某個神秘组织暗殺。

=== 第二代 ===
* 第二代「怪盜基德」：黑羽快斗（現任）（注：未註明情況下，“怪盜基德”指第二代基德，即黑羽快斗）
: 快斗是江古田高中二年B班學生，熱愛[[魔術|魔術]]的17歲少年，因意外發現家中密室，得知了父親黑羽盜一就是曾活躍一時並於八年前銷聲匿跡的怪盜基德，其後得知八年前父親在魔術表演中遇險並非演出意外、而是被人謀害，為了追查背後的真相而繼承父親怪盜的身份，藉此引出犯人。
: 快斗有一位青梅竹馬兼同班同學[[魔術快斗#登場角色|中森青子]]，她並不知道快斗的怪盜基德身份。她的父親[[魔術快斗#登場角色|中森銀三]]為[[警視廳|警視廳]]搜查二課[[警部|警部]]，急欲逮捕怪盜基德，但是大都以失敗收場，也因此中森青子非常討厭怪盜基德。
: 劇中快斗與青子對彼此的喜歡之意顯而易見，雖未正式表白，卻是青山剛昌認定的三大青梅竹馬配對之一。

== 形象 ==
充滿傳奇色彩的夜盜，專門以藝術品（特別是寶石）為目標。身著白色禮服，佩特製單片眼鏡，背後的披風可以變為[[滑翔翼|滑翔翼]]。善於各種魔術手法，精通[[易容|易容]]，更能夠不借助道具模仿任何人的聲音，偽裝堪稱完美。記憶力過於常人。
其服裝疑參考1984年宮崎駿的《[[名偵探福爾摩斯|名偵探福爾摩斯]]》電視動畫中的莫里亞蒂教授。

舉止優雅，極其[[紳士|紳士]]，有時也會露出調皮的一面。神出鬼沒地盜取預告的目標，玩弄警察於掌中。雖是名通緝犯，但每次作案時並不會造成任何傷亡，而且每次作案也會吸引一大堆粉絲打氣。通常偷得的寶石、書畫等物品多半會歸還原主，或視情況給予真正需要的人。

== 持有道具 ==

* 撲克牌手槍：黑羽快斗平常就愛用的工具和武器。撲克紙牌可打旋也可拐彎，射出的力道可大可小，力道大時能切斷精鐵鍊條。也可射出帶鉤的繩索、旗子等其他物品。
* 閃光彈：逃生時用，有時是用於在眾人面前迅速換裝。
* 三角滑翔翼：缺點：
** 一、是滑翔要看風力，風力小了當然飛不起來，風力太大了也同樣會栽下來，與名偵探白馬探交手的那次就是吃虧在了七級大風上；同時風向也是個問題：M 3中的柯南和平次、OVA 10中的柯南和偵探團便從風向推理出了基德的逃走方向。
** 二、在高空目標太過明顯，在第5集中被斯內克打斷滑翔翼橫桿，在M 3中就乾脆連人一併被打了下來，在OVA 10中，被少年偵探團用很簡易的裝置就打下來了。
** 三、滑翔翼除了看風力還要看天氣，畢竟雷鳴閃電中用滑翔翼也無異於找死，在“麒麟角事件”中也因颱風而沒法使用(雖然也沒打算使用)。
* 竊聽器：將其裝於鴿子腿部、他人衣服或首飾上上進行竊聽。
* 催眠瓦斯：用於干擾他人。
* 吸盤：劃破玻璃時用吸盤吸住要切下的玻璃，防止其碎裂出聲。
* 萬能鑰匙：小偷專業道具，開鎖用。
* 走壁滑板：小型可操縱方向的“交通工具”，平時折疊安在滑翔翼旁邊，使用時拉出，可在傾角為90度的牆壁上來去自如。
* 溜冰鞋：拉動鞋上的機關，就可變成一雙滾軸三輪溜冰鞋，馬力十足且使用靈活，可以順樓梯扶手飛速而上，也能走鋼絲。
* 假人：即人形橡皮氣球，吹起後成人型，用於引開警察注意力，幾乎無失敗記錄（關於這點基德自己都很納悶，柯南也說：“警察們該吸取點教訓吧！”也許是警察見到基德太興奮了），但是在《怪盜淑女之卷》中被中森警官識破。
* 氣球：用於帶走所偷重物。
* 感應器視鏡：用於對抗防犯感應器。
* 發光二極管：將其扭曲，便可發出淡紅色的光，用於迷惑他人。
* 魔術師工具箱：內有手帕、大把的彩紙屑、紙牌、花朵、鴿子…魔術師有的和沒有的里面都有。
* 預告函：在盜竊寶石之前一定會發出的預告函，內容極難理解，幾乎只有高智商偵探才能破解。上面的圖案為怪盜基德親筆畫的。
* 單片眼鏡：配合逆光，讓人看不清楚他的面容。
* 鴿子：把竊聽器、監視器裝於鴿子​​腿部，可以用來竊聽、監視，變魔術時可吸引觀眾的視線。
* 面具：用於易容。從黑羽盜一的手中學來。
* 白色披風：進行換衣術與易容術時用於遮掩；以及作滑翔翼的覆布（即布製翼）。
* 煙幕彈：逃跑及表演開場時放煙幕。
* [[鐵撬|鐵撬]]：撬開升降機門。

== 其他 ==
在《[[名偵探柯南：世紀末的魔術師|名偵探柯南：世紀末的魔術師]]》中，怪盜基德就已知道[[江戶川柯南|江戶川柯南]]就是[[工藤新一|工藤新一]]（通過船上電話[[竊聽|竊聽]]；另在電視動畫第76集中有懷疑柯南身份的暗示），此事不影響到《[[名偵探柯南|名偵探柯南]]》原作的劇情發展和基本設定。

雖然沒有證據顯示其聰明程度勝於[[工藤新一|工藤新一]]、[[白馬探|白馬探]]等偵探角色（實際在校成績普通），但多次的盜寶行動最後被阻止，仍能臨機應變脫逃，臨場反應和決斷力絕佳，從白馬探找到的基德頭髮測出智商有400。
