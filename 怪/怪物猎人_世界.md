{{Vgname|怪物猎人 世界|ja=モンスターハンター：ワールド|en=Monster Hunter: World|cn=怪物猎人 世界|tw=魔物獵人 世界}}是一款由[[卡普空|卡普空]]制作并在[[PlayStation_4|PlayStation 4]]、[[Xbox_One|Xbox One]]和[[Windows|Windows]]平台上发行的动作角色扮演游戏，是该系列继2009年发售的《[[怪物猎人3|怪物猎人3]]》之后再次為[[家用游戏机|家用主机]]平台制作的新作{{efn|系列于2011年在[[PS3|PS3]]上发售过《MHP3》的高清移植版、同年在[[Wii_U|Wii U]]上发售过《MH3G》的高清移植版、于2016年在[[任天堂Switch|任天堂Switch]]上发售过《MHXX》的高清移植版。}}，也是系列首次在Windows上发售本傳作品，亦是本傳作品首次进行中文化：PlayStation 4版本和Windows上的[[Steam|Steam]]网络商店版本提供官方[[繁体中文|繁体中文]]化<ref name="vgtime 0710" />，PlayStation 4版本还将在未来发售简体中文版本；而[[腾讯|腾讯]]的[[Wegame|Wegame]]平台提前两日（以[[北京时间|北京时间]]计算）提供发售Windows版的官方简体中文版本<ref name="国服" />。

《怪物猎人 世界》最初在2017年[[E3游戏展|E3游戏展]]上正式公布，游戏的[[PlayStation_4|PlayStation 4]]和[[Xbox_One|Xbox One]]版本于2018年1月26日發售。PC版于同年8月9日发售；PS4平台在亞洲地區为游戏主机的獨佔版本。游戏在发售后获得媒體評論的好评<ref name="MCPS4" /><ref name="MCONE" /><ref name="MCPC" />。

==概述==
游戏最初于[[E3_2017|E3 2017]]上宣布。根据制作人辻本良三和德田优也的说法，《怪物猎人 世界》将会是《怪物猎人系列》的主系列作品<ref name="glixel dev" />。德田优也和藤冈要是《怪物猎人4》的总监。《怪物猎人 世界》在2017 E3的3年前即开始制作。在2017年第十九届[[香港动漫电玩节|香港动漫电玩节]]上，卡普空宣布本作在发售后会追加繁体中文化更新补丁{{efn|僅限PS4亞版，包含日版在內等其他地區版本無法更新中文版。}}，这是本傳作品的首次中文化<ref name="免費更新中文" />{{efn|此前由CAPCOM开发及运营的网游《[[怪物猎人Frontier_Online|怪物猎人Frontier Online]]》提供繁体中文版；而授权[[腾讯|腾讯]]开发及运营的网游《[[怪物猎人Online|怪物猎人Online]]》有简体中文版。}}。

《魔物獵人 世界》普遍獲得媒體的好評<ref name="MCPS4" /><ref name="MCONE" /><ref name="MCPC" />。但游戏在[[Steam|Steam]]平台收到了褒贬不一的评价，负面评价主要集中在Steam平台上频繁出现的断线问题。游戏发行商[[卡普空|卡普空]]称已注意到此问题并正在调查<ref>{{Cite web|url=https://steamcommunity.com/gid/103582791462692506/announcements/detail/1700558821197794896|title=MONSTER HUNTER: WORLD - Connection error after departing on quests several times
|accessdate=2018-08-13|author=MHW_CAPCOM|date=2018-08-13|publisher=[[Steam|Steam]]社区}}</ref>。

该游戏在[[中国大陆|中国大陆]]由[[腾讯|腾讯]][[WeGame|WeGame]]平台代理，并于2018年8月8日发售。但WeGame官方于2018年8月13日公告称，因“部分游戏内容未符合政策法规的要求，相关政府管理部门接到大量举报，该游戏相应运营资质文件现已取消”，应主管部门要求下下架整改、停止游戏售卖。已在WeGame购买游戏的玩家可选择在8月20日8时前无条件退款，也可继续进入游戏当前版本，但无法保障后续所有服务，相关事宜或其他影响有待另行公告<ref>{{Cite web|url=http://www.gamersky.com/news/201808/1085459.shtml|title=Wegame《怪物猎人：世界》将下架 因遭受大量举报|accessdate=2018-08-13|author=FoxJR|date=2018-08-13|publisher=[[游民星空|游民星空]]}}</ref><ref>{{cite web |author1=谁是大英雄 |title=腾讯给了我一次最痛快的退款，但我却怎么也高兴不起来 |url=http://www.geekpark.net/news/231912 |website=极客公园 |accessdate=2018-08-14 |date=2018-08-14}}</ref>。

由于[[网易|网易]]旗下的游戏媒体「网易爱玩」在WeGame版下架时发表的言论，有网友怀疑是网易方面进行了针对性的举报；在随后[[微信|微信]]公众号文章的煽动下，网友舆论转而大肆攻讦网易；这一情况引发了网易创始人[[丁磊|丁磊]]现身辟谣否认<ref>{{cite web |author1=C9 |title=WeGame版《怪物猎人：世界》下架后的那些事儿 |url=http://www.yystv.cn/p/3768 |website=游戏研究社 |accessdate=2018-09-10 |date=2018-08-14}}</ref>。下架风波次月的9月10日，[[新浪微博|新浪微博]]用户@灵石路黄师傅爆料，其通过向主管单位申请信息公开，得到回复，下架原因为腾讯送审版本和实际运营版本不符，并没有腾讯自称的“大量举报”；认为此一事件为腾讯WeGame平台自己的责任<ref>{{cite web |author1=灵石路黄师傅 |title=从来就没有举报这回事… |url=https://weibo.com/1893075304/GyL4Ox5MP |website=新浪微博 |accessdate=2018-09-10 |date=2018-08-14}}</ref>。

===銷售===
截至2018年5月，本作的全球累計出貨量（含下載版）達到了790萬份，為全系列作品中的最高紀錄<ref>http://www.vgchartz.com/article/276220/monster-hunter-world-ships-79-million-units/</ref>。

於Capcom的財報中，截至2018年4月16日，本作的實體出货量和已販售的下載版合計超过了800万份<ref>http://www.capcom.co.jp/ir/english/data/pdf/explanation/2017/full/explanation_2017_full_01.pdf</ref>。

截至2019年1月23日，本作全平台的全球累計出貨量（含下載版）達到了1200萬份<ref>https://www.famitsu.com/news/201902/08171693.html</ref>。
