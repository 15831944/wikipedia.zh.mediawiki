'''曾巩'''{{BD|1019年|9月30日|1083年|4月30日|catIdx=Z曾}}，字'''子固'''，[[建昌|建昌]][[南丰|南丰]]（今[[江西|江西]]南丰）人，[[汉族|汉族]][[江右民系|江右民系]]，[[北宋|北宋]][[散文家|散文家]]，被誉为“[[唐宋八大家|唐宋八大家]]”之一。

== 生平 ==
宋天禧三年(1019年)，八月二十五生於[[江西|江西]][[南豐|南豐]]一個官宦人家，十二歲作《六論》一篇，為當時所推祟。十六歲即篤志為古文。

十八歲隨父[[曾易占|曾易占]]遷移至[[玉山縣|玉山縣]]（在江西境內），周遊當地，寫成了《[[遊信州玉山小岩記|遊信州玉山小岩記]]》，內容主要是記述玉山縣的地貌，包括[[溶洞|溶洞]]、[[岩石|岩石]]等等。二十歲時再歷遊全國，得[[歐陽修|歐陽修]]的賞識，成為歐陽修的學生，並稱“歐曾”。

進士考試時，[[梅聖俞|梅聖俞]]為考官，發現了[[蘇軾|蘇軾]]寫的「[[s:刑賞忠厚之至論|刑賞忠厚之至論]]」，驚為天人，並推薦蘇軾的試卷給輔助主考官歐陽修批閱。歐陽修頗驚其才，但是試卷[[糊名|糊名]]，歐陽修認為很有可能是弟子曾鞏所寫（實際為蘇軾所寫），於是將此卷取為第二，將原本第二名的卷子取為第一。{{Citation needed}}欧阳修对[[梅尧臣|梅尧臣]]说：“老夫当避路，放他出一头地也。”<ref>欧阳修《与梅圣俞书》：“老夫当避路，放他出一头地也。”</ref>但卻事有碰巧，歐陽修為了避嫌，取為第一的卷子，恰好是曾鞏所寫【此处有误：嘉佑二年的状元是章惇的侄子章衡，并不是曾巩。而章惇羞于落后年长自己的侄子，坚辞进士及第，选择嘉佑四年重考，终得一甲第五名】。{{Citation needed}}[[王安石|王安石]]說：“曾子文章世稀有，水之江漢星之斗”。

1057年他中進士後，历任[[太平州|太平州]]司法[[参军|参军]]、馆阁[[校勘|校勘]]、[[越州|越州]][[通判|通判]]，[[济州|济州]]、[[福州|福州]][[知州|知州]]。後受[[宋神宗|宋神宗]]邀請，到京師擔任[[中書舍人|中書舍人]]，進行編修史書工作。曾鞏参与了整理并[[校勘|校勘]]《[[梁书|梁书]]》、《[[陈书|陈书]]》、《[[南齐书|南齐书]]》、《[[列女传|列女传]]》、《[[战国策|战国策]]》、《[[说苑|说苑]]》等书，寫有“敘錄”。著有《[[元丰类稿|元丰类稿]]》、《[[续元丰类稿|续元丰类稿]]》、《[[外集|外集]]》等。代表作为《[[墨池记|墨池记]]》。其文风细，笔墨犀利。

元豐五年，拜[[中書舍人|中書舍人]]。

元豐六年（1083年），因母喪免官，四月十一日卒于江寧府。理宗時追謚「文定」。

== 文风 ==
曾巩的文体风格为“古雅平正”，擅长引经据典；结构则平易理醇，章法开阖、承转、起伏、回环都有一定约束法度、严密、规矩。正因为其文章易于模仿和学习，他成为了唐宋文派和[[桐城派|桐城派]]学习的首要对象。<ref name=" fish" />

[[彭渊材|彭渊材]]说道：“恨曾子固不能作诗”<ref>宋释[[惠洪|惠洪]]《冷斋夜话》卷九</ref>。自此说一起，曾巩“不能诗”即盛传于世，他自己的學生[[秦观|秦观]]也认为他不会作诗<ref>《东坡题跋·卷三·记少游论诗文》：“曾子固以文名天下，而有韵者辄不工，此未易以理推之也。”</ref>，另一個學生[[陈师道|陈师道]]也说他“短于韵语”<ref>陈师道：《后山诗话》</ref>。這場筆墨官司一直延續至清朝<ref>[[王士祯|王士祯]]《池北偶谈》辩驳说：“刘渊才恨曾子固不能诗，今人以为口实。今观《类稿》中诸篇，亦荆公之亚，但天分微不及耳。若皇甫持正、苏明允、陈同父，乃真不能诗也。”</ref>。元初[[劉壎|劉壎]]亦为曾巩的诗辩护说：“宋人诗体多尚赋而比兴寡，先生之诗亦然。故惟当以赋体观之，即无憾矣”{{NoteTag|关于曾巩会不会做诗的争论，[[錢鍾書|錢鍾書]]在《宋詩選注》第63页注解裡提出十一種讨论此一问题的書目：孙觌《鸿庆居士集》卷十二《与曾端伯书》、刘克庄《后村大全集》卷一百七十五、方回《瀛奎律髓》卷十六、劉壎《隐居通议》卷七、杨慎《升庵外集》卷七十八、贺裳《载酒园诗话》卷五、王士祯《池北偶談》卷十四、何焯《义门读书记·元丰类稿》卷一、潘德舆《养一斋诗话》卷四、方东树《昭昧詹言》卷一、姚莹《后湘诗集》卷九《论诗绝句》、杨希闵《乡诗摭谭》卷三。另有學者又補充陆九渊《象山集》卷十七《与沈宰》、费衮《梁溪漫志》卷七《作诗当以学》、赵与时《宾退录》卷六、李东阳《怀麓堂集》卷四十一《书愧斋唱和诗序后》、杨慎《升庵集》卷五十五《曾子固诗》、郑方坤《全闽诗话》卷七《王慎中》、谢旻《江西通志》卷一四三载何乔新《书元丰类稿后》、陈思《海棠谱》卷上。詳見王兆鵬：〈钱钟书《宋诗选注》的文献价值及文献疏失〉一文。}}。

== 評論 ==
* [[王安石|王安石]]：“曾子文章世稀有，水之江汉星之斗。”<ref name=" fish" >{{cite web |url= http://news.ifeng.com/history/zhongguogudaishi/detail_2012_09/06/17404671_0.shtml|title=“唐宋八大家”之一曾巩满于宋朝现状 不忧国忧民 |accessdate= 2012年9月6日|publisher=凤凰网 |work= |language =zh-cn}}</ref>
* [[苏轼|苏轼]]：“醉翁门下士，杂从难为贤；曾子独超轶，孤芳陋群妍。”<ref name=" fish" />
* [[秦观|秦观]]说：“人才各有分限，[[杜子美|杜子美]]诗冠古今，而无韵者殆不可读；'''曾子固'''以文名天下，而有韵者辄不工。此未易以理推之也。”
* [[陈师道|陈师道]]说：“[[苏明允|苏明允]]不能[[诗|诗]]，[[欧阳永叔|欧阳永叔]]不能[[赋|赋]]，'''曾子固'''短于[[韻文|韵语]]，[[黄鲁直|黄鲁直]]烦于[[散文|散语]]，[[苏子瞻|苏子瞻]]词如[[诗|诗]]，[[秦少游|秦少游]]诗如词。”
* [[朱熹|朱熹]]说，“爱其词严而理正，居尝诵习”。<ref name=" fish" />
* [[元代|元代]][[方回|方回]]說：“子固诗一扫[[西崑體|崑体]]，所谓餖飣刻畫咸無之，平实清健，自为一家。”<ref>《瀛奎律髓》</ref>
* 《[[明史|明史]]·王慎中传》：“慎中为文，初主秦汉，谓东京之下无可取，已司欧、曾作文之法，乃尽焚旧作，一意师仿，尤得力于曾巩；顺之初不服，久亦变而从之。”<ref name=" fish" />
* [[钱锺书|钱锺书]]说：“一场笔墨官司直打到清朝，看来判他（曾巩）胜诉的批评家居多数”，就[[唐宋八大家|唐宋八大家]]而言，“他的诗远比[[苏洵|苏洵]]、[[苏辙|苏辙]]父子的诗好，七言[[绝句|绝句]]更有[[王安石|王安石]]的风致。”<ref>《宋诗选注》</ref>
