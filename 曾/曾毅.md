'''曾毅'''{{BD|1979年|11月6日}}[[湖南|湖南]][[益阳|益阳]]人，[[凤凰传奇|凤凰传奇]]组合中的男声和声、[[说唱|说唱]]，[[中国大陆|中国大陆]]著名歌手。

== 简介 ==
曾毅早年[[广州|广州]]打工，是一名无线电器件工厂的工人。曾毅与来自[[内蒙古|内蒙古]]的[[杨魏玲花|杨魏玲花]]认识后，对音乐狂热使得两人组成“[[凤凰传奇|凤凰传奇]]”，将民族唱腔与流行唱法相结合。他们的作品《[[月亮之上|月亮之上]]》、《[[自由飞翔|自由飞翔]]》等歌曲一蹴而红，凤凰传奇组合也被观众誉为“神曲天团”。 <ref name=" “凤凰传奇”曾毅完婚">{{cite web |url=http://roll.sohu.com/20111122/n326410084.shtml|title=《“凤凰传奇”曾毅完婚》|accessdate=2011年11月20日|work=|publisher=搜狐网|language= zh-Hans}}</ref>

=== 情感生活 ===
自从凤凰传奇组合出道以来，观众一直认为女歌手[[杨魏玲花|杨魏玲花]]和男歌手曾毅是情侣，实质是他们只是搭档。2011年3月，杨魏玲花和其相恋六年的男友[[徐明朝|徐明朝]]在内蒙古[[鄂尔多斯|鄂尔多斯]]结婚，曾毅也于2011年11月20日在湖南益阳结婚。<ref>{{cite web |url=http://www.tj20.com/news/2011/xinwen1477.html|title=凤凰传奇曾毅与李娜结婚回到益阳|accessdate=2011-11-23|work=|publisher=桃江新闻网|language= zh-Hans}}</ref> 婚后居住于[[北京|北京]]。[[刀郎_(歌手)|刀郎]]、[[江涛|江涛]]、[[刘思思|刘思思]]、[[刘嘉亮|刘嘉亮]]、[[誓言|誓言]]、[[侯旭|侯旭]]、[[湘海|湘海]]等好友出席了婚礼。<ref>{{cite web |url=http://finance.591hx.com/article/2011-11-24/0000095582s.shtml|title=凤凰传奇曾毅大婚 美丽新娘李娜身份公开|accessdate=2011年11月24日|work=|publisher=华讯财经|language= zh-Hans}}[https://www.nvsay.com/ent/news/38747.html 【图】杨魏玲花和曾毅睡觉图片私密大曝光 火爆场面不忍直视_娱乐新闻_女人说娱乐]</ref>新娘是[[新疆|新疆]]籍的李娜，[[大陆|大陆]]著名歌手[[刀郎_(歌手)|刀郎]]的好友。刀郎特意发微博表示祝贺：“曾记否北门13小[[八一剧场|八一剧场]]的门外门楼。那一抹倩兮笑兮娇俏低首的温柔。在你年少懵懂手叉烤肉多少次不经意的离别邂逅。等到了桃花开，等不到人再来。一别经年，她已被这个[[湖南|湖南]]伢子带走，在你放下了小老窖手抓肉惊然顿首。是否后悔没有喊出这声：诶，丫头。”<ref name=" “凤凰传奇”曾毅完婚"/>
