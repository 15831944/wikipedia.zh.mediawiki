'''曾侯乙青铜尊盘'''，通称'''青铜尊盘'''，1978年出土于[[湖北省|湖北省]][[随县|随县]][[曾侯乙墓|曾侯乙墓]]的一件[[战国|战国]]早期[[青铜器|青铜器]]，分为上面的[[尊|尊]]和下面的[[盘|盘]]两部分，整套器物纹饰繁缛且有很多[[镂空|镂空]]装饰，为[[失蜡法|失蜡法]]铸造。该器现藏[[湖北省博物馆|湖北省博物馆]]。

== 外观 ==
尊盘由[[尊|尊]]和[[盘|盘]]组成，尊置于盘中。尊盘的口沿是非常精细的镂空的变形龙纹和龙形雕饰，均可分辨出每条盘龙上的眼睛。

== 制作 ==
是采用“[[失蜡法|失蜡法]]”的铸造方法。

== 铭文 ==
尊和盘均铸有“曾侯乙作持用终”铭文。
