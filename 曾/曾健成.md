'''曾健成'''（{{lang-en|'''Tsang Kin-shing'''}}，{{bd|1956年|1月26日|catIdx=Z曾}}），別名'''曾阿牛'''或'''阿牛'''，[[香港|香港]]前[[香港立法局|立法局]]議員，曾任[[香港島|香港島]][[東區_(香港)|東區]][[樂康|樂康]]選區區議員，香港政治組織[[民主救港力量|民主救港力量]]召集人、[[社會民主連線|社會民主連線]]成員、[[保釣行動委員會|保釣行動委員會]]召集人、[[苦業主團結陣線|苦業主團結陣線]]召集人及[[香港民間電台|香港民間電台]]台長。現為[[社會民主連線|社會民主連線]]成員之一，並是該黨的常務委員。

== 生平 ==
曾健成本身是[[基督徒|基督徒]]，已婚，育兩子，加入政界純綷出於偶然。

1989年，當時香港社會因為[[六四事件|六四事件]]而引發起不少有心人以行動支持他們。作為一個地盤判頭、出身於草根階層的曾健成亦因為這個緣故，創建「民主台」，認識了[[支聯會|支聯會]]的一眾人士。其後積極參與社會運動。

== 政治生涯 ==
[[File:Carrie-lam-submit-nomination-form-3.jpg|thumb]]報名參選特首時向她示威（2017年2月28日）]]
曾加入[[民主黨_(香港)|民主黨]]，是民主黨的創黨成員之一，並於1995年的[[香港立法會|立法局議員]]選舉中，以曾任地盤判頭的身份代表[[新九組|新九組]]中的“漁農、礦產、能源及建造界”分組裡勝出，晉身議員行列。在第一次開會之前，曾與當時任[[香港總督|香港總督]]的[[彭定康|彭定康]]打賭會在首次會議穿西裝出席。開會當日，彭定康遵守承諾，給了他一張有親筆簽名的500元鈔票。

1998年與由[[陶君行|陶君行]]、[[鄭家富|鄭家富]]、[[陳國樑|陳國樑]]等民主黨內議員組成的「[[少壯派|少壯派]]」，爭取民主黨改革。

[[1999年香港區議會選舉|1999年香港區議會選舉]]，競逐連任[[東區區議會|東區區議會]]樂康選區議席，敗於[[民建聯|民建聯]]的陳達強。

[[2000年香港立法會選舉|2000年香港立法會選舉]]，因堅持競逐立法會香港島選區議席，而退出民主黨。與[[陳添勝|陳添勝]]及[[陳國樑|陳國樑]]以獨立民主派身份，打出「窮人要出聲」的口號，組成一張名單參選，但最終落敗，未能重返立法會。

[[2003年香港區議會選舉|2003年香港區議會選舉]]，在自己家居所在的[[東區_(香港)|東區]][[樂康|樂康]]選區（範圍為[[柴灣|柴灣]][[康翠臺|康翠臺]]、樂翠臺、山翠苑），擊敗[[香港孔教學院|香港孔教學院]]院長[[湯恩佳|湯恩佳]]之女兒[[湯棋淯|湯棋淯]]，重返區議會。

[[2004年香港立法會選舉|2004年香港立法會選舉]]中，曾健成以獨立候選人身分，夥拍[[香港職工會聯盟|香港職工會聯盟]]副主席[[鍾松輝|鍾松輝]]及前[[香港大學|香港大學]]學生會會長[[鄧徐中|鄧徐中]]參選港島選區，最後仍然落選。

2006年，曾健成分別籌組「SOS業主陣線」及「[[社會民主連線|社會民主連線]]」。

[[2007年香港區議會選舉|2007年香港區議會選舉]]，曾健成以1912票擊敗民建聯的[[黃家維|黃家維]]，成功連任。

[[2011年香港區議會選舉|2011年香港區議會選舉]]，競逐連任東區區議會樂康選區議席，以1871票對1916票，45票之差敗於報稱獨立的候選人[[李進秋|李進秋]]。

== 民間電台 ==
[[File:SDC10532.JPG|250px]]時出席支持[[民間電台|民間電台]]的行動（2009年）]]
2005年開始，曾健成投入爭取開辦公眾頻道及[[香港民間電台|民間電台]]的工作。2006年8月30日，因「非法佔用频道」的罪名，曾的电台被[[香港电讯管理局|香港電訊管理部門]]取缔<ref>[http://www.ausnz.net/news_sql/news_detail.asp?articleID=7677]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2008年1月，他被[[香港|香港]][[電訊管理局|電訊管理局]]檢控，指他「非法廣播」違反司法當局早前所訂下的禁制令。

2008年7月，曾健成代表[[社會民主連線|社會民主連線]]參選[[2008年香港立法會選舉|立法會選舉]]港島區，最後得票10,202票（3.25%）落選。

== 登上釣魚島 ==
1996年，曾健成率先發起前往[[釣魚台|釣魚台]]拆[[日本政府|日本政府]]興建的燈塔，以回應[[日本右翼團體|日本右翼團體]][[日本青年社|日本青年社]]在釣魚台矗立一座燈塔的行動。他曾成立「[[保釣行動委員會|保釣行動委員會]]」，並多次組織漁船，由[[中華民國|台灣]]和[[英屬香港|英屬香港]]出發赴釣魚台宣示[[中國|中國]][[主權|主權]]。最後，由「保釣行動委員會」成員[[陳裕南|陳裕南]]及時為[[新黨|新黨]]籍[[新北市|臺北縣]]議員[[金介壽|金介壽]]（已在2006年初由[[親民黨|親民黨]]轉投[[中國國民黨|中國國民黨]]），在釣魚台礁石上插上「[[青天白日滿地紅旗|青天白日滿地紅旗]]」及「[[五星紅旗|五星紅旗]]」（皆代表[[中國國旗|中國國旗]]）。自此之後，他曾接受訪問表明不再踏足[[日本|日本]]，也不會買日貨。

2012年8月15日，曾健成與香港保釣行動委員會人士一同登上[[釣魚台|釣魚台]]<ref name=ft>{{cite news|url=http://www.ft.com/intl/cms/s/0/8e1af1ce-f040-11e1-93fa-00144feabdc0.html#axzz251hBQ76G  |title=‘Bull’ proves hero and pariah for Beijing |author= Enid Tsui |publisher= 《金融时报》 |date= 2012-08-28 |accessdate=2012-08-30 }}</ref>。

== 其他事跡 ==
=== 港鐵站示威案 ===
2016年4月8日，曾健成在[[港鐵|港鐵]][[旺角站|旺角站]]抗議[[港鐵票價調整|港鐵加價]]，在站內張貼示威標語，港鐵職員上前勸阻但不獲理會，遂以違反《港鐵附例》作出票控。2017年3月7日，於[[觀塘裁判法院|觀塘裁判法院]]被裁定罪名成立，罰款1,500港元，裁判官指附例沒違反《[[基本法|基本法]]》及《人權法》，港鐵有法定理據去保障港鐵財產及防止公眾被滋擾，控方指出港鐵有檢控權，被告的示威自由並非絕對。曾健成則稱《附例》較《公安條例》更保守及壓制[[人權|人權]]<ref>{{cite web|title=曾健成港鐵貼標語罪成罰款|url=https://news.mingpao.com/pns/dailynews/web_tc/article/20170308/s00002/1488909199547|agency=明報|date=2017-03-08}}</ref><ref>{{cite web|title=抗港鐵加價貼標語 曾健成罰千五元|url=http://orientaldaily.on.cc/cnt/news/20170308/00176_042.html|agency=東方日報|date=2017-03-08}}</ref>。其後，他申請[[法律援助|法律援助]]<ref>{{cite web|title=曾健成就旺角站示威遭票控上訴 官容許押後再排期|url=http://hd.stheadline.com/news/realtime/hk/977234/|agency=頭條日報|date=2017-08-08}}</ref>。

2018年，他就控罪提出上訴<ref>{{cite web|title=反港鐵加價被指違例 曾健成上訴|url=https://www.inmediahk.net/node/1056474|agency=香港獨立媒體網|date=2018-04-19}}</ref>。6月，[[香港高等法院|高等法院]]駁回其上訴申請，維持原判，曾認為《港鐵附例》第32條（a）的定義太寬廣，即時再向高等法院申請上訴至[[香港終審法院|終審法院]]獲批准<ref>{{Cite web |url=https://topick.hket.com/article/2089683/%E6%B8%AF%E9%90%B5%E5%A4%A7%E5%A0%82%E5%85%A7%E7%A4%BA%E5%A8%81%E7%BD%B0%E6%AC%BE%E3%80%80%E6%9B%BE%E5%81%A5%E6%88%90%E5%87%86%E4%B8%8A%E8%A8%B4%E8%87%B3%E7%B5%82%E9%99%A2 |title=港鐵大堂內示威罰款 曾健成准上訴至終院 |date=2018-06-07 |publisher=香港經濟日報}}</ref>。

=== 時代廣場擺放劉曉波銅像 ===
2018年5月31日起，曾健成與多名義工在[[銅鑼灣|銅鑼灣]][[時代廣場|時代廣場]]對開鐘樓旁，先後擺放一個已故中國[[持不同政見者|持不同政見者]][[劉曉波|劉曉波]]的白色半身像和一個[[銅|銅]]製全身坐像，原定在露天廣場展出至2018年7月13日，即劉逝世1周年，再移師至[[金鐘|金鐘]][[添馬公園|添馬公園]]舉行追思會。6月7日，曾健成收到時代廣場寄出的信件，指因颱風逼近香港，天文台指風力將增強並發出[[三號強風信號|三號強風信號]]，「基於公共安全」以及「收到大量顧客和鄰近商戶投訴」指他們發出噪音和阻塞通道，要求他們盡快拆除搭建物，並離開該範圍<ref>{{Cite web |url=https://news.mingpao.com/ins/instantnews/web_tc/article/20180609/s00001/1528540809357 |title=時代廣場外搭帳篷放劉曉波像　曾健成收律師信　望擺雕像至劉逝世一周年 |date=2018-06-09 |publisher=明報}}</ref>。6月8日，現場義工收到專人派來的律師信，指他們於未經時代廣場授權下，在露天廣場搭建帳篷及展台，以揚聲器宣傳政治理念，並邀請及接受途人捐款，聲稱有關含商業元素的行為不被允許及會被視為干擾公眾，要求他們立即離開並移除物品，及保留索償權利，在無可奈何下廣場將向法院申請禁制令<ref name="眾新聞0612">{{Cite web |url=https://www.hkcnews.com/article/12753/%E6%9B%BE%E5%81%A5%E6%88%90-%E6%99%82%E4%BB%A3%E5%BB%A3%E5%A0%B4-%E5%8A%89%E6%9B%89%E6%B3%A2-12800/%E5%8A%89%E6%9B%89%E6%B3%A2 |title=「熱心市民」花$17萬製劉曉波坐像放時代廣場　曾健成上周收律師信　冀能擺至周年忌辰 |author=吳婉英 |date=2018-06-12 |publisher=眾新聞}}</ref>。

[[民主黨_(香港)|民主黨]][[何俊仁|何俊仁]]呼籲時代廣場能「有點人情味」，指劉曉波奉獻一生爭取民主與和平，希望能讓公眾向他表達敬意<ref>{{Cite web |url=https://www.inmediahk.net/node/1057640 |title=劉曉波銅像揭幕 支聯會社民連籲聯署促釋放劉霞 |author=何雪瑩、王沛嘉 |date=2018-06-12 |publisher=香港獨立媒體網}}</ref>。

最終，該些雕塑和物品在6月19日由[[社民連|社民連]]派員搬走，遷移至[[銅鑼灣|銅鑼灣]][[百德新街|百德新街]]和[[記利佐治街|記利佐治街]]交界空地擺放至7月13日劉曉波逝世1周年當天，呼籲市民繼續前往參觀，同時批評時代廣場出於政治理由打壓表達政見自由；時代廣場回應稱露天廣場位置地段屬私人擁有，擺放物品不符合契約准許的「開放予公眾作行人通道和靜態康樂用途」<ref>{{Cite web |url=https://news.mingpao.com/ins/instantnews/web_tc/article/20180619/s00001/1529416096875 |title=時代廣場稱申請禁制令　劉曉波銅像移往百德新街　社民連質疑打壓表達政見自由 |date=2018-06-19 |publisher=明報}}</ref><ref>{{cite web|url=http://hk.on.cc/hk/bkn/cnt/news/20180619/bkn-20180619155744593-0619_00822_001.html|title=劉曉波像放時代廣場被趕 社民連執拾離場|date=2018-06-19|work=東網}}</ref>。

《[[眾新聞|眾新聞]]》和《[[852郵報|852郵報]]》指出在2010年和2014年，分別有團體於廣場擺放[[民主女神像|民主女神像]]和[[六四事件|六四]]屠城浮雕與[[坦克|坦克]]，但未見阻撓，雖遭當局票控，但最後獲判上訴得直<ref name="眾新聞0612"/><ref>{{Cite web |url=https://www.post852.com/251777/%E6%99%82%E4%BB%A3%E5%BB%A3%E5%A0%B4%E5%87%BA%E5%BE%8B%E5%B8%AB%E4%BF%A1%E8%A6%81%E6%B1%82%E6%90%AC%E8%B5%B0%E5%8A%89%E6%9B%89%E6%B3%A2%E5%83%8F%E3%80%802010%E5%B9%B4%E6%93%BA%E6%B0%91%E4%B8%BB/ |title=時代廣場出律師信要求搬走劉曉波像　2010年擺民主女神像未見阻撓 |date=2018-06-10 |publisher=852郵報}}</ref>。

=== 擺放慰安婦銅像 ===
2017年8月15日起，曾健成連同[[保釣行動委員會|保釣行動委員會]]在[[中區行人天橋系統|中環行人天橋]]對正[[交易廣場|交易廣場]][[日本駐香港總領事館|日本駐香港總領事館]]外擺放[[慰安婦少女銅像|慰安婦少女銅像]]，抗議[[日本首相|日本首相]][[安倍晉三|安倍晉三]]的慰安婦及其右翼政策，要求[[日本政府|日本政府]]道歉賠償。
