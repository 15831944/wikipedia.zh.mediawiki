==生平==
1964年，蒙赫奥尔吉勒生于[[苏赫巴托尔省|苏赫巴托尔省]]西乌尔特县。1982年毕业于[[乌兰巴托市|乌兰巴托市]]23中学。1988年毕业于[[莫斯科国际关系学院|莫斯科国际关系学院]]。1996年毕业于[[美国|美国]][[哈佛大学|哈佛大学]]法学院。1988年至1991年，任[[蒙古人民共和国|蒙古人民共和国]]外交部随员。1991年至1995年，任蒙古常驻[[联合国|联合国]]代表团二秘。1996年至1997年，在“索比联合”律师事务所当律师。1997至2000年，在“蒙赫奥尔吉勒”律师事务所当律师。2000年至2004年，任[[蒙古国|蒙古国]]法律和内务部副部长。2004年至2008年，任[[国家大呼拉尔|国家大呼拉尔]]委员。2004年至2006年，任[[蒙古国|蒙古国]]外交部部长。2006年至2007年，担任[[国家大呼拉尔|国家大呼拉尔]]法律常设委员会主席。2007年至2008年，任蒙古国法律和内务部部长。2008年，再度当选为国家大呼拉尔委员<ref name=rq/>。2016年，又当选为国家大呼拉尔委员<ref name=mfam/>。2016年，出任[[额尔登巴特内阁|额尔登巴特内阁]]的对外关系部部长<ref>{{Cite web |url = http://news.163.com/17/0220/19/CDO99N5O00018AOQ.html |title = 李源潮会见蒙古国外长蒙赫奥尔吉勒 |author =  |date =2017-02-20  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。
