'''曾偉權'''（Savio Tsang Wai Kuen，{{bd|1961年|11月22日|||catIdx=曾}}），香港男演員，前[[香港電視網絡|香港電視網絡]]藝人，現為[[無綫電視藝員列表|無綫電視]]基本藝人合約藝員。

== 背景 ==
===早年生活===
曾偉權為[[羅馬天主教|羅馬天主教]]虔誠信徒。父母生產豬油，上有三兄一姊，為家中么子。因為父母工作忙碌，無暇照顧，自小便通街跑，經常流連佐敦碼頭、佐治五世公園等。<ref>[http://tvb4life.pixnet.net/blog/post/192737514-%E5%96%AE%E4%BA%BA%E8%A8%AA-%E6%94%BE%E4%B8%8B%E4%BE%86%E8%B5%B0%E5%87%BA%E5%8E%BB%E3%80%80%E6%9B%BE%E5%81%89%E6%AC%8A-%E5%BF%BD%E7%84%B61%E5%91%A8-1016%E6%9C%9F 單人訪 放下來走出去 曾偉權-《忽然一周》1016期]</ref>曾偉權曾形容自己小時候屬於街童，因此小學五年級時，父母就送他與年長兩歲的哥哥一同到[[澳門|澳門]]就讀軍訓式的寄宿男校[[慈幼中學|慈幼中學]]，校規異常嚴格，使他吃過不少苦頭，但那段經歷亦培養出他獨立刻苦的性格。團體生活也令他日後加入電視圈時更容易與人溝通協調。雖然個性沉靜，但在學期間熱衷參加各種課外活動，舉凡排球、推鉛球、跳遠、擲鐵餅、標槍等。每年比賽三甲必入，運動成績彪炳，是校內風雲人物。另外，他的花名是「會長」，因為曾做過班長與學生會會長。同時因喜歡看電影，曾在校內搞話劇和參與樂隊演出。他自言應該就是從那時培養出表演慾。曾偉權中學畢業後返港與父母住在九龍大角咀。在一次茶聚期間，他的同學提議他報考藝員訓練班。曾偉權於是跟著去湊熱鬧。當時[[麗的電視|麗的電視]]與[[無綫電視|無綫電視]]同一時間招生。當他到達[[九龍塘|九龍塘]]時發現無綫招生的隊伍很長，便選擇先排麗的，之後再去報無綫，豈料人其實是排在裡面，排完出來已經5點，而無綫的招生時間於下午6點結束，但外面還有百多人在排象。曾偉權索性直接放棄。{{noteTag|關於曾偉權早年生活和入行經過的資料出自香港電台第二台廣播節目《守下留情》於2012年9月29日播出的「近代豪俠系列」訪問。}}

===演藝發展===
曾偉權畢業於1981-1982年[[麗的電視|麗的電視]]最後一期演員訓練班，與[[何家勁|何家勁]]，[[葉玉萍|葉玉萍]]是同學。其第一部出鏡的電視劇是《[[愛情跑道|愛情跑道]]》，在大結局中客串一角，沒有台詞，只是實習性質。1982年，麗的電視台曾想在曾偉權與[[何家勁|何家勁]]之間選擇演出《[[大將軍|大將軍]]》男主角。因造型原因輸給[[何家勁|何家勁]]，轉而被派演《[[琥珀青龍|琥珀青龍]]》，與[[葉玉萍|葉玉萍]]演出第二男女主角。為了加深觀眾印象，電視台於宣傳時分別形容他倆為翻版[[潘志文|潘志文]]和縮水[[米雪|米雪]]。曾偉權接連參演多部劇集，逐漸嶄露頭角，於1983年憑《[[101拘捕令|101拘捕令]]》中充滿衝勁的警察「李長仔」一角，大受歡迎。令他奪得亞視台慶夜最佳新人獎，並當選當年度的《[[華僑日報|華僑日報]]》十大電視明星金球獎，這也是他個人最鐘愛的角色之一。1984年，他相繼演出《[[霍東閣|霍東閣]]》、《[[武則天|武則天]]》等大紅劇集，再次得到十大電視明星金球獎。他在[[亞洲電視|亞洲電視]]被列為重點栽培的小生，常以主角身份出演劇集。比較突出的演出是1984年《[[雲海玉弓緣|雲海玉弓緣]]》和1986年《[[白髮魔女傳|白髮魔女傳]]》。電視台亦有意將他與[[麥翠嫻|麥翠嫻]]塑造成螢幕情侶，在《[[天堂鳥|天堂鳥]]》、《[[霍東閣|霍東閣]]》、《[[靈界邊緣人|靈界邊緣人]]》等劇中多次配對合作。兩人亦曾一起擔任亞洲小姐競選的表演嘉賓，並於1985年代表亞視到[[台灣|台灣]]參加[[金鐘獎|金鐘獎]]。

曾偉權在1984年6月11日播出的劇集《[[十二金牌|十二金牌]]》（共30集）中曾被屬意演出[[岳飛|岳飛]]之子[[岳雲|岳雲]]。但後來由於演員調動，本來要改演[[張憲|張憲]]，但最後岳雲改由[[黎漢持|黎漢持]]飾演，張憲則由[[羅樂林|羅樂林]]頂上。而在1986年4月7日播出的《[[少林英雄|少林英雄]]》中，曾偉權被屬意飾演其中一位男主角，他卻以身體不適為由辭演，改由[[江華|江華]]頂替。同年因紀律問題被提前一年解約，在外拍攝獨立公司製作的影帶劇，如《[[飛渡長城|飛渡長城]]》、《[[禿鷹特警隊|禿鷹特警隊]]》等。後者曾於亞視播映。隨著電視台易主，曾偉權亦得以重返亞視，拍攝《[[霹靂神將|霹靂神將]]》、《[[傅紅雪傳奇|傅紅雪傳奇]]》等劇。1990年，他為了尋求事業上的突破，轉投[[無綫電視|無綫電視]]。但由於無綫人才眾多、加上注重力捧制度，在擔正過幾部電視劇後，一直徘徊在三、四線，演出的多屬閒角。而且觀眾對他的注意不多，至今未嘗大紅。{{noteTag|以上關於曾偉權演藝歷程的資料出自香港電台第二台廣播節目《守下留情》於2012年10月6日播出的「近代豪俠系列」訪問。}}曾偉權在2012年轉投[[香港電視網絡|香港電視網絡]]發展，2015年11月又重回[[無綫電視|無綫電視]]。

曾偉權出道以來演過不少角色，雖然其演出多屬閒角，但仍間中參演過戲份較吃重的角色，代表作有以《[[雲海玉弓緣|雲海玉弓緣]]》「金世遺」、《[[白髮魔女傳|白髮魔女傳]]》「卓一航」、《[[師奶股神|師奶股神]]》「'''朱玉麟'''」、《[[女人最痛|女人最痛]]》「'''林加文（GM）'''」、《[[點解阿Sir係阿Sir|點解阿Sir係阿Sir]]》進興集老大「'''倪鋒（Charles）'''」及《[[回到三國|回到三國]]》「'''[[張飛|張飛]]'''」等，皆受關注且廣獲好評。他雖然以飾演反派著名，但亦飾演了不少富正義感或搞笑的角色，如《[[師奶股神|師奶股神]]》「'''朱玉麟'''」和《[[回到三國|回到三國]]》「'''[[張飛|張飛]]'''」等等。在個人生活方面，他的圈內好友甚多如[[伍衛國|伍衛國]]、[[潘志文|潘志文]]等。

== 軼聞 ==
曾偉權是個愛犬人士，小時候每年暑假都要回家裡工廠幫忙，經常與附近聚集的三四十隻狗相處，最愛觀察牠們的一舉一動。他說很多藝人都熱愛動物，有些人會去餵街狗等，但曾偉權認為單靠餵不是辦法，所以很早以前已經開始擔任動物義工，替流浪狗做絕育放回的工作，至今收養過十多隻狗，目前飼養的5隻狗為丫B、豬古力、小吉、妹妹和多多。

他經常於個人臉書與微博發表保護貓狗的訊息，2014年8月港鐵撞死唐狗「未雪」事件，曾到場抗議<ref>[http://www.hkanimalpost.com/2014/09/30/%E3%80%8A%E6%83%85%E5%B8%B8%E5%9C%A8%E3%80%8B%E5%B0%88%E8%A8%AA%E8%97%9D%E4%BA%BA%E6%9B%BE%E5%81%89%E6%AC%8A%EF%BC%88%E4%B8%8B%EF%BC%89%EF%BC%9A%E9%82%A3%E4%BA%9B%E5%B9%B4%EF%BC%8C%E6%88%91%E8%AA%8D/ 《情常在》專訪藝人曾偉權<下>]</ref>。9年前亦曾與[[宣萱|宣萱]]一同在電視城救狗<ref>[http://www.hkanimalpost.com/2014/09/25/%E3%80%8A%E6%83%85%E5%B8%B8%E5%9C%A8%E3%80%8B-%E5%B0%88%E8%A8%AA%E8%97%9D%E4%BA%BA%E6%9B%BE%E5%81%89%E6%AC%8A-%E4%B8%8A-%E6%95%91%E7%8B%97%E8%A8%98/ 《情常在》專訪藝人曾偉權<上>]</ref>。

2011年左右為從事鋼琴教學的梁姓網友（花名「波兒」）解決養狗問題，一見如故，認她為乾妹，並出資開設琴行[[Twinkle_Music|Twinkle Music]]，擔任行政總監。

== 感情生活 ==
曾偉權曾與麗的藝員訓練班同學[[楊雅潔|楊雅潔]]相戀三年，兩人曾在《[[雲海玉弓緣|雲海玉弓緣]]》等戲中合作，但1984年7月便因性格不合而分手（10月一度復合）。1986年演出著名監製[[梅小青|梅小青]]所製作的賀歲喜劇《[[滿堂紅|滿堂紅]]》時，與梅小青之妹，同時也是該戲女主角[[梅小惠|梅小惠]]相識，雙方陷入愛河，交往13年，卻於1999年9月因為馬來西亞女歌手[[葉麗儀_(馬來西亞)|葉麗儀]]介入而分手，引起軒然大波，星途亦大受影響。<ref>{{web reference|URL=http://std.stheadline.com/archive/fullstory.asp?year1=2001&month1=3&day1=17&month2=3&day2=17&id=20010317f05|title=曾偉權主動公開戀情|work=星島日報|year=2001年|date=3月17日}}</ref>交往3年後，亦因遠距離戀愛維持不易而分手，目前單身。
