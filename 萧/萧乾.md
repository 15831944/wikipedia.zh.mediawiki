'''萧乾'''（{{bd|1910年|1月27日|1999年|2月11日|catIdx=Xiao,萧}}），生于[[北京|北京]]，[[中國蒙古族|蒙古族]]，原名'''萧秉乾'''、'''萧炳乾'''。是[[作家|作家]]、[[记者|记者]]和[[翻译家|翻译家]]。

== 生平 ==
1931年進入[[北平輔仁大學|北平輔仁大學]]，1933年轉到[[燕京大學|燕京大學]]就讀。1935年于燕大畢業，任职《[[大公报|大公报]]》。1939年远赴[[英国|英国]]，成为[[战地记者|战地记者]]。有《[[银风筝下的伦敦|银风筝下的伦敦]]》等著名报道。1949年萧乾回到北京，[[反右|反右]]期间被打为[[右派|右派]]，文革期間與[[愛人|愛人]][[文潔若|文潔若]]備受迫害，后平反。1999年，萧乾因[[心肌梗塞|心肌梗塞]]于北京逝世，终年89岁。

== 作品 ==
他的翻译作品有《[[莎士比亚戏曲故事集|莎士比亚戏曲故事集]]》（[[英國|英國]][[兰姆|兰姆]]姊弟合著）、《[[好兵帥克|好兵帥克]]》（從英語譯本轉譯）、《[[培尔·金特|培尔·金特]]》、加拿大里柯克幽默小品集等。1990年，萧乾和愛人[[文潔若|文潔若]]应[[译林出版社|译林出版社]]之邀，着手翻译[[詹姆斯·乔伊斯|詹姆斯·乔伊斯]]的巨作《[[尤利西斯|尤利西斯]]》，四年乃成，但他卻在《[[瑞士之行|瑞士之行]]》一文中表示：“这里躺着世界文学界一大叛徒。他使用自己的[[天才|天才]]和学识向极峰[[探险|探险]]，也可以说是浪费了一份禀赋去走死-{胡同}-。究竟是哪一样，本世纪恐难下断语。”

他與愛人曾合作翻譯多種英語文學作品，夫婦二人還合作從英語譯本轉譯[[俄國|俄國]][[托爾斯泰|托爾斯泰]]的短篇作品集。

== 友人 ==
萧乾自称“未带[[地图|地图]]的旅人”，在[[文坛|文坛]]前辈[[冰心|冰心]]口中，则是昵称“饼干”的弟弟。他与[[巴金|巴金]]、[[钱锺书|钱锺书]]等交好，与[[郭沫若|郭沫若]]、[[沈从文|沈从文]]、[[曹禺|曹禺]]等则颇有嫌隙。英国作家[[E·M·福斯特|E·M·福斯特]]曾经推荐萧乾去[[剑桥|剑桥]]读书，相交甚厚，后来则由于种种原因断交。
