《'''萧红'''》（英語：Falling Flowers），讲述著名女作家[[萧红|萧红]]生平故事，导演[[霍建起|霍建起]]，主演[[宋佳|宋佳]]、[[黄觉|黄觉]]等。中国于2013年3月8日上映。

== 剧情 ==
该片为纪念女作家萧红诞生100周年而拍摄，讲述了萧红从[[抗日战争|抗日战争]]、[[国共内战|国共内战]]，流亡于[[哈尔滨|哈尔滨]]、[[上海|上海]]、[[山西|山西]]，婚姻不幸，转而爱上[[端木蕻良|端木蕻良]]，最后客死[[英属香港|英属香港]]。<ref>{{cite web |url=http://news.163.com/12/0515/16/81ID2A1J00014AED.html |title= 电影《萧红》(组图)|accessdate=2012-05-15 |publisher=网易 |work= |language =zh-cn}}</ref>

==换角==
2011年8月4日，在北京人民大会堂举行于新闻发布会上，[[王珞丹|王珞丹]]饰演萧红。<ref>{{cite web|url=http://ent.sina.com.cn/m/c/2011-08-05/04053378093.shtml|title= 王珞丹新片饰演女作家萧红 和黄觉演绎夫妻 | date =2011年8月5日 | work= 新京报 | publisher= [[新浪|新浪]] | author =  sunny}}</ref>2012年1月4日剧组工作人员向记者解释，王珞丹的档期安排不开了，所以换成了小宋佳。<ref>{{cite web|url= http://ent.sina.com.cn/m/c/2012-01-04/16033524661.shtml |title= 小宋佳《萧红》含泪吻黄觉零下20度拍2小时 | date = 2012年1月4日 | work= 北国网-辽沈晚报 | publisher= [[新浪|新浪]] | author =  宋琰}}</ref>

== 花絮 ==
该片先后辗转于[[北京|北京]]、[[上海|上海]]、[[哈尔滨|哈尔滨]]、[[沈阳|沈阳]]、[[吉林|吉林]]、[[香港|香港]]、[[武汉|武汉]]、[[重庆|重庆]]等10多地取景。<ref name=" fish" >{{cite web |url=http://ent.qq.com/a/20111226/000327.htm#p=5 |title=《萧红》圣诞关机 主创分享幕后趣事尽显温情 |accessdate=2011年12月26日 |publisher=腾讯网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.cntv.cn/20111227/125885.shtml |title= 电影《萧红》封镜|accessdate=2011年12月27日 |publisher=央视 |work= |language =zh-cn}}</ref>

导演霍建起宣布结束时，来电通知他喜得贵子。<ref name=" fish" />

2012年5月24日在北京首映；该片中有激情戏。<ref>{{cite web|url=http://www.dmcc.gov.cn/publish/main/177/2012/20120525090201372507023/20120525090201372507023_.html|title=电影《萧红》在京首映 小宋佳纠结与黄觉激情戏|accessdate=2012-05-25|publisher=国家广电总局|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20141221225033/http://www.dmcc.gov.cn/publish/main/177/2012/20120525090201372507023/20120525090201372507023_.html|archivedate=2014-12-21}}</ref>

由于该片语言半文半白、镜头过分煽情，遭到新闻媒体批评。<ref>{{cite web |url=http://ent.ifeng.com/movie/special/15thsiff/zxbd/detail_2012_06/25/15535663_0.shtml?_from_ralated |title=电影《萧红》：悲罗襟之宵离 怨秋夜之未央 |accessdate=2012年6月25日 |publisher= 凤凰网|work= |language =zh-cn}}</ref>
