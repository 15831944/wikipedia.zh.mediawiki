'''萧山经济技术开发区'''是位于[[杭州|杭州]][[萧山区|萧山区]]、1993年5月建立的[[国家级经济技术开发区|国家级经济技术开发区]]。2009年萧山开发区与'''[[江东新城|江东新城]]'''（江东工业园区）党工委、管委会实行“一套班子、两块牌子”管理。<ref name="开发区">{{cite web |url=http://www.hangzhou.gov.cn/main/zjhz/hzlj/2010/gjjkfq/T350546.shtml |language= |title=萧山经济技术开发区 |publisher=中国杭州 |date=2011-04-14 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20130609075728/http://www.hangzhou.gov.cn/main/zjhz/hzlj/2010/gjjkfq/T350546.shtml |archivedate=2013-06-09 }}</ref>

== 简介 ==
开发区下辖市北、桥南、江东三大区块<ref>[http://www.xetz.gov.cn/gb/kfqgk/kfqgh.htm 开发区规划] {{webarchive|url=https://web.archive.org/web/20120906034603/http://www.xetz.gov.cn/gb/kfqgk/kfqgh.htm |date=2012-09-06 }}</ref>，总规划面积181平方公里，区内设有杭州江东国家'''新能源'''高新技术产业化基地、'''装备制造'''国家新型工业化产业示范基地、国家级杭州'''[[软件|软件]]产业'''基地萧山扩展区块，以及[[江东新城|江东新城]]和[[萧山高新技术产业园区|萧山高新技术产业园区]]这两个省级开发区。<ref name="简介">{{cite web |url=http://www.xetz.gov.cn/gb/kfqgk/kfqjj.htm |language= |title=开发区简介 |publisher=萧山经济技术开发区 |date= |accessdate=2012年9月 |deadurl=yes |archiveurl=https://web.archive.org/web/20120815154842/http://www.xetz.gov.cn/gb/kfqgk/kfqjj.htm |archivedate=2012-08-15 }}</ref>

截至2012年6月，开发区已引进外资企业550个，总投资84亿美元，其中有237个总投资达1000万美元以上的项目；批准内资项目357个；引进14家世界500强企业，基本形成了机械制造、电子电器、汽车整车及重点零部件、轻纺服装、医药食品、建材家具这六大支柱性产业。<ref name= "简介"/>
