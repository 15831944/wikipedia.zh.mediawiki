'''萧邺'''（{{bd|9世纪||874年|catIdx=X}}），字'''启之'''，[[唐朝|唐朝]]官员，[[唐宣宗|唐宣宗]]年间任[[宰相|宰相]]，[[唐懿宗|唐懿宗]]年间也短暂任相。

== 家世和早期仕途 ==
萧邺出自[[兰陵萧氏|兰陵萧氏]]齐梁房，是南梁长沙宣王[[萧懿|萧懿]]九世孙，八世祖[[萧渊明|萧渊明]]。萧邺的祖父[[萧直|萧直]]曾任[[给事中|给事中]]，父亲[[萧革|萧革]]曾任[[邵州|邵州]][[刺史|刺史]]。萧邺至少有一个哥哥[[萧嶰|萧嶰]]和一个弟弟[[萧岘|萧岘]]。萧邺考中[[进士|进士]]，屡次升任[[监察御史|监察御史]]、[[翰林学士|翰林学士]]，外任[[衡州|衡州]]刺史<ref name=NBT182>[[:s:新唐書/卷182|《新唐书》卷一百八十二]]</ref>。

== 唐宣宗年间 ==
唐宣宗[[大中_(唐朝)|大中]]（847年）元年，萧邺以[[监察御史|监察御史]]里行充[[翰林学士|翰林学士]]，<ref>[[:s:授萧邺李元监察御史制|崔嘏《授萧邺李元监察御史制》]]</ref><ref>[[:s:授萧邺翰林学士制|崔嘏《授萧邺翰林学士制》]]</ref>二年（848年）十一月加知制诰，三年（849年）九月责授[[衡州|衡州]][[刺史|刺史]]。<ref>《重修承旨学士壁记》</ref>后历任中书舍人、[[户部侍郎|户部侍郎]]，判本司。<ref name=NBT182/>四年（850年）六月，为[[高元裕|高元裕]]作《[[:s:大唐故吏部尚书赠尚书右仆射渤海高公神道碑|大唐故吏部尚书赠尚书右仆射渤海高公神道碑]]》。五年（851年），受病重的表叔伯[[韦正贯|韦正贯]]所托，<ref>《韦正贯墓志》：“公遗令薄葬，不请谥于太常，不用鼓吹，而请姑之孙翰林学士中书舍人萧邺纪行于豊碑”。</ref>在翰林学士、中书舍人任上在其死后作《[[:s:岭南节度使韦公神道碑|岭南节度使韦公神道碑]]》。后改判度支，宣宗欲任学士承旨[[韦澳|韦澳]]代之判户部，韦澳不肯。<ref>[[:s:旧唐书/卷158|《旧唐书》卷一百五十八]]</ref>十年（856年）十一月萧邺在[[兵部侍郎|兵部侍郎]]、判度支任上时，宣宗准备拜他为宰相，甚至已经派枢密宣旨了。但当他将旨意送往学士院时，枢密使[[王归长|王归长]]、[[马公儒|马公儒]]却问是否仍让萧邺判度支，导致宣宗怀疑他们和萧邺结党，因而改任[[户部侍郎|户部侍郎]]、判户部[[崔慎由|崔慎由]]为宰相。崔慎由和萧邺同在翰林院时就不和，崔慎由拜相后就罢免了萧邺的翰林学士。但这对萧邺拜相来说，仅仅是延迟。次年（857年）七月，他以朝散大夫、守尚书兵部侍郎、判度支、[[上柱国|上柱国]]、'''彭城县开国男'''、食邑三百户、赐紫金鱼袋被拜为[[同中书门下平章事|同中书门下平章事]]，<ref>[[:s:新唐书/卷8|《新唐书》卷八]]</ref><ref>《旧唐书·宣宗纪》《[[册府元龟|册府元龟]]》作六月拜相。</ref>恩宠隆重，引[[刘瑑|刘瑑]]同为宰相，且短期内仍判度支，<ref>[[:s:授萧邺平章事制|:s:授萧邺平章事制]]</ref>十月，授兼集贤殿学士，<ref>[[:s:授崔慎由监修国史萧邺集贤殿学士制|:s:授崔慎由监修国史萧邺集贤殿学士制]]</ref>十一月以尚书兼[[工部尚书|工部尚书]]，十二月才罢免判度支一职。宣宗余年，他维持相位，<ref name=ZZTJ249>[[:s:資治通鑑/卷249|《资治通鉴》卷二百四十九]]</ref>十二年（858年）二月，监修国史，<ref>[[:s:授萧邺监修国史刘瑑集贤殿大学士制|:s:授萧邺监修国史刘瑑集贤殿大学士制]]</ref>当月趁崔慎由因请求立[[皇太子|皇太子]]而为宣宗所恶之机陷害之，使其被出为[[东川节度使|东川节度使]]。<ref>[[:s:旧唐书/卷177|《旧唐书》卷一百七十七]]</ref><ref>[[:s:新唐书/卷114|《新唐书》卷一百一十四]]</ref>十三年（859年）三月，罢相，守[[吏部尚书|吏部尚书]]。<ref>[[:s:旧唐书/卷18下|《旧唐书》卷一十八下]]</ref>宣宗本欲以户部侍郎判度支[[杜胜|杜胜]]为宰相，但在萧邺罢相后，杜胜被宦官诋毁而不得志。<ref>[[:s:新唐书/卷169|《新唐书》卷一百六十九]]</ref>

== 唐宣宗身后 ==
八月，唐宣宗崩，萧邺以中书侍郎兼[[礼部尚书|礼部尚书]]平章事摄中书令奉册拥立皇长子继位为[[唐懿宗|唐懿宗]]。<ref>[[:s:命皇太子即位册文|:s:命皇太子即位册文]]</ref>九月，以[[门下侍郎|门下侍郎]]兼[[兵部尚书|兵部尚书]]平章事摄[[太尉|太尉]]，奉册上懿宗亡母尊谥[[元昭皇后|元昭皇太后]]。<ref>[[:s:唐懿宗元昭皇太后谥册文|夏侯孜《唐懿宗元昭皇太后谥册文》]]</ref>十月，诏兼[[尚书右仆射|尚书右仆射]]，复拜相，复官居中书侍郎兼礼部尚书平章事。<ref>[[:s:万敬儒孝行状碑|卢潘《万敬儒孝行状碑》]]</ref>十一月，以[[银青光禄大夫|银青光禄大夫]]、守门下侍郎兼工部尚书、同中书门下平章事、监修国史、上柱国、'''彭城县开国男'''、食邑三百户的身份，罢相改任[[检校|检校]][[右仆射|右仆射]]同中书门下平章事兼[[江陵|江陵]]尹充荆南节度管内观察处置等使。<ref name=ZZTJ249/><ref>[[:s:新唐书/卷9|《新唐书》卷九]]</ref><ref>[[:s:授萧邺荆南节度使制|:s:授萧邺荆南节度使制]]</ref><ref>《旧唐书·懿宗纪》误作二年二月以吏部尚书为检校尚书右仆射、太原尹、北都留守、河东节度观察等使。</ref>[[咸通|咸通]]三年（862年）迁[[西川节度使|西川节度使]]，辟[[卢知猷|卢知猷]]为荆南、剑南两府掌书记记室，<ref>[[:s:旧唐书/卷163|《旧唐书》卷一百六十三]]</ref><ref>[[:s:新唐书/卷177|《新唐书》卷一百七十七]]</ref>但无力阻止[[南诏|南诏]]入侵。五年（864年），率属蛮鬼主在大渡河迎战南诏，败之。<ref>[[:s:新唐书/卷222中|《新唐书》卷二百二十二中]]</ref>二月，贬山南西道观察使，<ref name=ZZTJ250>[[:s:資治通鑑/卷250|《资治通鉴》卷二百五十]]</ref>直至八年（867年），<ref>[[吴廷燮|吴廷燮]]《方镇年表》</ref>后又历任户部、吏部尚书，十一年（870年）正月、十二年（871年）三月、十三年（872年）三月，皆以吏部尚书被任为考官，但十三年三月科举时被右丞[[孔温裕|孔温裕]]替代。<ref>[[:s:旧唐书/卷19上|《旧唐书》卷一十九上]]</ref>十四年（873年）十一月又充[[河东节度使|河东节度使]]，任上无可称。约[[乾符|乾符]]元年（874年）三月前卒于任上。<ref name=NBT182/><ref>乾符元年三月[[窦澣|窦澣]]被任为河东节度使。详见[[郁贤皓|郁贤皓]]《[[唐刺史考全编|唐刺史考全编]]》。</ref>
