'''萧山区'''是[[中華人民共和國|中國]][[浙江省|浙江省]][[杭州市|杭州市]]的[[市辖区|市辖区]]，原先后为[[绍兴专区|绍兴专区]][[萧山县|萧山县]]、浙江省直辖萧山县、[[宁波专区|宁波专区]]萧山县、杭州市萧山县、浙江省[[萧山市|萧山市]]（[[县级市|县级市]]，地级杭州市代管），2001年设立萧山区。地处[[钱塘江|钱塘江]]南岸，为杭州南大门，东接绍兴市[[柯桥区|柯桥区]]，西接[[富阳区|富阳区]]和[[诸暨市|诸暨市]]。

== 历史 ==
[[File:Xiaoshan_district_government_service_center.jpg|缩略图]]
[[File:Zhejiang_Xiaoshan_Hospital_02.jpg|缩略图]]]]
公元2年建[[县|县]]，称[[余暨|余暨]]，属会稽郡。[[三国|三国]][[吴|吴]][[黄武|黄武]]年间（公元222至229年），改名永兴。[[唐|唐]][[天宝|天宝]]元年（公元742年）改名萧山，1949年成为省直属县。1949年6月底，划归绍兴专区。1952年起，复为省直属县。1957年划归宁波专区。1959年改属杭州市。1988年撤县设市，1996年5月8日原萧山市[[西兴街道|西兴镇]]、[[长河街道|长河镇]]、[[浦沿街道|浦沿镇]]三镇划归[[杭州市|杭州市]][[西湖区_(杭州市)|西湖区]]（同年12月12日以三镇为基础成立[[滨江区|滨江区]]），2001年3月撤市设[[杭州市|杭州市]]'''萧山区'''。

== 气候 ==
属亚热带季风气候，四季分明、雨热同期、日照充足。1月平均气温4.5℃，7月平均气温28.9℃，年平均气温16.8℃。极端最高气温42.2℃（2003年7月25日、8月1日，2013年7月30日），极端最低气温-15.0℃（1977年1月5日）。年降水量1440.5毫米。年日照时数1804.6小时。

== 交通 ==
===铁路===
[[File:20171105从西广场观看杭州南站.jpg|thumb]]站房]]
[[File:201607_Intersection_of_Main_Line_and_Subline_of_Shanghai–Kunming_Railway_near_Bailutang.jpg|thumb]][[萧山站|萧山站]]分岔]]
萧山有[[沪昆铁路|沪昆铁路]]经过，[[萧甬铁路|萧甬铁路]]在萧山接入沪昆铁路，同时设有杭州绕行线、萧萧联络线、萧浙联络线等联络线。在高速铁路方面，则有[[杭长客运专线|杭长客运专线]]、[[杭甬客运专线|杭甬客运专线]]经过，在[[杭州南站|杭州南站]]设有杭南信号所作为两条线路的联络线，未来还有[[杭黄客运专线|杭黄客运专线]]经过。原浙赣铁路与1932年完成，铁路横穿萧山并在[[城厢街道_(杭州市)|城厢街道]]设[[萧山西站|萧山站]]，办理客货运业务。1992年联通[[萧山站|白鹿塘站]]和[[笕桥站|笕桥站]]的浙赣铁路绕行线完工，并在萧山设有办理客运业务的[[杭州南站|新萧山站]]和[[盈宁站|盈宁站]]，部分经由杭州的列车不再经由横穿萧山的老浙赣线、直接由绕行萧山城区的绕行先北上进入[[沪杭铁路|沪杭铁路]]和[[宣杭铁路|宣杭铁路]]，同时原萧山站改名[[萧山西站|萧山西站]]。2006年，原浙赣铁路绕行线与浙赣铁路、沪杭铁路合并成为[[沪昆铁路|沪昆铁路]]，而横穿萧山城区的浙赣铁路改名[[杭州绕行线|杭州绕行线]]。

2010年1月1日起，萧山站更名为杭州南站。<ref>[http://news.cnnb.com.cn/system/2010/01/06/006382500.shtml 铁路：萧山站改名为“杭州南站”]</ref><ref>[http://www.xsnet.cn/news/shms/2010_1/1040365.shtml 火车站：近八十载风雨中的三次更名（萧山网）]</ref> 在[[杭州东站|杭州东站]]站改过程中，杭州南站和[[杭州站|杭州站]]是[[杭州|杭州]]地区主要的两个客运站。2013年7月1日，新杭州东站启用后，杭州南站开始进入闭关改造阶段。南站站房及相关工程于2014年11月正式开建，预计2017年6月完工<ref >{{cite web|url=http://hz.house.sina.com.cn/news/2014-09-14/08314449647.shtml| language= | title=铁路杭州南站11月改建 计划2017年6月30日完工及檢收，7月1日正式啟用 |publisher=都市快报 |date=2014-09-14 |accessdate= }}</ref>。而萧山站的名称则于2015年用于[[萧山站|白鹿塘站]]，同时在该站周围将兴建萧山货场，今后将成为铁路杭州枢纽南部的最重要的组成部分，与[[钱塘江|钱塘江]]北岸的[[乔司站_(铁路)|乔司站]]一同形成“北乔司、南萧山”的杭州铁路货运新格局。<ref>{{cite news|url=http://two.crec4.com/content-87-1505-1.html|title=白鹿塘经理部承建的萧山铁路货场正式运营|publisher=中国中铁四局集团第二工程有限公司|date=2016-01-11}}</ref>

萧山所有的铁路车站除[[萧甬铁路|萧甬铁路]][[夏家桥站|夏家桥站]]由[[上海铁路局|上海铁路局]][[宁波车务段|宁波车务段]]管理外，均由[[上海铁路局|上海铁路局]][[杭州直属站|杭州直属站]]管理。<ref>{{cite web|title=铁路杭州站经营文化手册震撼来袭！|url=http://mp.weixin.qq.com/s?__biz=MzA3NjIzMDMyOQ==&mid=2650036933&idx=1&sn=9798061aa88e25ef908c1a9951733f89&chksm=87648113b01308059fdd0d1367374f21ebe2136e33c2fda0c40a19320433d8f018968a8ae4a8&mpshare=1&scene=1&srcid=1020FoJQBqSwFOQ2Najn58C5#rd|date=2016-10-20|publisher=上铁杭州站}}</ref>从前上海铁路局杭州铁路分局设有萧山车务段，以管理萧山地区的铁路车站。后萧山车务段于2000年合并至诸暨车务段<ref>{{cite news|title=杭州年鉴2000 铁路|url=http://www.hangzhou.gov.cn/art/2003/1/27/art_806912_133972.html|publisher=杭州市人民政府|date=2003-01-27}}</ref>，后者又于2005并入新成立的[[金华车务段|金华车务段]]<ref>{{cite web|title=2006绍兴年鉴 铁路运输|url=http://www.sx.gov.cn/art/2008/6/3/art_1462297_16998695.html|publisher=绍兴市人民政府}}</ref>，萧山的大部分车站改由金华车务段管理。[[金华车务段|金华车务段]]管理的萧山铁路车站又于2009年10月1日起移交[[杭州站|杭州直属站]]管辖<ref>{{cite web|title=杭州年鉴2010 交通运输 铁路|url=http://www.hangzhou.gov.cn/art/2011/4/13/art_806361_160300.html|date=2011-04-13|publisher=杭州市人民政府}}</ref>。

===航空===
[[File:201607_Hangzhou_Xiaoshan_Intl_Airport.jpg|thumb]]]]
[[杭州萧山国际机场|杭州萧山国际机场]]位于区内[[瓜沥镇|瓜沥镇]]，是[[杭州市|杭州市]]的主要民航机场，同时也是[[浙江省|浙江省]]以及[[长三角|长三角]]地区的重要机场，处于华东第三，全国前十的水平。机场于2001年开航，以取代杭州城区军民合用的矛盾日益突出的[[笕桥机场|笕桥机场]]。但随着萧山机场航班量日益增加，尤其是第二跑道启用后，萧山机场造成机场周边以及萧山城区受到噪声影响日益严重的事实亦不容忽略，引起部分当地居民的不满<ref>{{cite web|title=【环保 求助】萧山机场噪音问题需要务实解决|publisher=人民网 地方领导留言板|url=http://liuyan.people.com.cn/thread.php?tid=3652540&display=&domainid=13&act=domain&page=1|date=2016-05-21}}</ref>。

===地铁===
[[File:201607_Exit_A2_of_Renmin_Road_Station.jpg|thumb]]A2出口]]
[[杭州地铁1号线|杭州地铁1号线]]和[[杭州地铁2号线|2号线]]连接萧山区与杭州市中心。2012年11月24日通车试运营的[[杭州地铁1号线|杭州地铁1号线]]在萧山区设有[[湘湖站|湘湖站]]，同时在与[[滨江区|滨江区]]交界设有[[滨康路站|滨康路站]]，联通杭州主城、[[临平|临平]]和[[下沙|下沙]]地区。1号线三期规划则将线路从下沙延伸至[[萧山机场|萧山机场]]<ref >{{cite web|url=http://biz.zjol.com.cn/system/2016/06/14/021187661.shtml | language= | title= 杭州地铁7号线确定进机场 杭绍城际铁路下月动工|publisher=杭州网 |date=2016-06-14 |accessdate= }}</ref>。2014年11月24日开通运营的[[杭州地铁2号线|杭州地铁2号线]]纵贯整个萧山区，区内设有{{站|钱江世纪城}}、{{站|盈丰路}}、{{站|飞虹路}}、{{站|振宁路}}、{{站|建设三路}}、{{站|建设一路}}、[[人民广场站_(杭州市)|人民广场]]、{{站|杭发厂}}、[[人民路站_(杭州市)|人民路]]、{{站|潘水}}、{{站|曹家桥}}、[[朝阳站_(杭州地铁)|朝阳]]等站，联通萧山城区和杭州主城以及城西地区。

规划中的[[杭州地铁5号线|杭州地铁5号线]]长48公里，西起工教路，东南至[[杭州南站|杭州南站]]。规划中的[[杭州地铁7号线|杭州地铁7号线]]长31公里，位于萧山区内，西起[[钱江世纪城|钱江世纪城]]、并与[[杭州地铁2号线|2号线]]形成的双向同站台换乘，向东走向与机场高速平行直至[[萧山国际机场|萧山国际机场]]，后沿北直到[[江东工业园|江东工业园]]。该线为连接线，目的是构建江东地区与钱江世纪城的联系。[[杭绍城际铁路|杭绍城际铁路]]也在规划中。

== 宗教 ==
[[File:萧山_江寺.jpg|right]]
[[佛教|佛教]]传入萧山，至少已有1600多年历史，东晋咸和六年（331）始建祇(qí)园寺（原名崇化寺，萧山体育路西端）。萧山佛教寺庙主要有曹洞正宗和临济正宗两派佛教禅宗支脉，前者如曹山寺、云门寺等，后者如白龙寺、冠山寺、宝寿禅寺等。寺庙作为佛教活动载体，在传播佛文化同时，产生了一批名僧高师。如竹林寺高昙禅师、宝寿禅寺独超法师、曹山寺净生起祖禅师等，他们问禅教化、济世治病，对萧山佛教文化的发展做出积极贡献。

萧山是浙江省内除[[温州|温州]]以外，另一个[[基督徒|基督徒]]众多的地区，人数超过10万，其中95%属于[[地方教会|地方教会]]背景。

== 方言 ==
因萧山古代一直属于会稽下的辖区，因此萧山方言不同于杭州，属于[[吴语|吴语]]临绍小片，与[[绍兴|绍兴]]方言基本相近。
