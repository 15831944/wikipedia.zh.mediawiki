'''萧灼基'''（{{bd|1933年|12月4日|2017年|11月14日}}），[[广东|广东]][[汕头|汕头]]人，[[中华人民共和国|中华人民共和国]]经济学家。

==生平==
1953年9月考入[[中国人民大学|中国人民大学]]经济系政治经济学专业，因品学兼优而在三年级时便被推荐读政治经济学专业研究生，师从经济学家[[宋涛_(经济学家)|宋涛]]教授。1959年7月研究生毕业后，被分配到[[北京大学|北京大学]]经济系任助教，1979年6月晋升为讲师，1980年12月晋升为副教授，1985年9月晋升为教授，1986年7月被评为[[博士生导师|博士生导师]]，1992年起享受国务院有突出贡献专家津贴<ref name=shishi>{{Cite web |url =http://finance.ifeng.com/a/20171115/15793757_0.shtml  |title = 著名经济学家、北大教授萧灼基逝世 享年84岁 |author =  |date = 2017-11-15 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

萧灼基曾任[[第九届全国政协委员|第九届全国政协委员]]、[[第十届全国政协委员|第十届全国政协常务委员]]、全国政协社会与法制委员会副主任，第六、第七届[[中国民主建国会|中国民主建国会]]常务委员、特别顾问、中国民主建国会经济委员会主任，第五届北京市[[海淀区|海淀区]]政协副主席，[[北京市|北京市]]、[[吉林省|吉林省]]、[[云南省|云南省]]等省、直辖市政府以及[[成都市|成都市]]、[[武汉市|武汉市]]、[[汕头市|汕头市]]等市政府顾问或咨询委员，[[中国国际贸易促进会|中国国际贸易促进会]]特聘顾问，[[北京大学经济学院|北京大学经济学院]]教授、博士生导师，北京市场经济研究所所长，《经济界》杂志社社长兼总编<ref name=jy>{{Cite web|url = http://economy.caijing.com.cn/20171115/4361219.shtml|title = 萧灼基去世 曾欲向总理建言：不要轻言房地产泡沫|author = |date = 2017-11-15|publisher = 财经网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20171115142708/http://economy.caijing.com.cn/20171115/4361219.shtml|archive-date = 2017-11-15|dead-url = yes}}</ref>。

萧灼基的主要研究领域是[[马克思主义|马克思主义]]经济理论、[[社会主义|社会主义]]宏观经济理论、经济发展战略、金融与证券市场。他是[[中华人民共和国|中华人民共和国]]较早从事[[证券市场|证券市场]]理论研究的学者之一，尤其是21世纪初发表的关于证券市场的观点，对中国证券市场的建设及完善发挥了重要作用<ref name=jy/>。萧灼基曾获首届[[孙冶方|孙冶方]]优秀论文奖（1984年）、[[辽宁省|辽宁省]]优秀理论文章奖（1984年）、北京大学首届优秀教学奖（1985年）、北京大学首届优秀科研著作奖（1986年）、全国改革与发展优秀论文金三角奖（1988年）、北方十三省市区优秀图书一等奖（1986年）、北京市改革十年优秀论文奖（1989年）、全国省级刊物（政治理论）优秀文章一等奖（1993年）、首届[[陈岱孙|陈岱孙]]经济学著作奖（1996年）等<ref name=shishi/>。

1985年他写的《恩格斯传》一书出版。2008年5月5日，正值马克思诞辰190周年纪念日，萧灼基出版了倾注他50年心血的《马克思传》，并再版了1985年出版的《恩格斯传》。这两本书都是中国学者写的第一本学术研究性的《马克思传》、《恩格斯传》<ref name=jy/>。

萧灼基一生培养了大批学生。1988年[[李克强|李克强]]（后出任[[中共中央政治局常委|中共中央政治局常委]]、[[中华人民共和国国务院总理|国务院总理]]）以《农村工业化：结构转换中的选择》一文获经济学硕士学位，指导老师就是萧灼基<ref name=jy/>。

2017年11月14日，萧灼基在北京病逝，享年84岁<ref>{{Cite web |url = http://news.ifeng.com/a/20171115/53275935_0.shtml |title =经济学家、北大教授萧灼基逝世 曾指导李克强硕士论文  |author =  |date = 2017-11-15 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

==著作==
萧灼基发表论文数百篇，出版专著十多部。主要著作有<ref name=shishi/>：
*《萧灼基选集》
*《萧灼基文集》
*《中国经济热点问题研究》
*《中国宏观经济纵论》
*《社会主义再生产理论研究》
*《提高经济效益，实现宏伟战略目标》
*《中国经济建设与体制改革》
*《[[马克思|马克思]]传》
*《[[恩格斯|恩格斯]]传》
*《[[马克思|马克思]]青年时代》
*《马克思恩格斯著作中的历史人物》<ref name=shishi/>

他主编的著作主要有：
*《中国证券全书》
*《中国经济概论》
*《哲学社会科学名人名著辞典》
*《中国证券市场1991、1992》
*1994年至2007年的《中国经济分析与预测》<ref name=shishi/>
