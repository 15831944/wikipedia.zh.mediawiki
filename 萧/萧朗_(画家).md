'''萧朗'''（{{bd|1917年|11月|2010年|5月27日}}）名'''印鈢'''，字'''朗'''，别署'''萍香阁主人'''，[[北京|北京]]人，[[中华人民共和国|中华人民共和国]]画家、美术教育家<ref name=shishi>{{Cite web |url =  http://news.163.com/10/0528/13/67P9M52J000146BD.html|title =国画大师萧朗逝世 生前热心公益事业  |author =  |date = 2010-05-28 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref><ref name=jj>{{Cite web|url =http://www.xiaolangart.com/page.cms?id=2c9082204cb0c77f014cb0d737070007|title =画家介绍|author =|date =|publisher =萧朗艺术网|language =|accessdate =2017-11-21|quote =}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

==生平==
萧朗1930年代末拜画家[[王雪涛|王雪涛]]为师，随王雪涛研习书画十多年，同时获[[齐白石|齐白石]]、[[陈半丁|陈半丁]]等人指点。1940年起，在[[北京|北京]][[琉璃厂|琉璃厂]]的画店、纸店挂笔单，成为职业画家。1948年与同为王雪涛弟子的画友在北京办画展，齐白石特为此题写 “迟园门下画展”。1949年入[[华北大学|华北大学]]艺术系学习。自1940年代起，先后在[[北京师范大学|北京师范大学]]、[[河北师范大学|河北艺术师范学院]]、[[广西艺术学院|广西艺术学院]]任教<ref name=jj/>。
 
1964年，花鸟画新作《浴罢》参加全国美展。1986年与画友在天津办画展。1991年应邀在[[深圳|深圳]]办“百吉（鸡）画展”并讲学。1992年应邀为大陆记者首次赴台湾采访团创作《芙蕖鹡鸰图》赠给[[台湾|台湾]][[海峡交流基金会|海峡交流基金会]]。1996年在北京[[中国美术馆|中国美术馆]]、1997年在[[广州美术馆|广州美术馆]]、1998年在[[广西博物馆|广西博物馆]]和天津人民美术出版社展览馆、2000年在天津文物公司、2012年在[[天津美术馆|天津美术馆]]、2013年在北京[[荣宝斋|荣宝斋]]等处举办个人画展。2008年3月萧朗将自己的25幅作品无偿捐赠[[中国国家博物馆|中国国家博物馆]]。他是[[中国美术家协会|中国美术家协会]]会员，天津美术家协会名誉副主席，[[天津美术学院|天津美术学院]]教授<ref name=jj/>。萧朗以花鸟画见长，擅作花鸟画小品<ref>{{Cite web |url = http://www.zhongjilian.com/hd/2013/12-19/19242_0.html |title = 高清图：萧朗花鸟小品画作—与灵性期会 用诗情浣尘 |author =  |date = 2013-12-19 |publisher = 中央驻津新闻工作者协会 |language =  |accessdate =  |quote =  }}</ref>。
 
2010年5月27日，萧朗在[[天津|天津]]病逝，享年93岁<ref name=shishi/>。
