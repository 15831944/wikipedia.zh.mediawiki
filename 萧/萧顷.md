'''萧顷'''（{{bd|862年||930年|6月19日|catIdx=X}}<ref name=HFD41>[[:s:舊五代史/卷41|《旧五代史》卷四十一]]</ref><ref name=HFD58>[[:s:舊五代史/卷58|《旧五代史》卷五十八]]</ref><ref>930年6月19日是[[后唐明宗|后唐明宗]]为萧顷之死废朝的日子，不能完全确定是否就是萧顷去世之日。</ref>），[[表字|字]]'''子澄'''，[[中国|中国]][[唐朝|唐朝]]及其在[[五代十国|五代十国]]时期的后继政权[[后梁|后梁]]、[[后唐|后唐]][[官员|官员]]，后梁年间为[[宰相|宰相]]。

== 家世 ==
萧顷生于[[唐懿宗|唐懿宗]][[咸通|咸通]]三年（862年），[[长安|长安]][[万年县|万年县]]人。他出身显赫的[[兰陵萧氏|兰陵萧氏]]，家族有多名宰相和要员，其祖父[[萧仿_(唐朝)|萧仿]]即[[唐僖宗|唐僖宗]]朝宰相。父[[萧廪_(唐朝)|萧廪]]，[[京兆尹|京兆尹]]。<ref name=BT172>[[:s:舊唐書/卷172|《旧唐书》卷一百七十二]]</ref>萧顷聪明有悟性，善于作文。<ref name=HFD58/>

== 唐朝年间 ==
[[唐昭宗|唐昭宗]]年间，萧顷中[[进士|进士]]，步入仕途。<ref name=BT172/>历任度支巡官、[[太常博士|太常博士]]、[[右补阙|右补阙]]。任内，多有强势军阀们奏请在自己的节镇为祖先设立家庙，萧顷上表反对，拒绝了这些奏请。<ref name=HFD58/>

萧顷任吏部员外郎期间，一次管吏部的[[仆射|仆射]]前宰相[[张濬|张濬]]的政治盟友大军阀[[宣武军节度使|宣武军节度使]][[朱全忠|朱全忠]]派判官高劭来请求给自己的一个儿子荫补出身官。<ref>张濬仅在896年任实职仆射，此事似在此时发生。见[[:s:舊唐書/卷179|《旧唐书》卷一百七十九]]。</ref>相关部门认为没有先例，拒绝。张濬为了讨好朱全忠，坚持要同意，甚至亲自去指挥，官吏惶恐。萧顷判道：“仆射不聚集郎官，就亲自赴省上指挥公事，非南宫（官员工作场所）旧仪。”张濬闻之，惭悚道歉，萧顷因此知名，朱全忠也赞赏他。<ref name=HFD58/>

== 后梁年间 ==
后朱全忠夺取帝位建立后梁，史称太祖皇帝，萧顷成为梁官员，历任[[给事中|给事中]]、[[谏议大夫|谏议大夫]]、[[御史中丞|御史中丞]]。[[开平_(后梁)|开平]]三年（909年），[[郓州|郓州]]百姓刘郁于驾前状告金吾大将军[[石彦辞|石彦辞]]卖宅不肯交割，经御史台审理，未能追查到底，于是萧顷与侍御史[[卢庥|卢庥]]各罚两月俸。十一月，诏萧顷与[[太常卿|太常卿]][[李燕|李燕]]、[[中书舍人|中书舍人]][[张兗|张兗]]、[[户部侍郎|户部侍郎]][[崔沂|崔沂]]、[[大理卿|大理卿]][[王鄯|王鄯]]、刑部郎中[[崔诰|崔诰]]一同删定律令格式。四年（910年）十二月，宰相[[薛贻矩|薛贻矩]]奏称李燕等重刊定律令三十卷，式二十卷，格一十卷，目录一十三卷，律疏三十卷，凡五部一十帙，共一百三卷，敕令中书舍人[[李仁俭|李仁俭]]由阁门奉进，请求取名为《大梁新定格式律令》并颁下施行，获准。<ref>[[:s:舊五代史/卷147|《旧五代史》卷一百四十七]]</ref>萧顷请[[孔邈_(五代)|孔邈]]为御史。<ref>《[[册府元龟|册府元龟]]》</ref>后又任[[礼部侍郎|礼部侍郎]]、主管科举，都以能力闻名。<ref name=HFD58/>

[[后梁末帝|后梁末帝]][[朱瑱|朱瑱]][[贞明_(后梁)|贞明]]四年（918年）四月，萧顷以[[金紫光禄大夫|金紫光禄大夫]]、行尚书[[吏部侍郎|吏部侍郎]]、[[上柱国|上柱国]]、'''兰陵县开国男'''、食邑三百户受任为[[中书侍郎|中书侍郎]]、[[同平章事|同平章事]]，拜为宰相。<ref>[[:s:新五代史/卷3|《新五代史》卷三]]</ref><ref name=ZZTJ270>[[:s:資治通鑑/卷270|《资治通鉴》卷二百七十]]</ref>进封'''兰陵县开国伯'''，加食邑四百户。同月，朱瑱召刑部郎中、充史馆修撰[[窦专|窦专]]入翰林院，遣崇政使[[李振|李振]]问宰相：“窦专是宰臣萧顷女婿，令中书商量可否？”中书省奏：“宰相的亲戚不居清显官职是避嫌之道，虽是旧规定，若蒙特恩，也有近例，不碍事。”朱瑱于是任窦专为翰林学士。<ref>[[:s:舊五代史/卷9|《旧五代史》卷九]]</ref>

贞明六年（920年）四月，萧顷被任为集贤殿大学士、判户部事；同月，尚书右丞[[李琪_(五代)|李琪]]因与朱瑱近臣姐夫[[赵岩_(后梁)|赵岩]]、国舅[[张汉杰|张汉杰]]交好而得以拜相，<ref>[[:s:舊五代史/卷10|《旧五代史》卷十]]</ref>很快萧顷与李琪有了冲突。因李琪结连赵岩、张汉杰，萧顷所言多不被采纳。萧顷外表谨慎细密，秘密观察李琪过失。李琪性情粗疏，收受贿赂，经常把行贿的摄官任为正职。萧顷奏报皇帝。朱瑱大怒，要流放李琪到远方，李琪在赵岩、张汉杰介入下得免，然而被罢相。<ref name=HFD58/><ref>[[:s:新五代史/卷54|《新五代史》卷五十四]]</ref><ref name=ZZTJ271>[[:s:資治通鑑/卷271|《资治通鉴》卷二百七十一]]</ref>

== 后唐年间 ==
[[龙德|龙德]]三年（923年）十月，自称唐朝合法继承者的后梁大敌[[后唐|后唐]]皇帝[[后唐庄宗|后唐庄宗]]突袭攻克梁都城[[大梁|大梁]]。城破之际，朱瑱自杀，梁亡，领地被唐接管。庄宗认为萧顷及其同僚宰相[[郑珏|郑珏]]等大量梁高官祖上世代为唐朝官员，本人却不忠于唐，应被惩罚，流放了他们。萧顷被贬为[[登州|登州]]司户。<ref>[[:s:舊五代史/卷30|《旧五代史》卷三十]]</ref><ref>[[:s:新五代史/卷5|《新五代史》卷五]]</ref><ref name=ZZTJ272>[[:s:資治通鑑/卷272|《资治通鉴》卷二百七十二]]</ref>后升[[濮州|濮州]]司马。数年后，召回[[洛阳|洛阳]]，迁[[太子宾客|太子宾客]]（荣衔，当时没有[[皇太子|皇太子]]）。<ref name=HFD58/>

庄宗的养兄和继位者[[后唐明宗|后唐明宗]][[天成_(后唐)|天成]]元年（926年）六月，萧顷被任为礼部尚书。<ref name=HFD58/><ref>[[:s:舊五代史/卷36|《旧五代史》卷三十六]]</ref>当年，[[中书舍人|中书舍人]][[马缟|马缟]]奏称明宗系旁系承统，应该如汉、晋以来惯例另立亲庙，追尊亲生祖先。诏下尚书省，集百官定议。萧顷等商议后依马缟所议。<ref name=HFD142>[[:s:舊五代史/卷142|《旧五代史》卷一百四十二]]</ref><ref>[[:s:新五代史/卷55|《新五代史》卷五十五]]</ref><ref>[[:s:覆准马缟议奏|《覆准马缟议奏》]]</ref>

二年（927年）二月，萧顷任[[太常卿|太常卿]]，五月任吏部尚书。<ref>[[:s:舊五代史/卷38|《旧五代史》卷三十八]]</ref>八月，太常礼院奏：“庄宗神主祔庙，七室之内应该有所祧迁（移出主殿、改于偏殿祭祀）。”中书门下奏议，祧[[唐懿祖|唐懿祖]]于偏殿，百官共议，萧顷等奏请从中书省所奏，<ref>[[:s:议覆中书请祧懿祖奏|《议覆中书请祧懿祖奏》]]</ref>获准。<ref name=HFD142/>

三年（928年）闰八月，萧顷改任[[太子少保|太子少保]]（仍是荣衔，当时没有皇太子）。<ref>[[:s:舊五代史/卷39|《旧五代史》卷三十九]]</ref>后[[致仕|致仕]]。<ref name=HFD58/>[[长兴_(后唐)|长兴]]元年（930年）五月，以[[太子少傅|太子少傅]]卒，废朝。<ref name=HFD41/>

== 评价 ==
*《旧五代史》史臣曰：如数君子者，皆互有所长，亦近代之良相也。<ref name=HFD58/>
