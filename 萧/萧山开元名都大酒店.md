'''萧山开元名都大酒店'''是位于[[杭州市|杭州市]][[萧山区|萧山区]]市心中路818号的一座[[五星级酒店|五星级酒店]]，是[[开元旅业集团|开元旅业集团]]投资建造的第11家旗舰连锁酒店<ref name="简介">{{cite web |url=http://xs.map008.com/hotel/HotelDetails.aspx?HotelId=1 |language= |title=杭州开元名都大酒店 |publisher=萧山地图 |date= |accessdate=2012年10月 |archiveurl=https://web.archive.org/web/20120927025920/http://xs.map008.com/hotel/HotelDetails.aspx?HotelId=1 |archivedate=2012-09-27 |deadurl=yes }}</ref><ref name= "开业"/>，于2005年1月18日开业<ref name="开业">{{cite web |url=http://xsrb.xsnet.cn/xsdaily/gb/content/2005-01/19/content_335301.htm |language= |title=杭州第一高楼开元名都大酒店昨开业迎宾 |publisher=萧山日报 |date=2005-01-19 |accessdate= }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

附近有[[杭州地铁2号线|杭州地铁2号线]][[人民广场站_(杭州市)|人民广场站]]。

== 简介 ==
总投资7亿元，总建筑面积12万平米，有512套豪华客房。<ref name= "开业"/>共47层高218米，在2016年杭州柏悦酒店开业之前是[[浙江省|浙江省]]最高的酒店。位于萧山中央商务区，临近[[沪杭甬高速公路|沪杭甬高速公路]]，距[[萧山国际机场|萧山国际机场]]8公里。<ref name= "简介"/>

曾获殊荣有：第三届"中国酒店『金枕头』奖"----区域最佳酒店；中国首届商务人士最受欢迎的百家商务酒店。<ref name= "简介"/>
