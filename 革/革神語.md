《'''革神語〜天啟劍神〜'''》（{{lang-ja|アラタカンガタリ〜革神語〜}}），是一部[[日本|日本]][[奇幻|奇幻漫畫]]風格的[[少年漫畫|少年漫畫]]，作者為[[渡瀨悠宇|渡瀨悠宇]]，從[[小學館|小學館]]《[[週刊少年Sunday|週刊少年Sunday]]》2008年44號開始連載。

== 故事簡介 ==
故事由天和國秘女王交接儀式展開，由於秘女族惟有少女可以繼任，秘女族的少年諽被其祖母逼迫假扮女裝參與秘女王交接儀式。儀式中發生秘女王的部屬12神鞘的背叛，諽被指稱行凶者而被追殺，卻被神開之森吞噬…

另一方面，現代日本，就讀高中一年級的日之原革，為了遺忘傷痛的過去，原本也決心在高中重新開始，但是沒想到在國中發生的噩夢再次襲來，絕望中期望自己從世上消失，在暗巷中聽到某人呼喚他的名字…

兩個異世界的少年革與諽彼此交換身分，諽替換革的身分來到日本，而革替換成諽來到天和國…
革在天和國接受了秘女王的請託，並帶著劍神「創世」一同開啟與其他12神鞘的降服大戰…

== 廣播劇 ==
《'''ラジオ アラタカンガタリ 〜らじかん〜'''》是2013年3月1日到9月27日於[[大阪放送|大阪廣播]]每週五24:30進行放送的廣播節目。主持人為[[岡本信彦|岡本信彦]]（日之原革 飾）與[[小野友樹|小野友樹]]（神薙 飾）。

; 來賓
* 7・8回 - [[木村良平|木村良平]]（門脇將人 飾）
* 9・10回 - [[鈴木達央|鈴木達央]]（赫血 飾）
* 11・12回 - [[皆川純子|皆川純子]]（金手 飾）
* 13・14回 - [[松岡禎丞|松岡禎丞]]（諽 飾）
* 15・16回 - [[高垣彩陽|高垣彩陽]]（言葉 飾）
