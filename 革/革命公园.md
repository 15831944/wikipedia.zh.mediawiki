'''革命公园'''是位于[[中華人民共和國|中華人民共和國]][[陕西省|陕西省]][[西安市|西安市]]城内的一座公园，是西安市最古老的公园（1927年）之一。<ref name=wanbao>{{Cite web |url= http://epaper.xiancn.com/xarb/html/2011-05/20/content_35701.htm |title= 改造后革命公园将焕然一新 |author=  |date= 2011年5月20日 |publisher= 《西安晚报》 |language= zh-hans |accessdate= 2013年2月17日 |quote=  }}</ref>革命公园是为了纪念[[二虎守长安|二虎守长安]]事件中死难的军民而建立的。革命公园在[[文化大革命|文革]]期间也曾改名为'''东风公园'''，划归新城区综合办管理，1972年恢复原名革命公园。<ref>{{Cite web|url=http://www.snwh.gov.cn/bmfw/gygc/content/gygc_06/gygc_06.htm|title=革命公园|author=|date=|publisher=陕西文化信息网|language=zh-hans|accessdate=2013年9月15日|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20120922232627/http://www.snwh.gov.cn/bmfw/gygc/content/gygc_06/gygc_06.htm|archivedate=2012年9月22日}}</ref>

革命公园被列为第四批陕西省文物保护单位，属于近现代重要史迹。<ref>{{Cite web |url= http://www.shaanxi.gov.cn/0/103/5616.htm |title= 陕西省人民政府关于公布第四批陕西省文物保护单位的通知 陕政发〔2003〕38号 |author=  |date= 2003年9月24日 |publisher= 陕西省人民政府 |language= zh-hans |accessdate= 2013年2月17日 |quote=  }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==历史沿革==
为了大祭在1926年[[第三次西安战斗|西安之战]]中死难的军民，[[国民联军|国民联军]]于1926年12月中旬发起成立了[[陕西省革命大祭筹备委员会|陕西省革命大祭筹备委员会]]，委员会决定在城内东北角建立革命公园。<ref name=history>{{Cite web |url= http://epaper.xiancn.com/xarb/html/2011-05/20/content_35698.htm |title= 革命公园——不能忘却的记忆 |author=  |date= 2011年5月20日 |publisher= 《[[西安晚报|西安晚报]]》 |language= zh-hans |accessdate= 2013年2月17日 |quote=  }}</ref>

1927年3月12日，国民联军驻陕总部在革命公园内举行规模盛大的追悼大会和公葬仪式，西安城内10万多军民群众纷纷赶到现场，[[冯玉祥|冯玉祥]]、[[刘伯坚|刘伯坚]]、[[于右任|于右任]]均出席哀悼大祭。大祭的所有参加者，到北郊[[草滩|草滩]]，装满一袋袋黄土，再背负到大祭现场，倾倒黄土于坟上，终成两座大冢。<ref name=history/>两座冢共掩埋围城期间死难的无名尸骨3043具，东为男冢、西为女冢。<ref name=hsb>{{Cite web |url= http://hsb.hsw.cn/2011-03/08/content_8018587.htm |title= 革命公园不封园改造将竖12块碑诉说沧桑 |author= 周艳涛 |date= 2011年3月8日 |publisher= 《华商报》 |language= zh-hans |accessdate= 2013年2月17日 |quote=  }}</ref>于右任为此事纂写了“西安负土坟殉难人民碑辞”。<ref name=bei>{{Cite web |url= http://xian.qq.com/a/20110308/000001_1.htm |title= 革命公园迎84年最大改造 历史残碑成新景观 |author= 陈黎 |date= 2011年3月8日 |publisher= 《西安晚报》 |language= zh-hans |accessdate= 2013年2月17日 |quote=  }}</ref>

1949年后，政府对公园进行了整修，种植花木，建筑花坛、亭台等设施，开辟了莲池。<ref name=xinhua>{{Cite web|url=http://www.sn.xinhuanet.com/misc/2012-07/23/content_25508980_3.htm|title=革命公园——西安守城之战的记忆|author=|date=2012年7月23日|publisher=新华网|language=zh-hans|accessdate=2013年2月17日|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20160304124339/http://www.sn.xinhuanet.com/misc/2012-07/23/content_25508980_3.htm|archivedate=2016年3月4日}}</ref>

1956年，当时流动马戏团和耍猴的、玩把戏协会在革命公园西北角组建了西安动物园，一直持续到1977年。<ref>{{Cite web |url= http://news.hsw.cn/system/2015/1101/320662.shtml?rand=5JZjqT7K |title= 莲湖公园明年百岁却“忘记”自己生日 历史待发掘 |author= 谢伟 |date= 2015-11-01 |publisher= 华商网 |language= zh-hans |accessdate= 2015年11月1日 |quote=  }}</ref>

==院内建筑==

公园内亦有三座雕塑，分别为：杨虎城将军铜像、[[刘志丹|刘志丹]]将军汉白玉石雕像以及 [[谢子长|谢子长]]将军铜像。<ref name=bei/>

==现状==
革命公园曾于2011年进行改造，主要是对园区内所有的路面、地下管线、革命亭广场地面等进行改造翻新。<ref name=hsb/>

经过数年的发展，革命公园已经成为西安市内较有影响的一个相亲平台。<ref>{{Cite web |url= http://news.hsw.cn/system/2012/03/12/051268692.shtml |title= 革命公园变“单身超市” 10元可买30名征婚者信息 |author=  |date= 2012年3月12日 |publisher= 华商网 |language= zh-hans |accessdate= 2013年2月17日 |quote=  }}</ref>2014年5月6日，革命公园管理方面表示将取缔这个“相亲角”。<ref>{{Cite web |url= http://travel.cnwest.com/content/2014-05/08/content_11108641.htm |title= 高清：革命公园将取缔“相亲角” 已存在近十年 |author=  |date= 2014-05-08 |publisher= 西部网 |language= zh-hans |accessdate= 2014年5月9日 |quote=  }}</ref>
