'''布倫特·海登'''（{{lang|en|Brent Matthew Hayden}}，{{bd|1983年|10月21日|||catIdx=Hayden, Brent}}）是一位[[加拿大|加拿大]]游泳選手。在[[2012年倫敦奧運會|2012年倫敦奧運會]]上，他得到[[2012年夏季奧林匹克運動會男子100米自由泳比賽|男子100米自由泳]]的銅牌<ref name=perseverance>{{cite news|title=Perseverance, sacrifice define Canada's Brent Hayden |author=Tony Care |url=http://www.cbc.ca/sports/opinion/olympics2012/2012/08/perseverance-and-sacrifice-define-canadas-brent-hayden.html |publisher=''[[CBC_Sports|CBC Sports]]'' |date=August 1, 2012}}</ref>。

[[2004年夏季奧林匹克運動會|2004年]]，海登隨加拿大代表團首度參與奧運會。然而，他在[[雅典|雅典]]街頭遇上示威，更被防暴警察以警棍打擊，因此無法參與男子100米自由泳賽事<ref name=perseverance/><ref name=starprofile>{{cite news|title=London 2012 profile |url=http://www.thestar.com/sports/london2012/swimming/athlete/418492--brent-hayden |publisher=''Toronto Star'' |accessdate=August 2, 2012}}</ref>。他在另外四項賽事作賽，最佳成績為4×200米自由泳接力中得第5名<ref name=COCprofile>{{cite web |title=Brent Hayden profile |url=http://olympic.ca/team-canada/brent-hayden |publisher=''Canadian Olympic Committee |accessdate=August 2, 2012}}</ref>，個人最佳成績則為200米自由泳賽事得第13名<ref name=COCprofile/>。

在[[2006年英聯邦運動會|2006年墨爾本英聯邦運動會]]中，海登在他本來不打算參與的50米自由泳項目中奪得銀牌<ref name=COCprofile/>；他在同屆英聯邦運動會也奪得三個銅牌。在[[2007年世界游泳錦標賽|2007年世界游泳錦標賽]]的男子100米自由泳項目中，他與意大利選手菲利頗·馬尼伊尼（{{lang|it|Filippo Magnini}}）同時沖綫，兩人同獲金牌。這是加拿大選手自1986年來首次在世界游泳錦標賽中奪冠，而他以48.43秒沖綫的成績亦刷新了自己創下的加拿大記錄；他在同屆賽事亦在4×200米自由泳接力中得銅牌<ref name=COCprofile/>。

海登於[[卑詩省|卑詩省]][[米遜_(卑詩省)|米遜]]出生，5歲時就開始游泳，但年幼時需重修游泳班<ref name=COCprofile/>。他的胸部有個超人紋身，因此被友人暱稱為「{{lang|en|Soup}}」<ref name=starprofile/>。
