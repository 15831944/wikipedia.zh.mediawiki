'''布莱恩·科比尔卡'''（{{lang-en|Brian Kobilka}}，{{bd|1955年|5月30日}}），美国[[斯坦福大学|斯坦福大学]]医学院分子和细胞生理学教授和博士。他也是ConfometRx，一家专注于G-蛋白偶联受体的生物技术公司的共同创办人。在2011年他成为[[美国国家科学院|美国国家科学院]]院士。2012年因在[[G蛋白偶联受体|G蛋白偶联受体]]方面的成就获得[[诺贝尔化学奖|诺贝尔化学奖]]<ref>http://www.ch.ntu.edu.tw/nobel/2012.html</ref>。

==早年==
科比尔卡生于明尼苏达州中部的小城[[利特尔福尔斯_(明尼苏达州)|利特尔福尔斯]]。他的祖父和父亲均为面包师，母亲做蛋糕。后来他进入[[明尼苏达大学|明尼苏达大学]]德卢斯分校学习生物学和化学，在那里他遇见了未来的妻子、马来西亚华人田东山。他和田东山一起在大学发育生物学实验室工作，曾经自制过组织培养皿。后来在[[耶鲁大学|耶鲁大学]]读研究生，开始对生死关头作用于G蛋白偶联受体的药物产生兴趣，尤其是肾上腺素和去甲肾上腺素受体，它们可以使呼吸道通畅，并提高心率。<ref name=nature>{{cite web |url=http://www.nature.com/news/2011/110824/pdf/476387a.pdf |title=Brian Kobilka|accessdate=2012-10-10}}</ref>

后进入圣路易斯[[华盛顿大学医学院|华盛顿大学医学院]]巴恩斯医院内科，在[[杜克大学|杜克大学]]的罗伯特·莱夫科维茨指导下作博士后研究。<ref>{{cite web |url=http://www.nature.com/nrd/journal/v3/n7/box/nrd1458_BX14.html |title=The state of GPCR research in 2004 : Nature Reviews Drug Discovery |accessdate=2008-10-16}}</ref>1987年至2003年任[[霍华德·休斯医学研究所|霍华德·休斯医学研究所]]研究员。<ref>{{cite web|title=Brian K. Kobilka, M.D.|url=http://www.hhmi.com/research/investigators/kobilka_bio.html|publisher=HHMI|accessdate=24 August 2012}}</ref>

==研究==
科比尔卡最出名的是贡献是对[[G蛋白偶联受体|G蛋白偶联受体]]（GPCR）的结构和活性的研究，特别是，科比尔卡实验室的工作确定了β2-肾上腺素受体的分子结构。科比尔卡加入时，实验室才刚刚开始思考如何克隆基因的β2-肾上腺素能受体（β2AR），并确定它的基因序列。但该受体如此之小，该小组只能从足够的蛋白质中搜集一些小块，利用其可能的基因序列作识别。科比尔卡决定建立一个哺乳动物的基因组序列库，并利用已有的小段序列筛选它们。这使得全序列的克隆成为可能。<ref ==荣誉==
1994年，科比尔卡获得美国药理学和实验疗法学会阿贝尔药理学奖。<ref>{{cite web |url=http://www.aspet.org/PUBLIC/awards/abel_award.html |title=ASPET J.J. Abel Award |accessdate=2008-10-16 |deadurl=yes |archiveurl=https://www.webcitation.org/6CstmUDbZ?url=http://www.aspet.org/awards/ |archivedate=2012-12-13 }}</ref>他在G蛋白偶联受体结构方面的工作被《科学》杂志评为2007年“年度突破奖”第二位。<ref>{{cite web |url=http://news-service.stanford.edu/news/2008/january9/med-kobilka-010908.html |title=Kobilka's work recognized in magazine award |accessdate=2008-10-16}}</ref>

==个人生活==
科比尔卡是[[波兰裔美国人|波兰裔美国人]]，他的祖父和父亲是面包师。他与妻子田東山育有两个孩子。<ref>{{cite journal|last=Buchen|first=Lizzie|title=Cell signalling: It's all about the structure|journal=Nature|date=24 August 2011|volume=476|issue=7361|pages=387–390|doi=10.1038/476387a}}</ref>
