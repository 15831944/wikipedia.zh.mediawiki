'''布韦岛'''（{{lang-no|Bouvetøya}}，又譯為-{zh-hans:'''布威岛'''（台湾）;zh-hk:'''布韋島'''（中國大陸）;zh-tw:'''鮑威特島'''（香港）;}-或-{zh-hans:'''鲍威特岛'''（香港）;zh-hk:'''布威島'''（台灣）;zh-tw:'''布韋島'''（中國大陸）;}-），是屬於[[挪威|挪威]][[南极领地|南极领地]]的一座[[岛屿|岛屿]]。

== 地理 ==
布韦岛位于{{coord|54|26|S|3|24|E}}，剛巧就在環繞南極洲的西風帶以外{{r|bbc}}。面积为58.5平方公里（22.6平方英哩）。全岛93%的面积被[[冰川|冰川]]覆盖；而島上平均每年的降雪量接近一公尺（2001年降的雪在15米深）{{r|bbc}}。沿岸有很多冰山。岛上没有港口或港湾，仅在岛外近岸处有临时锚地，通过小船与岸上聯絡。

布韦岛全岛最高处为奥拉夫峰（780米高）。沿海为冰崖和黑色火山岩。岛的东部在1955年到1958年露出了原生火山岩，有鸟类栖居。

布韦岛被认为是世界上距任何一个大陆最远的岛屿之一{{r|bbc}}，其他类似岛屿还有[[复活节岛|复活节岛]]和[[特里斯坦-达库尼亚岛|特-{里}-斯坦-达库尼亚岛]]。离布韦岛最近的大陆是[[南极洲|南极洲]]的[[毛德皇后地|毛德皇后地]]，有1600公里远。该岛[[国际域名缩写|国际域名缩写]]代码为.bv。

== 历史 ==

布韦岛于1739年1月1日由[[法国|法国]][[航海家|航海家]][[布韦|让-巴蒂斯特·夏尔·布韦]]（Jean-Baptiste Charles Bouvet de Lozier）发现。但是布韦没有对该岛进行考察，因此无法判定其究竟是一个岛屿还是南极大陆的一部分。

直至1808年，布韦岛才第二次被人发现，发现者是[[恩德比公司|恩德比公司]][[捕鲸船|捕鲸船]]“天鹅”号的船长林塞。1822年12月，捕海豹船“黄蜂”号的船长本杰明·莫雷尔第一次登上该岛，捕猎[[海豹|海豹]]。1825年12月10日，恩德比公司船东诺-{里}-斯船长将该岛命名为“利物浦岛”，并宣布其为英国屬地。1898年，[[德意志帝国|德国]]探险家库恩再次抵达该岛，但未上岛考察。

登上布韦岛的第一批科学考察者是1927年的一个挪威考察队。他们在岛上居住了一个月，并将其命名为布韦岛。英国稍后放弃了对该岛的主权要求。

[[第二次世界大战|第二次世界大战]]前及战争期间，布韦岛附近海域成为挪威捕鲸船作业基地，捕获了大量[[南露脊鲸|南露脊鲸]]和[[抹香鲸|抹香鲸]]。1941年1月中旬，[[德国海军|德国海军]][[辅助巡洋舰|辅助巡洋舰]]“[[企鹅|企鹅]]”号（HK-33）在布韦岛海域捕获了11艘挪威捕鲸船和3艘鲸鱼加工船。

1964年，在布韦岛海岸发现了1艘被遗弃的[[救生艇|救生艇]]，但乘客不知所蹤。

1971年，布韦岛被挪威政府宣布为自然保护区，禁止人员常年居住。1977年，挪威在岛上设立了一座自动气象站。

1979年9月22日，美國人造衛星Vela Hotel在布韦岛和[[爱德华王子群岛|爱德华王子群岛]]之間的海域偵測到一股強光。這道強光後來曾被推斷為[[核試|核試]]產生的爆炸或由[[隕石|隕石]]在空中爆炸所致，但相關的謎團至今仍未完全解開（[[船帆座事件|船帆座事件]]）。
