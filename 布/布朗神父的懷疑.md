《'''布朗神父的懷疑'''》（{{lang-en|''The Incredulity of Father Brown''}}）是[[英國|英國]]作家[[G·K·卻斯特頓|G·K·卻斯特頓]]的[[短篇小說|短篇小說]]集，收錄了八篇以[[布朗神父|布朗神父]]為主角的探案故事。於1926年出版<ref>{{Cite book|title=謀殺的魅影：世界推理小說簡史|last=褚盟|publisher=古吳軒出版社|year=2011年6月|isbn=978-7-80733-644-0|location=蘇州}}</ref><ref>{{Cite news|url=http://cathnews.com/archives/cn-perspectives-archive/18973-the-mysterious-appeal-of-father-brown|title=The mysterious appeal of Father Brown|author=|last=User|first=Fix|date=2014-09-04|newspaper=CathNews|accessdate=2018-11-20|language=en-gb}}</ref>。

== 內容 ==
該書包含以下八個短篇故事：

# 布朗神父復活
# 來自天空的箭
# 獵犬的預言
# 新月市的奇蹟
# 金十字架的詛咒
# 有翅膀的匕首
# 達爾納威家的厄運
# 吉迪恩·懷斯的鬼魂
