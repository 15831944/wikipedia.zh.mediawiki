'''布氏豹'''（[[学名|学名]]：''Panthera blytheae''）是5.95至4.1万年前（[[中新世|中新世]]晚期至[[上新世|上新世]]）曾经生存在[[青藏高原|青藏高原]]的[[猫科|猫科]][[豹属|豹属]]动物。<ref name = bsb01>{{Cite web | title = 西藏札达盆地发现最古老的豹类化石 | url = http://www.ivpp.cas.cn/xwdt/kydt/201311/t20131113_3977176.html | publisher = 中国科学院古脊椎动物与古人类研究所 | date = 2013-11-13 | language = zh-hans | }}</ref><ref name = bbc>{{Cite web | title = Oldest big cat fossil found in Tibet | url = http://www.bbc.co.uk/news/science-environment-24913291 | publisher = BBC news | author = James Morgan | date = 2013-11-13 | language = en}}</ref>

布氏豹的[[化石|化石]]于2010年8月7日在[[西藏自治区|西藏自治区]][[札达盆地|札达盆地]]被发现，是迄今为止[[人类|人类]]所发现最早的豹属动物化石。<ref name = bsb01/>
这一发现为豹属这一支大型食肉动物起源于亚洲的设想提供了有力的证据。

== 特征 ==
通过对布氏豹头盖骨的研究证明其特点是额宽，面短。虽然被证明与[[雪豹|雪豹]]是姐妹种，其体型大小更接近较小的[[云豹|云豹]]。<ref name = bbc/><ref name = canivora>{{Cite web | title = Panthera blytheae | url = http://carnivoraforum.com/topic/9985604/1/ | author = carnivoraforum | date = 2013-11-16 | language = en | access-date = 2014-02-14 | archive-url = https://web.archive.org/web/20140222191313/http://carnivoraforum.com/topic/9985604/1/ | archive-date = 2014-02-22 | dead-url = yes }}</ref>
主要捕食[[瞪羚|瞪羚]]，[[鼠兔|鼠兔]]和[[岩羊|岩羊]]等动物<ref>{{Cite web | title = Oldest big cat fossil found in Tibet | url = http://www.latimes.com/science/sciencenow/la-sci-sn-big-cat-fossil-20131112,0,4121925.story?track=rss#axzz2kVZNJ8QH | author = Geoffrey Mohan | date = 2013-11-12 | language = en}}</ref>
