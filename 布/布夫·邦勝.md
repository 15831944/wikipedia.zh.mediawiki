'''布夫·邦瑟'''（'''Boof Bonser'''，{{bd|1981年|10月14日|||Bonser, Boof}}）出生於[[美國|美國]][[佛羅里達州|佛羅里達州]][[聖彼德斯堡_(佛羅里達州)|聖彼德斯堡]]，前[[美國職棒大聯盟|美國職棒大聯盟]]投手，曾為[[統一7-ELEVEn獅|統一7-ELEVEn獅]]的投手。他曾用名'''約翰·保羅·邦瑟'''（'''John Paul Bonser'''），2001賽季結束後改成現在的名字。



== 高中時期 ==
邦瑟畢業於[[聖彼德斯堡_(佛羅里達州)|聖彼德斯堡]]的吉布斯高中，在高中四年的時間裡，作為投手的邦瑟一共投出了24勝9敗、防禦率1.99的成績。高中的最後一年邦瑟投出了7勝3敗、防禦率1.88的成績，在打擊方面，他擊出了11支[[全壘打|全壘打]]，打擊率為5成23。他被選為2000年[[皮尼拉斯縣_(佛羅里達州)|皮尼拉斯縣]]年度最佳高中球員，並參加了當年佛羅里達州全明星賽<ref>{{cite web | title = Player File: Boof Bonser Biography | publisher = [[MLB.com|MLB.com]] | url = http://minnesota.twins.mlb.com/team/player_career.jsp?player_id=425818 | accessdate = 2007-04-15}}</ref>。

== 小聯盟時期 ==
邦瑟在[[2000年美國職棒大聯盟選秀|2000年美國職棒大聯盟選秀]]上被[[舊金山巨人|舊金山巨人]]在第一輪21順位選中。隨後他在短1A的[[薩勒姆凱澤火山|薩勒姆凱澤火山]]開始自己的職業生涯。2001年他在1A的[[哈吉斯城太陽|哈吉斯城太陽]]表現出色，拿到[[南大西洋聯盟|南大西洋聯盟]]最多的16勝，三振數（178次）排名聯盟第二，並拿到了聯盟年度最佳投手的榮譽和入選了全明星賽。這個賽季後他將名字從約翰·保羅改成了他兒時的暱稱布夫。

隨後的兩年邦瑟在巨人隊小聯盟中穩步上升，2003年季末升到了3A的[[佛列斯諾灰熊|佛列斯諾灰熊]]。同年的11月14日，他和[[喬·內森|喬·內森]]、[[弗朗西斯科·利瑞安諾|弗朗西斯科·利瑞安諾]]一起被送到[[明尼蘇達雙城|明尼蘇達雙城]]，交換[[A·J·皮爾津斯基|A·J·皮爾津斯基]]（[[:en:A._J._Pierzynski|A. J. Pierzynski]]）。2004年邦瑟從雙城2A的[[新不列巔石貓|新不列巔石貓]]的起步，隨後升上3A的[[羅徹斯特紅翼|羅徹斯特紅翼]]。

2001年之後邦瑟再也沒有投出像當年出色的成績，作為頂級新秀的身份也有所褪色。直到2006年在紅翼隊開季投出防禦率2.01的成績，他才升上大聯盟。



== 大聯盟生涯 ==
=== 明尼蘇達雙城 ===
[[File:Boof_Bonser_2_crop.jpg|thumb]]隊時期。]]
{{MLBy|2006年}}5月17日，邦瑟首次被叫上大聯盟，頂替發揮不佳的[[凱爾·洛什|凱爾·洛什]]（[[:en:Kyle_Lohse|Kyle Lohse]]）的先發位置<ref>{{cite web |url=http://minnesota.twins.mlb.com/news/article.jsp?ymd=20060517&content_id=1458242&vkey=news_min&fext=.jsp&c_id=min |title=Twins demote Lohse, call up Bonser |accessdate=2009-09-22 |last=Hill |first=Justice B. |date=2006-05-17 |publisher=[[MLB.com|MLB.com]] |deadurl=yes |archiveurl=https://web.archive.org/web/20110617004446/http://minnesota.twins.mlb.com/news/article.jsp?ymd=20060517&content_id=1458242&vkey=news_min&fext=.jsp&c_id=min |archivedate=2011-06-17 }}</ref>。[[5月21|5月21]]在對[[密爾瓦基釀酒人|密爾瓦基釀酒人]]的比賽中，邦瑟首次登板亮相，投出六局送出8次三振，僅掉1分。6天之後他拿到了他在大聯盟的第一場勝投，他主投5局失4分，幫助球隊以9比4戰勝[[西雅圖水手|西雅圖水手]]。

在7月4日對[[堪薩斯市皇家|堪薩斯市皇家]]糟糕的表現之後，邦瑟被降回3A。一個月之後他又被升上大聯盟，頂替受傷的利瑞安諾。回到大聯盟後，雖然他連續兩場表現不佳，均吞下敗投，但雙城隊總教練[[羅恩·加登海爾|羅恩·加登海爾]]（[[:en:Ron_Gardenhire|Ron Gardenhire]]）還是決定將他留在先發輪值表中。他也沒有辜負教練的期望，在9月份投出4勝1負的成績，並拿到[[美國職棒大聯盟單月最佳新人|單月最佳新人]]的稱號。由於雙城投手群的受傷，他在季後賽中擔當2號投手的角色，在對[[奧克蘭運動家|奧克蘭運動家]]的比賽中主投6局失2分，無關勝敗，但雙城最終還是負於對手。

{{MLBy|2007年}}是邦瑟在大聯盟第一個完整的賽季。他開季擔任球隊的2號先發投手，僅次於[[尤漢·山塔納|尤漢·山塔納]]。他開季連續投出多場[[優質先發|優質先發]]的比賽，但在[[亡兵紀念日|亡兵紀念日]]後僅投出2勝9敗的成績。整個賽季他拿到8勝12敗、防禦率5.10的成績，並有136次的三振。這個賽季他的續航力受到球團的質疑，因此球團要求他減肥。邦瑟按照要求改成了更加健康的飲食結構，並加強鍛煉<ref>{{cite web | url = http://minnesota.twins.mlb.com/news/article.jsp?ymd=20080126&content_id=2356800&vkey=news_min&fext=.jsp&c_id=min | title = The Official Site of The Minnesota Twins: News: Notes: For Bonser, less is more | date = 2008-01-26 | author = Kelly Thesier | publisher = [[MLB.com|MLB.com]] | accessdate = 2010-01-22 | deadurl = yes | archiveurl = https://web.archive.org/web/20090903102228/http://minnesota.twins.mlb.com/news/article.jsp?ymd=20080126&content_id=2356800&vkey=news_min&fext=.jsp&c_id=min | archivedate = 2009-09-03 }}</ref>。{{MLBy|2008年}}賽季開始前邦瑟減了30磅<ref>{{cite web | url = http://bleacherreport.com/articles/15800-Minnesota_Twins-MinnesotaTwins_Notes_Boof_Bonser_Stumbles_in_First_Lightweight_Start | title = MinnesotaTwins Notes: Boof Bonser Stumbles in First Lightweight Start | author =  Marty Andrade}}</ref>。但減肥並沒有幫助他提高成績，前兩個月僅投出2勝6敗、防禦率6.16的成績。隨後他被調入牛棚，[[斯科特·貝克|斯科特·貝克]]（[[:en:Scott_Baker_(right-handed_pitcher)|Scott Baker]]）取代了他的先發位置<ref>{{cite web |url=http://sports.espn.go.com/mlb/news/story?id=3423130 |title=After catching line drive in face, Blackburn still slotted for next start |accessdate=2009-09-22|author=[[美聯社|美聯社]] |date=2008-06-02 |publisher=[[ESPN|ESPN.com]]}}</ref>。作為中繼投手，邦瑟也僅投出了防禦率5.88的成績。

{{MLBy|2009年}}2月25日邦瑟接受手術，修復肩肘，他也因此錯過了整個2009賽季<ref>{{cite web | url = http://mlb.mlb.com/news/article.jsp?ymd=20090225&content_id=3881988&vkey=news_mlb&fext=.jsp&c_id=mlb | title = Bonser likely out for the season | date = 2009-02-25 | accessdate = 2010-01-22 | publisher = [[MLB.com|MLB.com]] | author = Kelly Thesier}}</ref>。12月8日邦瑟被球隊[[指定派遣|DFA]]<ref>{{cite web | url = http://minnesota.twins.mlb.com/news/article.jsp?ymd=20091208&content_id=7775362&vkey=news_min&fext=.jsp&c_id=min | title = Twins designate Bonser for assignment | date = 2009-12-08 | accessdate = 2010-01-22 | publisher = [[MLB.com|MLB.com]] | author = Kelly Thesier | deadurl = yes | archiveurl = https://web.archive.org/web/20110617005103/http://minnesota.twins.mlb.com/news/article.jsp?ymd=20091208&content_id=7775362&vkey=news_min&fext=.jsp&c_id=min | archivedate = 2011-06-17 }}</ref>。

=== 波士頓紅襪 ===
{{MLBy|2009年}}12月10日，邦瑟被交易到[[波士頓紅襪|波士頓紅襪]]<ref>{{cite web | url = http://www.mlbtraderumors.com/2009/12/red-sox-acquire-boof-bonser.html | title = Red Sox Acquire Boof Bonser | author = Tim Dierkes}}</ref>。{{MLBy|2010年}}開季階段他一直待在傷兵名單中，直到6月9日才登上大聯盟。但他僅代表紅襪隊投了兩場比賽，在兩局的投球中失6分，隨後便被球隊DFA。
