'''布蘭佛德菊石'''（[[學名|學名]]：{{lang|la|'''''Blanfordiceras'''''}}），又名'''布蘭弗菊石'''，是生存在[[侏羅紀|侏羅紀]]末[[提通期|提通期]]至[[白堊紀|白堊紀]]初[[貝里亞期|貝里亞期]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]，和[[貝利亞斯菊石|貝利亞斯菊石]]有許多相似之處<ref name="Berriasella">{{Cite web |url = http://liusir.pixnet.net/blog/post/248636438-berriasella-%E8%B2%9D%E9%87%8C%E9%98%BF%E6%96%AF%E8%8F%8A%E7%9F%B3-%28%E6%B3%95%E5%9C%8B%29 |title = Berriasella 貝里阿斯菊石 (法國) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-06-06 }}</ref>，但其螺紋部份更[[圓|圓]]。其[[化石|化石]]被發現於[[南亞|南亞]]、[[南極洲|南極洲]]和[[中國|中國]]等地。<ref name="Fossilworks" />

== 特徵 ==
布蘭佛德菊石的側面外觀看似旋菊石類，都具有分岔的[[Y|Y]]形肋條。最大不同處在於其腹側呈「溝狀」。<ref name="Blanfordiceras">{{Cite web |url = http://liusir.pixnet.net/blog/post/248637674-blanfordiceras-%E5%B8%83%E8%98%AD%E4%BD%9B%E5%BE%B7%E8%8F%8A%E7%9F%B3-%28%E5%B0%BC%E6%B3%8A%E7%88%BE%29 |title = Blanfordiceras 布蘭佛德菊石 (尼泊爾) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-06-06 }}</ref>
