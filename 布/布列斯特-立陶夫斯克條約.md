《'''布列斯特-立陶夫斯克條約'''》（{{lang-ru|''Брестский мир''}}；''Friedensvertrag von Brest-Litowsk''，常被简称为布列斯特条约或布列斯特和约），是[[第一次世界大战|第一次世界大战]]期间，[[苏联|苏俄]]政府与[[同盟國_(第一次世界大戰)|同盟国]]（包括[[德意志帝国|德意志帝国]]、[[奥匈帝国|奥匈帝国]]、[[保加利亚|保加利亚]]、[[奥斯曼土耳其帝国|奥斯曼土耳其帝国]]）在[[布列斯特|布列斯特-立陶夫斯克]]签订的和约。

== 条约背景 ==

1917年，俄国[[十月革命|十月革命]]胜利之后，由于原来俄国属于[[协约国|协约国]]一方而与[[同盟國_(第一次世界大戰)|同盟国]]处于交战状态，为了退出战争，新成立不久的苏俄政府在提出的和平建议被协约国拒绝后，便与同盟国的德国进行和平谈判。

1917年12月3日，谈判开始，德国提出了把[[波兰|波兰]]、[[立陶宛|立陶宛]]、[[爱沙尼亚|爱沙尼亚]]的局部和[[拉脱维亚|拉脱维亚]]、[[白俄罗斯|白俄罗斯]]的全部割让给德国并赔款30亿卢布的苛刻条件，这引起了布尔什维克党内严重的分歧。[[列宁|列宁]]主张接受德国的条件，签订和约，为新生政权争得喘息机会，[[季诺维也夫|季诺维也夫]]、[[索柯里尼柯夫|索柯里尼柯夫]]、[[斯大林|斯大林]]、[[阿尔乔姆|阿尔乔姆]]、[[斯塔索娃|斯塔索娃]]、[[斯维尔德洛夫|斯维尔德洛夫]]等六名中央委员支持列宁；[[布哈林|布哈林]]为代表的“左派共产主义者”反对签订和约，主张对帝国主义继续世界大战，中央委员[[布勃诺夫|布勃诺夫]]、[[乌里茨基|乌里茨基]]、[[洛莫夫|洛莫夫]]支持布哈林；[[托洛茨基|托洛斯基]]则主张停战，復员军队，但不与德国签约（即“不战不和”），中央委员[[克列斯廷斯基|克列斯廷斯基]]、[[捷尔任斯基|捷尔任斯基]]、[[越飞|越飞]]支持托洛斯基。

1918年1月2日，苏俄政府召开中央和地方负责人会议。60人出席的会议上，赞成[[布哈林|布哈林]]主张的32人，赞成[[托洛茨基|托洛茨基]]主张的16人，赞成[[列宁|列宁]]主张的仅15人。最终，[[列宁|列宁]]的主张因处于少数而未能被通过。

1918年1月24日，苏俄政府召开中央会议，重新表决签约的问题。[[托洛茨基|托洛斯基]]的主张以9票对7票的多数通过，列宁的主张仍然没有被多数所接受。

1月30日，布列斯特谈判恢复。作为外交人民委员（即外交部长）、谈判代表团团长，[[托洛茨基|托洛斯基]]临行前与[[列宁|列宁]]约定：如果德国下了最后通牒就让步签约。结果德国果然向苏俄下了最后通牒，[[托洛茨基|托洛斯基]]向[[列宁|列宁]]电报询问对策，[[列宁|列宁]]立即覆电坚持：接受德国条件，立即签约。但是[[托洛茨基|托洛斯基]]没有接受[[列宁|列宁]]的建议，而是发表了拒绝签约的声明，率团离开布列斯特。结果，德国开始对苏俄大举进攻。

2月18日，在十分紧急的情况下，苏俄中央委员会召开了紧急会议。会上，列宁的主张又被以6：7的票数否决。随后，中央又连夜开会，经过激烈的争论，[[托洛茨基|托洛斯基]]转而支持列宁，会议结果以7票赞成、5票反对、1票弃权的结果通过了[[列宁|列宁]]的提案。苏俄政府连夜通知德国方面，同意签约。

但是，德国在得到通知后没有停止进攻，并于2月23日提出更加苛刻的条件。苏俄党中央又召开有15名委员参加的紧急会议。会议中，大多数委员表示不赞成[[列宁|列宁]]的主张。[[列宁|列宁]]只得提出，如果这种空谈继续下去，他就要辞职，退出政府和中央委员会。[[布哈林|布哈林]]对此毫不在乎，[[斯大林|史達林]]也未发生动摇。[[托洛茨基|托洛斯基]]不同意[[列宁|列宁]]的意见，但为了防止[[列宁|列宁]]辞职和党的分裂，他的态度发生变化。在他的影响下，出现了4票弃权。结果[[列宁|列宁]]的主张以7票赞成、4票弃权、4票反对获得通过。同日在[[蘇維埃中央執行委員會|蘇維埃中央執行委員會]]上再以116：85（26票棄權）之多數通過。[[托洛茨基|托洛斯基]]被解除了外交人民委员的职务，改由[[格奥尔基·契切林|格奥尔基·契切林]]接任。

2月24日，苏俄政府重新派出了谈判代表团与德国进行谈判。

== 条约内容 ==
[[File:Map_Treaty_of_Brest-Litovsk-en.jpg|thumb]]

3月3日，布列斯特和约正式签订。条约包括下列文件：

# 苏俄和德国、奥匈帝国、保加利亚、鄂圖曼土耳其的和约及附件
# 苏俄和德国的法律政治条约
# 苏俄和奥匈帝国的法律政治条约
# 苏俄和保加利亚的法律政治条约
# 苏俄和鄂圖曼土耳其的法律政治条约
# 苏俄和德国关于若干政治问题的附加条约#苏俄和德国的财政协定
# 苏俄和德国的私法协定

上述条约（一）至（五）项于1918年3月3日在布列斯特-立托夫斯克签订，3月29日在柏林互换批准书之后生效。（六）至（八）项于1918年8月27日在柏林签订，9月6日在柏林互换批准书生效。

条约主要內容如下：
# 苏俄终止与德国、奥匈帝国、保加利亚和鄂圖曼土耳其的战争状态；
# 苏俄军队全面复员，海军军舰驶回海港并解除武装；
# 苏俄承認[[芬蘭|芬蘭]]、[[烏克蘭|烏克蘭]]、[[白俄罗斯|白俄罗斯]]之獨立，并有义务同[[乌克兰人民共和国|乌克兰人民共和国]]立即缔结和约；
# 苏俄將[[爱沙尼亚|爱沙尼亚]]、[[拉脱维亚|拉脱维亚]]、[[立陶宛|立陶宛]]、[[库尔兰|库尔兰]]（今拉脱维亚西部）、[[俄属波兰|俄属波兰]]等地割與德意志帝國與奧匈帝國，上述各地今後之政治狀態，僅德奧二國有權決定；
# 苏俄将沙皇俄国于[[俄土战争_(1877年-1878年)|1877-1878年俄土战争]]中占领的[[阿達罕|阿達罕]]、[[卡爾斯|卡爾斯]]、[[巴統|巴統]]等地区归还给[[奥斯曼土耳其帝国|奥斯曼土耳其帝国]]；
# 苏俄拆除[[阿兰群岛|阿兰群岛]]上所有防禦工事，其未來地位由德奧瑞芬四國共同決定。

此外，在8月27日時於柏林所簽訂之補充條款中尚規定苏俄需給付德國60億馬克賠款，分五期償還，但最後付款日期不得超過1918年底。

== 影响 ==

此條約對俄國是相當沉重之打擊，俄方所受之損失大致如下：
# 國土：共126萬7千平方公里，相當於25%的國土。<ref name="李邁先">李邁先著，"俄國史（下卷）"（台北：國立編譯館出版，1969），頁433</ref>
# 人口：共6200萬人，相當於全國44%的人口。<ref name="李邁先 "/>
# 產業：
## 農產損失33%
## 工業損失54%
## 製糖業損失80%
## 鐵礦損失73%
## 煤礦損失75%
# 財政收入：共損失約27%<ref name="李邁先 "/>

由於此條約和俄方當初的理想“不割地，不賠款”相距實在過大，不僅[[蘇維埃|蘇維埃]]中的[[孟什維克派|孟什維克派]]代表大力反對，連布尔什维克党內部亦有人反對。但列寧一再為其主張辯護，並將此合約與1807年俄法的《[[提爾西特條約|提爾西特條約]]》相比，只是為了暫時退讓並謀求反擊之機。最後於3月8日布尔什维克党第七屆代表大會以30：12通過，同月18日再由蘇維埃臨時特別代表大會以784：261通過。

儘管損失是如此之慘重，但苏俄最後也成功地退出了[[第一次世界大战|第一次世界大战]]。戰後60億馬克的賠款，最後因為德國戰敗而免付，也因為德軍的崩潰，蘇軍在四年之內收回了部分被割讓的領土，包括過半數的烏克蘭與白俄羅斯領土，但蘇軍於[[波蘇戰爭|波蘇戰爭]]（1919-1921年）中受阻於[[波蘭第二共和國|波蘭第二共和國]]，故大部分的失土（包含波羅的海三小國、波蘭）要等到[[第二次世界大戰|第二次世界大戰]]末，才由[[史達林|史達林]]擊敗德軍而佔領（史達林唯一沒攻下的失土為[[芬蘭|芬蘭]]）。1991年12月26日[[苏联解体|苏联解体]]后[[俄罗斯|俄罗斯]]再次丧失二战后收回的布列斯特合约中割让的领土（[[2014年克里米亚危机|2014年克里米亚危机]]后俄罗斯实际占领克里米亚并建立了[[克里米亚联邦管区|克里米亚联邦管区]]，但未获得国际社会广泛承认）。

== 条约的废弃 ==

德国战败后，于11月11日同协约国签订了停战协定，其中第二款《关于德国东部边界的规定》之第15条规定德国放弃《布加勒斯特和约》及《布列斯特-立托夫斯克和约》及其补充条约。全俄中央执行委员会随后于11月13日通过决议单方面废除此条约，使得该条约的内容实际上成了一纸空文。

1922年，[[苏联|苏维埃俄国]]与[[魏玛共和国|德意志国]]签署了《[[1922年拉帕洛条约|拉帕洛条约]]》。两国借此宣布放弃在布列斯特-立陶夫斯克條約及一战后向对方提出的领土和金钱之要求。
