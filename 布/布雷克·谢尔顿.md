'''布雷克·托尔利森·谢尔顿'''（{{Lang-en|Blake Tollison Shelton}}，{{bd|1976年|6月18日|}}<ref name="bio">{{cite web |url={{Allmusic |class=artist |id=p480679|pure_url=yes}} |title=Blake Shelton biography |accessdate=December 29, 2007 |last=Cohoon|first=Rick|publisher=AllMusic|language=en}}</ref>）是一名[[美国|美国]][[乡村音乐|乡村音乐]][[男歌手|男歌手]]和[[作曲人|作曲人]]。2001年，他通过首张同名专辑《布雷克·谢尔顿》主打歌《Austin》一炮而红，蝉联[[公告牌_(杂志)|公告牌]]榜首数周<ref name="bio" />。

他的第二张、第三张专辑，即2003年发行的《{{Internal link helper/en|The Dreamer}}》和2004年的《{{Internal link helper/en|Blake Shelton's Barn＆Grill}}》，分别赢得[[黄金唱片|黄金唱片]]和[[白金唱片|白金唱片]]荣誉。他于2007年的发表第四张专辑《{{Internal link helper/en|Pure BS}}》，《{{Internal link helper/en|Pure BS}}》于2008年再次发行，其中多包含了[[迈克尔·布伯雷|迈克尔·布伯雷]]的流行歌曲“家”作为附赠曲目<ref name="pro quest2">{{cite web |title= 'Home' Is Where the Hit Is | accessdate= September 27, 2014 |last=Tucker |first=Ken |publisher=Prometheus Global Media}}</ref>。2008年11月发表第五张专辑《{{Internal link helper/en|Startin' Fires}}》，2010年发表《{{Internal link helper/en|Hillbilly Bone and All About Tonight}}》，2011年发表《{{Internal link helper/en|Red River Blue}}》。他曾八次获得[[格莱美奖|格莱美奖]]提名。

布雷克·谢尔顿近年来也常以评委身份出现在[[电视|电视]]歌唱[[选秀|选秀]][[节目|节目]]中，如《[[美国之声|美国之声]]》等。从2011年《美国之声》开播起，他至今参与了13季，他所带领的歌手拿下其中6季冠军（第2、3、4、7、11、13季）<ref>{{cite web|author1=Luke Levenson|title='The Voice': Team Blake's Chloe Kohanski Wins Season 13|url=https://www.rollingstone.com/country/recaps/the-voice-team-blakes-chloe-kohanski-wins-season-13-w514482|accessdate=2017-12-27|language=en}}</ref>。

2011年2015年间，他与歌手{{Internal link helper/en|米兰达·兰伯特|Miranda Lambert}}有过一段婚姻。
