'''布林宾萨里机场'''（{{lang-id|Bandara Blimbingsari}}，{{airport codes|BWX|WADY，曾用WARB}}）是[[印度尼西亚|印度尼西亚]][[东爪哇省|东爪哇省]][[外南梦县|外南梦县]]的一座机场，主要服务该县的县治[[外南梦|外南梦]]。机场于2010年12月建成开放，最初的跑道长{{convert|1,400|m}}，仅能供小型螺旋桨飞机起降，例如[[塞斯纳208|塞斯纳208]]等机型。2012年跑道被延长到{{convert|1,800|m}}，可以为更大型的[[涡轮螺旋桨发动机|涡轮螺旋桨发动机]]飞机提供服务，例如[[福克50|福克50]]和[[ATR_72|ATR 72]]。2011年机场仅运送旅客7,826人次，2015年增长到110,000人次<ref>{{cite web |url=http://news.detik.com/berita/d-3316594/ketua-dpr-puji-bandara-blimbingsari-yang-lebih-bagus-dari-tasmania |title=Ketua DPR Puji Bandara Blimbingsari yang Lebih Bagus dari Tasmania |author=Kartika S Tarigan |date=October 9, 2016}}</ref>。由于民众较为熟悉“外南梦”这个名称，印尼交通部通过2017年第830号决定，机场将被更名为'''外南梦机场'''<ref>{{cite web |url=http://www.qiandaoribao.com/news/102127 |title=Blimbingsari机场将改名为“外南梦机场” |last= |first= |date=2017-10-16 |website=www.qiandaoribao.co.id |publisher=点滴新闻网外南梦讯 |access-date=2017-11-11 |quote=}}</ref>。机场附近有几座火山，火山活动会对机场运作造成影响，例如2015年[[拉翁火山|拉翁火山]]喷发导致外南梦机场一度被关闭<ref>{{cite web |url=http://www.bbc.com/zhongwen/simp/world/2015/07/150717_indonesia_volcanos_airports |title=开斋节前夕印尼火山灰造成机场关闭 |last= |first= |date=2015-07-17 |website=www.bbc.com |publisher=BBC中文网 |access-date=2017-11-11 |quote=}}</ref><ref>{{cite web |url=http://news.seehua.com/?p=72600 |title=印尼机场重开　大部分航线恢復 |last= |first= |date=2015-07-11 |website=news.seehua.com |publisher=诗华日报 |access-date=2017-11-11 |quote=}}</ref>。

2015年开始第二次改扩建工程，跑道将被延长到{{convert|2500|m}}，跑道{{le|机场跑道编号|Pavement Classification Number|PCN}}值将从PCN16变为PCN21<ref>{{cite web |url=http://www.beritasatu.com/destinasi/260811-okupansi-garuda-rute-banyuwangi-capai-80-persen.html |title=Okupansi Garuda Rute Banyuwangi Capai 80 Persen |author=Feriawan Hidayat |date=March 27, 2015}}</ref>，使得机场能为喷气客机提供服务，例如[[空中客车A320|空客A320]]、[[波音737|波音737]]和[[庞巴迪CRJ|庞巴迪CRJ]]等机型。扩建计划还包括新建一座{{convert|5,000|m2}}的航站楼以及修建更大的停机坪<ref>{{cite web |url=http://www2.thejakartapost.com/news/2013/01/06/banyuwangi-improving-airport-soon-open-jakarta-banyuwangi-route.html |title=Archived copy |accessdate=2013-01-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20130107135104/http://www2.thejakartapost.com/news/2013/01/06/banyuwangi-improving-airport-soon-open-jakarta-banyuwangi-route.html |archivedate=2013-01-07 |df= }}</ref>。2017年8月21日，机场航站楼落成，共有两层，设施齐全，包括候机室、小清真寺和咖啡厅，每年能处理25万人次的客流，外南梦机场成为印尼首座绿化设计的机场<ref>{{cite web |url=http://guojiribao.com/shtml/gjrb/20170822/330790.shtml |title=外南梦机场航站绿化设计令人惊叹 |last= |first= |date=2017-08-22 |website=guojiribao.com |publisher=国际日报 |access-date=2017-11-11 |quote=}}</ref>，总占地面积达到7.6公顷<ref>{{cite web |url=http://www.shangbaoindonesia.com/indonesia-finance/%E5%88%B0%E5%A4%96%E5%8D%97%E6%A2%A6%E8%BF%9B%E8%A1%8C%E5%B7%A5%E4%BD%9C%E8%AE%BF%E9%97%AE%E5%89%AF%E6%80%BB%E7%BB%9F%E8%A7%86%E5%AF%9F%E6%9C%BA%E5%9C%BA%E6%96%B0%E5%80%99%E6%9C%BA%E6%A5%BC.html |title=到外南梦进行工作访问 副总统视察机场新候机楼 |last= |first= |date=2016-12-15 |website=www.shangbaoindonesia.com |publisher=印度尼西亚商报外南梦讯 |access-date=2017-11-11 |quote= |archive-url=https://web.archive.org/web/20171112021412/http://www.shangbaoindonesia.com/indonesia-finance/%E5%88%B0%E5%A4%96%E5%8D%97%E6%A2%A6%E8%BF%9B%E8%A1%8C%E5%B7%A5%E4%BD%9C%E8%AE%BF%E9%97%AE%E5%89%AF%E6%80%BB%E7%BB%9F%E8%A7%86%E5%AF%9F%E6%9C%BA%E5%9C%BA%E6%96%B0%E5%80%99%E6%9C%BA%E6%A5%BC.html |archive-date=2017-11-12 |dead-url=yes }}</ref>。

2017年9月8日，[[加鲁达印尼航空|加鲁达印尼航空]]开通了外南梦直飞雅加达的航线，每天有6个班次<ref>{{cite web |url=https://qiandaoribao.com/news/100233 |title=鹰航将开通雅加达-外南梦直飞航线 |last= |first= |date=2017-09-05 |website=www.qiandaoribao.co.id |publisher=安打拉通讯社外南梦讯 |access-date=2017-11-11 |quote=}}</ref>。此外，{{le|楠航空|Nam Air}}运营的外南梦至雅加达航线每天也有3个班次<ref>{{cite web |url=http://www.guojiribao.com/shtml/gjrb/20170906/332755.shtml |title=鹰航雅加达直飞外南梦班机9月8日首航 |last= |first= |date=2017-09-06 |website=www.guojiribao.com |publisher=东爪哇新闻网外南梦讯 |access-date=2017-11-11 |quote=}}</ref>。
