'''布田站'''（{{jpn|j=布田駅|hg=ふだえき|rm=Fuda eki|f=y}}）是一個位於[[東京都|東京都]][[調布市|調布市]]國領町五丁目，屬於[[京王電鐵|京王電鐵]][[京王線|京王線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KO17'''。

== 年表 ==
* 1917年，作為京王電氣軌道的車站啟用，當時車站位於[[甲州街道|甲州街道]]北邊。
* 1927年12月17日，搬至現址。
* 1944年5月31日，京王與[[東京急行電鐵|東京急行電鐵]]合併後，布田站作為京王線的車站運行。
* 1948年6月1日，東急與京王帝都電鐵分拆，布田站成為京王的車站。
* 2007年3月4日，暫時採用[[跨站式站房|跨站式站房]]，同時車站設備無障礙化，以便傷殘人士使用。
* 2012年8月19日，改建為[[地下站|地下站]]。
* 2013年2月22日，以'''KO17'''作為布田站的編號。

== 車站構造 ==
本站原是[[相對式月台|相對式月台]]2面2線地上站。在[[調布市#京王線連續立體交差事業|京王線調布站附近連續立體交差事業]]完成後，本站現為[[島式月台|島式月台]]1面2線地下站。

過去本站月台位於彎道上，地下化後的月台為直線上。

地下化後與[[調布站|調布站]]、[[國領站|國領站]]皆設置[[月台門|月台門]]。本站為全罩式月台門。

== 站名由來 ==
{{See also|調布市#地名由來}}
自古以來，車站周邊便是盛產「布」的土地，猶如「'''布'''之'''田'''」，因而稱為「'''布田'''」。布田除調布市外，[[多摩川|多摩川]]對岸的[[川崎市|川崎市]][[多摩區|多摩區]]也有同樣地名（[[中野島站|中野島站]]附近）。
