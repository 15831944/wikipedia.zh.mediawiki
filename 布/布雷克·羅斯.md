'''布雷克·羅斯'''（{{lang-en|'''Blake Ross'''}}，{{bd|1985年|6月12日|catIdx=R}}），[[Mozilla_Firefox|Mozilla Firefox]]的最初開發者之一<ref>{{cite web |last=Goodger |first=Ben |authorlink=Ben Goodger |title=Where Did Firefox Come From? |work=Inside Firefox |publisher=MozillaZine Weblogs |date=2006-02-06 |url= http://weblogs.mozillazine.org/ben/archives/009698.html |archiveurl= https://web.archive.org/web/20110623034401/http://weblogs.mozillazine.org/ben/archives/009698.html |archivedate=2011-06-23 |accessdate=2012-01-07}}</ref>。美國媒體給予他「天才少年」的稱號<ref>http://www.flickr.com/photos/foolswisdom/49103131/</ref>，他十歲起就自行架設了網站、並設計自己所需的應用程式與網路遊戲。羅斯十四歲獲得在矽谷[[Netscape|Netscape]]公司實習的機會，他就在家擔任Netscape公司的瀏覽器軟體的測試員，並能為瀏覽器進行除錯，期間認識鼓吹網路創新與多元選擇的[[非營利組織|非營利組織]][[Mozilla基金會|Mozilla基金會]]<ref name=MiamNewTime>{{cite news|last=Lowell|first=Mike|title=Best Local Boy Made Good 2005 - Blake Ross|url=http://www.miaminewtimes.com/bestof/2005/award/best-local-boy-made-good-42738/|accessdate=2013-07-27|newspaper=Miami New Times|date=2005-04-27}}</ref>。年僅十七歲時，羅斯與另一位軟體工程師[[戴夫·海厄特|戴夫·海厄特]]開始撰寫一種新的瀏覽器，即Firefox的原型。

他們將[[Mozilla_Suite|Mozilla Suite]]軟體去蕪存菁，只留最精華的部份。以期讓瀏覽器運行更有效率。而且由於軟體是開放原始碼授權的，只要是有興趣加入開發的志願者都能一起加入開發的行列。羅斯說：「我有一些很不錯的回憶，就是整晚呆在Netscape查看人們對我們軟體意見反應。」<ref>{{cite web |title=Firefox for Dummies |url=http://boston-legal.org/dana/article-tdi-6-16-2006.pdf |publisher=Petersburg Times |date=2006-06-16}}</ref>

知名科技雜誌《Business2.0》稱這個新程式是「[[微軟|微軟]]最大的夢魘」，並盛讚羅斯是軟體奇才<ref>{{cite news |url=http://money.cnn.com/magazines/business2/business2_archive/2004/11/01/8189378/index.htm |title=Microsoft's Worst Nightmare |publisher=Business 2.0 |date = 2004-11-01}}</ref>。羅斯的母親透露，羅斯不僅是電腦高手，也彈得一手好鋼琴，更是「令人歎為觀止的寫作奇才」。羅斯七歲迷上電玩[[模擬城市|模擬城市]]，在電玩世界裡，建造自己理想的[[烏托邦|烏托邦]]<ref>{{cite web |title=Outfoxing Microsoft |url=http://www.sptimes.com/2005/01/01/news_pf/Floridian/Outfoxing_Microsoft.shtml |publisher=Petersburg Times |date=2005-01-01}}</ref>。羅斯強調，他只是Firefox的催生者之一，投入Mozilla行列的龐大義工才是真正的幕後功臣，2003年，羅斯在[[斯坦福大学|-{zh-hans:斯坦福大学;zh-hant:史丹佛大學;}-]]主修[[信息技术|-{zh-hans:信息技术;zh-hant:電腦科技;}-]]，同時也是Mozilla的義工之一，致力於開發[[開放原始碼|開放原始碼]]程式<ref>{{cite|url=https://alumni.stanford.edu/get/page/magazine/article/?article_id=34293|title=Mister Firefox}}</ref>。
