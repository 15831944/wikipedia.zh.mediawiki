'''布雷特·迈克尔斯'''（{{lang-en|Bret Michaels}}，{{bd|1963年|3月15日|||catIdx= M迈克尔斯}}）出生于[[美国|美国]][[宾夕法尼亚州|宾夕法尼亚州]]巴特勒，为[[毒药乐队|毒药乐队]]主唱人，美国歌手、电影演员、电视演员。

== 生平 ==
1963年，迈克尔斯出生于美国[[宾夕法尼亚州|宾夕法尼亚州]]巴特勒。<ref name="biography.com">{{cite web|url = http://www.biography.com/search/article.do?id=278900|publisher = Biography.com|title = Bret Michaels|accessdate=April 1, 2009}}</ref>
迈克尔斯有两个姐妹，一个叫米歇尔，一个叫妮可。<ref>{{cite web|url= http://www.tvguide.com/celebrities/bret-michaels/bio/142699|title= TV Guide biography}}</ref> 

== 个人生活 ==
迈克尔斯和女友[[克里斯蒂·吉布森|克里斯蒂·吉布森]]（Kristi Gibson）已经恋爱长达16年，2010年，他在[[真人秀|真人秀]]《布雷特·迈克尔斯》（Bret Michaels: Life As I Know It）中向这个长线女友求婚，获得成功。<ref name=" rat" >{{cite web |url= http://ent.qq.com/a/20101223/000413.htm|title=布雷特·迈克尔斯与女友相恋16年 求婚成功 |accessdate=2010年12月23日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

2010年，美国媒体曾爆料迈克尔斯和歌手[[麦莉·赛勒斯|麦莉·赛勒斯]]（Miley Cyrus）的母亲蒂什（Tish）有不当关系，并且导致了其家庭破裂，不过两人都否认此事。<ref name=" cob" >{{cite web |url=http://ent.qq.com/a/20101110/000326.htm |title= 毒药乐队主唱否认偷情 致麦莉·赛勒斯父母离婚|accessdate=2010年11月10日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>
