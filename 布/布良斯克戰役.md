'''布良斯克戰役'''（1941年10月2日至21日）是[[第二次世界大戰|第二次世界大戰]][[德蘇戰爭|德蘇戰爭]]中的一場戰役，發生於[[布良斯克州|布良斯克州]]（1944年6月5日前[[布良斯克|布良斯克]]屬於[[奧廖爾州|奧廖爾州]]管轄）。

== 戰役背景 ==
1941年9月，德軍第47軍團（指揮官為裝甲部隊將軍[[約艾辛·列梅森|約艾辛·列梅森]]）向前推進，並在[[莫斯科|莫斯科]]的東南區域越過[[傑斯納河|傑斯納河]]。9月30日時，德軍在嘗試包圍[[蘇聯|蘇聯]][[第50集團軍|第50集團軍]]時擊敗了[[布良斯克方面軍|布良斯克方面軍]]的正面。這導致了該[[方面軍|方面軍]]無法有效連結第50及[[第13集團軍|第13集團軍]]。[[布良斯克方面軍|布良斯克方面軍]]的指揮官[[安德烈·伊萬諾維奇·葉廖緬科|安德烈·伊萬諾維奇·葉廖緬科]]中將採取了[[縱深防禦|縱深防禦]]的戰略。該方面軍擁有兩個步槍師、一個戰車師一個戰車旅擔任預備隊，而這些部隊都被集中至[[布良斯克|布良斯克]]地區。然而第47軍團拿下了[[卡拉齊夫|卡拉齊夫]]並於10月6日成功推進至該城東部，他們並嘗試與從反方向推進而來的第43軍團會合。

這迫使紅軍第13、第50集團軍與[[葉廖緬科|葉廖緬科]][[中將|中將]]的戰略集團撤退來避免被包圍的命運，結果在未經歷多少戰鬥後布良斯克便完全疏散。即便如此，城市依然遭到德軍的重型火炮及空軍的轟炸，摧毀了[[布良斯克|布良斯克]]的大片區域。<ref>{{cite web |url=http://velikvoy.narod.ru/bitvy/1941/oryol-bryansk.htm |title=存档副本 |accessdate=2008-05-01 |0=2008-06-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20080610111902/http://velikvoy.narod.ru/bitvy/1941/oryol-bryansk.htm |archivedate=2008-06-10 }} Orel-Bryansk Defensive Operation (in Russian)</ref>

[[蘇聯|蘇聯]]在[[奧廖爾|<nowiki/>]][[奧廖爾-布良斯克防禦行動|奧廖爾-布良斯克防禦行動]]（1941年9月30日至10月23日）中，光是[[布良斯克|布良斯克]]及週遭區域就折損了80,000多人。在第47軍團推進至它下一個目標[[奧廖爾|奧廖爾]]前，又有50,000名[[蘇軍|蘇軍]]被俘擄。
