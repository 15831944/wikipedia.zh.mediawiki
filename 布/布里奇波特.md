'''布里奇波特'''（{{lang-en|Bridgeport}}），或譯'''橋港'''，是[[美國|美國]][[康乃狄克州|康乃狄克州]]最大的城市 （2010年人口144,229人）<ref name=popest2>{{cite web | year = 2006年 | url = http://www.census.gov/popest/cities/files/SUB-EST2006_9.csv | title = Annual Estimates of the Population for All Incorporated Places in Connecticut | format = [[Csv|CSV]] | work = 2006 Population Estimates | publisher = U.S. Census Bureau, Population Division | accessdate =2007-06-28}}</ref> 、[[新英格蘭|新英格蘭]]第五大的城市。位於[[長島海灣|長島灣]]岸、[[佩闊諾克河|佩闊諾克河]]河口。

== 地理与气候 ==
桥港属温带[[大陆性气候|大陆性气候]]，四季分明，降雨与日照充足。冬季较冷，微潮，时而偏向寒冷，日最高气温低于{{convert|0|C}}的平均日数为20天，日最低气温低于{{convert|-10|C}}的平均日数为12天；夏季相对炎热潮湿，日最高气温超过{{convert|30|C}}的日数年均有21天，{{convert|35|C}}以上的气温只出现在48%的夏季。<ref name = "NWS Upton, NY"/>最冷月（1月）均温{{convert|30.1|F|1|disp=flip}}，极端最低气温{{convert|−7|F|0|disp=flip}}（1984年1月22日）。<ref name = "NWS Upton, NY"/>最热月（7月）均温{{convert|74.3|F|1|disp=flip}}，极端最高气温{{convert|103|F|0|disp=flip}}（1957年7月22日、2011年7月22日）。<ref name = "NWS Upton, NY"/>年均降水量约{{convert|42.7|in|sigfig=3|disp=flip}}，分布较为平均；年极端最少降水量为{{convert|23.03|in|disp=flip}}（1964年），最多为{{convert|73.93|in|disp=flip}}（1972年）。<ref name = "NWS Upton, NY"/>年均降雪量为{{convert|27.6|in|cm|disp=flip}}（北郊平均降雪量可达{{convert|75|cm|disp=/}}以上），但降雪年际变化较大；1972–73年的降雪量最少，积累降雪量只有{{convert|8.2|in|cm|disp=flip}}，1995–96年的降雪量最多，积累降雪量为{{convert|75.7|in|cm|disp=flip}}。<ref name = "NWS Upton, NY"/>无霜期平均为209天（4月9日至11月3日）；可测量降雪平均期为12月6日至3月19日。<ref name = "NWS Upton, NY"/>
