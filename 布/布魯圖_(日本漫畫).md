《'''布鲁图'''》是[[日本|日本]][[漫畫家|漫畫家]][[浦澤直樹|浦澤直樹]]所繪製的一部[[科幻|科幻]][[懸疑|懸疑]]漫畫。改編自手塚治虫的漫畫作品《[[原子小金剛|原子小金剛]]》中的〈地上最大機器人〉篇。單行本於日本發行超過850萬冊。<ref>[http://eiga.com/news/20101021/1/ 浦沢直樹「PLUTO」を米イルミネーションが実写化]映画.com 2010年10月21日</ref>

== 概要 ==
《原子小金剛》的〈地上最大機器人〉是作者浦澤直樹生平第一次受到感動的漫畫，也是這個作品讓浦澤立定志向要成為一位漫畫家，因此浦澤非常希望能夠重新繪製這篇作品。2002年的冬季，他向手塚治虫的兒子手塚真提出了這個構想，浦澤積極的態度和熱情打動了手塚真，於是在2003年3月讓浦澤開始重新繪製「地上最大機器人」，並於同年9月開始於日本的《Big Comic Original》（[[小學館|小學館]]）雜誌連載。

承繼手塚治虫作品的浦澤，並不希望因為尊敬該作品而只是重新描繪一遍。因此角色和故事是一半依循原作，另一半則有加入浦澤風格的設定。原本在手塚治虫原作中是由[[铁臂阿童木|铁臂阿童木]]擔任主角，而在浦澤版本中則以原作中的配角[[機器人|機器人]]刑警蓋吉特」（{{lang|de|Gesicht}}）的觀點來描述故事。此外在《[[铁臂阿童木|铁臂阿童木]]》其他故事中、以及手塚其他作品出現過的角色皆有在本作登場，是讓手塚漫畫愛好者會心一笑的安排。

本作品的主題之一為「機器人的人權要受到保障」，作品中描述的世界上機器人數量幾乎和人類一樣，機器人也和人類同樣以「生物」的生活方式表現。

曾獲得第9回[[手塚治虫文化賞|手塚治虫文化賞]]漫畫大獎、平成17年度[[文化廳媒體藝術祭|文化廳媒體藝術祭]]漫畫部門優秀賞。

== 故事 ==

這是一個人類與機器人共同生活的時代。瑞士最強的機器人—「蒙布朗」（{{lang|ja|モンブラン}}）在一場森林大火中被破壞。同時，在德國支援「機器人保護法」之團體的幹部也遭殺害。在這兩個人的屍體之頭部，都插著一個形狀類似「角」的不明物體。以高性能著稱的機器人刑警「蓋吉特」（{{lang|de|Gesicht}}）開始調查這一連串看似相關的案子，結果卻意外發現包括自己在內的7位世界上最優秀的機器人，都被兇手列入殺害的目標……

== 登場角色 ==
<small>以下登場角色名稱使用中文維基地區的官方譯名，部份字詞有其原本的含意，但因在此做姓名用故採音譯。</small>
;蓋吉特（{{lang|de|Gesicht}}、{{lang|ja|ゲジヒト}}）
:七個「世界上最優秀機器人」的其中一名，製作者是霍夫曼博士。曾參加第39次中亞衝突戰役的和平軍。替治安警察部隊在街頭臥底，並殲滅恐怖份子。解決過許多困難的案件，以特別搜查官機器人的身份在[[德國|德國]][[杜塞爾多夫|杜塞爾多夫]]（{{lang|de|Dusseldorf}}）活躍。現在的階級為「警部」。過去，在怒意的驅動之下，殺了不斷殘殺機械人小孩的人類(死者其中包含了蓋吉特的養子，羅比達)。歐盟當局為了掩蓋事實，不但消除了相關記憶，還添上在歐洲受訓的虛假記憶。被刪除的記憶包含，曾經與妻子有一個養子，養子被人殺掉，以及殺掉人類兇手的記憶。此段記憶在與小金剛交換記憶芯片時，被小金剛讀取到。
:形態：人類型。外表約為40歲左右的歐洲男性，頭髮稀少。身體的部份能自由變化形狀，例如能夠將自己的手變為槍枝進行戰鬥。

;阿童木（{{lang|en|Atom}}、{{lang|ja|アトム}}）
:七個「世界上最優秀機器人」的其中一名，曾參加第39次中亞衝突戰役的和平軍。現在以「和平使者」的身份受到全世界的歡迎，是人們的偶像。曾被賣到馬戲團。與家人一同居住在[[日本|日本]]東京市，目前就讀小學。在世界上最優秀的機器人中，他擁有最高等級的[[人工智慧|人工智慧]]。因此能夠準確的模擬人類的行為，並且能夠理解人類的感情。在與Pluto戰鬥後一直昏迷不醒，被確定為死亡,後來被其製作者天馬博士成功「修復」。但陷入昏迷不醒，之後植入蓋吉特的記憶晶片以單一情感的方式將陷入混亂的思緒迴路中清醒。
:形態：人類型。外表為日本男性小孩。頭上為普通的頭髮，並沒有原作中的角。（有趣的是、小金剛因睡眠而弄亂翹起的頭髮、讓人想到原作中有角的造型。） 

;蒙布朗（{{lang|en|Mont Blanc}}、{{lang|ja|モンブラン}}）
:七個「世界上最優秀機器人」的其中一名，曾參加第39次中亞衝突戰役的和平軍。在前線與伯朗德和海格力斯一起並肩驅逐波斯軍的機器人。在戰役之後進入[[瑞士|瑞士]]森林局工作，擔任盧塞恩林區的森林保護官。由於對解決中亞衝突的功績，以及喜愛作詩唱歌、溫和惇厚的個性，是全人類都很喜歡的機器人，被Pluto所殺，是這一連串事件最初的被害者。
:形態：7名「世界上最優秀機器人」其中唯一的舊式外型機器人。

;諾斯2號（{{lang|en|North 2}}、{{lang|ja|ノース2號}}）
:七個「世界上最優秀機器人」其中一名，曾擔任過[[英國|英國]]軍隊總司令安德烈·德古拉斯將軍的副官，參加過第39次中亞衝突戰役。之後經由人力仲介公司介紹，替隱居在歐洲聯邦[[蘇格蘭|蘇格蘭]]的音樂家波爾·鄧肯擔任管家。在與Pluto的戰鬥中死亡。
:形態：臉部帶著面具，唯一露出來的嘴巴部份是類似人類的樣子。全身擁有許多種致命武器（原作中的設定是有許多隻手臂，除了其中的兩隻，其他每隻手臂前端都是武器）。平常時除了普通的兩隻手臂以外，其它的手臂都藏在覆蓋全身的斗篷下。足部配備有噴射裝置可以飛行。

;布蘭德（{{lang|en|Brand}}、{{lang|ja|ブランド}}）
:七個「世界上最優秀機器人」其中一名，參加過第39次中亞衝突戰役。參與機械人格鬥，[[土耳其|土耳其]]的機械人格鬥冠軍。有機械人妻子和收養了不少小孩。穿著比賽用戰鬥服與Pluto戰鬥後死亡。

;海格立斯（{{lang|en|Heracles}}、{{lang|ja|ヘラクレス}}）
:七個「世界上最優秀機器人」其中一名，參加過第39次中亞衝突戰役。之後參與機械人格鬥，曾與布蘭德進行過四場比賽，不分勝負，是[[希臘|希臘]]的機械人格鬥冠軍。在布蘭德死後，想為他報仇，穿著軍用戰鬥服與Pluto戰鬥後死亡。

;艾普希倫（{{lang|en|Epsilon}}、{{lang|ja|エプシロン}}）
:七個「世界上最優秀機器人」其中一名，拒絕第39次中亞衝突戰役的徵兵，能發動強大的光子能量。戰爭後在[[澳大利亞|澳大利亞]]收養了不少孤兒。曾目擊海格立斯被Pluto所殺。不想殺死Pluto，堅持以正當手段把Pluto逮捕，但這信念在其製作者牛頓·霍華德博士被殺後似乎有點動搖。
:最後為了救出被綁架的孩子，與PLUTO一戰後死去。

;亞伯拉
:為波斯王國的首席科學家，曾參與國家的綠化計畫「波拉計畫」，製造名為「波拉」的綠化機器人。為了要完成機器人的電子頭腦,找來天馬博士設計出世界上最優秀的人工智能。但是，因為人工智能容納全世界60億人的個性，因此無法覺醒。在中亞戰爭中失去親人而產生憎恨。這份憎恨在他被空襲炸死之後，連同他事先準備好的記憶芯片一同裝入人工智能使其完成。而這個人工智能分裂成兩個人格，一個是製造PLUTO的亞伯拉，另一個是完成波拉的高地。

;高地
:外傳為替波斯王國設計大規模機器人兵器的科學家，實際上是天馬博士製造出來的人工智能。高地這個名字代表當地傳說中的賢者之名。在裝入帶有亞伯拉憎恨之記憶芯片後覺醒，同時以為自己是亞伯拉是人類，並且設計出帶有反質子炸彈的巨大機器人「博拉」想要毀滅一切。

;PLUTO
:人工智能的亞伯拉人格製造出來的巨大機器人。目的是為了向參與中亞戰爭的七位「世上最優秀的機器人」復仇而製作出來。其原來身分為薩哈多，是亞伯拉的機器人兒子。本來在荷蘭研究植物學想協助亞伯拉綠化波斯王國，但因為戰爭回國參戰。之後其電子頭腦被裝入PLUTO體內，成為PLUTO到處破壞。在和小金剛決戰後醒悟，並且犧牲自己阻止「博拉」爆炸。
:型態是帶有長角的巨大機器人，其角能夠任意延長、變形，而且拳頭也能飛出去攻擊。擁有無人可敵的龐大怪力，並且能夠刮起巨大龍捲風用來移動和攻擊，而且發出的電磁波也能用來控制機器人。

;布劳1589
:高智能机器人，由于是第一个违反《机器人法》“机器人不能杀人”原则的机器人而被处于求生不能求死不能的极刑。盖吉特探访时，他一语点破，人类连杀死他的勇气都没有。

;[[黑傑克|黑傑克]]
:[[怪醫黑傑克|怪醫黑傑克]]的主角，在故事中是一個「日本醫生」，雖然沒有明確表示身分、也沒有露臉，但可從無照行醫、醫術高超等特點推知。

== 用語 ==
*[[機器人|機器人]]（{{lang|ja|ロボット}}）
:這個時代的[[機器人|機器人]]與[[人類|人類]]一樣、擁有一定的人權、似乎也有公民權。外型方面、不管是與[[人類|人類]]十分相近者或是與一般印象中[[機器人|機器人]]的外型相符者都有。另外、也有可依作業之類的用途而改變自己身體(擁有複數的身體)的類型存在。
*第39次中亚纷争（第39次中央アジア紛争）
:中亚地区国家波斯王国由于大量破坏机器人并压榨机器人人权，并以机器人兵团侵犯邻国领土引发战争。由各国科学家组装成的波拉调查团在清真寺地下发现大量机器人残害，在色雷斯合众国的要求下以违反了国际条约为名派去世界上最强大的七个机器人推翻了波斯王国。然而，悲惨的战争场面深深地影响了七个机器人。
*波斯王国
:由达利奥十四世统治中亚独裁国家。强权君主制压迫下民不聊生且违反机器人人权法案，组成的机器人兵团不断侵犯邻国领土并意图统治整个中亚。后被世界上最强的七个机器人推翻，改为实行民主制。此“波斯王国”为作者虚构，与历史上波斯王国地理位置等均不相同。
*KR团
:反对机器人与人平等，要求废除机器人人权法案的极右团体，主张“机器人是奴隶”，包含知识精英、财阀、政客等有影响力的人士。
*波斯战争综合症
:战争对精神造成的巨大伤害，属于[[创伤后心理压力紧张综合症|创伤后心理压力紧张综合症]]，书中由艾普希伦谈到他抚养的战争遗孤时提出。
