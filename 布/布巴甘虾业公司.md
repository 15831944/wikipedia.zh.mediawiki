'''布巴甘虾业公司'''（{{lang-en|Bubba Gump Shrimp Company}}），是一所美国虾业公司，下属全球连锁的餐厅与市场，其创业灵感来自1994年的[[美国电影|美国电影]]《[[阿甘正传|阿甘正传]]》（{{lang|en|Forrest Gump}}）。截止2010年9月，全球共有32家连锁餐厅，其中22家位于美国、3家位于[[日本|日本]]、2家位于[[墨西哥|墨西哥]]与[[马来西亚|马来西亚]]，此外[[菲律宾|菲律宾]]、[[印度尼西亚|印度尼西亚]]、[[香港|香港]]各有一家。

== 概述 ==
[[File:BubbaGumpRestaurantHK.JPG|thumb]]
1996年，最早的布巴甘餐厅由[[維亞康姆|維亞康姆]]建立于[[加利福尼亚州|加利福尼亚州]]的[[蒙特雷_(加利福尼亚州)|蒙特雷]]，[[維亞康姆|維亞康姆]]也是[[派拉蒙電影公司|派拉蒙電影公司]]的股东（派拉蒙電影公司为电影《阿甘正传》的制片方）。布巴甘餐厅以《阿甘正传》的两位主角本杰明·巴弗布鲁“'''布巴'''”（{{lang|en|Benjamin Buford "Bubba"}}）与[[阿甘_(電影角色)|福雷斯特·'''甘''']]（{{lang|en|Forrest Gump}}）命名。电影中，布巴曾对阿甘提议了建立虾业公司的想法，而阿甘也在[[越南战争|越南战争]]后兑现了这个诺言，并以“布巴·甘”命名<ref>[http://nymag.com/listings/restaurant/bubba-gump-shrimp-co/ NY Magazine: Bubba Gump Shrimp Co.]</ref>。

布巴甘餐厅主要的菜谱为各类虾食，此外还有其他种类的[[海鲜|海鲜]]与{{link-en|肯瓊料理|Cajun cuisine|肯琼烹饪}}（{{lang|en|Cajun cuisine}}）。

== 著名員工 ==
美國演員[[克里斯·普瑞特|克里斯·普瑞特]]曾於[[夏威夷州|夏威夷州]][[茂宜島|茂宜島]]的布巴甘蝦業公司擔任服務員，2000年他被演員兼導演的Rae Dawn Chong發現，她曾演出其中一部柏特最喜愛的電影《[[獨闖龍潭|獨闖龍潭]]》。Chong於自己執導的短片中提供一個角色給柏特。<ref>{{Cite web |url = http://www.independent.co.uk/arts-entertainment/films/features/chris-pratt-from-parks-and-recreation-to-guardians-of-the-galaxy-9636202.html |title = Chris Pratt: From Parks and Recreation to Guardians of the Galaxy |author = Kaleem Aftab |date = 2014-07-30 |publisher = Independent |language = en}}</ref>
