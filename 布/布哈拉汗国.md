'''布哈拉汗国'''（{{lang-uz|Buxoro Xonligi}}，{{lang-tg|Χоноти Бухоро}}）是1500年至1920年间位于[[中亚|中亚]][[河中地区|河中地区]]的[[伊斯兰教|伊斯兰教]]封建国家，国名因16世纪中叶迁都至[[布哈拉|布哈拉]]而得名。布哈拉汗国共分为三个王朝：[[昔班尼王朝|昔班尼王朝]]（1500年—1599年）、[[阿斯特拉罕王朝|阿斯特拉罕王朝]]（又称札尼王朝，1599年－1785年）和[[曼吉特王朝|曼吉特王朝]]（又称海达尔王朝、[[布哈拉酋長國|布哈拉酋長國]]，1785年－1920年）。1920年，被[[苏维埃俄罗斯|苏维埃俄罗斯]]所吞并。

== 基本概况 ==
布哈拉汗国是位于中亚河中地区的伊斯兰教封建国家。19世纪初，其疆域除[[泽拉夫尚河|泽拉夫尚河]]和[[卡什卡河|卡什卡河]]流域外，还包括今[[土库曼斯坦|土库曼斯坦]]、[[塔吉克斯坦|塔吉克斯坦]]和[[阿富汗|阿富汗]]的部分地区，以及[[锡尔河|锡尔河]]北岸、[[土耳其斯坦城|土耳其斯坦城]]及其附近一带。布哈拉汗国原定都于[[撒马尔罕|撒马尔罕]]，16世纪中叶迁至布哈拉。主要居民为[[乌兹别克人|乌兹别克人]]，其次为[[塔吉克人|塔吉克人]]。居民多从事农业、牧业，其中灌溉工程较为发达。
== 相关历史 ==
1500年，乌兹别克首领[[昔班尼|昔班尼]]率兵自北进入河中地区，占领撒马尔罕，推翻[[帖木儿|帖木儿]]后裔在中亚的统治，建立昔班尼王朝。该王朝原定都于撒马尔罕，1561年迁都布哈拉，遂称布哈拉汗国。1505年，昔班尼夺取[[花剌子模|花剌子模]]。1507年，攻取[[赫拉特|赫拉特]]，灭[[帖木儿帝国|帖木儿帝国]]。1510年，昔班尼为[[费尔干纳|费尔干纳]]统治者[[巴布尔|巴布尔]]和[[萨非王朝|萨非王朝]]的[[伊斯迈尔一世|伊斯迈尔一世]]的联军所败，于[[梅爾夫戰役|梅爾夫戰役]]阵亡。昔班尼死后，国家濒于崩溃。1512年，昔班尼叔父[[库赤昆吉|库赤昆吉]]执政时，率军在[[尼尔杜万战役|尼尔杜万战役]]中打败萨非王朝和巴布尔的军队，恢复和巩固了乌兹别克人在河中地区的统治。在[[阿布都拉二世|阿布都拉二世]]统治时期，布哈拉汗国国势强盛，重新征服[[大呼罗珊|呼罗珊]]、花剌子模，夺取费尔干纳和[[塔什干|塔什干]]，并侵入[[哈萨克草原|哈萨克草原]]腹地。1598年，阿布都拉二世死后，国内动乱，呼罗珊等地复失。

1599年，阿斯特拉罕王朝开始统治布哈拉汗国。该朝[[伊玛目·库里|伊玛目·库里]]在位时，加强中央集权，以[[伊斯兰教法|伊斯兰教法]]治国，政治经济相对稳定，商业繁荣，纺织品、陶器、纸张远销国外。其后乌兹别克各氏族割据势力日渐强大，王族内讧加剧，[[印度|印度]][[莫卧儿王朝|莫卧儿王朝]]和[[希瓦汗国|希瓦汗国]]也乘机频繁入侵。[[苏布罕·库里|苏布罕·库里]]在抵御外部入侵和平定内乱中取得一定成就，但汗国日趋衰落和陷于分裂。18世纪前期，[[浩罕|浩罕]]（中心在费尔干纳）从布哈拉汗国分裂出去，建立了独立的[[浩罕汗国|浩罕汗国]]。1740年，[[阿夫沙尔王朝|阿夫沙尔王朝]][[纳迪尔沙|纳迪尔沙]]入侵，[[阿布都拉·哈兹|阿布都拉·哈兹]]投降，于1747年遇害。

阿斯特拉罕王朝末代可汗[[阿布·加齐|阿布·加齐]]统治期间，曼吉特首领[[马桑·沙·穆拉德|马桑·沙·穆拉德]]迎娶其女，建立曼吉特王朝。在他统治期间，采用[[埃米尔|埃米尔]]称号，实行行政和财政方面的改革，兴修水利，发展农业，加强了布哈拉汗国的中央政权。[[纳斯尔·阿拉赫·巴哈杜尔|纳斯尔·阿拉赫·巴哈杜尔]]统治时期，加强军队建设，不断扩张领土，力图称霸河中地区，并多次出兵对[[浩罕汗国|浩罕汗国]]进行军事干涉。1868年，在[[穆扎法尔丁·巴哈杜尔|穆扎法尔丁·巴哈杜尔]]统治时期，布哈拉汗国沦为[[沙皇俄国|沙皇俄国]]的附庸。1920年9月，当地人民和[[苏俄|苏俄]][[红军|红军]]推翻曼吉特王朝，建立[[布哈拉苏维埃人民共和国|布哈拉苏维埃人民共和国]]，布哈拉汗国正式灭亡。

== 政治文化 ==
布哈拉汗国为[[政教合一|政教合一]]的军事封建国家。汗国奉伊斯兰教[[逊尼派|逊尼派]]教义为国教，实行伊斯兰教法和部落习惯法相结合的司法制度。[[和卓|和卓]]家族、伊斯兰教长老在宗教和政治上享有较高的社会地位。[[苏菲派|苏菲派]][[纳格什班迪教团|纳格什班迪教团]]和[[库布拉维耶教团|库布拉维耶教团]]受到汗国的尊崇，赐封有大量土地，建有豪华的[[麻札|麻札]]。

布哈拉汗国将伊斯兰文化加以继承和发展。撒马尔罕和布哈拉为中亚伊斯兰学术文化中心，汗国除对原有的清真寺、图书馆、天文台进行修葺扩建外，还新建有多所清真寺和伊斯兰教经学院，其中撒马尔罕的[[希尔达尔伊斯兰经学院|希尔达尔伊斯兰经学院]]为中亚地区的伊斯兰教育中心，培养了大批学者。苏菲派学者对《[[古兰经|古兰经]]》和[[圣训|圣训]]作了注释和考证。此外，布哈拉汗国文学、史学盛极一时，建筑、绘画、书法等也达到较高的艺术水平。
