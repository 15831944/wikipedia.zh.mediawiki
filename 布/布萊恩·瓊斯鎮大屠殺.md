'''布萊恩·瓊斯鎮大屠殺'''（{{lang-en|'''The Brian Jonestown Massacre'''}}）是一個美國音樂團隊，由{{link-en|安東·紐康姆|Anton Newcombe}}領導。樂團於1990年在[[三藩市|三藩市]]成立。

布萊恩·瓊斯鎮大屠殺是由紐康姆於1990年組成的。他們早期的音樂風格是[[瞪鞋搖滾|瞪鞋搖滾]]。在他們的首張專輯《''{{link-en|Methodrone|Methodrone}}''》之後，樂團的音樂轉變為更廣泛的[[迷幻搖滾|迷幻搖滾]]風格，將如[[車庫搖滾|車庫搖滾]]、[[民謠搖滾|民謠搖滾]]，以及後來的[[電子音樂|電子音樂]]風格融入到他們的音樂中。他們繼續對多個音樂風格進行嘗試。

樂團是2004年紀錄片《''{{link-en|Dig!|Dig!}}''》的主題，並且因為他們動盪的工作關係，還有領導者紐康姆的古怪行為而獲得媒體的惡名昭著。到目前為止，布萊恩·瓊斯鎮大屠殺已經發行了十七張專輯、五張[[合輯|合輯]]、五張現場專輯、十三張[[迷你專輯|迷你專輯]]、十六張[[單曲|單曲]]，以及兩張不同表演者合輯。

== 1993年–1996年：早年 ==
樂隊的名稱是一個[[混成詞|混成詞]]，以搖滾樂隊[[滾石樂隊|滾石樂隊]]的創始人和吉他手，以及[[圭亞那|圭亞那]][[瓊斯鎮|瓊斯鎮]]臭名昭著的大型自殺[[邪教|邪教]]而命名。<ref>{{cite news|title=A timeless sound keeps psych-rockers BJM relevant|first=Kevin |last=Hopper |work=Albuquerque Journal|date=January 2, 2004|page=15}}</ref><ref>{{cite news|title=Town Without Pity; 30 Years Later, Memories of Jonestown Evoke Guilt, Anger and Mistrust|first=Charles A |last=Krause|work=The Washington Post|date=November 19, 2008|page=C.1}}</ref>

布萊恩·瓊斯鎮大屠殺的首張專輯《''{{link-en|Spacegirl and Other Favorites|Spacegirl and Other Favorites}}''》於1995年只作黑膠唱片發行，限量500張。<ref>{{Cite web |url = https://www.last.fm/music/The+Brian+Jonestown+Massacre/Spacegirl+and+Other+Favorites |title = Spacegirl and Other Favorites |publisher = Last.fm |language = en}}</ref>專輯包括1994年發行的"Hide and Seek"，樂隊接著的專輯《''{{link-en|Methodrone|Methodrone}}''》受到在發行前幾年獲得重視的[[瞪鞋搖滾|瞪鞋搖滾]]類型的很大影響。專輯的超凡搖滾音樂與例如[[Galaxie_500|Galaxie 500]]、{{link-en|Spacemen 3|Spacemen 3}}和{{link-en|My Bloody Valentine (band)|My Bloody Valentine (band)|My Bloody Valentine}}的樂隊相媲美。專輯中的兩首曲"She Made Me"和"Evergreen"，曾於1992年以雙[[A面|A面]]單曲發行。

樂隊在1996年經歷了熱烈的錄音時期，發行了三張大型專輯。《''{{link-en|Their Satanic Majesties' Second Request|Their Satanic Majesties' Second Request}}''》反映1960年代{{link-en|迷幻文化|Psychedelia}}的模仿作品。專輯還包括了與各種不同樂器的廣泛試驗，包括Indian drones、[[錫塔琴|錫塔琴]]、[[電子琴|電子琴]]、{{link-en|Farfisa|Farfisa}}、{{link-en|迪吉里杜管|Didgeridoo}}、[[塔布拉鼓|塔布拉鼓]]、[[康加鼓|康加鼓]]和[[鐵片琴|鐵片琴]]。專輯的標題是滾石樂隊的1967年專輯《''{{link-en|Their Satanic Majesties Request|Their Satanic Majesties Request}}''》的一個模仿作品。<ref>{{Cite web |url = https://www.allmusic.com/album/their-satanic-majesties-second-request-mw0000045070 |title = Their Satanic Majesties' Second Request |publisher = AllMusic |language = en}}</ref>

樂隊的第二張專輯是1996年發行的《''{{link-en|Take It from the Man!|Take It from the Man!}}''》，源於[[節奏藍調|節奏藍調]]，並受到滾石樂隊的極大影響。<ref>{{Cite web |url = https://www.allmusic.com/album/take-it-from-the-man%21-mw0000613104 |title =  Take It from the Man! |publisher = AllMusic |language = en}}</ref>專輯包括歌曲"Straight Up and Down"，後來被用作[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》（2010年-2014年）的[[主題曲|主題曲]]。

在那年發行的第三張和最後一張專輯是《''{{link-en|Thank God for Mental Illness|Thank God for Mental Illness}}''》。專輯探索了[[鄉村音樂|鄉村]]和節奏藍調等類型。<ref>{{Cite web |url = https://www.allmusic.com/artist/the-brian-jonestown-massacre-mn0000614455/biography |title = The Brian Jonestown Massacre Biography |author = Jason Ankeny |publisher = AllMusic |language = en}}</ref>專輯分為兩部分，第一部分主要是低音，原聲歌曲，而後半部分是一系列歌曲合併成一首名為"Sound of Confusion"的曲目。"Sound of Confusion"具有平常歌曲和非常抽象聲音的大雜燴。
