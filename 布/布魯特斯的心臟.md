《'''布魯特斯的心臟'''》是日本作家[[東野圭吾|東野圭吾]]的[[長篇小說|長篇]][[推理小說|推理小說]]。[[光文社|光文社]]於1989年發行了{{link-ja|新書|新書}}，1993年發行了[[光文社|光文社]][[文庫版|文庫版]]，副標題{{nihongo|〈完美犯罪殺人電驛〉|完全犯罪殺人リレー|romaji=Kanzen Hanzai Satsujin Relay}}是文庫本之後才掛上去的<ref>[http://from1985.pekori.to/keigotaku/review/buru.html ブルータスの心臓作品DATA]</ref>，2009年發行了新裝版改封面。本作講述擔任人工智慧機器人開發的拓也，原本將施行一項計畫，然而施行當中卻全然翻盤，故事隨之發展下去。

在2011年6月17日播放由[[藤原龍也|藤原龍也]]主演的電視劇。

== 故事簡介 ==
在產業機器製造廠裡參與開發[[人工智能|人工智能]][[機器人|機器人]]的末永拓也陷入了困境，因為他的情人雨宮康子懷孕了，但她拒絕[[墮胎|墮胎]]。一心要往上爬的拓也，計劃要成為公司社長的女婿。如果這件事暴露的話，他的計劃就會失敗。正當拓也苦無對策而感到焦慮之際，他發現另外有兩個人一樣和康子有親密關係。其中一人提出了一個意外的提議：合謀把康子殺掉。於是這群男人開始了完全犯罪的謀殺接力。

; {{nihongo|橋本 敦司|{{ruby|橋本|はしもと}} {{ruby|敦司|あつし}}|Hashimoto Atsushi}}

: MM重工開發企劃室職員，直樹的秘書。

; {{nihongo|酒井 悟郎|{{ruby|酒井|さかい}} {{ruby|悟郎|ごろう}}|Sakai Gorou}}

; {{nihongo|仁科 敏樹|{{ruby|仁科|にしな}} {{ruby|敏樹|としき}}|Nishina Toshiki}}
