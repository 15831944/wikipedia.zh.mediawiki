'''樱花'''是[[蔷薇科|蔷薇科]][[李屬|李屬]]中一些被稱為[[櫻|櫻]]的物種（包括典型樱亚属''Cerasus''和矮生樱亚属''Lithocerasus''）所開的[[花|花]]。

== 觀賞類櫻花 ==
{{refimprove|date=2015年3月}}

一些品種的櫻花所結的果实因味道欠佳不作食用，但可作繁殖櫻花樹之用，在溫帶地區平地即能結果，在亞熱帶地區[[海拔|海拔]]則需高於500公尺。{{Fact|亞洲品種的觀賞樱花最初的痕迹出现于喜马拉雅山区(参考资料：《樱大鉴》)。[[唐朝|唐朝]]時，这物种逐步传入[[印度|印度]]北部，中國長江流域以及南部偏西地区，台湾，韩国，日本地区。|time=2019-04-19}}这些地区，尤其是日本，成了相对独立的樱花的变种发展的地方。在[[日本|日本]]廣為種植，在現代被日本人視為花的代表。[[奈良縣|奈良縣]][[吉野山|吉野山]]的櫻花最為聞名，故被譽為“吉野千本櫻”。整個日本櫻花花期約為五十天，但每個地區從花開到花謝只有七至十天。櫻花種類繁多，故[[顏色|顏色]]並非純是粉紅色，相應的花有相符的顏色。春天[[賞櫻|賞櫻]]是日本傳統習俗之一，過去日本氣象廳每年會發表櫻花開花日期預測，是為“[[櫻前線|櫻前線]]”，目前櫻花開花預報已移轉至{{link-ja|日本氣象協會|日本気象協会}}發布。<ref>[http://www.tenki.jp/sakura/expectation.html 桜の開花予想 ]日本気象協会 </ref> 日本氣象廳僅保留櫻花等生物季節觀測資料。<ref>[http://www.data.jma.go.jp/sakura/data/ 生物季節観測の情報] 気象庁</ref>
至2012年為止氣象廳的櫻花標本木觀測對象:北海道是大山櫻，沖繩是寒緋櫻，其餘地區是染井吉野櫻。<ref>[http://www.data.jma.go.jp/sakura/data/sakura2012.pdf さくらの開花日と満開日 生物季節観測の情報] 気象庁</ref> 2011-2013詳細的各地櫻花標本木種類列表大致同前，惟根室是以千島櫻為觀測對象。<ref> [http://www.data.jma.go.jp/sakura/data/sakura004_03.html さくらの満開日(2011-2013年) 生物季節観測の情報] 気象庁</ref>

== 花期 ==
=== 日本櫻花花期 ===
日本的櫻花多數在3月下旬至四月上旬開花，但近年因[[全球暖化|全球暖化]]的影響，令櫻花開放的時間有所提前。而且太平洋的氣候變暖，亦導致花開後被風吹至散落。大大縮短了民眾欣賞櫻花的時間。櫻花季節不長。通常在初開（開花）後一星期便會盛開（滿開）。再過一星期，盛開尖峰就過了，花瓣就開始凋零。如有強風和下雨也會使得花期縮短。

日本每年都會有「櫻花前線」，即預報各地開花的時間。

日本獨立行政法人森林研究所多摩森林科學園櫻花保存林自1981年起即對所蒐集的櫻花開花狀況進行觀測。並有各品種30年平均開花日期等資料公布。
<ref>[http://www.ffpri-tmk.affrc.go.jp/cherry/bloom/bloom_main.html サクラの開花状況はこの観測結果]</ref>
<ref>[http://agriknowledge.affrc.go.jp/RN/2010813972.pdf 多摩森林科学園サクラ保存林における30年間のサクラの開花期観測] </ref>

== 食用 ==

樱的部分种类果实味美，通称[[櫻|樱桃]]，但观赏用樱花的果实通常不适合使用，而有些[[雜交|雜交]]品种是不結果的。因此，很多人以為櫻花和櫻桃一定是不同的物種。实际上有些物种既有观花品种（樱花），又有食用品种（樱桃）。比如[[中国樱桃|中国樱桃]]传统上作为果树栽培，但亦有观花品种，如泰山香樱（''Prunus pseudocerasus'' 'Taishan Xiang'）和剑桥樱（''Prunus pseudocerasus'' 'Cantabrigiensis'）。
