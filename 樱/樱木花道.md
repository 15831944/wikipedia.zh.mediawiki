'''櫻木花道'''是[[井上雄彥|井上雄彥]]的[[日本|日本]][[漫畫|漫畫]]及改編[[動畫|動畫]]《[[灌籃高手|灌籃高手]]》中的主角，湘北高中籃球隊的球員，擁有驚人的體能特質，經常以「天才」自稱。

球技進步後取代[[木暮公延|木暮公延]]，成為經常先發五人之一，在隊中的位置是[[大前鋒|大前鋒]]。

== 简介 ==
特徵是一頭紅髮，在就讀和光中學期間，三年來追求50个女生都被拒絕，最後一個拒絕他的女生是一個叫作島村葉子的女生，她表示自己喜歡的是籃球隊的小田，於是櫻木非常討厭籃球隊的人，聽到他人說籃球或讀音相近的字眼也會用頭撞擊對方。在进入高中后、他对[[赤木晴子|赤木晴子]]一見鍾情，为吸引她注意而进入篮球队。因为[[赤木晴子|赤木晴子]]暗恋[[流川枫|流川枫]]，所以他非常嫉妒流川枫；他與流川枫也经常争吵打架，不过他们各自心里还是佩服对方的球技。

故事一開始時是會犯下[[走步|走步]]的門外漢，但在短短的几个月裡成长为卓越的篮球手。他对感兴趣的东西学起来很快，擅长搶[[篮板球|篮板球]]、[[上籃|上籃]]和[[灌篮|灌篮]]，到後期也掌握了[[投篮|投篮]]。他有頂尖的速度和弹跳力，而且有着惊人的充沛体力，儘管經常會犯規，但常做出令人意想不到的举动，从而改变整个球队的气氛，甚至影響比賽的勝負，為湘北隊的靈魂人物。

櫻木第一場比賽是對陵南高中的練習賽中以後備身份出場。在經過短暫緊張後，他的潛力在該場比賽充分發揮。與對方兩米中鋒[[魚住純|魚住純]]抗衡、彈跳力、籃板球、瞬間的爆炸力、速度及運球能力等，讓對方教練[[田岡茂一|田岡茂一]]大跌眼鏡。完場前數秒射入反超前的入球更一度讓他高興不已，可是過早慶祝讓對方的王牌球員[[仙道彰|仙道彰]]有機可乘，再度超前了湘北並最終以一分之差險勝。這次失敗讓櫻木很不高興。

其後的大賽開始，櫻木一直以後備身份出賽，而且鮮有出場機會。即使可以上場比賽，亦以驚人方式極速[[五犯離場|五犯離場]]。後來吸收了多次失敗經驗，終於在對翔陽高中一戰成名。自此以後，櫻木漸漸成長為湘北不可或缺的籃板及內線球員。

櫻木花道從未學習如何投[[罰球|罰球]]時，頭一次站上[[罰球線|罰球線]]時還因為不知所措而第一投持球超時，之後用｢扔｣的出去。而後在幾次看了其他人的[[罰球|罰球]](如[[藤真健司|藤真健司]]、[[三井壽|三井壽]]、[[赤木剛憲|赤木剛憲]]、[[牧紳一|牧紳一]]；[[仙道彰|仙道彰]]等)然後自己學習改良，變成用｢拋｣的(與NBA金州勇士名人堂球星[[里克·巴里|里克·巴里]]一樣)，但[[命中率|命中率]]奇高，對海南隊時下半場兩次[[罰球|罰球]]三球全進，彈無虛發；對豐玉時四罰中三，對山王時六罰全進。

由於身材高大且天生神力，打架的戰績非常優異，在國中時是學校裡讓人害怕的小混混，這種傾向似乎延續到高中時期。就讀於和光國中時，跟好友水戶洋平、高宮望、大楠雄二及野間忠一郎組成幹架小團體“[[櫻木軍團|櫻木軍團]]”。水戶等人雖然有時會戲弄櫻木，但危急時常毫不猶豫地維護櫻木。在學業成績不佳的湘北先發球員陣容中（除[[赤木剛憲|赤木剛憲]]外），是成績最差的。

== 轉變 ==
湘北在地區預選賽的四強賽對上海南大附屬高中；在比賽時間結束之前，由於櫻木的傳球失誤，錯失了追平並進入延長戰線的機會，讓湘北最後一刻惜敗給海南。從此櫻木便[[剃頭|剃頭]]雪恥。

二次面對陵南高中時，一開始就展現出近距離射籃能力。雖說中間因為受傷下場休息，但在下半場，赤木因已有四次犯規在身對魚住的進攻有所疑慮時，櫻木在赤木身後硬是擋住了魚住的射籃。並在最後只領先兩分的情形下搶下赤木未能投進的籃板球並演出再見灌籃，將分數拉開成領先四分的保險局面。

地區預選賽之後，原本只會上籃與灌籃的櫻木，在球隊前往靜岡與常誠高中進行練習賽期間，留在學校體育館，接受安西教練兩萬球射籃集訓，趕在全國大賽前增進實力。

== 全国大赛 ==
樱木花道已学会投篮技巧，在第一场全国大赛投篮得分。令许多人赞叹不已，最后得到胜利。第二场全国大赛对垒到三连霸全国冠军山王，虽然上半场取得一分领先，但是山王使出全力后，湘北因落後24分(36:60)而开始绝望。但是只有樱木还没放弃，他站在记分台上高喊“我要打败山王”，虽然这个举动被许多人嘲笑，可是樱木花道成功地把湘北的气势鼓起。除此之外，樱木花道也以惊人的篮板能力挽救了湘北。但是在关键时刻时，樱木为了救一颗快要出界的球而伤到脊椎。安西教练不再让他上场，但是他的斗志却没熄灭。最后他对安西教练说了一句话“如果你的最辉煌的时候是全国大赛还是国家队……那么对于我就是现在了！”所以安西让樱木上场，虽然身负重伤可是还坚韧不拔地盖了‘人肉丸’(河田美紀男)和澤北大火锅。在不到2秒和一分之差时，說出一句「'''左手只是輔助'''」<ref>此句在尖端完成版中改譯為「左手扶著球就好」，雖然較為貼近原意，但為多數中文讀者所不喜。</ref>後，[[流川枫|流川枫]]传球给樱木花道，樱木花道以优美的投篮姿势投进了在最后一秒的絕殺球，兩人前所未有的默契，帮助湘北以79:78反败为胜。也成为《[[灌籃高手|灌籃高手]]》的經典台詞。

== 原型 ==
櫻木花道原型是井上雄彥在1990年發表作品喜歡紅色(赤が好き)的男主角（姓名亦爲櫻木花道）。喜歡紅色(赤が好き)發表後，[[灌籃高手|灌籃高手]]開始連載。

因為櫻木的特色－一頭紅髮（本人自稱是天生的）、搞怪、擅長抓籃板球、跑步的姿勢及飛身救球，一般認為作者[[井上雄彥|井上雄彥]]是依「小蟲」[[丹尼斯·罗德曼|-{zh-hans:丹尼斯·罗德曼;zh-hk:丹尼士·洛文;zh-tw:丹尼斯·羅德曼;}-]]為設計腳本。另外從下往上拋投的罰球姿勢則是以[[瑞克·貝瑞|瑞克·貝瑞]]為參考。
