'''樱乃空'''（暂译，原名：{{lang|ja|桜乃そら}}）是AH-Software推出的一套[[语音合成|语音合成]]技术[[库_(计算机)|资料库]]及附属的[[虚构角色|虚构角色]]，以日本[[声优|声优]][[井上喜久子|井上喜久子]]的声音为基础制作，由使用歌声合成技术[[Vocaloid|Vocaloid]]的《VOCALOID 樱乃空》和使用语音合成技术AITalk的[[Voiceroid|Voiceroid]]内声库《VOICEROID2 樱乃空》组成，并在2018年7月26日发售。<ref>{{Cite news |url = http://nlab.itmedia.co.jp/nl/articles/1807/12/news120.html|title = 本当に“永遠”の存在へ……　井上喜久子の声を持つボカロ「桜乃そら」発売決定！|author = 深戸進路|date = 2018-7-12|newspaper = itmedia|language = ja}}</ref><ref>{{cite web |title=永遠の17歳・井上喜久子が「ボーカロイド 桜乃そら」"17歳"を担当 |url=https://www.zaikei.co.jp/article/20180716/453969.html |website=財経新聞 |accessdate=2018-08-12}}</ref>

== Vocaloid ==
《VOCALOID 樱乃空》以Vocaloid 5引擎为基础开发，包含了两款不同声音的歌声资料库：温柔的“Natural”和具有力量的“Cool”，据官方介绍，“Natural”适合演唱[[谣曲|谣曲]]或是中速节奏的流行歌曲，“Cool”适合演唱[[演歌|演歌]]或是快节奏的[[摇滚乐|摇滚]]。
