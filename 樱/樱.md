'''樱'''，是某些[[李属|李属]][[植物|植物]]的統稱，包括典型櫻亞屬（''Cerasus''）和矮生櫻亞屬（''Lithocerasus''；異名：''Microcerasus''）。

==名称与用途==
{{see also|櫻花}}
有些種類的櫻作觀賞植物栽培，在[[日本|日本]]尤為盛行。觀賞品種的花稱為'''[[櫻花|櫻花]]'''，其果實一般味道欠佳而不作食用，有些[[雜交|雜交]]品種甚至不結實。

有些種類的果實可食用或製作果脯。食用品種的果實通稱'''櫻桃'''，但櫻桃也可特指[[中國櫻桃|中國櫻桃]]。櫻桃果樹的花也有人叫'''櫻桃花'''，以別於觀賞用櫻花。

有些种类既有观花品种（樱花），又有食用品种（樱桃）。比如[[中国樱桃|中国樱桃]]传统上作为果树栽培，但亦有观花品种，如泰山香樱（''Prunus pseudocerasus'' 'Taishan Xiang'）和剑桥樱（''Prunus pseudocerasus'' 'Cantabrigiensis'）。

世界上大多數地區水果市場銷售的櫻桃為[[歐洲甜櫻桃|歐洲甜櫻桃]]（''Prunus avium''），這種外銷的櫻桃別名[[車厘子|車厘子]]（英語cherries之音譯）。[[歐洲酸櫻桃|歐洲酸櫻桃]]（''Prunus cerasus''）栽培也較廣，但不用於直接食用，一般加工成罐頭、飲料、甜點等。中國大陸地區除了歐洲甜櫻桃，還廣泛種植[[中國櫻桃|中國櫻桃]]（''Prunus pseudocerasus''）。

櫻桃容易因碰撞受損，挑選上要注意，綠梗較新鮮，色澤深，鮮紅、暗紅色甜度高。

== 分佈 ==
[[File:CherryYield.png|thumb]]

櫻桃主要分布於[[歐洲|歐洲]]、[[澳洲|澳洲]]、[[美國|美國]]以及[[東亞|東亞]]地區。

每年4月中旬是樱桃成熟的季节。樱桃外形娇小，吃起来口感甜中帶微酸、果肉滋味纯美。其花、叶、果實也可加工制做为腌菜而食用或酿造为果酒。

[[中國大陸|中國大陸]][[贵州省|贵州省]][[安顺|安顺]][[镇宁县|镇宁县]]，[[辽宁省|辽宁省]][[大连市|大连市]]以及山东省[[烟台市|烟台市]]均盛产此种水果。

== 食用价值 ==
[[File:Italienische_Süßkirschen.JPG|thumb]]
[[中医|中医]]古籍称它能“滋润皮肤”、“令人好颜色，美态”<ref>《[[名医別錄|名医別錄]]》中有记载：“吃樱桃，令人好颜色，美志。”《[[備急千金方|備急千金方]]》上說︰“櫻桃味甘、平、澀，調中益氣……令人好顏色。”</ref>，常吃能够让皮肤更加光滑润泽。樱桃含鐵量相當低。每一百公克櫻桃含鐵量只有 0.2463毫克。

樱桃核含有[[氰甙|氰甙]]，若大量誤食可能引起[[氰化物|氰化物]]中毒，寻常食用不必担心。<ref>https://new.qq.com/omn/20180409/20180409A0HRPN.html</ref>

== 文化 ==
* [[后梁宣帝|后梁宣帝]]有《樱桃赋》：“懿夫樱桃之为树，先百果而含荣，既离离而春就，乍苒苒而东迎。”
* [[白居易|白居易]]有《吴樱桃》：“含桃最说出东吴，香色鲜农气味殊。洽恰举头千万颗，婆娑拂面两三株。鸟偷飞处衔将火，人争摘时踏破珠。可惜风吹兼雨打，明朝后日即应无。”
