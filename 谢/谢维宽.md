'''谢维宽'''（{{bd|1962年|||}}）籍贯不详，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
谢维宽曾任[[中国人民解放军总后勤部|中国人民解放军总后勤部]]司令部战勤计划局局长<ref>{{Cite web |url =  http://www.gov.cn/wszb/zhibo244/content_1013656.htm|title = 总后勤部司令部战勤计划局局长谢维宽回答记者提问 |author =  |date = 2008-06-11 |publisher = 中国政府网 |language =  |accessdate =  |quote =  }}</ref>。2011年7月任中国人民解放军总后勤部司令部副军职专员。2012年12月任中国人民解放军总后勤部司令部副参谋长<ref>{{Cite web |url = http://www.chinahuaxicun.com/content/2014-12/05/content_265160.htm |title =  总后勤部司令部副参谋长谢维宽盛赞华西|author =  |date =  2014-12-05|publisher =  中国华西|language =  |accessdate =  |quote =  }}</ref>。2015年11月任[[中国人民解放军后勤学院|中国人民解放军后勤学院]]院长<ref>{{Cite web |url = http://mil.sohu.com/20151117/n426754267.shtml |title = 解放军后勤学院年内三次换帅 谢维宽任院长 |author =  |date = 2015-11-17 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。2017年任[[中国人民解放军国防大学|中国人民解放军国防大学]]副校长<ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1734190|title =谢维宽少将任国防大学副校长，此前担任解放军后勤学院院长  |author =  |date = 2017-07-16 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

2012年7月晋升少将军衔。
