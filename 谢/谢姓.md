'''谢姓'''是[[漢族姓氏|漢族姓氏]]之一，姓谢人口约占[[中國大陆|中國大陆]]總人口的0.79%，在中國姓氏人数中排第23位。谢姓人士主要分布在[[中國|中國]]的[[江蘇|江蘇]]、[[浙江|浙江]]、[[江西|江西]]、[[河南|河南]]、[[四川|四川]]、[[福建|福建]]、[[廣東|廣東]]、[[廣西|廣西]]、[[海南|海南]]等各省。以及[[台湾|台湾]]、[[马来西亚|马来西亚]]、[[香港|香港]]、和[[東南亞|東南亞]]等國家與地區。

[[唐朝|唐朝]]詩人[[劉禹錫|劉禹錫]]有詩稱：「舊時'''王謝'''堂前燕，飛入尋常百姓家。」诗中“谢”系指[[东晋|东晋]]望族[[陈郡谢氏|陈郡谢氏]]。[[明朝|明朝]][[朱元璋|朱元璋]]曾對遠封的諸王說：「朕為[[天子|天子]]自有[[弓馬|弓馬]]，豈賴[[陳郡謝氏|謝家]]！」<ref>[[陸龍其|陸龍其]]《書[[傅維鱗|傅維鱗]][[明書|明書]]後》</ref>就是以[[晉朝|晉朝]]仰賴[[陳郡謝氏|謝家]]子弟為例，訓勉諸王自己努力防禦外敵。

== 姓氏起源 ==
谢姓源流有多种，以[[炎帝|炎帝]][[姜姓|姜姓]]後裔為多，稱為「正宗」。

;炎帝後裔（主流）
*谢氏为[[炎帝|炎帝]][[神農氏|神農氏]]之后代，即姜姓谢氏。
**[[周宣王|周宣王]]命[[召穆公|召穆公]]滅[[謝國|古謝國]]，封舅父[[申伯|申伯]]于[[謝國|古謝國]]領土（今[[河南|河南]][[南陽|南陽]][[唐河县|唐河]]一帶），稱為「[[申國|申國]]」，又稱「謝國」。後子孫以「謝」為氏。[[魏晉|魏晉]]時，[[謝纘|謝纘]]遷居[[陳郡|陳郡]]陽夏（今[[河南|河南]][[太康县|太康]]），號稱「[[陳郡謝氏|陳郡謝氏]]」。

;黄帝後裔
*谢氏为[[黄帝|黄帝]][[軒轅氏|軒轅氏]]之后代。
**一支出自[[任姓|任姓]]，相传[[黄帝|黄帝]]之子廿五人，得姓者十四人，为十二姓，其中「禹陽」为任姓。任姓後人分居十个小国，其中第一为[[谢国|谢国]]，後人改[[謝氏|謝氏]]。<ref>《世本·氏姓篇》：谢，任姓，黄帝之后。</ref><ref>《古今姓氏书辩证》：谢氏，出自黄帝之后，任姓之别为十族，谢其一也。其国在河南南阳宛县。</ref>
**出自[[姬姓|姬姓]]，[[周朝|周朝]][[宗室|宗室]]居於謝國者，以國為氏。
**出自[[子姓|子姓]]，[[商朝|商朝]][[宗室|宗室]]居於謝國者，以國為氏。

;少數民族後裔
*为他姓改谢姓。
**如卫州（今河南[[衛輝|衛輝]]）人[[谢偃|谢偃]]之本为[[鲜卑族|鲜卑族]]人，姓[[直勒氏|直勒氏]]，后改为谢氏，稱「衛州谢氏」，後人遷居[[洛陽|洛陽]]，亦稱为「洛陽谢氏」、「河南谢氏」。<ref>《[[旧唐书|旧唐书]]文苑传》</ref>
**另有[[苗族|苗族]]、[[回族|回族]]等[[少數民族|少數民族]][[漢化|漢化]]改謝姓的。

== 發展變遷 ==
;東晉
[[魏晉|魏晉]]遷居[[華南|華南]]的[[陳郡謝氏|陳郡謝氏]]，一部份輾轉由[[會稽|會稽]]來到了[[閩南|閩南]][[泉州|泉州]]、[[漳州|漳州]]，最後移居[[台灣|台灣]]。

;唐、五代
[[唐僖宗|唐僖宗]]光啟元年（885年），[[光州|光州]][[固始|固始]]軍人[[王潮|王潮]]、[[王審知|王審知]]兄弟率卅四姓[[世族|世族]]、軍民入閩，固始縣令謝之郡等謝姓人士即在其中。<ref>《台灣省通志》</ref>

安溪尤俊的謝家是「固始縣令謝之郡，與王審知、吳祭等入閩，經[[漳州|漳州]]抵泉州，並定居泉州。安溪厚安的謝家也是「自[[光州|光州]][[固始|固始]]，以五季梁開平間從[[王審知|王審知]]入閩，始遷泉州之屬安溪永安里東皋居焉。」

在[[泉州|泉州]]拓墾的謝氏，是著名望族，相當顯赫，在[[宋仁宗|宋仁宗]][[皇祐|皇祐]]元年（1049年），福建轉運使謝仲規曾被賜與紫金魚帶，人稱其家族為「金魚謝氏」或「金魚世第」。安溪等地謝氏不少人為「金魚謝氏」的後裔，後來一部分人遷居[[台灣|台灣]]、[[南洋|南洋]]等地。

;宋元以後
[[宋朝|宋朝]]忠臣[[謝枋得|謝枋得]]有子謝寄孝，其後人進入[[泉州|泉州]]、[[漳州|漳州]]等地拓墾，後來也逐漸移往海外、台灣等地。

宋、元以後，也有不少[[江西|江西]]謝姓[[客家人|客家人]]進入[[閩西|閩西]]、[[粵東|粵東]]拓墾，後來移居台灣。

[[元朝|元朝]]末年[[諸侯|諸侯]]，漢王[[陳友諒|陳友諒]]，其祖父千一本為謝姓，因[[入赘|入赘]]陈家而改姓[[陳氏|陳]]，[[明太祖|明太祖]][[鄱陽湖之戰|鄱陽湖之戰]]討伐友諒，友諒敗死後，一些同宗人士改回謝姓，以求免禍。

== 谢姓字輩排行 ==
;福建
*福建闽侯谢氏字辈：「喜添元世一文廷国君兆仲仕日向。」
*福建宁化谢氏字辈：「文得子宗宇乔宸祥宪朝显世贤逢良士应时起荣华望日崇。」
*福建惠安谢氏字辈：「昆肇文思仕居，建承宗定士邦，英谨希顺志，兴发庆友双。和平济世，忠孝传家。」
*福建院內谢氏字辈：「立德以希聖賢，睦族必由仁讓，子孫訓其成式，孝友乃振家聲。」
**《院內，宋属安仁乡永宁里，元、明、清属二十一都，古稱晉江縣溪前，現稱石獅市蚶江鎮溪前村(碧溪)院內》
*福建安溪谢氏字辈：「天上玉京，星中長庚，海乃神龍，山稱嶽衡，國廈棟擎，世代令名，文居金榜，武為干城，晉朝寶樹，陳郡家聲，一人元良，萬邦以貞。」
*福建漳浦谢氏字辈：「恩庆为良才，东山久万锦，文锡世道永，汝时孔敏达，宝树成瑞连，江左声名远，南邦德泽长，还基承继绍，百代盛流芳」

;廣東
*广东壶山公谢氏字辈：“桃練台苡明，公尔则子弘，殷有良承绍；邦伯卿夫士，格致诚正修；德礼昭平-{}-治，诗书合诒谋；傳家惟孝顺，弈世庆贤良。伯仲长进益，云仍养兴桢。光辉宜俊哲，肇武启群英”。

;安徽
*安徽太和谢氏字辈：“东晋兴玉树，芝兰世泽长；祖德恒保守，家道自丕昌；宗荣延繁茂，贤良治国邦。”

;湖北
据《谢氏宗谱》记载，公元1751年，以熚炜公为首，联吴楚各属百脉，在湖北省嘉鱼县桂花树村清风楼六修谢氏谱局订颁的大成谢氏派序，即此谱：「同宗继祖盛南邦，从守先模作俊良。中正懋修全秉绶，亮功班效佐朝章，恒培芳树堂阶映，高振宏义海宇扬。开定徽言登会纪，贤祠欣起卜延长。」

;台灣
*[[臺灣|臺灣]][[新竹縣|新竹縣]]十一世祖貴明公派下<ref>{{Cite web |url = http://www.xies5.com/info/1502.html |title = 陸河縣謝氏家族各地（村）啟用輩字 |author = 谢铖 |date = 2012-11-29 |publisher = 謝氏網 |language =  |accessdate = 2015-01-18 |quote = 昭穆：14世至41世輩字，14世“添”字啟用輩序，屬本榮六乙公二世遠公派系 }}</ref>：「添<ref>注：對照[[廣東省|廣東省]][[陸河縣|陸河縣]]昂湖謝氏宗祠輩序“洪”字輩：即是臺灣新竹縣貴明公後裔十四世“添”字輩：[[陸豐縣|陸豐縣]][[三溪鄉|三溪鄉]]南丫甲石碣屋村遷台十一世祖貴明公，即是“錫”字輩。</ref>永金維建紹光，元清興文富福昌，賢明隆盛鴻振瑞，慶正聖仁德安康<ref>[http://www.guiming11.com/yan-ge 新竹縣貴明公謝姓宗親會－沿革]</ref>」
