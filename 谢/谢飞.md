'''谢飞'''（{{bd|1942年|8月14日||catIdx=Xie,谢}}），出生于[[陕西|陕西]][[延安|延安]]，籍贯[[湖南省|湖南省]][[长沙市|长沙市]][[宁乡县|宁乡县]]，[[中国|中国]][[导演|导演]]。<ref>{{cite web |url=http://people.mtime.com/892969/ |title=谢飞 |accessdate=2012年12月16日 |publisher=时光网 |archiveurl=http://people.mtime.com/892969/ |archivedate=2012年12月16日 |work= |language =zh-cn}}</ref>曾就读于[[北京四中|北京四中]]，凭借《香魂女》获得1993年德国柏林电影节金熊奖。

== 生平 ==
1942年，谢飞出生在[[陕西|陕西]][[延安|延安]]。是[[谢觉哉|谢觉哉]]和[[王定国|王定国]]的第三个孩子<ref>{{Cite web |url=http://news.ifeng.com/a/20150810/44392413_0.shtml |title=谢觉哉家书透露元配夫人可叹命运|accessdate=2016-09-11 |author=作者：孙莹 |date=2015-08-10 |work= |publisher=凤凰网|language=Zh-hans }}</ref>。

1961年考入北京电影学院导演系，毕业后留在学校里执教，曾当过导演系的主任。<ref name="fish " />1985年，谢飞担任北京电影学院副院长、中国影协第五届理事。<ref name="fish ">{{cite web|url=http://ent.qq.com/a/20121216/000038.htm?pgv_ref=aio2012&ptlang=2052|title=谢飞公开信炮轰电影审查制 中国导演协会似支持|accessdate=2012-12-16|publisher=腾讯网|archiveurl=https://web.archive.org/web/20121219224229/http://ent.qq.com/a/20121216/000038.htm?pgv_ref=aio2012&ptlang=2052|archivedate=2012年12月19日|work=|language=zh-cn|deadurl=yes}}</ref>2000年，由于当局对电影行业的严密管控，谢飞表示要改行当电视剧导演。<ref name="fish " />2012年12月，谢飞在微博发表公开信《呼吁以电影分级制代替电影审查的公开信》，呼吁当局改革电影审查制度。<ref name="fish " />
