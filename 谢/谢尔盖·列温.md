'''谢尔盖·尼古拉耶维奇·列温'''（{{lang-ru|Серге́й Никола́евич Ре́вин}}，{{bd|1966年|1月12日|}}）是[[俄罗斯|俄罗斯]]已退役宇航员。
==早年==
出生于[[莫斯科|莫斯科]]。1989年毕业于莫斯科电子工程学院（现{{tsl|en|National Research University of Electronic Technology|莫斯科电子技术国家研究型大学}}）<ref name=nasa>{{cite web |title=Biographical Data |url=https://web.archive.org/web/20180517040845/https://www.jsc.nasa.gov/Bios/htmlbios/revin.html |publisher=[[NASA|NASA]] |accessdate=2019-04-13 |language=en |date=2012}}</ref>。是一名电子自动化工程师，自1998年起成为俄航天署的航天试验员<ref name=qq>{{cite web |author=（[[新华网|新华网]]）贺颖骏 |title=今年首批3名宇航员搭载俄飞船赴国际空间站 |url=http://tech.qq.com/a/20120516/000216.htm |publisher=[[腾讯|腾讯]]科技 |archiveurl=https://web.archive.org/web/20190413145421/http://tech.qq.com/a/20120516/000216.htm |archivedate=2019-04-13 |date=2012-05-16}}</ref>。
==远征31/32==
莫斯科时间2012年5月15日7时（北京时间15日11时），他和[[根纳季·帕达尔卡|根纳季·帕达尔卡]]、美国宇航员[[约瑟夫·M·阿卡巴|约瑟夫·M·阿卡巴]]乘坐[[联盟TMA-04M|联盟TMA-04M]]载人飞船，在哈萨克斯坦的[[拜科努尔发射场|拜科努尔发射场]]升空前往[[国际空间站|国际空间站]]。这是2012年首批飞往空间站的宇航员<ref name=qq/>。

哈萨克斯坦时间2012年9月17日，他和帕达尔卡以及阿卡巴乘坐联盟TMA-04M飞船在[[哈萨克斯坦|哈萨克斯坦]][[阿尔卡雷克|阿尔卡雷克]]地区成功着陆<ref>{{cite web |title=俄罗斯“联盟”号载三名宇航员平安返回 |url=http://world.huanqiu.com/photo/2012-09/2665793.html |publisher=[[环球网|环球网]] |archiveurl=https://web.archive.org/web/20121105152709/http://world.huanqiu.com/photo/2012-09/2665793.html |archivedate=2019-04-13 |date=2012-09-18}}</ref>。
==个人生活==
他热爱[[旅游|旅游]]和[[气球_(航空器)|气球]]飞行<ref name=nasa/>。

他是[[俄罗斯生态党“绿党”|俄罗斯生态党“绿党”]]党员<ref>{{cite web |url=http://greenparty.ru/news/1564/|title=Хроники космонавта Сергея Ревина|publisher=Сайт РЭП  «Зелёные»|accessdate=2018-03-19 |lang=}}</ref>。2018年被提名为莫斯科市长候选人<ref>{{Cite web|url=https://ria.ru/politics/20180405/1518017623.html|title="Зеленые" определились с кандидатурой на выборах мэра Москвы|website=РИА Новости|language=ru|access-date=2018-04-06}}</ref>。
