'''謝蕾蕾'''（{{bd|1998年|12月1日|catIdx=Xie, LeiLei}}），中國大型女子偶像團體[[GNZ48|GNZ48]] Team G的成員。她於2015年7月25日的[[SNH48第二屆總選舉|SNH48第二屆總選舉]]中披露，成為SNH48的5期生。2015年12月4日，於[[Team_B_5th_Stage「劇場的女神」#SNH48_Team_XII「劇場女神」公演|SNH48 Team XII「劇場女神」公演]]正式出道，加入SNH48 Team XII<ref name="sina1">{{Cite web |url = http://ent.sina.com.cn/y/yneidi/2015-12-07/doc-ifxmhqac0105704.shtml |title = SNH48五期生宣布出道 4日上海献公演首秀 |date = 2015-12-07 |accessdate = 2016-01-15 |publisher = 新浪娛樂 }}</ref>。2016年4月20日，移籍GNZ48 Team G。2016年7月30日获得[[SNH48第三屆總選舉|SNH48第三屆總決選]]第四十名，并获得GNZ48 TOP7第一名。2017年7月29日获得[[SNH48_Group第四屆總選舉|SNH48 Group第四屆總決選]]第三十二名，并获得GNZ48 TOP16第二名。2018年7月28日获得[[SNH48_Group第五屆總選舉|SNH48 Group第五屆總決選]]第十一名，并获得GNZ48 TOP16第一名。

== 經歷 ==
2015年7月25日，在[[SNH48第二屆總選舉|SNH48第二屆總選舉]]中披露，成為SNH48的5期生。12月4日，於[[Team_B_5th_Stage「劇場的女神」#SNH48_Team_XII「劇場女神」公演|SNH48 Team XII「劇場女神」公演]]出道<ref name="sina1" /><ref name="wmxa">{{Cite news|url = http://www.wmxa.cn/a/201512/291903_2.html |title = snh48五期生上海公演正式出道 完整成员名单流出颜值爆表 |date = 2015-12-08 |accessdate = 2016-01-15 |publisher = 西安文明網 |deadurl = yes |archiveurl = https://web.archive.org/web/20160512193236/http://www.wmxa.cn/a/201512/291903_2.html |archivedate = 2016-05-12 }}</ref>。12月26日 ，參與了「[[SNH48第二屆年度金曲大賞BEST_30|SNH48第二屆年度金曲大賞BEST 30]]」。

2016年4月20日，移籍至[[GNZ48|GNZ48]]，并加入GNZ48 Team G<ref>{{cite news|url=http://www.chinayule.cn/star/20160420/31362.html|title=SNH48公布北京、广州姐妹团，开启偶像生态产业新纪元|date=2016-04-20|website=娱乐中国}}</ref>。4月29日，以GNZ48成員名義於[[Team_B_5th_Stage「劇場的女神」#GNZ48_Team_G_1st_Stage「劇場女神」公演|GNZ48 Team G 1st Stage「劇場女神」公演]]出道<ref>{{Cite news |url=http://m2.21cn.com/news/china/a/2016/0429/22/30981774.shtml |title=SNH48姐妹团GNZ48正式出道 首演人气爆棚 |access-date=2018-09-09 |archive-url=https://web.archive.org/web/20170731192814/http://m2.21cn.com/news/china/a/2016/0429/22/30981774.shtml |archive-date=2017-07-31 |dead-url=yes }}</ref>。5月20日，入選《[[夢想島|夢想島]]》選拔組，也是謝蕾蕾首次參與SNH48選拔。7月30日，於[[SNH48第三屆總選舉|SNH48第三屆總決選]]，最終獲得13,550.0票，獲得第40名，入選第三屆總決選Next Girls<ref>{{cite web|url=http://www.snh48.com/html/allnews/zixun/2016/0730/1873.html|title=“比翼齐飞”SNH48第三届偶像年度人气总决选最终结果公布|website=SNH48官方网站|date=2016-07-30}}</ref>。而謝蕾蕾于GNZ48 TOP7中排名第1名<ref>{{cite web|url=http://www.gnz48.com/event/g007/|title=“比翼齐飞”SNH48第三届偶像年度人气总决选GNZ48 TOP7成员|website=GNZ48官方网站|date=2016-07-31}}</ref>。8月26日，參與了「『愛音樂 翼起來』GNZ48夢想起航 Show Case 演唱會」<ref>{{cite web|url=http://www.gnz48.com/html/news/info/201608/199.html|title=“爱音乐 翼起来”GNZ48梦想启航Showcase顺利落幕|date=2016-08-27|website=GNZ48官方网站}}</ref>。9月16日，出席BiggerFun乐活狂欢节。11月11日，代言遊戲《穿越火线》<ref>{{cite web|url=http://cf.qq.com/webplat/info/news_version3/125/860/861/m640/201611/519605.shtml|title=火线11.11不落单，穿越火线陪你战！GNZ48枪娘直播票选开启|website=[[穿越火线|穿越火线]]|publisher=[[腾讯网|腾讯网]]|date=2016-11-03}}</ref>。

2017年1月7日，參與了「[[SNH48第三屆年度金曲大賞BEST_50|SNH48第三屆年度金曲大賞BEST 50]]」。4月13日，於GNZ48的第3張EP《I.F》的選拔組曲目《蒲公英的腳印》中落選選拔組Center，也是謝蕾蕾首次落選選拔組Center。4月29日，當選GNZ48星夢劇院2016年年度MVP<ref>{{cite web|url=http://www.gnz48.com/html/news/activity/201704/327.html|title=GNZ48一周年纪念日回顾|date=2017-04-28|website=GNZ48官方网站}}</ref>。6月10日，出席SNH48代言產品JoyDay脆酸奶[[深圳|深圳]]發佈會。7月10日，出席由絲芭影視出品的古裝劇《[[芸汐传|芸汐传]]》开机仪式<ref>{{cite news|url=http://ent.qq.com/a/20170711/024190.htm|title=《芸汐传》今日横店开机 创新 “轻古风”来袭|website=腾讯娱乐|publisher=[[腾讯网|腾讯网]]|date=2017-07-11|access-date=2019-01-07}}</ref>。7月29日，於[[SNH48_Group第四屆總選舉|SNH48 Group第四屆總決選]]，最終獲得23927.8票，排名第32名，入選第四屆總決選Under Girls<ref>{{cite web|url=http://www.snh48.com/html/allnews/zixun/2017/0729/2453.html|title=“我心翱翔”SNH48GROUP第四届总决选最终结果公布 鞠婧祎连霸|date=2017-07-29|website=SNH48官方网站}}</ref><ref>{{cite web|url=http://www.snh48.com/html/allnews/zixun/2017/0730/2456.html|title=SNH48 GROUP第四届总决选落幕 鞠婧祎问鼎年度人气王|date=2017-07-30|website=SNH48官方网站}}</ref>，並且在GNZ48 TOP16中排名第2名<ref>{{cite web|url=http://www.gnz48.com/html/news/info/201707/343.html|title=“我心翱翔”SNH48 GROUP第四届偶像年度人气总决选发布演唱会回顾|date=2017-07-30|website=GNZ48官方网站}}</ref>。

2018年2月3日，參與了「[[SNH48_Group第四屆年度金曲大賞BEST_50|SNH48 Group第四屆年度金曲大賞BEST 50]]」<ref>{{cite web|title=SNH48 首次大重组各队成员名单公布 分层级打造彰显新格局|url=http://www.snh48.com/html/allnews/zixun/2018/0203/2883.html|date=2018-02-03|website=SNH48官方网站}}</ref>。4月29日，當選GNZ48星夢劇院2017年年度MVP<ref>{{cite web|url=http://www.gnz48.com/html/news/info/201805/369.html|title=GNZ48出道两周年庆典 与粉丝朋友们一起展望未来|date=2018-05-02|website=GNZ48官方网站}}</ref>。5月8日，其客串的网络电影《狄大人驾到》于[[腾讯视频|腾讯视频]]上线，此处客串为谢蕾蕾拍摄《[[芸汐传|芸汐传]]》期间探班《狄大人驾到》剧组时，导演突发加的一段戏。6月25日，其出演的《[[芸汐传|芸汐传]]》启播<ref>{{cite news|url=http://ent.ifeng.com/a/20180619/43058997_0.shtml|title=《芸汐传》定档6.25 鞠婧祎演绎古风甜爽爱恋|website=凤凰网娱乐|publisher=[[凤凰网|凤凰网]]}}</ref>。7月20日，主演舞台剧《双面偶像：又又》<ref>{{cite news|url=http://epaper.xkb.com.cn/view/1115461|title=女性成长及自我救赎的故事感动观众|author=徐绍娜|newspaper=《[[新快报|新快报]]》|pages=A16|date=2018-07-23|access-date=2018-12-02}}</ref>。7月28日，於[[SNH48_Group第五屆總選舉|SNH48 Group第五屆總決選]]，最終獲得61115.92票，獲得第11名，入選第五屆總決選選拔組，取得GNZ48第一及分团第一的成绩，刷新[[SNH48|SNH48 Group]]总决选分团历史最高排名和票数<ref>{{cite web|url=http://www.snh48.com/html/allnews/zixun/2018/0729/3362.html|title=SNH48 GROUP第五届偶像年度人气总决选收官 李艺彤斩获第一|date=2018-07-29|website=SNH48官方网站}}</ref>。10月6日，参与了“GNZ48 TOP16 PVC姐妹情深深圳巡演”<ref>{{cite web|url=http://www.gnz48.com/html/news/info/201810/387.html|title=星光闪耀 伴你同行丨GNZ48 TOP16 PVC姐妹情深 深圳巡演回顾|date=2018-10-07|website=GNZ48官方网站|access-date=2019-01-07}}</ref>。10月10日，出席由丝芭影视出品的电视剧《[[小夜曲_(电视剧)|小夜曲]]》开机仪式<ref>{{cite news|url=http://www.dzwww.com/yule/yulezhuanti/mtcbg/201810/t20181011_11358611.htm|title=电视剧《小夜曲》开机 陈学冬黄婷婷演绎“三明治青年”|work=红网|publisher=[[大众日报|大众网]]|date=2017-07-11|access-date=2019-01-07}}</ref>。

== 人物 ==
* 个人特长是古典民族舞、抓娃娃、发呆。<ref name="gnz48official" />
* 兴趣爱好是少女漫、宅、舞蹈。<ref name="gnz48official" />
* 因為謝蕾蕾移籍GNZ48時為该团人氣成員之一，并且最初为GNZ48的Ace和Center，故謝蕾蕾有「镇南王」一稱。
* 梦想是成为一名专业的演员。<ref name="djt_interview">{{cite news|author=冯丽均|coauthor=张耀文|title=“自己选的路哭着也要走完”|url=http://e.hznews.com/paper/djsb/20180726/A06A07/1/|date=2018-07-26|newspaper=《[[东江时报|东江时报]]》|pages=A06-A07|access-date=2018-12-05}}</ref>
* 长项是舞蹈，喜欢跳古典民族舞。<ref name="djt_interview"/>
* 因为有點傻和呆，被粉丝开玩笑称为「谢蠢蠢」。
* 劇場的自我介紹是「谢谢你，谢谢大家，还有别忘了'''谢谢蕾蕾'''」。
* 關係好的成員有[[段藝璇|段藝璇]]、胡曉慧、陈珂、高源婧、李沁洁、龙亦瑞等。
* 偶像使命感和责任心极强，曾经因于[[SNH48_Group第四屆總選舉|SNH48 Group第四屆總決選]]失落GNZ48第一名，而将自己的手机屏保换成第四屆總決選GNZ48的Top16排名表，时刻提醒自己要努力重夺GNZ48的第一名。<ref>{{Youku|XMzM4MjAxMDMxMg|三日为期：第六集 偶像与粉丝共栖的养成剧场}}</ref>
* 代表的发型是双马尾辫，被粉丝称为“蕾蕾头”。
* 喜欢的前辈成员是[[鞠婧祎|鞠婧祎]]、[[许佳琪|许佳琪]]、[[林思意|林思意]]、[[赵粤|赵粤]]等。
* 喜爱的食物包括客家酿三宝及酿辣椒。<ref name="djt_interview"/>
* GNZ48 Top3（'''谢蕾蕾'''、[[郑丹妮|郑丹妮]]、[[刘力菲|刘力菲]]）成员之一。
* 为SNH48 Group第五届总决选的分团第一名。
* 2018年在SNH48 Group第五届总决选发表感言时以“GNZ！呜呜呜！48！呜呜呜！谢谢大家……”开场，并大哭不止<ref>{{Youtube|HKELXrfiAdQ|SNH48GROUP第五届偶像年度人气总决选演唱会直播（第40名公布及之后部分）|h=1|s=12|language=zh-cn}}</ref>。此举引起粉丝取笑，并有粉丝将闲话歌曲《学喵叫》中的“我们一起学喵叫，一起喵喵喵喵喵”改成“我们一起学蕾叫，一起呜呜呜呜呜”作应援词使用；此外网络上还出现各种模仿及[[鬼畜|鬼畜]]视频。而其本人则以“小时候被爸妈打时才试过哭得这么惨”形容当日的哭态<ref>{{cite news|author=冯丽均|title=人气总决选谢蕾蕾分团排第一|url=http://e.hznews.com/paper/djsb/20180802/A08/2/|date=2018-08-02|newspaper=《[[东江时报|东江时报]]》|pages=A08|access-date=2019-03-29}}</ref>。
