'''谢尔盖·弗拉季连诺维奇·基里延科'''（[[俄语|俄语]]：'''{{lang|ru|Серге́й Владиле́нович Кирие́нко}}''' {{bd|1962年|7月26日|catIdx=К}}），[[俄罗斯|俄罗斯]]政治家，曾任[[俄罗斯总理|俄罗斯总理]]。

== 早年 ==
谢尔盖·基里延科生于[[苏联|苏联]][[喬治亞蘇維埃社會主義共和國|格鲁吉亚]][[黑海|黑海]]海滨城市[[苏呼米|苏呼米]]。1984年毕业于[[下诺夫哥罗德|高尔基]]水运工程学院经济系。1989年至1991年任共青团高尔基州委书记。

== 弃政经商 ==
1992年[[俄罗斯|俄罗斯]]开始经济改革之后，基里延科担任[[下诺夫戈罗德市|下诺夫戈罗德市]]“保障”社会商业银行行长。1996至1997年改任该市“诺尔西石油股份公司”经理。

== 政府总理 ==
1997年3月，基里延科被任命为俄罗斯燃料和动力部第一副部长，同年11月升任部长。1998年3月23日，[[叶利钦|叶利钦]]总统任命基里延科为第一副总理和代总理，几天后又提名他为政府新总理候选人。他是继盖达尔和切尔诺梅尔金之后俄罗斯政府的第三任总理，並以35歲之齡成為[[俄罗斯历史|俄罗斯历史]]上最年轻的[[总理|总理]]。
== 俄原子能署署长 ==
基里延科2000年5月18日出任俄原子能署署长，2005年11月14日去职。
