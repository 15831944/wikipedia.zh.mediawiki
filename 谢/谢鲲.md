'''谢鲲'''（{{bd|281年||323年|catIdx=X}}）<ref>卒年見於謝鯤墓志</ref>，[[表字|字]]'''幼舆'''，[[陈郡|陈郡]][[阳夏|阳夏]]（今[[河南|河南]][[太康|太康]]）人。出身[[陳郡謝氏|陳郡謝氏]]士族，東晉鎮西將軍[[謝尚|謝尚]]之父，[[太保|太保]][[谢安|谢安]]的伯父，[[两晋|两晋]]名士。官至豫章太守，故有稱'''謝豫章'''。

== 生平 ==
謝鯤年少知名。[[太安_(西晉)|太安]]元年（302年），長沙王[[司馬乂|司馬乂]]擊敗掌權的齊王[[司馬冏|司馬冏]]，以[[太尉|太尉]]、都督中外諸軍事的身份輔政掌權。時在洛陽的謝鯤被人中傷，聲稱他將迴避司馬乂而出奔。司馬乂於是就要鞭打他，但謝鯤竟主動脫衣受罰，沒有一絲不滿神情。獲司馬乂赦免後又不見喜悅。東海王[[司馬越|司馬越]]聽聞謝鯤的名聲，辟謝鯤為[[掾|掾]]，但不久就因家僮犯事而被牽連免職。事後司馬越又再辟他為掾，後轉[[參軍|參軍]]。但謝鯤見當時國家混亂，於是因病離職並到[[豫章郡|豫章郡]]避亂。

後來，時任左將軍的[[王敦|王敦]]任用他為[[長史|長史]]。[[建興_(西晉)|建興]]三年（315年），[[杜弢|杜弢]]的叛亂被平定，領導平亂的王敦因功獲升任鎮東大將軍，謝鯤亦被封為'''咸亭侯'''。謝鯤為官不求功名，不作磨煉，一舉一動都不要危及到自己。及見王敦不臣之心顯露，謝鯤心知王敦不能夠輔助，於是不理政事，更常與[[畢卓|畢卓]]、[[阮放|阮放]]等人縱酒。

永昌元年（322年），王敦以誅[[劉隗|劉隗]]、[[刁協|刁協]]之名起兵，謝鯤向王敦稱劉隗只是「城狐社鼠」，反對王敦起兵。王敦聽後大怒，命謝鯤出任豫章[[太守|太守]]，但卻不派他到豫章上任，反倒要他隨自己進攻建康，要借助他有才的聲望協助自己。

同年王敦攻陷建康，及後謝鯤建議任用極有名望的護軍將軍[[周顗|周顗]]和驃騎將軍[[戴淵|戴淵]]，令因王敦舉兵而不安紛亂的群眾安心；而王敦亦曾經和謝鯤說要以周顗為[[尚書令|尚書令]]，戴淵為[[尚書僕射|尚書僕射]]。然而，王敦最終卻因忌憚二人的名聲而將二人殺害。謝鯤在從未知王敦有殺周顗的意圖下得知周顗的死訊，驚愕不己，如親人逝世一般。王敦參軍[[王峤_(晋朝)|王峤]]因力圖諫止王敦殺周顗而被收捕並將處斬，一眾士人因畏懼王敦而不敢勸止，而謝鯤出言卻勸阻王敦，成功救了王嶠。

王敦自攻陷建康後，雖然已奪取朝政大權，然而都不肯朝見[[晉元帝|晉元帝]]，並將返回[[武昌|武昌]]。謝鯤此時試圖勸王敦朝見晉元帝，但王敦堅決不肯，並於當年返回武昌遙控朝政。謝鯤多次進端正之言但都不被王敦所用，心中不悅。謝鯤隨王敦回[[江州|江州]]後終於上任豫章，在政清正嚴明，甚得百姓愛戴。[[太寧_(晉)|太寧]]元年（323年）逝世，享年四十三歲。王敦之亂被平定後，謝鯤獲追贈[[太常|太常]]，[[諡號|諡號]]為'''康'''。

== 性格特徵 ==
*謝鯤豁達不拘細節，且有高明見識，不重服飾儀表，又喜好《[[老子|老子]]》和《[[易經|易經]]》，且能唱歌和鼓琴，令[[王衍_(西晉)|王衍]]和[[嵇紹|嵇紹]]驚奇。而又因謝鯤一改父親[[謝衡|謝衡]]崇尚[[儒學|儒學]]而改尚[[玄學|玄學]]，故此助陳郡謝氏於玄學盛行的[[兩晉|兩晉]]期間社會地位抬升。
*謝鯤因家僮犯罪而被免官後，名士[[王玄|王玄]]和[[阮脩|阮脩]]都為他而歎息，但謝鯤卻表現得毫不在乎，當時人都佩服他的高遠豁達，坦然面對榮辱。
*謝鯤曾經因鄰家高氏的女兒有美色，於是去輕薄她，遭對方以[[梭|梭]]投擲，更撞斷了兩隻[[牙齒|牙齒]]。當時的人因而說：「任達不已，幼輿折齒。」謝鯤聽後還很高傲的大聲呼叫：「這還不影響我高歌。」這是成語「[[wiktionary:投梭折齒|投梭折齒]]」的典故。
*一次謝鯤見當時還是太子的[[晉明帝|晉明帝]]，都十分親近和重視對方。晉明帝問：「人們都以你與[[庾亮|庾亮]]相比，你自己有甚麼看法？」謝鯤答：「以禮整治朝廷，為百官作榜樣，我不如庾亮；一丘一壑，我就認為我更佳。」丘指山丘，壑指山溝，一丘一壑就引申作寄情山水。這是成語[[wiktionary:一丘一壑|一丘一壑]]的典故。
*謝鯤與[[王澄_(西晉)|王澄]]、[[胡毋輔之|胡毋輔之]]及[[阮瞻|阮瞻]]等人皆自己上承[[阮藉|阮藉]]，得到大道之本。故此脫去衣服，放任如同禽獸。<ref>王隱《晉書》：「魏末，阮籍嗜酒荒放，露頭散髮，裸袒箕踞。其後貴游子弟阮瞻、王澄、謝鯤、胡毋輔之之徒，皆祖述於籍，謂得大道之本。故去巾幘，脫衣服，露醜惡，同禽獸。甚者名之為通，次者名之為達也。」</ref>又與王澄等仰慕[[竹林七賢|竹林七賢]]，披頭散髮，裸體箕踞而坐，號稱「八達」<ref>鄧粲《晉紀》</ref>。謝鯤亦為王澄所敬仰，謝鯤當王敦的長史時就曾經在王敦面前和謝鯤一直談話，更慨嘆只有謝鯤能和他暢談，竟然看也不看王敦。
*[[衞玠|衞玠]]於永嘉六年（312年）死，謝鯤在武昌發喪，十分激動。他人問他何故，謝鯤說：「國家損失了棟樑，怎能不哀傷。」可見謝鯤雖縱酒和崇尚玄學，但不忘國事。<ref>《世說新語·傷逝篇》註《永嘉流人名》</ref>

== 逸事 ==
*謝鯤在豫章避亂時，曾在一空亭留宿過夜，而曾有多人於空亭被妖怪所殺。快將天亮時，謝鯤聽到有個黃衣人叫他開門，而謝鯤卻不感畏懼，更從窗中伸手抓著黃衣人的手，在拉扯之間更將其手弄斷。謝鯤看著斷手，發現是一隻鹿腿，後出外根據血跡追縱，發現了那頭鹿。自以以後，該亭就再沒妖怪殺人的事。<ref>《晉書·謝鯤傳》：「鯤以時方多故，乃謝病去職，避地于豫章。嘗行經空亭中夜宿，此亭舊每殺人。將曉，有黃衣人呼鯤字令開戶，鯤憺然無懼色，便於窗中度手牽之，胛斷，視之，鹿也，尋血獲焉。爾後此亭無復妖怪。」</ref>
