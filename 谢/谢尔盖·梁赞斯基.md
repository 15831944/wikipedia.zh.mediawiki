'''谢尔盖·尼古拉耶维奇·梁赞斯基'''（{{lang-ru|Серге́й Никола́евич Ряза́нский}}；{{bd|1974年|11月13日|}}）是[[俄罗斯|俄罗斯]]已退役[[宇航员|宇航员]]。
==早年==
1974年出生，毕业于[[莫斯科国立大学|莫斯科国立大学]]生物系[[生物化学|生物化学]]专业，2005年获得宇航员资格证书，后担任[[俄罗斯科学院|俄罗斯科学院]][[俄罗斯科学院生物医学问题研究所|生物医学问题研究所]]工作人员，曾任“飞往火星之旅”试验队长<ref>{{cite web |author=张浩 |title=俄“飞往火星之旅”地面模拟实验顺利完成 |url=http://news.cctv.com/world/20071130/101635.shtml |publisher=[[中国中央电视台|CCTV]] |archiveurl=https://web.archive.org/web/20190414134322/http://news.cctv.com/world/20071130/101635.shtml |archivedate=2019-04-14}}</ref>。

==宇航员生涯==
2003年5月29日，他成为宇航员候选人之一，之后开始接受基础训练。2005年7月5日获得实验宇航员资格。
===远征37/38===
莫斯科时间26日0时59分（北京时间4时59分），[[联盟-FG|联盟-FG]]运载火箭在[[拜科努尔航天发射场|拜科努尔航天发射场]]升空，9分钟后[[联盟TMA-10M|联盟TMA-10M]]载人飞船脱离火箭进入预定轨道，后与国际空间站实现对接。他和[[奥列格·科托夫|奥列格·科托夫]]、[[迈克尔·霍普金斯|迈克尔·霍普金斯]]是本次的任务队员<ref>{{cite web |author=吴亚莉 |title=俄“联盟TMA-10M”飞船发射 与空间站实现对接 |url=http://mil.huanqiu.com/world/2013-09/4396424.html?agt=122_213.undefined |publisher=环球网 |archiveurl=https://web.archive.org/web/20190414145449/http://mil.huanqiu.com/world/2013-09/4396424.html?agt=122_213.undefined |archivedate=2019-04-14 |date=2013-09-26}}</ref>。

2013年11月9日，他和科托夫将未点燃的[[索契冬奥会|索契冬奥会]]火炬带出舱外，进行了太空传递，并拍摄了视频，这是奥运史上首次太空火炬传递活动<ref>{{cite web |title=冬季奥运会火炬首次在太空传递 |url=https://www.bbc.com/zhongwen/simp/world/2013/11/131109_russia_olympic_torch_spacerelay |publisher=[[BBC|BBC]]中文 |accessdate=2019-04-14 |date=2013-11-09}}</ref>。

莫斯科时间2013年12月27日晚，他和科托夫打开舱门，开始进行[[舱外活动|舱外活动]]。他们的主要任务是将一家[[加拿大|加拿大]]公司的高清[[摄像头|摄像头]]安装在[[星辰号服务舱|星辰号服务舱]]外表面，在安装完成后，地面控制中心未收到相关数据，他们不得不返回，又将其拆卸下来带回空间站<ref>{{cite web |author=岳连国 |title=俄宇航员太空行走刷新纪录 未完成全部预定任务 |url=http://www.chinanews.com/gj/2013/12-30/5679102.shtml |publisher=[[中国新闻网|中国新闻网]] |accessdate=2019-04-14 |archiveurl=https://web.archive.org/web/20140904101213/http://www.chinanews.com/gj/2013/12-30/5679102.shtml |archivedate=2014-09-04 |date=2013-12-30}}</ref>。他们的任务时间最终达到了8小时7分钟，创下了俄宇航员身穿俄制[[海鹰宇航服|海鹰-MK宇航服]]舱外工作时长的新记录<ref>{{cite web |title=Космонавты вернулись на МКС, проведя в открытом космосе свыше восьми часов |url=https://www.interfax.ru/world/349429 |publisher=[[国际文传电讯社|Интерфакс]] |accessdate=2019-04-14 |language=ru |date=2013-12-28}}</ref>。这一纪录在2018年被[[亚历山大·米苏尔金|亚历山大·米苏尔金]]以8小时13分钟的时长打破<ref>{{cite web |title=国际空间站俄两名宇航员打破外太空作业时间纪录 |url=http://news.sciencenet.cn/htmlnews/2018/2/402079.shtm |publisher=[[中国新闻网|中国新闻网]] |archiveurl=https://web.archive.org/web/20190414112503/http://news.sciencenet.cn/htmlnews/2018/2/402079.shtm |archivedate=2019-04-14 |date=2018-02-03}}</ref>。

2014年1月27日，他和科托夫再次进行了出舱工作，主要解决上次安装失败的问题，在[[星辰号服务舱|星辰号服务舱]]外表面安装了一个高等分辨率摄像头和一个中等分辨率摄像头，整个过程持续了约6个小时<ref>{{cite web |title=俄宇航员在国际空间站外表面安装两个摄像头 |url=http://www.chinanews.com/gj/2014/01-28/5793992.shtml |publisher=中国新闻网 |accessdate=2019-04-14 |archiveurl=https://web.archive.org/web/20140228025416/http://www.chinanews.com/gj/2014/01-28/5793992.shtml |archivedate=2014-02-28 |date=2014-01-28}}</ref>。

[[格林威治时间|格林威治时间]]2014年3月11日0时1分（北京时间11日8时1分），[[联盟TMA-10M|联盟TMA-10M]]号飞船与空间站脱离，1分半钟后脱离空间站进入返回轨道<ref>{{cite web |title=国际空间站3名宇航员将返回地球 工作满半年 |url=http://www.chinanews.com/gj/2014/03-10/5933045.shtml |publisher=中国新闻网 |accessdate=2019-04-14 |archiveurl=https://web.archive.org/web/20140311225455/http://www.chinanews.com/gj/2014/03-10/5933045.shtml |archivedate=2014-03-11 |date=2014-03-10}}</ref>。他和同事成功返回地球。

===远征52/53===
2017年7月28日，载有梁赞斯基等宇航员的[[联盟MS-05|联盟MS-05]]飞船从哈萨克斯坦[[拜科努尔航天发射场|拜科努尔航天发射场]]升空<ref>{{cite web|title=俄美意三国宇航员搭“联盟”飞船在哈升空|url=http://news.xinhuanet.com/asia/2017-04/21/c_129559939.htm|publisher=中新网|date=2017-07-29}}</ref> ，6小时13分钟后与国际空间站对接<ref>{{cite web|title=3名宇航员搭乘"联盟MS-05"载人飞船进入国际空间站|publisher=中新网|url=http://www.chinanews.com/gj/2017/07-29/8290874.shtml|date=2017-07-29}}</ref>。

北京时间2017年8月17日22时，他和[[费奥多尔·尤尔奇欣|尤尔奇欣]]穿着[[海鹰宇航服|海鹰-MKS宇航服]]进行了太空出舱行走，手动释放了5颗纳米卫星，其中包括一颗用于预测国际空间站运行情况的“TS530-镜子”卫星，一颗[[3D打印|3D打印]]卫星“Tomsk-[[托木斯克理工大学|TPU]]-120”<ref>{{cite web |title=俄首颗3D打印卫星已开始传输包括中文在内的11种语言信号 |url=http://sputniknews.cn/russia/201708181023399021/ |publisher=[[俄罗斯卫星通讯社|俄罗斯卫星通讯社]] |accessdate=2019-04-14 |date=2017-08-18}}</ref>，以及两颗用来庆祝首颗[[人造卫星|人造卫星]]发射60周年的“塔纽莎”卫星。整次太空行走共持续了7小时34分钟<ref>{{cite web |author=张艾京 |title=俄宇航员进行太空行走 手动释放五颗纳米卫星 |url=http://www.chinanews.com/gj/2017/08-18/8308065.shtml |publisher=[[中国新闻网|中国新闻网]] |accessdate=2019-04-14 |archiveurl=https://web.archive.org/web/20181111195920/http://www.chinanews.com/gj/2017/08-18/8308065.shtml |archivedate=2018-11-11 |date=2017-08-18}}</ref>。这是俄宇航员2017年的唯一一次太空行走<ref>{{cite web |title=俄宇航员太空出舱行走 手动释放5颗纳米卫星 |url=http://www.taikongmedia.com/Item/Show.asp?m=1&d=24530 |publisher=资格太空网 |archiveurl=https://web.archive.org/web/20190414133716/http://www.taikongmedia.com/Item/Show.asp?m=1&d=24530 |archivedate=2019-04-14 |date=2017-08-18}}</ref>。

他和尤尔奇欣拍摄了历史上首部外太空全景视频，该视频已经于2017年10月4日在[[莫斯科航天博物馆|莫斯科航天博物馆]]及网站公开发布<ref>{{cite web |title=俄罗斯公布史上首部太空全景视频 |url=http://world.people.com.cn/GB/n1/2017/1005/c1002-29573085.html |publisher=[[人民网|人民网]] |archiveurl=https://web.archive.org/web/20190414133254/http://world.people.com.cn/GB/n1/2017/1005/c1002-29573085.html |archivedate=2019-04-14 |date=2017-10-05}}</ref>。
