《'''等一個人咖啡'''》（{{lang-en|'''''Café·Waiting·Love'''''}}）是一部於[[2014年電影列表|2014年]]上映的[[爱情片|愛情電影]]，改編自[[臺灣|臺灣]]知名作家[[九把刀|九把刀]]小說《等一個人咖啡》，由[[江金霖|江金霖]]執導、九把刀編劇。

== 概要 ==
《等一個人咖啡》<small>（2004年出版）</small>是作家九把刀的第一本小說，亦是九把刀銷量僅次於《[[那些年，我們一起追的女孩_(小說)|那些年，我們一起追的女孩]]》<small>（2006年出版）</small>的作品。《等一個人咖啡》是繼電影《[[那些年，我們一起追的女孩|那些年，我們一起追的女孩]]》<small>（2011年上映）與</small>《[[殺手歐陽盆栽|殺手歐陽盆栽]]》<small>（2011年上映）</small>之後，九把刀第三部改編成電影的小說。

《等一個人咖啡》於2013年12月27日開鏡，並展開拍攝。導演江金霖是《那些年，我們一起追的女孩》的執行導演，由賴雅妍以及新人演員宋芸樺、禾浩辰主演，還有資深演員藍心湄、李㼈等人加盟演出，並請來港星周慧敏特別演出老闆娘。2014年1月17日舉行媒體發佈會，公布主要演員與其他製作班底。全片於同年3月上旬殺青，台灣口碑場8月8日每院每日一場，香港口碑場8月9號單一場。<ref>[http://news.ltn.com.tw/news/entertainment/breakingnews/1061900 九把刀打造新女神 港台同步飄出咖啡香],2014.07.22 自由時報 記者封以恩</ref>

== 劇情簡介 ==
大學新鮮人思螢（[[宋芸樺|宋芸樺]]飾），為了她的暗戀對象，喜歡坐在固定座位，看似身邊女友不斷的澤-{于}-（[[張立昂|張立昂]]飾），而來到「等一個人咖啡」店打工，結識了[[咖啡|咖啡]]沖調技術高超，任何客人點的特調咖啡都能做得到的超酷[[女同性戀|拉子]]——阿不思（[[賴雅妍|賴雅妍]]飾）、與每天都看似無所事事的神祕美麗老闆娘（[[周慧敏|周慧敏]]飾）。

在大學裡擁有許多離奇傳說的學長阿拓（[[布魯斯_(藝人)|禾浩辰]]飾）和一群好友到「等一個人咖啡」喝咖啡，卻碰上被其搶走前女友的[[女同性戀|拉子]]阿不思，很有正義感的思螢看不慣阿拓<ref name="ccu-econ-20041101">{{cite web|url=http://mag.udn.com/mag/edu/storypage.jsp?f_ART_ID=3649|title= 網路小說主角 阿拓走了 | date= 2004-11-01 | author = 謝恩得 | publisher = [[聯合報系|聯合報系]] | coauthors = 謝育宏 （校園特約記者） | work = [[聯合晚報|聯合晚報]]| location = [[嘉義縣|嘉義]][[民雄鄉|民雄]] }}</ref>被朋友們嘲笑而大罵那群損友，最後思螢和阿拓結為好友。開朗的阿拓熱血又真誠，為了完成去環遊世界的夢想到處打工，但因為與人打賭賭輸而不斷延畢，最終成為校園中的傳奇老學長；不過卻也因為打工認識了看似黑道大哥但其實曾是電影導演的海產店老闆暴哥（[[李㼈|李㼈]]飾），以及擁有一身驚人廚藝的洗衣店老闆金刀嬸（[[藍心湄|藍心湄]]飾）。

深深被思螢的正義感吸引的阿拓，常常介紹他那些有趣的好朋友給思螢認識，思螢的生活也因為有阿拓而變得多彩多姿。可是思螢一直暗戀著澤-{于}-而只是把阿拓當成無話不談的好朋友，直到阿拓決定要離開臺灣，出國去實現夢想時，思螢才明白原來她所等的人不是澤-{于}-。

== 和小說相異之處 ==
* 原著中女主角就讀高三直到大四，電影版主角剛進大學。
* 原著中咖啡店在[[國立清華大學|清大]]夜市巷子底，故事場景主要發生在[[國立清華大學|清大]]、[[國立交通大學|交大]]附近，電影版沒有指涉故事場景地點（根據場景判斷為政大、台科大和北醫）。
* 原著中思螢暗戀的澤-{于}-，和老闆娘沒有任何感情關係，原著中亦沒有死去。
* 原著中老闆娘最後在「等一個人咖啡」等到了真愛，電影版則改為在世界各地散心。
* 原著阿拓服兵役離開，電影版改為到世界各地遊玩。
* 原著中思螢在阿拓走後，遇到阿拓的好友，才開始想起阿拓，電影版則在最後遇到澤-{于}-後醒悟。
* 原著中咖啡店在晚上十時三十分打烊，而電影版中咖啡店營業時間至晚上九時。
* 原著中是蝴蝶刀阿拓，個性隨和悠閒；而不是比基尼阿拓或大白菜阿拓，個性較單純。
* 原著中暴哥是黑社會老大後來有女人，金刀嬸自己週日在洗衣店的二樓聚會並且有其老公金刀桑，二人並無甚關係。
* 原著中咖啡店有換位置並且改名，電影沒有。
* 原著中思螢的哥哥、室友、阿拓的朋友、亂點王、技安張及培信皆未出現於電影中。

== 軼聞 ==
=== 關於阿拓 ===
就讀國立中正大學經濟系三年級的曾元拓是九把刀的書迷。於2004年10月，阿拓騎[[機車|機車]]時，為閃避逆向而來的[[腳踏車|腳踏車]]發生車禍，昏迷四天後去世<ref>[http://www.appledaily.com.tw/appledaily/article/headline/20041023/1328941/ 網友集氣搶救「阿拓」人氣小說真實男主角車禍昏迷命危]台灣蘋果日報2004年10月23日</ref><ref>[http://blog.udn.com/likeking/3538880 九把刀小說主角曾元拓 閃車殞命]</ref>，被[[大愛電視台|大愛電視台]]將其故事拍攝成[[大愛劇場|大愛劇場]][[人生旅程(一)-緣|人生旅程(一)-緣]]。因此在片末有一小段懷念阿拓的文字——阿拓，我們都很想你。
