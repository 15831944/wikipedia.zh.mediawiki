'''酷狗音乐'''（KuGou）是一款[[P2P|P2P]]音乐共享软件及[[音频|音频]]媒体播放软件，可以实现音乐和音乐MV的[[搜索|搜索]]、[[下载|下载]]和播放<ref name=":1">{{Cite web|url = http://mt.sohu.com/20151020/n423687788.shtml|title = 移动音乐应用分析报告-酷狗音乐(上篇)|accessdate = |author = |date = 2015年10月20日|publisher = 搜狐}}</ref>。酷狗隶属于腾讯音乐娱乐（广州）有限公司（前身为广州酷狗计算机科技有限公司，已于2017年更名），公司于2004年在[[广州|广州]]成立。打开软件时的经典提示语为“Hello！酷狗”。酷狗面向大众用户，主要盈利方式为游戏、广告宣传和直播间业务<ref>{{Cite web|url = http://mt.sohu.com/20151015/n423320619.shtml|title = 曾经的霸主酷狗没有巨头撑腰，怎么走？|accessdate = |author = |date = 2015-10-15|publisher = 搜狐}}</ref>。酷狗购买音乐版权并提供给用户，同时为用户提供广告服务以此赚取广告流量费<ref name=":1">{{Cite web|url = http://mt.sohu.com/20151020/n423687788.shtml|title = 移动音乐应用分析报告-酷狗音乐(上篇)|accessdate = |author = |date = 2015年10月20日|publisher = 搜狐}}</ref>。

== 历史 ==
2014年4月酷狗音乐和酷我音乐合并，成立海洋音乐集团，预计2015年底前海外上市<ref>{{Cite web|url = http://tech.qq.com/a/20150313/009742.htm|title = 曝酷狗酷我在一起了 新公司年底冲刺上市|accessdate = |author = |date = 2015年3月13日|publisher = 腾讯}}</ref>。

2014年12月酷狗起诉网易云音乐称其侵权200首歌曲，2015年1月7日网易云音乐又起诉酷狗音乐侵权37张专辑，共300首歌曲<ref>{{Cite web|url = http://www.nbd.com.cn/articles/2015-01-09/889436.html|title = 网易反诉酷狗音乐侵权 版权混战将加速行业洗牌|accessdate = |author = |date = 2015-01-09|publisher = 每日经济新闻}}</ref>；2015年2月酷狗音乐起诉阿里音乐旗下[[天天动听|天天动听]]侵权，2015年6月阿里音乐又起诉酷狗涉嫌侵权歌曲260首<ref>{{Cite web|url = http://mt.sohu.com/20150617/n415163490.shtml|title = 数字音乐版权厮杀大战扑朔迷离，酷狗出尽风头？|accessdate = |author = |date = 2015-06-17|publisher = 搜狐}}</ref>。2015年7月，[[中国国家版权局|中国国家版权局]]发布《关于责令网络音乐服务商停止未经授权传播音乐的通知》，要求网络音乐服务商停止未经授权传播音乐作品<ref>{{Cite web|url = http://china.cnr.cn/ygxw/20151023/t20151023_520248601.shtml|title = 国家版权局解读“网盘”新规：仍能上传 限制分享|accessdate = |author = |date = 央广网|publisher = 2015-10-23}}</ref><ref>{{Cite web|url = http://ent.qq.com/a/20150709/056283.htm|title = 国家版权局整改网络音乐：停止未经授权传播|accessdate = |author = |date = 2015-07-10|publisher = 腾讯娱乐}}</ref>。

2015年7月酷狗推出新的音乐推广方式，用户可以花费100元[[人民币|人民币]]让酷狗进行用户的个人演唱推广，每个被推广人的价格为0.1元，100元最多可将歌曲推广给1000人<ref>{{Cite web|url =http://ent.ynet.com/3.1/1507/09/10216340.html|title =酷狗斥资千万 助力音乐人实现音乐梦想|accessdate =|author =|date =2015-07-09|publisher =北青网}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2015年8月10日酷狗举行“星梦工程·造星计划”发布会，宣布培养有潜质的明星<ref>{{Cite web|url =http://www.ycwb.com/ePaper/ycwb/html/2015-08/12/content_764811.htm?div=-1|title =酷狗音乐打造“星梦工程”|accessdate =|author =|date =2015-08-12|publisher =羊城晚报}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2015年8月时酷狗拥有7亿用户，平台日播放量达到30亿次，旗下有繁星网、5sing原创音乐平台等，是第一大网络音乐平台<ref>{{Cite web|url = http://www.ycwb.com/ePaper/ycwb/html/2015-08/17/content_768133.htm?div=-1|title = 酷狗音乐发力“粉丝经济” 开启音乐产业O2O元年|accessdate = |author = |date = 2015-08-17|publisher = 羊城晚报}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，同时与国内外超过500家唱片公司合作建立正版音乐库<ref>{{Cite web|url = http://news.163.com/15/0810/09/B0L9CCH900014AEE.html|title = 酷狗音乐将亮相南国香节 观众现场体验酷狗K歌系统|accessdate = |author = 金羊网|date = 2015-08-10|publisher = 搜狐}}</ref>。2015年9月，易观智库《2015年上半年中国移动音乐用户专题研究报告》调研显示酷狗[[流動應用程式|移动用户]]以28.8%的用户占据第二，而QQ音乐以40.7%超出酷狗10余个百分点占据第一<ref>{{Cite web|url = http://ent.news.cn/2015-09/15/c_128231861.htm|title = 移动音乐APP用户调查 QQ音乐领跑酷狗音乐第2|accessdate = |author = |date = 2015年9月15日|publisher = 中华网}}</ref>，同时该报告还显示酷狗音乐90后和00后用户加起来已占到40%以上，85后用户占比为29.8%<ref name=":0">{{Cite web|url = http://www.chinadaily.com.cn/micro-reading/dzh/2015-10-22/content_14276414.html|title = 酷狗引领音乐live直播行业：咱不差“小鲜肉”！|accessdate = |author = |date = 2015-10-22|publisher = 中国日报网}}</ref>。

2015年酷狗获得9月26日一直持续到10月25日[[湘江|湘江]]音乐节在线直播授权<ref>{{Cite web|url = http://news.ycwb.com/2015-09/21/content_20683606.htm|title = 酷狗酷我彩虹再次联手 开启中国最大在线音乐节|accessdate = |author = |date = 2015-09-21|publisher = 金羊网}}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，有超过1000万的用户通过酷狗在线播放观看<ref name=":0" />。

2016年7月22日，[[中国音乐集团|中国音乐集团]]（简称“CMC”）和腾讯集团日前共同宣布，双方已达成共识对数字音乐业务进行合并。在本次交易中，腾讯把旗下的QQ音乐业务与CMC进行合并，通过资产置换股权成为新的[[腾讯音乐娱乐集团|腾讯音乐娱乐集团]]大股东，CMC旗下的酷狗音乐、酷我音乐将与QQ音乐成为一家人。2017年，为强化集团地位，酷狗母公司更名为“腾讯音乐娱乐（广州）有限公司”。
