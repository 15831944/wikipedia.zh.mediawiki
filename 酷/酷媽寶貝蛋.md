《'''酷媽寶貝蛋'''》（{{lang-ja|'''ママはぽよぽよザウルスがお好き'''}}），是一套1993年8月於[[日本|日本]]刊登的漫畫。1995年改編成電視動畫。

== 作品概要 ==
本作是連載於育兒雜誌『プチ・タンファン』（[[婦人生活社|婦人生活社]]發行）的作品，是一部搭配作者本人育兒經驗談的'''實錄育兒漫画'''（全4集）。

描述要照顧雖然可愛卻活潑而又精力十足的小怪獸們（寶貝蛋＝[[小孩|小孩]]）的青沼家之中所發生親子交流的生活故事。

由於[[郵便局|郵便局]]（[[日本郵便|日本郵便]]前身）使用裡面角色拍攝｢学資保險｣的[[電視廣告|電視廣告]]而因此成為話題，進而動畫化。

== 單行本 ==
單行本從1993年8月連載到1998年5月，由婦人生活社發行。2003年11月到2004年2月由[[幻冬舍|幻冬舍]]再次發行。

=== 主要登場人物（原作）===
; 青沼貴子（あおぬま　たかこ）
: 小龍和小安的母親，也是本作的[[作者|作者]]。本作中雖然姓「青沼」，但正確來說這是作者的[[舊姓|舊姓]]以及[[筆名|筆名]]，育兒日記中則是姓「鈴木」。
; 青沼源大（あおぬま げんだい）
: 小龍和小安的父親及貴子的老公。全家人都叫他「親愛的」，但沒人叫過他的名字。職業是建築測量士。留著[[鬍子|鬍子]]，還是[[披頭四|披頭四]]的歌迷
; 青沼龍（あおぬま りゅう）
: 青沼家的長男。描寫他從嬰兒時到幼稚園～小学低年級的成長過程。因為活潑又愛破壞，而被叫作[[怪獸|怪獸]][[哥吉拉|哥吉拉]]。2014年3月的現在為25歳，幫忙父親的建築測量士的工作。之後雖然直接就任這份工作，但2015年時與父親產生激烈的衝突而辭職，還當了2個月的[[尼特族|尼特族]]。2016年4月時為28歳、一邊打工一邊求取想取得的職業資格。
; 青沼安（あおぬま あん）
: 青沼家的長女。描寫她從在媽媽的肚子裡到幼稚園時的成長過程。小時候是個重視自己而又不愛聽話的小孩，因為喜歡大叫而被稱為怪獸安基拉斯。17歳時從高中輟學，並且透過通信教育的高中自主學習，並希望成為舞者，21歳時想要上大學而成為考生。22歳成為短期大學生並在24歳畢業。之後就職7個月後就辭職，並很快決定要轉職，2016年4月時為25歳，並成為[[OL|OL]]。
; 青沼傑克（あおぬま じゃっく）
: 青沼家養的貓。2002年2月以18歳高齡過世。

== 電視動畫 ==
1995年9月2日到1996年8月31日、由[[TBS電視台|TBS]][[Japan_News_Network|系列局]]以、[[MBS電視台|毎日放送]]（MBS）・[[日本動畫公司|日本アニメーション]]製作為電視動畫，全52集（1集播放2話）。平均收視率7.8％。

各事件的故事發展，基本上都忠於原作，但原作中帶有嘲諷辛辣的橋段則被改為較為溫暖的內容。

動畫前半段的故事雖然以女主角‧未來的日記為視角構成，但後半段轉變成動畫原創故事，自此之後，就較少以未來的[[第一人稱|第一人稱]]視角為觀點來發展故事。
