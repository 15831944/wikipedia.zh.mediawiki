'''Intel Core'''属于[[Pentium_M|Pentium M]]架構的最后一代產品。它是[[英特爾|英特爾]]在2006年1月打造新形象時发布的新一代处理器。晶片產品代號為'''Yonah'''（來自[[希伯來語|希伯來語]]的{{lang|he|יונה|}}），並於2006年1月5日（东亚） / 1月6日（美国）發行。这是Intel的第一款32位双核移动低功耗处理器、第一款65nm制程的移动处理器。

最初的'''Core'''品牌是指[[Intel|Intel]]的[[32-bit|32-bit]]移动双核[[x86|x86]][[CPU|CPU]]，源于[[Pentium_M|Pentium M]]处理器。Intel Core处理器系列使用了增强版本的[[Intel_P6|Intel P6]]微架构。其[[微架构|微架构]]也被称为“Enhanced Pentium M”。Intel Core处理器是其后继产品——[[64位|64位]][[Intel_Core微处理器架构|Intel Core微处理器架构]]（商标名称为Core 2）CPU的先驱。Intel Core品牌处理器包括两个分支：'''Duo''' (双核)与'''Solo''' (即Duo处理器，但其中的一个核被停用，用来替代Pentium M品牌的单核移动处理器)。

== Yonah ==

Yonah為英特爾第一代65納米行動處理器的產品代號，基於包含Banias、Dothan代號的[[Pentium_M|Pentium M]]處理架構，並融合了[[LaGrande|LaGrande]]安全性技術。浮點運算在[[SSE3|SSE3]]的部分被加強並與[[SSE|SSE]]與[[SSE2|SSE2]]整合，雖然目前看來整體表現稍降，但它有較大的快取。另外，Yonah也支持[[NX_bit|NX bit]]。

英特爾在[[國際消費電子展|CES 2006]]發表了Core Solo與Core Duo，這些晶片與前代的命名方式不同。'''T'''代表的是「演算」部分，而'''L'''代表的是「低耗電」部分。這些字後面會包含4數字的代號，但不能相比較。所有的Core產品線都具備2[[MB|MB]]的L2快取（雙核版的會共享）。

Yonah的双核布局非常类似于单个芯片内布放两个互联的[[Pentium_M|Pentium M]]处理器。所以，这种32位的Core品牌CPU，更像是它的前驱产品Pentium M处理器，而不像它的后继产品——采用64位[[Intel_Core微处理器架构|Core微架构]]的品牌为[[Core_2|Core 2]]的CPU. 虽然Intel从2006年1月花了很大努力重新树立其“Core”的新品牌，但是仍然有很多使用了Yonah的计算机继续标示为Pentium M处理器。

Core系列处理器也是第一种用于Apple公司的[[Macintosh|Macintosh]]计算机的Intel的CPU。Core Duo被用于第一代MacBook Pro，Core Solo被用于[[Mac_mini|Mac mini]]产品线。Core Duo开始了Apple整个PC产品线采用Intel处理器。 

从2007年到2009年, [[Intel|Intel]]把32位Yonah核心的移动处理器，以及64位[[Intel_Core微处理器架构|Core微架构]]的台式机处理器，使用新的品牌—[[Pentium_Dual-Core|Pentium Dual-Core]]—作为Intel的中低端产品线的品牌。 

2007年9月与2008年1月4日，几种''Core''品牌的CPU，包括几种Core Solo, Core Duo, Celeron与一种Core 2 Quad停产<ref>{{cite web |title=Intel already phasing out first quad-core CPU |url=http://www.tgdaily.com/content/view/33752/135/ |publisher=TG Daily |accessdate=2007-09-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20070913123702/http://www.tgdaily.com/content/view/33752/135/ |archivedate=2007-09-13 }}</ref><ref>{{cite web |title=Intel to discontinue older Centrino CPUs in Q1 08 |url=http://www.tgdaily.com/content/view/34115/122/ |publisher=TG Daily |accessdate=2007-10-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20071102195746/http://www.tgdaily.com/content/view/34115/122/ |archivedate=2007-11-02 }}</ref>。

=== Sossaman ===
{{main|Xeon}}
Sossaman是一個Yonah的衍生品，產品代號為Sossaman的雙核心[[Xeon|Xeon]] LV處理器於2006年3月14日發行。除了在雙核心的配置外， Sossaman與Yonah幾乎是一樣的。

== 技術規格 ==

Core Duo處理器核心晶片中內含1.51億個電晶體，包含分享所搭載的2 MB L2[[快取|快取]]。Yonah運作核心內含12條管線，預測最大運行時脈可達2.33–2.50 GHz。此一L2快取記憶體與兩顆獨立運算核心之間溝通、協調的方式，乃是透過一稱為「仲裁匯流排單元」進行資料或訊號的存取，且可縮短前置匯流排運作時，因快取記憶體依附性所造成的資料壅塞。不過，這樣的一套機制卻可能會增加L2快取大約10個時脈週期的延遲時間。<!--of the Dothan Pentium M to 14 clock cycles. The increase in clock frequency offsets some of the increased clock cycle latency.-->

Yonah支援945GM、945PM與945GT晶片組。Core Duo與Core Solo皆使用FCPGA6（478針）平台，但針腳的排列與新的核心功能皆不能和前代的[[Pentium_M|Pentium M]]或是[[Centrino|Centrino]]平台相容。

=== 優點 ===

Yonah 代表一個在處理器製造工業的突破性進展和每瓦特（Watt）能量消耗的表現。它同時提供一個暫時最快取記憶體（Cache）的科技，在雙核心的情況下快取記憶體仍然有良好的連貫性和窺探；它支援 667 MHz 外頻，和[[雙通道|雙通道]]（Dual Channel）[[DDR2_SDRAM|DDR-II]] 記憶體。在處理器核心電源管理的特色是粒狀的溫度控制，還支持對於單個核心的獨立能耗控制（比如在運算負荷較輕的情況下完全關閉一個核心），大大減少能量消耗。

=== 缺點 ===
Yonah的缺點主要繼承於先前的Pentium M架構，比如，缺乏片上（on-die）內存控制器所導致的內存延遲偏大（在DDR2內存本身的延遲就大於DDR的背景下，這一缺憾尤為明顯）以及浮點運算單元數量減少所帶來的運算效能下降。在已有處理器支緩 [[x86-64|x86-64]]擴展技術能夠同時運行[[32位元|32位元]]和[[64位元|64位元]]程式的情況下，Core Duo仍是一個32-位元處理器。這將Core Duo的地址空間限制在<!--It limits the size of a process's (linear) address space to --> 4 [[gibibyte|GiB]]。

=== Core Duo ===
[[File:Intel_Core_Duo.png|thumb]]
'''Intel Core Duo'''<ref>{{cite web|url=http://www.intel.com/support/processors/mobile/coreduo/ |title=Support for the Intel Core Duo Processor |publisher=Intel.com |date= |accessdate=2010-12-13}}</ref> (產品編號：80539)是全球第一個低耗電的[[雙核|雙核]]處理器（低於25瓦特），在這之前的低耗電產品為[[Opteron|Opteron]] 260/860 HE的55瓦特。Core Duo於2006年1月5日發行，並作為Napa平台的部件之一。它也是第一款[[蘋果電腦|蘋果]][[麥金塔電腦|麥金塔電腦]]所使用的Intel處理器。（蘋果開發機器使用了[[Pentium_4|Pentium 4]]處理器）。

與早期報告不同，Intel Core Duo支持Intel的虛擬技術<ref>[http://www.intel.com/performance/resources/briefs/mobiletechnology.pdf IntelR CentrinoR Duo 內部行動演算]</ref>。不過，看來一些系统集成商已經讓此功能默认是不可用的，須從BIOS選項设置使之工作。

64位的[[指令集架构|指令集架构]]技术（Intel的[[x86-64|x86-64]]架構）並未被Yonah支持。然而Yonah的繼承者--[[Merom|Merom]]支持了X86-64架构。

'''Intel Core Duo'''芯片内为雙核心，由此二核心共享芯片内的2 [[MB|MB]] L2高速缓存，並由仲裁控制器负责L2高速缓存與FSB（front-side bus）的运作。Core Duo處理器可以暫停其中一个核心的电力供应以省電的功能。

=== Core Solo ===
[[File:Intel_Core_Solo.png|thumb]]
'''Intel Core Solo'''<ref>{{cite web|url=http://www.intel.com/support/processors/mobile/coresolo/ |title=Support for the Intel Core Solo processor |publisher=Intel.com |date= |accessdate=2010-12-13}}</ref> (產品編號：80538)實際上也有兩個核心，然而 Intel 故意關掉一個，以區別於 Core Duo。最初目的是確保即使有一個核心壞了，處理器依然能工作，從而減少由於新產品可能出現的質量問題給公司帶來的麻煩。也避免了开设专门生产单核CPU的生产线的麻烦。早在[[Intel_80486|486]] CPU的年代，Intel就采用过类似的策略。早期的[[486SX|486SX]] CPU实际上就是按照全功能的[[486DX|486DX]] CPU制造的，但是[[浮点单元|FPU]]被关闭了。

== 後繼者 ==
{{main|Intel Core 2}}

Core的後繼者為使用[[Intel_Core微處理器架構|Intel Core微處理器架構]]核心的Intel Core 2處理器，於2006年夏季推出。Core 2與只推出筆記型處理器的Core不同，它的推出代表著英特爾於筆記型處理器及桌上型處理器兩個品牌線的重整合。

另外，英特爾將Intel Core技術加以改良，成為新一代命名為Conroe-L 核心的處理器，但這新一代處理器不再使用Intel Core的處理器代號，取而代之會使用[[Pentium_E|Pentium E]]及[[Celeron|Celeron]] 400處理器代號，為入門級的電腦使用者提供選擇。
