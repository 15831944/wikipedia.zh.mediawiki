'''{{zy|黟|-{zh-cn:yī;zh-hk:ji<sup>1</sup>;zh-tw:ㄧ}-}}县'''，是[[中华人民共和国|中华人民共和国]][[安徽省|安徽省]][[黄山市|黄山市]]下辖的一个县。面积847平方公里，人口10万。黟县是黄山市最小的一个县，也是最古的县之一。

== 历史 ==
黟县古称'''黝'''（黟的异体字，音同黟），始建于[[秦|秦]]二十六年（公元前221年）。

[[西汉|西汉]][[鸿嘉|鸿嘉]]二年（公元前19年）六月，黝县改为广德国。鸿嘉五年，广德国被废除，复称黝县。[[王莽|王莽]]始建国二年（公元10年），黝县改称'''朔卤'''，属丹阳郡。[[建武|建武]]元年（公元25年），复称黝县。

1950年属太平、石台的美溪、柯村、宏潭划归黟县管辖。1958年1月2日，黟县县委、县政府机关奉令迁至[[祁门县|祁门县]]合并办公。1959年4月5日，恢复黟县建置。

== 自然地理 ==
黟县北枕黄山，南望白岳，四面群山环抱，自古便享有'''小桃源'''之美誉。黟县境内连绵的群峰与黄山联为一体，在历史上曾阻碍了古黟与外部世界的交往，造就了黟县“世外桃源”般的生态环境。

黟县耕地面积共0.403万公顷，山场面积7.04万公顷，森林覆盖率达76.3%。年平均气温15.8℃，降水量1686毫米，无霜期212-213天。

== 风景名胜 ==
黟县境内至今仍存有保护完整的古民居3600幢，为皖南之首。[[西递|西递]]、[[宏村|宏村]]、[[南屏|南屏]]、[[关麓|关麓]]、[[屏山|屏山]]等古民居建筑村落布局严谨，工艺精湛，蕴藏着极其丰富的文化内涵，其中西递、宏村已被联合国教科文组织列为[[世界文化遗产|世界文化遗产]]。《黟山竹枝词》是吟詠古徽州黟县的创作，多記风土民情、山川形胜。[[俞正燮|俞正燮]]有竹枝词：“两般胜贵米与钱，大业从来说垦田；人众真难为造物，苞芦已值到山颠。”又“几层小楼傍山隈，六尺地从三户开；游客不知人逼仄，间评都说好楼台。”

== 特产 ==
泗溪三宝：[[苦丁茶|苦丁茶]]、[[香榧|香榧]]、[[葛粉|葛粉]]。
* [[苦丁茶|苦丁茶]]，富含[[多酚类|多酚类]]、[[三萜类|三萜类]]、[[黄酮类化合物|黄酮类化合物]]、[[皂甙|皂甙]]、[[熊果酸|熊果酸]]、[[蛇麻脂醇|蛇麻脂醇]]、[[蒲公英赛醇|蒲公英赛醇]]及多种[[无机盐|无机盐]]、[[氨基酸|氨基酸]]、[[维生素|维生素]]，营养与药理成分丰富。含有多种生理活性天然物质和人体所需要的[[硒|硒]]、[[锌|锌]]等多种[[微量元素|微量元素]]。

* [[香榧|香榧]]仁肉有丰实的[[脂肪|脂肪]]、[[蛋白质|蛋白质]]、[[糖|糖]]份及[[碳水化合物|碳水化合物]]，并含有多种[[维生素|维生素]]和[[不饱和脂肪酸|不饱和脂肪酸]]，有极高的营养价值。

* 腊八豆腐，是[[安徽|安徽]]黟县民间风味特产，在[[春节|春节]]前夕的[[腊八|腊八]]，即农历十二月初八前后，黟县家家户户都要晒制豆腐，民间将这种自然晒制的豆腐，称作腊八豆腐。

* 食桃，由优质[[籼米|籼米]]掺入一定比例的糯米机成米粉制成。用水和粉，揉成粉团，压入模中成型，蒸熟即可食用。
