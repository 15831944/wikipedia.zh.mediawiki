'''叉袋角'''，又称'''沙袋角'''，系[[上海市|上海市]]市区北部一片区名，位于[[静安区|静安区]]北部[[海防路|海防路]]、[[淮安路_(上海)|淮安路]]、[[西苏州路|西苏州路]]交会处一带及普陀区莫干山路附近一带，因流经莫干山路附近处的[[苏州河|苏州河]]形状凸出角状，类似[[叉袋|叉袋]]这种上海一带常用的农具而得名。莫干山路以东河湾一带，后都被工厂占据，而叉袋角地名逐渐消失，惟安远路以南仍沿用此地名。<ref name="上海地名志">{{cite book |author1=《上海地名志》编纂委员会 |coauthors=上海市地方志办公室 |title=上海地名志 |publisher=上海社会科学院出版社 |location=上海 |isbn=7-80618-527-5 |url=http://www.shtong.gov.cn/Newsite/node2/node2245/node70862/node70869/node70914/node70918/userobject1ai73712.html |accessdate=2018-06-09|date=1998.12}}</ref>

== 得名 ==
《上海地名志》认为，此地因流经莫干山路附近处的[[苏州河|苏州河]]形状凸出角状，类似[[叉袋|叉袋]]这种上海一带常用的农具而得名<ref name="上海地名志" />。

但是，《法华乡志》则称此地为“沙袋角”。《普陀地名志》另记载一说：[[阜丰面粉厂|阜丰面粉厂]]兴建后，工人每日接触面粉袋、袋装面粉，两角叉开，形象河曲之角，乃将沙袋角改称为叉袋角。<ref name="普陀">{{cite book |author1=普陀区人民政府 |title=上海市普陀区地名志 |publisher=[[学林出版社|学林出版社]] |isbn=7-80510-137-X |accessdate=2018-09-05}}</ref>

== 地理形势 ==
该地南北有吴淞江两大河曲，曲度分别为90度和60度，北部河曲为凸出的三角半岛状，南部河曲为凹进的河湾。此处是明代中叶，吴淞江下游改道宋家浜的转折处。<ref name="普陀" />

== 西岸 ==
1896年，[[孙多森|孙多森]]、[[孙多鑫|孙多鑫]]在叉袋角创办[[阜丰面粉厂|阜丰面粉厂]]，是中国第一家机器制粉厂。1912年，无锡荣家二兄弟[[荣宗敬|荣宗敬]]、[[荣德生|荣德生]]也在相邻处创办[[福新面粉厂|福新面粉厂]]，二厂后并为[[阜丰福新面粉厂|阜丰福新面粉厂]]，即公私合营后的[[上海面粉厂|上海面粉厂]]<ref>{{cite book|author=薛理勇|isbn=9787560837369|title=街道背后：海上地名寻踪（续篇）|year=2008|publisher=[[同济大学出版社|同济大学出版社]]|location=上海|pp=174-176}}</ref>。今面粉厂已经作为文物进行保护<ref>{{cite web|language =|publisher =上海市文化广播影视管理局|title =普陀区县级文物保护单位、登记不可移动文物|url =http://wgj.sh.gov.cn/wgj/node1257/node1264/userobject1ai75816.html|author =上海市文广影视局|date =2011-11-08|accessdate =|deadurl =yes|archiveurl =https://web.archive.org/web/20140202092414/http://wgj.sh.gov.cn/wgj/node1257/node1264/userobject1ai75816.html|archivedate =2014-02-02}}</ref>。

== 东岸 ==
1909年，为抵御[[上海公共租界|租界]]渗透，[[两江总督|两江总督]][[张人骏|张人骏]]上书：“上海北市地方市面日渐繁盛，汲水不便……”后获准，授权[[李平书|李平书]]筹办[[闸北水电公司|闸北水电公司]]。于是，购地16亩于上海火车站西叉袋角一带，1911年竣工、同年9月27日落成，向闸北华界供水供电。1913年因债务纠纷改为官办，易名江苏省闸北水电公司；后因管理不善，1924年复为商办，易名江苏省立闸北水电股份有限公司。是时，营业区域为东北沿黄浦江至张华浜，西南沿苏州河至陈家渡，东南至公共租界，西北至彭浦、江湾。[[一二八事变|一二八事变]]后，日军入侵上海，华界地区遭轰炸炮击，水、电厂部分被毁，后修复恢复供水供电，受嘉奖。[[八一三事变|八一三事变]]后，日军再次入侵上海，闸北华界又遭日军轰炸炮击，位于叉袋角公司水电厂全毁<ref>{{cite book|author1=上海市闸北区志编纂委员会 |title=闸北区志 |url=http://shtong.gov.cn/newsite/node2/node4/node2249/zabei/node39114/node39155/node63294/userobject1ai24396.html|isbn=7-80618-439-2|date=1998.3|location=上海|publisher=上海社会科学院出版社 |accessdate=2018-06-09}}</ref>。

== 轮渡 ==
当地曾设置渡口，称为通济路船渡（1927年前称叉袋角义渡），南岸位于西苏州河路217号，北岸为光复西路1231号（通济路口）。1927年，承办人获准正式营业。1937年抗日战争爆发被迫停航，战争胜利后复航。1956年5月1日以资产入股方式并入[[上海轮渡|上海轮渡]]，改用钢质渡船、钢踏步，后于1984年因修筑防汛墙而停航撤销<ref>{{cite book |author1=上海内河航运志编纂委员会编 |coauthors=上海市地方志办公室 |title=上海内河航运志 |date=1999.6 |publisher=上海社会科学院出版社 |location=上海 |isbn=7-80618-597-6 |url=http://shtong.gov.cn/newsite/node2/node2245/node4501/node56018/node56020/userobject1ai43388.html |accessdate=2018-06-09}}</ref>。
