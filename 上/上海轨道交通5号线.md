'''上海轨道交通5号线'''，又名'''莘闵轻轨'''、'''莘闵线'''，是[[上海轨道交通|上海轨道交通]]第4条建成通车的线路。

== 概览 ==
5号线原名上海市莘闵轻轨交通线工程，由上海申通集团有限公司、上海闵行城市建设投资开发有限公司、上海闵行联合发展有限公司组建的上海莘闵轨道交通线发展有限公司投资<ref>{{cite journal |author1=上海年鉴社 |journal=上海年鉴 |title=闵行区 |date=2003 |url=http://www.shtong.gov.cn/Newsite/node2/node19828/node70632/node70685/node70834/userobject1ai72907.html |accessdate=2018-09-01 |issn=1008-1046}}</ref>。线路由闵行区政府提出筹建，是唯一一条由区政府（闵行区）投资建设的轨交线路。最初由上海现代轨道交通股份有限公司负责运营，该公司的设立是上海市轨道交通体制改革的成果，是落实市有关部门关于轨道交通投资、建设、运营、监管纵向“四分开”和横向适度竞争原则的重大举措。目前5号线已改为上海地铁第一运营有限公司运营。

相对于上海其他轨道交通线路，部分列车车辆编组只有每列4辆。采用的法国阿尔斯通列车（05C01）是首次由上海本地组装制造。在非南延伸段区间乘坐起步票价2元，超出该段的均为3元起步。线路标志色为{{space}}<span style="background-color:#{{上海地铁标志色|5}}"><font color=white>PANTONE 2582C</font></span>{{space}}（C=49 M=78 Y=0 K=0）<ref name=R1a_color>{{cite web | title = 《上海轨道交通网络近期规划建设线路标志色方案》征求市民意见公告| publisher = 上海地铁 | date = 2011-09-09 | url = http://www.shmetro.com/node49/201109/con109210.htm | accessdate = 2018-09-21 | language = zh }}</ref>。

=== 线路数据 ===
* 营运里程：37.376km
* 站数：19
* 轨距：1435mm
* 动力电源：1500V直流（架空电线方式）
* 复线区间：全线
* 电气化区间：全线
* 高架区间：[[莘庄站|莘庄]]—[[奉浦大道站|奉浦大道]]/[[闵行开发区站|闵行开发区]]
* 地下区间：[[奉浦大道站|奉浦大道]]—[[奉贤新城站|奉贤新城]]

== 工程建设 ==
5号线原为规划R1线的南段，南起现在的[[金山卫站|金山卫站]]，[[莘庄站|莘庄站]]以南沿沪杭公路、奉柘公路、江海路、南桥路、沪杭公路、沪闵路行驶，并规划沿东川路的支线，现在的5号线即为其中的沪闵路段和沪杭公路段一部分以及支线。但[[闵行区|闵行区]]希望尽早建成该段线路，故以“城市高架有轨电车莘闵线工程”<ref>{{Cite web |url=http://www.shucm.sh.cn/gb/jsjt2009/zdgc/node1563/userobject1ai84660.html |title=2001年重大工程一览表 |access-date=2018-03-30 |archive-url=https://web.archive.org/web/20120412024258/http://www.shucm.sh.cn/gb/jsjt2009/zdgc/node1563/userobject1ai84660.html |archive-date=2012-04-12 |dead-url=yes }}</ref>为名于2000年8月8日提前开工建设并独立运营。莘闵轻轨北起[[闵行区|闵行区]][[莘庄站|莘庄站]]，沿沪闵路往南至交通大学附近折向东川路。最初计划止于[[金平路站|金平路站]]<ref>{{Cite web |url=http://www.shucm.sh.cn/gb/jsjt2009/zdgc/node1561/userobject1ai84618.html |title=1999年重大工程介绍—轨道交通 |access-date=2018-03-30 |archive-url=https://web.archive.org/web/20120412101551/http://www.shucm.sh.cn/gb/jsjt2009/zdgc/node1561/userobject1ai84618.html |archive-date=2012-04-12 |dead-url=yes }}</ref>，后改为至[[闵行开发区站|闵行开发区站]]，全长17.206公里。除起点段的410米地面线外，均为高架线路。全线设11座车站和剑川路停车场1座。除莘庄站为地面站外，其余均为高架车站<ref>{{cite web|author=author,email address |url=http://www.shanghai.gov.cn/nw2/nw2314/nw24651/nw13107/nw13624/u21aw87734.html |title=轨道交通五号线全线贯通 |publisher=Shanghai.gov.cn |date= |accessdate=2018-05-06}}</ref>。工程总投资34.37亿元。线路于2002年12月28日建成贯通，并于2003年11月25日投入试运行。

配合[[闵浦二桥|闵浦二桥]]的建设，南延伸[[西渡站|西渡站]]与[[江川路站|江川路站]]的车站结构，以及西渡至沪闵路东川路南侧（距预留分岔约1公里处）的轨道已作为“沪闵路-沪杭公路地方交通越江工程调整”先行开工建设。“沪闵路-沪杭公路地方交通越江工程”简称为闵浦二桥，它北起闵行区沪闵路东川路以北，南至奉贤区沪杭公路西闸路以南，全长为4.73公里。工程总体设计方案为公路与轨道交通上下叠合的布置形式，上层为公路双向四车道，下层为轨道交通5号线。大桥公路部分于2010年5月21日通车。

[[File:Tracks_near_Dongchuan_Road_Station.jpg|thumb]]
[[File:20180331正在改建中的5号线折返段及建设中的莘庄辅助停车场.jpg|thumb]]
南延伸段原为规划R1线的东川路以南段。为了方便[[奉贤区|奉贤区]]人民出行，经上海市政府的协商，原本闵行区政府投资建造的区内轨交决定跨越黄浦江延伸至奉贤区。南延伸工程线路长约19.628公里，按规划要求预留继续向南延伸的条件。地下线长约7.747公里，敞开段约0.668公里，高架线约11.213公里，设地下站4座、高架站5座，另新建平庄公路车辆基地和莘庄停车场各一座。线路自[[东川路站|东川路站]]以北的预留分岔引出，沿[[沪闵路|沪闵路]]，经闵浦二桥过江后沿沪杭公路至西渡，再经团南路折向东，过[[莘奉金高速公路|S4高速公路]]，至金海路向南，跨[[上海绕城高速公路|G1501高速公路]]后，沿金海路至[[平庄公路站|平庄公路站]]。除了新建车站之外，此工程还包括[[东川路站|东川路站]]的改造，在现站台两侧各添加一条轨道，使站台由[[側式月台|侧式站台]]变为[[雙島式月台|双岛式站台]]。其中工程总投资106.19亿元。于2014年6月30日以“西渡站-环城北路区间高架桩基工程”为名开工，于2018年12月30日建成通车。

由于5号线原线车站只能容纳4节编组车辆，考虑到南延伸段车站为6节编组设计，需要对原车站进行改造，以适应开通后的客流增加。既有设施改造工程包括改建5号线莘庄站、[[春申路站|春申路站]]、[[银都路站|银都路站]]、[[颛桥站|颛桥站]]、[[北桥站|北桥站]]、[[剑川路站|剑川路站]]等6座车站，将有效站台长度从80米增加到120米。改造莘庄站站后[[折返线|折返线]]，新建莘庄停车场出入段线，以及与[[上海轨道交通1号线|1号线]]的[[联络线|联络线]]。改造相关的供电、通信、通风空调、给排水消防等系统，并结合环保要求在部分区段设置[[隔音屏障|隔音屏障]]。工程总投资2.32亿元。

== 运营组织==
2018年10月20日起，5号线进入信号改造施工二阶段后将同步启用新的列车运行图。莘庄站～东川路站在完成站台“4改6”扩展改造工程及信号系统升级改造后，首次全部开行新购入的6节编组列车，而东川路站～闵行开发区站将继续开行4节编组列车。运营从大小交路方式调整为分段运营方式，乘客需在东川路站进行“4/6”节编组列车的双向换乘，具体为：往闵行开发区站的乘客在东川路站下车，同站台换乘4节编组列车；反之，乘客抵达东川路站后，换乘往莘庄站方向的6节编组列车<ref>[http://www.shmetro.com/node49/201810/con115127.htm 5号线10月20日六编组列车首次投运]</ref>。

== 争议 ==
为有效控制列车开行噪音的影响，在不影响总体5号线运能的前提下，在东川路站—西渡站约2公里区段内，列车将限速20km/h运行<ref>[http://mini.eastday.com/a/181230051757672.html 5号线延伸段开通后局部区段限速运行]</ref>。至于为何在江面上也需要限速，并没有答复。

== 已取消的规划 ==
5号线北段由[[莘庄站|莘庄站]]至[[虹桥火车站站|虹桥火车站站]]，走向为自莘庄站向西北折入七莘路，向北进入虹桥枢纽，全长14.3公里，计划设11站，被列入2010~2020年的建设规划，同时在《上海市城市快速轨道交通近期建设规划(2010-2020年)》中，将北段列为“新建线”，而不是类似南延伸的“延伸线”。根据最新规划，5号线北段将并入[[上海轨道交通17号线_(2010年前)|原17号线]]<ref name="r1"/>。但根据2016年公布的《上海市轨道交通近期建设规划（2017-2025）》，该线路由[[上海轨道交通嘉闵线|嘉闵线]]代替。
