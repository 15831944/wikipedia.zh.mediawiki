'''上饶县'''是[[中国|中国]][[江西省|江西省]][[上饶市|上饶市]]所辖的一个[[县|县]]，位于江西省东北部，东与信州区、广丰区隔丰溪河为界，东北为玉山县界，西挨铅山县和横峰县，北临德兴市。南连福建省南平市浦城县、武夷山市。全县南北最长132千米，东西宽45千米，总面积2240平方千米，其中城区面积10.9平方千米。

== 历史沿革 ==
古属[[扬州_(古代)|扬州]]，[[春秋时期|春秋]][[戰國|战国]]先后分属[[吴国|吴]]、[[越國|越]]、[[楚国|楚]]。
[[秦朝|秦]]属[[九江郡|九江郡]]（治所今[[安徽省|安徽省]][[淮南市|淮南市]][[壽春縣|寿春县]]）。
[[汉朝|汉]]属[[豫章郡|豫章郡]]，[[东汉|东汉]][[建安_(东汉)|建安]]初（196年—204年），吴析余汗县东境建置上饶县，以山郁珍奇、上乘富饶得名。
[[西晋|西晋]][[元康_(西晋)|元康]]元年（291年）并入葛阳县。
[[南北朝|南朝]][[刘宋|宋]]年间（420年—479年）复置县，属鄱阳郡。[[隋朝|隋]][[开皇|开皇]]九年（589年）复并入葛阳县，十二年（592年）改名[[弋阳县|弋阳县]]。县地随弋阳隶[[鄱阳郡|鄱阳郡]]。 
[[唐朝|唐]][[武德|武德]]四年（621年）重置县，隶[[饶州_(隋朝)|饶州]]。七年（624年），再次并入弋阳县。又并入弋阳县。乾元元年（758年）析饶州置信州，析弋阳复置上饶县，隶属信州，为州治所。
[[宋朝|宋]][[开宝|开宝]]八年（975年），县属[[江南路|江南路]]，[[信州_(江西)|信州]]属信州郡，为郡治所。
[[元朝|元]][[至元_(忽必烈)|至元]]十四年（1277年），县属江浙行中书省信州路，为路治所。
[[明朝|明]][[洪武|洪武]]三年（1370年），信州路改称[[广信府|广信府]]，上饶县隶之，为府治。四年，广信府改属江西行中书省，县隶之。
[[清朝|清]]属广信府，为府治所（今上饶市信州区金龙岗），[[道光|道光]]十二年（1841年）改属[[豫章道|豫章道]]。
[[中華民國|民国]]元年（1912年）冬，废府设道，上饶县隶豫章道。民国十五年废道，隶于省直辖。民国二十一年（1932年）江西省分设为十三个行政区，二十四年为八个行政区，三十一年为九个行政区，上饶县均属第六行政区，治上饶。[[第一次国共内战|第二次国内革命战争]]时期（1930—1934年），置上饶县苏维埃政府，隶属闽浙赣省苏维埃政府。江西省第六行政区。
1949年5月3日，[[中国人民解放军|中国人民解放军]]解放上饶，5月14日成立上饶县人民政府，属赣东北行政区；同日析县治广平镇设立上饶市，隶赣东北行政区，9月隶上饶专区。1960年3月撤销上饶县建置并入上饶市，1964年3月恢复县置。1968年4月12日，成立上饶县革命委员会，取代县委、县人民政府一切权力。1980年12月，撤销县革命委员会，恢复县人民政府建制。2000年10月18日，上饶地区撤地设市，县隶上饶市。
== 行政区划 ==
上饶县下辖3个街道11个镇10个乡。
街道：旭日街道、罗桥街道、兴园街道
镇：田墩镇、花厅镇、皂头镇、上泸镇、四十八镇、煌固镇、华坛山镇、郑坊镇、五府山镇、茶亭镇、枫岭头镇）
乡：清水乡、石人乡、石狮乡、尊桥乡、铁山乡、黄沙岭乡、湖村乡、望仙乡、应家乡

== 地理 ==
县境中山、低山、[[丘陵|丘陵]]与河谷平原从南北两端向中部呈阶梯状递降，大致平行于[[信江|信江]]对称分布，明显构成南北高、中部低的马鞍状地形。南北均为高大的山脉，北部为[[怀玉山脉|怀玉山脉]]，南部为[[武夷山脉|武夷山脉]]，中部信江横贯。
中山分布在县境南部和北部，包括[[五府山|五府山]]、[[灵山_(上饶)|灵山]]、华坛山等，占全县土地总面积36.2%，海拔1000—1800米，南部最高点五府岗海拔1891.4米。北部最高点灵山天梯峰海拔1496米。灵山山峰切割强烈，[[瀑布|瀑布]]较多。低山主要分布在上泸、四十八、郑家坊一带，占全县总面积13.1%，海拔500—1000米。地形兼有中山与丘陵的特征，地表溶沟、溶槽、石芽多见，有地下溶孔、[[溶洞|溶洞]]和[[伏流|地下河]]。
丘陵低丘主要分布在县境中部信江两侧，占全县总面积48.8%，海拔100—500米，多为丹霞地貌，有月岩、南岩、七峰岩等洞穴奇观。县内河谷平原呈长条状分布于信江两岸，宽处达4000—5000米，海拔50—70米，占全县总面积1.9%，主要由河漫滩和河流阶地组成，属侵蚀堆积地貌。县境著名山川是灵山，著名河流是信江。

== 风景名胜 ==
北有灵山，南有[[五府山|五府山]]。七峰岩在“[[皖南事变|皖南事变]]”中关押过[[新四军|新四军]]军长[[叶挺|叶挺]]将军。

北有[[茗洋关水库|茗洋关水库]]，南有[[大坳水库|大坳水库]]，[[下会坑水库|下会坑水库]]。
