'''上海开放大学'''（'''上海远程教育集团'''），原名'''上海电视大学'''，是位于[[上海市|上海市]]的一所公办[[成人高等院校|成人高等院校]]，主要通过远程教育等方式，面向成人开展本专科教育<ref name=xxjj>{{cite web|url=http://www.sou.edu.cn/2017/0119/c3859a6621/page.htm|title=学校简介|publisher=上海开放大学|access-date=2017-09-19|archive-url=https://web.archive.org/web/20170925013206/http://www.sou.edu.cn/2017/0119/c3859a6621/page.htm|archive-date=2017-09-25|dead-url=no}}</ref>。

== 校史 ==
1960年4月6日，[[复旦大学|复旦大学]]、[[华东师范大学|华东师范大学]]、[[华东化工学院|华东化工学院]]等位于上海的高校和[[上海人民广播电台|上海人民广播电台]]联合成立上海电视大学，该校是中国大陆最早的广播电视大学<ref name="gyz" />，隶属于上海市高等教育局，时任华东师范大学党委书记、副校长[[常溪萍|常溪萍]]兼任首任校长；时任复旦大学副校长[[苏步青|苏步青]]、华东化工学院院长[[张江树|张江树]]、上海市总工会副主席沈涵和上海市教育局副局长[[马飞海|马飞海]]兼任副校长，总部设在[[上海永安公司大楼|永安大楼]]五楼。该校建立前，已经于3月招收学生17940人；而在学校创设初期，该校陆续开设了数学、物理、化学、中文等专业，并明确让部分课程的结业水平对标大学本科毕业水平。全上海市共建立了28个电大辅导站、576个收看点，公开招收学员，并通过自学、广播、面授辅导方式进行教学。1965年，该校招收第二届5218名新生。该校第一届学员于1966年毕业，获得的文凭于1981年被追认为有效的大学本科文凭。1966年6月，学校因文化大革命爆发，被迫停办<ref name="crjyz">{{cite book |author1=《上海成人教育志》编纂委员会 |title=上海成人教育志 |date=2007-05 |publisher=上海社会科学院出版社 |location=上海 |isbn=978-7-80745-013-9 |url=http://www.shtong.gov.cn/Newsite/node2/node2245/node82368/node82387/node82406/userobject1ai111853.html |accessdate=2019-01-28 |archive-url=https://web.archive.org/web/20170227194518/http://www.shtong.gov.cn/Newsite/node2/node2245/node82368/node82387/node82406/userobject1ai111853.html |archive-date=2017-02-27 |dead-url=no }}</ref>。

1978年3月，上海市革命委员会批准复办电视大学，举办成人大专教育，总部依旧设在永安大楼五楼内，和上海电视台社教组共用办公地。当年，上海电视大学录取了新生7707人。1979年2月，上海电大与上海业余工业大学（今[[上海第二工业大学|上海第二工业大学]]）联合办学，学生规模一度占到上海大学生总数的三分之一，弥补了文化大革命造成的人才和知识断层，较大提升了上海教育系统内教师的学历层次；1979年5月，学校迁入陕西北路80号，和业余工大合署办公。1983年7月，上海电大与市业余工大分开，校址迁至控江路1130号。1984年7月，电大校址和杨浦区业余大学交换校址，又迁至鞍山七村24号（今阜新路25号）<ref name="bainian">{{cite book |author1=杨浦区史志编纂办公室 |author2=杨浦区政协教育和文化委员会 |title=百年大学看杨浦 |date=2003 |publisher=上海高教电子音像出版社 |location=上海 |isbn=7-88392-271-8 |url=http://www.shtong.gov.cn/newsite/node2/node4429/n107749/n95272/n95285/index.html |accessdate=2019-01-29}}</ref>。1987年7月1日，[[上海电视台|上海电视台]]正式启用有线电视26频道，专门用于播出电视大学的教育节目；1989年开始，该校试点招收“[[往届生|往届生]]”；1994年，放送电大教育节目的26频道划拨给电视大学下设的[[上海教育电视台|上海教育电视台]]<ref>{{cite book |author1=《上海广播电视志》编辑委员会编 |title=上海广播电视志 |publisher=上海社会科学院出版社 |location=上海 |isbn=7-80618-658-1 |url=http://www.shtong.gov.cn/newsite/node2/node2245/node4510/node11505/node11512/node63816/userobject1ai12553.html |accessdate=2019-01-28 |archive-url=https://web.archive.org/web/20160427213230/http://www.shtong.gov.cn/Newsite/node2/node2245/node4510/node11505/node11512/node63816/userobject1ai12553.html |archive-date=2016-04-27 |dead-url=no }}</ref>。

上海电视大学自1980年代末以来，生源锐减。为了学校发展，电视大学开设了计算机应用、英语、金融等专科升本科（专升本）课程，并广泛投入社会培训之中，首先提出并开展了“上海市计算机应用能力考核”项目。1996年，依托[[中央电大|中央电大]]的资源，举办专升本教育。1998年，独立举办开放专科教育，考试权下放给上海市教委，采用“免试入学、宽进严出”的模式，实施学分制管理<ref name="gyz">{{cite book |author1=《上海工运志》编纂委员会 |title=上海工运志 |date=1997-09 |publisher=上海社会科学院出版社 |location=上海 |isbn=7-80618-351-5 |url=http://www.shtong.gov.cn/Newsite/node2/node2245/node4471/node56291/node56302/node56304/userobject1ai42883.html |accessdate=2019-01-28 |archive-url=https://web.archive.org/web/20160327113916/http://www.shtong.gov.cn/Newsite/node2/node2245/node4471/node56291/node56302/node56304/userobject1ai42883.html |archive-date=2016-03-27 |dead-url=no }}</ref><ref name="crjyz" />。

2000年1月24日，经上海市人民政府批准，上海电视大学吸收上海教育电视台、上海电视中等专科学校和上海市电化教育馆，联合组建上海远程教育集团<ref name="crjyz" />。2002年，上海市教委同意将华东师范大学国顺路校区（原上海教育学院）有偿划转上海电视大学<ref>{{cite web |title=大事记（5） |url=http://www.shtvu.edu.cn/introduce/dsj.asp?page=5 |website=上海电视大学 |accessdate=2019-01-29 |archiveurl=https://web.archive.org/web/20070103122146/http://www.shtvu.edu.cn/introduce/dsj.asp?page=5 |archivedate=2007-01-03}}</ref>。2004年9月，上海电视大学将其校训定为“有教无类，乐学致远”<ref>{{cite web |title=大事记（2） |url=http://www.shtvu.edu.cn/introduce/dsj.asp?page=2 |website=上海电视大学 |accessdate=2019-01-29 |archiveurl=https://web.archive.org/web/20070103192818/http://www.shtvu.edu.cn/introduce/dsj.asp?page=2 |archivedate=2007-01-03}}</ref>。同年11月，上海电视大学国顺路校区新教学楼竣工投用<ref>{{cite web |title=大事记（1） |archiveurl=https://web.archive.org/web/20070104012623/http://www.shtvu.edu.cn/introduce/dsj.asp?page=1 |website=上海电视大学 |accessdate=2019-01-29 |url=http://www.shtvu.edu.cn/introduce/dsj.asp?page=1 |archivedate=2007-01-04 }}</ref>。

2010年10月，国务院办公厅印发《关于开展国家教育体制改革试点的通知》，明确将上海电视大学列入开放大学试点<ref>{{cite web|url=http://www.moe.gov.cn/jyb_xxgk/xxgk_jyta/jyta_ghs/201812/t20181225_364813.html|date=2018-12-25|title=教育部对十三届全国人大一次会议第5120号建议的答复|website=中华人民共和国教育部|accessdate=2019-01-28}}</ref>。2012年6月，[[中华人民共和国教育部|教育部]]发布教发函〔2012〕105号，批准上海电视大学更名为上海开放大学，撤销原上海电大建制<ref>{{cite news |author1=张骞 |title=上海电视大学正式更名为“上海开放大学” |url=http://sh.eastday.com/m/20120814/u1a6784072.html |work=东方网上海频道（据新民晚报） |date=2012-08-14}}</ref>，并承办上海市终身教育学分银行<ref>{{cite web |author1=中华人民共和国教育部 |title=教育部关于同意上海电视大学更名为上海开放大学的批复 |url=http://kf.fjrtvu.edu.cn/info/1012/1468.htm |website=[[福建广播电视大学|福建广播电视大学]] |accessdate=2019-01-29 |date=2012-06-21 |archive-url=https://web.archive.org/web/20190108043206/http://kf.fjrtvu.edu.cn/info/1012/1468.htm |archive-date=2019-01-08 |dead-url=no }}</ref>；2014年，学校被上海市学位委员会增列为[[学士|学士]]学位授予单位<ref>{{cite web|url=http://www.sou.edu.cn/2017/0123/c3860a11929/page.htm|title=历史沿革|publisher=上海开放大学|access-date=2017-09-19|archive-url=https://web.archive.org/web/20171030032019/http://www.sou.edu.cn/2017/0123/c3860a11929/page.htm|archive-date=2017-10-30|dead-url=no}}</ref>。

== 院系设置 ==
学校设有公共管理学院、经济管理学院、理工学院和人文学院4个二级学院，开设46个专业，在上海市设有40所分校、教学点<ref name=xxjj /><ref name="zzjg">{{cite web|url=http://www.sou.edu.cn/2017/0119/c3862a6623/page.htm|title=组织结构|publisher=上海开放大学|access-date=2017-09-19|archive-url=https://web.archive.org/web/20171030032004/http://www.sou.edu.cn/2017/0119/c3862a6623/page.htm|archive-date=2017-10-30|dead-url=no}}</ref>。其中，国顺路校区、阜新路校区、中原路校区是该校的三个直属校区<ref name="bainian" />，其余均为挂牌校区或分校。
