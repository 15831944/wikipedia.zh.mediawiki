'''上海航空公司'''，原名'''上海航空公司'''、'''上海航空股份公司'''，简称'''上航'''，成立於1985年12月30日，是中國大陸改革開放後，首批以地方政府資金支持創立的航空公司之一，亦是首批完全獨立於當時國營民航系統的航空運輸服務營辦商。基地位于[[上海虹桥国际机场|上海虹桥国际机场]]。2002年10月11日，上海航空在[[上海证券交易所|上海证券交易所]]上市。

上海航空成立的初衷是为了实现[[中国大陆|中国大陆]]与[[台湾|台湾]]的「[[三通|三通]]」，避免在飞机机身上出现国名和国旗而引起不必要的困扰。<ref>范鸿喜的回忆，刊于《世纪》杂志2010年第1期。</ref>

尽管日后两次变更全名，但由[[汪道涵|汪道涵]]书写的上海航空公司中文标识和SHANGHAI AIRLINES英文标识却一直保留。

== 歷史 ==
[[File:Boeing_707-324C,_Shanghai_Airlines_AN0199408.jpg|缩略图]]
1980年代初，随着上海经济的飞速发展和对外开放，旅游事业日趋兴旺，上海[[虹桥机场|虹桥机场]]的旅客吞吐量不断增长，上海民航的客运能力无法满足当时的需求，当时出现了“乘机难”的问题，并成了上海经济发展和对外开放的一个制约因素。1984年起，[[中共上海市委|中共上海市委]]、[[上海市人民政府|上海市人民政府]]领导与市人民政府交通办公室（现[[上海市交通委员会|上海市交通委员会]]）的领导，开始组建上海航空公司。8月13日，[[中华人民共和国国务院总理|中华人民共和国国务院总理]][[赵紫阳|赵紫阳]]在[[北戴河|北戴河]]主持召开的[[中央财经领导小组|中央财经领导小组]]会议上，在听取[[陈国栋|陈国栋]]、[[汪道涵|汪道涵]]汇报了上海经济工作情况以后指出，民航事业发展的根本的出路在于政企分开，多家经营，并表示上海可以成立航空公司。10月10日，上海市人民政府交通办公室提出了3条意见：建议建立上海航空公司，作为独立经济核算企业，并应有自主经营权；以租赁或购买的方式增加客货运输机，以利迅速扩大运力；资金除国家拨款以外，可以自筹、银行贷款和国外贷款<ref name="shtong1">{{cite news|title=新中国建立后上海民航机构 |url=http://shtong.gov.cn/newsite/node2/node2245/node68464/node68469/node68493/userobject1ai66135.html |publisher=上海市地方志办公室 |date= |accessdate=2015年10月17日 }}</ref>。

1985年2月19日，上海市政府交通办副主任[[贺彭年|贺彭年]]，应上海市市长汪道涵召见，受命筹建上海航空公司。3月6日，上航筹备组成立，筹备组由[[贺彭年|贺彭年]]、[[沈兆康|沈兆康]]等6人组成，贺彭年任组长；8日，上海市人民政府将《关于成立“上航”的请示》报送[[中华人民共和国国务院|中华人民共和国国务院]]。4月3日，上航筹备组向美国购买5架[[波音707|波音707-300]]型客机，由上海市对外贸易总公司[[杨如棠|杨如棠]]在纽约签署合同。4月7日，上海市人民政府向国务院报送《关于“上航”筹备情况的报告》，并在同月11日，时任国务院副总理[[李鹏|李鹏]]在主持召开国务院第二次民航工作会议上，原则同意上海组建航空公司。5月14日，[[民航总局|民航总局]]下发《关于同意组建“上航”的批复》，并由上海市市长汪道涵书写“上海航空公司”六字。同年6月，首2架上航波音707型飞机先后飞抵虹桥机场。8月8日，上航董事会正式成立，贺彭年任董事长兼总经理。11月3日，第5架上航波音707型飞机抵沪，至此五架上航波音707客机全部运抵上海。当时上航还没有自己的停机坪，故这些飞机临时停放在[[大场机场|大场机场]]和虹桥机场。同年12月，上航先后获得[[民航总局|民航总局]]的航空运输企业经营许可证和[[国家工商行政管理局|国家工商行政管理局]]的营业执照。1985年12月30日，上海航空公司成立<ref name="shtong1"/>。

1986年1月11日，上航首次成功试飞上海至北京航线。4月上旬，上航首次执行上海至北京、广州、杭州不定期客货运航班，并完成了首次用波音707型包机运送150名上海外贸系统代表参加[[广交会|广交会]]，安全飞抵广州的任务。同年年底，上航第一期13500平方米停机坪工程竣工，结束了上航有飞机无停机坪的历史，并正式开通上海至北京、上海至广州的航线<ref name="shtong1"/>。

1988年5月31日，时任上海市市长[[朱镕基|朱镕基]]召开“上航汇报工作会议”，贺彭年作了《关于“上航”租赁3架波音757型飞机可行性研究报告》。同年上海市人民政府正式发文批准“上航”以租赁方式更新飞机，并获得[[国家计委|国家计委]]的批准，同时明确由[[中国银行|中国银行]]上海分行负责担保<ref name="shtong1"/>。

1989年，上海航空向波音引進一批[[波音757|波音757]]雙發動機客機，用以取代老舊的波音707，第一架于8月1日运抵<ref name="shtong1"/>。为方便旅客出行，上航售票处首创销售往返机票，并提供任何一个售票点可以购买上航任何一条航线机票的便民措施。截至1989年底，上航的2架波音757型飞机共飞行航班518班次，飞行884小时，运送旅客5183人次，运输货物1020吨<ref name="shtong1"/>。

1990年8月7日，第三架波音757型飞机抵沪，并获得民航总局颁发的适航证和国籍证。同年，上航又新开辟上海至福州、桂林、厦门及广州至南京、北京至南京航线<ref name="shtong1"/>。1991年8月，上海航空集团公司正式成立，该公司成员企业除包括上航外，还拥有上海航空设备厂、上海飞鹤国际旅游公司、上海交通技术公司、上海航空广告公司、上海千鹤宾馆、大通空运（中国）有限公司上海分公司、中国海洋直升机专业公司上海公司、上海航空贸易服务公司等10个企业<ref name="shtong1"/>。1994年，上航与[[日本航空|日本航空]]在日航龙柏饭店举行“派遣乘务员研修合同”签字仪式，通过派遣乘务员培养乘务员骨干，以达到提高客舱服务水平。同年7月，上航引进第一架[[波音767|波音767]]飞机<ref name="shtong1"/>。

1998年5月上航开辟了首条国际航线，即南京--澳门（当时澳门仍属于[[葡萄牙|葡萄牙]]）的定期地区航线，随后又开辟了上海--澳门的定期地区航线。並在2000年11月從地方政府企業改革成為由國家及地方所設的投資公司所控股的股份制有限公司——上海航空股份有限公司<ref>{{cite news|title=上航首次公开发行股票招股说明书摘要 |url=http://money.163.com/editor/020923/020923_108506.html |publisher=网易 |date=2002年9月23日 |accessdate=2015年10月17日 }}</ref>，2002年10月11日在上海證券交易所的A股市場上市<ref>{{cite news|title=上海航空股份有限公司上市公告书|url=http://www.cnstock.com/ssnews/2002-9-28/qitaban/200209280283.htm|publisher=中国证券网|date=2002-09-28|accessdate=2015年10月17日|deadurl=yes|archiveurl=https://web.archive.org/web/20081101212601/http://www.cnstock.com/ssnews/2002-9-28/qitaban/200209280283.htm|archivedate=2008年11月1日}}</ref>。

2006年6月30日，由上航、盛航企业有限公司、捷航企业有限公司共同投资组建的上海國際貨運航空正式營運，經營貨運業務<ref>{{cite news|title=2007上海年鉴：（九）航空运输 |url=http://www.shtong.gov.cn/newsite/node2/node19828/node78865/node78896/node78981/userobject1ai100011.html|publisher=上海市地方志办公室 |date=|accessdate=2015年10月17日 }}</ref>。2006年4月11日，上海航空签署《议定书》，成为[[星空联盟|星空联盟]]观察员<ref>{{cite news|title=告别单打独斗 上海航空“飞往”星空联盟 |url=http://finance.people.com.cn/GB/1039/4363395.html|publisher=东方早报 |date= 2006年5月11日 |accessdate=2015年10月17日 }}</ref>，并在2007年12月12日成為星空聯盟的正式成員<ref>{{cite news|title=2008上海年鉴：(九)航空运输 |url=http://www.shtong.gov.cn/newsite/node2/node19828/node81582/node81663/node81665/userobject1ai106957.html|publisher=上海市地方志办公室 |date=  |accessdate=2015年10月17日 }}</ref>。

2009年6月6日，上海航空与[[中国东方航空公司|中国东方航空公司]]获得政府许可，获准合并。东航采取换股的方式进行吸收合并上航，上海航空与东方航空的换股比例为1:1.3，即每股上海航空股份可换取1.3股东方航空的股份。为此东航拟分别非公开发行不超过13.5亿股A股和不超过4.9亿股H股，融资总额约70亿。合并后，上海航空品牌依然保留<ref> {{cite web | title = 东航换股吸收合并上航 13日双双复牌 | publisher = 中国证券网-上海证券报 | date = 2009年7月13日| url=http://finance.sina.com.cn/chanjing/gsnews/20090713/02526470415.shtml | accessdate = 2015年10月17日}} </ref>。由于东方航空於2011年加入[[天合聯盟|天合聯盟]]，上海航空于2010年10月30日退出星空联盟<ref> {{cite web | title = 上海航空退出星空联盟 | publisher = 上海航空 | date = 2010-05-24 | url = http://ffp.shanghai-air.com/News/wfrmShowInfo.aspx?id=2925191 | accessdate = 2010-07-31 | language = zh-cn | deadurl = yes | archiveurl = https://archive.is/20100805070424/http://ffp.shanghai-air.com/News/wfrmShowInfo.aspx?id=2925191 | archivedate = 2010-08-05 }} </ref>，并与东方航空一起加入天合联盟。

2010年3月16日，上海航空有限公司成立；12月16日，原上海航空股份有限公司被核准注销；由上海航空有限公司继承原上海航空股份有限公司的债权和债务。<ref> {{cite web | title = 东方航空预计去年净利润同比增约10倍 | publisher = 中国证券网(原载一财网) | date = 2011-01-04 | url = http://cnstock.xinhuanet.com/index/gdxw/201101/1084031.htm | accessdate = 2011-01-05 | language = zh-cn | deadurl = yes | archiveurl = https://archive.is/20120708150636/http://cnstock.xinhuanet.com/index/gdxw/201101/1084031.htm | archivedate = 2012-07-08 }} </ref>

2011年5月31日，上海國際貨運航空、[[长城航空|长城航空]]货运部门与[[中国货运航空|中国货运航空]]重组，并组建新的中国货运航空公司<ref>{{cite news|title=中货航正式挂牌成立|url=http://www.ckair.com/about_us/companyNews/004DAE731500000000203.html|publisher=中国货运航空 |date=2011-06-02  |accessdate=2015年10月17日 }}</ref>。

== 语言 ==
与大多数航空公司一样，上航的客舱广播以普通话（国语）、英语两种语言播报。但自2012年开始，上海航空推出“沪语进客舱”服务项目，所有由上航执飞飞往上海的航班，客舱广播语言同时增加[[上海话|上海话]]播报（飞行预报、客舱安全提醒、飞行信息等常规信息，则不提供上海话播报）。在飞机起飞及播放“动感之城”上海宣传片后，客舱将进行沪语广播，乘务员首先用上海话向旅客致辞，并教兴趣浓厚的旅客学说几句上海话。该服务最初只在北京、深圳、广州以及港澳台地区的6个返沪航班中推出，2013年9月17日完善了广播的内容，且更名为“学说上海话”客舱互动广播，2014年12月1日，该服务扩大到所有上航执飞飞往上海的航班<ref>{{cite news|title=“沪语进客舱”上航特色播报服务好评不断|url=http://news.sina.com.cn/o/2013-09-30/034928336002.shtml|publisher=东方今报 |date=2013年9月30日|accessdate=2015年10月17日 }}</ref><ref>{{cite news|title=上航又推上海话特色广播|url=http://www.bjcaac.com/fw/34531.html|publisher=国际空港信息网 |date=2015-01-04 |accessdate=2015年10月17日 }}</ref>。

== 飞行常客奖励计划 ==
