'''上海轨道交通11号线'''，原名'''申嘉线'''，是[[中华人民共和国|中国]][[上海轨道交通|上海轨道交通]]网络中的一条线路，系统编号'''R3'''。连接上海市西北地区-中心城-浦东新区的主干线，单线里程为82.386公里，是中國和世界上单线里程最长的地铁线路，也是首条跨省轨道交通线路。

== 概览 ==
11号线最初计划由[[嘉定区|嘉定]]至[[南汇区|南汇]]临港新城，全長120公里，以[[三林站|三林]]为界，分为北段和南段。实际建设中，以[[罗山路站|罗山路站]]为分界，其中罗山路以北称11号线，并向东延伸至黄楼（[[上海迪士尼|上海迪士尼]]所在地）；罗山路以南称[[上海轨道交通16号线|16号线]]，并向北延伸至[[龙阳路站|龙阳路站]]。代表色为<small> <font color="{{上海地铁标志色|11}}">█</font> </small>'''咖啡色'''。

* 运营里程：82.386 km
* 车站数：38
* 轨距：1435mm
* 供电方式：1500V DC 接触网供电
* 双线区间：全线
* 电气化区间：全线
* 地下区间：[[御桥站|御桥]]东—[[桃浦新村站|桃浦新村]]北、[[嘉定新城站|嘉定新城]]西—[[昌吉东路站|昌吉东路]]东北、[[康新公路站|康新公路]]—[[迪士尼站_(上海)|迪士尼]]
* 高架区间：[[桃浦新村站|桃浦新村]]北—[[嘉定北站|嘉定北]]以北、[[昌吉东路站|昌吉东路]]西南—[[花桥站_(地铁)|花桥]]、[[御桥站|御桥]]东—[[康新公路站|康新公路]]

==工程建设==
11号线一期工程全长46.008公里，其中地下线长21.676公里，高架线长22.283公里，地面线长2.049公里<ref>{{Cite web |url=http://www.zhb.gov.cn/gkml/hbb/spwj1/201502/t20150216_296005.htm |title=关于上海市轨道交通11号线北段工程项目一期竣工环境保护验收合格的函 |access-date=2017-12-12 |archive-url=https://web.archive.org/web/20171213010129/http://www.zhb.gov.cn/gkml/hbb/spwj1/201502/t20150216_296005.htm |archive-date=2017-12-13 |dead-url=yes }}</ref>。全线共设9座高架车站（其中1座预留）、11座地下车站和1座地面车站，在主线起点附近设嘉定辅助停车场1座，在支线赛车场附近设赛车场车辆段1处，另有控制中心1处，主变电站2处，工程总投资189.5亿元。工程包括主线嘉定北站—华山路中间风井、支线嘉定新城站—安亭站，位于上海市嘉定区、普陀区和长宁区。[[江苏路站|江苏路站]]至[[嘉定北站|嘉定北站]]于2009年12月31日通车试运营，线路长度33.2公里。支线除[[昌吉东路站|昌吉东路站]]外于2010年3月29日开通。

二期工程全长22.886公里，工程总投资129.65亿元。江苏路站至[[罗山路站|罗山路站]]于2013年8月31日通车<ref>{{Cite web |url= http://www.shmetro.com/node49/201308/con113351.htm |title= 11号线二期8月31日起通车试运营 |author= 上海地铁运管中心 |date=2013-08-29 |language= zh-cn |accessdate= 2013-08-29 }}</ref>，单线里程达到65公里，成为国内单线里程最长的地铁线路。

[[安亭站_(地铁)|安亭站]]至[[花桥站_(地铁)|花桥站]]段，全长6公里，均为高架线；其中，上海市境内段约450米，江苏省境内段约5.55公里，投资18.49亿元。于2013年10月16日开通，共设3座高架车站，成为全国首条跨省轨道交通，单线里程达到72公里<ref>[http://news.sina.com.cn/c/2013-10-16/045128444128.shtml 我国首条跨省地铁开通：江苏昆山至上海票价7元]</ref>。

罗山路站至[[迪士尼站_(上海)|迪士尼站]]段，原名浦东中部地区公共交通配套工程，全长约9.15公里，其中高架线约7.39公里，敞开段约0.3公里，地下线约1.46公里，设站3座，投资43.71亿元<ref>{{cite web |url=http://www.ccmetro.com/newsite/readnews.aspx?id=84614 |title=存档副本 |accessdate=2015-05-31 |deadurl=yes |archiveurl=https://archive.is/20150531122430/http://www.ccmetro.com/newsite/readnews.aspx?id=84614 |archivedate=2015-05-31 }}</ref>。2015年12月19日开通[[秀沿路站|秀沿路]]至[[康新公路站|康新公路]]区段<ref name="2015区段">{{Cite news|url = http://www.shmetro.com/node49/201512/con114518.htm|title = 12月19日起  11、12、13号线新延伸段建成试运营|last = |first = |date = |work = 上海地铁运管中心|accessdate = 2015年12月19日|publication-date = 2015年12月19日}}</ref>。2016年4月26日10时起，[[迪士尼站_(上海)|迪士尼]]实现载客试运营<ref>{{Cite web |title = 上海地铁迪士尼站下周二开站 末班车最晚22:30 |url=http://shanghai.xinmin.cn/msrx/2016/04/21/29883675.html|publisher = 新民网| date = 2016-04-21 | accessdate = 2016-04-21 }}</ref>。

目前11号线正在进行[[陈翔路站|陈翔路站]]的建设，投资2.23亿元。这是上海地铁第一个在既有运营线路增加车站的项目。

===11A01型电动客车===
* 全车以白色为主调，列车车身两边各有一条深红色的饰带。
* 牵引系统：西门子旋转电机
* 该批所有列车不设闪灯路线图，但在车厢连接处设有黄色LED显示屏。车头加设黄色LED显示屏显示终到站。车内地板为暗红色。

===11A02型电动客车===
* 全车以白色为主调，列车车身两边各有一条深红色的饰带。前方logo大小比11A01型电动客车更大，车头比11A01型电动客车平。
* 牵引系统：南车时代旋转电机
* 该批所有列车不设闪灯路线图，但在车厢连接处设有黄色LED显示屏。车头加设黄色LED显示屏显示终到站。车内地板为蓝色。

===11A03型电动客车===
* 全车以白色为主调，列车车身两边各有一条深红色的饰带。外观与11A01型列车完全相同。
* 牵引系统：南车时代旋转电机
* 该批所有列车不设闪灯路线图，但在车厢连接处设有黄色LED显示屏。车头加设黄色LED显示屏显示终到站。车内地板为蓝色。
