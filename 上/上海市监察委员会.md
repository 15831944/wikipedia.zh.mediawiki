'''上海市监察委员会'''，简称'''上海市监察委'''，是上海市的省级地方[[中华人民共和国国家监察机关|国家监察机关]]，与[[中国共产党上海市纪律检查委员会|中国共产党上海市纪律检查委员会]][[合署办公|合署办公]]，成立于2018年1月29日<ref>{{cite web |url = http://www.shjcw.gov.cn/2015jjw/n2276/u1ai72341.html |language=zh-hans |title = 上海市监察委今天正式挂牌成立，李强揭牌 |publisher = 上观新闻 |date=2018-01-29 |accessdate=2018-01-29 }}</ref>，取代[[上海市监察局|上海市监察局]]的职能。上海市监察委主任由[[上海市人民代表大会|上海市人民代表大会]]选举产生，并向其和上海市人大常委会以及中华人民共和国监察委员会负责。上海市监察委现任主任为中共上海市委常委、市纪委书记[[廖国勋|廖国勋]]<ref>{{cite web |url = http://www.spcsc.sh.cn/n1939/n4433/n4473/u1ai164747.html |language=zh-hans |title=上海市人民代表大会公告（第四号） |website = 上海人大 |date=2018-01-28 |accessdate = 2018-01-28 }}</ref>。
