'''上普法尔茨行政区'''（Regierungsbezirk Oberpfalz）是[[德国|德国]][[巴伐利亚州|巴伐利亚州]]的7个[[行政区_(德国)|行政区]]之一，首府[[雷根斯堡|雷根斯堡]]。上普法尔茨行政区位于巴伐利亚州东部，与[[捷克|捷克]]接壤，与[[上巴伐利亚行政区|上巴伐利亚行政区]]、[[下巴伐利亚行政区|下巴伐利亚行政区]]、[[中弗兰肯行政区|中弗兰肯行政区]]和[[上弗兰肯行政区|上弗兰肯行政区]]相邻。

== 華語譯名 ==
由於兩岸翻譯上的不同，因此[[台灣|台灣]]則將其音譯為「'''-{上法爾茲}-'''」；[[中國大陸|中國大陸]]則將其音譯為「'''-{上普法尔茨}-'''」。

== 行政区划 ==

上普法尔茨行政区下设3个直辖市：
# [[安贝格|安贝格市]]（Amberg）
# [[雷根斯堡|雷根斯堡市]]（Regensburg）
# [[魏登|魏登市]]（Weiden）
