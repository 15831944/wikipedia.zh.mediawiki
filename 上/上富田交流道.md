'''上富田交流道'''（{{lang|ja|かみとんだインターチェンジ}}）位於[[和歌山縣|和歌山縣]][[西牟婁郡|西牟婁郡]][[上富田町|上富田町]]，是[[紀勢自動車道|紀勢自動車道]]的[[交流道|交流道]]。2015年7月12日通車<ref name="Tanabe-Susami">[http://www.kkr.mlit.go.jp/kinan/zimusho/ir/pdf27/150528.pdf 近畿自動車道紀勢線（田辺〜すさみ） 南紀田辺ICから白浜までが7月に先行開通! 〜夏の観光シーズンに大きな効果を期待〜 〜あわせて路線名称、IC名称が決定〜] 国土交通省近畿地方整備局 紀南河川国道事務所　2015年5月28日掲載・29日閲覧</ref><ref name="Tanabe-Susami 2">[http://www.kkr.mlit.go.jp/kinan/zimusho/ir/pdf27/150612.pdf 紀勢自動車道（田辺〜すさみ）　7月12日（日） 南紀田辺IC〜南紀白浜ICが開通します! 〜夏の観光シーズンに大きな効果を期待〜] 国土交通省近畿地方整備局 紀南河川国道事務所　2015年6月12日掲載・閲覧</ref>。

== 歷史 ==
* [[1998年|1998年]]（[[平成|平成]]10年）[[12月25日|12月25日]] : 南紀田邊交流道 - 南紀白濱交流道間 施行命令<ref name="wakayama">{{Cite web|date = |title = 高規格幹線道路|url = http://www.pref.wakayama.lg.jp/prefg/080300/kokikaku/kokikaku.html#kinki|publisher = 和歌山県|accessdate = 2014-05-31|archive-url = https://web.archive.org/web/20140531125644/http://www.pref.wakayama.lg.jp/prefg/080300/kokikaku/kokikaku.html#kinki|archive-date = 2014-05-31|dead-url = yes}}</ref>。
* [[2006年|2006年]]（平成18年）[[2月7日|2月7日]] : 第二次國土開發幹線自動車道建設會議時決定將南紀田邊交流道 - 南紀白濱交流道間以'''新直轄方式'''管理<ref name="wakayama" />。
* [[2015年|2015年]]（平成27年）[[5月28日|5月28日]] : 交流道名稱正式確定為「上富田交流道」<ref name="Tanabe-Susami" />。
* 2015年（平成27年）[[7月12日|7月12日]] : 南紀田邊交流道 - 南紀白濱交流道間 通車啟用<ref name="Tanabe-Susami" /><ref name="Tanabe-Susami 2" />。
