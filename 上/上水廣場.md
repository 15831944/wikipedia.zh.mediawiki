'''上水廣場'''（'''{{lang|en|Landmark North}}'''）為[[香港|香港]][[北區_(香港)|北區]]的大型商場，位於[[港鐵|港鐵]][[上水站|上水站]]對面。商場樓高5層，其中2樓至5樓為購物樓層，設99間商舖；6樓至22樓為辦公大樓。發展商為[[新鴻基地產|新鴻基地產]]，於1995年正式開幕。由於上水廣場頂部外型設計獨特，也是區內最大型的購物商場，而地面層則爲新的[[上水巴士總站|上水巴士總站]]所在地。

== 商場內部 ==
上水廣場經過翻新後，引進多間珠寶和美容店，主力吸納大陸旅客市場。翻新後，2-5樓為商場，6-9樓為樓上舖，店舖超過150間。零售商稱，75%到90%的顧客是大陸人。<ref>{{cite news |author1=CHRIS HORTON |title=大陸購物客改變香港新界商業環境 |url=https://cn.nytimes.com/business/20140612/c12border/zh-hant/ |accessdate=2018-08-16 |publisher=紐約時報中文網 |date=2014-06-12}}</ref>

==歷史==
上水廣場原址為[[菜園村|菜園村]]，為一個農地，當時政府要發展[[新市鎮|新市鎮]]，因此強行收回菜園村，於1990年土地由[[新鴻基地產|新鴻基地產]]投得，發展為現有的上水廣場，於1995年落成啟用。由於當時的上水位於香港偏遠位置，生活水平較低，而且當時政府還未開放[[自由行|自由行]]來港，因此當時上水廣場店舖以廉價為主，平民化食肆也較多，如[[哈迪斯|哈迪斯]]快餐店、[[美心大酒樓|美心大酒樓]]、[[繽紛樂園|繽紛樂園]]、[[冒險樂園|冒險樂園]]、[[大快活|大快活]]、[[必勝客|必勝客]]、[[客家好棧|客家好棧]]、[[豐澤|豐澤]]、[[日本城|日本城]]、[[中國旅行社|中國旅行社]]、[[商務印書館|商務印書館]]、[[通利琴行|通利琴行]]、[[聖安娜餅店|聖安娜餅店]]、[[肯德基|肯德基]]、[[Delifrance|Delifrance]]、[[縱横二千集團|縱横二千集團]]、[[馬拉松|馬拉松]]、[[Walker_Shop|Walker Shop]]、[[b+ab|b+ab]]等。由於政府在2005年實行「一簽多行」政策後，很多内地自由行都選擇來港購物，因此大部分店舖都變成金店、化妝品店及珠寶鐘錶店，如[[太子珠寶鐘錶公司|太子珠寶鐘錶公司]]、[[周大福|周大福]]、[[周生生|周生生]]、[[六福珠寶|六福珠寶]]、[[謝瑞麟|謝瑞麟]]、[[精工錶|精工錶]]、[[天梭錶|天梭錶]]、[[Dior|Dior]]、[[SK-II|SK-II]]、[[Lancome|Lancome]]等,一些貴價時裝店，如[[chocolate|chocolate]]、[[ete!|ete!]]、[[twist|twist]]以及一些中價餐廳，如[[太興|太興]]、[[美心香港地|美心香港地]]、[[上海婆婆|上海婆婆]]、[[Starbucks|Starbucks]]等。

上水廣場目前也是香港境內距離中國內地最近的大型購物商場，地理位置相當接近[[深圳|深圳]]，與[[羅湖口岸|羅湖口岸]]僅一個[[港鐵車站|港鐵車站]]之隔，故已成為實際上的「邊境購物城」。自2003年實施[[港澳個人遊|港澳個人遊]]（自由行）後商場中國內地旅客數量大幅飆升，現今上水廣場如同區內很多其他商場一樣，以招呼自由行旅客為主。

== 翻新工程 ==
上水廣場於2008年至2009年斥資逾8,000萬元進行美化及翻新工程，工程分兩期進行，包括將商場內部全面翻新，[[中庭|中庭]]增設[[視訊牆|大型電視]]，播放場內推廣活動及優惠資訊、翻新三樓寫字樓大堂及五樓設露天平台花園，讓顧客可於戶外進餐或休憩。<ref>[http://www.mpfinance.com/htm/Finance/20070824/News/la_lac1.htm 上水廣場斥8000萬翻新 明報 2007-08-24]</ref>

引入時尚名牌，如[[TWIST|TWIST]]、[[:CHOCOLATE|:CHOCOLATE]]、[[Brand_Off|Brand Off]]、[[Dior|Dior]]等。
#[[中庭|中庭]]增加[[視訊牆|大型電視]],播放場內推廣活動及優惠資訊。
#五樓設露天平台花園，顧客可於戶外進餐或休憩

== 行人天橋 ==
商場設5條[[行人天橋|行人天橋]]連接，分別來往[[新都廣場|新都廣場]]、[[港鐵|港鐵]][[上水站|上水站]]、[[彩園邨|彩園邨]]、[[石湖墟|石湖墟]]及[[上水匯|上水匯]]。

== 停車場 ==
商場[[停車場|停車場]]設有375個停車位，全日24小時為顧客和租戶提供服務，購物更可享免費泊車優惠。

== 寫字樓 ==
6樓至22樓為寫字樓，是[[北區_(香港)|北區]]唯一的[[甲級寫字樓|甲級寫字樓]]，內設[[健身室|健身室]]，面積每層約23,800[[平方呎|平方呎]]。

== 交通 ==
毗鄰港鐵上水站，商場地下更設大型公共交通總匯，往返港九各區甚至深圳關口均十分方便，是全港唯一「五口通關」的商場。（鐵路：羅湖站、落馬洲站；跨境巴士：文錦渡、沙頭角、皇崗）
