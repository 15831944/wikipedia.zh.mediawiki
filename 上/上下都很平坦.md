《'''上下都很平坦'''》，著名[[先锋派|先锋派]][[小说家|小说家]][[马原|马原]]的代表作之一，为[[长篇小说|长篇小说]]，带有自传性质。<ref name=" lion" >{{cite book |author=马原  |title=《上下都很平坦》|year=2001年 |publisher=北岳文艺出版社 |location= |pages=前言 |isbn= 9787537822282|language = zh-cn}}</ref>

== 内容 ==
[[20世纪70年代|20世纪70年代]]，正是[[文化大革命|文化大革命]]如火如荼地进行中，[[毛泽东|毛泽东]]号召[[知识分子|知识分子]][[上山下乡|上山下乡]]，一大群原本在[[学校|学校]]里研究钻研[[学术|学术]]和[[科技|科技]]的知识分子瞬间变成了[[知青|知青]]，知青来到[[农村|农村]]之后，在农村过着困窘和饥荒的日子，为了解决温饱问题，他们偷鸡摸狗、打猎、下河抓鱼、上山采野果等，瓶子、小秀、江梅、肖丽这4个女子正是知青中的成员，姚亮、陆高、长脖、二狗4个男子也是知青中的成员，他们互相在这场盛大的革命运动中遭遇，因为正处青春韶华，[[心理|心理]]和[[生理|生理]]都已经成熟，他们便在这场轰动全国的运动中开辟出一条狭小的小天地，谈情说爱，安家落户，植根乡土。<ref name=" shark" >{{cite web |url=http://news.163.com/10/0510/06/66A772TA00014AED.html |title=马原：小说已死(组图) |accessdate=2010-05-10 |publisher= 网易|work= |language =zh-cn}}</ref>

== 艺术 ==
小说语言简洁和深刻，富有哲理意义和神秘气息。<ref name=" shark" />

== 社会反响 ==
《上下都很平坦》与[[王朔|王朔]]《[[动物凶猛|动物凶猛]]》、[[余华|余华]]《[[在细雨中呼喊|在细雨中呼喊]]》并称“先锋文学青春叙事的三部杰作”。<ref name=" shark" />
