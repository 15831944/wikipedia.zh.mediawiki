'''上海长征医院'''（'''中国人民解放军海军军医大学第二附属医院'''），位于[[上海市|上海市]]，隶属[[中国人民解放军海军军医大学|中国人民解放军海军军医大学]]，是[[三级甲等医院|三级甲等医院]]。

==简介==
医院前身是1900年[[德国|德国]]医生[[埃里希·宝隆|埃里希·宝隆]]博士创办的“'''宝隆医院'''”。1930年更名“'''国立同济大学附属医院'''”。1948年更名“'''中美医院'''”。1955年10月，由[[中华人民共和国国防部|中华人民共和国国防部]]部长[[彭德怀|彭德怀]]签署命令，成立“'''上海急症外科医院'''”<ref name="长征医院官网"/>。1956年5月，第二军医大学在汉口路515号（惠中旅社）正式创办上海急症外科医院<ref name=sht/>。1958年9月，列编为“'''中国人民解放军第二军医大学第二附属医院'''”<ref name="长征医院官网"/>。1959年10月，迁入凤阳路415号同济医院旧址，对外称'''上海同济医院'''<ref name=sht/>。1966年9月，经上海市批准对外称“'''上海长征医院'''”<ref name="长征医院官网">{{Cite web |url =http://www.shcz.com/front/danye.aspx?quest=yiyuanjieshao  |title =医院介绍  |author =  |date =  |publisher = 上海长征医院 |language =  |accessdate = 2017-06-15 |quote =  }}</ref>。1969年10月，随第二军医大学调驻[[陕西省|陕西省]][[西安市|西安市]]。1975年7月迁回上海原址<ref name=sht>{{Cite web |url = http://www.shtong.gov.cn/Newsite/node2/node2245/node67643/node67648/node67687/node67780/userobject1ai65006.html |title = 上海卫生志 第一篇医疗 第一章医疗机构 第六节　医院选介 |author =  |date = 2003-12-08 |publisher = 上海通 |language =  |accessdate =  |quote =  }}</ref>。2017年更名'''中国人民解放军海军军医大学第二附属医院'''。

截至2010年代，医院设有业务科室47个，临床科室35个，辅诊科室12个，临床教研室20个，拥有硕士授权学科33个，博士授权学科28个，并且是临床博士后流动站。医院有2个上海市临床医学中心，2个上海市医学领先专业重点学科，6个全军医学专科技术中心，4个全军医学研究所，10个国家重点学科，是全军实验诊断重点实验室，国家药品临床研究基地，军队药品临床研究基地<ref name="长征医院官网"/>。

医院具有骨科、肾内科、神经外科、泌尿外科、整形外科、急救科等传统优势学科<ref name="长征医院官网"/>。1960年，长征医院医学教育家[[屠开元|屠开元]]成功进行了中国首例断肢再植动物实验<ref name="断肢再植">{{cite web|url=http://www.shtong.gov.cn/newsite/node2/node19828/node20062/node20114/node64319/userobject1ai68379.html|title=著名人物、先进集体-逝世人物一节|publisher=上海年鉴2000| date=2000-01-01 |accessdate=2013-11-15 }}</ref>。1978年6月，医院在[[中国内地|中国内地]]率先进行肾脏移植<ref name="人民网">{{Cite web |url = http://www.people.com.cn/GB/keji/1057/2503610.html |title = 器官移植:高难见精彩 |author =  |date =2004-05-18  |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。1985年，何清廉教授完成了中国内地第一例公开报道的变性手术<ref> {{cite book|author=吴兴人|coauthors=|title=纠正上帝的错误——“中国变性手术之父”何清濂的非常记忆|year=2005年 |publisher=上海文艺出版社|location=上海|isbn=9787532128884}}</ref>。1996年，傅志仁教授手术创下中国内地存活时间最长的单次原位全肝移植<ref name="全肝移植">{{cite web|url=http://www.people.com.cn/GB/paper40/10689/971365.html|title=“换肝”高手傅志仁
|publisher=华东新闻| date=2003年11月21日 第二版 |accessdate=2013-11-15 }}</ref>。

== 分院 ==
*中国人民解放军海军军医大学长征医院南京分院：位于[[江苏省|江苏省]][[南京市|南京市]][[下关区|下关区]]建宁路200号。前身是'''华东海军南京医院'''，后曾更名'''华东海军第二医院'''。1955年正式命名为'''中国人民解放军第四一四医院'''，是南京市集医疗、教学、科研为一体的部队综合性三甲医院。1999年，随海军军医学院转隶[[中国人民解放军第二军医大学|中国人民解放军第二军医大学]]建制，编入中国人民解放军第二军医大学第二附属医院，对外命名为'''中国人民解放军第二军医大学长征医院南京分院'''<ref>{{Cite web |url = http://www.cznj-hosp.com/about/introduction.asp |title =医院简介  |author =  |date =  |publisher =中国人民解放军第二军医大学长征医院南京分院  |language =  |accessdate = 2017-06-15 |quote =  }}</ref>。2017年更名'''中国人民解放军海军军医大学长征医院南京分院'''。
