'''上海周公馆'''的正式名称是'''中国共产党代表团驻沪办事处'''，位于[[上海市|上海市]][[黄浦区|黄浦区]][[思南路|思南路]]73号，是[[中国共产党|中国共产党]]于[[国共谈判|国共谈判]]期间在上海设立的办事机构，对外宣称是[[周恩来|周恩来]]的寓所，所以也有了周公馆的名称。

==简介==
[[File:Statue_of_Zhou_Enlai_at_his_former_residence_in_Shanghai_(20151003132905).jpg|thumb]]
上海周公馆是一幢建于二十世纪20年代三层的花园别墅，[[抗日战争|抗日战争]]结束后后为了进行[[国共谈判|国共谈判]]，中国共产党于1946年6月在此设立驻沪办事处，[[周恩来|周恩来]]、[[董必武|董必武]]等多次在此举行记者招待会和会见中外人士。周公馆在1959年被列为[[上海市文物保护单位|上海市文物保护单位]]，并在1979年进行了修复，建立了纪念馆。1986年9月，思南路73号、71号两幢楼房作为中国共产党代表团驻沪办事处纪念馆对外开放，纪念馆占地面积2400平方米，建筑面积1049平方米，73号作为周公馆旧址，按照当年家具陈设等进行了复制并展出，中共代表团驻沪办事处图片史迹展览在临近的71号号别墅展出，花园内则有周恩来的铜像<ref>{{cite web |url = http://www.sh-aiguo.gov.cn/node2/node4/node7/userobject1ai492.html |title = 中国共产党代表团驻沪办事处纪念馆（周公馆） |publisher = [[中共上海市委|中共上海市委]]宣传部 |date =  |accessdate = 2012-05-14 }}</ref>。
