'''上海长江河口科技馆'''位于[[上海市|上海市]][[宝山区|宝山区]]宝杨路1号[[吴淞炮台湾湿地森林公园|吴淞炮台湾湿地森林公园]]内，由[[上海市科学技术委员会|上海市科学技术委员会]]、[[宝山区|宝山区]]政府和[[华东师范大学|华东师范大学]]合作共建，是世界上唯一以“河口”命名的科技馆。

该馆是中国[[全国科普教育基地|全国科普教育基地]]、全国海洋科普基地、[[上海市爱国主义教育基地|上海市爱国主义教育基地]]和华东师范大学[[河口海岸学国家重点实验室|河口海岸科学综合实验基地]]。<ref>[http://www.tourfest.org/htdocs/tf_news_item.aspx?ID=224 上海科技馆河口分馆揭牌]</ref>

== 场馆介绍 ==
上海长江河口科技馆于2011年10月18日落成开馆，总建筑面积7707平方米，是上海市[[宝山区|宝山区]]政府与[[华东师范大学|华东师范大学]]区校合作的重点项目，由华东师范大学国家重点实验室所在单位河口海岸研究院负责内容策划，由华东师范大学设计学院负责从建筑外观到所有馆内展项的设计。<ref>[http://bsq.sh.gov.cn/zwxx/zxbd/201406/t20140609_167909.html 上海市宝山区人民政府] {{webarchive|url=https://archive.is/20160216030942/http://bsq.sh.gov.cn/zwxx/zxbd/201406/t20140609_167909.html |date=2016-02-16 }}	</ref>

展馆外形犹如一条跃动的鱼，共设有四个常规展厅和一个临时展厅，重点介绍长江河口自然、生态、科技与历史，综合展示世界各主要河口相关知识。<ref>{{cite web |url = http://www.most.gov.cn/dfkj/sh/zxdt/201111/t20111104_90687.htm |title = 上海长江河口科技馆开馆 |publisher = [[中华人民共和国科学技术部|中华人民共和国科学技术部]] |date=2011-11-07 |accessdate=2012-07-24 }}</ref> 上海长江河口科技馆正对[[长江|长江]]、[[黄浦江|黄浦江]]的汇流处，与[[上海吴淞国际邮轮码头|上海吴淞国际邮轮码头]]、[[吴淞炮台湾湿地森林公园|吴淞炮台湾湿地森林公园]]共同构成滨江景观带。

该馆设有四个常规展厅：
*序厅
*资源与环境：以互动游戏的形式介绍河口资源知识；
*科技与应用：展示中国在河口科研方面所取得的成就；
*人文与历史：讲述[[长江三角洲|长江三角洲]]的成长历程。
