== 地理位置 ==
[[File:Sheshan_hill_Songjiang_District_Shanghai-2.jpg|thumb]]]]

上海位于[[北纬|北纬]]31度14分，[[东经|东经]]121度29分。正当中国海岸线中心点，守长江口。处[[长江三角洲|长江三角洲]][[冲积平原|冲积平原]]前缘，东濒[[东海|东海]]，北界[[长江|长江]]，南临[[杭州湾|杭州湾]]，西与[[江苏省|江苏省]]和[[浙江省|浙江省]]接壤。全市南北最长处约120公里，东西最宽处约100公里。

==地貌==

上海市全境为[[冲积平原|冲积平原]]，仅西南部有部分[[火山|火山]]岩丘。海拔平均高度在4米左右，地势平坦，山脉少而低小。西部有[[佘山|佘山]]、[[天马山|天马山]]等，但高度都在100米以下。上海市最高点是位于[[杭州湾|杭州湾]]内，距离杭州湾北岸陆地最近点金山嘴6.2公里的海拔105.03米的[[大金山|大金山]]。

==面積==
上海全市市域面积为7823.44平方公里（包括陸域、长江水面和沿海滩涂）。陆域面积6340.50平方公里（其中大陆面积5132平方公里，岛屿面积1208平方公里）；长江水面1107平方公里；沿海滩涂面积376平方公里<ref name="上海计划志规划内容">[http://www.shtong.gov.cn/newsite/node2/node2245/node73963/node73970/node74028/node74070/userobject1ai88918.html 《上海计划志》>>第三篇 专项计划>>第二章 国土规划>>第二节 规划内容]</ref>。

== 气候 ==
上海的气候属于[[亚热带季风气候|亚热带季风气候]]，[[四季|四季]]分明，日照充分，雨量充沛。气候温和湿润，[[春|春]]（4月-5月）、[[秋|秋]]（10月-11月）较短，[[冬|冬]]（12月-次年3月）、[[夏|夏]]（6月-9月）较长。有春雨、[[梅雨|梅雨]]、秋雨三个雨期，因而5月至9月间为上海的汛期，降水量达全年的60%左右<ref>{{cite web | title = 上海概览-上海气候概况 | publisher = 中国上海| url = http://www.shanghai.gov.cn/shanghai/node2314/node3766/node3773/node3777/index.html| accessdate = 2011-03-11 | language = zh }}</ref>。每年的7月至8月进入[[伏旱|伏旱]]天气，较之日常月份显得潮湿酷热，每年平均有8.7天最高氣溫超過攝氏35度。由于地处东南沿海，每年夏天和初秋时节常会有来自[[太平洋|太平洋]]上的[[热带气旋|热带气旋]]（[[颱風|颱風]]）过境或影响。每年1月中旬至2月初为全年最寒冷的季节，受北方[[蒙古高原|蒙古高原]]和[[西伯利亚|西伯利亚]]的冷空气影响，会出现霜冻现象。如遇强冷空气来袭并恰逢暖湿气流，上海也会出现雨夹雪或降雪天气<ref>{{cite web | title = 上海气候概况 | publisher = 中国网| date = 2009-09-08 | url = http://www.china.com.cn/aboutchina/zhuanti/09dfgl/2009-09/08/content_18488374.htm| accessdate = 2011-03-11 | language = zh }}</ref>。

== 水文 ==
[[File:Shanghaihectorgarcia.jpg|180px]]口]]
[[File:Woosungkow.jpg|180px]]]]
江南地区水网密布，天气湿润多雨，上海地区河网纵横，水量丰富。到2010年，本市的河道及湖泊总面积尚有642.7平方公里，河面率为10.1%。本市河网基本属于黄浦江水系。境内主要河流有[[黄浦江|黄浦江]]、吴淞江及[[川杨河|川杨河]]、[[淀浦河|淀浦河]]、[[虬江|虬江]]等。黄浦江起自[[太湖|太湖]]{{NoteTag|1999年，上海地理学会发表文章，确认以[[浙江省|浙江省]][[安吉县|安吉县]]的[[苕溪|苕溪]]之源作为黄浦江源头。但并未完全得到认同，现采用上海政府官网的数据为准。}}，全长113公里，流经市区，江道宽度300～770米不等，平均360米，为不冻河，是上海的水上交通要道和主要城市景观组成部分。吴淞江（俗称[[苏州河|苏州河]]）上海境内长54公里，河道平均宽度45米，原有入海口即今[[吴淞口|吴淞口]]，后“[[黄浦夺淞|黄浦夺淞]]”<ref>{{cite web | title = 上海港的发韧地——旧青浦 | publisher = 新沪商 | date = 2010-03-10 | url = http://old.jfdaily.com/newspaper/xhs_1/node49102_64857/node49101/201003/t20100310_875363.htm | accessdate = 2011-03-11 | language = zh | deadurl = yes | archiveurl = https://web.archive.org/web/20150121074124/http://old.jfdaily.com/newspaper/xhs_1/node49102_64857/node49101/201003/t20100310_875363.htm | archivedate = 2015-01-21 }}</ref>，现于[[外白渡桥|外白渡桥]]以东入[[黄浦江|黄浦江]]。上海境内的最大湖泊为[[淀山湖|淀山湖]]，面积为62平方公里。此外，由于城市发展，上海市区境内诸多河道已被填埋并修筑道路，最为知名的为公共租界与法租界的界河[[洋泾浜|洋泾浜]]，即今[[延安东路_(上海)|延安东路]]。

== 岛屿 ==
上海的自然岛屿分为海洋岛屿与内河岛屿两大系统。海洋岛屿主要分布在北部长江口和南部[[杭州湾|杭州湾]]一带。共有崇明岛、长兴岛、横沙岛3个有居民岛屿，大金山岛、佘山岛、九段沙等23个无居民岛屿（沙洲）。<ref>[http://www.shanghai.gov.cn/shanghai/node2314/node2319/node22396/node22403/u21ai655694.html 上海市海洋发展“十二五”规划]</ref>岛屿以自然岛屿为主。北部长江口处的岛屿以[[崇明岛|崇明岛]]、[[长兴岛_(上海)|长兴岛]]和[[横沙岛|横沙岛]]三个岛为主，并伴有一些小型沙洲，这是上海市最大的三个岛屿。而[[杭州湾|杭州湾]]处的岛屿则有[[金山区_(上海市)|金山区]]境内的的[[大金山|大金山]]等岛，崇明岛以东35公里有[[佘山岛|佘山岛]]。[[黄浦江|黄浦江]]游的杨浦区有一座上海市区唯一的内河岛屿—[[复兴岛|复兴岛]]。此外[[黄浦江|黄浦江]]中游还有[[泖岛|泖岛]]、[[浏河|浏河]]还有[[浏河岛|浏河岛]]等内河岛屿。

[[File:China_Shanghai_Chongming_Nanmen.jpg|thumb]]
崇明岛是由长江携带下来的大量泥沙冲积而成，[[唐朝|唐]]武德元年（618年）仅是长江中的两个小沙洲，面积只有十餘平方公里<ref>{{cite web | title = 崇明岛印象 | publisher = 嘉兴日报 | date = 2006-09-01 | url = http://www.cnjxol.com/oldepaper/jxrb/content/2006-09/01/content_190832.htm | accessdate = 2011-03-11 | language = zh | deadurl = yes | archiveurl = https://web.archive.org/web/20120112131635/http://www.cnjxol.com/oldepaper/jxrb/content/2006-09/01/content_190832.htm | archivedate = 2012-01-12 }}</ref>。后几经变迁，现在东西约80多公里，南北宽约10-15公里，面积为1041平方千米。被誉为“长江门户、东海瀛洲”是世界最大的河口[[冲积岛|冲积岛]]和中国第三大岛。

长兴岛和横沙岛位于崇明岛以南的江口，将江水分为南北二水道。[[长兴岛|长兴岛]]俗称鸭窝沙，露出水面不过百年左右，分为石头沙、瑞丰沙、潘家沙、鸭窝沙、金带沙五个部分。近40年来才逐渐成一岛。与[[吴淞|吴淞]]相对。横沙岛在长兴岛东面，由横沙岛和元沙岛组成，面积约为40平方公里。

[[九段沙|九段沙]]是在长江口深水航道南侧新淤积形成的沙岛，总面积为420平方公里（-6米），其中吴淞0米线以上面积145平方公里。目前无常住人口，2000年建立上海市九段沙湿地自然保护区<ref>{{cite web | title = 上海九段沙湿地成候鸟天堂 | publisher = 人民网| date = 2001-03-11 | url = http://env.people.com.cn/GB/1072/13113460.html| accessdate = 2010-11-02 | language = zh }}</ref>。
