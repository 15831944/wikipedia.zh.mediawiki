'''上海快鹿投资(集团)有限公司<ref name="b1"/>'''、又称'''快鹿集团'''，是一家位于上海市的金融公司，注册地在[[长宁区|长宁区]]<ref name="a1">{{cite web |url = https://finance.qq.com/a/20160924/009182.htm |title=上海经侦介入快鹿案 李耀新被抓或揭开更多内幕 |accessdate=2018-10-13 |author= 胡金华（记者） |date=2016-09-24 |publisher=腾讯网，来源：华夏时报|language=Zh-hans }}</ref>。2016年起，与相关公司涉及一系列[[非法集资|非法集资]]案件引人关注。原董事长[[施建祥|施建祥]]已中国政府通缉。

== 历史 ==
前身可追溯至1953年成立的上海快鹿电线电缆有限公司，是以生产快鹿牌电话线知名的国有线缆企业。1999年国企改革，企业家[[施建祥|施建祥]]收购上海宣布[[破产|破产]]的四家国有企业。其中，快鹿即是受长宁区人民政府委托而收购，他以1亿元资金收购10亿资产的快鹿<ref>{{cite journal en|quotes =

2010年、2012年，公司获得小额贷款、融资担保两块牌照，进军金融业。此后，建立起一系列公司，或称“快鹿系”。2015年，中国大陆的互联网金融、影视成为热门投资领域。快鹿集团以“互联网+金融+影视”的概念，吸收了过百亿的投资。在一个不完全统计中，旗下公司有328家<ref>{{Cite web |url = http://www.dzzq.com.cn/estate/39012106.html |title=揭秘快鹿集团施建祥圈钱术：官员站台 328家公司织网 |accessdate=2018-10-13 |author= |date=2018-07-04 |work= |publisher=大众证券网，来源：南方周末|language=Zh-hans }}</ref>。

快鹿集团拥有三个[[P2P借贷|P2P借贷]]平台——金鹿财行、当天财富、东虹桥小贷。而快鹿集团对外否认与金鹿财行所属公司——上海金鹿财行财富投资管理有限公司的隶属关系。根据工商公示信息，快鹿集团和东虹桥金融控股公司{{NoteTag|东虹桥金融控股公司已改名为中海投金控。}}确实是金鹿财行的发起方<ref>{{cite web |url = http://www.xinhuanet.com/fortune/2016-03/31/c_128853271.htm |title=金鹿财行陷入兑付危机 快鹿系资本迷局即将揭盅？ |accessdate=2018-10-13 |author=徐曼曼 |date=2016-03-31 |work=责任编辑：徐曼曼 |publisher=新华网 |language = zh-Hans }}</ref>。

== “快鹿系”集资诈骗案 ==
=== 案发 ===
2013年，电影《[[葉問3|叶问3]]》开拍，制作团队来自香港，但由快鹿集团旗下子公司投资拍摄。[[施建祥|施建祥]]担任电影制片人。此片是资本参与影视制作的样本。与快鹿集团相关的[[神开股份|神开股份]]、[[十方控股|十方控股]]、当天财富等多家公司、机构认购了《叶问3》在中国大陆的电影票房收益权。当时，神开股份由快鹿集团控股，施建祥则是十方控股第二大股东。2016年2月23日，十方控股以1.1亿元人民币的价格向合禾影视收购《叶问3》在大陆地区55%的票房收益权。25日，十方控股公告，施建祥将出任董事会主席。3月4日，《叶问3》在大陆上映。上映前，施建祥成为十方控股第一大股东<ref>{{cite web |url = http://finance.sina.com.cn/roll/2016-03-02/doc-ifxpvzah8536199.shtml |title=影视资本运作叶问3样本：票房和股价捆绑 |accessdate=2018-10-13 |author=记者：王俊丹|date=2016-03-02 |work= |publisher=新浪网，来源：21世纪经济报道 |language=zh-Hans }}</ref>。上映之初，《叶问3》在票房上取得巨大成功，至3月6日，[[2016年中國大陸一周票房冠軍|成为当周票房冠军]]。就在上映首日，十方控股股价大涨22%。然而上映不久，即爆出票房造假。受此质疑影响，在3个交易日后，至3月9日收盘，十方控股累计跌幅47.78%，神开股份累计跌幅10.26%<ref>{{cite web |url = http://finance.sina.com.cn/roll/2016-03-09/doc-ifxqhmve8976010.shtml |title=《叶问3》背后的票房“资产证券化” |accessdate=2018-10-13 |author= |date=2016-03-09 |editor = 责任编辑：赵文伟 SF182 |publisher=新浪网，来源：一财网 |language=zh-Hans }}</ref>。同月，金鹿财行出现兑付风波。由此引发一系列事件，称'''快鹿事件'''。4月6日，快鹿集团公告并购金鹿财行、当天财富<ref name="a1" />。

7月8日，离职的集团董事局主席徐琪{{NoteTag|徐琪为[[美国籍|美国籍]]<ref name="b1" />。}}对外指控，集团高管邵某某、韦某某等人侵吞50亿元集团资产。徐琪亦被控侵吞6千万元集团资产。9月8日，与快鹿集团关系密切的原上海市[[长宁区|长宁区]]区长、[[上海市经济和信息化委员会|上海市经济和信息化委员会]]主任，现[[上海仪电|上海仪电]]（集团）有限公司监事会主席[[李耀新|李耀新]]“落马”。9月13日，[[上海市公安局|上海市公安局]]长宁公安分局在[[新浪微博|微博]]发布案情通告，称金鹿财行、当天财富未取得合法资质的情况下，面向公众募集资金，涉嫌[[非法吸收公众存款罪|非法吸收公众存款罪]]。9月中旬，[[上海市公安局|上海市公安局]]经济犯罪侦查总队已对快鹿集团进入全面调查<ref name="a1"/>。

2018年5、6月间，与施建祥关系密切的[[崔永元|崔永元]]举报影视圈阴阳合同。10月，由此引发的[[范冰冰偷税漏税事件|范冰冰偷税漏税事件]]得出调查结论。崔永元披露在向[[国家税务总局|国家税务总局]]提供证据的过程中，被上海公安局经侦大队派员询问。同时，上海警方侦查他所参与的公司，审讯他之前的助理。[[崔永元#檢舉上海警方不作為|上海警方询问崔永元，因范冰冰案，或因快鹿集团案，不得而知]]。

=== 一审 ===
2018年9月25日、26日，[[上海市第一中级人民法院|上海市第一中级人民法院]]对“快鹿系”集资诈骗案进行一审。法院公开审理了快鹿集团、上海长宁东虹桥小额贷款股份有限公司、上海东虹桥融资担保股份有限公司以及徐琪、张蕾、黄家骝、孙晔等12名个人集资诈骗、非法吸收公众存款一案。公诉机关指控，快鹿集团以“互联网金融”名义进行非法集资并诈骗集资款。而上海长宁东虹桥小额贷款股份有限公司则应快鹿集团要求制造虚假债权凭证，并由上海东虹桥融资担保股份有限公司提供不可撤销连带责任保证。此案非法集资共计400亿余元，共造投资人实际损失100亿余元。26日晚8时37分，庭审结束。[[合议庭|合议庭]]将在评议后对案件择期宣判<ref name="b1">{{cite web |url = https://www.thepaper.cn/newsDetail_forward_2477952 |title = 上海一中院一审公开开庭审理“快鹿系”集资诈骗案 |accessdate=2018-10-25 |author=记者：朱奕奕 |date=2018-09-26 |editor = 责任编辑：郑浩 |publisher=澎湃新闻 |language=zh-Hans }}</ref>。

2019年1月16日，上海市第一中级人民法院公开宣判，对快鹿集团、东虹桥小贷公司、东虹桥担保公司分别以集资诈骗罪判处罚金人民币15亿元、2亿元、2亿元；对黄家骝、韦炎平以集资诈骗罪判处无期徒刑，并处罚金；对徐琪以集资诈骗罪、非法吸收公众存款罪两罪并罚判处有期徒刑13年，并处罚金；对周萌萌等其余12名被告人以集资诈骗罪分别判处有期徒刑15年至9年不等的刑罚，并处罚金<ref>{{cite web |url = http://www.xinhuanet.com/2019-01/16/c_1123999026.htm|title = 上海法院一审公开宣判“快鹿系”集资诈骗案 |accessdate=2019-01-16  |author=新华社记者黄安琪 |date=2019-01-16  |editor =  责任编辑： 张樵苏 |publisher=新华网 |language=zh-Hans }}</ref>。
