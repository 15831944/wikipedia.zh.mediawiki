'''上海银行'''（{{lang-en|Bank of Shanghai}}，{{sse|601229}}），原名'''上海城市合作银行'''，成立于1995年12月29日，是一家总部设立在上海的[[股份制商业银行|股份制商业银行]]。其是由[[上海市|上海市]]国有[[股份|股份]]、[[中资|中资]][[法人|法人]]股份、[[外资|外资]]股份及众多个人股份共同组成的。

截至2016年12月，上海银行在[[上海|上海]]、[[北京|北京]]、[[天津|天津]]、[[深圳|深圳]]、[[成都|成都]]、[[杭州|杭州]]、[[宁波|宁波]]、[[绍兴|绍兴]]、[[南京|南京]]、[[苏州|苏州]]、[[盐城|盐城]]、[[南通|南通]]、[[常州|常州]]、[[无锡|无锡]]都设有分支行和营业网点。此外，上海银行还发起设立了闵行上银村镇银行、衢江上银村镇银行、[[江苏|江苏]]江宁上银村镇银行和崇州上银村镇银行。银行子公司有上银基金管理公司90%以及在[[香港|香港]]设立的上海银行（香港）有限公司100%。

上海銀行2015年報顯示，截至2015年12月31日，银行總資產达14491.4亿元人民币，凈利潤达130.43亿元人民币，均较前一年有所增长<ref name="add"/>。客户存款和垫款总额为5365.08亿元人民币，客户存款总额为7926.8亿元人民币。资本充足率为12.65%，不良贷款率为01.19%<ref name="add"/>。

据[[英国|英国]]《{{link-en|银行家杂志|The Banker}}》统计，按一级资本排序，在2015年度全球1000家大银行中，上海银行排名为108位<ref name="add" />。在《亚洲银行家》“2012年亚洲银行500强”资产规模排名中排在第55位<ref name="add" />。

== 发展历程 ==
1999年9月和2001年12月，上海银行分别吸收了[[国际金融公司|国际金融公司]]和[[香港上海汇丰银行|香港上海汇丰银行]]、[[上海商业银行|上海商业银行]]等外资银行的参股投资。2006年，上海银行被允许办理异地业务，成为中国第一家异地经营的城市商业银行；随即上海银行[[宁波|宁波]]分行开业。2007年10月22日，上海银行[[南京|南京]]分行开业。2008年1月2日，上海银行[[杭州|杭州]]分行开业。2011年8月10日[[國際金融公司|國際金融公司]](IFC)出售持有的上海銀行7%股份給[[中國投資有限責任公司|中國投資有限責任公司]]。2013年9月29日，上海银行成为首批入驻[[中国（上海）自由贸易试验区|中国（上海）自由贸易试验区]]的银行之一<ref>{{cite web |publisher = [[新华网|新华网]] |date = 2013-09-29 |title = 首批入驻上海自贸区银行出炉：8家中资2家外资 |url = http://news.xinhuanet.com/fortune/2013-09/29/c_125469175.htm }}</ref>。

2013年5月收購建行（亞洲）財務銀行牌照後，隨即易名為上海銀行（香港）。<ref>[http://www.bosc-hk.com/financialfiles/Articles_of_Association_2015.pdf]上海銀行（香港）有限公司資本票據之條款及細則普通股</ref>[[上海銀行（香港）有限公司|上海銀行（香港）有限公司]]屬於有限制持牌銀行，於2013年6月7日正式啟業，是上海銀行首家境外子公司。據該行資料，上海銀行（香港）截至六月底總資產為2.61億元，普通股一級資本比率為28.7%。<ref>[http://www.bosc-hk.com/Channel/xinxi.html]上海銀行（香港）有限公司財務資訊</ref>

2013年12月10日，[[香港上海汇丰银行|香港上海汇丰银行]]将其所持有的上海银行8%的股权出售给[[桑坦德银行|桑坦德银行]]<ref>{{cite web |publisher = [[新华网|新华网]] |date = 2013-12-11 |title = 汇丰缘何出售上海银行股权 |url = http://news.xinhuanet.com/fortune/2013-12/11/c_118517500.htm }}</ref>。
