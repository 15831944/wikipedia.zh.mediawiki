'''上山口站'''（{{jpn|j=上山口駅|hg=かみやまぐちえき|rm=Kami-yamaguchi eki|f=y}}）是位於[[日本|日本]][[山口縣|山口縣]][[山口市|山口市]]道祖町3的[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）[[山口線|山口線]][[鐵路車站|鐵路車站]]。

== 車站構造 ==
[[單式月台|單式月台]]1面1線地上車站，為無站員駐守的車站。

== 車站周邊 ==
*山口紅十字病院
**山口紅十字看護専門學校
*山口縣[[紅十字血液中心|紅十字血液中心]]
*山口身體障害者福祉中心

== 历史 ==
*1953年（昭和28年）4月10日 開業。
*1987年（昭和62年）4月1日 國鐵分割民營化、成為西日本旅客鐵道車站。
