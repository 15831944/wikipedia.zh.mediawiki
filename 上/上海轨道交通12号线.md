'''上海轨道交通12号线'''， 又名闵浦线，系统代号'''M2'''。

==概览==
全線由[[闵行区|闵行区]][[莘庄镇|莘庄镇]][[七莘路站|七莘路站]]，经[[徐汇区|徐汇区]]、[[静安区|静安区]]、[[黄浦区|黄浦区]]、[[虹口区|虹口区]]、[[杨浦区|杨浦区]]，下穿[[黄浦江|黄浦江]]最终到达[[浦东新区|浦东新区]]金桥[[金海路站|金海路站]]<ref name=r3>[http://www.chinahighway.com/news/2012/648249.php 上海：轨道交通12号线开始铺轨施工]</ref>。线路全长40.417公里，均为地下线，工程总投资为233.99亿元。全線共有32座車站，设金桥车辆基地（与9号线、14号线共用）和中春路车辆基地，设控制中心1座（与8号线、10号线共用）；全线设龙华中路站主变电所（与7号线共用）和长阳路主变电所各1座。工程于2008年底动工，2013年12月29日开通[[天潼路站|天潼路站]]—[[金海路站|金海路站]]段<ref>[http://www.shmetro.com/node49/201312/con113574.htm 12号线、16号线12月29日起载客试运营]</ref>，2014年5月10日[[曲阜路站|曲阜路站]]开门迎客<ref>[http://www.sh.xinhuanet.com/2014-05/09/c_133319816.htm 12号线曲阜路站明开通 遇极堵或关闭换乘通道] {{webarchive|url=https://web.archive.org/web/20140512230052/http://www.sh.xinhuanet.com/2014-05/09/c_133319816.htm |date=2014-05-12 }}</ref>，2015年12月19日开通[[汉中路站|汉中路站]]—[[七莘路站|七莘路站]]段，至此全线通车<ref name="2015区段">{{Cite news|url = http://www.shmetro.com/node49/201512/con114518.htm|title = 12月19日起  11、12、13号线新延伸段建成试运营|last = |first = |date = |work = 上海地铁运管中心|accessdate = 2015年12月19日|publication-date = 2015年12月19日}}</ref>。代表色為<small> <font color="{{上海地铁标志色|12}}">█</font> </small>'''翠绿色'''。

12号线是上海地铁系统中可换乘线路最多的线路，全线设19个换乘站点，可换乘除5号线，16号线和17号线之外的大部分线路，其中[[漕宝路站|漕宝路站]]—天潼路站区间皆为换乘站（其中[[南京西路站|南京西路站]]为虚拟换乘）<ref>{{cite news |title=“换乘之王”12号线贯通 全线设19个换乘站点_大申网_腾讯网 |url=http://sh.qq.com/a/20150331/043672.htm |accessdate=2018-09-04 |work=腾讯网上海频道 |language=zh-CN}}</ref>。

===线路信息===
* 营运里程：40.417km
* 站数：32
* 轨距：1435mm
* 动力电源：1500V直流（架空电线方式）
* 双线区间：全线
* 电气化区间：全线
* 地下区间：[[金海路站|金海路]]—[[七莘路站|七莘路]]

== 沿革 ==
* 2005年10月12日 工程环境影响评价公示<ref>[http://www.envir.cn/an/20051012b/index.htm 上海市轨道交通12号线(M2线)工程环境影响评价公众意见调查问卷]</ref>
* 2005年12月28日 工程报建<ref>[http://www.ciac.sh.cn/xmbjwsbsweb/xmquery/XmDetail.aspx?bjbh=0501KQ0020 上海轨道交通12号线工程项目信息]</ref>
* 2006年1月19日 工程总体设计招标
* 2007年3月25日 工程环境影响评价第二次公示<ref>[http://www.envir.gov.cn/eia/2007/0325/ 上海市轨道交通12号线(M2线)工程环境影响评价公众意见调查问卷]</ref>
* 2007年11月22日 工程环境影响报告书通过批复
* 2008年3月31日 工程可行性研究报告通过批复<ref>[http://zfxxgk.ndrc.gov.cn/PublicItemView.aspx?ItemID=%7bfefd6268-384a-415c-937b-c4e0c86edd43%7d 上海市轨道交通12号线工程可行性研究报告通过批复]</ref>
* 2008年12月30日 工程正式开工建设<ref>[http://why.eastday.com/q/20081230/u1a518372.html "第二换乘大户"轨交12号线开工]</ref>
* 2012年3月29日 轨道工程举行首铺仪式<ref>[http://www.shmetro.com/node55/node57/201203/con111331.htm 上海轨交12号线轨道工程举行首铺仪式]</ref>
* 2012年12月9日 东段（[[天潼路站|天潼路站]]—[[金海路站|金海路站]]）洞通<ref>[http://www.shmetro.com/node49/201212/con112545.htm 12号线东段土建工程实现结构贯通]</ref>
* 2012年12月31日 金桥停车场地面检修库房在施工过程中浇筑平台发生坍塌，造成5人死亡，18人受伤<ref>[http://news.ifeng.com/mainland/detail_2013_01/01/20713425_0.shtml 上海地铁12号线工地遭塔吊压塌 已致5死18伤]</ref>
* 2013年3月25日 东段（[[曲阜路站|曲阜路站]]—金海路站）洞通
* 2013年12月29日 东段（天潼路站—金海路站）开通试运营
* 2014年5月10日 曲阜路站开通<ref>{{Cite web |url=http://www.shmetro.com/node49/201405/con113704.htm |title=12号线曲阜路站5月10日起开站迎客 |accessdate=2014-05-10 |publisher=上海地铁运管中心 |date=2014-05-05 |}}</ref>
* 2014年9月28日 西段（[[汉中路站|汉中路站]]—[[七莘路站|七莘路站]]）洞通<ref>[http://shanghai.xinmin.cn/msrx/2014/09/28/25525189.html 12号线西段结构贯通 明年底有望通车]</ref>
* 2015年3月25日 西段（汉中路站—七莘路站）轨通
* 2015年4月15日 西段（汉中路站—七莘路站）电通
* 2015年6月12日 西段（汉中路站—七莘路站）热滑开始

== 未来发展 ==
根据《松江区国家新型城镇化经综合试点总体规划（方案）》，明确提出延伸轨道交通12号线，向西衔接漕河泾开发区（松江）分区，继而延伸至9号线[[洞泾站|洞泾站]]<ref>[http://sh.sina.com.cn/news/b/2016-06-29/detail-ifxtmwei9486384.shtml 上海松江新城面积翻倍 规划将沪昆铁路南迁]</ref>。在G60北侧通道，预留了设置轨道交通康城站的空间，目前规划尚未确定<ref>[http://city.eastday.com/gk/20180117/u1ai11152117.html 一号线西延？康城通轨交？莘庄镇交通“利好”消息不断]</ref>。
