《'''上将许世友'''》，中国大陆电视剧，导演[[安澜|安澜]]，主演[[张秋歌|张秋歌]]、[[范明|范明]]、[[东方子|东方子]]、[[刘劲|刘劲]]、[[王伍福|王伍福]]、[[卢奇|卢奇]]等。

== 剧情 ==
{{See Also|许世友}}
该剧以1985年许世友逝世前回光返照追忆战争岁月的倒叙方法讲述了许世友参军打仗的战斗生涯。

== 花絮 ==
该剧是“纪念许世友100周年诞辰”的献礼剧。<ref>{{cite web|url=http://news.longhoo.net/gb/longhoo/news/ent/userobject1ai585444.html|title=献礼将军百年诞辰 央视强势推出《上将许世友》|accessdate=2006-12-18|publisher=龙虎网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20090109055459/http://news.longhoo.net/gb/longhoo/news/ent/userobject1ai585444.html|archivedate=2009-01-09}}</ref>

该剧由北京懋伯兰文化传播有限责任公司、江苏省文化产业集团、[[八一电影制品厂|八一电影制品厂]]、[[中国唱片总公司|中国唱片总公司]]、[[南京军区|南京军区]]前线文工团共同制作。<ref>{{cite web|url=http://www.ccdy.cn/chanye/chuanmei/201109/t20110926_82964.htm|title=《上将许世友》压轴央视年底大戏|accessdate=2006-12-22|publisher=中国文化传媒网|work=|language=zh-cn|deadurl=yes|archiveurl=https://archive.is/20130424152416/http://www.ccdy.cn/chanye/chuanmei/201109/t20110926_82964.htm|archivedate=2013-04-24}}</ref>

中共官方对该剧总体持肯定态度，总政领导批示：“这是一部好作品，有情有义、有血有肉，光辉形象、跃然心中。”国家重大革命历史题材领导小组：“这是重大革命历史题材的一个突破，人物传记片的一个典范。”<ref>{{cite web |url=http://military.people.com.cn/GB/42968/57661/5828353.html |title=电视剧《上将许世友》：浓浓硝烟塑造中国军魂——导演电视剧《上将许世友》随想 |accessdate=2007年6月6日 |publisher=人民网 |work= |language =zh-cn}}</ref>

2006年12月18日，该剧在[[中国中央电视台综合频道|中国中央电视台综合频道]]首播。<ref>{{cite web |url=http://ent.sina.com.cn/x/2006-12-18/08551374183.html |title=电视剧《上将许世友》今晚央视开播 |accessdate=2006年12月18日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

== 争议 ==
* 该剧由于穿帮镜头、简繁字体不符合时代背景、亲友关系辈分称呼错乱等引发拍砖。
* 2007年，作家陈某在[[北京市|北京市]][[海淀区|海淀区]]人民法院状告该剧侵权其作品《许世友传奇》、《许世友外传》。<ref>{{cite web |url=http://www.sipo.gov.cn/albd/2007/200804/t20080402_366278.html |title=《上将许世友》被诉侵犯著作权 |accessdate=07-03-08 |publisher=中华人民共和国国家知识产权局 |work= |language=zh-cn }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url=http://news.xinhuanet.com/legal/2007-03/02/content_5791222.htm |title=电视剧《上将许世友》被诉侵权 |accessdate=2007年3月2日 |publisher=新华网 |work= |language =zh-cn}}</ref>
