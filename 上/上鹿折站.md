'''上鹿折車站'''（{{jpn|j=上鹿折駅|hg=かみししおりえき|rm=Kami-shishiori eki|f=y}}），位於[[日本|日本]][[宮城縣|宮城縣]][[氣仙沼市|氣仙沼市]]，為[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[大船渡線|大船渡線]]的[[鐵路車站|鐵路車站]]。

1932年3月19日至1933年12月4日間，曾是一個[[鐵路車站#終點站|終點站]]。

== 車站構造 ==
[[對向式月台|對向式月台]]2面2線的[[地面車站|地面車站]]。各月台以站內[[平交道|平交道]]連絡。[[氣仙沼站|氣仙沼站]]管理的[[無人站|無人站]]。

== 利用狀況 ==
*招呼站，由[[氣仙沼車站|氣仙沼車站]]管理

== 歷史 ==
* 1932年3月19日 - 設站。
* 1986年11月1日 - 改為[[無人車站|無人車站]]，並指定由[[氣仙沼車站|氣仙沼車站]]管理。
* 1987年4月1日 -  [[日本國鐵分割民營化|日本國鐵分割民營化]]後成為JR東日本的車站。
* 2011年3月11日 - 被[[東北地方太平洋近海地震|東北地方太平洋近海地震]]所引發的[[海嘯|海嘯]]沖毀
* 2013年3月2日 - 變為[[BRT|BRT]]車站，不再以鐵路行走。

== 車站週邊 ==
*氣仙沼市立白山國小
*縣道34號氣仙沼陸前高田線
*縣道211號上鹿折停車場線
