'''上海市教育委员会'''，简称'''上海市教委'''，是主管[[上海市|上海市]]地方权限各类教育的[[上海市人民政府|上海市人民政府]]组成部门。同时领导[[上海市教育考试院|上海市教育考试院]]。上海市教育委员会机关行政编制为194名；其中，主任1名、副主任6名、秘书长1名，正副处级领导职数62名。<ref>{{cite web |url = http://www.shmec.gov.cn/web/xxgk/index.php |title = 上海教育：信息公开 |publisher = 上海市教育委员会 |access-date = 2010-08-27 |archive-url = https://web.archive.org/web/20100822041139/http://www.shmec.gov.cn/web/xxgk/index.php |archive-date = 2010-08-22 |dead-url = yes }}</ref>

== 现任领导 ==
# 主任：薛明扬
#* 全面负责市教委工作
#* 分管：研究室、办公室、信访办公室、财务处
#* 联系：上海市教委财务与资产管理中心、上海市教育评估院
# 副主任：高德毅
#* 分管：民办教育管理处、学校后勤保卫处、德育处
#* 联系：[[上海教育报刊总社|上海教育报刊总社]]、上海市教育后勤管理中心、上海高校后勤服务中心
#* 协助联系：[[上海开放大学|上海远程教育集团]]（[[上海教育电视台|上海教育电视台]]）
# 副主任：李瑞阳
#* 分管：发展规划处、学生处、国际交流处
#* 联系：上海市教育考试院、上海市教育科学研究院、上海市学生事务中心、上海市教育基建管理中心
# 副主任：李骏修
#* 分管：人事处、体育卫生艺术科普处、审计处
#* 联系：上海市科技艺术教育中心、上海市教育人才交流服务中心、上海市青少年校外活动营地（[[东方绿舟|东方绿舟]]）、上海师资培训中心、上海市教育发展基金会、上海市中小幼教师奖励基金会、上海申教投资有限公司、上海市教育发展有限公司 
# 巡视员：尹后庆
#* 分管：基础教育处、教育督导室、青少年保护工作处
#* 联系：上海市教委教学研究室、上海市教育技术装备部、上海市教育督导事务中心、上海中学、上海交通大学附属中学、复旦大学附属中学、上海师范大学附属中学、上海市实验学校、上海师范大学第四附属中学、上海市聋哑青年技术学校、上海市盲童学校、上海市行政管理学校（B角）
# 副主任：印  杰
#* 分管：职业教育处、高等教育处、科学技术处（信息化工作）
#* 联系：上海市教委信息中心、上海市经济管理学校、上海市行政管理学校（A角）                    
# 副主任：袁  雯
#* 分管：终身教育处、科学技术处、政策法规处、语言文字管理处
#* 联系：上海市教委科技发展中心、上海市语言文字水平测试中心、[[上海老年大学|上海老年大学]]、上海市高校浦东继续教育中心
#* 协助联系：上海远程教育集团（[[上海开放大学|上海开放大学]]）
