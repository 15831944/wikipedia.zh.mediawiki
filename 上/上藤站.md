'''上藤站'''位于[[福建省|福建省]][[福州市|福州市]][[仓山区|仓山区]]上藤路、下藤路与六一南路交叉路口，是[[福州地铁1号线|福州地铁1号线]]的一座{{FZM Dater|1|在建|}}中间站，也是闽江南岸的首个站点，车站于2011年8月20日动工<ref>[http://news.fznews.com.cn/fuzhou/2011-8-18/2011818XSE3RGX9PO203432.shtml 地铁上藤站周六起围挡　上藤路交通基本不受影响]</ref>，2017年1月6日开通。

== 车站结构 ==
上藤站位于仓山区上藤路、下藤路与六一南路交叉路口，基本呈东南至西北走向，主体结构为双柱三跨地下三层车站，岛式站台，地下一层为站厅层，地下二层为设备层，地下三层为设备层。<ref>{{cite news|url=http://news.fznews.com.cn/node/9900/20161108/58210cb389e7d.shtml|title=地铁上藤站年底具备通车条件　上藤路将恢复通车|date=2016-11-08|accessdate=2017-01-20}}</ref>车站主体全长137.6米，宽23.2米，深22.5米。<ref>{{Cite web|url=http://news.ycwb.com/2012-10/23/content_4064435.htm|title=“大家伙”助推福州地铁建设上藤站工期缩短八个月_新闻中心_金羊网|accessdate=2017-04-01|last=www.ycwb.com|work=news.ycwb.com|archive-url=https://web.archive.org/web/20170401232722/http://news.ycwb.com/2012-10/23/content_4064435.htm|archive-date=2017-04-01|dead-url=yes}}</ref>

=== 站厅 ===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = 20170325 Concourse of Shangteng Station.jpg | caption1 = 车站站厅
}}
上藤站站厅位于上藤路下方。车站站厅所使用吊顶以斜式屋檐为基础设计而成，意为展示福州仓山区烟台山近代建筑群文化，墙面装修采用白色搪瓷板。车站设客服中心1处，分布于站厅南端，服务设施设置智能导乘机2处、自动售货机2处，B出入口通道旁规划设有[[便利店|便利店]]，A出入口通道设有洗手间。

=== 站台 ===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = 20170325 Platform of Shangteng Station.jpg  | caption1 = 上藤站站台层
}}
上藤站共设置2个分别来往于[[象峰站|象峰]]、[[福州火车南站站|福州火车南站]]的1号线站台，采用岛式站台设计。车站站台采用白色斜式屋檐式吊顶与白色搪瓷板装修，服务设施设有智能导乘机与自动售货机。
