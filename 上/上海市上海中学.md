'''上海市上海中学'''简称“上海中学”或“上中”，始创于[[清朝|清朝]][[同治|同治]]四年（1865年），原名[[龙门书院|龙门书院]]，1927年改名'''江苏省立上海中学'''，是享誉全国的[[江南四大名中|江南四大名中]]之一。1950年改为'''上海市上海中学'''。

上海市上海中学是[[上海市教育委员会|上海市教育委员会]]直属的寄宿制[[上海市实验性、示范性高中名单|实验性、示范性高中]]，一般称为“市重点高中”。

== 历史沿革 ==
上海中学创始于1865年10月17日。担任苏淞太道兵备的[[丁日昌|丁日昌]]顺应洋务潮流“兴新学”，捐银倡设龙门书院，成为上海开埠以来最早的地方官办新学。1905年，学校承“革旧习、必立师范学堂”[[戊戌变法|戊戌变法]]要义，改龙门书院为龙门师范。[[辛亥革命|辛亥革命]]近代师范又以省立为制，校名遂改为江苏省立第二师范。1937年[[抗日战争|抗日战争]]全面爆发后，校园被日本军队占领，成为关押英美侨民的集中营。为拒绝向[[汪精卫政权|汪伪政权]]登记，改名“沪新中学”。抗战胜利后，上海中学得以迁回现址复校。 中华人民共和国成立后，学校从江苏省划出，成为上海市上海中学。[[文化大革命|文革]]期间，上海中学成为重灾区，直至1970年被迫宣布停办。1978年，经上海市委、市府同意复校。

== 学校概况 ==
1993年6月，上中创办了上海市第一所由华人管理的国际性学校──上海中学国际部，1995年成为上海第一所国际文凭学校。2003年成为[[联合国教科文组织|联合国教科文组织]]亚太地区联系学校。上海中学是历史悠久的著名中学，为[[江南四大名中|江南四大名中]]之一。上海也有四大名校的说法：上海市上海中学，[[华东师范大学第二附属中学|华东师范大学第二附属中学]]，[[复旦大学附属中学|复旦大学附属中学]]和[[上海交通大学附属中学|上海交通大学附属中学]]。

上中歷史上共有8名學生獲得了[[国际数学奥林匹克|国际数学奥林匹克]](IMO)金牌。<ref name="京沪高中生在历届国际奥赛的获奖情况">{{cite web| language =中文| title =京沪高中生在历届国际奥赛的获奖情况| url =http://bbs.eduu.com/thread-774480-1-1.html}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 本部学生重点大学录取率超过97%。近几年考入[[北大|北大]]、[[清华|清华]]、[[复旦|复旦]]、[[交大|交大]]四所名校超过60%，06年达到65.5%。[[高考状元#上海|上海高考2007年理科状元]]、2008年文理科状元、2009年文理科状元、2010年文科状元，2011年文理科状元、2012年文理科状元均出自上中。本部学生获[[世界数学奥赛|世界数学奥赛]]与世界中学生乒乓球锦标赛金牌22枚，银牌9枚。2003－2006年获世界、全国、上海市各类竞赛奖项700多个。其中有获世界因特尔大赛、中国青少年科技创新奖(邓小平奖)等高层次奖多项。

== 建筑 ==
* 龙门楼
上中的主教学楼命名为龙门楼，其含意十分明显，要求上中学生“鲤鱼跳龙门”，此典故在《三秦记》中叙述如下：“龙门山，……每岁春季有黄鲤鱼，自海及诸川争来赴之。一岁中，登龙门者，不过七十二。初登龙门，即有云雨随之，天火自后烧其尾，乃化为龙。”奋勇向前，由鱼升龙，即成为上中文化的形象标志。于是上中的标识纹章用了鲤鱼跳龙门的抽象图案。龙门楼装修时，门厅刻上了“龙”字的115种写法，中央吊灯的支架是一条条力争上游的鲤鱼，即使建筑的基色，也以灰黄象征辉煌。
[[File:Longmen_Building_and_Flag.jpg|thumb]]

* 念慈楼
图书馆，包括有期刊杂志阅览室，电子网络阅览室，校史陈列室，一号二号图书室。20世纪90年代中期建造，早期底楼还设有自修室，部分二楼房间曾做过美术教室。

* 逸夫楼
实验楼，包括有互联网自算机房，物理实验室，电学实验室，化学实验室，生物实验室，多媒体教室，会议室，每层楼面还设有较大的阶梯教室。20世纪90年代中末期建造，克林顿98年访华期间，重新装修了几间盥洗室。
[[File:Yifu_Building_3.jpg|thumb]]

* 中兴楼
原址位于前侧的跃鲤湖，[[淞沪抗战|淞沪抗战]]期间原中兴楼被日机炸毁，留有跃鲤湖作证，新楼初为上海中学初中部所使用，20世纪90年代后期，因学校改制，扩招国际部，此楼也陆续改为上海中学国际部使用，楼的两侧曾作为国际部的部分宿舍。
楼前跃鲤湖取自鲤鱼跃龙门之意，后因中兴楼一般被称为中兴湖。湖上有九曲桥和石亭，湖内有石质鲤鱼一尊。

* 先棉堂
因20世纪30年代迁校建址期间破坏了原先的[[黄道婆|黄道婆]]墓道，所以将该楼命名为先棉堂，以纪念这位宋朝的手工纺织业名师。20世纪90年代初期以前作为生物，化学，物理等实验室，国际部成立之后，改为国际部使用。
[[File:Xianmiantang.jpg|thumb]]

* 喷水池
20世纪90年代后期改作一个圆形大花坛。

* 大礼堂
艺术节演出，各种报告会演讲会的举办场所。20世纪90年代末期因计划接待美国总统[[克林顿|克林顿]]访问而获得资金修缮(后临时改由克林顿总统的夫人[[希拉里·克林顿|希拉里·克林顿]]访问[[市三女中|市三女中]])。
* 方厅
大礼堂的侧厅。也是各类小型报告会的举办场所。

== 学生生活 ==
艺术节是一年一度的学生活动，为期约两周，一般在秋季学期结束之前举行。以班级(除高三毕业班外)为单位参加各类文艺比赛，包括民族舞、现代舞、独唱、小组唱、小品等。此外还有科技節，為期两周，其中较多的比赛项目被安排在春季其中一個星期二的下午舉行。
