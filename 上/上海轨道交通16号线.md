'''上海轨道交通16号线'''（系统代号'''R3s'''），曾用名临港线、上海轨道交通21号线、[[上海轨道交通11号线|11号线]]南段。

== 概览 ==
16号线为[[上海市|上海]]一条轨道交通路线，北起[[龙阳路站|龙阳路站]]，经罗山路、周浦镇、航头镇、新场镇、宣桥镇、惠南新城至[[滴水湖站|滴水湖站]]。全长58.962公里，其中地下线路长13.694公里，高架线路长45.268公里；共设站13座，其中地下站3座，高架站10座，最大站间距10.625公里，最小站间距2.558公里。设川杨河定修段（与11号线、13号线共址）及治北停车场、3座主变电站（分别位于罗山路、惠南、临港北）、1座控制中心（纳入隆德路控制中心与11号线、13号线共址统一调度管理）<ref>{{cite web | title = 轨交11号线南段规划调整(原载解放日报)| publisher = eastday.com | date = 2009-07-23 | url = http://jfdaily.eastday.com/j/20090723/u1a604526.html | accessdate = 2009-07-23 | language = zh }}</ref>，工程总投资约181.6亿元<ref name=zty>{{cite web|author=2017-04-17 17:33 |url=http://www.sohu.com/a/134664821_249521 |title=上海地铁16号线工程喜获第十四届“詹天佑奖” |language=en |publisher=Sohu.com |date=2017-04-17 |accessdate=2018-05-25}}</ref>。其中[[书院站|书院]]至[[惠南东站|惠南东]]区间为目前上海轨道交通两站之间最长区间。[[罗山路站|罗山路站]]至滴水湖站长52.850公里，于2013年12月29日建成通车<ref>{{cite web|author=2013-12-28 06:39 |url=http://sh.qq.com/a/20131228/003773.htm |title=“双龙齐飞” 沪轨交全网总长567公里世界第一 |publisher=Sh.qq.com |date=2013-12-28 |accessdate=2018-05-25}}</ref>，而龙阳路站至罗山路站于2014年年12月28日启用。列车最高时速120公里，使用A型车，三轨受流（亦設有[[受电弓|受电弓]]）。线路标识色为<span style="background-color:#{{上海地铁标志色|16}}"><span style="color:black;">Pantone 3255C</span></span>（C=44 M=0 Y=30 K=0）<ref name=R3scolor>{{cite web | title = 《上海轨道交通网络近期规划建设线路标志色方案》征求市民意见公告| publisher = 上海地铁 | date = 2011-09-09 | url = http://www.shmetro.com/node49/201109/con109210.htm | accessdate = 2017-06-17 | language = zh }}</ref>。

本线报站除了采用[[普通话|普通话]]、[[英语|英语]]之外，还使用了[[沪语|沪语]]报站。这是[[上海地铁|上海地铁]]首次使用沪语报站。

=== 线路数据 ===
* 营运里程：58.962km
* 站数：13
* 轨距：1435mm
* 动力电源：1500V[[第三轨供电|第三轨供电]]（车辆段为接触网供电）
* 双线区间：全线
* 电气化区间：全线
* 地下区间：[[惠南站|惠南]]西—惠南东、[[临港大道站|临港大道]]北—[[滴水湖站|滴水湖]]
* 高架区间：[[龙阳路站|龙阳路]]—惠南西、惠南东—临港大道北

== 运营组织 ==
16号线在最初设计上，安排了普通车（站站停）、大站车、直达车三种运行模式<ref name=zty />。运行初期采取普通车和大站车结合的模式开行<ref>{{cite web|url=http://www.shmetro.com/node49/201312/con113574.htm |title=12号线、16号线12月29日起载客试运营 |publisher=Shmetro.com |date=2013-12-27 |accessdate=2018-05-25}}</ref>。当前实际运营时间为，普通车全程57分钟，大站车全程46分钟。

自2013年12月29日起开行的大站车停靠[[羅山路站|羅山路站]]、[[新場站_(上海)|新場站]]、[[惠南站|惠南站]]以及[[滴水湖站|滴水湖站]]。于工作日早晚高峰期间开行，每天16对。后于2014年1月30日暂时取消大站车，站站停普通列车运行间隔从原来的20分钟缩短到10分钟左右<ref>{{cite web | url=http://sh.eastday.com/m/20140131/u1a7916267.html | title=16号线暂时取消大站车 列车间隔缩一半 | publisher=东方网 | date=2014-01-31 | accessdate=2014-02-02}}</ref>。2016年3月21日起，16号线恢复大站车运营模式<ref>{{cite web | url=http://weibo.com/1742987497/DmPGBi1Zl?refer_flag=1001030103_&type=comment#_rnd1458796623314 | title=好消息：3月21日起 16号线开行大站车 | publisher=上海地铁官方微博 | date=2016-03-18 | accessdate=2016-03-24}}</ref><ref>[http://service.shmetro.com/yygg/1342.htm 16号线3月21日起开行大站车] {{webarchive|url=https://web.archive.org/web/20160412225839/http://service.shmetro.com/yygg/1342.htm |date=2016-04-12 }}上海地鐵官方2016-03-18 17:08:38。</ref><ref>[http://wap.eastday.com/node2/n112/n114/u1ai576344_t71.html 沪轨交16号线重开“大站车”部分站点仍限流]翱翔</ref>，并将运行范围扩大到[[龙阳路站|龙阳路站]]，运营时间覆盖到工作日和节假日。但值得注意的是，大站车的运营并未包括工作日早高峰开往龙阳路方向的通勤班次。

== 争议 ==
===拥挤===
自16号线剩余段于2014年开通以来，工作日早晚高峰拥挤不堪。地铁运营方表示，这是因为工作日16号线的潮汐现象非常严重。<ref>[http://www.sh.xinhuanet.com/2014-12/31/c_133888727.htm 16号线拥挤原因 “潮汐现象”严重 暂无法增能] {{webarchive|url=https://web.archive.org/web/20151001003938/http://www.sh.xinhuanet.com/2014-12/31/c_133888727.htm |date=2015-10-01 }}</ref>。而外界则认为地铁列车仅3节编组才是根本原因。后续，地铁运营方为缓解车厢拥挤，16号线拆除部分车厢座椅增加载客量<ref>{{cite web|url=http://sh.sina.com.cn/news/m/2015-01-20/detail-icczmvun5206495.shtml?from=wap |title=16号线列车拆除12个双人座增容 直放大站车疏导拥堵车站客流 |publisher=Sh.sina.com.cn |date=2015-01-20 |accessdate=2018-05-25}}</ref>。同时地铁运营方在部分站点采取了限流措施，另外调整了调度，使得工作日[[龙阳路站|龙阳路站]]与[[惠南站|惠南站]]之间的运行间隔缩短。面对抱怨，轨交运营方表示最初16号线为“旅游专线”进行设计，同样被质疑为16号线沿线的大型居住基地早于16号线启动建设。而在16号线开通前，地铁方也通过新闻媒体发布了“分分钟三节变六节”的报道，来回应社会各界对16号线三节编组是否能满足高峰运输需要的担忧。对于增能的延误，地铁方开始用“16号线欲增开班次、加挂车厢，则涉及信号系统、运营、安全测试、演练等一系列问题，难以在短时间内解决”的说法来解释，逐步增加了6节编组车辆<ref>{{cite web|author=2015-01-08 14:58:52 |url=http://shanghai.kankanews.com/c/2015-01-08/0016222844.shtml |title=16号线实行高峰特别调度+限流 6节车要等至少22个月 |publisher=Shanghai.kankanews.com |date=2015-01-08 |accessdate=2018-05-25}}</ref>。

===限速===
16号线的设计时速为120公里/小时，实际最高运行速度约100公里/小时。站站停列车的全程行驶时间不断拉长。
同时，16号线还容易受大雾影响进一步限速。上海地铁运营方解释称，之所以16号线限速，主要是由于该线路时速快、行驶中对瞭望的要求高，因此遇到大雾、强对流等天气都需要及时限速<ref>{{cite web|url=http://shanghai.xinmin.cn/msrx/2014/08/21/25167407.html |title=地铁16号线今晨早高峰遇大雾限速两小时 |publisher=Shanghai.xinmin.cn |date= |accessdate=2018-05-25}}</ref>。

===大站车===
16号线通车之初开行了“大站车”，但2014年1月30日宣布暂时取消，只开行站站停列车并缩短运行间隔。对于新场、惠南、滴水湖等原来大站附近的居民，暂停“大站车”后出行用时略有拉长。家住临港的王先生表示，他已习惯6点半左右从滴水湖坐“大站车”到罗山路换11号线，现在坐站站停列车要多花15分钟以上。而16号线项目方人士表示，根据近期客流特征及乘客需求，目前暂时取消了“大站车”。一旦客流变化大乘客需求增加，“大站车”还会再回来，届时有望尝试“一站直达”等新模式。全天站站停模式，“需要开行一段时间接受乘客检验。而大站车再开行，需要凭借丰富的客流数据作为依据，但应该不会太久。”地铁运营方一名工作人员透露，大站车”“再回来时应该更优化，在新车到位、增能，甚至考虑调整大站站点后，16号线再开行大站车时将相对更合理<ref>{{cite web|author=马景华 |url=http://sh.eastday.com/m/20140219/u1a7939340.html |title=16号线“大站车”归期未定 下一步运营方案还未出炉 |publisher=Sh.eastday.com |date=2014-02-19 |accessdate=2018-05-25}}</ref>。2016年3月21日，16号线“大站车”模式重启运营<ref>{{cite news|url=http://www.thepaper.cn/newsDetail_forward_1445922 |title=上海地铁16号线将开行大站车，单程运行时间缩短约12分钟|work=澎湃新闻|date=2016-03-19}}</ref>。

===车钩断裂===
2015年2月7日，一则有关16号线列车断裂的微博引来网友热议。随即上海地铁表示，“上海地铁16号线由于车钩问题在行驶途中断掉了”其实是网友的讹传。微博照片上断开的列车，其实位于维修基地内，正在接受相关性能测试及工作演习，为列车扩编做准备<ref>{{cite web|author=2015-02-08 07:21 |url=http://sh.qq.com/a/20150208/006534.htm |title=网曝“16号线车厢脱钩断裂” 实为接受性能测试 |publisher=Sh.qq.com |date=2015-02-08 |accessdate=2018-05-25}}</ref>。但实际上，车钩断裂是测试中的意外，当时正在进行4列车连挂救援实验，其中编号为1602的列车突然紧急制动，而其他三列正常运行，导致1602的车钩以及贯通道被拉断。该事故是真实发生的，只是没有发生在正常载客运行中。
