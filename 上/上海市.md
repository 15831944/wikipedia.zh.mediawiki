'''上海，'''简称'''沪'''，别称'''申'''，是[[中華人民共和國|中華人民共和國]][[直辖市#_中华人民共和国|直轄市]]，也是[[中国经济|中国经济]]发达的城市，国际金融、贸易和航运中心{{RefTag |1 = {{cite web |url = http://www.cw.com.tw/article/article.action?id=5003450 |publisher = 天下杂志 |title = 誰能稱霸亞洲金融中心？ |accessdate = 2016-11-25 }} }}，[[上海港|其港口]]为[[世界上最繁忙的港口|全球最繁忙的港口]]；主要产业包括[[商贸流通业|商贸流通]]、[[金融|金融]]、[[信息产业|-{zh-tw:資訊;zh-hans:信息}-]]、[[制造业|制造]]等。上海位於[[中国东部|中国东部]]弧形海岸线的正中间，[[长江三角洲|长江三角洲]]最东部，东临[[东海|东海]]，南濒[[杭州湾|杭州湾]]，西与[[江苏|江苏]]、[[浙江|浙江]]两省相接，北端的[[崇明岛|崇明岛]]处于[[长江入海口|长江入海口]]中，佔地面積6,340平方公里。上海市是[[世界最大城市|世界最大的城市之一]]，截至2018年，人口2423.78万，其中[[上海人|本地户籍人口]]占59.72%，达1447.57万{{RefTag |name = "2018年上海市国民经济和社会发展统计公报" |1 = {{cite web |title = 2018年上海市国民经济和社会发展统计公报 |author1 = [[上海市统计局|上海市统计局]] |author2 = [[国家统计局|国家统计局]]上海调查总队 |website = 上海市统计局 国家统计局上海调查总队 |date =2019-03-01 |language = zh |accessdate =2019-03-01 |url =http://www.stats-sh.gov.cn/html/sjfb/201903/1003219.html }} }}；近年来，上海市也与周围的江苏、浙江两省高速发展的多个城市共同构成[[长江三角洲城市群|长江三角洲城市群]]，是世界几大城市群之一{{RefTag |1 = {{cite news |author = 中国广播网 |title = 长三角跻身6大世界级城市群 |url = http://news.xinhuanet.com/city/2012-04/10/c_122954744.htm |accessdate = 2013-10-17 |website = 新华网 |date = 2012-04-10 }} }}。2018年生产总值為32679.87億元人民幣，按同年國際匯率折合4,872.94億美元，為世界一大經濟區域；人均生产总值則為135000元人民幣，按同年國際匯率折合20,130.02美元。2018年上海居民全年人均可支配收入為64183元人民幣{{RefTag |name = "2018年上海市国民经济和社会发展统计公报"}}。《2016年中国城市可持续发展报告：衡量生态投入与人类发展》中，人类发展指数达到0.848，位列中国主要城市第6位。

晋代，上海初步发展为一个渔港、盐产地和商贸集镇。唐代到元代，上海地区归[[华亭县_(唐朝)|华亭县]]、[[松江府|松江府]]管辖。明清两朝，已较为繁荣，棉纺织业发达{{RefTag |1 = {{cite web |title = 松江美布“衣被天下” |publisher = 文汇读书周报 |date = 2009-07-31 |url = http://www.dajianet.com/news/2009/0805/85622.shtml |accessdate = 2011-03-12}} }}。不過讓上海成為舉足輕重城市地位的，是1843年根据《[[南京条约|南京条约]]》的[[上海开埠|正式开埠]]，上海作为指定通商五口之一，由此开始[[上海租界|上海租界]]的历史。得益独特的政治环境，上海经济凭借口岸，空前迅速的发展，吸引苏、浙、粤、皖、鲁等周边省份及外国的移民，成为中国乃至[[远东|远东]]地区最大的都会至今{{RefTag |1 = {{cite web |title = 上海的开埠地文化及其民俗流变 |website = 中国网 |date = 2009-11-30 |url = http://www.china.com.cn/aboutchina/zhuanti/09dfgl/2009-11/30/content_18981086.htm |accessdate = 2011-03-13 }} }}。吸收[[江南|江南]]地区附近传统的[[吴越文化|吴越文化]]与各地移民带入的[[多元文化|多样文化]]（包括開埠後的西方近现代文化）融合，本地逐渐形成特有的[[海派文化|海派文化]]。

== 释名 ==

上海之名，最早见于《[[宋会要辑稿|宋会要辑稿]]》中北宋设置的上海务，之后熙宁年间成书的《水利书》记载了上海浦{{RefTag |name=diming |1={{cite book|editor=陈征琳 |title=上海地名志 |date=1998-09 |publisher=上海社会科学院出版社 |location=上海 |isbn=7-80618-527-5 |page = 98 }} }}。宋代，[[吴淞江|吴淞江]]下游有一条名为上海浦的支流。"浦"一词原指河川汇合之所或入海处，但在吴地常指河川的小支流。“上”一词，在[[江東|江左]]地区的河流名称中，又常指靠近源头一面。反之，下即指靠近河口附近的地区。在今虹口的[[提篮桥|提篮桥]]地区尚有[[下海庙|下海庙]]的地名。[[宋朝经济|宋朝经济]]发达，催生了酒业的发展，政府在上海浦附近设置酒务以征酒税，称为上海务，上海一名由此传播{{RefTag |1 = {{cite web |title=上海为什么叫上海“下海”又在哪里 |publisher=腾讯读书 |date=2009-12-07 |url = http://book.qq.com/a/20091207/000013.htm |accessdate=2011-03-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20111109100625/http://book.qq.com/a/20091207/000013.htm |archivedate=2011-11-09 }}}}。

上海地名的来源有多种说法。一说上海因上海浦而得名。一说因“地居海之上洋”而得名。另一种则以“海外商舶在此登岸”而名为上海{{RefTag |name=diming}}。

此外，上海的简称“-{滬}-（-{沪}-）”来源于古时当地人创造的一种名为“扈”的捕鱼工具，东晋时[[苏州河|松江]]入海口称之为沪渎（今上海市青浦区东北旧青浦镇西）{{RefTag |name=hudu|1={{cite web |title=第四节河流 - 上海市地方志办公室 |publisher=上海市地方志办公室 |date=2008-07-04 |url = http://www.shtong.gov.cn/Newsite/node2/node2247/node4563/node79108/node79119/userobject1ai102190.html |accessdate=2008-07-04 }}}}。《[[晉書|晉書]]》和《[[梁书|梁书]]》就曾多次提到過“沪渎”{{NoteTag|《晉書》卷83：（袁）山松历显位，为吴郡太守。孙恩作乱，山松守沪渎，城陷被害。}}{{NoteTag|《梁书》卷56：（侯景）乃与腹心人数十单舸走，推堕二子于水，自沪渎入海至胡豆洲。}}。别称“申”则因相传上海西部地区曾为[[楚国|楚国]]公子[[春申君|春申君]]的封地而得名{{RefTag |1 ={{cite web|url=http://www.sh.xinhuanet.com/shgk/lishi01.htm|publisher=新华网上海频道|title=“申”、“沪”的由来 |accessdate=2009-08-01 }}}}。

== 历史 ==

=== 早期历史 ===

約7000年前，上海西部已形成陆地，东部地区则为近2000年陆续形成。上海境内最早的人类文明遗迹可追溯至6000年前的[[新石器时代|新石器时代]]{{RefTag |1 = {{cite web |title = 上海-{崧}-泽遗址发现座新石器时代墓穴 |publisher = 人民网 |date = 2004-05-13 |url = http://www.people.com.cn/GB/keji/39327/3095906.html |accessdate = 2011-03-12}}}}，有[[马家浜文化|马家浜]]、[[崧澤文化|-{崧}-泽]]、[[良渚文化|良渚]]、[[马桥文化|马桥文化]]的遗迹{{RefTag |name=上海松江广富林遗址发掘取重大成果 |1 = {{cite web |title = 上海松江广富林遗址发掘取重大成果 |publisher =中国国家地理 |date = 2009-01-08 |url = http://discover.news.163.com/09/0108/10/4V4MBB8G000125LI.html |accessdate = 2011-03-09 }}}}。公元4至5世纪时的[[晋朝|晋朝]]，[[吴淞江|松江]]{{NoteTag|即现在的[[苏州河|苏州河]]。三国吴韦昭《国语·越语》注、晋郭璞《水经·沔水注》、《[[后汉书|后汉书]]·左慈传》均提到松江，《陈书·侯瑱传》“与（侯）景战于吴松江”和《梁书》卷56也提到[[侯景|侯景]]曾经来到过松江。此“吴松江”乃谓吴地之松江，非说江名叫吴松江。《通典》《元和郡县志》吴县下皆有松江。唐陆广微《吴地记》“松江一名松陵，又名笠泽也”。北宋郏亶、郏乔、单锷等水利书中松江、吴淞江两名已并见，吴淞江或作吴松江。元、明以后习称吴淞江，松江之名遂不用作江名{{RefTag |name=hudu}}。}}和滨海一带即有渔民聚集，逐渐发展为一个渔港和商贸集镇。[[唐朝|唐]][[天宝_(唐朝)|天宝]]十载（751年），设立[[华亭县_(唐朝)|华亭县]]，隸屬[[苏州_(隋至北宋)|蘇州]]{{NoteTag|《新唐書》卷四十一：華亭。上。天寶十載析嘉興置。}}，范围北到今天的[[虹口|虹口]]一带，南到海边，东到下沙（今[[浦东新区|浦东新区]][[航头镇|航头镇]]北部），此为松江建城之始{{RefTag |1 = {{cite web|title=上海的历史沿革 |publisher=上海企业信息网 |date=2005-12-27 |url=http://www.inc.sh.cn/touzishangahi/showinfo.asp?id=79 |archiveurl=https://web.archive.org/web/20111109114612/http://www.inc.sh.cn/touzishangahi/showinfo.asp?id=79 |archivedate=2011-11-09 |accessdate=2011-03-09 |deadurl=yes }}}}。

1843年，根据上一年[[清朝|中]][[英国|英]]两国签订的《[[南京条约|南京条约]]》，上海作为[[五口通商|通商五口]]正式开埠{{RefTag |1 = {{cite web |title = 谁的上海？——近代上海社会的主体性问题|publisher = 文汇报 |date = 2010-04-16 |url = http://theory.people.com.cn/GB/11382634.html |accessdate = 2011-03-11 }}}}。1845年，中英签订《[[上海租地章程|上海租地章程]]》，由此开始租界的历史。此后，[[法国|法国]]与[[美国|美国]]相继在上海设立租界。其后，租界逐渐形成不受中国政府管辖，拥有独立[[治外法权|司法]]、行政权力的地区。租界之后历经数次扩张，到1900年，[[上海公共租界|公共租界]]达33,503亩，[[上海法租界|法租界]]面积则为2,135亩，范围基本为今上海的核心区域的大部分地区{{NoteTag|1914年，法租界完成最后一次扩界。公共租界范围大致为今上海市的黄浦、静安和虹口、杨浦两个区的南部沿江地带。中区与西区，东至黄浦江、西抵静安寺、镇宁路，南起延安路，北达苏州河；北区与东区，西至浙江北路、北界至武进路、周家嘴路一线，南抵苏州河，东界至复兴岛。黄浦、徐汇两区的部分区域主要是上海法租界，其南线由徐家汇沿肇嘉浜路、徐家汇路、人民路一线，东抵黄浦江，北到延安路与公共租界相邻，西由静安寺沿华山路到徐家汇。长宁则是上海公共租界越界筑路区。而闸北和南市两片中国管理的华界则被租界分割为互不相连的两块。}}{{RefTag |1 = {{cite web |title = 上海租界志>>第一篇区域人口>>第一章区域 第三节租界扩张 |publisher = 上海地方志办公室|date = 2003-08-27 |url = http://www.shtong.gov.cn/dfz_web/DFZ/Info?idnode=64463&tableName=userobject1a&id=57956|accessdate = 2011-03-11}}}}。凭借这独特的政治制度和地理位置，[[上海开埠|上海开埠]]后逐渐发展为远东最繁荣的经济和商贸中心，被誉为“十里洋场”和“冒险家的乐园”{{RefTag |1 = {{cite web |title = 说说“十里洋场”的由来 |publisher = 中国新闻网|date = 2001-12-01 |url = http://www.chinanews.com/hb/2012/11-21/4347060.shtml|accessdate = 2011-03-11}}}}。由于租界的存在使得当时上海繁华地区未遭受[[太平天国|太平天国运动]]与[[义和团|义和团运动]]的影响，享有实际独立的地位和充分的国际联系，为上海之后的持续繁荣创造了条件{{RefTag |1 = {{cite book |title =《史林》1987年03期 论上海租界的双重影响 |publisher = 上海社会科学院历史研究所|date = 1987年5月|author= 熊月之 |accessdate =2016年12月23日}}}}。

1911年11月3日，[[上海光复|上海光复]]{{RefTag |1 = {{cite web |title = 1911年11月1日-上海光复陈其美任沪军都督 |publisher = 网易历史 |date = 2006-11-01 |url = http://news.163.com/06/1030/10/2UM3V51200011EBF.html|accessdate = 2011-03-11 }}}}。自1912年[[中华民国|中华民国]]成立后，由于保守势力与[[中国同盟会|革命党]]互相争夺中央及地方权力，全国局势较为混乱，[[上海县|上海华界]]也时常处于无序状态。与此同时，租界保持了相对的稳定与和平，在经济、人口等方面进一步加快了发展，由1914年开始至1936年成为上海租界高度繁荣的一个阶段{{RefTag |1 = {{cite book |title =《史林》1987年03期 论上海租界的双重影响 |publisher = 上海社会科学院历史研究所|date = 1987年5月|author= 熊月之 |accessdate =2016年12月23日}}}}。

1927年以前，[[租界|租界]]以外的[[闸北|闸北]]和[[南市_(上海)|南市]]属[[江苏省|江苏省]][[上海县|上海县]]管辖。由于租界的政治特殊性，20世纪初的一些影响中国的重大政治事件亦在上海发生。1919年因[[巴黎和会_(1919年)|巴黎和会]]，上海工人、学生及各界人士举行全市总[[罢工|罢工]]、罢课、罢市，该年6月5日起，[[五四运动|五四运动]]的中心从北京移向上海。1921年7月，[[中國共產黨第一次全國代表大會|中國共產黨第一次全國代表大會]]在上海的召開{{RefTag |1 = {{cite web |title = 中国共产党第一次全国代表大会(附图) |publisher = 新浪新闻 |date = 2002-01-27 |url =http://news.sina.com.cn/c/2002-01-27/457477.html |accessdate = 2011-03-11}}}}标志着[[中国共产党|中国共产党]]的诞生，上海也成为[[中共中央|中共中央]]的所在地直至1933年{{RefTag |1 = {{cite web |title = 1933年中共临时中央迁入苏区后中央与中央局组织若干问题考析 |publisher = 中国知网 |date = 2003-01 |url = http://www.cnki.com.cn/Article/CJFDTotal-DANG200301009.htm|accessdate = 2011-03-11}}}}。1925年5月30日，上海学生与市民抗议日资棉紗厂厂主枪杀罢工工人，公共租界巡捕向抗议人群开枪，造成[[五卅慘案|五卅慘案]]，引发了全国性的反英运动{{RefTag |1 = {{cite web|title =血染浦江的五卅惨案|publisher =新浪读书|date =2005-04-04|url =http://book.sina.com.cn/longbook/cha/lilisan/46.shtml|accessdate =2011-03-11|deadurl =yes|archiveurl =https://web.archive.org/web/20111109104707/http://book.sina.com.cn/longbook/cha/lilisan/46.shtml|archivedate =2011-11-09}}}}。1927年4月12日，[[四一二事件|四一二事件]]爆发，[[中国国民党|中国国民党]]内部进行清党，[[第一次国共合作|第一次国共合作]]濒临结束{{Reftag| 1={{cite book|author1=胡适|author2=曹伯言|title=胡适日记全集|url=http://books.google.com/books?id=AAMzSpm8rzwC&pg=PA747|accessdate=17 January 2013|year=2004|publisher=聯經出版|isbn=978-957-08-2710-1|pages=747–}}}}。

1927年3月29日，[[国民政府|国民政府]]在[[租界|租界]]以外地区设[[上海特别市|上海特别市]]，同时将原属[[江苏省|江苏省]]的上海县、宝山县17市乡并入，總面積494平方公里，上海正式脱离[[江苏|江苏]]管辖，直属[[中华民国行政院|行政院]]{{RefTag |1 = [http://www.todayonhistory.com/7/7/ShangHaiSheLi-TeBieShi.html 上海设立“特别市”--历史上的今天网]}}。因为紧邻[[首都|首都]][[南京|南京]]，加之为对外贸易中心，故而大批金融机构将总部设立于上海，其全国经济与金融中心地位更加巩固。上海无论租界还是华界在此后[[国民政府黄金十年|黄金十年]]内均得到飞速发展，被誉为远东金融中心{{RefTag |1 = [http://finance.sina.com.cn/roll/20120313/073211573705.shtml 清算中心:"防火墙"与"融通者"一肩挑--新浪财经]}}。1930年7月，改称上海市。1930年代，上海是[[远东|远东]]最大的都会以及商业和金融中心{{RefTag |name=金融中心排名|1 = {{cite web |url = http://www.longfinance.net/Publications/GFCI%2011.pdf |title = The Global Financial Centres Index 11 |accessdate = 2012-03-19 |publisher = Z/Yen |language = en |archive-url = https://web.archive.org/web/20120319194027/http://www.longfinance.net/Publications/GFCI%2011.pdf |archive-date = 2012-03-19 |dead-url = yes }}}}。

1937年8月13日，[[淞沪会战|淞沪会战]]爆发，[[中国抗日战争|全面抗战]]开始，中日于上海激战三个月，上海沦陷{{RefTag |1 = {{cite web |title = “八一三”淞沪会战 我运炮弹炸日军司令部 |publisher = 中国黄埔军校网 |url = http://www.hoplite.cn/templates/kscfls0047.html|accessdate = 2011-03-11}}}}。战后，原先繁华的闸北、南市市面受到严重破坏，华界基本沦为[[贫民区|贫民区]]，这对上海今后几十年城市格局产生重大影响。上海公共租界的北半部沦为日军防区，而[[上海法租界|法租界]]和[[苏州河|苏州河]]以南的[[上海公共租界|公共租界]]因为独特的情况，则产生了长达四年多、畸形繁荣的[[孤岛时期|孤岛时期]]。至1941年[[太平洋战争|太平洋战争]]爆发后，日军开始占领租界，自1943年起，租界不复存在{{RefTag |1 = {{cite web |title = 上海租界历经百年，不是殖民地却好似殖民地 |publisher = 界面新闻|date = 2016-08-26 |url = http://www.jiemian.com/article/819846.html|accessdate = }}}}。1945年抗战胜利，上海由[[国民政府|国民政府]]接管{{RefTag |1 = {{cite web |title = 日本无条件投降后 国民党“大接收”乱象丛生  |publisher = 扬子晚报(南京) |url = http://news.163.com/15/1121/12/B8UNTVR200014AED.html|accessdate = 2016-12-23}}}}。

1946年，[[第二次国共内战|第二次国共内战]]爆发，上海经济陷入崩溃，物价飞涨。1948年政府颁行《[[金圆券|经济紧急处分令]]》试图扭转局面，但以失败告终{{RefTag |1 = {{cite web |title = 国民党时期金圆券疯狂贬值 物价上涨千万倍  |publisher = 《 同舟共进》 |url = http://news.163.com/08/0513/08/4BQG1N6600011247.html |accessdate = 2016-12-23}}}}。1949年5月[[上海战役|上海战役]]后，[[中国人民解放军|解放军]]在5月27日控制上海全境，成立[[中国人民解放军上海市军事管制委员会|中国人民解放军上海市军事管制委员会]]和[[上海市人民政府|上海市人民政府]]。此间，许多与民国政府及海外机构有关的人员、财产和机构随同中华民国政府撤离上海{{RefTag |1 = {{cite web |title = 1949国民党溃败大上海  |publisher = 《档案：上海纪实》 |url = https://v.qq.com/x/cover/53rl4easz3zvgtd/o0020uhdvw9.html |accessdate = 2016-12-23}}}}。

=== 中华人民共和国时期 ===

1949年，[[中华人民共和国|中华人民共和国]]设上海为[[中央直辖市|中央直辖市]]，总面积618.02平方公里。其中市区面积86.66平方公里，郊区面积531.36平方公里。1954年，全市面积为630.11平方公里。1958年，[[江苏省|江苏省]][[上海县|上海]]、[[嘉定区|嘉定]]、[[宝山区_(上海市)|宝山]]、[[松江区|松江]]、[[金山区_(上海市)|金山]]、[[川沙县|川沙]]、[[南汇|南汇]]、[[奉贤|奉贤]]、[[青浦|青浦]]、[[崇明县|崇明]]共10个县划归上海市，使上海市的面积增大到5862.88平方公里{{RefTag |1 = {{cite web|title = 1958年上海市行政区划 |publisher = 行政区划网 |date= 2014-04-16 |url= http://www.xzqh.org/html/show/sh/5169.html |accessdate= 2011-03-11}}}}。1984年，上海市测绘处和上海市土壤普查办公室对全市的土地面积重新进行测量，至1986年底，全市土地面积为6340.50平方公里，其中市区为375.44平方公里，郊县为5965.06平方公里；全市水面面积为121.85平方公里。从1987年1月起全市正式使用以上的数据{{RefTag |1 = {{cite web |title = 上海民政志——第二章 行政区划管理——第二节 解放后区划管理  |publisher = 上海地方志办公室

|time = August 2013}}}}，空气污染与噪音污染时常被市民投诉{{RefTag |1 = {{cite web |title = 去年上海市噪声污染投诉量仍占环境污染投诉总量高位 |publisher = 新民晚报 |date = 2012-04-13 |url = http://shzw.eastday.com/shzw/G/20120413/userobject1ai77298.html|accessdate = 2013-08-20}}}}，也出现了环境污染事故损害市民健康的事件{{RefTag |1 = {{cite web |title = 上海公布儿童血铅超标调查结果 承认环境污染致血铅超标 |publisher = 搜狐科学 |date = 2012-02-26 |url = http://news.familydoctor.com.cn/highlights/201202/1654262111746.html |accessdate = 2012-04-17 }}}}。上海也是一个典型的水质型缺水城市，由于水质的逐步恶化，自来水厂的水源地屡次进行搬迁{{RefTag |1 = {{cite web |title =水质型缺水城市上海的20年找水路径 |publisher = 第一财经日报 |date = 2010-12-06 |url = http://city.sina.com.cn/focus/t/2010-12-03/111511251.html |accessdate = 2012-04-17}}}}。闵行[[吴泾|吴泾]]滨江分布着很多[[化工|化工]]、[[能源|能源]]、[[碳素|碳素]]、[[建材|建材]]等厂家，可能会进行搬迁改造{{RefTag | 1 = {{cite web | author = 覃丛丛 | title = 吴泾工业区综合整治 老工业基地计划搬迁 三大公园有序推进 滨江两岸将现闵行版“外滩” | publisher = 闵行报 | date = 2016-03-18 | language = zh | accessdate = 2018-06-21 | url = http://mhb.shmh.gov.cn/content/2016-03/18/content_6081.htm }}{{Dead link|date=2019年3月 |bot=InternetArchiveBot |fix-attempted=yes }}}}。另外，上海位于[[长江|长江]]入海口冲积平原，属于[[软土|软土]]地基，过量开采[[地下水|地下水]]、修建高层建筑和地下设施等原因导致了地面沉降{{RefTag |name = "中国工业化的环境账" |1 = {{cite web |author = 黄安伟 |title = 中国工业化的环境账 |publisher = 纽约时报 |date = 2013年4月1日 |accessdate = 2013年4月6日 |language = zh-cn |url = http://cn.nytimes.com/article/china/2013/04/01/c01environment/ |archive-url = https://web.archive.org/web/20130407040255/http://cn.nytimes.com/article/china/2013/04/01/c01environment/ |archive-date = 2013年4月7日 |dead-url = yes }}}}。
 
上海市政府对于所出现环境问题采取了许多相应的改善措施。市区汽车禁鸣喇叭已用法律进行规范，高架旁则竖立了隔音屏以降低噪声对沿线居民住宅的影响。曾经污染严重的苏州河，自1990年代以后进行了三个阶段的大规模治污工程，污染现已基本消除，河流生态亦得到恢复{{RefTag |1 = {{cite web |title =苏州河生态迅速恢复 松江鲈鱼有望提早游回家 |publisher =人民网|date = 2004-11-07 |url = http://www.people.com.cn/GB/huanbao/1072/2970783.html |accessdate = 2013-04-05}}}}。对于地下水的过度开采，则采用自来水回灌的手段达到基本平衡，但是地面依旧以每年6毫米的速度沉降{{RefTag |1 = {{cite web |title = 上海地下水抽取回灌达基本平衡 地面沉降量每年6毫米 |publisher = 东方早报 |date = 2012-04-09 |url = http://sh.sina.com.cn/news/s/2012-04-09/0818213588.html|accessdate = 2012-04-19}}}}。

[[淀山湖|淀山湖]][[水質|水質]]為[[Ⅴ類|Ⅴ類]]，黃浦江[[水質|水質]]為[[Ⅳ類|Ⅳ類]]，[[近岸海域|近岸海域]]水質極差属[[劣四類|劣四類]]；上海市空氣達標天數比例為67.4%，重度及以上污染天數比例為6.3%{{RefTag | 1 = {{cite web | author = | title = 2017中國生態環境狀況公報 | publisher = 中華人民共和國生態環境部 | date = 2018-05-31 | language = zh | accessdate = 2018-06-20 | url = http://www.zhb.gov.cn/hjzl/zghjzkgb/lnzghjzkgb/201805/P020180531534645032372.pdf}}}}{{RefTag | 1 = {{cite web | author = | title = 2016中國環境狀況公報 | publisher = 中華人民共和國環境保護部 | date = 2017-06-05 | language = zh | accessdate = 2017-07-01 | url = http://www.mee.gov.cn/hjzl/zghjzkgb/lnzghjzkgb/201706/P020170605833655914077.pdf}}}}{{RefTag | 1 = {{cite web | author = | title = 2013中國環境狀況公報 | publisher = 中華人民共和國環境保護部 | date = 2014-05-27 | language = zh | accessdate = 2017-07-01 | url = http://www.mep.gov.cn/gkml/hbb/qt/201407/W020140707500480541425.pdf}}}}。

=== 气候 ===

上海的气候类型属于[[亚热带季风气候|亚热带季风气候]]，四季分明，日照充分，雨量充沛。气候温和湿润，年平均气温17.0 °C。春（4月-5月）、秋（10月-11月）较短，冬（12月-次年3月）、夏（6月-9月）较长。有春雨、[[梅雨|梅雨]]、秋雨三个雨期，因而5月至9月间为上海的汛期，降水量达全年的60%左右{{RefTag |1 = {{cite web |title = 上海概览-上海气候概况 |publisher = 中国上海|url = http://www.shanghai.gov.cn/nw2/nw2314/nw3766/nw3773/nw3777/u1aw6.html |accessdate = 2011-03-11}}}}。每年的7月进入[[伏旱|伏旱]]天气，较之日常月份显得潮湿酷热，平均气温为28.7°C，每年平均有8.7天最高氣溫超過攝氏35度。近年来受环境影响，气温有上升趋势。每年1月为全年最寒冷的季节，平均气温4.8°C。市区徐家汇（1872年开始统计）极端最高气温为40.9°C（2017年7月21日）{{RefTag |1 = {{cite web |title = 上海气温达40.9℃ 创145年最高纪录 |publisher = 中国天气网 |url = http://news.weather.com.cn/2017/07/2743916.shtml |accessdate = 2017-07-21}}}}，极端最低气温为-12.1°C（1893年1月19日）。国际交换站（1951年开始统计）极端最高气温39.9℃（2013年8月6日、8日），极端最低气温-10.1℃（1977年1月31日）。受北方[[蒙古高原|蒙古高原]]和[[西伯利亚|西伯利亚]]的冷空气影响，会出现霜冻现象。如遇强冷空气来袭并恰逢暖湿气流，上海也会出现雨夹雪或降雪天气{{RefTag |1 = {{cite web |title = 上海气候概况 |publisher = 中国网 |date = 2009-09-08 |url = http://www.china.com.cn/aboutchina/zhuanti/09dfgl/2009-09/08/content_18488374.htm |accessdate = 2011-03-11}}}}。

=== 外交 ===
{{main|上海对外事务}}

由于[[租界|租界]]的开辟，上海华界当局的外事活动主要以划定租界土地和处理租界内中国籍公民的相关事宜，直至1949年[[治外法权|治外法权]]消失{{RefTag |1 = {{cite web |title = 第七节美国重获治外法权  |publisher = 《上海外事志》编辑室 |date = 1999-09 |url = http://d.wanfangdata.com.cn/LocalChronicleItem_1771151.aspx |accessdate = 2011-03-12}}}}。1949年后很长一段时间，上海对外交流几乎呈现中断状态。自1990年代以后，上海开始逐步承办众多重要国际会议和大型赛事，逐渐加强了与世界的交流。

另外作为曾经租界的存在，很多国家在沪开设领事机构，大多为[[总领事馆|总领事馆]]。上海与国外城市缔结友好关系的历史，始于1970年代。至2017年底，与上海建立友好关系的地区已有85个{{RefTag |1 = {{cite web |title =上海市国际友好城市及友好交流关系城市总体情况 |publisher =上海市人民政府外事办公室 |date = 2017年10月 |url =http://www.shfao.gov.cn/wsb/node466/node548/node549/index.html  |accessdate = 2017-11-22}}}}{{RefTag |1 = {{cite web |title =64个市级友好城市 |publisher =上海市人民政府外事办公室|date = 2017年10月 |url = http://www.shfao.gov.cn/wsb/node466/node548/node550/index.html |accessdate = 2017-11-22}}}}。

在1980年代，上海在中国的经济地位迅速下降，经济发展缓慢。1990年4月，[[中共中央|中共中央]]和[[中华人民共和国国务院|国务院]]作出开发开放[[浦东新区|浦东]]的决定{{RefTag |name=近十年北京與上海空間發展管理體制的比較分析}}，宣布加快浦东地区的开发，实行经济技术开发区和某些[[经济特区|经济特区]]的政策{{RefTag |1 = {{cite web|title = 1990年4月，中共中央、国务院作出开发开放上海浦东的决定|publisher = 中国中央电视台|date = 2008|url = http://space.tv.cctv.com/act/community/NationalDay/NationalDayVideoCarousel.jsp?scheduleId=SCHE1246435013686466&videoId=VIDE1236568121580496|accessdate = 2012-04-05|deadurl = yes|archiveurl = https://web.archive.org/web/20150603060435/http://space.tv.cctv.com/act/community/NationalDay/NationalDayVideoCarousel.jsp?scheduleId=SCHE1246435013686466&videoId=VIDE1236568121580496|archivedate = 2015-06-03}}}}。浦东的经济得以在20年间剧增60余倍{{RefTag |1 = {{cite web |title =浦东:数字20年---对比1990和2009 |publisher = 东方网 |date = 2010-04-13 |url = http://district.ce.cn/zt/2010/pdkfkf/bjzl/201004/13/t20100413_21268953.shtml|accessdate = 2012-04-05}}}}，上海经济也在之后连续十余年保持10%以上年增长，再度巩固了作为中国经济、金融及贸易中心的地位{{RefTag |name=近十年北京與上海空間發展管理體制的比較分析}}，现浦东新区占上海市的经济比重近30%。2013年7月[[上海自由贸易区|上海自由贸易区]]获得国家立项，上海拥有了中国大陆首个自由贸易区，经济发展将再次获得良好机遇{{RefTag |name ="zimaoqu"|1 = {{cite news|url=http://news.china.com/domestic/945/20130703/17926580.html|title=国务院通过上海自由贸易试验区总体方案|publisher=中国政府网|date=2013-07-03|accessdate=2013-08-07|deadurl=yes|archiveurl=https://web.archive.org/web/20130709095257/http://news.china.com/domestic/945/20130703/17926580.html|archivedate=2013-07-09}}}}。

从[[中华人民共和国|中华人民共和国]]成立至[[上海1990年代|1990年代]]初，由上海市上缴的[[财政收入|财政收入]]占全部中央财政的比重保持在16%以上，占上海市本地财政收入的80%以上。1990年代中央启动[[浦东开发开放|浦东开发开放]]政策，上海财政上缴的比重有所减少，目前仍达12~15%比例，在全国各地“净上缴财政”排名中，上海位居第一，大规模的财政上缴有力地支援了全国各地的经济建设，但也使上海本地投资资金紧张，造成土地财政从而引发高房价等问题。近年来由于上海将大量地方财政收入亦上缴中央，使上海本地投资减弱，也一定程度上拖累了上海经济的发展{{RefTag |name ="上海概况"|1 = {{cite news |url = http://finance.sina.com.cn/132/2004-07-14/2.html |title=上海概况 |publisher=新浪财经-地方经济|accessdate=2013-04-05}}}}{{RefTag |name ="上海12年来首现固定资产投资负增长"|1 = {{cite news |url = http://sh.sina.com.cn/news/f/2011-07-23/0943190167.html |title=上海12年来首现固定资产投资负增长 |publisher=新浪上海 |date=2011年7月23日|accessdate=2013-04-05}}}}{{RefTag |1 = {{cite news |url = http://nccuir.lib.nccu.edu.tw/bitstream/140.119/37386/7/202307.pdf|title=上海與中央政府互動之分析 |publisher=[[國立政治大學|台湾國立政治大學]]|accessdate=2013-08-15}}}}。

=== 第三產業 ===

1949年以前，上海是中国的商业金融中心、[[亚洲|亚洲]]和[[远东|远东]]的[[国际金融中心|国际金融中心]]。当时上海汇集着号称“四行两局一库”的[[中央银行_(中华民国)|中央银行]]、[[中国银行|中国银行]]、[[交通银行|交通银行]]、[[中国农民银行|中国农民银行]]、[[中央信托局|中央信托局]]、[[中华邮政|邮政储金汇业局]]和中央合作金库的国家资本金融机构{{RefTag |1 = {{cite web |title = 黄浦区区志-金融 |publisher = 上海市地方志办公室 |url = http://www.shtong.gov.cn/newsite/node2/node4/node2249/huangpu/node34608/index.html|accessdate = 2011-03-12 |language = zh }}}}，以及数量众多的外资、私有银行、钱庄和信托公司。当时，总部设在上海的国内银行占银行同业公会注册会员的81%。经过[[国民政府黄金十年|国民政府黄金十年]]的发展，至[[抗日战争|抗日战争]]爆发前，上海的各类私有银行、钱庄与信托已经达到了105家，在华外资银行共32家。其中落户上海的有27家，而同年香港地区只有17家。与外滩平行的江西路，则因坐落大批金融机构，如金城、盐业、浙江兴业等银行，所收存款额占全国存款总额的三分之一以上，而被誉为“东方华尔街”{{RefTag |1 = {{cite web|title=老上海滩的东方华尔街----江西路 |publisher=上海档案信息网 |date=2008-06-10 |url=http://www.archives.sh.cn/docs/200806/d_201536.html |accessdate=2011-03-12 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20111109120420/http://www.archives.sh.cn/docs/200806/d_201536.html |archivedate=2011-11-09 }}}}。

当年，上海金融市场非常发达、金融工具品类繁多。除了证券市场以外，还拥有[[同业拆借|同业拆借]]、贴现等货币市场，内汇、外汇市场和标金（黄金）市场与大条银（白银）市场。在资本市场方面，上海当时是仅次于[[纽约|纽约]]和[[伦敦|伦敦]]的全球第三大股票市场和第三大债券市场{{RefTag|name=金融中心定位 |1 = {{cite web |title = 解读上海的国际金融中心定位 |publisher = 《当代金融家》|date =2006年1月11日 |url =http://business.sohu.com/20060111/n241382293.shtml |accessdate = 2013-07-06 |language = zh |author = 干杏娣}}}}。上海当时既是全国资金调剂中心，也是远东国际交易活跃的金融中枢：1936年，上海的黄金市场全年成交数实际达到3116万条，折合[[美元|美元]]1100余万元。这一数字超过[[日本|日本]]的[[东京|东京]]和[[大阪|大阪]]之和，也超过了[[法国|法国]]的[[巴黎|巴黎]]{{RefTag |1 = {{cite web |title = 创新凝聚力决定国际金融中心形成（文/杨威月，原载《国际金融报》） |publisher = people.com.cn |date = 2002-10-18 |url = http://www.people.com.cn/GB/paper66/7494/718625.html |accessdate = 2009-07-21 |language = zh }}}}。“每日交易额之巨，虽不足与伦敦、纽约媲美，然凌驾法、[[印度|印]]、日之上”{{RefTag|name=金融中心定位}}。

1949年以后，一些上海的[[资本家|资本家]]已经逃离到[[香港|香港]]、[[台湾|台湾]]或者海外其他地方。也有一些留在上海，继续经营。最后在1956年初，都被[[公私合营|公私合营]]取代{{RefTag |1 = [http://business.sohu.com/20060316/n242322193.shtml 生存与消亡：上海私营金融业的公私合营]}}。1950年，[[上海证券交易所|上海证券交易所]]因参与投机和外汇交易被[[中国人民解放军上海市军事管制委员会|上海市军管会]]关闭{{RefTag |1 = {{cite web |title = 上海最后“攻破”的一座堡垒：取缔证券大楼[图] |publisher = 中华网军事 |date = 2009-12-11 |url = http://military.china.com/zh_cn/history4/news2/11078475/20091211/15733075.html |accessdate = 2011-03-12 |language = zh |deadurl = yes |archiveurl = https://web.archive.org/web/20120112171848/http://military.china.com/zh_cn/history4/news2/11078475/20091211/15733075.html |archivedate = 2012年1月12日 |df =  }}}}。[[上海1952年|1952年]]底，上海私营金融机构的数量减至60余家。1956年初，官方实行公私合营政策，最后全部合并为统一的公私合营银行联合总管理处。所有资本市场也被关闭，上海的亚洲金融中心地位后被东京、[[新加坡|新加坡]]和[[香港|香港]]取代。

1987年4月1日，[[交通银行|交通银行]]在上海重新恢复对外营业{{RefTag |1 = {{cite web |title = 交通银行详细资料 |publisher = 凤凰财经 |url =http://finance.ifeng.com/company/data/detail/1693.shtml  |accessdate = 2011-03-12 |language = zh }}}}。1990年11月26日，[[上海证券交易所|上海证券交易所]]再度成立，上海金融市场开始恢复。到2009年底，上证总市值为18.3万亿元人民币，位居全球第五、成交金额居全球第三、新股发行募集资金总额居全球第六，并在近几年吸引了众多优秀的大型企业来沪上市融资。2002年，[[中國銀聯|中國銀聯]]在上海創立。2005年8月，[[中国人民银行|中国人民银行上海总部]]在沪揭牌。2016年，[[新开发银行|新开发银行]]全面投入使用。目前总行设在上海的国有银行有[[交通银行|交通银行]]、[[上海浦东发展银行|上海浦东发展银行]]、[[上海银行|上海银行]]和[[上海农村商业银行|上海农村商业银行]]等。2010年2月21日，由全球著名的[[毕马威会计师事务所|毕马威咨询公司]]参与制定，法国[[巴黎大区|巴黎大区]]投资促进协会发表公布的“世界最具吸引力”城市排行榜中，[[英国|英国]][[伦敦|伦敦]]位居榜首，排名第二的是中国上海{{RefTag|name=吸引力|1 = {{cite web |title = 上海超纽约成仅次于伦敦的世界最具吸引力城市 |publisher = 环球网 |date = 2010-02-22 |url = http://world.huanqiu.com/roll/2010-02/723359.html |accessdate = 2011-03-12 }}}}。上海拥有的现代化研究中心数量和写字楼数量已经超过了[[英国|英国]][[伦敦|伦敦]]{{RefTag|name=吸引力}}。陆家嘴成为新的金融区，并且与浦江对岸的外滩金融街遥相辉映。

=== 第二產業 ===

轻工业是上海发展最早也是最为成熟的工业部门。丝绸与纺织行业是自[[明代|明代]]就已经颇具规模。[[上海开埠|上海开埠]]后，由于倚靠的江南地区的富庶，加上轻工业投入相对较少，资金回笼快，中外资本纷纷在沪开设轻工业的工厂，并形成激烈竞争。[[计划经济|计划经济]]时期，上海生产的轻工业品行销全国，上海基本成为质量和时尚产品的代名词。上海的一系列食品、日化、[[钟表|钟表]]、[[自行车|自行车]]、[[缝纫机|缝纫机]]、摄影器材、仪器、办公耗材等等轻工业品共有90多大类，两万多种花色品种，成为中国轻工业生产最重要的基地{{RefTag |1 = {{cite web |title = 上海轻工业志-总述 |publisher = 上海地方志办公室 |url=http://www.shtong.gov.cn/dfz_web/DFZ/Info?idnode=68933&tableName=userobject1a&id=66569 |accessdate = 2011-03-12 |language = zh }}}}。永久牌自行车、[[大白兔奶糖|大白兔奶糖]]、上海牌手表、蝴蝶牌缝纫机、六神花露水等等经典产品因品质过硬、价格公道，长期作为中国大陆市场的畅销品牌受到大量消费者的肯定。目前，上海在自行车制造、食品生产、日化用品中仍旧保持较大规模{{RefTag|name=上海改革开放二十年工业卷|1 = {{cite book |title = 上海改革开放二十年-工业卷|publisher =中共上海市委党史研究室, 中共上海市工业工作委员会 |ISBN = 7-208-02996-2 |accessdate = 2013-06-05|language = zh }}}}。

上海，自洋务运动伊始，便成为中国最重要的工业基地，而[[江南制造总局|江南制造总局]]则是上海重工业的起点。[[国民政府|国民政府]]时期和[[计划经济|计划经济]]时期，上海的工业发展水平就已在全国遥遥领先。由于上海交通便利、科学文化发达，因此众多重工业产品的科研基地和生产基地都位于上海。[[改革开放|改革开放]]初期，由于中央政策原因，中国东南地区快速发展，使上海的工业面临边缘化的危机，但自1990年代中期以来，随着浦东新区的开发，以及财政转移支付比重减少等多方面的原因，上海的工业又重新焕发了新的生机。上海工业主要以[[冶金|冶金]]、[[石油化工|石油化工]]、[[机械|机械]]、[[电子工业|电子工业]]、[[汽车|汽车]]、[[航空|航空]]、[[航天|航天]]、[[超级计算机|超级计算机]]、船舶制造等行业为主。[[张江高科技园区|张江高科]]汇集了大量的高端制造业{{RefTag|name=上海改革开放二十年工业卷}}。

目前，在交通工业方面，[[上海大众|上海大众]]{{RefTag |1 = {{cite web |title = 1984—2009 上海大众汽车发展历史回顾 |publisher = 新华网 |date =2009-10-14 |url =http://news.xinhuanet.com/auto/2009-10/14/content_12228009.htm |accessdate = 2011-03-12 |language = zh }}}}、[[上海通用汽车|上海通用]]成为中小型乘用车生产的重要基地，申沃客车则是重要的大型车生产基地，而[[阿尔斯通|上海阿尔斯通交通设备有限公司]]则承担了轨道交通方面的生产。而在航空工业领域，上海是中国航空工业的研发基地，是大型国产客机的生产商[[中国商用飞机有限责任公司|中国商用飞机有限责任公司]]的总部基地{{RefTag |1 = {{cite web |title = 中航工业与上海市政府签约共建民机航电产业基地 |publisher = 中国证券网 |date = 2010-07-12 |url = http://www.cnstock.com/index/gdbb/201007/667967.htm |accessdate = 2011-03-12 |language = zh |deadurl = yes |archiveurl = https://web.archive.org/web/20121019000635/http://www.cnstock.com/index/gdbb/201007/667967.htm |archivedate = 2012-10-19 }}}}。此外，[[江南造船|江南造船厂]]和[[沪东中华造船|沪东中华造船厂]]领衔的造船工业也是上海的一大支柱产业，年造船能力超过千万吨{{RefTag |1 = {{cite web |title = 长兴岛年造船能力欲超千万吨 |publisher = 上海国际海事信息与文献网 |date = 2010-11-03 |url = http://www.simic.net.cn/news/detail.jsp?id=64558 |accessdate = 2011-03-12 |language = zh }}{{Dead link|date=2019年3月 |bot=InternetArchiveBot |fix-attempted=yes }}{{Dead link|time=June 2013}}}}。而在冶金和石油化工方面，[[宝钢|上海宝钢]]和[[上海石化|上海石化]]也居于国内行业的领先地位，宝钢集团更是在全球钢铁企业中位列第三{{RefTag |1 = {{cite web |title = 宝钢500强排名上升39位 升至全球钢企第三 |publisher = 腾讯财经 |date = 2009-07-09 |url = http://finance.qq.com/a/20090709/006813.htm |accessdate = 2011-03-12 |language = zh }}}}。

上海是中国近现代以来的商业中心。上海自建城以来，由于处于富庶的江南平原地区，加之自身传统自然经济手工业的发达。因而早在宋明时期，便已经是有名的商埠。元朝政府在上海设[[江海关|市舶司]]，使得上海的商业和海外贸易得到了极大发展。明代，松江府成为全国纺织手工业和棉布贸易中心。清[[康熙|康熙]]年间，由于[[迁界令|海禁]]解除，加之[[江海关|江海关]]的设立，上海棉布贸易迅速扩大。至[[嘉庆|嘉庆]]、[[道光|道光]]年间，每年销往英国的上海棉布就多达80万匹{{RefTag |1 = {{cite web |title = 第01节五口开放后外国资本主义 |publisher = 劝学网 |url = http://www.quanxue.cn/LS_Qing2/QingHua/QingHua08.html |accessdate = 2011-03-12 |language = zh }}}}。

上海开埠后，海外商品源源不断的由上海进入[[中国内地|内地]]。上海的进口贸易得到了飞速发展。到1850年代中期起，上海已经超越[[广州|广州]]，成为全国的外贸中心，上海的市面也变得异常繁华。[[庚申之劫|庚申之劫]]之中，上海由于租界的保护未受到严重破坏，而周边的苏州、杭州等江南城市则经历了一场浩劫，因此吸引了苏州等地大批商户到此躲避战乱，上海的商业得到了迅速的发展，从此成为中国的商业中心。到1908年，据《上海华商行名簿册》记载，租界内共有包括银行、[[钱庄|钱庄]]、商店、工厂、交通等各类企业、店铺在内，总数已达10534户{{RefTag |1 = {{cite web |title = 就辛亥革命性质问题答台北学者 |publisher = 中华文史网 |date = 2008-05-08 |url = http://jds.cass.cn/ztyj/wqzzs/201605/t20160506_3323831.shtml |accessdate = 2011-03-12 |language = zh }}}}，如今世界知名的[[南京路_(上海)|南京路]]在此时已经初具规模。

一战爆发后，民族商业也得到飞速发展，一些日后知名的环球百货公司在此时入驻南京路和上海其他地区。[[永安百貨|永安]]、[[先施|先施]]、[[大新公司|大新]]、[[新新公司|新新公司]]被合称为[[南京路_(上海)|南京路]]的四大百货公司，销售许多全球各地最新颖的百货。[[孤岛时期|孤岛时期]]，上海商业呈现出了畸形的繁荣。

1956年以后，由于[[计划经济|计划经济]]和[[公私合营|公私合营]]的实行。上海商业进入一个新的时期。这一时间内，主要都是国营商店维系着上海的商业运行。居民日常购物，从粮食、副食品到电视机、自行车都依赖[[中国居民供应票证|票证]]进行购买。1983年起到1993年，票证逐步退出上海的商业，市场开始实行敞开供应。从1990年代中叶起，超市、卖场、便利店等现代化商业模式开始进入上海。传统的[[杂货店|杂货店]]和百货商店开始逐步消失或转变功能{{RefTag |1 = {{cite book |title = 上海改革开放二十年-商业卷 |publisher = 上海人民出版社 1998年 |author = 中共上海市商业工作委员会, 中共上海市委党史研究室 |accessdate = 2013-06-05 |language = zh }}}}。

到1990年代，上海拥有三条主要商业街区和几大商业中心，分别为[[南京东路_(上海)|南京东路]]、[[淮海中路|淮海中路]]、[[四川北路|四川北路]]、[[豫园|豫园]]、[[徐家汇|徐家汇]]等。后经过20年的再度高速发展，至2010年末，上海的商业中心已经扩展到十个{{RefTag |1 = {{cite web|title=上海将形成十大都市商业中心 |publisher=徐汇区统计局 |url=http://sjj.xh.sh.cn/two_web/e_redianguanzhu/detail/y06/m05/t20060523.htm |accessdate=2013-07-26 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20151222133859/http://sjj.xh.sh.cn/two_web/e_redianguanzhu/detail/y06/m05/t20060523.htm |archivedate=2015-12-22 }}}}，分别为：南京东路商业中心、南京西路商业中心、淮海中路商业中心、四川北路商业中心、徐家汇商业中心、张杨路商业中心、豫园商城商业中心、五角场商业中心、新客站不夜城商业中心、中山公园商业中心。商业中心也从市区的市中心位置逐步扩展至市区的次中心区域，辐射范围不断扩大。根据上海市政府2010年出台的商业报告显示，中山公园与五角场是在原八商业中心的基础上新增的两个商业中心{{RefTag |1 = {{cite web |title = 上海商业新布局:10大都市商业中心呼之欲出|publisher =新闻晚报|date = 2006年5月22日|url =http://www.ce.cn/macro/gnbd/dq/zdb/200605/22/t20060522_7050447.shtml |accessdate =2013-06-05|language = zh }}}}。

=== 第一產業 ===

[[File:上海农业总产值1990-2009.jpg|thumb]]

上海由于属于[[亚热带季风气候|亚热带季风气候]]，一年四季分明。全年平均气温15.5℃，自有气象记录以来，最高气温为40.6℃，最低气温为－12.1℃。常年11月中旬初霜，次年4月上旬终霜，年均无霜期225天。年均日照时达1934.4小时，太阳辐射总量4670兆焦耳/平方米，其中光合有效辐射占一半。年均有效积温2630℃·日。雨量充沛，年雨日130天，平均年雨量1145.4毫米。光、热、水基本同季，农业气候条件比较优越{{RefTag |1 = {{cite web |title = 上海农业志-总述 |publisher = 上海地方志 |url = http://www.shtong.gov.cn/dfz_web/DFZ/Info?idnode=66645&tableName=userobject1a&id=62974 |accessdate = 2011-03-11 |language = zh }}}}。自古代以来，上海一直为鱼米之乡，[[水稻|水稻]]、渔业、蔬菜的产量都相对较高。早在上海开埠后，随着城市化进程，农用耕地和产鱼河流减少和被湮埋，依靠传统耕种手法的农业一度受到很大影响。自1990年代起，上海农业开始向高科技和机械化发展。

上海的传统农业主要是精耕细作，以渔业、盐业和种植业为主。早在[[秦汉|秦汉]]时期上海地区就已经“海滨广斥，盐田相望”。到了元代，[[江浙等处行中书省|江浙地区]]的34个盐场中，松江府就独占五个，成为重要产盐地区。此外，江南地区水网纵横，气候适宜，成为渔业和水稻种植的最适合地区。唐代，上海地区的薄稻米作为贡米送往京师[[长安|长安]]。[[吴越国|吴越国]]时期，华亭县大肆兴建堰闸等水利设施，使得农业种植得到更好的保障。宋代，华亭县亩产稻谷更能达到两至三石，成为当时中国粮食的高产区{{RefTag |1 = {{cite web |title =《傍秋亭杂记》|author = 主纂明[[正德_(明朝)|正德]]年间《松江府志》的顾清|language = zh}}}}。进入明代以后，除粮食生产外，上海地区因为纺织技术的改革，开始大量种植[[棉花|棉花]]等[[经济作物|经济类作物]]。加上[[黄道婆|黄道婆]]的改革，松江棉布闻名国内，产量极高，被誉为“衣被天下”{{RefTag |1 = {{cite book |title = 《上海乡土历史》|publisher =上海教育出版社|date =1993年|ISBN= 9787532033034|accessdate = 2013-08-07}}}}。

1988年，在时任市长的[[朱镕基|朱镕基]]的主持下，为了解决城市粮农副食品的供应，开始建设“[[菜篮子工程|菜篮子工程]]”，由此上海的农业现代化开始高速发展{{RefTag |1 = {{cite web |title = 上海农业志-蔬菜 |publisher = 上海地方志办公室 |url= http://www.shtong.gov.cn/dfz_web/DFZ/PianInfo?idnode=66650&tableName=userobject1a&id=-1|accessdate = 2011-03-12 |language = zh }}}}。1990年代初，现代化农业就取得了明显的效果。目前，上海的农业基本实现现代化。依靠高科技、标准化、机械化生产、兴建水利、农业园区和农业保险补贴等六个方面构成现代化农业网络。

到2012年，上海市实现农业总产值320.76亿元。其中，种植业171.2亿元；[[林业|林业]]8.87亿元；牧业72.63亿元；渔业57.81亿元；农林牧渔服务业10.25亿元。上海域外市属农场实现农业总产值16.08亿元。全年全市粮食播种面积187.61千公顷；粮食产量122.39万吨；水产品产量27.21万吨。全年水稻良种覆盖率达99.8%。至2012年底，上海市有543家企业、3300个产品获得农产品质量认证。其中，绿色食品生产企业116家，绿色食品170个；无公害农产品生产企业420家，无公害农产品3106个；上海累计建成标准化畜禽养殖场270家，标准化水产养殖场293家；累计建成设施粮田面积86.5千公顷，蔬菜标准园60家。至年末，全市有农业产业化龙头企业388家，农民专业合作社3177家{{RefTag|name=2012公报 |1 = {{cite web |title = 2012年上海市国民经济和社会发展统计公报|publisher = 上海市统计局|url =http://www.stats-sh.gov.cn/html/sjfb/201302/253153.html|date=2013-02-26|accessdate = 2013-07-26|language = zh }}}}。

根據2016年的統計，上海市內共有2419.70万名常住人口，其中分別有980.20万名常住外來人口以及1439.50万名常住户籍人口{{RefTag | name = "2016年上海市国民经济和社会发展统计公报"}}。目前，上海市常住人口中[[少数民族|少数民族]]共有11.8万人，其中[[回族|回族]]约有7万余人。除常住人口以外，持居住证的外来人口为103万人。另外有16万外国人常年居住于上海{{RefTag |1 = {{cite web |title =上海老外人数激增钱包缩水 |publisher = 广东省政府 |date = 2012-01-29 |url = http://column.cankaoxiaoxi.com/2012/0129/11608.shtml |accessdate =2012-04-20}}}}。

在人口结构方面，上海正面临严重的[[老龄化|老龄化]]问题。现今戶籍設在上海的人口其平均期望寿命為82.75岁，其中男性和女性分別為80.83岁與女性85.61岁{{RefTag | name = "2016年上海市国民经济和社会发展统计公报"}}，而65岁及以上人口則有232.98万人，占总人口的10.1%。预计到了2030年時60岁以上的老年人口将超过500万人{{RefTag |1 = {{cite web |title = 20年后上海将达老龄化高峰 老年人口预计超过500万|publisher = 新民网|url = http://news.xinmin.cn/rollnews/2009/04/14/1813288.html}}}}。

上海市的生育率处于极低水平。根据人口普查和抽查，2000年以来，上海市[[总和生育率|总和生育率]]处于0.67和0.74之间，仅为世代更替水平的三分之一。

上海自开埠以后，各地人口迅速涌入，其中[[江苏|江苏]]、[[山东|山东]]、[[浙江|浙江]]、[[广东|广东]]、[[安徽|安徽]]是上海居民的主要来源地，并与上海本地居民融合，构成今日上海市民主体的[[上海人|上海人]]{{RefTag |1 = {{cite web |title = 开埠通商与上海人特性的形成|publisher = 凤凰新闻|url = http://news.ifeng.com/history/gundong/detail_2013_11/19/31367222_0.shtml}}}}。

1843年前，上海主要以本地居民为主。随着开埠后贸易的迅速增长以及相对稳定的政治环境，原先在[[粤|粤]]的大批[[买办|买办]]、因避[[太平天国|太平军]]战乱的吴越富商、受水灾逃难的[[苏北|苏北]]民众纷纷离开原籍并定居上海{{RefTag |1 = {{cite web |title = 山坳中的上海话：兴衰背后见证中国社会微妙变化 |publisher = 新闻周刊 |date = 2005-10-19 |url = http://www.china.com.cn/chinese/difang/1003270.htm |accessdate = 2012-04-20 }}}}。到1949年，上海全地区人口502.92万，市区人口418.94万{{RefTag |1 = {{cite web |title = 上海市地方志办公室-上海人口数量 |url = http://www.shtong.gov.cn/newsite/node2/node2247/node4564/node79123/node79137/userobject1ai103292.html}}}}。同时也聚集了例如[[宋嘉澍家族|宋嘉澍家族]]、[[荣毅仁|荣毅仁]]、[[邵逸夫|邵逸夫]]、[[徐悲鸿|徐悲鸿]]、[[胡蝶|胡蝶]]等众多中国近代史上的政商文艺名人。

1949年以后，随着支援内地、[[上山下乡|上山下乡]]等各类政治运动，一定数量的上海居民被迁移至中国各地，如[[安徽|安徽]]、[[云南|云南]]、[[东三省|东三省]]等地。而一些富裕阶层则在1950年代左右移民香港或经由香港前往台湾。1978年以后，大批上海籍知青、支内纷纷返回上海。1990年代，随着出国风潮，大量上海民众又移民[[日本|日本]]、[[澳大利亚|澳大利亚]]、[[美国|美国]]等地。

另外在1949年以前，有数量众多的英、法等国公民居住于上海。[[二战|二战]]时期，上海因无需签证、担保即可进入的地区而接纳了大批由[[欧洲|欧洲]]因躲避[[納粹德國|纳粹]]而逃难的[[犹太|犹太]]难民。1990年代以后，随着商务活动的增加，上海再度聚集了诸多外籍人士{{RefTag |1 = {{cite web |title = 浦东概述：浦东生活之公共服务 |publisher = 浦东新闻网 |date = 2010-11-12 |url = http://pudongnews.eastday.com/qxxc/node39/node51/u1a495.html |accessdate =2012-04-20}}}}。

尽管上海全市户籍人口自然增长率自1993年后一直为负增长，但上海每年以引进人才的方式给予2万余人户籍，户籍人口依旧保持了连年增加。上海曾长期对户籍进行严格控制，但近年来已有所放宽{{RefTag |1 = {{cite web |title = 上海户籍人口第17年负增长 |publisher = 新闻晨报 |date = 2010-03-09 |url = http://china.huanqiu.com/roll/2009-03/403114.html}}}}。另外随着流动人口的剧烈增加，上海市也正面临种种人口问题，首先上海市的土地、水源、环境、交通等资源承受沉重压力，且大部分犯罪为流动人员所为，对上海的治安造成了影响{{RefTag |1 = {{cite web |title = 中国大城市人口严重超载 人口流动带来犯罪问题 |publisher = 中国经济网 |date = 2010-02-23 |url = http://business.sohu.com/20100223/n270381164.shtml |accessdate = 2012-04-28}}}}。数据显示，截止2012年上海中心城区人口密度是[[东京|东京]]的2.4倍、[[伦敦|伦敦]]的3.5倍、[[巴黎|巴黎]]的4.8倍，而以2012年为基点，上海人口或许在3年内超过城市承载量{{RefTag |1 = {{cite web |title = 三年内，上海人口或超过承载量 |publisher = 解放日报 |date = 2012年8月15日 |url = http://sh.eastday.com/m/20120815/u1a6784981.html |accessdate = 2013-04-30}}}}。

=== 语言 ===
{{Main|上海话}}

[[普通话|普通话]]为上海的官方语言，在学校教学及正式场合使用。而市民在日常生活中多使用与普通话无法互通的本地語言[[上海话|上海话]]。上海话简称沪语，語言學上屬於[[吴语|吴语]]，一般意义上是指近代以來[[上海租界|上海租界]]範圍和周邊使用的一种现代吴语，由於上海是[[遠東|遠東]]的國際大都市，所以上海話由各地吴语的方言和英語融合而成。上海话也和[[旗袍|旗袍]]、洋房、弄堂等一样，成为了上海风情的代表性符号。

如今上海话的使用范围面临多方面的压缩。伴随外地人的大量湧入，本地户籍人口比例逐渐减小，加之政府有意在公共場所限制上海話的使用，使得普通话越来越成为公共场合语言。由于國家禁止校園內講上海話，也禁止上海話電視台，上海少年一代对于[[上海话|上海话]]的掌握能力开始下降{{RefTag |1 = {{cite web|title=保护断裂的上海话 |publisher=东方早报 |date=2011-03-03 |url=http://www.dfdaily.com/html/63/2011/3/3/575142.shtml |accessdate=2012-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20130326015122/http://www.dfdaily.com/html/63/2011/3/3/575142.shtml |archivedate=2013-03-26 }}}}，社会上也出现了一些保护上海话的行动{{RefTag |1 = {{cite web |title = 独家:《上海话大词典》面世 "说打"沪语先过"音标"关 |publisher = 东方网 |date = 2007-08-15 |url = http://sh.eastday.com/qtmt/20070815/u1a342489.html |accessdate = 2012-04-20 }}}}。到目前上海话仍然在日常占有主要地位。上海话的使用和发展一直是上海社会讨论的话题{{RefTag |1 = {{cite web|title=山坳中的上海话 |publisher=中国新闻周刊 |date=2005-10-17 |url=http://www.chinanewsweek.com.cn/2005-10-24/1/6473.html |archiveurl=https://web.archive.org/web/20090501184358/http://www.chinanewsweek.com.cn/2005-10-24/1/6473.html |archivedate=2009-05-01 |accessdate=2013-04-21 |deadurl=yes }}}}。

当今的上海，与[[中国内地|中国内地]]许多其他地方一样，共有五种获得官方认可的合法宗教在开展相应的活动，分别为：[[道教|道教]]、[[佛教|佛教]]、[[天主教|天主教]]、[[新教|基督教新教]]和[[伊斯兰教|伊斯兰教]]。这些宗教在1949年后一度势衰，并在[[文革|文革]]时受到较大影响，后又逐渐恢复。至2009年，有宗教信仰群众88.76万人，以佛教信徒为最多{{RefTag |1 = {{cite web |title = 上海年鉴2009>>四十三、民族·宗教>>第一节 综述 |publisher = 上海地方志办公室 |date = 2010 |url = http://www.shtong.gov.cn/newsite/node2/node19828/node81801/node81858/node81964/userobject1ai110557.html |accessdate =2012-04-20}}}}。历史上上海还曾经存在过许多其他宗教，例如[[犹太教|犹太教]]、[[东正教|东正教]]、[[锡克教|锡克教]]等，现在仅存遗址。

上海地区的佛教和道教均可追溯至[[三国|三国时期]][[孫吳|孙吴]]时期。现开放寺庙98所{{RefTag |1 = {{cite web |title = 上海市佛教概况 |publisher = 上海民族和宗教事务委员会 |date = 2010 |url = http://www.shmzw.gov.cn/gb/mzw/shzj/fj/index.html |accessdate = 2012-04-12 |archive-url = https://web.archive.org/web/20120531083302/http://www.shmzw.gov.cn/gb/mzw/shzj/fj/index.html |archive-date = 2012-05-31 |dead-url = yes }}}}，以始建于三国时期的[[龙华寺_(上海)|龙华寺]]历史最为悠久。而1949年时，上海尚有道观236座，如今仅存27座{{RefTag |1 = {{cite web |title = 上海市道教概况 |publisher = 上海民族和宗教事务委员会 |date = 2010 |url = http://www.shmzw.gov.cn/gb/mzw/shzj/dj/index.html |accessdate = 2012-04-20 |archive-url = https://web.archive.org/web/20120531083450/http://www.shmzw.gov.cn/gb/mzw/shzj/dj/index.html |archive-date = 2012-05-31 |dead-url = yes }}}}，最著名的非[[正一道|正一道]]的[[上海城隍庙|城隍庙]]莫属，每年都会举行盛大的[[庙会|庙会]]活动，周边也已是沪上知名的商业区域，各种[[小吃|小吃]]餐厅、[[古玩|古玩]]商铺和小商品市场鳞次栉比，成为重要的旅游景点，另一著名道观为[[全真道|全真道]]的[[海上白云观|海上白云观]]{{RefTag |1 = {{cite web |title = 白云观 |publisher = 上海市民族宗教委员会 |date = 2001-03-11 |url = http://www.shmzw.gov.cn/gb/mzw/shzj/zjcs/userobject8ai805.html |accessdate = 2012-04-20 |archive-url = https://web.archive.org/web/20111109124529/http://www.shmzw.gov.cn/gb/mzw/shzj/zjcs/userobject8ai805.html |archive-date = 2011-11-09 |dead-url = yes }}}}。[[伊斯兰教|伊斯兰教]]最早随[[宋代|宋代]]在上海镇经商的[[阿拉伯帝国|大食]]商人进入上海，现有7坊清真寺{{RefTag |1 = {{cite web |title = 上海市伊斯兰概况 |publisher = 上海民族和宗教事务委员会 |date = 2010 |url = http://www.shmzw.gov.cn/gb/mzw/shzj/yslj/index.html |accessdate = 2012-04-20 |archive-url = https://web.archive.org/web/20120531082602/http://www.shmzw.gov.cn/gb/mzw/shzj/yslj/index.html |archive-date = 2012-05-31 |dead-url = yes }}}}，信徒集中于[[回族|回族]]、[[维吾尔族|维吾尔族]]等10个少数民族。

[[天主教|天主教]]和[[基督教新教|基督教新教]]则为近代由[[传教士|传教士]]传入。[[明朝|明]]万历三十六年，来自[[意大利|意大利]]的天主教传教士开始在沪传道。1949年以后，由于[[罗马教廷|罗马教廷]]和中国政府[[中華人民共和國與梵蒂岡關係|长期不和]]，天主教在沪发展的较为艰难。在上海现存的107座天主教堂中{{RefTag |1 = {{cite web |title = 上海市天主教概况 |publisher = 上海民族和宗教事务委员会 |date = 2010 |url = http://www.shmzw.gov.cn/gb/mzw/shzj/tzj/index.html |accessdate = 2012-03-12 |archive-url = https://web.archive.org/web/20120108104456/http://www.shmzw.gov.cn/gb/mzw/shzj/tzj/index.html |archive-date = 2012-01-08 |dead-url = yes }}}}，地处上海西郊[[佘山|佘山]]的[[佘山进教之佑圣母大殿|圣母大殿]]，气势宏伟，为远东最大的天主教堂{{RefTag |1 = {{cite web |title = 远东第一大教堂--圣母大殿 |publisher = 松江旅游 |date =  |url = http://ly.songjiang.gov.cn/Web/sub/SjTop_Detail.aspx?id=262 |accessdate = 2012-04-21}}}}，也是重要的圣母朝圣地。而基督教新教在上海出现得最晚，是在开埠的1843年由英国传教士传入的。上海的基督教堂多达169处{{RefTag |1 = {{cite web |title = 上海市基督教概况 |publisher = 上海民族和宗教事务委员会 |date = 2010 |url = http://www.shmzw.gov.cn/gb/mzw/shzj/jdj/index.html |accessdate = 2012-04-20 |archive-url = https://web.archive.org/web/20120531082725/http://www.shmzw.gov.cn/gb/mzw/shzj/jdj/index.html |archive-date = 2012-05-31 |dead-url = yes }}}}，有相当一部分成为了[[上海市文物保护单位|上海市文物保护单位]]，且大多位于市中心，因而也被游客视为热门的旅游景点{{RefTag |1 = {{cite web |title = 上海宗教志>>第五编基督教>>第一章场所 第一节、开放教堂 |publisher = 上海地方志办公室 |url = http://www.shtong.gov.cn/dfz_web/DFZ/Info?idnode=75317&tableName=userobject1a&id=92054 |accessdate = 2012-04-20 }}}}。

上海的公交车最早出现于1908年{{RefTag |1 = {{cite web |title = 走过百年的上海公交1908年——2008年 |publisher = 新民网 |date = 2008-02-26 |url = http://sh.xinmin.cn/shizheng/2008/02/26/1062812.html |accessdate = 2011-03-11 |deadurl = yes |archiveurl = https://web.archive.org/web/20111109121746/http://sh.xinmin.cn/shizheng/2008/02/26/1062812.html |archivedate = 2011-11-09 }}}}，到2011年，全上海公交线路1100多条，公交车近17000辆，其中包括行駛在市區13條線路上的358輛無軌電車。公交网络已经覆盖每一个行政村{{RefTag |1 = {{cite web |title =上海将实现“村村通公交” |publisher = 票价搜索 |date = 2008-03-15 |url =http://www.piaojia.cn/news/show.asp?id=29188 |accessdate = 2012-04-15}}}}，日客运量为770万人次{{RefTag |1 = {{cite web |title =沪轨交日均516万次 每天335万元补贴公交 |publisher = 腾讯新闻 |date = 2011-03-11 |url =http://news.qq.com/a/20110311/000589.htm?pgv_ref=aio |accessdate = 2012-04-15}}}}。

轨道交通是又一重要的市内交通交通工具，在客流量上和公交车平分秋色。包括16条[[上海地铁|地铁]]线路，1条[[上海磁浮示范运营线|磁悬浮线路]]，1条市域铁路（[[金山铁路|金山铁路]]），1条[[张江有轨电车|有轨电车线路]]，于1995年4月正式开始运营。截至2018年3月31日，[[上海地铁|上海地铁]]运营总里程644公里（不含磁浮线），居世界首位{{RefTag |1 = {{cite journal |title = 2012年中国城市轨道交通运营线路统计与分析 |author = 鲁放、韩宝明、王芳玲 |journal = 都市快轨交通 |year = 2013 |issue = 1}}}}，亦是世界上最繁忙的城市地铁系统之一。地铁线路将中心城区密集覆盖，并连接了大多数区县，工作日日均客流超900万{{RefTag |1 = [http://news.southcn.com/z/2012-03/30/content_41968822.htm 上海地铁工作日日均客流逼近700万]}}。

于1999年起出现的[[上海公共交通卡|上海公共交通卡]]是市民出行的主要结算方式，发行总量已经超过3000万张{{RefTag |1 = {{cite web |title = 交通卡能否网上充值|publisher = 新浪新闻 |date = 2008-02-20 |url = http://news.sina.com.cn/o/2008-02-20/131313443498s.shtml|accessdate = 2012-04-15}}}}。这是一种可储值的非接触式[[IC卡|IC卡]]，可用于上海市公交、轨交、轮渡、出租车、停车场的付费，以及上海境内高速公路路段的收费、公用事业收费和加油站等多领域使用，并可在[[长江三角洲|长江三角洲]]的部分城市通用{{RefTag |1 = {{cite web|title =交通卡年底升级长三角通用|publisher =新民网|date =2009-04-26|url =http://sh.xinmin.cn/minsheng/2009/04/26/1872767.html|accessdate =2012-04-15|deadurl =yes|archiveurl =https://web.archive.org/web/20121120214111/http://sh.xinmin.cn/minsheng/2009/04/26/1872767.html|archivedate =2012-11-20}}}}。

上海是中国的三大航空枢纽之一，拥有[[上海虹桥国际机场|上海虹桥国际机场]]与[[上海浦东国际机场|上海浦东国际机场]]两座国际机场，2014年运送旅客8965万人次，年货邮吞吐量361万吨。两机场年起降飞机65万架次{{RefTag |name=航空运输|1 = {{cite web |title = 上海年鉴2010>>十一、交通运输业>>（九）航空运输 |publisher = 上海地方志办公室 |url = http://www.shtong.gov.cn/newsite/node2/node19828/node82611/node82647/node82661/userobject1ai115696.html|accessdate = 2011-03-12}}}}，均被[[中國東方航空|中國東方航空]]与[[中國國際航空|中國國際航空]]作为[[枢纽机场|枢纽机场]]使用。2016年12月12日，中国城市机场首次旅客突破1亿人次。纵观全球，上海之前，有英国伦敦、美国纽约、亚特兰大和日本东京，上海由此成为全球第5个进入亿级人次航空“俱乐部”的城市。兩大機場的貨運吞吐量，也可望連續9年位居全球第3名。

1921年，虹桥机场在上海西郊兴建并于1929年开始系统运营，之后进行过多次扩建，现已成为[[虹桥综合交通枢纽|虹桥枢纽]]的一部分。浦东国际机场于1999年投入使用，迅速成为中国最繁忙的机场之一，航线覆盖世界73个国家和地区以及62个国内城市{{RefTag |name=航空运输}}，并有两岸直航。

此外，[[中国东方航空|中国东方航空]]、[[上海航空|上海航空]]、[[春秋航空|春秋航空]]、[[吉祥航空|吉祥航空]]、[[中国货运航空|中国货运航空]]、上海国际货运航空、[[扬子江快运|扬子江快运航空]]、[[长城航空|长城航空]]等8家航空公司也以上海为基地。

[[北宋|北宋]][[元祐_(年號)|元祐]]年间，华亭县便已开设[[县学|县学]]。[[上海开埠|上海开埠]]后，外国传教士和教会组织开始在上海开设各类学校。1850年，法国天主教耶稣会建立了中国第一所西式现代中学圣依纳爵公学，即今日[[上海市徐汇中学|上海市徐汇中学]]。1863年，[[李鸿章|李鸿章]]奏请开设了上海首家华人新式学堂即[[上海广方言馆|上海广方言馆]]{{RefTag |1 = {{cite web |title = 卢湾区志>>第三十四编专记>>江南机器制造总局>> 五、广方言馆|publisher = 上海地方志办公室 |url = http://www.shtong.gov.cn/newsite/node2/node4/node2249/luwan/node37536/node62750/node62768/userobject1ai50872.html|accessdate = 2011-03-14 }}}}，[[美国圣公会|美国圣公会]]于1879年在上海创立了[[圣约翰书院|圣约翰书院]]，并自1892年起开设大学课程，而1896年创办的[[南洋公学|南洋公学]]则为中国近代最早的大学之一。至1936年，上海已有高校34所。[[淞沪会战|淞沪会战]]爆发后，大部分高校随[[国民政府|国民政府]]迁到内陆，并于1945年返回，同时上海开始实行[[国民教育|国民教育制度]]。

1949年5月，全市公立学校和[[国立大学|国立大学]]均被[[中国人民解放军|中国人民解放军]]接管，私立学校遭整顿，同时教会学校和外资学校被收回，最为典型的是上海[[圣约翰大学_(上海)|圣约翰大学]]和[[沪江大学|沪江大学]]两所教会大学在1952年的院系调整中被撤销，教学人员被分配至其他学校。1952年进行的[[中国高等院校院系调整|高校院系调整]]中沪上一些学校部分或全部前往内陆{{RefTag |1 = {{cite web |title = 西安交大举行西迁50周年纪念座谈会(图) |publisher = 中国教育在线 |date = 2006-04-10 |url = http://www.eol.cn/xin_wen_2042/20060410/t20060410_172790.shtml|accessdate = 2011-03-12}}}}，外地一些高校的师资则部分迁来上海。

在开埠后，上海由[[上海公共租界工部局|公共租界工部局]]进行了大量市政建设{{RefTag |1 = {{cite web |title = 上海租界志>>第二篇机构>>第三章行政管理机构 第一节 道路码头委员会 |publisher = 上海地方志办公室 |date = 2001-03-11 |url = http://www.shtong.gov.cn/dfz_web/DFZ/Info?idnode=64473&tableName=userobject1a&id=57975|accessdate = 2012-04-03}}}}，包括[[电厂|电厂]]、[[自来水|自来水]]厂等。1927年制定《[[大上海计划|大上海计划]]》则是上海最早城市总体规划方案。现在，上海的城市规划依据2001年5月由国务院批复的《上海市城市总体规划》（1999年-2020年）进行，以建设国际经济、金融、贸易、航运中心为核心{{RefTag |1 = {{cite web |title = 上海建科创中心需处理好五个关系 |publisher = 网易新闻 |date = 2015-09-01 |url = http://money.163.com/15/0902/01/B2FJFSSA00253B0H.html |accessdate = 2012-04-03 }}}}。在[[上海城市规划展示馆|上海城市规划展示馆]]中，可见到相应以1:500比例制作的内环线100余平方公里范围内城市规划模型{{RefTag |1 = {{cite web|title =规模最大的城市规划模型 —“上海市内环线内主体模型”|publisher =大世界吉尼斯|date =|url =http://www.dsjjns.com/msg.php?id=2103|accessdate =2016-04-14|deadurl =yes|archiveurl =https://web.archive.org/web/20160304113727/http://www.dsjjns.com/msg.php?id=2103|archivedate =2016-03-04}}}}。

1949年以后的几十年中，市政建停留在新建工人住房、置换电水管网等阶段。自1990年代起，上海开始大规模市政建设，广泛推行平改坡工程（将旧房屋的平顶改造为坡顶）{{RefTag |1 = {{cite web |title =上海"平改坡"工程将大扩容 |publisher = 东方网 |date = 2002-07-20 |url = http://news.eastday.com/epublish/gb/paper148/20020720/class014800006/hwz720533.htm|accessdate = 2011-04-18 |language = zh }}}}和屋顶水箱改造工程{{RefTag |1 = {{cite web |title =中心城区屋顶水箱改造工程已经纳入“十一五”规划|publisher = 东方网 |date = 2006-04-12 |url = http://news.eastday.com/epublish/gb/paper148/20020720/class014800006/hwz720533.htm |accessdate = 2012-04-03}}}}。对违法建筑和环境脏乱路段也年年进行整治{{RefTag |1 = {{cite web |title = 上海年鉴2009>>城市建设管理>>第九节市容环境卫生 |publisher = 中国上海 |date = 2001-03-11 |url = http://www.shtong.gov.cn/newsite/node2/node19828/node81801/node81840/node81928/userobject1ai110362.html |accessdate = 2012-04-03}}}}。以举办2010年[[上海世界博览会|上海世界博览会]]为契机，上海市政部门对城市景观、道路、居民住宅等均进行了大规模的整修。上海历史上遗留下了大面积的[[棚户区|棚户区]]，在1990年代的对其中365万平方米的危棚简屋进行了改造，但是曹家村、光复路一代的棚户区由于成本问题始终未能动工，与周边高楼行成鲜明对比的[[城中村|城中村]]景观留存至今{{RefTag |1 = {{cite web |title =“蜗居”上海的人们 上海棚户区生活现状特写 |publisher =21世纪经济报道 |date = 2011-03-11 |url = http://sh.house.163.com/10/0311/09/61G2JA6E00073SDJ.html|accessdate = 2012-04-03}}}}。

上海开埠以前，园林绿化全部都为私家园林，最为著名的有[[豫园|豫园]]、露香园、[[古猗园|古猗园]]、秋霞圃、醉白池等等；开埠后，公共园林开始出现，1868年诞生的[[黄浦公园|公共花园]]为第一座公园。但1928年前，中国人不得进入[[工部局|工部局]]和[[上海法租界公董局|公董局]]兴建的公园{{RefTag |1 = {{cite web |title = 上海园林志>>第二篇公园 |publisher = 上海地方志办公室 |url = http://www.shtong.gov.cn/newsite/node2/node2245/node69854/node69860/index.html|accessdate = 2011-03-11}}}}。目前，上海全市除[[上海动物园|上海动物园]]、[[上海野生动物园|上海野生动物园]]等个别公园外，已经取消公园门票{{RefTag |1 = {{cite web |title = 上海免费公园为何让人欢喜让人忧 |publisher = 人民网社会频道 |date = 2005-04-15 |url = http://society.people.com.cn/GB/41158/3324173.html|accessdate = 2011-03-11}}}}。自2000年以后，为了净化城市空气，美化环境，上海开始在城市中心和郊区兴建一批开放绿地。到2008年，上海全市绿地总面积34256公顷，公共绿地14777公顷，城市绿化覆盖率为38%。全市林地总面积到近10万公顷，森林覆盖率11.6%{{RefTag |1 = {{cite web |title = 上海森林覆盖率10年增3倍 |publisher = 文汇报 |date = 2011-01-23 |url = http://whb.news365.com.cn/zhxw/201101/t20110123_2942227.htm |accessdate = 2011-03-11 |deadurl = yes |archiveurl = https://archive.is/20130101224139/http://whb.news365.com.cn/zhxw/201101/t20110123_2942227.htm |archivedate = 2013-01-01 }}}}。

为解决自1990年代开始大规模市政建设对资金的大量需求，上海市政府部分官员主导了对上海的投融资体制的大幅改革，先利用政府手中的土地资源，引入[[土地批租|土地批租]]的融资方式。期间，上海通过土地批租共筹措资金1000多亿元。1997年，受[[亚洲金融危机|亚洲金融危机]]的影响，上海的土地批租大量减少，但国内证券市场规模迅速扩大，居民储蓄存款不断增长，社会富余资金增加。面对这种情况，上海市政府制定政策将有盈利的基础设施项目推向市场，实行社会融资，同时制定多项措施，降低企业参与融资的风险。[[上海1990年代|在1990年代]]，上海通过社会渠道共融资超过2000亿元。同时上海市积极借鉴国际经验，通过进一步完善开发机制；扩大资本市场、社会保险资金参与融资；探索采用BOT、BOO等国际通行的融资方式以及加大[[公共财政|公共财政]]的扶持力度等措施，进一步扩大融资范围和能力，以满足市政建设的需要{{RefTag |1 = {{cite news |url = http://news.xinhuanet.com/fortune/2002-05/09/content_386317.htm |title = 陈良宇谈上海现代化城市建设与投融资体制改革 |publisher = 新华网 |date = 2002年5月9日|accessdate = 2013-06-05}}}}。

=== 公共卫生 ===

上海开埠后，西医随同传教士和医生一同进入。1844年，出现现代医院{{RefTag |1 = {{cite web |title = 上海通志--第三十六卷、卫生--概述 |publisher = 上海地方志办公室 |url = http://www.shtong.gov.cn/newsite/node2/node2247/node4593/index.html|accessdate = 2011-03-12}}}}。1904年，上海万国红十字会成立，为[[中国红十字会|中国红十字会]]的前身{{RefTag |1 = {{cite web |title = 中国红十字会初创及初步发展时期_高清图集 |publisher = 新浪网 |url =http://slide.news.sina.com.cn/c/slide_1_1794_11803.html |accessdate = 2011-03-12}}}}。现在，上海的公共医疗体系以[[上海市卫生局|上海市卫生局]]、上海市公共卫生应急指挥中心和28家大型[[三级甲等医院|三级甲等医院]]为主，辅之以社区医疗机构。同时，在各郊区县开始建设三级甲等医院的分院，并进行村卫生室的标准化建设。截止2012年，上海市共有医疗卫生机构3465所，专业卫生技术人员14.61万人{{RefTag|name=2012公报}}。
1988年春季，上海市爆发了[[1988年上海市甲型肝炎大流行|甲型肝炎大流行]]，期间上海市卫生防疫部门全力动员进行积极防治，此次疫情共造成超过30万人感染和31人死亡，疫情过后上海市政府总结经验并逐步完善了突发传染病疫情的防控预警机制以及应急处理预案{{RefTag |1 = {{cite book |title = 亲历：上海改革开放30年 |publisher=上海辞书出版社 |author = [[曹景行|曹景行]]，复旦大学口述历史研究中心 |year=2008 |isbn = 9787532625895}}}}。2003年中国大陆[[SARS事件|SARS事件]]期间，得益于先前甲肝大爆发取得到经验和防控体系，上海市的防疫工作得力使得上海未发生大规模疫情，因而获得[[世界卫生组织|世界卫生组织]]在内的各界肯定和好评{{RefTag |1 = {{cite web |title = 聚焦：上海为何至今未大范围爆发非典(组图)|publisher =新华网 | url = http://news.sina.com.cn/c/2003-05-01/10351026434.shtml|accessdate = 2013-04-07 }}}}{{RefTag |1 = {{cite news |title = 上海如何防疫情 从“甲肝大流行”中吸取经验 |publisher =南风窗|date = 2003-05-16 |url = http://news.sohu.com/93/59/news209295993.shtml|accessdate = 2013-04-07 }}}}。2013年长三角地区发生了[[甲型流感病毒H7N9亞型|H7N9]]流感疫情，上海卫生部门反应迅速，及时扑杀相关市场内的禽鸟以切断传染源{{RefTag |1 = {{cite news |title = 上海连夜关闭一处活禽市场 捕杀所有禽鸟 |url = http://news.ifeng.com/photo/hdnews/detail_2013_04/05/23886264_0.shtml#p=1 |accessdate=2013-04-05 |newspaper=凤凰网资讯|date=2013年4月5日}}}}。由于上海市多管齐下的得利防控措施，目前已经基本消除疫情{{RefTag |1 = {{cite news |title = 上海连续20天无H7N9禽流感新发病例 转入常态化管理 |url = http://society.people.com.cn/n/2013/0510/c1008-21440669.html|accessdate=2013-06-03|publisher=人民网|date=2013年5月10日}}}}。

截至2013年，上海户籍人口的人均期望寿命已超过82岁，位列[[中华人民共和国各城市预期寿命列表|中国大陆第一位]]，人均基本公共卫生服务经费达53.7元。上海居民三大健康指标（期望寿命、孕妇死亡率和新生儿死亡率）均保持在[[发达国家|发达国家]]水平{{RefTag |1 = {{cite news|author=[[上海市人民政府|上海市人民政府]]|title=截至2013年 上海市人口预期寿命达到82.47岁|url=http://www.shanghai.gov.cn/shanghai/node2314/node2315/node17239/node17241/u21ai863898.html|accessdate=2014-08-10}}}}。

=== 公共媒體 ===

==== 出版 ====
[[File:Oriental_Library.jpg|thumb]]前，[[商务印书馆|商务印书馆]]的东方图书馆]]

{{參見|1=Template:上海出版机构|l1=上海出版机构}}

1843年，上海英商[[墨海书馆|墨海书馆]]引进[[中国|中国]]第一台印刷机，成为中国最早的现代化出版社{{RefTag |1 = {{cite web |title = 咸丰三年(1853)墨海书馆出版博物 |publisher = 上海印刷博物馆 |url = http://sppcm.com.gp06.datas.com.cn/bencandy.php?fid=20&id=358|date = 2014-03-13}}}}。1897年，[[商务印书馆|商务印书馆]]在上海成立{{RefTag |1 = {{cite web|title =商务印书馆与上海|publisher =文汇读书周报|date =2007-06-29|url =http://dszb.whdszb.com/bl/200706/t20070629_1471583.htm|accessdate =2011-03-14|deadurl =yes|archiveurl =https://archive.is/20130428134839/http://dszb.whdszb.com/bl/200706/t20070629_1471583.htm|archivedate =2013-04-28}}}}。1912年，[[中华书局|中华书局]]在沪开业{{RefTag |1 = {{cite web|title=希望是最后的个案——中华书局上海总厂的前世今生 |publisher=中华书局 |date=2010-06-07 |url=http://www.zhbc.com.cn/shownews.asp?id=857 |accessdate=2011-03-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20121107202311/http://www.zhbc.com.cn/shownews.asp?id=857 |archivedate=2012-11-07 }}}}，并迅速发展崛起。上海此后逐渐成为了中国出版业的中心。到1910年代底，上海各类出版销售机构增加到120余家，占全国出版机构的80%。到1936年，[[商务印书馆|商务印书馆]]、中华书局和世界书局的出版已占全中国出版图书的71%。

1949年起，政府开始对出版进行整顿，形成以[[新华书店|新华书店]]为核心的图书出版发行。进入1959年以后，包括商务印书馆、中华书局在内的大型印刷出版机构被迁移至北京{{RefTag |1 = {{cite web |title = 上海通志>>第四十一卷报业、通讯、出版、广播、电视>>第二章出版>>第一节出版机构 |publisher = 上海地方志办公室 |url = http://www.shtong.gov.cn/newsite/node2/node2247/node4598/node79770/node79779/userobject1ai104443.html|accessdate = 2011-03-12 }}}}，上海出版行业逐渐衰落，到1969年全年上海出版仅为16种。

进入2000年，上海各出版社开始被整合成为[[上海世纪出版集团|上海世纪出版集团]]和[[上海文艺出版集团|上海文艺出版集团]]。[[上海辞书出版社|上海辞书出版社]]出版的《[[辞海|辞海]]》颇为著名，而[[上海文艺出版社|上海文艺出版社]]则有著名期刊《[[故事会|故事会]]》。

==== 报业 ====
[[File:申报创刊号.jpg|thumb]]》为近代中国的重要报纸，图为其创刊号]]

上海现代化报业自开埠以后便开始发展，之前主要有一些邸钞、《[[京报|京报]]》等官方公报的传统新闻传播媒介。1850年，全英语的《[[北华捷报|北华捷报]]》创刊，成为上海最早的报纸{{RefTag |1 = {{cite web|title=《北华捷报》创刊号 |publisher=上海档案信息网 |date=2008-04-08 |url=http://www.archives.sh.cn/docs/200804/d_187647.html |accessdate=2011-03-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20111109094814/http://www.archives.sh.cn/docs/200804/d_187647.html |archivedate=2011-11-09 }}}}。1861年，汉语报纸《[[上海新报|上海新报]]》创刊。1872年，近代影响力最大的报纸之一的《[[申报|申报]]》创刊{{RefTag |1 = {{cite web |title = 1872年4月30日 《申报》创刊 |publisher = 人民网|date = 2010-04-30 |url = http://news.ifeng.com/history/today/detail_2010_04/30/1474661_0.shtml |accessdate = 2011-03-12 }}}}。至1936年，《申报》《新闻报》《[[时报_(上海)|时报]]》《时事新报》《[[大公报|大公报]]》等五大报纸，每日销售总数即达到50万份。至1937年淞沪会战开始后，《申报》等由于不接受日军的新闻审查而停刊。1941年上海完全沦陷后，上海报业基本停运。直到1945年才恢复，至1949年5月，上海全市有各类报纸70余种{{RefTag |1 = {{cite web |title = 上海新闻志>>第一编 报纸 |publisher = 上海地方志办公室 |url =http://www.shtong.gov.cn/newsite/node2/node2245/node4522/node5501/index.html |accessdate = 2011-03-14}}}}，为全国报业中心{{RefTag|name=上海通志第四十一卷概述|1 = {{cite web |title = 上海通志>>第四十一卷报业、通讯、出版、广播、电视-概述 |publisher= 上海地方志办公室 |url =http://www.shtong.gov.cn/dfz_web/DFZ/PianInfo?idnode=4598&tableName=userobject1a |accessdate = 2011-03-12}}}}。此外，民國時期长期发行的著名杂志有《东方杂志》、《科学》、《中华教育界》、《教育与职业》、《生活》、《世界知识》、《[[现代评论|现代评论]]》等。

1949年[[中国人民解放军|中国人民解放军]]接管上海以后，由于《申报》因属于中国国民党党产而被没收停刊，上海报业也开始转向全面国有，到了[[上海1960年代|1960年代]]中期以后，全市只有《[[解放日报|解放日报]]》和《[[文汇报_(上海)|文汇报]]》两种{{RefTag |1 = {{cite web |title = 上海中文报纸史略(1861～1949) |publisher = 中国知网 |date = 1992-03 |url = http://www.cnki.com.cn/Article/CJFDTOTAL-TNGZ199204044.htm |accessdate = 2011-03-12 }}}}。1979年起，上海大批报纸开始复刊。

目前上海主要的两大报业集团，解放日报和文汇新民两大报业集团，合并为上海报业集团。《解放日报》和《文汇报》等保留了党报性质，在一些企事业单位实行强制订阅{{RefTag |1 = {{cite web |title = 上海：确保党报党刊发行 |publisher =新闻晨报 |date = |url = http://www.people.com.cn/GB/14677/40606/3069185.html |accessdate = 2012-04-16}}}}。而较为贴近平民的报纸有《[[新民晚报|新民晚报]]》、《[[新闻晨报|新闻晨报]]》等，以《新民晚报》的发行量为最大。

==== 广播电视 ====
{{Main|上海广播电视台}}

1923年1月23日在上海开播的[[大陆报-中国无线电公司广播电台|大陆报-中国无线电公司广播电台]]是中国首家公共广播电台{{RefTag |1 = {{cite web |title = 外国人在华广播电台的建立 |publisher = 中国现代广播史 |date = 2007-06-21 |url = http://www.fmdiy.com/ReadNews.asp?NewsID=409|accessdate = 2011-03-12 }}}}。1920年代以后，随着相关无线广播的法规制定，私营电台与国有电台迅速发展。到1945年，拥有营业许可的国营电台和民营电台达108家{{RefTag|name=上海通志第四十一卷概述}}，广播节目除新闻时讯外，也经常播出苏州[[评弹|评弹]]和[[滑稽戏|滑稽戏]]等[[曲艺|曲艺]]节目。1949年5月27日，上海人民广播电台成立{{RefTag |1 = {{cite web |title = 上海通志>>第四十一卷报业、通讯、出版、广播、电视>>第三章广播>>第一节、无线广播 |publisher = 上海地方志办公室 |url = http://www.shtong.gov.cn/newsite/node2/node2247/node4598/node79771/node79778/userobject1ai104472.html |accessdate = 2011-03-12 }}}}，同时各类私营电台被陆续整合，至1953年，仅剩上海人民广播电台一家。1992年初，上海东方广播电台成立{{RefTag |1 = {{cite web |title = 上海广播电视志>>第一编广播电台>>第一章无线广播>>第五节、上海东方广播电台 |publisher = 上海地方志办公室 |url = http://www.shtong.gov.cn/newsite/node2/node2245/node4510/node10159/node10168/node63808/userobject1ai54375.html |accessdate = 2011-03-12}}}}，成为同上海人民广播电台有竞争关系的第二家省级广播电台。1990年起[[上海人民广播电台|上海人民广播电台]]播出以[[上海市公安局刑侦总队|上海市公安局刑侦总队]]为原型的大型系列[[广播剧|广播剧]]《[[刑警803_(广播剧)|刑警803]]》，为中国播出数量最多、时间跨度最长的广播剧{{RefTag |1 = {{cite web |url = http://city.ifeng.com/cskx/20111214/189297.shtml |language = zh-Hans |title = 《刑警803》成为中国最长寿广播剧 |publisher = 凤凰网 |date = 2011-12-14 |accessdate = 2012-03-25 |deadurl = yes |archiveurl = https://web.archive.org/web/20140302103110/http://city.ifeng.com/cskx/20111214/189297.shtml |archivedate = 2014-03-02 }}}}。

1958年[[上海电视台|上海电视台]]成立并播出黑白电视，而[[彩色电视|彩色电视]]于1973年出现{{RefTag |1 = {{cite book |title = 中国彩电工业发展回顾 |ISBN = 9787121074394 |publisher = 电子工业出版社 |accessdate = 2013-07-27}}}}，同时建成高210米的上海电视塔。为了引进竞争机制，在1993年，[[上海东方电视台|上海东方电视台]]作为上海第二家省级电视台开播{{RefTag |1 = {{cite web |title = 东方卫视总监田明:我只是折射阳光的一滴水 |publisher = 东方网 |date = 2009-10-23 |url = http://sh.eastday.com/qtmt/20091023/u1a645879.html |accessdate = 2011-03-12 }}}}，其新闻减少了部分上海电视台的政治内容，其他节目也更加贴近市民。1994年2月，上海教育电视台开播，成为主要以播送教育与科技信息为主的电视台{{RefTag |1 = {{cite web|title=教视简介 |publisher=上海教育电视台 |url=http://www.setv.sh.cn/web/Intro.htm |accessdate=2012-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20120330113915/http://www.setv.sh.cn/web/Intro.htm |archivedate=2012-03-30 }}}}。1992年起，开始播出[[有线电视|有线电视]]。

2001年，广播电视机构改革，上海人民广播电台、上海东方广播电台、上海电视台、上海东方电视台、上海有线电视台全部并入[[上海文广新闻传媒集团|上海文广新闻传媒集团]]（今[[上海广播电视台|上海广播电视台]]），几乎控制了上海市全部的广播电视节目播出{{RefTag |1 = {{cite web |title = 关于SMG |publisher = 上海广播电视台 |url = http://www.smg.cn/ |accessdate = 2013-07-27}}}}。各电视台也进行明确分工，原本竞争的格局不复存在。

=== 体育 ===

[[上海开埠|上海开埠]]后，近代体育传入上海，后通过新式学校的教学活动在华人学生中开始，促进了上海体育水平的提高。1936年在[[柏林|柏林]]举办的[[1936年夏季奥林匹克运动会|第11届奥林匹克运动会]]上，中国代表团的三分之一为上海运动员{{RefTag |1 = {{cite web |title = 上海通志-第三十七卷体育-第四章竞赛-第二节参加运动会 |publisher = 上海地方志办公室 |url = http://www.shtong.gov.cn/newsite/node2/node2247/node4594/node79706/node79711/userobject1ai103281.html|accessdate = 2011-03-11}}}}。

== 文化 ==
{{Main|海派}}

上海的文化被称为“[[海派|海派文化]]”。海派文化的主体，根植于[[江南|江南]]地区传统的[[吴越文化|吴越文化]]，并且融入了开埠以后来自欧美各国的[[西方文化|西方文化]]，而逐步形成的一种不同于中国其他地区的独特文化。而海派的来源，有说法指于[[清末|清末]][[民初|民初]]的[[海上画派|海上画派]]有关。海上画派的前源是以[[董其昌|董其昌]]为代表的[[吴淞江|松江]]画派。后随着[[上海商业|上海商业]]的繁荣，越来越多的人开始以卖画为生，海上画派之名渐广为人知。

=== 饮食文化 ===
{{Main|上海菜}}

上海菜，当地人称为[[本帮菜|本帮菜]]。最初为上海本地居民的家庭料理，后各地移民纷至沓来，上海菜开始吸收各地不同的风格，调和成今天以浓油赤酱、淡雅爽口、醇厚鲜美、咸淡适中的本帮菜。除一般烹调手法外，本帮菜擅长糟味，极富[[江南|江南]]地区特色。目前本帮菜以绿波廊、上海老饭店等两家百年老店最为著名{{RefTag |1 = {{cite web|title =迷倒众多国家元首的绿波廊特色小点心|publisher =上海旅游网|date =2008-06-25|url =http://www.shanghaitour.net/news/news.jsp?newsid=13224|accessdate =2012-04-20|deadurl =yes|archiveurl =https://web.archive.org/web/20120112100349/http://www.shanghaitour.net/news/news.jsp?newsid=13224|archivedate =2012-01-12}}}}。[[八宝鸭|八宝鸭]]、糟钵头、虾子大乌参、腌笃鲜等均为知名的菜肴，而油爆虾、红烧甩水则是较为知名的家常菜。此外，[[生煎馒头|生煎馒头]]和[[南翔小笼|小笼馒头]]则是本帮小吃的知名代表。

此外，由于上海开埠后大批移民涌入，上海可以见到来自全国各地的[[中国菜系|各菜系]]餐馆。其中[[粤菜|粤菜]]以[[上海杏花楼酒家|杏花楼]]和[[新雅粤菜馆|新雅粤菜馆]]{{RefTag |1 = {{cite web|title =新雅|publisher =新雅粤菜馆|url =http://www.sunya.com.cn/index.php/Home~Detail~dining_culture|accessdate =2013-07-26|deadurl =yes|archiveurl =https://web.archive.org/web/20130921225148/http://www.sunya.com.cn/index.php/Home~Detail~dining_culture|archivedate =2013-09-21}}}}最为著名，而苏菜则以德兴馆为其代表，宁波风格料理则可以在宁波汤团店品尝到。

而西方饮食文化随同上海开埠进入这座城市，1853年开业的老德记西餐馆为沪上首家西餐厅。最初西餐一般以应对外国顾客为主，1910年德大西餐馆开业，成为面向各界民众的西餐馆。[[十月革命|十月革命]]后，由于大批[[蘇維埃俄國|苏俄]]侨民涌入，俄式西餐一度成为大众西餐的典范，其中[[罗宋汤|罗宋汤]]颇受平民喜爱{{RefTag |1 = {{cite web|title =回忆让上海人泪流满面的罗宋汤|publisher =丁丁网|date =2009-12-25|url =http://sh.food.ddmap.com/food_topic_19776_113162_1.htm|accessdate =2012-04-20|deadurl =yes|archiveurl =https://web.archive.org/web/20111109103540/http://sh.food.ddmap.com/food_topic_19776_113162_1.htm|archivedate =2011-11-09}}}}。目前，上海经营各国菜肴的餐馆为数众多，风格来自世界各地。而西方糕点随同西餐传入后由于其价格较之正式西餐相对低廉，受到普通民众的喜爱。

上海民众日常饮食的主食以[[米饭|米饭]]为主，偶尔以[[馄饨|馄饨]]、[[面条|面条]]等作为调节。传统家庭早饭以隔夜的剩余米饭加水制成[[泡饭|泡饭]]，并辅以[[酱菜|酱菜]]、[[咸蛋|咸蛋]]和[[腐乳|腐乳]]等。如在小摊提供的早餐则以[[豆浆|豆浆]]、[[油条|油条]]、[[面饼|大饼]]、[[粢饭|粢饭]]最为常见，因而被沪上民众称为“四大金刚”{{RefTag |1 = {{cite web|title =上海四大金刚|publisher =中外书摘|url =http://ewen.cc/qikan/bkview.asp?bkid=141245&cid=428964|accessdate =2012-04-20}}{{Dead link|date=2019年3月 |bot=InternetArchiveBot |fix-attempted=yes }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}}}。

=== 文学 ===

[[魏晋|魏晋]]时期，华亭县人[[陆机|陆机]]、[[陆云|陆云]]曾为当时一代文豪。而自1919年后，由于租界独特的政治制度，吸引了众多名人，上海很快成为当时[[新文化运动|新文化运动]]基地{{RefTag |1 = {{cite web|title=五四运动的先声：新文化运动 |publisher=新华网 |url=http://news.xinhuanet.com/ziliao/2004-04/27/content_1441831.htm |accessdate=2012-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20111018004946/http://news.xinhuanet.com/ziliao/2004-04/27/content_1441831.htm |archivedate=2011-10-18 }}}}，聚集了[[中国左翼作家联盟|左联]]、[[新感觉派|新感觉派]]、[[鸳鸯蝴蝶派|鸳鸯蝴蝶派]]文学流派，[[鲁迅|鲁迅]]、[[郭沫若|郭沫若]]、[[茅盾|茅盾]]、[[巴金|巴金]]、[[叶圣陶|叶圣陶]]、[[郁达夫|郁达夫]]、[[王造时|王造时]]、[[丁玲|丁玲]]、[[张爱玲|张爱玲]]等名家也在上海从事创作，并产生了《[[子夜_(小說)|子夜]]》等以上海为背景的著名作品。

1980年代以后，海派文学出现了[[叶永烈|叶永烈]]、叶辛、[[王安忆|王安忆]]等知名作家，而以《[[三重门|三重门]]》一举成名的[[韩寒|韩寒]]以其特立独行的作风引发了诸多讨论，被称为韩寒现象{{RefTag |1 = {{cite web |title =“韩寒现象”始末 |publisher = 搜狐网 |url = http://news.sohu.com/20071112/n253201105.shtml |accessdate =2012-04-20}}}}。现如今的上海已成为众多作家的创作摇篮，越来越多的华文作家选择上海作为文学创作的根据地，包括[[陈思和|陈思和]]、秦文君、[[赵丽宏|赵丽宏]]、[[沙葉新|沙葉新]]、[[寧財神|寧財神]]、[[許紀霖|許紀霖]]、[[徐则臣|徐则臣]]和[[蔡骏|蔡骏]]、[[落落|落落]]等等知名作家。

=== 曲艺与传统戏剧 ===

在十九世纪后期，文艺演出成为民众主要的休闲娱乐形式。20世纪初在上海、江苏、浙江一带逐渐出现了[[独角戏|独角戏]]，多由一人以[[吴语|吴语]]方言。在1920年代末期，通过吸收[[话剧|话剧]]艺术和传统说笑艺术，在上海产生了[[滑稽戏|滑稽戏]]{{RefTag |1 = {{cite web|title =王无能|publisher =易文网|date =2006-11-30|url =http://www.ewen.cc/earbook/bkview.asp?bkid=124380&cid=366136|accessdate =2012-04-20}}{{Dead link|date=2019年3月 |bot=InternetArchiveBot |fix-attempted=yes }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}}}，主要使用上海话和其他[[吴语|吴语]]方言。1912年开业的[[大世界|大世界]]游乐场后逐渐成为重要的演出场地。

而随着[[苏州|苏州]]的四大昆班、[[徽班|徽班]]的皮黄、[[京剧|京剧]]艺术等陆续进入上海{{RefTag |1 = {{cite web |title = 历史上的今天 3月2日 |publisher = 中国网 |url = http://www.china.com.cn/aboutchina/data/lssdjt/2009-02/25/content_17334020.htm |accessdate =2012-04-20}}}}。评弹也籍着语言与地理位置的优势，从[[苏州|苏州]]扩展到上海，并随着商业电台的普及，评弹艺术迅速发展，到1930年代，当时每天的评弹节目多达103档。1920年代以后，上海并逐渐形成了以[[周信芳|周信芳]]、[[盖叫天|盖叫天]]为首的海派京剧风格，并吸引了[[梅兰芳|梅兰芳]]等京剧大家常住上海{{RefTag |1 = {{cite web|title=梅兰芳的几次出国演出(附图) |publisher=上海档案信息网 |date=2008-02-27 |url=http://www.archives.sh.cn/docs/200802/d_158621.html |accessdate=2012-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20111109120331/http://www.archives.sh.cn/docs/200802/d_158621.html |archivedate=2011-11-09 }}}}。同一时期，[[浙江|浙江]][[嵊县|嵊县]]（今浙江[[嵊州|嵊州]]）小歌班也在上海舞台上开始发扬[[越剧|越剧]]这一新剧种{{RefTag |1 = {{cite web |title = 怀想当年“越剧十姐妹”绍兴将共演《山河恋》 |publisher = 搜狐娱乐 |date = 2007-02-01 |url =http://yule.sohu.com/20070201/n247975025.shtml |accessdate =2012-04-20}}}}。而上海本地的传统唱腔在融合了现代戏剧形式后，形成了上海特有的[[沪剧|沪剧]]。

至2012年，上海主要的传统剧团有[[上海京剧院|上海京剧院]]、[[上海昆剧团|上海昆剧团]]、[[上海越剧院|上海越剧院]]、上海沪剧院等，昆剧团经常赴其他国家和地区进行演出{{RefTag |1 = {{cite web |title = 上海昆剧团将携全本《长生殿》登上台北舞台 |publisher = 东方网 |date = 2010-03-30 |url = http://sh.eastday.com/qtmt/20100330/u1a713760.html |accessdate = 2011-03-12}}}}，京剧交流活动也时常举办{{RefTag |1 = {{cite web |title = 京津沪京剧流派对口交流演唱会12月23、25日在沪举行 |publisher = 东方网 |date = 2010-12-17 |url = http://sh.eastday.com/qtmt/20101217/u1a836833.html |accessdate =2012-04-20 }}}}，同时借助于电视媒体，出现了《[[越女争锋|越女争锋]]》等选秀活动。上海滑稽剧团、上海人民滑稽剧团和上海青年滑稽剧团等也开始借助于电视进行演出，尤其《[[老娘舅|老娘舅]]》节目自1995年起连演12年，成为上海电视上最长寿的电视节目之一{{RefTag |1 = {{cite web |title = 老娘舅 |publisher = 豆瓣电影 |url = http://movie.douban.com/subject/4074185/|accessdate =2012-04-20}}}}。1999年，用于杂技表演的[[上海马戏城|上海马戏城]]落成{{RefTag |1 = {{cite web|title =上海欢乐马戏|publisher =上海马戏城官网|url =http://www.happycircus.cn/index.aspx|accessdate =2013-07-27|deadurl =yes|archiveurl =https://web.archive.org/web/20131125021431/http://www.happycircus.cn/index.aspx|archivedate =2013-11-25}}}}，并长期上演《时空之旅》节目。此外，以[[浦东说书|浦东说书]]、[[松江锣鼓书|松江锣鼓书]]等地区性曲艺形式通过今年来的发展和传承，逐渐得到重视，并于2008年入选国家第二批非物质文化遗产名录{{RefTag |1 = {{cite web |title = 国务院关于公布第二批国家级非物质文化遗产名录和第一批国家级非物质文化遗产扩展项目名录的通知 |publisher = 中国政府网 |date = 2008-06-14 |url = http://www.gov.cn/zwgk/2008-06/14/content_1016331.htm |accessdate =2012-04-20}}}}。

=== 现代戏剧 ===
[[File:Shanghai_opera.jpg|thumb]]是上海重要的藝術表演中心]]

19世纪末，[[话剧|话剧]]在上海的[[中国教会学校列表|教会学校]]中出现，当时主要以[[英语|英语]]演出西方名剧的片段。1899年上演的《官场丑史》成为有记录的最早的一出剧目{{RefTag |1 = {{cite web|title=剧变沧桑：第1集 舞台西洋风 |publisher=文明网 |date=2009-02-21 |url=http://www.godpp.gov.cn/wmzh/2008-02/21/content_12509540_2.htm |accessdate=2012-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20170211044131/http://www.godpp.gov.cn/wmzh/2008-02/21/content_12509540_2.htm |archivedate=2017-02-11 }}}}。1907年，《[[黑奴吁天录|黑奴吁天录]]》在[[兰心大戏院|兰心大戏院]]公演，成为早期话剧形成的标志。[[新文化运动|新文化运动]]后，青年学生和知识分子多用话剧来表达观点。进入1920年代以后，开始出现了《麻雀与小孩》等歌舞剧{{RefTag |1 = {{cite web|title=黎锦晖儿童歌舞剧《麻雀与小孩》与《小小画家》的差异性比较 |publisher=博景源论文网 |date=2007-03-24 |url=http://www.86qb.com/Article/yishuxue/yinyue/2663.html |accessdate=2012-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20120117040549/http://www.86qb.com/Article/yishuxue/yinyue/2663.html |archivedate=2012-01-17 }}}}。

1927年成立的[[上海音乐学院|国立音乐院]]{{RefTag |1 = {{cite web|title=学院简介 |publisher=上海音乐学院 |url=http://www.shcmusic.edu.cn/html/intro/xxjj.htm |accessdate=2012-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20120303125530/http://www.shcmusic.edu.cn/html/intro/xxjj.htm |archivedate=2012-03-03 }}}}和1950年代陆续成立的上海人民艺术剧院、上海歌剧院和[[上海戏剧学院|上海戏剧学院]]{{RefTag |1 = {{cite web |title = 学院简介 |publisher = 上海戏剧学院 |url = http://www.sta.edu.cn/content.aspx?id=17032 |accessdate =2012-04-20}}}}，现为上海现代戏剧的演出机构和专业培训机构。话剧以上海话剧艺术中心为主体，而音乐剧和歌剧的演出则主要借助于国际演出团队。[[上海音乐厅|上海音乐厅]]和[[上海大剧院|上海大剧院]]也分别进行了改造和兴建，之后上演了[[百老汇|百老汇]]的经典音乐剧《[[悲惨世界|悲惨世界]]》、《[[猫_(音乐剧)|猫]]》{{RefTag |1 = {{cite web |title = 经典音乐剧《猫》3月28日大剧院首演 |publisher = 新浪上海 |date = 2003-02-25 |url = http://sh.sina.com.cn/local/action/20030225/11446190.shtml |accessdate =2012-04-20}}}}，以及[[普契尼|普契尼]]的经典歌剧《[[波西米亚人_(歌剧)|波西米亚人]]》、《[[图兰朵|图兰朵]]》{{RefTag |1 = {{cite web|title =从《阿蒂拉》上演看意大利歌剧在上海|publisher =文汇报|date =2013-10-28|url =http://comment.whb.cn/redian/view/35322|accessdate =2013-11-20|deadurl =yes|archiveurl =https://web.archive.org/web/20171007172512/http://comment.whb.cn/redian/view/35322|archivedate =2017年10月7日|df =}}}}等。

如今的上海已成为众多话剧、舞台剧的背景城市。例如中文话剧大师[[赖声川|赖声川]]创作执导的《[[暗恋桃花源|暗恋桃花源]]》，李逸导演的《[[上海滩_(1980年电视剧)|上海滩]]》，上海话剧艺术中心演出出品的大型话剧《大哥》，带有怪诞色彩的[[黑色喜剧|黑色喜剧]]《秀才与刽子手》等等。众多充满海派风情的话剧作品纷纷在包括[[上海大剧院|上海大剧院]]、[[东方艺术中心|东方艺术中心]]、[[上海人民大舞台|上海人民大舞台]]、[[艺海剧院|艺海剧院]]、[[上海话剧艺术中心|上海话剧艺术中心]]、[[兰心大戏院|兰心大戏院]]等在内的上海各大表演中心上演，其中亦有不少精彩的精品走出国门，在各国巡回演出，获得好评无数{{RefTag |1 = {{cite web |title = 海派话剧首次大规模晋京，有3部原创剧和6部欧美编译剧 |publisher = 上观新闻 |date = 2017-05-17 |url = http://www.jfdaily.com/news/detail?id=53478 |accessdate= }}}}。

=== 音乐 ===
[[File:Concert_hall_shanghai.jpg|right]]]]

上海的早期音乐以传统的[[江南丝竹|江南丝竹]]为主，[[上海开埠|上海开埠]]后，[[西洋音乐|西洋音乐]]传入。1878年，上海管乐协会成立[[管乐|管乐]]队{{RefTag |1 = {{cite web|title=近代西方音乐传媒--上海工部局乐队 |publisher=上海档案信息网 |date=2008-04-08 |url=http://www.archives.sh.cn/docs/200804/d_187665.html |accessdate=2012-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20111109104823/http://www.archives.sh.cn/docs/200804/d_187665.html |archivedate=2011-11-09 }}}}，即今[[上海交响乐团|上海交响乐团]]的前身。进入1930年代，上海流行音乐进入繁荣时期。《[[夜上海|夜上海]]》、《苏州河畔》、《[[夜来香|夜来香]]》、《[[何日君再来|何日君再来]]》、《[[玫瑰玫瑰我爱你|玫瑰玫瑰我爱你]]》、《[[如果没有你|如果没有你]]》、《恭喜恭喜》、《明月千里寄相思》等，均创作并流行于这一时期。除流行音乐以外，鉴于当时局势，上海的大部分音乐人也兼创作爱国歌曲，出现了《[[毕业歌|毕业歌]]》、《[[大刀进行曲|大刀进行曲]]》等作品。现在用作中国国歌的《[[义勇军进行曲|义勇军进行曲]]》也是在由知名音乐家[[聂耳|聂耳]]创作于上海。

1949年以后，上海的[[流行音乐|流行音乐]]以及电影[[原声音乐|原声音乐]]为[[红色歌曲|红色歌曲]]所取代，少有新作品诞生，这一现象一直持续到“[[文化大革命|文化大革命]]”结束。至1978年，“上海之春”音乐舞蹈节恢复举行，即后来的“上海之春”国际音乐节。随着音乐教育的恢复，上海逐渐出现了[[孔祥东|孔祥东]]、[[黄英_(歌唱家)|黄英]]、[[廖昌永|廖昌永]]等著名艺术家。为普及高雅音乐，[[上海人民广播电台|上海人民广播电台]]创立[[星期广播音乐会|星期广播音乐会]]，并持续至今。[[帕尔曼|帕尔曼]]、[[小泽征尔|小泽征尔]]、[[马友友|马友友]]、[[柏林爱乐乐团|柏林爱乐乐团]]、[[纽约爱乐乐团|纽约爱乐乐团]]等也曾在[[上海音乐厅|上海音乐厅]]、[[上海大剧院|上海大剧院]]和[[东方艺术中心|东方艺术中心]]进行交流演出。自2012年4月起，上海首届[[户外交响音乐会|户外交响音乐会]]——辰山草地广播音乐会落户[[佘山|佘山]]附近的[[上海辰山植物园|上海辰山植物园]]绿色剧场，至今每年春夏之际举办一届。辰山草地广播音乐会推崇“赏花赏乐享人生”的理念，邀请世界知名乐团参演，品质不俗{{RefTag |1 = {{cite web |title = 辰山草地广播音乐会 五大版块重温美好回忆 |publisher = 腾讯娱乐 |date = 2013-04-22 |url= http://ent.qq.com/a/20130422/000410.htm|accessdate =2013-04-23}}}}。

流行音乐方面，上海本地的相关制作和乐团在1970年代以后开始逐渐复苏。1993年，为鼓励上海和中国的流行音乐的发展，上海东方广播电台设立专业的流行乐榜单，即[[东方风云榜|东方风云榜]]。而本土的流行乐也培养[[毛阿敏|毛阿敏]]、[[胡彦斌|胡彦斌]]等明星。一些国际流行乐坛的著名歌手和乐队如[[惠特尼·休斯敦|惠特尼·休斯敦]]、[[席琳·狄翁|席琳·狄翁]]、[[瑪麗亞·凱莉|瑪麗亞·凱莉]]{{RefTag |1 = {{cite web |title = 席琳-迪翁引爆上海 《我心依旧》压轴演唱会 |publisher = 新浪娱乐 |date = 2008-04-11 |url = http://ent.sina.com.cn/y/2008-04-12/09591984176.shtml|accessdate =2012-04-20}}}}、[[林肯公園|林肯公園]]、[[比吉斯|比吉斯]]{{RefTag |1 = {{cite web |title = Bee Gees上海演唱会专题 |publisher =  新浪娱乐 |url = http://ent.sina.com.cn/f/y/BeeGees/index.shtml |accessdate = 2012-04-20 }}}}、[[海莉·薇思特拉|海莉·薇思特拉]]{{RefTag |1 = {{cite web|title =2010上海大剧院世博演出季闭幕演出新生代跨界美声——海莉·薇思特拉独唱音乐会|publisher =上海大剧院|url =http://www.shgtheatre.com/ec/searchAllProgram.do|accessdate =2012-04-20|deadurl =yes|archiveurl =https://web.archive.org/web/20120415235914/http://www.shgtheatre.com/ec/searchAllProgram.do|archivedate =2012-04-15}}}}、[[滨崎步|滨崎步]]{{RefTag |1 = {{cite web |title = 滨崎步上海演唱会专题 |publisher = 新浪娱乐 |date = 2008 |url = http://ent.sina.com.cn/f/y/Ayumi2008/index.shtml|accessdate =2012-04-20 }}}}、[[滚石乐队|滚石乐队]]、[[東方神起|東方神起]]、[[碧昂斯|碧昂斯]]、[[泰勒·斯威夫特|泰勒·斯威夫特]]{{RefTag |1 = {{cite web |title = 欧美歌手来华演唱会时间表 霉霉上海连唱三场 缪斯乐队首来内地 |publisher = Mtime时光网|url = http://news.mtime.com/2015/07/08/1544406.html|date =2015-07-08}}}}、[[約翰·傳奇|約翰·傳奇]]等都在上海举办过演唱会。

=== 电影與电视剧 ===

1895年8月，上海出现了名为西洋影戏的电影。1909年，亚细亚影戏公司成立，为中国最早的电影公司。在1921年至1931年间，上海各类电影公司摄制的故事片有650余部。随着电影业的迅速发展，[[阮玲玉|阮玲玉]]、[[胡蝶|胡蝶]]、[[周璇|周璇]]、[[赵丹|赵丹]]、[[上官云珠|上官云珠]]、黎明晖、[[王人美|王人美]]等成了当时中国家喻户晓的明星。大量私营电影则1950年代被整关闭。上海的电影业界先主要由国有的[[上海电影制片厂|上海电影制片厂]]、[[上海美术电影制片厂|上海美术电影制片厂]]{{RefTag |1 = {{cite web|title=中国动画的摇篮--上海美术电影制片厂介绍 |publisher=上海美术电影制片厂 |url=http://www.ani-sh.com/08SAFS/html/aboutus.html |accessdate=2012-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20110808193905/http://www.ani-sh.com/08SAFS/html/aboutus.html |archivedate=2011-08-08 }}}}、[[上海电影译制厂|上海电影译制厂]]{{RefTag |1 = {{cite web |title = 上海电影译制片厂50周年：永不消失的声音 |publisher = 新浪网 |date = 2007-06-26 |url = http://news.sina.com.cn/c/2007-06-26/095113312463.shtml |accessdate = 2012-04-20}}}}等构成，并于2003年组建成[[上海电影集团|上海电影集团]]。1949年后诸多著名革命题材电影如《[[南征北战_(电影)|南征北战]]》《[[红色娘子军|红色娘子军]]》《[[鸡毛信|鸡毛信]]》等为上海电影制片厂所拍摄，而成立于1957年上海美术电影制片厂是中国规模最大的美术电影制片基地，创作了《[[小蝌蚪找妈妈|小蝌蚪找妈妈]]》《[[大闹天宫|大闹天宫]]》《[[哪吒闹海|哪吒闹海]]》《[[天书奇谭|天书奇谭]]》《[[三个和尚|三个和尚]]》《[[宝莲灯|宝莲灯]]》等经典作品。

随着全球化的发展和海内外的电影产业进一步崛起，越来越多的中外电影包括《[[变形金刚2|变形金刚2]]》、《[[太平轮|太平轮]]》、《[[碟中谍3|碟中谍3]]》、《[[小时代_(电影)|小时代]]》、《[[谍海风云|谍海风云]]》、《[[神奇四侠2|神奇四侠2]]》、《[[如果·爱|如果·爱]]》、《[[面纱_(电影)|面纱]]》、《[[七月与安生_(电影)|七月与安生]]》、《[[黃金時代_(2014年電影)|黄金时代]]》、《[[十月围城_(电影)|十月围城]]》和《[[007：大破天幕杀机|007：大破天幕杀机]]》等等作品纷纷前往上海取景拍摄。

[[上海國際電影節|上海國際電影節]]（英語：Shanghai International Film Festival，英文簡稱為SIFF）是中國國內第一個世界公認的[[國際電影節|國際電影節]]。每年6月在上海舉行。電影節在1993年首次舉辦，1994年獲得[[國際電影製片人協會|國際電影製片人協會]]承認成為國際A類電影節。上海國際電影節共分為4個主要部分，包括競賽部分[[金爵獎|金爵獎]]、國際電影展覽放映、國際電影交易市場，及金爵國際電影論壇暨亞洲新人獎評選。2004年增設傳媒大獎。這亦成為了上海電影與[[華語電影|華語電影]]發展的一大重要成就。

[[电视剧|电视剧]]方面，上海也有许多反映城市特色的知名作品。比如[[上海电视台|上海电视台]]与[[中央电视台|中央电视台]]合拍的《[[十六岁的花季|十六岁的花季]]》以上海多所中学为场景，反映了改革开放后成长起来的中学生的生活与思想，在全国各地电视重播了十余年。《[[孽债|孽债]]》则直指文革上山下乡运动的余留问题，真实地记录了90年代初期上海的城市和社会面貌，同时还有沪语版本，因而在成片十余年后重播仍然受到广泛关注。

=== 時裝周 ===

上海时装周于創始於2001年，于每年的10月份在上海市區舉辦。作为中国與整個[[大中華地區|大中華地區]]原创设计发展推广的最优化的交流平台，上海時裝周一直都致力于通过“立足本土兼备国际视野”和“创意设计与国际接轨”的定位，推动本土时尚产业链和国际一线时尚领域接轨，從而历年吸引了众多优秀的來自中國、[[日本|日本]]、[[美國|美國]]、[[法國|法國]]、[[意大利|意大利]]、[[瑞典|瑞典]]等地的自主設計品牌，尤其是每届的时装周主秀场的首场秀演，都由本土原创品牌担纲{{RefTag |1 = {{cite web |title = 2016上海春夏時裝周 |publisher = 新浪網 |url = http://fashion.sina.com.cn/z/s/2016ssSHFW/ |accessdate =2016-11-26}}}}。

=== 文化活动 ===

上海每年定期举办文化活动。[[上海电视节|上海电视节]]创办于1986年，是中国最早的国际电视节{{RefTag |1 = {{cite web |title = 上海电视节 |publisher = 上海电视节 |url = http://www.stvf.com |accessdate =2012-04-20}}}}，创办于1993年的[[上海国际电影节|上海国际电影节]]在海内外也较为知名{{RefTag |1 = {{cite web|title=上海国际电影节 |publisher=央视国际 |date=2008-06-24 |url=http://city.cctv.com/html/yingxiangchengshi/e9852b4b073076258ef2635ca6881dff.html |accessdate=2012-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20090519075526/http://city.cctv.com/html/yingxiangchengshi/e9852b4b073076258ef2635ca6881dff.html |archivedate=2009-05-19 }}}}。[[上海旅游节|上海旅游节]]创办于1990年，为每年9月至10月的文化节日，主要进行国际旅游推广活动。上海国际艺术节期间会举行多场画展、音乐会、艺术展，并聘请各国艺术家登台献艺。每年8月中旬会在[[上海展览中心|上海展览中心]]举行[[上海书展|上海书展]]，而[[上海国际音乐烟花节|上海国际音乐烟花节]]通常在[[世纪公园|世纪公园]]举行。

=== 建筑特色 ===
{{Main|上海建筑}}

上海的古代建筑，目前基本遗留的以明代建筑为主，而且大部分为民居和私家园林。除[[官署|官署]]、[[学堂|学堂]]、[[寺庙|寺庙]]外鲜有其他类型的建筑。民居大多为江南风格，砖木立帖式平房，在上海，除[[上海老城厢|老城厢]]以外，在各郊区的乡镇中仍能见到较多的此类建筑。上海的私家园林目前保存较为完整的为[[豫园|豫园]]，而古代居民区保存较为完整的有[[闵行区|闵行]][[七宝镇|七宝镇]]、[[青浦|青浦]][[朱家角镇|朱家角镇]]、[[浦东新区|浦东]][[新场镇_(上海市)|新场镇]]等。

上海开埠以后，随着租界建立和欧美文化大举进入，一些西式别墅和洋房开始兴建，上海也开始出现各种不同风格的建筑，因而被誉为“万国建筑博览会”，其中尤以外滩地区的金融建筑最为著名，诸如中式的[[上海中国银行大楼_(外滩)|中国银行原总行大楼]]，如折衷主义风格的[[江海关|江海关]]，新古典主义风格的[[汇丰银行大楼_(外滩)|汇丰银行上海分行大楼]]等，采用美国装饰艺术运动摩天楼风格的[[沙逊大厦|沙逊大厦]]等，现大量作为[[上海市优秀历史建筑|上海市优秀历史建筑]]予以保护。因为人口大量增加，这个阶段也产生了极富上海特色[[石库门|石库门]]住宅，直至1980年代一直是上海最常见的住宅样式{{RefTag |1 = {{cite web |title = 石库门，走过百年继续守望上海|publisher = 新民晚报 |date = 2010-11-21 |url =http://xmwb.xinmin.cn/history/xmwb/html/2010-11/21/content_593870.htm |accessdate = 2012-04-18}}}}。

进入1990年代后，上海开始广泛进行城市建筑和基础设施改造，新建了大量商务办公楼、大型商场、高层住宅等现代风格建筑。至2008年，楼层超过30层的高楼已有2000多栋{{RefTag |1 = {{cite web|title=Skyscraper Cities Ranking List |publisher=The World's Best Skylines |date=2012-04-05 |url=http://homepages.ipact.nl/~egram/skylines.html |accessdate=2012-04-05 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20080204231310/http://homepages.ipact.nl/~egram/skylines.html |archivedate=2008-02-04 }}}}，其中尤以而浦东[[陆家嘴金融贸易区|陆家嘴]]地区为代表，[[东方明珠电视塔|东方明珠电视塔]]、[[金茂大厦|金茂大厦]]、[[上海环球金融中心|上海环球金融中心]]等已成为上海地标，2014年底完工632米高[[上海中心大厦|上海中心大厦]]是上海已建成最高的建筑{{RefTag |1 = {{cite web |title = 中国第一高楼上海中心大厦完工 总高度632米|publisher = 新华网 |date= 2016-03-16 |url= http://news.sina.com.cn/c/nd/2016-03-16/doc-ifxqhmve9257487.shtml |accessdate = 2016-04-18}}}}。

== 旅游 ==

上海在租界时期遗留了众多富有特色的欧式建筑，而1990年代起又出现了诸多现代化建筑，同时，周边郊区分布着几处具有江南风格的[[江南水乡古镇|水乡古镇]]，均成为了旅游景观。上海于1986年被国务院定为[[国家历史文化名城|国家历史文化名城]]。丰富的旅游资源吸引了各国游客，到2012年，接待入境的国际旅游者达到800万人次{{RefTag |1 = {{cite web |title =2012年上海：每100元GDP，旅游业贡献7.5元 |publisher =中国经济网 |date = 2013年5月24日 |url = http://www.ce.cn/macro/more/201305/24/t20130524_24414499.shtml |accessdate = 2013-06-05}}}}。

上海的[[东方明珠广播电视塔|东方明珠广播电视塔]]、[[上海野生动物园|上海野生动物园]]、[[上海科技馆|上海科技馆]]3处属中国最高的[[5A景区|5A级旅游景点]]{{RefTag |1 = {{cite web |title = 上海科技馆成为5A级科普旅游景区 |publisher = 新华上海 |date = 2010-03-18 |url = http://www.sh.xinhuanet.com/2010-03/18/content_19285986.htm |accessdate = 2011-03-11 |deadurl = yes |archiveurl = https://web.archive.org/web/20111109103736/http://www.sh.xinhuanet.com/2010-03/18/content_19285986.htm |archivedate = 2011-11-09 }}}}，另有20处[[中国国家4A级旅游景区|4A级景区]]，[[佘山|佘山]]、[[东平国家森林公园|东平]]、[[上海共青国家森林公园|共青]]、[[上海海湾国家森林公园|海湾]]等4处[[国家森林公园|国家森林公园]]，以及[[沙逊大厦|沙逊大厦]]、[[外白渡桥|外白渡桥]]等617处[[上海市优秀历史建筑|上海市优秀历史建筑]]。

上海近年来已成为驰名中外的国际[[夜景|夜景]]都市。浦西的外滩夜景以及浦东陆家嘴的摩天高楼夜景相隔黄浦江而分布，吸引着成千上万的游客前来观光。而南京路商圈、徐家汇商圈、新天地甚至是[[宝山区_(上海市)|宝山]]等地区等地段也有着五光十色、十分绚丽的夜景建设{{RefTag |1 = {{cite web |title =优美画册展现宝山夜景建设成果 |publisher =上海市绿化和市容管理局 |date = 2015-07-30 |url = http://lhsr.sh.gov.cn/sites/wuzhangai_lhsr/neirong.aspx?ctgid=9d99a135-2438-4e21-bbd4-34a52407f5ca&infid=1b3fdb3d-5eea-43ea-ac4f-09434c2cc602 |accessdate = 2016-04-14}}}}。

上海每年举办的文化推广活动也推动了上海旅游业的发展，包括[[上海國際電影節|上海國際電影節]]、国际音乐节等。每年9月的[[上海旅游节|上海旅游节]]则是展示上海和吸引游客的一个重要舞台和途径。[[网球|网球]][[上海大师赛|上海大师赛]]和[[一级方程式赛车|一级方程式赛车]][[中国大奖赛|中国站]]也成为一些游客到上海旅游的目的。2010年5月至10月间，[[上海世界博览会|上海世界博览会]]的参观人数是达到了7300余万人次{{RefTag |1 = {{cite web |title = 外国媒体盛赞上海世博会 |publisher = 中国青年报 |date = 2010-11-01 |url = http://zqb.cyol.com/content/2010-11/01/content_3436101.htm |accessdate = 2011-03-14}}}}。

== 著名人物 ==
{{Main|上海人}}

[[上海开埠|上海开埠]]后，各地的精英移民来到上海。上海有“'''冒险家的乐园'''”之称，很多近代的上海名人并不出生于上海，如[[鲁迅|鲁迅]]（绍兴）、[[周璇|周璇]]（常州），但上海成功吸引这些移民和上海本地居民，并共同成就了上海的都市辉煌。

上海著名的政治與軍事人物有明朝内阁首辅[[徐阶|徐阶]]、內閣次輔[[徐光啟|徐光啟]]，近代的[[梁啟超|梁啟超]]、[[孫中山|孫中山]]、[[宋慶齡|宋慶齡]]、[[宋美齡|宋美齡]]、[[宋靄齡|宋靄齡]]、[[厉麟似|厲麟似]]、[[陳果夫|陳果夫]]等人也曾長居上海。上海籍的科学界著名人士有[[諾貝爾獎|諾貝爾獎]]得主[[李政道|李政道]]、[[高锟|高锟]]，[[生物學家|生物學家]][[谈家桢|谈家桢]]、[[陳竺|陳竺]]等人。文學界則人才輩出，包括了[[陶行知|陶行知]]、[[郭沫若|郭沫若]]、[[傅雷|傅雷]]、[[陆谷孙|陆谷孙]]、[[曹禺|曹禺]]、[[胡适|胡适]]、[[邹韬奋|邹韬奋]]、[[黄炎培|黄炎培]]、[[厉声教|厲聲教]]、[[季羡林|季羡林]]、[[马寅初|马寅初]]、[[張愛玲|張愛玲]]、[[梁实秋|梁实秋]]等包括了上海本地出生與外來而在上海定居過的眾多名人。藝術文化界的上海名人有內地著名演員[[章金莱|章金莱]]、[[胡歌|胡歌]]、[[孫儷|孫儷]]、[[黃聖依|黃聖依]]、[[陸毅|陸毅]]、[[李俊鋒|李俊鋒]]、[[鄭愷|鄭愷]]、[[唐嫣|唐嫣]]、[[杨洋_(男演员)|楊洋]]、[[王梓薇|王梓薇]]、[[喬任梁|喬任梁]]以及出生於上海的香港電影演員[[李修賢|李修賢]]、[[張敏_(演員)|張敏]]等人，亦有流行歌手[[薛之謙|薛之謙]]、[[尚雯婕|尚雯婕]]等人。來自上海的的体育界人士有110米栏原世界纪录保持者[[刘翔|刘翔]]、籃球運動員[[姚明|姚明]]、足球運動員[[范志毅|范志毅]]、[[孙雯|孙雯]]、跳水運動員[[吳敏霞|吳敏霞]]、圍棋冠軍[[常昊|常昊]]、赛车手[[韩寒|韩寒]]等人{{RefTag |1 ={{cite web |url = http://travel.cntv.cn/2013/04/15/ARTI1365990838608961.shtml|title = 上海人为何总被误读？其实你不懂魔都的心|publisher = 《环球人文地理》|date = 2013年4月15日|accessdate = 2014-06-14}}}}。
