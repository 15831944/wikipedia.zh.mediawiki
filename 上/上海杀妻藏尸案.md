'''上海杀妻藏尸案'''是2016年10月17日上午发生在[[中国|中国]][[上海|上海]]的一起杀人藏尸案，凶手'''朱晓东'''因为口角杀死自己的妻子'''杨丽萍'''，并藏尸冰柜105天<ref>{{cite web |title=殺妻藏屍105天 上海狠男判死刑 |url=https://www.worldjournal.com/5833758/article-%E6%AE%BA%E5%A6%BB%E8%97%8F%E5%B1%8D105%E5%A4%A9-%E4%B8%8A%E6%B5%B7%E7%8B%A0%E7%94%B7%E5%88%A4%E6%AD%BB%E5%88%91/ |publisher=世界日报 |accessdate=2018-9-29}}</ref><ref>{{cite news |title=「殺妻藏冰櫃105天」還帶女開房　恐怖夫自首仍判死|url=https://www.ettoday.net/news/20180823/1242015.htm |accessdate=2018-9-29 |work=ETtoday新聞雲}}</ref>。此案是在[[中华人民共和国司法体制|中国现行司法体系]]下，夫妻之间发生的故意杀人案，凶手有法定从轻处罚情节{{efn|[[自首|自首]]即法定从轻处罚情节。《[[中华人民共和国刑法|中华人民共和国刑法]]》规定，犯罪嫌疑人自首的，可以从轻或减轻处罚<ref name="a1"/>。}}，但却罕见的宣判最高刑罚——[[死刑|死刑]]的案件<ref name="a1">{{Cite web |url=http://dy.163.com/v2/article/detail/DPVTOA2K0519S96U.html |title=上海杀妻藏尸案判决死刑立即执行，为何南京吉星鹏杀妻案判死缓？|accessdate=2018-11-28 |author= |date=2018-08-24 |work= |publisher=网易|language=Zh-hans }}</ref>。

[[中華人民共和國死刑制度|中国司法机关对死刑]]实行“保留死刑，严格控制和慎重使用死刑”的政策<ref name="a1"/>，即所谓“少杀、慎杀”原则<ref name="a2">{{Cite web |url=http://zqb.cyol.com/html/2011-08/05/nw.D110000zgqnb_20110805_2-02.htm |title=“少杀慎杀”的前提是“当判必判”|accessdate=2018-11-28 |author=邓海建  |date=|work= |publisher=中青在线，来源：《中国青年报》（2011年08月05日02版）|language=Zh-hans }}</ref>。1999年，[[中华人民共和国最高人民法院|最高人民法院]]《全国法院维护农村稳定刑事审判工作座谈会纪要》明确指出：“对于因婚姻家庭、邻里纠纷等民间矛盾激化引发的故意杀人犯罪，适用死刑一定要十分慎重，应当与发生在社会上的严重危害社会治安的其他故意杀人犯罪案件有所区别。对于被害人一方有明显过错或对矛盾激化负有直接责任，或者被告人有法定从轻处罚情节的，一般不应判处死刑立即执行<ref name="a1"/>。”在此案发生前，全国各地都有性质更为恶劣、无法定从轻处罚情节的杀妻、杀女友凶犯，仅判处[[死缓|死缓]]的案例<ref name="a1"/><ref name="a2"/>。

==当事人==

*受害人(妻子)：杨丽萍
**2013年9月，杨俪萍与朱晓东确立了恋爱关系。2015年12月31日与朱登记结婚<ref>{{cite news |title=上海“杀妻藏尸案”一审宣判：凶手朱晓东被判死刑 |url=http://news.ifeng.com/a/20180823/59949209_0.shtml?_cpb_shehui |work=凤凰资讯}}</ref>。
*凶手(丈夫)：朱晓东
**30岁，上海本地人，商场店员 。

==案件经过==
===杀人===
2016年10月15日，朱、杨二人前往[[杭州|杭州]]旅游，期间发生争吵。10月17日早上7时许，二人再度发生争执。朱晓东怒而掐死了妻子杨丽萍<ref name="a3">{{cite web |title=震惊丨奶油小生出轨杀妻，透支妻子信用卡与其他女性开房 |url=http://news.sina.com.cn/o/2017-12-13/doc-ifypnyqi4517604.shtml |accessdate=2018-11-28 |author= |date=2017-12-13 |work= |publisher=网易新闻|language=Zh-hans}}</ref>。

===藏尸===
杀死杨丽萍后，朱晓东将杨的尸体装进一个被套里，并放进冰柜<ref name="a3"/>。

===案发===
杀人藏尸后，朱对家人朋友隐瞒了杨的死讯。并且案发后，用杨的手机，以杨的名义通过[[微信|微信]]和他人联系。试图让他人觉得杨还活着<ref>{{cite web |title=杀妻藏尸案嫌犯供述：勒她只是不想让她再唠叨 |url=http://news.sina.com.cn/s/wh/2017-11-29/doc-ifyphtze2362968.shtml |publisher=法制晚报}}</ref>。案发前，朱把杨手机里的钱转给自己，杨遇害后一个星期，朱便带着个女人去了[[韩国|韩国]]旅行。并且和其他女人开房。2017年2月1日，杨丽萍的父亲[[六十花甲|六十大寿]]，按照约定杨丽萍会回去为父亲祝寿。朱明白自己杀人的事瞒不住了。朱在其父母陪同下前往派出所[[自首|自首]]。<ref>{{cite news |title=上海“杀妻藏尸案”一审宣判：凶手朱晓东被判死刑 |url=http://news.ifeng.com/a/20180823/59949209_0.shtml?_cpb_shehui |work=凤凰资讯}}</ref><ref>{{cite web |title=上海杀妻藏尸案细节:买书模仿作案事后买男性补品 |url=http://news.sina.com.cn/s/2018-08-30/doc-ihiixyeu1471128.shtml |publisher=新浪新闻}}</ref>

===预谋杀妻===
有媒体认为朱是有预谋地杀妻，并不是[[激情犯罪|激情犯罪]]<ref>{{cite web |title=上海杀妻藏尸案细节曝光：凶手买解剖书模仿作案事后买男性补品 |url=http://365jia.cn/news/2018-08-31/6A6853FC52AE2489_all.html}}</ref>。

==审判==
2018年8月23日，[[上海市第二中级人民法院|上海市第二中级人民法院]]一审宣判，朱晓东涉嫌故意杀人罪被判[[死刑|死刑]]，[[剥夺政治权利|剥夺政治权利]]终生。法院认为，此案虽因婚姻家庭矛盾引发，且朱晓东自首，但朱晓东犯罪性质恶劣，作案后长时间藏匿被害人尸体，无悔罪表现，社会危害极大，罪行极其严重，故依法对其不予从轻处罚。依照《[[中华人民共和国刑法|中华人民共和国刑法]]》第二百三十二条、第五十七条第一款之规定，作出上述判决<ref>{{cite news |title=上海“杀妻藏尸案”一审宣判：凶手朱晓东被判死刑 |url=http://news.ifeng.com/a/20180823/59949209_0.shtml?_cpb_shehui |work=|accessdate=2018-11-28 |author= |date=2018-08-23 |work=责任编辑：李中琪 PV086 |publisher=凤凰网，来源：澎湃新闻网|language=Zh-hans}}</ref>。
