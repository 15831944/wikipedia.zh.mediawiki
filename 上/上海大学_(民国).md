'''上海大学'''于1922年由[[国共合作|国共合作]]创办。名义上为[[中国国民党|中国国民党]]党立学校，实际上是[[中国共产党|中国共产党]]主掌的且以中共党员为骨干的第一所用以培养革命分子的教育机构。1927年，以[[蒋介石|蒋介石]]为首的国民党右派发动[[四·一二政变|四·一二政变]]后，强行关闭了该校<ref name="S">{{cite web|url=http://www.shu.edu.cn/xxgk/sdxs2.htm|title=上大校史|publisher=上海大學}}</ref>。前后共存在4年多时间，期间校址曾多次搬迁。

== 历史沿革 ==
*1922年春，[[王理堂|王理堂]]假[[陈独秀|陈独秀]]等人之名，以提倡新文化为号召，在上海闸北青岛路青云坊（今青云路298号附近）<ref>上海市地方志办公室——青云路街道：http://www.shtong.gov.cn/newsite/node2/node4/node2249/zabei/node38978/node39008/node63267/userobject1ai24262.html</ref>，创办[[东南高等师范专科学校|东南高等师范专科学校]]，但因校方借学敛财、校政腐败，学生于10月掀起罢课风潮，要求“改造学校”。
*1921年底，中共創辦了第一所學校，即[[平民女校|平民女校]]，以半工半讀的方式培養搞婦女工作的幹部，校長是黨中央宣傳部長[[李達_(哲學家)|李達]]。在取得成功經驗後，為加速培養更多的幹部，中共總書記陳獨秀決定創辦一所高等學校，他曾與李大釗等人多次醞釀籌劃。此時正值東南高等師範學校事發，黨中央研究後，決定接受師生們的要求，將東南高等師範改建為上海大學。
*1922年10月23日，学校由[[东南高等专科师范学校|东南高等专科师范学校]]改组，邀请-{[[于右任|于右任]]}-出任校长。校舍为老式石库门2层楼房10余间，弄堂门即校门，人称“弄堂大学”。
*1923年8月，学校评议会决定“聘请有名人物充当教授”<ref name="S"/>。
*1924年1月，[[中國國民黨第一次全国代表大会|中國國民黨第一次全国代表大会]]决议，每月补助上大1000银元为办学经费。2月，學生從160人增加到400人，学校迁至租界西摩路132号（今陕西北路342弄）及对面299弄时恩里4～12号民房继续办学。
*1925年，上海大学成为[[五卅运动|五卅运动]]的先锋队，5月30日学生[[何秉彝|何秉彝]]在老闸巡捕房门口被枪杀。6月4日，学校被英国军队占领、查封。7月，迁回闸北中兴路，设临时办事处继续招生，后租闸北青云路师寿坊（今青云路167弄位置）15幢民房为校舍，于9月10日开学上课。弄堂口挂有於右任所书“上海大學臨時校舍”的牌子。當時學生已增加到800人，還附設有平民學校和青雲學校。
*1925年9月7日，[[广州国民政府|广州国民政府]]第十五次会议议决，补助上大校舍建筑经费2万银元。
*1926年9月7日，上海10万人参加[[辛丑条约|“九七”国耻]]纪念大会，追悼各地死难烈士，上大许多学生参加并组织领导。担任全国学总委员的[[李硕勋|李硕勋]]主持大会，[[林钧|林钧]]主祭。上大师生[[周水平|周水平]]、[[刘华_(工運领袖)|刘华]]、[[贺威圣|贺威圣]]先后被军阀政府杀害。
*1927年2月，上海工人在中共指揮下發動第二次武裝暴動，2月19日工人罷工後，學生罷課支援。<ref name="上海青年志">[http://www.shtong.gov.cn/newsite/node2/node2245/node66268/node66277/node66326/node66387/userobject1ai62314.html 第二節　反對北洋政府和國民黨政權統治的青年運動] {{webarchive|url=https://web.archive.org/web/20140201221119/http://www.shtong.gov.cn/newsite/node2/node2245/node66268/node66277/node66326/node66387/userobject1ai62314.html |date=2014-02-01 }}</ref>2月23日，直系军阀当局搜查上海大学，留校学生50余人被捕。
*1927年3月，上海工人在中共指揮下發動第三次武裝暴動，上海大學的學生糾察隊直接參加了閘北區的暴動，攻下警察署。<ref name="上海青年志"/>
*1927年4月1日，通过募捐和借款的辦法，新校舍在江灣鎮西端奎照路落成，新学期开学。
*1927年5月2日，国民党军警借口搜查军械，闯入上大逮捕学生。5月3日，[[蔣中正|蔣中正]]指令淞滬警備司令[[楊虎|楊虎]]和[[陈群_(民国)|陳群]]查封上海大学。上大被國民政府军警單位取缔封闭而终止。部分师生转入[[国立武昌中山大学|国立武昌中山大学]]。並把上海大學的新校址改成了[[國立勞動大學|國立勞動大學]]。校舍于一二八日军入侵上海时被毁。

== 办学影响 ==
瞿秋白主持创办的社會學系當時是中國高等教育的首創。陳望道的《修辭學發凡》、鄧中夏的《中國勞動問題》、肖楚女的《中國農民問題》、施存統的《社會思想史》、安體誠的《現代經濟學》、瞿秋白的《社會科學概論》、蔡和森的《社會發展史》等，都是在講稿的基礎上由上海書店和民智書店發行的。

上大成立时，正值孙中山于广州蒙难脱险，留驻上海，筹划改组国民党，培养革命人才，对上大甚为关注。他希望上大办成“以贯彻吾党之主张，而尽言论之职责”的革命学校。上大集中众多共产党员，成为中国共产党早期在上海一个重要活动据点。中共和青年团组织活动中心内容是发动师生参加反帝、反军阀的国民革命运动。为此，上大被人称之为“著名的党化学校”。[[四·一二事件|四·一二事件]]发生后，[[帝國主義|帝國主義]]和[[國民黨|國民黨]]称“上海大學是赤色大本營”<ref name="L">{{cite web|url=http://www.zgdsw.org.cn/BIG5/n1/2018/0704/c222139-30125509.html|title=紅色學府上海大學的澎湃往事|publisher=中國共產黨歷史網|author=吳基民|date=2018-07-04}}</ref>。

上大师生参加了上海工人三次武装起义。原上大学生时任中共闸北部委书记兼军委书记[[郭伯和|郭伯和]]，任[[上海工人第三次武装起义|上海工人第三次武装起义]]闸北总指挥。[[顾作霖|顾作霖]]任杨树浦和沪东区指挥，[[林钧|林钧]]任南市分指挥，[[刘荣简|刘荣简]]、[[赵希仁|赵希仁]]、[[沈仲宇|沈仲宇]]等各区团部委书记协助指挥。上大师生参加闸北、南市、沪东、沪西四个地区起义战斗，后第三次武装起义取得胜利。

上大因其历史功绩和地位，被称为“五四运动有北大，大革命时期有上大”、“北有北大，南有上大”、“武有黄埔，文有上大”而跻名当时的中国。培育出许多重要的中共、国民党，乃至日後[[台灣共產黨|台灣共產黨]]的干部。上大学生谢雪红、林木顺等人于1928年4月15日在上海成立了台湾共产党。

==遗址==
[[File:上海大学旧址铭牌.jpg|上海大学旧址铭牌，位于[[陕西北路]]299弄原址|thumb|300px]]
1980年，陕西北路299弄时恩里4～12号（今陕西北路南阳路口附近）上海大学旧址被列为[[上海市文物保护单位|上海市文物保护单位]]。但在1995年,因土地批租，原建筑已被拆除，仅存一块旧址铭牌。
