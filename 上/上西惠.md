'''上西惠'''（{{lang|ja|上西恵}}，{{bd|1995年|3月18日|catIdx=Jiyounishi Kei}}），[[滋賀縣|滋賀縣]]出身，為日本女子團體[[NMB48|NMB48]]的前成員，曾經是Team N的主力成員。前經紀公司為[[京樂吉本控股|京樂吉本]]。

==經歷==
; 2010年
* 9月20日 - 報名參加『NMB48一期生徵選會』，合格（參加總數7,256名、最終合格者26名）。 
* 10月9日 - 於[[AKB48|AKB48]]的演唱會『東京秋祭』中和其他的25名合格者一同首次登場，[[NMB48|NMB48]]正式出道。 
; 2011年
* 1月1日 - 於大阪市難波的NMB劇場落成、『NMB48 研究生 1st Stage「為了誰」』公演開始、並且以選拔成員16人之中的一人的身分劇場出道。
* 3月10日 - Team N結成，並在NMB48第一期研究生25名之中選出了16名做為N組的成員，上西亦是其中一員。
; 2012年
: 10月4日、[[讀賣電視台|讀賣電視台]]『[[毒_ポイズン#テレビドラマ|赤川次郎原作 毒<ポイズン>]]』中，初次出演電視劇。而且，是NMB48中第1位以女演員身分出道的成員。
; 2016年
* 6月28的NMB48 Team TeamBⅡ公演上，正式披露了NMB48第五期研究生，其中上西惠的妹妹上西怜也名列其中，是[[NMB48|NMB48]]中首對姊妹在籍的成員<ref>{{cite web |url=http://woman.infoseek.co.jp/news/entertainment/20160628hochi124 |script-title=ja:ＮＭＢ４８に姉妹メンバー誕生！上西恵の妹・怜が加入 |publisher=樂天WOMAN |language=ja |date=2016-06-28 |accessdate=2016-06-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20160701165644/http://woman.infoseek.co.jp/news/entertainment/20160628hochi124 |archivedate=2016-07-01 }}</ref>。
* 10月18日，於NMB48六週年演唱會上的組閣中，劇場經理[[金子剛|金子剛]]宣布上西將自NMB48畢業<ref>{{cite web|url=http://news.nmb48.com/etc/20161018/210615.html |script-title=ja:NMB48 6th Anniversary Liveにて組閣を発表いたしました|publisher=NMB48官方網站|language=ja |date=2016-10-18 |accessdate=2016-10-18}}</ref><ref>{{cite web|url=http://www.oricon.co.jp/news/2080160/full/ |script-title=ja:NMB上西恵が6周年公演で卒業発表 1期生卒業相次ぐ|publisher=[[oricon|ORICON STYLE]] |language=ja |date=2016-10-18|accessdate=2016-10-18}}</ref>。
