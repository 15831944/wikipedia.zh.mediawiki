'''上海市公安局'''，是负责[[上海市|上海市]]内[[公共安全|公共安全]]、交通秩序的[[上海市人民政府|上海市人民政府]]组成部门，接受上海市人民政府和[[中华人民共和国公安部|中华人民共和国公安部]]的双重领导，设有27个内设机构，机关办公地点为上海市静安区武宁南路128号<ref name="SGAJ">{{cite web |url = http://www.police.sh.cn/shga/gweb/zfgkxx/gkxx_view.jsp?pa=6873a50917f316de730455aa46ea556413f7301ee53f0d120c5ce939cd18ad648fa0f7bef7bf3297 |title = 上海市公安局信息公开：主要职能 |publisher = 上海市公安局 |access-date = 2013-12-19 |archive-url = https://web.archive.org/web/20131219065815/http://www.police.sh.cn/shga/gweb/zfgkxx/gkxx_view.jsp?pa=6873a50917f316de730455aa46ea556413f7301ee53f0d120c5ce939cd18ad648fa0f7bef7bf3297 |archive-date = 2013-12-19 |dead-url = yes }}</ref>。在上海家喻户晓的警察形象'''刑警803'''系[[上海市公安局刑事侦查总队|上海市公安局刑事侦查总队]]昵称，属于上海公安的刑事重案组。

==沿革==
1911年[[辛亥革命|辛亥革命]]后，上海巡警总局改称民政总局，下设南市警务公所和闸北警务公所。

1913年5月，上海商埠巡警局、闸北巡警局合组而成成立[[江苏淞沪警察厅|江苏淞沪警察厅]]，1914年1月，'''督办淞沪水陆警察署'''成立，[[萨镇冰|萨镇冰]]任督办，淞沪警察厅分为沪南、闸北两个分厅。同年7月，督办淞沪水陆警察署撤销，裁撤分厅，又组为淞沪警察厅。

1927年[[上海特别市|上海特别市]]成立后，淞沪警竂厅改组成立[[上海特别市公安局|上海特别市公安局]]，厅长[[沈毓麟|沈毓麟]]改任公安局长。<ref>{{cite web|url =http://www.shtong.gov.cn/newsite/node2/node2245/node4476/node58281/node58327/node58337/userobject1ai46189.html|title =江苏淞沪警察厅|work =《上海公安志》|publisher =上海市地方志办公室|deadurl =yes|archiveurl =https://web.archive.org/web/20131219064936/http://www.shtong.gov.cn/newsite/node2/node2245/node4476/node58281/node58327/node58337/userobject1ai46189.html|archivedate =2013-12-19}}</ref>

1930年调整为上海市公安局，1937年又改称上海市警察局。

1949年中华人民共和国成立，设置上海市公安局。

== 职责 ==
上海市公安局对治安、出入境、道路交通、消防、边防、监所、公共信息网络安全、水上治安、人口等方面进行综合管理，主要职责有，
* 预防、制止和侦查违法犯罪活动；
* 维护社会治安秩序，制止危害社会治安秩序的行为；
* 管理交通、消防、危险物品和特种行业；
* 管理户政、国籍、入境出境事务和外国人在境内居留等有关事务；
* 管理集会、游行、示威活动；
* 监督管理计算机信息系统的安全保卫工作；
* 指导和监督国家机关、社会团体、企业事业组织和重点建设工程的治安保卫工作，指导群众性组织的各类治安防范工作。<ref name="SGAJ" />

* “8·18”绑架案（案发于1982年）：中华人民共和国建立以来上海首起绑架案
* [[于双戈抢劫银行案|于双戈抢劫银行案]]（案发于1987年）：中华人民共和国首起持枪抢劫银行案
* [[戴厚英遇害案|戴厚英遇害案]]（案发于1996年）
* [[魏广秀敲头案|魏广秀敲头案]]（案发于1997年）：中国首起连环敲头凶杀案
* [[杨佳袭警案|杨佳袭警案]]（案发于2008年）：上海最严重袭警事件
* [[沪浙赣连环电击杀人案|沪浙赣连环电击杀人案]]（案发于1981年）：上海警方31年追凶，于2013年破案
