'''上海戰役'''，亦称淞沪战役，是指于1949年5月12日至27日于上海在中国人民解放军和中华民国国军之间发生的战役，解放军获胜，取得上海。[[中华民国政府|中华民国政府]]称'''上海沦陷'''，[[中国共產黨|中国共產黨]]及[[中华人民共和国政府|中华人民共和国政府]]官方稱'''上海解放'''。

== 背景 ==
1949年4月20日夜，[[中国人民解放军|解放军]]发起[[渡江战役|渡江战役]]。22日凌晨，[[江阴要塞起义|江阴要塞起义]]又成功的切开国军在长江下游布置的防线。此后，在江面宽阔的长江下游，国军仅余[[吳淞口|吴淞口]]一处门户。23日，解放軍佔領[[南京市_(中華民國)|南京]]。[[國民政府|國民政府]]已于此前遷都至[[廣州|廣州]]，解放軍在佔領南京後，便開始著手擬定進攻中國经济中心和最大的工业城市─[[上海市_(中華民國)|上海]]。

当时在上海市區中，擁有國府守軍25萬人、遍布市區內外的3,000個美式碉堡、4,000個鋼筋水泥永備工事、一萬多野戰衛星工事及兩萬多顆地雷。驻沪的海军部队有第一舰队主力舰只、第一巡防舰队及长江各防区逃回来的舰艇，共有军舰9艘、战舰11艘，分布在白龙江口至宝山城一线的长江及[[黄浦江|黄浦江]]江面，主力驻扎则在吴淞口<ref name=":0">{{Cite book|url=https://www.worldcat.org/oclc/953044905|title=怒潮狂飙：国民党海空军传奇|last=高|first=晓星|date=1999|publisher=江苏人民|year=1999|isbn=7214023156|location=|pages=176|oclc=953044905}}</ref>。

== 经过 ==
為了避免於市區內與國軍作戰以減少損失，上海戰役一開始，[[第三野戰軍|第三野戰軍]]總指揮粟裕指揮下轄第九、第十兵團共42萬人採用了雙層鉗形攻擊的方法合围國軍，第三野戰軍採取各種策略，將市區的守軍調到了外圍，從而逐步殲滅，隨後，解放軍採取多路「'''快速躍進、勇猛穿插、迂迴包圍'''」的戰術，從西、南兩側，向市區突進，將國軍分割包圍在街巷之中。{{請求來源}}
*北线 
**第十兵团'''第二十六军'''于5月12日夜攻占[[昆山|昆山]]，14日进至[[南翔|南翔]]。'''第二十八军'''、'''第二十九军'''挺进[[吴淞|吴淞]]，12日分别攻占[[浏河|浏河]]、[[太仓|太仓]]、[[嘉定|嘉定]]。14日猛攻[[月浦|月浦]]、[[杨行|杨行]]、[[刘行|刘行]]一线。国军火力网封锁，并在[[坦克|坦克]]、[[装甲车|装甲车]]的掩护下，连续反冲击。15日，国民党守军又增调第21军、第99师加强该地区防御，战斗异常激烈。
**第十兵团'''第二十九军'''调整进攻部署于15日攻入月浦街内，迫使守军退守街外碉堡群。17日第二十八军肃清[[刘行|刘行]]守军，19日攻下刘行[[国际无线电台|国际无线电台]]，并以远射程炮轰击[[江湾机场|江湾机场]]。
*南线
**第九兵团'''第二十军'''于12～14日攻占[[平湖|平湖]]、[[金山卫|金山卫]]、[[奉贤|奉贤]]（[[南桥镇|南桥镇]]）；'''第二十七军'''进占[[嘉兴|嘉兴]]、[[嘉善|嘉善]]、[[松江|松江]]、[[青浦|青浦]]等县城，待命攻击市区。
**'''第三十军'''沿杭州湾北侧公路向浦东高桥进击。14日占领[[南汇|南汇]]，进逼[[川沙|川沙]]，威胁国军防御重点[[高桥|高桥]]。守军被迫由市区急调第51军至[[白龙港|白龙港]]、[[林家码头|林家码头]]地区，以增强[[浦东|浦东地区]]防御。16日于川沙东北[[白龙港|白龙港]]围歼立足未稳的国军第51军、暂第8师全部和第37军一部。同日，第三十一军进攻周浦，经10小时激战，占领[[周浦镇|周浦镇]]。尔后与'''第三十军'''并肩向高桥攻击前进。18日进至高桥[[杨家楼|杨家楼]]、[[朱家木桥|朱家木桥]]一线，将国军第12军压缩于高桥地区，并割断其与浦东市区第37军的联系。为保障吴淞口出海通道，国军第75军第95师东调增防。19日起，守军在海、空军配合下，依托该地区濒江临海有利地形，频繁反冲击与共军反复争夺。23日下午，[[人民解放军特种兵重炮兵部队|人民解放军特种兵重炮兵部队]]赶到，立即占领[[王家湾|王家湾]]以西阵地，向高桥东北海面国民政府海军10余艘舰艇<ref>“营口”、“永靖”、“联利”等舰和四艘炮艇</ref>射击，击中7艘，余则逃脱。

至此，共军封锁高桥以东海面。在10天的外围作战中，共军完全击破国民党淞沪外围防御体系，突破其主阵地，迫使国民党守军将主力集中于吴淞口两侧地区。

== 总攻 ==
5月23日夜，共军发起总攻。
*24日，第九兵团第二十军攻占浦东市区，国军渡江西撤。第二十七军占领[[虹桥镇|虹桥镇]]、[[龙华镇|龙华镇]]和[[龙华机场|龙华机场]]，进至苏州河以南市区边缘。'''第二十三军'''进至[[漕河泾|漕河泾]]、[[龙华|龙华地区]]。24日晚，第二十七军由[[徐家汇|徐家汇]]、[[万航渡路|梵皇渡路]]一线攻入市区；第二十三军由[[龙华|龙华]]附近攻入市区；第二十军主力西渡[[黄浦江|黄浦江]]由[[高昌庙|高昌庙]]攻入苏州河以南市区。各部队采取'''多路跃进、快速勇猛穿插及迂回包围'''等战术，直插市区各街道。<ref>{{cite book|title=三野档案：第三野战军 三十五、上海战役 P200|publisher=国防大学出版社|author=曹宏 李莉}}</ref>共军严格执行不使用火炮等重兵器射击的规定，在[[中共地下党|中共地下党组织]]密切配合下，市区战斗进展顺利。25日上午完全控制[[苏州河|苏州河]]以南市区，并乘胜向苏州河以北追击。25日上午经[[上海地下党|上海地下党]]的联系，国民政府军淞沪警备副司令兼第51军军长[[刘昌义|刘昌义]]与解放军谈判，接受限期交出阵地、军队集结待命等条件。25日晚，第三十军、第三十一军攻克高桥守军，至26日完全肃清浦东之敌。第二十七军、第二十三军及第二十军一部占领苏州河以北地区。27日下午3时，第二十七军在[[杨树浦|杨树浦]]受降最后一批国民党军，占领上海。
*第十兵团第二十六军占领[[纪家港|纪家港]]、[[沈家桥|沈家桥]]等地，并向[[绿杨桥|绿杨桥]]、[[塘桥|塘桥]]攻击前进。同日，[[汤恩伯|汤恩伯]]等登舰出海，在[[吴淞口|吴淞口]]外指挥退却，国军第75军从高桥第6师抽调至吴淞，掩护撤逃。[[苏州河|苏州河]]以北国军主力向[[吴淞口|吴淞口]]收缩，苏州河以南交警北撤。共军各部立即发起追击，阻截溃逃守军。26日，第二十六军攻占[[大场|大场]]、[[江湾|江湾]]，第二十九军攻占[[宝山|宝山]]，'''第二十五军'''一部攻至[[吴淞江边码头|吴淞江边码头]]，另一部攻占[[吴淞要塞|吴淞要塞]]，第二十八军、'''第三十三军'''攻占[[杨行|杨行]]、[[吴淞|吴淞]]等地。<ref>{{cite web|title=上海通志第十卷军事第五章兵事第三节现代兵事四、解放上海战役|url=http://www.shtong.gov.cn/Newsite/node2/node2247/node4571/node79225/node79235/userobject1ai103563.html|publisher=上海地方志办公室|date=2008-07-10}}</ref>6月1日，第二十五军向[[崇明岛|崇明岛]]发起登陆攻击，歼第1绥靖区之暂1军3700余人，6月2日占领全岛。至此，战役结束。

5月27日中午，駐守於吳淞口的國軍仍持續頑抗，解放軍為保障上海市民生命安全及不影響上海市的生產機能，在地下黨組織的配合下，改用政治攻勢，從內部瓦解了國軍，最終攻佔上海。{{請求來源}}

== 战役前后国民党部队的起义或投诚 ==
*1月28日，浙江省政府主席陈仪派人去上海劝说[[汤恩伯|汤恩伯]]学北平[[傅作义|傅作义]]接受'''和平解放上海'''。被[[汤恩伯|汤恩伯]]向蒋介石告密。蒋介石令上海警察局局长[[毛森|毛森]]逮捕陈仪，押送[[衢州|衢州]]关押，浙江省解放前夕押送台湾。1950年6月18日陈在台湾伏法。
*4月14日，中共地下组织[[田云樵|田云樵]]、[[舒忻|舒忻]]、[[周其昌|周其昌]]策动国民党特种部队伞兵3团刘农畯率司令部，伞兵一、二团各一队共2500余人，在乘船去福州途中于上海吴淞口起义，开往苏北连云港。
*5月25日，
**面对共军强大的攻势，[[刘昌义|刘昌义]]<ref>国民党淞沪警备区中将副司令兼51军军长中将</ref>通过中共地下组织[[田云樵|田云樵]]与第二十七军第八十一师师长[[孙端夫|孙端夫]]、政治委员[[罗维道|罗维道]]接洽、投诚临阵起义，并率领万余人在上海接受人民解放军改编，从而使人民解放军迅速跨过苏州河，消灭残敌，至27日下午，上海宣告全部解放。<ref>{{cite web|title=上海军事志第八编战事第三章现代战事第十二节上海战役前后国民党部队的起义或投诚 |url=http://www.shtong.gov.cn/Newsite/node2/node2245/node75342/node75354/node75371/node75377/userobject1ai92276.html|publisher=上海地方志办公室|date=2007-03-26}}</ref>
**中共地下组织[[刘燕如|刘燕如]]、[[顾永金|顾永金]]策动国民党上海警备大队机动车队队长[[高莲荪|高莲荪]]率全队33人在上海市威海卫路警备大队部起义。该队全部配备短枪，装备有吉普车3辆，摩托车10辆，队部设在威海卫路147号。
**中共地下组织[[沙玉林|沙玉林]]策动国民党青年军202师606团[[淡寿延|淡寿延]]等2000余人在上海[[杨树浦|杨树浦]]一带起义。
*5月31日，中共地下组织[[黎平|黎平]]策动国民党暂一军11师上校副师长[[刘贺田|刘贺田]]率领1987人在上海[[崇明|崇明堡镇]]起义。

== 后续 ==
1947年派赴台灣鎮壓[[二二八事件|二二八事件]]的國軍[[國民革命軍第二十一軍|第21軍]]（原整編第21師）在此戰役之中，軍長[[王克俊|王克俊]]丟下部隊，於1949年5月25日被解放軍殲滅大部，殘部於26日列隊投降。

上海战役后，由于部分拥有大量资产的商人希望通过[[银元|银元]]的交易来企图阻止人民币进入上海市场流通，并维持接管前的经济，导致了与中共政权的的对抗，最终中共政权通过强硬的行政手段取得胜利，控制上海的金融市场，此事被称为[[银元之战|银元之战]]。
