'''上野幌站'''（{{jpn|j=上野幌駅|hg=かみのっぽろえき|rm=Kami-Nopporo eki|f=y}}）是位於[[北海道|北海道]][[札幌市|札幌市]][[厚別區|厚別區]]厚別區上野幌，[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[千歲線|千歲線]]的車站，亦是千歲線於札幌市最南的車站。車站編號為'''H06'''。

== 簡介 ==
本車站是千歲線上行列車中，從札幌站開始的第一個[[待避|待避站]]。由於車站月台興建在築堤之上，所以比較狹小但適合眺望周圍的景色。

車站的名稱源自厚別區上野幌。而「野幌」則是由[[阿伊努語|阿伊努語]]「ヌプ．オル．オ．ペッ（拉丁字母：nup-or-o-pet）」演變而來，意思為「原野中的河流」<ref>{{Cite_web|url=https://www.city.sapporo.jp/atsubetsu/gaiyo/furusato.html |title=厚別區的現況、歷史／札幌市厚別區 |publisher=厚別區役所 |accessdate=2018-11-01}}</ref>。雖然車站名為「上」野幌，它與位於[[江別市|江別市]]的[[函館本線|函館本線]][[野幌車站|野幌車站]]相距超過10公里。

== 歷史 ==
1926年，車站於現時位置西北方約1公里的位置開始營運，為木造站房2面3線的車站。當時車站種植了很多[[杜鵑花屬|杜鵑花]]及[[大麗菊|大麗菊]]，因而受到全國的注目。1973年千歲線進行路線更改，本車站亦遷移至現時的所在地。而舊車站於[[1991年|1991年]]（平成3年）建設成厚別南公園。
* [[1926年|1926年]]（[[大正|大正]]15年）[[8月21日|8月21日]] - {{Link-ja|北海道鐵道 (第2代)|北海道鉄道 (2代)}}札幌線設立'''上野幌站'''<ref>『恵庭市史』恵庭市、1979年、p.1026</ref>。並開始處理旅客業務。
* [[1943年|1943年]]（[[昭和|昭和]]18年）[[8月1日|8月1日]] - 北海道鐵道被當時[[鐵道省|鐵道省]]收購並國有化<ref>{{NDLDC|2961465/3}} 「鐵道省告示第204號」『官報』1943年7月26日]（國立國會圖書館數位化資料）</ref>。成為[[日本國有鐵道|日本國有鐵道]]千歲線的車站<ref name="sone24"/>。
* [[1945年|1945年]]（昭和20年）[[4月1日|4月1日]] - 開始處理貨物及行李託運的業務。
* [[1961年|1961年]]（昭和36年）[[1月14日|1月14日]] - 由於鐵道容量不足，本車站與北廣島之間設置西之里信號場<ref name="hyakunenshi1">北海道鐵道百年史　下卷　日本國有鐵道北海道總局　1981年發行、P62。</ref>。
* [[1973年|1973年]]（昭和48年）[[9月9日|9月9日]] - 搬遷至新線。同時車站變成複線、高架及無人化<ref name="hyakunenshi1"/>。
* [[1987年|1987年]]（昭和62年）4月1日 - 經[[國鐵分割民營化|國鐵分割民營化]]後由JR北海道繼承<ref name="sone24"/>。
* [[1998年|1998年]]（[[平成|平成]]10年）4月1日 - 車站有人化（業務委託站）。
* [[2008年|2008年]]（平成20年）[[10月25日|10月25日]] - 開始使用[[智慧卡|智慧卡]]（[[Kitaca|Kitaca]]）<ref name="sone24"/>。

== 車站構造 ==
* 車站主要由1個島式月台及1個側式月台組成的2面3線複合月台。軌道沿堤壩由西北方延伸至東南方。
* 車站入口位於東北側，前往西南側需要使用隧道。
* 開始營運初期為無人車站，近年由於周邊地區進行開法，現在成為業務委託站（北海道JR Servicenet），並設有[[綠窗口|綠窗口]]（營業時間7時00分至17時20分）、[[自動售票機|自動售票機]]<ref group="注釋">由於車站只設置一部自動售票機，因此維修期間需要在綠窗口購買車票。</ref>、簡單自動入閘機（對應Kitaca及磁式車票）。
* 車站由新札幌站管理，亦負責夜間聯絡服務。

== 車站周邊 ==
位於[[北廣島市|北廣島市]]以東的交界，丘陵的山谷之間，房屋密度偏低。2004年12月厚別東通通車後，市民可以更容易前往車站西南方的地區，並且發展不同住宅區如平岡公園Benny Town（{{lang|ja|平岡公園ベニータウン}}）及Live Hills（{{lang|ja|ライブヒルズ}}）等。而車站東北側設有供[[停車換乘|停車換乘]]之用的停車場，由南方穿越至北方的野津幌川，以及位於北廣島市虹丘地區的札幌日本大學中學、高等學校、柏青哥店、動物醫院（新札幌La Paz動物醫院）、即日來回的入浴設施（湯處穗香、{{lang|ja|ほのか}}）。而隨著開發虹丘商業地區的同時，也預定開發上野幌站的周邊地區。

車站前方雖然比較冷清，但沿新札幌站方向步行約500米，則開始顯現密集的札幌市區的邊緣地帶。由2007年開始住宅區不斷向外延伸，而車站南側從2006年開始也建設了不少住宅區及中小規模的公寓。此外，車站西南往厚別東通方向約200米的地方亦經營了便利店（Seicomart），於2007年7月開始營業。

* [[國道274號|國道274號]]
* 札幌日本大學中學、高等學校
* 北海道北廣島西高等學校
* 雪印種苗園藝中心
* 札幌報恩學園
* [[JR北海道巴士|JR北海道巴士]]「上野幌站通」巴士站（33線、大33線、大34線、大35線、新33線、新34線）
* {{Link-ja|夕張鐵道|夕張鉄道}}（夕張巴士）「上野幌站通」巴士站（札幌急行線）
