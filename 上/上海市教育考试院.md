'''上海市教育考试院'''，成立于1995年，是主要承担上海市义务教育阶段以后的各级各类教育考试招生工作的[[事业单位|事业单位]]，隶属于[[上海市教育委员会|上海市教育委员会]]，现任院长郑方贤<ref>{{cite web |title=上海市教育委员会直属事业单位负责人信息 |url=http://www.shmec.gov.cn/html/xxgk/201609/1052016002.html |website=www.shmec.gov.cn |language=zh}}</ref>。

== 职责 ==
上海市教育考试院组织实施普通高校考试、成人高校考试、研究生考试、高等教育（含中等专业教育）自学考试和国家学历文凭考试及招生工作，组织实施中等学校升学考试等，并承担有关教育证书和资格考试工作，指导监督区县招生办，开发考试信息资源，对外交流等<ref>{{cite web |title=上海市教育考试院 |url=http://www.shmec.gov.cn/web/jyzt/zsdw/jyzt_show.html?id=16 |website=www.shmec.gov.cn |accessdate=2018-07-19 |language=zh}}</ref>。

== 内设机构 ==
上海教育考试院设置以下机构：党委办公室，院办公室，高等学校招生办公室，中等学校招生办公室，社会考试办公室，高等教育自学考试办公室，命题办公室，招生考试研究所，网络信息中心，后勤保障中心<ref>{{cite web |title=上海招考热线-上海市教育考试院官网 |url=http://www.shmeea.edu.cn/page/01000/ |website=www.shmeea.edu.cn |language=en}}</ref>。
