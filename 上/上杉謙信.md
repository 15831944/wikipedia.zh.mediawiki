'''上杉謙信'''（{{bd|1530年|2月18日|1578年|4月19日|catIdx=Uesugi, Kenshin}}）是一名活躍於[[日本戰國時代|日本戰國時代]]的[[大名|大名]]，後世譽為'''越後之龍'''、'''越後之獅'''、'''軍神'''。[[越後國|越後]]守護代[[長尾為景|長尾為景]]幼子，幼名“虎千代”，成年後稱'''長尾景虎'''。

由於繼承了[[關東管領|關東管領]]「上杉」的姓氏，並先後得到[[關東管領|關東管領]][[上杉憲政|上杉憲政]]和[[室町幕府|室町幕府]][[征夷大將軍|將軍]][[足利義輝|足利義輝]]的[[偏諱|偏諱]]，故又稱'''上杉政虎'''、'''上杉輝虎'''，篤信[[佛教|佛教]]，[[法名|法名]]'''谦信'''，號'''宗心'''，[[齋號|齋號]]不識庵。官位為「從五位下彈正少弼」，死後由[[明治天皇|明治天皇]][[追贈|追贈]]從二位。

任內終結內亂頻仍的越後國，除了政務與軍事外，也致力於經濟產業的復興。另由於救援盟國及關東秩序的回復屢次出兵，在48年的生涯中，分別與[[武田信玄|武田信玄]]、北条氏康、[[織田信長|織田信長]]、越中一向一揆、[[蘆名盛氏|蘆名盛氏]]、能登畠山氏、佐野昌綱、神保長職、[[椎名康胤|椎名康胤]]等大名頻頻作戰，尤其5度與武田信玄在川中島地區作戰最為著名，廣為後世描述。之後應足利將軍家邀請而上洛，從越後國經北陸路西進越中國、能登國及加賀國等擴大勢力，但壯志未酬而去世。作戰的戰兜為飯綱明神前立鐵錆地張兜。

謙信共有四名[[養子|養子]]，為[[上杉景勝|上杉景勝]]、[[上杉景虎|上杉景虎]]、[[上條政繁|上條政繁]]和[[村上國清|山浦景國]]。

== 經歷 ==
=== 出生與初陣 ===
[[File:KasugayamaCastle-print.jpg|thumb]]]]
[[享祿|享祿]]3年（[[1530年|1530年]]）出生於[[春日山城|春日山城]]，是越後[[守護代|守護代]][[長尾為景|長尾為景]]的四男，母親為越後栖吉城主長尾房景（古志長尾家）之女虎御前。幼名“虎千代”。[[天文_(後奈良天皇)|天文]]5年（1536年）父親隱居，家督由兄長[[長尾晴景|晴景]]繼承，7歲的虎千代於城下{{tsl|ja|林泉寺 (上越市)|林泉寺}}出家，由住持{{tsl|ja|天室光育}}教育之。<br/>
天文11年（1542年）12月，為景病逝，越後豪族[[揚北眾|揚北眾]]叛變蜂起，病弱的晴景無力平亂，乃於翌年令14歲的虎千代元服並改名為景虎，駐守{{tsl|ja|栃尾城}}，由[[本庄實乃|本庄實乃]]輔佐之。1544年，越後豪族[[揚北眾|揚北眾]]進攻栃尾城，景虎將城兵一分為二，部分兵力迂迴至敵本陣的背後急襲，造成敵軍混亂，之後城內的本隊配合出擊，成功擊退三條城主長尾俊景與黑瀧城主[[黑田秀忠|黑田秀忠]]的侵攻，是為初陣{{tsl|ja|栃尾城の戦い|栃尾城之戰}}。
=== 繼承家督 ===
天文14年（1545年）10月，守護上杉家老臣黑瀧城主黑田秀忠再度謀反，並進攻守護代晴景的居城春日山城，景虎之兄長尾景康遭到殺害，黑田秀忠之後於黑瀧城籠城，景虎自上杉定實處接受討伐令，並代替兄長擔任總大將出兵討伐，成功降伏秀忠（黑瀧城之戰）。但翌年天文15年（1546年）2月，秀忠再度舉兵反叛，這次景虎澈底殲滅了再度興叛的黑田，景虎威望益張，家中因而分裂成擁立晴景及景虎的兩派，最終在天文17年（1548年），越後[[守護|守護]][[上杉定實|上杉定實]]介入調停，晴景隱居，景虎成為兄長的養子並繼承家督，時年19。1550年，[[上杉定實|上杉定實]]死去，將軍[[足利義輝|足利義輝]]准許景虎使用白傘袋與毛氈鞍覆，確定其越後守護之位。同年，同族的坂戶城主[[長尾政景|長尾政景]]不滿景虎繼位而謀反，景虎於翌年天文20年（1551年）1月進攻政景方發智長芳的居城板木城並獲得勝利，同年8月包圍坂戶城（坂戶城之戰）成功降伏政景，政景之後成為景虎的重臣，越後自此統一。<br/>
天文21年（[[1552年|1552年]]），[[北條氏康|北條氏康]]攻陷[[关东管领|关东管领]][[上杉宪政|上杉宪政]]的平井城，憲政逃向越後向景虎求援，景虎將之安置在御館，隨後遣平子孫三郎、[[本庄繁長|本庄繁長]]等關東諸將反攻上野沼田城，大破之，成功奪還平井城及平井金山城。翌年，敘任從五位下彈正少弼。

== 與武田、北條的衝突 ==
[[File:Sengoku_period_battle.jpg|thumb]]
{{main|川中島之戰}}
1552年，領地遭[[武田信玄|武田信玄]]佔奪的[[信濃|信濃]]守護[[小笠原長時|小笠原長時]]向景虎求援，翌年，信濃葛尾城主[[村上義清|村上義清]]亦來投奔，景虎一度奪還其領，五次的[[川中島之戰|川中島之戰]]由此發端。1553年，[[上洛|上洛]]拜謁天皇與將軍，蒙受討敵之敕令。1555年，親自出兵鎮壓暗通武田的[[北條高廣|北條高廣]]，同年4月，進行了第二次川中島之戰，[[今川義元|今川義元]]居中調解，雙方和睦。1557年，因信玄毀約而爆發了第三次川中島之戰。1559年，景虎再度上洛，得到了將軍如[[管領|管領]]一般的待遇。1561年3月，以[[上杉憲政|上杉憲政]]的名義，帶領十萬大軍包圍[[小田原城|小田原城]]（{{tsl|ja|小田原城の戦い (1560年)|小田原城之戰}}），隔月撤兵行經到[[鎌倉|鎌倉]][[鶴岡八幡宮|鶴岡八幡宮]]時，在憲政的請求下，於[[源賴朝|源賴朝]]的木像下繼承了[[關東管領|關東管領]]一職及[[山內上杉家|山內上杉家]]家名（長尾氏本是平氏，繼承上杉氏后便成藤原氏族人），改名'''上杉政虎'''，並得到了[[上野國|上野國]]的統治權。同年8月，進行了第四次川中島之戰，這是兩軍最大規模的戰爭，傳說他曾直闖敵陣砍傷[[武田信玄|武田信玄]]。<br/>
1561年11月，[[北條氏康|北條氏康]]開始反擊，於[[武藏國|武藏國]]生野山大敗在川中島損耗甚巨的上杉軍（{{tsl|ja|生野山の戦い|生野山之戰}}）。12月，將軍[[足利義輝|足利義輝]]下賜偏諱，改名'''上杉輝虎'''。<br/>
南下攻打關東同時，景虎也在1560年進軍[[越中國|越中國]]，打著協助[[椎名康胤|椎名康胤]]的旗號襲擊神保家的領土，攻陷了富山城，當主{{tsl|ja|神保長職}}逃往增山城。1562年，上杉軍包圍增山城，神保長職在能登畠山氏的仲介下投降。

== 西征及晚年 ==
[[File:Letter_from_Uesugi_Kenshin_to_Uesugi_Kagekatsu.jpg|thumb]]
[[永祿|永祿]]8年（1565年），[[北條氏康|北條氏康]]攻擊關宿城，輝虎聯合[[佐竹義重|佐竹義重]]欲救之，北條軍不戰而退。1568年12月，北條家為對抗武田家而改與上杉家結盟，翌年3月，「'''{{tsl|ja|越相同盟}}'''」成立，輝虎收[[北條氏康|北條氏康]]之子[[上杉景虎|三郎]]為養子，不久即以法名'''不識庵謙信'''自稱。<br/>
[[元龜|元龜]]2年（1571年）10月，氏康逝世，兩家同盟破裂，並於[[天正_(日本)|天正]]2年（1574年），與[[北條氏政|北條氏政]]軍在[[利根川|利根川]]對峙，北條家再度進攻關宿城，謙信救援失利，被[[北條氏照|北條氏照]]以寡軍襲營，而[[佐竹義重|佐竹義重]]不願配合反攻，謙信遂領軍撤退，此戰後關宿城乃至羽生城都被北條家佔據，上杉家在關東的勢力大為衰退。<br/>
[[元龜|元龜]]3年（1572年），謙信佔領由[[一向一揆|一向一揆]]眾控制的[[富山城|富山城]]，1575年與[[石山本願寺|石山本願寺]]結盟，1577年攻擊[[能登國|能登國]]的[[畠山氏|畠山氏]]，當時畠山氏為家臣長氏和遊佐氏所控制，謙信在親上杉派的重臣[[遊佐續光|遊佐續光]]內應下攻佔能登七尾城，畠山氏滅亡。爲了救援將近被消滅的親織田派的[[長續連|長續連]]，[[織田信長|織田信長]]派遣[[柴田勝家|柴田勝家]]、[[前田利家|前田利家]]和[[丹羽長秀|丹羽長秀]]等人出兵[[能登國|能登國]]，自己也有意出征。當織田軍抵達加賀之時，因上杉軍已攻佔末森城，收到消息的織田軍只好被迫撤退。上杉軍立即追擊，斬首千餘，獲得大勝(有爭議)，是為[[手取川之戰|手取川之戰]]。<br/>
[[天正_(日本)|天正]]6年（1578年），謙信先是策反蘆名家臣山內重勝、大槻政通，同年3月欲再次準備南下關東救援結城家之時，於春日山城如廁中病逝（這是[[腦溢血|腦溢血]]的常見症狀之一，後世認為謙信應是因腦溢血而死），終年49歲。但是所謂的西征說近年備受質疑，現在學界的說法是認為上杉謙信是要攻打關東而非上洛。辭世詩句為「四十九年一睡夢、一期榮華一盃酒」，戒名為「不識院殿真光謙信」。<br/>
謙信死後，上杉家內部發生[[御館之亂|御館之亂]]，最後由[[上杉景勝|上杉景勝]]獲勝繼承上杉家。

== 信仰 ==
謙信篤信[[佛教|佛教]]，曾經因為[[信仰|信仰]]打算往[[高野山|高野山]][[出家|出家]]，引起了家內的紛爭。特別信奉佛教的[[戰神|戰神]]－[[毘沙門天|毘沙門天]]，自詡為[[毘沙門天|毘沙門天]]的[[化身|化身]]。

== 評價 ==
日本史学界的权威{{tsl|ja|坂本太郎 (歴史学者)|坂本太郎}}在其著作《日本史概说》中评价谦信说：“在杀伐无常，狂争乱鬥的戰国武将當中，上杉谦信以尊神佛、重人伦、尚气节、好学问的高节之士见称，令人感到不愧是混乱中的一股清新气息。”

同時代的關東武將[[太田資正|太田資正]]也對上杉謙信有如下評語：「謙信公之人品，八分乃賢者，二分為惡人。恣肆怒氣，行事怪異，是其'惡'；除此而外，勇猛而無欲，清靜而無邪，廉直而無私，明敏好察，慈惠待下，喜聞人諫等，是為其善。雖有微瑕，不足掩其輝，實乃絕世罕有之良將。」

江户时期的儒者岩垣松苗在其著作《国史略》中，評論谦信：「然观其平生所为，则祝发披缁，不畜妻妾，不茹荤腥，俨然持律僧也。而至行军略地，则杀人盈野，血流为河。未尝有勤王济世之志矣。加之弃祖先之胤，养螟蛉之子，其尸未寒，生祸阋墙，与国破嗣绝者，仅一间也已。智计虽多乎？胆勇虽壮乎？徒足以行强暴于一世已。」

== 人物 ==
* 《完全圖解日本戰國武將54人》記載其長相為細長的眼睛，厚實的下巴，走路有些一跛一跛。
* 據傳謙信早年曾與上野國平井城主千葉采女作為人質的女兒伊勢姬相戀，後因柿崎景家強烈反對而分手，於是伊勢姬在青龍寺出家終其一生。
* 謙信性喜喝酒，甚至被人稱為酒豪，即使騎在馬上也可以大快暢飲，有著名的马上杯之说。而且喝的再多也可以保持心情平穩，因此他的死因也可能是喝酒過多引發[[腦溢血|腦溢血]]致死。此外，辭世詩句中有「酒」字，所以一般認為謙信乃好酒之人。
* 一般人對於謙信的刻板印象於內政能力比較弱，實際上他在領內實行精密的統治，例如：發展越後國府春日山城下町直江津（今町湊）；整備越後通往他國的道路；領地內金、銀礦山與港口經營；越後特產青苧、鹽、紙等產業之培植，與明朝、朝鮮貿易。因此謙信連年用兵，仍然擁有大量軍資金，以維持軍略。
* 謙信的漢學造詣極高，甚至能通漢語，雅好詩文、琴曲，有大量漢詩流傳於世， 是個文武兼優的名將。攻陷[[能登|能登]]國[[七尾城|七尾城]]時寫下日本至今最有名的賞月漢詩「十三夜」：「霜滿軍營秋氣清，數行過雁月三更；越山併得能州景，遮莫家鄉憶遠征。」（由[[林羅山|林羅山]]潤飾）
* 上杉謙信殺戮頗多，在進攻山根城、私市城、前桥城、小山城、佐野城、猿洼城時，破城後的屠殺都十分殘酷。
* 小說家八切止夫曾提出上杉謙信其實是女兒身之說，影響及部分小說、漫畫跟電子遊戲。

== 墓所・靈廟 ==
[[File:Uesugi_Kenshin_Grave.jpg|260px]]
* 遺骸安置於{{tsl|ja|米沢城|米澤城}}內，維新後改葬於米澤藩歷代藩主長眠的上杉家廟所。而在春日山林泉寺（[[新潟縣|新潟縣]]上越市）與[[高野山|高野山]]亦有謙信公之墓。[[明治|明治]]5年（1872年）於米澤城本陣遺跡處興建上杉神社供奉謙信公。上杉神社於[[大正|大正]]8年（1919年）米澤大火中燒毀，但在大正12年（1923年）由[[伊東忠太|伊東忠太]]博士設計重建完成，並保存至今。
