'''上高县'''位于[[中国|中国]][[江西省|江西省]]西北部，是[[宜春市|宜春市]]下辖的一个[[县|县]]。

== 历史沿革 ==
[[东汉|东汉]][[中平|中平]]年间[[汝南|汝南]][[上蔡|上蔡]]百姓迁到此地，故析建成县置上蔡县。

[[西晋|西晋]][[太康|太康]]元年（280年）改为望蔡县。

[[隋|隋]][[开皇|开皇]]九年（589年）并入建城县。

[[唐|唐]][[中和_(唐僖宗)|中和]]年间置上高镇。

[[南唐|南唐]][[升元|升元]]年间置上高场，[[保大|保大]]十年（952年）置上高县，属[[筠州|筠州]]（后改称[[瑞州|瑞州]]）。

明清属瑞州府。

现隶属于宜春市。

== 行政区划 ==
上高县下辖1个街道6个镇6个乡14个村委会。
*街道：[[敖阳街道|敖阳街道]]
*镇：[[田心鎮_(上高縣)|田心镇]]、[[徐家渡镇|徐家渡镇]]、[[锦江镇|锦江镇]]、[[泗溪镇|泗溪镇]]、[[翰堂镇|翰堂镇]]、[[南港镇|南港镇]]
*乡：[[芦洲乡|芦洲乡]]、[[塔下乡|塔下乡]]、[[蒙山乡|蒙山乡]]、[[镇渡乡|镇渡乡]]、[[野市乡|野市乡]]、[[新界埠乡|新界埠乡]]
*村委会：横江、洲上、堆峰、车溪、先锋、光明、湾溪、泽山、桐山、富港、五星、端溪、城陂、三星。

== 地理 ==
上高县东西最大长度68公里，南北最大宽度45公里，全县面积1350.25平方公里。

上高县属中亚热带季风气候。

== 经济 ==
2007年，全县实现生产总值43.88亿元，比上年增长14.5%，人均12944元；完成财政收入3.78亿元。
