'''上海浦东国际机场'''（{{airport codes|PVG|ZSPD}}）位于[[中華人民共和國|中華人民共和國]][[上海市|上海市]][[浦东新区|浦东新区]][[祝桥镇|祝桥镇]]的滨海地带，與[[上海虹橋國際機場|上海虹橋國際機場]]並列為[[上海|上海]]對外的兩大[[機場|機場]]。规划占地45.66平方公里，现有占地面积约37.8平方公里，距市中心约30公里<ref name="env">{{cite news |author = 上海市环境科学研究院 |url = http://www.envir.gov.cn/docs/2014/20140730394.pdf |title = 上海浦东国际机场三期扩建工程环境影响报告书（报批前公示版） |publisher = 上海环境网 |date = 2014-07-30 |accessdate = 2014-07-30}}</ref>，由[[上海机场集团|上海机场（集团）有限公司]]进行经营管理工作。往来于上海的绝大部分国际和大部分港澳航班在浦东机场起降，亦有国内航班，而上海市有另一机场离市中心较近──[[上海虹桥国际机场|上海虹桥国际机场]]则以中国国内航线为主，是上海最早的国际机场。

2017年，浦东国际机场旅客吞吐量7000万人次，位列中国大陆第二，世界第九。货邮吞吐量383.56万吨，位列中国大陆第一，世界第三。飞机起降近50万架次<ref>{{cite web |title = 上海浦东机场年旅客量突破7000万 仅次于北京香港 |publisher = 航空圈 |date = 2018-02-01 | url=http://www.sohu.com/a/214253753_600491|accessdate =2018-02-04|language = zh }}</ref>。浦东机场也是中国大陆最大的空港口岸，2017年出入境人数达3525万人次<ref>{{cite web |title = 上海浦东机场口岸出入境人数首破3500万 连续15年稳居全国空港口岸之首 |publisher = 看看新闻  |date = 2017-12-27 | url=http://news.sina.com.cn/o/2017-12-27/doc-ifyqcsft7253618.shtml|accessdate =2018-02-04|language = zh }}</ref>。

== 历史 ==
* 1997年10月，上海浦东国际机场一期工程建设全面开工。
* 1999年9月16日，上海浦东国际机场正式通航投入使用。
* 2002年，上海浦东国际机场年旅客吞吐量首次突破1000万人次。由于浦东机场仅凭一条跑道每日却有超过800架次航班起降，接纳的航班数量饱和，因此自2003年11月起，部分飞往山东半岛的航班改回虹桥机场起降直至第二跑道完工<ref>{{cite news|title=上海浦东机场趋于饱和 部分航班回迁虹桥机场|url=http://www.china.com.cn/chinese/TR-c/441697.htm|publisher=中国网|date=2003-11-13}}</ref>。
* 2004年，上海浦东国际机场年旅客吞吐量首次突破2000万人次。
* 2005年3月17日，上海浦东国际机场第二跑道（16/34）正式投入使用<ref>{{cite web|title=上海航空枢纽战略规划绘就 浦东机场第二跑道明年使用|url=http://www.shanghai.gov.cn/nw2/nw2314/nw2315/nw4411/u21aw95187.html|publisher=上海市人民政府|date=2014-11-02}}</ref>。
* 2007年12月20日，上海浦东国际机场第二航站楼工程通过竣工验收。
* 2008年3月26日，上海浦东国际机场第二航站楼和货运中心正式投入使用。
* 2008年2月19日，上海浦东国际机场第三跑道（17R/35L）试飞成功，并于3月13日正式投入使用<ref>{{cite news|title=上海浦东国际机场第三跑道试飞成功|url=http://www.gov.cn/jrzg/2008-02/19/content_893666.htm|publisher=中华人民共和国中央人民政府|date=2008-02-19}}</ref>。该跑道为主要用于降落，但停靠在3W货机坪的重型航空器可以申请使用该跑道起飞。第一跑道（17L/35R）则主要用于起飞。
* 2009年，上海浦东国际机场年旅客吞吐量首次突破3000万人次。
* 2009年8月，[[中华人民共和国国务院|中央人民政府]]正式批准设立[[上海浦东机场综合保税区|上海浦东机场综合保税区]]，区内实行与[[洋山保税港区|洋山保税港区]]相同的海关特殊监管等各项政策，其中包括营业税等方面的一系列优惠<ref>{{cite web |title = 浦东机场综合保税区获批 浦东形成"三港三区"格局(原载新华网) |publisher = eastday.com |date = 2009-08-19 |url = http://sh.eastday.com/qtmt/20090819/u1a617328.html |accessdate = 2009-08-19 |language = zh }}</ref>。
* 2010年，上海浦东国际机场年旅客吞吐量首次突破4000万人次。
* 2013年1月1日起，上海浦东国际机场对45国公民实行72小时过境免签证政策。
* 2013年8月22日，国务院正式批准设立[[中国（上海）自由贸易试验区|中国（上海）自由贸易试验区]]。试验区范围涵盖上海浦东机场综合保税区。
* 2014年12月19日，上海浦东国际机场年旅客吞吐量首次突破5000万人次。浦东机场由此正進入5000万人次世界超大型枢纽机场行列<ref>{{cite web |title = 上海浦东机场年旅客吞吐量首破5000万人次 |publisher = 民航资源网 | date = 2014-12-19 |url=http://news.carnoc.com/list/302/302047.html |accessdate = 2014-12-19 |language = zh }}</ref>。
* 2018年3月1日，上海浦东国际机场34L跑道Ⅲ类[[仪表着陆系统|仪表着陆系统]]（ILS）的正式投用，成为中国内地首家具有Ⅲ类ILS运行保障能力的民用机场<ref name="ils">{{cite news|title=明起浦东机场二跑道开放Ⅲ类运行 有助于提升航班起降准点率|url=http://newsxmwb.xinmin.cn/chengsh/2018/02/28/31364005.html|publisher=新民网|date=2018-02-28}}</ref>。

== 机场设施 ==
=== 跑道 ===
* 第一跑道（17L/35R）：1999年9月建成，同T1一起投入使用，平行建于T1的西侧，长4000米、宽60米，两侧道肩各宽1.5米，设有2条长4000米，宽29米，两侧道肩各宽7.5米的平行滑行道，4条联络道和6条快速出口滑行道。跑道主、次降方向均设Ⅱ类精密进近助航灯光系统，并按Ⅲ类精密进近要求预留。
* 第二跑道（16R/34L）：2005年3月建成，等级指标4F，位于T2东侧，第一跑道东侧2260米，与其平行，长3800米、宽60米，北端与第一跑道北端向南错开1000米。配套有西侧2条平行滑行道、跑滑之间6条垂直联络道（2条端联络道、2条旁通联络道），2条与第一跑道联通的东西向联络滑行道、快递机坪及通向平行滑行道的联络道、预留二跑道东侧三个通向规划第四跑道的道口。二跑道南段设置Ⅲ类进近灯光系统，北端设置Ⅱ类进近灯光系统<ref name="ils"/>。
* 第三跑道（17R/35L）：2008年3月建成，平行建于第一跑道西侧，与第一跑道南端对齐，间距460米，长3400米、宽60米，设东西两侧各6条快速脱离道。第三跑道助航灯光按双向I类设置。
* 第四跑道（16L/34R）：2015年3月建成，平行建于第二跑道東侧，与第二跑道南端对齐，间距460米，长3800米、宽60米，于2015年四月投入使用。
* 第五跑道（15/33）：在建中，平行建于第四跑道東侧1750米处，长3400米、宽45米，為一條填海造陸的跑道<ref name="4th runway">{{Cite web|title=四跑道时代：数字“量变”激发浦东机场“质变”_财经_中国网|url=http://finance.china.com.cn/roll/20150413/3054049.shtml|work=finance.china.com.cn|accessdate=2019-02-09}}</ref>。运行等级为4E级，可供除A380外的所有飞机起降<ref>{{Cite web|url=http://sh.people.com.cn/n/2015/1222/c134768-27371524.html|title=浦东时报特供：浦东机场第五跑道正加紧建设--上海频道--人民网|accessdate=2017-04-18|last=L_104116|work=sh.people.com.cn}}</ref>。

=== 航站楼 ===
上海浦东国际机场设有两座航站楼T1和T2。在两座航站楼间中转的旅客可利用三条连廊往返，连廊长400公尺，设有6段自动步道，步行时间约需15分钟。两座航站楼间另设有10分钟一班的短途巴士，巴士站分别在一号航站楼出发层的1号门（国内）、8号门（国际）和二号航站楼的27号门（国内）、23号门（国际）<ref>[http://www.shanghaiairport.com/content/content/contentListAction5.do?doType=-1&start=0&range=10&viewstyle=noback&id=A1208765616671CD57D80954C357818BFB59C7FD7E8033 上海机场(集团)有限公司]</ref>。

于1997年10月全面开工，1999年9月建成通航，由[[保罗·安德鲁|保罗·安德鲁]]设计<Ref>{{cite news|title=保罗·安德鲁辞世：穿过戴高乐、浦东机场，追寻一种轻盈自在|url=https://www.thepaper.cn/newsDetail_forward_2525105|publisher=澎湃新闻|date=2018-10-13}}</ref>。T1建有80万平米的机坪、有76个机位，货运库面积达5万平米，同时，装备有导航、通讯、监视、气象和后勤保障等系统，能提供24小时全天候服务。由主楼和候机长廊两大部分组成，均为三层结构，由两条信道连接，面积达27.8万平米，到港行李输送带13条，登机桥28座；候机楼内的商业餐饮设施和其它出租服务设施面积达6万平米。

T1航站楼于2012年末开始进行中转优化，航站楼扩建等改造工程。工程于2016年1月投入使用，极大提高了[[中国东方航空|中国东方航空]]及其他[[天合联盟|天合联盟]]成员航司间航班的中转效率<ref>{{cite news|title=浦东机场T1改造完成　东航或成最大受益者|url=http://news.carnoc.com/list/333/333154.html|publisher=民航资源网|date=2016-01-06}}</ref>。

==== 第二航站楼（T2） ====

于2007年12月20日全面竣工验收，并于2008年3月26日投入使用。T2位于T1的东侧成对称布置，T2以由主楼、连廊、候机长廊和固定登机桥4部分组成，面积达48.55万平方米。其中主楼长414米，宽138米，地下一层，地上三层；连廊长306米，宽60米，地上七层；候机长廊长1404米，宽41米，地下一层，地上四层。

T2是一项超大型公共建筑工程，以“满足基地航空公司及其联盟中枢运作的需要”和“以人为本方便旅客”为设计理念<ref>{{cite web|title=上海浦东|url=http://www.csair.com/cn/tourguide/airport_service/domestic/domestic/18h976osv3o9u.shtml|publisher=中国南方航空公司|accessdate=2017-09-20}}</ref>，工程结构体量超大。T2为乘客方便中转换乘，实现国际航班出发到达分流，国内航班出发、到达混流，提高了空间的利用率并减少乘客步行距离；T2设有固定登机桥共42座，长廊中部的登机桥可同时为国内和国际航班服务<ref>{{cite news|title=上海浦东机场T2航站楼演绎人性化机场新概念|url=http://news.sohu.com/20080326/n255922320.shtml|publisher=搜狐网|date=2008-03-26}}</ref>。

T2航站楼共打桩6798根桩；下部为钢筋混凝土结构，钢筋和混凝土用量分别为6.93万吨和30余万立方米；上部为约3万吨的大跨度钢结构，屋面采用约19万平方米的压型彩钢板和约1.6万平方米的梭型天窗；四周玻璃幕墙约9.6万平方米；室内装饰大吊顶12.6万平方米，楼层金属吊顶15.2万平方米；墙面装饰2.5万平方米，主楼和连廊地面采用18万平方米的花岗石石材，候机长廊地面采用约12万平方米的地毯以降低噪音<ref>{{cite news|title=浦东机场2号航站楼探营 航班中转无需"上上下下"|url=http://xwwb.eastday.com/x/20080309/u1a408980.html|publisher=东方网|date=2008-03-09}}</ref>，固定登机桥采用1.8万平方米橡胶地板。

===== 机场博物馆 =====
第二航站楼国际出发候机厅设有浦东机场博物馆。博物馆成立于2014年6月4日，成立时是中国开设的首家机场博物馆<ref name="人民网"/>。而在博物馆内举办的首个展览是“东风西渐——上海市历史博物馆馆藏欧洲瓷器展”，共展出53件瓷器展品，还在现场设有软陶制作互动活动<ref name="人民网">{{Cite web |title = 国内首家机场博物馆在浦东机场开馆 |url = http://art.people.com.cn/n/2014/0610/c206244-25129823.html |publisher = [[人民网|人民网]] | date = 2014-06-10 | accessdate = 2015-02-10 }}</ref>。

==== 南卫星厅（浦东机场三期工程） ====
[[File:201812_Satellite_Terminal_of_PVG.jpg|thumb]]
浦东机场已于2015年12月29日正式开建南卫星厅，即浦东机场三期工程。浦东机场南卫星厅是世界上最大的单体卫星厅，总建筑面积62.2万平方米，比浦东机场T2航站楼（48.55万平方米）还大。卫星厅由S1和S2两座大厅相连组成，形成工字型的整体构型，年处理旅客设计能力为3800万人次，工程总投资约206亿元。两座卫星厅与T2一样，为国内出发到达混流、国际分流的基本剖面布局，设有83座各类登机桥固定端，能够提供86-125个各型机位，并考虑了未来的适当预留<ref>{{cite news|title=浦东机场三期扩建启动 世界最大单体卫星厅总面积超T2[图]|url=http://sh.eastday.com/m/20151229/u1ai9160437.html|publisher=东方网|date=2015-12-29}}</ref>。西侧的S1与现有的T1航站楼共同运行，年旅客吞吐量3680万，而东侧的S2与现有的T2航站楼共同运行，年旅客吞吐量为4320万。2019年两座卫星厅建成后，浦东机场年旅客吞吐量保障能力将达到8000万人次，跻身全球十大机场行列。内部将建有[[浦东机场捷运|捷运]]连接两座主航站楼与卫星厅<ref>{{Cite web |title=上海浦东国际机场三期扩建工程全面开工 |url = http://www.shairport.com/cn/xwg/info_53559_itemid_178945637.html |publisher = 上海机场（集团）有限公司 |date= 2015-12-29}}</ref>。

=== 货运中心 ===
[[File:201812_Pudong_Airport_Freight_Center.jpg|thumb]]
上海浦东国际机场目前建成3个货运区，[[UPS|UPS]]也在这里设立有国际转运中心，负责中国除华南以外地区及北亚的快件。
* 一期货运区：建于T1的北面，配合T1航厦各航空公司基地中心及货物中心，进行物流运转工作。
* 东货运区：建于T2的北面，配合T2航厦各航空公司基地中心及货物中心，进行物流运转工作。
* 西货运区：建于第三跑道西侧，为规模及吞吐量最大的专属货运区，总建筑面积37万平方米，并可配合位于机场南方的[[洋山港|洋山港]]配合海空货运的运转。配合货运专用的第三跑道使用。

=== 机场巴士 ===
[[File:Shanghai_Pudong_Airport_T1-T2bus.JPG|thumb]]

与[[虹桥机场|虹桥机场]]不同，来往浦东机场的巴士全部以“机场X线”命名，其中机场1线至7线及机场夜宵线连接市区，主要行走快速道路，并使用不同于城市公交的大型巴士，可托运行李。浦东机场巴士共设置了八条线路：
* 机场一线：来往浦东国际机场及虹桥枢纽东交通中心,途径[[虹桥机场|虹桥机场]]2号航站楼、[[虹桥火车站|虹桥火车站]]，中途不设停靠站，票价30元。
* 机场二线：来往浦东国际机场及位于[[静安寺|静安寺]]的[[上海城市航站楼|城市航站楼]]，中途不设停靠站，票价22元。
* 机场四线：来往浦东国际机场及鲁迅公园，经停德平路浦东大道、[[江湾五角场|五角场]]、[[运光新村|运光新村]]，全程票价22元。
* 机场五线：来往浦东国际机场及[[上海火车站|上海火车站]]，经停[[杨高中路|杨高中路]]洋泾港桥、东方医院、[[成都北路|成都北路]][[延安中路_(上海)|延安中路]]（往上海站方向单行停靠），全程票价22元。
* 机场七线：来往浦东国际机场─[[上海南站_(铁路)|上海南站]]，行走[[上海华夏高架路|华夏高架路]]和[[上海中环路|中环路]]，经停华夏东路川沙路、华夏西路上南路，全程票价20元。
* 机场八线：来往浦东国际机场及南汇汽车站，经停当局楼、海天三路启航路、交通队、海关仓库、航油站、东方航空、河滨西路卡口、机场保税区、金闻路闻居路、祝潘公路川南奉公路、千汇路南祝公路、南祝公路周祝公路、南祝公路祝成路、南祝公路卫亭路、盐仓、人民公路城东路、南汇汽车站，全程票价10元。
* 机场九线：来往浦东国际机场和[[莘庄地铁站|莘庄地铁站]]南广场，全程票价22元。
* 机场环一线： 来往浦东国际机场及航城园，经停当局楼、公安分局、指挥部（非高峰站）、海关仓库、航空公司、施湾，全程票价3元。
* 机场守航线（夜宵线）：23:00后至当日航班结束后45分钟运营，由浦东国际机场开往虹桥机场1号航站楼，经停龙阳路地铁站、东方路张扬路、东方医院（浦东大道）、浙江中路（延安路人民广场）、石门一路（延安路）、华山路（延安路静安寺）、虹许路（延安路），全程票价30元。

== 重大事故和事件 ==
[[File:201801_Precheck_at_PVG_T2.jpg|thumb]]
* 2003年1月23日，[[法国航空公司|法国航空公司]]由[[巴黎夏尔·戴高乐机场|巴黎夏尔·戴高乐机场]]飞来的AF112次航班在放下[[起落架|起落架]]准备降落时，有两名藏在行李舱内的土耳其人从起落架处跌落，分别从空中坠落至[[南汇|南汇]]境内一处民居灶房间和旁边的一个小桔园内<ref>{{cite news|title=法航巴黎飞上海航班降落过程中掉下两人|url=http://news.sina.com.cn/c/2003-01-23/1629885490.shtml|publisher=新浪网|date=2003-01-23}}</ref>。[[国际刑警|国际刑警]]在事发后介入调查，发现两名土耳其籍死者为飞行爱好者，曾有多次藏匿在飞机行李舱偷搭飞机并偷渡的纪录，两人很可能死于高空缺氧寒冷<ref>{{cite news|title=法航坠人：是两名土耳其冒险青年 不幸上错飞机|url=http://news.sina.com.cn/c/2003-01-28/231441373s.shtml|publisher=新浪网|date=2003-01-28}}</ref><ref>{{cite news|title=法航客机空中坠机事件：高空缺氧寒冷导致丧命|url=http://news.sina.com.cn/c/2003-01-26/0717888065.shtml|publisher=新浪网|date=2003-01-26}}</ref><ref>{{cite news|title=组图：法航巴黎飞上海航班降落过程中摔出两人|url=http://news.sina.com.cn/c/2003-01-24/0102885808.shtml|publisher=新浪网|date=2001-01-24}}</ref>。
