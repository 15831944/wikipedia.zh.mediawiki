'''上海公交936路'''，线路配车空调化前原名'''ZX591路'''，曾是由[[上海市|上海市]][[闵行区|闵行区]][[龙柏新村|龙柏新村]]地区通往市中心的公交线路之一，开通于1994年。自2016年12月3日开始，该线路已经处于“线路暂停运营”状态<ref name=":2" />。由于该线路配车数在停运前已经减到0辆<ref name="jbst" />，这意味着该线路[[de_facto|事实上]]已经被撤销。

== 线路历史 ==
=== 2006年1月25日以前 ===
线路由弘华实业公司开通于1994年，开通时名为'''ZX（专线）591路'''，运行区间为夏碧路—[[龙柏新村|龙柏新村]]，开通时核准配车38辆，后经过多次转让，交由巴士四汽运营<ref>{{Cite web|url=http://www.shtong.gov.cn/Newsite/node2/node2245/node4516/node55031/node55113/node55129/userobject1ai42414.html|title=上海公用事业志|accessdate=2017-10-21|author=《上海公用事业志》编撰委员会）（上海市地方志办公室）|date=|work=www.shtong.gov.cn|publisher=上海社会科学院出版社}}</ref>。截至1997年，该线路配车已经空调化，并更改线路号码为936路<ref>{{Cite book|url=http://www.shtong.gov.cn/Newsite/node2/node4/node2249/node75735/node75749/node76320/node76326/userobject1ai91230.html|title=黄浦区续志|last=|first=|publisher=上海社会科学院出版社|year=2003|isbn=7806811672|location=上海|pages=}}</ref>。2006年1月25日开始，线路被调整到张杨北路。<ref name=":0" />

==== 线路首末站 ====
线路由夏碧路开始，到龙柏新村止。

==== 票价 ====
全程单价RMB¥2~5，有人售票，此时上海市尚未在公共汽电车线路上实施[[上海公共交通卡#上海本地优惠政策列表|公交换乘优惠]]<ref name=":1" />。

==== 线路走向 ====
在2007年被延伸以前，该线路运行走向是：
* 上行由夏碧路开始，经过[[杨高北路|杨高北路]]、[[杨高中路|杨高中路]]、[[杨高南路|杨高南路]]{{NoteTag|由停靠站点“华高苑”（杨高北路巨峰路）可知该线路走向|name="hgy"}}、[[峨山路|峨山路]]、[[东方路|东方路]]、[[浦东大道|浦东大道]]、[[世纪大道_(上海)|世纪大道]]、[[延安东路_(上海)|延安东路]]、[[延安中路_(上海)|延安中路]]、[[延安西路_(上海)|延安西路]]、[[虹桥路|虹桥路]]、程家桥路、延安西路、虹井路、红松路、黄桦路（黄桦路步行街建成后改行白樟路）、青杉路、龙柏新村终点站。
* 下行由龙柏新村开始，经过青杉路、兰竹路、红松路、虹井路、[[虹桥路|虹桥路]]、[[延安西路_(上海)|延安西路]]、[[延安中路_(上海)|延安中路]]、[[延安东路_(上海)|延安东路]]、[[世纪大道_(上海)|世纪大道]]、[[浦东大道|浦东大道]]、[[东方路|东方路]]、[[峨山路|峨山路]]、[[杨高南路|杨高南路]]、[[杨高中路|杨高中路]]、[[杨高北路|杨高北路]]{{NoteTag|name="hgy"}}、春晖路、张杨北路、夏碧路。

=== 2006年1月25日~2007年12月29日 ===
2006年1月25日起，调整后的936路起讫站为秋霞路张杨北路至龙柏新村。调整后线路走向为：自秋霞路起经张杨北路、夏碧路、杨高北路循原线至白樟路、青杉路；回程：自青杉路起循原线至杨高北路、夏碧路、季景路至秋霞路。线路调整后，首末班车时间也同步变更——秋霞路张杨北路为6:35—19:00，龙柏新村为5:00—17:00；唯华山路区间车首末班车时间不变。<ref name=":0">{{cite web|title=上海：公交547、936路25日首班车起调整线路|url=http://www.gov.cn/fwxx/sh/2006-01/25/content_170840.htm|website=www.gov.cn|accessdate=2017-10-28}}</ref>

2007年12月29日开始<ref name=":29">{{cite web|title=936为配合地铁6号线开通缩线至世纪公园地铁站|date=2007-12-27|website=www.84000.com|url=http://www.84000.com.cn/NewsDetail.aspx?actionNewsID=2263}}</ref>，为配合[[上海轨道交通6号线|上海轨道交通6号线]]开通，936路缩线到[[世纪公园站|世纪公园地铁站]]（梅花路）。<ref name=":jb" />

==== 线路首末站 ====
线路由张杨北路（秋霞路张杨北路）开始，到龙柏新村止。

==== 票价 ====
全程单价RMB¥2~5，有人售票，此时上海市逐渐开始在非空调车实施[[上海公共交通卡#上海本地优惠政策列表|公交换乘优惠]]，本线路未实施<ref name=":1">{{Cite web|url=http://news.sina.com.cn/o/2006-12-20/141310821573s.shtml|title=上海公交换乘优惠扩容_新闻中心_新浪网|accessdate=2017-10-28|work=news.sina.com.cn}}</ref>。

==== 线路走向 ====
相比于2006年及以前的线路，调整后线路走向为<ref name=":0" />：
* 上行：自秋霞路起经张杨北路、夏碧路、杨高北路循原线至白樟路、青杉路；
* 下行：自青杉路起循原线至杨高北路、夏碧路、季景路至秋霞路。

==== 首末班车时间 ====
线路调整后，首末班车时间也同步变更<ref name=":0" />：
* 秋霞路张杨北路：6:35—19:00
* 龙柏新村：5:00—17:00

===  2007年12月29日~2015年3月14日 ===
2007年12月29日起<ref name=":29" />，为配合[[上海轨道交通6号线|上海轨道交通6号线]]开通，936路以“复线优化”为由缩线到[[世纪公园站|世纪公园地铁站]]（梅花路）<ref name=":jb">{{cite news|title=多条公交线路调整尽量接驳轨交6号线|accessdate=2017-10-28|work=新闻晨报|agency=新闻报社|date=2007-12-28}}</ref>。

在此之后，线路改为：下行循原线、花木路、白杨路、梅花路、世纪公园地铁站；上行经过梅花路、花木路循原线<ref name=":2008" />。然而，自地铁开通以来，因为该地铁线路仅使用四节小运量（C车）车厢编组，故出现了严重的拥挤现象。与此同时，杨高路沿线的773、936、968等公交车均被同步取消或改道，乘客难以被疏导相邻其他道路的公交上去。有关部门的规划短视以及不当的公交调整举措受到沿线居民的抨击。<ref name="6hx">{{cite web|title=6号线为什么这样堵 早报记者体验最挤路段-轨交 6号线-东早都市—上海频道-东方网|url=http://dfdaily.eastday.com/d/20080103/u1a388635.html|website=dfdaily.eastday.com|accessdate=2017-10-28|date=2008-01-03}}</ref>

2010年1月，该线路上行增设延安西路娄山关路站，以配合[[长宁区|长宁区]]进一步发展虹桥开发区和虹桥商圈。这一举措，使得乘客乘坐上行方向在虹桥开发区下车后，无需再从遵义南路长距离行走以抵达位于娄山关路附近的虹桥开发区核心区。<ref>{{cite web|title=11条公交线路开进虹桥商圈_网易新闻|url=http://news.163.com/10/0127/00/5U0A8CV6000120GR.html|website=news.163.com}}</ref>

2015年3月14日0时开始，[[延安东路隧道|延安东路隧道]]开始大修。根据[[上海市公安局|上海市公安局]]交警总队的安排，936路缩线行驶，终点站改在[[人民广场_(上海)|普安路延安东路]]。<ref name="jbst" />

==== 首末站 ====
由世纪公园地铁站开始，到龙柏新村结束。

==== 收费 ====
全程票价RMB¥2~5，有人售票；2012年12月30日起改为单一票价RMB¥2，无人售票。

线路自2010年起实行对空调公交车的[[上海公共交通卡#上海本地优惠政策列表|公交换乘优惠]]。<ref>{{cite web|url=http://xjxjy.eastday.com/eastday/node3521/node144492/node194355/node228184/node228186/node228188/u1a3175751.html|title=上海公交刷卡优惠方案拟出台 2小时内换乘省5角|website=东方网}}</ref>

==== 线路走向 ====
在被缩线回浦西以前，线路走向为：<ref name=":2008" />
* 上行：下行经过[[梅花路_(上海)|梅花路]]、[[花木路|花木路]]、[[峨山路|峨山路]]、[[东方路|东方路]]、[[浦东大道|浦东大道]]、[[世纪大道_(上海)|世纪大道]]、[[延安东路_(上海)|延安东路]]、[[延安中路_(上海)|延安中路]]、[[延安西路_(上海)|延安西路]]、[[虹桥路|虹桥路]]、程家桥路、延安西路、虹井路、[[红松路_(上海)|红松路]]、白樟路（2008年~2009年修建[[上海轨道交通十号线|上海轨道交通十号线]]时，因红松路正在进行配套下水管道迁移，曾临时改道沿虹井路直行）、青杉路、龙柏新村终点站。
* 下行：下行由[[龙柏新村|龙柏新村]]开始，经过青杉路、兰竹路、红松路、虹井路、[[虹桥路|虹桥路]]、[[延安西路_(上海)|延安西路]]、[[延安中路_(上海)|延安中路]]、[[延安东路_(上海)|延安东路]]、[[世纪大道_(上海)|世纪大道]]、[[浦东大道|浦东大道]]、[[东方路|东方路]]、[[峨山路|峨山路]]、花木路、白杨路、梅花路到[[世纪公园站|世纪公园地铁站]]；

===  2015年3月14日~2016年12月3日 ===
[[File:上海公交936路龙柏新村站调度显示屏.jpg|thumb]]
2015年3月14日0时开始，[[延安东路隧道|延安东路隧道]]开始大修。根据[[上海市公安局|上海市公安局]]交警总队的安排，936路缩线行驶，终点站改在[[人民广场_(上海)|普安路延安东路]]，线路下行改为经原线、[[延安东路_(上海)|延安东路]]、黄陂南路、金陵东路、普安路到枢纽站，线路上行则改为自枢纽站开始经过普安路、延安东路、西藏南路、金陵东路、黄陂南路回原线；并改为定班车，间隔固定为30分钟<ref name="jbst" />。但是，除了935路{{NoteTag|当时就被暂停运营，截至2017年10月仍然未恢复，[[de_facto|事实上]]撤销}}、936路、983路{{NoteTag|该线路缩线在浦东。}}三条线路之外，其余大部分过[[黄浦江|江]]线路改道[[人民路隧道|人民路隧道]]{{NoteTag|唯有机场五线终点站在[[浦东机场|浦东机场]]，改走[[南浦大桥|南浦大桥]]。}}<ref>{{cite web|title=延安路隧道大修 交警总队发布交通组织方案_民生热线_新民网|url=http://shanghai.xinmin.cn/msrx/2015/03/04/26965007.html|website=shanghai.xinmin.cn}}</ref>。2015年12月25日，[[延安东路隧道|延安东路隧道]]大修提前结束，全线通车；但是936路仍“暂不恢复”<ref>{{cite web|title=延安东路隧道今天全线通车|url=http://app.why.com.cn/epaper/qnb/html/2015-12/25/content_279004.htm|website=app.why.com.cn}}</ref>，自此之后未能恢复，直到2016年12月3日为配套[[延安路中运量公交系统|延安路中运量公交系统]]而被撤销<ref name=":2" />。

==== 线路首末站 ====
线路由普安路延安东路开始，到龙柏新村止。

==== 收费 ====
截至该线路撤销，全程单一票价RMB¥2，无人售票，线路实行[[上海公共交通卡#上海本地优惠政策列表|公交换乘优惠]]。

== 未遂线路变更案 ==
=== 2011年撤销合并案 ===
2011年2月11日，[[上海市交通委员会|上海市城市交通和港口管理局]]公布了当年的第一批公交新辟及调整线路计划之市民意见征询稿，拟调整38条线路；由于该线路和延安路上例如[[上海公交1250路|71路]]、[[上海公交1251路|127路]]、[[上海公交925路|925路]]等线路重复，故在调整计划中提出了针对936路的撤线案，拟将936路撤销。<ref name="shzw">{{cite web|title=沪近百条公交线路将调整 市区公交网"鱼骨型"布局-上海政务|url=http://shzw.eastday.com/shzw/G/20130702/u1ai108166.html|website=shzw.eastday.com}}</ref>调整计划甫一公示，便引起市民、网友的强烈争议。据[[东方早报|东方早报]]报导，不少925路和936路乘客担心，现有线路的撤销将直接导致自己的出行成本上升<ref>{{cite web|title=925、936等公交撤销计划遭质疑 乘客担心成本上升_新浪上海_新浪网|url=http://sh.sina.com.cn/news/2011-02-15/0243172562.html|website=sh.sina.com.cn}}</ref>。最终，该计划中针对936路的部分无果而终。

=== 2013年缩线案 ===
2013年9月17日，[[上海市交通委员会|上海市交通港口局]]发布《2013年公交新辟及调整线路计划》进行公示，拟调整202条线路。根据方案，936路将不再从龙柏始发，调整为美丽园至世纪公园地铁站。缩线部分，由同线的公交57路承担，从而减少虹桥路、延安西路的重复线路，提高运营效率。但是，该方案中设计的内容后来未有实施，无果而终。<ref>{{cite web|title=上海调整202条公交线路 以共和新路等主干道为主_新浪上海|url=http://sh.sina.com.cn/news/b/2013-09-18/081562631.html?from=sh_ydph|website=sh.sina.com.cn}}</ref>

== 线路撤销 ==
{{see also|延安路中运量公交系统}}
2016年11月2日，上海市交通委员会发布《[[延安路中运量公交系统|延安路中运量系统]]公交线路优化调整计划》，征求市民意见<ref>{{cite web|title=《延安路中运量系统公交线路优化调整计划》网上征询市民意见的公告|url=http://www.jt.sh.cn/jtw/gsgg/n39/u1ai52106.html|website=上海交通网站|publisher=上海市交通委员会}}</ref>。根据该方案，由于和中运量线路重合度最高，936路将在[[上海公交71路|中运量71路]]开通前被撤销<ref>{{cite web|title=上海公布中运量公交运行路线 公交线网怎么调_房产上海站_腾讯网|url=http://sh.house.qq.com/a/20161125/008938.htm|website=sh.house.qq.com|language=zh-CN}}</ref>。2016年11月23日，[[巴士三公司|巴士三公司]]贴出公告，宣布线路将在12月3日暂停运营，936路就此谢幕<ref name=":2">{{cite web|author1=[[上海巴士第三公共交通有限公司|上海巴士第三公共交通有限公司]]|title=936路暂停营运|url=http://www.84000.com.cn/NewsDetail.aspx?actionNewsID=3202|website=www.84000.com.cn|archiveurl=https://web.archive.org/web/20171028161346/http://www.84000.com.cn/NewsDetail.aspx?actionNewsID=3202|archivedate=2010-10-29}}</ref>。
