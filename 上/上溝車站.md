'''上溝車站'''（{{jpn|j=上溝駅|hg=かみみぞえき|rm=Kamimizo eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[相模原市|相模原市]][[中央區_(相模原市)|中央區]]{{link-ja|上溝|上溝}}七丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[相模線|相模線]]的[[鐵路車站|鐵路車站]]。
== 車站結構 ==
[[側式月台|側式月台]]1面1線的[[高架車站|高架車站]]，站舍位於高架橋下。
== 使用情況 ==
