'''上海金融法院'''，是[[中华人民共和国|中华人民共和国]][[上海市|上海市]]的[[金融法院|金融法院]]，也是中华人民共和国首家金融法院。

==沿革==
2008年11月，中华人民共和国首家[[基层人民法院|基层人民法院]]金融审判庭在[[上海市浦东新区人民法院|上海市浦东新区人民法院]]成立。此后，[[上海市高级人民法院|上海市高院]]、中院及部分基层法院陆续成立专业的金融审判庭，统一管辖金融商事案件<ref name=nf/>。

2010年1月，在[[上海市政协|上海市政协]]十一届三次会议上，上海市政协委员[[张宁_(上海市政协常委)|张宁]]、[[吕红兵|吕红兵]]、[[谢荣兴|谢荣兴]]首次提交设立上海金融法院的提案。随后，上海市政协副主席[[朱晓明|朱晓明]]和[[高小玫|高小玫]]带着该提案赴[[上海市高级人民法院|上海市高院]]走访调研，上海市高院院长[[应勇|应勇]]对该提案评价很高，该提案由上海市高院承办，当年办理结果是“留作参考”<ref name=nf>{{Cite web |url =http://epaper.oeeee.com/epaper/A/html/2018-04/26/content_23809.htm  |title =  设立上海金融法院 政协委员连续呼吁8年|author =  |date = 2018-04-26 |publisher = 南方都市报 |language =  |accessdate =  |quote =  }}</ref>。

2015年1月24日，上海市政协十二届三次会议开幕，吕红兵执笔，张宁等11位上海市政协委员联署提交关于在上海设立金融法院的提案<ref name=nf/>，认为可借鉴[[上海知识产权法院|上海知识产权法院]]与[[上海市第三中级人民法院|上海市第三中级人民法院]]共同设立的先例，考虑将上海金融法院先行设立于[[上海市第一中级人民法院|上海市第一中级人民法院]]，在条件成熟时再考虑设立独立建制的上海金融法院<ref>{{Cite web |url =http://finance.people.com.cn/n/2015/0124/c1004-26443897.html  |title =存款丢了咋办？沪政协委员建言设立金融法院  |author =  |date =2015-01-24  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。上海市高院院长[[崔亚东|崔亚东]]也认为设立上海金融法院有必要，这一提案被列入上海市政协2015年重点协商办理的提案，办理结果是“列入计划拟解决”<ref name=nf/>。

2017年，上海市政协副主席[[李逸平|李逸平]]带队赴[[澳门|澳门]]、[[香港|香港]]调研金融审，调研组在专题报告中认为，上海建立一个专门的金融法院“非常必要”<ref name=nf/>。

2018年全国“[[两会|两会]]”期间，全国政协委员吕红兵联合[[上海外国语大学|上海外国语大学]]法学院副教授[[黄绮|黄绮]]等人再次提交“关于设立上海金融法院的建议”<ref name=nf/>。

2018年3月28日，[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军委主席|中央军委主席]]、[[中央全面深化改革委员会|中央全面深化改革委员会]]主任[[习近平|习近平]]主持召开中央全面深化改革委员会第一次会议并讲话，会议审议通过了《关于设立上海金融法院的方案》<ref>{{Cite web |url = http://www.xinhuanet.com/politics/leaders/2018-03/28/c_1122605838.htm |title = 习近平：加强和改善党对全面深化改革统筹领导 紧密结合深化机构改革推动改革工作 |author =  |date = 2018-03-28 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。

==管辖==
《最高人民法院关于上海金融法院案件管辖的规定》：

第一条 上海金融法院管辖上海市辖区内应由[[中级人民法院|中级人民法院]]受理的下列第一审金融民商事案件：
*（一）证券、期货交易、信托、保险、票据、信用证、金融借款合同、银行卡、融资租赁合同、委托理财合同、典当等纠纷；
*（二）独立保函、保理、私募基金、非银行支付机构网络支付、网络借贷、互联网股权众筹等新型金融民商事纠纷；
*（三）以金融机构为债务人的破产纠纷；
*（四）金融民商事纠纷的仲裁司法审查案件；
*（五）申请承认和执行外国法院金融民商事纠纷的判决、裁定案件。

第二条 上海金融法院管辖上海市辖区内应由中级人民法院受理的以金融监管机构为被告的第一审涉金融行政案件。

第三条 以住所地在上海市的金融市场基础设施为被告或者第三人与其履行职责相关的第一审金融民商事案件和涉金融行政案件，由上海金融法院管辖。

第四条 当事人对上海市[[基层人民法院|基层人民法院]]作出的第一审金融民商事案件和涉金融行政案件判决、裁定提起的上诉案件，由上海金融法院审理。

第五条 当事人对上海金融法院作出的第一审判决、裁定提起的上诉案件，由[[上海市高级人民法院|上海市高级人民法院]]审理。

第六条 上海市各中级人民法院在上海金融法院成立前已经受理但尚未审结的金融民商事案件和涉金融行政案件，由该中级人民法院继续审理<ref name=guanxia/>。
