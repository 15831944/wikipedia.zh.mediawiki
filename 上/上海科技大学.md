'''上海科技大学'''（{{lang|en|'''ShanghaiTech University'''}}，[[缩写|缩写]]作'''{{lang|en|ShanghaiTech}}'''；筹建时曾用名：'''上海高等研究大学'''<ref name="moe-reply">[http://www.moe.edu.cn/publicfiles/business/htmlfiles/moe/s5972/201205/135167.html 教育部关于同意筹建上海科技大学的通知]</ref>），[[简称|简称]]'''上科大'''，是主校区位于[[上海市|上海市]][[浦东|浦东新区]]张江高科技园区·中区的一所以理工科为主的公立[[普通高等学校|普通高等学校]]，由[[上海市人民政府|上海市人民政府]]和[[中国科学院|中国科学院]]共同举办、建设，由上海市人民政府主管。

== 院系设置 ==

学校的建设方向是一所小规模、高水平、国际化的研究型大学，分设五所学院、三所研究所以及一个通识教育中心。

=== 学院 ===

==== 物质科学与技术学院 ====

物质科学与技术学院的院长是[[美国艺术与科学院|美国艺术与科学院]]院士[[杨培东|杨培东]]。学院重点研究领域有：
* 材料发现、设计与合成（材料化学、生物材料、合成与工艺、催化、纳米、高分子、表面科学、固态化学、软硬材料界面，环境传感和恢复材料） 
* 凝聚态和材料物理（实验[[凝聚态物理|凝聚态物理]]、理论凝聚态物理、材料的物理行为、电子材料、储能材料、低维材料、[[量子材料|量子材料]]、超材料、材料理论） 
* 光谱与仪器科学（超快光谱学、散射和显微学、电子束和扫描探针技术及仪器、超快材料科学、原位电子显微学、原位超快光谱学、传感和检测）
* 材料系统工程
物质学院下设三个研究部：
* 光子科学与凝聚态物理研究部
* 材料生物学研究部
* 系统材料学研究部

==== 信息科学与技术学院 ====

信息科学与技术学院的创始院长是[[王雪红|王雪红]]。学院重点研究领域有：
* [[电子科学与技术|电子科学与技术]]（集成电路设计、物联网核心芯片、电力电子等）
* [[信息|信息]]与[[通信工程|通信工程]]（新一代无线通信网络、信息论等）
* [[计算机科学与技术|计算机科学与技术]]（计算机视觉、信息安全等）
信息学院下设多个研究中心<ref>{{Cite web |url=http://sist.shanghaitech.edu.cn/News.asp?mid=66 |title=Research Centers |access-date=2017-12-21 |archive-url=https://web.archive.org/web/20171222051922/http://sist.shanghaitech.edu.cn/News.asp?mid=66 |archive-date=2017-12-22 |dead-url=yes }}</ref>：
* 数据科学与机器智能研究中心
* 虚拟现实与视觉计算中心
* 网络科学中心
* 后摩尔时代器件与集成系统研究中心
* 先进电力与能源系统中心
* 自动化与机器人中心
* 计算机系统与安全中心

==== 生命科学与技术学院 ====

生命科学与技术学院的院长是[[林海帆|林海帆]]教授，执行院长是[[吴家睿|吴家睿]]教授，副院长是刘冀珑教授。学院重点研究领域有：
* [[表观遗传学|表观遗传学]]
* RNA生物学
* [[基因组学|基因组学]]/[[蛋白质组学|蛋白质组学]]/[[系统生物学|系统生物学]]
* [[干细胞生物学|干细胞生物学]]和再生医学
* 超分辨显微细胞学
* [[化学生物学|化学生物学]]
* [[药物研发|药物研发]]

==== 创意与艺术学院 ====

创意与艺术学院的创始院长是[[黎瑞刚|黎瑞刚]]，执行院长是杨旸教授，副院长是John McIntosh。开设与[[南加州大学|南加州大学]]合作的影视制作类非学历课程。

==== 创业与管理学院 ====

创业与管理学院的院长是方汉明，副院长为陆丁。

创管学院为学生开设的课程主要有创新创业和金融学两个方向，并从物质、信息、生命学院的在读本科生中招收这两个方向辅修专业的学生。

== 校园设施 ==

=== 主校区 ===

主校区又称张江校区，正门（南门）位于华夏中路393号，是张江中区内中国科学院上海[[浦东科技园|浦东科技园]]的主体部分。校区东临金科路，南临华夏中路（[[中环路|中环路]]），西近罗山路，北靠海科路。周边分布着[[中科院上海高等研究院|中科院上海高等研究院]]、[[国家蛋白质科学中心|国家蛋白质科学中心]]（筹）、[[中国商飞|中国商飞]][[上海飞机设计研究院|上海飞机设计研究院]]和中科院上海药物研究所。

校区由Moore Ruble Yudell建筑设计事务所提供设计咨询意见，该意见获得2012年AIACC颁发的城区设计奖<ref>[http://www.aiacc.org/2012/07/27/shanghai-advanced-research-inst-chinese-academy-of-sciences/ 2012 MERIT AWARD FOR URBAN DESIGN]</ref><ref>[http://old.shanghaitech.edu.cn/news.asp?id=163 上海科技大学校园规划设计荣获加州建筑师协会最佳城市设计奖]</ref>。实际最终设计由[[华东建筑设计院|华东建筑设计院]]、[[同济建筑设计院|同济建筑设计院]]、[[上海市政设计总院|上海市政设计总院]]共同完成。

张江校区采用地下连通的设计，各学院建筑群均设有联通地下车库的下沉式广场。校区内设有会议中心、学院大楼、体育馆、体育场、图书馆、书院宿舍、公共教室等。

物质学院、信息学院、创管学院与创艺学院的建筑被设计交汇在同一个十字路口，旨在促进学科交叉互动。而生命学院的L型楼与Y型楼在校区北侧，与国家蛋白质科学中心（上海）分居海科路两侧。

校区总建筑面积70.15万平方米，总投资41.69亿元，建设资金由上海市财力安排解决。其中新校区一期工程建筑面积58.78万平方米（教学与实验、体育等设施及辅助用房约28万平方米，学生宿舍和教师公寓约16万平方米，地下建筑约15万平方米），总投资35.03亿元；大学产学研基地（科技园二期）项目总建筑面积11.37万平方米，总投资6.66亿元。

=== 徐汇校区 ===

[[File:Yueyang_Road_Campus_of_ShanghaiTech_University.jpg|thumb]]

=== 过渡校区 ===

过渡校区的地址为海科路100号，为上海科技大学向中国科学院上海高等研究院租借，与主校区相通。

过渡校区内曾按大学需求分设行政楼、教学楼与科研楼。其中，餐厅、图书馆、教室、实验室等设备一应俱全。自2016年9月起，几乎所有的本科生教学活动都不再于过渡教学区进行，10月起，几乎所有的行政办公室都搬迁至新校区行政中心。自2018年1月起，为配合张江实验室建设，剩余的科研设施也陆续由过渡校区搬迁至主校区内。

== 学术 ==

=== 本科生培养 ===

本科生学年分为三个学期，秋季学期、春季学期与暑期小学期。其中，暑期小学期有短期的密集型专业课程、户外实践型课程、创新创业项目类课程、以及经典著作导读、外语、音乐、绘画课程可选。

本科生课程中有很大的比重是通识与跨学科课程，学校开设有一批原创性课程，如计算机科学导论、电子科技导论、现代生命科学导论，并在北京修远基金会支持下，与[[复旦大学|复旦大学]]、[[同济大学|同济大学]]、[[华东师范大学|华东师范大学]]合作开发了中华文明通论、世界文明通论、经典导读等课程。

自2017年起，随着校园内学生创新中心的开启，上科大也成为同济大学外，中国国内第一个落成“数制”工坊，并开设“智造学术”课程的高校。

创业与管理学院为所有专业的本科生提供经济学与设计思维方面的必修课与选修课，同时开设金融学和创新与创业两个辅修专业供高年级本科生修读。

== 招生 ==

=== 本科生招生 ===

上海科技大学本科招生纳入中国国家高等学校统一招生计划。2019年，学校面向[[北京|北京]]、[[上海|上海]]、[[山东|山东]]、[[江苏|江苏]]、[[浙江|浙江]]、[[江西|江西]]、[[福建|福建]]、[[河南|河南]]、[[四川|四川]]、[[贵州|贵州]]、[[云南|云南]]、[[辽宁|辽宁]]、[[天津|天津]]、[[湖北|湖北]]、[[陕西|陕西]]、[[湖南|湖南]]、[[甘肃|甘肃]]共17个省（市）进行本科招生。

上海科技大学列入高考提前批次招生，可直接报考。招生同时设有校园开放日，历时约24小时，分不同场次进行，每次内容大体相同但有细微差别，涉及个人的综合素质考核，并据个人材料与开放日表现给出A、B、C三种高考加分档次，在考生档案进入上科大后给予加分，依据加分后高考总分择优录取，并满足录取考生的第一志愿专业。<ref>[http://admission.shanghaitech.edu.cn/2019/0222/c916a39364/page.htm 上海科技大学2019年本科招生简章]</ref>在上海与浙江，要求考生高考选考科目中包含物理。在江苏，应省级主管部门要求，考生须参加校园开放日并获得加分方能报考。<ref>[http://www.jseea.cn/contents/channel_80/2019/03/1903151513405.html 省教育厅关于做好2019年普通高校综合评价录取改革试点工作的通知]</ref>

本科生招生专业为：物理学、化学、材料科学与工程、生物科学、生物医学工程、计算机科学与技术、电子信息工程。自2019年起，开设数学与应用数学专业，但仅接受通过其他专业录取入校后转入该专业的申请。

=== 研究生招生 ===

目前，上科大独立招收研究生。早期，上科大曾依托[[中国科学院大学|中国科学院大学]]，与中科院上海分院各研究所进行联合培养招生。

=== 讲座活动 ===

学校面向学生，开设有“文明之光”、“科技之光”、“艺术之光”、“产业之光”、“信仰的力量——上海市思政名师社会主义核心价值观进课堂”等系列讲座，[[James_E._Rothman|James E. Rothman]]、[[Elon_Musk|Elon Musk]]、[[库尔特·维特里希|Kurt Wüthrich]]、[[张益唐|张益唐]]等均在学校进行过讲座。学校还开设有更高规格的“上海科技大学讲座（ShanghaiTech Lecture）”，邀请过[[哈特穆特·米歇尔|哈特穆特·米歇尔]]、[[易纲|易纲]]、[[朱棣文|朱棣文]]、[[周小川|周小川]]等知名人士。
