'''CRH6型电力动车组'''是由[[中国|中国]][[中车青岛四方机车车辆|中车青岛四方机车车辆股份有限公司]]和[[中车南京浦镇车辆|中车南京浦镇车辆有限公司]]共同研制开发的[[和谐号电力动车组|CRH]]系列[[电力动车组|电力动车组]]。列車由四方技术总负责，浦镇四方联合设计，并分别在两公司及广东（新会）基地生产。CRH6型动车组适用于城市间以及市区和郊区间的短途[[通勤铁路|通勤客运]]，满足载客量大、快速乘降、快启快停的运营要求。

== 历史 ==
[[广东省|广东省]]是[[中国|中国]]国内最早提出建设[[城际轨道交通|城际轨道交通]]的省份，[[珠三角城际快速轨道交通|珠三角城际快速轨道交通]]的建设规划于2005年由[[中国国务院|中国国务院]]审议通过。2009年12月，原[[中国南车集团|中国南车集团]][[中车南京浦镇车辆|南车南京浦镇车辆有限公司]]旗下的[[中车南京浦镇城轨车辆|南京南车浦镇城轨车辆有限责任公司]]，与广东省东南城际轨道交通有限公司签署合同，为珠江三角洲城际轨道交通网络中的[[穗莞深城际轨道交通|穗莞深城际轨道交通]]及[[莞惠城际轨道交通|莞惠城际轨道交通]]提供车辆，合同总金额23.46亿元人民币<ref name=ifeng20091229>{{Cite web |url=http://finance.ifeng.com/roll/20091229/1644120.shtml |title=中国南车获国内首个城际列车项目订单 |accessdate=2010-12-23 |publisher=凤凰网财经 |date=2009-12-19 }}</ref>。按照合同，首列时速160公里等级的CRH6型城际列车将于2011年6月交付，而时速200公里等级的CRH6型城际列车将于2011年12月开始交付。然而，首列时速160km/h的CRH6F在2013年7月3号才下线<ref name=xinhuanet20130703>{{Cite web |url=http://news.xinhuanet.com/2013-07/03/c_116391878.htm |title=中国首列时速160公里城际动车下线 |publisher=新华网 |accessdate=2013-07-04 |date=2013-07-03}}</ref>。[[广珠城际轨道交通|广珠城际轨道交通]]日後亦采用此型号动车组。

== 技术特点 ==
CRH6型动车组采用8辆标准编组、编组长度201.4米。根据运输距离、站点和乘客群的不同，CRH6型动车组分为两大类型，运营速度分别为[[时速|时速]]200公里和160公里两个等级<ref name=jmrb20101217>{{Cite web |url=http://www.jmnews.com.cn/c/2010/12/17/02/c_1112842.shtml |title=新和谐号最多载客1488人 最高时速220公里 |accessdate=2010-12-23 |publisher=江门日报 |date=2010-12-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20101220055023/http://www.jmnews.com.cn/c/2010/12/17/02/c_1112842.shtml |archivedate=2010-12-20 }}</ref>。时速200公里的CRH6型动车组最高运营速度200公里/小时、试验速度220公里/小时，以“大站停”的模式运营；而时速160公里的CRH6型动车组最高运营速度160公里/小时、试验速度180公里/小时，以“站站停”模式运营<ref name=ifeng20091229 />。

CRH6型动车组具有轻量化大功率牵引设备，采用交流传动技术。由于通勤铁路站间距较小、客流量较大，因此CRH6型动车组具有快速停车启动、大载客容量的特点。时速200公里的车型由静止加速到200公里/小时需时183秒，加速距离6697米；时速160公里的车型由静止加速到160公里/小时需时102秒，加速距离2843米。列车采用微机控制的电空复合制动、高热容量的盘式制动装置，并按高减速度设计，确保动车组在最短的时间和最小的距离内实现快速停车；同时设有[[再生制动|再生制动]]，电制动功率最大发挥为牵引功率1.5倍，最大常用制动时产生的可再生电能可以100%回馈接触网。CRH6型动车组采用轻量化设计，降低运行能耗，轻量化转向架重量约比同类车型轻约25%，牵引设备重量平均约比同类车型轻43%。两种等级的CRH6型动车组均根据[[月台閘門|月台閘門]]设置需要，采用等距侧门设计<ref name=jmrb20101217 />。

=== CRH6A（200km/h） ===
[[File:CRH6A-0407_at_Pearl_River_West_Bridge_(20180924132130).jpg|thumb]]的CRH6A于[[珠江大桥|珠江大桥]]]]
CRH6A车型定员载客量557人（座席）<ref name=njrb20130507>{{Cite web |url=http://njrb.njdaily.cn/njrb/html/2013-05/07/content_63724.htm |title=车内站位比座位还多 车门大可快速上下客 |accessdate=2013-05-07 |publisher=南京日报 |date=2013-05-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20141221090839/http://njrb.njdaily.cn/njrb/html/2013-05/07/content_63724.htm |archivedate=2014-12-21 }}</ref>（双门中间车厢，一说554人）<ref name=smd20130516/>或477人（三门中间车厢）<ref>{{cite news|author=何颖晗|url=http://www.thepaper.cn/newsDetail_forward_1806343|title=上海金山铁路即将5岁，日均客流达3万人|publisher=[[澎湃新闻|澎湃新闻]]|date=2017-09-26|accessdate=2018-01-11}}</ref>，超员载客量1488人（按每平方米站立4人计算）。座位采用2+2布置、可调节座椅，局部设茶桌，端部设可翻转座椅；非端段部的车厢座椅编排与欧洲铁路车辆及大部分国铁车厢的软座較常用，全部座椅面向車廂中心的編排。另外1、3、5、7号车厢设置卫生间<ref name=smd20130516/>，列车采用真空集便器。

值得一提的是，CRH6A-4002和CRH6A-4502以及上海[[金山铁路|金山铁路]]、北京[[北京市郊铁路城市副中心线|副中心线]]用车的中间车厢为三门车厢，而其他的CRH6A车型均为双门车厢。

由[[广深铁路股份有限公司|广深铁路股份有限公司]]向四方厂定制的CRH6A-4132~4137采用与[[和谐号CRH2型电力动车组|统型CRH2型动车组]]相同的布局，头尾车厢不设驾驶室门，并设置一等座车（ZY），二等座车改为3+2布局，其中二等座中部空间位置设有茶桌采用2+2布局，所有座椅均可旋转。在5号车厢（ZEC）还设置[[餐车|吧台]]，亦设有残疾人士卫生间，车厢內部装潢与[[复兴号动车组|复兴号动车组]]相似。这一部分的CRH6A全部配属[[广深铁路股份有限公司|广深铁路股份有限公司]]广州东（石牌）动车所，担当[[广深动车组列车|广深动车组列车]]。

=== CRH6A-A（200km/h） ===
[[File:CRH6A-A_at_Xipu_Railway_Station.jpg|thumb]]上的CRH6A-A于[[犀浦站|犀浦站]]]]
CRH6A-A车型为CRH6A的4辆小编组版本，最大载客量达688人。该车型针对城际铁路站间距小的特点，提升了加减速能力，能够像地铁列车一样“快起快停”。列车还采用大宽度的双开塞拉门，并增加车门数量，保障乘客可以快速上下车，缩短了停站时间。车厢连接处贯通道的宽度达到1.1米，而且座椅采用“2+2”布置，可以使车内的通行空间更加宽敞。此外，CRH6A-A还设有[[Wi-Fi|Wi-Fi]]接入。该车型既可实现4辆编组单独运营，也可实现8辆编组重联运行<ref name=dzwww>{{cite web |url=http://www.dzwww.com/xinwen/guoneixinwen/201709/t20170909_16402658.htm|title=我国首例小编组城际动车组在中车四方下线|accessdate=2017-09-11|publisher=央广网 |date=2017-09-09 }}</ref>。

=== CRH6F（160km/h） ===
[[File:201712_CRH6F-0433_at_Changshaxi_Station.jpg|thumb]]使用的CRH6F列车停靠在[[长沙西站|长沙西站]]]]
CRH6F车型定员载客量达1502人（包括座席及站席，按每平方米站立4人计算），超员载客量达1998人（包括座席及站席，按每平方米站立6人计算）。列车座位同样采用2+2布置，但座椅不可调节或翻转；列车在3、6号车设[[洗手間|卫生间]]。与CRH6A不同，车门采用宽阔的对开塞拉门，每节车辆侧设有3个塞拉门（头尾车辆有2个，其中一个为驾驶室门）。该车牵引制动性能比CRH6A更优、载客量更大，更适合较短站间距的城际线路和站站停模式使用。

==问题==

===空调问题===
CRH6A大致沿袭了[[CRH2A|CRH2A型动车组]]的牵引动力系统设计，故列车每次通过[[分相区|分相区]]的时候，为广播、照明和空调等系统供电的[[辅助动力系统|辅助电源装置]]都会停电一段时间，直到[[受电弓|受电弓]]通过无电区以后才恢复。

但由于本车型主要运用于速度目标值较低、停站较为密集、分相区集中在车站附近的短途城际铁路中，故通过分相区的时间较长；同时本车型载客量大，在车厢满载乘客甚至超员时，空调系统的突然暂停会导致车厢迅速升温，以车门处最为严重。因此在本车型的实际运用中，分相区空调暂停对乘客体验的影响较为显著。

===卫生间问题===

CRH6仅在1、3、5、7号车厢设置卫生间，卫生间比其他系列的CRH动车组少一半，令不在这些车厢的乘客需要步行较长的距离前往卫生间。在高峰期，车厢的走道站满乘客，若是前往卫生间，需要穿越大量乘客才能到达卫生间，不仅不方便且有发生踩踏的风险。而且无障碍卫生间设于1号车厢，令在其他车厢的残疾人士更加难以前往。

===多用途空间的问题===

CRH6在车门靠两车厢接驳位的一侧设有3个折叠座椅，既可当临时座位使用，亦可作为轮椅人士的停泊位。在实际运营中，不少乘客会坐满这三个座位，令轮椅无法停泊，而CRH6的走道比其他CRH动车组的要狭窄一些，轮椅无处安放，令残疾人士甚为不便。
