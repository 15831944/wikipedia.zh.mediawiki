'''和平里站'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[香河园街道|香河园街道]]境内。2002年拆除。

==简介==
和平里站建于1917年。1969年[[望和支线|望和支线]]建成通车后，和平里站成为望和支线的终点站，是北京市区东北部主要货运站。截至1990年代中期，和平里站占地面积16万平方米，建筑面积2700平方米。站台呈南北走向。和平里站设有基本站台、货物线、仓库、货场、装卸机械等站区设施。有通往北京造纸一厂的专用铁路线。从和平里站北行与[[京包铁路|京包线]]接轨，折向南经[[望京站_(铁路)|望京站]]、[[星火站|星火站]]后转向东沟通[[京承铁路|京承线]]、[[京秦铁路|京秦线]]、[[京坨铁路|京坨线]]，转向南经[[百子湾站_(铁路)|百子湾站]]沟通东南环铁路线、[[双丰铁路|双丰线]]。南行至北京造纸一厂为专用铁路线终端<ref name=cyqz>{{Cite web |url =  http://www.bjchy.gov.cn/chaoyang/qznj/8a24f09a24d90f1f0124d94c73fe0019.html |title =朝阳区志第七编交通邮电  |author =  |date = 2009-11-09 |publisher =北京朝阳  |language =  |accessdate =  |quote =  }}</ref>。

1999年，北京城市轻轨铁路（即今[[北京地铁13号线|北京地铁13号线]]）开工建设。这条轻轨铁路是利用[[京包铁路|京包铁路]]、东北铁路内环线、望和支线这3段既有铁路的线路位置及其交通走廊建设<ref>北京城市轻轨正式破土动工，城市公共交通2000年01期</ref>。由此和平里站在2002年左右被拆除，原址改建为住宅小区，西侧为13号线[[光熙门站|光熙门站]]。
