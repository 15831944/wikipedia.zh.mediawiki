'''和谐号CRH380A型电力动车组'''，或称'''CRH2-380型'''，是原[[中华人民共和国|中华人民共和国]][[中华人民共和国铁道部|铁道部]]为营运新建的[[中国高速铁路|高速城际铁路及客运专线]]，由[[南车青岛四方机车车辆|中车青岛四方机车车辆股份有限公司]]在以原型为日本新干线[[新干线E2系电力动车组|E2系1000番台]]的[[和谐号CRH2型电力动车组|CRH2型]]电力动车组基础上改款的[[中国铁路高速|CRH]]系列高速[[动车组|动车组]]，最高营运速度350公里/小时。中国铁路总公司将所有引进国外技术、联合设计生产的[[中国铁路高速|中国铁路高速]]（CRH）车辆均命名为[[和谐号|和谐号]]。CRH380A系列为[[动力分散式|动力分散式]]、[[变频器|交流传动]]的电力动车组，采用了[[鋁合金|鋁合金]]空心型材车体。

== 概要 ==
[[File:Sifang_CRH2_380A_at_Shanghai_Expo.jpg|thumb]][[中国铁路馆|中国铁路馆]]展出的CRH380A头车]]
2009年6月，[[中华人民共和国铁道部|中华人民共和国铁道部]]向国内动车组制造企业招标采购共320列时速350公里的高速动车组<ref>{{Cite web |url=http://www.chinaitc.com.cn/389-1050-11823.aspx|title=铁道部时速350公里速度级铁路动车组项目招标公告 |accessdate=2011-11-21 |publisher=中技国际招标公司 |date=2009-06-10 }}</ref>，[[南车青岛四方机车车辆|南车青岛四方机车车辆股份有限公司]]为中标厂商之年9月28日，铁道部[[武汉铁路局|武汉铁路局]]与四方机车车辆在北京签署了140列时速350公里速度级动车组采购合同，包括100列16节长编组，及40列8节短编组的高速动车组订单，合同总金额约值450亿元[[人民币|人民币]]<ref>{{Cite web |url=http://finance.sdnews.com.cn/2009/10/1/820838.html |title=450亿元动车订单落青岛 时速350公里明年下线 |accessdate=2010-09-20 |publisher=山东新闻网 |date=2009-10-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20091004040114/http://finance.sdnews.com.cn/2009/10/1/820838.html |archivedate=2009-10-04 }}</ref><ref>{{Cite web |url=http://www.railcn.net/news/railway-express/105245.html |title=中国南车获铁道部巨额合同 |accessdate=2010-07-14 |publisher=中国铁道网 |date=2009-10-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20101122094324/http://www.railcn.net/news/railway-express/105245.html |archivedate=2010-11-22 }}</ref>。

南车四方机车车辆的时速350公里级别高速动车组研制项目名称为'''CRH2-380'''（或称'''CRH2-350'''），是在'''CRH2C'''第二阶段的基础上进行研发。持续运营时速为350公里，最高运营时速为380公里，最高试验时速400公里以上。由于CRH2C型动车组只是在时速250公里的CRH2-250型动车组（CRH2A）基础上加大牵引功率，仅仅满足目前最高运营时速350公里的要求，难以满足[[京沪高速铁路|京沪高速铁路]]上时速380公里的营运要求，因此CRH2-380需要在CRH2C的基础上全面提升列车整体性能，对动车组的牵引系统、空气动力外形作出了较大的改变<ref name=jhgstl>{{cite book |author= 张曙光 |authorlink= |coauthors= |title=《京沪高速铁路系统优化研究》 |year=2009 |publisher=中国铁道出版社 |location=北京 |isbn=9787113095178 }}</ref>。

[[中华人民共和国科技部|中华人民共和国科技部]]与原[[中华人民共和国铁道部|铁道部]]于2008年2月26日共同签署了《中国高速列车自主创新联合行动计划》，而CRH2-380型高速动车组正是该联合行动计划最重要的项目，2009年正式立项启动了列入“十一五”国家科技支撑计划的中国高速列车关键技术及装备研制重大项目。在分析[[京津城际铁路|京津城际铁路]]等高速铁路积累的大量数据和经验的基础上，铁道部提出了新一代动车组的性能提升方向、技术方案，并对方案进行了大量的分析计算，形成了一整套的列车系统设计方案和各子系统优化设计方案<ref>{{Cite web |url=http://www.qingdaonews.com/content/2009-06/06/content_8061039.htm|title=最快动车组青岛造 明年下线设计时速380公里 |accessdate=2012-04-10 |publisher=青岛新闻网|date=2009-06-06 }}</ref>。

为了预先获得CRH2-380型动车组新头型的空气动力效能和实车试验数据，铁道部于2009年决定将CRH<sub>2</sub>-150C作为CRH2-380的试验实体样车，改为使用下一代的新头型。车身仅标示为“试验车 CRH380A”的新头型CRH2C第二阶段动车组于2010年4月底下线，并在5月起开始在[[郑西客运专线|郑西客运专线]]试验。2010年5月，南车四方机车车辆又定制了一节CRH2-380A头车的1：1实体模型，在[[上海世博会|上海世博会]][[中国铁路馆|中国铁路馆]]展出，并亮相到世博会闭幕为止<ref>{{ cite web | url =http://www.tianya.cn/new/publicforum/Content.asp?idWriter=0&Key=0&strItem=worldlook&idArticle=265698 | title = 中国新一代超高速动车CRH380世博会亮相 | date = 2010-04-25 | publisher = [[天涯社区|天涯]] | accessdate = 2010-05-28}}</ref>。这辆模型后来于2010年12月参展于北京举行的[[世界高速铁路大会|第七届世界高速铁路大会]]，陈列于[[国家会议中心_(中国)|中国国家会议中心]]外，并在高铁大会结束后被送返南车四方位于青岛的厂房临时封存2个月，之后的“十一五”国家重大科技成就展在北京举行，模型重新涂装后，再一次送到[[国家会议中心_(中国)|中国国家会议中心]]外。

2010年9月，铁道部下发《关于新一代高速动车组型号、车号及坐席号的通知》，正式将四方机车车辆的CRH2-380型动车组型号名称更改，其中8辆编组的动车组被命名为'''CRH380A'''，而16辆编组的动车组被命名为'''CRH380AL'''。

由2014年7月1日起，所有CRH380动车组編號均作出了更改，原編號為（CRH380A-6001～CRH380A-6040、CRH380A-6041L～CRH380A-6140L、CRH380A-6141～CRH380A-6196），現改為（CRH380A-2501～CRH380A-2540、CRH380AL-2541～CRH380AL-2640、CRH380A-2641～CRH380A-2696），該編號之後的生產动车组均依照新規定之格式定編號<ref name="铁总运(2014)150号">{{Cite web|url=http://www.ditiezu.com/thread-375095-1-1.html|title=《中国铁路总公司关于印发动车组型号、车号和坐席号编号规则的通知》|accessdate=2015-09-22|work=中国铁路总公司|publisher=中国铁路总公司|date=2014-06-13}}</ref><ref name="铁总运(2014)150号-weibo">{{Cite web|url=http://weibo.com/1736649017/BaPm4rmfQ|title=《中国铁路总公司关于印发动车组型号、车号和坐席号编号规则的通知》|accessdate=2015-09-22|work=中国铁路总公司|publisher=中国铁路总公司|date=2014-06-13}}</ref>。

CRH2-380A列车总数为40列（CRH380A-2501～CRH380A-2540），采用6动2拖的编组方式，牵引功率为9600千瓦，使用DSA350型高速[[受电弓|受电弓]]，以及在受电弓的两侧为挡板。列车设有带一等包厢座位的一等座车（ZY）2辆、二等座车（ZE）3辆、带观光座的二等座车（ZEG）2辆和带酒吧的二等座车（ZEC）1辆。其中一等座采用2+2方式布置，二等座为2+3布置。除了带酒吧的二等座车、一等包厢座位外，其他车厢所有座位均能旋转。列车设有观光座定员12人，一等包座定员6人，一等座定员89人，二等座定员373人，全列定员480人。

2010年9月，CRH380A-2501～CRH380A-2510相继开始配属[[上海铁路局|上海铁路局]]，及进入[[沪宁高速铁路|沪宁高速铁路]]和[[沪杭高速铁路|沪杭高速铁路]]进行高速试验。2010年9月28日，CRH380A在沪杭客运专线进行高速试验，并先后两次刷新“中国铁路第一速”，并刷新了世界上“正常营运编组列车最高试验速度”（特殊改造列车的最高试验速度为[[法国高速列车|法国高速列车]]V150的574.8公里/小時）。当日上午10时40分，由上海虹桥开往杭州方向的CRH380A-2501列车（车次G55001）的试运行时速达到413.7公里/小时<ref>{{Cite web |url=http://news.xinhuanet.com/politics/2010-09/28/c_12615068.htm |title=沪杭高铁试运行时速达到413.7公里 再次刷新世界纪录 |accessdate=2010-09-28 |publisher=新华网 |date=2010-09-28 }}</ref>。不到一小时后的11时37分，由杭州折返到上海虹桥试运行（车次G55002）途中，最高时速达到416.6公里/小时<ref>{{Cite web |url=http://news.qq.com/a/20100928/001085.htm |title=沪杭高铁试运行 时速416.6公里全程仅40分钟 |accessdate=2010-09-28 |publisher=腾讯新闻网 |date=2010-09-28 }}</ref>。

为应付十一黄金周期间来往南京和上海的往返客流，上海铁路局在2010年9月30日、10月1日、10月7日三天，安排CRH380A首次载客运营，在[[沪宁高速铁路|沪宁高速铁路]]担当临时高速动车组旅客列车任务，为此车型第一次载客运营<ref>{{Cite web |url=http://www.js.xinhuanet.com/xin_wen_zhong_xin/2010-10/07/content_21065008.htm |title=南京火车站今日客流量或超10万 将增开15趟临客 |accessdate=2010-10-08 |publisher=新华网 |date=2010-10-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20160604163702/http://www.js.xinhuanet.com/xin_wen_zhong_xin/2010-10/07/content_21065008.htm |archivedate=2016-06-04 }}</ref>。2010年10月26日起，CRH380A正式在[[沪杭客运专线|沪杭客运专线]]投入服务。2010年12月3日起，CRH380A开始在[[武广客运专线|武广客运专线]]投入运营<ref>{{Cite web |url=http://www.chinanews.com.cn/cj/2010/12-02/2696911.shtml |title=新一代高速动车组CRH380A明起亮相武广高铁 |accessdate=2010-12-03 |publisher=中国新闻网 |date=2010-12-02 }}</ref>。2011年3月8日起，由于[[博鳌亚洲论坛|2011博鳌亚洲论坛]]的举行，2组CRH380A开始在[[海南东环铁路|海南东环铁路]]运行，但相关车体已于2016年转配北京铁路局。

2011年3月1日，CRH400A（CIT400A）高速综合检测车由南车青岛四方研制成功，于3日起奔赴京沪高铁对线路状态进行一系列检测，以确保[[京沪高速铁路|京沪高速铁路]]的安全可靠运营。该检测车的设计时速为400公里，以南车四方CRH380A新一代高速列车技术平台为基础，历时8个月研制成功。

=== 統型CRH380A ===

CRH380AL列车总数为113列（CRH380AL-2541～CRH380AL-2640、CRH380AL-2913～CRH380AL-2920、CRH380AL-2926～CRH380AL-2930），采用了14动2拖的编组方式，牵引功率为20440千瓦，7个动力单元，56台[[牵引电动机|牵引电动机]]，使用DSA350型高速[[受电弓|受电弓]]，以及在受电弓的两侧为挡板。列车设有带VIP座席的商务车（SW）1辆、一等座车（ZY）2辆、二等座车（ZE）10辆、带观光座的一等座车（ZYG）2辆和餐车（CA）1辆。其中一等座采用2+2方式布置，二等座为2+3布置，商务车和观光座为1+2布置。除了带酒吧的二等座车外，其他车厢所有座位均能旋转。前期列车商务座定员28人、一等座定员162人，二等座定员838人，全列定员1028人。后期列车商务座定员26人、一等座定员112人，二等座定员923人，全列定员1061人。

2010年10月底，CRH380A-6041L在南车青岛四方股份的青岛厂房下线；同年11月8日，被运送至[[中国国家铁道试验中心（东郊分院）|北京环行铁道]]进行安装试验设备及初步试验，随后在11月20日前往京沪高速铁路先导试验段（[[枣庄|枣庄]]至[[蚌埠|蚌埠]]）开始进行正式线路联调联试和综合试验，并在26日首次进行时速380公里的高速试验。2010年12月3日，CRH380A-6041L动车组在京沪高铁进行冲高速试验，由铁道部长[[刘志军|刘志军]]、铁道部总工程师[[何华武|何华武]]、副总工程师[[张曙光_(铁道部)|张曙光]]添乘<ref>{{cite web|author1=新闻联播|title=京沪高速线CRH380AL冲高486.1km/h+立鸡蛋|url=https://www.bilibili.com/video/av18381889?from=search&seid=3667137603372688193}}</ref>。当日上午11时6分，列车从[[枣庄站|枣庄站]]出发。11时28分，列车在[[宿州东站|宿州东站]]附近达到486.1公里/小时的最高运行时速<ref>{{Cite web |url=http://news.xinhuanet.com/fortune/2010-12/03/c_12844439_1.htm |title=和谐号CRH380A打破此前的世界铁路最高运营速度纪录 |accessdate=2010-12-03 |publisher=新华网 |date=2010-12-03 }}</ref>；11時39分到达[[蚌埠南站|蚌埠南站]]，列车以34分钟运行了220公里，全程平均时速达388公里/小时。这是继9月28日CRH380A型动车组在沪杭客运专线试运行创下时速416.6公里之后，中国高铁再次刷新“正常营运编组列车最高试验速度”；[[日本国土交通省|日本国土交通省]]亦表示，CRH380A型动车组已经成功超越了日本在1996年由[[新幹線955型電聯車|300X动车组]]创下的443公里/小时速度纪录<ref>{{Cite web |url=http://china.kyodo.co.jp/modules/fsStory/index.php?sel_lang=schinese&storyid=88048 |title=中国和谐号速度赶超日本新干线 |accessdate=2010-12-03 |publisher=共同社 |date=2010-12-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20101206134020/http://china.kyodo.co.jp/modules/fsStory/index.php?sel_lang=schinese&storyid=88048 |archivedate=2010-12-06 }}</ref><ref>{{Cite web |url=http://news.163.com/10/1203/11/6MVOUDUD00014JB5.html |title=京沪高铁试验时速486.1公里创世界铁路最高速 |accessdate=2010-12-03 |publisher=网易 |date=2010-12-03 }}</ref>。而在列车开始高速试验前，当日上午10时35分在[[徐州东站|徐州东站]]举行了“和谐号CRH380A新一代高速列车上线仪式”。

== 技术特性 ==

* 低阻力流线头型：南车四方机车车辆股份共设计了20种列车新头型方案，经过进行了气动阻力、气动升力、侧向力、隧道效应等大量的空气动力学的仿真计算，并通过三维流场数值仿真分析和多目标优化，进行了17项、75次仿真计算，确定了5种备选头型。继而又对备选方案制作1：8模型，分别进行了19个角度、8种风速的风洞气动力学实验和3种风速、4种编组的风洞噪声试验，对择优选出的方案进行了样车试制，完成了22项试验验证<ref>{{Cite web|url=http://www.railways.com.cn/ShowPub.aspx?id=172|title=中国南车：创新是一份沉甸甸的责任|accessdate=2010-09-28|publisher=|date=}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，经大量的比對、計算、試驗之後，最终確定了新一代高速列車的頭型方案<ref>{{Cite web |url=http://www.gov.cn/jrzg/2010-08/07/content_1673086.htm |title=创新中打造“中国名片” 中国南车在国际上拓展品牌 |accessdate=2010-09-28 |publisher=中国政府网 |date=2010-08-07 }}</ref>。实际运行时新头型的阻力系数小于0.13，尾车[[升力系数|升力系数]]小于0.08<ref name=IRJ>{{cite journal| author = | title = First Chinese designed HS train breaks cover| journal = International Railway Journal| volume = | issue = | pages = | publisher = | issn = | url = http://findarticles.com/p/articles/mi_m0BQQ/is_9_50/ai_n55394441/| language = | format = | accessdate = 2010-11-05| date = 2010年9月| archive-url = https://archive.is/20120708180407/findarticles.com/p/articles/mi_m0BQQ/is_9_50/ai_n55394441/| archive-date = 2012-07-08| dead-url = yes}}</ref>。
* 振动模态系统匹配：优化了转向架设计参数并改善车厢内部结构，以配合动车组车体的自然震动频率，有效抑制列车在高速运行时的车体结构性[[共振|共振]]，同时提高了乘坐舒适度。
* 高强度气密性：由于列车运行时速提高到380公里/小时，为满足两列动车同时双向通过隧道的气密需要，CRH380A型动车组进一步提升了气密性，车厢采用差压控制模式的全密封加压。车厢内压力從4000[[帕|帕]]下降到1000帕实际大于180秒，气压变化值小于200帕/秒<ref name=IRJ />。
* 高速转向架：CRH380A型动车组使用SWMB-400/SWTB-400型无摇枕[[转向架|转向架]]，由CRH2C第二阶段使用的SWMB-350/SWTB-350改良而来。两者相比，新的转向架增加了抗侧滚扭杆，带两组抗蛇行减震器，加强了二系悬挂空气弹簧柔度，提高了转向架的稳定性和减震效果，满足转向架临界失稳速度达550公里/小时的指标要求<ref name=jhgstl />。中国与[[欧盟|欧盟]]的列车脱轨系数安全标准是小于或等于0.8，实验结果表示，当CRH380A型动车组运行速度为386.3公里/小时，其最大脱轨系数为0.34，而CRH2A型动车组以250公里/小时运行时最大脱轨系数为0.72<ref name=IRJ />。
* 噪声控制技术：列车采用各种新型噪音吸收和阻隔技术材料，CRH380A型动车组在时速350公里/小时的情况下车厢内噪声保持67至69分贝，与CRH2A型动车组以250公里/小时运行时的情况相若。而低阻力新头型的使用亦减少超过5%的气动噪音<ref name=IRJ />。
* 车辆减重：由于列车牵引动力、结构质量、减噪水平的提高，车辆重量相应增加，但轴重仍维持在15吨的水平。当CRH380A动车组维持380公里/小时的旅行速度时，平均每位旅客的每百公里能量消耗小于5.2[[千瓦小时|千瓦小时]]<ref name=IRJ />。
* 高效率[[再生制动|再生制动]]：再生能量回馈电网效率达到90%<ref name=IRJ />。

== 车内布置 ==
列车内部的旅客空间“航空化”，座位号除了数字外还增加了英文字母。二等座每列设有“3+2”方式排列五个座位，以“A、B、C、D、F”代表；一等座每列设有“2+2”方式排列四个座位，以“A、C、D、F”代表。无论是一等车厢还是二等车厢，带字母“A”和“F”的座位靠窗，带字母“C”和“D”代表贴近中间走道。以二等车厢第四排的座椅为例，座号“4A”、“4F”靠窗，“4C”、“4D”临过道，4B为4A、4C中间一个座位。旅客在车站窗口、代售点购票时，可自行决定靠窗、靠过道与否。

与CRH3型动车组一样，CRH380A型动车组的驾驶室后面也有一个VIP包间，称为观光区。头车和尾车各设一个观光区，每个内设6个座椅（商务座则为5个），观光区的座椅亦有扬声器按钮，可通过这个按钮控制车内广播音量。CRH380A型电力动车组发车后的报站铃声和[[CRH2|CRH2]]相同，采用的是与日本[[东武铁道|东武铁道]][[东上线|东上线]][[常盘台站|常盘台站]]、[[和光市站|和光市站]]、[[成增站|成增站]]、[[朝霞站|朝霞站]]、[[朝霞台站|朝霞台站]]、[[四季站|四季站]]、[[富士见野站|富士见野站]]、[[川越站|川越站]]等车站相同的[[发车音乐|发车提示音]]，同样使用这段提示音的还有[[北京地铁4号线|北京地铁4号线]]<ref>{{cite web|url=http://www.2427junction.com/chinahighcrh2int.html|title=ＣＲＨ２車内設備|language=日语}}</ref><ref>{{cite web|url=http://www.tudou.com/programs/view/kHxyImlQnnw/|title=东武东上线发车音2|deadurl=yes|archiveurl=https://archive.is/20160506084402/http://www.tudou.com/programs/view/kHxyImlQnnw/|archivedate=2016-05-06}}</ref><ref>{{cite web|url=http://v.youku.com/v_show/id_XMzg2Mzk4NjEy.html?from=s1.8-1-1.2|title=地鐵圓明園站}}</ref>及[[北京地铁16号线|北京地铁16号线]]<ref>https://www.youtube.com/watch?v=nKRDQOcnKM4</ref>各站的站厅广播。

* ZEC：二等座車／餐車（Second Class Coach／Dining Car）

* ZES：二等／商务座车（Second Class／Business Coach）

;拉丁字母意思
* Z：Zuo（[[汉语拼音|拼音]]），座，座车
* Y：Yi（[[汉语拼音|拼音]]），一，一等
* E：Er（[[汉语拼音|拼音]]），二，二等
* C／CA：Can（[[汉语拼音|拼音]]），餐，餐车
* G：Guan（[[汉语拼音|拼音]]），觀，觀光
* S／SW：Shang（[[汉语拼音|拼音]]），商，商务
* T：Te（[[汉语拼音|拼音]]），特，特等
