'''和谐号CRH380B型电力动车组'''（或称'''CRH3-380型'''），是[[中华人民共和国|中华人民共和国]][[铁道部|铁道部]]为营运新建的[[中國高速鐵路|高速城际铁路及客运专线]]，由[[中国北车集团|中国北车集团]][[唐山軌道客車|唐山轨道客车有限責任公司]]和[[长春轨道客车|中国北车集团长春轨道客车有限责任公司]]在原型为德国西门子[[ICE-3列車|ICE3]]的[[和谐号CRH3型电力动车组|CRH3C型电力动车组]]基础上改款的[[中国铁路高速|CRH]]系列高速[[動車組|動車組]]，并以此为基础制造了时速400公里的[[和谐号CRH380C型电力动车组|CIT400B]]线路检测车。中国鐵道部将所有引进国外技术、联合设计生产的[[中国铁路高速|中国铁路高速]]（CRH）車輛均命名為「'''[[和谐号|和谐号]]'''」。

== 概要 ==
为[[京沪高速铁路|京沪高速铁路]]配套的新一代高速列车的开发是在消化吸收基础上一项庞大的再创新工程，当局为全面支撑中国高速列车技术＂自主创新＂的需求，[[中华人民共和国科技部|中华人民共和国科技部]]与[[中华人民共和国铁道部|铁道部]]于2008年2月26日共同签署了《中国高速列车自主创新联合行动计划》，提出研制符合京沪高铁运营需求的高速列车，由[[中国南车集团|中国南车]]、[[中国北车集团|北车集团]]在消化吸收外国相关技术的基础上，将动车组的时速从250～300公里提高到350公里及以上，为京沪高速铁路提供强而有力的装备保障。<ref>{{Cite web |url=http://xwcb.eastday.com/c/20080314/u1a410708.html |title=京沪高铁虹桥站年内开建 将成上海第一大火车站 |accessdate=2010-12-03 |publisher=新闻晨报 |date=2008-03-14 }}</ref>。其中，中国北车集团唐山轨道客车以CRH3型电力动车组为基础，研制新一代时速350公里级别高速动车组。2009年正式立项启动，当时研制项目名称为'''CRH3-350'''。

2008年9月，CRH3-350型动车组开始前期研发阶段，同时提高了车辆的设计标准，设计最高时速由时速350公里提高至380公里，项目名称为'''CRH3-380'''；列车在京沪高速铁路上的单程行车时间可由原规划的5小时，进一步缩短至4小时<ref>{{Cite web |url=http://www.railcn.net/news/railway-express/68405.html |title=京沪高铁国产列车 唐山投产　全球最快轮轨车 时速 |accessdate=2010-12-03 |publisher=铁道网 |date=2008-09-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20140227184314/http://www.railcn.net/news/railway-express/68405.html |archivedate=2014-02-27 }}</ref>。更改标准后，新一代动车组的产品设计工作于2009年1月27日正式启动。

2009年3月16日，[[中华人民共和国铁道部|中华人民共和国铁道部]]正式与中国北车集团旗下的[[唐山轨道客车|唐山轨道客车公司]]和[[长春轨道客车|长春轨道客车股份公司]]、以及[[中国铁道科学研究院|中国铁道科学研究院]]签署了一系列采购合同，合作生产100列新一代高速動車組，为16節車廂的大編組座車，合同总金额达392亿元人民币。其中70列是由[[唐山轨道客车|唐山轨道客车]]制造，其餘30列則由[[长春轨道客车|长春轨道客车]]制造<ref name=sina0402>{{cite web | url = http://news.sina.com.cn/c/2009-04-02/083617533481.shtml| title = 西门子将向中方提供百列高速列车 不卖核心技术| date = 2009-04-02| publisher = 新浪網| accessdate = 2010-05-28}}</ref>。

由于此次签订合同的消息中中国官方并没有提到[[西门子|西门子]]的参与，次日铁道部仅在网站上发表“百列国产新一代高速动车组将于2011年驶上京沪高铁”为主题的新闻。至3月20日，德国西门子集团在其网站发布信息称，“唐山轨道客车有限责任公司、长春轨道客车股份有限公司、中国铁道科学院和西门子签署了一份关于提供100列高速列车的合同，西门子获得价值7.5亿欧元份额”，此时[[德国|德国]]《[[商报|商报]]》也援引西门子内部人士谈话，西门子表示“不会、也绝不出让核心技术”<ref name=sina0402 />。这则消息在海内外一度引起广泛关注，许多媒体以“西门子宣布获得中国政府购买100列高速列车的订单”类似标题进行了报道<ref name=sina0402 />。由于西门子网站消息与铁道部官方发布信息，措辞明显有出入，尤其在动车组知识产权归属上存在疑义，引起国内公众疑虑。3月25日，铁道部公开回应称：“近期并没有跟西门子签订任何合同”，形势几乎演变成由铁道部和西门子主演的一场公关危机，引发诸多猜测。3月31日，[[西门子交通集团|西门子交通集团]]媒体发言人回应媒体时表示，此次100列动车组的合约确已签署，列车将主要在中国生产。而对有关技术转让的合约细节等问题，则并未有详细透露<ref name=sina0402 />。{{Fact|实际上，按铁道部的招标规则，在京沪高铁的设备采购中，外资企业不能成为总承包商，但是容许外资企业通过与中国本土企业合作，原则上铁道部并不可直接与外资签订合同。因此，中国北车股份有限公司在3月16日签署协议以后，会将部分无法国产的零部件生产份额转包给了西门子公司，而西门子此次通过提供电气牵引系统等的方式可获得约7.5亿欧元（约70亿元人民币）的订单。|time=2015-01-21T13:45:41+00:00}}而根据西门子的消息，这些部件的生产任务将分别由西门子位于[[德国|德国]][[纽伦堡|纽伦堡]]、[[克雷费尔德|克雷费尔德]]等地，以及[[奥地利|奥地利]]和[[中国|中国]][[上海|上海]]等工厂承担。

2009年6月，[[中华人民共和国铁道部|中华人民共和国铁道部]]向国内动车组制造企业招标采购共320列时速350公里的高速动车组<ref>{{Cite web |url=http://www.chinaitc.com.cn/389-1050-11823.aspx |title=铁道部时速350公里速度级铁路动车组项目招标公告 |accessdate=2011-12-02 |publisher=中技国际招标公司 |date=2009-06-10 }}</ref>，中國北車集团获得了100列动车组的訂單<ref>{{cite web| url = http://finance.people.com.cn/BIG5/10134968.html| title = 中國北車再獲140列新一代動車組大額訂單| date = 2008-09-28| publisher = 人民網}}</ref>，其中80列为新一代CRH3-380型动车组，包含40列16辆编组动车组和40列8辆编组动车组，由长春轨道客车制造<ref>{{cite web| url = http://jlrbszb.chinajilin.com.cn/html/2009-10/06/content_559836.htm| title = 长客签下235亿订单 为铁道部制造新一代动车组| date = 2008-09-28| publisher = 吉林日报| deadurl = yes| archiveurl = https://web.archive.org/web/20110707014542/http://jlrbszb.chinajilin.com.cn/html/2009-10/06/content_559836.htm| archivedate = 2011-07-07}}</ref>。2009年9月28日，铁道部、长春轨道客车股份有限公司、北京铁路局于北京正式签订合同，总值235亿元人民币<ref>{{Cite web |url=http://www.cccar.com.cn/news_content.asp?mainid=22&id=250 |title=新一代时速350公里动车组第二单在北京签订 |accessdate=2010-12-03 |publisher=长春轨道客车股份有限公司 |date= }}</ref>。这批动车组计划于2011年4月开始交付，全部80列动车组将于2012年6月完成交付。

2010年5月27日，首列CRH3-380列车完成样车试制，并在[[吉林省|吉林省]][[長春市|長春市]]绿园区轨道交通装备制造产业园首次向外界展示。

2010年7月17日，时任[[中共中央政治局常委|中共中央政治局常委]]、[[中国国家副主席|国家副主席]][[习近平|习近平]]在河北省唐山市调研期间，前往唐山轨道客车有限责任公司，考察新一代时速380公里CRH3-380“和谐号”动车组生产线，先后考察了高速动车组铝合金车体生产线和总装配厂，并登上一列即将下线的CRH3-380型动车组<ref>{{Cite web |url=http://news.163.com/10/0721/10/6C41E6UM000146BC.html |title=习近平考察中国北车唐山轨道客车有限责任公司 |accessdate=2010-12-03 |publisher=网易新闻 |date=2010-07-21 }}</ref>。

2010年9月，铁道部下发《关于新一代高速动车组型号、车号及坐席号的通知》，将70列由[[唐山轨道客车|唐山轨道客车]]制造，110列由[[长春轨道客车|长春轨道客车]]制造的CRH3-380型动车组定型为CRH380B系列，其中短编组动车为'''CRH380B'''，而长编组动车为'''CRH380BL'''。至2010年12月中旬，铁道部重新分配动车组车号，根据《运装客车【2010】807号》文件，将CRH380B分配为长春轨道客车生产的40列短编组座车动车组；CRH380BL分配为长春客车轨道的45列长编组座车动车组、唐山轨道客车的70列长编组座车动车组，CRH380B系列均采用原头型。而长春轨道客车的25列新头型长编组座车动车组则分配至'''[[和谐号CRH380C型电力动车组|CRH380C]]'''。

由2014年7月1日起，所有CRH380动车组編號均作出了更改。
*长春轨道客车生产的CRH380B，原編號為（CRH380B-6201L～CRH380B-6245L、CRH380B-6246～CRH380B-6300、CRH380B-6326～CRH380B-6383），現改為（CRH380BL-5501～CRH380BL-5545、CRH380BG-5546～CRH380BG-5600、CRH380BG-5626～CRH380BG-5636、CRH380B-5637～CRH380B-5681、CRH380BG-5682、CRH380BG-5683）。
*唐山轨道客车的CRH380B，原編號為（CRH380B-6401L～CRH380B-6470L、CRH380B-6471～CRH380B-6500），現改為（CRH380BL-3501～CRH380BL-3570、CRH380B-3571～CRH380B-3600），該編號之後的生產动车组均依照新規定之格式定編號，並將'''高寒型'''正式命名為CRH380BG<ref name="铁总运(2014)150号">{{Cite web|url=http://www.ditiezu.com/thread-375095-1-1.html|title=《中国铁路总公司关于印发动车组型号、车号和坐席号编号规则的通知》|accessdate=2015-09-22|work=中国铁路总公司|publisher=中国铁路总公司|date=2014-06-13}}</ref><ref name="铁总运(2014)150号-weibo">{{Cite web|url=http://weibo.com/1736649017/BaPm4rmfQ|title=《中国铁路总公司关于印发动车组型号、车号和坐席号编号规则的通知》|accessdate=2015-09-22|work=中国铁路总公司|publisher=中国铁路总公司|date=2014-06-13}}</ref>。

== 技术特性 ==
CRH380B型动车组是在CRH3C基础上研发的新一代高速动车组，与CRH3C相比，持续运营时速为由300公里提高至350公里，最高设计时速由350公里提高到380公里，最高试验时速为400公里以上，性能优化以提高牵引功率、降低传动比及动车组气动外形减阻为主，而列车舒适度优化方面主要采取提高列车减震性能、车厢降噪、加强车内气压控制等方式。

CRH380BL型动车组列车总数为121列，其中51列由长春轨道客车生产（CRH380BL-5501～CRH380BL-5545、CRH380BL-5823～CRH380BL-5828），另外76列由唐山轨道客车生产（CRH380BL-3501～CRH380BL-3570、CRH380BL-3732～CRH380BL-3737），采用了8动8拖的编组方式，牵引功率为18400千瓦，列车由1辆商务车（又称VIP座车）、4辆一等座车、10辆二等座车和1辆餐车组成，其中商务车28人，一等座186 人，二等座791人，定员为1005人<ref>{{cite web| url = http://news.ifeng.com/gundong/detail_2011_05/31/6721383_0.shtml| title = CRH380BL：“海豚”“猎豹”并驾齐驱| date = 2011-05-31| publisher = 凤凰网| accessdate = 2011-12-08}}</ref>。

2010年9月20日，首列CRH380B型高速动车组（6401L）在唐山轨道客车的厂房内下线<ref>{{Cite web |url=http://www.tskp.org.cn/article.asp?NewsID=9081 |title=高铁事业的领跑者 |accessdate=2010-12-03 |publisher=唐车公司科协、唐山科学技术协会 |date=2010-11-26 }}</ref>，而第二列（6402L）也在10月下线。2010年11月初，两列动车组前往[[中国铁道科学研究院环行铁道试验基地|中国铁道科学研究院环行铁道试验基地]]安装试验设备并进行初步试验，同年11月20日，CRH380B-6401L、CRH380B-6402L开始前往京沪高速铁路先导试验段（[[枣庄|枣庄]]至[[蚌埠|蚌埠]]）开始进行正式线路联调联试和综合试验。11月28日起，CRH380BL开始在[[沪宁城际高速铁路|沪宁城际高速铁路]]上空载试运行。2010年12月5日，CRH380B型动车组在京沪高铁先导段的最高试验速度达457公里/小时<ref>{{Cite web |url=http://beijing.qianlong.com/3825/2010/12/07/118@6396161.htm |title=时速500公里试验列车正在研制 |accessdate=2010-12-07 |publisher= |date=2010-12-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20101207174717/http://beijing.qianlong.com/3825/2010/12/07/118%406396161.htm |archivedate=2010-12-07 }}</ref>。

2011年1月9日，采用特殊试验编组（8M4T）的CRH380B-6402L动车组再次进行京沪高速铁路徐州至蚌埠先导段的运行试验。当天下午5点，列车由[[徐州东站|徐州东站]]出发，在启动后6分钟之内就达到了380公里/小时，并于5时15分创造了每小时487.3公里的最高试验速度<ref>{{Cite web |url=http://www.sasac.gov.cn/n1180/n1226/n2410/n314244/7793763.html |title=中国北车刷新高铁运营试验世界纪录速度 |accessdate=2010-12-30 |publisher=国资委信息中心 |date=2011-01-10 }}</ref>。

2011年1月13日起，配属[[上海铁路局|上海铁路局]]的首批9列CRH380BL型电力动车组，在春运前夕投入[[沪杭客运专线|沪杭客运专线]]的商业运营<ref name=cnr2011013>{{Cite web |url=http://www.chinacnr.com/272-619-3768.aspx |title=380系列长编型动车组首次上线运行并装备沪杭春运 |accessdate=2010-12-30 |publisher=中国北车股份有限公司 |date=2011-01-13 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 车内布置 ==
[[File:CRH380BL_Business_Coach_SW622203_20121114.JPG|thumb]]
[[File:Kitchen_of_CRH380B_20151031.JPG|thumb]]
CRH380B型动车组设有一等座、二等座、观光座、VIP座等座席等级。二等座车座席采用2+3布置；一等座车座席采用2+2方式布置，每组列车其中一辆一等座车设有一个4人包间、以及两个6人包间<ref name=cnr2011013 />。VIP座位于商务车车厢（又称VIP车），采用1+2方式布置，设置类似民航客机[[头等舱|头等舱]]的高级可躺座椅。与CRH3型动车组一样，CRH380B型动车组两端头车后方也设有包间，称为观光区，初期可通过可调透明度玻璃看到看到驾驶室的操作以及前方轨道视角，目前玻璃已全部锁定为不透明状态。

列车内部的旅客空间“航空化”，座位号除了数字外还增加了英文字母。二等座每列设有“3+2”方式排列五个座位，以“A、B、C、D、F”代表；一等座每列设有“2+2”方式排列四个座位，以“A、C、D、F”代表。无论是一等车厢还是二等车厢，带字母“A”和“F”的座位靠窗，带字母“C”和“D”代表贴近中间走道。以二等车厢第四排的座椅为例，座号“4A”、“4F”靠窗，“4C”、“4D”临过道，4B为两座中间一个座位。自2010年11月起铁道部陆续升级车站的售票系统后，旅客在人工售票窗口购票时可以选择指定车厢和席位号码；网络售票和自动售票机暂不支持自选车厢、席位，但网络售票在票源充足时支持选择靠窗/非靠窗位置。

== 召回事件 ==
2011年8月10日，中国北车表示，近日，配属上海局的CRH380B-6227L、6228L动车组，从北京回送上海的过程中，连续发生了热轴误报、自动降弓和牵引丢失故障。中国北车长客股份公司决定主动进行内部排查质量问题，暂缓CRH380BL列车交付，并将此事通报铁道部。<ref>{{cite web|url=http://epaper.bjnews.com.cn/html/2011-08/10/content_263298.htm?div=-1|title=京沪高铁一车型因故障暂缓出厂|publisher=[[新京报|新京报]]|language=zh-hans|date=2011年8月10日}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

8月11日晚，[[中国北车集团|中国北车集团]]发布公告称，召回全部[[唐山轨道客车|唐山轨道客车]]和[[长春轨道客车|长春轨道客车]]生产的CRH380BL动车组，共54列<ref>{{cite web|url=http://news.163.com/11/0812/04/7B7RCU4C00014AED.html|title=中国北车召回54辆京沪高铁动车
|publisher=新华网(广州)|date=2011年8月12日}}</ref>。受此影响，8月16日起京沪高铁部分车次（包括部分跨线车次）停运。

11月14日，铁道部称54辆CRH380BL已整改完成，并将自11月16日起分批投入运营。
