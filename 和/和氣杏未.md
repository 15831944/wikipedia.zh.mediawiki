'''和氣杏未'''（{{bd||1994年9月8日|catIdx=Waki Azumi}}）<ref name=profile>{{Cite web|date=|url=http://www.haikyo.co.jp/profile/profile.php?ActorID=12722|title=プロフィール|publisher=|accessdate=2016-05-31}}</ref>是[[日本|日本]]的女性[[聲優|聲優]]，[[東京都|東京都]]出身。[[東京俳優生活協同組合|東京俳優生活協同組合]]所屬。

== 人物 ==
[[東京動畫、聲優專門學校|東京動畫、聲優專門學校]]出身。中學時代原本想成為[[空中服務員|空中服務員]]，但進入高中後因為[[英文|英文]]表現不理想而改以聲優為目標<ref name="ddnavi">{{Cite web |date=2015-10-14|url=http://ddnavi.com/news/262783/a/|title=「和氣あず未」声優インタビュー＆撮り下ろしグラビア【声優図鑑】|publisher=DA VINCI新聞|accessdate=2015-10-17}}</ref>。

2014年開始加入東京俳優生活協同組合<ref name="haikyo">[http://www.haikyo.co.jp/profile/profile.php?ActorID=12722 東京俳優生活協同組合によるプロフィール]</ref><ref>{{Cite press release |title=新しく所属になりました。|publisher=東京俳優生活協同組合|date=2015-01-13|url=http://www.haikyo.co.jp/information/info15/info15.html|accessdate= 2015-10-17}}</ref>。

進入事務所後的初出道作品為《[[偶像大師_灰姑娘女孩|偶像大師 灰姑娘女孩]]》<ref name="ddnavi" />。

中學、高中時加入[[網球|網球]]社，對於運動神經很有自信<ref name="ddnavi" />。喜歡的動畫作品是「[[鋼之鍊金術師|鋼之鍊金術師]]」<ref name="ddnavi" />。

据本人在[[Twitter|Twitter]]上表示，本名「{{jp|j=あず未|rm=Azumi|help=no}}」取自[[长野县|长野县]]的地名{{jp|c={{tsl|ja|安曇野}}|j=安曇野|rm=Azumino|help=no}}<ref>{{Cite web |date=2016-04-26|url=https://twitter.com/azumi_waki/status/724778694766551040|script-title=ja:長野県楽しみだなぁ〜！ ちなみに長野県の安曇野って場所から、あずみって名前をとったらしい(^-^) 綺麗な場所なんだって〜〜…|publisher={{lang|ja|和氣あず未}}|accessdate=2016-04-26}}</ref>。
