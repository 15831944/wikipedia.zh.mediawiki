'''和站'''（{{jpn|j=和駅|hg=やわらえき|rm=Yawara eki|f=y}}）是位於[[北海道|北海道]][[雨龍郡|雨龍郡]][[北龍町|北龍町]]和，隸屬於[[日本國有鐵道|日本國有鐵道]]（國鐵）[[札沼線|札沼線]]，已廢除的[[鐵路車站|鐵路車站]]。

== 歷史 ==
* 1931年（昭和6年）10月10日 - [[札沼線|札沼北線]][[石狩沼田車站|石狩沼田]]至中德富（初代，現在的[[新十津川車站|新十津川]]）開始營運。
* 1944年（昭和19年）7月21日 - 由於[[太平洋戰爭|太平洋戰爭]]擴大，本線作為[[不要不急線|不要不急線]]要停止營運。
* 1956年（昭和31年）11月16日 - 札沼線[[雨龍車站|雨龍]]至石狩沼田路段恢復營運，全線回復正常服務。
* 1972年（昭和47年）6月19日 - 札沼線新十津川至石狩沼田路段廢除，本車站同時廢站。

== 車站構造 ==
站房位於札幌方向的右方西側，為[[側式月台|側式月台]]1面1線的地面車站，同時設有[[側線|側線]]通往車廠。而站房旁往石狩沼田方向設有貨物月台，並有1條引入線到達。本車站是廢線前，由[[浦臼車站|浦臼車站]]至石狩沼田站之間唯一能夠進行[[列車交會|列車交會]]的車站。

== 車站周邊 ==
遺址已興建縫製工廠，而舊站房部份建築亦被縫製工廠使用。
