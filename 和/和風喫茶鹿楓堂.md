《'''和風喫茶鹿楓堂'''》（{{lang-ja|鹿楓堂よついろ日和}}）是{{link-ja|清水ユウ}}的漫畫作品，以及改編的電視動畫。

== 概要 ==
原先在2013年10月發行的《{{link-ja|GO GO BUNCH|ゴーゴーバンチ}}》的vol.1上開始連載，至vol.20因《GO GO BUNCH》休刊而轉至同社的《{{link-ja|月刊COMIC@BUNCH|月刊コミック@バンチ}}》的2018年3月號上繼續連載，之後因雜誌重整為《月刊COMIC BUNCH》而在新刊創刊號2018年6月號上連載。

== 故事 ==
這裡是和風咖啡廳“鹿楓堂”，店主、負責茶水的水，負責拿鐵的格雷，負責料理的世貴，負責甜點的椿，各自身懷絕技的4人經營的隱藏人氣店，他們在“款待”客人的同時，偶爾還會解決客人的“煩惱”，以咖啡廳為舞台，完美的4人編織而成的真誠故事。好了，今天的客人是…？

== 出場人物 ==
=== 鹿楓堂 ===
; {{nihongo|東極京水|東極京水（スイ）|聲：[[諏訪部順一|諏訪部順一]] ，[[千本木彩花|千本木彩花]]（幼少期）}}
: 生日：4月7日 Ｏ型。
: 興趣：逛古書店、貓咪咖啡店、讀書。
: 特長：劍道、茶道。
: 喜歡的東西：日本酒、貓。
: 討厭的東西：鴿子。
: 本作的主人公，鹿楓堂的店主，負責茶水，不擅長料理。外表看起來成熟，卻有著天然的一面。狂熱貓咪愛好者，十分溺愛看板貓黃豆粉，時常給牠各種角度的攝影。認為鹿楓堂和其成員們比什麼都重要。

; {{nihongo|永江世貴|永江ときたか（ときたか）|聲：[[中村悠一|中村悠一]] }}
: 生日：10月19日 Ａ型。
: 興趣：看時代劇、電視劇、園藝、陶藝。
: 特長：料理、手工藝、乒乓球
: 喜歡的東西：陶藝品、植物、煮豆
: 討厭的東西：爬蟲類
: 鹿楓堂的料理廚師。性格溫和而冷靜，開了間陶藝教室，與附近的老人們交情很深。與水是中學時代的朋友。在鹿楓堂的後院弄了個家庭菜園，並逐年擴大。有隱性虎牙。

; {{nihongo|格雷戈里奧·瓦倫蒂諾|グレゴーリオ・ヴァレンティノ（ぐれ）|聲：[[小野大輔|小野大輔]] }}
: 生日：7月22日 Ｏ型
: 興趣：鴨子船、體育、電影鑑賞、旅行
: 特長：拿鐵拉花(自稱)、腳程快
: 喜歡的東西：運動、恐龍、冰淇淋
: 討厭的東西：魚糕
: 鹿楓堂的咖啡師，拿鐵拉花微妙，經常引發物議論（只有葉子的拉花堪稱完美，卻自認動物的拉花更好）義大利和日本的混血，是個氣氛製造者，像大哥哥般的存在，無論在什麼方面都十分積極，曾有段比較黑暗的過去。是「鴨船同好會」的會長擔任。

; {{nihongo|中尾椿|中尾椿（椿）|聲：[[山下大輝|山下大輝]] }}
: 生日：2月8日 AB型
: 興趣：邊逛邊吃，逛水族館、讀書
: 特長：製作日式和西式點心、背誦
: 喜歡的東西：甜食、辣的東西、大海
: 討厭的東西：蟲子、英語
: 鹿楓堂的糕點師，喜愛做甜點和吃甜點，假日經常一個人去不同的甜點店大開殺戒，曾被格雷說自己一半是砂糖做的。性格敏感細膩比較害羞認生，鹿楓堂中年紀最小的成員，對格雷經常有過激的吐槽。

; {{nihongo|東極八京|東極八京|聲：[[前野智昭|前野智昭]] }}
: 東極京水的雙胞胎哥哥“東區酒店集團”的副社長，外表冷酷，性格沈默寡言，幾乎不談自己的事，不喜歡別人踏入自己的隱私。

; {{nihongo|角崎英介|角崎英介|聲：[[鳥海浩輔|鳥海浩輔]] }}
: 

; {{nihongo|黃豆粉|きなこ|聲：[[天崎滉平|天崎滉平]] }}
: 鹿楓堂的看板猫。在後巷里被東極京水撿到後與鹿楓堂的大家一起生活。東極京水十分溺愛着它。喜歡頓煮雞胸肉。
