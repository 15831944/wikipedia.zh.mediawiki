'''和平教育'''旨在教導人們與自己以及他人、自然環境和諧相處。教育內容可能是傳遞此類價值觀和相關知識，或者培養這一類的態度、發展相關技能。<ref name="施正鋒"/>

[[聯合國|聯合國]]有許多聲明闡述了和平教育的重要性。<ref name ="James S"/>聯合國秘書長[[潘基文|潘基文]]將2013年[[國際和平日|國際和平日]]的主題定為「教育促進和平」（{{lang|en|Education for Peace}}），以呼籲各國將教育訂為政府的第一要務，並於教育中教授解決與預防衝突的技能等。 <ref name ="Countdown 2013"/><ref name ="C-zh 2013"/>[[松浦晃一郎|松浦晃一郎]]（[[聯合國教科文組織|聯合國教科文組織]]前任總幹事）曾寫道，和平教育「對教科文組織和聯合國的使命至關重要」。<ref name ="Koichiro"/>越來越多和平的研究者（如：{{le|貝蒂·里爾頓|Betty Reardon}}、{{le|道格拉斯·羅奇|Douglas Roche}}）強調接受和平教育是人們的[[權利|權利]]。<ref name ="Reardon"/><ref name ="Roche"/>和平教育能和[[人權教育|人權教育]]互相搭配。<ref name ="UNGA"/>
