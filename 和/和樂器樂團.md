'''和樂器樂團'''（{{lang-ja|和楽器バンド}}），是一個日本的八人組[[搖滾|搖滾]][[樂團|樂團]]。使用[[尺八|尺八]]、[[和箏|箏]]、[[三味線|三味線]]、[[和太鼓|和太鼓]]這些[[日本民族樂器|和樂器]]與[[吉他|吉他]]、[[貝斯|貝斯]]、[[鼓|鼓]]這三種西洋樂器，並加上擁有{{link-ja|詩吟|吟詩}}教師身分的主唱所組成。

== 經歷 ==
2013年3月，主唱[[鈴華優子|鈴華優子]]的號召下，以和風樂團「'''{{link-ja|華風月|華風月}}'''」為中心，並加入其他新成員以「'''鈴華ゆう子with和楽器バンド'''」這個暫時的樂團名字組成。

2014年1月，在「'''渋谷Club Asia'''」進行第一次的單獨公演，並正式將名字確立為「'''和楽器バンド'''」。

2014年4月，發行以「[[Vocaloid|Vocaloid]]」歌曲為主的出道專輯「'''ボカロ三昧'''」。

2014年7月，第一次的海外出演『'''第15回Japan Expo'''』<ref>{{cite news|url=http://www.barks.jp/news/?id=1000105427|title=パリっ子熱狂。＜JAPAN EXPO＞に和楽器バンド|newspaper=BARKS音楽ニュース|date=2014-07-07|accessdate=}}</ref>。

2014年8月，首張原創單曲「'''華火'''」發行。

2015年4月，首次登上朝日電視台的「'''[[MUSIC_STATION|MUSIC STATION]]'''」演出。

2015年4月23日，官方粉絲俱樂部「'''八重流 -yaeryu-'''」正式成立。

2015年5月9日及5月10日，首次來台灣進行「和楽器バンド 大台北演唱會」，地點在台北ATT SHOW BOX，二場門票均在3分鐘左右完售。

2015年11月7日，二次來台灣舉辦「台北繪卷」演唱會，地點於台北ATT SHOW BOX（10月2日門票開賣，30秒完售）。<ref>{{cite news|url=http://ent.ltn.com.tw/news/breakingnews/1501131|title=和樂器大秀中文 妖嬈舞姿讓台粉醉了|newspaper=自由娛樂|date=2015-11-07|accessdate=}}</ref>

2015年11月8日，宣布加場「台北繪卷 PREMIUM ENCORE」演唱會，（10月11日門票開賣，當日完售）。

== 團員 ==
; [[鈴華優子|鈴華優子]]（すずはな ゆうこ） - [[主唱|主唱]]
: 6月7日出生（生年非公開），[[茨城縣|茨城縣]][[水戶市|水戸市]]出身。3歲開始學習鋼琴，5歲開始學習{{link-ja|吟詩|吟詩}}和{{link-ja|詩舞|詩舞}}。[[東京音樂大學|東京音樂大學]]鋼琴系，吟詩師範。藝名「鈴華優子」的「華」是從吟名「晟華」而來。除了吟詩之外，也有用作曲家、鋼琴演奏者等身分活動。2011年，niconico動畫的祭典裡當選了ミスニコ生2011。然後同年12月獲得了第47回日本コロムビア全國吟詠コンクール大會優勝。並在那之後組成三人和風樂團「'''{{link-ja|華風月|華風月}}」'''，擔任[[鋼琴|鋼琴]]與{{link-ja|吟詩|吟詩}}，之後擴大組成「'''鈴華ゆう子with和楽器バンド'''」（樂團名之後改成「'''和樂器樂團'''」並與「'''{{link-ja|華風月|華風月}}'''」平行活動）。
: 家鄉茨城的電視台節目[[IBAKIRA_TV|IBAKIRA TV]]「[https://web.archive.org/web/20150216011632/http://www.ibakira.tv/archive/suzuhana/ 鈴華ゆう子のただいまIBARAKI]」擔任主持人。（該節目已完結）
; 衣袋聖志（いぶくろ きよし） - [[和箏|箏]]
: 1983年12月1日生、[[東京都|東京都]]出身。高中時代除了日本文化廳派遣至[[中國|中國]]演出，也在[[越南|越南]]、[[伊朗|伊朗]]、[[卡達|卡達]]、[[科威特|科威特]]、[[南韓|南韓]]等八個國家演出過。除了公演活動外，並還有在{{link-ja|全音楽譜出版社|全音楽譜出版社}}出版的二十五弦箏樂譜集，是二十五弦箏、二十一弦箏、十七弦箏、十三弦箏的箏奏者。多方結合[[日本民族樂器|和樂器]]與[[西洋樂器|西洋樂器]]彼此組合的可能性。另有參加與{{link-ja|中村仁樹|中村仁樹}}共同的樂團[[蓮-REN-|蓮-REN-]]、{{link-ja|華風月|華風月}}的成員之一。
; 神永大輔（かみなが だいすけ） - [[尺八|尺八]]
: 1985年8月17日生、[[福島県|福島県]][[磐城市|磐城市]]出身。十八歲時開始學習尺八，並在那之後取得[[尺八|尺八]]師範資格，獲得“耀山”稱號。有著在十餘國以上的海外公演經驗，參加過世界最大的音樂活動如[[西南偏南|西南偏南]]、[[Japan_Expo|Japan Expo]]、{{link-ja|Japanese Lounge Night|Japanese Lounge Night}}。並在大學教育機構擔任尺八講師。另一方面相當熱愛遊戲音樂，曾經參與遊戲配樂的音樂錄製，例如曾與知名作曲家攜手打造「Playing Works」系列的曲子。2013年春以「響-hibiki-」的名義加入黑流所組織的樂團「Crow×Class〜黒鴉組〜」並擔任[[尺八|尺八]]位置。
: 大輔在東京開了一間[[尺八|尺八]]音樂教室，並在多個大學與學院裡當講師，也在故鄉[[福島県|福島県]][[磐城市|磐城市]]，透過磐城藝術文化交流館所舉辦的活動，登記為Aliso音樂人，在各個中小學演奏給年輕人聽。
; 蜷川紅（にながわ べに） - {{link-ja|津輕三味線|津軽三味線}}
: 9月14日生（生年非公開），出生於[[京都|京都]]。四歲時開始學習[[大阪|大阪]]流派的民謠，7至8歲之間開始學習{{link-ja|津輕三味線|津軽三味線}}，也曾經擔任過數次模特兒。曾獲得日本民謠青少年大會冠軍、產經民謠大獎冠軍、內閣官房長官獎、本莊追分全國大會冠軍、文部科學大臣獎等等。15歲時因喉嚨受創後，而轉向致力於{{link-ja|津輕三味線|津軽三味線}}。為追求和樂器的可能性，開始在NicoNico社群網站上投稿影片並於東京進行活動。
; 黒流（くろな） - [[和太鼓|和太鼓]]
: 5月10日生（生年未公開），三歲開始打太鼓，九歲加入東京的和太鼓職業團體並在習得了傳統太鼓演奏後，獨立單飛。曾在15個以上的國家表演過太鼓以外，國內外也有雜誌專訪，電視及舞台劇等的表演活動。曾以傳統太鼓配合現代音樂，與多位音樂家及演奏家合作過。身為和樂器樂團中和太鼓的擔當，也在黑流自己所組織的樂團「Crow×Class〜黒鴉組〜」以「黒流-Kurona-」為名義擔當著黑鴉組中主唱及和太鼓的擔當。
: 曾擔任東京都教育委員會主催「東京都高等學校文化祭・鄉土藝能部門・中央大會」的審査員。也是東京迪士尼樂園＆迪士尼海洋樂園的「Blazing Rhythms」、「Countdown」、「The Legend Of Mythica -密西歌傳說」，以及「迪士尼盛夏祭典表演」等的太鼓表演指導師，與許多音樂家合作，也參加錄音活動。
; 町屋（まちや） - [[吉他|吉他]]
: 1982年9月13日出生於[[北海道|北海道]]。在Niconico社群網站裡壓倒性的人氣吉他手。在NicoNico動畫中投稿的作品「ナイト・オブ・ナイツを弾いてみた」的點閱率已經突破了150萬點閱次數。
: 身為和樂器樂團中吉他手及伴唱的擔當，也在知名獨立樂團「m:a.ture」中以「桜村 眞 -Oumura-」為名擔任主唱及吉他手。
; 亞沙（あさ） - [[電貝斯|電貝斯]]
: 12月18日（生年未公開）出生於[[埼玉県|埼玉]]。常以女性的妝扮來進行演出，但本身性別為男性。現在以[[Vocaloid|Vocaloid]]在NicoNico社群網站中順速竄紅。
: 2012年7月投稿的[[Vocaloid|Vocaloid]]歌曲「吉原哀歌」，至今已有突破100萬點閱次數的超高人氣，也被NicoNico動畫裡的其他音樂家表演過(甚至收錄於「ボカロ三昧」專輯裡，亦是和樂器樂團推出過的歌曲中已知唯一一首由亞沙代替町屋伴唱的歌曲)。此歌曲也被網路浮游都市之公司出版為小說，更與知名的電子音樂製作公司[[EXIT_TUNES|EXIT TUNES]]合作，發售了「吉原ラメント〜唄い手盤〜」「吉原ラメント〜UTAU盤〜」兩款單曲。
: 曾參加過由 EXIT TUNES 所舉辦的 「EXIT TUNES ACADEMY 」祭典。
; 山葵（わさび） - [[爵士鼓|爵士鼓]]
: 1988年11月5日出生於[[中國|中國]]，成長於[[岡山|岡山]]，持有著四分之一的日本血統（父親是中國人、母親是中日混血兒），故此為所有成員當中唯一懂得講中文的成員。在Niconico社群網站裡壓倒性的人氣鼓手王。15歲開始打鼓，除了流行[[搖滾樂|搖滾樂]]，也與[[舞蹈|舞蹈]]、[[演劇|演劇]]攜手合作，也參與唱片錄音。團員中年紀最小卻有沉穩的傲人演奏技巧，成名作莫過於與黑流合作的快速雙大鼓，為和樂器樂團中的知名表演。
: 在演奏時裸著上半身，並在背上書寫文字是他最大的特色。自己單獨表演時大部分以「ササキアキラ」為名來進行活動（在和樂器樂團的採訪中山葵曾經提到，若穿著衣服就是「ササキアキラ」，脫掉衣服時則會變成「山葵」）。
