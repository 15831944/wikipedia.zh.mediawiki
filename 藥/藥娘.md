'''藥娘'''廣義上指通過服用药物使自身女性化的人群，包括具有[[性別焦慮|性別焦慮]]的生理[[男性|男性]]、部分[[雙性別|雙性別]]群體等。<ref>{{Cite web |url = https://item.btime.com/30qo482d7r68dup7vrqvr3q4rhf|title = 药娘群像：与身体斗争的这些年|date=2017-09-07|publisher=北京时间|language=zh|accessdate=2017-11-26}}</ref>

狹義上指[[性別認同|性別認同]]為女性的生理男性（即跨性别女性）、[[雙性別|雙性別]]等，在進行[[性別重置手術|性別重置手術]]前的過渡階段，即進行[[女性化激素疗法|女性化激素疗法]]的階段。狹義上的藥娘即真正意義上的藥娘，僅僅是必要的特定階段，其最終的目的是通過[[性別重置手術|性別重置手術]]獲得符合心理預期的身份并使其合法化。

藥娘是中國大陸特有詞滙，她們通常活躍於較封閉的[[社交網絡|社交網絡]]，近来有逐漸公開化趨勢。

==自稱==
藥娘通常以[[跨性别女性|MtF]]（male to female，男跨女）、TS（Transsexual，常取拼音称做“天使”）自稱。

==藥物與手術==
藥娘通常服用的藥物有三大類別，分別是[[抗雄激素|抗雄激素]]藥物、[[雌激素|雌激素]]以及[[孕激素|孕激素]]。醫生通常會要求接受[[性別重置手術|性別重置手術]]者必須有一定時限的激素替代療法史，以達到相應的手術要求。
