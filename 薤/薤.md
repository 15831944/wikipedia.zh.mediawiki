'''薤'''（{{標音|字=薤|拼音=xiè|注音=ㄒㄧㄝˋ}}，[[学名|学名]]：''{{lang|la|Allium chinense}}''），为[[葷菜|五種葷菜]]之一。分布在[[柬埔寨|柬埔寨]]、[[越南|越南]]、[[日本|日本]]、[[美国|美国]]、[[老挝|老挝]]以及[[中国大陆|中国大陆]]的[[长江流域|长江流域]]和以南各省区等地，生长于海拔700米至2,700米的地区。目前已人工引种栽培，並常製作為[[醃菜|醃菜]]，是[[越南|越南]]新年时食品之一{{R|数据库}}。

== 名稱 ==
'''薤'''有多種名稱，包括'''藠頭'''（{{標音|字=藠|拼音=jiào或xiào|注音=ㄐㄧㄠˋ或ㄒㄧㄠˋ<ref>http://dict.variants.moe.edu.tw/yitic/frc/frc12041.htm</ref>}}）、'''薙頭'''、'''小蒜'''、'''薤白頭'''、'''野蒜'''、'''[[野韭|野韭]]'''等。

===台灣===
[[台語|台語]]稱為蕗藠、藠頭（藠，，俗寫為「蕎」）；[[阿美族|阿美族]]稱之為'''火蔥'''，族語稱為'''lakiu'''{{r|吳雪月|page=96}}。

===中國大陸===
在[[昆明话|昆明话]]裡把醃過後，半透明的薤稱為藠頭（音『叫』），是當地特色[[醃菜|醃菜]]之一；

而薤在[[内蒙|内蒙]]與[[山西|山西]]等地称为'''薤薤'''（俗寫為「害害」）。

== 形态 ==
薤为多年生宿根[[草本植物|草本植物]]，一般作二年生栽培，基生[[叶|叶]]数枚，半圆柱状线形，有不明显3-5棱，中空，葉鞘抱合成[[假莖|假莖]]，基部形成粗的[[鱗莖|鱗莖]]。鳞茎狭卵形，外觀初看似小型[[大蒜|蒜頭(大蒜)]]，故名'''小蒜'''，剖開來環形結構似[[洋葱|洋葱]]，呈白色，是主要的食用部分。秋季九至十月抽花莖，伞状[[花序|花序]]顶生，花小呈紫紅色。

「葉似[[蔥|蔥]]、[[韭|韭]]，鱗莖似[[蒜|蒜]]」，具有諸多[[蔥屬|蔥屬]]植物特徵是'''薤'''的特色。

== 產地 ==
原产中国，《[[漢書|漢書]]·龔逐傳》就有记载。目前中国南方諸省都有种植，北方人较少食薤。[[江西省|江西省]][[新建縣|新建縣]][[生米镇|生米镇]]被[[中華人民共和国农业部|中華人民共和国农业部]]命名「中国藠头之乡」。

台灣的薤由原住民移民時由南洋傳來，現為零星種植，其中栽植較大規模者包括[[雲林縣|雲林縣]][[古坑鄉|古坑鄉]]、[[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]、[[臺南市|臺南市]][[白河區|白河區]]，以及[[花蓮縣|花蓮縣]][[壽豐鄉|壽豐鄉]]月眉村與[[光復鄉|光復鄉]]等[[阿美族|阿美族]]部落，多在原住民部落才見得到{{r|吳雪月|page=96}}。

《[[黄帝内经|黄帝内经]]·素问》有“五菜为充”语，王冰注：“谓葵、藿、薤、葱、韭也。”《[[灵枢经|灵枢经]]·五味》又说：“葵甘，韭酸，藿咸，薤苦，葱辛。”

== 歷史 ==
中国古代最著名的輓歌《薤露》，起源於[[汉|漢代]]，至少沿唱至[[唐|唐代]]。

僅四句：「薤上露，何易晞，露晞明朝更復落，人死一去何時歸？」

== 食用方法 ==
[[File:醃薤白.jpg|200px]]中的醃薤白[[泡菜|泡菜]](右邊球形者)]]
華人地區以[[鹽|鹽]]漬、[[醋|醋]]漬鱗莖作為清脆口感的[[醃菜|醃菜]]為多；亦可炒食。

源於[[日本料理|日式]]的料理方法則包括：生蕗藠切片[[涼拌|涼拌]]、日式炸物（沾粉去炸）、蕗藠炒[[麵|麵]]、清炒、蕗藠炒[[蛋|蛋]]、蕗藠炒[[豬肉|豬肉]]、什錦蕗藠煎餅、雞肉芙蓉蕗藠蛋、海帶拌蕗藠、蕗藠[[海鮮|海鮮]][[義大利麵|義大利麵]]、蕗藠滑菇等等。

== 中醫學藥效 ==
{{無來源章節}}
{{ctm}}

[[File:W_rakkyou1101.jpg|thumb]]
藥用部位同食用部位，採[[白色|白色]][[鱗莖|鱗莖]]，稱為'''薤白'''。

=== 性味與歸經 ===
[[味辛|味辛]]、苦，[[性溫|性溫]]。入[[手太陰肺經|肺]]、[[足陽明胃經|胃]]、[[手陽明大腸經|大腸]]等經。

=== 功效 ===
功能[[行氣|行氣]]止痛、理氣寬胸、[[通陽|通陽]][[散結|散結]]；以上能溫通胸陽而散陰寒之結，治療[[胸痹|胸痹]]而痛、喘促等證；以下能通滯行氣，治療[[下痢|下痢]][[腹痛|腹痛]]等證。

治[[胸痹|胸痹]]代表方為[[栝蔞薤白白酒湯|栝蔞薤白白酒湯]]、[[栝蔞薤白半夏湯|栝蔞薤白半夏湯]]與[[枳实薤白桂枝汤|枳实薤白桂枝汤]]，出典於《[[傷寒雜病論桂林古本|傷寒雜病論桂林古本]]》卷十五「辨胸痹病脈證並治」篇，以及後世本《[[金匱要略|金匱要略]]》「胸痹心痛短氣病脈證治第九」篇。
