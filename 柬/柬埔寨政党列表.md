{{柬埔寨政治}}
'''柬埔寨政党列表'''列举的是[[柬埔寨王国|柬埔寨王国]]现存及曾经存在过的[[政党|政党]]。柬埔寨目前是由[[柬埔寨人民党|柬埔寨人民党]]执政的[[一党优势制|一党优势制]]国家。<ref>{{cite web |author1=顾佳赟 |title=柬埔寨坚拒西方干涉内政 |url=http://intl.cssn.cn/gj/gj_gjwtyj/gj_yt/201802/t20180228_3863209.shtml |website=[[中国社会科学院|中国社会科学网]] |publisher=[[文汇报_(上海)|文汇报]] |accessdate=2018-06-11}}</ref><ref>{{cite web |author1=许梅 |title=《中国周边国家概览：柬埔寨——国土、国民与国情；政治制度与政府》 |url=http://www.ntgshj.com/dangdiwanle/43049714.html |website=东南亚旅游攻略联盟 |publisher=复旦中国周边研究 |accessdate=2018-06-11}}</ref> 虽然柬埔寨允许反对党的存在，<ref>{{cite web |title=外交部长强调柬埔寨不搞一党专政 |url=https://cc-times.com/posts/202 |publisher=柬中时报 |accessdate=2018-06-11}}</ref>但是普遍认为反对党不具备与执政党在议会竞争的能力。<ref>{{cite web |title=没主要反对党参与·柬埔寨参院选举执政党大胜 |url=http://www.sinchew.com.my/node/1730837 |publisher=[[星洲网|星洲网]] |accessdate=2018-06-11}}</ref><ref>{{cite web |title=反对党再参政？门都没有！ |url=https://www.58cam.com/portal.php?mod=view&aid=5692 |website=柬单网 |publisher=华商日报 |accessdate=2018-06-11}}</ref>
