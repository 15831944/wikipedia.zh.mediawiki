'''充分必要條件'''（{{lang-en|sufficient and necessary condition}}）簡稱為'''充要條件'''。

在[[逻辑学|逻辑学]]中：
*当命题「若P則Q」为真时，P称为Q的'''充分条件'''，Q称为P的'''必要条件'''。

因此：
*当命题「若P則Q」與「若Q則P」皆为真时，P是Q的'''充分必要条件'''，同时，Q也是P的'''充分必要条件'''。
*当命题「若P則Q」为真，而「若Q則P」为假时，我们称P是Q的'''充分不必要条件'''，Q是P的'''必要不充分条件'''，反之亦然。

==必要條件==
P是Q的必要條件，代表「如果P是假，則Q是假」。

以[[逻辑符号表|邏輯符號]]表示：
: ~P → ~Q

通過[[否定後件|否定後件]]，得出「如果Q是真，則P是真」。
: Q → P

*例子1：對於大於2的數字，[[奇數|奇數]]是成為[[質數|質數]]的必要條件。
*例子2：年滿30歲是成為美國[[美國參議院|參議員]]的必要條件。如果他是參議員，他必定年滿30歲。

==充分條件==
P是Q的充分條件，代表「如果P是真，則Q是真」或「如果Q是假，則P是假」。

以[[逻辑符号表|邏輯符號]]表示：
: P → Q

*例子1：一個數字能被4整除，是成為[[偶數|偶數]]的充分（但不必要）條件。能被2整除，則是充分及必要條件。

==必要條件及充分條件==
P是Q的充分及必要條件，代表「若且唯若P是真，則Q是真」。

以[[逻辑符号表|邏輯符號]]表示：
: P ↔ Q

留意~P → ~Q可以推出Q → P。
:(~P → ~Q) & (P → Q)
:(Q → P) & (P → Q)
: P ↔ Q

==舉例==
1.若P表「人類生存」，Q表「人類呼吸」
: ~Q → ~P
: P → Q
:此時呼吸是生存的'''必要條件'''，生存是呼吸的'''充分條件'''，因为活着的人一定要呼吸，
: Q → P（錯誤）
:然而呼吸并非生存的'''充分条件'''，生存并非呼吸的'''必要条件'''，因為只會呼吸並不足以讓人生存下去，
:故P为Q的'''充分不必要条件'''，Q是P的'''必要不充分条件'''。

2.若P表「我打過他」，Q表「他被我打過」
: P ↔ Q
:此時這2個條件互為「'''充分（且）必要条件'''」。<ref>{{cite web|url = http://terms.naer.edu.tw/detail/1304291/ | title= 充分條件Sufficient Condition | publisher = [[中華民國教育部|中華民國教育部]]| work = 國家教育研究院 | location = 新北市三峽區}}</ref>
