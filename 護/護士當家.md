《'''護士當家'''》（{{lang-en|'''''Nurse Jackie'''''}}）是一部美國醫學喜劇類電視劇。本劇於2009年6月8日在[[Showtime電視網|Showtime電視網]]上首播<ref>{{Cite web |url = http://www.nydailynews.com/entertainment/tv-movies/nurse-jackie-star-edie-falco-mary-louise-parker-strong-women-lift-showtime-article-1.363757 |title = 'Nurse Jackie' star Edie Falco, Mary-Louise Parker, more strong women lift Showtime |author = Cristina Kinon |date = 2009-02-15 |publisher = New York Daily News |language = en}}</ref>，總共播放了七季，於2015年6月28日完結。

本劇的主演是[[艾迪·法柯|艾迪·法柯]]，飾演Jackie Peyton，是一名[[紐約市|紐約市]]All Saints'醫院的[[急症室|急症室]]護士。<ref>{{Cite web |url = http://www.thefutoncritic.com/reviews.aspx?id=20090608_nursejackie |title = The Futon's First Look: "Nurse Jackie" (Showtime) |date = 2009-06-08 |publisher = FutonCritic |language = en}}</ref><ref>{{Cite web |url = https://nypost.com/2008/06/30/nurse-edie/ |title = Nurse Edie: First Look at Sopranos Star's Dark, New Hospital Comedy |author = Michael Starr |date = 2008-06-30 |publisher = New York Post |language = en}}</ref>
