== 概要 ==
傳說是由開設[[京都|京都]][[石清水八幡宮|石清水八幡宮]]的[[行教|行教]]所開創，護國寺在[[明治時期|明治時期]][[神佛分離|神佛分離]]以前，是賀集八幡宮（賀集八幡神社、淡路國府八幡宮）的[[神宮寺|神宮寺]]。由於賀集莊在[[貞應|貞應]]2年（1223年）的[[淡路國|淡路國]]大田文<ref>大田文是指日本中世時期，尤其是鎌倉時期，以國為單位所紀載製作的國内公領、莊園別的田地面積和所有關係等等的文書。</ref>紀錄中，是屬於高野山寶幢院的領土，所以賀集八幡宮和護國寺在中世時期皆受到淡路守護[[細川氏|細川氏]]的庇護，但在[[至德_(日本)|至德]]3年（1386年）因火災而燒毀，到[[寛永|寛永]]8年（1631年）才再興。

祭祀的本尊是[[s:佛學大辭典/胎藏界大日如來|胎藏大日如來]]，是在[[平安时代|平安时代]]後期所作，被指定為日本國家[[重要文化財|重要文化財]]<ref>{{Cite web |url=http://www.city.minamiawaji.hyogo.jp/soshiki/shougai/dainichinyoraizazou.html |title=木造大日如来坐像 - 南あわじ市ホームページ |access-date=2012-08-07 |archive-url=https://web.archive.org/web/20121103070649/http://www.city.minamiawaji.hyogo.jp/soshiki/shougai/dainichinyoraizazou.html |archive-date=2012-11-03 |dead-url=yes }}</ref>。

在近世，是[[阿波國|阿波]][[蜂須賀氏|蜂須賀家]]的[[菩提寺_(日本)|菩提寺]]<ref>『全国寺院名鑑 近畿編』282頁</ref>。

池泉回遊式庭園是南淡路市的指定文化財（名勝）（[[1988年|1988年]]指定）<ref>{{Cite web |url=http://www.city.minamiawaji.hyogo.jp/soshiki/shougai/shishitei-bunkazai.html |title=市指定文化財 - 南あわじ市ホームページ |access-date=2012-08-07 |archive-url=https://web.archive.org/web/20120626155335/http://www.city.minamiawaji.hyogo.jp/soshiki/shougai/shishitei-bunkazai.html |archive-date=2012-06-26 |dead-url=yes }}</ref>。

除了本尊也祭祀[[布袋和尚|布袋]]，是屬於淡路[[七福神|七福神]]靈場會其一<ref name="ex01">『古寺名刹の百科事典』416頁
</ref>，據說會帶來財運。
