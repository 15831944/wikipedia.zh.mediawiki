'''耆那教'''（{{lang-sa|जैनधर्म}} ''{{IAST|Jainadharma}}''；{{lang-ta|சமணம்}} ''{{transl|ta|Samaṇam}}''；{{lang-en|Jainism}}），是起源于[[古印度|古印度]]的古老[[宗教|宗教]]之一<ref name="呂凱文">
{{cite journal 
  |author= 呂凱文
  |year= 2006
  |title= 當佛教遇見耆那教──初期佛教聖典中的宗教競爭與詮釋效應
  |journal= 中華佛學學報
  |volume= 19
  |pages= p179-207
  |url= http://www.chibs.edu.tw/ch_html/chbj/19/chbj1908.htm
  |accessdate= 2013-10-10
}}</ref>，有其独立的[[信仰|信仰]]和[[哲学|哲学]]。創始人為[[伐達摩那|伐達摩那]]（又稱'''[[摩訶毘羅|摩訶毘羅]]'''，意為大雄，前599年—前527年），是[[沙門思潮|沙門思潮]]中「[[六師外道|六師外道]]」之一的'''尼乾陀若提子'''<ref name="呂凱文"/>，他比[[佛教|佛教]]的創始人[[释迦牟尼|释迦牟尼]]早出生，耆那教的中心教義主要由他建立。耆那教[[教義|教義]]對現代印度的影響，大於同樣起源自印度的[[佛教|佛教]]，[[甘地|甘地]]就受到耆那教的許多影響<ref>[http://read.jd.com/16038/773013.html 與佛陀同時代的大雄 (3)]</ref>。目前耆那教有四五百萬信徒，大部分生活在[[印度|印度]]。

==名稱==
[[佛經|佛經]]中，稱呼這個教派為'''尼乾'''或'''尼犍'''，是離開束縛之意<ref>《慧琳音義》：「尼乾陀此-{云}-無繼，是外道總名也。」</ref><ref>《玄應音義》：「'''尼乾'''或作'''尼乾陀'''，應言'''泥健連他'''，譯-{云}-不繫也。」</ref>。將這個教派的信徒稱為'''尼乾子'''，又稱他們為[[裸形外道|裸形外道]]、裸體外道。

==起源 ==
[[File:Tirthankaras.jpg|thumb]]右為第二十四代祖師[[伐達摩那|伐達摩那]]。]]
[[File:Worshippers_leaving_the_temple_in_Ranakpur.jpg|thumb]][[拉那克普|拉那克普]]（Ranakpur）的耆那教廟]]

耆那教早期的歷史並不清楚，創始人[[伐達摩那|伐達摩那]]自稱自己是第24代[[蒂爾丹嘉拉|蒂爾丹嘉拉]]（Tīrthaṅkara），即祖師。他之上還有代代相傳的23代祖。伐達摩那被稱為'''大雄尊者'''（'''Mahavira'''，音譯'''摩訶毘羅'''，即'''偉大的英雄'''），是为第24代祖師，也是最后一位祖师。据耆那哲学所述，所有的祖师都是人类。但是，他们已经通过[[禅定|禅定]]和自我实现达到了完美或觉悟的状态——他们就是耆那（jina）即勝者。

耆那教對教主的尊稱其實為“佛陀”Budha，佛教稱呼釋迦牟尼為佛陀，實質上是沿用了耆那教的稱呼。佛教還未出現在印度前，耆那教已經先存在了。

伐達摩那诞生于2,593年前的{{en-link|Chaitra|Chaitra}}月上弦月第13天（前599年3月19日）。出生地在[[印度|印度]][[比哈爾邦|比哈爾邦]]。他的生日以后被作为了{{le|大雄节|Mahavir Jayanti}}，相傳他在三十歲出家修行，四十二歲成道開始[[傳道|傳道]]，他反對[[種姓制度|種姓制度]]，主張[[和平主義|和平主義]]，[[反暴力|反暴力]]，還有[[不殺生|不殺生]]、[[不妄語|不妄語]]、[[不偷盜|不偷盜]]、[[不邪淫|不邪淫]]及[[不執着|不執着]]（不贪着，通常专对钱财，故也称不蓄财或不贪财）的五條[[清規|清規]]。七十二歲時去世<ref name="呂凱文"/>。伐達摩那生前弟子多人，賢者十一人，不少門徒因修道嚴苦，斷食而死，只有[[印德拉菩提|印德拉菩提]]（Indrabhūti）與[[蘇達摩|蘇達摩]]（Sudharma）繼續[[傳道|傳道]]。

耆那教初時，[[羯陵伽|羯陵伽国]]的加拉衛羅王（Kharavela）是耆那教的護法名王（類似佛教的[[阿育王|阿育王]]），後來一度衰落，八世紀時復興（[[婆羅門教|婆羅門教]]亦於此時復興，演化為今日的[[印度教|印度教]]），後因回教徒統治又再度衰落。踏入近代後，耆那教有了新的發展，雖然人數不多，但逐漸為世界所認識，印度人自己亦重新審視這個古老的宗教。

== 耆那宇宙观 ==
耆那教中不存在[[婆羅門教|婆羅門教]][[三相神|三相神]]，即创造神，保护神和破坏神[[三位一體_(印度教)|三位一體]]的概念。另外，神为了[[降魔|降魔]]而[[化身|化身]]成人的想法在耆那教中也不被接受，伐達摩那認為世界是由[[極微|極微]]構成，不由神造<ref>

== 信仰和实践 ==
伐達摩那認為：“[[无始|无始]]以来，所有的生物（[[灵魂|灵魂]]）都是在[[业|业]]的束缚中，而业是善或恶行为的积聚。在业的影响下，[[灵魂|灵魂]]习惯于在[[物质|物质]]财富中寻找快乐，从而就有[[暴力|暴力]]的想法、行为、愤怒、怨恨、贪婪及诸如此类的罪恶在心中扎根。这些结果也导致业的进一步积聚。”並教导：“[[正知|正知]]（samyak-jnana）、[[正见|正见]]（samyak-darshana）和[[正行|正行]]（samyak-charitra）是自我解脱的真实之路。”

# 不偷窃（{{en-link|Asteya|Asteya}}）——不拿经由不当方法、途径得到的东西。
# 纯洁行（[[Brahmacharya|Brahmacharya]]）——不沉溺于肉欲的快乐。
# 不执着（{{en-link|Aparigraha|Aparigraha}}）——完全不粘着人、地和物，多指钱财，也有译作不贪财者。

耆那教徒以五誓言作为生活的核心。但如果不明智于非绝对论（{{en-link|Anekantavada|Anekantavada}}）和相对论（{{en-link|Syadvada|Syadvada}}）哲学，这些誓言律就不可能被充分且完整的实行。出家人严格地、完全地遵守这些誓言；另一方面，在家者也可只遵守与自己的实际生活相适应的部分誓言。

被伐達摩那吸引的人们包括了各阶层的人，上至[[王公|王公]]貴族，下至販夫走卒，有男有女、有貴有賤，伐達摩那将他的信徒归为4类，即男修士（Sadhu），女修士（Sadhvi），男在家众（Shravak）和女在家众（Shravika）<ref>{{Cite web

== 现代影响 ==
*1981年印度人口普查结果显示，印度的耆那教徒人数爲320万<ref>{{cite|url=http://www.cqvip.com/qk/92785x/199802/1002227650.html|title=“印度教国家”理论与印度的教派斗争|publisher=《当代亚太》1996年第3期|author=朱明忠(中国社科院亚太研究所 副研究员)}}</ref>；截至1991年3月1日，印度共有人口8.5亿，其中耆那教徒占0.41%，约340多万<ref>{{cite|url=http://www.cqvip.com/qk/92785x/199802/1002227650.html|title=印度的教派问题|publisher=《南亚研究》1998年第2期|author=姜景奎(北京大学东方学系)}}</ref>。其中白衣派流行於[[古吉拉特|古吉拉特]]地区，空衣派流行於[[邁索爾|邁索爾]]地区。他们拥有耆那教友谊协会、耆那教青年大会、世界耆那教传教会等组织；在国外也有组织，如英国耆那教兄弟会等。
