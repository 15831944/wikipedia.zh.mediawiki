'''皂化反应'''（{{lang-en|Saponification}}）是一种成[[皂|皂]]的化学反应。

== 化学反应 ==
皂化反应是一个[[放热反应|放热反应]]。

皂化反应是一个较慢的[[化学反应|化学反应]]，为了加快反应速度，可以在化学反应的过程中：

* 保持系统的较高温度。(加熱使熟成反應加速)

* 以物理方式不断搅拌溶液以增加分子碰撞的数量。

* 加入酒精（[[乙醇|乙醇]]），使混合得更充分。

== 反应机制 ==
*第一步：[[酯|酯]]与[[氢氧根|氢氧根]]的[[加成反应|加成反应]]
*第二步：[[消除反应|消除反应]]
*第三步：[[酸碱反应|酸碱反应]]（不可逆）

== 皂化值(皂化價） ==
习惯上，将1g油脂碱水解所消耗的[[氢氧化钾|氢氧化钾]]毫克数定义为'''皂化值'''。也可以利用它计算油脂的相对分子质量。

== 肥皂的制成 ==
[[脂肪|脂肪]]和[[植物油|植物油]]的主要成分是[[三酸甘油酯|三酸甘油酯]]，它的碱水解方程式为：

<math>\begin{array}{lcl}
CH_{2}OOCR\\
|\\
CHOOCR &+ 3NaOH \rightarrow 3RCOONa + CH_{2}OHCHOHCH_{2}OH\\
|\\
CH_{2}OOCR\end{array}</math>

R基可能不同，但生成的R-COONa都可以做[[肥皂|肥皂]]。常见的R-有：
*C{{sub|17}}H{{sub|33}}-：8-十七碳烯基。R-COOH为[[油酸|油酸]]。
*C{{sub|15}}H{{sub|31}}-：正十五烷基。R-COOH为[[软脂酸|软脂酸]]。
*C{{sub|17}}H{{sub|35}}-：正十七烷基。R-COOH为[[硬脂酸|硬脂酸]]。
油酸是[[单不饱和脂肪酸|单不饱和脂肪酸]]，由油水解得；软、硬脂酸都是[[饱和脂肪酸|饱和脂肪酸]]，由脂肪水解得。

如果使用氢氧化钾水解，得到的肥皂是软的。

向溶液中加入[[氯化钠|氯化钠]]可以减小脂肪酸盐的溶解度从而分离出脂肪酸盐，这一过程叫[[盐析|盐析]]。高级脂肪酸盐是肥皂的主要成分，经填充剂处理可得块状肥皂。

== 去污原理 ==
[[肥皂|肥皂]]分子有一端由许多[[碳|碳]]和[[氢|氢]]所组成的[[长链|长链]]，称为[[亲油端|亲油端]]；另一端则为亲水性的原子团，称为[[亲水端|亲水端]]。

使用肥皂时，油污被[[亲脂性|親油]]端吸附着，再由[[亲水|亲水]]端牵入水中，达到洗净效果。
