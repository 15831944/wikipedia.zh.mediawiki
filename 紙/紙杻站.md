'''纸杻站'''（{{kor|諺=지축역|漢=紙杻驛|羅=Jichuk yeok|f=y}}）是一個位於[[京畿道|京畿道]][[高陽市|高陽市]]{{link-ko|德陽區|덕양구}}孝子洞，屬於[[一山線|一山線]]與[[首爾地鐵3號線|首爾地鐵3號線]]的[[電鐵站|電鐵站]]。

== 车站结构 ==
地上2層規模的地面車站，月台位於地上2樓。候車室與站務設施分別開設於[[首爾交通公社|首爾交通公社]]（1號出口）側與[[韓國鐵道公社|韓國鐵道公社]]（2號出口）側，閘口與出口也是各設一個。各出口互相面向對方。

最初開通開因為當時只建設了[[首爾地鐵3號線|首爾地鐵3號線]]6節列車的{{link-ko|首爾地下鐵公社|서울지하철공사}}路段，[[一山線|一山線]]開工的同時增設至10節，需要[[大韓民國鐵道廳|大韓民國鐵道廳]]建設餘下4節的路段。因此[[首爾交通公社|首爾交通公社]]與[[韓國鐵道公社|韓國鐵道公社]]的月台管轄區域比例約為6:4，管轄的分界線標記於軌道旁邊。然而後來[[月台幕門|月台幕門]]、列車到着資訊展都是[[首爾交通公社|首爾交通公社]]設置的。另外單次用交通卡試行以前在2號出口的乘車券自動售票機，交通卡識別終端機則是由[[韓國鐵道公社|韓國鐵道公社]]管理，現在由[[首爾交通公社|首爾交通公社]]管理，但仍使用[[韓國鐵道公社|韓國鐵道公社]]作為資訊標誌版的製品使用。不過2號出口的結算由[[韓國鐵道公社|韓國鐵道公社]]進行，通過乘車量也是各自新設。

1990年7月13日開通當時月台是1面1線（[[側式月台|側式月台]]）規模，因為3號線全部列車的回車處理容量不足，只進行一部分列車的起終着。以後1996年[[一山線|一山線]]開通與月台擴建為1面2線成為現在的規模，回車線也一併拆除，從此停止在此站起終着。鄰近[[紙杻車輛基地|紙杻車輛基地]]，但車輛基地的引入線只連結[[三松站_(京畿道)|三松站]]和[[舊把撥站|舊把撥站]]。

== 历史 ==
* 1990年4月1日：站名決定為'''紙杻站'''（{{lang|ko|지축역}}）<ref>[http://www2.seoul.go.kr/snews/data/CN_MST/1990-1472.pdf#page=8&view=FitV&scrollbar=1&toolbar=1&statusbar=1&messages=1&navpanes=1 서울특별시고시 제38호], 1990년 2월 8일.</ref>。
* 1990年7月13日：中午12時起[[首爾地鐵3號線|首爾地鐵3號線]]延長開通的同時開始營業。
* 1996年1月30日：[[一山線|一山線]]開通，成為中途站。
== 使用人數 ==
站舍業務完全由[[首爾交通公社|首爾交通公社]]管理，但是閘口按照公司設置，收益精算也是分開的。
