'''吉格梅·納姆耶爾·旺楚克'''（'''Jigme Namgyel Wangchuck'''，{{bd|2016年|2月5日|catIdx=W|}}）<ref>{{Cite news|url=http://www.kuenselonline.com/his-royal-highness-the-gyalsey-is-born-2/|title=His Royal Highness The Gyalsey is born|author=|date=2016-02-06|work=|newspaper=KuenselOnline|accessdate=2017-10-01|language=en-US}}</ref>，是[[不丹|不丹]][[储君|王儲]]。他是不丹國王[[吉格梅·凯萨尔·纳姆耶尔·旺楚克|吉格梅·凯萨尔·纳姆耶尔·旺楚克]]和王后[[吉增·佩玛|吉增·佩玛]]的長子，為不丹第一位的王位繼承人。王儲的名字在他出生兩個月後在同年4月16日才被正式公布<ref>{{Cite news|url=http://www.kuenselonline.com/hrh-the-gyalsey-to-be-named-on-zhabdrung-kuchoe/|title=HRH The Gyalsey to be named on Zhabdrung Kuchoe|author=|date=2016-03-08|work=|newspaper=KuenselOnline|accessdate=2017-10-01|language=en-US}}</ref>。

==生平==
2016年2月，不丹国王旺楚克和王后佩玛的长子吉格梅·纳姆耶尔·旺楚克诞生。小王子出生后，其写真由不丹一个文化组织在网上发布。生下后一年多，小王子已随父母出席了不少重要场合。2017年10月31日，不丹国王旺楚克抵达[[印度|印度]]开启为期4天的访问，与不丹国王同行的有年仅1岁的不丹王子吉格梅·纳姆耶尔·旺楚克，这也是他出生以来首次外访，受到媒体和网友关注<ref>{{Cite web |url = http://news.ifeng.com/a/20171103/52930375_0.shtml |title = 随父亲出访印度会见莫迪 不丹小王子萌翻网友 |author =  |date = 2017-11-03 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。
