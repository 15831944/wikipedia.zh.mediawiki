'''吉维尼'''（{{lang-fr|Giverny}}）是[[法国|法国]][[厄尔省|厄尔省]]的一个[[市镇_(法国)|市镇]]，属于[[莱桑代利区|莱桑代利区]]（Les Andelys）[[埃科县|埃科县]]（Écos）。该市镇总面积6.46平方公里，2009年时的人口为500人。<ref>[http://www.insee.fr/fr/ppp/bases-de-donnees/recensement/populations-legales/france-departements.asp?annee=2009 法国INSEE人口数据-2009年]</ref>。

这个小镇因拥有[[克劳德·莫奈|克劳德·莫奈]]的故居及唯美的[[莫奈花园|莫奈花园]]而享誉全球。

== 地理位置 ==
吉维尼位于距维农五公里处的[[塞纳河|塞纳河]]与[[艾普特河|艾普特河]]（Epte）的交汇口。这里也是维新诺曼底的门户所在。

== 历史 ==

[[File:Monet's_house_in_Giverny.jpg|thumb]]

[[Image:Claude_Monet-Waterlilies.jpg|thumb]]

这个小镇是在[[墨洛溫王朝|墨洛溫王朝]]和[[圣·拉德贡德|圣·拉德贡德]](Sainte Radegonde)所在教会的共同赞助下建立的。公元863年，国王查尔斯二世作为圣·丹尼·勒弗曼（Saint-Denis-le-Ferment）修道院的一名僧侣，曾经拥有过吉维尼小镇。

11世纪时，吉维尼这片封地及其教会归在了位于鲁昂的圣旺修道院（Saint-Ouen）名下。在中世纪，曾先后有几位领主继承了这片领地，但它始终属于圣旺。在法国革命期间，卢瑞耶（Lorier）家族成为了吉维尼的领主。而卢瑞耶先生本人也在1791年成为本地的第一任市长。

1883年克劳德·莫奈开始定居在这里。起初他只是租了一栋别墅；1890年，他将其买了下来。他对别墅进行了改造，并将屋旁的一片果园完全改建成为一座异常漂亮的花园。1893年，在阿普特河畔，百合谷旁，他创作了传世名作《[[睡莲_(莫奈)|睡莲]]》系列。1926年12月5日，莫奈去世，并被安葬于毗邻的教堂墓地。在莫奈生活的时代，不断有一些艺术家，尤其是来自美国的艺术家，定居在这个美丽的小镇。有的还成为莫奈的朋友。

* 1992年，美国蒂拉(Terra)艺术基金会在这里成立了美国艺术博物馆。
* 1996年，美国[[总统夫人|总统夫人]][[希拉里·克林顿|希拉里·克林顿]]到访。
* 2001年，重新装修过莫奈故居及莫奈花园的房主 — [[拉尔德·范德·肯普|拉尔德·范德·肯普]]去世。他被埋葬于离莫奈墓地不远的地方。
* 2007年，[[日本|日本]][[天皇|天皇]]到访。
* 2009年，美国[[艺术博物馆|艺术博物馆]]改名为吉维尼印象画派艺术博物馆。

== 小镇名人 ==
* 莱尼·埃斯库德罗(生于1392年11月5日), 歌唱家, 曾在此居住.
* 勒内·梅耶 (1895-1972), 前总统，前部长，作家，小镇镇长，曾居住在小镇的法莱斯村。
* 克劳德·莫奈, 印象画派代表画家, 1926年5月26日卒于此。
* 让-玛丽·图勒古阿(Jean-Marie Toulgouat),画家；1927年9月15日生于此，2006年1月10日卒于此。
* 蒂埃里·胡瓯 (生于1958年4月25日), 城市景观设计工程师。
* 日尔曼·德拉维聂(1790-1868), 剧作家, 1790年二月生于吉维尼。
