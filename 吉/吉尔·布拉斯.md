《'''吉尔·布拉斯·德·桑蒂亚纳传'''》（{{lang-fr|''L'Histoire de Gil Blas de Santillane''}}，{{IPA-fr|listwaʁ də ʒil blɑ də sɑ̃tijan|}}，通称《'''吉尔·布拉斯'''》）是18世纪法国作家[[阿兰-勒内·勒萨日|阿兰-勒内·勒萨日]]所著的[[长篇小说|长篇]][[流浪汉小说|流浪汉小说]]。讲述了一个天真无知的西班牙青年冲破[[封建制度_(欧洲)|封建社会]]的障碍与束缚，最终过上幸福生活的故事<ref>{{cite book |author=萧枫 |title=世界古代文学发展概论 |page=第124页 |url=https://books.google.com/books?id=vv8jDwAAQBAJ&pg=PA124& |language=zh}}</ref>。
==情节概要==
吉尔·布拉斯是出生在[[西班牙|西班牙]]北部[[坎塔布里亚|坎塔布里亚]]地区[[海邊的散提亞拿|滨海桑蒂亚纳]]（Santillana）的贫苦[[青年|青年]]。但他天资聪慧，17岁时在[[舅舅|舅舅]]帮助下，从[[奥维耶多|奥维耶多]]前往[[萨拉曼卡大学|萨拉曼卡大学]]求学。一路上，他遭遇了[[土匪|土匪]]、[[骗子|骗子]]和[[强盗|强盗]]，甚至误入土匪窝，暂时与这些坏人成为同伙。在逃脱匪窝后，又被一女骗子骗取了钱财。在垂头丧气之际，他偶遇舅舅老友的儿子法布里斯。在法布里斯帮助下，吉尔·布拉斯开始重新寻找职业。他先后做过大[[司铎|司铎]]的[[佣人|佣人]]、[[医生|医生]]的助手、[[侯爵|侯爵]]夫人的侍从、公子的男仆、大[[主教|主教]]的秘书等。在被虚伪的大主教逐出家门后，他依靠学到的本领，成为了[[马德里|马德里]][[朝廷|朝廷]]首相的[[秘书|秘书]]，并受到了赏识和重用。此后，吉尔·布拉斯便开始以自己首相秘书的身份帮人买官办事，贪图钱财，对老家的亲人置之不理，最后还失去了好朋友法布里斯。最后，吉尔·布拉斯在经历一些事情后突然醒悟，回到老家探亲。不料，乡亲们却认为他不孝，欲投石砸他，但幸得母亲相救。在安顿好母亲后，吉尔·布拉斯离开家乡来到一个[[城堡|城堡]]，过上了幸福安逸的生活。<ref>{{cite web |title=吉尔·布拉斯 [法国]勒萨日 |url=https://www.kekeshici.com/shicizhoubian/xiaoshuo/waiguo/31074.html |publisher=可可诗词网 |archiveurl=https://web.archive.org/web/20190307072048/https://www.kekeshici.com/shicizhoubian/xiaoshuo/waiguo/31074.html |archivedate=2019-03-07 |language=zh |date=2018-11-06}}</ref>
==分析==
全书分两次出版，前后相隔二十年。第一、二部出版的1715年正是法国国王[[路易十四|路易十四]]去世，[[路易十五|路易十五]]即位的一年。小说里反映的是那两个朝代真实的法国社会<ref>{{cite book |author=杨绛 |title=杨绛作品集|pages=第220页|volume=第3卷 |date=1993年 |publisher=中国社会科学出版社 |url=https://books.google.com/books?id=AY4vAQAAIAAJ |language=zh}}</ref>。勒萨日不用工笔描画，只用粗线条勾勒，通过他的举动言谈和不同的境地中的动作，就将吉尔·布拉斯的个性鲜明地表现了出来<ref>{{cite journal |author=冯斌 |title=试析《吉尔·布拉斯》中的三位人物形象 |journal=参花(下) |date=2017年 |issue=第4期 |doi=10.3969/j.issn.1008-8407.2017.04.082 |language=zh |issn=1008-8407}}</ref>。
==中译本==
杨绛在译完《[[小癞子|小癞子]]》后，怕荒疏了[[法文|法文]]，就开始翻译《吉尔·布拉斯》。自1954年1月起，译文在《世界文学》分期发表。但杨绛自认为翻译得不够好，在经过了[[钱锺书|钱锺书]]的校对和磨合后，1955年才交付出版社。<ref>{{cite web |author=杨绛 |title=杨绛回忆与钱钟书的翻译往事 |url=http://book.ifeng.com/yeneizixun/special/fanyijia/detail_2012_08/09/16681366_0.shtml |publisher=[[凤凰网|凤凰网]] |archiveurl=https://web.archive.org/web/20160103025630/http://book.ifeng.com/yeneizixun/special/fanyijia/detail_2012_08/09/16681366_0.shtml |archivedate=2016年1月3日 |language=zh |date=2012年8月9日}}</ref>1956年1月，[[人民文学出版社|人民文学出版社]]出版初版，1962年9月再版<ref>{{cite web |author=范旭仑 |title=钱锺书翻译《吉尔·布拉斯》 |url=http://www.sohu.com/a/232216940_161795 |publisher=[[搜狐|搜狐]] |archiveurl=https://web.archive.org/web/20190307082922/http://www.sohu.com/a/232216940_161795 |archivedate=2019-03-07 |language=zh |date=2018-05-20}}</ref>。
