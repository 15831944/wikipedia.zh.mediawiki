'''吉田步美'''（{{lang|ja|吉田 歩美}}/{{lang|ja|よしだ あゆみ}}，Yoshida Ayumi），[[少年侦探团|少年侦探团]]的成员，是日本[[漫画|漫画]]和[[动画|动画]]作品《[[名侦探柯南|名侦探柯南]]》中的[[名偵探柯南角色列表|虚构人物]]。

命名來自[[日本|日本]][[推理小說|推理小說]]家'''{{tsl|ja|北川步實}}'''及日本推理劇中的[[刑事警察|刑警]]'''吉田'''茂（並非同名的日本前[[日本內閣總理大臣|首相]][[吉田茂|吉田茂]]）。

== 人物 ==
7岁，体重15公斤，就讀於[[东京都|东京都]]米花市帝丹小学一年B班。個性天真善良，待人親切有禮；時常微笑，顯得十分活泼可爱，令[[小岛元太|小島元太]]和[[圓谷光彥|圓谷光彥]]为之喜爱。

曾差點在圖書館讓少年偵探團滅團，當他們晚上潛入圖書館調查，因為步美開燈，讓館長(兇手)發現了他們。

== 形象 ==
有很強的正義感，經常幫助他人；也具有不錯的交际能力，會主動接觸、認識班上的新同学，如[[江户川柯南|江户川柯南]]、[[灰原哀|灰原哀]]等。有點自戀，常常說自己可愛，某次米花市發生幼童綁架事件，他很怕自己因為太可愛被綁架。

與元太光彥總認為自己計畫周詳，卻時常找死找麻煩，有次灰原落難，也不知哪里來的自信要救灰原，救不了人反倒被抓了。
 
此外，她喜愛'''假面超人'''，爱吃[[寿司|寿司]]，也曾梦想[[環遊世界|环游世界]]<ref group="參">漫畫第1卷，TV 4《大都會暗號地圖事件》</ref>。

== 情感 ==


曾多次受[[江戶川柯南|柯南]]幫助，因而欣賞其勇氣和聰穎；心裡非常喜欢[[江戶川柯南|柯南]]，成天纏著柯南，見不得其他女生接近他，但[[柯南|柯南]]總是把她當妹妹，遭遇危難時，会将他視為自己的精神支柱。不時會幻想著兩人長大後成为一对恋人的情景，也曾親吻[[江戶川柯南|柯南]]，但卻害他被[[圓谷光彥|光彥]]與[[小嶋元太|元太]]追打<ref group="參">TV 4《大都會暗號地圖事件》、109 (日版集数，海外版集数115)《偵探團大追擊事件》</ref>。
