'''吉奥拉·施皮格尔'''（{{lang-he|גיורא שפיגל}}；{{bd|1947年|7月27日||}}）是一名前[[以色列|以色列]][[足球|足球运动员]]和教练。<ref>[http://www.haaretz.com/hasen/spages/1128744.html Giora Spiegel is done coaching, but he's not done]</ref>目前他是在[[以色列國家男子足球隊|以色列国家队]]中待的最久的人，时间长达14年又357天。

== 早期生活 ==
吉奥拉·施皮格尔是{{le|埃利泽·斯皮格尔|Eliezer Spiegel}}的儿子，他曾效力于[[彼达迪华马卡比足球会|彼达迪华马卡比]]和[[以色列国家足球队|以色列国家队]]。<ref >{{Cite web|url=http://www.worldfootball.net/player_summary/giora-spiegel/|title=Giora Spiegel|publisher=worldfootball.net}}</ref>施皮格尔曾就读于{{le|赫兹利亚希伯来高中|Herzliya Hebrew Gymnasium}}。

== 职业球员生涯 ==
施皮格尔年轻时，曾效力于[[特拉維夫馬卡比足球會|特拉维夫马卡比队]]，他很早就被认为是未来的天才球员。到了17岁，他便带领以色列国家U-21队参加了亚锦赛。到18岁时，他就被提拔到了[[以色列國家男子足球隊|成年国家队]]。<ref >{{Cite web|url=http://www.fifa.com/fifa-tournaments/players-coaches/people=58340/index.html|title=Giora SPIEGEL|publisher=fifa.com}}</ref>1973年，他与马卡比的总经理：杰瑞·贝特·哈勒维（Jerry Beit haLevi）一起转会到了法国的一家俱乐部。之后他离开[[法国|法国]]，于1979年重回[[以色列|以色列]]加盟特拉维夫马卡比。<ref >{{Cite web|url=https://www.transfermarkt.com/giora-spiegel/profil/spieler/234702|title=Giora Spiegel|publisher=transfermarkt.com}}</ref>

在[[以色列国家足球队|以色列队]]中施皮格尔也有亮眼的发挥。他曾在[[1968年亚洲杯|1968年亚洲杯]]上攻进4球，当选该届杯赛的最佳射手。<ref >{{Cite web|url=http://www.hrszyl.com/ztnews/d5g97jjae455ag5b9j4i8cdj.html|title=历届“亚洲杯”冠军球队及比分和射手|publisher=和仁网}}</ref>

== 管理职业生涯 ==
20世纪80年代中期，施皮格尔在[[佩塔提克瓦工人足球俱乐部|佩塔提克瓦工人足球俱乐部]]开始了他的主教练生涯。几年后，他转任[[特拉维夫马卡比|特拉维夫马卡比]]任职主教练，并为球队赢得了[[以色列国家杯|以色列国家杯]]。因为与球员有所争执，在球队以10–0击败了[[海法马卡比足球俱乐部|海法马卡比]]后，施皮格尔被解雇了。1989年，他转会到了{{le|比尼特拉维夫足球俱乐部|Bnei Yehuda Tel Aviv F.C.|比尼特拉维夫}}，并于1990年率领球队赢得[[以色列超级杯|以色列超级杯]]冠军。1993年，他转会到了[[海法马卡比足球俱乐部|海法马卡比足球俱乐部]]执教。球队在那一年赢得了[[以色列足球超级联赛|以色列国家联赛]]冠军，全季48场保持不败，夺得联赛冠军。在他的率领下，海法马卡比又分别在1995年和1998年两次赢得了国家杯。1999年，施皮格尔回到{{le|比尼特拉维夫足球俱乐部|Bnei Yehuda Tel Aviv F.C.}}执教。在经过一个不算成功的赛季后，他转会到了[[里雄莱锡安夏普尔足球会|里雄莱锡安夏普尔足球会]]担任了两年的主教练。<ref >{{Cite web|url=http://www.redirectify.com/people/giora-spiegel.html|title=Giora Spiegel Sports manager|publisher=redirectify.com}}</ref>

2007年7月，在离开以色列足坛五年之后，[[耶路撒冷贝塔足球俱乐部|耶路撒冷贝塔足球俱乐部]]聘用了施皮格尔任[[总经理|总经理]]一职。那一年，球队赢得了[[雙料冠軍|雙料冠軍]]。2008年8月，施皮格尔宣布退休。<ref >{{Cite web|url=https://www.haaretz.com/soccer-giora-spiegel-is-done-coaching-but-he-s-not-done-1.4028|title=Soccer / Giora Spiegel Is Done Coaching, but He's Not Done|publisher=haaretz.com}}</ref>
