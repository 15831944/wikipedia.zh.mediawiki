'''吉隆坡火车总站'''（{{lang-ms|Stesen Kereta Api Kuala Lumpur}}）是一个位于[[马来西亚|马来西亚]][[吉隆坡|吉隆坡]]的[[火车站|火车站]]，1910年建成<ref>{{cite web|url=http://www.keretapi.com/the-kuala-lumpur-railway-station.html|title=Kuala Lampur Railway Station|publisher=keretapi.com|accessdate=25 August 2012|deadurl=yes|archiveurl=https://web.archive.org/web/20120531013954/http://www.keretapi.com/the-kuala-lumpur-railway-station.html|archivedate=2012年5月31日}}</ref>，以取代建于1886年的舊火车站。该站曾是吉隆坡的铁路枢纽，2001年起[[吉隆坡中央車站|吉隆坡中央車站]]承担了大部分的功能。该车站以独特的建筑著称，为采用东方和西方设计的混合体。

该站所在街道名为“苏丹希沙慕丁路”（''Jalan Sultan Hishamuddin''），原名胜利大街（Victory Avenue），属于白沙罗路（Damansara Road）的一部分。该站毗邻建筑风格相似的铁路局大楼，以及[[國家清真寺_(馬來西亞)|國家清真寺]]和达亚布米大厦。[[中央藝術坊站|中央藝術坊站]]位于400米外，[[巴生河|巴生河]]对岸。
