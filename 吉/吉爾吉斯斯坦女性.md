'''吉爾吉斯斯坦女性'''享有很高的自由但仍然比男性低<ref name= "Hays, 2008">{{cite web|url= http://factsanddetails.com/central-asia/Kyrgyzstan/sub8_5b/entry-4758.html|title= 吉爾吉斯斯坦的家庭和婦女|first= 杰弗里|last= 海斯|year= 2008|language= en}}</ref>，她們經常面臨[[強姦|強姦]]、[[家庭暴力|家庭暴力]]、[[性騷擾|性騷擾]]和[[歧視|歧視]]。

== 傳統角色 ==
吉爾吉斯斯坦女性被認為是民族文化的承擔者，因此他們在國家精神中培育後代的作用是不可替代的<ref>{{cite web|url= https://centralasien.org/2017/01/09/part-2-womens-roles-in-the-construction-of-identity-in-kyrgyzstan-ideas-and-expectations-of-women/|title= 女性在吉爾吉斯斯坦身份建構中的作用：女性的理念與期待|date= {{start_date|2017|01|09}}|first= 伊莎貝爾|last= 佩爾松|language= en}}</ref>。在傳統觀點中，她們有能力做出日常決定，並被期望打理家務，男子則在商業和政治部門中擁有更大的權力<ref name= "Hays, 2008"/>。

== 受教育情況 ==
據АКИpress網報導，吉國國家統計委員會報告顯示，目前吉爾吉斯斯坦15-17歲未成年少女的生育率呈現上升趨勢，在2015年的數據蒐集中，每一千名女性中有7名未成年少女生育，由於過早為人母，這些女性接受基礎教育的機會也大大減少<ref>{{cite web|url= http://2016cn.siluxgc.com/html/C1413/201703/6459743535894830593.shtml|title= 吉尔吉斯斯坦未成年少女生育率上升|date= {{start_date|2017|03|07}}|work= AKIpress|publisher= 絲路新觀察|language= zh-hans}}</ref>。

2018年，吉爾吉斯斯坦女性太空學校開展一項計劃－發射吉爾吉斯斯坦首顆衛星<ref>{{cite web|url= https://www.calvertjournal.com/news/show/9759/this-pioneering-space-school-for-girls-is-launching-kyrgyzstans-first-satel|title= 這個開創性的女孩太空學校正在發射吉爾吉斯斯坦的第一顆衛星|first= 凱蒂|last= 戴維斯|date= {{start_date|2018|03|12}}|work= The Calvert Journal|language= en}}</ref>，而19歲的Alina Anisimova和其他女性希望能在2020年完成這項壯舉<ref>{{cite web|url= https://www.bbc.com/news/av/world-europe-46347757/the-women-launching-kyrgyzstan-s-first-satellite|title= 吉爾吉斯斯坦婦女開發吉爾吉斯斯坦的第一顆衛星
|date= {{start_date|2018|11|27}}|work= [[BBC|BBC]]|language= en}}</ref><ref>{{cite web|url= http://video.caixin.com/2018-12-05/101355685.html|title= 吉爾吉斯斯坦的全女性太空學校|work= [[財新傳媒|財新傳媒]]|date= {{start_date|2018|12|05}}|language= zh-hans}}</ref>。

== 婚姻 ==
=== 一夫多妻制 ===
通過電話進行社會調查的「民主和公民社會聯盟」在報告中指出，大部分吉爾吉斯人，佔67.5%的人反對法律上允許在吉爾吉斯實施[[一夫多妻制|一夫多妻制]]；此外，25.9%的受訪者同意多妻制合法化可以解決[[妓女|妓女]]問題，但大部分居民（62.5%）不同意這個想法，持該意見的女性受訪者佔多數（女性佔69.7%，男性佔54.4%）<ref>{{cite web|url= http://kabar.kg/cn/news/1323/|title= 調查：67.5%的吉爾吉斯人反對一夫多妻制合法化|website= kabar.kg|date= {{start_date|2017|07|25}}|language= zh-hans}}</ref>。
== 強姦和性騷擾 ==
配偶強姦在內的強姦案件很少提交法院審理<ref>{{cite web|url= https://www.state.gov/documents/organization/277529.pdf|title= 2017年國家人權實踐報告 - 吉爾吉斯共和國|work= [[美國國務院|美國國務院]]|language= en-us}}</ref>，很多婦女選擇不報案，以免節外生枝。有人指出農村的傳統習俗－[[綁婚|綁婚]]<ref>{{cite web|url= https://www.newsweek.com/grab-and-run-1634|title= 吉爾吉斯斯坦綁婚的傳統習俗|date= {{start_date|2013|04|11}}|first= Noriko|last= Hayashi|work= [[新聞週刊|新聞週刊]]|language= en-us}}</ref>，是導致年輕女孩強姦問題的其中一個因素。女性在私營部門和大學被性騷擾的情況非常嚴重，但很少被報導<ref name= "Hays, 2008"/>。

== 家庭暴力 ==
2012年的一項政府調查發現，吉爾吉斯斯坦28％的女性或女孩遭受過虐待<ref name="The Guardian 01">{{cite web|url= https://www.theguardian.com/world/2015/nov/04/kyrgyzstan-domestic-violence-women|title= 吉爾吉斯斯坦被指控無視家庭暴力|date= {{start_date|2015|11|04}}|first= 彼得|last= 倫納德|work= [[衛報|《衛報》]]|language= en}}</ref>。 報告指出：「遭受過家庭暴力或強迫婚姻或早婚的女性常常感到巨大的家庭和社會壓力，要求將虐待保密，並為家庭著想。」，因此警察經常忽視家庭暴力案件<ref name="The Guardian 01"/>。

== 性別歧視 ==
由於法律執行不力，吉爾吉斯斯坦存在歧視，女性平均收入低於男性。 農村地區的婦女受到的影響更大，據報告她們的健康狀況較差，受虐待較多，難以在家外工作，財務獨立性較差<ref name= "Hays, 2008"/>。

== 女權運動 ==
'''「新節奏」'''是一個[[吉爾吉斯斯坦|吉爾吉斯斯坦]]小規模的女權主義團體，由三位年輕女性創立，總部設在[[奧什_(吉爾吉斯)|奧什]]。 最初，它不向公眾開放，僅由4名年輕女士組成，但在2016年，他們決定通過組織國際婦女節的遊行來公開。 該組織針對的是14至28歲之間的女性，組織主要宗旨之一是打擊家庭暴力、早婚或新娘綁架和性別歧視。 創始人還希望鼓勵女性接受高等教育。 該小組試圖利用圖像和圖片向公眾宣傳婦女的權利，並向她們展示婦女面臨的問題<ref>{{cite web|url= http://voicesoncentralasia.org/feminism-through-pictures-how-girls-in-the-south-of-the-kyrgyz-republic-fight-for-womens-rights/|title= 通過圖片宣傳女權主義：南吉爾吉斯斯坦女孩如何為婦女權利而戰|first= Elmurat|last=  Ashiraliev|date= {{start_date|2018|02|01}}|language= en}}</ref>。
