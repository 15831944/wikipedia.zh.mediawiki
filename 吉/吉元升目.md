'''吉元升目'''（{{lang-ja|吉元 ますめ}}），[[日本|日本]]女性[[漫畫家|漫畫家]]。出身於[[新潟縣|新潟縣]][[五泉市|五泉市]]。

== 來歷 ==
2003年以短篇「{{lang|ja|ハートについて考えてみる}}」（'''吉元{{lang|ja|マスメ}}'''名義）參加在地「第6回{{lang|ja|にいがたマンガ}}大賞」，入圍中學、高校生部門優秀作品賞（{{link-ja|魔夜峰央|魔夜峰央}}賞）<ref>{{Cite web |date= |url =http://award.manganime-niigata.jp/results_6.html|title =第6回にいがたマンガ大賞|work={{lang|ja|{{link-ja|にいがたマンガ大賞實行委員會|にいがたマンガ大賞}}}}|publisher=新潟市役所 新潟市文化體育部文化政策課|accessdate=2016-08-16|language=ja}}</ref>。

在[[Media_Factory|Media Factory]][[漫畫雜誌|漫畫雜誌]]《{{link-ja|Comic Flapper|コミックフラッパー}}》2013年5月號起連載《[[熊巫女|熊巫女]]》。2015年10月9日決定製作[[電視動畫|電視動畫]]<ref>{{Cite web |url=http://natalie.mu/comic/news/161670 |title=「くまみこ」TVアニメ化、田舎コンプレックスの巫女と喋るクマ描くコメディ|date=2015-10-09|publisher=[[Natalie_(網站)|Comic Natalie]]|accessdate=2016-08-16|language=ja}}</ref>，2016年4月至6月在[[AT-X|AT-X]]、[[東京都會電視台|TOKYO MX]]等台首播。
