{{Japanese|japanese=吉備津彦命|kana=きびつひこのみこと|romaji=Kibitsuhikonomikoto}}

[[Image:Emperor_family_tree_of_Japan_1-10_(ja).svg|thumb]]

'''吉备津彦命'''是古代日本的皇子，是[[孝灵天皇|孝灵天皇]]的第三个皇子，生母是倭国香媛（{{lang|ja|やまとのくにかひめ}}，出自《[[日本書紀|日本書紀]]》<ref>[http://miko.org/~uraki/kuon/furu/text/syoki/syoki04.htm 《日本書紀》卷第四 欠史八代紀]</ref>）；一说生母是意富夜麻登玖邇阿禮比賣命（{{lang|ja|おおやまとくにあれひめのみこと}}，出自《[[古事記|古事記]]》<ref>[http://miko.org/~uraki/kuon/furu/text/kojiki/09_1.htm 《古事記》卷之二 第九章 大和朝廷成立]</ref>）。本来的姓名是'''彦五十狭芹彦命'''（{{lang|ja|ひこいさせりびこのみこと}}），亦作'''比古伊佐勢理毘古命'''。称号是'''吉备冠者'''（{{lang|ja|きびのかじゃ}}）；{{link-ja|四道将軍|四道将軍}}中的一位，在[[山陽道|山陽道]]。

== 事迹 ==

吉备津彦命出生于{{link-ja|庵戸宮|庵戸宮}}（现在的[[奈良县|奈良县]][[磯城郡|磯城郡]][[田原本町|田原本町]]）；[[崇神天皇|崇神天皇]]十年九月九日，被勅命封为四道将軍之一，被派遣到山陽道，前往討罰謀反的[[武埴安彦命|武埴安彦命]]，之後平定了動亂，並殺死了武埴安彦命。
== 桃太郎 ==

吉備津彦命受命攻打[[播磨|播磨]]時遇上惡鬼[[溫羅|溫羅]]，吉備津彦命將其討伐，此事蹟是[[桃太郎|桃太郎]]故事的原型。
