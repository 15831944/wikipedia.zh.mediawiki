'''吉林站'''位于[[中国|中国]][[吉林省|吉林省]][[吉林市|吉林市]][[昌邑区|昌邑区]]重庆路1号，始建于1908年，1912年10月20日正式开始营运<ref>{{Cite web|url=http://www.sytlj.com/ShowArticle.aspx?id=5,22195|title=镶嵌在历史名城的吉林站_中国铁路沈阳局集团有限公司|accessdate=2018-10-13|work=www.sytlj.com}}</ref>，初名'''东关站'''，曾为[[吉林铁路局|吉林铁路局]]管辖的客货运[[特等站_(中鐵)|特等站]]，现为[[中国铁路沈阳局集团有限公司|中国铁路沈阳局集团有限公司]][[吉林车务段|吉林车务段]]管辖的客运[[二等站_(中鐵)|二等站]]。车站设有6个[[岛式站台|岛式站台]]，2个[[侧式站台|侧式站台]]，14条到发线。

== 历史 ==
===早期历史===
[[File:Jilin_Railway_Station_1912.jpg|thumb]]
[[File:Jilin_Railway_Station_1938.jpg|thumb]]
[[File:Jilin_Railway_Station.JPG|thumb]]
1912年，随着吉长铁路竣工通车，吉林站也投入使用。吉林站是全国最早一批火车站之一，在东北具有非常重要的地位。当时的吉林站只是一个平房，在房顶的四周，建有像城墙一样的围栏，站房顶部向上凸起，兼具中国传统建筑风格和俄罗斯建筑风格。<ref>{{cite web|url=http://news.xwhb.com/news/system/2011/01/12/010160600_03.shtml|title=新文化报：吉林站资料|accessdate=2011-08-24}}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

20世纪20年代，因原站房太小，所以拆掉在原址上重新建起了一栋两层小楼，作为新站房。1928年，接入吉敦铁路。而在1929年5月，[[吉海铁路|吉海铁路]]通车之时，却并未接入吉林站，而是选黄旗屯站（今[[吉林西站|吉林西站]]）为吉林总站。

1929年7月，吉海铁路吉林总站至吉林分站支线开始施工。因为日本人不准吉海路与吉长路接轨，所以设立吉林分站，以接近吉长路吉林站，满足中转需要。是年8月完成铺轨，12月27日投入运营，吉林分站改名为吉林车站。吉林分站旧址在今天的吉林站以北约0.5公里处，原吉林车辆段附近，现已全无踪影。<ref>百年吉林火车站的人文记忆.王力.江城日报.2012年11月28日</ref>1930年8月，吉海鐵路由[[黃旗屯站|吉林总站]]延修至吉林分站。 

1931年11月1日，满洲国财政部长[[熙洽|熙洽]]與[[滿鐵|滿鐵]]總裁[[內田康哉|內田康哉]]簽訂契約，出售吉林省全部鐵路业權。滿鐵迅將[[長圖鐵路|吉長]]、吉海兩路在吉林站接軌（另有资料称1931年10月12日吉海路与吉长路在吉林站接轨），后吉林总站改名为黄旗屯站，吉林车站被取消，吉海路的列车改为直接进吉林站。<ref>百年吉林火车站的人文记忆.王力.江城日报.2012年11月28日</ref>

1941年，因站房规模不够，又在站房的西北面新建了一个“票房子”当做候车室。

1980年，拆除使用了50多年的老站房，但新站房并未马上建成，而是建起临时候车室，临时候车室仅由一排平房构成，这个简陋的临时候车室使用了12年之久。

1992年，新站房建起，这个站房设计非常有特色，站房建筑为"JL"两个字母组合，远看又像船，契合了吉林“船厂”的旧称。站房分两层，2层设两个大候车室，1层也设计有2个候车室，但一直未开放，而是作为超市使用。此时的吉林站拥有1个侧式站台和2个岛式站台，并在当时的站台东侧拥有货场。

===2009年改造===
2009年9月1日，为配合长吉城际铁路工程建设，吉林站封站改造并拆除于1992年建成的站房，原吉林站客车被分流到[[吉林西站|吉林西站]]、[[江北站|江北站]]两个临时客站。<ref>{{cite web|url=http://jcrbszb.chinajilin.com.cn/html/2009-09/01/content_1862470.htm|title=江城日报：吉林站改扩建后到底啥样|accessdate=2009-09-01|deadurl=yes|archiveurl=https://web.archive.org/web/20131202223835/http://jcrbszb.chinajilin.com.cn/html/2009-09/01/content_1862470.htm|archivedate=2013-12-02}}</ref>

2011年1月11日，新站房主体建筑投入使用，长吉城际铁路开始运营，开行北京、沈阳、哈尔滨、长春等方向30对动车组列车，先期开通西站房和6-9号站台（后期为10-13号站台）使用。与此同时，东站房、高架候车大厅、未开通的站台都在施工中。

2011年8月28日，整个站房全部投入使用，吉林市江北站和西站分流业务停止。<ref>[http://www.jlsina.com/news/jilinshi/2011-8-23/32667.shtml 2011年8月28日起 吉林市江北站和西站分流业务停止] {{webarchive|url=https://web.archive.org/web/20141205004922/http://www.jlsina.com/news/jilinshi/2011-8-23/32667.shtml |date=2014年12月5日 }}</ref>

2012年12月1日，[[哈大客运专线|哈大客运专线]]开通运营，长吉城际铁路经长西联络线、长哈联络线与其接轨。吉林站增开7对大连方向动车组列车，同时调整北京、沈阳、哈尔滨、长春等方向动车组列车开行方案。

2014年6月16日，开通持二代居民身份证直接检票进出站功能。在中国铁路客户服务中心网站使用二代居民身份证购买G、D、C字头列车车票，乘车站和下车站均具备二代居民身份证检票条件时，未换取纸质车票（学生票、残疾军人票除外）的旅客可凭购票时所使用的二代居民身份证原件，直接通过车站自动检票机（闸机）办理进、出站检票手续。

2014年9月4日起，中铁银通卡在长吉城际铁路上开始发行。<ref>{{cite web |url=http://news.gmw.cn/newspaper/2014-09/04/content_100317286.htm |title=光明网：今起中铁银通卡可在长吉城际列车使用|accessdate=2014-09-04}}</ref>9月10日，吉林站开通中铁银通卡直接检票进站功能。<ref>{{Cite web|url=http://news.163.com/14/0910/06/A5OTI5JU00014AEE.html|title=今起乘长吉高铁可刷中铁银通卡_网易新闻|accessdate=2018-10-13|last=网易|work=news.163.com}}</ref>旅客可使用中铁银通卡电子现金功能直接进出站。中铁银通卡仅限乘坐长吉城际本线动车组列车（D50XX次），02车型1-4排为预留席位，刷卡进站时闸机屏幕将显示旅客席位号。

2014年12月18日，长吉城际列车对中铁银通卡旅客实行取号乘车制。旅客进站前需在取号机打印席位号后，凭席位号凭条、中铁银通卡方可进站乘车。<ref>{{Cite web|url=http://jl.people.com.cn/n/2014/1218/c351793-23267346.html|title=长吉城际列车对中铁银通卡旅客实行取号乘车制--吉林频道--人民网|accessdate=2018-10-13|last=L_103863|work=jl.people.com.cn}}</ref>

2015年9月20日，[[长珲城际铁路|长珲城际铁路]]正式开通运营。

== 车站结构 ==
=== 站房结构 ===
[[File:THE_WEST_WAITING_HALL_OF_JILIN_RAILWAY_STATION.jpg|thumb]]
[[File:吉林站候车室_-_panoramio.jpg|thumb]]
目前，吉林站拥有东、西两座站房、高架候车室、无站台柱雨棚及高站台、旅客地道及东西广场地下交通枢纽（属市政管辖），旅客区域分为地面站台层、高架层候车大厅及服务区域、地下到达层和市政交通枢纽区域。吉林站铁路管辖区总建筑面积229110平方米，站房的总建筑面积为53821平方米。<ref>{{cite web|url=http://www.jcwbs.com/a/news/20090828/1092.html|title=江城晚报：新吉林站建筑方案确定|accessdate=2011-08-24}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>高架候车室面积为20125平方米，分为软席候车区、普通候车区和动车候车区，可同时容纳近3000人。还设有东一层、西一层两个中型候车区，供两座基本站台使用。可承受6000人的高峰小时聚集人数。新建成的东广场占地面积约为5万平方米，广场周边建有配套的地下停车场、运输服务区、休闲景观区等，东广场及配套区域面积将达到48万平方米。

==使用情况==
[[File:Jilin_Railway_station_platform_with_CRH5_in_background.JPG|thumb]]
[[File:The_platform_3_of_jilin_railway_station201406.JPG|thumb]]
吉林站目前为吉林铁路枢纽内的主要客运站，并不办理货运业务。

吉林站地区是吉林市最大的交通枢纽，大量公交车由吉林站东、西广场公交场站始发。西广场地下设有吉林市公路客运站，中、短途公路班线在此始发，为公路运输枢纽之一。

本站线路有[[长图铁路|长图铁路]]、[[沈吉铁路|沈吉铁路]]、[[吉舒铁路|吉舒铁路]]、[[长珲城际铁路|长珲城际铁路]]，并可通过联络线与[[哈大客运专线|哈大客运专线]]连接。动车组列车可通达北京、北京南、天津西、沈阳、大连、哈尔滨西、长春、长春西、龙嘉等站，普通列车可通达宁波、上海、杭州东、天津、北京、青岛北、图们等站。截止2018年4月14日，吉林站日均办理客运业务的旅客列车有154列，其中动车组列车130列，普速列车24列。<ref> {{cite web |url=https://kyfw.12306.cn/otn/czxx/init|title=铁路客户服务中心 车站车次查询 |accessdate=2018-04-13}}</ref>

==站前广场==
[[File:Jilin_Station_Tower.png|thumb]]去思塔，1930年代末开发站前商业区时拆除<ref>吉林旧影 第66页 吉林人民出版社</ref>]]
吉林站前广场始建于1912年，当时主要是为停放人力车、马车及运送货物的车辆，广场西部建铁路公园。

1945年为纪念苏联红军烈士，在正对东候车室的广场中心建一座苏军烈士纪念塔。吉林站前苏军烈士纪念塔修建于1945年10月，塔高8米多，塔顶是一颗红五星。这座纪念塔一度成为吉林站前广场的标志性建筑物。<ref>百年吉林火车站的人文记忆.王力.江城日报.2012年11月28日</ref>

1985年首次对广场进行改造，东西方向修筑一条4317米长的柏油路。

1960年为修建无轨电车迥转线将公园分作两部分，1961年又将广场拓宽。

1968年将拆除铁路公园并炸毁苏军烈士塔，改建成群众集会广场，在广场西侧合建主席台。

1982年将会场拆除恢复站前公园。原主席台改为烤鸭店，后改为园中园旅行社，在原苏军烈士纪念处修建一座圆形喷水池，内置铁制芙蓉花。

1987年建设大世界地下商场（地下人防工事）时将站前公园及广场圆形喷水池全部拆除。

2009年吉林站封站改造，作为新吉林站工程的一部分，在新吉林站东站房前建立广场，是为吉林站东广场。

2011年吉林站新站房落成，但此时西广场（原站前广场）尚未改造，仅开通地面出站口，东广场也未施工完毕。

2012年，吉林站东进出站口（地面）开通，但广场尚未完工，乘客需经由临时便道出入。

2013年9月15日，吉林站东广场主体工程竣工，东进出站口（地面）开通，但东交通枢纽出站口（地下）暂未开通，其交通枢纽功能也在逐渐完善。吉林站东广场及吉林站东出站口（地下）预计在2013年10月正式投入使用。届时，吉林站西广场将封闭重建，吉林站西进站口和西出站口也将全部封闭。封闭的具体时间另行公告。<ref>[http://enews.xwh.cn/shtml/xwhb/20130926/content_159.shtml吉林站西广场10月进行整体封闭改造]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2014年7月1日，吉林站东广场地下1号、2号社会车辆停车场及地下出租车站投入使用，吉林站西广场开始封闭改造。<ref>{{cite web |url=http://www.shjnet.cn/xw/jcjsb/201407/t20140731_310856.html |title=松花江网：吉林站施工期间出站旅客需走东出口|accessdate=2014-09-14}}</ref>
