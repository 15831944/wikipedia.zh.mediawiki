'''吉阿吉阿語'''，或稱'''西布頓語'''屬於[[南島語系|南島語系]]，主要使用於靠近於[[印尼|印尼]][[苏拉威西岛|苏拉威西岛]]東南海岸旁之[[布顿岛|布顿岛]]南端[[巴務巴務市|巴務巴務市]]周邊。至2009年大約有六万人仍在使用吉阿吉阿語<ref>{{cite news|url=http://www.ethnologue.com/show_language.asp?code=cia|title=Ethnologue report for language code: cia|publisher=[[民族語|民族語]]|author=2005[[SIL國際|SIL國際]]|accessdate=2009-08-07|date=2009-07-01|language=en}}</ref>。

== 使用 ==
說吉阿吉阿語的人、大都是[[穆斯林|穆斯林]]，也同時使用[[窩里沃語|窩里沃語]]、[[印尼语|印尼语]]。窩里沃語是一種近似吉阿吉阿語的語言。

吉阿吉阿語一度使用一種稱為「{{link-en|Pegon alphabet|Pegon alphabet|袞度爾}}」（Gundhul）的文字。這種袞度爾源於以[[阿拉伯字母|阿拉伯字母]]為基礎的[[爪夷文|爪夷文]]，但袞度爾另發展有5個附加子音字母、不過沒有使用元音符號。從2008年開始，當地與[[韓國|韓國]][[訓民正音學會|訓民正音學會]]採用[[諺文|諺文]]（韓字）字母作為其現代化的文字<ref name=nocut 另外，為了準確表記吉阿吉阿語，在其正字法中特別恢復使用一個現代[[朝鮮語|韓語]]棄用的舊字母「{{lang|ko|[[ㅸ|ㅸ]]}}」以標示[[濁雙唇擦音|濁雙唇擦音]]{{IPA|[β]}}(v)<ref>{{cite news|title = 사라질 소수민족 언어 ‘한글’로 새 생명 | trans_title = 韓字為少數民族語言帶來新生命 |url = http://news.khan.co.kr/kh_news/khan_art_view.html?artid=200908061804195&code=940401 | publisher = [[京鄉新聞|경향신문]] ([[京鄉新聞|京鄉新聞]]) |date = 2009-08-06 |accessdate = 2009-08-08|language=ko}}</ref>。

以下為一個用[[諺文|諺文]]來拼寫吉阿吉阿語的例子<ref name=nocut>{{cite news|url=http://www.cbs.co.kr/nocut/show.asp?idx=1224657 |title=印尼 소수민족, '한글' 공식 문자로 채택| trans_title=印尼少數民族採用韓字作為正式文字 |language=ko| accessdate=2009-08-10| publisher=[[基督教放送|基督教放送]]：[[Nocut_News|Nocut News]]}}</ref>：

:아디 세링 빨리 노논또 뗄레{{古韓語|ᄫᅵ}}시. 아마노 노뽀옴바에 이아 나누몬또 뗄레{{古韓語|ᄫᅵ}}시 꼴리에 노몰렝오.
:''adi sering pali nononto televisi. amano nopo'ombae ia nanumonto televisi kolie nomoleo.''

從以上例子可以看到：當地人把「[[電視|電視]]」按當地語言的發音拼寫成「뗄레{{古韓語|ᄫᅵ}}시」；相反在[[韓語|韓語]]裡，「電視」的寫法卻是按其[[英語|英語]]發音的變化拼成：「[[电视|텔레비전]]」。
