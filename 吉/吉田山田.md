'''吉田山田'''（{{Jpn|j=吉田山田|kn=よしだやまだ}}），[[日本|日本]]的二人音乐组合，2005年结成。为[[Pony_Canyon|Pony Canyon]]旗下艺人。

==成员==
*吉田结威（{{Jpn|j=吉田 結威|kn=よしだ ゆい|rm=Yoshida Yui}}，{{Birth date and age|1983|9|9}}），出生于[[兵库县|兵库县]][[神户市|神户市]]。
*山田义孝（{{Jpn|j=山田 義孝|kn=やまだ よしたか|rm=Yamada Yoshitaka}}，{{Birth date and age|1983|12|13}}），出生于[[东京都|东京都]]。

==经历==
1999年，吉田与山田入读[[东京都立明正高等学校|东京都立明正高等学校]]。在学中山田对吉田的吉他弹唱产生了兴趣，有了一起做音乐的想法，但直到毕业，并未施行。毕业后吉田进入[[和光大学|和光大学]]学习心理学，山田则成为了一个龙套演员。一年后，山田联络吉田提议开始音乐活动，此时吉田在学业上遇到了瓶颈，接受了山田的提议。刚开始时期二人对于如何作曲都不了解，山田提议去纽约修行，这次修行也加深了二人的关系。

2005年回国后，开始正式音乐活动。最早以本名为组合名，粉丝则称呼为“吉田山田”。2006年开始在事务所相关人士的帮助下强化学习，并开始在全国范围内参与演出。到2009年10月发售第一张单曲《{{lang|ja|ガムシャランナー}}》并在波丽佳音正式主流出道<ref name="w">{{cite web |last1=Inc |first1=Natasha |title=吉田山田、新曲「ツボミ」特設サイトで相性チェック |url=https://natalie.mu/music/news/27241 |website=音楽ナタリー |accessdate=2019-02-27 |language=ja}}</ref>。

2010年10月发售首张专辑《{{lang|ja|と}}》，并进行了全国巡回演唱会。同时开始了ZIP-FM的常规广播节目。

2013年发售第8张单曲《{{lang|ja|魔法のような}}》获得Oricon周榜第14位，这也是吉田山田至今获得的最好成绩。同年二人参加[[大阪马拉松|大阪马拉松]]赛都完成了全程。

2013年12月发售第9张单曲《{{lang|ja|日々}}》，该曲是为[[NHK|NHK]]“大家的歌”节目所写，并在2013年12月至2014年1月间播放，此后4度重播<ref>{{cite web |last1=Inc |first1=Natasha |title=吉田山田「日々」、みんなのうたで4度目の再放送 |url=https://natalie.mu/music/news/136446 |website=音楽ナタリー |accessdate=2019-02-27 |language=ja}}</ref>。

2014年1月发售第3张专辑《吉田山田》，被选为第56届[[日本唱片大奖|日本唱片大奖]]优秀专辑奖<ref>{{cite web |last1=Inc |first1=Natasha |title=三代目 J Soul Brothers初のレコ大受賞＆最優秀新人賞は西内まりや |url=https://natalie.mu/music/news/135191 |website=音楽ナタリー |accessdate=2019-02-27 |language=ja}}</ref>。

2015年发售首个演唱会DVD作品<ref>{{cite web |last1=Inc |first1=Natasha |title=吉田山田、初の渋谷公会堂ライブをDVD化 |url=https://natalie.mu/music/news/151818 |website=音楽ナタリー |accessdate=2019-02-27 |language=ja}}</ref>。

2016年发售第4张、出道第7年的专辑《47》<ref>{{cite web |last1=Inc |first1=Natasha |title=吉田山田、新アルバムのタイトルは「47【ヨンナナ】」 |url=https://natalie.mu/music/news/173241 |website=音楽ナタリー |accessdate=2019-02-27 |language=ja}}</ref>，并开始日本全国47都道府县巡回<ref>{{cite web |last1=Inc |first1=Natasha |title=吉田山田、来年は47都道府県ツアー「二人また旅2016」 |url=https://natalie.mu/music/news/169940 |website=音楽ナタリー |language=ja}}</ref>。

2019年发售第13张单曲《{{lang|ja|桜咲け}}》成为电视动画《[[火之丸相扑|火之丸相扑]]》的结束曲<ref>{{cite web |last1=Inc |first1=Natasha |title=吉田山田「火ノ丸相撲」ED曲のMV公開（動画あり） |url=https://natalie.mu/music/news/320595 |website=音楽ナタリー |accessdate=2019-02-27 |language=ja}}</ref>。并预定再次进行47都道府县巡回<ref>{{cite web |last1=Inc |first1=Natasha |title=吉田山田、2度目の47都道府県ツアー開催決定「二人またまた旅2019」 |url=https://natalie.mu/music/news/308421 |website=音楽ナタリー |accessdate=2019-02-27 |language=ja}}</ref>。
