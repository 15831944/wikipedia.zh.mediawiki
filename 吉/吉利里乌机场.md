'''吉利里乌机场'''（[[印尼语|印尼语]]：Bandar Udara Tjilik Riwut，{{airport codes|PKY|WAGG}}）原名'''帕纳隆机场'''（Panarung），是[[印度尼西亚|印度尼西亚]][[中加里曼丹省|中加里曼丹省]][[帕朗卡拉亚|帕朗卡拉亚]]的一座机场。机场名称来自[[印度尼西亚民族英雄|印度尼西亚民族英雄]]、中加里曼丹第二任省长{{le|吉利里乌|Tjilik Riwut}}<ref>{{cite web |url=http://www.nila-riwut.com/id/tjilik-riwut/tjilik-riwut-my-father |title=Tjilik Riwut, My Father |last= |first= |date=2011-02-14 |website=nila-riwut |publisher= |access-date=2017-11-10 |quote=}}</ref>。2017年被列入印尼国家机场战略计划重点发展的8个机场之中<ref>{{cite web |url=http://www.tribunnews.com/bisnis/2017/09/07/kemenhub-kebut-penyelesaian-8-bandara-program-strategi-nasional |title=Kemenhub Kebut Penyelesaian 8 Bandara Program Strategi Nasional |author=Apfia Tioconny Billy |date=2017-09-07 |website=www.tribunnews.com |publisher=TRIBUNNEWS |access-date=2017-11-10 |quote=}}</ref><ref>{{cite web |url=https://finance.detik.com/berita-ekonomi-bisnis/3632365/jokowi-punya-7-proyek-strategis-bandara-begini-progresnya |title=Jokowi Punya 7 Proyek Strategis Bandara, Begini Progresnya |author=Muhammad Idris |date=2017-09-07 |website=finance.detik.com |publisher=detikcom |access-date=2017-11-10 |quote=}}</ref>。

==设施==

2014年10月10日启用了新的航站楼，旅客容量有所增加。航站楼总面积15,000平方米，共有两层，每天可服务1,000名旅客。航站楼内有几家商店和餐馆。目前机场正在计划修建一个新的航站楼。<ref>{{cite web |url=https://finance.detik.com/foto/3586068/perubahan-wajah-bandara-tjilik-riwut-yang-jadi-prioritas-jokowi/ |title=Perubahan Wajah Bandara Tjilik Riwut yang Jadi Prioritas Jokowi |author=Eduardo Simorangkir |date=2017-08-04 |website=finance.detik.com |publisher=detikcom |access-date=2017-11-10 |quote=}}</ref>
