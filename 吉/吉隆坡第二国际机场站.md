'''吉隆坡第二国际机场站'''（{{lang-ms|Stesen ERL klia2}}）位于[[马来西亚|马来西亚]][[雪兰莪州|雪兰莪州]][[雪邦市|雪邦市]][[吉隆坡第二国际机场|吉隆坡第二国际机场]]，于2014年5月2日和[[吉隆坡第二国际机场|吉隆坡第二国际机场]]（klia2）同时开放运营<ref name="sinchew.com.my">{{cite news|url=http://www.sinchew.com.my/node/1383539|title=【更新】502同日启用．机场快铁延伸至KLIA2|date=2014-04-22|publisher=[[星洲网|星洲网]]}}</ref>。

== 车站概要 ==
本站设在吉隆坡第二国际机场的综合大厦的二楼（Gateway@KLIA），其中设有230尺长的岛式站台、升降机、验票闸门及售票及办公范围。<ref>{{cite news|url=http://www.sinchew.com.my/node/1324438|title=机场快铁延长路线试跑|date=2013-05-30|publisher=[[星洲网|星洲网]]}}</ref>车站形式为岛式站台，站台A往机场快铁，而站台B则往机场支线。

机场快铁延伸线于2011年7月15日开始建造，并预期在2012年10月31日完工。然而该线路迟至2013年9月28日才完工，并在2014年4月14日获得[[陆路公共交通委员会|陆路公共交通委员会]]发出同意延伸快铁的执照。最终本站与吉隆坡第二国际机场同时于2014年5月2日开放，成为两条机场铁路的新南端终点站。<ref>{{cite web|url=http://www.kleptocrazy.my/the-erl-extension-project-from-klia-to-klia2/|title=The ERL Extension Project From KLIA to KLIA2 - Kleptocrazy Malaysia|website=www.kleptocrazy.my}}</ref>
