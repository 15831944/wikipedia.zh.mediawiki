'''小詹姆斯·亨利·「吉姆」·韋布'''（{{lang-en|'''James Henry "Jim" Webb, Jr.'''}}；{{bd|1946年|2月9日||}}），是一位美國[[政治人物|政治人物]]與作家<ref>{{cite web|url=http://www.presidency.ucsb.edu/ws/index.php?pid=33823&st=&st1=|title=Ronald Reagan: Nomination of James H. Webb, Jr., To Be Secretary of the Navy, February 18, 1987|work=The American Presidency Project |first1 = Gerhard |last1 = Peters |first2 = John T. |last2 = Woolley}}</ref><ref>{{cite web |url=http://articles.latimes.com/1987-02-18/news/mn-2692_1_retirement-plans|title=The Nation|work=Los Angeles Times |date = February 18, 1987}}</ref>，曾任[[維吉尼亞州|維吉尼亞州]][[美國參議院|參議院議員]]及[[雷根|雷根]]政府的[[美國海軍部長|美國海軍部長]]。

== 參議院 ==
[[2006年美國中期選舉|2006年美國中期選舉]]，韋布當選參議員。2012年，他不尋求連任參議員。

== 2016年美國總統選舉 ==
2014年11月19日，吉姆公佈會成立一個討論委員會，以評估[[2016年美國總統選舉|在2016年參選美國總統]]的可能性<ref>{{cite web |url=http://webb2016.com |title=Dear Friends |work=webb2016.com |publisher=Webb 2016 Exploratory Committee |date=November 19, 2014 |deadurl=yes |archiveurl=https://web.archive.org/web/20141120130902/http://webb2016.com/ |archivedate=2014年11月20日 }}</ref><ref>{{cite web |url=http://www.politico.com/story/2014/05/jim-webb-president-race-2016-no-vice-president-virginia-106838.html|title=Jim Webb: No to VP, door open for 2016|first=Jonathan |last = Topaz|date =  May 19, 2014 |work=[[Politico|Politico]]}}</ref>。

他最終在2015年7月2日宣佈以民主黨籍參選2016年美國總統<ref name="nyti_JimW">{{Cite web |title = Jim Webb Announces Democratic Bid for Presidency | last = Haberman |first = Maggie |work = The New York Times |date = 2 July 2015 |accessdate = 2015-07-02 |url = http://www.nytimes.com/2015/07/03/us/politics/jim-webb-presidential-campaign.html?_r=0 |quote =  }}</ref>；但後來因黨內支持率一直在包尾（不超過1%），在2015年10月20日第一場民主黨辯論舉行後，宣布退選<ref>[https://www.yahoo.com/politics/jim-webb-plans-to-drop-out-of-democratic-primary-153500314.html Jim Webb drops out of Democratic primary race]</ref>，並拒絕支持[[希拉蕊·克林頓|希拉蕊·克林頓]]，曾考慮<ref>[https://www.washingtonpost.com/opinions/thinking-outside-the-two-party-box/2015/10/30/a8827dda-7dae-11e5-beba-927fd8634498_story.html Jim Webb "America needs an independent presidential candidate" Washington Post, November 1, 2015]</ref>改為以獨立候選人參選<ref>{{Cite web|title = FEC Form 2, Statement of Candidacy|url = http://docquery.fec.gov/pdf/994/201510309003259994/201510309003259994.pdf|website = FEC.gov|accessdate = 2015-10-30}}</ref>。
