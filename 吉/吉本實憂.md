'''吉本實憂'''（{{Lang-ja|吉本 実憂}}，{{bd|1996年|12月28日|catIdx=Yoshimoto Miyu}}），[[日本|日本]]女[[藝人|藝人]]、[[模特兒|時裝模特兒]]，偶像團體「[[X21|X21]]」隊長，出身於[[福岡縣|福岡縣]][[北九州市|北九州市]]，隸屬[[奧斯卡傳播|奧斯卡傳播]]經紀公司。身高160公分、血型AB型。

== 經歷 ==
在2012年夏天，從102,564人的海選中脫穎而出，獲得第13回[[全日本國民美少女比賽|全日本國民美少女比賽]]的大獎，與[[小澤奈奈花|小澤奈奈花]]共同得到此一殊榮<ref>{{Cite web |url=http://www.oscarpro.co.jp/news/2012/08/13.html |title=『第13回全日本国民的美少女コンテスト』グランプリ決定！ |date=2012-08-21 |accessdate=2012-08-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20120913114558/http://www.oscarpro.co.jp/news/2012/08/13.html |archivedate=2012-09-13 }} - Oscar Promotion</ref>，並進入由該屆入選成員所組成的團體「[[X21|X21]]」，在隔年1月28日出道時作為其中的Leader。在正式大會審查時，選擇演唱了[[倖田來未|倖田來未]]的歌曲「[[WIND_(倖田來未單曲)|WIND]]」<ref>{{Cite web |url=http://www.sanspo.com/geino/news/20120822/oth12082205060020-n2.html |title=１０年ぶり！国民的美少女、２人が涙の頂点 |date=2012-08-22 |accessdate=2012-08-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20120824234217/http://www.sanspo.com/geino/news/20120822/oth12082205060020-n2.html |archivedate=2012-08-24 }} - SANSPO.COM</ref>。

在東京的第一個工作是在[[拉爾夫·洛朗|拉爾夫·洛朗]]的直營旗艦店所舉辦的開幕宴會，負責來賓接待的服務<ref>{{Cite web |url=http://www.oricon.co.jp/news/entertainment/2017170/full/ |title=“国民的美少女”吉本＆小澤がイベント初登場「緊張してる」 |date=2012-09-27 |accessdate=2012-09-28}} - [[Oricon|ORICON STYLE]]</ref><ref>{{Cite web |url=http://www.nikkansports.com/entertainment/news/f-et-tp0-20120927-1023972.html |title=美少女ＧＰ吉本実憂＆小沢奈々花が初仕事 |date=2012-09-27 |accessdate=2012-09-28}} - [[日刊體育|nikkansports.com]]</ref>。

2013年，擔任[[第85回紀念選拔高等學校野球大會|第85回紀念選拔高等學校野球大會]]史上首位「選拔應援海報人物」<ref>{{Cite web |url=http://mainichi.jp/sports/news/20130228k0000e050195000c.html |title=センバツ：吉本実憂さんが球児にエール　ポスター発表会|date=2013-02-28 |accessdate=2013-02-28}} - 每日jp</ref>。

2015年，在連續劇《[[表參道高校合唱部！|表參道高校合唱部！]]》與電影《{{link-ja|罪的留白|罪の余白#映画}}》中的反派色彩演繹獲得好評與注目<ref>[http://www.iza.ne.jp/kiji/entertainments/news/151004/ent15100417500019-n1.html 〈期待の若手女優（９）〉「イラッとさせる」ヒール女子校生を『罪の余白』で演じた吉本実憂]</ref>。
