'''吉尔罗伊'''（{{lang|en|'''Gilroy'''}}），是[[美国|美国]][[加利福尼亚州|加利福尼亚州]][[圣克拉拉县|圣克拉拉县]]下属的一个市。设市于1906年11月10日。是为[[硅谷|硅谷]]的一个[[卫星城|卫星城]]。根据[[美国2010年人口普查|美国2010年人口普查]]，该市有人口48,821人。该市以[[大蒜|大蒜]]种植与生产而闻名。每年会举行“吉尔罗伊大蒜节”，展售各种大蒜食品，甚至是大蒜[[冰淇淋|冰淇淋]]。<ref>[http://www.gilroygarlicfestival.com/ Gilroy Garlic Festival]</ref>吉尔罗伊市的昵称是“'''世界大蒜之都'''”，这个昵称的起源是该市不仅生产了大量的大蒜以及相关产品，该市的饮食文化也极具大蒜特色。<ref>[http://www.garlicworld.com/ garlicworld]</ref>
== 地势 ==
吉尔罗伊位于[[旧金山湾区|旧金山湾区]]东南部，[[圣荷西_(加利福尼亚州)|圣荷西]]市中心东南方大约55公里处，[[摩根山丘_(加利福尼亚州)|摩根山丘]]市东南方大约16公里处。往西距离[[太平洋|太平洋]]海岸大约31公里，西南部是[[圣塔克鲁兹_(加利福尼亚州)|圣塔克鲁兹]]市。市区坐落在[[圣克拉拉山谷|圣克拉拉山谷]]南部，[[海拔|海拔]]大约61米。市区面积42平方千米，水域面积极小（0.06%）。该市位于[[地震带|地震带]]附近，是地震多发区。

== 气候 ==
该市气候受到太平洋的影响，属于[[地中海气候|地中海气候]]。全年气温区间在0.9 °C (33.6 °F)至32.3 °C (90.2 °F)之间。全年[[降水量|降水量]]约为480毫米。夏季气候极为干燥，冬天雨量较多，但极少有降雪。在夏季，[[雾|雾]]较为常见。晚上10时左右会有来自海湾的雾飘至该市上空，第二天上午10时左右逐渐散去。在冬天，阵雨发生的几率较大。该地区的地形不适合[[龙卷风|龙卷风]]产生，也较少出现雷暴或风暴天气。这里的地中海气候使得一些热带植物，如[[棕榈树|棕榈树]]，适合在这里生长。总体上，由于距离相近，该地与北边的[[摩根山丘_(加利福尼亚州)|摩根山丘]]气候几乎完全相同。一月份的平均气温在40-60[[华氏|华氏]]度（4-15[[摄氏|摄氏]]度），七月平均在60-90华氏度（15-32摄氏度）。平均每年有33天日最高气温会超过100 °F (37.7 °C)，以及平均18天日最低气温低于32 °F (0 °C)。该地历史最高温纪录是1972年7月15日，温度为115 °F (46 °C)，最低温为1990年12月22日，17 °F (-8 °C)。平均每年有58天降雨。<ref>http://www.wrcc.dri.edu/cgi-bin/cliMAIN.pl?ca3417</ref>

== 人口 ==
[[File:Gilroy.JPG|thumb]]
=== 2000年 ===
根据[[2000年美国人口普查|2000年美国人口普查]]，2000年时，该市有11,869户，9,590个家庭，平均每户3.46人，每个家庭3.74人。12,152个住房单位。种族构成为：58.9%的[[白人|白人]]，4.4%[[亚裔|亚裔]]，1.8%[[黑人|非洲裔]]，1.6%[[原住民|原住民]]，0.3%太平洋岛屿居民，27.7%属于其它种族。5.4%同时来自两个或更多的种族。

2000年该市有47.7%的家庭拥有18岁以下的[[未成年人|未成年人]]。60.8%为有婚姻的家庭。

=== 2010年 ===
根据[[2010年美国人口普查|2010年美国人口普查]]，2010年时，该市有14,175户，11,336个家庭，14,854个住房单位。平均每个家庭有3.39人。种族构成为：58.7%的[[白人|白人]]，7.1%[[亚裔|亚裔]]，1.9%[[黑人|非洲裔]]，1.7%[[原住民|原住民]]，0.2%太平洋岛屿居民，25.2%属于其它种族。5.1%同时来自两个或更多的种族。

2010年该市有50.2%的家庭拥有18岁以下的[[未成年人|未成年人]]。57.5%为有正常婚姻的家庭，0.7%为[[同性恋|同性恋]]同居。该市居民男女比例为97.1:100。

=== 收入 ===
根据2000年的估计，该市每户平均年收入为$66,401。家庭平均年收入为$80,371。男性人均年收入为$45,759，女性为$34,710。有7.3%的家庭和10.4%的人口收入位于[[贫困线|贫困线]]以下，包括12.8%的未成年人和6.5%的65岁以上老年人。

== 设施 ==
[[太平洋瓦电公司|太平洋瓦电公司]]为该市提供燃气和电力。

该市有四所公立高中和一所大学、三所初中、八所小学。

[[圣克拉拉县图书馆|圣克拉拉县图书馆]]管理该市的图书馆 (Gilroy Library)业务。<ref>"[http://www.santaclaracountylib.org/gilroy/ Welcome to the Gilroy Library] {{webarchive|url=https://web.archive.org/web/20100407073247/http://www.santaclaracountylib.org/gilroy/ |date=2010-04-07 }}." Santa Clara County Library. Retrieved on March 27, 2010.</ref>

== 交通 ==
南北向的[[101号美国国道加利福尼亚州段|101号美国国道]]（US-101）高速公路经过该市，為居民通勤至北聖塔克拉拉郡的要道。途經吉爾羅伊的東西向[[152號加州州道|152號加州州道]]（CA-152）是南舊金山灣區往中谷[[5號州際公路|5號州際公路]]（I-5）的要道。

该市北部有一处小型的机场供大眾小型飛行器使用。而民用航班需要到55公里外的[[圣荷西国际机场|圣荷西国际机场]]，该市以南的[[萨利纳斯_(加利福尼亚州)|萨利纳斯]]市也有机场。

[[加州火车|加州火车]]设有[[吉尔罗伊站|吉尔罗伊站]]，服务往返硅谷的通勤者。

[[加利福尼亞高速鐵路|加利福尼亞高速鐵路]]預計於吉爾羅伊市中心設站。
