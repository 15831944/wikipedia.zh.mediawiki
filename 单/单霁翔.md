'''单霁翔'''（{{bd|1954年|7月|catIdx=S}}）[[汉族|汉族]]，[[江苏省|江苏省]][[江宁县|江宁县]]（今[[南京市|南京市]][[江宁区|江宁区]]）人。[[中华人民共和国|中华人民共和国]]官员。曾任[[国家文物局|国家文物局]]局长，2012年1月至2019年4月任北京[[故宫博物院|故宫博物院]]院长。[[中共中央党校|中共中央党校]]研究生学历。高级[[工程师|工程师]]、[[注册规划师|注册规划师]]。

== 生平 ==
1971年1月，单霁翔参加工作。1985年6月，单霁翔加入[[中国共产党|中国共产党]]。<ref name=xinhua>[http://news.xinhuanet.com/ziliao/2002-03/05/content_300534.htm 单霁翔简历，新华网，于2012-2-14查阅]</ref>

1992年1月到1994年5月，单霁翔任[[北京市城市规划管理局|北京市城市规划管理局]]副局长。1994年5月至1997年8月，单霁翔任[[北京市文物局|北京市文物局]]党组书记、局长。1997年8月至2000年1月，单霁翔任[[中共北京市房山区区委|中共北京市房山区区委]]书记。2001年1月至2002年8月，单霁翔任[[北京市规划委员会|北京市规划委员会]]（[[首都规划建设委员会办公室|首都规划建设委员会办公室]]）党组书记、主任。2002年8月，单霁翔任[[中华人民共和国文化部|中华人民共和国文化部]]党组成员，[[国家文物局|国家文物局]]党组书记、局长。2012年2月，他被免去[[国家文物局|国家文物局]]局长的职务。 <ref name=xinhua/>2012年1月，单霁翔任[[中华人民共和国文化部|中华人民共和国文化部]]党组成员、[[故宫博物院|故宫博物院]]院长，兼任[[故宫博物院|故宫博物院]]党委书记。<ref>[http://www.dpm.org.cn/shtml/115/@/9036.html#33 单霁翔，故宫博物院，于2012-2-14查阅] {{webarchive|url=https://web.archive.org/web/20161120112022/http://www.dpm.org.cn/shtml/115/%40/9036.html |date=2016-11-20 }}</ref><ref>{{Cite web |url = http://news.ifeng.com/a/20171109/53119045_0.shtml |title =今天故宫馆长火了，他原来一直是隐藏段子手  |author =  |date = 2017-11-09 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>2019年4月8日，单霁翔退休，故宫博物院院长由原[[敦煌研究院|敦煌研究院]]院长王旭东继任。<ref>{{cite news |url=https://finance.sina.com.cn/chanjing/gsnews/2019-04-08/doc-ihvhiewr4078572.shtml |title=故宫院长单霁翔今日退休 敦煌研究院院长王旭东接任 |website=新浪财经 |date=2019-04-08 }}</ref>

他是[[中国人民政治协商会议|中国人民政治协商会议]]第十届、第十一届全国委员会委员。他还是[[首都规划建设委员会|首都规划建设委员会]]委员。<ref>[http://politics.people.com.cn/GB/41223/16851976.html 郑欣淼卸任故宫博物院院长 单霁翔接任，人民网，2012年01月11日]</ref>
