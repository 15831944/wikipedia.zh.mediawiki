'''单斑裸颊鲷'''（[[学名|学名]]：{{lang|la|''Lethrinus harak''}}），又稱'''單斑龍占'''，为[[裸颊鲷科|裸颊鲷科]][[裸颊鲷属|裸颊鲷属]]的[[鱼类|鱼类]]，俗名单斑龙占。分布于台湾东港等。该物种的模式产地在红海。<ref name='id7473'/>

== 特徵 ==
單斑龍占魚體長而呈橢圓形。[[吻|吻]]中短而略鈍，吻上緣與上[[頜|頜]]間的角度為60°-70°。眼間隔凸起或幾平坦。眼睛大，位於頭背側。兩頜具[[犬齒|犬齒]]及絨毛狀齒，後方側齒呈圓形或[[臼齒|臼齒]]狀；上頜骨上緣平滑或稍呈鋸齒狀。[[頰|頰]]部無鱗，胸鰭基部內側具鱗，側線鱗數46-47:側線上鱗列數4.5-5.5(大部分為5.5)；側線下鱗列數13-14。背鰭單一，不具深刻。尾鰭分叉，兩葉先端尖型。體背側綠褐色或灰色。

== 生態 ==
單斑龍占主要棲息於沿岸[[珊瑚礁|珊瑚礁]]、[[岩礁|岩礁]]外緣、[[沼澤|沼澤]]、[[紅樹林|紅樹林]]或海藻床的砂泥地。主要分布的深度在5-20公尺。獨居或成小群活動。主要以[[軟體動物|軟體動物]]、甲殼類及小魚為食。

== 分布 ==
單斑龍占分布於[[印度|印度]]到西[[太平洋|太平洋]]，西起[[東非|東非]]、[[紅海|紅海]]，東至[[薩摩亞|薩摩亞]]，北至日本南部，南至澳洲北部。[[台灣|台灣]]分布於南部、西部及[[小琉球|小琉球]]海域。

== 利用 ==
單斑龍占一般以[[延繩釣|延繩釣]]、[[一支釣|一支釣]]或[[底拖網|底拖網]]等方式捕獲，全年都可以捕撈到，尤其夏天特別多。屬於中型食用魚種，小魚可供觀賞。
