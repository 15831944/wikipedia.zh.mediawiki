'''单田芳'''（{{bd|1934年|12月17日|2018年|9月11日|catIdx=Shan单田芳}}），本名'''单传忠'''<ref name="自传">[http://data.book.163.com/book/section/000BCcAB/000BCcAB2.html]{{Dead link|date=2019年4月 |bot=InternetArchiveBot |fix-attempted=yes }}单田芳自传</ref>，生于今[[中国|中国]][[辽宁|辽宁]][[营口市|营口]]，[[籍贯|籍贯]][[河北|河北]][[保定市|保定]][[涞水县|涞水县]]（父系祖籍[[山东|山东]][[德州市|德州]][[临邑县|临邑县]]，祖父入赘）<ref>单田芳自传 言归正传:单田芳说单田芳</ref>，[[中国大陆|中国大陆]]知名[[评书|评书]]表演艺术家、第三批国家[[非物质文化遗产|非物质文化遗产]][[国家非物质文化遗产传承人|传承人]]<ref>[http://www.ccnt.gov.cn/sjzz/fwzwhycs/flfg/201001/W020100113612095898690.doc  第三批国家级非物质文化遗产项目] {{webarchive|url=https://web.archive.org/web/20110623174325/http://ccnt.gov.cn/sjzz/fwzwhycs/flfg/201001/W020100113612095898690.doc |date=2011-06-23 }}doc文件</ref>、[[中国曲艺家协会|中国曲艺家协会]]会员以及[[中国通俗小说研究会|中国通俗小说研究会]]会员。其评书表演在语言上比较质朴，以声绘形绘色绘情上尤为见长；其个人评书[[電視節目|节目]]和相关[[视频|视频]]也自1990年代中后期开始，在[[中国中央电视台|中国中央电视台]]等[[中国內地|中国內地]]各大[[电视频道|电视频道]]上相继播出，其听众多达6亿人<ref name=zgzs>{{Cite web|url=http://finance.chinanews.com/cul/2018/09-12/8624915.shtml|title=单田芳：我曾给《隋唐演义》做过“大手术” |accessdate=2018-09-12|author=|date=2018-09-12|publisher中国之声}}</ref>。2018年9月11日下午3点30分因病在中日友好医院去世。

== 生平 ==
===早年===
50年代，单田芳父亲蒙冤入狱，母亲与父亲离婚。1953年考入[[东北大学_(中国)|东北大学]]，后因身体与家庭原因放弃学业从事评书艺术，师从[[李庆海|李庆海]]。1955年，单田芳进入[[鞍山|鞍山]]曲艺团，得到[[西河大鼓|西河大鼓]]名家[[赵玉峰|赵玉峰]]和评书名家[[杨田荣|杨田荣]]的指点。24岁正式登台。不仅表演传统评书，也有许多新书作品。
===文革十年===
[[文化大革命|文革]]中[[因言获罪|因言获罪]]，遭到迫害，甚至曾被[[造反派|造反派]]踢碎牙齿<ref>[http://book.sina.com.cn/excerpt/sz/rw/2009-04-20/1409254912_2.shtml 单田芳文革时受尽屈辱被人踢碎满口牙]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。身上两处骨折。

===重返舞台===
文革结束后，单田芳得以重新返回舞台说书。从[[鞍山人民广播电台|鞍山人民广播电台]]开始逐渐被全国评书爱好者接受，成为中国大陆首屈一指的大家。据统计，每天最高峰时有1.2亿听众收听单田芳的评书<ref name=zgzs/>。

===退休后===
单田芳继续发展评书艺术，创立北京单田芳艺术传播有限责任公司，任董事长。<ref name="自传"/><ref name="生平1">[http://www.m1905.com/mdb/star/1767/life/ 单田芳生平]</ref>

===逝世===
2018年9月11日下午3点30分，单田芳因病在[[中日友好医院|中日友好医院]]病逝，享壽84岁<ref>{{Cite web|url=http://ent.sina.com.cn/s/m/2018-09-11/doc-ihiixyeu6200625.shtml|title=评书大师单田芳11日下午病逝 享年84岁|accessdate=2018-09-11|author=|date=2018-09-11|publisher=新浪娱乐|archive-url=https://web.archive.org/web/20180911121230/http://ent.sina.com.cn/s/m/2018-09-11/doc-ihiixyeu6200625.shtml|archive-date=2018-09-11|dead-url=no}}</ref>。

== 作品简介 ==
《三國演義》，《西遊記》，《封神演義》，《水滸傳》，《大明英烈》，《民國英烈》，《曾國藩》

《亂世梟雄》，讲述張作霖及張學良父子的一生。

《三侠五义》，讲述北宋仁宗年间以三侠（指北侠欧阳春、南侠展昭、丁氏双侠丁兆兰、丁兆蕙（二人为一侠））；五义（指钻天鼠卢方、彻地鼠韩彰、钻山鼠徐庆，翻江鼠蒋平、锦毛鼠白玉堂这五鼠弟兄）为首的侠义之士协助青天包公平奇冤办要案的故事。

《白眉大侠》，讲述[[北宋|北宋]]初期以徐良为首的正义之士除暴安良、替天行道的故事，故事承接自《三侠五义》与《七杰小五义》。后改编为同名武侠电视剧。

《[[林则徐|林则徐]]》，80集[[视频|视频]]评书，讲述[[清朝|清朝]][[道光|道光]]时期，林则徐以钦差大臣身份在[[中国|中国]][[广东|广东]]禁烟及被革职后的故事。

== 影响 ==
“凡有井水处，皆听单田芳。”艺术生涯作品累计100多部，在全国600多家电台、电视台播出，影响几代中国人<ref name=zgzs/>。由于单田芳知名度颇高且嗓音独特，常有人模仿他的声音<ref>[http://6.cn/watch/5847582.html 杨光模仿单田芳]</ref>。

单田芳在其著作《言归正传:单田芳说单田芳》中涉及了[[长春围困战|长春围困战]]，其他没有提到任何共军对普通老百姓的救援。指出事实上当时老百姓被围在城中，或者被围在国共两军对峙的中间地带，他们唯一的希望是「逃出长春这座魔窟」。没有老百姓希望留在这种朝不保夕的危险地带等着「白食」。而进攻方的共军完全可以让老百姓自行离去，所以中共官方这种所謂「救援」是没有任何逻辑。单田芳又提到他父亲花了「七两黄金外加一顿饭」买通国軍[[连长|连长]]，冒充「起义的国军」才逃出长春，而后又因为害怕而自首，承认自己不是国军。间接证明了老百姓不被共军放行，救援对象仅为国军投诚者的事實<ref>单田芳《言归正传:单田芳说单田芳》，「少年经历大变局」部分）</ref>。

== 家庭 ==
单田芳出身曲艺世家，外祖父王福义是[[闯关东|闯关东]]进沈阳最早的[[竹板书|竹板书]]老艺人；母亲王香桂是三四十年代著名的西河大鼓演员，人称“白丫头”；父亲单永魁是弦师；大伯单永生和三叔单永槐分别是西河大鼓和评书演员。妻子王全桂，比单田芳年长8岁。
