《'''風之谷的娜烏西卡'''》（{{lang-ja|風の谷のナウシカ}}）是由日本女星[[安田成美|安田成美]]於1984年發行的個人首張單曲，並且是同年上映動畫《[[風之谷|風之谷]]》的預告片宣傳歌。

== 發表過程 ==
在動畫導演[[宮崎駿|宮崎駿]]進行將他所創作的漫畫《[[風之谷_(漫畫)|風之谷]]》給改編成[[風之谷|同名動畫]]時，影片出版商[[德間書店|德間書店]]舉行了一場挑選出能符合劇中女主角[[娜烏西卡|娜烏西卡]]形象、以及演唱此篇動畫片尾曲的女星選拔賽。<ref>{{cite web |url=https://ciatr.jp/topics/258926 |script-title=ja:安田成美『風の谷のナウシカ』のイメージガールは今も美しいまま！ |work=Ciatr |language=ja |date=2018年3月7日 |accessdate=2019年2月16日}}</ref>選拔賽最後是由[[安田成美|安田成美]]在一共7611名參賽者中勝出，獲得演唱此曲目的機會。<ref>{{cite book |script-title=ja:《ロマンアルバム　風の谷のナウシカ》 |publisher=德間書店 |location=日本 |isbn=978-4-19-720155-6 |date=2001年11月10日 |chapter=〈{{lang|ja|安田成美}}〉|page=128頁}}</ref>

雖然《風之谷的娜烏西卡》有被安排穿插在一些《風之谷》的宣傳影片中，但宮崎駿與影片製作人[[高畑勳|高畑勳]]不認為該曲目的歌詞涵義與曲調風格能夠表達出影片涵義，不接受將此採納為片尾曲，加上當時負責影片配樂的[[久石讓|久石讓]]得到宮崎駿與高畑勳的肯定，便將《風之谷》片尾結束的音樂改成也一同交由久石讓處理。<ref>{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=60至61頁}}</ref>

== 反應 ==
此歌曲推出後，有獲得[[Oricon|Oricon]]排行榜中的年度單曲第63名成績。<ref>{{cite web |url=http://www.http.cc/music/rewrite.php/category-412.html |title=日本Oricon年度單曲排行-日本音樂排行 - 1984年 |work=http.cc音樂 |language=zh-tw |accessdate=2019年2月16日}}</ref>在音樂節目《[[The_Best_Ten|The Best Ten]]》的排行部份，曾擠進該節目週內排行榜的第13名、以及年度百大單曲第96名。<ref>{{cite book |script-title=ja:《ザ・ベストテン―蘇る!80’sポップスHITヒストリー》 |publisher={{lang|ja|角川インタラクティブ・メディア}} |location=日本 |isbn=978-404-894-453-3 |date=20064年12月}}</ref>
