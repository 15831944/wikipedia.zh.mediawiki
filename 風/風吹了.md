《'''風吹了'''》（{{lang-ja|風立ちぬ、かぜたちぬ}}，另譯名《起風了》、《风雪黄昏》、《風起》及《起風》）是日本小說家[[堀辰雄|堀辰雄]]於1936年發表的[[中篇小說|中篇小說]]，小说内容建立在作者自身的经历之上，一共由五章构成，是堀辰雄的代表作品，也被视为一部名作。<ref name="album">{{lang|ja|『新潮日本文学アルバム17 堀辰雄』}}（[[新潮社|新潮社]]、1984年）</ref><ref name="miyashita">{{lang|ja|宮下奈都「カバー解説」}}（{{lang|ja|文庫版『風立ちぬ』}}）（{{lang|ja|角川春樹事務所}}{{lang|ja|ハルキ文庫}}、2012年4月15日）</ref>

== 发表经过 ==
小说的<序曲>和<起风了/风吹了>（{{lang|ja|「風立ちぬ」}}）两章最初以《起风了/风吹了》为题于1936年（[[昭和|昭和]]11年）开始在《改造》杂志12月号（第18卷第12号）上刊载。<ref name="zenshu"/>翌年1937年（昭和12年）在杂志《文藝春秋》1月号（第15卷第1号）发表了<冬>这一章，之后在《新女苑》杂志4月号（第1卷第4号）上登载了<婚約>（即后来的<春>）<ref name="zenshu"/>，小说的最后一章<死荫谷>（{{lang|ja|「死のかげの谷」}}）刊载于《[[新潮_(日本雜誌)|新潮]]》杂志3月号（第35卷第3号），并在同年的4月10日，上述各章节经整合以《起风了/风吹了》为名的单行本由野田书房出版发行<ref name="zenshu">{{lang|ja|「解題」}}（{{lang|ja|『堀辰雄全集第1巻』}}）（{{lang|ja|筑摩書房}}、1977年）</ref><ref>{{cite news|author1=石剑峰|title=《起风了》同名小说出中文版|url=http://www.dfdaily.com/html/150/2013/10/9/1077291.shtml|accessdate=2015-08-22|agency=东方早报|date=2013-10-09}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。现在的日文版由新潮和岩波等文库再版发行。小说也被翻译为了英文（英文标题：''The Wind Has Risen''）、法文（法文标题：''Le vent se lève''）和意大利语（意大利标题：''Si alza il vento''）

== 作品概要 ==
劇情是以1934年至1936年期間，堀辰雄與染上[[肺結核|肺結核]]的未婚妻矢野凌子在[[長野縣|長野縣]]富士見高原療養所生活的點滴為題材、所改編的故事。<ref>{{cite book |title=《風吹了》 |publisher=新雨出版社 |author=[[堀辰雄|堀辰雄]] |isbn=957-733-681-7 |page=4頁 |year=2001年12月 |accessdate=2012月12月5日}}</ref>

在故事的開頭及一些段落中，有引用法國詩人[[保羅·瓦勒里|保羅·瓦勒里]]的著作《海濱墓園》（{{lang|fr|Le cimetière marin}}）中的一句有名诗句：

{{Cquote|風吹了/起风了！当试着生存！/唯有努力试着生存！<ref>{{cite book |title=《風吹了》 |publisher=新雨出版社 |author=堀辰雄 |isbn=957-733-681-7 |page=12、25頁 |year=2001年12月 |accessdate=2012月12月5日}}</ref>}}

日文的「{{lang|ja|風立ちぬ、いざ生きめやも}}」是堀辰雄本人根据法语原文「{{lang|fr|Le vent se lève! Il faut tenter de vivre!}}」翻译的<ref name="album"/><ref name="miyashita2">{{lang|ja|宮下奈都「さあ、生きようじゃないか」}}（{{lang|ja|文庫版『風立ちぬ』}}）（{{lang|ja|ハルキ文庫}}、2012年4月15日）</ref>。

“{{lang|ja|風立ちぬ}}”的“{{lang|ja|ぬ}}”表过去、已完成的助动词，这一句意为“风已经来了”。“{{lang|ja|いざ生きめやも}}”中的“{{lang|ja|いざ}}”表邀强烈的邀约之意。“{{lang|ja|生きめ}}”的语境下是助动词“{{lang|ja|む}}”的已然形，并与后面的反语助词“{{lang|ja|やも}}”连接使这句话意为“来，活下去吗？不还是死去吧”<ref>{{lang|ja|粂川光樹}}「{{lang|ja|『いざ生きめやも』考」『古典と現代』}}1978年10月</ref>，根据前后文，又有“活不下了吗?不一定要活下去”的意思。

== 評價 ==
这部小说是堀辰雄的代表作，小说中的風景描绘语言也十分精彩，風景描写的手法也被指出称赞。小說家[[丸岡明|丸岡明]]認為劇情中描述時間流逝的表現上，有著美麗的詞彙表現。<ref>{{cite book |script-title=ja:《『風立ちぬ・美しい村』について》 |publisher=[[新潮文庫|新潮文庫]] |author=[[丸岡明|丸岡明]] |isbn=978-410-100-402-0  |year=1951年1月 |accessdate=2012月12月5日 }}</ref>

而[[三島由紀夫|三島由紀夫]]則對於堀辰雄在作品中混合著日本及西歐文化兩者風情的感官世界、文體有著好評。<ref>{{cite book |script-title=ja:《日本の文学〈第34〉内田百間,牧野信一,稲垣足穂》 |publisher=中央公論社 |author=[[三島由紀夫|三島由紀夫]]  |year=1970年 |accessdate=2012月12月5日 }}</ref>

== 改編及引用==
《風吹了》在1954年由[[岛耕二|岛耕二]]执导首次改编为电影，在1976年又由[[若杉光夫|若杉光夫]]導演改編成電影《逝風殘夢》，由[[山口百惠|山口百惠]]及[[三浦友和|三浦友和]]主演。<ref>{{Cite web |url=http://www.allcinema.net/prog/show_c.php?num_c=145576 |script-title=ja:風立ちぬ(1976) |publisher=itmedia.com |accessdate=2012年12月15日 }}</ref>

另動畫導演[[宮崎駿|宮崎駿]]在2013年發表的動畫《[[風起|風起]]》，也引用了此小說部份內容。<ref>{{Cite web |url=http://nlab.itmedia.co.jp/nl/articles/1212/13/news096.html |script-title=ja:ジブリ新作は宮崎駿「風立ちぬ」＆高畑勲「かぐや姫の物語」　2013年夏2本同日！ |publisher=itmedia.com |date=2012年12月13日 |accessdate=2012年12月15日 }}</ref>台灣新版書名也一併改為《風起》<ref>[http://www.books.com.tw/exep/activity/activity.php?id=0000055649&sid=0000055649&page=1 宮崎駿2013年動畫《風起》的創作源頭，日本文壇名家崛辰雄生涯代表作：《風起》]</ref>。
