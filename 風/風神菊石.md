'''風神菊石'''（[[學名|學名]]：{{lang|la|'''''Aioloceras'''''}}）是生存在[[白堊紀|白堊紀]][[阿爾布期|阿爾布期]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]。其化石被發現於[[馬達加斯加|馬達加斯加]]、[[巴塔哥尼亞|巴塔哥尼亞]]和[[澳大利亞|澳大利亞]][[昆士蘭省|昆士蘭省]]等地。<ref name="Fossilworks" />

== 特徵 ==
風神菊石的外殼緊縮，旋風狀的螺紋覆蓋殼階的大部分。本屬和''[[Neosaynella|Neosaynella]]''屬菊石及[[克里翁菊石|克里翁菊石]]相似，但風神菊石的殼飾較為光滑而不突出，尤其是殼體側面靠近臍部的肋條相當細緻，至於克里翁菊石的肋條則十分突出，且於靠近臍部的部分明顯隆起。<ref name="Aioloceras">{{Cite web |url = http://liusir.pixnet.net/blog/post/248637581-%E8%8F%8A%E7%9F%B3%E6%AD%A3%E5%90%8D%EF%BC%9A%E9%A2%A8%E7%A5%9E%E8%8F%8A%E7%9F%B3-aioloceras |title = 菊石正名：風神菊石 Aioloceras @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-09 }}</ref>
