《'''風之谷'''》（{{jpn|j=風の谷のナウシカ}}）是由日本動畫工作室[[Topcraft|Topcraft]]製作、[[宫崎骏|宫崎骏]]執導，內容改編自宮崎駿本人連載的[[風之谷_(漫畫)|同名漫畫作品]]，於1984年3月11日上映的動畫電影。<ref>{{cite web|url=http://movie.walkerplus.com/mv17243/|title=風の谷のナウシカ|publisher=Walkerplus|language=ja|accessdate=2013年9月3日}}</ref>

劇情是描述在一個過去古文明曾遭受到毀滅的末世界中，一位叫做「[[娜烏西卡|娜烏西卡]]」少女的冒險故事。在上映後除了獲得來自於《[[電影旬報|電影旬報]]》、[[每日電影獎|每日電影獎]]等方面的獎項外，往後也被日本[[文化廳|文化廳]]視為是最好的國產動畫之一，以及被[[世界自然基金會|世界自然基金會]]挑選為呼籲社會大眾關注[[汙染|環境公害]]、動物保育議題的作品。<ref name="MiyazakiPage34">{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=34頁}}</ref><ref>{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=64頁}}</ref>

== 劇情 ==
過去千年前，世界的產業文明曾發展至巔峰，但經歷一場被稱為{{nihongo|「火之七日」|火の七日間}}的戰爭後，原有的文明毀於一旦。之後世界被一種由菌類構成，名叫{{nihongo|「腐海」|腐海}}的新生態體系所掩蓋。然而腐海只有變種的巨型昆蟲能夠適應其中，人類僅能生活在剩餘的小面積土地上，面對巨型昆蟲和會釋放[[瘴氣|瘴氣]]的腐海森林包圍威脅下積極求存。<ref name="GuideBookPage41">{{cite book |script-title=ja:《風の谷のナウシカ GUILE BOOK 復刻版》 |publisher=[[德間書店|德間書店]] |location=日本 |isbn=978-4-19-720309-3 |date=2010年7月15日 |chapter=〈{{lang|ja|物語}}〉|page=41頁}}</ref>

{{nihongo|「風之谷」|風の谷}}是一個靠海的小王國，因為海風的庇護能不被腐海森林的[[孢子|孢子]]給侵入，而得以保持原貌。風之谷的公主則是一位善良勇敢、名叫[[娜烏西卡|娜烏西卡]]的少女。<ref name="GuideBookPage41"/>

某一天，一架來自{{nihongo|多魯美奇亞|トルメキア}}王國，裡頭載著從{{nihongo|培吉特市|ペジテ市}}所挖掘出過去在火之七日戰爭中毀滅一切的兵器「[[巨神兵|巨神兵]]」之[[運輸機|運輸機]]遭受大批巨型昆蟲攻擊，不幸墜毀於風之谷。之後該王國的第四王女{{nihongo|庫夏娜|クシャナ}}為了搶回巨神兵，派出軍隊入侵風之谷。多魯美奇亞軍隊在庫夏娜的指揮下佔領了風之谷，之後將巨神兵放在風之谷內進行復活工程，並把娜烏西卡等風之谷其他人作為人質送上軍機載走。<ref>{{cite book |script-title=ja:《風の谷のナウシカ GUILE BOOK 復刻版》 |publisher=德間書店 |location=日本 |isbn=978-4-19-720309-3 |date=2010年7月15日 |chapter=〈{{lang|ja|物語}}〉|page=42頁}}</ref>

在飛行途中，多魯美奇亞軍機遭受來自培吉特的少年{{nihongo|阿斯貝魯|アスベル}}所操作之戰機攻擊，混亂中多架多魯美奇亞軍機皆受墜毀，只有娜烏西卡及庫夏娜等一些人駕駛飛行砲艇逃出，降落至腐海森林裡的湖旁。<ref>{{cite book |script-title=ja:《ロマンアルバム　風の谷のナウシカ》 |publisher=德間書店 |location=日本 |isbn=978-4-19-720155-6 |date=2001年11月10日 |chapter=〈{{lang|ja|王蟲の章}}〉|page=30至34頁}}</ref>

之後娜烏西卡為了幫助也同樣來到腐海裡頭、遭受大量昆蟲追擊的阿斯貝魯逃脫，而一同沉落到腐海的地底處。在地底裡，娜烏西卡與阿斯貝魯隨後發現到腐海會將污染大地給淨化的秘密，雖然在過程中會產生劇毒的瘴氣，但被淨化後的腐海地底有著乾淨的空氣和水源，而巨大的王虫則是守護腐海的生物。<ref>{{cite book |script-title=ja:《ロマンアルバム　風の谷のナウシカ》 |publisher=德間書店 |location=日本 |isbn=978-4-19-720155-6 |date=2001年11月10日 |chapter=〈{{lang|ja|王蟲の章}}〉|page=36至37頁}}</ref>

回到地面上的娜烏西卡與阿斯貝魯，意外得知培吉特市市長打算拿小王虫來誘使王蟲群攻擊風之谷，要在混亂之際從多魯美奇亞人手中搶回巨神兵。<ref>{{cite book |script-title=ja:《風の谷のナウシカ GUILE BOOK 復刻版》 |publisher=德間書店 |location=日本 |isbn=978-4-19-720309-3 |date=2010年7月15日 |chapter=〈{{lang|ja|物語}}〉|page=44頁}}</ref>之後娜烏西卡解救了小王蟲後，想以自身為盾來制止憤怒衝來的王蟲群。遭王蟲群衝撞而犧牲死去的娜烏西卡，隨後被冷靜下來的王蟲群以特殊能力給救活。復活後的娜烏西卡，在晨曦金色陽光下站立於王虫群的觸鬚上，而她身上沾著王蟲藍色血液的衣裳，讓她以風之谷過去古老傳說「藍衣人將降臨在金色的草原，並引導人們且帶來希望」的形象現身於大家面前。<ref>{{cite book |script-title=ja:《ロマンアルバム　風の谷のナウシカ》 |publisher=德間書店 |location=日本 |isbn=978-4-19-720155-6 |date=2001年11月10日 |chapter=〈{{lang|ja|金色の草原の章}}〉|page=60頁}}</ref>

== 製作 ==
=== 過程 ===
於1980年代初期，宮崎駿在參與日本和美國兩方動畫公司合力將漫畫家[[溫瑟·麥凱|溫瑟·麥凱]]創作的[[連環漫畫|連環漫畫]]《[[小尼莫_(漫畫)|小尼莫]]》給改編成電影動畫的專案時，內心認為《小尼莫》是屬於各篇幅劇情獨立的短篇漫畫；不容易製作成長篇動畫作品，而開始摸索自行創作的故事。之後宮崎駿曾想出了一篇名叫{{nihongo|《食屍鬼的王女娜烏西卡》|グールの王女ナウシカ}}作品，內容上是受到英國作家[[J·R·R·托爾金|J·R·R·托爾金]]撰寫的奇幻小說裡所提及到[[哥布林|哥布林]]之類的奇幻生物所啓發，劇情方面是描述一位帶領著常受被眾人歧視的食屍鬼民族、名叫娜烏西卡的王女的故事；以及一篇{{nihongo|《土龍與庫夏娜》|土竜とクシャナ}}，講述著一名平時與巨大土龍共同生活、名叫庫夏娜的少女的作品。<ref>{{cite book |title=《[[宮崎駿水彩繪畫作集：風之谷|宮崎駿水彩繪畫作集：風之谷]]》 |publisher=德間書店 |author=[[宮崎駿|宮崎駿]] |location=日本 |isbn=978-419-810-001-8 |date=1996年9月5日 |chapter=〈{{lang|ja|『ナウシカ』誕生までの試行錯誤}}〉|page=151頁}}</ref><ref name="MiyazakiBookPage37">{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=37頁}}</ref>

在同一時期，出版社[[德間書店|德間書店]]正有著想整合影像、音樂等文化媒體作品的營運方針，當時宮崎駿的工作夥伴且身任動漫雜誌《[[Animage|Animage]]》副編輯長的[[鈴木敏夫|鈴木敏夫]]，便建議他提出一份能創作成電影動畫的企劃書。因《小尼莫》改編動畫一事難有進度，宮崎駿便重新構思了一篇以日本古代神話為背景，內容有著各種鎧甲武者、幻想飛天機器的{{nihongo|《戰國魔城》|戦国魔城}}來作為提案。但《戰國魔城》最後被德間書店一方高層以「用沒有原作的題材拍出來的動畫，缺乏足夠的賣點吸引觀眾入場」的理由給回退。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=[[台灣東販|台灣東販]] |author=[[鈴木敏夫|鈴木敏夫]] |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈『風之谷的娜烏西卡』及創立STUDIO GHIBLI〉 |page=34頁}}</ref><ref name="MiyazakiBookPage37"/>

得知被回退的理由後，宮崎駿便在《Animage》動畫雜誌上開始創作一份原作漫畫。而連載的漫畫內容也從原先以日本傳統故事為背影的作品，改成以末世界為舞台的《風之谷》。<ref>{{cite book |script-title=ja:《映画道楽》 |publisher={{lang|ja|びあ}} |author=鈴木敏夫 |location=日本 |isbn=4-8356-1540-9 |date=2005年4月11日 |chapter=〈{{lang|ja|プロデューサー事始め。}}〉|page=66頁}}</ref>

創作《風之谷》後沒多久，宮崎駿內心覺得如果是依據為了要拍影片的前提來創作這份漫畫，對於該漫畫本身是很失敬的事，便想專心畫好漫畫後再談拍片一事。<ref name="ShigotoPage35">{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈『風之谷的娜烏西卡』及創立STUDIO GHIBLI〉 |page=35頁}}</ref>之後鈴木敏夫的上司《Animage》編輯長[[尾形英夫|尾形英夫]]建議宮崎駿可以先試著做出幾分鐘長度的《風之谷》短片，再來確認看看是否有拍攝成影片的可行性，宮崎駿也在對方的激勵下開始有意製作影片。<ref name="ShigotoPage35"/>在確認要進行製作《風之谷》動畫時，原先資助一方的[[德間書店|德間書店]]集團有提出拍攝約半小時長度電視動畫的方案，宮崎駿則表示希望要製作出長度約70分鐘的作品，之後在尾形英夫的反應下敲定成是一部長度約110分鐘的電影動畫。<ref name="MiyazakiBookPage41">{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=41頁}}</ref>

製作影片前，宮崎駿面臨到選擇製作人和找出製作場所方面的難題。當時宮崎駿中意的製作人人選是與他長年合作的前輩[[高畑勳|高畑勳]]，但起初高畑勳認為以自身的行事風格並不適合來擔任製作人，便未接受宮崎駿的要求，之後高畑勳是在鈴木敏夫的遊說之下，答應要求出面。<ref>{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈鈴木敏夫成為鈴木敏夫的瞬間〉 |page=104頁}}</ref>動畫的製作場地部份，鈴木敏夫在透過同事[[龜山修|龜山修]]的消息之下，得知過去曾在[[東映動畫|東映動畫]]與宮崎駿和高畑勳共事過的人士[[原徹|原徹]]有成立一間名為[[Topcraft|Topcraft]]的動畫室，便讓宮崎駿去向原徹會面，以請求借用場地資源。<ref>{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈鈴木敏夫成為鈴木敏夫的瞬間〉 |page=105頁}}</ref>

在之前為了要讓德間書店高層願意將《風之谷》翻拍成動畫，當時鈴木敏夫用盡方式設法讓對方點頭。之後他與龜山修刻意私下找來德間的宣傳部長遊玩{{nihongo|骰子博弈遊戲|チンチロリン}}並有意居下風，讓對方在心情愉悅的情況下趁機提出有關《風之谷》動畫化一事。<ref>{{cite book |script-title=ja:《映画道楽》 |publisher={{lang|ja|びあ}} |author=鈴木敏夫 |location=日本 |isbn=4-8356-1540-9 |date=2005年4月11日 |chapter=〈{{lang|ja|プロデューサー事始め。}}〉|page=67～68頁}}</ref>鈴木敏夫也在廣告公司[[博報堂|博報堂]]來電詢問有關《風之谷》企劃時，臨時誇大《風之谷》原作漫畫的銷售量，想藉此讓對方感興趣來出面贊助。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈「小工廠」 GHIBLI〉 |page=155頁}}</ref>而當時也在博報堂任職的宮崎駿弟弟[[宮崎至朗|宮崎至朗]]，也出面協助讓此影片有上映發表的機會。<ref>{{cite web |url=https://webcache.googleusercontent.com/search?q=cache:OoY35X4tErAJ:https://www.kci.go.kr/kciportal/ci/sereArticleSearch/ciSereArtiOrteServHistIFrame.kci%3FsereArticleSearchBean.artiId%3DART001153587%26sereArticleSearchBean.orteFileId%3DKCI_FI001153587+&cd=3&hl=ja&ct=clnk&gl=tw |script-title=ja:宮崎駿の挫折と成功 - アニメ版『風の谷のナウシカ』に至る道程 |work=東西大學 |author=岡田良平 |language=ja |format=[[PDF|PDF]] |date=2004年5月4日 |accessdate=2019年2月16日}}</ref>其中在和博報堂攜手合作之前，玩具公司[[萬代|萬代]]曾是德間書店想尋找的夥伴，但此構想最後沒有成行。<ref>{{cite book |script-title=ja:《あの旗を撃て!―『アニメージュ』血風録》 |publisher=Oakla |author=[[尾形英夫|尾形英夫]] |location=日本 |isbn=978-477-550-480-2 |date=2004年11月1日 |page=178頁}}</ref>

動畫的製作是在1983年2月1日開始進行，宮崎駿也在連載漫畫至第16回時休刊全心投入執導。在耗時一年多時間繪製5萬多張畫數後於隔年3月2日完成，並在完工後沒多久的11日正式於戲院公開。<ref name="MiyazakiPage34"/><ref>{{cite book |title=《宮崎駿水彩繪畫作集：風之谷》 |publisher=德間書店 |author=宮崎駿 |location=日本 |isbn=978-419-810-001-8 |date=1996年9月5日 |chapter=〈SPECIAL ISSUE〉|page=206頁}}</ref>

=== 劇本內容 ===
{{quote box|width=20%|salign=center|quote={{nihongo|「我將作品的重心放在人類與自然之間的關聯性上：在大自然的環繞下，人類與自然之間的依存關係。」|戦いそのものを描くのではありません。人間を取りかこみ、人間が依存する自然そのものとのかかわりが、作品の重要な主題になるはずです。}}|source=──宮崎駿在《風之谷》的企劃書裡，對於作品內容主旨的看法。<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈即使是沒看過原著的人，也能樂在其中……〉 |page=367頁}}</ref><ref>{{cite book |script-title=ja:《スタジオジブリ作品関連資料集 I》 |publisher=德間書店 |location=日本 |isbn=978-419-860-525-4 |date=1996年6月1日 |page=13頁}}</ref>}}
最早宮崎駿所想要做出長度約70分鐘的《風之谷》動畫裡，劇情上是孩童時期的娜烏西卡學習如何擔任一名御風使，以及會見她的劍術師傅猶巴、以及腐海森林裡王蟲的過程，一部內容類似前傳的作品。<ref name="ShigotoPage35"/>之後是以介於原作漫畫第一、第二集之間的劇情來改編。

在確定要製作成將近兩小時長度的電影後，一開始有找來在《[[福星小子|福星小子]]》電視版動畫裡擔任編劇的[[伊藤和典|伊藤和典]]，來為宮崎駿的原作漫畫內容改寫劇本。<ref name="MiyazakiBookPage43">{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=43頁}}</ref>其中伊藤和典想以劇中角色庫夏娜所屬的國度「多魯美奇亞」，來將電影版動畫改名成「多魯美奇亞戰記」，雖然當時宮崎駿對於《多魯美奇亞戰記》這名稱頗感興趣，但遭到製作人高畑勳表示該名稱無法表現出這是一部講述娜烏西卡的故事而反對。<ref>{{cite book |title=《暢所欲言！押井守漫談吉卜力秘辛》 |publisher=台灣東販 |author=[[押井守|押井守]] |location=台灣 |isbn=978-986-475-850-0 |date=2018年12月1日 |chapter=〈吉卜力工作室成立前夕『風之谷』〉|page=16頁}}</ref>之後伊藤和典因身邊事務緣故退出製作團隊，留下讓宮崎駿自行完成電影劇本，但伊藤和典當時讓庫夏娜身上手臂是[[義肢|義肢]]的設定有採用在作品裡。<ref name="MiyazakiBookPage43"/>

結局內容方面，原先宮崎駿計畫要讓娜烏西卡在面臨眾多大群王蟲時的那一幕就結束劃下句點，之後高畑勳及鈴木敏夫另提出「讓娜烏西卡被迎向而來的大群王蟲給弄傷身亡，成為事後人們心中留下的傳奇」以及「讓娜烏西卡雖然被王蟲給弄傷身亡，但死後又再度復活」的兩種方案。最後宮崎駿在鈴木敏夫的提議下，選擇了現今娜烏西卡死後又復活的結局。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈『風之谷的娜烏西卡』及創立STUDIO GHIBLI〉 |page=45至48頁}}</ref>

其它放棄採用的橋段，有像是在劇情末段裡庫夏娜下令啓動巨神兵時，身上裝置的人造[[子宮|子宮]]突然開始[[羊膜囊破裂|破水]]，之後她便在身上衣裳沾滿鮮血的模樣，仍重新返回到戰場上。<ref name="MiyazakiBookPage54">{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=54頁}}</ref>或是結尾處娜烏西卡與阿斯貝魯一同在風車上看著遠方的夕陽，這種類似青春劇風格的一幕。<ref name="MiyazakiBookPage54"/>

=== 作畫 ===
<!-- 注释出：[[File:Kyoshinhei.jpg|thumb]]所繪製的[[巨神兵|巨神兵]]，達到了宮崎駿想要呈現給觀眾一種肉體帶有「黏性」的感覺。<ref>{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈談庵野秀明 「沒有人知道」的庵野秀明〉 |page=336至337頁}}</ref>]] -->
在選取[[Topcraft|Topcraft]]作為製作場所時，高畑勳為了確保畫面表現上的品質，對原先的Topcraft成員做了測試。但測試結束後因Topcraft成員的作畫能力多數達不到高畑勳的個人要求，讓他最後只願意不動Topcraft原有的攝影人員，在作畫以及美術等部份人選則幾乎都重新找過。<ref name="KazePage106-107">{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈鈴木敏夫成為鈴木敏夫的瞬間〉 |page=106至107頁}}</ref>之後請來了[[中村光毅|中村光毅]]擔任此片的美術指導、[[小松原一男|小松原一男]]負責作畫指導、[[原畫|原畫]]部份則由[[金田伊功|金田伊功]]帶領的陣容。<ref name="KazePage106-107"/><ref name="MiyazakiBookPage54"/>

嘗試的手法上，劇中王蟲移動的畫面裡，宮崎駿採用了一種稱為{{nihongo|「ゴムマルチ」|Gum-Multi}}的技術交付給執導助手[[片山一良|片山一良]]執行。<ref name="ShuubbatsutenPage450">{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈豐饒的大自然，同時也是兇殘的大自然〉 |page=450頁}}</ref>該方法是將繪製王蟲軀幹的[[賽璐珞|賽璐珞]]畫面切分成各部份，之後將各切分部份以繫帶相接，用拉動畫面配合攝影鏡頭的方式來表現出移動模樣。<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈豐饒的大自然，同時也是兇殘的大自然〉 |page=452頁}}</ref><ref>{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=53頁}}</ref>

影片的製作途中因人力仍不足，有在雜誌《[[Animage|Animage]]》上徵求動畫師，當時剛投入動畫界的[[庵野秀明|庵野秀明]]在雜誌上得知消息後，便前往東京向宮崎駿自薦。宮崎駿在看了庵野攜帶的個人作品集後，肯定了對方的能力，讓他成為團隊的一員，並將作品末段裡[[巨神兵|巨神兵]]發射破壞光束攻擊來襲的王蟲群段落交付給他經手處理。<ref>{{cite AV media |title=《風之谷》DVD特別收錄影片：《吉卜力工作室誕生物語》 |medium=[[DVD|DVD]] |format=[[MPEG2|MPEG2]] |time=7分53秒至9分45秒 |publisher=[[博偉|博偉]] |language=ja |location=台灣 |date =2004年1月9日}}</ref>

主角娜烏西卡的配音人選，由先前曾在宮崎駿執導的《[[魯邦三世卡里奧斯特羅之城|魯邦三世卡里奧斯特羅之城]]》替劇中女主角{{nihongo|克蕾莉絲|クラリス}}的[[島本須美|島本須美]]擔任。當時島本須美在人選面試過程時，雖罹患感冒緣故導致發聲情況不佳，但仍被宮崎駿挑選下來希望能與她再次合作。<ref name="MiyazakiBookPage59">{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=59頁}}</ref>宮崎駿在島本須美為娜烏西卡配音的要求條件是「要比起男人更有男人模樣，比起女人更有女人味道」，而她在娜烏西卡回憶起小時一幕的演出表現也得到宮崎駿的肯定。<ref name="MiyazakiBookPage59"/><ref name="ShuubbatsutenPage450"/>

男主角阿斯貝魯是由當時僅十六歲、在電視劇《[[家族遊戲#1983年TBS電視台電視版|家族遊戲]]》演出而獲得關注的青年影星[[松田洋治|松田洋治]]擔任，多年後他有與島本須美一同在同由宮崎駿所執導的動畫《[[魔法公主|魔法公主]]》再次合作。<ref name="MiyazakiBookPage59"/>

在一些需要特殊音效效果的場景部份，劇情裡衆角色戴著防毒面具對話的段落裡，當時是採用[[紙杯|紙杯]]套上[[橡皮筋|橡皮筋]]作成類似面具模樣的小道具，請配音人員套上來錄製出的。<ref>{{cite book |script-title=ja:《ロマンアルバム　風の谷のナウシカ》 |publisher=德間書店 |location=日本 |isbn=978-4-19-720155-6 |date=2001年11月10日 |chapter=〈{{lang|ja|プリンセス．ボイスは銀の色}}〉|page=130頁}}</ref>巨型蟲子王蟲的鳴叫聲，則是請搖滾歌手[[布袋寅泰|布袋寅泰]]用[[吉他|吉他]]所演奏出的聲音。<ref>{{cite web |url=https://www.cinematoday.jp/news/N0030279 |script-title=ja:『風の谷のナウシカ』王蟲の声は布袋寅泰のギター！本人がツイッターで明かす！ |work=Cinema Today |language=ja |date=2011年2月10日 |accessdate=2019年2月16日}}</ref>

=== 配樂 ===
[[File:Joe_Hisaishi_in_Krakow.JPG|thumb]]。當時高畑勳選擇他作為人選的決定，間接促成了往後宮崎駿與他的長年合作。<ref>{{cite book |script-title=ja:《ジブリの哲学――変わるものと変わらないもの》 |publisher=[[岩波書店|岩波書店]] |author=鈴木敏夫 |location=日本 |isbn=978-400-023-495-5 |date=2011年8月11日 |chapter=〈{{lang|ja|宮崎・久石コンビはこうして生まれた}}〉|page=77頁}}</ref>]]
考慮到《風之谷》內容會像是史詩般壯大的動畫，宮崎駿等人難以挑選出適合擔任配樂的人選，之後[[德間日本傳播|德間日本傳播]]一方人士推薦了當時尚未特別有名氣的[[久石讓|久石讓]]，高畑勳在評估對方之後，決定將他作為人選。<ref>{{cite book |script-title=ja:《ジブリの哲学――変わるものと変わらないもの》 |publisher=[[岩波書店|岩波書店]] |author=鈴木敏夫 |location=日本 |isbn=978-400-023-495-5 |date=2011年8月11日 |chapter=〈{{lang|ja|宮崎・久石コンビはこうして生まれた}}〉|page=76頁}}</ref>

敲定好配樂人選之後，高畑勳另提了一個在對製作完成的動畫進行配樂之前，先做出「印象專輯」的方案。他指示要宮崎駿先寫出幾首描述此部作品的詩，來讓久石讓透過這些詩構想出之後會用在正式配樂上的雛形樂曲。<ref>{{cite book |script-title=ja:《ジブリの哲学――変わるものと変わらないもの》 |publisher=岩波書店 |author=鈴木敏夫 |location=日本 |isbn=978-400-023-495-5 |date=2011年8月11日 |chapter=〈{{lang|ja|イメージ．アルバムを作ろう}}〉|page=74頁}}</ref>

隨後久石讓在八個月內，完成了以[[合成器|合成器]]為中心，搭配歐洲[[德西馬琴|德西馬琴]]與南美洲[[克納笛|克納笛]]等民族樂器的印象專輯、和採用在影片配樂的原聲帶、以及額外製作出以[[管弦樂團|管弦樂團]]演奏的[[交響曲|交響曲]]版專輯以上三份成品。<ref>{{cite book |script-title=ja:《ロマンアルバム　風の谷のナウシカ》 |publisher=德間書店 |location=日本 |isbn=978-4-19-720155-6 |date=2001年11月10日 |chapter=〈{{lang|ja|久石 譲「ナウシカを通して音楽を入れてみました」}}〉|page=170頁}}</ref><ref name="MiyazakiBookPage61">{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=61頁}}</ref>此期間他構想過八十種以上的樂曲，並有讓他當時四歲的女兒[[藤澤麻衣|藤澤麻衣]]在影片裡娜烏西卡死而復活的段落中，詠唱該段搭配的音樂。<ref name="MiyazakiBookPage61"/><ref>{{cite web |url=http://www.zakzak.co.jp/entertainment/ent-news/news/20121114/enn1211141214009-n1.htm |script-title=ja:久石譲の娘・麻衣が語る“親子の葛藤”　歌手になる…父の説得に３年 |work=產經 Digital |language=ja |date=2012年11月14日 |accessdate=2019年2月16日}}</ref>

片尾的主題歌方面，原先一開始是計劃採用由[[松本隆|松本隆]]作詞、[[細野清臣|細野清臣]]作曲、[[萩田光雄|萩田光雄]]編曲、女星[[安田成美|安田成美]]演唱的《[[風之谷的娜烏西卡_(單曲)|風之谷的娜烏西卡]]》，但此歌曲內容因未能得到宮崎駿和高畑勳的青睞遭到捨棄，最後在影片結尾處是改用由久石讓創作的樂曲{{nihongo|《鳥人》|鳥の人}}。<ref name="MiyazakiBookPage61"/>而宮崎駿也曾向久石讓提議來將[[蘇聯|蘇聯]]歌手[[弗拉基米爾·維索茨基|弗拉基米爾·維索茨基]]所創作的《地球之歌》（{{lang|ru|Песня О Земле}}）給改編曲調，再搭配自己所撰寫的歌詞來作為《風之谷》的片尾曲，但此提議因弗拉基米爾已去逝的因素下，在取得版權方面有難度而放棄。<ref name="MiyazakiBookPage61"/><ref name="MiyazakiPage62">{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=62頁}}</ref>

=== 宣傳標語 ===
此影片是鈴木敏夫首次接觸到宣傳事務的作品，他當時與曾替《[[永遠的大和號|永遠的大和號]]》、《[[銀河鐵道999|銀河鐵道999]]》劇場版等動畫電影擔任過宣傳的[[德山雅也|德山雅也]]合作，替《風之谷》討論出好的宣傳標語。<ref>{{cite book |script-title=ja:《ジブリの仲間たち》 |publisher=新潮社 |author=鈴木敏夫 |location=日本 |isbn=978-410-610-674-3 |date=2016年6月20日 |chapter=〈{{lang|ja|作ることにしか興味がなかった僕が宣伝を始めるまで}}〉|page=13頁}}</ref>

之後[[博報堂|博報堂]]一方人士也向鈴木敏夫提出他們所想到的{{nihongo|「人類全滅亡了？」|人間はもういらないのか？}}這樣標語，不過遭到鈴木敏夫反對，他認為博報堂所想出的內容完全沒把作品要傳遞的主題給呈現出。<ref>{{cite book |script-title=ja:《映画道楽》 |publisher={{lang|ja|びあ}} |author=鈴木敏夫 |location=日本 |isbn=4-8356-1540-9 |date=2005年4月11日 |chapter=〈{{lang|ja|映画の宣伝は三位一体で作られる。}}〉|page=130頁}}</ref>隨後德山雅也想出的標語為{{nihongo|「少女的愛呼喚了奇蹟」|少女の愛が奇跡を呼んだ}}，但鈴木敏夫對此內容覺得落於俗套而有所顧慮。<ref name="GhibliNakamaPage15">{{cite book |script-title=ja:《ジブリの仲間たち》 |publisher=新潮社 |author=鈴木敏夫 |location=日本 |isbn=978-410-610-674-3 |date=2016年6月20日 |chapter=〈{{lang|ja|作ることにしか興味がなかった僕が宣伝を始めるまで}}〉|page=15頁}}</ref>德山雅也後續進一步向鈴木敏夫提出必須要更改片名的想法，他認為「風之谷」這幾個字讓觀眾不易理解，而建議應改成{{nihongo|《風之戰士娜烏西卡》|風の戦士ナウシカ}}。<ref name="GhibliNakamaPage15"/>在標語不好決定以及被要求更改片名的情況時，鈴木敏夫詢問了高畑勳的看法，高畑勳表示著博報堂一方的標語確實不佳，德山雅也部份則是可接受他所想出的標語；但不應更改片名。<ref name="GhibliNakamaPage15"/>

最後作品主要的宣傳標語是「少女的愛呼喚了奇蹟」，另外在其它一小部份的廣告上也採用了德山雅也另外想出的{{nihongo|「火之七日改變了地球…」|火の7日間が地球変えた…}}，以及鈴木敏夫個人所提的{{nihongo|「關愛樹林、與蟲交談、召喚風的鳥人…」|木々を愛で 虫と語り 風をまねく鳥の人…}}等標語。<ref>{{cite book |script-title=ja:《ジブリの仲間たち》 |publisher=新潮社 |author=鈴木敏夫 |location=日本 |isbn=978-410-610-674-3 |date=2016年6月20日 |chapter=〈{{lang|ja|作ることにしか興味がなかった僕が宣伝を始めるまで}}〉|page=16頁}}</ref>

負責影片上映的[[東映|東映]]公司起初並沒有特別對《風之谷》投入宣傳，之後靠德間書店社長[[德間康快|德間康快]]親自會面東映社長[[岡田茂|岡田茂]]後，讓對方有較積極的動作。<ref>{{cite book |script-title=ja:《徳間康快　夢を背負って、坂道をのぼり続けた男》 |publisher=文化通信社 |author=金澤誠 |location=日本 |isbn=978-484-590-687-1 |date=2010年9月16日 |page=130至131頁}}</ref>

=== 外語版本爭議 ===
[[File:Nausicaa_of_the_Valley_of_the_Wind(USA_VHS).jpg|150px]]
影片公司New World Pictures於1985年6月期間，在美國當地上映了《風之谷》。<ref>{{cite book |title=《Hayao Miyazaki Master of Japanese Animation》 |publisher=Stone Bridge Press |author=Helen McCarthy |location=美國 |isbn=978-188-065-641-9 |date=1999年9月1日 |page=72至99頁}}</ref>但當時New World Pictures除了將片名修改成《風之戰士》（Warriors of the Wind）之外，另將影片剪輯縮短成97分鐘，其中劇中角色與場景名稱也遭替換，如娜烏西卡的名字被換成「桑德拉公主」（Princess Zandra）、阿斯貝魯則改為「米洛王子」（Prince Milo）、腐海則是「毒叢林」（Toxic Jungle）。<ref>{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=68頁}}</ref>在劇情部份New World Pictures有將一些節奏看起來「比較緩慢」的段落給刪減掉，導致當時《風之谷》英語配音人員在錄製中有難以融入劇情狀況。<ref name="Nausicaa.Net_FAQ">{{cite web |url=http://www.nausicaa.net/miyazaki/nausicaa/faq.html#warrior |title=FAQ |work=Nausicaa.net |language=en |accessdate=2011年5月29日}}</ref>到了該年12月New World Pictures發行了北美版的《風之谷》[[錄影帶|錄影帶]]，當時外盒上有請專人做重新繪製的封面，不過封面中出現許多劇情裡所沒有的角色，如持槍的金髮男子、骷髏造型戰士、騎乘飛馬的勇者，娜烏西卡也被繪製成身著緊身迷你裙的西洋金髮女郎風格造型。<ref name="Nausicaa.Net_FAQ"/>

最初宮崎駿並未得知New World Pictures的篡改行為，直至1985年9月期間讀到《[[朝日新聞|朝日新聞]]》刊登的文章{{nihongo|《現今是動畫時代》|いまアニメの時代}}才了解此事，並對該情況感到不悅。<ref name="MiyazakiBookPage67"/>之後鈴木敏夫於1995年在[[安錫國際動畫影展|安錫國際動畫影展]]上演講時，表示若有人曾看過過去北美版的《風之谷》，希望他們能忘掉這個版本的存在。<ref name="MiyazakiBookPage69">{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|風の谷のナウシカ 1984}}〉|page=69頁}}</ref>1980年代的北美版內容問題，到了2005年由[[華特迪士尼影業|華特迪士尼影業]]取得發行版權後，做了重新配音、不異動影片長度，並將譯名還原為《Nausicaä of the Valley of the Wind》後獲得解決。<ref name="MiyazakiBookPage69"/>不過迪士尼所做的翻譯內容，有被[[多特蒙德工業大學|多特蒙德工業大學]]的講師荻原-施克·繪里子（Eriko Ogihara-Schuck）指出迪士尼為了讓美國當地觀眾較好理解內容，字幕上沒表現出日語原版所傳遞出的[[泛靈論|萬物有靈論]]訊息，這樣導致觀看迪士尼翻譯內容的觀眾會認為娜烏西卡是為了人類做出自我犧牲；而不是為了自然。<ref>{{cite book |title=《Miyazaki's Animism Abroad: The Reception of Japanese Religious Themes by American and German Audiences》 |publisher=McFarland Books |author=Eriko Ogihara-Schuck |location=美國 |isbn=978-0-7864-7262-8 |date=2014年10月16日 |chapter=〈The Christianizing of Animism in Manga and Anime: American Translations of Hayao Miyazaki's Nausicaä of the Valley of the Wind〉 |pages=133至146頁}}</ref> 

歐洲地方發行的版本起初也有類似北美版情況，德語版一開始發行的錄影帶將片名取為《星際戰士》（{{lang|de|Sternenkrieger}}）。<ref name="Nausicaa.Net_Video">{{cite web |url=http://www.nausicaa.net/miyazaki/video/nausicaa/ |title=Video List: Kaze no Tani no Naushika |work=Nausicaa.net |language=en |accessdate=2019年2月16日}}</ref>而法語版由VIP International發行時片名被改為《幽靈船》（{{lang|fr|Le Vaisseau Fantôme}}），另一家法語片商Blue Kid's Video則是用《星星公主》（{{lang|fr|La Princesse des Etoiles}}）如此片名。<ref name="Nausicaa.Net_Video"/>德語和法語版，則分別各在2005和2007年期間重新推出還原內容的[[DVD|DVD]]版本。<ref name="Nausicaa.Net_Video"/>

在亞洲方面，香港的AV亞洲影帶曾有讓[[徐克|徐克]]負責監製，邀請[[黃霑|黃霑]]、[[陳慧嫻|陳慧嫻]]、[[沈殿霞|沈殿霞]]等明星錄製配音，且把台詞篡改成偏向[[賀歲片|賀歲片]]的詼諧內容。<ref name="hkwenweipo">{{cite web |url=http://paper.wenweipo.com/2011/02/16/OT1102160005.htm |title=影碟別注：《風之谷》藍光碟 |work=[[文匯報_(香港)|香港文匯報]] |author=亞里安 |language=zh-hk |date=2011年2月26日 |accessdate=2014年5月22日}}</ref>往後香港推出的[[VCD|VCD]]和[[DVD|DVD]]版本，則未有當時AV亞洲影帶篡改台詞的爭議情況，並在2011年的[[藍光光碟|藍光]]版本中再一次重新選角配音。<ref name="hkwenweipo"/>

一些傳播媒體的看法上，網站[[動畫新聞網|動畫新聞網]]肯定《風之谷》劇中的角色設計、執導方式、配樂，認為此作品足以在過去至今經典動畫清單裡佔有一席之地。<ref>{{cite web |url=http://www.animenewsnetwork.com/review/nausicaa-of-the-valley-of-wind/dvd |title=Nausicaä of the Valley of Wind DVD |work=[[動畫新聞網|動畫新聞網]] |author=Theron Martin |language=en |date=2005年3月16日 |accessdate=2019年2月16日}}</ref>Comicbook.com則提到雖然沒有明確證明，但2015年上映的《[[星際大戰|星際大戰]]》系列第七部電影《[[STAR_WARS：原力覺醒|STAR WARS：原力覺醒]]》裡頭，有被影迷指出不少會令人聯想到《風之谷》的影子，包含劇中一些荒涼場景的鏡頭、女主角[[芮|芮]]的面罩打扮與娜烏西卡戴上瘴氣面具模樣之間的相似度等等。<ref>{{cite web |url=https://comicbook.com/anime/2017/12/19/star-wars-hayao-miyazaki-connection-anime-nausicaa/ |title=Did You Notice This Hayao Miyazaki ’Star Wars' Connection? |work=ComicBook.Com |author=Megan Peters |language=en |date=2017年12月18日 |accessdate=2019年2月24日}}</ref>

英國作家海倫·麥卡錫（Helen McCarthy）在個人著書《500部必看動畫電影》（500 Essential Anime Movies）裡，表示此部作品的強勢在於劇本內容、令人感到興奮與激動的橋段、以及配樂等方面。<ref>{{cite book |title=《500 Essential Anime Movies: The Ultimate Guide》 |publisher=Harper Design |author=Helen McCarthy |location=英國 |isbn=978-006-147-450-7 |date=2009年1月6日 |page=24頁}}</ref>日本漫畫家[[星野桂|星野桂]]提及《風之谷》是她在小學時觀賞多次，令她內心大受感動的個人最愛動畫。<ref>{{cite web |url=http://jumpsq.shueisha.co.jp/contents/m_int_hoshino/index.html |archive-url=https://web.archive.org/web/20100315091947/http://jumpsq.shueisha.co.jp/contents/m_int_hoshino/index.html |dead-url=yes |script-title=ja:ジャンプスクエア　マンガ家 直撃インタビュー［モノガタリ］ |work=[[週刊少年Jump|週刊少年Jump]] |language=ja |archive-date=2010年3月15日 |accessdate=2019年2月16日}}</ref>美國科幻小說家[[歐內斯特·卡倫巴赫|歐內斯特·卡倫巴赫]]認為此影片的部份橋段，恰巧與他的著作《[[生態烏托邦|生態烏托邦]]》裡談論內容有所共鳴，並認為是部有機會在美國上映成功的作品。<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈談論「風之谷」中的未來，捨棄火苗的「風之谷」與備有冰箱的「生態理想國（ECOTOPIA）」〉 |page=303頁}}</ref>

製作群之一的高畑勳認為此部影片帶給了觀眾過去前所未見內容，在節奏上也拿捏得相當好，整體上具備所有賣座電影應有的條件，以他個人身為這部影片製作人的立場上，他認為《風之谷》達到完美一百分的成績；但若是身為宮崎駿朋友的立場上，他希望宮崎駿應該得從這部電影開始去朝向新的境界邁進，只願意打出三十分的成績。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈『風之谷的娜烏西卡』及創立STUDIO GHIBLI〉|page=50至51頁}}</ref>宮崎駿的友人動畫導演[[押井守|押井守]]表示《風之谷》從不是他個人喜愛的作品，他不認同劇中娜烏西卡一方面形象太過完美、一方面又像過去的[[布爾什維克|布爾什維克]]黨派般地隨意訴諸武力，也批評沒有對觀眾解釋清楚娜烏西卡是出於何種原因來願意做出自我犧牲的行為。<ref>{{cite book |title=《暢所欲言！押井守漫談吉卜力秘辛》 |publisher=台灣東販 |author=押井守 |location=台灣 |isbn=978-986-475-850-0 |date=2018年12月1日 |chapter=〈吉卜力工作室成立前夕『風之谷』〉|page=21至26頁}}</ref>

宮崎駿則自評在片尾結局的部份雖然設法想排除宗教方面的色彩，但娜烏西卡最後復活被王蟲舉起在半空的那一幕，仍讓他感覺帶有宗教意味的表現或是在塑造[[聖女貞德|聖女貞德]]般地形象，而成為他個人相當介意的地方。<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈豐饒的大自然，同時也是兇殘的大自然〉 |page=447頁}}</ref>另宮崎駿當時聽從鈴木敏夫的提議，將結局安排成娜烏西卡死後又再度復活的內容，也讓他事後內心不時質疑這是否為正確的抉擇。<ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=[[台灣東販|台灣東販]] |author=[[鈴木敏夫|鈴木敏夫]] |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈『風之谷的娜烏西卡』及創立STUDIO GHIBLI〉 |page=49頁}}</ref>

=== 社會現象 ===
[[File:Nausicaä_of_the_Valley_of_the_Wind_-_Möwe_Glider.jpg|thumb]]展示，由[[八谷和彥|八谷和彥]]設計仿造劇情中娜烏西卡所使用滑翔翼的「[[OpenSky_M-02|M-02J]]」。]]
影片上映後，多數大眾媒體將此片視為對於維護自然、保育環境方面的象徵作品。<ref name="MiyazakiBookPage66"/>如1985年1月8日《[[朝日新聞|朝日新聞]]》刊登的一篇探討自然問題的文章，便引用了此作品做為例子。<ref name="MiyazakiBookPage66"/>1988年在[[奈良縣|奈良縣]]舉行的{{nihongo|「奈良絲路博覽會」|なら・シルクロード博覧会}}裡，會場中展示的一段內容關於由過去地球誕生至現代公害問題影片裡，也採用了《風之谷》的一些場面。<ref name="MiyazakiBookPage66"/>

作品裡登場的虛構飛行具方面，2003年日本新媒體藝術家[[八谷和彥|八谷和彥]]以《Open Sky Project》為名義的專案，著手設計出劇中娜烏西卡所使用滑翔翼的仿真成品，之後依序弄出了一架名為「Möwe1/2」按原作設定二分之一比例的遙控機，以及兩架按原作一比一比例各名為「M-01」、「M-02」不俱馬達的滑翔機。<ref>{{cite web |url=https://www.gizmodo.jp/2013/09/post_13101.html |script-title=ja:本当に飛べる！ ナウシカのメーヴェを実際に作った八谷和彦さんにお会いしてきたよ！ |work=[[Gizmodo|Gizmodo]] |author=嘉島唯 |language=ja |date=2013年9月5日 |accessdate=2019年2月16日}}</ref>到了2014年，八谷和彥做出除了按原作一比一比例之外，且具有噴射引擎的成品「[[OpenSky_M-02|M-02J]]」，M-02J在2014年夏季於[[北海道|北海道]][[瀧川市|瀧川市]]公開飛行，並有在2016年達到飛行70公尺高度的記錄。<ref>{{cite web |url=https://www.sankei.com/life/news/160731/lif1607310019-n1.html |script-title=ja:「ナウシカ」の翼、空舞う　北海道で公開テスト飛行 |work=[[產經新聞|產經新聞]] |language=ja |date=2016年7月31日 |accessdate=2019年2月16日}}</ref>

== 週邊 ==
=== 影片 ===
[[錄影帶|錄影帶]]版本是在影片公開的同一月發行，之後在日本有賣出約12萬份的成績，也是當時[[Oricon|Oricon]]的錄影帶銷售排行榜首。<ref name="MiyazakiBookPage66"/><ref name="Navicon">{{cite web |url=https://navicon.jp/news/14641/ |script-title=ja:宮崎駿不朽の名作「風の谷のナウシカ」がニューマスターで金曜ロードSHOW!に登場－日テレ |work=Navicon |language=ja |date=2012年5月11日 |accessdate=2019年2月16日}}</ref>[[DVD|DVD]]版本則在上映的19年後，於2003年11月期間推出，之後有達到約75萬份的銷售量，並曾拿下2003年12月首週日本當地DVD影片銷售的第一名成績。<ref name="MiyazakiBookPage66"/><ref>{{cite web |url=https://www.oricon.co.jp/prof/34386/products/517773/1/ |script-title=ja:風の谷のナウシカ アニメーション |work=[[Oricon|Oricon]] |language=ja |accessdate=2019年2月16日}}</ref>

高畫質的[[藍光光碟|藍光光碟]]版本於2010年夏季發行。<ref>{{cite web |url=https://av.watch.impress.co.jp/docs/news/360753.html |script-title=ja:「風の谷のナウシカ」Blu-ray版の詳細仕様が発表 |work=AV Watch |author=山崎健太郎 |language=ja |date=2010年4月12日 |accessdate=2019年2月16日}}</ref>在《風之谷》藍光版推出之前，2009年年底吉卜力便有以影片《[[崖上的波妞|崖上的波妞]]》製作出公司首個的藍光動畫商品。<ref>{{cite web |url=https://av.watch.impress.co.jp/docs/news/298713.html |script-title=ja:「崖の上のポニョ」ドキュメンタリーBD/DVDが大幅発売延期 |work=AV Watch |author=山崎健太郎 |language=ja |date=2009年7月1日 |accessdate=2019年2月16日}}</ref>之後日本的華特迪士尼，前去吉卜力討論要將過去至今的吉卜力動畫重新做成一系列的藍光影碟發行。雖然當時日本華特迪士尼與吉卜力多數人士考慮先推出《[[天空之城|天空之城]]》的藍光版，但鈴木敏夫獨排眾議，表明應先從《風之谷》開始推出。<ref name="GhibliLiteraturePage3-5">{{cite book |script-title=ja:《ジブリの文学》 |publisher=岩波書店 |author=鈴木敏夫 |location=日本 |isbn=978-400-061-194-7 |date=2017年3月28日 |chapter=〈{{lang|ja|『ナウシカ』は、まだ終わっていない}}〉|page=3至5頁}}</ref>

鈴木敏夫會提議要先重新推出《風之谷》，是回想到先前宮崎駿在[[迪士尼頻道|迪士尼頻道]]上，觀看經過數位技術調整色彩的迪士尼經典動畫時內心有所不滿，他認為將色彩給調整成過於清晰反而褻瀆了這些老作品。<ref name="GhibliLiteraturePage3-5"/>而製作《風之谷》時所用到的[[底片#負片|負片]]，在過去上映結束後有被保留下來，便想利用庫存的負片，來還原出當時呈現在劇院裡的色彩。<ref name="GhibliLiteraturePage3-5"/>之後鈴木敏夫請吉卜力攝影指導[[奧井敦|奧井敦]]處理這項請求，奧井敦則以一格一格掃描負片的手法，還原畫面到藍光版上，並依照宮崎駿建議來特別針對部份場景的[[綠色|綠色]]做出調整。<ref name="GhibliLiteraturePage3-5"/>最後奧井敦做出的藍光版畫面成果，有令宮崎駿忍不住感嘆落淚。<ref name="GhibliLiteraturePage3-5"/>藍光版上市後，有成為2010年7月末週期間日本影音市場藍光影片銷售的第一名，也與過去的《風之谷》錄影帶版與DVD版相同，都有拿下過影音銷售排行首位的紀錄。<ref>{{cite web |url=https://www.oricon.co.jp/prof/34386/products/865550/1/ |script-title=ja:風の谷のナウシカ アニメーション |work=Oricon |language=ja |accessdate=2019年2月16日}}</ref><ref name="Navicon"/>

=== 電子遊戲 ===
[[德間書店|德間書店]]有與旗下的Technopolis Soft製作了幾份改編自《風之谷》的遊戲。第一款是適用於[[NEC|NEC]]的[[PC-8800|PC-8800]]主機平台，與動畫同名的的[[冒險遊戲|冒險遊戲]]。第二款是用在NEC[[PC-6001|PC-6001]]主機，名叫{{nihongo|《娜烏西卡危機一髮》|ナウシカ危機一髪}}的[[射擊遊戲|射擊遊戲]]，內容是讓娜烏西卡駕著風之谷的砲艇，擊毀原作漫畫裡所登場國家「土鬼」一方所駕駛的飛行船。最後一款是使用在[[MSX|MSX]]主機，名叫{{nihongo|《被遺忘的娜烏西卡遊戲》|忘れじのナウシカ・ゲーム}}的作品，遊戲裡是讓娜烏西卡對抗土鬼一方，並且有要讓玩家擊倒王蟲的場合。而電子遊戲所改編的內容，都未能得到宮崎駿認同。<ref>{{cite web | url=https://kotaku.com/5862547/ni-no-kuni-isnt-miyazakis-first-video-game-appearance |title=Ni No Kuni Isn't Miyazaki/Ghibli's First Video Game Appearance |work=[[Kotaku|Kotaku]] |author=Luke Plunkett |language=en |date=2011年11月25日 |accessdate=2019年2月16日}}</ref>

== 後續 ==
=== 吉卜力 ===
《風之谷》上映雖獲得票房迴響，但原先的製作場所[[Topcraft|Topcraft]]在1985年便停止營運。之後為了讓高畑勳計劃拍攝的紀錄片《[[柳川堀割物語|柳川堀割物語]]》能有足夠的預算資金，高畑勳、宮崎駿、鈴木敏夫等人便開始籌辦一間獨立的動畫室來拍攝作品賺取費用，後續所成立的工作室為現今的[[吉卜力工作室|吉卜力工作室]]，並有邀請當時《風之谷》製作團隊裡的一些人員進入。<ref>{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈鈴木敏夫成為鈴木敏夫的瞬間〉 |page=115至116頁}}</ref><ref>{{cite book |title=《樂在工作：進入宮崎駿．高畑勳的動畫世界》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-176-961-5 |date=2009年7月15日 |chapter=〈『風之谷的娜烏西卡』及創立STUDIO GHIBLI〉 |page=52頁}}</ref>

雖然《風之谷》是吉卜力工作室成立前就完成的作品，但事後動畫版權被歸類在吉卜力的名下，並被列為吉卜力的長篇動畫之一。<ref>{{cite web |url=http://www.ghibli.jp/chronology/ |script-title=ja:スタジオジブリの年表 |work=[[吉卜力工作室|吉卜力工作室]] |language=ja |accessdate=2019年2月16日}}</ref>

=== 續集問題 ===
影片上映結束後，[[德間書店|德間書店]]一方曾希望宮崎駿能盡快推出《風之谷》的續集動畫，但當時宮崎駿無意為了商業目的推出續作。<ref name="MiyazakiBookPage67"/>在推出吉卜力的首部電影動畫《[[天空之城|天空之城]]》後，當時宮崎駿認為該部作品具備著和《風之谷》相同的「漫畫電影」要素，某種程度涵義上能視為是《風之谷》的姐妹作品。<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈對個人而言，它是「風之谷」的延續〉 |page=460頁}}</ref>

之後小說家[[夢枕獏|夢枕獏]]於1986年在雜誌《[[Animage|Animage]]》上與宮崎駿對談時，對於宮崎駿所提到原作漫畫裡[[巨神兵|巨神兵]]的未來情節走向很感興趣，便提議對方往後若有機會，可以嘗試作出《風之谷》動畫續集。<ref>{{cite book |title=《出發點 1979～1996》 |publisher=台灣東販 |author=宮崎駿 |location=台灣 |isbn=978-957-473-843-4 |date=2006年2月1日 |chapter=〈有個故事，希望能借重宮崎先生將其拍成電影〉 |page=317頁}}</ref>參與過《風之谷》動畫團隊的[[庵野秀明|庵野秀明]]後續也曾提議想以劇中庫夏娜為主角，來作出《風之谷》的外傳，但宮崎駿對於庵野的企劃內容純粹在描述戰爭，而否決了他的提議。<ref>{{cite journal |script-title=ja:〈少し前よりもナウシカの事が少しわかるようになった。 ロング・インタビュー宮崎駿〉 |journal=COMICBOX月刊 |volume=98 |date=1995年1月 |pages=21頁}}</ref>

在1993年期間原作漫畫劇情已來到末段即將連載結束之時，宮崎駿仍表示未有拍攝動畫續作的意圖。<ref>{{cite web |url=http://premium.nikkeibp.co.jp/itm/col/suzuki/143/ |archive-url=https://web.archive.org/web/20081004233709/http://premium.nikkeibp.co.jp/itm/col/suzuki/143/ |dead-url=yes |script-title=ja:映画『風の谷のナウシカ』に続編があった？ |work=Nikkei Business Publications |language=ja |date=2008年9月3日 |archive-date=2008年10月4日 |accessdate=2019年2月16日}}</ref>到了2013年9月6日宮崎駿在個人引退的記者會上，對於媒體詢問也表示並沒有預定製作《風之谷》續集的想法。<ref>{{cite web |url=http://mainichi.jp/enta/news/20130906mog00m200014000c.html |archive-url=https://web.archive.org/web/20130907031341/http://mainichi.jp/enta/news/20130906mog00m200014000c.html |dead-url=yes |script-title=ja:宮崎駿監督:ナウシカ続編「ありません」 引退会見の主な発言 |work=[[每日新聞|每日新聞]] |language=ja |date=2013年9月6日 |archive-date=2013年9月7日 |accessdate=2019年2月16日}}</ref>

不過在2012年期間，庵野秀明有與特效導演[[樋口真嗣|樋口真嗣]]合作，拍攝出以巨神兵為主角來入侵21世紀[[東京|東京]]的特攝短片《[[巨神兵現身東京|巨神兵現身東京]]》。<ref>{{cite web |url=https://www.oricon.co.jp/news/2011378/ |script-title=ja:ナウシカの“巨神兵”が特撮短編映画化 企画は“エヴァ”の庵野秀明 |work=Oricon |author= |language=ja |date=2012年5月10日 |accessdate=2019年2月16日}}</ref>
