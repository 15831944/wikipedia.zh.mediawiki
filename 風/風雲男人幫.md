{{作品名稱|name=''Gangster Squad''|hk=反黑暴隊|tw=風雲男人幫|cn=匪帮传奇}}是一部2013年[[美國|美國]][[犯罪片|犯罪]]、[[劇情片|劇情]]題材的電影，由[[魯賓·弗來舍|魯賓·弗來舍]]執導<ref>{{cite web |url=http://screenrant.com/the-gangster-squad-movie-ruben-fleischer-robf-90199 |title=‘Zombieland’ Director Tapped for Crime Drama ‘The Gangster Squad’ |author=Frappier, Rob |work=Screen Rant |accessdate=2011-09-13 }}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>，[[喬許·布洛林|喬許·布洛林]]、[[瑞恩·高斯林|賴恩·高斯寧]]、[[尼克·諾特|尼克·諾特]]、[[艾瑪·史東|艾瑪·史東]]和[[西恩·潘|西恩·潘]]主演。

電影最初於2012年9月7日發行<ref name="InsideMovies1">{{cite web |last=Franich |first=Darren |url=http://insidemovies.ew.com/2012/07/25/warner-bros-pushes-gangster-squad |title='Gangster Squad: Warner Bros. pushing back release date |work=InsideMovies.EW.com |accessdate=2012-09-14}}</ref>，但在[[2012年奥罗拉枪击事件|2012年奥罗拉枪击事件]]之後，發行日期便被延後至2013年1月11日<ref name="WarnerBros1">{{cite web |url=http://www.warnerbros.com/movies/in-theaters-coming-soon/gangster-squad/075a4928-d88a-430d-a8ee-f623a0610423.html |title=GANGSTER SQUAD |work=WarnerBros.com |accessdate=2012-09-14}}</ref>。

== 劇情 ==
1949年，冷酷無情的黑道老大[[米奇·柯罕|米奇·柯罕]]（[[西恩·潘|西恩·潘]]飾）主宰洛杉磯大小事情，針對被腐敗警界保護的猖獗黑道，警察總長比爾·帕克（[[尼克·諾特|尼克·諾特]]飾）採取了極端手段，他徵召身經百戰的英雄警官約翰·歐馬拉（[[喬許·布洛林|喬許·布洛林]]飾）成立一個嚴正的執法小組，他們的工作不是逮捕壞人，而是開戰。當城市爆發了槍戰，疲憊不堪的警官傑瑞·伍特斯（[[雷恩·葛斯林|雷恩·葛斯林]]飾）很不情願地捲入這場對抗之中，因為他愛上被柯恩視為禁臠的優雅尤物葛莉絲·法拉戴（[[艾瑪·史東|艾瑪·史東]]飾）。

== 製作 ==
主要拍攝於2011年9月6日在[[洛杉磯|洛杉磯]]開始，取景地點包括洛杉磯縣從[[圣费尔南多谷|聖費爾南多谷]]北部至南部。

== 發行 ==
美國原訂2012年9月7日上映<ref name="InsideMovies1">{{cite web |last=Franich |first=Darren |url=http://insidemovies.ew.com/2012/07/25/warner-bros-pushes-gangster-squad |title='Gangster Squad: Warner Bros. pushing back release date |work=InsideMovies.EW.com |accessdate=2012-09-14}}</ref>，但前夕遭逢[[2012年奥罗拉枪击事件|奧羅拉槍擊事件]]，才延遲至2013年1月11日發行<ref name="WarnerBros1">{{cite web |url=http://www.warnerbros.com/movies/in-theaters-coming-soon/gangster-squad/075a4928-d88a-430d-a8ee-f623a0610423.html |title=GANGSTER SQUAD |work=WarnerBros.com |accessdate=2012-09-14}}</ref>，部分畫面重拍。
