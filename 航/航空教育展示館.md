'''航空教育展示館'''在[[台灣|台灣]][[高雄市|高雄市]][[岡山區|岡山區]]，是位於[[中華民國空軍軍官學校|中華民國空軍軍官學校]]校區內的[[博物館|博物館]]之一<ref>{{cite web |url = https://newtalk.tw/news/view/2017-08-09/94095|title=戰機眷村博物館 最酷岡山小旅行看這裡 |accessdate=2017-08-09 |author =新頭殼newtalk 丘秝榕  }}</ref>（另外還有[[空軍軍史館|空軍軍史館]]和軍機展示場）。於2015年9月19日開幕<ref>{{cite web |url = http://www.president.gov.tw/NEWS/19761|title=總統出席空軍官校航空教育展示館開幕典禮|accessdate=2015-09-19 |author =總統府新聞 }}</ref> <ref>{{cite web |url = https://n.yam.com/Article/20160213552919|title=空軍航空教育展示館　傳揚國軍航空史 |accessdate=2016-02-13 |author =軍聞社 周力行}}</ref>，並於2016年3月1日改為不需預約參觀<ref>{{cite web |url = http://news.ltn.com.tw/news/life/breakingnews/2021630 |title=亞洲首座懸吊國寶級戰機展示館 4月2日岡山開張 |accessdate=2017-03-30 |author =自由時報 蘇福男 }}</ref> <ref>{{cite web |url = http://news.ltn.com.tw/news/life/breakingnews/2024334 |title=航空教育展示館開張 4日前兒童免費參觀 |accessdate=2017-04-02 |author =自由時報 蘇福男 }}</ref>。主要展示品為退役軍機，亦有展示武器裝備與航空引擎。<ref>{{cite web |url = https://www.ydn.com.tw/News/246930 |title=慶祝814 航空教育館嘉年華會迎賓  |accessdate=2017-07-26 |author =青年日報 陳盈蒨 }}</ref> <ref>{{cite web |url = http://hk.on.cc/hk/bkn/cnt/lifestyle/20171010/bkn-20171010150128111-1010_00982_001.html|title=高雄：360度睇經典戰機 |accessdate=2017-10-10 |author =東網 }}</ref>
