'''航拍中国'''是中国中央电视台拍摄的一部全航拍纪录片。涉及23个省、5个自治区、4个直辖市和2个特别行政区。使用了16架载人直升机、57架无人机，总行程接近15万公里。本片第一季于2017年1月底播出了前四集，随后在同年3月播出完整版。<ref>[http://news.hainan.net/hainan/yaowen/datu/2017/02/01/3316766.shtml 央视《航拍中国》第一季第一集聚焦大美海南] 海南在线 2017-02-01 </ref>2016年11月30日，《航拍中国》宣传片登上了位于美国纽约时代广场的纳斯达克巨型屏幕，连续播放3天。
<ref>{{cite news|title=《航拍中国》：前所未有的视角看中国|url=http://media.people.com.cn/n1/2017/0301/c14677-29116092.html|accessdate=2017-03-18|agency=人民}}</ref>2019年2月18日，央视纪录频道和航拍中国纪录片的官方微博发布消息，称第二季即将播出。<ref>{{cite news|title=久等了！《航拍中国 》第二季正式官宣！|url=https://weibo.com/6979282670/Hhf5mrHO0?type=comment#_rnd1550487293640|accessdate=2019-02-18|agency=新浪微博.CCTV9航拍中国官方微博}}</ref>2月26日，《航拍中国》第二季发布会举行，会上宣布第二季于3月3日起在[[中央电视台综合频道|CCTV-1]]和[[中央电视台纪录频道|CCTV-9]]播出。<ref>{{cite web|url=http://tv.cctv.com/2019/02/26/VIDEF4SjUKdZDCtEdoRNuU9u190226.shtml |title=《新闻联播》20190226 |website=央视网 |date=2019-02-26 }}</ref>

==评价==

===正面===
豆瓣网上获得9.2分的高分。每集都有一条清晰的线路，可以被观众用作一个省份的旅游指南。<ref>[http://ent.newssc.org/system/20170316/000760105_2.htm 四川新闻网]</ref>有观众看完之后认为本纪录片对普及中国地理知识有作用，有地理老师甚至梳理出相关考点供[[高考|高考]]考生参考。<ref>{{cite web |url=https://www.zhihu.com/question/55191600 |title=如何评价央视纪录片《航拍中国》？ |website=知乎网 |date=2017-03-19 }}</ref>

《航拍中国》导演余乐在接受采访时表示，“自然如何决定人文”是摄制组在制作时特别关注的一个问题。他认为，“（空中视角）弥补大家很多认识上的空白处。对大家的知识构建来说，我们可能有很多摄影图片很多很漂亮的，但就像苏东坡那句诗，横看成岭侧成峰，每个摄影记者拍到的都是侧面。但是从天空的角度看，真的是最全面知道它是什么样。”<ref>{{cite web |url=https://www.cqcb.com/personage/2017-03-09/270137.html |title=《航拍中国》总导演余乐：不给观众上课，看到前所未见的中国 |website=澎湃新闻 |date=2017-03-09 }}</ref>

===负面===
对《航拍中国》的负面评价主要集中在第二季的制作上。有观众认为第二季飞行路线逻辑混乱，内容杂驳，主线不清。<ref>{{cite web |url=https://mp.weixin.qq.com/s?__biz=MzI4MjU1MjQ4Ng==&mid=2247490800&idx=1&sn=4bad81ca9cf6c19f9e46bace72580f6f&chksm=eb996d39dceee42f20af0852774715965ca695acf474a007f9bf6948ecb844c5fd638ba4d0c2&mpshare=1&scene=23&srcid=#rd |title=豆瓣评分8.7，看完第二集，我觉得高了 |author=传媒大咖 |date=2019-03-06 }}</ref>此外，根据哔哩哔哩弹幕实时反馈，部分观众认为片头动画切换效果太快，观看体验不舒适。
