'''航空应急频率'''（{{lang-en|'''aircraft emergency frequency'''}}，也称'''GUARD'''）是[[航空|航空]][[波段|波段]]上保留的一个用于{{tsl|en|Emergency||遇险}}飞机[[通信|通信]]的[[频率|频率]]。民用的该频率为121.5 [[赫兹|MHz]]，也称国际航空遇险信号（International Air Distress，'''IAD'''）或甚高频守听（[[VHF|VHF]] Guard）；而243.0 MHz则供军事用途，也称军用航空遇险信号（Military Air Distress，'''MAD'''）或特高频守听（[[UHF|UHF]] Guard）。早期的{{tsl|en|Emergency position-indicating radiobeacon station|应急定位发射机}}（ELT）使用守听频率发信，但更现代的应急定位发射机已改用406 MHz频率。

== 历史 ==
[[国际民用航空组织|国际民用航空组织]]（ICAO）协调[[ARINC|ARINC]]和[[国际电信联盟|国际电信联盟]]（ITU）选择了121.5 MHz频率。

== 监控 ==
[[File:CessnaARC-RT-359ATransponder04.jpg|缩略图]]无线电]]
民用语音频率121.5 MHz受到大多数空中事务单位监控，包括[[航空交通管制|空管]]塔台、{{tsl|en|Flight service station||FSS}}服务、国家空管中心、军事防空、其他飞行和紧急服务，以及许多商用飞机。

在[[英国|英国]]，121.5 MHz由位于{{tsl|en|London Terminal Control Centre|伦敦终端管制中心}}和{{tsl|en|Shanwick Oceanic Control|山威克海上管制}}的[[英國皇家空軍|英國皇家空軍]]遇险与转移小组（也称D&D）通过全国范围的天线网监控。如果飞机“消失”或“暂时无法确定方位”，取决于飞机的高度和位置，中心的工作人员可能使用[[三角測量|三角測量]]法确定对飞行员有益的确切位置。

== 使用 ==
[[File:AIR14_(15306814355).jpg|缩略图]]燃料箱]]
任何遇险或遇到紧急情况的飞机都可以使用两个守听频率，[[航空交通管制|空管]]也可以用它来警告飞机不要进入{{tsl|en|Restricted airspace|限制空域|限制}}或{{tsl|en|Prohibited airspace|禁飞空域}}。

在飞机被防空飞机拦截时，飞机可能在121.5 MHz被要求表明身份和意图并得到指示。

== 定位信标 ==
{{Listen|filename=Radio beacon of distress on 121,5 MHz.ogg|title=遇险无线电信标的调制|description=遇险信标在121.5 MHz和243 MHz的发信|format=[[Ogg|Ogg]]}}
早期的应急定位发射机在撞击后于121.5 MHz发射信标。而近年的应急定位发射机改在[[406_MHz|406 MHz]]上发信，而在121.5 MHz发射低功率信标供当地定位。卫星则收听信号并提醒当地人员进入紧急状态，信标使搜索和抢救人员能更快找到事故现场。工作在406 MHz的信标会进行编码，从而确定载具的国籍和迅速验证误告警。卫星对纯121.5 MHz版本的支持已在2009年初停止。<ref>{{cite web|url=http://www.sarsat.noaa.gov/emerbcns.html|title=Emergency Beacons|publisher=[[NOAA|NOAA]]}}</ref><ref>{{cite web|url=http://www.navcen.uscg.gov/?pageName=mtEpirb|title=EMERGENCY POSITION INDICATING RADIOBEACON (EPIRB)|publisher=[[U.S._Coast_Guard|U.S. Coast Guard]]}}</ref>
