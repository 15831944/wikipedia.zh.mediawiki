'''航宙軍軍官成為冒險者'''（'''{{lang|ja|航宙軍士官、冒険者になる}}'''）是由[[伊藤暖彥|伊藤暖彥]]所著的日本輕小說。日文文庫版由[[Fami通文庫|Fami通文庫]]出版，並由[[himesuz|himesuz]]擔任插畫與人物設計，[[2019年|2019年]]3月為止出版2卷。日文(線上)漫畫版由{{Tsl|ja|たくま朋正|たくま朋正}}編製<ref>{{Cite news|url=https://comic-walker.com/contents/detail/KDCW_AM19200711010000_68/|title=航宙軍士官、冒険者になる|newspaper=ComicWalker - 人気マンガが無料で読める！|accessdate=2019-02-17|language=ja}}</ref>。

== 概要 ==
從[[2017年|2017年]][[8月10日|8月10日]]起在「[[成為小說家吧|成為小說家吧]]」上連載<ref name="なろう">{{Citation|title=航宙軍士官、冒険者になる|url=https://ncode.syosetu.com/n3490ee/ |author=a_itoh（伊藤 暖彦） |publisher=小説家になろう|accessdate=2019-02-17|language=ja}}</ref>。[[2018年|2018年]][[11月30日|11月30日]]由[[Fami通文庫|Fami通文庫]]開始出版文庫版，文庫化後仍繼續在「[[成為小說家吧|成為小說家吧]]」連載。後續記述內容以文庫版為準，提及「[[成為小說家吧|成為小說家吧]]」版時會以Web版記述。

== 簡介 ==
帝國航宙<ref>航宙大抵是指「航空暨宇宙」之意，來自NASA（{{tsl|ja|アメリカ航空宇宙局}}），包含在重力圈大氣層內航空飛行與航行太空宇宙兩重意義內</ref>軍軍官[[#艾倫・格林多|艾倫・格林多]]所搭乘的軍艦在超空間航行中受到未知的攻擊，艾倫成了唯一的倖存者。<br />
航宙艦變成無法航行的狀態，於是艾倫乘坐逃生艙緊急降落到眼前的行星上。<br />
雖然他感到絕望，但是對於在降落的行星上，有繁榮著與艾倫的遺傳子系譜相通的人類而感到驚訝，再加上這個行星上的人竟然會使用「魔法」。<br />
艾倫與他共生的奈米機器〔[[#奈諾姆|奈諾姆]]〕，只好一邊運用科學技術來調查「劍與魔法的世界」，一邊度過求生的生活……<ref name="JPBook1" />

== 主要登場角色 ==
; <span id="艾倫・格林多">{{nihongo|艾倫・格林多|アラン・コリント|}}</span><ref>「格林多（コリント）」的語源來自「[[格林多前書|格林多前書]]（{{tsl|ja|コリントの信徒への手紙一}}）」；伊莉絲・康拉德准將的丈夫名字為「亞倫（アーロン）」，與艾倫發音有點接近</ref>
: 銀河帝國航宙軍宙兵隊的軍官。由於除了他以外的乘組員都已經死亡了，於是就任航宙艦伊莉絲的艦長<ref>原本是非正規船員的中尉軍官，接任艦長後升級為正規船員的准將將官，Web版還有提到從宙兵隊升到航宙艦隊，不過不知道有何差別</ref>，在那不久後緊急逃生到未知的行星上。<ref name="JPBook1" />
; <span id="伊莉絲・康拉德">{{nihongo|伊莉絲・康拉德|イーリス・コンラート|}}</span>
: 航宙艦伊莉絲・康拉德的AI<ref>艦名是取自伊莉絲・康拉德准將，艦上AI全名職稱是伊莉絲・康拉德名譽上尉。傳統上，艦內會收藏同名官兵的(部份)遺體</ref>。其虛擬人物外觀是戰死的伊莉絲・康拉德准將<ref>為了對抗蟲族（バグス）而戰死，死前為中校，死後二階級特進為准將</ref>生前的容貌。<ref name="JPBook1" />
; <span id="克萊莉婭・斯塔拜因">{{nihongo|克萊莉婭・斯塔拜因|クレリア・スターヴァイン|}}</span>
: 斯塔維克（スターヴェーク）王國王女。因為（王族）被謀反而離開了國家。擅長火魔法與劍術。<ref name="JPBook1" />
; <span id="艾爾娜・諾里安">{{nihongo|艾爾娜・諾里安|エルナ・ノリアン|}}</span>
: 年輕時被提拔為克萊莉婭近衛騎士的劍術和魔法好手。愛好是縫紉，其本領是能獨自一人製作出禮服。<ref name="JPBook1" />

== 用語、世界觀 ==

; <span id="奈諾姆">{{nihongo|奈諾姆|ナノム|NanoM}}</span>
: 所謂奈諾姆，只要是航宙軍士兵都會在體內共生・培養的，軍用奈米機器（Nanometer-scale Machine）的簡稱。
: 它是一種需用電子顯微鏡才能看見的極小型機器，單體上沒有什麼功能，最低限度也要形成一千億個以上的集合體才能發揮功能的系統。
: 具備AI機能、體內的監視器、醫療・修復功能、能夠完全存取來自五官的感覺、短距離通信機能等等，可以說是一種多功能系統。
: 無法從人類的常吃下肚的成分來製作，需要特殊的稀有金屬<ref>通常要吃下「稀有金屬錠」才能在人體內製作出來</ref>來製作。<ref name="JPBook1" />

; <span id="蟲族">{{nihongo|蟲族|バグス|BUGs}}</span>
: 外觀看起來像是昆蟲變大的智慧生命體，現在已經有各式各樣的類型被確認了<ref>其中一種是「ビートルタイプⅡ型」</ref>。
: 最基本的類型是，長的像八條腿的蟑螂身形稍微拉長而上半身向後仰起的類型。
: 身體被硬邦邦的體毛和外殼覆蓋著，光是這樣就讓人類充分動搖並感到恐懼，更加上極其骯髒、非常油膩膩地，在其上寄宿著數種的寄生蟲。
: 擁有遠超人類的戰鬥能力，赤手空拳與之戰鬥是完全敵不過的。
: 但是，在經歷千年的戰鬥中人類一直努力著改善其差距，現在已經成功提高了戰鬥力，只要是航宙軍航宙兵的話，就可以在空手戰鬥中進行對等的互戰。
: 雖然蟲族也擁有能够進行超空間航行的科技，但在科技方面，人類一直領先數個階段。
: 蟲族的可怕之處在於其壓倒性的數量。蟲族瞄準的殖民行星會被陸續出現的蟲族艦隊襲擊。
: 專家分析，人類和蟲族的生產率、種族數量的比為1比6。但是，一般認為被發現的蟲族殖民星上沒有特別的生產力，殖民星的數量和人類也沒有太大差別。
: 基於此，專家們認為在某處有存在著被認為是蟲族母星系的大規模繁殖・生產據點的說法比較有力，只要能够發現並殲滅它們，就能結束這場長期的戰爭。<ref name="JPBook1" />
