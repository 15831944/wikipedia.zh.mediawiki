'''虔誠者路易'''，即'''路易一世'''（{{lang-fr|Louis I<sup>er</sup>}}，{{bd|778年|4月16日|840年|6月20日}}），[[法蘭克王國|法蘭克王國]]的國王和皇帝（814年1月28日—840年6月20日在位），[[查理大帝|查理大帝]]的兒子與繼位者。

== 阿基坦國王 ==

781年，查理大帝平定[[阿基坦|阿基坦]]後，讓尚為年幼的路易擔任阿基坦的國王。

== 繼承神聖羅馬皇帝帝位 ==

由於他的兩個哥哥[[意大利的丕平|意大利的丕平]]和[[小查理|小查理]]都先於父親[[查理大帝|查理大帝]]去世。於是在813年9月，[[查理大帝|查理大帝]]於[[亞琛|亞琛]]加冕路易為皇帝，繼承帝位。但軟弱的路易沒有其父的巨大威望，也沒有足夠的軍事政治能力。

== 第一次分封國土 ==

817年，路易依照法蘭克的傳統，將帝國一分為三，分給他的三個兒子——
* 長子[[洛泰爾一世_(中法蘭克)|洛泰尔]]得到[[意大利|意大利]]，並且作為皇帝繼承人輔佐朝政。
* 次子[[阿基坦丕平|丕平一世]]得到[[阿基坦|阿基坦]]。
* 幼子[[日耳曼人路易|日耳曼人路易]]分到[[巴伐利亞|巴伐利亞]]。

這次分封引起各地封建主不滿。在[[亞琛|亞琛]]召開的一次民眾大會上，路易的堂兄弟瓦拉把國家的一切混亂都歸罪於路易，他公開指責道：「你就是罪魁禍首。」

== 第二次分封國土 ==

823年，他的第二任皇后[[巴伐利亞的朱迪絲|巴伐利亞的朱迪絲]]又為他生了一個兒子[[禿頭查理|禿頭查理]]，路易重分其帝國，以讓查理獲得[[分封|分封]]。后来，他的其他三個兒子感到不滿，發起了武裝[[叛亂|叛亂]]。教皇[[額我略四世|額我略四世]]也威脅皇帝，自己的權威必須要高過皇帝的權威。路易被兒子們[[罷黜|罷黜]]。在[[蘇瓦松|蘇瓦松]]的大教堂，皇帝解下自己的肩帶與佩劍，被迫在蘭斯大主教面前公開[[懺悔|懺悔]]自己[[瀆神|瀆神]]。一年之後，由於爭奪戰利品產生的不和，為對抗[[洛泰爾一世_(中法蘭克)|洛泰尔]]，[[日耳曼人路易|日耳曼人路易]]和[[阿基坦丕平|丕平一世]]恢復了虔誠者路易的皇位，禿頭查理也被釋放。840年6月20日，路易死於[[美因茨|美因茨]]附近。

==法蘭克王國的分裂==

路易在840年死後，他的兒子[[洛泰爾一世|洛泰爾一世]]成為唯一的統治者，但是843年他的三個兒子還是在《[[凡爾登條約|凡爾登條約]]》裡決定三分法蘭克王國。王國後來在855年的[[普呂姆分割|普呂姆分割]]、870年的《[[墨爾森條約|墨爾森條約]]》和880年的《[[利貝蒙條約|利貝蒙條約]]》中又再分裂成多個細小獨立國家，此後法蘭克再沒有被統一過。
