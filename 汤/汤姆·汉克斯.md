'''托马斯·杰弗里·汉克斯'''（{{lang-en|Thomas Jeffrey Hanks}}，{{bd|1956年|7月9日||catIdx=Hanks, Tom}}），暱稱'''湯姆·漢克'''（{{lang|en|Tom Hanks}}），是著名[[男演員|男演員]]、[[电影演员|电影演员]]及電視製作人，他是[[好萊塢|好萊塢]]具有廣泛影響力的影星，以演技精湛著称。曾参演多部不同类型电影，饰演角色包括《[[费城故事_(1993年)|费城故事]]》中的同性戀律師、《[[阿甘正傳|阿甘正傳]]》中憨直的男主角[[阿甘_(電影角色)|福雷斯特·甘]]、在《[[阿波罗13号_(电影)|阿波罗13号]]》中絕不言死的指挥官、在温馨喜剧《[[西雅图不眠夜|西雅图不眠夜]]》中的魅力男士、《[[雷霆救兵|雷霆救兵]]》中堅毅不拔的[[美國陸軍遊騎兵|美國陸軍遊騎兵]][[上尉|上尉]]約翰·米勒。他是歷史上继[[斯潘塞·特雷西|斯潘塞·特雷西]]之后的第2位連續兩屆获得[[奧斯卡金像獎|奧斯卡金像獎]]最佳男主角奖的演员。2016年獲頒美國最高的平民榮譽—[[總統自由勳章|總統自由勳章]]。

== 經歷 ==
湯姆·漢克出生在[[加州|加州]]的[[康科德_(加利福尼亞州)|康科德]]市，父親為[[英格蘭|英格蘭]]裔，母親為[[葡萄牙|葡萄牙]]裔，在[[奧克蘭|奧克蘭]]市的天際高中就读時加入戲劇社，大學在加州[[海沃德|海沃德]]的CHABOT學院就讀兩年後，轉入[[沙加緬度州立大学|沙加緬度州立大学]]戲劇系。后来他受导演邀请去了[[俄亥俄州|俄亥俄州]]的[[克里夫蘭|克里夫蘭]]，在那里他开始了表演之路。湯·漢斯是{{link-en|國家太空學會|National Space Society}}（National Space Society，簡稱NSS）中的一员，也是[[HBO|HBO]]电视连续短剧《[[从地球到月球_(电视短剧)|从地球到月球]]》的制作人，该短剧讲述了送人上月球的[[阿波羅計畫|阿波羅計畫]]。

雖然湯姆·漢克斯早期的作品绝大多數以喜劇為主—他第一次引起觀眾注意的是1980—1982年[[情景喜剧|情景喜剧]]《{{link-en|亲密伙伴|Bosom Buddies}}》（''Bosom Buddies''）—但是此后他放弃了喜剧开始转入比较严肃的戏剧。最近，他轮流交替在喜剧和严肃戏剧角色之间。

== 家庭 ==
湯·漢斯声称透過[[林肯|林肯]]的母親南西·漢克斯（Nancy Hanks）而与林肯有亲戚关系。

湯·漢斯在1988年同演员{{link-en|丽塔·威尔逊|Rita Wilson}}（Rita Wilson）结婚。他们是在拍攝《志愿者》時相识的。他们有2个孩子。在此之前的1978到1987，湯·漢斯与珊曼莎·路易斯（Samantha Lewes）曾有过一段婚姻。同样也生育了2个孩子（其中一个是演员[[柯林·漢克斯|柯林·漢克斯]]）。通过与威尔逊的接触，湯·漢斯加入了[[希腊正教|希腊正教]]。

== 荣誉 ==
[[File:Tom_Hanks_-_footprint.JPG|thumb]]
湯·漢斯因为《[[費城_(電影)|費城]]》（1993）、《[[阿甘正傳|阿甘正傳]]》（1994）曾接连2届赢得了[[奥斯卡最佳男主角奖|奥斯卡最佳男主角奖]]。他是最年轻的获得[[美國電影學會|美國電影學會]]终身成就奖（2002年6月12日，时年46岁）的演员。

2006年，汤姆·汉克斯被选为[[美国陆军|美国陆军突击队名人堂]]荣誉成员，他亦是第一位获此荣誉的好莱坞演员。<ref>http://news.xinhuanet.com/ent/2006-07/03/content_4785809.htm</ref>

2016年12月時任美國總統[[奧巴馬|奧巴馬]]在[[白宮|白宮]]為湯姆·漢克斯頒發了總統自由勳章。

== 其他 ==
湯·漢斯是美国[[民主党|民主党]]黨員，曾支持许多民主党[[政客|政客]]，包括[[希拉里·克林顿|希拉里·克林顿]]、[[黛安·范士丹|黛安·范士丹]]、[[戈爾|戈爾]]、[[约翰·克里|约翰·克里]]。湯·漢斯同样也是一个著名的[[环境主义|环境主义]]者，他驾驶两部[[混合动力|混合动力]][[汽车|汽车]]分别是丰田RAV4 EV第一批次的混合动力eBox。他还是[[美国自然保育协会|美国自然保育协会]]一员，他为了组织拍了一些在[[收音机|收音机]]、[[电视|电视]]、及其他公共场合均可看見的通告，甚至在组织於他家乡[[愛達荷州|愛達荷州]]的分支中作[[志願者|志願者]]。他支持同性婚姻。<ref>{{cite web |url=http://vids.myspace.com/index.cfm?fuseaction=vids.individual&videoid=33546997 |title=存档副本 |accessdate=2012-06-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20091004182056/http://vids.myspace.com/index.cfm?fuseaction=vids.individual&videoid=33546997 |archivedate=2009-10-04 }}</ref>

湯·漢斯是[[英超|英超]]球隊[[阿士東維拉|阿士東維拉]]的球迷。
