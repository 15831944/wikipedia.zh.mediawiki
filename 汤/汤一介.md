'''汤一介'''（{{bd|1927年|2月16日|2014年|9月9日}}）<ref name="学者春秋：汤一介 ">{{cite web |url=http://www.guoxue.com/xzcq/ddxz/tangyijie/tyj.htm|title=《学者春秋：汤一介》|accessdate=|work=|publisher=国学网|language=zh-hans}}</ref>，[[天津|天津]]人 ，祖籍[[湖北|湖北]][[黄梅|黄梅]]，[[北京大学|北京大学]][[哲学|哲学]]系教授，[[国学|国学]]大师。<ref name="学者春秋：汤一介 ">{{cite web |url=http://www.guoxue.com/xzcq/ddxz/tangyijie/tyj.htm|title=《学者春秋：汤一介》|accessdate=|work=|publisher=国学网|language=zh-hans}}</ref><ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>。

== 生平 ==
1927年，汤一介出生于一个书香门第，他的祖父[[汤霖|汤霖]]是[[清朝|清朝]][[光绪|光绪]]十六年（1891年）的[[进士|进士]]，父亲[[汤用彤|汤用彤]]是近现代[[中国|中国]][[哲学|哲学]]界融会中西、并接通[[中文|中文]]和[[梵文|梵文]]的学术大师之一，任职[[北京大学|北京大学]]副校长。<ref name="学者春秋：汤一介 ">{{cite web |url=http://www.guoxue.com/xzcq/ddxz/tangyijie/tyj.htm|title=《学者春秋：汤一介》|accessdate=|work=|publisher=国学网|language=zh-hans}}</ref><ref name="84岁国学大家汤一介唯一传记出版 ">{{cite web |url=http://book.sina.com.cn/news/b/2011-11-25/1119293105.shtml|title=《84岁国学大家汤一介唯一传记出版》|accessdate=2011年11月25日|work=|publisher=新浪网|language= zh-hans}}</ref><ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

[[高中|高中]]时，汤一介对[[西方|西方]][[文学|文学]]发生了浓厚兴趣，而且试着写作了一些[[散文|散文]]、[[杂文|杂文]]。<ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

1946年，因为落榜，<ref name="学者春秋：汤一介 ">{{cite web |url=http://www.guoxue.com/xzcq/ddxz/tangyijie/tyj.htm|title=《学者春秋：汤一介》|accessdate=|work=|publisher=国学网|language=zh-hans}}</ref>汤一介进入[[北京大学|北京大学]]先修班。在此期间，他阅读了[[西方哲学|西方哲学]]和[[美学|美学]]方面的作品，写了《论善》、《论死》、《论人为什么活着》几篇文章，表达了自己对人生的思考。<ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

1947年，汤一介考取了[[北京大学|北京大学]][[哲学|哲学]]系，<ref name="学者春秋：汤一介 ">{{cite web |url=http://www.guoxue.com/xzcq/ddxz/tangyijie/tyj.htm|title=《学者春秋：汤一介》|accessdate=|work=|publisher=国学网|language=zh-hans}}</ref>于1951年毕业。<ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

1949年，[[北京大学|北京大学]]成立了[[新民主主义|新民主主义]]青年团，[[文学|文学]]院有一个团总支，汤一介和他未来的妻子[[乐黛云|乐黛云]]就是这这里互相认识的。<ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>同年，汤一介加入[[中国共产党|中国共产党]]。<ref>{{cite news|title=87岁北大教授汤一介去世：千年儒学的当代布道者|url=http://culture.ifeng.com/a/20140910/41914724_0.shtml|accessdate=2014-09-10|newspaper=凤凰网文化|date=2014-09-10}}</ref>

1952年，汤一介和[[乐黛云|乐黛云]]结婚。<ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

1957年后，由于各种[[政治运动|政治运动]]和[[学术|学术]]批判风起云涌，汤一介也被投入其中。1957年初，[[北京大学|北京大学]]召开了“[[中国|中国]][[哲学|哲学]]史座谈会”，他写了一篇文章《谈谈哲学遗产的继承问题》，这是汤一介在中共建政后发表的第一篇论文。

1958年后，汤一介受到‘[[反右倾运动|反右倾运动]]’的冲击和打压。<ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

1966年，[[文革|文革]]开始，汤一介被终止讲课资格，下放[[干校|干校]]。<ref name="学者春秋：汤一介 ">{{cite web |url=http://www.guoxue.com/xzcq/ddxz/tangyijie/tyj.htm|title=《学者春秋：汤一介》|accessdate=|work=|publisher=国学网|language=zh-hans}}</ref>

1973年，汤一介加入“北京大学、清华大学大批判组”（即“梁效”），曾任材料组组长。<ref name="“梁效”的成立与终结">{{cite web |url=http://view.news.qq.com/a/20140810/006237.htm|title=《“梁效”的成立与终结》|accessdate=|author=范达人|publisher=炎黄春秋|language=zh-hans}}</ref><ref name="“梁效”成员最终结局">{{cite web |url=http://news.xinhuanet.com/mrdx/2013-01/25/c_132127346.htm|title=《“梁效”成员最终结局》|accessdate=|work=|publisher=新华网|language=zh-hans}}</ref>四人帮垮台后他又被隔离审查，直至1978年才被解放。<ref name="汤一介和\"梁效\"">{{cite web |url=http://www.gmw.cn/01ds/2006-02/08/content_370825.htm|title=《汤一介和"梁效"》|accessdate=|work=|publisher=光明网|language=zh-hans}}</ref>

1980年，53岁的汤一介恢复了讲课资格，教授《[[魏晋|魏晋]][[玄学|玄学]]与[[佛教|佛教]]、[[道教|道教]]》。<ref name="学者春秋：汤一介 ">{{cite web |url=http://www.guoxue.com/xzcq/ddxz/tangyijie/tyj.htm|title=《学者春秋：汤一介》|accessdate=|work=|publisher=国学网|language=zh-hans}}</ref><ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

1981年，汤一介发表了《论[[中国|中国]][[传统|传统]][[哲学|哲学]]范畴体系诸问题》，率先打破“[[唯物主义|唯物主义]]”和“[[唯心主义|唯心主义]]”互相对峙的陈旧思想，提出认识发展史来看待[[哲学|哲学]]问题。<ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

1983年，汤一介任[[美国|美国]][[哈佛大学|哈佛大学]]访问[[学者|学者]]。汤一介参加了在[[加拿大|加拿大]][[蒙特利尔|蒙特利尔]]召开的第17届[[世界|世界]][[哲学|哲学]]大会。<ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

1984年，[[中国文化书院|中国文化书院]]成立，是国内第一家[[民间|民间]][[学术|学术]]团体。书院获得了[[冯友兰|冯友兰]]、[[张岱年|张岱年]]、[[朱伯崑|朱伯崑]]、[[李泽厚|李泽厚]]、[[庞朴|庞朴]]等[[哲学家|哲学家]]的倾力支持。<ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

2002年10月，汤一介向[[北京大学|北京大学]]学校领导提出编纂《[[儒藏|儒藏]]》的构想，得到了了[[张岱年|张岱年]]、[[季羡林|季羡林]]的鼓励和支持，<ref name="84岁国学大家汤一介唯一传记出版 ">{{cite web |url=http://book.sina.com.cn/news/b/2011-11-25/1119293105.shtml|title=《84岁国学大家汤一介唯一传记出版》|accessdate=2011年11月25日|work=|publisher=新浪网|language= zh-hans}}</ref>2003年国家[[教育部|教育部]]批准立项，[[北京大学|北京大学]]整合文科院系的力量并联合有关高等院校和[[学术|学术]]机构，正式启动了《[[儒藏|儒藏]]》工程，这也是[[教育部|教育部]]迄今为止最为重大的[[哲学|哲学]][[社会|社会]]科学研究攻关项目。汤一介任《[[儒藏|儒藏]]》首席专家，另外有[[日本|日本]]、[[韩国|韩国]]和[[越南|越南]]等国家的学者一共400多人。<ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

2010年6月29日，[[北京大学|北京大学]][[儒学|儒学]]研究院成立，汤一介出任院长。<ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

2014年9月9日晚8时56分，汤一介于北京逝世<ref>{{cite book | url=http://cpc.people.com.cn/n/2014/0910/c83084-25633522.html | title=87岁国学大师汤一介去世 | publisher=中国共产党新闻网 | accessdate=2014-09-10}}</ref>。

== 学术思想 ==
汤一介性格儒雅平和，内敛沉静，熟知[[中国|中国]][[传统文化|传统文化]]的精髓，又对[[西方文学|西方文学]]、[[哲学|哲学]]以及[[西方|西方]][[古典音乐|古典音乐]]有极大的兴趣。他曾自述“[[俄国|俄国]]的[[托尔斯泰|托尔斯泰]]、[[法国|法国]]的[[罗曼·罗兰|罗曼·罗兰]]，还有[[法国|法国]]的[[纪德|纪德]]（1947年[[诺贝尔文学奖|诺贝尔文学奖]]得主）对我的影响都非常大。我也非常喜欢听[[西方|西方]][[古典音乐|古典音乐]]。”并且有着每天晚上睡前听一小时[[西方|西方]][[古典音乐|古典音乐]]的习惯。<ref name=" 国学大家汤一介：“我不是哲学家”">{{cite web |url=http://news.xinhuanet.com/edu/2011-03/27/c_121236072.htm|title=《国学大家汤一介：“我不是哲学家”》|accessdate=2011年3月27日|work=|publisher=新华网|language=zh-hans}}</ref>

== 轶事 ==
汤一介曾多次公开支持学生民主运动，包括在六四天安门时期，与知识界和文化界33人联署致函中国全国人大常委会及中共中央，提出“释放[[魏京生|魏京生]]等政治犯”等建议；1994年，中国维权运动风起云涌，40多名学者和作家联名签署“纪念联合国宽容年”公开信，汤一介也参与其中。[[习近平|习近平]]在[[六四事件|六四事件]]25周年来临的敏感时刻，在北大高调接见汤一介，当时曾引起境外媒体关注。
