'''汤涛'''{{BD|1963年||}}是一名计算数学家，出生于[[中华人民共和国|中国]][[安徽省|安徽省]][[舒城县|舒城县]]，1984年毕业于[[北京大学|北京大学]]数学系，1989年获得[[英国|英国]][[利兹大学|利兹大学]]博士学位，2017年当选[[中国科学院|中国科学院]][[中国科学院院士|院士]]。现任[[北京师范大学-香港浸会大学联合国际学院|北京师范大学-香港浸会大学联合国际学院]]校长<ref>{{Cite web|title=校长 - UIC Official Website|url=https://uic.edu.hk/cn/about-us/organisation-and-management-structure/president|work=uic.edu.hk|accessdate=2019-02-13|language=cn|first=MPRO|last=Manager}}</ref>。

汤涛于1990年至1998年间，在[[加拿大|加拿大]][[西门菲莎大学|西门菲莎大学]]执教，并且取得终身教职；1998年加入[[香港浸会大学|香港浸会大学]]，曾经担任数学系主任、研究生院院长、理学院院长等职务。2015年加入[[南方科技大学|南方科技大学]]，任[[南方科技大学|南方科技大学]]副校长、研究生院院长、教务长、数学系讲座教授<ref>{{cite web |title=南方都市报：南科大副校长汤涛当选中科院院士，30年来他只做了一件事 |url=https://m.mp.oeeee.com/a/BAAFRD00002017113059836.html}}</ref>。

汤涛的研究领域为计算数学，在高精度和自适应计算方法研究领域做出了重要学术成就，因此荣获冯康科学计算奖，教育部自然科学一等奖，国家自然科学二等奖<ref>{{cite web |title=中工网：南科大汤涛教授当选中科院院士 |url=http://character.workercn.cn/364/201711/29/171129130713744.shtml}}</ref>。他是教育部长江讲座教授、国家千人计划学者，曾担任香港数学会理事长等重要学术职务。鉴于其学术影响力，他获选美国数学学会会士、美国工业与应用数学学会会士，并于2018年获邀成为国际数学家大会报告人<ref>{{cite web |title=科学网：北大八位教师校友在国际数学家大会作邀请报告 |url=http://news.sciencenet.cn/htmlnews/2018/8/416402.shtm}}</ref>。

汤涛于2010年合作创办数学科普刊物《数学文化》<ref>{{cite web | title=《数学文化》期刊 | url=http://www.global-sci.org/mc.html}}</ref>。
