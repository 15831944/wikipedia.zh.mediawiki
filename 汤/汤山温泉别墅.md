'''汤山温泉别墅'''又被称为'''蒋介石温泉别墅'''，位于[[中国|中国]][[江苏省|江苏省]][[南京市|南京市]][[江宁区|江宁区]][[汤山镇|汤山镇]]汤水河畔温泉路3号，始建于1920年，关于其最初的房主有[[陶保晋|陶保晋]]、[[张静江|张静江]]两种说法，以前一说法为主<ref>{{cite web |url=http://www.ahda.gov.cn/DocHtml/1/2009/3/16/151673661585.html|title=《档案解开温泉别墅身份之谜》|accessdate=2012-09-04|work=|publisher=安徽档案信息网网}}</ref>。1946年由南京市政府接收，并于当年秋至1947年3月重修，改为[[蒋中正|蒋中正]]、[[宋美龄|宋美龄]]专用的温泉别墅。1949后成为部队接待场所，现已对外开放。该建筑包括两层楼房5间和平房10间，楼房面朝东南，大青石墙面，悬山顶上覆以棕色筒形琉璃瓦。顶层中部为会客厅，东侧为卧室，西侧为书房和餐厅等，底层一半入地，沿走廊自西向东依次为随员池、卫士池、美龄池和总统池<ref>{{cite web|url=http://njdfz.nje.cn/HTMLNEWS/1182/200974175804.htm|title=《汤山温泉别墅》|accessdate=2012-09-04|work=|publisher=南京地方志网|deadurl=yes|archiveurl=https://web.archive.org/web/20121102030801/http://njdfz.nje.cn/HTMLNEWS/1182/200974175804.htm|archivedate=2012-11-02}}</ref>，走廊尽头设有暗道，通往当时的某处宪兵队驻扎地（现状已被封堵）<ref>{{cite web |url=http://house.focus.cn/news/2010-05-21/939811.html|title=《蒋介石南京汤山温泉别墅暗藏的两大谜团》|accessdate=2012-09-04|work=|publisher=搜狐}}</ref>。
