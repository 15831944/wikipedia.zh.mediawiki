'''汤兰兰事件'''<ref name="fhw1"/><ref name="b2"/>或称'''寻找汤兰兰事件'''<ref>{{Cite web |url=http://www.sohu.com/a/220481410_499808 |title=“寻找汤兰兰”事件 网友质疑原发媒体 |accessdate=2019-03-15 |author= |date=2018-02-02 |work= |publisher=搜狐网|language=Zh-hans }}</ref>是2018年初，在中国大陆发生的一起舆论争议事件，由[[澎湃新闻|澎湃新闻]]和《[[新京报|新京报]]》记者所做2008年[[黑龙江省|黑龙江省]]未成年人被[[性侵|性侵]]案的报道所引发。报道中，记者公开当年未成年受害人的身份信息，试图“寻找”被警方保护、更换身份的受害人<ref name="a1"/><ref name="a2"/>。记者此举，引起网友反感，反被[[人肉搜索|人肉搜索]]。事件所涉性侵案本身、[[新闻伦理|新闻伦理]]、网友反应均引发争议。

== 背景 ==

强奸案涉及[[隐私|个人隐私]]，根据《[[:s:中华人民共和国刑事诉讼法|中华人民共和国刑事诉讼法]]》第一百八十八条，可以不公开审理。中国司法对强奸案受害人保护无制度化规定。

== 案件 ==

'''汤兰兰案'''是[[黑龙江省|黑龙江省]]女孩汤兰兰被家人和村民强奸的案件。2008年10月3日，不满14周岁的汤兰兰（化名）向黑龙江省[[五大连池市|五大连池市]][[龙镇_(五大连池市)|龙镇]]警方写信称她从7岁开始被父亲、爷爷、叔叔、姑父等亲属及村民强奸。10月27日，汤兰兰在寄宿房东兼干妈陪同下报案，检查结果表明她有过性行为。10月底，警方先后抓获[[犯罪嫌疑人|犯罪嫌疑人]]15人。其中有汤父汤继海、汤母万秀玲及爷爷汤瑞景。12月13日，汤瑞景在看守所内咳血，经抢救无效死亡，被鉴定死于肺癌。{{r|a}}

2009年9月22日，黑河市中级人民法院一审开庭，2010年10月20日，以[[强奸罪|强奸罪]]、[[强迫卖淫罪|强迫卖淫罪]]、[[嫖宿幼女罪|嫖宿幼女罪]]判处1名被告无期徒刑，10名被告有期徒刑5至15年不等。11名被告向[[黑龙江省高级人民法院|黑龙江省高级人民法院]]（简称“黑龙江高法”）上诉，2012年10月26日，黑龙江高法驳回上诉{{r|a}}。

2014年，黑龙江高法驳回被告之一刘长海的申诉。刘长海再向[[黑龙江省人民检察院|黑龙江省人民检察院]]的申诉，于2016年被驳回。再向[[最高人民检察院|最高人民检察院]]的申诉，于2017年被驳回<ref name="aa1"/>。

被告汤继海、刘万友、陈春付、于东军四人以被诬陷、被[[刑讯逼供|刑讯逼供]]等为由向黑龙江高法申诉。2018年2月8日，黑龙江高法决定立案并组成合议庭复查。同案被告万秀玲提出书面申诉，纪广才、梁利权、王占军、李宝才、徐俊生提出口头申诉。7月27日，黑龙江高法公开开庭宣布驳回申诉<ref name="aa1">{{Cite news|url=http://www.xinhuanet.com/legal/2018-07/27/c_1123185946.htm|title=黑龙江省高院驳回汤继海等人的申诉|author=|date=2018-07-27|work=新华社|accessdate=}}</ref>。同日，[[中国中央电视台新闻频道|中国中央电视台新闻频道]]发布报道，首次采访汤兰兰，汤兰兰表示11人无一冤枉。<ref name=a>{{Cite news|url=http://news.cctv.com/2018/07/27/ARTISrIwgEFSHSnBarBb0xo7180727.shtml|title=【独家调查】九问汤兰兰案！|author=央视记者 吴闯 丹霓 程琴 文杰 张赛 郝亮 亚峰|date=2018-07-27|work=央视网新闻|accessdate=}}</ref>

== 新闻争议 ==

2018年1月30日，[[澎湃新闻|澎湃新闻]]发布报道《寻找汤兰兰：少女称遭亲友性侵，11人入狱多年其人“失联”》，报道开头即指汤兰兰“把全家人送进了监狱”，称案件疑点颇多，证据不足，需重审，呼吁寻找汤兰兰，同时公布了案件被告之一汤母万秀玲提供的汤兰兰新的身份信息<ref>{{Cite web|url=https://www.thepaper.cn/newsDetail_forward_1971755|archiveurl=https://web.archive.org/web/20180130173956/https://www.thepaper.cn/newsDetail_forward_1971755|archivedate=2018-01-30|title=寻找汤兰兰：少女称遭亲友性侵，11人入狱多年其人“失联”|accessdate=|author=|date=2018-01-30|publisher=澎湃新闻}}</ref>。次日，《[[新京报|新京报]]》发布评论《被全家“性侵”的女孩，不能就这么“失联”着》<ref>{{Cite news|url=http://www.bjnews.com.cn/opinion/2018/01/31/474713.html|title=被全家“性侵”的女孩，不能就这么“失联”着 {{!}}新京报快评|author=佘宗明|date=2018-01-31|work=新京报|accessdate=}}</ref>。

报道发出后，既引发中国大陆舆论争议。媒体“寻找汤兰兰”替代案件本身成为舆论关注的焦点。在[[新浪微博|微博]]上，澎湃新闻和《新京报》遭到网友大规模的批评<ref name="b1"/>，报道记者亦被网友[[人肉搜索|人肉搜索]]<ref name="b1"/><ref>{{Cite web |url=https://new.qq.com/omn/20180210/20180210G05ELL.html |title=汤兰兰事件：如何在信息中寻找真相？|accessdate=2019-03-15 |author= |date=2018-02-10 |work= |publisher=腾讯网|language=Zh-hans }}</ref>。有评论指澎湃新闻30日的报道已触及新闻伦理和法律边界<ref name="a2">{{Cite web |url=http://news.ifeng.com/a/20180205/55780819_0.shtml |title=所有人都搞偏方向！汤兰兰案最详细调查在此|accessdate=2019-03-15 |author=阚大神 |date=2018-02-05 |work=责任编辑：雒效文 PN066 |publisher=凤凰网|language=Zh-hans }}</ref>。此前《[[南方周末|南方周末]]》记者王瑞峰{{efn|有称<ref name="a2"/>《[[南方周末|南方周末]]》记者王瑞峰是[[于欢案|于欢案]]的最早的报道者。案件经报导引发舆论关注后，最终于欢在二审中获得减刑。}}曾采访过案件，在报道公布前被撤稿。1月19日，记者在个人社交帐号发表后被删除，未引起公众关注<ref name="b1">{{Cite web |url=http://www.chinadevelopmentbrief.org.cn/news-20910.html |title=“汤兰兰案”报道：容易的事和困难的事|accessdate=2019-03-15 |author=方可成 |date=2018-02-02 |work= |publisher=发展简报网站|language=Zh-hans }}</ref>。

虽然目前，中国司法对强奸案受害人保护无制度化规定。但汤兰兰案中，有认为出于对受害人的保护，警方为汤兰兰迁移户口、提供新身份是符合常识的<ref>{{Cite web |url=http://www.mnw.cn/news/shehui/1934487.html |title=汤兰兰事件真相：汤兰兰强奸案始末回放 被判刑人员名单|accessdate=2019-03-15 |author= |date=2018-02-01 |work=责任编辑：郑莉莉 |publisher=闽南网|language=Zh-hans }}</ref><ref name="a2"/>。对所谓“失联”，[[北京市人民检察院列表|北京市人民检察院]]的两位人士撰文指出<ref name="fhw1">{{Cite web |url=http://news.ifeng.com/a/20180203/55727788_0.shtml |title=汤兰兰事件中，网友最关心的问题都有权威解答了|accessdate=2019-03-15 |author=张洋、赵鹏 |date=2018-02-03 |work= |publisher=凤凰网，来源：环球时报|language=Zh-hans }}</ref>，事件中汤兰兰“从来没有跟媒体联系过，谈不上失联。并且汤兰兰也没有义务与媒体联系。”“让性侵案件的被害人反复回忆被害经历本身就是对人的再次伤害，媒体并无这个权利。”故有评论者，将澎湃新闻“寻找汤兰兰”定义为“[[人肉搜索|人肉搜索]]”<ref name="b2">{{Cite web |url=https://new.qq.com/omn/20180201/20180201G110XD.html |title=澎湃新闻及汤兰兰事件详细始末|accessdate=2019-03-15 |author= |date=2018-02-01 |work= |publisher=腾讯网|language=Zh-hans }}</ref>。另有评论指，在现实环境下，澎湃新闻的报道“追问汤兰兰下落是容易的，质疑公检法是困难的”，而对记者进行人肉搜索“不仅找错了靶子，而且用错了方法。”<ref name="b1"/>

2月2日，[[江苏新闻广播|江苏新闻广播]]发布文章《被父亲爷爷等亲友性侵7年，受害人汤兰兰不能“消失”吗？》，认为澎湃新闻“张贴出受害人的户籍信息，帮助刑满释放人员寻找受害人”<ref name="a1">{{cite news|title=被父亲爷爷等性侵7年 受害人汤兰兰不能“消失”?|author=泊南|url=https://www.weibo.com/ttarticle/p/show?id=2309404202926670222265|accessdate=2019-01-09|agency=江苏新闻广播|publisher=|date=2018-02-02}}</ref>。2月3日<ref>{{Cite web |url=https://gongyi.ifeng.com/a/20180204/44870170_0.shtml|title=人民日报评汤兰兰案:揪出真相不等于让受害人出列|accessdate=2019-03-15 |author= |date=2018-02-04 |work=责任编辑：赵歆 PP019 |publisher=凤凰网|language=Zh-hans }}</ref>，《[[人民日报|人民日报]]》的微博帐号发表评论“公众渴望真相，但同样相信：媒体监督亦有法治规范与伦理边界。尤其是对一桩烈度极大的未成年人案而言，知情权本就让位于隐私权，揪出真相，绝不等于让受害人出列。以持中的姿态谋求法治，更有助于守护正义。”
