'''托马斯·威廉·“汤米”·海因索恩'''（{{lang-en|'''Thomas William "Tommy" Heinsohn'''}}，{{bd|1934年|8月26日}}），美國前職業籃球運動員和教練。

== 球員生涯 ==
1956年，海因索恩被[[波士頓塞爾提克|波士頓塞爾提克]]以[[地域指名|地域指名]]的方式選中。第一年，他獲得[[NBA最佳新秀|NBA最佳新秀]]。之後，他也成為塞爾提克主力前鋒之一，並奪下8次冠軍。在NBA歷史上，只有隊友[[比爾·羅素|比爾·羅素]]和[[山姆·瓊斯|山姆·瓊斯]]的冠軍次數比他多。而1965年，他選擇退休。

== 教練生涯 ==
他在1969年以教練身分重回塞爾提克，從1969年到1978年。其中他拿下2座[[NBA總冠軍|NBA總冠軍]]和一次[[NBA最佳教練|NBA最佳教練]]。

== 廣播 ==
當完教練後，現在海因索恩以播報員的身份當工作。

== 榮譽 ==
他在1986年被選入篮球名人堂
