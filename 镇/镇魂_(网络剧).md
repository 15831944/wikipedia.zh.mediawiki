《'''镇魂'''》，改编自[[Priest_(作家)|Priest]][[镇魂_(小说)|同名小说]]的[[网剧|网剧]]，2018年6、7月在[[优酷视频|优酷视频]]播出后，迅速爆红，成为中国大陆2018年夏天现象级地影视作品<ref name="a1"/>。剧集粉丝亦以“镇魂女（男）孩”知名。剧集爆红后，即在播出完结不久的8月2日被下架，原因不详<ref name="a2"/>。至8月底，或称有地方台复播此剧<ref name="a7"/>。9月下旬，或称优酷视频复播其中两集，不久迅速屏蔽<ref name="b1"/>。11月10日，经过删减的剧集于优酷重新上架<ref name="a8"/>。

== 制作、播出与下架 ==

=== 制作、播出 ===
该剧由时悦影视出品，浙江晟喜华视、泰洋川禾文化传媒、幻秘文化传播联合出品<ref>{{Cite web |url=http://ent.163.com/17/0417/08/CI7B2IQ700038793.html |title=奇幻剧《镇魂》上海开机 首创中式软奇幻|accessdate=2017-11-14 |author= |date=2017-04-17 |work=李彤_NK4455 |publisher=网易娱乐|language=Zh-hans }}</ref>，[[周远舟|周远舟]]执导，[[白宇|白宇]]、[[朱一龙|朱一龙]]等主演<ref>{{Cite web |url=http://ent.163.com/17/0323/10/CG72NR0400038793.html |title=《镇魂》男主曝光 黄金班底造年度"镇"网之作|accessdate=2017-11-14 |author= |date=2017-03-23 |work=李彤_NK4455 |publisher=网易娱乐|language=Zh-hans }}</ref>。2017年4月15日，在上海开机，同年7月13日杀青<ref>{{Cite web |url=http://ent.qianlong.com/2017/0718/1865166.shtml |title=《镇魂》杀青曝剧照 白宇朱一龙黄金兄弟飙演技|accessdate=2017-11-14 |author= |date=2017-07-18 |work=高骞（QN037） |publisher=网易娱乐|language=Zh-hans }}</ref>。2018年1月15日，公布首支片花<ref>{{Cite web |url=http://www.yule.com.cn/html/201801/37403.html |title=《镇魂》首曝片花 白宇“洒脱不羁”挑战全新角色|accessdate=2018-02-23 |author= |date=2018-01-16 |work= |publisher=中国娱乐网|language=Zh-hans }}</ref>，3月26日公布第二支片花。6月13日，于网络平台[[优酷视频|优酷视频]]<ref>{{Cite web |url=http://ent.sina.com.cn/v/m/2017-10-27/doc-ifynhhay5912777.shtml |title=一大波新剧来袭!杨洋范冰冰赵丽颖陈乔恩蓄势待发|accessdate=2017-11-23 |author= |date=2017-10-27 |work=得得 |publisher=新浪娱乐|language=Zh-hans }}</ref>播出。

=== 镇魂女孩（男孩） ===

剧集的[[爱好者|粉丝]]被称为镇魂女孩（男孩<ref>{{Cite web |url=http://baijiahao.baidu.com/s?id=1604495196416146499&wfr=spider&for=pc |title=《镇魂》热播，网上出现镇魂女孩，镇魂男孩之说，这是什么梗？|accessdate=2018-08-20 |author= |date=2018-06-28|work= |publisher=|language=Zh-hans }}</ref>），亦称镇魂女鬼<ref name="a3"/>，或说因善于发掘两位主演甚至演员好友的“黑历史”而由镇魂女孩升级为镇魂女鬼<ref>{{Cite web |url=http://www.sohu.com/a/238930188_114787 |title=镇魂/这么尴尬的剧，为什么那么多人在追？|accessdate=2018-08-20 |author= |date=2018-07-02|work= |publisher=搜狐网|language=Zh-hans }}</ref>。剧集和主演的爆红，与镇魂女孩群体性的[[二次創作|二次創作]]和推广息息相关。以剧集衍生的[[表情包|表情包]]为例，在播出的6月间，即已被广泛使用。甚至被[[人民网|人民网]]、[[中国气象局|中国气象局]]、[[中央气象台|中央气象台]]等机构的官方[[新浪微博|微博]]使用。当微博网友评论“气象官博突然集体嗑镇魂”时，中央气象台官方微博则回复“是自己人遍布全行业的节奏”<ref>{{Cite web |url=http://lady.163.com/18/0628/18/DLDIQNSO00267VA9.html |title=一朋友圈都是“镇魂女孩”究竟什么情况？|accessdate=2018-08-20 |author= |date=|work= |publisher=搜狐网|language=Zh-hans }}</ref>。而有评论在剧集播出的初期，即已指出镇魂女孩“无论是转发和自制表情包、产出视频、剧改漫画、精修截图的节奏，还是发现朱一龙白宇在演戏这条路上的兢兢业业，镇魂女孩都整齐划一，盘活了各种宣传形式和手段，以原著+影视+演员三个元素为基础的复合宣传水准相当高，给了巍澜[[配對_(同人)|CP]]出圈的机会，也让路人知道了两位优秀的演员朱一龙和白宇。”<ref>{{Cite web |url=http://www.sohu.com/a/238242925_100069859 |title=为什么给镇魂女孩一个支点，她们可以撬起整个微博？|accessdate=2018-08-20 |author= |date=2018-06-28|work= |publisher=搜狐网|language=Zh-hans }}</ref>

镇魂女孩亦获得优酷的官方认可，优酷于7月20日<ref>{{Cite web |url=https://baijiahao.baidu.com/s?id=1606563706386211275&wfr=spider&for=pc |title=镇魂女孩C位出道！导师朱一龙深夜发文：请你们一直走花路！|accessdate=2018-08-20 |author= |date=2018-07-21|work= |publisher=|language=Zh-hans }}</ref>投放的户外广告中，打出“镇魂女孩”“C位出道”的口号<ref name="a3">{{Cite web |url=https://baijiahao.baidu.com/s?id=1606518277914768479&wfr=spider&for=pc |title=“镇魂女孩”C位出道？全国人都知道？你们是不是每天都开心？|accessdate=2018-08-20 |author= |date=2018-07-20|work= |publisher=|language=Zh-hans }}</ref>。

=== 剧集下架、地方台播出 ===

全剧在7月25日<ref name="a8"/>播出完结，即于8月2日被下架，优酷视频和制作方并未公布原因。有评论认为，下架原因并非一般认为的“宣傳封建迷信、過度渲染社會陰暗面、部分畫面血腥暴力。”，而是原著的[[耽美|耽美]]属性。虽然剧集已将两位主角之间的[[同性恋|同性恋情]]改为兄弟情。而此前已有[[APP|APP]]下架原著的有声小说，视为剧集下架的前兆<ref name="a2">{{Cite web |url=http://www.sohu.com/a/244984591_509883 |title=《镇魂》或永久下线：“兄弟情”倒下的真实原因到底是……|accessdate=2018-08-14 |author= |date=2018-08-03|work= |publisher=搜狐网|language=Zh-hans }}</ref>。实际上，自2016年初，耽美网剧《[[上癮_(網路劇)|上瘾]]》在播出过程中被下架，耽美题材已成为中国大陆影视的禁区。有指《上瘾》下架之后，出演耽美或改编作品的演员多为非表演院校出身、且知名度较低的演员。《镇魂》亦是近两年来第一次由表演院校出身的演员出演的耽美题材改编作品<ref name="a4"/>。

除剧集下架外，有指包括[[花絮|花絮]]在内的相关视频被下架、禁播<ref name="a5">{{Cite web |url=http://www.redquan.com/hot/2018081818927.html |title=镇魂回来了是真的吗 官方宣布复播视频逐渐解禁|accessdate=2018-08-20 |author= |date=2018-08-18|work=编辑：宇诺 |publisher=红圈新闻网站|language=Zh-hans }}</ref>。同时，[[新浪微博|新浪微博]]上禁止搜索“镇魂”二字，并于8月9日<ref name="a8"/>在超话功能中屏蔽“镇魂”这一话题<ref name="a6"/>，包括与《镇魂》名字相近的网络作品《[[镇魂街|镇魂街]]》亦受牵连<ref>{{Cite web |url=http://www.sohu.com/a/246418392_259403 |title=《镇魂街》惨遭微博封杀？这其中似乎有点隐情|accessdate=2018-08-20 |author= |date=2018-08-10|work= |publisher=搜狐网|language=Zh-hans }}</ref>。8月15日，[[中国中央电视台音乐频道|中国中央电视台音乐频道]]播出《镇魂》推广曲《时间飞行》，被视为剧集可能重新上架的预兆。此后，下架的花絮重新上线<ref name="a5"/>。8月18日时，新浪微博已在超话功能中解禁“镇魂”<ref name="a6">{{Cite web |url=https://baijiahao.baidu.com/s?id=1609105154062834447&wfr=spider&for=pc |title=《镇魂》微博超话已解禁，镇魂女孩喜极而泣，重播正片还会远吗？|accessdate=2018-08-20 |author= |date=2018-08-18|work= |publisher=|language=Zh-hans }}</ref>，但剧集仍处于下架状态。8月底，网友发现湖北的地方台有播出此剧<ref name="a7">{{Cite web |url=https://baijiahao.baidu.com/s?id=1609665063171694705&wfr=spider&for=pc |title=《镇魂》现身地方电视台，这些现象表明再有可能上架？|accessdate=2018-08-29 |author= |date=2018-08-24|work= |publisher=|language=Zh-hans }}</ref>。9月22日的报道，或称优酷视频复播其中第1、第6集，不久又迅速屏蔽。优酷官方无公开回应<ref name="b1">{{Cite web |url=https://baijiahao.baidu.com/s?id=1612312484905687635&wfr=spider&for=pc |title=《镇魂》再上架，镇魂女孩：这是中秋节的惊喜？怎么突然又没了！|accessdate=2018-09-25 |author= |date=2018-09-22|work= |publisher=|language=Zh-hans }}</ref>。11月10日，优酷在官方微博发布《镇魂》海报，配文称“让这个夏天的感动，延续成冬天的温暖”。后证实，《镇魂》重新上架，称之为镇魂女孩过年<ref>{{Cite web |url=http://ent.sina.com.cn/v/m/2018-11-10/doc-ihmutuea8881563.shtml |title=镇魂女孩过年了！网剧《镇魂》重新上架|accessdate=2018-11-19 |author= |date=2018-11-10|work= |publisher=新浪网|language=Zh-hans }}</ref>。次日，观众反馈，相较之前播放的版本，某些CP剧情和经典场面被删除，[[弹幕|弹幕]]数量减少<ref name="a8">{{Cite web |url=http://www.bjnews.com.cn/ent/2018/11/11/520288.html |title=《镇魂》时隔3个月重新上架，剧情有删减|accessdate=2018-11-19 |author=记者：张赫 |date=2018-11-11|work=编辑：郭冠华 |publisher=新京报网|language=Zh-hans }}</ref>。

== 剧情 ==

剧集对原著情节有较大改编，将以上古[[中国神话|中国神话]]为背景的故事设定改为以地星人和海星人、亚兽族人之间的争斗。

位于龙城市光明路4号的特别调查处是一处特殊机构，它的职责是保护市民、维护和平，并处理地星人私逃至海星的相关事件。龙城大学发生奇案后，赵云澜带领特调处一干人等前往调查，由此结识沈巍。沈巍在之后，亦搬家成为他的邻居。特调处在龙城大学开始查案，以及之后发生的数起奇异事件，沈巍皆身陷其中。此事，引起了赵云澜的怀疑。他不知道，沈巍的真实身份实际是与特调处合作已久的黑袍使。

== 轶事 ==

=== “沈美人”与“面面好看” ===

剧中朱一龙分饰沈巍、夜尊两角。剧集播出后，沈巍便赢得“沈美人”的昵称<ref>{{Cite web |url=http://k.sina.com.cn/article_6523106404_184ceb464001009u1v.html |title=确认过眼神，《镇魂》优酷官方运营应该是自己人，小标题说明一切|accessdate=2018-09-25 |author= |date=2018-07-11 |work= |publisher=新浪网|language=Zh-hans }}</ref>。夜尊在原著中被称为鬼面，镇魂女孩称之为“面面”。夜尊在剧集后半部出场，脸部佩戴面具示人。第30、31、32集剧情中，沈巍被夜尊绑于柱上。7月12日，优酷视频播出三分钟的第33集[[预告片|预告片]]<ref>{{Cite web |url=http://www.sohu.com/a/240970878_100157846 |title=镇魂33集高甜来袭|accessdate=2018-09-25 |author= |date=2018-07-13|work= |publisher=|language=Zh-hans }}</ref><ref name="c1"/>，预告片<ref>{{Cite web |url=https://baijiahao.baidu.com/s?id=1605792781712947434&wfr=spider&for=pc |title=《镇魂》最让人心疼的反派，夜尊：面面委屈、面面不知道……|accessdate=2018-09-25 |author= |date=2018-07-12|work= |publisher=|language=Zh-hans }}</ref>中，夜尊摘下面具以真面目示人。结果，“面面好看”成为[[新浪微博|新浪微博]]热搜第一。朱一龙本人在12日，以沈巍的口气回应“弟弟你够了啊，后台很硬嘛”，同时贴出[[表情包|表情包]]，表示“前30集白演了”。镇魂女孩的反应则是：面面太好看了！对不起，巍巍，我爬墙一个星期下周三我再回来！你在柱子上等着我！<ref>{{Cite web |url=http://www.sohu.com/a/241635254_602191 |title=面面一出道沈巍成旧爱？不仅好看，还有一个重要原因他哥比不过！|accessdate=2018-09-25 |author= |date=2018-07-17 |work= |publisher=搜狐网|language=Zh-hans }}</ref>，亦有评论调侃沈巍要夺回粉丝，只有吐个血与之一搏<ref name="c1">{{Cite web |url=https://www.sohu.com/a/240845431_788840 |title=《镇魂》开播以来第一名的热搜竟然是沈面，沈巍：前三十集白演了|accessdate=2018-09-25 |author= |date=2018-07-12|work= |publisher=搜狐网|language=Zh-hans }}</ref>。

=== 被诟病的剧集结局与原著作者新增的番外 ===

7月25日<ref name="a8"/>，播出剧集结局。原著中的团圆结局被改成两位[[主角|主角]]牺牲。原著作者Priest为安抚粉丝，于30日在[[新浪微博|微博]]发表[[番外|番外]]，设定81个芥子世界，指剧集为赵云澜所经历的长达一万年的第81个芥子世界。番外中，赵云澜突破第81个芥子世界后，得以与沈巍团圆结局<ref>{{Cite web |url=https://baijiahao.baidu.com/s?id=1607382812983670096&wfr=spider&for=pc |title=priest为《镇魂》写番外！原作亲自改结局，网友：这才是真结局！|accessdate=2018-08-14 |author= |date=2018-07-30 |work= |publisher=|language=Zh-hans }}</ref>。

=== 因剧嗑成CP的“湖北天气”、“陕西气象” ===

剧集播出时，新浪微博的官方微博广泛使用《镇魂》和演员的[[表情包|表情包]]。在诸多官方微博中，主演白宇家乡的陕西气象台（微博帐号“陕西气象”）和朱一龙家乡的湖北气象台（微博帐号“湖北天气”）的官方微博，被戏称为“最浪”的官博<ref>{{Cite web |url=https://baijiahao.baidu.com/s?id=1607300183141365819&wfr=spider&for=pc |title=看完《镇魂》后变成魔鬼的官博，最浪还数湖北天气和陕西气象|accessdate=2018-11-19 |author= |date=2018-07-29 |work= |publisher=|language=Zh-hans }}</ref>。也由于它们之间频繁地互动而成为一对[[配對_(同人)|CP]]——珊瑚CP<ref>{{Cite web |url=https://baijiahao.baidu.com/s?id=1608478656346997491&wfr=spider&for=pc |title=当“珊瑚”CP遇上白宇朱一龙，既皮又燃的兄弟情你招架得住吗？|accessdate=2018-11-19 |author= |date=2018-08-11 |work= |publisher=|language=Zh-hans }}</ref>。2019年1月22日，《[[人民日报|人民日报]]》在北京主办2019政务V影响力峰会。会议上，[[人民网|人民网]]发布《2018年度人民日报政务指数·微博影响力报告》<ref>{{Cite web |url=http://yuqing.people.com.cn/n1/2019/0121/c209043-30581156.html |title=《2018年度人民日报政务指数·微博影响力报告》发布|accessdate=2019-02-22 |author= |date=2019-01-22 |work=责编：王堃、章翔 |publisher=人民网|language=Zh-hans }}</ref>。报告中，虽未将两家单位的微博帐号列入全国十大气象微博，但对两家单位给予了积极的评价。称“珊瑚CP”巧借热播剧，树立气象“大IP”。指它们“巧妙利用影视剧热度，跟进传播、适时联动，将明星粉丝转化为自身粉丝，有效借力提升自身影响力和吸引力，成立树立了新的气象新媒体‘大IP’。<ref>{{Cite web |url=http://yuqing.people.com.cn/NMediaFile/2019/0121/MAIN201901211335000329860253572.pdf|title=《2018年度人民日报政务指数·微博影响力报告》……三、2018年度政务微博影响力分榜……31.全国十大气象微博......60|accessdate=2019-02-22 |author= |date= |work= |publisher=人民网|language=Zh-hans }}</ref>”
