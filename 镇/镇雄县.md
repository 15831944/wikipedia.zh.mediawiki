'''镇雄县'''是[[中华人民共和国|中华人民共和国]][[云南省|云南省]][[昭通市|昭通市]]下属的一个[[县|县]]。

== 历史沿革 ==
镇雄县名取“镇守雄关”之意。

[[夏朝|夏]]、[[商朝|商]]属[[梁州|梁州]]、[[雍州|雍州]]区域。[[周朝|周]]时名屈流大雄甸。[[西汉|西汉]][[建元|建元]]六年（前135年），为南广县，属[[犍为郡|犍为郡]]。[[建安|建安]]十九年（214年），改置南昌县，属[[朱提郡|朱提郡]]。[[蜀汉|蜀汉]][[延熙|延熙]]元年（238年）置[[南广郡|南广郡]]。

[[隋|隋]][[开皇|开皇]]三年（583年），改设协州，不久废。[[唐|唐]][[武德|武德]]元年（618年），复置协州。天宝战争（754年）后，入南诏，改为芒部。

[[北宋|北宋]][[熙宁|熙宁]]七年（1074年），置西南番部都大巡检司。宋代，云南境内，继唐代南诏之后。937年，白蛮段氏建立了[[大理国|大理国]]，仍以大理为政治中心。滇东北地区不属于大理国范围，是宋叙州[[羁縻|羁縻]]之地。有乌蒙部（今昭通、鲁甸）、芒部（今镇雄）、乌撒部（今威宁）、閟畔部（今会泽、巧家）、科部（
四川宁南）、易娘部（今彝良）、易溪部（今威信）、阿头部（贵州赫章）。

[[元朝|元朝]][[至元|至元]]十年（1273年），置芒部路军民总管府。[[明|明]][[洪武|洪武]]十五年（1382年），设芒部卫，相继改为芒部府，隶云南布政司，十六年（1383年）正月，改隶四川布政司，十七年（1384年），升为芒部军民府；[[嘉靖|嘉靖]]五年（1526年），援引“大雄”古名，改称镇雄军民府；[[万历|万历]]三十六年（1609年），改称镇雄府。

[[清|清]][[雍正|雍正]]五年（1727年），改隶云南；六年（1728年），降府为州，属昭通府。[[宣统|宣统]]三年（1910年），升为直隶州。[[民国|民国]]二年（1913年），改州为县，分置[[彝良县|彝良县]]，设立威信行政委员（后改为设治局）。

1949年后，镇雄县隶属昭通地区；2001年8月，昭通撤地设市，镇雄县仍隶属昭通市。

== 地理概况 ==
镇雄位于云南省东北，云贵川三省结合部，地跨东经104°18'至105°19'，北纬27°17'至27°50'。东以[[赤水河|赤水河]]为界与[[四川|四川]][[叙永|叙永]]相邻，南连[[贵州|贵州]][[毕节|毕节]]、[[赫章|赫章]]，西毗[[彝良|彝良]]，北抵[[威信|威信]]。县属坡头乡德隆村，即“鸡鸣三省”之地。县境从东至西99公里，从南到北54公里，总面积3696平方公里。县城驻地乌峰镇，距省城[[昆明|昆明]]598公里，距[[贵阳|贵阳]]326公里，距[[重庆|重庆]]505公里，距[[成都|成都]]618公里，距昭通265公里。县城与毗邻市（县）城距离，距叙永182公里，距威信87公里，距赫章76公里，距毕节73公里，距彝良220公里。

== 行政区划 ==
2012年至今，镇雄县辖21镇7乡：乌峰镇、罗坎镇、雨河镇、大湾镇、牛场镇、五德镇、芒部镇、以勒镇、赤水源镇、母享镇、泼机镇、黑树镇、碗厂镇、盐源镇、木卓镇、以古镇、坪上镇、塘房镇、场坝镇、中屯镇、坡头镇、花朗乡、花山乡、杉树乡、鱼洞乡、尖山乡、果珠彝族乡、林口彝族苗族乡。

== 自然资源 ==
自然资源较为丰富，历史上与滇西的[[腾冲|腾冲]]共同享有“金腾越、银镇雄”的美称。

=== 水土资源 ===
县境内有[[横江|横江]]、[[赤水河|赤水河]]、[[乌江|乌江]]三大水系，含大小河流39条，主要河流有[[白水江|白水江]]、[[罗甸河|罗甸河]]、[[翟底河|翟底河]]等。全县每年降水量33.65亿立方米，地下水储量10.38亿立方米，水能理论蕴藏量26万千瓦，实际可开发量22万千瓦，现已开发3.69万千瓦。现有耕地124.3万亩，人均耕地0.98亩。

=== 矿产资源 ===
境内矿藏主要有[[煤炭|煤炭]]、[[硫铁矿|硫铁矿]]、[[大理石|大理石]]、[[水晶石|水晶石]]、[[石灰石|石灰石]]等30多种。其中，煤炭和硫铁矿分布很广，储量丰富，煤炭远景储量74亿吨，占全省储量28.01%，工业储量45.17亿吨，占全省储量的17.1%。

=== 生物资源 ===
有植物资源600多种。有[[银杏|银杏]]、[[珙桐|珙桐]]等世界珍稀树种及价值极高的濒危古木“红豆杉”。被誉为“涂料之王”的镇雄木漆，在国内外市场久负盛名，畅销不衰；被誉为“果中珍珠”的镇雄[[樱桃|樱桃]]，年产鲜果45万公斤以上。有筇竹、方竹等竹笋珍品和享誉全国的苦丁茶。有[[天麻|天麻]]、[[黄连|黄连]]、[[半夏|半夏]]、[[云木香|云木香]]、[[杜仲|杜仲]]等100多种中药材和竹荪、[[香菌|香菌]]、[[木耳|木耳]]等菌类植物。有野生动物150多种，其中有国家二级保护动物大鲵（[[娃娃鱼|娃娃鱼]]）。
      
== 地貌气候 ==
镇雄位处[[云贵高原|云贵高原]]北部斜坡地带，境内喀斯特地貌比较丰富，喀斯特溶洞，天坑以及天生桥等自然风光众多，是发展玻璃栈道观景平台的理想之地！境内山峦起伏，沟壑纵横，全县有芒部镇 中屯平坝村 泼机镇小平坝等少量坝区，以及半山区、山区和高寒山区。地势西南高，东北低，中部和南部稍平缓。最高点是安尔乡麦车村的戛么山，海拔2416米；最低处为罗坎镇桐坪村大滩，海拔630米。镇雄大多数地方属暖温带季风气候，少数河谷地区属北亚热带气候，年平均气温11.3℃，日照1341小时，无霜期218.6天，年平均降水量914.6毫米。

== 特产 ==
竹荪又名[[竹笙|竹笙]]、竹参、僧竺蕈，是云南特产的一种名贵食用菌。昭通地区北部的大关、彝良、盐津、镇雄、威信等县生长的竹荪产量多，质量好，其中以镇雄县生产的竹荪为佳。美丽奇特，被称为菌中皇后。

镇雄[[木漆|木漆]]产量居全省第一，在全国木漆生产中占重要地位。木漆又称割漆，是从漆树上割取的天然漆，色呈黑色，耐酸碱、耐高热、防腐性好，有“涂料之王”的称誉。木漆现已走向国际市场。 

各个乡镇特产如下：
*镇雄县：特产天麻、竹荪、樱桃、五加皮等，尤以木漆质量最佳。镇雄生漆主产于镇雄县，为全国产漆重点县之一。该省主要集中于滇东北的昭通地区和滇西北的怒江僳僳族自治州。镇雄生漆分色多，质地纯净，色泽深黑，防腐性强。
*乌峰镇：豆粑
*雨河镇：白酒
*母享镇：糯米
*大湾镇立石碑村：棕片、柿子
*芒部镇：中药“鸣钟五加皮”。牛场镇：竹笋、天麻
*堰塘乡平桥村：棕片
*果珠彝族乡：柿子
*尖山乡：木漆
*盐源乡：木漆、油桐籽
*碗厂乡：天麻、竹笋、香菌
*杉树乡：天麻
*坪上乡老场村：蚕茧
*场坝乡：天麻、木漆
*场坝乡杨梅山村：杨梅、核桃
*中屯乡、塘房乡：樱桃

== 景胜古迹 ==
镇雄，古今自然景胜尤多，只因地处僻壤之乡，无从开发游览。

清[[乾隆|乾隆]]四十九年(1784)，收入《镇雄州志》的景胜，有乌峰耸翠、白人仙影、月窟禅光、一星拱斗、平坝春耕、二水怀珠、桂矾秋钓、梅坳流云。同时，在《景胜》篇记入佛寺垂杨、呐林飞瀑、桃坝晚霞、凤岭樵歌，岁关渔唱、墨池流香、羊梯石蹬、柳林春雨、西山筇竹共十七景。

前人还把一些山岩峭壁、石洞、河塘作名胜古迹载入景胜篇”。[[光绪|光绪]]十三年(1887)续修《镇雄州志》，又增入指阁灵芝、观音竹韵、玉水洄珠、鱼跳听涛、鳖尾黄葛、义笔彩云、长坡远眺、龙凤钟声。

[[民国|民国]]间民间所命的景致，有犀牛望月、平因秋水、仙姑留影、石关月朗、龙洞晴波、仙人撒网、王泄点灯、——品当朝、珠笔点相．但末形成文字。一九八七年编纂《镇雄县志》，又新收入夫妻石罗、鸡鸣三省、罗甸风光、白骡腾空、飞瀑流珠、葡萄三井等。为充实一地之景胜，在编写《镇雄风物志》的过程中，又新命增设鱼跳龙门、罗汉撞钟、芒部风情、间歇泉水、玉带龙宫、玉带天宫、南天门、白马龙潭、古洞天门、美女梳妆、白水天桥落景致．同时将原中山公园和今开辟的人民公园，纳入此篇，以壮此篇之宏观也。

== 鸡鸣三省 ==
云贵川鸡鸣三省位于云贵川三省交界处。东为贵州毕节林口，北位四川叙永县水潦乡岔河村，西为云南镇雄县坡头镇德隆村白车。此地风景优美，悬崖峭壁，绝处逢生。有白车溶洞、交界河滩、白车碑文、献山白车胡家老包、贵州老鹰洞头碑处、四川大堰、岔河丛洞、云南白车凉风洞、白车滴水穿石台、白车小梯子、梯子岩犀牛洞等优美景点。稍远有云南官曾岩、威信扎西会议旧址、四渡赤水旧址、半坡过哈等旅游的好地方。
