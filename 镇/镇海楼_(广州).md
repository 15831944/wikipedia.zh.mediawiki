'''镇海楼'''（'''广州博物馆'''），又名'''望海楼'''，俗称'''五层楼'''。位于[[中华人民共和国|中国]][[廣東省|廣東省]][[广州市|广州市]][[越秀山|越秀山]]（越秀公园）小蟠龙冈上，为广州城市标志之一。現是專門收藏、展覽關於廣州歷史文物和史料的場所。

== 历史 ==
[[明朝|明朝]][[洪武|洪武]]十三年（1380年），永嘉候[[朱亮祖|朱亮祖]]扩建广州城，将宋三城（子城、东、西两城）合并，并开拓北城800余丈，城墙横跨越秀山，在上面建一座五层高楼以壮观瞻。建筑有雄镇海疆之意，故名'''镇海楼'''。大楼于[[嘉靖|嘉靖]]年间大修，到[[清朝|清朝]]毁于[[三藩之乱|三藩之乱]]，[[康熙|康熙]]26年（1687年）重修，[[中华民国|中华民国]]时期再被[[桂系|桂系]]军阀所毁。1928年重修时，将楼内木结构改为钢筋混凝土结构，砖石墙壁基本为明代旧物。1929年起成为广州市立博物院，1950年改称[[广州博物馆|广州博物馆]]。1989年6月被列为[[广东省文物保护单位|广东省文物保护单位]]（3-27），并在1996年被评为广州十大旅游美景之一。[[清代|清代]]以“镇海层楼”及1986年以“越秀层楼”先后列为[[羊城八景|羊城八景]]之一。2013年3月，镇海楼被2013年3月列入[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]。有人以“烟锁池塘柳，灰堆镇海楼”诉说其饱受沧桑。

== 建筑 ==
[[File:Canton,_China;_the_Five-Storied_Pagoda_and_Old_Fort._Wellcome_V0037377.jpg|thumb]]的鎮海樓。]]
镇海楼因楼高五层，俗称五层楼。该楼又名“望海楼”，因当时珠海河道甚宽，故将“望江”变为“望海”。楼前碑廊有历代碑刻，右侧陈列有12门古炮。是广州现存最完好、最具气势，也最富有民族特色的古建筑。镇海楼楼高28[[米|米]]，呈长方形，宽31米、深16米。下两层围墙用红石砌筑，以上为青砖墙。底层墙厚3.86米，以上逐层递减。有复檐5层，绿琉璃瓦盖。饰有石湾鳌鱼花脊。這五層高建築逐步向上收減寬度和深度，令到鎮海樓形似塔又似樓。廣府人將鎮海樓稱為「五層樓」，亦因為登樓所看的美景不絕，也被冠以「五嶺以南第一樓」和誉为“岭南第一胜览”。

鎮海樓自十四世紀後期至今六百多年間曾被多數火毀和人為破壞，其中在明朝嘉靖年間所立的《重修鎮海樓碑》可作為証。此後在1928年再度重修鎮海樓時，將原來木建築樓層改為鋼筋水泥結構，並引入當地石灣彩釉陶燒作為瓦脊和檐角的鰲魚裝飾。

楼顶正面悬挂“镇海楼”横匾，两旁悬挂一副对联：“千万劫，危楼尚存，问谁摘斗摩霄，目空今古；五百年，故侯安在，使我倚栏看剑，泪洒英雄！”。楼内常设《广州历史陈列》展览，展示广州五、六千年来的文化、风俗和城市发展等历程。楼前碑廊有历代碑刻，右侧陈列有12门古炮。它是[[广州城市中轴线|广州传统中轴线]]的起点和最高点。

城防大炮包括明朝[[崇禎|崇禎]]年間以至清朝中葉[[鴉片戰爭|鴉片戰爭]]廣州地區鑄造的防衛大炮，現放置於鎮海樓西側的炮座，並建立碑廊以作紀念。

登上鎮海樓最頂層，可一矙廣州全景，直望珠江。[[改革開放|改革開放]]後，大量高樓矗立，使之江景被阻擋，廣州的古塔大多已被遮蔽，無法直觀。

== 博物館仲元樓展區 ==
仲元樓展區所在建築物原是[[鄧鏗|仲元]]圖書館，1957年2月被改為[[廣州美術館|廣州美術館]]並對外開放。主體建築採青磚綠瓦、重樓飛簷，是仿[[北京|北京]][[故宮|故宮]][[文華殿|文華殿]]的宮殿式建築。館內有佔地面積300多平方米的陳列大廳和100多平方米的名家畫廊。展廳外，廣場前五星崗環山碑廊落成於1987年，嵌有自晉至清的名家書法刻石118方，1998年又在館址東邊續建碑廊，嵌有《尺素遺芬》的刻石58方，均是清末[[海山仙館|海山仙館]]的石刻。

展區收藏有[[中國畫|中國畫]]、[[油畫|油畫]]、[[版畫|版畫]]、[[水粉畫|水粉畫]]、[[水彩畫|水彩畫]]、[[連環畫|連環畫]]、[[雕塑|雕塑]]等各門類的美術作品1萬多件，其中有[[宋朝|宋朝]][[文同|文同]]《墨竹圖》、[[元朝|元朝]][[李珩|李珩]]《紆竹圖》、[[明朝|明朝]][[林良|林良]]《秋樹聚禽圖》、[[清朝|清朝]][[石濤|石濤]]和[[朱耷|朱耷]]合作的《蘭竹圖》等珍品。大門入口處停泊了最後一輛[[廣州雙層巴士|廣州牌雙層巴士]]。
