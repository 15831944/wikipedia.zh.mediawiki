'''-{zh-hans:榴梿; zh-hant:榴槤; zh-cn:榴莲; zh-sg:榴梿;}-'''（[[学名|学名]]：{{lang|la|''Durio zibethinus''}}），又作'''-{zh-hans:榴莲; zh-hant:榴蓮; zh-cn:榴梿; zh-sg:榴莲;}-'''，別名'''麝香猫果'''。为[[锦葵科|锦葵科]][[榴槤属|榴槤属]]植物。一般认为原产地在热带国家。例如：[[泰国|泰国]]、[[马来西亚|马来西亚]]、[[印度尼西亚|印度尼西亚]]、[[文莱|文莱]]。目前[[泰国|泰国]]、[[越南|越南]]、[[中国|中国]]等都栽培<ref>{{cite web| url =http://www.bioinfo.cn/db05/KmzwSpecies.php?action=view&id=1709| title =榴莲| work =《中国高等植物数据库全库》| publisher =中国科学院微生物研究所| author =昆明植物研究所| accessdate =2009-02-24| deadurl =yes| archiveurl =https://web.archive.org/web/20131213025243/http://www.bioinfo.cn/db05/KmzwSpecies.php?action=view&id=1709| archivedate =2013-12-13}}</ref>。榴槤在東南亞被譽為“水果之王”，榴槤與眾不同的地方是它獨特的氣味以及外殼上的刺。榴槤果實可以成長至30厘米（12英寸）長，直徑15厘米（6英寸）。榴槤的形狀多是長橢圓形或圓形，颜色为綠色帶棕色，果肉呈淺黃色。

本种是一種巨型的[[熱帶|熱帶]][[常綠乔木|常綠乔木]]，树高可达45米{{r|worldagroforestry}}。叶片长圆，顶端较尖，[[聚伞|聚伞]][[花序|花序]]，花色淡黄，果实大小如[[足球|足球]]，果皮坚实，密生三角形刺，果肉是由[[假种皮|假种皮]]的肉包皮组成，肉色黃至白皆有，粘性多汁，酥软味甜，吃起来有冰淇淋的口感，但因味道濃重異於其他水果，不是普羅大眾都喜歡的。

== 產地 ==
原产[[东南亚|东南亚]]，有「果王」之稱。原產地包括[[汶萊|汶萊]]，[[印度尼西亞|印度尼西亞]]和[[馬來西亞|馬來西亞]]。榴槤生長地遍佈[[東南亞|東南亞]]，其主要生長在[[泰國|泰國]]、[[马来西亚|马来西亚]]、印尼等地，其他種植榴槤地方包括[[柬埔寨|柬埔寨]]、[[老撾|老撾]]、[[越南|越南]]、[[緬甸|緬甸]]、[[印度|印度]]、[[斯里蘭卡|斯里蘭卡]]、[[西印度群島|西印度群島]]、[[佛羅里達州|佛羅里達州]]、[[夏威夷|夏威夷]]、[[巴布亞新幾內亞|巴布亞新幾內亞]]、[[波利尼西亞群島|波利尼西亞群島]]、[[馬達加斯加|馬達加斯加]]、[[海南島|海南島]]、[[澳大利亞|澳大利亞]]北部和[[新加坡|新加坡]]。

== 種類 ==
本种是国际贸易上最常见的[[榴槤屬|榴槤屬]]物种，人工栽培的品系已有上百样。除了本种，还有另外9种可食用的榴梿属物种，多只在原生地的市场上出现。

== 營養 ==
榴莲营养丰富，100克榴莲中含有热量147（千）[[卡路里|卡路里]]、[[蛋白质|蛋白质]]1.47克、[[脂肪|脂肪]]5.33克、糖类27.09克、[[维生素B|维生素B]]20.13毫克、[[维生素C|维生素C]]19.7毫克，还有许多微量元素，包括[[鉀|鉀]]。{{or|榴槤含豐富的[[雌激素|雌激素]]可能增加女性的生育機會。}}

== 歷史 ==
東南亞早於史前時代已食用榴槤，但西方世界和中国認識榴槤只約600年。已知歐洲最早榴梿的紀錄是[[意大利|意大利]][[旅行家|旅行家]][[尼科洛·達·康提|尼科洛·達·康提]]，在15世紀前往東南亞時紀錄。[[葡萄牙|葡萄牙]]醫生{{le|加西亞·德奧爾塔|Garcia de Orta}}在1563年出版的書中描述榴梿在印度用作藥用。同时期，随[[郑和下西洋|郑和下西洋]]的穆斯林[[马欢|马欢]]在《[[瀛涯胜览|瀛涯胜览]]》记述苏门答剌国的榴梿“赌尔焉”{{notetag|赌尔焉，漢語拼音：dǔ'ér yān，是马来语“durian”音譯。}}：“有一等臭果，番名‘赌尔焉’，如中国[[水鸡头|水鸡头]]样，长八九寸，皮生尖剌，熟则五六瓣裂开，若臭牛肉之臭，内有粟子大酥白肉十四五块，甚甜美好吃，中有子可炒吃，其味如栗。<ref>[[马欢|马欢]]《[[瀛涯胜览|瀛涯胜览]]·苏门答剌国》</ref>”

== 成熟 ==
榴莲是一种典型的热带水果，产地为[[泰国|泰国]]、[[马来西亚|马来西亚]]等国。某些品種的榴槤（例如生長於馬來西亞的種）不可以采摘，否则不能继续成熟，果树也会受损伤。只有等其成熟后自己落下。榴蓮樹種类眾多，有些種四、五年即可結果，有的要六至八年才能結果。絕大多數的榴蓮樹每年結果實一次，一棵樹每年可産七、八十個榴蓮<ref name="熱帶水果之王"/>。它还有一个奇特现象就是“老茎生花”，因为它的一个果实大的重达1－2公斤，树上细小的树枝难以承受这么重，它就在[[主干|主干]]上开花结果。是适应环境的结果。一個榴蓮从樹上掉下後不斷成熟的二至四天内是最美味的，但經過五，六天後，大多數人會認為這时爛熟又難吃。

== 氣味 ==
[[File:Singapore_MRT_Fines.jpg|right]]上禁止携带榴蓮的告示牌（右下圖）]]
人类对榴莲的气味有非常极端的嗅觉反映，一部分人认为非常芳香，爱吃的人觉得很香，越吃越爱吃，越吃越上瘾；而另一部分人认为恶臭无比。它的种子也可以吃。一般很少有一种水果有如此明显的嗅觉反差，但由于考虑到公众反映，不少国家如[[马来西亚|马来西亚]]，[[泰国|泰国]]和[[新加坡|新加坡]]禁止[[酒店|酒店]]、[[飞机|飞机]],[[餐馆|餐馆]]和其他[[公共交通工具|公共交通工具]]携带打开的榴莲。

成熟后自然裂口的榴莲，存放时间不能太久，当嗅闻时有一股[[酒精|酒精]]味时，就是变质了。未开口的榴莲，不成熟的有一股青草味，成熟的散发出榴莲固有的气味。

如果用手直接抓著榴蓮吃，手上的味道會殘留兩至三天，據說用榴蓮的殼當容器盛水洗手的话，其去除氣味的效果比用肥皂洗手更好，而且马上味道就会消除。<ref name="熱帶水果之王"/>。

== 用途 ==
在馬來西亞，榴莲的葉子和根部用來解熱，而葉汁用來治療光發燒的病人。它也可製成不同美食，例如：榴蓮月餅和榴蓮炸彈

== 注意 ==
{{ctm}}
因榴莲含[[糖|糖]]较高，[[糖尿病|糖尿病]]患者应不吃或少吃。榴莲含[[钾|钾]]较高，故[[肾|肾]]病、[[心脏病|心脏病]]患者应慎吃。
