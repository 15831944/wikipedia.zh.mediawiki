
'''紹熙內禪'''，又稱'''紹熙禪位'''、'''宋光宗內禪'''等。是一次[[南宋|南宋]]的政治事件，發生於[[宋光宗|宋光宗]]紹熙五年（1194年），[[宋光宗|宋光宗]][[禪位|禪位]]予嘉王[[趙擴|趙擴]]，是為[[宋寧宗|宋寧宗]]。名义上是光宗自行禅位，实际上是被迫[[退位|退位]]，是[[赵汝愚|赵汝愚]]、[[趙彥逾|趙彥逾]]、[[葉適|葉適]]、[[徐誼|徐誼]]、[[郭杲|郭杲]]、[[韓侂冑|韓侂冑]]等人，在獲得[[憲聖慈烈皇后|隆慈太皇太后]]支持之下，所發動廢除宋光宗的宫廷[[政变|政变]]。

== 背景 ==
[[淳熙|淳熙]]十四年（1187年）十月，任二十五年太上皇的[[宋高宗|宋高宗]]駕崩於德壽宮中，皇帝[[宋孝宗|宋孝宗]]表示要服喪三年，为了服丧，让太子[[赵惇|赵惇]]参预政事。[[淳熙|淳熙]]十六年二月初二（1189年2月18日）又[[禪位|禪位]]，命太子赵惇即位，太子即位后，是为[[宋光宗|宋光宗]]。太上皇孝宗被尊為「壽皇聖帝」，簡稱「壽皇」。

[[慶遠|慶遠]][[節度使|節度使]][[李道|李道]]的次女[[李鳳娘|李鳳娘]]姿色豔麗，曾有道士[[皇甫坦|皇甫坦]]看她的[[面相|面相]]，稱此有大貴之相，言于[[宋高宗|宋高宗]]說：「此女當母儀天下。」高宗命其孫趙惇（後來的宋光宗）聘李鳳娘為妃，封榮國夫人，後改封定國夫人，[[乾道|乾道]]四年（1168年）生子趙擴。

李氏雖姿色豔麗，卻性喜搬弄是非，性妒无礼，行事霸道，為[[太子妃|太子妃]]時就已令宋高宗和宋孝宗對她反感，宋高宗還曾痛罵，說自己被[[皇甫坦|皇甫坦]]所誤。李鳳娘為皇后後，恃著光宗生性懦弱，又疏於朝政，大事小事都要取決於她，權力漸被驕恣凶悍的李鳳娘操控。

== 李皇后 ==
宋光宗即位後，李皇后曾朝見壽皇，請立[[趙擴|趙擴]]為皇太子，但壽皇認為應該由光宗親自來商議，李皇后對壽皇咆哮，壽皇大怒，自此李皇后不斷阻止光宗朝見壽皇。由於壽皇在位時每幾天就朝見高宗一次，光宗不朝見壽皇，兩人的孝道成為很大的對比。

李皇后剽悍，壓抑光宗，光宗生病，精神也不太正常。绍熙二年冬十一月，李皇后又杀害了有孕在身的黄贵妃，光宗知道之后，又更生病了。<ref>宋史/卷243，黄贵妃，淳熙末在德寿宫，封和义郡夫人。光宗为皇太子，傍无侍姬，上皇以夫人赐之，遂专宠。即位，拜贵妃。绍熙二年冬十一月，为皇后李氏所杀。帝闻而成疾。</ref>壽皇聽說後便送來了藥丸，但李皇后卻造謠說那是毒藥，離間光宗與壽皇父子關係。有一次，光宗宴請大臣，大臣請光宗探望壽皇，李皇后知道了以後立即阻止。

壽皇駕崩以前，光宗始終沒有探視過壽皇。

== 退位 ==
由於壽皇的喪禮需要身為長子的[[宋光宗|宋光宗]]主持，而光宗在李皇后的阻止下不主持，也不服喪，使得朝臣紛紛不滿，[[相國|相國]][[留正|留正]]及[[樞密使|樞密使]][[趙汝愚|趙汝愚]]（[[宋太宗|宋太宗]]之後人）请示光宗立嘉王为[[皇太子|皇太子]]，而光宗在奏章批示「歷事歲久，念欲退閒」<ref>《宋史》本纪第三十七.宁宗一，「是夕，御批付丞相云：历事岁久，念欲退闲。」</ref>后，留正也因為不滿朝政而离开朝廷；趙汝愚则聯合[[趙彥逾|趙彥逾]]、[[葉適|葉適]]、[[徐誼|徐誼]]、[[郭杲|郭杲]]、[[韓侂冑|韓侂冑]]等人策畫政變，[[韓侂冑|韓侂冑]]請高宗之妻[[隆慈太皇太后|隆慈太皇太后]]出面（韓侂冑是隆慈的外甥）[[垂簾聽政|垂簾聽政]]，又以防不测，命殿帅[[郭杲|郭杲]]，步帅[[阎仲|阎仲]]连夜带兵控制南皇宫南北，[[傅昌朝|傅昌朝]]暗中制造黄袍<ref>宋史/卷392，「侂胄复命，汝愚始以其事语陈骙、馀端礼，使郭杲及步帅阎仲夜以兵卫南北内，礼使其姻党宣赞舍人傅昌朝密制黄袍。」</ref>，趙汝愚兵谏逼迫光宗退位。使得尚未成为太子的赵扩，就受禪[[稱帝|稱帝]]。

== 結果與影響 ==
結果，[[趙汝愚|趙汝愚]]一手策划导演了这场政变，隆慈太皇太后垂帘听政主持禫祭；嘉王赵括由「立為太子」变为「稱帝」，即史上的[[宋寧宗|宋寧宗]]。宋光宗被迫退位，尊為[[太上皇|太上皇]]，李皇后被尊為太上皇后。

[[趙汝愚|趙汝愚]]因推立之功出任宰相，而[[韓侂冑|韓侂冑]]的姪女被立為皇后。而由於[[趙汝愚|趙汝愚]]拒絕向[[韓侂冑|韓侂冑]]讓步，[[韓侂冑|韓侂冑]]向宋寧宗進言罷免[[趙汝愚|趙汝愚]]，[[韓侂冑|韓侂冑]]得以專權。

==评价==
*清[[赵翼|赵翼]]：「光宗虽-{云}-内禅，其实废也。宁宗背其生父，正其不孝之罪大，而急夺其位，且以扶立者为其有大劳而报之」。而「人知光宗之不孝，而不知宁宗之不孝，尤倍于光宗。」
