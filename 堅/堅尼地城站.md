'''堅尼地城站'''（{{lang-en|Kennedy Town Station}}）是[[港鐵|港鐵]][[港島綫|港島綫]]西端終點[[鐵路車站|鐵路車站]]，位於[[香港島|香港島]][[中西區_(香港)|中西區]][[堅尼地城|堅尼地城]][[士美菲路|士美菲路]]和[[科士街|科士街]]之間的地底，於2014年12月28日隨[[港島綫西延|西港島綫]]通車正式啟用並取代[[上環站|上環站]]成為港島綫的西面終點站。

==車站設計==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Kennedy Town Station Concourse 201412.jpg | caption1 = 大堂
| image6 = Kennedy Town Station Control Room.JPG | caption6 = 車站控制室
}}
堅尼地城站上方為公園及公共交通交匯處，出入口設計與上方的兩個公園及交通交匯處互相結合，並使用啡黃色掛石作外牆裝飾，頂棚用料為玻璃，可以讓天然光滲透進車站入口處內；而地底部分則分為兩層，地庫一層為車站大堂，地庫二層為兩個側式月台，月台設計並與[[上環站|上環站]]相似，用作分隔出站及入站客流之用，以避免乘客坐反方向誤入掉頭隧道之內。

港鐵解釋堅尼地城站鄰近海旁，故此車站大堂及月台以湖水藍色為車站主色。<ref name="cite_ref-9" />

===大堂===
堅尼地城站內提供6間商店供乘客購物或進食，包括兩間7-11便利店、La Boheme Bakery、意大利雪糕店Heyyo、uji bedding床上用品店及銀行櫃員機等服務。車站亦自助服務設施、以至男、女、及傷殘洗手間。另外，大堂近C出入口設有1部「[[會員服務站|會員服務站]]」，供乘客享用港鐵友禮會的會員優惠<ref>{{Cite web|language=zh-hk|publisher=港鐵公司|title=鐵路四處遊－「會員服務站」|url=http://www.mtr.com.hk/ch/customer/services/csp.html|author=港鐵公司|accessdate=2016-09-26}}</ref>。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Kennedy Town Station 2018 01 part1.jpg | caption1 = 2號月台（落客專用）
}}
堅尼地城站港島綫月台為兩個[[側式月台|側式月台]]，全部月台均設有[[月台幕門|月台幕門]]。1號及2號月台在乘客眼中看似兩個位於不同層數的單向式月台，但事實上它們是對向式月台設計，只是兩個月台之間被牆壁完全阻隔，令乘客無從得知兩月台位於同一層，與[[南昌站_(香港)|南昌站]]（[[西鐵綫|西鐵綫]]）、[[美孚站|美孚站]]（西鐵綫）、[[荃灣西站|荃灣西站]]、[[坑口站_(香港)|坑口站]]、[[上環站|上環站]]、[[何文田站|何文田站]]（[[觀塘綫|觀塘綫]]）及[[青衣站|青衣站]]的設計類似。每當上下班繁忙時間之後，部份港島綫的港鐵列車在此站清客後，便會暫停服務，先使用調頭隧道折返1號月台，再返回[[柴灣車廠|柴灣車廠]]。

==利用狀況==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = MTR KET (14).JPG | caption1 = 2號月台頭端可看到列車經由調頭隧道駛入同層1號月台
}}
堅尼地城站鄰近只有[[住宅|住宅]]、[[辦公室|辦公室]]及社區綜合大樓，鄰近並無大型的購物商場和[[百貨公司|百貨公司]]。因此，本站的服務對象主要是居住於鄰近的堅尼地城居民和上班人士。所以，除了上下班的繁忙時間外，整體的出入閘人流會很少。在每天的晚上、星期六、星期日和公眾假期，人流會更少。所以，堅尼地城站是[[港鐵|港鐵]][[港島綫|港島綫]]17個車站中其中一個人流較少的車站。不過，有少部分[[南區|南區]]西部的居民及到[[瑪麗醫院|瑪麗醫院]]探病的民眾，都會使用本站轉乘小巴往其他地方，對本站的人流仍有一定支持。

==歷史==
堅尼地城站最早在1970年的《集體運輸計劃總報告書》出現。該站本應在1980年代與港島綫一起興建。但是當時地鐵公司（即現時的[[香港鐵路有限公司|香港鐵路有限公司]]）以[[上環站|上環站]]以西的路段客量不足和存在技術困難為由，而沒有展開工程。在1980年代或之前的方案中，這個車站位於[[加多近街|加多近街]]近[[堅尼地城巴士總站|堅尼地城巴士總站]]的地底，同時計劃配合將[[港澳客輪碼頭|港澳客輪碼頭]]搬遷至車站旁邊。

根據2005年6月的建議，堅尼地城站將會是港島綫延伸段──西港島綫的車站，而不是在較早前的建議中，一個[[南區_(香港)|南區]]地鐵支綫（當時也被稱為西港島綫，即現時[[南港島綫|南港島綫]]的前身）的車站。根據地鐵公司在2006年8月發表的西港島綫的修訂建議方案<ref>[http://www.legco.gov.hk/yr06-07/chinese/panels/tp/tp_rdp/papers/tp_rdp0330cb1-1279-1-ec.pdf 地鐵公司就西港島線向香港立法會提供的電腦投影片資料]，2007年3月30日</ref>，堅尼地城站位於科士街臨時遊樂場地底，即[[爹核士街|爹核士街]]及[[士美菲路|士美菲路]]一帶。而車站地面出口，即舊[[堅尼地城游泳池|堅尼地城游泳池]]的位置，設有[[公共運輸交匯處|公共運輸交匯處]]，方便鐵路乘客轉乘其他公共交通工具<ref>[http://www.legco.gov.hk/yr08-09/chinese/panels/tp/tp_rdp/papers/tp_rdp0601cb1-1782-1-ec.pdf 港鐵公司就西港島綫向香港立法會提供的電腦投影片資料（2008-09）]</ref>。

堅尼地城站於2013年4月10日平頂，成為[[港島綫西延|西港島綫]]首個平頂的車站，標誌著該站的[[土木工程|土木工程]]完成，隨即正式展開軌道鋪設工程、[[機電工程|機電工程]]及其他後期裝修工程。<ref>{{zh-hant}}{{cite web|author= |title=港鐵西港島綫堅尼地城站平頂 |url=http://mtr.com.hk/eng/corporate/file_rep/PR-13-024-C.pdf |work=[[港鐵公司|港鐵公司]] |date=2013年4月10日}}</ref>本站於2014年9月30日竣工{{請求來源}}，並已於同年12月28日正式通車<ref name="東方日報20141018"/>。

港鐵在2014年12月13及14日（星期六、日）分別開放[[香港大學站|香港大學站]]及堅尼地城站予公眾參觀，由中西區區議會派發入場券。12月14日有逾二千名西區居民參加本站開放日及嘉年華會。<ref>{{cite news |title=堅尼地城站舉行嘉年華會　慶祝港島綫西延12月28日開通 |url=http://www.mtr.com.hk/archive/corporate/en/press_release/PR-14-127-C.pdf |publisher=[[港鐵公司|港鐵公司]]}}{{zh-hant}}</ref>

根據堅尼地城站的照片，牆上以大型的書法體展示站名，與港島線其他車站一致。惟書法體的「堅尼地城」四字在引起[[網民|網民]]爭議，多位網民質疑「-{堅}-」字採用了[[簡體字|簡體字]]「-{坚}-」。有網民指出，「-{堅}-」在[[行書|行書]]中亦有寫為「-{坚}-」字的習慣<ref>{{cite web|title=【坚】行书书法字典|url=http://shufa.guoxuedashi.com/575A/2/|publisher=国学大师}}</ref>。而港鐵發言人澄清，西港島線站名並無採用簡體字，3座新鐵路站內的書法站名均是出自同一位前港鐵建築師[[區傑棠|區傑棠]]之手筆，字型設計融入了作者的個人書法藝術風格。為了讓鐵路站增添特色，港島線最初啟用時就採用書法字在月台顯示車站名稱，令車站添加中國傳統文化氣息，與車站主色調配合，形成獨有的設計風格；由於西港島綫是港島線的延線，故車站內展示的站名沿用港島線的書法字體及風格，保持連貫性。<ref name="cite_ref-9">[http://hk.apple.nextmedia.com/realtime/news/20140321/52307925  蘋果日報即時新聞：車站名現簡體字疑雲 港鐵：無採用簡體字樣]</ref>

===塗鴉惡搞港鐵 數小時內被清理===
2018年10月23日早上，網民在facebook「西環變幻時」群組貼文，發現車站B出口的石屎牆上被塗鴉上諷刺港鐵的標誌和字句。港鐵標誌上看起來像「屎」字。而標誌的旁邊畫上「Most Troublesome Railway」的字句。塗鴉很快已被黑色膠袋遮蓋。到下午牆壁已重新上油。<ref>{{cite news |author1=陳展希 |title=堅尼地城站現塗鴉惡搞港鐵Logo　下午已重新上油　網民：效率奇高 |url=https://www.hk01.com/%E7%86%B1%E7%88%86%E8%A9%B1%E9%A1%8C/249738/%E5%A0%85%E5%B0%BC%E5%9C%B0%E5%9F%8E%E7%AB%99%E7%8F%BE%E5%A1%97%E9%B4%89%E6%83%A1%E6%90%9E%E6%B8%AF%E9%90%B5logo-%E4%B8%8B%E5%8D%88%E5%B7%B2%E9%87%8D%E6%96%B0%E4%B8%8A%E6%B2%B9-%E7%B6%B2%E6%B0%91-%E6%95%88%E7%8E%87%E5%A5%87%E9%AB%98 |accessdate=2018-10-22 |work=香港01 |date=2018-10-22}}</ref>
