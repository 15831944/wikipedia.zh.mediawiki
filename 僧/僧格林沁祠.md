'''僧格林沁祠'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[地安门东大街|地安门东大街]]47号，是为[[清朝|清朝]]大臣[[僧格林沁|僧格林沁]]所立祠堂。

==历史==
[[僧格林沁|僧格林沁]]，[[博尔济吉特氏|博尔济吉特氏]]，[[内蒙古|内蒙古]][[科尔沁左翼后旗|科尔沁左翼后旗]]人，科尔沁扎萨克郡王[[索特纳木多布济|索特纳木多布济]]的嗣子。清朝[[同治|同治]]四年（1865年），在[[山东|山东]]追击[[捻军|捻军]]时，被[[赖文光|赖文光]]、[[张宗禹|张宗禹]]率领的捻军在山东曹州（今[[山东省|山东省]][[菏泽市|菏泽市]]）高楼寨周边的吴家店围歼并斩首，所部骑兵全军覆没<ref name=dhly>{{Cite web |url = http://www.bjdclib.com/subdb/laneculture/historybuilding/200906/t20090616_2149.html |title = 僧格林沁祠堂 |author =  |date =  2009-06-16|publisher = 东华流韵 |language =  |accessdate =  |quote =  }}</ref><ref name=wf>{{Cite web |url = http://www.bjdclib.com/dclib/subjectdb/dhly/newsphoto/200911/t20091112_28210.html |title =僧格林沁王府  |author =  |date = 2009-11-12 |publisher =东华流韵   |language =  |accessdate =  |quote =  }}</ref>。

僧格林沁灵柩运回北京后，[[同治帝|同治帝]]与两宫皇太后亲临祭奠。僧格林沁谥曰“忠”，配飨[[北京太庙|太庙]]，绘像[[紫光阁|紫光阁]]，其子[[伯彦讷谟祜|伯彦讷谟祜]]承袭亲王爵，并且赏“博多勒噶台”王号。[[光绪|光绪]]年间，清廷在在今北京地安门东大街北侧为僧格林沁立专祠“显忠祠”（即僧格林沁祠），春秋派官员致祭<ref name=dhly/><ref name=wf/>。

除了这处祠堂外，清廷还在僧格林沁征战之地建有祠堂，例如在[[山东|山东]][[济宁|济宁]][[崇觉寺铁塔|铁塔寺]]内便设有僧格林沁祠堂，因为铁塔寺是僧格林沁剿办捻军的前线指挥部和大本营，僧格林沁死后也曾停尸在铁塔寺东路第三进院。据说祠内供奉着随他征战并殉国的将领牌位，如内阁学士[[全顺|全顺]]等人。现铁塔寺内的僧格林沁祠早已无存<ref>{{Cite web |url =http://www.fjdh.cn/ffzt/fjhy/ahsy2013/11/150830302758.html  |title = 济宁铁塔寺—济宁—山东寺院 |author =  |date = 2013-11-01 |publisher =佛教导航  |language =  |accessdate =  |quote =  }}</ref>。

[[中华民国|中华民国]]时期，北京的僧格林沁祠改为怀幼小学，后来改名为进步小学，又改名为宽街小学<ref name=dhly/>。2000年9月1日，北京市东城区黑芝麻胡同小学、前圆恩寺小学、宽街小学（部分）三校合并后统称“北京市东城区黑芝麻胡同小学”。后来僧格林沁祠不再用作学校，而是由北京市东城区教育局房管所使用。

==建筑==
僧格林沁祠坐北朝南，现状为前殿、享殿、享殿的东西配殿构成的两进[[四合院|四合院]]。其他建筑已无存<ref name=dhly/>。
*仪门：位于地安门东大街北侧。面阔三间。仪门已拆除<ref name=dhly/>。
*前殿：面阔三间（12.6米），进深10.2米，绿琉璃瓦大式[[硬山顶|硬山顶]]，带吻兽和垂兽，前后出廊，[[旋子彩画|旋子彩画]]。现在前殿前后廊均推出<ref name=dhly/>。
**东、西配房：前殿东西配房各五间，为灰筒瓦箍头脊顶，已拆除<ref name=dhly/>。
*碑亭：位于二门前，六角[[攒尖顶|攒尖顶]]。亭内石碑正面朝南，龟趺螭首。碑身宽1.9米，高4.5米，厚0.5米。龟趺高l.1米。碑身和龟趺之间有长方形的碑座，碑座宽1.3米，高0.5米，厚0.56米。碑阳为[[满文|满文]]，碑阴无文字，碑侧雕龙。碑亭已拆除。石碑已于1984年运到[[北京石刻艺术博物馆|北京石刻艺术博物馆]]<ref name=dhly/>。
*二门：砖门楼，面阔4.8米，进深2.9米，绿琉璃瓦[[歇山顶|歇山顶]]，带吻兽和垂兽。二门已拆除<ref name=dhly/>。
*享殿：面阔三间，绿琉璃瓦大式歇山顶，带吻兽和垂兽，旋子彩画，檐下五踩重昂斗栱。前有三出陛月台。现在享殿左右两侧已新建了仿古隔断墙，月台因院内地面不断加高而已经垫平<ref name=dhly/>。
**东、西配殿：享殿东西配殿各三间，都是绿琉璃瓦硬山顶<ref name=dhly/>。
**燎炉：东配殿南侧有一座正方形的砖砌燎炉，每边长2.83米，硬山灰筒瓦调大脊，有吻兽及垂兽，下带方座。燎炉已拆除<ref name=dhly/>。
*围墙：该祠四周原有砖墙，砖墙涂有红垩<ref name=dhly/>。
