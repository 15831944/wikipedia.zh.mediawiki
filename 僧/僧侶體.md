'''僧侶體'''（'''hieratic'''）是[[古埃及|古埃及]]時期[[書吏|書吏]]用來快速記錄的[[手寫體|手寫體]]，與同一時期發展的[[聖書體|聖書體]]有密切的關係<ref name="Goedicke 1988:vii–viii">Goedicke 1988:vii–viii.</ref>。「僧侶體」由公元2世紀[[亞歷山大的革利免|亞歷山大的革利免]]命名<ref>Goedicke 1988:vii; Wente 2001:2006. The reference is made in Clement's ''[http://www.earlychristianwritings.com/text/clement-stromata-book5.html Stromata]'' 5:4.</ref>，詞源來於[[希臘文|希臘文]]{{Polytonic|γράμματα ἱερατικά}}（''grammata hieratika''），意思是「僧侶的文字」，這是由於僧侶體是數千年來都只用作宗教文字。

== 發展 ==
僧侶體在[[埃及早期王朝|埃及早期王朝]]（Protodynastic Period）開始使用，但不是由聖書體演變而成，因為沒有聖書體字形的證據。聖書體用於石碑銘刻最早在[[埃及第一王朝|埃及第一王朝]]，那時候[[書吏|書吏]]已經使用僧侶體，可見兩種文字在同一時期發展而成<ref name="Goedicke 1988:vii–viii"/>。（但这种说法目前还有相当的争议）

僧侶體一直在古埃及時期和[[埃及_(羅馬帝國行省)|羅馬統治時期]]使用。公元前660年[[世俗體|世俗體]]（Demotic）取代僧侶體，用來紀錄非宗教事務。直至公元3世紀，僧侶仍然使用僧侶體。

== 特徵 ==
僧侶體跟一般手寫體不同，閱讀方向從右到左。起初僧侶體可以直書和橫書，但在[[埃及第十二王朝|埃及第十二王朝]]後橫書成為標準書寫方法，除了可避免書寫時弄糟紙上其他文字和增加書寫速度，也使閱讀書卷更方便。

僧侶體有潦草和[[合字|合字]]的特徵，文字較聖書體規範化。僧侶體有時使用[[附加符號|附加符號]]以識別相似字形，一些複雜的符號可以在一劃中完成。

從[[埃及第二十王朝|埃及第二十王朝]]中期至[[埃及第二十六王朝|埃及第二十六王朝]]早期，[[底比斯_(埃及)|底比斯]]地區使用由[[上埃及|上埃及]]行政文件衍生的「僧侶異體」（Abnormal Hieratic）<ref>Wente 2001:210. See also Malinine [1974].</ref>，後來被[[下埃及|下埃及]]書史使用的世俗體所取代。

== 影響 ==
僧侶體對數個其他文字有影響，最明顯的是其演化而成的世俗體，還有[[麥羅埃文|麥羅埃文]]、[[科普特字母|科普特字母]]（Coptic alphabet）和[[古努比亞字母|古努比亞字母]]（Old Nubian alphabet）。早期[[希伯來字母|希伯來字母]]使用僧侶體數字<ref>Aharoni 1966; Goldwasser 1991.</ref>。
