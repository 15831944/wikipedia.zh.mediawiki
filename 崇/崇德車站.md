'''崇德車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[秀林鄉|秀林鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[北迴線|北迴線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[島式月台|島式月台]]兩座。
*設有多卡通刷卡機。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
*僅有[[臺鐵區間車|區間車]]及北上4043次[[台鐵區間快車|區間快車]]停靠。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1979年2月8日：設站。
*因附近有著名的[[清水斷崖|清水斷崖]]之故，該站景色曾經成為1980年代至1990年代間的國中[[社會課|社會課本]]封面。
*2006年3月10日深夜，由樹林開往花蓮的1073次自強號通過本站時撞上於西正線上進行軌道電路維修之道班工與電務工，導致5人當場殉職，事故後台灣鐵路管理局長[[徐達文|徐達文]]為此請辭下台負責，相關人員遭到連坐處分，一年後於站場後方立紀念碑紀念此事故。
*2016年3月1日：啟用多卡通刷卡機。
