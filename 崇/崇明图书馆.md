'''崇明县图书馆'''位于[[上海市|上海市]][[崇明县|崇明县]][[城桥镇|城桥镇]]崇明大道7897号，是[[国家一级图书馆|国家一级图书馆]]。

==概况==
崇明图书馆的前身是'''崇明私立第一图书馆'''，创建于1918年。在1980年、1984年分两期建造新馆，1998再次建造了人民路128号的新馆。当时位于人民路的图书馆建筑面积7129平方米，其中分馆750平方米，少儿馆646平方米，拥有藏书33万册。<ref>{{Cite web|url =http://www.cmlib.cn/shownews.asp?newsid=556|title =崇明县图书馆基本概况|publisher =崇明图书馆|date =|accessdate =2012-09-03|deadurl =yes|archiveurl =https://web.archive.org/web/20110829145506/http://www.cmlib.cn/shownews.asp?newsid=556|archivedate =2011-08-29}}</ref>2012年，图书馆又迁入了位于崇明大道7897号的新馆，规模再次扩大，藏书53万册，占地面积1.6万平方米，建筑面积1.1万平方米<ref>{{Cite web |url = http://www.cmlib.com.cn/lib/Sub.html#!Module/Article/Type/Show/ColumnId/894ab370-95bb-4bbf-bc26-3bd66369e629/ItemId/83f65690-e7f2-4cee-90c2-bbb22112aa67 |title = 崇明区图书馆基本概况 |publisher = 崇明图书馆 |date = |accessdate = 2012-12-01 }}</ref>
