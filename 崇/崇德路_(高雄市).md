'''崇德路（Chongde Rd.）'''為[[高雄市|高雄市]]的東西向道路之一，全線皆位於[[左營區|左營區]]境內，起端為[[翠華路_(高雄市)|翠華路]]接[[新莊仔路|新莊仔路]]近[[蓮池潭|蓮池潭]]風景區，末端止於崇德路8巷微笑公園前。
== 銜接道路 ==
往西:
* 新莊仔路
* 環潭路
往東:
* 崇德路8巷
* 文奇路110巷22弄

== 主要結點 ==
（由東至西）
* 博愛三路
* 華夏路
* 翠華路

== 近期施工 ==
* [[高雄鐵路地下化|高雄鐵路地下化]]

崇德路平交道是高雄市等待時間最久的平交道，市區鐵路地下化之後，此平交道將已被拆除。
