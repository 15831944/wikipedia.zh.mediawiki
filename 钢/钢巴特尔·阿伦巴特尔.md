'''钢巴特尔·阿伦巴特尔'''（Ganbaatar Ariunbaatar，{{bd|1988年|||}}）[[蒙古国|蒙古国]][[乌兰巴托|乌兰巴托]]人，蒙古国男中音歌唱家。

==生平==
1988年，阿伦巴特尔出生在乌兰巴托以西的游牧部落<ref name="newyorkerprizewinner">{{cite news|last1=Frazier|first1=Ian|title=PRIZE-WINNER|url=http://www.newyorker.com/magazine/2015/11/16/prize-winner-from-mongolia-to-carnegie-hall|accessdate=February 20, 2017|work=The New Yorker|date=November 16, 2015}}</ref>。长大后，他进入[[蒙古文化艺术大学|蒙古文化艺术大学]]学习两年<ref name="newyorkerprizewinner"/>，2010年毕业<ref name="marinskybio">{{cite web|title=ARIUNBAATAR GANBAATAR|url=https://www.mariinsky.ru/en/company/opera_guest/ganbaatar/|website=Marinsky Theatre|accessdate=February 20, 2017}}</ref>。2005年至2010年在蒙古文化艺术大学学习期间，师从O. Ichinkhorloo教授<ref name=alun>{{Cite web |url = http://www.concerthall.com.cn/newpage.php?id=7239  |title = 阿伦巴特尔简介 |author =  |date =  |publisher =广东省星海音乐厅 |language =  |accessdate = 2017-07-10 |quote =  }}</ref>。随后他在乌兰巴托当交通管理员，直至2014年加入[[俄罗斯|俄罗斯]][[乌兰乌德|乌兰乌德]]的{{link-en|布里亚特国立学术歌剧芭蕾舞剧院|Buryat State Academic Opera and Ballet Theatre|布里亚特国立学术歌剧芭蕾舞剧院}}担任独唱演员<ref name="newyorkerprizewinner"/><ref name="marinskybio"/>。

2011年，在[[俄罗斯|俄罗斯]][[莫斯科|莫斯科]]举行的第24届格林卡国际声乐比赛中，他获授“最佳男中音”及“[[帕威尔·利西齐安|帕威尔·利西齐安]]（Pavel Lisitsian）”奖。2012年获蒙古国“最优秀歌唱家”奖。2014年获莫斯科第3届[[穆斯利姆·马戈马耶夫|穆斯利姆·马戈马耶夫]]（Muslim Magomaev）国际声乐比赛第一名<ref name=alun/>。2015年，在第15届[[柴可夫斯基国际音乐比赛|柴可夫斯基国际音乐比赛]]中获得声乐组金奖<ref name="newyorkerprizewinner"/><ref name="marinskybio"/><ref name=alun/>。2016年，获[[蒙古国总统|蒙古国总统]][[查希亚·额勒贝格道尔吉|查希亚·额勒贝格道尔吉]]在[[国家宫_(蒙古国)|国家宫]]颁发[[成吉思汗勋章|成吉思汗勋章]]<ref>{{Cite web |url =http://www.sohu.com/a/117886717_508040  |title =蒙古国功勋演员钢.艾润巴特尔荣获国家最高奖“成吉思汗勋章"总统亲自颁奖  |author =  |date = 2016-11-01 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。在2017年{{link-en|英国广播公司卡迪夫世界歌手大赛|BBC Cardiff Singer of the World competition|英国广播公司卡迪夫世界歌手大赛}}中获得最佳世界歌手奖，演唱的歌曲《Mongoliin tal nutag》荣获最佳歌曲奖<ref>{{Cite web |url = http://www.sohu.com/a/151844264_286345  |title =荣获“世界卡迪夫歌手大赛”最佳歌手奖的艾润巴特尔歌曲欣赏  |author =  |date = 2017-06-25 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。
