Retrieved 01-05-2009.</ref><ref name="ironsky1">[http://www.ironsky.net/info.php Iron Sky info site] {{webarchive|url=https://web.archive.org/web/20090114062938/http://www.ironsky.net/info.php |date=2009-01-14 }}</ref>。于2012年4月4日上映<ref name="ironsky1"/>。本片的[[加長版|導演剪輯版]]於2014年3月11日在DVD和藍光碟上發售，內容加長了約20分鐘。<ref>{{Cite web |url = http://www.psnation.com/2014/03/11/review-iron-sky-directors-cut-blu-raydvd/ |title = Review: Iron Sky: Director's Cut (Blu-ray/DVD) |author = Josh Langford |date = 2014-03-11 |publisher = PS Nation |language = en}}</ref>

2012年10月發售改編電子遊戲《''{{link-en|Iron Sky: Invasion|Iron Sky: Invasion}}''》。续集《{{link-en|钢铁苍穹2：即临种族|Iron Sky: The Coming Race}}》（''Iron Sky: The Coming Race''）于2019年1月16日在芬兰首映。

== 剧情 ==
{{剧情摘要不完整|time=2018-05-23T21:06:38+00:00|small=left}}
2018年，是美国自1970年代以来的那几次登月之后的首次月球登陆。与此同时，纳粹酝酿了70年的入侵正式开始，而整个世界正在踏着正步走向它的毁灭。

二战结束前，[[纳粹|纳粹]]科学家在[[反重力|反重力]]研究中取得了突破性进展。1945年后期，在建于[[南极|南极]]的一个秘密基地中，第一批纳粹宇宙飞船被发射向月球，去在月背建立“Schwarze Sonne ”（黑太阳）军事基地。战败后，最精英的纳粹分子纷纷逃往月球。在那里，科学家不分昼夜地研究新技术、改善武器装备，试图建立起一支强大的[[納粹飛碟|納粹飛碟]]部队，伺机反攻地球。

2018年，美国人再次启动登月计划，将世界上第一位登月的黑人宇航员詹姆斯·华盛顿（克里斯托弗·科尔比）送上月球。这也是美国新上任的女总统（斯黛芬妮·保罗饰演，以[[萨拉·佩林|萨拉·佩林]]为原型）主持下的的形象工程。当华盛顿和他的伙伴一走出太空舱，就遭到了月球纳粹军的袭击。

华盛顿被俘后被带往基地审问。纳粹的现任元首名叫科茨弗莱施（乌多·奇尔），但由于性格软弱，领导大权实际上掌控在军官阿德勒（戈茨·奥托）手中。阿德勒的情人瑞娜特·里切特（茱莉亚·迭泽）是一名单纯善良的教师，脑子里充满了对纳粹主义的美丽憧憬。里切特的父亲是一名科学家（提洛·普鲁克纳）。他检测了华盛顿的[[智能手机|智能手机]]，意识到它是纳粹的宇宙戰艦諸神黃昏（Götterdämmerung）上所缺少的部件。有了这种手机，纳粹反攻地球的计划就能实现。为此，阿德勒给华盛顿注射“白化素”，让他变成了白人。阿德勒带着华盛顿一起来到地球，开始部署他自己的阴谋计划……

== 喜剧元素 ==
作为一部[[黑色幽默|黑色幽默]]喜剧片，影片中多次出现[[黑色幽默|黑色幽默]]，16分钟处，影片模仿了《[[帝国的毁灭|帝国的毁灭]]》中经典镜头（元首的愤怒）。27分处“引用”了《[[奇爱博士|奇爱博士]]》的那只“不听话的[[手|手]]”（左手拼命按住想行纳粹礼的右手）。1小时处的情节讽刺了各国在太空问题上互相欺骗，各自隐瞒（整個[[國際太空站|國際太空站]]其實是各國太空戰艦組成，會席上唯獨同時屬於[[永久中立國|永久中立國]]和電影製作組所在國的[[芬蘭|芬蘭]]沒有參與其中）。面對聯合國艦隊先進武器被壓著打的[[齊格菲|齊格菲]]級流星閃電戰部隊的場景（外型似[[齊柏林飛船|齊柏林飛船]]、比前者大近十倍的太空船），背景音樂撥放的卻是[[華格納|華格納]]歌劇[[尼伯龍根的指環|尼伯龍根的指環]]中著名的[[女武神的騎行|女武神的騎行]]。1小时15分处各国为了争夺月球上的[[氦-3|氦-3]]资源，在[[联合国|联合国]]大会上争吵，到最后各国反目成仇，从各国战舰在月球上互相攻击，到地球上各国之间竟然展开了[[核战争|核战争]]。1小时16分出现向[[美国总统|美国总统]]扔鞋经典场景。
