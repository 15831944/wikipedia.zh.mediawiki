'''都知监'''是明朝内廷管理宦官与宫内事务的“[[二十四衙門|十二监]]”之一。晚明太监[[刘若愚_(明朝)|刘若愚]]著《酌中志》，历述内府各衙门执掌，将该监置于十二监的最末。并说它“甚不显贵”、“极寒苦”、“下下衙门”。

==官制==
*掌印太监，一员；
*佥书，无定员；
*掌司，无定员；
*长随，无定员；
*奉御，无定员。

== 发展 ==
都知监始设于洪武三十年（1398年）七月，在十二监中置立最晚。其发展经历两个阶段，由早期的重要部门到后期的可有可无。
《[[明太祖实录|明太祖实录]]》载：置都知监，秩正四品，掌内府各监方移、一应关支勘合。
清修《明史·职官三》载都知监设官及职掌云：都知监，掌印太监一员，佥书、掌司、长随、奉御无定员。旧掌各监行移、关知（应为支，编者注）、勘合之事，后惟随驾前导警跸。

== 拟名由来 ==
“都知”为宋代内侍职官衔名。宋朝宦官机构分为“内侍省”与“入内待省”。两省均设都都知、左右班都知、押班等官。《[[文献通考|文献通考]]·职官十一》云：“押班次迁副都知，次迁都都都知，遂为内臣之极品”。“都知”是宋朝宦寺本官的最高等级。

== 没落 ==
都知监实为分首重之监（初为[[内官监|内官监]]，后为[[司礼监|司礼监]]）的权力，符合分而治之、相互监督的政治原则，这正是[[朱元璋|朱元璋]]分职设官的一贯原则。

[[明宣宗|明宣宗]]起，司礼监崛起。都知监两项基本职掌（各监文移、关支勘合）被其禠夺，造成地位的沦丧。

[[嘉靖|嘉靖]]中期尽数裁撤镇守内官，造成了对都知监第二次，也是最为致命的打击。彻底沦为清道开路的下役（后惟随驾前导警跸）。
