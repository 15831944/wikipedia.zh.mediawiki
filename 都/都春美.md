'''都春美'''（{{lang-ja|都 はるみ}}，本名'''北村春美'''；{{bd|1948年|2月2日|||Miyako Harumi}}），出生於[[日本|日本]][[京都府|京都府]][[京都市|京都市]]，日本女性[[演歌|演歌]][[歌手|歌手]]，父親為[[日本韓國人|日本韓國人]]<ref>2002年7月4日、NHK教育台放送節目《ETV2002》</ref>，母親為[[日本人|日本人]]。

== 經歷 ==
=== 幼時 ===
受喜愛藝術表演的母親影響，都春美5歲時便被安排學習日本舞蹈及[[芭蕾舞|芭蕾舞]]，之後在6歲的6月6日當天開始接受母親的浪曲及民謠教學。之後青年期就讀洛陽女子高校便有當歌手的打算而中途退學。

=== 歌手期 ===
在1964年，都春美發表了從事歌手的第一張單曲{{nihongo|《真麻煩哪》|困るのことヨ}}。之後在同年的第3張單曲{{nihongo|《島嶼姑娘的山茶花是戀之花》|アンコ椿は恋の花}}獲得好評，並以此獲得第6屆[[日本唱片大獎|日本唱片大獎]]的新人獎。<ref>{{Cite web |url=http://www.tbs.co.jp/recordaward/award/shinjin.html |script-title=ja:過去の 『 最優秀新人賞 』 受賞者一覧 |work=[[TBS電視台|TBS電視台]] |language=ja |accessdate=2012年11月28日 }}</ref>

於1973年都春美將經紀公司轉移到{{nihongo|SunMusic Production|サンミュージックプロダクション}}下，<ref>{{cite book |author= 相澤秀禎 |script-title=ja:人気づくりの法則 |publisher=東洋經濟新報社  |language=ja |isbn=978-449-255-309-1 |date=1988年4月23日 }}</ref>3年後以{{nihongo|《自北方旅店》|北の宿から}}再獲第18屆日本唱片大獎的獎項。<ref>{{Cite web |url=http://www.uta-net.com/user/record_taisyo/index.html#1970 |script-title=ja:年代別日本レコード大賞特集 - 歌ネット |work=Uta-Net |language=ja |accessdate=2012年11月28日 }}</ref>之後以1980年的單曲{{nihongo|《大阪時雨》|大阪しぐれ}}獲得第22屆日本唱片大獎的最優秀歌唱獎後，<ref>{{Cite web |url=http://www.tbs.co.jp/recordaward/award/taishou.html |script-title=ja:過去の 『 レコード大賞 』 受賞者一覧 |work=TBS電視台 |language=ja |accessdate=2012年11月28日 |deadurl=yes |archiveurl=https://web.archive.org/web/20120312071003/http://www.tbs.co.jp/recordaward/award/taishou.html |archivedate=2012年3月12日 }}</ref>成為首位曾在日本唱片大獎獲得新人獎、大獎、最優秀歌唱獎三項榮譽的藝人。

=== 引退 ===
之後在1984年3月，雖然仍處於個人名聲的高峰、都春美對外宣佈想回歸普通女性的生活而從歌壇引退。當時的告別作是在第35屆[[NHK紅白歌合戰|NHK紅白歌合戰]]上演出。

=== 復出 ===
1987年、都春美改以[[音樂製作人|音樂製作人]]的身份重回音樂界擔任數名藝人的事務，並在1989年的第40屆NHK紅白歌合戰上再次登場復出。

自2012年前都春美已發表超過130張的音樂單曲，並曾在2010年春季接受[[紫綬褒章|紫綬褒章]]的榮譽。<ref>{{Cite web |url=http://www.news24.jp/articles/2010/04/28/04158238.html |script-title=ja:春の褒章決まる　紫綬褒章に都はるみさん |work=News24 |language=ja |date=2010年4月28日 |accessdate=2012年11月28日 }}</ref>
