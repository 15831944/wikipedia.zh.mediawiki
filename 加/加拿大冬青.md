'''加拿大冬青'''（學名：''{{lang|la|Ilex verticillata}}''，常見於以本物種製作之[[精油|精油]]的成份列表）<ref name="Nova Scotia Wild Flora" />，又名'''北美冬青'''、'''黑榿木'''<ref name="Nova Scotia Wild Flora">{{cite web |url=http://www.nswildflora.ca/species/Aquifoliaceae/IlexVert/species.html |title=''Ilex verticillata'' (common winterberry) |website=Nova Scotia Wild Flora |archive-url=https://web.archive.org/web/20160303174445/http://www.nswildflora.ca/species/Aquifoliaceae/IlexVert/species.html |archive-date=2016-03-03}}</ref><ref name="Floridata">{{cite web |title=''Ilex verticillata'' |url=https://floridata.com/Plants/Aquifoliaceae/Ilex%20verticillata/870 |website=Floridata |first=Steve |last=Christman |date=2005 |access-date=2018-09-01}}</ref>、'''珊瑚莓''' <ref name="Floridata" />、'''密西根冬青'''<ref name="Floridata" />等名稱，是[[北美洲|北美洲]]東部的一個[[冬青|冬青]]物種。本物種是[[美国|美国]]及[[加拿大|加拿大]]東南部的特有種，分佈於東起[[纽芬兰岛|纽芬兰岛]]以西至[[安大略省|安大略省]]、南至[[明尼蘇達州|明尼蘇達州]]及[[亚拉巴马州|亚拉巴马州]]<ref name="PLANTS">{{PLANTS |symbol=ILVE |taxon=Ilex verticillata |access-date=2011-11-01}}</ref><ref name=grin>{{GRIN | access-date = 2017-12-17 }}</ref>。可藥用或作提煉精油。

本物種一般只見於[[湿地|湿地]]，偶爾也會於沙丘或草地出現。北美冬青的漿果是多種鳥類的重要食物來源，例如[[旅鶇|旅鶇]]<ref name=nymf>{{cite web |title=''Ilex verticillata'' |url=http://nymf.bbg.org/profile_species_tech.asp?id=293 |first=Steven D. |last=Glenn |date=2013 |website=New York Metropolitan Flora Project |publisher=Brooklyn Botanic Garden}}</ref>。此外，現時也常被用作園藝用植物，特別是[[聖誕節|聖誕節]]前後期間。
