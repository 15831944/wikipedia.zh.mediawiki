'''加州州立大學多明格斯山分校'''（'''California State University, Dominguez Hills'''，亦稱：'''CSUDH'''、'''Dominguez Hills'''、'''Cal State Dominguez Hills'''或'''Dominguez Hills State University'''）是[[加利福尼亞州立大學|加利福尼亞州立大學]]系統內、位於美國加利福尼亞州[[卡森_(加利福尼亞州)|卡森]]的一所公立大學，成立於1960年<ref>{{cite web|url=http://degrees.calstate.edu/csu_degree_search2?noCache=437:1358364639 |title=CSU Degrees |publisher=Degrees.calstate.edu |date= |accessdate=October 20, 2013}}</ref>。2019年《[[美國新聞與世界報道|美國新聞與世界報道]]》將其排在“西部地區大學”中的第88位。<ref>{{cite web|title=California State University, Dominguez Hills|url=http://colleges.usnews.rankingsandreviews.com/best-colleges/csudh-1141|publisher=U.S. News and World Report|accessdate=2019-02-18}}</ref>
