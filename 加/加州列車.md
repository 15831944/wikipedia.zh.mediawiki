'''加州列車'''（{{lang-en|Caltrain}}），又稱'''加州鐵道'''、'''加州通勤鐵路'''，為[[美國|美國]][[加州|加州]][[舊金山灣區|舊金山灣區]]的[[通勤列車|通勤列車]]，連接[[舊金山|舊金山]]至[[矽谷|矽谷]]。最北端車站位在[[舊金山|舊金山]]，最南端車站為[[吉爾羅伊_(加利福尼亞州)|吉爾羅伊]]。

== 列车等级 ==
[[File:Caltrain_Numbers.svg|thumb]]
加州列车共有3种列车等级，分别是{{link-ja|各站停车|各駅停車}}列车（Local）、快速列车（Limited，部分车站不停车）和特急列车（即Baby Bullet，停站最少）。加州列车的车次为三位数，首位表明列车等级，后两位为车次序号，车次尾号的奇偶代表运行方向，偶数车次为南行，奇数车次为北行；但部分车种的车次牌只显示后两位数。
* 工作日开行的列车车次首位为1、2或3，其中1字头为各站停车车次，2字头为快速车次，3字头为特急车次。<ref>{{cite web |url=http://www.caltrain.com/schedules/weekdaytimetable.html |title=Weekday Timetable |author=<!--Staff writer(s); no by-line.--> |date=5 October 2014 |publisher=Joint Powers Board |website=Caltrain |accessdate=5 April 2015 }}</ref>
* 周末开行的列车车次首位为4或8，其中4字头为各站停车车次，8字头为周末特急车次。<ref>{{cite web |url=http://www.caltrain.com/schedules/weekend-timetable.html |title=Weekend Timetable |author=<!--Staff writer(s); no by-line.--> |date=5 October 2014 |publisher=Joint Powers Board |website=Caltrain |accessdate=5 April 2015 }}</ref>

==車票種類與收費標準==

在車上沒有出售車票，乘客必須在上車前在車站購買車票，或是使用[[路路通|路路通卡]]刷卡。搭車時如果未持有效的票務證明，將會收到罰單。

車票分以下幾種:<ref>http://www.caltrain.com/Fares/tickettypes.html</ref><ref>http://www.caltrain.com/Fares/farechart.html?PageMode=Print</ref>

*單程票: 收費方式按起站與終站所在區域之間隔收取，共劃分為六區，購買後四個鐘頭內有效。
*一日票: 一日之內可以在購買時所指定的區間內, 不限次數搭乘。
*八次票: 此一票種只能透過路路通卡使用，在購買時所指定的區間不限方向使用8次，自購買後30天內有效。
*月票: 此一票種只能透過路路通卡使用，一個月內可任意不限次數搭乘購買時所指定的區間，在週末與假日可不限區間搭乘。
*跨區間加價票: 此一票種只可以配合單程票、一日票或月票使用。在使用前述三種票時，如欲搭乘超過車票上所指定的區間時，就必須加購此一種車票。
*65歲以上老人、殘障人士、18歲以下兒童票價約是全票的一半。

==未來計劃== 

===加州高速鐵路共用系統===
早期[[加州高速鐵路|加州高速鐵路]]規劃在[[舊金山半島|舊金山半島]]鋪設新的軌道，但後來在與多方民意和地方政府討論後，決定在舊金山到聖荷西之間與加州列車共用軌道<ref>http://www.caltrain.com/projectsplans/CaltrainModernization/BlendedSystem.html</ref>。
在一份運行模擬計劃報告書中提到<ref>http://www.caltrain.com/Assets/Caltrain+Modernization+Program/Documents/Final-Caltrain-California+HSR+Blended+Operations+Analysis.pdf</ref>，在不使用[[待避|列車待避]]的運行方式下，尖峰時刻每小時將可運行六班加州列車與兩班高鐵列車。在利用[[待避|列車待避]]的運行方式時，每小時則運行六班加州列車與四班高鐵列車。

此計劃可大略分為以下幾個部分：
# 將加州列車現代化
# 建設新的高鐵車站 
# 在部分路段新增軌道以利[[列車交會|列車交會]]與[[待避|列車待避]]
# 在舊金山將軌道由現有的終點站延伸至新的舊金山轉運站(Transbay Transit Center)
# 更新平交道，或是平交道分離

===加州列車現代化===

此計劃為加州高速鐵路共用路線計劃的一部份，有三項子計劃：
<ref>http://www.caltrain.com/projectsplans/CaltrainModernization/Modernization.html</ref>
<ref>http://www.caltrain.com/projectsplans/CaltrainModernization/Modernization/PeninsulaCorridorElectrificationProject/Electric-Multiple-Unit-EMU-trains.html</ref>
# 更新號誌與與行車控制系統: 目前正在施工中。預計於2015年秋季完工。
# 在[[舊金山|舊金山]]與[[聖荷西|聖荷西]]之間執行[[電氣化鐵路|鐵路電氣化]]:已經完成[[环境影响评价|環境影響評估]]。預計於2020年完成。
# 將現有之柴油動力列車更換為[[電聯車|電聯車]](EMU): 預計於2020年完成。初期將會把75%之柴油動力列車更新為[[電聯車|電聯車]]，其餘的則視資金與列車的使用年限來更新。

此計劃預計耗資十七億美元，其中七億美金來自選民公投通過的加州高鐵建設公債，其餘的資金則是來自地方政府與聯邦的資助。加州列車現代化之後將可提高運能，縮短列車間距以提高發車頻率，與降低噪音。
