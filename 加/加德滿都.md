'''加德滿都'''（<small>[[尼泊尔语|尼泊尔语]]</small>：'''काठमाडौं, काठमान्डु'''，{{IPA-all|kɑːʈʰ.mɑːɳ.ɖuː|}}）是[[尼泊爾|尼泊爾]]的[[首都|首都]]也是尼泊尔最大的[[城市|城市]]。加德滿都的海拔約1350米，三面環山，市區即於山間的[[加德滿都谷地|加德滿都谷地]]之中，氣候宜人，有“山中天堂”之美誉。

== 名字来历 ==
在加德满都的[[杜巴广场|杜巴广场]]有一个建造于12世纪[[独木庙|独木庙]]（又称“加萨满达木屋”、“木城寺”，英文名称“Kasthamandap”），这个木庙共三层，整栋建造没有任何的铁钉或者钢铁做支撑，而是完全由一根独木做成的，加德满都的名字据说就是来自于这个木庙。

有时候加德满都也被称为“坎提普尔”，“坎提”是神妃[[吉祥天女|吉祥天女]]的别称，“普尔”的意思是神仙居住的地方，所以坎提普尔就表明该地是[[吉祥天女|吉祥天女]]居住的地方。

== 历史 ==
723年，李查继王朝统治者那加玛德瓦大王在这里正式建城，将它命名为坎提普尔，意思是「美丽之城」。

10世纪，[[李查维|李查维]]王朝在此地定都，并以一棵大树的木料修造了一幢三重檐庙式的建筑，取梵文名为“加斯特满达普”（Kasthamandap），转为尼泊尔语为加德满都，意思是"独木寺"。后来以这座建筑为中心逐步修造房舍，向外扩展，加德满都就成为城市的名称。

13世纪初，来自西部的马拉人占领了加德满都河谷，玛拉王朝是尼泊尔历史上最繁荣的时期，艺术、文字和灵魂论都在这一时期取得了伟大的成就，现今保存下来的佛塔、神庙、雕刻，大都建造于这一朝代。

14世纪，[[孟加拉蘇丹國|孟加拉蘇丹國]]的军队攻入加德满都，烧杀抢劫，使城内建筑遭到严重破坏。

15世纪末，马拉王朝分裂，加德满都、帕坦和巴德岗等城市各自成为独立王国，河谷地区形成三足鼎立局面。

1768年，尼泊尔中西部的[[廓尔喀|廓尔喀]]国王攻占了加德满都，建立[[沙阿王朝|沙阿王朝]]，随后统一了整个尼泊尔，从此加德满都就成为全国首都。在沙阿王朝建政不到100年的时候，沙阿国王将他们的政权输给了出身[[拉傑普特人|拉傑普特人]]的拉纳家族，他们将近一个世纪掌握着国家权力。

1951年[[拉纳王朝|拉纳王朝]]被彻底推翻。

== 地理 ==
加德满都市位于[[加德滿都谷地|加德滿都谷地]]的西北部，[[巴格马提河|巴格马提河]]和[[比兴马提河|比兴马提河]]交汇处，全市面积50.67平方千米，平均海拔1350米。

=== 气候 ===
加德满都气候温和，一年中的大部分时间比较舒适宜人。从10月到来年3月，早晨常有浓雾，气温起伏较大。4月份起，气温回升，下午常有阵雨。从6月中旬雨季伴随着夏季来临。白天在30℃-35℃左右，晚上在15℃-20℃之间。

加德满都的[[空气污染|空气污染]]严重，除了工业污染，城区交通拥挤，老旧的车辆排放大量超标尾气之外，寺庙焚烧大量物品也是造成空气污染的一个重要原因，加德满都地处盆地的谷底，四周的高山阻挡了空气流动，加重了污染程度。[[巴格玛蒂河|巴格玛蒂河]]（尼泊尔语：बागमती नदी）是加德满都的圣河，同时也是一条污染严重的河流，河道里布满了生活垃圾與骨灰。

== 教育 ==
市中的[[杜巴高中|杜巴高中]]创立于1892年，是尼泊尔历史最悠久的中学。

成立于1959年的[[特里布文大学|特里布文大学]]位于加德满都南郊，根据在校学生数量和学院的数量计算，该大学是尼泊尔最大的大学。

在加德满都市郊有一所[[加德满都大学|加德满都大学]]，该校建立于1991年1月，是尼泊尔第二大的大学，也是尼泊尔第二古老的大学。
== 交通 ==
* 公路：
** [[中尼公路|中尼公路]]，于1963年6月动工，1967年5月建成通车，这条公路是连接加德满都和[[西藏|西藏]][[拉萨|拉萨]]的通道。
** [[特里布文公路|特里布文公路]]（又名“拉帕公路”），这条公路是将加德满都与外界相连的第一条公路。
* 机场：
** [[特里布万国际机场|特里布万国际机场]]，是尼泊尔唯一的国际机场。
