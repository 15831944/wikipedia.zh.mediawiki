'''加查拉戰役'''是[[第二次世界大戰|第二次世界大戰]][[北非戰場|北非戰場]]中一次極其重要之戰役，戰事從1942年5月26日至6月21日，主要發生在[[利比亞|利比亞]][[圖卜魯格|圖卜魯格]]附近。

== 背景 ==
隨著1941年底{{en-link|十字軍行動|Operation Crusader}}的成功後，英國第8軍團將軸心國軍逐出[[昔蘭尼加|昔蘭尼加]]地區。但到了1942年1月，[[埃尔温·隆美尔|埃尔温·隆美尔]]獲得新到之步兵及坦克增援，有足夠力量橫越利比亞，攻佔[[地中海|地中海]]海岸之圖卜魯格及南面之[[比爾哈凱姆|比爾哈凱姆]]。

== 戰役過程 ==
[[File:GenRitchie.jpg|thumb]]會見屬下指揮官]]
5月27日，隆美尔率領[[德意志非洲军|非洲裝甲軍團]]之主力 - [[德國非洲軍|德國非洲軍]]、[[義大利第20機械化軍|義大利第20機械化軍]]及[[德國第90輕機械化師|德國第90輕機械化師]]; 危險但絕妙地迂迴到盟軍防線南方，因為他相信敵人已經佈下地雷以保衛防線側翼及背後。義大利第10軍在前線攻擊以牽制盟軍，義大利第101機械化師從西面進攻比爾哈凱姆之洼地，同時在隆美尔左翼的義大利第132步兵師則迂迴到側後。當時，防守洼地之[[自由法國|自由法國]]第1旅進行了激烈抵抗，其激烈程度完全超出了隆美尔的想象。而非洲軍發現自己被困於在比爾哈凱姆和圖卜魯格之間的大釜陣地。

隆美尔做了多個步驟解決困難，第一，他向西反攻以打通補給通道，因此他向英軍第150步兵旅進攻以突破包圍。之後，他粉碎了盟軍亳無組織的反擊，親率主力向南將自由法軍逐出比爾哈凱姆。隨後，恢復向東進攻以消滅英軍殘餘部隊。

== 結果 ==
6月14日，[[克勞德·奧金萊克|克勞德·奧金萊克]]下令主力部隊向東撤退至阿拉曼防線，只留下南非第2軍防守圖卜魯格。由於圖卜魯格曾被堅守九個月後在1941年12月於十字軍行動被解救出來，盟國領袖們以為今次亦會一樣。但是，奧金萊克認為盟軍兵力不足以防守圖卜魯格，因此，下令[[艾倫·康寧漢|艾倫·康寧漢]]保存實力向東撤退。

7天之後，6月21日，35,000名盟軍向軸心國軍投降，這是繼[[新加坡|新加坡]]被日軍攻陷後軸心國另一次重大勝利。

加查拉戰役之失敗及圖卜魯格之陷落令康寧漢及奧金萊克被解除指揮權，分別由[[伯納德·勞·蒙哥馬利|伯納德·勞·蒙哥馬利]]及[[哈羅德·亞歷山大|哈羅德·亞歷山大]]代替。

攻陷圖卜魯格之後，非洲裝甲軍團向[[埃及|埃及]]推進，於7月1日在[[第一次阿拉曼戰役|第一次阿拉曼戰役]]中進攻[[阿拉曼|阿拉曼]]防線。
