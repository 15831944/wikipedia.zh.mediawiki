'''加藤 哲郎'''{{BD|1947年|1月16日||}}是一名[[日本|日本]]的政治學家，目前擔任[[一橋大学|一橋大学]]名譽教授、[[早稻田大學|早稻田大學]]客員教授。

== 簡歷 ==
*[[岩手縣|岩手縣]][[盛岡市|盛岡市]]出身。
*[[1970年|1970年]] - [[東京大学|東京大学]]法学部第3類畢業，加入株式会社[[大月書店|大月書店]]。
*[[1978年|1978年]] - 就任[[名古屋大学|名古屋大学]]法学部助手。
*[[1980年|1980年]] - 就任[[一橋大学|一橋大学]]社会学部專任講師。
*[[1981年|1981年]] - 就任一橋大学社会学部助教授。
*[[1989年|1989年]] - 就任一橋大学社会学部教授。
*[[1993年|1993年]] - 以論文「コミンテルンの世界像 ―世界政党の政治学的研究-」（[[共產國際|共產國際]]的世界形象 ―世界政黨的政治學研究-）為題的論文取得名古屋大学法學博士。
*[[1996年|1996年]] - 就任[[德里大學|德里大學]]中国日本研究所客員教授。
*[[1999年|1999年]] - 就任[[墨西哥學院|墨西哥學院]]客員教授。
*[[2000年|2000年]] - 就任一橋大学[[大学院|大学院]]社会学研究科教授
*[[2010年|2010年]] - 就任[[早稻田大學|早稻田大學]]大学院政治学研究科客員教授。一橋大学定年退職、改任名誉教授。

== 人物 ==
* 持續推進以「永続民主主義革命」為名的[[公民社会|公民社会]]建構。
* 對[[丸山真男|丸山真男]]擁有很高評價。
* [[北朝鮮による日本人拉致問題|北朝鮮による日本人拉致問題]]解決を求める[[意見広告|意見広告]]を[[ニューヨーク・タイムズ|ニューヨーク・タイムズ]]に掲載する運動を行なった「[[意見広告7人の会|意見広告7人の会]]」呼びかけ人（残りの6人は[[有田芳生|有田芳生]]、[[勝谷誠彦|勝谷誠彦]]、[[重村智計|重村智計]]、[[高世仁|高世仁]]、[[日垣隆|日垣隆]]、[[湯川れい子|湯川れい子]]）。
* 指導学生包括[[丹野清人|丹野清人]]（首都大学東京教授）、[[堀江孝司|堀江孝司]]（首都大学東京教授）、[[白井聰|白井聰]]（京都精華大学専任講師）等<ref>[https://cakes.mu/posts/11520]cakes</ref>。
