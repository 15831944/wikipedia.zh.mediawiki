'''加影站'''（{{lang-ms|Stesen Kereta Api Kajang}}）位於[[雪兰莪|雪兰莪]][[加影|加影]]，是[[巴生谷捷运|巴生谷]][[双溪毛糯－加影线|双溪毛糯－加影线]]的南端终点站，也是[[KTM通勤铁路|KTM通勤铁路]]的其中一站，距离市中心约1公里。<ref>{{cite web|url=http://www.orientaldaily.com.my/s/63996|title=捷運追蹤（完結篇）：靠近火車軌道 加影站限時段施工|work=東方網 - 馬來西亞東方日報}}</ref>

本站属于[[双溪毛糯－加影线|双溪毛糯－加影线]]第二阶段线路
，于2017年7月17日開始通行。<ref>{{cite news|title=双加捷运17日全面运作‧乘客料每天40万人次|url=http://www.sinchew.com.my/node/1663303|accessdate=2017-07-17|publisher=[[星洲网|星洲网]]|date=2017-07-16|language=zh-hans}}</ref>

== 历史 ==
原先的车站于1897年开放。从20世纪初开始，该车站开始为镇内提供城际列车服务，而在90年代中期引进了KTM通勤铁路服务，当时位于轨道东侧，毗邻武吉路的新车站在完成后便取代位于轨道东侧，毗邻流古路的旧车站。

目前的车站在其服务过程中重建了一些。车站的一个站台被延长以供城际列车停靠，而这些火车偶尔会比车站的站台更长。位于轨道西侧的第二个入口于2004年完成，但没有闸门因当时只提供手动售票服务。2007年上半年，车站安装了更大的站台顶棚。

目前两站无法进行站内转换，乘客需要再次购票不同的车票以前往其他车站。<ref>{{cite web|url=http://www.orientaldaily.com.my/s/205605|title=第一捷運7站　銜接4種公交|work=東方網 - 馬來西亞東方日報}}</ref>
