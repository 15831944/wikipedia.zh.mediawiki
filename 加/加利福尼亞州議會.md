'''加利福尼亚州议会'''（{{lang-en|California State Legislature}}），是[[美国|美国]][[加利福尼亚州|加利福尼亚州]]的立法机构。加利福尼亚州议会为[[两院制|两院制]]，包含[[加利福尼亚州参议院|加利福尼亚州参议院]]（英语：California State Senate）与[[加利福尼亚州众议院|加利福尼亚州众议院]]（英语：California State Assembly）。

加利福尼亚州参议院为[[上议院|上议院]]，共40个席位，每届任期4年；加利福尼亚州众议院为[[下议院|下议院]]，共80个席位，每届任期2年。兩院自1970年以來，一直由民主黨控制。加州州議會的議場位於州首府[[沙加緬度|沙加緬度]]的[[加利福尼亚州議會大廈|加利福尼亚州議會大廈]]。
