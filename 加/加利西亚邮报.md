'''加利西亚邮报'''（{{lang-es|''El Correo Gallego''}}）是西班牙[[加利西亚_(西班牙)|加利西亚]]自治区的一份报纸。它是加利西亚最古老的出版物，也是[[西班牙|西班牙]]第四古老的出版物<ref>{{cite web |title=O Grupo Correo Gallego confianca VITELSA para assisntecia tecnica e manutencao de Correo TV, TDT locais Santiago de Compostela |url=http://www.vitelsa.es/pt/noticias?n=247 |publisher=vitelsa |archiveurl=https://web.archive.org/web/20190110050351/http://www.vitelsa.es/pt/noticias?n=247 |archivedate=2019-01-10 |language=es |date=2015-11-12}}</ref>。
==历史==
《加利西亚邮报》于1878年8月1日创刊于[[费罗尔|费罗尔]]市<ref>{{cite web |title=EL CORREO GALLEGO UN PERIÓDICO, UN PAÍS, UN COMPROMISO|publisher=El Correo Gallego |url=https://web.archive.org/web/20170913082446/http://www.elcorreogallego.es:80/canales/grupo/paxinas/el-correo-gallego.php|accessdate=2017-09-13|language=es}}</ref>。在[[西班牙内战|内战]]期间的1938年，该报总部迁至[[聖地亞哥-德孔波斯特拉|聖地亞哥-德孔波斯特拉]]<ref>{{cite book|author=Laura Callahan|title=Spanish/English Codeswitching in a Written Corpus|url=https://books.google.com/books?id=gUc9AAAAQBAJ&pg=PA92|date=2004-06-24|publisher=John Benjamins Publishing Company|isbn=978-90-272-9537-8|page=92|language=en}}</ref>。[[西班牙民主转型|西班牙民主转型]]后的1984年，它成为加利西亚邮报集团（{{lang|es|Grupo Correo Gallego}}）的一部分。1995年10月，《加利西亚邮报》的[[网站|网站]]正式开通<ref>{{cite web |title=EL CORREO, 140 años de historia del periodismo y entrega a Galicia |url=https://web.archive.org/web/20170906024004/http://www.elcorreogallego.es:80/tendencias/ecg/correo-140-anos-historia-periodismo-entrega-galicia/idEdicion-2017-08-01/idNoticia-1067116 |publisher=El Correo Gallego |accessdate=2017-09-06 |language=es |date=2017-08-01}}</ref>。
