'''加隈亞衣'''（{{lang-ja|加隈亜衣}}，{{bd|1988年|9月9日}}<ref name="birthblood">{{Cite web |url=https://twitter.com/aimausu/status/376735810038407168|title=2013年9月9日付、加隈のツイートより |accessdate=2013-09-09}}</ref>），出身於[[日本|日本]][[福岡縣|福岡縣]]<ref name="mausu blog">{{Cite web |url=http://ameblo.jp/mausu-y/entry-10924568648.html |title=初☆26期Aクラス加隈亜衣｜マウスプロモーション付属養成所ブログ |accessdate=2012-05-13}}</ref><ref name="famitsu">{{Cite web |url=http://www.famitsu.com/news/201201/30009039.html |title=7次元さきの世界に住む少女とコミュニケーションせよ！　ガスト新作『シェルノサージュ ～失われた星へ捧ぐ詩～』発表 - ファミ通.com|accessdate=2012-05-15}}</ref>，[[日本|日本]][[聲優|女性聲優]]，屬於[[Mausu_Promotion|Mausu Promotion]]所属。

== 經歷 ==
Mausu Promotion付屬養成所的第26期生<ref name="mausu blog" />。

從2011年秋天開始在廣播節目和動畫片，遊戲軟體裡演出、2012年4月26日發售的[[PlayStation_Vita|PS Vita]]遊戲、《[[靜籟之空_～獻給失落之星的詩～|靜籟之空 ～獻給失落之星的詩～]]》女主角依恩為首度配音的角色。<ref>デビューに関する記述はこの情報以外にも様々な場で公開されているため、本サイトはあくまでも一例。{{Cite web |url=http://www.famitsu.com/news/201210/07022420.html |title=『シェルノサージュ』トークイベントでチャイナ服イオン公開！　試練編最終幕の歌を歌うのは霜月はるか【マチ★アソビ vol.9】|accessdate=2012-10-09}}</ref>

== 人物 ==
* 出生於福岡縣、講博多方言。其他大阪方言也精通<ref name="famitsu" />。
* 拿手的運動為游泳和乒乓球。其他也擅長電腦打字<ref name="famitsu" />。
