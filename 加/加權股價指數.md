'''加權股價指數'''（簡稱'''加權指數'''、'''TAIEX'''）是由[[臺灣證券交易所|臺灣證券交易所]]所編製的股價指數，是[[台灣|台灣]]最為人熟悉的[[股票指數|股票指數]]，被視為是呈現[[台灣經濟|台灣經濟]]走向的櫥窗<ref>{{Cite web |url=https://ctee.com.tw/News/ViewCateNews.aspx?newsid=143890&cateid=cjzc |title=台股上攻萬點？李瑞倉：正面看待|accessdate=2018-01-056 |date=2017年3月20日 |work=工商時報|language=zh-cn|archiveurl=|archivedate=}}</ref>。

== 背景 ==
發行量加權股價指數係以1966年為基期，基期指數設為100。

== 內容 ==
[[台灣證券交易所|台灣證券交易所]]採用「[[柏謝加權算式|柏謝加權算式]]」（Paasche Formula），與美國[[S&P_500|S&P 500]]的公式相同，是反應整體市場股票價值變動的指標。其係以上市股票之市值當作權數來計算股價指數，採樣樣本為所有掛牌交易中的[[普通股|普通股]]，但初上市股票與全額交割股票除外。

*其計算公式為：
:''' 發行量加權股價指數 =（當期總發行市值 / 基值）× 基期指數 '''<ref>{{Cite web |url=http://www.twse.com.tw/zh/page/products/indices/series.html |title=TWSE 臺灣證券交易所 > 產品與服務 > 指數編製與授權 > 指數系列
|accessdate=2017-06-27 |date=2016年5月31日 |work=台灣證券交易所|language=zh|archiveurl=|archivedate=}}</ref>

當期總發行市值為各採樣股票價格乘以發行股數所得市值之總和，因此，發行公司的市值愈高，占指數的權重就愈高，這就是典型的權值股。發行量加權股價指數係以1966年為基期，基期指數設為100，其採樣樣本除[[特別股|特別股]]、[[全額交割股|全額交割股]]外，其餘[[股票上市|上市]]股票均包括在內。

== 特點 ==
發行量加權股價指數的特色是股本較大的股票對指數的影響會大於股本小的股票，市值高者如[[台積電|台積電]]、[[鴻海|鴻海]]、[[國泰金|國泰金]]及[[大立光|大立光]]、[[台塑|台塑]]三寶等更是其中的重要代表。新上市公司股票納入指數計算，得以當時上市股數為準，且於上市當月過後隔兩月的第一交易日才列入統計。如果成分股進行[[除權|除權]]、[[新股權利證書|新股權利證書]]上市、可轉換[[特別股|特別股]]轉換為[[普通股|普通股]]……等，基值就會進行調整，以維持加權指數之連續性，但除息時則不調整基值，直接扣減，形成「指數蒸發」現象，如果將除息所扣減的指數「還原」回去的話，則成為「臺灣加權總報酬指數」<ref>{{Cite news|url=http://smartold.businessweekly.com.tw/webarticle.php?id=63179 |title=Smart智富比台股會漲的股市很多　跨國投資更能提高報酬|accessdate=2017-06-27 |date=2016-11-28|author=張雍川、游頴鴻 |work=城邦文化事業股份有限公司|language=zh|archiveurl=|archivedate=}}</ref>。

== 指數歷史 ==
[[file:Twse_Histary.png|500px]]
此指數計算採成分股加權制，最高與最低係以交易中出現的極值為準，在1990年2月12日（星期一）盤中，台灣股市因為資產泡沫曾達到12682.41點的歷史高點<ref>{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20170509/1080937/|title=【有片看透透】那些年追的萬點　4次下場是這樣！|accessdate=2017-10-30 |date=2017-05-09 |work=蘋果日報|language=zh-tw|archiveurl=|archivedate=}}</ref>，此記錄迄今仍未打破。

*1986年10月17日，首次站上1000點關卡<ref>{{Cite news|url=http://www.china.com.cn/overseas/txt/2008-09/26/content_16541263.htm |title=台湾股市三十年：风风雨雨造就“经济温度计”|accessdate=2017-06-27 |date=2008-09-26 |work=中国网|language=zh|archiveurl=|archivedate=}}</ref>。
*1989年6月19日，首次站上10000點關卡<ref>{{Cite news |url=http://www.chinatimes.com/newspapers/20170528000076-260202 |title=台股 拚改寫萬點第3長紀錄|accessdate=2017-06-27 |date=2017-05-28|work=中國時報|language=zh|archiveurl=|archivedate=}}</ref>。
*1990年2月12日，指數創下12682.41的歷史最高記錄<ref>{{Cite news|url=http://www.businessweekly.com.tw/article.aspx?id=11953&type=Blog |title=台股攻破萬點！給投資人一句話：高點未知，但現在絕不是撿便宜的時機|accessdate=2017-06-27 |date=2015-04-27|work=商業週刊|language=zh|archiveurl=|archivedate=}}</ref>，但隨後便出現達8個月的大崩盤走勢，同年10月12日，出現2485.25的低點，波段跌幅達10197點。
*1997年7月，指數因電子股、金融股榮景第二度站上萬點。
*2000年2月，指數第三度站上萬點，同年4月初再短暫重返萬點之上，但隨後遭逢[[網際網路泡沫|科技股泡沫]]，再度跌破萬點。
*2001年9月26日，指數跌至3411.68點，為2000年以來最低記錄。
*2015年4月27日，指數第四度站上萬點<ref>{{Cite news|url=https://finance.technews.tw/2015/04/27/tw-stock-10000/|title=睽違 15 年！台股第四度站上萬點，這次因素大不同|accessdate=2017-07-02 |date=2015-04-27|work=財經新報|author=liu milo |language=zh|archiveurl=|archivedate=}}</ref>。
*2017年5月9日，指數第五度站上萬點。
*2017年5月23日，指數本月以來第七度站上萬點、第五度收盤上萬點，並持續史上最長近17個月<ref>{{Cite news|url=https://udn.com/news/story/7251/3159245|title=史上最長的萬點行情 謝金河直指三個重點|author=董俞佳|date=2018-05-24|work=聯合報|accessdate=2018-05-25|language=zh-tw|archiveurl=|archivedate=}}</ref>。
*2017年6月21日，指數到達10379.82點，創下17年最高記錄。
*2017年11月22日，指數到達10882.65點，創下27年最高記錄。
*2018年1月23日，指數到達11270.18點，創下28年最高記錄<ref>{{Cite news|url=https://news.cnyes.com/news/id/4010683|title=【華冠投顧】丞唐講股：台股空方不止、多方不息 - 何丞唐 分析師|accessdate=2018-01-09 |date=2018-01-09|work=鉅亨台北資料中心|author=華冠投顧 |language=zh-tw|archiveurl=|archivedate=}}</ref>。
*2018年2月12日，台股首度在年度封關日以萬點之上收盤（農曆封關日）<ref>{{Cite news|url=http://news.ltn.com.tw/news/business/breakingnews/2171757|title=連66天站上萬點 台股破最長萬點紀錄|accessdate=2017-08-23 |date=2017-08-23|work=自由時報|author=洪友芳 |language=zh-tw|archiveurl=|archivedate=}}</ref>。
*2018年5月23日，台股上萬點滿周年，破最長萬點紀錄。
*2018年10月11日，由于受美国股市大跌拖累與诸多原因影响，創史上最大跌點，跌下万点。

== 其他 ==
臺灣證券交易所除了有編製此指數外，另有第一、二類發行量加權股價指數（1997年7月1日起不再分第一、二類）<ref>{{cite web|url=http://nccur.lib.nccu.edu.tw/bitstream/140.119/34180/6/35350106.pdf|title=第二章 下市緩衝機制  第二節 台灣相關法令概述|accessdate=2010-09-27}}</ref>，及1987年開始計算的不含金融股發行量加權股價指數、2000年開始計算的不含電子股發行量加權股價指數，和產業分類發行量加權股價指數。在1987年，產業分類分為8大類別，1995年增至19類，2007年7月再增至30類。
