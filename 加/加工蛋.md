'''加工蛋'''（{{lang-ja|'''加工卵'''，かこうらん}}）是[[食品工业|食品工业]]和[[餐饮|餐饮]]业使用的[[雞蛋|雞蛋]]的初加工产品。2006年统计的[[日本|日本]]国内鸡蛋出货量中，约20%为加工蛋消费<ref>[http://www.keimei.ne.jp/article/20070625t3.html 社団法人日本養鶏協会調べ。2007年6月25日鶏鳴新聞] {{webarchive|url=https://web.archive.org/web/20070811044719/http://www.keimei.ne.jp/article/20070625t3.html |date=2007年8月11日 }}</ref>。<ref>[http://www.foodmate.net/tech/jiagong/jiagong5/194103.html 中国蛋品加工行业现状及发展方向]</ref><ref>[https://m.sohu.com/a/161879236_233104?_f=m-article_24_feeds_1 我国蛋制品加工率不高 深加工技术亟待提升]，搜狐</ref><ref>[http://news.7wsh.com/zixun/122098.html 机械化进程加速将带动蛋制品发展]</ref><ref name=jilin>{{cite news|title=吉林金翼蛋品的供给侧改革：小鸡蛋也有大“买卖”|url=http://china.chinadaily.com.cn/2017-06/23/content_29864379.htm|accessdate=2017-11-03|agency=新华社|publisher=中国日报中文网|date=2017-06-23}}</ref>

== 液态蛋 ==
液态蛋<ref>{{cite book|author1=迟玉杰|title=蛋制品加工技术|date=2009年1月1日|publisher=中国轻工业出版社|isbn=9787501966707|pages=320|url=https://books.google.com/books?id=-bSBDQAAQBAJ&pg=PT112&lpg=PT112|accessdate=2017-11-03}}</ref>（{{lang|ja|液卵}}）是将鸡蛋去除[[雞蛋|卵殻]]，只保留内容物。保留蛋黄形状的被称为全形蛋（{{lang|ja|ホール液卵}}），[[蛋黃|蛋黃]]与[[蛋白|蛋白]]被均匀混合的被称为「{{lang|ja|全液卵}}」或「{{lang|ja|液状全卵}}」，卵黄与卵白被分离的称为「{{lang|ja|卵黄液}}」「{{lang|ja|卵白液}}」或「{{lang|ja|液状卵黄}}」「{{lang|ja|液状卵白}}」。有些制品还添加糖或盐。[[蛋白质|蛋白质]]因遇热变性特质不能进行加热杀菌，对其[[沙門氏菌屬|沙门氏菌]]和[[大腸桿菌|大腸桿菌]]的[[殺菌|殺菌]]需采用[[巴斯德消毒法|低温保持殺菌法]]。食品加工行业中广泛将液态蛋用于[[蛋黃醬|蛋黃醬]]制造、[[餜子|製菓]]、{{tsl|ja|製パン||面包}}、{{tsl|ja|製麺|面条生产|面条}}等[[消费品|消费品]]的生产。蛋清液也被用于制作[[膠囊劑|膠囊劑]]。

== 冰冻蛋 ==
冰冻蛋是采用-30[[℃|℃]]以下的温度急速冷冻的液态蛋。其保存和流转均在-18℃或更低温度下进行。因会发生蛋清[[黏度|粘度]]降低、[[泡沫|起泡性]]降低、卵黄[[凝胶|凝胶]]化等冷冻变性情况，冷冻前会加入约10%的[[蔗糖|蔗糖]]或3~5%的[[食盐|食盐]]来防止蛋黄凝胶化。

== 干燥蛋 ==
干燥蛋（{{lang|ja|乾燥卵}}）是将液态蛋去除水分后制成粉末状或薄片。在工业上广泛使用{{tsl|ja|噴霧乾燥}}法。在蛋白粉生产中，因蛋白中所含[[葡萄糖|葡萄糖]]在保存中会引起变色等问题，需先进行脱糖处理。除蛋白粉外，亦有根据需要生产蛋黄粉、全蛋粉等<ref name=jilin />。

== 浓缩蛋 ==
鸡蛋全蛋約76%、蛋清约88%、蛋黄約48%为所含水分，浓缩蛋（{{lang-ja|濃縮卵}}）是采用反[[渗透压|渗透压]]法、限制[[过滤|过滤]]法和全蛋加热减压法来降低蛋清中的所含水分。加热減圧法会加温到60℃，以及添加蔗糖来防止蛋白质的热变性。

== 二次加工 ==

=== 微波加工蛋 ===
{{tsl|ja|マイクロ波加熱|采用微波加热}}使其膨胀、干燥。产品有助于长期保存，放入热水中可以使其在短时间内复原。主要用于[[杯麵|杯麵]]{{tsl|ja|加薬||配菜}}、[[拌飯素|拌飯素]]。

=== 筒状加工蛋 ===
将蛋液均匀涂抹在加热的圆形{{tsl|ja|ドラム缶||铁桶}}内壁，可以制成薄片状鸡蛋。主要用于{{tsl|ja|冷やし中華|中华冷面}}和[[散壽司|樱花寿司]]。有时会将液态[[小麥麵粉|小麦粉]]和[[黄油|黄油]]加入来制成[[可麗餅|可麗餅]]。

=== 长蛋 ===
长蛋又名卷蛋<ref>《蛋制品加工技术》</ref>。形似[[金太郎飴|金太郎飴]]的长条形[[水煮蛋|水煮蛋]]自[[1970年代|1970年代]]起在日本就有生产。生产过程是将蛋白液放在金属管外，将其加热凝固。然后将蛋黄液注入管状的蛋清体中，再次加热使其凝固。然后将其[[真空包装|真空包装]]并进行加热杀菌。蛋白与蛋黄的比例为62%:38%，这与全蛋中蛋白与蛋黄的比例基本相同。这种产品主要用于在[[便利店|便利店]]等地以片状加入所销售的[[沙律|沙律]]、[[面条|面条]]等食品中。
