'''超時工作'''（{{lang-en|'''Overtime'''}}，{{lang|en|'''OT'''}}），俗稱'''加班'''，泛指超過「正常工作時間」以外的延長[[工作時數|工作時數]]的行為。

所謂「正常工作時間」，可以間根據以下4種方式來決定：
# 依照健康考慮或合理性。
# 依照所屬行業的約定俗成。
# 通過法例的規範。
# 通過雇主和雇員之間的協定。

== 各地情況 ==

=== {{ROC-TW}} ===
2011年4月一份由台灣[[人力銀行|人力銀行]]業者[[yes123求職網|yes123求職網]]所公佈的網路[[問卷調查|問卷調查]]顯示，在受訪的2140位[[上班族|上班族]]中，有63.9%每天工作超過8小時、每天平均只睡6.2小時，同時也有近7成上班族表示加班也領不到加班費，上班族自嘆「爆肝換[[工作貧窮|窮忙]]」<ref>{{cite news| title =錢少事多睡不飽 上班族怨「爆肝換窮忙」| date =2011-04-30| url =http://life.chinatimes.com/2009Cti/Channel/Life/life-article/0,5047,130518+132011043000835,00.html| work =[[中廣新聞|中廣新聞]]| accessdate =2011-05-04}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

=== {{HKG}} ===
[[香港中文大學亞太研究所|香港中文大學亞太研究所]]聯同香港中文大學社工系，在2010年11月以電話訪問1000名18歲以上全職人士，結果發現有4成受訪者表示工作時間偏長，平均每日工作10小時或以上，近1成受訪者每日工作更逾14小時。調查更建議政府及私營機構，應推廣五天工作及減少員工超時工作<ref>{{cite news| title =四成僱員日做逾十小時| date =2011-03-26| url =http://hk.news.yahoo.com/article/110325/3/nghe.html| work =星島日報| accessdate =2011-05-04}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

香港向來以工時長見稱：過去幾十年，香港平均工時達每周48小時，即一星期有六天工作8小時或五天工作9.6小時。在香港，工時長跟勤工被畫上等號。不少管理層至今也認為加班是升職的要素，甚至拒絕加班是懶散的表現，以致勞工的工時不斷延長。另外，在97金融風暴之後，無償加班變成了理所當然的事，同意員工加班工半（即加班費為時薪1.5倍）的老闆反過來變成稀有動物。有核數師表示，[http://topick.hket.com/article/499888/%E5%B0%88%E6%A5%AD%E4%BA%BA%E5%A3%AB%E5%8B%81OT%20%E6%AF%8F%E5%91%A8%E5%8A%A0%E7%8F%AD40%E5%B0%8F%E6%99%82 旺季時每周超時工作逾40小時，沒薪金補償]

近年香港醫護人員工時已有所減少，每星期工時大約55小時，雖然比一般文職人員略長，但相比其他勞動行業如保安人員、飲食業從業員為低。香港大型醫療機構，包括醫院管理局都有為醫護人員提供超時津貼。

[https://web.archive.org/web/20150417074106/http://www.singpao.com/xw/gat/201501/t20150124_546293.html 2015-01-24香港成報網報導], 標準工時委員會顧問公司報告推算，接近73萬名僱員每周超時工作五小時，其中超過七成屬無償加班。

對於有僱主視員工超時工作是理所當然，香港政府至今2015年亦未立法確立標準工或就無償加班問題加強規管。理大社會政策研究中心主任鍾劍指，這是香港環境競爭大，以致社會風氣不健康。大家均認為超時工作是拼搏表現，大多數僱主沒提供薪金或假期補償，而在這種氣氛下，令準時下班的人感到壓力。

== 各地的常見相關限制 ==
加班時間的工資要加成計算，除了因為加班時員工身體負擔會大幅提升外，這樣可以讓企業傾向以增聘員工應對高業務量，減少高失業率高工時問題。

=== {{ROC-TW}} ===
# 延長勞工之工時連同正常工時，一日不得超過12小時。延長之工作時間，一個月不得超過46小時。
# 除了[[責任制|責任制]]以外，延長工時的加班行為，都必需額外計算加班費的薪資，或進行[[補休|補休]]。但實際上台灣受僱者大多被迫使用責任制，甚至有[[無薪假|無薪假]]、責任制一起實施的現象；而補休則常常是期限短，過期後就不給補休也不給加班費，且少有公司有補休加成計算的規定。

== 對健康的影響 ==

雖然超時工作帶來的健康的影響，在短期沒有明顯危害，但若長期不予理會，有可能加劇病情，影響健康和工作能力。超時工作除了本身可增加壓力以外，部分人因為長時間工作，導致食無定時，加上缺乏[[運動|運動]]，間接容易引致各類[[腸胃疾病|腸胃疾病]]及增加罹患[[心血管疾病|心血管疾病]]的風險<ref>{{cite news| title =嚴分輕重 五級制加班隱疾| date =2011-05-02| url =http://hk.news.yahoo.com/article/110501/3/o3b1.html| work =星島日報| accessdate =2011-05-04| deadurl =yes| archiveurl =https://web.archive.org/web/20110508140953/http://hk.news.yahoo.com/article/110501/3/o3b1.html| archivedate =2011-05-08}}</ref>。
