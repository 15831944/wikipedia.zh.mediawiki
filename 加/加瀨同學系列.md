{{日本作品|加瀨同學系列|加瀬さんシリーズ}}是[[日本|日本]][[漫畫家|漫畫家]][[高嶋博美|高嶋博美]]起先以《'''牽牛花與加瀨同學。'''》（{{lang-ja|あさがおと加瀬さん。}}）為開端的系列漫畫統稱。

== 作品介紹 ==
2010年8月開始在[[新書館|新書館]]刊行的[[百合_(類型)|百合]]漫畫集《Hirari、（{{lang|ja|ひらり、}}）》上連載，後來在2014年7月因該刊物休刊，經過漫畫網站「FLASH WINGS」之後，轉移到《{{link-ja|Wings (雜誌)|ウィングス (雑誌)|Wings}}》繼續連載。

2017年3月23日，於影音分享網站[[YouTube|YouTube]]先行公開《牽牛花與加瀨同學。》的官方動畫宣傳短片<ref>{{Cite web|url=http://asagao-anime.com/news/archives/20|script-title=ja:「あさがおと加瀬さん。」アニメーションクリップ制作決定|publisher=「牽牛花與加瀨同學。」動畫官方網站|accessdate=2017-03-25|language=ja}}</ref>，同年5月7日以《妳的光芒（{{lang|ja|キミノヒカリ}}）》作為另一部動畫短片的標題播出<ref>{{Cite web|url=https://twitter.com/asagao_anime/status/861128476165668865|script-title=ja:【あさがおと加瀬さん。】 アニメーションクリップ「キミノヒカリ」|publisher=「牽牛花與加瀨同學。」動畫官方[[Twitter|Twitter]]|accessdate=2017-05-09|language=ja}}</ref>，該短片後來收錄在同年7月25日發行的單行本《圍裙與加瀨同學。》特裝版當中。

2017年10月宣布新作動畫以[[OVA|OVA]]形式在劇院上映，2018年6月9日在日本公開<ref>{{Cite web|url=https://twitter.com/asagao_anime/status/914777060987502592|script-title=ja:【新作】2018年初夏に「あさがおと加瀬さん。」OVAの期間限定劇場上映が決定いたしました！|publisher=「牽牛花與加瀨同學。」動畫官方Twitter|accessdate=2017-10-02|language=ja}}</ref>。

== 故事簡介 ==
{{expand-section}}
主角'''山田結衣'''是位個性內向、喜歡[[植物|植物]]的高中2年級學生，班上的園藝股長。

在山田她隔壁班的'''加瀨同學'''是[[田徑|田徑]]隊的王牌及校內風雲人物。

某一天，山田與加瀨她們2人因山田種的[[牽牛花|牽牛花]]而相識，並成為好朋友，而后逐渐开始交往。

== 登場人物 ==
以下角色的聲優均為OVA版聲優。
; {{nihongo|山田結衣|山田 結衣（やまだ ゆい）|聲：[[高橋未奈美|高橋未奈美]]}}
: 高中二年級學生，性格內向，對綠化很感興趣，同時也是學校的綠化委員，在放學後以及休息時都在花壇除草。
: 除了家人以外別人都以「山田」來稱呼她。
: 唱歌與運動都不擅長。

; {{nihongo|加瀨友香|加瀬 友香（かせ ともか）|聲：[[佐倉綾音|佐倉綾音]]}}
: 與山田結衣同校的高二學生，生日為5月1日，星座為[[金牛座|金牛座]]。
: 學校田徑部的王牌人物，本人在學校十分受女生們的歡迎。

; {{nihongo|三河|三河（みかわ）|聲：[[木戶衣吹|木戶衣吹]]}}
: 山田結衣的好友，山田稱呼她為“三河親”（{{lang|ja|三河っち}}），對山田和加瀨的關係十分感興趣。

; {{nihongo|井上茜|井上茜(いのうえ あかね)|声：[[壽美菜子|壽美菜子]]}}
: 高三學生，隸屬于田徑部，由於三河的言語而誤認為是加瀨的前女友。
