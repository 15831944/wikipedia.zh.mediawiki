'''加尔各答地铁'''是位于[[印度|印度]][[西孟加拉邦|西孟加拉邦]][[加尔各答|加尔各答]]的[[地铁|地铁]]系统，自1984年开始正式运营。除了正在运营的一号线外，尚有5条线路正在建设。其一号线是印度第一条地铁。<ref name="KOLKATA METRO – THE 1ST METRO SYSTEM IN INDIA">{{cite web | url=http://kmrc.in/overview.php | title=KOLKATA METRO – THE 1ST METRO SYSTEM IN INDIA | accessdate=2013-04-06}}</ref>
==历史==
印度独立之后，加尔各答市内的交通状况唤醒了城市规划者、[[西孟加拉邦|西孟加拉邦]]政府及[[印度政府|印度政府]]的重视。试图增加现有公交系统车辆的努力被证明是无用功，因为加尔各答地面上道路面积仅占5%，而在[[德里|德里]]道路面积已达21%。<ref name="THREE CHEERS">{{cite web | url=http://www.telegraphindia.com/1111027/jsp/opinion/story_14666108.jsp | title=THREE CHEERS | publisher=The Telegraph | accessdate=2013-04-07}}</ref><ref name="TRANSPORTATION">{{cite web | url=http://dda.org.in/planning/docs/12_Transportation.pdf | title=TRANSPORTATION | accessdate=2013-04-07}}</ref><ref name="History">{{cite web | url=http://www.mtp.indianrailways.gov.in/view_section.jsp?lang=0&id=0,1,283 | title=History | accessdate=2013-04-10}}</ref>

为了寻找其他的解决之道，在1969年成立了都市交通工程局，而该局则推荐当地政府考虑[[大众捷运系统|大众捷运系统]]。该局准备了一份自1971年开始在该市建立5条地铁线路的总计划，全长97.5公里。这其中首先要修建的，就是繁忙的南北轴线地铁，全长25公里。<ref name="Kolkata Metro Website">{{cite web|title=Kolkata Metro Website|url=http://www.mtp.indianrailways.gov.in/view_section.jsp?lang=0&id=0,1,283 |publisher=Kolkata Metro Authority}}</ref>这条线路于1972年6月1日通过审批，1972年12月29日由时任[[印度总理|印度总理]]的[[英迪拉·甘地|英迪拉·甘地]]奠基。<ref name="METRO RAILWAY KOLKATA">{{cite web | url=http://www.urbanindia.nic.in/programme/ut/Metro_Kolkat_PPT.pdf | title=METRO RAILWAY KOLKATA  PRESENTATION  FOR  ADVISORY BOARD OF METRO RAILWAYS ON 29.6.2012 | accessdate=2013-04-06 | deadurl=yes | archiveurl=https://web.archive.org/web/20130628183415/http://urbanindia.nic.in/programme/ut/Metro_Kolkat_PPT.pdf | archivedate=2013-06-28 }}</ref>

在最早的总计划中，一共规畫了五条线路。其中有三条被选中进行施工，但这其中只有一号线真正建成。<ref name="KOLKATA METRO – THE 1ST METRO SYSTEM IN INDIA">{{cite web | url=http://kmrc.in/overview.php | title=KOLKATA METRO – THE 1ST METRO SYSTEM IN INDIA | accessdate=2013-04-06}}</ref>

自开工之日起该工程就遭遇了种种的障碍，直到1978年才真正开始动工。最早开通的一段未与线路所设计的两个车库相连，所以只能单线运营，两年以后才修到南边的车库，开始复线运营。1995年 Dum Dum 和 Tollygunge 之间的线路开始运营，才实现全线运营。<ref name="History">{{cite web | url=http://www.mtp.indianrailways.gov.in/view_section.jsp?lang=0&id=0,1,283 | title=History | accessdate=2013-04-10}}</ref>

=== 1號線 ===
[[File:Shahid_Khudiram_Metro_Station.jpg|thumb]]

加尔各答地铁1號線共有24座車站運營中，其中15座位於地下，2座位於地面，7座為高架站。所有地下車站（Park Street站除外）均設[[島式月台|島式月台]]；所有高架車站（Noapara站除外）均設[[側式月台|側式月台]]。Noapara站將會成為1、4號線的轉乘站，因此設有島式月台。1號線平均站距為1.14公里。由於1號線使用第三軌供電， Jatin Das Park, Central和 Shyambazar站設有變電站。

目前加尔各答地铁的乘客可於1號線的Dum Dum, Rabindra Sarobar 及 Kavi Subhash 站換乘[[加爾各答郊區鐵路|加爾各答郊區鐵路]]。

=== 2號線（建設中）===
加尔各答地铁2號線（東西走廊）共有12座車站，並以地下隧道穿越[[胡格利河|胡格利河]]。2號線工程造價為₹4874.6 [[克若爾|克若爾]]（7.5億美元），原規劃路線全長14.67公里(8.9公里位於地底，5.77公里為架空路段)。2號線於2009年3月動工建設，但工程屢次因為土地徵收及貧民窟遷移問題而停頓。2016年6月，2號線的走向有所變動，使線路總長度增至16.55公里（10.81公里位於地下，5.77公里為架空路段）。工程延誤和外匯虧損使工程造價上升80%至₹9000[[克若爾|克若爾]]（11.5億美元）。

2號線一期 （[[比丹納加爾|比丹納加爾]]Sector V 至Sealdah: 全長9公里）預計於2018年6月啟用，二期 （Sealdah至Howrah Maidan: 全長7公里）則預計於2019年8月啟用。<ref name=":0">{{Cite web|url=http://timesofindia.indiatimes.com/city/kolkata/East-West-Metro-project-cost-to-shoot-up-to-nearly-Rs-9000-cr/articleshow/52590594.cms|title=East West Metro project cost to shoot up to nearly Rs 9,000 cr - Times of India|access-date=2016-09-25}}</ref>
2號線將使用[[標準軌距|標準軌距]]，每列列車將有6卡車廂，並設有空調系統。所有車站月台均會設月台幕門。

=== 3號線（建設中）===
加尔各答地铁3號線將連接加爾各答的極南端和市中心（[[B._B._D._Bagh|B. B. D. Bagh]]），全長16.72公里。

== 交通接駁 ==
目前加尔各答地铁的乘客可於1號線的Dum Dum, Rabindra Sarobar 及 Kavi Subhash 站轉乘[[加爾各答郊區鐵路|加爾各答郊區鐵路]]。

建設中的3號線將會途經加爾各答循環鐵路的Majerhat火車站；建設中的4號線和6號線則會於[[內塔吉·蘇巴斯·錢德拉·鮑斯國際機場|內塔吉·蘇巴斯·錢德拉·鮑斯國際機場]]設站。


== 运营 ==
[[Image:New_Garia_Metro_Station.jpg|thumb]]
加尔各答地铁的每节车厢都有为女性和老人预留的座位，2012年也在地下隧道部分引入了手机信号。<ref name="মেট্রো রেলে এবার মোবাইল-সুবিধা">{{cite web|author=|url=http://www.abpananda.newsbullet.in/kolkata/59-more/26003-2012-07-23-13-35-58|title=রেলে এবার মোবাইল-সুবিধা|publisher=|date=|accessdate=2012-07-24|archive-url=https://web.archive.org/web/20120725050655/http://abpananda.newsbullet.in/kolkata/59-more/26003-2012-07-23-13-35-58#|archive-date=2012-07-25|dead-url=yes}}</ref>大部分地铁车站都没有空调，只有电扇。<ref name="Durga Puja bonanza: Enjoy AC Metro ride up to New Garia">{{cite web | url=http://articles.timesofindia.indiatimes.com/2010-07-28/kolkata/28292580_1_rake-metro-passengers-kavi-nazrul | title=Durga Puja bonanza: Enjoy AC Metro ride up to New Garia | accessdate=2013-04-06}}</ref>

=== 运营时间 ===
该地铁自早上6點45分运营到晚上9点55分。高峰时间间隔为5分钟（高峰自早上八点半运营到中午十二点，再从下午四点半到晚上七点半），平峰为10分钟一班。周日运营時間為早上9点45分至晚上9点55分，间隔15分钟。<ref name="KOLKATA">{{cite web|url=http://urbanrail.net/as/kolk/kolkata.htm |title=KOLKATA |accessdate=2013-04-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20110629132602/http://urbanrail.net/as/kolk/kolkata.htm |archivedate=2011-06-29 }}</ref>

=== 车票===
[[File:Kolkata_metro_ticket.jpg|right]]
加尔各答地铁可以使用[[智慧卡|智慧卡]]购票。车票从{{INR}}4到{{INR}}12不等，<ref name="Fare Chart">{{cite web | url=http://www.mtp.indianrailways.gov.in/view_section.jsp?lang=0&id=0,2,396 | title=Fare Chart | accessdate=2013-04-10}}</ref>是该市最便宜的交通方式。<ref name="Expect a Metro fare hike in January">{{cite web | url=http://articles.timesofindia.indiatimes.com/2012-12-25/kolkata/35998931_1_metro-fare-hike-delhi-metro-minimum-fare| title=Expect a Metro fare hike in January | accessdate=2013-04-06}}</ref>

=== 安全===
