'''加拿大的簽證政策'''主要取決於[[加拿大|加拿大]]與各國的外交關係。<ref>{{cite web|url=http://www.canadaimmigrationvisa.com/visatype.html|title=加拿大移民政策、規則和規定（Canadian Immigration Policy, Rules and regulations）|year=2009|publisher=Canada Immigration Visa H1 solution Inc|accessdate=2011-05-02}}</ref> 許多[[英聯邦|英聯邦]]國家的人民在需要進入加拿大國境時，都能獲得免[[簽證|簽證]]的待遇。一般而言，被要求簽證的國家通常都與加拿大有[[移民|移民]]或[[外交|外交]]方面的問題（例如：[[非法入境|非法入境]]）。
从2016年起，仿效[[美国|美国]][[旅游许可电子系统|旅游许可电子系统]]，加拿大也推出类似的[[加拿大旅游许可电子系统|eTA]](electronic travel authorization)计划，针对入境加拿大免签的国家。<ref>[http://www.cic.gc.ca/english/department/acts-regulations/forward-regulatory-plan/eta.asp Electronic Travel Authorization (eTA)] {{webarchive|url=https://web.archive.org/web/20150320235357/http://www.cic.gc.ca/english/department/acts-regulations/forward-regulatory-plan/eta.asp |date=2015-03-20 }}</ref>

==簽證豁免國家及地區列表之變更==
{{OftenUpdate}}
===重啟沙特阿拉伯簽證需求===

2002年9月5日，重啟前往加拿大旅遊的沙地阿拉伯公民的簽證限制，因為「沙特阿拉伯沒有表明其具有意願或其擁有基礎措施拒絕恐怖分子、犯罪分子或其他被拒入境者使用其護照」。<ref>{{cite web |url=http://www.gazette.gc.ca/archives/p2/2002/2002-09-25/html/sor-dors326-eng.html |title=加拿大公報 |publisher=Gazette.gc.ca |date=2010-09-29 |accessdate=2013-01-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20130523075830/http://www.gazette.gc.ca/archives/p2/2002/2002-09-25/html/sor-dors326-eng.html |archivedate=2013-05-23 }}</ref>

===重啟馬來西亞簽證需求===

2002年9月24日，重啟前往加拿大旅遊的馬來西亞公民的簽證限制，因為「馬來西亞護照和護照系統容易被濫用」。<ref>{{cite web|url=http://canadagazette.gc.ca/archives/p2/2002/2002-10-09/html/sor-dors332-eng.html|title=加拿大公報|publisher=加拿大公報|date=2010-09-29|accessdate=2013-01-05}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web|url=http://www.malaysiakini.com/news/13085 |title=加拿大對馬來西亞人實施簽證要求（Canada imposes visa requirements on Malaysians） |publisher=Malaysiakini |date=2002-09-23 |accessdate=2013-01-05}}</ref>

===重啟哥斯達黎加簽證需求===

2004年5月11日，重啟前往加拿大旅遊的哥斯達黎加公民的簽證限制，因為「哥斯達黎加籍人士進入加拿大申請難民保護或將加拿大作為非法進入美國過境點的數量持續增加」，並且「哥斯達黎加檔案被該國籍人士和鄰近國家濫用的案例有增加」。<ref>{{cite web |url=http://gazette.gc.ca/archives/p2/2004/2004-05-19/html/sor-dors111-eng.html |title=加拿大公報 |publisher=Gazette.gc.ca |date=2010-07-08 |accessdate=2013-01-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20130523072732/http://www.gazette.gc.ca/archives/p2/2004/2004-05-19/html/sor-dors111-eng.html |archivedate=2013-05-23 }}</ref>

===啟動克羅地亞公民簽證豁免===

2009年3月26日，對於克羅地亞公民進入加拿大旅遊的簽證要求被取消，因為「克羅地亞非法移民以及簽證拒簽率在過去5年持續穩步下降，並且申請難民和遣返的數量維持低水平」。<ref>{{cite web |url=http://www.gazette.gc.ca/rp-pr/p2/2009/2009-04-15/html/sor-dors105-eng.html |title=檔案-加拿大公報-移民和難民保護法規修正（臨時對克羅地亞國籍人士豁免簽證） |publisher=Gazette.gc.ca |date=2012-11-09 |accessdate=2013-01-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20130523061634/http://www.gazette.gc.ca/rp-pr/p2/2009/2009-04-15/html/sor-dors105-eng.html |archivedate=2013-05-23 }}</ref>

===<del>重啟墨西哥簽證需求</del>===

2009年7月13日，重啟對墨西哥公民進入加拿大旅遊的簽證要求，因為以下三個主要原因：<ref>[http://www.gazette.gc.ca/rp-pr/p2/2009/2009-08-05/html/sor-dors207-eng.html5 ]{{dead link|date=January 2013}}</ref>

*墨西哥籍人士的難民申請從2005年的3,500例上升到2008年的9,500例；
*在過去3年，非法移民的數量持續穩步上升；
*有關旅遊證件、組織犯罪和腐敗的風險；

2016年12月1日再次免除簽證要求。

===重啟捷克簽證需求===
加拿大公民可以免簽證進入[[申根地區|申根地區]]，其中包括[[捷克共和國|捷克共和國]]。當捷克共和國在2004年與其他9個國家加入[[歐盟|歐盟]]時，歐盟與加拿大政府曾展開對話希望解除這些國家的公民簽證要求，以獲得所有申根國家和加拿大之間的簽證互惠。其結果是捷克公民的簽證要求於2007年10月被取消。<ref>[http://eur-lex.europa.eu/LexUriServ/LexUriServ.do?uri=CELEX:52008DC0486R%2801%29:EN:NOT 第四份簽證互惠報告（Fourth Visa reciprocity report）]歐盟</ref> 

然而，2009年7月16日，加拿大重新啟動對捷克公民的簽證要求因為很多[[羅姆人|羅姆人]]進行政治避難申請，導致逾期居留的比率上升至很高的水平。<ref>{{cite web|url=http://www.cic.gc.ca/english/department/media/releases/2009/2009-07-13a.asp |title=新聞發布-加拿大實施對捷克公民的簽證要求 |publisher=Cic.gc.ca |date= |accessdate=2013-01-05}}</ref><ref>[http://www.euractiv.com/en/justice/prague-seeks-eu-solidarity-visa-row-canada/article-184099 歐洲議會議員要求加拿大解除對所有歐盟公民的簽證要求] Euractive.com</ref>歐盟敦促加拿大「這種非常令人遺憾的情況應盡快結束」。<ref>[http://europa.eu/rapid/pressReleasesAction.do?reference=MEMO/09/466&format=HTML&aged=0&language=EN&guiLanguage=en 第五份簽證互惠報告]歐盟</ref>簽證要求在2013年11月被取消。

===啟動台灣簽證豁免需求===
自2010年11月22日起，擁有個人身份證件的[[中華民國護照|中華民國護照]]持有者可以無需簽證進入加拿大，因為「中華民國護照持有者的簽證拒簽率和非法移民、遣返和申請政治庇護的數量很低」。<ref>{{cite web |url=http://www.gazette.gc.ca/rp-pr/p2/2010/2010-12-08/html/sor-dors265-eng.html |title=檔案-加拿大公報-Regulations; Parliament; Law; Instruments; Media; Regulation; Governor General; Cabinet; Information |publisher=Gazette.gc.ca |date=2012-11-09 |accessdate=2013-01-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20130523055529/http://www.gazette.gc.ca/rp-pr/p2/2010/2010-12-08/html/sor-dors265-eng.html |archivedate=2013-05-23 }}</ref>

===重啟五國簽證需求===
2012年9月13日，[[博茨瓦納|博茨瓦納]]，[[納米比亞|納米比亞]]，[[聖盧西亞|聖盧西亞]]，[[聖文森特和格林納丁斯|聖文森特和格林納丁斯]]以及[[斯威士蘭|斯威士蘭]]被從簽證豁免國家名單中刪除。因此，該五國公民都必須事先獲得簽證以前往或過境加拿大。<ref>{{cite web|url=http://www.cic.gc.ca/english/department/media/releases/2012/2012-09-11.asp |title=加拿大公民和移民部：新聞發布-加拿大實施對[[博茨瓦納|博茨瓦納]]，[[納米比亞|納米比亞]]，[[聖盧西亞|聖盧西亞]]，[[聖文森特和格林納丁斯|聖文森特和格林納丁斯]]以及[[斯威士蘭|斯威士蘭]]公民的簽證要求 |publisher=Cic.gc.ca |date= |accessdate=2013-01-05}}</ref>

博茨瓦納，納米比亞和斯威士蘭被剔除的主要原因與販賣人口（尤其是未成年人）和使用偽造文書有關。此外，納米比亞有最高的非法移民比率，高達81%在加的公民承認違反移民法。2011年，有71%的納米比亞旅客在加拿大申請政治庇護。

聖盧西亞以及聖文森特和格林納丁斯被移除的主因是其不可靠的旅行檔案，特別是因為「犯罪分子在這些國家可以合法變更其姓名並獲取新的護照」。在某些情況下，這兩個國家的公民「因為安全理由被遣出加拿大之後再度使用不同的護照進入」。此外，聖盧西亞以及聖文森特和格林納丁斯被移出還由於「來自聖盧西亞和聖文森特不可接受的政治庇護申請高比率，有高達該國人口1.5%至3%的公民在過去5年於加拿大尋求政治庇護申請」。
