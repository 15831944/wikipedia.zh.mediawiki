'''加藤莉娜'''（{{lang-ja|加藤リナ}}，{{bd|1990年|6月4日|}}），[[東京都|東京都]]出身的[[日本|日本]][[AV女優|AV女優]]，曾是[[Prestige|PRESTIGE]]專属女优。所属於[[CLAP_Promotion|CLAP Promotion]]事务所。于2013年4月引退。

== 人物 ==
在成為AV女優之前是一位[[宣傳模特兒|展場模特兒]]。<ref>FLASH 2012年9月25日号 AV女優70人プライベート性調査23</ref>

兴趣是[[飞镖|飞镖]]，而最喜歡的運動是足球，最喜歡的球員是2012年12月當時在[[德甲|德甲]][[紐倫堡足球俱樂部|FC紐倫堡]]效力的[[清武弘嗣|清武弘嗣]]選手。她也是[[日本國家足球隊|日本國家足球隊]]的死忠球迷。

== 簡歷 ==
2010年8月，在[[Prestige|Prestige]]以《ダマし撮りじぇねれーしょん 01》AV[[初登場|初登場]]。<ref>{{cite web|title=加藤 リナ(かとう りな) プレステージ |url=http://www.prestige-av.com/actress/katorina.php |publisher=[[Prestige|プレステージ]]|date=2010-08-20|}}</ref>

2012年4月，在限定的半年內與[[佐藤遥希|佐藤遥希]]、[[蕾_(AV女優)|蕾]]、[[小倉奈奈|小倉奈奈]]、[[成瀬心美|成瀬心美]]等女優共同組成[[Sky_PerfecTV!|Sky PerfecTV!]]成人啦啦隊『AIDL5』<ref>{{cite web|url=http://news.searchina.ne.jp/disp.cgi?y=2012&d=0405&f=national_0405_162.shtml |title=AV女優5名による、太っ腹企画 市販作品をファンに無料配信 |publisher=[[新秦|サーチナ]] |date=2012-04-05 |accessdate=2012-11-25 }}</ref>。在私底下與小倉奈奈的關係特別好。

2013年1月16日，所屬片商Prestige宣布加藤莉娜在4月6日的工作结束后将会从此引退。<ref>{{cite web|title=加藤リナ‧AV完全引退 |url=http://www.playno1.com/article-9655-1.html#.Vu2JlfkrKhc|publisher=AVNo.1-PLAYNO.1|date=2013-01-16|}}</ref>而她本人在自己的Twitter帐号上也确认了这个消息，并说这是自己的决定。<ref>[https://twitter.com/katou_rina/status/291444147942989824 @katou_rina : 皆さん！突然引退発表をしてしまいビックリさせてしまってごめんなさい。引退についてBlogに想いを書かせて頂いたのでみてください。 加藤リナのカトリナ日記♪ : 加藤リナ、引退します。]，加藤リナのカトリナ日記♪ : 加藤リナ、引退します。</ref>

2013年4月6日，從AV界引退。

2013年4月7日，她在自己的官方Twitter上贴出告别推文。

2013年4月16日，她注销了自己在Twitter上的帐号，并正式引退。
