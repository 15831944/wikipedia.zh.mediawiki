'''{{ruby-py|汨|-{zh-cn:mì;zh-tw:ㄇㄧˋ;zh-hk:mik<sup>9</sup>}-}}罗市'''{{NoteTag|1=“汨”字為左氵右[[日|日]]}}，别称'''罗城'''，是[[中华人民共和国|中华人民共和国]][[湖南省|湖南省]]的[[县级市|县级市]]，由[[岳阳市|岳阳市]]代管。市人民政府驻[[归义镇_(汨罗市)|归义镇]]。

位于[[湖南省|湖南省]]东北部，[[汨罗江|汨罗江]]的下游，因汨罗江而得名。全市总面积1669.8平方公里，人口73.69万<ref name=":0">{{Cite web|url=http://www.miluo.gov.cn/todaymiluo/14860.htm |title=汨罗市乡镇区划调整方案：调整后下辖1乡17镇1街道 |accessdate=2016-10-17 |work=www.miluo.gov.cn |deadurl=yes |archiveurl=https://web.archive.org/web/20161018214732/http://www.miluo.gov.cn/todaymiluo/14860.htm |archivedate=2016-10-18 }}</ref>。交通发达，[[京广铁路|京广铁路]]、[[京珠高速|京珠高速]]、[[107国道|107国道]]纵贯市境。[[中国肉类生产百强县列表|湖南省第五大、中国第22大肉类生产大县市]]，[[农业|农业]]基础坚实，是中国商品粮基地县(市)；再生资源集散市场被[[中华人民共和国国务院|国务院]]批准列为国家[[循环经济|循环经济]]试点。[[战国时期|战国时期]][[文学家|文学家]][[屈原|屈原]]自沉之地，现有[[屈子祠|屈子祠]]、[[屈原墓|屈原墓]]等古迹纪念建筑。

== 历史<ref>{{cite web|url =http://www.miluo.gov.cn/front/zhengfu/miluo/zjml3.asp|title =汨罗人文历史|deadurl =yes|archiveurl =https://web.archive.org/web/20081105113031/http://www.miluo.gov.cn/front/zhengfu/miluo/zjml3.asp|archivedate =2008-11-05}}</ref> ==

* 根据位于城郊窑州村[[附山园遗址|附山园遗址]]考证，最迟9000年前的[[新石器时代|新石器时代]]，就有史前人类在汨罗江畔生活，属于[[大溪文化|大溪]][[龙山文化|龙山文化]]范畴。
* [[东周|东周]]莊王七年（前690年），楚武王灭[[罗国|罗国]]；次年，罗国遗民迁至[[湘江|湘江]]流域，筑城于汨罗江尾闾南岸，名罗城。境域包括今[[湘阴县|湘阴]]、'''汨罗'''、[[平江县|平江]]三县（市）及[[岳阳县|岳阳县]]南部和[[沅江市|沅江市]]东部地区。
* [[秦始皇|秦始皇]]二十六年（前221年），设置[[罗县|罗县]]，治所设罗城，境域为罗子国移民领地。
* [[南朝宋|南朝宋]]元徽二年（474年），析罗县、益阳、湘西三县的沿江沿湖地区设置[[湘阴县|湘阴县]]。
* 唐[[武德|武德]]八年（625年），撤罗县并入湘阴县，治所今汨罗川山坪镇城墙村。
* 1966年1月18日析湘阴县东部置汨罗县，以湘阴县的汨罗、白水、桃林、长乐、弼时5区为其行政区域，隶岳阳地区。
* 1987年9月23日撤县设市，以原汨罗县为汨罗市辖域。

== 地理 ==

=== 位置 ===

汨罗地处[[东经|东经]]112°51'-113°27'，[[北纬|北纬]]28°28'-29°27'之间。最东处为八景乡山阳村的山阳寨，最西处是芦苇场的石湖包，最南端是高家坊镇鹿峰村的陈家湾，最北端是磊石乡汨岳村的汨岳界<ref>{{cite web|url =http://www.miluo.gov.cn/front/zhengfu/gongyeyuan/jtqw.html|title =汨罗交通位置}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。南北相距66.75公里，东西相距62.5公里，全境周长301.84公里，总面积1561.95平方公里，汨罗城区面积12.37平方公里<ref>《2007年湖南统计年鉴》</ref>。

=== 地势 ===

汨罗属[[幕阜山|幕阜山脉]]与[[洞庭湖平原|洞庭湖平原]]的过渡地带，东南部高，西北部低，由[[山地|山地]]向滨湖[[平原|平原]]过渡。山峰东部有幕阜山余脉[[智峰山|智峰山]]等；南部有[[玉池山|玉池山]]等。其中玉池山主峰达摩岭海拔777.5，为汨罗最高峰。
[[岗地|岗地]]是汨罗分布最广的地貌类型，占全市总面积39.28%。
平原位于汨罗江及其支流溪谷两侧，水文面积296.01平方公里，是全市[[粮食|粮食]]、[[棉花|棉花]]的主要产区。<ref name="miluo1">{{cite web|url =http://www.mlcn.cn/lcfq/jrml/jrlc/200612/5725.html|title =汨罗地理特点}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>

=== 水文 ===

境内[[水利|水利]]丰富。流域面积6.5平方公里以上的河流44条。其中，流域面积在100平方公里以上的河流10条。[[洞庭湖|洞庭湖]]水系有汨罗江及支流汨江、罗水，为[[洞庭湖水系|洞庭湖水系]]中仅次于[[湘江|湘]]、[[资江|资]]、[[沅水|沅]]、[[澧水|澧]]的第五大水系。汨罗多年平均[[降水量|降水量]]1345.4毫米，降水总量 21.31亿[[立方米|立方米]]，地表水资源总量44.65亿立方米，[[地下水|地下水]]储量24.21亿立方米。<ref name="miluo1" />

=== 资源<ref>{{cite web|url =http://www.miluo.gov.cn/front/zhengfu/miluo/zjml4.asp|title =汨罗物产资源|deadurl =yes|archiveurl =https://web.archive.org/web/20081105113556/http://www.miluo.gov.cn/front/zhengfu/miluo/zjml4.asp|archivedate =2008-11-05}}</ref> ===

* 土地资源 全市[[耕地|耕地]]总面积51.16万亩，宜林地87万亩，草场54.76万亩。
* 植物资源 [[蕨类植物|蕨类植物]]15科，25种；[[裸子植物|裸子植物]]7科，13种；[[被子植物|被子植物]]94科，383种。国家保护植物有[[水杉|水杉]]、[[银杏|银杏]]、[[杜仲|杜仲]]等。森林覆盖率达41．8％，积蓄林木218．3万立方米。
* 动物资源 [[昆虫|昆虫]]65科，168种；[[鱼类|鱼类]]20科，90种；[[鸟类|鸟类]]28科，50种；[[哺乳类|哺乳类]]16科，29种。国家保护动物有[[穿山甲|鲮鲤]]（穿山甲）、[[金钱豹|金钱豹]]、[[大鲵|大鲵]]（娃娃鱼）、[[猴面鹰|猴面鹰]]、[[江豚|江豚]]（江猪）、[[大灵猫|大灵猫]]等。
* 矿藏资源 汨罗江[[砂金矿|砂金矿]]是已探明的[[长江|长江]]以南最大的河流矿床，地质储量20吨左右。
* 水力资源 水资源的理论蕴藏能量4.01万千瓦。

== 人口语言 ==

全市有15个[[少数民族|少数民族]]，[[汉族|汉族]]人口占99.95%。据2000年11月1日零时全国第五次人口普查结果显示：全市总户数19.40万户，总人口65.89万人。其中非农业户人口8.83万人。男女性比例107.6：100。<ref>{{cite web |url = http://www.miluo.gov.cn/front/zhengfu/miluo/zjml2.asp |title = 汨罗行政区划 |deadurl = yes |archiveurl = https://web.archive.org/web/20081105113606/http://www.miluo.gov.cn/front/zhengfu/miluo/zjml2.asp |archivedate = 2008-11-05 }}</ref>

[[汨罗话|汨罗话]]属[[湘方言|湘方言]]长(沙)益(阳)片，城关等部分地区与[[长沙话|长沙话]]和[[湘阴|湘阴]]话能直接沟通，与岳阳下辖其它市如[[平江|平江]]、[[临湘|临湘]]方言（属[[贛語方言|贛語方言]]区）无法直接交谈。
部分地区，如长乐因地域与平江、[[岳阳县|岳阳县]]毗连，受[[赣语|赣语]]的影响较大，但仍属湘语长益片 <ref>{{Cite journal zh |author=陈山青|title=湖南汨罗长乐方言语音分析|journal= 湘潭大学学报（哲学社会科学版）|volume = 第30卷 |issue = 第04期|date=2006年}}</ref>。

== 经济 ==

汨罗经济发展迅速，[[农业|农业]]保持平稳发展，[[工业|工业]]和[[第三产业|第三产业]]快速发展。以2007年为例：全市实现地区[[生产总值|生产总值]]85.1万元，比上年增长17.7%。其中[[第一产业|第一产业]]增长2.0%；[[第二产业|第二产业]]增长28.6%；[[第三产业|第三产业]]增长14.1%。三产业结构比例为27.7：49.9：22.4。主要经济指标排岳阳六县（市）第一位，2006年度中首次进入湖南经济强县十强，列第九。

=== 第一产业 ===

汨罗是[[农业|农业]]大市，农业经济基础坚实，主要粮食作物为[[水稻|水稻]]、[[玉米|玉米]]、[[红薯|红薯]]等，经济作物有大棚蔬菜，[[西瓜|西瓜]]，[[棉花|棉花]]等，畜牧业渔业生产保持稳定发展。农民现金收入与生活消费同步增长，农村固定资产投资保持较快发展势头。

* 1987年，率先在湖南省内[[绿化|绿化]]宜林荒山，获“'''湖南省消灭荒山第一市（县）'''”的称号
* 1988年被确定为'''湖南省瘦肉型猪生产基地县（市）'''
* 1989年被确定为'''全国商品粮基地县（市）'''
* 1993年列入'''国家农业综合开发项目县（市）'''
* 1994年成为'''全国水稻机械化第一县（市）'''
* 2000年被确定为湖南省5个'''农业农村现代化试点县（市）'''之一<ref>{{cite web | url = http://www.miluo.gov.cn/front/zhengfu/miluo/zjml6.asp | title = 汨罗发展概况 | deadurl = yes | archiveurl = https://web.archive.org/web/20081105113611/http://www.miluo.gov.cn/front/zhengfu/miluo/zjml6.asp | archivedate = 2008-11-05 }}</ref>

=== 第二产业 ===

[[工业|工业]]为主的第二产业成为汨罗经济增长的主动力，以可再生资源深加工、农产品加工、绿色环保和农机制造为四大主要产业。 2005年10月，汨罗再生资源集散市场被国务院批准列为国家循环经济试点。国家“[[十一五规划|十一五]]”规划明确提出：“建设湖南汨罗等再生资源回收利用市场和加工示范基地”<ref>{{cite web|url=http://news.xinhuanet.com/misc/2006-03/16/content_4309517_10.htm | title = 中华人民共和国国民经济和社会发展第十一个五年规划纲要| work= 专栏11循环经济示范试点工程 | quote = 再生资源回收利用 建设湖南汨罗等再生资源回收利用市场和加工示范基地。 }}</ref>。再生资源产业为主的工业园正成为湖南及中国知名的循环经济示范园区。

=== 第三产业 ===

汨罗传统服务行业稳步发展，其中商贸流通和[[餐饮|餐饮]][[娱乐|娱乐]]业发展势头强劲，交通运输业和邮电通讯业快速发展；新兴服务也发展良好，房地产业逐步规范，金融保险业健康发展以及旅游业和节日经济发展迅速。
但发展仍滞后于经济发展的整体水平，2007年[[第三产业|第三产业]]所占[[GDP|GDP]]比重为22.4%，远低于中国平均40.5%，湖南省39.5%，岳阳市33.8%。
<ref>{{cite web | url = http://miluo.mofcom.gov.cn/accessory/200706/1182408883204.doc | title = 汨罗市第三产业发展情况汇报 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

体现在第三产业：

* 与[[工业化|工业化]]的快速推进不协调。
* 与[[城市化|城市化]]进程不适应。
* 对[[GDP|GDP]]的贡献率不大。
* 体制不顺。

== 交通 ==

汨罗西临[[洞庭|洞庭]]，南连[[长沙|长沙]]，北通[[岳阳|岳阳]]；[[京广铁路|京广铁路]]、[[武广客运专线|武广客运专线]]、[[京珠高速|京珠高速]]、[[107国道|107国道]]、[[湖南省道308线|湖南省道308线]]、[[汨罗江|汨罗江]]横穿东西。还有正在建设中的京珠高速公路复线、湖南省道201线和规划中的长岳城际铁路、汨罗-益阳-衡阳城际铁路和浏阳到溆浦铁路。

京广铁路沿途经桃林、范家园、汨罗、古培塘、越江（白水）、川山坪、高家坊等多个站。

== 科教 ==

汨罗的[[初等教育|初等教育]]和[[中等教育|中等教育]]以[[素质教育|素质教育]]而著名<ref>提高民族素质的基础工程——湖南省汨罗市13年全面推行中小学素质教育的调查与思考，《'''求是'''》，1996年23期</ref>。从1984年开始大面积推行素质教育探索<ref>素质教育再看汨罗，《'''人民教育'''》，2006年11期</ref> 。1996年，时任国务院副总理的[[李岚清|李岚清]]视察汨罗市教育，指示“汨罗经验非常可贵，要逐步在全国推广”。1996年7月由原[[中华人民共和国国家教育委员会|国家教委]]在岳阳召开的“全国构建督导评估机制推动素质教育”会议，汨罗素质教育经验推向全国。 <ref> 中国教育实验20年之运作回眸与走势前瞻，《'''当代教育科学'''》，2001年11期</ref> 2006年5月，[[中国教育报|中国教育报刊社]]、[[湖南省教育学会|湖南省教育学会]]、[[湖南省教育科学研究院|湖南省教育科学研究院]]、[[湖南教育报|湖南教育报刊社]]在长沙联合举办了“汨罗素质教育论坛”。<ref> “汨罗素质教育论坛”举行，《'''中国教育报'''》，2006年5月24日</ref>

至2007年，汨罗市共有[[职业高中|职业高中]]12所，普通[[高中|高中]]6所，初中38所，小学143所，[[特殊教育|特殊教育]]学校1所。其中[[汨罗职业中学|汨罗职业中学]]是全国示范性职业中学。[[汨罗市第一中学|汨罗市第一中学]]和[[汨罗市第二中学|汨罗市第二中学]]是湖南省示范性中学。

== 文化习俗 ==

2009年汨罗江畔端午习俗入选[[人類非物質文化遺產代表名錄|世界非物质文化遗产]] 。2011年长乐故事会入选[[国家级非物质文化遗产|国家级非物质文化遗产]]。

=== 龙舟竞渡 ===
{{See Also|端午节|中華端午節|赛龙舟}}

端午源头，龙舟故里。

龙舟竞渡吃粽子的习俗起源于汨罗。传说当时楚人以舟拯救屈原，借[[粽子|粽子]]驱江鱼，期望阻止鱼吃掉屈原的身体。

2005年，建中国汨罗江国际龙舟竞渡中心，举办中国汨罗江国际龙舟节。
另外龙舟竞渡中心也是2008年[[2008年夏季奥林匹克运动会火炬接力|奥运火炬]]在湖南汨罗传递的终点。

* [[任凯南|任凯南]]

* [[白杨_(演员)|白杨]]

* [[韩少功|韩少功]]

* [[杨敬年|杨敬年]]
