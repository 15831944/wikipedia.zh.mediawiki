'''铜陵市'''是[[中华人民共和国|中华人民共和国]][[安徽省|安徽省]]下辖的[[地级市|地级市]]，位于安徽省南部，[[宁铜铁路|宁铜铁路]]终点和[[铜九铁路|铜九铁路]][[庐铜铁路|庐铜铁路]]起点，濒临[[长江|长江]]，是长江下游重要的[[港口|港口]]之一<ref>{{cite web|url=http://www.tl.gov.cn/ |title=铜陵市政府网站 |date= |accessdate=2017-06-10}}</ref>。全市总面积3,008平方公里，2017年末户籍人口171.1万人，常住人口160.8万人<ref>{{cite web|url=http://www.tl.gov.cn/sjtl/tjgb/201803/t20180323_438225.html |title=铜陵市2017年国民经济和社会发展统计公报 |publisher=铜陵市政府网站 |date= |accessdate=2018-03-23}}</ref>。现辖3[[市轄區|市轄區]]（[[铜官区|铜官区]]、[[义安区|义安区]]、[[郊区_(铜陵市)|悦江区]]）1[[县_(中华人民共和国)|县]]（[[枞阳县|枞阳县]]）。

铜陵以产[[铜|铜]]著称于世，是中华民族[[青铜|青铜]]文化的重要发祥地之一。铜开采、冶炼的历史始于三千年前的商周时代，故有“中国古铜都”之誉。

== 历史 ==
* [[南北朝|南北朝]][[齐|齐]][[梁|梁]]时期(公元479~557年),官府在此设立采铜机构,兴办官营手工[[工场|工场]]。
* [[唐朝|唐朝]]时期(公元618~907年)采铜规模扩大,兴建了官办铜矿,称之[[铜官场|铜官场]]。
* 公元888年，[[庐州|庐州]][[刺史|刺史]][[杨行密|杨行密]]统兵袭[[宣州|宣州]]，扎营[[铜官场|铜官场]]之山麓，此山遂以[[铜官山|铜官山]]之名首见于史册。
* [[南唐|南唐]]，曾封[[铜官山|铜官山]]为[[利国山|利国山]]。
* [[清|清]]初，这里曾设立过[[铜官督|铜官督]]。
* [[鸦片战争|鸦片战争]]后，[[帝国主义|帝国主义]]列强为了掠夺[[铜矿|铜矿]]资源，曾在此设立过[[安裕公司|安裕公司]]、[[伦华公司|伦华公司]]。
* 1938年，[[日本|日本]]侵占铜陵后，设立[[铜官山矿业有限公司|铜官山矿业有限公司]]。
* 1949年4月21日，[[渡江战役|渡江战役]]后铜陵成为江南第一个解放的城市。
* 1952年11月，[[人民政府|人民政府]]在此成立了[[铜官山矿务局|铜官山矿务局]]。
* 1956年10月12日，成立[[铜官山市|铜官山市]]。
* 1958年，[[铜官山市|铜官山市]]与[[铜陵县|铜陵县]]合为铜陵市。同时由省辖改隶属于[[安庆专区|安庆专区]]。
* 1959年，市县分开,设置铜陵市和[[铜陵县|铜陵县]]。
* 1964年，撤销市制，改为[[铜陵特区|铜陵特区]]。
* 1965年，恢复[[池州专区|池州专区]]，专署驻[[贵池县|贵池县]]。辖原属[[安庆专区|安庆专区]]的[[贵池|贵池]]、[[青阳|青阳]]、[[东至|东至]]、[[铜陵|铜陵]]4县；由[[太平|太平]]、[[贵池|贵池]]2县析置[[石台县|石台县]]。池州专区辖5县。
* 1971年，撤销[[池州专区|池州专区]]，改置[[池州地区|池州地区]]。[[铜陵县|铜陵县]]属于池州地区。
* 1972年1月，撤消[[铜陵特区|铜陵特区]],恢复铜陵市。
* 1974年，铜陵市成为省辖市，辖三区（[[铜官山区|铜官山区]]、[[狮子山区|狮子山区]]、[[郊区_(铜陵市)|郊区]]）一县（[[铜陵县|铜陵县]]）。
* 2015年，经国务院批准，将安庆市枞阳县划归铜陵市管辖；撤销铜陵市铜官山区、狮子山区，设立铜陵市铜官区；撤销铜陵县，设立铜陵市义安区。
* 2019年，铜陵市郊区更名为“悦江区”。<ref>{{Cite web |author= |date= |url=http://www.tl.gov.cn/zxzx/xwzx/bmdt/420/201901/t20190125_564385.html|title=郊区抓重点攻难点推动高质量发展|work=铜陵市人民政府网}}</ref>

== 矿产资源 ==
铜陵自然资源尤以矿产资源储量丰富，矿种全而闻名。已探明的矿种主要有铜、硫、铁、金、银、煤、石灰石等，其中铜、金、银、硫铁矿和石灰石储量均在全省名列前茅。大理石、石灰岩、膨润土、铅锌矿、珍珠岩、钾长石、石英石等，也有一定储量。铜陵八宝分别为：金，银，铜，铁，锡，生姜，老蒜，麻。

铜陵位于[[宁铜铁路|宁铜铁路]]的终点以及[[铜九铁路|铜九铁路]]的起点，境内亦有[[宁安客运专线|宁安客运专线]]、[[合福客运专线|合福客运专线]]通过，截止2016年5月15日，[[铜陵站|铜陵站]]每日开行36对旅客列车，[[铜陵北站|铜陵北站]]每日开行32对旅客列车。铜陵是[[京台高速公路|京台高速公路]]（原合铜黄高速公路，现已并入京台高速公路）和[[沪渝高速公路|沪渝高速公路]]的交汇点。铜陵东距[[南京|南京]]约200千米，西距[[九江|九江]][[庐山|庐山]]约300千米，南距[[九华山|九华山]]约80千米，距[[黄山|黄山]]约120千米，北距[[合肥|合肥]]约130千米。[[铜陵长江大桥|铜陵长江大桥]]于1995年12月26日建成通车，[[铜陵长江公铁大桥|铜陵长江公铁大桥]]于2015年7月1日建成通车。

铜陵距[[合肥新桥国际机场|合肥新桥国际机场]]约150千米，可以在市内购买机票；距[[池州九华山机场|池州九华山机场]]约20千米。
