'''铜陵职业技术学院'''，为[[安徽省|安徽省]][[铜陵市|铜陵市]]的一所公办全日制普通高等职业院校。

== 简介 ==
学院现有教职工388人，其中副高级以上职称88人。拥有省级专业带头人5人，省级教学团队3个，省级教坛新秀4人。建有实习实训基地167个，其中省级示范实训中心6个。<ref name=xyjj>{{Cite web |url=http://www.tlpt.net.cn/main/1-xyjj.asp?typeid=23&bigclassid=62 |title=铜陵职业技术学院学院简介 |access-date=2016-08-08 |archive-url=https://web.archive.org/web/20170329212819/http://www.tlpt.net.cn/main/1-xyjj.asp?typeid=23&bigclassid=62 |archive-date=2017-03-29 |dead-url=yes }}</ref>

== 学院 ==
现有管理系、经贸系、医学系、护理系、信息工程系、艺术传媒系、机械工程系、电气工程系、基础教学部、体育部、思政部11个系部。设有48个专业，其中省级特色专业7个。<ref name=xyjj/>
