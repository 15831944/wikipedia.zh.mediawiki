'''司打口總站'''（[[葡萄牙文|葡萄牙文]]：'''PRAÇA PONTE HORTA/ TERMINAL'''、{{lang-en|'''Praca Ponte Horta Termianl'''}}）位於[[澳門|澳門]]下環區河邊新街的一個終點站，該終點站設於兩個方向行車道中間，毗鄰[[內港客運碼頭|內港客運碼頭]]。現時只有[[澳巴|澳巴]]'''[[新福利3A路線|3A路線]]'''停靠該站。另外，[[澳門新福利公共汽車有限公司|新福利]]'''[[新福利5路線|5]]'''、'''[[新福利16路線|16]]'''、'''[[澳門巴士26路線|26]]'''、'''[[澳門巴士MT4路線|MT4]]'''路線，[[澳巴|澳巴]]'''[[澳門巴士1路線|1]]'''、[[澳門巴士2路線|2]]'''、'''[[澳門巴士6A、6B路線|6B]]'''、'''[[澳門巴士10路線|10]]'''、'''[[澳門巴士10A路線|10A]]'''、'''[[澳門巴士11路線|11]]'''、'''[[澳門巴士21A路線|21A]]'''、'''[[澳門巴士55路線|55]]'''、'''[[澳門巴士N3路線|N3]]'''路線會經過該站旁邊的「司打口」站。

== 沿革 ==
== 歷史 ==
*2007年12月4日，為配合[[內港客運碼頭|內港客運碼頭]]啟用，[[土地工務運輸局|土地工務運輸局]]將原設於[[內港客運碼頭|內港客運碼頭]]的終點站遷往司打口對開的新建終點站，以方便市民乘坐。而原站位置附近則改為有時限的遊覽車上下客區。<ref>[http://www.gcs.gov.mo/showNews.php?DataUcn=28333&PageLang=C 澳門政府新聞局 - 司打口新巴士總站周二啟用2007-12-02]</ref>

== 設施 ==
*[[澳巴|澳巴]]站長更亭。
