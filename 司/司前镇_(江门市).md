'''司前镇'''是[[广东省|广东省]][[江门市|江门市]][[新会区|新会区]]下辖的一个镇。位於新會區西端，总面积88.77平方公里，人口6.5484萬。港澳臺華僑鄉親4.8萬。

== 地理 ==
'''司前'''地处[[珠江三角洲|珠江三角洲]]西部，位于新会区的西部，东为[[大泽|大泽]]镇，西接[[开平|开平]]市[[水口|水口]]镇，北接[[鹤山|鹤山]]市[[共和镇|共和镇]]与[[址山镇|址山镇]]，南隔[[潭江|潭江]]与[[罗坑|罗坑]]镇相望。

==历史==
[[唐|唐]][[元和|元和]]年间（806年—820年）此地已有居民居住，[[明|明]]、[[清|清]]时为牛肚湾司驻地，因在司衙前建墟，故名司前。

清朝时，司前分属石碑都、遵名都和得行都。民國時，属第五区。1955年設立區公所，改名为司前区。1958年改為人民公社，1984年恢復為司前區，1986年改為鎮至今。

== 气候 ==
司前[[气候|气候]]温和湿润，夏热冬温，少霜无雪，年平均气温约22摄士度，年降水量约2000毫米，春夏及秋初雨水较集中。夏秋易受[[台风|台风]]袭击，但因离海边有一段距离（约50公里），台风灾害没有海边地区那么严重。

== 经济 ==
司前是新会经济强镇，人均收入多年居新会区第二（仅次于县城会城镇）。民营企业蓬勃发展，以[[制造业|制造业]]为主，尤其以[[不锈钢|不锈钢]][[产业|产业]]比较有名，镇内的不锈钢制品、产品远销国内外，享有″不锈钢制品之乡″的美誉，为[[广东|广东]]省的五金不锈钢专业镇技术创新试点。2002年被定为广东省270个[[中心镇|中心镇]]之一，2005年初，更成为首批118个[[全国发展改革试点小城镇|全国发展改革试点小城镇]]之一。在河村工业区内有闻名的[[广东天健实业集团有限公司|广东天健实业集团有限公司]]，该公司是全国最大的钢家具出口基地之一，被国家工商总局评为中国[[私营企业|私营企业]]500强，位列第32位。

== 方言 ==
司前镇有着非常独特的语言：[[司前话|司前话]]。司前话应归属[[粤语|粤语]]中的四邑方言片，然而她与粤语标准音[[广州话|广州话]]的差异很大，与周边地区的[[四邑话|四邑话]]差异也很大，讲廣州話、四邑话的人难以听懂司前话，而司前人却能听懂甚至會講廣州話、四邑话。随着与周边地区，尤其是广佛地区[[经济|经济]][[文化|文化]]交往加深，司前话受广州话影响较大，主要表现在词汇方面，现时老一辈司前人与年轻人讲的司前话在[[语音|语音]]、[[词汇|词汇]]上均已有差异。但大体上，司前话仍然保持着一些其特有的词汇。另外，在三益乡一带的司前话与其他地区的司前话相比，其[[口音|口音]]比较接近会城话（会城镇主要说四邑话，但其语音、词汇受[[广州话|广州话]]影响很大，已十分接近广州话，只是[[声调|声调]]与广州话有明显差异，但仍能与广州话直接通话。而广州人则很难听懂司前话）。

== 其他 ==
司前有两个经济文化中心：[[河村|河村]]和司前墟镇。从一些河村出土以及在河村保存下来的古文物分析，有考古学家认为，[[河村|河村]]是新会在[[秦|秦]][[汉|汉]]时期刚开埠时的经济文化中心（从地理上看，司前恰好处于[[五邑|五邑]]地区地理中心，古新会包括目前五邑地区）。而司前墟镇已成为司前实际的经济文化中心，镇政府及大部分的[[商铺|商铺]]、[[食肆|食肆]]及司前影剧院、[[司前文沧广场|司前文沧广场]]都位于司前墟镇。
