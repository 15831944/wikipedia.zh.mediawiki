'''司马'''是[[中国|中国]]古代的一个[[官职|官职]]名称，在不同的朝代有不同的职权。

== 中央官名 ==
[[西周|西周]]始置，位次[[三公|三公]]，与[[六卿|六卿]]相当，与[[司徒|司徒]]、[[司空|司空]]、[[司士|司士]]、[[司寇|司寇]]并称[[五官|五官]]，掌军政和军赋，[[春秋時期|春秋]]、[[戰國_(中國)|战国]]沿置。楚國設置「[[令尹|令尹]]」與「大司馬」二職，皆有權領兵，如《左傳》：「司馬將中軍,令尹將左軍,右尹子辛將右軍。」等。究竟由什麼職務統軍，端看實際需要而定，並不是很嚴格的規範。
[[汉武帝|汉武帝]]时置[[大司马|大司马]]，作为[[大将军|大将军]]的加号，后亦加于[[骠骑将军|骠骑将军]]，[[后汉|后汉]]单独设置，皆[[开府|开府]]。[[隋朝|隋]][[唐朝|唐]]以后为[[兵部尚书|兵部尚书]]的别称。<ref>《[[史记|史记]]·楚世家》：“立子比为王，公子子皙为令尹，充疾为司马。”</ref>

== 军官名 ==
[[東周|東周]][[戰國|戰國]]時期，各軍之下亦設有司馬，地位不如各軍主帥與軍佐，領的是大夫銜，《[[尉繚子|尉繚子]]》：「今百人一卒，千人一司馬，萬人一將，以少誅眾，以弱誅疆。」可見司馬之職位在將軍之下。
汉武帝定-{}-制，司馬，主武也，掌管軍事之職。大将军所属军队分为五部，各置司马一人领之。[[魏晋南北朝|魏晋南北朝]]，诸将军开府，府置司马一人，地位僅次於将军，掌本府軍事，相当于后世的参谋长。[[司馬昭|司馬昭]]即以[[山涛|山涛]]为行军司马，[[胡三省|胡三省]]注稱：“行军司马之号始此。”。<ref>《[[資治通鑑|資治通鑑]]·魏元帝咸熙元年》：“昭自将大军从帝幸长安，以诸王公皆在鄴，乃以山涛为行军司马。”胡三省注：“行军司马之号始此。”</ref>

宋制，司馬銅印墨綬，絳朝服，武冠。至隋時廢州府之任，不置司馬，改置[[治中|治中]]。<ref>《[[三国志|三国志]]·蜀志·[[姜维|姜维]]传》：“[[延熙|延熙]]元年，随大将军[[蒋琬|蒋琬]]往[[汉中|汉中]]。琬既迁大司马，以维为司马，数率偏军西入。”</ref>

== 地方官名 ==
[[唐朝|唐朝]]，[[州|州]]设司马，作为[[刺史|刺史]]的佐官，实际无职掌，多用于处置贬官；著名詩人[[白居易|白居易]]即曾因得罪权贵，被貶為[[江州|江州]]（今[[江西省|江西省]][[九江市|九江市]]）司馬，並在元和十一年寫下《[[琵琶行|琵琶行]]》這篇千古名作，其诗云：“同是天涯沦落人，相逢何必曾相识”、“坐中泣下谁最多？江州司马青衫湿”。[[明朝|明]][[清朝|清]][[士大夫|士大夫]]雅稱[[同知|同知]]為司馬。
