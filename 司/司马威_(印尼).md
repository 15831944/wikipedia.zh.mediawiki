'''司馬威'''是[[印度尼西亞|印度尼西亞]]的城市，由[[亞齊省|亞齊省]]負責管轄，位於該國西北部[[蘇門答臘島|蘇門答臘島]]北部，面積181.06平方公里，2010年人口170,504，人口密度為每平方公里940人。城市附近有丰富的石油气矿藏，为[[苏哈托|苏哈托]]政府与美国[[埃克森美孚|埃克森美孚]]合作开发，于1978年投产，为当地经济带来兴旺景气，合约期满由政府收回，成立司马威经济特区<ref name="归谁">{{cite web |url=http://www.guojiribao.com/shtml/gjrb/20170706/323550.shtml |title=阿伦经济特区应归谁经管? |last= |first= |date=2017-07-06 |website=www.guojiribao.com |publisher=国际日报 |access-date=2017-09-22 |quote=}}</ref>。

==经济特区==

2017年2月17日，[[印度尼西亚总统|印尼总统]][[佐科·维多多|佐科·维多多]]签署了2017年第5号关于建立司马威经济特区的政府条例，经济特区主要发展出口加工、物流、工业品制造、能源加工和旅游等行业，经管权完全归亚齐省政府和亚齐省政府企业。司马威经济特区总面积2622.48公顷，其中位于司马威城区阿伦炼油厂区域的有1840.8公顷，[[北亚齐县|北亚齐县]]德宛塔拉（Dewantara，有炼油厂）区域有582.08公顷，以及北亚齐县的耶木安（Jamuan，亚齐纸浆厂所在地）区域有199.6公顷，司马威经济特区预计最迟在3年时间内建成并开始运作。<ref name="归谁"></ref><ref>{{cite web |url=http://www.shangbaoindonesia.com/indonesia-finance/%E4%B8%BA%E5%8F%91%E5%B1%95%E5%8C%97%E4%BA%9A%E9%BD%90%E5%8E%BF%E4%B8%8E%E6%B4%9B%E5%8F%B8%E9%A9%AC%E5%A8%81%E5%9F%8E%E6%88%98%E7%95%A5%E7%BB%8F%E6%B5%8E%E5%8C%BA%E6%94%BF%E5%BA%9C%E6%88%90%E7%AB%8B.html |title=为发展北亚齐县与洛司马威城战略经济区 政府成立阿伦洛司马威经济特区 |last= |first= |date=2017-02-28 |website=www.shangbaoindonesia.com |publisher=印度尼西亚商报 |access-date=2017-09-22 |quote=}}</ref><ref>{{cite web |url=http://www.mofcom.gov.cn/article/i/jyjl/j/201703/20170302525052.shtml |title=印尼成立阿伦洛司马威经济特区 |last= |first= |date=2017-03-01 |website=中华人民共和国商务部电子商务和信息化司 |publisher=中华人民共和国驻棉兰总领馆经商室 |access-date=2017-09-22 |quote=}}</ref>
