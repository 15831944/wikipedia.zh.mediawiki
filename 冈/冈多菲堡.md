'''冈多菲堡'''（{{lang-it|'''Castel Gandolfo'''}}；亦稱為'''岡道爾夫堡'''）是在[[意大利|意大利]][[罗马|罗马]]东南方30[[公里|公里]]的小镇，位於[[拉齐奥|拉齐奥]]大区，人口約7,000人。冈多菲堡高踞在[[阿尔巴诺山|阿尔巴诺山]]上，俯视[[阿尔巴诺湖|阿尔巴诺湖]]，风景优美，是[[教宗|教宗]]的夏日别墅所在地，至今已有400多年历史。

==名字来源==
{{lang|it|'''Castel Gandolfo'''}}这个意大利地名来自于[[拉丁语|拉丁语]]的{{lang|la|''Castrum Gandulphi''}}，城堡的名字可能是来自[[日内瓦|日内瓦]]的{{lang|it|Gandolfi}}家族。另一个说法是教宗[[庇護二世|庇護二世]]在他的自传《笔记》（{{lang|la|Commentaries}}）（1462年）中说，来自萨宾的冈都尔菲（{{lang|la|Gandulphi Sabinorum}}）这个人，即{{Tsl|en|Savelli family|薩維利家族}}的冈道尔夫（{{lang|it|Gandolfo Savelli}}）的拉丁形式。

== 历史 ==
阿尔巴诺对意大利人有特殊意义。公元前7世纪，阿尔巴诺山是拉丁同盟的首都[[阿尔巴朗格|阿尔巴朗格]]的所在地。而阿尔巴诺湖则是一个古老的火山口。考古学家曾在阿尔巴诺湖底发掘出公元前16世纪的古文明遗物。有人认阿尔巴诺是罗马文明的发源地。

1世纪时，阿尔巴诺曾是[[罗马帝国|罗马帝国]][[图密善|图密善]]皇帝的夏宫所在地，原占地14平方公里。

[[File:Pope's_summer_residence.jpg|thumb]]

冈多菲堡的名称来自本地一座12世纪时的冈多菲公爵堡垒。冈多菲堡垒后归罗马望族[[萨维里|萨维里]]所有。1596年，[[教廷国|教廷国]]的[[使徒内阁|使徒内阁]]以十五万[[斯库度|斯库度]]教廷币的代价，从萨维里家族手中将冈多菲堡垒购买下来。教宗[[克莱孟八世|克莱孟八世]]最先入住冈多菲堡。1624年教宗[[乌尔巴诺八世|乌尔巴诺八世]]委派[[瑞士|瑞士]][[建筑师|建筑师]]，[[巴洛克|巴洛克]]风格之父[[卡罗·马德诺|卡罗·马德诺]]（Carlo Maderno）重建冈多菲古堡。此后，教宗[[亚历山大七世|亚历山大七世]]委任杰出的巴洛克雕塑家和建筑设计家[[济安·贝尼尼|济安·贝尼尼]]设计冈多菲古堡广场的圣托马斯教堂。教宗[[庇护十一世|庇护十一世]]加建毗邻的巴伯里尼别墅。

[[File:Lake_Albano.jpg|thumb]]

从1870年至1929年，教宗别墅曾空置近60年。1929年教宗[[庇护十一世|庇护十一世]]和[[贝尼托·墨索里尼|贝尼托·墨索里尼]]签订了《[[1929年条约|1929年条约]]》，条约中规定教宗的冈多菲堡别墅和毗邻的巴伯里尼别墅享有[[治外法权|治外法权]]。此后庇护十一世重新到冈多菲堡别墅避暑。教宗[[庇护十二世|庇护十二世]]和教宗[[保罗六世|保罗六世]]都先后在冈多菲堡别墅逝世。

== 若望·保禄二世 ==
[[File:John_Paul_II_George_W._Bush_July_2001.jpg|thumb]]陪同[[喬治·沃克·布什|布什总统]]偕第一夫人劳拉参观冈多菲堡]]

[[若望·保禄二世|若望·保禄二世]]于1978年10月16日被选为教宗，过后几天教宗若望·保禄二世就前往视察冈多菲堡别墅。保禄二世在冈多菲堡别墅住过的日子很多，他曾在此会见[[美国总统|美国总统]][[喬治·沃克·布什|乔治·布什]]，会见[[巴勒斯坦|巴勒斯坦]][[法塔赫|法塔赫]]的领导人[[亚西尔·阿拉法特|亚西尔·阿拉法特]]，但他更多的时间在别墅接见社会上的名流，包括哲学家，深入探讨当代各种他关心的问题。教宗每个夏季都要在冈多菲堡别墅接见多至15位哲学家、科学家。若望·保禄二世所著《回忆与身份》一书，便是教宗在1993年同两位[[波兰|波兰]]哲学家[[克里斯朵夫·米凯尔斯基|克里斯朵夫·米凯尔斯基]]、[[约瑟夫·提希纳|约瑟夫·提希纳]]在冈多菲堡长谈的结果。

== 本篤十六世 ==
[[本篤十六世|本篤十六世]]继承四百多年歷代教宗的传统，继续以冈多菲堡为夏日别墅。

== 交通 ==
从[[罗马|罗马]]市往东南方向取SS7公路往阿尔巴诺湖方向行驶约30公里，冈多菲堡在阿尔巴诺湖西的阿尔巴诺山岗上。四围是郁郁葱葱的橄榄树林和葡萄园。

== 建筑 ==
教宗别墅是一巴洛克式座红砖大厦，四面封闭如口字形，内院广场有石像。教宗别墅正面正中楼顶有一面意大利古式六分度时钟，钟面沿圆周顺时针方向等距离排列 I、II至VI等6个罗马数字。这种古意大利6分度时钟，在意大利也很少见。教宗别墅灰色的拱形大门两侧有教廷的[[瑞士近卫队|瑞士卫兵]]把守。教宗别墅门前的广场，称为自由广场，广场中央有一个意大利式喷水池。广场一边是多间意大利餐厅，一长列餐桌，常游客满座。别墅一侧有约一米高石栏杆，可供眺望阿尔巴诺湖。一条碎砖路可通山下。

== 梵蒂冈天文台 ==
梵蒂冈天文台的总部设在冈多菲堡，按《[[1929年条约|1929年条约]]》享有[[治外法权|治外法权]]。兩台1930年代从[[梵蒂冈|梵蒂冈]]搬到冈多菲堡的[[天文望远镜|天文望远镜]]，沿用到1980年代。1985年教宗[[若望·保禄二世|若望·保禄二世]]曾考虑将这两台天文望远镜赠送[[中国|中国]][[合肥大学|合肥大学]]，后因各种原因未果。<ref>[http://www.hsstudyc.org.hk/big5/tripod_b5/b5_tripod_137_06.html 教宗若望保祿二世的畢生中國之旅]</ref>
