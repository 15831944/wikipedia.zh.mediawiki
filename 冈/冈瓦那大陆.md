== 名字的由来 ==
冈瓦那本是[[印度|印度]]的一个地名。在这个地方发现了岩石与[[南半球|南半球]]其他大陆的共性，[[爱德华·修斯|爱德华·修斯]]於是以此來起名冈瓦那大陆。

== [[大陆漂移|大陆漂移]]过程 ==
由[[罗迪尼亚大陆|罗迪尼亚大陆]]的南部分裂而成。进一步分裂成：[[新西兰|新西兰]]岛屿、[[澳洲大陆|澳洲大陆]]、[[南美洲|南美大陆]]、[[南极大陆|南极大陆]]、[[馬達加斯加|馬達加斯加]]及[[非洲|非洲]]大陆，以及已經漂流到北半球的[[印度|印度]]古陆、[[阿拉伯半島|阿拉伯半島]]。

== 对动物地理的影响 ==
很多动物起源于冈瓦那大陆，如长鼻类（大象）、古腭类（鸵鸟等），解释了南亚和非洲，非洲和南美之间动物区系的类似性。解释了新西兰没有土生哺乳动物（蝙蝠除外）的原因等。
