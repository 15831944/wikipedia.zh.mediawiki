2016年獲得《[[這本輕小說真厲害！|這本輕小說真厲害！]]》年度文庫排名第8名，新作第3名<ref>{{cite book |author= |title= 《このライトノベルがすごい! 2017》 |publisher= 寶島社 |isbn= 978-4-8002-6345-2 |date=2016年11月}}</ref>，2017年獲得第7名<ref>{{cite book |author= |title= 《このライトノベルがすごい! 2018》 |publisher= 寶島社 |ISBN= 978-4-8002-7798-5 |date=2017年11月}}</ref>。

== 故事簡介 ==
就讀高中二年級的友崎文也，認為有些人不管怎麼努力，都會因為長相、體格等因素導致失敗，而某些人能因為先天優勢而獲得成功，於是得出「人生是款糞作遊戲」的結論，並自認為自己是人生的「弱角」。

但在某一天的因緣際會下，被友崎認為是人生強角的完美女生日南葵，因為看不慣友崎的人生觀，決定要教會友崎人生的「遊戲」規則，就這樣，由弱角實行挑戰的人生攻略論開始了。

== 登場角色 ==
; {{nihongo|友崎文也|友崎 文也（ともざき ふみや）}}
: 就讀關友高中二年級，另一個身分是線上連機對戰遊戲「AttaFami」全日本第一的玩家「nanashi」，有著「人生是款糞作遊戲」的消極人生觀，沒有什麼朋友，認為自己是人生的「弱角」。
: 某天在與AttaFami全日本第二名玩家「NO NAME」的網聚中發現對方的身分為校園女王日南葵，並遭到對方的唾棄，之後便在日南的指導下開始攻略「人生」。
: 在日南的指導下日益進步，過程中也逐漸對自己的改變燃起自信，並與班上的幾個現充慢慢混熟。
; {{nihongo|日南葵|日南 葵（ひなみ あおい）}}
: 就讀關友高中二年級，另一個身分是線上連機對戰遊戲「AttaFami」全日本第二的玩家「NO NAME」，友崎的同班同學，田徑社。
: 成績優異、運動萬能，在學校被視為完美的代名詞，友崎也認為她是人生的「強角」。
: 有著對勝利十分執著的一面，很尊敬唯一能贏過自己的日本第一nanashi，在得知其真實身分為友崎後表示失望，在聽過友崎的抱怨後決定指導他攻略人生。
: 其本身的完美似乎不是與生俱來，而是靠者後天的努力所塑造，因此非常不認同友崎的人生觀，認為就算是人生也可以經由努力得到成果。
; {{nihongo|七海深奈實|七海 みなみ（ななみ みなみ）}}
: 關友高中二年級，友崎的同班同學，班上的現充之一，與日南同屬田徑社。
: 是群體中的開心果，與夏林花火有著接近百合的友情，經常對她性騷擾。
; {{nihongo|菊池風香|菊池 風香（ふうか きくち）}}
: 關友高中二年級，友崎的同班同學，喜歡看書。
: 被日南預定為友崎的女朋友人選。
; {{nihongo|泉優鈴|泉 優鈴（いずみ ゆず）}}
: 關友高中二年級，友崎的同班同學，班上的現充之一。
: 表面上會與友崎正常互動，實際上卻對其不以為然，直到接受友崎的遊戲指導後才對他改觀。
: 對同班的中村有好感，但對方沒有發覺，與友崎關係變好後一直希望雙方能和好相處，第三卷集體外宿後與中村的距離明顯拉近。
; {{nihongo|夏林花火|夏林 花火（なつバやし はなビ）}}
: 關友高中二年級，友崎的同班同學，個子很矮。
: 七海的好友，兩個人的關係好到接近百合。
; {{nihongo|水澤孝弘|水澤 孝弘（みずさわ たかひろ）}}

: 與日南一樣非常完美，不同的是自己的完美是天生的。
: 感覺敏銳，很早就察覺到友崎正在嘗試改變的事情，對努力改變的友崎表示敬佩，本身的談話技巧也是友崎的學習對象。
