《侠客行》是一款中国大陆[[MUD|MUD]]游戏，于1995年运行。

== 简介 ==

1995年，[[方舟子|方舟子]]因对台湾MUD游戏《[[东方故事2_天朝帝国|东方故事2]]》加入了大量玄幻要素不满<ref name="digi163">{{Cite web|url =http://digi.163.com/17/0505/06/CJLDH5VU001687H3.html|title=中国唯一连续运营20余年的网络游戏，还有什么人在玩？|publisher = digi163|date = 2017-05-05 |accessdate = 2017-05-05}}</ref>，集结翔少爷、时空、丁、草鱼组成了一个五人小组，开发文字MUD《侠客行》并在北美运营。<ref name="sina">{{Cite web|url =http://games.sina.com.cn/ol/n/2016-11-29/fxyawmm3812295.shtml|title=方舟子开发侠客行 国产网游都有过哪些尝试？|publisher = sina|date = 2016-11-29 |accessdate = 2017-05-05}}</ref>

1996年月，《侠客行》服务器遭受一名玩家的黑客攻击，玩家将其中的程序全部拷贝。团队警告了该玩家，结果玩家将所有文件公开在一个站点上供免费下载。《侠客行》小组为应对源代码泄露选择了主动将源代码开源化。<ref name="digi163"></ref><ref name="qq"></ref>

1996年5月更换服务器前后，方舟子和几个创始人因为工作变动，只留了一人看顾游戏。因为人手的关系，该创始人从玩家中提拔了几个人手（被称为巫师），赋予很大的权限来协助管理。但由于进入的巫师与其他创始人之间并不熟悉，但权限却平齐，引起权限的混乱。权限的混乱引起了玩家的不满，也让系统处于极度不稳定中。<ref name="qq">{{Cite web|url =http://tech.qq.com/a/20080716/000393.htm|title=方舟子误入《侠客行》：中国网游鼻祖轶事|publisher = qq|date = 2008-07-216 |accessdate = 2017-05-05}}</ref>

方舟子为了整顿下巫师队伍，收回部分权限，引起巫师队伍的不满。1997年10月，方舟子发出一个备忘录，就权限问题与巫师们进行沟通，但到了后来，巫师们自建小团队，并排斥创始人们。<ref name="qq"></ref>

1997年5月，方舟子宣布退出《侠客行》。1997年10月，方舟子发表声明，宣布只要在不商业化的前提下，任何人都可以任意使用、改写《侠客行》，任何人都无权垄断《侠客行》的使用、开发或商业化。<ref name="qq"></ref>

==影响==
《北大侠客行》于1996年运行，使用侠客行原作代码，最初服务器在设置于北京大学东门物理楼的一台服务器上，至今仍在运营。<ref>{{cite web|title=中国唯一连续运营20余年的网络游戏，还有什么人在玩？|url=https://zhuanlan.zhihu.com/p/26718501|website=知乎专栏|language=zh}}</ref>
