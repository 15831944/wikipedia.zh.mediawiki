《'''梦溪笔谈'''》是[[北宋|北宋]]的[[沈括|沈括]]所著的[[笔记|笔记体]]著作，大约成书于1086年－1093年，分为26卷，又《补笔谈》3卷，《续笔谈》1卷。因為写于[[润州|润州]]（今[[镇江|镇江]]）梦溪园而得名，收录了沈括一生的所见所闻和见解。

== 內容 ==
现存《梦溪笔谈》分故事、辩证、乐律、象数、人事、官政、权智、艺文、书画、技艺、器用、神奇、异事、谬误、讥谑、杂志、药议17目共609条。内容涉及[[天文|天文]]、[[数学|数学]]、[[物理|物理]]、[[化学|化学]]、[[生物|生物]]、[[地质|地质]]、[[地理|地理]]、[[气象|气象]]、[[医学|医学]]、[[工程技术|工程技术]]、[[文学|文学]]、史事、[[美术|美术]]及[[音乐|音乐]]等学科。

== 意義 ==
《夢溪筆談》是[[百科全书|百科全书]]式的著作、中国科学史上的重要文献，记载了北宋科学家[[沈括|沈括]]的科学知识及成就。

;[[数学|数学]]
*开创了“[[隙积术|隙积术]]”和“[[会圆术|会圆术]]”（[[s:夢溪筆談/卷18|卷十八 技藝]]）

;[[天文|天文]]
*指出[[北極星|极星]]不在[[天极|天极]]（[[s:夢溪筆談/卷07|卷七 象數一]]）
*得出[[冬至|冬至]]日短、[[夏至|夏至]]日長的结论
*改進了天文仪器
*[[历法|历法]]上大胆创新，提出《[[十二氣曆|十二氣曆]]》

;[[地理|地理]]
*以流水[[侵蚀作用|侵蚀作用]]解释奇异地貌成因。
*記載了植物和動物的[[化石|化石]]。（[[s:夢溪筆談/卷21|卷二十一 異事]]）

;物理
*發現[[磁偏角|磁偏角]]的存在（[[s:夢溪筆談/卷24|卷二十四 雜誌一]]）
*利用[[凹面镜|凹面镜]]（陽燧）聚光取火，並尋找凹面镜的焦點，發現其成像規律（[[s:夢溪筆談/卷03|卷三 辯證一]]）
*記載了「古人鑄鑑，鑑大則平，鑑小則凸」，即利用[[凹面镜|凹面镜]]成像小的特點鑄鏡（[[s:夢溪筆談/卷19|卷十九 器用]]）
*声音[[共振|共振]]实验（[[s:補筆談#樂律|《補筆談》樂律]]）

;技术
*[[毕昇|毕昇]]发明的[[活字印刷术|活字印刷术]]（[[s:夢溪筆談/卷18|卷十八 技藝]]）
*河工高超关于合龙堵口的方法
*建筑工匠[[喻皓|喻皓]]的《木经》及其建筑成就（卷十八 技藝）
*[[指南针|指南针]]
*炼铜、炼钢
*首次使用“[[石油|石油]]”一詞。沈括還嘗試用石油燃燒產生的黑煙製墨，效果頗佳。“石油”一詞一直沿用至今，但製墨技術未能寫入書中，今已失傳<ref name="千年一笔谈">{{Cite book|author=钱斌|title=《千年一笔谈》|publisher=商務印書館|isbn=978-7-100-08996-8}}</ref>（[[s:夢溪筆談/卷24|卷二十四 雜誌一]]）

关于社会历史现象部分，比如叛亂軍[[李顺|李顺]]的事迹、赋税实况、北宋时西北和北方军事、典制礼仪的演变，也有较详细的记载。

《夢溪筆談》屬於[[筆記|筆記]]體著作，這類著作發行量一般不大，但《夢溪筆談》卻是一個例外，甚至政府沒錢時也會印刷《夢溪筆談》以填補財政虧空。[[南宋|南宋]]期間，[[揚州|揚州]][[知州|知州]]要重建州學，就以刊印《夢溪筆談》補充經費。<ref name="千年一笔谈"/>

== 評價 ==
相較於[[李約瑟|李約瑟]]對《笔谈》的高度評價，中國学者近來對此書熱情稍減，首先沈括當年寫作動機只是“談謔”，<ref>《笔谈》自序说：“所录唯山间木荫，率意谈谑，不系人之利害者；下至闾巷之言，靡所不有；亦有得于传闻者，其间不能无缺谬。”</ref>這是宋代達官士子撰寫笔记的普遍動機，與[[吳曾|吳曾]]的《能改斋漫录》、[[王辟之|王辟之]]的《澠水燕談錄》及其他宋代筆記在本質上沒太大差異。相對[[秦九韶|秦九韶]]的著作《數書九章》而言，《筆談》在結構體制上明顯缺乏一種科學家對著述規範的自覺意識。<ref>虞雲國：《給《夢溪筆談》的評價潑點冷水》</ref>學者[[张家驹|张家驹]]指出：《笔谈》“采用笔记体裁，以至他的科学成就，受到许多限制。人们阅读这书，所得到的知识，多是一些零星的片段，而不是系统的完整的学说”。
