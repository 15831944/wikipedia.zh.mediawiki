《'''梦想演播厅'''》（{{lang-en|Dream Studio 48}}）是[[SNH48|SNH48 Group]]打造的大型竞演综艺，节目在筹备时曾用《星梦演播厅》这一名称。节目是为[[SNH48_Group第五屆總選舉|SNH48 Group偶像年度人氣總決選]]特别打造的大型竞演综艺，为给SNH48 Group成员们更多展示才华的舞台，同时也为了升级历年[[星夢劇院|SNH48公演]]中的总选拉票环节的形式及内容，节目将由66名成员同台进行才艺比拼。该节目6月10日起每周日在[[爱奇艺|爱奇艺]]正式播出<ref>[http://www.snh48.com/html/allnews/zixun/2018/0511/3067.html SNH48 GROUP 大型竞演综艺将启 66名成员同台比拼]</ref><ref>[http://www.snh48.com/html/allnews/zixun/2018/0609/3160.html SNH48 GROUP第五届总决选启动仪式落幕 投票通道正式开启]</ref>。

== 概要 ==
《梦想演播厅》是为[[SNH48_Group第五屆總選舉|SNH48 Group偶像年度人氣總決選]]特别打造的大型竞演综艺，为了给SNH48 Group成员们更多展示才华的舞台，同时也为了升级历年SNH48公演中的总选拉票环节的形式及内容。节目由来自[[SNH48|SNH48]]、[[BEJ48|BEJ48]]、[[GNZ48|GNZ48]]、[[SHY48|SHY48]]、[[CKG48|CKG48]]共计66名成员同台进行才艺比拼，每周人气冠军不仅将得到丰厚的总选推广资源，更有机会角逐SNH48 Group年度总决选演唱会开场曲的C位资格。在节目中，参演成员们既可以选择熟悉的歌舞进行对抗PK，也可以展示在其他方面的特殊才艺。全面开放的表演类型选择，将不仅铸就SNH48 GROUP历史上规模最庞大的成员才艺对抗，也将给成员们除了剧场拉票外，一个展示多面自我的全新舞台<ref>[http://www.snh48.com/html/allnews/zixun/2018/0511/3066.html “砥砺前行”SNH48 GROUP第五届偶像年度人气总决选5月11日启动]</ref>。

== 比賽方式 ==
#报名：SNH48 Group所有在籍成员都可报名，共选取66名成員获得竞演机会。上届总决选TOP 66成员报名即可获得席位，而剩余席位由助力数最高的成员获得。任何成员因通告、学业、健康等不可抗力缺席录制。参赛名额将顺延给口袋48中未入选的助力数最高的成员<ref>[https://h5.48.cn/2018apppage/signupXM/index.html 《星梦演播厅》口袋48报名页面]</ref>。
#选位：参与节目的66名成員，将通过口袋48抽签决定录制期数。
#表演内容：每集将有竞演进行现场竞演。成員將獲得3－5分钟的才艺展示时间及拉票环节。成員可根据自身要求，自行选择单人solo或邀请辅助嘉宾进行表演。
#投票：成员的最终票数由成员互投和粉丝线上投票产生。当集表演结束后，由表演的成员互相投票，每位成员拥有5票，3票明票，2票暗票，不可弃权，最多为同一成员投2票。而当集表演结束后，粉丝可在24小时内通过口袋48的投票通道为支持的成员投票。以保证任何一方的投票，将无法单方面左右最终结果。这也将是对成员实力和人气的双重考验<ref>[https://h5.48.cn/2018apppage/signupXM/rule.html 《星梦演播厅》口袋48查看活动规则&奖励页面]</ref>。
#競猜活動：每一期11位选手在名单公示后，在口袋48会有相应的排名竞猜进行。粉丝可以预测每期《梦想演播厅》第1-11的排名，顺位完全相同即可成功，第一个成功的粉丝将获得当期11位成员每人1张，总计11张的拍立得照片，其余成功的粉丝能获得1张随机拍立得照片<ref>[https://weibo.com/2689280541/GjPYR1kye 【#SNH48梦想演播厅# 第一期11人参演名单正式公布】]</ref>。

== 节目奖励 ==
每一周的周冠军将获得五地[[星夢劇院|星夢劇院]]单人一周推广资源（包括但不限于个人宣传片播放、显著广告位等）、SNH48 Group五团官博置顶、微博推广资源及口袋48单人开屏等奖励。同时，66名参赛成员还将获得[[SNH48_Group第五屆總選舉|SNH48 Group第五屆總選舉]]开场曲的表演资格。而依据各周成绩，在对66名成员进行综合排序后，最终的总冠军将获得开场曲的C位名额。
