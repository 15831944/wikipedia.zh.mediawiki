'''梦鸽'''{{BD|1966年||||catIdx= L}}，原名'''刘清娣'''，湖北省沙市人，毕业于中央社会音乐学院、[[中国音乐学院|中国音乐学院]]、[[解放军艺术学院|解放军艺术学院]]，著名歌唱家[[李双江|李双江]]的妻子，儿子[[李天一|李天一]]，现为[[总政歌舞团|总政歌舞团]]歌唱家、[[全国青联|全国青联]]常委、[[中共中央|中共中央]]直属机关青联委员，国资委特邀青联委员，[[中国音乐家协会|中国音乐家协会]]会员。梦鸽、[[韩红|韩红]]、[[谭晶|谭晶]]、[[许飞|许飞]]被誉为李双江“四大关门女弟子”。<ref>{{cite web |url=http://edu.sina.com.cn/bschool/2013-02-26/1713372183.shtml |title=商院人物：盘点李双江四大关门女弟子  |accessdate= 2013年2月26日 |publisher= 新浪网 |work= 人民网|language=zh-cn}}</ref>

== 生平 ==
1966年，梦鸽出生在[[湖北省|湖北省]][[沙市|沙市]]，受到家中母亲的影响，孩提时就热爱唱歌。梦鸽第一次登台演出时她还只有8岁。1977年梦鸽考入沙市歌舞团，之后，梦鸽又考入中央社会音乐学院、[[中国音乐学院|中国音乐学院]]，其老师是[[金铁霖|金铁霖]]和[[邹文琴|邹文琴]]。梦鸽大学毕业之后，被[[总政歌舞团|总政歌舞团]]录取。

== 家庭 ==
1990年，24岁的梦鸽和51岁的歌唱家[[李双江|李双江]]在[[北京市|北京市]]结婚，梦鸽曾是李双江的学生。两人育有一子[[李天一|李天一]]（后改名李冠丰）。<ref>{{cite web|url=http://ent.people.com.cn/n/2012/0710/c1012-18485461-2.html |title=揭李双江梦鸽相差27岁师生恋  老来得子曾流产 |accessdate=2012年7月10日 |publisher=人民网 |work= |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20130226180724/http://ent.people.com.cn/n/2012/0710/c1012-18485461-2.html |archivedate=2013年2月26日 }}</ref><ref>{{cite web |url= http://ent.163.com/13/0223/02/8OC5KUJB00032DGD.html|title= 李双江梦鸽相差27岁“师生恋”修成正果 58岁得子 溺爱非常|accessdate=2013-02-23 |publisher=网易 |work= |language=zh-cn}}</ref>
