'''癮科技'''（Cool3C）是一家[[臺灣|台灣]]的3C新聞網站，報導與3C產品相關的新聞消息，前身為 [[Engadget|Engadget]] 的中文版網站<ref>{{Cite news|url=https://www.cool3c.com/article/47984|title=【新聞稿】 - 終止協助經營Engadget中文版網站 癮科技媒體平台正式成立 - 癮科技|last=atticus|date=2011-06-28|newspaper=癮科技|accessdate=2018-06-01|language=zh-hant}}</ref>，曾隸屬於 FunMakr Media 博客邦有限公司<ref>{{Cite web|url=https://foursquare.com/v/funmakr-media-%E5%8D%9A%E5%AE%A2%E9%82%A6%E6%9C%89%E9%99%90%E5%85%AC%E5%8F%B8/4d01e66de5f16ea8d106a828|title=FunMakr Media 博客邦有限公司|accessdate=2018-06-01|work=Foursquare}}</ref>，目前由台灣公司上癮科技股份有限公司獨立經營，站長為吳顯二（Atticus Wu）<ref>{{Cite web|url=https://elite.ezprice.com.tw/elite/4908-%E5%90%B3%E9%A1%AF%E4%BA%8C/|title=吳顯二 - 癮科技網站站長 - EZprice 電商百大人才|accessdate=2018-06-01|work=elite.ezprice.com.tw}}</ref>。

相關網站有：[https://shop.cool3c.com 好東西賣場]、[https://uccu.cool3c.com/ 壞朋友]預售平台、[https://course.cool3c.com/ 慕課]線上課程服務等。

近期於 2017年1月進行改版<ref>{{Cite news|url=https://www.cool3c.com/article/116148|title=癮科技2017改版搶先看：全新版面、獨家專欄、年度活動 - 癮科技|last=Tandee|date=2017-01-03|newspaper=癮科技|accessdate=2018-06-01|language=zh-hant}}</ref>。

== 特點 ==
癮科技以編輯獨特觀點報導各種科技相關內容。站長吳顯二（Atticus Wu）本人即為第一代編輯，是第一代的科技媒體，最早的文章可回溯至 2005 年。

除日常消費科技產品最新新聞報導、開箱文等內容，亦提供獨具編輯特色的各種專欄，如「[https://www.cool3c.com/column/one-pic 一圖看懂]」系列以簡明的手繪圖片說明科技、生活相關資訊為特色、「[https://www.cool3c.com/column/old-game-collection 巷口遊戲攤]」則介紹從第一代電玩遊戲至今的主機、與經典遊戲等內容。以及年度科技活動整理如「COMPUTEX台北國際電腦[https://www.cool3c.com/event/computex-taipei 展]」、「蘋果 iPhone 秋季發表[https://www.cool3c.com/event/apple-iphone-special-event 會]」等。

== 網站規模 ==
根據 [https://www.comscore.com/ ComScore 調查數據],癮科技每月約有 283 萬不重複使用者、900 萬次頁面瀏覽，目前在台灣不分類網站中排名 37。

== 團隊組成 ==
* 開發部：癮科技系統維運，新功能研究與開發，做出有趣的功能讓大家玩，還有努力不要讓網站長蟲
* 業務部：主要就是賺大家的薪水！ 開發客戶、廣告操作、等等都難不倒他們
* 編輯部：是癮科技大家看到的門面，產出讓大家認為是「癮科技」的文章、負責熬夜看發表會和去人擠人的活動會場，還有整理資料，努力寫稿
其他還有專門提出爆炸性意見以及開會總是最開心的的吳老闆和負責營運的單位確保薪水都有發出來

== 開發技術 == 
網站使用 PHP+MySQL 進行開發，最早期的系統是 Drupal，經歷了版本5~7，由於是功能及架構皆已完成，對於當時開發資源較缺少的狀況有很大的幫助。在準備進入 Drupal 8 時，開發團隊已經成長很多，網站需求也越來越五花八門，所以開始討論使用其他的 Framework 來適應現在的狀況。期間有試過了 CodeIgniter，但後來發現架構雖然很彈性，但對新PHP技術的相容性沒那麼好，很多套件也需要再開發過，所以就轉而試用了當時常被討論的 Laravel<ref>[https://laravelconf.tw/zh-TW/sponsors 2018贊助 Laravel Conference 網站資料]。</ref>。
目前大多的系統後端皆為 Laravel，前端有 Vue, Bootstrap, jQuery, Font Awesome。

== 網站歷史發展 ==
前身 Engadget 中文版從 2005 年站長 Atticus Wu 發表第一篇文章開始,為台灣 3C 科技部落格先驅。2011 年每月不重複訪客達 120 萬人,為當時台灣最大 3C 科技數位媒體。 2011年6月28日，[[Engadget|Engadget]]因與 Engadget 母公司 AOL 經營理念上的歧見,結束六年的合作,<ref>[http://bbs.cool3c.com/article/47984 【新聞稿】 - 終止協助經營Engadget中文版網站癮科技媒體平台正式成立，同年六月並同時推出癮科技 2.0。
Cool3c{{webarchive|url=https://web.archive.org/web/20110701112331/http://bbs.cool3c.com/article/47984 |date=2011-07-01 }}</ref>，原 Engadget 繁體中文站更名為「Engadget中文版」，並由Engadget香港編輯接手。現在癮科技與 Engadget 兩家網站已無任何關係，為獨立經營的兩個不同網站。
