'''罘罳峰'''（拼音：fú sī），地处于[[宁乡县|宁乡县]][[流沙河镇|流沙河镇]]西北面，<ref name="罘罳峰的迁居潮 ">{{cite web |url=http://news.163.com/10/0219/09/5VSGJV0K000120GR.html|title=《罘罳峰的迁居潮》|accessdate=2010-02-19|work=|publisher=网易|language=zh-hans}}</ref>是横贯于[[湖南省|湖南省]]境内的[[雪峰山|雪峰山]]一座尾峰，海拔近500米，平地突起，仿佛是帝王寝宫中的[[屏风|屏风]]，古谓[[罘罳|罘罳]]，故而得名。又因为其形状酷似[[猴子|猴子]]，故又名“猴子峰”。

== 地理 ==
[[File:Fusi_Mountain_3.jpg|thumb]]

罘罳峰地质地貌为[[丘陵|丘陵]][[砂类土|砂类土]]，故而[[流沙河镇|流沙河镇]]盛产[[水沙|水沙]]。

== 交通 ==
[[宁乡县城|宁乡县城]]沿[[209省道|209省道]]往[[娄底|娄底]]，过了[[烂山峡|烂山峡]]，而后出现的就是突兀的罘罳峰。<ref name="罘罳峰的迁居潮 " />

== 文化 ==
罘罳峰在[[清朝|清朝]]曾列为“[[宁乡十景|宁乡十景]]”之一。[[隋末|隋末]][[唐初|唐初]]，山顶曾建有[[金风寺|金风寺]]，先道后佛并存，曾为[[回龙山|回龙山]]之分寺。清[[康熙|康熙]]年间重修。[[民国|民国]]二十二年再维修。每年六月十七日，由回龙山主持和尚带上弟子到此院与驻寺和尚一道做法事三天，同时唱[[木偶戏|木偶戏]]。<ref name="罘罳峰的迁居潮 " />据传，[[芙蓉山|芙蓉山]]的[[清庵祖师|清庵祖师]]曾投师于[[金风寺|金风寺]]。金凤寺内有清庵祖师殿。香火鼎盛时期，金风寺规模较大，有大小房屋十二间，有四个神殿。正殿是玄帝殿，内供玄帝神象，是寺内最大的殿堂。右为清庵祖师殿，左为侠神殿．后为九天玄女殿。两边有[[二十四位诸天佛祖|二十四位诸天佛祖]]和[[十八罗汉|十八罗汉]]神像。神像均用木雕制，金装其表，神态各异，栩栩如生。寺院毁于二十世纪五十年代，罘罳峰上只留下金风寺遗址。民国时期，罘罳峰下的林山寺改建成[[罘罳完小|罘罳完小]]，也称[[国立宁乡第五高小|国立宁乡第五高小]]，即现今[[宁乡七中|宁乡七中]]和[[草冲中学|草冲中学]]的前身。[[中华人民共和国|中华人民共和国]][[刘瑞林|刘瑞林]]将军、[[刘雪初|刘雪初]]部长等都曾就读于此。
