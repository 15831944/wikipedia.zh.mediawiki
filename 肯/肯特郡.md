'''肯特郡'''（{{lang-en|Kent}}，[[英國|英國]][[英格蘭|英格蘭]][[東南英格蘭|東南部]]的[[英格蘭的郡|郡]]。以人口計算，[[坎特伯雷市|坎特伯雷市]]是第1大（亦是唯一一個）[[英國的城市地位|城市]]、第1大[[英國的自治市鎮|自治市鎮]]（Borough），[[梅德斯通區|梅德斯通]]是第2大自治市鎮；[[達特福德|達特福德]]是第1大鎮（Town），[[梅德斯通|梅德斯通]]是第2大鎮（亦是[[郡治|郡治]]）。[[坎特伯雷|坎特伯雷]]有2個[[世界文化遺產|世界文化遺產]]─[[聖奧古斯丁修道院|聖奧古斯丁修道院]]、[[坎特伯雷座堂|坎特伯雷座堂]]，又是[[坎特伯雷大主教|坎特伯雷大主教]]的起源地，是郡內最著名的歷史名城。

肯特南與[[東薩塞克斯郡|東薩塞克斯]]郡相鄰，西南（下小半部分）與[[西薩塞克斯郡|西薩塞克斯]]郡相鄰，西南（上大半部分）與[[薩里郡|薩里]]郡相鄰，西北（下半部）與[[大倫敦|大倫敦]]相鄰，西北（上半部）與[[埃塞克斯郡|埃塞克斯]]郡相鄰。肯特北部的[[達特福福德|達特福福德]]、[[格雷夫森德_(肯特郡)|格雷夫森德]]、[[胡半島|胡半島]]北臨[[泰晤士河河口|泰晤士河河口]]。[[英法海底隧道|英法海底隧道]]在肯特東部的貫穿[[多佛爾區|多佛爾區]]、[[多佛尔海峡|多佛尔海峡]]、[[法国|法国]][[加來海峽省|加來海峽省]]。

肯特是34個[[非都市郡|非都市郡]]之一，實際管轄12個[[非都市區|非都市區]]，[[面積|佔地]]3,544[[平方公里|平方公里]]（[[英格蘭的非都市郡列表_(以面積排列)|第10]]），有1,382,800[[人口|人口]]（[[英格蘭的非都市郡列表_(以人口排列)|第1]]）；如看待成48個[[名譽郡|名譽郡]]之一，它名義上包含多1個[[單一管理區|單一管理區]]─[[梅德韋|梅德韋]]，[[面積|佔地]]增至3,736[[平方公里|平方公里]]（[[英格蘭的名譽郡列表_(以面積排列)|第10]]），[[人口|人口]]增至1,634,500（[[英格蘭的名譽郡列表_(以人口排列)|第7]]）。

肯特被称为“英格兰的花园”，但是在过去当地的田园风景已被农庄工业取代了。此外在过去数世纪中其它工业也在肯特立足。棉纺、造铁、造纸、水泥和工程都曾在此兴旺。渔业和旅游业今天依然在海岸地区非常重要。20世纪东肯特煤田被开采，今天在[[邓杰内斯|邓杰内斯]]有一座核电站。不过[[泰乃特|泰乃特]]被看作是英格兰东南发展最少的地区。海港、海底隧道和两条英国机动车路是英国通向欧洲大陆的连接。漫斯顿和罗彻斯特各有一座机场。著名的肯特人有[[查尔斯·狄更斯|查尔斯·狄更斯]]和[[查尔斯·罗伯特·达尔文|查尔斯·罗伯特·达尔文]]、[[温斯顿·丘吉尔|温斯顿·丘吉尔]]等。

== 历史 ==
[[斯旺斯扎|斯旺斯扎]]的采石场证明早在[[旧石器时代|旧石器时代]]肯特就已有人居住。麦德伟的巨石阵是[[新石器时代|新石器时代]]建造的。从[[青铜器时代|青铜器时代]]、[[铁器时代|铁器时代]]到[[古罗马|罗马时代]]都有遗迹留存下来。肯特的名字来自[[布立吞亚支|布立吞亚支]]中的''Cantus''（边缘），它本来指肯特东部的海岸地区。[[恺撒|恺撒]]于前51年称之为''Cantium''。5世纪时肯特东部有一个独立的王国。

坎特伯雷是[[圣公会|圣公会]]的宗教中心，[[圣奥格司丁|圣奥格司丁]]的教座。一般圣奥格司丁被看作是将[[基督教|基督教]]带到英格兰的人（597年）。[[威廉一世_(英格兰)|威廉一世]]入侵后肯特人自称为是“不败的”，（错误地）宣称他们将[[诺尔曼人|诺尔曼人]]吓跑了。中世纪里肯特爆发多次农民起义，包括[[瓦特·泰勒|瓦特·泰勒]]领导的起义。1553年[[托马斯·怀特|托马斯·怀特]]领军从肯特进入伦敦反对[[玛丽一世_(英格兰)|玛丽一世]]。[[托马斯·贝凯|托马斯·贝凯]]死后肯特成为一个朝圣地。坎特伯雷的宗教地位促使[[杰弗里·乔叟|杰弗里·乔叟]]写了对英国的书写语言有非常关键的意义的《坎特伯雷故事》。

17世纪英国与欧洲大陆上[[荷兰|荷兰]]和法国的关系紧张使得肯特成为一个军事重镇。1667年荷兰海军袭击麦德伟船坞后沿海岸修建了许多要塞。[[第二次世界大战|第二次世界大战]]中肯特的机场在[[不列颠战役|不列颠战役]]中起了重要作用。肯特的居民区则经常受空袭。

== 工业 ==
中世纪时北唐斯丘陵对两个工业非常重要：炼铁业和纺织业。肯特农业发达，郡内有许多大的兰花和[[啤酒花|啤酒花]]园，因此有时被称为“英格兰的花园”。农村里有许多特别的[[烤房|烤房]]用来烤干啤酒花。不过今天许多这些烤房已经被改为民居。伦敦附近的市场上鲜花众多。后来造纸、水泥和煤矿比较重要。造纸业的发达是出于肯特的众多水资源。一开始纸磨都位于河流附近。19世纪末大型的现代化造纸厂出现。水泥工业是在19世纪出现的，当时[[石灰岩|石灰岩]]很丰富，而大型建筑的建造需要许多水泥。今天依然可以看到这些工业留下来的石灰岩堆。煤矿主要集中在东肯特。最旺时有1900多个矿运行，今天所有煤矿都已关闭。

== 政治 ==
=== 传统 ===
传统地肯特被麦德伟河分为东西两部分。宗教上肯特被分为坎特伯雷和罗彻斯特两个教区。

== 地理 ==
下圖顯示英格蘭48個名譽郡的分佈情況。肯特郡，南與[[東薩塞克斯郡|東薩塞克斯郡]]郡相鄰，西南（下小半部分）與[[西薩塞克斯郡|西薩塞克斯郡]]郡相鄰，西南（上大半部分）與[[薩里郡|舒梨郡]]相鄰，西北（下半部）與[[大倫敦|大倫敦]]相鄰，西北（上半部）與[[埃塞克斯郡|雅息士郡]]相鄰。

肯特是英格兰最东南的郡，向北它滨泰晤士河和[[北海_(大西洋)|北海]]，向南滨多佛尔海峡和[[英吉利海峡|英吉利海峡]]。欧洲大陆隔海峡仅21英里。郡内有一系列从东向西走向的小山脊。这些山脊是1000到2000万年前的造山运动中形成的。风化使得山脊之间形成了沟谷。郡内最高点高251米。肯特最显眼的地势是其白色的悬崖。[[维尔德地带|维尔德地带]]的名称来自于日耳曼语中的''wald''，意为森林。今天那里依然有许多茂密的森林。一些地区长年无法穿越。

[[麦德伟河|麦德伟河]]是肯特的水道之一，它源于[[伊甸桥|伊甸桥]]附近，首先向东流约40千米，然后再梅德斯通附近北折。在罗彻斯特它穿越[[北唐斯丘陵|北唐斯丘陵]]最后在[[希尔内斯|希尔内斯]]附近注入泰晤士河。它是泰晤士河的最后一条支流。涨潮时潮水可以一直流到[[阿灵顿|阿灵顿]]。过去船只一直可以行到[[汤布里奇|汤布里奇]]。
