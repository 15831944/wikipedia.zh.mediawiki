'''凯里市'''是[[贵州省|贵州省]][[黔东南苗族侗族自治州|黔东南苗族侗族自治州]]州政府所在地，是一个以[[苗族|苗族]]为主体，多[[民族|民族]]聚居的城市。“凯里”系[[苗語黔東方言|苗語黔東方言]]北部土語音译，意为[[木佬人|木佬人]]的田地。苗语称木佬人为“凯”，田为“里”，“凯里”即“木佬人的田那个地方”。

截止2014年底，凯里市总面积1616.51平方公里，其中建成区面积67平方公里，超过遵义市（遵义市2013年底建成区61平方公里）为贵州省建成区第二大城市，凯里市总人口65.61万，其中城区常驻人口48万人。

== 历史 ==
凯里设治历史悠久，早在明清时期曾有“小京州”美称。
* [[春秋时期|春秋时期]]，现凯里属[[南蛮|南蛮]][[牂牁|牂牁]]国。
* [[战国时期|战国时期]]，现凯里属[[夜郎|夜郎]][[且兰国|且兰国]]。
* [[秦朝|秦]][[汉朝|汉]]时期，现凯里属且兰县。
* [[隋朝|隋朝]]，现凯里属宾化县。
* [[元朝|元朝]]始设沾，有凯里安抚司，恭焦溪蛮夷军民长官司，臻洞长官司等。
* [[明|明]][[洪武|洪武]]十四年（公元1381年）置清平堡。
* 洪武二十二年（公元1389年）升为长官司，属[[平越卫|平越卫]]。
* 洪武二十三年六月初四（公元1390年7月16日）置[[清平卫|清平卫]]于司北，改隶清平卫。
* [[弘治|弘治]]七年五月二十七日（公元1494年6月23日）置[[都匀府|都匀府]]，同时清平长官司废，置清平县，隶都匀府。
* [[嘉靖|嘉靖]]八年二月二十八日（公元1529年4月6日）凯里安抚司由四川布政司改隶贵州布政使司，属清平卫。
* [[万历|万历]]二十九年（公元1601年）改隶平越军民府。
* [[万历|万历]]三十五年（公元1607年），降凯里安抚司为凯里长官司。
* [[康熙|康熙]]七年七月安二（公元1668年8月9日），清平县入麻哈州。
* [[康熙|康熙]]十年，恢复清平县，凯里长官司隶清平县
* [[雍正|雍正]]十年（公元1732年）开辟苗疆。进一步改土归流，兵设凯里已，隶属丹江厅，领十三屯堡军
* [[雍正|雍正]]十二年，添设清平县丞一员分驻凯里。称凯里县丞
* [[乾隆|乾隆]]三年十一月（公元1739年1 月），将镇远县臻洞长官司革除，以该司所属臻洞司，赏郞等江21寨就近改隶清平县凯里县丞
* [[民国|民国]]3年（公元1914年），因清平县与山东省东 昌府清平县重名，故以境内有香炉山改称炉山县，凯里县丞改称凯里分县。
* 1949年11月，中共政权占领[[炉山县|炉山县]]。
* 1951年1月，凯里苗族自治区成立。
* 1956年7月，凯里设[[县|县]]并成为[[黔东南苗族侗族自治州|黔东南苗族侗族自治州]]首府。
* 1983年8月19日，[[中华人民共和国国务院|国务院]]批准撤消凯里县，并设立凯里市（县级市）。
* 1984年1月，凯里市正式成立。

== 地理 ==
[[File:Xijiang_Qianhu_Miaozhai.Skyline.jpg|thumb]]]]
凯里市地处于云贵高原东侧的梯级状大斜坡地带，属[[亚热带|亚热带]]湿润[[亞熱帶季風氣候|季风气候]]，年均气温 16.1℃，最高气温37℃，最低气温零下4-7℃。境内有清水江、重安江、巴拉河等大小河溪153条，河流径流总量39.89亿立方米，水能资源理论蕴藏量5.4万千瓦。

凯里全市有野生动物有数百种、药用动植物有428种、矿产资源主要有煤、铁、石英砂岩、石油和天然气等，此外还有石油和天然气矿点各一处。

== 行政区划 ==
2012年，凯里市将两个省级经济开发区炉碧经济开发区、凯里经济开发区所管辖的麻江县碧波镇、下司镇纳入凯里市行政范围，至此凯里市共管辖7个街道，9个镇，2个乡。

* 乡：[[大风洞乡|大风洞乡]]、[[凯棠乡|凯棠乡]]。

== 旅游 ==
自然风光主要有香炉山、金泉湖和一批景观独特的溶洞。

另外，凯里还有大大小小的[[民族节日|民族节日]]136个，被称为“百节之乡”。主要的传统节日有[[苗年|苗年]]、[[爬坡节|爬坡节]]、[[吃新节|吃新节]]、[[姊妹节|姊妹节]]、[[六月六|六月六]]、[[牯藏节|牯藏节]]、[[芦笙会|芦笙会]]等。

凯里市盛行各种民族工艺，其中以挑花、刺绣、织锦、蜡染、银饰品最为著名。

== 饮食 ==
凯里市的饮食以酸、辣为主。
