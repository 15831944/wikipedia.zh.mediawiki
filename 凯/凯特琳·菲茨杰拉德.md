'''凯特琳·菲茨杰拉德'''（{{lang|en|Caitlin FitzGerald}}）是美国演员兼编剧，知名角色有[[Showtime|Showtime]]电视网的热门美剧《[[性爱大师|性爱大师]]》中的莉比·马斯特一角。

== 早年经历 ==

凯特琳自小就对表演产生兴趣，并在许多社区剧院和学校的活动中参与表演。她在美国顶级私立中学康科德中学（Concord Academy）读完寄宿制高中后<ref>{{cite web |url=http://www.concordacademy.org/news/detail.aspx?pageaction=ViewSinglePublic&LinkID=2155&ModuleID=211 |title=存档副本 |accessdate=2015-07-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20160305022653/http://www.concordacademy.org/news/detail.aspx?pageaction=ViewSinglePublic&LinkID=2155&ModuleID=211 |archivedate=2016-03-05 }}</ref><ref>http://aritzia.com/Profile%3A-Caitlin-FitzGerald/i21-ea-caitlin-fitzgerald,default,pg.html</ref>，被[[纽约大学|纽约大学]]提克斯艺术学院录取，主攻戏剧专业<ref>http://www.elle.com/pop-culture/reviews/caitlin-fitzgerald-390126</ref>。她还在伦敦的[[皇家戏剧艺术学院|皇家戏剧艺术学院]]学习过[[莎士比亚|莎士比亚]]<ref>http://www.accesshollywood.com/masters-of-sex-caitlin-fitzgerald-on-libbys-marriage-struggles_article_86834</ref>。

== 演艺经历 ==

凯特琳·菲茨杰拉德在《[[愛_找麻煩|-{zh:愛 找麻煩;zh-hans:爱 找麻烦;zh-hant:愛 找麻煩;zh-hk:複雜關係;zh-mo:複雜關係;zh-tw:愛 找麻煩;}-]]》（ It's Complicated ），《[[绯闻女孩|绯闻女孩]]》（ Gossip Girl ）（S4E13）, 《待解救的少女》（ [http://en.wikipedia.org/wiki/Damsels_in_Distress Damsels in Distress]）以及《新婚夫妇》（[http://en.wikipedia.org/wiki/Newlyweds_(film) Newlyweds]）等影视作品中都有过不俗表现。

2010年，凯特琳参演了[[易卜生|易卜生]]的话剧《海达 高布乐》（[http://en.wikipedia.org/wiki/Hedda_Gabler Hedda Gabler]）。这是一部非百老汇传统的先锋戏剧，在仅能容纳25名观众的私人住宅里演出。凯特琳对于能出演“出自非莎士比亚之手的最伟大的女性角色”<ref>http://www.bloomberg.com/news/2010-08-10/secret-set-streep-s-movie-kid-make-hedda-hotter-than-pacino.html</ref>感到十分兴奋。《[[纽约时报|纽约时报]]》的首席戏剧评论家 Ben Brantley 对她的表演进行评价说: "无论是腿、眼睛还是颧骨，外加仿佛就是为拍特写而生的脸庞，她的美让人陶醉。" 但是，他觉得凯特琳缺乏应有的庄重，总给人一种印象，像是“一个刚进贵族学校的转校生，可能离受人欢迎还有一大段差距”<ref>http://theater.nytimes.com/2010/08/30/theater/reviews/30hedda.html</ref>。

2012年，凯特琳出演了独立电影 Like the Water, 同时也是该剧的编剧之一。该戏在她的老家[[缅因州|缅因州]]拍摄。 电影讲述了一位童年好友的死亡，灵感来源于凯特琳的一名同学的突然离世<ref>http://www.nytimes.com/2012/07/29/nyregion/the-short-life-and-lonely-death-of-sabrina-seelig.html</ref>。同年她还参演了电影《菲茨杰拉德家族的圣诞节》，该片由艾德·伯恩斯（[http://en.wikipedia.org/wiki/Ed_Burns Ed Burns]）指导<ref>http://www.imdb.com/title/tt2331880/</ref>。

2013年，凯特琳出演热门美剧《[[性爱大师|性爱大师]]》， 影评人奖（[http://en.wikipedia.org/wiki/Indiewire Indiewire]） 认为她的角色是所有电视剧角色中最没有得到应有评价的一个: "莉比·马斯特这个角色非常容易让人觉得无聊，而不会让人心生怜悯……菲茨杰拉德完美地演绎了莉比的娇弱婀娜，展现出了养尊处优的高贵形象，但又不失少女的天真——她有的时候会叫自己的丈夫“爸比” （Daddy）—— 她用自己的表演告诉大家，她不是一个表现保守价值观或无知形象的替身……菲茨杰拉德把莉比的感性和率真表露无遗。"<ref>http://www.indiewire.com/article/television/8-underappreciated-supporting-actors?page=2#articleHeaderPanel</ref>

接下来她将于2014年出演罗斯·卡兹（Ross Katz）导演的独立喜剧Brother’s Keeper。其中她扮演尼克·克罗（[http://en.wikipedia.org/wiki/Nick_Kroll Nick Kroll] ）的女朋友 Kat, 在男友的科技发明失败后把他甩了<ref>http://www.deadline.com/2013/12/colm-feore-joins/</ref>。

== 家庭 ==

菲茨杰拉德（FitzGerald）是一个在历史上有名的[[新英格兰|新英格兰]]家族<ref>http://ohnotheydidnt.livejournal.com/80292158.html</ref>。凯特琳出生并成长在美国的缅因州卡姆登市<ref>http://freepressonline.com/main.asp?SectionID=41&SubSectionID=59&ArticleID=13860&TM=63721</ref>。她的父亲Des FitzGerald是美国跨国企业康地集团（ContiGroup）的前任CEO<ref>https://bangordailynews.com/2012/12/03/business/maines-venture-capital-fund-hires-well-known-entrepreneur-to-mentor-companies/</ref>，母亲 Pam Allen是《针织达人迷》(Knitting for Dummies)一书的作者，同时也是Quince & Co毛纱公司的创始人<ref>http://www.mainebiz.biz/article/20120806/CURRENTEDITION/308029992/profitable-patterns-%7C-pam-allen-quince-&-co</ref>。她的弟弟 Ryan是[[耶鲁大学|耶鲁大学]]环境管理专业的硕士<ref>http://www.mainebiz.biz/article/20120806/CURRENTEDITION/308029992/profitable-patterns-%7C-pam-allen-quince-&-co</ref>。

凯特琳从小父母离异， 她有两个同父异母的弟弟，Alexander 和Cooper<ref>http://books.google.com.hk/books?id=1bZqj3R9viEC&lpg=PA146&ots=gmiJ8emT9S&dq=lucinda+ziesing+sons&pg=PA146&redir_esc=y&hl=zh-CN&sourceid=cndr</ref>。 她的爷爷Desmond FitzFerald是[[肯尼迪|肯尼迪]]政府时期的CIA（[[美国中央情报局|美国中央情报局]]）副局长<ref>http://www.foodandwine.com/articles/the-maine-event</ref>，她的姑姑Frances FitzGerald是[[普利策奖|普利策奖]]的获得者<ref>http://www.foodandwine.com/articles/the-maine-event</ref>。

== 感情 ==

2013年，凯特琳被狗仔队抓拍到和《[[性爱大师|性爱大师]]》的男一号[[米高·辛|麦克·辛]]疑似约会，麦克·辛在剧中扮演凯特琳的丈夫。两人被拍到在加州马里布的一次晚餐后接吻。两人经常一起出现在慈善活动、音乐节、餐馆等各种场合<ref>http://ohnotheydidnt.livejournal.com/80292158.html</ref>。
