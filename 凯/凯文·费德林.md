'''凯文·费德林'''（{{lang-en|'''Kevin Federline'''}}，{{bd|1978年|3月21日|||catIdx= F费德林}}）出生于[[美国|美国]][[加利福尼亚州|加利福尼亚州]][[弗雷斯诺|弗雷斯诺]]，为美国舞蹈演员、饶舌歌手和时尚模特，前妻是美国巨星[[小甜甜布蘭妮|小甜甜布蘭妮]]，因為嫁給布蘭妮才開始走紅，備受爭議性。

== 生平 ==
1978年，费德林出生于[[美国|美国]][[加利福尼亚州|加利福尼亚州]][[弗雷斯诺|弗雷斯诺]]，父亲迈克是一个[[汽车|汽车]]修理工，母亲朱莉是来自[[俄勒冈州|俄勒冈州]]的一个银行出纳员。费德林有一个哥哥克里斯。8岁时，费德林的父母离婚。11岁时，他和哥哥克里斯跟着父亲搬迁到弗雷斯诺的房子居住。费德林读[[高中|高中]]的9年级时，就辍学了，他自学[[跳舞|跳舞]]，并且加入了一个非营利性质的舞蹈组织。<ref name="people">Dinh, Mai; Murphy, Janet, ed. [http://www.people.com/people/kevin_federline/biography Kevin Federline biography (1986-2006)]. [[People_(magazine)|People.com]]</ref>

== 个人生活 ==
费德林的第一任妻子是女演员[[莎尔·杰克逊|莎尔·杰克逊]]，两人育有2个孩子，一个是女儿柯丽，一个是儿子卡勒博。<ref name="fish ">{{cite web|url=http://www2.sdnews.com.cn/ent/2006-12/8_408055.html|title=布兰妮老公凯文·费德林：我现在感觉很不错|accessdate=2006-12-08|publisher=山东新闻网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20070120033607/http://www2.sdnews.com.cn/ent/2006-12/8_408055.html|archivedate=2007-01-20}}</ref>

2004年，费德林和美国女巨星[[布兰妮·斯皮尔斯|布兰妮·斯皮尔斯]]结婚，2年后的2006年11月，布兰妮提出离婚，他们育有2个孩子，一个叫肖恩·普雷斯顿，一个叫杰登·詹姆士，布兰妮在离婚官司上提出抚养孩子的要求，只同意给费德林探视权。<ref name="fish " /> 即使两人已经离婚，但是费德林表示会永远爱孩子和家人，永远支持布兰妮：“我会永远爱我这些孩子们的妈妈，只要她们需要我，我就会出现在她们身边表示支持。与孩子和家庭的关系对我来说是一辈子的，所以我会永远珍惜。”<ref>{{cite web |url= http://yule.sohu.com/20080328/n255968811.shtml|title=布兰妮前夫凯文-费德林 表示永远爱她支持她 |accessdate= 2008年3月28日|publisher= 搜狐网|work= |language =zh-cn}}</ref>
