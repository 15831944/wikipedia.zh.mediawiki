'''凯尔特女人'''（{{lang|en|Celtic Woman}}），大陸譯作「凯尔特-{女人}-」、臺灣及香港作「-{天使女伶}-」，是一个由四名[[爱尔兰|爱尔兰]]女艺术家组成的乐团。该乐团的音乐类型覆盖了传统的爱尔兰音乐和现代歌曲。迄今为止，已经出版了五张唱片:《{{lang|en|Celtic Woman}}》同名专辑、《圣诞祝福》（{{lang|en|''A Christmas Celebration''}}）和《新的旅行》（{{lang|en|''A New Journey''}}）、《最好的旅行》（{{lang|en|''The Greatest Journey''}}）、《心灵之歌》（{{lang|en|''Songs from the Heart''}}）。

==成员==

过去的成员包括[[海莉·薇思特拉|海莉·薇思特拉]]（{{lang|en|Hayley Westenra}}）、[[克洛伊·艾諾|克洛伊·艾諾]]（{{lang|en|Chloë Agnew}}）、[[莉莎·凱莉|莉莎·凱莉]]（{{lang|en|Lisa Kelly}}）、[[阿莱克丝·夏普|阿莱克丝·夏普]]（{{lang|en|Alex Sharpe}}）和小提琴手[[瑪麗·内斯比|瑪麗·内斯比]]（{{lang|en|Máiréad Nesbitt}}）

== 成员 ==
凯尔特女人最初的成员为克洛伊·艾諾、歐拉·霍倫、莉莎·凱莉、琳·希拉里、美芙·威尔卡哈及瑪麗·塞斯比。2005年美芙·威尔卡哈怀孕期间，[[迪尔德丽·香农|迪尔德丽·香农]]被选中代替她进行巡演。香农于2006年2月离开，美芙在录制专辑《新的旅行》期间重新加入，并参与该专辑巡演。

第二次成员调整于2006年9月6日公布，其中[[海莉·薇思特拉|海莉·薇思特拉]]于8月24日正式加入该团体。<ref name="CW211">{{cite web |url=http://www.celticwoman.com/viewSection.asp?idSection=211 |title=September 06 |work=CelticWoman.com |date=2006-09-06 |accessdate=2007-02-13 |archiveurl=https://web.archive.org/web/20070223143839/http://www.celticwoman.com/viewSection.asp?idSection=211 |archivedate=2007-02-23 |deadurl=yes }}</ref>她参与录制专辑《新的旅行》，并在巡演中与美芙交替出现以维持5人阵容。<ref name="CW218">{{cite web |url=http://www.celticwoman.com/viewSection.asp?idSection=218 |title=Hayley and Méav dates announced |work=CelticWoman.com |accessdate=2007-02-13 |archiveurl=https://web.archive.org/web/20070223143932/http://www.celticwoman.com/viewSection.asp?idSection=218 |archivedate=2007-02-23 |deadurl=yes }}</ref>

2007年8月20日，美芙宣布永久退出该团体以专注个人演唱事业。她的接替者琳恩·希拉里，于2007年10月10日在[[美国|美国]][[佛罗里达州|佛罗里达州]][[斯特罗|斯特罗]]首次亮相。<ref name="CW226">{{cite web |url=http://www.celticwoman.com/trellis/News/ |title=Welcoming Our New Celtic Woman |work=CelticWoman.com |date=2007-08-20 |accessdate=2007-08-21 |archiveurl=https://www.webcitation.org/5TytGlb6b?url=http://www.celticwoman.com/trellis/News/ |archivedate=2007-12-10 |deadurl=yes }}</ref><ref name="meav-news1">{{cite web |url=http://www.meav.com/home.htm |title=Méav Concert Tour Kicks off in New York |work=Meav.com |accessdate=2007-12-03 |archiveurl=https://www.webcitation.org/5Tob23PmX?url=http://www.meav.com/home.htm |archivedate=2007-12-03 |deadurl=yes }}</ref>

2007年12月，凯莉宣布她将在2008年因休产假离开该团体。爱丽丝·夏普将在《新的旅行》巡演中代替凯莉<ref name="cw-sharpe1">{{cite web |url=http://www.celticwoman.com/ |title=And So... |work=CelticWoman.com |accessdate=2007-12-09 |archiveurl=http://www.webcitation.org/5Tyt4BvGl?url=http://www.celticwoman.com/ |archivedate=2007-12-10 |date=December 2007 |deadurl=yes }}</ref>。

当凯尔特女人组合被问到成员之间如何相处，莉莎·凱莉回答道：“我们相处融洽是因为我们各不相同。克萊爾·艾諾有些忧郁，美芙·妮·威尔卡哈比较理性，歐拉·霍倫如同天使一般，瑪麗·内斯比则很有活力。”<ref name="harmony">{{cite web |url=http://www.parade.com/articles/editions/2007/edition_06-03-2007/Personality_Parade |title=Walter Scott's Personality Parade, June 3, 2007 |work=Parade.com |accessdate=2007-12-18}}</ref>根据克萊爾·艾諾所说，她们之间的友谊是她们最多被问到的问题。“我认为人们总在寻找《[[绝望主妇|绝望主妇]]》式的故事，女人之间互相憎恨，难以相处，但那只是电视节目。事实上我们之间相处得很好，是亲如姐妹的好朋友。”<ref name="harmoniously">{{cite web |url=http://www.postcrescent.com/apps/pbcs.dll/article?AID=/20080402/APC05/80402152 |title=Celtic Woman members get along harmoniously |work=Post Crescent |accessdate=2008-04-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20140821205951/http://www.postcrescent.com/apps/pbcs.dll/article?AID=%2F20080402%2FAPC05%2F80402152 |archivedate=2014-08-21 }}</ref>
