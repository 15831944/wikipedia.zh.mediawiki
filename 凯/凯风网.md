'''凯风网'''是一家由[[中国反邪教协会|中国反邪教协会]]建立的网站。其網站在首頁標示其創立宗旨為「以传播[[中华传统文化|中华传统文化]]，提倡[[精神健康|精神健康]]，揭露邪教危害为特色的综合性网站」。
==背景==
其网站声称自己是批判[[法轮功|法轮功]]的民间网站，但有中国地方政府文件称該网由[[610办公室|中共610办公室]]创办<ref name="泸县">{{cite web|url = http://www.2250s.com/file.php/4/5542/2014_m.pdf |title = 方舟子2014年十大要闻|page=30}}</ref>。[[法轮功|法轮功]]方面媒体称其主要的负责人为[[何祚庥|何祚庥]]<ref>{{Cite web | url=http://www.renminbao.com/rmb/articles/2007/6/7/44535.html | title=胡出访 江夹缝里求生存（多图）| date=2007-06-07 | publisher=人民报}}</ref>。

其中，四川泸县人民政府公众信息网一份政府文件显示，[[610办公室|610办公室]]以民间方式开办《凯风网》批[[法轮功|法轮功]]，省、市、县将为《凯风网》撰稿列入目标考核<ref name="泸县" />。而山东滕州[[政法委|政法委]]网站曾宣布，被凯风网采用刊登的宣传稿件“字数（按版面计数）超过500字的，每篇奖励500元人民币；不足500字的，按400元奖励。”<ref name="zzszfw">{{cite web |url=http://www.2250s.com/file.php/4/5542/2014_m.pdf |page=30 |title=滕州政法委加强对政法调研信息宣传工作的奖励 |archiveurl=https://www.webcitation.org/69YmJKijZ?url=http://www.zzszfw.com/shownews.asp?newsid=993 |archivedate=2012-07-31 |deadurl=yes }}</ref>。
