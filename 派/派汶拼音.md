'''派汶拼音'''（泰語：ไพบูลย์；拼音：Pai-buun），是[[派汶出版社|派汶出版社]]制定的[[泰語拼音|泰語拼音]]系統，由於[[皇家泰語轉寫通用系統|皇家泰語音譯通用系統]]刪除了泰語中不符合歐洲語言的音韻特徵（如音調和部分送氣信息），泰英詞典中常常采用這套系統，如英文維基詞典<ref>[[:en:wikt:Wiktionary:Thai_romanization|官方文檔]]</ref>、《Three-Way Thai–English, English–Thai Compact Dictionary》<ref>[http://www.paiboonpublishing.com/ 派汶出版社]</ref>等等。這裏介紹的是2009年2月啓用的“派汶＋”系統之第二版。

==特點==
*無損地記錄泰語的完整音韻信息。
*特例字依照實際發音拼寫，而非照字母轉寫。
*由於泰語的古全濁聲母均已淸化爲吐氣音，派汶拼音使用“g”（j）、“k”（ch）表示全淸與次淸。逢該組聲母存在喉化音化濁的情況的，化濁的記“b”（d），不化濁的記“bp”（dt）。

# 該字母現代泰語中已棄用
# 零聲母
# 雙元音之尾部
# “ร”作爲作爲附標標在無元音符號的音節上作“'''ɔɔn'''”轉寫；特殊用法“◌รร◌”中轉寫作“'''◌a'''”；特殊用法“◌รร”中轉寫作“'''◌an'''”；特殊用法“◌รรม”中轉寫作“'''◌am'''”
# 梵語舌尖元音ฤ = รึ (rʉ); ฤๅ = รือ: (rʉʉ); 舌邊元音ฦ = ลึ (lʉ); ฦๅ = ลือ: (lʉʉ)。但他們的音在具體詞匯中可能有變化。
# 在來自歐洲語言的借詞中韻尾也寫“l”。
# 括號中的韻尾在實際泰文中不會出現。

==雜項==
* 音節閒有隔音符號，永不省略。“-”前的音節重讀；“~”前的音節輕讀。
* 未定序的其他符號：{{l|th|ก็}}、{{l|th|ก์}}、{{l|th|ฯ}}、{{l|th|ๆ}}
* 外來或古代符號：{{l|th|อํ}}（單獨）、{{l|th|อฺ}}、{{l|th|อ๎}}
