《'''穆桂英掛帥'''》是1959年，京劇大師[[梅蘭芳|梅蘭芳]]晚年創作的最後一部[[梅派|梅派]]經典作品<ref>{{Cite web |url = https://weihaimin.com/mu-guiying/  |title = 《穆桂英掛帥》穆桂英-梅派經典 |author =  |date =  |publisher =  |language =  |accessdate = 2019-04-01 |quote = }}</ref>，他曾在《我怎樣創作穆桂英掛帥》一文中說明，《穆桂英掛帥》原是[[豫劇|豫劇]]經典，在改編為京劇的過程，考量劇情氣氛的飽滿度，故加入一段「穆桂英初始因宋朝王室刻薄寡恩不願出戰，經由佘太君說服後，展現忠君愛國情操」情節，為角色建構出更完整的情緒轉折。<ref>[http://www.sohu.com/a/238555577_680915 梅兰芳：我怎样创作《穆桂英挂帅》 ]</ref>


==故事大綱==
本劇來自民間故事改編，講述退隱多年[[穆桂英|穆桂英]]遇上了西夏來犯[[宋朝|宋朝]]邊疆的時局。宋王透過比武方式，希望能夠甄選合適的將領率兵防禦。楊家子弟[[楊金花|楊金花]]及[[楊文廣|楊文廣]]在這場比武活動得勝，而宋王擔心兩姐弟太過年輕，因此命同為楊家女將的穆桂英掛帥。由於穆桂英想起宋朝廷曾迫害忠良，因此不願出征，經過長輩[[佘太君|佘太君]]勸說後決定慷慨誓師出征。

==穆桂英生平與愛情==
穆桂英是出生於[[穆柯寨|穆柯寨]]，從小在綠林好漢雲集的山寨成長，精通武藝，是知名女俠。在《[[楊家將|楊家將]]》演義中，[[楊宗保|楊宗保]]為破解遼國設下的天門陣，需要穆柯寨的降龍木，因此率軍至穆柯寨與穆桂英過招。穆桂英以過人機智及武藝精湛，生擒楊宗保。
在鬥智鬥武的過程中，穆桂英與楊宗保產生情愫，楊宗保一方面為求降龍木破天門陣，一方面也希望能夠與棋逢敵手的穆桂英締結連理，一同為宋朝廷盡忠，因此答應穆桂英的求婚。驍勇善戰的穆桂英嫁給楊家優秀的子孫-楊宗保成了一段佳話，也讓穆桂英投順宋朝成為楊門女將中重要一員。<ref>[http://www.epochtimes.com/b5/15/8/5/n4496297.htm 【文史】楊門女將穆桂英傳奇]</ref>
