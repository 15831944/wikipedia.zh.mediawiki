'''穆拉亚清真寺'''（{{lang-ar|المسجد المعلق}} ''Masjid Al-Muallaq'', {{lang-he|מסגד אל-מועלק}} ''Misgad Al-Muallak''）又名"查希尔·欧麦尔清真寺"({{lang-ar|مسجد ظاهر العمر}}) 是[[以色列|以色列]][[阿卡|阿卡]]的一个[[清真寺|清真寺]]。

==历史==
该清真寺由阿卡的[[阿拉伯|阿拉伯]]统治者查希尔·欧麦尔建于1758年。其所在庭院原为[[十字军|十字军]]建筑的遗址，后来曾是该城的[[热那亚|热那亚]]区的大门。1746年以前，该建筑被阿卡的犹太居民用作[[犹太会堂|犹太会堂]]，<ref name="Sharon38">Sharon, 1997, p. 38.</ref> 称为Ramchal犹太会堂。<ref>{{Cite web|url=http://www.aguide2israel.com/index.php/fuseaction/destination.home/a/1664/title/Religious%20Sites |title=Acre: Religious and prayer sites |deadurl=yes |archiveurl=https://web.archive.org/web/20090609151755/http://www.aguide2israel.com/index.php/fuseaction/destination.home/a/1664/title/Religious%20Sites |archivedate=2009-06-09 }}</ref> 犹太人拥有这幢建筑，直到查希尔将它改造成清真寺，另外补偿一座位于犹太区的建筑物<ref name="Sharon38"/>。犹太会堂的残余特征包括放圣约柜的壁龛和[[希伯来语|希伯来语]]题字<ref name = "archnet">{{cite web|url=http://archnet.org/sites/3495|title=Muallaq Mosque|work= ArchNet |accessdate=2008-12-31}}</ref>。

==建筑==
清真寺坐落在阿卡旧城市场的边缘<ref name="Sharon38"/>，清真寺的入口位于原[[叫拜楼|叫拜楼]]底座下方<ref name = "archnet"/>，但叫拜楼已在1950年被市政当局以公共安全风险理由拆除<ref name = "archnet"/>。清真寺主要包括一个方形的祈祷大厅,<ref name="Sharon38"/><ref name = "archnet"/> 一个三重圆顶的门廊在祷告大厅的入口处。<ref name = "archnet"/>在祷告大厅旁边是一个小房间, 目前用作图书馆。<ref name="Sharon38"/> 一座有顶楼梯通向下面的庭院<ref name = "archnet"/>。
