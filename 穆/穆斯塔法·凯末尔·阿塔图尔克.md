'''穆斯塔法·凱末爾·阿塔图尔克'''（{{lang-tr|'''Mustafa Kemal Atatürk'''}}，{{bd|1881年|5月19日|1938年|11月10日|catIdx=K}}）是[[土耳其|土耳其]]的軍官、改革家、作家和領導人，[[土耳其共和國|土耳其共和國]]第一任[[土耳其總統|總統]]、[[土耳其總理|總理]]及[[土耳其大國民議會|國民議會]]議長<ref name=yatsen>

凱末爾最為人所知是他在[[第一次世界大战|一戰]]傑出的軍事能力，同時伴隨著[[奧斯曼帝國|奧斯曼帝國]]的崩解，他帶領[[土耳其國民運動|土耳其國民運動]]並成功在[[安卡拉|安卡拉]]建立獨立政府，他打敗當時[[協約國|協約國]]所帶領的同盟軍隊，並以傑出的優越能力解放了國家並建立了現今的土耳其共和國，在他就任土耳其領導人的時候，凱末爾進行一連串政治，經濟和文化上的變革，啟蒙土耳其並讓土耳其成為現代化和[[世俗主義|世俗主義]]的國家，凱末爾帶領土耳其現代化的作為被稱為[[凱末爾主義|凱末爾主義]]，其中也包括減少[[伊斯蘭教|伊斯蘭教]]對政治及教育的影響<ref>{{cite book |author=孙振玉 |title=传统与现实：土耳其的伊斯兰教与穆斯林 |year=2001 |publisher=民族出版社 |isbn=9787105046171 |url=http://wapread.189.cn/goChapterContent.action?bookId=100000215337455&chapterId=4&curpage=5&surplus=1}}</ref>。

== 早年 ==
1881年3月12日出生于奥斯曼帝国的萨罗尼卡，父亲名为[[阿里·里薩埃芬迪|阿里·里薩埃芬迪]]。1887年其先后就读于[[法特玛太太区立小学|法特玛太太区立小学]]和[[沙姆西先生小学|沙姆西先生小学]]。1893年转入[[萨洛尼卡幼年学校|萨洛尼卡幼年学校]]，学习成绩优异。因与一位老师同名，被该老师命名为'''凯末尔·穆斯塔法'''。1895年升入[[玛纳斯提军事预备学校|玛纳斯提军事预备学校]]。1899年进入[[伊斯坦布尔军官学校|伊斯坦布尔军官学校]]，1902年进入[[哈拜参谋学院|哈拜参谋学院]]，因[[数学|数学]]成绩优秀，获得“凯末尔”之称。1905年毕业，授上尉军衔。

== 军事生涯 ==
[[File:Mustafa_Kemal_during_the_Gallipoli_Campaign.jpg|thumb]]
凱末爾因参与政治活动被逮捕，后被苏丹[[阿卜杜勒·哈米德二世|阿卜杜勒·哈米德二世]]放逐到[[大马士革|大马士革]]第五军营服役。1911年，凯末尔因批评当局而被调离伊斯坦布尔，不久参加了土耳其与[[意大利|意大利]]在[[利比亚|利比亚]]进行的战争，晋升为少校。1912年10月—1913年5月，调任驻[[保加利亚|保加利亚]]大使馆武官，晋升为中校。[[第一次世界大战|第一次世界大战]]爆发以后，被任命为新编第19师师长。1915年晋升为上校，指挥了著名的[[加里波利之战|加里波利之战]]，击退了[[英国|英国]]、[[澳大利亚|澳大利亚]]、[[新西兰|新西兰]]、[[法国|法国]]、[[印度|印度]]和[[紐芬蘭|紐芬蘭]]军队的登陆进攻。1916年8月，因保卫[[奥斯曼帝国|奥斯曼帝国]]首都伊斯坦布尔，获得“伊斯坦布尔的救星”和“帕夏”的称号，名扬全国和欧洲。后被任命为第二军团司令，晋升为将军。<ref name=yuan2013>{{cite journal|author=袁灿兴|title=凯末尔与土耳其的现代化之路|journal=《文史天地》|year=2013|issue=02|url=http://www.wstdw.com/html/87/3/3291/1.htm|accessdate=2013-11-20|deadurl=yes|archiveurl=https://web.archive.org/web/20131210131009/http://www.wstdw.com/html/87/3/3291/1.htm|archivedate=2013-12-10}}</ref>

== 政治生涯 ==
[[File:Mustafa_Kemal_Atatürk_(1918).jpg|thumb]]   
1919年，他受命為駐紮在[[薩姆松|薩姆松]]的第9軍督察使（Inspector general），負責殘存的土耳其軍隊的復員和維持治安的工作。他利用這一身份，著手聯合各民族主義抵抗力量，組建[[土耳其革命者|國民軍]]，7月在埃爾祖魯姆舉行的東部各省護權協會代表大會，凱末爾被選為主席。

1920年[[英國|英國]]重新佔領[[伊斯坦布爾|伊斯坦布爾]]，逮捕議員並強迫蘇丹解散國民議會，凱末爾在[[安卡拉|安卡拉]]召開[[大國民議會|大國民議會]]，宣佈組成代表國家的臨時政府，並擔任新成立的大國民議會主席。伊斯坦布爾的蘇丹政府簽訂了[[塞夫爾條約|塞夫爾條約]]之後，凱末爾領導的安卡拉政府堅決拒絕這一條約。1921年8月他要求並說服了大國民議會選舉他為國民軍全權總司令，親自指揮國民軍與[[希臘|希臘]]軍隊在離安卡拉80公里外的[[薩卡里亞河|薩卡里亞河]]進行戰略決戰，戰役中他被流彈打斷一根肋骨，但仍然紮著繃帶指揮，最後以弱勝強，重挫希軍主力，取得了決定性的勝利。9月被大國民議會授予「加齊」（「勝利者」）稱號，晉陞為元帥。

1922年11月1日，大國民議會通過了廢除蘇丹制的決議案。1923年10月29日，為了消除土耳其政制的曖昧狀態，凱末爾向新選出的大國民議會提出了一項憲法修正案，正式成立[[土耳其共和國|土耳其共和國]]，凱末爾隨後當選為共和國第一任總統。

1923年9月凱末爾在安納托利亞與羅麥里亞護權協會基礎上建立[[共和人民黨_(土耳其)|人民黨]]，1924年4月，大國民議會通過了新憲法。11月人民黨改名為[[共和人民黨_(土耳其)|共和人民黨]]，由總統兼任[[政黨領袖|主席]]，總理為副主席。自1923年直至1938年凱末爾去世，凱末爾一直以黨領政、連選連任共和國總統、黨主席和武裝部隊總司令，其權力幾乎不受任何限制。凱末爾逝世十五年後遺體入葬落成於1953年的[[安卡拉|安卡拉]]{{link-en|凱末爾紀念館|Anitkabir}}。

== 世俗化 ==
{{main|凱末爾改革}}
1924年3月，凯末尔废除了源自伊斯蘭教先知[[穆罕默德|穆罕默德]]後人的[[哈里發|哈里發]]制度，将奥斯曼王室成员全部驱逐出境，並進行政治改革。他废除了历史悠久的[[伊斯兰教长制|伊斯兰教长制]]、撤消伊斯兰教法（沙里亚）部、停办独立的宗教学校和经院、关闭宗教法庭（特别沙里亚法庭）以及废除被奉为神圣法典的沙里亚法、制订和采用依据西欧国家法律为摹本的新民法等等，从而为土耳其的[[世俗化|世俗化]]扫清了障碍。

<blockquote>“我不是精神上的，沒有教條，沒有冷靜，刻板的統治作為精神遺產。我的精神遺產是科學和思想。”</blockquote>

=== 服饰革命 ===
凯末尔利用1925年大国民议会授与政府的特别权力，以激烈的手段完成了颇具象征意义的土耳其服饰革命。他颁布命令，強制所有政府人员必须穿戴[[西装|西装]]与礼帽，同时颁布一项禁令，禁止非神职人员穿着宗教袍服或宗教徽记；11月25日，又颁布新的法律，強制禁止所有男子戴礼帽，凡戴[[土耳其帽|土耳其帽]]者将依律治罪。他并带头脱下军服，换上西服，以为国民表率。

=== 解放婦女 ===
凯末尔推動了一系列提高土耳其婦女地位的改革。包括在法律中明文強制不准婦女穿戴蓋頭面紗、廢除[[一夫多妻|一夫多妻]]、確立[[離婚|離婚]]制度、保障婦女在教育就業參政及財產繼承的平等權利。

1934年修改憲法，婦女21歲擁有選舉權，30歲則擁有被選舉權，這項舉措甚至比許多歐洲國家更早，如[[法國|法國]]和[[瑞士|瑞士]]。

1935年大選後，十八名女性國會議員加入土耳其國會。

== 文字改革 ==
[[File:Ataturk-September_20,_1928.jpg|thumb]]（1928年9月20日）]]
1928年5月24日大国民议会立法通过以国际[[拉丁字母|拉丁字母]]取代以前使用的[[阿拉伯字母|阿拉伯字母]]。8月9日在共和人民党举行的一个有党的许多重要人物参加的游园会上，凯末尔宣布实行文字改革。他号召“把这件事（指推广新字母）看成是一种爱国行为和国民义务”，要求土耳其人民把“自己从多少世纪以来像铁箍似的束缚着我们思想的那些令人无法理解的符号中解放出来”<ref>{{cite web|url=http://www.da-kiss.com/a/lishizhenxiang/20130402/3603.html|title=土耳其融入欧洲文明的极端“字母革命”|publisher=奇闻异事网|accessdate=2013-10-28|deadurl=yes|archiveurl=https://web.archive.org/web/20131029202107/http://www.da-kiss.com/a/lishizhenxiang/20130402/3603.html|archivedate=2013-10-29}}</ref>。而只有这样，土耳其民族才能“以它的文字和它的思想，表明自己在文明世界中的地位”。他前往全国各地，亲自教人们学习新字母。11月3日，大国民议会通过一项立法，以拉丁字母為基礎，确定了土耳其新字母<ref>{{cite web |url=http://www.goturkey.com.cn/i/521.html |title=土耳其語言 |publisher=土耳其旅遊局 |accessdate=2013-10-28}}</ref>，并规定翌年起不再公开使用旧的阿拉伯字母。

凯末尔聲稱，与阿拉伯字母相比，新字母清楚、简洁又适合[[土耳其语|土耳其语]]发音，在土耳其提高成人识字率及发展文教事业方面产生了显著的效果。但是，實際上新的拉丁土耳其字母，合併了幾個不同的音，k和q不分，é和e不分，h和x不分，反而產生了新的混淆。如其实行的一切变革一样，凯末尔将这一切都视为政治行动，是他改造土耳其国家和社会的一部分，其目的就是要削弱与割裂同奥斯曼帝国历史和伊斯兰文化的联系。凯末尔从另一方面否定了伊斯兰文化的本体特征，反过来强调了[[突厥|突厥]]史和前[[安纳托利亚|安纳托利亚]]历史的重要性<ref>{{cite news |author=韓宗佳 |title=土耳其能否重歸中東領導者？ |url=http://news.sina.com.tw/article/20130605/9840139.html |work=新浪新聞 |date=2013-06-05 |accessdate=2013-10-28}}</ref>。

== 遗产和影响 ==
[[File:Anitkabir.JPG|thumb]]的{{link-en|凱末爾紀念館|Anitkabir}}]]
凯末尔执政期间实施一系列社会政治与经济改革措施，并逐步形成土耳其民族的思想理论体系——[[凯末尔主义|凯末尔主义]]。

当今土耳其国内最大机场——[[伊斯坦布尔|伊斯坦布尔]]的[[阿塔蒂尔克国际机场|阿塔蒂尔克国际机场]]即以他的名字命名<ref>{{cite web|author=李显超|url=http://site.nihaowang.com/site/7821f|title=阿塔蒂尔克国际机场|publisher=你好網|accessdate=2013-10-28|deadurl=yes|archiveurl=https://web.archive.org/web/20160304122310/http://site.nihaowang.com/site/7821f|archivedate=2016-03-04}}</ref>。

== 評价 ==
凯末尔在各方面试图使土耳其走向[[西方世界|西方]]、走向现代，也确实在某种程度上实现了目标。土耳其民族通过他的努力，摆脱了内部和外来的束缚，开始走向工业化、现代化迈进<ref>{{cite news |author=何清漣 |title=民主的毒藥：軍人干政背後的威權邏輯 |url= http://www.epochtimes.com/b5/13/7/9/n3912164.htm |work=大紀元 |date=2013-07-08 |accessdate=2013-10-28}}</ref>。仅凭这一点，“现代土耳其之父”的名号是当之无愧的。

凯末尔雖然使得土耳其人從[[君主專制|君主專制]]體制走向了[[民主共和|民主共和]]，但是也走向了其個人[[強人政治|強人政治]]和[[威權主義|威權主義]]的另一個極端。他試圖擺脫了[[阿拉伯|阿拉伯]]與[[波斯|波斯]]文化和宗教治國的影響，卻一味地向[[歐美|歐美]]文化靠攏。

凯末尔認为，當時[[伊斯兰|伊斯兰]]社會有許多的生活方式，太過宗教化且不夠進步，與[[共和制度|共和制度]]与现代进步是對立的，与现代[[民主|民主]]制度是矛盾的，所以，要實行土耳其的现代化，必需倡导世俗化和全面西化的这样一个假设为前提，实行了与[[奥斯曼帝国|奥斯曼帝国]]和伊斯兰文明決裂，将[[土耳其|土耳其]]融入“唯一”的文明── “[[欧洲|欧洲]]文明”的一系列措施。凯末尔本人拥有的民族英雄光环和一支对其绝对服从的军队。在军事力量支持下，凯末尔依靠暴力和强权推行全盘西化政策，在社會各個方面，以[[西方世界|西方]]為藍本进行一系列强力的改革。

由於凯末尔的獨裁指導思想，導致了土耳其政壇在20世紀存在[[军官|军官]]的長期干政。历史上，[[土耳其軍隊|土耳其軍隊]]曾多次发动军事[[政变|政变]]强迫民选的、支持放鬆宗教管制的右翼[[凱末爾主義|凱末爾主義]]（[[民主黨|民主黨]]和[[正義黨|正義黨]]政府）下台，把國家重新維持強制世俗主義。所有[[法律|法律]]必须经过由军方及宪法法庭审查才能生效，民主變成了土耳其军人把持国家命运的美麗外衣，近年[[埃爾多安|埃爾多安]]領導的伊斯兰民主主義（[[保守民主|保守民主]]）的文人政府通過修憲限制軍方權力，而軍方對政府的影響力也開始下降。不過在做出一些自由化改革後，埃爾多安在13年廣場遊行事件後也變成了獨裁者，他開始從反對並試圖淡化凱末爾的個人崇拜影響變為大肆鼓吹這位共和國的國父宣傳[[伊斯蘭主義|伊斯蘭主義]]為表實則推崇土耳其的[[民族保守主義|民族保守主義]]來弘揚土耳其的奧斯曼精神，並在[[2016年土耳其政變|2016年土耳其政變]]後在全國各地大肆拘捕反對人士、支持恐怖主義者（即[[庫爾德工人黨|庫爾德工人黨]]）及新聞工作者。但因凱末爾世俗和[[進步主義|進步主義]]的改革，不少居住於土西和大城市的土耳其人對凱末爾的評價遠高於埃爾多安<ref>{{cite news |author=人民網 |title=土耳其告別軍人干政年代 |url=http://big5.gmw.cn/g2b/history.gmw.cn/2011-08/26/content_2537535.htm |work=光明日報 |date=2011-08-26 |accessdate=2013-10-28 |archive-url=https://web.archive.org/web/20131029212304/http://big5.gmw.cn/g2b/history.gmw.cn/2011-08/26/content_2537535.htm |archive-date=2013-10-29 |dead-url=yes }}</ref>。
