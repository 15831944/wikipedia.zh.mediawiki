'''穆罕默德·兰吉巴'''（1935年1月1日- 2004年6月29日）是一名[[伊朗|伊朗]][[足球运动员|足球运动员]]和教练。兰吉巴曾效力于[[德黑兰独立足球俱乐部|德黑兰塔吉]]于[[德黑兰帕斯足球俱乐部|德黑兰帕斯]]。他曾是[[伊朗國家男子足球隊|伊朗国家足球队]]的队长。1972年，他还担任国家队的主教练一职。

== 早年 ==
兰吉巴于1935年1月1日出生在[[伊朗|伊朗]]的[[克尔曼沙赫|克尔曼沙赫]]，直到高中毕业都住在那。之后他在1950年加入了{{le|德黑兰沙欣足球俱乐部|Shahin Tehran F.C.|科曼莎沙欣}}。在兰吉巴结束学业后，他被调到[[德黑兰|德黑兰]]，毕业于达内什卡迪·阿夫萨里。

== 俱乐部生涯 ==
他于1956年加入[[德黑兰独立足球俱乐部|塔吉足球俱乐部]]，在球队踢了八年后，于1964年转会至[[德黑兰|德黑兰]]的[[德黑兰帕斯足球俱乐部|帕斯俱乐部]]。他从1965年开始到1970年担任球队队长，之后他在该俱乐部退役。

== 国家队生涯 ==
他于1959年首度入选[[伊朗國家男子足球隊|伊朗國家队]]。并参加了[[1960年亞足聯亞洲盃|1960年亚洲杯]]预选赛，但球队未能晋级决赛圈。1965年，在前任伊朗国家队的队长{{le|帕尔维兹·德哈达利|Parviz Dehdari}}退役后，兰吉巴接过了队长袖标，直到两年后在对阵[[土耳其國家足球隊|土耳其]]的比赛后宣布退役。<ref >{{Cite web|url=http://www.rsssf.com/miscellaneous/iran-coach.html|title=Player: Mohammad Ranjbar|publisher=rsssf.com}}</ref>

== 主教练生涯 ==
退役后，兰吉巴于1968年开始担任{{le|伊朗20岁以下国家足球队|Iran national under-23 football team|伊朗U-20}}的教练。1970年，他被选为[[伊朗国家足球队|伊朗]]助理教练，在1972年{{le|帕尔维兹·德哈达利|Parviz Dehdari}}辞职后，兰杰巴被任命为国家队的临时主教练。由于在[[1972年亞足聯亞洲盃|1972年亚洲杯]]率队取得了冠军，<ref >{{Cite web|url=http://www.rasekhoon.net/mashahir/show-905732.aspx|title=Ranjbar, Mohammad|publisher=rasekhoon.net}}</ref>伊朗足协正式任命兰吉巴为球队的主教练，并与他签订了两年的合同，但他在两个月后便辞职。从1972年到1985年他都没有再次执教球队。1985年，他就任[[德黑兰独立足球俱乐部|德黑兰独立足球俱乐部]]的技术总监。之后兰吉巴在1998年至2002年曾担任国家队的领队。

== 疾病去世 ==
2004年1月12日，69岁的兰吉巴身患致命疾病，几个月后被送往[[德国|德国]][[汉诺威|汉诺威]]进行手术，但德国医生由于病情危急，不愿动手术。最终他于2004年6月29日因大脑紊乱死于[[德黑兰|德黑兰]]以北的Iranmehr医院。<ref >{{Cite web|url=http://www.payvand.com/news/04/jun/1190.html|title=Iran's football manager Mohammad Ranjbar dies of cerebral disorder|publisher=payvand.com}}</ref>
