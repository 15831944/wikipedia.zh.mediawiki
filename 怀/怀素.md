'''怀素'''（{{bd|725年||785年|catIdx=H怀素}}）， 字'''藏真'''，[[长沙|长沙]]人（另一说[[零陵|零陵]]人），俗姓[[钱姓|钱]]，[[唐朝|唐朝]]僧人、书法家。精勤学书，以善[[狂草|狂草]]出名。唐朝人稱他為'''醉僧'''。

== 生平 ==
懷素的伯祖是[[惠融禪師|惠融禪師]]<ref>[[陸羽|陸羽]]，僧懷素傳：懷素伯祖，惠融禪師者也</ref>。

怀素幼年出家，最初修习[[佛经|佛经]]、[[历律|历律]]书，后来留意于[[书法|书法]]。

因貧而無紙可書，就在家鄉故里種植芭蕉萬餘株以供揮灑，以蕉叶代纸练字，所以把他居住的地方叫“绿天庵”。書不足，乃漆一盤書之，又漆一方板，書至再三，盤板皆穿。

== 風格 ==
怀素早年书法作品，先學[[歐陽詢|歐陽詢]]書，世莫能辯，至是鄉中呼為大錢師小錢。之後的笔势纵横驰突，体态健缓跌宕，因以婆娑烂漫形象，别开生面，情韵不减而境界别出。

懷素的性情踈放，不拘於細行，萬緣皆繆，心自得之。於是飲酒以養性，書[[草書|草書]]以暢志，每当酒酣興發，一遇到寺壁里墙、衣裳器皿，无所不书，唐朝人称为“醉僧”。

== 评价 ==
前人评其狂草继承[[张旭|张旭]]，而有所发展，谓“以狂继颠”，并称“颠张醉素”，对后世影响很大。

[[唐朝|唐朝]]诗仙[[李白|李白]]：“少年上人号怀素，[[草书|草书]]天下称独步，墨池飞出北溟鱼，笔锋杀尽中山兔。……吾师醉后倚绳床，须臾扫尽数千张。飘风骤雨惊飒飒，落花飞雪何茫茫……如闻神鬼惊，时时只见龙蛇走。左盘右蹙如惊电，状如楚汉相攻战……”<ref>{{cite book |language =zh-hans|author=李白|title=《李太白全集》|year=2011-03-01|publisher=中华书局|location=北京|pages=第8卷|isbn=9787101077780 }}</ref>
