'''怀柔关公庙'''，位于[[北京市|北京市]][[怀柔区|怀柔区]][[雁栖镇|雁栖镇]]范各庄村，是[[道教|道教]]宫观。

==简介==
关公庙始建于[[明朝|明朝]]初期，明末清初达到鼎盛。后来毁于一场大火。2014年，范各庄村实施燕城古街步行街建设，在原址复建关公庙。2017年7月15日（农历丁酉年六月二十二日），关公庙宗教活动场所颁证仪式暨[[关帝|关帝]]圣诞开光盛典举行，[[中国道教协会|中国道教协会]]副会长、[[北京市道教协会|北京市道教协会]]会长[[黄信阳|黄信阳]]，中国道教协会副秘书长[[周高德|周高德]]，北京市宗教局副局长[[刘先传|刘先传]]，怀柔区副区长[[于海波|于海波]]，北京市道教协会副会长[[刘崇尧|刘崇尧]]、[[孔理元|孔理元]]、秘书长姜涛等出席。刘先传颁发宗教活动场所牌，于海波颁发宗教活动场所证书，四位书画家赠送墨宝。关公庙住持黄章杰致答谢词<ref>{{Cite web |url = http://www.daoisms.org/article/sort028/info-31099.html?1500164971 |title =  北京怀柔关公庙颁发活动场所证暨圣像开光庆典举行|author =  |date = 2017-07-15 |publisher =道教之音  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.china.com.cn/guoqing/2017-07/15/content_41221057.htm  |title =关公庙宗教活动场所颁证仪式暨关帝圣诞神像开光盛典今日举行  |author =  |date =2017-07-15  |publisher = 中国网 |language =  |accessdate =  |quote =  }}</ref>。2017年9月7日，[[中共北京市委|中共北京市委]]统战部部长[[齐静|齐静]]，在中共怀柔区委书记[[常卫|常卫]]、区委副书记[[姜泽廷|姜泽廷]]、区委统战部部长迟行刚、区常委任武军、雁栖镇党委书记周福刚、范各庄村党支部书记白玉亭等陪同下参观考察了关公庙<ref>{{Cite web |url = http://www.daoisms.org/article/sort028/info-32155.html?from=groupmessage |title = 北京市委统战部部长齐静参观考察怀柔关公庙 |author =  |date = 2017-09-07 |publisher =  道教之音|language =  |accessdate =  |quote =  }}</ref>。

此后关公庙举办了许多文艺活动。2017年10月4日（农历八月十五日）晚，关公庙举办[[中秋|中秋]]夜“恭祝[[月老|月老]]圣诞《道·民俗》”李建科原创歌曲演唱会，此次演唱会由关公庙住持黄章杰特邀音乐人李建科演出<ref>{{Cite web |url = https://www.sohu.com/a/196473081_814666 |title = 北京怀柔关公庙举办“恭祝月老圣诞《道·民俗》”李建科原创歌曲演唱会 |author =  |date = 2017-10-05  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。2018年7月11日，关公庙举办首届“翰墨怀柔 醉美燕城”书画艺术节开幕式，该活动为期5天<ref>{{Cite web |url =http://www.bjethnic.gov.cn/xinwenzhongxin/gzdt/20180718/6078.shtml  |title =怀柔区雁栖关公庙首届“翰墨怀柔 醉美燕城”书画艺术节开幕  |author =  |date =  2008-07-18|publisher =北京市民族事务委员会  |language =  |accessdate =  |quote =  }}</ref>。
