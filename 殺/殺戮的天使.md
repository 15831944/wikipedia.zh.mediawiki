《'''殺戮的天使'''》（{{lang-ja|殺戮の天使}}）是一款由星屑KRNKRN（真田まこと）製作、[[AGM_PLAYISM|AGM PLAYISM]]及Mugen Creations發行的解謎類免費遊戲。遊戲對應[[Microsoft_Windows|Microsoft Windows]]平台，於2015年8月14日發行。

漫畫由{{lang|ja|名束くだん}}負責作畫，於[[月刊Comic_Gene|月刊Comic Gene]]上連載。

2017年7月，KADOKAWA於本作官方Twitter上宣佈動畫化的消息，並將於2018年7月6日播放<ref>{{Cite web |url = https://www.inside-games.jp/article/2017/07/25/108744.html |title = フリーゲーム『殺戮の天使』がTVアニメに！ 記憶をなくした少女と殺人鬼の物語をアニメで描く |author = INSIDE |date = 2017-07-25 |language = ja |accessdate = 2018-03-27}}</ref><ref>{{Cite web |url = http://jp-anime-trend.com/3655 |title = 免費遊戲『殺戮的天使』TV動畫化決定！ |author = 日本動漫趨勢報 |date = 2017-07-25 |accessdate = 2018-03-27}}</ref>。

本作的Nintendo Switch版預訂由[[Active_Gaming_Media|Active Gaming Media]]營運的[[Playism|Playism]]發售<ref>{{cite web|url=http://www.4gamer.net/games/135/G013541/20180511035/|title=Nintendo Switch版の発売やTVアニメ化も決定。若者層を中心に大きな人気を獲得している「殺戮の天使」の仕掛け人，バカー斉藤大地氏ミニインタビュー|date=2018-05-11|accessdate=2018-06-02}}</ref>。
Nintendo Switch版跟免費版無太大差異，預訂收錄手機版的特典圖像。

== 故事簡介 ==
13歲的瑞吉兒·加德納醒來後發現自己被困在一棟廢棄建築物的地下室。她失去了記憶，甚至不明白自己為什麼在這種地方。

在建築物內徘徊並尋找出口時，她遇到了一個用繃帶覆蓋臉部並手持鐮刀的殺人魔——艾札克·佛斯特。

在兩人向對方作出承諾後，她開始在這棟建築物內發現更多關於自己的事情。<ref>{{Cite web |url = https://www.animatetimes.com/news/details.php?id=1500885542 |title = 「電ファミニコゲームマガジン」の人気自作ゲーム『殺戮の天使』のTVアニメ化が決定！　7月27日にはコミカライズ最新刊が登場 |author = animate Times |date = 2017-07-24 |language = ja |accessdate = 2018-03-27}}</ref>

== 登場人物 ==
; {{nihongo|瑞吉兒·加德納/瑞依|レイチェル・ガードナー/レイ|Rachel·Gardner/Ray|聲：[[千菅春香|千菅春香]]}}
: 本作女主角；Floor B1的層主。
: 13歲，金髮藍眼的無口少女，脖子上戴着黑色項圈，身穿黑白條紋上衣和白色外套及黑色短褲。黑色斜肩包裡放著裁縫用具，以便隨時可以拿出針線來進行縫紉。
: 時常面無表情，聰明且冷靜沉著。擅長縫紉。認為自己是不被允許活著的。因为圣经说不能自杀，所以求札克殺掉她，對信仰相當執著。

; {{nihongo|艾札克·佛斯特/札克|アイザック・フォスター/ザック|Isaac·Foster/Zack|聲：[[岡本信彦|岡本信彦]]}}
: 本作男主角；Floor B6的層主。
: 約20歲，異色瞳（左眼為黃色，右眼為黑色），黑髮，手持巨型鐮刀的青年。小時候因為被燒傷所以上半身包著繃帶，身穿灰色連帽衫及紅色長褲。
: 只要發現看起來很幸福的人就會想要殺掉對方。頭腦不太好，個性極其衝動，經常用暴力來解決問題，對人的感情和心境的變化相當敏銳。不識字。非常怕火。不會說謊也討厭別人對自己說謊。在大樓中和瑞吉兒定下誓約，答應瑞吉兒只要完成誓言就會殺了她。自稱「正常的成年男性」。經常被人稱作「怪物」或「工具」。稱瑞吉兒為「瑞依」。

; {{nihongo|丹尼爾·狄更斯/丹尼|ダニエル・ディッケンズ/ダニー|Daniel·Dickens/Danny|聲：[[櫻井孝宏|櫻井孝宏]]}}
: Floor B5的層主。
: 身穿醫生的白色長袍，戴著眼鏡。茶色頭髮。左眼視力不太好，右眼為義眼（平常是黑色），現出本性時會戴上像亞歷山大石一般有紅綠兩色瞳孔的義眼。
: 職業為心理醫生。外表非常溫和。對於眼球異常執著，最喜歡與母親相似的藍色眼球；對眼球以外的事物都毫無興趣，而對於眼球沒受他中意的人則不會表示任何關心。

; {{nihongo|愛德華·梅森/艾迪|エドワード・メイソン/エデイ|Edward·Mason/Eddie|聲：[[藤原夏海|藤原夏海]]}}
: Floor B4的層主。
: 頭戴麻袋頭套，身穿粗棉布製短袖T恤和藍色背帶褲，脖子上圍著紅色圍巾。眼睛為綠色。隨身帶著鐵鏟。年齡與瑞吉兒相近。
: 擅長製作墳墓。做事熱心認真。對喜歡的事物會露出天真無邪的表情。常會淘氣的跳來跳去。對於自己尊敬的人會很尊敬，面對自己不喜歡的人會表現的非常臭屁。

; {{nihongo|凱薩琳·華德/凱西|キャサリン・ワード/キャシー|Catherine·Ward/Cathy|聲：[[伊瀬茉莉也|伊瀬茉莉也]]}}
: Floor B3的層主。
: 身穿黑色上衣和緊身裙，搭配黑色長靴，紅色手套和領帶，頭戴黑色看守帽。眼睛為綠色。豐滿的身材和美豔的容貌。頭髮為金色和粉紅色的漸變。手持紅色鞭子（機關遙控器）。
: 自稱為「處刑人」（日語寫作『斷罪人』）。興奮時會發出尖銳的笑聲。只要看到被自己判為罪犯的人受盡痛苦折磨就會很開心。雖然說話方式十分爽朗又俏皮，但是會說出看不起對方的話。常被札克叫做抖S女或SM女。

; {{nihongo|亞伯拉罕·葛雷/格雷|エイブラハム・グレイ|Abraham·Gray|聲：[[大塚芳忠|大塚芳忠]]}}
: Floor B2的層主。
: 梳著油頭的中年男性。眼睛沒有瞳孔。脖子上戴著十字架項鏈，身穿如神父一般的紫色長袍。
: 被人稱作「神父大人」。表情嚴肅，經常會為了試探對方而做出提問。會用神秘的紫色煙霧使人陷入幻覺中。大樓的創建者。將自己定義為大樓內的「神」，並將大樓裡的殺人魔稱為「天使」。

== 迴響 ==
除了前作《[[霧雨飄零之森|霧雨飄零之森]]》的粉絲，本作憑藉著其獨特設定又獲得新粉絲，此外，更透過遊戲實況，將影響力擴及到未持有電腦的客群<ref name="4GamerInterview"/>。
電玩誌（ゲームマガジン）的營運商βακα公司代表齊藤大地，在4Gamer.net的訪談中表示，殺戮的天使以十幾歲的女性粉絲為主要人氣客群<ref name="4GamerInterview">{{cite web|url=http://www.4gamer.net/games/419/G041905/20180514056/|title=Nintendo Switch版の発売やTVアニメ化も決定。若者層を中心に大きな人気を獲得している「殺戮の天使」の仕掛け人，バカー斉藤大地氏ミニインタビュー|date=2018-05-14|accessdate=2018-06-02}}</ref>。
