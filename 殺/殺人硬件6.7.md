== 反響 ==
本片得到大部分負面評價。[[爛番茄|爛番茄]]上基於30條評論，本片獲得分數為33％，32％的觀眾表示喜歡本片。<ref>{{Cite web |url = https://www.rottentomatoes.com/m/virtuosity/ |title = VIRTUOSITY (1995) |publisher = Rotten Tomatoes |language = en}}</ref>[[Metacritic|Metacritic]]上基於17條評論，得分為百分之39。<ref>{{Cite web |url = http://www.metacritic.com/movie/virtuosity |title = Virtuosity 1995 |publisher = Metacritic |language = en}}</ref>

本片在[[錫切斯電影節|錫切斯電影節]]上被提名為最佳電影，但輸給了《[[公民X|公民X]]》。<ref>{{Cite web |url = http://sitgesfilmfestival.com/eng/arxiu/1995/programacio#01 |title = 28ed. Festival Internaciona de Cinema Fantàstic de Sitges (7/10 - 14/10) |publisher = Sitges Film Festival}}</ref>
