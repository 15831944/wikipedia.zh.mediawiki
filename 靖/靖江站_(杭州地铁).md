'''靖江站'''位于[[杭州|杭州]][[萧山区|萧山区]]青六南路与永盛路交叉路口<ref>{{cite web |title=最新！杭城3条地铁50个站点位置公布！独家整理沿线53个楼盘 你家门口有吗 - 杭州新闻中心 - 杭州网 |url=http://hznews.hangzhou.com.cn/chengshi/content/2017-12/16/content_6743453_3.htm?Token= |website=杭州网}}</ref>，为杭州地铁[[杭州地铁7号线|7号线]]及[[杭州地铁中轴快线|中轴快线]]<ref>{{cite web |title=杭州地铁三期新增机场轨道快线、3号线北延、5号线西延-浙江新闻-浙江在线 |url=http://zjnews.zjol.com.cn/zjnews/hznews/201803/t20180306_6735701.shtml |website=浙江在线}}</ref>的地下同台换乘站，预计2021年开放。

==车站结构==
靖江站为地下二层双岛四线[[同台换乘站|同台换乘站]]，长约587米，主体标准段宽40.7米，开挖深约16.64米，车站设置4组风亭和4个出入口，12个消防出口，以及3个物流出入口。<ref>{{cite web |title=中铁一局五公司杭州地铁7号线靖江站首幅地连墙开始施工 |url=http://www.crfeb.com.cn/1j/_47/1797187/index.html |website=中铁一局集团有限公司}}</ref>
