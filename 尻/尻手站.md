'''尻手站'''（{{jpn|j=尻手駅|hg=しってえき|rm=Shitte eki|f=y}}）是一個位於[[神奈川縣|神奈川縣]][[川崎市|川崎市]][[幸區|幸區]]南幸町三丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[南武線|南武線]]的[[鐵路車站|鐵路車站]]。

除了南武線的本線外，[[濱川崎站|濱川崎站]]方向的支線（濱川崎支線、旅客導覽內為「南武支線」）以及往{{link-ja|新鶴見信號場|新鶴見信号場}}的支線（尻手短絡線、[[貨物列車|貨物列車]]專用）在此站起分岔。

此站位於[[川崎市|川崎市內]]，但在車費區分上視作[[特定都區市內#設定區域一覽|橫濱市內]]的車站處理。

== 历史 ==
* [[1927年|1927年]]（[[昭和|昭和]]2年）[[3月9日|3月9日]]：伴随南武鐵道線[[川崎站|川崎站]] - [[登户站|登户站]]間开通运营以尻手[[停车场_(铁路)|停留場]]的形式开业。
* [[1929年|1929年]]（昭和4年）：升格为车站。
* [[1930年|1930年]]（昭和5年）
** [[3月25日|3月25日]]：貨物支線（本站-[[滨川崎站|滨川崎站]]）开始运营。
** [[4月10日|4月10日]]：本站 - [[滨川崎站|新滨川崎站]]之间开始办理客运业务。
* [[1944年|1944年]]（昭和19年）[[4月1日|4月1日]]：南武鐵道線国有化，本站成为[[日本國有鐵道|國鐵]]南武線的车站。
* [[1973年|1973年]]（昭和48年）[[10月1日|10月1日]]：貨物支線[[滨川崎站|滨川崎站]]-[[新鶴見信號場|新鶴見信號場]]（現在的新鶴見信号場）间开始运营。
* [[1987年|1987年]]（昭和62年）4月1日：[[国铁分割民营化|国铁分割民营化]]，成为東日本旅客鐵道的车站。
* [[2001年|2001年]]（[[平成|平成]]13年）[[11月18日|11月18日]]：IC卡[[Suica|Suica]]启用。

== 车站结构 ==
本站为2面3线的高架車站。由侧式月台1面1線和岛式月台1面2線组成。

== 使用人數 ==
