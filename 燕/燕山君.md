'''燕山君'''{{efn|「[[燕|燕]]」音：Yān／ㄧㄢ這裡的「燕」作地方名解，讀作「煙」，不讀「燕子」的「燕」。}}（{{kor|k=연산군|rm=Yeonsan gun|hanja=燕山君|f=h}}；{{bd|1476年||1506年|catIdx=L李}}），姓'''李'''，名'''{{僻字|㦕|左「忄」右「隆」}}'''（{{kor|k=이융|rm=Yi Yung|hanja=李㦕}}），兒名'''無作金'''<ref>[http://sillok.history.go.kr/id/kja_11112007_002 《燕山君日記》60卷，燕山君11年(1505年)12月7日(丁巳)2번째기사]</ref>，是[[朝鮮王朝|朝鮮王朝]]第10代君主，1494年至1506年在位。第7代君主[[朝鮮世祖|世祖]]曾孫、第11代君主中宗異母兄，也是继[[朝鮮端宗|端宗]]（鲁山君）之后，朝鮮王朝歷史上第二位廢王。

== 生平 ==
燕山君由[[廢妃尹氏|廢妃尹氏]]所生，為成宗的[[嫡長子|嫡長子]]。尹氏本是成宗的繼妃，但因[[妒忌|妒忌心]]異常強烈，不但意圖殺害與成宗有關係的[[宮女|宮女]]，還弄傷成宗臉部，因而在成宗十年（1479年）被廢，當時燕山君3歲。成宗十三年（1482年）尹氏被賜死藥，所以燕山君實際上是由[[貞顯王后|貞顯王后]]所撫養而長大的{{efn|《[[朝鲜王朝實錄|朝鲜王朝實錄]]》中並未有任何記載廢妃尹氏與貞顯王后爭寵的事，與尹氏爭寵的應該是鄭氏與嚴氏，事實上貞顯王后在當上王妃之前在成宗後宮並不起眼。}}{{efn|燕山君在殺害鄭氏與嚴氏時確實有一度想殺害貞顯王后，但一方面因為王后慎氏求情，一方面顧念貞顯王后的撫養之恩，所以並未下手。}}。

燕山君7歲時（1483年）被封為[[王世子|王世子]]。其性格粗暴且自幼就厭惡讀書，也有人認為他作為王位繼承者並不適任。成宗雖然也認為燕山君不足以繼位，但仍未下廢嫡的決定。1494年成宗過世，由王世子燕山君繼位。

== 即位 ==
受了信仰[[佛教|佛教]]的貞顯王后影響，燕山君不喜[[儒家|儒家]]文化，因而經常與尊崇儒學的百官發生衝突。燕山君任內曾發生兩次士禍，分別為1498年的[[戊午士禍|戊午士禍]]及1504年的[[甲子士禍|甲子士禍]]。甲子士禍的成因，是燕山君聽信[[外戚|外戚]][[任士洪|任士洪]]讒言，追捕當年與生母尹氏的案件相關人士並加以殺害。燕山君經常與[[官妓|官妓]]四處遊玩，並對進諫的大臣们一個個施以[[酷刑|酷刑]]，是朝鲜王朝史無前例的極惡無道又冷酷無情的君主。

== 被廢 ==
1506年[[中宗反正|中宗反正]]後，燕山君遭[[成希顏|成希顏]]、[[朴元宗|朴元宗]]等朝臣廢除，被流放至[[江華島|江華島]]，兩個月後病死<ref>[http://sillok.history.go.kr/inspection/insp_king.jsp?id=wka_10111008_002&tabid=w 《中宗實錄》1卷，中宗元年（1506年）11月8日紀錄二]</ref>，得年30歲。因以暴君身分被廢，因而無廟號、尊號、諡號、陵名等。

== 評價 ==
朝鮮史書評價燕山君是「悖惡」。為鎮壓朝臣，燕山君制定[[寸斬|寸斬]]、[[炮烙|炮烙]]、[[拆胸|拆胸]]、[[碎骨飄風|碎骨飄風]]等酷刑，並不時派[[醫女|醫女]]賜死藥予政敵；由於官妓制度在他登基前就被廢去，他為了宴樂而將醫女強充官妓，又派[[採紅駿使|採紅駿使]]強搶民女進宮為官妓。燕山君極為寵愛[[妓生|妓生]]出身的[[張綠水|張綠水]]，後因官妓人數眾多，將[[成均館|成均館]]和[[塔洞公園|佛寺]]改建為[[妓院|妓院]]，這些行徑都被指為“淫亂宮庭”。此外，燕山君亦因有人用訓字（現[[韓語|韓語]]所用的[[諺文|諺文]]）羞辱他，因而下令禁止使用訓字。
