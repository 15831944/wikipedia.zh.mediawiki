'''燕雀科'''（[[学名|学名]]：{{lang|la|Fringillidae}}）又名'''雀科'''，是[[鸟纲|鸟纲]][[雀形目|雀形目]]中的一个[[科_(生物)|科]]，包含了多种称为“雀”的鸟类。

== 外形 ==
本科的鸟体形较小，外形似[[麻雀|麻雀]]。锥状[[喙|喙]]厚实有力，嘴峰隆起，上下嘴边缘平滑，闭合性好，这一点与[[鹀科|鹀科]]的鸟有着明显的区别。本科中[[交嘴雀属|交嘴雀属]]的鸟喙的形态特殊。在[[解剖学|解剖学]]上，角质腭两侧的纵棱平行，这是与文鸟科的重要区别点。[[羽毛|尾羽]]12枚，初级飞羽10枚，但通常外侧的第一枚退化。

== 习性 ==
本科的鸟[[鸣肌|鸣肌]]发达，善于鸣叫，有些种雌雄都善鸣叫。

本科的鸟多以[[植物|植物]]的[[果实|果实]]、[[种子|种子]]为食，繁殖期时以昆虫哺育雏鸟。本科的鸟巧于营巢，常在树上或地面上筑巢。巢一般是开放式程碗状。

本科的鸟是晚成雏。

== 与人类的关系 ==
本科的鸟由于鸣叫动听、外形美丽，在世界各地深受人们的喜爱，也是深受[[鸟类买卖|鸟类买卖]]的危害。本科鸟由于食性复杂，容易饲养，也是人们喜欢的原因之一。
