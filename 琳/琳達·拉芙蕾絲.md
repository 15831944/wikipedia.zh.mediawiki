'''琳達·蘇珊·波曼'''（{{lang-en|Linda Susan Boreman}}，{{bd|1949年|1月10日|2002年|4月22日|catIdx=Lovelace, Linda}}），[[藝名|藝名]]'''琳達·拉芙蕾絲'''（{{lang-en|Linda Lovelace}}），生於美國[[紐約市|紐約市]][[布朗克斯|布朗克斯]]，著名[[色情演員|色情演員]]、作家，以演出電影《[[深喉嚨_(電影)|深喉嚨]]》聞名。她因為她的前夫而成為色情演員，在離開色情電影業之後，她也加入了教會，並成為[[反色情運動|反色情運動]]（Anti-pornography movement）的發言人。

== 生平 ==

=== 早年生活 ===
拉芙蕾絲生於一個不快樂的勞工家庭，母親桃樂絲·波曼（Dorothy Boreman）對她相當的嚴苛，父親約翰·波曼（John Borman）為警察，鮮少在家。<ref name="Linda Lovelace">{{Cite web |url= http://www.telegraph.co.uk/news/obituaries/1391959/Linda-Lovelace.html |title=Linda Lovelace }} </ref>中學時期她就讀於私立的[[天主教|天主教]]學校，且與交往對象維持一段很安全的距離，還因此得到「神聖小姐」的暱稱。<ref name="Linda's Life">{{Cite web  |url=http://old.nationalreview.com/comment/comment-briggs042502.asp  |title=Linda's Life  |deadurl=yes  |archiveurl=https://archive.is/20120605051617/old.nationalreview.com/comment/comment-briggs042502.asp  |archivedate=2012-06-05  }} </ref>當她16歲時，她的父親從紐約市警隊退休，波曼一家搬到佛羅里達州。<ref>{{Cite web |url=http://www.arlindo-correia.org/linda_lovelace.html  |title=Linda Lovelace; Star of 'Deep Throat' }} </ref>後來她於20歲產下一子。<ref name="Linda Lovelace" />不久之後，她回到紐約居住，並開始上電腦學校。<ref name="Linda's Life" />
