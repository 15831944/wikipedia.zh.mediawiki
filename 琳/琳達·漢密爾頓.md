'''琳達·卡蘿爾·漢密爾頓'''（{{lang|en|'''Linda Carroll Hamilton'''}}，{{bd|1956年|9月26日|catIdx=}}），是一位[[美國|美國]][[女演員|女演員]]，因《[[魔鬼終結者|魔鬼終結者]]》演出莎拉·康納一角而聞名。

== 經歴 ==
小時候在劇團當兒童演員，高中畢業後，入讀當地藝術學院，後來為了要學習戲劇演出，搬到紐約成為[[演員|演員]]。1979年第一次出演電影作品[[夜花|夜花]]，1984年第二次出演就是她的代表作品《[[魔鬼終結者|魔鬼終結者]]》，在當中飾演莎拉·康納。1987年主演電視劇集《[[俠膽雄獅_(1987年電視劇)|俠膽雄獅]]》兩度獲獎，成為[[金球獎|金球獎]]和[[艾美獎|艾美獎]]的得獎者。1991年，在[[终结者2：审判日|魔鬼終結者續集]]再度飾演莎拉·康納，並以此獲[[土星獎|土星獎]]。

== 拒演 ==
在《[[终结者3|-{zh-hans:终结者3：机器的崛起;zh-hk:未來戰士3：殲滅者TX;zh-tw:魔鬼終結者3：機器來臨;}-]]》中，因為琳達·漢密爾頓對劇本沒興趣，所以沒有參與演出。

== 感情生活 ==
在私人生活中，琳達·漢密爾頓在1982年與{{link-en|布魯斯·艾伯特|Bruce Abbott}}結婚，與布魯斯·艾伯特生下了兒子，1989年琳達·漢密爾頓與布魯斯·艾伯特正式[[離婚|離婚]]。

琳達·漢密爾頓在1997年又與[[詹姆斯·卡麥隆|詹姆斯·卡麥隆]]（魔鬼終結者導演）再婚，其後生下了女兒，1999年琳達·漢密爾頓與詹姆斯·卡麥隆正式[[離婚|離婚]]。
