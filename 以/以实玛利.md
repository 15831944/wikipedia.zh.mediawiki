'''以實瑪利'''（{{lang-he|'''יִשְׁמָעֵאל‎'''}}，{{lang-en|'''Ishmael'''}}），[[基督新教|基督新教]]譯作'''以實瑪利'''；[[天主教|天主教]]譯作'''依市瑪耳'''；[[伊斯兰教|伊斯兰教]]譯作'''易司馬儀 '''，按照[[塔纳赫|塔纳赫]]与[[可蘭經|可蘭經]]记载。

== 犹太教中的以实玛利 ==
在[[犹太教|犹太教]]的《[[希伯来圣经|希伯来圣经]]》或称《[[塔纳赫|塔纳赫]]》中，以实玛利是[[亚伯拉罕|亚伯拉罕]]的妾侍[[夏甲|夏甲]]所生的兒子，上帝對他的預言是「為人必像野驢。他的手要攻打人，人的手也要攻打他；他必住在眾弟兄的東邊，後裔極其繁多甚至不可勝數。」（《[[塔纳赫|塔纳赫]]﹒[[创世纪|创世纪]]-16》）

因[[夏甲|夏甲]]在生下[[以實瑪利|以實瑪利]]後輕視[[撒拉|撒拉]]，[[撒拉|撒拉]]便虐待[[夏甲|夏甲]]及[[以實瑪利|以實瑪利]]，[[撒拉|撒拉]]更要求[[亞伯拉罕|亞伯拉罕]]趕走夏甲母子。後來以實馬利住在曠野，成為弓箭手，神諭令要其結婚生子（《[[创世纪|创世纪]]-21》）並生下十二族長（《[[创世纪|创世纪]]-25》），受上帝應許成為大國的祖先。以撒生[[雅各_(旧约圣经)|雅各]]和[[以扫|以掃]]，雅各後改名為[[雅各_(舊約聖經)|以色列]]（《[[創世紀|創世紀]]-32》），他兒子的後代後發展為以色列的十二支派（《[[創世紀|創世紀]]-49》）。日後的[[阿拉伯人|阿拉伯人]]即為[[以實瑪利|以實瑪利]]的後代，[[猶太人|猶太人]]則是[[以撒|以撒]]的後代。

== 伊斯兰教的以实玛利 ==
{{main|以實瑪利 (伊斯蘭教)}}
以实玛利的来历与犹太教中的说法相同，[[伊斯兰教|伊斯兰教]]的宣道者[[穆罕默德|穆罕默德]]曾以[[阿拉伯人|阿拉伯人]]為以实玛利后裔的理由，同當時的猶太拉比證明自己的先知身分。学者发现有人把穆罕默德的祖先追溯到尼拜约，也有人追溯到基达。<ref>以实玛利 {{cite web |url=http://www.answering-islam.org/Chinese/project/gb_topic13/gb_13_c.htm |title=存档副本 |accessdate=2009-04-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20100620044759/http://answering-islam.org/Chinese/project/gb_topic13/gb_13_c.htm |archivedate=2010-06-20 }}</ref>
