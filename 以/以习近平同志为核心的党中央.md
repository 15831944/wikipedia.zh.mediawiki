'''习近平同志为核心的党中央'''，又称“党中央的核心、全党的核心”<ref>{{cite web|url=http://www.bbc.com/zhongwen/simp/china/2016/10/161031_cr_cpc_sixth_plenary_session |title=点评中国：六中“核心论”意味着什么 |website = BBC中文网 |date=2016-10-31 }}</ref>，是[[中国共产党中央委员会总书记|中共中央总书记]][[习近平|习近平]]自2016年[[中国共产党第十八届中央委员会第六次全体会议|中共十八届六中全会]]获得的称号，取代之前[[胡锦涛|胡锦涛]]时期沿用下来的“习近平同志为总书记的党中央”<ref>{{cite web |url = http://zt.ccln.gov.cn/lzqh/duj/49496-all.shtml |title=明确习近平总书记的核心地位至关重要 |website = 中国干部学习网 |date=2016-11-03 |access-date=2018-03-25 |archive-url = https://web.archive.org/web/20180325233057/http://zt.ccln.gov.cn/lzqh/duj/49496-all.shtml |archive-date=2018-03-25 |dead-url=yes }}</ref>。2017年[[中国共产党第十九次全国代表大会|中共十九大]]召开以后，随着“[[习近平新时代中国特色社会主义思想|习近平新时代中国特色社会主义思想]]”写入《[[中国共产党章程|党章]]》，进一步形成以习近平为[[中国共产党|全党]]最高领袖的领导体制<ref>{{cite web |url = http://www.zaobao.com/znews/greater-china/story20180211-834475 |title = 分析：微视频以“人民领袖”称习近平 中央媒体宣传为“领袖”形成共识 |publisher=联合早报 |date=2018-02-11 }}</ref>，实际取代了[[中华人民共和国|中国]][[改革开放|改革开放]]以来实行的[[中国共产党集体领导制度|集体领导制度]]<ref>{{cite web |url = https://www.voachinese.com/a/china-politics-20171025/4085541.html |title=中共新领导班子出台，未明确习近平的接班人 |website=美国之音中文网 |date=2017-10-25 }}</ref>，确立了习近平作为唯一[[领导核心|领导核心]]的地位<ref>{{cite web |url = http://theory.people.com.cn/n1/2017/1124/c40531-29665457.html |title=学习时报评论员：深入认识维护习近平总书记核心地位的重大意义|website=人民网|date=2018-11-24}}</ref>。

2017年10月27日，中共中央总书记[[习近平|习近平]]主持召开[[中共中央政治局|中共中央政治局]]会议，会议通过《中共中央政治局关于加强和维护党中央集中统一领导的若干规定》，其中除了突出习近平总书记作为全党的核心地位，还规定全体中央政治局委员和常委每年都要向党中央和总书记书面述职<ref>{{cite web |url = http://www.xinhuanet.com/politics/19cpcnc/2017-10/27/c_1121868508.htm |title=中共中央政治局召开会议 研究部署学习宣传贯彻党的十九大精神 |website=新华网|date=2017-10-27 }}</ref>。这样的安排颠覆了[[中国共产党第十二次全国代表大会|中共十二大]]废除[[中国共产党中央委员会主席|主席制]]以来总书记与其他[[中国共产党中央政治局常务委员会|政治局常委]]集体领导的安排<ref>{{cite web|url=http://www.chinadaily.com.cn/hqzg/2007-10/20/content_6193339.htm|title=中国共产党进一步加强集体领导|publisher=中国日报|date=2007-10-20}}</ref>，形成总书记对政治局的垂直领导，政治局委员成为总书记的下属<ref>{{cite web|url=http://www.bbc.com/zhongwen/simp/chinese-news-41752378|title=中共新政治局常委揭晓 集体领导或被削弱|publisher=BBC 中文网|date=2017-10-25}}</ref>。2018年3月，习近平首次审阅中央政治局各委员提交的报告，他又向各政治局委员提出自己对他们的重要要求，展示他作为党总书记的绝对权威<ref>{{cite web|url=http://www.xinhuanet.com/politics/2018-03/21/c_1122569929.htm|title=中央政治局同志向党中央和习近平总书记述职|publisher=新华网|date=2018-03-21}}</ref>。

2018年2月26日，[[中国共产党第十九届中央委员会第三次全体会议|中共十九届三中全会]]召开，会议通过[[深化党和国家机构改革|深化党和国家机构改革]]，其中包括组建[[中央全面依法治国委员会|中央全面依法治国委员会]]、[[中央审计委员会|中央审计委员会]]、[[中央教育工作领导小组|中央教育工作领导小组]]等中共领导机构<ref>{{cite web|url=http://cpc.people.com.cn/n1/2018/0305/c64094-29847159.html|title=中共中央关于深化党和国家机构改革的决定|publisher=人民网|date=2018-03-05}}</ref>，还改组原来的领导小组，改为[[中央全面深化改革委员会|中央全面深化改革委员会]]、[[中央网络安全和信息化委员会|中央网络安全和信息化委员会]]、[[中央财经委员会|中央财经委员会]]、[[中央外事工作委员会|中央外事工作委员会]]等委员会。<ref>{{cite web|url=http://www.gov.cn/zhengce/2018-03/21/content_5276191.htm#1|title=中共中央印发《深化党和国家机构改革方案》|publisher=中国政府网|date=2018-03-21}}</ref>除此以外，[[中共中央组织部|中共中央组织部]]、[[中共中央宣传部|中共中央宣传部]]、[[中共中央统战部|中共中央统战部]]的职能扩张，这些党内机构兼并部分原来隶属于[[中华人民共和国国务院|国务院]]行政部门的职权<ref>{{cite web|url=http://www.bbc.com/zhongwen/simp/chinese-news-43484361|title=学者称中国机构改革回归“党政合一”、国务院权力被削|publisher=BBC 中文网|date=2018-03-21}}</ref>。新设立的委员会覆盖党政各个层面，直接向以总书记为首的中央政治局负责，其中多个重要委员会更加由习近平亲自挂帅。
