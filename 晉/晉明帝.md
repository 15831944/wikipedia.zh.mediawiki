'''晉明帝 司馬紹'''（{{bd|299年||325年||10月18日|catIdx=S司馬}}），字'''道畿'''，[[東晉|東晉]]的第二代皇帝，[[晉元帝|晉元帝]]司馬睿長子。母親是[[豫章郡君|豫章郡君荀氏]]。在位不足三年，但在位期間平定了[[王敦之亂|王敦之亂]]。

== 生平 ==
司馬紹自小聰慧，故此特別受父親司馬睿所寵愛。後於[[永嘉_(西晋)|永嘉]]元年（307年）隨父親一同移鎮[[建業|建業]]（後改建康，今[[江蘇|江蘇]][[南京市|南京市]]）。[[建兴_(西晋)|建興]]元年（313年），司馬睿升任左丞相，拜司馬紹為東中郎將，鎮守[[廣陵|廣陵]]。316年，[[晉愍帝|晉愍帝]]所在的[[長安|長安]]被[[前趙|前趙]]攻陷，晉愍帝出降，西晋灭亡。有鉴于此，317年，司馬睿稱晉王，建元[[建武_(东晋)|建武]]，並立司馬紹為晉王太子。318年，司馬睿即位称帝，改元[[大兴_(东晋)|太興]]，司馬紹被立為皇太子。

[[永昌_(东晋)|永昌]]元年（322年）發生[[王敦之亂|王敦之亂]]，[[大將軍|大將軍]][[王敦|王敦]]領兵進攻建康並佔領[[石頭城|石頭城]]，晉元帝派[[王導|王導]]等人進攻石頭城但都被王敦擊敗，司馬紹於是打算率領將士與王敦決一死戰，即將出發時因遭[[太子中庶子|太子中庶子]][[溫嶠|溫嶠]]極力勸阻而沒有實行。隨後王敦自任丞相並掌握朝政，見司馬紹勇而有謀，而且朝野中亦有很高名望，於是打算誣陷他不孝而將他廢掉，但因溫嶠等大臣支持司馬紹，王敦終也不能廢掉司馬紹。

晉元帝因王敦之亂而憂憤成疾，於當年閏十一月己丑日（323年1月3日）病逝，司馬紹在次日繼位，为晉明帝，並由[[司空|司空]]王導輔政。

王敦雖於[[永昌_(东晋)|永昌]]元年（322年）就回到[[武昌|武昌]]遙控朝廷，但因為圖謀篡位，於[[太宁_(东晋)|太寧]]元年（323年）暗示要朝廷徵召自己入朝，晉明帝於是以手詔徵召王敦。同年，[[王允之|王允之]]乘酒宴而知道王敦的圖謀，於是回京告訴其父[[王舒|王舒]]，王舒於是與王導一同報告晉明帝，得以早作防備。

次年，晉明帝既心知王敦意圖，於是騎馬微服去視察王敦於[[于湖|于湖]]的營地，但遭到軍人發現，並派五名騎兵追捕。晉明帝逃走時，用水浸濕所騎馬匹的粪便来使其降温，又拿出七寶鞭交給路旁賣食物的婆婆，並要她出示給追來的騎兵。晉明帝走後不久，追兵就來到，並詢問婆婆，婆婆於是取出七寶鞭，並稱那人已經走得很遠。騎兵們顧著傳玩七寶鞭而在那裏停留了很久，而且見馬糞已冷，以為追不及了，於是都沒有再追，晉明帝因此成功逃脫。

及後，晉明帝積極準備京師[[建康|建康]]的防護，最終於當年成功擊敗王敦派來進攻的軍隊，平定了王敦之亂。王敦之亂後，晉明帝下令不再問罪於王敦一眾官屬，又分別以[[應詹|應詹]]為[[江州|江州]]刺史、[[劉遐|劉遐]]為[[徐州_(古代)|徐州]]刺史、[[陶侃|陶侃]]為[[荊州|荊州]]刺史、王舒為[[湘州|湘州]]刺史，重整各州形勢，消除王敦以王氏宗族各領諸州以凌弱帝室的失衡情形{{NoteTag|[[王敦|王敦]]以[[王邃|王邃]]為徐州刺史、王舒為荊州刺史、王彬為江州刺史。湘州在[[司馬承|司馬承]]被殺後不明，王舒前任刺史劉顗不知何時上任。}}。

[[太宁_(东晋)|太寧]]三年閏八月戊子（325年10月18日），司馬紹病逝於東堂，年僅二十七歲。葬於武平陵，[[廟號|廟號]]'''[[肅祖|肅祖]]'''{{NoteTag|name=庙号}}。

== 性格特徵 ==
* 司馬紹年少聰明，小時候便曾經與父親就「太陽與長安孰近」的問題作出不同答案的爭辯。長大後聰明有機斷，精於事理，於是能讓國家從王敦之亂的亂局回復平定。
* 司馬紹性至孝，有文武才略，敬重賢人，素好文辭，於是當時如王導、[[庾亮|庾亮]]、溫嶠、[[桓彝|桓彝]]、[[阮放|阮放]]等名臣都親待他。而因他習武藝，善於安撫將士，於是任太子時東宮聚集很多人，亦得遠近各人歸心。
* 王敦曾称呼晋明帝为：「黄鬚[[鲜卑|鲜卑]]奴」，這是因為其母建安郡君荀氏是燕代人，混雜了當地鮮卑人血統，故明帝可能也長得有一些像外族，[[鬚|鬚]]為黃色。

== 軼事 ==
* 司馬紹任太子時，想修建池苑樓臺，但元帝不許。司馬紹於是命手下的武士在一晚之間修好太子西池。<ref>《世說新語·豪爽》</ref>
* 司馬紹有寵妃[[宋褘|宋褘]]，褘國色天香，善吹笛，乃[[石崇|石崇]]妾[[綠珠|綠珠]]之女弟子，不久司馬紹病篤，群臣进谏，请出宋袆，最後宋褘被送給吏部尚书[[阮孚|阮孚]]。<ref>[[沈約|沈約]]《[[s:俗說|俗說]]》</ref>
* 司馬紹在位時，曾問晉室得天下的事。[[王導|王導]]於是告訴他[[司馬懿|司馬懿]]當日發動[[高平陵之變|高平陵之變]]誅除[[曹爽|曹爽]]，樹立[[蔣濟|蔣濟]]等與自己同心的大臣；又說道曹髦被[[司馬昭|司馬昭]]親信[[賈充|賈充]]所命的[[成濟|成濟]]弒殺一事。司馬紹聽後，將面龐伏在牀上，說：「若真的像你所說，晉室國祚又怎能夠長遠！」<ref>《晉書·宣帝紀》及《[[世說新語|世說新語]]·尤悔篇》</ref>

== 評價 ==
* 唐代[[房玄齡|房玄齡]]於《[[晉書|晉書]]》的「[[史臣|史臣]]曰」評論說：「維揚作宇，憑帶洪流，楚江恆戰，方城對敵，不得不推誠將相，以總戎麾。樓船萬計，兵倍王室，處其利而無心者，[[周公|周公]]其人也。威權外假，嫌隙內興，[[王敦|彼]]有[[荊州|順流之師]]，此無強籓之援。商逢九亂，堯止八音，明皇（案：晉明帝）負圖，屬在茲日。運龍韜於掌握，起天旆於江靡，燎其餘燼，有若秋原。去縗絰而踐戎場，斬鯨鯢而拜園闕。鎮削威權，州分江漢，覆車不踐，貽厥孫謀。其後七十餘年，終罹[[桓玄|敬道]]之害。或曰：『興亡在運，非止上流。』豈創制不殊，而弘之者異也。」
* 唐代某貴族「[[公子|公子]]」{{NoteTag|學者李錦繡懷疑此公子實指[[唐太宗|唐太宗]]，陳虎則認為就是太宗無誤，不過是登基前的秦王[[李世民|李世民]]與虞世南的對話紀錄，見李錦繡，〈读敦煌P2636《帝王略论》文書札记〉，《论史传经》（上海：上海古籍出版社，2004）、（唐）虞世南撰、陳虎譯註，《帝王略論》（北京：中華書局，2008）卷一}}與[[虞世南|虞世南]]的對話：「公子曰：『東晉自元帝已下，何為賢主？』先生曰：『晉自遷都江左，強臣擅命，（天子）垂拱南面，政非己出。王敦以磐石之宗，居上流之地，負才矜地，志懷沖問鼎，非明帝之雄斷，王導之忠誠，則晉祚其移於他族矣。若使[[壽命|降年]]永久，佐任群賢，因洛、澗之遺黎，乘[[前趙|劉]]、[[後趙|石]]之衰運，興復中原，不難圖也。』」<ref>《唐文拾遺》卷十三；亦收入（唐）虞世南 撰、陳虎譯 註，《帝王略論》（北京：中華書局，2008）</ref>
