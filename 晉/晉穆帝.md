'''晉穆帝司馬聃'''（{{bd|343年||361年|7月10日|晉穆帝}}），字'''彭子'''，[[東晉|東晉]]第五代皇帝，[[廟號|廟號]]孝宗。晉穆帝是[[晉康帝|晉康帝]]之子，母[[褚蒜子|褚蒜子]]<ref>《[[s:晉書/卷008|晉書·卷八·帝紀第八 穆帝 哀帝 海西公(廢帝)]]》：穆皇帝諱聃，字彭子，康帝子也。建元二年九月丙申，立為[[皇太子|皇太子]]。戊戌，康帝崩。己亥，[[太子|太子]]即皇帝位，時年二歲。</ref>。

== 生平 ==
[[建元_(晉)|建元]]二年（344年），由於晉康帝僅22歲便駕崩，晉穆帝即位，時年兩歲；由於年幼而由[[褚太后|褚太后]]掌政<ref>《[[s:晉書/卷008|晉書·卷八·帝紀第八 穆帝 哀帝 海西公(廢帝)]]》：大赦，尊皇后為皇太后。壬寅，皇太后臨朝攝政。永和元年春正月甲戌朔，皇太后設白紗帷於太極殿，抱帝臨軒。改元。甲申，進鎮軍將軍、武陵王晞為鎮軍大將軍、開府儀同三司，以鎮軍將軍顧眾為尚書右僕射。</ref>，並由[[何充|何充]]輔政。何充過世後改由[[蔡謨|蔡謨]]與[[司馬昱|司馬昱]]輔政<ref>《[[s:晉書/卷008|晉書·卷八·帝紀第八 穆帝 哀帝 海西公(廢帝)]]》：己卯，使持節、侍中、都督揚州諸軍事、揚州刺史、驃騎將軍、錄尚書事、都鄉侯何充卒。二月癸醜，以左光祿大夫葵謨領司徒，錄尚書六條事、撫軍大將軍、會稽王昱及謨並輔政。</ref>。晉穆帝在位期間東晉雖然北伐失敗，但是由於[[桓溫|桓溫]]消滅了在[[四川|四川]]立國的[[成漢|成漢]]<ref>《[[s:晉書/卷008|晉書·卷八·帝紀第八 穆帝 哀帝 海西公(廢帝)]]》：三年春正月乙卯，桓溫攻成都，克之。丁亥，李勢降，益州平。</ref>，並且於[[永和_(晋穆帝)|永和]]十二年（356年）三月，奪回[[洛陽|洛陽]]<ref>《[[s:晉書/卷008|晉書·卷八·帝紀第八 穆帝 哀帝 海西公(廢帝)]]》：秋八月己亥，桓溫及姚襄戰于伊水，大敗之，襄走平陽，徙其餘眾三千余家于江漢之間，執周成而歸。使揚武將軍毛穆之，督護陳午，輔國將軍、河南太守戴施鎮洛陽。</ref>，雖然不久就因為糧運不繼而撤退，東晉的版圖仍然有所擴大。[[昇平|昇平]]元年 ( 357年 ) 晉穆帝行[[冠禮|冠禮]]後褚太后歸政，晉穆帝開始親政。

361年晉穆帝過世，得年十八歲，葬永平陵<ref>《[[s:晉書/卷008|晉書·卷八·帝紀第八 穆帝 哀帝 海西公(廢帝)]]》：五月丁巳，帝崩于顯陽殿，時年十九。葬永平陵，廟號孝宗。</ref>。由晉成帝長子琅邪王[[司馬丕|司馬丕]]繼位<ref>《[[s:晉書/卷008|晉書·卷八·帝紀第八 穆帝 哀帝 海西公(廢帝)]]》：五年五月丁巳，穆帝崩。皇太后令曰：「帝奄不救疾，胤嗣未建。琅邪王丕，中興正統，明德懋親。昔在咸康，屬當儲貳。以年在幼沖，未堪國難，故顯宗高讓。今義望情地，莫與為比，其以王奉大統。」於是百官備法駕，迎於琅邪第。</ref>。
