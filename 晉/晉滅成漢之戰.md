'''晉滅成漢之戰'''是[[中國|中國]]東晉時期的一場戰爭，由東晉荊州刺史桓溫所領導，進攻據有益州的成漢政權。桓溫於永和二年（346年）十一月出兵，至次年三月攻下成都並接受李勢投降，歷時約四個月。戰後東晉吞併成漢全境。

== 背景 ==
[[File:Map_of_Sixteen_Kingdoms_2.png|280px]]與[[東晉|東晉]]形勢圖。]]
[[西晉|西晉]][[永興_(西晉)|永興]]元年（304年），[[李雄_(十六國)|李雄]]於成都稱成都王，自此割據益州自立。[[咸康_(東晉)|咸康]]四年（338年），成漢宗室[[李壽|李壽]]發動政變篡位。李壽至建元元年（343年）去世，其子李勢繼位，但李勢為人荒淫而不恤國事，親近小人，少見大臣，而且常猜忌他人，因而誅殺大臣和濫加刑獄，終令到國內「夷獠叛亂，軍守離缺，境宇日蹙，人懷危懼。」<ref>《晉書·李勢傳》：「勢既驕吝，而性愛財色，常殺人而取其妻，荒淫不恤國事。夷獠叛亂，軍守離缺，境宇日蹙。加之荒儉，性多忌害，誅殘大臣，刑獄濫加，人懷危懼。斥外父祖臣佐，親任左右小人，群小因行威福。又常居內，少見公卿。」</ref>

與此同時，東晉荊州刺史[[庾翼|庾翼]]亦有北伐之心，並向晉康帝上表陳述北伐計劃，更移鎮[[襄陽|襄陽]]。其時庾翼籌備北伐徵集了他所領[[荊州_(古代)|荊]][[江州|江]][[司州|司]][[雍州|雍]][[梁州|梁]]益六州的編戶奴及車牛驢馬，引來了百姓嗟怨；而移鎮襄陽一事雖已先藉詞移鎮[[安陸|安陸]]，但仍遭朝中大臣反對，唯有兄長[[庾冰|庾冰]]及桓溫和譙王[[司馬無忌|司馬無忌]]支持<ref>《晉書·庾翼傳》：「時欲向襄陽，慮朝廷不許，故以安陸為辭。帝及朝士皆遣使譬止，車騎參軍孫綽亦致書諫。」「初，翼遷襄陽，舉朝謂之不可，議者或謂避衰，唯兄冰意同，桓溫及譙王無忌贊成其計。」</ref>。庾翼雖繕修兵器和積聚糧食，但尚未正式實行就已在永和元年（345年）去世。時在朝中輔政的[[何充|何充]]則以桓溫接替庾翼為荊州刺史。

桓溫上任後見成漢國力衰弱，於是意圖出兵滅了成漢以建立功勳<ref>《晉書·桓溫傳》：「時李勢微弱，溫志在立勳於蜀。」</ref>，但當時桓溫將佐，乃至當時大眾都認為此事難以成功<ref>《晉書·劉惔傳》：「及溫伐蜀，時咸謂未易可濟。」</ref><ref>《資治通鑑·卷九十七》永和二年：「安西將軍桓溫將伐漢，將佐皆以為不可」</ref>。其時[[江夏郡|江夏]]相[[袁喬|袁喬]]則力勸桓溫伐蜀，並分析了成漢人恃仗天險而不作防備的行為，建議輕兵深入，攻其無備；又分析了後趙和東晉長江守軍，表示出征不會影響對北方的防禦，令人有機可乘，最後又指出收復蜀地的益處<ref>《晉書·袁喬傳》：「喬勸溫曰：『夫經略大事，故非常情所具，智者了於胸心，然後舉無遺筭耳。今天下之難，二寇而已。蜀雖險固，方胡為弱，將欲除之，先從易者。今泝流萬里，經歷天險，彼或有備，不必可克。然蜀人自以斗絕一方，恃其完固，不修攻戰之具，若以精卒一萬，輕軍速進，比彼聞之，我已入其險要，李勢君臣不過自力一戰，擒之必矣。論者恐大軍既西，胡必闚覦，此又似是而非。何者？胡聞萬里征伐，以為內有重備，必不敢動。縱復越逸江渚，諸軍足以守境，此無憂矣。蜀土富實，號稱天府，昔諸葛武侯欲以抗衡中國。今誠不能為害，然勢據上流，易為寇盜。若襲而取之者，有其人衆，此國之大利也。』」</ref>。桓溫同意袁喬所言，決心攻伐成漢。

== 經過 ==
永和二年十一月辛未日（346年12月10日），桓溫率領益州刺史[[周撫|周撫]]、南郡太守譙王司馬無忌攻伐成漢，並命袁喬領二千兵作為前鋒。桓溫上表出征後便出發，當時朝廷卻以桓溫孤軍深入蜀地而感到憂慮<ref>《晉書·桓溫傳》：「溫將發，上疏而行。朝廷以蜀險遠，而溫兵寡少，深入敵場，甚以為憂。」</ref>。

桓溫到[[青衣縣|青衣]]時，李勢興兵預備抵抗，又派了右衞將軍[[李福|李福]]、鎮南將軍[[李權|李權]]和前將軍[[昝堅|昝堅]]等領兵自[[山陽|山陽]]（今[[四川|四川]][[樂山縣|樂山縣]]西南）兵向合水（[[大渡河|大渡河]]入江口），以作抵禦。當時眾將都意圖於長江南岸設下伏兵等待晉軍，但昝堅不聽，堅持要率兵自[[鴛鴦碕|鴛鴦碕]]兵向[[犍為|犍為]]（今四川[[宜賓市|宜賓市]]西南）。桓溫則一直沿江前進，繞過犍為而於永和三年（347年）三月到達[[彭模|彭模]]（今四川[[彭山縣|彭山縣]]）。

桓溫到彭模後，有建議要桓溫兵分兩路進攻，分散敵方防禦兵力。不過袁喬認為應當全軍齊心合力進攻，分兵只會令人心不一，萬一有一軍失利就大勢已去，反建議桓溫拋棄炊具，讓士兵只帶三日糧食，直奔[[成都|成都]]（今四川成都市），刺激士兵士氣<ref>《晉書·袁喬傳》：「喬曰：『今深入萬里，置之死地，士無反顧之心，所謂人自為戰者也。今分為兩軍，軍力不一，萬一偏敗，則大事去矣。不如全軍而進，棄去釜甑，齎三日糧，勝可必矣。』」</ref>。桓溫於是留下[[參軍|參軍]][[孫盛|孫盛]]和[[周楚|周楚]]領弱兵在彭模守輜重，自己就親率主力進攻成都。

當時李福曾經進攻彭模，不過留守的孫盛等人奮戰，成功擊退了李福。而桓溫主力就與李權發生了戰事，三戰三勝，兵敗的成漢軍都逃返成都，成漢鎮軍將軍[[李位都|李位都]]更歸降。到了犍為的昝堅此時才知已錯過了桓溫大軍，於是退返。昝堅追上桓溫時，桓溫已經到了成都附近的[[十里陌|十里陌]]，昝堅軍因而自潰。李勢則在成都西南的[[笮橋|笮橋]]迎戰桓溫，初戰桓溫前鋒失利，參軍[[龔護|龔護]]戰死，士兵都恐懼後撤。不過其時擊打戰鼓的人卻擂鼓進攻，袁喬因此乘勢指揮軍隊奮戰，士氣漸漸高漲，終大敗成漢守軍。桓溫於是繼續攻向成都，火燒大城各個城門<ref>《晉書·李勢傳》：「溫至城下，縱火燒其大城諸門。」</ref>。成漢軍民都十分惶恐，再無鬥志，當時成漢[[中書監|中書監]][[王嘏|王嘏]]及[[散騎常侍|散騎常侍]][[常璩|常璩]]都勸李勢歸降，不過李勢慮及昔日[[東漢|東漢]][[大司馬|大司馬]][[吳漢|吳漢]]攻滅[[成家|成家]]時族誅[[公孫述|公孫述]]，怕投降會遭屠戮<ref>《晉書·李勢傳》：「勢以問侍中[[馮孚|馮孚]]，孚言：『昔吳漢征蜀，盡誅公孫氏。今晉下書，不赦諸李，雖降，恐無全理。』」</ref>，只得乘夜逃出成都，逃到[[晉壽|晉壽]]的[[葭萌城|葭萌城]]。其時[[鄧嵩|鄧嵩]]和昝堅都勸其投降，李勢終也決定投降，命[[王幼|王幼]]送降書到桓溫那處，並反縛雙手，載著棺材以示投降。桓溫為其鬆縛並燒掉棺材，送李勢和其他宗室共十多人到建康。成漢滅亡。

== 結果 ==
桓溫於此戰消滅了成漢，晉朝重奪了益、梁州地區，是東晉建國以來最成功的一次對外戰爭。不過，桓溫雖然選任成漢良臣如[[王誓|王誓]]、[[鄧定|鄧定]]、[[常據|常據]]等以安撫蜀地，但桓溫尚未撤軍就發生了王誓與鄧定、隗文及王潤的叛變<ref>《晉書·桓溫傳》：「軍未旋而王誓、鄧定、隗文等反。」</ref>。及至桓溫撤軍西歸後隗文和王潤仍趁機佔領成都，奉前成漢國師[[范長生|范長生]]子[[范賁|范賁]]為帝，招來很多蜀人支持<ref>《資治通鑑·卷九十七》永和三年：「夏，四月，丁巳，鄧、隗文等人入據成都，征虜將軍楊謙棄涪城，退保德陽。……秋，七月……隗文、鄧定等立故國師范長生之子賁為帝而奉之，以妖異惑眾，蜀人多歸之。」</ref>；晉將[[蕭敬文|蕭敬文]]也乘機叛亂<ref>《資治通鑑·卷九十七》永和三年：「十二月，振威護軍蕭敬文殺征虜將軍楊謙，攻涪城，陷之，自稱益州牧；遂取巴西，通于漢中。」</ref>。兩事先後在永和五年（349年）<ref>《晉書·穆帝紀》：「（永和五年）夏四月，益州刺史周撫、龍驤將軍[[朱燾|朱燾]]擊范賁，獲之，益州平。」</ref>及永和八年（352年）<ref>《晉書·穆帝紀》：「（永和八年）八月，平西將軍周撫討蕭敬文于涪城，斬之。」</ref>才得解決。

桓溫亦因此戰而提高聲望，朝廷更曾打算以[[豫章郡|豫章郡]]進封桓溫<ref>《晉書·荀蕤傳》：「時桓溫平蜀，朝廷欲以豫章郡封溫。蕤言於帝曰：「若溫復假王盛，北平河洛，修復園陵，將何以加此！」於是乃止。</ref>，不過其聲望提高卻令朝廷忌憚他，當時輔政的會稽王[[司馬昱|司馬昱]]因而提拔名士[[殷浩_(東晉)|殷浩]]為揚州刺史，抗衡桓溫<ref>《晉書·殷浩傳》：「時桓溫既滅蜀，威勢轉振，朝廷憚之。簡文以浩有盛名，朝野推伏，故引為心膂，以抗於溫。」</ref>。故及至後來後趙皇帝[[石虎|石虎]]去世，桓溫上請北伐時朝廷亦久久不作批示<ref>《晉書·桓溫傳》：「及石季龍死，溫欲率眾北征，先上疏求朝廷議水陸之宜，久不報。時知朝廷杖殷浩等以抗己。」</ref>，反讓殷浩主持北伐，以抗衡桓溫<ref>《晉書·殷浩傳》：「及石季龍死，胡中大亂，朝廷欲遂蕩平關河，於是以浩為中軍將軍、假節、都督揚豫徐兗青五州軍事。」</ref>最終桓溫憑借殷浩北伐頻頻失利而上書彈劾殷浩，總掌內外大權，開始專掌東晉朝政<ref>《晉書·桓溫傳》：「時殷浩至洛陽修復園陵，經涉數年，屢戰屢敗，器械都盡。溫復進督司州，因朝野之怨，戶奏廢浩，自此內外大權一歸溫矣。」</ref>。

明末史家[[王夫之|王夫之]]認為，當時李勢昏庸，成漢衰弱，當立即興師吞併。但桓溫欲起兵時，朝廷卻在憂慮他不能取勝，桓溫因而看出朝廷之人皆見識淺陋，所以上表後便立即出兵，無視朝廷中人。最終則如劉惔所顧慮那樣，桓溫攻滅成漢後，得以專制朝廷<ref>《讀通鑑論》：「李勢立，驕淫虐殺，此天亡李氏之日，不待再計而宜興師者也。桓溫西討，晉廷惴惴然憂其不克，溫目笑而心鄙之，拜表即行，知晉之無人也。劉惔曰：『但恐克蜀之後，專制朝廷。』其言驗矣。</ref>。
