以下是'''自[[1959年太平洋颱風季|1959年]]至[[1999年太平洋颱風季|1999年]]期間，由[[聯合颱風警報中心|聯合颱風警報中心]]-{除名}-的[[太平洋颱風季|太平洋颱風]]名字'''之列表。該時段共有'''13個'''[[熱帶氣旋|熱帶氣旋]]名字被除名，包括再次被加入和替換的「貝絲」（Bess）。

== 概述 ==
自1959年[[1959年太平洋颱風季|太平洋颱風季]]起，[[聯合颱風警報中心|聯合颱風警報中心]]開始為西北[[太平洋|太平洋]]的[[熱帶氣旋|熱帶氣旋]]進行命名工作，在此之前由[[美國海軍|美國海軍]]直接負責。<ref>{{cite web |year=1999 |author=Gary Padgett |url=http://australiasevereweather.com/cyclones/2000/summ9901.htm |archivedate=2010-09-29 |title=Monthly Global Tropical Cyclone Summary January 1999 |accessdate=2010-09-29 |archiveurl=https://www.webcitation.org/5t60dYVRr?url=http://australiasevereweather.com/cyclones/2000/summ9912.htm |publisher=Australian Severe Weather |deadurl=yes }}</ref>命名表由四份名單組成，名字會由各[[英語|英文]]字母順序使用，每份名單有21個，命名表有84個名字，當用完四份名單之後，就會回到第一個重新開始。早期是以西方[[女性|女性]]名字依英文字母排列命名（除了[[Q|Q]]、[[U|U]]、[[X|X]]、[[Y|Y]]、[[Z|Z]]），所以當時的颱風又稱為「風姐」；到了1979年4月開始改以[[男性|男]]女名字相間的順序命名，但每份名單仍維持21個名字；直到1990年開始加入Y、Z字首的人名，令每份名單名字數目增至有23個，命名表就有92個名字。在該段時期的熱帶氣旋都非正式地採用這些名字，有關[[亞洲|亞洲]]國家氣象部門以及其轄區內的天氣報告是否使用這些颱風名稱，則由她們自行決定。'''當聯合颱風警報中心認為某個熱帶氣旋在某地區造成嚴重破壞及（或）引致很多人死亡，他們會自行將有關名字除去，再以另一個名字作為替補。'''在這40年的時期裡，總共有13個名字因造成了嚴重影響而被聯合颱風警報中心自行-{除名}-和替換。<ref>{{cite web|publisher=香港天文台|year=2007年|title=二零零六熱帶氣旋|accessdate=2012-09-19|url=http://www.weather.gov.hk/publica/tc/tc2006/chinese/section1.htm#1.3}}</ref>值得留意的是，於1974年-{除名}-的「貝絲」（Bess）在1979年命名表改革中再次被加入，但於1982年因[[1982年颱風貝絲|超級颱風貝絲]]嚴重影響[[日本|日本]]而再次被取代，所以仍會列入在列表內。<ref name="bess"/>

在1981年之前，聯合颱風警報中心更同時為西北太平洋熱帶氣旋編配一個國際編號，後來交由[[氣象廳_(日本)|日本氣象廳]]負責。聯合颱風警報中心於1995年因應「基地關閉與重整法案」開始撤離[[關島|關島]]，於是有部分[[亞洲|亞洲]]國家代表在[[世界氣象組織|世界氣象組織]][[颱風委員會|颱風委員會]]第31屆會議中，以此為理由，提出改革颱風命名表的建議，以彰顯亞洲各地區特色。<ref>{{cite web |url=http://news.bbc.co.uk/1/hi/world/asia-pacific/639275.stm |title=Chinese Dragon replaces Typhoon Ted |date=2000-02-11 |author=Staff Writer |publisher=BBC News |accessdate=2010-10-07 |archiveurl=https://www.webcitation.org/5tItchU2T?url=http://news.bbc.co.uk/2/hi/asia-pacific/639275.stm |archivedate=2010-10-07 |deadurl=yes }}</ref>颱風委員會接納此建議，並著手制訂新命名表，新命名表最終於2000年1月1日正式生效，並交由日本氣象廳為所有往後達到[[熱帶風暴|熱帶風暴]]強度的熱帶氣旋命名。<ref>{{cite web |url=http://australiasevereweather.com/cyclones/2000/summ9912.htm |title=Monthly Global Tropical Cyclone Summary December 1999 |author=Gary Padgett |year=2000 |publisher=Australian Severe Weather |accessdate=2010-09-29 |archiveurl=https://www.webcitation.org/5t60dYVRr?url=http://australiasevereweather.com/cyclones/2000/summ9912.htm |archivedate=2010-09-29 |deadurl=yes }}</ref>而[[菲律賓氣象局|菲律賓氣象局]]（1972年12月8日起名為菲律賓大氣地球物理和天文管理局【PAGASA】）則於1963年始使用自己一套命名法，為所有進入或產生於[[菲律賓|菲律賓]]風暴責任範圍以內的熱帶氣旋進行命名，作當地警報用途，名單每四年循環再用。<ref> {{cite web|author=Chris Landsea|year=2007|title=How are Tropical Cyclones Named?|publisher=Hurricane Research Division|accessdate=2008-03-05|url=http://www.aoml.noaa.gov/hrd/tcfaq/TCFAQ_B.txt}} </ref>

== 已退役熱帶氣旋名字名單 ==
列表內名單會列出其巔峰強度，最高持續風速是以一分鐘平均風速來計算；而有關風災帶來的經濟損失是以[[美元|美元]]計算，有些數字由於缺乏某些受影響地區的資料，可能與實際數字有所出入。另外，據聯合颱風警報中心對熱帶氣旋等級的定義，此列表中會以不同顏色代表不同級別的熱帶氣旋：

*{{colorbox|#{{storm colour|storm}}}}青色背景代表[[熱帶風暴|熱帶風暴]]（一分鐘平均風速：63－118[[公里每小時|公里／小时]]；包括[[氣象廳_(日本)|日本氣象廳]]定義下的[[強烈熱帶風暴|強烈熱帶風暴]]）
*{{colorbox|#{{storm colour|1}}}}淡黃色背景代表一級颱風（一分鐘平均風速：119－153公里／小时）
*{{colorbox|#{{storm colour|2}}}}黃色背景代表二級颱風（一分鐘平均風速：154－177公里／小时）
*{{colorbox|#{{storm colour|3}}}}淡[[橙色|橙色]]背景代表三級颱風（一分鐘平均風速：178－208公里／小时）
*{{colorbox|#{{storm colour|4}}}}橙色背景代表四級颱風（一分鐘平均風速：209－251公里／小时）
*{{colorbox|#{{storm colour|5}}}}橙紅色背景代表五級颱風（一分鐘平均風速：252公里／小时或以上）

請注意於1974年-{除名}-的「貝絲」（Bess）在1979年命名表改革中再次被加入，但於1982年因[[1982年颱風貝絲|超級颱風貝絲]]嚴重影響[[日本|日本]]而再次被取代。關於1958年或以前退役的風暴名稱，請參看[[已退役太平洋颱風名稱列表_(1947年-1958年)|已退役太平洋颱風名稱列表 (1947年-1958年)]]。
