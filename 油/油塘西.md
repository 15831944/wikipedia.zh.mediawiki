'''油塘西'''是香港[[觀塘區議會|觀塘區議會]]下轄的[[選區|選區]]，代號'''J26'''，成立於2003年，現任區議員為[[九龍社團聯會|九龍社團聯會]]／[[創建力量|創建力量]]成員[[呂東孩|呂東孩]]。

==範圍==
選區範圍包含[[油塘邨|油塘邨]]、[[油麗邨|油麗邨]]翠麗樓、康麗樓、仁麗樓、[[油塘站|油塘站]]、[[鯉灣天下|鯉灣天下]]、[[嘉賢居|嘉賢居]]、[[Ocean_One|Ocean One]]、[[Peninsula_East|Peninsula East]]、[[三家村|三家村]]、[[晒草灣|晒草灣]]和[[茶果嶺|茶果嶺]]等，與其相連的選區有[[平田_(選區)|平田]]、[[油塘東|油塘東]]、[[油麗|油麗]]、[[翠翔|翠翔]]、[[麗港城_(選區)|麗港城]]、[[景田|景田]]和[[西貢區議會|西貢區議會]]選區[[維景|維景]]選區。

隨著[[鯉魚門邨|鯉魚門邨]]和[[油塘邨|油塘邨]]等相繼落成，[[選舉事務處|選舉事務處]]在2003年將[[油塘四山|油塘四山]]分拆成油塘四山西和[[油塘東|油塘四山東]]，當時代號為J21。2005年，由於[[油麗邨|油麗邨]]落成後有大量人口遷入，選舉事務處因而在2007年將屬於油塘四山西的油麗邨和[[油美苑|油美苑]]與屬於油塘四山東範圍內除了高龍閣、高靜閣、高鳳閣和高康閣的[[高翔苑|高翔苑]]合併成為[[油塘中|油塘中]]，同時將油塘四山西改稱油塘西，並且將屬於油塘西的[[嶺南新村|嶺南新村]]劃入油塘東範圍。

2011年，選舉事務處將嶺南新村重新歸入油塘西範圍，油塘西的代號也改為J23<ref>{{zh-hant}}{{cite web|url=http://www.eac.gov.hk/pdf/distco/maps/dc2003j.pdf |title=2003年區議會選舉 - 選區分界概要 |publisher=[[香港區議會|香港區議會]] |date=2011-10-06 |accessdate=2011-11-12}}</ref><ref>{{zh-hant}}{{cite web|url=http://www.eac.gov.hk/pdf/distco/maps/dc2007j.pdf |title=2007年區議會選舉 - 選區分界概要 |publisher=[[香港區議會|香港區議會]] |date=2011-10-06 |accessdate=2011-11-12}}</ref><ref>{{zh-hant}}{{cite web|url=http://www.elections.gov.hk/dc2011/maps/dc2011j.pdf |title=2011年區議會選舉 - 選區分界概要 |publisher=[[香港區議會|香港區議會]] |date=2011-10-06 |accessdate=2011-11-12}}</ref><ref>{{zh-hant}}{{cite web|url=http://www.elections.gov.hk/dc2011/maps/dc2011q2.pdf |title=2011年區議會選舉 - 選區分界概要 |publisher=[[香港區議會|香港區議會]] |date=2011-10-06 |accessdate=2011-11-12}}</ref>。

2015年，選區代號由J23改為'''J24'''。

2019年，預計選區代號將由J24改為J26。

==選區特色==
[[油塘西|油塘西]]選區範圍頗大，既有傳統村落，如茶果嶺村、[[三家村|三家村]]，又有工廠大廈等，以及於[[千禧年|千禧年]]落成的新型[[公共屋邨|公共屋邨]][[油塘邨|油塘邨]]。後來，區內更落成[[油麗邨|油麗邨]]第三期以及幾個[[私人屋苑|私人屋苑]]，人口大增。但區內卻缺乏[[圖書館|圖書館]]、[[泳池|泳池]]、普通科門診等。將來，[[油塘西|油塘西]]區內的油塘灣將會有油塘灣綜合發展計劃、區內的[[工業區|工業區]]亦會有大規模的新發展，預計會興建[[私人屋苑|私人屋苑]]、[[商業大廈|商業大廈]]、[[海濱長廊|海濱長廊]]、和[[酒店|酒店]]，更會因地理位置優勢興建遊艇中心等。

==沿革==
[[2003年區議會選舉|2003年區議會選舉]]，當時人口估算約15,979人<ref>{{zh-hant}}{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/summary/summary_kt.html |title=2003年區議會選舉 - 選區分界概要 |publisher=[[香港區議會|香港區議會]] |date=2011-10-06 |accessdate=2011-12-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20111108185439/http://www.elections.gov.hk/elections/dc2003/tc_chi/summary/summary_kt.html |archivedate=2011-11-08 }}</ref>，其中有6,093位是選民<ref>{{zh-hant}}{{cite web|url=http://www.elections.gov.hk/elections/dc2003/tc_chi/eb/eb_num_kt.html|title=觀塘區|publisher=[[香港區議會|香港區議會]]|accessdate=2011-12-01|deadurl=yes|archiveurl=https://web.archive.org/web/20110115012327/http://www.elections.gov.hk/elections/dc2003/tc_chi/eb/eb_num_kt.html|archivedate=2011-01-15}}</ref>，當時投票站設於路德會聖馬可堂幼稚園和[[海濱學校|海濱學校]]<ref>{{zh-hant}}{{cite web|url=http://www.gld.gov.hk/egazette/pdf/20030743/cgn200307437196.pdf |title=投票站及點票站名單 |publisher=[[香港區議會|香港區議會]] |accessdate=2011-12-01}}</ref>，選區議席由[[無黨派|無黨派]]人士[[呂東孩|呂東孩]]與[[民主黨_(香港)|民主黨]]成員[[陳惠達|陳惠達]]爭奪<ref>{{zh-hant}}{{cite web|url=http://www.elections.gov.hk/elections/dc2003/common/platforms/J21.pdf|title=候選人簡介|publisher=[[香港區議會|香港區議會]]|accessdate=2011-12-01}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。最終，呂東孩以2,096票勝出<ref>{{zh-hant}}{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/result/result_kt.html |title=2003年區議會選舉 - 選舉結果(觀塘) |publisher=[[香港區議會|香港區議會]] |accessdate=2011-12-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20110511072608/http://www.elections.gov.hk/elections/dc2003/tc_chi/result/result_kt.html |archivedate=2011-05-11 }}</ref>。

[[2007年區議會選舉|2007年區議會選舉]]，當時人口估算約18,781人<ref>{{zh-hant}}{{cite web |url=http://www.elections.gov.hk/dc2007/chi/summaryj.html |title=2003年區議會選舉 - 選區分界概要 |publisher=[[香港區議會|香港區議會]] |date=2011-10-06 |accessdate=2011-12-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20150621192227/http://www.elections.gov.hk/dc2007/chi/summaryj.html |archivedate=2015-06-21 }}</ref>，其中有7,223位是選民<ref>{{zh-hant}}{{cite web|url=http://www.elections.gov.hk/dc2007/chi/num_electors_kwun_tong.html|title=觀塘區|publisher=[[香港區議會|香港區議會]]|accessdate=2011-12-01|deadurl=yes|archiveurl=https://web.archive.org/web/20110115151542/http://www.elections.gov.hk/dc2007/chi/num_electors_kwun_tong.html|archivedate=2011-01-15}}</ref>，尋求連任的呂東孩在沒有對手的情況下自動當選<ref>{{zh-hant}}{{cite web |url=http://www.elections.gov.hk/dc2007/chi/results_kwun_tong.html |title=2007年區議會選舉 - 選舉結果(觀塘) |publisher=[[香港區議會|香港區議會]] |accessdate=2011-12-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20111202180926/http://www.elections.gov.hk/dc2007/chi/results_kwun_tong.html |archivedate=2011-12-02 }}</ref>。

[[2011年區議會選舉|2011年區議會選舉]]，當時人口估算約20,567人<ref>{{zh-hant}}{{cite web|url=http://www.elections.gov.hk/dc2011/chi/summaryj.html |title=2011年區議會選舉 - 選區分界概要 |publisher=[[香港區議會|香港區議會]] |date=2011-10-06 |accessdate=2011-11-12}}</ref>，其中有9,027位是選民<ref>{{zh-hant}}{{cite web|url=http://www.elections.gov.hk/dc2011/chi/ebriefsj.html |title=觀塘區 |publisher=[[香港區議會|香港區議會]] |accessdate=2011-11-12}}</ref>，投票站改設於[[天主教普照中學|天主教普照中學]]和[[鯉魚門體育館|鯉魚門體育館]]<ref>{{zh-hant}}{{cite web|url=http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&extra=e&year=2011&month=10&day=10&vol=15&no=43&gn=61&header=1&part=1&df=1&nt=gn&newfile=1&acurrentpage=12&agree=1&gaz_type=mg |title=投票站及點票站名單 |publisher=[[香港區議會|香港區議會]] |accessdate=2011-11-12}}</ref>，呂東孩再次參選，對手是民主黨成員[[韓家銘|韓家銘]]<ref>{{zh-hant}}{{cite web|url=http://www.elections.gov.hk/dc2011/pdf/J23_01.pdf |title=候選人簡介 |publisher=[[香港區議會|香港區議會]] |accessdate=2011-11-12}}</ref>。最終，呂東孩以3,229票大比數擊敗取得779票的韓家銘而勝出<ref>{{zh-hant}}{{cite web|url=http://www.elections.gov.hk/dc2011/chi/results_kwun_tong.html |title=2011年區議會選舉 - 選舉結果(觀塘) |publisher=[[香港區議會|香港區議會]] |accessdate=2011-11-12}}</ref>。

[[2015年區議會選舉|2015年區議會選舉]]，當時人口估算約20,481人<ref>{{zh-hant}}{{cite web|url=http://www.elections.gov.hk/dc2015/chi/summaryj.html |title=觀塘區 |publisher=[[香港區議會|香港區議會]] |accessdate=2015-10-14}}</ref>，其中有9,689位是選民<ref>{{zh-hant}}{{cite web|url=http://www.elections.gov.hk/dc2015/chi/ebriefsj.html |title=觀塘區 |publisher=[[香港區議會|香港區議會]] |accessdate=2015-10-14}}</ref>。呂東孩第二次自動當選。
