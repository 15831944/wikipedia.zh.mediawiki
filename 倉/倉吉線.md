'''倉吉線'''（{{jpn|j=倉吉線|hg=くらよしせん|rm=Kurayoshi sen|f=y}}）是一條連接[[鳥取縣|鳥取縣]][[倉吉市|倉吉市]]的[[倉吉站|倉吉站]]與[[東伯郡|東伯郡]][[關金町|關金町]]（現時為倉吉市的一部分）[[山守站|山守站]]的[[鐵路線|鐵路線]]（[[地方交通線|地方交通線]]），由[[日本國有鐵道|日本國有鐵道]]（國鉄）運營。經過[[日本國有鐵道經營再建促進特別措置法|國鐵再建法]]的施行，在1981年被定為[[特定地方交通線|第1次特定地方交通線]]，列入停用鐵道線表中。最後在1985年4月1日全線[[停運|停運]]。

== 路線概要（停運前） ==
* 路線距離（營運公里）：20.0km
* [[軌距|軌距]]：1,067mm
* 站數：9驛（包括起終點站）
* [[雙線鐵路|雙線]]區間：沒有（全線[[單線鐵路|單線]]）
* 電化區間：沒有（全線非電氣化）
* [[閉塞_(鐵路)|閉塞方式]]：片劑閉塞式

== 運行狀態（停運前） ==
[[File:JNR_Kurayoshi_Line_Timetable_at_Kuyayoshi_Sta.jpg|thumb]]

全線列車都是普通列車，大部分列車都是行走全線，其餘列車只會由[[打吹站|打吹站]]、[[西倉吉站|西倉吉站]]、[[關金站|關金站]]出發或到達的區間列車。此外，[[泰久寺站|泰久寺]]與[[山守站|山守]]兩站是不可列車交會，即是關金站以南的區間只可一架列車運行，由往山守的列車班次會使用[[柴油動車組|柴油動車組]]，而其他班次會使用客卡列車。

1984年2月的時間表修正中，此路線還會使用混合列車<ref name="rm13-17">『鐵路雜誌』通巻13號 p17</ref>。此外在1980年，全日上下行共有27班列車運行，行走班次低於20km/h的列車班次共9班<ref name="korono1-83">『コロタン文庫 鐵道No.1全百科』p83（1981年・小學館）</ref>，被評為「全日本最慢的鐵路區間」<ref name="korono1-83"/>，因此，有書籍是記載了此路線的普通列車<ref name="korono1-83"/>。當中有介紹「424列車」，由西倉吉至倉吉之間的6.8公里路段需時27分鐘<ref>日本交通公社『國鐵監修 交通公社之時刻表 1981年8月號』196頁，424列車，在西倉吉10:02出發，在打吹10:16出發，上灘10:21出發，倉吉10:29到達。</ref>，此列車的平均車速為15.1km/h，也被評為「慢過一名馬拉松運動員的列車」的稱呼<ref name="korono1-83"/>。原因在於此路線等級低（簡易線），使用30kg鋼軌，後來亦未施作高規格化工程，致使軸重大的[[國鐵DE10形內燃機車|國鐵DE10形內燃機車]]不能高速行駛。

== 歷史 ==
路線原本是由山陰本線上井站（現時為倉吉站）至倉吉町（現時為倉吉市）的市內的倉吉[[輕便線|輕便線]]，在1912年啟用。

在1941年，位於[[關金溫泉|關金溫泉]]附近的關金站啟用，在1958年，山守站啟用。之後的修正[[鐵道敷設法|鐵道敷設法]]，當中的會計畫由山守站向南伸長，經過中國山區到達[[姫新線|姫新線]][[中國勝山站|中國勝山站]]，並將會名為[[南勝線|南勝線]]，但由於將會興建的沿路上有許多人會使用私家車與巴士，在1985年3月31日，「再見倉吉線」的記念牌附在倉吉線最後的臨時列車（也是最後的一班列車），在當時22:07到達山守站，長達72年的鐵路線終於停運。

在停運後，便由[[日本交通_(鳥取)|日本交通]]（倉吉站 - 打吹 - 西倉吉 - [[關金溫泉|關金溫泉]] - 山守之間）、[[日之丸自動車|日之丸自動車]]（倉吉站 - 打吹 - 西倉吉之間）間[[中鐵巴士|中鐵巴士]]（1986年為湯原巴士的分社，現時為[[中鐵美作巴士|中鐵美作巴士]]、倉吉站 - 打吹 - 關金溫泉 - [[蒜山高原|蒜山高原]]之間）的三間巴士公司替代鐵路服務，在2007年11月1日，中鐵美作巴士的倉吉站 - 打吹 - 關金溫泉 - 蒜山高原之間的巴士路線由於客量少而取消。

=== 年表 ===
* 1912年（[[明治|明治]]45年）6月1日：倉吉輕便線 上井與倉吉之間（2.6英里，大約為4.18公里）啟用。倉吉站（第二代）啟用。
* 1912年（[[大正|大正]]元年）10月1日：新設上灘站。
* 1922年（大正11年）9月2日：倉吉輕便線改名為倉吉線。
* 1930年（[[昭和|昭和]]5年）4月1日：營業距離的單位變為公制（2.6英里變為4.2公里）。
* 1941年（昭和16年）5月17日：倉吉至關金之間 (11.0公里) 開通。新設西倉吉站、小鴨站、上小鴨站、關金站。
* 1958年（昭和33年）12月20日：關金至山守之間 (4.8公里) 開業（全線20公里開通）。新設泰久寺站、山守站。
* 1972年（昭和47年）1月10日：倉吉站（第二代）改名為打吹站。
* 1972年（昭和47年）2月14日：上井站改名為倉吉站（第三代）。
* 1974年（昭和49年）10月1日：西倉吉至關金之間的貨物運輸停止。
* 1981年（昭和56年）9月18日：指定為第1次特定地方交通線。列入停用鐵道線表中。
* 1983年（昭和58年）12月31日：倉吉至西倉吉之間的貨物運輸停止。
* 1985年（昭和60年）4月1日：倉吉至山守之間 (20.0公里)的倉吉線停運。

== 現狀 ==
此鐵路線中，一些路段變為街道（西倉吉與上小鴨之間為[[鳥取縣道501號倉吉東郷汽車線|鳥取縣道501號倉吉東郷汽車線]]（伯耆自動車道）），上小鴨站附近變為[[國道313號|國道313號]]。經過20多年後，上小鴨站與山守站之間的路軌與道碴還存在。泰久寺站月台還是存在的，月台上還有車站名稱牌，以及部分路軌<ref>[http://www1.harenet.ne.jp/~xb-35/kurayosisen/kurayosisen_index.htm 倉吉線遺跡] {{webarchive|url=https://archive.is/20120805192809/http://www1.harenet.ne.jp/~xb-35/kurayosisen/kurayosisen_index.htm |date=2012-08-05 }}</ref>。西倉吉站中，東側的島式月台還在使用，用為候車區與設置公共廁所，而西側的島式月台變成公園的一部分。泰久寺站與山守站之間的[[山守隧道|山守隧道]]由於有崩塌的危險，兩邊的隧道口已經填滿，不可進入。此外，打吹站的位置設置了[[倉吉線鐵道記念館|倉吉線鐵道記念館]]，館內設有[[國鐵C11形蒸氣機車|國鐵C11形蒸氣機車]]75號。
