'''倉木麻衣'''（{{bd|1982年|10月28日|catIdx=Kuraki, Mai}}，本名青野真衣），[[日本|日本]]女[[歌手|歌手]]，生於日本[[千葉縣|千葉縣]][[船橋市|船橋市]]，現居[[東京|東京]]。现隶属[[NORTHERN_MUSIC|NORTHERN MUSIC]]公司LOOP事務所（2007年－）。她的父亲是电影导演、實業家{{tsl|ja|山前五十洋}}。祖父是诗人{{tsl|ja|山前実治|山前实治}}。2007年之前曾隶属[[GIZA_Studio|GIZA Studio]]。2001年4月進入[[立命館大學|立命館大學]]產業社會学院人文科学系就讀，並於2005年3月畢業。截止2014年12月，她举办了超過300次的个人演唱会。CD总销量突破1600万枚（日本公信榜[[ORICON|ORICON]]调查，海外销量以及与松本孝弘合作的单曲未算入其中）。出道以来，目前为止，全部41张单曲均进入ORICON TOP10，是ORICON史上唯一一位享有此纪录的女性独唱艺人（不含团体）。2017年7月，因為21次為「[[名偵探柯南|名偵探柯南]]」系列作品獻唱的關係，獲得了「同一歌手為同一動畫演唱主題曲次數最多」的金氏世界紀錄<ref>{{Cite web |url = http://ent.sina.com.cn/y/yrihan/2017-07-25/doc-ifyihrit1389059.shtml |title = 仓木麻衣为柯南献唱21首主题曲 破吉尼斯世界纪录 |author =  |date = 2017-07-25 |publisher = 新浪 |language = zh-cn |accessdate = 2017-07-25 }}</ref>。

== 音樂特徵 ==
=== 作曲者 ===
仓木麻衣的作品不全是由她本人作曲。其中相当一部分由其他作曲家的提供。从出道至2004年，作品的曲多由[[大野愛果|大野愛果]]、[[德永曉人|德永曉人]]、YOKO Black. Stone等3位作曲家提供。特別是大野愛果，她提供的作品大約佔了所有作品的一半。編曲則多由作曲者自己擔任，或[[波士頓|波士頓]]的「Cybersound」負責。2005年以後則多由大野愛果、德永曉人為中心的GIZA所屬作曲者提供。

=== 作詞 ===
除了一部份的翻唱或合唱歌之外，仓木麻衣自己创作了她大部分歌曲的歌词。

== 音樂事業 ==
=== 童年與早期生活 ===
: 在小學高年級時看了[[麥可·傑克森|麥可·傑克森]]所主演的電影開始，到後來看[[惠特妮·休斯頓|惠特妮·休斯頓]]的MV、LIVE，感到“我也想成為這樣的歌手”，於是對西洋音樂產生極大的興趣。

: 當時倉木正在練習鋼琴，隱隱約約對於音樂事業也有了初步的考慮。在鋼琴課中，一邊看著音符一邊哼唱著歌曲時，被鋼琴課老師聽到，老師說“與鋼琴相比，你還是向歌唱方面發展吧”，以這句話為契機，倉木開始第一次考慮開始自己的歌唱事業。

: 在成為中學生之後，倉木開始接觸洋樂，特別是女性歌手的作品。而最早接觸的洋樂作品，則是Michael Jackson的音樂錄影帶。之後Mariah Carey（[[瑪麗亞·凱莉|瑪麗亞·凱莉]]）、Whitney Houston（[[惠特妮·休斯頓|惠特妮·休斯頓]]）、Lauryn Hill（勞倫·希爾）的作品相繼接觸，從此開始認真考慮成為像她們那樣的歌手。

: 此時，與BEING的長戶大幸相識。長戶在聽過倉木的demo tape之後，建議倉木在成為高中生之後正式出道，而在此之前，則要加強英語學習以及積累其他各方面的經驗。在成為高中生之後，由長戶介紹，與GIZA studio的工作人員相識，在試聽過翻唱至Lauryn Hill的《To Zion》之後（此後有在FC EVENT展示），決定正式出道。

=== 1999年：正式出道 ===
: 1999年10月，當時還只有16歲的她在美國出道，以Mai-K名義推出首張英文《[[Baby_I_Like|Baby I Like]]》，由美國的獨立廠牌Bip·Record發行，以外國輸入盤的形式在日本發售，旋即被抢购一空。其後於12月8日發行以倉木麻衣名義發行的首張日文單曲《[[Love,_Day_After_Tomorrow|Love, Day After Tomorrow]]》，正式於日本出道。這張單曲初動銷量只得約3萬張，可是其後每週銷量一直上升，並於2000年3月正式突破百萬銷量，合计138万张。成為2000年最耀眼的新人歌手。

=== 2000年－2007年：GIZA Studio時期 ===
: 首張單曲達成百萬銷售後，倉木隨即推出《[[Stay_by_my_side|Stay by my side]]》與《[[Secret_of_my_heart|Secret of my heart]]》兩張單曲，皆獲得近百萬的高銷量。其中《Secret of my heart》更是首次跟[[動畫|動畫]]《[[名偵探柯南_(動畫)|名偵探柯南]]》合作，至今倉木仍有不少歌曲成為名偵探柯南的使用歌。

: 2000年，她於6月7日發行了專輯的先行單曲《[[NEVER_GONNA_GIVE_YOU_UP|NEVER GONNA GIVE YOU UP]]》。6月28日發行首張專輯《[[delicious_way|delicious way]]》，專輯驚人的初動銷量達211萬，打破由去年[[宇多田光|宇多田光]]的《[[First_Love_(宇多田光專輯)|First Love]]》所創下的最高出道專輯初動銷量。專輯累積銷量達353萬，力壓當年[[濱崎步|濱崎步]]的《[[Duty|Duty]]》成為該年專輯銷量冠軍。获得了2000年度细碟销量冠军、大碟销量亚军、最佳作词、最出色新人奖和最高唱片收入亚军等五个奖项。而《delicious way》至今仍是日本歷代銷量第9專輯。在首張專輯獲得巨大成功後，她便開始轉換曲風，於2000年年底發行《[[Simply_Wonderful|Simply Wonderful]]》跟《[[Reach_for_the_sky|Reach for the sky]]》兩張單曲，皆獲得30萬至40萬的高銷量。數張高銷量的唱片為她帶來約160億日元的總銷售額，成為該年總銷售額歌手亞軍。

: 2001年，她先後推出三張單曲，包括《[[冰冷的海／Start_in_my_life|冰冷的海／Start in my life]]》、《[[Stand_Up|Stand Up]]》及《[[always_(倉木麻衣單曲)|always]]》，皆獲得數十萬的穩定銷量。而《Start in my life》為動畫《名偵探柯南》的片尾曲，該單曲更在首周以二十萬銷量獲得公信榜單曲榜亞軍<ref>{{Cite web |url = http://www.musictvprogram.com/corner-ranking-2001-2-19.html |title = 2001年2月19日付 シングルチャート TOP10 |author = オリコンさん |language = ja |accessdate = 2017-07-25 |archive-url = https://web.archive.org/web/20160410074420/http://www.musictvprogram.com/corner-ranking-2001-2-19.html |archive-date = 2016年4月10日 |dead-url = yes }}</ref>。4月初，她於立命館宇治高等學校畢業，並入讀立命館大學産業社會學部人間文化課程。7月4日，推出第二張個人專輯《[[Perfect_Crime|Perfect Crime]]》，銷量旋即突破百萬，达到131万张。8月19日至9月4日，舉辦了與爽健美茶合作的首次個人演唱會「爽健美茶 Natural Breeze 2001 happy live」。總結全年的唱片總銷量達二百多萬，而總銷售額達60億日元，為當年女性個人歌手第四名。

: 在完成第二張專輯後，她便於2001年至2002年間陸續推出了《[[Can't_forget_your_love／PERFECT_CRIME_-Single_Edit-|Can't forget your love／PERFECT CRIME -Single Edit-]]》、《[[Winter_Bells|Winter Bells]]》、《[[Feel_fine!|Feel fine!]]》與《[[Like_a_star_in_the_night|Like a star in the night]]》等單曲，皆獲得不俗銷量。2002年1月10日，她以Mai-K的名義於美國推出了首張英文專輯《[[Secret_of_my_heart_(專輯)|Secret of my heart]]》。當中收錄了單曲《Baby I Like》及日文專輯《delicious way》內多曲的英文版本。其後便回到日本活動，舉行了首場全國巡迴演唱會「Mai Kuraki Loving You…Tour 2002」。而從這年她開始較頻繁於電視上亮相，她首次接拍了[[資生堂|資生堂]]的SEA BREEZE廣告，並出演了「2002 FIFA [[世界盃足球賽|WORLD CUP]]官方演唱會」，為首次於電視上演出。10月23日，她推出了以童話故事為主題第三張個人專輯《[[FAIRY_TALE|FAIRY TALE]]》，銷量達73萬。其後開始了她的全國巡迴演唱會「Mai Kuraki FAIRY TALE TOUR02-03」，並當中度過了她的20歲生日。

: 2002年底至2003年初，她陸續推出了單曲《[[Make_my_day_(倉木麻衣單曲)|Make my day]]》、《[[Time_after_time_～在落花紛飛的街道上～|Time after time ～在落花紛飛的街道上～]]》、《[[Kiss|Kiss]]》與《[[風的啦啦啦|風的啦啦啦]]》。2月，於東京、大阪等地舉行了4天的GIZA studio情人節演唱會，與同事務所的[[愛内里菜|愛内里菜]]、[[GARNET_CROW|GARNET CROW]]等歌手一同出演，觀眾皆獲得[[巧克力|巧克力]]與卡片作禮物。而在[[Windows_Media_Player|Windows Media Player]]9系列更同時發表了「Digital Media Night」的情人節現場表演，可於Windows Media Player9系列上點播。7月9日，推出了第四張個人專輯《[[If_I_Believe|If I Believe]]》，當中收錄跟[[新加坡|新加坡]]歌手[[孫燕姿|孫燕姿]]的合作歌曲《Tonight, I feel close to you》。發售後連續第四次獲得專輯冠軍。專輯發售後，倉木在創刊55周年的「[[FamilyMart|FamilyMart]]Dream Match [[橫濱水手|橫濱F．水手]]對[[雷吉納足球俱樂部|雷吉納]]」中演唱了專輯中的《SAME》。10月8日，推出了與[[B'z|B'z]]的松本孝弘的合作單曲《Imitation gold》，登上該週銷量榜冠軍。12月31日，首次出演「[[NHK紅白歌合戰|第54回NHK紅白歌合戰]]」，演唱《Stay by my side》，為翌日發售的首張精選輯作宣傳。

: 2004，她於1月1日推出首張精選輯《[[Wish_You_The_Best|Wish You The Best]]》。專輯連續四週高據銷量榜冠軍，並錄得近百萬的高銷量，是繼出道專輯《delicious way》後再次進入年榜頭十位，更獲[[日本唱片協會|日本唱片協會]]認証為百萬專輯<ref>{{Cite web |url = http://www.riaj.or.jp/f/data/cert/gd.html |title = 2004年1月1日から2004年1月31日の期間中に発売 |author = RIAJ |language = ja |accessdate = 2017-07-25 }}</ref>。4月至7月，開始舉行全國巡迴演唱會「Mai Kuraki 2004 Live Tour Wish You The Best〜Grow, Step by Step〜」，而在7月4日的[[仙台|仙台]]Sun Plaza演唱會則完成了100場演唱會<ref>{{Cite web |url = http://news.livedoor.com/article/detail/2321952/ |title = 倉木麻衣、ツアーブログをスタート |author = livedoor |date = 2006-08-15 |language = ja |accessdate = 2017-07-25 }}</ref>。在演唱會期間，更於[[活力門|活力門]]創立以「Mai-K Diary」為名的[[博客|博客]]，主要是倉木的個人日記及演唱會報告。5月19日，推出全年唯一的單曲《[[通往明日的橋樑|通往明日的橋樑]]》，更在年底的[[NHK紅白歌合戰|第55回NHK紅白歌合戰]]中獻唱此曲<ref>{{Cite web |url = http://www1.nhk.or.jp/kouhaku/history/history_55.html |title = 第55回NHK紅白歌合戰 |author = NHK |date = 2006-08-15 |language = ja |accessdate = 2017-07-25}}</ref>。
