'''蚌埠仁和集机场'''（{{airport codes|BFU|ZSBB}}），位于[[中华人民共和国|中华人民共和国]][[安徽省|安徽省]][[蚌埠市|蚌埠市]][[禹会区|禹会区]]迎宾大道西侧，是一座空军二级永备机场。<ref>[http://epaper.bbnews.cn/shtml/bbrb/20150104/305678.shtml 城市中心为何有个军用机场？ ——— 蚌埠老机场迁建路 蚌埠日报]</ref>

== 历史 ==
蚌埠机场始建于1926年，系[[中华民国国军|中华民国国军]]遗留下来的土质机场，位于目前[[蚌山区|蚌山区]]、[[龙子湖区|龙子湖区]]交界处。1928年改建为军用机场，1953年、1961年分别扩建为三级、二级军用机场。由于该机场处于蚌埠市区中部，严重制约了蚌埠市的城市建设和发展，也对部队的正常训练和飞行安全以及人民群众的生命财产安全带来潜在隐患，1989年全国人大七届二次会议上提出搬迁蚌埠机场的建议。1998年5月，新机场获准使用土地4030亩，并于同年12月开工建设。新建的军民合用机场位于距[[蚌埠市|蚌埠市]][[禹会区|禹会区]][[秦集镇|秦集镇]]仁和集。迁建的机场按军用二级和民用4C设计于2002年8月竣工验收后交付部队使用。<ref>{{Cite web |url =http://data.carnoc.com/corp/airport/bfu.html |title=蚌埠机场 |accessdate=2015-05-28 |date=2018-04-21|publisher=民航资源网 }}</ref>

== 机场设施 ==
飞行跑道共一条，长2600米、宽40米，[[机场飞行区等级|机场飞行区等级]]为4C。<ref>{{GCM|BFU|Bengbu Airport}}</ref>
