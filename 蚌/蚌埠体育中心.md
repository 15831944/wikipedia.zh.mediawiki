'''蚌埠体育中心'''建于2016年，位于[[安徽省|安徽省]][[蚌埠市|蚌埠市]][[蚌山区|蚌山区]]城南新区，与[[湖上升明月|湖上升明月]]景区相望，为综合性甲级体育中心。体育中心占地面积546[[亩|亩]]，投资概算约11.6亿元，包含体育场、体育馆、多功能综合馆、体育运动学校和景观塔等部分组成<ref>{{cite web |author1=颜婕 |title=五个体育场馆今年陆续开建 |url=http://www.bbnews.cn/xw/jrtj/2016/03/520187.shtml |website=蚌埠新闻网 |accessdate=2018-07-24 |date=2016-03-30}}</ref>，2018年建成投入使用。蚌埠体育中心以“龙行天下、龙腾戏珠”为设计理念，建筑主体呈龙形，外立面为红底白鳞。

==建筑构造==
*'''体育场'''：主体育场可容纳观众30000人，座椅为红白相间的“满天星”设计，主要举办足球和田径赛事。
*'''体育馆'''：建筑面积36400平方米，可容纳观众8000人，主要举行[[篮球|篮球]]、[[排球|排球]]等球类比赛，也可用于演艺活动。场内活动桌椅可以收起，形成多个篮球、羽毛球、乒乓球、溜冰场场地<ref>{{cite web |author1=蚌埠发布 |title=蚌埠体育中心要交付使用啦，计划在…… |url=http://wemedia.ifeng.com/64298268/wemedia.shtml |publisher=大风号 |accessdate=2018-07-27 |date=2018-06-09}}</ref>。
*'''多功能馆'''：建筑面积41733平方米，主要举办[[射击|射击]]、[[击剑|击剑]]、国际式[[摔跤|摔跤]]、[[柔道|柔道]]、[[拳击|拳击]]、[[跆拳道|跆拳道]]等比赛项目。
*'''景观塔'''：位于体育场和体育馆中心位置，高约70米。<ref>{{cite web |author1=苗成韬 |title=蚌埠体育中心将竣工 两条巨龙呼之欲出 |url=http://bb.anhuinews.com/system/2018/04/26/007856183.shtml|website=中安在线 |accessdate=2018-07-24 |date=2018-04-26}}</ref>
