'''蚌埠第二中学'''，简称'''蚌埠二中'''，是[[安徽省|安徽省]][[蚌埠市|蚌埠市]]的一所[[高级中学|高级中学]]，为安徽省示范高中。

== 校史 ==
蚌埠二中始建于1928年。1978年，被列入“安徽省重点中学”。2000年，被批准为“安徽省示范性普通高中”。2007年，学校被国家教育部和人事部评选为全国教育系统先进集体。<ref name=jj>{{cite web|url=http://www.bbez.com/detail-619|title=学校概况|publisher=蚌埠二中|accessdate=2018-07-03}}</ref>

== 现状 ==
学校现有49个教学班，在校学生近3000名。教职工159人，其中高级教师81人，特级教师14人。<ref name=jj/>

== 知名校友 ==
*[[蔡其华|蔡其华]]：曾任水利部副部长、水利部[[长江水利委员会|长江水利委员会]]主任和安徽省水利厅厅长
*[[杨建兴|杨建兴]]：曾任铁道部建设司司长
*[[王晓淮|王晓淮]]：曾任蚌埠市政协主席
*[[谈骏生|谈骏生]]：曾任南京医科大学第二附属医院院长
