《'''沁园春·长沙'''》，是[[毛泽东诗词|毛泽东诗词]]，作于1925年，最早发表在《[[诗刊|诗刊]]》1957年第一期，同时刊载的还有《[[七律·长征|七律·长征]]》《[[沁园春·雪|沁园春·雪]]》等17首。
[[File:Qinyuanchun_Changsha.JPG|thumb]]手书《沁园春·长沙》，摄于[[长沙|长沙]][[橘子洲|橘子洲]]]]
== 创作背景 ==
1925年，[[国民大革命|国民大革命]]已经开始，同年春天，毛泽东从[[上海市|上海市]]回到[[湖南省|湖南省]]，组织了20个多个农会，開始了革命事业，秋季，毛泽东从湖南省到[[广东省|广东省]][[广州市|广州市]]主持农民运动讲习所，在长沙市的时候，他重新游历了[[橘子洲头|橘子洲头]]，看到深秋时节的橘子洲头的景色，他写了这首词。<ref name=" duck" >{{cite book |author= 张岚 |title=《高中文言文译著及赏析》|year=2005年 |publisher=长春出版社 |location=长春市 |pages=1到2页 |isbn=7-80604-287-3 |language = zh-cn}}</ref>

== 诗词赏析 ==
全词上阕写景，下阕抒情。上阕从[[岳麓山|岳麓山]]、[[湘江|湘江]]、天空、水底选取实景，远近结合，动静映照，用文字描绘了一幅湘江秋景图，最后一句诗表现毛泽东对主宰国家命运，掌握民族的前途的期望。

下阕通过对战友和战斗生活的回忆和追述，以及不惧怕当时的大军阀敢于反抗，最后三句用象征的手法表现了在革命运动中也可以激流勇进完成革命主宰中华大地。<ref name="duck" />

== 出版发表 ==
*1957年，《[[诗刊|诗刊]]》第一期刊载了毛泽东的18首旧体诗词，该词为其中一首。
*1958年，由叶君健、于宝榘翻译的上述18首词的英译版在《中国文学》第三期发表。
