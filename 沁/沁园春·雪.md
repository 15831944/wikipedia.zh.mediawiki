《'''沁园春·雪'''》为1936年[[毛泽东|毛泽东]]在[[陕西|陕西]]时所作，1945年10月，毛泽东從延安赴[[重庆谈判|重庆谈判]]，将诗作抄录送给诗人[[柳亚子|柳亚子]]，随后刊登在重庆各大报纸，广为流传。<ref name="lion " />

== 历史背景 ==
1935年12月25日，中国共产党在[[瓦窑堡|瓦窑堡]]召开了[[瓦窑堡会议|瓦窑堡会议]]，制定了抗日民族统一战线。1936年2月，毛泽东带领红军抗日先锋队从陕西北部出发，准备东渡[[黄河|黄河]]，开赴[[河北|河北]]同日本帝国主义作战。在陕西北部的[[清涧县|清涧县]][[高杰村|高杰村]]袁家沟，恰好落了一场大雪，毛泽东被周围的壮阔雪景深深感染，于是创作了这首气势磅礴的词章。<ref name="lion " /><ref name="duck " >{{cite web |url=http://www.chinanews.com/cul/2011/12-26/3560396.shtml |title=读《沁园春·雪》:感受一般诗人难以达到的境界 |accessdate=2011年12月26日  |publisher=中国新闻网 |work= |language =zh-cn}}</ref>

1945年8月25日，中共中央发表了《对日时局的宣言》，次日毛泽东起草了《中共中央关于同国民党进行和平谈判的通知》。8月28日，毛泽东、[[周恩来|周恩来]]、[[王若飞|王若飞]]等乘坐飞机抵达重庆，同中国国民党进行了43天的长时间谈判。毛泽东在重庆的时间里，会见了诗人[[柳亚子|柳亚子]]，柳亚子多次写诗呈送并且索要毛泽东的诗作，毛泽东在红岩村用第十八集团军重庆办事处便笺手书一份《沁园春·雪》，又在纪念册上手书了一份并且送给柳亚子，柳亚子将它发表在1945年11月14日的《[[新华日报|新华日报]]》，随后《[[新民报|新民报]]》、《[[大公报|大公报]]》相互转载。该词发表后，和作50余首，评论20余篇。<ref name="lion " /><ref name="duck " />

== 鉴赏 ==
全词一共114个字，上阕赞颂了中华大地的锦绣山河，下阕讲述了中国历史的前进进程。<ref name="duck " />

== 评价 ==
毛泽东对英文秘书[[林克_(1923年出生)|林克]]说：“《沁园春·雪》这首词是反封建的，‘惜秦皇汉武，略输文采；唐宗宋祖，稍逊风骚’，是从一个侧面来批判[[封建主义|封建主义]]制度的。”<ref>{{cite web |url= http://news.ifeng.com/history/zhongguojindaishi/detail_2011_11/23/10852224_0.shtml|title=毛泽东谈《沁园春·雪》：这首词是反封建的 |accessdate=2011年11月23日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

[[陈布雷|陈布雷]]：“气势磅礴、气吞山河，可称盖世之精品。”<ref name="lion " >{{cite web |url= http://news.ifeng.com/history/1/midang/200809/0920_2664_794985.shtml|title=毛泽东《沁园春-雪》发表引轰动 蒋介石恼火 |accessdate=2008年9月20日 |publisher=凤凰网 |work= |language =zh-cn}}</ref><ref name=" cat" />

[[蒋中正|蒋中正]]对陈布雷说：“我看他毛泽东野心勃勃，想当帝王称王称霸，想复古，想倒退。你（陈布雷）要赶快组织一批人，写文章批判他。”<ref name="lion " /><ref name=" cat" />

柳亚子：“中国有词以来第一手，虽[[苏轼|苏]]、[[辛弃疾|辛]]犹未能抗手，况余子乎？”<ref name=" cat" >{{cite web |url=http://news.sina.com.cn/c/2005-10-27/10168131373.shtml |title=1945年《沁园春·雪》发表以后 |accessdate=2005年10月27日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

[[倪匡|倪匡]]小說《在數難逃》，[[衛斯理|衛斯理]]批判毛澤東：『我道：「這最高領袖，是一個極端自負的人，氣概之高，舉世無雙，連唐宗漢武成吉思汗，都不放在他的眼中，和他自己相比，都被他比了下去——有著如此心態的一個人，怎會把別人的命數收放在心中。這類心態的人，要成霸業，千萬人的性命，在他看來，不過是霸業一塊基石。他如何會把他人的命數，放在心中！」』
