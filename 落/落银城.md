{{Vgname|落银城|fr=Silverfall}}是一款由[[基督山游戏|基督山游戏]]开发的一款Windows平台的[[砍杀游戏|砍杀类]][[电子角色扮演游戏|角色扮演游戏]]。本作后被移植至PSP平台。2008年2月在Windows平台推出本作的[[资料片|资料片]]《落银城：大地觉醒》（Silverfall: Earth Awakening），同年3月推出包含本体和资料片的“落银城：黄金版”。

落银城与[[暗黑破坏神|暗黑破坏神]]、[[暗黑破坏神II|暗黑破坏神II]]有着很多相似点<ref>{{Cite web |url = http://www.gamersky.com/news/201109/180241_2.shtml |title = 精彩不落单！细数那些与“暗黑”有关的追随者 |author =  |date = 2011-09-14 |publisher = 游民星空 |language = zh-cn |accessdate = 2018-02-18 }}</ref>。曾被人称为单机版的[[魔兽世界|魔兽世界]]<ref name="QQ" />。

== 系统 ==
=== 角色 ===
游戏开始时，玩家可以在人类（Human）、精灵（Elf）、洞穴巨人（Troll）、哥布林（Goblin）这四种种族中选择一个作为自己[[玩家角色|角色]]的种族，并可选择包括战士、弓箭手、魔法师等职业在内的任一种职业。本作支持可选[[玩家角色|角色]]性别。<ref name="sina">{{Cite web |url = http://games.sina.com.cn/j/n/2008-05-14/1819248846.shtml |title = 北京娱乐通即将推出ARPG大作《落银城》 |author =  |date = 2008-05-14 |publisher = 新浪 |language = zh-cn |accessdate = 2018-02-16 }}</ref>
