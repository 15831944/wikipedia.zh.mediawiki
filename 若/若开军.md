'''若开军'''（{{lang-my|ရက္ခိုင့်တပ်မတော်}}，{{lang-en|Arakan Army}}，[[縮寫|縮寫]] '''AA'''），是[[缅甸|缅甸]]的一支以[[若开人|若开人]]为主的少数民族地方武装部队，该武装成立于2009年4月10日，<ref name="atimes">{{cite news|last1=Mathieson|first1=David Scott|title=Shadowy rebels extend Myanmar’s wars|url=http://www.atimes.com/article/shadowy-rebels-extend-myanmars-wars/|accessdate=2017-06-13|work=Asia Times|date=2017-06-11}}</ref>为[[若开民族联盟|若开民族联盟]]（ULA）的武装翼，目前由[[總司令|總司令]][[通米亚良|通米亚良]]所领导。若开军与[[若开民族军|若开民族军]]、[[若开解放军|若开解放军]]合称为“若开三军”。<ref>{{cite web|title=新闻背景：长缨在手-若开三军联盟|url=http://blog.sina.com.cn/s/blog_63ec24f50102veit.html|work=当红军的哥哥的博客|date=2015-01-13|access-date=2017-08-16}}</ref>

若开军参与了{{le|克钦冲突|Kachin conflict}}，与[[克钦独立军|克钦独立军]]（KIA）一起对抗[[缅甸国防军|缅甸国防军]]。大多数若开军的士兵都是在克钦独立军的军营内接受训练，然而，若开军在若开邦有额外的训练营。根据缅甸和平监测中心（Myanmar Peace Monitor）的说法，若开军2014年的兵力为1500多人，<ref name="MMPeaceMonitor" />包括驻扎在缅甸与孟加拉国边界若开邦的人员。<ref>{{cite web|title = Myanmar: New Front in an Old War|url = http://thediplomat.com/2015/07/myanmar-new-front-in-an-old-war/|accessdate = 2015-08-31|first = Richard Potter, The|last = Diplomat}}</ref><ref>{{cite web|url = http://mmpeacemonitor.org/stakeholders/armed-ethnic-groups|title = Far From Home, Arakan Rebels Fight on Kachin Frontline|publisher = Irrawaddy.org|date = |accessdate = 2014-05-05}}</ref><ref>{{Cite web|title = Arakan Army Calls for Calm After Bangladesh Border Clash|url = http://www.irrawaddy.org/burma/arakan-army-calls-for-calm-after-bangladesh-border-clash.html|accessdate = 2015-08-31}}</ref>《[[伊洛瓦底|伊洛瓦底]]<nowiki/>》声称，截至2015年9月，若开军有2500名士兵和1万名支持者。<ref>{{Cite web|title = ‘I Want to Stress That We Are Not the Enemy’|url = http://www.irrawaddy.org/interview/i-want-to-stress-that-we-are-not-the-enemy.html|accessdate = 2015-09-28}}</ref>

==历史==
若开军成立于2009年4月10日，政治翼为[[若开民族联盟|若开民族联盟]]，成立地点为被称作“临时总部”的克钦邦[[拉咱|拉咱]]。<ref>{{cite web|title = AA (Kachin Region)|url = http://www.mmpeacemonitor.org/stakeholders/myanmar-peace-center/153-aa|website = www.mmpeacemonitor.org|accessdate = 2015-09-28|last = Administrator}}</ref>

培训后，该集团计划返回若开邦为争取民族自决而战；然而，随着2011年6月克钦邦的战争爆发，他们无法返回。于是，他们便拿起武器支援克钦独立军对缅甸国防军的战斗。2014年，若开军在与孟加拉国接壤的边界附近的若开邦以及泰缅边境附近驻扎下来，队伍有所壮大，作战能力也有所提升。

2015年2月，若开军与[[緬甸民族民主同盟軍|緬甸民族民主同盟軍]]（MNDAA）、[[德昂民族解放军|德昂民族解放军]]一道参与了[[2015年果敢军事冲突|2015年果敢军事冲突]]，同缅甸国防军进行了战斗。<ref>{{Cite web|title = Ethnic allies join Kokang fight|url = http://www.mmtimes.com/index.php/national-news/13108-tnla-arakan-army-join-kokang-fight.html|website = www.mmtimes.com|accessdate = 2015-09-28|first = The Myanmar|last = Times}}</ref>这次武装冲突造成数百名政府军士兵死亡。

2015年4月，若开军和缅甸政府军在若开邦北部的[[皎道镇|皎道镇区]]和[[百力瓦镇区|百力瓦镇区]]爆发武装冲突。<ref>{{Cite web|title = Refugees From Ruined Village Say Myanmar Army Trapped Them|url = http://www.voanews.com/content/refugees-ruined-village-trapped-myanmar-army/2731123.html|accessdate = 2015-09-28}}</ref>2015年8月27日，若开军与孟加拉国边防部队在缅孟边界[[班多尔班县|班多尔班县]]{{le|坦奇|Thanchi Upazila|坦奇乡}}Boro Modak地区爆发了军事冲突。<ref>{{Cite web|title = Arakan Army Calls for Calm After Bangladesh Border Clash|url = http://www.irrawaddy.org/burma/arakan-army-calls-for-calm-after-bangladesh-border-clash.html|accessdate = 2015-09-09}}</ref>

2015年8月20日，若开军与[[孟加拉国步枪队|孟加拉国步枪队]]的一支巡逻队爆发了{{le|2015年孟加拉国 - 若开军边界冲突|2015 Bangladesh–Arakan Army border clash|冲突}}，起因是孟加拉国步枪队当日早些时候没收了若开军的10匹马。<ref>{{cite web|title=Arakan Army attack Border Guard Bangladesh patrol|url=http://mizzima.com/news-domestic/arakan-army-attack-border-guard-bangladesh-patrol}}</ref>

2015年12月，缅甸国防军与若开军在[[实兑|实兑]]以北60公里的[[皎道镇|皎道镇区]]和[[妙乌镇区|妙乌镇区]]交界地区进行了数日的战斗，若开军方面战斗伤亡情况未知。<ref>{{Cite web|title = Rakhine refugees await return after Arakan Army clashes|url = http://www.frontiermyanmar.net/en/news/rakhine-refugees-await-return-arakan-army-clashes|website = Frontier Myanmar|access-date = 2016-02-16|first = Mratt Kyaw|last = Thu}}</ref>包括1名指挥官在内的几名政府军士兵被狙击身亡，还有数名士兵受伤。<ref>{{Cite news|title = Myanmar army clashes with ethnic Rakhine rebels|url = https://www.theguardian.com/world/2016/jan/08/myanmar-army-clashes-ethnic-rakhine-rebels-killed|newspaper = The Guardian|date = 2016-01-08|access-date = 2016-02-16|issn = 0261-3077|language = en-GB|first = Oliver|last = Holmes}}</ref>

2016年2月，缅甸国防军与另一支反政府武装[[若开罗兴亚救世军|若开罗兴亚救世军]]在若开邦北部爆发了[[2016-2017年若開邦北部衝突|冲突]]，若开军随即发表声明，称行凶者（指[[若开罗兴亚救世军|若开罗兴亚救世军]]）为“野蛮的孟加拉穆斯林恐怖分子”，并将这次暴力事件称之为“一场由若开邦北部孟加拉[[伊斯兰原教旨主义|伊斯兰原教旨主义]]武装分子发起的狂暴行动。”<ref name="atimes" />

2017年4月10日，若开军举行了建军8周年的纪念仪式。<ref>{{cite web|title=同盟军发文祝贺若开军建军8周年|url=http://blog.sina.com.cn/s/blog_146e99be00102x53t.html|work=果敢资讯网|date=2017-04-13|access-date=2017-08-14}}</ref>

2017年4月16日中午，若开军与缅甸国防军在若开邦耶德岛和崩那岛两地两次交火，交火中双方均有伤亡，具体情况不详。<ref>{{cite web|title=果敢北部杏塘一线传激烈枪声 16日缅军与若开军两度交火|url=http://www.kokang123.com/thread-23613-1-1.html|work=果敢资讯网|date=2017-04-18|access-date=2017-08-14}}</ref>
