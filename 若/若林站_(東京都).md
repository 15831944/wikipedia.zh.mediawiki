'''若林站'''（{{jpn|j=若林駅|hg=わかばやしえき|rm=Wakabayashi eki|f=y}}）位於[[日本|日本]][[東京都|東京都]][[世田谷區|世田谷區]][[若林_(世田谷區)|若林]]四丁目，是[[東京急行電鐵|東京急行電鐵]][[世田谷線|世田谷線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''SG03'''。

== 歷史 ==
* 1925年（[[大正|大正]]14年）1月18日 - 開業。
* 1939年（[[昭和|昭和]]14年）10月16日 - 改稱'''玉電若林站'''。
* 1969年（昭和44年）5月11日 - 改稱'''若林站'''。
* 2001年（[[平成|平成]]13年）2月11日 - 開始進行月台提升工程。

== 車站構造 ==
對向[[側式月台|側式月台]]2面2線[[地面車站|地面車站]]。平日早上[[尖峰時段|尖峰時段]]的上行月台有職員駐守，負責收取車費。

上行月台近三軒茶屋入口曾經有商店營業。

== 若林踏切 ==
本站往三軒茶屋方向有個[[平交道|平交道]]，東急稱之為「西太子堂5號踏切」<ref>{{Cite web |title=踏切安全通行カルテ 「西太子堂5号」 |url=http://www.ktr.mlit.go.jp/honkyoku/road/fumikiri/13tokyo/369_nishitaishidou5.pdf |publisher=国土交通省関東地方整備局 |format=PDF |accessdate=2017-12-13}}</ref>，是唯一橫斷環七通的平交道。與一般鐵路平交道不同，此平交道沒有柵欄，而是用[[交通號誌|交通號誌]]阻斷車流。運作方式為，電車會先在平交道旁停下，透過軌道上的感應線路傳達指令給交通號誌切換為紅燈，車輛停止行進，接著行人號誌與列車號誌亮起，列車再通過平交道。

若是列車停留在感應區中一定時間以上，為了避免環七通堵塞，汽車交通號誌會改為閃紅燈。此時必須通報警察才能解除閃紅燈<ref>「日本路面電車カタログ2011-2012」p26-28 イカロス出版</ref>。

若林踏切在過去也是使用柵欄的一般平交道，但在環七通交通量逐漸增大的情況下，1966年（[[昭和|昭和]]41年）改為現行形式。
1972年（昭和47年）東急提出立體交差化計畫構想，除了[[東急池上線|東急池上線]][[戶越銀座站|戶越銀座站]] - [[旗之台站|旗之台站]]間（1989年（[[平成|平成]]元年）地下化）、[[東急東橫線|東急東橫線]][[多摩川站|多摩川園前站]] - [[新丸子站|新丸子站]]間（1986年（昭和61年）高架化），也提出將若林踏切在內的世田谷線西太子堂站 - 若林站區間交通立體化<ref>鉄道ピクトリアル 269号、p.42-46</ref>。
