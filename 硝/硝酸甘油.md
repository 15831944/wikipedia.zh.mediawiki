'''硝酸甘油'''（Nitroglycerin）(C<sub>3</sub>H<sub>5</sub>N<sub>3</sub>O<sub>9</sub>)，又稱硝酸-{}-甘油酯、三硝酸-{}-甘油酯、三硝酸丙三酯，是[[甘油|甘油]]的三[[硝酸酯|硝酸酯]]，是一种爆炸能力极强的[[炸药|炸药]]。1847年由[[都灵大学|都灵大学]]的化學家[[索布雷洛|索布雷洛]]發明。常有人誤解「硝酸甘油」是[[瑞典|瑞典]][[化學家|化學家]][[阿爾弗雷德·諾貝爾|阿爾弗雷德·諾貝爾]]發明的，事實上諾貝爾發明的是在1866年利用硝酸甘油發展高穩定性、防誤爆的[[矽藻土炸藥|矽藻土炸藥]]。

== 物理特性 ==
纯品无色透明。在13℃时冻结，[[沸点|沸点]]256℃，相对密度1.5931（20/4℃）。微溶于水。

== 化学特性 ==
本品具有一般硝基化合物的反应性，如硝基可被还原至亚硝基、氨基。

此外，本品极易由于受热、撞击而爆炸，使用时应注意防护。

== 制备方法 ==
由[[甘油|甘油]]和[[浓硝酸|浓硝酸]]：[[浓硫酸|浓硫酸]]为1：3体积比的混酸硝化制得。此实验具有爆炸危险性：硝酸甘油在室温下或由桌面高度掉落地面即可引爆。

[[file:Nitroglycerine_Destruction_V.1.svg|200px]]

== 应用 ==
诺贝尔发明了一种使硝酸甘油稳定的方法，至今已发展到数百种配方，但原理都是把硝酸甘油和挥发性低的次级炸药，各类其他配料，粘结剂，填充料等混合在一起，如[[硝化纤维|硝化纤维]]，[[矽藻土|矽藻土]]等等。

===醫療使用===
{{main|硝酸甘油 (藥物)}}
医药上用作[[血管扩张劑|血管扩张劑]]，其作用机理为扩张小动脉和小静脉，但扩张小静脉的作用超过小动脉。制成0.3%硝酸甘油片剂，舌下给药，吞服无效，作用迅速而短暂，效果极强。用于治疗[[冠狀動脈|冠狀動脈]]狹窄引起的急性[[心绞痛|心绞痛]]，如多次服用会降低药效，[[低血壓|血壓過低]]及有服用[[威而鋼|威而鋼]]等會使血管擴張的藥物時，可能引發[[昏倒|昏倒]]甚至[[休克|休克]]。且對於冠狀動脈堵塞無效，應盡速就醫治療<ref>[https://video.udn.com/news/576762 台南副市長盼耐絞寧列指示藥？專家驚訝：絕對不可！]</ref>。
