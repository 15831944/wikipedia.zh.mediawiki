{{ Refimprove|硝酸钠|date=2013年6月 }}
'''硝酸鈉'''為[[无机鹽|无机鹽]]的一種，化学式为NaNO<sub>3</sub>，白色固體粉末，又稱為'''智利硝石'''或祕魯硝石（較不常見）。世上最大的硝酸鈉礦位於[[智利|智利]][[阿塔卡馬沙漠|阿塔卡馬沙漠]]。

可由下列化學式合成:
: NaOH(aq) + HNO<sub>3</sub>(aq) → NaNO<sub>3</sub>(aq) + H<sub>2</sub>O(l)

硝酸钠加热至380℃分解产生[[亚硝酸钠|亚硝酸钠]]和[[氧气|氧气]] <ref>中学教师化学手册（1981 科学普及出版社）3.5无机化合物的物理常数.P<sub>154，411</sub> </ref><ref>R.C.Weast, Handbook of Chemistry and Physics,60thed.,1979-1980</ref>。

硝酸钠具有[[氧化性|氧化性]]，可与[[铅|铅]]共热反应产生亚硝酸钠和[[氧化铅|氧化铅]]<ref>中学教师化学手册（1981 科学普及出版社）6.2.20氮及其化合物的反应.P<sub>411</sub></ref>。还在常温下将[[氢碘酸|氢碘酸]]氧化成[[碘|碘]][[单质|单质]]并形成[[一氧化氮|一氧化氮]]：

:  2NaNO<sub>3</sub> + 8HI = 2NaI + 3I<sub>2</sub> + 2NO↑ + 4H<sub>2</sub>O <ref>中学教师化学手册（1981 科学普及出版社）6.2.20氮及其化合物的反应.P<sub>412</sub></ref>

硝酸钠[[溶液|溶液]]中引入[[氢离子|氢离子]]后会表现出[[硝酸|硝酸]]的特性：
:  4NaNO<sub>3</sub> + 4H<sub>2</sub>SO<sub>4</sub> →（加热）4NaHSO<sub>4</sub> + 4NO<sub>2</sub>↑ + O<sub>2</sub>↑ + 2H<sub>2</sub>O

:  NaNO<sub>3</sub>(aq) + 3FeCl<sub>2</sub> + 4HCl →（加热） NaCl+ 3FeCl<sub>3</sub> + NO↑ + 2H<sub>2</sub>O<ref>中学教师化学手册（1981 科学普及出版社）6.2.20氮及其化合物的反应.P<sub>412</sub></ref>

将硝酸钠与[[氧化钠|氧化钠]]置于[[坩埚|银坩埚]]中以700°C加熱，大约7天后，会生成白色晶体[[原硝酸钠|原硝酸钠]](Na<sub>3</sub>NO<sub>4</sub>)，它对[[水蒸气|水蒸气]]及[[二氧化碳|二氧化碳]]十分敏感。

硝酸鈉為{{le|醃製鹽|Curing salt}}的成分之一。
