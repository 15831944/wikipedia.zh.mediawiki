'''硝酸鉀'''是[[钾|钾]]的[[硝酸盐|硝酸盐]]，實驗式KNO<sub>3</sub>(硝酸鉀是離子化合物，並沒有分子，所以沒有分子量，只有式量)。外觀为透明無色或白色粉末，無味，比重（水=1）为2.11。在水中的溶解度为13 g/100mL（因溫度而異，溫度越高溶解度越高，在化學物質之中，硝酸鉀溶解度變化是相當明顯的）。

潮解性較硝酸鈉為低，有冷卻刺激鹽味。溶於水，稍溶於[[乙醇|乙醇]]。

== 用途 ==
硝酸钾俗称火硝或土硝。主要用于焰火、[[黑色火药|黑色火药]]、[[火柴|火柴]]、[[导火索|导火索]]、[[烛芯|烛芯]]、[[烟草|烟草]]、[[布朗管|彩电显像管]]、药物、化学试剂、[[催化剂|催化剂]]、陶瓷釉彩、[[玻璃|玻璃]]、复合肥料、及花卉、蔬菜、果树等经济作物的叶面喷施[[肥料|肥料]]等。對敏感牙齒有舒緩作用，所以有很多[[牙膏|牙膏]]也有硝酸钾。硝酸鉀也是[[天氣瓶|天氣瓶]]內的一種鹽類。
在医学上，硝酸钾用于治疗[[肾结石|肾结石]]，目前发现人类泌尿结石有26种成分，其中没有硝酸钾，因为硝酸钾非常活泼，不易形成结石，反而能够与其它结石中的钙结合并且排出体外。是治疗含钙结石的主要药物，为了防止硝酸钾在大肠杆菌作用下形成有致癌性的[[亚硝酸盐|亚硝酸盐]]，一般都与[[硫酸铝钾|硫酸铝钾]]一起服用。

== 可燃性危险特性 ==
高热放出氧气;遇有机物、还原剂、木炭、硫、磷等易燃物可燃;燃烧产生有毒氮氧化物烟雾。操作时应佩戴氯丁橡胶手套。<ref>[http://china.guidechem.com/15472/  硝酸钾可燃性—盖德化工网]</ref>
== 化学性质 ==
* 可参与氧化还原反应
: S + 2KNO<sub>3</sub>+ 3C → K<sub>2</sub>S + N<sub>2</sub>↑ + 3CO<sub>2</sub>↑ 
* 酸性环境下具有氧化性
: 6FeSO<sub>4</sub>+ 2KNO<sub>3</sub>+ 4H<sub>2</sub>SO<sub>4</sub> → K<sub>2</sub>SO<sub>4</sub>+ 3Fe<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub>+ 2NO↑ + 4H<sub>2</sub>O 　　
* 加热分解生成氧气
: 2KNO<sub>3</sub> → 2KNO<sub>2</sub>+O<sub>2</sub>↑
* 与有机物或硫、磷等摩擦或撞击会燃烧或爆炸
