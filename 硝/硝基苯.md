'''硝基苯'''（俗称'''人造苦杏仁油'''）是一种剧毒有机物{{citation needed}}，有像杏仁油的特殊气味。[[化学式|化学式]]为[[碳|C]]<sub>6</sub>[[氢|H]]<sub>5</sub>[[氮|N]][[氧|O]]<sub>2</sub>。纯品是几乎无色至淡黄色的晶体或油状液体，不溶于水。

硝基苯用作溶剂和温和的[[氧化剂|氧化剂]]。它的主要用途是制造[[苯胺|苯胺]]，也常用作绝缘物质和光泽剂。中毒可用亚甲蓝解救。

== 合成 ==

硝基苯主要是通过使用浓硝酸和浓硫酸的混合物对[[苯|苯]]进行[[硝化反应|硝化反应]]来制备。这个反应是一个典型的[[亲电芳香取代反应|亲电芳香取代反应]]。

== 化学性质 ==

硝基苯较易通过[[催化|催化]]氢化等方法还原，还原的最终产物是[[苯胺|苯胺]]。硝基苯作为温和的氧化剂，在合成[[喹啉|喹啉]]的[[斯克劳普合成法|斯克劳普合成法]]中得到了良好的应用，它负责将中间产物[[1,2-二氢喹啉|1,2-二氢喹啉]]氧化为喹啉。

硝基是一个吸电子基团，这使得苯环上的[[π键|π电子]]密度大大降低，从而使硝基苯参与亲电取代反应的能力有所减弱，同时使硝基成为了间位定位基。硝基苯仍可进行[[硝化反应|硝化反应]]和[[卤代反应|卤代反应]]，得到相应的间位衍生物，但不参与[[傅-克反应|傅-克反应]]。

== 工业应用 ==

硝基苯是工业上制备苯胺和苯胺衍生物（如[[扑热息痛|扑热息痛]]）的重要原料，同时也被广泛用于[[橡胶|橡胶]]、[[杀虫剂|杀虫剂]]、[[染料|染料]]以及[[药物|药物]]的生产。硝基苯也被用于[[涂料|涂料]][[溶剂|溶剂]]、[[皮革|皮革]]上光剂、地板抛光剂等，在这里硝基苯主要用于掩蔽这些材料本身的异味。值得一提的是，硝基苯甚至还曾被用于[[肥皂|肥皂]]的廉价[[香料|香料]]。
