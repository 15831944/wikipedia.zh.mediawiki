'''硝酸铍'''是[[铍|铍]]元素的[[硝酸盐|硝酸盐]]，它是一种[[离子化合物|离子化合物]]，化学式为Be(NO<sub>3</sub>)<sub>2</sub>。<ref name = "INCHEM">{{cite web |url=http://www.inchem.org/documents/icsc/icsc/eics1352.htm|title=Beryllium Nitrate (ICSC)|author= |date= |work= |publisher=IPCS INCHEM|accessdate=13 September 2010}}</ref>每个[[化学式单位|化学式单位]]由一个Be<sup>2+</sup>[[阳离子|阳离子]]和两个[[硝酸根|NO<sub>3</sub><sup>-</sup>]][[阴离子|阴离子]]所构成。

== 危害 ==
硝酸铍像其他的铍化合物一样有剧毒。<ref name = "INCHEM" />小剂量摄入时它是一种刺激剂。它燃烧时产生刺激性的有毒烟雾。然而，短时大量摄入硝酸铍会导致急性[[化学性肺炎|化学性肺炎]]，但症状在3天之内都不会表现出来。<ref name = "INCHEM" />
