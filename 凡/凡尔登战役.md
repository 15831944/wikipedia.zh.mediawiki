'''凡尔登战役'''是[[第一次世界大战|第一次世界大战]]中破坏性最大，时间最长的[[战役|战役]]，战事从1916年2月21日延续到12月19日，[[德国|德]][[法国|法]]两国投入100多个[[師_(軍隊)|师]]兵力，军队死亡超过25万人，50多万人受伤。伤亡人数仅次于[[索姆河战役|索姆河战役]]，被称为“'''凡尔登绞肉机'''”。

== 背景 ==
1914年德国没能取得迅速胜利，[[欧洲|欧洲]]战场在[[西线|西线]]形成胶着状态的[[战壕|战壕]]战，交战双方谁都不能前进一步。1915年，德国的几次进攻努力都没有能够奏效。德国将军[[冯·法金汉|冯·法金汉]]认为如果对法国加强打击，迫使法军投入所有兵力，会使法国因消耗而亡。他选择了位于[[默兹高地|默兹高地]]上的凡尔登要塞作为主要进攻点，凡尔登是法国东北部的一座小城市，是德国通往[[巴黎|巴黎]]的主要交通要道，有“巴黎钥匙”之称。此外，凡爾登亦是[[高盧|高盧]]民族情感的發源地，無論是情感還是戰略層面，法國都不可能放棄凡爾登。因此，進攻此處可以讓法国“流尽最后一滴血”。
1916年发生于此的著名的凡尔登战役，是第一次世界大战的转折点，德意志帝国也从此逐步走向最后失败。

== 战斗过程 ==
凡尔登是巴黎的东部门户，是法军整个战线的支撑点，法军统帅部也非常重视保卫这个战略要地。1916年2月，德国皇太子亲率21万德军，向凡尔登发动了猛烈进攻。

凡尔登的保卫力量比较虚弱，但由于气候条件不佳，德国的进攻被迫延迟，为法国调集援兵提供了时间，法国调集了两个师进入这个地区。

1916年2月21日德军動用1,200门大炮对40公里的战线进行了9个小时的集中轰击，发射了100万发炮弹，然后用3个军（第3、第7、第18军）的兵力进攻，先头部队用[[火焰喷射器|火焰喷射器]]扫清战壕，23日德国军队已经前进了将近5公里，法国军队被迫后撤。

[[File:Battle_of_Verdun_map.png|thumb]]

24日法国军队被迫放弃第二道防线，25日德军用强大的火力进攻占领了{{le|杜奥蒙要塞|Fort Douaumont}}。

2月25日法军总司令[[霞飞|霞飞]]此时才任命[[亨利·菲利普·贝当|亨利·菲利普·贝当]]将军为凡尔登地区总指挥，贝当将军指挥的第20军进入凡尔登的当天，就收到了杜奥蒙要塞失陷的战报，由于法军第33团的顽强抵抗，再加上当时天降大雪，德军对杜奥蒙村的进攻受阻，法军利用唯一一条与后方保持联系的{{le|巴勒迪克-凡尔登公路|Voie Sacrée}}（法国人称为“圣路”）把握时间向凡尔登运送人员补给，一周内组织3,900辆卡车，运送人员190,000和25,000吨的物资，这是人类战史上首次大规模的汽车运输。法军的大批援军及时赶到，加强了纵深防御，扭转了战役开始阶段的不利局面，使战役陷入拉锯战。

德军由于前进过快，丧失了炮火支援的优势，反而使自己暴露在法军位于[[默兹河|默兹河]]西岸的炮火射程内，战场由于持续炮火的攻击已经变成一片泥潭，火炮很难向前移动，到了3月2日德军最终占领杜奥蒙村时，已经丧失了4个团。

由于无法向凡尔登前进，德军转过来准备拿下{{le|莫尔翁山|Le Mort Homme|霍姆山}}，经过3个月的激烈战斗，德军攻占了凡尔登西部的两个村庄，6月7日凡尔登东部的{{le|沃要塞|Fort Vaux}}投降。但双方损失惨重，贝当将军准备保存实力用于更好的防御，但他于5月1日被解职，由更具有进攻性的[[罗贝尔·尼维勒|尼维勒]]将军取代。

德军下一个目标是{{link-fr|苏维要塞|Fort de Souville}}，1916年6月22日德军向法军阵地投放[[毒气|毒气]]弹，第二天以60,000兵力进攻，但没有能拿下苏维要塞，战斗持续到9月6日。

1916年[[索姆河战役|索姆河战役]]打响，迫使德军从凡尔登抽调部分兵力去对付北面的[[英国|英]]法联军。到了秋季，德军已经非常疲惫，德军总司令[[保罗·冯·兴登堡|保罗·冯·兴登堡]]撤换了法金汉将军，任命[[埃里希·鲁登道夫|埃里希·鲁登道夫]]为凡尔登战役司令。

1916年10月21日法军发起反攻，使用了新式400毫米口径的大炮和徐进弹幕射击技术，10月24日夺回杜奥蒙要塞，11月2日德军放弃沃要塞，12月11日德军完全退到战役开始时的战线。

== 损失 ==
这场战役使双方都遭受了惨重的损失，法军损失54.3万人，德军损失43.3万人，虽然法军损失更为严重，但没有达到德军预想的2:1水平。凡尔登虽然使法国到了濒临崩溃的地步，但还没有崩溃，贝当由于坚持战场上的部队必须定期轮换，致使70%的法军都参与了这场战役，而德军只有20%参与了。残酷的战斗造成了心理上的影响，几乎导致法军叛变，由于答应士兵今后再也不参加向德军进攻的严酷的战斗，才使叛变没有真正发生。以后法军士兵拒绝进行进攻性的战斗，只是守在战壕中进行保卫战。其中在[[凡尔登|凡尔登]]要塞爭奪戰，法軍一直在[[要塞|要塞]][[碉堡|碉堡]]以[[機槍|機槍]]、[[迫擊砲|迫擊砲]]、強力[[彈弓|彈弓]](發射[[手榴彈|手榴彈]])對抗來襲的德軍。雙方激戰數晝夜，死傷甚為慘重，因此又被稱為凡爾登絞肉機。



== 后果 ==
凡尔登战役的胜利使得法国更相信要塞防御系统，从而导致修建[[马奇诺防线|马奇诺防线]]，凡尔登战役也成为法国决心保卫自己的信心的象征。
