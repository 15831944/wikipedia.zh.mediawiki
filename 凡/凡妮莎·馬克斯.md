'''凡妮莎·馬奎斯'''（{{lang-en|'''Vanessa Marquez'''}}，{{bd|1968年|12月21日|2018年|8月30日|catIdx=Marquez, Vanessa}}），美國女演員。她最出名是在《[[仁心仁術|仁心仁術]]》的首三季中飾演護士Wendy Goldman。她是{{link-en|ACTRA|ACTRA}}的成員。

== 個人生活 ==
馬奎斯曾患有[[重性抑鬱障礙|抑鬱症]]、強迫症和購物成癮。2005年A＆E的節目《{{link-en|Intervention (TV series)|Intervention (TV series)|Intervention}}》中記錄了她對購物成癮的掙扎。<ref>{{Cite web |url = https://losangeles.cbslocal.com/2018/08/31/er-actress-vanessa-marquez-shot-killed-by-south-pasadena-pd-after-brandishing-bb-gun/ |title = ‘ER’ Actress Vanessa Marquez Shot, Killed By Police After Brandishing BB Gun |date = 2018-08-31 |publisher = CBSLA |language = en |access-date = 2018-09-01 |archive-url = https://web.archive.org/web/20180831225958/https://losangeles.cbslocal.com/2018/08/31/er-actress-vanessa-marquez-shot-killed-by-south-pasadena-pd-after-brandishing-bb-gun/ |archive-date = 2018-08-31 |dead-url = no }}</ref>

=== 逝世 ===
2018年8月30日，馬奎斯在她位於加州[[南帕薩迪納_(加利福尼亞州)|南帕薩迪納]]的家中發生事故後，被警察殺害。當局曾打了電話到她的家中進行社福檢查，並發現馬奎斯有癲癇發作，而且患有精神健康的問題。在與當局進行了90分鐘的談話，其中包括一名洛杉磯縣心理健康人員的介入後，她用一枝看起來像手槍的東西指著警察，警察便開火。後來才發現只是一把BB槍。<ref>{{Cite web |url = https://www.huffingtonpost.com/entry/er-actress-killed-by-police_us_5b89644ee4b0511db3d7d371 |title = Former ‘ER’ Actress Killed By Police After She Took Out BB Gun |author = Marina Fang |date = 2018-08-31 |publisher = Huffington Post |language = en}}</ref><ref>{{Cite web |url = https://variety.com/2018/tv/news/vanessa-marquez-er-shot-killed-police-1202923337/ |title = Former ‘ER’ Actress Vanessa Marquez Shot and Killed by Police |author = Gene Maddaus |date = 2018-08-31 |publisher = Variety |language = en |access-date = 2018-09-01 |archive-url = https://web.archive.org/web/20180831200533/https://variety.com/2018/tv/news/vanessa-marquez-er-shot-killed-police-1202923337/ |archive-date = 2018-08-31 |dead-url = no }}</ref>
