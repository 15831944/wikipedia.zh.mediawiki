'''凡妮莎·梅（[[德语|德語]]：Vanessa Mai，{{bd|1992年|5月2日}}），出生於[[德国|德國]][[阿斯巴赫-西肯贝格|阿斯帕赫]] ([[巴登-符腾堡|巴登-符騰堡]]<ref>{{Cite web|url=http://www.billboard.com/artist/7348932/vanessa-mai/biography|title=Vanessa Mai - Biography {{!}} Billboard|accessdate=2017-08-01|work=www.billboard.com|language=en}}</ref>)，是[[克罗地亚|克羅埃西亞]]裔的[[德国|德國]]流行音樂女歌手。曾是{{link-de|無雲樂隊|Wolkenfrei}}一員，由於樂隊其他成員的退出使得凡妮莎·梅扛下樂團之責，凡妮莎·梅在宣傳期告一段落後，也決定單飛追求個人歌唱事業。'''

== 生平 ==
'''凡妮莎·梅是一位有魅力的歌手和舞者 ，以電子舞蹈流行音樂為主。凡妮莎·梅出生於1992年5月2日在[[德国|德國]][[阿斯巴赫-西肯贝格|阿斯帕赫]]，父親是[[克罗地亚|克羅埃西亞]]的音樂家，母親則為[[德国|德國]]人，而凡妮莎·梅就在一個音樂家庭中長大。鼓勵從小就開始唱歌，甚至經常和父親一起在舞台上演出，後來加入學校音樂隊。在她青少年時期，曾經也是嘻哈樂團[[GettinCrazd|Getting Craz'd]]的成員之一。<ref>{{Cite web|url=http://www.billboard.com/artist/7348932/vanessa-mai/biography|title=Vanessa Mai - Biography {{!}} Billboard|accessdate=2017-08-01|work=www.billboard.com|language=en}}</ref>'''

'''在2013年 ，凡妮莎·梅認識他的經理Andreas Ferber。2016年年初與Andreas Ferber訂婚<ref>{{Cite news|url=http://www.vip.de/cms/dsds-jurorin-vanessa-mai-und-andreas-ferber-haben-sich-auf-mallorca-verlobt-2610351.html|title=DSDS-Jurorin Vanessa Mai und Andreas Ferber haben sich auf Mallorca verlobt|newspaper=vip.de|accessdate=2017-08-01|language=de-DE}}</ref>，他們在2017年6月6日舉行民間結婚儀式，並在2017年6月12日在[[馬略卡島|馬約卡島]]波連斯薩 ( Mallorca Pollença ) 教堂正式結婚。<ref>{{Cite news|url=http://www.t-online.de/unterhaltung/stars/id_81418104/vanessa-mai-andreas-ferber-heimliche-traumhochzeit-auf-mallorca.html|title=Vanessa Mai & Andreas Ferber: Heimliche Traumhochzeit auf Mallorca|newspaper=www.t-online.de|accessdate=2017-08-01}}</ref>'''

'''Vanessa Mandekic 決定以自己出生的月份 ，5月當作藝名 Vanessa Mai ( Mai 是[[德语|德語]]的5月)。'''

== 音樂事業 ==


=== Wolkenfrei ( 2012 ~ 2015 ) ===
'''因為 ( 2011年 ) 主唱[[Heike_Wanner|Heike Wanner]]健康因素而退出，在2012年，電子鋼琴手[[Marc_Fischer|Marc Fischer]]和吉他手[[Stefan_Kinski|Stefan Kinski]]所成立的樂團，Marc Fischer Band ( 樂團舊名 ) 決定更深入流行領域，隨後更名為[[Wolkenfrei|Wolkenfrei]]，同年5月演唱會Marc Fischer找來歌舞雙才的凡妮莎·梅入替新主唱一職。<ref>{{Cite news|url=https://www.sonymusic.com.tw/album/%E7%8D%BB%E7%B5%A6%E4%BD%A0-2cd%E9%99%90%E9%87%8F%E8%B1%AA%E8%8F%AF%E7%9B%A4-%E5%87%A1%E5%A6%AE%E8%8E%8E%E7%8E%AB-vanessa-mai-88985305232/|title=獻給你 (2CD限量豪華盤) - 台灣索尼音樂娛樂股份有限公司|newspaper=台灣索尼音樂娛樂股份有限公司|accessdate=2017-08-01|language=en-US}}</ref>'''

'''2013年7月，發行出道單曲《 Jeans, T-Shirt und Freiheit 》。隨後在2014年2月，發行《 Endlos Verliebt 》，在專輯中誕生了數張單曲，更達到了18首歌曲。<ref>{{Cite web|url=http://www.billboard.com/artist/7348932/vanessa-mai/biography|title=Vanessa Mai - Biography {{!}} Billboard|accessdate=2017-08-01|work=www.billboard.com|language=en}}</ref>'''

'''2015年，就在籌備《 Wachgeküsst 》新作品之際，Marc 選擇退居幕後，Stefan 則回歸家庭與職場，兩人皆離開[[Wolkenfrei|Wolkenfrei]]，獨剩 Vanessa 扛下樂團之責，專輯空降[[德国|德國]]榜 TOP7，開出亮眼佳績。凡妮莎在宣傳期告一段落後，也決定單飛追求個人歌唱事業！<ref>{{Cite news|url=https://www.sonymusic.com.tw/album/%E9%AD%94%E5%8A%9B%E4%B9%8B%E5%90%BB-%E6%99%B4%E7%A9%BA%E6%A8%82%E5%9C%98-wolkenfrei-88875050292-0/|title=魔力之吻 - 台灣索尼音樂娛樂股份有限公司|newspaper=台灣索尼音樂娛樂股份有限公司|accessdate=2017-08-01|language=en-US}}</ref>'''

=== 獨唱生涯 ( 2016至今 ) ===
'''成功之後，凡妮莎·梅參加第13季[[德国|德國]]現場節目歌唱比賽“超級巨星”。此外，在這段時間裡，透露出，Marc 和 Stefan 退出 [[Wolkenfrei|Wolkenfrei]] 之後，樂隊將以凡妮莎·梅的名義前進。<ref>{{Cite web|url=http://www.billboard.com/artist/7348932/vanessa-mai/biography|title=Vanessa Mai - Biography {{!}} Billboard|accessdate=2017-08-01|work=www.billboard.com|language=en}}</ref>'''

'''2016年5月4日，發行凡妮莎·梅出道的第3張專輯《 Für dich 》，也是她單飛的第一張大碟，並以“Ich Sterb für dich”為單曲。'''

'''2017年1月6日，發行《 Für dich - Live aus Berlin 》，是凡妮莎·梅於2016年10月10日在[[柏林|柏林]]現場演唱會所收錄的專輯。'''

'''2017年8月11日，發行專輯《 Regenbogen 》，這次的專輯可帶著俏皮的風格。'''

'''於2018年1月12日，再次發行專輯《 Regenbogen 》，不過這次是( Gold Edition )，意味著曲風不同和加上了一些新歌，讓歌迷們有著不同的新感受。'''

'''2018年8月3日，新專輯《Schlager》，這次專輯名稱就是德語系國家的一種流行歌舞曲意思，主打《Wir 2 Immer 1 》，還找了Olexesh饒舌歌手一同演唱，又為此次專輯帶來前所未有的新風格。'''
