== 發行 ==
《制服模特兒》與上作《[[衝吧！Bicycle|衝吧！Bicycle]]》相隔約四個月發售，是乃木坂46於2012年發行的第四張單曲。單曲分為附送DVD的「Type-A」、「Type-B」和「Type-C」，以及只有CD的「通常盤」和「動畫盤」（{{lang|ja|アニメ盤}}），共五種形式發行。<ref name="natalie20121105" />Type-A、Type-B、Type-C和動畫盤在首批發行期間均附送全國[[握手會|握手會]]應徵券兼特別禮物應徵券（{{lang|ja|全国握手会参加券兼スペシャルプレゼント応募券}}）。<ref>{{cite web|url=http://www.nogizaka46.com/discography/?d=SRCL-8201 |language=ja |title=<span lang="ja">制服のマネキン【CD+DVD盤/Type-A】</span> |publisher=[[乃木坂46|乃木坂46]] |date=2012 |accessdate=2012-12-28}}</ref>

五種形式的封面各不相同。Type-A、Type-B、Type-C和通常盤的封面各以[[乃木坂46#選拔成員制|八福神]]的其中兩名成員為主角，動畫盤的封面則以《[[魔奇少年#電視動畫|魔奇少年]]》中的三名主角——阿拉丁（{{lang|ja|アラジン}}）、阿里爸爸（{{lang|ja|アリババ}}）和莫路珍娜（{{lang|ja|モルジアナ}}）——為主角。<ref>{{cite web|url=http://natalie.mu/music/news/80618 |language=ja |title=<span lang="ja">乃木坂46「制服のマネキン」ジャケットに八福神＆マギ</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2012-11-27 |accessdate=2012-12-28}}</ref>

這張單曲亦經[[台灣索尼音樂娛樂|台灣索尼音樂娛樂]]在台灣於同年12月21日發行<ref name="台灣索尼" />，台灣版的封面和收錄曲目都跟日本版的[[#通常盤|通常盤]]相同。<ref>{{cite web|url=https://www.sonymusic.com.tw/main/album/%E5%88%B6%E6%9C%8D%E6%A8%A1%E7%89%B9%E5%85%92-%E4%B9%83%E6%9C%A8%E5%9D%8246-nogizaka46-88765444582|language=|title=制服模特兒 - 乃木坂46 Nogizaka46 - 88765444582|publisher=[[台灣索尼音樂娛樂|台灣索尼音樂娛樂]]|date=2012|accessdate=2012-12-28}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 宣傳 ==
乃木坂46於2012年11月5日公布單曲名稱為「{{lang|ja|制服のマネキン}}」。<ref>{{cite web|url=http://www.nogizaka46.com/news/2012/11/4th-1.php |language=ja |title=<span lang="ja">4thシングルのタイトル発表</span> |publisher=[[乃木坂46|乃木坂46]] |date=2012-11-05 |accessdate=2012-12-28}}</ref>同年12月14日至21日，為紀念新單曲發行，官方網站設立特別頁面「乃木坂46中心測驗」（{{lang|ja|乃木坂46センター試験}}），「測驗」中有一些關於成員自身的困難題目，如果全對就可以獲得獨家圖像。<ref>{{cite web|url=http://www.nogizaka46.com/center/ |language=ja |title=<span lang="ja">乃木坂46 センター試験</span> |publisher=[[乃木坂46|乃木坂46]] |date=2012 |accessdate=2012-12-28}}</ref><ref>{{cite web|url=http://natalie.mu/music/news/81624 |language=ja |title=<span lang="ja">乃木坂46、問題日替わり「乃木坂46センター試験」開始</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2012-12-14 |accessdate=2012-12-28}}</ref>

2012年12月13日至2013年1月14日，在[[大江戶線|都營大江戶線]]沿線28個車站張貼了這張單曲的宣傳海報，海報上乃木坂46的成員穿上[[東京都交通局|東京都交通局]]以及六家企業的制服。該六家企業包括[[伊勢丹|伊勢丹]]、[[宏達國際電子|HTC]]、[[全日空航空公司|全日空航空]]、[[中日龍|中日龍]]、[[哈多巴士|哈多巴士]]和Heritage飯店（{{lang|ja|ホテルヘリテイジ}}）。<ref>{{cite web|url=http://mfound.jp/news/2012/12/015883.html |language=ja |title=<span lang="ja">乃木坂46、新曲「制服のマネキン」のPRキャンペーンで、都営大江戸線をジャック！</span> |publisher=mFound |date=2012-12-13 |accessdate=2012-12-28}}</ref>

== 歌曲 ==
A面曲《制服模特兒》（{{lang|ja|制服のマネキン}}）是[[HTC_J_Butterfly|HTC J Butterfly]]的廣告歌曲。<ref name="natalie20121210">{{cite web|url=http://natalie.mu/music/news/81329 |language=ja |title=<span lang="ja">HTC J新CMで乃木坂46八福神の素顔をフィーチャー</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2012-12-10 |accessdate=2012-12-28}}</ref>{{noteTag|1=《制服模特兒》的[[乃木坂46#選拔成員制|八福神]]（[[秋元真夏|秋元真夏]]、[[生田繪梨花|生田繪梨花]]、[[生駒里奈|生駒里奈]]、[[櫻井玲香|櫻井玲香]]、[[白石麻衣|白石麻衣]]、[[橋本奈奈未|橋本奈奈未]]、[[星野南|星野南]]和[[松村沙友理|松村沙友理]]）參與了這個廣告的演出。<ref name="natalie20121210" />}}B面曲《指望遠鏡》（{{lang|ja|指望遠鏡}}）是動畫《[[魔奇少年#電視動畫|魔奇少年]]》的片尾曲。<ref>{{cite web|url=http://natalie.mu/music/news/79903 |language=ja |title=<span lang="ja">乃木坂46「マギ」ED曲アニメバージョンが配信開始</span> |publisher=[[娜塔莉_(網站)|natalie]] |date=2012-11-14 |accessdate=2012-12-28}}</ref>

== 影片 ==
跟以往的單曲一樣，DVD收錄了乃木坂46全體成員的個人影片。由於成員[[柏幸奈|柏幸奈]]因考試而沒有參與拍攝，所以這次共有32段影片，分別收錄於Type-A、Type-B和Type-C的DVD內。這次的個人影片以「克服不擅長！成員32人向着明天的一步」（{{lang|ja|苦手克服！メンバー32人の明日への一歩！}}）為主題，內容為成員去克服其各自不擅長的事物，例如[[生駒里奈|生駒里奈]]是「水」、[[生田繪梨花|生田繪梨花]]是「鳥」、[[松村沙友理|松村沙友理]]是「蛇」等等。<ref>{{cite web|url=http://www.oricon.co.jp/news/music/2018794/full/ |language=ja |title=<span lang="ja">乃木坂46、苦手克服に挑戦　生駒は制服でプールへ</span> |publisher=[[Oricon|Oricon]] |date=2012-11-15 |accessdate=2012-12-28}}</ref>

== 反應 ==
《制服模特兒》於發行首日的銷量達19萬張，比上一張單曲《[[衝吧！Bicycle|衝吧！Bicycle]]》的首周銷量約18.7萬張多，取得當日[[Oricon公信榜|Oricon公信榜]]單曲日榜首位。<ref name="台灣索尼" />

單曲發行首周的銷量約為23.3萬張，取得該周Oricon公信榜單曲周榜首位。這是乃木坂46首次有單曲首周銷量突破20萬張。在2012年開始發行單曲的乃木坂46於該年共有三張單曲取得該排行榜首位，包括《[[來吧香波|來吧香波]]》、《衝吧！Bicycle》和這張單曲，為該榜首次有女性藝人於出道年達成此紀錄；亦是自6年前的[[KAT-TUN|KAT-TUN]]以來再次達成此紀錄。<ref name="oricon20121225" />{{noteTag|1=在這之前，女性藝人於出道年取得[[Oricon公信榜|Oricon公信榜]]單曲周榜首位次數的最高紀錄為2張，由[[岩崎宏美|岩崎宏美]]（1975年）和[[NMB48|NMB48]]（2011年）保持。<ref name="oricon20121225" />}}
