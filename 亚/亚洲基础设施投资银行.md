'''亞洲基礎設施投資銀行'''（{{lang-en|Asian Infrastructure Investment Bank}}，[[縮寫|縮寫]]為{{lang|en|'''AIIB'''}}），[[簡稱|簡稱]]'''亞投行'''，是一個向[[亚洲国家和地区列表|亚洲各国家和地区]]政府提供資金以支持[[基础设施|基础设施]]建設之區域多邊開發機構，成立宗旨在促進亞洲區域内的[[互联互通|互联互通]]建設和經濟一體化進程，並且加強[[中华人民共和国|中华人民共和国]]及其他亞洲國家和地區的合作。總部設在[[中华人民共和国|中国]][[北京市|北京]]，[[授權資本|法定资本]]為1,000億[[美元|美元]]。<ref name="21国签约">{{cite news |language = zh-hans |author = 韩洁 |coauthors = 何雨欣 |url = http://news.xinhuanet.com/fortune/2014-10/24/c_1112965880.htm |title = 21国在京签约决定成立亚洲基础设施投资银行|publisher = 新华网 |date = 2014-10-24 |accessdate = 2015-04-01}}</ref>

==起源==
[[中华人民共和国主席|中华人民共和国主席]][[习近平|习近平]]于2013年10月2日在[[雅加达|雅加达]]-{zh-hans:同;zh-hant:與}-時任[[印度尼西亚总统|印尼總統]][[苏西洛·班邦·尤多约诺|苏西洛]]举行会谈时首次倡议筹建亚投行<ref name="习近平倡议"/>，[[中华人民共和国国务院总理|中国国务院总理]][[李克强|李克强]]同月出访[[东南亚|东南亚]]时，也向东南亚国家提出相关倡议<ref name="李克强提出"/>。2014年10月24日，中華人民共和國、[[印度|印度]]、[[新加坡|新加坡]]等21国在北京正式签署《筹建亚投行[[備忘錄|備忘錄]]》<ref name="21国签约"/>，[[印尼|印尼]]因政府换届，於2014年11月25日签署备忘录，成为第22个意向创始成员国<ref name="印尼签署" />。

2015年3月12日，[[英国|英国]]報名加入亚投行的意向创始成员国<ref >{{Cite web |url = http://www.financialnews.com.cn/gj/gjyw/201503/t20150314_72523.html |title = 英国正式申请加入亚投行 专家称不足为奇但意义特别}}</ref>，成为首个計劃加入亞投行的[[歐洲國家和地區列表|歐洲]]及[[西方世界|西方国家]]，更是经济[[大国|大国]]和第一個[[七國集團|七國集團]]成員<ref>{{Cite web|url=http://www.renminbao.com/rmb/articles/2015/9/21/62143pb.html|title=英國財相奧斯本訪華 推「新時代中英關係」(圖)|accessdate=2015-09-21|author=|date=2015-09-21|publisher=人民日报}}</ref>。次日[[瑞士|瑞士]]也提出申请意願，随后，[[法国|法国]]、[[意大利|意大利]]、[[德国|德国]]等发达国家也表態跟進<ref name="法意德申请" />。[[大韩民国|韩国]]、[[俄罗斯|俄罗斯]]等域內国家和[[巴西|巴西]]这一重要[[新兴市场|新兴经济体]]也在申请截止日期3月31日前相繼申请加入意向創始成員國<ref name="韩国加入_2" /><ref name="俄罗斯" /><ref name="荷兰、巴西、格鲁吉亚加入" />。至此，全球[[五大洲|五大洲]]均有國家參與。

==始創階段==
2015年4月15日，報名亞投行的截止日，亚投行意向创始成员国确定为57个，其中域内国家37个、域外国家20个<ref name="AIIB Agr." />。[[东南亚国家联盟|东南亚国家联盟]]（東盟）10国全数加入，拥有28个成员國的[[欧洲联盟|欧洲联盟]]（歐盟）有14国加入，[[二十國集團|二十國集團]]（G20）中也有14国加入，而[[金磚國家|金磚國家]]則全部跻身首发阵容<ref>{{Cite web |url = http://www.guancha.cn/economy/2015_04_15_316010.shtml |title = 亞投行創始成員國名單確定 |date = 2015年4月15日 |publisher = 觀察者}}</ref>，其他国家和地区今后仍可以作为普通成员加入。各國将于2015年年中完成亚投行章程谈判并签署，年底前完成章程生效程序，正式成立亞洲基礎設施投資銀行<ref name="2015.3.20楼部长答记者问" />。

2015年12月25日，[[中华人民共和国财政部|中国财政部]]部长[[楼继伟|楼继伟]]宣布，《亚洲基础设施投资银行协定》正式生效，亚投行宣告成立。至当日为止，包括[[缅甸|缅甸]]、[[新加坡|新加坡]]、[[文莱|文莱]]、[[澳大利亚|澳大利亚]]、中国、[[蒙古国|蒙古]]、[[奥地利|奥地利]]、[[英国|英国]]、[[新西兰|新西兰]]、[[卢森堡|卢森堡]]、[[大韩民国|韩国]]、[[格鲁吉亚|格鲁吉亚]]、[[荷兰|荷兰]]、[[德国|德国]]、[[挪威|挪威]]、[[巴基斯坦|巴基斯坦]]、[[约旦|约旦]]等在内的17个意向创始成员国（股份总和占比50.1%）已批准《亚洲基础设施投资银行协定》并提交批准书。根据筹建工作计划，亚洲基础设施投资银行开业仪式暨理事会和董事会成立大会将于2016年1月16日至18日在北京举行<ref name="成立">{{Cite web |url = http://epaper.bjnews.com.cn/html/2015-12/26/content_615696.htm?div=-1|title =亚投行正式成立下月16日开业|date = 2015年12月26日 |publisher = 新京报}}</ref>。

2016年1月16日至18日，亚洲基础设施投资银行的开业仪式在北京举行，16日上午在北京[[钓鱼台国宾馆|钓鱼台国宾馆]]，国家主席[[习近平|习近平]]出席亚投行的开业仪式并致辞，分别致辞的还有亚投行行长[[金立群|金立群]]<ref name="开业仪式2"/>。金立群表示，亚投行的核心价值观是“精干、廉洁和绿色”<ref>{{Cite news|url=http://www.mof.gov.cn/zhengwuxinxi/caizhengxinwen/201601/t20160118_1651612.html|title=亚投行启航十大看点|date=2018-06-29|newspaper=archive.is|accessdate=2018-06-29|deadurl=bot: unknown|archiveurl=https://archive.today/20180629141118/http://www.mof.gov.cn/zhengwuxinxi/caizhengxinwen/201601/t20160118_1651612.html|archivedate=2018年6月29日|df=}}</ref>。同日下午，国务院总理[[李克强|李克强]]出席亚投行理事会成立大会并致辞<ref name="开业">{{Cite web |url = http://www.fmprc.gov.cn/web/fyrbt_673021/t1330339.shtml |title = 亚洲基础设施投资银行开业仪式系列活动将在北京举行 |date = 2016年1月8日 |publisher = [[中华人民共和国外交部|中华人民共和国外交部]]}}</ref>。在亚投行理事会成立大会第一部分议程中，[[中华人民共和国财政部|中国财政部]]部长[[楼继伟|楼继伟]]被选举为首届理事会主席<ref name="开业仪式"/>。

== 歷史发展 ==
=== 背景 ===
[[亞洲經濟|亞洲經濟]]占[[世界經濟|全球经济]]总量的1/3，是当今世界上最具经济活力和增长潜力的地区，拥有全球六成[[世界人口|人口]]。但因建设资金有限，一些国家與地區的铁路、公路、桥梁、港口、机场和通讯等基础設施严重不足，这在一定程度上限制了该区域的经济发展。<ref name="亚投行路线图猜想">[http://paper.people.com.cn/gjjrb/html/2014-11/24/content_1501990.htm 亚投行路线图猜想《国际金融报》人民网]</ref>

根据[[亞洲開發銀行|亞洲開發銀行]]估计，2010年至2020年的10年间，[[亞洲國家和地區列表|亚洲國家與地區]]要想维持现有经济增长水平，内部基础设施的投资至少需要8万亿美元，平均每年需投资8000亿美元。8万亿美元中，68%用于新增基础设施的投资，32%是维护或维修现有基础设施的所需资金。现有的多边机构并不能提供如此巨额的资金，亞洲開發銀行的总资金约为1600亿美元，[[世界银行|世界银行]]也仅有2230亿美元，目前，两家银行每年能够提供给亚洲国家與地區的资金只有約200亿美元。<ref name="亚投行路线图猜想"/>

=== 成立 ===
2015年6月29日，随着亚投行“基本法”《亚洲基础设施投资银行协定》當天上午在北京正式签署，57个意向创始成员国中，已完成国内审批的50国参加签字仪式。各国签署《亚投行协定》后，还需经本国立法机构批准。年底之前，经合法数量的国家批准后，《协定》即告生效，亚投行正式成立。<ref name="亚投行协议签署" />
根据《亚投行协定》，《协定》由中华人民共和国政府（保存人）保存，《协定》附件一所列各国政府应在2015年12月31日前完成签署。保存人应将《协定》经过核定无误的副本寄给所有签署方及其他已成为银行成员的国家。《协定》须经签署方批准、接受或核准。批准书、接受书或核准书应于2016年12月31日之前向保存人交存，或如有必要，在理事会依照本协定第二十八条规定经特别多数投票通过的稍晚日期之前向保存人交存。保存人应及时将每次交存及交存日期通知其他签署方。在《协定》生效日之前交存批准书、接受书或核准书的签署方，在协定生效之日成为银行成员。任何其他履行本条第一款规定的签署方，在交存批准书、接受书或核准书之日起成为银行成员。 至少有十个签署方已交存批准书、接受书或核准书，且签署方在本协定附件一列出初始认缴股本的加总数额不少于认缴股本总额的百分之五十，《协定》即告生效。

2015年12月25日，中华人民共和国财政部部长楼继伟宣布，《亚洲基础设施投资银行协定》正式生效，亚投行宣告成立。至当日为止，包括缅甸、新加坡、文莱、澳大利亚、中国、蒙古、奥地利、英国、新西兰、卢森堡、韩国、格鲁吉亚、荷兰、德国、挪威、巴基斯坦、约旦等在内的17个意向创始成员国（股份总和占比50.1%）已批准《协定》并提交批准书。根据筹建工作计划，亚洲基础设施投资银行开业仪式暨理事会和董事会成立大会将于2016年1月16日至18日在北京举行<ref name="成立"/>。

==运作==
=== 筹建阶段工作机制 ===
亚投行筹建工作确立了以各国财政部参与的谈判代表会议为章程谈判的主渠道、以筹建多边临时秘书处为技术支撑机构的工作机制。中國作为亚投行的发起國和[[东道主|东道国]]担任谈判代表会议的常设主席，承办会议的成员国担任当次会议的联合主席。秘书处从专业角度为章程谈判提供技术支持，[[金立群|金立群]]为秘书长<ref name="2015.3.25史部长答记者问">[http://gjs.mof.gov.cn/pindaoliebiao/gongzuodongtai/201503/t20150325_1206668.html 2015年3月25日史耀斌副部长就亚投行筹建有关问题答记者问]</ref>。

作为意向创始成员国，拥有在筹建阶段讨论亚投行章程的权利，并参与到亚投行的筹建进程。开放申请成为意向创始成员国的日程中，[[美国|美国]]认为亚投行的成立与其[[国家利益|国家利益]]相冲突，因此抱持保留态度<ref name="亞投行創始國申請截止，美日拒絕">{{Cite web |url = http://www.thenewslens.com/post/145035/ |title = 亞投行創始國申請截止，美日拒絕 |date = 2015年4月1日 |publisher = The News Lens 關鍵評論}}</ref>，並试图劝说其盟友不要加入，然而除[[日本|日本]]外，收效甚微<ref name="美國孤立亞投行計劃破產">{{Cite web |url = http://www.135editor.com/zh-tw/articles/20150320/550d0922-2694-48fe-9164-671202734e20.html |title = 美國孤立亞投行計劃破產 |date = 2015年3月20日 |access-date = 2015-05-04 |archive-url = https://web.archive.org/web/20150509233352/http://www.135editor.com/zh-tw/articles/20150320/550d0922-2694-48fe-9164-671202734e20.html |archive-date = 2015-05-09 |dead-url = yes }}</ref>。

=== 治理结构与政策 ===
作为一家新成立的多边开发银行，亚投行在治理结构、环境和社会保障政策方面将充分借鉴现有多边开发银行通行的经验與好的做法，同时也要避免其走过的弯路，以降低成本和提高运营效率，更好地为成员国服务<ref name="2015.3.20楼部长答记者问">{{Cite web|url=http://www.mof.gov.cn/zhengwuxinxi/caizhengxinwen/201503/t20150320_1205011.html|title=楼继伟部长就亚投行筹建情况答记者问|accessdate=2017-08-29|author=|date=2015-03-20|work=|publisher=中华人民共和国外交部}}</ref>。

亚投行将设立[[理事会|理事会]]、[[董事会|董事会]]和[[管理學#管理层|管理层]]三层管理架构，并将建立有效的监督机制，确保决策的高效、公开和透明。目前各創始國正在认真研究现有多边开发银行的治理模式和经验，并广泛听取包括非成员国和其他多边开发银行在内的有关方面意见，在此基础上对如何设计亚投行的治理模式进行深入讨论<ref name="2015.3.20楼部长答记者问" />。董事会12个席位被划分为9个亚洲国家的地区性席位和3个亚洲以外国家的非地区性席位，这对亚洲较小国家有利<ref name="菲財長稱亞洲以外三國獲董事席位">{{Cite web |url = http://www.guancha.cn/economy/2015_05_20_320324.shtml |title = 菲財長稱亞洲以外三國獲董事席位 |publisher = 觀察者 |date = 2015年5月20日}}</ref>。

亚投行将根据公开、透明、择优的原则选聘行长和高层管理人员。根据现有多边开发银行的通行做法，亚投行将在正式成立后召开部长级理事会任命首任行长<ref name="2015.4.15史部长答记者问" />。关于是否在其他国家设立区域中心等问题，将根据未来亚投行的业务开展情况來协商确定<ref name="2015.3.25史部长答记者问"/>。

=== 投票权与缴纳资本 ===
依据《亚投行协议》，亚投行成员的总投票权由股份投票权、基本投票权以及创始成员享有的创始成员投票权组成。每个成员的股份投票权等于其持有的亚投行股份数，基本投票权占总投票权的12%，由全体成员(包括创始成员和今后加入的普通成员)平均分配。每个创始成员同时拥有600票创始成员投票权，基本投票权和创始成员投票权占总投票权的比重约为15%。每一位成员的初始认缴股本分五次缴清，每次缴纳百分之二十；第一次缴付应在本协定生效后三十天内完成，或在成员国内批准程序完成后相关文件交付于保存人当日缴清，以后发生者为准。第二次缴付在本协定生效期满一年内完成。其余三次将相继在上一次到期一年内完成。初始认缴中原始实缴股本的每次缴付均应使用美元或其他可兑换货币（这意味着可以缴付该成员国国本币，只要满足其他有关限制性规定）。银行可随时将此类缴付转换为美元。如若到期未能完成缴付，则相应的实缴和待缴股本所赋予的权利，包括投票权等都将中止，直至银行收到到期股本的缴付。银行的待缴股本，仅在银行需偿付债务时方予催缴，成员可选择美元或银行偿债所需货币进行缴付。在催缴待缴股本时，所有待缴股份的催缴比例应一致。各种缴付的地点由银行决定，但在理事会举行首次会议之前，首次付款应支付给银行的托管方：中华人民共和国政府。欠发达成员在缴纳资本时会有优待措施。<ref name="AIIB Agr." />

== 迴響 ==
=== 与现有多边开发银行的关系 ===
亚投行是国际发展领域的新成员、新伙伴，在亚洲基础设施[[借贷|融资]]需求巨大的情况下，由于定位和业务重点不同，亚投行与现有多边开发银行是互补而非竞争关系。亚投行侧重于[[基础设施|基础设施]]建设，而现有的[[世界银行|世界银行]]（世行）、[[亞洲開發銀行|亞洲開發銀行]]（亞行）等多边开发银行则强调以减贫为主要宗旨。从历史经验看，包括亚洲开发银行和[[欧洲复兴开发银行|欧洲复兴开发银行]]（EBRD）在内的区域性多边开发银行的设立，不仅没有削弱世界银行等现有全球性多边开发银行的影响力，反而增强了多边开发性金融的整体力量，更有力地推动了全球经济的发展<ref name="2015.3.20楼部长答记者问" />。亚投行将致力于促进亚洲地区基础设施建设和互联互通，其中包括“[[一带一路|一带一路]]”沿线亚投行成员国的相关基础设施建设项目。中國也欢迎世界银行、亚洲开发银行等机构积极参与“一带一路”相关项目<ref name="2015.3.20楼部长答记者问" />。

在亚投行筹建以及未来运作过程中，中國都将积极推动亚投行与世界银行、亞洲開發銀行等现有多边开发银行在[[创作共用|知识共享]]、能力建设、人员交流、项目融资等方面开展合作，共同提高亞洲地区基础设施融资水平，促进本地区的经济和社会发展。中國作为亞投行的發起國，世界银行、亚洲开发银行的重要[[股东|股东]]国，将一如既往地支持现有多边开发银行在促进全球减贫和发展事业方面做出积极贡献。自從筹建亚投行的倡议提出以来，世行行長[[金墉|金墉]]、[[國際貨幣基金組織|国际货币基金组织]]（IMF）總裁[[克里斯蒂娜·拉加德|拉加德]]和亚行行長[[中尾武彥|中尾武彦]]分别在多个场合表态积极支持筹建亚投行，表示将与其开展合作。世行、亚行等多边开发银行已与亚投行筹建多边临时秘书处建立了工作联系，在许多方面给予了支持。<ref name="2015.3.20楼部长答记者问" /><ref name="2015.4.15史部长答记者问" />
