'''亚历山大·米哈伊洛维奇·萨莫库佳耶夫'''（{{lang-ru|Алекса́ндр Миха́йлович Самокутя́ев}}，{{bd|1970年|3月13日|}}）是[[俄罗斯|俄罗斯]]已退役[[宇航员|宇航员]]。

==早年==
1970年3月13日出生，在[[奔萨|奔萨]]长大。1987年至1988年在奔萨理工学院学习，1992年毕业于[[切尔尼戈夫|切尔尼戈夫]]高等飞行员航空学校<ref>{{Cite web|url=http://www.roscosmos.ru/main.php?id=2&nid=11591 |title=Soyuz TMA-21 Prime Crew. Aleksander SAMOKUTYAEV – Commander |author=Russian Federal Space Agency |date=2011-03-27 |accessdate=2011-04-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20110420001934/http://roscosmos.ru/main.php?id=2 |archivedate=2011-04-29}}</ref>。2000年毕业于[[加加林空军学院|加加林空军学院]]，获得飞行工程师资格<ref>{{cite web|url=http://www.esa.int/SPECIALS/magisstra/SEM871TOREG_0.html|title=Alexander Samokutyayev|publisher=[[欧洲航天局|European Space Agency]]|date=2010-10-07}}</ref>。

==宇航员生涯==
2003年成为[[加加林宇航员培训中心|加加林宇航员培训中心]]宇航员候选人。2003年6月至2005年6月间接受了基本太空飞行训练，以优秀成绩通过考验。2005年7月获得测试宇航员资格。2005年8月至2008年11月参加了[[国际空间站|国际空间站]]项目高级培训。2008年12月开始接受[[远征23|远征23]]/24任务备用成员培训<ref name=bio>{{cite web |title=Biographical Data - Aleksandr Samokutyayev|url=https://web.archive.org/web/20160330024916/http://www.jsc.nasa.gov:80/Bios/htmlbios/samokutyaev.html|publisher=[[NASA|NASA]] |accessdate=2019-04-14}}</ref>。
===远征27/28===
[[莫斯科时间|莫斯科时间]]2011年4月5日2时18分（[[北京时间|北京时间]]6时18分），他和[[安德烈·鲍里先科|安德烈·鲍里先科]]、[[罗恩·加兰|罗恩·加兰]]在哈萨克斯坦[[拜科努尔航天发射场|拜科努尔航天发射场]]乘坐俄制[[联盟TMA-21|联盟TMA-21]]宇宙飞船前往[[国际空间站|国际空间站]]<ref>{{cite web |author=耿锐斌 |title=俄加加林号载人飞船飞赴国际空间站 |url=http://news.sciencenet.cn/sbhtmlnews/2011/4/243049.html?id=243049 |publisher=科学时报 |accessdate=2019-04-10 |date=2011-04-05}}</ref>。他此行还带着女儿送给他的[[玩具|玩具]][[狗|狗]]礼物<ref>{{cite web |title=美俄宇航员携手升空 庆祝加加林“飞天”50周年 |url=http://news.sohu.com/20110406/n280147554.shtml |publisher=[[搜狐|搜狐]] |accessdate=2019-04-15 |archiveurl=https://web.archive.org/web/20120303122749/http://news.sohu.com/20110406/n280147554.shtml |archivedate=2012-03-03 |date=2011-04-06}}</ref>。经过2天的绕地飞行后，联盟TMA-21号航天器于[[国际协调时|国际协调时]]4月6日23时9分与国际空间站对接<ref>{{Cite web|url=http://en.rian.ru/science/20110405/163373769.html|title=Russia's Soyuz TMA-21 with new ISS crew launched from Baikonur|publisher=[[俄罗斯新闻社|RIA Novosti]]|date=2011-04-05|accessdate=2011-04-06| archiveurl= https://web.archive.org/web/20110515145336/http://en.rian.ru/science/20110405/163373769.html| archivedate= 2011-05-15|deadurl= no}}</ref>。

莫斯科时间2011年8月3日18时51分（北京时间3日22时51分），他和谢尔盖·沃尔科夫打开空间站[[码头号对接舱|码头号对接舱]]舱门，开始了舱外任务。他们手动发射了“雪松”号微型卫星，并为[[星辰号服务舱|星辰号服务舱]]表面安装了新的激光通信终端设备。任务中，萨莫库佳耶夫负责从[[码头号对接舱|码头号对接舱]]上拆卸起重臂，而沃尔科夫则临时充当“[[起重机|起重机]]驾驶员”。此外，他们还把装有[[微生物|微生物]]和[[蘑菇|蘑菇]][[孢子|孢子]]的[[容器|容器]]从[[码头号对接舱|码头号对接舱]]运至星辰号服务舱，最后以地球为背景，拍摄了[[尤里·加加林|尤里·加加林]]、[[谢尔盖·科罗廖夫|谢尔盖·科罗廖夫]]和[[康斯坦丁·齐奥尔科夫斯基|康斯坦丁·齐奥尔科夫斯基]]3人的肖像照片，以纪念人类[[载人航天|载人航天]]50周年<ref>{{cite web |title=2名俄罗斯宇航员顺利完成6小时太空作业 |url=http://www.chinanews.com/gj/2011/08-04/3233662.shtml |publisher=[[中国新闻网|中国新闻网]] |archiveurl=https://web.archive.org/web/20190415150333/http://www.chinanews.com/gj/2011/08-04/3233662.shtml |archivedate=2019-04-15 |date=2011-08-04}}</ref>。

[[协调世界时|协调世界时]]2011年9月16日0时38分，联盟TMA-21飞船与国际空间站俄罗斯[[探索号实验舱|探索号实验舱]]脱离；16日3时59分，返回舱在哈萨克斯坦[[杰兹卡兹甘|杰兹卡兹甘]]市东南93英里处成功降落<ref>{{cite web|url=https://www.telegraph.co.uk/science/space/8767703/Soyuz-astronauts-land-in-Kazakhstan.html|title=Soyuz astronauts land in Kazakhstan|publisher=[[每日电讯报|The Telegraph]]|date= 2011-09-16|accessdate=2011-09-17}}</ref>。

===远征41/42===
2014年9月26日，他和任务队员乘坐[[联盟TMA-14M|联盟TMA-14M]]载人飞船飞往国际空间站<ref>{{cite web |author=张继业 |title=国际空间站３名宇航员结束任务进入返回舱 |url=http://www.xinhuanet.com/world/2015-03/12/c_1114608612.htm |publisher=新华网 |archiveurl=https://web.archive.org/web/20190414040430/http://www.xinhuanet.com/world/2015-03/12/c_1114608612.htm |archivedate=2019-04-14 |location=[[莫斯科|莫斯科]] |date=2015-03-12}}</ref>。

2014年10月22日，他和指令长[[马克西姆·苏拉耶夫|马克西姆·苏拉耶夫]]穿着俄制[[海鹰舱外航天服|海鹰舱外航天服]]从[[码头号对接舱|码头号对接舱]]出舱。他们拆掉了1台旧地震监测仪和2副[[天线|天线]]，并抛弃到了太空。它们会在[[大气层|大气层]]中迅速烧毁。任务共持续了3小时38分钟<ref>{{cite web|url=http://en.itar-tass.com/non-political/755916|title=Russian cosmonauts complete spacewalk, return to space station|author=[[俄通社-塔斯社|TASS]]|date=2014-10-22}}</ref>。

2014年11月27日[[感恩节|感恩节]]，他和国际空间站5名航天员共同享用了[[火鸡|火鸡]]、甜[[山芋|山芋]]、玉米[[面包|面包]]以及水果[[馅饼|馅饼]]等经过冻干以及恒温等处理过的特制感恩节美食<ref>{{cite web |title=国际空间站公布宇航员感恩节菜单 |url=http://www.cnsa.gov.cn/n6758823/n6759010/c6774362/content.html |publisher=国家航天局 |archiveurl=https://web.archive.org/web/20190414035454/http://www.cnsa.gov.cn/n6758823/n6759010/c6774362/content.html |archivedate=2019-04-14 |date=2014-12-01}}</ref>。

莫斯科时间2015年3月12日上午5时8分（北京时间10时8分），搭载萨莫库佳耶夫、[[叶莲娜·谢罗娃|谢罗娃]]和{{tsl|en|Barry E. Wilmore|巴里·威玛}}的[[联盟TMA-14M|联盟TMA-14M]]号飞船在[[哈萨克斯坦|哈萨克斯坦]]预定区域着陆<ref>{{cite web |title=俄联盟号飞船载三名宇航员成功返回着陆 |url=http://www.xinhuanet.com/world/2015-03/12/c_127573669.htm |publisher=[[新华网|新华网]] |archiveurl=https://web.archive.org/web/20190415143310/http://www.xinhuanet.com/world/2015-03/12/c_127573669.htm |archivedate=2019-04-15 |date=2015-03-12}}</ref>。
