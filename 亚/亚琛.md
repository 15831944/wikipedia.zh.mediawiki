'''亞琛'''（{{lang-de|Aachen}}，{{lang-nl|Aken}}，{{lang-fr|Aix-la-Chapelle}}）是位於[[德國|德國]][[北萊茵-威斯特法倫|北萊茵-威斯特法倫]]州的一個城市，靠近[[比利時|比利時]]與[[荷蘭|荷蘭]]邊境，是德國最西部的城市。

亞琛以[[溫泉|溫泉]]（歐洲中部最熱的溫泉）著名，從西元1世紀起这里就是療養地。這個城市是一個重要的[[鐵路|鐵路]]匯集點，也是[[工業|工業]]與集會中心。經濟以[[食品工業|食品工業]]為主，主要生產[[杏仁糖|杏仁糖]]、[[巧克力|巧克力]]、Printen餅。 此外亦有[[機械|機械]]、[[鐵路設備|鐵路設備]]及[[紡織業|紡織業]]。此外，德國約20%的[[毛織品|毛織品]]在亚琛生產。

== 歷史 ==
這個城市早年由[[凱爾特人|凱爾特人]]與[[羅馬人|羅馬人]]相继統治，具有豐富的歷史見證。人們將亞琛視為[[查理曼|查理曼]]的出生地。在查理曼統治的時期（800年－814年），於亞琛建造宮廷禮拜堂，也就是[[亞琛大教堂|亞琛大教堂]]。由於其無與倫比的歷史地位、建築風格，以及收藏於其內的曠世藝術品，亞琛大教堂成為德國第一座列入[[聯合國教科文組織|聯合國教科文組織]][[世界文化遗产|世界文化遗产]]的建筑瑰寶。

[[查理曼|查理曼]]將這個城市建造成[[卡洛林|卡洛林]]文化的中心，開創了[[中世紀|中世紀]]早期第一個具代表性的文化復興。西元813年至1532年間，有三十二個[[神聖羅馬帝國|神聖羅馬帝國]][[神聖羅馬帝國皇帝|皇帝]]在此加冕。1166年和1215年兩次獲得[[城市特許狀|城市特許狀]]。1250年左右成為帝國的[[自由城市|自由城市]]。不過由於接近[[法國歷史|法國]]邊境，又遠離[[神聖羅馬帝國|神聖羅馬帝國]]中心，1562年皇帝加冕儀式改至[[法蘭克福|法蘭克福]]進行。1668年起成為數次[[和平會議|和平會議]]會址。

[[法國大革命|法國大革命]]期間（1789年－1797年），亞琛為[[法國|法國]]所佔領，1801年正式割讓給[[法國|法國]]，直到1815年[[拿破崙|拿破崙]]戰敗，根据[[维也纳会议|维也纳会议]]，亞琛由[[普魯士王国|普魯士王国]]管理。這個城市在[[第二次世界大战|第二次世界大战]]末期，成為被盟軍攻佔的第一座德國城市（1944年10月20日），並受創於強大的空襲與陸戰。1966年，[[亞琛|亞琛]]開始進行重建。1999年1月31日為[[亞琛大教堂|亞琛大教堂]]成立一個歐洲基金委員會，在這委員會中，許多國家的市長答應支持大教堂的維護。這個源於[[卡洛林|卡洛林]]時代的建築紀念物，在1978年由[[UNESCO|UNESCO]]列為[[世界文化遺產|世界文化遺產]]。目前[[亞琛|亞琛]]的人口在2001年統計約為24萬4千人。

亞琛國際性的[[查理曼獎|查理曼獎]]於1950年開始舉辦，是最老也是最有名的獎項，頒給在行為和研究機關表現優異，而且對[[歐洲|歐洲]]國家統一有貢獻者，2000年[[美國總統|美國總統]][[比尔·克林顿|比尔·克林顿]]也曾在此受頒獎項。

世界知名單練和[[马术|马术]]競技活動「CHIO」也是亞珅的民間節日，每年的八月都會舉辦活動，吸引來自世界各地愛馬的朋友來此聚會。

== 景點 ==
值得參觀的景點如[[亞琛市政廳|市政廳]]（Rathaus）、及[[亞琛大教堂|亞琛大教堂]]（Aachener Dom）。市政廳在1353年從[[查理大帝|查理大帝]]（或{{link-en|卡爾大帝|Karl der Große}}）皇宫的廢墟中建起。亞琛大教堂之祭壇建於約13至14世紀左右，其中還有查理大帝的寶座及墳墓。

[[報紙博物館|報紙博物館]]是為了紀念名為Paul Julius von reuter的德國記者而設置的，其中收集了約三十種語言、超過十二萬的報紙與雜誌。 

1870年[[亞琛工業大學|亞琛工業大學]]（RWTH Aachen）設立，孕育出不少傑出的工程界人士。

[https://web.archive.org/web/20050824133827/http://museen.aachen.de/content/mus/slm/index.html Suermondt ludwig美術館]為[[威尼斯|威尼斯]]式[[文藝復興|文藝復興]]建築，在十九世紀中葉為著名的亞琛城市官邸，本為Cassalette住處，其中保留著豪華的入口階梯玄關，裝飾屋頂，及壁畫。1992年到1994年，建築師Bussmann與Haberer在舊建築物一旁設計新的藝術中心，呈現新與舊建築藝術的融合。美術館現為美術展示與美術教育用之空間。

位在[[亞琛|亞琛]]市中心外北端的[[盧斯山|盧斯山]]（Lousberg），不只有著有趣的傳說，亦是值得踏青的地方。亞琛溫泉也是相當有名。

== 嘉年華會 ==
「亚琛位於[[萊茵河|萊茵河]]畔」，此一宣稱並不為眾人所熟悉，因該城距離萊茵河尚有75公里，但當人們認識亚琛嘉年華，就不會對這個地理上的歸屬性感到懷疑，因為除了[[科隆|科隆]]、[[杜塞尔多夫|杜塞尔多夫]]和[[美因茲|美因茲]]之外，亚琛也被視為一個信徒的堡壘，而且早在19世紀，嘉年華的第一個協會就在這裡成立。嘉年華的格言是跟朋友玩笑逗趣，這裡的居民以唱歌、跳舞、喝酒來歡度這個節日。在11月11日的11點11分開始到2月的綠色星期四（grüner Donnerstag），會有超過五十個嘉年華聚會在亞琛舉行。

每年的復活節的前一個星期到復活節後的一個星期，在阿亨的 Bendplatz 廣场會有各式各樣吸引人的活動和玩樂， 例如[[雲霄飛車|雲霄飛車]]、[[碰碰車|碰碰車]]、迷宮等這些遊戲，除此之外還賣一些應節食品，吸引了很多住在亞琛當地和周圍城市的人前來。
