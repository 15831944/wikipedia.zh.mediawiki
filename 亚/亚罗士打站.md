'''亚罗士打站'''（{{lang-ms|Stesen Kereta Api Alor Setar}}）是[[马来西亚|马来西亚]][[吉打州|吉打州]][[亚罗士打|亚罗士打]]市的火车站，属于[[KTM电动列车服务|KTM电动列车服务]]（ETS）和[[KTM通勤铁路北马区|KTM通勤铁路北马区]]的其中一站。[[马来亚铁道公司|马来亚铁道公司]]是该车站的主要经营者。

==主要路线==
*'''亚罗士打'''至[[吉隆坡|吉隆坡]]至[[新加坡|新加坡]]
*'''亚罗士打'''至[[曼谷|曼谷]]
