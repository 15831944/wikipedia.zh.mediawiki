'''亚历山大·罗弗斯'''（{{lang-de|'''Alexander Roovers'''}}，{{bd|1987年|3月17日}}），[[德國|德國]]男子[[羽毛球|羽毛球]]運動員。

== 簡歷 ==
2017年4月，亚历山大·罗弗斯出戰[[2017年克羅地亞羽毛球國際賽|克羅地亞羽毛球國際賽]]，在準决赛中以0比2 （18-21、18-21）不敌赛会头号种子、克罗埃西亚的[[兹沃尼米尔·杜尔金雅克|兹沃尼米尔·杜尔金雅克]]。同时，他与[[Patrick_Scheiel|Patrick Scheiel]]在男双赛事中以0比2 （24-26、15-21）惜败于跨国组合 （克罗埃西亚 :伊戈尔·吉姆布尔 / 泰国 ：图万那格森·沙玛差）。同年8月，他出戰[[2017年懷卡托羽毛球國際賽|懷卡托羽毛球國際賽]]，在决赛中以0比2 （12-21、16-21）不敌赛会头号种子、荷兰的[[埃里克·迈斯|埃里克·迈斯]]，赢得了男單亞軍。年尾12月，他出戰[[2017年愛爾蘭羽毛球公開賽|愛爾蘭羽毛球公開賽]]，在决赛中以2比0 （21-19、21-11）击败了爱尔兰的[[阮日|阮日]]，首次赢得了个人国际赛的男单冠军。
