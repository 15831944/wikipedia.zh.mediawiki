'''小亚历山大·梅格斯·“艾尔”·黑格'''（{{lang-en|'''Alexander Meigs "Al" Haig Jr. '''}}，{{bd|1924年|12月2日|2010年|2月20日|catIdx=Haig, Alexander Meigs}}），出生于[[宾夕法尼亚州|宾夕法尼亚州]][[费城_(宾夕法尼亚州)|费城]]，是[[美国|美国]]政治家、外交官及[[美國陸軍|美國陸軍]]退役[[上將|上將]]，曾任[[白宫幕僚長|白宫幕僚長]]、[[美国国务卿|美国国务卿]]及陸軍副參謀長。他曾经辅佐过三位美国总统。他以成功的军旅生涯、杰出的外交才能和他的一句颇具争议的言论而闻名于世。黑格曾是总统信任的顾问、杰出的外交家。

== 职业生涯 ==
[[File:General_Alexander_M._Haig,_Jr.jpg|thumb]]
黑格年輕時就讀美國[[聖母大學|聖母大學]]，但讀兩年後即轉學念[[西點軍校|西點軍校]]，後來又取得哥倫比亞大學企管碩士和[[喬治城大學|喬治城大學]]國際關係學士。

黑格曾出任国家安全顾问[[亨利·基辛格|亨利·基辛格]]的助手。他为巴黎和平协议铺平道路的一系列谈判中起到关键作用，该协议的签署结束了[[越战|越战]]。黑格还为尼克松总统1972年历史性的中国之行打前站。

黑格出任[[理查·尼克松|理查·尼克松]]总统的白宮幕僚長，并且据信在尼克松总统1974年辞职、副总统[[傑拉德·福特|傑拉德·福特]]接任总统职而展开的谈判中发挥了作用。

黑格重返军队后出任驻欧美军总司令与北约盟军最高司令。由于对[[伊朗人质危机|伊朗人质危机]]的处理问题，在[[吉米·卡特|吉米·卡特]]总统执政期间，黑格辞职。退役后的黑格将军进入私营企业，但不久后，他又重返政坛，出任[[朗奴·列根|朗奴·列根]]总统的国务卿。黑格的国务卿生涯历时一年零五个月，在那期间，他以对[[苏联|苏联]]强硬立场而闻名，并陷入一场跟其他政府高官的争斗。

黑格在任期間，还在英国政府与阿根廷政府之间展开穿梭外交，为阻止两国为[[福克兰群岛|福克兰群岛]][[马岛战争|开战]]，但他没能成功。

== 雷根遇刺案越權爭議 ==
黑格最著名的可能是他在1981年担任[[里根|里根]]总统的国务卿时所说的一句话。当里根遇刺住院时，當時由於副總統[[老布希|老布希]]正在外訪，黑格在全国电视广播上说，现在由他来主事。他的这句话看起来属于违反美国宪法的继位顺序。黑格后来写到，他当时“措辞不当”。其後他於1982年辭職，結束1年半的国务卿生涯。

按照[[美国宪法|美国宪法]]，[[美國總統繼任順序|总统权力的继承顺序]]依次是[[美国副总统|副总统]]、[[美国众议院议长|众议院议长]]、[[參議院臨時議長|参议院多数党中资历最深的参议员]]，然后才是国务卿。

== 卸任國務卿后 ==
卸任国务卿後在私营企业工作，并经常就外交政策发表演讲。

==去世==
2010年2月20日他在[[马里兰州|马里兰州]][[巴尔迪摩|巴尔迪摩]]的[[约翰·霍普金斯医院|约翰·霍普金斯医院]]里因感染引发的并发症病逝。他的遗孀与他结俪60年，两人育有三名子女，八名孙辈。

== 评价 ==
[[美国总统|美国总统]][[巴拉克·奥巴马|巴拉克·奥巴马]]在一份书面声明中将黑格称作一位为国家贡献卓越的伟人。国务卿[[希拉里·克林顿|希拉里·克林顿]]发表声明说，她听到黑格去世的消息深感悲痛。
