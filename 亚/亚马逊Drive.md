'''Amazon Cloud Drive'''，是[[Amazon.com|Amazon]]旗下的个人[[网络硬盘|网络硬盘]]服务。<ref>{{cite web |url=http://www.digitaltrends.com/mobile/amazon-launches-cloud-drive-digital-locker-cloud-player/ |title=Amazon launches Cloud Drive 'digital locker,' Cloud Player |last=Couts |first=Andrew |date=2011-03-29 |work=Digital Trends}}</ref> 

== 概况 ==
[[Amazon.com|Amazon]]于2011年3月29日向其会员推出这项服务。Amazon Cloud Drive提供图片、文档、视频等各种内容的存储，并开发有网页版、[[Windows|Windows]]、[[Android|Android]]和[[iOS|iOS]]设备的客户端，不同客户端之间可同步存储内容。[[Android|Android]]和[[iOS|iOS]]系统的客户端命名为Cloud Drive Photos，仅可显示、存储、同步图片和视频内容。

Amazon Cloud Drive初始免费容量为5[[Gigabyte|GB]]，用户可按年为单位，每[[Gigabyte|GB]]0.5[[美元|美元]]的价格，选择将容量扩充至20GB、50GB、100GB、200GB、500GB、1000GB的扩容计划。<ref>{{cite web |url=https://www.amazon.com/clouddrive/learnmore?ref_=cd_home_pricing#plans-section |title=Amazon Cloud Drive   Learn More |publisher=Amazon.com |accessdate=2014-07-31}}[[美国|美国]]地区服务内容，[[中华人民共和国|中国]]地区（[[亚马逊中国|亚马逊中国]]）目前不提供扩容方案。</ref> {{Tsl|en|Fire Phone}}和[[Amazon_Prime|Amazon Prime]]用户拥有无限免费容量的照片存储空间。

[[Amazon.com|Amazon]]不同国家站点的Cloud Drive系统互相独立，所有存储数据均使用[[Amazon_S3|Amazon S3]]服务，储存和传输过程则进行[[高级加密标准|AES]]-128的[[TLS|TLS]]加密。

== Kindle个人文档服务 ==
Amazon Cloud Drive储存[[Kindle|Kindle]]用户通过电子邮件发送的个人文档。个人文档统一储存在“我的发送至Kindle文档”文件夹中，并会占用Cloud Drive容量。<ref>{{cite web |url=http://www.amazon.cn/gp/help/customer/display.html?nodeId=200767340 |title=Kindle个人文档服务 |publisher=[[亚马逊中国|亚马逊中国]] |accessdate=2015-01-05}}</ref>用户在亚马逊网站上购买的[[Kindle|Kindle]][[电子书|电子书]]则储存于独立空间（Kindle图书馆），与Cloud Drive无关。
