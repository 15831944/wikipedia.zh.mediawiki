'''亚历山大·尤里耶维奇·卡列里'''（{{lang-ru|Александр Юрьевич Калери|Aleksandr Yuriyevich Kaleri}}，{{bd|1956年|5月13日|}}）是俄罗斯第73位[[宇航员|宇航员]]，世界第265位宇航员<ref>{{cite web |title=Позывные космонавтов |url=http://www.gctc.ru/main.php?id=156 |publisher=GCTC |archiveurl=https://web.archive.org/web/20190421055051/http://www.gctc.ru/main.php?id=156 |archivedate=2019-04-21 |language=ru}}</ref>。他曾五次前往太空，五次进行太空行走任务。
==早年==
1956年5月13日出生于苏联[[拉脱维亚苏维埃社会主义共和国|拉脱维亚苏维埃社会主义共和国]][[尤爾馬拉|尤爾馬拉]]。1973年毕业于尤尔马拉市第5中学，1979年毕业于[[莫斯科物理技术学院|莫斯科物理技术学院]]飞行动力学专业，毕业后在[[科罗廖夫能源火箭航天集团|“能源”科研生产联合公司]]工作，担任设计工程师。1983年获得莫斯科物理技术学院“液体、气体和[[等离子体|等离子体]]”[[力学|力学]]硕士学位<ref>{{cite web |title=Биографии космонавтов СССР и РФ |url=http://www.gctc.ru/main.php?id=178 |publisher=GCTC |accessdate=2019-04-21}}</ref>。 
==宇航员生涯==
1984年4月入选“能源”集团宇航员候选人，1985年至1986年在[[加加林宇航员培训中心|加加林宇航员培训中心]]接受基础太空飞行训练，并获得了测试宇航员资格。他有22小时的[[L-39信天翁|L-39信天翁]]教练机飞行经验<ref name=nasabio>{{cite web |title=Aleksandr Kaleri|url=https://web.archive.org/web/20100527212453/http://www.jsc.nasa.gov/Bios/htmlbios/kaleri-ay.html|date=2009 |publisher=NASA |accessdate=2019-04-20}}</ref>。 

1987年4月至5月，他参与了[[联盟TM-4|联盟TM-4]]飞船“和平号空间站远征”备用任务人员培训。1988年1月至3月参与了“和平号空间站第4次远征”随航工程师培训；1990年5月至11月参与了“和平号空间站第8次远征”任务随航工程师培训；1991年1月至4月参与了“和平号空间站第9次远征”随航工程师培训，1991年5月18日作为[[联盟TM-12|联盟TM-12]]飞船中[[谢尔盖·康斯坦丁诺维奇·克里卡列夫|克里卡列夫]]的替补；1991年5月至1992年2月参与了第9和第10次远征随航工程师培训<ref name=ru>{{cite web |title=Юрий Иванович Онуфриенко |url=https://www.energia.ru/en/iss/iss04/onufrienko.html|publisher=[[科罗廖夫能源火箭航天集团|SP Korolev RSC Energia]] |accessdate=2019-04-21}}</ref>。
===和平号空间站===
1992年、1996年和2000年三次作为随航飞行工程师前往[[和平号空间站|和平号空间站]]执行任务<ref name=ru/>。

===远征8===
2003年10月18日5时38分，[[联盟-FG|联盟-FG]]号运载火箭在[[哈萨克斯坦|哈萨克斯坦]][[拜科努尔发射场|拜科努尔发射场]]发射成功，载有3名宇航员的俄[[联盟TMA-3|联盟TMA-3]]载人飞船顺利送入预定轨道。这是自2月1日[[哥伦比亚号航天飞机灾难|哥伦比亚号航天飞机灾难]]以来俄罗斯向国际空间站第二次运送宇航员。其中卡列里任任务指令长，和美国宇航员[[迈克尔·福尔|迈克尔·福尔]]是常驻成员，[[西班牙|西班牙]]籍短期考察员[[佩德罗·杜克|佩德罗·杜克]]任随航工程师<ref>{{cite web |author=董映璧 |title=俄联盟号飞船升空 第八批宇航员奔赴国际空间站 |url=http://www.people.com.cn/GB/keji/1056/2142685.html |publisher=科技日报 |accessdate=2019-04-21 |archiveurl=https://web.archive.org/web/20031030233958/http://www.people.com.cn/GB/keji/1056/2142685.html |archivedate=2003-10-30 |location=[[莫斯科|莫斯科]] |date=2003-10-18}}</ref>。

2004年1月5日，[[美国国家航空航天局|美国国家航空航天局]]地面控制中心告诉卡列里和福尔，空间站中的[[气压|气压]]出现了微弱的下降。但仍处于安全状态，因为一昼夜间气压下降幅度每[[平方英寸|平方英寸]]并没有超过0.04[[磅|磅]]<ref>{{cite web |author1=陈永辉 |title=国际空间站气压趋于稳定 宇航员安全无恙 |url=http://tech.sina.com.cn/other/2004-01-08/0842279146.shtml |publisher=新浪科技 |accessdate=2019-04-21 |archiveurl=https://web.archive.org/web/20050506214719/http://tech.sina.com.cn/other/2004-01-08/0842279146.shtml |archivedate=2005-05-06 |date=2004-01-08}}</ref>。经过几天的排查，他们终于发现防止舷舱多层[[玻璃|玻璃]]之间的水汽，原来是剩余气压的[[排水|排水]]软管出现了裂缝。他们很快修复了软管裂缝<ref>{{cite web |title=国际空间站“漏气”了！ |url=http://st.zjol.com.cn/kjjsb/201809/t20180901_8158580.shtml |publisher=[[浙江|浙江]]科技新闻网 |archiveurl=https://web.archive.org/web/20190421131428/http://st.zjol.com.cn/kjjsb/201809/t20180901_8158580.shtml |archivedate=2019-04-21 |date=2018-09-01}}</ref>。

2004年2月26日，他和福尔在[[舱外活动|舱外活动]]时，太空服内[[温度|温度]]控制装置出现故障，他不得不提前结束首次在无人留守国际空间站的状态下进行太空行走的“[[冒险|冒险]]之旅”<ref>{{cite web |title=宇航员提前结束出舱行走 |url=http://www.taikongmedia.com/Item/Show.asp?m=1&d=13316 |archiveurl=https://web.archive.org/web/20190421124810/http://www.taikongmedia.com/Item/Show.asp?m=1&d=13316 |archivedate=2019-04-21 |date=2004-03-01}}</ref>。在这次持续3时55分的任务中，他们把一个“实验人”安放到了[[星辰号服务舱|星辰号服务舱]]外，以测量空间站外[[辐射|辐射]]强度与辐射量<ref>{{cite journal |author=孟奂 |title=空间站外的“实验人” |journal=太空探索 |date=2004年 |issue=5 |pages=18 |issn=1009-6205}}</ref>。

2003年12月和2004年4月，他通过保密渠道与委托人通话，在[[国家杜马|杜马]]选举及[[2004年俄罗斯总统选举|总统选举]]中投上了一票<ref>{{cite web |author=董爱波 |title=俄宇航员从太空投票参加杜马选举 |url=http://news.ifeng.com/world/Europe/200712/1202_1395_318766.shtml |publisher=新华网 |archiveurl=https://web.archive.org/web/20190421125302/http://news.ifeng.com/world/Europe/200712/1202_1395_318766.shtml |archivedate=2019-04-21 |date=2007-12-02}}</ref>。

===远征25/26===
2010年10月7日23时10分，载有卡列里、[[奥列格·斯克里波奇卡|奥列格·斯克里波奇卡]]和[[斯科特·凯利|斯科特·凯利]]的[[联盟TMA-01M|联盟TMA-01M]]宇宙飞船从哈萨克斯坦境内的拜科努尔发射基地发射升空，飞往国际空间站。8日凌晨0时02分与空间站成功对接<ref>{{cite web |title=中国画首度登陆国际空间站 |url=https://finance.qq.com/a/20110714/003689.htm |publisher=雅昌艺术网 |archiveurl=https://web.archive.org/web/20190421140607/https://finance.qq.com/a/20110714/003689.htm |archivedate=2019-04-21 |date=2011-07-14}}</ref>。

2011年3月16日7时54分，卡列里、[[奥列格·斯克里波奇卡|斯克里波奇卡]]和凯利所乘坐的联盟TMA-01M载人飞船返回舱成功降落在哈萨克斯坦[[阿尔卡雷克|阿尔卡雷克]]市以北85公里处的草原上<ref>{{cite web |author1=耿锐斌 |title=国际空间站3名宇航员乘俄联盟飞船返回地球 |url=http://news.66wz.com/system/2011/03/16/102459344.shtml |archiveurl=https://web.archive.org/web/20190421130309/http://news.66wz.com/system/2011/03/16/102459344.shtml |archivedate=2019-04-21 |date=2011-03-16}}</ref>。
