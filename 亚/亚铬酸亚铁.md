'''亚铬酸亚铁'''是[[离子化合物|离子化合物]]，[[化学式|化学式]]为FeCr<sub>2</sub>O<sub>4</sub>。

== 制备 ==
* 自然界中存在于[[铬铁矿|铬铁矿]]，虽然有许多杂质。

* 高温灼烧[[三氧化二铬|三氧化二铬]]与[[氧化铁|氧化铁]]：
:<math>\mathsf{FeO + Cr_2O_3 \ \xrightarrow{1600^oC}\ Fe(CrO_2)_2 }</math>

== 物理性质 ==
黑褐色[[立方晶系|立方晶体]]，[[空间群|空间群]]'''''F d3m'''''，其中''a'' = 0.8374 nm，Z = 8。

== 化学性质 ==
* 与浓[[盐酸|盐酸]]反应：
:<math>\mathsf{Fe(CrO_2)_2 + 8HCl \ \xrightarrow{}\ 2CrCl_3 + FeCl_2 + 4H_2O }</math>

* 与热的浓[[硝酸|硝酸]]反应：
:<math>\mathsf{Fe(CrO_2)_2 + 10HNO_3 \ \xrightarrow{}\ 2Cr(NO_3)_3 + Fe(NO_3)_3 + NO_2\uparrow + 5H_2O }</math>

* 在不活泼气体保护下，缓慢溶于碱中：
:<math>\mathsf{Fe(CrO_2)_2 + 6NaOH + 4H_2O \ \xrightarrow{\tau, N_2}\ 2Na_3[Cr(OH)_6] + Fe(OH)_2\downarrow }</math>

* 在空气中反应更快：
:<math>\mathsf{4Fe(CrO_2)_2 + O_2 + 24NaOH + 14H_2O \ \xrightarrow{}\ 8Na_3[Cr(OH)_6] + 4FeO(OH)\downarrow }</math>

* 被[[碳|碳]]（[[焦炭|焦炭]]）还原为铬和铁：
:<math>\mathsf{Fe(CrO_2)_2 + 4C \ \xrightarrow{1100-1200^oC}\ [Fe + 2Cr] + 4CO }</math>
