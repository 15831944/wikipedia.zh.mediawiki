'''亞洲足球聯盟'''（简称'''亚協'''或'''亚足联'''；{{lang-en|'''A'''sian '''F'''ootball '''C'''onfederation，[[縮寫|縮寫]]：'''AFC'''}}）是負責管理[[亞洲|亞洲]]區[[足球|足球]]事務的[[體育|體育]][[组织_(社会学)|組織]]，舉辦各項[[國家|國家]]級及[[足球會|球會]]級賽事，包括協助[[國際足協|國際足協]]舉行世界盃外圍賽及4年一度的[[亞洲盃|亞洲盃]]。於1954年5月8日在[[菲律賓|菲律賓]]的[[馬尼拉|馬尼拉]]成立，創會成員包括[[阿富汗|阿富汗]]、[[緬甸|緬甸]]、[[中華民國|中華民國]]、[[香港|香港]]、[[印度|印度]]、[[印度尼西亞|印度尼西亞]]、[[日本|日本]]、[[南韓|南韓]]、[[巴基斯坦|巴基斯坦]]、[[菲律賓|菲律賓]]、[[新加坡|新加坡]]及[[越南|越南]]，首任會長及秘書長由香港人[[羅文錦|羅文錦]]及[[李惠堂|李惠堂]]擔任。總部現設在[[馬來西亞|馬來西亞]]的[[吉隆坡|吉隆坡]]。

亞洲足協每年舉辦大型的足球賽事，較為人認識的有[[亞洲盃足球賽|亞洲盃]]及2003年改制[[亞足聯冠軍聯賽|亞足聯冠軍聯賽]]，前者是最高等級的國家級賽事，早在1956年舉辦；後者是最高等級的球會級賽事，在新制定的賽事模式與[[歐洲冠軍聯賽|歐洲冠軍聯賽]]無異，之前由[[亞洲冠軍盃|亞洲冠軍盃]]及[[亞洲盃賽冠軍盃|亞洲盃賽冠軍盃]]合併而成。

2004年亞洲足協推出名為「亞洲展望」（Vision Asia）的足球計劃，目的是統一亞洲區在有關足球運動的水平，包括管理及運動科學，包括同等級別的國家安排在同一級別比賽，首先推行的是為改革亞洲聯賽冠軍盃首先將14個實力最強的亞洲國家，稱為「成熟國家」，所屬球會可以參加[[亞足聯冠軍聯賽|亞足聯冠軍聯賽]]；隨後的14個的亞洲國家，稱為「發展中國家」，所屬球會則可以參加[[亞洲足聯盃|亞洲足聯盃]]，最後其他足球發展比較落後的國家稱為「冒起國家」，他們的所屬球會參加[[亞足聯主席盃|亞足聯主席盃]]。

現時的亞洲足球大致可分為兩大勢力——[[東亞|東亞]]及[[西亞|西亞]]，東亞包括有[[日本國家足球隊|日本]]、[[南韓國家足球隊|南韓]]、[[澳大利亚国家足球队|澳大利亚]]及中國等（2006年開始來自[[大洋洲|大洋洲]]的[[澳大利亚国家足球队|澳大利亚]]新加入[[亞洲足協|亞洲足協]]），而西亞則有[[伊拉克|伊拉克]]、[[約旦|約旦]]、[[伊朗國家足球隊|伊朗]]、[[沙地阿拉伯國家足球隊|沙烏地阿拉伯]]、[[巴林國家足球隊|巴林]]、[[烏茲別克國家足球隊|烏茲別克]]及[[阿聯酋國家足球隊|阿聯酋]]。到现在为止，亚足联共有47个会员协会，包括46个正式会员和1个准会员（北马里亚纳群岛）<ref>{{Cite web |url = http://www.the-afc.com/about-afc |title = About AFC |publisher = AFC官网 |language = en}}</ref>，其中仅[[北马里亚纳群岛足球协会|北马里亚纳群岛]]尚未加入国际足联。

雖然亞洲幅員廣大，人口眾多，但基於過去傳統歷史和文化因素，相對於世界其他洲，亞洲對於足球熱衷程度在世界各洲之中敬陪末座，也讓亞足聯會員國的總體足球實力僅優於[[大洋洲足球联合会|大洋洲]]，遠不如[[欧洲足球联合会|歐洲]]和[[南美洲足球联合会|南美洲]]，亦輸給[[中北美洲及加勒比海足球协会|中北美洲]]和[[非洲足球联合会|非洲]]；一些重大事件也嚴重打擊了亞洲的足球發展。因此在很多重大比賽當中，諸如[[奧運|奧運]]和[[世界杯足球赛|世界盃]]，亞洲球隊成績也不盡理想。

== 亚足联执委会 ==
{{main|亚足联执委会}}
亚足联执委会由亚足联大会选举产生。西亚、东亚、东南亚、中南亚各有一个副主席和三个执行委员名额，此外亚足联大会选举还会产生两名国际足联执委、一名国际足联副主席、一名亚足联主席。亚足联主席自动成为国际足联执委。
