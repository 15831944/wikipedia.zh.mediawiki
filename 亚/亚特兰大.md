'''亚特兰大'''（{{lang-en|'''Atlanta'''}}）是[[美国|美国]][[佐治亚州|佐治亚州]]首府及最大城市，也是[[富爾頓縣_(喬治亞州)|富尔顿县]]的县政府驻地。人口456,002人（2014年）<ref>http://www.census.gov/popest/data/index.html</ref>，而正在快速发展的亚特兰大都會区拥有人口5,884,736<ref>https://en.wikipedia.org/wiki/Atlanta#cite_note-United_States_Census_Bureau-7</ref>人，是美国第9大都市区。

作为一个铁路枢纽，亚特兰大的发展始于19世纪早期，在[[南北战争|南北战争]]时被摧毁，但在當选为州府后迅速重建。20世纪，它是美国民权运动的中心，并举办了[[1996年夏季奧林匹克運動會|1996年夏季奧林匹克運動會]]。

==歷史 ==
亞特蘭大在1837年建城，很快成為南方重要交通樞紐。[[美國內戰|美國內戰]]期間為[[美利堅聯盟國|美利堅聯盟國]]重要城市，1864年在此與[[合眾國|合眾國]][[謝爾曼|謝爾曼]]軍團進行長達一年的圍城戰，南軍戰敗後全市幾乎全毀，但在[[重建時期|重建時期]]快速恢復。20世紀人口在[[喬治亞州|喬治亞州]]首府遷入後快速增加，除[[非裔美國人|非裔美國人]]外，亞裔與西班牙語裔美國人也大量遷入。<ref>http://www.atlantahistorycenter.com/</ref>

=== 氣候 ===
亚特兰大属典型的[[副热带湿润气候|副热带湿润气候]]，四季分明，气温变化相对和缓，全年降水分配均匀，代表美国东南部内陆地区的气候。冬季温和，微潮，时而偏向温暖，日最高气温低于{{convert|0|C}}的平均日数为1.7天，日最低气温低于{{convert|0|C}}的平均日数为40天，低于{{convert|-5|C}}的平均日数为8.3天；夏季相对炎热潮湿，日最高气温超过{{convert|30|C}}的日数年均有88天，超过{{convert|35|C}}的有11天。<ref name = "NWS Peachtree City, GA"/>最冷月（1月）均温{{convert|43.3|F|1|disp=flip}}，极端最低气温{{convert|−9|F|0|disp=flip}}（1899年2月13日）。<ref name = "NWS Peachtree City, GA"/>最热月（7月）均温{{convert|80.2|F|1|disp=flip}}，极端最高气温{{convert|106|F|0|disp=flip}}（2012年6月30日）。<ref name = "NWS Peachtree City, GA"/>无霜期平均为239天（3月22日至11月15日）。<ref name = "NWS Peachtree City, GA"/>年均降水量约{{convert|49.7|in|sigfig=3|disp=flip}}；年极端最少降水量为{{convert|31.80|in|disp=flip}}（1954年），最多为{{convert|71.45|in|disp=flip}}（1948年）。<ref name = "NWS Peachtree City, GA"/>年均降雪量为{{convert|2.9|in|cm|disp=flip}}，<ref name = "NWS Peachtree City, GA"/>但许多冬季的积累降雪量微少以致难以测量，1935–36年的降雪量最多，积累降雪量为{{convert|10.9|in|cm|disp=flip}}。<ref name = "NWS Peachtree City, GA"/>

=== 主要景点 ===
* [[可口可乐|可口可乐]]世界：位于奥林匹克公园内，距[[CNN|CNN]]电视中心仅数百米。占地面积不大，但是内容非常丰富，可以看到可口可乐历史上的相关产品与历史图片，此外，可以参观生产线以及观摩可口可乐出资拍摄的可乐主题动画片，其中，以4D影院播放的短片最为精彩，效果几乎可以乱真，是可口可乐世界里最受欢迎的项目。在出口处，游客可免费品尝可口可乐研发的近100种不同口味的饮料，它们大部分都在世界范围内分地域销售，非常难得才能见到，因此，能够在亚特兰大品尝全世界的可口可乐产品也非常有吸引力。
* 喬治亞水族館：该水族馆是世界最大的水族馆之一，票价26美元。水族馆分上下两层，游客可以看到白鲸、虎鲨等珍稀海洋动物，另外，可以和水母、海星等生物作亲密接触。
* [[瑪格麗特·米切爾|瑪格麗特·米切爾]]之家：包含两个项目，一个是《[[乱世佳人|乱世佳人]]》博物馆，一个是瑪格麗特·米切爾的故居，票价13美元。在博物馆，可以欣赏到瑪格麗特·米切爾生平纪录片，以及《乱世佳人》的一些相关历史图片资料。瑪格麗特·米切爾的故居曾在大火中付于一旦，如今的故居是重新修缮过的，但内部装潢仍旧复原了她居住时的样貌。
* [[石山|石山]]公园：南北战争时的著名遗迹，离亚特兰大市中心约有40分钟车程。石山公园是市民周末出游的去处，内有小型的主题公园，还保留了上世纪的火车。
* [[CNN|CNN]]电视中心：位于亚特兰大会展中心对面，也是市内小有名气的购物中心。电视中心一楼是用餐以及售卖小商品的地方，游客可在一楼购买該电视台的参观券，票价13美元。整个参观过程约55分钟，会有专职人员带领你进入电视台内部，了解CNN的运作方式，并且你还可以坐在模拟新闻主播台前播报新闻，部分游览区域不得拍照。

== 交通 ==
=== 轨道交通系统 ===
亞特蘭大捷运（MARTA - Metropolitan Atlanta Rapid Transit Authority）自1972年开始营运，目前拥有700辆巴士，150条巴士路线，总共行驶哩程2400公里。另外有240辆重型捷运电联车，行驶于36个捷运站之间，总哩程60公里。在奥运時期当局亦設計了讓捷运负荷多數访客的計畫，可将平日运输量达五十万人次的捷运，在奥运期间提升二倍至三倍。
MARTA使用[[微風卡|微風卡]](Breeze Card)作為代幣支付工具。公車與地鐵的報站系統採[[英語|英語]]與[[西班牙語|西班牙語]]雙語。
捷运设计原是为每日工作来往的人潮所用，因此行驶间隔不密集，每十五分钟一班。捷运票价一般单张2.5美元。转换至不同路线捷运或巴士可持转换证免费搭乘，转换以一次为限。为了鼓励大家將公共运输做為主要交通工具，减少私人交通工具的使用，因此亦有不少优惠措施。例如：

* 一日券：无限搭乘次数，票价為9美元
* 二日券：无限搭乘次数，票价為14美元
* 三日券：无限搭乘次数，票价為16美元
* 四日券：无限搭乘次数，票价為19美元
* 周票：无限搭乘次数，票价為23.75美元
* 月票：无限搭乘次数，票价為95美元
