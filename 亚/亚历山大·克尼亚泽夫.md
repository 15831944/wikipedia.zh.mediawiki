'''亚历山大 克尼亚泽夫''' （俄語：Александр Князев, 1976年7月19日－）俄罗斯晶体化学学科专家、物理化学学科专家<ref>{{Cite web|title = УНЦ "Химия": Руководство|url = http://www.unn.ru/chem/leadership.php|website = www.unn.ru|access-date = 2016-01-24}}</ref>。下诺夫哥罗德国立大学教授、化学系副系主任（主管科研工作）、化学研究所副所长（主管科研工作）、基础无机化学新材料相关课题博士生导师。1998年毕业于下诺夫哥罗德国立大学，2000年获得副博士学位，2009年获得化学博士学位<ref>{{Cite web|title = Князев Александр Владимирович - Диссертации - Известные ученые|url = http://www.famous-scientists.ru/list/3306|website = www.famous-scientists.ru|access-date = 2016-01-24}}</ref>，2011年受聘为教授。


==履历==
于国内外知名刊物发行220余篇学术文章，主要著有三本专业领域书籍及三本专业教学参考书。赫希索引第13位（2019年1月，Web of Science网站）<ref>{{Cite web|title = Alexander Knyazev E-7898-2013 - ResearcherID.com|url = http://www.researcherid.com/rid/E-7898-2013|website = www.researcherid.com|access-date = 2016-01-24}}</ref>。2015年俄罗斯国际化学技术峰会（RCCT-2015）副主席<ref>{{Cite web|title = RCCT-2015 {{!}} Nizhni Novgorod 22–26 June 2015|url = http://www.rcct2015.unn.ru|website = www.rcct2015.unn.ru|access-date = 2016-01-24}}</ref>。
