'''亚历山大·亚历山德罗维奇·米苏尔金'''（{{lang-ru|Алекса́ндр Алекса́ндрович Мису́ркин}}，{{bd|1977年|9月23日|}}）是[[俄罗斯|俄罗斯]][[宇航员|宇航员]]。

==早年==
1994年毕业于[[奧廖爾|奧廖爾]]市第一中学。之后在{{tsl|en|Kacha Higher Military Aviation School of Pilots|卡恰军事航空学校}}学习，1998年9月开始在[[阿爾馬維爾_(克拉斯諾達爾邊疆區)|阿尔马维尔]]军事航空学校学习，1999年10月以优异成绩获得飞行工程师学位<ref name=bio/>。
==宇航员生涯==
2006年10月，他成为宇航员候选人，开始接受基础培训，2009年6月2日以优异成绩通过考验，6月9日获得实验宇航员资格<ref name=bio>{{cite web |title=Biographical Data - Aleksandr Aleksandrovich Misurkin |url=https://web.archive.org/web/20170414153552/https://www.jsc.nasa.gov/Bios/htmlbios/misurkin.html |publisher=[[NASA|NASA]] |accessdate=2019-04-14}}</ref>。
===远征35/36===
[[莫斯科时间|莫斯科时间]]2013年3月29日0时43分（[[北京时间|北京时间]]29日4时43分），载有米苏尔金等宇航员的[[联盟TMA-08M|联盟TMA-08M]]号载人飞船从[[拜科努尔航天发射场|拜科努尔航天发射场]]成功升空，与[[联盟-FG|联盟-FG]]运载火箭分离后进入预定绕地轨道<ref>{{cite web |author1=胥文琦 |title=俄罗斯“联盟”号载人飞船29日成功发射升空 |url=http://mil.huanqiu.com/world/2013-03/3779832.html?agt=15422 |publisher=[[环球网|环球网]] |archiveurl=https://web.archive.org/web/20190414115913/http://mil.huanqiu.com/world/2013-03/3779832.html?agt=15422 |archivedate=2019-04-14 |date=2013-03-29}}</ref>。

莫斯科时间2013年8月16日18时40分（北京时间22时40分），他和[[费奥多尔·尤尔齐欣|费奥多尔·尤尔齐欣]]穿着俄制[[海鹰航天服|“海鹰-MK”航天服]]，打开[[码头号对接舱|码头号对接舱]]舱门进入太空，开始了2013年的第三次舱外任务。这次[[舱外活动|舱外活动]]的主要是为即将于12月与[[国际空间站|国际空间站]]对接的俄罗斯科学号多功能实验舱铺设电缆<ref>{{cite web |author=贺颖骏 |title=国际空间站俄宇航员实施今年第三次太空行走 |url=http://news.sciencenet.cn/htmlnews/2013/8/281497.shtm |publisher=[[新华社|新华社]] |archiveurl=https://web.archive.org/web/20160403081733/http://news.sciencenet.cn/htmlnews/2013/8/281497.shtm |archivedate=2016-04-03 |date=2013-08-21}}</ref>。

莫斯科时间22日15时40分（北京时间19时40分），他和尤尔齐欣打开舱门，进行本月第二次舱外任务。他们拆卸了[[星辰号服务舱|星辰号服务舱]]表面为太空实验“激光通信系统”而设置的机载[[激光|激光]]通信终端。他们还在星辰号服务舱表面安装了外设工位，并取回安装在[[探索号实验舱|探索号实验舱]]外的实验涂片<ref>{{cite web |author=贺颖骏 |title=俄两名宇航员一周内完成两次太空行走 |url=http://www.chinanews.com/mil/2013/08-23/5201196.shtml |publisher=[[新华网|新华网]] |archiveurl=https://web.archive.org/web/20190414111348/http://www.chinanews.com/mil/2013/08-23/5201196.shtml |archivedate=2019-04-14 |date=2013-08-23}}</ref>。
===远征53/54===
2017年9月13日，搭载[[远征53|远征53]]/54考察组的[[联盟MS-06|联盟MS-06]]宇宙飞船从哈萨克斯坦[[拜科努尔航天发射场|拜科努尔航天发射场]]升空。米苏尔金担任指挥长<ref>{{cite web |title=“联盟MS-06”宇宙飞船在哈萨克斯坦发射升空 |url=http://www.chinanews.com/tp/hd2011/2017/09-13/770986.shtml |archiveurl=https://web.archive.org/web/20190414113629/http://www.chinanews.com/tp/hd2011/2017/09-13/770986.shtml |archivedate=2019-04-14 |date=2017-09-13}}</ref>。

2018年2月2日，他和[[安东·什卡普列罗夫|安东·什卡普列罗夫]]在舱外进行了复杂的安装作业，无意中打破了单次空间站外作业时长纪录。以前的纪录为8小时7分钟，此次为8小时13分钟<ref>{{cite web |title=国际空间站俄两名宇航员打破外太空作业时间纪录 |url=http://news.sciencenet.cn/htmlnews/2018/2/402079.shtm |publisher=[[中国新闻网|中国新闻网]] |archiveurl=https://web.archive.org/web/20190414112503/http://news.sciencenet.cn/htmlnews/2018/2/402079.shtm |archivedate=2019-04-14 |date=2018-02-03}}</ref>。

2018年2月27日晚，他和[[约瑟夫·阿卡巴|阿卡巴]]、马克∙范德赫与留在空间站的任务队员告别，坐上了联盟MS-06飞船。莫斯科时间28日2时08分，飞船与国际空间站分离。2018年2月28日清晨，飞船返回舱降落在哈萨克斯坦[[杰兹卡兹甘|杰兹卡兹甘]]东南146公里处<ref>{{cite web |title=国际空间站宇航员搭载“联盟MS-06”飞船返回地球 |url=http://www.chinanews.com/gj/2018/02-28/8456307.shtml |publisher=中国新闻网 |accessdate=2019-04-14 |archiveurl=https://web.archive.org/web/20180727092912/http://www.chinanews.com/gj/2018/02-28/8456307.shtml |archivedate=2018-07-27}}</ref><ref>{{cite book |author=吴刚 |title=国际空间站3名宇航员返回地球 |date=2018年3月1日 |publisher=[[人民日报|人民日报]] |location=[[符拉迪沃斯托克|符拉迪沃斯托克]]}}</ref>。
