'''亚历山大·武尔茨'''（'''Alexander Wurz'''） (1974年2月15日出生在[[下奧地利州|下奧地利州]],[[Thaya|Thaya]]) 是奥地利职业赛车运动员，曾为[[一级方程式赛车|一级方程式赛车]][[威廉姆斯车队|威廉姆斯车队]]车手。他是前拉力赛车运动员，1974年,1976年和1982年欧洲锦标赛冠军弗兰克·伍尔茨的第二个儿子。

== 威廉姆斯时期 ==
[[File:Alex_Wurz_Canada_2006.jpg|thumb]]中驾驶着威廉姆斯三号赛车]]
亚历山大从2006年起成为了[[威廉姆斯车队|威廉姆斯车队]]各分站赛德的试车手。在2006年8月3日伍尔茨取代了[[马克·韦伯|马克·韦伯]]作为正式车手参加2007年度的F1赛事。这是自2000年以后伍尔茨首次成为全职的F1车手，他的队友是[[尼科·罗斯伯格|尼科·罗斯伯格]]。

== 个人资料 ==
武尔茨结婚并有两个儿子查理(Charlie)和菲力克斯(Felix)。他又一个有趣的习惯，就是在驾驶赛车时穿的是不同颜色的靴子<ref>{{cite web |url=http://www.btinternet.com/~rhobbs/mindgames.htm |title=Mind Games |archiveurl=https://archive.is/20120630225123/http://www.btinternet.com/~rhobbs/mindgames.htm |archivedate=2012-06-30 |deadurl=yes }}</ref>，但是在2007年返回F1赛场后他改变了这一习惯。
