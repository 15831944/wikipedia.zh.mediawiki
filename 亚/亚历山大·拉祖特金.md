'''亚历山大·伊万诺维奇·拉祖特金'''（{{lang-ru|Александр Иванович Лазуткин}}，{{bd|1957年|10月30日|}}）是[[俄罗斯|俄罗斯]]已退役[[宇航员|宇航员]]。
==早年==
1957年10月30日出生于莫斯科，1981年毕业于[[莫斯科航空学院|莫斯科航空学院]]，获得飞机[[机械工程|机械工程]]学位。1981年至1984年留校担任[[工程师|工程师]]。1984年至1992年在[[科罗廖夫能源火箭航天集团|能源科研生产联合公司]]担任设计工程师。1992年入选宇航员队伍，1994年3月获得测试宇航员资格<ref>{{cite web |title=Александр Иванович Лазуткин |url=http://www.warheroes.ru/hero/hero.asp?Hero_id=883 |website=warheroes.ru |accessdate=2019-04-20 |language=ru}}</ref>。
==生涯==
1997年2月10日，他和队员乘坐联盟TM-25，前往[[和平号空间站|和平号空间站]]执行任务。6月25日，[[瓦西里·齐布利耶夫|瓦西里·齐布利耶夫]]在引导[[進步號太空船|進步號]]货运飞船与和平号空间站对接时，由于飞船的移动速度太快，飞船撞向了空间站的[[光谱号|光谱号]]增压舱，撞掉了一块[[太阳能电池板|太阳能电池板]]阵列<ref>{{Cite book|title = This New Ocean: The Story of the First Space Age|url = https://books.google.com/books?id=TLKCmJSFJnYC|publisher = [[兰登书屋|Random House Publishing Group]]|date = 2010-09-29|isbn = 9780307765482|language = en|first = William E.|last = Burrows}}</ref> ，留下了撞击痕迹，机舱压力开始降低<ref>{{Cite book|title = Surviving the Extremes: A Doctor's Journey to the Limits of Human Endurance|url = https://books.google.com/books?id=2czSJB5BSHoC|publisher = Macmillan|date = 2004-01-20|isbn = 9781429976114|language = en|first = Kenneth|last = Kamler}}</ref>。

拉祖特金等队员迅速切断了与损坏舱体的连接，空间站内部压力开始稳定。但是太阳能电池装置的损坏使散热器保护盖移位。他们关闭了陀螺控制系统，并人工操作大型姿态控制推力器，使空间站实现机动，让其他舱体上的电池板朝向太阳以获取电能。最终空间站成功脱险<ref>{{cite web |title=十次大胆的太空救援 |url=http://www.sohu.com/a/118843396_418353 |publisher=搜狐 |accessdate=2019-04-20 |archiveurl=https://web.archive.org/web/20180102184157/http://www.sohu.com/a/118843396_418353 |archivedate=2018-01-02 |date=2016-11-13}}</ref>。

==退役==
2007年从宇航员队退役。2011年7月至2014年2月担任[[莫斯科航天博物馆|莫斯科航天博物馆]]馆长<ref>{{cite web |title=专访俄前宇航员：天宫一号升空迫使俄美修正航天战略 |url=http://world.huanqiu.com/roll/2011-09/2048690.html |publisher=[[环球网|环球网]] |accessdate=2019-04-20 |date=2011-09-29}}</ref><ref>{{cite web |title=СМИ: Героя России Александра Лазуткина уволили с поста директора Музея космонавтики |url=https://tass.ru/obschestvo/954633 |publisher=[[俄通社-塔斯社|TASS]] |accessdate=2019-04-20 |language=ru |date=2014-02-11}}</ref>。

2014年9月12日，他出席了[[深圳會展中心|深圳會展中心]]2號館生物展，現場參與了太空探索者協會年會社會活動日活動，之后參觀了[[華為技術有限公司|華為技術有限公司]]、[[華大基因|華大基因]]、北科生物科技有限公司等多家展商展台，體驗了生物科技的前沿技術<ref>{{cite web |title=航天員現身深圳國際生物/生命健康業展覽會 |url=https://news.cnyes.com/news/id/1083643 |publisher=北美新浪 |archiveurl=https://web.archive.org/web/20190420093936/https://news.cnyes.com/news/id/1083643?exp=a |archivedate=2019-04-20 |date=2014-09-15}}</ref>。
