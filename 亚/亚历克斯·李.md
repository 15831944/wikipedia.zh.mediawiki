'''亚历克斯·弗朗西斯·李'''（{{lang-en|Alex Francis Lee}}，{{bd|1990年|1月15日||}}）是一名[[关岛|关岛]]足球运动员，目前效力于[[美国足球联赛|美国足球联赛]]球队{{link-en|里士满踢球者|Richmond Kickers}}，司职[[后卫_(足球)|后卫]]。

== 足球生涯 ==
=== 大学时期 ===
亚历克斯·弗朗西斯·李曾在2008年至2011年期间效力于[[马里兰大学学院市分校|马里兰大学学院市分校]]足球队<ref>{{cite web |url=http://www.umterps.com/ViewArticle.dbml?DB_OEM_ID=29700&ATCLID=207385927 |language=en |title=Alex Lee Biography |accessdate=2018-07-30 |work=[[马里兰大学|马里兰大学]] |archive-url=https://web.archive.org/web/20170707141944/http://www.umterps.com/ViewArticle.dbml?DB_OEM_ID=29700&ATCLID=207385927 |archive-date=2017-07-07 |dead-url=no }}</ref>。2009年，由于一场意外车祸，亚历克斯在二年级时休学了一年<ref name="baltimoresun">{{cite web|url=http://articles.baltimoresun.com/2010-11-25/sports/bs-sp-terps-soccer-1128-20101125_1_terps-ncaa-alex-lee-family-affair|title=Alex Lee Accident Maryland's Lee, recovered from auto accident, faces twin brother Sunday - Baltimore Sun|publisher=baltimoresun|accessdate=2018-07-30|language=en|archive-url=https://web.archive.org/web/20130622053912/http://articles.baltimoresun.com/2010-11-25/sports/bs-sp-terps-soccer-1128-20101125_1_terps-ncaa-alex-lee-family-affair|archive-date=2013-06-22|dead-url=no}}</ref>。2011年，亚历克斯开始效力于{{link-en|超级发展联赛|Premier Development League}}球队{{link-en|皇家里士满足球俱乐部|Real Maryland F.C.}}<ref name="uslsoccer">{{cite web|url=http://www.uslsoccer.com/teams/2011/8969534.html#STATS|title=United Soccer Leagues (USL)|publisher=uslsoccer|accessdate=2018-07-30|language=en|archive-url=https://web.archive.org/web/20140503055812/http://www.uslsoccer.com/teams/2011/8969534.html#STATS|archive-date=2014-05-03|dead-url=yes}}</ref>。

=== 职业生涯时期 ===
2012年，亚历克斯在{{link-en|2012年美国职业足球大联盟二次选秀|2012 MLS Supplemental Draft}}中获得第11名的成绩，于3月13日与[[达拉斯足球俱乐部|达拉斯足球俱乐部]]签下职业合同<ref name="go">{{cite web|url=http://espn.go.com/blog/dallas/soccer/post/_/id/14729/fc-dallas-signs-scott-sealy-and-alex-lee|title=FC Dallas signs Scott Sealy and Alex Lee - FC Dallas Report - ESPN Dallas|publisher=espn.go.com|accessdate=2018-07-30|language=en|archive-url=https://web.archive.org/web/20140503071337/http://espn.go.com/blog/dallas/soccer/post/_/id/14729/fc-dallas-signs-scott-sealy-and-alex-lee|archive-date=2014-05-03|dead-url=no}}</ref>。然而，亚历克斯在达拉斯期间并未获得出场机会，最后在赛季末解约。2013年3月5日，亚历克斯加盟[[美国足球联赛|美国足球联赛]]球队{{link-en|里士满踢球者|Richmond Kickers}}<ref name="richmondkickers">{{cite web|url=http://www.richmondkickers.com/home/700103.html|title=KICKERS SIGN ALEX LEE - Richmond Kickers|publisher=richmondkickers.com|accessdate=2018-07-30|language=en|archive-url=https://web.archive.org/web/20160303230309/http://www.richmondkickers.com/home/700103.html|archive-date=2016-03-03|dead-url=yes}}</ref>。

== 生活 ==
亚历克斯·李有两个同样效力于[[关岛国家足球队|关岛国家足球队]]的兄弟，分别为他的双胞胎兄弟{{link-en|贾斯廷·李|Alex Lee (soccer)}}和弟弟{{link-en|内特·李|Nate Lee (footballer)}}<ref name="Nittany Lions Profile">{{cite web |url=http://www.gopsusports.com/sports/m-soccer/mtt/justin_lee_345090.html |title=Justin Lee |accessdate=2018-07-30 |work=Penn State Nittany Lions |language=en |archive-url=https://web.archive.org/web/20171025191748/http://www.gopsusports.com/sports/m-soccer/mtt/justin_lee_345090.html |archive-date=2017-10-25 |dead-url=no }}</ref>。他们三人均在关岛对阵[[香港足球队|香港]]的比赛中完成了国家队首秀<ref name="world football">{{cite web |url=http://www.worldfootball.net/report/freundschaft-2015-maerz-hongkong-guam/ |title=Hong Kong 1-0 Guam |language=en |date=2015-03-25 |accessdate=2018-07-30 |website=World Football |archive-url=https://web.archive.org/web/20170925093122/http://www.worldfootball.net/report/freundschaft-2015-maerz-hongkong-guam/ |archive-date=2017-09-25 |dead-url=no }}</ref>。
