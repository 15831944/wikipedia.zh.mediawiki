《'''橙路'''》（{{lang-ja|きまぐれオレンジ☆ロード}}）是[[松本泉|松本泉]]創作的[[日本漫畫|日本漫畫]]，後來被改編成[[日本電視動畫|電視動畫]]，另外有[[OVA|OVA]]、劇場版電影及小說。系列漫畫的發行量總計達到2000萬冊<ref>http://comipress.com/article/2008/12/31/3733.html</ref>。

== 概要 ==
故事讲述拥有[[超能力|超能力]]的男主角春日恭介与鮎川圆、桧山光两个女孩之间三角恋情。

本作品系列包括[[漫画|漫画]]、[[动画|动画]]、[[小说|小说]]、電影和[[OVA|OVA]]。

* 漫画于1984年15號至1987年42號在日本[[漫画杂志|漫画杂志]]《[[少年Jump週刊|少年Jump週刊]]》上连载。
* 電視動畫於1987年4月6日至1988年3月7日間在日本首播。
* OVA分為三期，第一期在1989年發售，第二期在1990年發售，第三期在1991年發售。
* 有兩套電影版，第一套在1988年上映，第二套則在1996年上映。
* 早期（1980年代後期至90年代初），臺灣有些公司翻版了不少日本動畫歌曲歌集，由依照日本原版專輯到臺灣自行編輯也有，初期以錄音帶為主，後來CD開始普及，很快便取代錄音帶。翻版在香港亦曾流行一時，有些專售動漫商品的店舖亦曾銷售臺灣翻版唱片。
* 中國大陸仿製日本正版，多張盜版原聲大碟SOUND COLOUR系列1-3集、OVA原聲大碟以及精選歌集LOVING HEART，于2000年在中國大陸市埸湧現，其中以LOVING HEART最為盜版嚴重。起初只是於大陸拍賣網站出售，其後充斥於全世界各大拍賣熱點，以 全新未開100%NEW 作售賣招徠，由此可見此作品多年來一直有極高人氣。
* 電視版香港首播是1989年ATV[[亞洲電視|亞洲電視]]，電影版也在日本公映的翌年於香港的[[亞洲電視|亞洲電視]]以節目名稱 動畫冰點 於星期六下午首播，但宣傳不足，令一眾香港橙路迷錯過欣賞。橙路熱潮不斷，同年[[安樂影片|安樂影片]]於香港會議展覽中心舉行動畫影展，橙路的 OVA SPECIAL 也在其中播放，多埸爆滿。數月後TVB[[無線電視|無線電視]]於星期六深夜時段首播放多套OVA作品，TVB自家配音版鮎川圓，同用[[相聚一刻|相聚一刻]]音無響子的配音員[[林元春|林元春]]，加上兩套作品動畫版同是[[高田明美|高田明美]]人物設計，致令香港一些對動畫不太認識的觀眾，時常弄錯兩套作品女主角的主要原因。
* 1993年，ATV亞視重新配音《橙路》，由香港名配音員[[林保全|林保全]]配演春日恭介。這版本最廣為流傳，也是配演鮎川圓的女新人[[黃玉娟|黃玉娟]]的成名作。

== 故事概要 ==
故事的主角春日恭介，因為其家族春日家擁有[[超能力|超能力]]（超能力來源是母親的家系，父親是普通人），每當被人發現其家族秘密便需要搬家。這次因為他的妹妹春日久留美在運動會運用超能力，在3秒內跑完100公尺，使他們不得不搬第七次家。

剛轉入新學校的春日恭介，在一條長樓梯邂逅少女鮎川圓，後來發現是同班同學。另一方面同校學妹檜山光，自幼便認識鮎川圓，因為檜山光在體育館裡準備抽菸時見到單獨在體育館裡垂頭喪氣的恭介使用超能力將籃球從地上投到對向去進中籃網而大吃一驚，從而對春日恭介產生好感，而在一次倆人於學校走廊相撞後更確認三角戀情的開始。

== 登場角色 ==
; {{nihongo|[[春日恭介|春日恭介]]|春日恭介（かすが きょうすけ）||[[配音員|聲]]: [[古谷徹|古谷徹]]}}
::'''香港配音'''：[[葉少榮|葉少榮]]（亞視1989年、劇場版亞視1991年）、[[鄧榮銾|鄧榮銾]]（OVA無綫1991年）、[[林保全|林保全]]（亞視1993年重配）、[[羅偉傑|羅偉傑]]（Pop & In 博意影碟 OVA）、[[陳廷軒|陳廷軒]]（Pop & In 博意影碟劇場版）
: 1969年11月15日生，[[天蠍宮|天蠍座]]，成績、長相與運動能力都很普通的少年，個性優柔寡斷，夾在同班同學（從國三到高三）鮎川圓與小兩屆的學妹檜山光之間，周遭大部份的人均認為他與檜山光較為親近，實際上他對鮎川圓的好感遠遠超過檜山光。在漫畫版最後一集(第15集)裡就清楚描述他對檜山光的感覺只是像「兄妹之間的喜歡」，但對鮎川圓的感覺則是「情侶之間的愛」。
: 擁有的[[超能力|超能力]]包括瞬間移動、念動力、透視、千里眼、順風耳、自我催眠、物體位置互換等等，並且能藉著滾落樓梯之類的方法回到過去，在某些情況下能進入平行世界，與他人甚至是動物頭部互相撞擊則能交換身體，動畫版中甚至能使出類似電光的力量，有著多樣化的超能力，並會隨著年齡增長而覺醒新的能力。
: 在漫畫版裡春日恭介常常無意識地（如遭久留美催眠、或自我催眠、在游泳池遭小茜惡搞、或是遭爺爺及一彌惡搞）做出對鮎川圓不禮貌的動作而被鮎川圓搧了很多巴掌，並且優柔寡斷的關係時常讓早期的鮎川圓吃醋，最後在其妹真奈美的揭露下而被迫作出三角戀情的抉擇；而劇場版則是鮎川圓又再度吃醋下自己必須面對三角關係的決擇。大部分的漫畫迷批評劇場版「但願回到過去」的春日恭介太過冷血，不符合松本泉描繪對春日優柔寡斷的個性。
: 常常在關鍵時刻拯救鮎川圓，比如在游泳池邊鮎川圓因喝醉酒無力抵抗差點遭遇強暴、在滑雪中遇到幽靈而決定和鮎川圓殉情、在纜車上因為過度失溫而差點凍死、在前往無人島上鮎川圓不斷遭雷擊時使用迅速移動能力拯救鮎川圓，甚至在平行世界拯救不認識春日恭介的鮎川圓而致使平行世界裡pick阿圓對春日產生好感；也會在鮎川圓重感冒時想辦法替她解決問題（比如做難吃的菜、或是叫表妹小茜冒充鮎川圓前往補考），長期累積下來使的鮎川圓開始對春日恭介萌生愛意，甚至漫畫版後幾集兩人在接吻關鍵時刻總被檜山光撞見而倆人趕快躲開，直到三角戀情結束後倆人才光明正大接吻而結束整部漫畫劇情。

; {{nihongo|[[鮎川圓|鮎川圓]]|鮎川まどか（あゆかわ まどか）||聲: [[鶴弘美|鶴弘美]]、[[島津冴子|島津冴子]]（Jump特別版動畫）、[[櫻井智|櫻井智]]（廣播劇）}}
::'''香港配音'''：[[龍寶鈿|龍寶鈿]]（亞視1989年）、[[林元春|林元春]]（OVA無綫1991年）、[[黃玉娟|黃玉娟]]（劇場版亞視1991年、亞視1993年重配、Pop & In 博意影碟）
::（[[衛視中文台|衛視中文台]]首播時譯為'''鮎川丸子'''）
: 1969年5月25日生，[[雙子宮|雙子座]]，性格獨立堅強，在與恭介認識前是抽煙打架無照駕駛等前科累累的典型不良少女，因過去個性強悍且時常只用吉他pick或撲克牌及徒手就能把對手打成重傷，故又有pick阿圓之稱；在春日出現後收斂許多，然而過去的威名仍然使不少人頗為忌憚，看似無所畏懼的她其實很怕幽靈一類的東西，平常瞞著學校在ABCB打工。
: 從未交過男朋友，她小時候的初戀情人和以後的男朋友都是春日恭介一人而已。對於這個角色設定，松本泉曾在漫畫版序裡面提及，本來是希望鮎川圓交過很多男朋友且也已經不是處女，但後來受到漫畫迷的抗議而把她設定成從未交過男朋友。
: 在漫畫版和動漫版鮎川圓小時候是一個短髮且愛穿短褲的女生，直到春日恭介來到六年前才改變了鮎川圓決定留長髮以及穿裙子，而鮎川圓在初期遇見春日恭介並送給他的紅色草帽亦是鮎川圓小時候遇到穿越時空的春日恭介送給她的。鮎川圓小時候的過去和長大後性格逐漸轉變為溫柔，有很多都跟遇到春日恭介離不開關係。
: 漫畫版第一集在抽菸中的鮎川圓故意裝作不認識春日恭介，在後來在第十五集春日真奈美決定幫恭介攤開三角關係時（末集），檜山光流淚說「女孩子被喜歡的人看到自己在抽菸會被認為是不良少女」，而真奈美則接著說「會這麼想的不是只有妳而已」便已經暗示了還有另外一個人這麼想，那個人就是鮎川圓。阿圓一開始便對春日恭介有好感，但為了掩飾自己正在抽菸而被春日看到的醜態，所以故做鎮定當作不認識春日。而真奈美的那一句話讓小光決定查清楚到底春日愛的是誰。
: 漫畫版裡許多對鮎川圓有偏見的同學隨著鮎川圓的個性改變較溫柔而開始逐漸轉而同情鮎川圓，甚至在最後一集鮎川圓曾參與演唱會後被大家誤會是因為去參與演唱會和在ABCB打工所以被退學處份，故大家集體請願自願退學而逼校長取消退學處分決定，後來證實是誤會一場，因為是鮎川圓為了結束三角戀情關係而主動要求退學。而鮎川圓從容易對檜山光主動和春日恭介親密的接觸而吃醋逐漸變成對檜山光默默地隱藏和春日的關係、甚至最後選擇默默退出三角戀情而去美國留學，引起漫畫迷廣泛的同情及迴響，故對於劇場版「但願回到過去」的鮎川圓反而直指不符原本角色設定，還有漫畫迷呼籲「就將鮎川圓的印象停留在漫畫版和TV版就好」。
: 音樂世家出身，有一位已婚的姊姊，遺傳自父母的優秀音樂才華在原作中曾經讓大批經紀公司找上門，動畫版中也有著熟練吹奏[[薩克斯風|薩克斯風]]的鏡頭出現。
: 由於在台灣人氣極高的緣故，而與[[福星小子|福星小子]]的拉姆以及[[相聚一刻|相聚一刻]]的音無響子及[[幸運女神|幸運女神]]的蓓兒丹娣（另一派說法為[[棒球英豪|鄰家女孩]]的淺倉南）被台灣動漫迷稱為80～90年代的動漫界四女神。直至現在只要到5月25日[[PTT|PTT]]仍會在進場畫面出現鮎川圓的肖像並且寫上「阿圓生日快樂」，可見人氣之高到現在仍未衰減。

== OVA ==
OVA版的故事和TV版無連接，在TV版結局中鮎川圓（15歲時，國中三年級）已得知春日恭介是超能力者且兩人已互表明心意，然而OVA版依照漫畫原著，時間點發生在15~17歲（高中時期）鮎川圓仍不知道恭介的秘密，三人持續三角戀關係。

=== 第一期 ===
; {{nihongo|白色戀人|白い恋人たち[OVA 1]}}
: 1989年2月15日發售。
: 故事講述恭介和鮎川等人來到爺爺家旅行滑雪。當地有一個傳說：情侶不可一起滑雪；小光為避開傳說，變成恭介和鮎川一起滑雪，終於在山洞中遇上了一連串怪事。
; {{nihongo|夏威夷驚魂記|ハワイアン・サスペンス [OVA 2]}}
: 1989年3月15日發售。
: 故事講述恭介等人夏威夷遊玩，小光卻遭匪徒綁架的故事。

=== 第二期 ===
; {{nihongo|Heart On Fire! 春天偶像|恋のステージ HEART ON FIRE! 春はアイドル [OVA 3]}}
: 1989年12月16日發售。
: 故事講述恭介與偶像歌手早川互換身體。
; {{nihongo|Heart On Fire! 巨星誕生|恋のステージ HEART ON FIRE! スター誕生 [OVA 4]}}
: 1989年12月16日發售。
: 故事講述恭介害怕偶像歌手早川會說出他擁有超能力的秘密。
; {{nihongo|旋風！變身少女小茜|ハリケーン! 変身少女あかね [OVA 5]}}
: 1989年12月16日發售。
: 故事講述恭介擁有變身超能力的表妺小茜來到恭介家，因為她認為檜山光是恭介的女朋友而引起連串事件。
; {{nihongo|我是貓也是魚|吾輩は猫であったり おサカナであったり [OVA 6]}}
: 1989年12月16日發售。
: 故事講述恭介得到一條能夠交換身體的繩，但一次意外下，恭介和金魚交換了身體，然後又變成一隻貓而發生連串故事。

== 劇場版 ==
; {{nihongo|但願回到過去|あの日にかえりたい}}
: 1988年10月8日上映，是動畫版的完結篇，交代了春日恭介在三角戀的選擇。
: 講述鮎川圓和春日恭介考進大學前一刻，回想前年夏天他和鮎川圓和檜山光的三角戀的解決。為銜接動畫版完結的劇情，劇場版的鮎川圓和檜山光對三角戀情的態度和解決方法與漫畫原著有所出入。大部分的漫畫迷批評劇場版「但願回到過去」描繪的春日恭介太過冷血，不符合松本泉描繪對春日優柔寡斷的個性，也批評劇情中的鮎川圓跟松本泉所設定的鮎川圓個性不同，松本泉在漫畫版所設定的鮎川圓是默默地想獨自退出三角戀情而成全恭介和小光在一起，然而到了劇場版卻變成鮎川圓想要獨自佔有恭介反而使人同情小光，因而許多支持鮎川圓的漫畫迷呼籲「就把印象停留在漫畫版跟TV版就好」。

== 雜項資料 ==
* 日本原名{{lang|ja|きまぐれ}}在日语中有着捉摸不定的意思，正好说明了三个人之间暧昧不定的关系。
* {{lang|ja|オレンジ☆ロード}}是英语“Orange Road”的[[片假名|片假名]]表示，也就是“-{橙路}-”这个译名的由来。
* 作者[[松本泉|松本泉]]在連載期間就常因為健康狀況不良，而導致後來的長期休載。休載前和再連載後的一段時間，因為畫風差異很大，讀者便猜測是由{{link-ja|萩原一至|萩原一至}}代筆。事實上只有在松本無法執筆暫時由萩原代畫，在連載完結前萩原就離開了工作室，松本也在官方的Fanbook裡否定這項流言。另外長期讓松本困擾的症狀在2004年查明是{{link-ja|腦髓液減少症|脳脊髄液減少症}}。
* 單行本化時就有相當多修改，特別是最後一集修改幅度最大。愛藏版發行時也經過了修改，所以連載版、JC（Jump Comics）版和愛藏版的最終回是完全不一樣的。
