'''目連救母'''是一則佛經故事，有諸多版本，在中國流傳最廣的[[變文|變文]]版本，是[[釋迦牟尼佛|釋迦牟尼佛]]的神通第一弟子[[目犍連|目犍連]]（簡稱'''目連'''、'''目蓮'''，{{lang-sa|Maudgalyayana}}；{{lang-pi|Moggallāna}}）以[[神通|神通]]能力到達[[饿鬼道|饿鬼道]]，見母親身受[[饿鬼|饿鬼]]之苦，最終借十方眾僧威神之力使母親解脫。但最初故事原型是[[目連|目連]]請佛陀為其母說法，讓他已經成為[[天人|天人]]的母親修成果位，解脫輪迴，因說法布施是大功德，而非儒教影響下轉變的忠孝故事。

== 故事简介 ==
相傳[[目犍连|目犍连]][[羅漢|羅漢]]的母親做了很多壞事，死後變成了[[餓鬼|餓鬼]]，目連通过神通看到後十分傷心，就運用法力，將一些飯菜拿給母親食用，可是飯一到母親口邊就化為焰灰，目犍连大声向[[釋迦牟尼佛|釋迦牟尼佛]]哭救<ref>[[西晋|西晋]][[竺法护|竺法护]]译，《佛说盂兰盆经》：“一时佛在舍卫国祇树给孤独园。大目犍连始得六通，欲度父母，报乳哺之恩。即以道眼观视世间，见其亡母生饿鬼中，不见饮食，皮骨连立。目连悲哀，即以钵盛饭，往饷其母，母得钵饭，即以左手障钵，右手搏食，食未入口，化成火炭，遂不得食。目连大叫，悲号涕泣，驰还白佛，具陈如此。”</ref>。[[世尊|世尊]]表示目連雖已是神通第一，但個人的神通力不敵[[業力|業力]]，因此必須依大福田力才能救拔，為此目連特別在洞穴外準備盥洗用品，等待[[結夏安居|結夏安居]]結束的修行羅漢使用盥洗，并於每年七月中以百味五果，置於盆中，供養十方僧人，以此般功德，其母方能濟度<ref>[[西晋|西晋]][[竺法护|竺法护]]译，《佛说盂兰盆经》：“佛告目连：‘十方众生，七月十五日，僧自恣时，当为七世父母及现在父母厄难中者，具饭、百味五果、汲灌盆器、香油锭烛、床敷卧具、尽世甘美以著盆中，供养十方大德众僧。当此之日，一切圣众，或在山间禅定、或得四道果、或在树下经行、或六通自在教化声闻缘觉、或十地菩萨大人，权现比丘，在大众中，皆同一心，受钵和罗饭，具清净戒，圣众之道，其德汪洋。其有供养此等自恣僧者，现世父母、六亲眷属，得出三涂之苦应时解脱，衣食自然；若父母现在者，福乐百年；若七世父母生天，自在化生，入天华光。’”</ref>。目蓮依佛意行事，其母終得解脫<ref>[[西晋|西晋]][[竺法护|竺法护]]译，《佛说盂兰盆经》：“时目连母即于是日，得脱一劫饿鬼之苦。”</ref>。

== 故事原型 ==
目連救母故事原型並沒有[[儒教|儒教]]忠孝色彩，而是目連請佛陀為亡母說法開示，度其于苦厄。見於《根本說一切有部毗奈耶》<ref>《大正藏》冊24，頁16中-下。</ref><ref>[http://homepage.ntu.edu.tw/~ntubudda/tjbs/fulltexts/012/012-1f.pdf 漢譯大小乘佛典的「神足飛行」的空間敘事]，台北政治大學，丁敏[著]。</ref>。目連思念亡母，想飛往[[摩利支天|摩利支天]]「以法教化」亡母，但摩利支天路途遙遠，只有世尊能前往說法，遂目連請世尊一同前往。到了摩利支世界，母子相認，目連的母親已變成年輕的天女，母少子老，大家都不相信。目連便說：「此女養我，是我生母。」接著佛陀為目連亡母說[[四聖諦|四聖諦]]，使其得證[[初果|初果]]，永不再墜惡道。在此故事中目連的母親是天女，和《盂蘭盆經》中目連的母親是餓鬼、以及變文中目連的母親身在地獄中，是截然不同的形象和救度起點；而亦可以發現「目連救母」故事最初的敘述聲調，亦非表彰目連的孝行。

== 變文 ==
漢傳《目連救母》故事最早起源於[[西晉|西晉]]三藏[[竺法護|竺法護]]所譯《{{lj|佛說盂蘭盆經|盂蘭盆経}}》，[[張祜|張祜]]曾對[[白居易|白居易]]說：“明公亦有《目連經》。《長恨詞》-{云}-：‘上窮碧落下黃泉，兩處茫茫皆不見。’此不是目連訪母耶？”<ref>[[王定保|王定保]]：《[[唐摭言|唐摭言]]》</ref>。[[敦煌|敦煌]][[變文|變文]]寫本有《大目犍連變文》、《大目連變文》和《大目連緣起》，[[元代|元代]]末年有《目蓮救母出離地獄昇天》。[[明|明]]代戲曲作家[[鄭之珍|鄭之珍]]著有[[傳奇|傳奇]]《目連救母》，全名《目連救母勸善戲文》。[[乾隆|乾隆]]年間，[[張照|張照]]參考鄭之珍《目連救母》，編撰《勸善金科》二百四十集。在[[中國民間信仰|中國民間信仰]]中，《目連救母》故事被改編，不再是透過供養佛僧以救母，而是自己持著[[佛祖|佛祖]]的禪杖，到地獄中，打破了地獄之門，救出母親。

== 異說 ==
有的版本中，目連得到[[如來佛祖|如來佛祖]]的指示，在七月十五日供养许多[[阿罗汉|阿罗汉]]，每人一钵净水、一枝杨枝，喝时先用杨枝净水布施无量饿鬼，将功德回向目连的亡母，亡母承僧人们的加持脱离[[饿鬼道|饿鬼道]]，转世至[[畜生道|畜生道]]做了一隻黑狗，后来听目连说法七天，領悟了[[佛法|佛法]]，转生[[三十三天|三十三天]]为[[仙女|仙女]]，才算救母成功。

在《目連三世寶卷》和一些民間傳說的版本中，將目連救母與[[黃巢之亂|黃巢之亂]]的傳說合併，說目連救母時，有惡鬼趁目連以錫杖頓地打開地獄門時逃出，轉世為人或豬羊，[[地藏王菩薩|地藏王菩薩]]與[[十殿閻君|十殿閻君]]要求目連收回這些惡鬼，才赦免其母。於是目連二世轉生為[[唐朝|唐朝]]民變首領[[黃巢|黃巢]]，[[屠殺|屠殺]]八十萬（或稱八百萬）餘百姓，將這些當初由地獄逃出的人殺死，靈魂收至地獄。而後三世又化身為屠夫，替閻君收回豬羊性命，目蓮母才得到釋放<ref>{{cite book|title=目連救母故事之演進及其有關文學之研究|url=https://books.google.com.tw/books?id=mTinAAAAQBAJ&pg=PT194&dq=%E7%9B%AE%E9%80%A3%E6%95%91%E6%AF%8D%E6%95%85%E4%BA%8B%E4%B9%8B%E6%BC%94%E9%80%B2%E5%8F%8A%E5%85%B6%E6%9C%89%E9%97%9C%E6%96%87%E5%AD%B8%E4%B9%8B%E7%A0%94%E7%A9%B6&hl=zh-TW&sa=X&ved=0CB0Q6AEwAGoVChMI7uqx-4TLxwIVRCmUCh28dA-T#v=onepage&q=%E7%9B%AE%E9%80%A3%E6%95%91%E6%AF%8D%E6%95%85%E4%BA%8B%E4%B9%8B%E6%BC%94%E9%80%B2%E5%8F%8A%E5%85%B6%E6%9C%89%E9%97%9C%E6%96%87%E5%AD%B8%E4%B9%8B%E7%A0%94%E7%A9%B6&f=false | author = 陳芳英 | publisher = 國立臺灣大學出版中心 | page = 36-38 |date=1983年6月}}</ref>。然而依佛教教義言，目連作為[[羅漢|羅漢]]及[[釋迦牟尼|釋迦牟尼]]親授大弟子，應持戒嚴謹，不至以[[殺生|殺生]]補過。[[殺生|殺生]]者乃佛教之大戒（十惡之首），再者就覆罪惡深重之魂，既已經過[[六道輪迴|六道輪迴]]超脫[[三惡道|三惡道]]而進入[[人道|人道]]，可見此些[[靈魂|靈魂]]皆以脫離[[餓鬼道|餓鬼道]]，此故事根本與佛教教義相駁。佛教界多駁斥此說，認為此不合理的情節有辱教中聖者。<ref>{{cite web | url = http://www.templevisit.org/Selection_content.asp?AutoNo=9508 | title=佛學問答：目蓮救母 | work = 古今寺廟巡禮 }}</ref>

== 发展 ==
最初盂兰盆节中，佛教僧众用盆子裝滿百味五果，供養的是[[釋迦佛|釋迦佛]]、眾[[阿羅漢|阿羅漢]]、僧侶，以拯救入地獄的苦難眾生<ref>[[西晋|西晋]][[竺法护|竺法护]]译，《佛说盂兰盆经》：“于七月十五日，佛欢喜日，僧自恣日，以百味饭食，安盂兰盆中，施十方自恣僧，愿使现在父母，寿命百年无病、无一切苦恼之患，乃至七世父母离恶鬼苦，生人天中，福乐无极。是佛弟子修孝顺者，应念念中，常忆父母，乃至七世父母。年年七月十五日，常以孝慈，忆所生父母，为作盂兰盆，施佛及僧，以报父母长养慈爱之恩。若一切佛弟子，应常奉持是法。”</ref>。

在[[中國民間信仰|中國民間信仰]]，已經成為重要的一項[[民俗|民俗]]，{{Fact|同時[[破地獄|破地獄]]成功時，[[地獄|地獄]]的群鬼也跟著出來，此也是[[農曆|農曆]]七月被稱為[[鬼月|鬼月]]之故|time=2015-5-5}}，也是[[盂蘭盆節|盂蘭盆節]]（中元節）法會、祭典舉辦之因，故目連尊者，是中元普渡中民間奉祀的對象，民間甚至認為[[目連|目連]]尊者是[[地藏菩薩|地藏菩薩]]的輔佐者、[[枉死城|枉死城]]眾生的救度者，部下有四大[[判官|判官]]：查察司判官[[李玄邃|李玄邃]]、賞善司判官[[楊玄感|楊玄感]]、罰惡司判官[[韓子通|韓子通]]、陰律司判官[[崔子玉|崔子玉]]，四判官專門計算死者功德，故各大地藏庵時常配祀目連尊者，如[[新莊地藏庵|新莊地藏庵]]。

目蓮救母的故事，也成為[[戲曲|戲曲]]演出的重要題材，被稱作[[目蓮戲|目蓮戲]]，目蓮戲常與[[鄉民|鄉民]]的宗教儀式相結合，尤其中元節的酬神者，[[神功戲|神功戲]]更是必邀目蓮戲。

[[福建|福建]]的[[拗九节|拗九节]]相传源於目連救母的典故。[[福州人|福州人]]傳說佛陀弟子目連的母親平素蠻橫，死後[[靈魂|靈魂]][[禁錮|禁錮]]於[[地獄|地獄]]，罰做沒有飯喫的[[餓鬼|餓鬼]]。目連是個[[羅漢|羅漢]]，能看到母親的慘狀，這位目連[[孝子|孝子]]於是每天用[[神通|神通]]到地府幫母親送飯，卻都被[[陰間|陰間]]的[[無常鬼|無常鬼卒]]喫掉。直到[[農曆|農曆]]正月廿九，目連送一碗[[黑芝麻|黑芝麻]]、[[糯米|糯米]]、[[花生|花生]]、[[紅棗|紅棗]]、[[桂圓|桂圓]]等等加[[紅糖|紅糖]]煮成的黑色甜[[粥|粥]]，鬼卒以為這粥裏都是「垢」，不肯拿。目連母親才得以飽餐一頓。後世為了紀念目連孝心，一定要在正月廿九這一天煮放了[[紅糖|紅糖]]的熬九粥給父母、長輩食用。因[[福州话|福州话]]中“垢”、“九”同音，此粥又被称为“[[拗九粥|拗九粥]]”，正月廿九日则成为了“拗九节”。<ref>[http://www.cctv.com/geography/mfms/20010808/31.html 拗九节与二月二]，央视网</ref>
