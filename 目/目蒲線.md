'''目蒲線'''（{{jpn|j=目蒲線|hg=めかません|rm=Mekama sen|f=y}}）曾經是一條連接[[東京都|東京都]][[品川區|品川區]][[目黑站|目黑站]]與[[大田區|大田區]][[蒲田站|蒲田站]]，屬於[[東京急行電鐵|東京急行電鐵]]（東急）的鐵路線。

東京急行電鐵的母體{{link-ja|目黑蒲田電鐵|目黒蒲田電鉄}}最初開業的路線。2000年8月6日起以[[多摩川站|多摩川站]]（同日由多摩川園站改名）為界分割為以下路線，消滅「目蒲線」的名稱。

; 作為系統名的分割
:* 目黑站－多摩川站－武藏小杉站：[[目黑線|目黑線]]（與分割同時延伸至[[武藏小杉站|武藏小杉站]]。2008年6月22日再延伸至[[日吉站_(神奈川縣)|日吉站]]）
:* 多摩川站－蒲田站：[[東急多摩川線|東急多摩川線]]
:
; 作為路線名的分割
:* 目黑站－[[田園調布站|田園調布站]]：目黑線
:* 田園調布站－多摩川站：改為使用與向來以重複路段（並走）的[[東橫線|東橫線]][[四線鐵路|複複線]]之內2軌
:* 多摩川站－蒲田站：東急多摩川線

== 路線資料 ==
路線分割以前的資料。
* 路線距離：12.9公里
* [[軌距|軌距]]：1067毫米
* 站數：15個（包括起終點）
* 複線路段：全線
* 電氣化路段：全線（直流1500V）
* [[閉塞_(鐵路)|閉塞方式]]：自動閉塞式
