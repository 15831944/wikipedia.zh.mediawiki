'''目黑站'''（{{jpn|j=目黒駅|hg=めぐろえき|rm=Meguro eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[品川區|品川區]][[上大崎|上大崎]]，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）、[[東京急行電鐵|東京急行電鐵]]（東急）、[[東京地下鐵|東京地下鐵]]、[[東京都交通局|東京都交通局]]（[[都營地下鐵|都營地下鐵]]）的[[鐵路車站|鐵路車站]]。此站是品川區最北端車站。

== 停靠路線 ==
此站有以下4間公司4條路線停靠，是轉乘站。

* JR東日本：[[File:JR_JY_line_symbol.svg|15px]] [[山手線|山手線]] - 僅停靠運行於電車線的環狀線山手線電車。依據[[特定都區市內|特定都區市內]]制度，本站屬於「[[特定都區市內#設定區域一覧|東京都區內]]」與「[[東京山手線內|東京山手線內]]」<ref name="kippu">{{Cite web |url=https://www.jreast.co.jp/kippu/11041.html |title=JR東日本：きっぷに関するご案内>特定の都区市内駅を発着する場合の特例 |publisher=東日本旅客鉄道 |accessdate=2014-08-02}}</ref>。車站編號為「'''JY 22'''」。
* 東急：[[File:Tokyu_MG_line_symbol.svg|15px]] [[目黑線|目黑線]] - 本站為起點。與下述2路線互相直通運行。過去稱為「目蒲線」。車站編號為「'''MG01'''」。
* 東京地下鐵：[[File:Subway_TokyoNamboku.png|15px]] [[南北線_(東京地下鐵)|南北線]] - 本站為起點。通車起有許多列車從[[赤羽岩淵站|赤羽岩淵站]]直通埼玉高速鐵道線。車站編號為「'''N 01'''」。
* 都營地下鐵：[[File:Subway_TokyoMita.png|15px]] [[三田線_(都營地下鐵)|三田線]] - 本站為起點。車站編號為「'''I 01'''」。

東急目黑線、東京地下鐵南北線、都營地下鐵三田線之間實施[[直通運行|互相直通運行]]，為站內共用的[[共構與共站|共同使用站]]（東急管轄）。

另外，南北線與三田線從本站至[[白金高輪站|白金高輪站]]是共用車站與軌道設備。該區間由東京地下鐵作為第一種鐵道事業者，而東京都交通局則為第二種鐵道事業者。

== 歷史 ==
* 1885年（[[明治|明治]]18年）3月16日：[[日本鐵道|日本鐵道]]目黑站開業<ref name="jtb84"/>，只開辦客運服務。
* 1906年（明治39年）11月1日：日本鐵道被國有化，成為[[日本國有鐵道|日本國鐵]]的車站。
* 1909年（明治42年）10月12日：日本國鐵重整路線，本站成為山手線所屬車站。
* 1923年（[[大正|大正]]12年）3月11日：目黑蒲田電鐵（現東急）目黑站開業<ref name="jtb82">[[#jtb|東急車站]]、pp.82-83。</ref><ref name="jtb84">[[#jtb|東急車站]]、pp.84-85。</ref>。
* 1929年（[[昭和|昭和]]4年）：目蒲線目黑站開設東橫第2食堂<ref name="jtb84"/>。
* 1936年（昭和11年）：目蒲線站房改建為木造砂漿2層建築<ref name="jtb84"/>。
* 1942年（昭和17年）：目蒲線月台對應3輛編組<ref name="jtb84"/>。
* 1953年（昭和28年）12月12日：目蒲線站房改建為鋼筋混擬土地上2層、地下1層建築<ref name="jtb84"/>。
* 1967年（昭和42年）4月1日：目蒲線月台對應4輛編組，廢止3號線<ref name="jtb84"/>。
* 1987年（昭和62年）4月1日：[[國鐵分割民營化|國鐵分割民營化]]，本站由JR東日本接手管理。
* 1992年（[[平成|平成]]4年）：JR東日本在本站月台試驗月台閘門，閘門在試驗完成後被拆除。
* 1997年（平成9年）7月27日：東急目黑站地下化<ref name="jtb84"/>。
* 2000年（平成12年）
** 8月6日：目蒲線分割為目黑線和[[東急多摩川線|多摩川線]]，本站改屬目黑線<ref name="jtb82"/>。
** 9月26日：[[帝都高速度交通營團|帝都高速度交通營團]]（營團地下鐵）南北線、都營地下鐵三田線經本站連接東急目黑線。同日開始相互直通運行<ref name="jtb84"/>。
* 2001年（平成13年）
** 4月28日：連結東急車站與JR車站西口的連絡地下道開通<ref name="rail fan2003-02p8">{{Cite journal|author = 鉄道友の会東京支部東急部会 |date = 2003-02-01 |title = 2001年度 東急総決算 |journal = RAIL FAN |issue = 2 |volume = 50 |publisher = 鉄道友の会 |pages = 8 }}</ref>。
** 11月18日：JR東日本智慧卡系統[[Suica|Suica]]正式投入服務。
* 2004年（平成16年）4月1日：營團地下鐵民營化，本站由東京地下鐵接手管理。
* 2006年（平成18年）4月28日：東京地下鐵定期券售票處結束營業<ref>[https://web.archive.org/web/20060621140656/http://www.tokyometro.jp/rosen/teiki/index.html 定期券うりばのご案内（インターネット・アーカイブ）]</ref>。
* 2010年（平成22年）8月28日：JR山手線月台啟用可動式月台門<ref>{{Cite press release|url=http://www.jreast.co.jp/press/2009/20100307.pdf|format=PDF|title=山手線恵比寿駅、目黒駅のホームドア使用開始日について|publisher=東日本旅客鉄道|date=2010-03-04}}</ref>。

=== 站名和所在地 ===
雖然本站站名為目黑站，但不是位於[[目黑區|目黑區]]而是品川區。就算是開業時車站也不在目黑村附近，而是位於大崎村附近。一般相信原因是在建設鐵路（現在的山手線）時目黑川附近的農民擔憂[[蒸氣機車|蒸氣機車]]駛過時會令他們的農作物受損而強烈反對，造成原本沿目黑川興建的計畫受挫。

事實上，這種站名與所處地名相異的情況並不罕見，如[[品川站|品川站]]（位在[[港區_(東京都)|港區]]）、[[厚木站|厚木站]]（位在[[海老名市|海老名市]]）、[[志木站|志木站]]（位在[[新座市|新座市]]）等也有類似的情況。

島式月台1面2線的地下車站。月台為目黑線～南北線、三田線相互直達運行共用。此外，管理的業務歸屬於東急電鐵。

往武藏小杉方向設有剪式[[轉轍器|轉轍器]]。2006年9月25日至2008年6月21日，目黑線上行列車中的本站終停各站停車先利用該轉轍器至1號線下行月台，與後面進入2號線上行月台的上行急行接續，再載客發往武藏小杉。但自同年6月22日起，多數班次已直通南北線、三田線，折返列車也不再使用該轉轍器。

南北線車站採用全罩式月台門，但由於本站為東急管轄站，因此成為南北線中唯一安裝可動式月台門的車站。東急線開業時為2面3線的地上[[側式月台|側式月台]]港灣式地面車站<ref name="jtb84"/>，之後1953年建造車站大樓，{{link-ja|東急store|東急ストア}}入駐地下1樓<ref name="jtb84"/>。當時正面設有剪票口，另外1號線月台中央還有一座往山手線月台的連絡剪票口。1997年7月27日完成現在的站體構造。

站內設有東急、東京地下鐵、都營地下鐵3種自動售票機，須依據不同目的地在不同售票機購票。若是前往[[白金台站|白金台站]]、[[白金高輪站|白金高輪站]]，由於是東京地下鐵、都營地下鐵共用去間，兩者機器皆可購票。另外，[[PASMO|PASMO]]、[[Suica|Suica]]等IC卡乘車券儲值、PASMO發售、定期券（磁氣、PASMO）發行等業務是由管理車站的東急負責，東京地下鐵、東京都交通局的自動售票機無法儲值，也不發行東京地下鐵、東京都交通局的定期券。（東急⇔東京地下鐵、東京都交通局的連絡定期券由東急定期券窗口及有「定期券」標誌的售票機發售）

本站是東京急行電鐵的站長所在站，管理「目黑站管內」本站 - [[奧澤站|奧澤站]]間<ref>鉄道ピクトリアル2015年12月臨時増刊号 【特集】東京急行電鉄　p.47</ref>。東京地下鐵車站部分是[[霞關站_(東京都)|霞關]]站務管區[[溜池山王站|溜池山王]]地域的被管理站<ref>鉄道ピクトリアル2016年12月臨時増刊号 【特集】東京地下鉄　p.17</ref>，都營地下車站部分則是[[日比谷站|日比谷]]站務管區日比谷站務區的被管理站<ref>[http://www.reiki.metro.tokyo.jp/reiki_honbun/ag10115671.html 東京都交通局駅務管区処務規程]</ref>。

目黑站是東京地下鐵的最南端車站。

== 使用情況 ==
本站1日平均上下車人次在1999年度以前約30萬人左右，之後[[都營地下鐵三田線|都營地下鐵三田線]]、[[東京地下鐵南北線|東京地下鐵南北線]]通車並與[[東急目黑線|東急目黑線]]直通開始，使用人數不斷增加。2017年度1日平均上下車人次總和達約71.6萬人，年上下車人次約2億6100萬人。

* '''JR東日本''' - 2017年度1日平均上車人次為'''111,655人'''<ref group="利用客数">[http://www.jreast.co.jp/passenger/index.html 各駅の乗車人員] - JR東日本</ref>。
*: JR東日本車站中次於[[戶塚站|戶塚站]]位居第32位。
* '''東京急行電鐵''' - 2017年度1日平均上下車人次為'''276,680人'''<ref group="利用客数">[http://www.tokyu.co.jp/railway/data/passengers/ 2016年度乗降人員] - 東急電鉄</ref>。
*: 該公司車站中次於[[澀谷站|澀谷站]]、[[橫濱站|橫濱站]]名列第3位。包括東京地下鐵南北線、都營地下鐵三田線。
* '''東京地下鐵''' - 2017年度1日平均上下車人次為'''118,326人'''<ref group="利用客数">[http://www.tokyometro.jp/corporate/enterprise/passenger_rail/transportation/passengers/index.html 各駅の乗降人員ランキング] - 東京メトロ</ref>。
*: 包括東急目黑線直通人次。
* '''都營地下鐵''' - 2017年度1日平均上下車人次為'''97,236人'''（上車人次49,675人，下車人次47,561人）<ref group="利用客数">[http://www.kotsu.metro.tokyo.jp/subway/kanren/passengers.html 各駅乗降人員一覧] - 東京都交通局</ref>。
*: 包括東急目黑線直通人次。
