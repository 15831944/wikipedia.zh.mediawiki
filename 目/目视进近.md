'''目视进近'''在飞行员遵循[[儀器飛行|仪表飞行规则]]（IFR）、同时确认能目视到跑道的情况下着陆机场跑道的方法。飞行员在进近过程必须能看见机场或是前方飞机。此种进场方法必须经过授权且受合适的[[航空交通管制|空管]]设施管制。<ref name="AIM_visual_approach">{{cite web
 |title = Arrival Procedures: Visual Approach
 |journal = Aeronautical Information Manual (AIM)
 |publisher= [[美国联邦航空管理局|Federal Aviation Administration]] (FAA)
 |date = 2010-11-10
 |pages = 5-4-23
 |format = PDF
 |accessdate = 2012-08-27}}
</ref>[[国际民用航空组织|国际民用航空组织]]（ICAO）的定义补充道：在“部分或完全未完成[[仪表进近|仪表进近]]的情况下”{{NoteTag|英文：either part or all of an instrument approach is not completed|name=英文}}，可进行目视进近。此点与[[美国联邦航空管理局|美国联邦航空管理局]]的规定大同小异。<ref name="AIM_visual_approach"/>

==目的==
目视进近得以让飞行员在不必执行仪表进近的情况下着陆跑道。<ref name="aopa">{{cite journal
 |title        = Instrument Insights Part 8 of 12: Tricks of the Trade
 |journal      = AOPA Pilot Magazine
 |date         = 1988-08-01
 |first        = Horne
 |last         = Thomas
 |volume       = 41
 |issue        = 8
 |url          = http://www.aopa.org/pilot/features/ii_9808.html
 |accessdate   = 2010-11-26
 |archive-url  = https://web.archive.org/web/20100521060414/http://www.aopa.org/pilot/features/ii_9808.html
 |archive-date = 2010-05-21
 |dead-url     = yes
 |df           = 
}}
</ref>这可极大地减轻飞行员及空中管制员的负担，同时还能保证至机场航线的畅通。<ref name="AIM_visual_approach"/>相比于执行复杂的{{tsl|en|instrument approach procedure|仪器进场程序|仪器进场程序}}（IAP），以较短航程飞行至机场可以保证飞行员的安全，因为最为重要的进场和着陆程序均发生在飞行员最为疲惫的时候。<ref>{{cite journal
 |title = Out of the Pattern: The longest cross-countries
 |journal = Flight Training Magazine
 |date = 1999-01-05
 |first = Amy
 |last = Laboda
 |accessdate = 2010-11-27}}
</ref><ref name="aopa"/>不仅如此，空中管制员也可通过目视进近来最大化机场流量（特别是在极为繁忙的机场）。而且，除非是在特定规则及情形下，否则空中管制的仪表飞行规则要求均可被减少。<ref>{{cite web
 |title = Problems with Visual Approaches: The Visual Trap
 |journal = Aviation Safety Reporting System (NASA)
 |date = 1992-07-01
 |first = Perry
 |last = Thomas

 |accessdate = 2010-11-27}}
</ref> 

目视进近同时也可造成额外的安全隐患，如错认着陆跑道（例：[[亞特拉斯航空|亞特拉斯航空]][[波音747-400_LCF|波音747 LCF]]型飞机错误着陆在{{tsl|en|Colonel James Jabara Airport|詹姆斯·贾巴拉中校机场|詹姆斯·贾巴拉中校机场}}而非目的地机场{{tsl|en|McConnell Air Force Base|麦康奈尔空军基地|麦康奈尔空军基地}}）。<ref>{{cite web |title=Incident: Atlas BLCF at Wichita on Nov 20th 2013, landed at wrong airport |url=http://avherald.com/h?article=46bcd7c5 |website=The Aviation Herald |accessdate=12 April 2019}}</ref>除此之外，现实中还曾发生其他类似的小事件。<ref>{{cite web |title=Incident: Delta A320 at Rapid City on Jul 7th 2016, landed on wrong airport |url=http://avherald.com/h?article=49adef6e |website=The Aviation Herald}}</ref><ref>{{cite web |title=Incident: American A320 at Fort Myers on Aug 30th 2018, approached wrong airport and went around |url=http://avherald.com/h?article=4bd139c6 |website=The Aviation Herald}}</ref>由于需要特别注意外界环境，目视进近尤其在拥挤空域中会增加飞行员的工作量。当飞行员接受目视进近时，其也要承担保持安全距离、规避尾流和保持云下可见的责任。

根据国际民航组织第4444号文档（6.5.3、6.5.4.3、8.9.5号定义）所述，当未完成部分或全部仪表进近程序时，应进行目视进近至地形。飞行员应目视地障，确保高度正处于或高于许可的初期进场高度。测量情况可鉴于目视进近及着陆可否被完成。对于雷达矢量，应仅在飞行员汇报机场或前方飞机可见时可授予目视进近许可，此时矢量通常应已停止。Part-NCO.OP第27页规定：不允许在跑道能见距离低于800米的情况下执行目视进近。
