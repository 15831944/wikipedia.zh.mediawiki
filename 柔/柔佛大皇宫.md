'''柔佛大王宮'''（{{lang-ms|Istana Besar}}），又称[[苏丹|苏丹]][[阿武峇卡|阿武峇卡]]皇宫，现名'''柔佛皇家博物館'''。王宮坐落在[[柔佛|柔佛]]海峡边的敦依斯邁醫生路（Jalan Tun Dr. Ismail），面向[[柔佛海峡|柔佛海峡]]，与[[新加坡|新加坡]]隔岸相望，是[[柔佛苏丹王朝|柔佛苏丹王朝]]时期苏丹的九个皇宫之一。

== 历史 ==
1864年，时任柔佛蘇丹[[阿武峇卡|阿武峇卡]]主持了柔佛大王宮奠基儀式，并命当时[[新山|新山]][[华人|华人]][[港主|港主]][[黄亞福|黄亞福]]所建。经过两年建造，於1866年正式完工。

== 简介 ==
早期剛建竣的柔佛大王宮是當時[[馬來半島|馬來半島]]最宏偉的宮殿。它不僅是作為皇家住所及[[柔佛蘇丹王朝|柔佛蘇丹王朝]]政府的行政中心<ref name="JOHOR MESTI SENTIASA JADI JOHOR - BAHAGIAN 12">[http://johorsoutherntigers.com.my/johor-mesti-sentiasa-jadi-johor-bahagian-12/ 《JOHOR MESTI SENTIASA JADI JOHOR - BAHAGIAN 12》，JOHOR Southern Tigers]</ref>，亦是接待來各地貴賓、柔佛王室舉辦冊立王儲、王儲新婚盛典，以及歷代蘇丹華誕封賜有功勳銜予有功人士之地。<ref name="柔王室博物館近期開放‧“華人禮堂”受矚目">[http://news.sinchew.com.my/node/448157 《柔王室博物館近期開放‧“華人禮堂”受矚目》，星洲網，2015-10-15]</ref>

柔佛大王宮不僅凸顯了獨特以及現代化的設計之外，也是突出了其皇家的威嚴。王宮擁有兩個擁有皇家大階梯的主要入口；面向柔佛海峽的階梯，主要是用來迎接從新加坡乘船抵達的貴賓使用的。另一頭面向花園的階梯則可迎接貴賓直接抵達大殿<ref name="JOHOR MESTI SENTIASA JADI JOHOR - BAHAGIAN 12"/>。

蘇丹阿武峇卡曾经在柔佛大王宫接见了许多国内外的贵宾，当中包括了英国[[維多利亞女皇|維多利亞女皇]]的孙子[[阿尔伯特·维克托王子_(克拉伦斯和阿文代尔公爵)|阿尔伯特·维克托王子]]以及[[夏威夷王國|夏威夷王國]]的[[卡拉卡瓦|卡拉卡瓦國王]]<ref name="JOHOR MESTI SENTIASA JADI JOHOR - BAHAGIAN 12"/>。

毗鄰柔佛大王宮則有一座禮堂，於1875年完工，主要是用於舉辦正式場合如柔佛蘇丹的華誕，亦是財政部以及其他政府部門的所在地。此禮堂裡的柱上有懸掛由當年華社領袖及港主為慶賀柔佛大皇宮落成而進貢的漢字賀聯：<ref name="JOHOR MESTI SENTIASA JADI JOHOR - BAHAGIAN 12"/><ref name="柔王室博物館近期開放‧“華人禮堂”受矚目"/><ref>[http://www.chinapress.com.my/20150915/%E7%94%B1%E8%8F%AF%E5%95%86%E9%80%B2%E8%B2%A2%E6%8E%9B%E6%BC%A2%E5%AD%97%E8%B3%80%E8%81%AF%E6%9F%94%E7%8E%8B%E5%AE%AE%E6%9C%89%E5%BA%A7%E8%8F%AF%E4%BA%BA%E7%A6%AE%E5%A0%82/ 《由華商進貢 掛漢字賀聯 柔王宮有座華人禮堂》，中國報，2015-09-15]</ref>

在柔佛大王宮區內還建有以蘇丹阿武峇卡王妹翁姑扎哈拉來命名的紮哈拉王宮以及扎哈拉禮堂。早期扎哈拉王宮是作為由新加坡直落布蘭雅搬遷至新山的蘇丹阿武峇卡家族的住所，亦是數個政府部門的辦事處，直至1940年這些部門遷至[[蘇丹依布拉欣大樓|蘇丹依布拉欣大樓]]<ref name="JOHOR MESTI SENTIASA JADI JOHOR - BAHAGIAN 12"/>。

其他位於柔佛大王宮區內的宮殿還有Marbal王宮以及斯瑪央王宮。Marbal王宮是以[[倫敦|倫敦]]的[[馬爾博羅大樓|馬爾博羅大樓]]來命名的，起初是時任衛生署署長G.H. Garlick醫生的住所，之後便成為柔佛蘇丹依布拉欣第二王子東姑阿武峇卡的住所。現在的Marbal王宮則作為柔佛王宮花園部的辦事處。斯瑪央王宮則是蘇丹後法蒂瑪的寢宮，亦是之後蘇丹依斯邁還是柔佛王儲時的住所。現在這間王宮已經拆除，僅剩一部分王宮的階梯還留著<ref name="JOHOR MESTI SENTIASA JADI JOHOR - BAHAGIAN 12"/>。
