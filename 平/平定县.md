'''平定县'''位于[[中华人民共和国|中华人民共和国]][[山西省|山西省]]东部，是[[阳泉市|阳泉市]]下辖的一个县。

== 地理 ==
===位置===
平定县地处太行山西侧，县城西北是阳泉市，东北是[[盂县|盂县]]，东边是[[河北省|河北省]][[井陉县|井陉县]]，南边是[[昔阳县|昔阳县]]，西边是[[寿阳县|寿阳县]]。县城距离阳泉市中心9公里，[[太原市|太原市]]中心127公里。

== 历史沿革 ==
平定在[[战国|战国]]时期属[[赵国|赵国]]，称上艾邑。

[[西汉|西汉]]時，[[韩信|韩信]]攻打赵国，曾在此驻兵，并修建城池，以[[榆树|榆树]]塞门，故名榆关城，也就是后来的上城。不久西漢将上艾邑改为上艾县，归[[太原|太原]]郡管理，县城在今日县城南边的[[张庄镇_(平定县)|张庄镇]]新城村。

[[曹魏|曹魏]][[黄初|黄初]]元年（公元220年），归乐平郡管理。

[[北魏|北魏]][[建国_(北魏)|建国]]元年（公元386年），因胡语谐音更名为石艾县。

[[北魏|北魏]][[太平真君|太平真君]]九年（公元448年），改名为上艾县。

[[唐朝|唐朝]][[天宝_(唐朝)|天宝]]元年（公元742年），改名为广阳县，县城迁至现在[[昔阳县|昔阳县]]的广阳村。

[[北宋|北宋]][[太平兴国|太平兴国]]二年（公元977年），宋太宗[[赵光义|赵光义]]在广阳建立了平定军，归镇州（现在的[[河北|河北]][[正定|正定]]）管理。太平兴国四年（公元979年），宋太宗把广阳县改为平定县，并把县城从40公里外的广阳城迁至榆关城，也就是现在平定县的上城。并在上城外侧新建城池，成为下城。

[[金朝|金]][[大定_(金)|大定]]二年（公元1162年），平定县升为平定州，管辖平定、乐平（今[[昔阳|昔阳]]）两县。

[[元朝|元]][[至元|至元]]二年（公元1265年），平定县、乐平县并入平定州。

[[清朝|清]][[雍正|雍正]]二年（公元1724年），把平定州升为直隶州，管辖[[盂县|盂县]]，[[寿阳县|寿阳县]]，乐平县。

1912年，[[辛亥革命|辛亥革命]]后，把平定州改为平定县。

1949年，平定县归[[晋中|晋中]]地区管理。1983年，平定县划归[[阳泉市|阳泉市]]。

== 文化 ==
===方言===
平定县方言属于[[晋语|晋语]][[并州片|并州片]]下的阳泉片（也称平辽小片）。共有声母25个，韵母39个，声调5个：阴平、阳平、上声、去声、入声。和太原话的主要区别是入声不分阴阳，而太原话则有阴入和阳入之分。<ref>{{cite web|url=http://www.docin.com/p-74934373.html|title=平定方言研究}}</ref>

== 旅游 ==
*[[娘子关|娘子关]]：是万里长城的著名关隘，有“天下第九关”之称，为历代兵家必争之地。现存关城为明嘉靖二十年（1542）所建。
*天宁寺双塔：在平定县城南关土垣上。寺址高耸，双塔并峙。塔为楼阁式，平面八角形，每层收刹甚急，轮廊形制奇异。
