'''平壤地鐵'''（{{kor|諺=평양 지하철|漢=平壤 地下鐵|羅=P'yŏngyang Chihachŏl|f=y}}）是[[朝鮮民主主義人民共和國|朝鮮]]首都[[平壤|平壤]]的[[城市軌道交通系統|城市軌道交通系統]]，始建於1960年代，1973年9月正式投入營運。地铁隧道为朝鲜战争时期已挖掘好的防空洞<ref>{{cite book|title=北京城建设计研究总院院志简编（1958-2008）|date=2008|publisher=北京城建设计研究总院}}</ref>，深度普遍在22-100米之間，某些山區路段更深入150米。

平壤地鐵於1968年動工，1973年9月6日第一條地鐵線路千里馬線通車；第二條革新線在1975年的朝鮮國慶節通車。現時系統約長24[[公里|公里]]，分成千里馬及革新等兩條支線，合共17個車站，然而其中革新線上的[[光明站_(平壤)|光明站]]則在[[金日成|金日成]]去世後被關閉，不再使用。

平壤地鐵剛通車時採用了中國[[吉林省|吉林省]][[長春|長春]]客車廠（現[[長春軌道客車|長春軌道客車]]股份有限公司）的DK4型列車，平壤地鐵目前使用的車廂都是從[[德國|德國]]進口的車廂和朝鮮仿製的德國的車廂，2016年1月1日，朝鮮自主研发生产的新型列車投入營運。

除地下鐵路系統外，平壤亦有三條地面[[有軌電車|有軌電車]]鐵路，採用 [[捷克|捷克]]、德國等國機車以及自行制造的列车。

== 平壤地铁车型 ==
[[File:MetroPyongyang.jpg|thumb]] 
[[File:Pyongyang_metro_station_2.jpg|缩略图]]
[[File:People_in_Pyongyang_Metro_01.JPG|thumb]]

中國聲稱，1973年平壤地铁正式运营时，在线路上运行的地铁车辆均为新车。其中的4列車廂来自[[中国|中国]]的[[平壤地铁DK4型电动车组|DK4]]车辆，部分于1998年卖回中国，部分几组列车则经过改造，开始运行于[[平义线|平义线]]。平壤地铁车辆宽2.7米，长18.8米（每节），由825伏[[直流电|直流电]][[第三轨供电|第三轨]]驱动。1998年起平壤市地铁启用了[[柏林地铁|柏林地铁]]使用的旧车。2016年，平壤地铁部分使用本国[[金钟泰电力机车联合企业|金钟泰电力机车联合企业]]製造的新车。

分别使用的如下三种类型的地铁：

* [[平壤地铁Gl型电动车组|Gl]]（Gisela）“迪斯拉”，源自[[东柏林|东柏林]]，于1978年—1982年间制造，60节双驱动地铁列车，现已停用。
* [[平壤地铁D型电动车组|D]]（Dora）“多拉”，源自[[西柏林|西柏林]]，于1957年—1965年间制造，108节双驱动地铁列车。

这些车辆原有的广告被覆盖后，外表被重新喷成奶色和红色。部分车厢内部悬掛有[[金日成|金日成]]、[[金正日|金正日]]的画像。由於列車都是[[德國|德國]]制造的，所以控制台上的標籤都用德語書寫。

平壤地铁的新车型於2016年投入服務，外型比舊车型更加现代化，车厢光线更明亮，设有显示停车站和行驶速度等的屏幕，以及老年人和残障人士等专座，朝鲜官方称该车辆其为朝鲜自行生产。<ref>[http://www.ditiezu.com/thread-431586-1-1.html] 平壤地鐵新車</ref>

== 特点 ==
*由於朝鮮政府並不容許商家賣廣告，所以不論地鐵車卡、候車區、車站還是電梯都是有关[[朝鮮勞動黨|朝鮮勞動黨]]和朝鲜历史的掛畫或壁畫。
*2013年以前，國外觀光客僅能在導遊的陪伴下，在兩個指定的車站間（[[千里馬線|千里馬線]][[復興站|復興站]]至[[榮光站|榮光站]]的路段）搭乘平壤地鐵一個站，不過2013年後,朝鮮政府允許遊客坐六個站（[[千里馬線|千里馬線]][[復興站|復興站]]至[[紅星站|紅星站]]的路段，也就是千里馬線全線）。
*大部份地鐵站的候車區都設有多個由[[勞動新聞|勞動新聞]]提供的閱報支架，以供候車的乘客閱讀。
