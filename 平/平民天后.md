《'''莉琪的異想世界'''》（{{lang-en|''The Lizzie McGuire Movie''}}），又译《'''麻雀变凤凰'''》、《'''麻雀变明星'''》、《'''平民天后'''》，是一部基于[[华特迪士尼公司|迪士尼]][[电视剧|电视剧]]《[[新成长的烦恼|新成长的烦恼]]》的[[喜剧|喜剧]][[电影|电影]]，于5月2日2003年（见[[2003年电影|2003年电影]]）由[[华特迪士尼影片|华特迪士尼影片]]发行。电影从主角莉琪（[[希拉里·达夫|希拉里·达夫]]饰）她从[[初级中学|初级中学]]毕业并随班级旅行至[[罗马|罗马]]的开头开始。

该影片的摄制完成于2002年末，上映于2003年5月，虽然在评论界获得的反响褒贬不一，但在当时达到了[[美国|美国]][[票房|票房]]第二名。

== 剧情 ==
Lizzie在高中畢業後，她組織了到[[羅馬|羅馬]]的旅行。

== 製作 ==
* 該影片在[[羅馬|羅馬]]拍攝。
* [[Lalaine_Vergara|Lalaine Vergara]]（饰演Miranda Sanchez）沒有在電影中出現的原因從沒有得到明確的解釋，而且成為了支持者們的話題。（她同時沒有在電視劇集最後的幾集的拍攝中出現。） 有指她的缺席是由於她當時在[[墨西哥城|墨西哥城]]度假，但真正的原因是她決定退出電視劇集的拍攝。
* 原始的電影標題是"Ciao, Lizzie!"。
* 該電影原本計畫在2003年3月上映。
* 在[[羅馬競技場|羅馬競技場]]舉行演唱會的虛構場景拍攝於[[羅馬奧林匹克體育場|羅馬奧林匹克體育場]]。
* 電影中的機場是[[溫哥華國際機場|溫哥華國際機場]]（YVR），座落於[[溫哥華|溫哥華]]。
* Lizzie與Paolo在Vespa上時，參照了[[奧黛麗·赫本|奧黛麗·赫本]]主演的《羅馬假期》。
* 在該電視劇集中常見但在電影中沒有出現的角色有 Miranda、Claire、Larry、Lanny及Violet。

== 流行文化 ==
* 在影片的開頭，Lizzie正在假唱[[英國|英國]]女子組合[[Atomic_Kitten|Atomic Kitten]]的《[[The_Tide_Is_High|The Tide Is High]]》（[[Blondie_(band)|Blondie]]原唱）。
* 《[[平民天后電影原聲大碟|平民天后電影原聲大碟]]》指當中[[希拉蕊·杜芙|希拉蕊·杜芙]]的歌曲《[[Girl_In_The_Band|Girl In The Band]]》及[[Vitamin_C|Vitamin C]]的歌曲《[[Volare|Volare]]》並沒有在電影中出現，但在電視最後所有人在找Lizzie時，Ethan正戴著耳筒聽著杜芙的歌曲；當Paolo帶Lizzie坐Vespa時，背景音樂正是Vitamin C的歌曲。
* Lizzie的爸爸曾在機場提及過[[黛安·索耶|黛安·索耶]]。該對白是「誰會寄錄影帶給[[黛安·索耶|黛安·索耶]]去令你如此尷尬？」
* 在《[[辛普森一家|辛普森一家]]》的"[[Co-Dependent's_Day|Co-Dependent's Day]]"一集中，[[荷馬|荷馬]]與[[馬芝|馬芝]]正在收看《平民天后》。
* 《[[威爾與格蕾絲|威爾與格蕾絲]]》的一集中，威爾提及格蕾絲想去看《平民天后》。

== 軼事 ==
* Lizzie的全名是Elizabeth，而在義大利Elizabeth的發音是Elisabetta。
* 在[[Disney_Channel's_So_Hot_Summer!_2006|Disney Channel's So Hot Summer! 2006]]中，這部電影於"Click It to Pick It"電影對決中勝出，打敗了電影《[[公主日記|公主日記]]》。
* 當電影在[[迪士尼頻道|迪士尼頻道]]初次上演時，任何人說的"天啊"（原文："God"）都被刪去。
