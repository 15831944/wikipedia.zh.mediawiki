《'''平沙万里行'''》（{{lang-th|มุ่งไกลในรอยทราย}}）是由[[诗琳通公主|诗琳通公主]]写作的一部[[游记|游记]]，讲述的是1990年4月7日至4月21日，诗琳通公主第二次访华；在中国的[[北京市|北京市]]，[[陕西省|陕西省]][[西安市|西安市]]，[[甘肃省|甘肃省]][[兰州市|兰州市]]、[[嘉峪关市|嘉峪关市]]、[[酒泉市|酒泉市]]、[[敦煌市|敦煌市]]，[[新疆维吾尔自治区|新疆维吾尔自治区]][[吐鲁番市|吐鲁番市]]、[[乌鲁木齐市|乌鲁木齐市]]、[[喀什市|喀什市]]，[[香港|香港]]等地游览了15天<ref>{{cite web |url=http://www.gsfao.gov.cn/news/gzkx/2016/316/16316113353664911II28AHD7DCB20B.html |title=李荣灿会见泰国驻华大使 |last= |first= |date=2016-03-16 |website=甘肃省人民政府 |publisher= |access-date=2018-08-13 |quote=}}</ref>，向泰国民众了解[[丝绸之路|丝绸之路]]文化<ref>{{cite web |url=http://public.dha.ac.cn/content.aspx?id=418640037314 |title=敦煌“佛陀之莲”，连接中泰友谊 |last= |first= |date=2018-06-25 |website=敦煌研究院 |publisher= |access-date=2018-08-13 |quote=}}</ref><ref>{{cite web |url=http://bclc-tu.com/sand1.html |title=bclc-tu มุ่งไกลในรอยทราย |last= |first= |date= |website=bclc-tu |publisher= |access-date=2018-08-13 |quote=}}</ref>。

中文版标题由诗琳通公主御笔题写，书中有公主自己手绘的插图10幅。另有诗琳通公主的中文老师顾雅炯的译本，名为《'''随公主踏沙远行'''》（{{lang-th|ตามเสด็จ “มุ่งไกลในรอยทราย”}}），与林长茂译《平沙万里行》相比少了公主的自画插图。

== 内容 ==
===第1天（1990年4月7日）===
从[[曼谷|曼谷]]出發，下午4時45分来到[[北京机场|北京机场]]。當時的国务院副總理[[吳學谦|吳學谦]]邀請詩琳通公主于[[钓鱼台国宾馆|钓鱼台国宾馆]]芳菲园举办晚宴<ref>{{cite web |url=http://bclc-tu.com/sand25330407_page1.html |title=" มุ่งไกลในรอยทราย วันเสาร์ที่ 7 เมษายน 2533 " |last= |first= |date= |website=bclc-tu |publisher= |access-date=2018-08-13 |quote=}}</ref>。

===第2天（1990年4月8日）===
