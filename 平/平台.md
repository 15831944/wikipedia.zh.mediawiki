== 科技、技術 ==
* [[系統平台|系統平台]]，一個可讓應用程式運行的框架
* [[平台遊戲|平台遊戲]]，一種電子遊戲類型
* {{link-en|平台技術|Platform technology}}，一種讓產品和製作過程被創造並得以存續的科學技術。
* [[汽車平台|汽車平台]]，一系列在不同汽車型號間通用的零部件

== 物體、構造 ==
* {{link-en|潛水台|Diving platform}}，競技跳水用的平台
* {{link-en|跳台|Jumping platform}}，自然存在的平台，或者臨時造來{{tsl|en|cliff-jumping}}的平台
* [[鑽油台|鑽油台]]，一個用來鑽探石油的平台
* [[地台|地台]]，地理學上由沉積岩覆蓋的穩定地塊
* {{link-en|平台式土墩|Platform mound}}，支撑结构的土方工程
* {{link-en|剧院平台|Platform (theatre)}}
* [[車站月台|車站月台]]，火車站中讓火車或電車停靠的區域

==经济==
