'''平安通站'''（{{jpn|j=平安通駅|hg=へいあんどおりえき|rm=Heiandōri eki|f=y}}）是位於[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[北區_(名古屋市)|北區]]平安通一丁目，為[[名古屋市營地下鐵|名古屋市營地下鐵]]之車站。

== 可利用之鐵路路線 ==
*名古屋市營地下鐵
**[[名城線|名城線]] - 車站編號為'''M11'''。
**[[上飯田線|上飯田線]] - 車站編號為'''K02'''。

== 車站結構 ==
名城線為擁有2面2線[[側式月台|側式月台]]之[[地下車站|地下車站]]，上飯田線為擁有1面2線[[島式月台|島式月台]]之地下車站，直接交叉於名城線之下方。為了乘客們的安全起見，月台設有可動式[[月台門|月台門]]。

== 歷史 ==
*1971年（[[昭和|昭和]]46年）12月20日 - 伴隨名城線市役所站至大曾根站間延伸時開業。
*2003年（[[平成|平成]]15年）3月27日 - 上飯田線開業。成為轉乘站。
