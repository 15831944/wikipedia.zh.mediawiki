{{Vgname|1=平壤賽車|ko=평양 레이서|en=Pyongyang Racer}}是由進駐[[北朝鮮|北朝鮮]]的西方科技廠商{{link-en|Nosotek|Nosotek}}開發；於2012年推出的北朝鮮史上第一款電子遊戲。<ref>{{cite web|url=http://www.ilbe.com/index.php?listStyle=webzine&mid=ilbe&document_srl=535280211|title=빨갱이필독 북한산 레이싱게임 평양레이서를 해보자|work=iLBE|date=2012年12月21日|accessdate=2013年10月20日}}</ref><ref name="AniArc">{{cite web|url=http://news.aniarc.com/news/43851/pyongyangracer|title=北朝鮮第一款網頁遊戲《Pyongyang Racer 平壤賽車》 交通美女帶領觀光北朝鮮首都|work=AniArc News|date=2012年12月21日|accessdate=2013年10月20日|deadurl=yes|archiveurl=https://web.archive.org/web/20131101193029/http://news.aniarc.com/news/43851/pyongyangracer|archivedate=2013年11月1日}}</ref>

== 契機 ==
此為[[高麗旅行社|高麗旅行社]]為了能讓更多人知道[[平壤|平壤]]的觀光景點及特色，而與NOSOTEK合作推出的遊戲。

另遊戲的主旨不在於刻意開發出華麗的作品，而是希望讓玩家能在遊戲中藉由駕駛著車輛、觀賞平壤的風味。<ref>{{cite web|url=http://www.pyongyangracer.co/racer_about.html |title=ABOUT KORYO TOURS PYONGYANG RACER |work=平壤賽車 |accessdate=2013年10月20日 }}</ref>

== 遊戲方式 ==
遊玩方法同大多數的賽車遊戲，即設法在最少的時間內跑完全程、以便獲得較高的成績。而遊戲中引用的車輛則是北韓自製的[[平和汽車|平和汽車]]。<ref name="AniArc"/>

在遊戲過程中跑道裡不時穿插著印有平壤當地代表景點如[[平壤體育館|平壤體育館]]、[[凱旋門_(平壤)|凱旋門]]、[[柳京飯店|柳京飯店]]等圖片，當玩家接觸到那些圖片時便會顯示該景點的觀光資訊。<ref name="AniArc"/>

另外若玩家未能即時補充汽車燃料、或是造成三次違規紀錄時便會造成遊戲結束。<ref>{{cite web|url=http://www.appledaily.com.tw/realtimenews/article/new/20121223/158031/ |title=北韓首推「平壤賽車」電玩　美女警抓違規 |work=[[蘋果日報_(台灣)|蘋果日報]] |date=2012年12月23日 |accessdate=2013年10月20日 }}</ref>

== 反應 ==
遊戲評論網站HarcoreGaming 101認為在遊玩《平壤賽車》的過程中，遊戲裡的平壤市中除了玩家所駕駛的車輛外、幾乎無其它任何交通工具在路上，這種虛假、死氣沉沉的氣氛讓人不夠自在。<ref>{{cite web|url=http://www.hardcoregaming101.net/pyongyang/racer.htm|title=Pyongyang Racer (평양 레이서) - Web Browser (2012)|work=Hardcoregaming101|author=Iwant|date=2013年9月21日|accessdate=2013年10月20日}}</ref>

資訊網站Dvice.Com則表示雖然《平壤賽車》在官方網頁上聲稱這是一個有趣的作品，但看了實際的畫面卻令人傻眼，另外有著不佳的遊戲操控方式、詭異的音樂風格等問題，但整體還比另一款爛遊戲《[[大貨車極限競賽|大貨車極限競賽]]》有可玩性。<ref>{{cite web|url=http://www.dvice.com/2012-12-20/pyongyang-racer-north-koreas-first-online-video-game |title=Pyongyang Racer: North Korea's first online video game |work=Dvice.Com |author=Evan Ackerman |date=2012年12月20日 |accessdate=2013年10月20日}}</ref>

Gamesinasia評論此作品遊戲畫面像是[[Play_Station|Play Station]]一代主機的水準，而遊戲操控的靈活性方面不佳、像是一頭大熊騎在[[單輪車|單輪車]]上陷入泥沼後動彈不得的模樣。另外遊戲中有著沒有特別作用的毫無意義設定，例如像是讓玩家駕駛的車輛發出喇叭聲的按鍵功能，而令人發噱的遊戲配樂是少數有趣的地方。<ref>{{cite web|url=http://www.gamesinasia.com/north-korea-computer-game-pyongyang-racer/|title=I Just Played North Korea’s First Computer Game: Pyongyang Racer|work=Gam In Asia|author=C. Custer|date=2012年12月21日|accessdate=2013年10月20日}}</ref>
