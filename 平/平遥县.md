'''平遥'''是中国[[山西省|山西省]][[晋中市|晋中市]]下辖的一个县，面积1260平方公里，人口49万，县人民政府驻[[古陶镇|古陶镇]]。平遥是[[国家历史文化名城|国家历史文化名城]]，包括[[双林寺_(平遥)|双林寺]]和[[镇国寺|镇国寺]]在内的[[平遥古城|平遥古城]]已被列为[[世界文化遗产|世界文化遗产]]。

== 历史 ==
[[北魏太武帝|北魏太武帝]]时，将平陶县（原址在今[[文水县|文水县]]西南）东迁至今址，并因避太武帝拓跋焘名讳，改平陶为平遥。

== 地理 ==
平遥地处[[汾河|汾河]]东岸、[[太原盆地|太原盆地]]的西南端，与另一座国家历史文化名城[[祁县|祁县]]相毗邻。

== 交通 ==
大西高铁、[[南同蒲铁路|南同蒲铁路]]、[[京昆高速公路|京昆高速公路]]、汾邢高速纵贯县境。

== 人口 ==
2010年[[第六次全国人口普查|第六次全国人口普查]]全县常住人口为502712人。<ref>[http://www.pyonline.net/lh2008/info.asp?id=31742 平遥县第六次全国人口普查主要数据公报显示，全县常住人口为502712人]</ref>

== 经济 ==
经济以旅游业和农业为主，主产[[粮食|粮食]]、[[棉花|棉花]]，特产[[平遥牛肉|平遥牛肉]]、推光[[漆器|漆器]]等。其中平遥牛肉名声颇大，有“[[平遥牛肉|平遥牛肉]][[太谷饼|太谷饼]]”的民歌歌词。
