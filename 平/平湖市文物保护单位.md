'''平湖市文物保护单位'''是中国[[浙江省|浙江省]][[平湖市|平湖市]]的市级[[文物保护单位|文物保护单位]]，截止到2017年共公布八批，共44处（已升级至浙江省文物保护单位和全国重点文物保护单位的不再列入）<ref>{{cite web |title=《平湖市新增２处省级文物保护单位》|publisher= 浙江省文物局 |url=http://www.zjww.gov.cn/news/2017-02-23/1166742822.shtml}}</ref>。

*第一批：1984年1月23日公布，数量不详。
*第二批：1985年11月14日公布，数量不详。
*第三批：1998年2月5日公布，数量不详。
*第四批：2004年6月18日公布，数量不详。
*第五批：公布时间及数量不详。
*第六批：2010年2月10日公布，共5处，含[[庄桥坟遗址|庄桥坟遗址]]（2013年列为全国重点文物保护单位）、[[陆氏宅第|陆氏宅第]]、[[侵华日军登陆处|侵华日军登陆处]]、[[百人坑_(平湖)|百人坑]]和[[溪漾指挥坟|溪漾指挥坟]]<ref>{{cite web|title=《平湖年鉴（2011）》|publisher=平湖史志网|url=http://phszw.pinghu.gov.cn/ucms/html/phszw/col982/2012-05/11/20120511101141884745368_1.html|access-date=2018-08-07|archive-url=https://web.archive.org/web/20180807124921/http://phszw.pinghu.gov.cn/ucms/html/phszw/col982/2012-05/11/20120511101141884745368_1.html|archive-date=2018-08-07|dead-url=yes}}</ref>。
*第七批：2011年6月3日公布，共7处，含[[俞创硕故居|俞创硕故居]]、[[葛氏祠堂|葛氏祠堂]]、[[坟庵桥遗址|坟庵桥遗址]]等。
*第八批：2015年1月12日公布，共20处<ref>{{cite web |title=《平湖市人民政府关于公布平湖市第八批文物保护单位的通知》|publisher= 平湖市政府 |url=http://www.pinghu.gov.cn/ucms/html/main/col124/2015-01/22/20150122093630071293592_1.html}}</ref>。
