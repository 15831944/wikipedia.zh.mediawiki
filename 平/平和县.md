'''平和县'''位于中国[[福建省|福建省]]南部，是[[漳州市|漳州市]]下辖的一个县。

== 行政区划 ==
辖10个镇、5个乡：<ref>{{PRC admin/ref sgc|35/06/28/000/000}}</ref>
{{PRC admin/children|35/06/28/000/000}}。

== 民族 ==
平和是福建省重点侨乡之一，也是[[臺灣|臺灣]][[閩南裔台灣人|閩南人]]移民的重要[[祖籍|祖籍]]地，居民以[[闽南人|闽南人]]為主，[[客家人|客家人]]次之，大致通行[[閩南語|閩南語]]。

== 资源 ==
平和境内矿藏种类有[[金|金]]、[[银|银]]、[[铜|铜]]等10多种，其中[[石英石|石英石]]、[[高岭土|高岭土]]、[[花岗岩|花岗岩]]分布广、贮量大、品位比较高。

南胜境内有两座大山，名为大矾山、小矾山。山名其名，有丰富的[[矾|矾]]矿。

== 特产 ==
[[File:Dapingqiao_-_pomelo_-_Walmart_sign_-_DSCF4121.JPG|thumb]]
* 水果：著名的有琯溪[[蜜柚|蜜柚]]，坂仔[[香蕉|香蕉]]。
* 茶叶：平和是[[乌龙茶|乌龙茶]]主要产区，以[[黄旦|黄旦]]、[[奇兰|奇兰]]、[[八仙茶|八仙茶]]出名。
* 食品：南胜麻枣，平和枕头饼等。
* 陶瓷︰宋朝至明朝中葉，時海上絲路出口茶葉與陶瓷，有相當多民生用陶瓷器即由平和窯場出產，後來因清鄭交戰與海禁，泉、漳、潮淪為戰區，漳州窯遂匠去窯癈，令歐洲人痴迷的[[克拉克瓷|克拉克瓷]]產區難以考據，至1990年代才確認精美的克拉克瓷即產自[[漳州窯|漳州窯]]。

== 名胜古迹 ==
平和县有1个[[国家4A级旅游景区|国家4A级旅游景区]]、3个[[全国重点文物保护单位|全国重点文物保护单位]]、2个省级风景名胜区、1个省级农业旅游示范点、9个省级文物保护单位<ref>[http://www.hdzxw.com/news/93/n-208293.html 平和一地游 站站都精彩]</ref>。

境内有千年古刹的[[佛教|佛教]]圣地[[三平寺塔殿|三平寺]]，有“闽南第一山”的[[灵通岩|灵通岩]]。南胜五寨交界处南胜克拉克瓷遗址。
