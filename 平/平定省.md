'''平定省'''（{{vie|v='''Tỉnh Bình Định'''|hn=省平定}}）位於[[越南|越南]]，區屬[[南中部|中南沿海地區]]，為越南的武術之鄉，流行各種越南傳統武術。

== 行政區劃 ==
平定省下轄1市1市社9縣，省會位於[[歸仁市|歸仁市]]。
* [[歸仁市|歸仁市]]（Thành phố Qui Nhơn）
* [[安仁市社|安仁市社]]（Thị xã An Nhơn）
* [[安老縣_(平定省)|安老縣]]（Huyện An Lão）
* [[懷恩縣|懷恩縣]]（Huyện Hoài Ân）
* [[懷仁縣_(越南)|懷仁縣]]（Huyện Hoài Nhơn）
* [[符吉縣|符吉縣]]（Huyện Phù Cát）
* [[符美縣|符美縣]]（Huyện Phù Mỹ）
* [[綏福縣|綏福縣]]（Huyện Tuy Phước）
* [[西山縣|西山縣]]（Huyện Tây Sơn）
* [[耘耕縣|耘耕縣]]（Huyện Vân Canh）
* [[永盛縣_(平定省)|永盛縣]]（Huyện Vĩnh Thạnh）

== 地理 ==
此省地形多變，由西向東依次是高原、丘陵、平原及海岸。

== 經濟 ==
* 以[[漁業|漁業]]與[[農耕|農耕]]為主
* 2010年歸仁市將成為特別行政區

== 歷史 ==
平定省的歷史多與占族古國有關。
* 980年[[占族|占族]]放棄首都[[聖子修道院|美山]]（{{lang|vi|Mỹ Sơn}}），遷至平定省的Vijaya([[毘闍耶|毘闍耶]])。
* 1470年被[[越族|越族]]入侵
