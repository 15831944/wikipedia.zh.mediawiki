'''福建省平潭综合实验区'''，简称'''平潭综合实验区'''或'''平潭实验区'''，是[[福建省|福建省]]下属的一个[[综合实验区|综合实验区]]，位于福建省[[福州市|福州市]][[平潭县|平潭县]]。

2009年7月，福州市人民政府在平潭县设立“福州（平潭）综合实验区”。2012年，更名为“福建省平潭综合实验区”。<ref name="nvs " >{{cite web |url = http://news.xinhuanet.com/politics/2012-02/15/c_122703407.htm |title=福建平潭“五个共同”探索两岸合作新模式 |accessdate= 2012年|work= |publisher= 新华网 |language =zh-cn}}</ref><ref name="xlwac " >{{cite web |url = http://news.sina.com.cn/c/2012-03-23/111024163302.shtml|title=学者称台方不应对平潭扣一国两制实验区帽子 |accessdate= 2012年|work= |publisher=新浪网 |language =zh-cn}}</ref>2013年7月，平潭实验区取得行使[[设区市|设区市]]的管理权限，不再是福州市政府的下属机构，改归福建省政府直属。

平潭由126个岛屿组成，主岛[[海坛岛|海坛岛]]面积271平方公里，为全国第五大岛，同時為離台灣最近的陸地，2009年5月14日，国务院正式下发《关于支持福建省加快建设海峡西岸经济区的若干意见》，[[海西經濟特區|海西經濟特區]]建设从区域战略上升为国家战略。同年7月底召开的福建省委八届六次全会上正式作出了设立福州（平潭）综合实验区的决定，建立两岸更加紧密合作交流的区域平台，決議把平潭建设成为探索两岸合作新模式的示范区。

== 历史 ==
2009年，[[中华人民共和国国务院|中华人民共和国国务院]]发表了《关于支持福建省加快建设海峡西岸经济区的若干意见》，随后在[[福建省委|福建省委]]八届六次全会宣布“平潭综合实验区”正式成立。<ref name="nvs " /><ref name="xlwac " />国务院总理[[温家宝|温家宝]]在十一届全国人大五次会议上作政府工作报告时指出，要全面深化经济[[金融|金融]]合作，推动两岸经济合作框架协议后续商谈取得新进展;加快海峡西岸经济区建设;积极扩大各界往来，开展文化、教育等交流。之後[[福建|福建]]对平潭投资逐年增长，由2010年的100亿元人民币到2011年的303亿元，[[十二五規劃|十二五規劃]]期间更将达到2500亿元的规模。

2011年起平潭綜合實驗區管委會開始招聘台灣專才前往，並以區管委會副主任、區經濟發展局副局長...等職務徵人，年薪皆達台幣數百萬，[[中華民國政府|中華民國政府]]則表示依現行法規若是中華民國國民前往大陸任官職為非法，會被處以罰金。<ref>[http://hunter.1111.com.tw/ch/kk1.asp 平潭綜合實驗區管委會開始招聘台灣專才] {{webarchive|url=https://web.archive.org/web/20141006122122/http://hunter.1111.com.tw/ch/kk1.asp |date=2014-10-06 }}</ref>

2014年3月[[全国人大代表|全国人大代表]]、福建省省长[[苏树林|苏树林]]表示，平潭作为对台先行先试区，不能复制大陆任何一个地区或城市的模式，必须开创出属于福建的平潭符号。<ref>中新网北京訊 2014.3.6。平潭全面深化对台先行先试区</ref>苏树林说“海西建设离不开平潭，而平潭最突出的就是对台”，除了建设面积达18平方公里的两岸区域金融中心外，平潭还将重点培养[[石化|石化]]、汽车、电子、精密制造等10家年销售收入超百亿的台资企业，并打造闽台合作石化、信息产业超千亿产业集群。

2017年的报道称，台籍人士除医疗、养老保险进入大陆体系外，在行政、医疗等亦享受[[超国民待遇|超国民待遇]]。主政平潭的平潭综合实验区党工委书记[[张兆民|张兆民]]更主张对台籍人士“高看一等，厚爱一筹，宁可我们没有，也要台胞优先。”，公开要求大陆民众为台籍人士放弃自身权益<ref>{{Cite web |url=http://news.cnr.cn/native/gd/20170307/t20170307_523642425.shtml |title=人大代表张兆民：对台胞“高看一等、厚爱一筹” 平潭发展“三多一快” |accessdate=2017-08-20 |author=记者：张子亚、郭婕妤 |date=2017-03-07|work= |publisher=央广网|language=Zh-hans }}</ref>。

2018年[[BBC|BBC]]中文網報導：「事隔多年，平潭島硬體建設有所進展，免稅市場、創意園等簇新建築都已經落成營運，但島上人煙仍然稀少、連打車都不太容易。有曾經對平潭雄心壯志的台商抵受不住經營艱難，舉手投降、黯然撒出。」<ref>[http://www.bbc.com/zhongwen/trad/chinese-news-41620671 「兩岸共同家園」——福建平潭讓台灣人親近大陸了嗎？ - BBC 中文网]</ref>

== 发展模式 ==
福建省省长[[苏树林|苏树林]]表示，平潭综合实验区是以“五个共同”、“三个放”来开发和治理，即：共同规划、共同开发、共同经营、共同管理、共同受益；三个放：放地、放权、放利。<ref name="nvs " /><ref>{{cite web |url=http://news.xinhuanet.com/politics/2012-02/17/c_122713450.htm?prolongation=1 |title= 权威部门详解平潭发展规划|accessdate= 2012年|work= |publisher=新华网 |language =zh-cn}}</ref>

== 制度 ==
=== 管理制度 ===
[[社区|社区]]、[[学校|学校]]、[[医院|医院]]、[[公共文化设施|公共文化设施]]等都将设立台籍官员。<ref name=" vns" >{{cite web |url=http://news.xinhuanet.com/politics/2012-02/16/c_122710715.htm?prolongation=1 |title=福建平潭：面向台湾选干部 实行“双币制” |accessdate= 2012年|work= |publisher=新华网 |language =zh-cn}}</ref>

=== 通关制度 ===
一线放宽、二线管住、人货分离、分类管理<ref name="nvs " />

=== 税收政策 ===
减免台籍投资租金；企业所得税和个人所得税优惠，由[[福建省政府|福建省政府]]补贴；设立口岸免税店。<ref name="nvs " />

=== 货币制度 ===
“[[双币制|双币制]]”，平潭综合实验区允许平潭的银行机构和台湾地区银行之间开立人民币同业往来账户和[[新台币|新台币]]同业往来账户；允许平潭符合条件的银行机构为境内外企业以及个人开人民币账户和新台币账户。<ref name="nvs " /><ref name=" vns" />

=== 水路 ===
海峡两岸的船舶可以直航，目前已经开通了「[[海峡号|海峡号]]」和「丽娜号」客货滚装船，每周固定班次，分别往返：平潭--台北，平潭--台中。<ref name="nvs " /><ref>{{cite web | url = http://www.taiwancsf.com/Content.aspx?sn=7 | title = CSF海峽高速-船班與票價資訊 | accessdate = 2016-09-08}}</ref>

=== 公路 ===
台湾[[机动车|机动车]]，可以在平潭综合实验区自由行驶。<ref name="nvs " />

=== 鐵路 ===
[[福平鐵路|福平鐵路]]已於2013年底動工，預計於2019年通車。

== 人力资源 ==
平潭综合实验区实行“四个一千”的人才工程，即：“从2012年起，用5年时间，面向台湾引进1000名专才，面向海内外招聘1000名高层次人才，从省内选派1000名年轻干部到平潭工作，培养1000名实验区人才。”<ref name="nvs " /><ref name=" vns" />

== 争议 ==
=== 正面論述 ===
2012年3月20日，[[新党|新党]][[新黨#新黨歷任領導人|主席]][[郁慕明|郁慕明]]宣布成立“[[台湾平潭关系协会|台湾平潭关系协会]]”。他说：“平潭实验区在大陆，不是在台湾，‘[[一国两制|一国两制]]’不是台湾需要考虑的问题，况且在他们的领土上，接受我们的管理制度，有什么好怕的。”<ref name=" jvs" /><ref>{{cite web |url = http://news.ifeng.com/taiwan/3/detail_2012_03/20/13321635_0.shtml |title=新党成立台湾平潭关系协会 积极开发平潭实验区 |accessdate= 2012年|work= |publisher=凤凰网 |language =zh-cn}}</ref>

2012年3月22日，[[中华人民共和国|中华人民共和国]][[国台办|国台办]]主任[[王毅|王毅]]表示：“平潭实验区是一个经济项目，不是政治项目，也不是‘[[一国两制|一国两制]]’项目，台湾没搞清楚是怎么回事。”这是中华人民共和国官方表态平潭综合实验区属于[[经济体系|经济体系]]而不是[[政治体系|政治体系]]。<ref name=" jvs" /><ref>{{cite web |url = http://news.ifeng.com/taiwan/3/detail_2012_03/21/13347602_0.shtml |title=王毅：平潭开发区不是政治项目 也不是一国两制项目 |accessdate= 2012年|work= |publisher=凤凰网 |language =zh-cn}}</ref>

2012年3月28日，[[中华人民共和国|中华人民共和国]][[国台办|国台办]]新闻发言人[[杨毅|杨毅]]在回答记者问时说：“平潭综合实验区的开放开发是属于经济范畴的事，不是在进行‘[[一国两制|一国两制]]’实验。平潭对所有境内外投资者开放，欢迎台湾同胞自愿参与，互利合作。”<ref>{{cite web|url=http://taiwan.huanqiu.com/liangan/2012-03/2564239.html|title=台办：平潭实验区不是在进行“一国两制”实验|accessdate=2012年|work=|publisher=环球时报|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120331040727/http://taiwan.huanqiu.com/liangan/2012-03/2564239.html|archivedate=2012-03-31}}</ref>

=== 负面論述 ===
[[中华民国行政院长|中华民国行政院长]][[陈冲|陈冲]]表示：“平潭去年才改为[[特别市|特别市]]，当地面积只有300多平方公里，人口39万人，基础建设条件不完善，但口号喊得漂亮，目的不单纯。”<ref name=" jvs" /><ref name="shwx " >{{cite web |url = http://roll.sohu.com/20120323/n338684232.shtml |title=国台办主任王毅：平潭综合实验区不是政治项目 |accessdate= 2012年|work= |publisher=搜狐网 |language =zh-cn}}</ref>

2012年3月15日，[[中华民国陆委会|中华民国陆委会]][[刘德勋|刘德勋]]表示：“不接受平潭实验区‘[[一国两制|一国两制]]’原则，台湾中小企业别被误导。按照《[[两岸人民关系条例|两岸人民关系条例]]》，台湾人民若前往大陆担任公职将处以10万至50万元新台币罚款。”<ref name="jvs">{{cite web|url=http://taiwan.huanqiu.com/exclusive/2012-03/2549856.html|title=平潭实验区引爆两岸舌战|accessdate=2012年|work=|publisher=环球时报|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120325192136/http://taiwan.huanqiu.com/exclusive/2012-03/2549856.html|archivedate=2012-03-25}}</ref><ref name="shwx " /><ref>{{cite web |url = http://news.sina.com.cn/c/2012-03-16/135924126423.shtml |title=台当局否认与大陆共同规划福建平潭实验区 |accessdate= 2012年 |work= |publisher=新浪网 |language = zh-cn}}</ref>

前[[中华民国陆委会|中华民国陆委会]]主委[[吴钊燮|吴钊燮]]表示：“平潭岛是[[海峽西岸經濟區|海西特区]]第一个试点，未来若发展成功，台湾恐怕失去存在价值；而且大陆想借海西特区把台湾矮化为[[地方政府|地方政府]]，趁机推行‘[[一国两制|一国两制]]’。”<ref name="jvs" /><ref name="shwx" />

[[台联党|台联党]]主席[[黄昆辉|黄昆辉]]表示：“平潭实验区是‘统一特区’，支持者就是‘统一中介商’和‘红色代言人。”<ref name="jvs" />
