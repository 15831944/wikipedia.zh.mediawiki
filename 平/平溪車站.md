'''平溪車站'''位於[[台灣|台灣]][[新北市|新北市]][[平溪區|平溪區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[平溪線|平溪線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]兩座，僅包夾一股道，形成[[西班牙式月台布局|西班牙式月台布局]]。<ref>{{cite news|title=平溪天燈節 火車上下車分流、分段營運|url=https://video.udn.com/news/437603|publisher=聯合影音|date=2016-02-05|accessdate=2017-02-20}}</ref>

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#簡易站|簡易站]]。
*行駛車輛為[[台鐵DR1000型柴油客車|DR1000型柴油客車]]，等級為[[臺鐵區間車|區間車]]。
*昔日可供列車交會，現全部於[[十分車站|十分站]]交會。
*站外的石底街設有「平溪」公車站牌，可透過新北市轄795路公車前往[[菁桐車站|菁桐車站]]、[[深坑區|深坑區]]、[[木柵|木柵]]等地（經[[台北捷運|台北捷運]][[文湖線|文湖線]][[木柵站|木柵站]]）。
*本站有 "鐵路之旅-小站巡禮紀念章"之戳章,可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1929年10月1日：設「石底驛」。
*1954年2月1日：改為「平溪」並由簡易站改為三等站<ref>《台灣鐵路管理局公報》43年第15號，臺灣鐵理管理局</ref>。
*2001年7月16日：管理站由[[十分車站|十分站]]改為[[瑞芳車站|瑞芳站]]。
*2013年7月1日：本站站房、廁所開始整修。
*2013年9月30日：啟用多卡通刷卡機。
*2014年4月：本站站房、廁所完工啟用。
*2016年2月14日：第二月台啟用。
