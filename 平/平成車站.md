'''平成站'''（{{jpn|j=平成駅|hg=へいせいえき|rm=Heisei eki|f=y}}）是位於[[日本|日本]][[熊本縣|熊本縣]][[熊本市|熊本市]][[中央區_(熊本市)|中央區]]{{link-ja|平成 (熊本市)|平成 (熊本市)|平成}}2丁目的[[九州旅客鐵道|九州旅客鐵道]]（JR九州）車站，屬於[[豐肥本線|豐肥本線]]。

== 站名由來 ==
站名來源於所在地地名，而地名則是來源於[[元號|元號]]「[[平成|平成]]」。車站周圍主要為[[住宅區|住宅區]]，因在1990年代初期完成[[團地|團地]]開發，因此命名為「平成」。<ref>{{Cite web|title=「平成駅」「昭和駅」「大正駅」はどこにある？ 平成最後に元号駅で“タイムスリップ”|url=https://bunshun.jp/articles/-/9633|work=文春オンライン|accessdate=2019-03-24|first=鼠入|last=昌史}}</ref>

== 歷史 ==
* 1992年7月15日 - 開業<ref name="sone27">{{Cite book|author=[[曽根悟|曽根悟]]|title=週刊「歴史でめぐる鉄道全路線 国鉄・JR」|editor=朝日新聞出版分冊百科編集部（編輯）|publisher=[[朝日新聞出版|朝日新聞出版]]|series=週刊朝日百科|volume=27号・豊肥本線／久大本線|pages=14-15|date=2010-01-24}}</ref>。
* 1999年10月1日 - [[豐肥本線|豐肥本線]]改為[[電氣化鐵路|電氣化鐵道]]，在該站設置[[列車交會|交會]]用的2號站台。
* 2006年3月18日 - 改為[[有人車站|有人]][[業務委託車站|業務委託車站]]。<ref name="委託"/>
* 2012年12月1日 - 導入交通[[智能卡|智能卡]][[SUGOCA|SUGOCA]]。<ref>{{Cite web|title=SUGOCAのご利用可能エリアを平成24年12月1日（土）に拡大します|url=http://www.goto-sangyo.co.jp/wp-content/uploads/2012/10/f9137a10e4aff4daa19867a15da896b7.pdf|publisher=[[五島產業汽船|五島產業汽船]]|author=[[九州旅客鐵道|九州旅客鐵道]]|date=2012.9.19|accessdate=2019/03/24}}</ref>

== 車站構造 ==
2面2線[[側式站台|側式站台]]的[[地面車站|地面車站]]，兩座站台之間設有跨線橋，而車站上方則為一座公路橋。<ref name="zeneki38"/><ref>{{Cite web|title=変わりゆく時代乗せ 熊本・平成駅（探検！九州・沖縄）|url=https://www.nikkei.com/article/DGXMZO41078880Y9A200C1MLY000/|work=日本経済新聞 電子版|accessdate=2019-03-26|language=ja}}</ref>
