'''平凉市'''是[[中华人民共和国|中华人民共和国]][[甘肃省|甘肃省]]下辖的[[地级市|地级市]]，位于甘肃省东部。此地有“旱码头”之称是古「絲綢之路」必經重鎮，史稱「西出長安第一城」。

== 历史 ==
[[北周|北周]][[建德|建德]]元年（572年），置平凉县，隶原州长城郡。[[唐|唐]][[天寶_(唐朝)|天宝]]元年（742年）改州为郡，原州改为平凉郡。[[贞元|贞元]]四年（788年），恢复平凉县建制。[[贞元|贞元]]十九年（803年），原侨置于灵台县百里城的行原州移至平凉城。[[元和|元和]]四年（809年），置行渭州于平凉，省平凉县。[[广明|广明]]元年（880年），[[吐蕃|吐蕃]]攻占平凉城。[[后唐|后唐]][[清泰|清泰]]二年（935年），复置平凉县，初属泾州，后改属渭州。

[[北宋|北宋]][[政和|政和]]七年（1117年），渭州升为平凉军节度。[[金|金]][[天会|天会]]九年（1131年）改渭州为平凉府。[[天眷|天眷]]二年（1139年）复称渭州。[[大定|大定]]二十七年（1187年）升渭州为平凉府。[[兴定|兴定]]三年（1219年）置陕西西路行省于平凉。

[[元代|元代]]置平凉府，属[[陕西等处行中书省|陕西等处行中书省]]。[[明|明]][[洪武|洪武]]九年（1376年）平凉府属[[陕西布政使司|陕西布政使司]]。[[清|清]][[康熙|康熙]]八年（1669年），平凉改属甘肃平庆道。[[咸丰|咸丰]]九年（1859年），改为分巡平庆泾固道。[[同治|同治]]八年（1869年），改称平庆泾固化盐法兵备道。[[光绪|光绪]]三十三年（1907年），改为分巡平庆泾固化道。

民国2年（1913年），改分巡平庆泾固化道为陇东道。民国3年（1914年），改陇东道为泾原道。民国23年（1934年），设平凉行政督察区。民国25年（1936年），平凉行政督察区改称第二行政督察区。1949年7月，中共在华池县悦乐镇置平凉分区，8月徙治平凉县。

1950年2月，析平凉县城区设平凉市；5月，静宁、海原、西吉3县由定西分区划归平凉分区，庄浪县划归天水分区；10月，化平县改名泾源县。1953年10月29日，析出西吉、海原、固原3县成立甘肃省西海固回族自治区。1955年10月，[[庆阳专区|庆阳专区]]并入[[平凉专区|平凉专区]]。1956年1月，庄浪县划归平凉专区。1958年3月，隆德、泾源县划归[[宁夏回族自治区|宁夏回族自治区]]；12月，平凉县市合并。1961年11月，复设庆阳专区。12月15日，恢复华亭、灵台、庄浪和崇信县建制；1964年6月，平凉市改为[[平凉县|平凉县]]。

1983年7月，复设平凉市（县级）。2002年6月，撤销[[平凉地区|平凉地区]]和县级平凉市，设立地级平凉市，原县级平凉市改为崆峒区。2018年8月，撤销华亭县，设立县级华亭市。

== 地理 ==
平凉的地理位置在北纬34°54′-35°46′和东经105°20′-107°51′之间，东接[[陕西省|陕西省]][[咸阳市|咸阳市]]，南邻陕西省[[宝鸡市|宝鸡市]]和甘肃省[[天水市|天水市]]，西接甘肃省[[定西市|定西市]]和[[白银市|白银市]]，北邻[[宁夏回族自治区|宁夏回族自治区]][[固原市|固原市]]和甘肃省[[庆阳市|庆阳市]]。其地理形貌主要是[[黄土高原|黄土高原]]的丘陵地带，森林覆盖率为22.3%，高度在海拔890米和2957米之间。面积1.1万平方公里，人口224万。

=== 气候 ===
平凉市年平均气温为7.4至10.1摄氏度，属于[[温带干旱半干旱气候|温带干旱半干旱气候]]，半干旱，半潮湿。年降水量在420至600毫米之间。平均日照总时数为每年2144至2380小时，无霜期为156至188天。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为2068033人<ref>{{cite book |author=平凉市统计局|title=《平凉市2010年第六次全国人口普查主要数据公报》}}</ref>。其中，男性人口为1040603人，占50.32%；女性人口为1027430人，占49.68%。人口性别比（以女性为100）为101.28。0－14岁人口为398005人，占19.25%；15－64岁人口为1487252人，占71.92%；65岁及以上人口为182776人，占8.84%。

到2015年末，全市常住人口209.8万人,比2014年末增加0.57万人。城镇人口76.09万人,占常住人口的36.27%,比上年提高1.8个百分点。按年龄分,0-14岁人口37.83万人,占18.03%；15-64岁人口152.73万人,占72.8%；　65周岁及以上人口19.24万人,占9.17%。

全市有回、蒙古、满、藏、彝、维吾尔、苗、壮、布依、朝鲜、侗、瑶、白、土家、哈尼、哈萨克、黎、僳僳、佤、畲、东乡、土、达斡尔、羌、撒拉、锡伯、普米、塔吉克、怒、保安、裕固、独龙、赫哲族共33个少数民族，少数民族人口16.2万多人，占全市总人口的7.3%，其中回族约16万人，占少数民族总人口的98.8%；分布在崆峒区的大寨、上杨、峡门、白庙、寨河、大秦、西阳等7个乡和华亭县神峪、山寨2个乡，共有110个回族村，530个纯回族社。

全年人口出生率12.31‰,比2014年上升0.12个千分点；人口死亡率6.23‰,比2014年上升0.04个千分点；人口自然增长率6.08‰。<ref>{{Cite web|url=http://pl.plmh.cn/zjpl/rkmz/201510/t20151028_1634.html|title=中国·平凉-人口民族|accessdate=2017-01-22|work=pl.plmh.cn}}</ref>

== 经济 ==
平凉市最重要的产业是农业，其次有[[煤矿|煤矿]]和[[石灰岩|石灰岩]]矿。2006年市内生产总值为123亿人民币元。

== 交通 ==
[[312国道|312国道]]和[[宝中铁路|宝中铁路]]通过平凉市。

[[平定高速公路|平定高速公路]]、[[银武高速公路|银武高速公路]]、[[西平铁路|西平铁路]]和[[天平铁路|天平铁路]]正在建筑中。

[[平凉站|平凉站]]是宝中铁路沿线车站，距离[[宝鸡站|宝鸡站]]203公里，隶属[[兰州铁路局|兰州铁路局]]，为一个[[三等站|三等站]]。
