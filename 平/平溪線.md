{{平溪線RDT}}
'''平溪線'''是由[[臺灣鐵路管理局|臺灣鐵路管理局]]所經營的傳統鐵路支線，最初是為了運輸自沿線礦坑開採而得的[[煤礦|煤礦]]而興建此路線，後來並兼辦客運。

== 路線資料 ==
*經營管轄：[[臺灣鐵路管理局|臺灣鐵路管理局]]
*路線距離：三貂嶺＝菁桐間12.9 公里 <ref>{{cite web|title=各站營業里程（台灣鐵路管理局）|url=http://service.tra.gov.tw/tw/CP/16324/mileages-5.aspx|deadurl=yes|archiveurl=https://web.archive.org/web/20090720114134/http://service.tra.gov.tw/tw/CP/16324/mileages-5.aspx|archivedate=2009-07-20}}</ref>
*[[軌距|軌距]]：1,067毫米
*車站數：7（含起訖車站）
*開業時間：1921年7月全線開通
*雙線區間：無，全線單線
*[[電氣化鐵路|電化]]區間：無
雖然只有一座設在[[市道106號|市道106號]]的公路平交道，但仍有無數個行人穿越區，因其單線非[[電氣化鐵路|電氣化鐵路]]，眾多民眾也直接在鐵軌上步行。

== 簡介 ==
[[File:Pingxi_Line_Edmondson_Tickets.JPG|thumb]]。]]
平溪線於1921年（大正10年）7月全線完工，原本是[[台陽礦業株式會社|台陽礦業株式會社]]所出資興建的運煤專用鐵路，時官方稱之為台陽礦業株式會社石底線，簡稱石底線。1929年（昭和4年）[[台灣總督府|台灣總督府]][[台灣總督府交通局鐵道部|鐵道部]]收購路線，定名為平溪線，經過整建之後開始兼辦客運。

1980年代時，由於平溪區內煤礦場相繼關閉及人口流失等因素，平溪線的營運虧損。臺灣鐵路管理局曾打算停止平溪線鐵路，但經過地方人士的積極爭取而保留。

平溪線為台鐵六條客運支線之一（平溪線、[[內灣線|內灣線]]、[[六家線|六家線]]、[[集集線|集集線]]、[[深澳線|深澳線]]、[[沙崙線|沙崙線]]），也為目前仍營運的鐵路支線當中歷史最悠久，也是風景最美麗的客運支線。平溪線沿[[基隆河|基隆河]]河谷興建，沿途有原始的河谷景觀、[[壺穴|壺穴]]、瀑布，生態資源豐富。平溪線沿線旅遊景點甚多，包括三貂嶺瀑布群、[[十分瀑布|十分瀑布]]、[[眼鏡洞|眼鏡洞]]、四廣潭、[[台灣煤礦博物館|台灣煤礦博物館]]、平溪老街、孝子山、平溪三尖、菁桐老街、[[石底煤礦|石底煤礦]]遺址等；近年來平溪每年固定舉辦「[[天燈|天燈]]節」，平溪線也發揮了大量輸送遊客的功能。平溪線沿線的基隆河溪谷有「平溪[[耶馬溪|耶馬溪]]」的美名，也被喻為「台灣的[[保津峽|保津峽]]」<ref>[http://www.naruhodo.com.tw/katakura/index.php?page=31&tcnt=32 な～るほど・ザ・台湾] {{webarchive|url=https://web.archive.org/web/20131012021757/http://www.naruhodo.com.tw/katakura/index.php?page=31&tcnt=32 |date=2013-10-12 }}</ref>。

*三貂嶺～大華：三瓜子橋
*大華～十分：眼鏡洞旁的基隆河鋼梁橋
*大華～十分：十分老街上的平溪線鐵道

===路線延伸===
早在[[台灣日治時期|日治時代]]，即曾有將平溪線延伸到景尾（[[景美|景美]]），以與[[新店線_(台鐵)|新店線]]接軌，形成「'''臺北裏線'''」的構想。[[昭和|昭和]]5年（1930年）臺灣總督府產業調查會決議事項，在新建的地方鐵道中，即列出「景尾菁桐坑間 25.7㎞」的計畫線。

戰後，[[臺灣鐵路管理局|臺灣鐵路管理局]]曾考慮以[[美援|美援]]興建「菁桐坑-景美支線」。探查報告中，設置景美、木柵、深坑、雙溪口、文山、武坑、菁桐坑等車站<ref>[https://aa.archives.gov.tw/archivesData.aspx?SystemID=0000270904 鐵路支線平面圖]，臺灣鐵路管理局，1945年11月30日，國家檔案資訊網</ref><ref>[https://aa.archives.gov.tw/archivesData.aspx?SystemID=0000270372 坡度地形圖]，臺灣鐵路管理局，1977年12月31日，國家檔案資訊網</ref>。

在1996年復提出平溪線延伸[[木柵|木柵]]案。[[行政院|行政院]][[行政院經濟建設委員會|經建會]](現國發會)也曾於2004年做過「臺北都會區休閒遊憩鐵路暨沿線土地規劃利用之研究」，規劃從捷運木柵站起，沿途設深坑、楓子林、石碇子埔、姑娘廟等站，最後在菁桐接軌平溪線<ref>[http://www.ndc.gov.tw/News_Content.aspx?n=DE76C5BA163408BF&sms=3067CBEFA95E83AB&s=380A8D060CE17939 台北都會區休閒遊憩鐵路暨沿線土地規劃利用之研究]，國家發展委員會，2004</ref>，但最後都因為種種因素而未成功<ref>《日人曾想延平溪線到景美 墈敗垂成》，中國時報，2007年3月21日</ref><ref>黃智偉，《臺陽公司志》，臺北：臺北縣文化局，2004，P 156~157</ref>。

本延伸案在2007年交通部運研所、2010年[[蔡英文|蔡英文]]市長候選人<ref>[http://www.libertytimes.com.tw/2010/new/sep/25/today-fo4.htm 新北再生 五都心五軸帶] {{webarchive|url=https://web.archive.org/web/20101028192804/http://www.libertytimes.com.tw/2010/new/sep/25/today-fo4.htm |date=2010-10-28 }}</ref>及2011年[[蔡英文|蔡英文]]總統候選人<ref>{{Cite web |url=http://cdnews.com.tw/cdnews_site/docDetail.jsp?coluid=107&docid=101709450 |title=推觀光 連接捷運與平溪線 |access-date=2017-03-23 |archive-url=https://web.archive.org/web/20170323234153/http://cdnews.com.tw/cdnews_site/docDetail.jsp?coluid=107&docid=101709450 |archive-date=2017-03-23 |dead-url=yes }}</ref>等都提議延伸至[[深坑區|深坑]]，不過因為興建成本高、[[台灣鐵路管理局|台鐵]]態度消極，以及地形過於險峻，地方政府只允諾會評估。2017年，[[中華民國總統|總統]][[蔡英文|蔡英文]]任內提出之[[前瞻基礎建設計畫|前瞻基礎建設計畫]]並未納入本線延伸計畫，讓本線延伸案僅止於紙上談兵。

未來若是侯硐雙溪區間線型改善計畫長隧道通過，平溪線與宜蘭線分岔站將由三貂嶺站調整為侯硐車站，原宜蘭線侯硐三貂嶺區間將成為平溪線延伸段。
