'''平安南道'''（{{kor|諺=평안남도|漢=平安南道|羅=Phyŏngannam do|f=y}}），是[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]西北部的地方行政区。面积12,330 平方公里，人口3,597,557人。道政府所在[[平城|平城]]市。

== 地理 ==
西面[[黄海|黄海]]（朝鲜名称，西海），南邻首都[[平壤|平壤]]和[[黄海北道|黄海北道]]，东接[[咸镜南道|咸镜南道]]，北接[[平安北道|平安北道]]和[[慈江道|慈江道]]。

== 历史 ==
* 1896年把[[平安道|平安道]]南北两分而成立。
* 1946年9月平壤成为特别市，脱离平安南道。
* 1980年代[[南浦|南浦]]市脱离平安南道，成为直辖市
* 2004年1月9日南浦做为特级市、重新隶属平安南道。
* 2010年，江西、大安、温泉、龙岗、千里马五郡劃入南浦特別市。

== 觀光熱點 ==
* [[石湯溫泉|石湯溫泉]]：位於[[陽德郡|陽德郡]][[溫井里|溫井-{里}-]]境內。水溫73-78攝氏度，[[硫化氫|硫化氫]]為1公升中含有2.0至3.5毫克，[[礦物質|礦物質]]為1公升中含有0.4至0.5克。據說對各種[[皮膚病|皮膚病]]、神經痛、關節發炎、慢性濕疹、婦科病等有療效。
* [[龍崗溫泉|龍崗溫泉]]（Ryonggang Hot Spring）：位於「溫泉郡」。
* [[法興寺_(朝鮮)|法興寺]]（Pophung Temple）：位於「江龍山」的山腳的[[高句麗|高句麗]]時代之[[佛教|佛教]]寺廟。
* [[藥水里壁畫古墓|藥水-{里}-壁畫古墓]]：[[高句麗|高句麗]]時代的壁畫[[墳墓|墳墓]]

== 天然風物 ==
* 北倉朝鮮刺槐群落：位于「北倉郡」的「南陽里」境內。刺槐群落發現于1919年間，樹高30-50厘米，地下莖中生出新枝。刺槐開花，很適宜栽種于園林。 
 
* 龍浦[[細葉松樹|細葉松樹]]林：位于「北倉郡」的「芢浦勞動者區」境內，此樹最高30米，周長2.4米，樹冠直徑4米，此樹根部較發達，不被風吹倒。
 
* 安國寺[[銀杏|銀杏]]樹：位于平城市的鳳鶴洞境內。此樹種植于14世紀。樹高27米，周長10米，樹冠直徑18米左右，此樹枝杈有10多個。

* [[白鷺|白鷺]]、[[蒼鷺|蒼鷺]]繁殖地：[[价川|价川]]市的「龍雲里」境內，妙香山脈南西走向，「龍雲里」東坡上有[[白鷺|白鷺]]、[[蒼鷺|蒼鷺]]棲息處。從每年3月到10月初[[白鷺|白鷺]]、[[蒼鷺|蒼鷺]]到此棲息繁殖。

== 特產 ==
* [[成川栗|成川栗]]：又稱「葯栗」，產於[[成川郡|成川郡]] ，每顆約7-8克。
* [[東陽泉水|東陽泉水]]：位於[[陽德郡|陽德郡]]的[[雙龍山|雙龍山]]的泉水。
* [[新德山冷泉|新德山冷泉]]：位於[[龍岡郡|龍岡郡]]境內，從深1,000米的[[花崗石|花崗石]]層涌出的[[冷泉|冷泉]]，含有[[鉀|鉀]]、[[鈣|鈣]]等對人體有好處的物質。
