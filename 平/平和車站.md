'''平和車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[壽豐鄉|壽豐鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[側式月台|側式月台]]一座。
*車站內無人駐守與售票，提供多卡通刷卡機。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，由[[壽豐車站|壽豐站]]管理。
*停靠狀態：
**僅停靠[[臺鐵區間車|區間車]]。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1934年4月20日：應當時[[木瓜山林場|木瓜山林場]]的請求而設立，因地處[[鯉魚潭_(花蓮縣)|鯉魚潭]]的南方，故稱為'''池南'''。
*1962年4月1日：因本站行政區屬[[壽豐鄉|壽豐鄉]][[平和村_(壽豐鄉)|平和村]]內，故改稱今名。
*2012年2月23日：配合花東新車站運動，將原本已不使用的站房重新整修完成改善月台設施。
*2016年6月28日：啟用多卡通刷卡機。
