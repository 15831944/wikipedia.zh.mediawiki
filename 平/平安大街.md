'''平安大街'''是北京城东西方向的第二条交通大动脉，是[[平安里西大街|平安里西大街]]、[[地安门西大街|地安门西大街]]、[[地安门东大街|地安门东大街]]、[[张自忠路|张自忠路]]、[[东四十条|东四十条]]的总称。于1999年8月28日正式竣工通车<ref>{{cite news|url=http://news.sina.com.cn/china/1999-8-28/9842.html|title=北京平安大街等四条干道全部竣工|publisher=[[中新社|中新社]]|date=1999-08-28|accessdate=2019-02-07}}</ref>。平安大街东起[[东四十条桥|东四十条桥]]，西至[[官园桥|官园桥]]（全部位于二环内），全长7公里。沿线建筑包含[[北海公园|北海公园]]等文化旅游景点<ref>{{cite news|url=http://news.sina.com.cn/china/9907/070536.html|title=平安大街驶来旅游专线车|publisher=[[北京晚报|北京晚报]]|date=1999-07-05|accessdate=2019-02-07}}</ref>，从[[官园桥|官园桥]]和[[东四十条|东四十条]]延长的道路为平安大街延长线，东段连接[[机场第二通道高速公路|机场第二通道高速公路]]至[[北京首都机场|北京首都机场]]三号航站楼，西段到达五路居。平安大街是继长安街后第二条北京东西向交通大动脉。

== 布局规划 ==
平安大街具有很多富有老北京特色的景点，并且毗邻北海公园，平安大街两侧主要有[[北京孙中山行馆|孙中山行馆]]、[[和敬公主府|和敬公主府]]、[[清陆军部和海军部旧址|清陆军部和海军部旧址]]等，同时连接[[北京钟鼓楼|北京钟鼓楼]]、[[景山公园|景山公园]]、[[什刹海|什刹海]]等名胜古迹，沿线建筑皆为仿古式建筑与四合院。

== 交通 ==
平安大街在早晚高峰时段会实行较为严格的交通管制，平时也较经常塞车。[[北京地铁6号线|北京地铁6号线]]和[[北京地铁3号线|北京地铁3号线]]途经平安大街西段和东段。很多电车（如107路与111路）都途经平安大街。

== 延长线 ==
西沿线从[[官园桥|官园桥]]起经过[[北京三环路|北京三环路]][[花园桥|花园桥]]，最终到达西四环；东沿线从[[东四十条桥|东四十条桥]]途经[[工人体育场|工人体育场]]与[[朝阳公园_(北京)|朝阳公园]]，经过[[北京三环路|北京三环路]][[长虹桥|长虹桥]]与[[北京四环路|北京四环路]][[朝阳公园桥|朝阳公园桥]]至东五环[[平房桥|平房桥]]连接[[机场第二通道高速公路|机场第二通道高速公路]]途经[[楼梓庄桥|楼梓庄桥]]、[[金盏桥|金盏桥]]、[[管头桥|管头桥]]到达首都机场3号航站楼。
