{{nihongo|《'''平成狸合戰'''》|平成狸合戦ぽんぽこ}}為1994年由[[吉卜力工作室|吉卜力工作室]]推出、[[高畑勳|高畑勳]]執導的一部[[动画|动画]]电影。

== 剧情简介 ==
故事為一群住在東京[[多摩地域|多摩]]丘陵森林的[[貉|狸子]]們，在原本安逸的生活環境突然遭到人類的住宿建設計劃給破壞掉。

備感生存危機的狸子們為了自保；決定重新練起古時狸子們所留下的變身術，並派遣兩名使者前去佐渡及四國尋找狸子長老們協助、以能夠和人類一決勝負… 。

== 製作 ==
=== 經緯 ===
1989年高畑勳與[[鈴木敏夫|鈴木敏夫]]在私下閒聊時，曾表示著「狸子雖然是日本特有的生物，但放眼望去都找不到有關狸子的動畫，希望能看到一個像故事《[[阿波狸合戰|阿波狸合戰]]》那樣的動畫」。<ref name="GhibliWorkPage19">{{lang|ja|スタジオジブリ作品関連資料集 Vol.5}}</ref>

之後鈴木敏夫將和高畑勳對談的內容轉述給宮崎駿時，宮崎駿想到自己是漫畫家[[杉浦茂|杉浦茂]]的作品《八百八狸》愛好者，也嘗試畫了幾張構想的內容，但隨後因宮崎駿當時忙於製作《[[魔女宅急便|魔女宅急便]]》而無下文。<ref name="GhibliWorkPage19"></ref>

1992年宮崎駿完成《[[紅豬|紅豬]]》後，想到先前的狸子題材事情，便向鈴木敏夫提出「{{nihongo|豬|ブ'''タ'''}}的下一部作品，就是{{nihongo|狸子|'''タ'''ヌキ}}了」。而高畑勳之後便接受提案擔任此片導演，但高畑勳一開始所構思的作品方向並非宮崎駿期待的杉浦茂漫畫作品《八百八狸》的類型，而是想以《阿波狸合戰》的風格為主；並隨後從《[[平家物語|平家物語]]》的內容中獲取靈感，再次修正了《平成狸合戰》的劇情路線。<ref name="GhibliWorkPage19"></ref>

《平成狸合戰》片中雖有許多狸子施展變身術的鏡頭，但比起劇中奇幻的元素，高畑勳更希望本作能呈現出像是[[紀錄片|紀錄片]]模樣的風格。<ref>{{lang|ja|映画を作りながら考えたことII}}：68頁</ref>

=== 片名刊頭 ===
動畫片名「平成狸合戰」為導演高畑勳故意取的一個讓人聽起來覺得水準不高的名稱，而宮崎駿也曾一直反對此片名，<ref>樂在工作：107頁</ref>但鈴木敏夫則認為「平成狸合戰」的片名能夠讓人注意到這是一部純粹以狸子眼光所拍攝的作品、而非一些意在檢舉責備人類破壞自然的電影<ref>樂在工作：152頁</ref>

平成貍合戰也有在海報上與影片開頭裡刻意在片名旁邊標示著「{{lang|ja|総天然色漫画映画}}」。「{{lang|ja|総天然色映画}}」為日本在昭和30年代時期開始上映的彩色電影中、為了區別於先前黑白電影的不同，而特別使用的廣告宣傳詞。

== 反應 ==
=== 票房 ===
*《平成狸合戰》是1994年日本國產電影的票房年度冠軍，票房收入為26億日元。<ref>{{cite web|url=http://www.eiren.org/toukei/1994.html |title=1994年度日本電影票房收入 |publisher=一般社團法人日本電影製作者連盟 |date= |accessdate=2011-12-08}}</ref>

=== 獎項記錄 ===
*1994年第49回[[每日電影獎|每日電影獎]]：動畫電影獎
*1995年[[安錫國際動畫影展|安錫國際動畫影展]]：長篇動畫獎

== 影片小細節 ==
*動畫中一些狸子的模樣是採用漫畫家[[杉浦茂|杉浦茂]]的作畫風格。（如一隻用樹葉變成金錢的狸子被火球奶奶敲頭的那一幕）<ref>{{lang|ja|スタジオジブリの軌跡 1984-2011}}：92頁</ref>
*正吉與其牠狸子於變身術的街頭實習段落裡、在化成人類走在街上時，一家商店的門口貼著吉卜力電視動畫「[[海潮之聲|海潮之聲]]」女主角武藤里伽子的大頭海報。<ref>{{Cite web |url= http://umikiko.cocolog-nifty.com/ |title= 漫談作品《海潮之聲》|author= 中島秋則 |date=2008-01-22 |accessdate=2011-12-08 }}</ref>
*描述人類把林地挖光、並出現一名側臥佛陀的橋段中，佛陀臉蛋的模特兒是[[日本電視台|日本電視台]]的電影製片人[[奧田誠治_(電影製片人)|奧田誠治]]；為宮崎駿及鈴木敏夫的友人。<ref>{{cite web|archiveurl=https://web.archive.org/web/20040621234339/http://www.yomiuri.co.jp/entertainment/ghibli/cnt_interview_okuda02.htm |url=http://www.yomiuri.co.jp/entertainment/ghibli/cnt_interview_okuda02.htm |title=讀賣新聞上的奧田誠治採訪 |publisher=[[讀賣新聞|讀賣新聞]] |date=2004年6月15日 |archivedate=2004年6月21日 |accessdate=2014年8月20日 |language=Ja |deadurl=yes }}</ref>
*「雙子星作戰」的劇情靈感來自日本作家「[[宮澤賢治|宮澤賢治]]」的著作《孿生星子》（{{lang|ja|双子の星}}）。<ref>影片片尾的注釋</ref>
*影片中四國的狸子長老們與玉三郎搭乘電車下站的「[[聖蹟櫻丘車站|聖蹟櫻丘車站]]」，與吉卜力次回1995年動畫「[[心之谷|心之谷]]」為同一採用的舞台。
*「妖怪大作戰」裡狸子們變化成龍在天空飛行的段落中，空中出現了「[[魔女宅急便|魔女宅急便]]」裡乘著掃帚的魔女琪琪、「[[紅豬|紅豬]]」裡架駛著飛機的波魯克、「[[龍貓|龍貓]]」裡拿著雨傘飛行的大龍貓、以及「[[兒時的點點滴滴|兒時的點點滴滴]]」裡的妙子（背著書包的小學時期造型）等吉卜力動畫角色。<ref name="PomPokoFaq">{{cite web|url=http://www.nausicaa.net/miyazaki/pompoko/faq.html |title=Nausicaa.Net的《平成狸合戰》問答集|publisher={{Link-en|Nausicaa.net|Nausicaa.net}}|date= |accessdate=2011-12-08|language=en}}</ref>
*妖怪大作戰中有一段遊行貌似日本導演[[黑澤明|黑澤明]]的電影《夢》、裡頭第一小段描述狐狸娶親的故事{{nihongo|「太陽雨」|日照り雨}}。<ref name="PomPokoFaq"></ref>
