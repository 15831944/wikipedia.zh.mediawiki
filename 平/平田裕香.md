'''平田裕香'''（{{lang|en|'''Hirata Yuka'''}}，{{bd|1983年|9月15日<ref name="niiba">{{Cite web|url=http://www.nigun-niiba.co.jp/hirata/ |title=プロフィール |publisher=オフィスニグンニイバ |date=2013-01-18 |accessdate=2013-01-18 |archiveurl=https://web.archive.org/web/20121122062801/http://www.nigun-niiba.co.jp/hirata/ |archivedate=2012-11-22 |deadurl=yes }}</ref><ref name="across">{{Cite web |url=http://across-ent.com/talent/women/yuka_hirata.html |title=プロフィール |publisher=アクロス エンタテインメント |accessdate=2016-11-01}}</ref>|catIdx=Hirata Yuka}}），[[日本|日本]][[女演員|女演員]]、[[聲優|聲優]]、[[寫真偶像|寫真偶像]]。出身於[[北海道|北海道]][[夕張郡|夕張郡]][[栗山町|栗山町]]，原隸屬 [https://archive.is/20130111204244/http://www.nigun-niiba.co.jp/hirata/index.html  オフィスニグンニイバ]（Office Nigun Niiba<ref name="niiba"/>），現隸屬 Across Entertainment。

== 個人經歷 ==
   * 來自[[北海道|北海道]]的平田裕香，1983年9月15日在[[札幌市|札幌市]]醫院出生<ref name="bomb"/>。
   * 興趣是做手藝、看書、拍攝寫真、遊賣茶店和吃忌廉[[泡芙|泡芙]]，擅長[[打籃球|打籃球]]<ref name="niiba"/>（小學至中學都有參加籃球部）<ref name="bomb"/>、[[滑雪|滑雪]]和吹[[牧童笛|牧童笛]]<ref>{{Cite web |url=http://ascii.jp/elem/000/000/331/331783/ |title=【番外レポート】平田裕香が2ndビデオ発売記念握手会！裕香スマイルで「私と一緒に冒険して！」|publisher=[[ASCII_Media_Works|アスキー・メディアワークス]] |date=2002-06-23 |accessdate=2016-11-01}}</ref>。
   * 小時因就讀小學四年級而搬到[[夕張郡|夕張郡]][[栗山町|栗山町]]居住，此後便以當地為正式出身地<ref name="bomb"/>。
   * 於東京都立代代木高等學校（2004年3月閉校）畢業。
   * 由於小時曾看過資深日本藝人[[大竹忍|大竹忍]]兩小時電視劇演出，受其精湛而老練的演技深深打動，自此立志要成為真正演員<ref name="bomb">{{Cite journal |title=平田裕香の記事（《{{link-ja|BOMB|BOMB}}》2001年8月号） |year=2001 |publisher={{link-ja|学研パブリッシング|学研パブリッシング}} |journal=BOMB |issue=2001年8月号 |page=51-61}}</ref>。
   * 1998年，參加日本著名藝能公司 {{link-ja|Jumper Artist Office|Jumper Artist Office}} 所舉辦的第32屆「新人Presentation」比賽，並從此投身藝能界。
   * 1998年，在[[日本|日本]][[集英社|集英社]][[週刊YOUNG_JUMP|週刊YOUNG JUMP]]所主辦的「第10屆 全國女子高中生制服收藏」選美活動中獲提名。
   * 自從2007年在[[特攝|特攝]]影片「[[獸拳戰隊激氣連者|獸拳戰隊激氣連者]]」內飾演臨獸殿幹部且使用[[變色龍|變色龍]]拳法美女‧「'''魅莉（メレ）'''」後，憑著可愛造型及極具性格的演繹而人氣急增。外界對
     平田裕香演技亦有不俗評價，並認為她是位得體有禮的藝人。另外，她對[[聲優|聲優]]及相關行業相當有興趣。
   * 作為聲優以廣播劇「{{link-ja|NISSAN あ、安部礼司〜BEYOND THE AVERAGE|NISSAN あ、安部礼司〜BEYOND THE AVERAGE}}」裡飾演「大日本ジェネラル」開發本部 OL
    「'''姫川皐月'''」一角較為人所知，只因該角色憑「經常妄想、具冷靜頭腦及厲害分析能力」而搏得聽眾大力支持。
   * 2013年1月28日，在自己[[部落格|部落格]]中，道出「隨著時間和年紀逐漸增長，想完成的事情亦變多了。雖然曾多次向經理人公司商量辭退一事，並為此感到擔憂，但為了解決
     兩難局面，故以[[部落格|部落格]]向大家發表辭退聲明」，並離開「オフィスニグンニイバ」。
   * 2013年12月22日於「NISSAN あ、安部礼司〜BEYOND THE AVERAGE」放送400回突破記念大感謝祭「安部魂(あべコン)」活動內，與一眾廣播劇演員成功挑戰
     [[健力士世界紀錄|健力士世界紀錄]] ———— '''連續8小時不休息而現場直播的廣播劇（The Longest Radio Play）'''<ref>{{Cite web |url=https://www.tfm.co.jp/abe/abecon2013/report.php |title=あ、安部礼司│『安部魂（あべコン）』イベントレポート |publisher=[[FM東京|FM東京]] |date=2014-03-09 |accessdate=2014-03-10}}</ref>。
   * 2014年4月3日，獲邀加入「アクロス エンタテインメント」，並出任業務提攜（聲優管理工作）一職。
   * 2014年10月31日，因『「NISSAN あ、安部礼司〜BEYOND THE AVERAGE」チーム安部礼司の青森出張 in 三内丸山遺跡』活動裡，成功聚集2,500名聽眾到場參與，並對宣揚
     以申報現今[[世界遺産|世界遺産]]名錄為目標的「北海道・北東北の縄文遺跡群」有所貢獻，故獲[[青森縣知事|青森縣知事]]・三村申吾任命為無限期「'''あおもり名誉縄文人'''」<ref>{{Cite web |url=http://news.infoseek.co.jp/article/prtimes_000000491_000004829/ |title=大人気ラジオドラマ『あ、安部礼司』 安部礼司らが「あおもり名誉縄文人」に就任！ 青森県三内丸山遺跡の世界遺産登録目指しPR |publisher={{link-ja|Infoseek|Infoseek}}ニュース |date=2014-11-06 |accessdate=2014-11-07}}</ref>。

== 軼事 ==
   * 自覺個性是不服輸。
   * 一直嫌胸部稍大<ref name="bomb"/>。
   * 討厭洗好的衣服上沾着殘餘洗衣液。
   * 因「香」字劃數好，故與其姊姊一樣，名字中皆含有此字<ref name="bomb"/>。
   * 因近視關係，故常要配戴[[眼鏡|眼鏡]]或[[隱形眼鏡|隱形眼鏡]]；連親友都不察覺她有[[雙眼皮|雙眼皮]]<ref name="bomb"/><ref name="bomb0301">{{Cite journal |title=平田裕香の記事（《BOMB》2003年1月号） |year=2003 |publisher=学研パブリッシング |journal=BOMB |issue=2003年1月号 |page=51-65}}</ref>。
   * 在小學時相當能幹，曾擔任兒童會會長、應援團團長（啦啦隊長）及籃球部副部長。
   * 得到[[安達充|安達充]]提拔和照顧，故視他為「東京的亞父」。
   * [[東清彥|東清彥]]的漫畫作品《[[四葉妹妹！|四葉妹妹！]]》中，「'''綾瀨風香'''」一角是以她作角色原型。
   * 與[[日本|日本]]漫畫家[[安達充|安達充]]及[[東清彥|東清彥]]、[[小學館|小學館]]關係良好，常於演出舞台劇時，收到他們的花牌祝賀。
   * 圈中好友包括 [[寺崎裕香|寺崎裕香]]、[[折井步|折井步]]、[[井上奈奈|井上奈奈]]、[[佐藤寬子|佐藤寬子]]、[[福井未菜|福井未菜]]、金田美香、齊藤貴美子和 もたい陽子 等。
   * 與[[寺崎裕香|寺崎裕香]]關係甚佳，閑時常結伴同遊。
