'''平嶋 夏海'''（{{bd|1992年|5月28日|catIdx=Hirajima Natsumi}}），是[[AKB48|AKB48]]的前成員。出生於東京都。隸屬於演藝經紀公司{{lang|ja|[[Production尾木|Production尾木]]}}（2008年6月 - ）。暱稱是「'''{{lang|ja|なっちゃん}}'''」。

== 来歴 ==
'''2005年'''
*10月30日、參加『AKB48第一期的甄選』合格（參加人數7924名、最終合格者24名）。
* 12月8日、作為甄選成員候補生的20名之一，在[[AKB48劇場|AKB48劇場]]初次登場（当時是旧Team A所屬）。
'''2007年'''
* 2月、與[[浦野一美|浦野一美]]、[[渡邊志穗|渡邊志穗]]移至旧Team B作為支援成員。
'''2009年'''
* 1月開始、與同事務所所屬的[[多田愛佳|多田愛佳]]、[[仲川遥香|仲川遥香]]、[[渡邊麻友|渡邊麻友]]組成「走廊奔跑隊」開始活動（之後改為[[走廊奔跑隊7|走廊奔跑隊7]]）。
* 6月至7月舉行的『[[AKB48第13張單曲選拔總選舉「向神發誓，動真格」|AKB48第13張單曲選拔總選舉「向神發誓，動真格」]]』中獲得第26位，成為Under Girls。
'''2010年'''
* 5月至6月舉行的『[[AKB48第17張單曲選拔總選舉「向母親發誓，這是真的」|AKB48第17張單曲選拔總選舉「向母親發誓，這是真的」]]』中獲得第26位，成為Under Girls。
'''2011年'''
* 2月25日、『[[TeamB_5th_Stage「劇場的女神」|TeamB 5th Stage「劇場的女神」]]』公演中是劇場公演共計出演回數777回<ref name="777 times">[http://blog.oricon.co.jp/watarirouka/archive/6915/0 「777回目゛∩(^ ∀^*)なっちゃん」] {{webarchive|url=https://web.archive.org/web/20110228032937/http://blog.oricon.co.jp/watarirouka/archive/6915/0 |date=2011-02-28 }} - 渡り廊下走り隊公式ブログ（2011年2月26日）</ref>。
* 5月至6月舉行的『[[AKB48第22張單曲選拔總選舉|AKB48第22張單曲選拔總選舉「今年也要全力以赴」]]』第3次連續獲得第26位，成為Under Girls<ref>[http://ameblo.jp/akihabara48/entry-10918551053.html#main 「AKB48 22ndシングル選抜総選挙結果」] - AKB48オフィシャルブログ（2011年6月10日）</ref>。
'''2012年'''
* 1月28日、因與男友的私人照片流出<ref>[http://www.sponichi.co.jp/entertainment/news/2012/01/28/kiji/K20120128002522520.html 男性とパジャマ姿で…AKB2人? の写真流出 発端はツイッター] - スポーツニッポン 2012年1月28日（2012年1月28日閲覧）</ref><ref>[http://hochi.yomiuri.co.jp/entertainment/news/20120128-OHT1T00033.htm 平嶋夏海、米沢瑠美がAKB脱退! 男性との写真流出] {{webarchive|url=https://web.archive.org/web/20120422165946/http://hochi.yomiuri.co.jp/entertainment/news/20120128-OHT1T00033.htm |date=2012-04-22 }} - スポーツ報知 2012年1月28日（2012年1月28日閲覧）</ref>，宣佈退出AKB48的所有活動<ref name="jitai1">[http://ameblo.jp/akihabara48/entry-11148085263.html 【お知らせ】] - AKB48オフィシャルブログ（2012年1月28日）</ref><ref name="jitai2">[http://blog.oricon.co.jp/watarirouka/archive/9199/0 【お知らせ】] {{webarchive|url=https://web.archive.org/web/20120328192933/http://blog.oricon.co.jp/watarirouka/archive/9199/0 |date=2012-03-28 }} - 渡り廊下走り隊のオフィシャルブログ（2012年1月28日閲覧）</ref>。
* 2月5日、「[[崇尚麻里子|崇尚麻里子]]」劇場盤大握手会宣佈所有有關AKB48的活動完結<ref>[http://ameblo.jp/akihabara48/entry-11148384107.html 【辞退に関して】] - AKB48オフィシャルブログ（2012年1月28日）</ref><ref>[http://www.sanspo.com/geino/news/120206/gnj1202060513015-n1.htm 平嶋夏海&米沢瑠美、AKB最後の握手会] {{Wayback|url=http://www.sanspo.com/geino/news/120206/gnj1202060513015-n1.htm |date=20120208071213 }} - sanspo.com 2012年2月6日（2012年2月6日閲覧）</ref>。

* 10月24日、從這日開始，籍舞台劇「{{lang|ja|GO, JET!GO!GO!〜I LOVE YOUが言えなくて〜}}」正式復出藝能界。
'''2013年'''
*在2月1日公開<ref>台灣於4月3日公開</ref>的《AKB48心程紀實3：少女眼淚的背後》<small>{{lang|ja|（[[AKB48_笑淚交織|DOCUMENTARY of AKB48 No flower without rain 少女たちは涙の後に何を見る?]])}}</small>中，參與了這部她自AKB48退出以來的電影面談映像。
*4月7日，報名參選[[AKB48第32張單曲選拔總選舉|第五屆AKB48選拔總選舉]]。
*6月8日，[[AKB48第32張單曲選拔總選舉|AKB48第32張單曲選拔總選舉]]獲得第62名，是唯一一位參加候選而入圍的畢業成員。

== 人物 ==
有[[花粉症|花粉症]]<ref>[http://blog.oricon.co.jp/watarirouka/archive/6924/0 「花粉症さんとんでけー゛∩(^ ∀^*)なっちゃん」] {{webarchive|url=https://web.archive.org/web/20110302083331/http://blog.oricon.co.jp/watarirouka/archive/6924/0 |date=2011-03-02 }} - 渡り廊下走り隊オフィシャルブログ（2011年2月26日）</ref>。

=== AKB48相關 ===
在第4張單曲「BINGO!」初次被選為選拔成員。第8張單曲「櫻花的花瓣們2008」再度被選為選拔成員。

[[AKB48劇場|AKB48劇場]]公演的出演回數是包括畢業生是AKB48全成員中最多，至2011年2月25日出演有777回<ref name="777 times"/>，同年8月20日達成出演800回<ref>[http://blog.oricon.co.jp/watarirouka/archive/8402/0 「800゛∩(^ ∀^*)なっちゃん」] {{webarchive|url=https://web.archive.org/web/20110823080300/http://blog.oricon.co.jp/watarirouka/archive/8402/0 |date=2011-08-23 }} - 渡り廊下走り隊オフィシャルブログ（2011年8月21日）</ref>，總共出演833回。至2012年3月28日現在，AKB48劇場公演凡演回數已超過800回，平嶋和浦野（在2011年5月4日出演800回達成，總共828回出演）2人是出演最多<ref>但是，浦野在AKB48（690回）→SDN48（138回）的總共回數，只計AKB48平嶋是第1位。</ref>。

研究生（2011年2月時侯）的首推成員是[[入山杏奈|入山杏奈]]<ref>[http://blog.oricon.co.jp/watarirouka/archive/6853/0 「あんさー゛∩(^ ∀^*)なっちゃん」] {{webarchive|url=https://web.archive.org/web/20110223005843/http://blog.oricon.co.jp/watarirouka/archive/6853/0 |date=2011-02-23 }} - 渡り廊下走り隊オフィシャルブログ（2011年2月19日）</ref>。

在AKB48活動完結之後，取得電單車駕駛執照，喜歡到烹飪班去。而且，在2012年4月開始到大學上學。
