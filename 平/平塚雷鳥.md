'''平塚雷鸟'''（{{lang-ja|平塚らいてう}}，{{bd|1886年|2月10日|1971年|5月24日}}），本名'''平塚明'''，是[[日本|日本]][[思想家|思想家]]、[[評論家|評論家]]、[[作家|作家]]、[[女性主義|女权主义者]]，毕业于[[日本女子大学|日本女子大学]]家政系<ref name="Hiratsuka">{{cite web |title=女性・平和運動のパイオニア　平塚らいてう(1886-1971) ～元始、女性は実に太陽であった～(青鞜) |url=https://www.jwu.ac.jp/unv/about/jidai/index/pioneer002.html |website=日本女子大学 |accessdate=2018-11-17 |archive-url=https://web.archive.org/web/20170729104118/https://www.jwu.ac.jp/unv/about/jidai/index/pioneer002.html# |archive-date=2017-07-29 |dead-url=no|language=日语 }}</ref>，在战后主要参与反战运动。平塚在1911年9月庆祝杂志《青鞜》创刊撰写的文章《女性原本是太阳》是女权运动的重要象征之一。其就读的[[日本女子大学|日本女子大学]]于2005年设立了平塚雷鸟奖<ref name="Yoshidai">{{cite web |title=平塚らいてう賞とは |url=https://www.jwu.ac.jp/st/grp/raiteu/about.html |website=日本女子大学 |accessdate=2018-11-17 |archive-url=https://web.archive.org/web/20180408141343/https://www.jwu.ac.jp/st/rp/raiteu/about.html# |archive-date=2018-04-08 |dead-url=no|language=日语 }}</ref>。

平塚雷鸟的日文姓名一般写作{{lang|ja|平塚らいてう}}，但也有都是[[日本汉字|汉字]]的情况（{{lang|ja|平塚雷鳥}}），也有被写成{{lang|ja|平塚明子}}的情况。<ref>{{cite web |author1=長谷川時雨 |title=平塚明子（らいてう） |url=https://www.aozora.gr.jp/cards/000726/files/45985_26598.html |website=青森文库 |accessdate=2018-11-18 |archive-url=https://web.archive.org/web/20170307171252/http://www.aozora.gr.jp/cards/000726/files/45985_26598.html# |archive-date=2017-03-07 |dead-url=no|language=日语 }}</ref>

==生平==
雷鸟于1886年出生于[[东京|东京]]，是高级官员平塚定二郎的次女，母亲的娘家在[[明治维新|明治维新]]前是[[和歌山县|和歌山]]藩主家的医生。1892年雷鸟进入富士见小学，尔后转学至诚之小学，毕业后进入东京女子高等师范学校附属高等女校。1903年雷鸟进入日本女子大学学习，毕业后又进入[[二松學舍大学|二松學舍大学]]学习<ref>{{cite web |title=卒業生紹介 |url=https://www.nishogakusha-u.ac.jp/about/f2.html |website=二松學舍大学 |accessdate=2018-11-18 |archive-url=https://web.archive.org/web/20180630063205/http://www.nishogakusha-u.ac.jp/about/f2.html# |archive-date=2018-06-30 |dead-url=no|language=日语 }}</ref>。1907年6月，她开始创作短歌和短篇小说。<ref name="Xianqu">{{cite journal |authors=李先瑞 |title=平塚雷鸟:日本女性解放运动的先驱 |journal=世界文化 |date=2015 |issue=10 |page=20页}}</ref>

1908年发生了“{{link-ja|盐原事件|煤煙_(小説)#心中未遂事件}}”。3月21日晚上，平塚雷鸟和{{link-ja|森田草平}}两人搭乘前往[[大宮區|大宮區]]、{{link-ja|盐原|塩原温泉郷}}方向的电车出游。在途中森田草平企图杀死雷鸟后殉情，没有成功。这个殉情未遂事件造成社会哗然，许多人开始批评身为处女的雷鸟和男子共处一室的行为。自此，雷鸟开始关注性别歧视和男尊女卑社会中女性的自我解放问题。<ref name="Xianqu"/>

1911年9月，在{{link-ja|生田長江}}的推荐下，她创办了日本首个女性文艺杂志《青鞜》，撰写了发刊辞《女性原本是太阳》。1919年11月24日，她和{{link-ja|市川房枝}}、{{link-ja|奥武芽绪|奥むめお}}等人组织創建“{{link-ja|新妇女协会|新婦人協会}}”，把《女性同盟》定为机关杂志，提出妇女参政、男女机会平等、男女同校、保障妇女儿童合法权益、争取世界和平等口号，积极开展妇女解放运动。“新妇女协会”的最大目标是要求修改《{{link-ja|治安警察法}}》第五条，成员们为此四处奔波宣讲。1921年，雷鸟由于过度劳累，加之跟市川房枝产生分歧，不得不从协会退出。此外，伊藤野枝和山川菊荣等社会主义者也对雷鸟进行批判和攻击。1922年，《治安警察法》第五条修正案获得通过，之後平塚雷鸟开始了文学创作。[[第二次世界大战|第二次世界大战]]后，平塚雷鸟在推进妇女运动的同时积极推进和平运动，坚持反战。<ref name="Xianqu"/>
