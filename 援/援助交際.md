'''援助交際'''（{{jpn|j=援助交際|hg=えんじょこうさい|rm=enjokōsai|f=t}}）簡稱「'''援交'''」，是源自[[日本|日本]]的一個[[委婉语|委婉语]]，指[[未成年人|未成年人]]為獲得[[金錢|金錢]]而答應與[[成年人|成年人]][[約會|約會]]。從字面上望文生義似乎未伴有[[人类性行为|性接觸]]，現今意義卻成為未成年人自行尋找客人進行[[性交易|性交易]]的代名詞。

依據[[中華民國|中華民國]][[內政部警政署刑事警察局|內政部警政署刑事警察局]]的定義，援交是一種特殊的「雙向互動」[[情色|情色]][[性交易|性交易]]：「未成年人（特別是尚未走向[[社會|社會]]的「[[中學|中學]]生」）接受成年人的「援助」，包括金錢、[[服裝|服裝]]、[[飾品|飾品]]和[[食物|食物]]等物質享受；成年人接受未成年人的「援助」——[[性_(生物學)|性]]的奉獻。」

現時援交並不限於少女向成年人賣春，也有少男向成年人賣春，顧客通常是家庭關係不佳的中年婦女或[[男同性戀|男同性戀]]。

在[[日本|日本]]，有些[[初中|初中]][[女|女]][[學生|學生]]通過[[網路|網路]]、[[電話|電話]]等[[聯絡|聯絡]]方式，尋找[[陌生人|陌生人]]進行援交。參與援交的[[未成年人|未成年人]]認為，在[[交友|交友]]過程中具備[[自主|自主]][[选择權|选择權]]、並與對方有所[[溝通|溝通]]、[[交流|交流]]和[[了解|了解]]，是這種[[活動|活動]]得以開展和持續的[[前提|前提]]；相較之下，他們除了對於約會與[[婚前性行為|婚前性行為]]的人際[[概念|概念]]似乎並不深厚，也不太明瞭這樣是否涉及被[[法律|法律]]所禁止的[[性交易|性交易]]。

至於未成年人找成年人進行[[性交易|性交易]]而給予成年人金錢就是'''逆援助交際'''（簡稱「'''逆援'''」），由於未成年人經濟地位提高以及性觀念的開放<ref>[[蘋果日報|蘋果日報]]：《[http://www1.hk.apple.nextmedia.com/template/apple_sub/art_main.php?iss_id=20070422&sec_id=12187396&art_id=7025768 網上淫：逆援助交際玩番男人轉頭]》</ref>，這現象便有所增長，當中主要是多金少女找尋成年男子以排解慾望，另也有男學生援助少婦的情況。

== 歷史 ==
援助交際行為最早於1985年在[[東京|東京]]，伴隨男來店女來電的「電話交友」營業場所風行而出現。後來有學生借助深夜播出的[[電視|電視]]交友節目而達到援交的目的。援交未成年人的問題，最早由《[[朝日新聞|朝日新聞]]》晚報於1994年9月20日報導。​关于“援助交际”在女高中生中的参与比例有两份数据，一是1996年10月东京都政府相关部门针对东京5500名女中学生所做的问卷调查（得到的有效回答问卷是1291份），其中高中生回答“有”的比例占4.4%，初中生是3.8%。二是财团法人“亚洲女性和平国民基金会”于1997年10月针对首都圈960名女高中生（15-18岁）所做的问卷调查（有效回答600份），其中约5%的人回答“有”。不过这2份调查都是在援助交际最盛行的东京地区抽样的，在日本其他地区应该没这么高比例。
<ref>[http://d.hatena.ne.jp/keyword/%B1%E7%BD%F5%B8%F2%BA%DD 援助交際とは - はてなダイアリー]</ref>

在2015年10月期間，聯合國特使布基契奧（Maud de Boer-Buquicchio）發表報告指日本有13%的女中學生從事過援助交際，但日本政府指報告失實並提出抗議。事後也親口證實13%是沒有證據支持的數據,撤回報告並道歉。<ref>{{cite web |url=http://www.esame.cn/hot/20151104/672144.html |title=存档副本 |accessdate=2015年11月17日 |deadurl=yes |archiveurl=https://web.archive.org/web/20151117230452/http://www.esame.cn/hot/20151104/672144.html |archivedate=2015年11月17日 |df= }}</ref>

=== 香港 ===
{{main|老泥妹}}
自2000年代起，援交[[網站|網站]]在[[香港|香港]]紛紛出現，使[[未成年人|未成年人]]（尤其是[[女性|女性]]）得以在網上與[[嫖客|客人]]討價及相約<ref>[http://appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20080630&sec_id=4104&subsec_id=11867&art_id=11290299 警鎖定五大援交網 擬派卧底調查--蘋果日報 2008年6月30日] {{webarchive|url=https://web.archive.org/web/20080702015504/http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20080630&sec_id=4104&subsec_id=11867&art_id=11290299 |date=2008年7月2日 }}</ref>，導致「援助交際」開始成為一個嚴重的社會問題，直至[[王嘉梅命案|王嘉梅命案]]發生後，這個問題才真正被社會人士所重視。在香港，這群以[[女生|女生]]為群體的[[未成年|未成年]]「援交女」一般被人[[戲稱|戲稱]]為「[[老泥妹|老泥妹]]」，皆因她們平時不愛[[回家|回家]]，只靠與其他人進行[[性交易|性交易]]之後才有機會[[洗澡|洗澡]]，所以常常「成身老泥」（滿身都是[[汙垢|汙垢]]）。

== 詐騙 ==
由於許多援交是透過網路論壇尋找對象，在現實世界中並不一定看過彼此，因此「假援交真詐騙」的情形層出不窮。

==網路語言==
'''PTGF'''為英文直接縮寫，全寫為「Part Time Girl Friend」，中譯「兼職女情人」，實解作進行'''援助交際'''的少女，常見於討論區與社交網站等，尤其是'''援助交際'''專題網站。因為內容較為敏感，而且可能不合法，為了規避管方（父母、學校等）、執法部門（警方、保護兒童組織等）和過濾軟件等的關注（[[突破網絡審查|突破網絡審查]]），而使用的字眼。
