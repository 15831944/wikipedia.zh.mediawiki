'''臨沂市'''是[[中華人民共和國|中華人民共和國]][[山東省|山東省]]下轄的[[地級市|地級市]]，位于山東省东南部，[[黄海|黄海]]以西，因临[[沂河|沂河]]而得名，1994年12月由[[臨沂地區|臨沂地區]]改制而來，是山東省面积最大、人口最多的地級市。古代政區主體隶属[[琅琊|琅琊]]、[[沂州|沂州]]，是東夷文化和鳳凰文化的重要发祥地，是著名的歷史文化名城、中国書法名城，亦是近代著名的沂蒙革命老區、前[[華東局|中共中央華東局]]、[[华东军区|华东军区]]、[[华东野战军|华东野战军]]、[[山東省政府|山東省政府]]、[[山东军区|山东军区]]等诞生地及所在地。1972年，因出土《孙子兵法》《孙膑兵法》而闻名于世。臨沂是第三批[[全國文明城市|全國文明城市]]、全國第一個[[水生态文明城市|水生态文明城市]]以及[[中国优秀旅游城市|中国优秀旅游城市]]、[[國家园林城市|國家园林城市]]、[[國家衛生城市|國家衛生城市]]、[[國家环境保护模范城市|國家环境保护模范城市]]、[[國家森林城市|國家森林城市]]、[[中國金刚石之都|中國金刚石之都]]、[[世界滑水之城|世界滑水之城]]。辖區内山高水长，[[沂河|沂河]]穿越其中，[[蒙山_(山东)|蒙山]][[沂水|沂水]]甲天下，环境优美，宜居宜商。在福布斯[[大陸最佳商业城市|大陸最佳商业城市]]<ref>{{cite web |url=http://www.forbeschina.com/list/1988 |title=存档副本 |accessdate=2013-01-28 |deadurl=yes |archiveurl=https://archive.is/20130425234629/http://www.forbeschina.com/list/1988 |archivedate=2013-04-25 }}</ref>评选中近三年连续入围50强。2013年1月，國家[[海关总署|海关总署]]批准'''臨沂商城'''采用“旅游购物商品”贸易方式出口商品，臨沂成为继[[浙江|浙江]][[义乌|义乌]]之后，全國第二個享受出口商品就地一次报检、一次通关政策的城市。如今臨沂已成为中國商贸物流之都。

== 歷史 ==
商朝时期，这块土地上出现了郯、莒、费诸方国。

周灭商后， [[鲁国|鲁国]]和[[齐国|齐国]]是[[周王朝|周王朝]]控制东方的重要支柱，临沂地域除分属齐、鲁等国外，见于《春秋》的还有颛、阳、向、莒、郯、根牟、於馀丘、杞等国。

春秋时期，除上述古国外，见于《春秋》的有启阳，中丘、祝丘、费、防、台、东阳、武城、丘舆、向、次室、蒙、郓、堂阜、盖、 艾、纪障、密、鄢陵等20个。

战国时期，域内诸封国先后为齐、楚所兼并，至战国末期，南部属楚，北部属齐。

秦朝统一，地方实行[[郡县制|郡县制]]，全国分为三十六郡，临沂地域分属[[琅琊郡|琅琊郡]]和[[郯郡|郯郡]]。

西汉时，郡国并行。临沂地域分属徐州之琅琊国、东海郡、城阳国和兖州之泰山郡。

东汉承西汉制度，分属徐州之利城郡、东海郡、琅琊郡、琅琊国和兖州之泰山郡、东莞郡、东安郡和城阳国。

三国时期，属魏国。至魏末，分属东海国、琅琊国、东莞郡、泰山郡。

西晋时期，分属徐州之琅琊国、东海郡、兰陵郡、东莞郡、兖州之泰山郡。

晋室南渡后，地域先后属于后赵、东晋、前燕、后燕、南燕、前秦和北魏。<br />
东晋元帝在[[江乘县|江乘县]]侨置[[南东海郡|南东海郡]]、[[南琅琊郡|南琅琊郡]]、[[南东平郡|南东平郡]]、[[南兰陵郡|南兰陵郡]]。南琅琊郡寄治在江乘县的金城。金城为三国东吴修筑。《三国志·吴书·孙和传》中记载：宝鼎二年(267年)十二月，“皓于金城外露宿。”到晋成帝咸康元年(335年)，[[桓温|桓温]]任南琅琊郡内史，出镇江乘县金城，请求割丹阳郡江乘县境以立郡。不久，南琅琊郡正式在江乘县划土立定，江乘县改隶南琅琊郡。咸康七年(341年)，割江乘县西境设立临沂县。南朝齐永明六年(488年)，武帝以为江乘县白下垒（位于今南京市中央门外北崮山）依带江山，将南琅琊郡治从金城迁至白下垒，名白下城。陈太建十年(578年)废除南琅琊郡，置建兴郡。

隋朝分属沂州（琅琊郡）、泗州（下邳郡）、密州（高密郡）、海州（东海郡）、徐州（彭城郡）。

唐朝分属沂州（琅琊郡）、密州（高密郡）、徐州（彭城郡）。

宋朝分属沂州（琅琊郡）、密州（高密郡）、淮郡军等。

宋室南渡后，分属于金朝山东东路的沂州、莒州、邳州、泰安州。

元朝，属中书省山东东西道宣慰司益都路。

明朝，分属山东布政司兖州府和青州府。

清朝初因之，清雍正十二年（1734年）升沂州为府，置附郭兰山县，降莒州为散州。属沂州府。有兰山、郯城、费县、沂水、蒙阴、日照、莒州6县1州。

中华民国成立后，废府撤州，实行省、道、县三级制。1913年撤销沂州府，改莒州为莒县。1914年分山东省为4道，改兰山县为临沂县，临沂地域分属济宁道和胶东道。1918年撤销。1925年10月22日，新建琅琊道，辖区即沂州府旧治全境，治所驻临沂县。1928年，琅琊道撤销，各县直隶于省。1936年，置山东省第三区行政督察专员公署。

抗日战争时期，中国共产党这里创建抗日根据地。1940年8月，在沂南县青驼寺成立山东省战时工作推行委员会。1943年9月改为山东省战时行政委员会。<br />
1945年8月在莒南县大店镇成立山东省政府。

解放战争时期，共产党的行政区划层次仍沿袭抗日战争时期，只是对区划作了某些调整。1948年7月，鲁中、鲁南、滨海三个行政区合并为鲁中南行政区，设鲁中南行政公署（驻临沂城），下辖7个专区，49个县。至1949年10月1日，临沂地域分属沂蒙专区、尼山专区、台枣专区、滨海专区。<br />

中华人民共和国成立后，对行政区划进行调整。1950年5月，[[鲁中南行政区|鲁中南行政区]]撤销。以[[沂蒙专区|沂蒙专区]]为基础，成立[[沂水专区|沂水专区]]，辖9县。以[[滨海专区|滨海专区]]为基础，成立[[临沂专区|临沂专区]]，辖9县。1953年1月将原属临沂专区的赣榆、东海、邳县、新海连市划入江苏省。1953年7月，沂水专区撤销，除日照县划入胶州专区外，所辖其余各县皆划归临沂专区，同时，原划入[[滕县专区|滕县专区]]的[[平邑县|平邑县]]亦复归临沂专区。1956年3月，[[日照县|日照县]]划归临沂专区。至1961年4月，[[临沂专区|临沂专区]]辖'''临沂市'''、[[郯城|郯城]]、[[兰陵县|苍山]]、[[临沭|临沭]]、[[莒南|莒南]]、[[沂南|沂南]]、[[沂水|沂水]]、沂源、蒙阴、平邑、费县、日照、莒县1市12县。1985年3月22日，撤销日照县和石臼办事处改为县级日照市。1989年6月12日，[[日照市|日照市]]升格为地级市。1989年12月2日，沂源县划入淄博市。1992年1月，莒县划入日照市。至此，临沂地区辖临沂市、郯城、苍山、莒南、沂水、蒙阴、平邑、费县、沂南、临沭1市9县。1994年12月，撤销[[临沂地区|临沂地区]]和县级临沂市，设立地级临沂市。原临沂市分为兰山、河东、罗庄3个县级区。地级临沂市辖兰山、罗庄、河东3区和郯城、苍山、莒南、沂水、沂南、平邑、费县、蒙阴、[[临沭|临沭]]9县。 2014年1月21日，[[苍山县|苍山县]]复名[[兰陵县|兰陵县]]。

== 地理 ==
'''临沂市'''位于[[山东|山东]]的南部，东部连接[[日照|日照]]，地近[[黄海|黄海]]，西接[[枣庄|枣庄]]、[[济宁|济宁]]、[[泰安|泰安]]，北靠[[淄博|淄博]]、[[潍坊|潍坊]]，南临[[江苏|江苏]]的[[徐州|徐州]]、[[连云港|连云港]]，大致位于京沪连线的中心位置，境内有京沪高速公路穿过。临沂市以临近横贯临沂南北的山东省第二大河亦即山东省第一大内河——[[沂河|沂河]]，而得名。地跨东经117度24秒─119度11秒，北纬34度22秒─36度22秒，南北最大长距228公里，东西最大宽度161公里，总面积17184平方公里，是[[山东省|山东省]]面积最大、人口最多的地市。

=== 民族 ===
临沂市是一个多民族聚居区，有[[汉族|汉族]]、[[回族|回族]]、[[满族|满族]]、[[朝鲜族|朝鲜族]]、[[壮族|壮族]]、[[蒙古族|蒙古族]]、[[藏族|藏族]]、[[苗族|苗族]]等32个民族，除[[汉族|汉族]]以外的其他[[少数民族|少数民族]]人口占总人口的0.33%。

== 经济 ==
随着京沪高速、胶新铁路、荷兖日铁路等的建设与电气化和临沂机场复航吞吐量突破56万人次，和临沂批发市场的扩大，其经济发展势头强劲，现正全力打造省会和半岛外的黄金三角经济第三极。气候湿润温和，适宜于农、林、牧、副、渔综合发展，盛产农副土特产品。以[[建材|建材]]为主的非金属矿藏储量大且质量好，已被省定为建材基地，水资源充足，水质优良，发达的公路、铁路交通运输网。拥有建材、[[轻纺|轻纺]]、[[食品|食品]]、[[化工|化工]]、[[机械|机械]]、[[电子|电子]]、[[电力|电力]]、[[煤炭|煤炭]]等十几个工业门类。[[手工业|手工业]]生产历史较长，风格独特。通讯设施完备，覆盖全市。

== 著名人物 ==
===历史人物===
*[[蒙恬|蒙恬]]：秦朝著名将领，祖父[[蒙骜|蒙骜]]、父亲[[蒙武|蒙武]]均为秦国名将、其弟[[蒙毅|蒙毅]]官拜上卿。蒙恬收复河套地区，开发宁夏，修筑长城，防御匈奴，改良毛笔、改良古筝等，为一代名将。
*'''[[琅邪王氏|琅邪王氏]]'''
**[[王崇|王崇]]：西汉后期大臣，王骏之子，王吉之孙，御史大夫，封扶平侯。
**[[王睿|王睿]]：东汉末荆州刺史，[[王祥|王祥]]之伯父，为[[孙坚|孙坚]]所杀。
**[[王祥|王祥]]：三国至晋大臣，二十四孝人物之一（[[二十四孝|二十四孝]]--臥冰求鯉）。
**[[王览|王览]]：王祥之弟，二十四悌人物之一。
**[[王导|王导]]：[[东晋|东晋]]初宰相，书法家，[[王览|王览]]之孙。
**[[王戎|王戎]]：三国末期至晋初大臣，[[王导|王导]]之族兄，[[竹林七賢|竹林七賢]]之一，後變得貪財好利。
**[[王衍|王衍]]：西晋大臣，王戎族弟，以清谈误国著称。
**[[王敦|王敦]]：晋朝大臣，王导之从兄，东晋初年叛乱，失败后悒郁而终。
**[[王澄|王澄]]：晋代政治人物，[[王衍|王衍]]之弟，曾仿效[[竹林七贤|竹林七贤]]聚会肆为。精通技击，因与族兄[[王敦|王敦]]政见不合，为其王敦所害。
**[[王羲之|王羲之]]：东晋时期[[書法|書法]]家。
**[[王献之|王献之]]：[[書法|書法]]家，王羲之第七子。
**[[王融|王融]]：[[文學|文學]]家。
*'''[[琅邪顏氏|琅邪顏氏]]'''
**[[颜延之|颜延之]]（384年－456年）：南朝宋文学家。
**[[顏之推|顏之推]]（531年－595年）：《[[顏氏家訓|顏氏家訓]]》的作者。
**[[颜师古|颜师古]]（581年－645年）：唐初儒家学者，经学家、语言文字学家、历史学家,顏之推之孙。
**[[颜杲卿|颜杲卿]]（692年－756年）：颜师古的堂曾孙，安史之乱中著名忠臣。
**[[顏真卿|顏真卿]]（709年－784年）：颜师古的堂曾孙，颜杲卿从弟，唐代著名政治家、书法家。创立“[[颜体|颜体]]”楷书。
*[[刘洪|刘洪]]：创建《乾象历》。
*[[左宝贵|左宝贵]]：甲午中日战争英雄。
*[[孙镗|孙镗]]：明朝抗倭名将。
*[[王璟|王璟]]：明朝政治家。
*[[曾子|曾子]]：宗圣。
*[[匡衡|匡衡]]：[[西汉|西汉]]末[[丞相|丞相]]，[[凿壁偷光|凿壁偷光]]。
*[[荀子|荀子]]：乱世哲人。
*[[蒙恬|蒙恬]]：[[秦朝|秦朝]]名将。
*'''[[瑯邪諸葛氏|瑯邪諸葛氏]]''' 
**[[諸葛豐|諸葛豐]]：[[西漢|西漢]][[司隸校尉|司隸校尉]]。
**[[諸葛珪|諸葛珪]]：[[東漢|東漢]]諸葛亮之父。
**[[諸葛玄|諸葛玄]]：[[東漢|東漢]][[豫章|豫章]]太守，後投靠[[劉表|劉表]]，諸葛亮叔父 照顧諸葛亮和[[諸葛均|諸葛均]]。
**[[諸葛亮|諸葛亮]]：汉末三国时期[[刘备|刘备]]集团重要谋士，任军师中郎将，[[蜀漢|蜀漢]]建立后任丞相，死后谥号为“武侯”，故亦称“诸葛武侯”。
**[[諸葛瑾|諸葛瑾]]：汉末三国时期[[東吳|東吳]]謀士及大将，[[諸葛亮|諸葛亮]]长兄，在江东先后任长史、中司马，后为绥南将军宣城侯，东吴建国后被封为宛陵侯，历任左将军、大将军左都护领豫州牧。
**[[诸葛诞|诸葛诞]]：汉末三国时期[[曹魏|曹魏]]大臣，[[诸葛亮|诸葛亮]]族弟，先后任吏部郎、御史中丞尚书，后免官，起用后为御史中丞尚书并出为扬州刺史，加昭武将军衔。[[王凌|王凌]]叛乱平定后，任镇东将军，假节都督扬州诸军事，封山阳亭侯，坐镇东南。后反，败，自杀。
**[[诸葛靓|诸葛靓]]：[[诸葛诞|诸葛诞]]之子，诞反，欲以吴为应，遣质于吴，诞亡，因留于吴。吴亡后自匿拒仕于晋。因与[[晋武帝|晋武帝]][[司马炎|司马炎]]幼时交好，且其姐为琅琊王夫人，故得善终。
**[[诸葛长民|诸葛长民]]：东晋大臣，为[[宋武帝|宋武帝]][[刘裕|刘裕]]所杀。
