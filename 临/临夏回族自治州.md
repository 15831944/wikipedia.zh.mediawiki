'''臨夏回族自治州'''，简称'''临夏州'''，是[[中华人民共和国|中华人民共和国]][[甘肃省|甘肃省]]下辖的[[自治州|自治州]]，位于甘肃省中部，[[黄河|黄河]]上游，[[黄土高原|黄土高原]]与[[青藏高原|青藏高原]]交界。首府临夏市是古代[[丝绸之路|丝绸之路]]、唐蕃古道的重镇，[[明代|明代]]四大[[茶马|茶马]]司之一，有“河湟雄镇”之称。临夏州[[汉族|汉族]]人口比例约40%，[[回族|回族]]人口比例约32%。

== 历史 ==
[[秦代|秦代]]置枹罕县（fú hǎn），属[[陇西郡|陇西郡]]，治所在今临夏县[[韩集镇_(临夏县)|韩集镇]]双城。[[西汉|西汉]][[始元|始元]]六年（前81年），改属[[金城郡|金城郡]]。[[东汉|东汉]][[建武|建武]]十二年（36年），撤金城郡，枹罕县改属陇西郡。[[西晋|西晋]][[太康|太康]]二年（281年），废枹罕县，立枹罕护军。[[永嘉|永嘉]]年间，枹罕县属[[晋兴郡|晋兴郡]]。[[十六国|十六国]][[前凉|前凉]][[太元|太元]]二十一年（344年），析[[凉州|凉州]]新置[[河州_(前凉)|河州]]，州、郡均治枹罕。后境地分別为[[前秦|前秦]]、[[西秦|西秦]]、[[后秦|后秦]]、[[西夏|西夏]]、[[北魏|北魏]]、[[西魏|西魏]]、[[北周|北周]]统治。
[[File:5637-Linxia-City-Dongguan-back-street.jpg|缩略图]]
[[隋|隋]][[开皇|开皇]]三年（583年）废枹罕县入河州。[[大业|大业]]三年（607年），河州改枹罕郡。[[唐朝|唐朝]][[武德|武德]]元年（618年），复置枹罕县，为河州治所。[[天宝_(唐朝)|天宝]]元年（742年），改河州为[[安乡郡|安乡郡]]。[[宝应_(唐朝)|宝应]]元年（762年），安乡郡为[[吐蕃|吐蕃]]所占。
[[File:5665-Linxia-City-Hongyuan-Park-east-gate.jpg|缩略图]]
[[北宋|北宋]][[熙宁|熙宁]]六年（1073年），大将[[王韶|王韶]]收复河州，复置枹罕县，属[[熙河路|熙河路]]。熙宁九年（1076年）省枹罕县入安乡县。[[金朝|金朝]][[天会_(金朝)|天会]]九年（1131年），河州为金所据。[[皇统|皇统]]二年（1142年），移州治至枹罕城。[[贞元_(金朝)|贞元]]二年（1154年）复置枹罕县，为河州治。[[大定_(金朝)|大定]]二十七年（1187年），枹罕县改属临洮路。[[成吉思汗|成吉思汗]]二十一年（1227年），蒙古军攻占河州。[[元朝|元朝]][[至元_(忽必烈)|至元]]六年（1269年）省枹罕县入安乡县。[[明朝|明朝]][[洪武|洪武]]四年（1371年），置[[河州卫|河州卫]]。

[[清朝|清朝]][[乾隆|乾隆]]年间，[[苏四十三|苏四十三]]反清，是为[[第一次河煌事变|第一次河煌事变]]。[[同治|同治]]元年（1862年），中国西北地区发生[[同治陕甘回变|同治陕甘回变]]。回民、撒拉族等[[穆斯林|穆斯林]]与汉民之间相互仇杀，战乱持续十余年，终被清政府镇压。其中，河州莫尼沟[[阿訇|阿訇]][[马占鳌|马占鳌]]起兵反清，即[[第二次河湟事变|第二次河湟事变]]。1872年，马占鳌率众归降朝廷，奠定了河州[[马家军|马家军]]崛起的基础，终成为[[民国时期|民国时期]]实际控制甘肃、[[宁夏|宁夏]]、[[青海|青海]]等地的地方[[军阀|军阀]]。[[光绪|光绪]]二十一年（1895年），[[第三次河湟事变|第三次河湟事变]]。

[[民国|民国]]二年（1913年），改设导河县，属[[兰山道|兰山道]]。1927年，废道改区，属兰山区。1928年，[[马仲英|马仲英]]反叛，引发[[第四次河湟事变|第四次河湟事变]]。民国18年（1929年），改称临夏县。民国33年（1944年），属甘肃省临夏行政督察专员公署，仍辖临夏、临洮、宁定、和政、永靖、夏河等6县。1936年5月，改临夏行政督察专员公署为甘肃省第五区行政督察专员公署，将临洮县划归第一行政督察区。1939年4月，将夏河县划归第一行政督察区。1940年，改[[康乐设治局|康乐设治局]]为康乐县。

1949年8月，[[陕甘宁边区|陕甘宁边区人民政府]]发布第120号命令，设立[[临夏专区|临夏行政督察专员公署]]，辖临夏、临洮、洮沙（治今临洮县[[太石镇|太石镇]]）、宁定（治今广河县[[城关镇_(广河县)|城关镇]]）、和政、康乐、永靖、夏河等8县。1949年8月下旬，[[中国人民解放军第一野战军|中国人民解放军第一野战军]]第一兵团解放临夏地区。1949年9月初，在临夏县成立洮西行政督察专员公署，辖上述8县。1949年9月下旬，改洮西专区为临夏专区；将临洮县划归新设的临洮专区。

1950年5月，洮沙县划归临洮专区，夏河县改由省直辖；将岷县分区的临潭县划归临夏分区。以临夏县东乡以及永靖、和政、宁定等县部分地区设立东乡自治区（县级）。1950年6月，析临夏县城区置临夏市（县级），临夏县由城区迁驻韩家集。1950年9月，改临夏行政督察专员公署为临夏分区专员公署。

1953年，临夏市改由省直辖；宁定县改设[[广河县|广通回族自治区]]（县级）；临潭县划归[[甘南藏族自治区|甘南藏族自治区]]；10月[[东乡族自治县|东乡自治区]]改为[[东乡族自治县|东乡族自治区]]。

1955年7月，东乡族自治区改名东乡族自治县；广通回族自治区改名广通回族自治县。1956年11月，撤销临夏专区，设立'''临夏回族自治州'''。辖临夏市和临夏、和政、永靖、康乐4县及东乡族自治县、广通回族自治县。寻改广通回族自治县为广通县。

1957年，[[广通县|广通县]]改称[[广河县|广河县]]。1958年，永靖、临夏2县并入临夏市；广河、康乐2县并入和政县，1961年恢复。1973年12月，甘肃省报告国务院：撤销临夏市，并入临夏县，临夏县由韩家集迁驻城关镇（今临夏市城区）。1980年6月，国务院批准：析临夏县西北部置积石山保安族东乡族撒拉族自治县。1983年8月，国务院批准：恢复临夏市（县级），以临夏县的城关镇和城关、南龙、折桥、枹罕4个公社为临夏市的行政区域。临夏县迁驻韩家集（第三次）。

== 地理 ==
临夏州位于甘肃中部，省会[[兰州|兰州]]以南，西为[[青海省|青海省]]边界，南面是[[甘南藏族自治州|甘南藏族自治州]]，东邻[[定西市|定西市]]。地形以高原、山区、黄土丘陵区为主。平均海拔2000米以上。[[黄河|黄河]]贯穿西北部，在永靖县[[刘家峡|刘家峡]]前形成[[刘家峡水库|刘家峡水库]]。黄河主要支流——大夏河和[[洮河|洮河]]从邻近的[[甘南州|甘南州]]流向刘家峡水库。洮河亦是临夏州与定西市的一部分边界。全年平均气温为8℃，无霜期155天，为[[温带半干旱气候|温带半干旱气候]]，年均降水量只有442毫米。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全州[[常住人口|常住人口]]为1946677人<ref>{{cite book |author=临夏回族自治州统计局|title=《临夏回族自治州2010年第六次全国人口普查主要数据公报》}}</ref>。其中，男性人口为985235人，占50.61%；女性人口为961442人，占49.39%。人口性别比（以女性为100）为102.47。0－14岁人口为422288人，占21.69%；15－64岁人口为1365200人，占70.13%；65岁及以上人口为159189人，占8.18%。

=== 民族 ===
全州常住人口中，[[汉族|汉族]]人口为772872人，占39.70%；各[[少数民族|少数民族]]人口为1173805人，占60.30%。

临夏是个多元民族、宗教的地方，回族占境内的人口约三分之一。信奉[[伊斯兰教|伊斯兰教]]的民族包括[[回族|回族]]、[[东乡族|东乡族]]、[[保安族|保安族]]、[[撒拉族|撒拉族]]等，占全州人口超过一半。另外还有[[土族|土族]]、[[藏族|藏族]]等少数民族。

== 经济建设 ==
[[File:6058-Liujiaxia-Dam.jpg|thumb]]

临夏州是全国最贫困的地区之一，全州除首府临夏市外，其余县份均为[[国家级贫困县|国家级贫困县]]。2010年，全州人均生产总值5441元，约804美元，东乡族自治县更只有2785元，即411美元。

2011年，全州[[生产总值|生产总值]]128.78亿元，比上年增长13.1%。其中，第一产业增加值27亿元，比上年增长5.2%；第二产业增加值39.94亿元，比上年增长15.3%；第三产业增加值61.84亿元，比上年增长15.4%。<ref>[http://www.gstj.gov.cn/doc/ShowArticle.asp?ArticleID=13386 《2011年临夏回族自治州国民经济和社会发展统计公报》] {{webarchive|url=https://archive.is/20130422230325/http://www.gstj.gov.cn/doc/ShowArticle.asp?ArticleID=13386 |date=2013-04-22 }}</ref>

* [[刘家峡水电站|刘家峡水电站]]位于[[永靖县|永靖县]]境内[[黄河|黄河]]干流上，是当时亚洲第一座百万千瓦级大型水电站。始建于1958年9月，1974年12月竣工。中共中央总书记[[胡锦涛|胡锦涛]]和夫人[[刘永清|刘永清]]，曾亲身参加刘家峡水电站的建设。

== 著名景点 ==
* [[炳灵寺石窟|炳灵寺石窟]]——位于永靖县[[积石山|积石山]]大寺沟西侧的崖壁上。[[西晋|西晋]]初年开凿在黄河北岸大寺沟的峭壁之上，正式建立于[[西秦|西秦]][[建弘|建弘]]元年（420年），[[明朝|明朝]][[永乐|永乐]]后称[[炳灵寺|炳灵寺]]。
* [[临夏南关大寺|临夏南关大寺]]——[[伊斯兰教|伊斯兰教]][[清真寺|清真寺]]，位于[[临夏市|临夏市]]南门广场，是甘肃规模较大，历史较悠久的清真寺。
* [[临夏老华寺|临夏老华寺]]——伊斯兰教清真寺，亦称大华寺，位于[[临夏市|临夏市]]华寺街。始建于1368年，距今已经有600多年的历史，为甘肃规模较大、历史较久远的清真寺。
* [[临夏东关大清真寺|临夏东关大清真寺]]
* [[临夏万寿观|临夏万寿观]]
* [[八坊十三巷|八坊十三巷]]——位于[[临夏市|临夏市]]主城区，是一片具有回族民族特色的历史文化街区，[[国家4A级景区|国家4A级景区]]，被称为民族建筑艺术“大观园”、“甘肃新地标”和临夏文化旅游新名片。
