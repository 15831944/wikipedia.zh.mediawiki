《'''兵临城下'''》，2011年上映于[[中国大陆|中国大陆]]的战争剧，讲述了[[抗日战争|抗日战争]]以及[[国共内战|国共内战]]时期[[四川省|四川省]][[成都|成都]]的谍战和暗战的故事，主演是[[于和伟|于和伟]]、[[王媛可|王媛可]]、[[潘虹|潘虹]]、[[邵汶|邵汶]]、[[午马|午马]]、[[石天琦|石天琦]]。於2010年12月30日在湖南影视频道首播<ref>{{cite web|url=http://ent.sina.com.cn/v/m/2010-12-30/16023193748.shtml | language= | title="《兵临城下》成跨年大戏 湖南影视抢滩首播 |publisher=新浪娱乐 |date=2010-12-30 |accessdate= }}</ref>。

== 剧情 ==
[[抗日战争|抗日战争]]胜利之后，[[中国共产党|中国共产党]]和[[中国国民党|中国国民党]]陷入了[[国共内战|国共内战]]，[[四川省|四川省]][[成都|成都]]这个西部城市迎来了各派势力的明争暗斗。

== 花絮 ==
电视剧改编自[[袁春雨|袁春雨]]的同名小说。<ref>{{cite web |url=http://news.163.com/11/1019/10/7GNLJHGE00014AED.html |title=谍战剧《兵临城下》登陆卫视 |accessdate=2011-10-19  |publisher=网易 |work= |language =zh-cn}}</ref>

男演员刘欢的表演受到了观众的追捧。<ref>{{cite web |url=http://yule.sohu.com/20110915/n319464533.shtml |title= 刘欢《兵临城下》表现亮眼 两剧连播受关注|accessdate= 2011年9月15日 |publisher= 搜狐网|work= |language =zh-cn}}</ref>
