《'''兵团日报'''》，是[[中国共产党新疆生产建设兵团委员会|中国共产党新疆生产建设兵团委员会]]的[[机关报|机关报]]。

==简介==
中国共产党新疆生产建设兵团委员会机关报起初是1953年5月22日创刊的《新疆解放军》报分版出刊的《生产战线》。1957年元旦，更名为《新疆生产战线报》。1969年4月，更名为《军垦战报》。1983年8月1日，更名为《新疆军垦》报。1991年元旦，更名为《新疆军垦报》。1994年元旦，更名为《兵团日报》<ref>康红梅，新疆生产建设兵团报、刊文献资源的研究与利用，农业图书情报学刊2009, 21(7):189-191</ref>。《兵团日报》用[[汉文|汉文]]、[[维吾尔文|维吾尔文]]两种文字出版<ref>{{Cite web|url =http://bt.xinhuanet.com/2012-08/10/c_112689206.htm|title =兵团日报|author =|date =2012-08-10|publisher =兵团网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170401144348/http://bt.xinhuanet.com/2012-08/10/c_112689206.htm|archivedate =2017-04-01}}</ref>。

《兵团日报》由兵团日报社出版。兵团日报社是[[中国共产党新疆生产建设兵团委员会宣传部|中国共产党新疆生产建设兵团委员会宣传部]]下属正厅级事业单位，下辖兵团网、兵团手机报、生活晚报、传播论坛杂志等等<ref>{{Cite web |url = http://yjsc.xjnu.edu.cn/s/35/t/45/38/c2/info80066.htm |title = 兵团日报社招聘会公告 |author =  |date = 2015-03-30 |publisher = 新疆师范大学研究生处 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.xjbt.gov.cn/c/2015-03-13/543616.shtml |title = 郭永辉与兵团日报社、当代兵团杂志社工作人员座谈 |author =  |date = 2015-03-13 |publisher = 新疆生产建设兵团 |language =  |accessdate =  |quote =  }}</ref>。
