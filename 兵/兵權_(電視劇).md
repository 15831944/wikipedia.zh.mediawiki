{{engname|《兵權》|'' Bing Kuen ''}}，又名《陳橋兵變》，是[[電視廣播有限公司|電視廣播有限公司]]在1988年播出的古裝電視劇集，由[[劉仕裕|劉仕裕]]監製，[[劉家輝|劉家輝]]、[[黎明|黎明]]、[[邵美琪|邵美琪]]等演出<ref>{{Cite web |url =http://www.chinadaily.com.cn/interface/yidian/1120781/2016-05-03/cd_25031707.html|title =黎明演過的電影 黎明主演過哪些電影|date = 2016年5月3日|publisher =網易娛樂|language =中文}}</ref><ref>{{Cite web |url =http://big5.ent.qianzhan.com/ent/star/c5f2e623.html|title =商天娥個人資料簡歷檔案| publisher =前瞻網娛樂|language =中文}}</ref>，於1988年12月1日首次在[[無綫電視|無綫電視]][[翡翠台|翡翠台]]播出，合共20集<ref>{{Cite web |url =http://vipo.tvb.com/mam/test/page:10|title =兵權首播日期| publisher =tvb.com|language =中文}}</ref><ref>[http://blog.sina.com.cn/s/blog_7a103c860102v22q.html  1988年无线《兵权》（惠天赐、刘家辉、黎明、邵美琪等）]</ref><ref>[http://yule.bandao.cn/news_html/201803/20180331/news_20180331_2818283.shtml  黎明压轴加盟《将夜》 首曝剧照帝王霸气归来]</ref>。

==劇情簡介==
電視劇《兵權》是根據[[五代十國|五代十國]]及[[宋朝|宋朝]]的史實改編而成，講述[[趙匡胤|趙匡胤]]和[[柴榮|柴榮]]的故事。趙匡胤與柴榮、鄭恩、賀彤等，一起協助[[郭威|郭威]]建立[[後周|後周]]。[[張永德|張永德]]是郭威的女婿，因不滿郭威建立後周而勾結[[契丹|契丹]]的耶律世藩，密謀殺害郭威，柴榮為了得到帝位，故意不出伸援手，郭威最終因失救而傷重死亡。郭威死後，柴榮繼承帝位，是為周世宗。秦傲是趙匡胤的朋友，他們同時戀上耶律納蘭，耶律納蘭卻與賀彤一同愛上趙匡胤，然而趙匡胤與來自契丹的納蘭在身份上是敵人，不容許結合，趙匡胤最終與賀彤成親。柴榮一方面派遣趙匡胤到契丹與耶律安圖議和，另一方面卻暗中派兵攻打契丹，趙匡胤與柴榮因此事而互相猜忌。趙匡胤多次立下大功，受到其他官員和民眾的愛戴，柴榮擔心掌管兵權的趙匡胤乘機判變，於是設計奪走趙匡胤的兵權，雙方陣營發生衝突，後來柴榮因病逝世，將領們把[[黃袍|黃袍]]穿在趙匡胤的身上，推舉他為皇帝，國號為宋，歷史稱趙匡胤為宋太祖<ref>{{Cite web |url =https://www.tvmao.com/drama/aWFgayNi|title =兵權劇情介紹|publisher =電視貓|language =中文}}</ref><ref>{{Cite web |url =https://movie.douban.com/subject/3115208/|title =兵權（1988）| publisher =豆瓣電影|language =中文}}</ref>。
