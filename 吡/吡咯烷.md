'''吡咯啶'''，又称'''四氢吡咯'''，是五元含[[氮|氮]]的[[不饱和化合物|饱和]][[杂环化合物|杂环化合物]]。

== 性质 ==

吡咯烷是无色透明有特殊[[氨|氨]]气味的液体，见光或潮湿空气易变黄色，有毒。与水、[[乙醇|乙醇]]、[[乙醚|乙醚]]、[[氯仿|氯仿]]混溶。四氢吡咯的衍生物广泛存在于许多[[天然产物|天然产物]]中，如烟草中的成分[[尼古丁|尼古丁]]和[[蛋白质|蛋白质]]中的[[脯氨酸|脯氨酸]]就是四氢吡咯的2-取代物。

== 制取 ==

由[[吡咯|吡咯]]的[[催化氢化|催化氢化]]而得。吡咯的氢化比[[呋喃|呋喃]]要困难得多，在温和条件下不易氢化，在较高反应用[[铂|铂]]催化氢化可得四氢吡咯。氢化后失去了吡咯环的[[芳香性|芳香性]]，因此产物[[二氢吡咯|二氢吡咯]]（[[吡咯啉|吡咯啉]]）和四氢吡咯的[[碱#酸碱电子理论中碱的定义|碱性]]都有所增强。

== 用途 ==

用作医药原料、特殊有机溶剂，也用于有机合成。四氢吡咯作为[[仲胺|仲胺]]，可以与[[酮|酮]]类生成[[烯胺|烯胺]]，这一点在有机合成上很有用。
