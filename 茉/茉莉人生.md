《'''茉莉人生'''》（{{lang|fr|Persepolis}}，直譯为[[波斯波利斯|波斯波利斯]]）是一部2007年上映的[[動畫|動畫]][[電影|電影]]，根據[[瑪嘉·莎塔碧|瑪嘉·莎塔碧]]所創作的[[波斯波利斯_(漫畫)|同名的漫畫]]所改編的。

== 劇情簡介 ==
瑪嘉是一位異想天開的9歲[[少女|少女]]，出身于[[德黑兰|德黑兰]]的一个[[左翼|左翼]]知识分子家庭，外祖父是[[卡扎尔王朝|卡扎尔王朝]]的一个王子，也是一个[[共产主义者|共产主义者]]，最后死在[[伊朗君主列表|伊朗沙阿]]的监狱中，父母都在政治上支持[[马克思主义|马克思主义]]事业并反对[[穆罕默德-礼萨·巴列维|巴列维国王]]的统治，最敬爱的阿鲁什叔叔则是信仰[[共产主义|共产主义]]的[[伊朗人民党|伊朗人民党]]的成员，尝尽[[流亡|流亡]]和坐牢的苦头。她喜愛[[西方文化|西方文化]]，酷愛偶像[[李小龍|李小龍]]。[[真主|真主]]和[[马克思|马克思]]是她的密友，成天愛幻想，夢想是成為全[[宇宙|宇宙]]最後一位[[先知|先知]]。[[伊朗伊斯兰革命|伊朗伊斯兰革命]]后，[[伊斯兰原教旨主义|伊斯兰原教旨主义]]者掌握了大权，在国内实行[[政教合一|政教合一]]制度，推行极其保守的社会政策，严格控制西方文化传播，在学校强制灌输[[伊斯兰教|伊斯兰教义]]，女性被迫戴上头巾，宗教警察在街头监视着人们的一言一行，饮酒及各种为[[伊斯兰教法|伊斯兰教法]]所不容忍的娱乐都被禁止，阿鲁什叔叔等共产主义者被处决，神权当局的蛊惑驱使大量青年在[[两伊战争|两伊战争]]中成为炮灰。这一切让瑪嘉感到极其压抑。她的叛逆和不安分让雙親和外祖母不时感到担忧，週遭的人也認為她是怪孩子，保守的[[伊朗|伊朗]]婦女認為她是西方腐敗的象徵。她喜愛的一切不被保守的祖國所接受，她只能偷偷地去買[[黑膠唱片|黑膠唱片]]和所有喜愛的東西。原本以為日子可以如此幻想地過下去，直到国内的政治局势继续恶化时，雙親决定把她送到[[歐洲|歐洲]]。在成長的歲月中不斷地發生苦難，在被愛情刺得遍體鱗傷後，在西方的颓废事物让她迷茫时，開始懷念家鄉和雙親，還有外祖母和她身上的[[茉莉|茉莉]]香。她選擇回到家鄉，大家一起尋找快樂彌補過去失去的時光，但當自由的真諦出現在眼前時，她又不得不離鄉去追夢......

瑪嘉·莎塔碧说：
{{Cquote2|我知道，當我選擇說出這個故事，我就再也回不去了。我的故鄉，波斯波利斯。那覆蓋著藹藹白雪的阿爾伯茲山，以及外婆身上迷人的茉莉花香。
 
然而，我選擇告訴你們這個故事，從古波斯文明的波斯波利斯， 直到現代伊斯蘭下的神秘面紗。我永遠記得外婆的話：不要忘記妳從哪裡來。
 
這是我純真的革命童年，我青春的愛情流浪。這是我的伊朗，我的家鄉。這是我的《波斯波利斯》。 }}

== 技术 ==

影片按照原始的[[绘本|绘本]]以黑白作为主色调。影片中除了“现代”的部分有颜色外，历史记叙的部分采用类似[[皮影戏|皮影戏]]的效果。将[[漫画|漫画]]版转变为[[动画|动画]]版的过程，由艺术指导和执行制片人[[Marc_Jousset|Marc Jousset]] 重新设计。动画版署名是[[Perseprod_Studio|Perseprod Studio]]，同时还由[[Je_Suis_Bien_Cotent|Je Suis Bien Cotent]]和[[Pumpkin_3D|Pumpkin 3D]]这两个专业工作室负责制作。

== 評價 ==
《茉莉人生》受到許多影評人與觀眾給予高度的評價，並在[[爛番茄|爛番茄]]與[[Metacritic|Metacritic]]都得到相當高的分數。
