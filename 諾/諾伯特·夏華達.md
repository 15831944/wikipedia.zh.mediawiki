'''諾伯特·夏華達'''（{{lang|en|Norbert Hauata}}，{{bd|1979年|6月8日|||}}）是[[塔希提|塔希提]]的[[足球裁判|足球裁判]]。他於2008年成為[[國際足協|國際足協]]的國際裁判，得以為國際比賽判法<ref>[http://worldreferee.com/site/copy.php?linkID=1970&linkType=referee&contextType=stats "Palmares for Norbert Hauata"]</ref>。同年9月10日，為[[紐西蘭國家足球隊|紐西蘭]]對[[新喀里多尼亞國家足球隊|新喀里多尼亞]]的[[大洋洲國家盃|大洋洲國家盃]]賽事判法，這是他首場裁決的國際比賽。2010年，為[[大洋洲聯賽冠軍盃|大洋洲聯賽冠軍盃]]決賽次回合判法。翌年，又兩度出任為[[2011年U17世界盃足球賽|U17世界盃]]分組賽的球證<ref>[http://www.fifa.com/tournaments/archive/u17worldcup/mexico2011/matches/round=255973/match=300151848/report.html "Match Report - Panama - Germany 0:2 (0:2)"]</ref>。2012年，他為三場的大洋洲國家盃判法，當中包括一場半決賽<ref>[http://www.fifa.com/worldcup/preliminaries/oceania/matches/round=258415/match=300200739/report.html "Match Report - New Zealand - New Caledonia 0:2 (0:0)"]</ref>。之後，他又被委任為[[2013年U17世界盃足球賽|2013年U17世界盃足球賽]]的球證，這次，他同樣為兩場的分組賽判法。2014年，他被國際足協選為[[2014年世界盃|2014年世界盃]]的替補球證，並在[[墨西哥國家足球隊|墨西哥]]對[[喀麥隆國家足球隊|喀麥隆]]的分組賽中擔當第四球證<ref>[http://resources.fifa.com/mm/document/tournament/competition/02/36/99/76/eng_02_0613_mex-cmr_fulltime.pdf Match report]</ref>。
