'''諾福克'''（'''Norfolk''' {{IPAc-en|ˈ|n|ɔr|f|ʉ|k}} {{respell|NOR|fək}}；{{IPAc-en|local|ˈ|n|ɒ|f|ʊ|k}} {{respell|NOF|uuk}}）是[[美國|美國]][[維吉尼亞州|維吉尼亞州]]東南部[[漢普頓錨地|漢普頓錨地]]的一個[[獨立市_(美国)|獨立市]]，是該州第二大城市。

諾福克成立於1945年2月11日。城名來自[[英國|英國]]的[[諾福克郡|諾福克郡]]。<ref>Kane, J. N. and C. C. Aiken. The American Counties: Origins of County Names Dates of Creation and Population Data 1950-2000. Lanham, MD.: Scarecrow Press, 2005.</ref>[[諾福克海军基地|諾福克海军基地]]是美國最大的[[海军基地|海军基地]]，美國海軍絕大部分艦艇都在這裡生産。

==地理与气候==
诺福克属[[副热带湿润气候|副热带湿润气候]]，四季分明，气温变化相对和缓，全年降水分配均匀。冬季温和，微潮，时而偏向温暖，日最高气温低于{{convert|0|C}}的平均日数为3.3天，日最低气温低于{{convert|0|C}}的平均日数为46天，低于{{convert|-5|C}}的平均日数为8.6天；夏季相对炎热潮湿，日最高气温超过{{convert|30|C}}的日数年均有64天，超过{{convert|35|C}}的有9.1天。<ref name = "NWS Wakefield, VA"/>最冷月（1月）均温{{convert|40.4|F|1|disp=flip}}，极端最低气温{{convert|−3|F|0|disp=flip}}（1985年1月21日）。<ref name = "NWS Wakefield, VA"/>最热月（7月）均温{{convert|79.6|F|1|disp=flip}}，极端最高气温{{convert|105|F|0|disp=flip}}（上次出现于2010年7月24日与7月25日）。<ref name = "NWS Wakefield, VA"/>无霜期平均为250天（3月18日至11月22日）。<ref name = "NWS Wakefield, VA"/>年均降水量约{{convert|46.5|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|26.48|in|disp=flip}}（1986年），最多为{{convert|70.72|in|disp=flip}}（1889年）。<ref name = "NWS Wakefield, VA"/>年均降雪量为{{convert|5.8|in|cm|disp=flip}}，<ref name = "NWS Wakefield, VA"/>但降雪年际变化较大；1998–99年的降雪量最少，积累降雪量微少以致难以测量，1979–80年的降雪量最多，积累降雪量为{{convert|41.9|in|cm|disp=flip}}。<ref name = "NWS Wakefield, VA"/>
