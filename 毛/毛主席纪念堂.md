'''毛主席纪念堂'''是时任[[中国共产党中央委员会主席|中国共产党中央委员会主席]]、[[中国共产党中央军事委员会主席|中国共产党中央军事委员会主席]]、[[中国人民政治协商会议全国委员会|中国人民政治协商会议全国委员会]]名誉主席[[毛泽东|毛泽东]]于1976年9月9日[[毛泽东逝世|逝世]]后，根据同年10月8日《中国共产党中央委员会、中华人民共和国全国人民代表大会常务委员会、中华人民共和国国务院、中国共产党中央军事委员会关于建立伟大领袖和导师毛泽东主席纪念堂的决定》而建造的纪念性建筑物，位於[[天安门广场|天安门广场]]南侧、[[北京中轴线|北京中轴线]]上，其中陈列有经特殊处理保存的毛泽东的[[遗体|遗体]]。其外形和美國的[[林肯紀念堂|林肯紀念堂]]類似。毛主席纪念堂由[[中共中央办公厅|中共中央办公厅]][[毛主席纪念堂管理局|毛主席纪念堂管理局]]管理。

== 历史 ==
毛主席纪念堂的原址为[[中华门_(北京)|中华门]]，于1958年後被拆除。1976年11月24日，毛主席纪念堂奠基。1977年9月9日举行落成典礼并对外开放<ref name="gaikuang">{{Cite web |url = http://cpc.people.com.cn/GB/69112/113427/6694839.html  |title = 毛主席纪念堂概况 |author =  |date =  |publisher = 毛主席纪念堂官方网站 |language =  |accessdate =2014-01-03  |quote =  }}</ref>。

1976年9月9日，[[毛泽东逝世|毛泽东逝世]]。1976年9月14日，[[国家计委|国家计委]]组织全国八省市十多家单位的设计人员集中到[[前门饭店|前门饭店]]，开始毛主席纪念堂的选址及方案设计工作。设在[[前门饭店|前门饭店]]的毛主席纪念堂规划设计领导小组由[[赵鹏飞|赵鹏飞]]、[[袁镜身|袁镜身]]、[[沈勃|沈勃]]等人组成，参加设计的人员来自[[中国建筑科学研究院|中国建筑科学研究院]]、[[北京市规划局|北京市规划局]]、[[北京市建筑设计研究院|北京市建筑设计研究院]]、[[清华大学|清华大学]]、[[天津大学|天津大学]]、上海市民用建筑设计院、广东省建筑设计研究院、广州市建筑设计研究院、[[南京工学院|南京工学院]]、中国建筑西北设计研究院、辽宁省建筑设计研究院、黑龙江省建筑设计研究院、[[中国人民解放军基本建设工程兵|中国人民解放军基本建设工程兵]]设计院。《建筑学报》上曾经总结道：“毛主席纪念堂的设计方案，是来自八个省市的老中青设计人员共同进行的。以后由北京市建筑设计研究院等在京单位的工人、干部和技术人员组成毛主席纪念堂规划设计组，继续完成了纪念堂的综合方案设计和整个工程的全部设计工作。”<ref name="ma" />

1976年10月8日，《[[s:中国共产党中央委员会、中华人民共和国全国人民代表大会常务委员会、中华人民共和国国务院、中国共产党中央军事委员会关于建立伟大的领袖和导师毛泽东主席纪念堂的决定|中国共产党中央委员会、中华人民共和国全国人民代表大会常务委员会、中华人民共和国国务院、中国共产党中央军事委员会关于建立伟大的领袖和导师毛泽东主席纪念堂的决定]]》发布。1976年10月15日，北京市建筑设计研究院成立毛主席纪念堂规划设计组，并建立临时党支部。1976年11月6日，[[中共中央政治局|中共中央政治局]]审查毛主席纪念堂方案。1976年11月9日，毛主席纪念堂工程指挥部成立，时任北京市建委副主任[[李瑞环|李瑞环]]出任总指挥。[[中华人民共和国国务院副总理|国务院副总理]][[谷牧|谷牧]]负责纪念堂建设的领导工作。1976年11月24日，中共中央政治局最后审定毛主席纪念堂方案，毛主席纪念堂奠基<ref name="ma">马国馨，三十年前纪念堂，建筑创作2007年09期</ref><ref name=daxiu/>。[[华国锋|华国锋]]参加了奠基仪式，并亲自为纪念堂基石培土<ref name=daxiu/>。

1977年2月，毛主席纪念堂工程指挥部召开[[工业学大庆|工业学大庆]]会议。1977年3月22日，毛主席纪念堂主体结构提前完成。1977年5月24日毛主席纪念堂建筑工程完工。1977年6月17日，毛主席纪念堂工程指挥部召开总结工作、表彰先进大会。1977年7月23日，[[中共十届三中全会|中共十届三中全会]]公报发表。1977年8月18日，[[中国共产党第十一次全国代表大会|中国共产党第十一次全国代表大会]]新闻公报发表。1977年8月22日，出席中共十一大的代表瞻仰毛主席遗容，同日发表[[中共十一届一中全会|中共十一届一中全会]]公报。1977年8月30日，《[[人民日报|人民日报]]》、《[[北京日报|北京日报]]》头版发表“毛主席纪念堂全部胜利建成”消息。1977年8月31日，[[南斯拉夫总统|南斯拉夫总统]][[铁托|铁托]]赴毛主席纪念堂瞻仰毛泽东遗容。1977年9月9日，[[中共中央|中共中央]]、[[全国人大常委会|全国人大常委会]]、[[中华人民共和国国务院|国务院]]、[[中共中央军委|中共中央军委]]在毛主席纪念堂北门广场召开“纪念伟大领袖和导师毛主席逝世一周年及毛主席纪念堂落成典礼大会”。此后，全国各省、自治区、直辖市的代表先后前来瞻仰，截至1977年9月26日达到16万人<ref name="ma" />。

毛主席纪念堂每年都会接受例行设备维护和检修，时间约为十余天。自2009年以来，时间一般安排在3月1日至20日。一般维护和检修项目包括对电器设备如空调、照明设施及其他机器设备进行维护等。

1997年4月1日至12月31日，毛主席纪念堂大修，重点维修下层屋面和下檐天沟，以免造成渗漏，维修期间暂停对外开放<ref name=daxiu/>。

2007年3月3日至9月20日，毛主席纪念堂大修，进行建筑和内部设备维修施工，施工期间暂停对外开放<ref>{{Cite web |url = http://news.sina.com.cn/c/2007-03-03/125911330509s.shtml |title = 毛主席纪念堂即将闭馆修缮 |author =  |date =2007-03-03  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.cctv.com/china/20070228/107353.shtml  |title = 毛主席纪念堂3月3日至9月20日施工暂停对外开放 |author =  |date = 2007-02-28 |publisher = 央视国际 |language =  |accessdate =  |quote =  }}</ref>。

2013年7月，有关部门对毛主席纪念堂门前雕塑、屋顶和廊檐等外部结构进行局部修缮；7月14日，工人开始对毛主席纪念堂大门[[匾额|匾额]]上的“毛主席纪念堂”六个大字重新喷涂金粉。此次维修期间照常开放<ref>{{Cite web |url = http://news.ifeng.com/photo/hdnews/detail_2013_07/15/27500112_0.shtml#p=1  |title = 毛主席纪念堂进行修缮工作 |author =  |date = 2013-07-15 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref name=daxiu/>。

2013年6月2日至5日，毛主席纪念堂南北庭院路面施工，暂停对外开放。同年11月10日、11月12日（11月11日是周一，例行不开放），因系统软件升级，毛主席纪念堂暂停开放团队预约<ref name=daxiu/>。

2017年3月1日至8月31日，毛主席纪念堂大修，进行建筑和内部设备维修施工，施工期间暂停对外开放。此次改造施工由中国新兴建设开发总公司承建。2017年9月1日起毛主席纪念堂正式恢复对外开放，9月2日起开放团队预约<ref name=daxiu>{{Cite web |url = http://news.ifeng.com/a/20170901/51842063_0.shtml |title =大修183天，毛主席纪念堂再开放  |author =  |date = 2017-09-01 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

[[中共中央|中共中央]]曾在毛主席纪念堂落成典礼（1977年）、毛泽东诞辰90周年（1983年）、100周年（1993年）、110周年（2003年）、120周年（2013年）之际，在毛主席纪念堂隆重举行纪念活动<ref name=gaikuang/><ref>{{Cite web |url =  http://www.chinanews.com/gn/2013/12-26/5665113.shtml|title =毛泽东诞辰120周年 习近平等赴纪念堂瞻仰遗容  |author =  |date =  2013-12-26|publisher =  中新网|language =  |accessdate =  |quote =  }}</ref>。

为纪念毛泽东诞辰114周年，[[人民网|人民网]]—中国共产党新闻网与毛主席纪念堂管理局联合推出“毛主席纪念堂专题网页”，2007年12月26日上午9时9分正式开通<ref>{{Cite web |url = http://cpc.people.com.cn/GB/64093/64387/6694932.html |title =毛主席纪念堂专题网页将于2007年12月26日开通  |author =  |date =2007-12-25  |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。

==建筑==
毛主席纪念堂南侧为[[正阳门|正阳门]]城楼，北侧为[[人民英雄纪念碑|人民英雄纪念碑]]，毛主席纪念堂与人民英雄纪念碑和正阳门城楼各相距200米。<ref name=dangshi/>

毛主席纪念堂南北长260米，东西宽220米，占地面积57200平方米，总建筑面积33867平方米。主体建筑长宽各105.5米，高33.6米，是柱廊型正方体，坐南朝北，南北正面上方镶嵌着刻有[[中共中央主席|中共中央主席]]、[[中华人民共和国国务院总理|国务院总理]][[华国锋|华国锋]]题写的“毛主席纪念堂”六个金色大字的[[汉白玉|汉白玉]]匾额。外廊由44根[[福建|福建]]黄色方形[[花岗岩|花岗岩]]明柱环抱，每根柱高17.5米。明柱间装饰有[[广州|广州]]石湾花饰陶板，明柱通体采用[[青岛|青岛]][[花岗岩|花岗岩]]贴面。屋顶有两层玻璃飞檐，檐间镶有[[葵花|葵花]]浮雕。基座有两层平台，台基高4米，边长105.5米；台帮全部采用[[四川|四川]][[大渡河|大渡河]]畔的枣红色花岗岩砌成，四周环以[[北京|北京]][[房山|房山]][[汉白玉|汉白玉]]万年青花饰栏板。南、北门的台阶中间，各有两条汉白玉垂带，上面雕刻葵花、万年青、腊梅、青松图案。<ref name=gaikuang/><ref name=dangshi>[http://dangshi.people.com.cn/GB/165617/166498/9989408.html 毛主席纪念堂]，中国共产党新闻网，于2014-01-03查阅</ref><ref name=dhly>[http://www.bjdclib.com/subdb/laneculture/historybuilding/200906/t20090611_1897.html 毛主席纪念堂]，东华流韵，于2014-01-03查阅</ref>毛主席纪念堂南门外的30面红旗，代表全国30个省、自治区、直辖市。<ref>[http://news.takungpao.com/history/oldpic/2013-08/1338634_8.html 毛主席纪念堂轰轰烈烈营造过程]，大公网，于2013-01-04查阅</ref>

=== 设施 ===
毛主席纪念堂向群众免费開放的廳室有北大廳、瞻仰廳、南大廳，以及毛澤東、[[周恩來|周恩來]]、[[劉少奇|劉少奇]]、[[朱德|朱德]]、[[鄧小平|鄧小平]]、[[陳雲|陳雲]]同志革命業績紀念室和電影廳等等<ref name=gaikuang/><ref name=dangshi/>。

; 毛主席纪念堂一层开放的厅室
* 北大厅：北大厅是瞻仰参观入口以及举办纪念仪式的场所，可容纳700多人。北大厅正中安放着汉白玉坐像《毛泽东主席》，总高3.45米，毛泽东面含微笑，端坐在沙发上。背景是巨幅绒绣壁画《祖国大地》，高6.6米，宽23.74米，展现中国壮美的山河<ref name=gaikuang/><ref name=dangshi/><ref name=dhly/>。
* 瞻仰厅：瞻仰厅正面的汉白玉墙面上，镶着金字[[隶书|隶书]]“伟大的领袖和导师毛泽东主席永垂不朽”，为[[侯德昌|侯德昌]]书写<ref>{{Cite web |url =http://culture.ifeng.com/whrd/detail_2012_11/15/19201014_0.shtml  |title =解析中共新常委亮相背景国画《幽燕金秋图》  |author =  |date =2012-11-15  |publisher =  凤凰网|language =  |accessdate =  |quote =  }}</ref>。在瞻仰厅正中间的[[水晶棺|水晶棺]]中，停放着毛泽东的遗体。四個表情嚴肅的衛兵守護著毛泽东的水晶棺。毛泽东遺体身着灰色[[中山装|中山装]]，覆盖着[[中国共产党党旗|中国共产党党旗]]。水晶棺的基座采用[[泰山|泰山]]黑色花岗岩制成，棺座四周分别镶嵌着金饰[[中国共产党党徽|中国共产党党徽]]、[[中华人民共和国国徽|中华人民共和国国徽]]、[[中国人民解放军军徽|中国人民解放军军徽]]、毛泽东生卒年份。水晶棺周围是[[君子兰|君子兰]]以及玻璃栏杆。<ref name=gaikuang/><ref name=dangshi/>
* 南大厅：南大厅是瞻仰参观出口。北侧白色大理石墙面上，刻有鎏金的毛泽东词《[[满江红|满江红]]·和[[郭沫若|郭沫若]]同志》手迹。<ref name=gaikuang/><ref name=dangshi/><ref name=dhly/>

; 毛主席纪念堂二层开放的厅室
* “毛泽东、周恩来、刘少奇、朱德、邓小平、陈云同志革命业绩纪念室”：1983年12月26日，经[[中共中央书记处|中共中央书记处]]决定，在毛主席纪念堂设立的“毛泽东、周恩来、刘少奇、朱德同志革命业绩纪念室”正式开放<ref>{{Cite web |url =http://cpc.people.com.cn/GB/64162/64164/4416129.html |title = 中国共产党大事记·1983年 |author =  |date =  |publisher = 中国共产党新闻网 |language =  |accessdate =  2017-09-03|quote =  }}</ref>。2001年，为纪念[[中国共产党|中国共产党]]建党80周年，经[[中共中央|中共中央]]批准，又增设“邓小平、陈云同志革命业绩纪念室”，同时对1983年建成的“毛泽东、周恩来、刘少奇、朱德同志革命业绩纪念室”的陈列内容及形式作出调整及补充。这六个纪念室共陈列文物102件、图片490张、文献224件<ref>{{Cite web |url = http://www.people.com.cn/GB/shizheng/19/20010630/500569.html |title = 毛主席纪念堂增设邓小平陈云同志革命业绩纪念室 对毛泽东周恩来刘少奇朱德同志革命业绩纪念室进行调整补充 |author =  |date =  2001-06-30|publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref><ref name=gaikuang/><ref name=dangshi/>。
* 电影厅：放映文献纪录片《怀念》<ref name=gaikuang/>。

2012年，在[[毛泽东|毛泽东]]诞辰120周年前夕（2013年为毛泽东诞辰120周年），中共中央办公厅决定在毛主席纪念堂设立韶山厅、井冈山厅、遵义厅、延安厅，并在四个厅内陈列反映毛泽东革命生涯中具有重大意义的地方的山水画。其中，遵义厅主题绘画作品由[[贵州省|贵州省]]承担，贵州省的张润生、乔任侠、王忠才、杜小牧、马庆中、韩亚明等6名画家为此创作了《遵义颂》、《雄关漫道》、《赤水红军渡口》、《赤水丹霞》等4幅作品，并于2012年交接给毛主席纪念堂管理局<ref>{{Cite web |url =  http://news.ifeng.com/mainland/detail_2012_10/22/18427204_0.shtml|title = 中央办公厅决定在毛主席纪念堂设立井冈山延安等4厅 |author =  |date =2012-10-22  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://hn.rednet.cn/c/2012/05/06/2604605.htm  |title =湖南画家的国画韶山将亮相毛主席纪念堂  |author =  |date =2012-05-06  |publisher =红网  |language =  |accessdate =  |quote =  }}</ref>。

== 雕塑 ==
[[File:MausoleumMaoZedongpic.jpg|thumb]]

毛主席纪念堂有五座雕塑。“这次创作的五座雕塑，毛主席汉白玉雕刻坐像高3.5米。北门两座群雕每座高8米、长15米，人物平均高3.5米左右。南门的两座，每座高6米、长7米，人物高3.5米左右，四座群像共60多个人物。创作时间约半年，放大制作只有2个月。参加创作的有来自全国18个省市的103位雕塑工作者，可以说是一次空前的大会战。”雕塑家[[田金铎|田金铎]]、[[刘逢刚|刘逢刚]]、[[叶毓山|叶毓山]]、[[孙家彬|孙家彬]]、[[盛扬|盛扬]]、[[陈启南|陈启南]]、[[仇志海|仇志海]]、[[王维力|王维力]]、[[苏晖|苏晖]]、[[仲兆乃|仲兆乃]]、[[白澜生|白澜生]]、[[曹承恩|曹承恩]]、[[龙德辉|龙德辉]]、[[马改户|马改户]]、[[郭其祥|郭其祥]]、[[孙纪元|孙纪元]]、[[陈长庚|陈长庚]]、[[铁恩厚|铁恩厚]]等人均参加了雕塑的创作。老一辈雕塑家中，[[刘开渠|刘开渠]]、[[王朝闻|王朝闻]]担任顾问，[[滑田友|滑田友]]、[[曾竹韶|曾竹韶]]、[[王临乙|王临乙]]、[[王合内|王合内]]等指导或参加制作<ref name="ms">让工农兵看了满意——毛主席纪念堂雕塑组部分作者座谈摘要，美术1977年第5期</ref>。

其中，北大厅内有漢白玉座像《毛泽东主席》。中共中央主席华国锋、副主席[[叶剑英|叶剑英]]和其他中央领导亲自审稿，选定了毛主席塑像的方案。华国锋主席指示：“最主要的是要靠艺术家们把毛主席的形象塑造好，让工农兵看了满意。”中央领导指示：“台座不要太高，像不要太大，要让观众看了感到亲切。无产阶级领袖和资产阶级领袖的一个根本区别是无产阶级领袖代表人民、为人民服务，与人民群众保持密切的联系。资产阶级领袖雕像往往被做成了十分高傲，摆出一副威严的架势。表现毛主席就不能那样，要亲切、慈祥，寓伟大于亲切之中。”最后确定了采用坐像而不是立像，坐像“最后确定了毛主席还是像往常一样地在[[中南海|中南海]]，亲切慈祥地坐在沙发上这样一个构思。”整座雕像由许多石雕师傅共同雕刻完成<ref name="ms" />。在创作中，坐像的方案始终有坐像交腿还是不交腿这两种方案。中央选定了交腿的方案。坐像制成后，有人认为交腿像生动、亲切，但和纪念堂庄严肃穆的氛围不够协调。中央研究讨论了这些意见，决定重塑一尊平腿坐像，以替换交腿坐像。但执行该决定很繁琐，最后[[邓小平|邓小平]]拍板决定不用再更换，因此坐像仍维持交腿<ref name=daxiu/>。

广场雕塑群像共有四座，均为泥塑，北面正门的东西两侧各有一座，南面后门的东西两侧各有一座。“前边两座雕塑应是表现丰功伟绩，后边两座表现继续革命，要使群众瞻仰时思绪万千，更加激发起革命、团结战斗的决心。”<ref name="ms" />

北门东侧雕塑表现了民主革命。“表现民主革命的一组，着重体现[[农村包围城市|农村包围城市]]、[[人民战争|人民战争]]、[[枪杆子里面出政权|枪杆子里面出政权]]的思想，主要设计了[[第一次国共内战|井冈星火]]、[[抗日战争|抗日战争]]、[[第二次国共内战|解放全中国]]三段主要内容。”<ref name="ms" />

北门西侧雕塑表现了社会主义革命和建设。“表现社会主义革命和建设的一组，着重体现[[毛泽东思想#主张“继续革命论”|无产阶级专政条件下继续革命]]的思想，分[[社会主义改造|走集体化道路]]、[[工业学大庆|工业学大庆]]·[[农业学大寨|农业学大寨]]、[[无产阶级文化大革命|无产阶级文化大革命]]三段主要内容。”<ref name="ms" />

南门两侧的两座雕塑表现了继承遗志、继续革命。<ref name="ms" />
