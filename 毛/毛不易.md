'''毛不易'''（{{bd|1994年|10月1日|catIdx=W王}}），本名'''王維家'''，出生于[[黑龙江省|黑龙江省]][[齐齐哈尔市|齐齐哈尔市]][[泰来县|泰来县]]，[[中國大陸|中國大陸]][[歌手|歌手]]。毕业于[[杭州师范大学|杭州师范大学]]护理专业。2017年，參加騰訊視頻選秀音樂娛樂節目《[[明日之子|明日之子]]》，獲得全國總決賽冠軍，從而正式進入演藝圈。2018年，推出個人第一張全創作專輯《平凡的一天》，並在同年舉行第一場個人全國巡迴演唱會。官方粉絲名為「暴發戶」，由毛不易本人親自取名，應援色為金色。

==演艺历程==

=== 出道前 ===
*1994年10月1日，出生於[[黑龍江省|黑龍江省]][[齊齊哈爾市|齊齊哈爾市]][[泰來縣|泰來縣]]，自小喜歡唱歌，綽號是“王毛毛”
*初中时，他给自己取名“不易”，意为平凡、不改变
*一直將唱歌當作自己的興趣，並從2012年開始到2017參加《[[明日之子|明日之子]]》前都持續在[[唱吧|唱吧]]上上傳自己的翻唱作品，累計至今約三百餘首
*2013年，進入[[杭州師範大學|杭州師範大學]]就讀，學習[[護理學|護理學]]專業
*在校期間参加了校园十佳歌手的比赛 ，并担任過杭州师范大学理学院“百万音镑”十佳歌手决赛评委
*2016年，到醫院擔任實習護士，期間在杭州租了房號“203”的房間，並開始接觸吉他，嘗試自己寫歌，期間完成三、四十餘首詞曲創作，第一首作品〈出嫁〉是寫給自己要遠嫁的姊姊

=== 2017年 ===
*在朋友推薦下參加音樂選秀節目《[[明日之子|明日之子]]》，海選時因原創歌曲《如果有一天我變得很有錢》而受到大家的關注與喜愛
*在比賽中推出單曲《消愁》廣受歡迎，在騰訊視頻的播放量突破6000萬<ref>{{cite news |language = |author =  |coauthors =  |url = http://ent.qianlong.com/2017/0831/1990553.shtml |title = 8月酷狗单曲畅销榜 毛不易消愁夺冠-千龙网·中国首都网 |publisher =  |pages =  |date =  |accessdate = }}</ref><ref>{{cite news |language = |author =  |coauthors =  |url = http://ent.sina.com.cn/music/zy/2017-09-25/doc-ifymfcih4458586.shtml |title = 《明日之子》毛不易夺冠 杨幂希望以后有机会合作_新浪娱乐_新浪网 |publisher =  |pages =  |date =  |accessdate = }}</ref>，且在推出一周便突破了上億次的播放量，毛不易本人也因此爆紅，被稱為2017年現象級的歌手。
*2017年9月1日，推出首张个人現場专辑《巨星不易工作室 No.1》，收录了包括〈像我这样的人〉、〈消愁〉等在内的7首原创歌曲，该专辑陆续在海外各大音乐平台上架，并登上KKBOX新曲速报歌单推荐榜，他也因此进军海外
*2017年9月23日，在《明日之子》中獲最強廠牌稱號，奪冠出道
*2017年11月11日，推出首支个人錄音室原创单曲〈项羽虞姬〉
*出道不到一年即帶著原創曲目《消愁》登上江蘇衛視的跨年演唱會舞台

=== 2018年 ===
*常駐參與的第一檔音樂綜藝《[[無限歌謠季|無限歌謠季]]》播出
*2018年5月31日，毛不易推出首张个人录音室专辑《平凡的一天》，该专辑由[[李健|李健]]、[[赵兆|赵兆]]担任制作人，上线仅15分钟主打歌单曲销量就突破了50万首，首日銷量破百萬，造成話題<ref>{{cite news |language = |author =  |coauthors =  |url = http://yue.ifeng.com/a/20180601/39847942_0.shtml |title = 毛不易《平凡的一天》首发销量破百万，制作人李健称其无可取代起点高 |publisher =  |pages =  |date =  |accessdate = }}</ref>
*2018年9月22日，出道一周年的前一天，在上海舉辦個人首場演唱會《像我这样的人》
*獲多項音樂獎項，並在華人歌曲音樂盛典中被提名五大獎項，並獲得年度最受歡迎新人獎、年度最佳創作新人獎、年度金曲獎，三個獎項
*2018年11月2日，应邀出任“中欧旅游年欧盟旅游亲善大使”
*2018年11月17日，個人首場演唱會《像我这样的人》於成都大魔方收官，同時於騰訊平台上進行全程直播，線上平台觀看人次達350多萬<ref>{{cite news |language = |author =  |coauthors =  |url = http://v.qq.com/live/p/topic/59196/review.html?from=groupmessage&isappinstalled=0&fbclid=IwAR0AfT73PFg-1s3MrDaCIFVRASJl4kQw0Jpc7m9ASs-2g-wHDFBLRx94aD8#player |title = 毛不易2018像我这样的人巡演成都站 |publisher =  |pages =  |date =  |accessdate = }}</ref>
*2018年12月31日，二度登上江蘇衛視跨年演唱會的舞台，演唱代表作之一《像我这样的人》，以及大熱影視主題曲《不染》，獲熱烈回響
