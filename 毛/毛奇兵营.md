'''毛奇兵营'''（{{lang-de|Moltke-Kasernen}}），又译毛尔提克兵营，旧址位于[[中国|中国]][[山东省|山东省]][[青岛市|青岛市]][[市北区|市北区]][[登州路|登州路]]77号海军大院内，是驻青[[德意志帝国海军|德军]]于1905-1909年间建设的一座[[兵营|兵营]]建筑群，也是德军继[[伊尔蒂斯兵营|伊尔蒂斯兵营]]和[[俾斯麦兵营|俾斯麦兵营]]之后在青岛建设的第三座永久性兵营。整个建筑群现为[[青岛市文物保护单位|青岛市文物保护单位]]，但目前仅余部分营房<ref Name=he>{{Cite news|url=http://qd.sohu.com/20130903/n385730667.shtml|title=俾斯麦兵营和毛奇兵营一豪华一简陋|author=|date=2013-09-03|work=城市信报|accessdate=2017-05-08}}</ref><ref Name=ren>{{Cite book|title=《人文青岛丛书：老楼故事》|last=鲁海|publisher=青岛出版社|year=2010|isbn=9787543600447|location=青岛}}</ref>。

==建筑历史==
===德军兵营===
[[File:Bundesarchiv_Bild_134-C2113,_Tsingtau,_Unteroffizier_der_Marine-Feldbatterie.jpg|thumb]]
[[File:青岛毛奇兵营_03.jpg|thumb]]
1897年11月14日，德军占领青岛。1898年1月26日，一批卫戍部队乘{{link-de|达姆施塔特号蒸汽船|Darmstadt (1890)|达姆施塔特号蒸汽船}}到达青岛，并于同年6月13日组成海军第三营（III. Seebataillon）<ref>{{cite web |url=http://www.marine-infanterie.de/html/3_8.html|title=Aufstellung des III. Seebataillons |accessdate=2017-07-23|work=|publisher=Geschichte der Marine-Infanterie (1675-1919)}}</ref>。海军第三营到达青岛不久，便先后派兵进驻[[胶州|胶州]]、[[即墨|即墨]]、[[高密|高密]]、[[沙子口|沙子口]]等青岛周边市镇要地。[[义和团运动|义和团运动]]期间，第三营除赴京津地区参战部队外，其他部队负责镇压[[山东|山东]]部分地区的义和团、稳定局势。至[[八国联军之役|八国联军之役]]结束数年后，尚有一连驻扎胶州、五连（骑兵连）驻扎高密<ref>{{cite web |url=http://www.tsingtau.org/wp-content/uploads/2015/02/Adressbuch_1902.pdf|title=Adress-Buch des deutschen Kiautschou-Gebiets 1902|accessdate=2017-07-23|date=1902-09-08|work=|publisher=Verlag von Otto Rose, Tsingtau}}</ref>。

1905年，德国与[[清政府|清政府]]签订《[[胶高撤兵善后条款|胶高撤兵善后条款]]》。驻扎在[[胶州|胶州]]、[[高密|高密]]的德军两个[[连|连]]于此后撤回青岛。为安置撤回的部队，胶澳总督府于1905-1906年扩建了[[俾斯麦兵营|俾斯麦兵营]]，并在[[台东镇|台东镇]]西南侧邻近[[青岛啤酒博物馆|啤酒厂]]处修建了一座新的兵营，以安置自高密撤回青岛的第三营五连（5. Kompagnie）。兵营西侧邻近以德国名将[[赫尔穆特·卡尔·贝恩哈特·冯·毛奇|赫尔穆特·冯·毛奇]]（Helmuth von Moltke）命名的毛奇山（Moltke-Berg，今[[贮水山|贮水山]]），因此被命名为毛奇兵营<ref Name=xun>{{cite web |url=http://club.qingdaonews.com/touch/show_57_3824049_1_0.htm|title=寻访德国三大兵营的前世今生|accessdate=2017-07-23|date=2010-11-12|work=|publisher=}}</ref>。

兵营的建设工程由总督府建筑师[[舒备德|舒备德]]（Heinrich Schubart）负责监督指导<ref>{{cite web |url=http://www.tsingtau.org/schubart-j-heinrich-f-1878-1955-dr-ing-architekt-und-ministerialdirigent/|title=Schubart, J. Heinrich F. (1878-1955) – Dr.-Ing., Architekt und Ministerialdirigent|accessdate=2017-07-23|date=2017-05-01|work=Dr. Helga Landau|publisher=}}</ref>。根据1905-1906年度《胶澳租借地发展备忘录》的记载，兵营内已经有一座马厩和一所武器鞍具车间建成<ref Name=Denk1906>{{cite web |url=http://digital.staatsbibliothek-berlin.de/werkansicht?PPN=PPN770823831&PHYSID=PHYS_0052&DMDID=DMDLOG_0001|title=胶澳租借地发展备忘录 1905-1906年度|accessdate=2017-07-23|work=德意志帝国海军部|publisher=柏林国家图书馆}}</ref>。同时，跑马场、一幢宿舍楼和一幢办公楼也已经开工。1906-1907年度，马场、宿舍楼、办公楼、火炮库及车库相继完成，I号营房已开工，II号营房也已着手备料的工作。至1908年10月，守卫室已建成，I号营房内部装修接近完成，II号营房的屋顶也接近完成<ref>{{cite web |url=http://digital.staatsbibliothek-berlin.de/werkansicht?PPN=PPN770823904&PHYSID=PHYS_0061&DMDID=DMDLOG_0001|title=胶澳租借地发展备忘录 1907-1908年度|accessdate=2017-07-23|work=德意志帝国海军部|publisher=柏林国家图书馆}}</ref>。兵营各设施至下一年度已竣工，同时准备继续扩建，建设隔离马厩和第二座车间<ref>{{cite web |url=http://digital.staatsbibliothek-berlin.de/werkansicht?PPN=PPN77082420X&PHYSID=PHYS_0059&DMDID=DMDLOG_0001|title=胶澳租借地发展备忘录 1908-1909年度|accessdate=2017-07-23|work=德意志帝国海军部|publisher=柏林国家图书馆}}</ref>。兵营建成后，除了第五骑兵连外，海军第三营的机枪连和海军野战炮兵连（Feldbatterie）也相继入驻该营<ref Name=xun/><ref Name=mao>{{cite web |url=http://arc.dailyqd.com/2016-04/12/content_326068.htm|title=毛奇兵营：胶澳中学和青岛海军学校都曾在此办学|accessdate=2017-07-23|date=2016-04-12|work=|publisher=青岛日报}}</ref>。

===1914年至1938年===
[[File:青岛战役日英联军军官合影.jpg|thumb]]
1914年8月，[[青岛战役|青岛战役]]爆发。在德军三大兵营中，毛奇兵营与德国守军前线的五大步兵堡垒距离最近，便于战时支援，因此成为兵员中转站及前线士兵的休整场所。10月31日，日军发起总攻，毛奇兵营遭到炮火轰炸，建筑受损严重。11月7日，德军投降，日军于16日完全占领青岛。该兵营于11月20日改名为若鹤兵营（毛奇山亦改名为若鹤山），为日军所使用<ref Name=ren/><ref Name=mao/><ref Name=ti>{{cite web |url=http://media.china.com.cn/cmys/2016-12-27/936468.html|title=毛爾提克兵營：日德最後一戰|accessdate=2017-07-23|date=2016-12-27|work=张帆|publisher=青岛日报}}</ref>。

1922年12月10日，[[北洋政府|北洋政府]]收回青岛主权。[[山东|山东]]各地[[士绅|士绅]]于1924年2月筹资开办的私立胶澳中学（1950年代至今为[[青岛一中|青岛一中]]）将该处作为校舍使用，同年10月，兵营为[[北洋陆军|北洋陆军]]所占用，胶澳中学迁往原德军[[伊尔蒂斯兵营|伊尔蒂斯兵营]]址<ref Name=ren/><ref Name=mao/><ref Name=ti/>。

1933年，沈鸿烈就任青岛市长。同年8月，沈鸿烈将其创办的葫芦岛军校迁到此地，改名[[青岛海军学校|青岛海军学校]]。1937年11月，青岛海军学校因[[抗战|抗战]]内迁<ref Name=ren/><ref Name=mao/><ref Name=ti/>。

[[File:青岛毛奇兵营II号营房北侧.jpg|thumb]]
===1938年后===
1938年1月，日军再次占领青岛并占用兵营。1945年8月，[[日本投降|日本投降]]。兵营曾先后为{{link-en|美国海军陆战队第六师|6th Marine Division (United States)}}与[[中华民国国军|国军]]所使用<ref Name=ren/><ref Name=mao/><ref Name=ti/>。

1949年6月2日，[[解放军|解放军]]进驻青岛，毛奇兵营旧址仍为部队所使用。1950及1960年代，毛奇兵营曾先后成为多所海军院校的校址。1952年6月8日，解放军[[中国人民解放军海军大连舰艇学院#中国人民解放军海军政治学院|海军联合学校]]第二分校自[[南京|南京]]迁至此处，后迁往湛山大路26号。1952年7月成立的海军政治干部学校亦曾设于此处。1953年3月20日，第四海军学校（后改称[[青岛海军潜艇学院|海军潜水艇学校]]）在此处成立，1954年9月迁至[[浮山所|浮山所]]附近的湛流干路23号（今山东路10号）<ref>{{cite web|url=http://qingdao.iqilu.com/qdminsheng/2017/0722/3630678.shtml|title=上世纪50年代，潜水艇学校是这样在青岛从无到有的|date=2017-07-22|work=|publisher=青岛日报 齐鲁网}}</ref><ref>{{cite web|url=http://lib.sdsqw.cn/bin/mse.exe?seachword=%u6F5C%u8247%u5B66%u9662&K=b2&A=12&rec=19&list=&page=&run=13|title=青岛市志·军事志·驻军与军事机构·人民军队·军事院校|author=青岛市史志办|date=|publisher=山东省情网|accessdate=2018-09-25}}</ref>{{注|部分资料称毛奇兵营旧址1960年代曾为海军通讯联络学校的驻地<ref Name=mao/><ref Name=ti/>，但根据其他资料，1957年以原第五海军学校（原海军联合学校）第三分校组建的[[中国人民解放军海军工程大学#中国人民解放军海军电子工程学院|海军通信学校]]设于沙岭庄而非登州路。}}。

该址现仍为海军军产，有时被称为“[[登州路|登州路]]海军大院”或“北航大院”，部分为[[中国人民解放军北部战区海军航空兵|北海舰队航空兵]]后勤部队所使用<ref Name=chai>{{cite web |url=http://club.qingdaonews.com/showAnnounce_2_4910356_1_1.htm|title=毛奇兵营旧址一德式营房建筑近日被拆毁|accessdate=2017-07-23|date=2011-11-01|work=|publisher=}}</ref>。2005年，该建筑被列为[[青岛市文物保护单位|青岛市文物保护单位]]。尽管如此，该建筑群多次被使用单位破坏，多座主要营房已被拆除<ref Name=chai/><ref Name=fa>{{cite web |url=http://club.qingdaonews.com/touch/show_57_3840135_1_0.htm|title=毛奇兵营发现百年老瓦|accessdate=2017-07-23|date=2017-04-01|work=|publisher=}}</ref>。

==建筑特色及保护状况==
[[File:青岛毛奇兵营地图.jpg|thumb]]
初建时的毛奇兵营坐落于毛奇山（Moltke-Berg，今[[贮水山|贮水山]]）东麓、伊尔蒂斯山（Iltis-Berg，今[[太平山_(青岛)|太平山]]）北麓，位于[[台东镇|台东镇]]与[[青岛啤酒博物馆|啤酒厂]]之间，大门朝向西北，临穆勒上尉街（Hauptmann-Müller-Straße，今[[登州路|登州路]]）<ref Name=Denk1906/>。整个建筑群由两座营房、一座礼堂、两座马厩及若干辅助建筑组成，大小建筑共十余栋，建筑群规模相对于[[俾斯麦兵营|俾斯麦兵营]]与[[伊尔蒂斯兵营|伊尔蒂斯兵营]]较小。各建筑的外立面装饰风格与前两者具有极大不同，前两者外立面装饰性较强，毛奇兵营则大大简化外立面装饰，放大实用性。在前两者出现的[[三角楣饰|山花]]、[[塔式建筑|塔楼]]、[[敞廊|敞廊]]、[[壁柱|壁柱]]等装饰元素在此均未出现，而仅使用[[花岗岩|花岗岩]]窗套及砖拱作为装饰<ref Name=xun/>。

相对于其他两座兵营，毛奇兵营目前的保护状况较差。原本十余栋营房如今仅剩七栋，另有一栋残缺不全。其中位于建筑群南端的I号、II号营房外墙面均被贴上[[瓷砖|瓷砖]]，[[花岗岩|花岗岩]]石基被红色油漆覆盖，II号营房的正门也被改建，其南侧已扩建出新的建筑。北侧临登州路的一座马厩被截成两段，作为门房使用。原本作为门房的守卫室被平房包围。临[[登州路|登州路]]的礼堂南侧已被改建，其北半部分被租给临街小餐馆使用<ref Name=chai/><ref Name=fa/>。

礼堂与I号、II号营房之间原本有三栋并排的营房，仅剩中间一栋。其中北侧一栋已于2000年被拆除改建幼儿园，南侧一栋于2011年10月30日上午被完全拆除以建造篮球场。后者被拆除时，毛奇兵营已成为[[青岛市文物保护单位|市级文保单位]]，但年久失修。[[中国人民解放军海军|海军]]负责人对此表示，文物部门将毛奇兵营列入文保单位并未经过军方同意。由于涉及军产，本地主流媒体均不敢报导此事<ref Name=chai/><ref Name=fa/>。

兵营东侧的长营房（可能为马厩）原为单层，现已扩建为双层。西侧长营房（可能为马厩）北端已于1990年代拆除建居民楼，南端已面目全非。2015年9月10日，西侧长营房中段被发现正在拆除，该址现残留营房南端，中段仅剩一小段房屋、一侧墙壁和碎砖瓦<ref Name=fa/>。

==毛奇营房==
[[File:青岛战役日德双方司令官会面处_01.jpg|thumb]]
毛奇营房（Moltke-Baracken）位于[[贮水山|贮水山]]东坡，毛奇兵营及[[青岛啤酒博物馆|啤酒厂]]以西，原址在今[[登州路|登州路]]38号[[青岛医学院|青岛大学医学院]]松山路校区。该建筑群大部分为一些临时平房，自1899年便已存在，但早期没有特定名称，在地图上仅以“临时棚屋”（Baracken）标注<ref>{{cite web |url=http://digital.staatsbibliothek-berlin.de/werkansicht?PPN=PPN770823599&PHYSID=PHYS_0057&DMDID=DMDLOG_0001|title=胶澳租借地发展备忘录 1900-1901年度|accessdate=2017-07-23|date=1901-10|work=德意志帝国海军部|publisher=柏林国家图书馆}}</ref>。后亦冠以“毛奇”之称。该营房曾为海军第三营工兵连驻地。1914年11月10日，胶澳总督府末代总督{{link-de|麦维德|Alfred Meyer-Waldeck}}与日军司令官{{link-ja|神尾光臣|神尾光臣}}中将在此会面<ref Name=mao/><ref Name=ti/><ref>{{Cite|author=三船写真馆|title=日独战役青岛名所写真帖|date=1914}}</ref>。该营房后改名为“若鹤厂舍”，日军临时铁道联队第三大队驻扎于此，同时在此设立临时俘虏收容所。1940年代，该建筑群拆除，原址建设[[青岛日本第三国民学校|日本第三国民学校]]。
