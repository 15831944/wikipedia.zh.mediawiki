'''毛振飛'''（{{bd|1949年}}）是[[中華民國|中華民國]]資深[[勞工運動|勞工運動]]代表人物，多年來參與勞工運動<ref name = "自由時報 01">{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/2304556 |title = 「70歲也該退了！」勞團老將毛振飛 宣布退出工運決策圈 |author =  |date = 2018年1月6日 |publisher = 《[[自由時報|自由時報]]》 |language = Zh-hant |accessdate = 2018年5月27日 |quote =  }}</ref>。他是[[全國關廠工人連線|全國關廠工人連線]]代表，還曾任[[桃園市產業總工會|桃園市產業總工會]]理事長、[[桃園航勤|桃園航勤]]企業工會常務理事、[[團結工聯_(台灣)|團結工聯]]發言人<ref>{{Cite web |url = http://talk.news.pts.org.tw/guest/14047 |title = 毛振飛 |author =  |date =  |publisher = 《[[有話好說|有話好說]]》 |language = Zh-hant |accessdate = 2018年5月27日 |quote =  }}</ref>。

== 早年生活 ==
毛振飛出生於1949年<ref name = "自由時報 01"/>，從小生長在[[基隆市|基隆市]]的深藍家庭，是家中長子，父親和母親都是福建省人<ref name = "壹週刊 01">{{Cite web |url = http://www.nextmag.com.tw/realtimenews/news/372402 |title = 【壹號人物】再幹一場工運 毛振飛 |author =  |date = 2017年12月21日 |publisher = 《[[壹週刊_(台灣)|壹週刊]]》 |language = Zh-hant |accessdate = 2018年5月27日 |quote =  }}</ref>。毛振飛的性格從小喜好打抱不平，遇見不公義的事情也絕不妥協<ref name = "壹週刊 01"/>。他退伍後，便在[[新北市|新北市]][[瑞芳區|瑞芳區]][[大福煤礦|大福煤礦]]礦坑內工作，因此特別能體認基層勞工的辛酸<ref name = "壹週刊 01"/>。由於知道工人是拿生命在換錢，毛振飛在妻子王璧玉的勸說下更換工作，考進[[桃園航勤|桃園航勤]]維修部<ref name = "壹週刊 01"/>。

== 工運抗爭 ==
1987年，因看不慣軍事化管理，毛振飛便與同仁在任職的桃園航勤私下發起組織工會，成為解嚴後臺灣第一個最強悍、參與人數最多的勞工工會<ref name = "壹週刊 01"/>。擔任幹部的毛振飛馬上被桃園航勤高層視為眼中釘，桃園航勤高層轉而想「收編」他，但毛振飛拒絕<ref name = "壹週刊 01"/>。在執政者眼裡，毛振飛亦成為黑名單<ref name = "壹週刊 01"/>。1990年，[[郝柏村|郝柏村]]擔任[[行政院院長|行政院院長]]期間，他便被提報為十大惡寇之一<ref name = "壹週刊 01"/>。

2005年，勞工運動界公認最具公信力的[[桃園市產業總工會|桃園市產業總工會]]剛成立，便由毛振飛擔任理事長<ref name="自由時報 01" /><ref name = "壹週刊 01"/>，是勞工運動代表性領導人物之一<ref name = "風傳媒 01">{{Cite web |url = http://www.storm.mg/article/381972 |title = 「我們也應該退了」工運領導、70歲毛振飛退出決策圈「掰掰了」 |author =  |date = 2018年1月6日 |publisher = [[風傳媒|風傳媒]] |language = Zh-hant |accessdate = 2018年5月27日 |quote =  }}</ref><ref name = "林朝億 01">{{Cite web |url = http://newtalk.tw/news/view/2018-01-06/109571 |title = 關鍵時刻 毛振飛以70歲為由退出桃產總決策圈 |author = 林朝億 |date = 2018年1月6日 |publisher = [[新頭殼|新頭殼]] |language = Zh-hant |accessdate = 2018年5月27日 |quote =  }}</ref>。同年，毛振飛公開幫[[民主進步黨|民主進步黨]][[立法委員|立法委員]][[王拓|王拓]]站台拉票，家族便認為他是叛徒<ref name = "壹週刊 01"/>。爾後，他帶領參與《勞動基準法》、[[全國關廠工人連線抗爭歷史|全國關廠工人連線]]、[[中華民國國道收費員資遣抗爭運動|中華民國國道收費員資遣抗爭]]等街頭勞工運動<ref name = "風傳媒 01"/><ref name = "林朝億 01"/>。

毛振飛對抗政府的激烈行徑，包括有蛋洗[[行政院|行政院]]和[[凱達格蘭大道|凱達格蘭大道]]、發動300多人向[[中華民國總統官邸|中華民國總統官邸]]丟「糞」、參與關廠工人們在[[台北車站|台北車站]]臥軌<ref name = "壹週刊 01"/>。毛振飛每場抗爭都站在前線，也因此背負數件訴訟案、十幾條[[刑責|刑責]]，他也曾為了抗爭被判處[[拘役|拘役]]20天<ref name="自由時報 01" /><ref name = "壹週刊 01"/>，並且入獄服刑<ref name = "林朝億 01"/>。

== 退休活動 ==
2015年，毛振飛退休，但只要勞工團體通知，他仍會持續參與勞工抗爭運動，他還擔任桃園市產業總工會顧問<ref name = "壹週刊 01"/>。[[桃園市市長|桃園市市長]][[鄭文燦|鄭文燦]]曾問他是否能接任[[桃園市政府勞工局|桃園市政府勞工局]]局長，還有小黨提議推派他出來參選，毛振飛通通拒絕這些美意<ref name = "壹週刊 01"/>。2016年，[[桃園市空服員職業工會|桃園市空服員職業工會]]抗議高工時等問題，[[中華航空|中華航空]]空服員集結在總部前[[2016年中華航空空服員罷工事件|發動閃電罷工]]，其幕後推手亦包括毛振飛<ref name = "壹週刊 01"/>。

為抗議[[一例一休|一例一休]]，專程從基隆市前來的毛振飛與其他勞工團體，一大早跑到行政院門口拉布條抗議，並且在[[中華民國勞動部|中華民國勞動部]]前的帳篷下展開長達11天的絕食抗議行動，甚至還專程南下參與高雄市舉辦的抗議大遊行<ref name = "壹週刊 01"/>。而立法院在審查《勞動基準法》修正草案的當晚，他還帶頭想推倒立法院群賢樓門口前的拒馬<ref name = "壹週刊 01"/>。

2017年12月23日的勞工大遊行，毛振飛曾要求群眾在晚上6點時進行衝撞後就地解散，但引爆聲援團體強烈不滿，也有工會成員與堅持撤退的毛振飛爭吵起來<ref name = "風傳媒 01"/>。最後有不少群眾不理勞工團體，演變成街頭流竄抗議<ref name="自由時報 01" />。2018年1月6日，儘管在抗爭前夕的關鍵時刻，由於自己已經虛歲70歲<ref name = "林朝億 01"/>，他在Facebook上貼文宣布將退出勞工運動決策圈<ref name = "風傳媒 01"/>，但往後還是會參加抗爭活動<ref name="自由時報 01" />。

== 思想主張 ==
毛振飛堅持一生挺勞工、熱衷勞工運動，且總是站在[[執政黨|執政黨]]對立面，只替勞工立場發聲<ref name = "壹週刊 01"/>。

== 個人生活 ==
毛振飛身材壯碩，脾氣火爆<ref name = "壹週刊 01"/>。朋友們喜歡幫毛振飛取外號，經常稱他為「老毛」或「毛主席」<ref name = "壹週刊 01"/>。私底下的毛振飛臉上總掛著微笑，平常興趣是一人宅在家觀看電視轉播橄欖球球賽等體育頻道<ref name = "壹週刊 01"/>。毛振飛住在基隆市的國民住宅老公寓<ref name = "壹週刊 01"/>。

毛振飛的妻子為王璧玉，育有女兒毛靜翊和兒子毛裕民<ref name = "壹週刊 01"/>。他與妻子在貨櫃工廠認識，他曾寫上百封情書給妻子展開熱烈追求，兩人最後結婚<ref name = "壹週刊 01"/>。然而毛振飛自覺長年忽略家庭，妻子病逝對他造成極大的打擊，他形容自己像是沒有靈魂的人<ref name = "壹週刊 01"/>。消沉1年，毛振飛[[皈依|皈依]]佛教、尋求宗教的救贖，才走出喪妻之慟<ref name = "壹週刊 01"/>。
