《'''毛毛蟲波羅'''》（{{lang-ja|毛虫のボロ}}）是部由日本[[吉卜力工作室|吉卜力工作室]]製作，動畫導演[[宮崎駿|宮崎駿]]所執導於2018年春季上映的以[[毛毛蟲|毛毛蟲]]為題材之短篇動畫<ref>{{cite web|url=https://www.asahi.com/articles/ASL194VSXL19UTIL02C.html |script-title=ja:宮崎駿監督「毛虫のボロ」３月公開　５年ぶり新作 |work=[[朝日新聞|朝日新聞]] |language=ja |date=2018年1月14日 |accessdate=2018年1月21日}}</ref>。

== 劇情 ==
在草堆裡頭，有隻在黎明前從卵裡孵化出的毛毛蟲，之後它在太陽露出時刻踏上了會遇見天敵或是夥伴的未知旅程<ref>{{cite web|url=https://mainichi.jp/articles/20180110/k00/00m/040/043000c |script-title=ja:「毛虫のボロ」３月公開 |work=[[每日新聞|每日新聞]] |language=ja |date=2018年1月9日 |accessdate=2018年1月21日}}</ref>。

== 過程 ==
先前在宮崎駿決定要拍攝1997年動畫電影《[[魔法公主|魔法公主]]》時，他曾另有準備一套名為《毛毛蟲波羅》的作品企劃書<ref name="MiyazakiBook189Page">{{cite book |script-title=ja:《宮崎駿全書》 |publisher={{lang|ja|フィルムアート社}} |author=葉精二 |location=日本 |isbn=978-484-590-687-1 |date=2006年3月31日 |chapter=〈{{lang|ja|もののけ姫 1997}}〉|page=190頁}}</ref>。當時《毛毛蟲波羅》的劇情構思內容，主要是描述一條毛毛蟲從大街上的樹木，前進至另一棵樹之過程當中所展開的冒險故事<ref>{{cite book |script-title=ja:《「もののけ姫」はそうして生まれた》 |publisher=德間書店 |author=浦谷年良 |location=日本 |isbn=978-419-860-930-6 |date=1998年10月31日 |chapter=〈{{lang|ja|企画構想,遥かな道のり}}〉|page=40頁}}</ref>。

之後因吉卜力影片製作人[[鈴木敏夫|鈴木敏夫]]當時認為那時期已將近六十歲的宮崎駿，往後應該很難再有製作動畫的體力。便希望他挑選有著大量動作場面的《魔法公主》，來作為是他個人最後執導，以及聲勢最浩大的一部作品<ref name="MiyazakiBook189Page"/><ref>{{cite book |title=《順風而起》 |publisher=台灣東販 |author=鈴木敏夫 |location=台灣 |isbn=978-986-331-438-7 |date=2014年8月15日 |chapter=〈全國電影院清一色吉卜力〉 |page=176～177頁}}</ref>。

不過拍攝完《魔法公主》的宮崎駿往後仍繼續製作多部動畫電影，在執導完2013年作品《[[風起|風起]]》時雖曾發表退休聲言表示不再從事長篇動畫製作，但在休憩一陣子後於2015年夏季開始進行想將過去的構想《毛毛蟲波羅》拍攝成短篇動畫<ref>{{cite web|url=https://www.cinematoday.jp/news/N0091692 |script-title=ja:宮崎駿監督の新作、始動！ジブリがスタッフ募集を開始　今度こそ本当に最後の監督作品に… |work=Cinema Today |language=ja |date=2017年5月19日 |accessdate=2017年5月19日}}</ref><ref>{{cite web|url=https://www.cinematoday.jp/news/N0084615 |script-title=ja:宮崎駿監督の新作短編『毛虫のボロ』はCG×手描きに！「怪獣えいがになっちゃいそう |work=Cinema Today |language=ja |date=2016年7月15日 |accessdate=2017年5月19日}}</ref>。

《毛毛蟲波羅》的手稿分鏡以及拍攝情況，之後有在2016年[[NHK|NHK]]電視台播放的紀錄片{{nihongo|《永不休止的宮崎駿》|終わらない人 宮崎駿}}裡透露出來<ref>{{cite web|url=http://www.nikkansports.com/entertainment/news/1815235.html |script-title=ja:鈴木Ｐ、宮崎駿監督の短編は「久々にすごいもの」 |work=[[日刊體育|日刊體育]] |language=ja |date=2017年4月29日 |accessdate=2017年5月19日}}</ref>，

== 製作 ==
{{quote box|quoted=true|width=25%|salign=center|quote={{nihongo|毛毛蟲會看得到空氣中的粒子嗎？咀嚼的葉子對它來說會像是果凍的味道嗎？而寄生蜂應該就會像是現代戰場上飛來飛去的無人戰鬥機吧…就因為以上種種我做了這部電影。|毛虫には空気の粒は見えるのかなぁとか、葉っぱをかじった時はゼリーのような味がするのかなぁとか、狩人蜂は今の戦場で飛び回っている無人攻撃機みたいなものかなぁとか...。それでこんな映画ができてしまいました。}}|source=──宮崎駿對此影片的說明。<ref>{{cite web|url=https://www.cinematoday.jp/news/N0099263 |script-title=ja:宮崎駿監督の新作短編、タモリが声と音を一人で表現！「タモリさんに感謝」 |work=Cinema Today |language=ja |date=2018年3月14日 |accessdate=2018年3月24日}}</ref>}}
劇情內容的啟發，來自於宮崎駿小學時期所學的植物[[光合作用|光合作用]]現象，當時他對於光合作用感到興趣，並幻想著如果是以蟲子的角度是否能目睹到該過程變化<ref name="Huffington">{{cite web|url=https://www.huffingtonpost.jp/2018/03/15/boro_a_23386239/ |script-title=ja:宮崎駿監督の新作『毛虫のボロ』は、タモリの“密室芸”とアニメが融合した新世界すぎてワクワクした |work=[[哈芬登郵報|哈芬登郵報]] |author=古川慧 |language=ja |date=2018年3月15日 |accessdate=2018年3月24日}}</ref>。另外他小時看過的由[[花野原芳明|花野原芳明]]創作的1948年漫畫{{nihongo|《小蜘蛛的冒險》|コグモの冒険}}，當中所收錄一則名叫{{nihongo|《飛天的惡魔》|空飛ぶあくま}}的短篇裡頭，擬人化的綠毛毛蟲角色在身體感到痛苦時；腹部突然爆裂冒出一堆寄居在他體內寄生蜂的一幕，也讓當時的宮崎駿帶來深刻印象<ref name="Huffington"/>。

作畫上是採用電腦的[[CG|CG]]繪圖技術，為宮崎駿首次想嘗試的挑戰<ref>{{cite web |url=https://www.iwate-np.co.jp/article/oricon/2107518 |script-title=ja:宮崎駿監督最新短編アニメ『毛虫のボロ』完成　声と音はタモリが担当 |work=岩手日報 |language=ja |date=2018年3月16日 |accessdate=2018年3月24日 |title={title} |archive-url=https://web.archive.org/web/20180324224337/https://www.iwate-np.co.jp/article/oricon/2107518 |archive-date=2018-03-24 |dead-url=yes }}</ref>。電腦繪圖團隊則是以曾在2015年動畫《[[花與愛麗絲殺人事件|花與愛麗絲殺人事件]]》裡擔任過CG指導的[[櫻木優平|櫻木優平]]為中心<ref>{{cite web|url=https://prtimes.jp/main/html/rd/p/000000010.000023394.html |script-title=ja:Hulu×クラフター　櫻木優平監督　スマートCGアニメーション 発表 |work=PR Times |language=ja |date=2017年3月25日 |accessdate=2018年3月24日}}</ref>。

內容裡波羅所發出的聲音以及一些背景音效是由藝人[[塔摩利|塔摩利]]所表現，他之前就有在以各種生活周遭音效為主題的吉卜力短篇動畫《[[尋找棲所|尋找棲所]]》裡演出<ref name="Mainichi">{{cite web |url=https://mainichi.jp/articles/20180314/orc/00m/200/062000c |script-title=ja:宮崎駿監督最新短編アニメ『毛虫のボロ』完成　声と音はタモリが担当 |work=[[每日新聞|每日新聞]] |language=ja |date=2018年3月14日 |accessdate=2018年3月24日 |title={title} |archive-url=https://web.archive.org/web/20180325045027/https://mainichi.jp/articles/20180314/orc/00m/200/062000c |archive-date=2018-03-25 |dead-url=yes }}</ref>。影片片尾曲則是與宮崎駿長期合作的音樂家[[久石讓|久石讓]]提供<ref>{{cite web|url=https://natalie.mu/comic/news/273596 |script-title=ja:「毛虫のボロ」宮崎駿がボロの声や効果音を担当したタモリに「ありがとう」 |work=[[Natalie_(網站)|Natalie]] |language=ja |date=2018年3月14日 |accessdate=2018年3月24日}}</ref>。

== 公開 ==
作品原先預計於2017年7月期間上映<ref>{{cite web|url=http://news.livedoor.com/article/detail/13001303/ |script-title=ja:ジブリ鈴木P、宮崎駿の新作は7月公開 長編2019年公開は否定 |work=[[活力門|活力門]] |language=ja |date=2017年4月29日 |accessdate=2017年5月19日}}</ref>，之後延遲到2018年3月21日公開，並且僅在[[吉卜力美術館|吉卜力美術館]]的土星座電影館裡播放<ref>{{cite web|url=https://www.oita-press.co.jp/1009000000/2018/01/09/ORI2103776 |script-title=ja:宮崎駿監督、最新作『毛虫のボロ』　ジブリ美術館で3・21上映開始 |work=大分合同新聞 |language=ja |date=2018年1月9日 |accessdate=2018年1月21日}}</ref>。上映前的試映會是在公開前一週裡3月14日舉行<ref name="Mainichi"/>。
