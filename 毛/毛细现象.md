'''毛細現象'''（又稱'''毛細管作用'''）是指[[液體|液體]]在細管狀物體內側，由液體與物體之間的[[黏附|附著力]]和因[[內聚力|內聚力]]而產生的[[表面張力|表面張力]]組合而成，令液體在不需施加外力的情況下，流向細管狀物體的現象，該現象甚至令液體克服[[地心引力|地心引力]]而上升。[[植物|植物]][[根|根部]]吸收的水分能夠經由[[莖|莖]]內[[維管束|維管束]]上升，即是毛細現象最常見的例子。當[[液體|液體]]和[[固體|固體]]（管壁）之間的[[黏附|附著力]]大於液體本身[[內聚力|內聚力]]時，就會產生毛細現象。[[液體|液體]]在[[垂直|垂直]]的細管中時液面呈凹或凸狀、以及多孔材質物體能吸收[[液體|液體]]皆為此現象所造成的影響。

== 水的毛細現象 ==
[[File:Capillarity.svg|left]]與[[附著力|附著力]]的差異，[[水|水]]在毛細管中，中央較四周凹下；[[汞|汞]]在毛細管中，中央較四周凸起。]]
毛細管常被用來說明毛細現象，當[[垂直|垂直]]的細[[玻璃|玻璃]]管底部置於[[液體|液體]]中（例如[[水|水]]）時，管壁對[[水|水]]的[[附著力|附著力]]便會使液面四周稍比中央高出一些；直到[[液體|液體]][[表面張力|表面張力]]已經無法克服其[[重量|重量]]時，才會停止繼續上升。在毛細管中，液柱[[重量|重量]]與管徑的[[平方|平方]]成[[正比|正比]]，但是[[液體|液體]]與管壁的接觸[[面積|面積]]只與管徑成正比；這使得較窄的毛細管吸水會比較寬的毛細管來得高。例如，一根管徑0.5毫米的玻璃細管，理論上能夠將水抬升2.8厘米，但實際觀察時其高度會略低些。

== 汞的毛細現象 ==
在某些[[液體|液體]]與[[固體|固體]]的組合中，與毛細管吸水的狀況略為不同，例如細[[玻璃|玻璃]]管與[[水銀|水銀]]（汞），汞柱本身的[[原子|原子]][[內聚力|內聚力]]大於汞柱與管壁之間的[[附著力|附著力]]，故汞柱液面中央會稍比四周凸起，這和毛細管吸[[水|水]]的狀況恰為相反。

== 毛細現象應用 ==
[[File:TLC_black_ink.jpg|200px]]利用了毛細現象。]]
[[File:Capillary_papertowel.PNG|180px]]
* 在[[水文學|水文學]]中，毛細現象常用來解釋[[土壤|土壤]]對[[水|水]]的吸引力；在[[土壤|土壤]]中，[[水|水]]分會由較[[潮溼|潮溼]]處移動到[[乾燥|乾燥]]處，即是毛細現象所致。
* 毛細現象也是[[眼淚|眼淚]]能夠自[[眼睛|眼睛]]不斷流出的必要因素。
* 現今某些材質的[[運動|運動]]衣料，會透過毛細現象吸[[汗|汗]]。
* [[化學家|化學家]]常利用毛細現象來進行[[薄板層析|薄板層析]]（薄板色譜分析）。
* [[自来水笔|自来水笔]]的笔管也是通过毛细现象维持笔头湿润
* [[紙巾|紙巾]]即是透過毛細現象吸收[[液體|液體]]，其充滿細孔的材質使得[[液體|液體]]能夠被紙巾吸收。
* [[海綿|海綿]]有非常多的細小孔洞（相當於毛細管），這使得[[海綿|海綿]]能夠吸收大量的[[液體|液體]]。
* [[蠟燭|蠟燭]]芯將蠟引到火附近。

== 公式 ==

液柱上升高度是：
: <math>h={{2 \gamma \cos{\theta}}\over{\rho g r}}</math>

此處：

:''γ'' = [[表面張力係數|表面張力係數]]
:''θ'' = 接觸角
:''ρ'' = 液體[[密度|密度]]
:''g'' = [[重力加速度|重力加速度]]
:''r'' = 細管半徑

當''θ''>90度，這表示彎液面為凸面；同時''h''<0，表示流體在毛細管下降，即[[汞|汞]]在玻璃管的情況。

對於在[[海平面|海平面]]上，裝了水的玻璃管，

:''γ'' = 0.0728 J m<sup>-2</sup>
:''θ'' = 20°
:''ρ'' = 1000 kg m<sup>-3</sup>
:''g'' = 9.8 m s<sup>-2</sup>

液柱高度為：

: <math>h\approx {{1.4 \times 10^{-5}}\over r} \ \mbox{m}</math> .

根據此方程式，理論上在1米寬的管中，水可以上升0. 000 014米（因此極不容易被察覺）；另外在1厘米寬的管中，水可以上升0.14厘米；而在半徑0.1毫米的毛細管中，水可以上升140毫米。

=== 推導 ===

* 方法一：考慮表面張力的力

: <math>2\pi r\gamma\cos\theta = \rho gh \pi r^2 \;</math>.

其中
:表面張力引起的力為<math>F = 2\pi r\gamma\;</math>，而其垂直向上的部分為<math>F \cdot \cos\theta\;</math>；
:升起的液體部分的體積為<math>V = \pi r^2 h\;</math>，其[[重量|重量]]（[[重力|重力]]的作用力）為<math>\rho Vg \;</math>；。

* 方法二：考慮流體內非常接近彎液面的點A和非常接近毛細管外表面的點B的壓力，按[[伯努利定律|伯努利定律]]有：

: <math>P_0 - \frac{2\gamma}{R} + \rho g h = P_0</math>

其中，''R''為彎液面的半徑，<math>R = \frac{r}{\cos\theta}</math>；<math>P_0=P_A=P_B</math>則為大氣壓力。
