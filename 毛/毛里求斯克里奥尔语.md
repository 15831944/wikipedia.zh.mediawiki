'''毛里裘斯克里奧爾語'''（毛里裘斯克里奧爾語：'''Morisien'''或'''Kreol morisien'''；{{lang-en|'''Mauritian Creole'''}}；{{lang-fr|'''créole mauricien'''}}）是一種於[[毛里裘斯|毛里裘斯]]使用的[[克里奧爾語|克里奧爾語言]]。這種語言的詞彙大多源自[[法語|法語]]，另有小部份來自[[英語|英語]]和島上使用的其他[[非洲|非洲]]及[[亞洲|亞洲]]語言。

==社會語言地位==
毛里裘斯克里奧爾語是毛里裘斯的[[通用語|通用語]]。縱然英語是毛里裘斯的政府機構及教學的[[官方語言|官方語言]]，而毛里裘斯人亦普遍學習標準法語，毛里裘斯克里奧爾語仍然是島上最常用的[[口語|口語]]。由於毛里裘斯人口在種族和語言上有著極大的差異，不同族群之間的交流非常頻繁，毛里裘斯克里奧爾語在這種不同族群之間的非正式溝通中極為常用。毛里裘斯人大部份祖籍[[印度|印度]]，但他們所操的語言亦各式各樣，包括[[泰米爾語|泰米爾語]]、[[旁遮普語|旁遮普語]]、[[泰盧固語|泰盧固語]]、[[印地語|印地語]]、[[古吉拉特語|古吉拉特語]]，再加上英語、法語和克里奧爾語。同樣情況亦發生在當地的少數[[歐洲|歐洲]]和[[華人|華人]]人口，他們分別操法語、英語、[[廣東話|廣東話]]、[[普通話|普通話]]和[[客家話|客家話]]。另一方面，克里奧爾人佔島上人口三分之一，他們是[[大英帝國|大英帝國]]於1830年代廢止[[奴隸制度|奴隸制度]]前由非洲和亞洲帶來的[[奴隸|奴隸]]的後裔。克里奧爾人最常以克里奧爾語作為他們的第一語言，雖然他們都很常講法語或英語。

==分類==
由於毛里裘斯克里奧爾語的大多數詞彙來源自法語，所以語言學家將其分類為「以法語為基本的克里奧爾語言」。語言學家亦同意[[塞舌爾|塞舌爾]]、[[查戈斯群島|查戈斯群島]]和[[羅德里格斯島|羅德里格斯島]]的克里奧爾語跟毛里裘斯的克里奧爾語有密切關係，但它與其他同以法語為基本的克里奧爾語言之間的關係則仍有爭議。[[留尼汪|留尼汪島]]與毛里裘斯相鄰，卻比毛里裘斯更早受法語殖民者統治。語言學家羅伯特·舒登遜（{{lang|fr|Robert Chaudenson}}）認為毛里裘斯克里奧爾語與留尼汪克里奧爾語同樣有密切連繫。諸如羅伯特·A·帕彭（{{lang|fr|Robert A. Papen}}）、菲利普·貝克（{{lang|en|Philip Baker}}）及克里斯·科爾訥（{{lang|en|Chris Corne}}）等的另一些語言學家則認為留尼汪克里奧爾語對毛里裘斯克里奧爾語的影響微不足道，並指兩者之間只不過比其他同以法語為基本的克里奧爾語言更相似而已。

==歷史==
首先發現毛里裘斯島的是[[葡萄牙|葡萄牙人]]，但他們沒有在島上殖民。毛里裘斯克里奧爾語詞彙中的少量[[葡萄牙語|葡萄牙語]]元素，實際上是來自歐洲[[航海|航海]][[術語|術語]]中的葡萄牙語元素（例如「通用語」（{{lang|en|Lingua Franca}}）一詞），或者由來自與葡萄牙人經商頻繁地區的非裔或亞裔奴隸傳入。

[[荷蘭|荷蘭人]]於1638年起統治毛里裘斯，至1710年便完全撤離，期間還不夠一百年，只留下少量逃跑了的奴隸在島上，根本對毛里裘斯克里奧爾語沒有帶來什麼重大的影響。

五年後即1715年起，法國人到來並開始在島上殖民。法國殖民者將他們在留尼汪和[[西印度群島|西印度群島]]實行的以奴隸工人為基本的莊園經濟，複製到毛里裘斯去。1730年開始，奴隸成為島上人口的多數，至1777年更是人口的85%。這些奴隸來自[[西非|西非]]、[[東非|東非]]、[[馬達加斯加|馬達加斯加]]和印度等各地，令他們之間出現了所謂「語言分裂」（{{lang|en|linguistic fragmentation}}）情況，無法用其中任何一方的語言來溝通，法語在這情況下便逐漸成為了不同種族奴隸之間的通用語。可是由於以法語為母語的人（即殖民者）在島上的人口比例很低，而且跟奴隸們保持距離，極少溝通，再加上奴隸獲得的正規教育又是少之又少，這種種原因導致奴隸們所用的「法語」必然跟殖民者所用的標準法語相去甚遠，成為現代毛里裘斯克里奧爾語的雛形。早於1773年便已有歷史文獻提及島上奴隸所使用的克里奧爾語言。

英國人於[[拿破崙|拿破崙]]時代接收了毛里裘斯，但往島上殖民者不多，而那時毛里裘斯克里奧爾語亦已大致發展成熟。1830年代英國人廢除奴隸制度後，大量毛里裘斯克里奧爾奴隸離開莊園，莊園主人便轉向從[[印度|印度]]輸入契約勞工來代替。雖然印度人此後便成為島上的最大族群，但由於印度族群內部本身亦存在語言分裂的情況，加上他們與操英語或法語的白人族群之間的疏離，使這些印度工人也選擇使用毛里裘斯克里奧爾語作為他們主要的通用語。英語和法語長久以來在毛里裘斯都享有較高的社會地位，並成為政府機關、商業、教育及大眾傳媒的主要語言，但在眾多非正式的場合中，毛里裘斯克里奧爾語卻確立了其地位。

== 音韻與拼字==
毛里裘斯克里奧爾語的[[音韻學|音韻系統]]跟法語十分類近。不過，法語中的 {{lang|fr|j}} 和 {{lang|fr|g}} 音在毛里裘斯克里奧爾語被唸成 {{lang|fr|z}} 一般，而法語中的圓唇元音 {{lang|fr|u}} 和 {{lang|fr|eu}} 會分別被唸成 {{lang|fr|i}} 或 {{lang|fr|u}} （但不圓唇，類似法語中的 {{lang|fr|ou}}）和 {{lang|fr|e}} 或 {{lang|fr|o}}。

雖然毛里裘斯克里奧爾語言至今還沒有正式的標準拼字系統，但仍然出版了數本單語或雙語的[[字典|字典]]，分別由菲利普·貝克、''Ledikasyon pu travayer''（{{lang|en|Education for Workers}}，工人教育）小組和阿爾諾·卡普朗（{{lang|fr|Arnaud Capooran}}）等編纂。以毛里裘斯克里奧爾語言編寫的出版物越來越多，非正式的標準拼字系統亦在慢慢醞釀中。這套系統主要沿襲自法語拼字系統，但在拼字中去除了不發音的字母，並盡量減少同一發音的不同拼寫方式。

2005年，[[毛里裘斯大學|毛里裘斯大學]]的Vinesh Hookoomsing教授出版了一本名為《''Grafi Larmoni''》的報告，試圖協調各個不同的拼寫系統。

== 詞彙 ==
縱然大部份毛里裘斯克里奧爾語[[單字|詞語]]的來源是法語，但他們的用法卻不一定與法語中的一樣。比如說，來自法語的[[定冠詞|定冠詞]]''{{lang|fr|le}}''或''{{lang|fr|la}}''，在毛里裘斯克里奧爾語中常常會與其指明的[[名詞|名詞]]融合為一個詞。法語中的''{{lang|fr|le rat}}''（老鼠）在毛里裘斯克里奧爾語中變為'''''lera'''''，法語中的''{{lang|fr|le temps}}''（時間）則變成'''''letan'''''。這情況偶爾亦會發生在部份[[形容詞|形容詞]]和[[前置詞|前置詞]]，例如法語的''{{lang|fr|femme}}''（女人）和''{{lang|fr|riz}}''（米飯），分別加上形容詞''{{lang|fr|bonne}}''（好）和前置詞''{{lang|fr|du}}''（屬於...的，類似英文中的''{{lang|en|of}}''）後，在毛里裘斯克里奧爾語中便成了'''''bolfam'''''（來自法語''{{lang|fr|bonne femme}}''）和'''''duri'''''（來自法語''{{lang|fr|du riz}}''）。有些詞語在融合後甚至連語意也有所不同，例如毛里裘斯克里奧爾語的'''''gayh'''''（有、擁有）來自法語''{{lang|fr|gagner}}''（贏得、勝出）。

毛里裘斯克里奧爾語裡亦有些詞語[[外來語|借用]]自[[非洲|非洲]][[黑奴|黑奴]]們的語言：[[馬拉加什人|馬拉加什人]]為毛里裘斯從[[馬拉加斯語|馬拉加斯語]]帶來了'''''lapang'''''（馬拉加斯語''ampango''，指黏在壺底的飯）、'''''lafus'''''（馬拉加斯語''hafotsa''，一種樹木）、'''''zahtak'''''（馬拉加斯語''antaka''，一種植物）等詞語。以上例子與借用自法語的詞語一樣，原來的詞語與法語定冠詞''{{lang|fr|le}}''、''{{lang|fr|la}}''或''{{lang|fr|les}}''融合為一個字。擁有東非語源的詞語包括'''''makutu'''''（來自{{en-link|馬庫阿語|Makua language}}的''makhwatta''，指膿瘡）、'''''matak'''''（來自[[斯瓦希里語|斯瓦希里語]]及{{en-link|馬孔德語|Makonde language}}的''matako''，指臀部）。

== 文法 ==
毛里裘斯克里奧爾語名詞[[眾數_(語言學)|眾數]]形式不變形，只能依靠上文下理判斷名詞是[[單數_(語言學)|單數]]或眾數。如果必需明確標示眾數式，一般會在名詞前加上[[助詞|助詞]]'''''ban'''''（來自法語''{{lang|fr|bande}}''）。法語中的''{{lang|fr|un}}''和''{{lang|fr|une}}''對應毛里裘斯克里奧爾語的'''''en'''''，但使用規則有少許相異之處。另外，毛里裘斯克里奧爾語的冠詞'''''la'''''置於被其修飾的名詞之後，比較法語：''{{lang|fr|un rat}}''、''{{lang|fr|le rat}}''、''{{lang|fr|les rats}}''和毛里裘斯克里奧爾語：'''''en lera'''''、'''''lera-la'''''、'''''ban-lera'''''。

毛里裘斯克里奧爾語中，每個[[代詞|代詞]]只有一個形式，不論[[主格|主格]]、[[賓格|賓格]]或[[屬格|屬格]]，亦不論[[性_(語法)|性別]]，因此，毛里裘斯克里奧爾語的代詞 '''''li''''' 根據情況可分別譯成英語的''{{lang|en|he}}''、''{{lang|en|she}}''、''{{lang|en|it}}''、''{{lang|en|him}}''、''{{lang|en|his}}''、''{{lang|en|her}}''或''{{lang|en|hers}}''。

與名詞一樣，毛里裘斯克里奧爾語的[[動詞|動詞]]同樣不會因應[[時態|時態]]或[[人稱|人稱]]而變形。反而以所依附的名詞或代名詞來判斷是誰行動，並在動詞之前獨立或結合使用一些[[小品詞|小品詞]]來指示時態：'''''ti'''''（來自法語''{{lang|fr|etais}}''，「是」）指[[時態|過去時態]]；'''''pe'''''（來自''{{lang|fr|apres}}''，「之後」）指[[時態|進行時態]]；'''''(f)in'''''（來自''{{lang|fr|fin}}''，「結束」）指[[時態|完成時態]]；'''''a'''''（來自''{{lang|fr|va}}''，「將」）指[[時態|未來時態]]。例句：'''''li fin gayh'''''意即「他／她／它（過去）有」，又可略寫為'''''li n gayh'''''，並可看成一個詞那樣發音。
