== 花絮 ==
该片是“纪念毛岸英牺牲60周年”的纪念剧。<ref>{{cite web |url= http://culture.people.com.cn/GB/87423/13472490.html|title= 电视剧《毛岸英》：一曲青春激情的颂歌|accessdate=2010年12月14日 |publisher=人民网 |work= |language =zh-cn}}</ref>

该片讲述了毛岸英和毛泽东的父子关系、毛岸英和妻子刘思齐的爱情故事以及在朝鲜战争中牺牲的原因。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2010-10-20/10113119279.shtml |title= 《毛岸英》登陆央一 英雄史诗永远激励后人|accessdate= 2010年10月20日|publisher=新浪网 |work= |language =zh-cn}}</ref>

该剧拍摄时，毛岸英的妻子刘思齐尚在世，已经改嫁。<ref>{{cite web|url=http://culture.gmw.cn/2010-11/22/content_1402722.htm|title=电视剧《毛岸英》荡气回肠令人感奋|accessdate=2010-11-22|publisher=光明网|work=|language=zh-cn}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

该剧以“感情戏”为主旋律。<ref>{{cite web |url=http://ent.qq.com/a/20101031/000038.htm |title=《毛岸英》以情动人 主旋律电视剧打感情牌 |accessdate=2010年10月31日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

该剧在[[北京市|北京市]][[中南海|中南海]]拍摄，首次破例允许实景拍摄了3天，后在[[湖南省|湖南省]][[韶山市|韶山市]]、[[俄罗斯|俄罗斯]][[莫斯科|莫斯科]]等地拍摄。<ref>{{cite web |url= http://news.sina.com.cn/c/2010-10-15/181521284207.shtml|title=电视剧毛岸英将开播 中南海破例允许实景拍摄 |accessdate= 2010年10月15日|publisher=新浪网 |work= |language =zh-cn}}</ref>
