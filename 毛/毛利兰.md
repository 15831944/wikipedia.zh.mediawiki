'''毛利兰'''（日語：{{lang|ja|毛利 蘭}}/{{lang|ja|もうり らん}}，Mōri Ran），是[[日本|日本]][[漫画|漫画]]和[[动画|动画]]作品《[[名侦探柯南|名侦探柯南]]》中的女主角。

命名來自[[法国|法国]]小说家'''[[莫理斯·盧布朗|莫理斯·盧布朗]]'''（'''Mauri'''ce Leb'''Lan'''c，《[[亚森·罗宾|亚森·羅賓]]》作者）。

== 人物 ==
東京都米花市帝丹高中二年B班的學生，[[空手道|空手道]]社的女主將，是[[江户川柯南|工藤新一]]的[[青梅竹馬|青梅竹馬]]兼女友，[[鈴木園子|鈴木園子]]及[[遠山和葉|遠山和葉]]的好友；父親是名偵探[[毛利小五郎|毛利小五郎]]，母親是知名[[律師|律師]][[妃英理|妃英理]]；父母在她7歲時分居，和經營毛利偵探事務所的父親同居。

1年前在紐約救下銀髮殺人魔（[[黑暗組織#黑暗組織的重要幹部|苦艾酒]]易容），苦艾酒對小蘭心存感激並稱她為「Angel」<ref group="參" name="TV 286"/>。

從小代替母親承擔所有家事，同時在事務所擔任助理的工作。對於分居中的父母，常暗中撮合雙方感情，雖然到目前為止都因為父親的花心導致一直無法成功，但兩人感情確實有進展，只是很慢。

在新一失蹤後，小蘭說服父親收留化名為[[江户川柯南|江户川柯南]]的新一。她曾數次察覺柯南驚人的推理能力，怀疑兩者為同一人，在最懷疑、最極端時還對柯南稱「新一」<ref group="參">漫畫第3卷，TV 7《每月一件禮物威脅事件》</ref>，但每次都在柯南或其他人的幫助下成功瞞過，至今仍不知道「江戶川柯南=工藤新一」。

運動神經發達，擅長[[空手道|空手道]]，因崇拜前田聰（6年前日本空手道冠軍）而開始練習<ref group="參">漫畫第8卷，TV 68《暗夜男爵殺人事件（事件篇）》</ref>。曾獲得「全國高中空手道[[關東地方|關東]]大賽（個人賽）」冠軍。在電影《漆黑的追蹤者》與愛爾蘭對峙時，躲過秒速350米的子彈。

曾和[[江户川柯南|新一]]及[[鈴木園子|園子]]在其他動畫中客串<ref>分別是《[[犬夜叉|犬夜叉]]》第128集和《[[青山剛昌短篇集|青山剛昌短篇集]]》。</ref>。

== 形象 ==
性格溫柔善良、樂觀隨和、陽光親切、真誠熱忱、天真浪漫、堅強積極、強悍倔強、樂於助人，和每個人都能相處得十分融洽。很喜歡小朋友，深受少年偵探團成員的仰慕。 

情感豐富，擁有強烈的責任感和正義感，容易為別人的事落淚，卻常把自己的悲傷隱藏在笑容之下<ref group="參">漫画第33卷，TV 269《犯罪的紀念品（前篇）》</ref>，曾說過「勇氣這個詞，是形容人挺身而出的正義字眼，不能用來當作殺人的理由」<ref group="參">漫画第31卷，TV 247《網中謎（後篇）》</ref>。遇上困境時從不輕易示弱，更不輕言放棄，能爆發出驚人的決斷力和行動力，並全力保護他人。

喜歡可愛的小動物，愛好文學和電影，喜歡的顏色是紅色<ref group="參">M1《[[名偵探柯南：引爆摩天樓|引爆摩天樓]]》</ref>。害怕打雷<ref group="參">漫畫第5卷，TV34-35《山莊繃帶怪人之殺人事件》</ref>、妖魔鬼怪等可怕事物。

第六感非常準，擁有與生俱來的好運，抽獎幾乎百發百中<ref group="參">TV 174《二十年後的殺機 交響樂號連續殺人事件》、243-244《毛利小五郎鬧雙胞》、589-590《最糟糕的生日》</ref>，玩麻將和撲克牌也經常抽到好牌<ref group="參">TV 75《金融公司社長殺人事件》、174《二十年後的殺機 交響樂號連續殺人事件》、219《名偵探齊聚一堂 工藤新一vs怪盜基德》</ref>。

善於整理家務、料理、護理<ref group="參">TV 170《黑暗中的死角（前篇）》、511《推理對決！新一vs沖矢昴》</ref>、手工藝（編織<ref group="參">漫畫第24卷，TV 171《黑暗中的死角（後篇）》</ref>、陶藝<ref group="參">漫畫第30卷，TV 228-229《暗藏殺機的陶藝教室》</ref>等），會彈鋼琴<ref>TV 11中彈奏《[[第14钢琴奏鸣曲_(贝多芬)|月光奏鸣曲]]》；M12《[[名偵探柯南：戰慄的樂譜|顫慄的樂譜]]》中為合唱團伴奏。</ref>，具有一定程度的樂理知識<ref>TV 194-195《意味深長的音樂盒》中能根據音符很快寫出樂譜；M12《[[名偵探柯南：戰慄的樂譜|顫慄的樂譜]]》中能迅速分辨出新一拉小提琴。</ref>，對於人文歷史也相當熟悉<ref>TV 558-561《死亡之館、赤壁》中講出中國的[[三国|三國]]歷史。</ref>，但方向感很差，獨自行動時常迷路。

[[吉田步美|步美]]說，[[灰原哀|灰原哀]]曾向柯南提到小蘭是「順產型」（做事情有計畫）的女生<ref group="參">漫画第31卷，TV 246《網中謎（前篇）》</ref>。

== 情感 ==
因為新一在[[紐約|紐約]]救下銀髮殺人魔後說「我不明白殺人要有什麼理由，但人救人是不需要理由的」，真正開始對其產生感情<ref group="參" name="TV 286"/>。

和青梅竹馬的新一是兩情相悅，但一直不確定他對自己的心意，也只向柯南一人坦白過喜歡新一<ref group="參">漫画第1卷，TV 2《董事長千金綁架事件》</ref>。後來新一在[[倫敦|倫敦]]對自己表白後，卻遲遲不予以正面回覆<ref group="參">漫畫第72卷，TV 621《福爾摩斯默示錄（0 is Start）》</ref>，直到[[京都|京都]][[修學旅行|修學旅行]]期間，才親吻新一臉頰回應自己的心意並正式發出交往的請求<ref group="參">漫畫第95卷，TV 928《紅色的修學旅行（戀紅篇）》</ref>。

== 影響 ==
人物最大特徵是右側前額有個角狀的髮型（約30°～50°），常被網友吐槽為'''角蘭'''，'''獨角蘭'''或凶器等等<ref group="參">{{Cite web |url=http://comic.qq.com/a/20150717/030802.htm |title=日媒分析毛利兰的角是什么时候出现的 |work=騰訊動漫 |date=[2015年07月17日]}}</ref>。初次登場及真人飾演時並無此特徵，但隨著連載而日趨明顯，約於漫畫第10期定型。在2016年的訪談中，作者青山剛昌表示：「我想的是給小蘭畫個標誌性的髮型，像這樣的記號，即使站得很遠畫得很小也能被認出來。就算在人群中也可以，僅此而已。抱歉，就跟[[阿童木|阿童木]]的尖頭一樣。」
