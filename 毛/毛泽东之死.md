'''毛泽东'''1976年9月9日逝于北京，享壽82岁（虚龄83岁）。

1976年9月9日下午4时，[[中央人民广播电台|中央人民广播电台]]以《[[告全党全军全国各族人民书|告全党全军全国各族人民书]]》的形式公布消息称：“1976年9月9日0时10分，中国共产党中央委员会主席、[[中国共产党中央军事委员会主席|中国共产党中央军事委员会主席]]、[[中国人民政治协商会议全国委员会|中国人民政治协商会议全国委员会]]名誉主席毛泽东在[[北京|北京]]逝世，享年83岁”<ref>[http://dangshi.people.com.cn/GB/165617/166495/10002553.html 毛泽东逝世--中国共产党新闻--人民网]</ref>。毛泽东是继时任[[中华人民共和国国务院总理|中华人民共和国国务院总理]][[周恩来|周恩来]]和时任[[全国人大常委会委员长|全国人大常委会委员长]][[朱德|朱德]]后，第三位于1976年去世的[[党和国家领导人|党和国家主要领导人]]<ref>[http://media.people.com.cn/GB/40606/4706912.html 毛泽东主席逝世公开广播的背后]</ref>。

== 晚年病情 ==
1971年9月13日[[九一三事件|林彪事件]]之后，毛泽东的身体状况急剧衰退，在1972年初曾发生严重[[休克|休克]]<ref>{{cite book |title = 《毛泽东传：1949-1976》 |editor = [[中共中央文献研究室|中共中央文献研究室]]（金冲及 主编） |publisher = 中央文献出版社 |year = 2003 |page = 1615}}</ref>。此后毛泽东又发生[[白内障|白内障]]，双目[[失明|失明]]，直至1975年[[手术|手术]]后恢复一半视力<ref>{{cite book |title = 《红镜头中的毛泽东》 |author = 顾保孜、杜修贤 |publisher = 辽宁人民出版社 |year = 2004}}</ref>。

1976年9月9日0时10分，毛泽东在北京[[中南海|中南海]]202別墅內病逝。新华社发布通讯《中国共产党中央委员会、中华人民共和国全国人民代表大会常务委员会、中华人民共和国国务院、中国共产党中央军事委员会公告》。[[中共中央|中共中央]]至今尚未公布毛泽东因何病而死。毛泽东醫疗小组主要成员、[[中国人民解放军第三〇五医院|中国人民解放军第三〇五医院]]院長[[李志绥|李志绥]]在[[毛泽东私人医生回忆录|回忆录]]中称，兩位醫生診斷毛澤東患有[[肌萎縮性脊髓側索硬化症|肌萎縮性脊髓側索硬化症]]（俗稱「漸凍人症」），而不是[[帕金森氏症|帕金森氏症]]。<ref name="LiZhi-Sui">{{cite book

== 追悼大会 ==
1976年9月18日下午3时，“伟大的领袖和导师毛泽东主席追悼大会”在[[天安门广场|天安门广场]]举行，[[中共中央副主席|中共中央副主席]][[王洪文|王洪文]]主持大会，[[中共中央副主席|中共中央第一副主席]]、[[国务院总理|国务院总理]][[华国锋|华国锋]]致悼词。当时天安门广场上站了前来悼念毛泽东的民众，人数估计多达十多万，估计北京有幾十万民众参加，全国部分地區举行相关的悼念活动。

== 台湾反应 ==
[[中國國民黨|中國國民黨]]黨營報紙《[[中央日報|中央日報]]》在9月10日的[[頭版|頭版]]以標題「禍國殃民·百死莫贖 毛匪澤東斃命」刊登毛澤東的死訊，並在副標題稱「死訊宣佈時奪權布置顯未完成 為匪偽留下無法收拾的爛攤子」。<ref>「禍國殃民·百死莫贖 毛匪澤東斃命」，《中央日報》頭版，1976年9月10日</ref>
{{quote|
〔中央社臺北九日電〕為禍中國六十年的匪酋[[毛澤東|毛澤東]]，於今天凌晨結束了他醜惡的一生。

共匪「[[新華社|新華社]]」於今天下午四時播出匪黨「中央委員會」、匪偽「人大常務委員會」、匪偽「國務院」與匪黨「中央軍事委員會」的一封所謂「告全黨全軍全國全族人民書」中，宣布毛酋因病情惡化，醫治無效，於今天零時十分在北平死亡。

共匪這一文件，匪黨未提及毛酋曾留下任何遺言，也未提及毛酋死後對繼承人問題，作了什麼安排。

「新華社」在發表這一文件中透露，毛酋不但與「疾病鬥爭」，而且在奄奄一息的時候，仍不忘掌握匪黨匪軍的權力，「一直戰鬥到生命的最後一息」，更顯示毛酋未能完成奪權的布置，而在權力鬥爭中的煎熬中黯然死去，替匪偽頭目留下了一個無法收拾的爛攤子。

毛酋顯然是在四月五日[[四五天安门事件|天安門事件]]後病情加劇，他最後一次露面是於五月二十七日會見巴基斯坦總理[[佐勒菲卡尔·阿里·布托|布托]]，六月十五日匪黨「中央」宣布毛酋不再會見外賓。由於天安門事件引發了大陸人民的反毛反共怒火，毛酋雖在一怒之下整肅了[[鄧小平|鄧小平]]，但是卻無法平抑爆炸性的不滿情緒，天安門事件成了毛酋的催命符。七月底唐山發生[[唐山大地震|大地震]]，整個大陸流傳著毛酋已臨末日的傳說，這一次大地震敲響了毛酋的喪鐘。

毛匪澤東畢生迷信權力，強調「階級鬥爭」，製造仇恨，煽動暴亂，為中國有史以來最暴虐的[[極權主義|極權主義]]者，竊據大陸以來屠殺了七千萬的大陸人民，他自以為製造矛盾、發動階級鬥爭可以鞏固他的極權統治，但終於在無情的鬥爭下喪生。

他的死，正如大陸人民在天安門喊出的口號：「[[秦始皇|秦始皇]]的時代已經一去不復返了。」<ref>《中央日報》，臺北：中央日報社，民國六十五年九月十日</ref>

中華民國[[國民大會|國民大會]]也于9月10日發表聲明，呼籲大陸同胞於毛匪斃命、匪黨內部權力鬥爭加劇之際，加強反共抗暴鬥爭，早日摧毀反人性、反人權的匪偽政權。[[中華民國體育協進會|中華民國體育協進會]]理事長[[黎玉璽|黎玉璽]]與四十五個單項運動協會則呼籲大陸運動員，把握出國訪問及比賽的機會，投奔自由。<ref>{{Cite news|url=|title=國代呼籲大陸同胞 及時奮起摧毀暴政|author=|date=1976-09-11|work=聯合報|accessdate=}}</ref>

而在和大陸隔海相望的金門，[[中華民國國軍|中華民國國軍]]加強[[心戰|空飄廣播及喊話]]，號召大陸軍民同胞，在此毛匪斃命的變亂中，乘時奮起，摧燬暴政。

台灣宗教界對毛澤東之死普遍感到高興，[[樞機主教|樞機主教]][[于斌|于斌]]說，「毛澤東死亡，可以說是我們反攻復國的契機」

由於香港[[電視廣播有限公司|無線電視]]製播《毛澤東特輯》，被認為是「通共媚匪」，台灣三家電視台[[中國電視公司|中視]]、[[中華電視公司|華視]]、[[臺灣電視公司|台視]]發表共同聲明，抵制無線電視的節目、影片，停止各方面合作，<ref>{{Cite news|url=|title=港無線電視台媚匪 國內三台決予抵制|author=中央社|date=1976-10-01|work=聯合報|accessdate=}}</ref>這一行為一直到80年代《[[楚留香_(電視劇)|楚留香]]》在台灣掀起熱潮，才宣告結束。

== 国际反应==
在消息公布后的15分钟内，世界主要的媒体，包括[[路透社|路透社]]、[[美联社|美联社]]、[[法新社|法新社]]等都报道毛泽东的死讯。

*{{USA}}：美國總統[[杰拉尔德·福特|福特]]第一時間向北京致唁電，他在唁電中表示，「一九七五年十二月我訪問北京時曾有幸會見了毛主席。我們的談話促進了[[中美关系|美中關係]]沿著我們兩國早先設想的路線發展。請讓我現在象那時一樣申明，美國决心在[[中美三个联合公报|上海公報]]的基礎上完成我們關係的正常化。這將是讚揚他的遠見的恰當的管道，也將有益於我們兩國人民。」<ref>{{Cite news|url=|title=美国总统福特的唁电|author=|date=1976-09-14|work=人民日報|accessdate=}}</ref>此後他又在記者會上發表了聲明，他說：「美國人不會忘記，正是在毛主席的領導下，中國同美國一起採取行動結束了一代人的敵對情緒，在我們兩國關係上開創了一個新的和比較積極的時代。」但美國民間也有不同的聲音，[[华尔街日报|華爾街日報]]評論指出，「毛澤東不會為中國人所懷念，而會使他們所懷恨。」，[[纽约时报|紐約時報]]則稱呼毛澤東為「中國的[[约瑟夫·维萨里奥诺维奇·斯大林|史達林]]，瘋狂廢除舊秩序的制度」。
*{{USSR}}：由於[[中苏交恶|中蘇交惡]]，蘇聯方面只在官方報紙的角落簡短提及毛澤東去世。
*{{DPRK-PRK}}：1976年9月9日，[[金日成|金日成]]向中共發出唁電，唁電中說：「在朝鮮人民為反對美帝國主義武裝侵略者而進行激烈鬥爭的[[朝鲜战争|祖國解放戰爭]]時期，毛澤東同志粉碎了國內外敵人的一切阻撓，掀起[[朝鲜战争|抗美援朝]]運動，用鮮血援助了我國人民的正義鬥爭。」<ref>{{Cite news|url=|title=金日成同志的唁电|author=金日成|date=1976-09-11|work=人民日報|accessdate=}}</ref>朝鮮政府還將9月10日至18日定為全國哀悼期，下半旗誌哀，并舉行了大規模的悼念活動。
*{{ROU}}：羅馬尼亞總統[[尼古拉·齐奥塞斯库|齊奧塞斯庫]]在唁電中說，「毛澤東是羅馬尼亞人民親密的朋友。他努力發展我們兩黨、兩國和兩國人民之間的關係，以利於兩國人民、利於社會主義與和平事業。  」<ref>{{Cite news|url=|title=齐奥塞斯库同志的唁电|author=齊奧塞斯庫|date=1976-09-11|work=人民日報|accessdate=}}</ref>
*{{PAK}}：9月9日，[[巴基斯坦总统|巴基斯坦總統]]喬德里發表聲明：「作為改變了占人類四分之一的人民生活的中國革命之父，毛主席是有史以來最傑出的領袖之一。作為政治家和思想家，他在人類史册上留下了不可磨滅的印記。他的逝世不僅對中國人民而且對世界各國人民都是一個巨大的損失。」[[巴基斯坦总理|巴基斯坦總理]]布托則表示，「毛澤東主席一貫關心巴基斯坦的幸福和進步，這使他永遠活在我們心中。巴基斯坦人民同兄弟的中國人民一起哀悼這位偉人的逝世。」<ref>{{Cite news|url=|title=巴基斯坦總統、總理發來的唁電|author=|date=1976-09-11|work=人民日報|accessdate=}}</ref>
*{{YUG}}：南斯拉夫總統[[约瑟普·布罗兹·铁托|鐵托]]評價：「毛澤東主席的逝世使中國人民失去了最傑出的領袖。如果沒有他，現代中國是難以想像的。」<ref>{{Cite news|url=|title=南斯拉夫总统铁托的唁电|author=约瑟普·布罗兹·铁托|date=1976-09-11|work=人民日報|accessdate=}}</ref>
*{{VNM}}：[[越南共产党|越南勞動黨]]以最高規格發表唁電，同時在唁電中說「越南人民永遠銘記敬愛的毛主席所說的“七億中國人民是越南人民的堅強後盾，遼闊的中國領土是越南人民的可靠後方”這一充滿情誼的話語。我們越南人民無比深切地感謝毛澤東主席、[[中国共产党|中國共產黨]]、中國政府和兄弟的中國人民給予我們革命事業的巨大的、寶貴的支持和援助。」<ref>{{Cite news|url=|title=越南劳动党中央、国会常务委员会和政府会议的唁电|author=|date=1976-09-12|work=人民日報|accessdate=}}</ref>
*{{AUS}}：[[澳大利亚总督|澳洲總督]]约翰·罗伯特·柯尔爵士說：「我知道中华民族的每一个成员，今天都象一个家庭失去最敬爱的家长一样」，總理[[约翰·马尔科姆·弗雷泽|约翰·马尔科姆·弗雷泽]]評價：「在他的指引和鼓舞下，中国重新赢得了民族尊严和国际威望」<ref>{{Cite news|url=|title=澳大利亚总理弗雷泽的唁电|author=|date=1976-09-12|work=人民日報|accessdate=}}</ref>
*{{JPN}}：日本首相[[三木武夫|三木武夫]]等人到中國大使館親自誌哀，并發表聲明稱「现在，日中两国关系正在发展的时候，失去了这位伟大的领导人，衷心感到惋惜」，而新華社也在日本官方釋出極大善意之後重申北京支持日本主張收復被蘇俄佔領的[[北方四島|北方四島]]的要求。
*{{GBR}}：英國女王[[伊丽莎白二世|伊利莎白二世]]親致唁電，首相[[詹姆斯·卡拉漢|詹姆斯·卡拉汉]]說，「他的影響遠遠超出了中國的疆界，無疑他將作為世界聞名的偉大政治家而被人們所緬懷。」<ref>{{Cite news|url=|title=英国首相卡拉汉的唁电|author=詹姆斯·卡拉汉|date=1976-09-12|work=人民日報|accessdate=}}</ref>
*{{FRA}}：法國總統[[瓦勒里·季斯卡·德斯坦|德斯坦]]表示，「他使中國擺脫了昔日的屈辱，使中國恢復了歷史賦予她的中心地位。法國不會忘記，正是毛澤東主席和深切敬仰他的[[夏爾·戴高樂|戴高樂將軍]]一起促成了我們兩國互相接近。」<ref>{{Cite news|url=|title=法国总统德斯坦的唁电|author=|date=1976-09-12|work=人民日報|accessdate=}}</ref>9月11日，巴黎有市民舉行遊行集會，哀悼毛澤東的去世。

== 身后安排 ==
虽然毛泽东生前鼓励[[火化|火化]]、并在火化同意书上签名，但他的遗体仍被[[中共中央政治局|中共中央政治局]]决定做[[屍體防腐处理|屍體防腐处理]]后永久保留。初时暂厝[[人民大会堂|人民大会堂]]，完成防腐处理后被安放在[[水晶棺|水晶棺]]内，放置在新建成的[[毛主席纪念堂|毛主席纪念堂]]供后世瞻仰，由[[中共中央办公厅|中共中央办公厅]][[毛主席纪念堂管理局|毛主席纪念堂管理局]]维护和运营。后来，参观毛主席纪念堂成为特色的[[思想政治教育|思想政治教育]]和[[红色旅游|红色旅游]]项目。<ref>[http://news.china.com/zh_cn/history/all/11025807/20050909/12641245.html 毛泽东逝世后遗体是如何进行保护的]</ref>
