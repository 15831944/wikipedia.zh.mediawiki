'''毛泽东的早年生涯'''是指从[[毛泽东|毛泽东]]1893年12月26日出生到1921年加入[[中国共产党|中国共产党]]之前这段时间里的生活，主要是他在学校的求学生涯。

== 家世 ==
{{See also|毛贻昌|文素勤}}
[[File:_Wen_Qimei.jpg|thumb]]
[[File:_Mao_Yichang.jpg|thumb]]

毛泽东的父亲叫[[毛贻昌|毛贻昌]]，字顺生，号良弼。祖籍[[浙江|浙江]][[衢州|衢州]][[江山市|江山市]]石门镇清漾村<ref>2007年浙江省江山市委宣传部公布：毛泽东的祖籍地是浙江省江山市石门镇清漾村。（[http://www.ycwb.com/xkb//2007-03/23/content_1424613.htm 毛泽东祖籍源自浙江江山] {{Wayback|url=http://www.ycwb.com/xkb//2007-03/23/content_1424613.htm |date=20170830012327 }}）</ref><ref>《毛泽东韶山家谱》记载“毛氏祖居三衢”，“三衢”就是今浙江衢州地区</ref>。毛泽东的20代祖先[[毛太华|毛太华]]曾客居云南。2002年版《韶山毛氏五修族谱》载：
<blockquote>我族始祖太华公，元[[至正|至正]]时人也，避乱由江西[[吉州|吉州]]龙城迁云南之[[澜沧卫|澜沧卫]]，娶王氏，生八子。明洪武十三年庚申，以军功官入楚省，携长子清一、四子清四与之偕行。解组时居[[湘乡|湘乡]]北门外绯紫桥。十余年后，清一、清四两公卜居[[湘潭|湘潭]]三十九都，今之七都七甲[[韶山|韶山]]家焉。</blockquote>
即毛太华（毛泽东为其第20代后人）于1380年（[[明朝|明朝]][[洪武|洪武]]十三年庚申年），携长子毛清一、幼子毛清四移居湖南[[湘乡县|湘乡县]]，后来他的两个儿子移居到湘潭县韶山冲。{{r|bird|page=1}} 

元[[至正|至正]]年间，江西吉安的毛太华是如何远涉几千里到达云南[[北胜州|北胜州]]（今[[永胜县|永胜县]]）？元至正十八年（1358）四月，反元起义领袖[[徐寿辉|徐寿辉]]的[[陈友谅|陈友谅]]大军攻占江西吉安。同年，徐寿辉的[[明玉珍|明玉珍]]大军攻克嘉定(今四川[[乐山|乐山]])，控制了四川大部。徐寿辉从湖广等地向四川增兵10多万。毛太华在其中，从家乡江西入川当兵。至正二十三年（1363）春，明玉珍分兵3路南征云南。其中，中路军[[万胜|万胜]]带3万兵攻取昆明后，又经楚雄进兵大理，元大理总管[[段功|段功]]率领部落头人武装击败万胜的反元军。万胜军又遭两场败绩，经[[毕节|毕节]]退往重庆。毛太华即在这次远征中流散于云南民间。

元至正年间，云南[[北胜州|北胜州]]土司高姓，为元大理总管[[段功|段功]]的姻亲。毛太华在北胜州娶白蛮女子王氏为妻，生子8人，一住16年。毛太华内迁湖南湘乡是在洪武十三年，次年即洪武十四年，朱元璋派出[[傅友德|傅友德]]、[[蓝玉|蓝玉]]、[[沐英|沐英]]率大军远征云南，消灭元朝驻昆明的梁王政权。此前，元梁王谋害大理总管段功，段功儿子段宝继承大理政权，并通过其舅家北胜州的高氏土司与朱元璋的四川行省取得联系夹击昆明的元梁王。毛太华即以汉人信使身份，匆促间仅带了二子从北胜州取道四川回到内地，被安置于湖南湘乡县。

据《明太祖实录》，云南的'''澜沧卫'''，是[[岷王_(明朝第十一任)|岷王]][[朱楩|朱楩]]（是[[朱鎔基|朱鎔基]]的祖先）的护卫军，于洪武二十四年（1391年）组建编成，驻守于南京附近的[[南直隶|南直隶]]，人员以湖广籍为主。原计划珉王要封在甘肃，所以该部队称为西河中护卫。洪武二十八年（1395年）夏四月，“岷府西河中护卫并仪卫司官军校尉往云南镇守”，为朱楩就藩昆明打前哨。同年九月初三西河中护卫改称云南中护卫；同月廿四，云南中护卫调往[[北胜州|北胜州]]（即今云南丽江市[[永胜县|永胜县]]），置'''澜沧卫'''。次年（洪武二十九年），正式成立''澜沧卫军民指挥使司''，“寓兵于农，屯民实边”，[[改土归流|改土归流]]，当地的高姓土司下台，而从[[南直隶|南直隶]]调来的这支部队连带家属上万人，成了北胜州的人口主体民族。这时年龄不足三十岁的毛清二，以其有汉人父亲的身份背景，被驻防明军吸收，成了澜沧卫军户，后升为明军军官，成为毛太华留在当地的孤儿寡母的顶梁柱。今永胜县[[程海乡|程海乡]]凤羽（行政村）毛家湾（自然村），以及[[河口乡|河口乡]]毛家村、[[金官镇|金官镇]]西湖毛家湾、[[翁彭乡|翁彭乡]]毛家村、[[清河乡|清河乡]]毛家井村、[[片角乡|片角乡]]毛家洼村等地，至今生活着毛清二的后代3000多人。康熙七年立的永胜毛清二的墓碑：“始祖宦游北赕越数载，卜居凤山之北”，即指毛清二在北赕（即北胜州）为宦数年，并开创了凤山毛家湾的祖村的局面。现今永胜坝区乡镇依然延用着当年屯戍时以第一任长官姓氏及军马营伍之设命名的村庄、地名。“明故'''一世祖'''旌表忠义、武德将军毛公讳用、字清二老大人之墓。”至今，永胜县毛氏族人奉毛清二为一世祖。自毛清二起，一至七世祖族祖都冠以武德将军爵位，八至十一世族祖是指挥千户或前所千总。

明初西南边陲统一，全国局面安定后。云南永胜的毛清二与湖南湘乡的父亲毛太华、兄弟毛清一、毛清四互相知道对方下落。所以云南、湖南两地的历代毛氏族谱都清楚记载了另一支系的存在。 

毛贻昌原本是一个贫苦的农民，读过两年书，10岁时，毛贻昌和[[文七妹|文七妹]]订婚，14岁时，毛贻昌的母亲去世，15岁时，毛贻昌和18岁的文七妹结婚。{{r|bird|page=1-2}} 16岁时加入[[湘军|湘军]]，原因是负债过多，后回家务农和作小生意，毛贻昌累积了一点钱，买回了自己的土地，早年，毛贻昌和文七妹生了二个儿子，但是不幸夭折，后来生了两个女儿也不幸夭折。{{r|ant|page=46}} 毛泽东出生时，毛贻昌已经有了15亩地，成了中农，每年可以收60担谷，当时他们家一共5口人，毛贻昌、文七妹、祖父[[毛恩普|毛恩普]]、弟弟[[毛泽民|毛泽民]]，一家五口人一年吃35担谷，可以剩余25担谷，毛贻昌利用剩余的稻谷，又买了7亩地，这时候他爷爷去世了，而家里变成了富农，每年可以收获84担谷，而此时也增加了一个弟弟[[毛泽覃|毛泽覃]]，每年剩余稻谷49担。{{r|ant|page=46}} 毛贻昌早年不信[[佛教|佛教]]，有一天在路上遇到一只[[老虎|老虎]]，老虎没有咬他，反而逃走了，毛贻昌认为是神佛保佑，才开始信奉佛教。{{r|ant|page=50}}{{r|bull|page=4-5}} 毛贻昌是一个中农的时候就开始贩卖稻谷，逐渐成为富农，并且家里雇了[[长工|长工]]，6岁时，毛泽东就开始和家里人一起在地里干活，并且开了“毛义顺堂”的股票，他克勤克俭、精打细算，[[粮食|粮食]]和[[生猪|生猪]]生意非常红火。{{r|ant|page=46}}{{r|bull|page=1-2}}

毛泽东的母亲叫文其美，湘乡县[[文芝仪|文芝仪]]的女儿，真名文素勤，俗称文七妹，不识字，文七妹心地善良，慷慨厚道，虔诚信佛，吃观音斋，[[初一|初一]][[十五|十五]]带着毛泽东到[[凤凰山|凤凰山]]去拜佛烧香，经常接济穷人，施舍财物，前提是毛贻昌不在场。{{r|ant|page=48}}{{r|bull|page=4-5}} 毛泽东的外婆贺老太太生育两子三女，长子文玉瑞，次子文玉钦，大女嫁给钟家，二女嫁给王家，满女七妹嫁给韶山土地冲毛家毛贻昌。{{r|bird|page=3}}

1893年12月26日（[[清朝|清朝]][[光绪|光绪]]十九年），毛泽东诞生在[[湖南省|湖南省]][[长沙府|长沙府]][[湘潭县|湘潭县]][[韶山冲|韶山冲]]的一个[[农民|农民]]家，[[乳名|乳名]]“石三伢子”。

== 文正莹私塾 ==
8岁时，毛泽东开始在韶山冲附件的[[私塾|私塾]]小学堂读书，一直读到13岁，先后有6个老师教过他，也正是在私塾的学习，培养了他对[[中国历史|中国历史]]和[[中国文化|中国文化]]的兴趣。除了读书时间，大部分时间他都在地里干活，学堂教的是[[孔子|孔子]]的《[[论语|论语]]》和《[[四书|四书]]》。{{r|ant|page=47}} 毛泽东的启蒙老师[[文正莹|文正莹]]是毛泽东的亲二舅，毛泽东称他叫“八舅父”，他的工作是开私塾，教育族中子弟识字读书。文正莹教会了毛泽东《[[三字经|三字经]]》、《[[百家姓|百家姓]]》、《[[千字文|千字文]]》和《[[六言杂诗|六言杂诗]]》，并且经常借[[报刊|报刊]][[图书|图书]]给毛泽东。{{r|ant|page=47}}{{r|bird|page=3-6}} 毛泽东熟读经书，但是并不喜欢它们，他喜欢阅读中国古代小说，虽然它们当时是清朝的[[禁书|禁书]]，《[[精忠传|精忠传]]》、《[[水浒传|水浒传]]》、《[[隋唐演义|隋唐演义]]》、《[[三国演义|三国演义]]》和《[[西游记|西游记]]》这些禁书都被他熟读，并且和同学反复讨论。{{r|ant|page=47}}{{r|bird|page=3-6}}{{r|bull|page=2-3}} 后来毛泽东回家乡私塾读书时，送给他一本《康熙字典》。<ref name=" deer" >{{cite web |url=http://news.xinhuanet.com/xhfk/2011-01/18/c_12991974.htm |title=影响毛泽东的七位老师 |accessdate= 2011年1月18日|publisher= 新华网|work= |language =zh-cn}}</ref>

== 南岸私塾 ==
1902年，毛贻昌把毛泽东接回韶山冲的南岸私塾读书，老师是[[邹春培|邹春培]]，绰号叫“省先生”，由于毛泽东调皮捣蛋，一日，邹春培出对联考他，他说“濯足”，毛泽东回答“修身”；邹春培接着说“[[牛皮菜|牛皮菜]]”，毛泽东回答“[[马齿苋|马屎苋]]”。邹春培后来跑到毛贻昌那里说：“润之了不得啊，他的才学比我高，我已经教不了啦。”{{r|bird|page=6-8}}<ref name=" deer" />

1903年，毛泽东10岁，这年，他因为逃学不敢回家，又害怕父亲责打他，便往湘潭县县城跑，跑了三天，才跑出8里路，就被家人找到带回了家。{{r|ant|page=47}}{{r|bull|page=3}}

== 毛润生私塾 ==
1904年，毛泽东在韶山[[关公桥|关公桥]]私塾[[毛润生|毛润生]]那儿读了半年私塾。{{r|ant|page=47}}

== 毛宇居私塾 ==
[[File:_Mao_Yuju.jpg|thumb]]
13岁时，在一次家宴上，毛贻昌点名道姓骂儿子毛泽东懒惰无用，两人当着客人吵了起来，随后毛泽东离家出走，毛泽东跑到一个池塘边，威胁父亲跳下去寻死，毛贻昌提出毛泽东磕头认错，最后，毛泽东跪一条腿磕头认错告终。{{r|ant|page=47}}{{r|bull|page=1}} 同年，毛泽东离开了私塾，回到家中，白天在地里帮忙干活，晚上帮父亲记账，闲暇时间如饥似渴的阅读能够找到的一切书籍，父亲责备他懒惰，他则引经据典“父慈子孝”等来反驳他。{{r|ant|page=47}} 晚上，毛泽东用被子罩住窗户读书，这时候，毛泽东熟读了一本叫《[[盛世危言|盛世危言]]》{{efn|1= 《盛世危言》，清朝[[郑观应|郑观应]]在1894年写作的书籍，讲述“富国强兵”之道，内容涵盖政治、经济、军事、外交、文化等。}} 的书籍。{{r|ant|page=47}}{{r|bull|page=7-10}} 这本书介绍了[[西方|西方]]先进的[[科学|科学]]，[[铁路|铁路]]、[[电话|电话]]、[[电报|电报]]、[[轮船|轮船]]等，激起了毛泽东的求知欲望，他和父亲提起恢复学业的事情，遭到父亲的反对，为此，他离家出走，在井湾里私塾[[毛宇居|毛宇居]]家读了半年书。{{r|ant|page=47}}<ref name="fish " >{{cite web |url= http://book.sina.com.cn/1070500318_laoyingji/excerpt/sz/2008-04-25/1431234357.shtml|title=毛泽东之父毛顺生让贫困毛家发财致富 |accessdate=2008年4月25日 |publisher=新浪网 |work= |language =zh-cn}}</ref> 一次，毛宇居叫学生在私塾温习功课，但毛泽东带着书本到山上摘[[板栗|毛栗子]]去了，回来时，毛宇居大怒，指着一口天井叫毛泽东作诗，毛泽东脱口而出：

== 东山小学 ==
1907年到1908年，14岁时，毛泽东在家里务农和读书自学，老师[[李漱清|李漱清]]送给他一本瓜分中国的小册子《支那瓜分之命运》，书中提到“呜呼！中国其将亡矣！”{{r|ant|page=49-51}}<ref name=" deer" /> 书中介绍了[[大日本帝国|大日本帝国]]占领[[朝鲜|朝鲜]]、[[台湾|台湾]]的经过，也介绍了[[大英帝国|大英帝国]]占领[[缅甸|缅甸]]、[[法国|法国]]占领[[越南|越南]]的史实，这本书激发了毛泽东的救国意识。{{r|ant|page=49-52}} 同年，毛贻昌送毛泽东到湘潭的一家米店去做学徒，但是毛泽东提出要上学，学堂在湘乡县，名叫“湘乡[[东山学校|东山小学]]”，并且学堂里毛泽东的一个表哥[[王季范|王季范]]也在那读书，起初毛贻昌反对此事，经过王季范的劝说和八舅捐助学费才同意。到湘乡东山小学读书前，毛泽东留给父亲毛贻昌一首改自[[西乡隆盛|西乡隆盛]]《锵东游题壁二首》的《改西乡隆盛诗赠父亲》{{efn|1=西乡隆盛《锵东游题壁二首》，其一：二十七年云水身，又寻师友向三津。儿乌反哺应无日，忍别北堂垂白亲。其二：男儿立志出乡关，学若无成不复还。埋骨何须坟墓地，人间到处有青山。}} 。<ref name="fish " />

== 湘乡驻省中学堂 ==
毛泽东到长沙的[[湘乡市第一中学|湘乡驻省中学堂]]读了半年书，因为时局混乱、战争频繁而退学，{{r|ant|page=55-57}}  6月份，学校举行作文比赛，毛泽东的《商鞅徙木立信论》独占鳌头，国文教员[[柳潜|柳潜]]点评说：“实切社会立论，目光如炬，落墨大方，恰似报笔，而义法亦入古。逆折而入，笔力挺拔。历观生作，练成一色文字，自是伟大之器，再加功候，吾不知其所至。力能扛鼎，积理宏富。有法律知识，具哲理思想，借题发挥，纯以唱叹之笔出之，是为压题法，至推论[[商鞅|商君]]之法为从来未有之大政策，言之凿凿，绝无浮烟涨墨绕其笔端，是有功于社会文字。”<ref name="rat " >{{cite web |url=http://view.news.qq.com/a/20091101/000020.htm |title=现存毛泽东最早文稿：《商鞅徙木立信论》 |accessdate=2009年11月1日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

== 长沙参军 ==
在长沙，毛泽东读到《[[民力报|民力报]]》，使他认识到[[黄兴|黄兴]]、[[孙中山|孙中山]]、[[黄花岗七十二烈士|黄花岗七十二烈士]]、[[同盟会|同盟会]]等革命志士，他写了一篇文章，贴在学校的墙上，他在文章中提到，“把孙中山从大日本帝国请回来当新政府的[[总统|总统]]，康有为当[[总理|国务总理]]，梁启超当[[外交部长|外交部长]]！”{{r|ant|page=55-57}}{{r|bull|page=14-16}} 当时，[[川汉铁路|川汉铁路]]正在修筑，因为反对外资投入，学生们剪掉辫子，要求[[君主立宪制|立宪]]。{{r|ant|page=55-57}} [[黎元洪|黎元洪]]在[[武汉|武汉]]起义后，革命党人在学校演讲，号召学生行动起来，建立[[中华民国|民国]]，毛泽东听后，从同学们手中筹集了一点钱，打算到汉口去参军。{{r|ant|page=55-57}} 当时起义军和长沙清兵正在交战，守城的士兵没有放行。{{r|ant|page=55-57}} 第二天，长沙成立了都督府，[[哥老会|哥老会]]的[[焦达峰|焦达峰]]和[[陈作新|陈作新]]分别出任[[都督|都督]]和副都督。{{r|ant|page=55-57}} 不久之后，[[谭延闿|谭延闿]]带兵发动叛乱，杀死了焦达峰和陈作新。{{r|ant|page=55-57}} 此时，一支学生军引起了毛泽东的注意，他加入了这个军队，可拿每月7元的军饷，每月伙食花费他2元，还有买水的一些钱，剩余的钱他购买了报纸《[[湘江日报|湘江日报]]》。{{r|ant|page=55-57}} 北京的[[袁世凯|袁世凯]]和南方的孙中山达成议和后，湘军原本打算攻打袁世凯的计划落空，毛泽东认为革命已经结束，便退出了军队，他当了半年的兵。{{r|ant|page=55-57}} 

== 图书馆自修 ==
退役之后，毛泽东「曾經受到报章廣告的吸引而投身警校，可是在他註冊入學之後，卻又看到另一則[[肥皂|肥皂]]制造学校的廣告而改變主意。」<ref name="首都经济信息报19950905">《[[首都经济信息报|首都经济信息报]]》，[[北京|北京]]，1995年9月5日。</ref>因為該学校除了“不收学费、供食宿还有津贴”，因此毛花了一個[[銀元|銀元]]的註冊費便獲准入學。<ref name="首都经济信息报19950905" />{{r|ant|page=59-61}}{{r|bull|page=17}}「後來又受到朋友的慫恿」<ref name="首都经济信息报19950904">《首都经济信息报》，北京，1995年9月4日。</ref>，说国家处于经济战争之中，需要[[经济学家|经济学家]]，他「分別以一個銀元1元投考法律學校以及商科分校」<ref name="首都经济信息报19950904" />去读书，「但是最終因为学校以[[英语|英语]]」<ref name="首都经济信息报19950904" />授课，毛泽东不懂英语，「在入读一个月後」<ref name="首都经济信息报19950904" />，在月底「中途退学」<ref name="首都经济信息报19950904" />。{{r|ant|page=59-61}}{{r|bull|page=18}} 到最後，他「回到湖南老家投考當地有名的」<ref name="首都经济信息报19950904" />[[湖南省立第一中学|湖南省立第一中学]]，花费报名费1元，結果在入学考试中以卓越的成績——其中文作文更獲甲等而獲校方取錄」<ref name="首都经济信息报19950904" />，名列第一。在学校读了《[[资治通鉴|御批通鉴辑览]]》，由于学校的课程有限和校规麻烦，毛泽东在学校读了6个月就退学了，他定了一个自修计划，每天到[[湖南省立图书馆|湖南省立图书馆]]去看书，从每天早晨[[图书馆|图书馆]]开门到关门，他都在图书馆度过，中午吃两块[[米糕|米糕]]，晚上住在[[湘乡会馆|湘乡会馆]]，一直持续了半年。{{r|ant|page=59-61}}{{r|bull|page=20-25}} 在图书馆，毛泽东主要学习了[[世界地理|世界地理]]和[[世界历史|世界历史]]，第一次看到了[[世界地图|世界地图]]《世界堪舆大地图》，研究了[[亚当·斯密|亚当·斯密]]的《[[原富|原富]]》、[[达尔文|达尔文]]的《[[物种起源|物种起源]]》、[[约翰·穆勒|约翰·穆勒]]的《[[伦理学|伦理学]]》、[[斯宾塞|斯宾塞]]的《逻辑》、[[孟德斯鸠|孟德斯鸠]]的[[法律|法律]]专著等。{{r|ant|page=59-61}}

== 湖南省立第一师范 ==
[[File:_YangChangJi.jpg|thumb]]
[[File:_Mao_Zedong_and_Xu_Teli_1936.jpg|thumb]]
1913年，由于家里不供应生活费，毛泽东在会馆呆不下去了，开始寻找新去处。一日，他看到了湖南公立第四师范的招生广告，“不收学费、膳食低廉”，他除了自己写作文报考，同时还替两人写了作文报考，结果三人都被录取了。{{r|ant|page=61}}1914年，湖南公立第四师范合并到[[湖南第一师范学院|湖南省立第一师范]]，<ref name="shark ">{{cite web |url=http://www.xxcb.cn/show.asp?id=875381 |title=毛主席一师读书期间共花了160元 |accessdate=2007-08-26 |publisher=潇湘晨报 |work= |language=zh-cn }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 毛泽东在湖南省立第一师范读了5年书，他结识了老师[[杨昌济|杨昌济]]、[[孔昭绶|孔昭绶]]、[[方维夏|方维夏]]、[[徐特立|徐特立]]、[[袁吉六|袁吉六]]、[[易培基|易培基]]、[[饶伯斯|饶伯斯]]、[[费尔康|费尔康]]、[[黄澍涛|黄澍涛]]、[[雷明亮|雷明亮]]、[[王立庵|王立庵]]等，结识了同辈学生[[蔡和森|蔡和森]]、[[萧子升|萧子升]]、[[萧三|萧三]]、[[李立三|李立三]]、[[陶斯咏|陶斯咏]]、[[向警予|向警予]]、[[何叔衡|何叔衡]]、[[张昆弟|张昆弟]]、[[罗学瓚|罗学瓚]]、[[陈章甫|陈章甫]]、[[郭亮|郭亮]]、[[蔡畅|蔡畅]]、[[李维汉|李维汉]]、[[罗章龙|罗章龙]]、[[周世钊|周世钊]]等。{{r|ant|page=61}}{{r|bull|page=30-40}}

在湖南省立第一师范，毛泽东喜欢[[社会科学|社会科学]]，厌恶[[自然科学|自然科学]]。{{r|ant|page=62}} 在一次写生课上，他画了一条直线，上面有个半圆，表示“半壁见海日”；另外一次写生课，画蛋，他画了一个圆圈，结果图画课只有40分，图画课老师是[[谭柄锷|谭柄锷]]。{{r|ant|page=62}}<ref name=" lion" />{{r|bull|page=26-30}}

毛泽东喜欢梁启超的文体，在作文课上被老师袁吉六批评说是“新闻记者的手笔”，并且叫借给他一套[[韩愈|韩愈]]的文集，告诫他：“古文之兴，盛于唐宋，[[唐宋八大家|唐宋八大家]]，又以昌黎先生开千古文风之滥觞，读通了韩文，就读通了[[文言文|古文]]，也就懂得什么是真文章。你的文章，缺的就是古之大家的凝练、平稳、含蓄、从容，如满弦之弓，只张不驰，令人全无回味，这是作文的大忌！”他钻研了韩愈的文章，学会了古文。{{r|ant|page=62}} 毛泽东的伦理学老师是杨昌济，一次，他写了一篇《心之力》的文章，得了100加5分。{{r|ant|page=62}}{{r|bull|page=30}}

第二年的暑假，老师[[汤增璧|汤增璧]]给毛泽东一些《[[民报|民报]]》阅读，毛泽东看到上面中国学生周游列国的故事，深受启发，但是由于没钱，决定在湖南旅行。毛泽东开始徒步旅行，同行者萧子升，他们游历了湖南[[宁乡县|宁乡县]]、[[安化县|安化县]]等5个县，没有花一分钱。<ref name=" lion" >{{cite web |url=http://news.xinhuanet.com/politics/2011-05/09/c_121394701.htm |title= 毛主席的老师，都还好吗？|accessdate=2011年5月9日 |publisher=新华网 |work= |language =zh-cn}}</ref>

一次，[[杨昌济|杨昌济]]推荐毛泽东看[[胡林翼|胡林翼]]（字润芝）的《胡文忠公全集》，毛泽东阅读后改字“学润”，经杨昌济建议改字“润芝”，也写作“润之”。1915年一日，毛泽东在报纸上刊登了“二十八画生征友”的新闻，得到三个半人的响应，一个是罗章龙，两个青年，半个是李立三。{{r|ant|page=63}} 

毛泽东在第一师范注重体育锻炼，经常雨中爬[[岳麓山|岳麓山]]，晴天在[[湘江|湘江]]游泳。{{r|ant|page=64}} 在游泳时，他写下了一首对联：

有志未伸，有求不获。精神痛苦，以此为卓。 

天乎人欤，倾地一角。次则儿辈，育之成行。 

如果未熟，介在青黄。病时揽手，酸心结肠。 

但呼儿辈，各务为良。又次所怀，好亲至爱。 

或属素恩，或多劳瘁。大小亲疏，均待报赉。 

总兹所述，盛德所辉。必秉悃忱，则效不违。 

致于所恨，必补遗缺。念兹在兹，此心不越。 

养育深恩，春晖朝霭。报之何时，精禽大海。 

呜呼吾母！母终未死。躯壳虽隳，灵则万古。 

有生一日，皆报恩时。有生一日，皆伴亲时。 

今也言长，时则苦短。惟挈大端，置其粗浅。 

此时家奠，尽此一觞。后有言陈，与日俱长。 

尚飨！
</poem>{{hidden end}}|salign=right|source=}}
{{clear}}

毛泽东的湖南省立第一师范一共读了5年，花费了160元，三分之一用在订报纸上，每月一元，常常还在书摊买杂志买书，这也培养了他读报的习惯。在湖南省立第一师范，毛泽东订立了规矩“三不谈”：“不谈金钱，不谈男女，不谈家庭琐事”。{{r|ant|page=65}}<ref name=" lion" />

== 北京大学图书馆助理 ==
1918年夏天，毛泽东和一些学生远赴北京，募捐善款资助学生留学[[欧洲|欧洲]]。{{r|ant|page=66}} 毛泽东的老师杨昌济介绍他到[[北京大学图书馆|北京大学图书馆]]主任[[李大钊|李大钊]]那儿，做了助理员，每月8元。在北京大学，毛泽东认识了未来的妻子[[杨开慧|杨开慧]]，并且两人相恋。

1919年年初，毛泽东陪同去[[法国|法国]]留学的学生前往[[上海|上海]]，毛泽东借到了10元，买了到[[浦口|浦口]]的票，中途还在[[曲阜|曲阜]]下车，瞻仰了孔子故里，并且登了[[泰山|泰山]]。到了浦口后，毛泽东身无分文，而且鞋子被盗，幸亏一个湖南的老乡借钱给他，才圆满到达上海。{{r|ant|page=67}} 护送完学生上船之后，毛泽东回到长沙。{{r|ant|page=67}} 

== 驱张运动 ==
{{main|驱张运动}}
1918年4月，皖系军阀[[张敬尧|张敬尧]]和直系军阀[[吴佩孚|吴佩孚]]带兵占据湖南，随后，张敬尧被[[段祺瑞|段祺瑞]]任命为“湖南督军”兼“省长”，张敬尧烧杀抢掠、掠夺财富、压制言论自由，无恶不作，湖南人称他为“堂堂呼张，尧舜禹汤，一二三四，虎豹豺狼，张毒不除，湖南无望。”。<ref name=" july" >{{cite web |url=http://cdwb.cdyee.com/html/2011-06/03/content_219992.htm |title= 常德城内的“驱张”斗争|accessdate= 2011年6月3日|publisher=常德晚报 |work= |language =zh-cn}}</ref> 此时，毛泽东在《[[湘江评论|湘江评论]]》担任主笔，并且帮助创办了文化书社，毛泽东领导学生罢课反对湖南督军张敬尧，结果《湘江评论》被张敬尧查封。{{r|ant|page=69-71}} 随后，毛泽东再次远赴北京，担任新民学会的代表，宣传反对张敬尧和军阀的活动，并且担任一个通讯社的社长。{{r|ant|page=69-71}}

1919年9月，毛泽东带领新民学会会员，透过湖南学生联合会，组织学生和进步人士发动“驱张运动”，在全国各地得到了积极响应。<ref name=" july" />

1920年6月11日，张敬尧被谭延闿攻打而大败，6月26日，张敬尧逃出湖南省。<ref name=" july" />

== 湖南省独立运动 ==
{{main|湖南省独立运动}}
1919年，毛泽东再度前往上海，他和陈独秀讨论了“改造湖南联盟”的计划，紧接着，毛泽东回到长沙，组织联盟，他一边担任一师附小主事并兼任师范部22班国文教员<ref name=" lion" />，一边在新民学会活动，主要任务是争取湖南省独立。{{r|ant|page=69-71}} 谭延闿不久被[[赵恒惕|赵恒惕]]赶下台，赵恒惕提出“[[联省自治|联省自治]]”，实质上却镇压民主运动。{{r|ant|page=69-71}} 毛泽东在《新湖南》发表文章鼓吹改革。{{r|ant|page=69-71}}

1920年1月，[[杨昌济|杨昌济]]在北京病逝，临终前写信给广州军政府秘书长[[章士钊|章士钊]]推荐[[毛泽东|毛泽东]]和蔡和森：“吾郑重语君，二子海内人才，前程远大，君不言救国则已，救国必先重二子”。<ref>{{cite web |url= http://www.chinanews.com/cul/2011/07-05/3159225.shtml|title= 青年毛泽东在北大韬光养晦 职位低微别人不理睬|accessdate= 2011年7月5日|publisher= 中国新闻网|work= |language =zh-cn}}</ref>

1920年，毛泽东组织示威游行，被赵恒惕的警察镇压。{{r|ant|page=69-71}} 3月，毛泽东和老师[[黎锦熙|黎锦熙]]讨论湖南省自治运动的问题：
{{Quote box
|title = 
|quote = 奉上‘湖南建设问题条件’二份，有好些处所尚应大加斟酌。弟于吾湘将来究竟应该怎样改革，本不明白。并且湖南是中国里面的一省，除非将来改变局势，地位变成美之‘[[州|州]]’或德之‘[[邦|邦]]’，是不容易有独立创设的。又从中国现下全般局势而论，稍有觉悟的人，应该就从如先生所说的‘根本解决’下手，目前情况的为善为恶，尽可置之不闻不问，听他们去自生自灭。这样支支节节的向老虎口里讨碎肉，就使坐定一个‘可以办到’，论益处，是始终没有多大的数量的。不过，这一回我们已经骑在老虎背上，连这一着‘次货’—在中国现状内实在是‘上货’—都不做，便觉太不好意思了。”<br />湖南是中国里面的一省，除非将来改变局势，地位变成美之‘州’或德之‘邦’，是不容易有独立创设的。湘事糟透，皆由于人民之多数不能自觉，不能奋起主张，有话不说，有意不伸，南北武人乃得乘隙凌侮，据湖南为地盘，括民财归己豪。……消极方面，莫如废督裁兵；积极方面，莫如建设民治。……吾人主张，‘湘人自决主义’，其意义并非部落主义，又非割据主义，乃以在湖南一块地域之文明，湖南人应自负其创造之责任。……湘人自决主者，门罗主义也。湖南者湖南人之湖南，湖南人不干涉外省事，外省人亦切不可干涉湖南事。

1920年9月3日，毛泽东在湖南长沙《大公报》发表文章《湖南建设问题的根本问题——湖南共和国》：
{{Quote box
 |title = 
 |quote = 乡居寂静，一卧兼旬。九月一号到省，翻阅《大公报》，封面打了红色，中间有许多我所最喜欢的议论，引起我的高兴，很愿意继着将我的一些意思写出。<br />我是反对‘大中华民国’的，我是主张‘湖南共和国’的。有甚么理由呢？<br />大概从前有一种谬论，就是‘在今后世界能够争存的国家，必定是大国家’。这种议论的流毒，扩充帝国主义，压抑自国的小弱民族，在争海外殖民地，使半开化未开化之民族变成完全奴隶，窒其生存向上，而惟使恭顺驯屈于己。最著的例是[[英国|英]]、[[美国|美]]、[[德国|德]]、[[法国|法]]、[[俄国|俄]]、[[奥匈帝国|奥]]，他们幸都收了其实没有成功的成功。还有一个就是中国，连‘其实没有成功的成功’都没收得，收得的是[[满洲|满洲]]人消灭，[[蒙古族|蒙人]][[回族|回人]][[藏族|藏人]]奄奄欲死，十八省乱七八糟，造成三个政府，三个国会，二十个以上督军王巡按使王总司令王，老百姓天天被人杀死奸死，财产荡空，外债如麻。号称共和民国，没有几个个懂得‘甚么是共和’的国民，四万万人至少有三万九千万不晓得写信看报。全国没有一条自主的铁路。不能办邮政，不能驾‘洋船’，不能经理食盐。十八省中像湖南[[四川|四川]][[广东|广东]][[福建|福建]][[浙江|浙江]]湖北一类的省，通变成被征服省，屡践他人的马蹄，受害无极。这些果都是谁之罪呢？我敢说，是帝国之罪，是大国之罪，是‘在世界能够争存的国家必定是大国家’一种谬论的罪。根本的说，是人民的罪。<br />现在我们知道，世界的大国多半瓦解了。俄国的旗子变成红了色，完全是世界主义的平民天下。德国也染成了半红。[[波兰|波兰]]独立，[[捷克|截克]]独立，[[匈牙利|匈牙利]]独立。尤太、[[阿刺伯|阿刺伯]]、[[亚美尼亚|亚美尼亚]]，都重新建国。[[爱尔兰|爱尔兰]]狂欲脱离[[大英帝国|英吉利]]，[[朝鲜|朝鲜]]狂欲脱离[[大日本帝国|日本]]。在我们[[东北|东北]]的[[西伯利亚|西伯利亚]][[远东|远东]]片上，亦建了三个政府。全世界风起云涌，‘民族自决’高唱入云。打破大国迷梦，知道是野心家欺人的鬼话。推翻[[帝国主义|帝国主义]]，不许他再来作祟，全世界盖有好些人民业已醒觉了。<br />中国呢？也醒觉了（除开政客官僚军阀）。二九年假共和大战乱的经验，迫人不得不醒觉，知道全国的总建设在一个期内完全无望。最好办法，是索性不谋总建设、索性分裂，去谋各省的分建设，实行‘各省人民自决主义’。二十二行省三特区两藩地，合共二十七个地方，最好分为二十七国。<br />湖南呢？至于我们湖南，尤其三千万人个个应该醒觉了！湖南人没有别的法子，唯一的法子是湖南人自决自治，是湖南人在湖南地域建设一个‘湖南共和国’。我曾着实想过，救湖南，救中国，图与全世界解放的民族携手，均非这样不行。湖南人没有把湖南自建为国的决心和勇气，湖南终究是没办法。<br />谈湖南建设问题，我觉得这是一个根本问题。我颇有一点意思要发表出来，乞吾三千万同胞的聪听，希望共起讨论这一个顶有意思的大问题。今大是个发端，余侯明日以后继续讨论。

== 转向马克思主义 ==
1920年9月，湖南第一师范学校校长易培基聘请毛泽东担任一师附小的主事。<ref>[http://politics.people.com.cn/h/2011/0912/c226651-1658862205.html 解密毛泽东当年如何当老师]</ref>毛泽东住在[[朱劍凡|朱劍凡]]创办的[[周南中学|周南女中]]教工宿舍。<ref>[http://dangshi.people.com.cn/GB/85040/14164395.html 毛泽东与明朝皇族后裔朱伯深的故事]</ref>

1920年10月下旬-11月上旬，在长沙参与主办了[[伯特兰·罗素|罗素]]、[[蔡元培|蔡元培]]、[[张东荪|张东荪]]、[[李石岑|李石岑]]、[[杨端六|杨端六]]等中外学者的公开讲座，并担任长沙《[[大公报|大公报]]》特邀演讲速记员与评论撰稿人。对罗素在长沙的讲演《布尔什维克与世界政治》，毛泽东听后反倒是更倾向于马克思主义：
<blockquote>
我对[[萧子升|子升]][[李维汉|和笙]]两人的意见。【用平和的手段，谋全体的幸福】在真理上是赞成的，但在事实上认为做不到。[[伯特兰·罗素|罗素]]在长沙演兑，意与子升及和笙同，主张共产主义，但反对劳农专政，谓宜用教育的方法使有产阶级觉悟，可不至要妨碍自由，兴起战争，革命流血。但我于罗素讲演后，曾和[[彭璜|殷柏]]，[[易礼容|礼容]]等有极详细之辩论。我对于罗素的主张，有两句评语：“论上说得通，事实上做不到”。罗素和子升和笙主张的要点，是“用教育的方法”。但教育一要有钱，二要有人，三要有机关。现在世界，钱尽在资本家的手，主持教育的人尽是一些资本家，或资本家的奴隶。总言之，现在世界的学校及报馆两种最主要的教育机关，又尽在资本家的掌握中，现在世界的教育是一种资本主义的教育。以资本主义教育儿童，这些儿童大了又转而用资本主义教育第二代的儿童。教育所以落在资本家手里，则因为资本家有“议会”以制定保护资本家并防制无产阶级的法律，有“政府”执行这些法律，以积极的实现其所保护与所禁止。有“军队”与“警察”，以消极的保障资本家的安乐与禁止无产者的要求。有“银行”以为其财货流通的府库。有工厂以为其生产品垄断的机关。如此，共产党人非取政权，且不能安息于其守下，更要能握得其教育权；如此资本家久握教育权，大鼓吹其资本主义，使共产党人的共产主义宣传，信者日见其微。所以我觉得教育的方法是行不通的。我看俄国式的革命是无可如何的山穷水尽诸路皆走不通的一个变计。并不是有更好的方法弃而不采，单要采取这个恐怖的方法。<ref>毛泽东：“给肖旭东、蔡林彬并在法诸会友的信”，《毛泽东书信选集》，中央文献出版社，2003年11月版，ISBN 7507314804 </ref>
</blockquote>

[[File:_YangKaiHui_and_her_son.jpg|thumb]]
这年冬天，毛泽东受到[[马克思主义|马克思主义]]和[[俄国|俄国]][[十月革命|十月革命]]的影响，开始接近[[共产主义|共产主义]]，并且认为自己是一个马克思主义者。{{r|ant|page=69-71}} 同年，他和杨开慧结婚，结婚地址是一师附小主事室，<ref name="shark " /> 证婚人是楚怡校长[[陈润霖|陈润霖]]，主婚人是一师校长易培基；<ref name=" lion" /> 而他的父亲毛贻昌因为[[伤寒|伤寒]]在该年逝世。{{r|ant|page=69-71}}

1921年夏，毛泽东受聘为湖南第一师范学校的国文教员，结束了短暂的附小岁月。

1921年6月，毛泽东奔赴上海，参加[[中国共产党第一次全国代表大会|中国共产党第一次全国代表大会]]，开启了他的革命生涯。<ref name=" lion" />
