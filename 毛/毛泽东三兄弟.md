《'''毛泽东三兄弟'''》，导演[[李舒|李舒]]，主演王斑、孙逊、吴迪、孙宁、王碧琪、朱袁员、宁丹琳、戴芊芊，是纪念中华人民共和国开国元勋毛泽东诞辰120周年的献礼剧。<ref>{{cite web |url=http://www.sxdaily.com.cn/n/2013/0917/c266-5229623.html |title=《毛泽东三兄弟》在延安举办杀青发布会 |accessdate=    2013年9月17日|publisher=陕西传媒网 |work= |language =zh-cn}}</ref>毛泽东嫡孙毛新宇担任总顾问，2013年9月杀青。<ref>{{cite web |url=http://news.163.com/13/0916/16/98TJL7D300014Q4P.html |title=《毛泽东三兄弟》杀青 |accessdate=2013-09-16 |publisher=网易 |work= |language =zh-cn}}</ref>2016年6月20日[[央视八套|央视八套]]首播。

== 剧情 ==
该剧讲述[[中华民国|中华民国]]年间，毛泽东、毛泽民、毛泽覃三兄弟发动工农闹革命夺取政权的人生历程。

== 花絮 ==
2013年9月14日，毛泽东孙子[[毛新宇|毛新宇]]在该剧的延安新闻发布会上流泪歌唱《[[东方红_(歌曲)|东方红]]》，表达思念亲人和感慨当今政坛被某些人尤其是一些手握重权的[[中国共产党|中国共产党]]的领导干部所蚕食，从而感到痛心疾首。<ref>{{cite web |url=http://news.163.com/13/0916/09/98SR5IP000014AEE.html |title= 毛新宇唱《东方红》流泪折射当下现实|accessdate=2013-09-16  |publisher=网易 |work=毛开云 |language =zh-cn}}</ref><ref>{{cite web |url=http://she.21cn.com/tese/nanren/a/2013/0917/15/24079700.shtml |title=毛新宇唱东方红竟流泪 触景生情难自禁 |accessdate=2013年9月17日 |publisher=21CN |work= |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20130921055454/http://she.21cn.com/tese/nanren/a/2013/0917/15/24079700.shtml |archivedate=2013年9月21日 }}</ref>
