'''毛黴屬'''（學名：{{lang|la|''Mucor''}}）是一個包含大約有十多個品種的[[黴菌|黴菌]]屬，普遍可在泥土、植物的表面、腐爛的蔬果或[[消化系統|消化系統]]中找到它，它可以制作豆腐乳。

== 描述 ==
黴菌是一種[[真菌|真菌]]，它的[[菌落|菌落]]的顏色一般呈現白色、米黃色到灰色，生長快速，在適合的培植介質中可以生長至數-{A|厘米}-高。當菌落成熟，由於孢子的發展，使菌落的顏色變成灰或褐色。

== 繁殖 ==
部分品種（{{lang|la|''Mucor mucedo''}}）[[無性繁殖|無性繁殖]]。當[[孢子囊柄|孢子囊柄]]（sporangiophore）直立及發育成熟，孢子囊柄頂端脹大形成球狀[[孢子囊|孢子囊]]（sporangium），內裝單核[[單倍體|單倍體]][[孢子|孢子]]（sporangiospore）。孢子囊頂端的[[囊軸|囊軸]]向孢子囊伸出，囊壁隨之爆開並釋出孢子。 

[[有性繁殖|有性繁殖]]時，部分短小[[菌絲|菌絲]]形成配子囊（gametangia），當兩合適之配子囊接合，便形成球狀、具厚壁的接合孢子囊（zygosporangium）。接合孢子囊內藏一個接合孢子（zygospore）。接合孢子囊內進行核配（karyogamy）和[[減數分裂|減數分裂]]，可能可抵禦外在的惡劣環境。毛黴屬包括同宗配合（雌雄同株）和异宗配合（雌雄异株）的種。

== 其他 ==
因為大部分毛黴屬無法在溫暖環境中生長（接近攝氏37度），它們多不會感染人類和[[温血動物|温血動物]]。耐熱品種（{{lang|la|''Mucor indicus''}}）可致坏死性且能快速传播的{{Link-en|接合菌病|Zygomycosis|接合菌病}}。
