'''毛南族'''以农业为主，以饲养业和副业为辅，主要种植杂粮、少数地区种植水稻。毛南族聚居地在[[广西壮族自治区|广西壮族自治区]]西北部的[[环江毛南族自治县|环江毛南族自治县]]，其余散居在[[南丹县|南丹]]、[[金城江区|金城江]]、[[都安县|都安]]等地，[[贵州省|贵州省]][[平塘县|平塘县]][[卡蒲毛南族乡|卡蒲]]有集中分布。这个地区紧靠[[云贵高原|云贵高原]]东麓，中部是茅南山，东北部是九万大山，西北部是凤凰山，西南部是大石山区，属亚热带气候，适合于农耕畜牧。 　　

== 语言 ==
毛南族使用[[毛南语|毛南语]]，属[[侗台语族|侗台语族]][[侗水语支|侗水语支]]（一说为''[[南島语系|南島语系]]''或''[[泰－卡岱语系|泰－卡岱语系]]''）。2010年前该语言没有书写系统，曾用[[汉字|汉字]]音义拼记或以汉字为基础创造新字来记录毛南语的[[民歌|民歌]]和宗教诵本等，称“[[土俗字|土俗字]]”。2010年后才出现了以拉丁字母为基础的[http://maonan.org/wenzi “毛南语拼音方案”]。语言特点粗略来说句法属SVO类型（即主语＋动词＋宾语），如 Maenz（他／她）bai（去）her（集市）“他／她去赶集／赶墟”；中心语位于修饰语之前，如 mus（猪）naem（黑）“黑色的猪”。现代政治经济词汇多为汉语[[西南官话|西南官话]]借词，旧的汉语借词多为[[广西平话|广西平话]]借词。语音方面，一般认为有8个能区别意义的声调；毛南语的声母系统（80多个声母）比汉语（约20个声母）、壮语（约30个声母）等复杂。毛南人多通[[壮语|壮语]]和[[汉语|汉语]]，通用汉语文。小学、中学采用汉语课本，用汉语教授，但小学老师讨论教案多使用毛南语[http://maonan.org/language_cn.asp]。毛南族聚居的中心区－－下南乡政府日常语言一般是[[毛南语。环江毛南族自治县政府官方语言是汉语[[西南官话。毛南族的另外一个分支（约占人口1／3）聚居于贵州南部，通用“扬黄话”，所以也叫“扬黄人”（见陆天桥著[http://maonan.org/book_cn.asp 《毛南语语法》]。美国佛罗里达：环球出版社。2008）。

== 历史 ==
毛南族自称“阿南”{{IPA|/ʔai˦˨naːn˧˩˨/}}，或者“毛南”{{IPA|/maːu˨˦naːn˧˩˨/}}，意为“当地人”。[[唐代|唐代]]以前的“[[僚|僚]]”，宋、元、明 时代的“伶”，与毛南人的先祖有渊源关系。史籍中曾先后出现过“茅滩”、“茆滩”、“茅难”、“冒南” 、“毛难”、“毛南”等名称。

1986年以前称'''毛难族'''。

== 文化习俗 ==
毛南族人擅长竹编工艺。编制的竹帽做为当地男女订情之物。细密的竹帽代表两人的情意绵绵。

毛南族多信仰[[道教|道教]]，也有少数信仰[[佛教|佛教]]。

明伦白切香猪是毛南族的名菜，用当地产香猪加佐料蒸制而成。此外还有：甜[[红薯|红薯]]；[[螺蛳|螺蛳]]酸，蒜头酸水，鸭血酱，[[豆腐|豆腐]]肴。

毛南族的“[[南瓜节|南瓜节]]”为农历九月九日。各家把收获到家的形状各异，桔黄色的大[[南瓜|南瓜]]摆满楼板，逐一挑选。年轻人走门串户，到各家评选“ 南瓜王”。不仅要看外观，而且要透过表面看到瓜籽。 待到众人意见基本一致，由一身强力壮者用砍刀劈开“ 南瓜王”，主人掏出瓜瓢，把饱满的籽留作来年的种子。再把南瓜切成块，放进[[小米|小米]]粥锅裡，以文火煨炖到烂熟，先盛一碗供在香火堂前敬奉“南瓜王”，然后众人共餐同享。

“分龙节”，又叫“五月庙节”，是毛南族特有的节日，在阴历的“分龙”日前两天开始举行，主要是祭祀神灵与祖先，全村男女以及外嫁的女子和远道的亲友都赶来参加，隆重而热烈。
