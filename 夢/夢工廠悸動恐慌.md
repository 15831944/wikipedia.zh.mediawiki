{{Vgname|夢工廠悸動恐慌|ja=夢工場ドキドキパニック|aka=夢工廠心跳恐慌}}是1987年夏季所推出，由[[任天堂情報開發本部|任天堂]]所開發、[[富士電視台|富士電視台]]提供遊戲角色造型的[[FC磁碟機|FC磁碟機]]平台[[橫向捲軸遊戲|橫向捲軸]]電子遊戲。

此作品後續有被任天堂將裡頭角色替換成《[[超級瑪利歐|超級瑪利歐]]》系列人物，於1988年在[[美國|美國]]以「[[超級瑪利歐兄弟USA|超級瑪利歐兄弟2]]」名義發行。

== 劇情 ==

有一天；雙胞胎{{nihongo|皮奇|ピキ}}與{{nihongo|波奇|ポキ}}發現他們所飼養的寵物猴子{{nihongo|露莎|ルーサ}}從某處撿到一本專門介紹「夢宇宙」事蹟的繪本。夢宇宙是個人們會利用{{nihongo|「夢想機器」|ドリームマシン}}；來製造出美好事物的神祕和平世界。但某日突然冒出名叫{{nihongo|魔夢|マムー}}的怪物出現，將夢想機器給改造成專門製造出魔物的東西，使得夢宇宙的人們感到恐慌。之後夢宇宙的人們靠著魔夢討厭蔬菜之類東西的弱點，將對方給打敗。<ref name="VideoGamden">{{cite web |url=http://www.videogameden.com/fds.htm?dok |title=YUME KŌJŌ - DOKI DOKI PANIC |work=Video Gameden |language=en |accessdate=2017年12月8日 |deadurl=no |archiveurl=https://web.archive.org/web/20180104091517/http://www.videogameden.com/fds.htm?dok |archivedate=2018年1月4日 }}</ref>

在皮奇與波奇翻閱過程中不小心將書本最後一頁弄破時，書裡突然冒出魔夢的手將他們給拉進書中，一旁的露莎發現這不可思議的情況後，立刻跑去找人求救。

之後皮奇與波奇的家族成員；大哥{{nihongo|伊瑪吉|イマジン}}、大姐{{nihongo|莉娜|リーナ}}、以及{{nihongo|媽媽|ママ}}與{{nihongo|爸爸|パパ}}一同潛入了古書，來拯救被綁走的皮奇與波奇。<ref>{{cite web |url=http://themushroomkingdom.net/smb2_ddp.shtml |title=Super Mario Bros. 2 From Doki Doki Panic to Super Mario Bros. 2 |work=The Mushroom Kingdom |language=en |accessdate=2017年12月8日 |deadurl=no |archiveurl=https://web.archive.org/web/20080725023959/http://themushroomkingdom.net/smb2_ddp.shtml |archivedate=2008年7月25日 }}</ref><ref name="VideoGamden" />

== 玩法 ==

遊戲場景是以[[橫向捲軸遊戲|橫向捲軸]]方式呈現，過程中玩家可以透過挖取埋在地裡的物品、與拿取擺置在地上的道具、或甚至是將敵方角色給直接抓起來當作是扔擲的攻擊武器。<ref name="VideoGamden" />

內容總共包含七大章節、二十個小關卡，在每個大章節的開頭會讓玩家從伊瑪吉、莉娜、媽媽、爸爸四人當中選取一位來闖關。

四名角色中各擁有不同的能力屬性。伊瑪吉是各方面能力都有一定水準的角色，莉娜是四人當中挖掘物品動作最慢、但擁有可在跳躍後能緩慢下降的特殊能力，媽媽是挖掘物品動作也不快、但具備四人當中最強的跳躍能力，爸爸則是四人當中跳躍能力最差、但有最快挖出物品的力氣和衝刺速度。<ref name="VideoGamden" />

== 開發 ==

[[File:Yumekojo'87.JPG|thumb]]

此為[[田邊賢輔|田邊賢輔]]在進入[[任天堂|任天堂]]裡首款著手設計的作品。<ref>{{cite web |url=http://www.nintendoworldreport.com/interview/2181 |archiveurl=https://web.archive.org/web/20131105143103/http://www.nintendoworldreport.com/interview/2181 |dead-url=yes |title=Kensuke Tanabe Talks Metroid Prime 2: Echoes |work=Nintendo World Report |language=en |date=2004年5月18日 |archivedate=2013年11月5日 |accessdate=2017年12月8日}}</ref>過去他曾構想過一種兩人對戰的遊戲，內容是設法將遍佈在地上四周的物品給堆積成長柱狀後往上爬，並且可以把對手當成物品給抓起來朝一旁扔開；來妨礙對方進度獲取勝利。<ref name="Wired">{{cite web |url=https://www.wired.com/2011/04/super-mario-bros-2/ |title=The secret history of Super Mario Bros. 2 |work=[[連線|連線]] |author=Chris Kohler |language=en |date=2011年4月1日 |accessdate=2017年12月8日 |deadurl=no |archiveurl=https://web.archive.org/web/20161228000622/https://www.wired.com/2011/04/super-mario-bros-2/ |archivedate=2016年12月28日 }}</ref>當田邊賢輔把他的想法告訴《[[超級瑪利歐|超級瑪利歐]]》系列創造者[[宮本茂|宮本茂]]後，他們依此概念弄出了一款試玩作品。但受限於任天堂[[紅白機|FC]]遊戲機的硬體性能並未將遊戲概念給完美表現出來，若改設計成獨自一人遊玩作品又有乏味不有趣等等情況，讓此點子被擱置一旁。<ref name="Wired" />

到了1987年日本[[富士產經集團|富士產經集團]]以旗下的[[富士電視台|富士電視台]]為主首，規劃當年夏季在[[東京|東京]][[晴海|晴海]]、以及[[大阪|大阪]][[住之江區|住之江區]]兩地舉辦出有著各種娛樂設施、多媒體技術展示的活動{{nihongo|「通訊嘉年華夢工廠'87」|コミュニケーションカーニバル 夢工場'87}}。<ref>{{cite web |url=https://www.fujitv.co.jp/company/info/history.html |script-title=ja:沿革 |work=[[富士電視台|富士電視台]] |language=ja |accessdate=2017年12月8日 |deadurl=no |archiveurl=https://web.archive.org/web/20160722071045/http://www.fujitv.co.jp/company/info/history.html |archivedate=2016年7月22日 }}</ref><ref>{{cite web |url=http://news.livedoor.com/article/detail/9100849/ |script-title=ja:フジは売上180億円！民放夏のイベント“ボロ儲け”のカラクリ |work=[[活力門|活力門]] |language=ja |date=2014年7月31日 |accessdate=2017年12月8日 |deadurl=no |archiveurl=https://web.archive.org/web/20140731212217/http://news.livedoor.com/article/detail/9100849/ |archivedate=2014年7月31日 }}</ref>當時富士電視台為活動所設計出的形象角色；是外觀穿著為[[阿拉伯世界|阿拉伯世界]]風格的{{nihongo|伊瑪吉家族|イマジンファミリー}}。之後考慮[[商業搭配|商業搭配]]的需求，富士電視台向任天堂提出設計一款以伊瑪吉家族所打造的遊戲。<ref name="Wired" />

田邊賢輔接下富士電視台的請求後，把之前所構想兩人互搶地上物品來堆疊爬高的競賽點子；改變成將地上物品抓起後朝敵方攻擊的遊戲，宮本茂也提議可以試著做成像是《[[超級瑪利歐兄弟|超級瑪利歐兄弟]]》般的[[橫向捲軸遊戲|橫向捲軸]]作品。<ref name="Wired" />

最後所製作出的成品《夢工廠悸動恐慌》，是推出在任天堂的[[FC磁碟機|FC磁碟機]]平台上。<ref>{{cite web |url=https://www.gamerankings.com/famicomds/578449-yume-koujou-doki-doki-panic/index.html |title=Yume Koujou: Doki Doki Panic |work=[[GameRankings|GameRankings]] |language=en |accessdate=2017年12月8日 |deadurl=no |archiveurl=https://web.archive.org/web/20150911105355/http://www.gamerankings.com/famicomds/578449-yume-koujou-doki-doki-panic/index.html |archivedate=2015年9月11日 }}</ref>

== 影響 ==

在《夢工廠悸動恐慌》推出前；1986年任天堂已在日本發行了《[[超級瑪利歐兄弟|超級瑪利歐兄弟]]》的續作《[[超級瑪利歐兄弟2|超級瑪利歐兄弟2]]》，內容上沿用著前作的圖形與音效介面，除了編譯難度更高的關卡、添加些許新敵人和調整部份細節外，與前作並沒有太多差別。<ref name="IGN20120615">{{cite web |url=http://www.ign.com/articles/2012/06/15/this-is-shigeru-miyamotos-favorite-mario-game |title=THIS IS SHIGERU MIYAMOTO'S FAVORITE MARIO GAME |work=IGN |author=Samuel Claiborn |language=en |date=2012年6月15日 |accessdate=2017年12月8日 |deadurl=no |archiveurl=https://www.webcitation.org/6XT3CyWey?url=http://www.ign.com/articles/2012/06/15/this-is-shigeru-miyamotos-favorite-mario-game |archivedate=2015年4月1日 }}</ref><ref name="IGN2010913" />當時任天堂的美國分部試玩了《超級瑪利歐兄弟2》內容後，向日本總部反應該遊戲難度偏高，容易讓玩家產生挫折感，對這種不會讓人感到愉悅的作品有賣相不佳疑慮，便提議不應讓此遊戲在美國當地上市發行。<ref name="IGN20120615" /><ref name="IGN2010913" />

之後任天堂日本總部弄了替代方案，將《夢工廠悸動恐慌》的圖形與操作功能做調整後，在1988年以「[[超級瑪利歐兄弟USA|超級瑪利歐兄弟2]]」名義於美國推出。改版遊戲將《夢工廠悸動恐慌》替換為瑪利歐系列角色，伊瑪吉能力屬性的角色是[[瑪利歐|瑪利歐]]、莉娜是[[碧姬公主|碧姬公主]]、媽媽是[[路易吉|路易吉]]、爸爸則是[[奇諾比奧|奇諾比奧]]，而魔夢則改以「渥特」（Wart）稱之。1992年，遊戲以「超級瑪利歐兄弟USA」名稱逆向返回到日本當地發行。<ref name="IGN2010913">{{cite web |url=http://www.ign.com/articles/2010/09/14/ign-presents-the-history-of-super-mario-bros?page=3 |title=IGN PRESENTS: THE HISTORY OF SUPER MARIO BROS. |work=[[IGN|IGN]] |author=Rus Mclaughlin |language=en |date=2010年9月13日 |accessdate=2017年12月8日 |deadurl=no |archiveurl=https://www.webcitation.org/6XStk6Plq?url=http://www.ign.com/articles/2010/09/14/ign-presents-the-history-of-super-mario-bros?page=3 |archivedate=2015年4月1日 }}</ref><ref>{{cite web |url=https://kotaku.com/5675480/super-mario-all-stars-coming-to-america |title=Super Mario All-Stars Coming To America |work=[[Kotaku|Kotaku]] |author=Brian Ashcraft |language=en |date=2010年10月28日 |accessdate=2017年12月8日 |deadurl=no |archiveurl=https://web.archive.org/web/20170710182237/http://kotaku.com/5675480/super-mario-all-stars-coming-to-america |archivedate=2017年7月10日 }}</ref>後來日本版《超級瑪利歐兄弟2》在日本海外發行時，改採用《超級瑪利歐兄弟：遺失的關卡》（Super Mario Bros.: The Lost Levels）名稱。
