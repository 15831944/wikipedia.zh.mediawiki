'''夢時代站'''位於[[高雄市|高雄市]][[前鎮區|前鎮區]]，為[[高雄捷運環狀輕軌|高雄捷運環狀輕軌]]的捷運車站。

== 車站概要 ==
車站代碼為'''C5'''。站體位於[[成功路_(高雄市)|成功二路]]與[[時代大道|時代大道]]口，緊臨[[統一夢時代購物中心|統一夢時代購物中心]]。為[[側式月台|側式月台]]。

== 車站構造 ==

== 歷史 ==
===高雄環狀輕軌第一段C1-C14車站全民參與命名活動===
* 命名活動第一階段（2014年10月18日至11月20日）建議站名：成功時代站、時代大道站、夢時代站
* 命名活動第二階段（2015年2月16日至3月6日）票選選項：夢時代站、時代大道站、成功時代站
* 命名結果（2015年3月29日公告）：夢時代站<ref>[http://www.thenewslens.com/post/144007/ 高雄環狀輕軌C1-C14站有名字了！最快8月通車]</ref>

===2016===
* 2016年6月26日：隨著環狀輕軌第一階段C5-C8完工試營運啟用<ref>自由時報—[http://news.ltn.com.tw/news/life/breakingnews/1248359 高雄輕軌 力拚8月試營運]</ref><ref>中時電子報—[https://tw.news.yahoo.com/%E6%B0%B4%E5%B2%B8%E8%BC%95%E8%BB%8C-8%E6%9C%88%E5%BA%95%E9%9A%8E%E6%AE%B5%E9%80%9A%E8%BB%8A-215031837--finance.html 水岸輕軌 8月底階段通車]，2015年3月2日。</ref>。
===2017===
*2017年12月31日：本站12/31－1/1跨年活動期間不開放上下車，請往夢時代站的旅客要在[[凱旋中華站|凱旋中華站]]下車。<ref>[https://www.kcg.gov.tw/News_Content.aspx?n=F29A02A9D36C47F0&sms=19902EF36D6B551D&s=3DCD2E1AE339FA82 夢時代跨年派對資訊]</ref>
