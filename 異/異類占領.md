{{作品名稱|name=''Captive State''|hk=天逆反擊戰：地球起義|tw=異類占領|cn=俘虏国度}}是一部2019年美國[[科幻片|科幻]][[驚悚片|驚悚片]]，由[[魯柏·華爾特|魯柏·華爾特]]執導並與艾莉卡·貝尼（{{lang|en|Erica Beeney}}）共同撰寫劇本，[[約翰·古德曼|約翰·古德曼]]、[[艾許頓·桑德斯|艾許頓·桑德斯]]、{{le|喬納森·梅傑斯|Jonathan Majors}}、{{le|機關槍凱利 (饒舌歌手)|Machine Gun Kelly (rapper)|寇爾森·貝克}}與[[薇拉·法蜜嘉|薇拉·法蜜嘉]]主演。該片由[[焦點影業|焦點影業]]定於2019年3月15日在美國上映。

== 劇情 ==
故事敘述一群外星人降臨地球並以和平統一的幌子奴役人類。[[芝加哥|芝加哥]]警官墨瑞根為了將世上的人們聯合起來對付敵軍，而招募了反叛組織少年蓋布瑞一起行動。

== 發行 ==
2017年5月，據報導，發行商[[焦點影業|焦點影業]]將該片計劃於2018年8月17日在美國上映<ref name="Release">{{cite web|url=http://bloody-disgusting.com/movie/3437474/john-goodman-prepares-extraterrestrial-war-captive-state/|title=John Goodman Prepares for Extraterrestrial War in 'Captive State'|website=Bloody Disgusting|first=Brad|last=Miska|date=2017-05-15|accessdate=2017-05-16}}</ref>。之後，美國上映日被改至2019年3月29日；不久後，上映日又提前為2019年3月15日<ref>{{Citation|title=Captive State (2019)|url=https://www.rottentomatoes.com/m/captive_state|language=en|access-date=2019-01-22}}</ref>。
