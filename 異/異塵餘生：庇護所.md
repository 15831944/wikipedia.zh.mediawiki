{{Vgname|異塵餘生：庇護所|en=Fallout Shelter}}是一款由{{tsl|en|Bethesda Game Studios|貝塞斯達遊戲工作室}}及{{link-en|Behaviour Interactive}}合作開發，並由[[貝塞斯達軟件|貝塞斯達軟件]]發行的免費[[建造與經營模擬遊戲|模擬遊戲]]。作為《[[異塵餘生系列|異塵餘生]]》系列的一部分，貝塞斯達軟件先於2015年6月14日發行本作至[[iOS|iOS]]平台，其後擴展至[[Android|Android]]平台、[[Microsoft_Windows|Microsoft Windows]]、[[Xbox_One|Xbox One]]、[[PlayStation_4|PlayStation 4]]及[[任天堂Switch|任天堂Switch]]。

在發行後，本作獲得了褒貶不一的評價。評論家喜歡本作延續了《異塵餘生》宇宙的設定、核心玩法及其視覺風格。有部分批評則指本作缺乏深度、使用了不必要的[[微收費|微收費]]，以及缺乏結局。本作在發行後的兩星期錄得遊戲內的微收費額度為510萬美元。

== 開發及發行 ==
2009年6月，[[Engadget|Engadget]]的記者採訪了貝塞斯達軟體的{{tsl|en|Todd Howard (video game designer)|托德·霍華德}}，而當記者談及會否有一款適用於iOS平台的《異塵餘生》遊戲時，霍華德則表示《異塵餘生》世界「具有足夠的獨特性及可以移植至任何平台」，並揭示了遊戲製作團隊曾經進行這類研究<ref>{{Cite web |url = https://www.engadget.com/2009/06/05/bethesda-has-designed-iphone-versions-of-fallout-3/ |title = Bethesda has designed iPhone versions of Fallout 3 |author = Justin McElroy |date = 2009-06-05 |publisher = Engadget |language = en |accessdate = 2018-06-20 |deadurl = no |archiveurl = https://web.archive.org/web/20150924044029/http://www.engadget.com/2009/06/05/bethesda-has-designed-iphone-versions-of-fallout-3/ |archivedate = 2015-09-24 }}</ref>。同年11月5日，當時在[[id_Software|id Software]]工作的[[約翰·卡馬克|約翰·卡馬克]]聲稱自己擁有《異塵餘生》的iPhone遊戲的內部[[概念驗證|概念驗證]]，更表示自己可能會親自參與遊戲製作，儘管他當時正在參與其他項目<ref name="Kotaku-BC@091105">{{Cite web |url = https://kotaku.com/5397350/carmack-on-iphone-fallout-quake-live-and-elves-and-orcs |title = Carmack on iPhone Fallout, Quake Live and Elves and Orcs |author = Brian Crecente |date = 2009-11-05 |publisher = Kotaku |language = en |accessdate = 2018-06-20 |deadurl = no |archiveurl = https://web.archive.org/web/20180621105542/https://kotaku.com/5397350/carmack-on-iphone-fallout-quake-live-and-elves-and-orcs |archivedate = 2018-06-21 }}</ref>。約翰·卡馬克其後補充，表示自己至少會提供代碼<ref name="Kotaku-BC@091105"/>。

貝塞斯達在2015年6月14日舉行的[[E3|E3遊戲展]]上公佈本作，並確認本作是一款免費遊戲，而且會在同日於iOS平台上發行<ref name="GI-KH@150614"/><ref name="巴哈@150615"/>。本作是由{{tsl|en|Bethesda Game Studios|貝塞斯達遊戲工作室}}與{{link-en|Behaviour Interactive}}合作開發<ref>{{Cite web |url = https://venturebeat.com/2015/07/17/the-deanbeat-fallout-shelter-is-a-rare-free-to-play-hardcore-mobile-gaming-hit/ |title = The DeanBeat: Fallout Shelter is a rare free-to-play hardcore mobile gaming hit |author = Dean Takahashi |date = 2015-07-17 |publisher = GamesBeat |language = en |accessdate = 2018-06-20 |deadurl = no |archiveurl = https://web.archive.org/web/20170726102120/https://venturebeat.com/2015/07/17/the-deanbeat-fallout-shelter-is-a-rare-free-to-play-hardcore-mobile-gaming-hit/ |archivedate = 2017-07-26 }}</ref>，使用[[Unity_(遊戲引擎)|Unity]]遊戲引擎製作而成<ref>{{Cite web |url = https://www.theguardian.com/technology/2015/jul/06/unity-indie-gamings-biggest-engine-john-riccitiello |title = Unity - does indie gaming's biggest engine have an image problem? |author = Laura Kate Dale |date = 2015-07-06 |publisher = The Guardian |language = en |accessdate = 2018-06-20 |deadurl = no |archiveurl = https://web.archive.org/web/20180621105542/https://www.theguardian.com/technology/2015/jul/06/unity-indie-gamings-biggest-engine-john-riccitiello |archivedate = 2018-06-21 }}</ref>。2015年8月13日，本作亦於Android平台上發行<ref>{{Cite web |url = https://www.gamespot.com/articles/fallout-shelter-available-now-on-android/1100-6429720/ |title = Fallout Shelter Available Now on Android |author = Eddie Makuch |date = 2015-08-13 |publisher = GameSpot |language = en |accessdate = 2018-06-20 |deadurl = no |archiveurl = https://web.archive.org/web/20170709193712/https://www.gamespot.com/articles/fallout-shelter-available-now-on-android/1100-6429720/ |archivedate = 2017-07-09 }}</ref>。

根據貝塞斯達副總裁皮特·漢斯（Pete Hines）的說法，本作的靈感來自《{{tsl|en|Little Computer People|小電腦人}}》、《[[Progress_Quest|Progress Quest]]》、《{{link-en|X-COM|X-COM}}》、《[[模擬城市系列|模擬城市]]》及《{{tsl|en|FTL: Faster Than Light|超越光速}}》等的電子遊戲<ref>{{Cite web |url = https://kotaku.com/fallout-shelter-announced-out-tonight-1711317184 |title = Fallout Shelter Announced, Out Tonight |author = Luke Plunkett |date = 2015-06-14 |publisher = Kotaku |language = en |accessdate = 2018-06-20 |deadurl = no |archiveurl = https://web.archive.org/web/20170622062259/http://kotaku.com/fallout-shelter-announced-out-tonight-1711317184 |archivedate = 2017-06-22 }}</ref><ref>{{Cite web |url = https://www.cnet.com/news/surprise-fallout-mobile-game-arrives-for-ios/ |title = Surprise! 評論家普遍喜歡本作的玩法，但同時批評本作缺乏深度。[[Destructoid|Destructoid]]的克里斯·卡特（Chris Carter）表示自己不會長期玩這款遊戲，但卻值得投入時間<ref name="Destructoid-CC@150615"/>。[[IGN|IGN]]的賈斯汀·戴維斯（Justin Davis）指本作需要設下終局目標或一些可以導致終局的稀有資源，才能令玩家抱有期待並向著這些目標前進<ref name="IGN-JD@150626"/>。[[紐約每日新聞|紐約每日新聞]]的埃比尼澤·塞繆爾（Ebenezer Samuel）指本作雖然不是一款具有深度的角色扮演遊戲，但仍能給予玩家一種有趣的體驗<ref name="NYDN-ES@150616"/>。

部份評論家亦讚賞本作的視覺效果。[[Game_Informer|Game Informer]]的丹尼爾·塔克（Daniel Tack）指出本作的視覺效果十分符合避難所小子的美學<ref name="GI-DT@150701"/>。Destructoid的克里斯·卡特亦表示本作在視覺上較其他同類型遊戲更令人印象深刻<ref name="Destructoid-CC@150615"/>。[[Gamezebo|Gamezebo]]的傑森·福克納（Jason Faulkner）則表示本作的圖形令人印象深刻，更說「不僅是避難所居民呈現出濃厚的《異塵餘生》風格，避難所本身也非常漂亮」<ref NAME="GZ-JF@150618"/>。
