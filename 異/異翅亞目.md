'''異翅亞目'''（學名：{{lang|la|Heteroptera}}）是[[昆蟲綱|昆蟲綱]][[半翅目|半翅目]]底下三個亞目的其中之一，全世界約有超過4萬種。異翅亞目包括[[椿象|椿象]]、[[水黽|水黽]]、[[紅娘華|紅娘華]]、[[水螳螂|水螳螂]]等昆蟲。由於透過[[分子系統解析|分子系統解析]]，發現異翅亞目的物種與[[頸吻群|頸吻群]]的物種近似，有討論指這兩類物種的再分類有商榷的餘地。

== 分類的問題 ==
在現代的分類，「異翅亞目」這名詞有以下兩個完全不同的意思：
* 一般來說，異翅亞目的昆蟲都會被列為[[昆蟲綱|昆蟲綱]][[半翅目|半翅目]]底下三個亞目的其中一個；
* 或會被列為[[半翅目|半翅目]][[前喙亞目|前喙亞目]](Prosorrhyncha)底下一個無級別(rankless)的非[[林奈|林奈]][[單子群|單子群]]（monophyletic grouping）。

[[水黽下目|水黽]]（Gerromorpha）及[[隱角椿象下目|隱角椿象]]（Nepomorpha）兩個下目包含了這個亞目大多數的水生及半水生物種。餘下的物種，都被歸入[[臭蟲下目|臭蟲下目]]（Cimicomorpha）及[[椿象下目|椿象下目]]（Pentatomomorpha）中。因此，亦有其他分類把本亞目的物種先分為[[陸棲椿下目|陸棲椿下目]]（Infraorder GEOCORISA）及[[兩棲椿象下目|兩棲椿象下目]]（Infraorder AMPHIBICORISA），然後再細分下去。<ref>譯名來源：[http://www.tari.gov.tw/%E9%9C%A7%E5%B3%B0%E7%B8%BD%E6%89%80/%E6%87%89%E7%94%A8%E5%8B%95%E7%89%A9%E7%B3%BB/%E7%A0%94%E7%A9%B6%E5%AE%A4/%E5%88%86%E9%A1%9E/%E6%98%86%E8%9F%B2%E7%A7%91%E5%90%8D/%E5%8D%8A%E7%BF%85%E7%9B%AE.htm Class INSECTA 台灣行政院農業委員會農業試驗所：應用動物學首頁：昆蟲綱：有翅亞網：半翅目] {{webarchive|url=https://web.archive.org/web/20070928042222/http://www.tari.gov.tw/%E9%9C%A7%E5%B3%B0%E7%B8%BD%E6%89%80/%E6%87%89%E7%94%A8%E5%8B%95%E7%89%A9%E7%B3%BB/%E7%A0%94%E7%A9%B6%E5%AE%A4/%E5%88%86%E9%A1%9E/%E6%98%86%E8%9F%B2%E7%A7%91%E5%90%8D/%E5%8D%8A%E7%BF%85%E7%9B%AE.htm |date=2007-09-28 }}，在這個中的分類與本條目完全不同。</ref>
