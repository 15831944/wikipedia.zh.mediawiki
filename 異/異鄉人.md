《'''異鄉人'''》（原名：{{lang|fr|L'Étranger}}，又译为“'''局外人'''”），[[卡缪|卡缪]]之小說，1942年發出版，1957年[[諾貝爾文學獎|諾貝爾文學獎]]得獎作品。卡謬藉此作表達人是存在於孤立疏離之間以及生活荒謬。此書之主題和觀點經常有學者引用作為卡繆[[荒謬主義|荒謬主義]]和[[存在主義|存在主義]]哲學之例子。

此書主角是莫梭（Meursault），一個冷漠的法國[[阿爾及利亞|阿爾及利亞]]人，他冷漠地參加了他母親的葬禮。幾天後，他殺死了一名[[阿爾及爾|阿爾及爾]]的阿拉伯男子，之後被審判並被判處死刑。故事分為兩部分，以莫梭之[[第一人稱|第一人稱]]敘事。卡繆於1955年1月總結，《異鄉人》中的主角是擺脫社會習俗的英雄：「我很久以前總結過『異鄉人』這句話，我承認這是一個非常自相矛盾的說法：『在我們的社會中，任何不會在母親的葬禮上哭泣的男人都有可能被判處死刑。』我只是意味著我的書中的英雄被譴責，因為他不玩遊戲。」<ref>{{cite book |last=Carroll |first=David |date= |title=Albert Camus the Algerian: Colonialism, Terrorism, Justice |location= |publisher=Columbia University Press |pages= 27}}</ref>

異鄉人的第一版，共印4400本，並不是即時就暢銷。但這部小說之後很受歡迎，部分原因是因為[[尚-保羅·沙特|沙特]]的文章《分析異鄉人》（''Explication de L'Etranger''）。<ref>{{Cite book|title = The Stranger (Albert Camus)|last = McCarthy|first = Patrick|publisher = Cambridge University Press|year = 2004|isbn = 0-521-8321-01|location = New York|pages = 12}}</ref>此小說長期是20世紀[[現實主義|現實主義]]或者存在主義[[文學|文學]]之經典，譯成許多其他語言。此小說在[[世界報20世紀百大書籍|世界報20世紀百大書籍]]中排名第一。

==劇情==
作品主要描述生活在[[阿爾及利亞|阿爾及利亞]]首都[[阿尔及尔|阿尔及尔]]的主人公莫梭收到一封来自养老院的[[電報|電報]]告知其母親的死訊。主人翁在母亲的葬礼上並沒有流露出傷心難過，反而無視於道德教條在[[葬禮|葬禮]]隔天與女友做愛。之後又继续着与从前一般无二的生活，直到被卷入朋友的麻烦事，枪杀了一个阿拉伯人。种种行為既無關於他是否不愛他母親，也無關他是否討厭那阿拉伯人。在等待審判的期間，作者用了很多篇幅凌亂片段地交代莫梭過往生活中的荒謬事件。最後在面對審判時，主角莫梭表现得满不在乎，当被问到杀人动机时，他回答：“都是太陽惹的禍”，期待着在人们的咒骂声中面对行刑。小說最後以荒謬的判決將其處死作為結束。

== 分析==
[[李怡|李怡]]認為，《異鄉人》寫出現代人孤獨，以及同社會疏離，開頭第一句：「今天，媽媽死了，也許是昨天，我不能確定。」就以冷淡的漠視，形塑出主人公莫梭內心與現實的衝突，並且《異鄉人》亦展示出，社會上人與人的關係是虛偽，許多行為都不是自主而是荒謬，莫梭因無法擺脫「獨立個人」的困窘，因此他注定成為與社會疏離的局外人，也就是永遠的異鄉人。<ref>{{cite web |author1=[[李怡|李怡]] |title=《異鄉人》與荒謬主義|url = https://hk.lifestyle.appledaily.com/lifestyle/columnist/daily/article/20091119/13434651|agency=[[蘋果日報_(香港)|蘋果日報]]|date=2009-11-19|accessdate=2019-02-16|}}</ref>

== 电影版 ==
1967年在意大利被拍摄成电影。电影名为《异乡人》（[[意大利语|意大利语]]：Lo straniero）。该片导演是[[意大利|意大利]]名導[[卢奇诺·维斯康蒂|卢奇诺·维斯康蒂]]，《邮差总按两次铃》、《[[氣蓋山河|氣蓋山河]]》、《诅咒》、《[[魂断威尼斯|魂断威尼斯]]》都是其作品。制片人[[迪诺·德·劳伦提斯|迪诺·德·劳伦提斯]]。出演者[[馬切洛·馬斯楚安尼|馬切洛·馬斯楚安尼]]、[[安娜·卡里娜|安娜·卡里娜]]等。
