'''异姓王'''，指与[[皇帝|皇帝]][[姓氏|姓氏]]不同的[[王爵|王爵]]，亦即不是皇帝[[宗室|宗室]]的[[诸侯|諸侯王]]。在东周、春秋、战国时期，[[封建|封建制度]]盛行，天子分封的大多数是异姓王。但是自从[[秦朝|秦]]、[[汉朝|汉]]统一王朝建立后，绝大多数非皇族的大臣最高只能得到[[侯爵|侯爵]]，而异姓封王则成为殊遇极少数重臣的特例。由于这种地位，各朝各代的异姓王中多有反叛、篡权之人，即使无心对抗朝廷的异姓王也往往成为中央政权针对的对象。异姓封王也因此经常成为中国历史上引人关注的政治焦点。

== 西汉 ==
{{main|消滅異姓王風潮}}
[[秦始皇|秦始皇]]统一中国后，废除[[封建制度_(中国)|封建]]，设立[[郡|郡]][[县|县]]，即皇子亦不得封王。[[秦朝|秦]]末[[陳勝|陳勝]]、[[吳廣|吳廣]]發動了[[大澤之變|大澤之變]]後，抗秦起义风起云涌，實力軍人纷纷稱王。[[刘邦|汉高祖刘邦]]建立[[西汉|西汉]]後，一方面继续推行[[郡县制|郡县制]]，同时保留了一部[[分封制|分封国]]，大者为[[诸侯王|王]]，小者为[[侯|侯]]，分封给异姓功臣，主要的有七人：即（原齊王）楚王[[韩信|韩信]]、梁王[[彭越|彭越]]、淮南王[[英布|英布]]，赵王[[张耳_(汉朝)|张耳]]、燕王[[臧荼|臧荼]]（后为[[卢绾|卢绾]]）、长沙王[[吴芮|吴芮]]、[[韩王信|韩王信]]。

劉邦心中非常提防這些异姓王，於是稱帝不久，就開始磨刀霍霍，這些王国受到漢朝的攻擊，大多不能得到[[自然死亡|善終]]。赵王张耳病逝，赵国第二代赵王[[张敖|张敖]]因涉嫌行刺被问罪，王国除，降为宣平侯。燕王臧荼谋反，被攻灭，处死。后立的燕王[[卢绾|卢绾]]勾结匈奴，结果被刘邦察觉，发兵击之，卢绾率部属逃入匈奴，不久病死。楚王韩信被贬为淮阴侯，后被[[吕后|吕后]]处死。梁王彭越被贬，之后也被吕后处以[[醢刑|醢刑]]。淮南王英布此時被迫造反，刘邦抱病攻讨，英布退卻，被妹夫长沙王[[吴臣|吴臣]]所刺殺。韓王信擔心會被漢朝誅殺，便投降匈奴，約一同攻漢，後戰死。汉初唯一幸存的异姓王国只有吴芮的长沙国，因為勢力實在太小，漢朝沒有將它滅亡，得传五世，最後因无后而王国滅亡。

在刘邦滅亡了大部分异姓王之後，曾召集各刘姓王结下[[白马之盟|白马之盟]]，约定“非刘氏而王，天下共击之”，从制度上取消了汉朝再封异姓王的可行性。雖然劉邦死後，吕后[[訓政|訓政]]时，曾广封吕姓子弟為王，但是随着吕后死亡，吕姓諸王很快被[[元老|元老]]、[[大臣|大臣]]們給[[消滅諸呂|消滅]]。

== 东汉 ==
东汉建立后，承袭西汉[[七国之乱|七国之乱]]后对于汉廷对于诸侯王国的严格控制，即使同姓王都严格约束，更不设异姓王。但是这一制度到东汉末年最终被打破，实际控制了汉献帝的丞相[[曹操|曹操]]进魏公之后再封魏王，并且一举打破王国不得超过一郡的汉制，魏国占据朝廷核心地区十郡之地，声势超过了汉廷[[傀儡皇帝|傀儡皇帝]]。曹操死后，一年之内嗣位魏王的曹丕即篡汉自立，建立[[曹魏|曹魏]]。

== 曹魏 ==
曹魏末年，掌权的[[司马昭|司马昭]]效仿曹操，封晋王。司马昭死后，其子[[司马炎|司马炎]]立刻篡魏自立，建立[[西晋|西晋]]。

== 唐 ==
历经[[南北朝|南北朝]]诸多变乱，[[唐朝|唐]]初统一后只有极少数异姓王，但是之后逐渐开始增多。唐中叶，东平郡王[[安禄山|安禄山]]反叛，开始[[安史之乱|安史之乱]]，成为唐朝由盛转衰的分水岭。此后国家名虽统一，实际割据严重，各地军阀亦有异姓王，最终东平王[[朱全忠|朱全忠]]控制朝廷，加封梁王，数年后篡唐自立，建立后梁。

== 宋以后 ==
五代十国以后，[[北宋|北宋]]除延续后周几个异姓王之外，較少封异姓王。[[王爵|王爵]]多为功臣死后[[追封|追封]]的手段，故不再成为对于政权的威胁。異姓王一般為藩屬如南平王、安南王、大理國王、定難軍夏王、武威郡王、齊郡王，外戚如常山郡王、秦安僖王、沂王、樂平僖靖王，功臣如廣陽郡王童貫、同安郡王張邦昌、蘄忠武王韓世忠、循忠烈王張俊、信武順王吳璘、和武恭王楊存中、申忠獻王秦檜。

== 清 ==
清代，異姓王非常少，多为[[外藩蒙古|外藩蒙古]]各部及[[回部|回部]]首领。汉族勋贵封王仅见于清初[[孔有德|孔有德]]、[[尚可喜|尚可喜]]、[[耿仲明|耿仲明]]、[[吴三桂|吴三桂]]、[[孙可望|孙可望]]五家，[[黄芳度|黄芳度]]则在死后[[追赠|追赠]]王爵。但事實上只有[[吳三桂|吳三桂]]、[[尚可喜|尚可喜]]、[[耿仲明|耿仲明]]三王有實力，孔有德早死，孙可望稱王不久即退位。吴三桂與[[尚可喜|尚可喜]]之子[[尚之信|尚之信]]、[[耿仲明|耿仲明]]之孫[[耿精忠|耿精忠]]谋反，引发[[三藩之乱|三藩之乱]]，最后失败，至此再无汉人藩王。

[[满族|满族]]勋贵封王仅有两例，一为[[扬古利|扬古利]]在清初追封武勋王，其后[[福康安|福康安]]在乾隆年间因战功破例晋封[[贝子|贝子]]，死后追赠嘉勇郡王。

起事反清的[[太平天国|太平天国]]封了很多异姓王。

[[中華民國|中華民國]]建立後，[[辮軍復辟|辮軍復辟]]时，[[溥儀|溥儀]]封[[张勋|张勋]]為'''忠勇亲王'''。
