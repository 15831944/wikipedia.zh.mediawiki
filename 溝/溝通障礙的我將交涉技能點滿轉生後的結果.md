== 故事簡介 ==
'''森岡弘人'''因一場交通意外死去，醒來時發現面前出現了一名自稱女神的美女！！由於弘人生前在遊玩一款名叫《永恆幻境》的遊戲而獲得轉生到遊戲世界的權利，並將以自己生前的不幸所換來的額外技能點數全加進「交涉術」，展開超過激鬼畜異世界奇幻冒險。

=== 主要人物 ===
; {{Nihongo|希洛特·齊格里特|ヒロト・ジークリッド|Hiroto Jikuriddo}}／前世舊名：{{nihongo|森岡弘人|森岡 弘人（もりおか ひろと）}}
: 本作主角。本身有著嚴重的溝通障礙，前世因救自己的青梅竹馬而遭大貨車輾斃，但因自己是遊戲《永恆幻境》的最資深玩家而被女神邀請，並轉生到異世界。
: 由於使用技能點數將技能「交涉術」點滿的緣故，進而獲得了「魅惑」和「隸屬化」等技能，因此能將同性、異性或魔物魅惑，進而向他們下達一定程度上的指令，或是將魔物馴服成自己的護衛獸。
: 小說第一卷時，因為了取得菲莉亞妮絲所擁有的技能「【神聖】劍技」，而漸漸克服溝通障礙的老毛病。
: 第二卷因母親打算生第二胎時，卻因她自己的身體因素而面臨死亡的威脅，使得他用光剩餘技能點數，隻身前往龍的巢穴尋找秘藥的最後一樣材料，途中得到其他熟人的協助而成功進入深處並遭遇優希婭，因優希婭不打算將材料交給他而發生戰鬥，卻因自身不敵而落敗，在最後因莉歐娜的亂入而取得材料，並將優希婭變成護衛獸。

; {{Nihongo|塞拉·戚風|セラ・ハリケーン|Sela Hariken}}
: [[人魚|人魚]]。儘管只有7歲，但卻有著宛如高中生的外表<ref>推測有些種族的身體成長速度比實際年齡的成長速度更快。</ref>。
: 在傳教路上經過希洛特家時，由於進入了希洛特的魅惑技能範圍而遭到魅惑，進而對希洛特授乳，使得希洛特獲得歌唱技能。
: 原本是位於伊修亞拉爾村的神殿中的歌姬，負責演唱獻給神的歌曲與表演祭祀舞蹈，卻不明原因的喪失了歌唱的能力，因此辭去歌姬的身份，目前在米賽爾教會擔任傳教士。

=== 惡徒 ===
; {{Nihongo|帕梅菈·布蘭尼|パメラ・ブランニー|Pamela Buranni}}
: 隸屬盜賊公會的山賊集團「毒薔薇」的首領。
: 在希洛特還是嬰兒時，曾闖入他家意圖行竊，卻被希洛特的魅惑能力魅惑，進而替希洛特授乳，最後因自己魔力耗盡而昏倒，並因為被嚇到而將當時的希洛特稱為「'''惡魔之子'''」。在希洛特8歲時又再度遇上希洛特並被擊倒，最後與希洛特約定不再從事不法的工作。

=== 冒險者公會 ===
; {{Nihongo|里克·布魯瓦斯|リック・ブルワーズ|Likku Buruwazu}}
: 米賽爾冒險者公會會長。是一名理著光頭、渾身肌肉的男性，在遊戲時期被玩家稱為「'''布魯大哥'''」。

; {{Nihongo|夏莉·布魯瓦斯||Xiali Buluwasi}}
: 米賽爾冒險者公會接待小姐，里克的妹妹。是一名有著灰金色頭髮的女性，在遊戲時期有著不少男性玩家粉絲，但可能也是因為她有著一對爆乳的緣故。
: 因發高燒而在家靜養，後來在希洛特前往探望她時，被希洛特的魅惑能力魅惑，進而替他授乳。

; {{Nihongo|蘇珊娜·蘇·阿德爾海德|スザンナ・ス・アデルハイド|Suzanna Su Aderuhaido}}
: 冒險者公會的[[特務|特務]]。實力強悍，被希洛特形容為「即使被綁在椅子上，也能瞬間擺平傑斯塔等人」。
: 因某些原因而在希洛特家擔任女僕，在辭去女僕工作前，帶著希洛特前去救援被傑斯塔等人下藥迷昏的希露美妲與無名，並在讓希洛特護衛希露美妲等人時，將傑斯塔等人帶出去進行懲罰。

=== 其他人物 ===
; {{Nihongo|莉莉姆|リリム}}
: 魔王。多年前曾和優希婭的母親戰鬥過，後來被封印。之後再度復活時，利用古魯德公爵的野心控制他，並將他變為不死者，與希洛特交戰時，一度藉由控制命的身體佔上風，並吸取了希洛特的生命力，最後卻被優希婭擊敗，但並未確實死亡，目前下落不明。
