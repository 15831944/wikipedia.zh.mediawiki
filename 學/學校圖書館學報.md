{{作品名稱|學校圖書館學報|name=''School Library Journal''}}是一本主要以經常接觸年輕人的[[學校|學校]][[圖書館員|圖書館員]]、媒體專家或[[公共圖書館|公共圖書館]]員等為讀者導向之[[美國|美國]]月刊[[雜誌|雜誌]]。其內容廣納許多主題範圍的文章，並較著重於科技和多媒體方面的資訊。雜誌亦包含許多書本的評論，並依適讀者分類為「學齡前至四年級」、「五年級以上」以及「青少年」三群；評論的書本對象同時包括虛構與非虛構作品、[[視覺文學|圖像小說]]以及多媒體和數位資源。此外，雜誌中也常有適用於圖書館員之專業參考書籍的評論文章。<ref>{{cite news|last=Alvarez|first=Alex|title=RBI Sells Library Journal and School Library Journal to Media Source Inc.|url=http://www.mediabistro.com/fishbowlny/rbi-sells-library-journal-and-school-library-journal-to-media-source-inc_b13900|newspaper=Fishbowl NY|date=March 1, 2010}}</ref>
