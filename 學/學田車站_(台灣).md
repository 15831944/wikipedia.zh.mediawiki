'''學田驛'''（'''學田車站'''），是位於[[台灣|台灣]][[臺中市|臺中市]][[烏日區|烏日區]]一座已廢除的[[臺中線|臺中線]][[鐵路車站|鐵路車站]]。

== 簡介 ==
本站是一座[[通勤|通勤]]用汽油車招呼站，係緣於1930年代初期[[臺灣總督府交通局鐵道部|臺灣總督府交通局鐵道部]]為了和[[公路|公路]][[巴士|巴士]]競爭，引進[[台鐵DR2300型柴油車|輕便汽油客車]]來行駛台灣北、中、南部的都圈近郊通勤區間，而廣設的汽油車招呼站之一<ref name="台中都會圈的風景"/>。

== 車站構造 ==
不詳

== 利用狀況 ==
*已廢除，現無跡可尋。

== 車站週邊 ==
*[[烏日#歷史|頂朥胥]]

== 歷史 ==
*1934年12月1日：設立「'''學田驛'''」，僅停汽油車。
*1942年：因[[太平洋戰爭|太平洋戰爭]]時汽油短缺，汽油車停駛。
