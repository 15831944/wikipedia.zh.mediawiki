{{otheruses}}
'''字典'''是為单[[汉字|字]]提供[[音韻|音韻]]、意思[[解釋|解釋]]、例句、用法等等的[[工具書|工具書]]。使用[[字母|字母]]文字作爲文字的人羣沒有字典這個概念。<ref>多数国家所用的全是拼音文字，所有詞都是由字母所拼在一起，所以最基本的單位只有詞。在使用表意文字的地區，字常常是可以單獨使用，每一個字有各自的意義。把兩個到三個字連在一起來使用的稱為詞。因此，才有字典及詞典兩個稱呼。</ref>'''字典'''收字為主，亦會收詞。'''[[詞典|詞典]]'''或'''[[辭典|辭典]]'''收詞為主，也會收字。

=== 中國 ===
[[File:Shuowen.jpg|thumb]]》十五卷]]
東方最早的字典可算是在公元30年-124年，[[漢朝|漢朝]][[許慎|許慎]]編寫《[[說文解字|說文解字]]》，創立了[[六書|六書]]理論，制定了中文字[[部首|部首]]的基礎，是字書中的佼佼者。（說文解字應為字書，非字典）

1190年，即[[西夏|西夏]][[乾祐_(西夏)|乾佑]]庚戍二十一年，[[黨項|黨項人]][[骨勒茂|骨勒茂]]才完成了[[西夏|西夏]]的第一部[[西夏文|西夏文]][[中文|中文]]雙語字典－《[[番漢合時掌中珠|番漢合時掌中珠]]》，成為考古學家翻譯西夏文的依據。

在1716年（[[康熙|康熙]]五十五年），第一部正名為字典的《[[康熙字典|康熙字典]]》正式面世。當中除了列舉字的出處外，還羅列《[[唐韻|唐韻]]》、《[[廣韻|廣韻]]》、《[[集韻|集韻]]》、《[[韻會|韻會]]》、《[[正韻|正韻]]》等等的反切，並對同音切語加以歸併。

1815年，[[英國|英國]]傳教士[[馬禮遜|馬禮遜]]在[[澳門|澳門]]為了翻譯工作，編寫了中國第一部英語學習字典《[[華英字典|華英字典]]》。[[馬禮遜|馬禮遜]]在倫敦時候，曾經得到一名中國人的教導下學習一年漢語，抵達[[廣東|廣東]]後，曾翻譯《[[三字經|三字經]]》及《[[大學_(經傳)|大學]]》，並且編寫過漢語語法書籍，所以對中國文化及語言有一定了解。因此在《[[華英字典|華英字典]]》可以找到很多出自《[[紅樓夢|紅樓夢]]》和《[[論語|論語]]》的例句。《[[華英字典|華英字典]]》是世界上第一本英漢－漢英對照的字典，篇幅大內容豐，有豐富的例句及解釋，並收錄大量成語、俗語。1844年[[衛三畏|衛三畏]]的《[[英華韻府歷階|英華韻府歷階]]》及1847年[[麥都思|麥都思]]的《[[英漢字典|英漢字典]]》都把它當作參照基礎。

1866年，[[德國|德國]]傳教士[[羅存德|羅存德]]在[[香港|香港]]出版一部兩卷本的《[[英華字典|英華字典]]》，可算是香港最早的雙語字典。[[羅存德|羅存德]]在1848年到香港傳福音，於1853年成為香港的中國福音傳道會的主要負責人。他曾編寫過《[[英話文法小引|英話文法小引]]》及《[[英華行篋便覽|英華行篋便覽]]》。

1953年，《[[新华字典|新华字典]]》出版。

=== 越南 ===
1624年左右，[[法國|法國]]的傳教士Loutek（Alexandre de Rhodes）來到[[越南|越南]]。開始使用羅馬字母來拼寫[[越南語|越南語]]，在1651年出版第一本《越南-葡萄牙-拉丁語字典》（Dictionarium Annamaticum， Lusitanum et Latinum）。[[羅馬字|羅馬字]]自從17世紀傳入[[越南|越南]]了，一直長期為少數的天主教徒所使用。這套羅馬字系統後來經過多番修改，直到1986年[[胡大緯|胡大緯]]宣佈了越南.獨立的成為唯一的國定書寫系統。

== 編纂理念 ==
{{See|詞典#編纂理念}}
[[規範主義|規範主義]]或[[描述主義|描述主義]]從來是編纂字典的兩個重要派別。

[[中國|中國]]正式使用「字典」一詞始於《[[康熙字典|康熙字典]]》。根據《[[說文解字|說文解字]]》，'''典'''是[[五帝|五帝]]的書本，神聖尊貴的大冊。

在香港，漢字的[[粵語|粵語]]讀音應該從[[反切|反切]]還是從俗的爭議即是一例。⽀持「以古代辭書切音為準」的，一般稱為「從切」派；⽀持「以現代通行讀音為準」的，一般稱為「從眾」派。相關爭議引發了[[粵語正音運動|粵語正音運動]]，並導致市場上大量字詞典改從[[何文匯|何文匯]]等人提倡的「正音」，不過也有不少專家不承認其「正音」，並批評所謂「正音運動」其實是「滅音運動」。<ref>{{Cite book|author=石見田|url=http://www.savepropercantonese.com|title=《解·救·正讀》|isbn=978-988-8058-42-6|language=zh-hk}}</ref>

== 字的排序 ==
當'''字典'''所用的是[[拼音文字|拼音文字]]，字詞多數以拼音字母來排序。規律是以最常用音讀為首、訓讀為次。

中文字是表意文字，傳統上字書排列方式是根據[[部首|部首]]（如《康熙字典》），韻書則以韻部分類（如《廣韻》）。部首在[[許慎|許慎]]創立時，共有540個，後來不斷歸納淘汰，《辭源》中的部首只剩下240餘個，到了《漢語大辭典》只有200餘個。在1925年之後，部分中文字典開始使用由[[王雲五|王雲五]]發明的[[四角號碼|四角號碼]]檢字法。

現在大部分[[簡體字|簡體字]]的漢語字典的字詞是根據[[普通話|普通話]]音標來排列的，而[[繁體中文|繁體中文]]字典則有部首、漢語拼音、[[注音符號|注音符號]]等不同的排序方式。

== 標音系統 ==
{{See also|詞典#標音系統}}
各地民族都為自己的語言發展一套音標系統。中文受[[梵文|梵文]]啟發，創造出的最早的音標系統是[[反切|反切]]，即第一個字的聲加上第二個字的韻，再轉調而成，如「龍，盧容切」。也有標註直音字的，如「龍，从音籠」。

《[[現代漢語詞典|現代漢語詞典]]》使用[[普通話|普通話]]的羅馬字母及4聲調符號（[[汉语拼音|汉语拼音]]）作為標音系統，[[台灣|台灣]]出版的辭典則以[[注音符號|注音符號]]標音。

== 類別 ==
===雙語字典===
雙語字典有單向及雙向之分。主要是把[[外語|外語]]字詞用[[母語|母語]]來解釋，或者把母語的字詞用外語來解釋。雙向雙語字典基本上是兩本單向雙語字典合併在一起，更同時使用外語及[[母語|母語]]作解釋。

===學生字典===
針對學生學習的要求，字典收詞方面、意思解釋及例句選材方面儘量以容易明白為原則，並且加入插圖幫助解釋。

== 其他 ==
對於拼音文字，句子結構很重要，[[文法|文法]]字典更是不可少。為了不同需要，更有[[諺語|諺語]]字典、分類詞詞典、[[同義詞|同義詞]]詞典、[[反義詞|反義詞]]詞典、同韻尾詞典等等。對於中文，更有[[成語|成語]]字典。

百科全书式学者[[钱锺书|钱锺书]]愛读字典，在旅途中“怡然自得，手不釋卷”，輪船上惟以《[[約翰生字典|約翰生字典]]》自隨<ref>[[鄒文海|鄒文海]]：《憶錢鍾書》</ref>；又例如《[[韦氏大词典|韦氏大词典]]》，他竟能读三遍。他有一部40年代末流行的《[[简明牛津字典|简明牛津字典]]》，在每一页的空白处都密密麻麻写满批注。<ref>孔庆茂編：《钱锺书与杨绛》</ref>

== 常見內容 ==
為了提供更多功能，辭典會加入其他內容。例如：地方名稱、常見人名列表、度量衡、其他語言的字母表等等。

== 形式 ==
字典常常以不同的型態出現。袖珍版本為了方便攜帶。袋裝版本容易放在口袋裡。精簡版本只收錄非常常用的字詞。軟皮版本價錢便宜一點。硬皮版本耐用一點。電子字典方便查詢。網上字典方便連接其他系統。更有發聲字典讓學生容易掌握發音。
