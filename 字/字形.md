'''字形'''（{{lang-en|'''glyph'''}}），又稱'''字圖'''或'''書形'''，是指[[文字|字]]的形体。[[中华人民共和国国家标准|中华人民共和国国家标准]][[GB/T_16964|GB/T 16964]]《信息技术·字型信息交换》中定义字形为“'''一个可以辨认的抽象的图形符号，它不依赖于任何特定的设计'''”。在[[語言學|語言學]]中，[[字|字]]（character）是[[語意|語意]]的最基本單位，即[[語素|語素]]；字形是指為了表達這個意義的具體表達。同一字可以有不同的字形，而不影響其表達的意思，例如[[拉丁字母|拉丁字母]]第一個字母可以寫作{{lang|en|a}}或{{lang|en|ɑ}}，漢字中的“-{強／强}-”、“-{戶／户／戸}-”。

== 東方（漢字）字形大 ==
=== 形音義 ===
每個漢字均有三個屬性：形狀（形）、聲韻（音）、意義（義），統稱「形音義」。

=== 新舊字形 ===
{{Main|新字形}}

[[中國大陸|中國大陸]]於1965年出版的《[[印刷通用漢字字形表|印刷通用漢字字形表]]》規定了印刷字形的標準，在表中規定的字形俗稱為''新字形''，之前通用的字形則稱為''舊字形''<ref>http://www.yywzw.com/show.aspx?cid=114&id=613</ref>。其字形標準於1988年由《[[現代漢語通用字表|現代漢語通用字表]]》取代。

由於大部分新字形的筆劃都較舊字形少，它們很多時會被錯認為[[簡化字|簡化字]]。事實上新字形除了適用於簡化字，還適用於繁體字，例如「<span lang="zh-cn">-{過}-（-{过}-）</span>」、「<span lang="zh-cn">-{編}-（-{编}-）</span>」；較接近傳統的台灣寫法是「<span lang="zh-tw">-{過}-</span>」、「<span lang="zh-tw">-{編}-</span>」。在中國大陸廢除之異體字，於收入[[中日韓統一表意文字|中日韓統一表意文字]]時亦會在不會產生[[重形字|重形字]]的情況下根據新字形調整，稱為「無重形GB化」，例如「<span lang="zh-cn">-{諮}-（-{咨}-）</span>」、「<span lang="zh-cn">-{僱}-（-{雇}-）</span>」；較接近傳統的台灣寫法是「<span lang="zh-tw">-{諮}-</span>」、「<span lang="zh-tw">-{僱}-</span>」，Unicode Code Chart[http://www.unicode.org/charts/] 所示的就是這種字形。原则上[[Unicode|Unicode]]/[[ISO_10646|ISO 10646]]只对字，而非字形[[编码|编码]]。[[Microsoft_Windows|Microsoft Windows]] [[簡體中文|簡體中文]]版中[[宋體|宋體]]Simsun是中國大陸新字形Font，但是卻也包含不少舊字形漢字。其實這些舊字形漢字並非是宋體新舊漢字同時收錄，舊字形是來源於Unicode/ISO 10646中臺灣韓國日本區域的漢字，臺灣韓國日本區域的某些漢字則是中國大陸的舊字形。

下面所列舉的新舊字形，在Unicode/ISO 10646中，是中國大陸跟香港、臺灣、日本、韓國同源漢字不同形的關係，對於中國大陸而言，則是新舊字形的關係。中國大陸並沒有給舊字形編碼，給舊字形編碼的主要是臺灣和日本區域。

=== 字形標準 ===
{{See also|異體字}}

以上所稱字形分別通常只是很小分別，筆劃差異在一兩筆之間。差異很大的通常歸類為[[異體字|異體字]]，中國大陸標準為《[[第一批異體字整理表|第一批異體字整理表]]》。和中國大陸不同，香港和台灣將同一字的不同的字形，即使只有很小分別，都歸類為[[異體字|異體字]]。香港教育界所據標準為[[常用字字形表|常用字字形表]]。台灣所據標準為[[常用國字標準字體表|常用國字標準字體表]]、[[次常用國字標準字體表|次常用國字標準字體表]]、[[罕用字體表|罕用字體表]]及[[異體字表|異體字表]]。

== 西方字形 ==
[[File:LatinAgraphemeVariations.svg|thumb]]「a」之兩種常用字形，一為[[手寫體|手寫體]]，一為[[印刷體|印刷體]]]]

最常舉的例子是貨幣「圓」之符號「[[$|$]]」，使用一豎、兩豎、穿透或是不穿透皆不影響該符號之意義。

和文字有關：
* 多於一個字母連寫時產生的[[合字|連寫體]]（Ligature）
* 同一字母，因寫於字（Word）的不同位置而改變形狀的[[變寫體|變寫體]]（例子：[[s|s]] vs [[長s|長s]]）
