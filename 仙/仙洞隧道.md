'''仙洞隧道'''是位於[[臺灣|臺灣]][[基隆市|基隆市]][[中山區_(基隆市)|中山區]]的[[鐵路隧道|鐵路隧道]]，興建於1917年，原供[[基隆臨港線|基隆臨港線]]使用，2004年停用、並拆除鐵軌<ref name=":0">{{Cite web|url=https://nchdb.boch.gov.tw/assets/advanceSearch/historicalBuilding/20060720000002|title=仙洞隧道|accessdate=|date=|last=|work=文化部文化資產局}}</ref>。目前列為[[臺灣歷史建築|歷史建築]]保護。

== 簡介 ==
仙洞隧道之開鑿是為提供鐵路貨運列車裝載散裝物及軍需品於[[基隆車站|基隆火車站]]與[[基隆港|基隆港]]西岸12至16號碼頭之間。<ref name=":0" />

== 構造 ==
隧道主體為磚石拱頂，側壁由粗石砌成。全長155[[米_(单位)|公尺]]，寬14公尺、高15公尺，為單線型式隧道。<ref name=":0" />

== 現況 ==
隧道內於鐵軌拆除後改鋪柏油路面。<ref name=":0" />

== 周邊 ==
基隆築港殉職者紀念碑、[[仙洞巖|仙洞巖]]、高遠新村港務局局長官舍等。<ref name=":0" />
