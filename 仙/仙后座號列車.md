'''「仙后座」'''（{{jpn|j=カシオペア|rm=kashiopea|f=y}}；{{lang-en|Cassiopeia}}）是隸屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）旗下的超級豪華[[寝车|臥鋪列車]]，在[[JR九州|JR九州]]推出[[九州七星號列車|九州七星號列車]]之前曾是日本最豪華的臥鋪列車。1999年7月16日首次運行，往返於[[東京|東京]]的[[上野|上野]]與[[北海道|北海道]]的[[札幌|札幌]]之間，不同於一般日式臥鋪列車，仙后座號是以豪華的美式風格為概念所設計，全列車都採私人包廂型式，雙層車體，只有餐車與展望室等設施才是開放式的。

因[[北海道新幹線|北海道新幹線]]會於2016年3月26日通車營運，新幹線與在來線共用[[海峽線|海峽線]][[青函隧道|青函隧道]]，其[[架空電車線|架空電車線]]電壓將於新幹線開始營運時從現行20,000[[伏特|伏特]]50Hz交流電，升壓至25,000伏特。除了[[JR貨物|JR貨物]]新採購的[[JR貨物EH800型電力機車|EH800型電力機車]]以外，其他電力機車與電聯車都無法在升壓後駛入海峽線。2015年9月15日，JR東日本與JR北海道宣佈將停駛仙后座號、{{link-ja|超級白鳥號列車|スーパー白鳥|超級白鳥號/白鳥號}}及[[玫瑰號列車|玫瑰號]]等行駛海峽線的在來線客運列車{{R|JRE_150914}}，並於12月公布了這些列車的最後行駛日期。<ref>{{Cite press release |language = ja |url = http://www.jreast.co.jp/press/2015/20151212.pdf |title = 北海道新幹線設備切替に伴う列車の運休について |publisher = JR東日本 |date = 2015-12-18 |accessdate = 2016-03-20}}</ref>

仙后座號的最終列車於2016年3月20日由札幌發車，隔日抵達上野。<ref>{{cite news |language = ja |url = http://www.saga-s.co.jp/news/national/10206/291347 |title = カシオペア上り最終列車が出発 |publisher = 佐賀新聞 |date = 2016-03-20 |accessdate = 2016-03-20 |deadurl = yes |archiveurl = https://web.archive.org/web/20160402090610/http://www.saga-s.co.jp/news/national/10206/291347 |archivedate = 2016-04-02 }}</ref>

如同日本另外一輛也是往返於上野～札幌間的豪華臥鋪列車[[北斗星號列車|北斗星號]]，仙后座號也是採用了北方天空中特別醒目的星宿——[[仙后座|仙后座]]來作為命名由來，以便強調它的北國風情。<br>
仙后座號平常是以兩天內往返各一趟的方式運行於上野與札幌兩個終點站之間，因此除了少數旺季與假日班次比較密集之外，它通常都是在每週日、二、五的傍晚由[[上野車站|上野車站]]發車，迎著夕陽往北方前進，穿過星空之下，於隔日上午抵達[[札幌車站|札幌車站]]，再於當天傍晚由札幌出發，於第三天的上午回到東京上野。由於JR東日本底下只擁有一列仙后座號客房數量有限，因此需要事先提早購票預約才不會遇到客滿的狀況。

在車內配置上，仙后座號全車都配置JR方面稱為「A寢台」的高級臥鋪包廂（等級相當於一般列車的「綠席」座位，也就是日本的頭等席之意）與特殊列車才有的「特A寢台」包廂。包廂共有三級，分別是套房（{{lang|ja|カシオペアスイート}}，{{lang-en|Cassiopeia Suite}}），包括全車只有一間的家庭房——仙后座豪華房（展望室），6間套房（採上下雙層設計形式），及一間位於3號車上層的套房，總計8間；一間頂級的仙后座豪華房（{{lang|ja|カシオペアデラックス}}，{{lang-en|Cassiopeia Deluxe}}），位於2號車下層；與79間仙后座雙人房（{{lang|ja|カシオペアツイン}}，{{lang-en|Cassiopeia Twin}}），其中一間是方便[[輪椅|輪椅]]乘客使用的[[無障礙空間|無障礙空間房]]（{{lang|ja|カシオペアコンパート}}，{{lang-en|Cassiopeia Compart}}）。除此之外列車的3號車是餐車，列車末端（面對札幌方向）的12號車是展望室車，餐車裡面每天提供號稱一流國際飯店水準的[[怀石料理|懷石御膳]]或[[法國料理|法國料理]]作為晚餐，但因為餐車座位有限，因此規定分一天三梯次輪流用餐，每梯次約1.5小時。至於想要在客房內悠閒享受晚餐的旅客，也可以選擇訂購只有此列車上才買得到的仙后座特別便當（{{lang|ja|カシオペアスペシャル弁当}}，{{lang-en|Cassiopeia Special Lunchbox}}）。

不過，豪華的代價是沉重的，仙后座號擁有日本國內最高昂的鐵路票價，縱使是最入門等級的仙后座雙人房，一人份單程票價仍然高達三萬多[[日圓|日圓]]。除此之外，因為全車都是雙人房的設計（必要時部分車廂可以加床變成三人以上），因此即使只有一人搭乘也需擔負兩人份的票價，使得來回一趟的車資，幾乎可超過兩個人一同搭乘國內線班機往返外加高級旅館的一夜住房費用，也難怪自從開始營運以來，仙后座號一直被JR當作是旗艦或看板列車般對待，經常在一些介紹日本鐵道的書刊上密集曝光。

總體上來說，由於[[新幹線|新幹線]]的開通、[[長途巴士|長途巴士]]的普及與國內航空業的發達，日本的臥鋪列車不再負擔長距離旅客運輸的主要功能，因此利用豪華的列車客房與精緻的餐飲服務來吸引觀光取向的乘客，是仙后座號這一類豪華臥鋪列車主要的存在訴求，也是傳統產業面對新式競爭時的一種變通。

2016年3月19日，取消定期仙后座號列車，於同年6月，JR東日本子公司[[びゅうトラベルサービス|びゅうトラベルサービス]]主導辦理，開行カシオペアクルーズ(Cassiopeia cruise)與カシオペア紀行(Cassiopeia 紀行)。

== 硬體 ==

仙后座號的列車車身等硬體設備是特別為了其特殊用途而另行打造，不屬於JR原有的車輛系列。車系代號'''E26'''的仙后座號共有12節不同詳細配置的車身（不包含動力機車部份），與一輛有必要時才加掛的臨時電源車。

=== 動力部份 ===
在動力牽引部份，由於仙后座號的運行範圍橫跨[[JR東日本|JR東日本]]與[[JR北海道|JR北海道]]兩家公司的營運範圍，共三種不同的設施路段，因此沿途它總共會更換三種不同的動力車頭，分別是：

==== 目前使用 ====
* [[日本国铁EF81型电力机车|EF81]]：EF81是隸屬於JR東日本旗下的交直流兩用電力機車，因仙后座列車退出常態運用後、JR東日本將旗下所有[[EF510|EF510]]-500番台售予[[JR貨物|JR貨物]]，重新啟用投入臨時列車カシオペアクルーズ與カシオペア 紀行的牽引任務。
* [[日本國鐵EF64型電力機車|日本國鐵EF64型電力機車]]: 因臨時列車カシオペアクルーズ途經[[上越線|上越線]]面臨陡坡路線、採用本型機關車擔任[[上野車站|上野車站]]→[[長岡車站|長岡車站]]間之前補機。
* [[JR貨物EH800型電力機車|JR貨物EH800型電力機車]]:因[[青函隧道|青函隧道]]架空線規格變更為25000V 50Hz，以往使用之[[日本國鐵ED79型電力機車|日本國鐵ED79型電力機車]]無法繼續使用(且車輛已全數退役)、故改以此型機關車牽引カシオペア 紀行行駛[[青森車站|青森車站]]→[[函館車站|函館車站]]間。
* [[JR貨物DF200型柴油機車|JR貨物DF200型柴油機車]]:因[[JR北海道|JR北海道]]旗下之[[日本國鐵DD51型柴油機車|日本國鐵DD51型柴油機車]]已全數退役、故改以此型機關車牽引カシオペア 紀行[[函館車站|函館車站]]→[[札幌車站|札幌車站]]間。

==== 過往使用 ====

* [[EF510|EF510]]：EF510是隸屬於JR東日本旗下的[[交直流兩用電力機車|交直流兩用電力機車]]，負責仙后座號在[[上野|上野]]到[[青森|青森]]路段間的牽引任務。2010年6月25日，EF510正式開始牽引仙后座號的載客列車<ref>[http://2nd-train.net/2-topic/201006/666.html {{lang|ja|寝台特急「カシオペア」 牽引機がEF510に置き換え}}] {{webarchive|url=https://archive.is/20120707002902/http://2nd-train.net/2-topic/201006/666.html |date=2012-07-07 }}</ref>。
* [[ED79|ED79]]：ED79是隸屬於JR北海道旗下的[[交流電力機車|交流電力機車]]，主要的用途是負責[[津輕海峽線|津輕海峽線]]路段的車輛牽引（也就是[[青函隧道|青函隧道]]專用車頭）。ED79不是仙后座號專用車頭，而是與包括[[海峽號列車|海峽號]]在內所有往返於青森與[[函館|函館]]間的列車，共用編號1、2、4到21號的許多輛ED79車頭，但ED79在牽引仙后座時，仍會在車頭前方掛上圓形的仙后座號專屬列車標誌。
* [[DD51|DD51]]：DD51是JR北海道旗下所使用的[[柴液機車|柴液機車]]，使用於該公司營運範圍內，尚未全部[[电气化铁路|鐵路電氣化]]的路線。仙后座號與其他的長途豪華列車，例如[[黃昏特快號列車|黃昏特快號]]（<span lang="ja">トワイライトエクスプレス</span>，Twilight Express）與[[北斗星號列車|北斗星號]]，共用編號1006、1054、1068、1083、1095、1100、1102、1137、1138、1140到1143，與1148號等的DD51車頭。DD51以兩輛重連的方式，牽引仙后座號往返於函館與札幌之間。
