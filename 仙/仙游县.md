'''仙游县'''（{{lang-cpx|Sing-iú gā̤ⁿ}}）位于中国[[福建省|福建省]][[莆田市|莆田市]]的西南部，是福建著名的侨乡。

== 地理 ==
仙游地处福建东南沿海中部，位于东经118°27'—118°56'，北纬25°11'—25°43'之间，濒临[[湄洲湾|湄洲湾]]，距福州130公里、厦门153公里、莆田市区44公里，与[[台湾岛|台湾岛]]隔海相望。境内“七山一水二分田”，山地203.7万亩、耕地41.8万亩、林地191.1万亩。

地形西北高、东南低，最高点西苑凤山石谷解海拔1803.3米，最低处枫亭海滨村海拔仅5米。主要有木兰溪、延寿溪、枫慈溪、粗溪和九溪5条溪流，流域面积1805平方公里；[[木兰溪|木兰溪]]为境内第一大河流，流经县境东南部，从莆田三江口注入兴化湾，为潮汐所吞纳。 

== 历史 ==
[[唐朝|唐朝]][[圣历|圣历]]二年（699年）析莆田县地西半部设清源县，属武荣州。唐玄宗[[天宝|天宝]]元年（742年），泉州易名为[[清源郡|清源郡]]，属[[岭南道|岭南道]]，别驾赵颐正以“县名同郡非便，奏请改之”，因境内有[[何氏九仙|何氏九兄弟]]在九鲤湖羽化登仙的民间传说，改为仙游县，隶属清源郡。

宋代兴化军管辖莆田、仙游、兴化三县<ref>《[[宋史|宋史]]·地理志》</ref>。[[明|明]][[嘉靖|嘉靖]]四十二年，名將[[戚繼光|戚繼光]]曾在仙游縣與[[倭寇|倭寇]]爆發主要戰事，史稱仙游大戰。倭寇被殲滅萬餘人，元氣大傷，為倭戰中的關鍵性戰役。

== 行政区划 ==
下辖1个街道办事处、12个镇、5个乡、4个管委会。
*[[街道|街道]]：鯉城街道；
*镇：枫亭镇、榜頭镇、郊尾镇、度尾镇、鯉南镇、賴店镇、蓋尾镇、園-{庄}-镇、大濟镇、龍華镇、鐘山镇、游洋镇；
*乡：西苑乡、石蒼乡、社硎乡、書峰乡、象溪乡。

== 经济 ==
全县有工艺企业3350多家，包括古典工艺家具、仙游石、古玩、国画、油画等，2011年产值突破160亿元；从业人员近15万人。2012年1-9月份，86家规模以上工艺美术企业完成产值75.4亿元，占全县规模以上工业产值的41.8%。

== 资源 ==
[[File:1964-06_1964年_福建木兰溪仙游糖厂.jpg|thumb]]
仙游物产丰富，资源充裕，盛产[[大米|大米]]、[[水果|水果]]、[[食菇|食菇]]、[[鳗鱼|鳗鱼]]、[[茶叶|茶叶]]等。兴化桂圆为历代朝廷贡品，可谓“兴化桂元甲天下”，度尾“无籽蜜柚”质优味美，名闻海内外；书峰[[枇杷|枇杷]]以果质脆嫩、酸甜适度而备受欢迎；[[香菇|香菇]]、[[蘑菇|蘑菇]]、[[巴西菇|巴西菇]]产量在全省名列前茅，是新兴的“食用菌之乡”；[[高岭土|高岭土]]、[[叶腊石|叶腊石]]、[[花岗石|花岗石]]、[[石英|石英石]]、[[温泉|温泉]]、[[矿泉水|矿泉水]]等矿产资源储量大、品位高；境内溪流纵横，[[水力资源|水力资源]]居沿海县区之最。

== 文化 ==
仙游历史悠久，素以“海滨邹鲁、文献名邦”著称，是著名的“工艺美术之乡”、“国画之乡”、“戏曲之乡”、“田径之乡”、“武术之乡”。县城连年被授予“全省精神文明建设先进县城”称号。

仙游县的枫亭元宵游灯习俗被列入[[国家级非物质文化遗产名录|国家级非物质文化遗产名录]]。

仙游通行[[莆仙话|莆仙话]]。

== 旅游 ==
仙游地理条件优越，气候宜人，木兰溪流域山青水秀、风光旖旎。

境内名胜古迹有[[九鲤湖|九鲤湖]]、菜溪岩、麦斜岩、天马山、蔡襄墓等。
