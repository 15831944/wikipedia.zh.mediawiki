'''仙后水母'''（[[學名|學名]]：{{lang|la|''Cassiopea andromeda''}}），中文別名'''倒立水母'''、'''朝天水母'''等{{noteTag|1=仙后水母的其他中文別名：倒吊水母、車輪水母、反口水母<ref name = "科博館"/>}}，英文俗名為 {{lang|en|Upside-down jellyfish}}<ref name = "科博館">{{cite web|website = 自然與人文數位博物館 |publisher = [[國立自然科學博物館|國立自然科學博物館]] |title = 仙后水母 |url = http://digimuse.nmns.edu.tw/Demo_2011/NewModule.aspx?ObjectId=0b00000181eb8ecc&ParentID=0b00000181eb8ecc&Type=&Part=&Domain=az&Field=cn&Language=CHI |accessdate = 2017-08-20 |archiveurl = https://web.archive.org/web/20170820122410/http://digimuse.nmns.edu.tw/Demo_2011/NewModule.aspx?ObjectId=0b00000181eb8ecc&ParentID=0b00000181eb8ecc&Type=&Part=&Domain=az&Field=cn&Language=CHI |archivedate = 2017-08-20 |deadurl = no |language = zh-tw}}</ref>，分佈於全球[[熱帶海域|熱帶海域]]，主要棲息於陽光充足且無海流的淺海<ref name = "環保署"/>，如[[潮間帶|潮間帶]]至水深10公尺的沙質環境，有時也可見於淺水泥質底、海草床上或[[瀉湖|瀉湖]]地形。<ref name = "科博館"/><ref>{{cite web|website = [[環境資訊中心|環境資訊中心]] |publisher = [[臺灣環境資訊協會|臺灣環境資訊協會]] |author = 莫聞 |title = 東沙環礁上的有趣生物：倒立水母 |url = http://e-info.org.tw/node/52387 |date = 2010-03-05 |accessdate = 2017-08-20 |archiveurl = https://web.archive.org/web/20170820131023/http://e-info.org.tw/node/52387 |archivedate = 2017-08-20 |deadurl = no |language = zh-tw |location = 台北}}</ref>臺灣地區曾在[[屏東縣|屏東]]、[[綠島|綠島]]、[[澎湖群島|澎湖群島]]與[[東沙島|東沙島]]潟湖等地發現過本物種<ref name = "科博館"/>。

== 形態與行為 ==
[[File:國立海洋生物博物館展示的仙后水母.webm|thumb]]展示的仙后水母|150px]]
仙后水母擁有八個分枝狀口腕，[[個體|個體]]大小約20至30公分，體色為褐色、淺褐、灰色或淺綠，表面有時佈有斑點或淺色條紋。牠們時常以觸手朝上、傘頂朝下的姿態漂浮於海底，為其別名「倒立水母」等的由來。仙后水母藉「倒立」使[[共生|共生]]於其觸手及身體的[[蟲黃藻|蟲黃藻]]行[[光合作用|光合作用]]，供應自身能量。不過牠們仍會捕食小型[[浮游生物|浮游生物]]，為肉食性動物。仙后水母不常游泳或移動，其觸手具有微弱毒性，不可觸摸。<ref name = "科博館"/><ref name="環保署">{{cite|publisher =[[行政院環境保護署|行政院環境保護署]]|author =|title =東沙島周邊指標生物族群調查與復育評估 —標本製作與管理規劃|url =https://www.marine.gov.tw/filesys/dlarea/979/file2.pdf|date =2013-12|accessdate =2017-08-20|archiveurl =https://www.webcitation.org/6ssCNimRw?url=https://www.marine.gov.tw/filesys/dlarea/979/file2.pdf|archivedate =2017-08-21|deadurl =no|language =zh-tw}}</ref>

== 經濟 ==
仙后水母主要用於商業觀賞，飼養時可餵食[[豐年蝦|豐年蝦]]活體或水母液體飼料<ref name = "科博館"/>。其對水質要求不高，為水母中較易養殖的品種。<ref>{{cite web|website = 海洋教育中心 |publisher = [[國立海洋生物博物館|國立海洋生物博物館]] |title = 水族實驗中心 |url = http://oec.nmmba.gov.tw/Area_Detail.aspx?ID=f33c9fde-7370-479c-bc65-c25ec3647b09&ModuleID=B01 |accessdate = 2017-08-20 |archiveurl = https://web.archive.org/web/20170820145659/http://oec.nmmba.gov.tw/Area_Detail.aspx?ID=f33c9fde-7370-479c-bc65-c25ec3647b09&ModuleID=B01 |archivedate = 2017-08-20 |deadurl = no |language = zh-tw}}</ref>其萃取液經研究發現，能幫助人類皮膚[[上皮細胞|上皮細胞]]、人類[[纖維母細胞|纖維母細胞]]增生，可開發為美容產品。<ref>{{cite web|website = [[自由時報|自由時報]] |publisher = 自由時報 |author = 邱芷柔 |title = 〈南部〉美和科大研發 獨創水母面膜 |url = http://news.ltn.com.tw/news/local/paper/1055892 |date = 2016-11-26 |accessdate = 2017-08-20 |archiveurl = https://web.archive.org/web/20170820182407/http://news.ltn.com.tw/news/local/paper/1055892 |archivedate = 2017-08-20 |deadurl = no |language = zh-tw |location = [[內埔|內埔]]}}</ref>
