'''仙茅'''（[[学名|学名]]：{{lang|la|''Curculigo orchioides''}}），又名'''地棕根'''、'''地棕'''、'''独茅根'''、'''独茅'''、'''独脚仙茅'''、'''山党参'''、'''仙茅参'''、'''海南参'''、'''婆罗门参'''、'''芽瓜子'''，为[[石蒜科|石蒜科]][[仙茅属|仙茅属]]多年生[[草本植物|草本植物]]。分布于[[印度尼西亚|印度尼西亚]]、[[日本|日本]]、[[东南亚|东南亚]]、[[台湾岛|台湾岛]]以及[[中国大陆|中国大陆]]的[[广西|广西]]、[[广东|广东]]、[[贵州|贵州]]、[[福建|福建]]、[[云南|云南]]、[[浙江|浙江]]、[[四川|四川]]、[[湖南|湖南]]、[[江西|江西]]等地，生长于海拔500米至1,600米的地区，一般生长在草地、林中或荒坡上，属中国原产的植物（非从国外引种）。

==形态==
*多年生[[草本|草本]]。高10～14厘米。地下常有单一、不分支的根茎，黑褐色，肉质断面白色。
*叶根生，3～6枚，具柄，披针形，革质，长约10～30厘米，宽0.6～2厘米，有散生长毛。
*花茎极短，藏于[[叶鞘|叶鞘]]之内，花黄色。

==生长环境==
*喜生于山坡、丘陵的草地、灌木丛旁。
*[[长江|长江]]以南均有生长。主产[[四川|四川]]。

==加工==
药用根茎。春初、夏末采集。除去须根和根头，洗净，切断，晒干备用。
==药用==
{{medical}}
中医上讲仙茅辛苦而甘温，有小毒。补肾壮阳，散寒除痹。西医则发现其中所含的[[仙茅苷|仙茅苷]]可用来抑制[[β淀粉样蛋白|β淀粉样蛋白]]聚合。<ref>{{cite journal | doi = 10.1016/j.bmcl.2007.11.028| title = New polyphenols active on β-amyloid aggregation| journal = Bioorganic & Medicinal Chemistry Letters| volume = 18| issue = 2| pages = 828| year = 2008| last1 = Rivière| first1 = Céline| last2 = Richard| first2 = Tristan| last3 = Vitrac| first3 = Xavier| last4 = Mérillon| first4 = Jean-Michel| last5 = Valls| first5 = Josep| last6 = Monti| first6 = Jean-Pierre}}</ref>

===主治===
*肾虚腰痛，[[神经衰弱|神经衰弱]]，[[阳萎|阳萎]]。
*妇女[[更年期|更年期]][[高血压|高血压]]。
*慢性肾炎。
*无名肿毒。
