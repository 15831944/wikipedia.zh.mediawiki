{{engname|《仙人掌》|''Run ''}}是1994年出品的一部香港電影，由張文幹、蕭忠漢執導，[[黎明|黎明]]、[[林海峰|林海峰]]、[[葉玉卿|葉玉卿]]主演，在[[墨西哥|墨西哥]]進行拍攝。<ref>{{Cite web |url= http://hkmdb.com/db/movies/view.mhtml?id=7894&display_set=big5|title =仙人掌（1994）| publisher = 香港影庫|language = 中文}}</ref><ref>{{Cite web |url= http://www.hkbo.com.hk/search-details-new.php?filmid=1712&run=1|title =電影資料庫| publisher = 香港票房有限公司|language = 中文}}</ref><ref>{{cite web|url=http://www.allaboutleon.com/index.cgi?md=read;id=16941|title=永遠的香港電影1994|date=1994年|publisher=香港電影雜誌|via=www.allaboutleon.com|language = 中文}}</ref>

==情節介紹==
失意流浪漢林文輝帶著前女友送給他的一把結他，從香港來到墨西哥準備定居，卻被誤認為結他殺手，到處被人追殺，在逃亡中被飛刀女俠賀凝所救。原來結他殺手，是賀凝兄長賀泉的好友阿占，阿占和賀泉行俠仗義，挑戰黑幫首領卡洛斯，卻被卡洛斯設計陷害，賀泉慘死，阿占入獄。卡洛斯欲追求賀凝，賀凝一心想為兄報仇，不為所動，卡洛斯派手下到獄中暗殺阿占，阿占越獄逃亡，隨身帶著一個結他包，裡面放著槍支，因此被視為結他殺手。林文輝與賀凝日久生情，卡洛斯的手下以為林文輝是阿占，把他擄走，賀凝和阿占前往營救，雙方展開一場大戰，結果賀凝和阿占被卡洛斯槍殺，阿輝用賀凝教他的刀法，成功殺死卡洛斯，繼續他的流浪之旅。<ref>{{Cite web |url= https://www.yesasia.com/global/%E4%BB%99%E4%BA%BA%E6%8E%8C/1004037376-0-0-0-zh_TW/info.html|title =仙人掌－ 葉玉卿, 黎明|publisher =www.yesasia.com|language = 中文}}</ref>
