'''仙台鹽釜港'''（{{jp|j=仙台塩釜港|hg=せんだいしおがまこう|rm=Sendaishiogama kō|lead=yes}}）是位於[[日本|日本]][[宮城縣|宮城縣]]、{{link-ja|仙台灣|仙台湾}}內的[[海港|海港]]，由[[宮城縣|宮城縣政府]]負責港務營運。仙台鹽釜港包含了仙台、鹽釜、石卷及松島四大[[港區|港區]]，是日本官方指定的{{link-ja|中核國際港口|中核国際港湾}}及{{link-ja|國際據點港口|国際拠点港湾}}；2017年，其貨物吞吐量位居日本港口第11位<ref name=":0">{{Cite web|url=http://www.mlit.go.jp/common/001246780.pdf|script-title=ja:港湾別コンテナ取扱貨物量一覧 - 国土交通省|accessdate=2018-11-02|author=|date=2017|publisher=[[國土交通省|國土交通省]]}}</ref>。

== 名稱 ==
仙台鹽釜港名稱中的「鹽釜」（{{jpn|j=塩釜|link=no}}）与港口所在城市[[鹽竈市|鹽竈市]]（{{jpn|j=塩竈市|link=no}}）的「鹽竈」为同一名称的不同写法，讀音皆為「しおがま」（Shiogama）。<ref>{{Cite web |url=http://www.city.shiogama.miyagi.jp/seisaku/shise/gaiyo/kamado.html |script-title=ja:塩竈市公式サイト-「竈」の字について|publisher=鹽竈市|language=ja}}</ref><ref>{{Cite web|url=http://www.city.shiogama.miyagi.jp/shiminsomu/shise/joho/shimin/zenpan/kamadonoji.html|script-title=ja:「塩竈」の「竈」の字の表記について|publisher=鹽竈市|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20140813222847/http://www.city.shiogama.miyagi.jp/shiminsomu/shise/joho/shimin/zenpan/kamadonoji.html|archivedate=2014-08-13}}</ref>。

== 歷史 ==
=== 開發背景 ===
自[[奈良時代|奈良時代]]起，今日的仙台鹽釜港所在地{{link-ja|仙鹽地區|仙塩}}即為日本東北諸內陸都市所倚靠的海上交通要道；當時的[[令制國|令制國]]之一[[陸奧國|陸奧國]]還曾在[[國府_(日本歷史)|國府]][[多賀城|多賀城]]（今[[鹽竈市|鹽竈市]]）的鹽竈津設置{{link-ja|外港}}，成為今日{{link-ja|鹽釜港|塩釜港}}的前身<ref>『仙台市史』通史編3（近世1）仙台市史編纂委員会. 仙台市. 2001. p.336 {{ja icon}}</ref><ref>『仙台市史』通史編4（近世2）仙台市史編纂委員会. 仙台市. 2003. pp.281-283 {{ja icon}}</ref>。

1876年（[[明治|明治]]9年），[[明治天皇|明治天皇]][[巡幸|巡幸]]日本東北，並指示東北各地官員加緊振興產業。之後在內務卿[[大久保利通|大久保利通]]的協助支持下，促成了現代化港口{{link-ja|野蒜築港}}的建立，一度取代鹽釜港的地位<ref name=":1">{{Cite web|url=http://www.pa.thr.mlit.go.jp/shiogama/kids/history/history04.html|title=幻の野蒜築港|accessdate=2018-11-11|last=shiogama|work=www.pa.thr.mlit.go.jp|language=ja}}</ref>；然而1884年9月侵襲當地的[[颱風|颱風]]使野蒜築港的海港突堤等港口設施嚴重毀損，使政府最終放棄港口，撤出當地<ref name=":1" /><ref>{{Cite news|date=2014年4月6日|script-title=ja:“幻の港”復興、運河再生は 「野蒜築港跡」講演通し学ぶ 東松島|newspaper=石巻かほく|url=http://ishinomaki.kahoku.co.jp/news/2014/04/20140406t13005.htm|accessdate=2015-08-09|language=ja}}</ref>。此後，多數野蒜築港預定具備的機能移至附近的鹽釜港。1887年（[[明治|明治]]20年），{{link-ja|鹽釜車站|塩釜港駅}}通車營運；隔年，港口北側的工業用地建成，吸引大量工廠進駐，造就了鹽竈一時榮景<ref>『仙台駅百年史』JR東日本仙台駅. 1987. p.21 {{ja icon}}</ref>。1910年，[[大日本帝國|日本政府]]指定鹽釜港為第二種重要港口；1951年（[[昭和|昭和]]26年），指定為{{link-ja|重要港口|重要港湾}}。1950年代後，鹽釜港除了原有的貿易港口機能外，還完善了工業原料運輸及漁業機能<ref>『宮城縣史』復刻版5（地誌交通史）宮城縣史編纂委員会. 宮城県. 1987. pp.628-632 {{ja icon}}</ref>。

[[二戰|二戰]]後，日本[[日本戰後經濟奇蹟|經濟高度成長]]，其中仙台地區指定為「{{link-ja|新產業都市|新産業都市}}」。當地政府認為仙台原有的{{link-ja|仙台港}}不敷使用，決定另建新港<ref name="SendaiHistory">『仙台市史』 通史編8（現代1）仙台市史編纂委員会. 仙台市. 2011. p.137 {{ja icon}}</ref>。1971年，現代化的仙台新港於仙台市東部的長濱落成，並編入鹽釜港編制內，正式名稱為「鹽釜港仙台港區」<ref>『仙台市史』通史編8（現代1）仙台市史編纂委員会. 仙台市. 2011. p.165 {{ja icon}}</ref>。自此之後，鹽釜港的大部分貿易機能便逐漸轉移至仙台港。2001年（[[平成|平成]]13年），「鹽釜港」正式更名為「仙台鹽釜港」，其中鹽釜港本身與仙台港分別更名為「仙台鹽釜港鹽釜港區」及「仙台鹽釜港仙台港區」<ref>[http://www.pref.miyagi.jp/kouwan/portplan-sendai/portplan-sendai4.htm 仙台塩釜港の港湾計画について]{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }} {{ja icon}}</ref>。

自[[江戶時代|江戶時代]]以來，{{link-ja|石卷港|石巻港}}便以{{link-ja|東廻海運}}航線所倚靠的據點港口活躍，並於1964年成為重點港口<ref>[http://www.pref.miyagi.jp/is-kouwan/ishikou/rekishi.htm 石巻港の歴史]{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }} {{ja icon}}</ref>。然而，在2010年日本政府自全國103個重要港口中選出43個{{link-ja|重點港口|重点港湾}}時，石卷港出乎意料的並未包含在名單內<ref>[http://www.kahoku.co.jp/news/2010/08/20100803t11024.htm 石巻「選外」非常に遺憾　重点港湾選定で知事が批判] {{Wayback|url=http://www.kahoku.co.jp/news/2010/08/20100803t11024.htm# |date=20100806141343 }} [[河北新報|河北新報]]. 2010-08-03. {{ja icon}}</ref>。同年5月，時任[[國土交通大臣|國土交通大臣]][[前原誠司|前原誠司]]視察石卷港時，首次提出將該港與仙台鹽釜港合併的想法<ref>[http://www.kahoku.co.jp/news/2010/08/20100819t11030.htm 石巻港、重点港湾根強い復活論　一体整備なお温度差] {{Wayback|url=http://www.kahoku.co.jp/news/2010/08/20100819t11030.htm# |date=20100822232402 }} 河北新報. 2010-08-19. {{ja icon}}</ref>。

=== 合併及現況 ===
2011年3月，[[東日本大震災|東日本大震災]]發生，使包括仙台鹽釜港、石卷港及位於觀光勝地[[松島|松島]]的[[松島港|松島港]]在內的日本東部沿岸各港口受到嚴重損害。為整合協調各港口運輸事務，並藉此加快震後港口設施、貿易運輸網及產業復興，宮城縣政府正式決定將上述三港整合，經日本{{link-ja|閣議|閣議 (日本)}}通過，於2012年10月將原有的仙台鹽釜港（含仙台及鹽釜港區）、石卷港和松島港重整為仙台、鹽釜、石卷及松島四大港區，整體名稱則維持「仙台鹽釜港」不變<ref>{{Cite web|url=http://www.mlit.go.jp/report/press/port01_hh_000120.html|title=報道発表資料：仙台塩釜港、石巻港、松島港の統合について - 国土交通省|accessdate=2018-11-10|work=www.mlit.go.jp|language=ja}}</ref>。

今日的仙台鹽釜港是日本官方指定的{{link-ja|中核國際港口|中核国際港湾}}及{{link-ja|國際據點港口|国際拠点港湾}}之一；2017年，其貨物吞吐量位居日本港口第11位<ref name=":0"/>。
