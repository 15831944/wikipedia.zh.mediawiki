'''煲蠟'''指利用容器盛着大量高溫[[蠟|液態蠟]]作玩耍的行為，最常見於[[中秋節|中秋節]]迎月、[[中秋節|中秋節]]正日和追月。煲蠟約自90年代在香港出現，於[[兒童|兒童]]及青少年間興起，經口耳相傳或互相模仿流傳至今，具體由來典故已不可考。玩者會焚燒[[報紙|報紙]]或使用[[蠟燭|蠟燭]]等作火源，將蠟燭盛載在汽水樽蓋、[[月餅|月餅]]罐等器皿中加熱至[[熔化|熔化]]、[[燃燒|燃燒]]。部分人會向已溶化的蠟液噴[[水|水]]，使濺起燃燒中的蠟液和產生高溫[[水蒸氣|水蒸氣]]。煲蠟極為容易引起[[火災|火災]]，而滾燙的蠟液接觸到皮膚也可能造成第三級[[燒傷|燒傷]]。

== 香港政府的態度 ==
[[File:Waxburningwarmingbanner.JPG|right]]
1990年代以前，[[香港警察|香港警察]]考慮到節日關係，很少會在中秋節干預煲蠟行為。但由於煲蠟會使公園等公眾場所留下大量難以清理的蠟漬，而且易生危險，[[香港政府|香港政府]]界定煲蠟為「將蠟燃燒或溶化，或將任何液體灑在或潑在熱蠟之上，而可能導致任何人受傷或任何財產受損。」<ref>http://www.info.gov.hk/gia/general/200309/06/0906157.htm</ref>，自1990年代末期開始呼籲市民不要煲蠟，[[康樂及文化事務署|康樂及文化事務署]]會在中秋節的3天加密巡邏公園等場所，要求市民賞月時勿點燃太多蠟燭，並避免留下蠟跡，又會勸止煲蠟行為，亦可根據《遊樂場地規例》，檢控康文署轄下公園、泳灘和燒烤場地煲蠟的人，最高可被罰款2000港元及入獄14天<ref>{{cite web |url=http://www.takungpao.com/news/2005-9-9/GW-454338.htm |title=存档副本 |accessdate=2005-09-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20070930062112/http://www.takungpao.com/news/2005-9-9/GW-454338.htm |archivedate=2007-09-30 }}</ref>。康樂及文化事務署可將留下蠟跡視同亂拋垃圾，發出1500港元的定額罰款通知書。

房屋署可按「屋邨清潔扣分制」，對在[[房屋署|房屋署]]轄下屋邨範圍內公眾地方煲蠟的[[香港公共屋邨|公共屋邨]]住戶扣除5分，但不適用於[[租者置其屋計劃|已售出的單位]]，另外如有弄污地方，可被罰款1500港元，此外，市民若在[[香港房屋協會|香港房屋協會]]轄下屋邨範圍公共地方煲蠟，該協會將有關個案轉交警方和[[食物環境衛生署|食物環境衛生署]]處理。<ref>[http://www.info.gov.hk/gia/general/200610/03/P200610030168.htm 《房屋署提醒居民切勿煲蠟》]，載於www.info.gov.hk，2006年10月3日。</ref>
