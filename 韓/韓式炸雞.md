'''韓式炸雞'''（{{lang-ko|(양념)치킨}}，又称“'''韩国炸鸡'''”<ref name=hgy>{{cite web|url=http://www.hanguoyou.org/public/food/foodContent/14?xyz=0.41096096900740325|title=炸鸡|publisher=韩国游|date=2014-04-21}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，韓語中亦常簡稱為'''{{lang|ko|치킨}}'''）<ref>
“양념”即[[藥念|藥念]]，是韩式复合香料；치킨为英文“chicken”音译。</ref>是一种[[韩国|韩国]]风味的[[炸鸡|炸鸡]]，既可以是[[快餐店|快餐店]]、[[餐馆|餐厅]]和[[酒吧|酒吧]]的主菜，也可以被当做[[零食|零食]]。韩式炸鸡口味很多，有原味、香辣酱料味、醬油味、烧烤味、大蒜味、葱丝味和蜂蜜等，其中原味和香辣酱料味被“韩国游”网站认为是最受欢迎的口味<ref name=hgy />。

==特點==
韩式炸鸡在做法上有着独到之处。首先，韩式炸鸡在炸之前要进行[[腌泡|腌泡]]。其次，韩式炸鸡要炸两遍，因此比普通炸鸡要脆，含油量要低。最后，韩式炸鸡在第二次炸的时候使用特制韩式炸鸡[[佐料|佐料]]<ref Name="作法">{{cite web |url =http://www.28ms.cn/caipu/waiguocaipu/hanguoliaoli/239903.html |title=正宗韩式炸鸡的做法 |publisher =家常美食网 |date=2014-04-22 }}</ref>。与普通快餐店不同，韩式炸鸡店里的炸鸡都是现点现做的，而不是像普通炸鸡那样事先做好，因此要等上至少20分钟才能吃到<ref name=bjtv>{{cite web |url= http://www.rbc.cn/tj/2014/0303/1029237.htm |title= “来自星星的炸鸡配啤酒”到底有多好吃？ |publisher= 北京人民广播电台 |date= 2014-03-03 |deadurl= yes |archiveurl= https://web.archive.org/web/20140713210821/http://www.rbc.cn/tj/2014/0303/1029237.htm |archivedate= 2014-07-13 }}</ref>。

==韩剧引发的热潮==
韩式炸鸡一般与[[啤酒|啤酒]]一起搭配食用。隨着韓劇《[[來自星星的你|來自星星的你]]》興起風潮，{{link-en|炸雞和啤酒|Chimaek}}成為最新流行語：「下雪了，怎麼能沒有炸雞和啤酒呢？」這台詞為韓國添加了一個新的標籤，所以到韓國品嚐真正炸雞和啤酒也被遊客們列入必食清單中，同时也使得该剧播放地的韩式炸鸡店的生意异常興隆<ref >{{cite web |url=http://www.ce.cn/culture/gd/201402/19/t20140219_2329754.shtml |title=《來自星星的你》引爆粉絲經濟加速外國品牌入華 |publisher=中国经济网 |date=2014-02-19 }}</ref><ref>{{cite web |language=zh-hk |url=http://meishi.xooob.com/tsxc/200911/394886_1044947.html |title=韩式炸鸡 |publisher=美食排行榜 |deadurl=yes |archiveurl=https://web.archive.org/web/20130908033837/http://meishi.xooob.com/tsxc/200911/394886_1044947.html |archivedate=2013-09-08 }}</ref><ref>{{cite web |url=http://media.people.com.cn/n/2014/0221/c40606-24423154.html |title="来自星星的你"火的绝不止炸鸡全智贤单品卖脱销 |publisher=人民网 |date=2014-02-21}}</ref><ref name=bjtv />。據統計韓國雞隻的消耗量達到了每年6億隻、每天27萬隻。炸雞店也成為了人們聚會、觀看競賽等主要場所<ref>{{cite web |language=zh-hk |url=http://big5chinese.visitkorea.or.kr/cht/SI/SI_CH_2_14.jsp?cid=1919799 |title=韓式炸雞特蒐各式口味大集合 |publisher=[[韓國觀光公社|韓國觀光公社]] |access-date=2015-04-24 |archive-url=https://web.archive.org/web/20160305203525/http://big5chinese.visitkorea.or.kr/cht/SI/SI_CH_2_14.jsp?cid=1919799 |archive-date=2016-03-05 |dead-url=yes }}</ref>。
