'''韓光渭'''（{{lang-en|Kuang-Wei Han}}，{{bd|1930年}}），[[山東省|山東省]][[即墨縣|即墨縣]]人，已退休[[工程科學|工程科學]][[博士|博士]]，是[[台灣|台灣]]重要國防武器[[雄風飛彈|雄風飛彈]]一型、二型計畫主持人，被稱為「雄風飛彈之父」。<ref name = "中央研究院近代史研究所 Institute of Modern History, Academia Sinica_2018"> {{ cite web | title = 中央研究院近代史研究所 Institute of Modern History, Academia Sinica | url = http://www.mh.sinica.edu.tw/PGPublication_Detail.aspx?pubid=544&majorTypeCode=1&minorTypeCode=10 | publisher = 中央研究院 | language = zh-tw | accessdate = 2018-09-04 }} </ref><ref name = "材料也神奇:科技學者許樹恩的一生傳奇_2004"> {{ cite book | title = 材料也神奇:科技學者許樹恩的一生傳奇 | url = https://books.google.com.tw/books?id=a9xe358wOnEC&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false | publisher = 秀威出版, 2004 | author = 作者：許樹恩 | isbn = 9789867614391 | language = zh-tw | date = 2004-08-01 }} </ref><ref name = "學習的人生：韓光渭回憶錄_2010"> {{ cite web | title = 學習的人生：韓光渭回憶錄 | url = http://m.sanmin.com.tw/product/index/001313360 | publisher = 中研院 | author = 韓光渭 | language = zh-tw | date = 2010-06-01 }} </ref>

== 學歷 ==
[[青島市|青島市]]私立崇德中學、省立青島臨時中學、私立中正中學。<ref name = "中央研究院近代史研究所 Institute of Modern History, Academia Sinica_2018" />

1955年，[[海軍機械學校|海軍機械學校]]電機系學士。

1958年，海軍公費留美，就讀美國海軍研究院電機系。1961年，美國海軍研究院電機系博士。<ref name = "中央研究院近代史研究所 Institute of Modern History, Academia Sinica_2018" /><ref name = "院士會議_2018"> {{ cite web | title = 院士會議 | url = https://academicians.sinica.edu.tw/02.php?func=22&_op=?ID:E009 | publisher = 中央研究院 | language = zh-tw | accessdate = 2018-09-04 }} </ref>

== 經歷 ==
1941年，隨部隊離開青島，經過台灣前往[[海南島|海南島]]，1950年回防台灣駐防在[[澎湖|澎湖]]。1956年，太康軍艦電工官。1957年，[[海軍官校|海軍官校]]電機系助教。<ref name = "中央研究院近代史研究所 Institute of Modern History, Academia Sinica_2018" />

1962年到1964年，[[海軍工程學院|海軍工程學院]]副教授。

1965年，[[美國柏克萊加州大學|美國柏克萊加州大學]]研究。

1966年到1982年，[[中山科學研究院|中山科學研究院]]電子研究所控制組組長。1982年到1984年，中山科學研究院電子研究所副所長。1982年到1995年，中山科學研究院雄風飛彈計畫主持人。1984年到1995年，中山科學研究院系統發展中心副主任。

1985年，晉升[[中華民國海軍|中華民國海軍]][[少將|少將]]。<ref name = "學習的人生：韓光渭回憶錄_2010" />

1962年到1992年、1997年到1999年，[[交通大學|交通大學]]兼任教授。1992年到1999年，[[台灣大學|台灣大學]]兼任教授。1995年到2000年，[[元智大學|元智大學]]講座教授。<ref name = "院士會議_2018"> {{ cite web | title = 院士會議 | url = https://academicians.sinica.edu.tw/02.php?func=22&_op=?ID:E009 | publisher = 中央研究院 | language = zh-tw | accessdate = 2018-09-04 }} </ref>

== 評價 ==
孫又予先生在其回憶錄中對時任中山科學研究院控制組組長的韓光渭先生評價：是一位有實學而又肯實幹的人物，他讀書用功的程度無人能及。到日本東京出差，在星期假日時，大家都出去觀光遊玩，只有他一個人留在旅館裡閉門讀書。由他主持雄風飛彈計畫，就很成功，只是像韓光渭先生這樣的人太少了。<ref name = "憶浮生往事: 一名中科院退休員工對過去的回顧_2015"> {{ cite book | title = 憶浮生往事: 一名中科院退休員工對過去的回顧 | url = https://books.google.com.tw/books?id=K0HDCgAAQBAJ&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false | publisher = 獨立作家－獵海人, 2015 | author = 作者：孫又予 | isbn = 9789574327256 | language = zh-tw | date = 2015-10-23 }} </ref>

== 榮譽 ==
* 1962年，[[教育部|教育部]]學術獎章。
* 1966年，中山學術著作獎。
* 1990年，[[中央研究院院士|中央研究院院士]]。
* 1995年，交通大學名譽博士。<ref name = "韓光渭_2018"> {{ cite web | title = 韓光渭 | url = http://www.nctu.edu.tw/outstandingnctu/honorarydr/202-83/418-2014-07-16-07-40-07 | language = zh-tw | accessdate = 2018-09-04 }} </ref>
* 1997年，交通大學榮譽教授。
* [[美國|美國]][[電機電子工程師學會|電機電子工程師學會]]學術論文獎2次。
* 2006年，系統學會終身成就獎。
