'''韓語維基百科'''（{{kor|諺=한국어 위키백과|漢=韓國語 위키百科|羅=Hangugeo Wikibaekgwa|f=y}}）是[[維基百科|維基百科]]的[[韓語|韓語]]版本，一個自由、開放內容的百科全書計劃。目前是在各語言維基百科計畫中，規模排名第22（依條目量計算）<ref>[[:meta:List_of_Wikipedias|元维基上的“维基百科列表”]]</ref>。開始於2002年10月11日（星期五）下午四時，截至2018年2月7日，韓語維基百科已有408,557篇條目。

== 歷史 ==
=== 創立伊始 ===
韓語版原始的網址為[//ko.wikipedia.com ko.wikipedia.com]。第一篇有意思的條目，是由現時的管理員之一[[:ko:User:Xaos|Xaos]]在2002年10月12日14:37創建的[[吉米·卡特|吉米·卡特]]（[[:ko:지미_카터|지미 카터]]），當時Xaos還未正式注冊，到了15:37才注冊成為第一位用戶。

== 發展狀況 ==
雖然韓國國內的上網十分普及，但韓語維基百科初期屬於發展較遲緩的維基百科語種。可能的原因是韓國許多[[入口網站|入口網站]]，如[[NAVER|NAVER]]、[[雅虎|雅虎]]、[[DAUM|DAUM]]、[[PARAN|PARAN]]等皆提供免費百科查閱。為此，借助韓國當地的各種資訊科技服務，韓語維基有以下各種獨特的派生服務。

現今韓語維基百科已經呈現穩定發展的狀態，如果不計拉丁文字圈的語言則名列俄語、日語、中文、波斯語和阿拉伯語之後，排在第六位。

== 派生服務 ==
從2005年8月11日開始，[[Empas|Empas]]把韓語維基百科的內容與其內部的百科數據庫統合，使兩者皆可被檢索<ref>{{cite news|url=http://itnews.inews24.com/php/news_view.php?g_serial=165046&g_menu=020100&pay_news=0 |title=네티즌이 만드는 '위키백과사전' 검색 서비스...엠파스|publisher=아니뉴스24|date=2005-08-11| accessdate=2009-06-08}}</ref>。2007年7月6日開始，[[鮮京電信|鮮京電信]]（SK Telecom）的[[行動通訊服務|行動通訊服務]][[NATE|NATE]]亦提供為其客戶透過手機檢索韓語維基百科的服務<ref>{{cite news|url=http://www.etnews.co.kr/news/detail.html?id=200707050039|title=휴대폰으로 위키피디아를~|publisher=전자신문| date=2007-07-06}}</ref>。

從8月21日開始，韓國入門網站[[Daum|Daum]]的百科全書服務亦包括了對韓語維基百科及[[英語維基百科|英語維基百科]]的查詢<ref>{{cite news| url=http://www.edaily.co.kr/invest/stock/newsRead.asp?newsid=01866326583229616 |title=다음, 백과사전 서비스 강화..`위키피디아` 제공 |publisher=이데일리 (e-Daily 電子新聞)| date=2007-08-21|accessdate=2009-08-06|language= ko}}</ref>，而[[NAVER|NAVER]]亦從2008年1月11日開始提供對韓語及英語維基百科的檢索服務<ref>{{cite news|url=http://www.etnews.co.kr/news/detail.html?id=200801110013 |title=네이버, 위키피디아 검색 오픈 |title_zh=NAVER: 維基百科檢索啟用 |date=2008-01-11 |accessdate=2009-08-06| publisher=전자신문 (電子新聞)| language= ko}}{{quote|네이버의 검색 결과 페이지는 ‘사전’ 카테고리의 검색 결과가 ‘웹 문서’ 카테고리의 검색 결과보다 상위에 노출되며, 위키백과 검색 결과를 사전 카테고리에 노출하고 있다.}}</ref>。
