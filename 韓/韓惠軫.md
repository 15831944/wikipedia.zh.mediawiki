'''韓惠軫'''<ref>[http://weibo.com/3856949271/B5CGpdwZb Namoo Actors微博公布旗下藝人漢字名]</ref>（{{lang-ko|한혜진}}，{{bd|1981年|10月27日|||catIdx=Han, Hye Jin}}），[[韓國|韓國]][[女演員|女演員]]。

== 人物 ==

2002年出道，出道時都是演出一些不起眼的配角。2004年，她被選中出演《[[你的星星|你的星星]]》的女主角，憑此劇一舉贏得了KBS演技大賞新人獎。2005年接拍《[[加油！金順|加油！金順]]》更令她一炮而紅。2006年接拍大型史詩電視劇《[[朱蒙_(電視劇)|朱蒙]]》創下平均收視率高達41%的好成績。

== 個人生活 ==
姊夫為男演員[[金剛于|金剛-{于}-]]。

2004年與韓國知名靈魂樂歌手[[Naul_(韓國歌手)|NAUL]]公開戀情。2012年12月21日傳出分手消息，兩人因為工作不能經常見面而逐漸疏遠，九年戀愛關係畫上句點。<ref>{{Cite web |url= http://tw.enewsworld.mnet.com/enews/contents.asp?idx=24627|title= 韓惠珍與Naul近期分手 9年戀情告終|date=2010-12-22 |publisher=enewsworld|language= zh}}</ref>

2013年3月27日，證實與年齡小8歲、效力於[[英格蘭足球冠軍聯賽|英冠]][[斯旺西城足球俱乐部|斯旺西城足球俱樂部]]（該俱樂部該年隸屬[[英格兰足球超级联赛|英超]]等級）的韓國國家足球隊選手[[奇誠庸|奇誠庸]]已交往兩個月。由於兩段感情開始和結束的時間太接近，所以一直有韓惠軫腳踏兩條船的傳聞出現，而她在節目上澄清和NAUL其實早已在2012年夏天分手。與奇誠庸的戀情公開不久後即宣布婚訊，兩人於同年7月1日舉行婚禮。<ref>[http://cn.chosun.com/big5/site/data/html_dir/2013/03/28/20130328000009.html 姐弟戀奇誠庸告白神速承認亦爽快] 朝鮮日報中文網</ref><ref>[http://www.koreastardaily.com/tc/news/27985 韓惠珍7月嫁球星寄誠庸 8歲姐弟戀修成正果] 韓星網</ref><ref>[http://mwave.interest.me/enewsworld/article/39238 韓惠珍撇腳踏兩條船　弟弟奇誠庸一秒變男人]</ref>2015年9月13日，長女誕生。

2017年起將暫時前往英國協助[[奇誠庸|奇誠庸]]與家庭，並預定與同年5月返韓<ref>http://kpopn.niusnews.com/=K3tip8u1</ref>。
