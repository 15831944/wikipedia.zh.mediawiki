'''韓浩'''（{{bd|?||?||catIdx=Han韓浩}}），字'''元嗣'''，[[河內郡_(中國)|河內]]（郡治在今[[河南省|河南省]][[焦作市|焦作市]]境内）人，[[東漢|東漢]]末年[[三国|三国]]時期[[曹操|曹操]]手下的將領。與[[史渙|史渙]]因忠勇而聞名，韓浩官至中護軍，史渙官至中領軍，皆掌禁兵，封列侯。

汉末大乱，浩聚徒众护县。太守王匡辟浩为从事，领兵拒董卓于盟津。卓执其舅招浩，不从，因而知名。袁术任浩为骑都尉。后夏侯惇奇其才，使浩领兵从征伐。时荒乱乏粮，浩议急农救荒，曹操遂兴屯田，迁浩护军。从破柳城，改任中护军，置长史、司马，拜为万岁亭侯。从破张鲁，议者欲留浩镇汉中，都督诸军。曹操不舍浩，与俱还，后卒。

== 生平 ==
[[東漢|東漢]]末年，天下大亂，因韓浩的鄉縣周圍多山，所以賊寇特別多，韓浩便聚起群眾保衛鄉縣。後[[太守|太守]][[王匡_(三国)|王匡]]辟他為從事，領兵到[[盟津|盟津]]拒[[董卓|董卓]]。當時他的舅父[[杜陽|杜陽]]為[[河陰|河陰]]令，董卓要脅韓浩投靠，韓浩不從。之後，[[袁術|袁術]]知為人壯烈，任作騎都尉。[[曹操|曹操]]將領[[夏侯惇|夏侯惇]]聞得其名，與他相見，大感驚奇，使韓浩領兵跟隨。

當時曹操未能解決糧食問題，韩浩與[[枣祗|枣祗]]等向曹操提议[[屯田制|屯田制]]，曹操接納，遷他為護軍。194年，[[呂布|呂布]]偷襲[[兗州|兗州]]，夏侯惇混亂中被呂布詐降軍劫持，軍中大亂。韓浩率兵到夏侯惇陣營，先平定混亂，再到夏侯惇帳篷，大罵詐降軍：“汝等凶逆，乃敢執劫大將軍，復欲望生邪！且吾受命討賊，寧能以一將軍之故，而縱汝乎？”再為夏侯惇流淚，領兵殺入，詐降軍叩頭謝罪，韓浩不受，反而大罵詐降軍，斬殺眾人。事後曹操稱他處理得當，定明以後凡是發生劫持事件都不用顧及人質，全力攻擊劫持者，而後劫持事件沒有發生了。

[[官渡之戰|官渡之戰]]後，袁家勢力被曹操一步步消滅，曹操追殺[[袁尚|袁尚]]、[[袁熙|袁熙]]至[[柳城|柳城]]，史渙認為深入遠道非萬全之計，想與韓浩一同諫阻，韓浩說：“今兵勢彊盛，威加四海，戰勝攻取，無不如志，不以此時遂除天下之患，將為後憂。且公神武，舉無遺策，吾與君為中軍主，不宜沮眾。”果然攻破柳城，改升為[[中護軍|中護軍]]，置長史、司馬，掌管禁兵。

215年，曹操進攻[[漢中|漢中]][[張魯|張魯]]，張魯投降。眾人認為韓浩的智略足以防守邊疆，可以留鎮漢中，統率諸軍。曹操說：“吾安可以無護軍？”于是帶他還都城。韓浩死後，曹操甚覺可惜。

== 特徵 ==
韓浩之忠勇而為人所知，而且能冷靜分析事件。如夏侯惇劫持事件、柳城征伐對史渙的解釋都顯示其冷靜分析的頭腦。曹操對其重用有加。

== 評價 ==

議者：“以浩智略足以綏邊。”

[[陳壽|陳壽]]：“浩忠勇。”

== 演義中的韓浩 ==
被虛構為長沙太守[[韓玄|韓玄]]之弟。曾在[[博望坡之戰|博望坡之戰]]與漢中攻防戰中登場，為報韓玄之仇而與[[黃忠|黃忠]]交手，結局被黃忠斬殺。
