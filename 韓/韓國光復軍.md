'''韓國光復軍'''（{{lang-ko|한국 광복군}}）是[[第二次世界大战|第二次世界大战]]时期[[大韓民国臨時政府|大韓民国臨時政府]]（[[朝鲜|朝鲜]]在华[[流亡政府|流亡政府]]）的軍隊，1940年9月17日成立于[[中华民国|中华民国]][[重庆陪都时期|战时陪都重庆]]<ref>{{cite web |language = zh-hans |url = http://www.hoplite.cn/templates/gfjwsg0002.html |title = 抗战时期的韩国光复军 |publisher = 中国黄埔军校网>>韩国光复军文史馆|date = 未知 |accessdate = 2010年5月20日}}</ref>，主要由在华的[[朝鲜人|朝鲜人]]组成，对日军进行[[游击战|游击战]]。总司令由[[池青天|池青天]]（化名李青天）担任，参谋长为[[李范奭|李范奭]]。

== 背景 ==
自[[日韩合并|日本吞并朝鲜]]建立殖民统治以来，朝鲜人民抵抗运动不断。1919年爆发的[[三一运动|三一运动]]被镇压后，韩国独立人士在[[上海|上海]]建立了[[大韩民国临时政府|大韩民国临时政府]]，企图逐步将分散于亚洲各地的朝鲜民族运动统一。后由于中国国土的逐步沦陷，韩国临时政府也几经辗转，最终于1940年到达[[陪都|陪都]][[重庆|重庆]]。此时临时政府内部的党派斗争基本结束，新成立的[[韩国独立党|韩国独立党]]开始执政，朝鲜独立运动人士[[金九|金九]]为执行委员长<ref name="163ls">{{cite web |language = zh-hans |url = http://history.news.163.com/09/0816/17/5GRTH6F100011247.html |title = 韩国国父金九与蒋介石秘室单独笔谈 |publisher = 网易历史 |date = 2009-08-18 |accessdate = 2010年5月20日}}</ref>。此时[[朝鲜独立运动|朝鲜独立运动]]的激进派——[[金元凤|金元凤]]领导的[[朝鲜民族革命党|朝鲜民族革命党]]尚未加入韩国临时政府。

== 组建 ==
[[File:大韩民国临时政府对日宣战书.JPG|thumb]]

早在1933年，中国[[国民政府主席|国民政府主席]][[蒋介石|蒋介石]]曾于[[南京|南京]]秘密会见[[金九|金九]]先生，两人[[漢文|漢文]][[笔谈|笔谈]]密商合作事宜。蒋介石对[[朝鲜独立运动|朝鲜独立运动]]目前以[[暗杀|暗杀]]行为为主的方針，不以为然，提议朝鲜仁人志士自主建立武装<ref name="163ls" />。其后，[[河南省|河南省]]中央军官学校洛阳分校内开设了韩人特别班，对朝鲜人进行军事培训<ref name="lbh">{{cite web|language =zh-hans|url =http://www.luobinghui.com/j/m/200712/18479.html|title =韩国第一任总理李范奭|publisher =云南陆军讲武堂|date =2007-12-13|accessdate =2010年5月20日|deadurl =yes|archiveurl =https://web.archive.org/web/20100405082751/http://www.luobinghui.com/j/m/200712/18479.html|archivedate =2010年4月5日}}</ref>。1937年7月，韩国临时政府召开国务会议，决定成立军事委员会，着手进行战争<ref name="zh">{{cite web |language = zh-hans |url = http://www.china.com.cn/international/txt/2007-10/22/content_9105719_3.htm |title = 中国抗战洪流中的朝鲜志士 |publisher = 中国网 |date = 2007-10-22 |accessdate = 2010年5月20日}}</ref>。1939年，以金九为首的“光复阵线”与以金元凤为首的“民族阵线”发表联合宣言，号召民族团结一致抗战<ref name="jszh">{{cite web|language =zh-hans|url =http://military.china.com/zh_cn/history4/62/20070930/14375562.html|title =中国人尹呈辅曾经出任韩国光复军参谋长|publisher =中华网军事频道|date =2007-09-30|accessdate =2010年5月20日}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1940年，金九发表《韩国独立与东亚和平》向中国官方呼吁请求援助<ref name="360doc">{{cite web |language = zh-hans |url = http://www.360doc.com/content/08/0206/23/48648_1029522.shtml |title = 重庆不仅曾是民国的陪都，也是韩国流亡政权基地 |publisher = 个人图书馆 |date = 2008-02-06 |accessdate = 2010年5月20日}}</ref>。

1940年9月17日，在中国[[国民政府|国民政府]]的直接援助下，韩国光复军总司令部在重庆嘉陇宾馆正式成立，[[宋美齡|宋美龄]]还特别向光复军捐赠慰劳金10万元<ref name=":0">[http://jds.cssn.cn/webpic/web/jdsww/UploadFiles/2010/11/201011031627331813.pdf 蒋介石与韩国独立运动] 杨天石 中国社会科学院近代史研究所</ref>。光复军总司令由韩国抗日名将[[池青天|池青天]]担任，参谋长为[[青山里戰鬥|青山-{}-里戰鬥]]中的英雄[[李范奭|李范奭]]，后者在二战结束后成为[[大韩民国|大韩民国]]第一任[[韩国总理|内阁总理]]兼国防部长。

金九等发表宜言，声明韩国光复军决“与友邦中国抗日大军并肩杀敌”，“不但调动白山黑水间枕戈待旦的三韩健儿和散在华北一带的白衣大军，更能以此国内的三千万革命大众闻风而起，冲断倭寇的铁蹄锁链而遂行圣洁的天职”<ref name="163ls" />。[[蒋介石|蒋介石]]的亲信，时任重庆卫戍司令[[刘峙|刘峙]]出席了典礼。包括美军顾问团在内的许多外国客人也参加了成立典礼。同年10月，[[大韩民国临时政府|大韩民国临时政府]]制定了《光复军司令部组织条例》和《大韩民国临时统帅部管理制度》{{r|回忆录|page=207}}。

在取得蒋介石明确指示后，1941年11月13日，[[重庆|重庆]][[国民政府|国民政府]]颁发《光复军九个行动准绳》，将光复军正式纳入中国军事委员会辖下<ref name=":0" />，军服装备等均由中方负担<ref>{{cite web |language = zh-hans |url = http://www.wangchao.net.cn/junshi/detail_37136.html |title = 韩国光复军 - 整个一国民党军队 |publisher = 王朝网络 |date = 2009-11-08 |accessdate = 2010年5月20日}}</ref>。根据协议，重庆政府抽调部分文官和医疗救护人员对其进行资助<ref name="360doc" />。1941年12月10日，大韩民国临时政府对日正式宣战{{r|回忆录|page=324}}。

== 实力增强 ==
[[File:在西安接受检阅的韩国光复军.JPG|thumb]]
[[File:철기_이범석과_광복군,_미_OSS_요원.jpg|thumb]]（OSS）教官]]
光复军成立之初，仅有300人之众，主要由韩国临时政府内部人士构成。1941年l0月，蒋介石采取措施，将光复军与另一支朝鲜抵抗军队朝鲜义勇队两者都置于中国军事委员会统辖之下，并编入中国军队从而合二为一<ref name="lbh" />。其后，朝鲜独立运动的另一大派别[[朝鲜民族革命党|朝鲜民族革命党]]宣布加入韩国临时政府，[[金元凤|金元凤]]领导其朝鲜义勇队于1942年5月正式加入光复军<ref name="zh" />。原韩国光复军第1、2、5支队合并成第2支队。民族革命党的武装力量朝鲜义勇队被改编成为光复军第1支队。同年9月，韩国光复军总司令部由重庆前往西安{{r|回忆录|page=324}}。

1943年8月，韩国光复军向印缅战区派了工作队，参加了英军对日的作战。同年9月，中国方面决定恢复大韩民国临时政府对韩国光复军的管辖。1944年8月23日，韩国光复军重新获得作战和其它军事行动的自主权{{r|回忆录|page=324}}。

1944年10月，韩国光复军与美国[[战略情报局|战略情报局]]（OSS）取得了联系，探讨韩美联合在韩国作战的可能性。1945年1月，美军上尉受李范奭的邀请访问了西安第2支队总部。之后，大韩民国临时政府决定力促韩美军事合作，让美国OSS教官负责对第2支队的训练，以便在适当时候将受训人员投入战斗{{r|回忆录|page=324}}。韩国光复军司令总部从重庆迁往西安后，兵力由起初的几百人发展到5万多人<ref>{{cite web|url=http://history.people.com.cn/n/2015/0615/c372327-27155230-2.html |title=抗战时的西安：曾是韩国复国者向往的“圣地” |publisher=人民网 |date=2015-06-15 |archivedate=2015-10-03 |archiveurl=https://web.archive.org/web/20151003211928/http://history.people.com.cn/n/2015/0615/c372327-27155230-2.html |deadurl=yes }}</ref>。

== 作用 ==
光复军积极参加了中国抗日战争，在对日本方面的宣传、情报刺探以及争取国际援助等方面做了大量工作<ref>{{cite web |language = zh-hans |url = http://qkzz.net/article/827eebef-f48c-4d96-9a25-fb02e65c474c.htm |title = 蒋介石曾援助韩国抵抗日本 |publisher = 全刊杂志赏析网 |date = 未知 |accessdate = 2010年5月20日}}</ref>。光复军在河北省、熱河省、湖北省、山西省、山東省、安徽省、江西省等地对日军展开了游击战，但实际上因战斗效果欠佳而未被日本收入作战记录中。

== 战后 ==
[[日本投降|日本投降]]后，[[美国|美国]]與[[蘇聯|蘇聯]]两国以[[三八线|三八线]]为界，分别[[軍事占領|軍事占領]]朝鲜南北，韩国临时政府的面临着许多亟待解决的问题。1945年8月24日，蒋介石发表了《完成民族主义，维持国际和平》的讲演，明确表示应“恢复高丽的独立自由”，并借给韩国临时政府[[法币|法币]]1亿元、美金20万。1945年11月23日，大韩民国临时政府迁回朝鲜，光复军也于1946年返国<ref name="jszh" />。
