'''韓國郵政'''（{{朝|우정사업본부|郵政事業本部}}）是一家[[韩国|韩国]]公立事業單位，总部位于韩国[[世宗特別自治市|世宗特別自治市]]<ref>{{cite web |url=http://www.koreapost.go.kr/eng/sub/subpage.jsp?contId=e5050000 |title=Location |publisher=韩国邮政官网 |accessdate=2015-07-20 |language=en}}</ref>，主要业务包括[[邮政|邮政]]、[[邮政储蓄|邮政储蓄]]和[[保险|保险]]服务<ref>{{cite web|title=Introduction |url=http://www.koreapost.go.kr/eng/sub/subpage.jsp?contId=e5010000 |accessdate=2015-07-20 |newspaper=韩国邮政官网 |language=en}}</ref>。韩国邮政原由韩国[[大韩民国知识经济部|知识经济部]]管辖。2013年[[朴槿惠|朴槿惠]]行政机构改革后，韩国邮政被划归{{link-en|未来创造科学部|Ministry of Science, ICT and Future Planning}}管理<ref>{{cite web |url=http://chinese.joins.com/gb/article.do?method=detail&art_id=98695 |title=未来部权力大增，设两名副部长并接管邮政本部 |publisher=中央日报 |date=2015-07-20}}</ref>。韩国邮政目前员工人数大约有42000名，拥有3692个[[邮局|邮局]]、22个分拣中心和1个交换中心，每年投递约49.75亿件[[邮件|邮件]]<ref>{{cite web |url=http://www.chinapostnews.com.cn/newspaper/b2009/783/07830701.htm |title=加快业务发展的步伐———韩国邮政一览 |publisher=中国邮政报 |accessdate=2015-07-20}}</ref>。

==历史==
自1884年{{link-ko|韩国邮政总局|우정총국}}成立<ref>{{cite web |url=http://news1.kr/articles/?792511 |title=국내 최초 우체국 ‘우정총국’, 128년만에 우체국 기능 복원 |publisher=News1 Korea  |accessdate=2015-07-20 |language=ko}}</ref>，开始[[邮政|邮政]]服务以来，韩国邮政至今已有一百多年的历史。1900年，韩国加入[[万国邮政联盟|万国邮政联盟]]。1905年7月，韩国邮政开始办理储蓄业务；1966年，实现每个村镇至少一个邮局。1970年7月，韩国开始使用5位数的[[邮政编码|邮政编码]]。1974年1月，标准化[[信封|信封]]开始使用，提高了信件投递效率。1988年2月，邮政编码升为6位数，加快了邮件机械化管理。1990年，韩国开始建设计算机化邮政中心，大大提高了信件投递速度和效率。2002年，已有22个邮政中心配备了最新的分拣机。2005年，韩国共有3692个[[邮局|邮局]]、22个分拣中心、和1个交换中心，邮局总量为49.75亿件，绝大多数地区邮件1天内就可到达{{r|概况|韩文化|page1=68-69|page2=127}}<ref>{{cite web |url=http://wenku.baidu.com/link?url=m1ulqHCSBAy0FbWlAaEH0giO3Yx-3Ib4wlahKrvH62932vrU8vzHVcsoB_O2YdrNjsrPGUMbkwqRWGWIIvzymhSvKuoqGBtIdKt6XKDRoJa |title=韩国邮政储蓄的发展及启示 |publisher=《上海金融》 |date=2003年第7期}}</ref>。
