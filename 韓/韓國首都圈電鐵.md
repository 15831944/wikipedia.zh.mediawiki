'''韓國首都圈電鐵'''（{{kor|諺=수도권 전철|漢=首都圈 電鐵|羅=Sudogwon jeoncheol}}）是[[世界城市軌道交通年乘客量列表|世界第三大載客量]]的[[鐵路|鐵路]]系統，车站数量仅次于[[纽约地铁|纽约地铁]]，截至2013年4月26日，路线长度[[世界各城市地铁列表|世界第七]]。其服務範圍為[[韓國|韓國]][[首爾特別市|首爾特別市]]和周邊的[[首都圈|首都圈]]，日均載客量超过1000万人次（2013年统计）<ref>只限[[首爾地鐵|首爾地鐵]]、[[首爾特別市都市鐵道公社|首爾特別市都市鐵道公社]]和9号线的合併數據</ref>。首都圈電鐵以首爾的九條[[地下鐵路|地下鐵路]]為主，並輔以[[韓國鐵道公社|韓國鐵道公社]]的[[盆唐線|盆唐線]]及[[仁川地鐵|仁川地鐵]]等線路，合共19條路線。現時整個鐵路系統裡，單單是[[首尔地下铁|首尔地下铁]]中[[首爾交通公社|首爾交通公社]]和[[首爾市地鐵9號線株式會社|首爾市地鐵9號線株式會社]]的營運路段，其總長度已達975.4公里。而未來更有一條新路線及多條路線的延長工程正在開展中。

首都圈電鐵主要由兩家公司經營：一號線至八號線由[[首爾交通公社|首爾交通公社]]（一號線至四號線由2005年10月27日前稱為首爾地下鐵公社及後改稱首爾地鐵，而五號線至八號線由[[首爾特別市都市鐵道公社|首爾特別市都市鐵道公社]]營運，兩者營運至2017年5月30日）。而九號線由[[首尔地铁9号线株式会社|首爾市地鐵9號線株式會社]]營運。此外，[[韓國鐵道公社|KORAIL]]亦參與營運一號線的地上區間、三號線、四號線、中央線、京義線、盆唐線及水仁線的首爾市至市外區間。至於[[仁川國際機場鐵路|仁川國際機場鐵路]]由[http://www.arex.or.kr 空港鐵道株式會社]營運、[[仁川都市鐵道1號線|仁川1號線]]由[http://www.ictr.or.kr 仁川交通公社]營運，而[[新盆唐線|新盆唐線]]則由[[新盆唐線株式會社|新盆唐線株式會社]]經營。

== 使用车辆 ==
{{main|首都圈电铁车辆}}
首都圈电铁车辆迄今拥有超过26种系列的形式，行走于首都圈电铁的各条线路中，其中[[首尔地铁|首尔地铁]]和[[韩国铁道公社|韩国铁道公社]]的车辆可以在指定的区间内互相乘入。这些车辆早期为从日本引进，现在多为韩国国产。

* [[新盆唐線|新盆唐線]]<ref name="donga-20050822">{{lang|ko|홍수용}}，(2005年)，[http://china.donga.com/gb/srv/service.php3?bicode=020000&biid=2005082231358 首爾江南～龍山地鐵將被建成雙線地鐵]{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}，《[[東亞日報|東亞日報]]》，2005年8月22日號。</ref>預計在2019年在[[光教|光教]]再伸延至好梅實，光教至京畿大一段分拆為支線，在2018年向北伸延至[[龍山|龍山]]，日後再在2020年或以後由[[好梅實站|好梅實站]]南延至[[鄉南|鄉南]]。
* [[首爾地鐵7號線|首爾地鐵7號線]]延長工程：在2018年，該線會由[[富平區廳站|富平區廳]]延伸到[[石南|石南]]，與[[仁川地鐵2號線|仁川地鐵2號線]]交會
* [[盆唐線|盆唐線]]：本路線預計到2018年路線完成時，盆唐線將連接新的[[水仁線|水仁線]]（由[[水原市|水原]]至[[仁川廣域市|仁川]]），連同往仁川的路段，本路段將會和[[首爾地鐵4號線|首爾地鐵4號線]]共同組成首都圈循環線的南段路線。

* 以[[往十里站|往十里站]]（[[首爾地鐵2號線|首爾地鐵2號線]]、[[首爾地鐵5號線|首爾地鐵5號線]]、[[中央線_(韓國)|中央線]]及[[盆唐線|盆唐線]]的轉乘站）作為起始站的[[東北輕軌線|東北輕軌線]]正計劃中。<ref>[http://frdb.new21.org] {{webarchive|url=https://web.archive.org/web/20070930224559/http://frdb.new21.org/ |date=2007-09-30 }}</ref>

== 票價與車票 ==

目前票價由基本的10公里1250[[韓圓|韓圓]]（即約-{zh-hk:8.8港元; zh-cn:7元人民币; zh-tw:35元新台幣}-）起跳，每5公里增加100韓圓。

乘客可以使用電子票證“[[T-money|T-money]]”的儲值金額搭車（使用T-money卡每趟可折扣100韓圓）。T-money卡可在車站內和便利商店購買。另外，過去曾經推出過一種橙色的小型儲值車票，每張售價￦10,000，但內有￦11,000。車票適用於整個電鐵的範圍，但不適用於跨地域的城際國鐵線路。

== 歷史 ==
國鐵最早在1974年開始運營1號線，並直到首爾市地下鐵公社建立前的1970年代到1980年代之間運營首爾地鐵第一期工程的1-4號線。都市鐵道公社則於1994年起運營第二期工程的5-8號線。原訂的第三期工程的9-12號線因為[[亞洲金融風暴|亞洲金融風暴]]而被逼終止，但後來9號線及3號線延長工程因為有逼切需要而得以繼續，原10號線簡化成為[[新安山線|新安山線]]，而原11號線則簡化成為[[新盆唐線|新盆唐線]]。
* 1974年8月15日：首爾地鐵1號線[[首爾站|首爾站前]]（當時）—[[清涼里站|清涼里站]]（地下）段和[[韓國鐵道|國鐵]]清涼里站（地下）—[[城北站_(首尔)|城北站]]、[[首爾站|首爾站前]]（當時）—[[水原站_(京畿道)|水原站]]、[[九老站|九老站]]—[[仁川站_(仁川廣域市)|仁川站]]和[[龙山站|龙山站]]—[[回基站|回基站]]段开通。
* 1984年： 2號線完全開通。 
* 1985年： 3號線、4號線開通，首尔地鐵段服務自動化施行和車費山頂方法段車費零邊緣土地。 
* 1991年： 國铁段服務自動化施行。 
* 1996年： 5號線開通。 
* 1999年： 8號線開通。 
* 2000年： 6號線、7號線開通。 關於下面集成它處理首爾地鐵段，并且開始國铁段。 
* 2004年： 車費山頂方法距離比例零的邊緣土地。 
* 2005年： [[首都圈電鐵1号線|首都圈電鐵1号線]]延伸到[[天安站|天安站]]，[[首都圈電鐵中央线|首都圈電鐵中央线]]开通。
* 2007年： [[首都圈電鐵中央线|首都圈電鐵中央线]]延伸到[[八堂站|八堂站]]。
* 2009年： [[首爾地鐵9號線|首爾地鐵9號線]]開通。
* 2012年： [[首爾地鐵7號線|首爾地鐵7號線]]延伸到[[富平区厅站|富平区厅站]]
* 2018年： [[新盆唐線|新盆唐線]][[美金站|美金站]]開通
