'''剑桥大学露西·卡文迪许学院'''（[[英语|英语]]：'''{{lang|en|Lucy Cavendish College, Cambridge}}'''）是[[剑桥大学|剑桥大学]]的一个学院。 学院拥有220名学生，本科生和研究生各占一半。该学院是[[剑桥大学|剑桥大学]]三所女子学院之一，其他两所是纽纳姆学院和默里·爱德华兹学院。<ref name="LCC_info">{{cite web|url=http://www.lucy-cav.cam.ac.uk/pages/the-college/college-information.php|title=关于露西·卡文迪许学院的信息|publisher=露西·卡文迪许学院}}</ref>

学院于1965年由一批剑桥大学的女性研究员和讲师建立，她们对女性在大学内没有代表权持有异议。学院原先的名字是露西·卡文迪许学院协会，于1970年转移到现在的位置，并于1986年获准更名为露西·卡文迪学院。学院于1997年成为剑桥大学正式的学院。<ref name="cavendishstatutes">{{cite web|url=http://www.lucy-cav.cam.ac.uk/media/college/documents/statutes%20and%20preface.pdf|title=露西·卡文迪学院在剑桥大学的地位|deadurl=yes|archiveurl=https://web.archive.org/web/20090205094629/http://www.lucy-cav.cam.ac.uk/media/college/documents/statutes%20and%20preface.pdf|archivedate=2009-02-05}}</ref>

该学院以[[露西·卡文迪许|露西·卡文迪许]](1841-1925)命名，她是女性教育改革的先驱。

== 学院历史 ==
卡文迪许学院的建立要追溯到一个名叫[[晚餐|晚餐]]小组的组织，这个组织为那些不是[[剑桥大学学院|剑桥大学学院]][[院士|院士]]的组员提供[[高桌|高桌]]会谈的机会。<ref name="dininggrouppapers">{{cite web|url=http://www.londonmet.ac.uk/genesis/search/$-search-results.cfm?CCODE=1696|title=晚餐小组报告 1951-1966}}</ref>  在当时，剑桥大学仅有两个女子学院，它们无法提供足够的宿舍给大量前来教书或提供学术服务的女性。<ref name="bidderobit">{{cite news|url=http://www.independent.co.uk/news/obituaries/anna-bidder-729480.html|title=安娜·彼得讣告|date=2001年10月9日|publisher=英国独立报|first=安妮|last=柏桐}}</ref>

安娜·彼得是1965年至1970年卡文迪许学院的首位院长，她是晚餐小组的会员之一,也是一位动物学家。<ref name="bidderobit"/> 这就是学院的院徽上有[[鹦鹉螺|鹦鹉螺]]花纹的原因。<ref>{{cite web|url=http://www.lucy-cav.cam.ac.uk/media/college/documents/crest.pdf|title=露西·卡文迪许学院院徽|deadurl=yes|archiveurl=https://web.archive.org/web/20090205094643/http://www.lucy-cav.cam.ac.uk/media/college/documents/crest.pdf|archivedate=2009-02-05}}</ref>

安娜·彼得的继任者依次是凯特·贝塔姆、 菲利斯·赫兹、安妮·柏桐 (英国首位女性外交官)、波利·娜佩里和维罗妮卡·萨瑟兰。

珍妮·托德教授是现任院长，也是学院第7任院长，她于2008年9月继任。

== 建筑和庭院 ==
卡文迪许学院现位于麦丁利道和夫人玛格丽特道的交汇处，在西敏寺学院和[[剑桥大学圣约翰学院|圣约翰学院]]附近（圣约翰学院曾经提供土地给卡文迪许学院）<ref name="cavbldgs"/> 。在学院建院初的几年，卡文迪许学院占用了斯维尔街和北安普敦街的几间房间，直至1970年才转移到现在的位置。

学院主要建筑包括柏桐厅和学院图书馆，先后于90年代完工。<ref name="cavbldgs">{{cite web|url=http://www.lucy-cav.cam.ac.uk/media/college/documents/site-buildings.pdf |title=露西·卡文迪许学院庭院和建筑 |publisher=露西·卡文迪许学院 |deadurl=yes |archiveurl=https://web.archive.org/web/20110927015027/http://www.lucy-cav.cam.ac.uk/media/college/documents/site-buildings.pdf |archivedate=2011-09-27 }}</ref>

1991年，学院购入了附近的巴利奥尔农场，该农场曾经由著名的经济学家[[艾尔弗雷德·马歇尔|艾尔弗雷德·马歇尔]]和他的妻子玛丽·马歇尔居住，他们合著了马歇尔第一部经济学著作。该建筑被重新命名为马歇尔楼，用作学生宿舍直至2001年。其后马歇尔楼被恢复成原先的构造，用作院长室。<ref>{{cite web|url=http://www.cambridge2000.com/cambridge2000/html/0004/P4070638.html|title=剑桥 2000 - 露西·卡文迪许学院的马歇尔楼 (巴利奥尔农场)}}</ref>

== 学院学生 ==
卡文迪许学院是剑桥大学国际化程度很高的学生团体之一，学院拥有220名学生，本科生和研究生各占一半。

{{Cquote|学院的学生来自英国的各个角落乃至全世界，任何年龄的女性都可以在学院就读任何科目。|露西·卡文迪许学院网站}}

== 著名校友 ==
卡文迪许学院的校友包括现任[[联合国|联合国]]妇女发展基金执行主任的诺埃琳·海泽。

卡文迪许学院的荣誉院士有英国女演员[[茱蒂·丹契|朱迪·丹奇]]、现任丹麦女王[[玛格丽特二世_(丹麦)|玛格丽特二世]]和新西兰首位女性总督[[凯瑟琳·蒂泽德|凯瑟琳·蒂泽德]]。
