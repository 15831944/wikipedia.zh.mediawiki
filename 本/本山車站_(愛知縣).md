'''本山站'''（{{jpn|j=本山駅|hg=もとやまえき|rm=Motoyama eki|f=y}}）是位於[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[千種區|千種區]]四谷通一丁目，為[[名古屋市營地下鐵|名古屋市營地下鐵]]之車站。本站為[[東山線|東山線]]與[[名城線|名城線]]之轉乘站。

== 可利用之鐵路路線 ==
*名古屋市營地下鐵
**[[東山線|東山線]] - 車站編號為'''H16'''。
**[[名城線|名城線]] - 車站編號為'''M17'''。

== 車站結構 ==
本站東山線為2面2線[[側式月台|側式月台]]，名城線為1面2線[[島式月台|島式月台]]之[[地下車站|地下車站]]。

== 歷史 ==
*1963年（[[昭和|昭和]]38年）4月1日 - 東山線本山站開始營運。
*2003年（[[平成|平成]]15年）12月13日 - 本站名城線開始營運，成為兩線之轉乘站。
