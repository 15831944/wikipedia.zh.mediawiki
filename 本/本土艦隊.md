'''本土艦隊'''（The Home Fleet），負責防衛英國周邊海域的英國海軍艦隊。

== 第一次世界大戰 ==
1902年，沿岸警備為主要目的之本土艦隊編成，1904年改稱海峽艦隊，1909年取消海峽艦隊稱號。

第一次世界大戰爆發後，本土艦隊與[[大西洋艦隊（英國）|大西洋艦隊（英國）]]整合，並參與了[[日德蘭海戰|日德蘭海戰]]，和[[德意志帝國海軍|德意志帝國海軍]]對峙。

== 戰間期至二戰 ==
新的本土艦隊再次建立，此時有[[戰艦|戰艦]]6艘、[[巡洋戰艦|巡洋戰艦]]2艘、[[巡洋艦|巡洋艦]]3艘、[[驅逐艦|驅逐艦]]27艘、[[潛艦|潛艦]]6艘、[[航空母艦|航空母艦]]2艘。

1941年，參與追擊[[俾斯麥號戰列艦|俾斯麥號戰列艦]]、[[歐根親王號重巡洋艦|歐根親王號重巡洋艦]]行動。

== 戰後至今 ==
[[冷戰|冷戰]]時期，強調與[[北約|北約]]合作，在[[北大西洋|北大西洋]]防禦[[蘇聯|蘇聯]]海軍。至1967年，改編成'''西方艦隊'''（Western Fleet）。
