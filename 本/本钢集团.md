'''本钢集团有限公司'''簡稱'''本鋼'''，是中國一間地方國企，總部設於[[遼寧省|遼寧省]][[本溪市|本溪市]]。本鋼集團由[[遼寧省國有資產監督管理委員會|遼寧省國有資產監督管理委員會]]管理。[[世界钢铁协会|世界钢铁协会]]根據[[中国钢铁工业协会|中国钢铁工业协会]]提供的數據，本鋼鋼鐵於2015年產量全球排名第21名。<ref>{{cite web |url=https://www.worldsteel.org/zh/statistics/top-producers.html |title=存档副本 |accessdate=2015-12-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20151222133740/https://www.worldsteel.org/zh/statistics/top-producers.html |archivedate=2015-12-22 }}</ref>集团董事長兼[[黨委書記|黨委書記]]為[[張曉芳|張曉芳]]女士。<ref>{{cite web |url=http://www.bxsteel.com/GenSit.asp?page=6 |title=存档副本 |accessdate=2016年10月1日 |deadurl=yes |archiveurl=https://web.archive.org/web/20170320114609/http://www.bxsteel.com/GenSit.asp?page=6 |archivedate=2017年3月20日 |df= }}</ref>

==歷史沿革==
[[日俄戰爭|日俄戰爭]]後，[[奉天省|奉天省]]成為[[瓜分中國|日本勢力範圍]]。日本人{{link-en|大倉喜八郎|Okura Kihachiro}}在1905年於[[本溪湖|本溪湖]]非法建立煤礦。及後[[清朝政府|清政府]]向日方干涉，在1910年中日合伙成立'''本溪湖商辦煤礦業有限公司'''。1911年公司改組成'''本溪湖煤鐵有限公司'''。1931年[[九一八事變|九一八事變]]後公司全盤由日方控制，直至1945年[[第二次世界大戰|第二次世界大戰]]結束。1948年[[國共內戰|國共內戰]]期間落入[[解放區|解放區]]控制，並成為[[第一個五年計劃_(中國)|第一個五年計劃]]的[[156項重點工程|156項重點工程]]之一('''本溪鋼鐵公司''')，接受蘇聯專家援助改建。1953年[[本溪湖煤礦|本溪湖煤礦]]改屬[[本溪礦務局|本溪礦務局]]管理(本溪礦務局今屬[[瀋煤集團|瀋煤集團]]一部分)。<ref>https://web.archive.org/web/20110522120744/http://www.bxsteel.com/bxsteel/bwg/WZ01.html</ref><ref>https://web.archive.org/web/20110522130407/http://www.bxsteel.com/bxsteel/bwg/WZ041.html</ref>

1996年本溪鋼鐵公司改制為'''本溪钢铁(集团)有限责任公司'''。<ref name=BIS2015AR/>

1997年4月，本钢成為[[中華人民共和國國務院|國務院]]120家「深化大型企业集团试点工作」企业，512家重點國有企業之一。<ref>[http://cpc.people.com.cn/GB/64184/64186/66698/4495124.html 国务院批转国家计委、国家经贸委、国家体改委《关于深化大型企业集团试点工作的意见》的通知]</ref><ref>[http://blog.sina.com.cn/s/blog_544a7adf0100mwmj.html 《关于深化大型企业集团试点工作的意见》的通知 附件一]</ref><ref>[https://books.google.com.au/books?id=ECi0RzsyCKAC&source=gbs_navlinks_s 《中國金融法律》中國法制出版社]</ref>

2005年本钢與[[國務院國有資產監督管理委員會|央企]][[鞍山钢铁|鞍山钢铁]]宣佈合併。同年鞍本钢铁集团举行揭牌仪式，<ref>[http://chinaneast.xinhuanet.com/2005-08/16/content_4897149.htm 鞍钢本钢实现强强联合 中国钢铁业诞生“新航母”] {{webarchive|url=https://web.archive.org/web/20120622050320/http://chinaneast.xinhuanet.com/2005-08/16/content_4897149.htm |date=2012-06-22 }}</ref>惜至今未完成合併。

2008年本鋼一鐵廠正式停運，<ref>{{cite web |url=http://www.bxsteel.com/News2.asp?serialNo=3194 |title=存档副本 |accessdate=2016年10月1日 |deadurl=yes |archiveurl=https://web.archive.org/web/20161002170914/http://www.bxsteel.com/News2.asp?serialNo=3194 |archivedate=2016年10月2日 |df= }}</ref>被集團其他現代化廠房取代。

2010年，地方國企[[北营钢铁|北台钢铁(集团)有限责任公司]](今名為北营钢铁，本屬[[本溪市國有資產監督管理委員會|本溪市國資委]])與本鋼合併，成立'''本鋼集團有限公司'''管理兩間公司。遼寧省國資委則取代本溪市國資統一管理國有持股。

2016年本鋼集團公開發行5億元公司[[債券|債券]]: "16本钢债01"。<ref>[http://www.chinabond.com.cn/Info/23326592 2016年第一期本钢集团有限公司公司债券发行文件]</ref>子公司亦曾發行"11 北营钢MTN1" (15億元)、"14 北营钢CP001"  (30億元)<ref>[http://www.shclearing.com/xxpl/cwbg/sjb/201510/t20151030_112880.html 本溪北营钢铁(集团)股份有限公司2015年度三季度财务报表]</ref>、"12 本钢MTN1" (30億元)、"14 本钢MTN1"(20億元)、"14 本钢PPN1"(27億元)、"15本钢01"(15億元)、"15本钢SCP002"(10億元)、"15本钢SCP003"(10億元)、"15本钢SCP004"(10億元)、"16本钢SCP001"(10億元)、"16本钢SCP002"(20億元)公司債券。<ref name=BIS2015AR/>

同年6月遼寧省國資委將20%股權無偿划转至[[辽宁省社保基金|辽宁省社会保障基金理事会]]。<ref>{{Cite web |url=http://epaper.stcn.com/paper/zqsb/html/epaper/index/content_838819.htm |title=本钢板材股份有限公司关于实际控制人股权无偿划转事项的公告 |access-date=2016-10-01 |archive-url=https://web.archive.org/web/20170305001846/http://epaper.stcn.com/paper/zqsb/html/epaper/index/content_838819.htm |archive-date=2017-03-05 |dead-url=yes }}</ref>

2016年8月遼寧省國資委招標出售本钢集团、[[华晨汽车集团|华晨汽车集团]]、[[辽宁交通集团|辽宁交通集团]]、[[辽宁省水资源集团|辽宁省水资源集团]]、[[辽宁能源集团|辽宁能源集团]]、[[辽渔集团|辽渔集团]]、[[抚矿集团|抚矿集团]]、[[瀋煤集團|瀋煤集團]]和[[铁法能源|铁法能源]]9家地方國企。<ref>[http://www.sprtc.com/html/2016/zsxx_0823/3819.html 辽宁发展混合所有制经济引入战略投资者股权转让公告] {{Wayback|url=http://www.sprtc.com/html/2016/zsxx_0823/3819.html |date=20161016205906 }} 2016-08-23</ref> 但在同月31日本钢集团不在推介名單之中。<ref>[http://www.sprtc.com/html/2016/zsxx_0831/3840.html 辽宁省属7户国有企业股权转让推介]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>
