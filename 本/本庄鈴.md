'''本庄鈴'''（{{lang-ja|本庄鈴}}，{{bd|1997年|1月12日}}），[[日本|日本]]的[[AV女優|AV女優]]，[[山梨縣|山梨縣]]出身<ref name="SOD"/>。現為「[[Soft_on_Demand|SOD Create]]」專屬女優<ref name="SOD"/>，所屬於「C-more Entertainment」事務所<ref name="C-more"/>。

== 人物 ==
*[[山梨縣|山梨縣]][[鄉村|鄉村]]出身，父母的家中是[[葡萄|葡萄]][[果園|果園]]<ref name="週プレ"/>。[[高中|高中]]是讀{{link-ja|農業高中|農業高等学校}}並主修[[烹飪|烹飪]]，但是大學之後則在[[日本行政區劃|縣]]內的[[護理學系|看護系]]就讀<ref name="週プレ"/>。出道前仍住在山梨縣，在2018年6月23日前往[[東京|東京]]<ref name="週プレ"/>。
*興趣是[[閱讀|閱讀]]、聽[[音樂|音樂]]和[[料理|料理]]。擅長[[算盤_(東亞)|算盤]]跟[[運動|運動]]<ref name="C-more"/>。
*初體驗是在高二的冬天。在出道之前與三個人發生過關係。
*討厭條狀的食物像是[[豌豆|豌豆]]，最喜歡[[蛋糕|蛋糕]]跟[[生魚片|生魚片]]。
*在高中時期就非常仰慕AV女優[[明日花綺羅|明日花綺羅]]，特別是明日花的時尚感就像[[名流|名流]]一樣，讓她覺得與其他女優很不同，所以想要成為像明日花一樣的名人<ref name="週プレ"/>。她也是[[明里紬|明里紬]]的粉絲<ref name="しらべぇ">{{Cite web |url=https://sirabee.com/2018/03/05/20161513476/ |title=「1万本の事前予告でAV女優に」の美少女　本庄鈴が目標達成でデビュー決定 |publisher=しらべぇ |date=2018-03-05 |accessdate=2018-10-16|language=ja}}</ref>。
*藝名是自己取的<ref name="しらべぇ"/>。也常常泡[[三溫暖|三溫暖]]來保養皮膚<ref name="しらべぇ"/>。最終目標希望可以成為「世界級的模特兒」<ref name="週プレ"/>。

== 簡歷 ==
[[File:Honnjousuzu_Sign.png|200px]]
在出道之前她曾問自己是否需要裸體，為此她仍然無法下定決心，後來日本[[AV片商|AV片商]]「[[Soft_on_Demand|SOD Create]]」說服了她，並在2018年1月啟動了名為「予約1万本で超絶美少女がAVデビュー!?」的互動式企劃且尚未公布女優姓名，聲稱作品預約達到1萬次該女優才會出道<ref>{{Cite web |url=https://sirabee.com/2018/01/18/20161461066/ |title=予約1万本で超絶美少女がAVデビュー？　前代未聞のプロジェクトを本人に直撃 |publisher=しらべぇ |date=2018-01-18 |accessdate=2018-10-16|language=ja}}</ref>，她也一直到2月28日作品預約截止前確認達到1萬次才決定[[AV|AV]][[出道|出道]]<ref>{{cite web|url=http://www.playno1.com/article-26297-1.html#.W8WRYGgzaUk |title=打敗高橋しょう子！一萬片預約成功！本庄鈴、出道！ |publisher=AVNo.1-PLAYNO.1 |date=2018-03-05}}</ref><ref name="週プレ">{{Cite web |url=https://wpb.shueisha.co.jp/news/mens/2018/07/11/106546/ |title=山梨の天然美少女が予約１万本達成でＡＶデビュー！　本庄鈴、最終目標は「世界で通用するモデルに...」|author=河合桃子 |publisher=[[週刊Playboy|週プレNEWS]] |date=2018-07-11 |accessdate=2018-10-16|language=ja}}</ref>。

2018年3月5日，她的第一個裸體寫真出版在當期的《[[週刊Playboy|週刊プレイボーイ]]》中。3月9日，她在《SOD AWARD 2018ユーザー大賞》上進行了一般性的自我介紹。

2018年6月15日，個人出道作獲得成人網站《[[DMM.com|DMM.R18]]》上半年銷售排行榜的第一名<ref>{{cite web|url=http://www.playno1.com/article-27206-1.html#.W8Wf52gzaUk |title=2018上半年銷售排行榜公布！這支作品賣得最好！ |publisher=AVNo.1-PLAYNO.1 |date=2018-06-16}}</ref>。

2018年10月6日，參與[[香港|香港]][[YouTuber|YouTuber]][[大J|大J]]的影片<ref>{{YouTube|st7Cjvseils|與本庄鈴玩野球骰子！衣服全部脫掉 ？！}}</ref>。
