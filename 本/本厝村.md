'''柳溝村'''位於嘉義縣溪口鄉東邊，是臺灣省嘉義縣溪口鄉轄下的行政區。面積約3.24平方公里，轄內包含17鄰，至西元2017年6月止，人口為1470人，全境地勢平緩，為一純樸農村，村內阿蓮庄聚落中之天后宮是間已有167年歷史的廟宇。

== 地理位置 ==
'''柳溝村'''是台灣省嘉義縣溪口鄉轄下的14個行政區之一，東與溪口鄉[[疊溪村|疊溪村]]為鄰、西接溪口鄉[[坪頂村|坪頂]]、[[妙崙村|妙崙]]及[[本厝村|本厝]]三村、北隔[[三疊溪|三疊溪]]和[[大林鎮_(台灣)|大林鎮]]相對、南與[[民雄鄉|民雄鄉]]比鄰<ref name=":0">{{Cite web|url=https://www.google.com.tw/maps/place/623%E5%98%89%E7%BE%A9%E7%B8%A3%E6%BA%AA%E5%8F%A3%E9%84%89%E6%9F%B3%E6%BA%9D%E6%9D%91/@23.5847746,120.4099116,15z/data=!3m1!4b1!4m5!3m4!1s0x346ebc3134f7b527:0x62f58140ec006fcb!8m2!3d23.5814155!4d120.4155972?hl=zh-TW|title=GOOGLE MAP(嘉義縣溪口鄉柳溝村)|accessdate=2017-07-05|author=|date=|publisher=}}</ref>。

全村由北往南包括阿蓮庄、柳仔溝及南靖厝等三大聚落，日治時期皆屬打貓南堡柳子溝庄<ref name=":1">{{Cite book|title=溪口懷舊思想起|last=陳|first=席卿|last2=鄭|first2=淑琴|publisher=嘉義縣溪口鄉公所|year=2009|isbn=978-986-02-1929-6|location=嘉義縣|pages=25;26}}</ref>，由嘉93縣道貫穿串連，轄區最東側則有國道一號經過<ref name=":0" />，現有民雄分局柳溝派出所及柳溝國民小學兩政府機關<ref name=":2">{{Cite web|url=http://sikou.cyhg.gov.tw|title=嘉義縣溪口鄉公所-各村介紹-柳溝村|accessdate=2017-07-04|author=|date=|publisher=}}</ref>。

== 聚落歷史 ==
清朝時期屬嘉義縣打貓南堡，分有柳子溝庄及南靖厝庄；西元1901年屬嘉義廳打貓支廳雙溪口區打貓南堡柳子溝庄，並有柳子溝、南靖厝及阿連之土名；至西元1920年，改制屬台南州嘉義郡溪口庄柳子溝，前述三個土名仍存；直到國民政府來台，改制屬嘉義縣溪口鄉柳溝村，包含柳子溝、南靖厝與阿連庄三聚落<ref name=":1" />，茲將各聚落名稱由來分述於後：
# 阿連庄：地名因聚落範圍廣大，而取其庄頭連著庄頭之意，據嘉義縣溪口鄉土史料所載，本聚落為溪口鄉於清朝年間的四大庄之一（其他三大庄分別為：溪口庄、游厝庄及本廳庄），因庄內有土匪居住且吸食鴉片而有「鴉片庄」之稱（因其名不雅，日據時已改稱現名）。此聚落早期居民以莊、王、楊三姓最多，庄內有一聞人叫莊芋，是一個土匪，後受清廷招降，至大陸平番受封為五品官。在返回嘉義途中，行經淡水溪（今屏東、旗山附近）被人以毒餌計誘而死，其後代子孫現仍居於阿蓮庄內<ref name=":3">{{Cite web|url=http://gn.moi.gov.tw/geonames/index.aspx|title=地名資訊服務網|accessdate=2017-07-04|author=|date=|publisher=}}</ref>。
# 柳仔溝：名稱源於此聚落內的圳溝旁有著一棵大柳樹，故而得名<ref name=":3" />。
# 南靖厝：本聚落主要姓氏為莊姓，為福建省漳州府南靖縣遷徙而來之移民，聚落名稱是為紀念祖先來源而命名，開基祖為莊三郎，來此距今已經303年，在莊姓之外，曾姓為本聚落的第二大姓氏，但現在居民姓氏已混雜<ref name=":3" />。

== 教育 ==
柳溝村轄內的教育單位為柳溝國民小學，設立於西元1942年2月4日，為溪口國民學校(今溪口國民小學)柳溝分教場，教員劉興儀為首位分教場主任，西元1946年2月1日獨立為溪口鄉第二國民學校，由原任分校場主任的李傳龍先生升任首任校長，1947年2年24日改稱柳溝國民學校，1968年8月1日改稱柳溝國民小學，同年9月1日設立柳溝國民小學疊溪分校<ref>{{Cite web|url=http://www.lgps.cyc.edu.tw/|title=嘉義縣溪口鄉柳溝國民小學網頁|accessdate=2017-07-05|author=|date=|publisher=}}</ref>。

現今學區含括疊溪村、柳溝村、本厝村、妙崙村 (1、2、10鄰)等4村<ref>{{Cite web|url=http://edu.cyhg.gov.tw/News_Content.aspx?n=8B04CBD4471A6D45&s=FA655C580A05CA56|title=嘉義縣政府教育處-嘉義縣國民小學學區劃分表|accessdate=2017-07-05|author=|date=|publisher=}}</ref>。

== 文化資源或文化資產 ==
永興宮：位於柳溝村9鄰柳子溝29號，為柳仔溝信仰中心，主祀神農帝（五榖王）、徐三王爺、觀音菩薩及柳絲王。據當地耆老表示，柳絲王是相當特別罕見的神祇，曾顯赫一時，而主神神農帝也十分靈驗，每年農曆四月誕辰時，酬神戲碼要演上整個月。<ref name=":4">{{Cite web|url=http://crgis.rchss.sinica.edu.tw/|title=文化資源地理資料庫|accessdate=2017-07-04|author=|date=|publisher=}}</ref><ref name=":3" />

天后宮：位於柳溝村阿連庄35號，為阿蓮庄的庄頭廟，主祀神明為天上聖母，合祀觀音佛祖、福德正神、神農聖帝、三仙國王等。依嘉義縣鄉土史料記載，本廟曾於西元1982年重建，肇建於西元1850年（清道光三十年），廟內香爐可見「天上聖母於道光年間刻印」字樣。<ref name=":4" /><ref name=":3" />

保聖宮：位在柳溝村36之1號，創立於西元 1972年，以保生大帝為主祀神明，每年農曆一月十九日為廟方主要祭典日。<ref name=":4" /><ref name=":3" />

== 其他特色 ==
