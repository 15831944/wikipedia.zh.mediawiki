'''本傑明·「畢斯」·西格爾'''（{{lang-en|'''Benjamin "Bugsy" Siegel'''}}，{{bd|1906年|2月28日<ref name="California, Death Index, 1940–1997"/>|1947年|6月20日|catIdx=Siegel, Benjamin}}）是一名[[猶太裔美國人|猶太裔美國]]黑幫。西格爾被認為是「他的時期最臭名昭著和可怕的黑幫」之一。他被描述為英俊和有魅力，成為第一個頭版的名流黑幫之一。<ref>{{Cite web |url = http://query.nytimes.com/gst/abstract.html?res=9D06E3DB1339E632A25751C2A9609C946693D6CF&legacy=true |title = SIEGEL, GANGSTER, IS SLAIN ON COAST |date = 1947-06-22 |publisher = The New York Times |language = en}}</ref>他也是[[賭城大道|賭城大道]]發展背後的推動力。西格爾不僅在{{link-en|猶太裔美國有組織犯罪|Jewish-American organized crime|猶太黑幫}}中有影響力，而且像他的朋友和黑幫[[邁爾·蘭斯基|邁爾·蘭斯基]]一樣，也在[[意大利裔美國黑手黨|意大利裔美國黑手黨]]和意大利裔猶太[[全國犯罪集團|全國犯罪集團]]中擁有重要的影響力。

西格爾是{{link-en|謀殺公司|Murder, Inc.}}的創始人和領導者之一<ref>{{Cite web |url = http://www.laborers.org/Murder.html |title = Killer Ring Broken; 21 Murders Solved |date = 1940-03-19 |publisher = New York Daily News |language = en}}</ref>，並在[[禁酒時期|禁酒令]]期間成為一名{{link-en|朗姆酒飛車|Rum-running|私酒販}}。1933年禁酒令被廢除後，他轉向[[賭博|賭博]]方面。1936年，他離開[[紐約州|紐約]]，搬到[[加利福尼亞州|加利福尼亞州]]。在1939年，西格爾謀殺了黑幫{{link-en|哈里·格林伯格|Harry Greenberg}}。西格爾在1942年被宣判無罪。

西格爾前往[[內華達州|內華達州]]的[[拉斯維加斯|拉斯維加斯]]旅遊，他在那裡處理和資助了一些初期的賭場。他在開發商{{link-en|威廉·R·威爾克森|William R. Wilkerson}}用完資金後，便協助威爾克森的[[火鶴酒店|火鶴酒店]]。西格爾接管了這個項目，並管理最後的施工階段。火鶴酒店於1946年12月26日開張，但接待不善，不久便關閉了。酒店於1947年3月重新開張。三個月後，在1947年6月20日，西格爾在她女朋友{{link-en|維吉尼亞·希爾|Virginia Hill}}的[[比華利山|比華利山]]家中被槍殺。

== 流行文化 ==
* [[馬里奧·普佐|馬里奧·普佐]]的小說《[[教父_(小說)|教父]]》（1969年）中的角色{{link-en|莫·格林|Moe Greene}}是基於西格爾而成的。[[教父_(電影)|1972年改編電影]]中，由{{link-en|亞歷士·洛可|Alex Rocco}}飾演莫·格林。
* [[塞吉歐·李昂尼|塞吉歐·李昂尼]]的電影《[[四海兄弟|四海兄弟]]》（1984年）是鬆散地基於西格爾和蘭斯基的一生。<ref>{{Cite web |url = https://sabotagetimes.com/tv-film/once-upon-a-time-in-america-de-niro-on-the-film-that-broke-sergio-leone |title = Once Upon A Time In America: De Niro On The Film That Broke Sergio Leone |author = Richard Luck |date = 2013-11-05 |publisher = Sabotage Times |language = en}}</ref>
*《[[豪情四海|豪情四海]]》（1991年）是西格爾的半虛構傳記電影，由[[華倫·比提|華倫·比提]]飾演西格爾。
*1991年犯罪電影《[[江湖好漢|江湖好漢]]》描述了[[黑手黨委員會|黑手黨委員會]]的興起，由{{link-en|理查·格里科|Richard Grieco}}飾演西格爾。
* [[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》中，由[[米高·澤根|米高·澤根]]飾演西格爾。
