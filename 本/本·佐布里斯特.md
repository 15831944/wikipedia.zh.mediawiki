'''本杰明·托马斯·“-{zh-cn:本;zh-tw:班;zh-hk:賓;}-”·佐布里斯特'''（'''Benjamin Thomas "Ben" Zobrist'''，{{bd|1981年|5月26日|||Zobrist, Ben}}出生于[[美国|美国]][[伊利诺伊州|伊利诺伊州]][[尤里卡_(伊利诺伊州)|尤里卡]]）是[[美国职棒大联盟|美国职棒大联盟]][[坦帕湾光芒|坦帕湾光芒]]的球员。他曾经打过除[[投手|投手]]和[[捕手|捕手]]外所有的位置，{{MLBy|2009年}}起成为光芒队的主力[[二垒手|二垒手]]。他有一个昵称叫“'''佐里拉'''”（Zorilla），是芝加哥小熊总教练[[乔·梅登|乔·梅登]]给他取的<ref name="zorilla">{{cite web | url = http://www.nytimes.com/2009/07/19/sports/baseball/19zobrist.html?ref=sports | last = Schwarz | first = Alan | title = For Rays' Zobrist, Versatility Meets Opportunity | work = [[纽约时报|纽约时报]] | date = 2009-07-19}}</ref>。

==早年生活==
佐布里斯特在他的出生地尤里卡长大，从8岁开始喜爱打棒球，他和朋友在他的房子背后建了一座威浮球场。高中毕业后，并没有职棒和大学球探看重他，他认为他的棒球生涯要结束了。“在我的脑海中没有任何关于棒球的想法，”佐布里斯特说道，“在我最后一场高中棒球比赛结束之后，我驾车到处乱晃，只想着我的余生和棒球无缘了。”尽管如此，他的高中教练鼓励他参加在皮奥里亚举行的年度棒球赛事，来展示他的才华。佐布里斯特参加了该项赛事，并最终获得了[[奥立佛拿撒勒大学|奥立佛拿撒勒大学]]的青睐。在奥立佛时期，他担任过投手、游击手以及二垒手。大四那年他转学到[[达拉斯浸会大学|达拉斯浸会大学]]<ref name="tampabay.com">{{cite web | url = http://tampabay.com/sports/baseball/rays/article1018198.ece | last = Topkin | first = Mark | title = Tampa Bay Rays' Ben Zobrist has taken a surprising path to today's All-Star Game | work = 圣彼得堡时报 | date = 2009-07-14 | deadurl = yes | archiveurl = https://web.archive.org/web/20110806002653/http://www.tampabay.com/sports/baseball/rays/article1018198.ece | archivedate = 2011年8月6日 | df =  }}</ref>。

==职棒生涯==
===[[坦帕灣魔鬼魚|坦帕灣魔鬼魚]]===
佐布里斯特在[[2004年美国职棒大联盟选秀|2004年大联盟选秀大会]]上被[[休士顿太空人|休士顿太空人]]第六轮选中。

{{MLBy|2006年}}7月12日他被交易到魔鬼鱼<ref name="zorilla"/>。

佐布里斯特在魔鬼鱼队的前两个赛季打击方面始终不太理想，{{MLBy|2007年}}的打击率甚至只有1成55。之后他遇到了一位打击教练，那位教练给了他很大的帮助。佐布里斯特在{{MLBy|2008年}}有了很大的提升。虽然多数作为替补游击手出场，但整个赛季有12支全垒打的表现。守备方面他是一位工具人，2008赛季一共守过6个位置。在[[2008年世界大赛|世界大赛]]第三场面对[[费城费城人|费城费城人]]的比赛中，9局下半为了守住一人出局满垒的局面，佐布里斯特甚至充当了第五内野手。

{{MLBy|2009年}}开季初期佐布里斯特主要作为右外野手出场，在[[岩村明宪|岩村明宪]]受伤后他成为球队的主力二垒手。这个赛季他继续有着优异的表现，他一共击出27支全垒打，其中3支是[[满贯全垒打|满贯全垒打]]，长打率名列美联前茅。他也因此第一次入选了[[2009年美国职棒大联盟全明星赛|全明星赛]]<ref>{{cite web | url = http://www.tennessean.com/article/20090707/SPORTS04/907070348/1002/SPORTS/Franklin+s+Ben+Zobrist+earns+spot+as+All-Star+reserve | title = Franklin´s Ben Zobrist earns spot as All Star reserve }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。美国棒球作家协会评选他为光芒队年度MVP<ref>{{cite web | url = http://blogs.tampabay.com/rays/2009/10/ben-zobrist-named-rays-mvp-niemann-the-rookie-of-the-year.html | title = Ben Zobrist named Rays MVP; Niemann the Rookie of the Year}}</ref>。

===奧克蘭運動家===
2015年，佐布里斯特轉戰奧克蘭，隨後因運動家戰績不理想，在季中交易被出清到堪薩斯。

===堪薩斯皇家===
靠著多功能的防守與優秀的打擊火力，佐布里斯特成功幫助皇家拿下隊史第2座總冠軍。

===芝加哥小熊===
2016年，佐布里斯特與[[喬·梅登|喬·梅登]]再續前緣。不但以防守工具人的身分參與防守，豐富的經驗也正是年輕的小熊核心最需要的。同年也帶領著小熊隊奪得世界大賽冠軍，打破108年的魔咒，連續兩年幫助不同球隊拿下世界大賽冠軍，並獲得[[世界大賽最有價值球員獎|世界大賽最有價值球員獎]]。
[[File:Ben_Zobrist_with_2016_World_Series_MVP_trophy.jpg|thumb]]，獲得[[世界大賽最有價值球員獎|世界大賽最有價值球員獎]]。]]



==私人生活==
佐布里斯特的妻子朱莉安娜是一位基督歌手，他们住在[[田纳西州|田纳西州]]。他们拥有一个儿子，在2009年2月1日出生<ref>{{cite web | url = http://celebrity-babies.com/2009/02/12/ben-and-julianna-zobrist-welcome-son-zion-benjamin/ | title = Ben and Julianna Zobrist Welcome Son Zion Benjamin | work = Celebrity Baby Blog | date = 2009-02-12}}</ref>。

佐布里斯特常常谈论他的基督信仰，上帝是如何帮助他，使他认识到他会以棒球作为他的职业。他常常在接受采访时说道他整个棒球生涯是如何受到祝福。他和同样是基督徒的队友[[盖比·格罗斯|盖比·格罗斯]]（[[:en:Gabe_Gross|Gabe Gross]]）讨论过如何组织队友学习圣经<ref name="tampabay.com"/>。

佐布里斯特曾经还是冠军美国基督营的顾问<ref name="piniat">{{cite web|url=http://www.athletesinaction.org/news/post/Former-AIA-Player-Makes-it-to-the-e2809cBig-Leaguee2809d.aspx |last=Piniat |first=Elaine |title=Former AIA Player Makes it to the 'Big Leagues' |work=Athletes in Action |date=2009-06-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20110725023533/http://www.athletesinaction.org/news/post/Former-AIA-Player-Makes-it-to-the-e2809cBig-Leaguee2809d.aspx |archivedate=2011-07-25 }}</ref>。{{MLBy|2008年}}整个赛季和{{MLBy|2009年}}的开季阶段佐布里斯特使用基督Rap作曲家TobyMac的歌“Ignition”作为他的入场音乐，他现在使用的入场音乐是他妻子写的。
