'''本溪广播电视台'''由本溪电视台与本溪人民广播电台于2012年合并组成，其中本溪电视台始创于1983年10月，本溪人民广播电台成立于1951年1月。目前拥有5套电视节目和4套广播频率，电视频道和广播频率仍分别沿用原“本溪电视台”和“本溪人民广播电台”呼号。信号覆盖[[本溪市|本溪市]]及[[辽宁省|辽宁省]]东部市县，可收视人口约500万人。

2017年5月10日，本溪广播电视台及[[本溪日报|本溪日报]]社联合组建的本溪市新时代传媒（集团）有限公司正式挂牌成立。

== 频道<ref name="channel">[http://www.bxgdw.com/system/2013/03/01/010063688.shtml 本溪广播电视台3月1日华丽蜕变全新呈现]</ref> ==

=== 电视频道 ===
其中新闻频道为开路播出，其它频道为闭路播出。信号覆盖本溪市区及各个县区。
*新闻频道（本溪1套）：主频道，以新闻节目为主，主要节目有《本溪新闻》、《新视界》、《新闻早班车》、《记录本溪》等。
*文体频道（本溪2套）：原本溪有线电视台一套，每天播出18个小时的节目。由资讯、栏目、综艺、电视剧集、大型电视活动构成。
*生活频道（本溪3套）：原本溪有线电视台二套，以社教娱乐为主题，以普通市民为中心目标受众，以“提供生活信息，点滴为民服务”为基本内容。
*图文频道
*法制频道

=== 广播频道 ===
*综合广播（AM 1296kHz，FM 94.0MHz）
*经济广播（FM 107.4MHz）：又称“交通经济台”、“交通广播”
*生活广播（FM 98.0MHz）

==== 已停播频率 ====
以下频率从未出现在[[国家新闻出版广电总局|国家新闻出版广电总局]]的《广播电视播出机构及频道频率名录》中，为本溪广电擅自设立的频率<ref>{{cite web | url=http://www.sarft.gov.cn/shanty/resource/appendix/2013/01/14/20130116102112200275.xls | title=广播电视播出机构及频道频率名录 | publisher=国家新闻出版广电总局 | accessdate=2014-03-05}}</ref>，目前均已停播。
*生活娱乐台（曾先后使用FM 96.4MHz和AM 1296KHz）
*评书故事台（曾先后使用FM 104.1MHz和AM 900KHz）
*城乡广播（曾使用FM 104.1MHz）

==本溪广播电视报==
《本溪广播电视报》创刊于1994年12月22日，为本溪广播电视台旗下的综合性周报。平时出版4开40版。发行面向全省。自称是“本溪地区发行量最大的报纸”<ref>[http://www.bxmhw.com/gdb/index.shtml 《本溪广播电视报》官网]</ref>。

== 有线电视业务 ==
本溪有线电视网于1992年经[[本溪市|本溪市]]政府批准和原[[广播电影电视部|广播电影电视部]]审批成立，起初由原本溪有线电视台运营。本溪有线电视台与本溪电视台合并后，改由本溪电视台运营，之后又成立了“本溪有线电视网络公司”，负责本溪四个市辖区（[[平山|平山]]、[[明山|明山]]、[[溪湖區|溪湖]]、[[南芬|南芬]]）以及[[桓仁满族自治县|桓仁县]]的有线电视业务。2007年，本溪有线电视网络公司对全市[[有线电视|有线电视]]网络进行数字化升级改造，目前本溪有线电视网络传送基本电视节目、专业付费电视频道总计100多套，还提供多媒体数据广播、[[视频点播#准视频点播|NVOD]]等业务<ref>[http://www.bxmhw.com/szds/index.shtml 本溪数字电视介绍]</ref>。2012年年底，本溪有线电视开始试播[[高清|高清]]频道<ref>{{Cite web |url=http://bbs.asiatvro.com/thread-683384-1-1.html |title=辽宁省本溪市有线高清开始测试播出 |access-date=2013-09-30 |archive-url=https://web.archive.org/web/20150402192312/http://bbs.asiatvro.com/thread-683384-1-1.html |archive-date=2015-04-02 |dead-url=yes }}</ref>，并于次年9月正式开通<ref>[http://www.bxgdw.com/system/2013/09/04/010162041.shtml 本溪数字电视系统全面升级]，本溪电视台《本溪新闻》2013-09-04</ref>，但直到2017年底才开始正式发售[[高清|高清]][[机顶盒|机顶盒]]。
