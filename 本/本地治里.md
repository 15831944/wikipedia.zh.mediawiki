'''本地治里'''（[[泰米尔语|泰米尔语]]：பாண்டிச்சேரி，[[印地语|印地语]]：पॉंडिचेरी，[[法语|法语]]：Pondichéry，英文：Pondicherry，泰米尔语拉丁字母转译为作Puduchery），又譯為'''朋迪榭里'''，其名於[[泰米爾語|泰米爾語]]中意為「新村」。本地治里既是[[印度|印度]]的一個[[聯邦屬地|聯邦屬地]]的名稱，亦是該聯邦屬地轄下其中一個地區的名稱。為免混淆，前者可標明為'''本地治里聯邦屬地'''，後者可稱為'''[[本地治里市|本地治里市]]'''。

== 地理 ==
'''本地治里聯邦屬地'''由四個分散在[[印度|印度]]沿海、相互不連接的前[[法屬印度|法屬印度]]地區組成，分別為[[孟加拉灣|孟加拉灣]]沿岸的'''[[本地治里市|本地治里]]'''、'''[[卡來卡|卡來卡]]'''（Karaikal）和'''[[亚南|亚南]]'''（Yanam），和位於[[阿拉伯海|阿拉伯海]]沿岸的'''[[马埃|马埃]]'''（Mahé），四處共組一行政區域，以最大的本地治里命名。本地治里地區及卡來卡地區皆為[[泰米爾納德邦|泰米爾納德邦]]所包圍，雅南地區被[[安得拉邦|安得拉邦]]包圍，而马埃則地區為[[喀拉拉邦|喀拉拉邦]]所包圍。四個地區的面積分別為：本地治里地區293平方公里、卡來卡地區160平方公里、雅南地區30平方公里、马埃地區9平方公里。四處地區於2001年合共有人口97萬。
== 轄縣 ==
{{Main|本地治里行政区划}}
本地治里聯邦屬地下分4縣。

== 歷史 ==
{{Seealso|法屬印度}}

儘管本地治里很早已有[[泰米爾人|泰米爾人]]居住，但其近代史卻與[[歐洲|歐洲]]於[[南亞次大陸|南亞次大陸]]的[[殖民|殖民]]活動密不可分。[[法國東印度公司|法國東印度公司]]於1673年在本地治里建立商貿據點，成為[[法國|法國]]於南亞次大陸的主要根據地。其後[[荷蘭|荷蘭]]和[[英國|英國]]皆曾為爭奪該地而與法國發生戰爭。1693年荷蘭攻陷本地治里，但於1699年根據[[里斯維克和約|里斯維克和約]]將該地歸還法國。其後法國於1720年代取得马埃、於1731年取得雅南、於1738年取得卡來卡。於[[英法戰爭|英法戰爭]]期間（1742年至1763年），本地治里曾多次易手。1761年1月16日英國攻陷當地，但於1763年根據[[巴黎和約|巴黎和約]]歸還法國，然而英國於1793年再乘[[法國大革命|法國大革命]]再度攻下本地治里，但於1814年再度歸還法國。於1850年代後，英國取得了幾乎整個南亞次大陸的控制權，但仍允許法國繼續保留於本地治里、马埃、雅南、卡來卡等地的治權，成為法屬印度的一部份。

1947年[[印度|印度]]獨立，法國與印度於1948年達成協議，由法屬印度的人民公決其前途。於各法屬印度屬地中，[[金德訥格爾|金德訥格爾]]在1952年直接回歸印度並拼入相鄰的邦，而本地治里、卡來卡、雅南及马埃四個地區則以「本地治里」的名義於1954年11月1日加入印度成為一聯邦屬地，但[[法國國會|法國國會]]至1963年方確認與印度簽署的相關條約。

== 與法國的聯繫 ==
目前[[法語|法語]]依然是本地治里的常用語言之一。此外由於本地治里回歸印度時，法國政府允許當地人民選擇保留法國[[國籍|國籍]]或歸化印度國籍，因此當地不少泰米爾裔人及其後代至今依然保留法國國籍。法國於本地治里設有[[領事館|領事館]]，當地仍有[[法國文化協會|法國文化協會]]及[[法國遠東學院|法國遠東學院]]等機構。於本地治里，每年的[[法國國慶日|法國國慶日]]（7月14日）皆會有身穿法國軍服的巡遊活動，沿街高唱「[[馬賽曲|馬賽曲]]」，不少屋頂皆會於當日在[[印度國旗|印度國旗]]旁同時懸起[[法國國旗|法國國旗]]。

== 2004年海嘯 ==
[[2004年印度洋大地震|2004年印度洋大地震]]導致的[[海嘯|海嘯]]，使本地治里聯邦屬地下的本地治里地區及卡拉卡地區受到直接影響。根據本地治里聯邦屬地政府網站的資料顯示，直至2005年3月9日，本地治里地區有107人死亡，299人受傷；卡拉卡地區有492人死亡，280人受傷，31人失踪。
