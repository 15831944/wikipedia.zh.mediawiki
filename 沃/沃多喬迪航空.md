'''沃多喬迪航空'''是一個位於[[捷克|捷克]]（前[[捷克斯洛伐克|捷克斯洛伐克]]）的飛機製造公司。沃多喬迪航空成立於1919年，著名產品有[[L-29海豚|L-29海豚]]、[[L-39信天翁|L-39信天翁]]等。

自1989年的[[天鵝絨革命|天鵝絨革命]]及蘇聯解體後，沃多喬迪航空失去了許多在中東歐地區的噴射教練機市場，而北約國家不接受這東歐產品，使公司的軍用機銷銷售在1990年代下降，而在1998年至2004年間沃多喬迪航空更被[[波音公司|波音公司]]控制<ref>{{cite web |url=http://www.aero.cz/en/history.html |title=存档副本 |accessdate=2012-03-23 |0=2011-06-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20120207064353/http://www.aero.cz/en/history.html |archivedate=2012-02-07 }}</ref>。在2006年進行了私有化，捷克的投資公司更花30億[[捷克克朗|克朗]]購買公司的股份<ref>{{cite web |url=http://www.aero.cz/en/company-profile.html |title=存档副本 |accessdate=2011-07-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20110718164709/http://www.aero.cz/en/company-profile.html |archivedate=2011-07-18 }}</ref><ref>{{cite web |url=http://www.aero.cz/en/history.html |title=存档副本 |accessdate=2012-03-23 |0=2011-06-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20120207064353/http://www.aero.cz/en/history.html |archivedate=2012-02-07 }}</ref>。

目前沃多喬迪航空協助生產[[賽考斯基S-76精靈直升機|賽考斯基S-76精靈直升機]]，[[ERJ-170系列|ERJ-170系列]]的機門副生產線，[[UH-60黑鷹直升機|UH-60黑鷹直升機]]的駕駛艙等。

沃多喬迪航空亦希望提升其[[沃多喬迪機場|沃多喬迪機場]]的跑道，接近[[布拉格|布拉格]]國際機場的水平，吸引[[廉價航空公司|廉價航空公司]]進駐<ref>{{cite web |url=http://www.aero.cz/en/group-of-companies.html |title=存档副本 |accessdate=2009-12-31 |0=2010-01-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20100113130838/http://www.aero.cz/en/group-of-companies.html |archivedate=2010-01-13 }}</ref>。
