'''沃尔夫冈·奥弗拉特'''（{{lang-de|'''Wolfgang Overath'''}}，{{bd|1943年|9月29日||catIdx=Overath, Wolfgang}}），[[德国国家足球队|前西德]]足球运动员，<ref>{{cite web | url = http://www.kicker.de/news/fussball/bundesliga/vereine/1-bundesliga/1969-70/1-fc-koeln-16/12226/spieler_wolfgang-overath.html | title = Overath, Wolfgang | language = de | publisher = kicker.de | accessdate = 2012-03-25}}</ref>曾代表国家队三次征战[[世界杯足球赛|世界杯]]，<ref name = FIFA>{{Cite web |url = http://www.fifa.com/classicfootball/players/player=174828/index.html |title = Wolfgang OVERATH German thriller, king of Cologne |work = fifa.com |language = en |accessdate = 2012-06-23}}</ref>并在1974年本土举办的世界杯中随队夺魁。<ref>{{Cite web |url = http://www.espnstar.com.cn/pub/international/2012/0112/248990_4.htm |title = Goal.com视点：细数那些无缘金球奖的巨星 |work = espnstar.com.cn |language = zh-hans |accessdate = 2012-06-23}}</ref>作为一名颇具创造力的中场组织者，奥弗拉特以其精准的传球和出众的左脚技术闻名。

== 职业生涯 ==
=== 俱乐部 ===
奥弗拉特效力的第一家俱乐部是家乡球队SSV锡格堡04，但他的足球生涯主要是在在[[科隆足球俱乐部|科隆]]度过的。奥弗拉特在科隆队效力的14个赛季中共为球队出场765次，斩获287球。1964年，20岁的奥弗拉特在加盟球队的首个赛季就有9球入账，帮助科隆队获得[[德国足球甲级联赛|德甲联赛]]冠军。1968-69赛季，奥弗拉特最后一轮的进球又帮助科隆队战胜卫冕冠军[[纽伦堡足球俱乐部|纽伦堡]]，在赛季末成功保级。<ref>{{Cite web |url = http://football.17mcp.com/team/History.aspx?id=16 |title = 资料库>科隆 |work = football.17mcp.com |language = zh-hans |accessdate = 2012-06-23}}</ref>1976年，维斯韦勒从[[巴塞罗那足球俱乐部|-{zh-hk:巴塞隆拿; zh-hans:巴塞罗那; zh-tw:巴塞隆納;}-]]帅位离职，重新回到球员时代曾效力的科隆队执教。但这位新帅与奥弗拉特关系不佳。后者在经历了一个赛季的争吵之后于1977年退役。
=== 国家队 ===
奥弗拉特在1963-1974年代表西德国家队出赛81场，入17球。在66年、70年和74年的世界杯中，他一直是德国队的中场核心。<ref>{{Cite web|url = http://www.gerfans.cn/2010/0831/54.html|title = 名人堂>奥维拉特（Wolfgang Overath）|work = gerfans.cn|language = zh-hans|accessdate = 2012-06-23|deadurl = yes|archiveurl = https://web.archive.org/web/20120512060858/http://www.gerfans.cn/2010/0831/54.html|archivedate = 2012-05-12}}</ref>[[1970年世界杯足球赛|1970年世界杯]]，奥弗拉特在三、四名决赛中攻破[[乌拉圭国家足球队|乌拉圭]]球门，随队赢得季军，<ref>{{Cite web |url = http://www.espnstar.com.cn/pub/international/2010/0711/166329.htm |title = 四夺安慰奖创纪录 复制1974神迹仍需努力 |work = espnstar.com.cn |language = zh-hans |accessdate = 2012-06-23}}</ref>而国际媒体记者也将奥弗拉特评为本届杯赛最佳中场。<ref>{{Cite web |url = http://www.planetworldcup.com/LEGENDS/overath.html |title = World Cup Legends - Wolfgang Overath |work = planetworldcup.com |language = en |accessdate = 2012-06-23}}</ref>作为世界杯历史上仅有的两名“大满贯”球员（获得过世界杯冠军、亚军和季军，另一名是奥弗拉特的队友[[弗朗茨·贝肯鲍尔|-{zh-hans:贝肯鲍尔;zh-hk:法蘭斯·碧根鮑華;zh-tw:法蘭茲·貝肯鮑爾;}-]]），奥弗拉特本应在国家队赢得更多的出场机会。但两届[[德国足球先生|德国足球先生]]得主[[君特·内策尔|-{zh-hans:内策尔;zh-hk:納薩;zh-tw:君特·内策爾;}-]]的杰出表现却只能让他在一段时间内成为教练的第二选择。不过，奥弗拉特很快又取代了内策尔的位置，如他所说，奥弗拉特“就是为德国队而生的。”<ref>{{Cite web |url = http://www.fifa.com/classicfootball/winners/code=39/player=174828/interview.html |title = Overath: Our achievement will stand forever |work = fifa.com |language = en |accessdate = 2012-06-23}}</ref>
== 其他 ==
2004年，奥弗拉特出任科隆俱乐部主席。他与两名副主席诺伊基尔希(Friedrich Neukirch)和格洛瓦茨(Jürgen Glowacz)在2005年获得连任。<ref>{{Cite web |url = http://sports.sina.com.cn/g/2005-11-15/22091882211.shtml |title = 科隆公布财政状况支出减少 主席奥弗拉特获连任 |work = 新浪体育 |language = zh-hans |accessdate = 2012-06-23}}</ref>但在2011年11月13日，奥弗拉特在俱乐部年会上突然宣布辞职。<ref>{{Cite web |url = http://www.fjsen.com/p/2011-11/14/content_6775103.htm |title = 奥弗拉特承受重压 科隆主席年会上辞职 |work = fjsen.com |language = zh-hans |accessdate = 2012-06-23 }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
奥弗拉特同样热衷公益事业。1994年，奥弗拉特建立了以自己名字命名的基金会以提供社会援助。他同时组织了若干场前国家对球员参与的义赛，将筹集的善款用于援助有需要的人士。<ref name = FIFA/>
