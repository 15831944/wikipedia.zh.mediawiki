'''觀塘瑪利諾書院'''（簡稱觀瑪、KTMC；{{lang-en|Kwun Tong Maryknoll College}}）是[[香港|香港]]一所[[全日制|全日制]]男子[[香港英文授課中學|英文中學]]。該校由[[瑪利諾外方傳教會|美國瑪利諾神父教會]]於1971年成立，學校現由[[天主教香港教區|天主教香港教區]]管理，屬天主教香港教區學校聯會成員之一，亦是香港一所擁有四十多年歷史的男生英文天主教文法學校。校舍位於[[九龍|九龍]][[觀塘|觀塘]][[翠屏道|翠屏道]]100號。

== 歷史 ==
[[File:Bwphoto1.jpg|thumb]]導致尚未啟用的校舍之東南面發生山泥傾瀉，釀成71死52傷<ref>{{cite news | title=雨災調查委員會昨繼續聆訊 警官供搶救情形 翠屏道近曉明街之安置區第一台木屋全受泥蓋 第九座徒廈外載火水貨車被山泥翻倒着火焚燒 |publisher=[[華僑日報|華僑日報]] |date=1972年7月7日 |page=第三張第二頁 }}{{zh-hant}}</ref>]]
觀塘瑪利諾書院由[[瑪利諾外方傳教會|美國瑪利諾神父教會]]於1971年創立。校歌在創校時由任教音樂的羅銳志老師作曲，首任校長江天文神父填詞，一直沿用至今。校舍於同年動工興建，興建期間由[[聖言中學|聖言中學]]借予地方作為教室。受1972年的[[六一八雨災|翠屏道山泥傾瀉]]影響，因要重新評估校舍地基安全及影響工程進度，校舍的啟用日期遭到延遲。在校舍啟用前，須借用[[聖安當女書院|聖安當女書院]]校舍上課。書院校舍最終於1972年九月正式啟用。1973年，由[[徐誠斌|徐誠斌]]主教主禮校舍奠基儀式。 

隨著觀塘瑪利諾書院的發展，學校各方面的設施和運作變得完善。首屆學生代表會於1974年成立，並草擬學生會會章。1975年，首份學生會刊物《學生專訊》出版，也產生首屆中五畢業生，[[校友會|校友會]]亦同年成立。到了1976年，首屆學生會幹事會成立，並修訂學生會大選細則，確定以全校學生一人一票方式選出幹事會。1977年，首屆中七學生畢業，同年開辦觀塘瑪利諾夜中學。 

學校在1982年開始由[[香港天主教教區|香港天主教教區]]接手管理。<ref>{{cite web |url=http://www.ktmc.edu.hk/index.php?section=History |title= 學校歷史|accessdate= 2003-09-01 |work= |publisher= ktmc|date= 2003-09-01}}</ref>1993年，中七學生提出修改學生會會章，獲全體學生投票通過。新章訂明學生會權利來自全體學生，對學生會校內活動「有獨立的行政管理權」，確立學生會四權分立的體制。修章後，學生會成立旗下的仲議會及編委會，分別負責仲裁會內糾紛及出版學生刊物。

踏入千禧，觀塘瑪利諾書院校舍新翼於2004年正式啟用。在2007年，停辦觀塘瑪利諾夜中學。為響應[[教育局|教育局]]及天主教香港教區呼籲，學校決定在2011年自願減班，由2011至2012年度起中一由5班減至4班。<ref>{{cite news |author= |last= |first= |date= 2011-02-18日|title= 教區15屬校全參與減班|url= http://www.singtao.com/yesterday/edu/0218go01.html|newspaper= 星島日報|location= 香港|access-date= 2011-02-19}}</ref>在2015至2016教學年度，學校取消「忠」、「毅」兩社，並在2016至2017年度開始將中四級的四班分拆為五班，推行[[小班教學|小班教學]]。

== 校舍 ==
校舍位於山坡上。學校主樓由首任校監任澤民神父畫則興建，依中國四合院佈局設計。

現時校舍分別設有主樓及新翼。主樓樓高三層，呈四方形狀，地下中央有一片草地，草地上豎立了一座[[耶穌|耶穌]]像石雕。主樓採用了類似天井的設計，有別於區內其他學校的設計；主樓二樓亦設有一個中庭，現為園藝學會會址，中庭的樓下為禮堂；主樓提供各班的課室，並設有校務處、教員室、科學實驗室、地理室、圖書館等等。新翼校舍則樓高五層，設有教員室、會議室、語言室、社會科學室、多用途室，等。

== 學校成就 ==
觀塘瑪利諾書院分別於2002年及2010年出產了兩位會考狀元，為[[觀塘區|觀塘區]]內的第一位與最後一位十優狀元，空前絕後，一直被視為觀塘區名校之一。

至於在新學制文憑試中，學校每年亦有約五至六成考生升讀聯招八大（港大、中大、科大、理大、城大、浸大、教大、嶺大）或其他自資院校的本地學士課程；自資副學士或高級文憑課程則有3-4成畢業生選擇修讀；另有約1成學生選擇重修文憑試。<ref>[http://www.ktmc.edu.hk/Administration/Plans/Annual_School_Report_2014-2015.pdf "Kwun Tong Maryknoll College School Report (2014-2015)"]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

此外，學校亦旨在全人發展，著重學生的體藝水平。學校的運動校隊例如足球隊、彈網隊，泳隊均屢獲佳績；2008年由謝金文老師成立的學校棒球隊更曾代表香港出賽台灣。

== 班別架構 ==

於2015年，學校約有學生900人，分佈於26個班別。中一至中五設4個班別（A、B、C、D），每班約36人；中六設5個班。每班約36人，高中後，學生可以於文、理、商三個科目範疇中自由配搭選科組合（A、B班兩科，C、D 三科）。另外，學校中六及中七兼收女生數名。但2006年後，因為夜校關閉，校方行政議會檢討政策，認為只招收三數名女生意義不大，故此不再於中六招收女生。

2011-12年度,學校有34個班別。中一級設4個班別（A、B、C、D），每班約36人，中二至中六設5個班別（A、B、C、D、E），每班約36-40人；中七設3個班別（Arts、Biology、Mathematics），每班約30人。

於2016-2017年度， 學校將中四級由四班分拆為五班， 每班約28人，學生同樣可以於文、理、商三個科目範疇中自由配搭選科組合(A、B班：4C+2X，C、D、E班：4C+3X，D、E班：4C+3X+M1/M2）

於2016-2017年度，初中每級有四班，中四有五班，中五至中六每級四班。

== 觀塘瑪利諾書院學生會 ==

觀塘瑪利諾書院學生會由學校的學生所組成，為享有高度自治權的校內學生組織，由全校學生以一人一票的方式選出內閣。

觀塘瑪利諾書院學生會主要由四大中央組織組成，分別為：學生會仲議會、學生會代表會、學生會幹事會及學生會學生報編輯委員會，以上組織以四權分立形式運作，互不從屬、互相制衡。

仲議會：由顧問老師與學生商討選出；負責一切仲裁事務，處理投訴。

代表會：由學會等代表選出；學生負責一切有關管理會內財務及執行會章的事務，例如是由學生召開會議審核各學會的資助申請、召開會議修訂學生會會章等等。

幹事會：由全體學生選出；為學生會的最高行政組織，籌辦學生活動，例如聖誕舞會、聯校口試練習、畢業班足球比賽等等。

編委會：由全體學生選出；負責編輯和印刷校報、學校刊物《長青》等。

此外，學生會還包括附屬組織，例如社、學會、班會等等。

==學生活動==

[[運動|運動]]方面，學校擁有各類運動校隊，包括[[游泳|游泳]]、[[越野跑|越野跑]]、[[田徑|田徑]]、[[足球|足球]]、[[籃球|籃球]]、[[壘球|壘球]]、彈網([[trampoline|trampoline]])、[[羽毛球|羽毛球]]及[[乒乓球|乒乓球]]等，部分多年參加各項校際比賽亦屢獲殊榮。<ref>{{cite web |url=http://www.ktmc.edu.hk/index.php?section=Achievement|title= Roll of Honor |accessdate= 2011-01-01 |work= |publisher= ktmc|date= 2011-01-01}}</ref>

在[[音樂|音樂]]演藝方面，學校擁有[[合唱團|合唱團]]、[[弦樂團|弦樂團]]、[[中樂團|中樂團]]及話劇團。學校多年來積極參與[[香港學校音樂及朗誦協會|香港學校音樂及朗誦協會]]學校音樂節多項個人及團體比賽，於多個比賽中獲獎，同時亦投入參與地區音樂交流活動；話劇團亦參加學校話劇節，每年均有獲奬。

學校目前提供了數個[[制服團體|制服團體]]，例如有[[紅十字會|紅十字會]](YU95)，[[公益少年團|公益少年團]]等。

此外，學生會屬下亦設超過三十個學會，屬下於學生會。學生可自由選擇參加兩至三個學會，亦可在老師的指導下籌辦新學會，以培養學生對各方面的知識和興趣。

另外，學校每年舉辦一次校慶紀念活動，而十年會有一次大型紀念活動，活動包括開放日、展覽、嘉年華會、音樂會、同學會聚餐等，通常在校慶年的五月期間舉行。學校2011年為慶祝創校40週年，舉辦了步行籌款、文藝綜合晚會、聚餐等慶祝活動。

為慶祝創校45週年，學校於2016年舉辦嘉年華，設立多個遊戲攤位。

== 傳統 ==
四十年來，學校師生均以「觀瑪仔」作為學生的稱呼。

每一年，在學校模擬試後“Go Over Paper”最後一日都會有「畢業前惜別會」，該年畢業禮的同學（2010年及以前是中五、中七；2011年是純中七；2012年是新高中中六、舊制中七；2013及以後是純新高中中六）與老師進入學校對面的[[耶穌復活堂|耶穌復活堂]]進行祈禱聚會，此告別禮年年舉行以帶給該年畢業生最後一個回憶，為觀瑪一個傳統； 另畢業禮同學都會身穿短袖裇衫及打上領呔，即觀瑪仔所稱的「春/秋季校服」，與平日上課的服飾不同。
