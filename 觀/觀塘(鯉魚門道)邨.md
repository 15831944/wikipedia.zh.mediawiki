'''觀塘（鯉魚門道）邨''' (下稱鯉魚門道邨) 是[[香港|香港]][[九龍|九龍]][[觀塘區|觀塘區]]第一條[[政府廉租屋邨|政府廉租屋邨]]，現已拆卸。

鯉魚門道邨共有7座舊型[[政府廉租屋|政府廉租屋]][[大廈|大廈]]，於1962年落成，此等大廈這與[[徙置大廈#第三型|第三型徙置大廈]]類似，唯[[廚房|廚房]]設備兩者有分別。該邨於1973年之前稱為'''觀塘政府廉租屋邨'''，於1997年正式拆卸。重建後成為現時[[鯉安苑|鯉安苑]]。而[[鯉魚門邨|鯉魚門邨]]與該邨同樣是座落於[[鯉魚門道|鯉魚門道]]，但鯉魚門邨位於[[油塘|油塘]]及[[鯉魚門|鯉魚門]]交界，[[油塘中心|油塘中心]]對面，其前身為嶺南新村，二者並無關係。此外，以海鮮聞名的鯉魚門寮屋區與該邨同樣並無關係。

==事蹟==
* 鯉魚門道邨是香港八九十年代著名男子組合[[草蜢_(組合)|草蜢]]三子（[[蔡一智|蔡一智]]、[[蔡一傑|蔡一傑]]及[[蘇志威|蘇志威]]）的舊居<ref>{{Cite web |url = http://hk.ulifestyle.com.hk/topic/detail.html?id=ABFGD1ozBw1ZFANt |title = HK港生活：他們都是屋邨長大的　這些名人可能曾經是你鄰居 (九龍篇)  |date = 2015-06-19 |publisher = 香港經濟日報——U Liftstyle |language = zh-hk |accessdate = 2016-05-23 }}</ref>。當年蔡一智﹑一傑兩兄弟住在本邨第四座二樓某單位<ref>[https://www.youtube.com/watch?v=CPV0l-D5Wfs “官塘鯉魚門道邨集體回憶”內留言]</ref>，蘇志威則為同屋邨兼同班同學，深厚的感情讓他們成長後一同在演藝界工作。
