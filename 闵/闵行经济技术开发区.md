'''闵行经济技术开发区'''，是位于[[上海市|上海市]][[闵行区|闵行区]]的[[国家级经济技术开发区|国家级经济技术开发区]]，简称为'''闵行开发区'''，规划时以南北向主干道称为'''昆阳工业点'''，前身为批准于1982年的'''闵行新工业区'''，总面积3.5平方公里<ref>{{cite book |title=上海通志——第三十二卷·开发区 |date=1996 |publisher=上海人民出版社 |location=上海 |url=http://www.shtong.gov.cn/newsite/node2/node2247/node4589/node79616/node79622/userobject1ai103681.html |access-date=2018-09-01 |archive-url=https://web.archive.org/web/20150902011804/http://www.shtong.gov.cn/newsite/node2/node2247/node4589/node79616/node79622/userobject1ai103681.html |archive-date=2015-09-02 |dead-url=no }}</ref>。开发区由上海闵行联合发展有限公司负责开发，自称是国家级开发区中占地面积最小的工业开发区<ref>{{cite web |author1=上海闵行经济技术开发区 |title=上海闵行经济技术开发区简介 |url=http://ezone.mofcom.gov.cn/article/ao/201210/20121008402186.shtml |website=[[中华人民共和国商务部|中华人民共和国商务部]] |accessdate=2018-09-01 |date=2012-10-25}}</ref>。

== 历史 ==
1981年3月初，上海市向时任[[中华人民共和国国务院副总理|中华人民共和国国务院副总理]][[谷牧|谷牧]]提出利用[[外资|外资]]创建出口产品工业区的设想，谷牧表示支持。[[上海市规划局|上海市规划局]]应进出口办公室的申请，提出了奉贤[[星火农场|星火农场]]、[[江川路街道|闵行]]两个选址方案。上海市政府各部门分析认为，闵行地区因基础设施较为完备，适合设立出口产品工业区。同年4月6日，进出口办公室向上海市人民政府请示建立闵行出口产品工业区；7月3日，[[上海市人民政府|上海市人民政府]]批复同意并要求市规划局编制规划方案报批；1982年8月21日，上海市规划局向市人民政府上报《闵行昆阳工业点开发规划方案》，规划建设闵行新工业区，东起沙港，西至昆阳路，北沿[[剑川路|剑川路]]，南至[[江川路|江川路]]，规划面积2.13平方公里；1982年11月8日，上海市人民政府根据市计划委员会和市建设委员会的意见，原则同意上述开发规划方案，并决定冻结[[中华人民共和国户籍制度|户口]]，在规划区停建其他工程项目。1983年，上海市开发公司管理处委托上海工业设计院编制详细规划，把昆阳工业点的名称改为闵行开发区，将昆阳路以东的高压供电走廊用地划入闵行开发区。<ref name="规划志">{{cite book |author1=《上海城市规划志》编纂委员会 |title=上海城市规划志 |date=1999 |publisher=上海社会科学院出版社 |isbn=7-80618-531-3 |url=http://www.shtong.gov.cn/Newsite/node2/node2245/node64620/node64633/node64725/node64731/userobject1ai58547.html |access-date=2018-09-01 |archive-url=https://web.archive.org/web/20160627182859/http://www.shtong.gov.cn/Newsite/node2/node2245/node64620/node64633/node64725/node64731/userobject1ai58547.html |archive-date=2016-06-27 |dead-url=no }}</ref>

1983年4月3日，时任上海市副市长[[裴先白|裴先白]]主持召开了闵行开发区征地动迁工作会议，成立了闵行开发区征地动迁领导小组。1983年6月2日，上海市人民政府决定将[[虹桥开发区|虹桥开发区]]、闵行开发区的开发机构合一，正式成立上海市闵行虹桥开发公司；同月，该公司首次邀请200多名外国商人，进行招商引资活动。1984年5月，沪港合资上海环球玩具有限公司成立在开发区成立，成为了第一家进驻开发区的企业。1985年2月，经上海市人民政府、[[对外经济贸易部|对外经济贸易部]]批准，成立了开发企业——上海闵行联合发展有限公司，总投资1.7亿元，注册资本1亿元。这家企业是一家内地与香港合资的企业，上海市闵行虹桥开发公司出资65％，[[中國銀行（香港）#中銀集團|中银集团]]5家成员行出资25％，中国银行上海市分行出资10％，全权负责闵行开发区的发展。<ref name="大事记">{{cite book |author1=上海市闵行区志编纂委员会 |title=闵行区志 |date=1996 |publisher=上海社会科学院出版社 |isbn=7-80618-083-4 |url=http://www.shtong.gov.cn/newsite/node2/node4/node2249/node2251/node2255/index.html |section=大事记 |access-date=2018-09-01 |archive-url=https://web.archive.org/web/20160410083859/http://www.shtong.gov.cn/newsite/node2/node4/node2249/node2251/node2255/index.html |archive-date=2016-04-10 |dead-url=no }}</ref>

1986年，[[中华人民共和国国务院|中华人民共和国国务院]]正式批准成立国家级闵行经济技术开发区，并约定建设资金由上海市自筹，借用外资由上海市自还<ref>{{cite wikisource|title=国务院关于上海市建立闵行和虹桥两个经济技术开发区的批复|date=1986-08-29|wslanguage=zh}}</ref>。

1991年8月，上海市人民政府上报[[国务院特区办公室|国务院特区办公室]]、[[国家计委|国家计委]]及[[海关总署|海关总署]]批准后，闵行开发区用地面积扩大至3.5平方公里。第二期扩大工程分为两块：南块由江川路扩展至黄浦江边；西块东起昆阳路，西至碧溪路，北沿大中华橡胶厂闵行分厂，南到上海重型机器厂。<ref name="经贸">{{cite book|url=http://www.shtong.gov.cn/newsite/node2/node2245/node74728/node74741/node74845/node74853/userobject1ai89840.html|title=上海对外经济贸易志|date=2001|publisher=上海社会科学院出版社|accessdate=2018-09-01|isbn=7-80618-887-8|author1=《上海对外经济贸易志》编纂委员会}}</ref>

2014年，上海闵行联合发展有限公司和上海虹桥经济技术开发区联合发展有限公司（[[虹桥开发区|虹桥开发区]]开发实体）进行了资产重组，成立了上级控股公司，即上海地产闵虹（集团）有限公司<ref>{{cite web |title=国家级闵行、虹桥开发区成立30周年纪念 |url=http://www.smudc.com/news/news_pop.asp?id=1638 |website=闵行经济技术开发区 |accessdate=2018-07-17 |archive-url=https://web.archive.org/web/20170607061113/http://www.smudc.com/news/news_pop.asp?id=1638 |archive-date=2017-06-07 |dead-url=no }}</ref>。

== 对外商优惠政策 ==
据1987年3月23日颁布、2001年1月9日失效的《[[s:上海市闵行、虹桥经济技术开发区外商投资优惠规定|上海市闵行、虹桥经济技术开发区外商投资优惠规定]]》，对入驻开发区的外商投资企业，自1987年4月1日开始，上海市对入驻闵行开发区、虹桥开发区的外商投资企业免征地方[[所得税|所得税]]，至1995年结束（后上海市地方税务局批准延长至2000年<ref>{{cite journal |author1=上海市地方税务局 |title=关于对延长本市经济技术开发区生产性外商投资企业免征地方所得税至2000年底问题的批复（沪税外[96]4号） |date=1996-01-31 |quote=你司沪闵联发（95）第82号《请求延长开发区外资生产性企业所得税免税期的报告》悉。根据《上海市闵行、虹桥经济技术开发区外商投资优惠规定》和《上海市经济技术开发区条例》有关规定，对经济技术开发区内生产性外商投资企业，在1995年底之前免征地方所得税，这一优惠政策现已届满。为了继续发挥浦西原有三个经济技术开发区的整体功能，推动浦东、浦西地区联动发展的优势，经请示市政府同意，对上海市经济技术开发区内的生产性外商投资企业免征地方所得税可自1995年底延长至2000年底。}}</ref><ref>{{cite journal |author1=上海市财政局 |author2=上海市国家税务局 |author3=上海市地方税务局 |title=关于公布废止和失效的财政税务规范性文件目录（第十一批）的通知（沪财法[2002]36号） |date=2002-03-12 |url=https://www.czj.sh.gov.cn/zys_8908/zcfg_8983/zcfb_8985/wjyxx/201509/t20150916_166203.shtml |accessdate=2018-09-01}}</ref>）。

此外，闵行开发区的开发公司也给予入驻外商企业一定优惠。入驻外资研发机构，所需土地的场地开发费给予一定的优惠；所需租用的办公、实验用房，首两年零租金，第三年租金打四折，第四年租金打八折；所需的科研设备及按合同随之进口的技术及配套件、备件，除国家规定不予免税的的商品外，免征[[关税|关税]]和进口环节[[增值税|增值税]]。<ref>{{cite web |author1=闵行经济技术开发区 |title=上海闵行经济技术开发区鼓励支持外商投资设立研发机构的办法和优惠政策 |url=http://ezone.mofcom.gov.cn/article/al/201210/20121008402182.shtml |website=[[中华人民共和国商务部|中华人民共和国商务部]] |accessdate=2018-09-01}}</ref>

== 服务中心 ==
1987年，闵行开发区实行“一个窗口、一个图章、一个机构”对外，为外商提供“一站式”服务<ref name="经贸" />。1988年1月，闵行开发区建立行政管理服务中心，该中心由上海市外资委牵头举办，为开发区内企业的生产经营、办理申报手续、处理进出口等业务提供服务，号称为全国首例；截至1992年底，上海经济技术开发区海关、上海海关进出口商品检验局、上海市税务局、国家外汇管理局上海分局、上海市财政局、中国人民保险、中国银行、中国工商银行、交通银行、中国建设银行、中国对外贸易运输总公司、上海外轮代理公司、中国外轮理货总公司上海分公司等21家单位入驻<ref>{{cite book |author1=上海市闵行区志编纂委员会 |title=闵行区志 |date=1996 |publisher=上海社会科学院出版社 |isbn=7-80618-083-4 |url=http://www.shtong.gov.cn/newsite/node2/node4/node2249/node2251/node2418/node2420/node2422/userobject1ai238.html |chapter=行政管理服务中心 |access-date=2018-09-01 |archive-url=https://web.archive.org/web/20040925220327/http://www.shtong.gov.cn/newsite/node2/node4/node2249/node2251/node2418/node2420/node2422/userobject1ai238.html |archive-date=2004-09-25 |dead-url=no }}</ref>。

此外，2002年9月28日，闵行经济技术开发区外商投资服务中心正式揭牌，首批引进了16家各类中介服务机构，服务开发区内企业<ref name="上海年鉴2003">{{cite journal |author1=上海年鉴社 |journal=上海年鉴 |title=闵行区 |date=2003 |url=http://www.shtong.gov.cn/Newsite/node2/node19828/node70632/node70685/node70834/userobject1ai72907.html |accessdate=2018-09-01 |issn=1008-1046 |author= |archive-url=https://web.archive.org/web/20171115024155/http://shtong.gov.cn/Newsite/node2/node19828/node70632/node70685/node70834/userobject1ai72907.html |archive-date=2017-11-15 |dead-url=no }}</ref>。

== 入驻企业 ==
入驻开发区的企业有中美上海[[施贵宝|施贵宝]]制药有限公司、上海环球玩具有限公司（工商执照已被吊销）、上海[[三菱电机|三菱]]电梯有限公司、上海[[太古飲料控股#資產|申美饮料]]食品有限公司、上海新华控制技术 (集团) 有限公司、上海世界针织有限公司、上海[[强生|强生]]有限公司等。<ref>{{cite book |author1=上海市闵行区志编纂委员会 |title=闵行区志 |date=1996 |publisher=上海社会科学院出版社 |isbn=7-80618-083-4 |url=http://www.shtong.gov.cn/newsite/node2/node4/node2249/node2251/node2418/node2426/node2428/userobject1ai245.html |chapter=企业选介 |access-date=2018-09-01 |archive-url=https://web.archive.org/web/20040925220426/http://www.shtong.gov.cn/newsite/node2/node4/node2249/node2251/node2418/node2426/node2428/userobject1ai245.html |archive-date=2004-09-25 |dead-url=no }}</ref>

== 对外投资 ==
闵行开发区所属上海闵行联合发展有限公司联合[[申通地铁|申通地铁]]、上海闵行城市建设投资开发有限公司组建上海莘闵轨道交通线发展有限公司，投资兴建了莘闵轻轨交通线工程，即[[上海轨道交通5号线|上海轨道交通5号线]]（[[莘庄站|莘庄站]]—[[闵行开发区站|闵行开发区站]]）。线路全长17.04公里，2002年12月8日建成试通车<ref name="上海年鉴2003" />。
