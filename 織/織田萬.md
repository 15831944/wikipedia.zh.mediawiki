'''織田 萬'''（おだ よろず、[[慶應|慶應]]4年[[七月初四|7月4日]]（[[1868年|1868年]][[8月21日|8月21日]]） - [[昭和|昭和]]20年（[[1945年|1945年]]）[[5月26日|5月26日]]）<ref name = "100-Kinen">『織田萬』（著・赤澤史朗）「立命館大学法学部 創立百周年記念誌（64頁-71頁）」発行・立命館大学法学部（2000年9月22日発行）</ref>為[[日本|日本]][[法學者|法學者]]。専門[[行政法|行政法]]。[[京都帝國大學|京都帝國大學]][[名譽教授|名譽教授]]、[[常设国际法院|常设国际法院]]判事、[[关西大学|关西大学]]學長。[[學校法人立命館|學校法人立命館]][[名譽總長|名譽總長]]<ref name = "100-Kinen"/>。[[佐賀縣|佐賀縣]]須古邑出生<ref name = "100-Kinen"/>。
