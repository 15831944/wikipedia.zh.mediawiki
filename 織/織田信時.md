'''織田信時'''（生年不詳—1556年6月）是[[日本戰國時代|日本戰國時代]]的武將。父親是[[織田信秀|織田信秀]]。別名'''秀俊'''。

== 生平 ==
生年不詳，家中六男（一說五男）。幼名'''喜藏'''。成為[[犬山城|犬山城]]城主[[織田信康|織田信康]]的養子（『[[寬政重修諸家譜|寬政重修諸家譜]]』）。[[天文_(後奈良天皇)|天文]]24年4月20日（1555年5月10日），[[織田信長|織田信長]]成功奪取[[清須城|清須城]]，把[[那古野城|那古野城]]交予立下大功的叔父兼{{tsl|ja|守山城}}城主[[織田信光|織田信光]]（『[[信長公記|信長公記]]』）。而信光的弟弟[[織田信次|信次]]則成為守山城城主。6月26日，信次的家臣[[洲賀才藏|洲賀才藏]]在守山城下[[龍泉寺|龍泉寺]]附近的{{tsl|ja|庄内川|庄內川}}以弓箭射殺正在乘馬的[[織田秀孝|織田秀孝]]（信長等的弟弟），得知此事的信次迅速逃亡，[[織田信行|織田信行]]率軍在守山城下放火。而在信長的重臣[[佐久間信盛|佐久間信盛]]的誘導下，留在城內的家臣[[角田新五|角田新五]]和[[坂井喜左衛門|坂井喜左衛門]]投向信長。此後，成為守山城城主，並賜予信盛[[知行|知行]]百石（『信長公記』）。

在信長的[[家老|家老]][[林秀貞|林秀貞]]與弟弟[[林美作守|林美作守]]和[[柴田勝家|柴田勝家]]共謀，打算背叛信長時，跟隨信長進入林兄弟所在的那古野城，因為秀貞猶疑是否要背離仕奉三代的主君，於是生還（『信長公記』）。

後來，因為提拔坂井喜左衛門的兒子[[坂井孫平次|孫平次]]成為[[若眾|若眾]]，角田新五對此感到相當嫉妒。[[弘治_(後奈良天皇)|弘治]]2年（1556年）6月某日，角田新五假裝要修理城內的塀、柵，率軍入城，此時被迫[[切腹|切腹]]（『信長公記』）。

== 其他 ==
根據『[[織田系圖|織田系圖]]』和『[[寬政重修諸家譜|寬政重修諸家譜]]』等記載，是[[織田信秀|織田信秀]]的五男或六男，不過在『[[信長公記|信長公記]]』中記載，在弘治年間已經有活動，而如果根據前記系圖，已經育有一女，因此應早生於[[天文_(後奈良天皇)|天文]]12年（1543年）出生的[[織田信包|信包]]。而『信長公記』中亦記載「[[織田信廣|織田三郎五郎（信廣）]]殿，是[[織田信長|信長]]公的異母兄。其弟為安房守（信時），是口才不錯的人」（{{lang|ja|織田三郎五郎殿と申すは、信長公の御腹かはりの御舎兄なり。其弟に安房守と申候て、利口なる人あり}}），並沒有寫成信長的弟弟，而是寫成信長的異母兄信廣的弟弟，因此被推測應是信廣的同母弟，而非信長的弟弟，可能是信長的異母兄。

被認為與'''織田秀俊'''是同一人物，在弘治2年2月，有織田安房守秀俊發予{{tsl|ja|雲興寺}}的禁制（『[[愛知縣史料叢刊|愛知縣史料叢刊]]』「織田秀俊禁制」）。

死後，[[角田新五|角田新五]]並無受到處罰，並受到林秀貞的庇護，因此實際原因不明，不過信時對於信長而言是阻礙，因此有多種說法。

== 子孫 ==
生有一女，在信時死後，成為[[池田恒興|池田恒興]]的養女，嫁予[[飯尾敏成|飯尾敏成]]，在敏成戰死後，再嫁予[[下間賴龍|下間賴龍]]，誕下下間賴廣（後來的[[池田重利|池田重利]]）（『寬政重修諸家譜』）。因此，{{tsl|ja|播磨新宮藩}}藩主[[池田氏|池田氏]]是信時的女系子孫。
