'''烟酸'''（{{lang-en|niacin}}、{{lang|en|nicotinic acid}}，也称'''維他命B<sub>3</sub>'''、'''維他命PP'''、'''吡啶-3羧酸'''），分子式：[[碳|C]]<sub>6</sub>[[氢|H]]<sub>5</sub>[[氮|N]][[氧|O]]<sub>2</sub>，耐热，能升华。首次描述於Hugo Weidel於1873年對[[尼古丁|尼古丁]]的研究。它是人体必需的13种[[维生素|维生素]]之一，是一种水溶性维生素，属于[[维生素B|维生素B]]族。烟酸在人体内转化为[[烟酰胺|烟酰胺]]，烟酰胺是[[辅酶Ⅰ|辅酶Ⅰ]]和[[辅酶Ⅱ|辅酶Ⅱ]]的组成部分，参与体内脂质代谢，组织[[呼吸作用|呼吸]]的[[氧化|氧化]]过程和[[糖类|糖类]]无氧分解的过程。<br />

== 功能 ==
# 生理上為傳遞搬運電子，使末稍血管舒張作用。
# 具糖解作用，組織呼吸，脂肪合成。（功能類似於維他命B群）

== 攝取不足後遺症 ==
# 疲勞，倦怠，頭痛，噁心，嘔吐，背痛，體重減輕。
# 食慾喪失，舌炎，胃酸缺乏，對稱性皮膚炎，神經症狀。
# 癩皮病末期時出現嚴重之皮膚炎，腹瀉，癡呆，甚而死亡。
[[File:Pellagra_NIH.jpg|none]],飲食長期缺乏維生素B<SUB>3</SUB>]]

== 攝食過量之副作用 ==
# 皮膚潮紅及癢。
# 血清尿酸值升高。
# 葡萄糖耐受性減損。
# 肝臟方面的損害。

== 一般需求量 ==
成人每天攝取16毫克，女子為14毫克，發育中的青年為20毫克，懷孕期及哺乳期為18毫克。每天最高摄取量35毫克.
