'''烟花易冷'''是2010年由方文山作词，[[周杰伦|周杰伦作曲]]演唱的[[中国风|中国风]]歌曲，收录于专辑《[[跨时代|跨时代]]》<ref>{{Cite web|url = http://www.chinanews.com/yl/news/2010/05-06/2266289.shtml|title = 周杰伦新专辑歌曲泄露？ 网友最爱“烟花易冷”|accessdate = |author = |date = 2010年5月6日|publisher = 中国新闻网}}</ref>，由于歌曲吐字清晰，歌曲曾被怀疑不是周杰伦唱的，周杰伦坦言新歌“满矫情”<ref name=":0">{{Cite web|url = http://www.chinanews.com/tw/tw-ylzy/news/2010/06-02/2318749.shtml|title = 周杰伦新歌《烟花易冷》罕见字正腔圆 被疑山寨版|accessdate = |author = |date = 2010年6月2日|publisher = 中国新闻网}}</ref>。网上曾流传这首歌的真名是《伽蓝雨》<ref name=":0">{{Cite web|url = http://www.chinanews.com/tw/tw-ylzy/news/2010/06-02/2318749.shtml|title = 周杰伦新歌《烟花易冷》罕见字正腔圆 被疑山寨版|accessdate = |author = |date = 2010年6月2日|publisher = 中国新闻网}}</ref><ref>{{Cite web|url = http://ent.qq.com/a/20100702/000045.htm|title = 视频：周杰伦做客腾讯娱乐《名人访》聊新专辑|accessdate = |author = |date = 2010-07-02|publisher = 腾讯}}</ref>。《烟花易冷》跟《[[洛阳伽蓝记|洛阳伽蓝记]]》没有关系，烟花易冷描述了一千五百多年前盛极繁华后倾塌颓圮的[[洛阳|洛阳]]中，一名皇家将领与其所倾慕之女子间的爱情故事<ref>{{Cite web|url = http://www.ntjoy.com/wltx/2013/02/2013-02-25197460_3.html|title = 林志炫《烟花易冷》掳获人心 知道这首歌的背后故事吗？|accessdate = |author = |date = 2013-02-25|publisher = 江海明珠网}}</ref>。
