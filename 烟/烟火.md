'''烟火'''，又称'''烟花'''、'''花火'''、'''焰火'''，是一種以[[火藥|火藥]]和[[金屬|金屬]]的粉末製成的物體，以火點燃後會燃燒、爆炸並綻放出聲音和五顏六色的光線，通常在戶外使用。

== 歷史 ==
煙火於10世紀已有記載在[[中国|中国]]发明，主要用于盛大的典礼或表演中，《后武林旧事》记有[[宋孝宗|宋孝宗]]观海潮放烟火的情景说：“淳熙十年（公元1183年）八月十八日，上诣德寿宫，共请两殿往浙江观潮……管军命于江面分布五阵，乘骑弄旗，标枪舞刀，如履平地。点放五色烟炮满江、及烟收、炮息，则诸船尽藏，不见一只。”[[宋理宗|宋理宗]]（1225年至1264年）时，[[周密|周密]]在《[[齐东野语|齐东野语]]》〈御宴煙火〉中也记载了当时皇宫观看煙火的故事：「穆陵初年，嘗於上元日清燕殿排當，恭請恭聖太后。既而燒煙火於庭」。

到[[元|元]]、[[明|明]]年间，许多诗人、文学家都有有关[[鞭爆|鞭爆]]烟火这方面的记述。如明[[張岱_(明朝)|張岱]]《[[陶庵夢憶|陶庵夢憶]]》〈魯藩煙火〉：「魯藩煙火妙天下……及放煙火，燈中景物又收為煙火中景物。天下之看燈者，看燈燈外；看煙火者，看煙火煙火外。未有身入燈中、光中、影中、煙中、火中，閃爍變幻，不知其為王宮內之煙火，亦不知其為煙火內之王宮也。」

== 原理 ==
煙火的[[化學原理|化學原理]]和[[爆竹|爆竹]]大同小異，其結構都包含[[黑火藥|黑火藥]]和[[藥引|藥引]]。燃放煙花後，產生化學反應引發爆炸，而爆炸過程中所釋放出來的能量，絕大部分轉化成光能呈現在我們眼中。[[烟火制造术|製作煙火]]的過程中加入一些[[發光劑|發光劑]]和[[發色劑|發色劑]]能夠使煙火放出五彩繽紛的顏色。

發光劑原理為金屬[[鎂|鎂]]或金屬[[鋁|鋁]]的粉末[[氧化|氧化]]。當這些金屬燃燒時，會發出的強光及熱能。發色劑是一些金屬[[化合物|化合物]]。金屬化合物含有金屬離子。當這些金屬離子被燃燒時，會發放出獨特的火焰顏色。不同種類的金屬化合物在燃燒時，會發放出不同顏色的光芒。如：[[氯化鈉|氯化鈉]]和[[硫酸鈉|硫酸鈉]]都屬於[[鈉|鈉]]的化合物，他們在燃燒時便會發出金黃色火焰。同理，[[硝酸鈣|硝酸鈣]]和[[碳酸鈣|碳酸鈣]]在燃燒時會發出[[磚紅色|磚紅色]][[火焰|火焰]]。煙火便是利用金屬的這種特性製成的。製作煙火的人經過巧妙的排列，決定燃燒的先後次序。這樣，煙火引燃後，便能在漆黑的天空中綻放出鮮豔奪目、五彩繽紛的圖案。

== 世界各地煙花匯演 ==
[[File:Liuyang-Firework_production-China_-_panoramio.jpg|缩略图]]煙火工業區]]
一般而言，煙花匯演與煙火匯演是沒有任何分別（如：中國大陸說法為煙花、台灣說法為煙火）。但是煙花匯演與煙火匯演在[[香港|香港]]是有不同意思。煙火（Pyrotechnics）威力遠低於煙花，一般會在燃放後於空中消散，部分甚至可安全在戶內環境，如演唱會發放，受《娛樂特別效果條例》規管。煙花（Fireworks）則火力較大，射到高空後再爆炸，藉此拼湊出各式各樣圖案，受《危險品（一般）規例》規管。煙花匯演泛指大型匯演，如[http://www.info.gov.hk/gia/general/200609/28/P200609280177.htm 維多利亞港煙花匯演]，相反煙火匯演是指小型匯演，如[http://www.info.gov.hk/gia/general/201109/15/P201109150140.htm 荃灣煙火匯演]。

== 危險物品 ==
{{main|新興堂金紙鋪爆炸事件}}
煙火成份是[[火藥|火藥]]，是[[易燃物品|易燃物品]]，時有玩樂者出[[意外|意外]]，樂極生悲新聞不少，例如[[深圳舞王俱樂部火災|深圳舞王俱樂部火災]]、[[2009年曼谷夜總會大火|2009年曼谷夜總會大火]]、[[2009年中國中央電視台新大樓大火|2009年中國中央電視台新大樓大火]]。

近年來台灣婚喪喜慶及廟會，因為大量暴增的煙火需求，常有店家將炮竹、煙火囤積轉售給廟宇及一般民眾的情形，但煙火囤積容易引起爆炸，2011年4月的新興堂金紙鋪爆炸事件即是以上情形引起的案例。此外，逢節慶、廟會、選舉時，更有在都會區街道上施放煙火的情況，煙火施放產生的噪音、交通阻塞、安全等問題，也引起相當程度的民怨，2011年9月4日下午於嘉義「建國百年全國城隍大會師」活動中，還發生煙火爆炸事件，造成12人輕重傷，縱使台灣有煙火相關法令規範，然而在法律對宗教信仰的保護之下(或是政府行政對地方勢力仍有所忌憚)，法律仍然形同虛設，煙火引發的相關問題在台灣層出不窮。
