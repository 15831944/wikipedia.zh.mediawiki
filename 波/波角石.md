'''波角石'''（[[學名|學名]]：{{lang|la|'''''Cymatoceras'''''}}）是一[[屬|屬]]已[[滅絕|滅絕]]的[[鸚鵡螺亞綱|鸚鵡螺類]]。牠們可說是最常見的鸚鵡螺類化石之一，尤其以產於[[馬達加斯加|馬達加斯加]]的波角石最為著名。<ref name="Cymatoceras">{{Cite web |url = http://liusir.pixnet.net/blog/post/248636783-cymatoceras-%E6%B3%A2%E8%A7%92%E7%9F%B3-%28%E9%A6%AC%E9%81%94%E5%8A%A0%E6%96%AF%E5%8A%A0%29 |title = Cymatoceras 波角石 (馬達加斯加) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-05-14 }}</ref>

== 特徵 ==
波角石的外殼上的肋紋和波狀的縫合線和[[現生種|現生種]]的[[鸚鵡螺|鸚鵡螺]]非常相似，但波角石的殼口形狀較為渾圓，整體外觀也更為圓滾一些。<ref name="Cymatoceras2">{{Cite web |url = http://liusir.pixnet.net/blog/post/248637221-%E9%A6%AC%E9%81%94%E5%8A%A0%E6%96%AF%E5%8A%A0-%E6%B3%A2%E8%A7%92%E7%9F%B3-%28%E6%8B%8B%E5%85%89%E4%BA%8C%29-cymatoceras |title = 馬達加斯加 波角石 (拋光二) Cymatoceras @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-05-14 }}</ref>
