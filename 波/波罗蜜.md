'''波罗蜜'''（[[学名|学名]]：''{{lang|la|Artocarpus heterophyllus}}''），亦称'''菠萝蜜'''，在[[廣州|廣州]]、[[香港|香港]]、[[澳門|澳門]]及其他[[粵語|粵語]]地區，波罗蜜一般被稱為'''大樹菠蘿'''，在广西等地称为'''木波罗'''，[[云南省|滇西地区]]称'''牛肚子果'''，是[[桑科|桑科]][[波羅蜜屬|波羅蜜屬]][[常绿|常绿]][[乔木|乔木]]。

==产地==
原产於[[印度|印度]]、[[东南亚|东南亚]]，[[隋|隋]][[唐|唐]]时传入中国，《隋书·真腊传》称为“频那挲”（[[梵文|梵文]] Panasa 对音），[[宋代|宋代]]改称'''波罗蜜'''，沿用至今。<ref>{{cite web | url=http://kmweb.coa.gov.tw/ct.asp?xItem=1034859&ctNode=1581&mp=1&kpi=0&rowId=1 | title=波羅蜜屬新興果樹|website=農業知識入口網}}</ref>现在波罗蜜盛产于中国[[岭南|岭南]]、[[福建|福建]]、[[云南|云南]]和[[印度|印度]]、[[中南半岛|中南半岛]]、[[南洋群岛|南洋群岛]]、[[孟加拉国|孟加拉国]]和[[巴西|巴西]]等地。

== 外觀 ==
波羅蜜樹高可達20公尺以上，體液乳白色。葉互生，長橢圓形或倒卵形，革質，全緣，幼株葉片偶有淺裂。頭狀花序，幹出或腋出。聚合果卵狀橢圓形或長圓筒形，常著生於樹幹，未熟果呈綠色，成熟呈黃或黃褐色，果面有小六角形龜甲狀突起。假種皮(果肉)金黃色，柔軟甜蜜，散發香蕉和鳳梨融合的特殊香氣，可食用。種子白色，種仁肉質，煮時似菱角，烤時味如栗。

==用途==
[[明代|明代]][[鄭和下西洋|鄭和下西洋]]，隨鄭同行的[[馬歡|馬歡]]所著的《[[瀛涯勝覽|瀛涯勝覽]]》提及此物：「其波羅蜜如冬瓜之様，外皮似川荔枝，皮內有雞子大塊黃肉，味如蜜甜。中有子如腰子樣，炒吃味如栗子。」新鲜波罗蜜果肉是水果，气味香甜，常用作烹调材料，或制成波罗蜜果脯和波罗蜜罐头；种子也可食用，炒食或煮食均可口，但生食或许会引起中毒。波罗蜜木材常用来制造南洋各地的[[乐器|乐器]]。

== 种子 ==
波罗蜜的种子状似鹅卵石，光洁而有石头样花纹，质硬如木，但尚可嚼动。生食或有微毒，可引起些许头痛、恶心和呕吐等症状。不推荐生食。熟食味如花生或板栗。
