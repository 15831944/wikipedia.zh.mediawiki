'''博茨瓦纳共和国'''（{{lang-en|Republic of Botswana}}；{{lang-tn|Lefatshe la Botswana}}），通稱'''博茨瓦纳'''，是一個位於[[非洲|非洲]]南部的[[內陸國|內陸國]]，其以擁有強大的個人自由而著稱，在[[新聞自由|新聞自由]]和個人[[財產權|財產權]]都有顯著優勢<ref>{{Cite web|url=http://www.bbc.com/travel/story/20180107-what-its-like-to-live-in-a-well-governed-country|title=What it's like to live in a well governed country|accessdate=|author=|date=2018-01-07|publisher=BBC}}</ref>。全國國境皆為乾燥的[[台地|台地]]地形，南鄰[[南非|南非]]，西邊為[[納米比亞|納米比亞]]，東北與[[辛巴威|津巴布韦]]接壤。其國土北端只有在[[維多利亞瀑布|維多利亞瀑布]]附近與[[赞比亚|赞比亚]]接觸。該國獨立後沒有太多動亂變動，保留了西方殖民統治的政治、金融體系，特別是鑽石開採業，是投資環境較佳，在非洲發展水準可比上南非的熱點地區。

== 历史 ==
{{main|博茨瓦纳历史}}
13－14世纪，[[茨瓦纳人|茨瓦纳人]]由北方迁居于此。

在1885年時，為了避免被南邊鄰國南非境內[[荷蘭|荷蘭]]裔的[[波耳人|波耳人]]併吞，英國應該地區的民眾要求建立了“[[貝專納|貝專納]][[保護地|保護地]]”（Bechuanaland Protectorate），而成為[[聯合王國|聯合王國]]的一部份。

1966年9月30日脫離英國宣告独立，改名为博茨瓦纳共和国，實施民主化政治，積極的社會政策與豐富的礦藏（尤其是[[鑽石|鑽石]]礦，佔該國輸出利潤的八成）帶來的財富讓波札那擁有[[非洲|非洲]]大部分地區難得一見的開發水準。除此之外，由於該國積極地與國際社會合作，在國內建立為了保護野生動物而設的大面積國家公園，而使得[[觀光業|觀光業]]有機會逐漸在該國成長興盛。

波札那的首都為[[嘉柏隆里|哈博罗内]]，人口约23万（2011年），是一位於南邊國界上的城市，19世紀末時因為附近的礦業開採而興起。

前任总统是[[伊恩·卡马|伊恩·卡马]]，2008年4月1日就任，2014年10月28日连任。

现任总统是[[莫克维齐·马西西|莫克维齐·马西西]]，2015年7月当选执政党博茨瓦纳民主党全国主席，2017年7月连任。2018年4月接任总统、民主党总裁。

== 地理 ==
[[File:Botswana_map.png|thumb]]

博茨瓦纳面積600,370平方公里，是世界上第45大國家，次于[[乌克兰|乌克兰]]。其面積與[[马达加斯加|马达加斯加]]相當，比美國[[德克萨斯州|德克萨斯州]]略小。

博茨瓦纳地勢大致平坦，擁有起伏高原，一般海拔800～1100米。国土中部和西南部属[[卡拉哈里沙漠|卡拉哈里沙漠]]，覆蓋了博茨瓦納接近七成土地。位於西北方的[[奧卡万戈沼泽|奧卡万戈沼泽]]是世上最大的內陸[[三角洲|三角洲]]。多草原和盐沼，[[馬卡迪卡迪鹽沼|馬卡迪卡迪鹽沼]]（Makgadikgadi Pan）面積廣大，盤據博茨瓦纳北方。东南部为丘陵。

===气候===
大部分地区属于热带干旱草原气候，西部为沙漠、半沙漠气候。年降水量从东北部600毫米向西南递减至200毫米以下。

===动植物===
博茨瓦納擁有多樣的野生动植物棲息地，包括奧卡萬戈三角洲、喀拉哈里沙漠、草原和[[熱帶乾濕季氣候|熱帶草原]]。熱帶草原上，有[[黑尾牛羚|黑尾牛羚]]、[[羚羊亚科|亞科羚羊]]、其他哺乳類和雀鳥等野生動物。
===资源===
主要矿藏为钻石，其次为铜镍、煤、苏打灰、铂、金、锰等。已探明的铜镍矿蕴藏量为4600万吨，煤蕴藏量170亿吨。
== 经济 ==
[[File:Tree_map_export_2009_Botswana.jpeg|thumb]]
{{主条目|博茨瓦纳经济}}
波札那的经济支柱是钻石业，产值约占国内生产总值的三分之一；其次是旅游业。主要出口钻石、铜镍矿产品、苏打灰、牛肉产品、纺织品和汽车配件等；进口汽车及交通设备、机械及电子产品、食品 及饮料、金属、化工和橡胶制品、燃料和烟草等。<ref name=fmprc/>

相對於其他非洲國家，波札那擁有良好的金融管理紀錄，並在2013年被[[國際透明組織|國際透明組織]]列為全非洲最不腐敗的國家，全世界排名第30，甚至高過許多歐洲及亞洲國家<ref>{{cite web|url=http://cpi.transparency.org/cpi2013/results/ |title=Corruption Perceptions Index 2013 |publisher=[[國際透明組織|國際透明組織]]}}</ref>。[[世界經濟論壇|世界經濟論壇]]評估波札那為非洲兩個最有競爭力的國家之一。2004年博茨瓦納再一次被[[穆迪|穆迪]]投资服务公司（Moody's Investors Service）與[[史丹普公司|史丹普公司]]評估信用為「A」級，而讓波札那成為非洲[[信用風險|信用風險]]最低的國家，與[[中歐|中歐]]、[[東亞|東亞]]及[[拉丁美洲|拉丁美洲]]的國家旗鼓相當。

== 医疗卫生 ==
全国约36%（2008年10月）的成年人感染[[HIV|HIV]]，是全世界感染艾滋病病毒第二高的国家<ref>[http://www.berlinonline.de/berliner-zeitung/archiv/.bin/dump.fcgi/2008/1023/politik/0020/index.html Berlin Online]{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，平均每3个小时就有一人死于[[艾滋病|艾滋病]]。2013年艾滋病平均感染率为18.5%。目前疾病严重威胁着国家的社会经济安全。
