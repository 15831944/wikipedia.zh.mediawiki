'''波仔'''（'''Eateast'''），註冊商標為一個[[紅|紅]]底[[白|白]]字的「波仔」標誌。波仔飯，是[[香港|香港]]一個[[微波爐便當|微波爐便當]]品牌，於2001年由[[香港|香港]]飯盒供應商[[陽光一代|陽光一代]]（Sun Generation）老闆[[羅開彌|羅開彌]]發明<ref>[陽光飯盒出事有前科]，《明報》，2005年10月1日</ref>，現有40多間直營店，日銷量6萬多飯盒，同時也在惠康、大昌食品、Vango便利店、7-eleven發售 <ref>{{Cite web |url=http://www.eateast.cn/Cooperation.asp |title=招商合作-波仔 |access-date=2014-12-29 |archive-url=https://web.archive.org/web/20150208202639/http://eateast.cn/Cooperation.asp |archive-date=2015-02-08 |dead-url=yes }}</ref>。在2014年，波仔轉手至蔬果供應商創新蔬果食品董事長[[董品春|董品春]]。<ref>[https://www.mpfinance.com/fin/daily2.php?node=1491159664511&issue=20170403 售價要考慮負擔能力 去年收入按年升三成 轉標誌研素食菜單]</ref>

== 售賣地點 ==
波仔飯在[[香港|香港]][[便利店|便利店]]、大型連鎖[[超級市場|超級市場]]及[[澳門|澳門]]超級市場等均有發售，一般有三十款以上口味可供選擇。
高峰時期，曾經開設了約90間專門店。但由於租金持續大幅增加，2016年餘下約44間專門店。<ref>{{cite web|url= |title=《唔撈學生飯盒 波仔攻陷無飯夫婦》|publisher=商場脈博，壹週刊|issue=第1008期|date=2009年2月7日}}</ref> <ref>[http://orientaldaily.on.cc/cnt/news/20090831/00176_005.html 學生餐「無肉食」飯盒大王撤退]</ref>，專門店內有手推車，售賣旗下品牌各種口味的飯盒。

== 產品類別 ==
* 波仔飯，分為速涼和真空急凍。速涼以[[攝氏|攝氏]]四度冷藏，份量較多，可保存三十日。真空急凍的份量較少，並要存放於[[冰箱|冰箱]]，可保存一年。
* 叮叮菜
* 叮叮糖水/ 湯品$9~$15（2017-3-9） 
* 24小時叮早餐（2011售出約1年半） 
* 叮叮焗飯焗意粉$21~$28（2012-11起）
* 叮叮漢堡$15（2013-2）

== 軼事 ==
波仔飯專門店設有[[微波炉|微波炉]]，以供客人將飯盒加熱再攜走或留在波仔飯專門店裏堂食，服務員可代客加熱飯盒，但會要求顧客做-{zh-hans:摁钮; zh-hant:撳鈕; zh-hk:按掣;}-這個動作，據店員透露是因為店子沒有熟食[[牌照|牌照]]，若由服務員按動微波爐則會構成「煮食」的行為，為免觸犯法例，因此該店衍生出這個「顧客動手按」的服務安排。

波仔飯曾邀請香港著名已故球評家[[林尚義|林尚義]]拍攝電視廣告，使波仔飯的品牌一時廣為人知。<ref>[https://www.youtube.com/watch?v=8uz7TZQjAQY Microbo 波仔飯 TVC ]，[[YouTube|YouTube]]</ref>。波仔飯還有一首電視廣告歌，舊曲新詞，內容通俗。
