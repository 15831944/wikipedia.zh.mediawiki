'''波城'''（[[法语|法语]]：'''{{lang|fr|Pau}}'''；{{IPA-fr|po}}；{{Lang-eu|Paue}}）位于[[法国|法国]]西南部[[阿基坦大区|阿基坦大区]]，是[[大西洋岸比利牛斯省|大西洋岸比利牛斯省]]的首府，人口81,166（2010年）。 

==地理==
距离大西洋沿岸城市[[巴约讷_(比利牛斯-大西洋省)|巴约讷]]直线距离110公里。<ref>{{Cite web |url=http://www.geoportail.fr |title=GeoPortail |accessdate=2018-08-31|publisher=[[GeoPortail|GeoPortail]]}}</ref>
根据法国[[国家地理林业信息研究所|国家地理林业信息研究所]]的数据，该市的占地面积为31.51平方公里。<ref>{{Cite web |url=http://professionnels.ign.fr/rgc#tab-3 |title=地理信息 |accessdate=2018-08-31 |author=法国[[国家地理林业信息研究所|国家地理林业信息研究所]]}}</ref>而市政厅显示的数据为31.45平方公里，海拔高度在165米到245米之间。<ref name="pfr">{{Cite web |url=http://www.pau.fr/303-pau-en-un-clic.htm |title=Pau Information |accessdate=2018-08-31 |author=pau.fr}}</ref>
法国诗人[[阿尔方斯·德·拉马丁|拉马丁]]曾发出“看山至波城，看海至[[那不勒斯|那不勒斯]]”的感叹。<ref>{{fr}}{{Cite web|url=http://www.agglo-pau.fr/pau.html|title= « Présentation de la ville de Pau »|date=2018-08-31|quote=Pau est la plus belle vue de terre comme Naples est la plus belle vue de mer.}}</ref>
