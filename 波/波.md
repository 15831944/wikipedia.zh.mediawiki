'''波'''或'''波动'''是扰动或物理信息在空间上传播的一种[[物理|物理]]現象，扰动的形式任意，傳遞路徑上的其他介質也作同一形式振動。波的传播[[速度|速度]]总是有限的。除了[[电磁波|电磁波]]、[[引力波|引力波]]（又稱「[[重力波|重力波]]」）能够在[[真空|真空]]中传播外，大部分波如[[机械波|机械波]]只能在[[介质|介质]]中传播。波速與介質的彈性與慣性有關，但與波源的性質無關。

==数学描述==
在数学上，任何一个沿某一方向运动的函数形状都可以认为是一个波。考虑一种最简单的情况：二维平面波，波的形状可以用<math>xy</math>平面上的曲线<math>y = f(x)</math>描述。

如果这个曲线沿着<math>x</math>轴以<math>\omega</math>的速度向右运动，不难看出，这样的函数应该满足如下方程：<math>y = f(x-\omega t)</math>

如果沿x轴以ω的速度向左运动，则为：<math>y = f(x+\omega t)</math>

以上两个方程都满足如下形式的[[微分方程|微分方程]]：
:<math>\frac{\partial ^2 f}{\partial t^2} = c ^2\frac{\partial ^2 f}{\partial x^2}</math>
这里'''c'''通常是一个固定[[常数|常数]]，代表波的传播速率。这个方程称为'''一维波动方程'''。

它的通解可以表示为：
:<math>y(x,t)=f(x+\omega t) + g(x-\omega t)</math>

它表示一个向左传播的波和一个向右传播的波的叠加。

===行進波===
'''行進波'''，又稱為'''前進波'''，是一種在空間與時間裏的擾動，可以表達為

:<math>y(z,t) = A(z,\ t)\sin(kz - \omega t + \phi)\,\!</math>；

其中，<math>A(z,\ t)\,\!</math>是波的[[振幅|振幅]]，<math>z\,\!</math>是位置，<math>t\,\!</math>是[[時間|時間]]，<math>k\,\!</math>是波數，<math>\phi\,\!</math>是相數。

波的相速度<math>v_p\,\!</math>可以表達為

:<math>v_p = \frac{\omega}{k}= \lambda f\,\!</math>；

其中，<math>\lambda\,\!</math>是[[波長|波長]]。

=== 一维简谐波 ===
[[File:Simple_harmonic_motion_animation.gif|thumb]]。]]
一种最基本、最常见的波是[[簡諧運動|简谐波]]。它可以表示为：
:<math>f=Ae^{i(kx-\omega t)} \,</math>
其中<math>k</math>是[[波数|波数]]，<math>\omega</math>是[[角频率|角频率]]，<math>A</math>是[[振幅|振幅]]。

波数倚赖于[[波长|波长]]<math>\lambda</math>，<math>k=\frac{2\pi}{\lambda}</math>。角频率倚赖于[[周期|周期]]<math>T</math>，<math>\omega=\frac{2\pi}{T}</math>。

波速<math>v=\frac{\omega}{k}</math>。

== 影響波速的因素 ==
'''1.傳播的介質種類'''

在固體中的波速最高，液體次之，氣體最小（例如聲音）。溫度越高，空氣分子運動的速率越快，傳遞波的速度亦愈快。在同一介質中，波的速率與頻率無關。

'''2.溫度的高低'''

溫度越高，空氣分子運動的速率越快，所以傳遞波的速度亦越快。在一大氣壓下，0℃時空氣中的聲速為331公尺/秒，溫度每升降1℃，聲速約增减0.6公尺/秒。

==特征参量==
任何一种波都可以用如下的参量进行描述：
* 色散关系，即波的频率ω与波矢量'''''k'''''之间的关系：<math>\omega=\omega(\boldsymbol{k})</math>。其中，波矢量的方向是垂直于波阵面的，其数值等于[[波数|波数]]，即k=2π/λ。
* 波的[[相速度|相速度]]<math>v_{p}=\omega/k</math>与[[群速度|群速度]]<math>\boldsymbol{v}_g=\mathrm{d}\omega/\mathrm{d}\boldsymbol{k}</math>。相速度的方向与波矢量'''''k'''''的方向平行，而群速度表示波内能量转移的大小和方向。
* 波的衰减率γ
* 波的偏振。可以是无偏振、线偏振、椭圆偏振或者是圆偏振。

==类型==
波根據振動源的次數可以分為
*[[脈波|脈波]]：脈波的[[波源|波源]]只對介質作一短暫的擾動。波通過介質時，介質中的質點在短暫振動後，隨即靜止於原位置。
*[[週期波|週期波]]：週期波的波源對介質作連續有規律的振動。

波在均勻、無向性的介質中傳遞時，依介質的振動方向分可以分為
*[[縱波|縱波]]：縱波的特點是介質的振動方向與傳播方向相同，比如空氣中的[[聲波|聲波]]、[[地震波|地震波]]中的[[P波|P波]]。
*[[橫波|橫波]]：橫波的特點是介質的振動方向與傳播方向垂直。如：[[電磁波|電磁波]]、[[地震波|地震波]]中的[[S波|S波]]。

如果在非均質介質中傳遞時，介質振動的行為就不是只有橫向與縱向兩種，亦存在像[[表面波|表面波]]、海浪這種類型的振動。譬如：[[雷利波|雷利波]]其振動方式為橢圓形。

依波動傳遞需要介質來劃分，波可以分為
*[[力學波|力學波]]（機械波）：如[[聲波|聲波]]、[[水波|水波]]等
*[[非力學波|非力學波]]（非機械波）：如[[電磁波|電磁波]]、[[重力波|重力波]]等
*[[物質波|物質波]]則是在近代物理中敘述物質具有粒子與波動的二元性（[[波粒二象性|波粒二象性]]），近一步的探討則認為物質波是物質在空間中分布的機率，如電子的軌域。

== 传播 ==
有些波的传播需要[[介质|介质]]，比如[[声波|声波]]等[[机械波|机械波]]。有些则不需要介质，在[[真空|真空]]中也能传播。如[[电磁波|电磁波]]。

波在介質中傳播時，介質的質點並未隨波前進，而是在原處附近運動。

波的行進[[速度|速度]]v為其[[頻率|頻率]]f和[[波長|波長]]λ的[[乘積|乘積]]，即波長λ和[[週期|週期]]T的比值：<math>v = f \lambda = \frac{\lambda}{T}</math>

波在繩子上傳播時，波的行進速度v（SI單位[[米_(计量单位)|m]]/s）與繩子所受的[[張力|張力]]F（單位[[力|N]]）及繩子的線密度μ（單位kg/[[米_(计量单位)|m]]）有關：<math>v = \sqrt{F \over \mu}</math>
