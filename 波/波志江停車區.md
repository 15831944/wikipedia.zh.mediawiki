'''波志江停车区'''（平假名：はしえパーキングエリア）是位於[[群馬縣|群馬縣]][[伊勢崎市|伊勢崎市]]的[[北關東自動車道|北關東自動車道]]之[[休息站|停车区]]。此IC同時設置[[智能IC|智能IC]]。

== 連接道路 ==
* [[北關東自動車道|北關東自動車道]]

== 历史 ==
* 2008年3月1日 - 此停车区啟用。
* 2008年3月29日 - 開始試用[[智能IC|智能IC]]。
* 2009年4月1日 - 啟用[[智能IC|智能IC]]。
* 2011年4月29日 - 設置伊勢崎、波志江高速巴士站。
