'''妙蛙花'''（日文：'''{{lang|ja|フシギバナ}}'''，英文：'''{{lang|en|Venusaur}}'''，香港舊譯'''奇異花'''）是[[精靈寶可夢|精靈寶可夢]]系列登場的{{NUMBEROFPOKEMON}}種虛構[[角色|角色]]（[[怪物|怪獸]]）中的一種。

== 特徵 ==
妙蛙花為[[妙蛙草|妙蛙草]]的進化型。由淡藍色的身軀、翠綠的芭蕉葉、深咖啡色的樹幹及紅色的花瓣組成。據說只要有充足的養分及日光，背上的花就會更加鮮豔。而花散發的味道可以撫平人的心情。是草屬性及毒屬性的種子[[精靈寶可夢|寶可夢]]。

== [[神奇寶貝遊戲列表|遊戲]]中的妙蛙花 ==
初次登場於精靈寶可夢紅／綠版，妙蛙花於遊戲中由[[妙蛙草|妙蛙草]]在等級32級時進化，是[[妙蛙種子|妙蛙種子]]系列寶可夢的終極形態，無法再次進化。日文名稱原義為「不思議之花」。

妙蛙花[[神奇宝贝列表|全國圖鑑編號]]為3，城都聯盟編號為205，從[[妙蛙草|妙蛙草]]進化而成，代表技能是日光束及瘋狂植物，特性為茂盛（しんりょく）。

此外，妙蛙花也是精靈寶可夢綠版的封面獸。

== 動畫中的妙蛙花 ==
妙蛙花在動畫中比較地罕見，而在超世代的第168集中戰鬥皇宮的首領宇康有使用妙蛙花迎戰[[小智|小智]]的[[赫拉克羅斯|赫拉克羅斯]]和[[大王燕|大王燕]]。

在《[[神奇寶貝鑽石&珍珠|神奇寶貝鑽石&珍珠]]》中，小遙的妙蛙種子進化成妙蛙花。
