'''妙蛙種子'''（{{lang-ja|フシギダネ}}，{{lang-en|Bulbasaur}}，香港舊譯'''奇異種子'''）是[[精靈寶可夢|精靈寶可夢]]系列登場的{{NUMBEROFPOKEMON}}種虛構[[角色|角色]]（[[精靈寶可夢|精靈寶可夢]]）中的一種。

== 特徵 ==
妙蛙種子為草屬性及毒屬性的種子[[精靈寶可夢系列|寶可夢]]，有如[[青蛙|青蛙]]般的身體上背著大顆充滿營養的[[種子|種子]]，與種子是共生關係。這顆種子從出生時開始種植，出生後暫時由種子中獲取養分成長。成為[[妙蛙草|妙蛙草]]後由中生長出草，成為[[妙蛙花|妙蛙花]]後開花。<ref name="妙蛙种子1">'''图鉴:''' 它出生的时候背上就有一个奇怪的种子。这种子跟随它发芽成长。{{cite video game|title=口袋妖怪 红·绿|developer=Game Freak |publisher=任天堂 |date=1998-09-30 |platform=Game Boy }}</ref><ref name="妙蛙种子2">'''图鉴:''' {{cite video game|title=口袋妖怪 钻石·珍珠|developer=Game Freak |publisher=任天堂 |date=2006-09-28 |platform=Nintendo DS}}</ref>

身體顏色主要由青色及綠色組成。

== [[神奇寶貝遊戲列表|遊戲]]中的妙蛙種子 ==

初次登場於精靈寶可夢紅／綠版，為『精靈寶可夢 紅・綠・藍版』和『精靈寶可夢 火紅・葉綠版』中最先可挑選的[[精靈寶可夢|寶可夢]]（妙蛙種子、[[小火龍|小火龍]]、[[傑尼龜|傑尼龜]]）。等級16級進化成[[妙蛙草|妙蛙草]]。全國圖鑑及關都圖鑑中編號為001，是位於最前面的寶可夢。

寶石版編號為203，代表技能為寄生種子（{{lang|ja|やどりぎのタネ}}）及藤鞭（{{lang|ja|つるのムチ}}），特性為茂盛（{{lang|ja|しんりょく}}）。

『精靈寶可夢 黃版』中，[[皮卡丘|皮卡丘]]在親密狀態時與華藍市（{{lang|ja|ハナダシティ}}）民家裡的女性交談可得到。

等級16級時進化為[[妙蛙草|妙蛙草]]。

== 動畫中的妙蛙種子 ==
妙蛙種子為關都地區（{{lang|ja|カントー}}）中，給新人訓練家的三隻神奇寶貝之一。[[動畫|動畫]]版中，主角[[小智|小智]]也有一隻，是在十集中從一個民間的護士取得，個性負責、擅於領導、頑固，目前於[[大木博士|大木博士]]研究所中維持寶可夢間的和平。在第51集中，小智的妙蛙種子已經有進化的條件，但牠與[[皮卡丘|皮卡丘]]一樣不願意進化，但這亦讓牠早一點學會日光束（舊稱陽光烈焰）。
而豐緣聯盟篇的女主角小遙亦有一隻，牠額頭上有個心形斑紋，目前也在大木博士研究所中，後來進化成妙蛙花。

== 漫画中的妙蛙種子 ==
在一个基于游戏[[口袋妖怪_红·绿|红·绿]]改编的漫画[[神奇寶貝特別篇|神奇寶貝特別篇]]中，主角小智从大木博士那里拿到一只妙蛙种子，并起昵称叫“妙蛙”。在第15章“VS卡咪龟”中，它在对战猴怪后进化成[[妙蛙草|妙蛙草]]。
