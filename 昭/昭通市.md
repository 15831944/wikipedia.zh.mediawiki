'''昭通市'''是[[中华人民共和国|中华人民共和国]][[云南省|云南省]]下辖的[[地级市|地级市]]，位于云南省东北部。市境南界[[曲靖市|曲靖市]]和[[贵州省|贵州省]][[毕节市|毕节市]]，东接[[四川省|四川省]][[泸州市|泸州市]]，北邻四川省[[宜宾市|宜宾市]]，西临四川省[[凉山州|凉山州]]。地处滇、川、黔三省结合部，[[云贵高原|云贵高原]]与[[四川盆地|四川盆地]]交界，[[乌蒙山|乌蒙山]]西段。[[金沙江|金沙江]]为西、北两面与四川省的界河，主要河流还有[[牛栏江|牛栏江]]、[[横江|横江]]、[[洛泽河|洛泽河]]、[[白水江|白水江]]等，坡度落差大，水力资源丰富。总面积2.24万平方公里，人口543万。昭通是全省矿产资源重要产地，其中以[[煤|煤]]和[[硫黄|硫黄]]的储量最丰。

==历史==
[[秦昭王|秦昭王]]在位时，命[[李冰|李冰]]出任[[蜀郡|蜀郡]]太守，期间修筑了一条连接[[中原|中原]]、[[四川|四川]]与[[云南|云南]]的通道，因路宽五尺，又称为“[[五尺道|五尺道]]”。该通道经过今昭通市境内，成为中原与云南一带的交通要塞。[[西汉|西汉]][[建元|建元]]六年（前135年）于今昭通市区置'''[[朱提县|朱提县]]'''，属[[犍为郡|犍为郡]]。[[东汉|东汉]][[永初|永初]]元年（107年）析置[[犍为属国|犍为属国]]。[[建安_(东汉)|建安]]十九年（214年）改置[[朱提郡|朱提郡]]，治朱提县。[[蜀汉|蜀汉]]至[[西晋|西晋]]初朱提郡属[[益州|益州]]。[[太安_(西晋)|太安]]二年（303年）朱提郡改属[[宁州|宁州]]。[[永嘉|永嘉]]五年（311年）分朱提郡置[[南广郡|南广郡]]，治[[南广县|南广县]]（今[[盐津县|盐津县]]）。[[南朝齐|南朝齐]]改朱提郡为[[南朱提郡|南朱提郡]]，后废。[[南朝梁|南朝梁]]废[[朱提县|朱提县]]。

[[隋朝|隋]][[开皇|开皇]]四年（584年）置[[恭州|恭州]]，治今昭通市区；改南广郡为[[协州|协州]]，并置[[东安县|东安县]]为治所（今[[彝良县|彝良县]]）。[[大业|大业]]初省恭州、协州。[[唐朝|唐]][[武德|武德]]元年（618年）复置恭州、协州，并于故朱提县置[[安上县|安上县]]。武德七年（624年）又改为朱提县；次年改恭州为[[曲州|曲州]]。天宝年间，今市境地入[[南诏|南诏]]。南诏初设[[拓东节度|拓东节度]]，驻拓东城（今[[昆明市|昆明市]]），今市区属拓东节度乌蒙部。南诏后期改拓东节度为[[善阐府|善阐府]]。唐代至[[宋代|宋代]]年间，今市境处于[[中原王朝|中原王朝]]与南诏、[[大理国|大理国]]之间，直至[[蒙古|蒙古]]灭大理国。

[[元朝|元]][[至元|至元]]十五年（1278年）于乌蒙部地置'''[[乌蒙路|乌蒙路]]'''，治今[[昭阳区|昭阳区]][[旧圃镇|旧圃镇]]，属[[云南等处行中书省|云南等处行中书省]][[乌撒乌蒙宣慰司|乌撒乌蒙宣慰司]]。至元二十二年（1285年）于南诏芒布部置[[芒部路|芒部路]]（治今[[镇雄县|镇雄县]][[芒部镇|芒部镇]]），属云南行省；又于易娘部置[[益良州|益良州]]（治今[[彝良县|彝良县]]），属芒部路。[[明朝|明]]初改芒部路为[[芒部府|芒部府]]。[[洪武|洪武]]十四年，[[朱元璋|朱元璋]]遣征南将军颖川侯[[傅友德|傅友德]]、左副将军永昌侯[[蓝玉|蓝玉]]、右副将军西平侯[[沐英|沐英]]等征讨乌蒙，平定後，在滇东北置乌蒙、芒部、东川诸卫指挥使司，以“约束其酋长”。洪武十五年（1382年）改乌蒙路为'''[[乌蒙府|乌蒙府]]'''，改隶[[四川承宣布政使司|四川承宣布政使司]]；次年，芒部府改隶四川布政司；十七年省益良州入芒部府。[[嘉靖|嘉靖]]二年（1523年）改芒部府为[[镇雄军民府|镇雄军民府]]。嘉靖五年（1526年）置[[归化长官司|归化长官司]]（治今彝良县[[奎阳镇|奎阳镇]]），属镇雄军民府。[[万历|万历]]三十七年（1609年）改镇雄军民府为[[镇雄府|镇雄府]]。

[[清朝|清]][[雍正|雍正]]年間，[[鄂尔泰|鄂尔泰]]先後在[[东川|东川]]，乌蒙進行[[改土歸流|改土歸流]]，“先怀以德，继畏以威，然后徐议改流”。乌蒙镇总兵[[刘起元|刘起元]]贪暴残忍，“于是远近夷民皆无生之气，有死之心。”禄万福发动大规模的土司叛乱，总兵刘起元、游击江仁、知县赛枝大被杀。鄂尔泰调集云南、贵州、四川三省兵力，“分兵穷搜，务获渠魁，尽屠丑类”。雍正五年（1727年）乌蒙府、镇雄府改属云南省；次年乌蒙府[[改土归流|改土归流]]，并置[[恩安县|恩安县]]（治今昭阳区）、[[永善县|永善县]]（治今永善县南），乌蒙府治永善县，改镇雄府为[[镇雄州|镇雄州]]，属乌蒙府，并改归化长官司置[[彝良州同|彝良州同]]，隶镇雄州；同年于故阿旁部地置[[大关厅|大关厅]]（治今[[大关县|大关县]]）；于米贴地方置[[永善县|永善县]]（治今永善县北），属乌蒙府。雍正八年（1730年）设盐井渡巡检司（治今[[盐津县|盐津县]]）。雍正九年（1731年）改乌蒙府为'''[[昭通府|昭通府]]'''，取“昭明宣通”之意，移治[[恩安县|恩安县]]；同年于故阿竿路部置[[鲁甸厅|鲁甸厅]]（治今[[鲁甸县|鲁甸县]]），属昭通府。[[嘉庆|嘉庆]]十六年（1811年）析[[会泽县|会泽县]]置[[巧家厅|巧家厅]]（治今[[巧家县|巧家县]]）。[[光绪|光绪]]三十四年（1908年）析永善县富官村置[[靖江县|靖江县]]（治今[[绥江县|绥江县]]），属昭通府；同年升镇雄州为[[直隶州|直隶州]]；废彝良州同入镇雄州。

[[民国|民国]]二年（1913年）废府、州、厅改县，恩安县改名[[昭通县|昭通县]]，并析镇雄县置[[彝良县|彝良县]]，属[[滇中道|滇中道]]。1914年，因靖江县与[[江苏省|江苏省]][[靖江市|靖江县]]重名，改名[[绥江县|绥江县]]。民国六年（1917年），析大关县盐井渡置[[盐津县|盐津县]]，析镇雄县置[[威信行政区|威信行政区]]。1929年废滇中道。1932年改威信行政区为[[威信设治局|威信设治局]]，1934年改[[威信县|威信县]]。民国三十一年（1942年）各县属云南省第一行政督察区。

[[中华人民共和国|中华人民共和国]]成立后，1950年置[[昭通专区|昭通专区]]，专署驻昭通县，辖昭通、大关、绥江、盐津、威信、镇雄、彝良、鲁甸、会泽、巧家、永善11县。1958年撤销盐津县，并入大关县；撤销鲁甸县，并入昭通县。1962年恢复盐津县。1963年恢复鲁甸县。1964年将会泽县划归[[曲靖专区|曲靖专区]]。1970年昭通专区改称[[昭通地区|昭通地区]]。1981年1月，析昭通县设立昭通市；同年8月，析绥江、盐津两县设立水富县。1983年9月，撤销昭通县，并入昭通市。2001年1月，撤销昭通地区和县级昭通市，设立地级[[昭通市|昭通市]]；原县级昭通市改为[[昭阳区|昭阳区]]<ref>[http://www.gov.cn/gongbao/content/2001/content_61278.htm 国务院关于同意云南省撤销昭通地区设立地级昭通市的批复]</ref>。2018年7月2日，经国务院批准，撤销水富县，设立县级[[水富市|水富市]]，以原水富县的行政区域为水富市的行政区域<ref>{{cite web |author1=云南省人民政府办公厅 |title=国家同意云南省撤销水富县设立县级水富市 |url=http://www.yn.gov.cn/yn_zwlanmu/yn_dfzw/201809/t20180907_33871.html}}</ref>。

== 地理 ==
地形复杂，气候多样。昭通位居于云贵高原与四川盆地的结合地带，云贵川三省交汇中间地带。山高谷深，海拔高差大，最高海拔4040米（巧家县药山），最低海拔267米（水富县滚坎坝）。昭通市域总面积2.3万平方公里。南侧紧邻云南昆明[[东川区|东川区]]，云南[[曲靖市|曲靖市]]，东侧紧邻，贵州[[毕节地区|毕节地区]]，北侧紧邻，四川[[宜宾市|宜宾市]]，四川[[凉山彝族自治州|凉山彝族自治州]]。西侧由南至北和四川凉山，四川宜宾以[[金沙江|金沙江]]为界相邻。

=== 气候 ===
气候属亚热带、暖温带共存，年平均气温12.6℃，年平均日照1900小时，年平均降雨量704至760毫米。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为5213533人<ref>{{cite book |author=昭通市统计局|title=《昭通市2010年第六次全国人口普查主要数据公报》}}</ref>。同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加了621145人，增长13.53%。平均每年增加62114.5人，年平均增长率为1.28%。其中，男性为2738005人，占总人口的52.52%；女性为2475528人，占总人口的47.48%。总人口性别比（以女性为100）为110.60。0－14岁的人口为1400577人，占总人口的26.86%；15－59岁人口为3312738人，占63.54%；60岁及以上人口为500218人，占9.60%，其中65岁及以上的人口为354953人，占6.81%。居住在城镇的人口为1094842人，占总人口的21.00%；居住在乡村的人口为4118691人，占总人口的79.00%。

== 交通 ==
处于昆明、成都、贵阳、重庆等中心城市辐射的交汇地，是规划中的的“攀西——六盘水经济开发区”腹心地带<ref>[http://www.cgp.gov.cn/dzzh/px-lps.htm 攀西——六盘水经济开发区]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

* 铁路，[[内昆铁路|内昆铁路]] 2001年建成通车，市辖境内有 昭通 彝良 大关 盐津 盐津北 水富 火车站。 正在规划中的[[渝昆铁路|渝昆高速铁路]]，初步设想走向为 [[重庆|重庆]]－[[永川|永川]]－[[泸州|泸州]]－[[昭通|昭通]]－[[昆明|昆明]]。  和规划中的 [[丽江|丽江]]-[[攀枝花|攀枝花]]-[[昭通|昭通]]铁路。

* 航空，[[昭通机场|昭通机场]] , “昆明－昭通”“昆明-昭通-重庆”“昭通-成都”“昭通-北京”“昭通-深圳”“昭通-贵阳”往返航班

* 水运，水富港，为云南省第一大港，也是长江上自源头以来的第一座较大规模的港口。

* 高速公路，昭通待补高速公路，149公里<ref>[http://www.yn.xinhuanet.com/info/zdgl/ 云南昭待高速公路]</ref>，和水富至麻柳湾高速公路<ref>[http://www.yn.xinhuanet.com/nets/2007-10/11/content_11372557.htm 水麻高速公路] {{webarchive|url=https://web.archive.org/web/20080906220950/http://www.yn.xinhuanet.com/nets/2007-10/11/content_11372557.htm |date=2008-09-06 }}</ref>（[[渝昆高速公路|渝昆高速公路]]云南段），136公里。

* 公路，[[213国道|213国道]]南北贯穿昭通，连接境内 [[绥江|绥江]] [[大关|大关]] [[昭阳区|昭阳区]] 等地。

==资源物产==
昭通市物产丰富，已知矿产资源33种，其中，[[煤|褐煤]]储量达81.98亿吨，中国南方最大的褐煤田<ref>[http://zhaotong.mofcom.gov.cn/aarticle/zhongyaozt/200603/20060301697577.html 昭通市的褐煤资源位置和地理简况] {{webarchive|url=https://web.archive.org/web/20080507190114/http://zhaotong.mofcom.gov.cn/aarticle/zhongyaozt/200603/20060301697577.html |date=2008-05-07 }}</ref>。

水能资源丰富，水能蕴藏量为2080万千瓦，可开发装机容量1612万千瓦，在[[金沙江|金沙江]]下游昭通境内已建设成的 [[溪洛渡电站|溪洛渡电站]]成为世界第三大水电站，中国第二、[[向家坝|向家坝]]（建成后，将为世界第四大水电站，中国第三）、和规划中的[[白鹤滩|白鹤滩]]三座巨型水电站。

昭通市昭阳区，被誉为中国[[黑颈鹤|黑颈鹤]]之乡，昭通[[大山包|大山包]]海拔高，空气清新，为黑颈鹤主要过冬栖息地。乌蒙山区又是乌蒙马的原产地<ref>《滇南见闻录》记：“滇中之马善走山路，其力最健，乌蒙者尤佳。体质高大，精神力量分外出色，列于凡马内，不啻鹤立鸡群。”</ref>。

昭通也是中国南方优质苹果基地，昭通[[苹果|苹果]]，[[天麻|天麻]]，[[杜仲|杜仲]]等特产驰名省内外。

== 教育 ==
* [[昭通一中|昭通一中]]，前身 "凤池书院"，建于1913年，省立第二中学。为云南省一级高级中学。 校友[[姜亮夫|姜亮夫]]等。

== 名人 ==
* [[龙云|龙云]]，云南昭通人，中华民国滇军将领，云南省政府主席。主政云南17年，被称为云南王。彝族人。抗日战争期间，云南地处后方，但龙云先后派遣滇军20多万赴抗战前线，为抗战胜利作出巨大贡献。 

* [[卢汉|卢汉]]，生于云南昭通炎山，国民革命军将领。彝族人。对日抗战期间任第60军军长，后任第一集团军总司令，曾参与台儿庄战役、武汉会战、长沙会战。 

* [[罗炳辉|罗炳辉]]，昭通彝良人，中国工农红军、八路军和新四军高级将领。 

* [[姜亮夫|姜亮夫]]，云南昭通人，中国卓越的教育家，楚辞学家、敦煌学家、语言学家、文献学家、一代国学大师。云南现代学人中成就最高、影响最大的学者之一。
