'''鍾祥'''（{{bd|？||？|}}）字'''云亭'''，汉姓'''杨'''，[[清朝|清朝]][[内务府汉军|内务府汉军]][[镶黄旗|镶黄旗]]人。清朝官员。

==生平==
[[嘉慶|嘉慶]]十三年（1808年），鍾祥中戊辰科[[进士|进士]]。此後历任山东按察使、浙江布政使、[[山东巡抚|山东巡抚]]等职。[[道光|道光]]十六年（1836年）授[[闽浙总督|闽浙总督]]。道光二十二年（1842年）奉召来北京，授[[库伦办事大臣|库伦办事大臣]]，改授东河[[河道总督|河道总督]]<ref name=yin>{{Cite web |url = http://www.tjdag.gov.cn/tjdag/wwwroot/root/template/main/jgsl/lsbl_article.shtml?id=687 |title =  清末内务府的“四大家族”|author = 尹伊君 |date =  |publisher = 天津档案网 |language =  |accessdate = 2017-01-23 |quote =  }}</ref>。

鍾祥的家族世称“钟杨家”，累代巨富。钟杨家的宅邸在北京[[旧鼓楼大街|旧鼓楼大街]]马家厂（今[[前马厂胡同|前马厂胡同]]、[[后马厂胡同|后马厂胡同]]），“户舍连云，几遍前后两街，四乡田地尤广，有终年取不尽之租”（此处所称“前后两街”即如今的前马厂胡同、后马厂胡同）。后来因年深日久，有人不明白内务府旗人姓氏称谓的习惯，讹传“钟杨家”是铸钟的工匠世家，由马家厂附近的[[铸钟胡同|铸钟胡同]]敷衍出这家的事迹<ref name=yin/><ref name=ma>{{Cite web|url =http://bjrb.bjd.com.cn/html/2014-03/04/content_157347.htm|title =马与胡同不了情|author =朱小平|date =2014-03-04|publisher =北京日报|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170202020255/http://bjrb.bjd.com.cn/html/2014-03/04/content_157347.htm|archivedate =2017-02-02}}</ref><ref>{{Cite web |url =http://www.bjdclib.com/dclib/subjectdb/dhly/newsphoto/201101/t20110120_37737.html  |title = 北京胡同故事:铸钟厂胡同 钟杨家代代铸钟成首富 |author = 朱小平 |date = 2011-01-20 |publisher =东华流韵  |language =  |accessdate =  |quote =  }}</ref>。
