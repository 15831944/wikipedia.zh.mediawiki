'''鍾理和'''（{{bd|1915年|12月15日|1960年|8月4日|catIdx=Zhong鍾理和}}），[[筆名|筆名]]'''江流'''、'''里禾'''、'''鍾錚'''、'''鍾堅'''，是一位出身臺灣[[六堆|六堆]]的[[臺灣客家人|客家]]籍[[作家|作家]]。

他的代表作有[[長篇小說|長篇小說]]《笠山農場》、[[中篇小說|中篇小說]]《雨》及[[短篇小說|短篇小說]]《[[原乡人_(小说)|原鄉人]]》、《貧賤夫妻》等，一生創作共計長篇小說一部、中篇小說七部、短篇小說四十一篇，還有部分[[詩歌|詩歌]]和[[散文|散文]]等。被稱為「'''倒在血泊裡的筆耕者'''」。

== 生平 ==

=== 家族 ===
鍾理和的父親是鍾鎮榮（[[诨名|偏名]]蕃薯），母親是劉水妹；他的父親是[[六堆|六堆]]客家地區當時著名的[[土地主|地主]]和[[企業家|企業家]]，當鍾理和出生時他已年事漸高，並決定在[[美濃區|美濃]]經營[[農場|農場]]準備[[退休|退休]]養老。<ref>{{Cite news|url=http://chunghouse.pixnet.net/blog/post/178065387|title=六堆農企業家--鍾蕃薯 @ 鍾理和故居（鍾理和个老屋） :: 痞客邦 PIXNET ::|date=2016-10-26|newspaper=archive.is|accessdate=2017-03-12|deadurl=yes|archiveurl=https://archive.is/20161026235529/http://chunghouse.pixnet.net/blog/post/178065387|archivedate=2016-10-26}}</ref>

鍾理和另有一同父異母弟[[鍾和鳴|鍾和鳴]]（後改名鍾浩東）；他在擔任[[基隆中學|基隆中學]]校長時因成立[[中國共產黨|中國共產黨]]基隆中學支部遭[[國民黨|國民黨]]政府逮捕槍決。

=== 早年生活 ===
1915年，鍾理和出生於[[阿緱廳|阿緱廳]][[阿里港支廳|阿里港支廳]][[鹽埔區|鹽埔區]]大路關-{庄}-新大路關（今[[屏東縣|屏東縣]][[高樹鄉|高樹鄉]]廣興村，傳統[[六堆|六堆]]聚落之一）。<ref>鍾鐵民總編輯，《探訪鍾理和：紀念館暨文學地景》，高雄市：春暉，2010年1月，頁10。</ref>

1928年，鍾理和自鹽埔[[公學校|公學校]]畢業。後來，他進入長興公學校（現稱屏東縣長興國小）就讀高等科，又於1930年至1932年間接受[[私塾|私塾]]漢文教育；此後，他大量閱讀古典[[文言文|漢文]]詩詞和新體[[小说|小說]]，並開始嘗試創作。

1932年，鍾理和在結束私塾課業後隨父親遷居[[高雄州|高雄州]][[旗山郡|旗山郡]][[美濃街|美濃-{庄}-]]（今[[高雄市|高雄市]][[美濃區|美濃區]]）經營笠山農場；此時，他在當地認識了他日後的妻子[[鍾台妹|鍾台妹]]。<ref>{{Cite web|url=http://flipliterature.tw/%E9%8D%BE%E7%90%86%E5%92%8C%E3%80%8A%E8%B2%A7%E8%B3%A4%E5%A4%AB%E5%A6%BB%E3%80%8B/|title=《貧賤夫妻》|accessdate=2017-03-12|author=鍾理和|date=|work=flipliterature.tw|publisher=[[亞洲大學_(臺灣)|亞洲大學]]|language=zh-TW}}</ref>

=== 前往滿洲國 ===
[[File:鍾理和與鍾台妹.jpg|250px]]
1938年，他與鍾台妹[[同姓不婚|同姓結婚]]受阻，於是隻身前往[[滿洲國|滿洲國]]境內[[瀋陽|瀋陽]]進入滿洲[[自動車|自動車]]學校以取得經濟自立。

1940年，他返回臺灣攜帶妻子再度前往滿洲，並於[[瀋陽|瀋陽]]生下長子[[鍾鐵民|鍾鐵民]]。<ref>{{Cite book|title=〈門〉|last=鍾理和|first=|publisher=|year=|isbn=|location=|pages=}}</ref><ref>{{Cite book|title=〈泰東旅館〉|last=鍾理和|first=|publisher=|year=|isbn=|location=|pages=}}</ref><ref name=":0">{{Cite web|url=http://forums.chinatimes.com/report/people_story/20081011/inside3.html|title=中時電子報│名人家族故事|accessdate=2017-03-12|work=forums.chinatimes.com}}</ref>

1941年，他攜帶妻小遷居[[北平|北平]]；期間，他曾經營煤炭零售業，更花費許多時間專事寫作，並出版《夾竹桃》（北平：馬德增書局，1945）一書；該書收錄〈夾竹桃〉、〈新生〉、〈游絲〉、〈薄芒〉四篇中篇小說，富有[[五四文學作家|五四文學]]色彩，並深受[[魯迅|魯迅]]風格影響，在對中國國民性的批判與反省之相關主題有所承續。<ref>{{Cite book|url=https://books.google.com.tw/books?id=t8lYvgAACAAJ&dq=%E5%A4%BE%E7%AB%B9%E6%A1%83&hl=zh-TW&sa=X&ved=0ahUKEwj1h-Dc29HSAhXImZQKHbGPDIAQ6AEILDAD|title=夾竹桃|last=鍾理和|first=|date=1977-01-01|publisher=遠行出版社|year=|isbn=|location=|pages=|language=zh|chapter=}}</ref>其中〈薄芒〉以故鄉台灣美濃竹頭庄為故事舞台。

=== 返回臺灣、肺病發作 ===
1946年3月底，鍾氏返回臺灣；同年4月中，他應聘於屏東縣立內埔初級中學擔任代用國文教師；但是，他也在此間被診斷罹患[[肺病|肺病]]。

1947年元月，由於肺病再度惡化，鍾理和北上國立臺灣大學醫學院附屬醫院診療；期間，他經歷了[[二二八事件|二二八事件]]，並將[[日記|日記]]寫於內服藥袋之上；二二八事件發生後，他決定南返美濃定居。

1947年8月，他應[[鍾和鳴|鍾和鳴]]之邀再度北上擔任[[基隆中學|基隆中學]]總務主任，但因肺病復發而僅任職兩個月。同年10月底，他進入[[臺北市立聯合醫院松德院區|松山療養院]]接受治療，並於此後三年未曾返家。1950年5月中之後，他歷經二次[[手術|手術]]，並切除六根[[肋骨|肋骨]]，病情轉趨穩定；但是，他日後因此身體孱弱而無法進行一般勞動。同年10月23日，他在出院三日後返回家鄉。[[File:鍾理和、鍾浩東故居.jpg|thumb]]

1954年，他的次子鍾立民因病過世。<ref name=":0" />

=== 獲獎 ===
1956年，他的[[長篇小說|長篇小說]]〈笠山農場〉獲得[[中華文藝獎金委員會|中華文藝獎金委員會]]頒發國父誕辰紀念獎第二名（首獎從缺，第三名有兩位：[[彭歌|彭歌]]的〈落月〉、[[王藍|王藍]]的〈藍與黑〉）。同年12月，中華文藝獎金委員會停辦，導致鍾理和的〈笠山農場〉無法出版；歷經多次求情，他終於要回原稿，但又於尋覓出版機會時處處碰壁。

1957年3月，廖清秀主動贈與《恩仇血淚記》（1952年獲文獎會長篇小說第三獎）；此時，鍾理和開始與其他[[臺灣人|臺灣籍]]作家建立連結，並加入[[鍾肇政|鍾肇政]]發起的《[[文友通訊|文友通訊]]》。同年11月，廖清秀到美濃笠山腳下探望鍾理和，成為唯一見過鍾理和的文壇友人。<ref>〈台灣文學的祕密結社：文友通訊時期的台灣文學〉，主編莊華堂，《戰後台灣文學發展史十二講》（台北：唐山，2007年），頁58-59。</ref>

由於文壇友人相助與[[林海音|林海音]]提攜，鍾理和的作品開始較為頻繁地獲得刊登機會，尤其在1959年到1960年之間於《[[聯合報|聯合報]]》[[副刊|副刊]]（林海音主編）。

=== 病逝 ===
1960年8月4日，他在修改中篇小說《雨》時喀血而死；他因鮮血濺灑在書稿之上而被後人稱為「倒在血泊裡的筆耕者」（[[陳火泉|陳火泉]]語）。<ref>{{Cite news|url=http://www.shs.edu.tw/works/essay/2009/03/2009033116481668.pdf|title=「倒在血泊裡的筆耕者-鍾理和」|date=2017-03-12|newspaper=archive.is|accessdate=2017-03-12|deadurl=yes|archiveurl=https://archive.is/20170312200320/http://www.shs.edu.tw/works/essay/2009/03/2009033116481668.pdf|archivedate=2017-03-12}}</ref>此後，親族依其遺囑[[火葬|火化遺體]]，並將之安厝於家鄉附近[[朝元寺_(美濃)|朝元寺]]奉祀。同年9月1日，〈雨〉在聯合報的副刊連載；10月，林海音、鍾肇政和文心等友人組成「[[鍾理和遺著出版委員會|鍾理和遺著出版委員會]]」，出版其中短篇小說集《雨》（[[文星書店|文星書店]]發行）<ref>共收入：〈雨〉〈復活〉〈閣樓之冬〉〈貧賤夫妻〉〈錢的故事〉〈菸樓〉〈柳蔭〉〈蒼蠅〉〈還鄉記〉〈草坡上〉〈楊紀寬病友〉〈同姓之婚〉〈奔逃〉〈阿遠〉〈假黎婆〉〈初戀〉共16篇。</ref>，[[跋|跋]]曰：「在他的百日祭的供桌上，除了清茶鮮果之外，多了一本他的著作，對於這位一生熱愛寫作的人，應當是一件較有意義的事。」1961年8月4日，遺著委員會於其逝世周年時出版《笠山農場》（學生書店發行），終於完成其生前心願。<ref>張良澤，〈鍾理和全集總序〉，《鍾理和殘集》（遠行出版社，1976年），頁8-9。</ref>

== 影響 ==
*2004年3月14日，[[中国總理|中国總理]][[溫家寶|溫家寶]]在北京[[人民大會堂|人民大會堂]]新聞發佈會上回应台湾记者的发问时，朗诵了鍾理和作品《原鄉人》的“原鄉人的血，必須流返原鄉，才會停止沸騰！”这一句，以表示对台湾的「关切」，卻不慎將原鄉人念成原野人。<ref>[http://news.xinhuanet.com/book/2004-03/25/content_1381487.htm 走近原鄉人——鐘理和]新华网，2004-03-25，来自[[中国青年报|中国青年报]]</ref>。然而，鍾理和《夾竹桃》顯示旅居中國多年的鍾理和對中國原鄉的幻滅，原鄉人是鍾理和過世前的作品，代表的是鍾理和心中客家人的原鄉廣東，以及青年時對日本殖民壓迫尋求文化與理想中的「中國」。

== 評論摘錄 ==
*「臺灣因為有鍾理和這樣的作家出現，所以才會影響到後來，慢慢的，台灣作家強調我們台灣鄉土。」（[[陳火泉|張良澤]]語）<ref>{{Cite web|url=http://web.pts.org.tw/hakka/news/detail.php?id=67257|title=客家電視台 CH 17|accessdate=2017-03-12|work=web.pts.org.tw}}</ref><ref>鍾鐵民，〈編印《鍾理和全集》〉，鍾怡彥主編，《鍾鐵民全集（七）》，頁240。</ref><ref>張良澤，〈鍾理和全集總序〉，《鍾理和殘集》（遠行出版社，1976年），頁25。</ref>
*「由於中國文學一向都是由士大夫以至於「都市才子」的拿手好戲， 文學的偏食症侯極為嚴重，鍾理和的「農民文學」因此顯得特別珍貴，通過鍾理和，我們希望了解五十年代的台灣農村生活，他的小說也 必然有一定的文學以外的社會價值吧。」（[[陳火泉|唐文標]]，〈來喜愛鍾理和〉，《文季》2期，1973年11月）
*「在當時日本欺凌中國人，以及偉大的民族抗日戰爭，他沒有採取更積極的立場，沒有參與更建設的行動，更很少看他提及，這一點不能不說他的世界觀太狹隘，只能在個人的愛情生活轉迷宮之故了。」（出處同上 )<ref>{{Cite web|url=http://teacher.whsh.tc.edu.tw/huanyin/mofa_chungliho_9.htm|title=鍾理和|accessdate=2017-03-12|work=teacher.whsh.tc.edu.tw}}</ref>
*「我們讀了七十年代各家批評，不免感慨，「殖民地作家」真是難當──〈夾竹桃〉「抵抗」「批評」得太多，《笠山農場》則「抵抗」得太少，都成了批評家批評的箭靶；「太勇敢」或「太不夠勇敢」都顯得可疑。」（應鳳凰，〈鍾理和文學發展史（代序）〉，《鍾理和論述1960-2000》，高雄市：春暉，2004年）
*「近幾十年台灣文壇儘管經歷幾次論戰與意識形態的變遷，鍾理和握筆在文學園地裡認真、嚴肅耕耘的身影，經歷幾場風吹雨打，影像反而更為清晰。也可以說，是他對『文學』二字的嚴肅態度，激發他創作了不僅具個人風格，更代表台灣這塊土地特性的作品。一個社會的文學思潮，可以隨著時間改變而吹著不同的風向，然而作為好的，嚴肅文學作品的本質條件應該是不變的。」（出處同上 )
*「50年代台灣文學歷史，如果沒有鍾理和文學存在，景觀將變得偏頗而單調，失去台灣站在東亞特殊位置的關鍵性與複雜性。」（應鳳凰：〈鍾理和與他的《笠山農場》〉，《文訊》289 期，2009年11月）
*「鍾理和是戰後台灣第一位以優美中文創作的本省籍作家，也是最早刻劃台灣土地之美的農民文學創作者。」（出處同上 )
*「如果沒有鍾理和文學，50年代台灣便只是『大陸文人的文壇』，只有反共小說與懷鄉散文。有了《笠山農場》以及由這塊土地吸納出來的辛酸、愛戀與嘆息的作品，『台灣鄉土文學』的歷史與特色，便可由戰前到戰後連成一貫，而不是遲至70年代的黃春明、王禎和才出現。它是一條長長的線，而不是斷裂的零星現象。」（出處同上 )

== 紀念 ==

*鍾理和紀念館：一座位於高雄市[[美濃區|美濃區]]的文學紀念館，也是鍾理和文教基金會所在地。1979年，該館建立於[[鄉土文學論戰|鄉土文學論戰]]不久之後由林海音、鍾肇政、[[葉石濤|葉石濤]]、[[鄭清文|鄭清文]]、[[李喬|李喬]]、張良澤等人具名發起籌建。[[柴松林|柴松林]]、[[曾貴海|曾貴海]]分別為基金會首、二任董事長。館外設有台灣文學步道。<ref>{{Cite news|url=http://www.cw.com.tw/article/article.action?id=5026593|title=鍾鐵民  美濃鄉土的守護者|last=|first=|date=2011-09-06|work=|newspaper=[[天下雜誌|天下雜誌]]第480期|accessdate=2017-03-12|language=zh-TW}}</ref><ref>鍾鐵民，〈我的父親及「鍾理和紀念館」〉，《台灣時報》，1982年8月7日。</ref><ref>{{Cite news|url=http://literature.ihakka.net/hakka/author/zhong_li_he/default_memorial.htm|title=鍾理和紀念館|date=2013-05-02|newspaper=archive.is|accessdate=2017-03-12|deadurl=yes|archiveurl=https://archive.is/20130502084009/http://literature.ihakka.net/hakka/author/zhong_li_he/default_memorial.htm#selection-161.6-161.46|archivedate=2013-05-02}}</ref><ref>{{Cite web|url=http://chungliher.blogspot.tw/2012/11/blog-post.html|title=人與土地的書寫─鍾理和的文學世界，鍾理和紀念館二樓常設展開幕式|accessdate=2017-03-12|work=chungliher.blogspot.tw}}</ref>
