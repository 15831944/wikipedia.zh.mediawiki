'''鍾邦柱'''，[[中研院|中研院]]分子生物所特聘研究員，第五屆台灣女科學家「傑出獎」得主。<ref>{{Cite web|url=http://www.imb.sinica.edu.tw/chung/|title=Bon-chu Chung Lab|accessdate=20180324|author=|date=|publisher=}}</ref>專長是類固醇功能與調控, 斑馬魚發育, 小鼠動物模式與基因調控。<ref>{{Cite web|url=http://biochem.web.ym.edu.tw/files/15-1256-15952,c22-1.php|title=生化暨分子生物學組師資陣容|accessdate=20180324|author=|date=|publisher=}}</ref>中央研究院生命科学组院士。

== 個人生平 ==
鍾邦柱博士高中時就讀[[臺北市立第一女子高級中學|北一女中]]，升高二時她同時對語言及數理科學都有興趣，但最終選擇了理組。讓她下定決心選擇理組的關鍵，是因為[[陳之藩|陳之藩]]。由於她很喜歡陳之藩的作品，原來讀[[科学|科學]]的人也可以寫出優美的文學作品，因此決定以他為典範，期許自己未來可以跟他一樣以文理雙向發展。這就是鍾邦柱走上科學之路的開始。<ref>{{Cite web|url=http://epaper.ntu.edu.tw/view.php?id=14578|title=生化女先鋒：鍾邦柱的科學研究之路／CASE編輯整理|accessdate=20180324|author=|date=|publisher=}}</ref>

而後就讀[[國立臺灣大學|台大]]化學系，並於1979年美國[[宾夕法尼亚大学|賓州大學]]取得生物化學博士學位之後，繼續在[[普渡大學|普度大學]]生物化學系進行博士後研究。1982年開始，鍾邦柱博士擔任[[加利福尼亞大學|加州大學]][[旧金山|舊金山]]分校博士後研究員，受到指導教授的影響，決定踏入類固醇荷爾蒙的研究。<ref name=":0">{{Cite web|url=http://wcscamp-register.com/wcs_web/wcs_scientists_2012(1).html|title=第五屆台灣女科學家「傑出獎」得主－鍾邦柱 - 簡介|accessdate=20180324|author=|date=|publisher=}}</ref>

鍾邦柱博士於中研院分生所甫成立時即加入工作，為[[臺灣|台灣]]的研究紮根。二十多年來，研究荷爾蒙基因的突變與調控，成果斐然，其研究特色為用多種不同方法來探討荷爾蒙的調控，有豐富且深入的成果，其著作多發表在國際上一流期刊，包括著名的《[[自然_(期刊)|自然]]》期刊(Nature)。她的發現，不僅具學術意義，也有臨床與應用價值。<ref name=":0" />

== 研究成果 ==
類固醇是重要的荷爾蒙，在醫學與農學上用途很多，但是副作用也很廣泛。而它的調控方式，變化很多，也是目前醫藥界努力想要了解的題目。鍾邦柱博士的研究工作，正是沿著這個方向進行。成果有以下諸項：
# 利用[[斑馬魚|斑馬魚]]，了解類固醇的新功能：使用斑馬魚，發現固醇類的[[孕烯醇酮|孕烯醇酮]]，有促進細胞移動的功能，論文刊登於著名的「自然」期刊。了解胚胎的發育以及其機轉，促成新藥開發，以保護神經細胞。
# 建立小鼠的動物模式：製造出類固醇合成有缺陷的小鼠，用以研究類固醇缺失所造成的小鼠的問題，藉以了解類固醇的功能與調控。對醫藥的研究很重要。
# 了解類固醇[[基因|基因]]調控的機制：知道類固醇的分泌，受到一重要[[轉錄因子|轉錄因子]][[类固醇生成因子1|SF-1]]的作用。此因子如何受到調節，也被仔細探討，成果發表在重要國際期刊上。
# 闡述[[類固醇|類固醇]]缺失遺傳疾病的原因及檢測：了解人類先天性[[激素|荷爾蒙]]不平衡的重要原因，是CYP21基因的突變。對此疾病的基因突變與病情的關係，做詳細的檢測。設計了產前診斷，嘉惠病人及家屬。

* 2011～迄今，中央研究院分子生物研究所副所長
