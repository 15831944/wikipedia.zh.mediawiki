'''鍾白蘭'''（{{lang-en|'''Brenda Leigh Johnson'''}}）是美國警匪電視劇《[[真相追擊|真相追擊]]》中的一名虛構角色，為該劇的主角，由[[姬娜·薛域|姬娜·薛域]]飾演。

鍾白蘭在[[洛杉磯警察局|洛杉磯警局]]中當[[美国警衔|副總警監]]（Deputy Chief），率領重案組（Major Crimes Division）。她經驗豐富兼[[破案率|破案率]]極高，因而得到「結案者」（the closer，即劇集英文原名）這個花名。

== 背景 ==
父親Clay Johnson為[[美國陸軍|美國陸軍]][[上尉|上尉]]<ref name="1:4, Show Yourself">1:4, "Show Yourself"</ref>（現已退役），母親為Willie Rae Johnson；幼時一家三口需時常隨當兵的父親四處搬家。

大學畢業於[[乔治城大学|乔治城大学]]，其後在[[中央情報局|中央情報局]]受訓及工作，7年後轉到首府[[华盛顿哥伦比亚特区|華盛頓]]警局，期間認識了現時的長官包維爾（William Pope）。在華盛頓警局4年後，再轉到[[亚特兰大|亚特兰大]]警局當[[美国警衔|警監]]（Captain），卻因當時丈夫的干擾只待了3年半。與前夫離婚後，決定找個新環境，所以轉來[[洛杉磯|洛杉磯]]。

== 私人生活 ==
在華盛頓警局期間與已婚的包維爾有過一段感情。

在第二季開始與[[FBI|FBI]]探員何輝智（Fritz Howard）同居，兩人於第四季季尾結婚。

在第五季中，鍾白蘭的姪女Charlie Johnson（演員為[[姬娜·薛域|姬娜·薛域]]現實生活中的女兒）因與父母不和，跟著祖父母Clay及Willie Rae跑了到洛杉磯，最後獲鍾白蘭及何輝智收留。

有一隻名為Kitty的[[貓|貓]]；鍾白蘭本以為Kitty是雄性，但它在第一季中卻生下了小貓。在第五季，Kitty因病重而最後受人道毀滅；同季季尾，何輝智送了一隻新的小貓給鍾白蘭作生日禮物。

== 特點 ==
說話帶[[美國南部|南部]]口音，嗜吃甜食，對生活上的事情很不擅長。

在得知Kitty是雌性後，卻依然用男性代名詞來稱呼Kitty。
