'''观察者网'''是[[中华人民共和国|中华人民共和国]]一家以新闻及评论为主的互联网新闻媒体，创立于2012年，总部位于[[上海市|上海市]]<ref name=jj/>。

== 简介 ==
观察者网原本是《社会观察》的网络平台。《社会观察》（月刊）2003年创刊，是由[[上海社会科学院|上海社会科学院]]主办的[[社会科学|社会科学]]刊物。2010年，《社会观察》由春秋综合研究院接办。2011年上半年改为“《社会观察》编辑部”出版，“上海观察者文化传媒有限公司”（成立于2010年5月7日，法人代表[[金仲伟|金仲伟]]）运营，同时春秋综合研究院继续提供“学术支持”。后来又改为“上海社会科学杂志社”出版，“《社会观察》编辑部”编辑，并沿袭至2014年。2014年6月起，《社会观察》与观察者网脱离关系<ref name="wyz">{{Cite web |url = http://www.wyzxwk.com/Article/shidai/2015/01/335970.html |title = 观察者网、春秋综合研究院和华尔街资本 |author =  |date =2015-01-05  |publisher = 乌有之乡 |language =  |accessdate =  |quote =  }}</ref>。

观察者网域名归上海观察者信息技术有限公司（成立于2012年11月6日，法人代表金仲伟）所有。2012年4月16日，观察者网www.guancha.cn正式上线。2012年10月31日，观察者网手机站m.guancha.cn开通。2013年2月1日，观察者网苹果app上架。同年，观察者网的微信（ID：guanchacn）、微博（ID：@观察者网 ）陆续上线<ref name=jj>{{Cite web |url = http://www.guancha.cn/about/about.shtml |title = 关于观察者网 |author =  |date =  |publisher = 观察者网 |language =  |accessdate =2017-12-23  |quote =  }}</ref>。

据传观察者网和[[四月网|四月网]]都是由[[李世默|李世默]]投资创办，不过该传言从未获得证实<ref name="wyz"/><ref name="21ccom">{{cite web | url=http://www.21ccom.net/articles/zgyj/gqmq/article_2013102994411_2.html | title=赵新宇：谁在养活四月青年？(2) | publisher=[[共识网|共识网]] | accessdate=2014-07-05 | date=2013-10-29|archiveurl=https://web.archive.org/web/20160305012010/http://www.21ccom.net/articles/zgyj/gqmq/article_2013102994411_2.html|archivedate=2016-03-05|deadurl=yes}}</ref>。观察者网是由[[上海春秋发展战略研究院|上海春秋发展战略研究院]]（2014年正式成立，法人代表金仲伟）实际运作。观察者网的自我定位是“新闻时评集成网站”<ref name=jj/>。觀察者網是在上海市互聯網信息辦公室登記的互聯網新聞信息服務單位，獲得「轉載服務」類別的「互聯網新聞信息服務許可」，不過觀察者網的網站及微博都刊載了大量自行編採的國內及國際政治、經濟、外交新聞訊息。<ref>{{cite web |title=上海市互联网新闻信息服务单位许可信息 |author1=上海市互联网信息办公室 |url=http://www.shio.gov.cn/sh/xwb/n805/n808/u1ai16750.html |date=2018-04-20}}</ref>

观察者网自称其立场是“中国关怀全球视野；从这里读懂世界；独立而负责任”。其特约观察员涵盖了工商业界、媒体文化、高校研究所等领域的专家学者，包括[[史正富|史正富]]、[[张军_(经济学家)|张军]]、[[寒竹|寒竹]]、[[胡鞍钢|胡鞍钢]]、[[张维为|张维为]]、[[陈平_(经济学者)|陈平]]、[[张文木|张文木]]、[[朱云汉|朱云汉]]、[[强世功|强世功]]、[[潘维|潘维]]、[[李世默|李世默]]、[[宋鲁郑|宋鲁郑]]、[[郑若麟|郑若麟]]、[[边芹|边芹]]、[[罗思义|罗思义]]（John Ross）、[[王文|王文]]等等<ref name=jj/>。该网站刊登的文章包括了不同政治立场<ref name="wyz"/>。以[[乌有之乡|乌有之乡]]为代表的左派认为该网站发表的一些学者的文章具有[[右翼|右翼]][[民族主义|民族主义]]和[[国家主义|国家主义]]色彩<ref name="wyz"/><ref>{{cite web | url=http://www.guancha.cn/FaZhi/2014_01_22_201482.shtml | title=陈有西称中国一些律师是煽动家 袁裕来公开信称其为“恶毒的栽脏和诋毁” | publisher=观察者网 | accessdate=2014-07-05 | date=2014-01-22}}</ref><ref>{{cite web | url=http://www.guancha.cn/GuanChaZheWang/2013_12_20_194121.shtml | title=观察者网一周图说：2013他们眼里的中国 | publisher=观察者网 | accessdate=2014-07-05 | date=2013-12-20}}</ref>，该网站也曾刊文质疑美式选举制度<ref>{{cite web | url=http://www.guancha.cn/Martin-Gilens/2014_06_12_236911_s.shtml | title=马丁·季伦思、本杰明·佩奇：美国政治理论检验 ——精英、利益集团和普通民众 | publisher=观察者网 | accessdate=2014-07-05 | date=2014-06-12}}</ref>，抨击[[新自由主义|新自由主义]]<ref>{{cite web | url=http://www.guancha.cn/Kojin-Karatani/2014_06_11_235721.shtml | title=重新思考“帝国主义”——《民族与美学》中文版序言 | publisher=观察者网 | accessdate=2014-07-05 | date=2014-06-11}}</ref>；而自由派则认为该网站的立场偏左<ref name="21ccom"/>。

观察者网每个月都会组织一场“观察者网·春秋论坛”，请工商业界、知识界、传媒界的青年学人、有识之士到观察者网总部就某话题展开交流探讨<ref name=jj/>。
