'''敖厂长'''（{{bd|1991年|11月10日}}），中国大陆網路播客，游戏娱乐解说人，活跃于bilibili弹幕网（B站）以及YouTube，在bilibili弹幕网拥有约500万粉丝。自稱成都養雞二廠{{efn|实际上并不存在}}名譽廠長，出生于中國[[四川省|四川]][[成都市|成都]]，毕业于[[西南石油大学|西南石油大学]]。以[[四川普通話|四川普通話]]和尖锐的语句吸引观众，擅长影片后期处理。2008年在百度贴吧“命运与征服吧”发布《囧的呼唤》第一期视频，《囧的呼唤》系列影片是中国大陆点击率最高的游戏讲解类视频之一<ref>{{cite news|title = 自制游戏视频 敖厂长川普+麻辣的点评火了|url = http://www.sc.xinhuanet.com/content/2012-09/17/c_113100237.htm|accessdate = 2012年12月29日|newspaper = 四川在线-天府早报|date = 2012年9月17日|language = zh-cn|deadurl = yes|archiveurl = https://web.archive.org/web/20140727043257/http://www.sc.xinhuanet.com/content/2012-09/17/c_113100237.htm|archivedate = 2014年7月27日}}</ref>。本名未知，网传的真名'''敖缘凤'''也只是出自'''囧的呼唤'''中敖厂长对自己虚拟形象的设定，本人在其微信公众账号中表示自己其实并不姓敖。{{efn|微信ID：hawkaoao，评论日期：2015年1月18日}}

== 个人经历 ==
敖厂长于2008年开始制作以游戏娱乐讲解和游戏文化探讨为主旋律的在线系列影片《囧的呼唤》，并在百度贴吧命令与征服吧进行周期性更新。早期在《囧的呼唤》中以“流精电视台主持人Mr. Jiong（囧先生）”的身份出现，作品大多上传到[[bilibili|bilibili]]和[[优酷|优酷]]（目前在优酷上已停更）。他曾以敖尼玛的身份客串《暴走大事件》（第四季還有出現），后来的《敖尼玛的脱口秀》和《暴走敖尼玛》也是经暴走漫画<ref>{{Cite web|url = http://i.youku.com/u/UNTY5MDQ5Njcy|title = 暴走漫画的优酷空间|accessdate = 2015年2月21日|language = zh-cn}}</ref>发布的。

2013年6月26日，敖厂长在百度贴吧发帖承认自己抄袭另外一位游戏解说人“[[老E|老E]]”的行为，并进行道歉<ref>{{Cite web|url = http://tieba.baidu.com/p/2418923356|title = 给老E的道歉声明|accessdate = 2015年2月21日|language = zh-cn}}</ref>。（事後老E選擇原諒敖廠長，之後老E製作的新一期影片「史上最能摳的國產FC遊戲」當中再度與敖廠長合作，但這次敖廠長是以客串的身份來現身）<ref>{{Cite web|url = http://i.youku.com/u/UMzcyODEwOTAw|title = 老E的优酷空间|accessdate = 2015年2月24日|language = zh-cn}}</ref>。

2014年9月，在[[bilibili|bilibili]]发布的游戏区报告中，敖厂长的[[Minecraft|Minecraft]]系列点击硬币数为第一名<ref>{{Cite web|url = http://www.bilibili.com/topic/494.html|title = BILIBILI游戏区⑨月报告|accessdate = 2015-08-03|author = |date = |publisher = Bilibili|language = zh-cn}}</ref>。

2015年6月，敖厂长指责杂志《[[电脑爱好者|电脑爱好者]]》在2015年12期第21页关于[[极品飞车|极品飞车]]1的介绍完全照搬《囧的呼唤》179期内容，后该杂志社道歉<ref>{{Cite web|url = http://m.weibo.cn/1671959152/CoNal7s5k|title = 《电脑爱好者》抄袭|accessdate = 2015-08-03|author = |date = 2015-06-29|publisher = 新浪微博|language = zh-cn}}</ref><ref>{{Cite web|url = http://card.weibo.com/article/h5/s#cid=1001603859452599105185&from=&wm=&ip=10.13.112.62|title = 《电脑爱好者》致歉声明|accessdate = 2015-08-03|author = |date = 2015-06-30|publisher = 新浪微博|language = zh-cn}}</ref><ref>{{Cite web|url = http://m.weibo.cn/1671959152/CoWOokKCp|title = 敖厂长对致歉声明的回应|accessdate = 2015-08-03|author = |date = 2015-06-30|publisher = 新浪微博|language = zh-cn}}</ref>。

2016年9月，敖厂长的YouTube频道获得了10万订阅数，获得官方创作者奖（YouTube Creator Awards）颁发的白银播放按钮（Silver Play Button），成为中国大陆游戏播客的第一人。

2017年3月9日，敖厂长删除《囧的呼唤》第214期影片（影片中指责游族的《[[盜墓筆記|盗墓笔记]]》（影片未公开运营商为[[游族网络|游族]]并遮挡游戏名称）虚假宣传），在微博上表示“关键时刻还是得服软”“不想因为自己的兴趣爱好被人威胁”。不久后这条微博被删除。其后游族发表声明，否认威胁敖厂长，相关的素材获得使用权。<ref>[http://m.jiangsu.china.com.cn/mobile/2017/kuaixun_0316/9776215.html 敖厂长被威胁事件 是自主行为还是受人胁迫粉丝直呼还敖厂长一个公道] {{webarchive|url=https://web.archive.org/web/20170320053508/http://m.jiangsu.china.com.cn/mobile/2017/kuaixun_0316/9776215.html |date=2017-03-20 }}</ref>
但相同的影片在Youtube上仍能看到<ref>{{cite web| title = 【中国游戏虚假宣传】囧的呼唤214期 - YouTube| url = https://www.youtube.com/watch?v=Ix6tRZ3Mwu4 | accessdate = 2017-06-30 }}</ref>
。
