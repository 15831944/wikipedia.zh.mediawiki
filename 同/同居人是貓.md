== 故事簡介 ==
推理作家朏素晴在為雙親掃墓時意外發現一隻[[流浪貓|流浪貓]]，因而產生了創作靈感，並為了參考而把該貓隻帶回家中飼養。如是者，一個人和一隻貓開始了同居生活，並漸漸成為「家人」。<ref name="木棉花國際"/>

== 登場人物 ==
; {{nihongo|朏素晴|朏素晴（みかづき すばる）|聲：[[小野賢章|小野賢章]]}}<ref name="Character">{{Cite web |url = https://hizaue.com/character/ |title = Character |author = 電視動畫《同居人是貓》官方網站 |language = ja |accessdate = 2019-02-24}}</ref>
: 人類視點的主角。不擅長與人交流的年青[[推理作家|推理作家]]。雙親在旅行時意外去世。
: 在摸索新小說的題材時偶然遇到一隻流浪貓，把牠撿回家後因而產生了創作靈感。其後把流浪貓命名為「陽」，並開始與牠一同生活。

; {{nihongo|陽|陽（はる）|聲：[[山崎遙|山崎遙]]}}<ref name="Character"/>
: 動物視點的主角。雌性的黑白賓士貓，眼神銳利，被人棄養後與弟妹過著艱苦的流浪生活。
: 偶然被素晴撿到，並開始一起生活。本來因「食物」這一個單純的原因而留在素晴家中，其後開始以貓的方式來開始照顧素晴。

; {{nihongo|河瀬篤|河瀬篤（かわせ あつし）|聲：[[下野紘|下野紘]]}}<ref name="Character"/>
: 在「小熊出版社」工作，素晴的擔當編輯。相當積極地與怕生的素晴交流。非常喜歡貓，也很疼愛陽。

; {{nihongo|矢坂大翔|矢坂大翔（やさか ひろと）|聲：[[堀江瞬|堀江瞬]]、[[長谷川育美|長谷川育美]]{{small|（小学生時期）}}}}<ref name="Character"/>
: 素晴的兒時好友。外表看似輕浮，實際上很會社交，很在意失去雙親的素晴。

; {{nihongo|押守奈奈|押守なな（おうかみ なな）|聲：[[安濟知佳|安濟知佳]]}}<ref name="Character"/>
: 寵物商店的親切女店員。給開始與貓共同生活而感到苦惱的素晴提供了建議。和「小六」、「小八」兩隻貓一起生活。
