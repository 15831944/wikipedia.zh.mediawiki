'''《同音汉字书写规则》'''是[[日文|日文]]的[[正寫法|正寫法]]，以[[同音通假|同音通假]]为方法，凡《當用漢字表》外的字，都以同音且通假的当用字替代。[[國語審議會|國語審議會]]于1956年（[[昭和|昭和]]31年）7月5日公布该方案，為業界出版文字的指引，其中包括同音異義語、同音異形詞。

==簡介==
根據1946年（[[昭和|昭和]]21年）的《[[当用漢字表|当用漢字表]]》，凡是《当用汉字表》表外的字，都得用其他方式替換，可以别的詞彙替代，或仅把词中的表外汉字转为[[假名|假名]]。<ref>例如：{{lang|ja|斡旋→あっせん・あっ旋、明瞭→めいりょう・明りょう}}</ref>然而，用其他的詞，則詞不達意，甚或会使文意冗赘，而转写为假名又不符合阅读习惯。相反，[[同音通假|同音通假]]更为直接便利。最初，出版界、各新聞社各自為政，各有词形的標準，使社會用字混亂。于是，國語審議就制定了的标准，以同音通假为计，并规范词形，从而的出《同音汉字书写规则》。

《同音汉字书写规则》分别規定了「代用字」、「代用語」。代用字，指以同音的當用漢字而通假词中的表外漢字。譬如，「{{lang|ja|稀}}」是表外汉字，所以，本作「{{lang|ja|稀少}}」、「{{lang|ja|稀薄}}」的两个詞，都得改为「{{lang|ja|希少}}」、「{{lang|ja|希薄}}」。如此类推，以「{{lang|ja|希}}」为「{{lang|ja|稀}}」的代用字，并取代熟語中的「{{lang|ja|稀}}」字。但是，有的代用字仅限于某些詞組，如：

* 【{{lang|ja|訣}}】
:可變:「{{lang|ja|訣別}}」→「{{lang|ja|決別}}」　
:不可:「{{lang|ja|秘訣}}」→「{{lang|ja|秘決}}」
* 【{{lang|ja|澱}}】
:可變:「{{lang|ja|沈澱}}」→「{{lang|ja|沈殿}}」　
:不可:「{{lang|ja|澱粉}}」→「{{lang|ja|殿粉}}」
* 【{{lang|ja|撥}}】
:可變:「{{lang|ja|反撥}}」→「{{lang|ja|反発}}」　
:不可:「{{lang|ja|撥音便}}」→「{{lang|ja|発音便}}」
* 【{{lang|ja|顚}}】
: 可變:「{{lang|ja|顚倒}}」「{{lang|ja|動顚}}」→「{{lang|ja|転倒}}」「{{lang|ja|動転}}」等　
: 不可:「{{lang|ja|顛末}}」→「{{lang|ja|転末}}」
* 【{{lang|ja|綜}}】
:可變:「{{lang|ja|綜合}}」→「{{lang|ja|総合}}」　
:不可:「{{lang|ja|錯綜}}」→「{{lang|ja|錯総}}」

其外，「{{lang|ja|甚}}」、「{{lang|ja|磨}}」、「{{lang|ja|妄}}」本是表外汉字，且已有了代用字，如「{{lang|ja|蝕甚}}」→「{{lang|ja|食尽}}」、「{{lang|ja|研磨}}」→「{{lang|ja|研摩}}」、「{{lang|ja|妄動}}」→「{{lang|ja|盲動}}」，后来[[常用漢字|常用漢字]]表却收录了以上三字，如今又書寫原字。其实，[[日本新聞協会|日本新聞協会]]、[[学術用語集|学術用語集]]也有非官方的同音替代語，并且已经廣泛于民間。

同音通假罕用於人名、固有名詞，但「{{lang|ja|満洲}}」一般表記為「{{lang|ja|満州}}」（「{{lang|ja|洲}}」是表外漢字）。其实，替代語也有用于仍未採用的詞語中，诸如「{{lang|ja|叮嚀}}」、「{{lang|ja|意嚮}}」、「{{lang|ja|蹶起}}」、「{{lang|ja|媾和}}」、「{{lang|ja|銓衡}}」、「{{lang|ja|繃帯}}」等。可是，「{{lang|ja|下剋上}}」、「{{lang|ja|古稀}}」、「{{lang|ja|玉石混淆}}」、「{{lang|ja|車輛}}」、「{{lang|ja|漁撈}}」、「{{lang|ja|鈑金}}」之类，本应按照规范的术语，却仍然见于文學作品中。而类似「{{lang|ja|燻製}}」的詞語也仍未按照规范寫法。近年來，「{{lang|ja|障害}}」中的「{{lang|ja|害}}」字受人避讳，某些地方政府并将其转为假名，即「{{lang|ja|障がい}}」。然而，反对假名通假的人（如[[佐賀縣|佐賀縣]]的知事古川康）則提议收录「{{lang|ja|碍}}」字到常用漢字表，并书为「{{lang|ja|障碍}}」。

對此，[[文化審議會|文化審議會]]則否決了提案，原因为「提出者理由中有事實錯誤」。2009年12月，「殘障者制度改革推進本部（{{lang|ja|障がい者制度改革推進本部}}）」和文化審議在內閣府中討論了在公文中采用「{{lang|ja|障碍}}」一词与否，且又提议了收录「{{lang|ja|碍}}」字到《常用漢字》。

《同音汉字书写规则》中有不少字认为是本来的錯誤寫法。此外，虽然许多当用漢字内的熟語以同音替代詞的写法标记，而也有特例，如「{{lang|ja|旱魃}}」→「{{lang|ja|干魃}}」（「{{lang|ja|干ばつ}}」）、「{{lang|ja|稀覯}}」→「{{lang|ja|希覯}}」等轉写，从未在報告中提及过。

== 替换而混同的同音異義詞（范例） ==
* 「{{lang|ja|掩護}}」和「{{lang|ja|援護}}」
: {{lang|ja|掩護}}是在敵人的攻撃下保護隊友的行動，如「{{lang|ja|―射撃}}」
: {{lang|ja|援護}}是救助處於困境的人，如「{{lang|ja|被災者を―する}}（救助受災者）」
* 「{{lang|ja|徽章}}」和「{{lang|ja|記章}}」
: {{lang|ja|徽章}}是表明身分、資格、所屬團體而附在衣服、帽子上的物品。「{{lang|ja|制帽の―}}（學生帽上的徽章）」
: {{lang|ja|記章}}是發予给參加活動的有關人士的紀念物，如「{{lang|ja|従軍―}}」
* 「{{lang|ja|綺談}}」和「{{lang|ja|奇談}}」
: {{lang|ja|綺談}}指有趣又精闢的谈话。
: {{lang|ja|奇談}}指异乎寻常的故事，如「{{lang|ja|珍談―}}」
* 「{{lang|ja|教誨}}」和「{{lang|ja|教戒}}」
: {{lang|ja|教誨}}是受刑者的教育，如「{{lang|ja|―師}}」
: {{lang|ja|教戒（教誡）}}是告誡的意思，在口語中幾乎不用。
* 「{{lang|ja|昏迷}}」和「{{lang|ja|混迷}}」
: {{lang|ja|昏迷}}是迷惘且不知所措的意思。又是醫學术语，指有意識而對外部刺激沒有反應的狀態。實際上第一项意思极其少用，一般僅作為醫學术语。
: {{lang|ja|混迷}}指难以理解、真伪莫辨，如「{{lang|ja|―する政局}}（混亂的政局）」
* 「{{lang|ja|蒐集}}」和「{{lang|ja|収集}}」
: {{lang|ja|蒐集}}指收藏，如「{{lang|ja|切手の―}}（收藏郵票）」
: {{lang|ja|収集}}僅指收集的動作，如「{{lang|ja|ごみの―}}（垃圾收集）」
* 「{{lang|ja|情誼}}」和「{{lang|ja|情義}}」
: {{lang|ja|情誼}}指真心實意，如「{{lang|ja|―に厚い}}（情深誼長）」
: {{lang|ja|情義}}指人情義理，如「{{lang|ja|―を欠く}}（欠人情）」
* 「{{lang|ja|牆壁}}」和「{{lang|ja|障壁}}」
: {{lang|ja|牆壁}}仅指牆壁。「{{lang|ja|防火―}}（防火牆）」
: {{lang|ja|障壁}}指阻碍行動的東西，主要是抽象概念，如「{{lang|ja|関税―}}（關稅壁壘）」
* 「{{lang|ja|棲息}}」和「{{lang|ja|生息}}」
: {{lang|ja|棲息}}指棲息在某種場所。生物學中用於動物。「{{lang|ja|猿の―地}}（猴的棲息地）」
: {{lang|ja|生息}}指生活、生存。生物學中用於植物。「{{lang|ja|都会に―する}}（生活在都市）」「{{lang|ja|高山に―する植物}}（生長在高山上的植物）」
* 「{{lang|ja|洗滌}}」和「{{lang|ja|洗浄}}」
: {{lang|ja|洗滌}}指洗得十分乾淨，又指用水或薬剤等洗滌。「{{lang|ja|洗滌}}」本來的讀音是「{{lang|ja|せんでき}}」，而慣用讀法是「{{lang|ja|せんじょう}}」，具體例子：「{{lang|ja|胃を―する}}（洗胃）」
: {{lang|ja|洗浄}}指佛教中身心的洗濯，少用词。
* 「{{lang|ja|尖端}}」和「{{lang|ja|先端}}」
: {{lang|ja|尖端}}指尖銳物體的尖端，又指時代或流行事物的開始。「{{lang|ja|錐（きり）の―}}（錐尖）」「{{lang|ja|岬の―}}（海角）」「{{lang|ja|時代の―}}（時代的前奏）」「{{lang|ja|流行の―}}（流行前奏）」「{{lang|ja|―技術}}（尖端技術）」
: {{lang|ja|先端}}指长的物体的前端。「{{lang|ja|棒の―}}（棒的前端）」
* 「{{lang|ja|破毀}}」和「{{lang|ja|破棄}}」
: {{lang|ja|破毀}}是法律用語，指高級法院承認上訴成立而取消原判。如「{{lang|ja|原判決を―する}}（取消原判）」
: {{lang|ja|破棄}}指破壞、拋棄。又指契約、協議等的單方面消除「{{lang|ja|書類を―する}}（毀棄資料）」「{{lang|ja|契約を―する}}（取消契約）」
* 「{{lang|ja|哺育}}」和「{{lang|ja|保育}}」
: {{lang|ja|哺育}}指動物哺育下一代。
: {{lang|ja|保育}}指保護、養育嬰幼兒。「{{lang|ja|―園}}」「{{lang|ja|―器}}」「{{lang|ja|―所}}」
* 「{{lang|ja|妨碍}}」和「{{lang|ja|妨害}}」
: {{lang|ja|妨碍}}指阻礙通行<ref name="zh">這兩個詞語在[[漢語|漢語]]中有明確的區別。參見《{{lang|ja|デイリーコンサイス中日辞典}}》（[[三省堂|三省堂]]）的「[http://www.excite.co.jp/dictionary/chinese_japanese/?search=%E5%A6%A8%E7%A2%8D&match=beginswith&id=6554 妨碍]」和「[http://www.excite.co.jp/dictionary/chinese_japanese/?search=%E5%A6%A8%E5%AE%B3&match=beginswith&id=6555 妨害]」的對照。</ref>。
: {{lang|ja|妨害}}指为了阻止对方行動而加害<ref name="zh" />。
* 「{{lang|ja|厖大}}」和「{{lang|ja|膨大}}」
: {{lang|ja|厖大}}是形容詞，指数量或規模大。「{{lang|ja|―な計画}}（宏大的計劃）」「{{lang|ja|―な予算}}（龐大的預算）」
: {{lang|ja|膨大}}是動詞。「{{lang|ja|予算が―する}}（預算增多）」
* 「{{lang|ja|椿事}}」和「{{lang|ja|珍事}}」
: {{lang|ja|椿事}}是意料之外的大事，如「{{lang|ja|春先の―}}（早春的意外事件）」
: {{lang|ja|珍事}}指罕見、异常的往事，如「{{lang|ja|前代未聞の―}}（未曾聽聞的怪事）」

== 替換前已有的异形词 ==
以下詞語在规范前已有異型詞，後來凡詞中有表外漢字，都以原含表內漢字者为准。
* {{lang|ja|饑餓→飢餓}}：「{{lang|ja|饑}}」含「{{lang|ja|飢}}」义。
* {{lang|ja|障碍→障害}}：[[漢字圏|漢字圏]]以内，「{{lang|ja|碍（礙）}}」、「{{lang|ja|害}}」音义不同，故不用「{{lang|ja|障害}}」一詞，「{{lang|ja|障害}}」唯日本独有。
* {{lang|ja|歎願→嘆願}}：「{{lang|ja|歎}}」通「{{lang|ja|嘆}}」。
* {{lang|ja|磨滅→摩滅}}：戰前已通用。

== 對新詞的影響 ==
譬如，「{{lang|ja|姆}}」义为「[[乳母|乳母]]」，本有「{{lang|ja|保母}}」、「{{lang|ja|保姆}}」之分，后以「母」代「姆」，而從事同類職業的男性，则稱為「保父」，从而衍生出了「父」的对立。在日本，现在的职称凡有有阴阳之分，多改取中性者，如「保母」、「保夫」已经統稱為「{{lang|ja|[[褓姆|保育士]]}}」了。

== 《改定常用漢字表》的影響 ==
[[文化審議会|文化審議会]]于2010年6月，在《改定常用漢字表》中追加了196字。其中「{{lang|ja|臆}}、{{lang|ja|潰}}、{{lang|ja|毀}}、{{lang|ja|窟}}、{{lang|ja|腎}}、{{lang|ja|汎}}、{{lang|ja|哺}}、{{lang|ja|闇}}」八字，增至《同音漢字替代詞》。唯「{{lang|ja|闇}}」字特别处理，转写为片假名「{{lang|ja|アン}}」。而余下7字，则规范了与其相关的13個词组。《同音漢字替代詞》之增，仅为配合《当用漢字》而变动。至于当前的《常用漢字》、《改定常用漢字》，均以「目安」为准，即滿足一般的閱讀需求。此外，更避免了不宜同音假借的词组。

* 【{{lang|ja|臆説→憶説}}】「{{lang|ja|臆}}」字的词例有「{{lang|ja|臆説}}」，註釋了〈{{lang|ja|「憶説」とも書く。}}〉一句，译为「又寫作『{{lang|ja|憶説}}』。」
* 【{{lang|ja|臆測→憶測}}】「{{lang|ja|臆}}」字的词例有「{{lang|ja|臆測}}」，註釋了〈{{lang|ja|「憶測」とも書く。}}〉一句，译为「又寫作『{{lang|ja|憶測}}』。」
* 【{{lang|ja|潰滅→壊滅}}】
* 【{{lang|ja|潰乱→壊乱}}】
* 【{{lang|ja|肝腎→肝心}}】「{{lang|ja|腎}}」字的词例有「{{lang|ja|肝腎}}」，註釋了〈{{lang|ja|「肝心」とも書く。}}〉一句，译为「又寫作『{{lang|ja|肝心}}』。」
* 【{{lang|ja|決潰→決壊}}】
* 【{{lang|ja|広汎→広範}}】
* 【{{lang|ja|全潰→全壊}}】
* 【{{lang|ja|倒潰→倒壊}}】
* 【{{lang|ja|破毀→破棄}}】
* 【{{lang|ja|哺育→保育}}】
* 【{{lang|ja|崩潰→崩壊}}】
* 【{{lang|ja|理窟→理屈}}】

其中包括戰後已用的词组，如以上「{{lang|ja|憶説}}」、「{{lang|ja|憶測}}」、「{{lang|ja|広範}}」三詞。而「{{lang|ja|破毀}}·{{lang|ja|破棄}}」、「{{lang|ja|哺育}}·{{lang|ja|保育}}」则是不同的詞语。余下的八詞，早已有之，不是新造的。

== 類似个例 ==
*757年，[[統一新羅|統一新羅]]為了以兩字表記地名而在各地推行改名，并以同音、近音的漢字更换地名。新羅改名仅求美化，而日本則为限制汉字。此外，某些新的地名和以諺文表記的發音也因此相差。

== 對华影響 ==
近代以来，华语借用了大量的和製漢語，包括以代用字書寫的，且此类舶来词都不书以原词形，直接照搬而用。以下为汉语中常见的代用字词形。例如：
*「{{lang|ja|智識}}」→「{{lang|ja|知識}}」→「知識」<ref>在正式公佈代用字以前，部分通俗的代用寫法就已傳入中國。例如用“知識”代替“智識”作爲“knowledge”的譯詞，儘管中文已經有「智識」一詞。</ref>
*「{{lang|ja|崩潰}}」→「{{lang|ja|崩壊}}」→「崩壞」
*「{{lang|ja|辨当}}」→「{{lang|ja|弁当}}」→「便當」
