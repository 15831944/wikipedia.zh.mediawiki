'''同位素列表'''列出了所有已知的[[化学元素|化学元素]]的[[同位素|同位素]]。<ref>The data for these tables came from [[Brookhaven_National_Laboratory|Brookhaven National Laboratory]] which has an interactive [http://www-nds.iaea.org/nudat2/ Table of Nuclides] {{webarchive|url=https://web.archive.org/web/20100309215750/http://www-nds.iaea.org/nudat2/ |date=2010-03-09 }} with data on ~3000 nuclides.</ref>

此表由左到右按照[[原子序|原子序]]数的增长而排列，由下到上依照[[中子|中子]]数目由少到多排列。

表格中的颜色表示各个[[同位素|同位素]]的[[半衰期|半衰期]]（参见图例），表格边缘的颜色表示最稳定的[[核素|核素]]的[[半衰期|半衰期]]。
== 元素週期表 ==
{{Anchor|元素週期表|元素周期表}}這是一個元素週期表，該表列出各元素的穩定同位素個數，若無穩定同位素則列出最穩定的同位素之半衰期。

這個週期表有助於[[#部份表格|選擇所需的表格]]，以便檢視該元素[[同位素|同位素]]的詳細資料。
