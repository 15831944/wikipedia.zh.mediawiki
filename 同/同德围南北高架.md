'''同德围南北高架'''，为[[中国|中国]][[广州市|广州市]]兴建的一条[[高架路|高架路]]，位于[[白云区_(广州市)|白云区]][[同德围|同德围]]，南接[[西湾路|西湾路]]，北接[[德康路|德康路]]。全线设置双向四车道，设计时速40公里/小时。该道路可望减轻[[同德围|同德围]]的交通堵塞情况。

同德围南北高架已于2014年12月28日中午12时正式通车<ref name=sina>{{cite news|title=同德围南北高架本月通车_新浪新闻|url=http://news.sina.com.cn/o/2014-12-17/031931292999.shtml}}</ref>。

== 历史 ==
[[同德围|同德围]]东西面分别被[[石井河|石井河]]和[[京广铁路|京广铁路]]包围，只有[[西湾路|西湾路]]和[[西槎路|西槎路]]贯穿南北，而且加上约三十万的居住人口，交通堵塞问题异常严重。为解决这一问题，[[广州市政府|广州市政府]]决定兴建一条南北走向的[[高架路|高架路]]，以解决居民出行难得问题。<ref name="multiple">{{Cite web |url= http://informationtimes.dayoo.com/html/2012-05/30/content_1718715.htm### |title= 同德围东线高架两方案PK |author= 信息时报 |date= 2012年5月30日 |deadurl= yes |archiveurl= https://web.archive.org/web/20160304090532/http://informationtimes.dayoo.com/html/2012-05/30/content_1718715.htm |archivedate= 2016年3月4日 }}</ref>
在规划初期，曾提出沿[[石井河|石井河]]兴建的'''西线方案'''以及沿[[京广铁路|京广铁路]]兴建的'''东线方案'''，由于西线方案有可能破坏[[石井河|石井河]]的景观，最终采用东线方案。而东线方案又分为'''绕货场方案'''和'''穿越货场方案'''<ref name="multiple" />。最终为避免影响货场运作，选择了'''绕货场方案'''<ref name=sina />。

== 环保措施 ==
由于该道路沿线是密集的居民区，因此采用一系列措施以减低对居民的影响<ref>{{Cite web|url=http://news.xkb.com.cn/guangzhou/2012/0530/203133.html|title=同德围确定建高架桥 出入不再“华山一条路”|author=新快报|date=2012年5月30日}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
*高架桥离居民建筑的一般距离大于10米
*路面采用SMA沥青以降低噪音
*通过居民区等敏感点的路段设隔音屏障
*部分路段为居民安装隔音门窗

== 工程难点 ==
沿线居民及建筑密集，需要大量拆迁，据有关工程人士透露，总拆迁量达到2万平方米，而且某些建筑没有房产证，到时商讨拆迁补偿协议可能存在较大的争议。此外，棠溪货场所在的地块属于[[越秀区|越秀区]][[矿泉街|矿泉街]][[瑶台村|瑶台村]]所有，有村民对高架桥穿过表示担忧，可能会阻挠道路的建设<ref>{{Cite web |url= http://news.gz.soufun.com/2012-06-05/7817588_all.html |title= 同德围高架沿线拆迁量或达2万平米 |author= 南方日报 |date=2012年6月5日  }}</ref>。

2014年9月12日，[[广州环城高速公路|环城高速]]北段与南北高架相交处有两座桥墩发生超出安全范围的倾斜，导致环城高速北段封闭维修，市内同时进行的道路维修引致市内交通堵塞。<ref>{{cite news |title=广州日报 - 2014年9月12日 - A1:头版版 - 歪了俩桥墩 堵了半边城 |url=http://gzdaily.dayoo.com/html/2014-09/12/content_2747036.htm |deadurl=yes |archiveurl=https://web.archive.org/web/20140912125253/http://gzdaily.dayoo.com/html/2014-09/12/content_2747036.htm |archivedate=2014年9月12日 }}</ref>
