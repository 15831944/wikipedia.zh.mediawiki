'''同安縣'''，雅稱'''銀同'''。旧县名。辖域包括现在的[[中華民國政府|中華民國政府]]統治的[[金门县|金门县]]、[[中华人民共和国政府|中华人民共和国政府]]統治的[[厦门市|厦门市]]全境及[[龙海市|龙海市]]东北部。有許多名人[[籍貫|籍貫]]為'''同安縣'''，如[[鄭成功|鄭成功]][[參軍|參軍]][[陳永華_(明朝將領)|陳永華]]、[[江南|江南]][[提督|提督]][[陳化成|陳化成]]、知名華僑企業家[[陳嘉庚|陳嘉庚]]、[[孫中山|孫中山]]的[[如夫人|如夫人]][[陳粹芬|陳粹芬]]等。

== 縣名由來 ==
[[西晉|西晉]][[太康_(西晋)|太康]]三年（282年）時，東南沿海社會秩序仍不夠安定，故當時縣名多寓有「平安」之意，如[[南安|南安]]、[[惠安|惠安]]等，而同安一地因古稱「大同」，境內有座梅山，古名「同山」，遂將「同」字與「安」字結合，設同安縣{{r|郭}}。

而「銀同」之稱則來自[[南宋|南宋]][[紹興_(南宋)|紹興]]十五年（1145年）所建於大輪山下的同安城，該城東西長，南北窄，形似[[銀錠|銀錠]]，遂名銀同；該城又有「銅魚城」之稱，其名來自城南東溪溪底有三塊因長年溪水浸染成古銅色的魚形巨石{{r|郭}}。

== 沿革 ==
[[西晋|西晋]][[太康_(西晋)|太康]]三年（282年），析分[[建安郡|建安郡]]地設[[晉安郡|晉安郡]]（約今[[南安縣|南安縣]]），同年又分晉安郡地置同安縣，但不久又裁撤併回晉安郡{{r|郭}}。唐貞元十九年（803年），析南安縣西南四鄉設置大同場，粗定後來的同安縣域{{r|郭}}。[[五代|五代]][[后唐|后唐]][[长兴_(李嗣源)|长兴]]四年（933年），閩王[[王延鈞|王延鈞]]稱帝建[[閩_(十國)|閩國]]將大同場升为[[同安县|同安县]]，此後沿設{{r|郭}}。[[宋朝|宋]]属[[清源军|清源军]]、平海军、泉州。[[元朝|元]]属[[泉州路|泉州路]]，[[明清|明清]]時，屬於[[泉州府|泉州府]]。[[清兵入關|清兵入關]]之後，曾屠城四日，同安縣軍民死難五萬多人。同安縣[[教諭|教諭]]陳鼎[[自縊|自縊]]殉國，其子[[陳永華_(明朝將領)|陳永華]]即是[[金庸|金庸]]小說筆下「平生不見陳近南，便稱英雄也枉然！」[[陳近南|陳近南]]的原型。

1912年，[[中華民國歷史#建國之初|中華民國肇建]]，[[清朝滅亡|清朝滅亡]]。[[民國|民國]]三年（1914年），[[金門|金門]]、[[廈門島|廈門]]兩地，自同安縣獨立，設思明縣。民國四年（1915年）金門獨立設縣，管轄[[金門島|大]]、[[小金門|小金門]]，[[大嶝|大嶝]]、小嶝及周邊島嶼，是為[[金門縣|金門縣]]。民国时，省县直辖，同安县属福建省第四行政督察区。

1949年[[國共戰爭|國共戰爭]]後，[[中國大陸|中國大陸]]易主，'''同安縣'''及原属金门的大嶝乡由[[中华人民共和国政府|中华人民共和国政府]]統治，[[中華民國政府|中華民國政府]]持續統治金門县除大嶝乡外剩余的大部分地区。

1997年5月，廢'''同安縣'''，全縣劃入[[廈門市|廈門市]]，改為[[廈門市|廈門市]][[同安區|同安區]]。2003年9月，同安區的馬巷鎮、新店鎮、新圩鎮等組成[[翔安區|翔安區]]，是以昔日同安縣'''翔風里'''與'''民安里'''為核心而成的區{{r|郭}}。

== 行政區划 ==
宋朝時，同安縣下有三鄉、三十里，後整合為三鄉、十一里，元朝依此改為四十四都，明朝則增置「在坊里」而都不變，直到清乾隆四年（1775年）析置[[馬巷廳|馬巷廳]]為止，同安縣的都里基本上皆沿此置{{r|郭}}。
