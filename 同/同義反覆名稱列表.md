'''同義反覆名稱'''指的是一個名稱的兩個部分以上表達了相同的意思，一般是在翻譯過程或古名傳承中產生，一般是一個普遍名稱專指一個特定事物後，再由其他語言承接後添加該語言中這個普遍名稱的詞綴而成，例如[[撒哈拉沙漠|撒哈拉沙漠]]是阿拉伯語的沙漠一詞和漢語的沙漠一詞組合而成。
== 自然 ==
* [[撒哈拉沙漠|撒哈拉沙漠]]，位於[[非洲|非洲]]的[[沙漠|沙漠]]，「撒哈拉（{{lang-ar|الصحراء}}）」是[[阿拉伯語|阿拉伯語]]的沙漠一詞。
* [[雅芳河|雅芳河]]，位於[[英國|英國]]的多條[[河流|河流]]，「雅芳（{{lang-en|Avon}}）」是[[古布立吞語|古布立吞語]]的河流一詞。
* [[襟裳岬|襟裳岬]]，位於[[日本|日本]]北海道的[[海岬|海岬]]，「襟裳（{{jpn|j=襟裳|hg=えりも|rm=erimo}}）」是[[日語|日語]]轉譯[[阿伊努語|阿伊努語]]的海岬一詞（{{lang-ain|エンルム|enrum}}）。
* [[印度河|印度河]]，位於[[巴基斯坦|巴基斯坦]]的[[河流|河流]]，「印度（{{lang-la|Indus}}）」是[[拉丁語|拉丁語]]轉譯[[梵語|梵語]]的河流一詞（{{lang-sa|सिंधु|translit-std=IAST|translit=síndhu}}）。
* [[湄公河|湄公河]]，位於[[亞洲|亞洲]]的河流，「湄公（{{lang-vi|Mê Kông}}、{{lang-km|ទន្លេមេគង្គ}}、{{lang-my|မဲခေါင်မြစ်}}）」是各語言轉譯[[泰語|泰語]]、[[老撾語|老撾語]]的湄公河一詞（{{lang-th|แม่น้ำโขง}}、{{lang-lo|ແມ່ນ້ຳຂອງ}}），其中前兩個音節是大河的意思，來源於「母親河」，而後一個一音節則專指湄公河，可能與上古漢語的「江」同源，所以「湄公河」即相當於「河江河」。
* [[湄南河|湄南河]]，即[[昭拍耶河|昭拍耶河]]，「湄南」在[[泰語|泰語]]中指大河，來源於「母親河」。
* [[阿穆爾河|阿穆爾河]]，即[[黑龍江|黑龍江]]，「阿穆爾（{{lang-ru|Амур}}）」是[[俄語|俄語]]轉譯[[通古斯語族|通古斯諸語]]的河流一詞。

== 民族 ==
* [[阿伊努人|阿伊努人]]，位於[[日本|日本]]北海道等地的一個民族，「阿伊努（{{lang-ain|アイヌ|ainu}}）」是[[阿伊努語|阿伊努語]]的人一詞。

== 人文 ==
* [[什葉派|什葉派]]，[[伊斯蘭教|伊斯蘭教]]的一支，「什葉（{{lang-ar|شيعة}}）」是[[阿拉伯語|阿拉伯語]]的追隨者、黨派一詞。
* [[布爾什維克黨|布爾什維克黨]]，[[蘇聯|蘇聯]]的黨派，「布爾什維克（{{lang-ru|большевик}}）」是[[俄語|俄語]]的多數派一詞。
