'''同治新疆回变'''（1864年－1877年），是[[清朝|清朝]][[同治|同治]]年間[[新疆|新疆]]各地發生的叛亂，與[[浩罕汗國|浩罕]]及[[俄羅斯|俄羅斯]]的入侵交織在一起。

清同治三年（1864年）四月，[[庫車|庫車]]回民舉事，殺死庫車辦事大臣及[[阿奇木伯克|阿奇木伯克]]。因為[[陝甘回亂|陝甘回亂]]影響，數月之內，新疆各地相繼發生暴動和叛亂。暴動者以[[回族|回族]]（漢回）及[[維吾爾族|維吾爾族]]（纏回）阿訇、農民為主，也有[[漢族|漢族]]、[[柯爾克孜族|柯爾克孜族]]及[[哈薩克族|哈薩克族]]加入。各地起事者在攻佔官署及兵營後，推舉[[阿訇|阿訇]]為首領，建立政教合一的地方性割據政權。隨後，各政權之間又發生混戰與兼併。至同治四年（1865年）末，新疆大部份地方失陷，官軍僅保有[[塔爾巴哈臺|塔爾巴哈臺]]、[[庫爾喀喇烏蘇|庫爾喀喇烏蘇]]、[[巴里坤|巴里坤]]及[[哈密|哈密]]一帶。此時，西北官軍主力正在平定陝甘回亂，無暇顧及新疆。[[俄羅斯|俄羅斯]]乘亂入侵[[塔爾巴哈臺|塔爾巴哈臺]]及[[額爾齊斯河|額爾齊斯河]]流域，於1864年10月逼迫[[烏里雅蘇臺將軍|烏里雅蘇臺將軍]]簽訂《[[中俄勘分西北界約記|中俄勘分西北界約記]]》，吞併了中國[[巴爾喀什湖|巴爾喀什湖]]以東的大片領土。

1864年年底，[[喀什噶爾|喀什噶爾]]的叛乱者向[[浩罕汗國|浩罕汗國]]求援，導致[[阿古柏事件|阿古柏入侵]]。1865年至1867年間，浩罕權臣[[阿古柏|阿古柏]]率領浩罕軍隊消滅了[[天山南路|天山以南]]各地的割据政權，在新疆南部建立了[[哲德沙爾汗國|哲德沙爾汗國]]。同治十年（1871年），俄羅斯藉口“代收”，派兵侵佔[[伊犁|伊犁]]。[[光绪|光绪]]元年（1875年），清廷命[[陝甘總督|陝甘總督]][[左宗棠|左宗棠]]督辦新疆軍務。次年，清軍西征，收復[[迪化|迪化]]一帶。光绪三年（1877年），清軍消滅阿古柏政權，收復了除伊犁外的新疆全境，持續10餘年的新疆叛亂最終結束。光绪七年（1881年），清朝通過交涉收回被俄羅斯侵佔的伊犁。光绪十年（1884年），清廷下旨設立[[新疆省|新疆省]]，廢除了[[乾隆|乾隆]]以來的[[伯克制|伯克制]]和[[軍府制|軍府制]]，實行與內地相同的[[郡縣制|郡縣制]]。

此次動亂大致分為以下幾個階段：
# 新疆各地發生叛動，[[駐紮大臣|駐紮大臣]]等官員被殺或者[[自盡|自盡]]，清兵潰敗。
# 各地起事者建立多個互不統屬的割據政權，各政權之間相互混戰。
# 俄國軍隊入侵[[塔城|塔城]]等地，清廷被迫與之簽訂《[[中俄勘分西北界約記|中俄勘分西北界約記]]》。
# 浩罕軍官阿古柏攜[[白山派|白山派]][[和卓|和卓]]後裔入侵新疆，消滅[[天山南路|天山南路]]的清軍據點。
# 阿古柏逐一消滅各地割据政權，驅逐[[和卓|和卓]]後裔，建立哲德沙爾國。
# 清軍西征，收復除[[伊犁|伊犁]]以外的新疆全境，並進駐帕米爾高原中部。

== 背景 ==
同治初年，因[[太平天國|太平天國]]、[[捻軍|捻軍]]等持續多年的內亂及向西方列強賠款，清廷面臨嚴重的財政危機。每年由內地各省撥解新疆的二三百萬[[協餉|協餉]]逐漸縮減，最終斷絕。大小[[伯克制|伯克]]官吏任意攤派勒索，致使民怨沸騰。

=== 庫車 ===
同治三年（1864年）四月，修建[[渭幹河|渭幹河]]水利的庫車河工托乎提尼牙孜哈里等人殺死監工和伯克，引發暴動。同月，庫車土回{{NoteTag|清朝所谓“土回”，指居住在[[新疆|新疆]]的[[回族|回族]]。}}馬漋、馬三保、[[玉門縣|玉門]]人楊春，聯絡外地人田拉滿、蘇拉滿，聚集當地[[漢回|漢回]]（[[回族|回族]]）、漢民、纏回（維吾爾族）起事。庫車辦事大臣[[薩凌阿|薩凌阿]]諮報[[喀喇沙爾|喀喇沙爾]]（今[[焉耆縣|焉耆縣]]）求援。五月初三，馬漋部眾攻入庫車城內，殺死庫車辦事大臣薩凌阿、英吉沙爾領隊大臣[[文藝_(清朝)|文藝]]、葉爾羌幫辦大臣[[武仁布|武仁布]]、庫車[[阿奇木伯克|阿奇木伯克]]庫爾班等人，逼迫曾任喀什噶爾阿奇木伯克的庫車閒散郡王[[愛瑪特|愛瑪特]]（艾合買提王伯克）充當暴動首領。愛瑪特為庫車伯克[[鄂對|鄂對]]之後，不肯屈從，被當場殺死。<ref name="戡定新疆記1">《戡定新疆記》武功記一</ref>

暴動者找到庫車熱西丁家族的[[阿訇|阿訇]][[熱西丁和卓|熱西丁和卓]]{{NoteTag|[[熱西丁和卓|熱西丁和卓]]的祖先[[熱西丁|熱西丁]]曾勸說東察合台汗[[禿忽魯帖木兒|禿黑魯帖木兒]]皈依伊斯蘭教，見[[海答兒|海答兒]]《[[拉失德史|拉失德史]]》。此時的熱西丁和卓為其祖先[[麻扎|麻扎]]的看守人。}}，推舉其為首領。熱西丁自稱“先知穆罕默德最偉大的後裔、宇宙力量的主宰者”<ref>[[毛拉|毛拉]][[穆萨·赛拉米|穆萨·赛拉米]]《[[伊米德史|伊米德史]]》</ref>，人稱“汗和卓”（清代文獻譌作“黃和卓”），煽動民眾對異教徒（[[卡菲爾|卡菲爾]]）進行“[[聖戰|聖戰]]”。熱西丁派兵向西攻佔了阿克蘇和[[烏什|烏什]]，向東襲取[[喀喇沙爾|喀喇沙爾]]，基本控制了[[回部|回部]]東四城一帶。

=== 阿克蘇、喀喇沙爾 ===
五月初七，庫車以東的[[玉古爾|玉古爾]]（今[[輪臺縣|輪臺縣]]）被暴動者攻陷。初九日至十四日，[[阿克蘇|阿克蘇]]所屬的[[賽喇木|賽喇木]]、[[拜城|拜城]]，[[喀喇沙爾|喀喇沙爾]]及其屬城[[庫隴勒|庫隴勒]]（今[[庫爾勒市|庫爾勒市]]）也爆發民變，喀喇沙爾辦事大臣[[依奇哩|依奇哩]]死難。阿克蘇辦事大臣[[富蘇哩|富蘇哩]]馳書伊犁求援，未幾阿克蘇城陷，富蘇哩舉家自焚<ref name="戡定新疆記1" />。[[回部|回部]]東四城陷落後，為庫車的熱西丁所控制。

=== 烏魯木齊 ===
早在同治元年（1862年），[[甘肅|甘肅]][[河州|河州]]（今[[臨夏回族自治州|臨夏回族自治州]]）[[阿訇|阿訇]][[妥得璘|妥得璘]]（一名妥明）潛逃出關，來到烏魯木齊，寄居於烏魯木齊[[綠營|綠營]]署理中營提標參將[[索煥章|索煥章]]{{NoteTag|[[索煥章|索煥章]]為[[甘肅提督|甘肅提督]][[索文|索文]]之子。}}家中。索煥章師事之，一同謀劃起事。

同治三年五月，[[烏魯木齊都統|烏魯木齊都統]][[平瑞|平瑞]]得知庫車發生暴動，派文光領兵前去平叛，途中被亂軍擊潰於烏沙塔巴克，全軍覆沒<ref>《清史稿》穆宗本紀一</ref>。索煥章屬下的漢回兵丁夜間在烏魯木齊南關[[清真寺|清真寺]]集會議事，被戍卒朱小貴窺見。朱小貴報告烏魯木齊綠營提督[[業布沖額|業布沖額]]，業布沖額詢問索煥章。索煥章誣稱朱小貴造謠惑眾，將其[[斬首|斬首]]。六月十二日，妥得璘、索煥章起兵反叛，攻下烏魯木齊漢城（迪化城）。業布沖額被俘，囚禁於索煥章宅中，八日後仰藥自盡。都統平瑞與少量[[屯練|屯練]]、[[勇營|勇營]]困守[[鞏寧城|鞏寧城]]（烏魯木齊滿城）。同日，[[昌吉縣|昌吉縣]]事起，知縣[[恩錦|恩錦]]逃走，典史秦某闔家就戮。<ref name="戡定新疆記1" />

困於鞏寧城內的清兵多為老弱，不習操練，平瑞惟有等待援兵。然而“自六月至八月，無一至者。”<ref>[[楊毓秀|楊毓秀]]《[[平回志|平回志]]》卷七</ref>[[伊犁將軍|伊犁將軍]][[常清|常清]]、[[烏里雅蘇臺將軍|烏里雅蘇臺將軍]][[明誼|明誼]]派兵救援，皆被阻於烏魯木齊以北。平瑞勉強督兵出戰，清軍一觸即潰，死五百餘人。八、九月間，[[阜康|阜康]]、[[呼圖壁|呼圖壁]]、[[綏來縣|綏來]]、烏蘭烏蘇等地失陷。妥得璘自稱“清真王”（[[哈里發|哈里發]]），控制了[[庫爾喀喇烏蘇|庫爾喀喇烏蘇]]（今[[烏蘇市|烏蘇市]]）以東、[[濟木薩|濟木薩]]（今[[吉木薩爾縣|吉木薩爾縣]]）以西一帶地方。九月辛丑，在困守百餘日後，鞏寧城破，平瑞自焚，護理[[鎮迪道|鎮迪道]][[伊昌阿|伊昌阿]]投井死，協領榮慶以下死難者一百餘人。

== 阿古柏入侵 ==
喀什噶爾的[[思的克|思的克]]、[[金相印|金相印]]因無力攻克清軍駐守的[[恢武城|喀什噶爾漢城]]（今[[疏勒縣|疏勒縣]]），於1864年底派人邀請流亡[[浩罕汗國|浩罕汗國]]的[[白山派|白山派]][[和卓|和卓]]後裔入境。浩罕執政者隨即派[[阿古柏|阿古柏]]領兵攜[[波羅尼都|波羅尼都]]曾孫[[布素魯克|布素魯克]]入侵新疆。1865年9月至1870年間，阿古柏率領浩罕[[烏茲別克族|烏茲別克族]]軍隊相繼攻佔了喀什噶爾漢城、葉爾羌、和闐、[[阿克蘇|阿克蘇]]、庫車、[[庫爾勒|庫爾勒]]等地，將[[天山南路|天山南路]]各地的暴動首領全部消滅，最後控制了[[迪化|迪化]]，宣佈建立[[哲德沙爾汗國|哲德沙爾汗國]]。至清軍由甘肅平定時，阿古柏及其浩罕軍隊佔據新疆南部達十三年之久。

== 俄羅斯侵佔伊犁 ==
{{Main|伊犁危機}}

同治十年（1871年），俄國以防備阿古柏襲擾為由，派兵佔領了伊犁一帶，聲稱“代為收復”，待中國軍隊肅清內亂時歸還，實則圖謀永久割佔。

== 清軍收復新疆 ==
{{Main|清軍收復新疆之戰}}

同治十二年（1873年），清軍基本平息了陝西、甘肅的叛亂，籌備恢復新疆。[[光緒|光緒]]元年（1875年），清廷以[[陝甘總督|陝甘總督]][[左宗棠|左宗棠]]為欽差大臣，督辦新疆軍務。次年，清軍收復迪化、昌吉、[[綏來縣|綏來]]（今[[瑪納斯縣|瑪納斯縣]]）。光緒三年，[[阿古柏|阿古柏]]服毒自盡，清軍收復天山南麓。
