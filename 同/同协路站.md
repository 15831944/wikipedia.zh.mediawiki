'''同协路站'''是杭州地铁3号线的中途站，位于同协路与杭玻街交叉口北侧。预计2020年4月30日完工<ref>{{cite web |title=杭州地铁3号线同协路站在全线率先完成交改 |url=http://www.hzsz.com/news_detail.php?sid=1&fid=291&id=1302&h=3 |website=杭州市政集团}}</ref>

== 结构 ==
同协路站为地下二层岛式站台，沿同协路南北向布置，该车站为地下两层双柱三跨箱型框架结构，长207.77米，标准段宽21.7米、挖深16.4米；盾构井段宽25.6米、挖深18.3米；采用800毫米厚度的地下连续墙做围护结构（分别长35米、40米）；覆土厚度约3米；采用明挖顺作法施工<ref>{{cite web |title=杭州地铁3号线一期工程土建施工SG3-10标同协路站主体结构基坑开挖 |url=http://www.hzsz.com/news_detail.php?sid=1&fid=291&id=1361&h=3 |website=杭州市政集团}}</ref>。有效站台宽度12.6米，总建筑面积13004平方米。<ref>{{cite web |title=杭州地铁3号线一期走向和站点敲定 看看是否经过你家-浙江新闻-浙江在线 |url=http://zjnews.zjol.com.cn/zjnews/hznews/201801/t20180112_6314347.shtml |website=浙江在线}}</ref>
