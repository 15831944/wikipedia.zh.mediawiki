《'''同級生2'''》是[[élf|élf]]在1995年1月31日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]。<ref>{{cite web|url=http://www.getchu.com/soft.phtml?id=334|title=同級生2|publisher=Getchu.com|accessdate=2015年10月21日|language=ja}}</ref><ref>[https://vndb.org/v2337 Doukyuusei 2]The Visual Novel Database{{en}}</ref>Windows中文版由[[歡樂盒|歡樂盒]]代理，是將DOS版（一般將DOS/V版與[[PC-9801|PC-9801]]版統稱DOS版）加了中文[[配音|配音]]而成的版本（色盤只有[[16色|16色]]，即可證明），一般版售價[[新台幣|新台幣]]680元，附贈「同級生合唱團」一張[[音樂專輯|音樂專輯]][[CDDA|CD]]；但Windows中文版評價極差，附贈的CD更成為同級生合唱團唯一的專輯。Windows版和[[SEGA_Saturn|SEGA Saturn]]版有[[資料片|資料片]]《卒業生》。

== 人物介紹 ==
=== 主要角色 ===
; 龍之介（遊戲預設名字）、近藤誠（OVA）、沖田等（廣播劇）
: 本作的男主角，八十八學園3年B班的學生。八十八學園創校以來有名的問題學生。
; 鳴澤唯
:美佐子的女兒，8歲時就跟隨母親和龍之介同居。
; 水野友美
:住在龍之介家隔壁的[[青梅竹馬|青梅竹馬]]和同學。
; 舞島可憐
:龍之介的同學，8歲就進入演藝圈的偶像。
; 篠原いずみ
:篠原重工社長的女兒，龍之介的同學，弓道社的社長。
; 加藤みのり
:龍之介的同學。
; 南川洋子
:龍之介的同學，機車店的獨生女。
; 杉本櫻子
:因為罹患不明的疾病而在3年前就長期住院的少女，原本應該是和男主角同校同年級的學生。
; 鳴澤美佐子
:唯的母親，龍之介的繼母。
; 都築こずえ
:八十八學園一年級學生，圖書委員和網球社的社員。
; 片桐美鈴
:八十八學園的日文老師，龍之介的班級老師。
; 野野村美里
:入梅興業巴士公司的[[車掌小姐|車掌小姐]]。
; [[田中美沙|田中美沙]]
: 《[[同級生_(遊戲)|同級生]]》的女主角之一，本作登場時是體育大學二年級學生。
; 安田愛美
: 聖花園幼稚園的保母。
; 永島佐知子
: 永島旅館第5代老闆娘和寡婦。
; 永島久美子
: 佐知子的女兒，很嚮往都市的生活。
; 西御寺静乃
: 西御寺有友的同父異母妹妹，國中三年級學生，PS版的新角色。
; 山本律子
: 在可憐所屬事務所的搖滾樂手，PS版的新角色。
; 南雲麻耶
: 如月神社的神主孫女，SFC版的新角色。

=== 其他角色 ===
; 西御寺有友
:西御寺財閥的少爺。
; 長岡芳樹
:攝影社的社長，沒有社員。
; 川尻あきら
:男主角的同學和朋友，柔道社的社員。
; 天道新幹線
:八十八學園的體育老師和柔道社顧問。對愛美一見鍾情而展開追求。

== OVA ==
=== 同級生2 ===
《同級生2》[[OVA|OVA]]版是在1996年3月22日開始發售，原本預定出9集，因為銷量還不錯而再繼續出10～12集。<ref>{{cite web|url=http://www.animenewsnetwork.com/encyclopedia/anime.php?id=2183|title=Dōkyūsei 2 (OAV)|publisher=[[Anime_News_Network|Anime News Network]]|accessdate=2015年10月21日|language=en}}</ref>OVA後來改編成[[電視動畫|電視動畫]]，在1998年7月2日到8月27日播放，全部共9集。<ref>[http://www.tsundere.com/history/1998.html テレビアニメ年表 1998年～1999年]アニメワショーイ{{ja}}</ref>2013年6月28日發售合集《'''同級生2 DVD Perfect Collection'''》，收錄《同級生2》和《同級生2 Special 卒業生》。<ref>[http://www.getchu.com/soft.phtml?id=770161 同級生2 DVD Perfect Collection]Getchu.com{{ja}}</ref>
