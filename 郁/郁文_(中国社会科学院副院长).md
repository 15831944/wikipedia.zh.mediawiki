'''郁文'''（{{bd|1918年|12月|2010年|4月1日}}）原名'''孙保安'''，[[河北|河北]][[满城|满城]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员。[[中共中央宣传部|中共中央宣传部]]常务副部长，[[中国社会科学院|中国社会科学院]][[党组|党组]]书记、副院长<ref name=ccp>{{Cite web |url =  http://dangshi.people.com.cn/GB/165617/165618/166491/167909/11968339.html|title =  郁文|author =  |date =  |publisher = 人民网 |language =  |accessdate = 2017-10-12 |quote =  }}</ref>。

==生平==
1936年加入[[中华民族解放先锋队|中华民族解放先锋队]]。1937年2月加入[[中国共产党|中国共产党]]，历任《新中华报》、《解放日报》采访通讯部主任，《晋绥日报》采访通讯部主任兼[[新华社|新华社]]晋绥总分社社长，中共[[西安市|西安市]]委宣传部副部长。[[中华人民共和国|中华人民共和国]]成立后，历任[[新疆日报|新疆日报]]社社长，[[新疆省_(中华人民共和国)|新疆省]]文教委员会副主任委员，[[中共中央新疆分局|中共中央新疆分局]]宣传部副部长<ref name=ccp/><ref name=zhiyi/>。

1952年底，调到[[中国科学院|中国科学院]]，历任干部局局长、副秘书长、秘书长<ref name=ccp/>。1955年初，任中国科学院干部局局长兼机关党总支书记。1955年底，中国科学院机关党总支改为中国科学院机关党委，郁文任机关党委书记。1965年5月，经[[中共中央|中共中央]]批准，中国科学院党组改为中国科学院党委，并成立政治部，郁文任院党委委员、政治部主任。1979年7月，[[方毅|方毅]]任中国科学院院长、院党组书记，[[李昌|李昌]]、[[胡克实|胡克实]]、郁文任院党组副书记<ref name=chutan>{{Cite web |url =http://yyh.cas.cn/kyws/201412/t20141205_4269657.html  |title = 中国科学院党组织的历史沿革初探 |author =  |date =2014-12-05  |publisher =中国科学院院友服务网  |language =  |accessdate =  |quote =  }}</ref><ref name=ccp/>。1958年9月至1963年4月，郁文任[[中国科学技术大学|中国科学技术大学]]首任党委书记<ref name=zhiyi>{{Cite web |url =http://news.ustc.edu.cn/kdb/201004/t20100409_62470.html  |title =我校创始人之一、首任党委书记郁文同志逝世  |author =  |date =  |publisher = 中国科大报第656期  |language =  |accessdate =  |quote =  }}</ref>。

1982年，任[[中共中央宣传部|中共中央宣传部]]常务副部长兼[[中央宣传思想工作领导小组|中央对外宣传小组]]副组长。1989年，任[[中国社会科学院|中国社会科学院]]党组书记、副院长<ref name=ccp/>。

郁文是第六届、第七届[[全国人大常委会|全国人大常委会]]委员、民族委员会副主任委员，[[中共十二大|中共十二大]]、[[中共十四大|中共十四大]]代表<ref name=ccp/>。

2010年4月1日，郁文在北京病逝，享年92岁<ref name=ccp/>。
