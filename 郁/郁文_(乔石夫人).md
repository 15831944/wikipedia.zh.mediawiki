'''郁文'''（{{bd|1926年|10月|2013年|1月28日}}）女，原名'''翁郁文'''，投身革命后简化作'''郁文'''，曾用名'''凌澜'''、'''林兰'''，[[中國|中國]][[浙江|浙江]][[慈溪|慈溪]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[乔石|乔石]]夫人<ref name=shishi>{{Cite web |url =http://cpc.people.com.cn/n/2013/0207/c87393-20463732.html  |title =郁文同志逝世 享年87岁  |author =  |date =2013-02-07  |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1926年10月生于浙江慈溪。1944年11月，在浙东抗日根据地鲁迅学院参加[[新四军|新四军]]，同年12月加入[[中国共产党|中国共产党]]。[[抗日战争|抗日战争]]胜利后，在[[上海|上海]][[之江大学|之江大学]]参加组织学生运动，担任中共上海地下党主办的上海《联合晚报》记者和《青年知识》杂志特约编辑、撰稿人。1947年秋，任[[国统区|国统区]][[全国学联|全国学联]][[党组|党组]]成员<ref name=shishi/>。

[[中华人民共和国|中华人民共和国]]成立后，历任中央青委、[[共青团中央|团中央]]办公室秘书，中共[[杭州市|杭州市]]委青委宣传部部长，[[中共中央华东局|中共中央华东局]]青委秘书室副主任、宣传部科长。1955年4月起，任[[鞍山钢铁公司|鞍山钢铁建设公司]]党委办公室调研科科长、办公室副主任，[[酒泉钢铁公司|酒泉钢铁公司]]党委副秘书长兼调研室主任。1963年4月，到[[中共中央对外联络部|中共中央对外联络部]]工作，历任编译室研究员，研究组（室）综合组副组长，研究室副主任、主任。1986年12月起，任[[中国国际交流协会|中国国际交流协会]]副总干事、总干事、副会长。1999年7月[[离休|离休]]<ref name=shishi/>。

郁文是第七届、第八届[[全国政协委员|全国政协委员]]及外事委员会委员<ref name=shishi/>。

2013年1月28日，郁文在[[北京|北京]]病逝，享年87岁<ref name=shishi/>。

==家庭==
*父：翁祖望，出身于浙江书香门第，是当地有名的学者，曾任陈布雷的机要秘书，陈布雷将其五妹陈若希嫁给翁祖望<ref name=fumu>{{Cite web |url = http://www.tuanjiebao.com/lishi/2015-06/23/content_19235_3.htm |title = 乔石和他的平民父母 |author =  |date = 2015-06-23 |publisher = 团结网 |language =  |accessdate =  |quote =  }}</ref>。
*母：陈若希，出身于江南“官桥陈氏”（今[[余姚|余姚]]官桥村），是[[蔣介石|蔣介石]]心腹大秘[[陳布雷|陳布雷]]的五妹<ref name=fumu/>。
*舅：[[陈布雷|陈布雷]]<ref name=fumu/>
*堂舅：[[陈屺怀|陈屺怀]]，陈布雷的堂兄，早年随[[孙中山|孙中山]]从事革命<ref name=fumu/>。
*兄弟姐妹：郁文有两个哥哥、一个姐姐和一个妹妹。大哥翁泽水在[[抗日战争|抗日战争]]时期担任[[郭沫若|郭沫若]]的秘书，在[[周恩来|周恩来]]和[[陈布雷|陈布雷]]间传话<ref name=fumu/>。
*丈夫：[[乔石|乔石]]：1924年12月生，浙江定海人。郁文在《联合晚报》任职时，与同事乔石相识相知，后来又一起从事学联工作。1952年冬，二人在[[杭州|杭州]]结婚。因为乔石是陈布雷的外甥女婿，所以当时乔石和妻子郁文都未受重用<ref name=fumu/>。
*子女：乔石、翁郁文夫妇育有两子两女。小女儿大学毕业于[[北京医学院|北京医学院]]，硕士毕业于[[协和医科大学|协和医科大学]]。留学[[美国|美国]]后，从[[休斯顿|休斯顿]]的BAYLOR医学院获得博士学位（1993年），在[[南加州大学|南加州大学]]做过[[博士后|博士后]]。后来是一所美国正规研究型大学的助理教授。
