'''碳酸锂'''（Li<sub>2</sub>CO<sub>3</sub>）是一種无色至白色结晶，能溶于[[水分子|水]]，但溶解性不佳。用于治疗精神疾病，多用於[[躁狂症|躁狂症]]的治療。

== 性质 ==
白色[[单斜晶系|单斜]]，鹼性晶体，无[[潮解|潮解]]性，在空气中是稳定的。在水中的溶解度很小，因此可以通过用[[碳酸根|碳酸根]]离子沉淀溶液中锂离子的方法来制备。溶解度随温度的升高而降低。从溶液中析出的碳酸锂不含结晶水。由于锂离子具有很强的极化性，因此碳酸锂的热稳定性比其他碱金属碳酸盐要差一些，加热到熔点以上时，便会发生分解，产生[[氧化锂|氧化锂]]和[[二氧化碳|二氧化碳]]。

: <math>\ \mbox{Li}_2\mbox{CO}_3\longrightarrow \mbox{Li}_2 \mbox{O}+\mbox{C} \mbox{O}_2\uparrow</math>

将二氧化碳通入碳酸锂的水悬浮液中时，碳酸锂即转变为[[酸式碳酸锂|酸式碳酸锂]]而溶解。如果再将酸式碳酸锂的溶液加热，则又放出二氧化碳，并沉淀出碳酸锂。碳酸锂的这个性质可用于除去其中的杂质。

== 历史 ==
碳酸锂在[[精神科|精神科]]临床的应用其实相当偶然。1940年初，[[氯化锂|氯化锂]]曾代替[[氯化钠|氯化钠]]用于[[高血压|高血压]]和[[心脏病|心脏病]]患者，以降低钠离子的摄入量；后因其毒性作用而于1949年放弃使用。1949年，澳洲医生约翰·凯德（John Cade）在研究[[躁郁症|躁郁症]]的成因时估计它与[[尿酸|尿酸]]有关，不过由于尿酸在水中的溶解性很小，难以进行动物实验，而[[尿酸锂|尿酸锂]]是最易溶于水的尿酸盐，于是他就用尿酸锂来进行实验。结果发现，接受过尿酸锂的动物都变得十分温驯和安静，攻击行为得到控制。经过进一步研究，他发现尿酸锂中的锂离子才是关键所在，于是他便进一步进行实验，并在期刊上发表文章《锂治疗精神兴奋》，正式将锂引入精神科临床用于躁狂症的治疗。

== 用途 ==
工业上碳酸锂用于制造其他锂盐，如[[氯化锂|氯化锂]]和[[溴化锂|溴化锂]]等。它还在搪瓷、玻璃、陶瓷和瓷釉中作为[[氧化锂|氧化锂]]的原料，也被加到电解制铝的电解槽中，用以增加电流效率和降低电解槽的内阻和槽温。在医学上，则用于[[躁狂症|躁狂症]]的治疗。
