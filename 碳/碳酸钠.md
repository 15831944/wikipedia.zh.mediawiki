'''碳酸钠'''（{{Lang-en|'''sodium carbonate'''}}），俗名'''苏打'''(soda)、'''纯碱'''(soda ash 、soda crystals)、'''洗滌鹼'''(washing soda)，生活中亦常称“'''碱'''”。[[化学式|化学式]]：Na<sub>2</sub>CO<sub>3</sub>，普通情况下为白色粉末，为强[[电解质|电解质]]。密度为2.532g/cm<sup>3</sup>，熔点为850℃，易溶于水，具有[[鹽_(化學)|鹽]]的通性。吃起來有苦和淡淡的澀味

== 製取方法 ==
=== 侯氏制碱法 ===
{{main|氨碱法}}反应分三個步驟进行：

# NH<sub>3</sub> + CO<sub>2</sub> + H<sub>2</sub>O → NH<sub>4</sub>HCO<sub>3</sub>
# NH<sub>4</sub>HCO<sub>3</sub> + NaCl → NaHCO<sub>3</sub> + NH<sub>4</sub>Cl
# 2NaHCO<sub>3</sub> → Na<sub>2</sub>CO<sub>3</sub> + CO<sub>2</sub> + H<sub>2</sub>O

反应生成的二氧化碳（CO<sub>2</sub>）可以回收再用，而[[氯化銨|氯化銨]]（NH<sub>4</sub>Cl）又可以与[[生石灰|生石灰]]反应，产生NH<sub>3</sub>，重新作为原料使用：

: 2NH<sub>4</sub>Cl + CaO → 2NH<sub>3</sub> + CaCl<sub>2</sub> + H<sub>2</sub>O

氨碱法使生产实现了连续性生产，食盐的利用率得到提高，产品质量纯净，因而被称为「纯碱」，但最大的优点还在于成本低廉。

=== 联合制碱法 ===
{{Main|联合制碱法}}侯氏（联合制碱法）与索氏两法的区别是对于氯化铵的处理。索氏是加入生石灰发使氨逸出，而侯氏是加入食盐使氯化铵结晶而出。

== 历史 ==
在联合制碱法发明以前，[[氨碱法|氨碱法]]（亦稱為索爾維製碱法）应用最为广泛。是比利时人[[欧内斯特·索尔维|歐內斯特·索爾維]]（Ernest Solvay）於1862年發明的。

1867年[[苏威集团|索尔维公司]]制造的产品在[[巴黎世界博览会|巴黎世界博览会]]上获得铜制奖章，此法被正式命名为[[氨碱法|索尔维法]]。此时，纯碱的价格大大下降。消息传到英国，正在从事[[路布兰法|路布兰法]]制碱的英国[[哈琴森公司|哈琴森公司]]取得了两年独占索尔维法的权利。1873年哈琴森公司改组为[[卜内门公司|卜内门公司]]（Brunner Mond，即[[帝國化學工業|帝國化學工業]]（ICI）的前身），建立了大规模生产纯碱的工厂，后来，法、德、美等国相继建厂。这些国家发起组织[[索尔维会议|索尔维公会]]，设计图纸只向会员国公开，对外绝对保守秘密。凡有改良或新发现，会员国之间彼此通气，并相约不申请[[专利|专利]]，以防泄露。除了技术之外，营业也有限制，他们采取分区售货的办法，例如中国市场由英国卜内门公司独占。由于如此严密的组织方式，凡是不得[[索尔维会议|索尔维公会]]特许权者，根本无从问津氨碱法生产详情。直至20世纪初，许多国家要想探索[[氨碱法|索尔维法]]奥秘的厂商，无不以失败而告终。直至有關專利告終，此制作方法才大白於世。

由[[氨|氨]]、[[二氧化碳|二氧化碳]]和饱和食盐水为原料制得，称为[[联合制碱法|联合制碱法]]，是中国化学家[[侯德榜|侯德榜]]在[[氨碱法|氨碱法]]的基础上改进而成，该法在[[世界博览会|世界博览会]]上获得金制奖章。在北美，由于有大量天然碳酸钠矿藏存在，用天然的碱人工加工精制是製取碳酸钠的主要手段（天然碱法）。碳酸钠是化学及工业中的重要原材料。2003年，中国碳酸钠产量已经超过美国跃居世界第一。

== 结晶水合物 ==

碳酸钠的[[结晶水合物|结晶水合物]]俗称'''石碱'''，其[[分子式|分子式]]为Na<sub>2</sub>CO<sub>3</sub>·10H<sub>2</sub>O，带十个结晶水，在干燥的空气中易[[风化|风化]]。

== 水溶液 ==

其水溶液显弱碱性（所以被称为纯碱），有滑腻感，可以用于洗涤油污。

* 在不断通入二氧化碳气体的情况下，碳酸钠会与二氧化碳、[[水|水]]共同反应生成[[碳酸氢钠|碳酸氢钠]]：
: Na<sub>2</sub>CO<sub>3</sub> + CO<sub>2</sub> + H<sub>2</sub>O → 2NaHCO<sub>3</sub>
同時會發生副反應：

:{{chem|CO|2}} + {{chem|H|2|O}} ⇌ {{chem|H|2|CO|3|}} 

[[二氧化碳|二氧化碳]]（{{lang|en|[[碳|C]][[氧|O]]<sub>2</sub>}}）溶於[[水|水]]後，一部分[[二氧化碳|二氧化碳]]會與[[水|水]]化合，形成[[碳酸|碳酸]]。該反應是一個[[可逆反應|可逆反應]]，<ref>{{Greenwood&Earnshaw2nd|page=310}}</ref>
該反應在常溫下的[[平衡常数|平衡常數]]是<font lang=en>''K<sub>h</sub>''</font>=1.70×10<sup>−3</sup>；因此大部分[[二氧化碳|二氧化碳]]都不會參與反應。假若沒有[[催化劑|催化劑]]存在，[[反應速率|反應速率]]十分緩慢，其[[反應速率|反應速率]]常數僅為0.039 s<sup>−1</sup>（正反應）以及23 s<sup>−1</sup>（逆反應）。
* 碳酸钠遇[[酸|酸]]会发生剧烈反应，同时生成[[二氧化碳|二氧化碳]]和[[水|水]]：
: Na<sub>2</sub>CO<sub>3</sub> + 2H<sup>+</sup> → 2 Na<sup>+</sup> + CO<sub>2</sub> + H<sub>2</sub>O

该性质被利用于制造气体[[灭火器|灭火器]]。

* 碳酸钠与[[氢氧化钙|氢氧化钙]]反应生成[[氢氧化钠|氢氧化钠]]和[[碳酸钙|碳酸钙]]，工业上用来制[[氢氧化钠|氢氧化钠]]：
: Na<sub>2</sub>CO<sub>3</sub> + Ca(OH)<sub>2</sub> → 2 NaOH + CaCO<sub>3</sub>

* 在水溶液中，碳酸钠会完全[[电解|电解]]为[[碳酸根|碳酸根]]与[[钠|钠]]离子，[[碳酸根|碳酸根]]会发生[[水解|水解]]：  
: {{chem|CO|3|2-}} + H<sub>2</sub>O → {{chem|HCO|3|-}} + OH<sup>−</sup>

== 应用 ==

碳酸钠的主要工业应用领域为平板玻璃、日用玻璃、合成洗衣粉、中和酸性廢料、氧化铝等行业。

在生产生活中，碳酸钠用于制作麵点，制取[[氢氧化钠|氢氧化钠]]等，或作为多种洗涤剂的配方。水的净化用它来做软化剂。

碳酸钠在极高的温度（1000℃以上）会少量分解，但是在没有外加助剂的情况下，即使到1270℃，在数小时内仍然只是少部份分解（<2％）。

: Na<sub>2</sub>CO<sub>3</sub> → Na<sub>2</sub>O + CO<sub>2</sub>

这个反应是勒布朗法生产[[氢氧化钠|氢氧化钠]]的原理。
