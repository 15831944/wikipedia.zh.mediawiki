'''碳酸钙'''，俗稱'''灰石'''、'''石灰石'''、'''石粉'''，是一種[[化合物|化合物]]，[[化學式|化學式]]為[[鈣|Ca]][[碳酸鹽|CO<sub>3</sub>]]，呈碱性，在純[[水|水]]中溶解度甚小（[[溶度积|K<sub>sp</sub>]] = 4.8×10<sup>-9</sup>），可溶于[[酸|酸]]中。

碳酸鈣在地球上存量豐富，並以許多形式存在於[[岩石|岩石]]、[[礦物|礦物]]與[[生物體|生物體]]，如：[[霰石|霰石]]、[[方解石|方解石]]、[[白堊|白堊]]、[[石灰岩|石灰岩]]、[[大理石|大理石]]、[[石灰華|石灰華]]。亦為動物[[骨骼|骨骼]]或外[[螺壳|殼]]（如：[[爬蟲類|爬蟲類]]與[[双壳纲|双壳纲]]）的主要成份。碳酸鈣亦為[[水垢|水垢]]的主要成分，通常藉由水中的鈣離子與碳酸根離子結合所生成的，時常發生在[[硬水|水質硬度]]較高的地區。碳酸鈣亦為農用石灰中的有效成分之一，可用於中和土壤酸性，但過度使用也有造成土壤[[鹽鹼化|鹽鹼化]]的風險。在醫學應用方面，碳酸鈣亦是常用的[[鈣質|鈣質]]來源，也常做為[[抑酸剂|抑酸剂]]中的有效成分<ref>Frost RW, Lasseter KC, Noe AJ, Shamblen EC, Lettieri JT. Effects of aluminum hydroxide and calcium carbonate antacids on the bioavailability of ciprofloxacin. ''Antimicrobial Agents and Chemotherapy''. 1992;36（4）:830-832. [http://www.ncbi.nlm.nih.gov/pmc/articles/PMC189440/pdf/aac00038-0166.pdf] </ref>之一。

碳酸鈣外观为白色粉末或无色结晶<ref name=food>{{cite book|author1=張哲朗|coauthors=黃種華|title=圖解食品添加物與實務|date=2015-10-01|publisher=五南圖書出版股份有限公司|isbn=9571182893|pages=90-91|chapter=6.4 碳酸钙}}</ref>，无臭无味<ref>{{cite web|title=碳酸钙|url=http://www.basechem.org/chemical/3441|website=化学品数据库|accessdate=2017-07-16}}</ref>。碳酸钙有[[方解石|方解石]]、[[文石|文石]]、{{le|球霰石|Vaterite}}三种同质异形体，化學成分雖然相同，物理性質則多有差異。方解石（β-CaCO<sub>3</sub>，英语：calcite）是[[六方晶系|六方晶系]]的菱面體結構，有兩組交叉的菱形解理；[[霰石|霰石]]（λ-CaCO<sub>3</sub>，英语：aragonite）则是[[斜方晶系|斜方晶系]]，只有平行長軸的解理，而且不明顯<ref name ="Ropp">{{cite book|last=R C Ropp Elsevier|title=Encyclopedia of the alkaline earth compounds|publisher=Elsevier|isbn=9780444595508|pages=359–370}}</ref>；而球霰石的结构尚未被完全理解<ref name="DemichelisRaiteri2013">{{cite journal|last1=Demichelis|first1=Raffaella|last2=Raiteri|first2=Paolo|last3=Gale|first3=Julian D.|last4=Dovesi|first4=Roberto|title=The Multiple Structures of Vaterite|journal=Crystal Growth & Design|volume=13|issue=6|year=2013|pages=2247–2251|issn=1528-7483|doi=10.1021/cg4002972}}</ref>。霰石的比重為2.9～3.0，比方解石之2.71重一點<ref>{{cite web|title=霰石Aragonite|url=http://twgeoref.moeacgs.gov.tw/GipOpenWeb/wSite/ct?xItem=125828&mp=105&ctNode=1233|website=經濟部中央地質調查所|accessdate=2017-07-16}}</ref>。
* 天然[[石灰石|石灰石]]（limestone），几乎完全是方解石。[[白垩|白垩]]（chalk）为非晶质的石灰岩，是由古生物的骨骼积聚形成的<ref>{{cite web|title=白垩|url=https://bkrs.info/slovo.php?ch=%E7%99%BD%E5%9E%A9|accessdate=2017-07-16}}</ref>。[[汉白玉|汉白玉]]是纯白色的[[大理石|大理石]]，内含闪光晶体，也是石灰石的一种形态。
* [[洞石|洞石]]（travertine）或[[石灰华|石灰华]]（tufa）常见于[[温泉|温泉]]，由于沉积速度快，岩石中会保留很多小型气泡<ref>{{cite web|author1=Hershel Friedman|title=TUFA|url=http://www.minerals.net/mineral/tufa.aspx|website=Minerals.net||accessdate=2017-07-16}}</ref>。洞石通常为霰石，但也可能是方解石<ref>{{cite web|author1=Hershel Friedman|title=TRAVERTINE|url=http://www.minerals.net/mineral/travertine.aspx|website=Minerals.net|accessdate=2017-07-16}}</ref>。

===化学性质===
在一大气压下將碳酸鈣加熱到900℃會分解成[[氧化鈣|氧化鈣]]和[[二氧化碳|二氧化碳]]

: <math>\rm CaCO_3 \rightarrow CaO+CO_2 \uparrow </math>
碳酸鈣會和稀[[鹽酸|鹽酸]]反應，會呈[[泡騰現象|泡騰現象]]，並生成[[氯化钙|氯化钙]]、[[水|水]]和[[二氧化碳|二氧化碳]]

:<math>\rm CaCO_3+2HCl \rightarrow CaCl_2+H_2O+CO_2 \uparrow </math>

碳酸鈣悬浊液通入過量[[二氧化碳|二氧化碳]]，會生成[[碳酸氫鈣|碳酸氫鈣]]溶液

: CaCO<sub>3</sub> + CO<sub>2</sub> + H<sub>2</sub>O → Ca(HCO<sub>3</sub>)<sub>2</sub>

碳酸鈣和[[碳酸|碳酸]]溶液（雨水）反應，生成[[碳酸氫鈣|碳酸氫鈣]]

: CaCO<sub>3</sub> + H<sub>2</sub>CO<sub>3</sub> → Ca(HCO<sub>3</sub>)<sub>2</sub>

把[[二氧化碳|二氧化碳]]通入[[氫氧化鈣|氫氧化鈣]]會生成碳酸鈣和水

: CO<sub>2</sub> + Ca (OH)<sub>2</sub> → CaCO<sub>3</sub>↓ + H<sub>2</sub>O

== 製備 ==
工業用碳酸鈣主要是來源是由礦場或採石場用机械方法直接粉碎天然的方解石、石灰石、白垩、贝壳等而制得。由于这种碳酸钙的沉降体积比轻质碳酸钙的沉降体积小，所以称之为重质碳酸钙。純度較高的碳酸鈣（用於食品或醫藥產業者）可由較純粹的礦物來源（如：[[大理石|大理石]]）提取而得。

另外，碳酸鈣亦可由[[石灰|石灰]]為原料至取而得。首先将石灰石等原料煅烧生成石灰（主要成分为[[氧化钙|氧化钙]]）和[[二氧化碳|二氧化碳]]；再加[[水|水]]消化石灰生成石灰乳（主要成分为[[氢氧化钙|氢氧化钙]]）；再通入[[二氧化碳|二氧化碳]]氣體，使之生成碳酸钙沉淀；最后经脱水、干燥和粉碎而制得。以此方法製得的碳酸鈣又稱之為沉澱碳酸鈣（Precipitated Calcium Carbonate, PCC）或輕質碳酸鈣。<ref name="PCC">{{cite web|title = Precipitated Calcium Carbonate|accessdate = 2014-01-11|url = http://www.lime.org/uses_of_lime/other_uses/precip_cc.asp|deadurl = yes|archiveurl = https://web.archive.org/web/20140111165543/http://www.lime.org/uses_of_lime/other_uses/precip_cc.asp|archivedate = 2014-01-11}}</ref>
:: CaCO<sub>3(s)</sub> → CaO<sub>(s)</sub> + CO<sub>2(g)</sub>↑
:: CaO<sub>(s)</sub> + H<sub>2</sub>O<sub>(l)</sub> → Ca (OH)<sub>2(aq)</sub>
:: Ca (OH)<sub>2(aq)</sub> + CO<sub>2(g)</sub> → CaCO<sub>3</sub><sub>(s)</sub>↓ + H<sub>2</sub>O<sub>(l)</sub>

高純度碳酸鈣可藉由純化過的[[碳酸钠|碳酸钠]]和[[氯化钙|氯化钙]]的水溶液复分解反应生成沉淀，然后经脱水、干燥和粉碎而制得<ref>Production of precipitated calcium carbonate, US 2962350 A, Trubey Philip W [https://docs.google.com/viewer?url=patentimages.storage.googleapis.com/pdfs/US2962350.pdf]. </ref>。
:: Na<sub>2</sub>CO<sub>3</sub><sub>(aq)</sub> + CaCl<sub>2</sub><sub>(aq)</sub> → CaCO<sub>3</sub><sub>(s)</sub>↓ + 2NaCl<sub>(aq)</sub>

== 分类 ==
碳酸钙可以分为重质碳酸钙、轻质碳酸钙和活性碳酸钙三大类。<ref name="南台科技大學">[https://eportal.stust.edu.tw/eshare/EshareFile/2012_12/2012_12_ff3bd161.docx 化學與工程材料　化學（一）期末報告] - [[南台科技大學|南台科技大學]]</ref>
* '''重质碳酸钙'''：简称重钙，又称研磨碳酸钙，是用机械方法直接粉碎天然的方解石、石灰石、白垩、贝壳等而制得（粒径为30～50µm）。重质碳酸钙的[[沉降体积|沉降体积]]（1.1-1.9mL/g）比轻质碳酸钙小，所以称之为重质碳酸钙。<ref>{{cite book|editor=高彦祥|title=食品添加剂基础|date=2012-05-01|publisher=中国轻工业出版社|isbn=7501987548|edition=第二版}}</ref>
* '''轻质碳酸钙'''，又称沉淀碳酸钙，简称轻钙。一种方法是将石灰石等原料煅烧生成[[石灰|石灰]]（CaO）和[[二氧化碳|二氧化碳]]（CO<sub>2</sub>），再加[[水|水]]消化石灰生成[[石灰乳|石灰乳]]，然后再通入[[二氧化碳|二氧化碳]]碳化石灰乳生成碳酸钙沉淀；或者用[[碳酸钠|碳酸钠]]和[[氯化钙|氯化钙]]进行复分解反应生成碳酸钙沉淀。最后经脱水、干燥和粉碎而制得。轻质碳酸钙的沉降体积（2.4-2.8mL/g）比重质碳酸钙大，所以称为轻质碳酸钙。按[[粒径|粒径]]大小分为微粒钙（>5µm）、微粉钙（1µm～5µm）、超细钙（0.1µm～1µm）、纳米钙（≤0.1µm）等<ref name=gxso>{{cite web|author1=小者|title=碳酸钙的用途 9介绍|url=http://www.gxso.net/news/1/881.html|website=新贸网|accessdate=2017-07-16|date=2016-08-01}}</ref>。
* '''活性碳酸钙'''，又称改性碳酸钙、表面处理碳酸钙、胶质碳酸钙、白燕華，简称活钙，是用表面改性剂对轻钙或重钙的结晶形态、粒子大小、粒度分布及表面性能等方面进行表面改性而制得。由于经表面改性剂改性后的碳酸钙一般都具有补强作用，即所谓的“活性”。<ref name="南台科技大學"/>

== 用途 ==
工业碳酸钙按粉碎细度的不同分为四种规格：单飞、双飞、三飞、四飞。单飞粉用于生产无水[[氯化钙|氯化钙]]，是[[重铬酸钠|重铬酸钠]]生产的辅助原料，也是[[玻璃|玻璃]]及[[水泥|水泥]]生产的主要原料，此外，还用于[[建筑材料|建筑材料]]和家禽[[饲料|饲料]]等；双飞粉是生产无水氯化钙和玻璃等的原料，也是橡胶和油漆的白色填料，还用于建筑材料等；三飞粉可用作塑料、涂料及油漆的填料；四飞粉可用作电线绝缘层之填料、橡胶模压制品以及沥青制油毡的填料。<ref>{{cite web|title=碳酸钙|url=http://www.chemicalbook.com/ChemicalProductProperty_CN_CB2154602.htm|website=ChemicalBook|accessdate=2017-07-16}}</ref>
[[File:500_mg_calcium_supplements_with_vitamin_D.jpg|thumb]]
碳酸钙在[[医疗|医疗]]上用作抗酸药，能中和胃酸、保护溃疡面，用于胃酸过多、胃和十二指肠溃疡等病<ref>Frost RW, Lasseter KC, Noe AJ, Shamblen EC, Lettieri JT. Effects of aluminum hydroxide and calcium carbonate antacids on the bioavailability of ciprofloxacin. ''Antimicrobial Agents and Chemotherapy''. 1992;36(4):830-832. [http://www.ncbi.nlm.nih.gov/pmc/articles/PMC189440/pdf/aac00038-0166.pdf] </ref>。碳酸钙作为[[食品添加剂|食品添加剂]]，以保证人体所必需的钙的摄入，但通常不超过2%。除了用于医药食品，碳酸钙还可用作牙粉、牙膏及其他化妆品的原料。在口香糖、巧克力中，可以作为强化剂，既降低成本，又作为基质材料。在牙膏中，重质碳酸钙作为摩擦剂使用；在化妆品中，较细的优质碳酸钙可以作为填充剂。<ref name=food/>

其他用途还有：
* [[冶金|冶金]]工业中，主要用于助溶剂
* 用于制造建築材料[[水泥|水泥]]、[[石灰|石灰]]、人造石
* 中和[[二氧化硫|二氧化硫]]、[[水|水]]、酸性土壤或農田
* 製造[[玻璃|玻璃]]、[[鈣片|鈣片]]等
* 塑料、橡胶、涂料、硅酮胶等行业主要用来当作填料，以降低成本。还可以提高塑料制品的稳定性、硬度、刚性、耐热性以及加工性能。<ref name=gxso/>
