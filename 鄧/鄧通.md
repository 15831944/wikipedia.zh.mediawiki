'''鄧通'''，[[西漢|西漢]][[蜀郡|蜀郡]]人，擅長搖槳划船。原為船夫，後來成為[[漢文帝|漢文帝]]的[[佞幸|佞幸]]，得到文帝的寵幸，並賜給銅山成為富豪，卻得罪太子[[劉啟|劉啟]]，太子[[登基|登基]]後，剝奪鄧通所有財產。鄧通最後餓死。

== 飞黄腾达 ==
鄧通原是一個[[船夫|船夫]]，漢朝時稱之為「黃頭郎」。話說有一天漢文帝做了一個[[夢|夢]]：夢到自己在往天上爬，卻上不去，有個黃頭郎從身後推一把，終於爬到天上。漢文帝認為喻意自己可以登天[[成仙|成仙]]，故此積極憑夢境中「黃頭郎」的模樣，按圖索驥，尋遍天下，把「黃頭郎」找出來，封官賞賜，找到了鄧通。

== 吮痈舐痔 ==
更剛巧漢文帝身上長了一個大[[瘡|瘡]]，鄧通更加不錯過這個給[[皇帝|皇帝]]獻媚的絕佳機會，便趴在這個大瘡上，忍著噁心[[嘔吐|嘔吐]]，啜去大瘡的膿汁。鄧通的舉動打動了漢文帝的心，問鄧通道：「普天下誰最愛朕呢？」鄧通運用小聰明，工於心計的答道：「當然是[[太子|太子]]啊！」這時正好太子前來向皇帝問病，皇帝要太子給他吸啜身上的脓汁，想不到太子感到為難。令漢文帝感到鄧通比親兒子還要疼愛自己。鄧通自始得到漢文帝的歡心。但是邓通也因此得罪了太子（汉景帝）。

== 鄧通錢 ==
及後，皇帝命[[相士|相士]]為鄧通看相，相士看過後稱鄧通最後會因貧窮而[[餓死|餓死]]，文帝心想「對自己那麼好的人，能讓他餓死嗎？」於是文帝把蜀郡的一座[[銅|銅]]山賜給鄧通，准許他私人鑄錢，從此，'''鄧通錢'''流通天下，鄧通亦富甲一方。

== 下场 ==
有次邓通在朝堂上有怠慢之舉，[[宰相|宰相]][[申屠嘉|申屠嘉]]向文帝说：“陛下愛幸臣，則富貴之；至於朝廷之禮，不可以不肅！”。下朝后，相府下令逮捕邓通。邓通知道后，非常惊恐，跑去告诉文帝，希望文帝能保全。文帝說：“汝第往，吾今使人召若。”鄧通只好摘去帽子，光著腳，前往丞相府。申屠嘉責備他是區區小臣，竟敢在殿上嬉皮笑臉，“[[大不敬|大不敬]]，當斬!”鄧通嚇壞了，連連叩首，額頭鮮血直流。申屠嘉仍不平怒火。最後文帝派使者[[持节|持节]]召鄧通，而拜谢丞相說：“此吾弄臣，君释之。”皇帝亲自为邓通求情，邓通才得以释免。邓通哭著對文帝說：“丞相几杀臣。”

公元前156年，[[汉景帝|汉景帝]]继位。因早年邓通得罪他，汉景帝便以过境采矿的罪名<ref>《史记·佞幸列传》：“人有告邓通盗出徼外铸钱。”</ref>，罢其官，没收其财产。邓通饿死[[雅安|雅安]]。成語「'''不名一錢'''」「'''一文不名'''」就是在說鄧通。
