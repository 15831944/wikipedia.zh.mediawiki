'''鄧安寧'''{{BD|1956年||||catIdx=D鄧}}，是[[台灣|台灣]]電視連續劇導演、演員。

== 經歷 ==
青年時期鄧安寧以與友人組團、玩樂器為主，並在多家餐廳駐唱。當時樂團的主唱[[張小雯|張小雯]]之後成為他的妻子。

之後鄧安寧加入[[蘭陵劇坊|蘭陵劇坊]]學習演戲。<ref>{{Cite web  |url=http://ge.tnua.edu.tw/artstory/p01-94-4.htm |title=在散播歡樂中圓夢|publisher=[[國立台北藝術大學|國立台北藝術大學]] |author=[[趙自強|趙自強]] |date=2005-11-01 |accessdate=2012-09-18 }}</ref>後續因劇坊的同伙[[許效舜|許效舜]]、[[楊麗音|楊麗音]]等人進入電視節目《[[連環泡|連環泡]]》演出時推薦鄧安寧一同加入，鄧安寧便開始從事電視演員工作。<ref name="BIYASUNA">{{Cite web |url=http://www.eforu.com.tw/www/literature/human_directer02.htm |title=BI YA SU NA大愛劇場金鐘導演—鄧安寧 |publisher=中華民國剪輯協會 |author=陳韻年 |date=2003 |accessdate=2012-09-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20140222205327/http://www.eforu.com.tw/www/literature/human_directer02.htm |archivedate=2014-02-22 }}</ref>

鄧安寧後續除參與電視劇演出外，並在多家電視台擔任戲劇導演，並曾以2002年電視劇《[[別來無恙|別來無恙]]》獲得[[金鐘獎|金鐘獎]]的最佳導演提名。<ref name="BIYASUNA" />
