'''鄧永健'''（英文名：{{lang|en| Paul Tang Wing Kin}}，{{bd|1983年|6月9日|catIdx=T鄧}}），香港男演員，籍貫[[廣東|廣東]][[番禺|番禺]]，香港[[無綫電視|無綫電視]]基本藝人合約藝員及自由身粤語配音員。

== 簡歷 ==
鄧永健早年就讀[[青衣|青衣]]已停辦的[[閩僑第二小學|閩僑第二小學]]<ref name="Jim Tang's primary school, print screened on 2018-6-15">[https://drive.google.com/file/d/1w3JSSHcR3-H8FCQQXBPewwsr6CJY5n3J/view?usp=sharing Jim Tang's primary school, print screened on 2018-6-15]</ref>，為該校1996年第24屆畢業生。後在2001年畢業於[[皇仁舊生會中學|皇仁舊生會中學]]<ref>{{web reference|URL=http://elearning.qcobass.edu.hk/~qcobassaa/act.php|title=「師友計劃」專題講座|work=皇仁舊生會中學校友會|year=2010年|date=1月28日}}</ref>。曾為[[海洋公園|海洋公園]]的小丑藝人，後參與[[無綫電視|無綫電視]]第20期[[無綫電視藝員訓練班|無綫電視藝員訓練班]]<ref>[http://hk.myblog.yahoo.com/tangwingkin8369/profile 鄧永健blog簡介]</ref>，畢業後加入[[無綫電視|無綫電視]]。首齣拍攝的電視劇是[[千謊百計|千謊百計]]，2007年起參與處境劇[[同事三分親|同事三分親]]（劇中飾演Gary）；自第178集<ref>{{Cite web |url=http://tvcity.tvb.com/drama/best_selling_secrets/story/178.html |title=劇集同事三分親官方網頁：第178集--賤男真面目 |access-date=2007-07-21 |archive-url=https://web.archive.org/web/20071117133740/http://tvcity.tvb.com/drama/best_selling_secrets/story/178.html |archive-date=2007-11-17 |dead-url=yes }}</ref> 中佔主要戲份後，他的演出取得該劇觀眾正面評價。<ref>[http://www8.discuss.com.hk/viewthread.php?tid=5529555&extra=page%3D2&page=79 香港討論區：《同事三分親》討論專區] {{webarchive|url=https://web.archive.org/web/20160313221925/http://www.discuss.com.hk/viewthread.php?tid=5529555&extra=page%253D2&page=79 |date=2016-03-13 }}</ref> <ref>[http://www8.discuss.com.hk/viewthread.php?tid=5875937&extra=page%3D3　香港討論區：有關新人佔戲的討論]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref> <ref>{{Cite web |url=http://community.she.com/messageboard/entertainment/?action=view.topic&id=1956706&ref=com.home |title=存档副本 |access-date=2007-12-06 |archive-url=https://web.archive.org/web/20150619231822/http://community.she.com/messageboard/entertainment/?action=view.topic&id=1956706&ref=com.home |archive-date=2015-06-19 |dead-url=yes }}</ref><ref>[http://www4.uwants.com/viewthread.php?tid=5271434&extra=page%3D11　Uwants討論區：有關的鄧永健的討論]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

2008年10月開始播出的《[[畢打自己人|畢打自己人]]》（《[[同事三分親|同事三分親]]》姐妹處境劇）中，飾演其中一個主要角色－蘇同和（亦為Gary），並憑該角色的突出表現，打入[[2009年萬千星輝頒獎典禮|2009年萬千星輝頒獎典禮]]-飛躍進步男藝員提名最後五強，2012年開始兼任自由身配音員。

在《[[放學ICU|放學ICU]]》完結前，曾為主持之一。

現時為日本演員[[小栗旬|小栗旬]]、韓國演員[[李準基|李準基]]的常任配音員。
