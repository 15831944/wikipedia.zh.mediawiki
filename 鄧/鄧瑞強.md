'''鄧瑞強'''博士（{{lang-en|Dr. TANG Sui Keung}}），是[[香港神學院|香港神學院]]神學及倫理學副教授，[[香港中文大學|香港中文大學]][[崇基學院|崇基學院]]榮譽助理校牧。<ref>{{ cite web |url=https://www.cpr.cuhk.edu.hk/tc/events_detail.php?id=24601&t=%E5%B4%87%E5%9F%BA%E5%AD%B8%E9%99%A2%E7%A6%AE%E6%8B%9C%E5%A0%82%E4%B8%BB%E6%97%A5%E5%B4%87%E6%8B%9C-%E7%AB%8B%E6%96%BC%E4%BA%82%E4%B8%96%E4%B9%8B%E4%B8%AD&id=24601&t=%E5%B4%87%E5%9F%BA%E5%AD%B8%E9%99%A2%E7%A6%AE%E6%8B%9C%E5%A0%82%E4%B8%BB%E6%97%A5%E5%B4%87%E6%8B%9C-%E7%AB%8B%E6%96%BC%E4%BA%82%E4%B8%96%E4%B9%8B%E4%B8%AD |title=立於亂世之中 |date=2018-09-17|publisher=香港中文大學傳訊及公共關係處}}</ref><ref>{{ cite news |url=http://chinese.gospelherald.com/articles/26608/20180322/%E5%8F%97%E9%9B%A3%E7%AF%80%E5%B0%87%E8%87%A8-%E9%84%A7%E7%91%9E%E5%BC%B7-%E8%8B%A6%E9%9B%A3%E4%B8%96%E7%95%8C%E4%B8%AD%E7%9A%84%E8%BA%AB%E5%88%86%E5%8F%8A%E7%9B%BC%E6%9C%9B.htm |title=受難節將臨 ，鄧瑞強：苦難世界中的身分及盼望 |date=2018-03-22  |publisher=基督日報 }}</ref>

==簡歷==
鄧瑞強，1981年入讀[[香港中文大學|香港中文大學]]哲學系，1985年本科畢業後入讀香港中文大學研究院，師從[[陳特|陳特]]，1987年獲哲學碩士學位，碩士論文為《祈克果之生命三階段研究》。之後入讀[[中國神學研究院|中國神學研究院]]，1989年獲基督教研究碩士學位。<ref>{{ cite web |url=https://www.bshk.edu.hk/index.php/team/lecturer?id=31 |title=鄧瑞強博士 |date= |publisher=香港神學院 |author= }}</ref><ref name="logo69">{{ cite book |url= |title=基道文字（第69期）|date=2016年4月 |publisher=基道出版社 |author= |location=香港 | pages=16  |ISBN= }} </ref><ref>{{ cite web |url=https://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=83232&Pid=2&Version=1393&Cid=915&Charset=big5_hkscs |title=趙崇明鄧瑞強蔡式平談，在現代社會反思靈修觀 |date=2014-05-07 |publisher=時代論壇 |author=韋珮華}}</ref>

1989年至2003年，鄧瑞強於五旬節聖潔會永光堂任職傳道。之從返回香港中文大學研究院攻讀宗教研究哲學博士，專研基督教神學家[[潘霍華|潘霍華]]的倫理學，師從[[賴品超|賴品超]]，2006年畢業，博士論文為《回應他者：潘霍華倫理學在中國處境下的意義》。2006年至2008年，他於香港中文大學文化及宗教研究系擔任一級導師。2008年轉任[[香港神學院|香港神學院]]，先擔任神學及歷史科專任講師，現任神學及倫理學副教授。<ref>{{cite book |url= |title=香港中文大學概況(2007-2008) |date=2007 |publisher=香港中文大學 |author= |location=香港 | pages=252  |ISBN= }}</ref><ref>{{cite book |url= |title=多元、分歧與認同：神學與文化的探索 |date=2011-05-12 |publisher=橄欖 |author=賴品超 |location=香港 | pages=134  |ISBN=9789866131035 }}</ref>
<ref>{{ cite web |url=http://www2.crs.cuhk.edu.hk/academic-research/thesis-of-the-graduate-research-students/ph-d-in-religious-studies |title=學術研究 |date= |publisher=香港中文大學文化及宗教研究系 |author= }}</ref>

2008年至2013年，鄧瑞強同時擔任香港中文大學文化及宗教研究系兼任講師。由2008年起，鄧氏兼任香港中文大學崇基學院榮譽助理校牧<ref>{{ cite web |url=https://www.cco.cuhk.edu.hk/chaplaincy/chi/contact-us/ |title=聯絡我們 |date= |publisher=香港中文大學崇基學院禮拜堂 |author= }}</ref><ref name="logo69" />。 另外，他亦經常在不同教會講道及擔任公開講座主講<ref>{{ cite web  |title=「當憂慮遇上平安」講座，鄧瑞強：以神的恩典克服憂慮 |date=2009-04-27 |publisher=時代論壇 |author=韋珮華 |url=https://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=83232&Pid=2&Version=1393&Cid=915&Charset=big5_hkscs }}</ref><ref>{{ cite web  |title=參悟死生
 |date=2012-06-06 |publisher=時代論壇 |author= |url=https://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=83232&Pid=2&Version=1393&Cid=915&Charset=big5_hkscs }}</ref><ref>{{ cite web  |title=反思現代靈修觀 |date=2014-05-04 |publisher=基督教週報 |author= |url=http://www.christianweekly.net/2014/ta2022373.htm  }}</ref><ref>{{ cite web  |title=2016年公開講座：「死啊，你的毒鉤還在嗎？」 |date=20166-03-03 |publisher= |author= |url=http://www.christianweekly.net/2014/ta2022373.htm  }}</ref><ref>{{ cite web  |title=教會倫理系列講座：耶穌的倫理學 |date=2011-03-27 |publisher=基道書樓 |author= |url=http://www.logos.com.hk/acms/content.asp?site=logos&op=showbyid&id=6623 }}</ref>。

== 個人生活 ==
鄧瑞強的妻子楊燕玲，乃五旬節聖潔會永光堂拿撒勒之家資深長老，因病於2017年3月9日辭世。<ref>{{ cite web |url=https://www.wkphc.org/12774/ |title=[大堂報告]4月2日 |date=2017-04-02 |publisher=五旬節聖潔會永光堂 |author= }}</ref>
