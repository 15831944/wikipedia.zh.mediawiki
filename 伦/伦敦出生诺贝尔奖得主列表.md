'''伦敦出生诺贝尔奖得主列表'''，列出在[[英国|英国]]'''[[伦敦|伦敦]]'''出生的'''[[诺贝尔奖|诺贝尔奖]]'''得主。[[伦敦|伦敦]]是[[英国|英国]][[首都|首都]]和最大城市，也是[[欧盟|欧盟]]的最大[[都会区|都会区]]。该列表为''不完全列表''。

==诺贝尔物理学奖==
在伦敦出生的[[诺贝尔物理学奖|诺贝尔物理学奖]]得主有：
# 2003年诺贝尔物理学奖，'''[[安東尼·萊格特|安東尼·萊格特]]'''，英国/[[美国|美国]][[理论物理学|理论物理学]]家。1938年出生于英国伦敦的[[坎伯韦尔|坎伯韦尔]]。<ref name="2003－莱格特">{{cite web |url=http://nobelprize.org/nobel_prizes/physics/laureates/2003/leggett-autobio.html |title=2003年诺贝尔物理学奖得主安東尼·萊格特自传 |publisher=[[诺贝尔基金会|诺贝尔基金会]] |accessdate=2011年1月10日 |language=en |archive-url=https://web.archive.org/web/20111225024910/http://www.nobelprize.org/nobel_prizes/physics/laureates/2003/leggett-autobio.html |archive-date=2011-12-25 |dead-url=yes }}</ref>
# 1956年诺贝尔物理学奖，'''[[威廉·肖克利|威廉·肖克利]]'''，美国物理学家、发明家、微电子学家。1910年出生于英国伦敦。
# 1948年诺贝尔物理学奖，'''[[帕特里克·布萊克特|帕特里克·布萊克特]]'''，英国实验物理学家。1897年出生于英国伦敦。

==诺贝尔化学奖==
在伦敦出生的[[诺贝尔化学奖|诺贝尔化学奖]]得主有：
# 1956年诺贝尔化学奖，'''[[西里尔·欣谢尔伍德|西里尔·欣谢尔伍德]]'''，英国[[物理化学|物理化学]]家。1897年出生于英国伦敦。<ref name="1956－欣谢尔伍德">{{cite web|url=http://nobelprize.org/nobel_prizes/chemistry/laureates/1956/hinshelwood-bio.html |title=1956年诺贝尔化学奖得主西里尔·欣谢尔伍德自传 |publisher=[[诺贝尔基金会|诺贝尔基金会]] |accessdate=2011年1月10日 |language=en}}</ref>
# 1952年诺贝尔化学奖，'''[[阿彻·约翰·波特·马丁|阿彻·约翰·波特·马丁]]'''，英国[[化学|化学]]家。1910年出生于英国伦敦。

==诺贝尔生理学或医学奖==
在伦敦出生的[[诺贝尔生理学或医学奖|诺贝尔生理学或医学奖]]得主有：
# 2009年诺贝尔生理学或医学奖，'''[[杰克·绍斯塔克|杰克·绍斯塔克]]'''，美国[[生物学|生物学]]家。1952年出生于英国伦敦。<ref name="2009－绍斯塔克">{{cite web|url=http://nobelprize.org/nobel_prizes/medicine/laureates/2009/szostak.html |title=2009年诺贝尔生理学或医学奖得主杰克·绍斯塔克自传 |publisher=[[诺贝尔基金会|诺贝尔基金会]] |accessdate=2011年1月10日 |language=en}}</ref>
# 2003年诺贝尔生理学或医学奖，'''[[彼得·曼斯菲尔德|彼得·曼斯菲尔德]]'''），英国物理学家。1933年出生于英国伦敦南区的[[兰贝斯|兰贝斯]]。
# 1963年诺贝尔生理学或医学奖，'''[[安德魯·赫胥黎|安德魯·赫胥黎]]'''，英国[[生理学|生理学]]家、[[生物物理学|生物物理学]]家。1917年出生于英国伦敦的[[汉普斯特德|汉普斯特德]]。
# 1936年诺贝尔生理学或医学奖，'''[[亨利·哈利特·戴尔|亨利·哈利特·戴尔]]'''，英国[[神经生物学|神经生物学]]家、[[药理学|药理学]]家。1875年出生于英国伦敦。
# 1932年诺贝尔生理学或医学奖，'''[[查尔斯·斯科特·谢灵顿|查尔斯·斯科特·谢灵顿]]'''，英国神经生物学家、[[医学|医学]]家、[[病理学|病理学]]家、[[微生物学|微生物学]]家。1857年出生于英国伦敦的[[伊斯靈頓|伊斯靈頓]]。
# 1932年诺贝尔生理学或医学奖，'''[[埃德加·阿德里安，第一代阿德里安男爵|埃德加·阿德里安，第一代阿德里安男爵]]'''，英国神经生物学家。1889年出生于英国伦敦的[[汉普斯特德|汉普斯特德]]。

==诺贝尔和平奖==
在伦敦出生的[[诺贝尔和平奖|诺贝尔和平奖]]得主有：
# 1959年诺贝尔和平奖得主，'''[[菲利普·诺埃尔-贝克|菲利普·诺埃尔-贝克]]'''，英国[[政治家|政治家]]、[[外交家|外交家]]、[[学者|学者]]、[[运动员|运动员]]。1889年出生于英国伦敦西北区的[[亨頓|亨頓]]。
# 1937年诺贝尔和平奖得主，'''[[罗伯特·塞西尔，第一代切尔伍德的塞西尔子爵|罗伯特·塞西尔，第一代切尔伍德的塞西尔子爵]]'''，英国[[律师|律师]]、政治家、外交家。1864年出生于英国伦敦的[[卡文迪什广场|卡文迪什广场]]。
