'''伦敦桥行动'''是一个用于指代英国女王[[伊丽莎白二世|伊丽莎白二世]]駕崩後的進行相關事項的代号。<ref name="theguardian.com">{{cite newspaper|url=https://www.theguardian.com/uk-news/2017/mar/16/what-happens-when-queen-elizabeth-dies-london-bridge|work=[[The_Guardian|The Guardian]]|title=Operation London Bridge: the secret plan for the days after the Queen’s death|date=16 March 2017|accessdate=17 March 2017|first=Sam|last=Knight}}</ref><ref name="independent.co.uk">{{cite newspaper|url=https://www.independent.co.uk/news/uk/home-news/operation-london-bridge-queen-dead-elizabeth-ii-secret-plan-buckingham-palace-a7632891.html|first=Maya|last=Oppenheim|title=This is the secret code word when the Queen dies|work=The Independent|date=16 March 2017|publisher=}}</ref><ref>{{Cite news|url=http://www.huffingtonpost.co.uk/entry/operation-london-bridge-what-happens-when-the-queen-dies_uk_58ca6f33e4b0ec9d29d8ed39|title=5 Things We've Learned About 'London Bridge' – The Queen's Death Protocol|last=Bowden|first=George|date=16 March 2017}}</ref><ref>{{Cite news|url=http://metro.co.uk/2017/03/16/theres-a-secret-code-word-for-when-the-queen-dies-6512708/|title=There's a secret code word for when the Queen dies|last=Meyjes|first=Toby|date=16 March 2017|work=[[Metro_(British_newspaper)|Metro]]}}</ref> 该计划最初于20世纪60年代制定，此后每年更新若干次。 它涉及政府部门、[[英格兰教会|英格兰教会]]、[[伦敦警察厅|伦敦警察厅]]、[[英国军队|英国军队]]、媒体和[[伦敦御苑|伦敦御苑]]的规划。 与该计划有关的一些关键决定是女王自己做出的，而另一些决定只能由她的继任者（目前的法定继承人為[[威尔士亲王|威尔士亲王]]）在她身後做出。

截至2017年初，“伦敦桥塌了”一词预计将用于向首相和关键人员宣布女王去世，从而启动该计划。 2017年6月伦敦桥袭击事件发生后，这个短语是否还是指定的短语，或者这个代码现在已经公开的事实还没有得到证实。 也不知道这个代码短语最初是什么时候决定的。

== 背景 ==
当1952年国王[[乔治六世|乔治六世]]去世时，重要的政府官员被告知“海德公园之角”一词。这样做是为了防止白金汉宫电话总机接线员过早得知这一消息。

代号为“泰伊桥行动”的[[伊丽莎白王太后|伊丽莎白王太后]]的葬礼计划排练了22年，并且在1997年被用作[[戴安娜王妃|戴安娜王妃]]的葬礼的基础。<ref>{{Cite news|url=https://www.theguardian.com/uk/2002/apr/01/queenmother.monarchy1|title=A week of mourning for the last empress|date=1 April 2002|work=The Guardian}}</ref><ref name="Guard">{{Cite news|url=https://www.theguardian.com/uk/2002/apr/01/queenmother.monarchy1|title=A week of mourning for the last empress|date=1 April 2002|work=The Guardian}}</ref>

其他王室死者的葬礼计划也使用了英国著名桥梁的名字，包括：用于[[爱丁堡公爵菲利普亲王|爱丁堡公爵]]葬礼计划的“福斯桥行动”、以及用于[[威尔士亲王查尔斯|威尔士亲王]]葬礼计划的“梅奈桥行动”。<ref>{{Cite news|url=http://www.newstatesman.com/node/155725|title=The Insider – Paul Routledge|date=17 June 2002|work=New Statesman}}</ref>

== 计划 ==
{{Multiple image}}女王的私人秘书将是第一个接到消息的官员（亦即不是女王的亲属或者医疗团队的一员）。私人秘书的第一个行动将是与[[英国首相|英国首相]]联系，公务员将使用安全电话线传达“伦敦桥塌了”的代号短语。位于伦敦某处的[[英国外交部|英国外交部]]全球反应中心将把这一消息传达给其他十五个[[英联邦王国|英联邦王国]]的政府，以及其他[[英联邦|英联邦]]国家的政府。

媒体将通过一个公告被通知。该公告将通过无线电警报传输系统(RATS)向[[新闻协会|新闻协会]]和[[英国广播公司|英国广播公司]](BBC)发布，并通过一个蓝色“讣告灯”网络向商业电台发布，提醒主持人播放“无害音乐”并准备一条新闻快讯；而[[BBC第二台|BBC第二台]]将暂停当天的节目安排，转而播放[[BBC第一台|BBC第一台]]的通知。<ref>{{Cite web|url=https://www.gloucestershirelive.co.uk/news/uk-world-news/operation-london-bridge-what-happen-1759578|title=Operation London Bridge: This is what will happen when the Queen dies|accessdate=26 August 2018|last=Gogarty|first=Conor}}</ref>[[BBC新闻|BBC新闻]]将播放一系列预先录制好的女王肖像，期间当值主持人将穿上专门为此目的准备的暗色服装，为正式宣布死讯做准备。据《[[卫报|卫报]]》报道，《[[纽约时报|纽约时报]]》已经准备了11天的预先准备好的报道，[[ITN|ITN]]和[[天空新闻台|天空新闻台]](Sky News)早就排练过女王的死亡，只是使用了“罗宾逊夫人”这个名字。

一个仆人会在[[白金汉宫|白金汉宫]]的大门上钉一张黑边的告示。与此同时，白金汉宫的网站也将展示同样的告示。[[英国议会|英国议会]]将被召回；如果可能的话，会议将在数小时内举行；首相将在[[英国下议院|英国下议院]]发表讲话。

在死讯宣布之后的日子里，英国皇家铸币局将开始生产印有新君主头像的新硬币，供君主即位后发行。（尽管按照驾崩的说法，新君主将在伊丽莎白二世死后即刻继承王位。)

根据女王死亡地点的不同，移动女王棺材的安排也有所不同。例如，如果女王死在[[温莎城堡|温莎城堡]]或[[桑德林汉姆府|桑德林汉姆府]]，几天之内就会被开车运到白金汉宫。如果女王在海外去世，将由[[英国皇家空军|皇家空军]]第32中队送往皇家空军诺斯奥尔特基地，然后再开车送往白金汉宫。如果女王在[[苏格兰|苏格兰]]去世（例如在[[荷里路德宫|荷里路德宫]]或[[巴尔莫勒尔城堡|巴尔莫勒尔城堡]]），灵柩将首先安放在荷里路德宫，然后在爱丁堡[[圣吉尔斯大教堂|圣吉尔斯大教堂]]举行招待会。随后，棺材将被运往[[爱丁堡威瓦利站|爱丁堡威瓦利站]]，然后由[[皇家列车|皇家列车]]运往伦敦。在任何情况下，棺材都会被送到白金汉宫的王座室。女王去世后四天，它将被移至威斯敏斯特大厅，供公众瞻仰四天。

女王去世九天后，[[国葬|国葬]]将在[[西敏寺|西敏寺]]举行，之后她的遗体将被安葬在温莎城堡[[圣乔治礼拜堂_(温莎城堡)|圣乔治礼拜堂]]的一个准备好的坟墓中。
