'''-{zh-cn:艺术体操; zh-hk:藝術體操; zh-tw:韻律體操}-'''（Rhythmic gymnastics），又稱'''-{zh-cn:韵律体操; zh-hk:韻律體操; zh-tw:藝術體操}-'''、'''新體操'''，起源于[[欧洲|欧洲]]。1962年被国际体操联合会确定为比赛项目。1963年举办第一届世界艺术体操锦标赛，1984年被列为[[奥运会|奥运会]]正式比赛项目<ref>{{Cite news|url=http://www.ha.xinhuanet.com/sports/2014-03/19/c_119835892.htm|title=体操|work=新华网河南频道|deadurl=yes|archiveurl=https://web.archive.org/web/20140701053018/http://www.ha.xinhuanet.com/sports/2014-03/19/c_119835892.htm|archivedate=2014-07-01}}</ref>。

韻律體操原來只有女子比赛项目，但現在[[日本|日本]]、[[美国|美国]]、[[加拿大|加拿大]]、[[澳大利亚|澳大利亚]]、[[俄罗斯|俄罗斯]]、[[韩国|韩国]]、[[马来西亚|马来西亚]]、[[墨西哥|墨西哥]]等國家，已有男子進行韻律體操運動，称为[[男子艺术体操|男子艺术体操]]。首届[[世界男子艺术体操锦标赛|世界男子艺术体操锦标赛]]于2003年11月27日到29日召开。

與女子項目不同的是，女子的艺术体操主要有：绳操、圈操、球操、棒操和带操五种；而男子的通常沒有球操和棒操，但加入了棍操。但[[西班牙|西班牙]]的鲁本·奥利维拉（{{link-en|Ruben Orihuela}}）却将全部女子项目同时加以男子化；其身体所展现的柔韧性不逊于任何女子艺术体操运动员。

== 绳操 ==
绳操使用的轻器械是麻[[绳|绳]]或纤维制成的绳子，比赛动作主要有：过强绳跳、摆动、抛接、跳跃等。2011年，[[國際體操總會|國際體操總會]]決定從韻律體操比賽中廢除繩操這個項目。

== 球操 ==
球操使用的器械是直径20厘米左右的橡胶[[体操球|体操球]]，比赛动作主要有：拍球、滚动、抛接、跳跃、转动等。

== 棒操 ==
{{le|棒操|Baton twirling}}（也称舞棒）是使用长40至50厘米，形状如瓶的体操棒作为器械，表演动作主要有：空中转动、抛接、摆动、跳跃等，参与者目前多为女性。<ref>{{cite web|author1=民視|title=萬花叢中一點綠 日本小男生「舞棒」|url=https://tw.news.yahoo.com/%E8%90%AC%E8%8A%B1%E5%8F%A2%E4%B8%AD-%E9%BB%9E%E7%B6%A0-%E6%97%A5%E6%9C%AC%E5%B0%8F%E7%94%B7%E7%94%9F-%E8%88%9E%E6%A3%92-131651412.html|website=Yahoo新闻|accessdate=2017-09-24|date=2015-10-14}}</ref>

== 带操 ==
[[File:Ribbon_(rhythmic_gymnastics).jpg|thumb]]
带操使用的器械是长约6米，宽5厘米左右的[[彩带|彩带]]作为表演器械，彩带的一端与长50厘米左右的棍子相连，表演动作主要有：绕环、螺形、摆动、抛接、转体等。

== 圈操 ==
圈操使用的器械是内径约90厘米的木制或塑料制成的[[圈|圈]]，表演的动作主要有：钻圈、抛接、绕环、滚动等。
