'''赣州市'''，简称'''虔'''，是[[中华人民共和国|中华人民共和国]][[江西省|江西省]]下辖的[[地级市|地级市]]，常称为'''赣南'''地区，是中国七十个大中城市之一、江西省省域副中心城市，是江西省的南大门，也是江西省面积最大、人口最多、下辖县市最多的地级市。赣州东接[[福建省|福建省]][[三明市|三明市]]和[[龙岩市|龙岩市]]，南至西南临[[广东省|广东省]][[梅州市|梅州市]]、[[河源市|河源市]]和[[韶关市|韶关市]]，西靠[[湖南省|湖南省]][[郴州市|郴州市]]，北连江西省[[吉安市|吉安市]]、[[抚州市|抚州市]]，处于东南沿海地区向中部内地延伸的过渡地带，是内地通向东南沿海的重要通道。

赣州历史悠久，有着2200多年的建城史，是[[国家历史文化名城|国家历史文化名城]]<ref>{{cite web |url = http://www.wenbao.net/html/whyichan/lsmc/lsmc.htm |language=zh-cn |title=全国历史文化名城目录  }}</ref>。赣州也是客家人主要聚居地之一，客家文化浓厚。同时，赣州也是[[中国优秀旅游城市|中国优秀旅游城市]]、[[全国文明城市|全国文明城市]]，有着千里赣江第一城、江南宋城、客家摇篮、红色故都、世界橙乡、世界钨都等美誉。

赣州是全国稀有金属产业基地和先进制造业基地、原中央苏区振兴发展示范区、红色文化传承创新区和旅游目的地、区域性综合交通枢纽<ref>{{Cite Web|url=http://www.gov.cn/zhengce/content/2017-02/28/content_5171345.htm|title=国务院关于印发“十三五”现代综合交通运输体系发展规划的通知|author=中国政府网 | accessdate = 2018-6-19}}</ref>、赣粤闽湘四省通衢的区域性现代化中心城市。赣州都市区是江西省重点培育和发展的三大都市区之一。

== 历史沿革 ==
[[秦始皇|秦始皇]]三十三年（前214年），置[[南壄|南壄]]县，隶[[九江郡|九江郡]]，为赣南建置之始。

[[三国|三国]][[孫吳|吴]][[嘉禾|嘉禾]]五年（236年），置庐陵南部都尉，治[[雩都|雩都]]。

[[晋代|晋]][[太康|太康]]三年（282年），改置[[南康郡|南康郡]]，治雩都。[[永和_(晋穆帝)|永和]]五年（349），郡治从雩都迁至[[赣县|赣县]]（章、贡两水间，今章贡区）。

[[隋代|隋]][[开皇|开皇]]九年（589年），改南康郡为[[虔州|虔州]]。唐朝虔州治赣县，在今赣州市区，而不是今天的[[赣县|赣县]]。

[[宋代|宋]][[淳化|淳化]]元年（990年），以虔州原辖南康、大庾、上犹3县另置[[南安军|南安军]]，治[[大庾|大庾]]。[[绍兴|绍兴]]二十三年（1153年），改虔州为赣州。

[[明代|明]][[吴_(年號)|吳]]二年（1365年，元[[至正|至正]]二十五年），赣州、南安两路改为府。

[[民国|民国]]元年（1912年）废府，县直隶于省。三年（1914年）设赣南道。十五年（1926年），废赣南道，县直隶于省。

[[File:Chiang_Ching-kuo_in_Gannan_15.jpg|缩略图]]於赣州視察(1939)]]
1949年7月，成立赣州分区行政督察专员公署。8月析赣州镇设赣州市。1949年9月成立宁都（瑞金）分区行政督察专员公署。1952年8月，撤销宁都分区。1954年5月，成立赣南行政公署。1964年5月成立赣州专区。1971年1月改称赣州地区。

1999年7月，撤销赣州地区设立赣州市，原县级赣州市改设章贡区。<ref>[http://www.ganzhou.gov.cn/gzgk/jzyg/ 赣州市历史渊源] {{webarchive|url=https://web.archive.org/web/20110402073129/http://www.ganzhou.gov.cn/gzgk/jzyg/ |date=2011-04-02 }}</ref>

2013年11月12日国务院批准撤县级市南康市设南康区<ref>[http://www.jiangxi.gov.cn/dtxx/zwxx/201311/t20131113_992349.htm 国务院批准赣州调整部分行政区划] {{webarchive|url=https://web.archive.org/web/20131113024705/http://www.jiangxi.gov.cn/dtxx/zwxx/201311/t20131113_992349.htm |date=2013-11-13 }}</ref><ref>[http://jiangxi.jxnews.com.cn/system/2013/11/12/012792438.shtml 赣州：南康撤市设区获国务院批复] {{webarchive|url=https://web.archive.org/web/20131113024130/http://jiangxi.jxnews.com.cn/system/2013/11/12/012792438.shtml |date=2013-11-13 }}</ref><ref>[http://jx.sina.com.cn/news/dfdt/2013-11-18/073552773.html 赣州自行解决南康撤市设区所需人员编制和经费] {{webarchive|url=https://web.archive.org/web/20160304084926/http://jx.sina.com.cn/news/dfdt/2013-11-18/073552773.html |date=2016-03-04 }}</ref>。

2016年10月，赣县撤县设区获批复<ref name="Ganxian1">{{cite web | url = http://jiangxi.jxnews.com.cn/system/2016/10/12/015266887.shtml | title = 赣州赣县撤县设区获批复 | author = 袁思东 | publisher = 中国江西网 | date = 2016-10-12 | accessdate = 2016-10-15 | deadurl = no | archiveurl = https://web.archive.org/web/20161013130618/http://jiangxi.jxnews.com.cn/system/2016/10/12/015266887.shtml | archivedate = 2016-10-13 }}</ref>。设立赣州市赣县区<ref name="Ganxian2">{{cite web | url = http://jiangxi.jxnews.com.cn/system/2016/10/15/015281414.shtml | title = 赣县撤县设区为“赣县区” | author = 陈斌华 | publisher = 中国江西网 | date = 2016-10-15 | accessdate = 2016-10-15 | deadurl = no | archiveurl = https://web.archive.org/web/20161018073824/http://jiangxi.jxnews.com.cn/system/2016/10/15/015281414.shtml | archivedate = 2016-10-18 }}</ref>。

== 地理 ==
[[File:Ganzhousat.jpg|thumb]]

赣州市位于江西省南部，是江西省面积最大的地级行政区，全市以山脉、丘陵地形为主，占总面积的80.98%。[[章水_(赣江)|章江]]、[[贡江|贡江]]在赣州主城汇合为[[赣江|赣江]]。

2011年赣州市总户数275.18万户，总人口918.26万人，比年初增加10.99 万人。农业人口729.59万人，占总人口数的79.5%；非农业人口188.67万人，占总人口的20.5%。男性476.92万人，女性441.34万人。平均每户人口数为3.34人，人口密度每平方千米233人。全市人口出生率13.53‰，全市人口自然增长率为7.52‰<ref>{{cite web |url=http://www.ganzhou.gov.cn/jcgz/gzjj/gzgk/201206/t20120619_583356.htm |title=存档副本 |accessdate=2013-09-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20150412164342/http://www.ganzhou.gov.cn/jcgz/gzjj/gzgk/201206/t20120619_583356.htm |archivedate=2015-04-12 }}</ref>。

=== 民族 ===
全市汉族人口888.95万人，占总人口的99.1%。有杂散居少数民族成分41个，人口8.04万人（含外出人口），分布在19个县（市、区）（含赣州经济技术开发区）、204个乡（镇）、486个村，以畲族为主，回族集中在章贡区，瑶族集中在全南，有民族村30个。少数民族分别是：畲、回、蒙古、藏、维吾尔、苗、彝、壮、布依、朝鲜、满、侗、瑶、白、土家、哈尼、哈萨克、傣、黎、傈僳、高山、佤、拉祜、水、东乡、纳西、景颇、独龙、土、达斡尔、仫佬、羌、布朗、撒拉、毛南、仡佬、锡伯、塔吉克、怒、乌孜别克、鄂温克等民族<ref>{{cite web |url=http://www.ganzhou.gov.cn/jcgz/gzjj/gzgk/201206/t20120619_583356.htm |title=存档副本 |accessdate=2013-09-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20150412164342/http://www.ganzhou.gov.cn/jcgz/gzjj/gzgk/201206/t20120619_583356.htm |archivedate=2015-04-12 }}</ref>。

== 经济 ==
2014年赣州实现地区生产总值（GDP）1843.6亿元，增长10%，总量全省第二，增速全省第三。财政总收入328.5亿元，增长17.2%。固定资产投资1608.8亿元，增长20.9%，增速连续三年居全省首位。规模以上工业主营业务收入突破3000亿元，达3002.8亿元，增长14.9%。战略性新兴产业主营业务收入达400亿元，增长20%。全市民生类支出303.1亿元，增长11.6%，占财政总支出比重达56.6%，连续三年超过55%。城镇居民人均可支配收入22935元，增长10.3%，增速全省第二。农民人均可支配收入6946元，增长11.6%，增速连续两年全省第一。

长期以来，经济发展以[[农业|农业]]为主。种植的主要经济作物有[[水稻|水稻]]，[[甘蔗|甘蔗]]，[[烟草|烟草]]，[[西瓜|西瓜]]，[[柑桔|柑桔]]等。近年来大力发展[[脐橙|脐橙]]种植产业。工业主要有金属冶炼，木材加工，[[松香|松香]]采炼，制糖，烟草等，是中国主要的钨和稀土产地。近年来发展了一些劳动密集型产业以及与脐橙种植配套的脐橙加工产业和第三产业。

== 气候 ==
赣州市属中[[亚热带季风气候|亚热带季风气候]]，具有四季分明，热量充沛，降水充足，雨热同期等特点。冬季较为温和，偶有寒冷天气。夏季漫长，十分炎热。年平均气温19.4℃。年均降水量1461.2毫米，主要集中于春夏季。
