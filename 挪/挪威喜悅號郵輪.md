'''挪威喜悅號'''（'''Norwegian Joy'''，中國大陸翻譯為諾唯真喜悅號）是[[挪威郵輪|挪威郵輪]]旗下的一艘豪華[[郵輪|郵輪]]，她在2016年-2017年間在{{link-en|邁爾造船廠|Meyer Werft}}建造。目前為挪威郵輪旗下最大噸位的[[郵輪|郵輪]]及全球第八大郵輪，排水量達到167,725噸，屬於 Breakaway Plus 系列的郵輪之一，她與[[挪威遁逸號|挪威遁逸號]]是姊妹船，已於2017年下水並投入服務。

== 服務歷史 ==

== 設計 ==
[[挪威喜悅號|挪威喜悅號]]為[[中國|中國]]市場而設計的，郵輪不少地方與挪威遁逸號的設計非常不同，而且總噸位較挪威遁逸號大4,200噸，為Breakaway Plus Class系列中最大郵輪之一。

== 最新動態 ==
2018年3月30日，據《[[新民晚報|新民晚報]]》等媒體報道，因為上海吳淞口海域大霧，航道封鎖，原計劃在2018年3月27日-3月31日之間航行的[[挪威喜悅號|-{zh-hans:诺唯真;zh-hant:挪威}-喜悅號]]郵輪被迫滯留港口。不少遊客聚集在一起，對著郵輪工作人員高唱國歌，一旁還有人高聲叫好。遊客情緒失控後，與工作人員發生推搡……最終[[挪威郵輪|-{zh-hans:诺唯真;zh-hant:挪威}-郵輪]]官方決定取消當次航程，全額退款，並由合作的保險公司賠付港口取消的相關賠償。<ref>https://www.tapotiexie.com/Mobile/index.php/Mobile/content/id/5994</ref><ref>http://m.ce.cn/sh/sgg/201803/30/t20180330_28666737.shtml</ref><ref>http://gd.ifeng.com/a/20180330/6472177_0.shtml</ref>

2018年，公司通告「挪威喜悅號」將於2019年4月退出中國市場。<ref>{{Cite web|url=https://www.cruisecritic.com/news/news.cfm?ID=8765|title=Norwegian Cruise Line Will Move Another Mega Ship to Alaska in 2019 - Norwegian Cruise Line|website=www.cruisecritic.com|language=en|access-date=2018-07-30}}</ref><ref>{{Cite web |url=http://ec.ltn.com.tw/article/breakingnews/2580856 |title=豪華郵輪1年退出中國 網友直擊：被大媽吃垮 |work=自由財經|date=2018-10-15}}</ref>
