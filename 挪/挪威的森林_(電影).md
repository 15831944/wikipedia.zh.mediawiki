'''挪威的森林'''是於2010年上映的[[日本電影|日本電影]]，改編自日本作家[[村上春樹|村上春樹]]最廣為人知的作品《[[挪威的森林_(小說)|挪威的森林]]》，由法籍越南裔[[導演|導演]][[陳英雄|陳英雄]]執導，電影入圍競逐[[第67屆威尼斯國際電影節|第67屆威尼斯國際電影節]]的最高殊榮[[金獅獎|金獅獎]]。<ref name="venezia67">{{cite web |url=http://www.labiennale.org/en/cinema/archive/67th-festival/67/countries.html?back=true |title=Venezia 67 |accessdate=2011-01-13 |work=labiennale.org | date=2011-03-20}}</ref>

== 劇情 ==
故事在1967年開始，在[[神戶|神戶]]就讀高中的主角渡邊徹，有一名好友木月（キズキ）和他的青梅竹馬女友直子，木月其後自殺。渡邊徹獨自往[[東京|東京]]升讀大學，期間巧遇木月的前女友直子，兩人開始發展情侶關係。直子患上了[[憂鬱症|憂鬱症]]，進了京都一間名為「阿美寮」的集體精神病療養院。渡邊徹此時認識了性格活潑的女生小林綠，伴隨著主角的，是多愁善感的直子和野性活潑小林綠的三角關係，以及這[[學運|學運]]頻密的[[六十年代|六十年代]]。

== 製作 ==
改編自村上春樹的暢銷作品，原著小說中的重要角色均在角色名單中。此外本作小說的名稱，由[[披頭四|披頭四]]創作的曲目（Norwegian Wood）已確定會成為本作的主題曲，並用在目前所發放的一段31秒的預告當中。<ref name="n_wood">{{cite web|url=http://showbiz.chinatimes.com/2009Cti/Channel/Showbiz/showbiz-news-cnt/0,5020,100103+112010071500082,00.html|title=披頭四 認了陳英雄《挪威的森林》|accessdate=2010-08-07|work=中時電子報|date=2010-07-15|deadurl=yes|archiveurl=https://web.archive.org/web/20100913051933/http://showbiz.chinatimes.com/2009Cti/Channel/Showbiz/showbiz-news-cnt/0,5020,100103+112010071500082,00.html|archivedate=2010-09-13}}</ref>另一值得注意的是本作的配樂由歐美樂團[[電台司令|電台司令]]的第二創作靈魂——[[強尼·格林伍德|強尼·格林伍德]]負責。拍攝場景除了日本外，亦曾到[[法國|法國]]取景。<ref name="france">{{cite web|url=http://www.readingtimes.com.tw/TimesHtml/authors/murakami_haruki/whatsnew.html|title=《挪威的森林》拍攝現場演員模樣|accessdate=2010-08-07|work=時報悅讀網|date=2010-07-15}}</ref>
根據目前的拍攝現場報導，得知導演對細節的要求一絲不苟，包括對演員頭部的細微高度，或是燈光所拍攝的光度與距離等也有強烈標準。<ref name="light">{{cite web|url=http://sankei.jp.msn.com/entertainments/entertainers/100103/tnr1001031201003-n1.htm|title=【新春エンタメざんまい】透明感と官能　村上春樹の世界を緻密に映像化|accessdate=2010-08-07|work=|date=2010-01-03|deadurl=yes|archiveurl=https://web.archive.org/web/20100803061006/http://sankei.jp.msn.com/entertainments/entertainers/100103/tnr1001031201003-n1.htm|archivedate=2010-08-03}}</ref>此外為重現1969年的生活面貌，除了任何會出現在鏡頭前的物品雖與該年代的物品相符外，連不在拍攝範圍內的物品也放置了當時型號的物品。<ref name="light"/>
