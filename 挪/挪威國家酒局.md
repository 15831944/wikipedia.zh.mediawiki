'''挪威國家酒局'''（[[挪威文|挪威文]]：'''Vinmonopolet'''，符號：Ⓥ ，俗稱：'''Polet'''）是[[挪威政府|挪威政府]]掌管[[酒精飲料|酒精飲料]][[零售|零售]]的組織，為[[挪威|挪威]]唯一可以出售[[酒精含量|酒精含量]]在4.75%以上的酒精飲料的公司。

挪威國家酒局是挪威政府限制公民酒精消費的重要組織，并通過提高成本和有限享用來達成其目的。其主要目標是通過分配酒精產品來限制私人從酒精工業中獲得利益，防止銷售酒精產品給[[未成年人|未成年人]]和[[酒精中毒|嗜酒成癮者]]。

其批發商店位於全國各個城市與鄉村地區，停業時間早於其他商店，在工作日為下午6點，週六為下午3點。2007年，挪威國家酒局售出了71,100,000升酒精產品。<ref name=D2Nov1408/>

== 建立 ==

挪威國家酒局始建於1922年，當時便是一個[[國營企業|國營企業]]，其成立是與[[葡萄酒|葡萄酒]]出口商們（主要是[[法國|法國]]的）協商的結果。所有挪威的酒精銷售都需要通過這家公司來執行。<ref name=fiin-ch2>{{cite book|last = Hamran |first = Olav |coauthors = Myrvang, Christine |title = Fiin Gammel – Vinmonopolet 75 år |year = 1998 |publisher = Tano Aschehoug |isbn = 82-518-3738-3 |pages = 39–71 }}{{no icon}}</ref>

1939年，[[挪威財政部|挪威財政部]]和後來的[[挪威健康和保健服務部|挪威健康和保健服務部]]買下該公司的全部產權，使其成為一家[[股份公司|股份公司]]。<ref name=fiin-ch3>{{cite book|last = Hamran |first = Olav |coauthors = Myrvang, Christine |title = Fiin Gammel – Vinmonopolet 75 år |year = 1998 |publisher = Tano Aschehoug |isbn = 82-518-3738-3 |pages = 73–95 }}{{no icon}}</ref>

1996年，該公司的[[進口|進口]]和生產活動全部停止，原因是[[歐洲自由貿易聯盟法院|歐洲自由貿易聯盟法院]]認為其[[壟斷|壟斷]]行為違反了[[歐洲經濟區|歐洲經濟區]]協議。於是該公司將其烈酒的進出口和分配活動分離出來，轉交給了新成立的[[艾庫斯_(公司)|艾庫斯公司]]，而挪威國家酒局只負責零售業方面。<ref name=D2Nov1408>{{cite web|last= Anker |first= Nils, ''Dagens Næringsliv: D2'' |title= – Vi skal gjøre oss fortjent til folks støtte |url= http://www.dn.no/d2/d2mat/article1538626.ece  |date= November 14, 2008 }}{{no icon}}</ref>

== 與消費者的關係 ==

一項1990年代中期的調查顯示，大多數挪威人都支持取消其壟斷權，而允許私人銷售葡萄酒。<ref name=D2Nov1408/>自1996年重組以來，其顧客滿意度大幅上升，其[[行政總裁|行政總裁]][[凱·G·亨里克森|凱·G·亨里克森]]說：<ref name=D2Nov1408/>{{cquote|取消進口禁令是命運的一個巧妙安排。現在我們有210個進口商互相競爭以得到最合理的價格...}}

2008年[[BI挪威商學院|BI挪威商學院]]的一項調查顯示，該公司的顧客滿意度為81.5%，全挪威排名第四，<ref>kundebarometer.com [http://www.kundebarometer.com/index.php?content=nkbres2008&criteria=tilfredshet Customer poll, search criteria - satisfaction] {{webarchive|url=https://www.webcitation.org/5dEAtwnVN?url=http://www.kundebarometer.com/index.php?content=nkbres2008 |date=2008-12-21 }} {{no icon}}</ref><ref>{{cite web|last =Moe |first= Ingrid Elisabeth, ''Dagens Næringsliv'' |title= Disse hater og elsker vi |url= http://www.dn.no/privatokonomi/article1397936.ece |date= July 5, 2008 }}{{no icon}}</ref>隨後在次年升到了88%。<ref name=vinforum1-2010/>但是有質疑的聲音稱該公司雇傭的人也參加了這項調查。<ref name=dnSep08>{{cite web|last= Marthinsen |first= Tom, ''Dagens Næringsliv'' |title= Polets forskjellsbehandling |url= http://www.dn.no/vinguiden/article1484661.ece |date= September 6, 2008 }}{{no icon}}</ref>

挪威[[葡萄酒大師|葡萄酒大師]][[阿納·倫納德|阿納·倫納德]]指出現在的挪威國家酒局雖然在一些地區可以為顧客提供多種選擇，但在另外一些地區卻不能，而且雖然一些昂貴的酒在這裡會更便宜，但是這不過是一個副作用，“對普通消費者沒有任何好處”。<ref name=D2Nov1408/>

在挪威拍賣酒一度是非法的，<ref>{{cite web|last=''Dagbladet.no'' |date= December 31, 2008 |title= Vinmonopolet vil ha samlerauksjoner |url=http://www.dagbladet.no/2008/12/31/nyheter/vin/alkohol/4205246/ }}{{no icon}}</ref>2011年8月，[[挪威議會|挪威議會]]改變了這項法律，新法在2012年1月1日正式生效。<ref name=dnApr11>{{cite web|last=  Bjørndal|first= Bente, ''Dn.no'' |date= April 14, 2011 |url = http://www.dn.no/vinguiden/article2122819.ece |title=Nå blir det vinauksjoner på Polet}}{{no icon}}</ref>這使得2008年12月挪威國家酒局計劃建立的二手[[葡萄酒拍賣|葡萄酒拍賣]]系統得以實現，<ref name=dnDec08>{{cite web|last=  ''Dn.no'' |title= Vinmonopolet vil ha auksjoner |url= http://www.dn.no/vinguiden/article1574138.ece |date= December 31, 2008 }}{{no icon}}</ref>不過挪威國家酒局不會從中獲得任何利益。<ref name=dsJun12>{{cite web|last= Njarga |first= Berit B., ''Dinside.no'' |date= June 15, 2012|title= Vinmonopolet starter vinauksjon |url= http://www.dinside.no/897413/vinmonopolet-starter-vinauksjon }}{{no icon}}</ref>

挪威國家酒局提供超過12,000種產品，而相應的在2009年，[[瑞典酒類專賣局|瑞典酒類專賣局]]只有9,000種，[[芬蘭|芬蘭]]的Alko只有大約3,000種，[[英國|英國]]的[[維特羅斯|維特羅斯]]只有大約1,500種。<ref name=vinforum1-2010>{{Cite journal| last = Dybvik| first = Ola | title = Verdens beste vinbutikk | journal = Vinforum | volume = 24 | issue = 1| pages = 18–21| date = January 2010 }}{{no icon}}</ref>

== 受賄案 ==

1930年的Dysthe案（Dysthesaken）暴漏了挪威國家酒局商品收購程序的不完善，隨後酒局對此做了相應的調整，當時的領導層被指控混淆公司與私人利益。<ref name=fiin-ch3/>這一事件影響了1931年葡萄酒壟斷法（Vinmonopolloven）的頒佈。<ref>{{cite web|last= Lovdata.no |url= http://www.lovdata.no/all/hl-19310619-018.html |title= Vinmonopolloven, LOV 1931-06-19 nr 18: Lov om Aktieselskapet Vinmonopolet }}{{no icon}}</ref>''

2005年的Ekjord案（Ekjordsaken）是挪威國家酒局發生的另一項[[賄賂|受賄]]案，調查指出進口公司Ekjord A/S一直通過為酒局高層安排豪華宴席、住處及送禮來進行賄賂。<ref name=dnJan2005>{{cite web|last= Marthinsen, Tom; Linderud, Esben |first= ''Dagens Næringsliv''  |title= Vinimportør smurte polsjefer |url= http://www.dn.no/vinguiden/article420120.ece  |date= January 20, 2005}}{{no icon}}</ref>後來酒局高層的數人承認確實有上述的一些行為。<ref name=dnJDec205>{{cite web|last= Fadnes, Ole-Morten; Meyer, Henrik D  |first= ''Dagens Næringsliv'' |title= Polsjefen trekker seg - beholder lønnen |url= http://www.dn.no/vinguiden/article663086.ece  |date= December 2, 2005 }}{{no icon}}</ref>2006年，酒局前任CEOKnut Grøholt因此辭去職務，<ref name=dnJDec205/>8月凱·G·亨里克森繼任。<ref name=adr06>Dalheim, Ulf, ''Adresseavisen'' (April 21, 2006). == Vinbladet ==

1988年，挪威國家酒局刊發了《Vinbladet》（意為“葡萄酒雜誌”或者“葡萄葉”）雜誌，并免費發放給顧客。<ref name=fiin-ch13>{{cite book|last = Hamran |first = Olav |coauthors = Myrvang, Christine |title = Fiin Gammel – Vinmonopolet 75 år |year = 1998 |publisher = Tano Aschehoug |isbn = 82-518-3738-3 |pages = 369–399 }}{{no icon}}</ref>這個名字之前曾經作為公司內部刊物名使用了60年整。<ref name=fiin-ch2/>
