'''挪威廣播公司''' （[[挪威文|挪威文]]：Norsk rikskringkasting，簡寫「'''NRK'''」，也称'''挪威国家广播公司'''或'''挪威国家广播电视台'''），是[[挪威|挪威]]的國營公共廣播機構，也是挪威最大的广播电视机构。它由挪威政府出资，向其本土和海外提供[[电台|电台]]、[[电视|电视]]、[[互联网|互联网]]服务。它是該國首屈一指的[[公共媒體|公共媒體]]機構。所有的挪广电视频道和广播频率都可以通过其官网nrk.no在线观看和收听。挪威广播公司是[[欧洲广播联盟|欧洲广播联盟]]的创始成员。

== 財經情況 ==
該公司的出資並非國家預算的一部份，但是主要通過對公司的大股東收取牌照費用以及節目製作和[[DVD|DVD]]銷售等，另外還採用某種贊助形式。

== 公司簡史 ==
1925年，私營廣播公司：'''''Kringkastningselskapet A/S'''''開始在[[挪威|挪威]]國內進行廣播。1933年該公司以[[英國廣播公司|英國廣播公司]]（''[[BBC|BBC]]''）的模式創始──'''''NRK'''''公司。

新公司的總部設在[[奧斯陸|奧斯陸]]，旨在取代的各大城市里的眾多廣播電臺，原先她是覆蓋挪威全國的，在很長的時期內享有廣播專營權，直到1975年這種[[壟斷|壟斷地位]]才被逐漸打破。

== 频道 ==

===电台===
挪广现有13套广播频率，目前除挪威广播一台 
(NRK P1) 尚有模拟广播 (长波) 服务外，其他频率仅通过[[数码声音广播|数码声音广播]]和[[互联网广播|互联网]]传输。挪威广播一台在[[芬马克|芬马克]]设有长波发射台，以193 MHz的频率对挪威北部地区播音。

以下是挪广的广播频率表
* 挪威广播一台 (NRK P1)，主要播出[[新闻|新闻]]和谈话节目。
* 挪威广播一台经典金曲频率 (NRK P1+)，以播放各类经典流行音乐为主。
* 挪威广播二台 (NRK P2)，主要播出谈话和辩论节目。
* 挪威广播三台 (NRK P3)，主要播放[[流行音乐|流行音乐]]。
* 挪威广播三台青年频率 (NRK MP3)，播放流行音乐。
* 挪威广播新闻频率 (NRK Alltid nyheter)，以播放新闻节目为主，夜间会转播[[英国广播公司国际频道|英国广播公司国际频道]]的节目。
* 挪威广播少儿频率 (NRK Super)，播放少儿娱乐节目。
* 挪威广播十三台 (NRK P13)，播放[[摇滚乐|摇滚乐]]和[[独立音乐|独立音乐]]。
* 挪威广播萨米语频率 (NRK Sápmi)，以[[萨米语|萨米语]]广播，主要播出针对[[萨米族|萨米族]]地区的新闻、谈话、交通、音乐节目。
* 挪威广播体育频率 (NRK Sport)，播出[[体育|体育]]类节目。
* 挪威广播民间音乐频率 (NRK Folkemusikk)，主要播出挪威和英国的[[民间音乐|民间音乐]]。
* 挪威广播爵士乐频率 (NRK Jazz)，主要播出[[爵士乐|爵士乐]]。
* 挪威广播古典音乐频率 (NRK Klassisk)，主要播出[[古典音乐|古典音乐]]。
