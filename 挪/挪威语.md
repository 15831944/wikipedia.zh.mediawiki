'''挪威語'''（'''{{lang|no|norsk}}'''）是[[日耳曼語族|日耳曼語族]]的一個分支，普遍通用於[[挪威|挪威]]，也是挪威的[[官方語言|官方語言]]。挪威語與[[瑞典語|瑞典語]]和[[丹麥語|丹麥語]]十分相似，所以這三種語言的人可以互相溝通。由於丹麥語從十六世紀至十九世紀期間一直是挪威地區的標準[[書寫語言|書寫語言]]，以致近代的挪威語發展一直都受著[[愛國主義|愛國主義]]、城鄉隔閡以及[[挪威文學史|挪威文學史]]的爭議所影響。

根據挪威法律和政府政策，現行的挪威語有兩套書寫形式，分別是“[[書面挪威語|書面挪威語]]”（巴克摩挪威語，挪威語：{{lang|no|bokmål}}）和“[[新挪威語|新挪威語]]”（或稱耐諾斯克挪威語，挪威語：{{lang|no|nynorsk}}）。兩種書寫挪威語的形式分別溫和地代表著保守和激進的書寫表達方式。[[巴克摩挪威語|巴克摩挪威語]]和[[耐諾斯克挪威語|耐諾斯克挪威語]]還有他們非官方的版本，分別稱為“{{lang|no|riksmål}}”和“{{lang|no|høgnorsk}}”。

== 歷史 ==

挪威語是從古諾爾斯語發展出來。它與丹麥及瑞典地區的語言分別不大。事實上，維京商人促使了古諾爾斯語在[[俄羅斯|俄羅斯]]及[[歐洲|歐洲]]各地散佈。公元872年，[[哈拉爾一世|哈拉爾一世]]統一了[[挪威|挪威]]。當時開始使用[[盧恩字母|盧恩字母]]。

== 使用情況 ==

在小學和初中裏，大約85.3%接受{{lang|no|bokmål}}的教育，而14.5%則接受{{lang|no|nynorsk}}的教育。而到了初二，學生則要同時學習兩種語言。在挪威的433個政區裏，161個宣佈他們希望使用{{lang|no|bokmål}}與中央政府溝通，116個希望使用 {{lang|no|nynorsk}}，其餘的，即156個表示中立。在挪威的4549份刊物當中，92%使用{{lang|no|bokmål}}，8%使用{{lang|no|nynorsk}}。

== 方言 ==

挪威語方言通常被分為四塊方言區：北挪威方言（nordnorsk）、中部方言（trøndersk）、西部方言（vestlandsk）和東部方言（østnorsk）。有時候內地方言（midlandsmål）和南部方言（sørlandsk）會被視為第五和第六個方言區。這些方言的使用者通常能互相理解對方，但不同方言仍有獨特的口音、文法、單字和句法。

=== 方言語法 ===

==== 未來式 ====
未來式在不同地區有不同用法 "他要去旅行":
: ''Han kommer/kjem til å reise.''
: ''Han blir å reise.''
: ''Han blir reisan.''
: ''Han skal reise.''
==== 句法 ====
不同方言有不同的句法。舉例來說，問句不一定要以「疑問詞」（如何、哪裡、甚麼、誰……）開頭。

''Hvor mye er klokken?''（書面挪威語） / ''Kor mykje er klokka?''（新挪威語），直譯：時間多少？即為「幾點了？」可以用不同的形式表達：
* ''E klokka mykje?''（直譯：時間很多嗎？）（強調「時間」）
* ''E a mytti, klokka?''（直譯：很多嗎，時間？）（強調係動詞「是」）
* ''Ka e klokka?'' （直譯：時間是甚麼？）
* ''Ka klokka e?'' （直譯：時間是甚麼？）（倒裝），「時間」可以用另外一個單字表達，''Ke ure' e?''
* ''Å er 'o?'' （直譯：她是甚麼？）

== 書寫語言 ==

{{Main|書面挪威語|新挪威語}}

就像其他歐洲語言，挪威語有一個官方的諮詢的委員會——挪威語言會（{{lang|no|Språkrådet}}）——挪威文化部決定和認可挪威語官方的文法、拼法和詞語。
