'''裕德龄'''（{{bd|1885年|6月8日|1944年|11月22日|catIdx=Y裕}}），[[汉军正白旗|汉军正白旗]]人，因曾担任[[慈禧太后|慈禧太后]]的御前[[女官|女官]]并用英文写作了这段经历而闻名。

== 早年 ==

1885年（存疑），德龄出生于[[武昌|武昌]]，在兄妹五人中排行第三，在[[荆州|荆州]]、[[沙市|沙市]]长大。其父[[裕庚|裕庚]]为[[外交官|外交官]]，母亲是[[法国人|法国人]]。家族本姓徐，以[[随名姓|随名姓]]的方式称裕德齡。1895年，父亲任出使日本特命全权大臣三年，后又任[[中国驻法国大使列表|驻法公使]]三年。德龄随父在[[日本|日本]]、法国生活六年，不但会外语，還具有开阔的视野和渊博的学识，精通各国国情，曾是现代舞蹈大师[[艾莎道拉·邓肯|邓肯]]的弟子。

== 女官生涯 ==
[[File:The_Qing_Dynasty_Cixi_Imperial_Dowager_Empress_of_China_On_Throne_5.JPG|right]]、德龄、[[慈禧太后|慈禧太后]]、[[裕容龄|裕容龄]]、容龄母、[[隆裕太后|光绪皇后,静芬]]]]
1902年冬，裕庚任满归国，被赏给[[太仆寺|太仆寺]]卿衔，留京养病。17岁的德龄随父回京。此时列強入侵，[[慈禧太后|慈禧太后]]急欲讨好各国使节和他们的夫人，她从[[奕劻|庆亲王奕劻]]口中得知裕庚的女儿通晓外文及西方[[礼仪|礼仪]]，便下旨召裕庚夫人带同德齡、容齡姊妹入宫觐见，後來德齡与妹妹[[裕容龄|容龄]]一起成为[[紫禁城|紫禁城]]八[[女官|女官]]之一，为慈禧与西方国家使节夫人们交往担任[[翻译|翻译]]。直到1905年3月因父病才离宫到上海。期间慈禧太后曾有意将其许配给[[荣禄|荣禄]]之子巴龙，但为[[光绪帝|光绪帝]]设计解脱。

== 出宫之后 ==

1907年5月21日，德龄与[[美国|美国]]驻沪领事馆副领事[[迪厄斯·怀特|迪厄斯·怀特]]结婚。1915年，随夫赴美。

在美开始用[[英文|英文]]撰写回忆录和纪实文学作品，披露了许多慈禧及清宫的生活情景和晚清政局见闻，都是第一手资料，很受西方读者欢迎(但亦有批評稱德齡所著謬誤頗多，如把將[[珍妃|珍妃]]推入井中之事歸罪於[[李連英|李連英]]而非[[崔玉貴|崔玉貴]])，也具有丰富的历史价值。著作被[[顾秋心|顾秋心]]、[[秦瘦鸥|秦瘦鸥]]等译为中文，在《[[申报|申报]]》等媒体上刊载，影响较大。

抗战期间，参与[[宋庆龄|宋庆龄]]发起的[[保卫中国同盟|保卫中国同盟]]活动。

晚期則一個人獨居加州柏克萊市，擔任柏克萊大學的漢語教師。1944年11月22日早晨，在加州伯克利死于车祸，於Bancroft Way和Telegraph Ave交叉口被疾駛而來的雜貨卡車撞上。
