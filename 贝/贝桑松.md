'''贝桑松市'''（{{lang-fr|Besançon}}）位于[[法国|法国]]东部邻近[[瑞士|瑞士]]边境，是[[弗朗什孔泰大区|弗朗什孔泰大区]]和[[杜省|杜省]]的首府，人口117,733（1999年）。

[[Image:0_Besançon_-_Doubs_-_Quai_Vauban_(3).JPG|thumb]]

== 地理 ==
[[File:Besançon_NASA.png|thumb]]
贝桑松位于[[法国|法国]]东部[[汝拉山|汝拉山]]边缘，[[杜河|杜河]]沿岸，法国首都[[巴黎|巴黎]]以东325公里，[[勃艮第|勃艮第]]首府[[第戎|第戎]]以东100公里，[[瑞士|瑞士]][[洛桑|洛桑]]西北125公里，[[弗朗什孔泰大区|弗朗什孔泰大区]]的[[Belfort|Belfort]]西南100公里。 

贝桑松的早期城址是位于[[杜河|杜河]]的一个天然的弯曲形成的大约1公里直径的环状范围内，内部平坦，海拔大约250米，南面是海拔高371米的圣艾蒂安山（''Mont Saint-Étienne''），另外6座海拔400米至500米的小山环抱着这座城市：Brégille, Griffon, Planoise, Chaudanne, Montfaucon, 和 Montboucon。有一条运河穿过圣艾蒂安山下的岩石。 

== 体育 ==
贝桑松有一个足球学校
足球奥尔尚普贝桑松的学院是年轻球员，从4至13岁的学院。自2004年由标签法国足协标记的，其目的是为了促进年轻球员在足球场上的各个方面，成为强有力的球员在足球11-14年的高考。
其操作是基于通过游戏进展是在实践的中心。教学方法是通过把孩子学习的心脏功能。这给意义，以推动及让玩家更快的进步，因为他们在现实中，因此更多地参与。最后，孩子们一起训练的水平的球员，而不只是他们的年龄，使他们无法停止他们的进展。
总的目标是推动尽可能多的玩家让他们的蓬勃发展，通过活动的主要乐趣，再投资自己的学习其他活动长期。
足球学校的校长由毕业于法国足协和欧足联，拥有超过15年的在不同的国家足球学校的经验和不同的法国地区。这也是教育者与法国足协自2007年以来一个教练。[http://www.asobfoot.fr ASOB Foot]

== 气候 ==
贝桑松同时受到[[海洋性气候|海洋性气候]]和[[大陆性气候|大陆性气候]]的影响，降雨频繁，冬季寒冷，夏季干热，年平均气温为10.2 °C。最热的月份是7月（18.9 °C），最冷的是1月（1.6 °C）。贝桑松每年的[[降水量|降水量]]约为1108毫米。降雨最多的月份是5月（111.4毫米），最干旱的是7月（80.5毫米）。有史以来的最高气温，为38.8℃（1983年7月31日），最低气温为 −20.7 °C（1985年1月1日）。

== 人口 ==
根据1999年人口统计，贝桑松市人口为117,733人，低于历史最高值120,315人（1975年）。截止2004年2月，该市市内人口估计为114,900人。贝桑松建成区面积122km2，包括11个[[市镇|市镇]](''communes'') ，人口134,376人；都会区(''[[aire_urbaine|aire urbaine]]'')面积1652km2，包括234个[[市镇|市镇]]，人口222,381人。

== 政治 ==
贝桑松是法国[[弗朗什孔泰大区|弗朗什孔泰大区]]的首府，该大区包括4个省：杜省、[[上索恩省|上索恩省]]、[[汝拉省|汝拉省]]和[[貝爾福地區|貝爾福地區]]。

== 经济 ==
贝桑松以[[微技术|微技术]]和[[钟表|钟表]]业著称，主办一年两次的微技术领域的交易会。贝桑松是法国[[钟表|钟表]]工业的首都，在1970年代，其传统的钟表业受到来自来自亚洲地区的石英手表的冲击，而经历重大危机。贝桑松最负盛名的LIP 表厂的工人拒绝关闭工厂，成立了工人合作社经营工厂。这一行动非常有名，使得贝桑松具有激进左翼城市的名声。合作社不久倒闭，到1980年代，贝桑松的钟表业才在其历史声誉的基础上得以复苏，包括定制手表，高品质手表和时尚用品。1990年代以来，该市建立起法国各个领域领先技术中心的声誉，包括在电信和生物技术方面。 

== 教育 ==
贝桑松是[[弗朗什孔泰大学|弗朗什孔泰大学]]的所在地。截至2006年，大约有20,000名学生就读于该大学，其中包括约3000名外国留学生。该市还拥有[[国立高等机械和微技术学院|国立高等机械和微技术学院]](École Nationale Supérieure de Mécanique et des Microtechniques，ENSMM)，这所学校在[[微技术|微技术]]和[[机械|机械]]领域拥有良好声誉；以及世界著名的应用语言学中心，向非母语学习者教授10种语言（法语，阿拉伯语，汉语，英语，德语，意大利语，日语，葡萄牙语，俄语，西班牙语），以及向有要求者教授任何其他已知的语言，每年接受超过3000名来自世界各地的学生。

== 名胜 ==
[[File:StJeanCathedral_Besancon.jpg|right]]内部]]
[[File:Citadelle_Besancon.JPG|250px]]
贝桑松拥有法国主要城市中最美丽的老城区之一。[[杜河|杜河]]的“环”（"la Boucle"）环绕着老城区，而[[沃邦|沃邦]]的城堡戴在颈部。老城区如同古典石头建筑的合奏曲，其中有些可以追溯到[[中世纪|中世纪]]，另一些可追溯到西班牙文艺复兴。其中参观者较多的历史古迹有： 

* 一些罗马遗迹，
* 16世纪格朗韦勒宫 Palais Granvelle,
* 沃邦的[[贝桑松城堡|贝桑松城堡]]
* [[贝桑松主教座堂|圣若望主教座堂]]
* 几座西班牙文艺复兴风格建筑
* 玛达肋纳教堂（Église de la Madeleine）
* 河滨
* [[贝桑松_猶太會堂|贝桑松 猶太會堂]]

=== 其他 ===
* 时间博物馆
* 弗朗什孔泰博物馆
* 自然史博物馆，包含一个动物园，水族馆，昆虫馆，以及气候展览

== 交通 ==
[[贝桑松维奥特站|贝桑松维奥特站]]和[[贝桑松-弗朗什-孔泰TGV站|贝桑松-弗朗什-孔泰TGV站]]是贝桑松的两个主要铁路车站，也是两大主要交通线的交汇点：东北-西南路线，沿着杜河河谷，连接[[德国|德国]]、北欧和[[里昂|里昂]]、西南欧；南北路线，连接法国北部、[[荷兰|荷兰]]和[[瑞士|瑞士]]。它是斯特拉斯堡-里昂（德国-西班牙）线路的要站，也有[[高速列车|高速列车]]直通巴黎、[[巴黎－夏尔·戴高乐机场|巴黎－夏尔·戴高乐机场]]和[[里尔|里尔]]。贝桑松自身没有商业机场，距离[[巴塞尔－米卢斯－弗赖堡欧洲机场|巴塞尔－米卢斯－弗赖堡欧洲机场]]和[[里昂圣-埃克苏佩里机场|里昂圣-埃克苏佩里机场]]2个国际机场大约2小时路程。
