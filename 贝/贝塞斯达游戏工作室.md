'''贝塞斯达游戏工作室'''（{{lang-en|Bethesda Game Studios}}，缩写'''BGS'''）是美国[[游戏开发者|游戏开发商]]，[[贝塞斯达软件|贝塞斯达软件]]的分部，总部位于[[馬里蘭州|馬里蘭州]][[羅克維爾_(馬里蘭州)|羅克維爾]]。工作室成立于2001年，分拆自贝塞斯达软件的开发部门，贝塞斯达软件本身仅保留出版功能。工作室由[[执行制作人|执行制作人]][[托德·霍华德|托德·霍华德]]和工作室总监阿什利·程领导。BGS经营三个子工作室，一个在[[蒙特利尔|蒙特利尔]]，两个在[[德克萨斯州|德克萨斯州]]，截至2018年7月雇佣了400名员工。

== 历史 ==
2001年，[[贝塞斯达软件|贝塞斯达软件]]的母公司[[ZeniMax_Media|ZeniMax Media]]决定将贝塞斯达软件的两个运营部门（开发部和出版部）分离。随后，贝塞斯达软件保留了出版权，而开发组则转移至新成立的贝塞斯达游戏工作室<ref>{{cite web|url=https://www.youtube.com/watch?v=QKn9yiLVlMM|title=The History of Bethesda Game Studios - Elder Scrolls / Fallout Documentary|first=|last=Noclip|date=2018-06-05|publisher=|accessdate=2018-11-12|via=YouTube}}</ref>。

2015年12月9日，ZeniMax Media宣布成立贝塞斯达游戏工作室蒙特利尔，位于魁北克省蒙特利尔，由{{link-en|Behaviour Interactive}}的前负责人Yves Lachance领导，工作室将拓宽BGS在全游戏平台上的游戏投资。<ref>{{cite web |url=http://www.gameinformer.com/b/news/archive/2015/12/09/bethesda-opens-new-studio-in-montreal.aspx |title=Bethesda Opens New Studio In Montreal |first=Matthew |last=Kato |work=[[Game_Informer|Game Informer]] |date=2015-12-09 |accessdate=2015-12-10}}</ref>

2018年3月9日，总部位于[[德克萨斯州|德克萨斯州]][[奥斯汀|奥斯汀]]的另一家ZeniMax Media子公司BattleCry Studios更名为贝塞斯达游戏工作室奥斯汀，成为BGS的一部分<ref>{{cite web |url=https://www.gamesindustry.biz/articles/2018-03-09-battlecry-studios-becomes-third-bethesda-game-studios-office |title=BattleCry Studios becomes third Bethesda Game Studios office |first=Christopher |last=Dring |date=2018-03-09 |accessdate=2018-03-09 |work=[[Eurogamer|GamesIndustry.biz]]}}</ref>。

2018年8月10日，Escalation Studios更名为贝塞斯达游戏工作室达拉斯<ref>{{cite web|url=https://variety.com/2018/gaming/news/bethesda-game-studios-dallas-announcement-1202901832/|title=Escalation Studios Is Now Bethesda Game Studios Dallas|website=[[綜藝_(雜誌)|Variety]] |first=Stefanie |last=Fogel |date=2018-08-10|accessdate=2018-08-10}}</ref>。

== 子公司 ==
* 贝塞斯达游戏工作室蒙特利尔，2015年12月成立于魁北克省[[蒙特利尔|蒙特利尔]]。
* 贝塞斯达游戏工作室[[奥斯汀|奥斯汀]]，前身BattleCry Studios成立于2012年10月，是ZeniMax的子公司，于2018年3月成为BGS旗下。
* 贝塞斯达游戏工作室[[达拉斯|达拉斯]]，前身Escalation Studios成立于2007年，于2017年2月被ZeniMax收购，2018年8月成为BGS旗下。

== 开发的游戏 ==
贝塞斯达游戏工作室成立以来，主要为[[電子遊戲機|遊戲機]]和[[个人电脑|个人电脑]]开发[[电子角色扮演游戏|电子角色扮演游戏]][[上古卷轴系列|上古卷轴系列]]和[[異塵餘生系列|異塵餘生系列]]，这些游戏几乎都取得了商业成功。<ref>{{cite web |last1=Petty |first1=Jared |title=Bethesda Says Two Upcoming Games Are as Big as Skyrim and Fallout |url=http://uk.ign.com/articles/2016/06/15/bethesda-says-two-upcoming-games-are-as-big-as-skyrim-and-fallout |website=IGN |date=2016-06-14}}</ref><ref>{{cite web |url=http://www.glixel.com/news/bethesdas-todd-howard-hints-at-3-new-games-w467778 |title=Bethesda's Todd Howard Hints at 3 New Games |first=John |last=Davidson |date=2017-02-18 |accessdate=2017-02-18 |work=[[滾石_(雜誌)|Glixel]]}}</ref>

2015年，工作室凭基于同系列的《[[異塵餘生：庇護所|異塵餘生：庇護所]]》进军[[手机游戏|手机游戏]]市场，至2016年中期吸引了5000万玩家。2017年2月，霍华德表示，继《異塵餘生：庇護所》成功后，他们正在开发另一款手机游戏<ref>{{Cite web |url=http://www.gamespot.com/articles/skyrim-director-teases-next-mobile-game/1100-6447976/ |title=Skyrim Director Teases Next Mobile Game |first=Eddie |last=Makuch |date=2017-02-17 |accessdate=2017-02-17 |work=[[GameSpot|GameSpot]]}}</ref>。2018年游戏正式公布，名称为《[[上古卷轴：刀锋|上古卷轴：刀锋]]》。

2016年，霍华德证实，虽然他们正在开发《[[上古卷轴系列|上古卷轴VI]]》，但仍需要很长时间。与此同时，另外两个重要项目正在开发，预计将在《上古卷轴VI》之前发行<ref>{{cite web |last1=Chalk |first1=Andy |title=The Elder Scrolls 6 is in development, Todd Howard confirms |url=https://www.pcgamer.com/the-elder-scrolls-6-is-in-development-todd-howard-confirms/ |accessdate=2018-04-30 |work=PC Gamer}}</ref>。2018年5月30日，《[[辐射76|辐射76]]》公布<ref>{{cite web |url=https://kotaku.com/bethesda-announces-fallout-76-1826417383 |title=Bethesda Announces Fallout: 76 |last=Schreier |first=Jason |date=2018-05-30 |work=Kotaku|access-date=2018-06-05}}</ref>。[[托德·霍华德|托德·霍华德]]透露，《[[辐射76|辐射76]]》是贝塞斯达游戏工作室的首个在线角色扮演生存游戏。然而，游戏受到了粉丝和评论员的大量批评。2018年6月10日，在贝塞斯达的[[2018年E3电子娱乐展|E3 2018]]发布会上，公司透露正在开发另一个项目《星空》（''Starfield''），这是25年来首个新知识产权。
