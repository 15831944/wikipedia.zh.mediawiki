'''贝鲁特'''（{{lang-ar|'''بيروت'''|Bayrūt}}；{{lang-hy|Պէյրութ|Peyrut}}<ref>Պէյրութ ''Peyrut’'' is in [[Western_Armenian|Western Armenian]], spoken by [[Lebanese_Armenians|Lebanese Armenians]]. In [[Eastern_Armenian|Eastern Armenian]], the variation used in [[Armenia|Armenia]] the city's name is Բեյրութ ''Beyrut’''.</ref> {{lang-el|Βηρυτός|Bērytós}}；{{Lang-la|Berytus}}；{{Lang-he|בְּאֵרוֹת|Be’erot}}；[[阿拉姆語|阿拉姆語]]：Birot בירות；{{lang-en|Beirut}}；{{lang-fr|Beyrouth}}；{{lang-tr|Beyrut}}）是[[黎巴嫩|黎巴嫩]]的[[首都|首都]]，位於[[贝鲁特省|贝鲁特省]]的[[地中海|地中海]]沿岸，是該國最大的海港。因為最近一直沒有進行人口普查，故其具體人口未知。2007年時估計有100萬到200萬之間。該地名首次出現於[[古埃及|古埃及]]的[[阿馬爾奈文書|阿馬爾奈文書]]中，歷史可追溯至公元前15世紀，并一直有人類居住，與鄰國敘利亞的[[大馬士革|大馬士革]]並列為世界最古老城市之一。

貝魯特不僅是黎巴嫩政府所在地，同時對黎巴嫩經濟也有著不可忽視的影響。[[貝魯特中央區|貝魯特中央區]]、[[哈姆拉街|哈姆拉街]]、[[阿什拉斐葉區|阿什拉斐葉區]]等地有眾多公司與銀行。此外還有許多著名出版社、劇院，夜生活豐富，因此也成爲了區域內的文化中心。20世紀末期因[[黎巴嫩內戰|黎巴嫩內戰]]被毀，經歷了大重建。<ref>[http://www.macalester.edu/courses/geog61/jmhamilton/reconstruct.html Reconstruction of Beirut], Macalester College</ref><ref>[http://www.voanews.com/english/archive/2004-10/2004-10-27-voa47.cfm?CFID=234705748&CFTOKEN=74239565 Lebanon's Reconstruction: A Work in Progress], VOA News</ref><ref>[http://worldviewcities.org/beirut/elipsis.html Beirut: Between Memory and Desire] {{webarchive|url=https://web.archive.org/web/20160303175036/http://worldviewcities.org/beirut/elipsis.html |date=2016-03-03 }}, Worldview</ref>

== 历史 ==
[[File:بيروت_القرن_١٩_أسود_أبيض.jpg|thumb]]
[[File:beirut_old.jpg|thumb]]

貝魯特古名為“be'erot”（意即「水井之城」），由[[腓尼基|腓尼基]]人所建立。最早有關貝魯特的歷史文獻記載確切的年代為公元前15世紀左右，此時已經有刻上[[楔形文字|楔形文字]]的黏土板提及這座城市。最古老的殖民是在河中因日益增加的淤積的一個島上。城市在上古被稱為「Berytus」，這個名字在1934年被[[貝魯特美國大學|貝魯特美國大學]]藝術和科學系出版的考古學學報採用。

在公元前140年，貝魯特市被[[狄奧多特·特里豐|狄奧多特·特里豐]]與[[安条克七世|安条克七世]]之間的[[塞琉古帝国|塞琉古帝国]]王位爭奪中摧毀。貝魯特很快在[[希臘文化|希臘文化]]計劃中有條理地重建，並改名''Laodicea in Canaan''，以紀念[[塞琉西王朝|塞琉西王朝]]的一位王后。現在的貝魯特城建築在舊城之上，在1991年[[黎巴嫩|黎巴嫩]]內戰結束之前，只有少數對貝魯特的考古學能夠完成。現在大量位於市中心的古老遺址開放予考古學的研究。

公元前一世紀中期，Berytus（貝魯特古名）的硬幣印上命運女神[[堤喀|堤喀]]的頭像。硬幣背面則是城市的標誌：一隻海豚糾纏船錨。而這個標誌則在15世紀的[[威尼斯|威尼斯]]被印刷業者[[阿尔杜斯·马努提乌斯|阿尔杜斯·马努提乌斯]]所佔去。

在[[羅馬|羅馬]]的[[大希律王|大希律王]]統治下，貝魯特開始變得富裕。

[[第一次世界大戰|第一次世界大戰]]後，[[奧斯曼帝國|奧斯曼帝國]]崩潰，在短暂地成为[[叙利亚王国|叙利亚王国]]的一部分后，贝鲁特及整个黎巴嫩被交由[[法国|法国]]托管。但在法國的管理期間，政府没能处理好各教派的关系，以致在貝魯特造成了宗教關係緊張。而在[[第二次世界大戰|第二次世界大戰]]後，黎巴嫩被賦予了獨立，而貝魯特亦成為了首都。貝魯特一直維持著它作為[[阿拉伯世界|阿拉伯世界]]的文明都市，重要商業中心以及旅遊中心的地位，直至[[黎巴嫩|黎巴嫩]]在1975年爆發了[[黎巴嫩內戰|內戰]]為止。在這場戰爭之間，這個城市分裂為由穆斯林控制的西部和基督徒掌握的東部。而先前成為很多商業及文化活動焦點的市中心，亦頓時變成一個無人地帶。而很多住在這個城市的主要原居民亦紛紛逃亡至其他國家。至戰爭結束後，黎巴嫩人民已經著手重建貝魯特，而這座城市亦恢復它作為中東旅遊、文化中心和主要都市的地位，以及商業、時裝和媒體的中心。

== 地理 ==

貝魯特位於向東深入地中海的半島上，<ref name="autogenerated1">[http://www.macalester.edu/geography/courses/geog261/efarhat/%20beirut/geography.htm Beirut – The Pearl of the Middle East]{{Dead link|date=May 2009}}</ref> 位於黎巴嫩-以色列邊境以北{{convert|94|km}}。<ref>{{cite web| url= http://geography.howstuffworks.com/middle-east/geography-of-beirut.htm| title= Howstuffworks "Geography of Beirut"| publisher= Geography.howstuffworks.com| date= | accessdate= 2009-05-05| archiveurl= https://web.archive.org/web/20090428115539/http://geography.howstuffworks.com/middle-east/geography-of-beirut.htm| archivedate= 28 四月 2009| deadurl= no}}</ref> 兩側是呈三角環繞的[[黎巴嫩山|黎巴嫩山]]，貝魯特省佔地{{convert|18|km2|sqmi}}，城市的都會區則佔地{{convert|67|km2|sqmi}}。<ref name="autogenerated1"/> 其海岸富於變化，有岩灘、沙灘和懸崖等諸多景觀。

===氣候===
貝魯特的氣候屬於[[地中海氣候|地中海氣候]]中的暖夏型（[[柯本气候分类法|Köppen]]：Csa），夏季溫和少雨，春秋寒冷多雨。只有八月被認為是真正的熱月，當月平均最高氣溫為{{convert|32|°C|°F}}，一月和二月最冷，平均最低氣溫為{{convert|11|°C|°F}}。

平均年降雨量為{{convert|893|mm|in}}，主要降水集中於春、秋、冬季。春秋有短期的大暴雨，冬季則是綿延的小雨。除去東部高海拔地區，貝魯特很少有雪。

== 宗教 ==
贝鲁特是地中海沿岸最多样化的城市，这里的宗教有[[基督教|基督教]]（天主教[[马龙派|马龙派]]、[[希腊正教|希腊正教]]、[[希腊天主教|希腊天主教]]、[[亚美尼亚宗徒教会|亚美尼亚宗徒教会]]、亚美尼亚天主教、叙利亚正教、[[罗马天主教|罗马天主教]]及[[基督新教|基督新教]]）和伊斯兰教（[[逊尼派|逊尼派]]、[[什叶派|什叶派]]）以及独立的[[德鲁兹派|德鲁兹派]]。贝鲁特大多数[[犹太教|犹太教]]信徒自1975年战争爆发以后起，都永久地移居[[美国|美国]]，现在主要居住在[[布鲁克林|布鲁克林]]、[[纽约|纽约]]。贝鲁特在[[黎巴嫩|黎巴嫩]]内战中分成几个部分，主要是穆斯林为主的西贝鲁特和[[基督徒|基督徒]]为主的东贝鲁特。現任貝魯特的總主教是蘇法伊爾。

==教育==

貝魯特大多數高等教育機構都是私立的，唯一的公立高等教育機構是[[黎巴嫩大學|黎巴嫩大學]]。<ref name="UnescoEdu">{{RTFlink|[http://www.unesco.org/iau/onlinedatabases/systems_data/lb.rtf Lebanon – Education system]|26.5 [[Kibibyte|KiB]]<!-- application/rtf, 27,199 bytes -->}}, Unesco.org</ref> 有高等教育理事會負責統籌全市的大學、學院。<ref name="UnescoEdu"/>

貝魯特的私立學校有法國-黎巴嫩公立中學（Grand Lycée Franco-Libanais）、[[黎巴嫩美國社區學校|黎巴嫩美國社區學校]]、貝魯特國際學院等。<ref>{{cite web |url=http://www.educalys.edu.lb |title=Educalys website |publisher=Educalys.edu.lb |date= |accessdate=2011-09-05 |archiveurl=https://web.archive.org/web/20110722144453/http://www.educalys.edu.lb/ |archivedate=2011-07-22 |deadurl=yes }}</ref>

== 交通 ==

[[File:Three_Middle_East_Airlines_Airbus_A321.jpg|thumb]]]]
[[贝鲁特-拉菲克·哈里里国际机场|贝鲁特-拉菲克·哈里里国际机场]]位於此地南郊。[[貝魯特港|貝魯特港]]是地中海東部最大的商業港口之一，也是通衢，與[[塞浦路斯|塞浦路斯]]之間有較頻繁渡輪往來。<ref name=bigFire>{{cite web |url=http://www.ikamalebanon.com/info/transportation.htm |title=Transportation & Communication |publisher=Ikamalebanon.com |accessdate=17 January 2010 |archiveurl=https://web.archive.org/web/20100102003806/http://www.ikamalebanon.com/info/transportation.htm |archivedate=2 一月 2010 |deadurl=yes }}</ref>

貝魯特與黎巴嫩及一些主要[[敘利亞城市列表|敘利亞城市]]之間有巴士往來，該城的巴士等公共交通由多家私人公司負責，但也有上市公司“OCFTC”運營的巴士，該公司也經營鐵路運輸業。<ref>[http://www.lonelyplanet.com/worldguide/destinations/middle-east/lebanon/beirut/transport Beirut Transport] {{webarchive|url=https://web.archive.org/web/20071024173134/http://www.lonelyplanet.com/worldguide/destinations/middle-east/lebanon/beirut/transport |date=2007-10-24 }}</ref>

== 旅游 ==
在战争之前，贝鲁特是很受欢迎的[[旅游|旅游]]目的地，有“[[中东|中东]]的[[巴黎|巴黎]]”之美誉，有多家国际大酒店和繁荣的夜生活。在最近几年里，这个城市开始重建旅游业，在毁坏的城市中心开始建设主要建筑物和恢复工作，并建了一个新的散步道。在城市的其余部分还保守时，已经可以穿着西式服饰，包括短裤和短裙了，但在宗教性建筑里仍然是不允许的。

除了城市本身外，附近还有一些考古场所，包括 [[巴勒贝克|巴勒贝克]] 和[[比布鲁斯|比布鲁斯]]，据说是世界是最古老的连续有人居住的城市。在[[Faraya_Mzaar_Kfardebian|Faraya Mzaar Kfardebian]]这样的胜地，可以在山上进行冬季[[滑雪|滑雪]]。
