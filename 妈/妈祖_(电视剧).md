{{engname|《妈祖》|Mazu}}，[[中国大陆|中国大陆]]电视剧，导演[[路奇|路奇]]，主演[[刘涛|刘涛]]、[[严屹宽|严屹宽]]、[[刘德凯|刘德凯]]、[[黄嘉乐|黄嘉乐]]、[[呂晶晶|呂晶晶]]、[[赵鸿飞|赵鸿飞]]、[[林心如|林心如]]、[[魏宗万|魏宗万]]等。该剧以[[华人|华人]]地区“妈祖信仰”改编而成，于2012年12月31日在[[中央电视台电视剧频道|中央电视台电视剧频道]]首播。该剧与2013年7月31日在[[中视|中视]]播出。

== 剧情 ==
[[北宋|北宋]]，东南沿海地区经常发生水灾，赤脚大仙奏明玉帝，请求加封一位掌管海事的神仙，玉帝允诺。王母娘娘和观音菩萨经过查询，让福建莆田湄洲岛林愿的妻子王氏怀上了女婴，即未来的海神，赤脚大仙担任其师傅，教授其法术。东海水灾都是东海龙王的儿子睚眦引起的，為拯救苍生，林默和师傅赤脚大仙几经周折，终於制伏睚眦，而林默娘也被封為海神。

== 史实差异 ==
历史上妈祖是[[处女|处子]]之身而没有嫁人，该剧中妈祖也是[[处女|处子]]之身而妈祖卻有嫁人，但是，是嫁而不婚，婚而不娶，夫妻之名，無夫妻之實。<ref>{{cite web |url=http://lady.163.com/13/0108/15/8KN5V4AN00264M4F.html |title=古装剧与史实不符引"挑刺" "妈祖"嫁人惹争议 |accessdate= 2013-01-08|publisher=网易 |work= |language =zh-cn}}</ref>

该剧中妈祖用“[[英语|英语]]”和外国人交谈是穿帮镜头，没有遵循历史史实，因为北宋时期和宋朝商业往来的是[[阿拉伯|阿拉伯]]和[[波斯|波斯]]，[[英语史|现代英语]]当初还没有啟發，当时应用[[古英语|古英语]]进行交谈。<ref>{{cite web |url=http://edu.sina.com.cn/cet/2013-01-08/1535368391.shtml |title= 《妈祖》飚英文遭吐槽：考四六级拜妈祖|accessdate=2013年1月8日 |publisher=新浪网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.xinhuanet.com/photo/2013-01/09/c_124205018.htm |title=神话剧妈祖飆英文雷翻网友:走国际化路线全能女神 |accessdate=2013年1月9日  |publisher=新华网 |work= |language =zh-cn}}</ref>中国历史题材的影视有“英文”的还有《钟馗传说》（唐朝）、《大话西游》（唐朝）。<ref>{{cite web |url=http://news.xinhuanet.com/photo/2013-01/09/c_124205019.htm |title= 神仙古人说英语不止妈祖一人 影视剑走偏锋国际范|accessdate=2013年1月9日  |publisher=新华网 |work= |language =zh-cn}}</ref>

传说[[千里眼|千里眼]]、[[顺风耳|顺风耳]]被妈祖打败所收服，而剧中却为玉帝所赐，列位仙班比妈祖早。

== 花絮 ==
劇中林默娘（刘涛）與吴宗伦（严宽）一句劇中台词“你可以嫁而不婚，我可以婚而不娶，我们有夫妻之名，各居其屋，无夫妻之实。”引发了刘涛丈夫[[王珂|王珂]]在[[新浪微博|新浪微博]]叫屈：“苍天啊？这说的不是一个月来我这[[屌丝|屌丝]]过的日子吗？由於刘老师最近拍摄紧张，已经很久没逮著她人了！@严宽，你怎麼还出这种主意。”而刘涛在新浪微博回应丈夫王珂:“[[羊肉|羊肉]]吃多了[[上火|上火]]”。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2013-01-05/10323826992.shtml?c=from_123duba_focusnews_ent |title=刘涛老公诉相思情：一个月无夫妻之实 |accessdate=2013年1月5日 |publisher=新浪网 |work= |language =zh-cn}}</ref>
