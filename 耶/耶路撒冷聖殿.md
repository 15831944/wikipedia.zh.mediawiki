'''聖殿'''（{{Lang-he|בית המקדש}}），是古代[[以色列人|以色列人]]最高的祭祀場所。在聖殿建成之前，以色列人在[[會幕|會幕]]中進行祭祀。直到聖殿建成，聖殿隨即成為敬奉[[上帝|上帝]]的崇拜場所，也成了[[以色列人|以色列民族]]的象徵。歷史上的[[耶路撒冷|耶路撒冷]]曾先後出現兩座聖殿，現已被毀。

== 第一聖殿 ==
{{Main|所罗门圣殿}}

在[[以色列王國|以色列王國]]時代，第二代國王[[大衛王|大衛]]為建殿的材料作預備，由繼位的國王[[所羅門|所羅門]]在耶路撒冷[[摩利亞山|摩利亞山]]（[[圣殿山|圣殿山]]）興建第一座聖殿。故第一座聖殿又稱'''所羅門聖殿'''，聖殿長約30米，寬約10米，高約15米。聖殿落成後，隨即成為敬奉[[神名|上主]]的崇拜場所，也成了以色列人民族的象徵。直到公元前586年，[[新巴比倫王國|新巴比倫王國]]摧毀耶路撒冷，聖殿被毀。

== 第二聖殿 ==
{{Main|第二圣殿}}

公元前539年，[[波斯|波斯帝國]]推翻了新巴比倫王國。波斯王[[居鲁士二世|居魯士大帝]]頒旨，讓猶太人在公元前537年回耶路撒冷重建聖殿。由於重建過程並不順利，第二座聖殿於公元前515年完工。在公元前37年，聖殿由[[大希律王|大希律王]]得到擴建。直至公元70年，猶太人反抗[[羅馬帝國|羅馬帝國]]暴政，結果[[提多將軍|提多將軍]]率軍圍攻耶路撒冷，聖殿被焚毀，僅留下西邊一道圍牆（[[哭牆|哭牆]]）。

焚毀第二聖殿後，[[羅馬帝國|羅馬帝國]]曾在聖殿遺址上蓋一座[[羅馬|羅馬]]的最高神祇[[朱庇特|朱彼特]]之[[神廟|神廟]]，將[[猶太省|猶太省]]改名為[[巴勒斯坦|巴勒斯坦]]。公元637年，[[伊斯蘭教|伊斯蘭教]]的[[阿拉伯帝國|阿拉伯帝國]]興起，佔領巴勒斯坦後，在聖殿遺址上興建[[奧瑪清真寺|奧瑪清真寺]]，在毗鄰處興建[[阿克薩清真寺|阿克薩清真寺]]。

== 第三聖殿 ==
{{Main|第三圣殿}}

2001年7月29日，[[猶太教|猶太教]]极端團体“圣殿山忠誠者”在[[耶路撒冷旧城|耶路撒冷旧城]][[馬格里布門|馬格里布門]]外的一個停車場舉行簡短儀式，紀念[[猶太|猶太]]第一和第二圣殿遭到外族毀滅，并象征性地為重建「第三圣殿」舉行奠基[[儀式|儀式]]。結果奠基儀式引發了又一次[[巴以衝突|巴以衝突]]，終導致41名示威者和15名以警察受傷，另有27名示威者被捕。

== 現況 ==
第二座聖殿在公元70年被焚毀後，只餘[[西墙|西牆]]部分。公元691年，西牆成為[[伊斯蘭教|伊斯蘭教]]的[[圆顶清真寺|奧瑪清真寺]]和[[阿克薩清真寺|阿克薩清真寺]]的圍墙，聖殿原址現已是伊斯蘭教[[圆顶清真寺|奧瑪清真寺]]。而現今的西牆則成為露天的[[猶太會堂|猶太會堂]]。1981年，西墙，即哭牆，被列入[[世界遗产|世界文化遺產]]目錄。
