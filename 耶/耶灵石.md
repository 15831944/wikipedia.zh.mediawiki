'''耶灵石'''（Jelling stones）是公元10世纪巨大的经雕刻的[[卢恩石|卢恩石]]（runestone），发现于[[丹麦|丹麦]][[耶靈|耶靈]]。两块耶灵石中较古老的那块由国王[[老高姆|老高姆]]（Gorm the Old）为纪念其妻[[赛拉|赛拉]]（Thyra）所立。国王高姆是全丹麦的首位国王。这些石头上的[[卢恩字母|卢恩字母]]铭文被视为在丹麦最著名者。<ref>{{cite book|last=|first=|authorlink=|coauthors=|title=Jelling stones|year=2008|publisher=[[Encyclopædia_Britannica|Encyclopædia Britannica]]|quote=|url=http://www.britannica.com/bps/topic/302453/Jelling-stones|isbn=}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

两块石头中较大的那块是由国王高姆之子，[[哈拉爾一世_(丹麥)|蓝-{牙}-哈拉尔]]（Harald Bluetooth）为纪念其双亲并庆祝其征服丹麦与[[挪威|挪威]]，以及他令丹麦人皈依基督教而立<ref>[http://www.megalithic.co.uk/article.php?sid=17683 megalithic.co.uk] C. Michael Hogan, "Jelling Stones", Megalithic Portal, editor Andy Burnham</ref>。

这些石头位于地处两座坟冢之间的[[耶灵|耶灵教堂墓地]]。它们代表了丹麦自本土的[[北欧异教|北欧异教]]（Norse paganism）至[[基督教化|基督教化]]过程的转换阶段。这些石头被强烈视为丹麦作为一民族国家的创始。

在历经千年风霜雨雪之后，裂纹开始出现。2008年11月15日[[UNESCO|UNESCO]]专家检测了这些石头以确定其状况。将该石移至室内展示厅以防止进一步受天气损害的计划正在进行。<ref>{{cite web |url=http://www.dr.dk/Nyheder/Indland/2008/11/16/085024.htm |title=Eksperter: Runestenene skal reddes |accessdate=16 November 2008 |work= |publisher=Danmarks Radio |date=16 November 2008}}</ref>

== 高姆的卢恩石 ==
[[File:Runenstein_Gorm_2.jpg|left]]

较古老和较小的那块卢恩石写着：“国王高姆造此纪念碑以纪念其妻赛拉，丹麦的装饰。”（''"King Gormr made this monument in memory of Thyrvé, his wife, Denmark's adornment."''）（[[Rundata|Rundata]], DR 41）在[[后弗萨克文|后弗萨克文]]（Younger Futhark）[[卢恩字母|卢恩文字]]中的[[古诺尔斯语|古诺尔斯文]]（Old Norse）铭文展现如下：
[[File:Runenstein_Gorm_1.jpg|100px]]

: ''(A面)'' ᛬ ᚴᚢᚱᛘᛦ ᛬ ᚴᚢᚾᚢᚴᛦ ᛬
: ᛬ ᚴ(ᛅᚱ)ᚦᛁ ᛬ ᚴᚢᛒᛚ ᛬ ᚦᚢᛋᛁ ᛬
: ᛬ ᛅ(ᚠᛏ) ᛬ ᚦᚢᚱᚢᛁ ᛬ ᚴᚢᚾᚢ
: ''(B面)'' ᛬ ᛋᛁᚾᛅ ᛬ ᛏᛅᚾᛘᛅᚱᚴᛅᛦ ᛬ ᛒᚢᛏ ᛬

音译：
: ''(A面)'' : kurmR : kunukR :
: : k(ar)þi : kubl : þusi :
: : a(ft) : þurui : kunu
: ''(B面)'' | sina | tanmarkaR | but |

: (Jacobsen & Moltke, 1941-42, DR 41)

== 蓝牙哈拉尔的卢恩石 ==
[[File:Runenstein_Blauzahn_2.jpg|thumb]]）。]]

两块卢恩石中较大的那块写着：“国王哈拉尔命造此纪念碑以纪念其父高姆和其母赛拉；为自己赢得了全丹麦和挪威并令丹麦人成为基督徒的哈拉尔。”（''"King Haraldr ordered this monument made in memory of Gormr, his father, and in memory of Thyrvé, his mother; that Haraldr who won for himself all of Denmark and Norway and made the Danes Christian."'' ）（[[Rundata|Rundata]], DR 42）

: ᚴᚢᛒᛚ ᛬ ᚦᛅᚢᛋᛁ ᛬ ᛅᚠᛏ ᛬ ᚴᚢᚱᛘ ᚠᛅᚦᚢᚱ ᛋᛁᚾ
: ᛅᚢᚴ ᛅᚠᛏ ᛬ ᚦᚭᚢᚱᚢᛁ ᛬ ᛘᚢᚦᚢᚱ ᛬ ᛋᛁᚾᛅ ᛬ ᛋᛅ
: ᚼᛅᚱᛅᛚᛏᚱ (᛬) ᛁᛅᛋ ᛬ ᛋᚭᛦ ᛫ ᚢᛅᚾ ᛫ ᛏᛅᚾᛘᛅᚢᚱᚴ
: ''(B面)''
: ᛅᛚᛅ ᛫ ᛅᚢᚴ ᛫ ᚾᚢᚱᚢᛁᛅᚴ
: ''(C面)''
: ᛫ ᛅᚢᚴ ᛫ ᛏ(ᛅ)ᚾᛁ (᛫ ᚴᛅᚱᚦᛁ ᛫) ᚴᚱᛁᛋᛏᚾᚭ

: (Jacobsen & Moltke, 1941-42, DR 42)
