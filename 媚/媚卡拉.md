'''媚卡拉'''<ref name="hj">{{Cite web |url = https://th.hujiang.com/new/p626835/ |title = 台风“威马逊”取名字自泰国，你造啥意思咩？ |author = 沪江泰语 |date = 2014-07-23 |publisher =  |language = zh-cn |accessdate = 2017-09-05 |quote = }}</ref>（{{lang-th|เมขลา}}），又名'''玛尼媚卡拉'''<!--'''摩尼昧迦罗'''-->（{{lang-th|มณีเมขลา}}，{{lang-km|មណីមេខលា}}），是[[东南亚|东南亚]]佛教神话中的[[闪电|闪电]]女神和[[海洋|海洋]]女神，相当于[[中国民間信仰|中国民間信仰]]中的[[电母|电母]]。

== 传说 ==
媚卡拉喜欢拿着一颗[[如意宝珠|如意宝珠]]{{noteTag|有一说是[[红宝石|红宝石]]，但大多数造像手持如意宝珠。}}抛在各个云朵之间上玩，有一个叫[[拉玛逊|拉玛逊]]的[[夜叉|夜叉]]用[[斧头|斧头]]作为武器，看到宝珠请求媚卡拉索要，但是不愿意。这时拉玛逊很生气，扔着斧头追逐，媚卡拉每次都会及时闪避。

媚卡拉的宝珠发出的光明是闪电，而拉玛逊斧头劈开天空的声音是雷声，因此就有了雷电。<ref>{{cite web |url=http://www.thairath.co.th/column/pol/chuckthong/398279|title=เมขลาล่อแก้ว|author=กิเลน ประลองเชิง|date=24 มกราคม 2557|work= |publisher=ไทยรัฐ|accessdate=26 มกราคม 2557}}</ref>

== 流行文化 ==
* 2017年11月18日，于美国[[拉斯维加斯|拉斯维加斯]]举行的2017[[环球小姐|环球小姐]]比赛中，选手Maria以“闪电女神媚卡拉戏珠”为主题的民族服饰亮相。<ref>{{cite web |url=http://www.udnbkk.com/article-244502-1.html |title=環姐民族服飾PK Maria驚艷四座 |last= |first= |date=2017-11-23 |website=泰国世界日报 |publisher= |access-date= |quote=}} </ref>
* 泰语[[教科书|教科书]]《[[玛尼和她的朋友们|玛尼和她的朋友们]]》在小学三年级下册的第16课中有提到过媚卡拉和拉玛逊的故事。<ref>{{cite web |url=http://topicstock.pantip.com/library/topicstock/2009/05/K7910577/K7910577.html |title=Mr. Fusion ภูมิใจเสนอ "มานะ มานี ปิติ วีระ ชูใจ" ( ป.3 เล่ม 2 ) |last= |first= |date=2009-05-27 |website=Pantip |publisher=Mr. Fusion |access-date= |quote=}} </ref>
* [[泰国气象局|泰国气象局]]使用媚卡拉和拉玛逊的名字作为台风名称，如[[台风米克拉_(2015年)|台风米克拉 (2015年)]]、[[台风威马逊_(2014年)|台风威马逊 (2014年)]]。<ref name="hj" />
* 泰国[[MCOT_HD|MCOT HD]]电视台曾在1950年代使用黑白的媚卡拉图像作为台标。
* 泰国有一个娱乐电视奖项名为“{{link-th|媚卡拉奖|รางวัลเมขลา}}”。
