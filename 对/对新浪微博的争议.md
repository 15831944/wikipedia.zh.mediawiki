== 網路霸凌 ==

===侧面抹黑湖南卫视===
自[[新浪微博|新浪微博]]与[[湖南卫视|湖南卫视]]的对手[[浙江卫视|浙江卫视]]达成媒体跨界融合战略合作后<ref>[http://tech.sina.com.cn/i/2017-04-13/doc-ifyeimqy1167215.shtml 浙江卫视与新浪达成媒体跨界融合战略合作]</ref>，新浪微博等相关媒体到处充斥着有关湖南卫视的不利信息，甚至误读各种机构关系，包括一些新浪的下属官微（如微博“新浪娱乐”）亦会对湖南卫视以及下辖产业进行诽谤爆料，且微博亦会被持续保留，反而浙江卫视的各种负面舆论则会被全网屏蔽。

=== 热搜版块遭下线整改 ===
2018年1月27日，[[国务院新闻办公室|国家网信办]]、北京市网信办针对新浪微博对用户发布违法违规信息对该企业负责人进行约谈。北京网信办相关负责人指出，新浪微博违反国家有关互联网法律法规和管理要求，传播违法违规信息，存在严重导向问题，持续传播炒作导向错误、低俗色情、民族歧视等违法违规有害信息，对网上舆论生态造成恶劣影响，责令其立即自查自纠，全面深入整改<ref>{{cite news|title=北京网信办约谈新浪微博：热搜榜等版块暂时下线整改|url=http://news.sina.com.cn/o/2018-01-27/doc-ifyqyesy2860240.shtml|publisher=新浪新闻中心|date=2018-01-27|accessdate=2018-01-27}}</ref>。新浪微博负责人表示，将严格落实网信部门管理要求，对问题突出的热搜榜、热门话题榜、微博问答功能、热门微博榜明星和情感版块、广场头条栏目情感版块暂时下线一周进行整改，于2月3日恢复上线。

1月28日，新浪微博细化整改措施，并根据1月以来的统计情况，公布了刷榜行为突出的话题与热搜词31个，包括“王乐乐微博”、“魏千翔”等。新浪微博表示，以上相关明星、节目、事件名称，3个月内不能再上热门话题榜和热搜榜，相关账号3个月内不能主持话题<ref>{{cite news|title=刷榜话题及主持人3个月禁上热搜|url=http://news.sina.com.cn/c/2018-01-29/doc-ifyqyesy3365704.shtml|publisher=北京商报|date=2018-01-29|accessdate=2018-01-29}}</ref>。消息传出后，涉事明星[[周笔畅|周笔畅]]、[[张若昀|张若昀]]均表示，话题发起人、主持人及被查出的刷榜行为均非二位明星所在工作室操作<ref>{{cite news|title=周笔畅张若昀存在刷热搜榜的行为?工作室如此回应|url=http://ent.sina.com.cn/zz/2018-01-29/doc-ifyqyesy3505476.shtml|publisher=新浪娱乐|date=2018-01-29|accessdate=2018-01-29}}</ref>。

2月3日21时，新浪微博热搜版块经整改后重新上线。整改后的微博热搜在多个板块都增加了一条置顶内容，同时新增“[[习近平新时代中国特色社会主义思想|新时代]]”栏目。另外，微博热搜原有的明星娱乐八卦事件大幅度减少，侧重关注重要新闻、[[正能量|正能量]]事件和人物<ref>{{cite news|title=微博热搜“消失”一周重新上线 弃低俗挤水分|url=http://news.youth.cn/sh/201802/t20180204_11376231.htm|publisher= 中国青年网 |author=杨青山 |date=2018-02-04|accessdate=2018-02-04}}</ref>。

== 霸王著作權條款 ==
2017年改版的新浪微博《微博服務使用協議》1.3款規定：
{{Cquote2|用戶在微博上發布信息，等於授權微博平台作為該內容的「獨家發布平台」，用戶所發表的微博內容僅在微博平台上予以「獨家展示」。}}
儘管被法律界人士認爲是霸王條款，不具法律效力，新浪微博事後也承認條款不當並作出修改，但該條款已經造成衆多在新浪微博上分享原創作品的人士移除微博內容並離開微博。<ref>{{cite web|title=新浪微博推霸王條款　用戶作品成微博「獨家發布」禁轉發其他平台|url=https://www.hk01.com/%E5%85%A9%E5%B2%B8/119576/%E6%96%B0%E6%B5%AA%E5%BE%AE%E5%8D%9A%E6%8E%A8%E9%9C%B8%E7%8E%8B%E6%A2%9D%E6%AC%BE-%E7%94%A8%E6%88%B6%E4%BD%9C%E5%93%81%E6%88%90%E5%BE%AE%E5%8D%9A-%E7%8D%A8%E5%AE%B6%E7%99%BC%E5%B8%83-%E7%A6%81%E8%BD%89%E7%99%BC%E5%85%B6%E4%BB%96%E5%B9%B3%E5%8F%B0|website=香港01|language=en}}</ref>

== 殭屍粉絲 ==
'''[[殭屍|殭屍]]粉絲'''，又稱'''死粉絲'''，新浪對其定義為：一些长期没有动态、同一IP地址申请多个微博号码的用户<ref name="xinhua_2010May26">{{Cite web|url=http://news.xinhuanet.com/newmedia/2010-05/26/c_12143395.htm|title=微博“粉丝”网购产业链面临断裂|accessdate=2011-03-04|author=北京商报|work=新华网|date=2010年5月26日|language=zh-hans}}</ref>。殭屍粉絲會對認證用戶（即微博上帶V的用戶）加關注，非認證用戶（微博不帶V者）則不會被加關注。新浪公司解释這是手机用户注册时由系统自动产生的<ref name="oeeee_2010Aug29">{{Cite web|url=http://www.oeeee.com/a/20100829/926186.html|title=陈Sir回应关闭微博因由：难忍“僵尸粉”不再“织围脖”|accessdate=2011-03-04|author=沙龙|work=南方网|date=2010-08-29|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20110826031949/http://www.oeeee.com/a/20100829/926186.html|archivedate=2011-08-26}}</ref>。但網上有部分用戶表示，新注册的帐号會在完全不知情的情况下自动关注了很多名人<ref name="eNet_2011Feb18">{{Cite web|url=http://www.enet.com.cn/article/2011/0218/A20110218826213.shtml|title=微博“僵粉”频遭抱怨 注册机制引争议|accessdate=2011-03-04|author=计世网|work=硅谷动力|date=2011年2月18日|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20110221225045/http://www.enet.com.cn/article/2011/0218/A20110218826213.shtml|archivedate=2011年2月21日}}</ref>。

殭屍粉絲已形成一條產業鏈，網絡商店上有販賣殭屍粉絲的商家，以0.2元人民幣一個粉絲的價格販賣殭屍粉絲。但新浪現在已經會對殭屍粉絲進行審查和刪除，產業鏈也面臨斷裂。<ref name="xinhua_2010May26"/>

== 賬戶安全 ==

== 言论自由 ==
{{Original research}}
{{seealso|中國網路審查}}
由於新浪微博在中国境内拥有几亿的註冊用戶，注册用户庞大、微博消息傳播速度极快且消息影響面很廣，所以新浪微博在中國國內和國外的多次重大事件中均進行了极其严厉的[[審查|審查]]，手段均属中国境内微博之首。美国《[[时代周刊|时代周刊]]》曾报道指出，在新浪微博工作的审查员共有700多人<ref>{{Cite web|url=http://www.time.com/time/magazine/article/0,9171,2048171-1,00.html|title=Wired Up|publisher=时代周刊|accessdate=2011-02-17|date=2017-09-14}}</ref>。其中的審查手段包括：
* 對用户发言的內容进行審查。若发现含有敏感词的消息，可能根本无法发出、发送后会显示「微博发布成功，目前服务器数据同步可能会有延迟，请耐心等待1-2分钟。谢谢」但事实上该微博可能已经被直接删除，也可能发出后作者自己看起来一切正常，但别人完全看不到。
* 對搜索功能进行关键词过滤，若搜索含有关键词的字句，则提示「根據相關法律法規和政策，搜索結果未予顯示。」
* 用户的消息无法被公开阅读，只能登录后阅读，限制消息传播范围
* 禁止[[goo.gl|goo.gl]]等[[短链接|短缩链接服务]]和部分网站网址的传播
* 使用[[Unicode|Unicode]]编码形式的[[藏文|藏文]]发送的微博文章，虽然能成功发布，但在半小时至一小时之内，其他用户无法阅读，因为这些内容可能会被进行人工审查以确定是否含有“有害信息”

而新浪微博具体的中國國內外的多次重大事件中均进行了内容審查：
* 2010年7月11日开始，也就是[[2010年廣州撐粵語行動|2010年廣州撐粵語行動]]发生后
* 2010年10月8日及12月10日，也就是[[刘晓波|刘晓波]]宣布获得并得到[[2010年诺贝尔和平奖|2010年诺贝尔和平奖]]后
* 2011年1月25日开始，也就是[[2011年埃及革命|2011年埃及革命]]发生后
* 2011年2月20日开始，也就是[[中國茉莉花革命|中國茉莉花革命]]發生後，新浪微博暂时关闭了搜索功能、发图功能和转发功能，而试图发布相關內容想法的用户账户亦被暂时锁闭<ref name=Sina-Molihua>{{Cite web|url=http://blog.kenengba.com/2011/02/21/891|title=好一朵美丽的敏感词|accessdate=2011-02-21|date=2011-07-08}}</ref>。

==违规开展视听节目服务==
2017年6月22日，[[国家新闻出版广电总局|国家新闻出版广电总局]]已于近日发函，责成属地管理部门根据《互联网视听节目服务管理规定》对新浪微博在不具备《信息网络传播视听节目许可证》的情况下开展视听节目服务予以关停整顿<ref>{{cite news|title=国家新闻出版广电总局要求“新浪微博”、“ACFUN”等网站关停视听节目服务|url=http://www.sapprft.gov.cn/sapprft/contents/6588/338032.shtml|publisher=广电总局网站|date=2017-06-22|accessdate=2017-06-22}}</ref>。新浪微博对此回应称，微博用户上传非节目类视频不受影响<ref>{{cite news|title=新浪微博回应视听节目被关停：非节目类视频无影响|url=http://news.china.com/socialgd/10000169/20170623/30808037.html|publisher=中华网|date=2017-06-23|accessdate=2017-06-23}}</ref>。

6月28日，为配合广电总局整改，新浪微博宣布采取整改措施，其中包括关闭超过15分钟以上的长视频上传功能，以及自动解析、自动播放功能。同时新浪微博还要求用户如需传播视听节目，且已持有《信息网络传播视听节目许可证》的机构，应当按照许可证的各项要求开展业务；用户如需传播电影、电视剧，应当具有《电影片公映许可证》或《电视剧发行许可证》。未持有许可证的用户上传视听节目将予以屏蔽<ref>{{cite news|title=新浪微博回应广电总局：关闭超过15分钟以上长视频上传功能|url=http://news.163.com/17/0628/21/CO2465ST000187VE.html|publisher=澎湃新闻网(上海) |date=2017-06-28|accessdate=2017-06-29}}</ref>。

事发8个多月后，新浪微博再次爆出存在违规从事互联网视听节目服务行为。2018年2月24日，北京市新闻出版广电局、北京市文化市场行政执法总队约谈新浪微博网站负责人。经查，新浪微博存在未持有《信息网络视听节目许可证》擅自从事互联网视听节目服务、传播违反《互联网视听节目服务管理规定》第十六条规定的节目的行为。该网站未能有效履行主体责任，对向用户发布的违法违规视听节目未尽到审查义务，持续传播“炒作导向错误、低俗媚俗”等违法违规的视听节目内容，在内容审查和用户管理等方面存在较大管理漏洞，涉嫌违反《互联网视听节目服务管理规定》《互联网文化管理暂行规定》等相关法律法规。北京市新闻出版广电局、北京市文化市场行政执法总队责令新浪微博立即停止违规行为并限期整改<ref>{{cite news|title=北京依法责令新浪微博、凤凰网等６家网站限期整改 |url=http://www.xinhuanet.com/legal/2018-02/24/c_1122448785.htm|publisher=新华网 |date=2018-02-24 |accessdate=2018-02-25}}</ref>。
