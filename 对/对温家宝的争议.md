'''对温家宝的争议'''介绍与前[[中共中央政治局常委|中共中央政治局常委]]、[[中华人民共和国国务院总理|中华人民共和国国务院总理]][[温家宝|温家宝]]相关的批评和争议。


== 中国经济与社会形势 ==
评论指，温家宝任总理期间，没能继续前任总理[[朱镕基|朱镕基]]的经济改革方案，各地区发展落差和贫富差距增加、农民农田被剥夺，企业家和官僚经由土地买卖和房地产投机的暴利方式致富，同时土地和水的毒化、空气的污染均扩大<ref name=" wjbtz" >{{cite web |url=http://www.dw.de/%E6%B8%A9%E5%AE%B6%E5%AE%9D%E4%BB%BB%E6%9C%9F%E6%9C%AB%E6%AE%B5%E5%BD%A2%E8%B1%A1%E5%8F%97%E6%8D%9F/a-16644818 |title= “温家宝任期末段形象受损” |accessdate=2013年3月18日 |publisher=德国之声 |work= |language=zh-hans}}</ref>。2008年出台的[[四万亿投资计划|四万亿投资计划]]，虽然其本人不断辩解，但大批学者<ref>{{cite web|url=http://finance.sina.com.cn/china/20140409/170418748392.shtml|title=经济学家痛批四万亿计划 政府干预致结构失调|publisher=新浪网|date=2014年4月9日}}</ref>对其批判质疑并未间断，四万亿投资造成产能过剩、通货膨胀、巨额地方债务、银行坏账、生产率水平下降等，使中国错失经济结构调整的最佳时期<ref>{{cite web|url=http://finance.tianhenet.com.cn/2012/0531/297883.shtml|title=中国4万亿投资计划种下了四大恶果|publisher=天和网|date=2012-05-31|deadurl=yes|archiveurl=https://web.archive.org/web/20140429045038/http://finance.tianhenet.com.cn/2012/0531/297883.shtml|archivedate=2014年4月29日|df=}}</ref>，造成长期型结构性破坏<ref>{{cite web|url=http://jjckb.xinhuanet.com/2014-04/10/content_499484.htm|title=经济学家连番质疑四万亿计划 或导致结构失调|publisher=新华网|date=2014-04-10}}</ref>。中国经济学家[[张维迎|张维迎]]认为，胡温十年，中国的社会理念发生巨大倒退，从“民进国退”倒退到“国进民退”；从“发展是硬道理”倒退到“和谐社会”；从“建立法治国家”倒退到“稳定压倒一切”，理念的倒退造成了改革的停滞，并由此造成了社会矛盾激化、腐败更加严重、社会分配更加不公，同时使机会更不均等，社会流动性变弱。<ref>{{cite web|url=http://finance.ifeng.com/news/macro/20130611/8123937.shtml|title=张维迎：未来十年是改革关键期 过了这村没这店|publisher=凤凰网|date=2013年6月11日| language =zh-hans }}</ref>

虽然温家宝任职期间多次宣称进行[[打房|房地产调控]]并宣称调控有效果<ref>{{cite web|url=http://money.163.com/08/1111/06/4QEUASMM00252G50.html|title=温家宝部署7项工作 调控房地产促股市稳定|publisher=新华网|date=2008-11-11| language =zh-hans }}</ref><ref>{{cite web|url=http://news.xinhuanet.com/house/2011-10/27/c_122202614.htm|title=温家宝：房地产调控成果仍需巩固|publisher=中华工商时报|date=2011年10月27日| language =zh-hans }}</ref>，然而在其任内中国大陆房价却伴随着调控一路攀升而暴涨数倍<ref>{{cite web|url=http://finance.qq.com/a/20130408/002021.htm|title=中国式房产调控：十年调控房价屡调屡高|publisher=新文化报|date=2013-04-08| language =zh-hans }}</ref>，以[[上海|上海]]为例，平均一个上海家庭需要20年的收入才能买一套房<ref>{{cite web|url=http://house.qq.com/a/20130126/000033.htm|title=上海房价收入比何以高达20倍|publisher=金融投资报|date=2013-01-26| language =zh-hans }}</ref>。民众基本必需的住房未能得到保障造成了较为严重的[[中国社会问题|社会问题]]，也造成了许多家庭伦理悲剧<ref>{{cite web|url=http://www.vzton.com/Infomation/House/Talks/Comments/2011/0122/67006.html|title=中国住房问题与住房社会政策之现状 |publisher=华声在线|date=2011-01-22| language =zh-hans }}</ref>。另外，温家宝亦在不同场合表态将努力缩小贫富差距<ref>{{cite web|url=http://www.china.com.cn/news/zhuanti/2010lianghui/2010-03/05/content_19535788.htm|title=BBC关注温家宝工作报告中缩小贫富差距内容|publisher=中国网|date=2010-03-05| language =zh-hans }}</ref>，然而在其任内的十年，[[中国贫富差距|中国贫富差距]]不断扩大，报告称到2012年中国家庭的[[基尼系数|基尼系数]]达到了0.61，贫富差距举世罕见。<ref>{{cite web|url=http://news.qq.com/a/20121210/000069.htm|title=报告称中国家庭基尼系数0.61 贫富悬殊世所少见|publisher=京华时报|date=2012-12-10| language =zh-hans }}</ref>

2003年[[朱镕基|朱镕基]]卸任中国国务院总理时交给了温家宝一个中国经济社会的良性发展之局<ref>{{cite web|url=http://news.eastday.com/epublish/gb/paper148/20030308/class014800018/hwz899815.htm|title=如果在2103年评价朱镕基|publisher=南方周末|date=2003-03-08|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160913025550/http://news.eastday.com/epublish/gb/paper148/20030308/class014800018/hwz899815.htm|archivedate=2016年9月13日|df=}}</ref>，但经过温家宝十年任期，他交给继任者[[李克强|李克强]]的却是一个危机四伏、百病丛生的烂摊子<ref>{{cite web|url=http://www.voachinese.com/content/he-qing-lian-20141215/2558984.html|title=何清涟：总理大位纵易主 经济颓势也难挽| language =zh-hans }}</ref>，而且朱镕基时代三大改革所积累下的财富也在这十年间被耗尽<ref>{{cite web|url=http://i.ifeng.com/news?m=1&aid=48181118|title= 朱镕基时代三大改革财富耗尽|publisher=凤凰卫视|date=2012-11-28| language =zh-hans }}</ref>。

== 限塑令颁行争议 ==
2007年12月31日，温家宝领导的国务院办公厅下发了《国务院办公厅关于限制生产销售使用塑料购物袋的通知》。温家宝的这一政策将原来超市免费提供的塑料袋转嫁到消费者头上，一个200年都无法降解的塑料袋，出厂价以分计算，却以3毛左右的价格卖给消费者，使中国各大超市赚得盆满钵满。<ref>{{cite web|url=http://news.ifeng.com/a/20170612/51229092_0.shtml|title=“限塑令”沦为“卖塑令”，需要改改了|publisher=凤凰网|date=2017年6月12日| language =zh-hans }}</ref>统计显示，全球每年塑料总消费量约为4亿吨，中国消费6000万吨以上。在很多消费者看来，相比便携的需求，对几毛钱的塑料袋成本整体感受不明显，但在不少超市，靠着温家宝政府出台的政策，每年仅出售塑料袋就能赚上千万元。<ref>{{cite web|url=http://www.lyrb.com.cn/html/news/lynews/shms/2016/0217/57644.html|title=限塑令！卖塑令？实施7年名存实亡？|publisher=浏阳网|date=2016-02-17| language =zh-hans }}</ref>

==  河南经济发展 ==
批评指，他曾对部分区域经济造成伤害，如以违法用地为名对[[郑东新区|郑东新区]]和[[郑州加州工业城|郑州加州工业城]]的处理，沿海部分城市同样违规并未处理<ref name="bxwsb"/>，打压郑州经济的迹象明显，引起当地官员不满<ref name="bxwsb">{{cite web|url=http://www.peacehall.com/news/gb/china/2006/10/200610041929.shtml|title=温家宝痛下杀手惩治河南违规征地，官员怨中央不公|publisher=博讯网|date=2006年10月4日}}</ref>。其任内[[郑州铁路局|郑局]]被拆分，优质资产被剥离。河南周边相继得到其政策资金支持，唯[[中原城市群|中原城市群]]（李克强主政河南时提出）不予批复，《[[大河报|大河报]]》2010年发文称“河南塌陷”<ref>{{cite web|url=http://newpaper.dahe.cn/hnsb/html/2010-08/12/content_365499.htm|title=区域战略 河南塌陷|publisher=大河报|date= 2010年8月12日}}</ref>，[[卢展工|卢展工]]到豫后，为[[中原经济区|中原经济区]]造势，最终被快下台的温批复，媒体发文称是安慰奖，再不批复说不过去<ref>{{cite web|url=http://finance.ifeng.com/news/region/20111015/4848764.shtml|title=中原经济区被指是给河南安慰奖：再不批复说不过去|publisher=凤凰网|date=2011年10月15日}}</ref>。由于其不公政策，温当政时河南本省发展遭遇障碍，河南的廉价原材料和劳动力对外服务，同时还要提供廉价粮食。

== 上海经济发展 ==
[[File:上海自贸区四号门_20131017.jpg|thumb]]最早在2003年就有所规划，但却因为温家宝的强力阻挠而长期未能获得立项，最终在温家宝退休后，才由[[习近平|习近平]]、[[李克强|李克强]]、[[韩正|韩正]]等人成功设立]]
中国大陆第一个自由贸易区[[上海自贸区|上海自贸区]]，最早在2000年前后，[[陈良宇|陈良宇]]主政上海时期即有发展规划。2006年，[[胡锦涛|胡锦涛]]在[[上海市|上海市]]考察调研时提出：要把上海建设成为国际经济、金融、贸易、航运四大中心，但当时上海建立[[自由贸易园区|自贸区]]的许多经济改革创新的计划遭到当时中央政府的强大阻挠而基本没有进展，直到温家宝卸任半年前，才批示建立上海国际中心的提议，要求各个部门积极研讨与配合。但此后，上海自贸区建设还是被搁浅了，一直到温家宝退休之前，都没有任何实质性进展<ref name="autogenerated1">{{cite web|url=http://business.sohu.com/20130819/n384470025.shtml|title= 陈波：上海自贸区计划若难产 金融改革都会失败|publisher=财经网|date=2013-08-19 |accessdate=2014-06-25}}</ref>。直到温家宝卸任，[[李克强|李克强]]接任总理，上海自贸区才摆脱了长年难产的困境。温家宝任内，中央政府还在公务员薪酬、国产大飞机、迪斯尼乐园以及[[上海世博会|上海世博会]]等重大事宜上压制上海。 <ref>{{cite web|url=http://www.irmb.cn/news/show.php?itemid=789|title=上海自贸区内幕，从陈良宇到美日新加坡博弈|publisher=中国商人杂志|date=2013-10-19|accessdate=2014-06-25|deadurl=yes|archiveurl=https://web.archive.org/web/20140407073945/http://www.irmb.cn/news/show.php?itemid=789|archivedate=2014年4月7日|df=}}</ref> <ref name="autogenerated1"/>另一方面，[[上海港|上海港]]的重要组成部分——[[洋山深水港|洋山深水港]]在筹建时，温家宝政府以没有资金为由拒绝拨付款项支持，但与此同时温家宝却将国家4000亿元人民币的的巨额资金投入其家乡的港口——[[天津港|天津港]]的建设，但天津港由于地理条件原因，既非中转港也非不冻港，故这样的安排被认为是很不合理的，是扶植自己家乡的行为，最终[[天津港|天津港]]并没有发挥其当初所宣称的拉动北方经济的效果，因此被专家戏称为是一座名副其实的“废港”。<ref>{{cite web|url=http://www.singpao.com/xw/ht/201310/t20131012_465042.html|title=上海設立自貿區的歷史內幕|publisher=香港成报|date=2013-10-12|accessdate=2014-06-26|deadurl=yes|archiveurl=https://web.archive.org/web/20150101202331/http://www.singpao.com/xw/ht/201310/t20131012_465042.html|archivedate=2015年1月1日|df=}}</ref>
