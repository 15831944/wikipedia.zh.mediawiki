[[視頻分享網站|视频分享平台]][[YouTube|YouTube]]是2017年全球第二大受欢迎的网站<ref>[http://www.alexa.com/siteinfo/youtube.com "youtube.com Traffic Stats"] {{webarchive|url=https://web.archive.org/web/20161127004253/http://www.alexa.com/siteinfo/youtube.com |date=2016-11-27 }}, Alexa. Retrieved September 24, 2012</ref>。

== 軟體方面 ==
* [[Windows_Live_Messenger|Windows Live Messenger]]：2008年5月10日[[微软|微软]]停止預設功能上YouTube連結Windows Live Messenger的服務，雖然在21:30時已經取消此禁令，但微軟始終對此事並沒有解釋如佔用CPU資源等類似理由。原本連結是以<nowiki>http://</nowiki>或<nowiki>www</nowiki>做為開頭來啟動連結，但是在此事件中當有youtube.com的字眼時，將會出現一「The following message could not be delivered to all recipients: (original message here).」的錯誤提示。巧合的是，這一禮拜剛好微軟公布其一新的視訊服務。

== 教育機構 ==
* [[楊百翰大學|楊百翰大學]]：[[美國|美國]]一些高校也會阻擋學生對YouTube的瀏覽，如由[[摩爾門教|摩門教]]教會所創的私立大學[[楊百翰大學|楊百翰大學]]曾封鎖YouTube，但2009年6月這項決定已經改變。

* [[南加州建築學院|南加州建築學院]]：已畢業的研究生和尚未畢業的學生皆被禁止到YouTube或者其他常見的影片分享網站，如[[Google视频|Google影片]]；且並不考慮影片是否擁有教育內容。

* [[K-12|K-12]]：許多美國和[[加拿大|加拿大]]的K-12學校由於影片內容可能含有性、暴力和其他不尋常內容，會阻止學生訪問YouTube，但也有些學校會允許在教育目的時使用。

* [[英国|英国]]學校：英國各地的部分學校也禁止YouTube和其他影片分享網站的瀏覽，以「避免在工作上分心」。這大多是針對學生，在一些學校的教師被允許能前往YouTube瀏覽與教育相關的影片。這是透過使用代理檢查員系統，來通過理事會所提供的網路。但是在英國仍有些學校允許學生訪問YouTube，但他們只允許觀看被認為是適當的影片，且透過登錄到非自己所註冊的帳戶，並禁止上傳影片。

* [[马来西亚|马来西亚]][[亞太工藝與革新大學|亞太工藝與革新大學]]：學校盡可能阻止學生瀏覽YouTube，以減少學生堵塞了頻寬。

== 國家方面 ==

=== 阿拉伯联合酋长国 ===
2006年8月，阿拉伯聯合大公國電信監管當局封鎖YouTube，理由為網站影片「具有成人內容」，1個月後這禁令解除。

=== 巴西 ===
2006年YouTube巴西語言版本被巴西[[音樂電視網|MTV]]和其主持人[[丹妮埃拉·西卡雷莉|丹妮埃拉·西卡雷莉]]控告，指出該網站放置了由一名[[狗仔隊|狗仔隊]]偷拍她和她男友在[[西班牙|西班牙]]海灘的性愛影片。這起訴訟導致YouTube被封鎖，直到所有的影片副本皆被刪除。2007年1月6日禮拜六，巴西法院下達一法律強制令命令，在網路伺服器上放置過濾器以防止巴西用戶瀏覽YouTube。該措施在有效性方面受到質疑，因為影片除了可以在YouTube上播放，且隨著[[網路爆紅|網路快速傳播現象]]也可以出現在其他類似網站上。2007年1月9日禮拜二，同一個法院推翻其先前的決定，命令將過濾器刪除。雖然該影片本身仍被禁止，但已沒有技術性的方式來阻止使用戶瀏覽YouTube。

=== 泰国 ===
2006年，泰國当局禁止泰國IP地址用戶訪問YouTube。泰國當局確定的20個具有攻擊性的影片，並要求YouTube刪除之前也自行檢查其他影片的內容。2007年3月8日的這一周，YouTube再次被[[泰国|泰國]]封鎖。許多部落客認為是前任總理[[塔克辛·欽那瓦|塔克辛·欽那瓦]]在[[CNN|有線電視新聞網]]的訪問所導致，泰國政府則未對於這次禁令提供理由。YouTube於3月10日又再次被開放使用。2007年4月30日晚間，[[泰国|泰国]]政府再次封锁YouTube且长达近４个月。原因是该网站上的一段视频显示了两只脚对着泰国国王[[普密蓬·阿杜德|蒲美蓬·阿杜德]]，而这在泰国被认为是冒犯动作。不過政府之後聲稱將在幾天後解除封鎖YouTube網頁，只會封鎖網站內這類影片而不是整個網站。交通部長{{le|Sitthichai Pokai-udom|Sitthichai Pokai-udom}}指出：「當他們決定移除該影片時，我們將會撤銷禁令。」在這一事件中，[[互联网|網際網路]][[新闻|新聞]][[網誌|部落格]][[Mashable|Mashable]]也因報導對於YouTube影片的問題也被泰國當局封鎖。最后YouTube同意禁止在网站上出现此段视频之后，於8月30日获得解封。<ref>{{cite news|url=http://news.bbc.co.uk/chinese/simp/hi/newsid_6970000/newsid_6972100/6972187.stm|title=泰国对视频共享网站YouTube解禁|publisher=BBC|date=2007-08-31|accessdate=2009-03-25|deadurl=no|archiveurl=https://web.archive.org/web/20140301134238/http://news.bbc.co.uk/chinese/simp/hi/newsid_6970000/newsid_6972100/6972187.stm|archivedate=2014-03-01}}</ref>2007年9月21日，泰國當局透過法院想要禁止一最近出現的影片，內容指出[[泰國樞密院|泰國樞密院]]院長[[炳·廷素拉暖|炳·廷素拉暖]]企圖操縱皇室，使自己繼承成為[[泰国君主列表|泰國國王]]。

=== 伊朗 ===
2006年12月3日，伊朗政府由於YouTube網站有[[人类性行为|性]]相關的內容，遂宣布YouTube違反社會道德和行為準則，暫時封鎖其網站，而在[[2009年伊朗总统选举|2009年伊朗總統選舉]]時封鎖已取消。

=== 土耳其 ===
[[File:Youtube_censored_TR2008.png|thumb]]
[[File:YouTube_blocked_in_Turkey.jpg|thumb]]

土耳其法院自2007年3月開始曾數次禁止YouTube。第一次是2007年3月6日[[土耳其|土耳其]]一家地方法院判决封锁YouTube，直至该网站删除侮辱土耳其国父[[穆斯塔法·凯末尔·阿塔图尔克|穆斯塔法·凯末尔·阿塔图尔克]]的视频为止<ref>{{cite web |url=https://www.theguardian.com/world/2007/mar/07/turkey |title=存档副本 |accessdate=2013-03-21 |deadurl=no |archiveurl=https://web.archive.org/web/20130831035557/http://www.theguardian.com/world/2007/mar/07/turkey |archivedate=2013-08-31 }}</ref>。在YouTube律师稍后向该地方检察机关提供视频删除证明后，Youtube于2007年3月9日在土耳其恢复访问。其后由于此类视频在YouTube上反复出现，在2008年[[土耳其|土耳其]]又有多家法院前后数次发布封锁YouTube的禁令<ref>{{cite web |archivedate=2010-06-09 }}</ref>以諷刺法院的禁止令。2010年6月，[[土耳其總統|土耳其總統]][[阿卜杜拉·居尔|阿卜杜拉·居爾]]亦在其[[Twitter|Twitter]]帳戶上，表達了反對土耳其封鎖YouTube的看法。他說這影響到許多Google在土耳其的服務，并称已指示官員尋找法律途徑讓YouTube能被允許訪問<ref>{{cite web 在此一禁止令于2010年10月30日刚刚解除后，土耳其政治家丹尼斯·贝卡尔（Deniz Baykal）的视频导致该网站被短暂封锁，并被威胁如果网站不删除视频将受到新的封锁<ref>{{cite web |url=https://www.wsj.com/articles/SB10001424052748704462704575590420251199614?mod=googlenews_wsj |title=存档副本 |accessdate=2013-12-08 |deadurl=no |archiveurl=https://web.archive.org/web/20171107015843/https://www.wsj.com/articles/SB10001424052748704462704575590420251199614?mod=googlenews_wsj |archivedate=2017-11-07 }}</ref>。2014年3月27日，土耳其政府在一段描绘了土耳其的外交部长与情报官员、总司令讨论派兵进攻叙利亚圣战者武装的场景的视频被发布后的几个小时内封锁了YouTube<ref>{{cite web |url=https://www.wsj.com/articles/no-headline-available-1395933124?mod=wsj_streaming_stream&mg=reno64-wsj&url=http%3A%2F%2Fonline.wsj.com%2Farticle%2FSB10001424052702304418404579465283912697784.html%3Fmod%3Dwsj_streaming_stream&tesla=y |title=存档副本 |accessdate=2017-10-31 |deadurl=no |archiveurl=https://web.archive.org/web/20171107010056/https://www.wsj.com/articles/no-headline-available-1395933124?mod=wsj_streaming_stream&mg=reno64-wsj&url=http%3A%2F%2Fonline.wsj.com%2Farticle%2FSB10001424052702304418404579465283912697784.html%3Fmod%3Dwsj_streaming_stream&tesla=y |archivedate=2017-11-07 }}</ref>。随后一系列法院裁定该封锁应于2014年4月9日解除，但土耳其政府违抗法院的判决，继续封锁YouTube。直至5月29日，土耳其宪法法院发出判决，指此禁止令侵犯了宪法规定的言论自由权利，并命令恢复YouTube在土耳其的访问。

2014年6月1日，Youtube在土耳其境内的访问才恢复。2015年4月6日，由于广泛传播一名在人质危机期间被杀的士兵的视频，YouTube再次被短暂封锁，同时被封锁的还有[[Facebook|Facebook]]和[[Twitter|Twitter]]。2016年12月23日，据互联网监测组织[[土耳其封锁|土耳其封锁]]的报道，YouTube在土耳其再次无法访问，据称这是由于圣战者在该网站分享烧死被俘的土耳其士兵的视频<ref>{{cite web |url=https://turkeyblocks.org/2016/12/23/social-media-shutdowns-turkey-isis-releases-soldier-video/ |title=存档副本 |accessdate=2016-12-27 |deadurl=no |archiveurl=https://web.archive.org/web/20161227074815/https://turkeyblocks.org/2016/12/23/social-media-shutdowns-turkey-isis-releases-soldier-video/ |archivedate=2016-12-27 }}</ref><ref>{{cite web |url=https://apnews.com/c37a7dd383d44412baf37558095c1569 |title=存档副本 |accessdate=2016-12-27 |deadurl=no |archiveurl=https://web.archive.org/web/20161227131303/https://apnews.com/c37a7dd383d44412baf37558095c1569 |archivedate=2016-12-27 }}</ref>。自12月25日起，该网站的访问重新恢复至今。

=== 摩洛哥 ===
2007年5月25日，摩洛哥國有企业摩洛哥電信停止存取YouTube，而另外兩家私營的網路服務提供商Wana和Méditel則未受任何影響。摩洛哥電信並沒有任何理由解釋為何YouTube被封鎖，但據猜測可能是與站內擁有一些[[波利薩里奧陣線|波利薩里奧陣線]]的宣傳影片剪輯，或由於一些影片批評[[摩洛哥國王|摩洛哥國王]][[穆罕默德六世_(摩洛哥)|穆罕默德六世]]。5月30日，YouTube可以再次進行瀏覽，之後摩洛哥電信非正式地宣布，此次無法訪問YouTube僅是一個單純的「技術故障」。

=== 突尼斯 ===
2007年11月2日，突尼西亞封鎖了YouTube網頁，並會出現一假造的「HTTP 404錯誤消息」的訊息，目前仍沒有正式的消息解釋此一事件。

=== 日本 ===
