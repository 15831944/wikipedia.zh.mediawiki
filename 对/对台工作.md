'''对台工作'''是指中华人民共和国各级政府涉及[[台湾|台湾]]（中华民国）的政务工作。

[[统一中国|统一中国]]并最终解决[[台湾问题|台湾问题]]，一直是[[中华人民共和国政府|中华人民共和国政府]]的政治诉求，因此对台湾地区的[[统战|统战]]是政府工作重点。除各级政府定时召开[[对台工作会议|对台工作会议]]，处理相关问题外。从最高领导机构——[[中华人民共和国国务院|国务院]]到区县级地方政府，均设置[[台办|台办]]作为常设机构，配备人员、划拨经费，负责处理与台湾有关的政务工作。各级台办的日常工作以服务台湾人士、维护台湾利益为主要内容<ref name="a1">{{Cite web |url=http://www.huaxia.com/ytsc/tbxxg/llwz/2013/11/3606542.html |title=基层对台工作现状及对策思考 |accessdate=2017-08-19 |author=王庆春 |date=2013-11-07|work= |publisher=华夏经纬网·豫台视窗（河南省人民政府台湾事务办公室），|language=Zh-hans }}</ref>，多以利益输送的手法进行<ref name="a2">{{Cite web |url=http://news.dwnews.com/taiwan/news/2015-09-03/59679147.html |title=“对台工作完全错误”传习近平调查陈云林 |accessdate=2017-08-19 |author= |date=2015-09-03|work=综编：林犀 |publisher=多维新闻网|language=Zh-hant }}</ref>。对台让利是主要方针<ref name="a3">{{Cite web |url=http://phtv.ifeng.com/a/20150526/41091528_0.shtml |title=阮次山：大陆要防范对台让利却换不来真心 |accessdate=2017-08-19 |author= |date=2015-05-26|work= |publisher=凤凰网|language=Zh-hans }}</ref>，给予台湾人士“[[超国民待遇|超国民待遇]]”的同时，要求大陆民众为台湾人士放弃自身权益<ref name="a4">{{Cite web |url=http://news.cnr.cn/native/gd/20170307/t20170307_523642425.shtml |title=人大代表张兆民：对台胞“高看一等、厚爱一筹” 平潭发展“三多一快” |accessdate=2017-08-19 |author=记者：张子亚、郭婕妤 |date=2017-03-07|work= |publisher=央广网|language=Zh-hans }}</ref>。[[国台办|国台办]]亦被大陆民众蔑视的称为“[[跪台办|跪台办]]”，在具体事件中，有批评政府“一边宣传台湾人民是同胞，一边安排[[台独|台独]]分子羞辱大陆民众”<ref>{{Cite web |url=http://news.ltn.com.tw/news/politics/breakingnews/1026499 |title=賴清德高談台獨 中國網友又崩潰了 |accessdate=2017-08-19 |author= |date=2014-06-08|work= |publisher=自由时报电子报|language=Zh-hant }}</ref>。

2014年台湾青年学生主导了[[太陽花學運|太陽花學運]]，[[海峽兩岸服務貿易協議|海峽兩岸服務貿易協議]]在台遭到抵制。而中国大陸政府自认为这份协议自方让利，充分体现中国大陸政府对台湾的诚意和善意<ref>{{Cite web |url=http://news.ifeng.com/taiwan/3/detail_2013_06/21/26672806_0.shtml |title=商务部：《海峡两岸服务贸易协议》体现对台诚意善意 |accessdate=2017-08-19 |author= |date=2013-06-21|work= |publisher=凤凰网，来源：国际在线|language=Zh-hans }}</ref>。有报道称，中国大陸政府高层在太阳花学运后，承认对台工作的方法、策略和对象完全错误<ref name="a2"/>。同时有人指出，对台让利并不能换取台湾人士的[[国家认同|政治认同]]<ref name="a3"/>。此后，对台工作以维护台湾人士利益为目的，单方面向台湾输送利益的格局并无改变，譬如用[[财政|财政]]资助台湾人士前往大陆旅游的“[[统战团|统战团]]”持续至今。2017年，有身兼[[全国人大代表|全国人大代表]]的福建地方政府官员公开主张对台湾人士“高看一等，厚爱一筹，宁可我们没有，也要台胞优先。”<ref name="a4"/>

== 基层对台工作 ==

基层对台工作是指[[县级行政区|县级行政区]]人民政府及以下单位，包括乡镇、机关、学校、国有企业、非公有制企业、科研院社和医疗卫生单位的对台工作<ref name="c1"/>。工作对象除了台湾籍人士外，包括其家属、涉台婚姻对象，对于本地台籍人士、家属，建立[[档案|档案]]。

常见的工作内容有四节（[[元旦|元旦]]、[[春节|春节]]、[[清明节|清明节]]和[[中秋节|中秋节]]）的走访慰问<ref name="c1">{{Cite web |url=http://jz.rootinhenan.com/rootinhenan/html/2014/3/18463.htm |title=做好基层对台工作之浅见 |accessdate=2017-08-24 |author= |date=2014-03-25|work= |publisher=根在中原--中共河南省委统战部|language=Zh-hans }}</ref>，发放生活补助<ref>{{Cite web|url=http://www.puer.gov.cn/jqlb/lcxx/02679606168013190817|title=澜沧县台办深入基层走访困难台胞台属|accessdate=2017-08-22|author=李兆兰|date=2015-01-07|work=|publisher=普洱新闻网|language=Zh-hant}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，以及提供免费的各类服务（譬如医疗<ref>{{Cite web |url=http://fjrb.fjsen.com/fjrb/html/2016-05/10/content_924972.htm?div=-1 |title=晋江为年满60岁台胞免费体检 |accessdate=2017-08-22 |author= |date=2016-05-10 |work= |publisher=福建日报社数字报刊平台 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20170822180550/http://fjrb.fjsen.com/fjrb/html/2016-05/10/content_924972.htm?div=-1 |archivedate=2017-08-22 }}</ref>、司法服务<ref>{{Cite web |url=http://www.mnw.cn/xiamen/news/947238.html |title=组建台胞调解员队伍 在厦台商遇纠纷调解有门道 |accessdate=2017-08-22 |author= |date=2015-07-20|work= |publisher=闽南网，来源：海西晨报|language=Zh-hans }}</ref>等）。
