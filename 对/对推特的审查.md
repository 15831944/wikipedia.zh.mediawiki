== 埃及 ==
{{main|埃及网络审查}}
2011年1月25日，[[2011年埃及革命|2011年埃及革命]]期间，埃及民众无法访问推特。一些新闻报道指责埃及政府封锁推特<ref name="Dan Murphy 0125">{{cite news| last=  Murphy| first=  Dan| url=  http://www.csmonitor.com/World/Backchannels/2011/0125/Inspired-by-Tunisia-Egypt-s-protests-appear-unprecedented| title=  Inspired by Tunisia, Egypt's protests appear unprecedented| publisher=The Christian Science Monitor| date=2011-01-25| accessdate= }}</ref>，而埃及最大的移动网络运营商[[沃达丰埃及|沃达丰埃及]]（{{lang|en|Vodafone Egypt}}），澄清这不是他们的行动<ref>{{cite web| url=  https://twitter.com/VodafoneEgypt/statuses/29927017323433984| title=  Twitter / Vodafone Egypt: We didn't block twitter -  ...| publisher=  Vodafone Egypt on Twitter| date=2011-01-25| accessdate= 2011-01-25}}
</ref>。然而，推特的新闻稿并没有说明是哪一家公司承认它们提请封锁<ref name="Alex Sherman">{{cite news| last=  Sherman| first=  Alex| url=  http://www.businessweek.com/news/2011-01-26/twitter-says-access-to-service-in-egypt-is-blocked.html| title=  Twitter Says Access to Service in Egypt Is Blocked| publisher=Business Week| date=2011-01-26| accessdate= 2011-01-25}}</ref>，截至1月26日，推特仍无法在埃及访问。1月27日，各大报道声称埃及国内的整个互联网已被关闭<ref>{{cite news| url=http://www.huffingtonpost.com/2011/01/27/egypt-internet-goes-down-_n_815156.html | work=Huffington Post | first=Craig | last=Kanalley | title=Egypt's Internet Shut Down, According To Reports | date=2011-01-27}}</ref>。

互联网关闭不久后，[[谷歌|谷歌]]、[[推特|推特]]和[[SayNow|SayNow]]（1月份被谷歌收购的语音通讯创业公司）宣布启动[[Speak_to_Tweet|Speak to Tweet]]服务。谷歌在官方博客上宣布服务的目标是协助埃及示威者在互联网关闭期间保持联络。用户使用手机发布推特可以留下语音信箱留言，并使用推特标签#Egypt<ref name="Google Blog">{{cite web|last=Singh|first=Ujjwal|title=Some weekend work that will (hopefully allow more Egyptians to be heard.|url=http://googleblog.blogspot.com/2011/01/some-weekend-work-that-will-hopefully.html |publisher=Google|accessdate=2011-05-12}}</ref>。用户只需拨打同一个指定的电话号码，就可以在没有互联网连接的情况下访问这些推文。而接入互联网的用户可以透过访问twitter.com/speak2tweet收听这些推特。

2月2日，埃及四大服务提供商重新建立连接<ref>{{cite news|url=http://www.bbc.co.uk/news/technology-12346929|title=Egypt internet comes back online|date=2011-02-02|publisher=BBC News}}</ref><ref>{{cite web|url=http://asert.arbornetworks.com/2011/02/egypt-returns-to-the-internet/ |title=Egypt Returns to the Internet |author=Craig Labovitz |publisher=Arbor Networks |date=2011-02-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20111025112723/http://asert.arbornetworks.com/2011/02/egypt-returns-to-the-internet/ |archivedate=2011-10-25 }}</ref><ref>{{cite web|url=http://www.renesys.com/blog/2011/02/egypt-returns-to-the-internet.shtml|title=Egypt Returns To The Internet|publisher=Renesys|author=James Cowie|date=2011-02-02|deadurl=yes|archiveurl=https://web.archive.org/web/20110205190100/http://www.renesys.com/blog/2011/02/egypt-returns-to-the-internet.shtml|archivedate=2011年2月5日|df=}}</ref>。一个星期后，发生在革命高潮阶段的严格过滤结束。

== 法国 ==

== 韩国 ==
