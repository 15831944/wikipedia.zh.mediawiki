'''中国人民解放军海军诞生地纪念馆'''，位于[[江苏省|江苏省]][[泰州市|泰州市]][[高港区|高港区]][[白马镇_(泰州市)|白马镇]]，是在[[中国人民解放军海军|中国人民解放军海军]]诞生地的基础上兴建的反映中国人民解放军海军历史的纪念馆。

==简介==
1949年4月23日，根据[[中央军委|中央军委]]命令，[[华东军区|华东军区]]海军在[[泰州|泰州]]白马庙[[第三野战军|第三野战军]]东路渡江作战指挥部驻地成立，由华中军区副司令员[[张爱萍|张爱萍]]任华东军区海军司令员兼政治委员。1950年1月12日，[[毛泽东|毛泽东]]主席签发命令，任命[[萧劲光|萧劲光]]为中国人民解放军海军司令员，并开始组建海军领导机关。海军成为中国人民解放军的战略军种之一。1989年2月17日，中央军委批复海军，以1949年4月23日为人民海军诞生日<ref name=gmw>{{Cite web |url = http://www.gmw.cn/01gmrb/1999-04/30/GB/18042%5EGM2-3012.HTM?_t_t_t=0.3268445038702339 |title =人民海军诞生地纪念馆在泰州落成  |author =  |date =1999-04-30  |publisher =光明日报  |language =  |accessdate =  |quote =  }}</ref>。

经[[中共中央办公厅|中共中央办公厅]]、[[国务院办公厅|国务院办公厅]]批准，中国人民解放军海军诞生地纪念馆于1998年10月18日在泰州市白马乡（后改为白马镇）开工兴建。1999年4月29日，中国人民解放军海军诞生地纪念馆在泰州市白马乡举行落成开馆典礼，海军政治委员[[杨怀庆|杨怀庆]]、[[南京军区|南京军区]]副司令员[[何其宗|何其宗]]、[[中共江苏省委|中共江苏省委]]副书记[[曹克明|曹克明]]为纪念馆揭幕。该馆馆名由[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军委主席|中央军委主席]][[江泽民|江泽民]]题写<ref name=gmw/>。

2009年4月23日，中国人民解放军海军成立60周年纪念日，泰州市人民政府投资1200万元人民币对中国人民解放军海军诞生地纪念馆进行的场馆整修和布展改陈工程完工并对社会开放。重新布展后，馆内墙、门、窗、通道都是舰船的符号<ref name=cj/><ref>{{Cite web |url =http://www.china.com.cn/photo/txt/2009-04/22/content_17654330.htm  |title = 中国人民解放军海军诞生地纪念馆改造工程竣工 |author =  |date =2009-04-22  |publisher =中国网  |language =  |accessdate =  |quote =  }}</ref>。

中国人民解放军海军诞生地纪念馆的上级单位是泰州市文化广电新闻出版局<ref>{{Cite web|url = http://www.hjdsd.com/info.php?infos_id=1490|title = 海纪馆党支部召开换届选举大会|author = |date = 2016-11-11|publisher = 中国人民解放军海军诞生地纪念馆|language = |accessdate = |quote = }}{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

中国人民解放军海军诞生地纪念馆包含“人民海军诞生地”和新馆两部分。

'''人民海军诞生地'''，又名'''华东野战军（三野）渡江战役指挥部旧址'''。该建筑原为建于明末清初的王姓地主庄园，1949年4月初[[中国人民解放军|中国人民解放军]][[华东野战军|华东野战军]]（第三野战军）[[渡江战役|渡江战役]]指挥部设于此地，并于4月23日组建中国人民解放军第一支海军部队——华东军区海军。旧址占地面积约1500平方米，建筑面积约800平方米，现存清式二层小楼一座与平房数间，楼下为[[粟裕|粟裕]]、[[张震_(上将)|张震]]和张爱萍的卧室兼办公室，楼上为会议室<ref>{{cite web |url = http://www.hjdsd.com/about.php |title = 中国人民解放军海军诞生地纪念馆简介 |accessdate = 2012-10-16 |work =  |publisher = 中国人民解放军海军诞生地纪念馆 |archive-url = https://web.archive.org/web/20121009011824/http://www.hjdsd.com/about.php# |archive-date = 2012-10-09 |dead-url = yes }}</ref><ref name=cj>{{Cite web|url =http://www.tzcjjc.gov.cn/default.php?mod=article&do=detail&tid=1024509|title =中国人民解放军海军诞生地纪念馆|author =|date =2013-11-26|publisher =泰州市城建监察支队|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170118223640/http://www.tzcjjc.gov.cn/default.php?mod=article&do=detail&tid=1024509|archivedate =2017-01-18}}</ref>。1982年，被公布为第三批[[江苏省文物保护单位|江苏省文物保护单位]]。1996年和1997年先后被公布为江苏省全民国防教育基地、江苏省爱国主义教育基地。2006年，被公布为[[第六批全国重点文物保护单位|第六批全国重点文物保护单位]]<ref name=cj/>。

新馆由[[中国工程院院士|中国工程院院士]]、[[东南大学|东南大学]]建筑学家[[齐康|齐康]]主持设计，由东南大学建筑设计研究院、南京建筑设计院进行图纸设计。主体建筑为舰艏形状，整个场馆建筑面积4000平方米。室外广场左侧竖有桅杆<ref name=cj/>。

纪念馆以“近代沧桑”、“白马建军”、“威震海疆”、“发展壮大”、“鱼水情深”这五个主题展示中国人民解放军海军的发展历史，广场及室内陈列有海军退役的歼击机、轰炸机、超黄蜂直升机、鱼雷、舰炮、导弹、飞机舰艇模型、海军服饰等<ref name=cj/>。
