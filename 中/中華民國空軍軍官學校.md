'''中華民國空軍軍官學校（R.O.C. Air Force Academy），'''簡稱'''空軍官校'''、'''空官'''，為培養[[中華民國空軍|中華民國空軍]][[軍官|軍官]]的專門學校。

== 沿革 ==
<ref>{{cite web|url=http://www.cafa.edu.tw/content/index.asp?Parser=1,4,58|title=空軍官校簡介|publisher=中華民國空軍官校|date=2014年10月6日|language=zh}}</ref>
[[File:Central_Aviation_School.jpg|缩略图]]
[[File:臺灣空軍官校師生校外教學攝於某廟宇前_officers_and_students_of_Taiwan_Air_Force_Academy_in_front_of_a_temple.jpg|thumb]]
[[File:ROCAF_Academy_Military_Airplanes_Display_Area_Birdview_from_West_20111015.jpg|200px]]
[[File:Visitors'_Cars_Stopped_behind_Main_Gate_Waiting_for_Check_20111015.jpg|thumb]]

*1929年：[[國民政府|國民政府]]於[[中華民國陸軍軍官學校|南京中央陸軍軍官學校]]創立航空班。
*1932年：7月正式成立'''军政部航空学校'''，12月更名為「'''中央政府航空學校'''」，迁至[[杭州|杭州]][[筧橋|筧橋]]，現成為解放军[[笕桥机场|笕桥机场]]。
*[[中國抗日戰爭|抗戰]]爆發後遷至[[雲南|雲南]][[昆明|昆明]]，並於1938年正式定名為「'''空軍軍官學校'''」。
*抗戰勝利後遷回筧橋，之後因[[國共內戰|國共內戰]]爆發，於1949年遷臺，並於[[臺灣省|臺灣省]][[高雄縣|高雄縣]][[岡山鎮|岡山鎮]]（今[[高雄市|高雄市]]岡山區）現址復校。
*1960年9月：空軍官校改制為四年制大學教育，之後開始招收高中畢業之飛行專業軍官班。
*2008年6月：空軍官校招收女性正期班學生。

== 校徽涵義 ==
[[File:空軍官校校徽.JPG|thumb]]
一、橡樹葉象徵自強不息之意義。<br />
二、飛機機翼象徵八一四航空救國精神。<br />
三、藍色象徵蔚藍天空。<br />
四、整個校徽圖案象徵空軍筧橋精神。

== 校旗 ==
[[File:ROC_Air_Force_Academy_Flag.svg|thumb]]
以中華民國空軍軍旗製作，左邊加綴「空軍軍官學校」白底黑字。
*[[青天白日|青天白日]][[中華民國國徽|國徽]]象徵軍人效忠國家，保衛國家疆域之意涵。
*飛鷹雙翅：飛鷹是飛行速度最快、膽量最大的鳥類，世界上許多國家的空軍都用飛鷹當成象徵，在空軍軍徽的飛鷹代表「有我無敵，冒險敢死」之精神。 
*[[中華民國國花|國花]][[梅花|梅花]]象徵克服惡劣環境、堅忍不拔之精神，代表國軍軍人有克服自然堅忍不拔之毅力，為救國戰勝一切的本能，贏得最後之勝利。
*嘉禾代表祥瑞及豐收戰果之意。

== 校歌 ==
空軍官校校歌，由[[劉雪庵|劉雪庵]](盦)作曲，[[蔣中正|蔣中正]]頒詞。
歌詞：
得遂凌雲願，空際任迴旋，報國懷壯志，正好乘風飛去，長空萬里[[反攻大陸|復我舊河山]]，努力，努力，莫偷閒苟安，[[中華民族|民族]]興亡責任待吾肩，須具有犧牲精神，憑展雙翼，一衝天。

==榮譽徽==
*空軍軍官學校學生佩掛榮譽徽章，授予入伍成績合格者，並於新生入校後滿一年，升上二年級前頒發。
*象徵成為肩負「榮譽」、「責任」、「國家」的官校生。
