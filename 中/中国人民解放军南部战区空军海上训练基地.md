'''中国人民解放军南部战区空军海上训练基地'''，位于[[广西壮族自治区|广西壮族自治区]][[北海市|北海市]]，隶属[[中国人民解放军空军南宁基地|中国人民解放军空军南宁基地]]。

== 沿革 ==
2012年，'''中国人民解放军广州军区空军海上训练基地'''成立，隶属[[中国人民解放军广州军区空军|中国人民解放军广州军区空军]]<ref name=xin/><ref>{{Cite web |url = http://cpc.people.com.cn/n/2013/0420/c87228-21207927.html |title = 空军组织多兵机种海上搜索营救联合演练 |author =  |date =  2013-04-20|publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。该训练基地是空军海上训练保障专设机构，担负空军航空兵海上突防突击、飞行员海上跳伞救生等保障任务<ref name=talang>{{Cite web |url =  http://kj.81.cn/content/2015-10/14/content_6721929.htm  |title =踏浪逐涛海天间  |author =  |date =2015-10-14  |publisher =  中国空军网|language =  |accessdate =  |quote =  }}</ref>。成立后，该训练基地分住“两省四片”，面临“有部队缺营房、有船艇缺码头、有任务缺标准”的困难。2016年，更名'''中国人民解放军南部战区空军海上训练基地'''，隶属[[中国人民解放军空军南宁基地|中国人民解放军空军南宁基地]]。2017年1月22日，南部战区空军海上训练基地进驻新营区仪式举行，南部战区[[中国人民解放军空军南宁基地|空军南宁基地]]政委[[徐桂明|徐桂明]]，中共北海市委副书记[[蔡中平|蔡中平]]等30多位军地各单位领导以及海上训练基地全体官兵参加了进驻仪式，南部战区空军南宁基地副司令员杨明君宣布了进驻命令，该海上训练基地政委吴富伟作了进驻致辞<ref name=xin>{{Cite web |url = http://kj.81.cn/content/2017-02/07/content_7479345.htm |title =这些在海上训练的空军官兵终于有了新营区啦！  |author =  |date = 2017-02-07 |publisher = 中国空军网 |language =  |accessdate =  |quote =  }}</ref>。
