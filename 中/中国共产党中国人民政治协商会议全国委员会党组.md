'''中国共产党中国人民政治协商会议全国委员会党组'''（简称'''中共政协全国委员会党组'''，或'''中共全国政协党组'''），由[[中国共产党中央委员会|中国共产党中央委员会]]批准成立。

==沿革==
中国共产党[[中国人民政治协商会议全国委员会|中国人民政治协商会议全国委员会]][[党组|党组]]从[[中国人民政治协商会议第七届全国委员会|中国人民政治协商会议第七届全国委员会]]开始成立，此后历届也都设立<ref name=qgzxdz>{{Cite web |url = http://www.cppcc.gov.cn/2011/09/26/ARTI1317001118843697.shtml |title =中国共产党中国人民政治协商会议全国委员会党组  |author =  |date =  2011-09-26|publisher = 中国人民政治协商会议全国委员会 |language =  |accessdate =  |quote =  }}</ref>。

自第七届全国政协党组成立时，开始制定中共政协全国委员会党组工作规则，第八届全国政协党组作了较大的修改，规定了全国政协党组的领导关系，进一步明确、规范了全国政协党组的工作任务和会议制度。2003年3月26日，中共政协第十届全国委员会党组第一次会议通过了《中共政协全国委员会党组工作规则》<ref>{{Cite web |url =http://news.xinhuanet.com/newscenter/2004-09/08/content_1983488.htm  |title =以修改章程为契机大力推进政协工作的制度化  |author =  |date = 2004-09-08 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

==职责==
中国共产党中国人民政治协商会议全国委员会党组的任务主要是负责实现[[中国共产党|中国共产党]]的路线、方针、政策，讨论和决定中国人民政治协商会议全国委员会的重大问题，团结非党干部和群众，完成党和国家交给的任务，指导机关和直属单位中国共产党的组织的工作<ref name=qgzxdz/>。

中国共产党中国人民政治协商会议全国委员会党组负责领导[[中国共产党中国人民政治协商会议全国委员会机关党组|中国共产党中国人民政治协商会议全国委员会机关党组]]，以及[[中国人民政治协商会议全国委员会专门委员会|各专门委员会]]分党组。
