'''中国－西班牙关系'''（{{lang-es|Relaciones entre la China y España}}），又稱'''中西关系'''，是指[[西班牙|西班牙王国]]和[[中華人民共和國|中華人民共和國]]之間的雙邊關係，在1973年建立<ref name="xh">[http://news.xinhuanet.com/english/2009-01/30/content_10738354.htm China-Spain relations see steady growth]</ref>。現時[[中國|中國]]稱其關係為「全面戰略夥伴關係」。

== 歷史 ==
在1973年，當雙方剛結束了[[內戰|內戰]]時，雙方宣佈建交<ref name="xh"/>。1973年3月9日建交后，开始派遣驻[[西班牙|西班牙]]特命全权大使<ref>{{cite web|url=http://www.mfa.gov.cn/chn/pds/ziliao/wjrw/2167/2172/t9262.htm|title=驻西班牙王国历任大使|publisher=中华人民共和国外交部}}</ref>。自1994年6月29日中国与[[安道尔公国|安道尔公国]]建交后，驻[[西班牙|西班牙]]大使兼任驻[[安道尔|安道尔]]大使。

在幾次的交流後，雙方關係漸漸變好<ref>people.com.cn, June 10, 2008, [http://english.people.com.cn/90001/90776/90883/6426839.html Spain, China pledge to deepen relations]</ref>。中國參與了[[2008年世界博覽會|薩拉戈薩世界博覽會]]，在隨後的[[中國2010年上海世界博覽會|中國2010年上海世界博覽會]]西班牙也派了代表出席。

== 經貿關係 ==
雙方2004年的總貿易額為72億[[美元|美元]]，在2008年提升至227億[[美元|美元]]<ref name="xh"/>。2013年，双边贸易额为249.1亿美元，同比增长1.4%。其中中國出口189.3亿美元，同比增长3.8％，进口59.8亿美元，同比下降5.5％。

中國已成為了西班牙第六大的貿易夥伴<ref name="xh"/>。

双方签有引渡条约、被判刑人移管条约等協議<ref name="cn">[http://wcm.fmprc.gov.cn/pub/chn/gxh/cgb/zcgmzysx/oz/1206_41/1206x1/t7362.htm 中国同西班牙的关系] {{webarchive|url=https://web.archive.org/web/20140714123906/http://wcm.fmprc.gov.cn/pub/chn/gxh/cgb/zcgmzysx/oz/1206_41/1206x1/t7362.htm |date=2014-07-14 }}</ref>。
== 互訪 ==
西班牙王室成員曾多次訪問中國。费利佩六世作为王储期间曾多次访华<ref name="cn"/>。同樣的，時任[[中国国务院|中国国务院]]总理[[温家宝|温家宝]]也曾会见了西班牙王储费利佩·德博尔冯－格雷西亚。国家主席胡锦涛在2005年应西班牙国王胡安•卡洛斯一世的邀请下也到了西班牙訪問。

== 爭議 ==
2004年[[中国|中国]]向西班牙销售商品将近500万美元，购买[[西班牙|西班牙]]商品仅为约155万美元；而西班牙的经济资料称，从中国的进口几乎超过向中国出口的10倍，造成了[[西班牙人|西班牙人民]]對中國商人銷售不良產品的不滿<ref>[http://www.embajadachina.es/chn/ 中华人民共和国驻西班牙大使馆] {{Wayback|url=http://www.embajadachina.es/chn/ |date=20110714075019 }}</ref>。
