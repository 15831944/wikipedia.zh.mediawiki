'''中華民國與東加關係'''是指[[中華民國|中華民國]]與[[東加王國|東加王國]]之間的關係。兩國於1972－1998年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對東加的相關事務由'''中華民國駐[[斐濟|斐濟]][[台北经济文化代表处|商務代表團]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-264-41bde-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref><ref>{{Cite web |url = http://www.roc-taiwan.org/fj/post/7.html |title = 代表處簡介 |publisher = 中華民國駐外單位聯合網站}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，[[東加|東加]]當時雖已獨立但未加入，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
{{see also|中華民國外交|{{link-en|東加外交|Foreign relations of Tonga}}}}
{{see also|中國駐東加大使列表{{!}}中華民國駐東加王國外交官列表}}
1972年4月10日，中華民國與東加王國建立大使級[[外交#外交上的承認|外交關係]]。<ref name="鄭重聲明"/>

1975年6月，於首都[[努瓜婁發|努瓜婁發]]設立'''中華民國駐東加王國[[外交代表機構#大使馆|大使館]]'''，並派駐[[代辦|代辦]]。1977年，代辦升格為[[特命全權公使|大使]]。<ref name="鄭重聲明"/>

1987年8月2日，東加副總理杜伊達抵台訪問。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=1397F195EADEAB23 |author = 公眾外交協調會 |date = 1987年8月1日 |title = 東加王國副總理杜伊達閣下等一行三人 |publisher = 中華民國外交部}}</ref>

1991年2月，{{link-en|東加政府|Politics of Tonga}}任命[[大同公司|大同公司]]總經理[[林蔚山|林蔚山]]為'''駐台北[[領事|名譽總領事]]'''。<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-1.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1991年3月7日，東加王儲兼外交暨國防大臣[[乔治·图普五世|杜包]]親王抵台訪問。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=0D8608172C7EF665 |author = 公眾外交協調會 |date = 1991年3月6日 |title = 東加王國王儲兼外交暨國防部長杜包塔親王殿下 |publisher = 中華民國外交部}}</ref>

1992年10月，東加國防司令杜包中校抵台參加[[中華民國國慶|中華民國國慶]]活動。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=1EA956DAA598B877 |author = 公眾外交協調會 |date = 1992年10月10日 |title = 外交部於十月十日下午五時至七時在臺北賓館舉行盛大國慶酒會 |publisher = 中華民國外交部}}</ref>

1998年7月，東加國王[[陶法阿豪·圖普四世|杜包四世]]第9度訪問中華民國。<ref name="鄭重聲明"/>{{noteTag|前8次分別為1977年、1978年、1982年、1986年、1990年、1991年、1994年及1997年。<ref name="外交年鑑"/>}}

1998年11月2日，東加與中華民國斷交。<ref name="鄭重聲明">{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=5028B03CED127255&sms=5ED24855AD8E6C58&s=05CC4A6F4B8105B9 |title = 中華民國政府茲鄭重聲明自本年十一月二日起終止與東加王國之外交關係，同時中止中、東加兩國各項協定及合作關係，並召回駐東加王國大使 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境東加。]]
[[File:Visa_requirements_for_Tongan_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|東加簽證政策|{{link-en|東加公民簽證要求|Visa requirements for Tongan citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]與東加入境許可證的[[中華民國國民|中華民國公民]]在首都的[[努瓜婁發國際機場|努瓜婁發國際機場]]入境時，可獲得停留最多1個月的觀光簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-264-41bde-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持有{{link-en|東加護照|Tongan passport}}的{{link-en|東加人口|Demographics of Tonga|東加公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將東加歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2016年12月8日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-142-14617-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將東加歸類為'''第二級警示'''：對當地採取加強防護（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=439F527F9B583C7C |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
2018年，東加是中華民國第165大貿易夥伴、第223大[[進口|進口]]夥伴（在[[大洋洲國家和地區列表|大洋洲國家]]排名倒數第1；在[[世界政區索引|世界經濟體]]倒數第18）、第155大[[出口贸易|出口]]夥伴。出口至東加的金額為319萬7,189[[美元|美元]]，年減29.705%；自東加進口的金額為2,175美元，年增185.433%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）319萬5,014美元，年減29.741%。2017年，東加是中華民國第166大貿易夥伴、第230大進口夥伴（在大洋洲國家排名倒數第1；在世界經濟體倒數第7）、第144大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
