'''中国人民解放军第四五二医院'''，位于[[四川省|四川省]][[成都市|成都市]][[锦江区|锦江区]]工农院街1号，隶属[[中国人民解放军西部战区空军|中国人民解放军西部战区空军]]后勤部，为综合性[[三级甲等医院|三级甲等医院]]。

==简介==
医院创建于1950年2月，前身是[[中国人民解放军第一野战军|中国人民解放军第一野战军]]第三野战医院。1994年2月，正式更名为'''中国人民解放军第四五二医院'''。医院是综合性[[三级甲等医院|三级甲等医院]]，也是军队中心医院<ref name=jy>{{Cite web |url =http://news.163.com/10/0310/05/61D0MPQI000146BB.html  |title = 解放军第四五二医院60岁 |author =  |date =  2010-03-10|publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref name=youshi>{{Cite web|url =http://www.cdrb.com.cn/html/2015-10/28/content_2251207.htm|title =发挥五大品牌优势 打造群众信得过医院|author =|date =2015-10-28|publisher =成都日报|language =|accessdate =|quote =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。截至2015年，医院拥有1个全军重点专病中心（全军消化道肿瘤诊疗专病中心）、3个军区重点专科中心（成都军区医学矫形外科、泌尿外科、中西医结合肿瘤诊疗专科中心），以及成都军区医学妇科肿瘤专病中心<ref name=youshi/>。

医院曾先后两次参加[[对越自卫还击作战|对越自卫还击作战]]，参加了1986年[[两山轮战|“两山”作战]]，执行[[汶川大地震|汶川大地震]]、[[雅安地震|雅安地震]]抗震救灾等任务。2008年，因在汶川大地震抗震救灾中成绩突出，医院立集体三等功<ref name=youshi/><ref>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536914.htm |title =第四五二医院  |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，医院由[[中国人民解放军成都军区|中国人民解放军成都军区]]联勤部转隶[[中国人民解放军西部战区空军|中国人民解放军西部战区空军]]后勤部<ref name=tjp>{{Cite web |url = http://xb.81.cn/content/2016-07/20/content_7164513.htm |title = 解放军第452医院修建停机坪搭建“空中救援”快速通道 |author =  |date =  2016-07-20|publisher =军报记者  |language =  |accessdate =  |quote =  }}</ref>。
