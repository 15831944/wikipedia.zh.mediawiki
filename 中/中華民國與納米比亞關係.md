'''中華民國與納米比亞關係'''是指[[中華民國|中華民國]]與[[納米比亞共和國|納米比亞共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對納米比亞的相關事務由位於[[南非共和國|南非共和國]]的'''{{link-en|駐開普敦台北聯絡辦事處|Taipei Liaison Office in Cape Town}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-179-934da-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，[[納米比亞|納米比亞]]當時屬於[[南非|南非]]領土（[[西南非洲|西南非洲]]），由南非作為[[主權國家|主權國家]]參與投票。]]

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|納米比亞簽證政策|Visa policy of Namibia}}|{{link-en|納米比亞公民簽證要求|Visa requirements for Namibian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]與邀請函的[[中華民國國民|中華民國公民]]可前往納米比亞駐[[香港|香港]][[總領事館|總領事館]]申請簽證。經由[[南非機場列表|南非國際機場]]轉機入出境納米比亞，不需申辦南非[[過境簽證|過境簽證]]；若出機場再轉搭陸上交通工具前往納米比亞，因已實際入境南非，故須申辦[[南非簽證政策|南非簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-179-934da-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|納米比亞護照|Namibian passport}}的{{link-en|納米比亞人口|Demographics of Namibia|納米比亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將納米比亞歸類為'''灰色警示'''：提醒注意（2018年10月19日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-250-42d33-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|納米比亞經濟|Economy of Namibia}}}}
===貿易===
2017年，納米比亞是中華民國第103大貿易夥伴、第80大[[進口|進口]]夥伴、第157大[[出口贸易|出口]]夥伴。出口至納米比亞的金額為262萬9,266[[美元|美元]]，年減65.226%；自納米比亞進口的金額為5,083萬7,942美元，年增1,985.017%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）4,820萬8,676美元，年減1,041.087%。※2016年，納米比亞是中華民國第143大貿易夥伴、第138大進口夥伴、第128大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
