'''中央军委后勤保障部财务结算中心'''，位于[[北京市|北京市]]，隶属[[中央军委后勤保障部|中央军委后勤保障部]]。

==简介==
2016年[[深化国防和军队改革|深化国防和军队改革]]前，[[中国人民解放军总后勤部|中国人民解放军总后勤部]]设有'''中国人民解放军总后勤部财务结算中心'''，负责军委总部机关部门财务结算工作<ref>{{Cite web |url = http://politics.people.com.cn/n/2014/0109/c70731-24068785.html |title = 总后财务部负责人解读行政消耗费用限额管理办法 |author =  |date =2014-01-09  |publisher =  人民网|language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，撤销中国人民解放军总后勤部，成立[[中央军委后勤保障部|中央军委后勤保障部]]<ref>{{Cite web |url =http://theory.people.com.cn/n1/2016/0112/c40531-28040741.html  |title = 军委机关调整组建任务基本完成总部制改为多部门制 军委机关设15个职能部门 |author =  |date = 2016-01-12 |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。中央军委后勤保障部设有'''中央军委后勤保障部财务结算中心'''，负责该部的财务结算工作<ref>李岚，试论国库集中支付制度的改革与完善，经济管理(文摘版)2016年第04月14卷</ref>。
