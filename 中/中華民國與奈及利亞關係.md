'''中華民國與奈及利亞關係'''是指[[中華民國|中華民國]]與[[奈及利亞聯邦共和國|奈及利亞聯邦共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，但於對方最大城市互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[奈及利亞|奈及利亞]]。]]

尼日利亚在1960年10月1日宣佈獨立後，[[中華人民共和國政府|中華人民共和國政府]]的[[周恩来|周恩来]]和[[陈毅|陈毅]]分別與{{link-en|尼日利亚政府|Federal government of Nigeria}}高層通電話，表示祝贺<ref name=esta>{{cite book|author1=钱其琛|author2=王泰平|author3=张光佑|author4=马可铮|title=新中国外交50年|issue=2|publisher=北京出版社|year=1999}}</ref>{{Rp|692}}；但是，独立后的尼日利亚[[政变|政变]]頻繁，更曾有[[尼日利亚内战|内战]]爆發，政局不穩，因此中華人民共和國並未立即與尼日利亚[[外交#外交上的承認|建交]]<ref name=resist>{{cite book|author1=裴坚章|issue=3|title=中华人民共和国外交史: 1970-1978|publisher=世界知识出版社|year=1994}}</ref>{{Rp|173}}。

此後，[[中華民國政府|中華民國政府]]曾拉拢尼日利亚，派員訪問當地，但尼日利亚也沒有與中華民國建交<ref name=resist/>{{Rp|173}}。

1964年，[[台灣電力公司|台電]]總經理[[孫運璿|孫運璿]]因為在台電的成績受到[[世界銀行|世界銀行]]的青睞，且考慮到家中人口浩繁，薪水不敷開銷，便前往當時受世界銀行扶助的[[奈及利亞|奈及利亞]]國家電力公司，擔任執行長一職。<ref>{{Cite Web|url=http://smart-grid.tmcnet.com/news/2006/02/25/1406680.htm|title=Architect of Taiwan's industrial revolution|language=en|work=Smart Grid|date=2006-02-25|quote=From 1946 to 1964 Sun worked for Taiwan's electric utility, rising from senior engineer to president. He then worked for three years as chief executive officer and general manager of the Electricity Corporation of Nigeria, shortly after the African country became independent.|accessdate=2010-04-28}}</ref>在任職的3年內，孫運璿率領國際團隊推動[[尼日爾河|尼日河]]水力發電計畫，奈及利亞的發電量在任內增加了88%，1967年辭職返台。

1971年2月，尼日利亚與中華人民共和國建立外交關係<ref>{{cite book|title=中国近現代史大事记 (1840-1980)|publisher=知识出版社|year=1982|url=https://books.google.com.hk/books?id=LOs5AAAAMAAJ}}</ref>{{Rp|183}}。同年10月25日，奈及利亞支持中華人民共和國取代中華民國在[[聯合國|聯合國]]裡的[[中國與聯合國#代表權問題|「中國」席位]]。

====2017年尼日利亞要求中華民國代表團更名、遷址事件====
2017年1月11日，{{link-en|尼日利亚政府|Federal government of Nigeria}}發表聲明，基於「[[一個中國|一個中國]]」原則，不再承認臺灣是一個國家並終止外交關係，還要求中華民國代表團從首都阿布加遷往第一大城拉哥斯以及可能摘牌更名的處境。但{{Link-en|奈及利亞總統府|Aso Villa}}後來發布聲明澄清，表示有關奈及利亞與台灣中止外交關係的訊息並不正確，雙方關係是貿易代表處層級。聲明內容並未提及要求台灣遷館等情形。<ref>{{Cite web |url = https://udn.com/news/story/1/2225932 |author = 洪哲政 |date = 2017年1月12日 |title = 奈及利亞羞辱台灣有價 答案是這數字… |publisher = [[聯合新聞網|聯合新聞網]]}}</ref><ref>{{Cite web |url = http://www.ettoday.net/news/20170113/848245.htm |author = 國際中心 |date = 2017年1月13日 |title = 要台外館遷出首都？ 奈及利亞：台灣是貿易代表處層級 |publisher = [[ETtoday東森新聞雲|ETtoday東森新聞雲]]}}</ref><ref>{{Cite web |url = http://news.ltn.com.tw/news/focus/paper/1071041 |author = 呂伊萱 |date = 2017年1月14日 |title = 外交部：奈及利亞案 仍在努力中 |publisher = [[自由時報|自由時報]]}}</ref>

2017年3月31日，奈及利亞政府向中華民國代表施壓，限期一周內離開首都，否則將不保證其安全。代表[[趙家寶|趙家寶]]被召回，且自4月6日起，駐奈及利亞代表團暫停運作，改由'''{{link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''暫代。6月14日，[[中華民國外交部|中華民國外交部]]表示，已召見奈及利亞駐台代理代表嚴正抗議，將視後續狀況，對其駐台辦事處採取對等措施<ref>{{cite web|title=駐奈及利亞代表被迫回台 外交部抗議 |url=http://www.cna.com.tw/news/aipl/201706140357-1.aspx |publisher=[[中央社|中央社]] |accessdate=2017-06-14}}</ref><ref>{{cite web|title=我駐奈外館遭施壓 外交部要求奈國駐台代表處搬離 |url=http://news.ltn.com.tw/news/politics/breakingnews/2100137 |publisher=自由時報 |accessdate=2017-06-14}}</ref><ref name="駐館位置及聯絡方式">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-41-66c44-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref>。30日，奈及利亞政府派遣25名武警，封鎖中華民國駐奈及利亞代表團，並強制驅離駐代表團人員。中華民國外交部對於奈及利亞政府強制封鎖代表團表達嚴正抗議，並呼籲奈及利亞政府停止採取極端做法，立刻撤離武警，改以理性對話方式處理。<ref>{{cite web|title=粗暴！奈及利亞武警強驅我駐處人員 外交部嚴正抗議 |url=https://udn.com/news/story/6656/2557512 |publisher=聯合新聞網 |accessdate=2017-07-01}}</ref>7月4日，中華民國外交部決定將代表團從首都阿布加遷至舊都拉哥斯，預計三個月內完成遷址作業，但奈及利亞武警封鎖代表團阻礙遷址作業，中華民國外交部與奈及利亞政府交涉，希望武警儘速撤離，讓駐館人員繼續搬遷事宜。<ref>{{cite web|title=駐奈及利亞代表處遭封 搬遷作業延宕 |url=http://www.cna.com.tw/news/aipl/201707040118-1.aspx |publisher=中央社 |accessdate=2017-07-04}}</ref>9月7日，中華民國外交部表示奈及利亞駐館搬遷拉哥斯已找到適合地點，預計年底前可完成搬遷，同時依對等原則將要求位於台北市[[信義區_(臺北市)|信義區]]的奈及利亞駐華商務辦事處遷離首都台北市，對此奈及利亞駐華商務辦事處回應已收到通知，搬遷時程與地點等事宜還在等待奈國政府指示。<ref>{{cite web|title=外交部：年底前完成奈及利亞駐館搬遷 |url=https://money.udn.com/money/story/5641/2687974 |publisher=聯合報 |accessdate=2017-09-07}}</ref><ref>{{cite web|title=奈及利亞駐處搬遷 改名台北貿易辦事處 |url=http://www.cna.com.tw/news/aipl/201801050129-1.aspx |publisher=中央社 |accessdate=2018-01-05}}</ref>

2017年12月8日，駐館从阿布加搬迁到拉哥斯，2018年1月5日正式对外运作，3月5日起領事業務重新對外開放；然而駐館名称从中华民国驻奈及利亚联邦共和国商务代表团更名为'''{{Link-en|駐奈及利亞聯邦共和國臺北貿易辦事處|Taipei Trade Office in the Federal Republic of Nigeria}}'''（Taipei Trade Office in the Federal Republic of Nigeria）<ref name="駐館位置及聯絡方式"/><ref>{{Cite web |url = http://dailynews.sina.com/gb/tw/twpolitics/cna/20180104/21348200647.html |author = 侯姿莹 |date = 2018年1月4日 |title = 奈及利亚驻处搬迁 改名台北贸易办事处 |publisher = [[新浪|新浪]]}}</ref>。

2018年3月14日，奈及利亞駐華商務辦事處依對等原則搬遷到[[新北市|新北市]][[板橋區|板橋區]]。<ref>{{cite web|title=奈及利亞駐台辦事處遷至新北市 下週運作|url=http://www.cna.com.tw/news/aipl/201803210311-1.aspx|publisher=中央社|accessdate=2018-03-21|date=2018-03-21}}</ref>

兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往在新北市的奈及利亞駐華商務辦事處申請簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-41-66c44-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|奈及利亞護照|Nigerian passport}}的{{link-en|奈及利亞人口|Demographics of Nigeria|奈及利亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將奈及利亞按地區歸類為'''橙色警戒'''：避免非必要旅行，與'''紅色警戒'''：不宜前往，宜儘速離境（以上截至2018年8月31日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-129-ee7b4-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將奈及利亞歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年11月6日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=93FBC5CC064A0D4A |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於奈及利亞第1大城[[拉哥斯|拉哥斯]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#AFRICA |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>2015年7月2日，[[經濟部國際貿易局|經濟部國際貿易局]]則在首都[[阿布加|阿布加]]設立'''駐奈及利亞代表處經濟組'''。<ref>{{Cite web |url = http://www.taiwanembassy.org/ng/post/6.html |title = 代表處簡介 |publisher = 中華民國駐外單位聯合網站}}</ref><ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=323&pid=310534 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2017年，奈及利亞是中華民國第52大貿易夥伴、第43大[[進口|進口]]夥伴、第75大[[出口贸易|出口]]夥伴，在[[非洲国家和地区列表|非洲經濟體]]排名第3、3、4。出口至奈及利亞的金額為9,828萬2,680[[美元|美元]]，年減3.541%；自奈及利亞進口的金額為4億4,129萬1,009美元，年增69.262%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）3億4,300萬8,329美元，年增115.966%。※2016年，奈及利亞是中華民國第59大貿易夥伴、第46大進口夥伴、第73大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =經濟部國際貿易局}}</ref>

===投資===
[[臺商|臺商]]在奈及利亞投資有30餘家，總投資金額約2億美元。2016年10月9日，成立奈及利亞臺灣商會聯合總會。近年，由於奈及利亞景氣衰退，臺商人數不增反減。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

===合作===
中華民國根據1994年4月簽署的《[[加工出口區|加工出口區]]經濟合作協定》，於8月派駐顧問，在奈及利亞[[卡拉巴爾|卡拉巴爾]]加工出口區技術指導及運作，後因協定於1997年4月終止而未再派駐顧問。<ref name="86年外交年鑑"/>

===犯罪事件===
{{see also|尼日利亞騙徒{{!}}奈及利亞騙徒}}
2008年，中華民國[[內政部移民署|內政部移民署]]和新竹市警方聯手逮捕1名奈及利亞籍男子，以成立[[空殼公司|空殼公司]]投資[[汽車零組件列表|汽車零件]]出口為由，詐騙1名菲籍台裔女子數十萬[[新台幣|新台幣]]。<ref>{{Cite web |url = http://street530.pixnet.net/blog/post/26524037 |author = [[聯合報|聯合報]]記者彭淵燦、黃瑞典 |date = 2008年8月16日 |title = 奈及利亞人來台騙財騙色 |publisher = [[痞客邦|痞客邦]]}}</ref>

2010年，1名奈及利亞籍男子，於2001年利用在[[馬來西亞|馬來西亞]]購買的假護照入台[[臨時工|打工]]，被發現並遣返出境之後，又利用[[交友網站|交友網站]]和1名住在[[高雄|高雄]]的台灣女子結婚，再以依親方式入台打工，該女子發現這名男子結婚後就失蹤，懷疑被利用，於是向警方舉發，[[基隆市警察局|基隆市警方]]前往高雄，將準備潛逃出境的男子逮捕。<ref>{{Cite web |url = http://news.cts.com.tw/cts/society/201012/201012160631881.html#.WSN14cYlHDc |author = 綜合報導 |date = 2010年12月16日 |title = 奈國男子騙婚 異國戀破局! |publisher = [[華視新聞網|華視新聞網]]}}</ref>

2013年，2名奈及利亞籍的兄弟，哥哥多年前持觀光簽證入台逾期未返國，非法居留台灣多年，因為無合法身分，行動常受到限制，於是要弟弟以來台投資成立公司為由，順利取得[[中華民國居留證|居留簽證]]。兄弟倆利用台灣人分不清[[黑人|黑人]]長相的特性，哥哥便拿著弟弟的護照和居留簽證，再以自己的相片成功申辦外僑居留證和[[中華民國汽車駕駛執照|駕照]]，有了合法身分便趁機拐騙台灣女子。後來[[中華民國警察|警方]]和內政部移民署清查時，發現這2人身分竟然重複感到懷疑，經過數月蒐證後於[[新北市|新北市]]將2人逮捕，弟弟被逮時還自豪說「2年來我交往過的台灣女生超過百人！」<ref>{{Cite web |url = http://www.nownews.com/n/2013/03/21/307202 |author = 社會中心 |date = 2013年3月21日 |title = 台灣女性好好騙？黑人兄弟冒用身分 2年睡過上百女 |publisher = [[NOWnews_今日新聞|NOWnews 今日新聞]]}}</ref>

2016年，中華民國[[刑事警察局|刑事警察局]]破獲俗稱「[[奈及利亞419|奈及利亞419]]」的戀愛交友[[詐騙集團|詐騙集團]]，共逮捕[[中華民國籍|中華民國籍]]2名女子與4名[[車手|車手]]，其涉嫌與1名奈及利亞籍男子合謀詐騙。男子先以盜用的白人帥哥照片，在網路搭訕有錢女性取得個資，後續交由2名女同夥以流利的[[英文|英文]]行騙，被害人遍及[[東南亞|東南亞]]等12國，得手[[贓款|贓款]]轉由台灣的車手領取，獲利至少2,000萬新台幣。<ref>{{Cite web |url = http://www.chinatimes.com/realtimenews/20161226003702-260402 |author = 胡欣男 |date = 2016年12月26日 |title = 奈及利亞419詐騙 刑事局逮6台嫌 |publisher = [[中時電子報|中時電子報]]}}</ref>
