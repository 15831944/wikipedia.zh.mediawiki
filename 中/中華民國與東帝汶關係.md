'''中華民國與東帝汶關係'''是指[[中華民國|中華民國]]與[[東帝汶民主共和國|東帝汶民主共和國]]之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對東帝汶的相關事務由'''{{link-en|駐印尼台北經濟貿易代表處|Taipei Economic and Trade Office, Jakarta, Indonesia}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-120-cc80a-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，[[東帝汶|東帝汶]]當時屬於[[葡屬帝汶|葡萄牙殖民地]]，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
{{see also|中華民國外交|{{link-en|東帝汶外交|Foreign relations of East Timor}}}}
中華民國曾在[[葡屬帝汶|葡屬帝汶]]首都設有'''駐[[狄力|狄力]][[領事館|領事館]]'''，1974年4月25日，[[康乃馨革命|葡萄牙軍事政變]]後，葡屬帝汶亦發生內戰，導致[[东帝汶印度尼西亚占领时期|印尼軍事介入]]，中華民國於1975年12月28日關閉領事館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1998/section_3/page3-1-4.htm |title = 《中華民國87年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1998年11月26日，{{link-en|1999年東帝汶獨立公民投票|East Timorese independence referendum, 1999|東帝汶獨立運動}}領袖之一與1996年[[諾貝爾和平獎|諾貝爾和平獎]]得主[[若澤·拉莫斯·奧爾塔|霍塔]]應[[臺北市長|臺北市長]][[陳水扁|陳水扁]]的邀請訪問[[臺灣|臺灣]]，但被[[內政部移民署|內政部警政署入出境管理局]]列為不受歡迎人物而拒絕入境（由{{link-en|印尼政府|Government of Indonesia}}建議），後[[中華民國外交部|中華民國外交部]]查核後同意入境。<ref>{{Cite web |url = http://news.cts.com.tw/cts/general/199811/199811260021967.html |author = [[華視新聞|華視新聞]] |date = 1998年11月26日 |title = 霍塔入境被拒民進黨痛批 |publisher = [[中華電視公司|中華電視公司]]}}</ref><ref>{{Cite web |url = http://forums.chinatimes.com/report/mayor/98112604.htm |author = 劉永祥 |date = 1998年11月26日 |title = 霍塔：無意介入台灣選舉 |publisher = [[中時電子報|中時電子報]]}}</ref>

2002年4月17日，[[中華民國總統|中華民國總統]]陈水扁与[[中華民國外交部#歷任首長|外交部长]][[简又新|简又新]]电贺[[沙納納·古斯芒|古斯芒]]当选[[东帝汶总统|东帝汶总统]]。5月20日，東帝汶正式從[[印尼|印尼]]獨立，外交部再次致电{{link-en|东帝汶政府|Politics of East Timor}}，表達了[[外交#外交上的承認|建立外交關係]]的意愿但失敗，東帝汶獨立當天即與[[中華人民共和國|中華人民共和國]]建交。<ref>{{Cite web |url = http://blog.renren.com/share/220845417/14411070572 |author = 李轩成 |date = 2013年10月30日 |title = 中国与东帝汶关系的历史、现状及前景 |publisher = [[人人網|人人網]]}}</ref>

=== 簽證 ===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境東帝汶。]]
[[File:Visa_requirements_for_East_Timorese_citizens.png|thumb]]的{{link-en|東帝汶人口|Demographics of East Timor|東帝汶公民}}必須申請[[簽證|簽證]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|東帝汶簽證政策|Visa policy of East Timor}}|{{link-en|東帝汶公民簽證要求|Visa requirements for East Timorese citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可於抵達首都[[狄力|狄力]]的[[尼古勞·洛巴托總統國際機場|尼古勞·洛巴托總統國際機場]]後申請[[簽證#落地签证|落地簽證]]入境東帝汶，停留最多30天。或至{{link-en|東帝汶駐外機構列表|List of diplomatic missions of East Timor|東帝汶駐外館處}}申請「簽證許可函」。經東帝汶過境轉機，亦須辦理落地簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-120-cc80a-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&sms=B75E93A3DAB23A99 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有[[東帝汶護照|東帝汶護照]]的{{link-en|東帝汶人口|Demographics of East Timor|東帝汶公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將東帝汶歸類為'''橙色警戒'''：避免非必要旅行（2014年10月2日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-143-d6864-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|東帝汶經濟}}
===貿易===
2018年，東帝汶是中華民國第171大貿易夥伴、第169大[[進口|進口]]夥伴、第163大[[出口贸易|出口]]夥伴。出口至東帝汶的金額為195萬6,664[[美元|美元]]，年增53.291%；自東帝汶進口的金額為31萬8,929美元，年減23.512%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）163萬7,735美元，年增90.552%。2017年，東帝汶是中華民國第180大貿易夥伴、第169大進口夥伴、第172大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
