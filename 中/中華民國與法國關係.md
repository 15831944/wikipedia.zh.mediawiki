'''中華民國與法國關係'''是指[[中華民國|中華民國]]與[[法蘭西共和國|法蘭西共和國]]之間的關係。1913－1964年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[法國|法國]]。]]
{{see also|中華民國外交|法國外交}}
{{see also|中華民國驻法国大使列表{{!}}中華民國駐法國外交官列表|法国驻中華民國大使列表{{!}}法國駐中華民國外交官列表}}
1912年，中華民國成立後，派駐法國[[特命全權公使|公使]]。

1913年10月6日，法國[[外交#外交上的承認|外交承認]]中華民國[[北洋政府|北洋政府]]，於首都[[巴黎|巴黎]]設立'''中華民國駐法國[[外交代表機構#公使馆|公使館]]'''。

1928年12月22日，隨著[[國民革命軍北伐|國民革命軍北伐]]幾近尾聲，法國外交承認中華民國[[國民政府|國民政府]]。

1936年2月8日，公使館升格為'''中華民國駐法國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權公使|大使]]。

1942年1月1日，中華民國與法國在內的26國簽訂《[[聯合國共同宣言|聯合國共同宣言]]》。

1943年8月1日，因[[第二次世界大战德占法国军事管辖区|納粹德國佔領法國]]，中華民國宣布與[[維希法國|法國維琪政府]]斷交並撤館。

1944年10月23日，[[諾曼第戰役|諾曼第登陸]]，同盟國軍[[解放巴黎|解放巴黎]]後，中華民國承認[[法國臨時政府|法國臨時政府]]，並任命錢泰為大使（但自1949年[[中華人民共和國開國大典|中華人民共和國成立]]後至1964年中華民國與法國斷交為止，中華民國政府未再派任駐法大使）。<ref>1949年后在台北的中华民国政府曾多次与法商谈派全权大使但均被婉拒，相继由[[段茂澜|段茂澜]]公使、[[陈雄飞|陈雄飞]]公使[[外交官#代办|代办]]使事，1963年由[[参赞|参事]][[高士铭|高士铭]]代办使事。参见周谷《外交秘闻：一九六〇年代台北[[華盛頓哥倫比亞特區|华府]]外交秘辛》。</ref>

1946年2月28日，兩國簽訂《中法關於法國放棄在華[[治外法權|治外法權]]及其有關特權條約》（中法平等新約）。<ref>{{cite web |url=http://www.npm.gov.tw/exh100/diplomatic/page_ch03.html |date=2011年 |title=《百年傳承　走出活路》－中華民國外交史料特展：力爭平等 |publisher=[[國立故宮博物院|國立故宮博物院]]}}</ref>

1947年5月1日，[[中華民國外交部|中華民國外交部]]就該年3月8日、4月5日法軍兩次轟炸中國[[越南華人|在越華僑]]集中處一事，向[[法國駐中華民國大使館舊址|法國大使館]]提出嚴重抗議；次日，外交部次長[[葉公超|葉公超]]赴法國大使館提出口頭抗議。<ref name="民國史大事記">{{cite book|editors=[[李新_(1918年)|李新]]總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室，韓信夫、姜克夫主編|title=《中華民國大事記》|location=北京|publisher=[[中華書局|中華書局]]|date=2011-07}}</ref>{{rp|8346}}

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，法國亦在首都[[台北市|台北]]保留'''法蘭西共和國駐中華民國大使館'''，但僅派駐[[代辦|代辦]]而非大使。

1950年11月30日，法國在[[聯合國安全理事會|聯合國安全理事會]]上[[聯合國安全理事會否決權|否決]]中共代表[[伍修權|伍修權]]等提出之「美國侵略台灣案」。<ref>{{cite conference|title=聯合國安全理事會正式紀錄 第七十二號 第五三0次會議|date=1950-11-30|publisher=[[聯合國安全理事會|聯合國安全理事會]]|location=[[紐約|紐約]]發拉星草場|url=http://www.un.org/en/ga/search/view_doc.asp?symbol=S/PV.530&referer=/english/&Lang=C|accessdate=2015-08-24|language=zh-hant}}</ref>

1960年代，法國開始與[[中華人民共和國|中華人民共和國]]進行關係正常化行動，當時[[中華民國政府|中華民國政府]]極力阻止，其間[[中華民國總統|中華民國總統]][[蔣中正|蔣中正]]還致親筆信給時任[[法國總統|法國總統]][[戴高樂|戴高樂]]以穩固邦交。

1964年1月27日，法國和中華人民共和國建交，中華民國外交部抗議法國與中共政權建交，表明反對「[[兩個中國|兩個中國]]」；1月25日，法國總統戴高樂派前駐華大使{{link-en|貝志高|Zinovy Peshkov}}向總統蔣中正解釋；2月7日，外交部重申反對「兩個中國」，指[[戴高樂主義|戴高樂中立政策]]是分期投降；2月10日，宣布與法國斷交。<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|698}}12日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>除了總部在法國巴黎的[[聯合國教育、科學及文化組織|聯合國教育、科學及文化組織]]（UNESCO）之外，兩國關係全面中斷。<ref name="代表處簡介"/><ref name="駐館與駐地關係">{{Cite web |url = http://www.taiwanembassy.org/fr/post/2936.html |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref>

1971年10月25日，作為[[聯合國安全理事會常任理事國|聯合國安全理事會常任理事國]]之一的法國支持中華人民共和國取得在聯合國裡的「[[中國與聯合國#代表權問題|中國]]」席次。

1996年5月，{{link-en|班基協議|Bangui Agreements|中非共和國發生兵變}}，中華民國政府請{{link-fr|法國駐中非共和國大使館|Ambassade de France en République centrafricaine|法國駐中非大使館}}與法國政府於必要時提供協助，以維護中華民國駐中非大使館團員眷屬之安全。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=180DB36A793AAC89 |author = 公眾外交協調會 |date = 1996年5月25日 |title = 中非共和國首都班基市於本（八十五）年五月十八日再度爆發士兵譁變事件 |publisher = 中華民國外交部}}</ref>

1972年8月12日，中華民國在首都巴黎設立[[法人#社團法人|具社團法人]]性質的'''法華經濟貿易觀光促進會'''（[[法文|法文]]：{{lang|fr|Association pour la Promotion des Echanges Commerciaux et Touristiques avec Taiwan, ASPECT}}）。1976年7月，[[中華民國經濟部|中華民國經濟部]]亦設立'''亞洲貿易促進會駐巴黎辦事處'''（法文：{{lang|fr|Centre Asia-tique de Promotion Economique et Commerciale, CAPEC}}），負責雙邊經貿事務。1995年5月15日，法國同意將兩機構合署更名為[[實質大使館|具大使館性質]]的'''[[駐法國台北代表處|駐法國台北代表處]]'''（法文：{{lang|fr|Bureau de Représentation de Taipei en France}}）。<ref name="代表處簡介">{{Cite web |url = http://www.taiwanembassy.org/fr/post/13.html |title = 代表處簡介 |publisher = 中華民國駐外單位聯合網站}}</ref><ref name="駐館與駐地關係"/><ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-4.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref name="103年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/09.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref><ref name="駐外館處">{{Cite web |url = http://www.mofa.gov.tw/OverseasOffice_Content.aspx?n=6357834932B83C83&sms=5A961ED2E4BA25E8&s=E6436B9A96D5AEE5&os=D7B9F1DAF6192D32 |title = 駐外館處 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>並於2007年11月起大幅改善駐處人員及眷屬的居留簽證待遇。<ref name="96年外交年鑑"/>

1978年10月，法國在首都台北設立'''法亞貿易促進會'''（法文：{{lang|fr|Association Française pour le Développement du Commerce avec l′Asie}}），推廣雙邊經貿關係。1980年1月，[[法國外交部|法國外交部]]亦以非官方名義設立'''台北法國文化科技中心'''（法文：{{lang|fr|Centre Culturel et Scientifique Française de Taipei}}），負責台法文化科技及高等教育的往來，對上則隸屬於{{Link-fr|法國駐香港總領事館|Consulat général de France à Hong Kong}}。1985年9月，成立'''旅遊簽證組'''，辦理[[欧洲联盟签证政策|赴法簽證]]。1989年2月，法國政府為加強台法雙邊關係，以[[美國在台協會|美國在台協會]]為參考先例，將台北法國文化科技中心更名為'''[[法國在台協會|法國在台協會]]'''（法文：{{lang|fr|Bureau Français de Taipei}}）。1994年，將法亞貿易促進會、旅遊簽證組、法國在台協會三機構合署辦公，統稱為法國在台協會。<ref name="駐館與駐地關係"/><ref name="86年外交年鑑"/><ref name="103年外交年鑑"/><ref>{{lang|en|{{Citation |url=http://www.sciencespo.fr/ceri/sites/sciencespo.fr.ceri/files/jpcabest.pdf |format=pdf |title=France's Taiwan Policy: A Case of Shopkeeper Diplomacy |author=[[高敬文|高敬文]] |date=2001 |accessdate=2014-10-03}}}}{{en}}</ref>{{rp|p4}}<ref name="駐華外國機構">{{Cite web |url = http://www.mofa.gov.tw/OfficesInROC_Content.aspx?n=8CEB2B5F5436B997&sms=8EBFADC1592C7BFE&s=E6436B9A96D5AEE5&os=D7B9F1DAF6192D32 |title = 駐華外國機構 |publisher = 中華民國外交部}}</ref>

1999年11月，馬特拉集團接替德國{{link-en|多尼爾公司|Dornier Flugzeugwerke}}與行政院國家科學委員會簽約，承製「[[中華衛星二號|中華衛星二號]]」本體。後來法國總統[[雅克·希拉克|席哈克]]於2000年10月訪問[[中華人民共和國|中華人民共和國]]時遭抗議該案，但法國仍堅持決定不變。1999年，法國[[達梭航太|達梭航太]]總裁{{link-zh-yue|賽吉達梭|塞爾日達梭}}及投資負責人{{link-fr|勞倫達梭|Laurent Dassault}}等人抵台訪問。<ref name="88年外交年鑑"/><ref name="89年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_8.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2016年9月30日，駐法國代表張銘忠應邀出席在[[土魯斯|土魯斯]]的[[空中巴士|空中巴士]]總部舉行[[中華航空|中華航空]]新機[[空中客车A350_XWB|A350-900]]首架交機儀式，並於2017年起投入歐洲航線。<ref name="105年外交年鑑"/>

2017年5月，中華民國[[經濟部工業局|經濟部工業局]]率領「臺法航空產業訪問團」參訪法國土魯斯的航空[[供應鏈|供應鏈]]，強化臺法航空製造與維修商機合作。6月，臺中市長[[林佳龍|林佳龍]]率領業者拜訪{{link-en|法國企業行動聯盟|Mouvement des Entreprises de France}}、{{link-en|商務投資署|Business France}}，並共同舉辦臺法智慧機械暨航太[[複合材料|複合材料]]媒合活動；經濟部「歐洲創新研發合作拓展團」參訪[[巴黎國際航空太空展|巴黎國際航空太空展]]，並與[[達梭系統|達梭系統]]簽署合作備忘錄。<ref name="106年外交年鑑"/>
