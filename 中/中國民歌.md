{{中国音乐}}
'''中國民歌'''包括用以[[演唱|演唱]]的[[歌|歌]]和用于[[吟诵|吟诵]]的[[谣|谣]]两种形式。中国民歌是中国民族民间音乐的一部分，由劳动人民集体创作，口头传唱，是民间文学的一种形式，民歌多为以吟诵为主的韵文。

==历史==
公元前5世纪左右编成的中国第一部诗歌总集《[[诗经|诗经]]》，分风、雅、颂三个部分，其中风的部分，就是当时各地的民歌。战国时期[[屈原|屈原]]、[[宋玉|宋玉]]等人所著《[[楚辞|楚辞]]》中也搜集了楚地的民歌。[[西汉|西汉]]时期，政府设立“[[乐府|乐府]]”，收集各地民歌，以察民情。宋朝的[[词_(文学)|宋词]]，元朝的[[元曲|元曲]]也都是有代表性的中国民歌。中华人民共和国成立之初，中国的民歌多以红色革命题材为主。改革开放之后，中国的民歌受到西方文化的影响，与西方文化交融，并逐渐类似于西方的流行音乐，著名的民歌组合以[[凤凰传奇|凤凰传奇]]，[[玖月奇迹|玖月奇迹]]为代表。

== 分类 ==
中国民歌当前不同的学者有着不同的分法：有的学者将其分为号子、山歌、小调三大类；还有的学者将其分为号子、山歌、小调、长歌与多声部歌曲四大类；还有的学者将其细化，分为号子、山歌、小调、儿歌、摇儿曲、宗教歌曲、长歌与多声部歌曲等。具体分类的依据有歌曲体裁，歌词题材与方言。

中國民歌大分为三类：
# '''号子'''：（也称之为劳动号子）是人们在体力劳动时候编唱、并直接为之服务的民歌。在中国，号子有不同的称呼：北方常称之为“吆号子”，南方常称之为“喊号子”。号子具有一领众和的演唱形式，歌曲节奏与劳动节奏统一，旋律节奏固定，一般演唱时为领者即兴，和者衬词的形式。号子从歌词题材上分为农事号子、工程号子、林业号子、船工号子、搬运号子与作坊号子。
# '''[[山歌|山歌]]'''：是指各种山野歌曲，它是人们在上山砍柴、田间劳作、野外放牧、停脚小歇时，为抒发内心情感，或向远处的人传递情意、对答传语的即兴编唱，多在户外演唱。具有直畅性、单纯性、自由性，同时旋律起伏大，多采用上下句结构，句子多对偶。可大致分为爬山调、山曲、[[信天游|信天游]]、花儿、客家山歌、闽粤赣山歌。
# '''[[小调_(中國音樂)|小调 (中國音樂)]]'''：小调即“里巷之乐”，也称“小令”，是人们日常生活的休息、娱乐、集会等场合中演唱的民间歌曲，也就是说，除了劳动号子、山歌以外的各种日常歌唱的小型民歌都可以归入这一类之中。小调常由职业或半职业的艺人演唱，流传性广，节奏均匀，旋律丰富，歌词多采用长短句。

中國民歌細分为六类：
# '''劳动歌'''：指劳动号子等各种在劳动中唱的歌。有的直接配合劳动动作，[[节奏|节奏]]十分鲜明；有的只是调节劳动情绪，如采茶歌、草原牧歌等。
# '''仪礼歌'''：专门用于各种[[民间仪式|民间仪式]]和[[民间礼俗|民间礼俗]]的民歌。如民间婚礼上使用的撒帐歌，民间建房时使用的建房歌。
# '''时政歌'''：主要内容是[[时事政治|时事政治]]评论的民间歌谣。其[[政治意义|政治意义]]高，有一定的[[文献|文献]]价值。
# '''生活歌'''：表现群众一般的[[家庭生活|家庭生活]]与[[劳动生活|劳动生活]]内容的民歌。如反映农民生活的《十二月长工歌》，集中揭示了[[长工|长工]]与[[地主|地主]]之间的尖锐矛盾。
# '''情歌'''：即有关爱情的民歌。它是民歌中数量最多，也最动人的一种。如[[四川|四川]]民歌《高高山上一树槐》，它所表现的是一位姑娘久等情人不至后的怅惘。情歌在中国部分少数民族中是青年人恋爱、择偶过程中不可缺少的[[交际|交际]]工具。
# '''儿歌'''：儿童演唱或[[朗诵|朗诵]]的民间歌谣。中国古代特定时期产生的童谣由于是时政预言性质的民歌，不属于儿歌。儿歌的内容主要是游戏儿歌，[[绕口令|绕口令]]以及大人逗小孩的歌。

== 特点与地位 ==
古代中國民歌是一种富于生命力的[[口头创作|口头创作]]，其优秀作品具有高度的[[艺术价值|艺术价值]]，相反，大部分近代中國民歌的艺术价值都很低。在[[中国文学史|中国文学史]]上，民歌是[[诗歌|诗歌]]与[[文学|文学]]的始祖，是[[作家|作家]]创作的源泉之一。四言诗、五言诗、七言诗以及词、[[元曲|曲]]等[[文学体裁|文学体裁]]都起源于古代民歌而后为作家所采用的。

== 各地民歌 ==
中国民歌的形式非常丰富。不同[[地区|地区]]各有区别。

中国民歌依照地区可分成以下几种种：

* 中原地区和东南沿海汉族民歌
* 北方草原牧区民歌
* 西部受佛教影响藏族民歌
* 西南高原多民族原始民歌
* 东北受[[萨满教|萨满教]]影响的狩猎民歌
* 西北高原半农半牧民歌
* 西部受[[伊斯兰文化|伊斯兰文化]]影响的新疆民歌

中国民歌四句一段者居多，但[[陕|陕]]北、[[晋|晋]]西北有两句一段的民歌。除[[七言体|七言体]]以外，又有五言、六言、八言以至十几言的[[诗句|诗句]]。在[[用韵|用韵]]方面，中國人民歌基本上都押[[尾韵|尾韵]]。南方的较为柔美含蓄，北方的较为直率粗放。
