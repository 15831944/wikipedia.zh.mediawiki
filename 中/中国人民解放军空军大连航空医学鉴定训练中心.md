'''中国人民解放军空军大连航空医学鉴定训练中心'''，位于[[辽宁省|辽宁省]][[大连市|大连市]]，隶属[[中国人民解放军北部战区空军|中国人民解放军北部战区空军]]后勤部。

==简介==
空军大连航空医学鉴定训练中心的前身是1953年成立的'''中国人民解放军空军大连疗养院'''。自成立后直到2009年，空军大连疗养院先后完成12万人次的疗养保健任务，完成了[[抗美援朝战争|抗美援朝]]特级战斗英雄群体、[[中国航天员大队|中国航天员大队]]、[[八一飞行表演队|空军八一飞行表演大队]]、中国第一次参加跨国军事演习的航空兵部队等的疗养任务，完成了8.5万多名飞行人员的体检鉴定任务<ref name=dltjw>{{Cite web|url = http://dalian.runsky.com/2009-09/01/content_3365146.htm|title = 空军大连航空医学鉴定训练中心成立|author = |date = 2009-09-01|publisher = 大连天健网|language = |accessdate = |quote = }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2009年7月，都江堰、杭州、大连、青岛、临潼五所疗养院统一由各大军区联勤部转隶各大军区空军后勤部，成立5个航空医学鉴定训练中心，构建为“[[中国人民解放军空军总医院|空军总医院]]、[[中国人民解放军空军航空医学研究所|空军航空医学研究所]]——区域性航空医学鉴定训练中心——航空兵部队卫生机构”三级航空医学鉴定训练体系<ref>何强、彭天国、董文，空军航空医学鉴定训练中心的发展与思考 ，全国航空航天医学学术会议暨云南省航空航天医学学术会议2010,27(3):180-181</ref><ref>{{Cite web |url = http://news.xinhuanet.com/mil/2011-12/25/c_111295752.htm |title = 一场从陆地打响的空战 |author =  |date = 2011-12-25 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。其中，2009年8月31日，'''中国人民解放军空军大连航空医学鉴定训练中心'''组建大会举行，[[中国人民解放军沈阳军区空军|中国人民解放军沈阳军区空军]]副司令员[[王国明|王国明]]少将，中共大连市委副书记[[里景瑞|里景瑞]]出席。大连航空医学鉴定训练中心隶属沈阳军区空军后勤部<ref name=dltjw/>。大连航空医学鉴定训练中心还经常为周边农村村民免费体检<ref>{{Cite web |url = http://www.ln.xinhuanet.com/gy/jigou/20160721/3296995_c.html |title =  大连：免费健康体检送到村民家门口|author =  |date =  2016-07-21|publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，转隶[[中国人民解放军北部战区空军|中国人民解放军北部战区空军]]后勤部。
