'''中央军委审计署驻中部战区审计局'''，位于[[北京市|北京市]]，是[[中央军委审计署|中央军委审计署]]下属局，负责[[中国人民解放军中部战区|中国人民解放军中部战区]]审计工作。

==沿革==
在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月组建[[中央军委审计署|中央军委审计署]]。中央军委审计署新设'''中央军委审计署驻中部战区审计局'''，驻[[中国人民解放军中部战区|中国人民解放军中部战区]]<ref>{{Cite web |url =http://jz.chinamil.com.cn/n2014/tp/content_8077198.htm  |title =  军委审计署驻中部战区审计局某审计组确保主题教育不断线|author =  |date = 2018-07-02 |publisher =军报记者  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.sohu.com/a/201325740_612650  |title = 军委审计署赴武警某部审计组：做为党执审、对党忠诚的审计尖兵 |author =  |date = 2017-10-30 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
