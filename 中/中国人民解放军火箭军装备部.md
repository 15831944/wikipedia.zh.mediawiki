'''中国人民解放军火箭军装备部'''，位于[[北京市|北京市]]，是[[中国人民解放军火箭军|中国人民解放军火箭军]]机关职能部门，负责中国人民解放军火箭军装备工作。

==沿革==
1966年7月，[[中国人民解放军第二炮兵部队|中国人民解放军第二炮兵部队]]正式成立。机关设中国人民解放军第二炮兵司令部、中国人民解放军第二炮兵政治部、中国人民解放军第二炮兵后勤部。1975年12月增设'''中国人民解放军第二炮兵科技部'''<ref name=zhengming/>。1983年6月，中央军委作出了在海军、空军和第二炮兵成立技术装备部的决定<ref>杨桓同志生平，2014年</ref>。第二炮兵科技部改称'''中国人民解放军第二炮兵技术装备部'''<ref name=zhengming/>。1992年左右，改称'''中国人民解放军第二炮兵装备技术部'''<ref>姜素然，六项遥测专业国家军用标准审查会在成都召开，遥测遥控1991(6):66-66</ref><ref>牛士俊、林武，表面工程技术在导弹武器装备维修中的应用，中国表面工程1995(2):12-13</ref>。1998年改设'''中国人民解放军第二炮兵装备部'''<ref name=zhengming>{{Cite web |url =http://www.81.cn/jmywyl/2016-02/29/content_6933950.htm?1456760091721  |title = 火箭军，为第二炮兵正名 |author =  |date = 2016-02-29 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://news.163.com/08/1223/10/4TRGPBQJ000120GU.html |title = 二炮装备部:十年打磨 "倚天长剑"终显锋芒 |author =  |date =  2008-12-23|publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。

2015年12月，在[[深化国防和军队改革|深化国防和军队改革]]中，成立[[中国人民解放军火箭军|中国人民解放军火箭军]]<ref>{{Cite web |url = http://news.xinhuanet.com/politics/2016-01/01/c_1117646667.htm  |title =陆军领导机构火箭军战略支援部队成立大会在京举行 习近平向中国人民解放军陆军火箭军战略支援部队授予军旗并致训词  |author =  |date =2016-01-01  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。中国人民解放军火箭军设'''中国人民解放军火箭军装备部'''。
