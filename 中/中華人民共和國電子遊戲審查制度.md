'''中華人民共和國電子遊戲審查制度'''由[[中华人民共和国|中华人民共和国]][[文化部|文化部]]所辖“内容审查委员会”对国内市场和国外[[进口|进口]]的网络游戏产品的内容进行审查，从2004年9月1日开始正式实行。规定所有在[[大陆|大陆]]在线传播或移动传播的网络游戏产品，必须报文化部进行内容审查后，方可正式投入运营。<ref name="xinhua1488275">{{cite news|title=我国将建立网络游戏内容审查制度|url=http://news.xinhuanet.com/fortune/2004-05/25/content_1488275.htm|newspaper=新华网|date=2004-05-25}}</ref>
== 网络游戏审查 ==
2004年5月24日中华人民共和国[[网络游戏|网络游戏]]产品内容[[审查制度|审查制度]]由文化部正式公布，将禁止进口非独家授权使用的网络游戏。中華人民共和國文化部第49號令中對網路遊戲中不能含有的內容作出了10條規定载有10个方面内容的网络游戏不得提供，分别为：“反对宪法确定的基本原则的；危害国家统一、主权和领土完整的；泄露国家秘密、危害国家安全或者损害国家荣誉和利益的；煽动民族仇恨、民族歧视，破坏民族团结，或者侵害民族风俗、习惯的；宣扬邪教、迷信的；散布谣言，扰乱社会秩序，破坏社会稳定的；宣扬淫秽、赌博、暴力或者教唆犯罪的；侮辱或者诽谤他人，侵害他人合法权益的；危害社会公德或者民族优秀文化传统的；有法律、行政法规和国家规定禁止的其他内容的”。并专门成立进口游戏产品内容审查委员会，由文化部所辖，专门负责对中华人民共和国境内的网络游戏产品的内容审查。<ref name="xinhua1488275"/>

== 实体游戏出版物 ==
2014年8月，[[中华人民共和国国务院|中华人民共和国国务院]]取消[[新闻|新闻]][[出版|出版]][[广电总局|广电总局]]对“电子出版物出版单位与境外机构合作出版电子出版物审批”的审批权限。<ref>{{cite web|title=国务院关于取消和调整一批行政审批项目等事项的决定|url=http://www.gov.cn/zhengce/content/2014-08/12/content_8974.htm|accessdate=2014-08-21}}</ref>此前，境外游戏进入中国市场均由文化部与新闻出版广电总局联合审批，此行政命令推出后境外游戏实际上只需文化部审批。不过根据之前发布的《中国(上海)自由贸易试验区文化市场开放项目实施细则》，自贸区内的外资企业生产制作的电子游戏内容（包括境外游戏）审查已被下放到上海市文广影视局。<ref name=N1>{{cite web|title=广电审批境外游戏职能被取消 暂仅由文化部负责|url=http://tech.qq.com/a/20140813/056904.htm|accessdate=2014-08-21}}</ref><ref>{{cite web|title=索尼＆微软报腿上海市委 游戏审核将不再困难|url=http://www.ali213.net/news/html/2014-4/102802.html|accessdate=2014-08-21}}</ref>

== 游戏分级 == 
有媒体称，未来中華人民共和國將文化部将计划对涉及暴力、色情、粗口、恐怖等内容的游戏实行「按遊戲暴力級別進行分級」的制度。<ref>{{cite web|title=暴力色情禁不禁？中国游戏分级制度路在何方|url=http://games.sina.com.cn/t/n/2014-02-27/0952766545.shtml|publisher=新浪游戏网|accessdate=2014年2月27日}}</ref>但至今此問題仍處於擱置狀態。
