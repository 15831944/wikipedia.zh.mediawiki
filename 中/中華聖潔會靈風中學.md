'''中華聖潔會靈風中學'''（{{lang-en|China Holiness Church Living Spirit College}}，簡稱：{{lang|en|LSC}}），位於[[香港|香港]][[新界|新界]][[大埔_(香港)|大埔]][[棟樑里|棟樑里]]1號，是一所[[基督教|基督教]]教會男女子中文文法中學，由辦學團體[[中華聖潔會_(香港)|中華聖潔會]]於1998年創立，現為[[津貼學校|津貼學校]]。該校鄰近[[那打素醫院|那打素醫院]]、[[富善邨|富善邨]]及[[富亨邨|富亨邨]]，佔地面積6800平方米。

== 校訓 ==
中華聖潔會靈風中學的校訓「育德育才，作鹽作光」是指該校致力培育學生的品德，發展他們的多元才能。從而讓他們成長成熟，抓緊[[聖經|聖經]]的教導。在世上發揮鹽的調和、防腐的功用，並在家庭、社會、國家及世界作美好的見證及貢獻。

== 辦學宗旨 == 
中華聖潔會靈風中學以[[聖經|聖經]]真理和基督仁愛肯定學生的尊嚴和價值，並誘發其潛能，建立自信、優良的品德、正確的價值觀及積極的人生態度。又激發他們求知、自學、創作、思考及情意發展。

== 學校資料 ==
2005-2006年，兩名中華聖潔會靈風中學教師陳夢熊及凌美儀在「數學教育學習領域」上獲得[[行政長官卓越教學獎|行政長官卓越教學獎]]嘉許狀。2006年，該校借出場地舉辦了第二場「名人訪校」地區性講座，予嘉賓[[香港話劇團|香港話劇團]]資深演員[[雷思蘭|雷思蘭]]及資深演員[[羅冠蘭|羅冠蘭]]進行演講。2009年，為鼓勵學生有較佳的學術發展，該校推行「靈風人獎勵計劃」。

=== 班級結構 ===
;中一至中三各設六班
*（以中文授課）中國語文及文學、數學、科學、物理、化學、生物、電腦、歷史、(中一除外)地理、通識教育、宗教、視覺藝術、音樂、體育、普通話、德育、查經、基本商業
*（以英文授課）英國語文及文學 
;中四、中五各設文科兩班、理科兩班、商科一班（若沒有指明，均用中文授課）
*必修科目：中國語文、英國語文（以中文授課）、數學（E班以中文授課）、聖經、音樂、體育
*文科科目：4A班：中國歷史、世界歷史  
*文科科目：4B班：中國歷史、地理
*商科科目：4C班：經濟、商業、會計（均用中文授課）
*理科科目：4D班：物理、化學、生物
*理科科目：4E班：附加數學、物理、化學、生物（均用中文授課）
*選修科目（一）：第六科：中國文學、旅遊、電腦、地理
*選修科目（二）：第七科：酒店管理、電影、電腦、會計（均用中文授課）、視覺藝術

;選修科目限制：
*E班不能選擇任何選修科目
*D班不能選擇第六科選修科目
*C班不能選擇第七科選修科目
*B班第六科選修科目不能選擇地理，因B班本是文科地理班
*第六科選修科目及第七科選修科目不能同時選擇電腦科

;選修科目注意：
*若人數眾多，不能選修所的選修科目的學生，學校會自行分配不獲接納的同學選修其他的選修科目
*選修科目是由中三升班時選擇
*學生必須在中四及中五主修被分派的選修科目，不能途中放棄及選擇其他選修科目
;中六、中七各設文科一班、理科一班
*（以中文授課）中國文語文及文化、中國文學、中國歷史、通識教育、宗教、體育、視覺藝術 
*（以英文授課）英文、經濟、地理、物理、生物、化學、純數、數學與統計、會計、電腦應用
