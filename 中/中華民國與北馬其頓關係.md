'''中華民國與北馬其頓關係'''是指[[中華民國|中華民國]]與[[北馬其頓共和國|北馬其頓共和國]]（舊稱'''馬其頓共和國'''、'''前南斯拉夫馬其頓共和國'''）之間的關係。兩國於1999－2001年有官方[[外交#外交上的承認|外交關係]]，是[[臺灣戰後時期|中華民國政府遷台]]後，唯二建立外交關係的[[歐洲國家和地區列表|歐洲新獨立國家]]（另一為[[拉脫維亞|拉脫維亞]]）。斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對北馬其頓的相關事務由'''駐[[義大利|義大利]][[台北经济文化代表处|台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-157-c2416-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，[[北馬其頓|北馬其頓]]當時屬於[[南斯拉夫社會主義聯邦共和國|南斯拉夫]]的[[馬其頓社會主義共和國|馬其頓社會主義共和國]]，由南斯拉夫作為[[主權國家|主權國家]]參與投票。]]

{{see also|中華民國與馬其頓共和國建交案|中華民國駐馬其頓共和國大使列表{{!}}中華民國駐馬其頓共和國公使|馬其頓共和國駐中華民國大使列表{{!}}馬其頓共和國駐中華民國公使}}
1997年起，中華民國派遣人員到馬其頓，與[[馬其頓內部革命組織民族統一民主黨|馬其頓內部革命組織民族統一民主黨]]、{{link-en|民主選擇黨 (馬其頓)|Democratic Alternative (Republic of Macedonia)|民主選擇黨}}等[[反对派|反對派]]政黨進行往來，這些反對黨在{{link-en|1998年馬其頓國會選舉|1998 Macedonian parliamentary election|1998年下旬舉行的國会選舉}}中表現不俗<ref name=hands>{{cite news |title=中国马其顿再度握手 |url=http://www.people.com.cn/GB/paper68/3637/450174.html |accessdate=2015-09-06 |agency=[[环球时报|环球时报]] |date=2001-06-22 |archiveurl=https://archive.is/20150906125825/http://www.people.com.cn/GB/paper68/3637/450174.html |archivedate=2015-09-06 |deadurl=no }}</ref>；在國會選舉後，[[馬其頓共和國總統|總統]]和[[馬其頓共和國總理|總理]]分別屬於不同的政黨，兩人是政敵關係，但總統才是真正的掌權者<ref>{{cite news |title=扁“外交”失敗拉蕭萬長下水　蕭立即反駁 |url=http://www.chinareviewnews.com/doc/1004/3/6/6/100436682.html?coluid=7&kindid=0&docid=100436682 |accessdate=2015-09-06 |agency=[[中國評論通訊社|中國評論新聞]] |date=2007-08-26 |archiveurl=https://archive.is/20150906130455/http://www.chinareviewnews.com/doc/1004/3/6/6/100436682.html?coluid=7&kindid=0&docid=100436682 |archivedate=2015-09-06 |deadurl=no }}</ref>。據說，中華民國承諾向馬其頓提供3億[[美元|美元]]的現金，並在當地建設一座[[工業園區|工業園區]]<ref name=Broken>{{cite news |author1=任逸 |title=馬其頓與台灣斷交內幕 |accessdate=2015-09-06 |agency=[[人民網|人民網]] |date=2001-06-15 |url=http://www.people.com.cn/BIG5/guoji/22/84/20010615/489710.html |archiveurl=https://archive.is/20150906144919/http://www.people.com.cn/BIG5/guoji/22/84/20010615/489710.html |archivedate=2015-09-06 |deadurl=no }}</ref>。

1999年1月27日，馬其頓總理[[柳普喬·格奧爾基耶夫斯基|格奧爾基耶夫斯基]]立場友台，故在其任內與中华民国建立大使级[[外交#外交上的承認|外交關係]]。2月，於首都[[史高比耶|史高比耶]]設立'''中華民國駐馬其頓共和國[[外交代表機構#大使馆|大使館]]'''；6月，於首都[[台北市|台北]]設立'''馬其頓共和國駐中華民國大使館'''。<ref name="89年外交年鑑">{{Cite web |url= http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_8.htm |title=《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher=中華民國外交部}}</ref>然而此舉卻遭到總統[[基羅·格利戈羅夫|格利戈羅夫]]的反對，拒絕接受中華民國[[特命全權大使|大使]]的[[國書|國書]]呈遞，故邦交期間由[[特命全權公使|公使]]使事。也因為[[中華人民共和國|中華人民共和國]]不滿馬其頓與中華民國建交，於1999年2月25日在[[聯合國安全理事會|聯合國安全理事會]]中對[[联合国维持和平行动列表|聯合國派駐馬其頓維和部隊]]的延期案投下[[聯合國安全理事會否決權|否決票]]。<ref>{{Cite web |url= http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1998/section_3/page3-1-4.htm |title=《中華民國87年外交年鑑》〈第三章 中外關係〉 |publisher=中華民國外交部}}</ref>

2000年2月27日，{{link-en|馬其頓外交部|Ministry of Foreign Affairs (Macedonia)|馬其頓外交部長}}狄米托夫抵台訪問4天。期間會見[[中華民國總統|總統]][[李登輝|李登輝]]，拜會[[立法院長|立法院長]][[王金平|王金平]]、[[中華民國外交部#歷任首長|外交部長]][[程建人|程建人]]，並參觀[[台北世界貿易中心|台北世界貿易中心]]。 <ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=A99D6F4BB0651F03 |author = 公眾外交協調會 |date = 2000年2月26日 |title = 馬其頓共和國外交部長狄米托夫閣下訂本（二）月廿七日率團乙行十人抵華訪問四天。 |publisher = 中華民國外交部}}</ref>

2001年5月，{{link-en|北馬其頓政府|Politics of North Macedonia|馬其頓政府}}高層發生人事變動，新上任的馬其頓副總理、外交部長和{{link-en|北馬其頓國防部長|Minister of Defense (North Macedonia)|國防部長}}均不希望繼續與中華民國保持外交關係<ref name=Broken/>。為了維護與馬其頓的邦交，中華民國外交部長[[田弘茂|田弘茂]]在同年6月出訪當地，但馬其頓外交部長{{link-en|伊林卡·米特雷瓦|Ilinka Mitreva|米特雷瓦}}拒絕接見他，而台灣[[輿論|輿論]]也普遍不認同以金錢外交挽回馬其頓；米特雷瓦亦曾在[[記者會|記者會]]上表示，馬其頓外交部已計劃與中華民國斷絕外交關係，並與中華人民共和國關係正常化<ref>{{cite news |author1=楊孟瑜 |title=台灣稱對馬其頓不委曲求全 |url=http://news.bbc.co.uk/chinese/trad/hi/newsid_1360000/newsid_1364600/1364693.stm |accessdate=2015-09-07 |agency=[[BBC#BBC中文部|BBC中文網]] |date=2001-06-01 |archiveurl=https://archive.is/20150907055712/http://news.bbc.co.uk/chinese/trad/hi/newsid_1360000/newsid_1364600/1364693.stm |archivedate=2015-09-07 |deadurl=no }}</ref>。[[中華民國外交部|中華民國外交部]]對此發言深表遺憾，駐馬其頓[[外交代表機構#大使馆|大使館]]立即發表聲明，指出無法接受兩國間政治關係的改變，亦不接受任何形式的替代方案<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=5ADBB19F243C6E99 |author = 公眾外交協調會 |date = 2001年5月27日 |title = 中華民國與馬其頓邦交 |publisher = 中華民國外交部}}</ref>。6月12日，馬其頓政府發言人在記者會上表示，該國政府已決定承認中華人民共和國，並恢復馬其頓與中華人民共和國之間的正常外交關係<ref name=Broken />；分析認為，馬其頓政府此舉是為了融入全體成員國皆承認中華人民共和國的[[歐盟|歐盟]]，亦是因為該國與中華民國的經貿合作成效不彰。<ref name=hands/>6月18日，兩國斷交，中華民國關閉駐馬其頓大使館，停止一切合作計畫並撤回專家技術團。<ref>{{Cite web |url= http://www.mofa.gov.tw/News_Content_M_2.aspx?n=5028B03CED127255&sms=5ED24855AD8E6C58&s=F9035687CF8F9658 |author=公眾外交協調會 |date=2001年6月18日 |title=中華民國政府對於馬其頓政府罔顧兩國友誼，屈從中共威脅利誘，深表遺憾，爰決定自即日起中止中、馬外交關係，關閉駐馬其頓共和國大使館，停止兩國一切合作計畫並撤回專家技術團 |publisher=中華民國外交部}}</ref>

2012年4月3日，馬其頓成為第127個給予[[中華民國國民|中華民國公民]]可以[[中华民国公民签证要求|免簽證]]入境<ref>{{cite news|title=馬其頓對台免簽 先給1年|url=http://newtalk.tw/news/view/2012-04-03/23847|language=zh-tw|publisher=[[中央通訊社|中央通訊社]]|date=2012-04-03|accessdate=2015-04-10}}</ref>。

===貿易===
2015年10月13日，[[中華民國外交部|外交部]]委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）與8家台灣業者首度參加「2015年馬其頓[[冶金學|冶鐵]]、[[電子學|電子]]及[[建築學|營建]]展（TEHNOMA 2015）」。該展為馬其頓規模最大的技術展，[[台灣|台灣]]館以國家館形象展出。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=675558&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date =  2015年10月15日 |title = 搶攻東南歐商機 貿協首度組團前進馬其頓拓銷 |publisher = 經貿透視網}}</ref>

2017年，馬其頓是中華民國第135大貿易夥伴、第114大[[進口|進口]]夥伴、第154大[[出口贸易|出口]]夥伴。出口至馬其頓的金額為294萬0,374[[美元|美元]]，年減32.886%；自馬其頓進口的金額為1,123萬6,725美元，年增408.214%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）829萬6,351美元，年減482.301%。※2016年，馬其頓是中華民國第160大貿易夥伴、第142大進口夥伴、第146大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===計畫===
建交期間，為協助馬其頓經濟重建及發展，中華民國協助設立史高比耶[[加工出口區|加工出口區]]，以提升對馬其頓的投資、增加就業機會。由[[中華開發工業銀行|中華開發工業銀行]]與財團法人國際合作發展基金會（國合會）合資設立的史高比耶開發管理公司（資本額100萬美元，中華開發投資55%，國合會投資45%）負責加工出口區的開發工作，國合會另提供1,155萬美元低利[[貸款|貸款]]作為開發經費。中華民國另提供1,050萬美元贈款，協助馬其頓進行包括[[馬其頓共和國交通|交通]]、{{link-en|馬其頓大眾傳播|Telecommunications in the Republic of Macedonia|大眾傳播}}、水利、{{link-en|馬其頓教育|Education in the Republic of Macedonia|教育}}、社會福利及醫療衛生等與民生相關的十項基礎建設計畫。另設立「中華民國—馬其頓共和國經濟發展基金」，由兩國派員組成管理委員會以協調兩國經濟發展合作計畫。<ref>{{Cite web |url= http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter1-4.htm |title=《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher=中華民國外交部}}</ref>
