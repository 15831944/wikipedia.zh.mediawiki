'''中國國家男子足球队'''，简称'''中国男足'''、'''国足'''，[[国际足联|国际足联]]以'''中華人民共和國'''称呼。最早代表中国的[[国家足球队|国家足球队]]可追溯至于1924年创建的[[中華足球聯合會|中華足球聯合會]]，于1931年加入[[国际足联|国际足联]]。1949年，在[[第二次国共内战|第二次国共内战]]中[[中国国民党|国民党]]战败，[[中华民国国家足球队|中华民国国家足球队]]跟随[[国府迁台|国府迁台]]。中华人民共和国建立后，政府在1950年代初组建国家足球队，以中国国家足球队的名义代表中国大陆参赛，直到1970年代才獲廣泛接納參加國際賽。现在中华人民共和国管辖下的[[香港|香港]]和[[澳门|澳门]]地区各自拥有國際足球代表队，并且均为[[国际足联|国际足联]]及[[亚洲足联|亚洲足联]]的成员，分别以[[香港足球代表队|香港足球代表队]]和[[澳门足球代表队|澳门足球代表队]]的名义参加国际赛事。

从1976年起，中国队参加[[亚洲杯足球赛|亚洲杯足球赛]]，于[[1984年亚洲杯足球赛|1984年]]和[[2004年亚洲杯足球赛|2004年]]两度晉身决赛。2002年，中國队首次晉身[[世界杯足球赛|世界杯决赛圈]]，也是迄今为止唯一的一次晉級。

== 历史 ==

[[File:China_national_football_team_06-JUN-2008-ANZstad.jpg|thumb]][[澳大利亞體育場|澳大利亞體育場]]舉辦的[[世界盃|世界盃]]會外賽前的攝影。]]

在[[第二次世界大戰|第二次世界大戰]]之前，[[中華民國國家足球隊|中華民國國家足球隊]]是远东地區實力最強的球队，曾經九次贏得[[遠東運動會|遠東運動會]]足球比賽冠軍，以及參加1936年[[柏林奧運|柏林奧運]]足球賽。1949年國民政府遷台後，[[中華民國國家足球隊|中華民國國家足球隊]]初期仍然代表中國參加國際賽，於1954年及1958年兩次贏得[[亞運|亞運]]足球金牌，直至1971年聯合國代表權轉變為止。

从1970年代末中华人民共和国國家足球隊开始代表中国參加各项國際赛事，至1980年代初也是[[亞洲|亞洲]]的一支強隊，[[1982年世界杯|1982年世界杯]]亚太区预选赛中，中国队几乎晋级決賽周，但本來落後於中國的[[新西蘭国家足球队|新西蘭]]在最後一場外圍賽，作客淨勝[[沙特阿拉伯國家足球队|沙特阿拉伯]]五球，令新西蘭與中國分數及得失球差相同，而須進行多一場附加賽，最终中國在附加赛中戰败，未能晋级决赛圈。

兩年後的[[亞洲杯|亞洲杯]]中，中国队首度打入決賽，最後敗給沙特阿拉伯得亞軍。不過，心理素质差成為1980-1990年代中國隊的通病，以至在比賽中往往被實力次一級的對手打敗。例如[[1986年世界杯|1986年世界杯]]外圍賽主場被[[香港足球代表队|香港足球代表队]][[五一九事件|打敗]]，[[1990年世界杯|1990年世界杯]]外圍賽其中兩場對[[阿聯酋国家足球队|阿聯酋]]及[[卡塔爾国家足球队|卡塔爾]]的賽事，皆在領先對手一球的情況下完場前三分鐘連失兩球反勝為敗，[[1994年世界杯|1994年世界杯]]外圍賽更在首圈意外敗給[[弱隊|弱隊]][[也門国家足球队|也門]]提早出局，令中國隊遲遲未能衝出亞洲，參與更高級別的國際賽事。

1990年代開始，隨著[[日本国家足球队|日本]]足球的崛起以及[[韓國国家足球队|南韩]]等其他亞洲國家足球水準也不斷提高，中國足球水準更顯得停滯不前，[[2002年世界杯|2002年世界杯]]，中國隊取得出線資格首次晉身決賽周，但中國隊在決賽周表現也未如人意，三場小組賽皆墨，且一球未进。<ref name="上海热线">{{Cite web |title = 国足冲击韩日世界杯：首度出线铸造唯一辉煌 |url =http://sports.qq.com/a/20080615/000975.htm |publisher = 腾讯体育 | date = 2008-06-15 | accessdate = 2014-06-21 }}</ref>在[[2004年亞洲杯|2004年亞洲杯]]，中國殺入決賽，結果以1-3敗給[[日本国家足球队|日本]]，第二次取得亞洲盃亞軍。

[[2004年亞洲杯|2004年亞洲杯]]之後，中國足球水平退步十分严重。在[[2006年世界杯|2006年]]、[[2010年世界杯|2010年]]及[[2014年世界盃足球賽|2014年]]連續三屆世界杯外圍賽都在首輪分組賽出局；[[亞洲杯|亞洲杯]]方面，亦在[[2007年亞洲盃足球賽|2007年]]及[[2011年亞洲盃足球賽|2011年]]连续两屇決賽周都在首輪分組賽出局。

[[2015年亞洲盃外圍賽|2015年亞洲盃外圍賽]]，中國僅以小組最佳第三名出線決賽周，賽前一致被看淡。豈料中國卻在[[2015年亞洲杯|分組初賽]]中，分別撃敗[[沙地阿拉伯国家足球队|沙地阿拉伯]]、[[烏茲別克国家足球队|烏茲別克]]及[[朝鮮国家足球队|朝鮮]]，歷史上首次在亞洲盃分組初賽取得三戰全勝成績。可惜中國隊晉級八強後就面對東道主[[澳洲国家足球队|澳洲]]，結果以0-2不敵澳洲出局。

[[2018年世界盃資格賽亞洲區第二輪|2018年世界盃外圍賽亞洲區第二圈]]，中國作客以0-1敗給[[卡塔爾国家足球队|卡塔爾]]及主客皆與[[香港足球代表队|香港]]打和0-0，一度陷於出局边缘。幸在最後一場賽事主場以2-0擊敗卡塔爾，力保小組次名，加上其他組別的次名球隊失分，中國隊得以跻身最佳四隊小組次名，繼[[2002年世界盃外圍賽|2002年世界盃外圍賽]]之後，再次晉級世界杯亞洲區外圍賽第三轮。中國晉級後與[[韓國国家足球队|南韩]]、[[伊朗国家足球队|伊朗]]、[[烏茲別克国家足球队|烏茲別克]]、[[敘利亞国家足球队|敘利亞]]及卡塔爾同組，但外圍賽首循環只取得兩和三負的成績，位列榜尾，出缐形勢惡劣。雖然中國在外圍賽次循環表現有所改善，分別擊敗南韓、烏茲別克及卡塔爾，但因之前失分太多，只取得小組第五名被淘汰出局。

[[2019年亞洲盃|2019年亞洲盃]]，中國隊由曾帶領[[意大利国家足球队|意大利]]奪取[[2006年世界杯|2006年世界杯]]冠軍的名帥[[納比|納比]]領軍，但中國隊表現一般，分組初賽擊敗實力較弱的[[吉爾吉斯国家足球队|吉爾吉斯]]及[[菲律賓国家足球队|菲律賓]]，但最後一仗分組賽不敵[[韓國国家足球队|南韩]]，以小組次名晉級十六強。中國在十六強賽以2-1險勝[[泰國国家足球队|泰國]]；晉級半準決賽對賽[[伊朗国家足球队|伊朗]]，後防線連續三次出現致命失誤，讓伊朗人抓住機會，最後以0-3大敗出局。中國國家隊主教練納比在賽後新聞發佈會宣佈與中國足協合同到期，不再續約。

== 现状 ==
2011年中国大陆的青少年足球人口相比1995年减少了将近99%<ref>注：根据官方统计，1995年中国的青少年足球人口为65万，2011年初，此数字为7000，http://sports.enorth.com.cn/system/2011/07/05/006867048.shtml http://sports.sina.com.cn/n/2010-02-08/17094831820.shtml</ref>，全国面临无人踢球的窘境，各年龄段国家队、俱乐部队选拔青训球员极度困难，如中国顶级联赛球会[[北京国安|北京国安]]曾举行选秀活动，本打算招收一个队的球员，最终竟然只招入2人<ref>[http://sports.sina.com.cn/j/2011-10-25/04045801480.shtml 国安寻苗子仅招到两人 杨璞：北京踢球的孩子太少了]</ref>。而众多足球学校大多摇摇欲坠濒临倒闭。据2011年的统计，中国境内的注册球员数量已不足[[越南国家足球队|越南]]注册球员之20%、[[日本国家足球队|日本]]注册球员之1%、以及[[法国国家足球队|法国]]之0.1%<ref>[http://sports.sina.com.cn/c/2011-10-24/12065800528.shtml 中国足球注册球员不足越南2成 省级几无11人制联赛]</ref>。

人才的匮乏亦导致其他国字号战绩不佳，其中[[中国国家青年足球队|中国国青队]]自[[2005年U20世界杯足球赛|2005年世青赛]]打进16强后已经连续缺席5届[[U20世界杯足球赛|世青赛]]，[[中国国家少年足球队|中国国少队]]自[[2005年U17世界足球足球锦标赛|2005年世少赛]]打进16强后也同样连续缺席5届[[U17世界杯足球赛|世少赛]]。

2015年2月27日，[[中央全面深化改革领导小组|中央全面深化改革领导小组]]第十次会议审议通过《中国足球改革发展总体方案》<ref>[http://sports.people.com.cn/n/2015/0227/c22134-26608118.html 中央全面深改小组通过《中国足球改革总体方案》]</ref>。3月16日，[[中华人民共和国国务院办公厅|国务院办公厅]]正式对外发布《中国足球改革发展总体方案》，《方案》涉及足协改革、联赛改革、国足建设、青训体系、申办世界杯、中国联赛足彩等内容，从国家层面明确了足球的战略意义，此举备受外界瞩目<ref>[http://www.gov.cn/zhengce/content/2015-03/16/content_9537.htm 国务院办公厅关于印发中国足球改革发展总体方案的通知]</ref>。

== 影响 ==
中国足球联赛是中国唯一拥有巨大利润支持的大型运动联赛（另两个有微薄利润的运动联赛是中国围棋联赛和中国象棋联赛），在中国民间足球运动的地位远高于有国球之称的[[乒乓球|乒乓球]]和[[羽毛球|羽毛球]]。中国球迷们往往把国足的胜负看成是国家的荣辱，以至對于国家队長期表现不尽如人意称为“国耻”<ref>[http://sports.daqi.com/ztnew_index/245651/7/index.html 惨败，又一个中国足球国耻日] {{webarchive|url=https://web.archive.org/web/20070827062500/http://sports.daqi.com/ztnew_index/245651/7/index.html |date=2007-08-27 }}</ref><ref>[http://sports.sohu.com/20070718/n251126925.shtml “打平就出线”由来以久，“5·19”成足球界国耻日]</ref><ref>[http://ent.sina.com.cn/h/2006-10-08/11271274796.html 赵本山提起足球就发火：再谈国足就是国耻]</ref>。[[2002年世界杯足球赛|2002年世界杯]]期间，通过电视观看国家队比赛的球迷总数达到3亿；而在[[2004年亚洲杯|2004年亚洲杯]]国足同[[日本国家足球队|日本队]]的决赛，有超过2.5亿的观众收看了这场赛事，是国际大型体育赛事单场比赛在中国的收视记录。不过随着国足战绩日趋糟糕以及负面新闻导致的不良社会影响，国足比赛开始遭到中国球迷抵制。<ref>[http://news.dayoo.com/sports/201010/12/53870_14144726.htm 武汉球迷组织抵制国足 苏亚雷斯吃哈根达斯] {{webarchive|url=https://web.archive.org/web/20140725152929/http://news.dayoo.com/sports/201010/12/53870_14144726.htm |date=2014-07-25 }}</ref>
	
中国足球队有时也会成为中国大陆流行文化的元素。糟糕的战绩加上大量的负面新闻，使得中国男足经常成为国内民众嘲讽的对象。曾有中国网民根据[[2008年北京奥运会|北京奥运会]]歌曲《[[北京欢迎你|北京欢迎你]]》改编成搞笑歌曲《[[国足欢迎你|国足欢迎你]]》，以嘲讽中国男足在北京奥运会上的糟糕表现。<ref>{{cite web|url=http://www.youtube.com/watch?v=BmKyZPGV76E|title=《国足欢迎你》男女声版-改自《北京欢迎你》‏|publisher=YouTube}}</ref>更有甚者曾在[[小品|小品]]中将部分队员直接点名道姓指出来进行调侃。<ref>{{cite web|url=http://video.sina.com.cn/v/b/18783096-1058127434.html|title=小品《球迷》99央视春晚‏|publisher=新浪视频}}</ref>

== 世界排名 ==

在1998年12月公佈的[[国际足联世界排名|国际足联世界排名]]中，中國國家男子足球隊名列第 37 位，是至今排名最高的成績。而在[[2013年3月|2013年3月]]的世界排名中，中国男足排在 109 位，是目前為止最差的一次排名。<small><ref name=FIFA_ranking>[http://www.fifa.com/associations/association=chn/ranking/gender=m/index.html FIFA_com - China PR FIFA-Coca-Cola World Ranking] {{en}}</ref></small>

* 積分：1320 分
* 世界排名：第 75 位 （共 211 個國家／地區）
* 亞洲排名：第 7 位 （共 46 個國家／地區）
