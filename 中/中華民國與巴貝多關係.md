'''中華民國與巴貝多關係'''是指[[中華民國|中華民國]]與[[巴貝多|巴貝多]]之間的關係。兩國於1967－1977年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對巴貝多的相關事務由'''中華民國駐[[聖露西亞|聖露西亞]][[中華民國駐外機構列表|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-229-31855-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]一案棄權，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[巴貝多|巴貝多]]。]]
{{see also|中華民國外交|{{link-en|巴貝多外交|Foreign relations of Barbados}}}}
1967年9月4日，巴貝多與中華民國建立大使級[[外交#外交上的承認|外交關係]]。於首都[[布里奇敦|橋鎮]]設立'''中華民國駐巴貝多[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1977年1月11日，兩國斷交。14日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境巴貝多。]]
[[File:Visa_requirements_for_Barbadian_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|巴貝多簽證政策|巴貝多公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]僅限申請觀光簽證，需寄至巴貝多移民局或{{link-en|巴貝多駐外機構列表|List of diplomatic missions of Barbados|駐北美、歐洲使領館}}，停留最多90天。<ref name="簽證及入境須知">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-229-31855-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|巴貝多護照|Barbados passport}}的{{link-en|巴貝多人口|Demographics of Barbados|巴貝多公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國衛生福利部|中華民國衛生福利部]]的國際間旅遊疫情建議等級表將巴貝多歸類為'''第二級警示'''：對當地採取加強防護（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=16533556D6300F4F |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|巴貝多經濟|Economy of Barbados}}}}
===貿易===
2018年，巴貝多是中華民國第157大貿易夥伴、第139大[[進口|進口]]夥伴、第154大[[出口贸易|出口]]夥伴。出口至巴貝多的金額為328萬3,649[[美元|美元]]，年減8.483%；自巴貝多進口的金額為254萬5,984美元，年增36.467%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）73萬7,665美元，年減57.171%。2017年，巴貝多是中華民國第161大貿易夥伴、第146大進口夥伴、第149大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
