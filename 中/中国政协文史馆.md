'''中国政协文史馆'''，位于[[北京市|北京市]][[西城区|西城区]][[太平桥大街|太平桥大街]]23号，是[[全国政协办公厅|全国政协办公厅]]直属[[事业单位|事业单位]]。

==简介==
2005年3月10日，在[[全国政协文史和学习委员会|全国政协文史和学习委员会]]主任会议上，主任[[王蒙|王蒙]]提议筹建全国政协文史资料馆，随后报送请示。12月28日，在第十届全国政协第三十次主席会议上，[[中共中央政治局常委|中共中央政治局常委]]、[[全国政协主席|全国政协主席]][[贾庆林|贾庆林]]同意开展专题研究。2006年2月27日全国政协文史和学习委员会办公室报送了《关于筹建全国政协文史资料馆的初步方案》，4月17日又报送了《关于筹建中国政协文史资料馆的启动方案》，2007年9月13日全国政协办公厅人事局报送了《关于申请设立中国政协文史馆的请示》，获得全国政协主席[[贾庆林|贾庆林]]、副主席[[王忠禹|王忠禹]]、秘书长[[郑万通|郑万通]]等的批示。2007年9月30日，[[全国政协机关党组|全国政协机关党组]]会议研究了该馆建设及机构设置等<ref name=dshj/>。

2008年3月3日、9月26日，[[中央机构编制委员会办公室|中央机构编制委员会办公室]]、[[国家发展和改革委员会|国家发展和改革委员会]]分别复函全国政协办公厅，同意设中国政协文史馆<ref>{{Cite web |url =http://money.163.com/10/0507/12/66347S7Q00253B0H.html  |title =  中国政协文史馆建设项目通过批准|author =  |date = 2010-05-07 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref><ref name=dshj/>。2010年4月16日，全国政协机关党组会议决定成立中国政协文史馆筹建办公室，[[张振山|张振山]]任主任，[[许水涛|许水涛]]任副主任<ref name=dshj/>。5月26日，贾庆林、[[王刚_(政治人物)|王刚]]等全国政协领导出席该馆奠基仪式<ref name=dshj/><ref>郭颜，中国政协文史馆奠基仪式在京举行，中国政协2010(6)</ref>。2011年1月13日，全国政协秘书长办公会决定成立中国政协文史馆筹建工作协调领导小组，组长为全国政协机关党组书记、全国政协副秘书长[[孙怀山|孙怀山]]，副组长为全国政协副秘书长[[卢昌华|卢昌华]]、全国政协文史和学习委员会驻会副主任[[卞晋平|卞晋平]]<ref name=dshj/>。

2012年3月3日，全国政协十一届五次会议开幕会在[[人民大会堂|人民大会堂]]举行，贾庆林主席作的政协全国委员会常务委员会工作报告称，“建成中国政协文史馆并对外开放，使之成为宣传[[中国共产党领导的多党合作和政治协商制度|中国共产党领导的多党合作和政治协商制度]]的重要窗口。”<ref>{{Cite web |url = https://news.qq.com/a/20120303/001148.htm |title = 贾庆林：将建成中国政协文史馆并对外开放 |author =  |date =  2012-03-03|publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref>

2012年5月14日，全国政协机关党组会议决定，[[赵珩|赵珩]]任该馆馆长，许水涛任副馆长。6月7日，[[北京市住房和城乡建设委员会|北京市住房和城乡建设委员会]]公示中国政协文史馆工程获结构长城杯金奖。6月30日该工程竣工验收。7月16日，全国政协副主席兼秘书长[[钱运录|钱运录]]和全国政协机关领导视察了该馆大楼。[[全国政协机关事务管理局|全国政协机关事务管理局]]局长[[刘小宁|刘小宁]]向赵珩馆长移交了该馆钥匙。8月23日，该馆“三定”（工作职责、机构设置、人员编制）方案印发<ref name=dshj/>。9月21日，贾庆林等领导出席中国政协文史馆揭幕仪式并讲话<ref name=dshj>{{Cite web|url =http://www.zgzxwsg.com/gdashiji/index.shtml|title =文史馆大事记|author =|date =2014-02-27|publisher =中国政协文史馆|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20140818123822/http://www.zgzxwsg.com/gdashiji/index.shtml|archivedate =2014年8月18日|df =}}</ref>。

2013年5月30日，中国政协文史馆[[言恭达|言恭达]]艺术研究院成立<ref>杨妤，中国政协文史馆言恭达艺术研究院成立，青少年书法:青年版2013(6):72-72</ref>。

2013年2月22日，中国政协文史馆委员传记中心成立仪式暨首批传主座谈会在北京举行，该中心由中国政协文史馆主办、中国善尔健康集团协办，第一批确定立传的全国政协委员包括戏曲界人物[[梅葆玖|梅葆玖]]、[[谭元寿|谭元寿]]、[[叶少兰|叶少兰]]、[[李世济|李世济]]、[[蔡正仁|蔡正仁]]等<ref>{{Cite web |url = http://www.chinaxwcb.com/2013-03/14/content_265161.htm |title = 中国政协文史馆委员传记中心成立 |author =  |date = 2013-03-14 |publisher =  中国新闻出版网|language =  |accessdate =  |quote =  }}</ref>。

全国政协主席[[俞正声|俞正声]]参观中国政协文史馆时指出：“应该做好文史资料电子化的工作，方便人们查阅和利用”。2014年4月，全国政协秘书长办公会议研究了中国政协文史馆报送的文史资料数据库工程建设方案，决定建设“中国政协文史资料数据库”<ref name=juli>{{Cite web |url = http://www.zgzx.com.cn/2016-06/24/content_8751951.htm |title =  聚力共谋政协文史馆发展良策|author =  |date =2016-06-24 |publisher =  中国政协传媒网|language =  |accessdate =  |quote =  }}</ref>。

中国政协文史馆成立后，各地方政协也纷纷成立文史馆。2016年5月18日至19日，中国政协文史馆在[[广西|广西]][[南宁|南宁]]召开“发挥政协文史馆在人民政协对外宣传展示暨文史资料工作中的作用”座谈会。这是该馆成立后首次召开全国性的会议，也是政协历史上首次召开涉及文史馆工作的全国性会议<ref name=juli/>。

中国政协文史馆的功能定位从起初的仅为文史资料服务，发展到“建设一个集征集、收藏、研究、展示、利用、交流功能于一体的国家级综合性文史馆”，成为“人民政协光辉历程和重要成就的展示窗口，政协理论、政协文化和文史资料的研究园地，人民政协与社会各界及海内外人士的交流平台。”<ref name=juli/>中国政协文史馆收藏着数亿字的文史资料手稿、图书及字画、照片、音像等<ref>{{Cite web|url = http://www.nsa.gov.cn/web/a/gw/jinxiuban/20140318/3468.html|title = 重温人民政协历史 切实增强“三个自信”——第13期青年干部培训班参观中国政协文史馆|author = |date = 2014-03-18|publisher = 国家行政学院|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180209182153/http://www.nsa.gov.cn/web/a/gw/jinxiuban/20140318/3468.html|archive-date = 2018-02-09|dead-url = yes}}</ref>。该馆向社会团体、高等院校、国内外重要组织开放<ref name=juli/>，并且经常举办艺术展览<ref name=ychysh/>。

==建筑==
中国政协文史馆位于[[大乘胡同|大乘胡同]]东西向段的南侧，[[武定侯街|武定侯街]]北侧，坐西朝东，东北是[[全国政协礼堂|全国政协礼堂]]。中国政协文史馆总建筑面积23000平方米，风格同毗邻的全国政协办公楼群、[[全国政协礼堂|全国政协礼堂]]呼应。一层铜门图案融合了政协会徽的元素。一层大厅中央悬挂巨幅[[丙烯|丙烯]]画《碧波荟萃》，象征着人民政协海纳百川；大厅两侧为[[汉白玉|汉白玉]]浮雕《日月光华》，北侧“[[金乌|金乌]]”代表太阳，南侧“[[玉兔|玉兔]]”代表月亮。浮雕底部是山形和水纹，中间是植物和云纹，整个大厅体现了日月同辉、天人合一的主题<ref name=ychysh>{{Cite web |url =http://exhibit.artron.net/institution-ORG25959.html  |title =中国政协文史馆  |author =  |date =  |publisher = 雅昌艺术网 |language =  |accessdate =2018-02-09  |quote =  }}</ref>。
