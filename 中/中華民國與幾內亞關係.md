'''中華民國與幾內亞關係'''是指[[中華民國|中華民國]]與[[幾內亞共和國|幾內亞共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對幾內亞的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-232-54f6e-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|幾內亞簽證政策|Visa policy of Guinea}}|{{link-en|幾內亞公民簽證要求|Visa requirements for Guinean citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可由[[旅行社|旅行社]]代辦簽證，或前往幾內亞駐[[南非|南非]][[外交代表機構#大使馆|大使館]]申請。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-232-54f6e-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|幾內亞護照|Guinean passport}}的{{link-en|幾內亞人口|Demographics of Ghana|幾內亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將幾內亞歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2016年10月27日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-278-b8fc4-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將幾內亞歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年11月6日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=5725B94ECD7E3A57 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|幾內亞經濟|Economy of Guinea}}}}
===貿易===
2017年，幾內亞是中華民國第137大貿易夥伴、第156大[[進口|進口]]夥伴、第115大[[出口贸易|出口]]夥伴。出口至幾內亞的金額為1,203萬7,778[[美元|美元]]，年增24.944%；自幾內亞進口的金額為91萬4,102美元，年增208.596%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1,112萬3,676美元，年增19.119%。※2016年，幾內亞是中華民國第144大貿易夥伴、第171大進口夥伴、第121大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
