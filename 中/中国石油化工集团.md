'''中国石油化工集团公司'''（[[英语|英语]]：'''Sinopec Group'''（'''SINOPEC'''）），简称中国石化集团，于1998年7月经国务院批准在原[[中国石油化工总公司|中国石油化工总公司]]的基础上重组成立的特大型[[石油化工|石油化工]]企业<ref>{{cite web |title = 国务院关于组建中国石油化工集团公司有关问题的批复  |author = [[中华人民共和国国务院|中华人民共和国国务院]] |publisher = [[中国政府网|中国政府网]] |url= http://www.gov.cn/xxgk/pub/govpublic/mrlm/201011/t20101120_62923.html |accessdate=2014-04-27 |language = zh-hans}}</ref>，简称'''中石化集团'''、'''中石化'''，是由[[国务院国有资产监督管理委员会|国务院国有资产监督管理委员会]]监督管理的[[中央企业|中央]][[国有独资企业|国有独资企业]]。注册资本2316亿元。中石化集团以独家发起方式于2000年2月设立的股份制企业[[中国石油化工股份有限公司|中国石油化工股份有限公司]]。

中石化集團在2011年[[财富|财富]][[世界500强|世界500强]]名單中排名第5位，营业收入为2734.22亿美元。
中石化集團在2014年[[财富|财富]][[世界500强|世界500强]]名單中排名第5位，营业收入为4572亿美元。<ref>{{cite web |accessdate = 2014-07-09 |url = http://intl.ce.cn/specials/zxgjzh/201407/08/t20140708_3114803.shtml |title = 今年《财富》世界500强发布 中国公司首进前三强|publisher=中国经济网}}</ref>

==收購==
2009年6月25日中石化集團以每股52.8[[加元|加元]]（357[[港元|港元]]），合共約82.7億加元（72.4億[[美元|美元]]，558億港元）的現金全面收購在加拿大和倫敦均有上市的瑞士Addax石油公司。

Addax資產則位於西非及中東，包括尼日利亞、喀麥隆、加蓬、伊拉克的庫爾德地區。

2010年10月1日，中石化集團以71億美元入股[[西班牙|西班牙]]石油公司Repsol YPF旗下的Repsol巴西40%股權。Repsol巴西主要開采離岸油氣資源，主要油氣資產分佈於[[巴西|巴西]]的Santos、Campos及Espiritu Santo盆地。<ref>[http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20101002&sec_id=15307&subsec=15320&art_id=14511056 中石化554億入股巴西油企] 蘋果日報，2010年10月2日</ref>

2012年9月3日將煉化工程業務註冊為獨立公司[[中石化煉化工程（集團）股份有限公司|中石化煉化工程（集團）股份有限公司]] [Sinopec Engineering (Group)，簡稱﹕煉化工程集團]，註冊資本為31億元[[人民幣|人民幣]]，有助集團日後更容易將煉化工程業務分拆上市。

2012年9月28日，中国石化長城能源化工公司在北京成立，負責中石化煤化工業務投資及經營，标志着中国石化专业化发展煤化工业务全面启动。

2012年12月28日，[[中石化石油工程技术服务有限公司|中石化石油工程技术服务有限公司]]（[[中石化石油工程技术服务股份有限公司|中石化石油工程技术服务股份有限公司]]前身）在北京成立。公司现有员工14万余人，固定资产766亿余元，2012年预计收入950亿元。

2014年9月12日，中國石化與25家境內外投資者簽訂了《增資協議》；據此，投資者會向中国石化销售有限公司認購共計人民幣1,070.94億元現金（含等值美元）中国石化销售有限公司29.99%的股權。

2014年3月17日，中石化易捷销售有限公司在北京正式揭牌成立，为中国石化销售有限公司的全资子公司。
