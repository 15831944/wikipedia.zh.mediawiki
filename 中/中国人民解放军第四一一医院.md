'''中国人民解放军第四一一医院'''，位于[[上海市|上海市]]，隶属[[中国人民解放军东部战区海军|中国人民解放军东部战区海军]]后勤部。

==简介==
前身是教会医院，[[日本|日本]]占领上海后成为[[日军|日军]]的海军医院。[[抗日战争|抗日战争]]胜利后成为[[南京国民政府|南京国民政府]]海军上海医院。1949年5月27日上海解放后，由[[华东军区海军|华东军区海军]]接管，组建为华东海军上海医院。1954年6月按全军统一序列编为中国人民解放军第四一一医院，隶属华东军区海军。后来划归[[中国人民解放军海军上海基地|海军上海基地]]管辖。2004年编制体制调整，该医院转隶[[南京军区|南京军区]]联勤第十三分部管辖<ref name=zgjw>{{Cite web |url =http://www.81.cn/zghjy/2011-07/14/content_5584637.htm  |title =  解放军第411医院|author =  |date = 2011-07-14 |publisher =中国军网 |language =  |accessdate =  |quote =  }}</ref>。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，转隶[[中国人民解放军东部战区海军|中国人民解放军东部战区海军]]后勤部<ref>{{Cite web |url = http://news.ifeng.com/a/20170304/50752908_0.shtml |title = 战舰上的女军医：手术时只能趁浪头间隙抓紧缝针 |author =  |date = 2017-03-04 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

2016年改革前，医院是[[第二军医大学|第二军医大学]]、[[中国药科大学|中国药科大学]]、[[江西中医学院|江西中医学院]]、[[皖南医学院|皖南医学院]]的教学医院，是江西中医学院药剂学和药物化学专业及皖南医学院口腔临床医学专业联合硕士培养点，是海军医学研究所的临床研究部。1993年，被[[中国人民解放军总后勤部|中国人民解放军总后勤部]]评为军队首批[[三级甲等医院|三级甲等医院]]。1996年，被纳入上海市首批医疗保险定点医院。多次被总部评为“白求恩杯”优质服务先进单位<ref name=zgjw/>。

2016年改革前，医院实施学科建设“4321”工程计划，即巩固提高脊柱外科、口腔、药学、计划生育优生优育4个海军专科中心，发展烧伤整形、心脑血管、空潜科3个重点学科，建成肿瘤治疗、血管介入治疗2个专病治疗中心，重点关注一批优势学科<ref name=zgjw/>。

医院担负体系部队的医疗保障任务，并承担航空兵、潜艇、潜水人员特勤医疗保障，[[神舟|神舟]]系列飞船后勤支援医院，海上医疗队卫勤保障以及舰艇出海、出访、南海巡航等医疗保障任务。2008年[[汶川大地震|汶川大地震]]发生后，医院派出2支医疗队共63人在90多天中完成了抗震救灾医疗保障任务，获[[中共中央|中共中央]]、[[中华人民共和国国务院|国务院]]、[[中央军事委员会|中央军委]]授予“全国抗震救灾英雄集体”称号，2人立二等功<ref name=zgjw/>。

医院营区主要由四个院落组成，占地总面积68亩。
*本部（医疗区）：上海市东江湾路15号，占地面积22亩，有外科大楼、内科大楼、门诊楼、服务大楼和伽玛刀楼。2008年初，服务大楼投入使用。住院大楼2010年下半年投入使用。至此医疗区总建筑面积达6万余平方米<ref name=zgjw/>。
*药厂、单身干部宿舍：上海市广粤路469号，占地面积6.7亩<ref name=zgjw/>。
*体检中心：上海市多伦路210号原白崇禧公馆，占地面积1.6亩<ref name=zgjw/>。
