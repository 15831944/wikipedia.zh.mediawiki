'''中華民國與馬利關係'''是指[[中華民國|中華民國]]與[[馬利共和國|馬利共和國]]之間的關係。兩國於1960年有短暫的官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對馬利的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-238-18c67-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[馬利|馬利]]。]]
{{see also|中華民國外交|{{link-en|馬利外交|Foreign relations of Mali}}}}
1960年6月20日，與[[法屬蘇丹|法屬蘇丹]]以及[[塞內加爾|塞內加爾]]組成的「[[馬利聯邦|馬利聯邦]]」建立大使級[[外交#外交上的承認|外交關係]]。於聯邦首都[[達卡_(塞內加爾)|達卡]]（今[[塞內加爾|塞內加爾]]首都）設立'''中華民國駐馬利聯邦[[外交代表機構#大使馆|大使館]]'''，但只派駐[[參事|參事]]等級外交人員。

1960年8月20日，塞內加爾退出馬利聯邦，聯邦制解體。9月22日，馬利退出[[法蘭西共同體|法蘭西共同體]]成為完全獨立國家。

1960年10月27日，馬利與中華民國斷交。

1986年12月5日，簽署《中（華民）國臺灣與馬利（共和國）間國際[[特快专递|快捷郵件]]服務協定》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e9%a6%ac%e5%88%a9&tycountry_e=%e9%a6%ac%e5%88%a9&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境馬利。]]
[[File:Visa_requirements_for_Malian_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|馬利簽證政策|Visa policy of Mali}}|{{link-en|馬利公民簽證要求|Visa requirements for Malian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往馬利駐[[南非|南非]][[外交代表機構#大使馆|大使館]]申請簽證。入境須出示國際[[疫苗接種|疫苗接種]]證明（黃皮書）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-238-18c67-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|馬利護照|Malian passport}}的{{link-en|馬利人口|Demographics of Mali|馬利公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將馬利歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2016年10月27日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-251-d8fc1-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|馬利經濟|Economy of Mali}}}}
===貿易===
2017年，馬利是中華民國第162大貿易夥伴、第124大[[進口|進口]]夥伴、第189大[[出口贸易|出口]]夥伴。出口至馬利的金額為52萬0,459[[美元|美元]]，年減21.880%；自馬利進口的金額為490萬2,127美元，年增84.848%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）438萬1,668美元，年增120.656%。※2016年，馬利是中華民國第173大貿易夥伴、第136大進口夥伴、第184大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
