'''中国人民解放军联勤保障部队第九〇〇医院'''，原名'''中国人民解放军福州总医院'''（简称'''福州总医院'''），又称'''厦门大学附属东方医院'''，本部位于[[福建省|福建省]][[福州市|福州市]]西二环北路156号，隶属[[无锡联勤保障中心|无锡联勤保障中心]]，是[[三级甲等医院|三级甲等医院]]。

==简介==
前身为[[中华民国|中华民国]]时期的'''[[國防部聯合後勤司令部|联勤总部]]第二肺病疗养院'''。1949年8月17日[[中国人民解放军|中国人民解放军]]攻克[[福州|福州]]，该疗养院由[[中国人民解放军福建省军事管制委员会|中国人民解放军福建省军事管制委员会]]接管并改编为'''[[福建省卫生厅|福建省卫生厅]]直属医院'''。1951年1月，被中国人民解放军第十兵团[[福建省军区|福建省军区]]接收，1954年11月扩编为'''中国人民解放军第九十三医院'''，院址从福州西禅寺迁到福州西门后县村。1959年1月[[福州军区|福州军区]]成立，该医院更名为'''中国人民解放军福州军区总医院'''。1985年9月，福州军区并入[[南京军区|南京军区]]，该医院更名为'''中国人民解放军南京军区福州总医院'''<ref>{{Cite web |url = http://www.fzzyy.com/keshi/yjxt/2014-07-17/158.html |title =医学影像科  |author =  |date =  2014-07-17|publisher =南京军区福州总医院  |language =  |accessdate =  |quote =  }}</ref>。2009年7月10日，厦门大学附属东方医院签约仪式在[[厦门大学|厦门大学]]举行，南京军区福州总医院成为'''厦门大学附属东方医院'''。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由中国人民解放军南京军区转隶[[无锡联勤保障中心|无锡联勤保障中心]]，并更名'''中国人民解放军福州总医院'''<ref>{{Cite web |url = http://military.people.com.cn/n1/2017/0426/c1011-29236947.html |title = 无锡联勤保障中心构建军属优先医保新模式 |author =  |date = 2017-04-26  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。2018年11月再次更名为'''中国人民解放军联勤保障部队第九〇〇医院'''<ref>{{Cite web |url = https://www.thepaper.cn/newsDetail_forward_2610757 |title =联勤保障部队调整改革：多地多所军队医院集中更名 |author =钟煜豪  |date = 2018-11-07 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

医院是大型综合性[[三级甲等医院|三级甲等医院]]，也是[[中华人民共和国卫生部|中华人民共和国卫生部]]国际紧急救援网络医院，并设有福建省医疗系统首家博士后科研工作站<ref name="jj" /><ref>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536733.htm |title =南京军区福州总医院  |author =  |date = 2014-09-12  |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。2016年改革前，医院由院本部、第二住院部、莆田九五临床部、宁德四四二临床部、福州四七六临床部组成，医院本部设有53个专业学科，医院拥有1个国家级重点学科，1个国家重点专科在建项目，1个[[中华人民共和国科学技术部|中华人民共和国科学技术部]]示范型国际科技合作基地，是国家药物临床试验机构，医院设全军重点学科和基地10个、南京军区重点学科13个、福建省创新平台2个、重点实验室2个<ref name=jj>{{Cite web |url =http://www.fzzyy.com/yiyuangaikuang/yyjj/  |title =  医院简介|author =  |date =  |publisher =  南京军区福州总医院 |language =  |accessdate = 2017-06-19 |quote =  }}</ref>。
