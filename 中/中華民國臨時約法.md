{{中国宪政史}}
《'''中華民國臨時約法'''》又名'''民元約法'''，是在1912年[[中华民国|中华民国]]成立後，由當時位於[[南京|南京]]的[[中華民國臨時參議院|臨時參議院]]所制定的具有[[憲制|憲制]]性的文件，實際上是中華民國的第一部[[憲法|憲法]]。中华民国成立後，[[孙中山|孙中山]]擔任[[臨時大總統|臨時大總統]]兩個月後，迫于当时的形势，同意辭職並推举袁世凯为第二任临时大总统，但为了限制袁世凯的权力<ref>{{cite web|url=http://news.ifeng.com/history/special/yuanshikai/200906/0622_6961_1213937_1.shtml|title=象征与实质:《临时约法》问题}}</ref>，遂制定《中華民國臨時約法》。《中華民國臨時約法》在1912年3月8日由臨時參議院通过，3月11日公布实施，取代《[[中華民國臨時政府組織大綱|中華民國臨時政府組織大綱]]》，在袁世凯就任後生效。1914年5月1日因[[中华民国大总统|大總統]][[袁世凯|袁世凯]]公布《[[中华民国约法|中华民国约法]]》而被取代。1916年6月29日為[[中华民国大总统|大總統]][[黎元洪|黎元洪]]所恢復。

== 主要內容 ==
[[File:中华民国临时约法.jpg|thumb]]
* [[行政權|行政權]]
# 仿[[法蘭西第三共和國|法國]]式之[[內閣制|責任內閣制]]：蓋當時之參議院（國會）為架空袁世凱，乃將原《[[中華民國臨時政府組織大綱|中華民國臨時政府組織大綱]]》之[[總統制|總統制]]改為[[內閣制|內閣制]]使[[袁世凱|袁世凱]]成為[[總統|虛位總統]]。
# 總綱以簡潔之文字，將國家之要素作原則性的規定。
# 人民權利義務之保障已有詳盡之規定，並設有[[法律保留|法律保留]]條款。
# 大總統副總統之選舉：仍沿《[[中華民國臨時政府組織大綱|中華民國臨時政府組織大綱]]》之精神由[[臨時參議院|臨時參議院]]選舉出任。
* [[立法權|立法權]]
# 國會議員由人民直接選出，設[[臨時參議院|臨時參議院]]，產生[[中華民國初年國會|國會]]。
* [[司法權|司法權]]
# 已有獨立審判規定，符合[[三權分立|三權分立]]原則：臨時約法規定，法院由臨時大總統及司法總長分別任命之[[大法官|大法官]]組成，法官的獨立審判不受上級官廳之干涉。

== 沿革 ==
中國於1917年至1928年為分裂时期，南北各有政府，互不從屬，但以「約法」為憲法及作為[[法統|法統]]，直至1925年廢止，南北政府均不再堅持約法。

=== [[北洋政府|北洋政府]]一系 ===
* 1917年（民國六年）7月1日，被复辟帝制的[[张勋|张勋]]破坏，随后的[[段祺瑞|段祺瑞]]政府拒绝恢复。
* 1922年（民國十一年）4月，被控制北京的[[曹锟|曹锟]]和[[吴佩孚|吴佩孚]]以“法统重光”為号召，再度恢复。
* 1923年（民國十）10月10日，被國人稱為“[[曹錕憲法|曹錕憲法]]”的《中華民國憲法》施行而取代。
* 1925年4月24日，[[段祺瑞|段祺瑞]][[臨時執政|臨時執政]]政府发布命令，称“法统已成陈迹”，臨時約法废除。

=== [[廣州軍政府|廣州軍政府]]及[[國民政府|國民政府]]一系 ===
* 1917年（民國六年）9月10日，以[[廣東|廣東]]為基地的[[中華民國軍政府|中華民國軍政府]]展開[[護法運動|護法運動]]，所護者即為《中華民國臨時約法》。
* 1925年7月1日，[[中國國民黨|中國國民黨]]建立[[國民政府|國民政府]]，[[蔣中正|蔣中正]]在南京另组[[國民政府|國民政府]]後即少談及《中華民國臨時約法》，但未正式廢止。
* 1931年6月1日，《[[中華民國訓政時期約法|中華民國訓政時期約法]]》公布，《中華民國臨時約法》依新法優於舊法原則而失其最高效力。

== 評價 ==
* [[中國共產黨|中國共產黨]][[李大釗|李大釗]]形容《臨時約法》“為集矢之的”，《約法》雖然規定人民享有言論自由，但在實踐中《約法》毫無約束力<ref>杜鋼建，《中國近百年人權思想》</ref>。
