'''中華民國與塞內加爾關係'''是指[[中華民國|中華民國]]與[[塞內加爾共和國|塞內加爾共和國]]之間的關係。兩國於1960－2005年有官方[[外交#外交上的承認|外交關係]]，期間多次斷交與復交。第3次斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對塞內加爾的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-236-9b1ff-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[塞內加爾|塞內加爾]]。]]

{{see also|中華民國駐塞內加爾大使列表{{!}}中華民國駐塞內加爾外交官列表|塞內加爾駐中華民國大使列表{{!}}塞內加爾駐中華民國外交官列表}}
1960年6月20日，與[[塞內加爾|塞內加爾]]以及[[法屬蘇丹|法屬蘇丹]]組成的「[[馬利聯邦|馬利聯邦]]」建立大使級[[外交#外交上的承認|外交關係]]。於聯邦首都[[達卡_(塞內加爾)|達卡]]（今塞內加爾首都）設立'''中華民國駐馬利聯邦[[外交代表機構#大使馆|大使館]]'''，但只派駐[[參事|參事]]等級外交人員。

1960年8月20日，塞內加爾退出馬利聯邦，聯邦制解體。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-3.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1960年9月23日，塞內加爾與中華民國建立公使級外交關係。<ref name="89年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_7.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1964年9月，派駐塞國[[特命全權公使|公使]]（未到任）。11月8日，與中華民國首次斷交。<ref name="89年外交年鑑"/>

1969年7月16日，恢復邦交，並升格為大使級外交關係。<ref name="89年外交年鑑"/>1970年，於首都達卡設立'''中華民國駐塞內加爾共和國大使館'''，並派駐[[特命全權大使|大使]]。

1971年12月7日，與中華民國第二次斷交。1972年4月12日，關閉駐塞國大使館，改設立[[實質大使館|具大使館性質]]的'''駐塞內加爾經濟暨技術合作[[辦事處|辦事處]]'''，於1976年10月31日關閉。<ref name="89年外交年鑑"/>

1996年1月3日，再度恢復大使級外交關係。1月12日，設立駐塞國大使館。<ref name="89年外交年鑑"/>

1998年10月，於首都[[台北市|台北]]設立'''塞內加爾共和國駐中華民國大使館'''，並派駐大使。<ref name="87年外交年鑑"/>

2004年9月23日，[[塞內加爾總統|塞內加爾總統]][[阿卜杜拉耶·瓦德|瓦德]]、[[宏都拉斯總統|宏都拉斯總統]][[里卡多·馬杜羅|馬度洛]]、{{link-en|帛琉副總統|Vice President of Palau}}[[桑德拉·皮耶兰托齐|皮蘭朵茲]]、[[史瓦濟蘭|史瓦濟蘭]]副總理夏班固（Albert H.N. Shabangu）在第59屆[[聯合國大會|聯合國大會]]總辯論（General Debate）發表演說時，為「中華民國參與[[聯合國|聯合國]]案」聲援。塞內加爾總統瓦德在總辯論發言內容：「個人認為，聯合國在推行普遍性之作法並不公平。在《[[聯合國憲章|聯合國憲章]]》揭櫫公義與普遍性之核心原則下，[[中華民國在台灣|台灣之中華民國]]擁有[[台灣人口|二千三百萬居民]]、健全的[[民主制度分类|民主制度]]，以及令人印象深刻之[[臺灣經濟|經濟活力]]，卻仍遭排除於[[联合国系统|聯合國活動]]之外。」<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=8742DCE7A2A28761&sms=491D0E5BF5F4BC36&s=7A30ACF3DE86847D |author = 公眾外交協調會 |date = 2004年9月24日 |title = 友邦塞內加爾及宏都拉斯兩國總統、帛琉副總統及史瓦濟蘭副總理分別在聯大總辯論期間為我執言 |publisher = 中華民國外交部}}</ref>

2005年10月25日，與中華民國第三次斷交。塞內加爾總統瓦德致函[[中華民國總統|中華民國總統]][[陳水扁|陳水扁]]，宣佈與[[中華人民共和國|中華人民共和國]]復交，而將只與中華民國維持經濟、商業暨文化關係，瓦德並在函中表明「國家間沒有朋友，只有利益」。據[[中華民國外交部|外交部]]了解，中華人民共和國方面提出2億[[美元|美元]]的援助作為建交（復交）的條件。而在9月16日，瓦德公開表示中華人民共和國願意替塞內加爾修建一條[[水底隧道|河底隧道]]，連結[[甘比亞|甘比亞]]與塞內加爾，金額不明。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=70BCE89F4594745D&sms=700DE7A3F880BAE6&s=7036A2E764EA61DC |author = 公眾外交協調會 |date = 2005年10月25日 |title = 「中華民國政府宣佈中止與塞內加爾共和國外交關係」新聞說明會紀要 |publisher = 中華民國外交部}}</ref>

===貿易===
2017年，塞內加爾是中華民國第140大貿易夥伴、第140大[[進口|進口]]夥伴、第124大[[出口贸易|出口]]夥伴。出口至塞內加爾的金額為884萬1,595[[美元|美元]]，年增44.998%；自塞內加爾進口的金額為264萬5,896美元，年減25.320%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）619萬5,699美元，年增142.515%。※2016年，塞內加爾是中華民國第145大貿易夥伴、第130大進口夥伴、第136大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

==交流==
===援助===
1996年1月兩國復交時，中華民國承諾協助塞內加爾發展[[稻|稻作]]，以達成糧食增產目標。7月，簽署《[[農業技術|農業技術]]合作協定》並設立駐塞內加爾農技團，工作項目為稻作增產、[[吳郭魚|吳郭魚]]及[[牡蠣|牡蠣]]養殖、[[園藝|園藝]]及[[蔬菜|蔬菜]]發展等；1996年1月，簽署《[[醫療|醫療]]合作協定》，承諾每年提供醫療援助金給予塞內加爾，用以採購[[醫療器械|醫療器械]]及[[藥品|藥品]]、聘請當地及國外醫事人員、派遣醫事人員赴國外深造等；此外，並協助塞內加爾發展[[中小企業|中小企業]]、<ref name="88年外交年鑑"/>開發小型[[工業園區|工業園區]]等。<ref name="90年外交年鑑"/>

===僑民===
