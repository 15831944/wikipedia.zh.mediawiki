'''中国人民解放军国防大学军事管理学院'''，位于[[北京市|北京市]][[海淀区|海淀区]]红山口甲3号，隶属[[中国人民解放军国防大学|中国人民解放军国防大学]]。

==沿革==
2017年[[深化国防和军队改革|深化国防和军队改革]]前，国防大学原设有多个教研机构，包括：战略教研部、战役教研部、马克思主义理论教研部、信息作战与指挥训练教研部、军队建设与军队政治工作教研部、军事后勤与军事科技装备教研部<ref name=xhw>{{Cite web |url =http://news.xinhuanet.com/mil_new/2007-11/23/content_7132872.htm |title =国防大学  |author =  |date = 2007-11-23 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。军队建设与军队政治工作教研部下设有国防大学军队建设研究所等机构<ref>{{Cite web |url = http://military.people.com.cn/GB/52578/52579/367200/index.html |title = 聚焦中国国防力量发展新变化 |author =  |date = 2013-07-26 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。另外还有国防大学国防经济研究中心<ref>姜鲁鸣，军民融合战略下的国防军工发展趋势，军工文化2015年06期</ref>等单位。

在2017年[[深化国防和军队改革|深化国防和军队改革]]中，重新组建新的国防大学，下设'''中国人民解放军国防大学军事管理学院'''，位于国防大学校本部<ref name=wangbao>{{Cite web |url = http://yz.chsi.com.cn/sch/viewBulletin--schId-367819,infoId-1873291567,categoryId-478769,mindex-7.dhtml |title =国防大学2018年网报说明  |author =  |date =  |publisher = 中国研究生招生信息网 |language =  |accessdate = 2017-11-07 |quote =  }}</ref>。该学院是由军队建设与军队政治工作教研部<ref>{{Cite web |url =  http://military.cnr.cn/ztch/sjd/jd/20171011/t20171011_523981790.html|title =“和平统一、一国两制”是实现国家统一的最佳方式  |author =  |date = 2017-10-12 |publisher =央广网  |language =  |accessdate =  |quote =  }}</ref>、军队党史党建研究中心<ref>{{Cite web |url =http://news.china.com/zw/news/13000776/20170922/31502838.html  |title =  刘志兵：“两论”的写作背景、历史贡献和现实意义|author =  |date = 2017-09-22 |publisher = 中华网 |language =  |accessdate =  |quote =  }}</ref>、国防经济研究中心、军民融合深度发展研究中心<ref>{{Cite web |url = http://dzb.studytimes.cn/shtml/xxsb/20171101/30030.shtml |title = 军民融合，怎么融？ |author =  罗永光|date =2017-11-01  |publisher = 学习时报 |language =  |accessdate =  |quote =  }}</ref>等部门合并重组而成。
