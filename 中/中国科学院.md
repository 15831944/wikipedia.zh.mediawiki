'''中国科学院'''，简称'''中科院'''，是[[中华人民共和国科技|中华人民共和国科学技术]]方面的最高学术机构，全国[[自然科学|自然科学]]与[[高新技术|高新技术]]综合研究发展中心，为正部级[[国务院直属事业单位|国务院直属事业单位]]<ref name="章程"/>。

中国科学院於1949年11月在[[北京市|北京]]成立。1977年5月，哲学社会科学学部独立為[[中国社会科学院|中国社会科学院]]，1994年，在技術科學部的基礎上及[[中华人民共和国科学技术部|國家科委]]的支持下，成立[[中国工程院|中国工程院]]。

中国科学院与中国工程院并称“[[两院|两院]]”。发展至今，中科院下设6个[[学部|学部]]（数学物理学部、化学部、生命科学和医学学部、地学部、信息技术科学部、技术科学部），以及12个分院（北京、沈阳、长春、上海、南京、武汉、广州、成都、昆明、西安、兰州、新疆）、84个研究院所、2所大学、2所学院、4个文献情报中心、3个技术支撑机构和2个新闻出版单位，分布在全国20多个省（市）。此外，还投资兴办了430余家科技型企业（含转制单位），涉及11个行业，其中包括8家上市公司。此外，中国科学院还拥有（含与其他单位共建）4个[[中华人民共和国国家实验室列表|国家实验室]]、85个[[中华人民共和国国家重点实验室列表|国家重点实验室]]和153个[[中国科学院重点实验室|中国科学院重点实验室]]。

== 历史 ==
=== 初创时期 ===
[[File:Academia_Sinica_site_nanjing.JPG|thumb]]
1949年11月，[[中华人民共和国|中华人民共和国]]成立后仅一个月，中国科学院在[[北京|北京]]成立。

=== 文革时期 ===
[[File:Chinese_nuclear_bomb_-_A2923.jpg|thumb]]
1966年[[文化大革命|文化大革命]]爆发后，[[国家科学技术委员会|国家科学技术委员会]]、[[中国科学技术协会|中国科学技术协会]]、中国科学院先后被“砸烂”。1968年底，中国科学院仅在北京的171位高级研究人员中，便有131位先后被列为打倒和审查对象。文革期间，全院被迫害致死者达229名。除被立案审查或打倒外，包括中国科学院人员在内的中国大批科技人员及科技干部被下放到[[五七干校|五七干校]]劳动。<ref name="kjrb">{{Cite web|url=http://www.most.gov.cn/ztzl/kxdct30/kxdct30ztxw/200803/t20080317_59810.htm|title=惊回眸，那个春天——记1978年全国科学大会召开的前前后后|accessdate=2018-04-04|date=2008-03-17|work=科技日报|publisher=中华人民共和国科学技术部}}</ref>1969年9月20日，[[毛泽东思想宣传队|毛泽东思想宣传队]]进驻中国科学院，1969年11月29日撤出。<ref>{{Cite web|url=http://www.cas.cn/jzzky/ysss/bns/200909/t20090928_2529147.shtml|title=1968|accessdate=2018-04-04|date=2009-09-28|publisher=中国科学院}}</ref>

1970年7月1日，中国科学院革命委员会与国家科学技术委员会军事管制委员会合并，组成新的中国科学院革命委员会，对外活动时仍用中国科学技术协会的名义。<ref name=kjrb/><ref>{{Cite web|url=http://www.cas.cn/jzzky/ysss/bns/200909/t20090928_2529179.shtml|title=1970|accessdate=2018-04-04|date=2009-09-28|publisher=中国科学院}}</ref>1972年7月，中国共产党中国科学院[[党的核心小组|核心小组]]成立，中共中央任命[[郭沫若|郭沫若]]为中国共产党中国科学院核心小组组长。<ref>{{Cite web|url=http://www.cas.cn/jzzky/ysss/bns/200909/t20090928_2529191.shtml|title=1972|accessdate=2018-04-04|date=2009-09-28|publisher=中国科学院}}</ref>1975年初[[邓小平|邓小平]]主持中央和国务院日常工作后，提出整顿中国科学院。这次整顿是对全国科技工作领导机构的整顿，重点是调整知识分子政策和科技政策。<ref name=kjrb/>1975年7月18日，[[胡耀邦|胡耀邦]]、[[李昌|李昌]]、[[王光伟|王光伟]]到中国科学院主持工作。8月17日，胡耀邦等人将汇报提纲《关于科技工作的几个问题》报送邓小平。9月26日，邓小平主持的国务院会议讨论并基本通过了《关于科技工作的几个问题》，邓小平让在国务院研究室工作的[[胡乔木|胡乔木]]帮助修改，胡乔木作较多修改，并将原题《关于科技工作的几个问题》改为《科学院工作汇报提纲》。10月7日，中央通知胡耀邦任中国共产党中国科学院核心小组第一副组长，李昌、王光伟任副组长，[[王屏_(少将)|王屏]]、[[刘华清|刘华清]]、[[胡克实|胡克实]]任核心小组成员。核心小组组长仍是郭沫若，[[武衡|武衡]]继续任核心小组副组长。<ref name="yjqw">{{Cite web|url=http://www.cas.cn/jzzky/ysss/bns/200909/t20090928_2529195.shtml|title=1975|accessdate=2018-04-04|date=2009-09-28|publisher=中国科学院}}</ref>1975年11月，[[毛泽东|毛泽东]]发动“[[批邓、反击右倾翻案风|批邓、反击右倾翻案风]]”。《论全党全国各项工作的总纲》、《科学院工作汇报提纲》、《关于加快工业发展的若干问题》被批判为“三株大毒草”，因此夭折。<ref name=kjrb/><ref name=yjqw/>1977年5月，中国科学院哲学社会科学部另组[[中国社会科学院|中国社会科学院]]。<ref name="yjqq">{{Cite web|url=http://www.cas.cn/jzzky/ysss/bns/200909/t20090928_2529237.shtml|title=1977|accessdate=2018-04-04|date=2009-09-28|publisher=中国科学院}}</ref>1977年6月22日至7月7日，中共中国科学院核心小组第一副组长、副院长方毅主持召开院工作会议。会议讨论了《1978年至1985年中国科学院发展规划纲要（草案）》，决定取消院、所两级[[革命委员会_(文革)|革命委员会]]。<ref name=yjqq/>1977年9月18日，[[中国共产党中央委员会|中共中央]]批准重建[[中华人民共和国科学技术部|国家科学技术委员会]]，作为国务院所属的一个主管科学技术工作的部门，方毅任主任。中国科学院不再承担国家科学技术委员会的职能。<ref name=yjqq/>1978年，中国科学院成立中国科学技术大学研究生院（北京）。

=== 改革开放 ===
2001年，中国科学技术大学研究生院（北京）改名中国科学院研究生院。2012年7月23号，教育部同意中国科学院研究生院更名为[[中国科学院大学|中国科学院大学]]的申请，同时教育部建议中科院撤销中科院研究生院的建制<ref>{{cite news|url=http://news.163.com/12/0717/10/86JV3LB300014JB6.html|title=中科院研究生院更名中科院大学|date=2012-07-17|work=网易|accessdate=2012-07-18}}</ref>。

== 职责 ==
根据《{{s|中国科学院职能配置、内设机构和人员编制规定}}》，中科院承担下列职能<ref name="三定方案">{{cite web |author1=国务院办公厅 |title=中国科学院职能配置、内设机构和人员编制规定|url=http://www.gov.cn/zhengce/content/2010-12/29/content_7702.htm |website=中华人民共和国中央人民政府门户网站 |accessdate=2018-12-05 |location=北京 |language=zh-cn}}</ref>：
{{columns-list|2|
# 瞄准国家发展战略目标和国际科技前沿，主要从事基础研究和战略性研究，重点研究和解决中国现代化建设中的基础性、战略性、综合性、前瞻性重大科技问题。
# 结合科学研究，培养和输送具有创新意识和创新能力的高素质科技人才。
# 促进中国高技术产业的发展和传统产业技术更新；为提高中国技术创新能力提供坚实基础。
# 加强中国科学院学部建设，为国家宏观决策提供咨询、提出建议和参与对国家重大科技问题的评议。
# 按照开放、流动、竞争、择优的原则，建设一支包括一批国际知名的科学家、一批中青年科研骨干和精干高效的技术专家、管理人员的高水平的科技队伍。
# 逐步建立和不断完善与国际基本接轨并符合中国国情的国家科研机构新体制和现代科研院所制度；重点建设和稳定支持一批具有国际水平的国家知识创新基地，发挥国家知识创新体系核心的作用。
# 加大开放力度，广泛开展国内外学术交流与合作，促进知识和人才交流；提高国际科技交流与合作的水平，拓展渠道，提高层次，扩大规模；发展与香港特别行政区、澳门及台湾地区的科技交流与合作；建立与社会和企业的联系，探索与跨国高技术企业之间的合作途径。
# 承办国务院交办的其他事项。
}}

=== 学部与院士 ===
{{Main|中国科学院学部|中国科学院院士}}

中国科学院学部成立于1955年，是中国国家最高科学技术咨询机构，现设[[中国科学院数学物理学部|数学物理学部]]、[[中国科学院化学部|化学部]]、[[中国科学院生命科学和医学学部|生命科学和医学学部]]、[[中国科学院地学部|地学部]]、[[中国科学院信息技术科学部|信息技术科学部]]和[[中国科学院技术科学部|技术科学部]]六个学部。

中国科学院院士从全国最优秀的科学家中选出，每两年增选一次。目前有院士702人，外籍院士53人。[[中国科学院院士大会|全体院士大会]]是最高权力机构，其常设领导机构是学部主席团，由中国科学院院长担任学部主席团执行主席。
