'''中国人民解放军空军临潼航空医学鉴定训练中心'''，位于[[陕西省|陕西省]][[西安市|西安市]][[临潼区|临潼区]]华清路26号，隶属[[中国人民解放军中部战区空军|中国人民解放军中部战区空军]]后勤部。

==简介==
空军临潼航空医学鉴定训练中心的前身是1952年创建的'''中国人民解放军空军临潼疗养院'''<ref name=xayx>{{Cite web |url =  http://www.xagdyz.com/info/1039/2327.htm|title = 空军临潼航空医学鉴定训练中心 |author =  |date = 2014-05-13 |publisher =西安医学高等专科学校  |language =  |accessdate =  |quote =  }}</ref>。2009年7月，都江堰、杭州、大连、青岛、临潼五所疗养院统一由各大军区联勤部转隶各大军区空军后勤部，成立5个航空医学鉴定训练中心，构建为“[[中国人民解放军空军总医院|空军总医院]]、[[中国人民解放军空军航空医学研究所|空军航空医学研究所]]——区域性航空医学鉴定训练中心——航空兵部队卫生机构”三级航空医学鉴定训练体系<ref>何强、彭天国、董文，空军航空医学鉴定训练中心的发展与思考 ，全国航空航天医学学术会议暨云南省航空航天医学学术会议2010,27(3):180-181</ref><ref>{{Cite web |url = http://news.xinhuanet.com/mil/2011-12/25/c_111295752.htm |title = 一场从陆地打响的空战 |author =  |date = 2011-12-25 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。2009年'''中国人民解放军空军临潼航空医学鉴定训练中心'''成立后，隶属[[兰州军区空军|兰州军区空军]]后勤部。担负飞行员、航天员的疗养、航空生理心理训练、航空医学鉴定等任务，并且配合招飞中心开展招飞初检<ref name=xayx/><ref>段虹、程书洋，空军航空医学鉴定训练中心建设实践与启示，空军医学杂志2011年02期</ref><ref>{{Cite web |url =http://www.chinanews.com/mil/2014/08-13/6486500.shtml  |title = 空军某医学鉴定训练中心演练突发事件急救 |author =  |date = 2014-08-13 |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，转隶[[中国人民解放军中部战区空军|中国人民解放军中部战区空军]]后勤部。
