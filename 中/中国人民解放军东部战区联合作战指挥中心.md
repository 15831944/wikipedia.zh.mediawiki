'''中国人民解放军东部战区联合作战指挥中心'''，位于[[江苏省|江苏省]][[南京市|南京市]]，隶属[[中国人民解放军东部战区联合参谋部|中国人民解放军东部战区联合参谋部]]，是[[中国人民解放军东部战区|中国人民解放军东部战区]]的联合作战指挥机构。

==简介==
早在[[深化国防和军队改革|深化国防和军队改革]]前，中国人民解放军便已在战区层次设有东海联合作战指挥中心。2014年7月31日，[[中华人民共和国国防部|中华人民共和国国防部]]新闻发言人[[耿雁生|耿雁生]]回应有关东海联合作战指挥中心已成立的新闻时，并未予以否认，他回答称：“中国军队在建立联合作战指挥体制方面进行了积极探索，将走出一条具有中国特色的联合作战指挥体系改革之路。”<ref name="ifeng140805">[http://news.ifeng.com/a/20140805/41452210_0.shtml?f=hao123 外媒：中共设立中央军委联合作战指挥中心]，凤凰网，2014-08-05</ref><ref>{{Cite web |url = http://news.qq.com/a/20140729/065071.htm|title = 外媒称中国设立东海联合作战指挥中心 |author =  |date = 2014-07-29 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>

2015年11月27日，[[中华人民共和国国防部|中华人民共和国国防部]]新闻发言人[[杨宇军|杨宇军]]就[[深化国防和军队改革|深化国防和军队改革]]在健全联合作战指挥体制上有何举措时称，“主要是适应打赢信息化战争、有效履行使命任务的要求，组建战区联合作战指挥机构，健全军委联合作战指挥机构，构建完善精干高效的战略战役指挥体系。”<ref>{{Cite web |url =http://military.china.com.cn/2015-11/27/content_37177664.htm  |title =国防部谈军队改革：组建战区联合作战指挥机构  |author =  |date =2015-11-27  |publisher =  中国网|language =  |accessdate =  |quote =  }}</ref>2016年2月1日，中华人民共和国国防部新闻局局长、国防部新闻发言人杨宇军在国防部专题新闻发布会上表示，战区是“本战略方向的唯一最高联合作战指挥机构”<ref>{{Cite web |url = http://news.163.com/16/0202/06/BEQ5BF2500014AED.html|title = 战区是唯一最高联合作战指挥机构|author =  |date =  2016-02-02 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2016年2月前后，新成立的五个战区分别设立战区联合作战指挥中心。战区联合作战指挥中心是在此前各[[大军区|大军区]]作战指挥中心的基础上建成，增加了空军、火箭军等军兵种，增设了测绘导航、空域管理等作战要素，调整了指挥席位的设置<ref>{{Cite web |url = http://mil.huanqiu.com/china/2016-02/8623038.html |title =西部战区砍原军区数十个指挥席 30分钟发7方案  |author =  |date = 2016-02-29 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref>。其中东部战区联合作战指挥中心设有数十个指挥席位<ref>{{Cite web|url =http://news.hexun.com/2016-04-21/183441426.html|title =军委联指中心座位分布复杂 挂两巨幅地图|author =|date =2016-04-21|publisher =和讯|language =|accessdate =|quote =}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2016年2月，东部战区联合作战指挥中心联合作战值班进入试运行，编印了《联合作战指挥中心运行规范》<ref>{{Cite web |url =http://military.people.com.cn/n1/2016/0216/c1011-28126703.html  |title = 东部战区联合作战指挥中心“硝烟味”浓烈 |author =  |date = 2016-02-16 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。2017年2月13日，东部战区组织东部战区联合作战指挥中心人员值班资格首批认证仪式，数百位指挥员和参谋人员获得初级值班资格证书和徽章，至此东部战区机关超过八成的人员通过了《战区联指中心值班人员资格认证实施办法（试行）》规定的值班人员资格认证<ref>{{Cite web |url = http://www.chinanews.com/mil/2017/02-23/8157253.shtml |title = 东部战区人员能不能指挥联合作战要“考证” |author =  |date = 2017-02-23 |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。
