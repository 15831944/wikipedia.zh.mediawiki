《'''中华人民共和国国歌法'''》是[[全國人民代表大會常務委員會法制工作委員會|全國人民代表大會常務委員會法制工作委員會]]根据《[[中华人民共和国宪法|中华人民共和国宪法]]》中关于[[义勇军进行曲|义勇军进行曲]]（国歌）的条款制定的规范对[[中华人民共和国|中华人民共和国]][[国歌|国歌]]的使用的全国性[[中华人民共和国法律|法律]]，其内容包括中华人民共和国国歌的奏唱场合、奏唱礼仪、[[公民教育|宣传教育]]，以及违反该法的后果。

《中华人民共和国国歌法》于2017年9月1日经[[第十二届全国人民代表大会常务委员会|第十二届全国人民代表大会常务委员会]]第二十九次会议决议通过，现已自2017年10月1日起施行。

==立法背景与过程==
[[全國人民代表大會常務委員會法制工作委員會|全国人大常委会法工委]]主任[[沈春耀|沈春耀]]在草案提审的说明中称：“多年来，国歌所承载的爱国情怀、忧患意识和奋勇前行的民族精神、深入人心，广大人民群众热爱祖国、尊崇国歌，国歌奏唱和使用总体情况是比较好的。但实际生活中也存在奏唱国歌不规范、参与者举止不得体，国歌标准曲谱未予发布，影响奏唱和播放效果等。”<ref>{{Cite web |url = http://www.npc.gov.cn/npc/xinwen/2017-09/01/content_2028101.htm |title = 关于《中华人民共和国国歌法（草案）》的说明}}</ref>

在《中华人民共和国国歌法》制定之前，有先后两个规范中华人民共和国国歌使用的指导意见，一个是于1984年8月1日由[[中共中央宣传部|中共中央宣传部]]为规范中华人民共和国国歌在中华人民共和国境内的奏唱所下发的《关于中华人民共和国国歌奏唱的暂行办法》，另一个是于2014年12月12日由[[中共中央办公厅|中共中央办公厅]]、[[中华人民共和国国务院办公厅|国务院办公厅]]印发的《[[:s:关于规范国歌奏唱礼仪的实施意见|关于规范国歌奏唱礼仪的实施意见]]》，内容主要涉及国歌奏唱场合、奏唱礼仪、[[公民教育|国歌宣传教育]]等，并且对外事活动、运动赛事、学校活动等活动中的国歌使用进行了详细指导。

2008年起，[[中國人民政治協商會議全國委員會|全国政协]]委员[[于海_(指挥家)|于海]]每年都在政协会议上提议应该对国歌进行立法<ref>{{Cite web |url = http://www.bj.xinhuanet.com/jzzg/2017-05/08/c_1120938610.htm |title = “国歌最后一句为什么多个‘进’字？”——专访提案为国歌立法的全国政协委员于海 |author =  |date =2017-05-08  |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。

2015年3月，[[中國人民政治協商會議全國委員會|全国政协]]委员[[言恭达|言恭达]]建议设立《中华人民共和国国歌法》。

2016年12月12日，[[张德江|张德江]]主持[[第十二届全国人民代表大会常务委员会|第十二届全国人民代表大会常务委员会]]第84次[[全国人民代表大会常务委员会委员长会议|委员长会议]]通过了《全国人大常委会2017年立法工作计划》，2017年4月11日第十二届全国人民代表大会常务委员会第93次委员长会议修改，2017年5月2日公开。该计划中将《中华人民共和国国歌法》列入“初次审议的法律案”中，并拟于2017年6月初次审议<ref>{{cite web|url=http://www.npc.gov.cn/npc/zgrdzz/2017-08/15/content_2026567.htm |title=全国人大常委会2017年立法工作计划 |publisher=中国人大网 |date=2017-08-15 |accessdate=2017-09-01}}</ref>。

2017年6月22日，由[[全国人大常委会法制工作委员会|全国人大常委会法制工作委员会]]拟定的《中华人民共和国国歌法（草案）<ref>[http://www.npc.gov.cn/npc/xinwen/2017-09/01/content_2028101.htm 关于《中华人民共和国国歌法（草案）》的说明]</ref>》首次提请第十二届全国人民代表大会常务委员会第28次会议审议。6月28日至7月27日期间，<ref>{{cite web|url=http://www.npc.gov.cn/npc/flcazqyj/2017-06/28/content_2026191.htm |title=中华人民共和国国歌法（草案）征求意见 |publisher=中国人大网 |date= |accessdate=2017-09-01}}</ref>《中华人民共和国国歌法（草案）》初审稿通过中国人大网向公众进行了首次意见征集。《中华人民共和国国歌法（草案）》初审稿全文共15个条款，在其内容中规范了对中华人民共和国国歌的奏唱、播放和使用，其内容<ref>{{Cite web |url =http://npc.people.com.cn/n1/2017/0622/c14576-29356272.html |title = 国歌法（草案）一审：私人丧事奏唱国歌或将违法 |author =  |date = 2017-06-22|publisher =  人民网|language =  |accessdate =  |quote =  }}</ref>包括了对国歌的地位、对奏唱国歌的场合、对国歌奏唱的形式和礼仪等内容。

2017年8月28日，<ref>[http://www.npc.gov.cn/npc/xinwen/2017-09/01/content_2028098.htm 全国人民代表大会法律委员会关于《中华人民共和国国歌法（草案）》审议结果的报告]</ref>《中华人民共和国国歌法（草案）》第二次提请第十二届全国人民代表大会常务委员会第29次会议审议。本次的审议，在原有的初审稿的基础上对草案的大部分内容进行了编修调整，使其内容更趋于合理化。在对《中华人民共和国国歌法》的历次草案的对比中，只有其第二条及第十二条是在首次制定后就未予变动的两个条款{{NoteTag|《中华人民共和国国歌法（草案）》在二审后其内容具备了颁布及施行的标准并已由全国人大常委会决议通过，因此《中华人民共和国国歌法》的正式颁布及施行的内容完全采用了二审稿草案内容，故而二审稿草案内容即为正式版本内容。}}。

2017年9月1日，<ref>{{Cite web|url=http://news.sina.com.cn/o/2017-09-01/doc-ifykpuui0257546.shtml|title=全国人大常委会表决通过国歌法 10月1日起施行|date=2017-09-01|work=新浪}}</ref>第十二届全国人民代表大会常务委员会第29次会议第三次全体会议表决通过《中华人民共和国国歌法》，并由[[中华人民共和国主席|中华人民共和国主席]][[习近平|习近平]]签署第75号[[中华人民共和国主席令|主席令]]颁布本法，确定自2017年10月1日起正式施行。

===中国内地===
《中华人民共和国国歌法》的完整全文主要针对[[中国内地|中国内地]]各省、直辖市、自治区依照地方实际情况制定有关行政法规予以具体实施。对于中国内地的各电视广播媒体则统一以执行<ref>[http://www.gov.cn/xinwen/2017-09/30/content_5228901.htm 新闻出版广电总局关于学习宣传贯彻《中华人民共和国国歌法》的通知]</ref>由[[国家新闻出版广电总局|国家新闻出版广电总局]]发布的《关于学习宣传贯彻〈中华人民共和国国歌法〉的通知》为准。

由国家新闻出版广电总局发布的《关于学习宣传贯彻〈中华人民共和国国歌法〉的通知》在针对地方台方面，大多数省级电视广播媒体实际上只有[[中国省会|省会]]和[[计划单列市|单列市]]广播电台、电视台的第一套频道播放国歌，而一些省市台第一套是非上星综合频道的（通常是第二套为上星综合频道的）也会在该规定时段播放国歌（如[[福建电视台综合频道|福建电视台综合频道]]、[[上海广播电视台新闻综合频道|上海广播电视台新闻综合频道]]、[[厦门电视台|厦门电视台]]一套、[[延边电视台|延边电视台]]一套、[[内蒙古电视台|内蒙古电视台]]新闻综合频道）。另外，一些呼号为“卫视”的外宣频道或方言频道亦有播放国歌（如[[南方卫视|南方卫视]]、[[厦门卫视|厦门卫视]]、[[三沙卫视|三沙卫视]]、[[海峡卫视|海峡卫视]]、[[延边卫视|延边卫视]]）；而民族语卫星频道则会播放带民族语字幕的版本（如内蒙古蒙语卫视、西藏藏语卫视）。

本法律正式施行后，原由[[中共中央办公厅|中共中央办公厅]]、[[国务院办公厅|国务院办公厅]]于2014年12月12日印发的《关于规范国歌奏唱礼仪的实施意见》实际上已失效。

=== 香港 ===
在香港立法前，[[全国人民代表大会常务委员会法制工作委员会|全国人大常委会法制工作委员会]]表示，《中华人民共和国国歌法》属于全国性法律，成法后会择机通过全国人大常委会的决定将其纳入[[香港特别行政区基本法|香港特别行政区基本法]]的附件三中，而使其适用[[香港|香港]]<ref>{{Cite news |url = http://www.bbc.com/zhongwen/simp/chinese-news-39801805 |title = 中国全国人大拟立《国歌法》剑指“港独”？ |last=Allen |first = Kerry |date=2017-05-04 |website = BBC中文网 |accessdate=2017-09-01 |language=zh-hans }}</ref><ref name="DWcontroversy">{{Cite web |url = http://www.dw.com/zh/%E5%9B%BD%E6%AD%8C%E6%B3%95%E8%BF%9B%E9%A6%99%E6%B8%AF-%E7%94%A8%E6%B3%95%E5%BE%8B%E5%BC%BA%E6%8E%A8%E7%88%B1%E5%9B%BD%E6%83%85%E6%80%80/a-40287995 |title=《国歌法》进香港 用法律强推爱国情怀？ |work=德国之声 |accessdate=2017-09-01 |author = Deutsche Welle |language=zh }}</ref><ref>{{Cite news |url = https://theinitium.com/roundtable/20170901-roundtable-hk-national-anthem-law/ |title = 在國旗法、國徽法後，香港制定國歌法是順理成章？ |author = 端小二 |work = 端傳媒 |accessdate=2017-09-01 |language=zh-HK }}</ref>。

就本地立法，香港的免费电视台主流频道（[[港台電視31、31A|港台電視31]]、[[翡翠台|翡翠台]]、[[香港開電視|香港開電視]]、[[ViuTV|ViuTV]]）亦会根据[[香港特别行政区政府|香港特区政府]]和[[通讯事务管理局|通讯事务管理局]]规定，在晚间18点档新闻时段开始前以《[[心系家国|心系家国]]》宣传片的形式播放国歌。此外，香港的一些收费电视台和卫星电视台（如[[有线新闻台|有线新闻台]]、[[凤凰卫视资讯台|凤凰卫视资讯台]]）也会在晚间的部分整点新闻时段开始前播放国歌。

2018年10月7日，[[虎牙直播|虎牙]]主播“莉哥”（本名[[杨凯莉|杨凯莉]]）在直播过程中，篡改国歌曲谱，以嬉皮笑脸的方式演唱国歌第一句，并将国歌作为自己“网络音乐会”的“开幕曲”。10月10日，虎牙发布通报，称其行为违反了《中华人民共和国国歌法》中奏唱国歌时应当举止庄重的相关规定。对此，虎牙平台决定即日起封禁主播“莉哥”直播间，冻结主播莉哥直播账号，下架全部相关影像作品，并对其进行整改教育<ref>{{cite news|title=网红直播时戏唱国歌 虎牙：已冻结账号并对其整改教育|url=http://www.bjnews.com.cn/news/2018/10/10/510039.html|date=2018-10-10|accessdate=2018-10-11|publisher=新京报 }}</ref>。10月13日，上海静安警方通报，已依法对杨凯莉处以行政拘留5日<ref>{{cite news|title=网红女主播“莉哥”直播中侮辱国歌 被行拘5日|url=http://ent.sina.com.cn/y/yneidi/2018-10-14/doc-ifxeuwws3994288.shtml|date=2018-10-14|accessdate=2018-10-14|publisher=上观  }}</ref>。
