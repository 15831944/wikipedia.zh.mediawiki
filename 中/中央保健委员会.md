'''中央保健委员会'''或'''中央保健局'''是[[中国共产党|中国共产党]][[中国共产党中央委员会|中央委员会]]中负责中共领导人保健的专门机构，办公室设在[[中华人民共和国国家卫生健康委员会|中华人民共和国国家卫生健康委员会]][[国家卫生健康委员会保健局|保健局]]，隶属于[[中共中央办公厅|中共中央办公厅]]。主要负责[[党和国家领导人|党和国家领导人]]的个人身体健康。负责人为中央保健委员会办公室主任（国家卫生健康委保健局局长）。<ref>{{cite web |author1=新华网 |title=中央保健局局长回忆：抢救国家领导人的经历 |url=http://news.ifeng.com/history/1/midang/200807/0721_2664_664125.shtml |website=凤凰网 |publisher=凤凰资讯 |accessdate=2018-12-02}}</ref>中央保健委员会在中共领导人办公地[[中南海|中南海]][[中共中央办公厅警卫局|中央警卫处]]设有保健科。<ref>{{cite web |author1=王力平 |title=保健医生回忆在刘少奇同志身边工作的日子 |url=http://cpc.people.com.cn/GB/64162/64172/85037/85038/6043584.html |website=中国共产党新闻 |accessdate=2018-12-02}}</ref>
