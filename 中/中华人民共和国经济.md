1978年，[[鄧小平|鄧小平]]推行[[改革開放|改革開放]]政策，將中國大陸經濟從舊有的緩慢、低效率的蘇聯計劃經濟體制轉為市場經濟為導向的經濟模式，但政府依然有某種程度上的主導地位。在這種改革的背景下，[[人民公社|人民公社]]經濟體制在1978年被取消，農村[[家庭聯產承包責任制|家庭聯產承包責任制]]逐漸取代了这一制度，土地由农民承包，并且农民可以自由在市场贩卖自己的农产品<ref>《剑桥中华人民共和国史》 下册 第四章 毛的接班人问题和毛主义的终结 邓小平的纲领</ref>。工商业上，1978年12月22日发表的[[中共十一届三中全会|中共十一届三中全会]]公报作为标志，强调下放更多的经营自主权<ref>《中共十一届三中全会公报》：“会议指出，现在我国经济管理体制的一个严重缺点是权力过于集中，应该有领导地大胆下放，让地方和工农业企业在国家统一计划的指导下有更多的经营管理自主权；”</ref>。1979年6月的全国人民代表大会[[国家计划委员会|国家计划委员会]]主任[[余秋里|余秋里]]宣布调整工业重心，从重工业转向农业和轻工，要求重视农业、轻工业和对外贸易的发展，将其作为经济首要的重点纺织品是最大的项目。1979年重工业的计划增长率调整为7.6％，比当年轻工业的8.3％的计划增长率要低。重工业的投资也开始下降，从1978年占总投资的54.7％压缩到1979年的46.8％，并重视出口。其结果是1979—1981年，出口增长率名义上是原来增长率以年均29%的速度增长，出口产品以纺织品是最大的项目，还有自行车、缝纫机等轻工业产品，另外还有一些矿产<ref>《剑桥中华人民共和国史》 下册 第六章 中国的经济政策及其贯彻情况 工业战略的变化（1977—1980年）</ref>，并在1981年贸易赤字消失和1982年出现了大量顺差<ref>《剑桥中华人民共和国史》 下册 第六章 中国的经济政策及其贯彻情况 加速工业增长（1982—1987年）</ref>。又因为中国能源开采总量已经停滞，中国经济开始转向单位耗能较少的消费品工业部门，并对高效能源设备进行投资。值得一提，1980年代中国开始大量向国外借款并吸纳投资，又利用旅游业赚取外汇，更多企业开始自组筹措资金和自主制定战略发展。

1980年代，中国基本处于农业和工业快速增长时期，工业结构开始从重工业向轻工业调整，出口大增，[[市场经济|市场经济]]开始萌芽，又长三角地、珠三角等地区开始进入工业化高速增长以及1985年中国开始贬值货币以利于出口。1981年到1986年的工业产值增长年均值为13.6%，工业产值为10300亿，比中国1981年翻倍。[[GDP|GDP]]年均增长率大约为9.75%。但是这个年代仍然存在不少问题，首先，大量商品价格依然为国家制定，而不是市场，也就是[[价格双轨制|价格双轨制]]，有关系者从中牟利，[[价格闯关|价格闯关]]的改革到1990年才宣告完成。其次，垄断依旧大量存在。再者，物价不稳，1980年代曾经出现过通货膨胀高企的时期，1987年大幅度增加固定资产投资以及启动价格闯关改革后,出现了大幅度物价上涨<ref name="王玉茹《中国经济史》P327">王玉茹《中国经济史》P327</ref>，1987年[[CPI|CPI]]上涨7.3%，1988年CPI上涨8.8%，1990年CPI上涨18%<ref>[http://money.163.com/special/00253L8T/econdata.html 网易中国各年经济数据 ]</ref>。

1989年[[六四事件|六四事件]]后，由于外国制裁，中国GDP增长率在1989和1990年大幅下跌，分别为4.1%和3.8%。在1991年开始反弹，为9.2%。1992年，[[邓小平南巡|邓小平南巡]]，中国继续改革开放政策，在1998年前继续延续1980年代政策，并在启动[[中国1994年税制改革|中国1994年税制改革]]。这次改革后中央政府税后大幅增加，但是地方财源开始枯竭，[[土地财政|土地财政]]开始成为地方重要财源，又这次改革后财政收入占国民生产总值的比例上升。值得一提的是，中国1993年到1994年曾出现过经济过热的情况，1994年CPI上涨达到24.1%<ref>[http://news.sohu.com/20080513/n256823035.shtml 专家：不会发生1994年那样的通胀]</ref>，创下1949年有记录以来新高<ref name="王玉茹《中国经济史》P327"/>。

===第三阶段（1998至今）===

1998年后中国政府调整经济增长战略，首先实行积极的财政政策以对抗[[亚洲金融危机|亚洲金融危机]]。其次，将资金投入到新技术和新产业研发。其三、中国政策国企大幅度改革，关停一批效益不佳的国企，但这个政策导致大量国企员工下岗。其四，住房改革，商品房开始大量进入市场，房贷大量出现<ref>王玉茹《中国经济史》P323</ref>。又中国政府再次开始向重化工行业投入资金。这段时期地区经济差距开始放大，一些内陆省份日渐被抛离<ref>王玉茹《中国经济史》P324</ref>。

2015年全国一般公共[[预算|预算]]收入152217亿元，其中税收收入124892亿元，非税收入27325亿元，全国一般公共[[预算|预算]]支出175768亿元；全国政府性基金预算收入42330亿元，支出42364亿元；全国国有资本经营预算收入2560亿元，支出2079亿元<ref>{{cite web|title=2015年财政收支情况|url=http://gks.mof.gov.cn/zhengfuxinxi/tongjishuju/201601/t20160129_1661457.html|website=中华人民共和国财政部|publisher=国库司|accessdate=2016-11-22|date=2016年1月29日}}</ref>。

1970年代，中国大陆实行改革开放之初，在缺少对外经济交往经验、国内法律体系不健全的形势下，设立经济特区为国内的进一步改革和开放、扩大对外经济交流起到了极为重要的作用。

1980年8月26日第五届全国人大常委会第十五次会议批准《广东省经济特区条例》，标志着经济特区正式成立的法律授权。设立经济特区是改革开放之初邓小平的改革开放思想的重要内容之一，经济特区作为对外交流的窗口，在一定时期内通过中央政府给予的政策优势和区位优势二者合力，经济以超出一般地区很多的速度成长，其人均经济实力接近或超过中等发达国家（或地区）的水平，后来更衍生了[[上海浦东新区|上海浦东新区]]、[[天津滨海新区|天津滨海新区]]和[[重庆|重庆]][[两江新区|两江新区]]。

2005年6月，[[国务院发展研究中心|国务院发展研究中心]]发布的《地区协调发展的战略和政策》报告提出，“[[十一五|十一五]]”期间内地经济划分为[[中国东部经济区|东部]]、[[中国东北地区|东北]]、[[中国中部经济区|中部]]和[[中国西部经济区|西部]]四大板块。[[国家统计局|国家统计局]]亦按此方法划分中国的经济区域。

2014年3月16日[[发改委|发改委]]发布《国家新型城镇化规划(2014-2020年)》，除以往的三大国家级都会區：[[珠三角城市群|珠三角都会區]]、[[长三角城市群|长三角都会區]]、以京津冀为核心的[[环渤海经济圈|环渤海经济圈]]之外，将[[长江中游城市群|长江中游城市群]]、[[中原城市群|中原都会區]]、[[成渝城市群|成渝都会區]]、[[哈长城市群|哈长都会區]]四大区域列入国家级城市群之列<ref>{{cite web | url=http://www.zyjjw.cn/news/henan/2014-04-16/158010.html | title=中原城市群突出重围 跻身七大国家级城市群" | accessdate=2014-04-21}}</ref>，使之成为推动国土空间均衡开发、引领区域经济发展的重要增长极。<ref>{{cite web | url=http://news.xinhuanet.com/house/wuxi/2014-03-17/c_119795674_5.htm | title=《国家新型城镇化规划(2014-2020年)》 | accessdate=2014-04-21}}</ref>

=== 第三產業 ===

現時中國正處於產業轉型，以第三產業替換舊有的第二產業經濟模式，改以第三產業帶動經濟成長，並自動化傳統的工業生產模式，以提升生產效率，滿足國內持續成長的第三產業對基礎工業的需求。大陸的第三產業在2012年首次超過第二產業，占國內生產總值最大比重，主導了中国的经济。中華人民共和國的軟體產業屬於第三產業，這領域最近十幾年發展迅速，規模從2000年的593億人民幣增長到2010年的18400億人民幣。複合年均增長率為36.65%。現時中國軟體產業首要面對的挑戰是填補自身的核心工業軟體領域的缺陷，例如對[[程式語言|程式語言]]、[[系統軟體|系統軟體]]、[[電子設計自動化|電子設計自動化軟體]]、[[電子設計自動化|計算機輔助設計軟體]]、[[電子設計自動化|計算機輔助工程軟體]]的研發，減少依賴外國工業軟體，去降低外部壓力造成的生產風險，以滿足未來中國在複雜工業的自動化控制需求。

=== 第一產業 ===
{{main|中华人民共和国农业}}

中国是农业生产大国。2006年农业总产值为2.5万亿元人民币，占中国GDP的12.5%。中国人口的40%从事农业生产。2006年中国农业出口额为2480 亿元人民币，比上年增长14.1 %。中国主要的农产品出口市场为日本、欧盟、韩国、美国、香港和东盟国家<ref>[[澳大利亚|澳大利亚]]驻华大使馆[http://china.embassy.gov.au/bjngchinese/FTAagri.html 农业概览]</ref>。2013年中国农林牧渔业总产值为9.7万亿元人民币<ref>[http://tongji.cnki.net/kns55/Publish/industry/Z009.html 中国农业与经济社会发展统计数据库]</ref>。

中华人民共和国农业也称[[第一产业|第一产业]]，包括[[种植业|种植业]]、[[林业|林业]]、[[畜牧业|畜牧业]]、[[渔业|渔业]]。其中种植业主要分布在中国东部湿润和半湿润的平原地区，80年代起大量利用[[地膜|地膜]]技術增產。主要產品有[[稻米|稻米]]、[[小麥|小麥]]、[[高粱|高粱]]、[[花生|花生]]、[[茶|茶]]、[[大麥|大麥]]、[[玉米|玉米]]、[[棉花|棉花]]、油料作物等。林業主要分布在[[中国东北|中国东北]]和西南的天然林区，以及东南部的人工林区。畜牧業主要分布在中国西部新疆、青海、西藏、内蒙古等地区。漁業主要分布在中国东部地区。

==== 重新開發頁岩油 ====

中國[[頁岩油|頁岩油]]儲量為7,200億噸，分布在47個油頁岩盆地的80個礦床里。 據估算，中國頁岩油實際儲量甚至超過美國。中國的油頁岩工業早在1920年代就已建立。1960年代後，中國頁岩油產量下降，這種狀況直到2008年才逆轉。2005年，中國成為世界最大的頁岩油生產國，但後來被美国超越。2011年，中國的頁岩油產量約為650,000噸。

中国重要的油页岩企业是抚顺矿业集团。另有多家煤炭和石油公司扩大了在油页岩抽提方面的生产能力。2011年，中国有7家页岩干馏厂，每年生产页岩油 650,000吨。这些企业位于抚顺、桦甸、吉林[[汪清县|汪清县]]、辽宁 [[北票|北票]]、龙口和[[黑龙江省|黑龙江省]][[东宁县|东宁县]]。中国绝大多数油页岩企业采用抚顺式干馏炉或改装的抚顺式干馏炉。抚顺矿业集团拥有36亿吨高品质油页岩地质储量，可开采储量为9.2亿吨，分为东西两个露天矿井，储量分别为7.6亿吨和1.6亿吨。生产活动在西矿井进行，东矿井尚未开采。抚顺的页岩油生产成本为每桶18.46美元，其中采矿成本为0.184美元，运输成本4.25美元，干馏成本13.84美元。

2008年8月，中石油下属的大庆石油公司在[[黑龙江省|黑龙江省]][[牡丹江市|牡丹江市]]开建一座页岩油炼厂，该工厂设计年油页岩处理量为120万吨，产出页岩油10万吨。

=== 長期 ===

==== 全面建成小康社會 ====

中國共產黨第十九次全國代表大會提到，中國要在本世紀初「基本建成小康社會」的基礎上，到2020年「全面建成小康社會」，從現在到2020年，是全面建成小康社會「決勝期」。再到2035年「基本實現社會主義現代化」，最終到2050年「全面建成富強、民主、文明、和諧、美麗的社會主義現代化強國」。十九大報告還明確確保到2020年中國現行標準下農村貧困人口實現脫貧，解決區域性整體貧困。

==== 民营商业航天 ====

[[小行星采矿|地外礦業]]將是未来外太空商业化的新興產業，預計会带來龐大利益。現時已知最小的越地小行星 3554 Amun 是一块两公里宽的，含有铁、镍、钴、铂和其他金属的金属块；它含有的金属是人类历史上挖掘的金属的三十倍，虽然它只是仅有的几十个已知的金属小行星中最小的，但如果把它开采起来，以满足2001年金属市场，它的价值会高达120万亿元人民币。現時地外採礦的相关技术已经將近成熟，日本的[[隼鸟号|隼鸟号]]已经于2010年6月13日携带糸川小行星的表面样本返回地球，如今继任的[[隼鸟2号|隼鸟2号]]和美国太空總署的[[OSIRIS-REx|OSIRIS-REx]]探测器都已经达到各自的目标小行星附近。據估算，将一颗直径10米含铂金的小行星带回地球，大概需要50亿人民币。

== 可能的危机 ==
{{see also|中國即將崩潰}}
[[File:The_Coming_Collapse_of_China_ver._CHT_2002.jpg|thumb]]著《[[中國即將崩潰|中國即將崩潰]]》中文版]]
中国经济数据以GDP为首多次遭到外界质疑是否造假，例如质疑中国GDP统计时通过物价指数的计算上调整从而拉高数据<ref>[http://cn.nikkei.com/columnviewpoint/kelongcolumn/16514-20151019.html 日经中文网-2015/10/19-中国的宏观经济统计不可信吗?]</ref>，又或者通过通胀报低调整GDP平减指数达到拉高GDP增速数据的目标<ref>[http://www.ftchinese.com/story/001064212 中国经济统计数据可信吗？]</ref>。新华社曾发文反击称：“（中国）GDP实际增速仅高估不到0.5个百分点”<ref>[http://business.sohu.com/20150715/n416823992.shtml 新华网驳斥“中国GDP被高估”：不到0.5个百分点]</ref>。

[[中国国家统计局|中国国家统计局]]和相关部门曾多次发现并处罚[[地方政府|地方政府]]经济数据造假。同时，因为国家统计局有自己的直接收集数据的系统，国家统计局表示，中国的统计数据和制度不会地方数据的真实性问题而受到影响。<ref>{{cite web |url = http://www.sohu.com/a/220020710_157078 |title = 天津GDP增速全国倒数第1 “新一线大哥”怎么了 |publisher =  |date =  |accessdate = }}</ref>

新华网2014年2月14日发文，称“[[湖南|湖南]]基层统计造假透视：数据虚涨百倍 造假触目惊心”，人民日报对此事评论是《基层统计造假实为得了GDP职业病》<ref>[http://news.xinhuanet.com/politics/2015-02/14/c_1114370350.htm 新华网-2015年02月14日-湖南基层统计造假透视：数据虚涨百倍 造假触目惊心]</ref>。又2015年中央纪委监察部、国家审计署发现东北经济数据造假严重，[[吉林省|吉林省]]人大财政经济委员会主任委员赵振起认为：“倘若依照各地汇报的产业成长性计算，东北一些县域经济规模都超过香港了。”<ref>[http://society.people.com.cn/n/2015/1211/c136657-27913293.html 人民网-2015年12月11日-东北三省经济数据造假惊动中央巡视组]</ref>

[[遼寧省|遼寧省]]省長陳求發2017年1月第12屆人大8次會議中公開坦承，2011至2014年間該省的財政收入被虛報了至少20%，其中2014年虛報最高，達23%。<ref>[http://news.ltn.com.tw/news/business/paper/1082015 遼寧GDP暴跌23% 凸顯中國數據嚴重造假]</ref>

2017年6月[[中紀委|中紀委]]網站公布[[內蒙古|內蒙古]]和[[吉林省|吉林省]]有地方經濟資料造假，而中國鐵路總公司有「黑中介」等問題，此次繼遼寧省承認虛報經濟數據及財政收入後，再有省份被揭造假。<ref>[https://thestandnews.com/china/%E4%B8%AD%E7%B4%80%E5%A7%94%E6%8C%87%E5%85%A7%E8%92%99%E5%8F%A4%E5%90%89%E6%9E%97-%E5%9C%B0%E6%96%B9%E7%B6%93%E6%BF%9F%E8%B3%87%E6%96%99%E6%B6%89%E9%80%A0%E5%81%87/ 中紀委爆內蒙古吉林經濟數據造假]</ref>

2018年天津的[[濱海新區|濱海新區]]2016年的GDP造假虛報，總量下修1/3從1萬億元（人民幣）大幅度調整為6654億元（人民幣）。<ref>[https://www.hk01.com/%E5%85%A9%E5%B2%B8/149261/%E5%A4%A9%E6%B4%A5%E6%BF%B1%E6%B5%B7%E6%96%B0%E5%8D%80GDP%E6%B3%A8%E6%B0%B41-3-%E5%85%A7%E8%92%99%E8%B2%A1%E6%94%BF%E6%94%B6%E5%85%A5%E8%85%B0%E6%96%AC-%E5%85%A7%E5%9C%B0%E7%B6%93%E6%BF%9F%E6%95%B8%E6%93%9A%E6%B0%B4%E4%BB%BD%E5%A4%A7 天津濱海新區GDP注水1/3、內蒙財政收入腰斬　內地經濟數據水份大]</ref>但后经查证并非为虚报数据，实际为相关统计口径改变为所属地，天津全市总体GDP未受影响。
