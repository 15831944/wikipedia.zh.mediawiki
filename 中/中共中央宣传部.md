'''中国共产党中央委员会宣传部'''{{NoteTag|全名见于其公章。}}，通称'''中共中央宣传部'''，简称'''中央宣传部'''、'''中宣部'''，是[[中国共产党中央委员会|中国共产党中央委员会]]主管宣传思想和社情舆论工作的[[中共中央直屬機構|职能部门]]。

== 沿革 ==
中宣部前身1921年[[中共一大|中共一大]]後根據共產國際直接指導幫助下制定的《中國共產黨第一個綱領》模仿蘇共{{link-ru|中央宣传鼓动部|Агитпропа}}（{{lang|ru|Агитпропа}}）成立之「中央宣傳局」。<ref>[http://cpc.people.com.cn/GB/64184/64186/66627/66627/4489306.html 中國共產黨第一個綱領]</ref><ref>[http://cpc.people.com.cn/GB/64162/64168/64553/4427950.html 中国共产党中央局通告—关于建立与发展党、团、工会组织及宣传工作等]</ref><ref>[http://media.people.com.cn/GB/22100/41466/41467/3027940.html 新闻宣传异同论]</ref>1924年5月《黨內組織及宣傳教育問題議決案》在中央宣傳局架構內正式成立中央宣傳部（另同時設立中央組織部與中央工農部）。<ref>[http://cpc.people.com.cn/GB/64184/64186/66627/66627/4489257.html 《党内组织及宣传教育问题议决案》]</ref>1925年[[中共四大|中共四大]]《對於宣傳工作之議決案》規定：“為使宣傳工作做得完美而有系統起見，中央應有一強固的宣傳部負責進行各事，並指導各地方宣傳部與之發生密切且有系統的關係”、“共產國際關於宣傳工作議決案，本黨有盡可能地使之實施的必要”。<ref>[http://cpc.people.com.cn/BIG5/64162/64168/64556/4428265.html 對於宣傳工作之議決案]</ref>1928年10月中央要求建立與健全中央宣傳部的組織；<ref>[http://news.xinhuanet.com/ziliao/2002-03/04/content_696164.htm 1928年7月10日中共六大党章]</ref>各省委要立即建立宣傳部；各縣市區委應設立宣傳科；黨支部要有宣傳幹事，負責宣傳工作。<ref>[http://cpc.people.com.cn/GB/64184/64186/66631/4489524.html 中央通告第四号——关于宣传鼓动工作]</ref><ref>[http://www.ccps.gov.cn/cadre_education/lsyj/201207/t20120727_21631.html 中国共产党民主革命时期宣传工作思想述论]</ref><ref>[http://cpc.people.com.cn/GB/64184/64186/66636/4489681.html 中央特别通告——关于组织兵士暴动问题]</ref><ref>[http://cpc.people.com.cn/GB/64184/64186/66634/4489591.html 中央通告第六十号——执行武装保护苏联的实际策略]</ref><ref>[http://news.xinhuanet.com/ziliao/2003-01/19/content_696147.htm 中国共产党第三次修正章程决案第十九条]</ref>

1941年《關於統一各根據地內對外宣傳的指示》中規定“一切對外宣傳的領導應統一於宣傳部”，<ref>[http://cpc.people.com.cn/GB/64184/64186/66644/4490554.html 《關於統一各根據地內對外宣傳的指示》]</ref><ref>[http://news.xinhuanet.com/ziliao/2003-01/19/content_696080.htm 对于组织问题之议决案]</ref><ref>[http://news.xinhuanet.com/ziliao/2004-11/25/content_2259212.htm 党的组织问题议决案]</ref>並建立了中宣部領導、審查報紙雜誌的制度<ref>[http://cpc.people.com.cn/GB/64184/64186/66644/4490542.html 《中央宣传部关于各抗日根据地报纸杂志的指示》]</ref>《中央宣传部关于党的宣传鼓动工作提纲》指出“一切理论、主张、教育、文化、文艺等等均属于宣传鼓动活动的范围”、“印刷业、无线电及电影是宣传鼓动的有力工具”。1943年以後，中宣部負責在文藝工作和新聞工作中執行黨的政策<ref>[http://cpc.people.com.cn/GB/64184/64186/66645/4490704.html 中央宣传部关于执行党的文艺政策的决定]</ref>。1946年，中宣部在《关于广播、报纸宣传方针的通知》对正面宣传提出了要求。<ref>[http://cpc.people.com.cn/GB/64184/64186/66648/4491295.html 中央宣传部关于广播、报纸宣传方针的通知]</ref>建國前，中宣部管理文化教育工作。建國後，在中央宣傳部領導下，組織了[[中央廣播事業管理處|中央廣播事業管理處]]、中央出版委員會、中央電影管理局等機構。<ref>[http://news.xinhuanet.com/ziliao/2004-12/15/content_2336734.htm 《关于中央人民政府成立后党的文化教育工作问题的指示》]</ref>1949年12月中宣部《關於今後各級黨報名義不要公開宣傳的指示》下令各級黨報“都不必公開宣傳是中共機關報，也不必說是政府機關報，只稱某地某報即可。”

1962年秋，中宣部开始取消文艺单位党委制，[[徐光霄|徐光霄]]在[[中华人民共和国文化部|中华人民共和国文化部]]坚决貢彻执行，不少单位取消了[[党委|党委]]制。1962年，[[姚文元|姚文元]]批《[[海瑞罷官|海瑞罷官]]》的文章受到以[[彭真|彭真]]為首的[[中共北京市委|中共北京市委]]和以[[陸定一|陸定一]]為首的中宣部的抵制。1966年，[[毛澤東|毛澤東]]指出，中宣部是“閻王殿”，要“打倒閻王，解放小鬼”，批評彭真、中宣部和北京市委包庇壞人、壓制[[左派和右派|左派]]、不准革命；還說，如果再包庇壞人，中宣部要解散，北京市委要解散，“五人小組”要解散。<ref>[http://cpc.people.com.cn/GB/69112/70190/70194/5206586.html 点燃“文化大革命”的三把火]</ref>[[文革|文革]]後不久中宣部被取消，根据1966年5月中共中央[[五一六通知|五一六通知]]職權由[[中央文化革命小組|中央文化革命小組]]兼。<ref>1967年5月18日人民日报《伟大的历史文件》：“我们伟大领袖毛主席正是总结了国际无产阶级专政的历史经验，发动亿万群众，进行这场史无前例的无产阶级文化大革命，这是使我们党和国家永不变色的最可靠的保证。这是毛泽东同志在理论上和实践上对国际无产阶级的最大的贡献。在这个文件中，宣布撤销原来由彭真反革命修正主义集团包办的“文化革命五人小组”，重新设立中央文化革命小组，隶属于中央政治局常委之下。这是进行无产阶级文化大革命的一个重要措施。”</ref>《五一六通知》附件中称：“中宣部是阎王殿”、“要打倒阎王解放小鬼”，“中宣部要解散”。中宣部部长陆定一被诬陷为“反革命分子”。中宣部九名副部长分别被诬陷为“叛徒”、“特务”、“国民党分子”。中宣部的部长、副部长、秘书长们首批被揪出来了，被称为阎王殿的阎王。中央政治局扩大会议作出《关于停止和撤销彭真、陆定一、罗瑞卿和杨尚昆四同志职务问题的决定》，并下达了《{{s|中共中央关于陆定一同志和杨尚昆同志错误问题的说明|关于陆定一同志和杨尚昆同志错误问题的说明}}》。调[[陶铸|陶铸]]担任中央书记处常务书记，并兼任中宣部部长。副部长[[许立群|许立群]]、[[姚溱|姚溱]]、[[林默涵|林默涵]]、秘书长[[童大林|童大林]]4人停职反省。6月23日，以陶铸为组长的中共中央宣传部文化革命小组成立。陶铸在中宣部全体工作人员大会上宣布《中央关于改组中宣部领导的决定》，并宣布取消原有的处室机构改设4处1室，即：毛泽东思想宣传处、党员干部教育处、干部管理处、秘书处、调查研究室。1968年7月27日，中共中央决定对原中宣部实行[[军事管制|军事管制]]，任命[[李虓|李虓]]为军事管制小组组长、[[王少平|王少平]]为副组长。继中央宣传部首先被当作“阎王殿”砸烂后，各级党政宣传、出版、文化部门也都被“砸烂”，然后是被“军管”。<ref>《中国共产党组织史资料: "文化大革命"时期》，第六卷，中共党史出版社，2000</ref>

在1977年10月召開之[[中國共產黨第十一次全國代表大會|十一大]]批准《關於成立中央宣傳部的報告》，恢復中宣部職權，任命[[張平化|張平化]]為部長，其辦事機構為一室五局:辦公室、理論局、宣教局、文化藝術局、新聞局、出版局<ref>[http://www.people.com.cn/GB/32306/32313/32330/3982251.html 中宣部宣教局网上系列座谈会盘点]</ref>

根据2018年3月21日[[中共中央|中共中央]]印发的《深化党和国家机构改革方案》中，原[[国家新闻出版广电总局|国家新闻出版广电总局]]的新闻出版管理职责、电影管理职责划入中共中央宣传部管理，加挂[[国家新闻出版署|国家新闻出版署]]（[[国家版权局|国家版权局]]）、[[国家电影局|国家电影局]]牌子<ref>{{Cite web |url =  http://news.ifeng.com/a/20180321/56928639_0.shtml|title = 中央宣传部统一管理新闻出版、电影工作|author =  |date = 2018-03-21 |publisher = 新华通讯社|language =  |accessdate =  |quote =  }}</ref>。

== 职责 ==
根据《{{s|中国共产党中央委员会宣传部职能配置、内设机构和人员编制规定}}》，中共中央宣传部承担下列职能：
{{columns-list|2|
# 负责指导全国马克思主义理论的研究、学习和宣传；
# 负责引导社会舆论，指导协调中央的各新闻媒体做好新闻宣传工作，搞好舆论引导；
# 从宏观上指导精神文化产品的创作和生产；
# 规划和部署全局性的思想政治工作的任务；
# 受中央的委托，协同和会同有关部门对我们宣传文化系统的重要岗位的领导干部进行管理。联系宣传文化系统的知识分子，协助有关部门做好知识分子的工作；
# 负责提出宣传文化事业发展的指导方针。指导宣传文化系统制定政策和法规；同时还要按照中央的统一工作部署，做好宣传文化系统各有关部门之间的协调工作；
# 为中央领导和中宣部领导的决策和指导全局工作提供舆情信息的服务，并且要负责组织协调和指导宣传文化系统的舆情信息工作；
# 负责文化体制改革，包括新闻出版、广播电视业的改革和发展的调研，提出政策性的建议。
