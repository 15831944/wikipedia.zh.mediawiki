'''中国探月工程'''（{{Lang-en|'''China Lunar Exploration Project'''}} 或 {{Lang|en|'''Chinese Lunar Exploration Program'''}}，[[縮寫|縮寫]]：'''{{Lang|en|CLEP}}'''）亦稱'''嫦娥工程'''，是[[中华人民共和国|中国]][[国家航天局|国家航天局]]启动的第一个[[探月|探月]]工程，于2003年3月1日正式启动。首先是发射绕月[[卫星|卫星]]，继而是发射无人探测装置，实现月面软着陆探测，最后為運輸[[机器人|机器人]]上[[月球|月球]]建立观测点，并且採取[[樣本|樣本]]返回[[地球|地球]]；整个计划将會历时20年<ref>{{cite newspaper |url = http://szsb.sznews.com/html/2010-10/01/content_1255289.htm |title = 探月工程二十年 |publisher = [[深圳商报|深圳商报]] |date = 2010-10-01 |language = zh-cn |deadurl = yes |archiveurl = https://web.archive.org/web/20101211165744/http://szsb.sznews.com/html/2010-10/01/content_1255289.htm |archivedate = 2010-12-11 }}</ref>，並為載人登月的長期目標作出準備。中国探月工程的科研成果发布在国家天文台月球与深空探测研究部中国探月工程地面应用系统网站上。<ref>{{cite news|publisher=中国国家天文台|author=中国国家天文台月球与深空探测部|url=http://moon.bao.ac.cn/|title=中国探月工程地面应用系统|accessdate=2016-06-05|newspaper=中国国家天文台|archive-url=https://web.archive.org/web/20160617175751/http://moon.bao.ac.cn/|archive-date=2016-06-17|dead-url=no}}</ref>

== 工程规划 ==

中国嫦娥工程整体可以分为“探”、“登”、“驻”三大步骤，概称“大三步”（“三部曲”），分别指无人探月、载人登月、长久驻月。其中探月部分便是目前实施的“中国探月工程”或称“嫦娥工程”，下分为“绕”、“落”、“回”三个期段，统称“小三步”（“三步走”）。<ref>{{Cite web|url=http://www.chinanews.com/mil/2013/12-01/5565595.shtml|title=中国嫦娥工程的“大三步”和“小三步”|language=zh-cn|access-date=2014-02-15|archive-url=https://web.archive.org/web/20140222024020/http://www.chinanews.com/mil/2013/12-01/5565595.shtml|archive-date=2014-02-22|dead-url=no}}</ref><ref>{{Cite web|url=http://www.cnsa.gov.cn/n615708/n676979/n2235653/n2235659/167489.html|title=中国探月三部曲|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20140222205826/http://www.cnsa.gov.cn/n615708/n676979/n2235653/n2235659/167489.html|archivedate=2014-02-22}}</ref>

* 一期工程“绕”：（[[嫦娥一号|嫦娥一号]]、[[嫦娥二号|二号]]），2007年启动，发射月球轨道器/硬着陆器，在距离月球表面兩千公里的高度绕月飞行，进行月球全球探测。
* 二期工程“落”：（[[嫦娥三号|嫦娥三号]]/[[玉兔号月球车|玉兔号]]、[[嫦娥四号|四号]]/[[玉兔二号月球车|玉兔二号]]{{notetag|嫦娥四号任务通常被认为是嫦娥工程二期的任务之一，但在2019年1月14日的探月工程嫦娥四号新闻发布会上，[[国家航天局|国家航天局]]副局长、探月工程副总指挥[[吴艳华|吴艳华]]提到嫦娥四号是探月四期的首次任务<ref>{{cite news |title=探月无止境 “嫦娥”继续飞——国家航天局有关负责人详解探月工程嫦娥四号任务 |url=http://www.xinhuanet.com//tech/2019-01/15/c_1123989798.htm |accessdate=2019-01-16 |work=[[新华网|新华网]] |date=2019-01-15}}</ref><ref>{{cite web |title=探月工程嫦娥四号新闻发布会 |url=http://www.cnsa.gov.cn/n6758967/n6758969/n6760061/index.html |publisher=[[国务院新闻办公室|国务院新闻办公室]] |accessdate=2019-01-16 |date=2019-01-14}}</ref>。}}），2013年启动，发射月球软着陆器/[[月球车|巡视器]]，降落到月球表面，释放一个月球车，进行着陆区附近局部详细探测，着陆器还将携带天文望远镜，从月亮上观测星空。
* 三期工程“回”：（[[嫦娥五号T1|嫦娥五号探路星]]、[[嫦娥五号|嫦娥五号]]），2019年启动，发射月球自动采样返回器，降落到月球表面后，机械手将采集月球土壤和岩石样品送上返回器，返回器再将月球样品带回地球，开展相关研究。<ref>{{cite web|url=http://www.kepu.net.cn/gb/special/200912_02_tygc/wz/02.html|title=嫦娥工程|publisher=中国科普博览|language=zh-hans|access-date=2012-06-15|archive-url=https://web.archive.org/web/20131010104830/http://www.kepu.net.cn/gb/special/200912_02_tygc/wz/02.html|archive-date=2013-10-10|dead-url=no}}</ref>
* 四期工程：（{{le|嫦娥六号|Chang'e 6}}、[[嫦娥七号|嫦娥七号]]、[[嫦娥八号|嫦娥八号]]）指“三步走”完成后后续对月球的科研活动，包括对月球南极的采样返回、月面试验等。<ref>{{cite news |title=国家航天局首次公开：嫦娥六号、七号、八号来了 |url=http://www.bjnews.com.cn/news/2019/01/14/539311.html |accessdate=2019-01-14 |work=[[新京报|新京报]] |date=2019-01-14}}</ref>

=== 中国载人登月工程 ===

中国载人登月工程即“大三步”中的第二步。将根据嫦娥三期工程的进展而逐步推启。若成功立项，将会在[[2030年代|2030年代]]实现中国航天员登月<ref>{{cite web|url=http://news.sina.com.cn/c/2013-11-30/185828856179.shtml|title=载人航天顾问组长:神十一或在2016年5月发射|publisher=新浪网引自《海西晨报》|date=2013年11月30日|language=zh-cn|author=陈航飞|access-date=2014-01-23|archive-url=https://web.archive.org/web/20140202183334/http://news.sina.com.cn/c/2013-11-30/185828856179.shtml|archive-date=2014-02-02|dead-url=no}}</ref>。2012年6月14日，中国探月工程首席科学家[[欧阳自远|欧阳自远]]院士在中科院第16次院士大会上表示，至2017年中国将會基本完成不载人登月探测任务，之后将择机开展载人登月探测等<ref>{{cite web|url=http://news.xinhuanet.com/politics/2012-06/14/c_112217044.htm|title=欧阳自远：嫦娥三号明年发射将实现着陆器与月球车联合探测|publisher=[[新华网|新华网]]|date=2012年6月14日|language=zh-hans|author=吴晶晶、刘奕湛|access-date=2012-06-15|archive-url=https://web.archive.org/web/20120617020908/http://news.xinhuanet.com/politics/2012-06/14/c_112217044.htm|archive-date=2012-06-17|dead-url=no}}</ref>。中国载人登月工程截止2018年还处于研究阶段，未到决策阶段<ref>{{cite news |title=嫦娥五号年内发射 载人登月还在研究 |url=http://www.stdaily.com/index/kejixinwen/2019-01/14/content_746623.shtml |accessdate=2019-01-14 |work=[[中国科技网|中国科技网]] |date=2019-01-14}}</ref>。目前几种方案包含两次发射载货的[[长征五号|长征五号登月运载火箭]]加一次发射载人的[[长征二号F运载火箭|长征二号F运载火箭]]，或两次发射[[长征九号运载火箭|长征九号运载火箭]]（一枚载货、一枚载人）。

== 目标 ==
绕月探测工程的科学目标：
# 获取月球表面三维立体影像。划分月球地形和地貌单元；统计撞击坑大小与密度，计算月球表面年龄，恢复月球早期演化历史；分析月面构造，编制月面断裂和环形构造影像和月球构造区划图，研究月球地质构造的演化历史。
# 分析月球表面元素含量和物质类型的分布特点。绘制相关元素的全月球含量与分布图、粗略划分岩石类型，研究月球的化学演化。
# 探测月壤特性。测量全月球不同波段的微波辐射亮度温度，反演月壤厚度的信息，评估月壤中[[氦-3|氦-3]]的资源量与分布。
# 探测地月空间环境。探测地月空间高能粒子和太阳风离子的成分、通量、能谱及其随时空的变化特征，研究太阳活动和地球磁层对近月空间环境的影响。
