==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，[[亞美尼亞|亞美尼亞]]當時屬於[[蘇聯|蘇聯]]的[[加盟共和國|加盟共和國]]（[[亞美尼亞蘇維埃社會主義共和國|亞美尼亞蘇維埃社會主義共和國]]），由蘇聯作為[[主權國家|主權國家]]參與投票。]]
{{see also|中華民國外交|{{link-en|亞美尼亞外交|Foreign relations of Armenia}}}}
根據統計，超過6,000名[[亞美尼亞籍|亞美尼亞籍]]的[[蘇聯紅軍|蘇聯紅軍]]在[[第二次世界大戰|第二次世界大戰]]時來到[[中國東北地區|中國東北地區]]，參與[[中國抗日戰爭|中國抗日戰爭]]。<ref>{{Cite web |url=http://www.chinesetoday.com/big/article/1036596 |author=赫佰靈 |date=2015年9月1日 |title=亞美尼亞89歲老人的中國抗戰情結 |publisher=《{{link-en|國際日報|International Daily News}}》 |deadurl=no |archiveurl=https://archive.is/20150920013349/http://www.chinesetoday.com/big/article/1036596 |archivedate=2015年9月20日 }}</ref>

1999年5月23日，亞美尼亞知識份子聯盟總裁畢里波亞揚（Piliposyan）抵台參加[[行政院青年輔導委員會|行政院青年輔導委員會]]舉辦的第一屆全球青年領袖會議。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter1-2.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2011年7月12日，簽訂《[[中華民國法務部調查局|中華民國法務部調查局]]洗錢防制處與[[亞美尼亞共和國中央銀行|亞美尼亞共和國中央銀行]]金融監測中心關於[[洗錢|洗錢]]及資助[[恐怖主义|恐怖分子]]相關{{link-en|金融情報|Financial intelligence|金融情資}}交換合作瞭解[[備忘錄|備忘錄]]》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e4%ba%9e%e7%be%8e%e5%b0%bc%e4%ba%9e&tycountry_e=%e4%ba%9e%e7%be%8e%e5%b0%bc%e4%ba%9e&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

2015年12月，[[2015年聯合國氣候變化大會|第21次聯合國氣候變遷大會]]（COP 21）第八次部長談判會議中，亞美尼亞代表因提及[[台灣|台灣]]而遭[[中華人民共和國|中華人民共和國]]代表當場稱此說法「不恰當」，並再次重申在[[聯合國氣候變化綱要公約|聯合國氣候變遷綱要公約]]的架構下須嚴守「[[一個中國原則|一個中國原則]]」。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/WebArchive/2153/%E4%B8%AD%E5%9C%8B%E5%A4%A7%E9%99%B8%E9%98%BB%E6%92%93%E6%88%91%E5%9C%8B%E9%9A%9B%E7%A9%BA%E9%96%93%E4%BA%8B%E4%BE%8B%EF%BC%88104%E5%B9%B4%EF%BC%89.pdf |author = 公眾外交協調會 |title = 中國大陸阻撓我國際空間事例（104年事例）|publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]／[[簽證#电子签证|電子簽證]]的方式入境亞美尼亞。]]
[[File:Visa_requirements_Armenian_citizens.png|thumb]]必須申請[[簽證|簽證]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|亞美尼亞簽證政策|亞美尼亞公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以在首都[[葉里溫|葉里溫]]的[[茲瓦爾特諾茨國際機場|茲瓦爾特諾茨國際機場]]辦理[[簽證#落地签证|落地簽證]]入境亞美尼亞，停留最多120天；亦可在入境前，至亞美尼亞[[簽證#电子签证|電子簽證]]網頁申辦簽證。在亞美尼亞機場轉機，若有第三國的機票或簽證，可申請[[過境簽證|過境簽證]]在境內停留72小時（可申請延長至4天）。若無入境亞美尼亞，則可在轉機區停留48小時。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-143-c29a3-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=7F220D7E656BE749 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|亞美尼亞護照|Armenian passport}}的[[亞美尼亞公民|亞美尼亞公民]]抵台參加國際會議、運動賽事、商展活動，可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將亞美尼亞按地區歸類為'''灰色警示'''：提醒注意，與'''橙色警戒'''：避免非必要旅行（以上截至2017年12月19日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-126-6ba61-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2013年，[[經濟部國際貿易局|經濟部國際貿易局]]挑選最具拓銷潛力的22國26城市推動執行「貿易尖兵任務」，其中[[東歐|東歐]]7國的[[新興市場|新興市場]]就包括[[亞美尼亞|亞美尼亞]]。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622045&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2013年10月30日 |title = 貿易尖兵勇闖22國26城市 新興市場商機無窮 |publisher = 經貿透視網}}</ref>

2018年，亞美尼亞是中華民國第161大貿易夥伴、第148大[[進口|進口]]夥伴、第161大[[出口贸易|出口]]夥伴。出口至亞美尼亞的金額為226萬4,618[[美元|美元]]，年增25.918%；自亞美尼亞進口的金額為148萬4,686美元，年增32.130%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）77萬9,932美元，年增15.574%。2017年，亞美尼亞是中華民國第172大貿易夥伴、第154大進口夥伴、第164大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
