'''《中国地》'''，2011年上映于[[中国大陆|中国大陆]]的[[抗日战争|抗日战争]]题材剧，改编自[[赵冬苓|赵冬苓]]同名小说，讲述了[[东北|东北]]广大人群抗日救国的可歌可泣的历史事迹。

== 剧情 ==
1931年9月18日，[[大日本帝国|大日本帝国]]悍然发动[[九一八事变|九一八事变]]，开始了对[[中华民国|中华民国]]的疯狂侵略战争，面对来势汹汹的[[关东军|关东军]]，[[张学良|张学良]]的[[东北军|东北军]]大败逃遁，关东军开始由[[城市|城市]]分散到[[农村|农村]]屠杀中国人民，大日本帝国扶持[[溥仪|溥仪]]建立了[[满洲国|满洲国]]，奴役中华民国国民，在[[奉天省|奉天省]]和[[热河特别区|热河特别区]]的地线（今辽宁省朝阳市[[清风岭|清风岭]]地区），朴实的农民赵天龙带领全家人口积极[[抗日救亡运动|抗日救亡运动]]，并且守护家园长达14年没有被日军侵占，后加入[[中国共产党|中国共产党]]。<ref name=" bull" >{{cite web |url=http://amuse.nen.com.cn/system/2012/09/25/010070753.shtml |title=作品简介：电视剧《中国地》 |accessdate=  2012-09-25 |publisher=东北新闻网 |work= |language =zh-cn}}</ref>

== 媒体评价 ==
《[[南方日报|南方日报]]》专文『《中国地》抗战“非典型” 惨变家庭伦理剧？』批评了该剧战斗场面少家长里短多；英雄优点少缺点多；剧情煽情等。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2011-08-02/11443375209.shtml|title=《中国地》抗战“非典型” 惨变家庭伦理剧？ |accessdate= 2011年8月2日|publisher= 新浪网|work= |language =zh-cn}}</ref>

《[[大连晚报|大连晚报]]》专文『《中国地》主创现身 要告诉观众中国人的血性』强调该剧的意义是唤醒中国人心中业已磨灭的“血性”和“英勇”。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2011-08-02/16273375526.shtml |title= 《中国地》主创现身 要告诉观众中国人的血性|accessdate= 2011年8月2日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

《[[新民晚报|新民晚报]]》专文『《中国地》：大义和小民同在 缺点与可爱共存』指出该剧的中心是“义”和“情”。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2011-08-04/17093377798.shtml |title= 《中国地》：大义和小民同在 缺点与可爱共存|accessdate= 2011年8月4日|publisher= 新浪网|work= |language =zh-cn}}</ref>
