'''中央军委审计署综合局'''，位于[[北京市|北京市]]，是[[中央军委审计署|中央军委审计署]]下属局，负责该署综合业务。

==沿革==
在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月组建[[中央军委审计署|中央军委审计署]]。中央军委审计署新设'''中央军委审计署综合局'''<ref name=hdrd>{{Cite web |url =http://hdrd.bjhd.gov.cn/dbgz/dbml/  |title =  代表名录|author =  |date =  |publisher =  海淀人大|language =  |accessdate = 2017-08-08 |quote =  }}</ref>。
