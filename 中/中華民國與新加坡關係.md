'''中華民國與新加坡關係'''是指[[中華民國|中華民國]]與[[新加坡共和國|新加坡共和國]]（通稱'''新加坡'''、'''星國'''）之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，但在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。雖無邦交，而兩國在政治、軍事、經濟等各領域皆保持密切關係。新加坡也數次在[[海峽兩岸|海峽兩岸]]官方交流間擔任第三方的角色。

1965年8月9日，[[新加坡独立|新加坡独立建國]]。8月11日，[[行政院長|行政院長]][[嚴家淦|嚴家淦]]致電[[新加坡總理|新加坡總理]][[李光耀|李光耀]]，承認新加坡政府<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|716}}。此後，[[中華民國總統|中華民國總統]][[蔣中正|蔣中正]]及[[蔣經國|蔣經國]]均派代表赴任新加坡，蔣經國甚至派次子[[蔣孝武|蔣孝武]]長駐於新加坡。新加坡總理李光耀任內更訪問[[台灣|台灣]]達二十五次，堪稱是訪問台灣最多次的[[各國領導人列表|外國領導人]]。

1971年10月，新加坡在聯合國大會上支持中華人民共和國取得「中國」席次。

1987年6月8日至12日，行政院長[[俞國華|俞國華]]伉儷訪問新加坡。<ref>[http://history.ey.gov.tw/Item/VideoDetail/e0f1b5d3-7239-42bb-847e-ada7f5088c81 俞國華院長訪問新加坡紀實]</ref>

1987年12月9日，[[新加坡环境及水源部|新加坡環境部]]長麥馬德抵台訪問。期間會見外交部長[[丁懋時|丁懋時]]及其他政府官員，並參觀環保設施。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=2504B5A39500F7F3 |author = 公眾外交協調會 |date = 1987年12月8日 |title = 新加坡環境發展部部長麥馬德博士等一行三人定本（十二）月九日來華訪問四天 |publisher = 中華民國外交部}}</ref>

1988年1月13日[[蔣經國逝世|蔣經國逝世]]後，總理李光耀抵台參加其葬禮，表示「我來向這位已故總統，同時也是我的好朋友致哀」。<ref>{{Cite web |url = https://news.pts.org.tw/article/293223?NEENO=293223 |author = 綜合報導 |date = 2015年3月23日 |title = 李光耀與蔣經國相知 台星合作40年 |publisher = [[公視新聞|公視新聞]]}}</ref>7月4日，李光耀夫婦率領財政部長[[胡賜道|胡賜道]]、勞工部長李玉全抵台訪問，總統[[李登輝|李登輝]]夫婦在[[臺北賓館|臺北賓館]]設宴款待。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=1E9AD37BD4929FB3 |author = 公眾外交協調會 |date = 1988年7月4日 |title = 李總統登輝先生暨夫人於本（七）月四日晚七時卅分 |publisher = 中華民國外交部}}</ref>

李登輝也于繼任總統後的1989年3月6日訪問新加坡，和李光耀會面，這是唯一在任的[[中華民國國家元首|中華民國國家元首]]以官式訪問新加坡。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=2FBD546975AB8D32 |author = 公眾外交協調會 |date = 1989年3月6日 |title = 李總統訪星行前機場聲明 |publisher = 中華民國外交部}}</ref>4月17日，新加坡第一副總理兼國防部長[[吳作棟|吳作棟]]夫婦亦抵台訪問。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=581BF51E656F4C10 |author = 公眾外交協調會 |date = 1989年4月16日 |title = 新加坡第一副總理兼國防部長吳作棟伉儷抵華訪問 |publisher = 中華民國外交部}}</ref>

1990年1月4日，李光耀夫婦抵台訪問，陪同官員包括第二副總理[[王鼎昌|王鼎昌]]、外交部政務次長宋比得等人。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=0550A892DA893ED2 |author = 公眾外交協調會 |date = 1990年1月3日 |title = 新加坡共和國總理李光耀偕夫人等一行十人 |publisher = 中華民國外交部}}</ref>10月3日，新加坡与中华人民共和国[[外交#外交上的承認|建交]]。10月27日，李光耀夫婦再度抵台作私人訪問，並會見總統李登輝與行政院長[[郝柏村|郝柏村]]。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=85F3A4C45F0C1B92 |author = 公眾外交協調會 |date = 1990年10月29日 |title = 新加坡總理李光燿伉儷於本年十月廿七日至廿九日來華作私人訪問 |publisher = 中華民國外交部}}</ref>

1993年4月27日，由中華民國[[陸委會|陸委會]]授權的[[海峽交流基金會|海峽交流基金會]]（海基會）董事長[[辜振甫|辜振甫]]，與中華人民共和國[[國台辦|國台辦]]授權的[[海峡两岸關係協會|海峡两岸關係協會]]（海協會）會長[[汪道涵|汪道涵]]於新加坡舉行會談。而[[辜汪會談|辜汪會談]]也是自1949年[[兩岸分治|兩岸分治]]以來，首度進行的準官方會晤。

1993年9月，新加坡外交部長[[黃根成|黃根成]]在第48屆[[聯合國大會|聯合國大會]]的總辯論中，表示國際社會應鼓勵台灣參與國際組織。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=01CE54E75B2C7C88 |author = 公眾外交協調會 |date = 1993年10月14日 |title = 聯大第四十八屆常會總辯論自本年九月廿七日至十月十三日止 |publisher = 中華民國外交部}}</ref>

1997年11月，新加坡總理吳作棟在參加{{link-en|1997年加拿大APEC峰會|APEC Canada 1997|亞洲太平洋經濟合作會議領導人非正式會議}}（APEC高峰會）後，過境台灣[[中正國際機場|中正國際機場]]，與中華民國高層官員會面。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-1.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1999年9月21日，[[921大地震|台灣發生大地震]]，[[新加坡政府|新加坡政府]]派遣緊急救難隊抵台協助搜救。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2004年，[[李显龙|李显龙]]在上任总理之前到访台湾，[[台湾媒体|台湾媒体]]對此做出大幅報導，引起了[[中华人民共和国政府|中华人民共和国政府]]的不满。<ref name="epoch 2006"/><ref name="mfa">[http://www.mfa.gov.sg/content/mfa/overseasmission/shanghai/archive_press_statements/2004/200407/press_200407.html Q& A with DPM Lee on his visit to Taiwan] {{webarchive|url=https://web.archive.org/web/20140826114656/http://www.mfa.gov.sg/content/mfa/overseasmission/shanghai/archive_press_statements/2004/200407/press_200407.html |date=2014-08-26 }} 李副总理到访台湾的问与答，2004年7月19日，最后访问于2014年8月22日。</ref>由于中国大陆主张对台湾拥有主权，中华人民共和国政府反对[[中华人民共和国建交列表|邦交国]]与台湾建立任何的正式关系。[[中華人民共和國外交部|中華人民共和國外交部]]发言人[[章启月|章启月]]声称李显龙“[[伤害中国人民的感情|伤害13亿中国人民感情]]”。<ref>[http://english.peopledaily.com.cn/200407/12/eng20040712_149197.html China protests against Lee Hsien Loong's visit to Taiwan] 中国抗议李显龙访台，《[[人民日报|人民日报]]》，2004年7月12日。</ref>这促使李显龙在当上总理后重申支持[[一个中国|一个中国]]政策，反对[[台湾独立|台湾独立]]<ref>[http://www.singapore-window.org/sw04/040826a2.htm China warns officials against future visits to Taiwan] {{webarchive|url=https://web.archive.org/web/20120716185655/http://www.singapore-window.org/sw04/040826a2.htm |date=2012-07-16 }} 中国告诫新加坡官员今后不要访问台湾，《[[法新社|法新社]]》，2004年8月26日。</ref>，并说台湾报章作出了推测性的报道，包括声称[[新加坡国防部|新加坡国防部长]][[张志贤|张志贤]]随同李显龙访台，即使张志贤当时人在新加坡。<ref name="mfa"/>[[中華民國外交部#歷任首長|中華民國外交部長]][[陳唐山|陳唐山]]以「鼻屎大的國家」來形容新加坡，引發批評。<ref>[http://news.bbc.co.uk/chinese/trad/hi/newsid_3690000/newsid_3696700/3696788.stm 台外長以"鼻屎"喻新加坡引發批評]，《[[英國廣播公司#BBC中文部|BBC中文網]]》，2004年9月28日，最后访问于2014年8月25日。</ref>

==經濟==
{{see also|臺灣經濟|新加坡經濟}}
===背景===
中華民國與新加坡同為資本輸出國家，與[[東南亞國家協會|東南亞國家協會]]（東協）其餘各國比較，低技術層次的企業較難在新加坡尋得投資機會，但由於新加坡位居東南亞金融、貿易及運輸樞紐地位，[[勞動力|勞動力]]相對較[[東南亞#劃分|東南亞國家]]高，故紛紛前往新加坡投資[[貿易|貿易]]、[[金融保險|金融保險]]、[[電子元件|電子製造]]等行業。此外，由於[[香港回歸|1997年香港主權移交]]，許多[[臺灣商人|台商]]已將原在[[香港|香港]]的據點遷往新加坡，以操作國際[[接單生產|接單]]、[[融資|融資]]、調撥零件、支援售後服務和蒐集國際市場最新情報等。因此自1996年起，台商赴新加坡投資大量增加。為整合台商的資源與力量，成立了「新加坡台北工商協會」。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/sg/pdf/103-18Singapore.pdf |title = 投資環境簡介 |date = 2014年12月 |publisher = 經濟部全球臺商服務網}}</ref><ref name="國家基本資料表">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 國家基本資料表 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

===貿易===
[[經濟部國際貿易局|經濟部國際貿易局]]於[[新加坡|新加坡]]設立'''駐新加坡代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=214&pid=305862 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>2009年9月1日，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）也設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#ASIA |title = 外貿協會全球據點|publisher = 中華民國對外貿易發展協會}}</ref><ref name="98年外交年鑑"/>

2018年，新加坡是中華民國第6大貿易夥伴、第9大[[進口|進口]]夥伴、第5大[[出口贸易|出口]]夥伴。出口至新加坡的金額為173億6,421萬6,118[[美元|美元]]，年減1.479%；自新加坡進口的金額為84億1,223萬1,909美元，年減3.485%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）89億5,198萬4,209美元，年增0.483%。2017年，新加坡是中華民國第6大貿易夥伴、第6大進口夥伴、第5大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資===
截至2017年，中華民國對新加坡的投資累計599件，總金額134億5,071萬美元；新加坡對中華民國的投資累計2,472件，總金額81億6,145萬美元。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

金融業是台商投資新加坡行業的首位，目前已有[[彰化銀行|彰化銀行]]、[[台灣銀行|台灣銀行]]、[[華南銀行|華南銀行]]、[[臺灣土地銀行|臺灣土地銀行]]、[[第一商業銀行|第一商業銀行]]、[[兆豐國際商業銀行|兆豐國際商業銀行]]、[[國泰世華商業銀行|國泰世華商業銀行]]、[[中國信託商業銀行|中國信託商業銀行]]、[[玉山商業銀行|玉山商業銀行]]與[[台新國際商業銀行|台新國際商業銀行]]共10家銀行在新加坡設立分行。<ref name="投資環境簡介"/>

由於新加坡位居[[歐亞大陸|歐亞]]航線樞紐地位，中華民國的[[中華航空|中華航空]]、[[長榮航空|長榮航空]]、[[陽明海運|陽明海運]]、[[長榮海運|長榮海運]]與[[萬海航運|萬海航運]]等大型[[航運|航運]]業均在新加坡設有據點與航點，並帶動相關船務與維修的投資。<ref name="投資環境簡介"/>

1999年，[[台灣積體電路製造公司|台灣積體電路製造公司]]（台積電）在新加坡所投資的[[晶圓|晶圓]]廠開始動工，隨後[[聯華電子|聯華電子]]（聯電）於2000年宣布投入36億美元興建12[[英吋|英吋]]晶圓廠，已於2003年上半年[[量產|量產]]；[[聯發科技|聯發科技]]（聯發科）也於2004年在新加坡設立公司。由於大型廠商投資案的緣故，帶動周邊設備、[[封裝|封裝]]測試及相關產業的發展，從而引進[[中小企業|中小企業]]從事此方面的投資。<ref name="投資環境簡介"/>

2010年，[[長春集團|長春石化集團]]在新加坡[[裕廊島|裕廊島]]石化園區，投資生產[[烯丙醇|烯丙醇]]、[[異丙苯|異丙苯]]、[[乙酸乙烯酯|乙酸乙烯酯]][[單體|單體]]等三類產品，已於2013年開始量產。第二期投資計畫也將展開。<ref name="投資環境簡介"/>

2014年，聯電投資1億1,000萬美元於新加坡設立特殊技術中心，專門開發300[[毫米|毫米]]晶圓片特殊[[積體電路|積體電路]]的加工技術；<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 主要產業概況 |publisher = 中華民國對外貿易發展協會}}</ref>[[王品集團|王品集團]]旗下的蔬果品牌與新加坡餐飲集團合作，在新加坡展店2間，以新型態的[[素食主義|素食主義]]概念進入傳統[[華人|華人]]、[[印度人|印度人]]素食市場，甚至[[穆斯林|穆斯林]]的[[清真饮食|清真食品]]市場。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 投資環境分析 |publisher = 中華民國對外貿易發展協會}}</ref>

===經貿問題===
{{see also|2013年臺灣毒澱粉事件|2014年台灣劣質油品事件}}
2009年2月，中華民國向[[世界動物衛生組織|世界動物衛生組織]]（OIE）通報區域性[[口蹄疫|口蹄疫]]（Foot and Mouth Diseases, FMD），新加坡便即刻暫停冷凍[[豬肉|豬肉]]銷往該國。2012年，又通報OIE多起口蹄疫情，因此，新加坡至今仍未解除冷凍豬肉的進口禁令。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref>

2010年1月，銷往新加坡的禽肉產品因「低致病性[[禽流感|禽流感]]」（Low Pathogenicity Notifiable Avian Influenza, LPAI, {{link-en|H5N2|Influenza A virus subtype H5N2}}）疫情，遭新加坡禁止進口。經過協調與提供資料，並於2011年2月15日將追蹤報告遞交世界動物衛生組織，表示疫情受控制無新增案例後，新加坡自2011年2月28日起解除對中華民國的冷凍[[家禽|禽肉]]、加工禽肉產品與活[[禽鳥|禽鳥]]的進口禁令。但2011年3月，[[嘉義縣|嘉義]]再傳出致病性禽流感，新加坡即自2011年3月24日暫停冷凍禽肉與活禽鳥進口至新加坡，至今仍未解除進口禁令。<ref name="與我國經貿關係"/>

2013年上半年，爆發食品製造業者違法使用未經核准的[[順丁烯二酸酐|順丁烯二酸酐]][[化製澱粉|化製澱粉]]於[[粉圓|粉圓]]、[[米粉|米粉]]、[[關廟麵|關廟麵]]等食品案，行政院衛生署（現今[[中華民國衛生福利部|中華民國衛生福利部]]）[[衛生福利部食品藥物管理署|食品藥物管理署]]於2013年5月13日公告有關「呼籲食品業者應使用經核准之化製澱粉」資訊。新加坡{{link-en|農糧暨獸醫局|Agri-Food and Veterinary Authority of Singapore}}（AVA）隨即於5月17日發布[[新聞稿|新聞稿]]說明該案件，受影響產品包括[[蒲鉾|魚板]]、[[關東煮|黑輪]]、[[粄條|粄條]]、[[芋圓|芋圓]]、[[粉圓|粉圓]]、[[番薯|番薯]]粉、[[米粉|米粉]]、[[關廟麵|關廟麵]]等，該局並呼籲新加坡民眾退還受影響產品避免食用，並採取回收下架措施。除官方公告外，順丁烯二酸案也引起[[新加坡媒體|新加坡媒體]]關注，事件發生後，[[泡沫紅茶|泡沫紅茶]]連鎖經營業者，包括[[50嵐|50嵐]]、[[貢茶_(連鎖店)|貢茶]]、[[歇腳亭|歇腳亭]]、[[紅太陽_(連鎖店)|紅太陽]]及[[阿甘綠茶|阿甘綠茶]]等業者雖皆主動將產品送檢或公告產品未添加，但已損及臺灣食品整體形象。且自2014年2月3日起，新加坡要求輸銷該國的每批貨船中的部分食品必須檢附順丁烯二酸檢驗證書及測試報告。<ref name="與我國經貿關係"/>

2014年9月4日，爆發[[2014年台灣劣質油品事件|問題油品事件]]，受牽連企業涉及各大食品製造業者如[[味全|味全]]、[[味王|味王]]、[[旺旺集團|旺旺]]、[[統一超商|統一超商]]、[[全家便利商店|全家便利商店]]、[[奇美實業|奇美食品]]、[[盛香珍|盛香珍]]、[[85度C|85度C]]、[[黑橋牌|黑橋牌]]等。[[新加坡媒体|新加坡媒體]]持續報導相關消息，農糧暨獸醫局於9月22日宣布奇美與盛香珍6項受影響產品暫停進口。之後問題油品事件持續延燒，農糧暨獸醫局於11月5日進一步公布暫停進口受影響的3項統一滿漢大餐[[速食麵|速食麵]]品項。至今尚未解除對上述受問題油品影響產品的進口禁令。<ref name="與我國經貿關係"/>

==交流==
===新加坡華人===
{{see also|新加坡華人}}
1945年8月15日[[日本無條件投降|日本無條件投降]]后，新加坡各地展开了庆祝[[中國抗日戰爭|抗日]]胜利的活动。

在1970年和1980年代，有更多的漢族移民到[[新加坡|新加坡]]，到新加坡投資，工作，定居或念書。他們通常屬於受過[[高等教育|高等教育]]的人士，並且在各專業領域如[[半導體|半導體]]、[[研究開發|研發]]、[[法律|法律]]、[[商業|商業]]、[[金融|金融]]、[[教育|教育]]、[[宗教|宗教]]、與[[傳媒|傳媒]][[記者|記者]]等。

===教育===
在1950年代和1960年代，有許多新加坡的[[漢語|華文]]老師來自台灣。

兩國每年皆籌組各80人的[[高中生|高中生]]友好訪問團進行互訪。[[中華民國政府|中華民國政府]]也提供「臺灣[[獎學金|獎學金]]」及「[[中華民國國語|華語文]]獎學金」給予新加坡青年抵台攻讀[[學位|學位]]或研習中文。<ref name="105年外交年鑑"/>[[新加坡大學列表|新加坡3所國立大學]]與[[台灣大專院校列表|台灣多所大學]]簽有合作[[備忘錄|備忘錄]]，定期交流；<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2010/2.1.1.pdf |title = 《中華民國99年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>[[新加坡教育部|新加坡教育部]]曾運用台灣製作的影片以供[[中學|中]][[小學生|小學生]]的電腦教學資料。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2008/html/08-1-1.htm |title = 《中華民國97年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2008年11月，中華民國[[刑事警察局|刑事警察局]]外事警官隊會同[[新加坡警察部队|新加坡警方]]共同追查藏身中華人民共和國的[[詐騙集團|詐騙集團]]，發現至少有8名新加坡民眾被騙[[匯款|匯出]]約合230萬[[新台幣|新台幣]]到台灣，再由台灣的[[車手|車手]]集團主謀率員在台領款。先將提領金額1成扣下做為[[佣金|佣金]]，再將其他9成匯到對岸。警方並查出該車手集團主謀同樣因擔任車手被捕遭[[羈押|羈押]]中，遂借提追查，全案偵破。<ref>{{Cite web |url = http://news.ltn.com.tw/news/business/breakingnews/151527 |author = 黃敦硯 |date = 2008年11月20日 |title = 詐騙集團要求錢滙台灣 騙倒8新加坡人 |publisher = 《自由時報》}}</ref>
