'''中央军委联合参谋部第五十五研究所'''，位于[[北京市|北京市]]，是[[中央军委联合参谋部|中央军委联合参谋部]]某局下属的科研机构。

==简介==
该所原名'''中国人民解放军总参谋部第五十五研究所'''，是[[中国人民解放军总参谋部情报部|中国人民解放军总参谋部情报部]]（又称中国人民解放军总参谋部第二部）第二局下属的科研机构<ref>{{Cite web |url = http://www.110.com/panli/panli_107236.html |title =  北京金龙日盛建筑装饰工程公司与中国人民解放军总参谋部第五十五研究所债务纠纷案|author =  |date =2008-10-15  |publisher =110判裁案例   |language =  |accessdate =  |quote = 中国人民解放军总参谋部第二部第二局（以下简称总参二部二局）证实，总参五十五所为总参二部二局下属单位 }}</ref>。总参谋部第五十五研究所主要从事信息加密系统<ref>古孟贤，卫星电视加密系统，计算机与网络1995(2):32-32</ref>、新材料<ref>{{Cite web |url =http://www.jx.xinhuanet.com/renshi/20170120/3629289_c.html  |title =萍乡学院院长范小林任赣南师范大学党委副书记  |author =  |date =2017-01-20  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>等方面的研究。总参谋部第五十五研究所曾涌现出精细化工军事应用专家、[[中国工程院院士|中国工程院院士]][[吴慰祖|吴慰祖]]等专家学者<ref>{{Cite web|url = http://www.nantong.gov.cn/art/2008/9/5/art_39304_1327322.html|title = 吴慰祖|author = |date = 2008-09-05|publisher = 中国南通|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170907214015/http://www.nantong.gov.cn/art/2008/9/5/art_39304_1327322.html|archive-date = 2017-09-07|dead-url = yes}}</ref>。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，总参谋部第五十五研究所更名为'''中央军委联合参谋部第五十五研究所'''<ref>{{Cite web |url = http://www.bjkw.gov.cn/wzwj/2016_gs1/2016/07.html |title =环境保护与资源利用  |author =  |date =  |publisher = 北京科学技术委员会 |language =  |accessdate = 2017-05-22 |quote =  }}</ref>。
