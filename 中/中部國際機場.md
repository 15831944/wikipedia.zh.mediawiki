'''中部國際機場'''（{{jpn|j=中部国際空港|hg=ちゅうぶこくさいくうこう|rm=Chūbu kokusai kūkō|f=y}}，{{Airport codes|NGO|RJGG}}），日文[[暱稱|暱稱]]「{{lang|ja|'''セントレア'''}}」（新特麗亞／CENTRAIR），是位於[[日本|日本]][[愛知縣|愛知縣]][[常滑市|常滑市]]的[[國際機場|國際機場]]，為[[名古屋|名古屋]]與[[中部地方|中部地方]]主要的聯外機場，由[[日本政府|日本政府]]指定的{{link-ja|特殊會社|特殊会社}}「{{Link-ja|中部國際機場公司|中部国際空港株式会社}}」經營，同時被[[日本政府|日本政府]]評定為[[日本機場列表#1._據點機場|據點機場]]。2005年2月17日正式啟用，做為[[名古屋機場|名古屋機場]]的後繼者，其使用原名古屋機場的[[IATA機場代碼|IATA機場代碼]]「NGO」，而[[ICAO機場代碼|ICAO機場代碼]]則為「RJGG」。

== 概況 ==
中部國際機場坐落於[[伊勢灣|伊勢灣]]內的[[人工島|人工島]]上，是日本第三座以[[填海造陸|填海造陸]]興建之機場，前兩座為[[長崎機場|長崎機場]]及[[關西國際機場|關西國際機場]]。機場在2005年2月啟用後，所有原由[[名古屋機場|名古屋機場]]處理的商業航班均轉為由中部國際機場處理。機場在[[2005年世界博覽會|2005年愛知世博]]開幕前啟用。其暱稱「新特麗亞」（{{jpn|j=セントレア|rm=centrair|f=y}}）係由英文「{{lang|en|Central}}」（代表[[中部地方|中部地方]]）與「{{lang|en|Airport}}」（機場）兩字組合而成，用於機場的設施標示、機場島的{{link-ja|住居表示|住居表示}}、以及[[航空交通管制|航空交通管制]]的呼號上，並由機場營運方登錄為[[註冊商標|註冊商標]]（商標登錄編號：第4566713號等<ref name="j-platpat">{{Cite web |url = https://www2.j-platpat.inpit.go.jp/syutsugan/TM_LIST_T.cgi?ITEM01=106&KEY01=Centrair&OPT01=01&ITEM02=107&KEY02=Centrair&OPT02=01&ITEM03=103&KEY03=Centrair&OPT03=01&ITEM04=705&KEY04=&OPT04=01&STIME=147193799898919107898208&HITCNT=7&S_FLAG=00&TERMOPT=01&PAGE=01 |title = 商標出願・登録情報（結果一覧）｜J-PlatPat |publisher = 独立行政法人 工業所有権情報・研修館 |accessdate = 2016-08-23 }}</ref>）。

中部國際機場在2000年8月開始興建，成本為7680億[[日圓|日圓]]（約73億[[美元|美元]]），但在有效管理成本下，節省了近1000億日圓成本。

除了各項減輕成本政策外，中部國際機場亦有多項為保護環境而作的設計。人工島建成英文字母P字的形狀，令伊勢灣內的水流維持暢通。其岸邊部份以天然石塊建成並傾斜，以協助海中生物在此棲息。

日本是[[波音787|波音787]]的機身組件製造國之一，因此[[波音公司|波音公司]]的超大型貨機[[波音747-400_LCF|747-400 LCF]]（由[[亞特拉斯航空|亞特拉斯航空]]代為操作）常會在中部國際機場起降，載運機身半成品到美國。2015年，該公司将首架波音787-8[[原型机|原型机]]（N787BA,ZA001）捐赠给日本[[名古屋|名古屋]]，在中部国际机场静态展览<ref>[http://boeing.mediaroom.com/2015-07-07-Boeing-Donates-First-Ever-787-Dreamliner-to-Nagoyas-Central-Japan-International-Airport Boeing Donates First-Ever 787 Dreamliner to Nagoya's Central Japan International Airport]</ref>。

== 客運大樓 ==
客運大樓形狀像英文字母T字，共有3條客運廊由中央伸展，此設計令旅客由登記櫃檯步行至登機閘口的距離少於300米，[[建築師|建築師]]原本構思的設計，是讓客運大樓建成像[[紙鶴|摺紙鶴]]的模樣，但由於成本原因而放棄。

抵達旅客會經2樓進行入境程序，而離開的旅客則使用3樓。最底層則用作維修、航空膳食及其他地勤服務，還有供停機坪的旅客[[巴士|巴士]]作上落之用。

== 機場交通 ==
* 搭乘電車
從中部國際機場起搭乘[[名鐵|名鐵]]特急到[[名鐵名古屋車站|名古屋站]]約需28分鐘。
