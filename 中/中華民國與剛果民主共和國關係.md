'''中華民國與剛果民主共和國關係'''是指[[中華民國|中華民國]]與[[剛果民主共和國|剛果民主共和國]]（通稱'''民主剛果'''、'''剛果（[[金夏沙|金]]）'''，1971－1997年稱'''[[薩伊|薩伊]]'''）之間的關係。兩國於1960－1973年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對民主剛果的相關事務由'''[[駐歐盟兼駐比利時代表處|駐歐盟兼駐比利時代表處]]'''{{Link-en|多重委派|Dual accreditation|管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-149-bbb26-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[剛果民主共和國|剛果民主共和國]]。]]
{{see also|中華民國外交|{{link-en|剛果民主共和國外交|Foreign relations of the Democratic Republic of the Congo}}}}
{{see also|中国驻刚果民主共和国大使列表{{!}}中華民國駐剛果民主共和國／薩伊共和國外交官列表|刚果民主共和国驻华大使列表{{!}}剛果民主共和國／薩伊共和國駐中華民國外交官列表}}
1960年8月10日，建立大使級[[外交#外交上的承認|外交關係]]。於首都[[利奧波德維爾|利奧波德維爾]]（1966年改名為金夏沙）設立'''中華民國駐剛果民主共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1964年10月13日，簽署《中華民國與剛果民主共和國技術合作協定》。1969年9月3日續簽。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e5%89%9b%e6%9e%9c&tycountry_e=%e5%89%9b%e6%9e%9c&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

1971年4月15日，剛果民主共和國總統[[蒙博托·塞塞·塞科|蒙博托·塞塞·塞科]]抵台訪問，並與[[蔣中正|蔣中正]]總統進行晤談。<ref>[http://www.ccfd.org.tw/ccef001/index.php?option=com_content&view=article&id=1464:0008-45&catid=198&Itemid=256　對剛果總統莫布杜訪華致歡迎詞]</ref>10月25日，民主剛果在[[聯合國大會|聯合國大會]]上支持中華民國保有聯合國內的[[中國與聯合國#代表權問題|「中國」席次]]。

1973年1月30日，[[薩伊共和國|薩伊共和國]]與中華民國斷交。

1991年6月10日，與薩伊簽署《相互設處特別協議》。9月，於首都金夏沙設立[[實質大使館|具大使館性質]]的'''[[臺北經濟文化代表處|中華民國代表團]]'''，享有[[外交特權|外交特權]]，並辦理領務。1997年5月，[[第一次剛果戰爭|薩伊叛軍攻入首都]]，恢復國號為[[剛果民主共和國|剛果民主共和國]]。後因[[第二次剛果戰爭|內戰情勢不穩]]及兩國實質關係無法推動，[[中華民國外交部|中華民國外交部]]於1998年8月撤離人員。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter1-3.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>代表團目前已關閉。

[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境剛果民主共和國。]]
[[File:Visa_requirements_for_DR_Congo_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|剛果民主共和國簽證政策|Visa policy of the Democratic Republic of the Congo}}|{{link-en|剛果民主共和國公民簽證要求|Visa requirements for Democratic Republic of the Congo citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往剛果民主共和國駐[[新加坡|新加坡]][[領事館|領事館]]申請簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-149-bbb26-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|剛果民主共和國護照|Democratic Republic of the Congo passport}}的{{link-en|剛果民主共和國人口|Demographics of the Democratic Republic of the Congo|民主剛果公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將剛果民主共和國歸類為'''橙色警戒'''：避免非必要旅行（2018年8月13日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-207-a2f40-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將該國按疾病歸類為'''第一級注意'''：提醒遵守當地的一般預防措施，與'''第二級警示'''：對當地採取加強防護（以上截至2018年11月6日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=43B8AAED11163074 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|剛果民主共和國經濟}}
===貿易===
2017年，剛果民主共和國是中華民國第93大貿易夥伴、第67大[[進口|進口]]夥伴、第143大[[出口贸易|出口]]夥伴。出口至民主剛果的金額為507萬5,541[[美元|美元]]，年減23.045%；自民主剛果進口的金額為8,883萬1,805美元，年增92.525%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）8,375萬6,264美元，年增111.801%。※2016年，剛果民主共和國是中華民國第100大貿易夥伴、第81大進口夥伴、第134大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===事件===
兩國在邦交期間，[[中國輸出入銀行|中國輸出入銀行]]（以下簡稱輸銀）提供[[貸款|貸款]]資助剛果民主共和國。斷交後，輸銀於2015年12月向[[美國紐約南區聯邦地區法院|美國紐約南區聯邦地區法院]]提告{{noteTag|中國輸出入銀行在[[紐約|紐約]]設有辦公室，過去貸款給[[中華民國外交#邦交國|中華民國的邦交國]]時由輸銀出面與對方國家銀行簽約，合約可以適用{{link-en|紐約州法律|Law of New York}}，對中華民國比較有保障。}}，求償包括未償還貸款與[[利息|利息]]。法院於2017年1月裁定剛果民主共和國須償還5,730萬美元給輸銀。<ref>{{Cite web |url = https://udn.com/news/story/5/2375838 |author = 田思怡 |date = 2017年3月31日 |title = 與台灣斷交的非洲國家小心了 銀行打官司討回64億貸款 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref><ref>{{Cite web |url = http://www.cw.com.tw/article/article.action?id=5081742 |author = [[德國之聲|德國之聲]] |date = 2017年4月2日 |title = 與台灣斷交 要付出代價！ |publisher = [[天下雜誌|天下雜誌]]}}</ref>
