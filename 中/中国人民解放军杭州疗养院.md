'''中国人民解放军杭州疗养院'''，总院位于[[浙江省|浙江省]][[杭州市|杭州市]][[西湖区_(杭州市)|西湖区]]杨公堤27号（东门）、龙井路5号（西门），隶属[[无锡联勤保障中心|无锡联勤保障中心]]。

==简介==
1950年5月1日，'''浙江军区干部疗养院'''在[[余杭|余杭]][[良渚|良渚]]崇福寺组建成立。1952年12月，该院自崇福寺迁驻杭州市保俶路。1953年5月，改称'''华东军区第三干部疗养院'''。同年底，迁驻杭州市黄泥岭。1954年12月，改称'''中国人民解放军第三三疗养院'''。1955年5月，改归[[中国人民解放军浙江省军区|浙江军区]]领导。1960年7月，第三三疗养院改称'''中国人民解放军南京军区杭州疗养院'''。1969年3月，改编为驻军治疗医院，番号为'''中国人民解放军第一二八医院'''。1979年7月，改称'''中国人民解放军南京军区杭州疗养院'''，并由团级单位改成师级单位。1980年8月，杭州疗养院直属南京军区后勤部领导。后该院下设分院，分院位于[[莫干山|莫干山]]<ref name=zjsjsz> {{Cite book | author =  浙江省军事志编纂委员会编| title =  浙江省军事志| location =  | publisher = 方志出版社 | date = 1999年 | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote =第三篇 驻军 第三章 中华人民共和国成立后驻军 第五节 医院(疗养院)  }} </ref>，后成为该疗养院莫干山疗养区<ref name=mogansh>{{Cite web |url = http://www.81.cn/zghjy/2016-02/03/content_6886742.htm |title = 解放军杭州疗养院党委领导顶风踏雪慰问偏远单位 |author =  |date =2016-02-03   |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.81.cn/zghjy/2017-01/25/content_7467114.htm?from=groupmessage&isappinstalled=0|title = 身处德清“寒极”，那份关怀与温暖却一直都在 |author =  |date = 2017-01-25  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.81.cn/zghjy/2016-08/14/content_7205291.htm |title =莫干山疗养区：白云深处的“守山兵”  |author =  |date = 2016-08-14 |publisher =  中国军网|language =  |accessdate =  |quote =  }}</ref>。 
 
1951年1月，开始筹建'''中国人民解放军海军杭州疗养院'''，至1954年7月3日在浙江省杭州市正式成立，初设3个疗养科、100张床位。1954年底，增设内科、外科、放射科、理疗科、门诊室、五官科，床位增至200张，番号称'''中国人民解放军第一五六医院'''。1963年5月，复称'''中国人民解放军海军杭州疗养院'''，床位增至250张，1979年10月增至400张<ref name=zjsjsz/>。

2004年，海军杭州疗养院、[[中国人民解放军空军杭州航空医学鉴定训练中心|空军杭州疗养院]]并入南京军区杭州疗养院，后分别改为南京军区杭州疗养院海勤疗养区、空勤疗养区；同年，中国人民解放军第一一七医院转隶南京军区杭州疗养院。2005年起，执行新的编制。2007年，成立直属院区党委。2010年，空勤疗养区重新划归[[南京军区空军|南京军区空军]]管理，南京军区杭州疗养院的编制由此变为：院机关、直属院区、海勤疗养区、莫干山疗养区、第一一七医院<ref>杭州疗养院简介，中国康复医学会疗养康复学术会议，2010年</ref>。

截至2014年，该疗养院有全军级中心3个，军区级中心2个，并先后与[[安徽医科大学|安徽医科大学]]、[[浙江大学|浙江大学]]、[[中国人民解放军第二军医大学|中国人民解放军第二军医大学]]等高等院校建立协作关系，是[[杭州师范大学|杭州师范大学]]健康管理学院临床实践基地
<ref>{{Cite web |url =http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536738.htm  |title = 南京军区杭州疗养院 |author =  |date =  2014-09-12|publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，南京军区杭州疗养院由[[中国人民解放军南京军区|中国人民解放军南京军区]]转隶[[无锡联勤保障中心|无锡联勤保障中心]]，更名'''中国人民解放军杭州疗养院'''<ref name=junsao>{{Cite web |url =http://www.junsaozg.com/a/junsaofc/2017/0620/86651.html  |title = 新规逐条学习 禁令刚性落实 |author =  |date = 2017-06-20  |publisher =军嫂网  |language =  |accessdate =  |quote =  }}</ref>。

==中国人民解放军第一一七医院==
前身为华东军区运输部医训队<ref name=jj/>。1951年9月，'''华东军区西湖陆军医院'''在杭州市九里松组建成立，1952年12月改称'''华东军区第十一陆军医院'''。1954年9月划归浙江军区领导，改称'''中国人民解放军第四一疗养院'''，1956年5月改编为驻军医院，番号为'''中国人民解放军第一一七医院'''。1958年6月，中国人民解放军第九六医院并入中国人民解放军第一一七医院<ref name=zjsjsz/>。1982年起，第一一七医院向社会开放。

中国人民解放军第九六医院：1950年12月，浙江军区接收杭州市卫生局所属浙江病院。1952年7月，浙江病院与浙江军区直属医院合并成立'''浙江军区医院'''，驻浙江省杭州市劳动路。1954年9月，改称'''中国人民解放军第九六医院'''。1958年6月，并入第一一七医院，番号撤销<ref name=zjsjsz/>。

中国人民解放军第一一七医院是全军[[三级甲等医院|三级甲等医院]]，是驻浙江省规模最大的集医疗、教学、科研、保健于一体的综合性部队医院。截至2016年，医院拥有1个全军治疗中心（全军脊柱外科治疗中心），7个军区（省）专科专病中心（老年病诊治中心、呼吸病诊治中心、整形修复治疗中心、内分泌代谢病诊治中心、检验质量控制中心、优生优育技术指导中心、浙江省癌症中心胸部肿瘤研究指导中心），2个浙江省重点学科（医学检验科、骨科），3个杭州市重点学科（整形外科、骨外科、医学影像科）、4个驻地特色学科（交通伤急救、肾病移植治疗、大肠肛门病治疗、心血管病治疗），3个大型设备诊疗中心（PET-CT诊断中心、伽玛刀治疗研究中心、射波刀治疗中心），1个健康管理中心。医院是[[浙江大学|浙江大学]]、[[中国人民解放军第二军医大学|中国人民解放军第二军医大学]]、[[江苏大学|江苏大学]]、[[安徽医科大学|安徽医科大学]]、[[温州医科大学|温州医科大学]]、[[杭州师范大学|杭州师范大学]]医学院、[[浙江中医药大学|浙江中医药大学]]等高等院校的教学医院
<ref name=jj>{{Cite web |url = http://www.117yiyuan.com/index.php?c=content&a=list&catid=104 |title =医院简介  |author =  |date =  |publisher = 中国人民解放军第一一七医院 |language =  |accessdate = 2017-06-23 |quote =  }}</ref>。

*九里松院区：位于浙江省杭州市[[西湖区_(杭州市)|西湖区]]灵隐路14号，地处[[杭州西湖|杭州西湖]]景区九里松<ref name=jj/>。
*机场路院区：位于浙江省杭州市[[江干区|江干区]]机场路40号，毗邻[[杭州东站|杭州火车东站]]、[[杭州地铁|杭州地铁]][[闸弄口站|闸弄口站]]<ref name=jj/>。
