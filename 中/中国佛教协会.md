'''中国佛教协会'''，是[[中华人民共和国|中华人民共和国]]成立后的[[中国佛教|中国佛教]]组织，1953年初成立。

== 历史 ==
1952年11月4日至5日，中国佛教协会发起人会议在北京举行。此次会议研究、讨论中国佛教协会的宗旨、任务、组织，并通过《中国佛教协会发起书》。而会议发起人有[[释虚云|虚云]]、[[喜饶嘉措|喜饶嘉措]]、[[噶喇藏|噶喇藏]]、[[釋圓瑛|圆瑛]]、[[柳霞·土登塔巴|柳霞·土登塔巴]]、[[丹巴日杰|丹巴日杰]]、[[罗桑巴桑|罗桑巴桑]]、[[多吉占东|多吉占东]]、[[能海|能海]]、[[法尊|法尊]]、[[巨赞|巨赞]]、[[陈铭枢|陈铭枢]]、[[吕澂|吕澂]]、[[趙樸初|赵朴初]]、[[董鲁安|董鲁安]]、[[叶恭绰|叶恭绰]]、[[林宰平|林宰平]]、[[向达|向达]]、[[周叔迦|周叔迦]]、[[郭朋|郭朋]]等。最终会议推举组成中国佛教协会筹备处，赵朴初任主任。<ref>{{cite web |url = http://fo.ifeng.com/zhuanti/fojiaozhounian/dashiji/200909/0920_456_58144.shtml |title =1 952年：赵朴初被选举成为中国佛教协会筹备处主任 |publisher = 凤凰网 来源：慧海佛教资源库 |author= |date=2009年9月20日 |accessdate=2011-11-25 |language = zh-hans}}</ref>

1953年5月30日，在[[广济寺_(北京市)|北京广济寺]]举办中国佛教协会成立会议。121位[[活佛|活佛]]、[[喇嘛|喇嘛]]、[[法师|法师]]、[[居士|居士]]参加大会，他们来自[[汉族|汉]]、[[藏族|藏]]、[[蒙古族|蒙]]、[[傣族|傣]]、[[满族|满]]、[[苗族|苗]]、撒里维吾尔（[[裕固族|裕固]]）等7个民族。喜饶嘉措致开幕词，赵朴初报告中国佛教协会成立、发起经过及筹备工作。<ref>{{cite web |url = http://fo.ifeng.com/zhuanti/fojiaozhounian/dashiji/200909/0920_456_58145.shtml |title = 1953年：中国佛教协会成立会议在北京广济寺隆重开幕 |publisher=凤凰网 来源：慧海佛教资源库 |author =|date=2009年9月20日 |accessdate=2011-11-25 |language = zh-hans}}</ref>
