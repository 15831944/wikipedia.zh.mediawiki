'''中山快速公交'''（{{lang-en|Zhongshan Bus Rapid Transit}}），又称'''中山BRT'''，缩写'''ZBRT'''，位于[[中华人民共和国|中国]][[广东省|广东省]][[中山市|中山市]]，為中山首条建成的[[快速公交系统|快速公交系统]]，设有7条路线及13个快速公交车站，2014年1月27日正式营运。

== 概况 ==
首期主线（利和广场—火炬公交枢纽站）呈东西走向，系统全长12.6[[公里|公里]]，其营运、管理由[[中山公交集團|中山公交集團]]负责，主要是负责营运系统的统一调度和管理工作。

== 历史 ==
*2011年，中山市人大十三届六次会议上，中山《政府工作报告》提出：实施“交通先行再先行，公交优先再优先”的交通基础设施发展战略。<ref>{{Cite news|url = http://www.zs.gov.cn/main/zwgk/newsview/index.action?id=97548|title = 2011年政府工作报告|last = |first = |date = |work = |accessdate = }}</ref>
*2011年3月，中山市启动《中山公共交通发展战略》研究，历经7个月的深入调研，《中山市公共交通发展战略》通过国家级专家评审，其中提出构建“一环、二交叉、三横、四纵、五辅”的快速公交骨干网络。
*2012年4月，十四届五次市政府常务会议研究同意实施《中山市公共交通发展战略》。<ref>{{Cite news|url = http://www.zsnews.cn/ZT/ZSBUS/2012/12/31/2334982.shtml|title = 公交的台阶和台阶上的公交
——中山公交集团落实薛晓峰书记“公交再上新台阶”指示纪实|last = |first = |date = |work = |accessdate = }}</ref>
*2013年5月31日，中山市快速公交系统示范线工程可行性研究通过评审，随即开展设计、报建等相关工作。
*2013年10月21日，快速公交示范线工程动工，工期90天。<ref>{{Cite news|url = http://www.zsnews.cn/zt/wenmingzs/News/2013/10/22/2532391.shtml|title = 中山快速公交示范线昨开工|last = |first = |date = |work = |accessdate = }}</ref>
*2014年1月27日上午9点30分，首班试运营的快速公交车发车。
*2014年7月18日，开通支线B13路“大岭-东利”、B15路“利和广场-中山树木园”，优化调整支线B11路“人民医院-城轨中山站”为“中仪花园-城轨中山北站”，调整主线B1路、支线B10路和支线B12路首末站时间。<ref>{{cite news| title=关于新开、优化调整快速公交线路的通告| url=http://www.zsbus.cn/zh-CN/displaynews008.html?newsID=269214| deadurl=yes| archiveurl=https://web.archive.org/web/20150227091450/http://www.zsbus.cn/zh-CN/displaynews008.html?newsID=269214| archivedate=2015-02-27}}</ref>

== 乘坐方式 ==
乘客可通过[[斑马线|斑马线]]进入和离开站台，在BRT通道内需要在站台投币或使用[[中山通|中山通]]刷卡进站，一票制2元/人次，使用中山通IC卡实施中山市政府规定的票价优惠政策。进入站台后按指引在指定闸口候车，在快速公交系统内同站台内均可免费换乘任意方向其他路线。不参与该系统营运的公共汽车路线则保留在道路两侧公共汽车站点停靠，不进入专用通道行驶。其他细则与普通公共汽车无异。火炬公交樞紐站較為特別，如乘搭另一方向路線，需下车後再付費進入站台。

== 路线 ==
具体路线采用1條主线及7条支线，其中主线从利和广场至火炬公交枢纽站，全程走BRT通道，其余的6条支线为部分路段行走BRT通道。所有行经BRT的巴士线统一以“B”字开头。
