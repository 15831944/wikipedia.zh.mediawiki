'''中国人民解放军国防科技大学气象海洋学院'''，位于[[江苏省|江苏省]][[南京市|南京市]]，隶属[[中国人民解放军国防科技大学|中国人民解放军国防科技大学]]。

==沿革==
*1950年12月，在[[北京市|北京市]]建立'''军委气象局气象干部训练班'''，隶属[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]气象局<ref name=shouwang>{{Cite web|url = http://www.qstheory.cn/kj/kjrw/201401/t20140117_313870.htm|title = 王鹏飞：守望气象|author = |date = 2014-01-17|publisher = 求是理论网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20140118031717/http://www.qstheory.cn/kj/kjrw/201401/t20140117_313870.htm#|archive-date = 2014-01-18|dead-url = yes}}</ref><ref name=njshyong>{{Cite web|url = http://mzj.nanjing.gov.cn/njsyw/69695/201707/t20170731_4888171.html|title = 金陵晚报双拥周刊244：气象海洋学院的南京情缘|author = |date = 2017-07-31|publisher = 南京双拥网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170804214137/http://mzj.nanjing.gov.cn/njsyw/69695/201707/t20170731_4888171.html|archive-date = 2017-08-04|dead-url = yes}}</ref>。
*1953年1月，气象干部训练班改为'''军委气象局气象干部学校'''<ref name=shouwang/>。
*1953年8月，军委气象局划归[[中央人民政府政务院|中央人民政府政务院]]管理，改称[[中国气象局|中央气象局]]，气象干部学校仍留在军委<ref name=shouwang/>。
*1953年12月，学校移交地方，改为'''北京气象专科学校'''，隶属[[中央人民政府高等教育部|中央人民政府高等教育部]]（1954年改为[[中华人民共和国高等教育部|中华人民共和国高等教育部]]）<ref name=jsszqxsyz>{{Cite book |url = http://www.jssdfz.gov.cn/book/qxsyz/D11/D3J3.HTM |title = 江苏省志·气象事业志 |author = 江苏省地方志编纂委员会 |date = 1996年 |publisher = 江苏科学技术出版社 |language =  | |quote = 三、空军气象学院 }}</ref>。
*1955年8月，[[中华人民共和国国务院|中华人民共和国国务院]]将隶属中华人民共和国高等教育部的北京气象专科学校移交[[中国人民解放军空军|中国人民解放军空军]]领导，更名为'''中国人民解放军气象专科学校'''<ref name=njshyong/><ref name=jsszqxsyz/>。
*1956年，[[中国人民解放军空军航空大学|中国人民解放军第九航空学校]]气象预报系并入中国人民解放军气象专科学校。
*1960年4月，改称'''中国人民解放军气象学校'''<ref name=njshyong/>。
*1962年，[[中国人民解放军军事工程学院|中国人民解放军军事工程学院]]空军工程系的机场建筑和气象专业并入中国人民解放军气象学校。
*1963年1月，中国人民解放军气象学校从北京市迁至[[江苏省|江苏省]][[南京市|南京市]]中和桥。1963年3月改为'''中国人民解放军空军第三高级专科学校'''<ref name=nanjing>{{Cite web|url = http://app.nanjing.gov.cn/njnj/sz8_12/8/html/08Noname374.html|title = 第三节 南京解放后军事院校|author = |date = |publisher = 中国南京|language = |accessdate = 2017-08-05|quote = }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name=njshyong/><ref name=jsszqxsyz/>。
*1966年至1968年，因[[文化大革命|文化大革命]]影响，学校停止招生三年。1968年11月复课<ref name=jsszqxsyz/>。
*1969年11月，改为'''中国人民解放军空军第三专科学校'''<ref name=nanjing/>。中专性质<ref name=jsszqxsyz/>。
*1970年1月，移驻[[江苏省|江苏省]][[江宁县|江宁县]]岔路口原南京步兵学校校址<ref name=nanjing/>。
*1975年，改为'''中国人民解放军空军气象学校'''<ref name=nanjing/>。
*1979年6月，改为'''中国人民解放军空军气象学院'''<ref name=nanjing/><ref name=njshyong/><ref name=jsszqxsyz/><ref>{{Cite web |url =http://www.mod.gov.cn/service/2007-09/25/content_4085897.htm  |title =  解放军空军气象学院|author =  |date =2007-09-25  |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
*1999年4月，由中国人民解放军通信工程学院、中国人民解放军工程兵工程学院、中国人民解放军空军气象学院、中国人民解放军总参谋部第六十三研究所合并组建[[中国人民解放军理工大学|中国人民解放军理工大学]]。其中，中国人民解放军空军气象学院改为'''中国人民解放军理工大学气象学院'''<ref name=njshyong/>。
*2003年，中国人民解放军理工大学气象学院正式成立军事海洋系，面向全军招收军事海洋学本科生。2006年成为海洋气象学博士点。2010年成为全军唯一的海洋科学一级学科硕士点<ref name=haiyang/>。
*2012年8月7日，中国人民解放军理工大学气象学院正式更名为'''中国人民解放军理工大学气象海洋学院'''<ref name=haiyang>{{Cite web |url = http://mil.news.sina.com.cn/2012-10-16/0833703744.html |title = 解放军理工大学气象学院更名气象海洋学院 |author =  |date = 2012-10-16 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，以[[中国人民解放军国防科学技术大学|中国人民解放军国防科学技术大学]]、[[中国人民解放军国际关系学院|中国人民解放军国际关系学院]]、[[中国人民解放军国防信息学院|中国人民解放军国防信息学院]]、[[中国人民解放军西安通信学院|中国人民解放军西安通信学院]]、[[中国人民解放军电子工程学院|中国人民解放军电子工程学院]]、中国人民解放军理工大学气象海洋学院为基础组建[[中国人民解放军国防科技大学|中国人民解放军国防科技大学]]<ref name=xin>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1699945  |title = 新国防科大今年以国防科大、国关学院等6所院校为基础重建 |author =  |date =  2017-06-03 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。其中，中国人民解放军理工大学气象海洋学院改为'''中国人民解放军国防科技大学气象海洋学院'''<ref>{{Cite web|url =http://cesi.cumt.edu.cn/portal.php?mod=view&aid=583|title =中国气象科学研究院研究员郭建平等来我院作大气环境遥感报告|author =|date =2017-07-09|publisher =中国矿业大学环境与测绘学院|language =|accessdate =|quote =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
