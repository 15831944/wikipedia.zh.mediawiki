'''中國國民黨第一次全國代表大會'''於1924年1月20日至30日期間，在當時[[广州|广州]]的[[國立廣東高等師範學校|國立廣東高等師範學校]]禮堂舉行。大會出席代表共196人，[[孫中山|孫中山]]任大會主席，[[胡漢民|胡漢民]]、[[汪精衛|汪精衛]]、[[林森|林森]]、[[李大釗|李大釗]]及[[謝持|謝持]]任主席團成員。苏联顾问[[鲍罗廷|鲍罗廷]]也出席此次会议。

== 背景 ==
第二次[[護法運動|護法運動]]後，孫中山開始考慮與共產黨合作的可能。1923年1月，孫中山與[[蘇聯|蘇聯]]政府全權代表[[越飛|越飛]]在[[上海|上海]]會面，正式討論與共產黨合作。孫、越會面曾後發表《[[孙文越飞宣言|孫文越飛聯合宣言]]》{{ref|4}}。

同年年初，[[陳炯明|陳炯明]]部被拥护孙中山的[[滇系|滇]]、[[桂系|桂]]、和倒戈[[粤系|粤]]军联合组成的西路讨贼军击败，退守[[东江|东江]]。孫中山得以在3月回廣州。12月29日，孫中山落實接受[[列寧|列寧]]和[[共產國際|共產國際]]的協助重建[[大元帥府|大元帥府]]，共產國際派出[[鮑羅廷|鮑羅廷]]到廣州為孫中山顧問，以[[蘇共|蘇共]]為模式重組[[中國國民黨|中國國民黨]]。1924年1月，孫中山在中國國民黨第一次全國代表大會上宣佈實行[[聯俄容共|聯俄容共]]政策，並發表[[s:中國國民黨第一次全國代表大會宣言|《中國國民黨第一次全國代表大會宣言》]]。在蘇聯援助下，於3月組建[[黃埔軍校|黃埔軍校]]，並以[[蔣中正|蔣中正]]為校長。

== 會議 ==
[[File:The-Principle-of-Minsheng_notice_in_Guangzhou_1924.jpg|thumb]] 
本次大會選出中央執行委員24人、以總理為主席、候補中央執行委員17人；中央監察委員5人、候補中央監察委員5人。中央執行委員會為全國代表大會閉會期間之最高權力機關。

 
重要決議：
# 通過「組織國民政府之必要案」。
# 制定「中國國民黨總章」、「中國國民黨政綱」。
# 通過「紀律問題」及「海關問題」案。
# 通過第1次全國代表大會宣言。

*总理：孙 文
*中央执行委员会常委：廖仲恺、戴季陶、谭平山（共产党员）
* 组织部长[[谭平山|谭平山]]；秘书 杨匏安（共产党员）
* 宣传部长[[戴季陶|戴季陶]]（汪兆铭继任）；秘书 刘芦隐
* 工人部长[[廖仲恺|廖仲恺]]；秘书 冯菊坡（共产党员）
* 农民部长[[林祖涵|林祖涵]]（共产党员）（黄居素、廖仲恺继任）；秘书 彭湃（共产党员）
* 军事部长[[许崇智|许崇智]]
* 青年部长[[邹鲁|邹鲁]]；秘书 孙甄陶。

*妇女部长[[曾醒|曾醒]]（何香凝继任）；秘书 唐允恭
