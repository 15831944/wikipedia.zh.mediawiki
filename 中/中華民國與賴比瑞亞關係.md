'''中華民國與賴比瑞亞關係'''是指[[中華民國|中華民國]]與[[賴比瑞亞共和國|賴比瑞亞共和國]]之間的關係。兩國於1957－2003年有官方[[外交#外交上的承認|外交關係]]但曾經中斷，二次斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對賴比瑞亞的相關事務由'''{{Link-en|駐奈及利亞聯邦共和國臺北貿易辦事處|Taipei Trade Office in the Federal Republic of Nigeria}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-211-1c71f-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[賴比瑞亞|賴比瑞亞]]。]]
{{see also|中華民國外交|{{link-en|賴比瑞亞外交|Foreign relations of Liberia}}}}
{{see also|中華民國駐賴比瑞亞大使列表{{!}}中華民國駐賴比瑞亞外交官列表|賴比瑞亞駐中華民國大使列表{{!}}賴比瑞亞駐中華民國外交官列表}}
1957年8月19日，建立公使級[[外交#外交上的承認|外交關係]]。於首都[[蒙羅維亞|蒙羅維亞]]設立'''中華民國駐賴比瑞亞共和國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-3.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1960年2月，升格為大使級外交關係。公使館升格為'''中華民國駐賴比瑞亞共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1965年4月24日，{{link-en|賴比瑞亞副總統|Vice President of Liberia}}{{link-en|陶伯特|William R. Tolbert, Jr.}}抵台訪問。<ref>[http://history.ey.gov.tw/Item/PhotoDetail/e515bf05-b1d8-4299-ac3f-153ca78bfa71 賴比瑞亞副總統陶伯特訪華]，[[行政院|行政院]]</ref>

1967年12月，總統府秘書長[[張群|張群]]任慶賀賴比瑞亞總統、副總統就職典禮特使。<ref>{{Cite web|url=http://www.history.com.tw/pe/11/11001301.htm|title=張群小傳|accessdate=2017-03-29|author=|date=|work=www.history.com.tw|publisher=}}</ref>

1977年2月23日，與中華民國首次斷交。<ref name="86年外交年鑑"/>

1989年10月2日，{{link-en|利比里亚政府|Politics of Liberia}}宣布将与[[中华民国政府|中华民国政府]]恢復大使級外交關係。9日，双方在[[台北市|台北]]签署《中华民国与賴比瑞亞共和国恢复外交关系联合公报》，正式复交。12月，{{link-en|第一次賴比瑞亞內戰|First Liberian Civil War|賴比瑞亞發生內戰}}，至1993年7月25日簽署《和平協定》，政權暫由各派系組成的「過渡政府」接掌。8月10日，該國{{link-en|阿莫斯·索耶|Amos Sawyer|索耶}}派「臨時政府」在政權移交前夕，宣布與[[中華人民共和國|中華人民共和國]]恢復外交關係，中華民國發表聲明，質疑此決定的合法性，而該國最大派系「{{link-en|賴比瑞亞全國愛國陣線|National Patriotic Front of Liberia|全國愛國陣線}}」領導人[[查尔斯·泰勒_(利比里亚政治人物)|泰勒]]於9月7日發表聲明，反對「臨時政府」與之復交的決定。此後，由於賴比瑞亞政局不明，[[海峽兩岸|海峽兩岸]]大使館得以並存於首都蒙羅維亞。1997年7月19日，泰勒{{link-en|1997年賴比瑞亞大選|Liberian general election, 1997|當選賴比瑞亞總統}}，其所領導的{{link-en|賴比瑞亞全國愛國黨|National Patriotic Party|全國愛國黨}}亦獲得[[利比里亚议会|國會]]絕對多數席次。泰勒當選後即表示繼續與中華民國維持外交關係，並致函邀請總統[[李登輝|李登輝]]出席就職典禮{{noteTag|由行政院政務委員[[蘇起|蘇起]]以總統特使身分率團前往。}}。賴比瑞亞政府發言人表示，同時承認[[一國兩府|中華民國與中華人民共和國兩個政府]]，又賴比瑞亞為[[主權國家|主權國家]]，有權與任何國家交往。中華人民共和國在此情形下宣布與賴比瑞亞中止外交關係，並撤消承認該國駐[[香港|香港]][[名譽領事|名譽領事]]。<ref name="86年外交年鑑"/>

2003年10月12日，與中華民國第二次斷交。當日，{{link-en|賴比瑞亞外交部|Ministry of Foreign Affairs (Liberia)}}代理部長接見[[中華民國駐賴比瑞亞大使列表|駐賴大使]]陳永綽表示，因受到[[西部非洲|鄰近國家]]、[[非洲聯盟|非洲聯盟]]、[[聯合國|聯合國]]之派駐代表的強大壓力，決定自當地時間10月12日起恢復與中華人民共和國的外交關係，稍後對外正式宣布。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=70BCE89F4594745D&sms=700DE7A3F880BAE6&s=6DAF303793D10CDA |author = 公眾外交協調會 |date = 2003年10月12日 |title = 我宣佈與賴比瑞亞斷交記者會紀要 |publisher = 中華民國外交部}}</ref>

2010年5月31日，《[[紐約時報|紐約時報]]》報導，賴比瑞亞前總統泰勒正在[[荷蘭|荷蘭]][[海牙國際法庭|海牙國際法庭]]受審，同時，賴比瑞亞政府也追討其[[貪污|貪污]]資產，追查過程中，發現有8筆總額超過2,000萬[[美元|美元]][[匯款|匯款]]來自中華民國政府，但全被泰勒中飽私囊。[[中華民國外交部|中華民國外交部]]副發言人章計平表示，當初援助的款項是用來興建[[醫院|醫院]]或幫助[[青少年|青少年]]復學等[[人道救援|人道救援]]用途，對於賴比瑞亞把這筆金錢用在其他用途，外交部深感遺憾，並譴責賴比瑞亞沒善用台灣的援助好意。至於是否有匯款以鞏固邦交，章計平不願回應。但知情人士表示，在當時兩岸[[烽火外交|烽火外交]]情況下，「台灣需要用各種方法防止中國搶走[[中華民國外交#中华民国的邦交国|我邦交國]]，會有金援外交的出現，也是當時不得不的背景造成的。」<ref>{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20100601/32553898/ |author = 蔡筱雯、陳郁仁 |date = 2010年6月1日 |title = 賴比瑞亞前總統 吞我金援6.4億 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref>

兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可在首都的[[羅勃茲國際機場|羅勃茲國際機場]]辦理[[簽證#落地签证|落地簽證]]入境，可停留7－30日。若停留超過30日，可前往賴比瑞亞駐[[日本|日本]][[外交代表機構#大使馆|大使館]]申請簽證。入境須出示[[黃熱病|黃熱病]]的[[疫苗接種|疫苗接種]]證明（黃皮書）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-211-1c71f-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

===貿易===
2017年，賴比瑞亞是中華民國第183大貿易夥伴、第207大[[進口|進口]]夥伴、第170大[[出口贸易|出口]]夥伴。出口至賴比瑞亞的金額為134萬3,294[[美元|美元]]，年增71.134%；自賴比瑞亞進口的金額為2萬5,047美元，年增41,645.000%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）131萬8,247美元，年增67.956%。※2016年，賴比瑞亞是中華民國第195大貿易夥伴、第233大進口夥伴、第180大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

==援助==
===中華民國===
由於{{link-en|第一次賴比瑞亞內戰|First Liberian Civil War|賴比瑞亞連年內戰}}，中華民國於1996年9月捐贈1萬[[公噸|公噸]][[食米|食米]]。<ref name="86年外交年鑑"/>

1990年，設立駐賴比瑞亞[[農業技術|農業技術]]團與[[醫療|醫療]]團，後因該國內戰而撤團。至1997年，恢復派駐農技團，<ref name="86年外交年鑑"/>以協助發展[[水稻|水稻]]、[[山稻|陸稻]]及[[蔬菜|蔬菜]]等作物。<ref name="87年外交年鑑"/>2002年，又因{{link-en|第二次賴比瑞亞內戰|Second Liberian Civil War|賴比瑞亞內亂紛擾}}，撤離部分團員。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__03.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

===賴比瑞亞===
