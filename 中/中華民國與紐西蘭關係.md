'''中華民國與紐西蘭關係'''是指[[中華民國|中華民國]]與[[紐西蘭|紐西蘭]]之間的關係。兩國於1912－1972年有官方[[外交#外交上的承認|外交關係]]，斷交後，在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構，保持密切的非官方往來。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[紐西蘭|紐西蘭]]。]]
{{see also|中華民國外交|{{link-en|紐西蘭外交|Foreign relations of New Zealand}}}}
{{see also|中国驻新西兰大使列表{{!}}中華民國駐威靈頓／紐西蘭外交官列表}}
1912年，中華民國成立後，繼承[[清朝政府|清朝政府]]於[[光緒|光緒]]二十九年（1903年）在[[紐西蘭自治領|紐西蘭自治領]]首都設立的'''駐[[威靈頓|威靈頓]][[領事館|領事館]]'''。<ref name="駐館與駐地關係"/>

1935年，[[國民政府|國民政府]]將領事館升格為'''駐威靈頓總領事館'''。<ref name="駐館與駐地關係">{{Cite web |url = http://www.taiwanembassy.org/nz/post/12.html |date = 2016年7月8日 |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref>

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持[[外交#外交上的承認|邦交]]。

1958年，[[八二三炮戰|八二三炮戰]]期間，[[紐西蘭總理|紐西蘭總理]][[華爾特·納許|華爾特·納許]]建議台灣自外島撤軍，而由國際間保證其地位，成為一個獨立而中立的國家。<ref>[[美國國防部|美國國防部]]助理部長辦公室委託[[蘭德公司|蘭德公司]]研究著作、謝永湉譯，1985年，《一九五八年台灣海峽危機》，[[中華民國國防部|中華民國國防部]]史政編譯局</ref>{{rp|227-236，266}}

1961年8月17日，駐威靈頓總領事館升格為'''駐紐西蘭[[外交代表機構#大使馆|大使館]]'''。<ref name="駐館與駐地關係"/>

1970年6月14日，中華民國外交部長[[魏道明|魏道明]]赴紐西蘭出席第五屆亞洲太平洋理事會部長級會議。<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|789}}

1971年10月25日，紐西蘭支持中華民國繼續保有[[聯合國|聯合國]]裡的[[中國與聯合國#代表權問題|中國席次]]（即'''中國代表權'''，《[[聯合國大會2758號決議|聯合國大會2758號決議]]》），但最終表決失敗。

1972年12月22日，紐西蘭與中華民國斷交，並與[[中華人民共和國|中華人民共和國]]建交。<ref name="外交年鑑"/>

2002年，{{link-en|紐西蘭外交暨貿易部|Ministry of Foreign Affairs and Trade (New Zealand)|紐西蘭外交部}}[[次長|次長]]理查·格蘭特（Richard Grant）抵台訪問。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__01.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2013年，威靈頓市長{{link-en|切利亞·韋德-布朗|Celia Wade-Brown}}抵台訪問。<ref name="104年外交年鑑"/>

2017年4月19日，民進黨立委[[陳瑩_(政治人物)|陳瑩]]表示，外交部亞太司副司長[[林恩真|林恩真]]在2016年底的立法院長[[蘇嘉全|蘇嘉全]]宴席上，當著聽得懂[[中文|中文]]的紐西蘭[[毛利人|毛利國王]]幕僚長朗基希羅亞（Rangihiroa Whakaruru）的面說她曾派駐紐西蘭6年，沒跟毛利國王接觸過，又說毛利國王「沒有用」，且不受當地原住民支持，[[新西兰总理|新總理]]也不會支持他，丟盡台灣的臉；但當時全程在場的民進黨立委[[谷辣斯·尤達卡|谷辣斯·尤達卡]]否認這個說法，表示整場宴席完全沒聽到這樣的事情，也向林恩真求證，她沒有這樣說過。<ref>{{Cite web |url = https://udn.com/news/story/6656/2418082 |author = 徐偉真 |date = 2017年4月21日 |title = 外交官說毛利國王沒用？綠委還原現場說沒有 |publisher = 《[[聯合新聞網|聯合新聞網]]》}}</ref>外交部27日表示，已將相關報告送交[[立法院外交及國防委員會|立法院外交及國防委員會]]，但「當晚談話細節難以還原」。<ref>{{Cite web |url = http://www.ettoday.net/news/20170427/912905.htm |author = 翁嫆琄 |date = 2017年4月27日 |title = 外交官遭疑說毛利國王沒用 外交部：難以還原當晚談話 |publisher = 《[[ETNEWS新聞雲|ETNEWS新聞雲]]》}}</ref>

===代表機構===

1973年5月，[[中華民國政府|中華民國政府]]在第1大城[[奧克蘭_(紐西蘭)|奧克蘭]]設立'''駐紐西蘭屋崙亞東貿易中心'''{{noteTag|[[華裔紐西蘭人|當地老華僑]]將奧克蘭譯作屋崙。其後為避免混淆，中華民國外交部於1992年4月30日更名為通用的奧克蘭。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=8E7F883A72EEA7D0 |author = 公眾外交協調會 |date = 1992年4月30日 |title = 我國原駐美之「羅安琪」 |publisher = 中華民國外交部}}</ref>}}。1980年7月，亞東貿易中心遷至首都[[威靈頓|威靈頓]]。1982年，增設屋崙分處。1991年11月，亞東貿易中心更名為[[實質大使館|具大使館性質]]的'''{{link-en|駐紐西蘭臺北經濟文化辦事處|Taipei Economic and Cultural Office in New Zealand}}'''（Taipei Economic and Cultural Office, New Zealand），並享有部分[[外交豁免權|特權與禮遇]]；屋崙分處亦更名為[[領事館|具領事館性質]]的'''駐奧克蘭台北經濟文化辦事處'''（Taipei Economic and Cultural Office in Auckland）。<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/06.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref><ref name="辦事處簡介">{{Cite web |url = http://www.roc-taiwan.org/nzakl/post/353.html |date = 2016年7月7日 |title = 辦事處簡介 |publisher = 中華民國駐外單位聯合網站}}</ref><ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-1.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref name="駐外館處">{{Cite web |url = http://www.mofa.gov.tw/OverseasOffice_Content.aspx?n=6357834932B83C83&sms=5A961ED2E4BA25E8&s=8A33013523400F35&os=882AC6F2C0F2D795 |title = 駐外館處 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>

1987年10月，[[紐西蘭政府|紐西蘭政府]]在首都[[臺北市|臺北]]設立同等性質的'''{{link-en|紐西蘭商工辦事處|New Zealand Commerce and Industry Office}}'''（New Zealand Commerce and Industry Office）。於1990年4月起開辦[[簽證|簽證]]業務、1991年11月增設經濟組及貿易發展組、1994年起恢復調派{{link-en|紐西蘭外交暨貿易部|Ministry of Foreign Affairs and Trade (New Zealand)|紐西蘭外交部}}高層人員擔任駐台代表。<ref name="駐館與駐地關係"/><ref name="90年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2001/Chap3/3-1-1.htm |title = 《中華民國90年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref name="駐華外國機構">{{Cite web |url = http://www.mofa.gov.tw/OfficesInROC_Content.aspx?n=8CEB2B5F5436B997&sms=8EBFADC1592C7BFE&s=8A33013523400F35&os=E5C0F6A40A13E849 |title = 駐華外國機構 |publisher = 中華民國外交部}}</ref>

1994年9月，[[紐西蘭國會|紐西蘭國會]]通過《外交特權豁免法》修正法案，於1995年1月起授權[[紐西蘭政府|相關部會]]賦予[[台灣|台灣]]駐處人員薪資[[所得稅|所得稅]]免稅之待遇。1997年10月，紐西蘭外交部再同意賦予台灣駐處及人員相當於[[外交代表機構|外交代表機構]]及領事館所應享有之財稅優惠特權待遇。<ref name="86年外交年鑑"/>

[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境紐西蘭。]]

持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境紐西蘭，停留最多90天。紐西蘭特別關注[[自助旅遊|自助行]]或以免簽證方式入境的旅客其旅行計畫與財力證明，應事先查詢有關規定。經紐西蘭轉機限於[[奧克蘭國際機場|奧克蘭國際機場]]，無須申請[[過境簽證|過境簽證]]，停留不得超過24小時。轉機前往[[澳洲|澳洲]]若持有{{link-en|澳大利亞簽證政策|Visa policy of Australia|澳洲簽證}}，亦無需申請過境簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-19-2e478-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&sms=B75E93A3DAB23A99 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持有[[紐西蘭護照|紐西蘭護照]]的[[紐西蘭人口|紐西蘭公民]]也可以免簽證的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

此外，兩國皆提供每年各600名額給18－30歲青年為期1年的[[工作假期签证|度假打工簽證]]（皆必須未曾取得此種簽證）。<ref>{{Cite web |url = http://www.youthtaiwan.net/WorkingHoliday/Cus_WHNation.aspx?n=4892E8B8F5C0E174&s=13E6E9C4BE15A3CA |title = 青年度假打工 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-82-43-18183-1.html |title = 紐西蘭籍人士申請中華民國度假打工事由簽證相關手續說明 |publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將紐西蘭歸類為'''灰色警示'''：提醒注意（2017年5月22日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-249-31637-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

[[經濟部國際貿易局|經濟部國際貿易局]]在威靈頓設立'''駐紐西蘭代表處經濟組'''。<ref name="本部駐外單位">{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=214&pid=305862 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

自從2013年7月10日簽署《[[臺紐經濟合作協定|紐西蘭與臺灣、澎湖、金門、馬祖個別關稅領域經濟合作協定]]》（ANZTEC），並於12月1日生效後，2015年11月，兩國就ANZTEC執行情況共同進行「整體檢視」，報告指出ANZTEC對兩國的經濟皆帶來正面效益，紐西蘭自中華民國[[進口|進口]]成長10%，紐西蘭[[出口贸易|出口]]至中華民國則成長22%。ANZTEC聯合委員會已建立{{link-en|食品安全檢驗與動植物防疫檢疫|Sanitary and phytosanitary measures and agreements|SPS}}、{{link-en|技術性貿易障礙|Technical barriers to trade|TBT}}、[[原住民|原民]]合作、[[貿易|貿易]]與[[勞工|勞工]]、貿易與[[環境|環境]]等5個次級委員會，以及[[服務業|服務業]]專家小組會議，逐步建立廣泛且定期性的交流合作機制。2017年11月，召開第3屆ANZTEC聯合委員會。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2018年，紐西蘭是中華民國第37大貿易夥伴、第33大進口夥伴、第37大出口夥伴，在[[大洋洲国家和地区列表|大洋洲經濟體]]皆排名第2。出口至紐西蘭的金額為4億8,418萬2,806[[美元|美元]]，年增3.402%；自紐西蘭進口的金額為9億0,938萬4,467美元，年增8.299%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）4億2,520萬1,661美元，年增14.472%。2017年，紐西蘭是中華民國第37大貿易夥伴、第34大進口夥伴、第38大出口夥伴，在大洋洲經濟體皆排名第2。<ref>{{Cite web|url =https://cus93.trade.gov.tw/|title =中華民國進出口貿易統計|publisher =經濟部國際貿易局}}</ref>

===投資===
[[臺灣商人|台商]]及移民投資紐西蘭的熱潮出現於1993年－1997年，尤其以1993年和1996年的投資金額7,426萬[[紐西蘭元|紐元]]和5,488萬紐元最多（1紐元換算[[新臺幣|新臺幣]]約20.6元），其餘年度均超過1,000萬紐元；但自1998年開始，投資金額即出現下滑，每年均不及400萬紐元，1999年和2004年更低於100萬紐元以下。<ref name="投資環境分析"/>

至2013年3月，台商在紐西蘭的總投資金額為24億8,000萬紐元，名列外資第9位，高於第10名的[[中華人民共和國|中華人民共和國]]的23億5,200萬紐元。但此項統計未包括機密不予公布的投資案件與金額，因此僅有參考價值，無法全然反應外資在紐西蘭的投資現況。<ref name="投資環境分析"/>

臺商的主要投資項目包括[[教育|教育]]、[[森林|森林]]、[[農場|農場]]、[[不動產|不動產]]、[[旅館|旅館]]、[[超市|超市]]、[[旅行社|旅行社]]、[[肉品|肉品]][[食品加工|加工]]、[[健康食品|健康食品]]製造、[[化妝品|化妝品]]製造、[[電腦|電腦]]維修、一般[[貿易|貿易]]等，但規模普遍不大，且未在該國經濟中扮演重要角色。較具規模的業者包括教育機構（ACG教育集團、奧克蘭商學院、紐西蘭商業學院）、高爾夫球球場旅館、中華電視網、皇家律師事務所、商勝集團不動產投資、三寶健康食品、環球肉品加工公司、[[東元電機|東元電機]]、璽龍有機烏龍茶、亞洲食品進口批發商OK Mark、T-Mark，以及「太紫」[[中国烧酒|白酒]]的盧氏兄弟等。<ref name="投資環境分析"/>

2005年停航台北－奧克蘭航線的[[紐西蘭航空|紐西蘭航空]]於2018年11月1日復航，董事長{{link-en|泰瑞瑟·沃爾什|Therese Walsh|沃爾什}}、首席營收長華萊士（Cam Wallace）等人搭乘首航班機抵達台北。其航線以[[波音787#787-9|波音787-9客機]]執飛，每週3航班往返，是兩地唯一直飛的航線。<ref>{{Cite web |url = https://udn.com/news/story/7241/2994117 |author = 楊文琪 |date = 2018年2月22日 |title = 紐西蘭航空台北─奧克蘭直飛航線 今年11月開航 |publisher = 《聯合新聞網》}}</ref><ref>{{Cite web |url = https://www.gq.com.tw/life/travel/content-35322.html |author = 徐毓、Kyle H.|date = 2018年2月23日 |title = 紐西蘭航空你終於回來了！台北直飛奧克蘭航線11月正式開航 |publisher = 《[[GQ|GQ]]》}}</ref><ref>{{Cite web |url = https://www.nownews.com/news/20181101/3045522/ |author = 陳致宇 |date = 2018年11月1日 |title = 睽違13年！紐西蘭航空重返台灣天空 首航載客率9.4成 |publisher = 《[[NOWnews今日新聞|NOWnews今日新聞]]》}}</ref>

===駕車===
外國人在紐西蘭合法駕車都必須持有[[國際駕駛執照|國際駕駛執照]]或[[中華民國汽車駕駛執照|本國駕駛執照]]英文譯本，有效期1年，滿1年後必須申請{{link-en|紐西蘭駕駛執照|Driver licence in New Zealand}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-19-2e478-05-1.html |title = 生活資訊 |publisher = 外交部領事事務局}}</ref>
