'''中華民國與巴拿馬關係'''是指[[中華民國|中華民國]]與[[巴拿馬共和國|巴拿馬共和國]]之間的關係，現今亦指[[臺灣|臺灣]]與巴拿馬之間的關係。

兩國於1910－2017年有正式[[外交#外交上的承認|外交關係]]，其歷史可追溯至[[清朝|清朝]][[宣統|宣統]]元年，在中華民國建立以及[[臺灣戰後時期|遷至台灣]]後，始終維持外交關係，直到2017年6月，巴拿馬與[[中華人民共和國|中華人民共和國]]建交，聲明中表示依「[[一個中國|一個中國]]」政策終止與「臺灣」之間的直接官方關係。{{noteTag|{{link-en|巴拿馬政治|Politics of Panama|巴拿馬共和國政府}}承認世界上只有一個中國，[[中華人民共和國政府|中華人民共和國政府]]是代表全中國的唯一[[合法政府|合法政府]]，[[中华人民共和国疆域|臺灣是中國領土不可分割的一部分]]。<ref>{{cite web|url=https://www.presidencia.gob.pa/en/News/JOINT-STATEMENT-BETWEEN-THE-REPUBLIC-OF-PANAMA-AND-THE-PEOPLE-S-REPUBLIC-OF-CHINA|title=JOINT STATEMENT BETWEEN THE REPUBLIC OF PANAMA AND THE PEOPLE'S REPUBLIC OF CHINA|publisher=Ministry of the Presidency, Republic of Panama|date=2017-06-12|language=en|quote=The Government of the Republic of Panama today breaks up their "diplomatic relations" with Taiwan, and undertakes to leave any official contact or relationship with Taiwan.|location=Panama City|deadurl=no|archiveurl=https://web.archive.org/web/20170728144639/https://www.presidencia.gob.pa/en/News/JOINT-STATEMENT-BETWEEN-THE-REPUBLIC-OF-PANAMA-AND-THE-PEOPLE-S-REPUBLIC-OF-CHINA|archivedate=2017-07-28}}</ref>}}<ref name=official>{{cite web|url=http://www.mire.gob.pa/index.php/es/noticias-mire/10366-panama-se-convierte-en-el-primer-pais-de-latinoamerica-que-firma-acuerdo-sobre-iniciativa-de-accion-global-contra-las-redes-fantasmas-2|title=COMUNICADO CONJUNTO ENTRE LA REPÚBLICA DE PANAMÁ Y LA REPÚBLICA POPULAR CHINA SOBRE EL ESTABLECIMIENTO DE RELACIONES DIPLOMÁTICAS|date=2017-06-12|language=es|location=Ciudad de Panamá|publisher=Gobierno de la República de Panamá|quote=El Gobierno de la República de Panamá rompe hoy mismo sus “relaciones Diplomáticas” con Taiwán, y se compromete a dejar toda relación o contacto oficial con Taiwán.|deadurl=no|archiveurl=https://web.archive.org/web/20170616145929/http://www.mire.gob.pa/index.php/es/noticias-mire/10366-panama-se-convierte-en-el-primer-pais-de-latinoamerica-que-firma-acuerdo-sobre-iniciativa-de-accion-global-contra-las-redes-fantasmas-2|archivedate=2017-06-16}}</ref>

斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對巴拿馬的相關事務由'''中華民國駐[[尼加拉瓜共和國|尼加拉瓜共和國]][[中華民國駐外機構列表|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-271-5ba45-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

中華民國與巴拿馬的[[外交#外交上的承認|外交關係]]可追溯於[[清朝|清朝]][[宣統|宣統]]元年，1909年12月6日，在首都[[巴拿馬城|巴拿馬城]]設立'''大清駐巴拿馬共和國[[總領事館|總領事館]]'''，<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部 |deadurl = no |archiveurl = https://web.archive.org/web/20130604074247/http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html |archivedate = 2013-06-04 }}</ref>後於1910年1月16日建立[[領事|領事級]]外交關係，並派駐總領事。

1912年2月12日，中華民國繼承清朝與巴拿馬的外交關係。

1921年12月，中華民國派駐巴拿馬[[特命全權公使|公使]]。1922年1月20日，總領事館升格為'''中華民國駐巴拿馬共和國[[外交代表機構#公使馆|公使館]]'''。1954年5月1日，升格為'''[[外交代表機構#大使馆|大使館]]'''，<ref name="105年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部 |deadurl = no |archiveurl = https://web.archive.org/web/20180129004249/http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |archivedate = 2018-01-29 }}</ref>並派駐[[特命全權大使|大使]]；1972年10月16日，在巴拿馬第二大城設立'''中華民國駐[[科隆_(巴拿马)|箇朗]]領事館'''，1989年6月19日，升格為'''總領事館'''，<ref name="86年外交年鑑"/>後於2009年7月關閉。

1933年6月，在首都[[南京市_(中華民國)|南京]]設立'''巴拿馬共和國駐中華民國公使館'''。1954年5月1日，在[[台北市|台北]]升格為'''大使館'''；<ref name="105年外交年鑑"/>1978年10月，增設'''巴拿馬共和國駐台北總領事館'''，由該國大使兼任總領事。<ref name="94年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2005/WebPages/MICO/08/08_01_06.htm |title = 《中華民國94年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

[[美國|美國]]在1989年12月[[美国入侵巴拿马|對巴拿馬發動入侵行動]]並逮捕最高領導人[[曼纽尔·诺列加|諾瑞嘉]]後，[[中華民國政府|中華民國政府]]在1990年1月5日宣布承認巴拿馬新政府。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=6193E07700915985 |author = 公眾外交協調會 |date = 1990年1月5日 |title = 中華民國政府決定自即日起正式承認巴拿馬新政府 |publisher = 中華民國外交部}}</ref>

== 經濟 ==
{{see also|臺灣經濟|巴拿馬經濟}}
===貿易===
2018年，巴拿馬是中華民國第79大貿易夥伴、第84大[[進口|進口]]夥伴、第72大[[出口贸易|出口]]夥伴。出口至巴拿馬的金額為1億1,171萬5,698[[美元|美元]]，年減12.243%；自巴拿馬進口的金額為4,078萬7,760美元，年增21.764%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）7,092萬7,938美元，年減24.387%。2017年，巴拿馬是中華民國第75大貿易夥伴、第88大進口夥伴、第64大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

據2017年統計，中華民國的航運公司共有68艘船在巴拿馬註冊，為巴拿馬的第五大[[權宜船|權宜船]]來源國，其中45艘為[[長榮海運|長榮海運]]所有，該公司在巴拿馬設有專用港。<ref>[http://times.hinet.net/news/20244633 台巴斷交 交通部：68艘權宜船不受影響]</ref>

===投資===
截至2016年，[[臺商|臺商]]對巴拿馬的總投資金額為16億6,638萬美元，共計83件，主要投資業別有[[運輸|運輸]]及[[倉儲|倉儲業]]、[[製造業|製造業]]、[[金融|金融]]及[[保險|保險業]]、[[批發|批發]]及[[零售業|零售業]]；巴拿馬對台灣總投資金額為8億8,682萬美元，共計72件。<ref name="105年外交年鑑"/>

[[兆豐國際商業銀行|兆豐國際商業銀行]]在巴拿馬設有分支機構。<ref name="103年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/11.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

===合作===
中華民國曾在巴拿馬派駐[[農業技術|農業技術]]團，合作項目有[[園藝|園藝]]、[[蔬菜|蔬菜]]種子繁殖及栽培、[[海蝦養殖|海水蝦繁養殖]]及核心示範農戶等計畫；<ref name="88年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter1-6.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>另根據《中巴[[漁業|漁業]]技術合作協定》，在巴拿馬派駐漁業技術團，從事漁撈示範、漁民訓練、[[漁場|漁場]]調查等工作。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=C34A597950A18ACD |author = 公眾外交協調會 |date = 1995年12月8日 |title = 國內部分媒體於十二月八日報導巴拿馬總統下令工商部中止中巴漁業協定 |publisher = 中華民國外交部}}</ref>

==僑民==
