[[中华人民共和国政府|中华人民共和国政府]]通过[[国家新闻出版广电总局|国家新闻出版广电总局]]对在[[中国大陆|中国大陆]]落地的[[香港|香港]]、[[澳门|澳门]]、[[台湾|台湾]]及[[境內與境外|其他境外]]卫星电视节目，及[[中国大陆电视|中国大陆的电视节目]]内容进行审查，这是一种政府行政行为。绝大多数的国家或地区也有电视内容审查，但审查方式比[[中华人民共和国|中华人民共和国]]的少。

在[[中华人民共和国|中华人民共和国]]内地的省市有线电视网（多指[[广东省|广东省]]）可以收看得到的境外电视台（绝大部分）都是香港特别行政区的电视台。大陆的媒体被政府部门层层监控，所以电视审查一般情况下也就只会应用到那几家可以在中国大陆播放的香港特区和澳门特区的电视台。另外针对大陆三星级或以上酒店及涉外单位的卫星电视节目也有审查。

== 背景 ==
{{Unreferenced section|time=2018-12-16T05:44:05+00:00}}
{{See also|广东电视广播}}
最先开始电视审查的是[[广东省|广东省]]（[[广州市|广州市]]）人民政府。在广东省，民众可以接收到香港的[[香港電視台列表|两家免费电视台]]（无线电视和亚洲电视，亚视在结束免费电视广播后，广东省内仅能收看无线电视）的信号。为了避免香港的新闻报导与大陆的新闻报道出现差别，电视审查应运而生。最初的审查发生在1989年[[六四事件|六四事件]]前后，之后的[[法轮功|法轮功]]，[[SARS事件|SARS事件]]，[[程翔|程翔]]间谍案等敏感话题以及每逢[[翡翠台|翡翠台]]、[[本港台|本港台]]、[[明珠台|明珠台]]和[[亞洲電視國際台|国际台]]播放[[六合彩|六合彩]]（目前香港唯一的合法彩票）搅珠结果时都被这个机制准时大量过滤信息。

与[[中华人民共和国网络审查|中国网络审查]]不同，因为香港特别行政区已有电视分级制度，免费电视台很少会播放成人节目，因此，电视审查也只局限审查时政节目的内容，缺少如网络审查那些正当的理由（过滤色情、暴力信息）。同时，即使是在香港被列为M级（Mature，[[香港電視節目分類制度|成人观众]]）的部分电视节目，如《[[欲望城市|欲望城市]]》（港名：《色欲都市》；英文名：Sex and the City），在广东省境内依然能正常收看。

== 审查对象 ==
{{Unreferenced section|time=2018-12-16T05:46:10+00:00}}
#政治性議題（包括[[文化大革命|文化大革命]]、[[六四事件|六四事件]]、对[[中國人權|中國人權]]自由的批評等）；
#宗教（[[梵蒂冈|梵蒂冈]]等）；
#冷处理国际事件（俄国阅兵式、梵蒂冈等，央视外语频道通常直播录像在CNTV被屏蔽）
#政治異見人士（如[[台灣|台灣]][[泛綠|泛綠]]、[[第十四世达赖喇嘛|第十四世达赖喇嘛]]、[[香港泛民主派|香港泛民议员]]、[[中國維權運動|中国维权人士]]等）相關；
#香港、澳门、台灣的大规模游行或社會運動（如[[七一遊行|七一遊行]]、[[五一遊行|五一遊行]]、[[雨傘革命|佔領中環]]、[[太陽花學運|太陽花學運]]等）；
#中國大陆民众游行、示威集会、警民衝突事件，以及海外国家或地区的[[反中|反中]]、反共游行集会；
#中国大陆尚未经官方认可公开的新闻（如[[愛滋病|愛滋病]]、[[SARS事件|SARS事件]]、[[豆腐渣工程|豆腐渣工程]]、[[礦難|礦難]]、[[上訪|上訪]]、大陆政府中的[[贪污腐败|貪腐問題]]等）；
#被中央政府和[[中华人民共和国公安部|中华人民共和国公安部]]定义为[[邪教|邪教]]的组织在海外的近况（[[法轮功|法轮功]]等）；

== 审查特点 ==
审查对象主要包括：
# 香港地面电视：[[电视广播有限公司|电视广播有限公司]]的[[翡翠台|翡翠台]]和[[明珠台|明珠台]]（广东省，2017年3月23日前用模拟信号源，24号起开始使用数字信号源）。
# [[亚太6号|亚太6号]]卫星境外平台上面的所有频道（中国大陆全境），包含凤凰卫视中文台、资讯台。

=== 卫星过滤 ===
==== 卫星接收资格 ====
{{Wikisourceshas|卫星电视广播地面接收设施管理规定|广播电视管理条例}}
在中国大陆，一般民众自行安装[[卫星电视|卫星电视]]接收装置属于“非法行为”。根据《卫星电视广播地面接收设施管理规定》（1993年10月5日国务院令第129号）<ref group="註">{{Wikisource-inline|卫星电视广播地面接收设施管理规定}}</ref>和《广播电视管理条例》（1997年9月1日国务院令第228号）<ref group="註">{{Wikisource-inline|广播电视管理条例}}</ref>，只有在接收不到当地电视台、电视转播台、电视差转台、有线电视台（站）的电视节目的地区，个人才可申请安装卫星地面接收设施接收'''境内电视节目'''（限[[中星6B|中星6B]]或[[中星9号|中星9号]]），以及相关解密卡申请（对于卫星用户要收看已加密的CCTV1和其他加密国内频道节目）。个人需设置卫星地面接收设施的，必须经所在单位同意并持其开具的证明，向当地县级以上（含县级）广播电视行政部门提出申请，经地、市级广播电视行政部门和国家安全部门签署意见后报省、自治区、直辖市广播电视行政部门审批。经审查批准的个人，凭审批机关开具的证明购买卫星地面接收设施。卫星地面接收设施安装完毕，经省、自治区、直辖市广播电视行政部门和国家安全部门检验合格后，由省、自治区、直辖市广播电视行政部门发给《接收卫星传送的境内电视节目许可证》。个人设置的卫星接收天线不得占用公共场所、影响环境美观和邻里日常生活。 <ref>{{cite web |title=《卫星电视广播地面接收设施管理规定》实施细则 |url=http://news.xinhuanet.com/newmedia/2003-05/31/content_897368.htm |work=广电总局 |publisher=新华网 |date=2003-05-31 |accessdate=2011-05-16 |archive-url=https://web.archive.org/web/20110907225225/http://news.xinhuanet.com/newmedia/2003-05/31/content_897368.htm |archive-date=2011-09-07 |dead-url=no }}</ref>规定也明确，一经发现个人私自安装卫星装置即处以5千元人民币以下的[[罚款|罚款]]，对单位处以5万元以下的罚款，并没收相关器材。不过在一般情况下，由于接收装置体积小、私密性较好和安装方便，政府部门难以察觉，而且普遍执法并不严格，所以在中国大陆非常普遍。<ref>{{cite web |title=卫星电视天线私装猖獗 有线收费高为重要原因 |url=http://tech.163.com/09/1019/09/5LVPJ17K000915BD.html |work=南方日报 |publisher=网易 |date=2009-10-19 |accessdate=2011-05-16 |archive-url=https://web.archive.org/web/20091022220835/http://tech.163.com/09/1019/09/5LVPJ17K000915BD.html |archive-date=2009-10-22 |dead-url=no }}</ref>

[[中华人民共和国政府|中华人民共和国政府]]对允许设置卫星地面接收设施接收卫星传送电视节目的单位设有相当严格的条件：<ref>{{cite web |title= 申办《广州地区接收卫星传送电视节目许可证》审核程序 |url= http://www.gzpi.gov.cn/gjgwy01/ggfwxwgf/szgdw/xwcbgd/t20050624_10788.htm |publisher= 广州市文化广电新闻出版局 |accessdate= 2014-06-29 |archive-url= https://web.archive.org/web/20061109092949/http://www.gzpi.gov.cn/gjgwy01/ggfwxwgf/szgdw/xwcbgd/t20050624_10788.htm |archive-date= 2006-11-09 |dead-url= yes }}</ref>
#有确定的接收方位、接收内容和收视对象范围；
#有符合国家标准的接收设备；
#有合格的专职管理人员；
#有健全的管理制度。

下列单位和场所可申请设置卫星地面接收设施接收卫星传送的境外电视节目：
#级别较高、规模较大的教育、科研、新闻、金融、经贸等确因业务工作需要的单位；
#三星级或国家标准二级以上的涉外宾馆（限亚太6号）；
#专供外国人和香港、澳门特别行政区、台湾地区人士办公或居住的公寓。

申请设置卫星地面接收设施接收卫星传送境内电视节目的单位，应当提交下列材料：
#申请报告；
#《设置卫星地面接收设施接收卫星传送的境内电视节目许可证申请表》；
#企业法人营业执照或法人代码证。

==== 广电总局批准频道 ====
每年[[国家新闻出版广电总局|国家新闻出版广电总局]]（前身为[[国家广播电影电视总局|国家广播电影电视总局]]）都会发布三星级以上涉外宾馆等单位接收卫星电视的通知，如《广电总局关于2010年度三星级以上涉外宾馆等单位可申请接收的境外卫星电视频道范围的通知》。新闻出版广电总局对已被允许接收卫星电视的单位设有严格限制，规定此类单位接收的境外节目范围限于下列频道，信号来源限于国家新闻出版广电总局境外卫星电视平台（亚太6号，东经134度，[[標準畫質電視|标清]]）。每年新闻出版广电总局会评估境外频道的播放内容，判定下一年会否在亚太6号卫星继续转播。

与其他卫星直播商不同，亚太6号属於中华人民共和国政府垄断性质，其正版解码器及收视卡均由全国各地广电管理部门或官方代理独家销售，其价格相当高昂，每条频道的价格也有所区别，例如单频道HBO，现时为最低每月每个终端23.24元人民币（130个终端起计费，不足130个按130个计算）；而[[凤凰卫视|凤凰卫视]]等总部在香港的频道以及[[今日俄罗斯|今日俄罗斯]]、[[彭博电视|彭博电视]]等少数国外频道则完全免费；此外专用解码器（含IC卡）价格为每台3800元人民币。相比而言，一般直播商通常每月只收取10－100的固定费用，即可收看几十个频道。另外，即使个人/单位获得安装资格，如选择官方指定的安装公司，价格也较高。如申请成为合法的安装公司，更加是没有可能。<ref>{{cite web|title=境外卫星电视为什么只能由一家合法公司安装|url=http://www.rftgd.gov.cn/wsbs/jzxx_view.dy?mid=7429|publisher=广东广播影视网 - 局长信箱|date=2011-10-31|accessdate=2011-10-31|deadurl=yes|archiveurl=https://web.archive.org/web/20140222205744/http://www.rftgd.gov.cn/wsbs/jzxx_view.dy?mid=7429|archivedate=2014-02-22}}</ref>

亚太6号上的凤凰卫视中文台被新闻出版广电总局获准在广东省全省的有线电视网播出，而澳亚卫视中文台、星空卫视和MTV此四条频道只能在珠三角（选传）播出，普通民众可以随意收看。深圳市有线电视公司天威视讯还特别获准在有线电视网播出所有亚太6号的频道，不过需要另外付费申请（获准落地的四个频道无需申请）。<ref>{{cite web|title=频道收费标准|url=http://www.topway.com.cn/sz/sz_ff1.html|publisher=深圳天威视讯|accessdate=2011-05-16|deadurl=yes|archiveurl=https://web.archive.org/web/20110522215803/http://www.topway.com.cn/sz/sz_ff1.html|archivedate=2011-05-22}}</ref>虽然广电总局规定亚太6号的指定频道只准在广东部分城市播放，但中国大陆仍有不少市县或乡镇的广电管理部门或有线电视网络不理会禁令，自行在当地转播亚太6号或其他卫星的任意电视频道，如[[吉林省|吉林省]]有线网吉视传媒延边分公司的高清电视一度可收看韩国的[[KBS第1频道|KBS 1TV]]、[[KBS第2频道|KBS 2TV]]、[[MBC|MBC]]和[[SBS_(韩国)|SBS]]。而酒店除此以外，也会播放一些C波段的免费频道（如[[半岛电视台|半岛电视台]]、[[德国之声电视台|德国之声电视台]]等）以降低成本。而凤凰卫视在全国大多地区是有接收包括其高清频道，但很多地方打“测试频道”EPG名称甚至收费开放（包括北京歌华有线也是收费开放凤凰卫视等境外频道），同样为逃过广电审查，如福建广电网络集团就传输了凤凰卫视两个频道（非亚太6号信号源标清信号）包括其高清频道，但目前仅省会福州地区免费开放（高清频道同样在高清基本包不额外收费），其他市基本收费开放（厦门、莆田分公司只传输省网统一前端的高清频道，同样是收费开放，且不提供其标清频道）。

==== 境外卫星电视控制历史 ====
1949年中华人民共和国建立前，[[中国共产党|中国共产党]]对外宣传的渠道主要是靠官方通讯社[[新华社|新华社]]。1955年12月，中国共产党中央委员会主席[[毛泽东|毛泽东]]要求新华社“把地球管起来”，“尽快做到在世界各地都能派有自己的记者，发出自己的消息”。

1991年1月，成立对外宣传最高机构——中国[[国务院新闻办公室|国务院新闻办公室]]。

1993年3月，成立国务院新闻办公室与中共中央对外宣传小组，“[[一个机构两块牌子|一个机构两块牌子]]”，列入中共中央直属机构序列，由中共中央直接领导。同年7月，[[新闻集团|新闻集团]]董事长[[鲁珀特·默多克|鲁珀特·默多克]]以5.25亿美元，买下香港“卫星电视有限公司”（Star TV，现[[星空传媒|星空传媒]]）63.6%的股权（再於1995年购入其余下股份），不久後其宣称，“卫视对全世界的极权政府构成了明确的威胁”。然而，在他讲这番话几星期之内，同年10月5日，中国国务院颁布了《卫星电视广播地面接收设施管理规定》，明确规定“个人不得安装和使用卫星地面接收设施”，“各地的电视台、有线电视台和电视转播台，一律禁止转播卫星传送的境外播放的电视节目”。<ref name="默多克抨击北京有多疑症">{{cite web |title=默多克抨击北京有"多疑症 |url=http://news.bbc.co.uk/chinese/simp/hi/newsid_4250000/newsid_4259500/4259566.stm |publisher=BBC |date=2005-09-19 |accessdate=2011-11-28 |archive-url=https://web.archive.org/web/20100818102947/http://news.bbc.co.uk/chinese/simp/hi/newsid_4250000/newsid_4259500/4259566.stm |archive-date=2010-08-18 |dead-url=no }}</ref>

为能在中国大陆赢得更大的市场份额，默多克改变其态度，开始尽力迎合中共。1994年4月，曾实播中国人权状况记录片而引起中共不满的英国广播公司（BBC）被新闻集团在亚洲禁播（停止卫星转发），令中国大陆、香港及台湾看不到BBC的节目；<ref>{{cite web|title=默多克批判（上）|url=http://www.culstudies.com/plus/view.php?aid=4429|author=袁爱中|publisher=中国新闻研究中心|date=2005-01-07|accessdate=2011-11-28|deadurl=yes|archiveurl=https://archive.is/20130425163539/http://www.culstudies.com/plus/view.php?aid=4429|archivedate=2013-04-25}}</ref>1996年3月25日，新闻集团全资拥有的香港卫星电视有限公司与两家中方公司合股在香港成立了“凤凰卫视有限公司”。这两家中方公司就是原[[中央人民广播电台|中央人民广播电台]]高级管理人员刘长乐（现为凤凰卫视董事局主席）旗下的“今日亚洲有限公司”和中国银行的“华颖国际有限公司”。1998年8月28日，[[凤凰卫视|凤凰卫视]]开播。

2001年年底，因中国大陆一名记者写了一篇“[[参考资料|内参]]”，说[[上海|上海]]、[[浙江|浙江]]、[[福建|福建]]等沿海地区，私接境外电视信号现象“异常猖獗”，并有向内陆省份蔓延的趋势。随即一场大规模的电视清查行动在全国范围里展开，所有境外电视台受到牵连。凤凰卫视电视信号一夜之间消失，连高校校园里的信号亦被关闭。翌日，[[北京大学|北京大学]]、[[清华大学|清华大学]]几十名教授联名上书国务院指这件事情让人觉得恶心。一周后，凤凰卫视信号在各地高校陆续恢复。<ref name = "tv"/>2002年，当时的国家广电总局（现[[国家新闻出版广电总局|国家新闻出版广电总局]]）允许的境外频道（持有执照）开始被规定必须通过中国官方的卫星放送，这样可以令中华人民共和国政府能够任意关闭或中断信号放送。同年7月，英国广播公司世界频道（现世界新闻频道）卫星信号被广电总局突然关闭，原因是报道含有法轮功的镜头。<ref>{{cite web |title=China blacks out BBC |url=http://news.bbc.co.uk/2/hi/entertainment/2097890.stm |publisher=BBC |date=2002-07-05 |accessdate=2011-05-16 |archive-url=https://web.archive.org/web/20110202154753/http://news.bbc.co.uk/2/hi/entertainment/2097890.stm |archive-date=2011-02-02 |dead-url=no }}</ref><ref>{{cite web|title= 北京切断BBC电视 |url=http://www.voanews.com/chinese/news/a-21-a-2002-07-05-5-1-63298637.html |publisher=VOA |date=2002-07-05 |accessdate=2011-05-16}}</ref>对此中国外交部指，因BBC WORLD（现BBC World News）近期多次违反英国广播公司与中国国际电视总公司达成的有关协议，中国国际电视总公司依据并履行该协议的规定，关闭了BBC WORLD在华落地信号（当时由[[鑫诺一号通信卫星|鑫诺一号通信卫星]]转播，东经110.5度），至于何时恢复，要看BBC对此事的态度后再定。<ref>{{cite web|title= 2002年7月11日外交部发言人在记者招待会上答记者问 |url=http://www.fmprc.gov.cn/chn/gxh/tyb/fyrbt/jzhsl/t3535.htm |publisher=中华人民共和国外交部  |date=2002-07-11 |accessdate=2011-05-16}}</ref>

2002年，因中华人民共和国加入[[世界贸易组织|世界贸易组织]]，为遵守外资媒体可进入中国经营的条文，当局开始特批几条境外娱乐卫星频道在广东省部分地方落地：
2月[[美国在线|美国在线]] / 华纳公司旗下的[[华娱卫视|华娱卫视]]正式在广东播出。

3月28日，新闻集团旗下[[星空卫视|星空卫视]]在广州开播，根据最初广东有线和星空卫视达成的协议，落地第一年星空卫视将被限定在广东省的广州市和[[肇庆市|肇庆市]]播出，以后每年将逐步开放几个城市。这是中国政府第一次允许境外电视频道通过大陆有线电视网落地播出。作为交换条件，新闻集团通过自己的电视网让[[CCTV-NEWS|中国中央电视台英文国际频道]]落地[[英国|英国]]和[[法国|法国]]。同年，[[中共中央党校|中共中央党校]]“科学社会主义专业”博士吴玉荣，提出“国外的传媒特别是一些跨国传媒集团说到底就是一个商业机构，其最终目的就是赚钱。既然是这样，我们就可以通过商业的规则来限制、影响国外的主流媒体，如通过调整与西方传媒公司的商业关系，通过限制或批准这些媒体公司在中国的经济活动，通过购买这些公司的股票来影响这些传媒公司的决策层”。吴玉荣还举了一些成功的经验：“90年代中期我们就先后通过允许新闻集团在中国发展卫星电视而使其终止了反华的BBC的“世界电视节目”，通过限制“迪士尼”在中国的经营活动打击其参与制作分裂西藏的节目也颇有成效”。<ref>{{cite web |title=传媒全球化时代的中国国际形象战略 |url=http://www.chinadaily.com.cn/hqzg/2007-04/16/content_851357.htm |author=吴玉荣 |publisher=中国日报 |date=2007-04-16 |accessdate=2011-11-28 |archive-url=https://web.archive.org/web/20140106032945/http://www.chinadaily.com.cn/hqzg/2007-04/16/content_851357.htm |archive-date=2014-01-06 |dead-url=no }}</ref>

2003年3月星空卫视再获广电总局批准有限落地全国。4月26日，[[全球音乐电视台|全球音乐电视台]]（MTV）获准在广东珠三角落地。10月8日，默多克应时任中共中央党校校长[[曾庆红|曾庆红]]邀请，向“中共掌管意识形态的高级官员发表的演讲”，其演讲词被时任[[中国人民大学|中国人民大学]]新闻学院教授，现新闻学院副院长喻国明形容为：“煞费苦心的、充满讨好意味的话语”，可以“深切地感受”到演讲者对于进入中国传媒市场的“渴望和焦虑”，但是“如果抛开其功利性的图谋不论”，“许多论点是深刻并且中肯的，特别是对于我们把握传媒产业的真正价值是具有相当大的启发意义的”。<ref>{{cite web |title=鲁伯特•默多克：文化产业的价值 |url=http://www.people.com.cn/GB/14677/22114/41180/41185/3015288.html |publisher=人民日报 |date=2003-10-08 |accessdate=2011-11-28 |archive-url=https://web.archive.org/web/20110716075527/http://www.people.com.cn/GB/14677/22114/41180/41185/3015288.html |archive-date=2011-07-16 |dead-url=no }}</ref>9日，时任中共中央政治局常委主管宣传的[[李长春|李长春]]在人民大会堂会见了默多克一行，李希望新闻集团在促进“让世界了解中国和让中国了解世界方面做出更大的努力”。<ref>{{cite web |title=李长春会见默多克 希望新闻集团与中国多合作 |url=http://news.xinhuanet.com/newmedia/2003-10/10/content_1116251.htm |publisher=新华社 |date=2003-10-10 |accessdate=2011-11-28 |archive-url=https://web.archive.org/web/20110905104322/http://news.xinhuanet.com/newmedia/2003-10/10/content_1116251.htm |archive-date=2011-09-05 |dead-url=no }}</ref>同年，新华社副总编辑夏林在“‘喉舌’的力量”一文中提出“信息是一种战略资源。谁掌握了信息传播源和信息传播载体，谁控制了国际广播、卫星电视和互联网络，谁就有能力影响整个社会，谁就能在意识形态的斗争中，抢夺并占领住制高点，掌握主动权”。

2004年，中华人民共和国政府没有按照当初的承诺，允许星空卫视在大陆其他城市落地。新闻集团因不满足星空卫视只能在广东省和在全国三星级以上宾馆和涉外小区落地的局面，与[[青海卫视|青海卫视]]合作，买断青海卫视每晚结束转播央视《[[新闻联播|新闻联播]]》以后的时段，以此方式曲线在中国大陆全面落地，播放星空卫视节目和广告，但这种曲线落地只保持3个月便被广电总局紧急叫停，新闻集团为此投资1亿美元，广电总局喊停后导致5000万美元坏帐。<ref>{{cite web|title=星空卫视否认撤离大陆市场 目前公司运转正常|url=http://sh.sina.com.cn/news/20080410/100689921.shtml|work=新闻晨报|publisher=新浪网|date=2008-04-10|accessdate=2011-05-16|archive-url=https://web.archive.org/web/20140106040209/http://sh.sina.com.cn/news/20080410/100689921.shtml|archive-date=2014-01-06|dead-url=no}}</ref>同年10月28日，由广电总局、中华人民共和国商务部发布的《中外合资、合作广播电视节目制作经营企业管理暂行规定》（44号令）出台，<ref>{{cite web |title=中外合资、合作广播电视节目制作经营企业管理暂行规定 |url=http://news.xinhuanet.com/zhengfu/2004-11/30/content_2275821.htm |publisher=新华网 |date=2004-11-30 |accessdate=2011-11-28 |archive-url=https://web.archive.org/web/20051116125354/http://news.xinhuanet.com/zhengfu/2004-11/30/content_2275821.htm |archive-date=2005-11-16 |dead-url=no }}</ref>自同年11月28日起实施。《规定》包括“不得设立外商独资广播电视节目制作经营企业”，“法定代表人须由中方委派”和“合营企业中的中方一家机构应在合营企业中拥有不低于51%的股份”，“合营企业不得制作时政新闻和同类的专题、专栏节目”等一系列对外资企业的苛刻条例。

=== 广东省 ===
{{seealso|广东电视广播#历史#香港地面电视引进}}
广州市在1998年之前的香港地面电视审查画面非常简陋，当敏感话题或新闻提及时，监管部门会立即中断电视信号，变成了七彩[[测色板|测色板]]（俗称彩条信号），到后来开始出现广告、山水画、旅游胜地图片。由于当时广东省各地市仍然可以自行在当地接收香港地面电视信号，故审查并非全省同步，有些地方会出现没有过滤的情况，有些地方就用自行制作的静态公益宣传语风景画遮盖，方式不尽相同。例如广州两家有线电视网转播的[[翡翠台|翡翠台]]的审查也不一定一样。

新闻出版广电总局要求广东省广播电影电视局需确保香港地面电视四条频道保持有效统一地审查，所以现时广东全省的香港四频道（现时是二频道）的审查工作由广东有线广播电视网络股份有限公司（简称广东有线，总部於南方电视台，属南方广播影视传媒集团）负责，所有市县需使用由广东有线发出的已过滤信号。四频道（现时是二频道）在广东有线接收地在中山市[[五桂山|五桂山]]的电视发射站，全部为[[数码电视|数码电视]]信号，信号接收后通过光纤传至广州市南方电视台（现时是广东广播电视台）过滤后再向全省发送。

对于此四条频道（现时是二条频道），广东有线及各地下级转播电视台会对其插播本地广告以谋取利益，具体方法是通过各类广告中介公司进行招商<ref>{{cite news |url=http://www.jiarui.cc/cn/priceview.asp?priceid=6313 |title=2011广东电视台省网翡翠台广告报价 |work=嘉瑞传播 |accessdate=2011-08-31 |archive-url=https://web.archive.org/web/20140224003527/http://www.jiarui.cc/cn/priceview.asp?priceid=6313 |archive-date=2014-02-24 |dead-url=no }}</ref>。广州方面有三种选择，第一是境外电视直接支付落地费，第二种是分割一部分广告时间，第三种是广东省支付版权费购买其节目在广东地区的播映权，并拥有在节目中插播广告的权利。目前，TVB与广东省政府签的协议是第三种，黄金时段基本都是插播广告，而绝大部分时候地方电视台也利用此机会进行插播当地赞助商的广告，镇区电视广播站也跟着一起插播广告，当最上层转回正常播放剧集时，第二层也坚持播放完毕赞助商的广告为止，第三层也是如此，所以短短十五分钟小节剧集，最后仅变为八分钟左右，有许多紧张剧情观众因而被逼错失（不过，现广东省内各市区观众使用广东广电网络U互动回看TVB节目的，则是省线的广告）；而亚视是选择第二种分割广告时段，有一半时间保留原版广告；凤凰中文台属于第一种，完全无插播（类似国内卫视频道那样的落地形式）。另外，有些地方（如[[石龙镇_(东莞市)|东莞市石龙镇]]）更会插播当地电视台制作的本地新闻，形式层出不穷。<ref>{{cite web|title=石龙新闻播放时间|url=http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=17443|publisher=广东广播影视网 - 咨询投诉|date=2012-02-04|accessdate=2012-03-20|deadurl=yes|archiveurl=https://web.archive.org/web/20140222205739/http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=17443|archivedate=2014-02-22}}</ref><ref>{{cite web |title=关于各大电视频道经常插播四会新闻的投诉 |url=http://sp.gdsihui.gov.cn/communicateWebAction.do?action=viewDetail&forwardName=appViewDetail&id=5036 |work=四会市行政审批网 |publisher=archive.today |date=2013-01-10 |accessdate=2013-03-14 |deadurl=yes |archiveurl=https://archive.is/20130424150018/http://sp.gdsihui.gov.cn/communicateWebAction.do?action=viewDetail&forwardName=appViewDetail&id=5036 |archivedate=2013-04-24 }}</ref>新闻出版广电总局对大陆电视台的任何广告条例，对港台和境外电视台无效。<ref name="tvb片尾"/>但有的地方同样不理会禁令，连同国内一线卫视台也一贯插广告的。

虽然新闻出版广电总局只批准广东省珠三角播放的亚太6号四条频道，但广东省各市县或乡镇并不一定按照其规定执行，有些地方有线电视网会有此四台外的其他卫星频道，如茂名市就经常转播香港有线的频道。有些非珠三角的县市也曾经有此四台，但后来被取消。

广东有线其余播放的境外卫星频道也经过延时，画质和音质会比原版为差。

==== 广东省广州市 ====
{{See also|广州电视广播}}

广州市共有两间有线电视公司，[[广东有线|广东有线]]和[[广州珠江数码集团有限公司|广州珠江数码集团有限公司]]（[[广州有线|广州有线]]，总部于[[广州市广播电视台|广州市广播电视台]]），简称“省线”及“市线”。

2003年非典型肺炎（SARS）的爆发，广州市人民政府需要大量过滤来自香港的新闻资讯，开始使用香港特区政府宣传片（或当地公益广告）过滤。至今，已经发展到预先录制相同电视频道的节目来拦截整个“不当”的时政节目。当局曾于2006年“六四”期间以预先录制好的一个美食节目，来过滤[[香港电台|香港电台]]制作的《[[頭條新聞_(香港電台)|头条新闻]]》。另外，當香港電視頻道播出某些在大陆被禁的電影，以及宗教类節目（如亞視的《恩雨之聲》和《星火飛騰》），亦被广电部门換上其他節目。

最初的电视审查也只过滤中文电视台的新闻。后来也开始对英文电视台进行过滤。而英文台新闻的过滤仅针对于画面内容而言，即使英文台节目的聲音內容属过滤范围，但画面片段不需要过滤，仍然可以收看而不会过滤。

現在，電視節目的屏蔽已經开始细致化。《[[亞洲電視夜間新聞|亞洲電視夜間新聞]]》自2009年2月2日改版後，因其節目下的[[新聞跑馬燈|新聞跑馬燈]]會不時顯示大陆的敏感信息可能造成這些文字新闻被內地民眾接收，而攔截這些文字的工作量也會比較大，因此廣東省在播出亞視夜間新聞的時候會用白/蓝色僅僅屏蔽新聞跑馬燈，其他的畫面不做屏蔽，這個現象可以在中場[[廣告|廣告]]時間前的下節預告時發現。但没多久就没有長期屏蔽跑马灯，僅在有敏感新闻的出現才会临时用白/蓝色屏蔽新闻跑马灯内容。

2009年6月12日前，市线和省线都是使用两条独立的信号源。市线使用自行录制的2004年香港政府宣传片（包括[[陈慧琳|陈慧琳]]为代言人的“职业英语运动”等等）屏蔽敏感片段，而省线使用较新的香港政府宣传片屏蔽敏感片段。由于有线电视网审查境外电视节目只有大约20秒的判断时间，容易造成误判或漏判。2009年6月六四事件20周年期间某天，市线因漏屏蔽了约十秒的出自翡翠台《[[六点半新闻报道|六点半新闻报道]]》的报道六四烛光晚会片段，導致部分編輯被停職或處罰，市线也因此失去了独立接收香港电视信号的权利。报道指，监督播放的助理编辑因饮水疏忽，没有及时切断节目中报道香港6月4日[[維園六四燭光晚會|維園六四燭光晚會]]的内容，导致晚会的画面被转播出去，估计有数百万广州市民看到了有关内容。虽然播出的时间只有10秒钟，但[[中国共产党广州市委员会|中国共产党广州市委员会]]宣传部视为“重大政治事件”，派人进行调查处理，当晚值班的广州电视台（现广州市广播电视台）两名编辑和三名助理被停职。电视台一名负责人对信息中心证实了这起事件，称事件涉及政治错误。台湾“[[中央社|中央社]]”说错误转播的广州电视台的有线频道，电视台5名编辑遭到停职处分。中共广州市委宣传部证实广州电视台的误播事件，并说：“我市广播电视行政管理部门的意见是，按照国家广播电视管理条例的规定，依法处理。”但广州电视台依然否认发生过这个事故。香港《[[苹果日报|苹果日报]]》指，广州电视台就曾发生过类似疏忽导致发生政治事件。2001年3月5日，该台正在转播时任中国国务院总理[[朱镕基|朱镕基]]在九届全国人大四次会议上作政府工作报告，突然出现“前[[法轮功|法轮功]]练习者”的字幕，虽然只持续了一秒多钟，但事件惊动中共中央高层。事后调查发现，是电视台新闻部技术人员按错键，字幕人员忙中出错所致。涉案工作人员事后被开除，台长和新闻部主任被降职。<ref>{{cite web |title= 广州承认错播香港六四画面 |url= http://www.voachinese.com/content/a-21-w2009-06-19-voa22-61399882/1031039.html |publisher= VOA |date= 2009-06-19 |accessdate= 2014-06-29 |archive-url= https://web.archive.org/web/20160305020652/http://www.voachinese.com/content/a-21-w2009-06-19-voa22-61399882/1031039.html |archive-date= 2016-03-05 |dead-url= no }}</ref>

新闻出版廣電總局規定從2009年6月12日凌晨起，广东省各市县转播播香港地面电视四条频道必須採用廣東省有線電視作為信號源，原廣州有線明珠台的[[丽音|丽音]]频道，因此措施而消失。自此，市线会双层插播省线播出的商业广告和省线的宣传广告，如果市线原有的广告播完，省线还正在插播省线的广告或香港的广告时，便会插播广州台自行制作的公益广告、广州台节目宣传片或自行录制的2003年至2004年香港政府宣传片，但屏蔽敏感片段的一般是来自省线提供的2016年至2017年香港政府宣传片，市线有时也会双层插播。另外，市有線的境外電視频道的画质（顏色）和声音也开始比省有线更差，翡翠、明珠台台徽[[三基色|三基色]]中的紅色經常變成黑白，這是有線電視先經省線再經市線轉播和延時信號損失所致，两层（双重）延时插播是其中一个主要因素。为了覆盖省线广告，市线的广告时间因此更长，几乎所有节目（除了深夜播出的节目外）都有插播超时的情况。

=== 其他特点 ===
世界上很多国家和地区的电视或电台的管理制度规定，从电台或电视台发出信号到观众收到信号，规定有几秒钟的缓冲时间，以应对紧急情况，例如直播场合，有人说脏话（粗口）。中国大陆的“进阶”审查通常就发生在这几秒钟内，而这些一般是利用电视台延迟信号的工具来完成的。早前这种延时只有十秒，现在的延时已达到15-20秒。因此，这种审查没有预警性，通常审查仅会审查敏感内容。但这也很拙劣，比如亚视新闻播出前曾发生过国歌节目（为香港政府宣传片《心系家国》系列）有最后10秒是重复播出的，也有TVB的《[[六点半新闻报道|六点半新闻报道]]》最后片段没有播尽就直接插入本地广告，甚至有节目已经开始播出却在此时插播商业广告的情况。这都能看出节目延时的情况。而部分节目如香港电台制作的《头条新闻》、亚视自制的《[[东宫西宫TV|东宫西宫TV]]》及《[[我要做特首|我要做特首]]》等，或转播国外电视台的新闻，才会以较早前录制的同频道饮食节目（深夜为大陆电视广告）完全覆盖过滤。例外的是，在2005年前中共中央总书记[[赵紫阳|赵紫阳]]病逝当天的TVB《六点半新闻报道》就曾从片头预告一直过滤至接近尾声的体育新闻才恢复正常播映。广州电视台播出部工作人员曾撰写了一篇名为《有线电视延时监控播出及其信号处理》的论文，详细说明了该系统是如何工作的。文章指，广州电视台播出部为保证电视节目播出的安全性，会对某些特殊频道采取延时监控播出。<ref>{{cite web |title= 有线电视延时监控播出及其信号处理 |url= http://edu.wanfangdata.com.cn/Periodical/Detail/dsjs200112019 |author= 林承德 |publisher= 万方数据 |date= 2001-12-01<!--年12期--> |accessdate= 2014-06-29 |archive-url= https://web.archive.org/web/20150503174215/http://edu.wanfangdata.com.cn/Periodical/Detail/dsjs200112019 |archive-date= 2015-05-03 |dead-url= no }}</ref>现时电视台使用ASI单机码流插播系统。

審查雖然有一致準則，但一般人較難掌握到哪些片段必會被過濾，況且並非所有「違反規定」的片段均會被過濾。如2008年1月香港的[[民間電台|民間電台]]事件，電台在鬧市廣播和電台主持人的訪問片段，會被過濾；但其他片段欲如常播放。但也有错误的时候。如2005年[[香港立法会|香港立法会]]全体议员到广东省考察，在对香港立法会议员[[梁国雄|梁国雄]]的采访时就没有过滤其谈论六四事件的片段。而2008年[[北京奥运会|北京奥运会]]前[[:n:北京奧運發售門票現場混亂公安與港記者推撞|香港记者与北京公安民警推撞]]的事件，最初中午的新闻片段没有过滤，但晚上的片段就被过滤，而次日播出的片段就没有再过滤。而2016年3月5日的亚视[[亚洲电视欠薪及不获续牌事件|不获续牌]]事件，广东有线试过过滤了本港台和翡翠台的相关新闻片段，但凤凰卫视的相关新闻片段并未过滤。

新闻的过滤时宽时紧，亦跟幕后监控人员的认知有关，曾有香港[[法轮功|法轮功]]学员摆街，以及梁国雄穿着“六四”衣服的画面没有被过滤，但部分[[新华社|新华社]]有报道的敏感性新闻（特别跟西藏有关的新闻），全部片段都被过滤，甚至连官方的文字报道亦不例外。

另外广东省人民政府的審查手法亦被指出版权争议，其被指違反香港政府、電視台和電影公司版權。廣東广电部门插播的香港政府宣傳片，並未經香港政府批准播映。此外，因為宣傳片大多从香港本地電視台錄製，片段前後可隱約看到電視台的節目宣傳片段或其他廣告。用來插播被禁的節目的其他節目都是電視台播映的電影或自行製作的節目。

2008年的全國哀悼日中，虽然廣東有線在香港四条频道中转播CCTV-1，但香港四条频道当中的新闻节目和爱心募捐节目照常播出。2010年玉树地震、甘肃泥石流后的国家哀悼日，香港四条频道被全部禁止除新聞外所有節目和廣告的播放，並顯示“在国家哀悼日期间不准播出娱乐及商业广告”的藍底白字畫面，卫星频道则仅保留凤凰卫视中文台。<ref>{{cite web |title= 三大报章哀悼日头版现胡锦涛笑脸引热议 |url= http://www.rfa.org/mandarin/yataibaodao/hu-08162010094048.html |publisher= RFA |date= 2010-08-16 |accessdate= 2011-05-17 |archive-url= https://web.archive.org/web/20150522155323/http://www.rfa.org/mandarin/yataibaodao/hu-08162010094048.html |archive-date= 2015-05-22 |dead-url= no }}</ref>3次哀悼日中，亚太6号卫星境外平台亦停止所有频道转播，全部改转播CCTV-1并伪造对应境外频道台标。

=== 大陆其他城市 ===
{{See also|有线电视前端插播广告}}

除了[[中央電視台|中央電視台]]的所有頻道之外，其他頻道均經常被[[地方电视台|地方电视台]]直接插广告，特别是一线卫视，经常被插医疗广告。而根據當地的環境，層級由高到低的省市政府均有自己設立的地方電視台，這些電視台一層一層地插播廣告，直至縣區政府也有自己的插播廣告，到當地的有線用戶收看效果最差。而且各地政府各自為政，對外地電視台有諸多限制，只能接收衛視頻道。而中央電視台也借廣電總局對各地的電視台有嚴格限制，不許其製作及廣播[[方言|方言]]節目（廣東省等特殊地方除外），限制[[選秀|選秀]]節目、甚至威脅中央台收視的節目。此類節目在[[北方城市|北方城市]]一旦出現並觸動中央台，則會被截停信號或事後在頻道表中刪除。虽然有已不少用戶多次投诉，不过始终沒有得到合理答復。

在部分地区，很多一二线的广告和部分节目也会遭到地方电视台的拦截，因为一线卫视的节目在中国大陆收视率较高，因此在部分地区就会[[插播广告|插播广告]]，以吸引观众可信，因为地方电视台在湖南卫视插播的广告会具有明显的地方特征例如使用粤语广播和打出本地的电话号码或者本地地址等，因此当地群众能够分辨出地方台插播的广告特征。甚至对央视和其他收视率高的省级卫视以及地面频道（如广东珠江、湖南经视）也会被插播当地广告。另外在停机检修和转播《新闻联播》时段也会被插播广告和节目。

目前，西藏自治区墨脱县电视台会在黄金时段，在有线前端的湖南卫视插播《墨脱新闻》等自办节目，当然也遭受到了一些网友的谴责。在国内在某些因政策或资金问题而不能自立频道的市或县，会在黄金时段占用一些一线卫视或公共频道播放当地的新闻和节目，甚至在一些有本地频道但收视低的地区也会发生，例如在广东一些地区甚至还会占据凤凰卫视、澳亚卫视、珠江台、广东公共频道、CCTV-7等指定的频道进行占领节目和广告。

== 数字和高清电视 ==
===中国大陆===
中国大陆所有已开通高清电视的有线电视网只有中国大陆各电视台和各地有线电视网络公司自办的[[高清晰度電視|高清频道]]，并且一般需要另外申请安装高清机顶盒，除央视和各省级卫视的高清频道外，其他专业节目则一般需要另外收费，一般普通的有线电视[[机顶盒|机顶盒]]很少[[16:9|16:9]]的频道。虽然凤凰卫视现在是16:9形式播出，不过很多地方为了降低成本而转播了原版（非亚太6号版），而亚太6号版的则压缩成4:3的，但目前有的地区也开始测试播出凤凰卫视的两条高清频道（即凤凰中文HD、凤凰资讯HD），凤凰卫视香港台在2014年停播了标清信号。

目前，[[厦门电视台|厦门电视台]]各套频道、[[湖南广播电视台|湖南广播电视台]]各套地面频道及数字频道快乐垂钓频道、[[石家庄广播电视台|石家庄广播电视台]]新闻综合频道、上海[[SMG|SMG]]部分地面频道（含上海新闻综合、电视剧、[[五星体育频道|五星体育频道]]、ICS）等地面频道，以及全国部分上星购物频道已经16:9播出，而且很多是高清频道开播开始而过渡的，即高标清共用字幕机。将来还有更多频道标清信号过渡16:9播出兼高清开播。但与香港和国外不同的是，中国大陆上的16:9只是形式主义，而真正标清信号的传输模式依旧是4:3，故中国的16:9标清频道在标清机顶盒是压扁播出；而全国大多地方接收的[[凤凰卫视|凤凰卫视]]（亚太7号原版）标清信号，在标清机顶盒则是上下黑带或切边播出。

2015年7月2日，[[东南卫视|东南卫视]]高清开播，与标清频道共用字幕机，故标清16:9播出。而在2015年8月17日开始，中央电视台各套公共免费频道也全部16:9播出，但并非完全16:9，因为囊括4:3节目是拉伸播出，只是台标缩小成16:9格式，且并不是与高清共用字幕机。之后[[安徽卫视|安徽卫视]]在[[安徽电视台|安徽电视台]]搬迁新址开始也开始形式上16:9播出，仅保留部分新闻节目依然是切边播出（兼顾安徽公共频道并机直播和下辖市级台转播，但2016年《超级新闻场》改为16:9挤压播出），且台标尺寸依然是4:3并未完全16:9格式。[[辽宁卫视|辽宁卫视]]在2015年12月31日晚22点标清开始16:9播出。[[北京卫视|北京卫视]]在2015年9月2日为阅兵式而标清临时过渡16:9，后变回4:3。2016年，[[天津卫视|天津卫视]]、[[东方卫视|东方卫视]]、[[兵团卫视|兵团卫视]]、[[河南卫视|河南卫视]]、[[青海卫视|青海卫视]]开始16:9播出，2016年8月31日[[湖南卫视|湖南卫视]]标清16:9播出，之后[[山东卫视|山东卫视]]16:9播出；2017年，[[广东卫视|广东卫视]]、[[贵州卫视|贵州卫视]]、[[江苏卫视|江苏卫视]]16:9播出。将来还会有更多的台逐渐过渡。因部分频道标清设备老旧原因，导致已经过渡16:9播出的标清频道至今存在拉伸或切边节目。

==违规传送频道整顿==
2014年3月，廣東有線電視東莞分公司稱，[[高清翡翠台|高清翡翠台]]、[[J2|J2]]、[[亞洲電視亞洲台|亞洲台]]是在前期作為測試頻道試播，後應國家政策的要求停止播出，以粵語高清HD（由深圳天华公司制作的轮播频道）和数码信号本港台墊播，中國內地未經政策允許，不得接收當局批准外的其他境外頻道。廣東省廣播電影電視局也轉發了這條答复。<ref>{{cite news |title=国内第二批高清何时上星 |url=http://wz.sun0769.com/index.php/question/show?id=234196 |publisher=阳光热线问政平台 |date=2014-03-07 |accessdate=2014-06-25 |archive-url=https://web.archive.org/web/20140819085834/http://wz.sun0769.com/index.php/question/show?id=234196 |archive-date=2014-08-19 |dead-url=no }}</ref><ref>{{cite news|title=关于高清问题|url=http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=20963|publisher=廣東省廣播電影電視局|date=2014-04-14|accessdate=2014-06-25|deadurl=yes|archiveurl=https://web.archive.org/web/20140819084505/http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=20963|archivedate=2014-08-19}}</ref>

2016年7月15日，广东南方传媒IPTV下架了[[凤凰卫视中文台|凤凰卫视中文台]]、[[澳亚卫视|澳亚卫视]]等境外频道（由于澳亚卫视含有广州电视台节目宣传的插播，故为广州珠江数码信号盗播所为，之后被广州有线方告发后而实行），中国广电总局规定，未经允许禁止在IPTV平台播出境外频道。同日，湖南IPTV也下架了凤凰卫视的两个频道（在传送期间则是台标盖上贴膏药“测试”两字）。福建IPTV（由福建省广影集团、福建电信联合百视通挂靠运营）于2017年5月27日上线凤凰卫视两套高清频道（且不提供回看功能），但没多久于7月21日下线。

2017年4月，为迎接“[[中共十九大|中共十九大]]”，再加上前期“限韩令”的规定，广电总局严查整顿有线网违规传输频道，多个省市有线网下架DOX、天华等未备案频道。这些高清数字付费轮播频道均未经备案（SITV文广高清除外），根据规定默认备案为PushVOD频道的（但亦可以在OTT和IPTV平台播出），却违反广电总局规定擅自更改成高清数字付费轮播频道透过国干网进行传送销售（易导致全国一些有线电视网络无视央视高清频道又带宽问题无法满足观众需求）。而SITV欢笑剧场由于是合法备案频道不受影响，因“限韩令”原因取消“韩流”频道定位，于2017年3月起改播港剧为主体；“DOX韩流”于2017年3月12日更为“DOX雅趣”，但由于违规定位模式已被通报陆续下线，同华数、天华等频道一同下线并去除台标（仅保留有备案的精彩影视和求索纪录）。

另外一些未经允许传送的境外频道也在多个地方陆续下架，其中包括[[凤凰卫视|凤凰卫视]]，但极少部分省市因特殊要求，在过了安全大检查之后又恢复传送凤凰卫视等节目，或者放到互动轮播专区播出（需要开通互动点播业务）。凤凰卫视目前于2017年设立面向全国广电网络有线电视互动业务的“凤凰专区”，目前已对接多半地区，通过该互动专区亦可以收看凤凰卫视电视直播，以及新闻资讯和节目的点播。同年9月，因迎接金砖会议，福建省政府和厦门市政府同样对广电方面进行安全大检查，福建广电网络厦门分公司于9月2日凌晨单方面暂停播出凤凰卫视节目，并用蓝色画面覆盖一直到9月7日恢复播出（而福州市以及福建全省的凤凰卫视信号亦未受影响）。

2017年6月22日，由于广电总局已于近日发函，要求[[凤凰网|凤凰网]]在不具备《信息网络传播视听节目许可证》的情况下开展视听节目服务予以关停整顿，故凤凰卫视在凤凰网的直播源、点播节目暂时下架予以整顿，待整顿完成后恢复上架。7月31日，凤凰卫视部分节目在凤凰网恢复上架，不过恢复上架的仅有5档，分别是《中国新移民》、《全媒体大开讲》、《鲁豫有约》、《问答神州》和《金石财经》。目前凤凰卫视部分点播节目因迎接十九大，现已再次全部下架。

2017年9月12日，有观众突然反映[[凤凰卫视|凤凰卫视]]突然消失了《[[锵锵三人行|锵锵三人行]]》节目，之后新浪微博该节目官微亦发布了消息“因公司节目调整，锵锵暂时停播，感谢大家多年厚爱，后会有期”同样遭到网友争议，依中国现状来看，不排除是跟迎接十九大召开准备有关。据香港《苹果日报》援引凤凰卫视内部人员信息称，此次停播是[[广电总局|广电总局]]所下达的命令（实际上中国广电总局并管不到港澳台和国外频道，但在十九大期间，防止敏感言论触发和凤凰卫视领导团队所牵连的责任而已，而在新闻资讯类专题栏目当中，涉及敏感的在中国国际电视总公司转播平台会逐步审查过滤）。此前[[新浪微博|新浪微博]]已经对该节目关键词全网屏蔽，凤凰视频网站全网下架该节目，中国大陆其他第三方視頻、音頻網站也全面下架该节目的视频（而全国各地广电有线电视网络的“凤凰专区”未受影响）。同时暂停播出的节目还有《[[时事辩论会|时事辩论会]]》和《[[震海聽風錄|震海聽風錄]]》，另外其他節目也遭到警告和約談。有消息称，《[[一虎一席談|一虎一席談]]》和《[[軍情觀察室|軍情觀察室]]》也会暂停播出<ref>[http://hk.apple.nextmedia.com/international/art/20170913/20151176 鳳凰衛視遭封殺《鏘鏘三人行》停播 網民斥︰新焚書坑儒] {{Wayback|url=http://hk.apple.nextmedia.com/international/art/20170913/20151176 |date=20170929130022 }}，《苹果日报》，2017年9月13日。</ref>。

== 法律规定相关答复 ==

当民众向广东省广播电影电视局质疑有线电视台在境外频道的处理手法时，当局一般都不会直接回应，只解释：<ref>{{cite web|title=广州对香港电视节目屏蔽令人讨厌及愤怒|url=http://www.rftgd.gov.cn/wsbs/jzxx_view.dy?mid=7467|publisher=广东广播影视网 - 局长信箱|date=2011-08-09|accessdate=2011-10-31|deadurl=yes|archiveurl=https://web.archive.org/web/20160304072505/http://www.rftgd.gov.cn/wsbs/jzxx_view.dy?mid=7467|archivedate=2016-03-04}}</ref><ref>{{cite web|title=“敏感”内容必须有一个严格界定|url=http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=16985|publisher=广东广播影视网 - 局长信箱|date=2011-10-27|accessdate=2011-10-31|deadurl=yes|archiveurl=https://archive.is/20120805171837/http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=16985|archivedate=2012-08-05}}</ref>

{{Cquote|经国家广电总局批准在广东部分地区落地的境外电视频道，按照双方的协议，其电视节目在进入我省有线电视网播出之前，由我省有关单位对国务院《广播电视管理条例》和广电总局明确禁止播出的内容实行遮盖，目的是为了维护国家利益和文化安全，并保证与境内电视频道内容管理执法的一致性。请对此予以理解，并支持广电行政部门依法管理，共同维护国家安全和社会稳定。}}

对于星期一至五在翡翠台剧集即将结束上升字幕时至预告之间的30秒节目时段内被插播广告（台徽未消失），广电部门一般也是三缄其口应对，有时指“会要求辖下广电单位或有线电视公司整改并会随后附上反馈或调查意见”，但通常都没有下文；有时又会指“在境外频道正常节目时段（非时政内容）插播商业广告也是合法的”，新闻出版广电总局的规定对其无效；有时更称由於是人工操作，因没有节目表而出现“失误”：<ref name="tvb片尾">{{cite web|title=tvb片尾插播广告|url=http://www.rftgd.gov.cn/wsbs/jzxx_view.dy?mid=17287|publisher=广东广播影视网 - 咨询投诉|date=2012-01-04|accessdate=2012-03-19|deadurl=yes|archiveurl=https://web.archive.org/web/20120719070427/http://www.rftgd.gov.cn/wsbs/jzxx_view.dy?mid=17287|archivedate=2012-07-19}}</ref><ref>{{cite web|title=广东有线（省台）不死没有用，愿早死早着|url=http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=17087|publisher=广东广播影视网 - 咨询投诉|date=2011-11-16|accessdate=2012-03-19|deadurl=yes|archiveurl=https://web.archive.org/web/20140222205747/http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=17087|archivedate=2014-02-22}}</ref><ref>{{cite web|title=有没搞错啊|url=http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=17580|publisher=广东广播影视网 - 咨询投诉|date=2012-02-23|accessdate=2012-03-19|deadurl=yes|archiveurl=https://web.archive.org/web/20140222205841/http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=17580|archivedate=2014-02-22}}</ref>

当被问到境外电视在广东省落地并与广东省签定的相关协议可能被其滥用，并提出是否应公示出来时，广电部门就声称该协议是属于不能对外公开的范畴。<ref>{{cite web|title=境外电视在我省落地与我省签定了相关协议|url=http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=17776|publisher=广东广播影视网 - 咨询投诉|date=2013-01-10|accessdate=2013-03-14|deadurl=yes|archiveurl=https://web.archive.org/web/20140222205741/http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=17776|archivedate=2014-02-22}}</ref>

因广电部门一般只对新闻政治类内容进行严格监管，但对随意插播广告的漠视，或发现後也没任何处罚措施，故而对有线电视没有形成阻吓力（大部分有线电视台实际也是国营单位），甚至连监管部门同样位于有线电视台内（广东省广播电影电视局位于广东台/南方台大院内），有“自己人管自己人”之嫌，故而广东省各地民众经常投诉有线电视台胡乱插播的情况基本没有得到改善。

== 影响 ==
在无法访问国际互联网或无法登录被[[防火长城|防火长城]]封锁的电视台网站的情况下，这种审查方式对没有亲友在中国大陆境外的群众是极其有效的。有大陆网民笑称审查为“被插（播）”，也有不少广州市网民在[[微博|微博]]上以香港封台广告词接龙的方式讽刺审查。<ref>{{cite news|url=http://news.sina.com.cn/o/2011-08-04/081322932252.shtml |title=“迎客松”当歌唱，这是个奇迹 |work=南方都市报|date=2011-08-04 |accessdate=2011-08-31|archiveurl=https://archive.is/20170523165948/news.sina.com.cn/o/2011-08-04/081322932252.shtml|archivedate=2017-05-23}}</ref>香港部分电视台包括凤凰卫视中文台在内所有自行制作的节目都可以通过卫星天线收看，无干扰影响；也可以从其网站上面付费下载重新收看。

由于境外电视频道无法在一般民众中普及，像美国国家地理频道、探索频道这类电视台只能和中国大陆各电视台合作，授权其播放节目。
