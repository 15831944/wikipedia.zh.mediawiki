'''中国（陕西）自由贸易试验区'''（{{lang-en|China (Shaanxi) Pilot Free Trade Zone}}，简称'''陕西自贸区'''）是中国大陆第三批[[自由贸易园区|自由贸易园区]]之一，也是中国至今为止唯一一个位于[[中国西北地区|西北地区]]的自贸区<ref name=ydyl>{{cite web | url=http://news.china.com.cn/rollnews/jiaodian/live/2017-05/05/content_38198370.htm | title=陕西自贸区：打造“一带一路”上的超级引擎 | publisher=新华网 | date=2017-05-05| accessdate=2017-05-05}}</ref>。陕西自贸区于2016年8月由国务院获批，2017年4月1日正式挂牌成立<ref name=sina>{{cite web | url=http://finance.sina.com.cn/roll/2016-12-15/doc-ifxytqqn8574091.shtml | title=第三批自贸区方案发布在即 最快有望在下月挂牌 | publisher=经济参考报| date=2016-12-15 | accessdate=2017-05-04}}</ref>。陕西自贸区涵盖三个片区，包括中心片区、[[西安国际港务区|西安国际港务区]]片区和[[杨凌农业高新技术产业示范区|杨凌示范片区]]。

==历史沿革==
2013年，陕西提出了自贸区建设的申请<ref name=ydyl/>。2014年3月，西安市市长[[董军|董军]]在全国人大陕西代表团全会上表示，推进[[一帶一路|丝绸之路经济带]]建设是事关全局、影响深远的重大战略。而他认为，从整体上看，经济带还存在区域功能定位趋同、产业布局重叠、同质化竞争等问题，因此他建议国家设立西安自由贸易试验区，使之成为联通欧亚、承东启西的丝绸之路经济带的枢纽<ref>{{cite news | url=http://www.chinanews.com/gn/2014/03-05/5915656.shtml | title=西安市长建议设丝绸之路经济带西安自由贸易试验区| publisher=中国新闻网| date=2014-03-05 | accessdate=2017-05-04}}</ref>。

2016年8月，国务院批准陕西成立自贸区，为中国第三批自贸区<ref name=sina/>。2017年3月15日，国务院印发《国务院关于印发中国（陕西）自由贸易试验区总体方案的通知》<ref name=gov>{{cite web | url=http://www.gov.cn/zhengce/content/2017-03/31/content_5182306.htm | title=国务院关于印发中国（陕西）自由贸易试验区总体方案的通知| publisher=国务院| date=2017-03-15 | accessdate=2017-05-04}}</ref>。

2017年4月1日，陕西自贸区正式挂牌成立，陕西省和西安市领导[[娄勤俭|娄勤俭]]、[[胡和平|胡和平]]、[[韩勇|韩勇]]、[[毛万春|毛万春]]、[[刘小燕|刘小燕]]、[[梁桂|梁桂]]、[[王永康|王永康]]、[[李晓东_(民革)|李晓东]]、[[张迈曾|张迈曾]]、[[上官吉庆|上官吉庆]]、[[岳华峰|岳华峰]]、[[陈国强_(陕西省)|陈国强]]等出席，陕西省副省长[[魏增军|魏增军]]主持。成立仪式的最后，娄勤俭和胡和平共同为陕西自贸区三大片区揭牌。同日上午，[[中国银行|中国银行]]西安自贸区支行、[[北京银行|北京银行]]陕西自贸区国际港务区支行、陕西自贸区仲裁院和陕西自贸区国际商事调解中心也在陕西自贸区西安国际港务区国际招商推介暨项目签约仪式上同时揭牌，13家企业也现场与西安国际港务区签约，投资金额超20亿元人民币<ref name=hsw>{{cite web | url=http://news.hsw.cn/system/2017/0402/711549.shtml | title=陕西自贸区挂牌运行 自贸区内企业将享受创新红利| publisher=华商报| date=2017-04-02 | accessdate=2017-05-04}}</ref>。

==规划方案==
===区域功能===
陕西自贸区由西安中心片区、西安国际港务区片区和杨凌示范区片区组成，总面积119.95平方公里<ref name=gov/>，具体的管辖范围、区域功能如下<ref name=hsw/><ref name=gov/><ref>{{cite web| url=http://www.xaftz.gov.cn/Pages/IntroductionDetaile.aspx?id=228| title=中国（陕西）自贸试验区西安核心区| publisher=西安自贸区| date=2017-03-29| accessdate=2017-05-04}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web | url=http://www.shaanxiftz.gov.cn/zjzmg/gygk/lbgc/3iUfee.htm| title=区域概况| publisher=陕西自贸区网站| date=2017-03-07  | accessdate=2017-05-04}}</ref>：
*中心片区（总面积87.76平方公里，含西安出口加工区A区0.75平方公里、B区0.79平方公里，西安高新区综合保税区3.64平方公里和陕西西咸保税物流中心〔B型〕0.36平方公里，管辖片区涵盖[[西安经济技术开发区|西安经济技术开发区]]、[[西安高新技术产业开发区|西安高新区]]和[[西咸新区|西咸新区]]，具体管辖范围为：东至经开区明光路，南至高新区锦业二路、滨河北路，西至秦汉新城秦英路，北至空港新城正平大街），中心片区重点发展战略性新兴产业和高新技术产业，着力发展高端制造、航空物流、贸易金融等产业，推进服务贸易促进体系建设，拓展科技、教育、文化、旅游、健康医疗等人文交流的深度和广度，打造面向“一带一路”的高端产业高地和人文交流高地；
*[[西安国际港务区|西安国际港务区]]片区（总面积26.43平方公里，含西安综合保税区6.17平方公里，管辖片区涵盖西安国际港务区、[[浐灞生态区|浐灞生态区]]，具体管辖范围为：东至西安国际港务区西韩公路，南至浐灞生态区香槐一路，西至西安国家港务区灞河东路，北至西安国家港务区铁路北环线），西安国际港务区片区重点发展国际贸易、现代物流、金融服务、旅游会展、电子商务等产业，建设“一带一路”国际中转内陆枢纽港、开放型金融产业创新高地及欧亚贸易和人文交流合作新平台；
*[[杨凌农业高新技术产业示范区|杨凌示范片区]]（总面积5.76平方公里，管辖片区属于杨凌农业高新技术产业示范区的一部分，具体管辖范围为：东至新桥路，南至城南路、河堤路、滨河路一线，西至民乐路，北至兴平路、永安路、西兰高铁杨凌段一线），杨凌示范区片区以农业科技创新、示范推广为重点，通过全面扩大农业领域国际合作交流，打造“一带一路”现代农业国际合作中心。

===发展目标===
根据《中国（陕西）自由贸易试验区总体方案》，陕西自贸区的发展目标是要形成与国际投资贸易通行规则相衔接的制度创新体系，营造法治化、国际化、便利化的营商环境，努力建成投资贸易便利、高端产业聚集、金融服务完善、人文交流深入、监管高效便捷、法治环境规范的高水平高标准自由贸易园区，推动“一带一路”建设和西部大开发战略的深入实施<ref>{{cite web |url = http://news.sina.com.cn/c/nd/2017-03-31/doc-ifycwyxr8967049.shtml |title = 中国新建7个自贸区 总体方案印发明确发展目标 |publisher = 中国新闻网 |accessdate =2017-03-31}}</ref>。

===相关政策===
根据《中国（陕西）自由贸易试验区总体方案》，陕西自贸区将改革创新政府管理方式，实施“多证合一”综合审批服务运行模式，建立“一口受理、并联审批”工作机制，开展知识产权综合管理改革试点。在推动贸易转型升级方面，陕西自贸区将拓展新型贸易方式，鼓励企业参与“自主报税、自助通关、自动审放、重点稽核”等监管制度创新试点，完善海关特殊监管区域功能。此外，陕西自贸区还将推动金融制度创新、增强金融服务功能、创新合作机制、推动[[西部大开发|西部大开发]]战略的深入实施<ref name=gov/>。陕西自贸区建成后，未来将会引进大量外资企业以及大量中小企业进驻自贸区开展跨境业务，另外日后农民在杨凌购买进口农业生产资料、城镇居民购买进口商品也会更加便捷<ref>{{cite web |url = http://news.163.com/17/0401/05/CGTNGUUV00018AOP.html|title =陕西自贸区，市民期待啥？ |publisher =华商报(西安) |accessdate =2017-04-01}}</ref>。而[[中华人民共和国公安部|中华人民共和国公安部]]也决定在陕西自贸区推出7项出入境政策措施，支持区域经济社会建设发展<ref>{{cite web |url = http://news.163.com/17/0310/03/CF4SA2GP00018AOP.html|title =公安部7出入境措施支持陕西等自贸区 |publisher =华商报(西安) |accessdate =2017-04-01}}</ref>。
