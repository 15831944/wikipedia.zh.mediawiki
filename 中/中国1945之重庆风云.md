《'''中国1945之重庆风云'''》，中国大陆电视剧，导演[[张国立|张国立]]、[[羅長安|罗长安]]，主演[[唐国强|唐国强]]、[[张国立|张国立]]、[[蒋勤勤|蒋勤勤]]、[[袁立|袁立]]、[[刘劲|刘劲]]、[[张铁林|张铁林]]、[[王伍福|王伍福]]、[[郭连文|郭连文]]、[[许还山|许还山]]、[[达式常|达式常]]。

== 剧情 ==
[[抗日战争|抗日战争]]胜利之后，国共两党为了争夺中国大陆政权爆发了[[国共内战|国共内战]]，该剧讲述国共内战中的史实。

== 花絮 ==
该剧是“建立[[中国共产党|中国共产党]]90周年”献礼剧。<ref name=" tiger" >{{cite web |url=http://ent.sina.com.cn/v/m/2011-06-20/14063339469.shtml |title=《中国1945之重庆风云》再现重庆谈判精彩故事 |accessdate= 2011年6月20日|publisher=新浪网 |work= |language =zh-cn}}</ref>

该剧的主轴是“重庆谈判”。<ref name=" tiger" />

该剧被誉为电视剧版“[[建国大业|建国大业]]”。<ref>{{cite web |url=http://www.people.com.cn/h/2011/0617/c25408-1410822523.html |title= 《中国1945之重庆风云》将打造剧版《建国大业》|accessdate=2011年6月17日 |publisher=人民网 |work= |language =zh-cn}}</ref>

唐国强出演毛泽东、张国立出演蒋中正，张国立原本打算出现《[[铁齿铜牙纪晓岚|铁齿铜牙纪晓岚]]》一样的铁三角局面，但是[[王刚_(演员)|王刚]]没有找到合适的角色。<ref name=" fish" >{{cite web |url= http://ent.163.com/11/0617/00/76NA2D0200032DGD.html|title=《重庆风云》将播 张国立遗憾“铁三角”凑不齐 |accessdate=2011-06-17 |publisher=网易 |work= |language =zh-cn}}</ref><ref name=" cag" >{{cite web |url= http://ent.sina.com.cn/v/m/2011-06-17/10093336723.shtml|title=《重庆风云》BTV首播 《纪晓岚》班底独缺王刚 |accessdate= 2011年6月17日|publisher=新浪网 |work= |language =zh-cn}}</ref>
