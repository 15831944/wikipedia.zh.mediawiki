'''中国人民解放军空军土门机场'''，位于[[湖北省|湖北省]][[宜昌市|宜昌市]]，隶属[[中国人民解放军中部战区空军|中国人民解放军中部战区空军]]某部。

==简介==
1937年至1938年间，[[国民政府|国民政府]]军队在铁路坝（今夷陵广场）修建军用机场用于飞行训练。1940年，[[日军|日军]]攻占宜昌，铁路坝机场沦为废墟。1941年，日军强征民夫万名，先后在宜昌县土门垭以及[[当阳县|当阳县]]修建机场，用于轰炸中国大后方。1945年8月[[日本投降|日本投降]]，机场关闭<ref name=bianqian>{{Cite web|url = http://www.ycjt.gov.cn/content-22511-281555-1.html|title = 宜昌交通之变迁|author = 黄传林|date = 2006-03-27|publisher = 宜昌市交通运输局|language = |accessdate = |quote = }}{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[中华人民共和国|中华人民共和国]]成立后，中央军委决定在宜昌土门修建机场，1953年开工，1954年通航。随后又在当阳县修建军用机场<ref name=bianqian/>。后在宜昌土门机场成立了'''中国人民解放军空军宜昌场站'''。[[杨树葆|杨树葆]]曾任空军宜昌场站站长（正团职）<ref>{{Cite web |url = http://hb.qq.com/a/20110405/000263.htm |title = 做一个像父亲一样的人 他的精神温暖着我的心灵 |author =  |date = 2011-04-05 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

后来，该机场成为军民两用机场，民航称“'''宜昌土门娅机场'''”。1995年12月31日，宜昌土门娅机场正式停飞<ref> {{Cite book | author =  宜昌年鉴编纂委员会| title =1996宜昌年鉴 | location =  | publisher = 中国三峡出版社 | date = 1996年10月 | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote =宜昌土门娅机场正式停飞  }} </ref>。此后土门机场仅供军用。2013年11月8日，《宜昌市综合交通体系规划（2011-2030年）》通过专家评审后，进入30天公示期，该规划提出宜昌市将提升现有的[[宜昌三峡机场|宜昌三峡机场]]功能，对[[中国人民解放军空军当阳场站|当阳军用机场]]在现状基础上调整完善供军民两用，并且外迁土门军用机场，同时在三峡坝区及黄柏河旅游区规划新建2处直升机场<ref>{{Cite web |url = http://hb.sina.com.cn/news/j/2013-11-09/1452119036.html |title =  宜昌交通规划通过专家评审 新建两处直升机场|author =  |date = 2013-11-09 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

2010年时，空军土门机场由武汉空军宜昌土门机场管理办公室管理<ref>{{Cite web |url = http://yichang.leju.com/2010-02-09/10126704.html |title =  郭有明李乐成等走访慰问驻宜部队、困难群众|author =  |date = 2010-02-09 |publisher = 宜昌乐居网 |language =  |accessdate =  |quote =  }}</ref>。2016年时，空军土门机场由空军土门机场保障队管理<ref>{{Cite web |url = http://hb.news.163.com/yichang/16/0802/08/BTEV0H6R07111D71.html |title = 宜昌市领导走访慰问驻宜部队官兵 |author =  |date = 2016-08-02 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。
