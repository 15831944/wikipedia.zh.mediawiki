'''中国国防科技信息中心'''，位于[[北京市|北京市]][[海淀区|海淀区]][[阜成路|阜成路]]26号，是[[中华人民共和国|中华人民共和国]]国家级信息机构之一。

==简介==
1959年3月，中国国防科技信息中心成立。到2010年代，中国国防科技信息中心已成为集情报咨询研究、软科学研究、信息服务、信息技术开发、成果和专利管理、人才培养为一体的科技信息机构。截至2014年，该中心图书馆馆藏260多万份文献资料，30余个数据库共3000余万条信息，4000余部声像资料<ref name=lishi/><ref name=zhaosheng/>。中国国防科技信息中心是中国人民解放军规模最大、系统资源最为丰富的数字化加工中心<ref>{{Cite web |url =  http://www.81.cn/jwgz/2016-03/23/content_6972538.htm|title =  揭秘高科技成果背后的“创新智囊”：中国国防科技信息中心|author =  |date = 2016-03-23 |publisher =  中国军网|language =  |accessdate =  |quote =  }}</ref>。

中国国防科技信息中心的研究方向主要为军事高新技术、武器系统发展和军备控制、武器装备建设管理等，先后完成了许多国防科技情报研究课题，获数百项国家级和部委级科技进步奖<ref name=lishi/><ref name=zhaosheng/>。

1986年，中国国防科技信息中心获得硕士学位授予权。2014年时，该中心有3个硕士学位授权点（情报学、军事装备学、计算机应用技术），并且和[[北京大学|北京大学]]、[[中国人民解放军军械工程学院|中国人民解放军军械工程学院]]等联合培养情报学、军事装备学博士研究生。2003年底获准设[[博士后|博士后]]科研工作站<ref name=lishi>{{Cite web|url = http://gz.southcn.com/content/2016-01/11/content_140527336.htm|title = 中央军委装备发展部成立 总装备部成为历史|author = |date = 2016-01-11|publisher = 南方网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170604215250/http://gz.southcn.com/content/2016-01/11/content_140527336.htm|archive-date = 2017-06-04|dead-url = yes}}</ref><ref name=zhaosheng>{{Cite web |url = http://yz.chsi.com.cn/sch/viewZszc--infoId-701442400,categoryId-10462749,schId-367834,mindex-12.dhtml |title =中国国防科技信息中心  |author =  |date =  |publisher = 中国研究生招生信息网 |language =  |accessdate = 2017-05-17 |quote =  }}</ref><ref>{{Cite web |url = http://yz.chsi.com.cn/sch/schoolInfo--schId-367834.dhtml |title = 中国国防科技信息中心 |author =  |date =  |publisher = 中国研究生招生信息网  |language =  |accessdate = 2017-05-17 |quote =  }}</ref>。

2010年代，中国国防科技信息中心瞄准国防科技前沿开展科技信息服务，加强对用户需求研究，推进信息资源建设和挖掘，其战略前瞻性研究的深度及力度均有提高<ref>{{Cite web |url = http://military.people.com.cn/n1/2016/1208/c1011-28933878.html |title =中国国防科技信息中心打造“国字头”高端智库  |author =  |date = 2016-12-08 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
