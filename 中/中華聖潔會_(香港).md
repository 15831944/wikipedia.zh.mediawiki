'''中華聖潔會'''（{{lang-en|The China Holiness Church}}），[[香港|香港]][[基督教|基督教]]機構、辦學團體，以傳揚[[福音|福音]]為終旨。

== 歷史 ==
1930年，吳仲連牧師伉儷及幾位信徒在[[深水埗|深水埗]]成立中華聖潔會，並開始於[[深水埗|深水埗]]街頭佈道和聚會。{{noteTag|資料出自蕭國健《簡明香港近代史》（2013年由三聯書店(香港)有限公司出版）第156頁。}}在成立初期，該會租用[[荔枝角道|荔枝角道]]一所車房進行聚會。及後因為信徒人數增長，中華聖潔會便遷址[[深水埗|深水埗]]福榮街95號。及後到了日治時期，該會繼續聚會，又遷址至鴨寮街6號三樓。為方便內地的[[福音|福音]]工作，該會在[[廣州|廣州]][[中山市|中山]]設置佈道所。直至1950年代初期，信徒的數目持續增加，香港中華聖潔會將地址遷到白楊街25號地下。由於認定教育與堂會發展的策略，於是中華聖潔會向政府申辦小學「中聖學校」。不久就集資四十餘萬在[[深水埗|深水埗]]懷惠道建成校舍，並作為堂址。校舍於1957年落成啟用，辦學要旨在於向學生、家長及坊眾傳播[[福音|福音]]。<ref name="中華聖潔會簡史重溫（邢福增院長86周年堂慶講道節錄）">[http://www.chc.org.hk/index.php/about/intro 中華聖潔會〈中華聖潔會簡史重溫（邢福增院長86周年堂慶講道節錄）〉]</ref>

1977年，中華聖潔會展開差傳工作。在1981年至1984年間，該會先後派遣宣教士周志豪和孫玉琴遠赴[[南韓|南韓]]參與[[中國|中國]][[福音|福音]]傳播工作。<ref name="母會中華聖潔會歷史">[http://www.cschurch.org.hk/page12d.php 中聖教會〈母會中華聖潔會歷史〉]</ref>四年後，中華聖潔會在[[將軍澳|將軍澳]]臨時房屋區開設睦鄰中心協助社區關懷與植堂工作，並以「中聖教會」名義進行聚會。而教會及中聖教會白普理社區服務中心也遷入[[寶林邨|寶林邨]]寶寧樓。<ref name="中聖教會有限公司- 中聖教會白普理社區服務中心">[http://dss.hkcss.org.hk/pdf/90.pdf 香港社會服務機構總覽〈中聖教會有限公司- 中聖教會白普理社區服務中心〉第134頁]</ref>

== 開辦學校 ==
1957年，香港中華聖潔會開辦私立小學「中聖學校」。學校到了1976年隨深水埗的人口轉變以及學位需求增加而調整轉型為直資中學[[中聖書院|中聖書院]]。但在2003年年底，因學校進行擴建工程，該會隨學校由[[深水埗|深水埗]]遷至[[荔景|荔景]]校舍。中聖書院已於2009年1月遷回現址。<ref>{{web reference|URL=https://www.singtaousa.com/la/13-%E9%A6%99%E6%B8%AF/140528-%E8%8D%94%E6%99%AF%E9%82%A8%E6%96%B0%E5%85%AC%E5%B1%8B%E7%94%B3%E5%A2%9E40%E4%BC%99/|title=荔景邨新公屋申增40伙|work=星島日報|year=2017年|date=6月2日}}</ref>另外，中華聖潔會於1998年在[[大埔_(香港)|大埔]]開辦津貼中學[[中華聖潔會靈風中學|中華聖潔會靈風中學]]。<ref name="中華聖潔會靈風中學">[http://www.chsc.hk/ssp2016/pdf/tp_06_456.pdf 香港中學2016年概覽〈中華聖潔會靈風中學〉]</ref>同時開設分堂「中華聖潔會大埔堂」。 <ref name="中華聖潔會80周年特刊">[http://etd.lib.nsysu.edu.tw/ETD-db/ETD-search-c/getfile?URN=etd-0704105-155357&filename=etd-0704105-155357.pdf 香港中華聖潔會《中華聖潔會80周年特刊》第9頁]</ref>
