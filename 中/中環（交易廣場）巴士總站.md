'''中環（交易廣場）巴士總站'''（英文：'''Central (Exchange Square) Bus Terminus'''）是[[香港|香港]]一個坑狀室內巴士總站，位於[[中環|中環]][[干諾道中|干諾道中]]與[[港景街|港景街]]之間，供多條巴士路線使用。另外於總站東面，設有公共小巴總站、專線小巴總站、的士總站和公眾上落客區。中環（交易廣場）巴士總站因為直接繼承前中環總站，已被大眾視為中環的中央車站。本站專利車坑數目更達到14條，是現時全港最多車坑的傳統坑狀車站。

==總站路線資料（巴士）==
===[[城巴6號線|城巴6號線]]===
*{{HK Bus Route HKIsland Show:6|format=inline}}
1933年6月11日配合專營權生效，由[[中華巴士|中巴]]營辦，1993年9月1日由[[城巴|城巴]]取代中巴經營本線，途經[[淺水灣|淺水灣]]、[[黃泥涌峽|黃泥涌峽]]。

===[[城巴6A線|城巴6A線]]===
*{{HK Bus Route HKIsland Show:6A|format=inline}}
1991年投入服務，由[[城巴|城巴]]營運，來往中環（交易廣場）及[[赤柱炮台|赤柱炮台]]之間，途經[[香港仔隧道|香港仔隧道]]，只在繁忙時間提供服務。
 
===[[城巴6X線|城巴6X線]]===
*{{HK Bus Route HKIsland Show:6X|format=inline}}
1994年11月10日投入服務，途經[[皇后大道東|皇后大道東]]、[[香港仔隧道|香港仔隧道]]、[[深水灣|深水灣]]、[[淺水灣|淺水灣]]、[[馬坑|馬坑]]。
 
===[[新巴15線|新巴15線]]（特別班次）===
*{{HK Bus Route HKIsland Show:15|format=special}}
1960年7月1日投入服務，現由[[新巴|新巴]]營運，來往中環5號碼頭及[[山頂|山頂]]之間，上午十時前的班次則以此為總站。本線是唯一一條每天行走的山頂區巴士路線。

===[[新巴66線|新巴66線]]===
*{{HK Bus Route HKIsland Show:66|format=inline}}
1994年1月22日，配合[[馬坑邨|馬坑邨]]落成而投入服務，只在每天繁忙時間提供服務。

===[[城巴70線|城巴70線]]===
*{{HK Bus Route HKIsland Show:70|format=inline}}
1975年1月1日，為配合[[中華巴士|中巴]][[南區_(香港)|南區]]路線重組而投入服務，現時由[[城巴|城巴]]營運，途經[[香港仔隧道|香港仔隧道]]、[[灣仔北|灣仔北]]及[[金鐘|金鐘]]，是[[香港仔|香港仔]]居民來往[[中環|中環]]上班下班的主要路線。

===[[城巴70線|城巴70線]] (特別班次)===
*{{HK Bus Route HKIsland Show:70|format=special}}
1975年1月1日投入服務，是[[香港仔|香港仔]]居民來往[[中環|中環]]上下班的主要路線。2015年5月17日起配合本線延長至華貴，新增由本站開出的特別班次。

===[[城巴70線|城巴70A線]]===
*{{HK Bus Route HKIsland Show:70A|format=inline}}
2017年7月31日投入服務，只於星期一至五下午繁忙時間服務。途經[[金鐘|金鐘]]、[[灣仔北|灣仔北]]及[[香港仔隧道|香港仔隧道]]，方便[[香港仔|香港仔]]居民下班。

===[[城巴70P線|城巴70P線]]===
*{{HK Bus Route HKIsland Show:70P|format=inline}}
於2007年9月24日投入服務，只於平日上午繁忙時間開出三班。

===[[城巴75線|城巴75線]]===
*{{HK Bus Route HKIsland Show:75|format=inline}}
1982年3月15日投入服務，是第一批[[城巴|城巴]]由[[中華巴士|中巴]]手上接辦的26條巴士路線之一。途經[[香港仔隧道|香港仔隧道]]、[[軒尼詩道|軒尼詩道]]、[[金鐘道|金鐘道]]，是[[黃竹坑|黃竹坑]]居民來往[[中環|中環]]上班下班的主要的路線。 

===[[城巴90線|城巴90線]]===
*{{HK Bus Route HKIsland Show:90|format=inline}}
1987年4月13日投入服務，是[[鴨脷洲邨|鴨脷洲邨]]及[[鴨脷洲大街|鴨脷洲大街]]居民來往[[中環|中環]]上班及給[[灣仔|灣仔]]和中環下班的市民的路線。

===[[城巴97線|城巴97線]]===
*{{HK Bus Route HKIsland Show:97|format=inline}}
1987年4月13日投入服務，以配合[[利東邨|利東邨]]入伙。途經[[香港仔隧道|香港仔隧道]]、[[灣仔|灣仔]]、[[金鐘|金鐘]]。

===[[城巴260線|城巴260線]]===
*{{HK Bus Route HKIsland Show:260|format=inline}}
1978年9月1日投入服務，現由[[城巴|城巴]]營辦，是[[港島|港島]]區目前唯一以2字頭及三位數為編號的巴士路線。

===[[新巴590線|新巴590線]]===
*{{HK Bus Route HKIsland Show:590|format=inline}}
2010年5月30日投入服務，途經[[黃竹坑道|黃竹坑道]]、[[香港仔隧道|香港仔隧道]]、[[告士打道|告士打道]]、[[干諾道中|干諾道中]]，在行走此線巴士的路線顯示牌上，「Express特快」字眼明顯可見，但官方網站未有標示此線為特快線。

===[[過海隧道巴士690線|過海隧道巴士690線]]===
*{{HK Bus Route Cross Harbour Show:690|format=inline}}
1990年8月13日投入服務，現由[[九巴|九巴]]及[[城巴|城巴]]聯合經營，提供[[康盛花園|康盛花園]]、[[翠林|翠林]]、[[寶琳|寶琳]]來往[[北角|北角]]、[[銅鑼灣|銅鑼灣]]、[[灣仔|灣仔]]、[[金鐘|金鐘]]、[[中環|中環]]的[[過海隧道巴士|過海隧道巴士]]服務。
 
===[[過海隧道巴士690P線|過海隧道巴士690P線]]===
*{{HK Bus Route Cross Harbour Show:690P|format=inline}}
2000年5月2日投入服務，由[[九巴|九巴]]及[[城巴|城巴]]聯合經營，途經[[寶琳北路|寶琳北路]]、[[將軍澳隧道|將軍澳隧道]]、[[啟田道|啟田道]]、[[東區海底隧道|東區海底隧道]]、[[東區走廊|東區走廊]]、[[告士打道|告士打道]]，只在平日上午提供服務，假日停開。

==途經路線資料（巴士）==
===[[新巴2號線|新巴2號線]]===
*{{HK Bus Route HKIsland Show:2|format=inline}}
提供穿梭嘉亨灣、[[愛秩序灣|愛秩序灣]]、[[筲箕灣|筲箕灣]]、[[西灣河|西灣河]]、[[鰂魚涌|鰂魚涌]]、[[北角|北角]]、[[炮台山|炮台山]]、[[銅鑼灣|銅鑼灣]]、[[灣仔|灣仔]]、[[金鐘|金鐘]]及[[中環|中環]]的流水線巴士服務。

===[[新巴4號線|新巴4號線]]（假日特別班次）===
*{{HK Bus Route HKIsland Show:4|format=inline}}
1933年6月投入服務，現由[[新巴|新巴]]營辦，提供由華富（南）來往[[中環|中環]]之[[循環線|循環線]]服務。

===[[新巴4X線|新巴4X線]]===
*{{HK Bus Route HKIsland Show:4X|format=inline}}
4X為4號線的特快線，於2008年3月17日開始服務。只在星期一至六服務，早上時段的班次在抵達-{干}-諾道中後，便取道林士街天橋，不經上環及西營盤，本路線並與4號線提供聯合班次。

===[[城巴7號線|城巴7號線]]===
*{{HK Bus Route HKIsland Show:7|format=inline}}
於1921年投入服務，途經[[香港仔|香港仔]]、[[田灣|田灣]]、[[薄扶林道|薄扶林道]]、[[香港大學|香港大學]]、[[西營盤|西營盤]]及[[上環|上環]]，為港島區首條巴士路線。

===[[城巴11線|城巴11線]]===
*{{HK Bus Route HKIsland Show:11|format=inline}}
提供渣甸山來往銅鑼灣、灣仔、金鐘及中環的巴士服務，於1954年10月1日起投入服務。

===[[城巴12線|城巴12線]]===
*{{HK Bus Route HKIsland Show:12|format=inline}}
該路線提供半山區（堅道、西摩道及羅便臣道）來往中環的港鐵接駁巴士服務，於1955年8月16日起投入服務。

===[[新巴15線|新巴15線]]===
*{{HK Bus Route HKIsland Show:15|format=inline}}
1960年7月1日投入服務，現由[[新巴|新巴]]營運，來往中環5號碼頭及[[山頂|山頂]]之間，上午十時前的班次則以中環（交易廣場）為總站。本線是唯一一條每天行走的[[山頂|山頂]]區巴士路線。

===[[新巴25線|新巴25線]]===
*{{HK Bus Route HKIsland Show:25|format=inline}}
1973年9月3日投入服務，現由[[新巴|新巴]]營辦，以[[循環線|循環線]]形式運作，來往中環（5號碼頭）及[[寶馬山|寶馬山]]之間。

===[[新巴30X線|新巴30X線]]===
*{{HK Bus Route HKIsland Show:30X|format=inline}}
2008年3月17日開辦，由[[新巴|新巴]]營辦，循環來往[[數碼港|數碼港]]及中環（交易廣場）之間。是第三條來回程均行經[[林士街天橋|林士街天橋]]的全日[[港島|港島]]路線。
 
===[[新巴91線|新巴91線]]===
*{{HK Bus Route HKIsland Show:91|format=inline}}
該路線提供鴨脷洲邨、利東邨及香港仔來往薄扶林道、上環及中環渡輪碼頭的巴士服務，於1982年11月16日起投入服務。

===[[新巴94線|新巴94線]]===
*{{HK Bus Route HKIsland Show:94|format=inline}}
該路線提供利東邨及香港仔來往薄扶林道、上環及中環渡輪碼頭的巴士服務，於1989年11月16日起投入服務，只於平日上午繁忙時間提供服務。

===[[城巴347線|城巴347線]]===
*{{HK Bus Route HKIsland Show:347|format=inline}}
經[[域多利道|域多利道]][[香港華人基督教聯會薄扶林道墳場|香港華人基督教聯會薄扶林道墳場]]，只在[[清明節|清明節]]及[[重陽節|重陽節]]期間提供服務

===[[過海隧道巴士621線|過海隧道巴士621線]]===
*{{HK Bus Route Cross Harbour Show:621|format=inline}}
1997年2月17日起投入服務，由[[九巴|九巴]]及[[城巴|城巴]]營運，輔助[[過海隧道巴士601線|601線]]線及[[過海隧道巴士619線|619線]]，為一條只在平日繁忙時間服務的特快過海隧道巴士線，所以價錢較高。

===[[城巴629線|城巴629線]]===
*{{HK Bus Route HKIsland Show:629|format=inline}}
提供往海洋公園的巴士服務。只在每日早上開5班車單向前往海洋公園，不提供回程服務。

===[[過海隧道巴士681線|過海隧道巴士681線]]===
*{{HK Bus Route Cross Harbour Show:681|format=inline}}
於1995年12月6日投入服務，由九巴及城巴聯營，往返[[馬鞍山_(香港市鎮)|馬鞍山]]及[[中環|中環]]。途經[[大老山隧道|大老山隧道]]、[[觀塘繞道|觀塘繞道]]、[[東區海底隧道|東區海底隧道]]、[[東區走廊|東區走廊]]、[[灣仔|灣仔]]。

===[[新巴722線|新巴722線]]===
*{{HK Bus Route HKIsland Show:722|format=inline}}
1984年6月8日配合[[東區走廊|東區走廊]]通車投入服務，由[[新巴|新巴]]營運，循環行走[[耀東邨|耀東邨]]及[[中環碼頭|中環碼頭]]之間。
