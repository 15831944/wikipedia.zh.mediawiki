===外交===
{{see also|中華民國外交|{{link-en|吉爾吉斯外交|Foreign relations of Kyrgyzstan}}}}
1916年，[[俄羅斯帝國|俄羅斯帝國]]對反抗運動的[[政治迫害|鎮壓]]，使不少[[柯爾克孜族|吉爾吉斯民族]]從[[中亞|中亞]]遷徙至新成立的[[中華民國|中華民國]]（[[北洋政府|北洋政府]]）。

2016年12月28日，吉爾吉斯副議長阿利別科夫傳出因在12月21日訪台與中華民國立法院長[[蘇嘉全|蘇嘉全]]見面，在中華人民共和國向{{link-en|吉爾吉斯外交部|Ministry of Foreign Affairs (Kyrgyzstan)}}抗議後就自行辭職<ref name="newstalk"/>。阿利別科夫告訴媒體“國家利益高於個人利益，需要注意與中國加強關係”，吉國議長{{link-tr|圖爾松別科夫|Çınıbay Tursunbekov}}於29日當天即表示，“這起會面違反了吉爾吉斯與中國於2002年的睦鄰條約”{{notetag|中华人民共和国和吉尔吉斯共和国睦邻友好合作条约}}。其中規定吉爾吉斯承認世界只有[[一個中國|一個中國]]，中華人民共和國是代表中國唯一合法政府，台灣是中國不可分割的一部分<ref>{{cite web |title=Deputy Speaker of Kyrgyz Parliament steps down after Taiwan visit |url=http://akipress.com/news:587107 |publisher=AKI新聞通訊社 |date=2016-12-29 |archiveurl=https://archive.is/20161229043857/http://akipress.com/news:587107?from=portal&place=imp-en |archivedate=2016-12-29 |deadurl=yes }}</ref>。此外，這起會面就觀光、產業和農業洽談合作，並無討論政治問題或簽署官方文件<ref name="newstalk">{{cite web |title=吉爾吉斯副議長訪台見蘇嘉全 回國丟飯碗 |url=https://newtalk.tw/news/view/2016-12-30/80558 |publisher=新頭殼 |accessdate=2016-12-30}}</ref>。

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|吉爾吉斯簽證政策|Visa policy of Kyrgyzstan}}|{{link-en|吉爾吉斯公民簽證要求|Visa requirements for Kyrgyzstani citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以預審制[[簽證#落地签证|落地簽證]]的方式入境吉爾吉斯，事先透過吉爾吉斯[[旅行社|旅行社]]出具落地簽證邀請函向{{link-en|吉爾吉斯外交部|Ministry of Foreign Affairs (Kyrgyzstan)}}申請，核准後持函於首都的[[瑪納斯國際機場|瑪納斯國際機場]]辦理落地簽證，停留最多30天。經吉爾吉斯轉機，若未離開機場轉機區，無須申辦[[過境簽證|過境簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-138-788aa-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=7F220D7E656BE749 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|吉爾吉斯護照|Kyrgyzstani passport}}的{{link-en|吉爾吉斯人口|Demographics of Kyrgyzstan|吉爾吉斯公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將吉爾吉斯歸類為'''灰色警示'''：提醒注意（2015年7月29日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-99-92922-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==

===貿易===
2018年，吉爾吉斯是中華民國第178大貿易夥伴、第159大[[進口|進口]]夥伴、第174大[[出口贸易|出口]]夥伴，在[[亞洲國家和地區列表|亞洲國家]]排名倒數第4、7、5；在[[世界政區索引|世界經濟體]]倒數第67、82、63。出口至吉爾吉斯的金額為107萬3,180[[美元|美元]]，年減2.796%；自吉爾吉斯進口的金額為52萬4,401美元，年增393.234%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）54萬8,779美元，年減44.997%。2017年，吉爾吉斯是中華民國第186大貿易夥伴、第190大進口夥伴、第176大出口夥伴，在亞洲國家排名倒數第3、4、6；在世界經濟體倒數第53、47、55。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
