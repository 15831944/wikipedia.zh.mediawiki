'''中華民國全國商業總會'''（{{lang-en|General Chamber of Commerce of the Republic of China}}）簡稱“商總”，原名「中華民國商會全國聯合會」，是[[中華民國|中華民國]]商業界的最高組織，為[[七大工商團體|七大工商團體]]之一。

== 歷史沿革及發展 ==
1941年6月，對日抗戰中期，中華民國商業界人士集會陪都[[重慶|重慶]]，開始計劃籌備商會聯合會。

1946年5月間由王曉籟（[[上海市|上海市]]代表）、陳勤士（[[浙江|浙江]]代表）、王繹齋（南京市代表）、潘仰山（曾任全國工業總會理事長）、胡夢洲（[[福建|福建]]代表，福建省商聯會理事）、曹成章（[[華北|華北]]代表），在首都南京召開籌備會議，會後全體與會人員赴[[中山陵|中山陵]]謁陵，並決定正式召開第一屆全國會員代表大會。11月1日，[[中華民國|中華民國]]各省市商會和海外各地[[華僑|華僑]]商會約50個單位，400餘位代表，經由社會部的召集，共同集會於首都[[南京|南京]]，成立“中華民國商會全國聯合會”，[[國民政府|國民政府]]主席[[蔣中正|蔣中正]]親臨大會致賀並題頒「同心拓業」四字勗勉。大會會期四天，通過多件重要議案並通過訂定該日為“[[商人節|商人節]]”，藉以慶祝此一推行全國商業的法定最高機構的誕生，並象徵商業界的團結。<ref name="認識商總">[http://www.roccoc.org.tw/About/ 認識商總]</ref>

“中華民國商會全國聯合會”第二屆全國會員代表大會於1948年11月上海舉行，1951年9月21在臺北復會，由第二屆全國會員代表大會選出之理監事繼續行使職權，推展各項工作。<ref name="認識商總" />

1973年3月1日依據新頒訂的“商業團體法”，正式更名為“中華民國全國商業總會”。<ref name="認識商總" />

隨著動員戡亂時期終止，原大陸選出之理監事退職，第三屆第一次會員代表大會於1993年2月27日召開，全面改選理監事。<ref name="認識商總" />

== 中華民國全國商業總會任務 ==
# 關於國內外商業調查、統計及研究、發展事項。<br />
# 關於國際貿易之聯繫、介紹及推廣事項。<br />
# 關於政府經濟政策與商業法令之協助推行及研究建議事項。<br />
# 關於商業糾紛之調處事項。<br />
# 關於舉辦商業及服務業各項創新技能、人才培育有關訓練與業務講習；相關專業如：珠算測驗、勞工安全衛生、消防等之講習訓練；與海內外有關機構合作辦理提升知識、技能之進修課程等，並核發各項結業證書等事項。<br />
# 關於會員商品之廣告、展覽及商業證明文件之簽發事項。<br />
# 關於會員與會員代表基本資料之建立及動態之調查、登記事項。<br />
# 關於會員委託證照之申請變更、換領及其他服務事項。<br />
# 關於會員或社會公益事業之舉辦事項。<br />
# 關於會員合法權益之維護事項。<br />
# 關於接受政府機關、團體之委託服務事項。<br />
# 關於社會運動之參加事項。<br />
# 關於商業、服務業之就業服務有關事項。<br />
# 關於會員之聯繫合作及輔導事項。<br />
# 關於政府機關之交辦事項。<br />
# 依其他法令規定應辦理之事項。
