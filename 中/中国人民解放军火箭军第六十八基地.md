'''中国人民解放军火箭军第六十八基地'''，位于[[河南省|河南省]][[洛阳市|洛阳市]]，是[[中国人民解放军火箭军|中国人民解放军火箭军]]工程基地。

==沿革==
1958年，根据国家试验和发展[[导弹|导弹]]和[[原子弹|原子弹]]的部署，中央军委决定上马导弹综合试验靶场工程和原子弹试验场工程，两项工程代号为0029工程。1958年4月11日，中央军委秘书长[[黄克诚|黄克诚]]主持0029工程建设会议，决定成立中国人民解放军工程兵特种工程指挥部（简称“特工指”），代号中国人民解放军7169部队，为兵团级单位，以[[中国人民志愿军|中国人民志愿军]]第十九兵团机关、中国人民志愿军工程指挥部、中国人民志愿军后勤部为基础组建，由军委工程兵司令员[[陈士榘|陈士榘]]兼任7169部队司令员和政委。7169部队负责导弹综合试验靶场（又称[[中国核试验基地|二十一基地]]）和原子弹试验场（又称[[中国酒泉卫星发射中心|二十基地]]）的工程建设。7169部队下属兵力10余万，其中有工程安装大队，这是第二炮兵工程技术总队的前身。

1961年1月，上级决定7169部队机关从[[内蒙古自治区|内蒙古自治区]][[额济纳旗|额济纳旗]]<ref>{{Cite web|url = http://www.ejnq.gov.cn/Item/18803.aspx|title = 额济纳旗东风革命烈士陵园被命名为第二批“内蒙古中共党史教育基地”|author = |date = 2016-02-15|publisher = 中国额济纳|language = |accessdate = |quote = }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>移驻[[河南省|河南省]][[洛阳市|洛阳市]]。1962年春，部队进驻洛阳市龙门新草店。1962年11月，7169部队改为中国人民解放军工程兵工程部，部长[[赵东寰|赵东寰]]，该部是军级单位，代号中国人民解放军8277部队，工程安装大队是工程部下属单位。

随着二十基地、二十一基地工程建设任务不断增加，工程安装大队难以胜任，工程兵乃向中央军委提交了扩建工程安装大队的报告。1962年10月18日，[[中华人民共和国国务院副总理|国务院副总理]]兼[[中国人民解放军总参谋长|中国人民解放军总参谋长]][[罗瑞卿|罗瑞卿]]签署命令，在工程兵特种指挥部工程技术大队和特种工程指挥部直属分队的基础上，组建'''中国人民解放军工程兵工程技术总队'''，代号'''中国人民解放军总字三五二部队'''，为师级单位，负责原子弹、导弹和人造地球卫星试验塔架的安装任务。随后，该总队为中国第一次核试验安装铁塔和起爆装置<ref name=shouji>{{Cite web |url =http://news.mod.gov.cn/forces/2010-04/12/content_4139340.htm  |title = 走近导弹筑巢人——二炮工程部队采访手记 |author =  |date =2010-04-12  |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref><ref name=daodan/>。

1964年10月16日，中国第一颗原子弹爆炸成功。工程兵工程技术总队70多名官兵受到[[周恩来|周恩来]]等[[党和国家领导人|党和国家领导人]]接见<ref name=daodan/>。8277部队完成了导弹综合试验靶场和原子弹试验场国防工程任务，陆续从二十基地、二十一基地撤出。工程兵工程技术总队进驻河南省洛阳市龙门。

1966年7月1日，中国地地战略导弹部队和常规战役战术导弹部队成立，[[周恩来|周恩来]]定名为[[中国人民解放军第二炮兵部队|中国人民解放军第二炮兵部队]]，代号中国人民解放军总字一〇二部队<ref name=shouji/>。起初，第二炮兵部队导弹均采用固定发射，当时第二炮兵第五十一基地、第五十二基地、第五十三基地、第五十四基地、第五十五基地、第五十六基地分布在中国各地的深山中，各基地所属的建筑工程兵承担导弹发射坑道建设任务。工程兵工程技术总队6个大队分别配置到上述6个基地，承担地质勘探、地形地貌测绘、导弹发弹平台机电工程安装任务。

1970年底，工程兵工程技术总队机关从河南省洛阳市龙门移驻[[江苏省|江苏省]][[徐州市|徐州市]]东甸子原装甲兵坦克学校营房。1975年8月，奉命迁回河南省洛阳市，进驻徐家营原第二工程兵学校校址。

1975年9月28日，工程兵工程技术总队划归第二炮兵，专门从事各种导弹阵地的安装建设<ref name=daodan/>。1975年12月，根据中央军委决定，工程兵工程技术总队正式划归第二炮兵，作为第二炮兵直属部队，番号为'''中国人民解放军第二炮兵工程技术总队'''，代号改为中国人民解放军80590部队，下属6个大队在施工过程中归所在第二炮兵各基地的统一指挥。

1985年[[百万大裁军|百万大裁军]]中，第二炮兵工程技术总队代号改为中国人民解放军96531部队，下属各大队整建制调整成为团的建制。部队职能除原先的地质勘探、地形地貌测绘、导弹发弹平台机电工程安装之外，增加了通信安装任务，第二炮兵工程技术总队的工程安装业务归[[中国人民解放军火箭军装备部|第二炮兵装备部]]管理。

第二炮兵工程技术总队及其前身先后完成了中国第一颗和第三颗原子弹试验塔架、中国第一颗人造地球卫星发射塔架（位于今[[中国西昌卫星发射中心|中国西昌卫星发射中心]]）的建设任务。1968年至1972年，安装了当时世界上最大的预警雷达玻璃钢天线罩。转隶第二炮兵部队后，第二炮兵工程技术总队从安装中国第一批导弹阵地开始，先后完成不同型号导弹的阵地安装任务。1984年2月，总队受命安装某国防重点工程，1985年成功安装中国第一口某大型号导弹试验井，1994年12月总队提前1年完成该项国防重点工程<ref name=shouji/><ref name=daodan/>。该总队高标准建成了中国第一个导弹试验基地、第一代核武库、第一批导弹阵地，还参与建设了[[人民大会堂|人民大会堂]]、[[毛主席纪念堂|毛主席纪念堂]]、[[北京首都机场|北京首都机场]]等工程<ref name=shizong>{{Cite web |url =http://news.sohu.com/20160826/n466172747.shtml  |title =揭秘火箭军工程部队：有士兵“失踪”9年  |author =  |date =2016-08-26  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

1990年代末，第二炮兵工程技术总队奉命承接大批阵地工程建设任务。当时，该总队已经数年未承接大的施工任务，但该总队克服困难完成了任务。2000年，该总队被[[中华人民共和国建设部|中华人民共和国建设部]]评为“国家一级安装企业”<ref name=gouzhu>{{Cite web |url = http://news.sohu.com/85/51/news203735185.shtml |title = 构筑共和国“地下长城”――记第二炮兵某工程技术总队 |author =  |date = 2002-10-18 |publisher =  搜狐|language =  |accessdate =  |quote =  }}</ref><ref name=daodan/>。

2000年代至2010年代，该部队是全军地下工程建设的主体力量，分散在中国20多个省（自治区、直辖市）、数十个施工点位上，担负中国人民解放军第二炮兵部队（2015年12月改为[[中国人民解放军火箭军|中国人民解放军火箭军]]）所有导弹阵地和部分军委、战区地下指挥工程施工任务<ref name=shizong/>。

2012年3月，该总队改为'''中国人民解放军第二炮兵工程基地'''，下属各团改为旅<ref>{{Cite web |url =  http://www.lyfn.gov.cn/content.aspx?id=103|title =领导介绍：靳灿欣 市林业局党组成员、纪检组长  |author =  |date =  |publisher =洛阳市林业局  |language =  |accessdate = 2017-10-09 |quote =  }}</ref>。2015年12月，中国人民解放军第二炮兵部队改为中国人民解放军火箭军，该基地改为'''中国人民解放军火箭军工程基地'''，仍是全军唯一一支军级建制的工程部队<ref name=shizong/><ref>{{Cite web |url = http://www.xcitc.edu.cn/info/1032/2282.htm |title = 我校2016年大学生征兵工作圆满结束 |author =  |date = 2016-09-20  |publisher =许昌职业技术学院  |language =  |accessdate =  |quote =  }}</ref>。

2017年4月18日，[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军事委员会主席|中央军委主席]][[习近平|习近平]]在[[北京|北京]][[八一大楼|八一大楼]]接见新调整组建的84个军级单位主官，并对各单位发布训令；[[中共中央政治局委员|中共中央政治局委员]]、[[中央军事委员会副主席|中央军委副主席]][[许其亮|许其亮]]宣读习近平主席签发的中央军委关于调整组建军兵种部队和省军区系统军级单位的命令、决定<ref>{{Cite web |url =http://news.xinhuanet.com/politics/2017-04/19/c_129550371.htm  |title = 全军新调整组建84个军级单位，习近平发布训令 |author =  |date =  2017-04-19|publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。在这次军级单位调整组建中，以中国人民解放军火箭军工程基地为基础，调整组建'''中国人民解放军火箭军第六十八基地'''<ref>{{Cite web|url = http://www.lsj.gov.cn/Item/51970.aspx|title = 冷水江市人武部开展新兵回访“三个一”活动|author = |date = 2017-09-30|publisher = 中国冷水江|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20171009195336/http://www.lsj.gov.cn/Item/51970.aspx|archive-date = 2017-10-09|dead-url = yes}}</ref>。
