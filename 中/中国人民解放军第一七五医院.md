'''中国人民解放军联勤保障部队第九〇九医院'''，原名'''中国人民解放军第一七五医院'''，又名'''厦门大学附属东南医院'''，位于[[福建省|福建省]][[漳州市|漳州市]]漳华中路269号，隶属[[无锡联勤保障中心|无锡联勤保障中心]]，为[[三级甲等医院|三级甲等医院]]。

==简介==
中国人民解放军第一七五医院于1948年10月创建于[[吉林省|吉林省]][[蛟河市|蛟河县]]，其前身是[[东北军区|东北军区]]第三十九后方医院和第四十二后方医院，先后参加了[[锦州战役_(1948年)|锦州战役]]和[[辽沈战役|辽沈战役]]、[[平津战役|平津战役]]等战役。1949年9月，医院随大军南下，进驻[[江西省|江西省]][[南昌市|南昌市]]。1954年8月，统编为'''中国人民解放军第一七五医院'''。其间，医院先后派出人员参加[[抗美援朝战争|抗美援朝战争]]、[[援越抗美|援越抗美]]卫勤保障及承担收治江西剿匪伤员任务。1970年9月，调防到福建省漳州市<ref>解放军第一七五医院/厦门大学附属东南医院，药学服务与研究2015年第06期</ref>。

中国人民解放军第一七五医院曾是[[中国人民解放军第六军医大学|中国人民解放军第六军医大学]]教学医院。后来，该医院成为全军首批[[三级甲等医院|三级甲等医院]]，担负驻地陆海空三军医疗保障任务，并且是全军应急机动医院。医院还主办了由[[中华人民共和国卫生部|中华人民共和国卫生部]]主管的全国性期刊《中国骨与关节损伤杂志》<ref>张东涛，第一七五医院剪影，解放军医院管理杂志2005年03期</ref>。医院是[[厦门大学|厦门大学]]附属医院，称'''厦门大学附属东南医院'''。同时还是[[福建医科大学|福建医科大学]]等多所高等院校的教学医院<ref>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536727.htm |title = 第一七五医院 |author =  |date = 2014-09-12 |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第一七五医院由[[中国人民解放军南京军区|中国人民解放军南京军区]]转隶[[无锡联勤保障中心|无锡联勤保障中心]]<ref>{{Cite web |url = http://news.xinhuanet.com/mil/2017-05/18/c_129606950.htm|title = 联勤保障部队所属医院改善护理队伍结构 |author =  |date =  2017-05-18|publisher =  新华网|language =  |accessdate =  |quote =  }}</ref>。
