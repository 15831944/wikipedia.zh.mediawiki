'''中華民國與馬達加斯加關係'''是指[[中華民國|中華民國]]與[[馬達加斯加共和國|馬達加斯加共和國]]（1958－1975年稱'''{{link-en|馬拉加西|Malagasy Republic}}'''）之間的關係。兩國於1960－1972年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對馬達加斯加的相關事務由'''{{link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-173-3db90-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的{{link-en|馬拉加西|Malagasy Republic}}。]]
{{see also|中華民國外交|{{link-en|馬達加斯加外交|Foreign relations of Madagascar}}}}
{{see also|中国驻马达加斯加大使列表{{!}}中華民國駐馬拉加西外交官列表|马达加斯加驻华大使列表{{!}}馬拉加西駐中華民國外交官列表}}
在馬達加斯加獨立之前，設有'''中華民國駐[[法屬馬達加斯加|法屬馬達加斯加]][[領事館|總領事館]]'''。<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter1-3.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1960年6月26日，馬達加斯加獨立。同日，建立大使級[[外交#外交上的承認|外交關係]]。於首都[[安塔那那利佛|安塔那那利佛]]設立'''中華民國駐{{link-en|馬拉加西共和國|Malagasy Republic}}[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。<ref name="外交年鑑"/>

1962年，[[馬達加斯加總統|馬達加斯加總統]][[菲利贝尔·齐拉纳纳|齊拉納]]抵台訪問，並會見總統[[蔣中正|蔣中正]]。

1972年12月15日，與中華民國斷交。<ref name="外交年鑑"/>

1991年6月17日，於馬達加斯加首都設立[[實質大使館|具大使館性質]]的'''[[臺北經濟文化代表處|中華民國特別代表團]]'''。代表團除具商務、領務功能外，並享有[[外交豁免權|外交優遇]]。1998年9月9日，馬達加斯加內閣會議片面廢止《中馬特別協議》，使代表團功能受到很大限制。<ref name="外交年鑑"/>目前已關閉。

1997年1月，馬達加斯加副總理阿坎姆夫婦應[[世界自由民主聯盟|世界自由民主聯盟]]（世盟）邀請訪台。<ref name="外交年鑑"/>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境馬達加斯加。]]
[[File:Visa_requirements_for_Malagasy_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|馬達加斯加簽證政策|Visa policy of Madagascar}}|{{link-en|馬達加斯加公民簽證要求|Visa requirements for Malagasy citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。{{link-en|馬達加斯加政府|Politics of Madagascar}}沒有給予[[世界政區索引|任何地區]]的公民[[簽證#签证优惠待遇|免簽證]]待遇，持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可於抵達首都的[[安塔那那利佛/伊瓦圖國際機場|伊瓦圖國際機場]]時辦理[[簽證#落地签证|落地簽證]]入境，簽證費視停留效期而定。簽證相關事宜，可詢問馬達加斯加駐[[新加坡|新加坡]][[領事館|領事館]]。若由[[黃熱病|黃熱病]]疫區國家入境，則須出示黃熱病[[疫苗接種|疫苗接種]]證明（黃皮書）。<ref>{{Cite web |url = http://cms.olympicair.com/timatic/webdocsI/countryinfo.html |author = Information sourced and provided by [[國際航空運輸協會|IATA]] |title = Visa Information |publisher = {{link-en|Timaticweb|Timatic}} }}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-173-3db90-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|馬達加斯加護照|Malagasy passport}}的{{link-en|馬達加斯加人口|Demographics of Madagascar|馬達加斯加公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將馬達加斯加歸類為'''橙色警戒'''：避免非必要旅行（2017年10月31日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-260-fda2f-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2017年，馬達加斯加是中華民國第98大貿易夥伴、第82大[[進口|進口]]夥伴、第111大[[出口贸易|出口]]夥伴。出口至馬達加斯加的金額為1,817萬8,077[[美元|美元]]，年增9.219%；自馬達加斯加進口的金額為4,593萬5,511美元，年減4.332%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）2,775萬7,434美元，年減11.522%。※2016年，馬達加斯加是中華民國第95大貿易夥伴、第80大進口夥伴、第109大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
