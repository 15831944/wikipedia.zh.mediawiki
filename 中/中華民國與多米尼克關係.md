'''中華民國與多米尼克關係'''是指[[中華民國|中華民國]]與[[多米尼克國|多米尼克國]]之間的關係。兩國於1983－2004年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對多米尼克的相關事務由'''中華民國駐[[聖露西亞|聖露西亞]][[中華民國駐外機構列表|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-230-498ea-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，[[多米尼克|多米尼克]]當時屬於{{link-en|英領西印度群島|British West Indies|英國殖民地}}，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]

1983年5月9日，[[多米尼克總理|多米尼克總理]]{{link-en|尤金妮亞·查爾斯|Eugenia Charles|查爾斯}}女士訪台並與[[中華民國政府|中華民國政府]]簽署《聯合公報》。10日，兩國[[外交#外交上的承認|建立外交關係]]。1984年8月，於首都[[羅索|羅索]]設立'''中華民國駐多米尼克國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]（由[[安地列斯群島|安地列斯群島]]的[[中華民國外交#邦交國|邦交國]]大使兼任）。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

多米尼克則在首都[[台北市|台北]]設立'''駐華[[名譽領事|名譽總領事]]'''。1990年3月，聘任[[台灣裔|台灣裔]]商人董立群為名譽總領事。<ref name="86年外交年鑑"/><ref name="紀要"/>1999年4月，聘任[[台灣大哥大|台灣大哥大電訊公司]]董事長[[孫道存|孫道存]]為名譽總領事。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__06.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1995年9月，多米尼克遭受{{link-en|颶風路易士|Hurricane Luis}}侵襲，損失慘重，中華民國外交部長錢復致電該國總理表示關切，並從「國際災難人道救濟基金」損贈20萬[[美元|美元]]供賑災重建。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=93F4AEE6707DC131 |author = 公眾外交協調會 |date = 1995年9月11日 |title = 位於東加勒比海地區之我兩友邦多米尼克及聖克里斯多福本（九）月五日遭強烈颶風路易士侵襲 |publisher = 中華民國外交部}}</ref>

2002年9月14日，[[吐瓦魯|吐瓦魯]]、[[多米尼克|多米尼克]]、[[馬拉威|馬拉威]]等3國代表於[[聯合國大會|聯合國大會]]總辯論發表演說時為中華民國參與聯合國案執言。其中，多米尼克外交部長瑞斐爾表示，熱切期待[[中華民國在台灣|在台灣之中華民國]]得早日以[[联合国系统|聯合國家庭]]的[[聯合國會員國列表|正式會員國]]身分參加各項會議。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=9E57026FB421F554&sms=28A3E17C7AEEE92F&s=DCA41E6BD31E702E |author = 公眾外交協調會 |date = 2002年9月15日 |title = 我國友邦吐瓦魯、多米尼克、馬拉威等三國代表於本（九）月十四日在聯大總辯論發表演說時曾為我參與聯合國案執言。 |publisher = 中華民國外交部}}</ref>

2003年7月，多米尼克《太陽週報》報導[[中国驻巴巴多斯大使列表|中華人民共和國駐巴貝多大使]]表示，根據其援外政策，只要多米尼克與其建交，便會提供經濟援助。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=B81B5A2FC67CDD90&sms=175F59D3F37F17B5&s=B760EB252F80C56C |author = 公眾外交協調會 |date = 2003年11月24日 |title = 中國大陸阻撓我國際空間事例（92年7月至9月事例）|publisher = 中華民國外交部}}</ref>

2004年3月29日，多米尼克總理[[羅斯福·斯凱里特|史卡利]]清楚指出此次與[[中華人民共和國|中華人民共和國]]建交，是其所給予之援助折合1億1,750萬美元（約3億1,725萬[[東加勒比元|東加勒比元]]）。30日，中華民國與多米尼克斷交。<ref name="紀要">{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=70BCE89F4594745D&sms=700DE7A3F880BAE6&s=84FEF5B1FB14333C |author = 公眾外交協調會 |date = 2004年3月30日 |title = 簡部長有關我國與多米尼克終止外交關係記者會答詢紀要 |publisher = 中華民國外交部}}</ref>
