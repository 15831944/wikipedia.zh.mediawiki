'''中国人民解放军火箭军疾病预防控制中心'''，位于[[北京市|北京市]]，隶属[[中国人民解放军火箭军后勤部|中国人民解放军火箭军后勤部]]。

==简介==
前身组建于1980年12月，隶属[[中国人民解放军第二炮兵|中国人民解放军第二炮兵]]后勤部。2014年3月更名为'''中国人民解放军第二炮兵疾病预防控制中心'''。平时承担中国人民解放军第二炮兵核与推进剂卫生防护、坑道卫生防护、卫生监督监测、疾病防控、媒介生物防治、健康教育与心理卫生服务、突发公共事件应急处置等卫生防护防疫服务保障任务，战时担负卫生防护防疫应急支援保障任务<ref>牢记职责使命 实践强军目标——第二炮兵疾病预防控制中心建设成果展示，解放军预防医学杂志2014(6)</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，改称'''中国人民解放军火箭军疾病预防控制中心'''，隶属中国人民解放军火箭军后勤部<ref>{{Cite web |url = http://www.81.cn/jfjbmap/content/2017-05/20/content_177878.htm |title =让导弹号手轻装走上发射岗位  |author =  |date = 2017-05-20 |publisher =  中国军网|language =  |accessdate =  |quote =  }}</ref>。
