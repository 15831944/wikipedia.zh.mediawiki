'''中华人民共和国第十三届国务院'''，是[[中华人民共和国|中华人民共和国]][[第十三届全国人民代表大会|第十三届全国人民代表大会]]（2018年3月至2023年3月）所产生的同一时期的[[中华人民共和国国务院|中华人民共和国国务院]]（即[[中华人民共和国中央人民政府|中央人民政府]]）。第十三届[[中华人民共和国国务院总理|国务院总理]]由[[中国共产党第十九届中央委员会|中共第十九届中央]][[中国共产党中央政治局常务委员会|政治局常委]][[李克强|李克强]]担任。本届政府对[[第十三届全国人民代表大会|第十三届全国人民代表大会]]及其[[第十三届全国人民代表大会常务委员会|常务委员会]]负责<ref>{{Cite web |url = http://www.gov.cn/guowuyuan/2018-03/18/content_5275215.htm|title = 十三届全国人大一次会议决定李克强为国务院总理 |date = 2018-03-18  |publisher = 中国政府网 |accessdate = 2018-3-18 }}</ref>。

除国务院办公厅外，第十三届国务院设置[[国务院组成部门|组成部门]]26个、[[国务院直属特设机构|直属特设机构]]1个、[[国务院直属机构|直属机构]]10个、[[国务院办事机构|办事机构]]2个、[[国务院直属事业单位|直属事业单位]]9个、[[国务院部委管理的国家局|部委管理的国家局]]16个。

== 机构设置 ==
2018年3月17日[[第十三届全国人民代表大会第一次会议|第十三届全国人民代表大会第一次会议]]通过《第十三届全国人民代表大会第一次会议关于国务院机构改革方案的决定》，批准《国务院机构改革方案》<ref>{{Cite web |url =  http://www.npc.gov.cn/npc/dbdhhy/13_1/2018-03/14/content_2048552.htm|title = 关于国务院机构改革方案的说明——2018年3月13日在第十三届全国人民代表大会第一次会议上 |author =王勇  |date = 2018-03-13 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.npc.gov.cn/npc/xinwen/2018-03/17/content_2049996.htm|title =  第十三届全国人民代表大会第一次会议关于国务院机构改革方案的决定|author =  |date =2018-03-17  |publisher =  中国人大网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.npc.gov.cn/npc/xinwen/2018-03/18/content_2050371.htm|title =  国务院机构改革方案|author =  |date = 2018-03-17  |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>。根据此次改革方案，国务院正部级机构减少8个，副部级机构减少7个<ref>{{cite news|url=http://www.bjnews.com.cn/news/2018/03/13/478782.html|title=国务院机构改革方案“出炉” 正部级机构减少8个|work=新京报|date=2018-03-13}}</ref>。
