'''中華民國與加拿大關係'''是指[[中華民國|中華民國]]與[[加拿大|加拿大]]之間的關係。1913－1970年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後直至1980年代，在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===

{{see also|中華民國駐加拿大大使列表{{!}}中華民國駐加拿大外交官列表|加拿大駐中華民國大使列表{{!}}加拿大駐中華民國外交官列表}}
1913年，[[中華民國|中華民國]]繼承[[清朝政府|清朝政府]]於[[光緒|光緒]]卅四年（1908年）在[[英屬北美|英屬加拿大]]設置的[[領事館|總領事館]]，更名為'''中華民國駐英屬加拿大總領事館'''。1931年，頒布《[[1931年西敏法令|西敏法令]]》，[[加拿大|加拿大]]成為獨立[[自治領|自治領]]。1932年，駐加拿大總領事館改稱'''中華民國駐[[渥太華|渥太華]]總領事館'''。

1941年8月2日，兩國[[外交#外交上的承認|建立外交關係]]。<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/10.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>8月29日，總領事館升格為'''中華民國駐加拿大[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。1944年2月17日，公使館升格為'''中華民國駐加拿大[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。1946年，加拿大在[[南京市_(中華民國)|南京]]设立'''[[加拿大驻中华民国大使馆旧址|驻中华民国大使馆]]'''<ref>{{Cite web|url=http://www.njmgjz.cn/ggsg/b37|title=加拿大大使馆◎南京民国建筑网|accessdate=2018-05-01|work=www.njmgjz.cn|archive-url=https://web.archive.org/web/20141222161334/http://www.njmgjz.cn/ggsg/b37#|archive-date=2014-12-22|dead-url=yes}}</ref>。

1947年8月20日，加拿大軍事代表團自[[日本|日本]]到南京。<ref name="民國史大事記">{{cite book|editor=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編|title=《中華民國史大事記》|location=[[北京|北京]]|publisher=[[中華書局|中華書局]]|date=2011年7月}}</ref>{{rp|8399}}

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，加拿大仍與中華民國維持邦交。

1967年10月6日，行政院副院長[[黃少谷|黃少谷]]代表總統[[蔣中正|蔣中正]]赴加拿大訪問；9日，與加國總理[[萊斯特·皮爾遜|皮爾遜]]（Lester Bowles Pearson）會商兩國加強合作問題；17日，返回臺灣。<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|742}}。

1968年7月16日，駐加拿大大使[[薛毓麒|薛毓麒]]遞交[[備忘錄|備忘錄]]，對加國有[[外交#外交上的承認|承認]][[中共|中共]]意圖表示關切；1969年2月10日，對加國與中共進行建交接觸，再提抗議。<ref name="記" />{{rp|756}}

1970年10月13日，加拿大與[[中華人民共和國|中華人民共和國]]建交<ref>[http://www.gov.cn/test/2012-04/20/content_2117961.htm 中華人民共和國與各國建立外交關係日期簡表]</ref>，並與中華民國斷交，<ref name="外交年鑑"/>15日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

1971年10月25日，加拿大投票反對[[中國與聯合國#代表權問題|中華民國續留聯合國]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。

1976年7月，加拿大[[蒙特婁|蒙特婁]]舉行[[1976年夏季奧林匹克運動會|第二十一屆夏季奧林匹克運動會]]，然而賽前[[加拿大政府|加拿大政府]]禁止中華民國代表團以其[[中華民國國旗|國旗]][[中華民國|國號]]入境參加，要求改名為「[[台灣|台灣]]」，在溝通無效後中華民國代表團於7月16日宣布退出該屆奧運會。

加拿大支持台灣有意義參與[[世界衛生組織|世界衛生組織]]（WHO）；2007年5月14日，[[加拿大衛生部|衛生部長]]{{link-en|甘禮民|Tony Clement}}（Tony Clement）首度在[[世界衛生大會|世界衛生大會]]（WHA）發言支持台灣有意義參與WHO活動<ref name="96年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2007/html/08-1-5.htm |title = 《中華民國96年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>；2017年5月17日，外交部長[[方慧蘭|方慧蘭]]在國會表示支持臺灣參加WHO等[[國際組織|國際組織]]，後於2019年再度重申此立場<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref><ref>[http://www.ntdtv.com.tw/b5/20190413/video/243638.html?美英加歐公開挺台參與WHO%20台籲世衛速邀請 美英加歐公開挺台參與WHO 台籲世衛速邀請]</ref>；[[加拿大參議院|加拿大參]]、[[加拿大眾議院|眾議院]]也多次通過有關決議案。

持有[[加拿大護照|加拿大護照]]的[[加拿大人|加拿大公民]]也可以免簽證的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

此外，兩國皆提供各1,000名額給18－35歲青年為期1年的[[工作假期签证|度假打工簽證]]（必須未曾參與加拿大IEC計畫或取得中華民國青年交流停留簽證）。<ref>{{Cite web |url = http://www.youthtaiwan.net/WorkingHoliday/Cus_WHNation.aspx?n=4892E8B8F5C0E174&s=99C67AA646CA5A6D |title = 青年度假打工 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-82-46-6b27c-1.html |title = 加拿大籍人士申請中華民國青年交流事由簽證相關手續說明 |publisher = 外交部領事事務局}}</ref>

===投資===
根據經濟部投資審議委員會統計，1952年－2013年，[[臺灣商人|台商]]對加拿大的總投資金額為3億9,300萬美元，共80件。自1980年代中期以來，台商在加拿大投資經營規模擴大，涉入產業類別廣泛，特別是[[大溫哥華區域局|大溫哥華地區]]及[[大多倫多地區|大多倫多地區]]已成為台商在[[北美地區|北美地區]]經營的重鎮之一。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/ca/pdf/103-28Canada.pdf |date = 2014年12月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

==交流==
===僑民===
{{see also|臺灣裔加拿大人|臺灣裔{{!}}海外臺僑|加拿大華人}}
截至2017年，定居在加拿大的[[台灣人|台灣人]]估計約26萬人；在台灣的[[加拿大人|加拿大人]]（含[[雙重國籍|雙重國籍]]）則約6萬人。<ref name="106年外交年鑑"/>臺灣人移民加拿大與[[美國|美國]]相比則晚得多，是於1980年代才開始蓬勃發展。移民者多以高經濟、高知識水平者，故多聚集在溫哥華、多倫多、[[卡加利|卡加利]]等城市的郊外[[豪宅|高級住宅]]區。

===學術===
{{see also|台灣教育|{{link-en|加拿大教育|Education in Canada}}}}
兩國定期召開雙邊[[高等教育|高等教育]]會議。[[加拿大政府|加拿大政府]]每年提供台灣學者「加拿大研究獎助金」；2004年起，[[中華民國政府|中華民國政府]]則開辦「臺灣[[獎學金|獎學金]]」及「[[中華民國國語|華語文]]獎學金」給予加拿大學生赴台研習。[[中華民國教育部|中華民國教育部]]與多所加拿大大學簽署設立「臺灣研究[[講座|講座]]」協議。截至2012年，[[加拿大大學列表|加拿大68所大學]]與[[台灣大專院校列表|台灣50所大學]]共簽署148項學術合作協議。<ref name="101年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2012/2.1.5.pdf |title = 《中華民國101年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>截至2017年，加拿大在台灣的[[留學生|留學生]]約600餘人；台灣在加拿大的留學生約3,000人。<ref name="106年外交年鑑"/>

2017年10月，中華民國科技部長[[陳良基|陳良基]]率團前往加拿大，參與和{{link-en|加拿大國家研究院|National Research Council (Canada)}}共同舉辦的「臺加科技合作20周年慶祝活動」及「[[医疗器械|醫療器材]][[物聯網|物聯網]]應用雙邊研討會」開幕儀式。<ref name="106年外交年鑑"/>

===觀光===
台灣人每年前往加拿大從事觀光、商務考察、探親及留學等各種活動約7萬人次；根據[[中華民國交通部觀光局|中華民國交通部觀光局]]統計，2017年加拿大赴台旅客則超過11萬人次。<ref name="106年外交年鑑"/>

===駕車===
持有中華民國[[國際駕駛執照|國際駕駛執照]]可在加拿大駕車。中華民國與加拿大[[魁北克省|魁北克省]]、[[安大略省|安大略省]]、[[紐芬蘭與拉布拉多省|紐芬蘭與拉布拉多省]]同意免試相互承認[[駕駛執照|駕駛執照]]，凡持有中華民國核發的駕駛執照且經由{{link-en|駐加拿大台北經濟文化代表處|Taipei Economic and Cultural Office in Canada}}驗證，並已取得該省[[居留證|居留證]]，可向該省申請免試換發駕駛執照。以[[簽證#签证优惠待遇|免簽證]]或{{link-en|停留簽證|Travel_visa#Short-stay_or_visitor_visas}}入境，不適用魁北克省免試換照。<ref name="生活資訊"/>
