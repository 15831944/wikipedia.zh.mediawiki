'''中華民國與不丹關係'''是指[[中華民國|中華民國]]與[[不丹王國|不丹王國]]之間的關係。兩國從未有正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對不丹的相關事務由'''[[駐印度台北經濟文化中心|駐印度台北經濟文化中心]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-123-d5f77-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[不丹|不丹]]。]]
{{see also|中華民國外交|不丹外交}}
不丹是唯一既沒有與中華民國[[外交#外交上的承認|建立外交關係]]，也沒有與[[中華人民共和國|中華人民共和國]]建交的[[聯合國會員國列表|聯合國會員國]]。

1971年10月25日，雖然不丹與中華人民共和國未建立外交關係，但支持中華人民共和國成為聯合國會員國的「[[中國|中國]]」席次，反對[[中國與聯合國#代表權問題|中華民國續留聯合國]]。

1993年9月11日，簽訂《中華民國台灣與不丹王國間交換國際[[特快專遞|快捷郵件]]服務瞭解[[備忘錄|備忘錄]]》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e4%b8%8d%e4%b8%b9&tycountry_e=%e4%b8%8d%e4%b8%b9&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

1994年3月18日，不丹公主[[旺楚克王朝|王巧克]]因[[走私|走私]][[犀牛|犀牛]]角及[[熊膽|熊膽]]抵台被法院[[有期徒刑#中華民國|判刑]]10個月。<ref>{{Cite web |url = http://memory.ncl.edu.tw/tm_cgi/hypage.cgi?HYPAGE=toolbox_vnews_detail.hpg&subject_name=%E8%87%BA%E7%81%A3%E6%96%B0%E8%81%9E%E5%BD%B1%E9%9F%B3%E8%B3%87%E6%96%99&dtd_id=8&project_id=news&xml_id=0000103906 |date = 1994年3月18日 |title = 不丹公主走私被判刑十個月 |publisher = [[國家圖書館_(中華民國)|國家圖書館]]}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境不丹。]]
[[File:Visa_requirements_for_Bhutanese_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|不丹簽證政策|{{link-en|不丹公民簽證要求|Visa requirements for Bhutanese citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。不丹只開放[[印度人口|印度公民]]、{{link-en|孟加拉人口|Demographics of Bangladesh|孟加拉公民}}、{{link-en|馬爾地夫人口|Demographics of the Maldives|馬爾地夫公民}}[[簽證#签证优惠待遇|免簽證]]待遇，且{{link-en|不丹政府|Politics of Bhutan}}不接受個人申辦簽證，須由當地[[旅行社|旅行社]]安排並每天消費約250[[美元|美元]]（約[[新臺幣|新臺幣]]7,600元），出發時需持旅行社所辦之簽證影本始可登機。經不丹轉機則可上網申請[[過境簽證|過境簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-123-d5f77-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|不丹護照|Bhutanese passport}}的{{link-en|不丹人口|Demographics of Bhutan|不丹公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將不丹歸類為'''灰色警示'''：提醒注意（2017年6月1日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-10-bdbc8-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|不丹經濟}}
===貿易===
2012年11月2日，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）首度邀請不丹商工會[[會長|會長]]率領17家廠商來台採購[[食品加工|食品加工]]機械、[[保全|安全器材]]、[[電子產品|電子]]、[[家庭劇院|家庭劇院]]、[[園藝|園藝]]、[[成衣|成衣]]及[[医疗器械|醫療]]等產品，成交金額超過3,300萬[[新臺幣|新臺幣]]。與外貿協會[[秘書長|秘書長]]共同簽署合作協議，加強雙方經貿合作，並前往[[高雄市|高雄]]參觀國際食品展。<ref>{{Cite web |url = http://www.trademag.org.tw/content02.asp?id=596869&type=49&url=/NewsSearch.asp |author = 李小慈 |date = 2012年11月14日 |title = 貿協與不丹商工會簽訂合作協議 |publisher = 經貿透視雙周刊第355期}}</ref><ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=596995&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2012年11月6日 |title = 貿協力邀 不丹王國商工會首次組團來台採購 |publisher = 經貿透視網}}</ref>

2018年，不丹是中華民國第191大貿易夥伴、第208大[[進口|進口]]夥伴、第180大[[出口贸易|出口]]夥伴，在[[亞洲國家和地區列表|亞洲國家]]排名倒數第3、3、4；在[[世界政區索引|世界經濟體]]倒數第54、33、57。出口至不丹的金額為88萬4,166[[美元|美元]]，年增555.112%；自不丹進口的金額為2萬7,214美元，年增44.709%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）85萬6,952美元，年增637.747%。2017年，不丹是中華民國第216大貿易夥伴、第209大進口夥伴、第207大出口夥伴，在亞洲國家排名倒數第1、2、2；在世界經濟體倒數第23、28、24。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
