'''中華民國與象牙海岸關係'''是指[[中華民國|中華民國]]與[[象牙海岸|象牙海岸共和國]]（[[音譯|音譯]]'''科-{}-特迪瓦'''；[[意譯|意譯]]'''象-{}-牙海岸'''）之間的關係。兩國於1963－1983年有官方[[外交#外交上的承認|外交關係]]。斷交後，在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構，但中華民國方面已於2017年關閉辦事處，目前對象牙海岸的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-241-cb0b3-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[象牙海岸|象牙海岸]]。]]
{{see also|中華民國外交|{{link-en|象牙海岸外交|Foreign relations of Ivory Coast}}}}
{{see also|中国驻科特迪瓦大使列表{{!}}中華民國駐象牙海岸外交官列表|科特迪瓦驻华大使列表{{!}}象牙海岸駐中華民國外交官列表}}
1963年7月20日，建立大使級[[外交#外交上的承認|外交關係]]。10月，於行政首都[[阿必尚|阿必尚]]設立'''中華民國駐象牙海岸共和國[[外交代表機構#大使馆|大使館]]'''。

1965年5月，派駐[[特命全權大使|大使]]。

1967年12月，[[總統府秘書長|總統府秘書長]][[張群|張群]]任慶賀[[利比里亚总统列表|賴比瑞亞總統]]、{{Link-en|賴比瑞亞副總統|Vice President of Liberia|副總統}}就職典禮特使，順道訪問象牙海岸。<ref>{{Cite web|url=http://www.history.com.tw/pe/11/11001301.htm|title=張群小傳|accessdate=2017-03-29|author=|date=|work=www.history.com.tw|publisher=}}</ref>

1983年3月3日，與中華民國斷交。4月17日，關閉駐在大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

1993年9月，象牙海岸[[常駐聯合國代表|常駐聯合國代表]]在第48屆[[聯合國大會|聯合國大會]]的總辯論中，提出「會籍普遍化原則」，間接表示支持[[中國與聯合國#台灣人民代表權問題及台灣入聯運動|中華民國參與聯合國]]。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=01CE54E75B2C7C88 |author = 公眾外交協調會 |date = 1993年10月14日 |title = 聯大第四十八屆常會總辯論自本年九月廿七日至十月十三日止 |publisher = 中華民國外交部}}</ref>

1995年10月，象牙海岸等20國在聯合國成立50週年特別紀念會議中，以直接或間接的方式為中華民國參與聯合國案執言。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=D7D21691928A05DE |author = 公眾外交協調會 |date = 1995年10月26日 |title = 聯合國成立五十週年聯大特別紀念會議自十月二十二日至十月二十四日召開期間 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#电子签证|電子簽證]]的方式入境象牙海岸。]]
[[File:Visa_requirements_for_Ivorian_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|象牙海岸簽證政策|Visa policy of Cote d'Ivoire}}|{{link-en|象牙海岸公民簽證要求|Visa requirements for Ivorian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以[[簽證#电子签证|電子簽證]]的方式入境象牙海岸的[[阿必尚國際機場|阿必尚國際機場]]，停留最多90天。也可至駐[[台北市|台北]]的象牙海岸共和國名譽[[領事館|領事館]]申請實體簽證，申辦相對便利，停留最多60天。該國是[[黃熱病|黃熱病]]疫區，入境須出示黃熱病的[[疫苗接種|疫苗接種]]證明（黃皮書）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-241-cb0b3-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ElectronicVisa.aspx?n=6F246645F10311D7&AreaSN=D33B55D537402BAA |title = 電子簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|象牙海岸護照|Ivorian passport}}的{{link-en|象牙海岸人口|Demographics of Ivory Coast|象牙海岸公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將象牙海岸歸類為'''橙色警戒'''：避免非必要旅行（2016年10月27日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-317-f8ea6-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將象牙海岸歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=815FF1A9B3BC5158 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|象牙海岸經濟}}
===貿易===
2017年，象牙海岸是中華民國第119大貿易夥伴、第126大[[進口|進口]]夥伴、第107大[[出口贸易|出口]]夥伴。出口至象牙海岸的金額為2,202萬4,649[[美元|美元]]，年減4.414%；自象牙海岸進口的金額為462萬9,853美元，年增9.121%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1,739萬4,796美元，年減7.469%。※2016年，象牙海岸是中華民國第116大貿易夥伴、第128大進口夥伴、第103大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
自[[象牙海岸革命戰爭|2002年內亂]]後，許多[[台灣商人|台商]]已撤離象牙海岸，有些選擇移居至他國，而多數則返回台灣另謀生計。目前當地台商及眷屬人數已自1980－1990年代最盛時期的300多人，剩下目前的23人（13戶）。經營的產業包括：[[国际贸易|進出口貿易]]、[[機械|機械]]進口[[代理|代理]]、[[橡膠|橡膠]]生產、[[蕃茄醬|蕃茄醬]]生產、[[農場|農場]]、[[餐馆|餐廳]]、[[糕點店|麵包店]]及冷凍[[漁業|漁產]]進口等，總投資金額約2,000萬美元。由於許多台商第1代已屆[[退休|退休]]年齡，因此事業經營呈現半停滯狀態；台商第2代有些在[[美國|美國]]、[[台灣|台灣]]及[[中华人民共和国|中國大陸]]等地發展，另有些則在[[西部非洲|西非國家]]從事[[貿易|貿易]]，總之，當地台商的商業活動並非十分活絡。<ref name="投資環境簡介"/>

目前舉行的有「台象經濟聯席會議」；<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>已成立的組織有「象牙海岸台灣商會」。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/ci/pdf/104-71IvoryCoast.pdf |date = 2015年8月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>
