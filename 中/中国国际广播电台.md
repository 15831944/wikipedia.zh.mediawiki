'''中国国际广播电台'''（简称'''国际台'''，中央广播电视总台及其所属中央电视台提到时称'''国广'''；{{lang-en|'''C'''hina '''R'''adio '''I'''nternational}}，{{lang|en|'''CRI'''}}）成立于1941年12月3日，前身是'''北京广播电台'''（{{lang-en|Radio Beijing/Radio Peking}}），是[[中国|中国]]唯一国家级从事对外广播的国家电台，它隶属于[[中央广播电视总台|中央广播电视总台]]。中国国际广播电台與[[中央人民廣播電台|中央人民廣播電台]]同為全国性[[广播电台|广播电台]]网络。

== 历史 ==
* 1941年12月3日，[[延安新华广播电台|延安新华广播电台]]的日语广播开播，首位播音员为日籍[[八路军|八路军]]女兵[[原清志|原清志]]（后加入[[中国国籍|中国国籍]]）。这一天后来也被中国国际广播电台认定为其开播日<ref>[http://gb.cri.cn/42071/2015/08/18/5311s5070493.htm 【老友记】原清志：帮中国人打日本鬼子是我最正确的选择(组图)]，国际在线</ref>。
* 1950年4月10日，'''中央人民广播电台'''的对外广播开始使用'''北京广播电台'''（'''Radio Peking'''）呼号，[[调谐信号|广播开始曲]]是歌曲《[[东方红|东方红]]》的钟声（开头两小节）。在那时，它在[[阿尔巴尼亚|阿尔巴尼亚]]有唯一的[[中继站|中继站]]。同时，'''北京广播电台'''开始对[[东南亚|东南亚]]广播。
* 1958年9月，位于[[北京|北京]][[复兴门|复兴门]]外的[[广播大厦|广播大厦]]建成，'''北京广播电台'''迁入该楼办公直至1997年，该大楼目前隶属于[[國家廣播電視總局|國家廣播電視總局]]。
* 1962年4月，'''北京广播电台'''在日语广播中开设“汉语讲座”，这是中国在对外广播中开设汉语教学节目的开始。
* 1966年，'''北京广播电台'''开始曲《[[东方红_(歌曲)|东方红]]》的钟声改为开头四小节，并播放《东方红》的完整管弦乐版。
* 1978年4月18日，正式更名为“'''中华人民共和国国际广播电台'''”，简称'''中国国际广播电台'''，并从[[中央人民广播电台|中央人民广播电台]]分离出来。
* 1983年，'''中国国际广播电台'''的英文名称改为“'''Radio Beijing'''”。
* 1984年，'''中国国际广播电台'''开始对首都北京地区正式进行英语广播，呼号为“Radio Beijing's Capital Service”。
* 1993年1月1日，中国国际广播电台的英文名称改为“'''China Radio International'''”，英文简称为“'''CRI'''”。
* 1997年，'''中国国际广播电台'''新大楼在'''石景山路甲16号'''建成，并迁入该楼办公至今。
* 2018年3月21日，根据《深化党和国家机构改革方案》，原[[中央电视台|中央电视台]]（[[中国国际电视台|中国国际电视台]]）、[[中央人民广播电台|中央人民广播电台]]、中国国际广播电台合并，组建新的[[中央广播电视总台|中央广播电视总台]]，作为[[国务院直属事业单位|国务院直属事业单位]]，归口[[中共中央宣传部|中共中央宣传部]]领导。原中央电视台（中国国际电视台）、中央人民广播电台、中国国际广播电台建制撤销，但对内保留原呼号，对外统一呼号为“中国之声”<ref>{{cite web |url = http://tech.ifeng.com/a/20180321/44915086_0.shtml |title=我国组建中央广播电视总台：中央电视台等撤销建制 |author=驱动之家 |website =驱动之家}}</ref>。中国国际广播电台在'''石景山路甲16号'''的办公区更名为[[中央广播电视总台鲁谷办公区|中央广播电视总台鲁谷办公区]]。<ref>{{cite news|title=中央广播电视总台正式揭牌亮相 三台新牌细微不同|url=http://news.china.com/news100/11038989/20180419/32327256.html|accessdate=2018-05-26|agency=澎湃新闻网|date=2018-04-19}}</ref>

== 发展 ==
中国国际广播电台在世界重要国家和地区建有32个驻外记者站，并在中国各省、市、自治区以及[[香港|香港]]、[[澳门|澳门]]特别行政区建有记者站，拥有庞大的信息网。

它每天以全世界43种语言广播1100多小时的节目，包括新闻、时事和有关政治、经济、文化、科学和技术的专题节目。对周边国家和地区播出。

中国国际广播电台采取[[短波|短波]]、[[中波|中波]]和[[调频|调频]]几种办法对外广播；对于远距离广播，采用大功率发射机和将节目送上卫星的办法，实施全球覆盖。

1995年，国际台将24套节目送上亚洲二号卫星；

1998年，开始租用泛美公司的泛美二号、三号、四号、五号卫星来传送节目。

CRI是全[[亚洲|亚洲]]最完备的国际广播服务，目前大约使用超过50台[[短波|短波]]发射机。

2011年，中国国际广播电台华语台之方言广播在原CRI海峡飞虹网络电台的基础上建立[[中国国际广播电台世界华声|中国国际广播电台世界华声]]以及同名的[[网络电台|网络电台]]，每天通过网络向全球播音。现拥有：[[闽南语|闽南语]]（[[厦门话|厦门话]]、[[潮州话|潮州话]]、[[台灣話|台灣話]]）、[[粤语|粤语]]（[[广府话|广府话]]）、[[客家话|客家话]]等四种[[语言|语言]]的短波、落地调频广播节目和相应的网络广播，另外还开办有一小时的[[温州话|温州话]]广播节目。

中国国际广播电台对外广播使用调频、中波、短波三个波段。其中，调频广播主要是通过与海外广电部门合作的形式，开办落地广播，每天播出固定时间和频率的节目。中波广播有一部分是境内的无线电台发射，主要是针对周边国家进行广播，如[[日语|日语]]、[[韩語|韩語]]、[[俄语|俄语]]等，也有在海外落地的中波电台，如阿尔巴尼亚周边等。短波广播是最主要的形式，其短波发射台分布在陕西宝鸡、北京房山、内蒙古呼和浩特、浙江金华、新疆喀什、云南昆明、广西南宁、河北石家庄、新疆乌鲁木齐、陕西咸阳，以及海外的古巴、马里、阿尔巴尼亚三国。

== 新媒体业务 ==
=== 网站 ===
{{main|国际在线}}
中国国际广播电台的“[[国际在线|国际在线]]”网站是中国中央重点新闻网站之一。目前，已发展成为61种语言的世界语种最多的网站。“国际在线”的访问者来自世界160多个国家和地区，网上节目时长达到每天221.5小时，日均收听人次70万。“国际在线”于2005年7月13日正式开播了多语种网络电台 Inet Radio（www.inetradio.cn），使用汉语普通话、英语、德语和日语播出。节目内容由资讯、谈话、音乐和外语教学四大类节目组成。2006年7月13日，“国际在线”推出了汉语、英语、日语和朝鲜语国内首家多语种[[播客|播客]]。2007年9月14日，由9种语言组成的11个境外网络电台（CRI WebCast）正式开播。

=== 网络电台 ===
中国国际广播电台网络电台是中国国际广播电台旗下的多语种网络电台，于2005年7月13日正式开播。

=== 手机广播电视 ===
2007年4月6日，中国国际广播电台(CRI)宣布获得手机广播电视牌照（[[IPTV|IPTV]]），这是中国第三张全国性的手机牌照，前两张分别由上海文广传媒集团(东方龙)和[[中央电视台|中央电视台]]持有。

2007年4月和6月，CRI手机广播电视分别在[[中国联通|中国联通]]和[[中国移动|中国移动]]平台顺利上线。CRI手机广播电视除集成国际台自有的[[广播|广播]]、[[电视|电视]]、[[互联网|互联网]]以及多语种信息等资源外，还整合了地方电台、电视台的节目内容。其广播电视业务由中国国际广播电台下属的国视通讯(北京)有限公司(GTV)运营。

== 电视节目 ==
中国国际广播电台从1999年10月开始制作并向全国传送国际新闻电视节目。目前，中国国际广播电台电视中心每天制作和传送上星的电视节目已达5个多小时，用户遍及全国300多个频道。另外，其旗下国广东方CIBN亦开设多套面向IPTV或OTT播送的轮播频道。

中国国际广播电台的数字电视频道有：
* 环球奇观
* 环球购物（现更名“聚鲨环球精选”）
* 中国交通频道

**CIBN冰雪体育频道
**CIBN马拉松频道
**CIBN每日健身频道

== 出版业务==
=== 出版社 ===
中国国际广播出版社创办于1985年，是由国家新闻出版广电总局主管、中国国际广播电台主办的中央级综合社科类出版社，以出版介绍中国各方面情况的多种外文图书为主，兼顾翻译出版各国经济、文化类图书，以及教外国人学汉语、教中国人学外语的教科书、工具书和音像制品等。

=== 报纸 ===
中国国际广播电台于90年代初开始主办《[[世界新闻报_(中国)|世界新闻报]]》，此报由[[国家新闻出版广电总局|国家新闻出版广电总局]]主管，每周一、周四出版。除了在北京印刷外，《世界新闻报》在上海、广州、成都、沈阳、西安、杭州、长春、武汉等多个城市设有分印点。

== 台呼 ==
除对内广播及英文国际广播外，中国国际广播电台广播节目开始前的ID音乐均为根据国歌《[[义勇军进行曲|义勇军进行曲]]》改编的音乐。中文广播呼号与外语广播ID有所不同，前者为华语广播的宣传语“中国不远，就在耳边”+电台名称，后者为中文电台名称（一次或两次）+播出语言的电台名称。这一台标音乐已被注册为商标。
