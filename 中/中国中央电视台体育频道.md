'''中国中央电视-{台}-体育频道'''（频道呼号：'''CCTV-5 体育'''）是[[中国中央电视台|中国中央电视台]]拥有的一条以[[普通话|普通话]]广播为主的体育频道，该频道为中国中央电视-{台}-开播最早、现在规模最大、拥有世界众多顶级赛事中国大陆地区独家报道权的专业体育频道，于1994年3月1日试播、1995年1月1日正式开播，通过中星6B、鑫诺3号卫星覆盖全国，卫星信号加密播出。全天24小时播出，在[[中国中央电视台体育赛事频道|中央电视台体育赛事频道]]开播以前一直是唯一一个覆盖全中国大陆的专业体育频道。

== 历史与发展 ==
{{Note24hUTC+8}}

1995年1月1日，该频道正式开播，通过亚洲2号卫星覆盖全国。<ref>[http://media.sohu.com/04/17/news209401704.shtml 中央电视台简介]</ref>，以及中央广播电视塔覆盖北京及周边地区。

2004年5月10日，该频道在中央广播电视塔发射的模拟信号停播。

2004年[[2004年歐洲足球錦標賽|葡萄牙欧洲杯]]期间，该频道推出由张斌主持的赛事杂志类节目《[[豪门盛宴|豪门盛宴]]》，从此开始，每逢[[世界盃|世界杯]]与[[欧洲足球锦标赛|欧洲国家杯]]，该频道均会安排此节目作为赛事直播的垫场预热。

2005年9月5日，该频道改为24小时播出。

从[[2008年夏季奥林匹克运动会|2008年北京奥运会]]开始，该频道台标不再只是出现在左上端，而是根据转播的具体情况，将台标移动到右上端。如比分牌放在荧幕左上方，以前CCTV-5的台标会遮挡住比分牌，而之后会移动到右上方。这样的安排，同样适合于其他频道在直播体育比赛时，如CCTV-1、CCTV-2等频道。

2011年1月1日，央视频道正式调整而更换频道标志，其中原有的该频道标志变得更清晰且在下方加上“'''体育'''”二字，原有的频道标志继续作为宣传标使用。<ref>[http://yule.sohu.com/20110102/n278638372.shtml 央视更换新台标：加注中文标识 频道区分清晰]</ref>   

2012年5月16日，体育频道首先搬入[[中央电视台总部大楼|新台址]]，并承担[[第30届奥林匹克运动会|伦敦奥运会]]的转播工作<ref>[http://news.sina.com.cn/c/2012-05-17/040924429052.shtml 中央电视台新台址主楼竣工 体育频道将率先入驻]</ref>。

2012年9月28日，该频道进行高清和标清同步播出<ref>[http://www.sarft.gov.cn/articles/2012/09/29/20120929152944640839.html 多套高清同播频道上星播出] {{webarchive|url=https://web.archive.org/web/20121111074921/http://www.sarft.gov.cn/articles/2012/09/29/20120929152944640839.html |date=2012-11-11 }} 2012年9月29日，国家广播电影电视总局</ref>。

2014年1月15日0时7分28秒<ref>[http://sports.cntv.cn/2014/01/15/VIDE1389726959146237.shtml 斯诺克大师赛第一轮 丁俊晖VS墨菲2（时间轴15:03）]，央视网</ref>，该频道的标清版本的[[縱橫比_(圖像)|16:9]]节目播出方式由上下加[[黑边|黑边]]（{{lang|en|Letterbox}}）方式显示[[縱橫比_(圖像)|16:9]]画面改为将[[縱橫比_(圖像)|16:9]]播出素材横向压缩成[[縱橫比_(圖像)|4:3]]，但是[[台标|台标]]还是原先的4:3比例，导致用画面比例为16:9的设备收看该频道标清版时节目没有变形，但台标横向拉伸变形，直到2015年4月1日台标调整才得以解决。

2015年8月，该频道获[[新英体育|新英体育]]授权直播2015-16赛季[[英格兰足球超级联赛|英超联赛]]的中国大陆区免费场次，这也是英超时隔12年再次回到全国性的卫星频道，但央视同时亦失去当季[[西班牙足球甲级联赛|西甲联赛]]的版权。<ref>{{Cite web|url=http://sports.qq.com/original/inside/sd347.html|title=【深度】疯狂版权时代逼央视牵手英超_腾讯网|accessdate=2018-08-11|last=9:56|first=shenshenyan 2014-04-09|work=sports.qq.com}}</ref>

2017年8月27日，CCTV-5、5+在天津全运会赛事期间的赛事片头（包括《体育新闻》等栏目）出现单线CCTV标识。

2018年初，CCTV-5再次更换包装和频道ID，使用“新址台标+SPORTS”为概念宣传徽标，同时改版包装和节目角标。旗下的[[CCTV-5+|CCTV-5+]]和[[风云足球频道|风云足球频道]]也一同改版统一换装。

== 播出 ==
[[国家新闻出版广电总局|广电总局]]禁止除央视以外的所有电视台，直接购买[[奥运会|奥运会]]、[[亚运会|亚运会]]和[[世界杯足球赛|世界杯]]及其预选赛的转播权。<ref>[https://archive.is/20130101182212/http://law.people.com.cn/showdetail.action?id=2568591 国家广播电影电视总局关于加强体育比赛电视报道和转播管理工作的通知]</ref>在[[夏季奥林匹克运动会|夏季奥运]]赛事期间，台标和呼号会作相应更改，频道会改名为'''“奥运频道”'''，台标则改成台标中的字母“CCTV”下加挂奥运五环标志（2000年开始使用，初期为1998年6月1日启用的单线透明“CCTV”下加“奥运五环”的台标，2001年换为双线“CCTV”台标后改为使用双线“CCTV”下加“奥运五环”的台标）。从2018年起，逢[[冬季奥林匹克运动会|冬季奥运]]赛事期间，亦会更换为“奥运频道”台标（在此之前的冬奥会期间仅宣传片花使用）。
[[File:CCTV_Olympics.svg|thumb]]

在奥运频道台标期间，按照[[奥林匹克宪章|奥林匹克宪章]]规定，奥运五环标志不得用于商业用途，因此广告期间奥运五环台标会消失，直至节目继续时恢复台标显示。其次为了记分牌不被遮盖，奥运期间台标放置到右上方，待奥运结束后台标恢复左上方几天后恢复CCTV5台标。

与央视其它免费频道一样，CCTV-5会在北京时间每天早上6:00前后播放[[中华人民共和国国歌|中华人民共和国国歌]]。早期与其它频道播出的画面不同，同步播出的画面为中国运动员在[[奥运会|奥运会]]上夺取金牌的场面；现与央视其它频道一样，使用统一的国歌MV<ref>[http://www.tudou.com/programs/view/DvgQTX0D75w/ CCTV-5早间播放的国歌画面]</ref>。

== 争议 ==
===CCTV-5遭非法盗播===
{{seealso|对中国中央电视台的争议#央视版权保护|l1=央视版权保护}}

===全面抵制中国足球===
2008年11月开始，CCTV-5中止转播[[中超联赛|中超联赛]]，体育新闻节目中也不再出现任何中国足球联赛相关报道；中国俱乐部以及[[中国国家足球队|中国国家足球队]]参加的国际赛事随后也遭到封杀。据《[[新快报|新快报]]》援引央视内部人士的话称，由于中国足球频出乱象，触怒了上级领导，导致央视做出封杀决定。央视对中国足球的封杀直至2010年初才告解除。

===赛事转播权===
2018年下半年，随着[[苏宁易购|苏宁体育]]获得[[德甲|德甲]]、[[意甲|意甲]]及[[欧冠|欧冠]]在中国大陆地区的独家版权，央视也失去了上述赛事的转播权。[[欧洲五大联赛|欧洲五大联赛]]中，央视只能直播[[英超|英超]]的免费场次以及关注度相对较低的[[法甲|法甲]]赛事。上述赛事版权的失去也使得CCTV-5部分节目受到影响。如国际足球专题节目《[[天下足球|天下足球]]》几乎变成“英法足球”甚至“法甲精华”，欧冠专题节目《[[冠军欧洲|冠军欧洲]]》更是一度停播，体育新闻类节目中也长期不见相关报道。最终，经过与苏宁方面的协商，央视重新获得了欧冠和意甲的转播权。<ref>{{cite news |title=特大喜讯!央视终又能直播欧冠 C罗后天凌晨"登场" |url=http://sports.163.com/18/1106/08/DVTRD6MQ00058781.html |accessdate=2018-12-07 |publisher=网易 |date=2018-11-06}}</ref><ref>{{cite news |title=喜大普奔！央视再收一“失地”，明晚能在电视上看意甲 |url=http://finance.ifeng.com/a/20181201/16599112_0.shtml |accessdate=2018-12-07 |publisher=每日经济新闻 |date=2018-12-01}}</ref>
