[2016年1月22日] {{ro}}.</ref>，采用“[[黨指揮槍|黨指揮槍]]”的军事领导体系，由[[中央軍事委員會|中央軍事委員會]]领导全国武装力量，采取[[中国共产党中央军事委员会主席|中央军委主席]]负责制<ref>[[江澤民|江澤民]]. 《[[江澤民文選|江澤民文選]]》第1卷. 中國北京: [[人民出版社|人民出版社]]. 2006年8月1日: 第487頁. ISBN 978-7010056746 {{zh-cn}}.</ref>。中华人民共和国武装力量在[[国家安全|国家安全]]和发展战略全局中具有重要地位和作用，肩负着维护国家主权、安全、发展利益的使命和职责。<ref>[http://www.81.cn/jwgz/2015-05/26/content_6508471.htm 中国武装力量的多样化运用]</ref>

中国的军事战略上采取积极防御和[[人民戰爭|人民戰爭]]。现在[[中国军事|中国军事]]的各项理论是[[中国共产党|中国共产党]]及其军队在长期的敌强我弱条件下的战争实践中形成的。在不同时期和阶段，有不同的内容和表现形式。[[土地革命战争|土地革命战争]]时期，[[中国工农红军|中国工农红军]]初期作战采取“敌进我退，敌驻我扰，敌疲我打，敌退我追”的作战原则；[[反围剿|反围剿]]后为打破敌人“围剿”与进攻，红军采取了“诱敌深入”的战略方针，实行[[游击战|游击性的运动战]]。[[抗日战争|抗日战争]]时期，表现为[[中华民国|全国]]实行持久战的总方针和[[八路军|八路军]]、[[新四军|新四军]]执行“基本的是游击战，但不放松有利条件下的运动战”的方针。[[第二次国共内战|第二次国共内战]]时期，则是“以歼灭[[中国国民党|国民党]]有生力量为主而不是以保守地方为主”。[[抗美援朝战争|朝鲜战争]]中，[[中国人民志愿军|志愿军]]在战略反攻阶段实行“以[[运动战|运动战]]为主，与部分的[[阵地战|阵地战]]、敌后游击战相结合”的方针。在战略防御阶段实行“持久作战，积极防御”的方针。

[[中华人民共和国|中华人民共和国]]成立以后，[[中国人民解放军|中国人民解放军]]的军事战略仍然是积极防御，但情况和内容有了很大的不同。人民解放军在战略上实行防御、自卫和后发制人的原则，不仅是着眼于敌强我弱的形势，还主要是由于作为国家军队；已不是夺取政权问题，而是要维护国家主权和安全。
* 1956年，[[中共中央军委|中共中央军委]]制定了和平建设时期的第一个战略方针，即“积极防御”战略方针。
* 1960年代至1970年代，主要是由于中国安全环境的急剧恶化，另外对[[战争|战争]]爆发的危险性估计得过于严重，因而采取“积极防御、诱敌深入”的方针，立足于应付最困难的情况，准备敌人早打、大打、打[[核武器|核战争]]。
* 1980年代初，针对未来全面反侵略战争的特点，确立了积极防御战略方针，坚持“后发制人、坚持人民战争、坚持持久作战、立足于以劣势装备战胜敌人、立足于最复杂最困难情况下作战”的战略指导原则。虽然没有发生[[世界大战|世界大战]]和大规模的外敌入侵，但是人民解放军曾胜利地进行了几场边境自卫防御作战，积累了在现代局部战争中贯彻积极防御战略思想的丰富经验。

1980年代中期以来，依据[[邓小平|邓小平]]提出的在较长时间内不发生大规模的世界战争是有可能的战略判断，指导思想改变，即从时刻准备早打、大打、打核战争的临战状态真正转到和平时期现代化建设的轨道上来，从立足于随时准备对付敌人大规模入侵转为着重对付局部战争和军事冲突。进入1990年代，国际战略格局变化，军事领域发生变革。积极防御的内容上把基点放在打赢现代技术特别是高技术条件下的局部战争。

2010年代中期，中国军队的积极防御军事战略方针的原则调整为<ref name="mod 2015" />：

* 服从服务于国家战略目标，贯彻总体国家安全观，加强军事斗争准备，预防危机、遏制战争、打赢战争
* 营造有利于国家和平发展的战略态势，坚持防御性国防政策，坚持政治、军事、经济、外交等领域斗争密切配合，积极应对国家可能面临的综合安全威胁
* 保持维权维稳平衡，统筹维权和维稳两个大局，维护国家领土主权和海洋权益，维护周边安全稳定；努力争取军事斗争战略主动，积极运筹谋划各方向各领域军事斗争，抓住机遇加快推进军队建设、改革和发展
* 运用灵活机动的战略战术，发挥联合作战整体效能，集中优势力量，综合运用战法手段；立足应对最复杂最困难情况，坚持底线思维，扎实做好各项准备工作，确保妥善应对、措置裕如
* 充分发挥人民军队特有的政治优势，坚持党对军队的绝对领导，重视战斗精神培育，严格部队组织纪律性，纯洁巩固部队，密切军政军民关系，鼓舞军心士气
* 发挥人民战争的整体威力，坚持把人民战争作为克敌制胜的重要法宝，拓展人民战争的内容和方式方法，推动战争动员以人力动员为主向以科技动员为主转变
* 积极拓展军事安全合作空间，深化与大国、周边、发展中国家的军事关系，促进建立地区安全和合作架构。

== 军队体系 ==
=== 领导制度 ===
中华人民共和国成立以来的各部宪法都规定了国家武装力量的统率者。1949年《[[中国人民政治协商会议共同纲领|中国人民政治协商会议共同纲领]]》规定：[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]统率军队。《[[:s:中华人民共和国宪法/1954年|中华人民共和国1954年宪法]]》规定：[[中华人民共和国主席|中华人民共和国主席]]统率全国武装力量，及兼任[[中华人民共和国国防委员会|国防委员会]]主席。《[[:s:_中華人民共和國憲法/1975年|1975年宪法]]》规定：[[中国共产党中央委员会主席|中国共产党中央委员会主席]]统率全国武装力量。《[[:s:中华人民共和国宪法/1978年|1978年宪法]]》规定：[[中国共产党中央委员会主席|中国共产党中央委员会主席]]统率中华人民共和国武装力量。《[[:s:中华人民共和国宪法/1982年|1982年宪法]]》规定：[[中华人民共和国中央军事委员会|中华人民共和国中央军事委员会]]领导全国武装力量。

[[中国共产党|中国共产党]]对[[中华人民共和国|中华人民共和国]]的军队，即[[中国人民解放军|中国人民解放军]]和其他一切人民武装力量，拥有绝对领导权。中共的[[中国共产党中央军事委员会主席|最高军事领导人]]往往也是国家武装力量的实际指挥者。军队中組成人員有党员，並排建有党小组，在基层连队设立党支部，在团以上单位设立党委。在军队中设立的各级党委是军队中的统一领导和团结的核心：党委对所属部队的一切组织、一切部门、一切人员、一切工作实行统一领导，部队的一切重大问题，除紧急情况下可以由首长临机处置外，都必须先由党委作出决定。[[中国人民解放军|中国人民解放军]]的中共组织，根据[[中共中央|中共中央]]的指示进行工作。未经中共中央、[[中共中央军委|中共中央军委]]的授权，任何组织或个人不得插手、过问或处理军队问题，更不允许擅自调动和指挥军队。此外，[[中国共产党|中国共产党]]禁止除[[共青团|共青团]]之外的其他任何党派、政治团体、政治组织在军队中建立组织和发展成员；其他组织和团体的成员如果参加军队，必须与原来的组织脱离关系；军队中，未经党组织的批准，不允许建立任何性质和形式的小团体、小组织；只有[[中国共产党|中国共产党]]的组织才能委派军队中的各级领导干部，其他任何组织和个人都不允许向军队委派干部<ref> [http://www.china.com.cn/ch-zhengzhi/zhengzhi7.htm 军事制度] 中国网</ref>。

=== 指挥机构 ===
[[全国人民代表大会|全国人民代表大会]]是[[人民代表大会制度|最高国家权力机关]]，拥有战争与和平问题的最高决定权。[[全国人民代表大会常务委员会|全国人民代表大会常务委员会]]是全国人大的常设机关，拥有[[宣战|宣战]]权与[[动员|动员]]权、全国和省级[[紧急状态|紧急状态]]决定权。[[中华人民共和国主席|中华人民共和国主席]]是[[虚位元首|虚位元首]]，只根据[[全国人民代表大会常务委员会|全国人大常委会]]的决定宣布进入[[紧急状态|紧急状态]]，宣布战争状态，发布[[动员|动员令]]。

[[中华人民共和国中央军事委员会|中华人民共和国中央军事委员会]]（与[[中国共产党中央军事委员会|中国共产党中央军事委员会]][[一个机构两块牌子|一个机构两块牌子]]）领导和指挥全国武装力量，中央军委机关内设[[中央军委办公厅|中央军委办公厅]]等15个职能部门。

[[中华人民共和国国务院|中华人民共和国国务院]]领导和管理国防建设事业（其中，中央军委会同国务院管理国防经费和国防资产）。下设的[[中华人民共和国国防部|中华人民共和国国防部]]是国务院“主管全国国防工作的部门”；[[中华人民共和国工业和信息化部|工业和信息化部]]管理的[[国家国防科技工业局|国家国防科技工业局]]（前身为[[中华人民共和国国防科学技术工业委员会|国防科工委]]）管理国防科研、生产（由中央军委协同）；[[国务院议事协调机构|国务院议事协调机构]]：[[国家国防动员委员会|国家国防动员委员会]]、[[国家边海防委员会|国家边海防委员会]]。

接受国务院与中央军委共同领导[[中國民兵|民兵]]建设；[[征兵|征兵]]工作、[[预备役|预备役]]工作；[[武警边防部队|边防]]、[[海防|海防]]和[[空防|空防]]管理工作。

[[中国人民解放军军事检察院|军事检察院]]是国家设在军队中的法律监督机关，受[[中央军委政法委员会|中央军委政法委]]和[[最高人民检察院|最高人民检察院]]共同领导。2016年6月起，[[中华人民共和国军事法院列表|中华人民共和国的军事法院]]和[[中华人民共和国军事检察院列表|中华人民共和国的军事检察院]]进行了调整组建。调整后的军事法院分为高级军事法院、中级军事法院、基层军事法院三级。<ref>[http://military.china.com/news/568/20160718/23082794.html 军事法院检察院已调整组建运行 打破隶属着眼惩腐]，中华网，2016-07-18</ref>

中国奉行防御性的国防政策。新世纪新阶段中国的国防政策主要包括：维护国家安全统一，保障国家发展利益；实现国防和军队建设全面协调可持续发展；加强以信息化为主要标志的军队质量建设；贯彻积极防御的军事战略方针；坚持自卫防御的核战略；营造有利于国家和平发展的安全环境。

自1949年中华人民共和国成立后的前30年间，军费占国内生产总值的比例一直较高，保持在5%以上（按中国官方公布数值），从1980年代开始进行控制和缩减（参见[[百万大裁军|百万大裁军]]）。<ref>[http://www.globalview.cn/ReadNews.asp?NewsID=24036 新中国军费开支的历史演变] {{webarchive|url=https://web.archive.org/web/20140815145458/http://www.globalview.cn/ReadNews.asp?NewsID=24036 |date=2014-08-15 }}</ref>军费占国内生产总值的比例在1989年降为2.5%，再进一步降至最低值为1997年的1.6%。之后军费比例略有调高，从2001年起一直保持在2.0%－2.2%之间，军费的数额也跟随[[中华人民共和国经济|中国经济]]的成长而大幅增加。2011年，中国的国防军费为9230亿人民币，占[[中华人民共和国国内生产总值核算|中国国内生产总值]]的2.0%，列世界第二，占[[各国国防预算列表|世界各国国防预算总额]]的8.2%<ref>[[斯德哥尔摩国际和平研究所|斯德哥尔摩国际和平研究所]]</ref>。

== 武装力量 ==
{{main|中国人民解放军|中国人民武装警察部队|中国民兵}}

[[File:KJ2000_at_2014_Zhuhai_Air_Show.jpg|thumb]]是世界上第一款裝備有[[電子掃描陣列雷達|有源電子掃描陣列]]雷達的[[空中預警機|空中預警機]]。]]

《[[中华人民共和国宪法|中华人民共和国宪法]]》第二十九条规定，“中华人民共和国的武装力量属于[[人民|人民]]”。《[[s:中华人民共和国国防法|中华人民共和国国防法]]》第十九条规定：“中华人民共和国的武装力量受[[中国共产党|中国共产党]]领导。武装力量中的中国共产党组织依照[[中国共产党章程|中国共产党章程]]进行活动。”中共及军方坚定宣称并维护“[[中国共产党|党]]对军队的绝对领导（简称“党指挥枪”）。2013年4月16日中國國務院發表的《中國武裝力量的多樣化運用》白皮書中，首次披露海陸空三軍的人數：陸軍機動作戰部隊85萬人、海軍23.5萬人、空軍近39.8萬人。

现行宪法下，中国国家武装力量由“军、警、民”三支力量构成：即[[中国人民解放军|中国人民解放军]]（包括现役部队和[[预备役|预备役]]部队）、[[中国人民武装警察部队|中国人民武装警察部队]]和[[中国民兵|中国民兵]]。其中[[中国人民解放军|中国人民解放军]]是最主要的国家武装力量。解放军的管理思想是“[[中华人民共和国中央军事委员会|军委]]管总、[[中国人民解放军战区|战区]]主战、[[中国人民解放军军兵种列表|军种]]主建”。[[中华人民共和国中央军事委员会|中华人民共和国中央军事委员会]]{{notetag|与[[中国共产党中央军事委员会|中国共产党中央军事委员会]][[一个机构两块牌子|一个机构两块牌子]]。}}领导和指挥全国武装力量。中央军委机关内设[[中央军委办公厅|办公厅]]、[[中央军委联合参谋部|联合参谋部]]、[[中央军委政治工作部|政治工作部]]、[[中央军委后勤保障部|后勤保障部]]、[[中央军委装备发展部|装备发展部]]、[[中央军事委员会训练管理部|训练管理部]]、[[中央军事委员会国防动员部|国防动员部]]、[[中央军事委员会纪律检查委员会|纪律检查委员会]]、[[中央军事委员会政法委员会|政法委员会]]、[[中央军事委员会科学技术委员会|科学技术委员会]]、[[中央军事委员会战略规划办公室|战略规划办公室]]、[[中央军事委员会改革和编制办公室|改革和编制办公室]]、[[中央军事委员会国际军事合作办公室|国际军事合作办公室]]、[[中央军委审计署|审计署]]和[[中央军事委员会机关事务管理总局|机关事务管理总局]]等15个职能部门。中国人民解放军划分为[[东部战区|东部战区]]、[[南部战区|南部战区]]、[[西部战区|西部战区]]、[[北部战区|北部战区]]、[[中部战区|中部战区]]等五个战区。

[[File:H-6K_20211_20151127.jpg|thumb]][[远程轰炸机|远程轰炸机]]主要擔任戰術戰略轟炸、偵察、反艦、巡邏監視等多種任務，為中國核三位一體的重要組成部分。<ref>董磊. [http://www.cankaoxiaoxi.com/mil/20150817/907056.shtml 美媒称轰-6K堪称“中国版B-52”：可挂载2大杀器]. 《[[參考消息|參考消息]]》. 2015年8月17日 [2016年2月19日] {{zh-cn}}.</ref>。]]

[[File:Chinese_HQ-9_launcher.jpg|thumb]]遠程[[面對空導彈|面對空導彈]]发射车]]

[[File:DF-31_ballistic_missiles_20170919.jpg|thumb]]核[[洲際彈道飛彈|洲際彈道飛彈]]发射车]]

[[中国人民解放军|中国人民解放军]]现有五个[[军种|军种]]：[[中国人民解放军陆军|陆军]]、[[中国人民解放军海军|海军]]、[[中国人民解放军空军|空军]]、[[中国人民解放军火箭军|火箭军]]和[[中国人民解放军战略支援部队|战略支援部队]]。[[中国人民解放军陆军|陆军]]兵种有[[步兵|步兵]]、[[炮兵|炮兵]]、[[装甲兵|装甲兵]]、[[工程兵|工程兵]]、[[通信兵|通信兵]]、[[防化兵|防化兵]]、[[防空兵|防空兵]]、[[陆军航空兵|陆军航空兵]]等。[[中国人民解放军空军|空军]]兵种有歼击航空兵、轰炸航空兵、强击航空兵、特种航空兵、运输航空兵、[[空降部队|空降兵]]等。[[中国人民解放军海军|海军]]现今随着中华人民共和国的强大得到了应有的发展，造船速度与数量不断的提高、工艺水平也越来越先进、同时潜艇发展也格外迅速、以及越来越多高水平舰船入役如：[[052D型驱逐舰|052D型驱逐舰]]，[[054A型护卫舰|054A型护卫舰]]或未来的[[055型驱逐舰|055型驱逐舰]]，[[075型两栖攻击舰|075型两栖攻击舰]]，[[001A型航空母舰|001A型航空母舰]]等，堪与欧美匹敌，是少数拥有航母的国家。随着中华人民共和国的影响力不断加强，海军行动也越来越受外界关注。[[中国人民解放军火箭军|火箭军]]是中國的戰略導彈部隊，是中國人民解放軍的重要力量，武器主要為[[核武器|核武器]]和常規戰略導彈，包括[[短程彈道導彈|短程]]、[[中程彈道導彈|中程]]、[[遠程彈道導彈|遠程]]和[[洲際彈道導彈|洲際彈道導彈]]。

== 国防工业 ==

[[File:F-22P_PNS_Zulfiquar.JPG|thumb]]建造的F-22P型护卫舰]]

中国自身具备较为齐全的[[中华人民共和国工业|工业体系]]，21世纪的中国是武器进口大国，亦是武器出口大国。中国共产党在建国之前，就开始创建自己的军工系统。建国后，[[国防工业|国防工业]]得到政府的大力扶持。

[[国家国防科技工业局|国家国防科技工业局]]受国务院和中央军委的双重领导，管理有关工业部门的国防科研、军品生产和军品外贸。军事科研机构有[[中国人民解放军军事医学科学院|中国人民解放军军事医学科学院]]、[[中国人民解放军防化研究院|中国人民解放军防化研究院]]等。
