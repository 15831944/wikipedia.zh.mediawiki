'''中国人民解放军第二六四医院'''，位于[[山西省|山西省]][[太原市|太原市]][[迎泽区|迎泽区]]桥东街30号，隶属[[中国人民解放军中部战区陆军|中国人民解放军中部战区陆军]]后勤部，为综合性[[三级甲等医院|三级甲等医院]]。

==简介==
[[中华人民共和国|中华人民共和国]]成立前，[[山西省|山西省]][[太原市|太原市]]有一家[[阎锡山|阎锡山]]的军队医院——“'''三六医院'''”，主要收治在山西省的国民党中央军伤病员。1949年4月24日，[[中国人民解放军|中国人民解放军]]取得[[太原战役|太原战役]]胜利，攻占太原；当天，太行、太岳、晋绥等解放区医院的联合接管组接管了三六院，合并为太原市唯一的军队医院——'''山西省军区直属医院'''。当时该院仅有内科、外科、五官科共3个科室。1950年底，一批[[日本|日本]]籍医务人员来到该医院工作，他们是1945年[[抗日战争|抗日战争]]胜利后被进入[[中国东北|中国东北]]的[[中共|中共]]军队留用的原日本[[关东军|关东军]]医院人员。1950年底因[[抗美援朝战争|抗美援朝战争]]爆发，[[中华人民共和国政府|中华人民共和国政府]]将一批日本籍医务、技术人员从东北转移至华北、西北等地，其中数十位日本籍医务人员从东北调到山西省太原市的山西省军区直属医院及太原铁路医院。不久该院增建了科室，更名为'''山西省军区分科医院'''。1952年秋，中华人民共和国政府把在华的日本籍人员遣返回国，1952年9月8日该医院日本籍医务人员在回国前与医院中国籍人员合影留念（题为“中日医务工作者离别合影留念”）<ref name=wly>王立业，和日本友人共事的日子里，文史月刊2007(12):25-26</ref><ref>魏纯久、程太生，我和我的日籍老师，文史月刊2005(11):24-25</ref>。后来该医院更名为'''中国人民解放军第二六四医院'''<ref>{{Cite web|url = http://www.sxrb.com/sxwb/aban/09/382429.html|title = 后方医院就是我的家|author = |date = 2007-10-26|publisher = 山西新闻网|language = |accessdate = |quote = }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1996年12月，第二六四医院被[[中国人民解放军总后勤部卫生部|中国人民解放军总后勤部卫生部]]批准为[[三级甲等医院|三级甲等医院]]。截至2014年，是[[中国人民解放军第三军医大学|中国人民解放军第三军医大学]]、[[山西医科大学|山西医科大学]]、[[中国人民解放军白求恩军医学院|中国人民解放军白求恩军医学院]]、[[山西中医学院|山西中医学院]]、[[长治医学院|长治医学院]]等院校的教学医院；医院设有3个医学专科中心（创伤骨科研究诊疗中心、医学信息技术中心、优生优育技术中心）、3个医学专病中心（风湿病诊疗中心、外周血管疾病诊疗中心、消化内镜诊疗中心）、6个重点诊疗中心（烧伤整形诊疗中心、肝病诊疗中心、心脑血管介入诊疗中心、普外微创诊疗中心、肿瘤诊疗中心、口腔数字化诊疗中心），是全军远程医疗会诊中心及卫生信息技术培训基地<ref>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536789.htm |title =第二六四医院  |author =  |date = 2014-09-12 |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，医院由[[中国人民解放军北京军区|中国人民解放军北京军区]]转隶[[中国人民解放军中部战区陆军|中国人民解放军中部战区陆军]]<ref>{{Cite web |url =  http://zb.81.cn/content/2017-01/12/content_7448852.htm|title =  第264医院野战医疗所组织新年度冬季适应性训练|author =  |date =2017-01-12  |publisher =  军报记者|language =  |accessdate =  |quote =  }}</ref>。是[[山西省|山西省]]驻地最大的部队中心医院<ref>{{Cite web |url = http://zb.81.cn/content/2016-10/20/content_7314044.htm  |title =第264医院持续做好军队改革期间部队医疗保障  |author =  |date = 2016-10-20 |publisher = 军报记者 |language =  |accessdate =  |quote =  }}</ref>。
