'''中華民國與科威特關係'''是指[[中華民國|中華民國]]與[[科威特國|科威特國]]之間的關係。兩國於1963－1971年有官方[[外交#外交上的承認|外交關係]]，斷交後，中華民國於[[首都|首都]][[科威特市|科威特市]]設有[[實質大使館|具大使館性質]]的[[台北经济文化代表处|代表機構]]。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[科威特|科威特]]。]]
{{see also|中華民國外交|{{link-en|科威特外交|Foreign relations of Kuwait}}}}
1963年11月21日，兩國建立大使級[[外交#外交上的承認|外交關係]]，於首都[[科威特市|科威特市]]設立'''中華民國駐[[科威特國|科威特國]][[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/07.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

1971年3月22日，科威特与[[中华人民共和国|中华人民共和国]]建交。3月28日，中华民国與科威特中止外交關係。<ref name="外交年鑑"/><ref>{{cite book|title=《台灣歷史辭典》（附錄）|author=許雪姬|publisher=[[中華民國文化部|行政院文化建設委員會]]|chapter=中華民國歷任駐外大使(遷臺後，1949~2003.10)}}</ref>

===代表機構===
{{see also|中華民國駐外機構列表|駐臺灣外交機構列表}}

1986年4月23日，在科威特設立[[實質大使館|具大使館性質]]的'''中華民國駐科威特商務[[辦事處|辦事處]]'''（Commercial Office of the Republic of China in the State of Kuwait）。<ref name="外交年鑑"/>

1996年8月26日，更名為'''駐科威特王國[[台北经济文化代表处|台北商務代表處]]'''（Taipei Commercial Representative Office in the State of Kuwait）。<ref name="外交年鑑"/><ref name="駐外館處">{{Cite web |url = http://www.mofa.gov.tw/OverseasOffice_Content.aspx?n=6357834932B83C83&sms=5A961ED2E4BA25E8&s=CE8524192720696F&os=E13057BB37942D3F |title = 駐外館處 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>

====事件====
1990年8月，[[入侵科威特|伊拉克佔領科威特]]後，在該國的駐科威特商務辦事處、[[台灣中油|中國石油公司]]、[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]、[[中華工程公司|中華工程公司]]，其員工及眷屬與[[臺灣裔|僑民]]共143人，撤離科威特並抵達[[約旦|約旦]]，由[[中華民國政府|中華民國政府]]派遣專機接運抵臺。至[[波斯灣戰爭|伊拉克撤軍]]後，於1991年3月27日恢復辦事處運作。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-2.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境科威特。]]
[[File:Visa_requirements_for_Kuwaiti_citizens.png|thumb]]的方式入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|科威特簽證政策|{{link-en|科威特公民簽證要求|Visa requirements for Kuwaiti citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]前往觀光或商務需辦理停留簽證，可經由當地旅館或當地保證人代為申請，取得簽證後以影本辦理登機，抵達科威特至機場移民局領取簽證正本，辦理入境手續。經科威特轉機，不需申辦[[過境簽證|過境簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-46-a835f-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref>

持{{link-en|科威特護照|Kuwaiti passport}}的{{link-en|科威特人口|Demographics of Kuwait|科威特公民}}則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將科威特歸類為'''灰色警示'''：提醒注意（2017年4月18日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-180-83451-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將科威特歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2015年9月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=8405E523839DB5AE |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|科威特經濟|Economy of Kuwait}}}}
===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於[[科威特|科威特]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#MID-EAST |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

[[台灣中油|台灣中油公司]]與[[台塑石化|台塑石油公司]]分別向[[科威特石油公司|科威特石油公司]]（KPC）建立長期合作機制，每日向科威特購買原油約20萬[[桶_(單位)|桶]]。<ref>{{Cite web |url = http://www.cieca.org.tw/CountryInfoList.aspx |title = 國家基本檔(簡約版) |date = 2014年4月9日 |publisher = 中華民國國際經濟合作協會 |deadurl = yes |archiveurl = https://web.archive.org/web/20151022171920/http://www.cieca.org.tw/CountryInfoList.aspx |archivedate = 2015年10月22日 }}</ref>2002年，科威特石油公司[[執行長|執行長]]（[[薩巴赫家族|王儲]]兼[[科威特首相|首相]]之女）夏荷（SHAIKHA Saad Al-Sabah）抵台訪問。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__02.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2010年10月，在科威特舉辦首屆「[[台灣精品獎#相關台灣品牌評選|臺灣精品]]展」。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2010/2.1.2.pdf |title = 《中華民國99年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2014年11月16日，[[經濟部國際貿易局|經濟部國際貿易局]]委託外貿協會籌組「2014年[[中東|中東]]利基產業拓銷團」在科威特舉行該團第二站貿易洽談會。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=650705&url=/NewsSearch.asp |author = 陳思雅 |date = 2014年11月17日 |title = 台灣產品人氣旺 拓銷科威特商機達290萬美元 |publisher = 經貿透視網}}</ref>

2018年，科威特是中華民國第21大貿易夥伴、第11大[[進口|進口]]夥伴、第62大[[出口贸易|出口]]夥伴。出口至科威特的金額為1億4,630萬3,037美元，年增5.543%；自科威特進口的金額為51億3,435萬6,767美元，年增43.322%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）49億8,805萬3,730美元，年增44.843%。2017年，科威特是中華民國第24大貿易夥伴、第14大進口夥伴、第61大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
根據[[中華民國經濟部|經濟部]]投資審議委員會資料顯示，截至2013年12月底止，[[臺灣商人|台商]]在科威特並無任何投資。<ref>{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/kw/pdf/103-68Kuwait.pdf |title = 投資環境簡介 |publisher = 經濟部全球臺商服務網}}</ref>

2014年，[[中鼎工程|中鼎工程]]積極搶進科威特第4[[煉油廠|煉油廠]]興建計畫，並與國際大型公司聯手合作，共同搶進科威特的營建工程。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 主要產業概況 |publisher = 中華民國對外貿易發展協會}}</ref>

此外，[[資訊工業策進會|資訊工業策進會]]（資策會）與科威特中央科學研究院有長期合作關係，雙方就科技發展進行交流與合作開發。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref>
