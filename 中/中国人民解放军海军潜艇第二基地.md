'''中国人民解放军海军潜艇第二基地'''，位于[[海南省|海南省]][[三亚市|三亚市]]，是[[中国人民解放军南部战区海军|中国人民解放军南部战区海军]]潜艇基地。

== 历史 ==
中国人民解放军海军潜艇第二基地是继[[中国人民解放军海军潜艇第一基地|中国人民解放军海军潜艇第一基地]]（原名中国人民解放军海军潜艇基地）之后兴建的又一个核潜艇基地。2000年代初期，海军潜艇第二基地在三亚建设。截至2008年，中国094战略核潜艇已进驻新建的该基地<ref>{{cite web |url =  http://www.hinews.cn/news/system/2006/05/30/000105617.shtml|title = 外媒称三亚进驻核潜艇 中国海南岛成军事重镇 |author =  |date = 2006-05-30 |publisher = 南海网 |language =  |accessdate =  |quote =  }}</ref><ref>{{cite web |url = http://mil.news.sina.com.cn/p/2008-05-07/0725498804.html |title =  外媒称中国三亚超大海军基地可隐藏20艘核潜艇 |author =  |date = 2008-05-07 |publisher  =新浪  |language =  |accessdate =  |quote =  }}</ref><ref>{{cite web |url = http://phtv.ifeng.com/program/jqgcs/detail_2008_04/21/1057017_0.shtml |title = 马鼎盛质疑简氏“解放军三亚泊核潜艇” |author =  |date = 2008-04-21  |publisher = 凤凰网  |language =  |accessdate =  |quote =  }}</ref>。海军潜艇第二基地在海南省三亚市[[亚龙湾|亚龙湾]]，这里是亚洲最大的核潜艇基地，在亚龙湾东侧建有四条潜艇用栈桥，长度均为229米，合计可靠泊各型潜艇16艘<ref>{{cite web |url = http://qd.sohu.com/20140128/n394320771.shtml |title =曝中国三大核潜艇基地 青岛沙子口至少容纳六艘  |author =  |date = 2014-01-28 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.xinhuanet.com/mil/2013-12/19/c_125887208.htm |title = 俄媒：中国研制096核潜艇 6年后达俄军规模 |author =  |date =  2013-12-19|publisher =  新华网|language =  |accessdate =  |quote =  }}</ref>。

2013年4月9日，中共中央总书记、国家主席、中央军委主席[[习近平|习近平]]在出席[[博鳌亚洲论坛|博鳌亚洲论坛]]2013年年会后，到海军驻三亚部队视察，其间视察了海军潜艇第二基地<ref>{{cite web |url = http://news.sina.com.cn/o/2013-04-12/073926804174.shtml |title = 习近平视察南海舰队登上新型潜艇  |author =  |date =  
2013-04-12 |website = 新浪  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://mil.news.sina.com.cn/china/2016-01-11/doc-ifxnkvtn9751502.shtml |title = 习总授旗后战略核潜艇部队都做什么 已突破装备极限 |author = |date = 2016-01-11  |website = 新浪 |accessdate =  |quote =  }}</ref>。

南部战区海军某潜艇支队支队长[[王红理|王红理]]率372潜艇在执行任务途中，突遇重大险情，经过排险，圆满完成后续任务。2014年8月27日，[[中央军委主席|中央军委主席]][[习近平|习近平]]签署通令，为92474部队部队长王红理记一等功<ref>{{Cite web |url =http://cpc.people.com.cn/n/2014/0828/c64094-25556429.html|title=习近平签署通令 给1个单位、24名个人记功|author = |date = 2014年8月28日07:10|publisher = 人民网－人民日报 |language =  |accessdate =  |quote =  }}</ref>。2014年9月2日，中央军委给王红理、海军给372潜艇记一等功庆功大会在南海舰队某潜艇支队礼堂举行，中央军委委员、海军司令员[[吴胜利|吴胜利]]宣读了中央军委和海军的通令，分别给王红理和372潜艇颁发了奖章证书和奖状<ref>{{Cite web |url= http://www.mod.gov.cn/auth/2014-09/03/content_4534864.htm|title =中央军委给王红理、海军给372潜艇记一等功庆功大会举行 |author = 徐叶青、特约记者高毅|date = 2014-09-03 07:32:27|publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
