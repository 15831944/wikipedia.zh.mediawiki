'''中央农村工作领导小组'''，是[[中国共产党中央委员会|中共中央]]领导农村工作、农业经济的议事协调机构，负责对农村、农业经济工作领域的重大问题作出决策，负责历年“中央农村工作会议”所需相关文件的起草工作。

== 沿革 ==
1993年3月，中共中央决定，成立'''中央农村工作领导小组'''；领导小组由组长、副组长、组员组成。其中，组长一般由国务院主管农村工作的副总理担任；副组长由专职领导（正部级）担任；小组成员由[[中华人民共和国农业部|农业部]]、[[中华人民共和国水利部|水利部]]、[[国家林业局|国家林业局]]、[[中华人民共和国国家发展和改革委员会|国家发改委]]、[[中华全国供销合作总社|全国供销合作总社]]理事会、国家扶贫办以及国务院主管副秘书长等涉农部门领导担任<ref>{{Cite web |url =  http://china.huanqiu.com/article/2014-10/5169126.html|title = 田成平已卸任中央农村工作领导小组副组长 |author =  |date = 2014-10-16 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref>。

在设立之初，中央农村工作领导小组并无独立的日常办事机构，曾和[[中央财经领导小组办公室|中央财经领导小组办公室]][[合署办公|合署办公]]，人员交叉任职。自2004年起，中央农村工作领导小组开始单独下设办公室<ref>{{Cite web|url =http://www.infzm.com/content/101628/|title =中央财经领导小组掀开面纱|author =|date =2014-06-19|publisher =南方周末|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20160912174312/http://www.infzm.com/content/101628|archivedate =2016-09-12}}</ref>。中央农村工作领导小组办公室设在[[中华人民共和国农业农村部|农业农村部]]。
