'''中国载人航天计划'''（英文：China Manned Space，縮寫CMS）是[[中华人民共和国|中华人民共和国]]的载人航天计划，于1992年9月21日建立。计划分为三个步驟：第一步驟目标是将[[太空人|太空人]]送入[[太空|太空]]天地往返（神舟一號至神舟六號），第二步驟建立第一階段短期的[[天宮一號|天宮一號]]目標飛行器（神舟七號至神舟十號）、第二階段中期三十天為基準的[[天宮二號|天宮二號]][[天空實驗室|太空實驗室]]（神舟十一號、天舟一號），第三步驟建立長期[[中國太空站|天宮]][[太空站|太空站]]。<ref>{{Cite web|url=http://www.cmse.gov.cn/project/show.php?itemid=480|title=中国载人航天工程简介|publisher=中国载人航天工程网|date=2011-04-23|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20111223072747/http://www.cmse.gov.cn/project/show.php?itemid=480|archivedate=2011-12-23}}</ref>

==历史==
1986年，[[鄧小平|鄧小平]]批准实施[[国家高技术研究发展计划|国家高技术研究发展计划]]（代号：863计划），将发展航天技术列入其中。当时论证了不少方案，最后专家建议以载人飞船起步，最终建成空间站。1992年中央批准[[国防科工委|国防科工委]]组织载人航天工程技术经济可行性论证（代号 921），并形成技术经济可行性论证报告。<ref name=zjpxww>[http://dangjian.people.com.cn/n/2012/1207/c117092-19823834.html 载人航天扬国威——访中国载人航天工程总设计师周建平，中国共产党新闻网，2012-12-07]</ref>

1992年9月21日，[[中共中央政治局常委会|中共中央政治局常委会]]会议批准中国载人航天工程（代号：921工程）按「三步走」发展战略实施：<ref name=ytj>[http://www.thepaper.cn/newsDetail_forward_1316698 中国载人航天工程办公室换帅，余同杰接棒王兆耀，澎湃新闻，2015-03-31]</ref><ref name=jj>[http://www.cmse.gov.cn/art/2011/4/23/art_24_1054.html 中国载人航天工程简介，中国载人航天，2011-04-23]</ref>

*第一步，“发射载人飞船，建成初步配套的试验性载人飞船工程，开展空间应用实验”；
*第二步，“突破航天员出舱活动技术、空间飞行器的交会对接技术，发射空间实验室，解决有一定规模的、短期有人照料的空间应用问题”；
*第三步，“建造空间站，解决有较大规模的、长期有人照料的空间应用问题。”<ref name=jj/>

2005年2月，中央决策实施载人航天工程第二步第一阶段任务。<ref name=zjpxww/>

2010年9月25日，中共中央政治局常委会会议批准《载人空间站工程实施方案》，正式启动载人空间站工程。<ref name=jj/>

中国载人航天工程按照行政、技术两条指挥线组织开展研制、建设工作，设中国载人航天工程总指挥、总设计师（简称“两总”）联席会议制度。中国载人航天工程总指挥、总设计师联席会议研究决定中国载人航天工程实施过程中的重要问题，重大决策报[[中华人民共和国国务院|中华人民共和国国务院]]批准后实施。中国载人航天工程总指挥、副总指挥分由[[中国人民解放军总装备部|中国人民解放军总装备部]]、[[中华人民共和国工业和信息化部|中华人民共和国工业和信息化部]]、[[中国科学院|中国科学院]]、[[中国航天科技集团公司|中国航天科技集团公司]]领导担任；中国载人航天工程总设计师、副总设计师由技术专家担任。<ref name=wzy>[http://district.ce.cn/newarea/roll/201203/27/t20120327_23192345.shtml 王兆耀、武平分别任921工程办正副主任（附机构简介），中国经济网，2012-03-27]</ref>

中国载人航天工程实行专项管理机制，设“[[中国载人航天工程办公室|中国载人航天工程办公室]]”（CMSEO），代表中国政府行使管理职能。中国载人航天工程办公室隶属[[中央军事委员会装备发展部|中央军事委员会装备发展部]]，是统一管理载人航天工程的专门机构和组织指挥部门，也是中国载人航天工程总指挥、总设计师和重大专项领导小组的办事机构。<ref name=ytj/><ref name=jj/>

== 发射计划 ==

中国载人航天计划的第一步是进入太空，而进入太空轨道飞行器被命名为[[神舟号飞船|神舟号飞船]]，最多乘员三人。飞船由[[长征二号F运载火箭|长征二号F火箭]]运载。

=== 中國載人飞船 ===
{{主条目|神舟飛船}}

迄今，[[神舟飛船|神舟号飞船]]共进行过11次发射，6次载人、5次无人（前四次和神八）。

==== 神舟一号 ====
{{主条目|神舟一号}}

[[神舟一号|神舟一号]]飞船是[[中华人民共和国|中华人民共和国]]载人航天计划中发射的第一艘实验飞船，于1999年11月20日 06:30:3.5秒在[[酒泉航天发射场|酒泉航天发射场]]发射升空，承担发射任务的是在[[长征2号|长征2号]]捆绑式火箭的基上改进研制的[[长征2号F|长征2号F]]载人航天火箭。北京时间11月21日 15:41顺利返回，降落在[[内蒙古|内蒙古]]中部地区，飞船在太空中共飞行了21个小时。

==== 神舟二号 ====
{{主条目|神舟二号}}

[[神舟二号|神舟二号飞船]]是中国发射的第二艘实验飞船，它也是中国第一艘正样无人航天飞船，2001年1月10日01時00分03.561秒在[[酒泉航天发射中心|酒泉航天发射中心]]发射升空，顺利进入预定轨道。1月16日在[[内蒙古|内蒙古]]中部的主着陆场成功着陆。飞船在太空中运行了近七天，绕地球飞行了117圈。

==== 神舟三号 ====
{{主条目|神舟三号}}

[[神舟三号|神舟三号飞船]]也是一艘正样无人飞船，2002年3月25日22:15在[[酒泉卫星发射中心|酒泉卫星发射中心]]成功发射升空。2002年4月1日，神舟三号飞船在太空绕地球飞行107圈后，准确降落在[[内蒙古自治区|内蒙古]]中部的着陆场。

==== 神舟四号 ====
{{主条目|神舟四号}}

[[神舟四号|神舟四号飞船]]是[[中国|中国]]第三艘正样无人飞船，于2002年12月30日00時40分03.543秒[[酒泉航天发射场|酒泉航天发射场]]由[[长征二号F火箭|长征二号F火箭]]发射升空，飞船按照预定计划在太空飞行了6天零18小时，飞船在环绕地球飞行了108圈后，于北京时间2003年1月5日19时16分，准确降落在[[内蒙古|内蒙古]]中部地区的着陆场。

==== 神舟五号 ====
[[文件:Shenzhou5-1.JPG|缩略图]]
{{主条目|神舟五号}}

[[神舟五號|神舟五号载人飞船]]是[[中华人民共和国|中国]]首次发射的载人[[航天飞行器|航天飞行器]]，于2003年10月15日09時00分03秒497毫秒将[[航天员|航天员]][[杨利伟|杨利伟]]送入太空。飞行时间21小时22分钟45秒。这次的成功发射标志着中国成为继前[[蘇聯|蘇聯]]（現由[[俄羅斯|俄羅斯]]承繼）和[[美国|美国]]之后，第三个有能力独自将人送上太空的国家。

==== 神舟六号 ====
{{主条目|神舟六号}}
[[神舟六号|神舟六号载人飞船]]于2005年10月12日09:00:03.583在酒泉发射升空，搭載了费俊龙和聂海胜两名中国航天员，2005年10月17日降落在内蒙古主着陆场，任務時間達100小時。

==== 神舟七号 ====
{{主条目|神舟七号}}

[[神舟七号|神舟七号载人飞船]]在2008年9月25日21:10:04.988由[[长征2F火箭|长征2F火箭]]发射升空，载有[[翟志刚|翟志刚]]（指令长）、[[劉伯明_(宇航員)|刘伯明]]和[[景海鹏|景海鹏]]三名[[宇航员|宇航员]]。2008年9月27日，景海鹏留守[[返回舱|返回舱]]，翟志刚出舱作业，劉伯明在轨道舱内协助，實現了中國歷史上第一次的[[太空漫步|太空漫步]]，令中國成為第三個有能力把太空人送上太空並進行太空漫步的國家。飞船于2008年9月28日成功着陆于中国[[内蒙古|内蒙古]][[四子王旗|四子王旗]]。神舟七号飞船共计飞行2天20小时27分钟。

==== 神舟八号 ====
{{主条目|神舟八号}}

[[神舟八号|神舟八号飞船]]采用长征2号F改进型遥八运载火箭发射，于2011年11月1日清晨05:58:10.430在[[酒泉卫星发射中心|酒泉卫星发射中心]]发射升空。神舟八号升空后在11月3日凌晨与天宫一号自动对接，组成一座小型的低地轨道“空间实验室”。间隔12天后脱离天宫一号，与其进行第二次交会对接。最后再次脱离天宫一号，返回舱返回地球。

==== 神舟九号 ====
{{主条目|神舟九号}}

[[神舟九号|神舟九号飞船]]于2012年6月16日18时37分24.558秒由长征2F遥九火箭送入太空，飞船搭载两名男航天员景海鹏、刘旺以及一名女航天员刘洋。<ref>[http://news.qq.com/a/20120616/001196.htm 神舟九号发射成功 中国航天员飞往“天宫”]</ref>将于天宫一号进行两次对接，其中一次为手动对接，由刘旺操作完成。<ref>[http://news.sina.com.cn/o/2012-06-16/055924601576.shtml “神九”逐“天宫”刘旺手控交会对接]</ref>

====神舟十号====
{{主条目|神舟十号}}

[[神舟十号|神舟十号]]於2013年6月11日17時38分02.666秒發射起飛，載三名航天員，[[聶海勝|聶海勝]]、[[張曉光_(航天員)|張曉光]]、[[王亞平|王亞平]]，与天宫一号对接。飞船由推进舱（服务舱）、返回舱、轨道舱组成，在轨飞行約15日，其中驻留天宮一號12天。2013年6月26日8时7分，神舟十号在内蒙古四子王旗着陆场着陆。
====神舟十一号====
