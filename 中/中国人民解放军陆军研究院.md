'''中国人民解放军陆军研究院'''，位于[[北京市|北京市]]，是[[中国人民解放军陆军|中国人民解放军陆军]]直属单位，是中国人民解放军陆军最高科研机构。

==沿革==
2017年以前，中国人民解放军陆军没有军种最高科研机构，各科研机构分散在军队院校等单位中。例如有[[中国人民解放军石家庄陆军指挥学院|中国人民解放军石家庄陆军指挥学院]]陆军研究所<ref>朱长来、陈国辉，美国陆军新版电子战条令的主要内容及特点分析，外军信息战2009年第5期</ref>、[[中国人民解放军军械工程学院|中国人民解放军军械工程学院]]军械技术研究所<ref>{{Cite web |url = http://tech.southcn.com/t/2014-08/27/content_107417171.htm |title =解放军首个通用弹药导弹保障技术实验室成立  |author =  |date =2014-08-27  |publisher =  南方网|language =  |accessdate =  |quote =  }}</ref><ref>杜峰坡、穆希辉、张宝富、来升，经济型防爆电动叉车设计技术研究，机械设计与制造2010(6):24-26</ref>等等。2015年底，[[中国人民解放军装甲兵工程学院|中国人民解放军装甲兵工程学院]]成立党的创新理论学习研究中心<ref>{{Cite web |url = http://tech.hexun.com/2015-12-29/181468771.html |title = 强军路上，铁骑飞歌铸辉煌 |author =  |date =2015-12-29  |publisher = 和讯 |language =  |accessdate =  |quote =  }}</ref>，2015年12月陆军领导机构成立后，该中心改为中国人民解放军陆军党的创新理论研究中心<ref>{{Cite web |url = http://www.sohu.com/a/163750834_157267 |title = 陆军加速转型建设纪事 |author =  |date = 2017-08-11 |publisher =搜狐 |language =  |accessdate =  |quote =  }}</ref>。
