'''中國－阿曼關係'''是指[[中國|中國]]和[[阿曼|阿曼]]之間的外交關係，兩國於1978年5月25日建交<ref>{{cite web|url=http://om.chineseembassy.org/chn/sbgx/t785291.htm|title=双边关系|publisher=中國駐阿曼大使館|date=2013-01-01}}</ref>。兩國建交後關係發展相當順利。
== 歷史 ==
[[佐法尔叛乱|佐法尔叛乱]]（阿曼内战）时期，中国曾支持叛军一方<ref>{{cite journal|author1=黄培昭|title=中国和阿曼关系|journal=中国社会科学院西亚非洲研究所|url=http://waas.cssn.cn/webpic/web/waas/upload/2011/06/d20110609192835359.pdf}}</ref>。1978年5月25日兩國建交。2008年4月[[2008年夏季奧林匹克運動會火炬接力|北京奧運聖火]]傳遞至阿曼首都[[馬斯喀特|馬斯喀特]]，是整個火炬接力中唯一的阿拉伯國家。

== 經貿關係 ==
2011年，雙邊貿易額為158.8亿美元，其中中國向阿曼出口9.9亿美元，阿曼向中國148.8亿美元其中，中國主要向阿曼出口机电产品、钢铁及其制品、高新技术产品、纺织品等；而阿曼則主要向中國出口原油。

== 文化關係 ==
1981年7月，为纪念公元6世纪连接阿曼北部城市苏哈尔和中国广州之间的“海上丝绸之路”，阿曼“苏哈尔号”仿古友谊船抵达广州。1995年12月，阿曼出资在广州建成“苏哈尔号”古船纪念碑。2000年12月，阿曼资助泉州海外交通史博物馆兴建伊斯兰阿拉伯文化陈列室。2009年10月，阿曼捐资修建泉州清净寺礼拜堂。2010年和2011年，中國残疾人艺术团连续两年前往阿曼演出，受到广泛欢迎。2005年，中國批准阿曼为中国公民出国旅游目的地国。
