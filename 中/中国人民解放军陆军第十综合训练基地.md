'''中国人民解放军陆军第十综合训练基地'''，机关驻地[[山西省|山西省]][[长治市|长治市]]，隶属[[中国人民解放军中部战区陆军|中国人民解放军中部战区陆军]]。

==沿革==
1967年7月15日经[[中央军委|中央军委]]批准，陆军第十二军第三十一师坦克自行火炮第二三六团、陆军第十二军第三十四师坦克自行火炮第二三九团、陆军第二十七军第一七九师坦克自行火炮第二八三团于1967年8月1日在[[安徽省|安徽省]][[嘉山县|嘉山县]]三界组成'''中国人民解放军坦克第九师'''，隶属[[南京军区|南京军区]]装甲兵。师长蔚锦茂，政委包可悦，副师长李喜，参谋长俞锡荣。1969年8月28日，坦克自行火炮第二三六团、坦克自行火炮第二三九团、坦克自行火炮第二八三团依次改称坦克第九师坦克第三十三、三十四、三十五团。1969年11月，坦克第九师调归[[北京军区|北京军区]]装甲兵建制，移驻山西省长治市。

1976年1月，坦克第九师撤销。以坦克第九师师部为基础，组建'''中国人民解放军北京军区坦克乘员训练基地'''，隶属北京军区，仍驻山西省长治市<ref>{{Cite web |url =http://mil.news.sina.com.cn/2003-08-23/145128.html  |title =  北京军区某坦克乘员训练基地特色文化队活动见闻|author =  |date =  2003-08-23|publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。所属坦克第三十三、三十四、三十五团依次改称陆军第六十六、二十一、二十八军坦克团。

2010年代初，北京军区坦克乘员训练基地改为'''中国人民解放军北京军区兵种训练基地'''<ref>{{Cite web |url =http://sxcz.wenming.cn/wmbb/201602/t20160205_2326358.html  |title = 长治市领导看望慰问老党员老同志及驻市部队官兵 |author =  |date = 2016-02-05 |publisher =长治文明网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://cz.sxgov.cn/content/2015-07/27/content_6053798.htm |title =  北京军区全国人大代表调研组在长治调研 许霞主持|author =  |date =2015-07-27  |publisher =黄河新闻网  |language =  |accessdate =  |quote =  }}</ref>。

[[湖北省|湖北省]][[孝昌县|孝昌县]]原有'''中国人民解放军济南军区坦克乘员训练基地四大队'''，隶属[[中国人民解放军济南军区|中国人民解放军济南军区]]<ref>{{Cite web |url =http://news.cntv.cn/20110820/100720.shtml  |title = 领导履职尽责不分心 |author =  |date =2011-08-20  |publisher =中国网络电视台  |language =  |accessdate =  |quote =  }}</ref>。后改为'''中国人民解放军济南军区兵种训练基地四大队'''。

2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，包括这两处训练基地在内的多个单位合并，调整组建'''中国人民解放军陆军第十综合训练基地'''，隶属[[中国人民解放军中部战区陆军|中国人民解放军中部战区陆军]]<ref>{{Cite web |url = http://wemedia.ifeng.com/71995845/wemedia.shtml |title = 市党政军领导参加军事日活动 |author =  |date = 2018-08-02 |publisher =凤凰新媒体  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.xiaochang.gov.cn/xwzx/xzbm/content_47421  |title =驻孝昌陆军第十综合训练基地战士到光荣院 开展学雷锋活动  |author =  |date = 2018-03-07 |publisher = 孝昌县人民政府 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://mil.news.sina.com.cn/2018-04-02/doc-ifysvnth3059337.shtml  |title = 【退役军人生存状态之三】只想见识一个精彩的世界 |author =  |date = 2018-04-02 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。
