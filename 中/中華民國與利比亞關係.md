'''中華民國與利比亞關係'''是指[[中華民國|中華民國]]與[[利比亞國|利比亞國]]之間的關係。兩國於1959－1978年有官方[[外交#外交上的承認|外交關係]]，斷交後中華民國曾在利比亞首都[[的黎波里|的黎波里]]設立[[實質大使館|具大使館性質]]的代表機構，但受到[[2011年利比亞內戰|2011年利比亞內戰]]期間影響暫時關閉，現已裁撤。<ref name="外館">[http://news.ltn.com.tw/news/politics/breakingnews/2144777 外交部宣布 暫關閉挪威、關島、吉達外館]</ref>目前對利比亞的相關事務由'''駐[[約旦|約旦]][[臺北經濟文化辦事處|臺北經濟文化辦事處]]'''、'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref name="駐館位置及聯絡方式">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-246-e5470-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[格達費治下的利比亞|阿拉伯利比亞共和國]]。]]
{{see also|中華民國外交|{{link-en|利比亞外交|Foreign relations of Libya}}}}
1959年5月10日，中华民国与利比亞[[外交#外交上的承認|建交]]，於首都[[的黎波里|的黎波里]]設立'''中華民國駐[[利比亞王國|利比亞聯合王國]][[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]，<ref name="99年">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2010/2.1.3.pdf |title = 《中華民國99年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>且[[中華民國國軍|中華民國國軍]]甚至派出通信人員在[[利比亚武装力量|利比亞軍中]]協助當時在利比亞通信兵營當指揮官的[[上尉|上尉]][[格達費|格達費]]。

1962年2月26日，簽署《中華民國與利比亞聯合王國之技術合作協定》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e5%88%a9%e6%af%94%e4%ba%9e&tycountry_e=%e5%88%a9%e6%af%94%e4%ba%9e&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

1969年8月31日，格達費率通信兵營革命[[利比亚绿色革命|取得政權]]，更改國號為[[卡扎菲治下的利比亚#阿拉伯利比亚共和国（1969年-1977年）|阿拉伯利比亞共和國]]。

1978年8月9日，[[大阿拉伯利比亚人民社会主义民众国|大阿拉伯利比亚人民社会主义民众国]]与[[中华人民共和国|中华人民共和国]]建交（1971年6月11日外交承認）。9月14日，中华民国與利比亞断交。10月22日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

1980年3月19日，於首都的黎波里設立[[實質大使館|具大使館性質]]的'''中華民國駐利比亞[[臺北經濟文化代表處|商務辦事處]]'''，駐外人員享有部分[[外交豁免權|外交人員待遇]]，並可以官方身分與{{link-en|利比亞政府|Politics of Libya}}洽辦業務。但自{{link-en|聯合國安理會第731號決議|United Nations Security Council Resolution 731|1992年利比亞遭受聯合國制裁}}後，{{link-en|利比亞外交部長|Minister of Foreign Affairs (Libya)|利比亞外交部}}在中華人民共和國壓力下，通知辦事處自1995年9月停止辦理領事業務，更於1997年5月規範辦事處不得從事政治活動。[[中華民國外交部|中華民國外交部]]以該處功能有限，報經[[行政院|行政院]]核准後，於9月12日關閉辦事處。<ref name="99年"/><ref name="86年">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-3.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1996年8月，利比亞外交部國際技術合作司長保文（Adad S. Burwin）抵台訪問。<ref name="86年"/>

2006年1月18日，格達費次子[[賽義夫|賽義夫]]抵達臺灣並與[[中華民國總統|中華民國總統]][[陳水扁|陳水扁]]見面，希望台灣在利比亞設立代表機構。<ref name="亞西司">{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=70BCE89F4594745D&sms=700DE7A3F880BAE6&s=C64B82F500F152DC |author = 國際合作及經濟事務司 |date = 2008年2月21日 |title = 本部單位主管例行新聞說明會紀要--亞西司 |publisher = 中華民國外交部}}</ref>2008年2月13日，恢復設立代表處，<ref name="99年"/>但目前已關閉。<ref name="外館"/>

2006年5月10日，總統陳水扁訪問利比亞時與格達費會面，<ref name="亞西司"/>當時利比亞給予[[中華民國國民|中華民國公民]][[中华民国公民签证要求|免簽證待遇]]。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境利比亞。]]
[[File:Visa_requirements_for_Libyan_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|利比亞簽證政策|Visa policy of Libya}}|{{link-en|利比亞公民簽證要求|Visa requirements for Libyan citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]無論申請觀光或商務簽證皆須備有利比亞當地出具之邀請函，可前往利比亞駐[[日本|日本]][[外交代表機構#大使馆|大使館]]申請簽證。護照內頁不得有[[以色列签证政策|以色列簽證]]或入出境章。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-246-e5470-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|利比亞護照|Libyan passport}}的{{link-en|利比亞人口|Demographics of Libya|利比亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將利比亞歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2017年4月25日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-109-58c44-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==

===貿易===
2017年，利比亞是中華民國第146大貿易夥伴、第155大[[進口|進口]]夥伴、第125大[[出口贸易|出口]]夥伴。出口至利比亞的金額為868萬3,407[[美元|美元]]，年減31.618%；自利比亞進口的金額為111萬3,417美元，年減90.093%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）756萬9,990美元，年增418.635%。※2016年，利比亞是中華民國第121大貿易夥伴、第108大進口夥伴、第115大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

==援助==
斷交前，兩國的技術合作範圍甚廣，包括[[醫療|醫療]]、[[通訊|通訊]]、[[土木|土木]]、[[水利|水利]]、[[氣象|氣象]]、車輛修理等。斷交後，駐利比亞技術人員均改以私人應聘方式，後因待遇偏低，工作人員已陸續撤離。<ref name="86年"/>

兩國曾在醫療衛生、[[人力資源|人力培訓]]擁有密切合作關係。[[國立臺灣大學醫學院附設醫院|國立臺灣大學醫學院附設醫院]]提供利比亞[[癌症|癌症]]醫師赴台受訓名額；財團法人國際合作發展基金會（國合會）舉辦各項研習班，以協助提升利比亞的人力素質。<ref name="99年"/><ref name="97年">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2008/html/08-1-3.htm |title = 《中華民國97年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2011年，國合會與國際慈善機構「{{link-en|美慈組織|Mercy Corps}}」（Mercy  Corps）合作進行「利比亞[[糧食安全|糧食安全]]計畫」。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/9652c596-1cea-4268-ae5c-7832ece3c1a2.pdf |author = 國際合作及經濟事務司 |date = 2012年4月30日 |title = 國際合作發展事務100年度報告（參、100年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref>
