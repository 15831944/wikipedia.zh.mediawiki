'''中國大鯢'''（[[學名|學名]]：''{{lang|la|Andrias davidianus}}''）<ref>{{ITIS|ID=202085|taxon=''Andrias davidianus'' |downloaded=2007年3月13日}}</ref>是[[大鯢|大鯢屬]]的一種，俗名'''中國娃娃魚'''、'''娃娃魚'''，乃生活在[[淡水|淡水]]中的[[兩棲動物|兩棲動物]]。目前世上最早的[[化石|化石]]出土於[[中國|中國]][[内蒙古|内蒙古]]，距今约1.65億年，所以中國大鯢可被稱為[[活化石|活化石]]。1988年《中華人民共和國野生動物保護法》將之列為[[国家二级重点保护野生动物名录|国家二级重点保护野生动物]]。

== 描述 ==
中國大鯢是現存[[兩棲類|兩棲類]]當中，體型最大的一種，一般身長約60至70公分，體重約5至6公斤，大的可長達1.8公尺，重達30公斤。背呈棕褐色，有黑色斑塊，腹色較淡，頭扁寬，生有成對的疣突，頭的背面有極小的鼻孔和眼，口大，軀扁而粗壯，尾短而側扁，四[[肢|肢]]很短，前肢4[[手指|指]]，後肢5[[趾|趾]]，趾間有[[蹼|蹼]]。

因叫聲像嬰兒啼哭，故又名「娃娃魚」，但實屬兩棲動物，並不屬於[[魚類|魚類]]。

另外中科院昆明動物所研究員車靜說，研究團隊與合作者們經過10年的考察和走訪，界定了陝西種、四川種、廣西種、貴州種和安徽種5個物種。另外，在養殖場種群中還意外發現了另外兩個獨特支系，加之記錄分佈於海拔4000米以上的青藏高原種群，推測中國大鯢可能包括8個物種。

== 生活情況 ==
娃娃魚棲息在清澈、低溫的溪流或者天然溶洞中，牠們是肉食性動物，主要的食物為[[魚類|魚類]]和[[甲殼類|甲殼類]][[動物|動物]]，於夏天末期產卵，約2至4個星期即可孵化。

現時中國[[湖南|湖南]]、[[山西|山西]]、[[陝西|陝西]]、[[廣東|廣東]]、[[廣西|廣西]]等地，都有人工馴養的娃娃魚分佈；[[野生動物|野生]]則極為稀少，主要於[[長江|長江]]、[[黃河|黃河]]及[[珠江|珠江]][[流域|流域]][[海拔|海拔]]1500公尺以上[[支流|支流]]被發現。

== 保育現狀 ==
由於娃娃魚生存年代久遠，其肉被認為具有神奇的療效，因此價格昂貴，常遭人類捕捉來食用或觀賞，再加上生活環境遭受破壞，幾瀕臨滅絕。20世紀末期，[[台灣|台灣]]也曾盛行食用，大量從中國内地走私，作為觀賞用途的也不計其數。

在[[IUCN紅色名錄|IUCN紅色名錄]]內，娃娃魚被評定為[[極危物種|極危物種]]，表示其野生種群所面臨[[滅絕|滅絕]]的機會極高。目前野生種群主要面對濫捕的情況，棲息地的破壞也減少了它們的生活空間。

== 人工養殖 ==
自1978年，[[陽愛生|陽愛生]]成功地實現了娃娃魚的人工繁殖，時至今日，中國人工繁殖娃娃魚的數量已至約10萬尾，並成立有[[湖南省|湖南省]]大鯢保護與發展協會。有報導指人工馴養的娃娃魚，將能在兩年後成為普羅大眾的食品<ref>[http://www.022net.com/2008/12-15/485731253315428.html “娃娃鱼”兩年後將擺上家庭餐桌] {{webarchive|url=https://web.archive.org/web/20090502132741/http://www.022net.com/2008/12-15/485731253315428.html |date=2009-05-02 }}，載人民網，2008年12月15日</ref>。野生的中国大鲵至少包括五个到八个不同物种，而养殖场相关业者不知道各地区大鲵物种的明显差异，无意间创造出了杂交品种，其對野外基因組成的適應，有令到多数大鲵物种灭绝的风险。<ref>[https://cn.nytimes.com/science/20180605/giant-salamanders-china/ 中国大鲵保育困境：放生反致野生品种走向灭绝] 纽约时报中文 2018-06-05</ref><ref>{{Cite web|title=科学松鼠会 » 认识它们，却只来得及说再见|url=https://songshuhui.net/archives/101234|accessdate=2019-03-16}}</ref>

2009年，娃娃魚主要聚居地之一的[[張家界|張家界]]大鯢自然保護區，正擬提供娃娃魚給[[香港|香港]][[海洋公園|海洋公園]]作科普宣傳及科學研究之用。此前香港有兩條娃娃魚，屬1990年代末非法偷運入境，经香港[[漁護署|漁護署]]搜出後交由[[嘉道理農場|嘉道理農場]]照料，2001年養於海洋公園至今<ref>[http://news.sina.com.hk/cgi-bin/nw/show.cgi/2/1/1/1005816/1.html 國寶娃娃魚贈海洋公園] {{Wayback|url=http://news.sina.com.hk/cgi-bin/nw/show.cgi/2/1/1/1005816/1.html |date=20090503013920 }}，載[[明報|明報]]，2009年1月15日</ref>。
