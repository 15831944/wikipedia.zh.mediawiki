'''中国民用航空局'''，簡稱'''中国民航局'''、'''民航局'''，是[[中华人民共和国国务院|中华人民共和国国务院]]主管[[民用航空|民用航空]]事業的[[国务院部委管理的国家局|国家局]]，由[[中华人民共和国交通运输部|交通运输部]]管理。

中国民航局在1987年以前曾承担中国民航的运营职能；2008年3月，原中国民用航空总局由[[国务院直属机构|国务院直属机构]]降格为[[国务院部委管理的国家局|国务院部委管理的国家局]]，同时更名为中国民用航空局<ref>{{Cite news |url = http://npc.people.com.cn/GB/28320/116286/116574/6985000.html |title = 华建敏：组建交通运输部、民用航空局 保留铁道部 |date  =2008-03-11 |publisher = 人民网}}</ref>。

== 沿革 ==
[[File:1952-08_中国人民航空公司成立.png|缩略图]]
[[File:CAAC_Boeing_707.jpg|缩略图]]为中国内地引进的首款波音客机。]]
{{See also|两航事件}}
1949年11月2日，[[中共中央|中共中央]]决定在[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]下设民用航空局（受[[中国人民解放军空军|空军]]司令部领导）；12月，军委民航局成立，在[[天津|天津]]、[[上海|上海]]、[[武汉|武汉]]、[[重庆|重庆]]、[[广州|广州]]设办事处<ref>[http://www.caac.gov.cn/D1/60years/jkcy/cl 成立军委民航局 - 中国民航局60周年档案展] {{webarchive|url=https://web.archive.org/web/20120426130325/http://www.caac.gov.cn/d1/60years/jkcy/cl/ |date=2012-04-26 }}</ref>。1950年3月10日，广州办事处印章正式启用，主要负责广东、广西、湖南三省（区）的民航管理及业务工作。1951年1月29日，民航武汉办事处与民航广州办事处合并为华中南民航办事处，不久改为中南民航办事处。设在广州，管辖范围为广东、广西、湖南、湖北四省（区）的民航业务。1951年10月27日，经政务院批准，调整军委民航局的组织机构，将各地区民航办事处改为地区管理分局。

中央军委、政务院于1952年5月7日作出了《关于整编民用航空的决定》，军委民航局改为军队建制，直辖于空军司令部，同时将民用航空的行政管理和企业经营机构分开，改设民用航空局和民用航空公司。根据这个决定，民航局于1951年7月至11月进行整编。整编后，军委民航局下设华东（上海）、中南（广州）、西南（重庆）、华北（天津）4个民航管理处，实施政府管理职能。华南民航管理分局改为中南民航管理处，管辖范围不变。1951年7月17日成立中国人民航空公司，经营航空运输和专业航空业务，在广州设立了广州民航站和中国人民航空公司广州营业处。由于民航初建，业务量不大，设两套机构，配两套干部，冗员过多，而且存在领导关系及工作上的矛盾，不能很好地保证飞行安全。在不到一年时间内，发生几起重大事故，在1953年6月，学习苏联民航“政企合一”的模式，中国人民航空公司并入军委民航局。 

1954年11月，“中央人民政府革命军事委员会民用航空局”更名为“中国民用航空局”（以下简称“民航局”），改为国务院直属局，受国务院与空军双重领导，空军主管技术、飞行、机务、通讯、人事管理和政治工作。　　

1958年2月27日，民航局改为[[中华人民共和国交通部|交通部]]部属局，并于6月17日批准交通部党组“关于体制下放意见的报告”，民航实行中央、地方双重领导，即：国际航线、国内干线和工业航空，集中性大，实行以中央为主的双重领导；地方航线和农业航空下放地方，实行以地方为主的双重领导。据此，全国大部分地区相继在原有航空站的基础上组建了省、自治区民航管理局（处）。

1960年11月17日，中国民用航空局改称“交通部民用航空总局”。1962年4月，[[第二届全国人民代表大会常务委员会|第二届全国人民代表大会常务委员会]]第53次会议决定，民航局改称“中国民用航空总局”，为[[中华人民共和国国务院|国务院]]直属局，由[[中国人民解放军空军|空军]]代管，各省的民航局（处）均由民航局垂直领导。 

1969年11月20日，民航划转为中国空军的一部分，执行军队制度；各省民航局以民航系统领导为主，同时接受大军区空军的领导；民航系统实行义务工役制，着工作服和制服，不着军装。

1980年3月，民航脱离军队建制（除航行管制权仍由[[中国人民解放军总参谋部作战部|总参作战部]]航行局和[[中国人民解放军空军|空军司令部]]航行局统一领导），中国民航局恢复为[[国务院直属机构|国务院直属机构]]，实行企业化管理；既是政府行政部门，又以“中国民航（CAAC）”的名义直接经营航空运输<ref>{{cite web|url=http://travel.sina.com.cn/air/2009-10-13/1204111070.shtml|title=60年成就光荣梦想：回顾新中国民航业发展史(组图)|author=|date=|website=sina.com.cn|accessdate=2018-11-29}}</ref>。

1987年，民航业进行了以“航空公司与机场分设”为特征的体制改革，将原“中国民航”拆分出6个独立的航空公司，实行自主经营、自负盈亏、平等竞争；中国民航局不再直接承担客运职责。

1993年4月19日，民航局更名为“中国民用航空总局”，属[[国务院直属机构|国务院直属机构]]；当年12月20日起，其行政级别由副部级调整为正部级。

2002年3月，原隶属于民航局管理的航空公司以及服务保障企业正式与其脱钩，成立了六大集团公司；除[[北京首都国际机场|首都机场]]、[[西藏自治区|西藏自治区]]内机场外，其他民用机场均改由地方政府，施行属地化管理。<ref>[http://www.caac.gov.cn/H1/H4/ 中国民用航空局行政体制沿革] {{webarchive|url=https://web.archive.org/web/20130124110502/http://www.caac.gov.cn/H1/H4/ |date=2013-01-24 }}</ref>

2008年3月15日，[[第十一届全国人民代表大会第一次会议|第十一届全国人民代表大会第一次会议]]通过《第十一届全国人民代表大会第一次会议关于国务院机构改革方案的决定》，批准《国务院机构改革方案》。方案规定：“组建[[中华人民共和国交通运输部|交通运输部]]。将[[中华人民共和国交通部|交通部]]、中国民用航空总局的职责，[[中华人民共和国建设部|建设部]]的指导城市客运职责，整合划入交通运输部。组建国家民用航空局，由交通运输部管理。[[国家邮政局|国家邮政局]]改由交通运输部管理。保留[[中华人民共和国铁道部|铁道部]]，继续推进改革。不再保留交通部、中国民用航空总局。”<ref>{{Cite web |url =http://www.npc.gov.cn/npc/xinwen/syxw/2008-03/12/content_1413911.htm  |title = 关于国务院机构改革方案的说明——2008年3月11日在第十一届全国人民代表大会第一次会议上 |author =华建敏  |date =2008-03-11  |publisher =中国人大网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.gov.cn/2008lh/content_921411.htm |title = 两会授权发布：国务院机构改革方案 |author =  |date = 2008-03-15 |publisher = 中国政府网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.gov.cn/2008lh/content_921283.htm  |title = v |author =  |date =  2008-03-15 |publisher = 中国政府网 |language =  |accessdate =  |quote =  }}</ref>

== 职责 ==
根据《{{s|中国民用航空局主要职责内设机构和人员编制规定}}》，中国民用航空局承担下列职能：
{{columns-list|2|
# 提出民航行业发展战略和中长期规划、与综合运输体系相关的专项规划建议，按规定拟订民航有关规划和年度计划并组织实施和监督检查。起草相关法律法规草案、规章草案、政策和标准，推进民航行业体制改革工作。
# 承担民航飞行安全和地面安全监管责任。负责民用航空器运营人、航空人员训练机构、民用航空产品及维修单位的审定和监督检查，负责危险品航空运输监管、民用航空器国籍登记和运行评审工作，负责机场飞行程序和运行最低标准监督管理工作，承担民航航空人员资格和民用航空卫生监督管理工作。
# 负责民航空中交通管理工作。编制民航空域规划，负责民航航路的建设和管理，负责民航通信导航监视、航行情报、航空气象的监督管理。
# 承担民航空防安全监管责任。负责民航安全保卫的监督管理，承担处置劫机、炸机及其他非法干扰民航事件相关工作，负责民航安全检查、机场公安及消防救援的监督管理。
# 拟订民用航空器事故及事故征候标准，按规定调查处理民用航空器事故。组织协调民航突发事件应急处置，组织协调重大航空运输和通用航空任务，承担国防动员有关工作。
# 负责民航机场建设和安全运行的监督管理。负责民用机场的场址、总体规划、工程设计审批和使用许可管理工作，承担民用机场的环境保护、土地使用、净空保护有关管理工作，负责民航专业工程质量的监督管理。
# 承担航空运输和通用航空市场监管责任。监督检查民航运输服务标准及质量，维护航空消费者权益，负责航空运输和通用航空活动有关许可管理工作。
# 拟订民航行业价格、收费政策并监督实施，提出民航行业财税等政策建议。按规定权限负责民航建设项目的投资和管理，审核(审批)购租民用航空器的申请。监测民航行业经济效益和运行情况，负责民航行业统计工作。
# 组织民航重大科技项目开发与应用，推进信息化建设。指导民航行业人力资源开发，科技、教育培训和节能减排工作。
# 负责民航国际合作与外事工作，维护国家航空权益，开展与港澳台的交流与合作。
# 管理民航地区行政机构、直属公安机构和空中警察队伍。
# 承办国务院及交通运输部交办的其他事项。
}}

'''中国民航'''，或称'''中国人民航空公司'''，是于1952至1987年由[[中华人民共和国国务院|中华人民共和国国务院]]主管的[[民用航空|民用航空]]事業运营机构。

=== 运营历史 ===
[[File:CAAC_Boeing_747SP-J6;_N1304E@LAX,_October_1982_BGO_(5659169999).jpg|缩略图]]客机，喷涂中国民航涂装。]]
[[File:CAAC_Boeing_737-2T4;_B-2512,_HKG,_December_1990_CEN_(5288956368).jpg|缩略图]]的中國民航波音737-200。]]
[[File:Ilyushin_Il-62_2026_CAAC_SVO_19.09.74_edited-3.jpg|缩略图]]在[[谢列梅捷沃国际机场|莫斯科谢列梅捷沃机场]]。]]
[[File:Antonov_An-24B,_CAAC_AN1018220.jpg|缩略图]]的An-24。]]
[[File:Boeing_767-2J6ER,_B-2554,_CAAC.jpg|缩略图]]的中国民航波音767-200ER。]]
[[File:1963-11_1963_飞机乘务员.jpg|缩略图]]》1963年11期刊载的中国民航乘务员图片。]]
中国民用航空总局曾經以“中国民航”的名義經營民用航空業務，20世纪50年代中期开始运营国际航线，但当时的目的地仅为[[苏联|苏联]]、[[蒙古人民共和国|蒙古]]、[[朝鲜|朝鲜]]、[[缅甸|缅甸]]等[[共产主义|共产主义]]阵营国家，机队也以苏联制客机为主。<ref>[http://timetableimages.com/ttimages/complete/caac57/caac57-5.jpg 1957 route map].</ref><ref>{{cite web|url=http://www.timetableimages.com/ttimages/ca3/ca64/|title=Index of /ttimages/ca3/ca64/|author=|date=|website=www.timetableimages.com|accessdate=2018-11-29}}</ref> 

1959年，中国民航接收首架[[伊尔-18|伊尔-18]]，1963年引进[[维克斯子爵|维克斯子爵型]]。1973年，中国民航接收首架[[波音707|波音707]]，次年使用波音707开通北京经[[卡拉奇|卡拉奇]]飞往[[巴黎|巴黎]]的中法航线。1980年，中国民航引进[[波音747SP|波音747SP]]<ref>{{cite news|author=俞泉福|title=波音747SP型宽体客机 将于四月一日载客航行|publisher=[[人民日报|人民日报]]|date=1980-03-14|pages=3}}</ref><ref>{{cite news|title=我民航波音747SP型客机首航巴黎|publisher=[[人民日报|人民日报]]|date=1980-04-03|pages=6}}</ref>，翌年1月7日用其开通首条中美定期航线，由北京经上海、[[旧金山|旧金山]]飞往[[纽约|纽约]]<ref>{{cite news|title=中国民航首航纽约|publisher=[[人民日报|人民日报]]|date=1981-01-08|pages=1}}</ref><ref>{{cite news|title=中国首航班机飞抵旧金山受到该市市长欢迎|publisher=[[参考消息|参考消息]]|date=1981-01-16|pages=2}}</ref><ref>{{Cite web |url= http://www.caac.gov.cn/D1/60ZNQD/JZMH/200910/t20091015_28317.html |title= 亲历中美正式通航_中国民用航空局 |author= 尹淦庭 |date= 2009-10-15 |publisher= 中国民航报社 |language= zh |deadurl= yes |archiveurl= https://web.archive.org/web/20120220233938/http://www.caac.gov.cn/D1/60ZNQD/JZMH/200910/t20091015_28317.html |archivedate= 2012-02-20 }}</ref>。

1984年，中国民航引进国产[[运-7|运-7]]，同年9月5日开通首条由北京经广州飞往[[悉尼|悉尼]]的航线<ref>{{cite news|title=中国民航首航大洋洲直飞悉尼|publisher=[[人民日报|人民日报]]|date=1984-09-06|pages=1}}</ref>。1985年，中国民航首次引进[[空中客车A310|空中客车A310]]<ref>{{cite news|title=中国民航出现“空中客车”|publisher=[[人民日报|人民日报]]|date=1985-07-02|pages=2}}</ref>。

1980年代中期，中国民航已开通飞往[[美国|美国]]、[[欧洲|欧洲]]、[[中东|中东]]、[[澳大利亚|澳大利亚]]的航班，且多用波音客机执行，而在前往[[东欧|东欧]]的航班上仍使用苏制客机，如[[伊尔-62|伊尔-62]]。<ref>{{cite web|url=http://airlineroute.net/2011/02/03/ca-w85/|title=1985/86: CAAC Network|first=UBM (UK) Ltd.|last=2018|date=|website=airlineroute.net|accessdate=2018-11-29}}</ref>
中国民航只在国际航线上使用IATA代码CA（如CA981），国内航线则无此代码（如1501）。<ref>{{cite web|url=http://pic.feeyo.com/pic/20070207/2007020709233650.jpg|title=1988年中国民航时刻表|author=|date=|website=feeyo.com|accessdate=2018-11-29}}</ref>

1987年，国务院决定对民航业进行以航空公司与机场分设为特征的体制改革，原“中国民航”拆分成6个独立的航空公司：
* [[中国国际航空公司|中国国际航空公司]]（总部驻北京）
* [[中国东方航空公司|中国东方航空公司]]（总部驻上海）
* [[中国南方航空公司|中国南方航空公司]]（总部驻广州）
* [[中国西北航空公司|中国西北航空公司]]（总部驻西安）
* [[中国北方航空公司|中国北方航空公司]]（总部驻沈阳）
* [[中国西南航空公司|中国西南航空公司]]（总部驻成都）

其中，[[中国西北航空公司|中国西北航空公司]]后被并入[[中国东方航空公司|中国东方航空公司]]；[[中国北方航空公司|中国北方航空公司]]后被并入[[中国南方航空公司|中国南方航空公司]]；[[中国西南航空公司|中国西南航空公司]]后被并入[[中国国际航空公司|中国国际航空公司]]。而各民用航空场站则下放地方管理。

現在，原“中国民航”的航空代碼（IATA：CA；ICAO：CCA）都歸併到了中國國際航空旗下。<ref>{{cite web|url=http://news.carnoc.com/list/169/169719.html|title=民航那些事（二）·激荡：CAAC改革重组之路_民航新闻_民航资源网|author=|date=|website=news.carnoc.com|accessdate=2018-11-29}}</ref>
