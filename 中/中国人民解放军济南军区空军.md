'''中国人民解放军济南军区空军'''，机关驻地[[山东省|山东省]][[济南市|济南市]]，是[[中国人民解放军空军|中国人民解放军空军]]驻[[中国人民解放军济南军区|中国人民解放军济南军区]]的战役军团，归中国人民解放军空军建制，受中国人民解放军空军、中国人民解放军济南军区双重领导。领导和指挥驻[[山东省|山东省]]、[[河南省|河南省]]空军部队。主要担负本区防空作战，空中进攻作战，协同陆、海军作战等任务<ref name="a1">{{cite news |title = 济南军区空军(Air Force of the Jinan Military Area Command)|url = http://military.people.com.cn/GB/8221/71065/137783/139906/139912/8431053.html |date=2008-11-28 |newspaper = 人民网}}</ref><ref name=zgkjw>{{Cite web |url = http://kj.81.cn/content/2012-12/11/content_5137124.htm |title =济南军区空军 |author =  |date = 2012-12-11 |publisher = 中国空军网 |language =  |accessdate =  |quote =  }}</ref>。

==沿革==
1967年6月1日，中央军委批准，将[[中国人民解放军空军第六军|中国人民解放军空军第六军]]军部扩编为'''中国人民解放军济南军区空军'''。1967年9月15日，正式行使军区空军职权。机关设有司令部、政治部、后勤部、工程部。1970年2月，撤销工程部。1976年2月增设航空工程部，1993年9月改称装备技术部，1998年12月整编为装备部<ref name="a1" /><ref name=zgkjw/>。

济南军区空军成立后，空军第六军下辖的驻[[山东省|山东省]]和徐海地区（即[[徐州|徐州]]和[[连云港|连云港]]地区）部队划归济南军区空军建制。1975年2月，济南军区空军辖区1所航空学校，从空军直属改归济南军区空军建制。1978年10月，所属航空学校和驻徐海地区部队划归[[中国人民解放军南京军区空军|中国人民解放军南京军区空军]]建制。1985年下半年，[[中国人民解放军武汉军区空军|中国人民解放军武汉军区空军]]撤销后，驻河南省空军部队、院校划归济南军区空军建制；同时调整、撤销、合并了部分单位。到20世纪末，济南军区空军已成为包括航空兵、地空导弹兵、雷达兵、通信兵及其他专业兵的合成军队<ref name="a1" /><ref name=zgkjw/>。

济南军区空军长期担负辖区内防空作战任务。1971年12月30日，空军航空兵第五师团长[[杨国祥|杨国祥]]驾机执行空投[[原子弹|原子弹]]试验任务时，投弹系统发生故障，在地面指挥员指挥下，杨国祥驾机载弹安全着陆。排除故障后，杨国祥于1972年1月7日空投原子弹试验成功。济南军区空军部队遂行[[黄河|黄河]]炸凌防洪、[[唐山大地震|唐山抗震救灾]]、抢救遭黄河水围困的[[胜利油田|胜利油田]]、[[青岛|青岛]][[黄岛油库8.12爆炸事故|黄岛油库灭火]]、飞播造林、人工增雨，以及援建[[徐州观音机场|徐州观音机场]]、胜利油田机场等<ref name="a1" /><ref name=zgkjw/>。

2016年1月至2月，在[[深化国防和军队改革|深化国防和军队改革]]中，撤销各大军区，成立战区。原济南军区空军部队转隶3个战区，被调整到10个单位<ref name=zhuanli>{{Cite web |url = http://news.youth.cn/js/201603/t20160308_7717604.htm |title =  原济南军区空军部队转隶3个战区 被调整到10个单位|author =  |date =  2016-03-08|publisher =中国青年网  |language =  |accessdate =  |quote =  }}</ref>。
