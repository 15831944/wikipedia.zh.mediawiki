'''中華民國與歐洲聯盟關係'''是指[[中華民國|中華民國]]與[[歐洲聯盟|歐洲聯盟]]（通稱'''歐盟'''）之間的關係。歐盟目前不承认中華民國，但有互設代表機構。

== 歷史 ==
===外交===
中華民國與歐盟早期關係之維護，由中華民國駐比利時大使館兼辦與歐盟前身「[[歐洲共同體|歐洲共同體]]」業務。

1971年10月25日，中華民國與[[比利時王國|比利時王國]]於中止[[中華民國與比利時關係|外交關係]]後，為維持雙邊實質關係，中華民國政府於1971年11月5日於比京[[布魯塞爾|布魯塞爾]]設置「孫逸仙文化中心」作為代表機構。1990年1月，更名為「台北經濟文化辦事處」。1995年11月，復更名為「駐比利時台北代表處」。在歐盟成形逐漸擴大之際，且晉身為世界政經要角後，為增進中華民國與歐盟之關係，於2001年4月再更名為「[[駐歐盟兼駐比利時代表處|駐歐盟兼駐比利時代表處]]」，此後通過歐盟可加強在沒有設立代表處的歐盟國家非官方外交关系。

1991年6月12日，[[歐洲議會|歐洲議會]]成立「台灣之友協會」。1997年4月，更名為「歐洲議會友台小組」。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2007/html/08-5.htm |title = 《中華民國96年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

1991年7月22日，歐洲議會副議長{{link-de|西格貝特·亞伯|Siegbert Alber|亞伯}}率領多位議員抵台訪問。期間會見立法院長[[梁肅戎|梁肅戎]]、外交部長[[錢復|錢復]]、政務次長房金炎、經濟部國濟貿易局長許柯生、交通部觀光局長[[毛治國|毛治國]]、行政院大陸委員會副主任委員[[馬英九|馬英九]]、行政院新聞局副局長黃老生、中歐貿易促進會理事長曹嶽維等人士。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=4772274E30BFB424 |author = 公眾外交協調會 |date = 1991年7月21日 |title = 歐洲議會議員訪華團一行十人由該議會副議長亞伯率領 |publisher = 中華民國外交部}}</ref>

1996年2月15日，歐洲議會通過緊急決議案，譴責中華人民共和國[[臺灣海峽飛彈危機|在福建沿岸部署重兵]]，威脅台灣即將於3月23日舉行的[[1996年中華民國總統選舉|首次中華民國總統直接選舉]]。該決議案要求[[歐洲聯盟理事會|歐洲聯盟理事會]]運用對其影響力，阻止其對台灣的侵略行為以及對台灣選舉的恫嚇干預。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=4C5565809492F19B |author = 公眾外交協調會 |date = 1996年2月16日 |title = 歐洲議會於本（二）月十五日下午六時 |publisher = 中華民國外交部}}</ref>3月8日，[[歐洲聯盟理事會主席國|歐洲聯盟理事會主席]]公開發表聲明譴責中華人民共和國。3月14日，歐洲議會通過第二次緊急決議案。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=98D17CA02B5AF0C6 |author = 公眾外交協調會 |date = 1996年3月15日 |title = 中共自本月八日起在台灣南北兩端外海舉行飛彈射擊以及在台灣海峽福建沿海進行軍事演習以來 |publisher = 中華民國外交部}}</ref>

2008年5月30日，[[中華民國立法院|中華民國立法院]]成立「中華民國與歐洲議會議員友好協會」，由[[國民黨|國民黨]]立委擔任會長；<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2009/html/08-5.htm |title = 《中華民國98年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>2012年4月20日，另成立「臺灣與歐洲議會議員友好協會」，由[[民進黨|民進黨]]立委擔任會長。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2012/2.5.pdf |title = 《中華民國101年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2018年2月1日，台北市長[[柯文哲|柯文哲]]率團訪問比利時，於布魯塞爾[[歐洲議會議員|歐洲議會議員]]沙龍發表演說，共有30位歐洲議會議員應邀出席，設宴者為歐洲議會友台小組副主席{{link-en|安德烈·科瓦契夫|Andrey Kovatchev}}。柯文哲也是首位於歐洲議會發表演說的[[中華民國自由地區鄉鎮市區列表|中華民國台灣縣市]]首長。<ref>{{Cite web |url = https://www.ettoday.net/news/20180201/1104996.htm |author = 陳家祥 |date = 2018年2月1日 |title = 登歐洲議會提台北價值 柯文哲：不一定要跟隨美國、大陸 |publisher = [[ETtoday新聞雲|ETtoday新聞雲]]}}</ref>

2018年5月8日，[[歐盟對外事務部|歐洲聯盟對外事務部]]表示，在符合歐盟的「[[一個中國|一中政策]]」以及歐盟的政策目標下，歐盟支持以務實方式處理台灣的國際參與問題，包括[[WHA|WHA]]、[[WHO|WHO]]及其所有技術性會議，並主張歐盟的「一中政策」與[[中國|中國]]的「[[一個中國_(中華人民共和國)|一中原則]]」不同，歐盟不斷強化與台灣在經貿關係、科學、教育與文化等各種領域的良好合作關係。<ref>{{Cite web |url = https://www.cna.com.tw/news/firstnews/201805095003.aspx |author = 唐佩君 |date = 2018年5月9日 |title = 歐盟代表28成員國 明確表達挺台參與WHA |publisher = [[中央社|中央社]]}}</ref>

2018年12月12日，歐洲議會通過「共同外交暨安全政策（CFSP）」年度執行報告決議案，強化[[印太地區|印太地區]]安全對歐盟及其會員國具重要利益；其中包括鼓勵中國與台灣迅速恢復雙邊對話，也重申堅定支持台灣有意義參與國際組織、機制及活動。與前一年的決議案相比，增加鼓勵兩岸迅速恢復雙邊對話，也是繼同年9月歐洲議會「[[中国－欧盟关系|歐中關係]]報告」之後再度鼓勵兩岸恢復官方對話。<ref>{{Cite web |url = https://www.cna.com.tw/news/firstnews/201812120397.aspx |author = 唐佩君 |date = 2018年12月13日 |title = 歐洲議會通過決議 鼓勵兩岸迅速恢復對話 |publisher = [[中央社|中央社]]}}</ref>

2019年1月31日，[[歐盟執委會|歐盟執委會]]人道援助及危機管理執委{{link-en|斯蒂里亞尼迪斯|Christos Stylianides}}受邀至歐洲議會上報告兩岸議題時表示，歐盟對近年兩岸關係趨於緊張感到遺憾，就中共總書記[[習近平|習近平]]在《[[告台灣同胞書|告台灣同胞書]]》發表40周年紀念會上的[[習五條|對台談話]]，歐盟希望兩岸能和平對話，並強調尊重台灣治理體系，且台灣與歐盟在民主、法治及人權具有共同價值，鼓勵台灣積極參與國際事務，亦指出自2016年以來中國限制兩岸高層交流，且欲壓縮台灣國際空間及誘使盟友改變立場，歐盟對此持續鼓勵兩岸進行建設性對話，樂見歐洲議會關注兩岸議題，歐盟籲兩岸維持現狀，並在與台灣共享價值及利益下，歐盟將繼續尋求與台灣在不同領域緊密合作。<ref>{{Cite web |url = https://www.cna.com.tw/news/firstnews/201901310020.aspx |author = 唐佩君 |date = 2019年1月31日 |title = 兩岸議題登歐洲議會 歐盟強調與台灣合作 |publisher = [[中央社|中央社]]}}</ref>

===協定===
1990年8月1日，駐比利時台北經濟文化辦事處代表[[舒梅生|舒梅生]]與歐洲商會主席R. Delorozoy在布魯塞爾簽署《中歐互免海運事業[[所得稅|所得稅]]議定書》。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=5BCEF92E2488FE42 |author = 公眾外交協調會 |date = 1990年8月2日 |title = 駐比利時舒代表梅生與歐洲商會主席R. Delorozoy |publisher = 中華民國外交部}}</ref>

為增進中華民國與[[歐洲國家和地區列表|中、東歐國家]]的合作關係，中華民國經濟部[[國際合作發展基金會|海外經濟合作發展基金管理委員會]]（海合會）於1994年4月17日在[[歐洲復興開發銀行|歐洲復興開發銀行]]的[[聖彼德堡|聖彼德堡]]年會中，由駐比利時台北經濟文化辦事處代表[[魯肇忠|魯肇忠]]與該行總裁{{link-en|戴拉賀謝|Jacques de Larosière}}簽署[[諒解備忘錄|諒解備忘錄]]，對中、東歐友好國家進行共同[[融資|融資]]。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=AED02C67F7DE466E |author = 公眾外交協調會 |date = 1994年4月19日 |title = 為增進我與中、東歐國家之合作關係 |publisher = 中華民國外交部}}</ref>

2017年9月，歐盟在貿易政策報告文件中指出，已準備與台灣展開投資談判。<ref>[http://www.cna.com.tw/news/firstnews/201709195004-1.aspx 歐盟指準備與台灣展開投資談判]</ref>
