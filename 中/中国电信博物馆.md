'''中国电信博物馆'''，位于[[北京市|北京市]][[海淀区|海淀区]][[学院路_(北京)|学院路]]42号，是[[中华人民共和国|中华人民共和国]][[中华人民共和国通信|通信行业]]综合性博物馆、[[国家二级博物馆|国家二级博物馆]]，隶属[[中国电信集团公司|中国电信集团公司]]。

== 简介 ==
1983年，[[中华人民共和国邮电部|中华人民共和国邮电部]]成立中国邮电博物馆筹建处。1999年8月30日，'''中国邮电博物馆'''首次对外接待第22届[[万国邮政联盟|万国邮政联盟]]大会代表团。2001年9月，更名为'''中国电信博物馆'''，隶属[[中国电信集团公司|中国电信集团公司]]。同年10月30日，中国电信博物馆正式对外开放。截至2012年有馆藏文物17000多件，展出面积7000多平方米<ref>{{cite web |url = http://museum.chinatelecom.com.cn/service/gk/201209/t20120913_634.html |title = 博物馆简介 |author =  |date = 2012-09-13 |website = 中国电信博物馆 |language = zh |accessdate =  |quote =  }}</ref>。2012年，该馆宣布“本馆因内部装修暂时关闭”<ref>{{cite web |url = http://museum.chinatelecom.com.cn/news/announcement/201209/t20120926_722.html  |title = 闭馆公告  |author =  |date = 2012-09-26  |website = 中国电信博物馆 |language = zh |accessdate =  |quote =  }}</ref>。

中国电信博物馆对外开放初期，设有通信史厅、科技科普厅等。通信史厅展示了自3000多年前通信活动萌芽到当代电信发展成果。陈列通信藏品千余件，包括清末龙图案电话机、共电式人工交换机、韦斯登重锤式发报机等。2008年，中国电信博物馆推出“中国电信专题展览”，分为企业形象展示、商务领航、电子政务、信息田园、个人信息化、我的e家等板块<ref>{{cite web |url = http://tech.qq.com/a/20091223/000340.htm |title = 程京生：中国电信创建的博物馆简述  |author =  |date = 2009-12-23 |website =  腾讯 |language = zh |accessdate =  |quote =  }}</ref>。
