'''中国人民解放军成都军区空军'''，机关驻地[[四川省|四川省]][[成都市|成都市]]，是[[中国人民解放军空军|中国人民解放军空军]]驻[[中国人民解放军成都军区|中国人民解放军成都军区]]的战役军团。归中国人民解放军空军建制，受中国人民解放军空军、中国人民解放军成都军区双重领导。领导和指挥驻[[四川省|四川省]]、[[云南省|云南省]]、[[贵州省|贵州省]]、[[重庆市|重庆市]]、[[西藏自治区|西藏自治区]]空军部队。主要担负本区防空作战，空中进攻作战，协同陆军作战等任务<ref name=cdjqkj>{{Cite web |url = http://kj.81.cn/content/2012-12/11/content_5137122.htm |title = 成都军区空军 |author =  |date =2012-12-11  |publisher = 中国空军网 |language =  |accessdate =  |quote =  }}</ref>。

== 沿革 ==
1960年8月1日，[[中国人民解放军昆明军区|中国人民解放军昆明军区]]空军指挥所在[[云南省|云南省]][[昆明市|昆明市]]组建。刘懋功任主任，崔文斌任政治委员。

1965年10月19日，中国人民解放军成都军区空军指挥所在四川省成都市成立。高厚良任主任，丁钊任政治委员。1969年1月底，[[中国人民解放军空军拉萨基地|空军拉萨指挥所]]及其所属驻藏部队，从[[兰州军区空军|兰州军区空军]]划归成都军区空军指挥所建制领导。

1976年4月，空军精简整编，将昆明军区空军指挥所、成都军区空军指挥所分别改编为空军第五军、第八军军部。1978年10月，空军第五军、第八军军部分别改称昆明军区空军指挥所、成都军区空军指挥所。

1985年8月15日，[[中央军事委员会|中央军委]]批准，成都军区空军指挥所、昆明军区空军指挥所合并，整编成'''中国人民解放军成都军区空军'''。9月9日，中国人民解放军成都军区空军在成都成立。机关设司令部、政治部、后勤部、航空工程部。1993年9月，航空工程部更名为装备技术部，1998年12月又整编为装备部<ref name=cdjqkj/>。

成都军区空军成立后，成都、昆明军区空军指挥所部队改归成都军区空军建制领导。同时还调整、撤销、合并了一部分师级、团级单位，在昆明成立了成都军区空军前方指挥所。1987年6月，该指挥所更名为空军昆明指挥所，归属成都军区空军建制，统一领导驻云南空军部队。1993年1月，中央军委下达空军编制体制调整命令，空军昆明指挥所改称空军昆明基地，执行副军级权限。2003年12月，空军昆明基地撤销，缩编为正师级的成都军区空军昆明指挥所。到20世纪末，成都军区空军已成为由航空兵、高射炮兵、地空导弹兵、通信兵、雷达兵及其他专业兵组成的合成部队<ref name=cdjqkj/>。

1985年12月，成都军区空军派出[[直升机|直升机]]飞越[[青藏高原|青藏高原]]抗雪救灾。1987年5月，成都军区空军紧急组织空军部队进入西藏自治区，这是[[中国人民解放军历史|中国人民解放军历史]]上在高原规模最大的空运任务。成都军区空军还先后参加了北京国庆[[阅兵|阅兵]]、科研试验等任务，并执行过抗震救灾、抗洪抢险、飞播造林等任务<ref name=cdjqkj/>。

2016年2月1日，[[中国人民解放军战区|中国人民解放军战区]]成立大会在北京[[八一大楼|八一大楼]]举行<ref>{{Cite web |url = http://news.xinhuanet.com/politics/2016-02/01/c_1117960554.htm |title = 中国人民解放军战区成立大会在北京举行 |author = |date = 2016-02-01 |website = 新华网 |language =  |accessdate = |quote =  }}</ref>。各[[大军区|大军区]]随之撤销，成都军区空军也由此撤销。
