'''中国人民解放军陆军装甲兵学院'''，简称'''陆军装甲兵学院'''，本部位于[[北京市|北京市]][[丰台区|丰台区]]，隶属[[中国人民解放军陆军|中国人民解放军陆军]]，是培养装甲兵指挥、参谋、技术保障军官和指挥管理、维修保障士官的综合性高等教育院校。

== 沿革 ==
;中国人民解放军装甲兵学院
*1950年，中国人民解放军坦克第一师第三团改建'''中国人民解放军坦克学校'''。1950年9月1日，坦克学校在天津光明电影院召开成立大会暨开学典礼<ref name=yanzhao/><ref>{{Cite web |url =http://dangshi.people.com.cn/n/2014/0120/c85037-24167047-2.html  |title =许光达情系军队教育事业  |author = 何立波 |date = 2014-01-20 |publisher =中国共产党新闻网  |language =  |accessdate =  |quote =  }}</ref>。原校址在[[天津市|天津市]]马场道和大王庄两地，后因驻地分散及缺乏训练场地等因素，分批迁至[[北京市|北京市]][[长辛店镇|长辛店]]<ref name=yi>一正，人民解放军第一所坦克学校，军事历史1993年03期</ref>。
*1951年3月，更名'''中国人民解放军第一战车学校'''<ref name=yi/>。
*1953年3月，更名'''中国人民解放军第一坦克学校'''<ref name=yi/><ref>{{Cite web |url =  http://dangshi.people.com.cn/GB/146570/165271/9802627.html|title = 人民解放军的“装甲兵之父” |author = 郭胜伟 |date =  2009-08-06|publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。
*1969年2月，撤销第一坦克学校<ref name=yi/>。
*1978年12月，经中央军委批准，重建为'''中国人民解放军坦克学校'''，位于安徽省[[宿县|宿县]]<ref name=yanzhao/>。校長白連，副校長朱海峰，教练团部队長陈森林。
*1982年7月，迁至安徽省[[蚌埠市|蚌埠市]]<ref name=yanzhao/><ref name=junshizhi/>。校長朱海峰。
*1986年6月，升格'''中国人民解放军蚌埠坦克学院'''<ref name=yanzhao/>。
*2011年11月，升格'''中国人民解放军装甲兵学院'''，副军级<ref name=yanzhao>{{Cite web |url =  http://yz.chsi.com.cn/sch/schoolInfo--schId-368222,categoryId-467572,mindex-1.dhtml|title = 装甲兵学院 |author =  |date =  |publisher = 中国研究生招生信息网 |language =  |accessdate = 2016-11-20 |quote =  }}</ref>。是培养全军装甲兵部队军事指挥与参谋人才的兵种任职教育院校，主要承担生长干部任职培训、现职干部任职培训、现职干部轮训、研究生教育等培训任务，同时承担培养来自20余个国家的外国留学生及部分外授装备接装培训任务<ref>{{Cite web |url =  http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536809.htm|title = 装甲兵学院 |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军总参谋部军训和兵种部|中国人民解放军总参谋部军训和兵种部]]转隶[[中国人民解放军陆军|中国人民解放军陆军]]<ref>{{cite web|title=装甲兵学院由原总参谋部军训和兵种部转隶陆军|url=http://www.thepaper.cn/newsDetail_forward_1523669|publisher=澎湃新闻|date=2016-09-03}}</ref>。

;中国人民解放军装甲兵工程学院
* 1961年，以[[中国人民解放军军事工程学院|中国人民解放军军事工程学院]]装甲兵工程系为基础，在[[西安|西安]]组建'''中国人民解放军装甲兵工程学院'''<ref name=rmwjj>{{Cite web |url =http://military.people.com.cn/GB/52578/52579/193612/11840023.html  |title =装甲兵工程学院简介  |author =  |date = 2010-06-10 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
* 1969年，装甲兵工程学院、第二坦克学校、第五坦克技术学校（部分）合并，在[[北京|北京]]组建'''中国人民解放军坦克技术学校'''。是全军首批对外开放单位之一，1971年起开始培训外国留学生<ref name=rmwjj/>。
**中国人民解放军第二坦克学校：1952年，步兵第一八四师直属部队、坦克第二十六师直属部队一部、坦克第五十一团修理连、坦克第五十二团大部合并，组建'''中国人民解放军第二战车学校'''。1953年，更名'''中国人民解放军第二坦克学校'''。1969年撤销建制，部分改编坦克第十二师，大部分参与组建中国人民解放军坦克技术学校。
* 1976年，更名'''中国人民解放军装甲兵技术学校'''<ref name=rmwjj/>。
* 1978年，升格'''中国人民解放军装甲兵技术学院'''<ref name=rmwjj/>。
* 1986年，更名'''中国人民解放军装甲兵工程学院'''<ref name=rmwjj/>。副军级。装甲兵工程学院是全国重点工科院校，“十一五”期间军队“2110工程”重点建设院校<ref name=rmwjj/>。
* 1999年5月，由[[中国人民解放军总参谋部|中国人民解放军总参谋部]]转隶[[中国人民解放军总装备部|中国人民解放军总装备部]]<ref name=rmwjj/>。
* 2004年，装甲兵技术研究所并入。
* 2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军总装备部|中国人民解放军总装备部]]转隶[[中国人民解放军陆军|中国人民解放军陆军]]<ref>{{cite web |title=全军后勤最高学府后勤学院已由原总后转隶中央军委训练管理部 |url = http://www.thepaper.cn/newsDetail_forward_1473027 |website = 澎湃新闻 |date=2016-05-23 }}</ref>。

;中国人民解放军陆军装甲兵学院
*2017年，以原中国人民解放军装甲兵学院、中国人民解放军装甲兵工程学院、中国人民解放军装甲兵技术学院为基础重建'''中国人民解放军陆军装甲兵学院'''<ref>{{Cite web |url = http://news.ifeng.com/a/20170530/51183107_0.shtml |title =  2017年军队招生单位名录发布 军改后院校新名称曝光|author =  |date = 2017-05-30 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。是培养装甲兵指挥、参谋、技术保障军官和指挥管理、维修保障士官的综合性高等教育院校。院本部为原中国人民解放军装甲兵工程学院，位于北京市丰台区卢沟桥畔；蚌埠校区为原中国人民解放军装甲兵学院，位于安徽省蚌埠市蚌山区；[[中国人民解放军陆军装甲兵学院士官学校|士官学校]]为原中国人民解放军装甲兵技术学院，位於吉林省长春市经济技术开发区<ref name=xunli/>。2017年8月1日[[中国人民解放军建军纪念日|中国人民解放军建军纪念日]]，中国人民解放军陆军装甲兵学院举行揭牌仪式，该院蚌埠校区、士官学校同步举行揭牌活动<ref name=jiepai>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1749302 |title = 陆军装甲兵学院今日举行揭牌仪式，徐航、杨戈分任院长政委 |author =  |date =2017-08-01  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

==专业设置==
2017年组建时，中国人民解放军陆军装甲兵学院拥有以工学、军事学为主体、多学科交叉融合的综合性学科体系，在工学、管理学、军事学3个学科门类有6个博士后科研流动（工作）站，取得5个一级学科博士学位授权，有19个博士点；取得6个一级学科、3个二级学科、8个专业学位硕士授权，有50个硕士点；开设18个本科专业。2017年招生的11个本科专业是<ref name=xunli/>：
*机械工程<ref name=xunli/>
*电气工程及其自动化<ref name=xunli/>
*通信工程<ref name=xunli/>
*光电信息科学与工程<ref name=xunli/>
*武器系统与工程<ref name=xunli/>
*装甲车辆工程<ref name=xunli/>
*作战指挥<ref name=xunli/>
*火力指挥与控制工程<ref name=xunli/>
*无人系统工程<ref name=xunli/>
*指挥信息系统工程<ref name=xunli/>
*仿真工程<ref name=xunli/>

==装甲兵装备技术研究所==
原装甲兵工程学院下设装甲兵装备技术研究所，是中国人民解放军唯一的装甲装备综合性研究所，也是中国人民解放军新型坦克装甲车辆的“质量检验站”和“保障研究中心”<ref name=kaoyan>{{Cite web |url = http://www.chinakaoyan.com/graduate/intro/schoolID/230.shtml |title = 装甲兵工程学院 |author =  |date =  |publisher = 中国考研网 |language =  |accessdate = 2017-05-17 |quote =  }}</ref>。

该研究所拥有装甲兵装备技术研究所试验场，这是[[亚洲|亚洲]]规模最大、现代化程度最高的坦克装甲车辆试验场<ref name=kaoyan/>，也是为中国人民解放军新型战车进行设计定型试验的唯一单位。该试验场于1960年成立<ref name=baoguang>{{Cite web |url =http://www.81.cn/rd/2016-01/26/content_6869226.htm  |title = 陆军装甲兵某试验场试验新装备过程曝光 |author =  |date =2016-01-26  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。1971年，该试验场在[[黑龙江省|黑龙江省]][[塔河县|塔河县]]设立北方站，用以考验新研制装备在严寒地区的适应能力<ref>{{Cite web |url = http://news.xinhuanet.com/politics/2016-01/22/c_1117868417.htm |title = 中国军队新研制装甲装备寒区试验见闻 |author =  |date = 2016-01-22 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref name="baoguang" />。截至2016年，该试验场已完成自59式坦克到新型主战坦克、自单一型号到装甲车族126型372台次装备的设计定型试验，获得100多项国家军队科技成果奖。2016年，该场正从粗放型向精细型、从任务型向研究型数字化发展<ref name="baoguang" /><ref>{{Cite web |url = http://news.youth.cn/jsxw/201601/t20160128_7577363.htm |title = 新闻特写：走进装甲试验场 这个冬天不够冷 |author =  |date = 2016-01-28 |publisher =中国青年网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.guancha.cn/military-affairs/2017_05_17_408788.shtml |title = 海防、预备役及试验部队转隶陆军 助推陆军转型 |author =  |date = 2017-05-17  |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.ifeng.com/mil/2/detail_2012_10/28/18616093_0.shtml |title =装甲兵某研究所试验场：让战车安全驶入“战场”  |author =  |date =2012-10-28  |publisher =  凤凰网|language =  |accessdate =  |quote =  }}</ref>。
