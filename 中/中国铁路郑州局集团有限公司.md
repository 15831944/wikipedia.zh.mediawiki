'''中国铁路郑州局集团有限公司'''<ref name="公司化改革">{{cite web|author1=徐书婷|title=铁路局公司制改革:四地负责人从局长变成董事长|url=http://m.guancha.cn/politics/2017_10_22_431855.shtml|website=观察者网|accessdate=2017-10-22|location=上海|language=zh-cn}}</ref>，原称'''郑州铁路局'''，是[[中国铁路总公司|中国铁路总公司]]下属公司，受[[武汉铁路监督管理局|武汉铁路监管局]]监管。管辖线路主要分布在[[河南省|河南省]]北部和[[山西省|山西省]]南部，另外[[陇海铁路|陇海铁路]][[太要站|太要站]]、[[宁西铁路|宁西铁路]][[富水站_(中国)|富水站]]（位于陕西省境内）及其附近线路区间以及[[新兖铁路|新兖铁路]][[算王庄站|算王庄站]]（位于山东省境内）及其以西线路区间也隶属于郑局。

郑铁集团位于中国铁路网中心，其中[[郑州铁路枢纽|郑州铁路枢纽]]地处南北大动脉[[京广铁路|京广铁路]]与东西大动脉[[陇海铁路|陇海铁路]]的交汇点，是中国重要的铁路交通枢纽，在中国铁路交通史上具有重要地位。

==沿革==
*1904年，郑县站始设立，为[[卢汉铁路|卢汉铁路]]（[[卢沟桥|卢沟桥]]至[[汉口|汉口]]）的业务站。
*1948年10月22日，[[中国人民解放军|中国人民解放军]]占领郑州，“中原陇海、平汉铁路郑州联合管理委员会”随后成立。中原临时人民政府交通部部长[[许子威|许子威]]兼任军事代表、党委书记。
*1949年2月，更名为“中原陇海、平汉铁路联合管理局”。同年3月，“中国人民革命军事委员会铁道部郑州铁路管理局”成立。
*1958年9月，原郑州铁路管理局一分为三，铁道部郑州、武汉、西安铁路局分别正式挂牌成立。
*1963年4月，武汉铁路局撤销，并入郑州铁路局。
*1971年7月，武汉铁路局恢复设立，郑州铁路局一分为二。
*在[[文革|文革]]期间，为保证全国铁路交通的正常运行秩序，郑州铁路局曾实施军事管制。
*1983年3月，武汉铁路局再次撤销，并入郑州铁路局。
*1984年10月，西安铁路局撤销，并入郑州铁路局。
*2005年3月，武汉铁路局、西安铁路局恢复设立，郑州铁路局一分为三。
*郑州铁路局在“2010中国企业500强排行榜”中排名第159位。

1996年，郑州铁路局进行了工商登记<ref name=guapai/>。

2017年9月23日，郑州铁路局召开电视电话会议，宣布京沪高速铁路股份有限公司原董事长、党委书记钱铭任中国铁路郑州局集团有限公司董事长、党委书记。这是铁路局公司化改革启动后，中铁总下属18个路局（公司）中，首个宣布路局董事长人选的企业。

2017年11月4日，济南、乌鲁木齐、南昌、北京、哈尔滨、武汉、郑州、上海、呼和浩特、昆明、青藏铁路公司等11家铁路局（公司）的名称变更通过[[国家工商行政管理总局|国家工商行政管理总局]]核准，郑州铁路局名称变更为'''中国铁路郑州局集团有限公司'''（(国)名称变核内字[2017]第5283号）<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1851596  |title = 15家铁路局将集体变身集团公司：名称变更已通过核准 |author =  |date =2017-11-06   |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月15日，在国家企业信用信息公示系统中，中国铁路郑州局集团有限公司已完成工商登记变更。工商登记变更后，中国铁路郑州局集团有限公司仍为中国铁路总公司100%控股的公司，公司由原先的全民所有制企业变成有限责任公司（非自然人投资或控股的法人独资）企业，注册资本不变，经营范围增加住宿餐饮业，建设项目承发包、勘察、设计、施工与监理，租赁，房地产开发经营和中介，保险经纪与代理服务，现代服务，广播、电视、电影和影视录音制作，文化艺术服务，招标采购及服务，科学研究和技术服务，园艺作物种植，林木育种、育苗、种植和管护，木材和竹材采运等<ref name=wu>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1865510|title =北京济南等14家铁路局正式变身路局集团：完成工商登记变更|author =  |date = 2017-11-15 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

==机务段==
[[File:HXD1B-0233_at_Zhengzhoubei_20180228_110025.jpg|thumb]]于郑州北整备场]]
机务段负责存放、维护、检修铁路机车，工作通常包括清洗、加油、加水、检修、除尘等。郑州铁路局下辖以下机务段：
*郑州机务段（郑局郑段）
*洛阳机务段（郑局洛段）
*新乡机务段（郑局乡段）
在生产力布局调整完成前，郑州铁路局还曾有下列机务段：
*宝丰机务段（郑局丰段，已并入洛阳机务段）
*三门峡西机务段（郑局峡段，已并入洛阳机务段）
*月山机务段（郑局月段，已并入新乡机务段）
*郑州南机务段（郑州南机务段，与郑州北机务段合并成立郑州机务段）
