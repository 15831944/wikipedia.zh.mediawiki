'''中国人民解放军第十八医院'''，位于[[新疆维吾尔自治区|新疆维吾尔自治区]][[叶城县|叶城县]]幸福南路9号，隶属[[中国人民解放军南疆军区|中国人民解放军南疆军区]]。

==简介==
中国人民解放军第十八医院位于新疆叶城县，叶城县是国家级贫困县，自然环境十分恶劣<ref name=qianli/>。该医院在搞好部队卫勤保障工作的同时，数十年坚持开展医疗助民、帮教助学、扶贫帮困等活动<ref name=jishi/>。2010年6月，[[兰州军区|兰州军区]]兰州总医院遵照[[中华人民共和国卫生部|中华人民共和国卫生部]]、[[中国人民解放军总后勤部卫生部|中国人民解放军总后勤部卫生部]]《关于进一步加强对新疆、西藏地区县医院对口支援工作的通知》要求，分别和新疆博尔塔拉自治州人民医院、中国人民解放军第十八医院签订帮扶协议书，结成帮扶对子<ref name=qianli>{{Cite web |url = http://news.cntv.cn/20110817/119479.shtml |title = 千里入疆送安康 |author =  |date = 2011-08-17 |publisher = 央视网 |language =  |accessdate =  |quote =  }}</ref>。

该医院隶属[[中国人民解放军南疆军区|中国人民解放军南疆军区]]<ref>{{Cite web |url = http://military.cnr.cn/ddtp/20160129/t20160129_521276985.html |title = 我军直升机穿越“飞行禁区”营救突患高原脑水肿战士 |author =  |date = 2016-01-29 |publisher = 央广军事 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://army.81.cn/content/2016-05/26/content_7073671.htm |title =解放军第十八医院大力开展义务巡诊活动  |author =  |date = 2016-05-26 |publisher =中国陆军  |language =  |accessdate =  |quote =  }}</ref>。
