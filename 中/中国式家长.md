《'''中国式家长'''》是墨魚玩工作室製作，椰島遊戲發行的一款模擬養成遊戲。玩家在游戏中扮演一位出生在普通的中式家庭的孩子。

== 背景与发行 ==
《中国式家长》由墨魚玩工作室製作，椰島遊戲發行。該遊戲以「[[中华人民共和国教育|中國教育]]」為主軸，開發團隊希望以「中國式」元素還原一段中國式的成長故事。<ref name="聯合">{{cite web |title=最真實的《中國式家長》 讓你回溫人生的成長歷程 |url=https://game.udn.com/game/story/10453/3397029 |website=聯合新聞網 |accessdate=2018-10-01|language=zh-tw}}</ref>游戏制作人刘祯浩于2018年7月表示，做这款游戏至少在一年前就被列入了计划。刘祯浩和另一名制作人杨葛一郎对“中国式家长”有充分体验，他们希望做出这款游戏让“孩子与家长能相互理解”。2018年春季，游戏基本完成；同年5月、6月，游戏在网上开放玩家试玩，并出现在了一些展会中。<ref name="中新网">{{cite web |title=当“中国式家长”成为游戏，你会更理解父母吗？|url=http://www.chinanews.com/cul/2018/07-12/8563940.shtml |website=中新网 |accessdate=2018-10-01|language=zh-cn}}</ref>9月29日，遊戲正式登陸[[Steam|Steam]]和[[WeGame|WeGame]]平台。<ref name="新浪">{{cite web |title=國產獨立遊戲《中國式家長》今日登陸WeGame平台 |url=https://m.sina.com.hk/news/article/20180929/5/48/2/%E5%9C%8B%E7%94%A2%E7%8D%A8%E7%AB%8B%E9%81%8A%E6%88%B2-%E4%B8%AD%E5%9C%8B%E5%BC%8F%E5%AE%B6%E9%95%B7-%E4%BB%8A%E6%97%A5%E7%99%BB%E9%99%B8WeGame%E5%B9%B3%E5%8F%B0-9280960.html |website=新浪香港 |accessdate=2018-10-01 |language=zh-hk}}</ref>而“墨鱼玩”工作室创始人杨葛一郎则表示，希望能推出智能手机版的游戏，并在2019年能推出游戏英文版。<ref name="NYT"/>

== 玩法 ==
在遊戲中，玩家扮演一位出生在普通的中式家庭的孩子。遊戲模擬了遊戲角色從出生到[[普通高等学校招生全国统一考试|高考]]的整段過程，每局游戏游戏共48轮。玩家在遊戲中安排游戏角色的学习时间，学习各种技能，达到家长所期待的目标，同时也可以给孩子买各种礼物，这些选择将塑造孩子在智商、情商、体魄、想象力、记忆力，以及魅力这六个方面的发展。但玩家需要在自我心理压力和父母满意度之间维持平衡。遊戲採取個性化培養系統，玩家可自主決定未來的發展方向，职业结局则最终由角色特长决定，有八十多种结局设定。此外，遊戲亦開發了面子對決、特長選秀、班干選舉等桥段，並加入了校園戀愛的劇情。而游戏结束后，游戏角色将会作为下一代孩子的父亲，并因其职业、性格、配偶，影响下一代的基础属性、零用钱等。<ref name="聯合"/><ref name="中新网"/><ref name="新浪"/><ref name="NYT"/>
