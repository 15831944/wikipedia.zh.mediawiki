'''中国人民解放军第五医院'''，位于[[宁夏回族自治区|宁夏回族自治区]][[银川市|银川市]][[兴庆区|兴庆区]]胜利南街839号，隶属[[西宁联勤保障中心|西宁联勤保障中心]]。

==简介==
1946年9月，中国人民解放军第五医院成立。医院为[[三级甲等医院|三级甲等医院]]。先后参加了[[宁夏战役|宁夏战役]]、[[抗美援朝战争|抗美援朝战争]]、[[对越自卫反击战|对越自卫反击战]]、[[小汤山医院|小汤山]]抗击“[[非典|非典]]”、[[舟曲泥石流|舟曲泥石流]]救灾、“[[神舟|神舟]]”系列飞船等卫勤保障任务。21世纪初，该医院建有国家级中西医结合肺病专科，军区级创伤骨科中心、眼科中心、泌尿外科中心，宁夏回族自治区白内障人工晶体植入中心等，并且建有全军区域性银川三级设备维修站、全军三级计量检测站、中国人民解放军银川血站<ref name=gfb>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536774.htm |title = 第五医院 |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。医院先后被全军评为“全军为部队服务先进单位”、“优质服务先进单位”、 被宁夏回族自治区评为“拥政爱民模范单位”等<ref>史纲、乔佳，一切服务于部队 一切服务于临床——解放军第五医院医学工程科发展纪实，医疗卫生装备2013(8):4-5</ref><ref name=gfb/>。

中国人民解放军第五医院原隶属于[[兰州军区|兰州军区]]联勤部<ref>{{Cite web |url = http://news.163.com/10/0415/07/649U95C000014AED.html |title =  第五医院专家医疗队走进社区开展便民服务|author =  |date = 2010-04-15  |publisher =网易  |language =  |accessdate =  |quote =  }}</ref><ref>兰州军区第五医院医务处，砺兵雪域高原 投身强军实践——兰州军区第五医院野战医疗队高原寒区使命课题训练纪实，解放军预防医学杂志2014年06期</ref>。2016年，兰州军区撤销，该医院转隶[[西宁联勤保障中心|西宁联勤保障中心]]<ref>{{Cite web |url = http://army.81.cn/content/2017-04/19/content_7568033.htm |title =解放军第五医院开展“健康军营行”活动  |author =  |date = 2017-04-19 |publisher = 中国陆军网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://army.81.cn/content/2017-05/12/content_7598907.htm|title = 解放军第五医院获评中医药工作先进单位 |author =  |date =2017-05-12  |publisher =中国陆军网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://army.81.cn/content/2017-05/03/content_7586195.htm  |title =解放军第五医院野战医疗保障演练磨砺保障硬功  |author =  |date =2017-05-03  |publisher =  中国陆军网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://army.81.cn/content/2017-06/21/content_7647403.htm|title =解放军第五医院举行“高原寒区使命训练”誓师大会  |author =  |date =2017-06-21  |publisher =中国陆军网  |language =  |accessdate =  |quote =  }}</ref>。
