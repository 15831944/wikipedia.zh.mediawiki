'''中華民國與克羅埃西亞關係'''是指[[中華民國|中華民國]]與[[克羅埃西亞共和國|克羅埃西亞共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對克羅埃西亞的相關事務由'''[[駐奧地利台北經濟文化代表處|駐奧地利台北經濟文化代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-151-73607-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
{{see also|中華民國外交|克羅埃西亞外交}}
1992年10月，克羅埃西亞觀光暨貿易部長{{link-en|宓米卡|Neven Mimica}}（Neven Mimica）抵台訪問。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-4.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1993年1月12日，在[[克羅埃西亞獨立戰爭|克羅埃西亞獨立戰爭]]期間，中華民國[[人道援助|人道援助]]該國10萬[[美元|美元]]。<ref name="86年外交年鑑"/>

1995年6月，中華民國經濟部長[[江丙坤|江丙坤]]率團赴[[瑞士|瑞士]]參加蒙太納[[論壇|論壇]]，期間會見克羅埃西亞副總理{{link-hr|史克葛羅|Borislav Škegro}}（[[克羅埃西亞文|克羅埃西亞文]]：{{lang|hr|Borislav Škegro}}），並就雙邊經貿關係交換意見。<ref name="86年外交年鑑"/>

2016年，臺北市議會議長[[吳碧珠|吳碧珠]]訪問克羅埃西亞。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|歐洲聯盟簽證政策|{{link-en|克羅埃西亞公民簽證要求|Visa requirements for Croatian citizens}}}}
[[欧洲联盟成员国|歐盟成員]]的克羅埃西亞尚未獲准加入[[申根區|申根區]]，但基於《[[申根公約|申根公約]]》，該國實行與申根區相同的簽證政策。<ref>{{Cite web |url = http://www.mvep.hr/en/consular-information/visas/visa-requirements-overview/ |title = Visa requirements overview |publisher = {{link-en|克羅埃西亞外交暨歐洲事務部|Ministry of Foreign and European Affairs (Croatia)}}}}</ref>歐洲申根區給予持有載明[[中華民國國民身份證|身份證]]字號的[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以[[歐洲聯盟簽證政策|歐盟免簽證]]入境，停留日數單獨計算，不與申根區合併，每6個月期間內總計可停留90天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-151-73607-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_FAQ.aspx?c=AD6908DFDDB62656&n=C1B19EF0ACDD3C91&sms=867F5398A49F758A |title = 歐盟免申根簽證常見問答集 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

[[克羅埃西亞人口|克羅埃西亞公民]]持有[[歐盟護照|歐盟護照]]者也可以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將克羅埃西亞歸類為'''灰色警示'''：提醒注意（2017年4月20日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-107-8fdac-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2013年，[[經濟部國際貿易局|經濟部國際貿易局]]挑選最具拓銷潛力的22國26城市推動執行「貿易尖兵任務」，其中[[東歐|東歐]]7國的[[新興市場|新興市場]]就包括[[克羅埃西亞|克羅埃西亞]]。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622045&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2013年10月30日 |title = 貿易尖兵勇闖22國26城市 新興市場商機無窮 |publisher = 經貿透視網}}</ref>

2018年，克羅埃西亞是中華民國第99大貿易夥伴、第129大[[進口|進口]]夥伴、第85大[[出口贸易|出口]]夥伴。出口至克羅埃西亞的金額為6,184萬3,283[[美元|美元]]，年增41.006%；自克羅埃西亞進口的金額為408萬6,535美元，年減31.112%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）5,775萬6,748美元，年增52.286%。2017年，克羅埃西亞是中華民國第105大貿易夥伴、第121大進口夥伴、第89大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
根據經濟部投資審議委員會統計，截至2015年4月，尚無前往克羅埃西亞投資的[[台灣商人|台商]]。<ref>{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/hr/pdf/104-65Croatia.pdf |date = 2015年7月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

===教育===
{{see also|台灣教育|{{link-en|克羅埃西亞教育|Education in Croatia}}}}
[[中華民國教育部|中華民國教育部]]曾於1991年7月起設立「[[獨立國家國協|獨立國家國協]]及[[東歐|東歐地區國家]]來華留學[[獎學金|獎學金]]」以研習[[中華民國國語|中文]]，其中包括克羅埃西亞。<ref name="86年外交年鑑"/>
