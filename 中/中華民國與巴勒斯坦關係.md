'''中華民國與巴勒斯坦關係'''是指[[中華民國|中華民國]]與[[巴勒斯坦國|巴勒斯坦國]]之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對巴勒斯坦[[約旦河西岸|約旦河西岸]]及[[加薩走廊|加薩走廊]]的相關事務由'''駐[[臺拉維夫|臺拉維夫]][[台北經濟文化辦事處|台北經濟文化辦事處]]'''、'''駐[[約旦|約旦]]臺北經濟文化辦事處'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = http://www.roc-taiwan.org/il/post/4.html |title = 代表處簡介 |publisher = 中華民國駐外單位聯合網站}}</ref>

1994年5月10日，[[中華民國總統|中華民國總統]][[李登輝|李登輝]]前往當時的[[中華民國外交#中华民国的邦交国|邦交國]]－[[南非|南非]]，參加{{link-en|1994年南非國會選舉|South African general election, 1994|新任南非總統}}[[曼德拉|曼德拉]]的就職典禮，其間與[[巴勒斯坦解放組織|巴勒斯坦領袖]][[阿拉法特|阿拉法特]]不期而遇並且短暫寒暄。李登輝日後說道，他與阿拉法特雖無共同之處，但阿拉法特的確是一位偉大的人物。<ref>{{Cite web|url =http://www.newtaiwan.com.tw/bulletinview.jsp?bulletinid=16271|author =李心怡|date =2004年3月15日|title =李登輝年表|publisher =《[[新台灣新聞周刊|新台灣新聞周刊]]》|deadurl =yes|archiveurl =https://web.archive.org/web/20161025142340/http://www.newtaiwan.com.tw/bulletinview.jsp?bulletinid=16271|archivedate =2016年10月25日}}</ref><ref>{{Cite web |url = http://www.taiwanus.us/MediaVideoAudio/books/lee/4.htm |author = 上土反冬子 |date = 2001年6月8日 |title = 《虎口的總統－李登輝與曾文惠》 |publisher = Taiwanus.net 台灣海外網}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|巴勒斯坦簽證政策|Visa policy of Palestine}}|{{link-en|巴勒斯坦公民簽證要求|Visa requirements for Palestinian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。[[中華民國國民|中華民國公民]]可持載有[[中華民國國民身分證|身分證]]字號的[[中華民國護照|中華民國護照]]以[[簽證#签证优惠待遇|免簽證]]的方式入境[[以色列|以色列]]後再前往[[巴勒斯坦|巴勒斯坦]]。停留期間若前往巴勒斯坦地區，要再返回以色列時，[[以色列國防軍|以國軍方]]檢查哨會查驗入境卡與[[護照|護照]]後才能放行。對於免簽證入境以色列的外國國民實施「入境卡」制度（entry card），為身分證大小的硬質紙卡，效力與入境章戳相同，於出境時由移民官收回。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-48-9bc3c-02-1.html |title = 簽證及入境須知 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=7F220D7E656BE749 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持有[[巴勒斯坦護照|巴勒斯坦護照]]的{{link-en|巴勒斯坦人口|Demographics of the Palestinian territories|巴勒斯坦公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將巴勒斯坦[[約旦河西岸|約旦河西岸]]與[[加薩走廊|加薩走廊]]歸類為'''橙色警戒'''：避免非必要旅行，與'''紅色警戒'''：不宜前往，宜儘速離境（以上截至2018年5月11日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-454-2b393-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|巴勒斯坦經濟|Economy of the Palestinian territories}}}}
===貿易===
2018年，巴勒斯坦是中華民國第195大貿易夥伴、第215大[[進口|進口]]夥伴、第185大[[出口贸易|出口]]夥伴，在[[亞洲國家和地區列表|亞洲國家]]排名倒數第2、2、3；在[[世界政區索引|世界經濟體]]倒數第50、26、52。出口至巴勒斯坦的金額為72萬1,967[[美元|美元]]，年增95.844%；自巴勒斯坦進口的金額為1萬2,351美元，年增279.564%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）70萬9,616美元，年增94.208%。2017年，巴勒斯坦是中華民國第202大貿易夥伴、第220大進口夥伴、第195大出口夥伴，在亞洲國家排名倒數第2、1、3；在世界經濟體倒數第37、17、36。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

====客運====
兩國無直航班機，且巴勒斯坦唯一的[[亞西爾·阿拉法特國際機場|亞西爾·阿拉法特國際機場]]也因故關閉，可先從[[臺灣機場|臺灣機場]]→直航第三地→{{link-en|以色列機場列表|List of Israeli airports|以色列機場}}、{{link-en|約旦機場列表|List of airports in Jordan|約旦機場}}、{{link-en|埃及機場列表|List of airports in Egypt|埃及機場}}等[[中東#國家及地區|周邊國家]]→巴勒斯坦。
