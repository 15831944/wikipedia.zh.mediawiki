'''中华人民共和国旗帜'''是指与[[中华人民共和国|中华人民共和国]]相关的[[旗帜|旗帜]]。除国旗外，还有军旗、党旗、社团旗帜以及地方旗帜等。
==国旗==
{{main|中华人民共和国国旗}}
[[File:Flag_of_the_People's_Republic_of_China.svg|thumb]]
中华人民共和国国旗称作'''五星红旗'''，为左上角镶有五颗黄色[[五角星|五角星]]的红色旗帜，旗帜图案中的四颗小五角星环绕在一颗大五角星右侧呈半环形。红色的旗面象征[[革命|革命]]，大五角星象征中国共产党，四颗小五角星象征工人阶级、农民阶级、民族资产阶级、小资产阶级，每颗小五角星各有一个角指向大五角星的中心，象征着[[中国人|中国人民]]在共产党领导下的团结。<ref>{{cite book | last = 马 | first = 全洲 | authorlink = | author2 = 周凯军 | title = 国旗·国徽·国歌的故事 | publisher = 解放军出版社 | date = 2009-04-01 | location = 北京 | pages = 1 | url = | doi = | id = | isbn = 9787506557290 }}</ref>

该旗的设计者是[[曾联松|曾联松]]，是一名来自[[浙江省|浙江]][[瑞安市|瑞安]]的普通公民。随着中国共产党在[[国共内战|国共内战]]中取得胜利，[[中国人民政治协商会议|新政治协商会议]]筹备会在1949年7月发出了征集国旗图案的通告，曾联松设计并提交了他的国旗样稿。在2992幅<ref name="peopleprcflag">{{cite news|url=http://dangshi.people.com.cn/GB/85040/9850276.html|title=全国人民踊跃参与设计 1个月内应征国旗图案达2992幅|author=陈宇|publisher=中国共产党新闻网|date=2009-08-13|accessdate=2009-11-05}}</ref>（一说为3012幅<ref name="govprcflag">{{cite web|url=http://www.gov.cn/test/2005-05/24/content_18247.htm|title=中华人民共和国国旗|publisher=中国政府网|date=2005-05-24|accessdate=2009-11-08}}</ref>）应征国旗图案中，曾联松的设计被选入38幅候选草图。经过多次讨论和少量修改，他的设计被选为了新政权的国旗。在1949年10月1日的[[中华人民共和国开国大典|开国大典]]上，[[毛泽东|毛泽东]]宣告了[[中华人民共和国中央人民政府_(1949年－1954年)|中华人民共和国中央人民政府]]成立，并按动电钮在[[天安门广场|天安门广场]]上升起了第一面五星红旗<ref name="mzdtam">{{cite web|author=陈宇|title=组图：开国大典 毛泽东升起第一面五星红旗 (5)|url=http://dangshi.people.com.cn/GB/151935/163464/9840022.html|publisher=中国共产党新闻网|accessdate=2012-08-20|date=2009年8月12日}}</ref>。

=== 特别行政区 ===
{{Main|香港特别行政区区旗|澳门特别行政区区旗}}

[[File:PRC_and_Hong_Kong_flags.jpg|thumb]]，并列使用时国旗尺寸一般须大于区旗。]]

[[香港|香港]]和[[澳门|澳门]]两个[[特别行政区|特别行政区]]拥有自己的区旗。[[全国人民代表大会|全国人民代表大会]]相继通过了两个特别行政区区旗的规范使用方法，规定国旗和区旗同时升挂、悬挂、使用时，国旗应大于区旗，国旗应置于中心、较高或者突出的位置，以旗帜正面面向观众为基准，国旗应在右、区旗应在左；列队行进时，国旗应在区旗之前<ref name="hongkong_flag">{{cite web |url = http://www.people.com.cn/zgrdxw/faguiku/dff/F50-1060.html |title = 中华人民共和国香港特别行政区区旗区徽使用暂行办法 |author = [[全国人民代表大会香港特别行政区筹备委员会|全国人民代表大会香港特别行政区筹备委员会]] |website = 人民网 |date=1996-08-11|accessdate=2009-11-19}}</ref><ref name="macau_flag">{{cite web |url = http://www.china.com.cn/law/flfg/txt/2006-08/08/content_7064255.htm|title=中华人民共和国澳门特别行政区区旗、区徽使用暂行办法 |website = 北方网 |author = [[全国人民代表大会澳门特别行政区筹备委员会|全国人民代表大会澳门特别行政区筹备委员会]] |date=1999-01-16 |accessdate=2009-11-19 }}</ref>。

香港与澳门两地的区旗设计都使用了国旗图案的某些元素。香港区旗以红色为底色，色度值与国旗红相同，并且采用了5颗白色五角星<ref name="hongkong_flag" />。澳门区旗有一大四小五颗黄色五角星，与国旗的五星图案一致，象征着澳门“是中國不可分割的一部分”。<ref name="ambiente">{{cite web |url = http://www.ambiente.gov.mo/tchinese/07/lotus05/l0508.asp |title=蓮花與澳門特別行政區的區旗、區徽 |accessdate=2009-11-02 |author=楊允中 |publisher=澳門特別行政區政府環境委員會 |work=澳門蓮藝文化協會 |deadurl=yes |archiveurl=https://web.archive.org/web/20071223002632/http://www.ambiente.gov.mo/tchinese/07/lotus05/l0508.asp |archivedate=2007-12-23 }}</ref>

除了香港、澳门两个特别行政区有区旗之外，中华人民共和国其他一些城市也曾设计过市旗。但1997年11月[[中共中央办公厅|中共中央办公厅]]、[[中华人民共和国国务院办公厅|中华人民共和国国务院办公厅]]发出《中共中央办公厅、国务院办公厅关于禁止自行制作和使用地方旗、徽的通知》后，这些城市的市旗都已停止使用<ref>{{cite web |url = http://zh.wikisource.org/wiki/中共中央办公厅、国务院办公厅关于禁止自行制作和使用地方旗、徽的通知 |title = 中共中央办公厅、国务院办公厅关于禁止自行制作和使用地方旗、徽的通知 |accessdate = 2009-11-10 |date = 1997-11-18 |website = [[维基文库|维基文库]] }}</ref>。

== 类似旗帜与标志 ==
五星红旗的颜色、布局及内涵与[[蘇聯国旗|蘇聯国旗]]相仿，五角星为[[社会主义国家|社会主义国家]]（如-{zh-hans:[[朝鲜|朝鲜]]; zh-hant:北韓;}-、[[越南|越南]]等）常用标志{{notetag|古巴虽为社会主义国家，但[[古巴国旗|古巴国旗]]自1902年启用，其白色五角星象征独立，并非因实行社会主义而采取五角星图案。}}。但苏联和朝鲜国旗的长宽比例为2:1<ref>{{cite book | language = lv, ru | last = Strods | first = Evalds | authorlink = | author2 =Regina Jozauska | author3 =Andris Nikolajevs | title = PSRS ģerboņi un karogi | title_zh = 苏联国旗和国徽 | publisher = Avots | date = 1987 | location = Rīga, Latvijas PSR | pages = 12 | url = | doi = | id = | isbn = }}</ref>，中华人民共和国和越南国旗采取3:2比例<ref>{{cite web|title=中华人民共和国国旗法|url=https://zh.wikisource.org/wiki/%E4%B8%AD%E5%8D%8E%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD%E5%9B%BD%E6%97%97%E6%B3%95|accessdate=2016-11-15|language=zh|quote=《国旗制法说明》（一）}}</ref>。[[李光耀|李光耀]]在其回忆录中称，[[新加坡国旗|新加坡国旗]]的五角星是为了照顾[[新加坡华人|华人]]民众希望效仿中华人民共和国国旗，在国旗上加五颗五角星<ref>{{cite book |author = 李光耀 |authorlink = 李光耀 |title = The Singapore Story: Memoirs of Lee Kuan Yew |location = Singapore |publisher = Times Editions |pages= 342–343 |isbn = 978-981-204-983-4|date=1998}}</ref>，惟代表意義略有不同{{notetag|新加坡国旗五颗五角星代表了国家的五大理想：[[民主|民主]]、[[和平|和平]]、[[進步主義|進步]]、[[公正|公正]]、[[平等|平等]]<ref name="rules">{{cite web |url=http://app.www.sg/data/usermedia/documents/Singapore_Arms_And_Flag_And_National_Anthem_Rules.pdf |format=PDF |archiveurl=https://www.webcitation.org/5r7lKWdGk?url=http://app.www.sg/data/usermedia/documents/Singapore_Arms_And_Flag_And_National_Anthem_Rules.pdf |archivedate=2010-07-10 |title=《新加坡国徽、国旗、国歌法》 (Cap. 296, R 1, 2004 Rev.
