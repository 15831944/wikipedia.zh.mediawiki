'''中华人民共和国国务院总理'''，简称'''国务院总理'''、'''总理'''，是[[中华人民共和国|中华人民共和国]]的[[政府首脑|政府首脑]]，是中华人民共和国最高[[人民政府|国家行政机关]][[中华人民共和国国务院|国务院]]的首长<ref>{{cite news |title = 主席是虚职 总理行实权 |url = http://star.news.sohu.com/s2013/zhuxi/ |accessdate=2013-03-15 |website = 搜狐评论 |date=2013年3月15日 }}</ref>，是国家政务上的最高负责人<ref>{{cite news |title = 揭秘国家主席与总理职权:主席是虚职 总理行实权 |url = http://zy.takungpao.com/2013/6_1_0315_35487_2.html |accessdate = 2013-03-15 |work = 大公中原新闻网综合 |date = 2013年3月15日 |deadurl = yes |archiveurl = https://web.archive.org/web/20131102203405/http://zy.takungpao.com/2013/6_1_0315_35487_2.html |archivedate = 2013年11月2日 }}</ref>。国务院总理全面领导国务院的工作，召集和主持[[国务院常务会议|国务院常务会议]]和[[国务院全体会议|国务院全体会议]]，可以签署和公布[[中华人民共和国国务院令|国务院令]]，应当根据[[中华人民共和国宪法|宪法]]对国务院赋予的职权行事。

中华人民共和国的[[國家政體|國家政體]]是[[人民代表大会制|人民代表大会制]]，国务院即中央人民政府，直接向[[全国人民代表大会|全国人民代表大会]]及其[[全国人民代表大会常务委员会|常务委员会]]負責。国务院总理每届任期为五年，1982年后规定连续任职不得超过两届。

在[[中华人民共和国公务员|中华人民共和国公务员]]体制中，国务院总理属於行政级别最高的第一级，为[[国家级正职|国家级正职]]。历任国务院总理全部由[[中共中央政治局常委|中共中央政治局常委]]担任，是除[[中共中央主要负责人|中共党魁]]外的唯一例子。

国务院总理的前身是中央人民政府政务院总理，是从1949年到1954年期间，[[中华人民共和国中央人民政府_(1949–1954)|中华人民共和国中央人民政府]]下设[[中央人民政府政务院|政务院]]的负责人。

== 职权 ==
根据《[[中华人民共和国宪法|中华人民共和国宪法]]》规定，国务院总理有以下职权：
* 总理全面领导[[中华人民共和国国务院|国务院]]工作，总理代表[[中华人民共和国国务院|国务院]]对[[全国人大|全国人大]]及其[[全国人大常委会|常委会]]负责<ref>《中华人民共和国宪法》（2004年修订）：“总理领导国务院的工作。副总理、国务委员协助总理工作。”“国务院实行总理负责制。各部、各委员会实行部长、主任负责制。”</ref>；
* [[中华人民共和国国务院副总理|国务院副总理]]、[[国务委员|国务委员]]协助总理工作，并与[[国务院秘书长|国务院秘书长]]、[[国务院组成部门负责人列表|各部部长]]、[[国务院组成部门负责人列表|各委员会主任]]、[[中国人民银行|中国人民银行]]行长、[[中华人民共和国审计署|审计署]]审计长一起对总理负责<ref>《中华人民共和国宪法》（2004年修订）：“第九十一条　国务院设立审计机关，对国务院各部门和地方各级政府的财政收支，对国家的财政金融机构和企业事业组织的财务收支，进行审计监督。审计机关在国务院总理领导下，依照法律规定独立行使审计监督权，不受其他行政机关、社会团体和个人的干涉。”</ref>；
* 国务院工作中的重大问题，总理具有最后决策权；
* 总理有权向全国人大及其常委会提出[[中华人民共和国国务院副总理|国务院副总理]]、[[国务委员|国务委员]]、[[国务院组成部门负责人列表|各部部长]]、[[国务院组成部门负责人列表|各委员会主任]]、[[中国人民银行|中国人民银行]]行长、[[中华人民共和国审计署|审计署]]审计长、[[国务院秘书长|国务院秘书长]]的任免人选。
* [[中华人民共和国国务院|国务院]]发布的决定、命令和行政法规，向全国人大及其常委会提出的议案，任免行政人员，须由总理签署，才具法律效力。
* 总理出国访问期间，由负责常务工作的[[中华人民共和国国务院副总理|副总理]][[中华人民共和国国务院代理总理|代行总理]]职务。
* 总理召集和主持[[国务院全体会议|国务院全体会议]]。会议一般讨论重大问题或涉及众多部门的重大事项，全体会议一般每一季度或每半年召开一次。
* 总理召集和主持[[国务院常务会议|国务院常务会议]]。会议讨论[[中华人民共和国国务院|国务院]]工作中的重要事项，讨论提请[[全国人大常委会|全国人大常委会]]的议案，[[中华人民共和国国务院|国务院]]准备发布的行政法规，各部门、各地方请示国务院的重大事项等。会议一般每周召开一次。
* 总理可用自己名义发布[[国务院令|国务院令]]，用来颁布和废止[[行政法规|行政法规]]、任免[[特别行政区|特别行政区]][[行政长官|行政长官]]、解除[[戒严|戒严]]等。
* 总理根据中华人民共和国国务院组织法第四条在国务院工作中的重大问题中，必须经国务院常务会议或者国务院全体会议讨论决定，例如决定省、自治区、直辖市的范围内部分地区进入紧急状态{{NoteTag|原称为“[[戒严|戒严]]”，后由[[s:中华人民共和国宪法修正案_(2004年)|2004年宪法修正案]]修改措辞为“进入紧急状态”。}}{{NoteTag|在1989年的[[六四事件|六四事件]]中，时任国务院总理[[李鹏|李鹏]]签署《[[s:国务院关于在北京市部分地区实行戒严的命令|国务院关于在北京市部分地区实行戒严的命令]]》，宣布自1989年5月20日10时起在北京市部分地区实行[[戒严|戒严]]，该戒严令要到1990年1月11日才解除。该戒严令未经过《[[s:中华人民共和国国务院组织法|中华人民共和国国务院组织法]]》第四条规定的程序“国务院工作中的重大问题，必须经国务院常务会议或者国务院全体会议讨论决定”。还有不同理由，该戒严令不时被指为[[违宪|违宪]]的戒严。}}。

== 任免程序 ==
* 程序上，国务院总理由[[中华人民共和国主席|国家主席]]提名，交由[[全国人民代表大会|全国人民代表大会]]全体投票通过，再由国家主席根据全国人大的决定任命。
* 只有全国人民代表大会全体会议有权力罢免国务院总理<ref>《中华人民共和国宪法》（2004年修订）：“第六十三条　全国人民代表大会有权罢免下列人员：（二）国务院总理、副总理、国务委员、各部部长、各委员会主任、审计长、秘书长；”</ref>。国家主席根据全国人民代表大会的决定，免去总理的职务。<ref>《中华人民共和国宪法》（2004年修订）：“第八十条　中华人民共和国主席根据全国人民代表大会的决定和全国人民代表大会常务委员会的决定，公布法律，任免国务院总理、副总理、国务委员、各部部长、各委员会主任、审计长、秘书长，……”</ref>
* [[文化大革命|文化大革命]]期间的《七五宪法》和《七八宪法》[[中华人民共和国主席存废之争|废除了国家主席的职位]]，改由[[中国共产党中央委员会主席|中国共产党中央委员会主席]]提名总理的人选。

== 党内排名 ==
虽然在第九至十一届国务院（1998年－2013年），[[朱镕基|朱镕基]]和[[温家宝|温家宝]]两位总理因为资历关系，在[[中共中央政治局常委|中共中央政治局常委]]排名中位于[[全国人大常委会委员长|全国人大常委会委员长]]之后{{NoteTag|李鹏于1985年进入政治局，1987年成为政治局常委，乔石也在1985年进入政治局，1987年成为政治局常委，但李鹏1981年3月升任正部级职务（电力工业部部长），乔石升正部级是在1982年4月（中共中央对外联络部部长），所以总理李鹏在委员长乔石前面。朱镕基1992年10月才进入政治局及常委会，资历比李鹏晚很多，所以他当总理时要排在当委员长的李鹏后面。吴邦国和温家宝同在1992年10月进入政治局，不过当时吴邦国是正式的政治局委员，温家宝只是政治局候补委员，所以吴邦国的排名要在温家宝前面。现任总理李克强排名在全国人大常委会委员长张德江前面，因为李克强比张德江早进入中央政治局常委会。}}，但作为政府的一把手，往往在[[中华人民共和国政治|中国政治]]中扮演更重要的角色。到了[[李克强|李克强]]担任国务院总理后，排名重新升至第二位，恢复[[李鹏|李鹏]]时期的排名。
