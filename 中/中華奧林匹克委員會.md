'''中華奧林匹克委員會'''（{{Lang-en|Chinese Taipei Olympic Committee}}），簡稱'''中華奧會'''，是[[中華民國|中華民國]]的[[國家奧林匹克委員會|國家奧林匹克委員會]]，負責處理中華民國在[[奧運|奧運]]、[[亞運|亞運]]等國際體育賽事的聯繫工作，並加強與[[國際奧會|國際奧會]]及國內外各運動團體聯繫，維護奧林匹克之名稱、標誌及會徽。中華奧林匹克委員會參加奧運及國際體育賽事時，使用「[[中華臺北|中華臺北]]」做為參賽代表團的名稱。

== 歷史 ==
中華奧林匹克委員會的前身為[[中國奧林匹克委員會_(中華民國)|中國奧林匹克委員會]]，其實體組織為[[中華全國體育協進會|中華全國體育協進會]]，成立於1922年。

1973年時因為奧運席位問題，中華全國體育協進會於7月17日分立出中華奧林匹克委員會以專司國際運動或比賽，原中華全國體育協進會則改稱中華民國體育協進會，即現今的[[中華民國體育運動總會|中華民國體育運動總會]]，專司國內運動或比賽。

== 會旗（梅花旗） ==
{{see also|梅花旗}}
*[[梅|梅花]]外環：[[中華民國國花|國花]]，象徵梅花凌冬耐寒，其所表現的堅貞剛潔；表現出[[中華民國|中華民國]]選手在艱苦的訓練過程中，出現佳績。
*外環顏色：從外到內的，藍、白、紅色，代表[[中華民國國旗|國旗]]。
*[[青天白日|青天白日]]徽：代表中華民國。須特別注意的是，依據1981年[[中華奧會|中華奧會]]與[[國際奧委會|國際奧委會]]所簽定的洛桑協議，國際奧委會承認的會徽中所使用的青天白日徽，白日的比例並非[[中華民國國徽|中華民國國徽]]標準，亦不是[[中國國民黨|中國國民黨]]黨徽標準，而是介於兩徽之間<ref>[http://www.tpenoc.net/center_b04_01.jsp 中華奧會會徽說明（中華奧會官網）] {{webarchive|url=https://web.archive.org/web/20160725164756/http://www.tpenoc.net/center_b04_01.jsp |date=2016-07-25 }}</ref><ref>[https://video.udn.com/news/543093 中華奧會會旗釋疑 設計者：不是黨徽 | 時事 | 聯合影音]</ref>。

*[[奧林匹克五環|奧林匹克五環]]：代表[[國際|國際]][[奧林匹克運動會|奧林匹克運動會]]。
**注釋：[[中華民國|中華民國]]大專院校、高中體育總會旗幟與標誌，都使用此面旗幟，差別除了在於五色環換成其他標誌，還有青天白日徽是使用中華民國國徽標準比例。

中華民國政府簽署奧運模式協議時，並未同意觀眾席場內禁用中華民國國旗，各國亦無明文禁止，但各項國際運動會實務上迫於[[中國大陸|中國大陸]]壓力仍經常禁止觀眾攜中華民國國旗入奧運會場加油，特別是同場有中國大陸運動員比賽時，依舊經常遭搶奪或驅趕<ref>[http://www.ettoday.net/news/20120802/82259.htm 倫敦奧運／不能用國旗？體委會：禁帶非參賽國旗幟入場]</ref><ref>[http://tw.news.yahoo.com/僑胞攜國旗加油被搶-英方致歉-054328412.html 僑胞攜國旗加油被搶 英方致歉]</ref><ref>[http://tw.news.yahoo.com/%E7%82%BA%E6%9B%BE%E6%AB%9F%E9%A8%81%E5%8A%A0%E6%B2%B9-%E7%95%99%E5%AD%B8%E7%94%9F%E5%B8%B6%E5%9C%8B%E6%97%97%E7%AA%81%E5%9C%8D-063102360.html?_esi=1 為曾櫟騁加油 留學生帶國旗突圍]</ref><ref>[http://www.appledaily.com.tw/realtimenews/article/new/20160810/926003/ 愛台灣老外國旗裝觀賽　遭奧會「強迫脫衣」]</ref>。

中華臺北於1981年3月23日《[[洛桑協議|洛桑協議]]》時和國際奧會簽約同意會名、會旗及會徽。1983年6月1日中華奧林匹克委員會提交的「會歌」（{{lang|en|anthem}}）被國際奧會接受<ref>{{Cite web |url = http://www.la84foundation.org/OlympicInformationCenter/OlympicReview/1983/ore190/ORE190zb.pdf |title="Chinese Taipei" and Olympism |accessdate=2010-07-10 |year=1983 |format=PDF |work=Olympic Review |publisher=國際奧林匹克委員會 |pages=593-594 |language=En}}</ref>。
