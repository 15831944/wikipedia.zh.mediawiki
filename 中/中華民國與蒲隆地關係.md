===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|蒲隆地簽證政策|Visa policy of Burundi}}|{{link-en|蒲隆地公民簽證要求|Visa requirements for Burundian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]與[[黃熱病|黃熱病]][[疫苗接種|疫苗接種]]證明的[[中華民國國民|中華民國公民]]可委託當地人、[[關係人|關係人]]或[[旅行社|旅行社]]，向該國航空、邊境暨外事警察局（Police de l′Air, des Frontieres et des Etrangers，PAFE）申請入境許可（Autorisation d′Entree），持許可在抵達該國唯一的[[布松布拉國際機場|布松布拉國際機場]]時繳交簽證費，停留最多30天，可延長。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-177-5d187-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|蒲隆地護照|Burundian passport}}的{{link-en|蒲隆地人口|Demographics of Burundi|蒲隆地公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將蒲隆地歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2017年4月27日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-357-3e5b1-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將蒲隆地歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=2103546156B815AD |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|蒲隆地經濟|Economy of Burundi}}}}
===貿易===
2017年，蒲隆地是中華民國第215大貿易夥伴、第181大[[進口|進口]]夥伴、第217大[[出口贸易|出口]]夥伴，在[[非洲國家和地區列表|非洲國家]]排名倒數第5、11、3；在[[世界政區索引|世界經濟體]]倒數第24、56、14。出口至蒲隆地的金額為3萬1,744[[美元|美元]]，年減12.500%；自蒲隆地進口的金額為16萬4,467美元，年減7.081%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）13萬2,723美元，年減5.684%。※2016年，蒲隆地是中華民國第215大貿易夥伴、第182大進口夥伴、第219大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
