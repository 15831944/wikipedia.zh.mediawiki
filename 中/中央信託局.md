'''中央信託局'''（簡稱'''中信局'''），原係[[中華民國|中華民國]][[金融|金融]]體系重要機構之一，已經被[[臺灣銀行|臺灣銀行]]合併。

== 沿革 ==
* 1934年8月 為了因應[[對日抗戰|對日抗戰]]緊急需要，[[國民政府|國民政府]]訓令[[中央銀行_(中華民國)|中央銀行]]設立「中央信託局籌備處」負責籌備創立中央信託局，同時命令央行全部撥充中央信託局成立所需[[資本總額|資本總額]][[法幣|國幣]]一千萬元。
* 1935年7月29日 中央銀行[[理事會|理事會]]第88次會議通過《中央信託局章程》，並呈奉國民政府令准備案。
* 1935年10月1日 中央信託局正式成立，'''定位為中央銀行之附設機構'''，總局設於[[上海|上海]]；當時的中央信託局與央行關係密切，被合稱為「行局一家」。
* 1942年6月 《中央信託局章程》修正，其第一條明定：「中央銀行遵照國民政府訓令，特設中央信託局經營[[信託|信託]]業務。」
* 1947年5月7日 國民政府公布《中央信託局條例》，全文24條；其第一條明定：「國民政府為執行國策，辦理特種信託[[保險|保險]][[儲蓄|儲蓄]]業務，設中央信託局，'''受[[中華民國財政部|財政部]]之監督'''，依本條例規定辦理之。」

* 1949年 國民政府遷往[[台灣|台灣]][[台北市|台北市]]，中央信託局總局隨行。
* 1950年6月9日 [[行政院|行政院]]頒布〈調整國營事業機構方案〉，規定中央信託局業務限於購料、易貨、儲運與保險。
* 1951年 政府核准中央信託局恢復信託業務，中央信託局陸續奉令辦理[[軍人保險|軍人保險]]、[[再保險|再保險]]、輸出保險及[[公務人員保險|公務人員保險]]等政策性業務。

1983年11月11日修正公布的《中央信託局條例》第一條規定：「為執行政府政策，辦理採購、貿易、保險、銀行、信託、儲運及其他經中央主管機關指定之特種業務，設中央信託局，受財政部之監督。」該局配合台灣[[經濟|經濟]]建設，協助政府機關及[[公營事業|公營事業]]機構採購物資，以及代辦重大建設所需設備，並協助政府辦理經濟[[外交|外交]]事宜。

* 1962年2月 台北市[[城中區_(台北市)|城中區]]武昌街一段49號中央信託局大樓落成。
* 1972年2月 [[中國產物保險公司|中國產物保險公司]]（[[兆豐產物保險|兆豐產物保險]]前身）合併'''中央信託局產物保險處'''。
* 1983年3月 中央信託局人壽保險業務以「'''中央人壽'''」為商標名稱。
* 2003年7月1日 中央信託局配合[[中華民國政府|中華民國政府]]「[[台灣金融改革|金融改革]]」政策，依據《[[公司法|公司法]]》向[[經濟部商業司|經濟部商業司]]辦妥公司登記，改制為[[股份有限公司|股份有限公司]]，全稱為「'''中央信託局股份有限公司'''」，[[英文|英文]]全稱維持不變。

中央信託局最後一任[[董事長|董事長]]為[[許嘉棟|許嘉棟]]，許嘉棟曾任央行副總裁與[[國立臺灣大學|國立臺灣大學]]經濟學系[[教授|教授]]，足見該局與央行仍是關係密切，不可以單純之[[商業銀行|商業銀行]]視之。

* 2007年 政府推動[[公股|公股]]金融機構整併，考量中央信託局與[[臺灣銀行|臺灣銀行]]業務具互補性、可[[多元化|多元化]]發展，經[[中華民國財政部|財政部]]報奉[[行政院|行政院]]核定中央信託局與臺灣銀行合併。
* 2007年7月1日 臺灣銀行以「吸收合併」方式整併中央信託局，'''臺灣銀行為存續公司，中央信託局為消滅公司'''。中央信託局各分局改為臺灣銀行各分行，中央信託局人壽保險處改-{制}-為「臺灣銀行股份有限公司人壽保險部」。
* 2008年1月1日，[[臺灣金融控股股份有限公司|臺灣金融控股股份有限公司]]成立，1月2日臺灣銀行人壽保險部改-{制}-為臺灣金控[[子公司|子公司]]「[[臺銀人壽保險股份有限公司|臺銀人壽保險股份有限公司]]」。

== 組織架構 ==
;局本部
:'''國內銀行處'''+'''外匯處'''：併入臺灣銀行後，在原址合併成立「'''臺灣銀行武昌分行'''」。
:'''貿易處'''：併入臺灣銀行後，改名為「'''臺灣銀行貿易部'''」，並接續辦理及推展'''黃金存摺'''業務，2009年11月16日改制為「'''臺灣銀行貴金屬部'''」。
:'''信託處'''：與臺灣銀行合併後，併入原「'''臺灣銀行信託部'''」，部址並從臺北市[[重慶南路_(台北市)|重慶南路]]一段58號搬遷於此。

; '''購料處'''
:併入臺灣銀行後，改名為「'''臺灣銀行採購部'''」，並接續代辦政府機關、公立學校及公共事業集中採購制度之規劃、推行等事項。

; '''公務人員保險處'''
:位於'''中央信託局公保信義大樓'''（今[[衛生福利部中央健康保險署|衛生福利部中央健康保險署]]大樓）。併入臺灣銀行後，改名為「'''臺灣銀行公教保險部'''」，辦公室仍設於該大樓。

; '''人壽保險處'''
:併入臺灣銀行後，改名為「'''臺灣銀行人壽保險部'''」，2008年1月2日改制為「'''[[臺銀人壽|臺銀人壽]]'''」。
