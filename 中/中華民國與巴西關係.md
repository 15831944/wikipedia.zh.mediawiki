'''中華民國與巴西關係'''是指[[中華民國|中華民國]]與[[巴西聯邦共和國|巴西聯邦共和國]]之間的關係。1913－1974年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[巴西|巴西]]。]]
{{see also|中華民國外交|巴西外交}}
{{see also|中国驻巴西大使列表{{!}}中華民國駐巴西合眾共和國／巴西合眾國／巴西聯邦共和國外交官列表|巴西驻华大使列表{{!}}巴西駐中華民國外交官列表}}
1913年4月8日，巴西是第一個[[外交#外交上的承認|承認]]並與中華民國建交的國家。<ref name=Immigration>{{cite|url=http://www.psc.ntu.edu.tw/outline/e_paper/pop46/%E6%B9%AF%E7%86%99%E5%8B%87.pdf|title=巴西招徠臺灣人移民─1960年代我國政府的態度與人民的反應|publisher=[[國立臺灣大學|國立臺灣大學]]|date=2013-06|format=PDF|deadurl=yes|archiveurl=https://web.archive.org/web/20150414030855/http://www.psc.ntu.edu.tw/outline/e_paper/pop46/%E6%B9%AF%E7%86%99%E5%8B%87.pdf|archivedate=2015-04-14}}</ref><ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/11.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>於原首都[[里約熱內盧|里約熱內盧]]設立'''中華民國駐{{link-en|巴西合眾共和國|First Brazilian Republic}}[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1943年5月4日，公使館升格為'''中華民國駐[[巴西合眾國|巴西合眾國]]／巴西聯邦共和國[[外交代表機構#大使馆|大使館]]'''，改派駐[[特命全權大使|大使]]。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國仍維持邦交。

1960年，將大使館由里約熱內盧遷至新首都[[巴西利亞|巴西利亞]]。

1971年10月25日，巴西在[[聯合國大會|聯合國大會]]投票支持[[中國與聯合國#代表權問題|中華民國續留聯合國]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，但在該案表決前，中華民國已宣布退出聯合國。

1974年8月15日，巴西與[[中華人民共和國|中華人民共和國]]建交，<ref>[http://www.gov.cn/test/2012-04/20/content_2117961.htm 中华人民共和国与各国建立外交关系日期简表]</ref>並根據「[[一個中國|一個中國]]」原則與中華民國[[外交#外交上的承認|斷交]]，<ref name="外交年鑑"/>9月15日關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

1991年3月31日，巴西國有企業民營化計畫指導委員會主席莫迪亞諾抵台訪問。期間會見中華民國外交部常務次長[[程建人|程建人]]、中華民國全國工業總會理事長[[許勝發|許勝發]]等人士，並舉辦「巴西[[國有企業|國有企業]][[民營化|民營化]]」說明會。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=815BEC0AC260E9B4 |author = 公眾外交協調會 |date = 1991年3月30日 |title = 巴西聯邦共和國國營企業民營化計畫指導委員會主席莫迪亞諾訂於本（三）月卅一日抵華訪問三天 |publisher = 中華民國外交部}}</ref>

2018年3月，[[2018年巴西大選|巴西總統候選人]]、聯邦眾議員[[雅伊爾·博索納羅|波索納洛]]抵台訪問，是巴西與中華民國斷交後首位訪問台灣的巴西總統候選人。當時中華人民共和國駐巴西大使館發函譴責波索納洛訪台「冒犯[[中华人民共和国疆域|中國主權和領土]]完整」。10月28日，波索納洛當選[[巴西總統|巴西總統]]。<ref>{{Cite web |url = https://udn.com/news/story/6656/3448810 |author = 中央社 |date = 2018年10月29日 |title = 巴西總統當選人波索納洛曾訪台 外交部：有助合作交流 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref>

兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]申請巴西簽證的所有文件必須掃描並上傳至線上申請表格，完成後，申請人仍須攜帶紙本文件至駐[[台北市|台北]]的{{Link-en|巴西商務辦事處|Commercial Office of Brazil to Taipei}}窗口遞件。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-93-4f28f-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref>無論持觀光或商務簽證入境巴西，均禁止從事有償或無償的工作。<ref name="投資環境簡介"/>

持有{{link-en|巴西護照|Brazilian passport}}的[[巴西人|巴西公民]]抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將巴西歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2018年1月26日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-21-1b16e-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將巴西歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=0A30CEDBFFAB4F8F |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於巴西第1大城[[聖保羅_(巴西)|聖保羅]]設立'''巴西台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#S-AMERICA |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]則在首都[[巴西利亞|巴西利亞]]設立'''駐巴西代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=215&pid=305863 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2018年，巴西是中華民國第26大貿易夥伴、第27大[[進口|進口]]夥伴、第23大[[出口贸易|出口]]夥伴，在[[美洲|美洲]]經濟體排名第4、3、4。出口至巴西的金額為13億4,564萬1,684[[美元|美元]]，年增6.342%；自巴西進口的金額為16億4,227萬6,135美元，年減36.790%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）2億9,663萬4,451美元，年減77.742%。2017年，巴西是中華民國第22大貿易夥伴、第20大進口夥伴、第24大出口夥伴，在美洲經濟體排名第2、2、4。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資===
根據[[巴西中央銀行|巴西中央銀行]]統計，[[臺灣商人|臺商]]在巴西的投資金額，2007年為942萬美元、2008年為1,921萬美元、2009年為1,508萬美元、2010年為1億1,298萬美元、2011年為5,200萬美元、2012年為5,500萬美元；另根據經濟部投資審議委員會統計，1952年－2013年，臺商對巴西的總投資金額為2億6,947萬美元。主要投資行業為[[資訊及通訊科技|資通訊產品]]、[[金屬加工|金屬加工]]、[[批發|批發]]及[[零售|零售]]業、[[飯店|旅館]]及[[飲食|餐飲]]業、[[房地產|房地產]]及[[国际贸易|進出口]]業。<ref name="投資環境簡介"/>巴西對中華民國的總投資金額為2,203萬美元。<ref name="國家基本資料表">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 國家基本資料表 |publisher = 中華民國對外貿易發展協會}}</ref>

近年來[[電子工業|電子工業]]的蓬勃發展，有礙於巴西高額的[[電子產品|電子產品]][[关税|進口稅]]，臺灣的電子廠商與[[電子專業製造服務|電子代工]]企業紛紛前往巴西設立據點以降低成本。前往巴西設廠或設立[[子公司|子]]、[[分公司|分公司]]的大型企業有：[[鴻海科技集團|鴻海科技集團]]、[[宏碁|宏碁]]、[[華碩電腦|華碩電腦]]、[[仁寶電腦|仁寶電腦]]、[[精英電腦|精英電腦]]、[[微星科技|微星科技]]、[[技嘉科技|技嘉科技]]、[[友訊科技|友訊科技]]、[[研華科技|研華科技]]等。<ref>[http://www.trade.gov.tw/World/ListArea.aspx?code=7020&nodeID=977&areaID=1&country=b645be06KW_&pw=3 中華民國經濟部 國際貿易局 國家檔案：巴西]</ref><ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/br/pdf/103-42Brazil.pdf |date = 2014年12月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

===僑民===
{{see also|臺灣裔{{!}}海外臺僑|巴西華人}}
[[臺灣人|臺灣人]]大規模移民巴西可以追溯到1960年代，巴西政府為了平衡在巴西的[[亞洲人|亞洲人]]勢力（當時以[[日本人|日本人]]為最多）因而向[[臺灣|臺灣]]與[[南韓|南韓]]招募移民。起初，不同於臺灣人移民[[北美|北美]]、[[大洋洲|紐澳]]是以學術、[[第三產業|服務業]]為主。臺灣人移民巴西是以[[第一產業|農業]]為多，且以[[高雄縣|高雄縣]][[六龜區|六龜]]、[[美濃區|美濃]]；[[屏東縣|屏東縣]][[內埔鄉|內埔]]、[[高樹鄉|高樹]]等鄉鎮為眾多。但[[中華民國政府|中華民國政府]]對於國民移民巴西的看法抱持著不支持的態度，因而限制國民移居巴西。但多數人向政府報名以[[農業技術|農業技術]]團的方式前往巴西，與巴西政府合約期滿時便申請並取得[[永久居留權|永久居留許可]]。<ref name = Immigration/>至第2代移民多已融入當地主流社會。<ref name="94年外交年鑑"/>

現今在巴西的臺灣裔族群多居住在巴西東南部的[[聖保羅_(巴西)|聖保羅]]以及其郊區約佔半數，其餘多分布在[[南大河州|南大河州]]、[[巴拉那州|巴拉那州]]及[[里約熱內盧|里約熱內盧]]。位於巴西東北的[[勒西菲|勒西菲]]是另一臺灣裔重點聚落，於1960年代大批高雄美濃居民移居至此經營農場，人數約數百名，現今含括其後裔則為千餘名。<ref name = Immigration/>臺灣裔後裔多事業有成，如聖保羅近郊之[[摩基達斯克魯易斯市|摩基達斯克魯易斯市]]，當地約有百戶臺灣僑民設立[[雙孢蘑菇|洋菇]]栽培農場，在巴西的洋菇[[市場佔有率|市場佔有率]]達到9成。<ref>[http://mag.chinatimes.com/mag-cnt.aspx?artid=5581 中國時報：台商在巴西 創造蘑菇傳奇]</ref>

2001年－2004年，中華民國與巴西出現因[[涉外婚姻|跨國婚姻]]而衍生的[[子女撫養權|子女監護權]]問題——[[吳憶樺|吳憶樺監護權之爭]]。

2005年，[[行政院客家委員會|行政院客家委員會]]主任委員[[羅文嘉|羅文嘉]]前往聖保羅出席[[客家|客家]]活動中心落成典禮及[[美洲|全美洲]]客家懇親會；[[僑務委員會|僑務委員會]]籌建的聖保羅[[華僑|華僑]]文教中心亦竣工啟用。<ref name="94年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2005/WebPages/MICO/08/08_01_06.htm |title = 《中華民國94年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

===學術===
{{see also|臺灣教育|{{link-en|巴西教育|Education in Brazil}}}}
[[中華民國外交部|中華民國外交部]]設立臺灣[[獎學金|獎學金]]、[[中華民國教育部|教育部]]設立[[中華民國國語|華語文]]獎學金、[[財團法人國際合作發展基金會|財團法人國際合作發展基金會]]（國合會）則設立高等人力培訓獎學金，提供巴西學生赴臺研習；外交部另提供臺灣獎助金給予巴西學者赴台從事[[漢學|漢學]]或臺灣研究。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>
