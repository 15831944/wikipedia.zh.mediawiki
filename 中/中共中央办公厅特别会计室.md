'''中共中央办公厅特别会计室'''，是[[中共中央办公厅|中共中央办公厅]]内设机构，负责特别会计工作。

==沿革==
[[抗日战争|抗日战争]]期间的1941年，由于受到[[中国国民党|中国国民党]]的严密封锁，[[中共中央|中共中央]]在经济内外交困的情况下，号召各单位建立自己的“家务”。于是各单位开荒种粮、纺纱织布，并抽出资金和人力到[[国统区|国统区]]乃至[[英属香港|香港]]做生意，从而为党积累了不少资金。1941年9月，为统一管理这些特别经费，中共中央决定设立'''[[中共中央书记处|中共中央书记处]]特别会计科'''（以下简称特会科），并且明确特会科主要负责管理中共地下党的活动经费、援外经费及中央会议所需全部经费。当时特会科的主要资金来源包括：（1）党的秘密企业（党员企业）所赚的钱；（2）各中央局上缴的黄金（因当时各[[解放区|解放区]]货币不统一，故只能上缴黄金）；（3）作战的战利品；（4）中央及各中央局统一收取的党费；（5）抗日战争时期海外爱国人士及华侨捐助的款项。特会科主任为[[赖祖烈|赖祖烈]]，直属[[任弼时|任弼时]]领导<ref name=wangyh>王彦红，揭秘战争年代党的特殊经费，四川党的建设(城市版)2012年07期</ref>。

1948年3月，[[中共中央前方委员会|中央前委]]、[[中共中央后方委员会|中央后委]]机关开始向[[河北省|河北省]][[平山县|平山县]][[西柏坡|西柏坡]]转移。此次转移穿越[[陕西省|陕西省]]、[[山西省|山西省]]、[[河北省|河北省]]的二十余个县。中央书记处特会科抵达西柏坡后，被安排到东、西柏坡两村口之间的一个大土棚子内，当时党的特别经费及人员都在该大棚子里。贵重物品都存在三眼[[窑洞|窑洞]]里，其中最东边一眼窑洞存放古董如宝石、象牙等，中间一眼窑洞放置一台保险柜，内存重要钱款和账本，西边一眼窑洞存放从陕北运来的黄金。这些党的特别经费在关键时刻曾起过重要作用。例如1946年6月[[李先念|李先念]]领导的[[新四军|新四军]]第五师6万余人被国民党30余万军队围困在鄂东[[宣化店|宣化店]]，中央领导指示特会科迅速筹备[[法币|法币]]、[[美元|美元]]，后来用飞机空投到五师驻地，解除了五师的经济困境。1946年夏，[[国民政府|国民政府]]还都[[南京|南京]]，[[八路军驻重庆办事处|八路军驻重庆办事处]]也随之迁至南京，并在[[上海|上海]]设办事处，在[[北平|北平]]、[[天津|天津]]、上海创办报纸，这些都使用了党的特别经费。[[毛泽东|毛泽东]]指示，经费开支要节省，一般情况下不能动用<ref name=wangyh/>。

1949年1月初的[[中共中央政治局|中共中央政治局]]会议上，毛泽东明确提出：党的特别经费，以不搞为好。中共中央决定，党不再搞贸易和商业活动。1949年3月，中共中央和[[中国人民解放军|中国人民解放军]]总部从西柏坡迁往北平。为确保党的特别经费的安全，特会科未随中央一同进京，而是在1949年3月底才离开西柏坡赴北平，当时动用了七辆卡车。党的特别经费全部放进[[中国人民银行|中国人民银行]]地下库房。进京后，党的特别经费就此完成任务，并分别处理：例如[[邓洁|邓洁]]在北平经营的一批商店、电影院及计划在[[密云|密云]]开挖的金矿，全部移交北平市；[[赖祖烈|赖祖烈]]在北平做生意留下的金条银元，及几方存在[[东交民巷|东交民巷]][[中国银行|中国银行]]保险库的古印，折价出售给中国人民银行，钱存在特会科；在香港的[[华润（集团）有限公司|华润公司]]、[[華潤五豐|五丰商行]]等，因香港仍为[[英国|英国]]控制，所以只能交给[[中共中央香港工作委员会|中共香港工委]]管理。经过上述处理，党的特别经费中凡与商贸有关的部分均脱钩。特会科仅留下折价出售所得的款项、党员上缴的党费和毛泽东的稿费<ref name=wangyh/>。

随后，特会科编入中共中央办公厅（简称中办），改为'''中共中央办公厅特别会计室'''，由[[杨尚昆|杨尚昆]]直接领导<ref name=wangyh/>。一直到“[[文化大革命|文化大革命]]”前，中办一直设有特别会计室<ref name=wg/>。特别会计室负责管理中共中央特别经费和协调使用全国各地党组织的经费，直接受[[中共中央政治局常委|中共中央政治局常委]]的领导。[[中共中央主席|中共中央主席]]毛泽东的个人稿费也由特别会计室存入银行保管办理<ref name=feng/>。该稿费毛泽东主要用于买书、办学、搞调查研究、资助身边工作人员、给[[民主党派|民主党派]]及无党派人士请客送礼、退赔身边工作人员及妻子[[江青|江青]]向地方单位索要的物品、为前妻[[贺子珍|贺子珍]]买药等等。每年毛泽东自己动用的稿费不到1万元。晚年他曾从稿费里批给江青3万元，1975年批给贺子珍2万元、[[毛岸青|毛岸青]]、[[李讷|李讷]]各8000元。1976年9月[[毛泽东之死|毛泽东逝世]]后，稿费多年未动，直到1981年中办找到[[李敏_(毛娇娇)|李敏]]，才从稿费中给她8000元以及一台[[电冰箱|电冰箱]]、一台[[彩色电视机|彩电]]，同时也给李讷一台电冰箱、一台彩电，余下稿费仍由特别会计室保管和掌握<ref>邬家能，毛泽东的稿费开支，党史纵览2004(6):10-11</ref>。一些中央领导的存款也放在特别会计室，如[[朱德|朱德]]便将发工资以来的存款存放在特别会计室，1977年1月15日其遗孀[[康克清|康克清]]致信中办主任[[汪东兴|汪东兴]]，遵其遗嘱将这笔存款20306.16元全部上交党组织，特别会计室不久给康克清寄来收据<ref>{{Cite web |url =  http://news.sohu.com/20091201/n268592151.shtml|title = 朱德之孙朱和平：爷爷临终的嘱托 |author =  |date = 2009-12-01 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

文革爆发后，中办的机构被搞乱，工作受破坏。1969年5月以后，中办特别会计室的业务划归中央办公厅管理处财务组<ref name=wg>{{Cite web |url =http://www.hprc.org.cn/gsw/detail_zzslk.jsp?channelid=75034&record=1875  |title =  中国共产党组织史资料 第六卷 第一编　中国共产党中央组织机构 第二章　中共中央工作机构|author =  |date = 2017-11-01 |publisher = 中华人民共和国国史网 |language =  |accessdate =  |quote =  }}</ref>。1976年10月时，中办下设12个处室（部、馆、校）。1988年，中办机构改革，调整后设调研室、秘书局、警卫局、机要局、机要交通局、中直管理局、中央档案馆、毛主席纪念堂管理局、特别会计室、老干部局、人事局、机关党委办公室12个局级机构。1993年12月，中办机构改革，调整后设一个副部级机构（中央档案馆暨国家档案局），10个职能局：调研室、秘书局（其中法规室是副局级机构，设在秘书局）、警卫局、机要局、机要交通局、中直管理局、特别会计室、老干部局、人事局和机关党委<ref name=qiliu>{{Cite web |url = http://www.hprc.org.cn/gsw/detail_zzslk.jsp?channelid=75034&record=1978 |title = 中国共产党组织史资料 第七卷 第一编　中国共产党中央组织机构 第二章　中共中央工作机构 |author =  |date =  |publisher =  中华人民共和国国史网|language =  |accessdate =2017-11-01  |quote =  }}</ref>。
