'''中国人民解放军海军'''是[[中国共产党|中国共产党]]及其领导的[[中華人民共和國|中華人民共和國]]的[[海军|海上武裝力量]]，是[[中国人民解放军|中国人民解放军]]的五大[[中国人民解放军军兵种列表|军兵种]]之一。解放军海军成立于1949年，先后参与了[[第二次国共内战|第二次国共内战]]、[[西沙海战|西沙海战]]、[[赤瓜礁海战|赤瓜礁海战]]等多次军事行动。

中国人民解放军海军自建立后数十年间主要负责近海防御工作，进入21世纪后发展迅速，寻求建构一支近海防御与远海护卫能力兼备的[[藍水海軍|藍水海軍]]。<ref>{{cite web |url =http://www.81.cn/jwzl/2015-11/26/content_6786823.htm|title = 军事力量建设发展 |publisher = 中国军网 |date =  |accessdate =  }}</ref>海军现役军人约29万人，舰船300余艘，包括1艘[[航母|航母]]、33艘[[驅逐艦|驅逐艦]]、95艘[[護衛艦|護衛艦]]、18艘[[核潜艇|核潛艇]]和58艘[[潜艇|常規攻擊型潛艇]]，其中[[中国人民解放军海军航空兵|海军航空兵]]拥有飞机600余架。解放军海军现役舰艇总吨位位居世界第二，是亚洲及西太平洋地区最大规模的海上武装力量，也被一些媒體認為是世界上仅次于[[美国海军|美国海军]]的综合实力第二强的海上军事力量<ref name="海军A">{{cite web |author = 大陸中心 |title=中國航母的「帶刀侍衛」　張軍社：055領先美勃克級艦 |url = http://www.ettoday.net/news/20170630/956023.htm |website=東森新聞雲 |accessdate=2017-07-02 |language=zh-Hans |date=2017-06-30 |quote=美國《國家利益》雜誌本月初列出世界5大海軍，將中國排在第2名，排在第一位的是美國，排在中國後面的分別是俄羅斯、英國、日本。文章指出，中國海軍在過去25年裡取得長足進展，中國海軍目前擁有1艘航母、6艘兩棲運輸艦、32艘驅逐艦、94艘護衛艦、18艘核潛艇和大約58艘常規攻擊型潛艇。}}</ref><ref>{{cite web |author = |title=The Five Most-Powerful Navies on the Planet|url = https://nationalinterest.org/feature/the-five-most-powerful-navies-the-planet-10610 |website= |accessdate=|language= |date=|quote=}}</ref><ref>{{cite web |author = |title=Top 10 Countries with Best Navy Force in the World|url = https://improb.com/best-military-navy-force/|website= |accessdate=|language= |date=|quote=}}</ref>。

== 历史 ==
=== 创立 ===
[[File:China_10th_Anniversary_Parade_in_Beijing_03.jpg|缩略图]]
中国人民解放军海军是在[[中国人民解放军陆军|中国人民解放军陆军]]一部和[[中华民国海军|中华民国海军]]舰队及人员的基础上组建起来的。

[[抗日战争|抗日战争]]时期，1944年驻[[威海|威海]][[刘公岛|刘公岛]]的[[汪精卫政权|汪精卫政权]]海军练兵营的卫队长[[郑道济|郑道济]]率600多人宣布起义，隨後加入[[八路军|八路军]]，1944年11月改编为[[胶东军区海军支队|胶东军区海军支队]]，成为[[中国共产党|中国共产党]]领导的第一支海上武装力量。消息传到[[延安|延安]]，[[毛泽东|毛泽东]]指示[[叶剑英|叶剑英]]成立了中共军队历史上第一个海军研究小组<ref name=qiangda>{{Cite web |url = http://mil.news.sina.com.cn/history/2016-12-26/doc-ifxyxusa5512319.shtml |title = 毛泽东心系海军建设:我们一定要建立强大的海军 |author =  |date =2016-12-26  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

1949年1月8日，在[[西柏坡|西柏坡]]召开的[[中共中央政治局|中共中央政治局]]会议上讨论并通过《目前形势和党在1949年的任务》的决议，其中指出：“1949年及1950年我们应当争取组成一支能够使用的空军及一支能够保卫沿海、沿江的海军。”此后，毛泽东多次强调建设海军的重要性，还指示[[中国人民解放军第三野战军|中国人民解放军第三野战军]]前敌委员会，在发起[[渡江战役|渡江战役]]前后，尽快组建华东军区海军<ref name=qiangda/>。

1949年3月4日，宣布起义的中华民国海军[[重庆号巡洋舰|“重庆”号巡洋舰]]舰长[[邓兆祥|邓兆祥]]率部抵达[[葫芦岛|葫芦岛]]港。3月24日，进军[[北平|北平]]的[[中共中央|中共中央]]机关途经[[河北省|河北省]][[涿县|涿县]]时，毛泽东、[[朱德|朱德]]联名签发了致“重庆”号巡洋舰舰长邓兆祥及全体官兵的嘉勉电。同时，毛泽东、朱德决定立即组建海军。4月23日，中华民国海防第二舰队司令[[林遵|林遵]]宣布起义，加入中国人民解放军<ref name=zhutuo/>；所部25艘舰艇、1200余名官兵中，有18艘艦艇拒絕加入，向長江下游突圍，其中11艘成功至上海，重回中華民國海軍陣營<ref>{{Cite web |url =https://udn.com/news/story/10930/3744490  |title = 黑歷史解禁！中共海軍70年 源自國軍一場喋血大叛變 |author = 程嘉文 |date =2019-04-09  |publisher =聯合報  |language =  |accessdate =2019-04-09  |quote =  }}</ref>。

1949年4月4日，[[中国人民解放军第三野战军|中国人民解放军第三野战军]]副司令员[[粟裕|粟裕]]、参谋长[[張震_(上將)|张震]]奉[[中共中央军委|中共中央军委]]命令，在[[江苏省|江苏省]][[泰县_(江苏县份)|泰县]][[白马镇_(泰州市)|白马庙乡]]（今位于[[江苏省|江苏省]][[泰州市|泰州市]][[高港区|高港区]]）建立[[渡江战役|渡江战役]]指挥部，接收中华民国海军起义舰艇以组建一支保卫沿海沿江的海军部队。1949年4月23日，华东军区海军领导机构在白马庙乡成立，[[张爱萍|张爱萍]]任司令员兼政委，中国人民解放军海军从此诞生。1989年2月17日，中央军委批复海军党委：以1949年4月23日华东军区海军成立日期为“人民海军成立日”<ref name=pa>{{Cite web |url =http://news.163.com/13/1225/11/9GUI1LPI00014JB6.html  |title = 毛泽东寄语海军建设：海军要搞好，使敌人怕 |author =  |date =2013-12-25  |publisher =网易  |language =  |accessdate =  |quote =  }}</ref><ref name=qiangda/>。

华东军区海军成立后，以华东军区海军司令部、政治部的名义在《[[大公报|大公报]]》上刊登了《关于招收原海军人员参加人民海军的通知》，并在[[福州|福州]]、[[青岛|青岛]]、[[厦门|厦门]]、[[广州|广州]]等地设登记国民党海军人员办事处。由此招收了大批原中华民国海军官兵。其中有[[曾以鼎|曾以鼎]]中将（原国民党海军第二舰队司令、江防舰队副总司令、海军司令部参谋长）、[[韩玉衡|韩玉衡]]少将（原国民党海军马尾造船所所长）等高级军官，也有海军专业技术人员如[[徐时辅|徐时辅]]（原国民党海军总司令部办公室副主任）、[[何希琨|何希琨]]（原国民党海军青岛军官学校上校教育长）等人。张爱萍将这些人员组织起来，成立海军研究委员会，协助华东军区海军首长接收装备、组建海军部队、主持舰船修复<ref name=zhutuo/>。

1949年8月28日，毛泽东在[[中南海|中南海]]怀仁堂正门大厅里接见了张爱萍、林遵、[[曾国晟|曾国晟]]（原中华民国海军司令部第六署署长）、[[金声_(海军)|金声]]（原中华民国海军司令部办公厅主任）、[[黄胜天|黄胜天]]（华东军区海军司令部参谋）等人一行。应张爱萍请求，毛泽东为华东军区海军题词：“我们一定要建设一支海军，这支海军要能保卫我们的海防，有效地防御帝国主义的可能的侵略。”另为华东军区海军机关报《[[人民海军_(报纸)|人民海军]]》报题写了“人民海军”四字报头。报头及题词在8月29日下午送交张爱萍<ref name=zhutuo>{{Cite web |url = http://news.xinhuanet.com/mil/2009-07/08/content_11674492.htm |title =毛泽东对人民海军谆谆嘱托  |author =  |date = 2009-07-08 |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref><ref>东方鹤，张爱萍与人民海军的创建，畅销书摘2000:20-23</ref>。

1949年11月11日，为了培养海军人才，在大连成立了第一海军学校，即今[[海军大连舰艇学院|海军大连舰艇学院]]。

1950年[[元旦|元旦]]，《人民海军》报创刊号发行，刊登了毛泽东的上述题词，8月27日[[朱德|朱德]]总司令、[[刘少奇|刘少奇]]副主席、[[周恩来|周恩来]]总理为华东军区海军的题词（朱德题词：“虚心学习，努力工作，建设一支人民的海军”；刘少奇题词：“建设人民的海军，巩固国防”；周恩来题词：“为建设中国人民海军而奋斗”），以及华东军区司令员[[陈毅|陈毅]]、政治委员[[饶漱石|饶漱石]]、副司令员[[粟裕|粟裕]]的题词<ref name=zhutuo/><ref name=qiangda/><ref>{{Cite web |url = http://mil.sohu.com/20160601/n452347458.shtml |title =  陆军机关报《人民陆军》问世 司令政委合写头条|author =  |date =2016-06-01  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。

1950年1月13日，正在苏联进行国事访问的毛泽东自[[莫斯科|莫斯科]]给正在主持中央工作的刘少奇发电报称：“可即任命[[萧劲光|萧劲光]]为海军司令。”据此，1950年1月15日中央军委命令：“（一）为了统一管理现有各地海军人员及舰艇，及建立人民海军，特任命萧劲光为中国人民解放军海军司令员。（二）现在大连创办之海军学校，及华东军区海军司令部，今后即直接向萧劲光司令员报告工作。”後來，正在苏联访问的毛泽东又發表意見支持萧劲光将海军机关设在北京的观点，毛泽东表示：“海军是一个军种，海军机关是一个战略决策机构，应单独成立司令部。海军机关要设在北京。”<ref name=zhutuo/>

=== 发展 ===
[[File:Type_65_frigate_PLANS_Haikou_(209).jpg|缩略图]]。]]
1950年4月14日，中国人民解放军海军领导机关成立大会在[[北京协和医院|北京协和医院]]礼堂举行。萧劲光任司令员，[[刘道生|刘道生]]任副政委兼政治部主任；同年任命[[王宏坤|王宏坤]]为副司令员，[[罗舜初|罗舜初]]为参谋长。1955年至1960年，相继组建了[[东海舰队|东海舰队]]、[[南海舰队|南海舰队]]和[[北海舰队|北海舰队]]<ref name="2008whitebook">{{cite web |url=http://download.china.cn/ch/pdf/090120.pdf |title=2008年中国的国防白皮书 |accessdate=2009-03-18 |work= |publisher=中华人民共和国国务院新闻办公室 |date=2009年1月 }}</ref><ref name=pa/><ref>{{Cite web |url = http://hunan.voc.com.cn/xhn/article/201607/201607180902115227.html |title = 这个湘籍开国中将 被誉为“陆海空全能”他是谁？|author =  |date = 2016-07-18 |publisher =  新湖南|language =  |accessdate =  |quote =  }}</ref>。

1950年3月中旬，毛泽东在中南海颐年堂听取了萧劲光有关海军筹建工作的汇报。随后毛泽东批准经费，先在北京[[贡院东街|贡院东街]]建起两栋三层楼作为海军司令部、政治部机关的办公地点。1953年后，请[[梁思成|梁思成]]主持设计，建造了使用至今的海军机关办公楼。后来又在海军机关办公楼周边逐步建设了直属部队营房、军官宿舍等，形成了“海军大院”<ref name=zhutuo/>。

1949年底至1950年初，毛泽东第一次率团访问[[苏联|苏联]]，同苏联政府签订了购置海军装备的协议，苏联政府向[[中华人民共和国|中华人民共和国]]提供1.52亿[[美元|美元]]贷款用于支付有关费用。但[[朝鲜战争|朝鲜战争]]爆发后，[[美国第七舰队|美国第七舰队]]进入[[台湾海峡|台湾海峡]]，中华人民共和国战略中心转向[[朝鲜半岛|朝鲜半岛]]，该笔贷款后經调整使用，海军首次购买苏联装备计划搁浅<ref name=qiangda/>。

1950年8月10日，海军建军会议在北京举行。会议决定向中共中央提出争取苏联援助的具体建议。不久，萧劲光向毛泽东汇报海军建军会议情况。1950年10月8日，毛泽东亲自给[[斯大林|斯大林]]写下一封信，以解释和推动海军军事订货问题。1952年4月底至6月底，萧劲光再度访问苏联莫斯科，与苏联海军商谈1952年内中国人民解放军海军订货落实及海军五年建设计划等问题。1952年7月10日，毛泽东再次给斯大林写信。1953年6月4日，中苏签订《关于海军交货和关于在建造军舰方面给予中国以技术援助的协定》。按照该协定，中华人民共和国自苏联购得舰艇137艘，其中成品舰42艘、需运回中国组装的半成品舰95艘<ref name=qiangda/>。

1950年11月底，中央军委电令各中央局、各中央分局、各[[大军区|大军区]]，责成各地的党政军民学机关设法清理、登记流散各地的原海军人员。“凡无政治问题，身体尚健，而有一技之长者，尽可能抽调来，以克服新海军建设中人才的困难”。此后，萧劲光从这批人以及华东军区海军研究委员会中选出20多人，请到北京成立中国人民解放军海军机关的研究委员会<ref name=zhutuo/>。
[[File:1967-10_1953年毛泽东视察海军.jpg|thumb]]
1952年2月14日，毛泽东冒雪视察中国人民解放军海军机关，并对海军装备建设作出指示<ref name=zhutuo/><ref name=pa/>。1953年，毛泽东主席视察海军，其间，5月21日为[[长江舰|长江舰]]、[[洛阳舰|洛阳舰]]，5月24日为[[南昌舰|南昌舰]]、[[广州舰|广州舰]]、[[黄河舰|黄河舰]]题词：“为了反对[[帝国主义|帝国主义]]的侵略，我们一定要建立强大的海军。”<ref>{{Cite web |url = http://www.china.com.cn/military/2013-11/27/content_30719572.htm |title = 揭秘毛泽东为海军题词：没写“一定”被丢弃 |author =  |date = 2013-11-27 |publisher = 中国网 |language =  |accessdate =  |quote =  }}</ref>1953年12月4日，中共中央政治局讨论海军建设计划时，毛泽东指示：“为了肃清海匪的骚扰，保障海道运输的安全；为了准备力量，于适当时机收復[[台湾|台湾]]，最后统一全部国土；为了准备力量，反对帝国主义从海上来的侵略，我们必须在一个较长的时期内，根据工业发展的情况和财政的情况，有计划地逐步地建设一支强大的海军。”<ref name=pa/>

1954年至1965年间，中国人民解放军海军水面舰艇部队、潜艇部队同[[中华民国海军|中华民国海军]]发生[[海战|海战]]十余次，主要以水面舰艇部队为主，初期损失较多，随着海战经验的丰富，取得了几次以弱势装备取胜的战例，随着装备水平的提高，海战逐渐减少；海岸炮兵参加了历次[[金门炮战|金门炮战]]；[[中国人民解放军海军航空兵|中国人民解放军海军航空兵]]在历次台海军事冲突中执行任务。1954年5月，海军参加东矶列岛战役并取胜。

1955年，海军派出部队，与陆军、空军协同作战，参加了[[一江山岛战役|一江山岛战役]]，取得胜利。1957年，国务院总理[[周恩来|周恩来]]在青岛检阅海军部队。1959年12月1日，东海舰队418潜艇在训练中因事故沉没，38名官兵遇难。

1965年，与[[中华民国海军|中华民国海军]]进行[[八六海战|八六海战]]和[[崇武以东海战|崇武以东海战]]并取得胜利。八六海战创造了小艇击沉两艘驱逐舰（PCE45劍門號巡邏艦、PC118章江號獵潛艦）的记录。1974年，[[南海舰队|南海舰队]]在[[西沙海战|西沙海战]]中击败南越海军，控制了[[西沙群岛|西沙群岛]]全部岛礁。

1975年5月2日晚至5月3日凌晨，毛泽东在[[中南海|中南海]]最后一次主持召开中共中央政治局会议，他同每位与会的[[中共中央政治局委员|中共中央政治局委员]]握手交谈，毛泽东握着海军政治委员[[苏振华|苏振华]]的手说：“管海军靠你。海军要搞好，使敌人怕。”随即伸出小拇指说：“我们海军只有这样大！”1975年5月8日，苏振华向海军党委常委传达了毛泽东的指示。随后，苏振华组织人员并邀请国务院工业部门人员共同研究制定了《海军舰艇十年发展规划》。1975年5月22日，苏振华以个人名义向毛泽东写了书面报告，报告中表示“力争在十年左右建成一支较强大的海军。”5月23日，毛泽东收阅报告后批示：“努力奋斗，十年达到目标。”<ref name=pa/><ref name=zhutuo/>

=== 改革开放初期 ===
[[File:US_Navy_031022-N-6346B-004_The_People's_Liberation_Army_Navy_(PLAN)_guided_missile_destroyer_Shenzhen_(DDG_167)_sails_into_Apra_Harbor,_Guam.jpg|缩略图]][[阿普拉港|阿普拉港]]的[[深圳号驱逐舰|深圳号驱逐舰]]。]]
1978年11月18日，公布《[[海军舰艇命名条例|海军舰艇命名条例]]》。1980年，重建[[中国人民解放军海军陆战队|海军陆战队]]。

1985年11月16日至1986年1月19日，东海舰队132合肥号导弹驱逐舰、东运615丰仓号综合补给舰（已改舷号舰名为882鄱阳湖号综合补给舰）组成舰艇编队出访[[巴基斯坦|巴基斯坦]]、[[斯里兰卡|斯里兰卡]]和[[孟加拉国|孟加拉国]]，此为中国人民解放军海军首次派出对外出访舰艇编队。

1988年3月14日，南海舰队在[[赤瓜礁海战|赤瓜礁海战]]中击败越南海军，收复[[赤瓜礁|赤瓜礁]]、[[华阳礁|华阳礁]]、[[永暑礁|永暑礁]]、[[东门礁|东门礁]]、[[渚碧礁|渚碧礁]]、[[南薰礁|南薰礁]]。

1989年3月31日至1989年5月2日，海军大连舰艇学院[[郑和号训练舰|郑和号训练舰]]出访美国，中国海军舰艇第一次出现在西半球。1995年，中央军委主席[[江泽民|江泽民]]在青岛检阅海军部队。

2001年4月1日，[[中国人民解放军海军航空兵|中国人民解放军海军航空兵]]派出2架[[歼-8II|歼-8II]]战斗机對一架在[[中国南海|中国南海]]执行侦查任务的[[美国海军|美国海军]][[EP-3|EP-3]]型[[偵察機|偵察機]]进行监视和拦截，其中一架僚机在中国[[海南岛|海南岛]]东南70[[海里|海里]]（110[[公里|公里]]）的中国[[专属经济区|专属经济区]]上空与美机发生發生碰撞，[[中美撞机事件|中國战斗机墜毀]]，飞行员王伟牺牲，而美國的軍機則迫降[[海南岛|海南岛]]陵水機場。

2003年，为了精简指挥层次，解放军海军撤銷了基地一级指挥机构。撤銷了海军航空兵部，把海军航空兵交由军区空军统一指挥，但保留了海军航空兵这个名称。2003年4月16日，北海舰队[[361号潜艇|361号潜艇]]在训练中因事故沉没，70名官兵遇难。事后，海军司令员[[石云生|石云生]]和海军政治委员的[[杨怀庆|杨怀庆]]被免职。

2013年7月14日，[[日本|日本]][[防衞省|防衞省]]稱5艘中國海軍艦艇和16艘[[俄羅斯海軍|俄羅斯海軍]]艦艇，於當日通過了位於日本[[北海道|北海道]]和俄羅斯[[薩哈林島|薩哈林島]]之間的[[宗谷海峽|宗谷海峽]]，從[[日本海|日本海]]駛向[[太平洋|太平洋]]。中俄艦艇沒有進入日本領海，這是中國海軍艦艇首次通過這一海峽。

2014年1月7日至6月23日，中国海军先后派出盐城舰、黄山舰分别与俄罗斯、丹麦、挪威舰艇执行了20批叙利亚化学武器海运出境至销毁点联合护航任务。<ref>[http://www.81.cn/jmywyl/2014-06/28/content_6019062.htm 中国海军圆满完成叙化武海运护航任务]</ref>

2014年6月23日至8月1日，中国海军派出由导弹驱逐舰海口舰、导弹护卫舰岳阳舰和综合补给舰千岛湖舰组成的舰艇编队以及和平方舟医院船参与在美国夏威夷举行的“环太平洋-2014”演习。<ref>[http://military.people.com.cn/GB/8221/69693/386312/index.html 中国海军首次派军舰参加环太平洋军演]</ref>

2014年7月14日中国人民解放军海军和[[俄羅斯海軍|俄羅斯海軍]]在东海举行海上协作-2014联合演习。

2014年9月20日解放军海军反海盜護航編隊訪問[[伊朗|伊朗]]阿巴斯军港，為史上首次中國艦隊訪問伊朗也是首次停靠[[波斯灣|波斯灣]]沿岸。<ref>[http://mil.huanqiu.com/photo_china/2014-09/2747496.html 環球時報-中方艦隊訪問伊朗]</ref>

2014年12月，中国海军北海、东海、南海三大舰队共18艘水面舰船，在西太平洋冲之鸟礁海域举行了代号“机动六号”的军事演习。规模超越2013年10月底的“机动五号”演习，是中国海军有史以来举行的最大规模的远海演习。<ref>[http://news.ifeng.com/a/20141225/42795212_0.shtml 外媒：解放军18艘军舰西太平洋演习再创纪录]</ref>

2015年3月29日至4月7日，在亚丁湾、索马里海域执行护航任务的中国海军第十九批护航舰艇编队临沂舰、潍坊舰、微山湖舰赶赴也门，执行撤离中国在也门人员任务。中国海军护航编队共执行了４次撤离任务，成功从也门撤离了中国公民600多人，以及来自15个国家的外国公民279人。<ref>[http://news.xinhuanet.com/world/2015-04/07/c_1114892269.htm 新闻链接:撤离也门行动大事记]</ref>

2015年5月11日至21日中俄两国海军在地中海海域举行了“海上联合—2015（Ⅰ）”演习。<ref>[http://jz.chinamil.com.cn/n2014/tp/content_6484543.htm 中俄“海上联合－2015（Ⅰ）”军事演习正式开始]</ref>

2015年8月20日至28日中俄两国海军在彼得大帝湾海域、克列尔卡角沿岸地区和日本海海空域举行“海上联合—2015（Ⅱ）”军事演习。<ref>[http://jz.chinamil.com.cn/n2014/tp/content_6632570.htm 我海军参加中俄“海上联合—2015（Ⅱ）”演习舰艇编队起航]</ref>

2016年初，在[[深化国防和军队改革|深化国防和军队改革]]中，以海军各舰队机关为基础，调整组建为各战区海军机关。<ref>[http://news.ifeng.com/a/20160202/47332944_0.shtml 国防部：战区军种机关调整组建已完成，凤凰网，2016-02-02]</ref>解放军海军现设有[[北部战区海军|北部战区海军]]、[[东部战区海军|东部战区海军]]、[[南部战区海军|南部战区海军]]三个战区海军，并在[[中部战区|中部战区]]编有海军人员。[[中部战区|中部战区]]地域（北京、天津、河北、山西、陕西、河南和湖北）内有海军单位，中部战区机关干部中也有海军干部，但中部战区不直辖海军部队，也不设中部战区海军领导机关<ref>{{Cite web |url = http://news.china.com/domestic/945/20160402/22358081.html |title = 五大战区除西部战区外均已建立海军 |author =  |date = 2016-04-02 |publisher = 中华网 |language = zh-CN |accessdate =  |quote = }}</ref>。

=== 军衔 === 
[[File:Chinese_sailors_qingdao.jpg|250px]]军衔：左侧为海军[[列兵|列兵]]，右侧为海军[[上等兵|上等兵]]。]]
{{Main|中国人民解放军海军军衔}}

海军现有军衔编制与陆空军相同，为5级19阶。大体上分为[[军官|军官]]与[[士兵|士兵]]两大类。
军官分为将校尉3级，将军分为[[上将|海军上]]、[[中将|海军中]]、[[少将|海军少]]3阶，校官分[[大校|海军大]]、[[上校|海军上]]、[[中校|海军中]]、[[少校|海军少]]4阶，尉官分[[上尉|海军上]]、[[中尉|海军中]]、[[少尉|海军少]]3阶；介于军官与士官之间的[[军校学员|军校学员]]；士兵分为[[士官|士官]]和士兵两级，士官分高级士官（海军一级军士长、海军二级军士长、海军三级军士长），中级士官（海军四级军士长、海军上士），初级士官（海军中士，海军下士），兵分海军[[上等兵|上等兵]]与海军[[列兵|列兵]]<ref name="中国士官军衔调整为7衔级 士官将有住房公积金">{{cite web |url = http://news.163.com/10/0729/03/6CNTHFOC000146BD.html |title = 中国士官军衔调整为7衔级 士官将有住房公积金 |accessdate=2010-07-29 |work= |publisher=网易新闻 |date= }}</ref>。

初级士官最高6年，中级士官最高8年，高级士官可以服现役14年以上。初级士官、中级士官在本级最高服现役年限内，按照岗位编制规定确定服现役时间<ref name="中国士官军衔调整为7衔级 士官将有住房公积金" />。

=== 歌曲 ===

* [[林春霖|林春霖]]（1961-1990）：在海难中救人牺牲的海军航空兵地勤兵。1990年10月，海军追授其“爱民英雄”荣誉称号。
* [[刘洪峰|刘洪峰]]（1971-1993）：在车辆事故中勇救群众牺牲。1993年，海军追授其“雷锋式的好战士”荣誉称号。
* [[刘志艳|刘志艳]]（1966-　　）：探亲期间见义勇为负重伤的海军汽车兵。1994年10月，海军授予其“舍己模范战士”荣誉称号。
* [[刘晓松|刘晓松]]（1978-1999）：海军工程大学毕业生，毕业赴部队报到期间列车出轨事故救人牺牲。1999年11月，海军授予其“舍己救人的军校优秀大学生”荣誉称号。
* [[朱桂全|朱桂全]]（1970-　　）：哈尔滨号导弹驱逐舰士官。2006年，海军授予其“爱舰精武模范士官”荣誉称号。
* [[梅雄|梅   雄]]（1986-2016）：交通事故中救人牺牲的海军通信兵。2015年9月，海军追授其“舍己救人爱民模范”荣誉称号。

=== 未获得荣誉称号的著名英模 ===
* [[高　翔|高　翔]]（1931-　　）：1965年在海南岛上空驾驶歼-6战机击落美军F-104战机，记一等功。
* [[卜凤刚|卜凤刚]]（1932-1982）：解放战争时期战斗英雄，曾被东北人民解放军授予“孤胆英雄”荣誉称号，并获毛泽东奖章。后为海军火炮装备军代表，积劳成疾逝世。国家副主席乌兰夫为《解放军报》撰文，称“优秀共产党员卜凤刚是我们学习的楷模”。
* [[刘德全|刘德全]]（1942-2014）：长期在北海舰队服役，海军大连舰艇学院原副院长，海军少将。20世纪80年代起成为著名学雷锋标兵，被誉为“雷锋团长”。徐向前元帅题词：“向雷锋式的好干部刘德全同志学习”。
* [[杨志亮|杨志亮]]（1962-　　）：1988年赤瓜礁海战中中方唯一受伤人员，记一等功。后为海军少将，海军研究院政治委员。
* [[柏耀平|柏耀平]]（1962-　　）：著名的飞行员舰长，2000年记一等功。后为海军少将，北部战区海军副参谋长。
