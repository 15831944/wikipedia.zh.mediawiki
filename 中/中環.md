'''中環'''（{{lang-en|'''Central'''}}）{{Notetag|“Central”之名原爲對應1850年代開始實行的當地的行政屬區「中區」（Central District），而中環則爲當時香港華人對當地的通俗區劃，由於「中環」與「中區」的範圍大致上重合，故多數人都將兩者視爲同義詞，後來當[[港英政府|港英政府]]正式採納「中環」作爲官方認可地名時，也直接將中區之英文簡稱“Central”作爲「中環」的官方英文翻譯，故中環沒有如[[上環|上環]]和[[堅尼地城|堅尼地城]]般以粵音拼讀作爲官方英文名稱，但這同時也誤導了很多香港市民以爲「[[四環九約|四環九約]]」是港英政府的官方行政區劃，及以爲「四環」中的「中環」是從英文“Central”翻譯而來，然而現存的歷史文獻均顯示港英政府從來沒有使用「四環」作爲行政區劃。}}，又稱'''中區'''（{{lang-en|'''Central District'''}}，簡稱'''Central'''）{{Notetag|中區雖然名爲「區」，但並非現今香港十八區區議會的區劃之一，僅具有地名之意，然而部分政府部門如[[香港警務處|香港警務處]]和[[香港消防處|香港消防處]]會以「中區」作爲其部門內部分區的一個地區單位。香港並無統一的行政分區，故不同部門會有不同的部門內部行政分區。使用「中區」作爲地區單位的部門不少會在英文名稱上直接使用簡稱“Central”而省略“District”一詞爲該地區單位命名，例如中區消防局的英文名稱爲“Central Fire Station”。}}，位於[[香港島|香港島]][[中西區_(香港)|中西區]]，是[[香港|香港]]的[[中心商業區|商業中心]]及政治中心。

[[香港交易所|香港交易所]]、多家大型[[銀行|銀行]]、跨國金融機構及外國[[領事館|領事館]]都設在中環。香港的[[香港終審法院|終審法院]]、[[禮賓府|禮賓府]]（前港督府）以及全港第二高建築物[[國際金融中心二期|國際金融中心二期]]也位於中環。[[香港特別行政區政府總部|香港特別行政區政府總部]]、[[立法會綜合大樓|立法會綜合大樓]]和[[解放軍駐港部隊|解放軍駐港部隊]][[中環軍營|中環軍營]]則位於鄰近中環的[[金鐘|金鐘]][[添馬|添馬]]。

中環之名源於19世紀時香港華人對香港島北岸城區的通俗分區「[[四環九約|四環九約]]」，「中環」為其中一環；至於港英政府對[[維多利亞城|維多利亞城]]{{Notetag|19世紀時港英政府稱其於香港島北岸的發展區爲「維多利亞城」（City of Victoria），城外地方皆爲郊區，「維多利亞城」在當時也被視爲[[英屬香港|英屬香港]]的首府。至20世紀中葉後香港整體被視爲單一城市進行發展，香港島北岸的發展區也已連成一片，「維多利亞城」之名便逐漸淡出。由於中環一帶一直都是港英政府的行政中心，而中環的[[聖約翰座堂_(香港)|聖約翰座堂]]更被認爲是維多利亞城的中心，故早期的官方地圖中「維多利亞城」之名一般多被標示在中環的位置，很多人因此誤以爲「維多利亞城」專指中環一地，但其實「維多利亞城」之名涵蓋整個香港島北岸從[[西環|西環]]到[[銅鑼灣|銅鑼灣]]一帶的發展區，港英政府更於1903年豎立了至少7塊「[[維多利亞城界碑|維多利亞城界碑]]」標示其範圍。}}的官方分區則可分爲東、中、西三區，中環之地即「中區」。在此之後雖然中區的界線略有變化，但其核心區域一直都是中環一帶；後來香港政府也採納「中環」作爲官方地名，「中環」和「中區」也幾乎成爲同義詞，一般坊間應用多稱「中環」，「中區」則爲行政性的區劃。

中環也是香港的交通樞紐，是四條[[港鐵|港鐵]]路線的交匯點{{noteTag|包括[[荃灣綫|荃灣綫]]、[[港島綫|港島綫]]、[[東涌綫|東涌綫]]和[[機場快綫|機場快綫]]（按通車次序排序）。}}，其中[[機場快綫|機場快綫]]通往[[赤鱲角|赤鱲角]][[香港國際機場|香港國際機場]]。[[中環天星碼頭|中環天星碼頭]]有頻繁渡輪橫越[[維多利亞港|維多利亞港]]往來[[尖沙咀|尖沙咀]]，[[港外線碼頭|港外線碼頭]]有定期航班往來[[香港離島|離島]]。

== 歷史 ==
中環是香港的心臟地帶，也是[[香港開埠|香港開埠]]後最早開發的地區，也是香港的[[商業中心|商業中心]]。

在1841年1月，[[英國人|英國人]]佔領並登陸港島時作人口統計，當時稱「群大路」的中上環一帶只在有50約人居住，反而位於南部的[[赤柱|赤柱]]有約2000人<ref>{{Cite web|url=https://hk.lifestyle.appledaily.com/lifestyle/special/daily/article/20180602/20408344|title=【詠物誌】香港最早人口普查　邊度人丁最旺|accessdate=2018-08-15|author=鄭天儀|date=2018-06-02|publisher=蘋果日報}}</ref>。不過基於中環鄰近以前還屬於[[清朝|清朝]]的[[九龍半島|九龍半島]][[尖沙咀|尖沙咀]]<ref>{{Cite book|title=香港二十八總督|last=張|first=連興|publisher=三聯書店(香港)有限公司|year=2012|isbn=9789620432491|location=香港|pages=41}}</ref>，有攻守之利，而且中環背靠高山而前有[[維多利亞港|深水港]]，適合船隻航行，最後便率先在中環建立其[[軍事基地|軍事基地]]，並迅速興建多條幹道。在今[[下亞厘畢道|下亞厘畢道]]的山坡，1841年開埠時已劃為政府專用地段，作為[[香港殖民地時期#香港政府|殖民地政府]]象徵的[[港督府|港督府]]便座落在[[政府山|政府山]]的中央位置。

1874年桂文燦編纂的《廣東圖說》同治刊本，當中以[[阿群帶路圖|群帶路]]標誌為[[上環|上環]]、中環、[[灣仔|下環]]。而1866年的《新安縣全圖》，當時的[[阿群帶路圖|群帶路]]所標示的位置亦在中環現址相近。因此歷史上，中環屬於[[維多利亞城|維多利亞城]]的其中一部份。

遠在[[第二次世界大戰|第二次世界大戰]]前及戰後初期，位於[[香港島|香港島]]的中環已是當時香港的主要商業中心。由於土地不敷應用，在[[香港總督|港督]][[德輔|德輔]]任內（1887年至1891年），中環進行了多次[[填海|填海]]工程。今日的[[香港會所|香港會所]]、[[皇后像廣場|皇后像廣場]]、[[立法會大樓_(香港)|前立法會大樓]]（現為[[香港終審法院|香港終審法院]]所在地）等，也是當時的填海工程所建成的。

1970至1980年代是中環的全盛時期，當時中環不斷興建[[摩天大廈|摩天大廈]]，包括各銀行總部。加上[[金融市場|金融市場]]開始興旺，不少香港重要的商業活動均在中環進行，不少香港人都以在中環上班為榮。<ref>{{cite book|author=萬里地圖組|title=會說話的地圖：香港篇|year=2011|publisher=萬里機構|location=香港|isbn=9789621446602|pages=24|language=zh-hk}}</ref>但由於中環土地即使多次填海都始終不能滿足需要，加上中環的[[辦公室|辦公室]]租金一直居高不下，這個港島的主要商業區便逐漸擴展至分別位處中環東西兩面的[[金鐘|金鐘]]和[[上環|上環]]，以及[[灣仔|灣仔北岸]]。不過貴為香港的心臟和金融市場中心，中環的商業活動仍然相當頻繁。

雖然中環至今仍然有多幢歷史建築得以留存，但由於[[中心商業區|中心商業區]]的發展重建壓力及政府保育政策的缺失，歷年來中環有不少歷史建築未及得到保育便已經被拆卸或重建，例如[[香港大會堂|香港大會堂]]、舊[[匯豐銀行|匯豐銀行]]總行、舊[[香港郵政總局|郵政總局]]、舊[[香港會|香港會]]會所等等。近年政府遷拆[[中環天星碼頭|中環天星碼頭]]及[[皇后碼頭|皇后碼頭]]更引起巨大爭議及衝突。2009年，時任特首[[曾蔭權|曾蔭權]]提出多項中環歷史建築保育計劃。擬保育的建築包括[[中區政府合署|中區政府合署]]東座及中座、前[[中央警署|中央警署]]、前[[域多利監獄|域多利監獄]]及前[[裁判司署|裁判司署]]、[[中環街市|中環街市]]、-{[[荷里活道|荷里活道]]}-前警察宿舍（[[英皇書院|英皇書院]]）等。此外政府亦和[[香港聖公會基恩小學|香港聖公會基恩小學]]洽商保育[[慈幼英文學校|慈幼英文學校]]建築群。

== 交通 ==
作為商業中心區，中環交通極之方便八達。[[港島綫|港島綫]]駛經[[中環站|中環站]]（{{lang-en|'''Central Station'''}}），[[荃灣綫|荃灣綫]]以[[中環站|中環站]]為終點站、[[東涌綫|東涌綫]]和[[機場快綫|機場快綫]]也以位處本區的[[香港站|香港站]]為市區總站，[[中環站|中環站]]—[[香港站|香港站]]是一個重要交通樞紐，兩站相連，乘客可通過站內通道在不拍卡出閘情況下在兩站間往來。港島綫穿梭香港島北岸、荃灣綫來往香港島與[[九龍|九龍]]；[[東涌綫|東涌綫]]往返[[香港島|香港島]]、[[西九龍|西九龍]]、[[新界|新界]][[荔景|荔景]]、[[青衣島|青衣島]]與[[大嶼山|大嶼山]]東涌之間，也可在中途站[[欣澳站|欣澳站]]轉乘[[迪士尼綫|迪士尼綫]]；[[機場快綫|機場快綫]]為旅客提供市區至[[香港國際機場|香港國際機場]]的快捷交通工具選擇，前往[[香港國際機場|香港國際機場]]只需時24分鐘。港鐵亦計劃延長東涌綫至[[添馬站|添馬站]]（{{lang-en|'''Tamar Station'''}}），接駁[[將軍澳綫|將軍澳綫]]前往[[將軍澳新市鎮|將軍澳新市鎮]]。

巴士方面，[[干諾道|干諾道]]中和[[德輔道|德輔道]]中都有許多巴士路線往來港九新界各區，當中不少更是以中環為總站。[[香港電車|香港電車]]則駛經德輔道中。

渡輪方面，中環近[[維多利亞港|維多利亞港]]海旁可謂香港渡輪服務交通樞紐，[[港外線碼頭|港外線碼頭]]設有多條航線往來[[香港離島|多座離島]]，包括[[中環至長洲航線|中環至長洲]]（[[新渡輪|新渡輪]]）<ref name="公司簡介-新渡輪"/>、中環至[[大嶼山|大嶼山]][[梅窩|梅窩]]（新渡輪）<ref name="公司簡介-新渡輪">{{Cite web |url=http://www.nwff.com.hk/public/getitem.php?lang=zh-hk&id=7f8911b8-d67f-4fc1-a76d-0d6fae26d16a&v=27&submenu_num=1 |title=公司簡介 |accessdate=2017-10-17 |publisher=新世界第一渡輪服務有限公司}}</ref>、中環至[[愉景灣|愉景灣]]（[[港九小輪|港九小輪]]）<ref>{{Cite web |url=http://www.hkkf.com.hk/index.php?op=timetable&page=pengchau&style=tc |title=中環 - 坪洲渡輪 |accessdate=2017-10-17 |publisher=港九小輪控股有限公司 ''www.hkkf.com.hk''}}</ref>、中環至[[南丫島|南丫島]][[榕樹灣|榕樹灣]]、南丫島[[索罟灣|索罟灣]]和中環至[[馬灣|馬灣]][[珀麗灣|珀麗灣]]等等<ref>{{Cite web |url=http://www.td.gov.hk/tc/transport_in_hong_kong/public_transport/ferries/service_details/ |title=專營及持牌渡輪服務 服務詳情 |accessdate=2017-10-17 |publisher=運輸署 ''www.td.gov.hk''}}</ref>。[[天星小輪|天星小輪]]是跨越[[維多利亞港|維多利亞港]]、往來香港島和九龍的歷史悠久和特色交通工具，[[中環天星碼頭|中環天星碼頭]]有頻繁渡輪[[中環至尖沙咀航線|往來九龍尖沙咀]]。中環往來尖沙咀航線服務的歷史更可追溯到1870年代的[[蒸汽船|蒸汽船]]<ref>《油尖旺區風物誌》，油尖旺區議會</ref>。昔日天星小輪還有[[中環至紅磡航線|中環至紅磡航線]]，惟因乘客量不足，於2011年4月1日起停辦。
