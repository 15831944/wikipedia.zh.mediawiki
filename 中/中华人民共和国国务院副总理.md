'''中华人民共和国国务院副总理'''是[[中华人民共和国|中华人民共和国]][[中华人民共和国国务院|国务院]]（中央人民政府）及其[[国务院常务会议|常务会议]]的重要[[中华人民共和国国务院组成人员|组成人员]]，为[[国家级正职|国家级正职]]（第一副总理）或[[国家级副职|国家级副职]][[中华人民共和国公务员|公务员]]、[[党和国家领导人|党和国家领导人]]，一般由[[中国共产党中央政治局常务委员会|中共中央政治局常务委员]]（第一副总理）和[[中共中央政治局委员|中共中央政治局委员]]担任。

根据《[[中华人民共和国宪法|中华人民共和国宪法]]》第八十八条和《[[s:中华人民共和国国务院组织法|中华人民共和国国务院组织法]]》第二条的规定，国务院副总理在国务院的总理负责制下协助[[中华人民共和国国务院总理|总理]]工作<ref>{{cite web |url = http://www.gov.cn/misc/2005-06/10/content_5548.htm |title = 中华人民共和国国务院组织法 |accessdate = 2007-12-14 |date = 1982-12-10 |publisher = 中国政府门户网站 |language = zh-cn }}</ref>。副总理的人选，由[[中华人民共和国国务院总理|国务院总理]]提名，经[[全国人民代表大会|全国人民代表大会]]表决后，由[[中华人民共和国主席|国家主席]]根据全国人大会议的决定任命。作为[[中华人民共和国国务院组成人员|国务院组成人员]]，副总理每届任期5年，连续任职不得超过两届。总理出国访问期间，由负责常务工作的副总理代行总理职务。现任排名第一、负责国务院常务工作的副总理是[[韩正|韩正]]，他同时兼任[[中共中央政治局|中共中央政治局]]常務委员和国务院党组副书记。排名位列韩正之后的副总理还有[[孙春兰|孙春兰]]、[[胡春华|胡春华]]和[[刘鹤|刘鹤]]，均是[[中国共产党中央政治局|中共中央政治局]]委员。

自[[Template:国务院/7|第七届国务院]]（李鹏内阁）开始，除[[邹家华|邹家华]]和[[朱镕基|朱镕基]]曾短暂以中央委员和中央候补委员身份担任外（1991年4月—1992年10月），国务院副总理均由[[中国共产党中央政治局|中共中央政治局]]委员担任，排名第一的副总理更是[[中国共产党中央政治局常务委员会|中共中央政治局常委]]。
