'''中專8線 溫雅寮－麗水'''。{{r|thb60}}起點位於[[臺中市|臺中市]][[梧棲區|梧棲區]]臺中港南堤路底，終點位於臺中市龍井區麗水里忠明橋。全長12.165公里。{{r|thb60}}全線位於[[臺中港|臺中港]]港區內。

== 路線說明 ==
;'''[[臺中市|臺中市]]'''
*[[梧棲區|梧棲區]]：'''溫雅寮'''（起點，南堤路底）→ 南堤路 → '''區界'''
*[[龍井區|龍井區]]：'''區界''' → 南堤路 → '''[[臺中火力發電廠|臺中火力發電廠]]''' → 龍昌路 → '''麗水'''（終點，忠明橋）

== 沿線地標、設施 ==
*[[臺中港|臺中港]]
**南防波堤
**西碼頭
*[[臺中火力發電廠|臺中火力發電廠]]

== 歷史 ==
1977年時本編號第一次被使用，當時路線為「''溫雅寮～梧棲''」{{r|gazette}}，係由1976年第二次公路普查時所編的中55線改編，起點位於當時[[台17線|台17線]]六公里處，終點位於當時[[台12線|台12線]]起點。1981年將中專8線改為「臺中港臨海道路線」{{r|gazette}}，即為今日南堤路路線，自此未有變動。
