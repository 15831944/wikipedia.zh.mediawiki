'''中国人民解放军中部战区联合作战指挥中心'''，位于[[北京市|北京市]]，隶属[[中国人民解放军中部战区联合参谋部|中国人民解放军中部战区联合参谋部]]，是[[中国人民解放军中部战区|中国人民解放军中部战区]]的联合作战指挥机构。

==简介==
2015年11月27日，[[中华人民共和国国防部|中华人民共和国国防部]]新闻发言人[[杨宇军|杨宇军]]就[[深化国防和军队改革|深化国防和军队改革]]在健全联合作战指挥体制上有何举措时称，“主要是适应打赢信息化战争、有效履行使命任务的要求，组建战区联合作战指挥机构，健全军委联合作战指挥机构，构建完善精干高效的战略战役指挥体系。”<ref>{{Cite web |url =http://military.china.com.cn/2015-11/27/content_37177664.htm  |title =国防部谈军队改革：组建战区联合作战指挥机构  |author =  |date =2015-11-27  |publisher =  中国网|language =  |accessdate =  |quote =  }}</ref>2016年2月1日，中华人民共和国国防部新闻局局长、国防部新闻发言人杨宇军在国防部专题新闻发布会上表示，战区是“本战略方向的唯一最高联合作战指挥机构”<ref>{{Cite web |url = http://news.163.com/16/0202/06/BEQ5BF2500014AED.html|title = 战区是唯一最高联合作战指挥机构|author =  |date =  2016-02-02 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2016年2月前后，新成立的五个战区分别设立战区联合作战指挥中心。战区联合作战指挥中心是在此前各[[大军区|大军区]]作战指挥中心的基础上建成，增加了空军、火箭军等军兵种，增设了测绘导航、空域管理等作战要素，调整了指挥席位的设置<ref>{{Cite web |url = http://mil.huanqiu.com/china/2016-02/8623038.html |title =西部战区砍原军区数十个指挥席 30分钟发7方案  |author =  |date = 2016-02-29 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref>。

2016年成立后，中部战区出台了《战区联合作战指挥中心值班暂行规定》，中部战区联合作战指挥中心总值班员由中部战区联合参谋部某局局长田桂民兼任<ref>{{Cite web |url =http://www.legaldaily.com.cn/army/content/2016-04/06/content_6555156.htm  |title = 中部战区出台联合作战值班和参谋部建设规定 |author =  |date =2016-04-06  |publisher = 法制网 |language =  |accessdate =  |quote =  }}</ref>。2016年3月，中部战区指挥机关岗位练兵进入阶段性考核验收，来自中部战区机关各军兵种要素的参谋人员分批分期完成命题考核，考核内容包括参谋业务、政治工作、后勤保障等基础专业，还包括联合作战筹划、联合作战指挥、信息分析处理等综合情况处置<ref>{{Cite web |url = http://military.people.com.cn/n1/2016/0401/c1011-28244657.html |title = 中部战区起步开局大力提升联合作战指挥能力 |author =  |date = 2016-04-01 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。2016年5月，中部战区首次通报表彰联合作战优秀参谋人员<ref>{{Cite web |url =http://www.mod.gov.cn/power/2016-05/29/content_4666620.htm  |title =中部战区狠抓能力建设实现联指高起点起步  |author =  |date = 2016-05-29 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。2016年，中部战区曾两次组织指挥所研究性演习<ref>{{Cite web |url = http://news.xinhuanet.com/mil/2017-01/16/c_129449068.htm |title =  联合演训聚焦“练官强将”——中部战区联合作战指挥中心见闻|author =  |date = 2017-01-16 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。
