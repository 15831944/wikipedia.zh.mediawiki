'''中華民國與阿拉伯聯合大公國關係'''是指[[中華民國|中華民國]]與[[阿拉伯聯合大公國|阿拉伯聯合大公國]]（通稱'''阿聯'''）之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，中華民國目前在第1大城[[杜拜|杜拜]]設立[[實質大使館|具大使館性質]]的代表機構。<ref name="駐外館處">{{Cite web |url = http://www.mofa.gov.tw/OverseasOffice_Content.aspx?n=6357834932B83C83&sms=5A961ED2E4BA25E8&s=0984A85A3A9A6677&os=20C1A3DAF6A74FCE |title = 駐外館處 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>

持[[阿拉伯聯合大公國護照|阿拉伯聯合大公國護照]]的{{link-en|阿拉伯聯合大公國人口|Demographics of the United Arab Emirates|阿聯公民}}也可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於阿拉伯聯合大公國第1大城[[杜拜|杜拜]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#MID-EAST |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

2018年，阿拉伯聯合大公國是中華民國第18大貿易夥伴、第12大[[進口|進口]]夥伴、第25大[[出口贸易|出口]]夥伴。出口至阿聯的金額為12億4,999萬0,192美元，年減6.428%；自阿聯進口的金額為46億3,675萬6,658美元，年增19.805%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）33億8,676萬6,466美元，年增33.632%。2017年，阿拉伯聯合大公國是中華民國第19大貿易夥伴、第13大進口夥伴、第23大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

出口至阿聯的主要項目有[[紡織|紡織]]纖維、[[電腦|電腦]]零組件、[[電子產品|電氣器材]]、[[汽車零組件列表|汽車零配件]]、[[五金|五金]]等；自阿聯進口的主要項目有[[鋁|鋁]]錠、[[石油|石油]]及[[石油化學|石化]]產品<ref name="駐館與駐地關係"/>。

===投資===
1977年－2013年，根據經濟部投資審議委員會統計，[[臺灣商人|台商]]在阿拉伯聯合大公國的投資件數共6件，總投資金額約337萬美元。<ref name="投資環境簡介"/>

台商在阿聯經營的產業以電腦業、[[通訊技術|通訊業]]與[[運輸|運輸業]]為主，其他包括[[水產養殖|水產]]加工、[[家具|家具]]和旅行用品等。電腦業、通訊業一般皆設廠在[[自由貿易區|自由貿易區]]內，其他廠商則依業務需要分設於大公國境內。台商經營方式一般為設立發貨倉庫直接配銷或[[轉口|轉口]]，以及進口[[半成品|半成品]]組裝等。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/ae/pdf/103-74United%20Arab%20Emirates.pdf |date = 2014年12月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

[[宏碁|宏碁]]、[[華碩電腦|華碩電腦]]、[[鴻海科技集團|鴻海科技集團]]等公司皆於杜拜自由貿易區內設立公司或發貨倉庫，業務範圍涵蓋[[中東#國家及地區|中東國家]]、[[北部非洲|北非]]、[[中亞|中亞]]及[[蘇聯加盟共和國|前蘇聯國家]]。台商的資通訊產品在中東地區的主要競爭者來自[[美國|美國]]、[[日本|日本]]、[[芬蘭|芬蘭]]、[[韓國|韓國]]等國際品牌。<ref name="投資環境簡介"/>

===會展===
由於杜拜已成為[[波斯灣|波斯灣地區]]各項國際展覽的中心，每年吸引台商組團前往參加如[[塑膠|塑膠]][[機器|機械]]、[[食品|食品]]、電腦及[[消費電子產品|電子消費用品]]、五金[[建材|建材]]及家具、安全器材、[[醫療|醫療]]、[[旅遊|旅遊]]、[[紙|紙類]]應用、[[美容|美容]]；[[太陽能|太陽能]]、[[電力|電力]]、[[照明|照明]]及[[能源|能源]]等大型商展。並由外貿協會、[[行政院農業委員會|行政院農業委員會]]等單位協助台商前往辦理「[[台灣精品|台灣精品]]發表會」、「醫療業務[[健檢|健檢]]推薦會」等拓銷推廣活動。<ref name="外交年鑑"/><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2015年2月8日，外貿協會組團參加於杜拜舉辦的「2015年[[波斯灣阿拉伯國家|波灣]]國際食品展」，共有21家臺灣食品業者參展。臺灣的[[清真|清真]]產業品質保證推廣協會（THIDA）所核發的清真認證（[[清真#符合“清真”的食物|Halal]]），於2014年正式獲得{{link-en|阿聯政府|Politics of the United Arab Emirates}}認可，此次食品展的臺灣館設置清真食品展示區並在認證廠商攤位處明顯標示「Halal認證」，吸引許多[[穆斯林|穆斯林]]買主興趣。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=657275&url=/NewsSearch.asp |author = 外貿協會 |date = 2015年2月13日 |title = 貿協赴杜拜拓銷臺灣食品創佳績 接單逾1850萬美元 |publisher = 經貿透視網}}</ref>

===航空===
2006年4月，[[中華航空|中華航空]]開闢[[中正國際機場|中正國際機場]]（臺北）↔[[阿布達比國際機場|阿布達比國際機場]]航線。中華民國外交部長[[黃志芳|黃志芳]]及交通部長[[郭瑤琪|郭瑤琪]]前往[[阿布達比|阿布達比]]，參加華航的首航慶祝酒會。<ref name="95年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2006/html/08-1-2.htm |title = 《中華民國95年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>但目前已停航。

2014年2月10日，[[阿聯酋航空|阿聯酋航空]]開闢[[臺灣桃園國際機場|臺灣桃園國際機場]]（台北）↔[[杜拜國際機場|杜拜國際機場]]間的每日1班直飛客運航班，成為首家直航臺灣的[[中東|中東]]籍[[國家航空公司|國家航空公司]]。採用[[波音777#777-300ER（77W）|波音777-300ER]]之[[廣體客機|廣體客機]]服務，夏季班表（3－9月）杜拜起飛時間為凌晨4:35，臺北起飛時間為上午10:55。阿聯酋航空並規劃增加第2班台北直飛航班，直飛航班開通後，有助於雙邊的經貿與人員往來。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 經濟回顧與展望 |publisher = 中華民國對外貿易發展協會}}</ref><ref>{{Cite web |url = http://www.boca.gov.tw/content.asp?CuItem=428&BaseDSD=13&CtUnit=17&mp=1 |title = 特殊資訊 |publisher = 外交部領事事務局}}</ref>

2015年7月19日，阿聯酋航空首度以[[空中巴士A380|空中巴士A380]]客機飛往台灣，比平常的波音777-300ER座位數多近40%。清晨於杜拜國際機場起飛，下午4:40抵達臺灣桃園國際機場。阿聯酋航空於2014年2月開航台灣後，載客率平均達80%以上，經評估決定先安排1班A380飛行往返台灣的航線。<ref>{{Cite web |url = http://www.cna.com.tw/news/aopl/201507190041-1.aspx |author = 汪淑芬 |date = 2015年7月19日 |title = 阿聯酋A380首飛台 座位多4成 |publisher = [[中央通訊社|中央通訊社]]}}</ref>

====事件====
2017年5月30日，阿聯酋航空（以下稱 Emirates）發函內部[[電子郵件|電子郵件]]，為增開[[亞洲國家和地區列表|亞洲]]尤其是中華人民共和國的航班並因應「[[一個中國政策|一個中國政策]]」，要求[[中華民國籍|中華民國籍]]空服員執勤時，在圍裙上移除[[中華民國國旗|中華民國國旗]]徽章並改配戴[[中華人民共和國國旗|中華人民共和國國旗]]徽章，此舉引發台灣[[空服員|空服員]]反彈。<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20170530/1129309/ |author = 許麗珍 |date = 2017年5月30日 |title = 阿聯酋強迫台籍空服員 配戴中國國旗執勤惹議 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref>對此，[[中華民國外交部|中華民國外交部]]訓令駐杜拜辦事處向Emirates[[總部|總部]]抗議，隨後，Emirates再表示先前決定是「不恰當」、「不正確」的，要求台灣空服員暫時移除國旗徽章，在下一步指示之前不需配戴任何徽章。並表示，現在空服員不管來自哪裡都不需要配戴國旗徽章，但Emirates沒有說明這是整個公司的[[制服|制服]]政策改變，或是特別針對台灣空服員。31日，駐處代表前往Emirates總部交涉，表達嚴正立場；另在台北約見Emirates駐臺辦公室人員，瞭解本案經過，並表達不滿與抗議。此事件中，有不少兩岸[[網民|網民]]在Emirates的[[Facebook|Facebook]]粉絲專頁上[https://www.facebook.com/Emirates/photos/a.196512987125125.39201.158307244279033/1198009420308805/?type=3&theater 留下正反意見]。<ref>{{Cite web |url = http://www.bbc.com/zhongwen/trad/chinese-news-40104383 |author = 綜合報導 |date = 2017年5月31日 |title = 台灣空服員改戴中國國旗？阿聯酋航空火速撤銷指令 |publisher = BBC中文網}}</ref><ref>{{Cite web |url = https://newtalk.tw/news/view/2017-05-31/88003 |author = 林朝億 |date = 2017年5月31日 |title = 阿聯酋航空致歉 仍要求勿帶台灣國旗 |publisher = [[新頭殼|新頭殼]]}}</ref>
