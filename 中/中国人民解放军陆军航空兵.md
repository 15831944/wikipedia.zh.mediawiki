'''中国人民解放军陆军航空兵'''，是[[中国人民解放军陆军|中国人民解放军陆军]]兵种。

==沿革==
1985年，中央军委主席[[邓小平|邓小平]]决定组建陆军航空兵。1986年10月3日，中央军委正式批准组建陆军航空兵的方案。1988年1月8日，中国人民解放军陆军第一支陆军航空兵部队在[[北京军区|北京军区]]某集团军成立<ref name=yuebing>{{Cite web|url = http://military.china.com/news/568/20150828/20288628_all.html|title = 中国陆军翱翔蓝天是从那次大阅兵开始！|author = |date = 2015-08-28|publisher = 中华网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170213164446/http://military.china.com/news/568/20150828/20288628_all.html|archive-date = 2017-02-13|dead-url = yes}}</ref><ref>{{Cite web |url =http://news.xinhuanet.com/mil/2004-11/03/content_2171583.htm  |title = 我军空中“轻骑兵” |author =  |date = 2004-11-03 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.xinhuanet.com/mil/2015-10/26/c_128358119.htm  |title = 海量组图：深度揭秘中国陆军航空兵 |author =  |date = 2015-10-26 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.xinhuanet.com/mil/2008-07/28/content_8823290_3.htm |title =中国军队30年：国防和军队现代化不断推进  |author =  |date =  2008-07-28|publisher =新华网  |language =  |accessdate =  |quote =  }}</ref><ref name=qmtsh>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1737042|title = 陆军航空兵将全面提升全域到达、立体突击和持续作战等能力 |author =  |date = 2017-07-19 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。1988年初，在部分[[大军区|大军区]]陆军机械化集团军陆续成立直升机大队，后发展为陆航团。2013年左右，[[北京军区|北京军区]]、[[沈阳军区|沈阳军区]]、[[兰州军区|兰州军区]]、[[广州军区|广州军区]]、[[南京军区|南京军区]]、[[成都军区|成都军区]]等六个大军区的陆航团改为陆航旅<ref>{{Cite web |url = http://mil.sohu.com/20140113/n393427471.shtml |title = 中国陆军适应信息化作战 六大军区陆航团转改旅 |author =  |date = 2014-01-13 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

中国人民解放军陆军航空兵成立之初，从[[中国人民解放军空军|中国人民解放军空军]]接收了大批装备，包括国产的直-5、直-6、直-9，以及美制S-70C“黑鹰”、俄制米-8及米-17、法制SA-342L“小羚羊”。同时中国人民解放军陆军航空兵开展专用机型的研制及引进工作，研制重点为以国产直-8、直-9为原型的专用改进型号，以及专用（中型）武装型和轻型反坦克型。其间，机载火箭发射器及反坦克导弹也进一步发展<ref name=yuebing/>。

1999年10月1日，[[中华人民共和国成立50周年阅兵|中华人民共和国成立50周年阅兵]]中，中国人民解放军陆军航空兵部队直升机参加阅兵。此后中国人民解放军陆军航空兵部队多次参加阅兵活动<ref name=yuebing/>。

中国人民解放军陆军航空兵原由中国人民解放军总参谋部陆军航空兵部管理。1986年10月，成立中国人民解放军总参谋部陆军航空兵办公室（副军级），属[[中国人民解放军总参谋部|中国人民解放军总参谋部]]建制，由总参谋部装备部代管。1988年10月，中国人民解放军总参谋部陆军航空兵办公室改为中国人民解放军总参谋部陆军航空兵局，由总参谋部直接领导。1989年2月，总参谋部陆军航空兵局正式开始办公。1992年9月，总参谋部炮兵部、装甲兵部、工程兵部、防化部、陆军航空兵局合并，组建[[中国人民解放军总参谋部军训和兵种部|中国人民解放军总参谋部兵种部]]，下设陆航局。2003年12月，陆航局扩为中国人民解放军总参谋部陆军航空兵部。2015年底至2016年初，在[[深化国防和军队改革|深化国防和军队改革]]中，撤销中国人民解放军总参谋部，成立[[中国人民解放军陆军|中国人民解放军陆军]]领导机构，中国人民解放军陆军航空兵改由[[中国人民解放军陆军参谋部|中国人民解放军陆军参谋部]]航空兵局管理<ref name=qmtsh/>。
