'''中国人民解放军空军武昌场站'''，位于[[湖北省|湖北省]][[武汉市|武汉市]][[江夏区|江夏区]][[山坡乡|山坡乡]]，隶属[[中国人民解放军中部战区空军|中国人民解放军中部战区空军]]某部。

==沿革==
原为1950年代末组建的'''中国人民解放军空军山坡场站'''，为团级单位，负责供应飞行部队的武器弹药、油料、航空器材备件、车辆运输、通信保障、军需给养工作。1962年，升格组建'''中国人民解放军空军山坡基地'''，为师级单位，将地勤维护保障任务划归场站建制。基地下设司令部、政治部、后勤部、机务处等机关。基地主任为[[樊马义|樊马义]]，参谋长[[张石川|张石川]]，还有两位副参谋长。1972年至1976年间，空某师调离山坡基地，移防四川大足机场和万县梁平机场；山坡基地撤销，恢复为'''中国人民解放军空军山坡场站'''；新组建的歼击机师驻山坡机场<ref name=xia>夏时周 口述、夏少芳 整理，我在山坡基地的工作经历，武汉文史资料2016(9):20-24</ref>。

[[文革|文革]]初期[[七二〇事件|七二〇事件]]中，毛泽东被困武汉[[东湖宾馆|东湖宾馆]]，所在地遭到军队人员冲击。1967年7月20日下午，[[周恩来|周恩来]]乘坐专机抵达山坡机场<ref name=cpc/>。同行的还有另一架大型客机，机上运载了[[8341部队|中国人民解放军8341部队]]一个连的兵力。飞机在山坡机场停留两天<ref name=xia/>。周恩来抵达山坡机场后，随即登上专车，在中央警卫一中队小分队的护卫下直抵武汉[[东湖宾馆|东湖宾馆]]。到东湖宾馆百花一号后，周恩来随即约见[[杨成武|杨成武]]、[[谢富治|谢富治]]、[[汪东兴|汪东兴]]，研究了解事态发展。百花一号和[[毛泽东|毛泽东]]居住的东湖宾馆梅岭一号有距离，徒步前往易被发现，周恩来、杨成武、谢富治、汪东兴4人乃乘坐一辆小[[吉普车|吉普车]]往见毛泽东。经他们劝说，毛泽东决定乘飞机离开武汉。随后决定毛泽东乘坐的飞机自[[武汉王家墩机场|王家墩机场]]起飞。自东湖宾馆至王家墩机场的沿途安全由[[中国人民解放军武汉军区|中国人民解放军武汉军区]]负责。同时决定将山坡机场的两架自北京来的飞机也调到王家墩机场待命，以便毛泽东离开武汉时乘用。1967年7月21日凌晨2时，毛泽东离开梅岭一号，很快抵达王家墩机场。天亮后，毛泽东登上专机。1967年7月21日上午11时，飞机在[[上海虹桥机场|上海虹桥机场]]降落。这是毛泽东自1958年（1958年[[中共中央政治局|中共中央政治局]]考虑到飞机有危险性，决定毛泽东出行不坐飞机，而坐火车专列）后首次乘飞机，也是毛泽东最后一次乘飞机<ref name=cpc>{{Cite web |url =http://dangshi.people.com.cn/n/2014/0613/c85037-25145686-3.html  |title = 毛泽东1967年在武汉的一次遇险经历 |author =  |date =2014-06-13  |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。

此后仍在1967年，山坡机场突降三架专机，其中一架是[[英国|英国]]制“子爵”式高级客机，为中央首长专机。但并机上并无中央首长，仅有工作人员。飞机在山坡机场停留三天，停留期间飞机停在起飞线上，旁边有消防车、救护车、通信车待命，机组人员也就地待命，随时准备起飞。但因毛泽东选择坐火车专列从[[北京|北京]]途经武汉，所以这三架专机作为预备行动方案未被使用<ref name=xia/>。

后来，空军山坡场站更名为'''中国人民解放军空军武昌场站'''<ref name=hannan>{{Cite web |url =  http://news.carnoc.com/list/410/410555.html|title =汉南通用机场飞行程序竣工设计通过军民航审查  |author =  |date = 2017-07-10 |publisher = 民航资源网 |language =  |accessdate =  |quote =  }}</ref>。2000年代，在[[中国人民解放军空军汉口场站|空军汉口场站]]准备从[[武汉王家墩机场|王家墩机场]]迁移，而阳逻机场尚未建成启用时，空军武昌场站成为武汉地区的军事重地<ref name=xia/>。

2012年10月，武汉市人民政府常务会议审议通过了《武汉“十二五”物流空间规划》，其中首次提出将山坡机场军民合用，改扩建成武汉江南航空物流港。2013年1月27日，[[海航|海航]]机场集团有限公司董事长兼总裁董桂国一行5人，在武汉市江夏区人民政府副区长[[范礼奎|范礼奎]]的陪同下，实地考察了山坡机场<ref>{{Cite web|url =http://www.whjt.gov.cn/zhxx/jxqjtj/2013/01/29/43998.htm|title =海航机场集团有限公司考察山坡机场|author =|date =2013-01-29|publisher =武汉市交通运输委员会|language =|accessdate =|quote =}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。但根据《[[长江日报|长江日报]]》2016年1月20日的报道，武汉规划中的第二机场[[武汉江南国际机场|武汉江南国际机场]]已不再选址山坡机场<ref>{{Cite web |url = http://e.cjn.cn/cjrb/html/2016-01/20/content_5505238.htm |title =  “十二五”武汉交通亮点再发现 新城区着眼民生出行 构建综合交通运输体系|author =  |date =2016-01-20  |publisher = 长江日报 |language =  |accessdate =  |quote =  }}</ref>。

空军武昌场站原隶属[[中国人民解放军广州军区空军|中国人民解放军广州军区空军]]空军航空兵第十八师，为该师第五十二团驻地。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，转隶[[中国人民解放军中部战区空军|中国人民解放军中部战区空军]]某部<ref name=hannan/>。
