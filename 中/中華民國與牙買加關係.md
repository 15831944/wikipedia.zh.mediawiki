'''中華民國與牙買加關係'''是指[[中華民國|中華民國]]與[[牙買加|牙買加]]之間的關係。兩國於1962－1972年有官方[[外交#外交上的承認|外交關係]]，斷交後，也曾於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。<ref name="改善"/>目前對牙買加的相關事務由'''中華民國駐[[聖克里斯多福及尼維斯|聖克里斯多福及尼維斯]][[中華民國駐外機構列表|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-217-1495f-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]一案棄權，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[牙買加|牙買加]]。]]
{{see also|中華民國外交|{{link-en|牙買加外交|Foreign relations of Jamaica}}}}
1962年8月9日，中華民國與牙買加建立大使級[[外交#外交上的承認|外交關係]]。9月3日，於首都[[京斯敦|京斯敦]]設立'''中華民國駐牙買加[[外交代表機構#大使馆|大使館]]'''。

1966年，開始派駐牙買加[[特命全權大使|大使]]。

1972年11月1日，牙買加與中華民國斷交。2日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

1991年11月，牙買加於首都[[台北市|台北]]設立'''牙買加商務暨貿易[[辦事處|辦事處]]'''。同時邀請[[中華民國政府|中華民國政府]]前往牙買加設處以推展雙方關係，中華民國政府提議先在首都京斯敦設立冠以正式國名的商務代表團，經由{{link-en|牙買加內閣|Cabinet of Jamaica}}會議通過，並分別以口頭及書面確認邀請設立'''[[台北经济文化代表处|中華民國商務代表團]]'''。但之後牙買加內閣在[[中華人民共和國政府|中華人民共和國政府]]強大壓力下，藉口避免民眾混淆、影響牙買加與[[中華人民共和國|中華人民共和國]]關係，以及該國[[選舉|大選]]等理由，要求駐館改用其他名稱。經過1年多的交涉，1993年6月9日，[[中華民國外交部|中華民國外交部]]決定中止雙方的代表機構。<ref name="改善">{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=50CD87327BF3477F |author = 公眾外交協調會 |date = 1993年6月10日 |title = 牙買加政府數年前主動與我洽談改善關係 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>

1991年11月18日，牙買加公共事業暨運輸部長抵臺訪問六天。期間前往拜會外交部、[[中華民國交通部|交通部]]、[[中華民國經濟部|經濟部]]、[[行政院農業委員會|行政院農業委員會]]（農委會）、[[中華民國中央銀行|中央銀行]]、[[台北市議會|台北市議會]]、[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）、[[長榮集團|長榮集團]]等政府部門與民間機構，並參加牙買加港務局駐華商務辦事處[[開幕典禮|開幕典禮]]暨酒會。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=222079BB07436860 |author = 公眾外交協調會 |date = 1991年11月17日 |title = 牙買加公共事業暨運輸部部長皮克斯基爾訂於本（十一）月十八日抵華訪問六天 |publisher = 中華民國外交部}}</ref>

1998年8月19日，簽署《中牙（牙買加）國際[[特快专递|快捷郵件]]業務細則（換文）》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e7%89%99%e8%b2%b7%e5%8a%a0&tycountry_e=%e7%89%99%e8%b2%b7%e5%8a%a0&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境牙買加。]]
[[File:Visa_requirements_for_Jamaican_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|牙買加簽證政策|{{link-en|牙買加公民簽證要求|Visa requirements for Jamaican citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可於抵達{{link-en|牙買加機場列表|List of airports in Jamaica|該國國際機場}}後申請[[簽證#落地签证|落地簽證]]入境牙買加。停留最多30天，但可延長至90天內。若以非觀光目的入境時，仍需申辦「身分切結書」（Affidavit of Identity），可前往牙買加駐[[日本|日本]][[外交代表機構#大使馆|大使館]]申辦簽證。搭乘[[遊輪|郵輪]]者可以[[簽證#签证优惠待遇|免簽證]]入境牙買加1天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-217-1495f-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=ABBF62618F53F8DE |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|牙買加護照|Jamaican passport}}的{{link-en|牙買加人口|Demographics of Jamaica|牙買加公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將牙買加歸類為'''灰色警示'''：提醒注意（2017年5月31日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-43-836a4-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將牙買加歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=D4090DC3C44F755E |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
2018年，牙買加是中華民國第129大貿易夥伴、第111大[[進口|進口]]夥伴、第117大[[出口贸易|出口]]夥伴。出口至牙買加的金額為1,280萬6,532[[美元|美元]]，年增9.580%；自牙買加進口的金額為1,069萬3,772美元，年增24.318%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）211萬2,760美元，年減31.514%。2017年，牙買加是中華民國第127大貿易夥伴、第116大進口夥伴、第117大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
