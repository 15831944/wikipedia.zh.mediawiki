'''中国人民解放军61889部队'''，又名'''中央警卫团'''，驻[[北京市|北京市]]，隶属[[中央军委联合参谋部|中央军委联合参谋部]]，由[[中共中央办公厅警卫局|中共中央办公厅警卫局]]指挥，负责[[要員保护|保护]][[中华人民共和国|中华人民共和国]][[党和国家领导人|党和国家主要领导人]]。

== 沿革 ==
1928年5月，[[朱德|朱德]]、[[毛泽东|毛泽东]]在[[井冈山会师|井冈山会师]]，组建[[中国工农红军第四军_(红一方面军)|中国工农红军第四军]]。第四军成立军直特务连。1930年8月，[[中國工農紅軍第一方面軍|中國工農紅軍第一方面軍]]（又称中央红军）成立，原特务连改称总前委特务连。同年10月，总前委特务连扩编为特务队。同年12月，特务队改称为特务大队。1931年11月，[[中华苏维埃共和国|中华苏维埃共和国]]临时中央政府成立，特务大队负责警卫任务。同年12月，特务大队改称国家政治保卫大队，隶属国家政治保卫处（后改成政治保卫局）。1934年7月，国家政治保卫大队扩编为政治保卫团。同年10月，中央红军开始[[长征|长征]]，政治保卫团大部随[[中国共产党中央委员会|中共中央]]和中央首长行动，负责警卫任务。1935年1月，中央红军长征抵达[[貴州省_(中華民國)|贵州省]][[遵义县|遵义县]]，政治保卫团缩编为政治保卫大队，执行了[[遵义会议|遵义会议]]的警卫任务。同年12月中旬，中央红军长征抵达[[陕北|陕北]]后，政治保卫大队改称总前委特务营。1936年5月，特务营改编为独立特务团一营。1937年4月，改编为中共中央军委警卫营，隶属中央军委总参谋部，负责保卫中共中央、中央军委、主席。

[[中国抗日战争|抗日战争]]爆发后，必须加强延安驻地、路线护送、会议和防空的警戒。1937年11月，从黄河边河防前线调回警卫营第三连，担负蓝家坪书记处、中宣部、军委直属政治部的警卫任务。为了加强防空警戒，1937年12月，由延安保卫营调来80名防空队员与警卫营机枪排合编为防空队，装备苏式马克沁重机枪六挺，负责对空瞭望，发出防空警报，开展对空射击。防空队与轰炸日机作战十余次。1938年2月7日，中央教导队成立，队长[[王金|王金]]（[[吴烈|吴烈]]继任），政治委员[[萧前|萧前]]；人员由八路军第三五九旅第七一八团、第三八五旅第七七○团等部抽调130余人，全部为党员干部，编为三个区队，1938年4月担负警卫对象内卫任务。1938年9月20日，由于日军不断轰炸延安，警卫营第二连随毛泽东搬到了杨家岭，四连随军委搬到王家坪。警卫营其他部队警戒中组部、中央党校、总政治部、总供给部、二局、解放社、抗大、锄奸部犯人看押。1939年6月，中央教导队扩编为中央教导大队，辖4个中队、骑兵连、训练队。1941年3月，中央军委警卫营改编成中央军委警卫大队。1942年10月，中央军委警卫大队、中央教导大队合编为中央警备团（第十八集团军总司令部警备团），隶属中央军委总参谋部和[[中共中央社会部|中共中央社会部]]领导。1944年9月5日，中央警备团直属警卫队战士[[张思德|张思德]]因公牺牲。同年9月8日，中共中央直属机关举行“张思德同志追悼大会”，[[中共中央主席|中共中央主席]][[毛泽东|毛泽东]]亲临大会并作《[[为人民服务|为人民服务]]》演讲。

1947年2月，临时编成临时特务队警卫，中共中央社会部一部赴[[山西省|山西省]][[临县|临县]]。同年3月，中央警备团分三路保卫中共中央领导和机关安全转移。同年8月，由[[东北民主联军|东北民主联军]]总部警卫团调来一个连，加入中央警备团建制。1948年5月间，中央警备团在[[河北省|河北省]][[平山县|平山县]]整编，临时特务队编入中央警备团建制。1949年2月，中央警备团第三、四营的7个连开入[[北平|北平]]，暂改称“北平市公安总队”，负责接收、警卫、训俘、肃特、维持社会治安。1949年3月，中央警备团护送中共中央、中国人民解放军总部自[[西柏坡|西柏坡]]迁至北平。7月，以北平市公安总队之名执行任务的部队归建。9月，以中央警备团等部队为基础，扩编为中国人民公安中央纵队第二师，师长[[刘辉山|刘辉山]]，政委[[张廷祯|张廷祯]]。其中中国人民公安中央纵队第二师第四团是以中央警备团为主成立。<ref name="zjkzx">[http://www.zjkonline.com/News/201502/New1254860.html 中央警卫局沿革 附首都警卫部队]，张家口在线，2015-02-09</ref>

1949年11月28日，[[中央人民政府公安部|中央人民政府公安部]]部长[[罗瑞卿|罗瑞卿]]根据[[毛泽东|毛泽东]]主席11月19日的批示，下令把[[胶东军区海军支队|中国人民公安中央纵队第一师]]、第二师已进[[中南海|中南海]]执勤的两个警卫营，与原[[华北人民政府|华北人民政府]]公安部警卫大队合编为中国人民公安中央纵队警卫团（又称中南海警卫团）。团长[[何有兴|何有兴]]，政委[[温良升|温良升]]。警卫团的建制及军政工作直属中央纵队，警卫业务及训练教育计划等工作统一由[[中共中央办公厅|中共中央办公厅]]警卫处[[汪东兴|汪东兴]]负责。1950年11月25日，中国人民公安中央纵队警卫团改编为中国人民解放军公安警卫师第一团，隶属公安部队建制。<ref name=zjkzx/>

1953年5月，中央军委命令从中央警卫师（公安警卫师）中抽组一个团，担负[[中国共产党中央委员会主席|中共中央主席]]、[[中国共产党中央书记处|中共中央书记处]]负责同志的警卫任务。1953年6月，以公安警卫师第一团为基础，成立[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]警卫团（简称“中央警卫团”），同时部队番号为“总字001”。部队建制归中央军委总参谋部领导，警卫任务归[[中共中央办公厅警卫局|公安部九局]]领导。1953年6月9日中央警卫团在[[中南海|中南海]]正式成立。首任团长[[张耀祠|张耀祠]]，政委[[杨德中|杨德中]]。1955年6月，中央警卫团归公安军（1955年7月18日至1957年9月1日存在）建制，业务仍由公安部九局领导。1955年8月，中央警卫团改编为中国人民解放军中央公安警卫团，定为副师级。1959年1月，中央公安警卫团改称中央警卫团，部队番号“3747”，划归[[北京卫戍区|北京卫戍区]]建制领导，定为正师级。1961年3月，中央警卫团调归[[中国人民解放军总参谋部|中国人民解放军总参谋部]]建制。中央警卫团初建时只有一千多人,主要任务是保卫[[毛泽东|毛泽东]]、[[刘少奇|刘少奇]]、[[朱德|朱德]]、[[周恩来|周恩来]]、[[陈云|陈云]]。<ref name="zjkzx" /><ref name="flz"/><ref name=wy/><ref name="yu">{{Cite web|url=http://news.163.com/14/1215/03/ADFNIP9800014Q4P.html|title=中央警卫团老兵佘兴发：我在8341部队的日子(图)|accessdate=2017-06-01|author=|date=2014-12-15|last=|work=|publisher=网易}}</ref>約1964年，首次使用8341部队番號，该番号由[[中国人民解放军总参谋部|中国人民解放军总参谋部]]下达。<ref name="wy">{{cite web |url = http://news.163.com/09/0828/01/5HP1DAAK00013LSN.html |title=毛泽东和他的“8341”部队|publisher=网易|date=2009-08-28}}</ref><ref name=yu/>
[[File:1968-04_1968年_支左人员学习毛泽东选集.jpg|thumb]]人员学习毛泽东选集]]
[[文化大革命|文化大革命]]期间[[七二〇事件|七二〇事件]]中，毛泽东被困武汉[[东湖宾馆|东湖宾馆]]，所在地遭到军队人员冲击。1967年7月20日下午，[[周恩来|周恩来]]乘坐一专机抵达山坡机场，同行的另一架专机是大型客机，运载了8341部队一个连的兵力。在其护送下，毛泽东乘飞机匆忙离开武汉<ref>{{cite book|author1=夏时周|editor=夏少芳|title=我在山坡基地的工作经历|date=2016(9)|publisher=武汉文史资料|pages=20-24}}</ref>。1969年10月，中央警卫团调归中共中央办公厅警卫处（简称中央警卫处，由公安部九局和中央警卫团合并组成），[[汪东兴|汪东兴]]具体负责领导、指挥。中央警卫团初建时，警卫单位或目标主要有[[玉泉山|玉泉山]]、[[新六所|新六所]]、[[中南海|中南海]]等地，后扩大到[[北戴河|北戴河]]、[[天安门|天安门]]、[[人民大会堂|人民大会堂]]、[[钓鱼台国宾馆|钓鱼台]]等，[[文化大革命|文革]]期间又增加了昌平农场、[[北京植物园|北京植物园]]、花园村、东官房等地。文革“[[三支两军|三支两军]]”中，先后到[[北京大学|北京大学]]、[[清华大学|清华大学]]、[[新华社|新华社]]、北京针织总厂、[[中国人民解放军八一电影制片厂|八一电影制片厂]]等单位支“左”<ref name="flz" />。1975年，部队番号一度改为57001部队，不久后又改回8341部队<ref name="yu" />。1976年[[毛泽东|毛泽东]]逝世后，又改为57003部队。1976年在[[粉碎四人帮|怀仁堂政变]]中执行抓扣“[[四人帮|四人帮]]”任务。1976年粉碎“四人帮”后，经[[中国共产党中央委员会|中共中央]]、[[中国共产党中央军事委员会|中央军委]]批准，1977年8月中央警卫团扩编为中央警卫师，为副军级。1982年5月，中央警卫师缩编为中央警卫团。1993年，部队定为正军级。2000年10月，改为61889部队。<ref name="zjkzx" /><ref name="flz">{{cite journal |title = 我所知道的8341部队 |author=冯立忠 |journal = 《领导文萃》 |year = 2008年 |issue = 第24期}}</ref><ref>[http://news.ifeng.com/history/vp/detail_2012_05/31/14952520_0.shtml 图说8341部队：直接参与粉碎四人帮后庆祝游行]，凤凰网，2012-05-31</ref><ref name=wjh>武健华：[http://sh.sina.com.cn/news/2010-03-30/0134138432.html 我所知道的8341部队]，新浪，2010-03-30</ref>

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，其原上级单位[[中国人民解放军总参谋部|中国人民解放军总参谋部]]撤销，该部队转隶新成立的[[中央军委联合参谋部|中央军委联合参谋部]]。

== 轶闻 ==
“8341”这組数字引起民间广泛猜測。有民间传说认为是毛泽东於1948年前往[[北京市|北京]]途中，与[[山西省|山西]][[五台山|五台山]][[塔院寺|塔院寺]]道士会面时，道士告之的4个数字，毛不知其意，便将其作为警卫部队的编号。意思是毛澤東享壽83歲，領導中國共產黨41年（由1935年[[遵义会议|遵义会议]]領導[[中国共产党|中国共产党]]至1976年）<ref>{{cite book |language = zh-hant |author = 冷思 |title = 十六大之後的中國 |url = |year = 2002年 |location =  |publisher = 有限出版社 |ISBN = 9789628647477 |page = 32 }}</ref>。亦有其他說法<ref>{{Cite web|url=http://www.chinawts.com/list/budnews3/125355303_10.htm |title=求得慧心五台归 |accessdate=2017-06-01 |author= |date=2006-01-11 |work= |publisher=五台山 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304030242/http://www.chinawts.com/list/budnews3/125355303_10.htm |archivedate=2016-03-04 }}</ref>。也有民间传说认为是因毛泽东参加[[新軍|新軍]]时所使用的第一支步枪的编号是“8341”<ref name="flz" />。然而，中央警卫团首任团长[[张耀祠|张耀祠]]在其回忆录中表示，这組番號為中国人民解放军总参谋部所定的部队序列编号，上述民间传闻毫无依据<ref name="flz" />。
