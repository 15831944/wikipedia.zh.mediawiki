'''中華民國與喬治亞關係'''是指[[中華民國|中華民國]]與[[喬治亞|喬治亞]]之間的關係。兩國從未建立[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，[[喬治亞|喬治亞]]當時屬於[[蘇聯|蘇聯]]的[[加盟共和國|加盟共和國]]（[[喬治亞蘇維埃社會主義共和國|喬治亞蘇維埃社會主義共和國]]），由蘇聯作為[[主權國家|主權國家]]參與投票。]]
===外交===
{{see also|中華民國外交|{{link-en|喬治亞外交|Foreign relations of Georgia}}}}
喬治亞自1991年[[蘇聯解體|從蘇聯獨立]]後未與中華民國有過正式外交關係，雙方以發展民間商務交流為主。目前對喬治亞的相關事務由'''[[臺北市|臺北]]－[[莫斯科|莫斯科]]經濟文化協調委員會[[臺北經濟文化代表處|駐莫斯科代表處]]'''、'''{{link-en|駐安卡拉台北經濟文化代表團|Taipei Economic and Cultural Mission in Ankara}}'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-139-c4698-03-1.html |title =  駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

2005年3月25日，[[中華人民共和國|中華人民共和國]]通过《[[反分裂國家法|反分裂國家法]]》后[[格鲁吉亚外交部|格鲁吉亚外交部]]发表声明，支持「[[一个中国|一个中国]]」的立场，格鲁吉亚欢迎中国在此范围内给予台湾广泛[[自治|自治权]]的善意<ref>{{cite web|title=一些国家政府支持中国通过《反分裂国家法》 |url=http://news.163.com/05/0325/20/1FNGRE4A0001124T.html |publisher=网易 |date=2005-03-25 |accessdate=2017-04-01}}</ref>。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境喬治亞。]]
[[File:Visa_requirements_for_Georgia_citizens.png|thumb]]必須申請簽證方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|喬治亞簽證政策|Visa policy of Georgia}}|格鲁吉亚公民签证要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。[[喬治亞政治|喬治亞政府]]對[[中華民國政府|中華民國政府]]並非友善，取得喬治亞簽證具相當難度。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-139-c4698-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持有{{link-en|喬治亞護照|Georgian passport}}的[[喬治亞人口|喬治亞公民]]抵台參加國際會議、運動賽事、商展活動，可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將喬治亞歸類為'''灰色警示'''：提醒注意（2017年5月23日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-277-31c96-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將喬治亞歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年9月4日）。<ref>{{Cite web |url = https://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=DA92A7D5BA13F45D |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|喬治亞經濟}}
2011年4月14日，喬治亞商工總會會長{{link-en|Jemal Inaishvili|Jemal Inaishvili}}受中華民國國際經濟合作協會（國經協會）邀請來台，並拜會[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）秘書長[[趙永全|趙永全]]，除了瞭解台灣經貿發展現況之外，也進一步促進雙方交流，以增加彼此合作機會。<ref>{{Cite web |url = http://www.trademag.org.tw/content02.asp?id=558965&type=22&url=/NewsSearch.asp |title = 橫跨歐亞 地理佳 喬治亞向台商招手 |date = 2011年5月4日 |publisher = 經貿透視雙周刊}}</ref>

2011年、2013年，分別舉辦第一、二屆「台喬經濟合作會議」。<ref>{{Cite web |url = http://www.cieca.org.tw/CountryInfoList.aspx |title = 國家基本檔(簡約版) |date = 2015年2月6日 |publisher = 中華民國國際經濟合作協會 |deadurl = yes |archiveurl = https://web.archive.org/web/20151022171920/http://www.cieca.org.tw/CountryInfoList.aspx |archivedate = 2015年10月22日 }}</ref>

2018年，喬治亞是中華民國第122大貿易夥伴、第108大[[進口|進口]]夥伴、第114大[[出口贸易|出口]]夥伴。出口至喬治亞的金額為1,540萬1,048[[美元|美元]]，年增32.300%；自喬治亞進口的金額為1,258萬2,246美元，年減40.765%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）281萬8,802美元，年減129.362%。2017年，喬治亞是中華民國第116大貿易夥伴、第99大進口夥伴、第118大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
