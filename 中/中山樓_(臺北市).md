'''中山樓'''是位於[[臺灣|臺灣]][[臺北市|臺北市]][[陽明山|陽明山]]的[[多功能大廳|多功能]][[會議中心|會議設施]]，1965年10月動工、1966年11月12日竣工啟用<ref>[[李松林|李松林]]著，《蔣介石的台灣時代》，台北，風雲時代出版，1993年，第229頁</ref>，由[[修澤蘭|修澤蘭]]設計，佔地約18,000平方公尺，外型的[[中國式建築|中式]][[宮殿式建築|宮殿建築]]樣式為主要特色；樓高三層，內部設有可容納約1,800人的會議廳（中華文化堂）、與可容納2,000人的餐廳各一座。目前由[[國立臺灣圖書館|國立臺灣圖書館]]轄下的陽明山中山樓管理所負責管理。

[[中華民國總統|中華民國總統]]在1996年改由[[中華民國總統及副總統在臺灣之公民直接選舉與罷免|公民直選]]前，皆由[[國民大會代表|國民大會代表]]選出，而中山樓正是[[國民大會|國民大會]]遷臺後的主要開會場所；因此中山樓除歷經多次總統選舉，也是[[中華民國憲法|中華民國憲法]][[中華民國憲法增修條文|歷次增修]]的會議所在地。中山樓建築亦常為[[新臺幣|新臺幣]]鈔券、[[中華民國郵票|郵票]]的主題圖像<ref>[http://www.currency.cbc.gov.tw/a1/T02.html 中央銀行新台幣介紹] {{webarchive|url=https://web.archive.org/web/20160824061033/http://www.currency.cbc.gov.tw/a1/T02.html |date=2016-08-24 }}</ref>，為[[第五套橫式新臺幣|現版新臺幣]]壹百元紙鈔背面的代表圖像。

興建中山樓，是時任[[中華民國總統|總統]][[蔣中正|蔣中正]]為紀念[[中華民國國父|國父]][[孫中山|孫中山]]百年誕辰，另一為發揚[[中華文化|中華文化]]，期使[[中華民國政府|中華民國政府]]在正式集會、慶典、接待國賓時，有適合之場地，並讓國際人士領略中華文化之精粹。<ref>{{cite book|author=曾一士總編輯|title=中山樓之美|year=2003|publisher=國立國父紀念館|location=臺北市|isbn=9789570228366}}</ref>

== 基址 ==
中山樓所在區域是昔日[[臺灣日治時期|臺灣日治時期]]的草山林間學校所在地，1935年[[始政四十周年記念臺灣博覽會|始政四十周年記念臺灣博覽會]]的草山分館（草山觀光館）即設在目前講堂之處。興建之初，由於地基為堅石、軟泥與硫磺土混雜，建築師在設計時將每根主要立柱依地勢與地質加長、加大，使之深入至硫磺土層以下，最深可逾12公尺。為防止硫磺土對建築之侵蝕，以熱[[柏油|柏油]]澆灌柱子與柱礎，再以鋁箔油毛氈包覆，隔絕硫氣。<ref>[http://reading.gov.taipei/ct.asp?xItem=6660987&CtNode=46534&mp=100021 40餘年的山中傳奇　中山樓] {{Wayback|url=http://reading.gov.taipei/ct.asp?xItem=6660987&CtNode=46534&mp=100021# |date=20170214003348 }}. 臺北市政府觀光傳播局</ref><ref>[http://www.ntl.edu.tw/fp.asp?fpage=cp&xItem=9032&CtNode=1268&mp=13 建築師的話]. 陽明山中山樓管理所</ref>中山樓亦是全世界迄今唯一一棟蓋於[[硫磺|硫磺]]口的建物<ref>[http://subtpg.tpg.gov.tw/web-life/taiwan/9608/9608-07.htm 陽明山中山樓]. 臺灣省政府</ref>，硫磺口是日治時代的[[草山溫泉|草山溫泉]]湧泉地，位於大會議廳之講台正下方，早年供應附近溫泉旅館使用，為減低地熱，則以十數根圓管鑽入講台下方，使地熱沿管線噴出。<ref>[http://hk.crntt.com/crn-webapp/touch/detail.jsp?coluid=192&kindid=0&docid=102944908 印在新台幣上的中山樓　卯足勁爭取遊客]. 中國評論通訊社</ref>

== 特色 ==
全樓以中國宮殿建築式樣為本，依山勢而建，以綠色琉璃瓦覆頂，搭配紅簷、白牆，建築主體之屋頂採單檐歇山頂。由於此為紀念中華民國國父孫中山百歲誕辰之建築，因此在設計與裝飾上融合許多吉祥元素，如：壽字台階、百壽橋、福字燈座、蝙蝠方格門等。建築照明部分，有多達48種造型各異之宮燈遍布內外。室內布置則有[[歐豪年|歐豪年]]、[[高逸鴻|高逸鴻]]、[[黃君璧|黃君璧]]、[[江兆申|江兆申]]、[[王壯為|王壯為]]、[[季康|季康]]之書畫作品。

== 組織沿革 ==
中山樓最早由[[陽明山管理局|陽明山管理局]]管轄，後改隸[[國立國父紀念館|國立國父紀念館]]，2012年5月20日改隸[[國立臺灣圖書館|國立臺灣圖書館]]管理<ref>[http://www.ntl.edu.tw/ct.asp?xItem=13002&CtNode=1182&mp=13 中山樓 ─ 組織沿革及編制]</ref>。

== 軼聞 ==
由於中山樓坐落於[[大屯火山群|大屯火山群]]的硫磺地帶上，故在中山樓舉行餐會的時候，接待方所準備的[[銀|銀]]製餐具容易接觸空氣中的[[硫|硫]]產生黑色的[[硫化銀|硫化銀]]斑點，故要平均20分鐘左右要更換一次。關於中山樓的風水傳說，則是其位於龍脈之上，位置與[[士林官邸|士林官邸]]、[[圓山行館|圓山行館]]及[[總統府_(臺灣)|總統府]]連成一線，左青龍、右白虎，前有[[紗帽山|紗帽山]]，左右還有松溪、磺溪環繞，形成“玉帶圍腰”。最特別的是，中山樓還是全世界罕見、直接蓋在硫磺坑口的大型建築物。

此外，中山樓當前列為禁區的二樓含有“總統套房”的設施，其中有[[蔣中正|蔣中正]]及夫人[[蔣宋美齡|蔣宋美齡]]當年的會客廳、寢室、衛生間，還有模擬[[臺海戰爭|臺海戰爭]]的兵棋室、一個成人身高的巨型特製地球儀。墻上挂了許多[[黃君璧|黃君璧]]、歐豪年的親筆字畫，還有四十三年前由外國進口時，據說為全臺第一套便後清洗的“下體盆”（或稱凈身盆），且至今還能使用。另一特別的是蔣中正用於指揮作戰的兵棋室，內有四台完全依照[[中國大陸|中國大陸]]沿海地形製作的立體模型圖，及對匪作戰地形圖；且兵棋室的屏風後方，還有一張床，為[[蔣經國|蔣經國]]的辦公室，據說是蔣經國為了尊敬先父，不願使用父親生前的房間，才在兵棋室下榻。

== 歷任總統使用 ==
* [[蔣中正|蔣中正]]總統：[[蔣中正|蔣中正]]總統與第一夫人[[宋美齡|宋美齡]]在中山樓二樓有總統套房，[[蔣中正|蔣中正]]總統與夫人[[宋美齡|宋美齡]]都會在夏天避暑時搬來[[陽明山|陽明山]]居住，甚至[[國民大會|國民大會]]或[[中國國民黨|中國國民黨]]黨代表大會、中全會在[[中山樓|中山樓]]召開，就會入住該樓[[總統套房|總統套房]]。
* [[蔣經國|蔣經國]]總統：[[蔣中正|蔣中正]]總統用於指揮作戰的兵棋室，兵棋室的屏風後方，還有一張床，為[[蔣經國|蔣經國]]總統的辦公室。[[蔣經國|蔣經國]]總統就任後，[[國民大會|國民大會]]或[[中國國民黨|中國國民黨]]黨代表大會、中全會的召開都會入住兵棋室。
* [[李登輝|李登輝]]總統：[[李登輝|李登輝]]總統在[[國民大會|國民大會]]或[[中國國民黨|中國國民黨]]黨代表大會、中全會召開時，會前往[[中山樓|中山樓]]致詞。外賓來訪時，[[李登輝|李登輝]]總統會帶著外賓遊歷中山樓。
* [[陳水扁|陳水扁]]總統：[[陳水扁|陳水扁]]總統任期只有[[任務型國民大會|任務型國民大會]]及2005年[[青年國是會議|青年國是會議]]開會時使用，2005年[[青年國是會議|青年國是會議]]開議時，[[陳水扁|陳水扁]]總統前往[[中山樓|中山樓]]致詞。
* [[馬英九|馬英九]]總統：[[馬英九|馬英九]]總統在2014年前往在[[中山樓|中山樓]]舉辦的「2014年師鐸獎表揚大會」，並在國宴廳上向與會老師們敬酒致意。
