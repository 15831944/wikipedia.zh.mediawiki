'''中国铁路乌鲁木齐局集团有限公司'''<ref>{{Cite web |url =  http://money.163.com/17/1023/15/D1EO750V002581PP.html|title = 中铁总公司制大进展:11家地方铁路集团新掌门亮相 |author =  |date = 2017-10-23  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>，原名'''乌鲁木齐铁路局'''，是[[中国铁路总公司|中国铁路总公司]]下属公司，管辖[[新疆维吾尔自治区|新疆维吾尔自治区]]全境和[[甘肃省|甘肃省]][[安北站|安北站]]以西164.2公里的铁路。管理线路总延展长度為7563.5公里，营业里程5764[[公里|公里]]。

==历史==
1971年1月1日，[[中华人民共和国交通部|交通部]]乌鲁木齐铁路局成立。1975年2月8日，乌鲁木齐铁路局改属[[铁道部|铁道部]]领导。1978年8月28日，在运输生产上，实行铁路局和段（站）两级管理。1980年6月，运输生产指挥权下放给乌鲁木齐办事处和哈密办事处，在运输生产上实行局—办事处—站段三级管理。1982年12月，乌、哈办事处改为分局，运输生产上实行铁路局—分局—站段三级管理。1994年4月25日根据铁道部铁劳函[1994]171号文件，“铁道部乌鲁木齐铁路局”更名为“乌鲁木齐铁路局”。

2005年按照乌鲁木齐铁路局管理体制改革的统一部署，先后撤消了4个分局级管理机构（乌鲁木齐、哈密铁路分局和北疆铁路公司、南疆铁路临管处），实行铁路局直接管理站段的体制，对11个运输站段进行了撤并和重组；全面完成了涉及社会职能单位的移交工作；推进多元系统的重组改制，房建生活系统、工辅业单位整建制划归多元系统管理。设立运输处、客运处、货运处、机务处、车辆处、工务处、电务处、对外合作处、安全监察室、办公室、总工程师室、企业管理和法律事务处、计划统计处、财务处、审计处、收入稽查处、人事处（党委干部部）、劳动和卫生处、建设管理处、物资管理处、土地房产管理处、人民武装部（人防战备处）。2005年末职工人数为53340人，其中：在岗职工48734人，在岗职工中长期职工48439人，临时职工295人；非在岗职工4606人，包括内部退养，长病、长假人员，长学人员，集体外出劳务，内部下岗人员。在岗职工年末人数中女职工占在岗职工人数的27.33%。

1995年，乌鲁木齐铁路局进行了工商登记<ref name=guapai/>。

2017年11月4日，经[[国家工商行政管理总局|国家工商行政管理总局]]核准，乌鲁木齐铁路局名称变更为'''中国铁路乌鲁木齐局集团有限公司'''（(国)名称变核内字[2017]第5278号）<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1851596  |title = 15家铁路局将集体变身集团公司：名称变更已通过核准 |author =  |date =2017-11-06   |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月13日，在国家企业信用信息公示系统中，中国铁路乌鲁木齐局集团有限公司已完成工商登记变更。工商登记变更后，中国铁路乌鲁木齐局集团有限公司仍为中国铁路总公司100%控股的公司，公司由原先的全民所有制企业变成有限责任公司（非自然人投资或控股的法人独资），注册资本由原先的1054.96512亿人民币变更为1118.5298亿人民币，经营范围增加矿产品、建材、化工产品、农畜产品、烟草制品的销售等<ref name=si>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1861836 |title = 4家铁路局率先变更工商登记：广州西安局集团增房产开发功能 |author =  |date = 2017-11-13 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

===车务段===
车务段负责管理列车运行，管辖二等以下的车站。乌鲁木齐铁路局下辖以下车务段：
*乌鲁木齐车务段
*哈密车务段
*奎屯车务段
*阿克苏车务段
*库尔勒车务段
*喀什车务段

===机务段===
[[File:乌鲁木齐.机务段大门_China_Xinjiang_Urumqi_Welcome_you_to_tour_t_-_panoramio.jpg|thumb]]
[[File:中国新疆乌鲁木齐市火车西站China_Xinjiang_Urumqi_West_Railway_Station_-_panoramio_-_罗布泊.jpg|thumb]]
机务段负责铁路机车的运用、检修和维护工作，当中分为乘务部门和检修部门，乘务部门是司机负责执乘工作，检修部门负责日常的机车保养。乌鲁木齐铁路局下辖以下机务段：
*乌鲁木齐机务段
*库尔勒机务段
*哈密机务段

===车辆段===
[[File:乌鲁木齐.火车西站车辆段_China_Xinjiang_Urumqi_Welcome_you_to_tour_-_panoramio_-_jun_jin_luo.jpg|thumb]]]]
车辆段主要负责铁路车辆的运用及基本检修。乌鲁木齐铁路局下辖以下车辆段：
*乌鲁木齐车辆段
*乌鲁木齐西车辆段
*库尔勒车辆段

===動车段===
*乌鲁木齐動车所(乌鲁木齐)

===客运段===
客运段负责管理客运列车乘务人员，旧称列车段。乌鲁木齐铁路局下辖以下客运段：
*乌鲁木齐客运段
*庫爾勒客運段

===工务段===
工务段负责铁路线路及桥隧设备的保养与维修工作。乌鲁木齐铁路局下辖以下工务段：
*乌鲁木齐工务段
*哈密工务段
*奎屯工务段
*库尔勒工务段
*阿克苏工务段
*喀什工务段

===工务机械段===
*乌鲁木齐工务机械段
===供电段===
[[File:20130208乌鲁木齐电务段.jpg|thumb]]
供电段负责电气化铁路的牵引供电、铁路运输信号供电、铁路地区的电力供应、电力设备的检修与保养等工作。乌鲁木齐铁路局下辖以下供电段：
*乌鲁木齐供电段
*库尔勒供电段
*哈密供电段

===电务段===
电务段负责管理和维护列车在运行途中的地面信号与机车信号及道岔正常工作，乌鲁木齐铁路局下辖以下电务段：
*乌鲁木齐电务段
*库尔勒电务段
