'''中華民國大總統'''，簡稱'''大總統'''，是[[中華民國|中華民國]][[北洋政府|北洋政府]]時期的[[中華民國國家元首列表|國家元首]]，為1947年[[行憲|行憲]]後的[[中華民國總統|中華民國總統]]前身。

對於首位中華民國大總統的說法不一。[[臺海兩岸|臺海兩岸]]主流意見、[[中華民國|中華民國]]和[[中華人民共和國|中華人民共和國]][[教科書|教科書]]均強調，[[孫文|孫中山]]在[[南京|南京]]成立[[中華民國政府|中華民國政府]]，是首位大總統。部分外國歷史學家則認為，首任大總統是在[[北京|北京]]掌握軍權的[[袁世凱|袁世凱]]。袁世凯作為[[清朝|清朝]]最後一任[[總理大臣|總理大臣]]，透過[[段祺瑞|段祺瑞]]等[[北洋軍|北洋軍]]將領發表《[[北洋五十將乞共和電|北洋五十將乞共和電]]》與《[[乞共和第二電|乞共和第二電]]》揚言發動[[兵變|兵變]]，才使[[監國|監國]][[載灃|載灃]]與[[隆裕太后|隆裕太后]]同意[[宣統退位|宣統退位]]；1913年10月10日，袁世凯就任中华民国首任「正式大總統」。

1916年6月6日袁世凯逝世後，中華民國大總統职位被[[北洋軍閥|北洋軍閥]]控制，成為[[虛位元首|虛位元首]]；期间孫文在[[廣州|廣州]]建立不被[[国际社会|国际社会]]承认的[[广州军政府|广州军政府]]，曾自任「非常大總統」，试图通过[[武力|武力]]取代北洋政府。[[中国国民党|中国国民党]]在[[苏俄|苏俄]]的资助下進行[[國民革命軍北伐|國民革命軍北伐]]且胜利，在[[南京|南京]]建立[[國民政府|國民政府]]，中華民國大總統一職由[[國民政府主席|國民政府主席]]取代，国民党的[[党国体制|党国体制]]逐步确立。

== 臨時大總統 ==
[[File:Sun_Yat-sen_2.jpg|thumb]]
中華民國臨時大總統是1912年1月1日至1913年10月10日对中华民国國家元首的稱呼。

首任臨時大總統是[[孫文|孫文]]，由1911年12月29日（農曆宣統三年十一月初十）召開的17個省代表（一省一票）臨時大總統選舉會上選出。

1912年1月1日，中華民國臨時大總統就職典禮在[[南京府|南京]][[南京總統府|總統府]]（清朝[[兩江總督|兩江總督]]署）舉行。但在北京的[[清朝政府|清朝政府]]仍存在，中國南北出現兩個[[政權|政權]]。同时，孫文的臨時大總統的程序及合法性也遭外界質疑。1911年12月31日出版的《[[紐約時報|紐約時報]]》報道指出参加這次選舉的代表屬[[:wikisource:en:The_New_York_Times/1911/12/31/DR._SUN'S_ELECTION_IS_NOT_TAKEN_SERIOUSLY|非法代表]]。<ref>{{cite web |url = http://query.nytimes.com/mem/archive-free/pdf?res=990CE3DC123AE633A25752C3A9649D946096D6CF |title=DR. SUN'S ELECTION IS NOT TAKEN SERIOUSLY |publisher=纽约时报 |date=1911-12-31 |language=en }}</ref>

[[File:孫文曾在選舉中舞弊？.png|thumb]]

孫文就職誓詞為：
<blockquote style="border: 1px solid blue; padding: 0.5em 0.8em;">
傾覆滿洲專制政府，鞏固中華民國，圖謀民生幸福，此國民之公意；文實遵之，以忠於國，為眾服務。至專制政府既倒，國內無變亂，民國卓立於世界、為列邦公認，斯時文當解臨時大總統之職。謹以此誓於國民。
</blockquote>

[[File:Seal_for_the_provisional_government_president_of_Republic_of_China_20050807.jpg|thumb]]]]

*临时大总统：[[孫文|孫文]]
*总统府秘书长：[[胡汉民|胡汉民]] 
*陆军总长：[[黄兴|黄兴]]，次长：[[蒋作宾|蒋作宾]]
*海军总长：[[黃鍾瑛|黃鍾瑛]]，次长：[[汤芗铭|汤芗铭]] 
*司法总长：[[伍廷芳|伍廷芳]]，次长：[[吕志伊|吕志伊]]
*財政總長：[[陳錦濤|陳錦濤]]，次長：[[王鴻猷|王鴻猷]]

*内务总长：[[程德全|程德全]]，次长：[[居正|居正]] 
*教育总长：[[蔡元培|蔡元培]]，次长：[[景耀月|景耀月]] 
*实业总长：[[张謇|张謇]]，次长：[[马君武|马君武]] 
*交通总长：[[汤寿潜|汤寿潜]]，次长：[[于右任|于右任]] 


自1912年1月12日開始，袁世凱開始勸說滿清皇族交出權力及[[宣統帝|宣統帝]]遜位，期間[[中國同盟會|中國同盟會]]對袁世凱的炸彈暗殺行動失敗，1月18日，孫中山以《五條要約》{{NoteTag|一、清帝退位，其一切政權同時消滅，不得私授於其臣；二、在北京不得更設臨時政府；三、得北京實行退位電，即由民國政府以清帝退位之故，電問各國，要求承認中華民國彼各國之回章；四、（孫）文即向參議院辭職，宣布定期解職；五、請參議院公舉袁世凱為大總統，如此方於事實上完善。}}的方式向袁世凱攤牌，試圖做實南京臨時政府，1月22日，孫中山又以公諸報端的方式將南北雙方幕後談判的内容全部曝光，这不但令袁世凱極其尷尬和不滿，也讓議和全權代表[[伍廷芳|伍廷芳]]盡失顏面，期間孫中山還試圖向日本借款，以籌備與袁世凱決戰，但未能成功。袁世凯為少生枝節，同意同盟會多人進入總長之列，同盟會最終如願以償。2月12日[[隆裕太后|隆裕太后]]頒降懿旨，宣統帝退位，袁世凱全權組織臨時共和政府，这是滿清統治正式結束的標誌，漢人重新統治全中國，中華民國正式取代滿清。

2月13日，袁世凱通電共和，根據南方獨立各省及部分革命黨人的意願，孫中山提出辭職咨文。2月15日，[[中華民國臨時參議院|臨時參議院]]（國會）正式選舉袁世凱為中華民國臨時大總統。3月8日通過《[[中華民國臨時約法|中華民國臨時約法]]》，將[[總統制|總統制]]改為[[內閣制|內閣制]]。袁世凱于3月10日在北京就職。

袁世凱就職誓詞：
<blockquote style="border: 1px solid blue; padding: 0.5em 0.8em;">
民國建設造端，百凡待治，世凱深願竭其能力，發揚共和之精神，滌蕩專制之瑕穢，謹守憲法，依國民之願望，達國家於安全彊固之域，俾五大民族同臻樂利。凡茲志願，率履弗渝。俟召集國會，選定第一期大總統，世凱即行解職。謹掬誠悃，誓告同胞。
大中華民國元年三月初十日
</blockquote>

== 大總統 ==
中華民國大總統任期五年，可以連任（無限制）。

第一任大總統是袁世凱，於1913年由國民黨主導的國會選出。袁世凱于1916年猝逝後，北洋軍閥成為北京政府的實際领导者，大總統及國務總理领導的內閣均沒有多大實權，重要决策由各大軍閥协商。1924年曹錕辭職後，大總統一職交由國務院攝行，之後由段祺瑞任臨時執政（代總統），段下野後交由國務院攝行，從顏惠慶復職後的幾任內閣再度攝行大總統職權，直到張作霖成立軍政府任陸海軍大元帥，大總統名稱正式走入歷史。

== 非常（時期）大總統 ==
'''中华民国非常大总统'''是[[孙文|孙文]]在[[广州|广州]]于1921年5月5日至1923年2月21日[[護法運動|第二次護法]]時的名称，其法統來自南下的[[中華民國第一屆國會|中華民國第一屆國會]]議員，與当时受到軍閥控制的[[中華民國北京政府|中華民國北京政府]]對抗。
