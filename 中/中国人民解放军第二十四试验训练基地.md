'''中国人民解放军第二十四试验训练基地'''（中国人民解放军89730部队），位于[[云南省|云南省]][[昆明市|昆明市]][[寻甸县|寻甸县]]大石洞。

==沿革==
该基地是[[中华人民共和国|中华人民共和国]]最早的反导弹试验基地，1970年自[[新疆维吾尔自治区|新疆维吾尔自治区]][[库尔勒县|库尔勒县]]迁至云南省寻甸县大石洞，时称'''[[中国酒泉卫星发射中心|中国人民解放军第二十训练基地]]第四试验部'''，隶属[[中国人民解放军国防科学技术委员会|中国人民解放军国防科学技术委员会]]（国防科委）<ref name=xrxrmzf>{{Cite web|url =http://www.gzxr.gov.cn/show-113-20997-1.html|title =二十四基地战友会来田湾开展爱心捐助|author =|date =2014-11-28|publisher =兴仁县人民政府|language =|accessdate =|quote =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1975年，第二十训练基地撤销，以第二十训练基地第四试验部为基础，组建'''中国人民解放军第二十四试验训练基地'''，执行军级权限。1970年代，该基地负责中华人民共和国反导工程“[[640工程|640工程]]”的试验任务，部署着用于导弹弹道测控的“110”超远程单脉冲跟踪雷达（该雷达可以在2000余公里外跟踪外空目标），还进行了“反击一号”反导导弹飞行试验<ref>{{Cite web |url =  http://mt.sohu.com/20170308/n482701558.shtml|title = 中国堪比“萨德”的武器，60年代我们就搞的工程 |author =  |date = 2017-03-08 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://military.china.com/jctp/11172988/20170213/30249281_27.html#photos |title = 矛和盾都要有！中国动能末段反导系统团队曝光 |author =  |date =2017-02-13  |publisher = 中华网 |language =  |accessdate =  |quote =  }}</ref><ref>姬文波，“640工程”中国第一代反导系统研制始末，凤凰周刊2010年04期</ref><ref>{{Cite web |url = http://news.xinhuanet.com/mil/2011-11/28/c_122342445.htm |title = 中国反导系统60年代便开始研制 外媒揭秘“640工程” |author =  |date = 2011-11-28 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://history.sohu.com/20140516/n399611118.shtml |title =揭秘640工程：中国反导研究已秘密实施45年  |author =  |date =2014-05-16  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://mt.sohu.com/20170404/n486411569.shtml |title =清明之计，向为24基地国防建设中献出生命的战友敬礼！  |author =  |date =2017-04-04  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。1982年7月，国防科委等部门合并组建为[[中华人民共和国国防科学技术工业委员会|中华人民共和国国防科学技术工业委员会]]（国防科工委），第二十四试验训练基地转隶国防科工委。1998年，国防科工委撤销，组建[[中国人民解放军总装备部|中国人民解放军总装备部]]，第二十四试验训练基地转隶总装备部。

2001年，第二十四试验训练基地撤编<ref name=xrxrmzf/>。
