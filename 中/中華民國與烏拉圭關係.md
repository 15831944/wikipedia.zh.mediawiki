'''中華民國與烏拉圭關係'''是指[[中華民國|中華民國]]與[[烏拉圭東方共和國|烏拉圭東方共和國]]之間的關係。1957－1988年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對烏拉圭的相關事務由'''[[駐阿根廷台北商務文化辦事處|駐阿根廷台北商務文化辦事處]]'''、'''{{link-en|駐哥倫比亞台北商務辦事處|Taipei Commercial Office in Bogotá, Colombia}}'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-186-b659f-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[烏拉圭|烏拉圭]]。]]
{{see also|中華民國外交|{{link-en|烏拉圭外交|Foreign relations of Uruguay}}}}
{{see also|中国驻乌拉圭大使列表{{!}}中華民國駐烏拉圭外交官列表|乌拉圭驻华大使列表{{!}}烏拉圭駐中華民國外交官列表}}
1957年11月15日，烏拉圭與中華民國建立公使級[[外交#外交上的承認|外交關係]]。於首都[[蒙特維多|蒙特維多]]設立'''中華民國駐烏拉圭東方共和國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]（由[[中国驻阿根廷大使列表|駐阿根廷大使]]兼任，後兼使改為專使）。<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_10.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1962年7月5日，建立大使級外交關係。公使館升格為'''中華民國駐烏拉圭東方共和國[[外交代表機構#大使馆|大使館]]'''。1966年3月，派駐[[特命全權大使|大使]]。<ref name="外交年鑑"/>

1985年2月24日，[[中華民國副總統|中華民國副總統]][[李登輝|李登輝]]率特使團前往烏拉圭參加[[烏拉圭總統列表|新任總統]][[胡利奧·瑪麗亞·桑吉內蒂|桑吉內提]]的{{link-en|就職典禮|Inauguration}}。並宣慰當地[[海外華人|僑民]]與駐在工作人員。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=0E0F1CC867B7DC4E |author = 公眾外交協調會 |date = 1985年2月24日 |title = 李副總統特使團赴烏拉圭離中正機場聲明 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=1A772EE466E13FC3 |author = 公眾外交協調會 |date = 1985年3月16日 |title = 登輝此次以特使身分赴烏拉圭共和國慶賀桑吉內提總統就職 |publisher = 中華民國外交部}}</ref>

1986年1月13日，{{link-en|烏拉圭眾議院|Chamber of Deputies of Uruguay}}議長{{link-en|馬契桑諾|Antonio Marchesano}}夫婦抵台訪問6天。期間拜會[[行政院長|行政院長]][[俞國華|俞國華]]、[[立法院長|立法院長]][[倪文亞|倪文亞]]等部門與人員。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=42F36DEF529A2AE2 |author = 公眾外交協調會 |date = 1986年1月12日 |title = 烏拉圭共和國眾議院議長馬契桑諾閣下暨夫人一行七人，訂本（元）月十三日來華訪問六天。 |publisher = 中華民國外交部}}</ref>5月18日，眾議院第一、第三副議長與眾議員，以及[[烏拉圭總統列表|總統]]宣傳顧問等人亦抵台訪問。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=8593A58D4EF2C53A |author = 公眾外交協調會 |date = 1986年5月17日 |title = 烏拉圭共和國眾議院訪華團一行十二人 |publisher = 中華民國外交部}}</ref>

1987年8月29日，烏拉圭前總統、駐[[巴拉圭|巴拉圭]]大使{{link-en|巴契哥|Jorge Pacheco Areco}}夫婦抵台訪問。期間拜會行政院長俞國華、外交部長[[丁懋時|丁懋時]]及其他政府官員。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=966E4F0CE3142ABC |author = 公眾外交協調會 |date = 1987年8月28日 |title = 烏拉圭前總統、現任駐巴拉圭共和國大使巴契哥 |publisher = 中華民國外交部}}</ref>

1988年2月3日，烏拉圭與[[中華人民共和國|中華人民共和國]]建交，4日，與中華民國斷交。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

1992年7月，於首都蒙特維多設立[[實質大使館|具大使館性質]]的'''駐烏拉圭[[臺北經濟文化代表處|臺北經濟辦事處]]'''，{{link-en|烏拉圭政府|Politics of Uruguay}}將該處視同[[國際組織|國際組織]]。<ref name="外交年鑑"/>但目前已關閉。

1993年9月，烏拉圭[[常駐聯合國代表|常駐聯合國代表]]在第48屆[[聯合國大會|聯合國大會]]的總辯論中，提出「會籍普遍化原則」，間接表示支持[[中國與聯合國#台灣人民代表權問題及台灣入聯運動|中華民國參與聯合國]]。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=01CE54E75B2C7C88 |author = 公眾外交協調會 |date = 1993年10月14日 |title = 聯大第四十八屆常會總辯論自本年九月廿七日至十月十三日止 |publisher = 中華民國外交部}}</ref>

1995年1月，台灣「豐盈十六號」[[魷|魷]]釣船於南大西洋[[公海|公海]]發生命案，3名[[中華人民共和國國籍法|中華人民共和國籍]]漁工涉嫌殺死[[中華民國籍|中華民國籍]]船員，後該船駛往烏拉圭首都蒙特維多靠岸並請求處理。因該案發生在公海，烏拉圭政府在經過初步偵訊後即依據[[國際法|國際法]]「[[司法管轄權|司法管轄權]]屬於船籍國」的規範，經由[[國際刑警組織|國際刑警組織]]通知[[中華民國警察|中華民國警方]]派員前往押解嫌犯，並於2月返抵台灣移送法辦。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=ED62A6D900019411 |author = 公眾外交協調會 |date = 1995年2月6日 |title = 我國刑事警察局透過外交部及駐外單位通力合作以及積極交涉聯繫 |publisher = 中華民國外交部}}</ref>

1999年1月，烏拉圭眾議院議長{{link-es|池洛博|Jaime Trobo}}抵台訪問。期間會見立法院長[[劉松藩|劉松藩]]、外交部長[[胡志強|胡志強]]及其他政府官員。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=1F125717FF9BD550 |author = 公眾外交協調會 |date = 1999年1月3日 |title = 烏拉圭眾議院議長池洛博，訂本（元）月四日抵華訪問七天。|publisher = 中華民國外交部}}</ref>11月，{{link-en|烏拉圭最高法院|Supreme Court of Uruguay}}院長抵台參加第42屆國際法官協會年會。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter1-6.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2015年，烏拉圭護民官署主席{{link-es|谷恩澤|Mirtha Guianze}}抵台訪問。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係 阿根廷篇〉 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境烏拉圭。]]
[[File:Visa_requirements_for_Uruguayan_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|烏拉圭簽證政策|烏拉圭公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往烏拉圭駐[[日本|日本]][[外交代表機構#大使馆|大使館]]申請簽證。<ref name="簽證及入境須知">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-186-b659f-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持有{{link-en|烏拉圭護照|Uruguayan passport}}的{{link-en|烏拉圭人口|Demographics of Uruguay|烏拉圭公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

===貿易===
2000年4月，[[經濟部國際貿易局|經濟部國際貿易局]]（國貿局）局長[[陳瑞隆|陳瑞隆]]率領「2000年[[南美洲#南美洲国家和地区|南美洲]]經貿投資訪問團」訪問[[巴拉圭|巴拉圭]]、[[巴西|巴西]]、[[阿根廷|阿根廷]]、[[智利|智利]]及[[烏拉圭|烏拉圭]]。<ref name="外交年鑑"/>

2011年3月16日，由國貿局委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）與40家廠商舉辦的「2011年南美洲拓銷團」繼[[秘魯|秘魯]]、智利及阿根廷之後，抵達最後1站烏拉圭，並於首都[[蒙特維多|蒙特維多]]進行貿易洽談會。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=557183&url=/NewsSearch.asp |author = 李建隆 |date = 2011年3月21日 |title = 「2011年南美洲拓銷團」烏拉圭貿易洽談會成果超乎預期劃下完美句點 |publisher = 經貿透視網}}</ref>

2018年，烏拉圭是中華民國第94大貿易夥伴、第102大[[進口|進口]]夥伴、第86大[[出口贸易|出口]]夥伴。出口至烏拉圭的金額為5,904萬4,656美元，年增4.576%；自烏拉圭進口的金額為1,854萬4,717美元，年增14.317%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）4,049萬9,939美元，年增0.650%。2017年，烏拉圭是中華民國第97大貿易夥伴、第105大進口夥伴、第84大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
