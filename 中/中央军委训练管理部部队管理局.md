'''中央军委训练管理部部队管理局'''，位于[[北京市|北京市]]，是[[中央军委训练管理部|中央军委训练管理部]]下属局，负责全军部队管理工作。

==沿革==
在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月组建[[中央军委训练管理部|中央军委训练管理部]]。中央军委训练管理部新设'''中央军委训练管理部部队管理局'''，负责军容风纪、营院管理等在内的部队管理工作<ref>{{Cite web |url = http://jz.81.cn/n2014/tp/content_7272285.htm |title = 首都驻军展开军容风纪纠治 重点是着装问题 |author =  |date =  2016-09-22|publisher =军报记者  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1788086 |title = 政知见：驻京部队纠治的50个突出问题是什么？ |author =  |date = 2017-09-08 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.163.com/17/0524/17/CL7HIATG000187VE.html  |title = 中央军委训练管理部：共享单车明起禁止进入首都地区驻军营院 |author =  |date = 2017-05-24  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。
