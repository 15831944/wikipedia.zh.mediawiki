'''中前圓唇元音'''是一個[[元音|元音]]，出現於一些口語中。就[[聲學|聲學]]的角度來說，這個元音實際上是'''中次前圓唇元音'''<ref>{{tsl|en|Geoff Lindsey|喬夫·琳賽}} (2013) [http://englishspeechservices.com/blog/the-vowel-space/ The vowel space], Speech Talk</ref> 。

由於在[[國際音標|國際音標]]中並沒有專門表示半閉{{IPA|[ø]}}與半開{{IPA|[œ]}}之間的前元音音標，習慣上通常將此音直接寫作{{angle bracket|{{IPA|ø}}}}。若要更精確地描述，可以加上變音符號，如{{angle bracket|{{IPA|ø̞}}}} 或 {{angle bracket|{{IPA|œ̝}}}}。

== 中前不突出元音 ==
{{IPA vowels|class=infobox}}
'''中前不突出元音'''是中前圓唇元音的一個變體，習慣上寫為{{angle bracket|{{IPA|ø̞}}}}或{{angle bracket|{{IPA|œ̝}}}}，本文為求簡潔與統一僅採用前者。國際音標中並沒有用來表示「不突出」（又稱「抿閉性」）的[[國際音標#變音符號|變音符號]]；然而這個將嘴唇「緊繃地聚合而不突出」的概念可以用{{IPAalink|β̞}}來表示。因此，此音可以寫成{{angle bracket|{{IPA|e̞͡β̞}}}} / {{angle bracket|{{IPA|ɛ̝͡β̞}}}}（{{IPA|[e̞]}} / {{IPA|[ɛ̝]}} 加上嘴唇不突出的符號）或{{angle bracket|{{IPA|e̞ᵝ}}}} / {{angle bracket|{{IPA|ɛ̝ᵝ}}}}（{{IPA|[e̞]}} / {{IPA|[ɛ̝]}}加上嘴唇不突出的符號）。

另外，用來表示[[元音圓唇度|不圓唇]]的變音符號{{angle bracket|{{IPA|  ͍ }}}}若與圓唇元音組合時，可以解釋為「嘴唇舒展」的意思。故此音還可以寫作{{angle bracket|{{IPA|ø͍˕}}}} / {{angle bracket|{{IPA|œ͍˔}}}}。

== 中前突出元音 ==
{{Infobox IPA
|above = 中前突出元音
|ipa symbol = ø̫˕
|ipa symbol2 = œ̫˔
|ipa symbol3 = ø̞ʷ
|ipa symbol4 = œ̝ʷ
|ipa symbol5 = e̞ʷ
|ipa symbol6 = ɛ̝ʷ
}}

語言學家{{tsl|en|J. C. Catford|約翰·坎尼森·卡特福德|卡特福德}}指出，許多擁有圓唇前元音與後元音的語言中會分別使用不同的[[元音圓唇度|圓唇度]]：即突出的後元音與不突出的前元音。然而，在[[北日耳曼語支|斯堪地那維亞諸語]]與一些其他的語言中，卻存在著突出的前元音。其中，[[瑞典語|瑞典語]]甚至有同一元音兩種不同圓唇度的情況（參見[[次閉次前圓唇元音|次閉次前圓唇元音]]）。

由於在國際音標中並沒有表示突出與不突出兩種圓唇度的音標，符號{{angle bracket|{{IPA|ø̞ʷ}}}}（中前圓唇元音加上表「突出」的符號）在此特別地被用作中前突出元音的音標。

聲學上，這個元音的位置介於較典型的中前不突出元音{{IPA|[ø̞]}}與中前不突出元音{{IPAblink|e̞}}之間。
