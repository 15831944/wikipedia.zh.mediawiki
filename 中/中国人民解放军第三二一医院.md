'''中国人民解放军第三二一医院'''，位于[[吉林省|吉林省]][[白城市|白城市]][[洮北区|洮北区]]海明西路81号，隶属[[沈阳联勤保障中心|沈阳联勤保障中心]]，为[[二级甲等医院|二级甲等医院]]。

==简介==
'''中国人民解放军第三二一医院'''始建于1964年<ref>赵继昌、吕书宝、王刚，励精图治谋发展 赤诚为兵献真情——解放军321医院全面建设掠影，人民军医2007(6)</ref>。第三二一医院每年为[[沈阳军区|沈阳军区]]体系部队上门送医送药，《[[解放军报|解放军报]]》曾以“十年保障百万兵”为题报道。医院眼科为“沈阳军区眼科治疗中心”。医院曾派医疗队参加[[唐山大地震|唐山大地震]]抗震救灾和[[1987年大兴安岭火灾|1987年大兴安岭扑火]]，先后获“功臣医疗队”、“烈火挡不住的医疗队”等称号，医院连续10年被沈阳军区联勤部评为“基层建设先进团单位”<ref>{{Cite web |url =  http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536705.htm|title =第三二一医院  |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。2014年10月14日，医院成功进行了一例全脑血管造影术诊疗，这标志着医院在[[沈阳军区总医院|沈阳军区总医院]]的帮带之下，已经能独立开展全脑血管造影术和颅内介入治疗<ref>{{Cite web |url =  http://www.mod.gov.cn/hospital/2014-10/29/content_4548300.htm|title = 解放军第321医院开辟介入治疗新领域 |author =  |date = 2014-10-29 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第三二一医院由[[中国人民解放军沈阳军区|中国人民解放军沈阳军区]]转隶[[沈阳联勤保障中心|沈阳联勤保障中心]]<ref>汪学潮、陈永林、高继宇，战友,你会用眼吗，解放军生活2017(3):78-78</ref>。
