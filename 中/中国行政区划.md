== 行政区划沿革 ==
中国行政区划的雏形从[[夏朝|夏朝]]建立[[国家|国家]]就开始了。《[[尚書_(書籍)|尚书]]》中的《[[禹貢|夏书·禹贡]]》记载，[[大禹|大禹]]的时候，天下分为九州，分别为[[冀州_(九州)|冀州]]、[[兖州_(古代)|兖州]]、[[青州_(古代)|青州]]、[[徐州_(古代)|徐州]]、[[扬州_(九州)|扬州]]、[[荆州_(古代)|荆州]]、[[梁州|梁州]]、[[雍州_(古代)|雍州]]、[[豫州|豫州]]。豫州是中心，其他州环绕豫州。

到了[[西周|西周]]时期，实行[[分封制|分封制]]，建立了[[诸侯|诸侯]]国。真正的地方行政区划制度则是萌芽发展于[[春秋时期|春秋时期]]，到[[秦朝|秦朝]]建立后才在全国范围内推行的。（參見：[[先秦行政区划|先秦行政区划]]）从秦朝起，中国历代王朝的地方行政区划制度大致可分为三个阶段：
# 秦汉魏晋南北朝时期，州、郡时代
# 隋唐五代宋辽金时期，道、路时代
# 元明清时期，行省时代

中国历代的行政区划，在不同的朝代同一政区其管辖范围并不见得相同，即使是同一朝代，也可能会发生变化。

=== 州郡時期（前221年－583年） ===
{{参见|秦朝行政区划|西汉行政区划|新朝行政区划|东汉行政区划|三国行政区划|西晋行政区划|东晋行政区划|南北朝行政區劃|隋朝行政區劃}}
[[秦朝行政区划|秦朝行政区划]]分为[[郡|郡]]、[[县|县]]两级，以郡统县。[[秦始皇|秦始皇]]在前221年分全国为[[三十六郡|36郡]]，到秦末增加到40多个郡。县共有1000多个，一个郡一般管十几到二三十个县。

到了[[汉朝|汉朝]]，有些郡分封给[[诸侯王|诸侯王]]，称为[[王国|王国]]。汉朝初年[[诸侯|诸侯]]王掌有王国的实权，一[[國_(中國古代制度)|国]]可以有几个郡。从[[汉景帝|汉景帝]]到[[汉武帝|汉武帝]]以后，撤销了[[诸侯王|诸侯王]]的统治权，规定一国只能管一郡，王国官吏由[[朝廷|朝廷]]直接任命，郡和国在[[行政区划|行政区划]]制度上已没有了区别，不同的只是名称而已（王国的[[税收|税收]]不交给朝廷）。[[汉武帝|汉武帝]]时，把全国的郡国分为13个[[刺史|刺史]]部，另外[[首都|首都]]附近的7个郡称为[[司隶校尉|司隶校尉]]部。这是一种监察区域，并不是行政区域，刺史只负责监察地方长官，不管地方行政。这些刺史部的名称大多以《[[禹贡|禹贡]]》、《职方》里的州名命名，所以当时人常称一部为一[[州|州]]。[[西汉|西汉]]后期，全国有103个郡、国。到[[东汉|东汉]]中期，又在边境的少数民族地区设立了[[属国|属国]]，相当于郡一级的政区。

[[黄巾起义|黄巾起义]]后，州由监察区变为行政区，成为郡以上的一级政区。这种格局到[[三国|三国]]时就固定下来了，维持了大约400年的时间。三国时，[[曹魏|曹魏]]有12州，[[孫吳|东吴]]有4州，[[蜀汉|蜀汉]]只有1州。[[西晋|西晋]]统一时共有19州，173郡国。西晋末年，增设两州，共有21州。之后州的辖区逐步缩小，到了[[南北朝|南北朝]]前期，共有五六十个州。到南北朝末期，州增加到300多个，郡增加到600多个。此时平均一个州只管2个郡，每个郡只管2、3个县。州郡县三级制的意义已经完全丧失了。所以到了后期，只有州刺史和县令长到任理事，郡的太守并不上任理事，实际上已经变成了州县二级制。[[隋文帝|隋文帝]]在[[开皇|开皇]]三年（583年）于是撤销了郡，只剩下州和县。

隋文帝废郡后，由于全国范围太大，由中央政府直接管辖近200个州非常困难。到了[[隋炀帝|隋炀帝]]时又将州改称郡，模仿汉武帝时的制度，设置了若干监察吏治的官员，分部巡查。但是由于缺少相关的详细资料，具体情况现在已经很难了解了。

=== 道路時期（627年－1276年） ===
{{参见|唐朝行政区划|五代行政区划|宋朝行政区划|遼朝行政区划|金朝行政区划}}

[[唐朝|唐朝]]初年，又将郡改称为州。当时有300多个州，由于朝廷难于管理，[[唐太宗|唐太宗]][[貞觀_(唐朝)|贞观]]元年（627年）又将全国分为十个道，每个道有时置官，有时不置官，也是监察性的。到了[[唐玄宗|唐玄宗]][[开元|开元]]年间，改为15道，设置[[采访处置使|采访处置使]]，相当于汉武帝时的刺史。另外，玄宗以后，还将一些地位特殊的州改称为[[府|府]]。至唐末共有十几个府。[[安史之乱|安史之乱]]以后，形成了以掌兵权的[[节度使|节度使]]作为地方行政长官的制度。一个节度使管几个州，其辖区也叫道，形成了道-州-县三级行政区划制度。全国分为40多个道，每道少则2、3州，多则10多州。原来的采访处置使已经撤销。

[[北宋|北宋]]初年，取消了节度使的实权，所有的州都直属中央。但是为了便于管理，到了[[宋太宗|宋太宗]]时，又在州之上设置了[[路_(行政区划)|路]]。宋朝时的路不归一个机构管辖，而是分属好几个机构，分管各自的职权。路的性质介于行政区和监察区之间的一种区划。[[宋朝|宋朝]]时有一个特别的现象。每一个州还有一个郡名。州名是正名，郡名则是别名。郡名大都沿袭唐朝时的旧名，改用新郡名的很少。新的州朝廷往往会赏赐一个郡名，没有郡名的州很少。宋朝时一些重要的州也升为了府，北宋末年已有三十几个府。到了[[南宋|南宋]]时，加上[[金朝|金朝]]的府共有50多个，是当时州的总数的1/7强。此外，还有军、监两种地位低于州的州级行政区划。

[[辽代|辽代]]全境分为5道，州一级的有府、州、军、城四种。金代全境分为20多路，州一级的初期有府、州、军三种，后来把军都升为了州，只剩下府和州。

=== 行省時期（1276年－1911年） ===
{{main|元朝行政區劃|明朝行政區劃|清朝行政區劃}}

從[[元朝|元朝]]開始，開始實行[[行省|行省]]制。它的全稱為某某等處[[行中書省|行中書省]]，簡稱某某行中書省或某某行省。元朝的行省制是沿襲自[[金朝|金朝]]的[[行尚書省|行尚書省]]。開始它和金朝一樣，只是一種中央政府派遣在外的臨時機構。由於長時間的[[戰爭|戰爭]]，行省也開始干預地方政務，逐漸變成了最高一級的地方行政區域。元朝初年的行省管轄範圍很大，改變也比較頻繁。到了中葉，開始穩定下來，將全國分為[[中書省|中書省]]直轄區（即腹里地區）、[[宣政院|宣政院]]轄地（今[[西藏|西藏]]），以及10個行中書省。省下有路、府、州、縣，路歸省管。府和州有的歸路管，有的歸省管，還有的州歸府管。縣有的歸路管，有的歸府管，有的歸州管。

到了[[明朝|明朝]]，[[明太祖|明太祖]]取消了行省，改為[[承宣布政使司|承宣布政使司]]。又將原來歸中書省直轄的地區改為京師直轄的地區，稱為[[直隸|直隸]]。 [[明成祖|明成祖]]以後，北京周圍的直轄區稱為北直隸，[[南京|南京]]周圍的直轄區稱為南直隸。 [[宣德|宣德]]以後，全國分為兩直隸，十三布政使司。但是民間一般還是將直隸或布政使司稱為省。

[[File:China_Proper.jpg|thumb]]

[[清朝|清朝]]初年，沿襲明制，改“承宣布政使司”為“[[布政使司|布政使司]]”，民間仍習慣稱為“省”。 [[康熙|康熙]]初年以前也只有15個省。到康熙六年，設為18個省，也就是所謂的“關內十八省”。另外在滿洲則設置[[奉天县_(中华民国)|奉天]]、吉林、黑龍江3個將軍轄區。新疆由[[伊犁將軍|伊犁將軍]]統轄。[[外蒙古|外蒙古]]喀爾喀四部由[[烏里雅蘇臺將軍|烏里雅蘇臺將軍]]、[[庫倫辦事大臣|庫倫辦事大臣]]各分管二部。[[唐努烏梁海|唐努烏梁海]]也由烏里雅蘇臺將軍管轄。[[科布多|科布多]]由[[科布多參贊大臣|科布多參贊大臣]]管轄。內蒙古採用札薩克制與[[盟|盟]][[旗|旗]]制度管理。西藏置[[駐藏大臣|駐藏大臣]]，青海置[[西寧辦事大臣|西寧辦事大臣]]管理。[[光緒|光緒]]十年（1884年）新疆建省，次年將原屬福建的台灣府升為[[臺灣省|臺灣省]]。光緒三十二年、三十三年間（1907年）分科布多置[[阿爾泰區域|阿爾泰]]，又將奉天、吉林、黑龍江三個將軍的轄區改為省。至清末，共有二十二省以及內蒙古、[[西套蒙古|西套蒙古]]、外蒙古、科布多、阿爾泰、青海、西藏等區域。

元明清以來，省以下的行政區劃各個時期都不相同。元朝時省下有路、府、州、縣，路歸省管。府和州有的歸路管，有的歸省管，還有的州歸府管。縣有的歸路管，有的歸府管，有的歸州管。明朝省下設府、州、縣。清朝省下設府、[[厅_(行政区划)|廳]]、[[州|州]]、縣。明清時的州，清朝的廳有的直屬於省，有的歸府管。元明時期的州有的有縣，有的不轄縣。清朝凡是屬於府的州一律不管縣。也就是說，元明清時期的州實際上就是比較重要的縣。清朝[[宣統|宣統]]末年則開始廢府。此外元明清也有道，但是作為省的派出機構，並不是行政區域。

== 中华人民共和国的行政区划（法定领土） ==
{{Main|中华人民共和国行政区划}}{{See also|香港行政区划|澳門行政区划}}
{{Divisions levels of China}}

理论上，[[中华人民共和国|中华人民共和国]]现在的行政区划基本上分为[[中国一级行政区|省]]、[[地|地]]、[[县|县]]、[[鄉_(中華人民共和國)|乡]]四级：
# 全国分为[[中国一级行政区|省]]、[[自治区|自治区]]、[[直辖市|直辖市]]和[[特别行政区|特别行政区]]四个类别的国家[[一级行政区|一级行政区]]；
# 省、自治区分为[[地级行政区|地级行政区]]（含[[副省级城市|副省级城市]]）；
#* [[地区|地区]]与[[自治州|自治州]]分为[[县|县]]、[[自治县|自治县]]和[[市|市]]；
#* 直辖市、[[地级市|地级市]]与[[副省级城市|副省级城市]]分[[县级行政区|县级行政区]]包括[[市辖区|市辖区]]、县、自治县与市等；
# 县、自治县分为[[鄉_(中華人民共和國)|乡]]、[[民族乡|民族乡]]、[[镇|镇]]等[[乡级行政区|乡级行政区]]，乡级为基层政权。

在行政区域管理实际中，现行的管理体制实际上沿用了国民政府那一套管理方式。除直辖市和特别行政区外，每个[[中国一级行政区|省]]、[[自治区|自治区]]行政区划仍旧按[[地市级行政区|地市级行政区]]（含副省级城市）、[[县级行政区|县级行政区]]（含副地司级行政区域）和乡镇（含副县级行政区域）三级管理，这也是[[中国大陆|中国大陆]]特色政治在行政区划管理上的体现。地市级行政区（含副省级城市）包括[[地级市|地级市]]、地区（以前为专区）、自治州和[[盟|盟]]（内蒙古），其机构设置上除了各有侧重以外其他大同小异。按照中华人民共和国的行政区划的[[行政级别|行政级别]]，最高行政长官也按照行政区的[[行政级别|行政级别]]对应。如“副省级城市”最高行政长官对应的行政级别为“副省级”，依此类推。

中华人民共和国现在包括4个中央直接管辖的直辖市，23个省，5个实行民族自治的自治区，还有2个特别行政区都是根据[[中国共产党|中国共产党]]前领导人[[邓小平|邓小平]]提出的[[一国两制|一国两制]]的概念所成立的。

传统上[[中华人民共和国|中华人民共和国]]的版图状似[[公鸡|公鸡]]，而[[中华民国|中华民国]]正式承认[[蒙古国|蒙古国]]独立前的版图因为状似[[秋海棠|秋海棠]]，因此成为一般人的通称。许多人認为只要将公鸡加上外蒙古，就是秋海棠，以下依据中华人民共和国与邻国之交界，由东北逆时针略述与中华民国版图之差异。當[[中華民國憲法|中華民國憲法]]規定疆域非經決議不得變更時，中華民國政府從不承認中华人民共和国與他國簽訂的國界協定。

* {{flag|朝鮮}}：1962年签订界约，将[[朝鲜民族|朝鲜民族]]视为民族源頭的「[[长白山|长白山]]」（白頭山）以南划予朝鮮民主主義人民共和國 。
* {{flag|俄羅斯}}：包括[[外東北|外東北]]（1858、1860年）、[[黑瞎子岛|黑瞎子岛]]（1945年）、[[江东六十四屯|江东六十四屯]]（1900年）、[[唐努乌梁海|唐努乌梁海]]（1944年）和[[外西北|外西北]]。1991年中华人民共和国承认除黑瞎子島外所有主要爭議領土的丧失，而黑瞎子岛則在2005年和俄罗斯平分解決領土爭議 。
* {{flag|蒙古国}}：1962年签订界约，[[諾門罕_(地名)|诺门罕谷地]]、[[察哈尔特别区|察哈尔]]北境[[達里岡愛牧場|達里岡愛牧場]]、新疆[[北塔山事件|北塔山]]划归蒙古国（当时是[[蒙古人民共和国|蒙古人民共和国]]）。
* {{flag|塔吉克}}：在[[苏联|苏联]]时期，中华人民共和国不承认英俄瓜分帕米尔，形成未定界。在苏联解体后，与塔吉克斯坦签约，放弃[[喷赤河|喷赤河]]以东、[[薩雷闊勒嶺|薩雷闊勒嶺]]以西的[[帕米尔高原|帕米尔]]北境，约收回1000平方公里。
* {{flag|阿富汗}}：1963年签订界约，承认阿富汗领有[[瓦罕帕米尔|瓦罕帕米尔]]与[[巴达克山|巴达克山]]地区主权。
* {{flag|巴基斯坦}}：1963年签订界约，放弃[[坎巨提|坎巨提]]的主权，[[喬戈里峰|喬戈里峰]]（K-2）共享。
* {{flag|尼泊尔}}：1961年签订界约，[[珠穆朗玛峰|珠穆朗玛峰]]（圣母峰）共享。
* {{flag|不丹}}：[[印度|印度]]占领[[藏南地区|藏南地区]]之后，将其西南一隅[[墨拉萨丁|墨拉萨丁]]地区送给不丹。
* {{flag|印度}}：由于1913年英国与西藏在[[西姆拉会议|西姆拉会议]]中，自行划定「[[麦克马洪线|麦克马洪线]]」，及新疆西南的[[阿克塞钦|阿克塞钦]]地区。印度主张拥有上述两地全境，因此爆发[[中印戰爭|边界战争]]，中华人民共和国在取胜后自行退兵，目前仍未签订界约。
* {{flag|缅甸}}：1960年签订界约，中华人民共和国放弃[[高黎贡山|高黎贡山]]以西7万平方公里土地（[[江心坡|江心坡]]），放弃[[南坎|南坎]]、[[果敢|果敢]]等縣城，收回[[片马镇|片马]]、[[班老乡|班老]]等地区。
* {{ROC|台灣地區}}：中華民國政府主张，[[台灣|台灣]]、[[澎湖|澎湖]]自1945年被[[國民政府|國民政府]]所光復，與金門、馬祖列島同屬於中華民國[[台灣地區|臺澎金馬地區]]的一部分，因[[兩岸分治|兩岸分治]]造成主權治權爭議，臺澎金馬不屬於[[中華人民共和國中央人民政府|中華人民共和國中央人民政府]]實際管轄。
