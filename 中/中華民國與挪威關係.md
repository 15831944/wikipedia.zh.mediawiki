'''中華民國與挪威關係'''是指[[中華民國|中華民國]]與[[挪威王國|挪威王國]]之間的關係。兩國於1913－1950年有官方[[外交#外交上的承認|外交關係]]，斷交後直至1980年代，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構，但挪威方面已於2004年關閉；中華民國方面則在2017年關閉，對挪威的相關事務由'''{{Link-en|駐瑞典臺北代表團|Taipei Mission in Sweden}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-65-81a85-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[挪威|挪威]]。]]
{{see also|中華民國外交|{{link-en|挪威外交|Foreign relations of Norway}}}}
{{see also|中国驻挪威大使列表{{!}}中華民國駐挪威外交官列表|{{link-en|挪威駐華大使列表|List of diplomats of Norway to China|挪威駐中華民國外交官列表}}}}
1913年10月9日，挪威[[外交#外交上的承認|外交承認]]中華民國[[北洋政府|北洋政府]]。

1920年5月27日，建立公使級外交關係。於首都[[奧斯陸|奧斯陸]]設立'''中華民國駐[[挪威王國|挪威王國]][[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]（[[中国驻瑞典大使列表|駐瑞典公使]]兼任）。

1928年11月12日，隨著[[國民革命軍北伐|國民革命軍北伐]]接近尾聲，挪威外交承認中華民國[[國民政府|國民政府]]。

1940年8月18日，因{{link-en|納粹德國佔領挪威|German occupation of Norway}}而閉館。

1943年7月17日，升格為大使級外交關係。公使館升格為'''中華民國駐挪威王國[[外交代表機構#大使馆|大使館]]'''（原設於[[英國|英國]]首都[[倫敦|倫敦]]，1945年5月，[[第二次世界大戰歐洲戰場的結束|納粹德國投降]]後遷回挪威），並派駐[[特命全權大使|大使]]。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1950年1月7日，與中華民國斷交。<ref name="103年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/09.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>11月30日，挪威在聯合國安全理事會上否決中共代表[[伍修權|伍修權]]等提出之「美國侵略台灣案」<ref>{{cite conference |title=聯合國安全理事會正式紀錄 第七十二號 第五三0次會議 |date=1950-11-30|publisher=[[聯合國安全理事會|聯合國安全理事會]] |location=[[紐約|紐約]]發拉星草場 |url = http://www.un.org/en/ga/search/view_doc.asp?symbol=S/PV.530&referer=/english/&Lang=C |accessdate=2015-08-24 |language=zh-hant}}</ref>。

1980年8月26日，於首都奧斯陸設立[[實質大使館|具大使館性質]]的'''駐挪威台北商務處'''（Taipei Trade Center）。1992年12月17日，更名為'''駐挪威[[臺北經濟文化辦事處|臺北經濟文化辦事處]]'''（Taipei Economic and Cultural Office）。1999年10月16日，再更名為'''{{link-en|駐挪威台北代表處|Taipei Representative Office in Norway}}'''（Taipei Representative Office in Norway）。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-4.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref name="103年外交年鑑"/><ref>{{Cite web |url = http://www.roc-taiwan.org/no/post/1003.html |title = 代表處簡介 |publisher = 中華民國駐外單位聯合網站}}</ref>2017年9月30日，代表處暫停運作，由'''{{Link-en|駐瑞典臺北代表團|Taipei Mission in Sweden}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>[http://news.rti.org.tw/news/detail/?recordId=359293 外交部：駐吉達、關島、挪威3館暫停運作]</ref>

1989年10月23日，在首都[[台北市|台北]]設立同等性質的'''挪威商務臺北[[辦事處|辦事處]]'''（Norwegian Trade Office, Taipei）。1992年11月，更名為'''挪威貿易委員會台北辦事處'''（Norwegian Trade Council）。1994年1月獲授權簽發赴挪威觀光簽證，1997年7月將簽發地點改為台北，並直接將簽證貼於[[中華民國護照|中華民國護照]]內頁。2002年6月，因經費關係精簡人事，撤回辦事處的[[挪威人|挪威籍]]人員，業務並改由駐東京大使館督導。2004年1月1日，辦事處關閉，原挪威簽證業務轉移至{{link-en|丹麥商務辦事處|Trade Council of Denmark, Taipei|丹麥駐台北辦事處}}。<ref name="86年外交年鑑"/><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2004/08/08_01_04.htm |title = 《中華民國93年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref name="103年外交年鑑"/><ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 國家基本資料表 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

2016年1月，挪威重申「[[一个中国|一个中国]]」原则，对于[[台湾|台湾]]的国籍登记一律标示[[中华人民共和国|China]]。{{link-en|挪威外交部|Ministry of Foreign Affairs (Norway)}}[[發言人|發言人]]表示：「[[挪威政治|挪威政府]]立場很清楚，對於一中政策並沒有改變，挪威也不承認台灣（There is no change in Norway's One China Policy. Norway do not recognize Taiwan）。」[[中華民國外交部|中華民國外交部]]發言人則表示：「對於挪威政府不當登錄我國人國籍情況，我駐挪威代表處將持續積極進洽挪威政府正視改善。」<ref>{{cite web|title=挪威外交部：一直坚持一个中国的立场 不会改变 |url=http://www.appledaily.com.tw/realtimenews/article/politics/20160121/780715/ |publisher=[[苹果日报_(台湾)|苹果日报]] |date=2016-01-21 |accessdate=2016-10-03}}</ref>2017年3月，台灣留學生向{{link-no|挪威移民局|Utlendingsdirektoratet}}提起[[訴願|訴願]]，挪威政府最後以「此一註記並未對當事人在挪威的權利和義務造成影響」為由駁回。因此，這些留學生決定聯合在挪威居留的台灣人一起對挪威政府提起[[訴訟|訴訟]]。<ref>{{Cite web |url = https://www.nownews.com/news/20180808/2798853 |author = 侯姿瑩 |date = 2018年8月8日 |title = 台留學生為正名要告挪威政府 外交部提供協助 |publisher = [[NOWnews今日新聞|NOWnews今日新聞]]}}</ref>

===貿易===
2017年，挪威是中華民國第56大貿易夥伴、第46大[[進口|進口]]夥伴、第55大[[出口贸易|出口]]夥伴。出口至挪威的金額為1億7,251萬8,189[[美元|美元]]，年增2.967%；自挪威進口的金額為2億8,734萬7,381美元，年減14.147%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）1億1,482萬9,192美元，年減31.302%。※2016年，挪威是中華民國第53大貿易夥伴、第43大進口夥伴、第54大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
根據經濟部投資審議委員會統計，1996年－2014年，[[臺灣商人|臺商]]對挪威的總投資金額為120萬美元，計有2件。

雙邊定期舉行臺挪經濟合作會議、資通訊科技研討會；在當地也已成立挪威台灣商會。<ref name="98年外交年鑑"/><ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/no/pdf/104-53Norway.pdf |date = 2015年8月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

===犯罪事件===
2016年1月，國際[[詐騙集團|詐騙集團]]利用2家在[[台灣的銀行|台灣的銀行]]開設[[國際金融業務分行|OBU]]（Offshore Banking Unit，境外銀行）人頭[[銀行帳戶|帳戶]]，再入侵挪威的被害公司[[電腦|電腦]]，假冒公司[[CEO|CEO]]（執行長），發[[電子郵件|電子郵件]]給[[會計|會計]]部門行騙，謊稱要求[[匯款|匯款]]給客戶，其中有家挪威公司被詐騙5,000萬[[歐元|歐元]]。其公司事後向[[中華民國警察|中華民國警方]]報案，並協請攔阻詐騙款項，經調查，這些匯入的款項當天便遭轉走，難追其下落。<ref>{{Cite web |url = http://news.ltn.com.tw/news/society/paper/968000 |author = 邱俊福 |date = 2016年3月14日 |title = 假冒CEO 詐挪威公司18億 |publisher = [[自由時報|自由時報]]}}</ref>

2008年，台灣自挪威引進[[導盲犬|導盲犬]]，挪威導盲犬學校並捐贈訓育率100%的導盲種犬，以協助[[台灣導盲犬協會|台灣導盲犬協會]]培育種犬。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2008/html/08-1-4.htm |title = 《中華民國97年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2011年8－9月，{{link-en|駐挪威台北代表處|Taipei Representative Office in Norway}}舉辦第1屆[[台灣電影|台灣電影]]節。<ref name="100年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2011/2.1.4.pdf |title = 《中華民國100年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2012年10月，[[國立臺北大學|國立臺北大學]]與[[奧斯陸大學|奧斯陸大學]]合辦第1屆臺挪社會[[論壇|論壇]]。<ref name="101年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2012/2.1.4.pdf |title = 《中華民國101年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2012年12月，挪威臺灣之友協會成立。<ref name="101年外交年鑑"/>

2016年，有6名挪威籍人士獲核發[[中華民國教育部|中華民國教育部]]的[[中華民國國語|華語文]]獎學金。<ref name="105年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>
