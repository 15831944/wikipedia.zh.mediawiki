'''中華民國與土耳其關係'''是指[[中華民國|中華民國]]與[[土耳其共和國|土耳其共和國]]之間的關係。兩國於1934－1971年有官方[[外交#外交上的承認|外交關係]]，斷交後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構，雙方在經濟文化等實質關係持續穩定發展。<ref name="代表處簡介"/>

==政治==
===外交===
{{see also|中華民國外交|{{link-en|土耳其外交|Foreign relations of Turkey}}}}
{{see also|中国驻土耳其大使列表{{!}}中華民國駐土耳其共和國外交官列表|土耳其驻华大使列表{{!}}土耳其共和國駐中華民國外交官列表}}
1934年4月4日，簽署《中國土耳其友好條約》，建立公使級[[外交#外交上的承認|外交關係]]，<ref name="條約協定查詢系統"/>於首都[[安卡拉|安卡拉]]設立'''中華民國駐土耳其共和國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

[[中華民國國民政府主席|中華民國國民政府主席]][[蒋中正|蒋中正]]赠送一張亲笔签名軍裝照給被譽為[[土耳其历史|現代土耳其]]的肇建者、[[国父|国父]]（Atatürk，意指「土耳其之父」）[[凯末尔|凯末尔]]，照片至今仍保存於{{link-en|凯末尔紀念館|Anıtkabir}}内。<ref name="中土关系">{{Cite web|url=http://euroasia.cass.cn/news/408445.htm |author=昝涛 |date=2010年 |title=中土关系及土耳其对中国崛起的看法 |publisher=[[中国社会科学院|中国社会科学院]]俄罗斯东欧中亚研究所 |deadurl=yes |archiveurl=https://web.archive.org/web/20141225080207/http://euroasia.cass.cn/news/408445.htm |archivedate=2014-12-25 }}</ref>

1945年1月1日，駐土耳其公使館升格為'''中華民國駐土耳其共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，由於土耳其政府在[[冷戰|冷戰]]時维持强烈的[[反共|反共]]立场，故仍與中華民國維持外交關係。中国則公开表示支持土耳其的[[左派|左派]]运动<ref name="中土关系"/>。[[阿德南·曼德列斯|阿德南·曼德列斯]]執政时期，中華人民共和國不断攻擊土耳其為“[[帝国主义|帝国主义]]走狗”，而土耳其輿論亦塑造了中國的負面形象<ref name="中土关系"/>。

1960年1月29日，土中友好協會在土耳其安卡拉成立。<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|654}}

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[土耳其|土耳其]]。]]
1971年8月4日，土耳其與[[中華人民共和國|中華人民共和國]]建交。8月5日，與中華民國斷交。10月24日，土耳其在[[聯合國大會|聯合國大會]]上就[[中國與聯合國#代表權問題|中國代表權]]問題，與絕大多數[[中東#國家及地區|中東]]及[[歐洲國家和地區列表|歐洲國家]]一樣，支持[[中華人民共和國|中華人民共和國]]取得「[[中國|中國]]」席次。

2001年、2005年，中華民國外交部長[[田弘茂|田弘茂]]、[[陳唐山|陳唐山]]先後訪問土耳其。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2001/Chap3/3-1-2.htm |title = 《中華民國90年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2005/WebPages/MICO/08/08_01_02.htm |title = 《中華民國94年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2018年2月3日，台北市長[[柯文哲|柯文哲]]率團訪問土耳其，成為首個到訪該國的[[中華民國自由地區鄉鎮市區列表|台灣縣市]]首長。在抵達第1大城[[伊斯坦堡|伊斯坦堡]]後於飯店接受土耳其政要宴請。因為[[台灣|台灣]]在[[敘利亞內戰難民|敘利亞難民問題]]上給予土耳其很大的協助，因此晚宴上[[正义与发展党_(土耳其)|該國執政黨]][[土耳其大国民议会|國會議員]]表達感謝，桌上還並列[[中華民國國旗|中華民國國旗]]與[[土耳其國旗|土耳其國旗]]。由於[[土耳其政治|土耳其政府]]計畫在[[台北市|台北市]]捐贈一座[[清真寺|清真寺]]，相關細節的商談也是此行出訪的重點。<ref>{{Cite web |url = https://www.ettoday.net/news/20180204/1107049.htm |author = 陳家祥 |date = 2018年2月4日 |title = 國旗登宴會！台灣協助解決難民問題樂勝歐洲　土耳其超感謝 |publisher = [[ETtoday新聞雲|ETtoday新聞雲]]}}</ref><ref name="首位訪土耳其市長">{{Cite web |url = https://www.ettoday.net/news/20180205/1107531.htm |author = 陳家祥 |date = 2018年2月5日 |title = 首位訪土耳其市長　柯文哲：城市外交只是台北的一小部分 |publisher = ETtoday新聞雲}}</ref>4日，參訪[[慈濟基金會|臺灣佛教慈濟基金會]]設立的敘利亞難民義診醫院。<ref>{{Cite web |url = https://www.ettoday.net/news/20180205/1107532.htm |author = 陳家祥 |date = 2018年2月5日 |title = 參觀敘利亞難民醫院　柯文哲：台灣人生在福中不知福 |publisher = ETtoday新聞雲}}</ref>5日，前往{{link-en|科曲大學|Koç University}}（Koç University）參觀，並討論[[產學合作|產學合作]]方案。<ref>{{Cite web |url = https://www.ettoday.net/news/20180205/1108155.htm |author = 陳家祥 |date = 2018年2月5日 |title = 留學為工作不為學位　柯文哲：土耳其、波蘭比歐美更好 |publisher = ETtoday新聞雲}}</ref>此外，原本規劃要去首都安卡拉，但最終取消。<ref name="首位訪土耳其市長"/>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#电子签证|電子簽證]]的方式入境土耳其。]]

{{see also|土耳其簽證政策|土耳其公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可上網申請單次[[簽證#电子签证|電子簽證]]入境土耳其，停留最多30天。若申請多次入境、就學、工作等簽證須先至土耳其駐外領事館網站預約，再親自前往申辦。經土耳其轉機且未入境，無須申辦[[過境簽證|過境簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-49-60be6-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ElectronicVisa.aspx?n=6F246645F10311D7&AreaSN=7F220D7E656BE749 |title = 電子簽證資訊 |publisher = 中華民國外交部}}</ref>

中華民國政府對持[[土耳其護照|土耳其護照]]的[[土耳其人口|土耳其公民]]給予[[中華民國簽證政策#互惠國相互提供落簽待遇|互惠國相互提供落簽待遇]]，停留最多30天，免簽證費。也可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期，免簽證費。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-4490-dce26-1.html |title = 電子簽證的申請費用？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將土耳其按地區歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往，與'''紅色警戒'''：不宜前往，宜儘速離境（以上截至2017年4月18日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-4-c9935-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於土耳其第1大城[[伊斯坦堡|伊斯坦堡]]設立'''台灣貿易中心'''<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#EUROPE |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>（前身為1987年設立的遠東貿易服務中心駐伊斯坦堡辦事處<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-2.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>）。[[經濟部國際貿易局|經濟部國際貿易局]]則在首都[[安卡拉|安卡拉]]設立'''駐土耳其代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

[[臺灣商人|台商]]在土耳其為數不多，大部分從事[[國際貿易|國際貿易]]，引進[[台灣|台灣]]或[[中华人民共和国|中國大陸]]產品銷售，少數經營[[餐館|餐廳]]。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/tr/pdf/103-61Turkey.pdf |date = 2014年12月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

2013年，經濟部國際貿易局，挑選最具拓銷潛力的22國26城市推動執行「貿易尖兵任務」，其中[[東歐|東歐]]7國的[[新興市場|新興市場]]就包括[[土耳其|土耳其]]。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622045&url=/NewsSearch.asp |author = 中華民國對外貿易發展協會 |date = 2013年10月30日 |title = 貿易尖兵勇闖22國26城市 新興市場商機無窮 |publisher = 經貿透視網}}</ref>

2015年1月11日，[[高盛|高盛]]發表報告指出，台灣、土耳其及[[印度|印度]]並列2015年度[[世界政區索引|全球]]3大表現最佳新興市場。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 經濟回顧與展望 |publisher = 中華民國對外貿易發展協會}}</ref>

2018年，土耳其是中華民國第32大貿易夥伴、第45大[[進口|進口]]夥伴、第22大[[出口贸易|出口]]夥伴。出口至土耳其的金額為13億7,119萬2,711[[美元|美元]]，年減12.116%；自土耳其進口的金額為3億7,031萬0,679美元，年增38.482%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）10億0,088萬2,032美元，年減22.582%。2017年，土耳其是中華民國第29大貿易夥伴、第51大進口夥伴、第21大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =經濟部國際貿易局}}</ref>

出口至土耳其的項目以高價位之精密[[機器|機械儀器]]為主；自土耳其進口的項目則以[[原料|原料]]及初級加工品為主。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

===投資===
根據經濟部投資審議委員會統計，1952年－2013年，臺商赴土耳其總投資金額為463萬美元，計有9件。主要投資：[[貿易業|貿易業]]、[[電腦|電腦]]／[[電子產品|電子產品]]／[[光学仪器|光學製品]]、[[汽車零組件列表|汽車及其零件]]、[[金屬|金屬]]製品、[[化學品列表|化學材料]]、[[紡織業|紡織業]]；另根據土耳其台商會調查，截至2013年，台商在土耳其總投資金額為950萬美元，計有27件，大多是[[股本|股本]]投資，未實際派員常駐土耳其。[[鴻海科技集團|鴻海科技集團]][[子公司|子公司]]富士康與[[惠普|惠普]]投資6,000萬美元，每月裝配生產[[筆記型電腦|筆記型電腦]]20萬台，是台商在土耳其最大的[[製造業|製造業]]投資案；[[緯創資通|緯創資通]]則在伊斯坦堡設立電腦維修廠，以服務電腦用戶群。<ref name="投資環境簡介"/>台商的資金來源可以是第3地，如鴻海富士康（[[捷克|捷克]]）、[[東元電機|東元電機]]（[[蓋曼群島|蓋曼群島]]）、[[陽明海運|陽明海運]]（[[新加坡|新加坡]]）等。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 投資環境資訊 |publisher = 中華民國對外貿易發展協會}}</ref>

根據經濟部投資審議委員會統計，1952年－2013年，土耳其廠商赴臺總投資金額約237萬美元，計有37件。主要投資：貿易業、[[皮革|皮革]]／[[毛皮|毛皮]]製品、電腦／電子產品／光學製品、[[電力|電力]]設備、[[住宿|住宿]]及[[餐飲業|餐飲業]]、[[金融|金融]]及[[保险公司|保險業]]。<ref name="投資環境簡介"/>

目前成立的組織有：土耳其台商會、<ref name="國家基本資料表">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 國家基本資料表 |publisher = 中華民國對外貿易發展協會}}</ref>臺灣－土耳其文化經濟協進會。<ref name="投資環境簡介"/>

===援助===
1990年8月，[[入侵科威特|伊拉克入侵科威特]]引發中東危機後，[[中華民國政府|中華民國政府]]於9月提供200萬[[美元|美元]]現款與300萬美元的[[医疗器械|醫療器材]]援助土耳其用在該地區難民，並協助紓解財政困境。1991年1月16日，由台灣駐土耳其代表於首都[[安卡拉|安卡拉]]致贈給{{link-en|土耳其衛生部|Ministry of Health (Turkey)}}次長；<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=C1A242E46E9C1F22 |author = 公眾外交協調會 |date = 1991年1月21日 |title = 自伊拉克入侵科威特事件發生並引發中東危機以來 |publisher = 中華民國外交部}}</ref>3月，{{link-en|1991年伊拉克暴動|1991 uprisings in Iraq|伊拉克爆發內戰}}後，其國內上百萬[[庫德族|庫德族]]難民湧入土耳其及[[伊朗|伊朗]]，中華民國政府提供1,000萬美元援助庫德族難民，其中，有200萬美元捐贈予{{link-en|土耳其紅新月會|Turkish Red Crescent}}。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=2DA8A151B2D9BC0E |author = 公眾外交協調會 |date = 1991年9月18日 |title = 中華民國政府鑒於伊拉克自本年三月爆發內戰後 |publisher = 中華民國外交部}}</ref>

1999年9月21日，臺灣中部發生[[921大地震|921大地震]]，{{link-en|土耳其搜救隊|AKUT Search and Rescue Association}}（AKUT）於9月22日（約震後50小時）飛抵台灣協助救難工作<ref>1999-09-23/《[[聯合報|聯合報]]》/10版/[[921大地震|集集大震]]特別報導</ref>，是最早抵達台灣的國際救難隊。921大地震前一個月的1999年8月17日，土耳其西北部甫發生[[1999年伊茲密特地震|大地震]]造成嚴重傷亡，台灣[[慈濟基金會|慈濟]]發起募款援助。在921大地震發生後50小時，土耳其與德國的搜救團隊和消防局救難人員在[[彰化縣|彰化縣]][[員林鎮|員林鎮]]龍邦富貴名門大樓，救出當地受困民眾。<ref>{{cite news|url=http://www.cna.com.tw/Topic/NewsTopic/352-1/201209210042-1.aspx |language=zh-tw |title=土搜救隊員憶921心有餘悸 |publisher=中央通訊社 | date=2012-09-21 |accessdate=2013-03-30 }}</ref>

援助土耳其境內[[敘利亞內戰難民|敘利亞難民]]計畫，協助改善[[難民營|難民營]]衛生環境、興建難民學校及[[電腦室|電腦教室]]、援贈義診中心之醫療器材等。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/f9783c18-6276-407e-8458-51dbc00060e7.pdf |author = 國際合作及經濟事務司 |date = 2016年4月22日 |title = 國際合作發展事務104年度報告（參、104年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/ab7236a0-e636-4439-b9ce-c6b86ac4339e.pdf |author = 國際合作及經濟事務司 |date = 2017年4月28日 |title = 國際合作發展事務105年度報告（參、105年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref>
