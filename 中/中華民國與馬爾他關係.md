'''中華民國與馬爾他關係'''是指[[中華民國|中華民國]]與[[馬爾他共和國|馬爾他共和國]]之間的關係。兩國於1967－1972年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對馬爾他的相關事務由'''駐[[義大利|義大利]][[台北经济文化代表处|台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref name="駐館位置">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-158-c65c0-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[馬爾他|馬爾他]]。]]
{{see also|中華民國外交|{{link-en|馬爾他外交|Foreign relations of Malta}}}}
1967年7月19日，馬爾他與中華民國建立大使級[[外交#外交上的承認|外交關係]]，並派駐[[特命全權大使|大使]]（由[[中国驻意大利大使列表|駐義大利]]或[[中華民國駐教廷大使列表|駐羅馬教廷大使]]兼任）。

1971年8月13日，簽署《[[中華民國政府|中華民國政府]]與[[馬爾他政治|馬爾他政府]][[獸醫|獸醫]]合作換函》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e9%a6%ac%e7%88%be%e4%bb%96&tycountry_e=%e9%a6%ac%e7%88%be%e4%bb%96&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

1971年10月25日，馬爾他支持[[中國與聯合國#代表權問題|中華民國續留聯合國]]，即反對《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，是唯一投下反對票的[[聯合國會員國列表|聯合國歐洲會員國]]。

1972年1月31日，與中華民國斷交。2月24日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以「[[歐洲聯盟簽證政策|申根區免簽證]]」的方式入境馬爾他。]]
[[File:Visa_requirements_for_Maltese_citizens.png|thumb]]的[[馬爾他護照|馬爾他護照]]之[[馬爾他公民|馬爾他公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}

歐洲[[申根區|申根區]]給予持有載明[[中華民國國民身份證|身份證]]字號的[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以[[歐洲聯盟簽證政策|申根區免簽證]]入境，停留日數與申根區合併計算，每6個月期間內總計可停留90天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-158-c65c0-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_FAQ.aspx?c=AD6908DFDDB62656&n=C1B19EF0ACDD3C91&sms=867F5398A49F758A |title = 歐盟免申根簽證常見問答集 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

[[馬爾他公民|馬爾他公民]]持有[[歐盟護照|歐盟護照]]者也可以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將馬爾他歸類為'''灰色警示'''：提醒注意（2016年11月4日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-261-82480-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|馬耳他經濟{{!}}馬爾他經濟}}
===貿易===
2017年，馬爾他是中華民國第104大貿易夥伴、第103大[[進口|進口]]夥伴、第97大[[出口贸易|出口]]夥伴。出口至馬爾他的金額為3,178萬1,852[[美元|美元]]，年增17.251%；自馬爾他進口的金額為1,805萬7,931美元，年減19.903%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1,372萬3,921美元，年增200.908%。※2016年，馬爾他是中華民國第101大貿易夥伴、第97大進口夥伴、第98大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
