'''中華民國與聖多美普林西比關係'''是指[[中華民國|中華民國]]與[[聖多美普林西比民主共和國|聖多美普林西比民主共和國]]（通稱'''聖普'''、'''聖多美'''）之間的關係。兩國於1997－2016年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對聖多美普林西比的相關事務由'''{{link-en|駐葡萄牙台北經濟文化中心|Taipei Economic and Cultural Centre, Lisbon, Portugal}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-251-8c594-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，[[聖多美普林西比|聖多美普林西比]]當時屬於[[葡萄牙帝國#範圍|葡萄牙殖民地]]，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
{{see also|中華民國外交|{{link-en|聖多美普林西比外交|Foreign relations of São Tomé and Príncipe}}}}
{{see also|中華民國駐聖多美普林西比大使列表{{!}}中華民國駐聖多美普林西比外交官列表|聖多美普林西比駐中華民國大使列表{{!}}聖多美普林西比駐中華民國外交官列表}}
1997年5月6日，聖多美普林西比主動要求與中華民國[[外交#外交上的承認|建立外交關係]]，由兩國外交部長在[[台北市|台北]]簽署《建交公報》。5月24日，於首都[[聖多美|聖多美]]設立'''中華民國駐聖多美普林西比民主共和國[[外交代表機構#大使馆|大使館]]'''，由[[中華民國駐布吉納法索大使列表|駐布吉納法索大使]]兼任。7月11日，[[中華人民共和國|中華人民共和國]]與聖多美普林西比斷交。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-3.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>1998年10月，中華民國派駐專任[[特命全權大使|大使]]。<ref name="87年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1998/section_3/page3-1-3.htm |title = 《中華民國87年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2000年3月，於首都台北設立'''聖多美普林西比民主共和國駐中華民國大使館'''，並派駐大使。<ref name="89年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_7.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2016年12月21日，聖多美和普林西比與中華民國斷交。[[中華人民共和國外交部|中華人民共和國外交部]]表示讚賞，並歡迎聖普回到「[[一個中國|一個中國]]」原則的正確道路。同日，[[中華民國外交部|中華民國外交部]]也宣布與聖普斷交<ref>[https://udn.com/news/story/10677/2182390 台聖邦交19年今中斷 中國：歡迎回到「一個中國」],2016.12.21 [[聯合新聞網|聯合新聞網]]</ref><ref>[http://www.mofa.gov.tw/News_Content.aspx?n=8742DCE7A2A28761&s=55927181C55F2CF6 中華民國政府基於維護國家尊嚴，決定自即日起終止與聖多美普林西比民主共和國之外交關係],2016.12.21 中華民國外交部</ref>。22日，駐聖普大使館舉行[[中華民國國旗|降旗]]儀式<ref>{{Cite web |url = https://www.mirrormedia.mg/story/20161222soc001/ |author = 社會組 |date = 2016年12月22日 |title = 【台聖斷交】獨家直擊 我駐聖大使館降下國旗 |publisher = [[鏡周刊|鏡周刊]]}}</ref>。26日，中華人民共和國與聖普恢復外交關係。<ref>{{Cite web |url = http://www.cna.com.tw/news/firstnews/201612265009-1.aspx |author = 尹俊傑 |date = 2016年12月26日 |title = 中國大陸與聖多美普林西比復交 |publisher = [[中央通訊社|中央通訊社]]}}</ref>

斷交前的2016年3月，中華民國外交部亞西及非洲司長陳俊賢在[[立法院|立法院]]備詢時已透露警訊，表示聖多美普林西比可能是「黃燈」，正「注意當中」。<ref>{{Cite web |url = https://udn.com/news/story/1/2182407 |author = 許依晨 |date = 2016年12月21日 |title = 3月外交部已警訊：聖多美普林西比「亮黃燈」 |publisher = 聯合新聞網}}</ref>斷交當日，外交部長[[李大維_(外交官)|李大維]]召開記者會證實聖多美普林西比曾要求金援，且這數字是天文數字（2億[[美元|美元]]），因[[中華民國政府|中華民國政府]]認為不能再走金錢外交而拒絕。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/1923578 |author = 綜合報導 |date = 2016年12月21日 |title = 外交部爆斷交原因！聖多美索金援64億被拒 |publisher = [[自由時報|自由時報]]}}</ref>

[[File:Visa_requirements_for_Santomean_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]

持{{link-en|聖多美普林西比護照|Santomean passport}}的{{link-en|聖多美普林西比人口|Demographics of São Tomé and Príncipe|聖普公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

===貿易===
2017年，聖多美普林西比是中華民國第222大貿易夥伴、第214大[[進口|進口]]夥伴、第219大[[出口贸易|出口]]夥伴，在[[非洲国家和地区列表|非洲國家]]排名倒數第2、3、2；在[[世界政區索引|世界經濟體]]倒數第17、23、12。出口至聖普的金額為1萬8,946[[美元|美元]]（僅5、7月有出口額），年減97.362%；自聖普進口的金額為1萬1,677美元，年減92.787%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）7,269美元，年減98.693%。※2016年，聖多美普林西比是中華民國第189大貿易夥伴、第183大進口夥伴、第182大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===援助===
1998年6月，兩國簽署《[[醫療|醫療]]合作特別協定》，中華民國每年提供醫療援款給予聖多美普林西比用於採購[[醫療設備|醫療設備]]及[[藥品|藥品]]、聘請當地及國外醫事人員、派遣醫事人員赴國外深造。7月，中華民國派駐醫療團。其後又派駐抗[[瘧疾|瘧疾]]顧問團執行「瘧疾防治計畫」，其發生率已由2003年的47%逐年降至2015年約1%；1999年1月，派駐[[農業技術|農業技術]]團，並於10月簽署《農業技術合作協定》，其後推動「振興農村[[永續發展|永續發展]]計畫」，改善農村社區[[灌溉|灌溉]]系統，設立「[[農業|農]][[牧業|牧]]綜合經營示範中心」，傳授農業新觀念及知識技能，推廣種植高經濟價值[[蔬菜|蔬]][[水果|果]]；<ref name="88年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter1-3.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref name="91年外交年鑑"/><ref name="104年外交年鑑"/><ref name="98年外交年鑑"/>2008年增加「縮短[[數位落差|數位落差]]計畫」，於{{link-en|聖多美普林西比國家圖書館|National Library of São Tomé and Príncipe}}設立「數位中心」，協助發展[[資訊|資訊]]教育。<ref name="97年外交年鑑"/>

====航班====
兩國無直航班機，亦無中華民國→直航第三地→聖多美普林西比，需多次轉機前往。

例如：

{{ROC}}
*[[臺灣桃園國際機場|臺北]]→[[戴高樂國際機場|巴黎]]→[[温贝托·德尔加多机场|里斯本]]→[[聖多美國際機場|聖多美國際機場]]。

===駕車===
持有中華民國[[國際駕駛執照|國際駕駛執照]]或申請聖多美普林西比駕駛執照，並投保[[車險|車險]]，可在該國駕車。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-251-8c594-05-1.html |title = 生活資訊 |publisher = 外交部領事事務局}}</ref>
