'''中華民國與哈薩克關係'''是指[[中華民國|中華民國]]與[[哈薩克共和國|哈薩克共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，[[哈薩克|哈薩克]]當時屬於[[蘇聯|蘇聯]]的[[加盟共和國|加盟共和國]]（[[哈薩克蘇維埃社會主義共和國|哈薩克蘇維埃社會主義共和國]]），由蘇聯作為[[主權國家|主權國家]]參與投票。]]
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境哈薩克。]]
[[File:Visa_requirements_for_Kazakhstani_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|哈薩克簽證政策|Visa policy of Kazakhstan}}|哈萨克斯坦公民签证要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。[[中華民國國民|中華民國公民]]持有[[中華民國護照|中華民國護照]]、{{link-en|哈薩克外交部|Ministry of Foreign Affairs (Kazakhstan)}}核准的邀請函（由邀請的哈薩克公司或[[旅行社|旅行社]]出具），可獲發單次、停留最多30天的觀光或商務[[簽證#落地签证|落地簽證]]。經哈薩克轉機，若未離開機場轉機區，無須申辦[[過境簽證|過境簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-137-c6e20-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=7F220D7E656BE749 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|哈薩克護照|Kazakhstani passport}}的{{link-en|哈薩克人口|Demographics of Kazakhstan|哈薩克公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國衛生福利部|中華民國衛生福利部]]的國際間旅遊疫情建議等級表將哈薩克歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年9月4日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=16C812A5052C98F5 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
2004年，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於[[哈薩克|哈薩克]]前首都、最大城市設立'''[[阿拉木圖|阿拉木圖]]台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#ASIA |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

2010年，中華民國出口至哈薩克的金額每年開始大幅成長，除2008、2009年因[[經濟大衰退|全球金融風暴]]分別出現負成長及個位數成長外，其他年度成長幅度均在30%以上。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref><ref name="歐亞團拓銷中亞明星哈薩克放異彩">{{Cite web |url = http://www.trademag.org.tw/News.asp?id=636099&url=/NewsSearch.asp |author = 李金玲 |date = 2014年5月20日 |title = 歐亞團拓銷中亞明星哈薩克放異彩 |publisher = 經貿透視網}}</ref>

2012年，出口金額首次達到1億[[美元|美元]]，為1億0,199萬美元，年增65%。<ref name="歐亞團拓銷中亞明星哈薩克放異彩"/>

2013年，出口金額為1億4,878萬美元，年增46%，主要成長項目為[[行動電話|手機]]，僅該項目即成長3,785萬美元，年增96%。<ref name="歐亞團拓銷中亞明星哈薩克放異彩"/>

2014年4月28日，[[經濟部國際貿易局|經濟部國際貿易局]]委託外貿協會籌組「2014年歐亞利基產業拓銷團」在哈薩克阿拉木圖舉行貿易洽談會。<ref name="歐亞團拓銷中亞明星哈薩克放異彩"/>

2018年，哈薩克是中華民國第88大貿易夥伴、第72大[[進口|進口]]夥伴、第98大[[出口贸易|出口]]夥伴。出口至哈薩克的金額為3,130萬9,657美元，年減9.550%；自哈薩克進口的金額為8,870萬0,851美元，年減12.491%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）5,739萬1,194美元，年減14.016%。2017年，哈薩克是中華民國第84大貿易夥伴、第64大進口夥伴、第94大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
中華民國與哈薩克尚未互設官方機構與官方常態性對話管道，至今也未有任何一家廠商正式前往哈薩克投資。目前[[宏碁|宏碁]]、[[華碩電腦|華碩電腦]]、[[合勤科技|合勤科技]]、[[技嘉科技|技嘉科技]]、[[圓剛科技|圓剛科技]]等，是在[[俄羅斯|俄羅斯]]首都[[莫斯科|莫斯科]]設立地區[[總部|總部]]，再以[[代理|代理]]商、代理人或[[商務旅遊|出差]]方式經營哈薩克市場。<ref name="與我國經貿關係"/>
