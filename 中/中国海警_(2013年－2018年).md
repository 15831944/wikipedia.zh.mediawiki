'''中国海警'''是[[中华人民共和国|中华人民共和国]]的[[海岸巡防隊|海洋边界警卫部队]]，建立于2013年，隶属于[[中华人民共和国国土资源部|国土资源部]]管理的[[国家海洋局|國家海洋局]]，同时接受[[中华人民共和国公安部|公安部]]的业务指导，對外以「'''中國海警局'''」名義進行海上武裝執法。原设有11个省级海警总队。截至2013年8月1日，中国海警拥有千吨级以上公务执法船135艘，总吨位超36万吨<ref name="A01" />。2018年，中国海警队伍整体划归武警部队领导指挥，称[[中国人民武装警察部队海警总队|中国人民武装警察部队海警总队]]<ref>{{Cite web |url = http://www.xinhuanet.com/mrdx/2018-06/29/c_137289498.htm |title =组建中国人民武装警察部队海警总队  |author =  |date =2018-06-29  |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。

== 沿革 ==
2013年3月，根据[[第十二届全国人民代表大会|第十二届全国人民代表大会第一次全体会议]]通过的《[[国务院机构改革|国务院机构改革]]和职能转变方案》，[[中华人民共和国政府|中华人民共和国政府]]将原[[中华人民共和国国土资源部|国土资源部]]轄下的[[国家海洋局|国家海洋局]]的[[中国海监总队|中国海监总队]]、[[中华人民共和国公安部|公安部]]所屬的[[中国公安边防海警部队|中国公安边防海警部队]]、[[中华人民共和国农业部|农业部]]所屬的[[农业部渔业渔政管理局|中国渔政]]、[[中华人民共和国海关总署|海关总署]]所屬的[[海关总署缉私局|海上缉私警察]]整合为“'''中国海警局'''”。中国海警局与[[国家海洋局|国家海洋局]]是一个单位，两块牌子<ref>{{Cite web|url=http://politics.people.com.cn/n/2013/0723/c1001-22285781.html|title=中国海警局正式挂牌 设南海北海东海3个分局|accessdate=2018-03-22|date=2013-07-23|publisher=人民网}}</ref>。中国海警局局长由公安部分管海警工作的[[正部级|正部级]]副部长兼任，政委由海洋局局长兼任。[[国家海洋局|国家海洋局]]對外以“中國海警局”名義进行執法活动<ref>{{cite news |title = 国家海洋局以中国海警局名义开展海上维权执法 |url = http://news.xinhuanet.com/legal/2013-03/11/c_124442592.htm |accessdate = 2013-04-23 |newspaper = 人民公安报 |publisher = 新华网}}</ref>，同时接受[[中华人民共和国公安部|中华人民共和国公安部]]的业务指导。

2018年3月21日，根据[[中国共产党第十九届中央委员会第三次全体会议|中国共产党第十九届中央委员会第三次全体会议]]审议通过的《[[深化党和国家机构改革|深化党和国家机构改革]]方案》，海警隊伍轉隸[[中国人民武装警察部队|武警部隊]]<ref>{{Cite web|url=http://mil.huanqiu.com/world/2018-03/11684208.html|title=武警部队编制改革力度空前 海警力量“当兵入伍”|accessdate=2018-03-22|date=2018-03-21|publisher=环球网}}</ref>。

2018年6月22日，根据[[第十三届全国人民代表大会常务委员会|第十三届全国人民代表大会常务委员会第三次会议]]通过的《全国人民代表大会常务委员会关于中国海警局行使海上维权执法职权的决定》，海警队伍整体划归中国人民武装警察部队领导指挥，调整组建'''中国人民武装警察部队海警总队'''，称'''中国海警局'''，统一履行海上维权执法职责<ref>{{Cite web|url=http://www.npc.gov.cn/npc/xinwen/2018-06/22/content_2056585.htm|title=全国人民代表大会常务委员会关于中国海警局行使海上维权执法职权的决定|accessdate=2018-11-05|date=2018-06-22|publisher=中国人大网}}</ref>。

2018年7月1日零时起，海警队伍整体划归武警部队领导指挥<ref>{{Cite web |url = http://www.xinhuanet.com/mrdx/2018-06/29/c_137289498.htm |title =组建中国人民武装警察部队海警总队  |author =  |date =2018-06-29  |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。
