'''中環站'''（{{lang-en|'''Central Station'''}}）是位於[[香港|香港]][[中環|中環]][[遮打道|遮打道]]、[[畢打街|畢打街]]和[[德輔道|德輔道中]]的[[港鐵|港鐵]]列車站，並設有4個月台（一個島式月台和兩個側式月台），是[[荃灣綫|荃灣綫]][[香港島|香港島]]的終點站與[[港島綫|港島綫]]的中途站。中環站設有收費區的行人隧道分別連接鄰近[[中心商务区|商業區]]及[[香港站|香港站]]，供乘客轉乘[[東涌綫|東涌綫]]和[[機場快綫|機場快綫]]，而該隧道位於[[環球大廈|環球大廈]]、[[干諾道|干諾道中]]和[[交易廣場|交易廣場]]地底<ref name="sta_info">{{Cite book|author=港鐵公司|title=《中環站車站指南》|publisher=港鐵公司|date=2009年8月}}</ref>。

== 車站設計 ==
=== 樓層 ===
中環站設有5層，地面為出入口，大堂和商店主要在中環站的L1、L2層。L2及L4層分別為港島綫3號月台（往柴灣）及4號月台（往堅尼地城），而L3層則為荃灣綫的1號及2號月台（往荃灣）<ref name="sta_info" />。

另外，在L1層的[[遮打道|遮打道]]大堂與J出入口之間有一層閣樓，港鐵公司在該層開放了「藝術管道」<ref name="art_tube">{{cite web|url=http://www.mtr.com.hk/chi/community/art_arttube.html|title=心繫港鐵－港鐵車站藝術－藝術管道|work=港鐵網頁|publisher=[[港鐵公司|港鐵公司]]|accessdate=2012年3月4日|language=zh-hk}}</ref>用作展出藝術作品。而在L2層的3號月台設有行人隧道連接香港站東涌綫大堂。

=== 大堂 ===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = HKMTR Central Station Chater Street Concourse.jpg | caption1 = 遮打道大堂
| image2 = Central Station Pedder Street Concourse 201504.JPG | caption2 = 畢打街大堂
| image3 = Central Station 2015 01 part5.JPG | caption3 = 位於大堂內的藝術品
}}
中環站共有兩個大堂——遮打道大堂與畢打街大堂<ref name="sta_info" />。中環站是全港首個設有雙層大堂的車站。中環站開放初期，上層大堂與下層大堂沒有非付費區連接，乘客需要經過付費區才可往返上下兩層大堂。

遮打道大堂位於[[遮打道|遮打道]]地底，在地鐵[[修正早期系統|修正早期系統]]啟用時已興建。畢打街大堂位於[[畢打街|畢打街]]地底，並與港島綫同時興建的。該大堂共有2層，上層是連接出入口，而下層則連接遮打道大堂及港島綫月台<ref name="sta_info" />。其後又因[[機場鐵路|機場鐵路]]連接通道而作翻新及擴建，並設有行人通道連接[[東涌綫|東涌綫]]和[[機場快綫|機場快綫]]的[[香港站|香港站]]相連。由於翻新後的大堂與香港站的設計相若，均利用白色的搪瓷板作為牆身，而非像月台以紅色的馬賽磚為主要牆壁，使乘客容易將兩站混淆，以為到達香港站範圍，但只是身處於中環站的畢打街大堂。

直至2001年翻新後，地鐵公司將通往3號月台至遮打道上層大堂的樓梯，重新劃分為非付費區通道，方便乘客利用該通道連接的中環站非付費大堂的下層。此外，地鐵公司加入咖啡店、快餐店和書店，增加地鐵公司的非車務收入及便利乘客購物<ref name="mtrshop_cen">{{Cite news|language=zh-hant|url=http://hk.apple.nextmedia.com/news/art/20020601/2633781|title=可售賣燒味涼茶 設藝展免費上網 中環地鐵站闢建飲食區|publisher=蘋果日報|page=A4|date=2002-06-01|accessdate=2009-07-27}}</ref>。

現時，中環站內提供不同類型的商店供乘客購物或進食，例如有[[便利店|便利店]]、[[麵包糕餅店|麵包糕餅店]]、[[咖啡店|咖啡店]]、[[書店|書店]]、[[花店|花店]]、[[銀行|銀行]]及[[速遞|速遞]]等零售業及其他服務<ref name="shop"/><ref name="sta_info" />。

中環站也有不少自助服務設施供乘客使用，包括[[自動櫃員機|自動櫃員機]]、[[自動販賣機|自動售賣機]]、自動照相機等<ref name="shop"/><ref name="sta_info" />。近F出入口的大堂更設有[[香港郵政|香港郵政]][[郵箱|郵箱]]<ref>{{Cite web|language=zh-hant|publisher=香港郵政|title=郵箱位置表－港島區|url=http://www.hongkongpost.com/chi/locations/spb/hk/index.htm|author=香港郵政|accessdate=2009-07-27|deadurl=yes|archiveurl=https://web.archive.org/web/20090130035301/http://www.hongkongpost.com/chi/locations/spb/hk/index.htm|archivedate=2009-01-30}}</ref>，方便乘客在車站內投寄[[郵件|郵件]]。

在[[互聯網|互聯網]]服務上，現時中環站大堂及月台均設有免費由[[電訊盈科|電訊盈科]]提供的[[Wi-Fi|Wi-Fi]]熱點<ref>{{Cite web|language=zh-hant |publisher=港鐵公司 |title=車站設施 - Wi-Fi無線寬頻 |url=http://www.mtr.com.hk/chi/facilities/wifi.html |author=港鐵公司 |accessdate=2009-07-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20090803180127/http://www.mtr.com.hk/chi/facilities/wifi.html |archivedate=2009-08-03 }}</ref>，乘客可在候車的時候透過任何有Wi-Fi功能的電子器材存取互聯網。同時，港鐵公司於3號月台的行人通道設有「iCentre」免費上網服務設施<ref name="icentre">{{Cite web|language=zh-hant |publisher=港鐵公司 |title=車站設施 - iCentre |url=http://www.mtr.com.hk/chi/facilities/icentre.html |author=港鐵公司 |accessdate=2009-07-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20090708045956/http://www.mtr.com.hk/chi/facilities/icentre.html |archivedate=2009-07-08 }}</ref>，使乘客可在該處瀏覽網上資訊。

另外，港鐵於畢打街大堂L1層非付費區及L2層往返香港站的付費區行人通道內，各設有1部「[[會員服務站|會員服務站]]」，供乘客享用港鐵友禮會的會員優惠<ref>{{Cite web|language=zh-hant|publisher=港鐵公司|title=車站設施 - 「會員服務站」|url=http://www.mtr.com.hk/ch/customer/services/csp.html|author=港鐵公司|accessdate=2016-10-03}}</ref>。

=== 月台 ===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Central Station 2018 01 part4.jpg | caption1 = 2號月台（往荃灣方向）
| image2 = Central Station 2018 01 part2.jpg | caption2 = 3號月台（往柴灣方向）
| image3 = Central Station 2018 01 part1.jpg | caption3 = 4號月台（往堅尼地城方向）
}}
中環站設有4個月台<ref name="sta_info" />，各個月台均裝設有[[月台幕門|月台幕門]]。1號和2號月台共用一個[[島式月台|島式月台]]，供荃灣綫往[[荃灣站|荃灣]]方向及從荃灣往本站的列車停靠，建在遮打道的地底，由[[德輔道|德輔道中]]伸延至[[會所街|會所街]]。月台不設落客專用月台及調頭隧道，列車到站後會在該月台不同時間進行上落客及調頭，乘客可按指示於1、2號月台乘搭荃灣綫往荃灣方向的列車。同時，荃灣綫1及2號月台下方為港鐵其中一條重要側線——設有3條停車側綫的金鐘側綫，供荃灣綫及港島綫列車駛往金鐘站不同月台作車務調度之用。路軌由金鐘站月台延伸至德輔道中地底。1及2號月台近金鐘方向頭端牆有爬梯前往此側線。

3號月台（港島綫往[[柴灣站|柴灣]]方向）和4號月台（往[[堅尼地城站|堅尼地城]]方向）設於[[德輔道中|德輔道中]]與[[畢打街|畢打街]]交界的地底，由[[環球大廈|環球大廈]]伸延至[[歷山大廈|歷山大廈]]。這兩個月台從上下把[[荃灣綫|荃灣綫]]月台夾在中間，兩個[[側式月台|側式月台]]設在路軌北面側，3號月台在上，4號月台在下。

另外，每當特殊的情況之下，如下班繁忙時間列車調頭往[[金鐘站|金鐘站]]接載擠滿月台的乘客，部份荃灣綫的列車會在中環站清客後暫停服務，屆時月台將會發出廣播。乘客如果往返該站及[[葵芳站|葵芳站]]至[[荃灣站|荃灣站]]，往[[香港站|香港站]]乘搭[[東涌綫|東涌綫]]再經[[荔景站|荔景站]]轉乘荃灣綫所需時間會較直接使用[[荃灣綫|荃灣綫]]為快，但基於非繁忙時間東涌綫班次疏落及需要在荔景站轉車，實際上直接乘坐荃灣綫會較方便。

== 利用狀況 ==
中環站位於[[中環|中環]]，是香港首要的[[商業中心區|商業中心區]]及各重要機關所在，周邊主要為地標性建築物（[[中環街市|中環街市]]、[[置地廣場|置地廣場]]）、辦公大樓（如[[滙豐總行大廈|滙豐總行大廈]]、[[恆生銀行總行大廈|恆生銀行總行大廈]]、[[交易廣場|交易廣場]]、[[環球大廈|環球大廈]]、[[萬宜大廈|萬宜大廈]]等）和政府部門（如[[郵政總局|郵政總局]]、[[終審法院大樓|終審法院大樓]]），還有酒店（如[[文華東方酒店|文華東方酒店]]）、文娛設施（如[[香港大會堂|香港大會堂]]）、公共空間（如[[皇后像廣場|皇后像廣場]]、[[遮打花園|遮打花園]]）和消遣地帶（如[[蘭桂坊|蘭桂坊]]）等，再加上本身是[[荃灣綫|荃灣綫]]、[[港島綫|港島綫]]及香港站的[[東涌綫|東涌綫]]及[[機場快綫|機場快綫]]的重要轉車站，附近又設有各種交通工具以供接駁，因此車站非常繁忙，下班繁忙時間往荃灣綫月台的扶手電梯更經常大排長龍，每天有約有25萬人次使用本站往返香港各區<ref>{{Cite book|author=迪仔、SP1900|title=《港鐵誌05》|publisher=千九網|date=2005年7月}}</ref>，為整個系統第三繁忙的車站<ref>{{Cite news|language=zh-hant |url=http://www.mingpaotor.com/htm/News/20070628/HK-gga3.htm |title=銅鑼灣站極擠迫 擬建地下街紓緩 |work=明報 |pages= |date=2007年6月28日 |accessdate=2008年11月18日 |deadurl=yes |archiveurl=https://web.archive.org/web/20090925084447/http://www.mingpaotor.com/htm/News/20070628/HK-gga3.htm |archivedate=2009年9月25日 }}</ref>。

部份在[[金鐘站|金鐘站]]往[[荃灣站|荃灣]]及沿途各站及來自[[港島綫|港島綫]][[灣仔站|灣仔站]]至[[柴灣站|柴灣站]]前往[[荃灣綫|荃灣綫]][[尖沙咀站|尖沙咀站]]至[[美孚站|美孚站]]的乘客或會選擇先到中環站調頭或在中環站轉車，以避開金鐘站的擠迫人潮，輪候時間可能較在金鐘站登車快。例如：部份的乘客會選擇在金鐘站的[[荃灣綫|荃灣綫]]往中環方向的月台上車，到達該站後不下車（但須留意下班繁忙時間有數班列車可能在此站清客及直接前往金鐘站載客，屆時會有廣播提示乘客離開車廂），留在車上，並進行調頭，稱之為「荃灣綫的過渡乘客」。又或者，乘客到達金鐘站時選擇不下車，坐多一個站直接到該站下車並轉車。另外，乘客如果往返[[南港島綫|南港島綫]]，需要前往金鐘站轉車。

由於本站附近並無住宅發展，而來往本站及半山區的屋村巴士服務大多已停止服務，故在晚上的出入站人流相對稀少，但當[[蘭桂坊|蘭桂坊]]和[[中環摩天輪|中環摩天輪]]舉辦活動時，不少市民遊客亦會使用本站前往那裏。

== 接駁交通 ==
由於中環站處於已發展的[[商業中心區|商業中心區]]內，該站並沒有空地設立地面[[公共運輸交匯處|公共運輸交匯處]]，但中環站落成初期亦有部份地鐵接駁巴士路線<ref>地鐵接駁巴士路線是指路線編號後附以「M」英文字母的路線</ref>以中環站為終點站或以循環線形式運作，惟其總站只設於街道旁邊巴士站。直到[[交易廣場|交易廣場]]設有大型巴士總站，不少巴士路線遷往該處作總站，而接駁巴士路線只是途徑中環街道中途站方便乘客轉乘。隨著港島綫落成，有不少來往[[中環|中環]]至[[香港島|香港島]]東面的接駁巴士路線失去原有功效，有不少巴士路線縮短或取消。

現時，仍有不少乘客在中環站步行前往交易廣場公共運輸交匯處或於[[德輔道|德輔道中]]轉乘[[香港巴士|巴士]]、[[香港小巴|小巴]]或[[香港電車|電車]]前往香港[[中西區_(香港)|中西區]]、[[南區_(香港)|南區]]；同時也有不少乘客前往海旁乘坐[[山頂纜車|山頂纜車]]接駁巴士往返纜車站，或是步行至[[中環碼頭|中環碼頭]]轉乘[[渡輪|渡輪]]前往[[離島區|離島區]]、[[九龍半島|九龍半島]]等地。此外，半山區不少屋苑都設有居民巴士在本站附近設站，為屋苑居民提供接駁至港鐵的服務。

此外，由於港鐵興建機場鐵路時，早已在環球大廈地底與建行人隧道連接香港站，現時有港島綫及荃灣綫乘客利用行人隧道轉乘機場快綫或東涌綫前往[[香港國際機場|香港國際機場]]、[[青衣島|青衣島]]、[[大嶼山|大嶼山]]等地，為中環站帶來龐大轉乘人流。

== 歷史 ==
===早期計劃===
根據1967年發表的《香港集體運輸研究》<ref name="MTS-1967">{{Cite book|author=Freeman, Fox, Wilbur Smith & Associates|title=《香港集體運輸研究》"Hong Kong mass transport study"（MTS-1967）|year=1967|month=|publisher=香港政府}}《香港集體運輸研究》是中文譯名，文件只有英文版本。</ref>，中環站初期與上環街市站（即現[[上環站|上環站]]）計劃為港島綫及[[觀塘綫|觀塘綫]]間的轉綫站，而荃灣綫則以[[金鐘站|金鐘站]]為尾站。當時中環站的計劃位置為[[租庇利街|租庇利街]]與[[畢打街|畢打街]]之間的一段[[德輔道中|德輔道中]]地底，稍西於現在中環站的位置。

===早期系統===
[[香港政府|香港政府]]委託的顧問公司於1970年發表了另一份《集體運輸計劃總報告書》，當中建議在中環商業區設置兩個車站，分別為位於遮打道地底'''遮打站（Chater Station）'''及位於必打街（現稱畢打街）地底'''必打站（Pedder Station）'''。

===修正早期系統===
中環站由四大聯營公司（Dragages, Hochtief, Gammon, Sentab) 負責興建，於1980年2月12日因應[[香港地鐵|地鐵]][[修正早期系統|修正早期系統]]由[[尖沙咀站|尖沙咀]]延伸而啟用，成為修正早期系統的終點站，乘客可直接乘坐列車前往[[觀塘站|觀塘]]，約需27.5分鐘<ref name="MIS open book">{{Cite book|author=地下鐵路公司|title=《地下鐵路首日通車紀念特刊》"Offical Souvenir Book To Commemorate The 1st Day Of Running MTRC Trains"|year=1979|month=|publisher=地下鐵路公司}}</ref>。啟用初期只開放1號和2號月台、位於遮打道大堂以及畢打街走廊層，而底層設有停車側綫，由遮打道月台延伸至德輔道中，當時車站深25米，長380米<ref name="MIS open book" />，並預留了部份結構以便港島綫（第二期）於83年3月由青木飛島共同企業體負責興建。

由於修正早期系統當時是包括荃灣綫的中環至[[旺角站|旺角]]的路段，因此荃灣綫於1982年5月10日通車時，中環站便成為荃灣綫南端的尾站。

中環站C及D出入口是於1984年建成，原本於畢打街近會德豐大廈對出的行人路設有兩個地面出入口，到了1984年因應會德豐大廈興建，順道把這兩個出入口搬遷至大廈內；另外K出入口是於1985年因應新的[[香港滙豐總行大廈|香港滙豐總行大廈]]落成才加建。

===車站命名===
中環站荃灣綫和港島綫部份在通車前後曾分別有多個名稱，包括3個英文名稱：Chater（遮打）、Pedder（必打）和Central（中環）及4個中文名稱：遮打、必打、中環、中環西。現時統一命名為「中環（Central）」。

1967年9月，費爾文霍士顧問工程公司（Freeman, Fox, Wilbur Smith & Associates）發表了一份有關香港在地底集體運輸城市軌道交通系統的研究《香港集體運輸研究》<ref name="MTS-1967"/>，報告中提及在中環興建一個車站，名為'''Central'''<ref name="MTS-1967" />。

1970年，顧問公司發表了另一份《集體運輸計劃總報告書》<ref>{{Cite book|author=費爾文霍士顧問工程公司，Freeman, Fox, Wilbur Smith & Associates. & Hong Kong Government|title=《集體運輸計劃總報告書》"Hong Kong mass transit further studies"（MTS）|year=1970|month=|publisher=香港政府}}</ref>，在報告書中，位於中環遮打道地底的車站名為'''遮打（Chater）'''，而位於中環必打街（今畢打街）地底的車站名為'''必打（Pedder）'''。

1979年，地鐵修正早期系統通車時，有關首日通車紀念特刊<ref>[[地鐵公司|地鐵公司]] - 《地下鐵路首日通車紀念特刊》"Offical Souvenir Book To Commemorate The 1st Day Of Running MTRC Trains"（1979）</ref>提及到位於遮打道地底的修正早期系統車站（1號及2號月台）名為'''中環（Chater）'''，而位於畢打街地底的港島綫車站名為'''中環西（Pedder）'''。因此，當修正早期系統於1980年2月12日由[[尖沙咀站|尖沙咀]]伸延至中環時，有關車站的中文名稱命名為中環，而不是遮打。

而根據1983年香港政府出版的街道圖<ref>{{Cite book|author=地政署測量部|title=《香港街道與地區。下冊，九龍及新界地圖及導遊指南》|year=1983|month=|publisher=地政署}}</ref>，畢打街地底的港島綫車站命名為中環（Pedder），因此可以證明車站的中文名稱統一為'''中環'''。

1985年5月31日，港島綫[[金鐘站|金鐘站]]至[[柴灣站|柴灣站]]一段正式投入服務，地鐵也同時為包括中環站的多個車站（另外還包括[[油麻地站|油麻地站]]、[[旺角站|旺角站]]及[[美孚站|美孚站]]）更名，中環站的英文名稱Chater統一為Central取代<ref>{{Cite web|language=en |publisher=HKMTIC |title=The History - Hong Kong Mass Transit InfoCenter |url=http://www.theskytrain.net/hkmtr/Past_History.htm |author=HKMTIC |accessdate=2009-05-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20071007153839/http://www.theskytrain.net/hkmtr/Past_History.htm |archivedate=2007-10-07 }}</ref>。當港島綫在1986年5月23日延伸到中環站時，全站英文名稱統一為'''Central'''。

===機場鐵路連接通道===
為配合1990年代[[香港機場核心計劃|香港機場核心計劃]]<ref>{{Cite web|language=zh-hant |publisher=香港政府 |title=香港機場核心計劃 |url=http://www.info.gov.hk/napco/index.htm |author=香港政府 |accessdate=2009-05-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20070523103406/http://www.info.gov.hk/napco/index.htm |archivedate=2007-05-23 }}</ref>，地鐵公司在[[環球大廈|環球大廈]]及[[交易廣場|交易廣場]]建造地底一條橫跨[[干諾道|干諾道中]]的地下行人通道，以連接中環站和香港站，供乘客從港島綫或荃灣綫轉乘機場快綫或東涌綫<ref>{{Cite book|author=Greg Pearce|title="Arup : Hong Kong station"|year=2001|month=|publisher=Fellbach}}</ref>。同時，地鐵公司亦翻新及擴建舊有畢打街大堂，加設新閘機及商店。工程於1998年峻工。畢打街大堂完成翻新後，與香港站設計近似，利用搪瓷板取代原有的[[紙皮石|紙皮石]]牆身。 

===車站翻新工程===
2001年，地鐵公司為改善中環站環境，再度翻新舊有遮打道大堂<ref name="mtrshop_cen" />，以配合畢打街大堂及香港站的一貫設計，部份紙皮石牆壁改以搪瓷板為主。是次翻新除了加設一般商店外，還把咖啡店、快餐店和書店引入至地鐵範圍，增加地鐵公司的非車務收入，吸引人流及方便乘客在車站購物。

此外，是次翻新工程亦將中環站畢打街及遮打道上下層的非收費區大堂連接，乘客再無需穿越地面繁忙的街道或途徑收費區往來兩大堂，整項工程於2002年4月完工。同時又於中環站月台引入8部公眾電腦作為「iCentre」免費上網服務設施<ref name="icentre" /><ref name="mtrshop_cen" />方便乘客在車站內瀏覽[[互聯網|互聯網]]的內容，及在J3出入口的商店改為作「藝術管道」<ref name="art_tube" /><ref name="mtrshop_cen" />為藝術家提供一個展覽場地。

2008年1月，港鐵公司翻新中環站，將大堂及港島綫3號月台沿用多年的地台更換為灰色地台，並且在部份位置更換假天花。

===西港島綫軌道接駁前期工程===
由於港鐵須進行西港島綫之重置軌道走線、信訊系統及架空電纜工程，[[上環站|上環站]]掉頭路軌暫時無法供列車使用，所以2011年8月5日晚上11時30分直至8月8日首班車前，港島綫曾以[[金鐘站|金鐘站]]為終點站，在此期間，上環站關閉，中環站港島綫月台亦暫停服務，至於荃灣綫來往荃灣站至中環站的列車服務則不受影響，港鐵在工程期間安排了免費接駁巴士接載受影響乘客直接來往金鐘站及上環站之間，中途不停站<ref>[http://www.mtr.com.hk/chi/whatsnew/shwclosure.html 2011年8月5日－2011年8月7日　上環站暫停服務　交通安排手冊]，[[港鐵公司|港鐵公司]]</ref><ref>[http://mtr.com.hk/eng/corporate/file_rep/PR-11-066-C.pdf 上環站由星期五晚至星期一清晨關閉]，2011年8月3日，港鐵公司</ref><ref>[http://mtr.com.hk/eng/corporate/file_rep/PR-11-068-C.pdf 上環站大型軌道工程進展順利 港島綫預計明早可恢復全綫服務]，2011年8月7日，港鐵公司</ref><ref>[http://mtr.com.hk/eng/corporate/file_rep/PR-11-069-C.pdf 港島綫恢復全綫服務]，2011年8月8日，港鐵公司</ref>。

===加建會所街行人隧道（L出入口）===
[[File:Entrance_and_exit_L_of_Central_Station.JPG|220px]]
港鐵公司於2010年將J3出入口旁機房位置加建一條長約18米的行人隧道連接[[中國建設銀行大廈|中國建設銀行大廈]]（前麗嘉酒店），連同原有機房位置新通道長約35米。<ref>{{cite web |title = 中環站會所街行人隧道－項目簡介 |url = http://www.mtr.com.hk/chi/projects/projects_improvement_subways_cen.html |publisher = 港鐵公司 |accessdate = 2012年3月2日 |language = zh-hk}}</ref>工程已於2010年年初展開<ref>{{cite web |title = 中環站會所街行人隧道－工程進度 |url = http://www.mtr.com.hk/chi/projects/projects_improvement_subways_cen_update.html |publisher = 港鐵公司 |accessdate = 2012年3月2日 |language = zh-hk}}</ref>，於2013年11月28日啟用。<ref>{{cite web |title = 港鐵中環站新設行人隧道及出入口方便市民 |url = http://www.mtr.com.hk/eng/corporate/file_rep/PR-13-104-C.pdf | publisher = 港鐵公司 |accessdate = 2013年11月28日 | language = zh-hk}}</ref>在[[雨傘運動|雨傘運動]]期間，L出入口一度關閉。

==車站事故==

===站內停電事件===
2016年6月7日下午3時許，中環站內一個供應冷氣的低壓電錶房跳掣，同時有少量煙冒出，滅火保護系統隨即啓動後截斷電源，導致站內一部升降機、兩部扶手電梯、3號及4號月台部份照明系統以及32間店舖中的其中14間停電及無冷氣。站內只剩下後備照明系統，漆黑一片。乘客如常出入閘，而港鐵職員手持電筒照明。到晚上9時10分全面恢復電力運作。<ref>[http://hk.apple.nextmedia.com/realtime/breaking/20160607/55196871 【港鐵故障】中環站大停電 乘客摸黑體驗4粒鐘 《蘋果日報》 2016-06-07]</ref>

===列車相撞事故===
[[File:20190318_MTR_TrainCollide_At_CentralConcourse.jpg|220px]]
{{hkcurrent}}
{{main|港鐵中環站列車相撞事故}}
2019年3月18日凌晨時分，兩列荃灣綫列車（A187/A112及A131/A218）於中環站附近進行新信號系統測試期間出現相撞事故，當第一部列車從中環站二號月台經渡線前往金鐘站一號月台時，第二部列車亦從金鐘站四號月台駛往中環站一號月台，當駛至中環站以東之渡線時，疑因訊號系統失靈，而第二部列車亦收制不及，撞向第一部列車之第三及第四節（D652和C451）車廂，兩車車門及車窗玻璃皆有飛脫，車身亦偏離路軌，涉事之兩名車長需送院治理，其中一人右腳受傷。受事故影響，中環站荃灣綫月台全日暫停使用兩天，荃灣綫列車於兩天內只能維持荃灣至金鐘之服務，而不前往中環，並維持約3.5分鐘一班列車，較正常情況下減少約一半班次。
