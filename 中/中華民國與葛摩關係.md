'''中華民國與葛摩關係'''是指[[中華民國|中華民國]]與[[葛摩聯盟|葛摩聯盟]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對葛摩的相關事務由'''{{link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''、'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-162-cbaa9-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|葛摩簽證政策|Visa policy of the Comoros}}|{{link-en|葛摩公民簽證要求|Visa requirements for Comorian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。{{link-en|葛摩政府|Politics of the Comoros}}沒有給予[[世界政區索引|任何地區]]的公民[[簽證#签证优惠待遇|免簽證]]待遇，持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可在首都[[莫洛尼|莫洛尼]]的[[賽義德·易卜拉欣王子國際機場|賽義德·易卜拉欣王子國際機場]]辦理[[簽證#落地签证|落地簽證]]入境葛摩，可停留2星期並得延長。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-162-cbaa9-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|葛摩護照|Comorian passport}}的{{link-en|葛摩人口|Demographics of the Comoros|葛摩公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將葛摩歸類為'''灰色警示'''：提醒注意（2017年7月3日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-352-76ac2-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

{{see also|臺灣經濟|葛摩經濟}}
===貿易===
2017年，葛摩是中華民國第218大貿易夥伴、第197大[[進口|進口]]夥伴、第215大[[出口贸易|出口]]夥伴。出口至葛摩的金額為4萬6,679[[美元|美元]]，年減54.714%；自葛摩進口的金額為5萬0,657美元，年增99.610%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）3,978美元，年減105.120%。※2016年，葛摩是中華民國第216大貿易夥伴、第202大進口夥伴、第213大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

====客運====
