'''中国马文化博物馆'''，位于[[北京市|北京市]][[延庆区|延庆区]][[八达岭镇|八达岭镇]]阳光路8号，是展示中国马文化的专题博物馆。

==简介==
中国马文化博物馆由[[中国马术协会|中国马术协会]]、[[中国马业协会|中国马业协会]]、[[中国文物学会|中国文物学会]]、延庆县人民政府联合主办，2003年开馆，位于[[八达岭长城|八达岭长城]]脚下的阳光山谷马术俱乐部内。2002年末成立的阳光山谷马术俱乐部是当时中国最大的马术俱乐部，拥有长1200米跑道的速度赛场，300米的马游泳池，长120米宽50米的室内马术表演场以及马舍长廊、山间马道。自2002年成立至2003年的一年中，该俱乐部承办了11次全国及北京市的各类马术比赛，并在2003年9月21日承办“通商杯”第三届亚洲马术锦标赛，同时举行亚洲马术联合会特别会议<ref>{{Cite web |url =  http://sports.sina.com.cn/s/2003-09-22/160057500s.shtml|title =第三届亚洲马术锦标赛结束  |author =  |date =2003-09-21  |publisher =  新浪|language =  |accessdate =  |quote =  }}</ref><ref name=mabei/>。

中国马文化博物馆占地面积约2700多平方米，馆内展品1300多件，按参观路线分为文字区、雕塑区、名画区、英式美式区、摄影区、蒙古区。另外，该馆还定期展出爱马人士收藏的藏品和艺术品<ref name=mabei>{{Cite web |url = http://news.sina.com.cn/s/2003-07-24/1654438349s.shtml |title = 马背上的行走 |author =  |date = 2003-07-24 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.beijingmuseum.gov.cn/art/2007/5/9/art_2683_67591.html |title =中国马文化博物馆  |author =  |date =2007-05-09  |publisher = 北京数字博物馆 |language =  |accessdate =  |quote =  }}</ref>。展品中有[[美国|美国]]1940年生产的马鞍，治疗用马蹄铁，[[三寸金莲|三寸金莲]]专用马镫，[[东汉|东汉]]陶砖，[[汉白玉|汉白玉]]雕马，镖车等等<ref>刘莹、唐志远，神骏入凡尘中国马文化博物馆，博物2014(2):62-65</ref>。
