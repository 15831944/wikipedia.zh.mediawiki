'''中国人民解放军第一六一医院'''，位于[[湖北省|湖北省]][[武汉市|武汉市]][[江岸区|江岸区]]黄浦大街68号，隶属[[武汉联勤保障基地|武汉联勤保障基地]]，为综合性[[三级甲等医院|三级甲等医院]]。

==简介==
前身为1944年4月在[[河南省|河南省]][[范县|范县]]始建的'''冀鲁豫军区八分区第二野战医疗所'''。1949年，随[[刘邓大军|刘邓大军]]南下抵达[[武汉|武汉]]。1950年，改编为正式医院。1954年8月，更名'''中国人民解放军第一六一医院'''。1983年5月，扩编为'''中国人民解放军第一六一中心医院'''。1994年4月起，隶属中国人民解放军武汉后方基地领导<ref>{{Cite book |url = http://www.whfz.gov.cn:81/was40/detail?record=1&primarykeyvalue=ID%3D21847&channelid=47777 |section = 第二节　驻汉部队医院 |author =  |date =  |title = 武汉市地方志 |language =  |accessdate =2017-06-16  |quote =  }}</ref>。后更名为'''中国人民解放军第一六一医院'''。该医院是综合性[[三级甲等医院|三级甲等医院]]，先后完成[[九八抗洪|九八抗洪]]、抗击[[非典|非典]]、[[利比里亚|利比里亚]]维和、[[汶川大地震|汶川大地震]]抗震救灾卫勤保障，以及多次联合军事演习任务<ref>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536929.htm |title = 第一六一医院 |author =  |date =  2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。2016年7月，中国人民解放军第一六一医院的22层新病房大楼投入使用<ref>{{Cite web |url = http://news.ifeng.com/a/20160801/49695763_0.shtml |title =图文：解放军第一六一医院新病房大楼启用  |author =  |date = 2016-08-01 |publisher =  凤凰网|language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第一六一医院的上级单位武汉后方基地改为[[武汉联勤保障基地|武汉联勤保障基地]]<ref>{{Cite web |url = http://www.81.cn/jwgz/2017-05/09/content_7592934.htm |title = 联勤保障部队医院这波为兵服务，可以给满分！ |author =  |date =  2017-05-09 |publisher =  中国军网|language =  |accessdate =  |quote =  }}</ref>。
