'''中華民國與印度關係'''是指[[中華民國|中華民國]]與[[印度共和國|印度共和國]]之間的關係。兩國於1942－1949年有正式[[外交#外交上的承認|外交關係]]，斷交後直至1990年代，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構，兩國在民間之商業以及文化往來日益密切。<!-- <ref name="FICCI">[http://www.ficci.com/international/countries/taiwan/taiwan.pdf]</ref>{{dead link}} --><!-- <ref name="Relations">{{cite web|url=http://www.ficci.com/international/countries/taiwan/taiwan.pdf|title=India Taiwan Commercial Relations|publisher=|accessdate=2018-11-13}}</ref>{{dead link}} --><ref name="VO">{{cite web|url=http://www.atimes.com/atimes/China_Business/HB15Cb02.html|title=Asia Times Online :: China News, China Business News, Taiwan and Hong Kong News and Business.|first=Asia Time|last=Online|website=www.atimes.com|accessdate=2018-11-13}}</ref>

===外交===
[[File:Training_at_Ramgarh.jpg|thumb]]期间[[中华民国|中华民国]][[国民革命军|国民革命军]]在印度蓝伽整训时的地图。]]
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[印度|印度]]。]]
{{see also|中華民國外交|印度外交|蒋介石访问印度}}
{{see also|中華民國駐印度大使列表{{!}}中華民國駐印度外交官列表|印度駐中華民國大使列表{{!}}印度駐中華民國外交官列表}}
中华民国在[[中国抗日战争|抗日战争]]時期，印度给予中华民国很多支持，1938年[[印度国会|印度国会]]派遣医疗队支援中国“抗战”，其中的队员[[柯棣华|柯棣华]]大夫病逝于中国；“抗战”时期战略物资运输线——[[滇缅公路|滇缅公路]]被[[日军|日军]]切断以后，随后开辟的“[[驼峰航线|驼峰航线]]”其始发地就在印度的[[阿萨姆邦|阿萨姆邦]]。1942年，[[中国远征军|中国远征军]]在[[缅甸|缅甸]]对日作战中战事失利，[[孙立人|孙立人]]部隊退入[[印度|印度]]成为[[中国驻印军|中国驻印军]]，在重新组装、训练后成为经由缅甸反攻回国的前锋部队。

1942年2月4日至21日，[[中國國民黨主席|中国国民党总裁]]、[[国民政府军事委员会|国民政府军事委员会]]委员长[[蒋中正|蒋中正]]偕夫人[[宋美龄|宋美龄]]及隨員[[王寵惠|王寵惠]]、[[張道藩|張道藩]]等访问[[英属印度|英属印度]]，2月9日抵[[新德里|新德里]]，受到[[印度总督|印度总督]][[维克托·霍普，第二代林利思戈侯爵|林里斯哥侯爵]]的歡迎。访问期间，蔣委員長在[[加爾各答|加爾各答]]會見印度精神领袖[[甘地|甘地]]。2月21日離印返國，發表《告印度人民書》。此行是中、英双方在[[南亚|南亚]]和[[西藏|西藏]]的博弈中，中国方面的重要行动，赢得了[[印度独立运动|印度独立运动]]领袖的热烈欢迎。1942年4月，中华民國駐印度專員公署在[[新德里|新德里]]設立。5月，印度駐華專員公署亦設立於[[重慶|重慶]]。<ref>{{cite web|url=http://ap6.pccu.edu.tw/Encyclopedia/data.asp?id=3227&forepage=1|title=+{中華百科全書‧典藏版}+|website=ap6.pccu.edu.tw|accessdate=2018-11-13}}</ref>

1943年，參與第二次世界大戰的中國遠征軍於印度賈坎德邦[[中華民國駐印軍蘭伽公墓|安葬國軍遺骸580具]]（現有667具），安葬者[[軍階|軍階]]最高為[[少將|少將]]，最低為[[上等兵|上等兵]]，雖然都有墓碑，但姓名可考者僅40位。[[中華民國國防部|中華民國國防部]]分別於1982年及2011年撥款修繕。<ref>[http://lov.vac.gov.tw/photo/Content.aspx?i=685&c=13&p=1 中華民國駐印軍蘭伽公墓 榮民文化網]</ref><ref>[http://ramgarh.nic.in/chainacemetery.htm Vashno Devi Temple]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

1946年10月21日，中、印使节互升为大使。1947年8月15日根据[[蒙巴顿方案|蒙巴顿方案]]成立[[印度自治領|印度自治領]]。

1949年12月30日，印度自治領宣布承认[[中华人民共和国|中华人民共和国]]，[[中华民国|中华民国]]宣布与印度断交。1950年1月10日，關閉大使館。4月1日，印度與中華人民共和國建立邦交，<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>在[[中華人民共和國－印度關係|中華人民共和國－印度建交公報]]中，印度也承諾會奉行「[[一個中國|一個中國]]」政策。並於1971年10月25日投票反對[[中國與聯合國#代表權問題|中華民國續留聯合國]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。

即便[[印度|印度]]與[[中華人民共和國|中華人民共和國]]之關係深固，但在[[印度政府|印度政府]]主導之[[印度東望政策|印度東望政策]]下，為了使印度的經濟更上一層樓，印度尋求與中華民國進行非政府性民間商業、文化、資金以及技術交流，而[[中華民國|中華民國]]也見證到「{{link-en|印度時代|Indian century}}」的到來而樂於近一步地投資印度。<!-- <ref name="FICCI"/>{{dead link}} -->2002年時，兩國簽訂《臺灣－印度促進投資暨保障協定》以再促進雙邊經濟往來加速發展。

1999年，[[印度人民黨|印度人民黨]]秘書長[[纳伦德拉·莫迪|莫迪]]（Narendra Modi）抵台訪問。 <ref name="88年外交年鑑"/>

2003年1月，第五屆亞洲安全會議在印度[[新德里|新德里]]舉行，由[[立法院外交及國防委員會|立法院外交委員會]]召集委員[[張旭成|張旭成]]及[[國立政治大學國際關係研究中心|國立政治大學國際關係研究中心]]研究員丁樹範聯袂出席。會中，[[中国驻印度大使馆|中華人民共和國駐印度大使館]][[参赞|參贊]]毛四維於張旭成發言時向印度抗議，並表示其出席該會是對「[[一個中國|一個中國]]」原則的違反，但印度方面未予理會。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=B81B5A2FC67CDD90&sms=175F59D3F37F17B5&s=DBF1970C62147FD8 |author = 公眾外交協調會 |date = 2003年4月14日 |title = 中國大陸阻撓我國際空間事例（92年1月至3月事例）|publisher = 中華民國外交部}}</ref>

2007年6月12日，曾任[[中國國民黨|中國國民黨]]黨主席、時任[[中華民國總統|中華民國總統]]參選人[[馬英九|馬英九]]訪問印度。<ref>{{Cite web |url = http://www.epochtimes.com/b5/7/6/12/n1741961.htm |author = 陳亦偉|date = 2007年6月13日 |title = 時隔十年再訪印度 馬英九認為印度進步多 |publisher = [[大紀元時報|大紀元時報]]}}</ref>

2012年4月9日，中華民國總統馬英九在訪問[[中華民國外交#邦交國|非洲邦交國]]的途中過境印度[[孟買|孟買]]，這是中華民國總統第一次過境印度。<ref>{{Cite web |url = https://www.voacantonese.com/a/article-20120409in-depth-look-into-ma-stop-over-india-146645535/1148317.html |author = 木風 |date = 2012年4月9日 |title = 小突破或大進展？論馬英九過境印度 |publisher = [[美國之音|美國之音]]}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#电子签证|電子簽證]]的方式入境印度。]]
[[File:Visa_requirements_for_Indian_citizens.png|thumb]]的[[印度人口|印度公民]]必須申請[[簽證|簽證]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|印度簽證政策|印度公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。印度簽證須事先向印度－台北協會申辦，持觀光簽證停留最多180天，其餘簽證若停留超過180天，須於入境14天內向{{link-en|印度內政部|Ministry of Home Affairs (India)}}外國人登記事務局辦理登記，或在預定啟程前至少4天上網申請[[簽證#电子签证|電子簽證]]。經印度轉機，若過境不超過24小時、在同一航廈特定待轉機區候機、無需辦理行李提領及行李重新報到，則不用申請簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-21-84369-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ElectronicVisa.aspx?n=6F246645F10311D7&sms=B75E93A3DAB23A99 |title = 電子簽證資訊 |publisher = 中華民國外交部}}</ref>

持[[印度護照|印度護照]]的[[印度人口|印度公民]]符合特定要求者可透過[[中華民國政府|中華民國政府]]的[[中華民國簽證政策#東南亞七國網路申辦入境中華民國許可|東南亞七國網路申辦入台許可]]申請[[簽證#签证优惠待遇|免簽證]]，停留最多30天；抵台參加團客旅遊、國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將印度按地區歸類為'''灰色警示'''：提醒注意、'''黃色警示'''：特別注意旅遊安全並檢討應否前往，與'''橙色警戒'''：避免非必要旅行（以上截至2018年10月23日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-92-e9faf-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將印度按疾病歸類為'''第一級注意'''：提醒遵守當地的一般預防措施，與'''第二級警示'''：對當地採取加強防護（以上截至2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=9635A6E58C4A5216 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於[[印度|印度]]首都[[新德里|新德里]]，以及[[孟買|孟買]]、[[加爾各答|加爾各答]]、[[清奈|清奈]]設立'''[[台北世界貿易中心|台北世界貿易中心]]辦事處'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#ASIA |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]也在新德里設立'''駐印度代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=214&pid=305862 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

兩國雙邊貿易自過去10多年來成長迅速，2000年時雙邊貿易僅12億[[美元|美元]]，至2014年雙邊貿易達59億美元。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref>

2014年3月5日，印度商工部建議對從中華民國進口的[[苯酚|苯酚]]課徵每[[公噸|公噸]]47.29－196.24美元的[[反傾銷稅|反傾銷稅]]，商工部另於2014年6月3日建議對從中華民國進口的[[聚氯乙烯|聚氯乙烯]]粉課徵每公噸9.47－61.25美元的反傾銷稅。<ref name="與我國經貿關係"/>

2017年4月26日，外貿協會在[[台北市|台北]]成立'''印度中心'''<REF>{{cite web|url=https://udn.com/news/story/7241/2426283|title=- - 聯合新聞網|publisher=|accessdate=2018-11-13}}</REF>。

2018年，印度是中華民國第16大貿易夥伴、第18大[[進口|進口]]夥伴、第14大[[出口贸易|出口]]夥伴。出口至印度的金額為37億8,711萬5,989[[美元|美元]]，年增14.714%；自印度進口的金額為32億5,940萬8,982美元，年增6.557%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）5億2,770萬7,007美元，年增117.604%。2017年，印度是中華民國第16大貿易夥伴、第18大進口夥伴、第14大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =經濟部國際貿易局}}</ref>

===投資<small><ref name="與我國經貿關係"/></small>===
兩國在簽訂雙邊協議後貿易額快速增長。駐印度代表處經濟組統計，至2014年4月，[[臺灣商人|台商]]在印度投資設點者約80餘家。投資產業包括[[資訊及通訊科技|資通訊]]、[[機械|機械]]、[[貿易|貿易]]、[[碳煙|碳煙]]、[[運輸|運輸]]、[[建筑工程|工程]]、[[金屬|金屬]]、製[[鞋|鞋]]、[[農業|農]][[漁業|漁]]、[[金融|金融]]等，其中以資通訊為主。台商在印度投資的金額為6,646萬美元，排名外商投資第45位，僅佔印度[[外商直接投資|外商直接投資]]金額0.03%。台商在印度主要集中在新德里、孟買、清奈、[[阿麦达巴|阿美達巴]]及[[班加羅爾|班加羅爾]]等地區。在新德里以市場開發、[[行銷|行銷]]及[[服務業|服務業]]為主；在孟買以服務業、[[製造業|製造業]]、市場行銷為主；在清奈以製造業為主；在阿美達巴以製造業為主。<ref name="VO"/><ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/in/pdf/103-23India.pdf |title = 投資環境簡介 |publisher = 經濟部全球臺商服務網}}</ref>

===臺商組織===
德里台商會、孟買台商會及南印度台商會（以清奈地區為主）均已成立，印度台灣商會聯合總會於2013年10月成立，由上述三地台商會共同組成，對增進會員廠商間的資訊交流、市場商機及協助新進投資者瞭解印度市場特性扮演重要角色。目前旅居印度的[[台灣人|台灣人]]相當稀少，預估全印度應不超過200人，多數為台籍幹部，實際創業投資者也甚少。<ref name="投資環境簡介"/>
