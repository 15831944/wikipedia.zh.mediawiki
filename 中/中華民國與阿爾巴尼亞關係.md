'''中華民國與阿爾巴尼亞關係'''是指[[中華民國|中華民國]]與[[阿爾巴尼亞共和國|阿爾巴尼亞共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對阿爾巴尼亞的相關事務由'''駐[[義大利|義大利]][[台北经济文化代表处|台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-155-efac2-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[阿爾巴尼亞人民共和國|阿爾巴尼亞人民共和國]]。]]
{{see also|中華民國外交|{{link-en|阿爾巴尼亞外交|Foreign relations of Albania}}}}
阿爾巴尼亞是歷屆[[聯合國大會|聯合國大會]]有關「[[中國與聯合國#代表權問題|中國代表權問題]]」的提案人，至[[冷戰|冷戰]]結束前的數十年間，對中華民國極不友好。<ref name="外交年鑑"/>

1971年10月25日，第26屆聯合國大會就阿爾巴尼亞、[[阿爾及利亞|阿爾及利亞]]等23國所提出的「恢復[[中華人民共和國|中華人民共和國]]在[[聯合國|聯合國]]的合法權利」議案進行[[投票|表決]]。最終以76票贊成、35票反對、17票棄權的結果通過。根據《[[聯合國憲章|聯合國憲章]]》和聯合國大會議事規則，提案通過即成為正式決議，是為《'''[[聯合國大會2758號決議|聯合國大會2758號決議]]'''》。[[中華民國|中華民國]]、[[馬爾地夫|馬爾地夫]]、[[阿曼|阿曼]]等3國未投票。代表'''[[中國|中國]]'''[[合法性|正統]]的[[中華民國政府|中華民國政府]]，以聯合國代表、[[中華民國外交部#歷任首長|外交部長]][[周書楷|周書楷]]，在表決之前數小時，於大會上主動宣布退出聯合國。日後，中華民國政府將此決議稱為「排我納[[共匪|匪]]案」。

1992年5月，[[中華民國駐教廷大使列表|中華民國駐教廷大使]]黃秀日訪問阿爾巴尼亞，並會見[[阿尔巴尼亚国家元首列表|總統]]等高層政府官員。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-4.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1995年6月，中華民國經濟部長[[江丙坤|江丙坤]]率團赴[[瑞士|瑞士]]參加蒙太納[[論壇|論壇]]，期間會見阿爾巴尼亞總統[[薩利·貝里沙|貝里沙]]。<ref name="86年外交年鑑"/>

1999年5月12日，[[阿爾巴尼亞國會|阿爾巴尼亞國會]]部份議員成立「阿台友好協會」，成員包括[[阿爾巴尼亞政黨列表|不同黨派]]之國會議員及學者，積極推動兩國關係。<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_8.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2004年11月25日，簽署《中華民國[[法務部調查局|法務部調查局]]洗錢防制中心與阿爾巴尼亞反洗錢理事會防制[[洗錢|洗錢]]{{link-en|金融情報|Financial intelligence|情報}}合作交換[[備忘錄|備忘錄]]》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e9%98%bf%e7%88%be%e5%b7%b4%e5%b0%bc%e4%ba%9e&tycountry_e=%e9%98%bf%e7%88%be%e5%b7%b4%e5%b0%bc%e4%ba%9e&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

2015年，阿爾巴尼亞國會領袖{{link-sq|達沙米爾·謝希|Dashamir Shehi|謝希}}抵台訪問。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係 義大利篇〉 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境阿爾巴尼亞。]]
[[File:Visa_requirements_for_Albanian_citizens.png|thumb]]的[[阿爾巴尼亞人口|阿爾巴尼亞公民]]必須申請[[簽證|簽證]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|阿爾巴尼亞簽證政策|Visa policy of Albania}}|{{link-en|阿爾巴尼亞公民簽證要求|Visa requirements for Albanian citizens}}}}
持有[[中華民國護照|中華民國護照]]與邀請函的[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境阿爾巴尼亞，每6個月期間內總計可停留90天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-155-efac2-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持有[[阿爾巴尼亞護照|阿爾巴尼亞護照]]的[[阿爾巴尼亞人口|阿爾巴尼亞公民]]抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將阿爾巴尼亞歸類為'''灰色警示'''：提醒注意（2016年11月4日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-168-02bed-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==

===貿易===
1990年3月，[[中華民國行政院|中華民國行政院]]開放對阿爾巴尼亞直接貿易。4月，開放[[直接投資|直接投資]]。行政院同時核定與阿爾巴尼亞通匯、通訊、通航等措施。<ref name="外交年鑑"/>

2013年11月23日，[[中華民國外交部|外交部]]委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）組團參加於[[阿尔巴尼亚议会|阿尔巴尼亚议会]]大廈舉辦的「2013[[地拉那|地拉那]]國際[[展覽會|博覽會]]」，此博覽會是該國規模最大的綜合性商展。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=624200&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2013年11月27日 |title = 前進巴爾幹貿協首度組團參加「地拉那國際博覽會」 |publisher = 經貿透視網}}</ref>

2014年11月16日，阿爾巴尼亞商業總會長首次訪臺，期間拜會外貿協會副[[秘書長|秘書長]]，進行首次經貿交流。阿爾巴尼亞商業總會是該國規模最大的商會。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=651964&url=/NewsSearch.asp |author = 諶怡如 |date = 2014年12月1日 |title = 阿爾巴尼亞商業總會首次來臺 加強與台灣企業經貿交流 |publisher = 經貿透視網}}</ref>

2018年，阿爾巴尼亞是中華民國第141大貿易夥伴、第118大[[進口|進口]]夥伴、第142大[[出口贸易|出口]]夥伴。出口至阿爾巴尼亞的金額為592萬5,014[[美元|美元]]，年減38.276%；自阿爾巴尼亞進口的金額為648萬0,650美元，年增417.771%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）55萬5,636美元，年減106.656%。2017年，阿爾巴尼亞是中華民國第142大貿易夥伴、第153大進口夥伴、第122大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
