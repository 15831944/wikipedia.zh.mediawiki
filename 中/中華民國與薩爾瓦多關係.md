'''中華民國與薩爾瓦多關係'''是指[[中華民國|中華民國]]與[[薩爾瓦多共和國|薩爾瓦多共和國]]之間的雙邊關係。兩國於1941－2018年有正式[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對薩爾瓦多的相關事務由'''[[中華民國－瓜地馬拉關係|中華民國駐瓜地馬拉共和國]][[中華民國駐外機構列表|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-90-f8499-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

1933年6月15日，中華民國與[[瓜地馬拉|瓜地馬拉]]建立領事級外交關係，由[[中華民國駐瓜地馬拉大使列表|駐瓜地馬拉總領事]]兼理薩爾瓦多[[海外華人|僑務]]。<ref name="86年外交年鑑">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html|title=《中華民國86年外交年鑑》〈第三章 中外關係〉|publisher=中華民國外交部|date=1998-06|accessdate=2017-12-16}}</ref>

1941年4月9日，中華民國與薩爾瓦多建立公使級外交關係，由[[中華民國駐巴拿馬大使列表|駐巴拿馬公使]]兼駐薩爾瓦多[[特命全權公使|公使]]。<ref name="105年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部 |deadurl = no |archiveurl = https://web.archive.org/web/20180129004249/http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |archivedate = 2018-01-29 }}</ref>

1957年7月17日，在首都[[聖薩爾瓦多|聖薩爾瓦多]]設立'''中華民國駐薩爾瓦多共和國[[外交代表機構#公使馆|公使館]]'''，並派駐專任公使。<ref name="86年外交年鑑"/><ref name="105年外交年鑑"/>

1961年6月1日，兩國升格大使級外交關係，公使館升格為'''中華民國駐薩爾瓦多共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權公使|大使]]。<ref name="105年外交年鑑"/>

1963年5月31日，薩爾瓦多首任駐中華民國大使呈遞[[國書|國書]]，由駐[[日本|日本]]大使兼任。<ref name="105年外交年鑑"/>

1971年10月25日，薩爾瓦多在[[聯合國大會|聯合國大會]]上支持中華民國繼續保有[[中國與聯合國#代表權問題|「中國」席位]]。

1976年10月，在首都[[臺北市|臺北]]設立'''薩爾瓦多共和國駐中華民國大使館'''。<ref name="86年外交年鑑"/>

1977年9月16日，薩爾瓦多首位駐中華民國專任大使呈遞國書。<ref name="105年外交年鑑"/>

1992年3月，[[中華民國立法院|中華民國立法院]]與[[萨尔瓦多共和国立法议会|薩爾瓦多立法議會]]締結為友好[[國會|國會]]；1993年10月，臺北與聖薩爾瓦多[[臺北市友好城市或姐妹城市列表|締結為姊妹市]]。<ref name="86年外交年鑑"/>

1996年3月，薩爾瓦多外交部與立法議會對中華人民共和國[[台灣海峽飛彈危機|在台灣海峽軍事演習]]，威脅中華民國即將於3月23日舉行的[[1996年中華民國總統選舉|首次總統直接選舉]]一事，發表聲明強烈譴責。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=5507ECB91425917B |author = 公眾外交協調會 |date = 1996年3月12日 |title = 薩爾瓦多外交部本月八日就中共在台灣臨近海域舉行軍事演習發表正式聲明 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=3ECA98472B9B86C0 |author = 公眾外交協調會 |date = 1996年3月20日 |title = 薩爾瓦多國會於三月十四日發表聲明 |publisher = 中華民國外交部}}</ref>薩爾瓦多並聯合[[哥斯大黎加|哥斯大黎加]]、[[瓜地馬拉|瓜地馬拉]]、[[宏都拉斯|宏都拉斯]]、[[尼加拉瓜|尼加拉瓜]]總統及[[貝里斯|貝里斯]]總理共同發表公報予以聲援。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=5B6A37274C59E4B3 |author = 公眾外交協調會 |date = 1996年3月20日 |title = 哥斯大黎加、薩爾多瓦、瓜地馬拉、宏都拉斯、尼加拉瓜總統及貝里斯總理 |publisher = 中華民國外交部}}</ref>

===斷交===
2018年8月21日，中華民國與薩爾瓦多斷交，結束長達77年的外交關係。<ref>{{Cite web|url = https://www.mofa.gov.tw/News_Content.aspx?n=8742DCE7A2A28761&s=EBE1B654F891C258 |author = 公眾外交協調會 |date = 2018年8月21日 |title = 中華民國政府基於維護國家尊嚴，決定自即日起終止與薩爾瓦多的外交關係 |publisher = 中華民國外交部}}</ref>23日，駐薩爾瓦多大使館舉行[[中華民國國旗|降旗]]儀式。<ref>{{Cite web|url = https://twitter.com/VOAChinese/status/1033064896399900672/video/1 |author = |date = 2018年8月24日 |title = 台湾驻萨尔瓦多大使馆23号最后一次降旗。四天前萨尔瓦多宣布与台湾断交，与中国建交。 |publisher = [[美国之音|美国之音]]中文网on [[Twitter|Twitter]]}}</ref>

先前中華民國曾允諾捐贈4架[[UH-1直升機|UH-1H直升機]]給薩爾瓦多，知情人士指出，這4架直升機「沒有起運」，在2018年6月掌握相關情資後，要求相關單位暫緩這批軍援，[[中華民國國防部|中華民國國防部]]對此也表示計畫並未執行。據指出，直升機已經裝箱，原本9月就要運出。而[[敦睦艦隊|中華民國海軍敦睦遠航訓練支隊]]在6月抵達薩爾瓦多，當時就捐贈[[救護車|救護車]]、[[乳房攝影術|乳房攝影]]、[[X射线机|C型臂Ｘ光機]]等60餘件醫療裝備。<ref>{{Cite web |url = https://tw.appledaily.com/new/realtime/20180821/1415006/ |author = 陳培煌 |date = 2018年8月21日 |title =【最即時專題22】軍援薩爾瓦多4架UH-1H直升機 國防部：並未執行 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref>

薩爾瓦多是[[蔡英文政府|蔡英文政府]]上任以來第5個斷交的國家，至2018年8月剩下[[中華民國外交#邦交國|17個邦交國]]。中華民國外交部長[[吳釗燮|吳釗燮]]7月中才前往薩爾瓦多訪問，並與該國總統[[萨尔瓦多·桑切斯·塞伦|桑契斯]]會談，桑契斯強調對雙邊邦誼的堅定支持，但在8月20日蔡英文結束對[[中華民國－巴拉圭關係|巴拉圭]]、[[中華民國－貝里斯關係|貝里斯]]訪問的「同慶之旅」回到台灣後的隔天，即宣布斷交。<ref>{{Cite web |url = https://udn.com/news/story/6656/3320865 |author = 徐偉真、許家瑜 |date = 2018年8月21日 |title = 時間點敏感！蔡總統出訪回國隔天 薩爾瓦多與我斷交 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref>

對於斷交一事，薩爾瓦多總統桑契斯表示，與台灣結束外交和貿易關係，並與中國建交是經過詳盡的分析，是符合[[國際法|國際法]]原則，將為國家帶來巨大利益。<ref name="在台留學生">{{Cite web |url = https://www.nownews.com/news/20180821/2804047 |author = 編輯中心 |date = 2018年8月21日 |title = 在台留學生近2百人 薩爾瓦多：全部送往大陸確保安全 |publisher = [[NOWnews今日新聞|NOWnews今日新聞]]}}</ref>但[[萨尔瓦多共和国立法议会|國會]]議長[[諾曼·基哈諾|基哈諾]]表示，台灣是一個非常支持薩國的國家，斷交讓他感到遺憾，這是對友好國家的背叛行為，並譴責總統桑契斯不得人心的政府態度，將會導致嚴重後果。他將在國會尋求表達反對政府這項可怕的決定，並向台灣人民道歉；首都[[聖薩爾瓦多|聖薩爾瓦多]]市長{{link-es|艾內斯托|Ernesto Muyshondt}}也認為，這是[[法拉本多·马蒂民族解放阵线|執政黨]]一個糟糕的決定，與[[民主國家|民主國家]]斷交，和另一個[[獨裁政體|獨裁政權]]建交，薩國政府在外交上做出可恥和可惡的決策，執政黨不能代表絕大多數薩爾瓦多人。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/2525894 |author = 綜合報導 |date = 2018年8月21日 |title =「向台灣人民道歉」薩爾瓦多國會議長砲轟斷交 |publisher = [[自由時報|自由時報]]}}</ref>美國[[白宮|白宮]]則在聲明中指出，「中國明顯干涉[[西半球|西半球]]國家內政，美國嚴重關切薩爾瓦多政府對此的接受程度，這將導致美國重新評估{{Link-en|薩爾瓦多－美國關係|El Salvador–United States relations|與薩爾瓦多的關係}}。」<ref>{{Cite web |url = http://news.ltn.com.tw/news/world/breakingnews/2529631 |author = 綜合報導 |date = 2018年8月24日 |title = 中國利誘薩爾瓦多與我斷交 白宮怒斥：破壞兩岸關係  |publisher = 自由時報}}</ref>9月7日，{{Link-en|美國國務院發言人|Spokesperson for the United States Department of State}}發布聲明，國務院已召回{{Link-en|美國駐薩爾瓦多大使列表|List of ambassadors of the United States to El Salvador|駐薩爾瓦多大使}}{{noteTag|以及2018年與中華民國斷交的{{Link-en|美國駐多明尼加大使列表|List of ambassadors of the United States to the Dominican Republic|駐多明尼加大使}}、2017年斷交的{{Link-en|美國駐巴拿馬大使列表|List of ambassadors of the United States to Panama|駐巴拿馬臨時代辦}}。}}，商討該國最近與台灣斷交的決定。<ref>{{Cite web |url = https://udn.com/news/story/6809/3355682 |author = 張加 |date = 2018年9月8日 |title = 不滿與台斷交 美召回駐巴國、多國、薩國使節 |publisher = 聯合新聞網}}</ref><ref>{{Cite web |url = http://www.peoplenews.tw/news/65607481-4591-46b9-a501-806823d8f1a8 |author = 綜合報導 |date = 2018年9月8日 |title = 挺台灣！美國下令召回與台斷交三國大使 |publisher = [[民報_(2014年)|民報]]}}</ref>

有關斷交的原因，外交部長吳釗燮表示是薩爾瓦多不斷要求大量資金，希望協助「聯合港開發案」的40億美元，經評估後，認為是不合適的開發案，恐讓兩國陷入高度債務風險。若再加上「[[自由經濟區|自由經濟區]]」的230億美元，加總就超過中華民國2019年國防預算的2倍。<ref>{{Cite web |url = https://www.chinatimes.com/realtimenews/20180821004115-260407 |author = 丁世傑 |date = 2018年8月21日 |title = 薩爾瓦多聯合港開發案 吳釗燮沒說的金額是多少？|publisher = [[中時電子報|中時電子報]]}}</ref>對於吳釗燮的說法，{{link-en|薩爾瓦多總統府|Casa Presidencial (El Salvador)}}發言人{{link-es|洛倫扎納 (政治人物)|Roberto Lorenzana|洛倫扎納}}反批台灣「卑鄙」且「完全錯誤」，並表示「[[中华人民共和国建交列表|其他國家都已經和北京建立了外交和經貿關係]]」、「我們不能違背世界潮流，忽略[[各國國內生產總值列表_(國際匯率)|中國已經是世界第二大國]]，在全球經貿出口的領導地位。」<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/2526989 |author = 綜合報導 |date = 2018年8月22日 |title = 薩爾瓦多翻臉不認人 竟反批台灣「卑鄙」|publisher = 自由時報}}</ref>2018年8月斷交時稱「沒有任何經濟前提」，但11月薩爾瓦多總統桑契斯訪問中國獲得金援1.5億美元（約[[新台幣|新台幣]]46.4億）。<ref>[http://news.ltn.com.tw/news/world/breakingnews/2606327 金援薩國46億認了金錢外交？中國外交部這麼說]</ref>

==經濟==

===貿易===
2018年，薩爾瓦多是中華民國第82大貿易夥伴、第89大[[進口|進口]]夥伴、第73大[[出口贸易|出口]]夥伴。出口至薩爾瓦多的金額為1億0,695萬3,144[[美元|美元]]，年增4.997%；自薩爾瓦多進口的金額為3,289萬2,796美元，年減45.438%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）7,406萬0,348美元，年增78.124%。2017年，薩爾瓦多是中華民國第74大貿易夥伴、第76大進口夥伴、第74大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

出口至薩爾瓦多的項目有：[[塑膠|塑膠]]製品及原料、[[橡膠|橡膠]]製品及原料、[[機器|機器]]及[[機械|機械]]用具、[[汽車零組件列表|汽機車零配件]]、[[電機機械|電機設備及零件]]、[[人造纖維|人造紡織]]材料、[[家具|家具]]及[[燈具|燈具]]、[[鋼鐵|鋼鐵]]製品、[[穀類|穀類]]、[[针织|針織品]]等；自薩爾瓦多進口的項目有：[[蔗糖|蔗糖]]、電機設備及零件、[[咖啡|咖啡]]、回收紙或[[紙板|紙板]]、鋼鐵、針織或[[鉤針編織|鉤針織]]之衣著、[[棉花|棉花]]、塑膠製品及原料、[[魚類|魚類]]及其他水產[[無脊椎動物|無脊椎動物]]、[[鋁|鋁]]製品等。<ref>{{Cite web |url = https://www.trade.gov.tw/World/ListArea.aspx?code=7020&nodeID=977&areaID=1&country=b646Jap54i-55Om5aSa&pw=3|title = 各國經貿資訊 |publisher =經濟部國際貿易局|date = 2018-02-01|accessdate = 2018-03-10}}</ref>

在薩爾瓦多的華僑約2,000人，其中約300人來自[[臺灣|臺灣]]。大多從事進口[[批發|批發]][[零售|零售]]、[[紡織|紡織]]及[[餐飲業|餐飲業]]。僑團組織有：華僑總會、中華文化協會、中華總商會（前三者合組「旅薩華僑聯合總會」）、中薩文化協會、臺灣商會等。<ref name="98年外交年鑑">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2009/html/08-1-6.htm|title=《中華民國98年外交年鑑》〈第二章 對外關係〉|publisher=中華民國外交部}}</ref><ref name="106年外交年鑑"/>

===援助===
斷交前，中華民國派駐薩爾瓦多技術團，合作項目有：建立中美洲區域海水養殖發展、強化鄉鎮家庭式[[水產養殖|水產養殖]]中心、{{link-en|農企業|Agribusiness}}發展及農民組織競爭力提升、熱帶蔬果健康種苗繁殖中心、恢復潛力地區種植[[稻米|稻米]]並提升生產及生產力、加強中美洲農牧保健組織（Organismo Internacional Regional de Sanidad Agropecuaria, OIRSA）轄區[[柑橘|柑橘]][[黃龍病|黃龍病]]防治及落實[[病虫害防治|病蟲害]]綜合管理、一鄉一特產、運用[[遙測|遙測]]及[[地理資訊系統|地理資訊系統]]合作、[[市場行銷|市場行銷]]技術指導及訓練等計畫。<ref name="87年外交年鑑">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1998/section_3/page3-1-6.htm|title=《中華民國87年外交年鑑》〈第三章 中外關係〉|publisher=中華民國外交部}}</ref><ref name="89年外交年鑑">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_10.htm|title=《中華民國89年外交年鑑》〈第三章 中外關係〉|publisher=中華民國外交部}}</ref><ref name="98年外交年鑑"/><ref name="102年外交年鑑"/><ref name="103年外交年鑑">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/11.pdf|title=《中華民國103年外交年鑑》〈第二章 對外關係〉|publisher=中華民國外交部}}</ref><ref name="104年外交年鑑"/>另派駐[[中小企業|中小企業]]專家團定期前往協助。<ref name="91年外交年鑑">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__06.htm|title=《中華民國91年外交年鑑》〈第三章 中外關係〉|publisher=中華民國外交部}}</ref>

2001年1－2月，薩爾瓦多{{link-en|2001年1月薩爾瓦多地震|January 2001 El Salvador earthquake|先}}、{{link-en|2001年2月薩爾瓦多地震|February 2001 El Salvador earthquake|後發生強烈地震}}，傷亡慘重，[[中華民國政府|中華民國政府]]即致電慰問該國政要，並迅速捐贈緊急慰問金、派遣30人救難隊前往支援。民間機構及團體亦陸續派遣救難及醫護人員前往救災或提供金錢、物資及運輸方面的協助。<ref name="90年外交年鑑">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2001/Chap3/3-1-6.htm|title=《中華民國90年外交年鑑》〈第三章 中外關係〉|publisher=中華民國外交部}}</ref>

2005年3月，中華民國副總統[[呂秀蓮|呂秀蓮]]率領農工商科技考察團訪問薩爾瓦多，與該國副總統{{link-en|艾絲柯芭|Ana Vilma de Escobar}}舉行「台薩園區」研討會，決議於[[萨尔瓦多奥斯卡·阿努尔福·罗梅罗-伊-加尔达梅斯主教国际机场|科馬拉帕國際機場]]週邊闢建[[物流|物流]]中心，峻工後將協調薩國免費提供土地及比照在[[台灣的科學園區|台灣的科學園區]]獎勵投資廠商優惠條件，並且引進[[中正國際機場|中正國際機場]]空運物流中心模式，使薩爾瓦多轉型為[[中美洲|中美洲]]營運中心。<ref name="94年外交年鑑"/>

===學術===
{{see also|臺灣教育|{{link-en|薩爾瓦多教育|Education in El Salvador}}}}
[[中華民國外交部|中華民國外交部]]提供「臺灣[[獎學金|獎學金]]」，[[財團法人國際合作發展基金會|財團法人國際合作發展基金會]]（國合會）也提供獎學金給與薩爾瓦多學生赴台就學深造。此外，國合會自2014年派遣[[中華民國國語|華語文]]教師赴該國任教。<ref name="106年外交年鑑"/>根據[[中華民國教育部|中華民國教育部]]清查，2017－2018年（106學年度），薩爾瓦多在台留學生有195人，其中領有台灣獎學金28人。兩國斷交後，薩爾瓦多總統[[萨尔瓦多·桑切斯·塞伦|桑契斯]]表示，為確保安全，所有在台的薩爾瓦多留學生都將送往中國大陆；而台灣沒有在薩爾瓦多留學的學生。<ref name="在台留學生"/>

[[中華民國文化部|中華民國文化部]]與薩爾瓦多{{link-es|唐波斯可大學|Universidad Don Bosco}}、基督教福音大學（Universidad Evangélica de El Salvador）、中美洲科技專科學院（Escuela Especializada en Ingeniería ITCA-FEPADE）簽署「[[臺灣書院|臺灣書院]]」聯絡點意向書。<ref name="102年外交年鑑">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2013/11.pdf|title=《中華民國102年外交年鑑》〈第二章 對外關係〉|publisher=中華民國外交部}}</ref>
