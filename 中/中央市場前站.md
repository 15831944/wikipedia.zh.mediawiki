'''中央市場前站'''（{{jpn|j=中央市場前駅|hg=ちゅうおういちばまええき|rm=Chūō-Ichiba-mae eki|f=y}}）是一個位於[[日本|日本]][[兵庫縣|兵庫縣]][[神戶市|神戶市]][[兵庫區|兵庫區]]，屬於[[神戶市營地下鐵|神戶市營地下鐵]][[海岸線_(神戶市營地下鐵)|海岸線]]的[[鐵路車站|鐵路車站]]。車站編號是'''K05'''。

== 歷史 ==
* 2001年（[[平成|平成]]13年）7月7日：[[神戶市營地下鐵|神戶市營地下鐵]][[海岸線_(神戶市營地下鐵)|海岸線]]的[[三宮·花時計前站|三宮·花時計前站]]－[[新長田站|新長田站]]間開通同時開業。

== 車站結構 ==
島式月台1面2線的地下車站。

== 使用情況 ==
2016年度1日平均'''上車'''人次為'''1,850人'''<ref>[http://www.city.kobe.lg.jp/life/access/transport/kotsu/gaiyou/index.html 神戸市：神戸市交通局　事業概要]</ref>。
