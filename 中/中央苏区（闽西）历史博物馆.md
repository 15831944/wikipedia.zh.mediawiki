'''中央苏区（闽西）历史博物馆'''，位于[[福建省|福建省]][[龙岩市|龙岩市]][[新罗区|新罗区]]北环西路51号，是龙岩市唯一全面反映闽西革命史、重点反映[[中央苏区|中央苏区]]（闽西）历史的综合性专题博物馆。

==简介==
1984年，在[[谭震林|谭震林]]等老一辈无产阶级革命家提议下，始建'''闽西革命历史博物馆'''，1986年建成，1989年正式对外开放。2009年陈列全面改版，同时改造了周边环境。2009年5月被[[国家文物局|国家文物局]]评为国家二级博物馆。2014年4月更名为'''中央苏区（闽西）历史博物馆'''<ref>{{Cite web |url =http://fj.people.com.cn/GB/n2/2017/0207/c234955-29679057.html  |title =中央苏区（闽西）历史博物馆获评第三批国家一级博物馆  |author =  |date =  2017-02-07|publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref><ref name=jj>{{Cite web |url = http://www.crt.com.cn/hsmx/mbjj.html# |title = 中央苏区（闽西）历史博物馆简介 |author =  |date =  |publisher =红色闽西  |language =  |accessdate = 2017-07-05 |quote =  }}</ref>。2017年被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。1988年[[中共中央办公厅|中共中央办公厅]]主任[[温家宝|温家宝]]、1996年[[中共福建省委|中共福建省委]]副书记[[习近平|习近平]]、2004年[[中共中央政治局常委|中共中央政治局常委]][[李长春|李长春]]等领导曾来本馆视察<ref name=jj/>。

中央苏区（闽西）历史博物馆占地面积30亩，建筑面积7000多平米，展厅面积5000多平米，馆藏文物将近两万件。馆内主要设有《红色闽西》基本陈列以及《中央苏区·福建》、《闽西红土名人》等专题陈列<ref name=jj/>。      

中央苏区（闽西）历史博物馆先后成为[[全国爱国主义教育示范基地|全国爱国主义教育示范基地]]、国家国防教育示范基地、全国青少年爱国主义教育基地、[[国家公务员局|国家公务员局]]公务员特色实践教育基地现场教学点、[[井冈山干部学院|井冈山干部学院]]现场教学点、全国社会科学普及教育基地等，并被评为国家3A级旅游景区<ref name=jj/>。
