'''中国－巴基斯坦关系'''（{{lang-ur|پاک چین تعلقات}}）是指中國和巴基斯坦之間的外交關係，兩國于1951年建交。当时剛從[[大英帝國|英國]]獨立不久的[[巴基斯坦|巴基斯坦]]，正式和遷[[台灣|台]]不久的[[中華民國|中華民國]]政府斷絕外交關係，改而和[[中華人民共和國|中華人民共和國]]建交，是世界上最早承認中華人民共和國的國家之一，也是首個和[[北京|北京]]政府建立外交關係的[[伊斯蘭|伊斯蘭]]國家。

1962年，[[中印戰爭|中印戰爭]]爆發後，作為[[印度|印度]]的傳統宿敵的巴基斯坦，和中國的關係變得更親密。冷戰時期，[[中華人民共和國|中華人民共和國]]和[[巴基斯坦|巴基斯坦]]是[[亞洲|亞洲]]區平衡[[蘇聯|蘇聯]]和其盟友[[印度|印度]]的主要力量。1979年，[[阿富汗戰爭_(1979年)|蘇聯入侵阿富汗]]企圖將勢力南下印度，中華人民共和國政府同美國一起，強力支持巴基斯坦聯合抵抗蘇聯的侵略，利用{{le|中国 - 巴基斯坦边境|China–Pakistan border|邊境}}對[[阿富汗|阿富汗]]大量援助。

冷戰後蘇聯解體，地區性的緊張得到舒緩，雖然使中印關係回暖，但中國同巴基斯坦的關係也依舊堅定，透過中方的調停使和平得到確保的條件下，目前在中巴交界處與印度還有領土爭端，為保持印巴地區穩定，中華人民共和國直到今日也是巴基斯坦在軍事、經濟和科技方面的主要支持者，中國民間將其堅固的關係稱作巴鐵。

== 歷史 ==
[[File:Pakistan_Air_Force_Chengdu_JF-17_Gu.jpg|缩略图]]
[[File:Gwadar_Port.jpg|缩略图]]節點-巴達爾港接近完工]]
中國和巴基斯坦的關係友好，双方官方称双方关系为“是長期、全天候和多方面發展的友好關係”，“ 兩國的長久友好關係建基於互信、互惠互利和互不干涉內政等原則”。中国政府用“'''好邻居、好朋友、好兄弟、好伙伴'''”这一组词语形容中巴关系，並且視巴基斯坦為關係最良好的國家之一。<ref name=" fish" >{{cite web |url= http://news.qq.com/a/20130523/020380.htm|title=李克强在巴演讲：宁舍金子 不舍中巴友谊 |accessdate=2013年5月23日 |publisher=腾讯网 |work= |language=zh-cn}}</ref>巴方官员也多次强调中巴之间是“'''特殊的关系'''”，而巴基斯坦往往被中国人称为“'''巴铁'''”。<ref name=" fish" />兩國同时被[[印度|印度]]視為[[假想敵|假想敵]]甚至潛在敵人也是兩國關係快速升溫的重要原因。

[[File:PRC_Towers_and_PNSC_Building_Karachi.jpg|缩略图]]的中華共和大樓(左)]]
巴基斯坦也是中國和西方國家的關係在[[冷戰|冷戰]]中後期正常化的重要橋樑。 1960年代，中國受到西方國家和[[前蘇聯|前蘇聯]]集團的雙重孤立，也和[[印度|印度]]爆發戰爭； 巴基斯坦也因[[克什米爾|克什米爾]]主權問題和印度處於敵對關係，在1965年爆发[[1965年印巴戰爭|第二次印巴戰爭]]。 然而巴基斯坦因為[[地緣政治|地緣]]關係，成為西方抵抗前蘇聯南下印度洋的橋頭堡。美國在[[越戰|越戰]]戰敗，導致對華外交政策改變，1970年在巴基斯坦的協助下，中美兩國雙方秘密接觸，促成美國國家安全顧問[[亨利·基辛格|基辛格]]在1971年訪華，奠定[[中華人民共和國|中華人民共和國]]取得联合国席位和中美建交的基礎。

1979年，前蘇聯[[阿富汗戰爭_(1979年)|入侵]]巴基斯坦的北方鄰國阿富汗，印度也相對偏向前蘇聯，巴基斯坦當局深感雙重威脅，中國則強力支持巴基斯坦抵禦前蘇聯對阿富汗的侵略。

1989年，中國发生[[六四事件|六四事件]]时，巴基斯坦也是少數幾個支持中国政府的国家。巴基斯坦在[[台灣問題|台灣問題]]、[[西藏問題|西藏問題]]上完全支持中華人民共和國，也反對[[西方國家|西方國家]]干涉中國內政。

中華人民共和國也參與和援建很多巴基斯坦的[[基建|基建]]項目，如電訊網絡、1978年落成的[[中巴公路|中巴公路]]和現在的核電廠等。 [[青藏鐵路|青藏鐵路]]通車後，巴基斯坦前總統[[佩爾韋茲·穆沙拉夫|穆沙拉夫]]希望巴基斯坦的鐵路網能和青藏鐵路連接。

中國也是巴基斯坦的主要武器輸出國，也協助巴基斯坦發展武器，[[MBT-2000式主战坦克|MBT 2000坦克]]和[[梟龍戰機|梟龍戰機]]等都是兩國共同研發的現代武器。兩國在1980年代亦達成[[核子|核子]]合作協議，巴基斯坦亦在1998年成功試爆核彈。在反恐問題，除了配合西方國家，巴基斯坦也配合中國在新疆對主張以[[恐怖主義|恐怖主義]]手段爭取[[新疆|新疆]]獨立的[[东突厥斯坦伊斯兰运动|東伊運]]組織的圍剿活動。巴基斯坦軍方在2003年一次[[巴基斯坦|巴]][[阿富汗|阿]]邊境的反恐怖聯合行動中，擊斃了東伊運首領[[艾山·買合蘇木|艾山·買合蘇木]]。 2004年，兩國更首次合作進行反恐演習。

在四川[[汶川大地震|汶川大地震]]发生后，巴基斯坦将国内所有储存的救灾帐篷共计两万多顶全部捐献给了中国，同时还派出了救援人员赴川。

2015年，也门战事突起，安全形势持续恶化。由于巴政府派出的飞机和两艘军舰一时无法满足上千名从也门撤离回国的巴公民的要求，当地情况也十分复杂，巴政府请求中方协助其撤离部分滞留在也门的公民。中方在接到这一请求后启动应急机制，协调有关各方，并派出了正在相关海域执行任务的中国军舰“临沂号”赴亚丁湾参与撤离救援行动。

2018年，巴基斯坦新總理[[伊姆蘭·汗|伊姆蘭·汗]]上任后，中巴關係產生齟齬。[[伊姆蘭·汗|伊姆蘭·汗]]曾批評中国没有给巴基斯坦带来繁荣，而是带来一场危机，並一度對[[中巴经济走廊|中巴经济走廊]]持保留态度<ref>[http://news.dwnews.com/china/news/2018-07-31/60074336.html 中巴关系被指遇冷 巴基斯坦新总理如此表态]</ref><ref>[https://www.bbc.com/zhongwen/simp/world-44956790 巴基斯坦大选：风流倜傥的伊姆兰·汗]</ref>。巴基斯坦部长级官员{{en-link|阿卜杜勒•拉扎克•达乌德|Abdul Razak Dawood}}認為[[中巴经济走廊|中巴经济走廊]]不公平地让中国企业受益<ref>[http://www.ftchinese.com/story/001079339?full=y&archive 巴基斯坦新政府对中巴经济走廊产生疑虑]</ref>。同時在[[新疆再教育营|新疆再教育营]]問題上，巴基斯坦宗教事务部长卡德里批評中国的新疆政策适得其反，只能强化极端分子的立场<ref>[https://www.voachinese.com/a/news-pakistan-urges-china-to-ease-pressure-on-muslims-minority-20180921/4581446.html 巴基斯坦罕见敦促中国善待新疆穆斯林]</ref>。

== 政治关系 ==
[[File:1965-8_1965_周恩来访问巴基斯坦会见总统阿尤布·汉.jpg|thumb]]访问巴基斯坦会见总统[[阿尤布·汗|阿尤布·汗]]]]
* 1950年－[[巴基斯坦|巴基斯坦]]宣佈和[[中華人民共和國|中華人民共和國]]建交，是世界最早承認北京政府的國家之一，也是首個和北京政府建立外交關係的[[伊斯蘭國家|伊斯蘭國家]]。
* 1951年5月21日－[[中華人民共和國|中華人民共和國]]和[[巴基斯坦|巴基斯坦]]正式建交。
* 1962年－[[中印戰爭|中印戰爭]]爆發，成了'''中巴關係'''發展的契機。
* 1963年－[[中華人民共和國|中]][[巴基斯坦|巴]]兩國達成第一份正式貿易協議和邊境協議。
* 1965年－中國在[[1965年印巴戰爭|第二次印巴戰爭]]中公開表明支持巴基斯坦對抗[[印度|印度]]的立場，在[[1971年印巴戰爭|第三次印巴戰爭]]也公開表明支持後者對抗印度和[[東巴基斯坦|東巴基斯坦]]（現在的[[孟加拉國|孟加拉國]]）的立場，中國還在[[聯合國|聯合國]]否決獨立後的孟加拉國的入聯申請，直至1975年兩國建交為止。
* 1970年－在巴基斯坦的協助下，中國和[[美國|美國]]秘密恢復外交接觸，更促成美國國家安全顧問[[亨利·基辛格|基辛格]]在1971年訪華。
* 1996年－中共中央總書記、國家主席、中央軍委主席[[江澤民|江澤民]]國事訪問巴基斯坦。
* 2003年－巴基斯坦軍隊在一次[[巴基斯坦|巴]][[阿富汗|阿]]邊境的反恐怖聯合行動中，擊斃主張以[[恐怖主義|恐怖主義]]手段爭取[[新疆|新疆]]獨立的[[东突厥斯坦伊斯兰运动|東伊運]]首領[[艾山·買合蘇木|艾山·買合蘇木]]。
* 2005年7月4日－在中國支持之下，在[[哈薩克斯坦|哈薩克斯坦]]首都[[阿斯塔納|阿斯塔納]]舉行的[[上海合作組織|上海合作組織]]首腦會議中，巴基斯坦隨[[伊朗|伊朗]]、[[印度|印度]]等獲接納為觀察員。 同年中國亦在巴基斯坦支持之下，獲接納為[[南亞區域合作聯盟|南亞區域合作聯盟]]的觀察員。
* 2013年5月22日－中共中央政治局常委、国务院总理[[李克强|李克强]]对巴基斯坦进行正式访问，巴基斯坦总统扎尔达里、总理霍索到机场迎接。
* 2015年4月20日–中共中央总书记、国家主席、中央军委主席[[习近平|习近平]]携夫人[[彭丽媛|彭丽媛]]访问巴基斯坦。

== 经济關係 ==
* 1978年－連接巴基斯坦東北部和[[新疆維吾爾自治區|新疆維吾爾自治區]]的[[中巴公路|中巴公路]]（接通[[紅其拉甫山口|紅其拉甫山口]]）正式開通。
* 1999年－中國援建巴基斯坦[[旁遮普省|旁遮普省]]的三十萬[[千瓦|千瓦]]核電廠正式投產。
* 2003年－中國國務院副總理[[吳邦國|吳邦國]]參加[[瓜達爾港|瓜達爾港]]深水港口的一期開工儀式，中國在該項目投資近兩億美元。
* 2008年－中巴兩國簽署自由貿易協議。 該年，中巴雙邊貿易總額達到69.81億美元。
* 2013年－[[中巴經濟走廊|中巴經濟走廊]]開始推動。

== 军事關係 ==
[[File:Fighter_plane_At_PAF_Museum.JPG|缩略图]]
* 1962年－[[中印戰爭|中印戰爭]]爆發，成了中巴關係發展的契機。
* 1965年－中國在[[1965年印巴戰爭|第二次印巴戰爭]]中公開表明支持巴基斯坦對抗[[印度|印度]]的立場，在[[1971年印巴戰爭|第三次印巴戰爭]]也公開表明支持後者對抗印度和[[東巴基斯坦|東巴基斯坦]]（現在的[[孟加拉國|孟加拉國]]）的立場。 同年，[[美國|美國]]為回應第二次印巴戰爭，減少對巴基斯坦的軍事援助，中國成為了巴基斯坦主要武器供應國。
* 1980年代－中國和美國均透過巴基斯坦支持[[阿富汗|阿富汗]]對抗[[阿富汗戰爭_(1979年)|蘇聯侵略]]。
* 1986年－中巴兩國達成[[核子|核子]]合作協議。
* 2001年－中巴兩國合作研制的[[坦克|坦克]] - [[MBT-2000式主战坦克|MBT 2000]]（巴基斯坦名稱：'''哈利德'''，'''Al-Khalid'''）全面投產。
* 2004年8月－中巴兩國在新疆[[塔什庫爾干塔吉克自治縣|塔什庫爾干塔吉克自治縣]]地區合作進行首次聯合反恐演習。
* 2007年－中巴兩國合作研制的 - [[梟龍戰機|梟龍戰機]]（巴基斯坦名稱：'''JF-17雷電'''，'''JF-17 Thunder'''）正式登場，巴基斯坦亦在2008年大量生產。
