'''中華民國與伊朗關係'''即[[中華民國|中華民國]]與[[伊朗伊斯蘭共和國|伊朗伊斯蘭共和國]]之間的關係。

1942－1971年，兩國有正式[[外交#外交上的承認|外交關係]]。斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對伊朗的相關事務由'''駐[[杜拜|杜拜]][[臺北經濟文化代表處|臺北商務辦事處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-199-96551-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）在伊朗首都設有'''[[德黑蘭|德黑蘭]]臺灣貿易中心'''，拓展雙方貿易往來業務。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#MID-EAST |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[巴勒维王朝|伊朗帝國]]。]]
{{see also|中華民國外交|{{link-en|伊朗外交|Foreign relations of Iran}}}}
{{see also|中國駐伊朗大使列表{{!}}中華民國駐伊朗外交官列表|伊朗驻华大使列表{{!}}伊朗駐中華民國外交官列表}}
1942年，建立公使級[[外交#外交上的承認|外交關係]]，於首都[[德黑蘭|德黑蘭]]設立'''中華民國駐[[巴勒维王朝|伊朗帝國]][[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1945年2月10日，升格為大使級外交關係。公使館升格為'''中華民國駐伊朗帝國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，基於[[冷戰|冷戰]]的背景且[[反共主義|反共]]立場一致，兩國繼續維持邦交。

1956年12月31日，總統[[蔣中正|蔣中正]]接見{{link-en|伊朗參議員|Senate of Iran}}馬蘇第時表示：「[[共匪|共匪]]侈言與吾人談和之時，其真義乃欲以政治手段達到彼等前此未能以武力獲得之目的。吾人以往與共匪談判，已有太多經驗，絕不致再受共匪最近和平攻勢之欺騙。我政府決不考慮任何共匪和平提議。」

1958年，伊朗國王[[穆罕默德-礼萨·巴勒维|穆罕默德-礼萨·巴勒维]]訪問中華民國，並與總統蔣中正會晤。巴勒維也是唯一曾造訪台灣的[[伊朗君主列表|伊朗君主]]。

1971年8月17日，兩國斷交。18日，關閉大使館；<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>10月25日，伊朗投票反對[[中國與聯合國#代表權問題|中華民國續留聯合國]]。

[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境伊朗。]]
[[File:Visa_requirements_for_Iranian_citizens.png|thumb]]的{{link-en|伊朗人口|Demographics of Iran|伊朗公民}}必須申請[[簽證|簽證]]方可入境[[中華民國|中華民國]]。]]

{{see also|伊朗簽證政策|{{link-en|伊朗公民簽證要求|Visa requirements for Iranian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境伊朗，但僅限觀光與商務：觀光停留15－30天、商務停留14天。護照文件有[[以色列签证政策|以色列簽證]]或入出境章戳者，得拒發落地簽證。若要申請一般簽證，則須前往伊朗駐外使領館申請（例如駐[[阿拉伯聯合大公國|阿拉伯聯合大公國]]的使領館）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-199-96551-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=7F220D7E656BE749 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref><ref name="與我國經貿關係"/>

持有[[伊朗護照|伊朗護照]]的{{link-en|伊朗人口|Demographics of Iran|伊朗公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將伊朗按地區歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往，與'''紅色警戒'''：不宜前往，宜儘速離境（以上截至2018年1月3日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-74-d1220-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將伊朗歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2015年9月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=600FC5F408F8353E |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|伊朗經濟}}
===貿易===
2000年上半年，台灣[[中鼎工程|中鼎工程]]與伊朗經過多次談判後，於次年在伊朗中部大城[[伊斯法罕|伊斯法罕]]合作設立ABS工廠，總工程費2,800多萬[[美元|美元]]。此為[[伊朗伊斯蘭革命|伊朗伊斯蘭革命]]後20多年來，兩國間第一項合作工程。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref>

2009年5月，[[中華民國經濟部|經濟部]]次長[[林聖忠|林聖忠]]率領「2009年[[中東|中東]]貿易訪問團」共59家廠商81位團員訪問伊朗首都[[德黑蘭|德黑蘭]]，為兩國斷交後，訪問伊朗層級最高的政府官員，該團也是中華民國歷年規模最大的中東地區拓銷活動。<ref name="與我國經貿關係"/>

2011年10月，中華民國與[[伊朗伊斯兰共和国中央银行|伊朗中央銀行]]合作，設立「與伊朗貿易金融管理措施」，接受6家非受聯合國[[經濟制裁|制裁]]的民營銀行之貿易文件，透過此一措施，雙方可以取得貨款。<ref name="與我國經貿關係"/>

2018年，伊朗是中華民國第35大貿易夥伴、第31大[[進口|進口]]夥伴、第41大[[出口贸易|出口]]夥伴。出口至伊朗的金額為4億4,245萬1,227[[美元|美元]]，年減32.235%；自伊朗進口的金額為11億4,330萬2,548美元，年增3.126%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）7億0,085萬1,321美元，年增53.790%。2017年，伊朗是中華民國第30大貿易夥伴、第31大進口夥伴、第31大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

中華民國是伊朗所生產之[[尿素|尿素]]、[[氨|氨]]、[[甲醇|甲醇]]、[[乙烯|乙烯]]的主要進口國之一。<ref>{{Cite web |url = https://www.taitraesource.com/total01.asp |title = 主要產業概況 |publisher = 中華民國對外貿易發展協會}}</ref>[[華碩|華碩]]（ASUS）及[[宏碁|宏碁]]（Acer）的[[個人電腦|個人電腦]]在伊朗[[市場佔有率|市場佔有率]]分別排名第一及第二位。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=667321&url=/NewsSearch.asp |title = 台灣產品在伊朗伊斯法罕備受歡迎 |author = 鄭雅福 |date = 2015年6月12日 |publisher = 經貿透視網}}</ref>

====對伊朗經濟制裁====
由於[[伊朗核問題|伊朗發展核計畫]]，受到聯合國[[禁運|禁運]]的[[經濟制裁|制裁]]，進口中華民國的[[機械|機械]]、[[電子零件|電子零組件]]等「戰略性[[高科技|高科技]]產品」不得不從[[阿拉伯聯合大公國|阿拉伯聯合大公國]]或[[土耳其|土耳其]]等第三國間接進入境內，導致中華民國實際對伊朗的出口值無法反映在[[海關|海關]]統計上，從而產生失真情形。也由於國際間對伊朗實施經濟制裁，產品出口至伊朗前，必須先送[[經濟部國際貿易局|經濟部國際貿易局]]審核，以取得核發的「戰略性高科技貨品輸出許可證」，項目達392項，間接影響業者對伊朗出口。<ref name="與我國經貿關係"/>直到2016年1月[[伊朗核問題|伊朗核問題]]解決後，貿易關係才逐漸恢復。

2018年11月5日，[[美國退出聯合全面行動計畫|美國重啟對伊朗經濟制裁]]，禁止各國向伊朗採購[[石油|石油]]，但也暫時允許包括中華民國（[[台灣|台灣]]）、[[印度|印度]]、[[韓國|韓國]]、[[土耳其|土耳其]]、[[義大利|義大利]]、[[希臘|希臘]]、[[日本|日本]]及[[中華人民共和國|中華人民共和國]]在內的8個國家地區繼續進口伊朗石油，豁免期限最高180天<ref>{{cite news|url=https://www.chinatimes.com/newspapers/20181107000299-260203|title=豁免國增購 伊朗石油出口回升|publisher=[[工商時報|工商時報]]|date=2018-11-07}}</ref>。根據中華民國[[經濟部能源局|經濟部能源局]]統計，2017年中華民國向伊朗採購石油約400萬[[桶_(单位)|桶]]，佔石油進口比率不到2%。除了伊朗，中華民國還向[[沙烏地阿拉伯|沙烏地阿拉伯]]、[[科威特|科威特]]、[[阿曼|阿曼]]、[[伊拉克|伊拉克]]、[[阿拉伯聯合大公國|阿拉伯聯合大公國]]、[[安哥拉|安哥拉]]和[[印尼|印尼]]等產油國採購石油，若豁免期滿不得再向伊朗採購，石油供應不會受到太大影響。<ref>{{Cite web |url = https://udn.com/news/story/11314/3463157 |title = 美啟動制裁 禁購伊朗石油 我在豁免名單內 |author = 陳韋廷 |date = 2018年11月6日 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref>
