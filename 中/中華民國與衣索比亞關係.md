===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|衣索比亞簽證政策|衣索比亞公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可於抵達首都的[[亚的斯亚贝巴博莱国际机场|阿迪斯阿貝巴博萊國際機場]]時辦理觀光[[簽證#落地签证|落地簽證]]入境，停留最多30天。非觀光類簽證可至{{link-en|衣索比亞駐外機構列表|List of diplomatic missions of Ethiopia|衣索比亞駐外使領館}}辦理。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-192-4eeea-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|衣索比亞護照|Ethiopian passport}}的{{link-en|衣索比亞人口|Demographics of Ethiopia|衣索比亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將衣索比亞歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2018年2月22日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-105-d140c-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

== 經濟 ==
{{see also|臺灣經濟|{{link-en|衣索比亞經濟|Economy of Ethiopia}}}}
=== 貿易 ===
2017年，衣索比亞是中華民國第108大貿易夥伴、第104大[[進口|進口]]夥伴、第103大[[出口贸易|出口]]夥伴。出口至衣索比亞的金額為2,549萬8,028[[美元|美元]]，年增7.743%；自衣索比亞進口的金額為1,783萬5,658美元，年增72.222%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）766萬2,370美元，年減42.429%。※2016年，衣索比亞是中華民國第110大貿易夥伴、第111大進口夥伴、第102大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
