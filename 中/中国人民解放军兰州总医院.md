'''中国人民解放军兰州总医院'''，位于[[甘肃省|甘肃省]][[兰州市|兰州市]][[七里河区|七里河区]]南滨河路333号，隶属[[西宁联勤保障中心|西宁联勤保障中心]]，为[[三级甲等医院|三级甲等医院]]。

==简介==
1938年11月筹办'''八路军军医院'''，院址设在[[延安|延安]]拐峁村（今属[[陕西省|陕西省]][[延安市|延安市]][[宝塔区|宝塔区]]李渠镇）。苏井观任院长，汪东兴任政委。1939年5月3日，八路军军医院开院典礼举行，八路军总政治部主任[[王稼祥|王稼祥]]、副主任[[谭政|谭政]]，军委参谋长[[滕代远|滕代远]]与会祝贺，[[印度援华医疗队|印度援华医疗队]]成员也参加开院典礼<ref name=yan'an/>。1939年11月12日，[[白求恩|白求恩]]殉职。1939年12月1日，八路军总部颁发命令，将八路军军医院更名为'''白求恩国际和平医院'''<ref>{{Cite web |url =http://mil.news.sina.com.cn/2005-10-10/0620324304.html  |title =抗战中的延安白求恩国际和平医院  |author =  |date = 2005-10-10 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>（后来设分院后改称'''白求恩国际和平医院总院'''<ref name=bicun>{{Cite web |url = http://www.jinsui.org/lishi/gemingjinianguan/2015/1201/3218.html |title =碧村  |author =  |date =2015-12-01  |publisher =  红色晋绥|language =  |accessdate =  |quote =  }}</ref>）。1940年初，白求恩国际和平医院迁驻柳树店（今属陕西省延安市宝塔区桥沟镇），院长鲁之俊，政委刘新权。1943年3月20日，与刘万家沟的八路军野战医院对换驻地，从柳树店迁驻刘万家沟（今属陕西省延安市宝塔区李渠镇）。八路军野战医院于1940年6月11日正式成立，隶属八路军留守兵团，1940年9月八路军医科大学成立后，八路军野战医院成为该校临床实习医院，1943年3月迁至柳树店<ref name=yan'an>涂通今，回忆延安和平医院，中国医院管理1985(3):41-44</ref>。

1944年春，美国医生[[马海德|马海德]]从军直门诊部调到延安白求恩国际和平医院外科工作。延安白求恩国际和平医院的医生还有[[祁开仁|祁开仁]]、[[谭壮|谭壮]]、[[黄树则|黄树则]]、[[李亭植|李亭植]]、[[徐根竹|徐根竹]]等，翻译有[[董均伦|董均伦]]。加拿大、美国援华医疗队，印度援华医疗队、公谊救护队的医务人员先后在延安白求恩国际和平医院工作。1946年10月30日，英国援华会会长克利浦斯夫人一行访问延安期间，曾参观该医院。1946年6月，全面内战爆发。1946年11月，白求恩国际和平医院总院奉命改编为'''陕甘宁晋绥联防军卫生部第二后方医院'''，撤出延安，同时将白求恩国际和平医院总院的名称转交给延安中央医院（后来演变为[[北京医院|北京医院]]、[[西安市儿童医院|西安市儿童医院]]、[[唐都医院|唐都医院]]等）<ref name=yan'an/><ref>{{Cite web |url =http://cpc.people.com.cn/GB/64162/64172/85037/85039/6040385.html  |title =中共领袖与延安中央医院往事  |author =  |date =  |publisher =中国共产党新闻网  |language =  |accessdate = 2017-06-16 |quote =  }}</ref>。

1947年7月，医院番号改为'''中国人民解放军西北野战军第二后方医院'''，归[[中国人民解放军西北野战军|中国人民解放军西北野战军]]后勤部领导。1949年2月，番号改为'''中国人民解放军第一野战军第二野战医院'''。1949年8月，[[兰州战役|兰州战役]]胜利，医院移驻[[兰州|兰州]]，奉命收编了国民党军队的第五十七医院和第二四九医院的医务人员。1950年7月，与已被西北军区接管的原[[西北军政长官公署|西北军政长官公署]]下辖的“兰州中央医院”合并，改编成'''中国人民解放军西北军区第一陆军医院'''<ref name=gsszjsz> {{Cite book | author = 甘肃省地方志编纂委员会 | title =  甘肃省志·军事志（下）| location =  | publisher = 甘肃人民出版社 | date = 2001年 | pages =  1192-1193| ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。1951年7月，西北军区第一陆军医院第二部分出，成立西北军区第四陆军医院（后改为[[中国人民解放军第一医院|中国人民解放军第一医院]]）；西北军区第一陆军医院第三部分出，成立西北军区第五陆军医院（后改为[[中国人民解放军第三医院|中国人民解放军第三医院]]）。

1954年，西北军区第一陆军医院改编为'''中国人民解放军西北军区总医院'''，归西北军区后勤部领导。1955年5月，更名为'''中国人民解放军兰州军区总医院'''，升格为正师级单位<ref name=gsszjsz/>。1994年改编为'''中国人民解放军兰州军区兰州总医院'''。医院是首批[[三级甲等医院|三级甲等医院]]<ref>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536819.htm |title = 兰州军区兰州总医院 |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

2004年6月1日，[[兰州军区空军|兰州军区空军]]的中国人民解放军第四七三医院正式被兰州军区兰州总医院接管，并更名为中国人民解放军兰州军区兰州总医院安宁分院<ref>{{Cite web|url =http://www.gs.xinhuanet.com/news/2004-06/11/content_2294695.htm|title =兰空473医院改为陆军总院安宁分院|author =|date =2004-06-11|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20040702221340/http://www.gs.xinhuanet.com/news/2004-06/11/content_2294695.htm|archivedate =2004-07-02}}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，兰州军区兰州总医院由[[中国人民解放军兰州军区|中国人民解放军兰州军区]]转隶[[西宁联勤保障中心|西宁联勤保障中心]]，并更名'''中国人民解放军兰州总医院'''<ref>{{Cite web |url = http://money.163.com/16/1112/16/C5MFBV6P002580S6.html |title = 高汛出任解放军西宁联勤保障中心政委 |author =  |date =  2016-11-12|publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1730559 |title = 解放军成都、兰州总医院已隶属于西宁联勤保障中心 |author =  |date =2017-07-11  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.81.cn/jwgz/2017-02/05/content_7476475.htm |title =草药“独一味”当年救治红军 如今助藏区群众致富  |author =  |date = 2017-02-05  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.gansu.gov.cn/art/2017/1/24/art_4785_298690.html |title = 甘肃省人民政府关于2016年度甘肃省科学技术奖励的决定 |author =  |date =2017-01-24  |publisher =甘肃省人民政府  |language =  |accessdate =  |quote =  }}</ref>。
