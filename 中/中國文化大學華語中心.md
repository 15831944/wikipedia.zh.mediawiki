
'''中國文化大學華語中心'''（Mandarin Learning Center，縮寫 MLC），隸屬於[[中國文化大學推廣教育部|中國文化大學推廣教育部]]。每年有將近1千位外籍生來MLC研習中文。

== 成立背景 ==
[[中國文化大學|中國文化大學]]華語中心於1992年成立，提供外籍生華語課程，由華語教師設計適合各程度的課程，包含了從適合成人、青少年、兒童、[[上班族|上班族]]到從事華語教學的[[台灣人|台灣人]]的訓練。

== 位置 ==
。位於[[中國文化大學推廣教育部|文大推廣部]][[大夏館|大夏館]]大樓內，附近有[[國立臺灣大學|國立臺灣大學]]、[[國立臺灣師範大學|國立臺灣師範大學]]、[[國立臺北教育大學|國立臺北教育大學]]等大專院校，鄰近[[大安森林公園|大安森林公園]]及[[臺北市立圖書館|臺北市立圖書館]]。位於臺北市大安區建國南路二段231號4樓。

== 設備 ==
MLC華語中心設有普通教室，與多種數位學習設備，並有數位學習中心提供網路與電腦的教學使用。</ref>。2013，<ref>{{cite web|author=now news|title=文大推廣部華語中心簡介|url=http://mlc.sce.pccu.edu.tw/?mainNum=0&submenuNum=5&lang=tw}}</ref>

== 課程 ==
MLC使用[[注音符號|注音符號]]和[[漢語拼音|漢語拼音]]兩種拼音系統教授中文，有部分的老師也具備雙語能力。成人班課程為週一至週五每天3小時的訓練，加上每週一次的線上TOP練習，做為老師評量學生進度與各班進度的統一標準。另外設計了HSK考前加強班和全修課程。MLC也提供多種課程長度，可配合學生在台居留期限做選擇。

== 教材 ==
MLC有使用師範大學出版的制式教材，也有設計不同的課程供學生選擇。這些課程包含了電影、中文歌曲、旅遊中文及HSK準備課程等。
