'''中華民國與安道爾關係'''是指[[中華民國|中華民國]]與[[安道爾公國|安道爾公國]]（又稱'''安道爾[[親王國|親王國]]'''）之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對安道爾的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-126-a55d3-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|安道爾簽證政策|Visa policy of Andorra}}|{{link-en|安道爾公民簽證要求|Visa requirements for Andorran citizens}}}}
安道爾沒有加入[[歐洲聯盟|歐洲聯盟]]與《[[申根公約|申根公約]]》，但根據1999年生效的《[[阿姆斯特丹條約|阿姆斯特丹條約]]》使得《申根既有規範》（Schengen acquis）正式成為[[歐盟法律|歐盟法律]]的一部分，能適用歐盟的簽證政策。歐洲申根區給予持有載明[[中華民國國民身份證|身份證]]字號的[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以[[歐洲聯盟簽證政策|申根區免簽證]]入境，停留日數與申根區合併計算，每6個月期間內總計可停留90天。經安道爾轉機，亦可免簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-126-a55d3-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_FAQ.aspx?c=AD6908DFDDB62656&n=C1B19EF0ACDD3C91&sms=867F5398A49F758A |title = 歐盟免申根簽證常見問答集 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|安道爾護照|Andorran passport}}的{{link-en|安道爾人口|Demographics of Andorra|安道爾公民}}也適用於歐盟國家公民[[簽證#签证优惠待遇|免簽證]]入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將安道爾歸類為'''灰色警示'''：提醒注意（2017年4月22日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-104-1f504-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2018年，安道爾是中華民國第208大貿易夥伴、第205大[[進口|進口]]夥伴、第203大[[出口贸易|出口]]夥伴，在[[歐洲國家和地區列表|歐洲國家]]排名倒數第2、3、2；在[[世界政區索引|世界經濟體]]倒數第37、36、34。出口至安道爾的金額為21萬7,393[[美元|美元]]，年增177.400%；自安道爾進口的金額為3萬4,530美元，年減72.439%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）18萬2,863美元，年減489.750%。2017年，安道爾是中華民國第211大貿易夥伴、第188大進口夥伴、第212大出口夥伴，在歐洲國家排名倒數第2、3、2；在世界經濟體倒數第28、49、19。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
