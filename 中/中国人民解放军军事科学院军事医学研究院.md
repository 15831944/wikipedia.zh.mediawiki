'''中国人民解放军军事科学院军事医学研究院'''，位于[[北京市|北京市]][[海淀区|海淀区]][[太平路_(海淀区)|太平路]]27号，是[[中国人民解放军军事科学院|中国人民解放军军事科学院]]的研究院之一，是[[中国人民解放军|中国人民解放军]]的最高[[军事医学|军事医学]]研究机构。

== 沿革 ==
1951年6月16日，[[中共中央|中共中央]]、[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]发出“关于迅速成立军事医学科学院”的命令，[[中国科学院|中国科学院]]被[[中央军委|中央军委]]赋予[[原子武器|原子武器]]、[[化学武器|化学武器]]、[[生物武器|生物武器]]等[[大规模杀伤性武器|大规模杀伤性武器]]军事医学防护研究的任务。1951年8月1日，'''中国人民解放军军事医学科学院'''在[[上海|上海]]成立，成为继[[中国科学院|中国科学院]]之后[[中华人民共和国|中华人民共和国]]成立的第二家国家级科学院。1956年，军事医学科学院制定了第一个“20年发展规划”。1958年，该院迁至北京<ref name=cpc>{{Cite web |url =  http://cpc.people.com.cn/GB/64093/64387/10830469.html|title =  祖国至上—军事医学科学院忠诚履行使命纪实|author =  |date =2010-01-24  |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://health.people.com.cn/GB/26466/232041/index.html  |title = 人民网专题：庆祝军事医学科学院成立60周年 |author =  |date = 2011 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。

1984年，院党委提出了《关于我院改革的初步设想》，经[[中国人民解放军总后勤部|中国人民解放军总后勤部]]批准开展改革试点；1985年，又作出《关于进一步搞好我院科技体制改革的决定》和12个暂行规定<ref>锐意改革，稳步前进——军事医学科学院科研改革取得初步成效，人民军医1987年06期</ref>。1985年，中央军委决定[[百万大裁军|百万大裁军]]，军事医学科学院推行了“从单纯科研型向科研开发型转变，从单纯军用型向军民结合型转变，从封闭型向开放型转变”的科技改革<ref name=cpc/>。

2003年8月10月，'''中国人民解放军疾病预防控制中心'''在军事医学科学院成立<ref name=cpc/>。2003年10月28日，中国人民解放军总后勤部卫生部卫勤研究室、中国人民解放军医学图书馆、中国人民解放军总后勤部卫生防疫队并入中国人民解放军军事医学科学院暨成立中国人民解放军疾病预防控制中心大会在北京的中国人民解放军军事医学科学院召开<ref>杨忠，中国人民解放军疾病预防控制中心在京成立，解放军预防医学杂志2004年第1期</ref>。2004年该中心被国家批准纳入公共卫生体系<ref name=cpc/>。2013年12月1日，由中国人民解放军总后勤部卫生部组织搭建的中国人民解放军疾病预防控制网（<nowiki>http://jkw.zh/</nowiki>）上线运行，这是中国人民解放军全军第一个面向基层部队的疾病预防控制网<ref>{{Cite web |url = http://mil.news.sina.com.cn/2013-12-13/0450754618.html |title =中国人民解放军疾病预防控制网开通  |author =  |date = 2013-12-13 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

军事医学科学院原来隶属[[中国人民解放军总后勤部|中国人民解放军总后勤部]]。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军总后勤部被撤销，成立[[中央军事委员会后勤保障部|中央军事委员会后勤保障部]]，军事医学科学院改隶中央军事委员会后勤保障部<ref>{{Cite web |url = http://news.163.com/16/0427/10/BLLEKOHC00014SEH.html  |title = 质检总局与军事医学科学院签署《公共卫生安全战略合作框架协议》落实军民融合 保卫公共安全 |author =  |date =  2016-04-27|publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。仍为正军级。

军事医学科学院承担[[军事医学|军事医学]]、[[基础医学|基础医学]]、[[生物技术|生物技术]]、卫生装备、药物研究等任务，负责军事斗争、反恐防恐、重大疾病防控的卫生勤务任务。

2017年7月19日，新调整组建的军事科学院、国防大学、国防科技大学成立大会暨军队院校、科研机构、训练机构主要领导座谈会在北京[[八一大楼|八一大楼]]举行，[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军委主席|中央军委主席]][[习近平|习近平]]向军事科学院、国防大学、国防科技大学授军旗、致训词，出席座谈会并发表讲话<ref name=xunci>{{Cite web |url =http://news.xinhuanet.com/politics/2017-07/19/c_1121347127.htm  |title = 习近平向军事科学院、国防大学、国防科技大学授军旗致训词 |author =  |date = 2017-07-19 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。军事医学科学院并入新调整组建的军事科学院，成立'''中国人民解放军军事科学院军事医学研究院'''，成为军事科学院8个研究院之一。其级别可能是副军级<ref name=yanjiuyuan>{{Cite web |url = http://www.sohu.com/a/168412197_495232 |title =  新军事科学院的几个研究院|author =  |date = 2017-08-30 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

== 机构设置 ==
2017年改革前，原中国人民解放军军事医学科学院下设：
* 卫生勤务与医学情报研究所
* 放射与辐射医学研究所
* 基础医学研究所
* 卫生学环境医学研究所
* 微生物流行病研究所
* 毒物药物研究所
* 卫生装备研究所
* 生物工程等研究所
* 野战输血研究所
* 疾病预防控制所
* 军事兽医研究所
* [[中国人民解放军第三〇七医院|解放军第三〇七医院]]
* [[中国人民解放军医学图书馆|解放军医学图书馆]]
* 实验仪器厂
* 实验动物中心
* [[研究生院|研究生队]]
* 眼科诊疗中心
* 耳鼻喉科诊疗中心
