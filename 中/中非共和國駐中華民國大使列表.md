==歷史==
1991年7月8日中華民國與中非共和國第二次恢復大使級外交關係。

1998年1月29日中非共和國與中華民國第三次斷交。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=5028B03CED127255&sms=5ED24855AD8E6C58&s=D7D460B5F6F6630F |author = 公眾外交協調會 |date = 1998年1月29日 |title = 中華民國外交部對於中非共和國於本（八十七）年元月廿九日與中共簽署所謂「復交」聯合公報發表聲明 |publisher = 中華民國外交部}}</ref>
