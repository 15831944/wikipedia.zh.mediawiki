'''中国一点都不能少'''，或作'''中国，一点都不能少'''，是2016年7月为抗议[[南海仲裁案|南海仲裁案]]在[[中国大陆|中国大陆]]出现的政治口号，最早出现于[[人民日报|人民日报]]官方微博发布的一张图片。

== 事件經過 ==
2016年7月11日，临近[[南海仲裁案|南海仲裁案]]仲裁结果公布，[[人民日报|人民日报]]新媒体中心策划推出“中国一点都不能少”报道专题，以“中国一点都不能少”为报道主题词，通过图片、[[H5|H5动图]]、海报、文章、视频、九宫格图解等形式及时表达中国态度、中国立场，详细解读《中国坚持通过谈判解决中国与菲律宾在南海的有关争议》白皮书，普及相关背景知识<ref name="推荐表">{{cite web |author1=苗苗 |title=中国一点都不能少：中国新闻奖网络新闻作品参评推荐表 |url=http://www.xinhuanet.com//zgjx/2017-06/17/c_136375501.htm |website=新华网 |accessdate=2018-11-19 |date=2017-06-17}}</ref>；该新闻专题後續更获得第二十七届中国新闻奖一等奖<ref>{{cite web |title=第二十七届中国新闻奖获奖作品目录 |url=http://www.wenming.cn/xwcb_pd/yw/201711/t20171102_4474337_1.shtml |website=中国文明网-新华社 |accessdate=2018-11-19 |date=2017-11-02}}</ref>。人民日報同時也在[[新浪微博|新浪微博]]的官方账号发布图片《中国一点都不能少》，配以文字“这才是中国，一点都不能少”，并发起微博话题“#中国一点都不能少#”。随后[[蒋劲夫|蒋劲夫]]、[[贾乃亮|贾乃亮]]、[[杨洋_(男演员)|杨洋]]、[[李晨_(演员)|李晨]]等大量明星转发人民日报的这一微博<ref>{{cite web |title=众明星为南海仲裁案发声 这才是中国一点都不能少 |url=http://www.1905.com/news/20160712/1044731.shtml |website=1905电影网-凤凰娱乐 |accessdate=2018-11-19 |date=2016-07-12}}</ref>。

[[张艺兴|张艺兴]]将自己的Instagram以及微博账号的头像替换为《中国一点都不能少》图片，[[宋茜|宋茜]]转发人民日报微博后在其Instagram账号发布《中国一点都不能少》图片<ref>{{cite news |title=中国一点都不能少！张艺兴、宋茜、wuli韬韬声援表明立场！ |url=http://www.vccoo.com/v/18e866 |date=2016-07-13}}</ref>，两人的Instagram账号下遭到大量[[台湾|台湾]]、[[越南|越南]]、[[菲律宾|菲律宾]]网民谩骂<ref>{{cite web |title=宋茜ins发中国一点都不能少掉粉40万+ 遭台独港独菲律宾喷子群讽 |url=https://www.henan100.com/news/2016/616904.shtml |website=河南一百度}}</ref>。

由于[[黃曉明|黃曉明]]转发“中国，一点都不能少”微博，其主演的电视剧《[[新上海灘_(2006年電視劇)|新上海灘]]》立刻遭到原本正在播出的越南[[平順省|平順省]]的停播<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20160719/910828/  【民報】黃曉明撂話「中國一點都不能少」 越南停播《新上海灘》]</ref>。

==後續引用==
2018年台灣[[第55屆金馬獎|第55屆金馬獎]]政治風波後，[[中國大陸|中國大陸]]藝人紛紛引用此圖表示立場。[[范冰冰|范冰冰]]及范冰冰男友[[李晨|李晨]]、台灣媳婦[[劉詩詩|劉詩詩]]、出爐影帝[[徐崢|徐崢]]、入圍影后的[[周迅|周迅]]及[[孫儷|孫儷]]、現身金馬獎的[[胡歌|胡歌]]及[[鄧超|鄧超]]等藝人，亦在微博轉發言論，表態支持。另外，香港藝人[[陳偉霆|陳偉霆]]、[[溫碧霞|溫碧霞]]、在韓發展的GOT7成員[[王嘉爾|王嘉爾]]等，也有轉發該微博<ref>[https://udn.com/news/story/7332/3489263 范冰冰轉發「中國，一點都不能少」…藉此洗白復出？] 2018.11.19 世界日報 中國新聞組</ref>。

台灣網友反擊貼上圖文「中國一點都不能少，但是可以缺一大塊」<ref>[https://tw.news.appledaily.com/new/realtime/20181118/1468707 中國一點都不能少？　台網友反擊：但可以缺一大塊] 2018.11.18 蘋果日報</ref>，顯示豐碩的中國[[原中華民國公告疆域行政區劃#範圍|秋海棠版圖]]，遭受他國割佔而失去[[國土|國土]]瘦成血色公雞<ref>{{cite news |title=「海棠血淚」 網傳一張中國地圖引熱議《海棠血淚》 |url=https://www.epochtimes.com.tw/n173984/海棠血淚-網傳一張中國地圖引熱議.html |date=2016-07-15}}</ref>，以此譏諷中國大陸公民根本不知道也無權知道中國國土的版圖，有多筆領土遭占，如[[唐努烏梁海|唐努烏梁海]]遭俄羅斯占領。
