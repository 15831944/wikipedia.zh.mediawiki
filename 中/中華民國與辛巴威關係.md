'''中華民國與辛巴威關係'''是指[[中華民國|中華民國]]與[[辛巴威共和國|辛巴威共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對辛巴威的相關事務由'''{{Link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-164-01334-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
[[File:Voting_res_2758.png|thumb]]，[[辛巴威|辛巴威]]當時屬於[[大英帝國#疆土|英國的殖民地]]（1965年單方面獨立為[[羅德西亞|羅德西亞共和國]]），因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境辛巴威。]]

===外交===
{{see also|中華民國外交|{{link-en|辛巴威外交|Foreign relations of Zimbabwe}}}}
2005年3月25日，津巴布韦支持[[中華人民共和國|中華人民共和國]]通过《[[反分裂國家法|反分裂國家法]]》、支持「[[一个中国|一个中国]]」政策，认为[[中华人民共和国疆域#领土争端|台湾是中国领土的一部份]]，并支持中国以[[一国两制|一国两制]]解决[[台湾问题|台湾问题]]<ref>{{cite web|title=一些国家政府支持中国通过《反分裂国家法》 |url=http://news.163.com/05/0325/20/1FNGRE4A0001124T.html |publisher=网易 |date=2005-03-25 |accessdate=2017-04-01}}</ref>。

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|辛巴威簽證政策|{{link-en|辛巴威公民簽證要求|Visa requirements for Zimbabwean citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可至辛巴威移民局[[簽證#电子签证|電子簽證]]網頁申請「入境同意函」，獲准後以[[電子郵件|電子郵件]]通知並列印，持函於抵達{{link-en|辛巴威機場列表|List of airports in Zimbabwe|該國機場}}時繳交簽證費入境。經由[[南非機場列表|南非國際機場]]轉機入出境辛巴威，不需申辦南非[[過境簽證|過境簽證]]；若出機場再轉搭陸上交通工具前往辛巴威，因已實際入境南非，故須申辦[[南非簽證政策|南非簽證]]。相關事宜可詢問辛巴威駐[[約翰尼斯堡|約翰尼斯堡]][[領事館|領事館]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-164-01334-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|辛巴威護照|Zimbabwean passport}}的{{link-en|辛巴威人口|Demographics of Zimbabwe|辛巴威公民}}抵台參加國際會議、運動賽事、商展活動，也可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將辛巴威歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2018年11月1日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-125-1bdd6-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==

===貿易===
2011年9月19日，{{link-en|辛巴威經濟企劃暨投資促進部|Ministry of Economic Planning and Investment Promotion (Zimbabwe)}}部長、全國商業總會主席，以及部長特別助理等人，抵台並訪問[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）。<ref>{{Cite web |url = http://www.trademag.org.tw/content02.asp?id=569416&type=22&url=/NewsSearch.asp |author = 馮翹楚 |date = 2011年10月19日 |title = 辛巴威歡迎台灣業者拓銷產品、爭取商機 |publisher = 經貿透視雙周刊第328期}}</ref>

2017年，辛巴威是中華民國第126大貿易夥伴、第102大[[進口|進口]]夥伴、第160大[[出口贸易|出口]]夥伴。出口至辛巴威的金額為206萬3,816[[美元|美元]]，年增7.929%；自辛巴威進口的金額為1,917萬5,448美元，年增233.504%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）1,711萬1,632美元，年增345.908%。※2016年，辛巴威是中華民國第153大貿易夥伴、第124大進口夥伴、第161大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===犯罪事件===
1名辛巴威籍男子從1991年起，就假冒多種身分詐騙台灣女子，2003年1名受害女老師指控其惡行，[[新北市政府警察局|台北縣警方]]循線將他逮捕，發現前後騙了11名女子，每人被騙約200萬[[新台幣|新台幣]]，後來[[臺北地方法院|臺北地方法院]]判他4年10個月[[有期徒刑|有期徒刑]]。2008年出獄便驅逐出境，數月後，改名並持[[英國護照|英國護照]]再度入境，繼續犯案，2年間又騙了7名女子。[[桃園地方法院檢察署|桃園地方法院檢察署]]還發現，2007年他在獄中，還曾寫信恐嚇他的台灣前妻，威脅要捏造前妻是台灣的[[塔利班|塔利班]]組織成員，向[[英國政府|英國政府]]舉發，[[桃園地方法院|桃園地方法院]]就以[[詐欺|詐欺]]、[[恐嚇|恐嚇]]等罪嫌，判處有期徒刑6年，執行完畢後驅逐出境。<ref>{{Cite web |url = http://news.ltn.com.tw/news/society/paper/481189 |author = 周敏鴻 |date = 2011年4月2日 |title = 假馬蓋先騙色 判6年 |publisher = [[自由時報|自由時報]]}}</ref>
