'''中华人民共和国领导人新年贺词'''是[[中华人民共和国|中华人民共和国]][[党和国家领导人|党和国家领导人]]发表的以庆祝[[格里历|格里历]][[新年|新年]]为主题的贺词。领导人发表的第一篇新年贺词是时任[[中国共产党中央委员会主席|中国共产党中央委员会主席]][[毛泽东|毛泽东]]的《[[将革命进行到底|将革命进行到底]]》，此贺词发表于1949年1月1日的《[[人民日报|人民日报]]》上。2001年，时任中共中央总书记、国家主席、中央军委主席[[江泽民|江泽民]]发表名为《共同促进世界的和平与发展》的新年贺词，通过[[电视|电视]]和[[电台广播|电台广播]]等途径传播。从这一年起，中华人民共和国领导人在每年的12月31日都会发表新年贺词，以[[视频|视频]]的方式通过电视、[[互联网|互联网]]等途径传播。新年贺词的格式也趋向于固定，主要包括对过去一年的总结反思以及对未来一年的展望与计划。<ref>{{cite web |author1=政析 |title=外国重视研究新年贺词的政治信号 |url=http://finance.ifeng.com/a/20090101/294332_0.shtml |website=[[凤凰网|凤凰网]]财经 |publisher=南方都市报 |accessdate=2018-12-31 |date=2009-01-01}}</ref><ref>{{cite web |title=揭秘领导人新年贺词(组图) |url=http://news.163.com/15/0108/10/AFEAGGEV00014AED.html |website=[[网易|网易]]新闻 |publisher=新疆天山网-新疆都市报(乌鲁木齐) |accessdate=2018-12-31 |date=2015-01-08}}</ref><ref>{{cite web |author1=子正君 |title=报纸记录20年国家主席新年贺词 |url=http://collection.sina.com.cn/yjjj/2016-01-01/doc-ifxneept3502945.shtml |website=[[新浪|新浪]]收藏 |accessdate=2018-12-31 |date=2016-01-01 |archive-url=https://web.archive.org/web/20180714011009/http://collection.sina.com.cn/yjjj/2016-01-01/doc-ifxneept3502945.shtml |archive-date=2018-07-14 |dead-url=no }}</ref><ref>{{cite web |title=解密领导人新年贺词 |url=http://news.sohu.com/s2013/dianji-1301/ |website=[[搜狐|搜狐]]新闻 点击今日 |accessdate=2018-12-31 |date=2014.1.1 |archive-url=https://web.archive.org/web/20180611192320/http://news.sohu.com/s2013/dianji-1301/ |archive-date=2018-06-11 |dead-url=no }}</ref>
