'''中国工农红军第二十七军'''（{{kj|1935年|1月30日|1935年|9月18日|H}}），简称'''红二十七军'''，为与同名红军区别，又称'''陕西红二十七军'''，是[[陕北红军|陕北红军]]主力部队之一。

== 历史 ==

1935年，[[中国共产党|中国共产党]]在陕西逐渐建立[[苏维埃|苏维埃]]政权。1月30日，根据[[中共中央军委|中共中央军委]]指示，在[[赤源县|赤源县]]白庙岔（在今[[陕西省|陕西省]][[子长县|子长县]]境）将陕北独立师改编为红二十七军第八十四师，师长[[杨琪_(中共烈士)|杨琪]]、政治委员[[张达志|张达志]]、参谋长[[朱子休|朱子休]]。全师仅700余人、300余枝枪。低于现代军事中[[團_(軍隊)|标准团]]的人数。2月15日，陕北游击队第六、十五支队在[[佳县|佳县]]张家峪合编为红二十七军第五团。团长[[王士杰|王士杰]]、政治委员[[慕生忠|慕生忠]]。3日，军第五团改为师第三团，原师第三团改神府独立团<ref name="a1"/>。

5月1日，与[[红二十五军|红二十五军]]、[[红二十六军|红二十六军]]在赤源县白庙岔[[会师|会师]]。5月，红二十七军扩至1200余人。同年9月18日，与红二十五军、红二十六军合编为[[红十五军团|红十五军团]]。红二十七军改编为红十五军团第八十一师<ref name="a1">{{Cite web |url=http://www.sxsm.gov.cn/zt/sfhjzt/sfhjda/201310/t20131010_164149.html |title=在陕中国工农红军 |accessdate=2016-05-30 |author= |date=2013-10-10 |work= |publisher=神木县人民政府网站，来源：神木新闻网 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160924200732/http://www.sxsm.gov.cn/zt/sfhjzt/sfhjda/201310/t20131010_164149.html |archivedate=2016-09-24 }}</ref>。1937年后改编为[[国民革命军|国民革命军]][[八路军|八路军]][[115师|115师]][[344旅|344旅]]一部，参加[[抗日战争_(中国)|抗日战争]]。1937年1月，以红八十一师为主，与陕北、关中各一个地方团再扩编为红二十七军。军长[[贺晋年|贺晋年]]，政治委员[[李赤然|李宗贵]]（后[[王平_(上将)|王平]]）。此后部队负责肃清延安以南地区土匪武装<ref>{{cite book|author=《中国人民解放军军史》编写组|title=中国人民解放军军史：第一卷|year=2010|publisher=军事科学出版社|pages=472|isbn=978-7-80237-381-5}}</ref>。8月，改编为八路军[[120师|120师]]直属炮兵营和辎重营。

== 编制 ==

*第八十四师<ref name="a1"/>：原陕北独立师第一、二、三团改编为第八十四师第一、二、三团。
**第一团，团长[[贺晋年|贺晋年]]，政治委员[[马佩勋|马佩勋]]，全团200余人，100余支枪
**第二团，团长[[郭玉人|郭玉人]]，政治委员[[马万里|马万里]]，全团300余人，100余支枪
**第三团，团长[[王兆相|王兆相]]，政治委员[[杨文谟|杨文谟]]，后改为神府独立团
**第三团（原军第五团），团长[[王士杰|王士杰]]、政治委员[[王国昌|王国昌]]，全团180余人，80余支枪
