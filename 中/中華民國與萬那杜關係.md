'''中華民國與萬那杜關係'''是指[[中華民國|中華民國]]與[[萬那杜共和國|萬那杜共和國]]之間的關係。兩國於2004年有官方[[外交#外交上的承認|外交關係]]，但7天後即斷交，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對萬那杜的相關事務由'''中華民國駐[[斐濟|斐濟]][[台北经济文化代表处|商務代表團]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-263-ce796-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref><ref>{{Cite web |url = http://www.roc-taiwan.org/fj/post/7.html |title = 代表處簡介 |publisher = 中華民國駐外單位聯合網站}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，[[萬那杜|萬那杜]]當時屬於[[英國|英]]、[[法國|法]]的[[新赫布里底群島共管地|新赫布里底群島共管地]]，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
{{see also|中華民國外交|萬那杜外交}}
1992年，{{link-en|瑟奇·沃霍尔|Serge Vohor}}出任瓦努阿圖外交部長。同年9月，中華民國與瓦努阿圖簽訂《政府间相互承认联合公报》；事後，[[中華民國政府|中華民國政府]]稱這是第一次以「相互承认」的方式与[[中华人民共和国建交列表|中華人民共和國的邦交国]]建立官方关系<ref>{{cite book|author1=高民政|title=台湾政治纵览|publisher=华文出版社|year=2000|isbn=9787507509960}}</ref>{{Rp|311}}。

2004年11月3日，[[萬那杜總理|萬那杜總理]]渥荷密訪[[台北市|台北]]，與[[中華民國外交部#歷任首長|中華民國外交部長]][[陳唐山|陳唐山]]簽署相互承認與建交公報，正式[[外交#外交上的承認|建立外交關係]]。當日於首都[[維拉港|維拉港]]設立'''中華民國駐萬那杜共和國[[外交代表機構#大使馆|大使館]]'''（暫設於[[艾美酒店|艾美酒店]]內）。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=5028B03CED127255&s=A077E424F6A610C5 |author = 公眾外交協調會 |date = 2004年11月3日 |title = 中華民國（台灣）政府與萬那杜共和國政府基於平等互惠之原則暨發展兩國友好關係之共同意願，決定自二○○四年十一月三日起建立大使級外交關係 |publisher = 中華民國外交部}}</ref>

2004年11月10日、18日，{{link-en|萬那杜內閣|Cabinet of Vanuatu}}兩度撤銷與中華民國的建交公報。12月10日，[[瓦努阿图议会|瓦努阿图议会]]對渥荷提出[[不信任動議|不信任案]]，最後決定罷免渥荷的總理職務。<ref>{{Cite web |url = http://www.people.com.cn/GB/guoji/14549/3056845.html |author = 郭拥军 |date = 2004年12月15日 |title = “建交”闹剧笑死人 瓦努阿图不欢迎台湾 |publisher = [[人民網|人民網]]}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境萬那杜。]]
[[File:Visa_requirements_for_Vanuatuan_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|萬那杜簽證政策|{{link-en|萬那杜公民簽證要求|Visa requirements for Vanuatuan citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可在[[瓦努阿圖機場列表|該國機場]]辦理[[簽證#落地签证|落地簽證]]入境萬那杜，停留最多30天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-263-ce796-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&sms=B75E93A3DAB23A99 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|萬那杜護照|Vanuatuan passport}}的{{link-en|萬那杜人口|Demographics of Vanuatu|萬那杜公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將萬那杜歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2016年12月8日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-343-861da-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|萬那杜經濟}}
===貿易===
2018年，萬那杜是中華民國第164大貿易夥伴、第138大[[進口|進口]]夥伴、第182大[[出口贸易|出口]]夥伴。出口至萬那杜的金額為79萬0,079[[美元|美元]]，年減10.063%；自萬那杜進口的金額為262萬1,005美元，年增52.673%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）183萬0,926美元，年增118.416%。2017年，萬那杜是中華民國第174大貿易夥伴、第149大進口夥伴、第180大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
