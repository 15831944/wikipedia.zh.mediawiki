'''中国共产党中央军事委员会常务委员会'''，是[[中国共产党中央军事委员会|中国共产党中央军事委员会]]的常设机关，自1950年代末至1970年代曾两次设立。

==第一次设立==
1959年[[庐山会议|庐山会议]]后，主持[[中国共产党中央军事委员会|中共中央军委]]工作的[[彭德怀|彭德怀]]、[[黄克诚|黄克诚]]（总参谋长）、[[洪学智|洪学智]]等人受到批判，被调离军委和总部领导岗位。1959年9月26日，[[中共中央政治局|中共中央政治局]]决定改组中央军委，新的军委由21人组成，增设[[中国共产党中央军事委员会副主席|副主席]]及'''常务委员会'''（简称军委常委）<ref name=xuping>徐平，中央军委三设办公会议，文史精华2008(3):52-54</ref>。

1959年10月20日，中央军委发出通知，决定在军委常委之下设立一个[[中央军委办公会议|办公会议]]<ref name=xuping/>。

[[文化大革命|文化大革命]]初期，1967年8月17日，根据中共中央、中央军委的决定，成立了由[[吴法宪|吴法宪]]（副总参谋长兼空军司令员）、[[叶群|叶群]]（林彪办公室主任）、邱会作（副总参谋长兼总后勤部部长）、[[张秀川|张秀川]]（海军副政委）组成的[[军委办事组|军委办事组]]，由吴法宪负责。军委办事组实际取代了中央军委常委和办公会议<ref name=xuping/>。

==第二次设立==
1971年[[九一三事件|九一三事件]]发生。1971年10月3日，中共中央决定撤销军委办事组，成立新的[[中央军委办公会议|中央军委办公会议]]。由军委副主席叶剑英主持军委日常工作<ref name=xuping/>。

1974年1月11日，[[南越|南越]]当局公布地图，将中国[[西沙群岛|西沙群岛]]全部划入其版图，并于1974年1月15日出动军舰突袭[[永乐群岛|永乐群岛]]。1974年1月17日，叶剑英指示总参谋部，组织班子，准备打仗。1974年1月18日，[[中共中央政治局|中共中央政治局]]会议决定成立“五人小组”，由叶剑英牵头，成员有[[王洪文|王洪文]]、[[张春桥|张春桥]]、[[邓小平|邓小平]]、[[陈锡联|陈锡联]]（后[[苏振华|苏振华]]也参加，小组改为“'''六人小组'''”）。小组的任务是处理中央军委的大事及紧急作战事项。1月19日凌晨，叶剑英接到报告，南越当局派兵强行登上[[琛航岛|琛航岛]]、[[广金岛|广金岛]]，并率先向[[中国民兵|中国民兵]]开炮。叶剑英当即请示[[周恩来|周恩来]]，周恩来让叶剑英立即组织领导该小组，由叶剑英负责指挥作战。1月20日，[[中国人民解放军|中国人民解放军]]向甘泉岛、珊瑚岛、金银岛的南越敌军发起反攻，全歼敌军，收复了西沙群岛中的永乐群岛，取得[[西沙之战|西沙海域自卫反击作战]]胜利<ref>{{Cite web |url =  http://dangshi.people.com.cn/GB/144956/11158277.html|title =历数中国八位国防部长：叶剑英坐镇指挥粉碎“四人帮”  |author =  |date =2010-03-17  |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。

1975年1月5日，中共中央任命[[邓小平|邓小平]]为中央军委副主席兼[[中国人民解放军总参谋长|中国人民解放军总参谋长]]<ref name=xuping/>。1975年1月26日，叶剑英致信毛泽东，提出鉴于1971年10月成立的中央军委办公会议成员情况有变化，为加强对军队和战备工作的具体领导，建议将[[西沙之战|西沙作战]]后成立的中央军委六人小组扩大，组成军委常务委员会，并增补[[刘伯承|刘伯承]]、[[徐向前|徐向前]]、[[聂荣臻|聂荣臻]]、[[粟裕|粟裕]]4人为常务委员。此信获得中共中央和毛泽东的批准<ref>{{Cite book |url =  |title = 历史的丰碑:中华人民共和国国史全鉴 （七 军事卷） |author =中共中央党校理论研究室  |date =2004年  |publisher =中共中央文献出版社  |language =  |accessdate =  |quote = 中央军委常务委员会 }}</ref><ref>{{Cite web |url =  http://cpc.people.com.cn/GB/64093/67507/4854137.html|title = 叶剑英大事年表 （1970年――1979年） |author =  |date = 2006-09-25 |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。 

1975年2月5日，中共中央决定取消军委办公会议，重新设立中央军委常务委员会，作为处理军委日常工作的办事机构，叶剑英等11人为常务委员，常务委员会由叶剑英主持。同时决定，[[中国人民解放军总参谋部|中国人民解放军总参谋部]]第一副总参谋长[[杨成武|杨成武]]、[[中国人民解放军总政治部|总政治部]]第一副主任[[梁必业|梁必业]]、[[中国人民解放军总后勤部|总后勤部]]党委第一书记（总后勤部部长）[[张宗逊|张宗逊]]、[[中央军委办公厅|军委办公厅]]主任[[胡炜|胡炜]]列席军委常务委员会会议，此即所谓列席常委。同年，增补[[李先念|李先念]]、[[王震|王震]]为中央军委常委<ref name=xuping/>。

1976年2月2日，叶剑英被停止主持军委日常工作，改由[[陈锡联|陈锡联]]主持。同年4月7日，中共中央军委副主席兼总参谋长邓小平被撤销党内外一切职务。1976年10月6日，[[粉碎四人帮|粉碎“四人帮”]]。1976年10月7日，中共中央政治局决定[[华国锋|华国锋]]任[[中共中央主席|中共中央主席]]、中共中央军委主席。1977年3月7日，中共中央通知，叶剑英恢复主持军委日常工作，陈锡联协助工作。1977年7月16日，[[中共十届三中全会|中共十届三中全会]]正式追任华国锋为中共中央主席、中央军委主席，并决定恢复邓小平的中共中央副主席、中共中央军委副主席、国务院副总理、中国人民解放军总参谋长等职务<ref name=xuping/>。

1977年8月19日，[[中共十一届一中全会|中共十一届一中全会]]通过了由63人组成的中共中央军委，军委主席为[[华国锋|华国锋]]，副主席为[[叶剑英|叶剑英]]、[[邓小平|邓小平]]、[[刘伯承|刘伯承]]、[[徐向前|徐向前]]、[[聂荣臻|聂荣臻]]；军委常委14人，列席常委3人<ref name=xuping/>。

1979年11月4日，中共中央决定，在中央军委常委领导下恢复[[中央军委办公会议|办公会议]]，负责处理军委日常工作。军委办公会议由耿飚主持。军委办公会议建立后，不再设军委列席常委<ref name=xuping/>。

1980年2月29日，[[中共十一届五中全会|中共十一届五中全会]]批准汪东兴、陈锡联的辞职请求，免去其中央军委常委职务。1981年6月27日到29日召开的[[中共十一届六中全会|中共十一届六中全会]]一致同意[[华国锋|华国锋]]辞去[[中共中央主席|中共中央主席]]、中央军委主席职务的请求，邓小平当选为中央军委主席。1981年7月10日，中共中央、中央军委决定，[[杨尚昆|杨尚昆]]任中共中央军委常委、秘书长，免去[[耿飚|耿飚]]的中央军委秘书长职务<ref name=xuping/>。

1982年起，[[中共中央军事委员会|中共中央军事委员会]]和[[中华人民共和国中央军事委员会|中华人民共和国中央军事委员会]]共同设立。中央军事委员会既是中共中央的军事机关，也是国家机构的组成部分<ref name=xuping/>。

1982年9月12日，[[中共十二届一中全会|中共十二届一中全会]]决定邓小平任中共中央军委主席，叶剑英、徐向前、聂荣臻、杨尚昆任中共中央军委副主席，其中杨尚昆为常务副主席。1982年9月30日，中共中央任命杨尚昆兼任军委秘书长，[[余秋里|余秋里]]、[[杨得志|杨得志]]、[[张爱萍|张爱萍]]、[[洪学智|洪学智]]任军委副秘书长，并决定中共中央军委由主席、副主席、秘书长、副秘书长组成，由秘书长、副秘书长组成[[中央军委常务会议|常务会议]]，负责处理军委日常工作。此后，中央军委不再设立常务委员会以及办公会议<ref name=xuping/>。
