'''中華民國與蘇利南關係'''是指[[中華民國|中華民國]]與[[蘇利南共和國|蘇利南共和國]]之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對蘇利南的相關事務由'''中華民國駐[[聖文森及格瑞那丁|聖文森國]][[中華民國駐外機構列表|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-259-b111c-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
[[File:Voting_res_2758.png|thumb]]，[[蘇利南|蘇利南]]當時屬於{{link-en|荷屬蘇利南|Suriname (Kingdom of the Netherlands)|荷蘭的海外自治省}}，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境蘇利南。]]
[[File:Visa_requirements_for_Surinamese_citizens.png|thumb]]的{{link-en|蘇利南人口|Surinamese people|蘇利南公民}}必須申請簽證方可入境[[中華民國|中華民國]]。]]
===外交===

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|蘇利南簽證政策|{{link-en|蘇利南公民簽證要求|Visa requirements for Surinamese citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]與邀請函的[[中華民國國民|中華民國公民]]可至[[苏里南驻外机构列表|蘇利南駐外使領館]]申請簽證。經蘇利南轉機，亦須簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-259-b111c-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持有[[蘇利南護照|蘇利南護照]]的{{link-en|蘇利南人口|Surinamese people|蘇利南公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將蘇利南歸類為'''灰色警示'''：提醒注意（2017年5月26日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-419-52f35-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將蘇利南歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=B3EFBE47CB5B528B |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|蘇利南經濟|Economy of Suriname}}}}
===貿易===
2018年，蘇利南是中華民國第137大貿易夥伴、第117大[[進口|進口]]夥伴、第129大[[出口贸易|出口]]夥伴。出口至蘇利南的金額為757萬5,279[[美元|美元]]，年增0.947%；自蘇利南進口的金額為668萬7,504美元，年增138.410%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）88萬7,775美元，年減81.108%。2017年，蘇利南是中華民國第143大貿易夥伴、第138大進口夥伴、第131大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
