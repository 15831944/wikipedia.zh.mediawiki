'''中華民國與模里西斯關係'''是指[[中華民國|中華民國]]與[[模里西斯共和國|模里西斯共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對模里西斯的相關事務由'''{{link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-176-94a33-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
{{see also|中華民國外交|{{link-en|模里西斯外交|Foreign relations of Mauritius}}}}
1971年10月25日，模里西斯在[[聯合國大會|聯合國大會]]上就「[[中國與聯合國#代表權問題|中國代表權]]」問題投票棄權，該國也是唯一棄權的[[聯合國會員國列表|非洲國家]]。

1984年9月，於首都[[路易港|路易港]]設立[[實質大使館|具大使館性質]]的'''中華民國駐模里西斯[[臺北經濟文化代表處|商務代表團]]'''；1988年3月，於首都[[台北市|台北]]設立同等性質的'''模里西斯外銷發展投資局駐華[[辦事處|辦事處]]'''，但因[[台商|台商]]投資有限、成效不彰，已於1992年底裁撤。<ref name="90年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2001/Chap3/3-1-3.htm |title = 《中華民國90年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>中華民國商務代表團目前也已關閉。

1997年4月，台北市長[[陳水扁|陳水扁]]參加在模里西斯舉行的{{link-zh-yue|城市與地方政府聯合會|世界城市同地方政府聯合組織}}第33屆世界大會。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-3.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|模里西斯簽證政策|{{link-en|模里西斯公民簽證要求|Visa requirements for Mauritian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]且在[[台灣|台灣]]有戶藉的[[中華民國國民|中華民國公民]]可於抵達[[西沃萨古尔·拉姆古兰爵士国际机场|模里西斯國際機場]]申請[[簽證#落地签证|落地簽證]]入境，停留最多60天，免簽證費。簽證相關事宜，可至模里西斯駐[[新加坡|新加坡]][[領事館|名譽領事館]]詢問。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-176-94a33-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|模里西斯護照|Mauritian passport}}的{{link-en|模里西斯人口|Demographics of Mauritius|模里西斯公民}}則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將模里西斯歸類為'''灰色警示'''：提醒注意（2017年6月3日）。<ref name="國外旅遊警示分級表">{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-369-bab3f-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

====事件====
2003年4月，台灣發生[[SARS事件|嚴重急性呼吸道症候群（SARS）疫情]]，模里西斯擔心會傳入該國，於是停止發放給台灣的[[漁船|漁船]]、[[船#分類|商船]]、[[航空|飛機]]入境許可。中華民國外交部與{{link-en|模里西斯政府|Politics of Mauritius}}交涉，指出模里西斯是台灣在[[非洲|非洲]]尤其是[[印度洋|印度洋]]一帶很大的[[漁場|漁業基地]]，有很多漁船在該國進行補給，若禁止入境，對台灣損害很大，並希望如果[[中華民國國民|台灣人民]]能提出健康證明，模里西斯政府能給予入境簽證。<ref>{{Cite web |url = http://www.epochtimes.com/b5/3/5/8/n309444.htm |author = [[中央通訊社|中央通訊社]]記者黃明興 |date = 2003年5月8日 |title = 模里西斯停發台灣旅客入境簽證 |publisher = [[大紀元時報|大紀元時報]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|模里西斯經濟|Economy of Mauritius}}}}
===貿易===
2017年，模里西斯是中華民國第91大貿易夥伴、第122大[[進口|進口]]夥伴、第72大[[出口贸易|出口]]夥伴，在[[非洲国家和地区列表|非洲經濟體]]排名第8、20、3。出口至模里西斯的金額為1億1,159萬7,985[[美元|美元]]，年減17.690%；自模里西斯進口的金額為565萬3,407美元，年減6.503%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1億0,594萬4,578美元，年減18.213%。※2016年，模里西斯是中華民國第77大貿易夥伴、第122大進口夥伴、第59大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
台灣[[老爺酒店集團|老爺酒店集團]]在該國建有五[[星級|星級]]的「模里西斯帝王酒店」（Sofitel Mauritius L′Impérial Resort & Spa）。<ref name="生活資訊">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-176-94a33-05-1.html |title = 生活資訊 |publisher = 外交部領事事務局}}</ref>

===漁業===
模里西斯是臺灣主要[[遠洋漁業|遠洋漁業]]基地之一，台籍[[漁船|漁船]]進入該國[[港口|港口]]維修、補給已有多年歷史，[[漁業|漁業]]事務成為兩國關係重要部分。<ref name="90年外交年鑑"/>[[行政院農業委員會漁業署|行政院農業委員會漁業署]]亦派駐模里西斯漁業專員。<ref name="國外旅遊警示分級表"/>
