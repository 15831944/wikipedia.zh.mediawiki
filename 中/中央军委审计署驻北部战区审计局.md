'''中央军委审计署驻北部战区审计局'''，位于[[辽宁省|辽宁省]][[沈阳市|沈阳市]]，是[[中央军委审计署|中央军委审计署]]下属局，负责[[中国人民解放军北部战区|中国人民解放军北部战区]]审计工作。

==沿革==
在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月组建[[中央军委审计署|中央军委审计署]]。中央军委审计署新设'''中央军委审计署驻北部战区审计局'''，驻[[中国人民解放军北部战区|中国人民解放军北部战区]]<ref>{{Cite web |url = http://zb.81.cn/content/2017-10/26/content_7800794.htm |title = 全军和武警部队官兵坚决拥护党的十九届一中全会决定 |author =  |date =2017-10-26  |publisher =军报记者  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://baijiahao.baidu.com/s?id=1596446033690004504&wfr=spider&for=pc |title = 军队人大代表吴春利赴审计一线宣讲会议精神侧记 |author =  |date =2018-03-31  |publisher =百家号  |language =  |accessdate =  |quote =  }}</ref>。
