'''中国人民解放军第一医院'''，又称'''三爱堂医院'''，位于[[甘肃省|甘肃省]][[兰州市|兰州市]][[城关区_(兰州市)|城关区]]静宁路72号，隶属[[西宁联勤保障中心|西宁联勤保障中心]]，为[[三级甲等医院|三级甲等医院]]。

==简介==
中国人民解放军第一医院的前身是[[中国工农红军|中国工农红军]]时期的中央直属干部休养连和[[抗日战争|抗日战争]]时期的军委直属医疗所，1939年在[[延安|延安]]扩建为'''[[中国人民解放军兰州总医院|八路军军医院]]'''，[[白求恩|白求恩]]曾在医院工作。同年白求恩殉职，1939年12月1日，八路军总部颁发命令，将八路军军医院更名为'''白求恩国际和平医院'''（后来设分院后改称'''白求恩国际和平医院总院'''）。1946年6月，[[第二次国共内战|全面内战]]爆发。1946年11月，白求恩国际和平医院总院奉命改编为'''陕甘宁晋绥联防军卫生部第二后方医院'''，撤出延安，同时将白求恩国际和平医院总院的名称转交给延安中央医院（后来演变为[[北京医院|北京医院]]、[[西安市儿童医院|西安市儿童医院]]、[[唐都医院|唐都医院]]等）。1947年7月，医院番号改为'''中国人民解放军西北野战军第二后方医院'''，归中国人民解放军西北野战军后勤部领导。1949年2月，番号改为'''中国人民解放军第一野战军第二野战医院'''。1949年8月26日，医院随同第一野战军前总机关进驻兰州。1950年7月，与已被西北军区接管的原西北军政长官公署下辖的“兰州中央医院”合并，改编成'''[[中国人民解放军兰州总医院|中国人民解放军西北军区第一陆军医院]]'''。1951年7月，西北军区第一陆军医院第二部分出，成立'''中国人民解放军西北军区第四陆军医院'''。1954年7月，更名为'''中国人民解放军第一医院'''，沿用至今<ref name=gsrb>{{Cite web |url = http://gsrb.gansudaily.com.cn/system/2009/10/20/011311832.shtml |title = 解放军第一医院庆祝建院70周年 |author =  |date = 2009-10-20 |publisher = 甘肃日报 |language =  |accessdate =  |quote =  }}</ref><ref name=xhw>{{Cite web|url =http://news.xinhuanet.com/mil/2008-01/14/content_7416010.htm|title =解放军第一医院|author =|date =2008-01-14|publisher =新华网|language =|accessdate =|quote =}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1955年，医院从兰州市木塔巷和金天观，迁入现址“三爱堂”。“三爱堂”原是[[张自忠|张自忠]]主政西北时的西北行政长官公署，兰州解放后曾为[[中国人民解放军第一野战军|中国人民解放军第一野战军]]司令部和[[彭德怀|彭德怀]]住地，“'''三爱堂医院'''”也由此得名<ref name=zhangxl/>。

1996年，医院被[[中国人民解放军总后勤部卫生部|中国人民解放军总后勤部卫生部]]评定为“[[三级甲等医院|三级甲等医院]]”<ref name=xhw/>。2000年代，先后参加援外医疗、抗击“[[非典|非典]]”、[[汶川大地震|汶川大地震]]抗震救灾、收治结石患儿等重大任务<ref name=gsrb/>。其中，2008年9月初，该院泌尿外科主任张伟致信《[[南方周末|南方周末]]》，揭露[[三鹿奶粉事件|三鹿奶粉导致婴儿患结石]]，后来他又和科室医生商量后，让患儿家属找《兰州晨报》曝光，最终促使[[三鹿奶粉事件|三鹿奶粉事件]]曝光，[[中华人民共和国卫生部|中华人民共和国卫生部]]部长[[陈竺|陈竺]]来该医院视察时说：“张伟是一名具有战略眼光，敢于承担责任的医生。”<ref name=jiaxm/>该医院也成为全国发现最早、救治最及时、抢救危重结石患儿最多的医院<ref name=zhangxl/>。截至2008年，医院设有高压氧治疗、肝胆外科、传染病防治等3个[[兰州军区|兰州军区]]专科中心<ref name=xhw/>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，医院由[[中国人民解放军兰州军区|中国人民解放军兰州军区]]联勤部转隶[[西宁联勤保障中心|西宁联勤保障中心]]<ref>{{Cite web |url = http://www.sohu.com/a/146687601_114731 |title = 西宁联勤保障中心科学统筹实现医疗资源利用最大化 |author =  |date = 2017-06-07 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。
