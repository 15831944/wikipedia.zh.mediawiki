'''中国人民解放军火箭军峨眉疗养院'''，位于[[四川省|四川省]][[峨眉山市|峨眉山市]]景区路3号，隶属[[中国人民解放军火箭军后勤部|中国人民解放军火箭军后勤部]]。

==简介==
峨眉疗养院的前身是陆军第四十医院（位于[[峨眉县|峨眉县]][[峨眉山|峨眉山]]下）。后来，陆军第四十医院改为'''中国人民解放军成都军区峨眉疗养院'''，隶属[[中国人民解放军成都军区|中国人民解放军成都军区]]；并且下设'''中国人民解放军成都军区峨眉疗养院都江堰分院'''<ref>成都军区峨眉疗养院简介，中国康复医学杂志1989年03期</ref><ref name=scrmzf>{{Cite web |url =http://www.sc.gov.cn/scgk1/tpzs/jyky/200801/t20080125_247536.shtml  |title =成都军区峨眉疗养院  |author =  |date = 2008-01-25 |publisher =  四川省人民政府|language =  |accessdate =  |quote =  }}</ref>。后来都江堰分院撤销。2004年5月26日，中国人民解放军空军都江堰疗养院（位于[[灌县|灌县]][[青城山|青城山]]旁）转隶成都军区联勤部，同年10月整编为'''中国人民解放军成都军区峨眉疗养院都江堰疗养区'''。2009年7月，成都军区峨眉疗养院都江堰疗养区转隶[[成都军区空军|成都军区空军]]后勤部，整编为[[中国人民解放军空军都江堰航空医学鉴定训练中心|中国人民解放军空军都江堰航空医学鉴定训练中心]]<ref name=scrmzf/><ref>何强、彭天国、董文，空军航空医学鉴定训练中心的发展与思考 ，全国航空航天医学学术会议暨云南省航空航天医学学术会议2010,27(3):180-181</ref><ref>{{Cite web |url = http://news.xinhuanet.com/mil/2011-12/25/c_111295752.htm |title = 一场从陆地打响的空战 |author =  |date = 2011-12-25 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

峨眉疗养院曾获[[中国人民解放军|中国人民解放军]]全军及成都军区“白求恩杯”优质服务先进单位及优胜单位，被[[中国人民解放军总后勤部卫生部|中国人民解放军总后勤部卫生部]]表彰为“为部队服务先进疗养院”。峨眉疗养院主要担负军队疗养员的疗养保健、健康鉴定、疾病矫治任务。2006年，峨眉疗养院招聘首批文职人员。2008年[[汶川大地震|汶川大地震]]期间，峨眉疗养院文职人员为灾后防疫及保证灾区群众健康发挥了作用<ref>{{Cite web |url =  http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536921.htm|title = 成都军区峨眉疗养院 |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。作为中国人民解放军全军高级干部、功臣模范、科技专家、飞行员等特勤人员的疗养康复及体检鉴定基地，峨眉疗养院曾接待多位中央军委和总部首长、[[杨利伟|杨利伟]]等全体[[中国航天员大队|中国航天员]]疗养<ref name=scrmzf/>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，峨眉疗养院的上级单位中国人民解放军成都军区撤销，峨眉疗养院转隶[[中国人民解放军火箭军后勤部|中国人民解放军火箭军后勤部]]，更名为'''中国人民解放军火箭军峨眉疗养院'''<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1689524  |title = 陆军驻新疆、西藏边远艰苦地区部队试点团职干部带配偶疗养 |author =  |date = 2017-05-19 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://photo.chinamil.com.cn/jypk/2016-11/22/content_7371239.htm |title = 火箭军峨眉疗养院迎来特殊“客人” |author =  |date =2016-11-22  |publisher =  中国军网|language =  |accessdate =  |quote =  }}</ref>。
