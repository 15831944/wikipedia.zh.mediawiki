'''中国人民武装警察部队警衔'''介绍[[中国人民武装警察部队|中国人民武装警察部队]]的[[警衔|警衔]]体系。1982年武警部队成立后，于1988年设立了武警警衔制度，之后又进行了一些调整<ref name="警衔丙">{{cite web |author = 徐平 |title = 我军建军九十年军衔制度沿革 |url = http://www.mod.gov.cn/education/2017-08/07/content_4788037.htm |website=中华人民共和国国防部网站 |accessdate=2017-08-20 |location=北京 |language=zh-cn }}</ref>。作为实行兵役制的[[国家宪兵|国家宪兵]]，中国人民武装警察部队的警衔与[[中国人民解放军军衔|中国人民解放军军衔]]基本相同，仅在用词和配色上有所区分<ref name="警衔丁">{{cite web |authors = 中华人民共和国国务院、中华人民共和国中央军事委员会 |title=中国人民武装警察部队实行警官

== 警官警衔 ==
现行的[[武警|武警]][[警衔|警衔]]于1988年实行，初设为6等17级。在首次授衔中，截至1988年年底，解放军全军各级军官58.7万余人，文职干部14.7万余人都获得了军衔。其中，[[中央军事委员会|中央军事委员会]]共批准授予上将17人，中将146人，少将1279人；校官17.8万余人；尉官48.8万人。同时，武警部队也单独举行了授衔仪式。<ref name="警衔丙"/>

1993年将[[士兵|士兵]][[警衔|警衔]]由3等6级增设为3等9级，1994年取消[[一级上将|一级上将]]衔，1999年将士兵警衔由3等9级精简为2等8级，2009年又改为2等9级，现行警衔分为5等19级：<ref name="警衔甲"/><ref name="警衔乙">{{cite web|author1=中央军事委员会|title=中国人民解放军军官军衔条例|url=http://www.mod.gov.cn/policy/2009-07/13/content_3098881.htm|website=中华人民共和国国防部网站|accessdate=2017-08-20|location=北京|language=zh-cn}}</ref>

[[中国人民武装警察部队|中国人民武装警察部队]]的警衔与[[中国人民解放军军衔|中国人民解放军军衔]]类似。1999年12月1日起，根据第二次修改的《中国人民解放军士兵服役条例》，武警设士官（志愿兵役制士兵）和兵（义务兵役制士兵）2等8级，即六至一级士官、上等兵和列兵警衔<ref>{{cite web|author1=徐平|title=武警部队的警衔制度|url=http://www.81.cn/jfdg/2011-07/13/content_5494521.htm|website=中国军网|accessdate=2017-02-08|date=2011-07-13}}</ref>。
