'''中国人民解放军国防科技大学国际关系学院'''，位于[[江苏省|江苏省]][[南京市|南京市]]，隶属[[中国人民解放军国防科技大学|中国人民解放军国防科技大学]]。

== 沿革 ==
;中国人民解放军外国语学院
{{参见|中国人民解放军战略支援部队信息工程大学外国语学院}}
* 1951年5月，组建'''中央军委技术部干部学校'''，校址在[[北京|北京]]<ref name=xhw/>。
* 1953年3月，中央军委技术部干部学校（部分）组建'''中央军委外国语文学校'''，将学员集中于[[张家口|张家口]]。张中如任校长。中央军委外国语文学校奠定了后来中国人民解放军国际关系学院的基础<ref name=xhw/>。
* 1957年，张家口的中央军委外国语文学校与北京的[[中国人民解放军外国语学院|军委技术部干部学校]]合并为'''中国人民解放军外国语文专科学校'''，校址在北京。
* 1960年1月3日，升格'''中国人民解放军外国语文学院'''。
* 1960年4月，校址迁至张家口。
* 1961年10月19日，更名'''中国人民解放军张家口外国语文学院'''。
* 1962年12月10日，更名'''中国人民解放军外国语学院'''。
* 1964年1月1日起，划归[[总参三部|总参三部]]。

;中国人民解放军南京外国语文学院
* 1952年，'''中国人民解放军华东军区直属文化速成中学'''成立。校址设在南京山西路（该地址后为南京军区军人俱乐部）<ref name=nanjing/>。
* 1954年，校址迁至南京西郊上新河，改建'''中国人民解放军第十七文化速成中学'''，隶属华东军区<ref name=nanjing/>。
* 1954年，'''中国人民解放军第二十文化速成中学'''由[[扬州|扬州]]迁至南京西郊板桥镇原华东军区第三公安干部学校旧址<ref name=nanjing/>。
* 1959年12月，第十七、第二十文化速成中学合并为'''中国人民解放军第十七文化速成中学'''，以板桥镇为校址<ref name=nanjing/>。
* 1960年9月，改建'''中国人民解放军南京军区第一文化学校'''，负责南京军区营、团、师职干部文化教学<ref name=nanjing/>。
* 1961年12月，改建升格为'''中国人民解放军南京外国语文学院'''<ref name=nanjing/>。
* 1962年7月，撤编，移交给[[中国人民解放军外国语学院|中国人民解放军外国语学院]]<ref name=nanjing/>。

;中国人民解放军国际关系学院
* 1962年7月，[[中国人民解放军外国语学院|中国人民解放军张家口外国语文学院]]第一系（1962年12月更名中国人民解放军外国语学院第一系）从张家口迁至南京板桥镇<ref name=nanjing>{{cite news |title = 第三节 南京解放后军事院校 |url = http://app.nanjing.gov.cn/njnj/sz8_12/8/html/08Noname374.html |website = 中国南京政府网站 |accessdate = 2016-12-01 }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>（即原中国人民解放军南京外国语文学院院址）。
* 1964年，中国人民解放军外国语学院第一系扩建为'''中国人民解放军外国语文学院'''<ref name=nanjing/>。隶属[[总参二部|总参二部]]。
* 1969年，改建'''中国人民解放军第二外国语学校'''<ref name=nanjing/>。
* 1976年，升格'''中国人民解放军外国语学院'''<ref name=nanjing/>。
* 1978年，改建'''中国人民解放军南京外国语学院'''<ref name=nanjing/>。
* 1986年，改建'''中国人民解放军国际关系学院'''<ref name=nanjing/><ref>{{cite news |title = 解放军国际关系学院 |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536702.htm |publisher = 中华人民共和国国防部 |date = 2014-09-12}}</ref>。隶属[[中国人民解放军总参谋部|中国人民解放军总参谋部]]，[[正军级|正军级]]。
* 2016年1月，在[[深化国防和军队改革|深化国防和军队改革]]中，原上级单位[[中国人民解放军总参谋部|中国人民解放军总参谋部]]撤销，转隶新组建的[[中央军事委员会训练管理部|中央军事委员会训练管理部]]<ref>{{cite news |title = 军队院校改革在即 解放军国际关系学院已转隶 |url = http://mil.sohu.com/20160118/n434836882.shtml |publisher=搜狐 |date=2016-01-18}}</ref><ref>{{cite news|title=爱心义诊进军营 温馨幸福满中秋 |url = http://www.91kq.com/news/201609/20/7336.html|publisher=中国口腔网|date=2016-09-20}}</ref>。

;中国人民解放军国防科技大学国际关系学院
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，[[中国人民解放军国防科学技术大学|中国人民解放军国防科学技术大学]]、[[中国人民解放军国际关系学院|中国人民解放军国际关系学院]]、[[中国人民解放军国防信息学院|中国人民解放军国防信息学院]]、[[中国人民解放军西安通信学院|中国人民解放军西安通信学院]]、[[中国人民解放军电子工程学院|中国人民解放军电子工程学院]]、[[中国人民解放军理工大学气象海洋学院|中国人民解放军理工大学气象海洋学院]]为基础组建[[中国人民解放军国防科技大学|中国人民解放军国防科技大学]]。校本部设在[[长沙|长沙]]，内设学院位于长沙、[[南京|南京]]、[[武汉|武汉]]、[[合肥|合肥]]等地<ref name=xin>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1699945  |title = 新国防科大今年以国防科大、国关学院等6所院校为基础重建 |author =  |date =  2017-06-03 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。其中，中国人民解放军国际关系学院改为'''中国人民解放军国防科技大学国际关系学院'''<ref>{{Cite web |url =http://www.tsinghua.edu.cn/publish/ir/3616/2017/20170602104703386875522/20170602104703386875522_.html  |title = 2017年政治学与国际关系学术共同体会议简讯（7） |author =  |date = 2017-06-02 |publisher = 清华大学国际关系学系 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.sohu.com/a/162051458_507486 |title = 从文学体验社会，为社会培育人才——“文学与社会”研修班暨外国文学研讨会成功举办 |author =  |date = 2017-08-03 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.njjlhw.cn/portal_jsps/site/hwzx/default/content.jsp?cntid=9500&chnid=4069 |title = 国防科技大学国际关系学院领导到我校访问交流 |author =  |date =2017-07-10   |publisher =南京市金陵汇文学校（初中部）  |language =  |accessdate =  |quote =  }}</ref>。
