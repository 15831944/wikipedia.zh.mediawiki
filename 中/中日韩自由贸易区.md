'''中日韩自由贸易区'''（{{lang-ko|'''한중일 자유무역지대'''}}；{{lang-ja|'''日中韓自由貿易協定（にっちゅうかんじゆうぼうえききょうてい）'''}}）是在[[中日韩领导人会议|中日韩领导人会议]]上提出来的构想，成员包括[[中华人民共和国|中华人民共和国]]、[[日本国|日本国]]、[[大韩民国|大韩民国]]。目的是为逐步消除三国之间的关税壁垒，加强[[贸易|贸易]]、[[投资|投资]]、[[金融|金融]]、[[货币|货币]]交流。<ref>[http://www.apdnews.com/news/28656.html 中日韩自贸区将促进东亚经济一体化] {{webarchive|url=https://web.archive.org/web/20130713093025/http://www.apdnews.com/news/28656.html |date=2013-07-13 }}，亚太日报，2013年8月1日</ref><ref>{{cite web |url= http://finance.ifeng.com/video/20100531/2257042.shtml|title= 中日韩三国将加快缔结自由贸易区|publisher=凤凰网 |accessdate=2010-05-31 |work= |language =zh-hans}}</ref><ref>{{Cite web|url=http://zh.cn.nikkei.com/columnviewpoint/column/10611-20140815.html|title=日本為什麼輸給德國  日經中文網|accessdate=2017-03-14|last=程巍|work=zh.cn.nikkei.com|language=ja}}</ref>

== 历史 ==
2002年，中国总理[[朱镕基|朱镕基]]、日本首相[[小泉纯一郎|小泉纯一郎]]、韩国总统[[金大中|金大中]]在[[中日韩领导人会议|中日韩领导人会议]]上提出了建立包括[[中华人民共和国|中华人民共和国]]、[[日本国|日本国]]、[[大韩民国|大韩民国]]的三国合作的[[自由贸易区|自由贸易区]]的构想，并开始在三国之间的民间学者之间展开研究和交流，三国承诺七年之内完成机制的构建和规划。

2010年11月25日，在[[中国|中国]][[山东省|山东省]][[青岛市|青岛市]]东亚经济交流机构第四届大会上，三国之间的十个会员城市签订了《“环黄海行动”备忘录》，在中国的[[天津市|天津市]]、[[青岛市|青岛市]]、[[大连市|大连市]]、[[烟台市|烟台市]]，[[日本|日本]]的[[北九州市|北九州市]]、[[下关市|下关市]]、[[福冈市|福冈市]]，[[韩国|韩国]]的[[釜山市|釜山市]]、[[仁川市|仁川市]]和[[蔚山市|蔚山市]]这10个城市开始中日韩自由贸易区展开先行先试。<ref>{{cite web |url= http://news.xinhuanet.com/fortune/2010-11/25/c_12817592.htm|title=中日韩十城市打造三国自由贸易区试验田 |publisher=新华网 |accessdate=2010-10-25 |work= |language =zh-hans}}</ref>

2011年，[[中日韩领导人会议|中日韩领导人会议]]上，中国总理[[温家宝|温家宝]]、日本首相[[野田佳彥|野田佳彥]]、韩国总统[[李明博|李明博]]三国领导人宣布在2011年年底完成“中日韩自由贸易区”的构建和规划设想，并且将在2012年启动谈判。<ref name=" xlws" >{{cite web |url=http://news.sina.com.cn/c/2011-11-20/143123495353.shtml |title=中日韩明年启动自贸区谈判 |publisher=新浪网 |accessdate=2011-11-20 |work= |language =zh-hans}}</ref>

2012年5月13日，三国在北京签署《[[中华人民共和国政府|中华人民共和国政府]]、[[日本国政府|日本国政府]]及[[大韩民国政府|大韩民国政府]]关于促进、便利和保护投资的协定》，这是三国签署的首个投资协议。此外三国领导人一致同意年内启动中日韩自由贸易区谈判，[[中華人民共和國|中華人民共和國]]倡议在[[中國大陸|中國大陸]][[山东|山东]]建设'''中日韩自由贸易区示范区'''，并积极考虑建立三国产业合作基地。<ref>{{Cite web|url=https://money.udn.com/money/story/5603/2218043|title=中日韓FTA談判 今登場|accessdate=2017-03-14|author=|date=|last=聯合財經網|work=聯合財經網|publisher=|deadurl=yes|archiveurl=https://web.archive.org/web/20170314164358/https://money.udn.com/money/story/5603/2218043|archivedate=2017年3月14日|df=}}</ref><ref>{{Cite web|url=http://zh.cn.nikkei.com/politicsaeconomy/politicsasociety/21159-20160825.html|title=中日韓飄搖的「三角關係」  日經中文網|accessdate=2017-03-14|last=徵|first=王|work=zh.cn.nikkei.com|language=ja}}</ref>2012年11月20日，三國經貿部長進行會晤舉行，並宣布正式啟動自貿區談判。2013年3月，在韓國[[首爾|首爾]]舉行第一輪談判；同年7月，在[[上海|上海]]舉行第二輪談判會議<ref>{{Cite web|url=https://www.chinatimes.com/newspapers/20181207000149-260309|title=推展合作 中日韓FTA重啟談判|accessdate=2019-02-28|author=林永富|date=2018-12-07|work=中國時報|publisher=|deadurl=yes|archiveurl=|archivedate=2017年3月14日|df=}}</ref>。

之後，發生了[[釣魚台國有化事件|釣魚台國有化事件]]、安倍晉三[[靖國神社爭議|參拜靖國神社]]、[[韓國部署薩德反導彈系統事件|韓國部署薩德事件]]，三國關係惡化，談判進度不前。

2019年3月，中国国务院总理[[李克强|李克强]]在[[国务院总理记者会|中外记者会]]上再次提出希望推动建设中日韩自由贸易区。<ref>{{cite web |url = https://news.rthk.hk/rthk/ch/component/k2/1447886-20190315.htm |title= 李克強冀推動建設中日韓自貿區 |work=香港電台 |date=2019-03-15}}</ref>
== 社会反映 ==
=== 中华人民共和国 ===
原[[中华人民共和国国务院副总理|国务院副总理]][[曾培炎|曾培炎]]曾力推加速中日韩自由贸易区的建设<ref>{{cite web |url = http://news.163.com/11/0426/05/72HSVU7T00014AED.html |title=应加快建立中日韩自由贸易区 |publisher=网易 |accessdate= 2011-04-26|work= |language =zh-hans}}</ref><ref name="yqs">{{cite web |url = http://info.shippingchina.com/bluenews/index/detail/id/80831.html |title=中日韩三国加速建设自由贸易区 |publisher=中国国际海运网 |accessdate= 2011-05-27|work= |language =zh-hans}}</ref>。

[[中华人民共和国国家发展和改革委员会|国家发改委]]对外经济研究所国际经济合作室主任[[张建平|张建平]]说：“伴随区内[[关税|关税]]和贸易限制措施的降低和取消，三国之间的商品流通更为顺畅和便利，同时由于中韩日分别位于产业链的低端、中端和高端，经济具有互补性，因此，建立自贸区后，三国相关经济指标将有明显提升，贸易关系将更为密切。”<ref name="yqs" />

[[中华人民共和国商务部|商务部]]国际贸易经济合作研究院国际市场研究部副主任[[白明|白明]]说：“一方面会为中国出口企业创收，出口厂商的成本会降低，市场利益会增加，企业动力增加。另一方面能刺激消费增长，消费者可以在自贸区内买到价格更低的商品，消费热情更大。”<ref name="yqs" />

[[中国现代国际关系研究院|中国现代国际关系研究院]]研究员[[刘军红|刘军红]]表示：“日本具备[[政策|政策]]、[[法律|法律]]、[[制度|制度]]，乃至[[技术|技术]]、[[资金|资金]]优势，想发挥主导作用，最终目的是为建立‘[[东亚共同体|东亚共同体]]’做准备；同时，由于中国市场不完善，[[民族企业|民族企业]]、[[工业|工业]]、[[农业|农业]]、[[高科技|高科技]]领域均落后于日本，而中国人口众多，签订中日韩自由贸易区协议，明显中国产业升级不利，甚至还有可能阻止和断送相关民族产业。”<ref>{{cite web |url = http://finance.sina.com.cn/roll/20100623/11348162574.shtml |title= 中日韩自由贸易区研究：战略方向现分歧|publisher=新浪网 |accessdate=2010-06-23 |work= |language =zh-hans}}</ref>

经济学家[[王蔚|王蔚]]指出：“日本认为自己经济实力雄厚，市场经济发达，[[科技|科技]]和管理水平高，理应在三国自由贸易区中起到主导地位。”<ref>{{cite web |url= http://business.sohu.com/77/14/article210391477.shtml|title=中国提出建中日韩自由贸易区 日本谋求主导地位 |publisher=搜狐网 |accessdate=2003-06-24 |work= |language =zh-hans}}</ref>

[[清华大学当代国际关系研究院|清华大学当代国际关系研究院]]教授[[刘江永|刘江永]]：“即便自贸区真的启动，也有一个逐步过渡的过程，很难一步到位。可能需要历经10年到20年，才能形成涵盖低关税、服务业、人员自由流动、知识产权等内容的高级自贸区。”<ref name="yqs" />

[[复旦大学经济学院|复旦大学经济学院]]副院长[[孙立坚|孙立坚]]说：“三国推进自由贸易协定之路还是比较艰难的。一旦中国向日本和南韩企业开放[[零关税|零关税]]，中国的企业是会通过竞争做大做强，还是被过大的竞争压力所挤兑淘汰，目前还应持谨慎态度。由于中国是一个非常庞大的市场，因此在推进自由贸易协定的过程中，相比日本南韩的积极主动，中国方面的态度比较保守，更希望采取逐步放开的方式，这样对中国企业的冲击也会相对弱一些。”<ref name="yqs" />

=== 日本国 ===
日本《[[读卖新闻|读卖新闻]]》称：“日本如果能在与[[中国|中国]]和[[东盟|东盟]]的自由贸易达成协议，对日本经济的好处要远远大于[[泛太平洋戰略經濟夥伴關係協議|TPP]]。”，“如果与中国在内的东盟+6框架内实现自由贸易，日本GDP将因此增加1.1%，而如果是TPP框架内的自由贸易，[[日本人|日本人]]的获益只有GDP的0.54%。”<ref name=" xlws" />

===大韓民國===
==影響==
===[[中華民國（台湾）|中華民國（台湾）]]===
一些观点认为对于台湾而言，由於韓國與臺湾在許多產業具競爭關係，而[[日本|日本]]及中國大陸分別是臺湾最大進口國與最大出口地區，中日韓的自由貿易區一旦建立，對於臺湾在區域整合及國際產業經貿競爭上會有相當衝擊，對中國大陸部分，臺湾以加強加快[[ECFA|ECFA]]後續談判及合作優惠範圍項目做因應，對日則以促進相關投資協定來提升雙邊關係。
