'''中華民國與拉脫維亞關係'''是指[[中華民國|中華民國]]與[[拉脫維亞共和國|拉脫維亞共和國]]之間的關係。兩國於1936－1940年有[[特命全權公使|公使級]]的[[外交#外交上的承認|外交關係]]，但因[[蘇聯佔領波羅的海國家|蘇聯併吞拉脫維亞]]而中止。自[[蘇聯解體|1991年獨立]]以來，於1992－1994年，兩國建立[[領事|總領事]]關係，是[[臺灣戰後時期|中華民國政府遷台]]後，唯二建立外交關係的[[歐洲國家和地區列表|歐洲新獨立國家]]（另一為[[馬其頓共和國|馬其頓共和國]]），以及唯一的[[蘇聯加盟共和國|前蘇聯國家]]。與拉脫維亞終止關係後，中華民國政府於[[首都|首都]][[里加|里加]]設立[[實質大使館|具大使館性質]]的代表機構。<ref name="駐外館處">{{Cite web |url = http://www.mofa.gov.tw/OverseasOffice_Content.aspx?n=6357834932B83C83&sms=5A961ED2E4BA25E8&s=E611B38FE00C641B&os=D73627CFF5C8BB60 |title = 駐外館處 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，[[拉脫維亞|拉脫維亞]]當時屬於[[蘇聯|蘇聯]]的[[加盟共和國|加盟共和國]]（[[拉脫維亞蘇維埃社會主義共和國|拉脫維亞蘇維埃社會主義共和國]]），由蘇聯作為[[主權國家|主權國家]]參與投票。]]
{{see also|中華民國外交|{{link-en|拉脫維亞外交|Foreign relations of Latvia}}}}
1936年6月25日，拉脫維亞與中華民國建立[[特命全權公使|公使級]]的[[外交#外交上的承認|外交關係]]。

1940年8月5日，因[[蘇聯佔領波羅的海國家|蘇聯吞併拉脫維亞]]而中止外交關係。

1991年8月21日，[[蘇聯解體|拉脫維亞自蘇聯獨立]]。11月，拉脫維亞外交部長{{link-en|尤肯斯|Jānis Jurkāns}}與中華民國簽署《互設[[實質大使館|貿易代表團]]協定》。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-4.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1992年1月29日，兩國發表《共同聲明》，建立[[領事|總領事級]]的外交關係。於首都設立'''中華民國駐[[里加|里加]][[領事館|總領事館]]'''。<ref name="86年外交年鑑"/><ref name="駐館"/>2月，中華人民共和國大使館關閉並撤離所有駐地人員，但沒有與拉脫維亞斷交。

1993年6月，{{link-en|1993年拉脫維亞國會選舉|Latvian parliamentary election, 1993|拉脫維亞國會選舉}}後，總理[[伊瓦尔斯·戈德马尼斯|戈德馬尼斯]]辭職，兩國關係呈現不穩。<ref name="86年外交年鑑"/>

1994年7月，拉脫維亞與[[中華人民共和國|中華人民共和國]]簽署《關係正常化聯合公報》。28日，終止與中華民國的總領事級外交關係。<ref name="86年外交年鑑"/><ref name="駐館">{{cite web|url=http://www.roc-taiwan.org/lv/post/69.html|title=駐館與駐在國關係|publisher=駐拉脫維亞代表處|date=2013-02-06|accessdate=2016-09-14}}</ref>

1994、1995年，拉脫維亞連續兩屆在[[聯合國大會|聯合國大會]]的總辯論中，提出「會籍普遍化原則」，間接呼應[[中國與聯合國#台灣人民代表權問題及台灣入聯運動|中華民國參與聯合國]]。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=4F8B8E40F8AEE541 |author = 公眾外交協調會 |date = 1994年10月14日 |title = 聯合國大會第四十九屆常會總辯論自九月二十六日至十月十二日召開 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=76ADD08F602C49F8 |author = 公眾外交協調會 |date = 1995年10月13日 |title = 聯合國大會第五十屆常會總辯論自九月二十五日至十月十一日召開期間 |publisher = 中華民國外交部}}</ref>

{{see also|臺灣經濟|拉脫維亞經濟}}
===貿易===
2008年10月，於[[台北市|台北]]召開「第1屆臺拉經濟合作會議」。第2－4屆會議在[[里加|里加]]舉行。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

2017年，拉脫維亞是中華民國第88大貿易夥伴、第117大[[進口|進口]]夥伴、第70大[[出口贸易|出口]]夥伴。出口至拉脫維亞的金額為1億1,202萬2,867[[美元|美元]]，年增4.263%；自拉脫維亞進口的金額為854萬8,064美元，年增9.089%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1億0,347萬4,803美元，年增3.883%。※2016年，拉脫維亞是中華民國第84大貿易夥伴、第116大進口夥伴、第70大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

==交流==
===學術===
{{see also|臺灣教育|{{link-en|拉脫維亞教育|Education in Latvia}}}}
[[中華民國教育部|中華民國教育部]]於1991年7月設立「[[獨立國協|獨立國協]]及東歐地區國家來台留學[[獎學金|獎學金]]」，其中包括拉脫維亞。<ref name="93年外交年鑑"/>該獎學金後來改為[[中華民國外交部|中華民國外交部]]的「臺灣獎學金」、教育部的「[[中華民國國語|華語文]]獎學金」，提供拉脫維亞學生申請赴台攻讀[[學位|學位]]及學習華語文；另有外交部提供的「臺灣獎助金」，提供拉脫維亞學者赴台研究。[[國立臺灣大學|國立臺灣大學]]、[[國立臺北科技大學|國立臺北科技大學]]、[[國立中山大學|國立中山大學]]與[[拉脫維亞大學|拉脫維亞大學]]及[[里加工業大學|里加工業大學]]簽有[[交換學生|交換學生]]計畫。<ref name="105年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

[[台北市政府|台北市政府]]自2004年起，每年提供獎學金1名給予[[臺北市友好城市或姐妹城市列表|姊妹市]]里加的學生赴台學習中文，「[[台北市|台北]]－[[里加|里加]]青年大使」活動亦由兩市政府各推派1名青年大使進行互訪。<ref name="93年外交年鑑"/>

財團法人國際合作發展基金會（國合會）等機構每年均提供拉脫維亞政府部門及民間機構人士赴台參加各項專技研習班。<ref name="93年外交年鑑"/>
