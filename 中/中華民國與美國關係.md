'''中華民國與美國關係'''是指[[中華民國|中華民國]]與[[美利堅合眾國|美利堅合眾國]]之間的雙邊關係。1913－1979年，兩國有官方[[外交#外交上的承認|外交關係]]，後因美國與[[中華人民共和國|中華人民共和國]]建交，而中斷與中華民國的外交關係。雙方斷交後，皆於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。由於美國對於中華民國在[[中華民國政治|政治]]、[[中華民國經濟|經濟]]及[[中華民國軍事|軍事]]上的高度重要性，被中華民國政府視作[[中華民國外交|外交]]工作的重點。

==代表機構==
===美國在台協會===
[[File:20170217_(IMG_0572)_New_compound_of_AIT_Taipei_Office_美國在台協會台北辦事處新館.jpg|thumb]]台北辦事處]]
1979年1月1日，[[美國|美國]]和[[中華人民共和國|中華人民共和國]]建交以後，與[[中華民國|中華民國]]之間即不存在正式的外交關係。但美國透過《[[台灣關係法|台灣關係法]]》、設立'''[[美國在台協會|美國在台協會]]'''（American Institute in Taiwan, AIT）與中華民國政府保持非官方的關係。

*[[美國在台協會#組織|美國在台協會主席]]：[[莫健_(美國外交官)|莫健]]。
*[[台北市|台北]]辦事處處長：[[酈英傑|酈英傑]]。另在[[高雄市|高雄]]設有[[美國在臺協會高雄分處|分處]]，<ref name="駐華外國機構">{{Cite web |url = http://www.mofa.gov.tw/OfficesInROC_Content.aspx?n=8CEB2B5F5436B997&sms=8EBFADC1592C7BFE&s=83376F561B7165E6&os=594FC080B4D63D73 |title = 駐華外國機構 |publisher = 中華民國外交部}}</ref>兩地也設有[[美國在台協會商務組|商務組]]。

'''美國在台協會台北辦事處'''位於[[信義路_(台北市)|信義路]]三段的土地是向[[臺北市政府教育局|臺北市政府教育局]]租用，由於[[臺北市議會|臺北市議會]]決議將該地收回興建國中，必須另覓新址。經過多次討論會勘後，在2000年12月選定[[內湖區|內湖區]][[金湖路_(台北市)|金湖路]]100號為新址地點，共約6.5[[公頃|公頃]]，自2005年1月起租期99年。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=9E57026FB421F554&sms=28A3E17C7AEEE92F&s=2D2C8C5C1010778B |author = 公眾外交協調會 |date = 2004年12月22日 |title = 美國在台協會台北辦事處將遷至內湖新址 |publisher = 中華民國外交部}}</ref>2018年6月12日落成啟用，新館前方有[[美國國旗|美國國旗]]、大門則有[[美國國徽|美國國徽]]。<ref>{{Cite web |url = https://udn.com/news/story/6656/3008892 |author = 洪哲政 |date = 2018年3月2日 |title = AIT證實：今年夏天啟用金湖路新館 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref><ref>{{Cite web |url = http://news.ltn.com.tw/news/focus/paper/1208532 |author = 呂伊萱 |date = 2018年6月13日 |title = AIT新館啟用「台美關係 推升到新高度」|publisher = [[自由時報|自由時報]]}}</ref>

中華民國前总统[[陈水扁|陈水扁]]曾这样表述台美關係，其在任期间为「[[美军|美军]]政府[[仲介|代理人]]」，听命于[[美國在臺協會|美國在臺協會]]（以下簡稱'''AIT'''）<ref>{{cite news| language =zh-tw| author =[[林濁水|林濁水]]| url =http://www.libertytimes.com.tw/2009/new/oct/19/today-o6.htm| title =論陳水扁控告美國政府| publisher =[[自由電子報|自由電子報]]| date =2009-10-19| accessdate =2011-06-18| deadurl =yes| archiveurl =https://web.archive.org/web/20091225161531/http://www.libertytimes.com.tw/2009/new/oct/19/today-o6.htm| archivedate =2009-12-25}}</ref><ref>{{cite news| language =zh-tw| author =曹郁芬| url =http://www.libertytimes.com.tw/2009/new/oct/11/today-p3-3.htm| title =向美請願 扁沒告歐巴馬| publisher =自由電子報| date =2009-10-11| accessdate =2011-06-18| deadurl =yes| archiveurl =https://web.archive.org/web/20091226020429/http://www.libertytimes.com.tw/2009/new/oct/11/today-p3-3.htm| archivedate =2009-12-26}}</ref>，美方对此予以否认。在[[維基解密|維基解密]]釋出AIT公文後，AIT被稱為「藍綠政治人物[[告解室|告解室]]」，因為能與AIT聊天、交換意見的台灣政治人物對AIT都頗為坦白；中華民國許多政府官員也習於直接跳過中華民國政府體系、向AIT報告重要資訊。<ref>{{cite news |language =zh-tw | author =王鼎鈞 | url =http://www.nownews.com/2011/09/08/301-2741085.htm| title =維基解密爆料時機曝光 部落客諷藍綠AIT大告解室打烊|publisher=[[NOWnews|NOWnews]] |date=2011-09-08 |accessdate=2011-09-08 }}</ref>

[[File:Taipei_Economic_and_Cultural_Representative_Office_in_the_United_States_from_VOA_(1).jpg|thumb]]]]

*[[北美事務協調委員會#歷任主委|北美事務協調委員會主任委員]]：[[林良蓉|林良蓉]]。

===駐處維安===
2018年7月29日，台灣《[[自由時報|自由時報]]》報導，美國透過管道向中華民國國安高層告知，美國在台協會台北辦事處新館於9月正式運作後，將派[[美國海軍陸戰隊|海軍陸戰隊]]進駐，其隊員隸屬於[[海軍陸戰隊使館警衛隊|海軍陸戰隊使館警衛隊]]，稱為「海軍陸戰隊衛兵」。這是兩國自1979年斷交以來，美國首度比照[[美國駐外機構列表|海外使領館]]，也在台灣設立「陸戰隊之家」（Marine House）。基於對等立場，中華民國軍方也爭取恢復[[中華民國憲兵|憲兵]]進駐駐美代表處及[[雙橡園|雙橡園]]，以彰顯兩國軍事合作交流進入新的里程碑。<ref>{{Cite web |url = http://www.chinatimes.com/realtimenews/20180729000852-260407 |author = 高德順 |date = 2018年7月29日 |title = 美陸戰隊 傳9月進駐AIT內湖新館 |publisher = [[中時電子報|中時電子報]]}}</ref>但[[有線電視新聞網|有線電視新聞網]]（CNN）在9月13日的報導指出，[[美國國防部|美國國防部]]已拒絕國務院的請求，原因是資源有限，且國務院未事先告知國防部在台北辦事處新館完工後，需要進駐陸戰隊，才導致請求被拒絕，並非怕激怒中國政府。<ref>{{Cite web |url = https://tw.appledaily.com/new/realtime/20180914/1429966/ |author = 鄭鴻達、曾盈瑜 |date = 2018年9月14日 |title = 美國防部拒派陸戰隊進駐AIT 總統府：不評論 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref>2019年4月3日，美國在台協會發言人孟雨荷表示，新館的維安如同舊館，由少數美方人員搭配人數較多的當地聘僱人員，並與地方當局合作共同負責。也首度證實自2005年起，派駐舊館的美方人員就有[[美軍|現役軍人]]，包括[[美國陸軍|陸軍]]、[[美國海軍|海軍]]、[[美國空軍|空軍]]與陸戰隊。<ref>{{Cite web |url = https://udn.com/news/story/10930/3735910 |author = 鄭媁 |date = 2019年4月3日 |title = AIT新館5月6日啟用 首度證實有陸戰隊軍官進駐 |publisher = 聯合新聞網}}</ref>

持有載明[[中華民國國民身份證|身份證]]字號的新版普通晶片[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以「[[免簽證計劃|免簽證計劃]]」的方式入境美國，停留最多90天。事先上網申請「[[旅遊許可電子系統|旅行授權電子系統]]（ESTA）」取得授權許可，且無其他特殊限制而無法適用者，即可免除預先申請[[B签证_(美国)|B1/B2簽證]]而直接前往美國。但免簽證不代表能自動獲准入境美國，仍需符合美國的入境規定與[[美國海關及邊境保衛局|美國海關及邊境保衛局]]官員核可。經美國轉機，可至[[美國在台協會|美國在台協會]]確認簽證申請及注意事項。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-100-57cb3-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=7DEC7150E6BAD606 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

所有參與免簽證計劃的38個國家都有較高的[[人類發展指數|人類發展指數]]而且大多被認為是[[已開發國家|已開發國家]]，中華民國更是此計劃唯一的{{link-en|駐華盛頓外交機構列表|List of diplomatic missions in Washington, D.C.|美國非邦交國}}。2017年11月1日，在加入免簽證計畫五周年時，另正式加入美國的「[[全球入境|全球入境計畫]]（Global Entry）」，只需事先申請、完成安全查核再通過面試後，入出境美國就可以使用專用通道，省去排隊查驗通關時間。<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20171026/1229674/ |author = 黃仲明 |date = 2017年10月26日 |title = 台灣加入美國全球入境計劃 赴美通關更快速 |publisher = 蘋果日報}}</ref>

2018年，美國是中華民國第2大貿易夥伴、第3大[[進口|進口]]夥伴、第3大[[出口贸易|出口]]夥伴，在[[美洲|美洲]]經濟體皆排名第1。出口至美國的金額為396億9,411萬1,138[[美元|美元]]，年增7.450%；自美國進口的金額為347億1,784萬6,127美元，年增14.821%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）49億7,626萬5,011美元，年減25.788%。2017年，美國是中華民國第2大貿易夥伴、第3大進口夥伴、第3大出口夥伴，在美洲經濟體皆排名第1。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

出口至美國主要產品包括：[[資訊及通訊科技|資通訊產品]]、[[積體電路|積體電路]]、[[電子產品|電子產品]]、[[腳踏車|腳踏車]]、[[汽車零組件列表|車輛零附件]]、[[塑膠|塑膠]]及其製品；自美國進口主要產品包括：製造[[半導體|半導體]]的零組件、積體電路及微組件、[[民用航空器|民用航空器]]（含[[航空發動機|引擎]]及零件）、[[鐵|鐵]]屬廢料及碎屑及重熔用[[鋼|鋼]]鑄錠，以及[[煤|煤]]、煤磚、煤球及煤製類似固體燃料。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref>

中華民國商品佔美國進口市場的比重，自1991年最高點的4.72%，逐年下降至2008年1.73%、2009年1.82%、2010年1.87%、2011年1.88%、2012年1.71%、2013年1.67%，以及2014年1.73%。銷往美國的主力產品仍以電機電子類為主，2014年，此類產品佔全部銷美產品金額的36.38%，年增6.20%。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 市場環境分析 |publisher = 中華民國對外貿易發展協會}}</ref>

==交流==
===僑民===
{{ expand section }}
{{see also|臺灣裔美國人|臺灣裔{{!}}海外臺僑}}
1971年的《[[聯合國大會第2758號決議|聯合國大會第2758號決議]]》，中華民國退出聯合國及1979年[[中美斷交|中美斷交]]時，{{citation needed|造成一定規模的臺灣人移美潮。|time=2015-09-22}}

1979年，美國通過的《[[臺灣關係法|臺灣關係法]]》，將給予中国的移民配額，區分為出生地在[[中華民國自由地區|臺灣]]者與出生地在[[中國大陸|中國大陸]]者。

另外，在[[台灣省戒嚴令|戒嚴]]時期，一些留學生由於被列入[[黑名單_(台灣)|黑名單]]而無法返國，形同被迫加入美國國籍，這些屬於{{citation needed|[[離散的臺灣人|離散的臺灣人]]}}（Taiwanese diaspora）。

1980年代，{{or|由於台灣經濟起飛，有一定數量的臺灣人基於經商、求學、或追求更好生活環境品質而移民美國。|time=2015-09-22}}

根據目前保守估計，在台美國僑民數量約為5萬人<ref>[http://www.idn.com.tw/news/news_content.php?catid=1&catsid=2&catdid=0&artid=20101206abcd015 自立晚報 -維基解密：台6海底電纜 列美安全設施]</ref>。

===學術===
有鑒於美國學界對[[美國政府|美國政府]]的政策制定及[[輿論|輿論]]形成具有影響力，中華民國官方與民間向來注重與美國重要[[智庫|智庫]]及[[美國大學列表|大學]]的合作與互動，積極參與相關[[研討會|研討會]]，雙方官員或專家學者並經常互訪。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2007/html/08-1-5.htm |title = 《中華民國96年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>以加強美國政策研究界對台灣的瞭解與支持、增進與美國[[意見領袖|意見領袖]]及未來政要的關係，進而對美國行政部門發揮影響力。目前的合作計畫甚多，項目包括政策研究計畫、[[訪問學者|訪問學者]]計畫、國際學術研討會及培訓台灣相關官員等。例如與[[美國國務院|美國國務院]]合作辦理[[傅爾布萊特計畫|傅爾布萊特國際教育計畫]]（The Fulbright Program），每年雙方共同捐助百餘名學者赴美國或台灣進修、協助[[中華民國國語|華語]]或[[英語|英語]]教學。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

中華民國前外交部長與駐美代表[[程建人|程建人]]指出台灣與美國交流的智庫包括：[[傳統基金會|傳統基金會]]、[[布魯金斯研究所|布魯金斯研究所]]、[[美國外交關係協會|美國外交關係協會]]、[[卡內基國際和平基金會|卡內基國際和平基金會]]、[[戰略暨國際研究中心|戰略暨國際研究中心]]、[[蘭德公司|蘭德公司]]、{{tsl|en|American Enterprise Institute|美國企業研究所}}、[[凱托研究所|凱托研究所]]、[[彼得森國際經濟研究所|彼得森國際經濟研究所]]、美國進步中心[[亞洲協會|亞洲協會]]、{{tsl|en|Atlantic Council|大西洋理事會}}、[[國家利益中心|國家利益中心]]與[[外交政策研究所|外交政策研究所]]等智庫。其中，傳統基金會、美國企業研究所、戰略暨國際研究中心等智庫傾向[[共和黨_(美國)|共和黨]]；布魯金斯研究所、卡內基國際和平基金會、美國進步中心亞洲協會等智庫傾向[[民主党_(美国)|民主黨]]。<ref name = "程建人專欄：談談傳統基金會及華府智庫（程建人） 蘋果日報_2019"> {{ cite web | title = 程建人專欄：談談傳統基金會及華府智庫（程建人） 蘋果日報 | url = https://tw.appledaily.com/headline/daily/20190327/38292705/ | publisher = 蘋果日報 | language = zh-tw | date = 2019-03-27 }} </ref>

中華民國與[[美國州份和領地列表|美國26州]]、[[華盛頓哥倫比亞特區|華盛頓哥倫比亞特區]]、[[關島|關島]]及[[波多黎各|波多黎各]]均有免試換發[[駕駛執照|駕駛執照]]的互惠安排，符合申請條件者，可持[[中華民國汽車駕駛執照|中華民國汽車駕駛執照]]免試（部分地區仍須筆試）換發{{link-en|美國駕駛執照|Driver's licenses in the United States}}；或事先申辦中華民國[[國際駕駛執照|國際駕駛執照]]，即可立即駕車，但宜知曉美國各州是否接受國際駕駛執照。<ref name="生活資訊"/>
