'''中華民國與敘利亞關係'''是指[[中華民國|中華民國]]與[[阿拉伯敘利亞共和國|阿拉伯敘利亞共和國]]之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對敘利亞的相關事務由'''駐[[約旦|約旦]][[臺北經濟文化代表處|臺北經濟文化辦事處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-245-ef906-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
{{see also|中華民國外交|{{link-en|敘利亞外交|Foreign relations of Syria}}}}
1949年，[[臺灣戰後時期|中華民國政府遷台]]後，因國際[[冷戰|冷戰]]情勢與[[地緣政治|地緣政治]]的影響，與敘利亞的官方交流甚少。

2018年4月14日，[[美國|美國]]聯合[[英國|英國]]、[[法國|法國]][[敘利亞內戰|空襲敘利亞境內化武設施]]，[[中華民國外交部|中華民國外交部]]發言人李憲章表示，強烈反對並譴責任何政府以[[化學武器|化學武器]]等[[大規模殺傷性武器|大規模殺傷性武器]]傷害無辜平民，支持國際社會採取必要制裁行動。<ref>{{Cite web |url = https://money.udn.com/money/story/5641/3086793 |author = 劉麗榮 |date = 2018年4月14日 |title = 美英法空襲敘利亞 外交部：支持國際制裁 |publisher = [[經濟日報_(臺灣)|經濟日報]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|敘利亞簽證政策|Visa policy of Syria}}|敘利亞公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]申請商務簽證需由敘利亞籍人士在國內做保以申請簽證許可，且護照上不得有[[以色列签证政策|以色列入出境紀錄或簽證]]，否則無法入境。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-245-ef906-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|敘利亞護照|Syrian passport}}的{{link-en|敘利亞人口|Demographics of Syria|敘利亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將敘利亞歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2018年4月17日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-267-0e33a-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|敘利亞經濟}}
===貿易===
在[[敘利亞內戰|敘利亞內戰]]發生之前，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）與各[[同業公會|同業公會]]邀集[[臺灣商人|台商]]常前往敘利亞參展，爭取商機，<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=511642&url=/NewsSearch.asp |author = 王傳強 |date = 2009年6月26日 |title = 歡迎報名「2009年五金建材業赴中東拓銷團」（沙烏地阿拉伯、約旦、敘利亞） |publisher = 經貿透視網}}</ref><ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=519946&url=/NewsSearch.asp |author = 王傳強 黃榮俊 |date = 2009年10月13日 |title = 2009年五金建材業赴中東拓銷團首站抵敘利亞，爭取商機逾800萬美元 |publisher = 經貿透視網}}</ref><ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=535070&url=/NewsSearch.asp |author = 陳誕華 |date = 2010年5月17日 |title = 敘利亞投入500億美元改善公共建設，貿協組團參展以搶攻敘國建材商機 |publisher = 經貿透視網}}</ref><ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=547168&url=/NewsSearch.asp |author = 李東森 |date = 2010年10月28日 |title = 歡迎參加2011敘利亞國際工業博覽會（中華民國中東經貿協會主辦） |publisher = 經貿透視網}}</ref><ref>{{Cite web |url = http://www.trademag.org.tw/content02.asp?id=524122&type=18&url=/NewsSearch.asp |author = [[莫斯科|莫斯科]]台貿中心 |date = 2011年1月12日 |title = 為台敘展覽業 搭起交流橋樑 |publisher = 經貿透視雙周刊第309期}}</ref>
但內戰發生後至今，趨於沉寂。

2018年，敘利亞是中華民國第106大貿易夥伴、第185大[[進口|進口]]夥伴、第93大[[出口贸易|出口]]夥伴。出口至敘利亞的金額為4,167萬4,974[[美元|美元]]，年增27.985%；自敘利亞進口的金額為18萬2,267美元，年減71.511%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）4,149萬2,707美元，年增29.979%。2017年，敘利亞是中華民國第115大貿易夥伴、第164大進口夥伴、第96大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
