'''中国－阿塞拜疆關係'''（{{lang-az|Azərbaycan–Çin münasibətləri}}），是指[[中國|歷史上的中國]]和[[阿塞拜疆歷史|阿塞拜疆]]（包括[[中华人民共和国|中华人民共和国]]和[[阿塞拜疆共和国|阿塞拜疆共和国]]）之間的雙邊關係。[[习近平|习近平]]曾表示中阿两国的友好合作关系正不断发展，两国「政治关系牢固、互信水平高、务实成果丰硕」，中方支持阿塞拜疆的主权、独立<ref>{{cite news |title = 习近平会见阿塞拜疆总统阿利耶夫 |url = http://mv.chineseembassy.org/chn/sgxw/t1157867.htm |publisher = 中华人民共和国驻马尔代夫共和国大使馆 |date = 2014-05-21}}</ref>；阿塞拜疆官員於2013年接受采访时表示該國「非常希望继续提升同中方的合作水平、拓宽合作领域」，又指「阿塞拜疆的大门永远对中国敞开」<ref>{{cite news |title = 阿塞拜疆官员：阿永远对中国敞开大门 |url = http://www.chinanews.com/gj/2013/06-03/4888107.shtml |publisher = 中新社 |date = 2013-06-03}}</ref>。中国是最先承认阿塞拜疆独立和与其建交的国家之一，有指阿塞拜疆視中国为其东亚外交的优先方向<ref name=euroasia/>。

== 歷史 ==
[[File:Stamps_of_Azerbaijan,_2002-608.jpg|缩略图]]
中国和阿塞拜疆的交往可追溯至两千年前<ref name=euroasia>{{cite conference|author =孙壮志|title =阿塞拜疆与中国的关系|booktitle =列国志|volume =阿塞拜疆|publisher =社会科学文献出版社|url =http://euroasia.cass.cn/news/101769.htm|date =2005年5月|deadurl =yes|archiveurl =https://web.archive.org/web/20141006080727/http://euroasia.cass.cn/news/101769.htm|archivedate =2014-10-06}}</ref>。东起[[长安|长安]]的[[丝绸之路|丝绸之路]]经阿塞拜疆通往[[欧洲|欧洲]]，为发展东西方的文明交流发挥重要作用<ref name=important>{{cite conference|author =帕里维兹|title =近20年以来中阿经贸关系浅析|year =2013|publisher =外交学院|url =http://124.93.245.86/KCMS/detail/detail.aspx?filename=1012038375.nh&dbcode=CMFD&dbname=CMFD2013|archiveurl =https://web.archive.org/web/20141003123249/http://124.93.245.86/KCMS/detail/detail.aspx?filename=1012038375.nh&dbcode=CMFD&dbname=CMFD2013|archivedate =2014-10-03|deadurl =yes}}</ref>，12世纪的阿塞拜疆[[诗人|诗人]]曾创作有關中国的文学作品<ref name=tour/>；[[元朝|元朝]]時，蒙古大军四-{出}-征战，[[中医|中医]]术由此傳播至阿塞拜疆<ref name=hist>{{cite conference |author = 卡里莫夫·拉沙德 |title = 阿中关系的现状与前景分析 |url = http://rlls.hlju.edu.cn/manage/upload/2013928/20139288922970.pdf}}</ref>。

[[苏联|苏联]]时期，中国与阿塞拜疆的关系是[[中苏关系|中苏关系]]的一部分。苏联阿塞拜疆国家歌舞团曾到中国进行为期2个月的访问演出<ref name=hist/>，而苏联派出的[[阿塞拜疆族|阿塞拜疆族]]專家帮助中国建立石油工业，[[克拉玛依油田|克拉玛依油田]]即是其中一例<ref name=important/>；1961年初，[[阿塞拜疆共产党|阿塞拜疆共产党]]中央委员会第一书记接见中苏友好协会代表团，双方並进行谈话<ref name=hist/>。後來，[[中苏交恶|中苏交恶]]，中国与阿塞拜疆的关系陷入停顿<ref name=hist/>。現代的中阿两国於1992年4月2日建交，中方同年8月设立[[中國駐亞塞拜然大使館|中國駐亞塞拜然大使館]]<ref name=emb>{{cite web|title=中国同阿塞拜疆的关系 |url=http://az.china-embassy.org/chn/sbjw/t899351.htm |publisher=中华人民共和国驻阿塞拜疆共和国大使馆 |date=2012-01-27 |archiveurl=https://web.archive.org/web/20141003112836/http://az.china-embassy.org/chn/sbjw/t899351.htm |archivedate=2014-10-03 |deadurl=yes }}</ref>。

== 經貿關係 ==
中国和阿塞拜疆两国的经贸合作自建交以来逐步展开，但两国相距远、交通不便，而且阿塞拜疆面臨经济困难，因此两国的经贸合作水平较低<ref name=emb/>。不過，两国的双边贸易额迅速增长，在2000年至2003年的增长速度達100至200%，两国於2006年的双边贸易额為3.68亿[[美元|美元]]<ref>{{cite news |title = 双边经贸 |url = http://az.mofcom.gov.cn/article/zxhz/sbmy/200801/20080105350930.shtml |publisher = 中华人民共和国驻阿塞拜疆共和国大使馆经济商务参赞处 |date = 2008-01-23}}</ref>。截至2011年，中国是阿塞拜疆第12大贸易合作伙伴，中国商品随处可见於阿塞拜疆的市场，中国生产的日用品亦成為居民的首选<ref name=important/>。

中国的[[石油|石油]]企业自2002年来在阿塞拜疆拥有2个开发权益项目，亦有中国企业在阿塞拜疆从事石油开采钻探服务，业务量不断增长，但两国在油气领域的合作较小<ref name=econ>{{cite news |title = 中阿经济技术合作基本情况 |url = http://az.mofcom.gov.cn/article/zxhz/hzjj/201211/20121108435385.shtml |publisher = 中华人民共和国驻阿塞拜疆共和国大使馆经济商务参赞处 |date = 2012-11-14}}</ref>；[[中国电工设备总公司|中国电工设备总公司]]在阿塞拜疆承包了一些工程项目，截至2007年底，通过招标在阿塞拜疆承包工程的中国公司签约合同金额累计有约5亿美元<ref>{{cite news |title = 中阿经济技术合作 |url = http://az.mofcom.gov.cn/article/zxhz/hzjj/200801/20080105349240.shtml |publisher = 中华人民共和国驻阿塞拜疆共和国大使馆经济商务参赞处 |date = 2008-01-23}}</ref>。亦有[[华人|华人]]在阿塞拜疆經營企业，包括[[餐饮业|餐饮业]]、[[保健品|保健品]]营销等<ref>{{cite news |title = 其它华人在阿企业 |url = http://az.mofcom.gov.cn/article/zxhz/zzjg/201307/20130700217267.shtml |publisher = 中华人民共和国驻阿塞拜疆共和国大使馆经济商务参赞处 |date = 2013-07-26}}</ref>；2004年至2011年中，阿塞拜疆有40家中国企业注册成立，中国亦有16家阿塞拜疆的公司<ref name=edu/>。

== 文化關係 ==
[[File:Stamp_of_Azerbaijan_464.jpg|thumb]]）]]
阿塞拜疆設有[[孔子学院|孔子学院]]，名為巴库国立大学孔子学院<ref>{{cite web|title =巴库国立大学孔子学院|url =http://college.chinese.cn/article/2010-11/11/content_190174.htm|publisher =网络孔子学院|date =2010-11-11|deadurl =yes|archiveurl =https://web.archive.org/web/20141006113322/http://college.chinese.cn/article/2010-11/11/content_190174.htm|archivedate =2014-10-06}}</ref>，由[[安徽大學|安徽大學]]和[[巴庫國立大學|巴庫國立大學]]合辦<ref>{{cite news |title = 阿塞拜疆巴庫國立大學孔子學院揭牌 |url = http://www.chinesetoday.com/big/article/485028 |publisher = 國際日報 |date = 2011-04-22}}</ref>，阿塞拜疆公民可以在此学习[[汉语|汉语]]和中国传统习俗<ref name=edu>{{cite news|author =陶琼|title =【40家中国企业】8年40家中国企业在阿塞拜疆注册成立|url =http://www.xj.xinhuanet.com/zt/2014-09/01/c_1112306777.htm|publisher =新华社|date =2014-09-01|deadurl =yes|archiveurl =https://web.archive.org/web/20141006093338/http://www.xj.xinhuanet.com/zt/2014-09/01/c_1112306777.htm|archivedate =2014-10-06}}</ref>；2004年起，中国向阿塞拜疆[[留学生|留学生]]派發[[獎學金|獎學金]]<ref name=emb/>。同年，中国的[[艺术体操|艺术体操]]代表队和[[拳击|拳击]]代表队到阿塞拜疆參與赛事，阿塞拜疆的相應代表队亦在华參加国际比赛和艺术节<ref name=emb/>。2011年中，中国舉行「阿塞拜疆文化日」<ref name=stamp/>。

為紀念兩國建交20周年，中阿兩國政府曾發行[[紀念幣|紀念幣]]和[[郵票|郵票]]<ref name=stamp>{{cite web |title = Cultural humanitarian events |url = http://www.azerbembassy.org.cn/index.php?m=content&c=index&a=lists&catid=77 |publisher = Embassy of the Republic of Azerbaijan in the People's Republic of China}}</ref>。兩國的一些新聞機構（如{{link-en|阿塞拜疆国家通讯社|Azerbaijan State Telegraph Agency}}和[[新华社|新华社]]）之間有发展合作关系<ref>{{cite news |title = 中国-亚欧伟大丝绸之路的复兴和媒体合作的加强 ——阿塞拜疆的经验 |url = http://www.chinatoday.com.cn/ctchinese/chinaworld/article/2014-08/31/content_637723.htm |publisher = 今日中国杂志社 |date = 2014-08-31}}</ref>；阿塞拜疆正在开拓在中国的航空旅游市场，如[[阿塞拜疆航空|阿塞拜疆航空]]开通的[[北京|北京]]航线即為其第一条开通长航线，巴库到[[伊斯坦布尔|伊斯坦布尔]]的航班亦成為部分中国旅客前往[[土耳其|土耳其]]的选择<ref name=tour>{{cite news |title = 阿塞拜疆加快开拓我国旅游市场 |url = http://finance.china.com.cn/roll/20140314/2258268.shtml |publisher = 中国网 |date = 2014-03-14}}</ref>。

== 軍事關係 ==
2005年，阿塞拜疆退伍軍人协会主席等人应邀出席[[北京|北京]]纪念[[抗战|抗战]]暨世界反[[法西斯|法西斯]]战争胜利60周年的庆祝活动<ref name=emb/>。於2007年和2008年，中阿两国在军事领域的交往活跃，{{link-en|阿塞拜疆武裝部隊總參謀長|General Staff of Azerbaijani Armed Forces}}{{link-en|纳季梅金·萨德科夫|Najmaddin Sadigov}}[[上将|上将]]曾访华，而[[中华人民共和国国防部|中华人民共和国国防部]]外事办公室副主任[[丁进攻|丁进攻]][[少将|少将]]也曾会见{{link-en|阿塞拜疆国防部|Ministry of Defense (Azerbaijan)}}部长{{link-en|薩法爾·阿比耶夫|Safar Abiyev}}[[上将|上将]]<ref name=emb/>。

2011年，中阿兩國签署国际协定书，中方向阿塞拜疆提供600万元的无偿军事援助；2013年1月，阿塞拜疆国防部与中国国防部签署协议，中方再向阿塞拜疆300万元军事援助<ref>{{cite news |title = 媒体称中国将为阿塞拜疆提供300万军事援助 |url = http://mil.huanqiu.com/observation/2013-05/3975628.html |publisher = 环球网 |date = 2013-05-28}}</ref>。2014年7月，阿塞拜疆语言大学中文和文化中心组织举行有關两国军事外交的研讨会，以纪念两国武装力量的创建，期間中国驻阿塞拜疆大使馆[[武官|武官]][[刘洪讲|刘洪讲]]講述了[[中华人民共和国军事|中华人民共和国军事]]概况<ref>{{cite news |title = 在阿塞拜疆与中国两国间合作扩大军事外交的作用 |url = http://azertag.az/chn/xeber/%E5%9C%A8%E9%98%BF%E5%A1%9E%E6%8B%9C%E7%96%86%E4%B8%8E%E4%B8%AD%E5%9B%BD%E4%B8%A4%E5%9B%BD%E9%97%B4%E5%90%88%E4%BD%9C%E6%89%A9%E5%A4%A7%E5%86%9B%E4%BA%8B%E5%A4%96%E4%BA%A4%E7%9A%84%E4%BD%9C%E7%94%A8-7319 |publisher = 阿塞拜疆国家通讯社 |date = 2014-07-18}}</ref>。
