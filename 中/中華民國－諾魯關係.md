'''中華民國－諾魯關係'''是指[[中華民國|中華民國]]與[[諾魯共和國|諾魯共和國]]之間的外交關係。

==歷史==
*1975年5月諾魯在[[臺北|臺北]]設立[[領事館|領事館]]。<ref name="中華民國外交部86年外交年鑑">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html|title=中華民國外交部86年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date = 1998-06 |accessdate=2018-03-01}}</ref>
*1980年5月4日[[諾魯總統|諾魯總統]]戴羅伯（Hammer DeRoburt）訪華，中華民國與諾魯正式建立領事級外交關係，並設立總領事館。<ref name="中華民國外交部86年外交年鑑"/>
*1990年8月17日中華民國政府將總領事館升格為[[大使館|大使館]]。<ref name="中華民國外交部86年外交年鑑"/>
*1995年2月28日諾魯政府因經費短絀暫時關閉臺北領事館。<ref name="中華民國外交部86年外交年鑑"/>
*2002年7月23日中華民國與諾魯中止外交關係。因為其和中華人民共和國建交。<ref name=h2>{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf|title=中華民國外交部105年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-05 |accessdate=2018-03-01}}</ref>
*2005年5月24日中華民國外交部長[[陳唐山|陳唐山]]與諾魯-{zh-cn:總統路德維格·斯科蒂;zh-sg:總統路德維格·斯科蒂;zh-hk:總統路德維格·斯科蒂;zh-tw:總統路德維格·史可迪}-在台北簽署復交公報，兩國恢復大使級外交關係<ref>{{cite news|url=http://www.mofa.gov.tw/News_Content_M_2.aspx?n=8742DCE7A2A28761&sms=491D0E5BF5F4BC36&s=31CAF663FD64CCEA|title=中華民國（台灣）政府與諾魯共和國政府基於平等互惠之原則及發展兩國友好關係之共同意願，決定自2005年5月14日起恢復大使級外交關係。|publisher=[[中華民國外交部|中華民國外交部]]|date=2005-05-14|accessdate=2015-08-16}}</ref>。
*2005年8月8日中華民國在諾魯[[愛和區|愛和區]]復設大使館。<ref name=h2>{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf|title=中華民國外交部105年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-05 |accessdate=2018-03-01}}</ref>
*2007年3月27日諾魯在臺北設立大使館。<ref name=h2>{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf|title=中華民國外交部105年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-05 |accessdate=2018-03-01}}</ref>

==兩國貿易==
2018年諾魯是中華民國第112名貿易伙伴，兩國貿易額32,683,528美元，其中中華民國對諾魯出口32,640,805美元，是第96大出口國，進口42,723美元，是第202大進口國，與2017年貿易額相比貿易總額約增加9309.85%、出口約增加10048.37%、進口約增加66.26%。<ref>{{cite book |url = http://cus93.trade.gov.tw/FSCI/ |title = 經濟部國際貿易局貿易統計系統 |publisher = [[經濟部國際貿易局|經濟部國際貿易局]] |date = 2019-02-23 |accessdate = 2019-03-05}}</ref>
*中華民國－諾魯主要出口項目：石油、燈具及照明配件、撚線、繩、索及纜、變壓器、其他鋼鐵製品等。<ref name=bft>{{Cite web |url = https://www.trade.gov.tw/World/Detail.aspx?nodeID=45&pid=581418|title = 各國經貿資訊 |publisher =[[經濟部國際貿易局|經濟部國際貿易局]]|date = 2018-02-08|accessdate = 2018-03-04}}</ref>
*中華民國－諾魯主要進口項目：纖維素紙漿、紙或紙板之製造或整理之機械、首飾及其配件等。<ref name=bft>{{Cite web |url = https://www.trade.gov.tw/World/Detail.aspx?nodeID=45&pid=581418|title = 各國經貿資訊 |publisher =[[經濟部國際貿易局|經濟部國際貿易局]]|date = 2018-02-08|accessdate = 2018-03-04}}</ref>

===航空客運===
兩國目前無直航班機，來往諾魯之客運航線有以下方式：

1.搭乘飛往{{flagicon|AUS}}[[布里斯班机场|布里斯本]]班機。

2.轉機即可到達{{flagicon|Nauru}}[[雅連區|雅連]]-[[諾魯國際機場|諾魯國際機場]]。

＊航線會因航空公司營運狀況更改或取消，出發前應與負責該航線的航空公司確認航線及航班，以免影響權益。
