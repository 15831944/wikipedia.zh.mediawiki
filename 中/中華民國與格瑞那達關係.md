'''中華民國與格瑞那達關係'''是指[[中華民國|中華民國]]與[[格瑞那達|格瑞那達]]之間的關係。兩國於1989－2005年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對格瑞那達的相關事務由'''中華民國駐[[聖文森及格瑞那丁|聖文森國]][[中華民國駐外機構列表|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-260-6799d-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，[[格瑞那達|格瑞那達]]當時屬於{{link-en|英領西印度群島|British West Indies|英國殖民地}}，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
{{see also|中華民國外交|{{link-en|格瑞那達外交|Foreign relations of Grenada}}}}
{{see also|中華民國駐格瑞那達大使列表{{!}}中華民國駐格瑞那達外交官列表}}
1989年7月20日，中華民國與格瑞那達建立大使級[[外交#外交上的承認|外交關係]]。31日，於首都[[圣乔治_(格林纳达)|聖喬治]]設立'''中華民國駐格瑞那達[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]（兼駐[[安地列斯群島|安地列斯群島]]的[[中華民國外交#邦交國|邦交國]]大使）。<ref name="91年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__06.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

格瑞那達則在首都[[臺北市|臺北]]設立'''駐華[[名譽領事|名譽領事]]'''。1991年3月，任命[[黃書瑋|黃書瑋]]為名譽領事。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1996年3月，{{link-en|格瑞那達政府|Politics of Grenada}}對中華人民共和國[[台灣海峽飛彈危機|在台灣海峽試射飛彈]]，威脅中華民國即將於3月23日舉行的[[1996年中華民國總統選舉|首次總統直接選舉]]一事，發表聲明極表關切。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=81638EAF4C30780A |author = 公眾外交協調會 |date = 1996年3月12日 |title = 格瑞那達總理宓契爾獲悉中共試射飛事件 |publisher = 中華民國外交部}}</ref>

2004年9月7日，格瑞那達遭遇[[飓风伊万的气象历史|艾文颶風]]侵襲，災情慘重，[[中華民國外交部|中華民國外交部]]提供[[人道援助|人道援助]]，外交部長[[陳唐山|陳唐山]]率團前往賑災慰問。除提供20萬[[美元|美元]]人道物資外，並將格瑞那達原用於「民生計畫」的150萬美元贈款改為免提計畫之賑災款，另同意提供800萬美元的災後重建款。<ref name="終止"/>

2004年12月22日，由於格瑞那達總理[[基思·米切尔|宓契爾]]前往[[北京|北京]]與[[中華人民共和國|中華人民共和國]]商談建交事宜，中華民國外交部於是召回駐在大使。<ref name="終止"/>

2005年1月27日，格瑞那達與中華民國斷交。<ref name="終止">{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=5028B03CED127255&sms=5ED24855AD8E6C58&s=DA55444F1D2BC6AE |author = 公眾外交協調會 |date = 2005年1月27日 |title = 格瑞那達政府在中國利誘下，決定與中國建交，中華民國政府至表遺憾。中華民國政府為維護國家尊嚴與利益，決定自即日起終止與格瑞那達之外交關係，並停止一切援助計畫。 |publisher = 中華民國外交部}}</ref>

斷交前，格瑞那達多次發言支持中華民國參與[[聯合國|聯合國]]，以及用[[世界衛生大會觀察員|觀察員]]身分參加[[世界衛生組織|世界衛生組織]]（WHO）的[[世界衛生大會|世界衛生大會]]（WHA）。

兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可將簽證申請表及相關資料以[[電子郵件|電子郵件]]傳至格瑞那達外交部，列印並攜帶許可函入境，於{{link-en|莫里斯·畢曉普國際機場|Maurice Bishop International Airport|機場}}繳交規費。經格瑞那達轉機，亦須申請簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-260-6799d-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持有[[格瑞那達護照|格瑞那達護照]]的{{link-en|格瑞那達人口|Demographics of Grenada|格瑞那達公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將格瑞那達歸類為'''灰色警示'''：提醒注意（2017年5月1日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-220-3e6bb-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將格瑞那達歸類為'''第二級警示'''：對當地採取加強防護（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=109B9B14D881EED2 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
2018年，格瑞那達是中華民國第204大貿易夥伴、第174大[[進口|進口]]夥伴、第201大[[出口贸易|出口]]夥伴，在[[美洲|美洲]]國家排名倒數第1、5、1；在[[世界政區索引|世界經濟體]]倒數第41、67、36。出口至格瑞那達的金額為26萬9,303[[美元|美元]]，年增64.131%；自格瑞那達進口的金額為25萬6,463美元，年增105.475%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1萬2,840美元，年減67.297%。2017年，格瑞那達是中華民國第206大貿易夥伴、第189大進口夥伴、第202大出口夥伴，在美洲國家排名倒數第2、3、2；在世界經濟體倒數第33、48、29。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===合作===
兩國在邦交期間，中華民國在格瑞那達派駐[[農業技術|農業技術]]團。合作項目有[[食品加工|食品加工]]、[[水利|水利]][[灌溉|灌溉]]、[[蔬果|蔬果]]、[[園藝|園藝]]、[[水產養殖|水產養殖]]，以及[[小农经济|小農]][[貸款|貸款]]等計畫。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter1-6.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref name="89年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_10.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref name="91年外交年鑑"/>
