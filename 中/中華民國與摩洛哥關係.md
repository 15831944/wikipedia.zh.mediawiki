===外交===
{{see also|中華民國外交|{{link-en|摩洛哥外交|Foreign relations of Morocco}}}}
2003年6月20日，由[[行政院新聞局|行政院新聞局]]副局長[[李雪津|李雪津]]率領之代表團前往摩洛哥舉行的第13屆全球婦女{{link-en|高峰會|Summit (meeting)}}（Global Summit of Women），據團員事後說明在[[中華人民共和國|中華人民共和國]]的幕後干擾下，代表團員雖已取得[[簽證|簽證]]，但{{link-en|摩洛哥政府|Politics of Morocco}}卻以其駐[[香港|香港]][[領事館|領事館]]所發簽證無效為由阻止團員入境。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=B81B5A2FC67CDD90&sms=175F59D3F37F17B5&s=0DBFCFB0D4042AA9 |author = 公眾外交協調會 |date = 2003年7月14日 |title = 中國大陸阻撓我國際空間事例（92年4月至6月事例）|publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=70BCE89F4594745D&sms=700DE7A3F880BAE6&s=B9B86A6E7060B312 |author = 公眾外交協調會 |date = 2003年6月30日 |title = 針對我參加「二○○三年全球婦女高峰會」代表團遭中共打壓事記者會說明紀要 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=70BCE89F4594745D&sms=700DE7A3F880BAE6&s=21E171208AC7F27B |author = 公眾外交協調會 |date = 2004年6月10日 |title = 單位主管新聞說明會紀要--(NGO委員會) |publisher = 中華民國外交部}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|摩洛哥簽證政策|摩洛哥公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]申辦簽證可郵寄至摩洛哥駐[[日本|日本]][[外交代表機構#大使馆|大使館]]或親自前往摩洛哥駐[[韓國|韓國]]大使館。郵寄辦理核准後會[[傳真|傳真]]許可文件（需1－2個月），持文件於抵達{{link-en|摩洛哥機場列表|List of airports in Morocco|摩洛哥機場}}時領取簽證並繳費入境；親自辦理則可當場准駁。經摩洛哥轉機須辦理[[過境簽證|過境簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-129-8e956-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

持[[摩洛哥護照|摩洛哥護照]]的{{link-en|摩洛哥人口|Demographics of Morocco|摩洛哥公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將摩洛哥歸類為'''灰色警示'''：提醒注意（2017年4月22日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-367-4fa6f-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
[[File:Two_females_riding_Kymco_scooter_in_Marrakech_(4593194590).jpg|thumb]]西南大城：[[馬拉喀什|馬拉喀什]]街頭的臺灣[[光陽工業|KYMCO]][[機車|機車]]]]
2010年11月24日，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）舉辦的「2010年[[北部非洲|北非]]貿易訪問團」於第1大城[[卡薩布蘭卡|卡薩布蘭卡]]舉辦貿易洽談會，由摩洛哥商工總會[[理事長|理事長]]開幕並致辭。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=549689&url=/NewsSearch.asp |author = 楊淨枝 |date =  2010年11月29日 |title = 「2010年北非貿易訪問團」摩洛哥站貿易洽談會，買主質量並重，創造豐富商機。 |publisher = 經貿透視網}}</ref>

2013年11月9日，[[經濟部國際貿易局|經濟部國際貿易局]]委託外貿協會與21家台灣業者執行「2013年[[西部非洲|西]]北非主力市場拓銷團」，共前往[[摩洛哥|摩洛哥]]、[[迦納|迦納]]、[[科特迪瓦|科特迪瓦]]、[[奈及利亞|奈及利亞]]等國。選擇摩洛哥是以其前進[[欧洲联盟成员国|歐盟市場]]的跳板，也是[[阿拉伯馬格里布聯盟|阿拉伯馬格里布聯盟]]（AMU）成員國，該聯盟人口約9,000萬，具有市場潛力。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622898&url=/NewsSearch.asp |author = 中華民國對外貿易發展協會 |date = 2013年11月11日 |title = 貿協領軍 前進西北非拓銷 |publisher = 經貿透視網}}</ref>

2017年，摩洛哥是中華民國第85大貿易夥伴、第74大[[進口|進口]]夥伴、第82大[[出口贸易|出口]]夥伴。出口至摩洛哥的金額為6,533萬4,460[[美元|美元]]，年增8.906%；自摩洛哥進口的金額為6,664萬8,565美元，年增25.863%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）131萬4,105美元，年減118.671%。※2016年，摩洛哥是中華民國第85大貿易夥伴、第76大進口夥伴、第81大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
