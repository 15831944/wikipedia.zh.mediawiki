== 經歷 ==
1984年出生於[[靜岡縣|靜岡縣]]陶器家庭。小時候受到父親影響，以及《[[七龙珠|七龙珠]]》的影響而開始畫漫畫。初中三年級時決定以[[漫畫家|漫畫家]]為目標。

2001年，16歲時（高中一年級）在《[[月刊GANGAN_WING|月刊GANGAN WING]]》11月號上刊載《{{lang|ja|[[海里の陶|海里の陶]]}}》出道。

2011年秋因生產而暫停兩部作品。11月16日在[[Twitter|Twitter]]上發表順利生產的消息<ref>{{ja}}[https://twitter.com/hikakom/status/136808792393400323 Twitter / hikakom: ご報告遅れましたが、無事赤さんが産まれました！ちょっとづつで ...]</ref>。2012年春恢復連載。

== 人物 ==
