'''中国人民解放军新闻传播中心广播电视部'''，位于[[北京市|北京市]]，隶属[[中国人民解放军新闻传播中心|中国人民解放军新闻传播中心]]。

==沿革==
1965年，经[[中央军委|中央军委]]批准，北京电视台（1978年5月1日更名为[[中国中央电视台|中国中央电视台]]）成立军事组，设有专职军事记者，承担电视军事节目摄制工作。1979年3月，中央电视台军事组改为'''中央电视台军事部'''，军事记者由现役军人出任<ref name=jm>{{Cite web |url = http://www.81.cn/jwgz/2016-12/27/content_7397414.htm |title =除了《解放军报》，这些军队媒体也同样精彩  |author =  |date =2016-12-27   |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref>。

1996年1月1日，[[中国中央电视台军事·农业频道|中央电视台军事·农业·少儿·科技频道]]开播，中国人民解放军电视宣传中心筹建期间设立的《神州军旅》、《军事新闻》、《军营文化广场》等新栏目陆续在该频道开播。1996年2月7日，[[中国人民解放军总参谋部|中国人民解放军总参谋部]]批准中国人民解放军电视宣传中心编制，编为[[正师级|正师级]]单位，两位主官高配（副军职）<ref name=jl>{{Cite web|url =http://www.js7tv.cn/news/201602_36182.html|title =1996年：初创之年筚路蓝缕肩担使命|author =|date =2016-02-01|publisher =中国军视网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170119052738/http://www.js7tv.cn/news/201602_36182.html|archivedate =2017-01-19}}</ref>。

1996年4月11日，中国人民解放军电视宣传中心成立大会在[[中国人民解放军总政治部|中国人民解放军总政治部]]大楼举行，中央军委委员、总政治部主任[[于永波|于永波]]，副主任[[周子玉|周子玉]]、[[唐天标|唐天标]]、[[袁守芳|袁守芳]]，[[中华人民共和国广播电影电视部|中华人民共和国广播电影电视部]]副部长兼中央电视台台长[[杨伟光|杨伟光]]等出席。大会宣读中央军委命令：中央电视台军事部、[[八一电影制片厂|八一电影制片厂]]新闻纪录片部、总政治部电教中心正式合并组建'''中国人民解放军电视宣传中心'''（对内称中国人民解放军电视宣传中心，对外称'''中央电视台军事节目中心'''），接受中国人民解放军总政治部和中央电视台双重领导，承担四项任务：“宣传党中央和中央军委决策指示、报道军委总部首长重要活动、报道国防和军队建设重大事件、普及国防知识和国防交通”<ref name=jm/><ref name=jl/>。

1996年5月15日至16日，中国人民解放军电视宣传中心在空军南苑招待所召开全军电视宣传工作会议，这是中国人民解放军历史上首次全军电视宣传工作会议。1996年7月，为庆祝中国人民解放军建军69周年，中国人民解放军电视宣传中心举办第一届军旅歌曲电视大赛“我们的队伍向太阳——96军旅歌曲电视大赛”，这是中国人民解放军历史上第一次举办军旅歌曲电视大赛。1996年9月8日至24日，[[中央军委副主席|中央军委副主席]][[刘华清|刘华清]]一行访问[[法国|法国]]、[[意大利|意大利]]，中国人民解放军电视宣传中心派出汪恒、张正梅随团采访，这是该中心成立后第一次派记者到国外采访<ref name=jl/>。

中国人民解放军电视宣传中心是[[中国人民解放军|中国人民解放军]]唯一的[[电视|电视]][[宣传|宣传]]常设机构，承担央视军事节目的制作和播出任务，主要节目在[[中国中央电视台军事·农业频道|中国中央电视台军事·农业频道]]播放，制作播出的节目主要有《[[军事报道|军事报道]]》、《[[军事纪实|军事纪实]]》、《[[百战经典|百战经典]]》、《[[红星艺苑|红星艺苑]]》、《[[军事科技|军事科技]]》、《[[和平年代|和平年代]]》、《[[超级战士|超级战士]]》、《[[军旅文化·大视野|军旅文化·大视野]]》、《[[军情连连看|军情连连看]]》、《[[人民子弟兵|人民子弟兵]]》、《[[防务新观察|防务新观察]]》、《[[中国武警|中国武警]]》等。

在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月，其原上级单位[[中国人民解放军总政治部|中国人民解放军总政治部]]撤销，成立[[中央军委政治工作部|中央军委政治工作部]]。该中心转隶中央军委政治工作部<ref name=wy>[http://money.163.com/16/0810/15/BU4A1BI500253B0H.html 原南京军区政治部宣传部部长徐黄生履新军委政治工作部宣传局，网易，2016-08-10]</ref>。

中国人民解放军电视宣传中心是中国人民解放军全军唯一的电视宣传常设机构，接受中央军委政治工作部、国家新闻出版广电总局和中央电视台的领导，承担中央电视台所有军事节目的制作和播出任务。截至2016年，中国人民解放军电视宣传中心在中国人民解放军和[[中国人民武装警察部队|中国人民武装警察部队]]共设有10多个记者站，在全军团以上单位设电视报道骨干或者通讯员，全军直接从事电视报道人员超过1000人，并且建立起一支超过100人的应急电视报道队伍。中国人民解放军电视宣传中心还创办了中国军视网、中国军视网微信、军视TV客户端<ref name=jm/>。

2018年，在[[深化国防和军队改革|深化国防和军队改革]]中，原中国人民解放军电视宣传中心、[[中国人民解放军新闻传播中心网络部|中央军委政治工作部文网中心]]、[[中国人民解放军出版社|中国人民解放军出版社]]、[[解放軍報|解放军报社]]等单位合并成立[[中国人民解放军新闻传播中心|中国人民解放军新闻传播中心]]<ref name=ding>{{Cite web |url =  http://media.people.com.cn/n1/2018/0620/c40606-30067803.html|title = 丁海明：军队媒体人有一个"强网梦"|author =  |date = 2018-06-20 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。其中，原中国人民解放军电视宣传中心等单位合并成立'''中国人民解放军新闻传播中心广播电视部'''<ref>{{Cite web |url =http://www.81.cn/jsjz/2018-06/14/content_8061974.htm  |title =积极培养新型军事广播电视人才  |author =  杜娟|date =  |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref>。
