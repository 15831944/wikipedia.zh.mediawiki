白俄羅斯曾是中華民國發展與[[獨立國家國協|獨立國家國協]]關係的重點國家。1996年1月，兩國代表簽署《[[中華民國政府|中華民國政府]]與[[白俄羅斯政府|白俄羅斯共和國政府]]有關經濟貿易、科學技術及文化合作暨設立經濟貿易代表團議定書》。7月1日，於首都設立'''駐明斯克[[台北经济文化代表处|台北經濟貿易代表團]]'''（Taipei Economic and Trade Mission in Minsk）<ref name="88年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter1-2.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref name="代表團">{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=B8A09D996D9B5FC0 |author = 公眾外交協調會 |date = 1996年7月5日 |title = 中華民國政府與白俄羅斯共和國政府有關經濟貿易、科學技術及文化合作暨設立經濟貿易代表團議定書 |publisher = 中華民國外交部}}</ref>。但之後基於外交實務考量且館務功能不彰，於2006年1月關閉代表團<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=1C70506E19AE137B |author = 公眾外交協調會 |date = 2006年1月3日 |title = 外交部宣布即日起關閉我駐白俄羅斯代表處 |publisher = 中華民國外交部}}</ref>。然而，[[中華民國外交部|中華民國外交部]]官员向媒体透露，闭馆原因与白俄罗斯反对台湾加入[[联合国|联合国]]和[[世界卫生组织|世界卫生组织]]，並多次拒发官员签证有关<ref>{{cite news|author=李佳霏 |authorlink=李佳霏| title=關閉駐白俄羅斯代表處 陳唐山：戰略考量 |url=http://www.epochtimes.com/b5/6/1/3/n1176892.htm |work=[[中央社|中央社]] |publisher=大紀元 |date=2006-01-03 |accessdate=2016-10-16}}</ref>。

1999年11月，[[白俄羅斯國民議會|白俄羅斯國民議會]]議員羅曼諾夫斯基（Romanovsky）、瓦羅士寧（Voroshnin）抵台訪問，這也是該國國會議員首次到訪。<ref name="88年外交年鑑"/>

2001年8月，{{link-be|白俄羅斯財政部|Міністэрства фінансаў Рэспублікі Беларусь}}第一副部長盧莫斯（Rumus）抵台訪問，並接受中華民國行政院長[[張俊雄|張俊雄]]代表政府捐贈2萬[[美元|美元]]給予白俄羅斯「[[切尔诺贝利核事故|車諾比災變]]善後委員會」。<ref name="90年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2001/Chap3/3-1-2.htm |title = 《中華民國90年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2002－2005年，白俄羅斯前總理[[弗拉基米尔·叶尔莫申|葉爾莫森]]（Yermoshin）、前副總理-{悉尼}-強（Sinitzin）、{{link-en|白俄羅斯紅十字會|Belarus Red Cross}}會長羅曼諾夫斯基（Romanovsky）、獨立國家國協常設委員會第一副主席布拉霍夫（Bulakhov）、[[自由民主党_(白俄罗斯)|白俄羅斯自由民主黨]]主席暨國會議員賈杜凱維奇（Gaidukevich）、副主席奇申科（khischenko）、{{link-en|白俄羅斯國防部|Defence Ministry of Belarus|國防部}}前第一副部長杜辛斯基（Tushinsky）等人相繼抵台訪問。<ref name="91年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__02.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2004/08/08_01_02.htm |title = 《中華民國93年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2005/WebPages/MICO/08/08_01_02.htm |title = 《中華民國94年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

===貿易===
2013年，出口至白俄羅斯的主要產品有：[[機器|機器]]零件、[[數位相機|數位相機]]、[[金屬|金屬]][[車床|車床]]、[[橡膠|橡]][[塑膠|塑膠]]加工機、[[化妝品|化妝品]]、[[儲存裝置|儲存裝置]]、[[自動資料收集|自動資料處理機]]、[[手工具|手工具]]等；自白俄羅斯進口的主要產品有：[[雜環化合物|雜環化合物]]、[[草木灰|鉀肥]]等，該2項產品佔進口總額97%。<ref name="白俄">{{Cite web |url = http://www.trademag.org.tw/News.asp?id=649283&url=/NewsSearch.asp |author = 市場拓展處 |date = 2014年10月30日 |title = 貿協東歐拓銷團白俄首站告捷  |publisher = 經貿透視網}}</ref>

2014年10月28日，[[經濟部國際貿易局|經濟部國際貿易局]]委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）籌組包含30家廠商的「2014年[[中歐|中]][[東歐|東歐]]暨以色列利基產業拓銷團」，共前往[[白俄羅斯|白俄羅斯]]、[[以色列|以色列]]、[[羅馬尼亞|羅馬尼亞]]及[[保加利亞|保加利亞]]4國爭取商機。首站白俄羅斯貿易洽談會於首都[[明斯克|明斯克]]舉行。<ref name="白俄"/>

2018年，白俄羅斯是中華民國第114大貿易夥伴、第110大[[進口|進口]]夥伴、第111大[[出口贸易|出口]]夥伴。出口至白俄羅斯的金額為2,079萬1,862[[美元|美元]]，年增17.313%；自白俄羅斯進口的金額為1,108萬3,575美元，年減70.757%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）970萬8,287美元，年減148.112%。2017年，白俄羅斯是中華民國第102大貿易夥伴、第86大進口夥伴、第112大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
