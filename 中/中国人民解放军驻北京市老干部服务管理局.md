'''中国人民解放军驻北京市老干部服务管理局'''，位于[[北京市|北京市]][[海淀区|海淀区]][[中关村南大街|中关村南大街]]28号，隶属[[中国人民解放军北京卫戍区|中国人民解放军北京卫戍区]]。

==沿革==
;中央军委政治工作部白石桥老干部服务管理局
前身是'''中国人民解放军总政治部白石桥老干部服务管理局'''<ref>{{Cite web |url = http://news.youth.cn/js/201509/t20150901_7069106.htm |title = 总政系统开展纪念抗战胜利70周年活动 |author =  |date = 2015-09-01 |publisher = 中国青年网 |language =  |accessdate =  |quote =  }}</ref>，隶属[[中国人民解放军总政治部|中国人民解放军总政治部]]。下设多个离职干部休养所。1986年成立中国人民解放军总政治部白石桥老干部服务管理局老年大学<ref>周宏美，总政白石桥老干部服务管理局党委——精心为老干部办好老年大学，军队党的生活2010:89-89</ref>。2016年在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军总政治部撤销。该局转隶[[中央军委政治工作部|中央军委政治工作部]]，更名'''中央军委政治工作部白石桥老干部服务管理局'''<ref>{{Cite web |url =http://www.js7tv.cn/news/201607_51183.html  |title =白石桥老干部服务管理局老干部学习党章纪实  |author =  |date =2016-07-01  |publisher = 中国军视网 |language =  |accessdate =  |quote =  }}</ref>，仍位于北京市海淀区中关村南大街28号。下设第一至第六离职干部休养所<ref>{{Cite web |url =  http://www.bidchance.com/calggnew/2017/06/27/20809960.html|title =  中央军委政治工作部白石桥老干部服务管理局第六离职干部休养所综合整治工程|author =  |date =2017-06-27  |publisher =中国招标网  |language =  |accessdate =  |quote =  }}</ref>

;中央军委政治工作部玉泉路老干部服务管理局
前身是'''中国人民解放军总政治部玉泉路老干部服务管理局'''，隶属[[中国人民解放军总政治部|中国人民解放军总政治部]]<ref name=xuxp>徐孝鹏:服务老干部要用真情献真爱，军队党的生活2012(5)</ref>。下设玉泉路、金沟河、朝阳、大兴、良乡离职干部休养所。1987年12月26日，成立中国人民解放军总政治部玉泉路老干部服务管理局老干部大学<ref>梁玲，十年情谊一刊牵，时事报告2011(2):86-87</ref><ref>{{Cite web |url = http://mil.news.sina.com.cn/2012-12-25/0440710650.html |title =  快乐的精神家园|author =  |date = 2012-12-25 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。2016年在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军总政治部撤销。该局转隶[[中央军委政治工作部|中央军委政治工作部]]，更名'''中央军委政治工作部玉泉路老干部服务管理局'''。仍位于北京市海淀区[[复兴路_(北京)|复兴路]]40号。

;中央军委机关事务管理总局北极寺老干部服务管理局
位于北京市[[海淀区|海淀区]]花园东路8号<ref>{{Cite web |url = http://www.bcactc.com/home/gcxx/sgzbgs_show.aspx?gcbh=11023120170011 |title =施工中标候选人公示  |author =  |date = 2017-04-07 |publisher =北京市建设工程信息网  |language =  |accessdate =  |quote =  }}</ref>。原为'''中国人民解放军总参谋部管理局北极寺离休干部服务管理处'''。2003年12月，中国人民解放军总参谋部管理局扩编为[[中国人民解放军总参谋部管理保障部|中国人民解放军总参谋部管理保障部]]，该处改为'''中国人民解放军总参谋部管理保障部北极寺离休干部服务管理处'''。2006年，在北京召开了中国人民解放军总参谋部管理保障部北极寺离休干部服务管理处更名'''中国人民解放军总参谋部管理保障部北极寺老干部服务管理局'''大会，总参谋部管理保障部[[李天策|李天策]]部长讲话要求北极寺老干部服务管理局作为全军最大的离休干部管理机构，高标准做好老干部“两高期”的服务管理工作<ref>高标准做好“两高期”服务管理工作，中国老年报2007年2月1日</ref>。

中国人民解放军总参谋部管理保障部北极寺老干部服务管理局隶属[[中国人民解放军总参谋部管理保障部|中国人民解放军总参谋部管理保障部]]。成立时担负着来自全军七个大单位1800名老干部的服务管理工作<ref>{{Cite web |url = http://politics.people.com.cn/GB/30178/4347205.html |title = 某部老干部学习科学发展观座谈会侧记 |author =  |date = 2006-05-03 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。下设幼儿园<ref>{{Cite web|url = http://edu.bjhd.gov.cn/syey/201604/t20160421_1261444.html|title = 中国人民解放军总参谋部管理保障部北极寺老干部服务管理局幼儿园|author = |date = 2016-04-21|publisher = 海淀教育|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180813175412/http://edu.bjhd.gov.cn/syey/201604/t20160421_1261444.html|archive-date = 2018-08-13|dead-url = yes}}</ref>、门诊部以及十多个干休所<ref>{{Cite web |url = https://www.lawxp.com/case/c24406844.html |title = 上诉人中国人民解放军总参谋部管理保障部北极寺老干部服务管理局第九干休所因机动车交通事故责任纠纷一案 |author =  |date = 2016-07-19 |publisher =汇法网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://renshi.people.com.cn/GB/139631/10011088.html |title = 受表彰先进干休所、先进离退休干部、先进老干部工作者名单 |author =  |date = 2009-09-08 |publisher =  中国共产党新闻网|language =  |accessdate =  |quote =  }}</ref>等机构。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军总参谋部管理保障部撤销。该局转隶[[中央军委机关事务管理总局|中央军委机关事务管理总局]]，更名'''中央军委机关事务管理总局北极寺老干部服务管理局'''。下辖33个干休所，服务的离退休老干部从副军级到正大军区级<ref name=huangtao>{{Cite web |url = https://baijiahao.baidu.com/s?id=1590259741879425830&wfr=spider&for=pc |title = 黄涛：“老干部健康长寿就是我们的工作成绩 |author =  |date =2018-01-22  |publisher = 百家号 |language =  |accessdate =  |quote =  }}</ref>。该局还下设有幼儿园<ref>{{Cite web |url = http://www.sohu.com/a/134886380_650096 |title = 收藏！2017年海淀区幼儿园名录（188家） |author =  |date =2017-04-18  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>等单位。

;中国人民解放军驻北京市老干部服务管理局
2016年，军队各级党委机关和地方各级党委政府，按照[[习近平|习近平]]关于做好老干部工作的指示，将军队离退休干部移交政府安置工作作为服务[[深化国防和军队改革|深化国防和军队改革]]的重要任务，2016年共向政府移交安置离退休干部7400多人，全军历年累计移交安置人数突破30万<ref>{{Cite web |url =http://www.mod.gov.cn/topnews/2017-01/18/content_4770293.htm  |title =民政部军委政治工作部北京市政府慰问移交政府安置的军队离退休干部 |author =  |date = 2017-01-18 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

2017年底，成立'''中国人民解放军驻北京市老干部服务管理局'''，隶属[[中国人民解放军北京卫戍区|中国人民解放军北京卫戍区]]。设有指挥监控中心、文化活动中心、门诊部、食堂等设施。2018年2月13日，[[中共中央政治局委员|中共中央政治局委员]]、[[中央军委副主席|中央军委副主席]][[张又侠|张又侠]]到中国人民解放军驻北京市老干部服务管理局调研慰问<ref>{{Cite web |url = http://www.81.cn/jwzb/2018-02/13/content_7944282.htm?_t=t |title = 张又侠慰问军队驻京老干部服务管理局 |author =  |date = 2008-02-13 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

驻北京市老干部服务管理局下设五个综合服务保障中心。其中，第三综合服务保障中心前身是原中央军委政治工作部玉泉路老干部服务管理局<ref>{{Cite web |url =http://mini.eastday.com/a/180731105028786.html  |title = 复兴路40号社区举办“不忘初心牢记使命城市清洁日志愿服务活动” |author =  |date = 2018-07-31 |publisher =东方头条  |language =  |accessdate =  |quote =  }}</ref>，第四综合服务保障中心前身是原中央军委机关事务管理总局北极寺老干部服务管理局<ref>{{Cite web |url =http://www.bidcenter.com.cn/newscontent-49718610-1.html  |title =住户零星维修工程二次招标公告  |author =  |date =2018-04-28  |publisher =采招网  |language =  |accessdate =  |quote =  }}</ref>，第五综合服务保障中心前身是原中央军委政治工作部白石桥老干部服务管理局<ref>{{Cite web |url = http://epaper.stcn.com/paper/zqsb/html/epaper/index/content_1171279.htm |title = 北京中关村科技发展（控股）股份有限公司重大诉讼进展公告 |author =  |date =2018-06-29  |publisher =证券时报电子版  |language =  |accessdate =  |quote =  }}</ref>。
