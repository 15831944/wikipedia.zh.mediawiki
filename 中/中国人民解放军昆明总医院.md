'''中国人民解放军昆明总医院'''，简称'''昆明总医院'''，常称'''四十三医院'''，位于[[云南省|云南省]][[昆明市|昆明市]][[西山区_(昆明市)|西山区]]大观路212号，隶属[[桂林联勤保障中心|桂林联勤保障中心]]，是[[三级甲等医院|三级甲等医院]]。

==简介==
1949年3月，在[[安徽省|安徽省]][[亳县|亳县]]创建'''中国人民解放军华东野战军第九医院'''。该医院及其前身曾参加[[孟良崮战役|孟良崮战役]]、[[淮海战役|淮海战役]]、[[渡江战役|渡江战役]]<ref name=xiehui/>。1950年4月进驻昆明，先后更名为'''中国人民解放军云南军区军医院'''、'''中国人民解放军西南军区第十八陆军医院'''、'''中国人民解放军第四十三医院'''、'''中国人民解放军昆明军区总医院'''。1985年精简整编为'''中国人民解放军成都军区昆明总医院'''<ref>张步振，成都军区昆明总医院，解放军医院管理杂志1999年05期</ref>。

[[中华人民共和国|中华人民共和国]]时期，医院曾先后参加中缅勘界作战、[[援越抗美|援越抗美]]、[[对越自卫还击战|对越自卫还击战]]，[[唐山大地震|唐山大地震]]、[[1996年丽江地震|丽江地震]]救治工作，以及抗击“[[非典|非典]]”等工作<ref>刘兰辉，以质量建设为根本 实现医院全面建设的新跨越——访成都军区昆明总医院院长赵升阳，中国卫生质量管理2008,15(5):103-105</ref><ref name=xiehui>{{Cite web |url =  http://www.ynha.org.cn/c/2017-04-07/1751449.shtml|title =解放军成都军区昆明总医院全军骨科中心  |author =  |date =2017-04-07   |publisher =  云南省医院协会|language =  |accessdate =  |quote =  }}</ref>。医院本部位于[[云南省|云南省]][[昆明市|昆明市]][[西山区_(昆明市)|西山区]]大观路212号，并在云南省昆明市北教场、云南省[[曲靖市|曲靖市]]各设分院一所<ref name=xiehui/><ref>{{Cite web |url =http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536920.htm  |title =成都军区昆明总医院  |author =  |date =2014-09-12  |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军成都军区|中国人民解放军成都军区]]转隶[[桂林联勤保障中心|桂林联勤保障中心]]，更名为'''中国人民解放军昆明总医院'''<ref>{{Cite web |url = http://news.ifeng.com/a/20170115/50577495_0.shtml |title = 爸爸转隶妈妈赴藏援建 10岁女儿的"独居生活" |author =  |date = 2017-01-15 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.81.cn/jmywyl/2017-04/17/content_7564588.htm |title =  跨越2095公里，一张转诊单的“联勤之旅”|author =  |date = 2017-04-17 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。
