'''中華民國與塞席爾關係'''是指[[中華民國|中華民國]]與[[塞席爾共和國|塞席爾共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對塞席爾的相關事務由'''{{link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''、'''[[駐英國台北代表處|駐英國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-169-69ec7-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
[[File:Voting_res_2758.png|thumb]]，[[塞席爾|塞席爾]]當時屬於[[直轄殖民地|英國殖民地]]，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境塞席爾。]]
[[File:Visa_requirements_for_Seychellois_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|塞席爾簽證政策|{{link-en|塞席爾公民簽證要求|Visa requirements for Seychellois citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可在該國首都的[[塞舌尔国际机场|國際機場]]辦理[[簽證#落地签证|落地簽證]]（入境許可，Visitor's Permit），停留最多30天，可延長至90天。入境應攜帶每日至少150[[美元|美元]]之旅費。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-169-69ec7-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|塞席爾護照|Seychellois passport}}的{{link-en|塞席爾人口|Demographics of Seychelles|塞席爾公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將塞席爾歸類為'''灰色警示'''：提醒注意（2017年6月3日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-328-65fa4-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|塞席爾經濟|Economy of Seychelles}}}}
===貿易===
2017年，塞席爾是中華民國第150大貿易夥伴、第143大[[進口|進口]]夥伴、第135大[[出口贸易|出口]]夥伴。出口至塞席爾的金額為666萬3,206[[美元|美元]]，年減23.975%；自塞席爾進口的金額為234萬1,055美元，年增5.992%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）432萬2,151美元，年減34.071%。※2016年，塞席爾是中華民國第140大貿易夥伴、第143大進口夥伴、第125大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
