'''中華民國與賽普勒斯關係'''是指[[中華民國|中華民國]]與[[賽普勒斯共和國|賽普勒斯共和國]]之間的關係。兩國於1960－1972年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對賽普勒斯的相關事務由'''駐[[希臘|希臘]][[台北经济文化代表处|台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}；對北部[[政權|政治實體]]－[[北賽普勒斯土耳其共和國|北賽普勒斯土耳其共和國]]的事務則由'''{{link-en|駐安卡拉台北經濟文化代表團|Taipei Economic and Cultural Mission in Ankara}}'''兼轄。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-252-ea841-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]一案棄權，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[賽普勒斯|賽普勒斯]]。]]
{{see also|中華民國外交|{{link-en|賽普勒斯外交|Foreign relations of Cyprus}}}}
1960年9月10日，中華民國與賽普勒斯[[外交#外交上的承認|建立外交關係]]。並於首都[[尼古西亞|尼古西亞]]設立'''中華民國駐賽普勒斯[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。<ref name="外交年鑑"/>

1971年12月14日，[[中華人民共和國|中華人民共和國]]與賽普勒斯建交。<ref name="88年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

1972年1月12日，中華民國與賽普勒斯斷交。17日，關閉大使館。<ref name="88年外交年鑑"/>3月10日，改設立[[實質大使館|具大使館性質]]的'''[[臺北經濟文化代表處|台灣商務辦事處]]'''（Taiwan Trade Office）。後因辦事處功能不彰，於1991年5月1日關閉。<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_8.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以「[[歐洲聯盟簽證政策|歐盟免簽證]]」的方式入境賽普勒斯。]]
[[File:Visa_requirements_for_Cypriot_citizens.png|thumb]]的[[賽普勒斯護照|賽普勒斯護照]]之[[塞浦路斯人口|賽普勒斯公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|歐洲聯盟簽證政策|{{link-en|賽普勒斯公民簽證要求|Visa requirements for Cypriot citizens}}}}
[[欧洲联盟成员国|歐盟成員]]的賽普勒斯尚未獲准加入[[申根區|申根區]]，但基於《[[申根公約|申根公約]]》，該國實行與申根區相同的簽證政策。<ref>{{Cite web |url = http://www.mfa.gov.cy/mfa/mfa2006.nsf/consular02_en/consular02_en?OpenDocument |title = 國外訪客旅遊資訊 |publisher = {{link-en|賽普勒斯外交部|List of Ministers of Foreign Affairs of Cyprus}} |deadurl = yes |archiveurl = https://web.archive.org/web/20140323194207/http://www.mfa.gov.cy/mfa/mfa2006.nsf/consular02_en/consular02_en?OpenDocument |archivedate = 2014-03-23 }}</ref>歐洲申根區給予持有載明[[中華民國國民身份證|身份證]]字號的[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以[[歐洲聯盟簽證政策|歐盟免簽證]]入境，停留日數單獨計算，不與申根區合併，每6個月期間內總計可停留90天。[[北賽普勒斯土耳其共和國|北賽普勒斯土耳其共和國]]也給予中華民國公民免簽證入境，可停留90天。有鑒於南北賽（[[賽普勒斯共和國|賽普勒斯共和國]]與北賽普勒斯土耳其共和國）仍有政治爭端，宜避免入境北賽或往來南北賽之間。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-252-ea841-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_FAQ.aspx?c=AD6908DFDDB62656&n=C1B19EF0ACDD3C91&sms=867F5398A49F758A |title = 歐盟免申根簽證常見問答集 |publisher = 中華民國外交部}}</ref>

[[塞浦路斯人口|賽普勒斯公民]]持有[[歐盟護照|歐盟護照]]者也可以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將賽普勒斯歸類為'''灰色警示'''：提醒注意（2017年5月30日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-401-b74e1-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|賽普勒斯經濟}}
===貿易===
2018年，賽普勒斯是中華民國第119大貿易夥伴、第127大[[進口|進口]]夥伴、第106大[[出口贸易|出口]]夥伴。出口至賽普勒斯的金額為2,519萬4,204[[美元|美元]]，年增18.833%；自賽普勒斯進口的金額為490萬8,611美元，年增4.146%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）2,028萬5,593美元，年增23.032%。2017年，賽普勒斯是中華民國第121大貿易夥伴、第125大進口夥伴、第108大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
