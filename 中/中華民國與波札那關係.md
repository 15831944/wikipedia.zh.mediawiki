'''中華民國與波札那關係'''是指[[中華民國|中華民國]]與[[波札那共和國|波札那共和國]]之間的關係。兩國於1966－1974年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對波札那的相關事務由'''{{link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-161-2cd9a-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[波札那|波札那]]。]]

==政治==
===外交===
{{see also|中華民國外交|{{link-en|波札那外交|Foreign relations of Botswana}}}}
1966年12月30日，建立大使級[[外交#外交上的承認|外交關係]]。於首都[[嘉柏隆里|嘉柏隆里]]設立'''中華民國駐[[波札那共和國|波札那共和國]][[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1968年9月10日，簽署《中華民國與波札那共和國技術合作協定》，於1972年4月28日續簽。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e6%b3%a2%e6%9c%ad%e9%82%a3&tycountry_e=%e6%b3%a2%e6%9c%ad%e9%82%a3&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

1974年4月5日，與中華民國斷交、關閉大使館，至今邦交未曾恢復。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境波札那。]]
[[File:Visa_requirements_for_Botswana_citizens.png|thumb]]的{{link-en|波札那人口|Demographics of Botswana|波札那公民}}必須申請簽證方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|波札那簽證政策|波札那公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]申請簽證可詢問波札那駐[[南非|南非]][[高級專員|高級專員公署]]（[[外交代表機構#大使馆|大使館]]），或{{link-en|波札那駐外機構列表|List of diplomatic missions of Botswana|波札那駐外館處}}。經由[[南非機場列表|南非國際機場]]轉機入出境波札那，不需申辦南非[[過境簽證|過境簽證]]；若出機場再轉搭陸上交通工具前往波札那，因已實際入境南非，故須申辦[[南非簽證政策|南非簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-161-2cd9a-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持[[波札那護照|波札那護照]]的{{link-en|波札那人口|Demographics of Botswana|波札那公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將波札那歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2018年11月1日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-145-3d076-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|波札那經濟}}
===貿易===
2017年，波札那是中華民國第167大貿易夥伴、第165大[[進口|進口]]夥伴、第150大[[出口贸易|出口]]夥伴。出口至波札那的金額為356萬6,197[[美元|美元]]，年減20.482%；自波札那進口的金額為62萬8,539美元，年增1,528.930%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）293萬7,658美元，年減33.929%。※2016年，波札那是中華民國第166大貿易夥伴、第197大進口夥伴、第145大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
