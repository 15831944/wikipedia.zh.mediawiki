'''中華紙漿股份有限公司'''，簡稱'''華紙'''，於1968年7月遵奉中央政府第四期經建計劃，以「改良林相、繁榮東部暨發展台灣造紙工業」為目標，成立於花蓮縣吉安鄉。2012年與永豐餘紙與紙板事業部合併，完成林漿紙一貫垂直整合生產，並轉型走向高附加價值的特殊紙開發。此外，華紙也對台灣環境生態造成極大汙染疑議。據環保署資料，華紙花蓮廠自2015年迄今，因空污、水污遭罰多達30次。環保署水保處長葉俊宏表示，會請花蓮縣環保局慎重考慮以重大情節開罰，處以最高罰款2000萬元，且依《水污染防治法》73條，1年內經2次限期改善，仍違反規定，華紙應勒令停工。

中華紙漿是一家以[[紙#紙漿|紙漿]]及[[紙|紙品]]加工製造為主要業務的[[企業|企業]]，主要產品有短纖闊葉樹紙漿、文化用紙、紙板與特殊紙。目前於大中華區共有四座生產基地，包含花蓮廠、久堂廠、台東廠與廣東鼎豐廠。其母公司為[[永豐餘投資控股股份有限公司|永豐餘投資控股股份有限公司]]。

== 沿革概略 ==
1968年7月5日，以[[新臺幣|新臺幣]]2億元之[[股本|股本]]成立。

1968年10月30日，花蓮廠破土動工，並於1969年底開始[[試車|試車]]。

1970年4月4日，花蓮廠正式開工。

1974年10月15日，協助[[象牙海岸|象牙海岸共和國]]以該國木材在花蓮廠製造漂白紙漿。

1975年2月7日，公司已發行的[[股票|股票]]經[[臺灣證券交易所|臺灣證券交易所]]批准後，在交易所[[股票上市|公開掛牌交易]]。

1976年12月7日，轉投資成立'''印尼永吉紙業公司'''。

1980年12月31日，年產量12萬噸之紙漿廠試車完成。

1982年3月31日，出售年產量10萬噸漂白紙漿之工廠設備予印尼永吉紙業公司。

1990年8月18日，轉投資成立[[大陸|大陸]][[廣東省|廣東省]]'''廣東鼎豐紙業有限公司'''。

2006年4月1日，轉投資成立廣東省'''肇慶鼎豐林業有限公司'''。

2000年9月8日，公司股票全面轉換為[[無實體發行|無實體發行]]。

2008年12月8日，轉投資成立'''永豐能源科技股份有限公司'''。<ref>[http://www.chp.com.tw/milestone/milestone.html 華紙紀事]</ref>

2012年10月1日，與永豐餘紙與紙板事業部完成合併，落實林漿紙一貫整合。

2014年，花蓮廠成立「'''氫能源示範基地'''」。

2016年，中華紙漿獲頒「'''亞洲企業社會責任獎-綠色領導獎'''」。

2018年12月18日，媒體報導「'''中華紙漿污水危生態 超標46倍 出海口慘變「陰陽河」'''」

== 產品簡介 ==
中華紙漿依產品別，共分為五大事業群：特殊紙事業群、紙漿事業群、紙與紙板事業群、特化材料事業群，及農林事業群。產品內容如下：
# 漂白闊葉樹紙漿(LBKP)：為造紙主要原料。
# 銅版紙類(Art Paper)：海報、月曆、雜誌、書冊、型錄、雜誌等。
# 道林、模造紙類(Woodfree)：教科書、叢書、產品說明書、工商日誌、萬用手冊內頁、簿冊、雜誌及宣傳單等。
# 紙板類：應用於撲克牌、吊牌、卡片、書籍之封面及化妝品、電子組零件、鞋盒、糕點、餅乾等包裝盒用。
# 環保認證紙類(FSC)。
# 其他特殊用途紙類：各等級紙管紙、標籤及膠帶用產品系列、各產業包裝用紙如醫療用紙、食品包裝用紙、光學襯紙及裝潢建材基紙系列產品。
# 農林產品：肥料與土壤改質劑。
# 特化材料：主要生產銷售液鹼、鹽酸、亞氯酸鈉、次氯酸鈉等。

== 產能狀況與生產能力 ==
* 大中華地區生產基地：
台灣地區生產基地包含花蓮廠(漿紙一貫廠)、久堂廠(特殊紙廠)及台東廠(再生環保紙廠)，紙漿年產能為25萬噸，文化用紙年產能為60萬噸，特殊用紙年產能為10萬噸。其中紙漿90%用於製造文化用紙、10%外銷。
 
中國大陸生產基地為廣東鼎豐廠(林漿紙一貫廠)，紙漿年產能為12萬噸，家庭用紙年產能為5萬噸。
* 近期重要投資：
公司投資台灣久堂廠200萬美金進行設備轉換，將4,000噸文化用紙產能轉移成2,500噸膠帶用紙產能，2015年11月已完成設備轉換，2016年1月已生產銷售。

公司為配合永豐餘衛生紙部門於中國大陸華南市場之布局，於中國大陸鼎豐廠購置2台新衛生紙機，此添購可為目前年產能增加5萬噸，已於2016年7月投產。
