'''中国人民解放军第九四医院'''，位于[[江西省|江西省]][[南昌市|南昌市]]井冈山大道1028号，隶属[[无锡联勤保障中心|无锡联勤保障中心]]，为[[三级甲等医院|三级甲等医院]]。

== 简介 ==
第九四医院是综合性[[三级甲等医院|三级甲等医院]]，是[[南方医科大学|南方医科大学]]等多所院校的教学医院，是[[中国人民解放军第四军医大学|中国人民解放军第四军医大学]][[西京医院|西京医院]]全军心血管外科研究所的临床基地<ref name=mod>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536723.htm |title = 第九四医院 |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。2012年10月18日，南京军区第九四医院举行“'''南昌大学附属长城医院'''”揭牌仪式暨门急诊、重症监护中心大楼改扩建竣工庆典活动，正式揭牌成为[[南昌大学|南昌大学]]非直属附属医院，同时启用第二名称“南昌大学附属长城医院”<ref>{{Cite web |url = http://www.94ebh.com/yygk/yyxw/2012830.html |title = 南京军区第94医院举行南昌大学附属长城医院揭牌仪式 |author =  |date =  |publisher = 南京医科大学邦德骨科医院 |language =  |accessdate = 2017-06-26 |quote =  }}</ref>。  
2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第九四医院由[[中国人民解放军南京军区|中国人民解放军南京军区]]联勤部转隶[[无锡联勤保障中心|无锡联勤保障中心]]<ref>{{Cite web |url =http://www.sohu.com/a/124841226_115402|title =中国第二十批援助赞比亚军医组出征 |author =  |date = 2017-01-20  |publisher =搜狐|language =  |accessdate =  |quote =  }}</ref>。
== 历史 ==
1946年9月成立'''[[华中野战军|华中野战军]]第十纵队医疗队'''。1947年2月，改为'''[[华东野战军|华东野战军]]第十二纵队医疗队'''。1948年5月，在[[江苏省|江苏省]][[金湖县|金湖县]]改为'''江淮军区后方医院'''。1949年4月改称'''皖北军区后方医院'''，先后驻[[安徽省|安徽省]][[肥东县|肥东县]]、[[肥西县|肥西县]]。1950年12月改称'''华东军区第十四野战医院'''，隶属华东军区卫生部，驻安徽省[[明光市|明光县]]。1951年3月改称'''华东军区第十四陆军医院'''，驻[[山东省|山东省]][[兖州市|兖州市]]。医院自成立后，先后参加[[苏中七战七捷|苏中七战七捷]]、[[淮海战役|淮海战役]]、[[渡江战役|渡江战役]]、[[炮击金门|炮击金门]]、[[东山保卫战|东山保卫战]]等战役，接收[[抗美援朝战争|抗美援朝战争]]的[[中国人民志愿军|中国人民志愿军]]伤病员等。1953年2月，医院迁址[[福建省|福建省]][[福州市|福州市]]，转隶[[中国人民解放军福建省军区|福建军区]]卫生部。1954年4月，医院迁址福建[[漳州|漳州]]，同年10月正式命名为'''中国人民解放军第九四医院'''。1958年10月转隶后勤第五分部。1961年11月转隶后勤第十八分部。1970年9月，奉军委命令与[[中国人民解放军第一七五医院|中国人民解放军第一七五医院]]对调，从福建漳州移防江西南昌，转隶[[中国人民解放军江西省军区|江西省军区]]<ref>{{Cite web |url = http://www.94yy.cn/yygk/yyls/197.html |title = 医院变迁 |author =  |date = 2014-04-02 |publisher = 中国人民解放军第九四医院 |language =  |accessdate =  |quote =  }}</ref>。
