'''中國威脅論'''，是闡述[[中華人民共和國|中華人民共和國]]的軍事霸權及[[威權主義|威權主義]]對世界各國的[[民主|民主]]、[[和平|和平]]、[[軍事|軍事]]、[[經濟|經濟]]等各方面造成嚴重威脅的論點。該論點一般認為起源於西方國家及日本<ref>[http://www.roc-taiwan.org/ct.asp?xItem=45458&ctNode=3591&mp=202 中国の脅威は台日が正視すべき共通課題] {{webarchive|url=https://web.archive.org/web/20150225133520/http://www.roc-taiwan.org/ct.asp?xItem=45458&ctNode=3591&mp=202 |date=2015-02-25 }}</ref>。

== 原因 ==
{{Unreferenced section|time=2018-10-10 T14:26:23+00:00}}
[[中国共产党|中国共产党]]高级党政干部，著作编辑副主任[[蒯辙元|蒯辙元]]提出，自中华人民共和国实行[[改革开放|改革开放]]以后中国在经济、政治、军事等综合实力方面的崛起，从[[日本|日本]]、[[韩国|韩国]]、[[美国|美国]]等邻国和世界大国便不斷传来所谓的“中国威胁论”，表示出外国人对于中国大陸的崛起会影响和威胁到本国的利益，试图通过这种舆论减缓中国的发展速度或在国际舞台中孤立中国<ref>{{cite web|author =蒯辙元|title =如何化解世界对中国崛起的恐惧|url =http://blog.ifeng.com/article/9924914.html|deadurl =yes|archiveurl =https://web.archive.org/web/20160305070420/http://blog.ifeng.com/article/9924914.html|archivedate =2016-03-05}}</ref>。蒯指出，外国提出“中国威胁论”的原因不外乎有三：

#崛起后的中国大陸近期的大国心态调整、战略调整有些过急、过激。这给国际社会，特别是给西方发达国家和中国周边国家以硬崛起而非软崛起的负面形象，因而产生了负面效应，造成负面影响，从而使之对中国大陸的强大、强硬担忧，甚至出现恐慌。近日，法国战略研究基金会分析家瓦莱里·尼凯强调：“近一两年来，发达国家意识到，中国大陸的形象不再像从前那样正面”。法国《回声报》2月3日载文指出：“每周都会出现中国的某个举动让我们的政治经济界人士恐慌的情况”。
#虽然新时期崛起的中国其自身的历史文明、现实文明，发展文明，在本质和理念上都与历史上崛起的帝国例如英德美日帝国有着根本的区别。但是中国大陸对自身先进的文明理念、发展理念、崛起理念，在文宣上严重滞后，在传播上严重落后，而且作为世界第二大经济体的话语权缺位，运用更是乏力。因而面对“中国威胁论”“中国恐惧症”，从理论到实际只是被动应对，破解无方，化解无力。

{{fact|针对上述原因，蒯辙元认为，从中共官方到民间都应深切反思，全面检讨，无论理论还是实践，无论战略还是策略，无论心态还是行为，都应做审时度势的调整，采取积极、主动，冷静、理性，且富于创意性的战略和战术行动来逐渐化解对中国大陸崛起的恐惧。}}

2017年初，英國媒體《星期天泰晤士報》報導日本出資渲染中國威脅論，被視為試圖壓制中國發展的一種外交手段。<ref>{{cite web |author = BBC |title = 英媒：日本資助英國智庫展開反華公關宣傳 |url = http://www.bbc.com/zhongwen/trad/press-review-38788712}}</ref>

== 历史 ==
{{Unreferenced section|time=2018-04-29T14:26:23+00:00}}
{{Seealso|反华|反中}}
“中国威胁论”在欧洲由来已久。在中華人民共和国[[改革开放|改革开放]]初期，很多人并不看好中国大陸，甚至有人预言中国大陸可能“崩溃”，《明镜》周刊曾对此表达了“恐惧”：“如果中国人不能解决温饱问题，全世界人都将挨饿。”早期的“中国威胁论”主要是文化意义上的。由于[[欧洲|欧洲]]在[[宗教|宗教]]、[[民主|民主]]、[[人权|人权]]等诸多价值關上與共产主义中國存在重大分歧，共产主义很难真正获得欧洲国家的好感。{{cn}}

[[19世纪|19世纪]]后期的“[[排华浪潮|排华浪潮]]”。当时[[美国|美国]][[白人种族主义者|白人种族主义者]]和劳工利益集团将在美国的[[华工|华工]]视为美国主流文化的的“威胁”，于是推动政府分别于1882年和1884年通过了《[[排华法案|排华法案]]》。这一时期的“中国威胁论”特指华人移民，不是现代意义上的“中国威胁论”，但其中体现的白人至上主义思想和东西方文明的冲突，则在其后的“中国威胁论”中仍能找到踪影。{{cn}}

中共建政之初，美国也曾兴起过一阵“中国威胁论”，即中国革命的胜利有可能在东南亚引起[[多米诺骨牌效应|多米诺骨牌效应]]，从而对美国形成“红色威胁”。这一时期的“中国威胁论”出现在美苏冷战的大背景下。{{cn}}

从历史上看，“中国威胁论”的版本很多，但共同的特点就是，他们的倡导者是反對[[中国共产党|中国共产党]]治理下的[[中华人民共和国|中华人民共和国]]。不過「反共」跟「反華」的觀點容或有部分重疊，但應就事論事，不應混為唯一談，中共傳統上特意不區分兩者，將所有對其體制或行為的批評，通通成了「反華」，「敵對勢力」等，從而迴避真正的問題。

== 概念 ==
中國威脅論是國際關係上因中國大陸崛起而產生憂慮與質疑的論述，[[中國崛起|中國崛起]]後，隨著中國大陸迅速發展，遂导致其周邊國家的利益和國際秩序備受挑戰，威脅包括[[經濟|經濟]]、[[軍事|軍事]]、[[糧食|糧食]]、[[人口|人口]]以至[[太空|太空]]等領域。有論者指中國大陸專政體制是產生威脅論的主因。中共官方則認為威脅論源於後冷戰時代西方國家用以壓制中國發展的說法。<ref>{{cite web  |title = 中國威脅論 |url = http://www.liberalstudies.hk/daily_concepts/?word=499}}</ref>

== 评价 ==

=== 态度 ===
時至今日，在不同的地區和國家在冷戰後，面對中國大陸所形成的威脅感有不同程度的加強或減弱，但“中国威胁论”并未直接的态度，有时甚至不明所以。

《[[中華民國憲法|中華民國憲法]]》規定，[[中華民國|中華民國]]的国家[[主權|主權]]及於全中國。因此，此處之「中國威脅論」指的是[[中共政權|中共政權]]的威脅。
	 
[[中華民國政府|中華民國政府]]於1987年解除[[台灣省戒嚴令|戒嚴]]後，隨著開放探親等措施，[[兩岸關係|兩岸關係]]日趨緩和；但是，首先是中國大陆[[六四天安門事件|六四天安門事件]]，之後的[[千島湖事件|千島湖事件]]等都讓台灣民眾普遍對中國大陆的觀感不佳；而後[[台灣海峽飛彈危機|95-96台海飛彈危機]]乃中共於1995年7月以飛彈試射威脅台灣，對後冷戰裁減軍備的國際社會是一大警訊，不但兩岸關係日趨緊張，而世界各國也出現圍堵中共的主張。

[[六四事件|六四事件]]后，据称，中国大陆當局开始推动民族主义和威权主义政治宣传以鞏固政權，相继推广《环球时报》等民族主义媒体，对台湾进行武力威胁，同时快速扩充军费，窃取各国军事、经济机密，对争议中的海域宣示主权并加强军事存在。外部观察家逐步提出中国大陆是一种军事政治威胁的观点。<ref>{{cite web|title=陈岳：“中国威胁论”与中国和平崛起|url=http://www.21ccom.net/articles/world/zlwj/20150508124457_all.html|date=2015-05-08|publisher=[[共识网|共识网]]|archiveurl=https://web.archive.org/web/20160813085245/http://www.21ccom.net/articles/world/zlwj/20150508124457_all.html|archivedate=2016-08-13|deadurl=yes}}</ref>
	
近年來，儘管台灣歷經對[[中華民國憲法|中華民國憲法]]的修訂，及1991年已終止[[動員戡亂臨時條款|動員戡亂]]，中華民國政府已不再視中國共產黨為叛亂敵人，但中共經常以中華民國政府特别是中国国民党無法接受的方式談論關於對日抗戰<ref>{{subst:fact/auto|中共不承認國民黨主導並領導抗日的歷史事實。}}</ref>、國共內戰<ref>中共在勝利后成立中華人民共和國，稱已取代中華民國成為全中國的政權，拒絕承認在台灣的中華民國政府。</ref>、不承認中華民國政府、[[台灣民主化|台灣民主化]]<ref>中國國民黨在台灣執政時已解除戒嚴和黨禁，並實現總統民選。</ref>、[[台灣本土化運動|台灣本土化運動]]及[[台灣獨立運動|台灣獨立運動]]的看法與立場，並不斷以武力展示恫嚇並在外交上打壓台灣。台湾媒体宣传及民众的看法认为，許多接受中國大陸國民教育的中國大陸民眾只會複述中共的觀點，使得頗多台灣人認為中國大陸民眾深受中共洗腦。
	
與台灣的衝突，特別是1995年至1996年在[[1996年中華民國總統選舉|第一次中華民國總統公民直接選舉]]前的飛彈試射，那時剛經過[[六四事件|六四事件]]、[[千島湖事件|千島湖事件]]的台灣，具象化中国共产党威脅論的實在性，使大陆的威脅論迄今仍影響[[台灣政治|台灣政治]]。[[中國共產黨|中國共產黨]]仍堅持[[一黨執政|一黨執政]]，有观点认为[[中華人民共和國人權狀況|中華人民共和國人權狀況]]在[[北京奧運會|北京奧運會]]後繼續惡化、以及不放棄以武力解決台灣問題的立場，並部署約兩千枚對準台灣的[[飛彈|飛彈]]<ref>[http://www.libertytimes.com.tw/2011/new/may/20/today-p12.htm 我軍方研判 中對台導彈 明年達1800枚] {{webarchive|url=https://web.archive.org/web/20110523070116/http://www.libertytimes.com.tw/2011/new/may/20/today-p12.htm |date=2011-05-23 }}, 自由電子報, 2011-5-20</ref><ref>[http://www.libertytimes.com.tw/2010/new/jul/18/today-t2.htm# 國防部：中國對台飛彈年底達1960枚] {{webarchive|url=https://web.archive.org/web/20120219203446/http://www.libertytimes.com.tw/2010/new/jul/18/today-t2.htm |date=2012-02-19 }}, 自由電子報, 2010-7-18</ref>，這些都讓中華民國政府和許多台灣人民對中國共產黨及中國大陸崛起厭惡、警覺及強烈反感。

原[[立法院院長|立法院院長]][[王金平|王金平]]表示，台灣面對中國大陸軍事崛起的威脅，應更積極參與國際組織活動、強化與美日同盟、持續建立自我防衛的力量，以維護台海和平與亞太區域安全。

[[中華民國總統|中華民國總統]][[蔡英文|蔡英文]]認為「兩岸關係的改善與穩定，主要取決於中國大陸是否走向民主，[[中國民主化|中國民主化]]一直影響了兩岸的進程跟台海關係穩定。民主與人權是台灣最重要的資產，台灣更應積極的在中國大陸民主進程上，助其一臂之力。只有中國大陸民主化，雙方才可能在相同制度的基礎上，穩定的互惠往來。」<ref name="democraticalliance">[http://www.epochtimes.com/b5/11/9/8/n3366474.htm%E9%98%B2%E4%B8%AD%E5%9C%8B%E8%BB%8D%E4%BA%8B%E5%A8%81%E8%84%85-%E5%8F%B0%E6%87%89%E5%BC%B7%E5%8C%96%E7%BE%8E%E6%97%A5%E5%90%8C%E7%9B%9F 防中國軍事威脅 台應強化美日同盟]，大紀元，2011年09月08日</ref>

==== {{ASEAN}} ====
[[新加坡|新加坡]]在1995年5月13日前往北京某國際企業年會演說時，就道出國際的矛盾心態—一方面視中國為商機，另一方面又是威脅。經媒體的報導後，「中國威脅論」更廣為流傳，特別在當時1995年2月中國和[[菲律賓|菲律賓]]（與新加坡同為[[東南亞國協|東南亞國協]]會員）的背景下，當時的[[中華人民共和國國務院副總理|中華人民共和國國務院副總理]]兼外長[[錢其琛|錢其琛]]正式駁斥該「中國威脅論」的說法，指「中國永不威脅或侵略別國」。

自2010年起，東南亞多國與中國在[[南海|南海]]的領土爭端升高，更促使這些國家合作對抗軍力不斷增長的中國。日本、韓國已及東盟各國對中共紛紛採取更加防範的態度，並重新鞏固與美國的軍事同盟關係，尋求保護。美國也順勢重新加強了與這些亞洲盟友的軍事互動與同盟關係。<ref>[http://ap.ntdtv.com/b5/20111023/video/78041.html 美國政策重點轉向亞太 直指中共]，[[新唐人電視台|新唐人電視台]]，2011年10月23日</ref>

==== {{USA}} ====
在後冷戰的背景下，1991年12月13日《[[紐約時報|紐約時報]]》專欄作家Leslie H. Gelb撰寫題為「Breaking China Apart」（分裂中國）的文章；後有Charles Krauthammer在1995年7月31日的《[[時代週刊|時代週刊]]》（Vol.146，Iss. 5）撰文「Why we must contain China」（為何我們必須圍堵中國）。[[中國人民解放軍|中國人民解放軍]]少將[[朱成虎|朱成虎]]的對美核武言論曾引起很大的爭議。

中國威脅論在[[新保守主義|新保守主義]]很流行。他們的想法是，美國有特別責任領導別的國家到一個民主的將來，也覺得[[美國|美國]]必須當世界唯一的超權國家。但是在整個的美國保守運動，中國威脅論是有爭議。美國“親[[商業|商業]]保守派”反對這個思潮，認為[[貿易|貿易]]和[[文化|文化]]接觸會對美國和[[中華人民共和國|中華人民共和國]]雙方有利。

[[习近平|习近平]]在2012年上任[[中国共产党中央委员会总书记|中共中央总书记]]后，由于中国放弃了“韬光养晦”的策略，推行“[[一带一路|一带一路]]”政策，加上在各领域的全面崛起及其带来的威胁日益增大，美国共和黨的保守派部分亲中派（如[[沈大伟|沈大伟]]）转变了看法，遂建议美国政府逐渐放弃对华的接触政策，转而采取强硬策略。共和黨和民主黨雖然在國內的內政有嚴重分歧和衝突，但皆認為中國威脅美國的軍事、經濟及知識產權。參看[[2018年美中貿易戰|2018年美中貿易戰]]。

===={{JPN}}====
2005年12月22日，[[日本外相|日本外相]][[麻生太郎|麻生太郎]]發言「中國正成為威脅」，這是[[小泉內閣|小泉內閣]]首次有閣員在公開場合提出「中國威脅論（{{lang|ja|中國脅威論}}）」。[[安倍晉三|安倍晉三]]亦表示，過去20年來，中國軍費快速增長，目前已達約20倍之多，特別是最近中國海軍力量的增長更為顯著；想要以力服人的行為，不只是增加巨額的財政負擔，也會使其失去區域各國的信賴。

2017年1月29日，英国《[[星期日泰晤士报|星期日泰晤士报]]》披露，[[日本驻英国大使馆|日本驻英国大使馆]]每月支付英国右翼智库{{link-en|亨利·杰克逊协会|Henry Jackson Society|亨利·杰克逊协会}}1万英镑，一些与亨利·杰克逊协会过从甚密的[[英国国会议员|英国国会议员]]曾被日本政府出资邀请访问日本，以帮助日本在英国制造及渲染中国威胁论；對此，[[今日俄罗斯|今日俄罗斯]]、《[[印度快报|印度快报]]》等媒体跟进报道，[[日本媒体|日本媒体]]一致保持沉默，英美主流媒体大多没有跟进报道<ref>{{cite news|url=http://www.globalview.cn/html/global/info_16102.html|title=揭秘被日本收买的英国反华智库|publisher=[[国际先驱导报|国际先驱导报]]|author=桂涛、冯武勇|date=2017-02-09|accessdate=2018-07-01|quote=}}</ref>。[[共同通訊社|共同通訊社]]客座论说委员冈田充说，很多日本记者私下感叹“不在稿子里加入批判中国的评论，稿子就通不过”，这不是上司的命令、而是记者的[[自我审查|自我审查]]，“中国威胁论在日本的言论空间中已经常态化”<ref>{{cite news|url=http://www.globalview.cn/html/societies/info_16109.html|title=日本记者：不加入批判中国的评论 稿子就通不过|publisher=环球视野|author=凤凰新闻|date=2017-02-09|accessdate=2018-07-01|quote=}}</ref>。

==== {{IND}} ====
中印軍事關係的緊張主要在於对双方争议边界问题，以及對[[巴基斯坦|巴基斯坦]]的軍事援助。1998年5月，[[印度|印度]]國防部長[[喬治·費爾南德斯|喬治·費爾南德斯]]在印度進行[[核試驗|核試驗]]後，宣揚「[[中國|中國]]是印度潛在的頭號威脅」的言論。及後，費爾南德斯本人收回這種說法，表示要和中國友好，但印度军事部署的加强，被各界军事家认为是针对中国和巴基斯坦。

===={{USSR}}/{{RUS}}====
1960年代初，中苏闹僵后，在苏联和世界上就存在着关于中国威胁的论调。[[法国总理|法国总理]][[达拉第|达拉第]]在会见[[赫鲁晓夫|赫鲁晓夫]]的时候便提出了“[[黄祸论|黄祸论]]”的问题。<ref>尼·赫鲁晓夫：《最后的遗言》</ref>而在1969年[[珍宝岛事件|珍宝岛事件]]之后，中苏陷入高度紧张状况，又由于中国继而实施了动员行为（“[[一号战备令|一号战备令]]”），而苏联[[西伯利亚|西伯利亚]]地区相当地广人稀、难以防御，整个苏联国内泛起了对中国军事威胁的恐惧。在[[持不同政见者|持不同政见者]]的苏联知识分子群体中，这种中国威胁论思想也非常普遍，如历史学家[[罗伊·麦德维杰夫|罗伊·麦德维杰夫]]写道：

{{cquote|斯大林在1941年没有充分考虑到希特勒的冒险主义。（……）他认为，希特勒会从分析实际的因素出发，而不是从幻想出发。就是在今天，特别是在苏中关系的现状下，我们更不应该忘记1941年的教训；因为毛泽东也不是从分析实际因素行事的，而是从他自己对实际情况有时候极其荒诞的想法行事的。因此，我们应该考虑到，来自毛泽东集团的毫无理智的冒险行为。|罗·麦德维杰夫：《让历史来审判：斯大林主义的起源及其后果》下册，第772页}}
