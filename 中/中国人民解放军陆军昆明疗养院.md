'''中国人民解放军陆军昆明疗养院'''，位于[[云南省|云南省]][[安宁市|安宁市]][[温泉镇_(安宁市)|温泉镇]]，隶属[[中国人民解放军陆军后勤部|中国人民解放军陆军后勤部]]。

==简介==
昆明疗养院于1950年10月建院，是[[中国人民解放军|中国人民解放军]]首批组建的疗养院，位于[[昆明市|昆明市]][[滇池|滇池]]以西40公里的安宁市温泉镇，占地面积163亩，海拔1795米。该院依山傍水，环境清幽，设施完善，曾被评为全军绿化先进单位，截至2016年曾经四次获得全军医院、疗养院优质服务“白求恩杯”。1960年代至1970年代，是[[党和国家领导人|党和国家领导人]]、[[中国共产党中央军事委员会|中共中央军委]]首长的休养地，也是中国共产党和国家、军队的政治外交场所。[[刘少奇|刘少奇]]、[[周恩来|周恩来]]、[[朱德|朱德]]、[[董必武|董必武]]、[[李先念|李先念]]、[[宋庆龄|宋庆龄]]、[[贺龙|贺龙]]、[[邓小平|邓小平]]、[[胡耀邦|胡耀邦]]等[[党和国家领导人|党和国家领导人]]以及[[老挝|老挝]]、[[越南|越南]]、[[缅甸|缅甸]]、[[阿尔巴尼亚|阿尔巴尼亚]]等国领导人及军政高级人员曾来院视察疗养。2004年和2013年，昆明疗养院先后接待[[杨利伟|杨利伟]]、[[聂海胜|聂海胜]]、[[费俊龙|费俊龙]]以及[[中国航天员大队|中国航天员大队]]的多位航天员，并且刻碑植树合影<ref name=mod>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536910.htm |title =成都军区昆明疗养院  |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref><ref name=crt>{{Cite web |url =http://www.crt.com.cn/news2007/News/tgjx/16615117359F6379CAB5EID75FD0FE.html  |title =李锦华：解放军昆明疗养院  |author =  |date =2016-06-15   |publisher =  中红网|language =  |accessdate =  |quote =  }}</ref>

2016年改革前，该疗养院名称是“'''中国人民解放军成都军区昆明疗养院'''”，隶属[[中国人民解放军成都军区|中国人民解放军成都军区]]<ref name=mod/>。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，成都军区撤销，该疗养院转隶[[中国人民解放军陆军后勤部|中国人民解放军陆军后勤部]]，名称改为“'''中国人民解放军陆军昆明疗养院'''”<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1689524  |title = 陆军驻新疆、西藏边远艰苦地区部队试点团职干部带配偶疗养 |author =  |date = 2017-05-19 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
