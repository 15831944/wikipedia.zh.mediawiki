'''中華民國與羅馬尼亞關係'''是指[[中華民國|中華民國]]與[[羅馬尼亞|羅馬尼亞]]之間的關係。兩國於1939－1941年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對羅馬尼亞的相關事務由'''駐[[斯洛伐克|斯洛伐克]][[台北经济文化代表处|台北代表處]]'''、'''駐[[匈牙利|匈牙利]]台北代表處'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-243-dff1b-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[羅馬尼亞社會主義共和國|羅馬尼亞社會主義共和國]]。]]
{{see also|中華民國外交|{{link-en|羅馬尼亞外交|Foreign relations of Romania}}}}
{{see also|中国驻罗马尼亚大使列表{{!}}中華民國駐羅馬尼亞外交官列表|罗马尼亚驻华大使列表{{!}}羅馬尼亞駐中華民國外交官列表}}
1939年7月3日，兩國建立公使級[[外交#外交上的承認|外交關係]]。於首都[[布加勒斯特|布加勒斯特]]設立'''中華民國駐[[羅馬尼亞王國|羅馬尼亞王國]][[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]

1941年7月10日，羅馬尼亞承認[[汪精衛政權|汪精衛政權]]及[[滿州國|滿州國]]，與中華民國斷交。7月15日，關閉駐羅馬尼亞公使館。至今邦交未曾恢復。

1950－1990年代東西方陣營的「[[冷戰|冷戰]]」期間，中華民國在[[意識形態|意識形態]]和[[政體列表|制度]]上均偏向以[[美國|美國]]為首的[[資本主義|資本主義]]陣營，復加上以[[蘇聯|蘇聯]]為首的[[社會主義國家|共產集團國家]]的封閉，導致從[[臺灣戰後時期|中華民國政府遷台]]後幾乎沒有官方往來。

1971年10月25日，身為共產陣營成員及蘇聯[[衛星國|衛星國]]之一的羅馬尼亞支持[[中華人民共和國|中華人民共和國]]取得[[聯合國|聯合國]]的「[[中國與聯合國#代表權問題|中國席位]]」（中國代表權問題）。

1989年的「[[東歐民主化|東歐劇變]]」使羅馬尼亞[[民主化|轉型為民主國家]]並重新對外開放，雙方始恢復往來。

1991年4月，羅馬尼亞外貿協會主席柯拉斯（Mircea Nicolae Coras）開始與台灣接觸。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-4.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1993年11月，駐匈牙利臺北商務辦事處代表[[魏武煉|魏武煉]]訪問羅馬尼亞。<ref name="86年外交年鑑"/>

1995年6月，中華民國經濟部長[[江丙坤|江丙坤]]率團赴[[瑞士|瑞士]]參加蒙太納[[論壇|論壇]]，期間會見羅馬尼亞副總理兼經濟改革政策協調會主席{{link-en|柯希亞|Mircea Coșea}}。<ref name="86年外交年鑑"/>

1995年8月，{{link-en|羅馬尼亞民主會議|Romanian Democratic Convention}}主席、[[布加勒斯特大學|布加勒斯特大學]]校長[[埃米爾·康斯坦丁奈斯科|康斯坦丁奈斯科]]抵台訪問。（於1996年11月當選[[羅馬尼亞總統|總統]]）<ref name="86年外交年鑑"/>

1996年5月，[[羅馬尼亞參議院|羅馬尼亞參議院]]經濟委員會主席{{link-ro|沙拉基恩|Viorel Sălăgean}}抵台參加[[1996年中華民國總統選舉|中華民國第九任總統副總統]]就職典禮。<ref name="86年外交年鑑"/>

1997年5月5日，簽署《中羅（馬尼亞）國際[[特快专递|快捷郵件]]服務瞭解[[備忘錄|備忘錄]]》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e7%be%85%e9%a6%ac%e5%b0%bc%e4%ba%9e&tycountry_e=%e7%be%85%e9%a6%ac%e5%b0%bc%e4%ba%9e&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

1997年9月，羅馬尼亞農業部長{{link-ro|卡瑞雷斯庫|Dinu Gavrilescu}}抵台訪問。<ref name="86年外交年鑑"/>

[[欧洲联盟成员国|歐盟成員]]的羅馬尼亞尚未獲准加入[[申根區|申根區]]，但基於《[[申根公約|申根公約]]》，該國實行與申根區相同的簽證政策。<ref>{{Cite web |url = http://www.mae.ro/en/node/2040 |title = V. Do I need a visa? |publisher = [[羅馬尼亞外交部|羅馬尼亞外交部]]}}</ref>歐洲申根區給予持有載明[[中華民國國民身份證|身份證]]字號的[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以[[歐洲聯盟簽證政策|歐盟免簽證]]入境，停留日數單獨計算，不與申根區合併，每6個月期間內總計可停留90天。近年[[中東#國家及地區|中東國家]]的[[恐怖活动|恐怖活動]]升高，同時為打擊[[人口販賣|人口走私]]，以及羅馬尼亞為加速進入申根區等因素，對外籍旅客入境審查從嚴。用免簽證方式入境以從[[歐洲國家和地區列表|西歐國家]]轉機為宜，避免經由中東或[[中華人民共和國|中華人民共和國]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-243-dff1b-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_FAQ.aspx?c=AD6908DFDDB62656&n=C1B19EF0ACDD3C91&sms=867F5398A49F758A |title = 歐盟免申根簽證常見問答集 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於首都[[布加勒斯特|布加勒斯特]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#EUROPE |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

2014年，羅馬尼亞的[[智慧型手機|智慧型手機]][[市場佔有率|市佔率]]，[[宏達國際電子|宏達國際電子]]（hTC）排名第3，佔10%，次於[[蘋果公司|蘋果公司]][[iPhone|iPhone]]的50%與[[三星電子|三星電子]]的35%。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/ro/pdf/104-59Romania%20.pdf |date = 2015年7月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>然而，在經營規模與宣傳手法方面，hTC仍不敵[[韓國|韓國]]的三星。<ref name="主要產業概況">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 主要產業概況 |publisher = 中華民國對外貿易發展協會}}</ref>

2017年，羅馬尼亞是中華民國第62大貿易夥伴、第63大[[進口|進口]]夥伴、第54大[[出口贸易|出口]]夥伴。出口至羅馬尼亞的金額為2億0,982萬4,512[[美元|美元]]，年增34.308%；自羅馬尼亞進口的金額為1億0,649萬6,612美元，年增16.587%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1億0,332萬7,900美元，年增59.259%。※2016年，羅馬尼亞是中華民國第63大貿易夥伴、第59大進口夥伴、第56大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
在羅馬尼亞的[[台灣商人|台商]]約有4家，包括投資設立[[人造纖維|化纖]]工廠，回收[[寶特瓶|寶特瓶]]製成纖維，銷往[[歐洲國家和地區列表|歐洲各國]]；利用該國[[林業|林業]]資源，專事[[木材|木材]]出口；從事進口[[牙醫學|牙科]]器材銷售；從事[[国际贸易|進口貿易]]業務，自台灣進口各種產品，售予當地[[批發商|批發商]]及[[通路行銷|通路商]]。其中由台商所成立的ROMCARBON SA是羅馬尼亞股票[[上市公司|上市公司]]，該集團擁有9家相關企業，主要以[[資源回收|回收再利用]]生產[[塑膠分類標誌#塑膠類別表|塑膠]]及[[化學品列表|化學製品]]，產品銷往西歐各國，集團員工約3,000人。2011年，與[[歐洲復興開發銀行|歐洲復興開發銀行]]（EBRD）、東南歐基金（SEEF）正式簽約合作，將共同挹注2,350萬[[歐元|歐元]]於羅馬尼亞成立「Green Group」集團，以建置羅馬尼亞首座整合性的[[高科技|高科技]]環保回收園區。<ref name="投資環境簡介"/>

電腦[[資訊科技|資訊科技]]廠商如[[華碩電腦|華碩電腦]]、[[訊舟科技|訊舟科技]]以歐洲駐點在羅馬尼亞設立[[分公司|分公司]]；[[宏碁|宏碁]]及其他台灣科技品牌則由當地[[代理|代理]]商經營；[[宏達電|宏達電]]是直接[[牌照|授權]]由歐洲市場開發[[總部|總部]]設點專責羅馬尼亞及[[摩爾多瓦|摩爾多瓦]]市場。<ref name="投資環境簡介"/>

===犯罪事件===
2016年4月，中華民國[[刑事警察局|刑事警察局]]破獲以羅馬尼亞人為首的偽卡盜刷集團，入台假觀光真取款。其中年資最久的羅馬尼亞籍嫌犯，不僅有盜錄、偽造[[信用卡|信用卡]]或[[提款卡|提款卡]]技術、統籌[[車手|車手]]取款，同夥入台也負責統籌、安排吃住、行程，堪稱集團內的「台灣通」。還誘騙1名台灣大[[土地主|地主]]的女兒，並和該女子生有1女。<ref>{{Cite web |url = http://www.ettoday.net/news/20160428/688791.htm |author = 楊佩琪 |date = 2016年4月28日 |title = 羅馬尼亞詐騙主腦泡夜店拐台女 富豪千金還幫生子 |publisher = [[ETtoday_東森新聞雲|ETtoday 東森新聞雲]]}}</ref>
