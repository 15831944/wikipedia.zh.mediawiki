'''中国人民解放军海军舰载航空兵部队'''，机关驻地[[辽宁省|辽宁省]]，是[[中国人民解放军海军航空兵|中国人民解放军海军航空兵]]首支舰载航空兵部队<ref>{{Cite web |url = http://liaoning.nen.com.cn/system/2016/12/01/019497844.shtml |title =  中央军委追授张超同志“逐梦海天的强军先锋”荣誉称号命名大会举行|author =  |date = 2016-12-01 |publisher =  东北新闻网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.mod.gov.cn/pic/2015-10/09/content_4623605.htm |title =“飞鲨”英雄戴明盟：中国舰载机首飞第一人  |author =  |date = 2015-10-09 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

==沿革==
2011年2月，海军舰载航空兵部队开始筹建。此后边组建、边试验、边训练，先后完成了舰载战斗机试飞员阻拦着舰、滑跃起飞等等任务。经[[中央军委|中央军委]]批准，2013年5月10日，中国人民解放军海军舰载航空兵部队在[[渤海湾|渤海湾]]畔正式组建，海军司令员[[吴胜利|吴胜利]]出席组建大会并向该部队授予军旗。海军舰载航空兵部队组建后，航母舰载机训练开始从科研试飞阶段向舰载机飞行员培训和舰载机上舰阶段过渡。<ref>{{Cite web |url =  http://www.chinanews.com/mil/2013/05-10/4807500.shtml|title =中国海军首支舰载航空兵部队正式组建  |author =  |date = 2013-05-10 |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>

该部队由舰载机飞行部队、机务保障部队等组成，装备[[歼-15|歼-15]]战斗机、教练机、多种型号的多用途直升机。该部队官兵是从全军飞行部队中优选而来，其中将近三分之一来自海军的“[[海空雄鹰团|海空雄鹰团]]”。<ref name=hkxyt/>

2015年9月3日，在[[纪念中国人民抗日战争暨世界反法西斯战争胜利70周年阅兵式|纪念中国人民抗日战争暨世界反法西斯战争胜利70周年阅兵式]]上，该部队部队长戴明盟带领5架歼-15舰载战斗机组成“V”字形梯队，飞过[[天安门广场|天安门广场]]上空，并放下尾钩向人们致敬，这是中国人民解放军海军固定翼舰载机第一次参加盛大阅兵。<ref name=dai/>

2016年4月27日，该部队舰载机飞行员[[张超_(飞行员)|张超]]在驾驶歼-15进行陆基模拟着舰训练时，飞机发生电传故障坠毁，张超牺牲。2016年11月，[[中央军委主席|中央军委主席]][[习近平|习近平]]签署命令，追授海军舰载航空兵部队一级飞行员张超“逐梦海天的强军先锋”称号。<ref>{{Cite web |url =http://news.163.com/16/1115/02/C5SMD4J9000187VI.html  |title =  追授海军某舰载航空兵部队一级飞行员张超“逐梦海天的强军先锋”荣誉称号|author =  |date =2016-11-15  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>
