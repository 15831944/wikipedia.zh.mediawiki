'''中国人民解放军海军研究院'''，总部位于[[北京市|北京市]]，机构分布在北京市、[[上海市|上海市]]，是[[中国人民解放军海军|中国人民解放军海军]]直属单位，是中国人民解放军海军最高科研机构。

==沿革==
1983年5月，经[[中国人民解放军总参谋部|中国人民解放军总参谋部]]批准，中国人民解放军海军组建了'''中国人民解放军海军装备论证研究中心'''，承担海军新型武器装备战术技术和可行性研究任务。该中心是在[[刘华清|刘华清]]倡议下组建<ref name=jiemi/>。1984年，[[柳克俊|柳克俊]]调海军装备论证研究中心任第一任总工程师<ref>陈涛，参与我国第一台军用计算机的研制者——海军装备论证研究中心总工程师柳克俊教授，科技成果管理与研究2011(2):97-97</ref>。该研究中心主要采用计算机等科技手段，通过[[系统论|系统论]]、[[控制论|控制论]]、[[信息论|信息论]]、[[运筹学|运筹学]]等，对海军装备发展决策所涉重大课题开展研究。截至2000年，海军装备论证研究中心已经初步建成计算仿真中心、战役战术对抗模拟实验室、舰艇战术应用软件开发实验室等10个实验室，成立了“海军软件开发中心”、“海军可靠性研究中心”等机构<ref>{{Cite web |url = http://www.people.com.cn/GB/channel1/15/20000810/181957.html |title =海军装备论证中心组建１７年科研成果逾４０００  |author =  |date = 2000-08-10 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。军事专家[[张召忠|张召忠]]、[[尹卓|尹卓]]都曾经在海军装备论证研究中心任职<ref name=zhudao/>。

2003年中国人民解放军体制编制改革时，该中心更名为'''中国人民解放军海军装备研究院'''。研究院拥有10个解放军全军特有学科、13个全军优势学科、20多个全军及海军装备技术研究和咨询机构<ref name=zhudao>{{Cite web |url =  http://mil.news.sina.com.cn/2013-10-23/0842745781.html|title = 谁在主导中国航母工程？解读我海军装备论证过程 |author =  |date =  2013-10-23|publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。海军装备研究院总部位于北京市，机构分布在北京市、[[上海市|上海市]]，是负责海军武器装备总体建设与发展及技术探索性研究的科研单位，是中国人民解放军海军直属单位<ref name=jj>{{Cite web |url = http://yz.chsi.com.cn/sch/schoolInfo--schId-367831.dhtml |title = 海军装备研究院 |author =  |date =  |publisher =中国研究生招生信息网  |language =  |accessdate = 2017-01-13 |quote =  }}</ref>。

该院在军内定期出版发行正式刊物《论证与研究》和《外国海军文集》。1986年起，开始招收硕士研究生。1995年与[[哈尔滨工程大学|哈尔滨工程大学]]联合设立管理科学与工程博士点。1999年起，开展授予具研究生毕业同等学力人员硕士学位工作。2002年，设立[[博士后|博士后]]科研工作站。<ref name=jj/>2008年9月至12月举办“海军装备发展前沿论坛”<ref name=lanwei/>。2014年，该院与[[北京理工大学|北京理工大学]]签署军民融合创新发展战略合作框架协议<ref>{{Cite web |url = http://www.bit.edu.cn/xww/zhxw/104757.htm |title =北京理工大学和海军装备研究院签署战略合作框架协议  |author =  |date = 2014-08-19 |publisher =  北京理工大学新闻网|language =  |accessdate =  |quote =  }}</ref>。

2017年7月19日，新调整组建的军事科学院、国防大学、国防科技大学成立大会暨军队院校、科研机构、训练机构主要领导座谈会在北京[[八一大楼|八一大楼]]举行，[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军委主席|中央军委主席]][[习近平|习近平]]向军事科学院、国防大学、国防科技大学授军旗、致训词，出席座谈会并发表讲话<ref name=xunci>{{Cite web |url =http://news.xinhuanet.com/politics/2017-07/19/c_1121347127.htm  |title = 习近平向军事科学院、国防大学、国防科技大学授军旗致训词 |author =  |date = 2017-07-19 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。在此次改革中，海军装备研究院调整组建为'''中国人民解放军海军研究院'''<ref name=wy>{{Cite web |url = http://war.163.com/17/0804/13/CR0HKDRC000181KT.html |title =  辽宁舰原政委梅文、原舰长张峥双双晋升少将军衔|author =  |date = 2017-08-04 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。
