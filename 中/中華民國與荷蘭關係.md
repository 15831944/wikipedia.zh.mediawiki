'''中華民國與荷蘭關係'''是指[[中華民國|中華民國]]與[[荷蘭王國|荷蘭王國]]（正式國名為'''尼德蘭'''）之間的關係。1913－1950年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交直至1970年代後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[荷蘭|荷蘭]]。]]
{{see also|中華民國外交|{{link-en|荷蘭外交|Foreign relations of Netherlands}}}}
{{see also|中国驻荷兰大使列表{{!}}中華民國駐荷蘭外交官列表|荷兰驻华大使列表{{!}}荷蘭駐中華民國外交官列表}}
1912年，中華民國成立後，向荷蘭派駐外交代表（同年改派駐[[特命全權公使|公使]]）。

1913年10月6日，荷蘭[[外交#外交上的承認|外交承認]]中華民國[[北洋政府|北洋政府]]，於行政首都[[海牙|海牙]]設立'''中華民國駐荷蘭王國[[外交代表機構#公使馆|公使館]]'''。

1928年12月19日，隨著[[國民革命軍北伐|國民革命軍北伐]]幾近尾聲，荷蘭外交承認中華民國[[國民政府|國民政府]]。

1943年2月15日，建立大使級外交關係。公使館升格為'''中華民國駐荷蘭王國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1947年，荷兰在[[南京市_(中華民國)|南京]]设立'''[[荷兰驻中华民国大使馆旧址|驻中华民国大使馆]]'''<ref>{{Cite web|url=http://news.jstv.com/a/20171127/1511760553126.shtml|title=荷兰驻中华民国大使馆旧址如今成这样_荔枝网新闻|accessdate=2018-05-01|work=news.jstv.com}}</ref>。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1950年3月27日，荷蘭與中華民國斷交。<ref name="103年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/09.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>31日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

1980年代，[[中華民國政府|中華民國政府]]執行「劍龍專案」，[[中華民國海軍|海軍]]計畫向荷蘭購買六艘[[劍龍級潛艇|劍龍級潛艇]]，但是[[中華人民共和國|中華人民共和國]]對荷蘭進行外交干涉，只完成[[海龙号潜艇|海龍號]]、[[海虎號潛艇|海虎號]]兩艘，後續四艘未能成交。中華人民共和國更因此採購案曾於1981年5月起將與荷蘭的外交關係降為[[代办|代辦級]]，直到1984年2月1日起才回復為大使級<ref name="cn">[http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/oz_606480/1206_606944/sbgx_606948/ 中国同荷兰的关系]</ref><ref name="ce">[http://nl.china-embassy.org/eng/zhgx/zzgx/t137589.htm China and the Netherlands]</ref>。

===環境===
為增進因應[[氣候變遷|氣候變遷]]及[[治水|治水]]防災能力，中華民國[[中央研究院|中央研究院]]在2014年9月與總部設在荷蘭的三角洲國際聯盟（Foundation Delta Alliance International，DA）簽署入會協定。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/7fb2e392-9e9e-4bf7-a52e-68579f6bfca4.pdf |author = 國際合作及經濟事務司 |date = 2015年4月14日 |title = 國際合作發展事務103年度報告（肆、呼應聯合國千禧年發展目標）|publisher = 中華民國外交部}}</ref>

===教育===
{{see also|臺灣教育|荷蘭教育}}
近年來，台灣每年約有200名學生赴荷蘭就讀；荷蘭則約有130名學生抵臺研習。兩國的大專院校至2015年共簽署50多項合作[[備忘錄|備忘錄]]，以推動學術研究及教師學生交換合作計畫。<ref name="104年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

荷蘭於2012年成立「荷蘭―亞洲榮譽生暑期學程計畫」，之後每年選送[[荷蘭大學列表|13所研究型大學]]學生前往亞洲研習。2013年選送15名學生赴臺、2014年選送20名、2015年亦選送20名。<ref name="104年外交年鑑"/>

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於第2大城[[鹿特丹|鹿特丹]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#EUROPE |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]則在行政首都[[海牙|海牙]]設立'''駐荷蘭代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2017年，荷蘭是中華民國第13大貿易夥伴、第15大[[進口|進口]]夥伴、第12大[[出口贸易|出口]]夥伴，在[[歐洲國家和地區列表|歐洲經濟體]]排名第2、3、2。出口至荷蘭的金額為49億7,812萬3,823[[美元|美元]]，年增11.388%；自荷蘭進口的金額為34億7,633萬3,767美元，年減16.139%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）15億0,179萬0,056美元，年增363.785%。※2016年，荷蘭是中華民國第13大貿易夥伴、第11大進口夥伴、第12大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資===
截至2017年，荷蘭對台灣的總投資金額為296億1,451萬美元，是第一大[[外商直接投資|歐洲外資來源國]]；[[臺灣商人|臺商]]對荷蘭的總投資金額為24億3,550萬美元。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref><ref name="96年外交年鑑"/>自1989年起，台商前往荷蘭投資設點，累計約150－200家，絕大多數是在台灣的[[上市公司|上市]]（[[柜台买卖|櫃]]）公司，其選擇荷蘭作為[[歐洲國家和地區列表|歐洲地區]]的[[市場行銷|行銷]]、[[倉庫|倉儲]]、提供售後維修服務、[[投資|投資]]、[[研究開發|研發]]等[[總部|總部]]。<ref name="投資環境簡介"/>

其中以[[資訊及通訊科技|資訊及通訊科技]]（ICT）相關產業為主，如：[[鴻海科技集團|鴻海科技集團]]、[[宏碁|宏碁]]、[[華碩電腦|華碩電腦]]、[[技嘉科技|技嘉科技]]、[[微星科技|微星科技]]、[[緯創資通|緯創資通]]、[[群創光電|群創光電]]、[[台灣積體電路|台灣積體電路]]、[[聯華電子|聯華電子]]、[[台達電子|台達電子]]、[[研華科技|研華科技]]等；[[傳統產業|傳統產業]]則有[[東元電機|東元電機]]、[[南港輪胎|南港輪胎]]、[[美利達工業|美利達工業]]、永光化學、桂盟集團、堤維西等；少數從事[[民用航空|航空]]、[[航運|海運]]、[[運輸|運輸]]、或[[金融|金融]]等[[第三產業|服務業]]，如[[兆豐國際商業銀行|兆豐國際商業銀行]]；[[和碩聯合科技|和碩聯合科技]]、[[台灣糖業公司|台灣糖業公司]]（養[[蘭科|蘭]]）、有大等公司則從事簡易[[裝配線|裝配]]或[[種植業|種植]]業務。台商選擇荷蘭做為歐洲營運總部最重要的理由是該國[[稅|賦稅]]環境相對優惠且富彈性，其他原因如：良好的[[英語|英文]]溝通環境、發達的[[物流|物流]]業、及荷蘭投資官員良好的「售後服務」等。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/nl/pdf/104-51Netherlands.pdf |date = 2015年7月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref><ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 國家基本資料表 |publisher = 中華民國對外貿易發展協會}}</ref>
