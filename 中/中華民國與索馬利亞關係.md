==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[索馬利亞|索馬利亞]]。]]
{{see also|中華民國外交|{{link-en|索馬利亞外交|Foreign relations of Somalia}}}}
2012年7月17日，遭[[索馬利亞海盜|索馬利亞海盜]]劫持18個月後，中華民國籍[[漁船|漁船]]「旭富一號」全體船員獲釋，船上26名船組人員全部平安。[[中華民國外交部|外交部]]對[[歐洲聯盟|歐洲聯盟]]、國際反海盜組織「海洋無海盜」（Oceans Beyond Piracy）和相關國家，以及[[中华人民共和国|中國大陸]]基於[[人道援助|人道]]所予之協助，表達感謝。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=8742DCE7A2A28761&sms=491D0E5BF5F4BC36&s=6240EBDF7EA94F9C |author = 公眾外交協調會 |date = 2012年7月18日 |title = 前遭索馬利亞海盜劫持之我國籍漁船「旭富一號」船員獲釋放 |publisher = 中華民國外交部}}</ref>

2012年3月，遭索馬利亞海盜在[[印度洋|印度洋]]劫持的[[阿曼|阿曼籍]]漁船「Naham3」與26名船員，包含1名[[中華民國籍|中華民國籍]][[輪機長|輪機長]]，在將近5年的拘禁後於2016年10月22日獲釋，搭機先至[[廣州|廣州]]，再各自返回母國。獲釋前，中華民國籍[[船長|船長]]在劫船過程中遭海盜殺害，另有兩名漁工病故。協助談判的國際組織「人質支持夥伴」（Hostage Support Partners）協調員史蒂德（John Steed）指出，這次[[人質|人質]]談判歷經18個月，Naham3船員獲釋意味著索馬利亞海盜猖獗時期綁架的最後一批船員終於獲釋，這也是該國海盜劫持人質時間第二久的事件。<ref>{{Cite web |url = http://news.ltn.com.tw/news/focus/paper/1045036 |author = 綜合報導 |date = 2016年10月24日 |title = 遭海盜囚5年 台籍輪機長獲釋 |publisher = [[自由時報|自由時報]]}}</ref>外交部亞西及非洲司長[[陳俊賢|陳俊賢]]在[[立法院|立法院]]答詢時指出，船東與台灣民間至少付出115萬[[美元|美元]][[贖金|贖金]]，其他國家也有出錢。此外，據輪機長家屬透露外交部要求簽切結書，日後須歸還外交部先代墊返台的機票費用；對此，前[[中央警察大學|中央警察大學]]教授[[葉毓蘭|葉毓蘭]]痛批「一樣是[[台灣人|台灣人]]落難在[[非洲|非洲]]，人質不如[[詐騙集團|詐騙集團]]」。針對外界質疑政府出錢包專機，將海外詐騙犯接回台，遭劫的台灣人還不如詐騙犯，陳俊賢澄清，不論是先前[[土耳其|土耳其]]、[[印尼|印尼]]與[[馬來西亞|馬來西亞]]送回台的[[台湾诈骗集团|台灣詐騙犯]]，機票錢都是他們自己付的。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/1868699 |author = 綜合報導 |date = 2016年10月27日 |title = Naham3漁船案 外交部：台灣民間出錢2700萬元 |publisher = 自由時報}}</ref><ref>{{Cite web |url = http://www.ettoday.net/news/20161026/799675.htm |author = 綜合報導 |date = 2016年10月26日 |title = 葉毓蘭批：一樣是台灣人落難在非洲，人質不如詐騙集團 |publisher = [[ETtoday_東森新聞雲|ETtoday 東森新聞雲]]}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境索馬利亞。]]
[[File:Visa_requirements_for_Somali_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|索馬利亞簽證政策|Visa policy of Somalia}}|{{link-en|索馬利亞公民簽證要求|Visa requirements for Somali citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。{{link-en|索馬利亞政府|Politics of Somalia}}沒有給予[[世界政區索引|任何地區]]的公民[[簽證#签证优惠待遇|免簽證]]待遇，<ref>{{Cite web |url = http://cms.olympicair.com/timatic/webdocsI/countryinfo.html |author = Information sourced and provided by [[國際航空運輸協會|IATA]] |title = Visa Information |publisher = {{link-en|Timaticweb|Timatic}} }}</ref>持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往索馬利亞駐[[馬來西亞|馬來西亞]][[外交代表機構#大使馆|大使館]]申請簽證。若護照上有[[以色列签证政策|以色列簽證]]將可能被拒絕入境。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-198-2e75a-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|索馬利亞護照|Somali passport}}的{{link-en|索馬利亞人口|Demographics of Somalia|索馬利亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將索馬利亞歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2017年10月30日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-247-70b60-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將索馬利亞歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2017年8月15日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=459E89EF3686B132 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
2017年，索馬利亞是中華民國第185大貿易夥伴、第187大[[進口|進口]]夥伴、第175大[[出口贸易|出口]]夥伴。出口至索馬利亞的金額為110萬7,465[[美元|美元]]，年增15.361%；自索馬利亞進口的金額為13萬3,172美元，年增313.282%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）97萬4,293美元，年增5.014%。※2016年，索馬利亞是中華民國第183大貿易夥伴、第198大進口夥伴、第173大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
