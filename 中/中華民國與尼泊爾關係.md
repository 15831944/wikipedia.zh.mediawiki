'''中華民國與尼泊爾關係'''是指[[中華民國|中華民國]]與[[尼泊爾聯邦民主共和國|尼泊爾聯邦民主共和國]]之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對尼泊爾的相關事務由'''[[駐印度台北經濟文化中心|駐印度台北經濟文化中心]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-124-24014-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境尼泊爾。]]
[[File:Visa_requirements_for_Nepalese_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|尼泊爾簽證政策|尼泊爾公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。除了[[印度人口|印度公民]]可[[簽證#签证优惠待遇|免簽證]]外，幾乎[[世界政區索引|所有地區]]的公民（12國除外）前往尼泊爾均可在首都[[加德滿都|加德滿都]]的[[特里布萬國際機場|特里布萬國際機場]]辦理[[簽證#落地签证|落地簽證]]入境，停留最多30天。經尼泊爾轉機，若不出機場，可停留24小時免[[過境簽證|過境簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-124-24014-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&sms=B75E93A3DAB23A99 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|尼泊爾護照|Nepalese passport}}的{{link-en|尼泊爾人口|Demographics of Nepal|尼泊爾公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將尼泊爾歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2017年6月1日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-65-66b48-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|尼泊爾經濟|Economy of Nepal}}}}
===貿易===
2011年4月17日，[[經濟部國際貿易局|經濟部國際貿易局]]委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）規劃的拓銷[[南亞|南亞]]市場首發團13家[[出口贸易|外銷]]廠商抵達尼泊爾首都[[加德滿都|加德滿都]]，成為[[台灣|台灣]]最先前往拓銷的經貿團體。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=559428&url=/NewsSearch.asp |author = 王裕宏 |date = 2011年4月22日 |title = 貿協率13家廠商首度赴尼泊爾拓銷 |publisher = 經貿透視網}}</ref>

2013年4月26日，外貿協會秘書長與尼泊爾商工會（Lalitpur Chamber of Commerce and Industry，LCCI）會長共同簽署合作[[備忘錄|備忘錄]]，是外貿協會第一次與尼泊爾工商團體簽署備忘錄，以增加企業交流及提升雙邊貿易。<ref>{{Cite web |url = http://www.trademag.org.tw/content02.asp?id=609484&type=49&url=/NewsSearch.asp |author = 高麗茹 |date = 2013年5月15日 |title = 貿協與尼泊爾簽署合作備忘錄 促進經貿交流 |publisher = 經貿透視雙周刊第367期}}</ref>另參加「2013台北國際[[文具|文具]][[禮物|禮品]]展」，承租16個攤位並設立尼泊爾國家主題館。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=609646&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2013年4月27日 |title = 外貿協會首次與尼泊爾簽署合作備忘錄 開啟經貿合作新契機 |publisher = 經貿透視網}}</ref>

2018年，尼泊爾是中華民國第123大貿易夥伴、第144大[[進口|進口]]夥伴、第109大[[出口贸易|出口]]夥伴。出口至尼泊爾的金額為2,492萬5,230[[美元|美元]]，年增160.643%；自尼泊爾進口的金額為207萬7,775美元，年增52.592%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）2,284萬7,455美元，年增178.583%。2017年，尼泊爾是中華民國第141大貿易夥伴、第151大進口夥伴、第123大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

==交流==
===援助===
[[奧比斯|國際奧比斯基金會]]（Orbis International）與台灣奧比斯基金會（Orbis Taiwan）合作進行「尼泊爾兒童[[眼科学|眼科]]醫療援助計畫」，包括兒童眼科中心及硬體設施、培訓專職眼科醫療人員、協助1,050名兒童進行[[視力|視力]]矯正及320名兒童進行眼疾手術，並提供[[加德滿都|加德滿都]]地區15歲以下之12萬名兒童提供視力篩檢服務。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/133bf67f-6547-4c62-8bfd-ef26e383f644.pdf |author = 國際合作及經濟事務司 |date = 2014年4月23日 |title = 國際合作發展事務102年度報告（參、102年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref>

2015年，[[2015年4月尼泊尔地震|尼泊爾發生地震]]，其中[[廓爾喀縣|廓爾喀縣]]為重災區之一。國際合作發展基金會（國合會）與國際關懷協會尼泊爾分會合作推動「尼泊爾廓爾喀縣[[糧食安全|糧食安全]]及生計支援計畫」，11月底結束，鑒於績效良好，於12月繼續推動「尼泊爾廓爾喀縣糧食安全及生計強化計畫」，強化該縣1,800戶受地震影響的脆弱家戶糧食安全與生計；增進農民參與農業價值鏈及與市場連結，促進受地震影響社區的{{link-en|農業商業化|Agribusiness}}；輔導脆弱家戶取得資金及市場管道，以強化微小型企業發展。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/ab7236a0-e636-4439-b9ce-c6b86ac4339e.pdf |author = 國際合作及經濟事務司 |date = 2017年4月28日 |title = 國際合作發展事務105年度報告（參、105年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref>
