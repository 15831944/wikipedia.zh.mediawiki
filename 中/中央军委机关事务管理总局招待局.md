'''中央军委机关事务管理总局招待局'''，位于[[北京市|北京市]]，是[[中央军委机关事务管理总局|中央军委机关事务管理总局]]下属局，负责中央军委机关的招待工作。

==沿革==
2016年改革前，[[中国人民解放军总参谋部管理保障部|中国人民解放军总参谋部管理保障部]]下设有中国人民解放军总参谋部管理保障部招待局，该局下辖：[[京东宾馆|京东宾馆]]（总参一所）、[[京南宾馆|京南宾馆]]（总参三所）、[[京津宾馆|京津宾馆]]（总参天津招待所）、[[东都宾馆|东都宾馆]]（总参泰安招待所）、[[翡翠宾馆|翡翠宾馆]]（总参黄山招待所）等宾馆。

在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月组建[[中央军委机关事务管理总局|中央军委机关事务管理总局]]。中央军委机关事务管理总局新设'''中央军委机关事务管理总局招待局'''<ref name=gg>{{Cite web |url =http://www.bidchance.com/calggnew/2016/12/16/17833922.html  |title =  更新电梯工程招标公告|author =  |date = 2016-12-16 |publisher = 中国招标网 |language =  |accessdate =  |quote =  }}</ref><ref name=xy>{{Cite web|url = http://www.ydl169.com/wenxin/article.asp?articleid=6|title = 馨怡宾馆简介|author = |date = |publisher = 馨怡网|language = |accessdate = 2017-03-28|quote = |archive-url = https://web.archive.org/web/20170904202104/http://www.ydl169.com/wenxin/article.asp?articleid=6|archive-date = 2017-09-04|dead-url = yes}}</ref>。

2016年，其下属的中央军委机关事务管理总局招待局第一招待所<ref name=diyi>{{Cite web |url = http://www.ccgp.gov.cn/cggg/dfgg/zbgg/201701/t20170117_7867097.htm |title = 北京地区（城六区）2016-2018年度会议定点单位协议采购项目中标公告：006中标公告附件1：第一包中标供应商名单 |author =  |date = 2017-01-17 |publisher =中国政府采购网  |language =  |accessdate =  |quote =  }}</ref>、第三招待所、[[馨怡宾馆|馨怡宾馆]]等单位获评为中央和北京市的北京地区（城六区）2016-2018年度会议定点单位<ref name=zhbiao>{{Cite web |url = http://www.ccgp.gov.cn/cggg/dfgg/zbgg/201701/t20170117_7867097.htm |title = 北京地区（城六区）2016-2018年度会议定点单位协议采购项目中标公告：006中标公告附件2：第二包中标供应商名单 |author =  |date = 2017-01-17 |publisher =中国政府采购网  |language =  |accessdate =  |quote =  }}</ref>。
