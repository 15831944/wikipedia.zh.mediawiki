'''中國共產黨'''，簡稱'''中共'''，在[[中國大陸|中國大陸]]常簡稱為'''共产党'''或'''党'''，目前是[[中華人民共和國|中華人民共和國]][[一黨制|唯一]]的[[執政黨|執政黨]]<ref>{{Cite web |url = http://news.xinhuanet.com/mil/2013-01/01/c_124173651.htm |title = 为建设一支听党指挥能打胜仗作风优良的人民军队而奋斗 |author = 《[[解放軍報|解放軍報]]》 |date = 2013年1月1日 |publisher = [[新華網|新華網]] |language = zh-Hans |accessdate = 2015年6月1日 |quote =  |archive-url = https://web.archive.org/web/20130927162356/http://news.xinhuanet.com/mil/2013-01/01/c_124173651.htm |archive-date = 2013-09-27 |dead-url = no }}</ref>。该党创立于[[中华民国大陆时期|中华民国大陆时期]]，由[[陳獨秀|陳獨秀]]和[[李大釗|李大釗]]於1921年7月组织成立，初期接受[[第三國際|共产国际]]的指导和援助，并与[[中国国民党|中国国民党]]合作。[[第一次国共合作|第一次国共合作]]破裂後，两黨进行了[[第一次國共內戰|長達10年的军事斗争]]。1940年代起，中國共產黨逐渐摆脱外國勢力的影響，改由以[[毛泽东|毛泽东]]为首的本土派掌权。在[[抗日战争|抗日战争]]期间，与国民黨进行了[[第二次国共合作|第二次国共合作]]。1945年抗战胜利后，中国共产党率領[[中國人民解放軍|中國人民解放軍]]在[[第二次國共內戰|第二次國共內戰]]中擊敗[[中華民國國軍|中華民國國軍]]，進而取得[[大陸地區|大陸地區]]及其絕大多數沿海島嶼的統治權，於1949年建立[[中華人民共和國|中華人民共和國]]。自1980年代開始，中國共產黨開始排除意識形態問題和许多非共產主義的政黨建立关系。1989年[[东欧剧变|东欧剧变]]和1991年[[蘇聯解體|蘇聯解體]]後，中國共產黨開始加强和其它社會主義國家執政黨的交流，並且繼續維持和世界各地非執政黨的共產黨聯繫。

中國共產黨的組織架構主要基於[[民主集中制|民主集中制]]，主張應共同維護已獲得共識的政策。中國共產黨最高機構是[[中國共產黨全國代表大會|全國代表大會]]，每5年召開一次。[[中國共產黨中央委員會|中央委員會]]作為全國代表大會閉會期間的核心權力機構，通常1年召開1次會議。因此，多數職權和日常工作都由[[中國共產黨中央政治局|中央政治局]]和其[[中國共產黨中央政治局常務委員會|常務委員會]]掌握。[[中共中央主要負責人|中央主要負責人]]自1993年開始會同時擔任負責黨內和民間事務的[[中國共產黨中央委員會總書記|中央委員會總書記]]、負責軍事事務的[[中國共產黨中央軍事委員會主席|中央軍事委員會主席]]和具外交禮儀性質的[[中華人民共和國主席|中華人民共和國主席]]，而其能有效掌握三個職位時常被外界稱作[[中華人民共和國最高領導人|最高領導人]]。[[習近平|習近平]]自2012年[[中國共產黨第十八次全國代表大會|第十八次全國代表大會]]至今担任中央委員會總書記成為中共最高領導人<ref>{{Cite web |url = http://news.xinhuanet.com/politics/2014-01/09/c_118895893.htm |title = 在实现中国梦征途上昂扬奋进——以习近平同志为总书记的党中央十八大以来治国理政纪实 |author = 杨婷 |date = 2014年1月9日 |publisher = [[新華網|新華網]] |language = zh-Hans |accessdate = 2015年6月1日 |quote =  |archive-url = https://web.archive.org/web/20150702215352/http://news.xinhuanet.com/politics/2014-01/09/c_118895893.htm |archive-date = 2015-07-02 |dead-url = no }}</ref>。截至2017年12月31日，中國共產黨共有8956.4萬名黨員、基层党组织則有457.2万个，其中女性占26.7%，少數民族占7.3%<ref name="number"></ref>。

中國共產黨始終宣稱其理想與目標是[[社會主義|社會主義]]，并將自身視為代表工人階級領導[[工農聯盟|工農聯盟]]和[[統一戰線|統一戰線]]的政黨。其中《[[中國共產黨章程|中國共產黨章程]]》指出中國共產黨堅持[[馬克思列寧主義|馬克思列寧主義]]、[[毛澤東思想|毛澤東思想]]、[[鄧小平理論|鄧小平理論]]、[[“三个代表”重要思想|“三个代表”重要思想]]、[[科學發展觀|科學發展觀]]和[[习近平新时代中国特色社会主义思想|习近平新时代中国特色社会主义思想]]等意识形态，並將中國共產黨表述為：「中國工人階級的先鋒隊，同時是中國人民和中華民族的先鋒隊，是中國特色社會主義事業的領導核心，代表中國先進生產力的發展要求，代表中國先進文化的前進方向，代表中國最廣大人民的根本利益。黨的最高理想和最終目標是實現共產社會。<ref>{{Cite web|url=http://news.xinhuanet.com/ziliao/2004-11/24/content_2255749.htm |title=中国共产党 |author= |date= |publisher=[[新華網|新華網]] |language=zh-Hans |accessdate=2015年6月1日 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20150810194006/http://news.xinhuanet.com/ziliao/2004-11/24/content_2255749.htm |archivedate=2015年8月10日 }}</ref><ref name = "中國共產黨章程 總綱">{{Cite book | author = [[中國共產黨第十六次全國代表大會|中國共產黨第十六次全國代表大會]] | title = 《[[中國共產黨章程|中國共產黨章程]]》總綱 | location = 中國北京 | publisher = [[中國共產黨全國代表大會|中國共產黨全國代表大會]] | date = 2002年11月14日 | pages =  | ISBN =  | accessdate = 2015年6月1日 | url =  | language = zh-Hans | quote =  }}</ref>」不過在[[毛澤東|毛澤東]]建立的[[計劃經濟|計劃經濟]]體制被認為效率低下的情況下，中國官方在政治理念的執行層面仍然從務實出發的，但在思想上為了確保路線一致不變動，[[鄧小平|鄧小平]]等人遂提出在國家處於[[社會主義初級階段|社會主義初級階段]]的條件下，應該先以逐步摸索的方式，來建設共產主義社會較為實際，利用此理由嘗試引入[[改革开放|经济改革]]，來發展{{Link-en|資本主義生產方式 (馬克思主義理論)|Capitalist mode of production (Marxist theory)|資本主義生產方式}}，於是中國大陸得以透過設立中國特色的社會主義爲綱的操作方式，以「[[實踐是檢驗真理的唯一標準|實踐是檢驗真理的唯一標準]]」這一口號而在20世紀末期順利轉型至[[社會主義市場經濟|社會主義市場經濟]]。

== 名称标识 ==
{{main|中國共產黨黨旗黨徽}}
[[File:18th_National_Congress_of_the_Communist_Party_of_China.jpg|thumb]]會場上的中國共產黨黨徽。]]

1920年9月1日，党派名称最早在陳獨秀在《[[新青年|新青年]]》發表的《對於時局之我見》中稱為「社會黨」，後來改稱為「共產黨」<ref name="名称由来">{{Cite web |url = http://cpc.people.com.cn/GB/64156/65682/4527309.html |title = “中国共产党”名称的由来 |author = 新華網 |date =  |publisher = 中国共产党新闻网 |language = zh-Hans |accessdate = 2016年8月20日 |quote =  |archive-url = https://web.archive.org/web/20161129222549/http://cpc.people.com.cn/GB/64156/65682/4527309.html |archive-date = 2016-11-29 |dead-url = no }}</ref>。在1920年8月13日和9月16日，留學法國的[[蔡和森|蔡和森]]在給毛澤東寫的兩封信中詳細地研討了共產黨問題，並且首先提出：「先要組織黨——共產黨，因為它是革命運動的發動者、宣傳者、先鋒隊、作戰部。」對此他在考察[[俄国共产党（布尔什维克）|俄国共产党（布尔什维克）]]等歐洲各國共產黨後提出具體的建黨步驟，其中包括宣告正式成立中國共產黨<ref name = "名称由来"/>。而當時人在上海市的陳獨秀與人在北京市的李大釗、張申府也通過信函討論名稱問題，最後決定採用「共產黨」作為中國無產階級政黨的名稱<ref name = "名称由来"/>。中国共产党的党歌事实上是《[[国际歌|国际歌]]》，但是並未正式写入《[[中国共产党章程|中国共产党章程]]》<ref name="党歌">{{Cite web |url = http://news.xinhuanet.com/politics/2008-01/28/content_7511238.htm |title = 瞿秋白：译词传谱国际歌 |author = 姚笛 |date = 2008年1月28日 |publisher = [[光明网_(中国)|光明网]] |language = zh-Hans |accessdate = 2015年6月29日 |quote =  |archive-url = https://web.archive.org/web/20160304105738/http://news.xinhuanet.com/politics/2008-01/28/content_7511238.htm |archive-date = 2016-03-04 |dead-url = no }}</ref>。

中國共產黨早期發展時並沒有設計統一官方標準的標誌，而是允許個別中國共產黨組織參考蘇聯共產黨製作標誌<ref name="Liang Jun">{{Cite web |url = http://english.cpc.people.com.cn/206972/206981/8188029.html |title = Flag and emblem of Communist Party of China |author = Liang Jun和Yao Chun |date = 2013年3月29日 |publisher = [[中國共產黨新聞網|中國共產黨新聞網]] |language = en |accessdate = 2015年6月1日 |quote =  |archive-url = https://web.archive.org/web/20150327215204/http://english.cpc.people.com.cn/206972/206981/8188029.html |archive-date = 2015-03-27 |dead-url = no }}</ref>。1942年4月28日，中國共產黨中央政治局下令制定統一的官方標誌，在決定中提到「中共黨旗樣式寬闊為三與二之比，左角上有斧頭鐮刀，無五角星，委託中央辦公廳制一批標準黨旗，分發各主要機關」<ref name = "Liang Jun"/>。隨後中國共產黨規定標準黨旗橫120公分、豎80公分，在左上角的中心（距離邊界4分之一的長度和寬度處）設計有直徑30公分的黃色錘子和鐮刀。旗杆左套為白色並且寬6.5公分，這寬度也就是旗幟橫長18分之一。

中國共產黨黨旗採用紅色背景視為象徵革命、錘子和鐮刀象徵工人和農民的工具，黃色標誌則象徵光明，這也意味著中國共產黨代表人民群眾和其利益<ref name = "Liang Jun"/>。發展至今日中國共產黨黨旗具有5種尺寸，其尺寸分別為橫388公分，豎192厘米（1號）；橫240公分，豎160公分（2號）；橫192公分，豎128公分（3號）；橫144公分，豎96公分（4號）；橫96公分，豎64公分（5號）<ref name = "Liang Jun"/>。1966年9月21日中共中央辦公廳印製《中國共產黨黨旗黨徽製作和使用的若干規定》，其中指出中國共產黨的黨徽黨旗是中國共產黨的官方符號和象徵<ref name = "Liang Jun"/>。之後則[[中國共產黨第十六次全國代表大會|中國共產黨第十六次全國代表大會]]通過《中國共產黨章程》修正案，在新增加的第53條中提到：「中國共產黨的黨徽黨旗是中國共產黨的象徵和標誌。黨的各級組織和每一個黨員都要維護黨徽黨旗的尊嚴。<ref>{{Cite book | author = [[中國共產黨第十六次全國代表大會|中國共產黨第十六次全國代表大會]] | title = 《[[中國共產黨章程|中國共產黨章程]]》第53條 | location = 中國北京 | publisher = [[中國共產黨全國代表大會|中國共產黨全國代表大會]] | date = 2002年11月14日 | pages =  | ISBN =  | accessdate = 2015年6月1日 | url =  | language = zh-Hans | quote =  }}</ref>」

=== 民主集中制 ===
{{main|民主集中制}}
中國共產黨的組織原則採取[[民主集中制|民主集中制]]，並且基於[[民主|民主]]和{{Link-en|中央集權政府|Centralized government|集中}}兩個原則而成，其中官方所指稱的民主主要是「社會主義民主」或者是「黨內民主」<ref name="王传志">{{Cite web |url = http://www.qstheory.cn/zxdk/2013/201310/201305/t20130513_230301.htm |title = 民主集中制： 我国政治制度的核心机制 |author = 王传志 |date = 2013年5月16日 |publisher = 《[[求是|求是]]》 |language = zh-Hans |accessdate = 2015年6月1日 |quote =  |archive-url = https://web.archive.org/web/20141121033129/http://www.qstheory.cn/zxdk/2013/201310/201305/t20130513_230301.htm |archive-date = 2014-11-21 |dead-url = no }}</ref>。而有關民主集中制的討論，自從1927年召開的[[中國共產黨第五次全國代表大會|中國共產黨第五次全國代表大會]]便一直視為中國共產黨的組織原則<ref name = "王传志"/>。其中在《中國共產黨章程》中提到：「黨是根據自己的綱領和章程，按照民主集中制組織起來的統一整體。<ref name = "王传志"/>」而毛澤東曾經表示民主集中制是「將民主和集中兩個似乎相衝突的東西，在一定形式上統一起來」，認為藉此結合能夠處理民主和集中之間的內部矛盾，同時民主集中制在[[自由|自由]]和{{Link-en|紀律|Discipline}}協調上更具優越性<ref name = "王传志"/>。

其中在《中國共產黨章程》中提到：「黨的領導主要是政治、思想和組織的領導。黨要適應改革開放和社會主義現代化建設的要求，加強和改善黨的領導。<ref name = "中國共產黨章程 總綱"/>」而當前中國共產黨表示民主同時是中國共產黨自身與社會主義重要的生命線<ref name = "王传志"/>，但是認為要實現且正常運作民主，可行的方法唯有實施集中制度<ref name = "王传志"/>。中國共產黨主張民主得以使用包括集中制度等任何形式展開，並且認為如果沒有集中制度就將會失去秩序<ref name = "王传志"/>。對此中國共產黨依據毛澤東對於民主集中制的觀點成立[[人民代表大會制度|人民代表大會制度]]，認為這是「在民主基礎上的集中，在集中指導下的民主。只有這個制度，才既能表現廣泛的民主，使各級人民代表大會有高度的權力；又能集中處理國事，使各級政府能集中地處理被各級人民代表大會所委託的一切事務，並保障人民的一切必要的民主活動。<ref name = "王传志"/>」
=== 集體領導 ===
{{main|中国共产党集体领导制度}}

Sullivan 183"/>。預備黨員的職責實際上和正式黨員相差不大，不同的是可能無法在中國共產黨黨內參與選舉投票，同時也沒有被選舉權<ref name = "Lawrence R. Sullivan 183"/>。許多中國共產黨黨員是透過加入中國共產黨旗下的[[中國共產主義青年團|中國共產主義青年團]]<ref name = "Lawrence R. Sullivan 183"/>，而江澤民以後民營企業家也被允許成為中國共產黨黨員<ref name = "Lawrence R. Sullivan 183"/>。根據《中國共產黨章程》，全體黨員必須服從命令、遵守紀律、堅持團結、服務於中國共產黨和人民，並且促進社會主義道路的發展<ref name = "中國共產黨章程"/>。

在加入中國共產黨將會獲得許多權限，這包括有參加中國共產黨的有關會議、閱讀中國共產黨有關文件、接受中國共產黨提供的教育培訓、在黨報黨刊上對中國共產黨政策提出討論、針對中國共產黨的工作提出建議和方案、在會議上有根據地批評中國共產黨任何組織和任何黨員、行使投票權和被選舉權。而在堅決執行該項決議或政策作為的前提下，中國共產黨黨員得以對對於決議和政策予以反對或者有不同意見的情況下，並且可向中國共產黨上級機關乃至於中央委員會提交任何意見，並且要求該組織提供答覆<ref name = "中國共產黨章程"/>。同時包括中國共產黨中央委員會等組織，都沒有權限得以剝奪中國共產黨黨員的這些權力<ref name = "中國共產黨章程"/>。

根據[[中共中央組織部|中共中央組織部]]於2016年6月30日的統計數據指出，截至2015年12月31日，中國共產黨黨員總數為8875.8萬名，基層組織則有441.3萬个。就中國共產黨黨員職業予以分類，農民、牧民和漁民有2602.5萬人；工廠員工有724.4萬人；企業單位和民办非企业单位专业技术人员1294.5万人；事業單位和民辦非企業單位管理人员911.4万人；黨政機關工作人員748.5万人；學生203.4万人；離職或退休人員1658.1万人；其他职业人员733万人。而就其他成員結構來看，女性中國共產黨黨員有2227.8萬人，占黨員總數的25.1%；少數民族黨員有618萬人，占黨員總數的7%；具有大專及以上學歷的黨員3932.4萬人，占黨員總數的44.3%；35歲及以下的黨員2254.4萬人。而自2013年開始，中國共產黨在保證發展黨員質量的前提下逐漸控制黨員人士增長速度，在發展受調控下使得總人數增加速度放緩<ref name = "陈剑">{{Cite web |url = http://dangjian.people.com.cn/n/2014/0701/c117092-25221092.html |title = 发展党员调控效果明显：党员8668.6万名，总量增速放缓 |author = 程宏毅和常雪梅 |date = 2014年7月1日 |publisher = [[中國共產黨新聞網|中國共產黨新聞網]] |language = zh-Hans |accessdate = 2015年6月1日 |quote =  |archive-url = https://web.archive.org/web/20150615112707/http://dangjian.people.com.cn/n/2014/0701/c117092-25221092.html |archive-date = 2015-06-15 |dead-url = no }}</ref>。截至2017年12月31日，中國共產黨內部總共有8956.4萬名黨員<ref name = "number"/>，是印度主要政黨[[印度人民黨|印度人民黨]]之後全世界上第二大的政黨<ref>{{cite news|title=印度人民党党员破亿“成为全球最大政党”|url=http://www.xinhuanet.com/world/2015-04/22/c_127715665.htm|publisher=新华网|date=2015-04-22|accessdate=2018-05-03|archive-url=https://web.archive.org/web/20180504012316/http://www.xinhuanet.com/world/2015-04/22/c_127715665.htm|archive-date=2018-05-04|dead-url=no}}</ref>。

早在1970年代期間，中國共產黨為了遏制「[[蘇維埃帝國|蘇維埃帝國]]」擴張而曾經有一陣子與[[極右派|極右派]]政黨締結短期關係。不過在1980年代初期，中國共產黨開始對歐洲的社會民主主義運動有着極大的興趣<ref name = "沈大偉 98"/>，並且與歐洲社會民主主義政黨大力發展政黨之間的互動，這也是中國共產黨首次與非共產黨的政黨展開交流<ref name = "沈大偉 98"/>。在1980年代之前，中國共產黨基於[[第二國際|第二國際]]、[[列寧主義|列寧主義]]和[[史達林主義|史達林主義]]對於社會民主主義運動的觀點，曾一度對於社會民主主義感到負面和輕視<ref name = "沈大偉 98"/>。不過在1980年代這一觀點發生了變化，中國共產黨開始認為歐洲的社會民主主義試圖創建「有人性的資本主義」<ref name = "沈大偉 98"/>，而實際上自身也可以從社會民主主義運動中學到部分經驗<ref name = "沈大偉 98"/>。中國共產黨在1980年代陸續派遣代表前往歐洲各地考察<ref name = "沈大偉 99">{{Cite book | author = [[沈大偉|沈大偉]] | title = China's Communist Party: Atrophy and Adaptation | location = 美國奧克蘭 | publisher = [[加利福尼亞大學出版社|加利福尼亞大學出版社]] |date = 2008年4月2日 | pages = 第99頁 | ISBN = 978-0520254923 | accessdate = 2015年6月1日 | url = https://books.google.com.tw/books?id=IOW6msgYWzwC&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false | language = en | quote =  }}</ref>，同時期大多數歐洲社會民主黨派別都面臨選舉支持率的下滑，以及開始對於自身展開改革<ref name = "沈大偉 99"/>。中國共產黨對此感到極大興趣，這之中又以[[工黨_(英國)|英國工黨]]和[[德國社會民主黨|德國社會民主黨]]的內部改革為首<ref name = "沈大偉 99"/>。中國共產黨對於两者再次取得執政權的原因歸咎於採行現代化措施，藉由轉而支持[[私有化|私有化]]而取代傳統的[[國家社會主義|國家社會主義]]原則，並且將原先的[[大政府|大政府]]思維轉型成為新的[[福利國家|福利國家]]觀點。至此改變對於市場的負面看法，而支持群眾也從傳統[[工會|工會]]轉型成企業家、年輕成員和學生<ref>{{Cite book | author = [[沈大偉|沈大偉]] | title = China's Communist Party: Atrophy and Adaptation |location = 美國奧克蘭 |publisher = [[加利福尼亞大學出版社|加利福尼亞大學出版社]] |date = 2008年4月2日 |pages = 第99頁至第100頁 |ISBN = 978-0520254923 |accessdate = 2015年6月1日 |url = https://books.google.com.tw/books?id=IOW6msgYWzwC&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false |language = en |quote =  }}</ref>。

中國共產黨過去與北韓的緊密關係，這使得中國共產黨是經常派遣代表訪問北韓的組織之一<ref name = "沈大偉 82"/>。儘管中國的朝鮮分析家往往向社會大眾表態支持北韓，並且發表多份有關北韓經濟改革的報告<ref name = "沈大偉 82"/>。但是在中國共產黨官方討論中則很大程度上蔑視北韓的經濟體制，認為在整個社會中瀰漫著[[朝鮮的個人崇拜|個人崇拜]]思想，而[[金日成|金日成]]家族更是在社會主義國家嘗試採納世襲繼承的做法。同時即便是在安全局勢時，由於[[朝鮮人民軍|朝鮮人民軍]]使用絕大多數資源，而讓北韓民眾生活普遍貧困<ref>{{Cite book |author = [[沈大偉|沈大偉]] | title = China's Communist Party: Atrophy and Adaptation |location = 美國奧克蘭 |publisher = [[加利福尼亞大學出版社|加利福尼亞大學出版社]] |date = 2008年4月2日 |pages = 第82頁至第83頁 |ISBN = 978-0520254923 |accessdate = 2015年6月1日 |url = https://books.google.com.tw/books?id=IOW6msgYWzwC&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false |language = en |quote =  }}</ref>，一些分析師還將北韓的現狀與中國過去的文化大革命對比<ref name = "沈大偉 83">{{Cite book |author = [[沈大偉|沈大偉]] |title = China's Communist Party: Atrophy and Adaptation |location = 美國奧克蘭 |publisher = [[加利福尼亞大學出版社|加利福尼亞大學出版社]] |date = 2008年4月2日 |pages = 第83頁 |ISBN = 978-0520254923 |accessdate = 2015年6月1日 | url = https://books.google.com.tw/books?id=IOW6msgYWzwC&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false | language = en | quote =  }}</ref>。多年來中國共產黨一直嘗試透過中國重要經濟基礎設施經驗，說服北韓的執政黨[[朝鮮勞動黨|朝鮮勞動黨]]展開經濟改革政策<ref name = "沈大偉 83"/>。例如在2006年時，中國共產黨邀請[[朝鮮勞動黨總書記|朝鮮勞動黨總書記]][[金正日|金正日]]前往[[廣東省|廣東省]]以展示中國經濟改革的成功<ref name = "沈大偉 83"/>。不過在一般情況下，中國共產黨仍認為朝鮮勞動黨和北韓是共產主義執政黨和社會主義國家的負面例子<ref name = "沈大偉 83"/>。

=== 腐敗問題 ===
中國共產黨黨員幹部腐敗而違法與違紀事件時有發生，對此中國共產黨對不合格黨員會予以開除黨籍處理，其中在2004年被開除黨籍人數約49,000名。到了2006年時人數則略有下降，當年度一共開除了20,000多名違紀黨員，這佔中國共產黨黨員1.7‰<ref>{{Cite web |url = http://politics.people.com.cn/BIG5/1026/4100246.html |title = 中共一年開除兩萬余名違紀黨員 佔總數1.7‰ |author = 孟娜 |date = 2006年2月13日 |publisher = [[人民網|人民網]] |language = zh-Hans |accessdate = 2015年6月1日 |quote =  |archive-url = https://web.archive.org/web/20160304092753/http://politics.people.com.cn/BIG5/1026/4100246.html |archive-date = 2016-03-04 |dead-url = no }}</ref>。不過許多人士認為這是中國共產黨本身的制度問題，只靠黨內紀律處分並不足夠。2007年9月13日，擁有更多職能和權力的[[國家預防腐敗局|國家預防腐敗局]]正式成立，主要的工作有反腐敗宣傳和教育、進行制度的建設、機制與體制的創新，以及在反腐敗案件上取得源頭性解決的工作，試圖能夠解決中國共產黨近年來黨員幹部腐敗案件之問題<ref>{{Cite web |url = http://cpc.people.com.cn/GB/64093/64371/6262134.html |title = 国家预防腐败局揭牌 |author = 姜洁 |date = 2007年9月14日 |publisher = [[中國共產黨新聞網|中國共產黨新聞網]] |language = zh-Hans |accessdate = 2015年6月1日 |quote =  |archive-url = https://web.archive.org/web/20160304083141/http://cpc.people.com.cn/GB/64093/64371/6262134.html |archive-date = 2016-03-04 |dead-url = no }}</ref>。
