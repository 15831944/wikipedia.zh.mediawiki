{{中华人民共和国政治}}

[[File:One_child_policy.jpg|thumb]][[德安县|德安县]]的一个宣传牌]]

'''计划生育'''是[[中华人民共和国|中华人民共和国]]的[[人口控制|人口控制]]政策，属于[[中华人民共和国基本国策|基本国策]]。现行计划生育政策以控制中国人口数量为目的，提倡晚婚晚育，实行限制人口数量的生育政策。

2015年以前，中国大陆计划生育以独生子女政策（[[一孩政策|一孩政策]]）为主，农村夫妻只有一个女孩、[[单独两孩|夫妻双方至少一方为独生子女]]、自治区和一些省的[[少数民族|少数民族]]（[[壮族|壮族]]、满族除外）等几种情况可以生育[[两孩政策|两个子女]]<ref name="steps up">{{cite news |url = http://news.bbc.co.uk/2/hi/asia-pacific/941511.stm |title=China steps up 'one child' policy |publisher=BBC News |date=2000-09-25 |accessdate=2009-07-31|language=en}}</ref>。新疆[[少数民族|少数民族]]农牧民、青海少数民族牧民可以生育三个子女<ref>{{cite web |language =zh-hans |url = http://unn.people.com.cn/GB/5975181.html |title=新疆维吾尔自治区人口与计划生育条例|publisher=人民网|date=2007年7月11日|accessdate=2010年9月18日}}</ref>。[[西藏自治区|西藏自治区]]对藏族实行自愿计划生育政策，但提倡已有三个孩子的夫妇不再生育。

根据国际法，计划生育必须以自愿为前提，以保障[[生育权|生育权]]为目标。

近年来，[[中国大陆人口|中国人口]]的结构问题凸显，[[中国大陆生育率|总和生育率]]接近世界最低水平，[[老龄化|老龄化]]严重，儿童[[性别比|性别比]]极高。计划生育的目标（原本预定仅執行25年左右，已经超过年限）和理论依据（计划未充分考虑各性別、种族的人口失衡）都受到广泛质疑<ref>{{cite news |author = 黄文政 |coauthors = 梁建章 |url = http://opinion.caixin.com/2014-03-06/100647580.html |title = 卫计委应公布计划生育目标和理论依据 |publisher = 财新网 |date = 2014年3月6日 |accessdate = 2014年5月25日}}</ref>，加上2013年推行單独二孩政策失敗后因错判形势而备受指责。2015年10月26日到29日中共在[[中國共產黨第十八屆中央委員會第五次全體會議|中共中央十八届五中全会]]宣布[[兩孩政策|全面二胎]]，在中国大陆实行35年的一胎化政策正式走入历史<ref>[http://www.storm.mg/article/72597 風傳媒：中國一胎化政策畫下句點！十八屆五中全會決議：一對夫婦可生兩個孩子]</ref><ref>{{cite news|url=https://tw.news.yahoo.com/105401078.html|date=2015-10-29|accessdate=2015-10-29|publisher=Yahoo奇摩（即時新聞）|title=中國大陸結束一胎化 實施全面二胎}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2018年5月，消息指，国务院最快将于年内取消计划生育政策，以减缓[[人口老龄化|人口老龄化]]趋势，执行了四十年的強制計划生育有望结束。

==历史==
{{main|中华人民共和国计划生育史}}
===1950年代至1970年代：从“赞成节育”到“最多两个”===
1954年12月，[[刘少奇|刘少奇]]召集国务院多个单位的负责人举行了节制生育问题的座谈<ref>{{cite book |title=刘少奇选集·下卷|chapter=提倡节育|editor=中共中央文献编辑委员会 |location=北京 |publisher=人民出版社 |date=1985年12月 |isbn=9787010039619 |url = http://cpc.people.com.cn/GB/64184/64185/66614/4488868.html |accessdate=2014-01-30 }}</ref>。1955年3月，中共中央在批示中表示“赞成适当地节制生育”<ref>{{cite web |title=中共中央对卫生部党组关于节制生育问题的报告的批示 |url=http://news.xinhuanet.com/ziliao/2004-12/29/content_2391355.htm |accessdate=2014-01-30}}</ref>。[[三年困难时期|三年困难时期]]结束以后，1962年12月，中共中央、国务院发出《关于认真提倡计划生育的指示》，提出“在城市和人口稠密的农村提倡节制生育”<ref>{{cite web|author=中共中央、国务院|title=关于认真提倡计划生育的指示|url=http://news.xinhuanet.com/ziliao/2002-03/04/content_2504428.htm|publisher=[[新华网|新华网]]|date=1962-12-28|accessdate=2014-08-08}}</ref>。

1973年12月，第一次全国计划生育工作汇报会确定了“晚、稀、少”方针，要求晚婚、一对夫妇生育两个孩子、间隔四年左右，计划生育工作全面推行。1978年10月，中共中央批转国务院计划生育领导小组会议报告，“提倡一对夫妇生育子女数最好一个最多两个，生育间隔三年以上。”<ref name="梁中堂">{{cite journal|author=梁中堂|title=艰难的历程：从“一胎化”到“女儿户”|journal=开放时代|year=2014|issue=3|url=http://www.opentimes.cn/bencandy.php?fid=375&aid=1806}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

===1980年代：一胎化===
70年代末期至80年代初，生育政策发生了革命性的变化，即从“晚稀少”到“一胎化”，再到“独女户”。

1978年2月26日，[[第五届全国人民代表大会第一次会议|第五届全国人民代表大会第一次会议]]召开。华国锋总理在《政府工作报告》中说：“计划生育很重要。有计划地控制人口增长，有利于国民经济的有计划发展，有利于保护母亲和儿童的健康，有利于广大群众的生产、工作和学习。必须继续认真抓好，争取在三年内把我国人口自然增长率降到百分之一以下”。1978年6月26-28日，新一届[[国务院计划生育领导小组|国务院计划生育领导小组]]召开第一次会议。[[中共中央政治局|政治局候补委员]]、国务院副总理[[陈慕华|陈慕华]]担任国务院计划生育领导小组组长。领导小组成员来自国务院所属的[[国家计划委员会|国家计划委员会]]、[[国家科委|国家科委]]、[[中华人民共和国财政部|财政部]]、[[中华人民共和国卫生部|卫生部]]等部委的领导人员。国务院计划生育领导小组第一次会议着重研究了贯彻落实华国锋主席在五届人大一次会议上提出的三年内把我国人口自然增长率降到百分之一以下的任务。1978年10月26日，中共中央批转《关于国务院计划生育领导小组第一次会议的报告》，即中发（1978）69号文件，提出:
<blockquote>
五届人大政府工作报告中提出，‘争取在三年内将我国人口自然增长率降到百分之一以下’，发展国民经济十年规划纲要中规定，从一九八一年到一九八五年把全国人口自然增长率降至千分之九左右，这个奋斗目标，是新时期总任务赋予全党全军和全国人民的一项艰巨而又光荣的任务。

解决好有关计划生育的各项政策。

晚育年龄，农村提倡女23周岁，男25周岁结婚，城市略高于农村。提倡一对夫妇生育子女数最好一个最多两个。生育间隔三年以上。

对于计划生先进个人和集体。要给予表扬和鼓励。对个别干部、党团员、职工，不实行晚婚、节育，要进行教育，在群众中影响很坏的，要开展批评和自我批评；情节恶劣的，要给予纪律处分。对破坏计划生育的阶级敌人要及时揭露，坚决打击。有关农村口粮分配，城市住房分配等社会经济政策和其他一些规定，要有利于计划生育工作的开展。
</blockquote>

1979年1月，在成都召开全国计划生育办公室主任会议，讨论了进一步贯彻党中央批转的国务院计划生育领导小组第一次会议报告精神，争取1980年把我国人口自然增长率降到百分之一以下的具体措施。此次会议第一次将计划生育作为政府倡导的一项活动，改变成为一种需要政府通过强有力的行政手段才能够推进的工作。要求在全国性法律颁布之前，各个省、直辖市、自治区先行制订地方的计划生育试行条例。此后，各地都出台了计划生育暂行规定，对违规生育者给予经济和行政的严厉处罚。很快，计划生育政策又变为要求一对夫妇只生一胎。1979年下半年起，多地又按照一胎化的要求修改计划生育规定。1979年一年里，全国29个省区市有27个出台了本省的计划生育条例，明确规定用经济的、行政的和法律的手段控制生育。除部分少数民族外，一胎化在全国城乡全面实行，仅云南、青海、宁夏、新疆农村可生育两孩。

1979年8月11日，国务院副总理陈慕华在《人民日报》发表《实现四个现代化，必须有计划地控制人口增长》长篇文章，提出20世纪末的人口目标和分两个阶段的发展方案：“从我国的实际情况出发，长期地、自觉地、有计划地控制人口增长，这就是我们的方针。我们必须坚持这个方针，争取本世纪末做到人口自然增长率为零，即人口不增长。为了达到这个目标，我们设想分两个阶段来努力。第一个阶段，争取到一九八五年把人口自然增长率从现在的千分之十二降到千分之五左右。第二个阶段，争取在二〇〇〇年人口自然增长率降到零。……必须大力提倡和推广一对夫妇只生一个孩子。这是使人口自然增长率降低到零的主要办法，也是群众可以接受的办法。”

1979年12月15-20日，国务院计划生育领导办公室在成都召开各省、市、自治区计划生育办公室主任会议，为确保1980年和1981年的人口自然增长率降到9.5‰和8‰的目标，“目前当务之急是尽快把计划生育工作的重点转移到提倡和奖励一对夫妇只生一个孩子的工作上来，逐年提高生一胎的比例，限制多胎生育。”

1980年2月2日，国务院计划生育领导办公室召开的婚姻、家庭、计划生育新风尚座谈会上提出：“只有逐步做到城市95%、农村90%的育龄夫妇只生一个孩子，到本世纪末，我国总人口才能控制在12亿左右。”

1981年3月，[[国家计划生育委员会|国家计划生育委员会]]成立，副总理[[陈慕华|陈慕华]]兼任首任国家计生委主任。1983年5月，[[钱信忠|钱信忠]]任[[国家计划生育委员会|国家计划生育委员会]]主任，他提出“一胎[[宫内节育器|上环]]，二胎[[绝育|绝育]]”，1983年创下上环、绝育、人工流产手术数量的多项记录。

1984年起，计划生育政策有所调整。各地陆续实施夫妻双方均为独生子女可以生育两个孩子的政策（[[双独两孩|双独两孩]]），仅四省区实施较晚：湖北、甘肃、内蒙古2002至2003年，河南2011年。多数省份对满足一定条件的农户实行只有一个女孩可以生育第二胎的政策（[[一孩半|一孩半]]）。

===1990年代至2010年代：一票否决制和极低生育率===
1991年5月，中共中央、国务院做出《关于加强计划生育工作、严格控制人口增长的决定》<ref>{{cite web |title=中共中央、国务院关于加强计划生育工作严格控制人口增长的决定 |url = http://news.xinhuanet.com/ziliao/2005-02/17/content_2587830.htm |date=1991-05-12|accessdate=2014-08-08}}</ref>，各地推行计划生育一票否决制。随着计划生育工作加强，中国大陆的[[中国大陆生育率|总和生育率]]快速下降。1990年代初，生育率下降到世代更替水平以下。[[中华人民共和国第五次全国人口普查|2000年人口普查]]显示，生育率仅为1.22，处于世界最低水平。

计划生育工作并没有因为生育率极低而放松，相反的反而認為政策執行十分成功。2000年3月和2006年12月，中共中央、国务院两次做出加强人口与计划生育工作、稳定低生育率水平的决定<ref>{{cite web |title = 中共中央、国务院关于加强人口与计划生育工作稳定低生育水平的决定 |url = http://cpc.people.com.cn/GB/64162/71380/71382/71481/4854373.html |date=2000-03-02 |accessdate=2014-08-08}}</ref><ref>{{cite web |title = 中共中央国务院关于全面加强人口和计划生育工作统筹解决人口问题的决定 |url = http://news.xinhuanet.com/politics/2007-01/22/content_5637713.htm |date=2007-01-22|accessdate=2014-08-08}}</ref>。[[中华人民共和国第六次全国人口普查|2010年人口普查]]显示，中国大陆的[[中国大陆生育率|总和生育率]]仅为1.18，仍然处于世界最低水平。

2011年11月，河南成为最后一个实施[[双独两孩|双独两孩]]政策的省份，但生育率没有提高。

===2013年至2015年秋：放開单独二孩政策與推廣結束一胎化===

2013年11月，中央决定坚持计划生育的基本国策，启动实施一方是独生子女的夫妇可生育两个孩子的政策（[[单独两孩|单独两孩]]），逐步调整完善生育政策，促进人口长期均衡发展<ref>[http://news.xinhuanet.com/politics/2013-11/15/c_118164489.htm 中国放开“单独二胎”] 人民网 2013-11-15 </ref><ref>[http://www.apdnews.com/news/46447.html 中国放开“单独两孩”将改变什么] {{webarchive|url=https://web.archive.org/web/20131121054655/http://www.apdnews.com/news/46447.html |date=2013-11-21 }}，亚太日报，2013年11月18日</ref>。2014年，各地陆续实施单独两孩政策，但多年的计划生育教育使政策遇冷，当年申请数仅106.9万，大大低于官方预期200萬對以上，加上計生委自開始打貪後，屢傳官員因貪腐落馬新聞，此事使得計劃生育受到人民廣泛質疑。

2015年，全国21所高校及科研机构的39位人口学者联名提交建议书，呼吁全面放开二孩生育、取消对公民的生育限制<ref>{{cite news|url=http://china.caixin.com/2015-03-02/100786969.html|title=放开二孩呼声再起 39学者吁促生育自由|date=2015-03-02|accessdate=2015-03-10|publisher=[[财新网|财新网]]}}</ref>；50位法律学者联名提交建议书，呼吁尽快废除生育审批制度，取消社会抚养费制度，实现公民自主生育<ref name="废止生育审批">{{cite news|url=http://news.qq.com/a/20150322/036597.htm|date=2015-03-22|accessdate=2015-03-29|publisher=[[腾讯|腾讯]]新闻|title=50名法学教授联名建议：还生育权于民 取消社会抚养费}}</ref>。在中國網路、電視傳媒上對人口老化數據的評判分析也逐漸增加，顯示一胎化的政策開始動搖。

===2015年秋：全面二孩政策實施===
在10月29日上的[[中国共产党第十八届中央委员会第五次全体会议|中共中央十八届五中全會]]突然決議：全面实施一对夫妇可生育两个孩子政策。<ref>[http://news.xinhuanet.com/politics/2015-10/29/c_128373482.htm 五中全会公报建议实施全面二孩政策] 新华网</ref>一胎化在[[中國大陸|中國大陸]]正式廢除。<ref>[http://news.163.com/15/1029/18/B746VT6C0001124J.html 中共全会公报允许普遍二孩政策] 网易 2015-10-29 </ref>但实际上，二孩政策的正式落地，除了要通过中共中央的決定，还要通过[[中华人民共和国国务院|國務院]]制定調整意見，[[全國人大常委會|全國人大常委會]]批准、各地實施方案報批、并由各[[中国一级行政区|一级行政区]][[人大常委会|人大常委会]]修訂計生條例这三个步骤。截至2015年秋，中国大陆有17個一级行政区仍实行一胎制<ref>{{cite news|title=全面放開二孩至少經歷4關鍵程序 搶生違法 |url=http://xj.people.com.cn/BIG5/n/2015/1030/c188514-26969049.html |publisher=重慶晨報  |date=2015年10月30日 |accessdate=2015年10月30日 }}</ref>。媒體預估，在2016年第一季度，中国大陆的多数一级行政区都能够具体实施全面二孩政策<ref>{{cite news |title=全面二孩预计明年一季度落地 只生一个将无优待 |url=http://china.huanqiu.com/article/2015-10/7880054.html?from=bdwz |publisher=新京报 |date=2015年10月31日 |accessdate=2015年10月31日 |deadurl=yes |archiveurl=https://web.archive.org/web/20151222083917/http://china.huanqiu.com/article/2015-10/7880054.html?from=bdwz |archivedate=2015年12月22日 }}</ref>。

== 概况 ==
===生育政策===
2015年以前的计划生育提倡一对夫妻生育一个子女，因此又称为“独生子女政策”、“一胎化政策”或“一孩政策”<ref>{{cite news |url = http://www.voanews.com/chinese/2009-07-25-voa33.cfm |title=上海没有改变一胎化政策 |author=申华 |publisher=美国之音中文网 |date=2009-07-25 |accessdate=2009-07-31}}</ref><ref>{{cite news|url=http://news.chinatimes.com/2007Cti/2007Cti-Rtn/2007Cti-Rtn-Content/0,4526,110109+112009073000980,00.html|title=官方統計中國每年墮掉一千三百萬胎|author=夏明珠|work=中國時報|publisher=中时电子报|date=2009-07-30|accessdate=2009-07-31}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。允许生育第二个孩子的常见情况有：
* 第一胎为无法成长为劳动力的非遗传性残疾儿；
* 再婚夫妇合计只有一个子女；
* 夫妻一方为独生子女（[[单独两孩|单独两孩]]）；
* 农村夫妇只有一个女儿（[[一孩半|一孩半]]，适用于19省份，直辖市和江苏、四川除外）<ref>{{cite web |url=http://www.china.org.cn/english/2002/Oct/46138.htm |title=Family Planning Law and China's Birth Control Situation|title_zh=计划生育法律和中国的生育控制情况 | publisher=中国网 | author=Hu Huiting |date=2002-10-18 | accessdate=2009-08-01|language=en}}</ref>；
* 海南、宁夏、新疆的农村夫妇；
* 云南（大中城市郊区、人口稠密或者生态恶化地区从严）、青海的农村困难户；
* 自治区和一些省的少数民族夫妇（[[壮族|壮族]]有时除外）。
具体生育政策由省级人大常委会制定，各省份之间有一定差别<ref>{{cite news|url=http://www.fj.xinhuanet.com/dszx/2005-03/21/content_3911756.htm|title=19种人享受“二胎政策”|publisher=新华网福建频道|work=泉州晚报|date=2005-03-21|author=陈文钊|accessdate=2009-08-01|deadurl=yes|archiveurl=https://web.archive.org/web/20050324171613/http://www.fj.xinhuanet.com/dszx/2005-03/21/content_3911756.htm|archivedate=2005-03-24}}</ref><ref>{{cite news|url=http://health.chinanews.cn/jk/zcdt/news/2008/03-07/1185590.shtml|title=江西修订计划生育条例 11种情形可再生第二胎|date=2008-03-07|publisher=中国新闻网|work=新法制报|author=陈佳|accessdate=2009-08-01}}</ref><ref>{{cite news|url=http://fzwb.ynet.com/article.jsp?oid=4544446|title=广州：留学一年以上并怀孕回国生二胎不算超生|date=2005-01-21|work=法制晚报|publisher=北青网|accessdate=2009-08-31|deadurl=yes|archiveurl=https://web.archive.org/web/20110718142316/http://fzwb.ynet.com/article.jsp?oid=4544446|archivedate=2011-07-18}}</ref><ref>{{cite news|url=http://www.sh.xinhuanet.com/2009-07/23/content_17176327.htm|title=上海: 鼓励符合条件夫妻生育第二个子女|work=文汇报|publisher=新华网上海频道|date=2009-07-23|accessdate=2009-07-31|deadurl=yes|archiveurl=https://web.archive.org/web/20100116110121/http://www.sh.xinhuanet.com/2009-07/23/content_17176327.htm|archivedate=2010-01-16}}</ref>。新疆的少数民族农牧民、青海少数民族牧民可生育三个子女，西藏农牧区提倡最多生育三个子女（不执行强制的计划生育政策）。

根据[[国家人口和计划生育委员会|国家人口计生委]]2007年估计，中国人口中，适用一孩政策的占35.9%，适用一孩半政策的占52.9%，适用两孩政策的占9.6%，适用多孩政策的占1.6%。<ref>[ http://news.sina.com.cn/c/2007-01-23/121512115296.shtml 计生委：中国计划生育不是一胎化政策]</ref><ref>{{cite news |url = http://news.xinhuanet.com/politics/2007-07/10/content_6354501.htm |title=于学军：除豫以外各省区市独生子女夫妻可生二胎|date=2007-07-10 |accessdate=2009-07-31|publisher=新华网}}</ref><ref>{{cite news |url = http://www2.chinadaily.com.cn/china/2007-07/11/content_5432238.htm |title = Most people free to have more child |date = 7/11/2007 |accessdate = 2008-10-30 |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20081017133413/http://www.chinadaily.com.cn/china/2007-07/11/content_5432238.htm |archivedate = 2008-10-17 }}</ref>。政策生育率约为1.5，也就是说根据政策一对夫妇平均可生育1.5个孩子。

生育第二胎通常需要受到[[生育间隔|生育间隔]]期（通常为3或4年）的限制，但近年来一些省份已放宽或取消了生育间隔的规定。<ref>{{cite news|url=http://www.gd.xinhuanet.com/sungov/2008-11/29/content_15050958.htm|title=粤“生二胎”条件适当放宽 取消四年间隔期|work=广州日报|author=翁淑贤|date=2008-11-29|accessdate=2009-08-01|publisher=新华网广东频道|deadurl=yes|archiveurl=https://web.archive.org/web/20140301103841/http://www.gd.xinhuanet.com/sungov/2008-11/29/content_15050958.htm|archivedate=2014-03-01}}</ref><ref>{{cite news |url = http://health.chinanews.cn/jk/zcdt/news/2007/09-30/1040515.shtml |title=湖南修改计划生育条例　取消合法再生生育间隔 |work=三湘都市报 |publisher=中国新闻网|author=胡忠民|date=2007-09-30 |accessdate=2009-08-01}}</ref><ref>{{cite news |url = http://www.chinanews.com.cn/n/2003-08-07/26/332948.html|title=北京适当放宽二胎生育政策 新条例9月1日起施行 |date=2003-08-07 |author=刘江、李珍玉 |publisher=中国新闻网 |accessdate=2009-08-02}}</ref><ref>{{cite news |url = http://news.sina.com.cn/c/2004-10-16/05473937532s.shtml |title=我国放宽二胎间隔 |date=2004-10-16|work=华商晨报|publisher=新浪网|accessdate=2009-08-02}}</ref>

[[香港|香港]]和[[澳门|澳门]]两个[[特别行政区|特别行政区]]并不实行计划生育政策，但中央对大陆居民涉在香港生育问题有着相关规定<ref>{{cite web|language=zh-hans|url=http://www.cgrkw.com/newsshow.asp?ArticleID=182|title=国家计划生育委员会关于中国内地居民涉外生育问题的规定|publisher=兰州市城关区人口和计划生育局|date=2006-09-26|accessdate=2010年9月18日|deadurl=yes|archiveurl=https://web.archive.org/web/20140303101243/http://www.cgrkw.com/newsshow.asp?ArticleID=182|archivedate=2014年3月3日}}</ref>。在外国出生的子女若未成为[[中国国籍|中国公民]]则不属于计划生育政策的计算范围，许多居民选择[[赴美生子|赴美生子]]等[[生育旅行|生育旅行]]。<ref>{{cite news|url=http://www.chinadaily.com.cn/china/2006-12/28/content_770107.htm|title=Are the rich challenging family planning policy?|title_zh=富人在挑战计划生育政策吗？|author=Guo Qiang|publisher=中国日报|date=2006-12-28|accessdate=2009-08-01|language=en}}</ref>

===节育措施===

===社会抚养费===

{{main|社会抚养费}}
2002年以前，超生的居民会被处以数额不等的[[罚款|罚款]]<ref>{{cite news|url = http://www.china.com.cn/law/txt/2001-12/31/content_5092743.htm|title=计划生育罚款被取消违规生育者缴纳社会抚养费|work=中国青年报|publisher=中国网|date=2001-12-31|accessdate=2009-07-31}}</ref>。2002年9月1日实施《[[中华人民共和国人口与计划生育法|中华人民共和国人口与计划生育法]]》后，违规生育子女的居民被要求依法缴纳“[[社会抚养费|社会抚养费]]”<ref>{{cite web |url = http://www.gov.cn/banshi/2005-08/21/content_25059.htm |title=中华人民共和国人口与计划生育法 |publisher=中国政府门户网站 |author = [[第九届全国人民代表大会常务委员会|第九届全国人民代表大会常务委员会]] |date=2001-12-29 |accessdate=2009-07-31}}</ref>，其中社会抚养费的征收标准按省级（即省、自治区和直辖市）规定，主要参考的是当地人均收入，当事人的实际收入以及超生情节<ref name="社会抚养费征收管理办法">{{cite web |url = http://www.gov.cn/zwgk/2005-06/02/content_3793.htm |title=社会抚养费征收管理办法|publisher=中国政府网 |date=2002-08-02 |accessdate=2009-07-31}}</ref> 。例如，在湖南省，违法多生一名子女的居民，将被征收的社会抚养费为双方上年度总收入的六到十倍；而重婚、婚外生育或生育更多子女的，受征收的倍数还会更高。<ref>{{cite news|url=http://news.xinhuanet.com/legal/2009-03/18/content_11030130.htm|title=湖南“重典”遏制富人超生 大幅提高社会抚养费|date=2009-03-18|accessdate=2009-08-02|publisher=新华网湖南频道}}</ref>

在收取社会抚养费的过程中，曾发生邵阳计生办公务人员为获得这一费用来作为地方财政收入而进行的[[邵阳当局没收婴儿事件|邵阳当局没收婴儿事件]]。

具体执法过程以湖南省双峰县为例 婴儿出生后，医院开出的出生证明直接与计生部门联网。如有超生，计生部门会在一个月左右送达《社会抚养费征收告知书》，过一段时间后会送达《社会抚养费催告书》，双峰是贫困县为了尽可能多收钱，社会抚养费征收单上缴费比例直接按照一年前人均收入6倍计算。如果按照要求缴费则会下达《社会抚养费征收决定书》，相当于收据。如果拒不执行缴费，计生部门会要求地方人民法院强制征缴。强制征缴手段包括直接从个人银行账户划扣，冻结个人存款，拉黑个人征信报告。

==法律问题==
===国际法===
{{see|生育权}}
国际公约和决议明确规定，[[计划生育|计划生育]]必须以自愿为前提。1968年联合国国际人权会议通过的《德黑蘭宣言》提出，“父母享有自由負責决定子女人數及其出生時距之基本人權”<ref name="德黑兰">{{cite report|title=國際人權會議蕆事文件|publisher=聯合國|id=A/CONF.32/41|url=http://www.un.org/Docs/asp/ws.asp?m=A/CONF.32/41|page=4|at=一六|date=1968-05-13|accessdate=2015-04-19}}</ref>。1969年联合国大会通过的《社會進步及發展宣言》重申，“父母有自由並負責決定其子女人數及生育間距之專有權利”<ref>{{cite report|title=社會進步及發展宣言|url=http://www.un.org/Docs/asp/ws.asp?m=A/RES/2542%20%28XXIV%29|id=A/RES/2542 (XXIV)|at=第四條|publisher=聯合國|date=1969年12月21日|accessdate=2014-12-06}}</ref>。中国1980年批准的《[[消除对妇女一切形式歧视公约|消除对妇女一切形式歧视公约]]》规定，男女“有相同的权利自由负责地决定子女人数和生育间隔”<ref>{{cite web|url=http://www.un.org/womenwatch/daw/cedaw/text/0360794c.pdf|title=消除对妇女一切形式歧视公约|at=第16条|date=1979|accessdate=2014-12-06}}</ref>。联合国《国际人口与发展会议行动纲领》指出，计划生育“不应采用任何形式的强迫形式。……不应以指标或配额方式强迫推行计划生育。”<ref>{{cite report|url=http://www.un.org/Docs/asp/ws.asp?m=A/CONF.171/13/Rev.1|title=国际人口与发展会议的报告|date=1995|accessdate=2015-04-29|publisher=联合国|id=A/CONF.171/13/Rev.1|page=39|at=7.12节}}《行动纲领》的正式语言是英文，原文：{{cite report|url=http://www.un.org/popin/icpd/conference/offeng/poa.html|quote=Any form of coercion has no part to play. (...) Demographic goals (...) should not be imposed on family-planning providers in the form of targets or quotas for the recruitment of clients.|title=Programme of Action of the International Conference on Population and Development|accessdate=2015-04-19}}</ref>

===国内法===
[[中华人民共和国宪法|中华人民共和国宪法]]、[[法律|法律]]、法规对计划生育都有规定，但与上述[[国际条约|国际条约]]和决议多有不和之处。参与制定《人口与计划生育法》的法学专家[[湛中乐|湛中乐]]教授2010年与多位学者联名发起《尽快启动〈人口与计划生育法〉全面修改的公民建议书》，建议全面修改《人口与计划生育法》，明确[[生育权|生育权]]是基本[[人权|人权]]，取消生育权限制，废止生育审批制度，取消社会抚养费制度<ref>{{cite news | author=湛中乐等 | title=《尽快启动〈人口与计划生育法〉全面修改的公民建议书》（全文） | url=http://china.caixin.com/2012-07-13/100410868.html | publisher=[[财新网|财新网]] | date=2012年7月13日 | accessdate=2014年5月26日 }}</ref>。2015年3月，包括[[江平_(学者)|江平]]在内的50位法律学者联名提交建议书，呼吁尽快废除生育审批制度，取消社会抚养费制度，实现公民自主生育<ref name="废止生育审批"/>。

====宪法====
1978年[[宪法|宪法]]第五十三条规定“国家提倡和推行计划生育”。

现行1982年《[[中华人民共和国宪法|中华人民共和国宪法]]》对计划生育有多项规定：
#第25条规定“国家推行计划生育，使[[人口|人口]]的增长同[[经济|经济]]和社会发展计划相适应”
#第49条规定“夫妻双方有实行计划生育的义务”
#第89条规定“[[国务院|国务院]]领导和管理全国计划生育工作”
#第107条规定“[[县级|县级]]以上地方各级[[人民政府|人民政府]]管理本行政区域的计划生育[[行政|行政]]工作”

2004年3月14日宪法修正案在宪法第三十三条中增加“国家尊重和保障人权”的规定。

====法律====
2001年12月29日人大常委会通过《中华人民共和国人口与计划生育法》，2002年9月1日施行。

《人口与计划生育法》第二条规定“实行计划生育是国家的基本国策”，第十七条规定“公民有生育的权利，也有依法实行计划生育的义务”。第十八条规定“提倡一对夫妻生育一个子女；符合法律、法规规定条件的，可以要求安排生育第二个子女。具体办法由[[省|省]]、[[自治区|自治区]]、[[直辖市|直辖市]][[人民代表大会|人民代表大会]]或者其常务委员会规定。”第四十一条规定“不符合本法第十八条规定生育子女的公民，应当依法缴纳[[社会抚养费|社会抚养费]]。”第四十二条规定“按照本法第四十一条规定缴纳社会抚养费的人员，是国家工作人员的，还应当依法给予行政处分；其他人员还应当由其所在单位或者组织给予纪律处分。”

《人口与计划生育法释义》指出“国家推行计划生育，对公民来说不是强制性义务，是倡导性义务”。<ref>{{cite web|url=http://www.npc.gov.cn/npc/flsyywd/shehui/2003-09/11/content_321085.htm|title=中华人民共和国人口与计划生育法释义|at=第十八条|publisher=全国人大|date=2003-09-11|accessdate=2014-12-07}}</ref>

参与制定《人口与计划生育法》的[[湛中乐|湛中乐]]教授认为，中国没有任何现行法律规定每对夫妇只能生一个孩子。他指出，一胎政策变成强制性政策，缺乏严格的法律依据。<ref>{{cite news | title=湛中乐：一胎政策有法律依据吗？ | url=http://www.caixin.com/2012-05-01/100385610.html | publisher=[[财新网|财新网]] | date=2012年5月1日 | accessdate=2014年5月26日 }}</ref>

====法规、规章====
国务院制定的现行计划生育类[[行政法规|行政法规]]有三部：
*《[[社会抚养费|社会抚养费]]征收管理办法》，2002年8月2日公布，2002年9月1日起施行。第三条规定“社会抚养费的征收标准，分别以当地城镇居民年人均可支配收入和农村居民年人均纯收入为计征的参考基本标准，结合当事人的实际收入水平和不符合法律、法规规定生育子女的情节，确定征收数额。社会抚养费的具体征收标准由省、自治区、直辖市规定。”
*《计划生育技术服务管理条例》，2001年6月13日公布，2001年10月1日起施行，2004年12月10日修改。
*《流动人口计划生育工作条例》，2009年5月11日公布，2009年10月1日起施行。1991年12月26日发布的《流动人口计划生育管理办法》和1998年9月22日发布、1999年1月1日起施行的《流动人口计划生育工作管理办法》已废止。

国家计划生育主管部门还制定了多项计划生育类部门规章<ref>{{cite web|title=计生类部门规章|url=http://www.nhfpc.gov.cn/zhuzhan/jslbmgz/lists.shtml|publisher=[[中华人民共和国国家卫生和计划生育委员会|中华人民共和国国家卫生和计划生育委员会]]|accessdate=2014年5月26日|deadurl=yes|archiveurl=https://web.archive.org/web/20140528010537/http://www.nhfpc.gov.cn/zhuzhan/jslbmgz/lists.shtml|archivedate=2014年5月28日}}</ref>。

除[[西藏自治区|西藏自治区]]外，各省级行政区都有人口与计划生育条例（地方性法规）。计划生育类省级地方政府规章一般有：社会抚养费征收管理办法、流动人口计划生育管理规定、生育服务证管理办法。一些自治州有执行省级人口与计划生育条例的变通规定。

==影响==
计划生育对中国的[[人口结构|人口结构]]、[[中国大陆人口|人口总量]]、生育文化、[[中华人民共和国经济|国民经济]]、社会保障、国家安全都产生了很大影响。它使中国的[[生育率|生育率]]降到极低水平，[[老龄化|老龄化]]日趋严重，出生婴儿男女[[性别比|性别比]]畸高。未来中国人口总量将急剧减少，经济增长受到限制，国家安全受到威胁<ref>{{cite web |author1=黄文政|author2=梁建章|title=人口雪崩——低生育率威胁国家安全（一） |url = http://opinion.caixin.com/2014-02-17/100639366.html|date=2014-02-17|accessdate=2014-09-07}}</ref>。

===人口总量===
[[中国人民大学|中国人民大学]][[社会与人口学院|社会与人口学院]]院长[[翟振武|翟振武]]认为，计划生育使中国人口减少了4亿。<ref>[http://www.tianjinwe.com/tianjin/tjyl/201110/t20111027_4477329.html 世界70亿人口日推迟5年] {{webarchive|url=https://web.archive.org/web/20140227150438/http://www.tianjinwe.com/tianjin/tjyl/201110/t20111027_4477329.html |date=2014-02-27 }}</ref>根据中国计生委相关机构研究，截至2000年，共少生2.5亿孩子<ref name="steps up"/><ref>{{cite news|url=http://dailynews.sina.com.cn/china/2000-09-22/129501.html|title=我国实行计划生育20年少生2.5亿个孩子|work=光明日报|author=薛冬|publisher=新浪网|date=2000-09-22|accessdate=2009-07-31|deadurl=yes|archiveurl=https://web.archive.org/web/20110707022114/http://dailynews.sina.com.cn/china/2000-09-22/129501.html|archivedate=2011-07-07}}</ref>。[[清华－布鲁金斯公共政策研究中心|清华－布鲁金斯公共政策研究中心]]主任王丰和美国[[北卡罗莱纳大学|北卡罗莱纳大学]]的助理教授[[蔡泳|蔡泳]]认为，在1979年到2009年期间，中国的计划生育政策避免了2亿人口的出生。<ref>[http://www.voanews.com/chinese/news/20111027-china-population-132737023.html 中国的计划生育究竟导致少生多少人？]</ref>

若继续维持现行生育政策，有人基于2000年为起算时间，以总和生育率水平为1.46计算，认为2300年[[中国人口|中国人口]]将降至7500万<ref>[http://business.sohu.com/20051215/n240983294.shtml 生育率太低 300年后中国人口将只剩下2800万？][[搜狐|搜狐]] 2005-12-15</ref>。

===人口结构===
====年龄结构====
一胎化政策造成中國[[421结构|「四二一」扭曲型家庭結構]]（意即四名祖父母與二名父母皆由一名獨生子女扶養），讓年輕夫婦背負沉重的扶養負擔，也讓社會高齡化問題日益嚴重。

聯合國估计，到2050年中国人口中三分之一年龄将超过60岁，只有48%的人处于工作年龄。

[[政協|政協]]人口資源環境委員會副主任[[鄭斯林|鄭斯林]]聲稱：中国人口老龄化和实行计划生育有关系，但这只是中国老龄化的原因之一，而不是全部；若不實行計劃生育，中國將會面臨更急迫的人口問題。<ref>{{cite news|url=http://news.cctv.com/china/20070310/101340.shtml|title=郑斯林:中国老龄化和计划生育有关但非全部原因|publisher=CCTV|date=2007年3月10日|accessdate=2007年8月23日}}</ref>

====男女比例====
由於傳統重男輕女觀念和客观条件的限制，一胎化政策造成男嬰比例的迅速增加，尤其是农村地区。依现行政策规定，农村人如果第一胎是男孩则不允许第二胎，如果第一胎是女孩，则允许第二胎。此政策本身不至於造成嚴重性別失衡，但是会有更多的农村家庭因此會違法选择胎儿性别的终止妊娠。所以在传统重男轻女的观念和社会保障不够要养儿防老的现实下，胎儿性别选择雖然違法，但非常常見——普遍是在农村等地——未來將會出現許多找不到妻子的男性。有學者及[[女性主義|女性主義]]者指出，女性過少對女性造成的傷害很明顯，如女性面臨更嚴重的人口販賣問題，[[性別選擇|性別選擇]]對母親的身心傷害；但異常的[[性別比|性別比]]對男性的傷害也很嚴重，這會讓男性心理痛苦、沒有娶妻帶來的成就感、責任感及安定感，大幅提高男性犯罪率、增加戰爭可能。目前中国的男性比女性要多3700万人，而2005年出生的人口男女性别比已经达到118.9:100，男性比女性多20%。<ref>[http://www.gov.cn/jrzg/2007-07/06/content_674626.htm 男性比女性多出3700万—治理"性别失衡"刻不容缓][[中国政府网|中国政府网]] 2007-07-06</ref><ref>{{cite news |url = http://news.xinhuanet.com/politics/2007-07/06/content_6334993.htm |title = 男性比女性多出3700万——治理“性别失衡”刻不容缓|date=2007-07-06|publisher=新华网|author=王君平、徐静|accessdate=2009-09-31}}</ref>这种性别的严重不均衡可能会在未来的二三十年之内导致严重的社会问题，大量「剩男」將找不到妻子。據中共官媒統計，這些「剩男」大約有2400萬至3300萬之間<ref>{{cite web|title=中國社會怪象：剩男缺妻 剩女難嫁|url=http://www.epochtimes.com/b5/16/10/14/n8398445.htm|publisher=大紀元|date=2016-10-15}}</ref>。而現代化社會讓女性有能力拒絕婚姻，更使此問題雪上加霜。

在一项关于中国的研究中，研究员发现女婴的出生情况存在少报或迟报的问题<ref>{{cite journal |author=M. G. Merli |coauthors=A. E. Raftery|year=1990 |title=Are births under-reported in rural China? Manipulation of statistical records in response to China's population policies|journal=Demography|volume=37|issue=2月|page=109－126页|language=en}}</ref>。

====民族构成====
由于汉族计划生育政策比少数民族更加严格，少数民族人口在总人口中的比例持续上升，从1964年人口普查的5.8%上升至[[中华人民共和国第六次全国人口普查|2010年人口普查]]的8.5%<ref>[http://www.gov.cn/test/2005-07/26/content_17366.htm 中国民族][[中国政府网|中国政府网]] 2005-07-26</ref><ref>[http://www.stats.gov.cn/tjsj/tjgb/rkpcgb/qgrkpcgb/201104/t20110428_30327.html 2010年第六次全国人口普查主要数据公报（第1号）] [[国家统计局|国家统计局]] 2011-04-28</ref>。但亦有人指出，这是由于[[民族识别|民族识别]]工作完善造成的，与计划生育无关。1979年认定了云南基诺山的基诺族，自1982年以来，中国民族成分恢复的人数在1200万人以上。年轻人中少数民族比例较高，2010年普查显示14岁以下儿童中少数民族比例已经达到11.3%。未来少数民族人口比例将持续上升，伴隨而來的通婚情況也越發明顯。

===经济===
有评论指目前中國適婚男性多於適婚女性的現象讓這些男性的家庭必須多存錢，這樣才能娶到老婆，中國儲蓄率過高的問題打擊中國及全球經濟。<ref>[http://news.sina.com.tw/article/20091121/2421200.html 中國昔日一胎化 如今導致資本流動不易]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>但指储蓄是为了娶老婆毫无根据，因当前中国人储蓄的目的主要是为了“医疗、教育、住房”三座大山。

有观点认为中国的农业社会中相信“人多力量大”，而且缺乏社会保障，少生将使得家庭经济能力降低。部分农村仍有男丁分田分红等的特权，故农村人都希望多生男儿。有观点认为[[421结构|“四二一”结构]]令一名独生子女能继承更多财产，又避免了子女争产的现象。另外社会有一说法：“越穷越生，越生越穷”，所以政府鼓励“少生优生”。

===国防===
长期计划生育政策可能造成征兵工作的困难，还使独生子成为兵员主体，其作战能力受到广泛质疑。各民族计划生育政策不同，导致边疆地区人口的民族构成改变，最终可能造成国家分裂<ref>{{cite news|author=席志刚|title=兵源之惑 中国强军面临新挑战|url=http://www.infzm.com/content/59932|publisher=南方周末|date=2011-06-03|accessdate=2015-11-09|deadurl=yes|archiveurl=https://archive.is/20151109100834/http://www.infzm.com/content/59932|archivedate=2015-11-09}}</ref><ref>{{cite web|author1=黄文政|author2=梁建章|title=国防之虞——低生育率威胁国家安全(四) |url=http://opinion.caixin.com/2014-02-25/100642709.html|date=2014-02-25|accessdate=2014-09-07}}</ref>。

===公民权利===
{{unreferencedsect}}
中國的計划生育政策被越來越多的國內外民眾及專家學者質疑，被指違背公民生育權，用工具實用主義觀點看待人口問題，後果嚴重。许多人认为“计划生育”和“[[生育控制|家庭计划]]”（Family planning）的最大区别在于“计划生育”是[[中华人民共和国宪法|宪法]]规定的公民义务，除中国计划生育外的[[人口控制|家庭计划]]则是通常可得到政府补助、服务和相关咨询，由民众自愿进行。部分反对者认为，[[社会|社会]]中的生育问题应该由[[家庭|家庭]]自愿决定，[[政府|政府]]无权干预。许多国外[[人权|人权]]组织一直批评计划生育政策侵犯人权。不少地方产后妇女需[[强制上环|强制上环]]，也倍受争议。

支持者认为计划生育是既定法律并实行已久，某些人执意要违法就必须承担后果。既然城镇能成功落实，就不存在农村、流动人员可以放宽的理由。
中国的计划生育的关键是缺乏有效监督。计生部门弹性执法违法执法已成常态，中央各省地市都有自己的一套规章制度。这些制度并没有被计生部门用来遵守而是被在方便的时候引用。

有些家庭会为了生儿子，在第一胎为女儿时，而选择离婚，另行娶妻，这样会多一次机会再生育多一个，而离婚后的女人就变成单亲家庭了。甚至有的家庭对未出生女婴进行非法[[堕胎|堕胎]]，或将婴儿遗弃，甚至杀害。

====强制堕胎====
虽然《中华人民共和国人口与计划生育法》并未规定对于违反计划生育义务的人可以进行强制终止妊娠，即强制堕胎。但在地方执行时，强制堕胎的事件时有发生。一些大月份孕妇因无准生证等原因被迫堕胎，<ref>{{cite web|url=http://news.sohu.com/20100905/n274716388.shtml |title=丈夫未满22妻子被引产 当地计生委称罚款后能生 |publisher=搜狐新闻 |date=2010年9月5日 |accessdate=2013年3月16日 }}</ref><ref>{{cite web|url=http://news.qq.com/a/20120614/000047.htm |title=陕西7月孕妇遭强制引产：交不起4万被逼按手印 |publisher=腾讯新闻 |date=2012年6月14日 |accessdate=2013年3月16日 }}</ref>甚至出现在怀孕九个月时强制堕胎的事件。<ref>{{cite web|url=http://news.panjk.com/201203/20120330354938.shtml |title=重庆磨山计生委强制为怀胎9月孕妇注射流产针 |publisher=平安健康网 |date=2012年3月30日 |accessdate=2013年3月16日 }}</ref>因强制堕胎导致母子双亡的情况也有发生。<ref>{{cite web|url=http://www.rfa.org/mandarin/yataibaodao/hunan-04142009100048.html |title=湖南计生办强制引产花季少女和胎儿双双殒命 |publisher=Radio Free Asia |date=2009年4月14日 |accessdate=2013年3月16日 }}</ref>

====儿童权利====
由于父母坚持自主生育权或是因各种原因（非婚生子、收养）<ref>{{cite web|url=http://news.zgkqw.com/news/system/2011/08/10/010169733.shtml|title=小悦终于见到在医院沉睡8年的出生证副页|publisher=中国柯桥网，来源：绍兴县报 |author=作者：曹健雄，编辑：钱瑶 |date=2011年8月10日|accessdate=2012-05-01 |language=Zh-hans}}</ref>，有些儿童自出生起就无法获得[[中华人民共和国户籍制度|户籍登记]]。“[[黑户|黑户]]”的身份，使他们遭受歧视和不公平待遇<ref>{{cite web|url=http://www.benxi.gov.cn/html/content_1030862.htm?CL=71|title=民警帮大忙 “黑孩” 落户了|publisher=本溪政务网，来源：洞天报|author=洞天报记者：甄映辉|date=2011-01-20|accessdate=2012-05-01|language=Zh-hans|deadurl=yes|archiveurl=https://archive.is/20120723172010/http://www.benxi.gov.cn/html/content_1030862.htm?CL=71|archivedate=2012-07-23}}</ref><ref>{{cite web|url=http://news.sina.com.cn/c/2009-05-08/005815587482s.shtml|title=民警帮大忙 “黑孩”落户了|publisher=新浪网，来源：新文化报|author=记者：刘长宇|date=2009年5月8日 |accessdate=2012-05-01 |language=Zh-hans}}</ref>。如果无法获得户籍登记，在现行制度下，这种受歧视和不公平待遇，将伴随他们终生，甚至有可能“遗传”给他们的子女。一些人非婚生子后，因避免受罚，而遗弃婴儿，也导致出现[[孤儿|孤儿]]的情况。

===城乡户籍的不平等===
城市人口比例：因为城镇居民大多只能生育一胎，而[[农民|农民]]大多可以生育两胎，造成城镇居民人口的萎缩，此举被认为和中国现在的[[城市化|城市化]]进程相矛盾。但是随着大量农民在事实上（即使未获得城市户口）移居城镇，事实上居住在城镇的人口比例在增加。而城镇居民只有一个子女，有些很渴望多生一个，而农民工则普遍生两个甚至超生，都引起城镇居民的不满，认为只是在限制城市人发展。

就学：然而城镇居民严格遵守计划生育，只有一个子女，往往得到更好的教育条件。外来农民工多数生二胎甚至超生，而且外地户口只能在城市入读民办私立学校，收费偏贵。公立学校也允许以赞助费的名义招收外地学生，使教学资源严重供不应求。

就业：目前城市就业形势十分严峻，当独生子女与农民工二代处于同一阶段互相竞争，同时形成互相仇视的心理。

===獨生子女===
{{Unreferenced|time=2015-11-16T03:20:20+00:00}}
“[[80後|80後]]”逐渐成为第一代獨生子女。因为只有一个后代，父母（祖父母）对孩子一般都过分骄纵，导致大多数的独生子女丧生社会生存能力，普遍以自我为中心，缺乏承受挫折的能力。或有的家长认为“学业为重”，而只顾读书，而忽视子女德育的发展。但有观点认为，中国新一代的独生子女所具有的独立的性格特征，无拘无束追求自由的生活态度，将有力地推动中国社会的转型，推动中国社会的民主法制进程。这些都引起了一些社会问题和相关讨论。

2007年3月15日，第十屆[[政協|政協]]會議上，以葉廷芳為首的29名政協委員聯名提案，要求儘快停止執行一胎化，恢復「一個不少，兩個正好」政策。葉廷芳說：「獨生子女缺乏兄弟姊妹，其自然成長的天性受到壓抑，必然導致人性的變異。」

===失独问题===
{{main|失独老人}}
失独老人指独生子去世，不能再生育或收养子女的父母。由于计划生育的实行，独生子女数激增，失独家庭逐渐增多。{{Fact|据统计，2010年中国失独老人达到200万人，而国家对此并没有一套系统的补/救助方案。}}

在中国，失去独生子女对于家庭的打击几乎是毁灭性的。一方面，失去独生子女的父母很容易失去精神寄托，造成一系列严重的心理问题；另一方面，因为国家社会保障体系的不完善，部分人养老需要依靠子女提供资金和照顾，一旦失去唯一的子女，他们很可能会陷入养老危机。失独老人是计划生育政策导致的社会问题之一，在近几年才逐步引起社会关注。

===社会问题===
中国的计划生育反应了中国的一个社会问题，民意和政令相抵触，出现和谐社会不和谐的问题。民众是不会同意制定一套法律来限制自己违背自己的意愿和宪法人权的法令。实际上中国民众是非常抵触和反对计划生育政策，因为大部分的家庭都经历过计划生育政策的影响。单边的计划生育政策只是政府单方面提出的政策，实际上并没有通过广大人民群众商议。中国民众绝大部分的人是有超生，部分政府公务员事业单位国企工作人员只是迫于丢饭碗不敢生。很多政府工作人员学者公知都有对反对计划生育提出自己的见解。计划生育的推行变成了一种绩效考核任务。实际上让基层居民委员会和村民委员会跟民众之间造成很多矛盾和冲突，部分地区都有报道出现殴打暴力冲突。

==部分评价==
===正面评价===
*[[国务院新闻办公室|国务院新闻办公室]]，在1995年发表《中国的计划生育》白皮书，认为中国的计划生育是“符合国情的战略决策、造福于民的社会事业、保障人权的正确选择”<ref>{{cite web|title=中国的计划生育|author=国务院新闻办公室|date=1995-08-23|accessdate=2015-03-25|url=http://www.seac.gov.cn/art/2004/6/25/art_2963_68482.html}}</ref>。

*[[习近平|习近平]]，中国国家主席，认为中国人口众多的基本国情、人口对经济社会发展的压力、人口与资源环境的紧张关系不会根本改变，计划生育基本国策必须长期坚持。<ref name="习近平">{{Cite web |url = http://news.12371.cn/2016/05/18/ARTI1463569857782101.shtml |title =习近平对人口与计划生育工作作出重要指示强调 推动计划生育基本国策贯彻落实 促进人口长期均衡发展与家庭和谐幸福 |date = 2016-05-18 |language = zh |accessdate = 2019-02-14 }}</ref>

*[[李克强|李克强]]，中国国务院总理，称计划生育基本国策促进了人口长期均衡发展，这关系到国家发展和民族未来和关系亿万家庭幸福。并对30多年来计生委做出的贡献予以肯定。<ref name="习近平"/>

*[[蒋正华|蒋正华]]，中国人口专家、[[全国人大常委会|全国人大常委会]]副委员长，认为三十多年来计划生育政策的实施，不论在人口和经济、社会发展等方面，都有重大的成就。<ref>{{Cite web |url = http://www.epochtimes.com/gb/5/8/7/n1010930.htm/amp |title = 中国计划生育政策三十年正反评价不一 |date = 2005-08-07 |language = zh |accessdate = 2019-02-14 }}</ref>

*[[王培安|王培安]]，国家[[卫计委|卫计委]]副主任，认为计划生育政策成就巨大，有效控制了人口过快增长，缓解了人口对资源环境的压力。并指出，如果不实行计划生育政策，现在的中国人口能达到17亿至18亿。后果是人均资源严重不足，环境难以承载发展的需要，经济无法发展到现在这个水平，生活环境、生活质量也将不如现在。<ref>{{Cite web |url = http://opinion.people.com.cn/n/2013/1118/c1003-23575752.html |title = 人民网评：以理性之思，评议计划生育政策 |date = 2013-11-18 |language = zh |accessdate = 2019-02-14 }}</ref>

*李小平，学者，中国计划生育的主要支持者，认为计划生育目的是为了提高中国经济发展，而家庭中第二个孩子是多余的，对中国人均收入提高没有意义。进一步希望在2200年将中国人口可以控制在3亿。<ref>[http://finance.sina.com.cn/economist/jingjixueren/20070406/08553478688.shtml 李小平：多添二胎孩子对中国有经济意义吗] [[新浪|新浪]] 2007-04-06</ref>

===负面评价===
*[[邹至庄|邹至庄]]，經濟學家，普林斯頓大學經濟學講座教授，認為中国人口太多问题已解决，放松控制会促进经济继续发展<ref>{{cite web|url=http://finance.ifeng.com/news/opinion/dlgc/20090609/762537.shtml |title=邹至庄：中國人口問題已解決 放鬆控制會促進經濟發展}}</ref>。
*[[汪蘋|汪蘋]]，中國全國政協委員，[[北京工商大學|北京工商大學]]教授，[[化学工程|化学工程]]和环境工程专家。在中國[[全国政协十一届二次会议|全国政协十一届二次会议]]上，遞交提案《关于“人口政策”适当放开的建议》，認為年輕人生育願望下降，應開放人口政策，解決老年化問題<ref>{{cite web|url=http://www.gqt.org.cn/zhuanti/09gzlh/zxxx/200903/t20090307_172857.htm |title=中国共青团网_年輕一代生育願望下降，汪苹委员：建议适当放开“人口政策”}}</ref><ref>{{cite web|url=http://www.eeo.com.cn/eobserve/Politics/beijing_news/2009/03/27/133566.shtml |title=經濟觀察網_全國政協委員汪蘋：生孩子是對社會做貢獻}}</ref>。
*[[黄敬孚|黄敬孚]]，中國全国政协委员、[[天津市儿童医院|天津市儿童医院]]儿科研究所所长。在2000年3月召开的中國[[全国政协九届三次会议|全国政协九届三次会议]]上，遞交提案《关于建议考虑调整计划生育政策，逐步将独生子女政策改为“二胎”政策的提案》，在2002年3月召开的[[全国政协九届五次会议|全国政协九届五次会议]]上，再次提交了《建议从更全面的角度考虑计划生育政策的提案》<ref name="历届政协会议上的二胎提案">{{cite web |url=http://guancha.gmw.cn/content/2009-05/06/content_917575.htm |title=历届政协会议上的二胎提案 |deadurl=yes |archiveurl=https://web.archive.org/web/20110519114317/http://guancha.gmw.cn/content/2009-05/06/content_917575.htm |archivedate=2011-05-19 }}</ref>。
*[[王翔|王翔]]，中國全國政協委員，[[江西民生集团|江西民生集团]]董事长。在2005年3月召开的中國[[全国政协十届三次会议|全国政协十届三次会议]]上，遞交提案，建议统筹推行“二胎制”<ref name="历届政协会议上的二胎提案"/>。
*[[叶廷芳|叶廷芳]]，中國全国政协委员、[[中国科学院|中国科学院]]研究员。在2007年3月召开的中國[[全国政协十届五次会议|全国政协十届五次会议]]上，叶廷芳等29名委员联名提交提案，要求尽快停止独生子女条例的执行，恢复原先的“一个不少（但必须是自愿），两个正好”的方针 <ref name="历届政协会议上的二胎提案"/>。
*[[温思美|温思美]]，中國全國政協委員，中國[[民盟|民盟]]中央副主席、[[广东省政协|广东省政协]]副主席、[[华南农业大学|华南农业大学]]副校长,教授，农村经济管理学专家。在2009年3月召开的中國全国政协十一届二次会议上，遞交提案《关于调整我国现行人口生育政策的建议》<ref name="历届政协会议上的二胎提案"/>。
*[[余杰|余-{杰}-]]：是当今中国最具争议的作家之一，基督徒，2008年9月写了一篇长文《谁也不能杀死孩子——写给所有的母亲，也写给所有的父亲》，反對強制计划生育。
*[[刘军宁|刘军宁]]，中國政治学者。2008年8月，他写了一篇题为《自由主义视野中的计划生育政策》的文章全面批判计划生育理论。
*[[旷新年|旷新年]]，北京清華大學中文系副教授，認為人口越多，國家才會越強大，不該限制人口<ref>{{cite web |url=http://guancha.gmw.cn/content/2009-03/23/content_900843.htm |title=大国的博弈归根结底是人口的博弈——《大国空巢》书评 |deadurl=yes |archiveurl=https://web.archive.org/web/20110519113653/http://guancha.gmw.cn/content/2009-03/23/content_900843.htm |archivedate=2011-05-19 }}</ref>。
*[[顾宝昌|顾宝昌]]，中国人民大学人口与发展研究中心教授、中国人口学会常务理事，認為认识到低生育率时代的挑战进而加以应对，比从前认识到高生育率会造成问题的过程更困難<ref>{{cite web|url=http://tech.qq.com/a/20080721/000133.htm|title= 韩国、伊朗低生育率之惑}}</ref>。
*[[纪宝成|纪宝成]]，原[[中國人民大學|中國人民大學]]校長。他建议将“一孩化”政策逐步转变为“提倡生一个，允许生两个，杜绝生三个”。<ref>{{cite web|url=http://finance.sina.com.cn/g/20090630/13046419401.shtml |title=计划生育谋变？——国际视野下的中国人口政策}}</ref>
*[[滕彪|滕彪]]，中國[[維權|維權]]律師，他認為不論是從人權的角度、生育權的角度，「計劃生育從很多方面看都是錯誤的，從根本上就是錯誤的」<ref>{{cite web|title=中國未來有3千萬剩男 專家：是社會重大隱患|url=http://www.epochtimes.com/b5/17/2/13/n8807691.htm|publisher=大紀元|date=2017-02-14}}</ref>。
