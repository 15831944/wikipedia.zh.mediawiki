'''中國大陸電視劇'''，是指主要由來自[[中國大陸|中國大陸]]的演員演出的[[電視劇|電視劇]]，在中国大陆一般简称'''国产剧'''，在与港台剧并称时一般称'''内地剧'''；而在台湾一般简称'''大陆剧'''或'''陆剧'''。投資拍攝陸劇者並不一定是來自中國大陸，外資或港台資金拍攝的電視劇也算是大陸劇。近年，由於媒體迅速發展，中國大陸的[[電影公司|電影公司]]及網絡公司也開始拍攝電視劇，形成一股新的電視劇力量。如今中國大陸已成为電視劇生产线的第一大国，2013年的年產量為到一萬五千集的電視劇<ref name="2013年產量"/>。

== 產量 ==
自2000年電視劇產量突破10000集開始，中國大陸電視劇平均每年以近千集的速度持續增長<ref name="2013年產量"/>，2003年上升到10381集<ref>{{cite web |url = http://news.sina.com.cn/s/2004-08-15/16373397532s.shtml | language= | title="中国电视剧产量年增12% 已成电视剧生产大国 |publisher=新浪娱乐 |date=2004-08-05 |accessdate= }}</ref>。踏入2010年，中國大陸電視劇年產量仍然增長，2010年全年已生產了436部<ref name="2012年產量">{{cite web |url = http://ent.sina.com.cn/v/m/2013-08-27/14403996602.shtml| language= | title="电视剧年产量：美国5000集 中国1.7万集 |publisher=新浪娱乐 |date=2013-08-27 |accessdate= }}</ref>，共1.4萬集電視劇，但是可以實際播出的電視劇卻很少，根據[[廣電總局|廣電總局]]在2010年公佈的數據顯示，如今每年只有3000多集電視劇能夠播出<ref name="2010年產量">{{cite web|url=http://tech.ifeng.com/media/detail_2011_08/31/8830179_0.shtml| language= | title="国内电视剧年生产量将超过2万集 80%难见天日 |publisher=凤凰网科技 |date=2011-08-31 |accessdate= }}</ref>。編劇王海林對此數據解釋道“央視的[[央視一套|1套]]和[[央視八套|8套]]的黃金檔，一天各播2集電視劇，一年最多播1500集；各衛視一天播2集，有時三四家衛視播一部劇，還要考慮有些台在播舊劇或不播電視劇，因此一年播放的總量也就3000集至5000集。”<ref name="2010年產量"/>。根據報導，2012年獲廣電總局批准發行的中國電視劇已達到506部、17703集，平均日產48集，相比美國一年的產量5000集左右，中國大陸電視劇產量多了很多<ref name="2012年產量"/>。2013年中國大陸總產量在十三年的持續增長後出現首次回落，總產量減少20000集為15000集<ref name="2013年產量">{{cite web |url = http://ent.sina.com.cn/v/m/2014-02-18/22174098772.shtml| language= | title="2013电视剧产量首回落 总局鼓励一剧一星 |publisher=新浪娱乐 |date=2014-02-18 |accessdate= }}</ref>。

以上所列出的数据，均是通过广电总局备案的数据，事实上中国大陆有部分剧集是以栏目剧的形式存在的，比较著名的有央视社会与法频道的《普法栏目剧》、湖南卫视的《青春进行时》等。因栏目剧不用取得正规电视剧所必须取得的制作许可证与发行许可证，因此栏目剧可边拍边播，部分针对正规电视剧的限制，如涉案剧的播出时间、部分正规电视剧不能涉及的题材等，栏目剧均可涉及。
