'''中華民國與瑞典關係'''是指[[中華民國|中華民國]]與[[瑞典王國|瑞典王國]]之間的關係。兩國於1913－1950年有官方[[外交#外交上的承認|外交關係]]，斷交直至1970年代後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。[[瑞典政府|瑞典政府]]仍遵行「[[一個中國|一個中國]]」政策，雖與中華民國加強經貿、科技、文化等實質關係與民間交流，但在有關政治與官方關係仍以務實主義為原則。<ref name="國家基本資料">{{Cite web |url = http://www.boca.gov.tw/content.asp?CuItem=505&BaseDSD=13&CtUnit=14&mp=1 |title = 國家基本資料 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref>

===外交===

{{see also|中国驻瑞典大使列表{{!}}中華民國駐瑞典外交官列表|瑞典驻华大使列表{{!}}瑞典駐中華民國外交官列表}}
1913年10月6日，瑞典[[外交#外交上的承認|外交承認]]中華民國[[北洋政府|北洋政府]]。<ref name="代表團簡介"/><ref name="97年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2008/html/08-1-4.htm |title = 《中華民國97年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

1916年，於首都[[斯德哥爾摩|斯德哥爾摩]]設立'''中華民國駐瑞典王國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]（歷任公使兼[[中国驻挪威大使列表|駐挪威]]，或[[中国驻丹麦大使列表|丹麥]]、[[中国驻芬兰大使列表|芬蘭]]等[[北歐|北歐]]國家）。<ref name="代表團簡介"/><ref name="97年外交年鑑"/>

1928年12月20日，隨著[[國民革命軍北伐|國民革命軍北伐]]幾近尾聲，瑞典外交承認中華民國[[國民政府|國民政府]]。

1947年9月18日，升格為大使級外交關係。公使館升格為'''中華民國駐瑞典王國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。<ref name="代表團簡介"/><ref name="97年外交年鑑"/>

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國維持短暫的邦交關係。

1950年1月19日，與中華民國斷交、關閉大使館。<ref name="代表團簡介"/><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

1971年10月25日，瑞典在[[聯合國大會|聯合國大會]]就[[中國與聯合國#代表權問題|中國代表權]]問題上，與絕大多數[[歐洲國家和地區列表|歐洲國家]]一樣支持[[中華人民共和國|中華人民共和國]]取得「中國」席次，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。

1999年9月21日，[[九二一大地震|台灣發生大地震]]，瑞典捐贈價值5,000萬新台幣物資，重達80噸，包括354個20人大帳篷、1,485個睡袋及1萬條毛毯；並派遣國會議員培爾松及救難服務局官員卡爾森抵台點交。<ref>[瑞典援我物資抵台]【1999-10-08/聯合報/9版/集集大震　討論】</ref>

2018年2月28日，{{link-en|瑞典稅務局|Swedish Tax Agency}}發表公告，依據[[聯合國|聯合國]]的[[ISO_3166-1|ISO編碼]]將台灣名稱「[[中華民國（台灣）|中華民國（台灣）]]，[[瑞典文|瑞典文]]：{{lang|sv|Republiken Kina (Taiwan)}}」，改為「[[中國臺灣|台灣，中國的一省]]，瑞典文：{{lang|sv|Taiwan, Provins i Kina}}」。[[中華民國外交部|中華民國外交部]]指示駐瑞典代表處表達抗議，並要求更正<ref>{{cite web|author=張嘉哲 |title=瑞典稅務局稱台灣是中國的一省 外交部：無法接受 |url=https://newtalk.tw/news/view/2018-03-04/116128 |publisher=[[新頭殼|新頭殼]] |date=2018-03-04 |accessdate=2018-03-07}}</ref><ref>{{Cite web |url = https://tw.appledaily.com/new/realtime/20180305/1308157/ |author = 政治中心 |date = 2018年3月5日 |title = 瑞典稅務局稱我為中國一省 涉外人士：兩岸外交戰重啟 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref>。

瑞典支持台灣成為[[世界衛生大會觀察員|世界衛生大會觀察員]]，有意義參與[[國際民航組織|國際民航組織]]、[[联合国气候变化框架公约|聯合國氣候變遷綱要公約]]等。近年亦協助台灣以{{link-en|觀察員身分|Observer status}}參與新成立的[[國際可再生能源機構|國際再生能源組織]]，支持延續台灣在[[經濟合作暨發展組織|經濟合作暨發展組織]]之競爭委員會、漁業委員會、鋼鐵委員會的參與資格。<ref name="104年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

[[File:Visa_requirements_for_Swedish_citizens.png|thumb]]的{{link-en|瑞典護照|Swedish passport}}之[[瑞典人口|瑞典公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境[[中華民國|中華民國]]。]]

{{see also|歐洲聯盟簽證政策|{{link-en|瑞典公民簽證要求|Visa requirements for Swedish citizens}}}}
歐洲[[申根區|申根區]]給予持有載明[[中華民國國民身份證|身份證]]字號的[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以[[歐洲聯盟簽證政策|申根區免簽證]]入境，停留日數與申根區合併計算，每6個月期間內總計可停留90天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-67-641bb-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_FAQ.aspx?c=AD6908DFDDB62656&n=C1B19EF0ACDD3C91&sms=867F5398A49F758A |title = 歐盟免申根簽證常見問答集 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

[[瑞典人口|瑞典公民]]持有[[歐盟護照|歐盟護照]]者也可以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將瑞典歸類為'''灰色警示'''：提醒注意（2017年5月26日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-341-66478-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|瑞典經濟}}
===貿易===
[[經濟部國際貿易局|經濟部國際貿易局]]於首都[[斯德哥爾摩|斯德哥爾摩]]設立'''駐瑞典代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2017年，瑞典是中華民國第39大貿易夥伴、第41大[[進口|進口]]夥伴、第34大[[出口贸易|出口]]夥伴。出口至瑞典的金額為6億1,390萬9,583[[美元|美元]]，年增5.964%；自瑞典進口的金額為5億9,858萬1,163美元，年增23.331%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1,532萬8,420美元，年減83.695%。※2016年，瑞典是中華民國第38大貿易夥伴、第41大進口夥伴、第34大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資===
相較[[歐洲國家和地區列表|歐洲其他地區]]，[[台灣商人|台商]]於瑞典所在的[[北歐|北歐]]投資有限，多半是設立[[行銷|行銷]]或純代銷據點，且多僱用當地人處理當地業務，鮮少自台灣派員進駐。根據經濟部投資審議委員會統計，1992年－2014年，臺商對瑞典的總投資金額為296萬美元，計有5件。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/se/pdf/104-52Sweden.pdf |date = 2015年8月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

在瑞典設有銷售及服務據點的台商有[[宏碁|宏碁]]、[[華碩電腦|華碩電腦]]、[[合勤科技|合勤科技]]、[[友訊科技|友訊科技]]等公司，除了華碩電腦外，負責人皆為瑞典籍人士；[[技嘉科技|技嘉科技]]則在瑞典聘有經銷[[代理|代理]]人。<ref name="投資環境簡介"/>

目前已成立瑞典台灣商會、Vi Taiwan [[台灣人|台灣人]]新生社。<ref name="國家基本資料表">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 國家基本資料表 |publisher = 中華民國對外貿易發展協會}}</ref>

===會議===
台瑞（典）雙邊經濟合作會議（Joint Business Council Meeting），自1986年起，每年輪流在對方首都舉辦，至2017年已舉辦33屆。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref><ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2017年9月25日，於斯德哥爾摩舉辦[[次長|次長]]級經貿諮商會議，由中華民國經濟部常務次長[[王美花|王美花]]與{{Link-en|瑞典外交部|Ministry for Foreign Affairs (Sweden)}}貿易政務次長史登史壯（Oscar Stenström）共同主持。會議的3大主題包括「[[電信|電信]]、[[物聯網|物聯網]]及{{link-en|數位經濟|Digital economy}}」、「[[生物科技|生物科技]]及[[醫療衛生科學|醫療科技]]」、「[[核电厂|核能]]除役、[[能源|能源]]與[[智慧城市|智慧城市]]」。<ref name="106年外交年鑑"/>

==交流==
===僑民===
{{see also|臺灣裔{{!}}海外臺僑|海外華人}}
台灣僑民大部份居住於首都[[斯德哥爾摩|斯德哥爾摩]]（以下簡稱斯京）、第2大城[[哥特堡|哥特堡]]及第3大城[[馬爾摩|馬爾摩]]。另有台僑散居各處，大部份集中於[[瑞典省份|瑞典中南部]]，全部人數約1,000人。瑞典台僑多從事[[飲食業|餐飲業]]及食品雜貨[[批發|批發]]業，少部份經營[[旅遊業|旅遊業]]及其他行業，以[[中小企業|中小企業]]為多。<ref name="國家基本資料"/>

台僑已在斯京成立「台灣瑞典華人聯合會」，於哥特堡及馬爾摩設有分會，並在斯京成立聯合會的附屬[[中文學校|中文學校]]。台灣[[佛光山|佛光山]]亦在斯京近郊成立「瑞典佛光山」。<ref name="國家基本資料"/>

===學術===
{{see also|臺灣教育|瑞典教育}}
[[中華民國教育部|中華民國教育部]]提供[[中華民國國語|華語文]]獎學金，給予瑞典學生申請赴臺研習中文或攻讀[[碩士|碩]][[博士|博士]]。[[中華民國外交部|外交部]]則提供臺灣獎助金給予瑞典學者赴臺從事研究。2017年，有150名臺灣學生赴瑞典留學，有88名瑞典學生赴臺留學。<ref name="106年外交年鑑"/>
