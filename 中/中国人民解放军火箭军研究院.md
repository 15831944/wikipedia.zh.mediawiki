'''中国人民解放军火箭军研究院'''，位于[[北京市|北京市]][[海淀区|海淀区]]小营西路32号，是[[中国人民解放军火箭军|中国人民解放军火箭军]]直属单位，是中国人民解放军火箭军最高科研机构。

==简介==
2003年12月，[[中国人民解放军第二炮兵部队|中国人民解放军第二炮兵部队]]以几个独立的研究所为基础，组建'''中国人民解放军第二炮兵装备研究院'''。此前，各研究所分别负责[[导弹|导弹]]的不同单元和系统的科研论证工作，制约了整体科研效能。为适应一体化作战和一体化装备建设的需要，中国人民解放军第二炮兵部队党委决定并所建院。该院成立后，建立了一体化科研信息网，并建设了一批国家、军队、第二炮兵重点实验室<ref>{{Cite web |url = http://news.sohu.com/20090211/n262169605.shtml |title = 第二炮兵装备研究院一体化科研5年取得成果6000余项 |author =  |date = 2009-02-11 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url =http://news.xinhuanet.com/mil/2006-06/19/content_4715042_3.htm|title =二炮初步形成核常兼备、射程衔接的威慑力量体系|author =|date =2006-06-19|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20090804134828/http://news.xinhuanet.com/mil/2006-06/19/content_4715042_3.htm|archivedate =2009-08-04}}</ref>。2010年代初，第二炮兵装备研究院试验中心转变培养模式<ref>{{Cite web |url = http://www.hinews.cn/news/system/2011/11/22/013699547.shtml |title =  第二炮兵装备研究院试验中心转变培养模式|author =  |date = 2011-11-22 |publisher =  南海网|language =  |accessdate =  |quote =  }}</ref>。2012年12月21日，第二炮兵战略研究中心成立大会在第二炮兵装备研究院举行<ref>{{Cite web |url =http://mil.news.sina.com.cn/2012-12-23/0439710496.html  |title =  第二炮兵战略研究中心成立|author =  |date = 2012-12-23 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。2015年，第二炮兵装备研究院政工网完成升级改造<ref>{{Cite web |url = http://www.81.cn/jwzl/2015-10/16/content_6738088.htm |title = 第二炮兵装备研究院新版院政工网关注度高 |author =  |date = 2015-10-16 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，第二炮兵装备研究院更名为'''中国人民解放军火箭军装备研究院'''，隶属[[中国人民解放军火箭军|中国人民解放军火箭军]]<ref>{{Cite web |url =  http://www.81.cn/depb/2016-01/26/content_6868835.htm|title = 火箭军装备研究院表彰10名科研标兵 |author =  |date = 2016-01-26  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

2017年7月19日，新调整组建的军事科学院、国防大学、国防科技大学成立大会暨军队院校、科研机构、训练机构主要领导座谈会在北京[[八一大楼|八一大楼]]举行，[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军委主席|中央军委主席]][[习近平|习近平]]向军事科学院、国防大学、国防科技大学授军旗、致训词，出席座谈会并发表讲话<ref name=xunci>{{Cite web |url =http://news.xinhuanet.com/politics/2017-07/19/c_1121347127.htm  |title = 习近平向军事科学院、国防大学、国防科技大学授军旗致训词 |author =  |date = 2017-07-19 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。在此次改革中，火箭军装备研究院调整组建为'''中国人民解放军火箭军研究院'''<ref>{{Cite web |url =http://news.ifeng.com/a/20170830/51803114_0.shtml  |title =开展专题教育 激发创业激情  |author =  |date =  2017-08-30|publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
