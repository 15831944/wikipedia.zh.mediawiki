《'''中華人民共和國國家安全法'''》，是一部[[中華人民共和國|中華人民共和國]]於2015年7月1日颁布的[[法律|法律]]。該法由[[全国人民代表大会常务委员会|全国人民代表大会常务委员会]]通過<ref>中華人民共和國主席令第29號 發布</ref>，內容共有7章，中國政府官方媒體稱該法對「政治安全、國土安全、軍事安全、文化安全、科技安全」等11個「國家安全任務」進行了明確規範<ref name=DOCH>{{cite web |url = http://news.163.com/15/0702/05/ATGDSL6P00014AED.html |title=新国家安全法 7月1日起施行 |publisher=[[重慶日報|重慶日報]] |language = zh-Hans |accessdate=2015-07-03}}</ref>。

==內容==
{{Wikisource|中华人民共和国国家安全法}}

《中華人民共和國國家安全法》計七章84條。該法中申明了[[中國共產黨|中國共產黨]][[一黨執政|一黨執政]]的合法性，稱中國要堅持「[[中國特色社會主義|中國特色社會主義]]制度」、「社會主義先進文化前進方向」、抵制「不良文化」。該法第25條規定「中國必須防範和懲治網絡攻擊和散佈‘違法有害信息’等行為，以維護國家網絡空間主權」，第28條規定「中國反對一切形式的恐怖主義和極端主義」。另外，該法第11條還規定：「維護國家主權、統一和領土完整，是包括[[香港|香港]]、[[澳門|澳門]]和[[台灣|台灣]]在內的共同義務。」<ref name=BBC1/>

中共官媒稱該法對「政治安全、國土安全、軍事安全、文化安全、科技安全」等11個「國家安全任務」進行了明確，並且其中不僅涉及「核安全」、「反恐」、「和平探索外层空间、海底和极地」等，還涉及影響普通民眾的內容<ref name=DOCH/><ref name=NOS/>。

該法將每年的4月15日定爲中华人民共和国的「[[全民国家安全教育日|全民国家安全教育日]]」<ref name=NOS/>。

==立法過程==
2014年11月1日，[[第十二届全国人民代表大会常务委员会|第十二届全国人民代表大会常务委员会]]第十一次会议通過了《中華人民共和國反間諜法》，並廢止了1993年通過的舊《中華人民共和國國家安全法》<ref name=Xinhua2>{{cite web |url=http://news.21cn.com/social/shixiang/a/2015/0701/18/29744027.shtml?_t=t |title=新国家安全法通过 揭秘为何再制定国家安全法 |publisher=新華網 |language=zh-hans |accessdate=2015-07-03 |archive-url=https://web.archive.org/web/20150704171946/http://news.21cn.com/social/shixiang/a/2015/0701/18/29744027.shtml?_t=t |archive-date=2015-07-04 |dead-url=yes }}</ref>。2015年5月，中華人民共和國又推出了新《中華人民共和國國家安全法》的草案<ref name="黃安偉1"/>。2015年7月1日，第十二届全國人大會常委會第十五次会议{{NoteTag|[[第十二届全国人民代表大会常务委员会|第十二届全国人民代表大会常务委员会]]第十五次会议通過該法時，全国人大常委会委员应到169人，出席155人。其中154人投赞成票，0人投反对票，1人投弃权票。}}通過該法，[[中华人民共和国主席|中华人民共和国主席]][[習近平|習近平]]根據全國人大常委會的決定於同日簽署第29号《[[中華人民共和國主席令|主席令]]》頒佈該法<ref name=DOCH/>。

== 各方反應 ==
=== 香港的反應 ===
[[香港特区政府|香港特区政府]]在該法頒佈後表示：「香港特区政府发表声明表示，香港有维护国家安全的责任，有关法律不在香港特别行政区实施。特区政府没有计划就《基本法》第二十三条立法。新国安法不在港实施。」<ref name=BBC1/><ref>{{cite web |url = http://news.ifeng.com/a/20150701/44082299_0.shtml?_hj_reload_&1435757017605 |title=港府：新国安法不在港实施|publisher=凤凰卫视| language=zh-hans |date=2015年7月1日|accessdate=2015年7月3日}}</ref>

香港[[特首|特首]][[梁振英|梁振英]]也表示：「这部全国性法律不在香港实施，当然香港作为国家一部分，我们有责任以及义务维护国家安全的，我们履行这些责任和义务的方式，通过基本法二十三条规定的本地立法，特区政府无计划就二十三条立法<ref>http://www.thenewslens.com/post/183426/</ref>。」

[[港區全國人大代表|港區全國人大代表]][[郑耀棠|郑耀棠]]表示：「我相信新国安法对国家的安全非常之重要，有法律的依据，怎样保障国家安全，当然香港作为特区政府，有二十三条基本法的规定，特区政府有不可推卸的责任，有宪制性责任在香港进行立法，但是立法必须在香港条件成熟时才能进行立法，但无论怎样，香港必须承担宪制性的责任<ref name=BBC1/><ref>{{cite web |url = http://news.ifeng.com/a/20150701/44082299_0.shtml?_hj_reload_&1435757017605 |title = 港府：新国安法不在港实施 |publisher=凤凰卫视 |language=zh-Hans |date=2015年7月1日 |accessdate=2015年7月3日}}</ref>。」

香港[[律政司|律政司]]司長[[袁國強|袁國強]]也表示：「毋須過分擔心。《國家安全法》在其他國家，美國、英國、加拿大、法國等，很早已經有，而且現今國際社會關係這樣複雜，國家在這時候通過這條法例，這法例不是針對香港。而且另一方面，不會於香港適用，也不會透過《基本法》第十八條加入附件三。這情況下，也看到是尊重『一國兩制』的做法。因為始終香港的《基本法》下，我們有二十三條，當然我們有憲法責任，現在未做，但在適當時候我們會處理。」<ref>{{cite web |url = http://www.info.gov.hk/gia/general/201507/05/P201507050696.htm |title = 律政司司長談《國家安全法》|publisher=香港特別行政區政府新聞處 |date=2015年7月5日 |accessdate=2015年7月8日 |language=zh |location=香港}}</ref>

但是，[[香港泛民主派|香港泛民主派]]認爲，新法會在香港人進入中國大陸地區時帶來問題，可能被追究境外的言行<ref name=BBC1>{{cite web|url=http://www.bbc.com/zhongwen/trad/china/2015/07/150703_china_national_security_law_reax|title=外界對中國國安法如何付諸實施表達關注|publisher=BBC|accessdate=2015-07-03}}</ref>，憂心若引入國安法等同撕毀香港《基本法》。{{cn}}

===中華民國的反應===
[[中華民國政府|中華民國政府]]官方機構[[行政院大陸委員會|行政院大陸委員會]]對此嚴正表達不滿<ref>{{cite news |title=中國大陸片面通過「國家安全法」，未正視兩岸現實及尊重臺灣民意，我政府再次表達不滿 |url = http://www.mac.gov.tw/ct.asp?xItem=111919&ctNode=5649 |agency=中華民國[[行政院大陸委員會|行政院大陸委員會]] |date=2015-07-01 |language=zh-tw}}</ref>。

[[中國國民黨|中國國民黨]]表示，「中國國民黨始終堅持中華民國是主權獨立的國家，堅持在《[[中華民國憲法|中華民國憲法]]》架構下，由臺灣2,300萬人民共同決定臺灣的未來；大陸單方面制定的任何法律，無法影響、改變臺灣人民的意志，也無法左右臺灣的未來<ref name=EPO0701>{{cite news |authors = 鍾元、吳旻洲、陳懿勝 |title=中共推出最新國安法 港臺說「不」 |url = http://www.epochtimes.com/b5/15/7/1/n4470638.htm |agency=[[大紀元時報|大紀元時報]]|date=2015-07-01}}</ref>」。

[[民主進步黨|民主進步黨]]發言人[[鄭運鵬|鄭運鵬]]表示：「臺灣社會難以接受」<ref name=CNA>{{cite news |title=陸國安法納港澳台　民進黨：難以接受 |url = http://www.cna.com.tw/news/aipl/201507010432-1.aspx |agency=中央社 |date=2015-07-01}}</ref>。民進黨大老[[呂秀蓮|呂秀蓮]]則批評，該法「顯示中國併吞台灣的急迫感」<ref name=BBC2>{{cite web |url = http://www.bbc.com/zhongwen/trad/china/2015/07/150703_taiwan_former-vp_annette_lu_hsiu-lien_panama-city |title=呂秀蓮：中國國安法顯示併吞台灣急迫感|publisher=BBC}}</ref>。

[[臺灣團結聯盟|臺灣團結聯盟]][[立法院|立法院]][[黨團|黨團]]總召集人[[賴振昌|賴振昌]]表示，中國通過國安法其實只是「自我陶醉、痴人說夢」，由於中共政權從未統治過臺灣，此法在台灣沒有法源依據；又呼籲，台灣應停止與中國交流。
