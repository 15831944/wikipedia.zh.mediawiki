'''中华人民共和国主席'''，简称'''中国国家主席'''、'''国家主席'''，是[[中华人民共和国|中华人民共和国]]礼仪性和象征性的[[虚位元首|虚位]][[国家元首|国家元首]]<ref>{{cite news |title = 主席是虚职 总理行实权 |url = http://star.news.sohu.com/s2013/zhuxi/ |accessdate=2013-03-15 |work = 搜狐评论 |date = 2013-03-15 }}</ref>，与[[全国人民代表大会常务委员会|全国人民代表大会常务委员会]]联合行使[[中华人民共和国国家元首列表|中华人民共和国国家元首]]的职权。

根据《[[中华人民共和国宪法|中华人民共和国宪法]]》规定，国家主席的主要职能是根据全国人大或其常委会的决定签发[[中华人民共和国主席令|主席令]]和代表[[中华人民共和国|国家]]进行国事和外交活动。国家主席也是[[中华人民共和国国家机构列表|国家机构]]之一，国家主席机构处于最高国家权力机关[[全国人民代表大会|全国人民代表大会]]的从属地位<ref>[http://211.chinalawinfo.com/edu/xfx/contents/class07_3_1r.htm 法律信息资源系统 宪法学] {{webarchive |url = https://web.archive.org/web/20141023013001/http://211.chinalawinfo.com/edu/xfx/contents/class07_3_1r.htm |date = 2014-10-23 }}</ref>。

中华人民共和国的[[國家政體|國家政體]]是[[人民代表大会制度|人民代表大会制度]]。国家主席、[[中华人民共和国副主席|副主席]]是[[虛位元首|虛位元首]]，代表中华人民共和国进行国事活动。国家主席不领导任何[[中华人民共和国国家机关|国家机关]]，不承担[[中华人民共和国法律|立法]]、[[中华人民共和国政府|行政]]、[[中华人民共和国军事|军事]]权力与责任。现任国家主席[[习近平|习近平]]因其同时担任[[中国共产党中央委员会总书记|中共中央总书记]]和[[中国共产党中央军事委员会主席|中央军委主席]]<ref>[http://cpc.people.com.cn/GB/64184/64185/180139/10818589.html 江泽民文选第三卷--党和国家主要领导人选集、文选、专集]</ref>，而成为[[中华人民共和国最高领导人|最高领导人]]。

国家主席职务自1954年召开的[[第一届全国人民代表大会|第一届全国人民代表大会]]以后开始设立。1968年[[文化大革命|文化大革命]]期间，时任国家主席[[刘少奇|刘少奇]]被罢免后开始缺位，1975年[[第四届全国人民代表大会|第四届全国人民代表大会]]通过的第二部《[[中华人民共和国宪法|宪法]]》[[中华人民共和国主席存废之争|废除了此一职务]]。1982年通过的第四部《[[s:中华人民共和国宪法_(1982年)|宪法]]》又恢复了主席和[[中华人民共和国副主席|副主席]]设置。

== 职权 ==
根据《[[中华人民共和国宪法|中华人民共和国宪法]]》第三章第二节規定，中华人民共和国主席行使下列职权：
{{cquote|'''第八十条'''　中华人民共和国主席根据全国人民代表大会的决定和全国人民代表大会常务委员会的决定，公布法律，任免国务院总理、副总理、国务委员、各部部长、各委员会主任、审计长、秘书长，授予国家的勋章和荣誉称号，发布特赦令，宣布进入紧急状态，宣布战争状态，发布动员令。

'''第八十一条'''　中华人民共和国主席代表中华人民共和国，进行国事活动，接受外国使节；根据全国人民代表大会常务委员会的决定，派遣和召回驻外全权代表，批准和废除同外国缔结的条约和重要协定。
}}

除第八十一條規定国家主席可以進行[[國事訪問|国事活动]]、接受外國使節外，国家主席的其他职权均需根据全国人大及其常委会的决定执行。

== 机构设置 ==
根据《[[中华人民共和国宪法|中华人民共和国宪法]]》第八十二条规定，[[中华人民共和国副主席|中华人民共和国副主席]]协助主席工作。中华人民共和国副主席受主席的委托，可以代行主席的部分职权。

中华人民共和国主席设置[[中华人民共和国主席办公室|中华人民共和国主席办公室]]，作为国家主席日常办事机构，为副部级。

== 沿革 ==
中华人民共和国主席的设立大致可分为四个阶段：即1949年[[中华人民共和国开国大典|中华人民共和国成立]]至1954年第一部《宪法》的颁布，此时只有中央人民政府主席一职；1954年国家主席的设立至1975年第二部《宪法》对国家主席的废除；自1975年国家主席在《宪法》上的缺位至1982年第四部《宪法》对国家主席的恢复；以及1982年至今国家主席制度的稳步发展时期，<ref>[http://news.xinhuanet.com/politics/2010-09/06/c_12521128.htm 新中国国家主席制度的变迁]</ref>其中在80年代至90年代初由资历深厚的[[中共元老|中共元老]]担任，至90年代初国家主席开始由具有实权的[[中国共产党中央委员会总书记|中共中央总书记]]兼任，在职位本身未变的情况下其实际地位职权发生了变化。

=== 建国初期 ===
[[中华人民共和国开国大典|中华人民共和国建国]]至1954年[[行憲|行憲]]之间，在[[中华人民共和国国家机构列表|国家机构]]体系中，没有设置专门的国家主席，行使国家主席职权的国家机关是[[中央人民政府委员会|中央人民政府委员会]]。中央人民政府委员会设主席1名（中央人民政府主席），副主席6名，委员56名，秘书长1名。中央人民政府主席主持中央人民政府委员会会议和领导中央人民政府委员会的工作，在国家政治生活中发挥着重要作用。但中央人民政府主席不是一个独立的国家机关，名义上不是国家元首，而只是中央人民政府委员会的组成人员，不过在实际上主席行使了一部份属于国家元首的职权。因此，在中华人民共和国建立至1954年9月，中央人民政府主席实际上被视为国家主席。

=== 1954年宪法 ===
1954年3月23日，在中华人民共和国宪法起草委员会第一次会议上，中共中央主席[[毛泽东|毛泽东]]在解释为什么要设主席时说：“为了保证国家安全起见，设了个主席。我们中国是一个大国，叠床架屋地设个主席，目的是为着使国家更加安全。有议长，有总理，又有个主席，就更安全些，不至于三个地方同时都出毛病。”<ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1631918 |title =全国人大为何设“委员长”而不是“议长”  |author =  |date =  2017-03-04 |publisher =澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>宪法起草委员会的成员对于国家主席的职权问题进行过多次讨论，他们曾设想将主席的权力定得低一些，超脱一点，让主席只相当于“半个[[克利缅特·叶夫列莫维奇·伏罗希洛夫|伏罗希洛夫]]（时任[[苏联最高苏维埃主席团|苏联最高苏维埃主席团]]主席）”，“主席可以提出建议，建议不起决定作用，人家愿理就理，不理拉倒，毫无办法。”但是，这并不是意味着主席什么事情也不干。因为主席还可以“担任国防委员会主席”，“在必要的时候召开最高国务会议”<ref>[http://dangshi.people.com.cn/GB/85040/12643529.html 虚与实：中国国家主席的地位及作用的演变--中国共产党新闻-人民网]</ref>。

1954年9月召开的[[第一届全国人民代表大会|第一届全国人民代表大会]]第一次會議制定了第一部《[[中华人民共和国宪法|中华人民共和国宪法]]》。《中华人民共和国宪法》规定，在全国人民代表大会之下设立常务委员会和中华人民共和国主席，取消中央人民政府委员会，国家主席是政治体制中一个独立的国家机关，既是国家的代表，又是国家的象征。同时第一部《宪法》还对国家主席的产生、任期、地位和职权等一系列问题予以明确的规定。

1954年至1965年，中華人民共和國的主席制度基本上正常运转。1954年9月至1959年4月，[[毛泽东|毛泽东]]和[[朱德|朱德]]分别担任国家主席和副主席。毛泽东在任期届满后不再连任国家主席，但继续担任[[中国共产党中央委员会主席|中共中央主席]]和[[中国共产党中央军事委员会主席|中共中央军委主席]]的职务。1959年和1965年，中共中央副主席[[刘少奇|刘少奇]]两次当选为国家主席。在1954年至1965年期间，国家主席根据全国人民代表大会及其常务委员会的决定，公布了大批的法律法令，多次召开了[[最高国务会议|最高国务会议]]，接见外国使节，並进行了其他许多有关的职务活动。

[[文化大革命|文化大革命]]期间，时任国家主席[[刘少奇|刘少奇]]在1968年10月召开的[[中国共产党第八届中央委员会第十二次全体会议|中共八届十二中全会]]上被“撤销其党内外一切职务”，由于没有召开全国人大会议，该罢免案属于违宪<ref>{{cite news|title=学者：1982年宪法出台邓小平发挥决定性作用|url=http://www.chinanews.com/cul/2014/08-22/6519039.shtml|author=唐云云|publisher=中国新闻网|quote=[[中国宪法学会|中国宪法学会]]副会长、[[北京大学|北京大学]]宪法学教授[[张千帆|张千帆]]：“文革”时无法无天，用“革命委员会”取代政府。“文革”中的一些手续也是高度违宪的，比如说罢免国家主席刘少奇，不是通过人大，而是中央政治局。}}</ref><ref>{{cite news|title=刘少奇遗物中的历史珍藏 (7)|url=http://cpc.people.com.cn/GB/64093/67507/8324495.html|author=李桂芳|publisher=中国共产党新闻网|quote=刘少奇大声抗议说：“我是中华人民共和国的主席。你们怎样对待我个人，这无关紧要，但我要捍卫国家主席的尊严。谁罢免了我的国家主席？要审判，也要通过人民代表大会。”}}</ref>。1980年2月召开的中共十一届五中全会通过了为刘少奇平反的决议，恢复其名誉。<ref>{{cite news|title=中国共产党第十一届中央委员会第五次全体会议关于为刘少奇同志平反的决议|url=http://cpc.people.com.cn/GB/64184/64186/66677/4493840.html|date=1980-02-29|publisher=中国共产党新闻网|quote=过去对于刘少奇同志的污蔑、诬陷、伪造的材料以及一切不实之词都应完全推倒。}}</ref>。1968年至1975年间，国家主席长期处于空缺状态，先后由国家副主席（[[宋庆龄|宋庆龄]]和[[董必武|董必武]]二人）和[[中华人民共和国代理主席|代主席]]（董必武一人）代理职务。

=== 缺位和废除 ===
{{main|中华人民共和国主席存废之争}}
刘少奇死后，[[毛泽东|毛泽东]]有意废除国家主席一职，但其法定接班人[[林彪|林彪]]及多数中央领导人均不同意，甚至为[[中华人民共和国主席存废之争|国家主席的存废]]，而引起毛林两人的争执及后来的[[九一三事件|九一三事件]]<ref>[http://news.xinhuanet.com/theory/2009-05/23/content_11413544.htm 毛泽东为何不设国家主席]</ref>。

1975年1月17日，[[第四届全国人民代表大会|第四届全国人民代表大会]]第一次会议通过了中华人民共和国成立后的第二部《宪法》。《宪法》正式取消了国家主席的建制，改由[[全国人民代表大会常务委员会|全國人大常委會]]行使[[國家元首|國家元首]]職責。1978年3月5日，[[第五届全国人民代表大会|第五届全国人民代表大会]]第一次会议通过了中华人民共和国成立后的第三部《宪法》。该部《宪法》仍然坚持不设置国家主席。不过，1978年《宪法》把1954年《宪法》所规定的由国家主席行使的一些职权，改为由[[中国共产党中央委员会|中国共产党中央委员会]]和[[中国共产党中央委员会主席|中央委员会主席]]行使，包括：提名[[中华人民共和国国务院总理|国务院总理]]及统率[[中国人民解放军|全国武装力量]]。另外，接受外国使节；根据[[全国人民代表大会|全国人民代表大会]]或者[[全国人民代表大会常务委员会|全国人民代表大会常务委员会]]的决定，公布法律和法令，派遣和召回驻外全权代表，批准同外国缔结的条约，授予国家的荣誉称号的重要职权，改为由[[全国人民代表大会常务委员会委员长|全国人大常委会委员长]]代表全国人大常委会行使，而委员长在当时被视为国家元首。

=== 1982年宪法 ===
1982年12月4日，[[第五届全国人民代表大会|第五届全国人民代表大会]]第五次会议通过了中华人民共和国成立后的第四部《宪法》，即《[[八二宪法|八二宪法]]》，在国家机构一章中恢复关于中华人民共和国主席的规定，重新设置国家主席、副主席职务<ref>{{cite news |title=国家主席是什么样的国家机构？ |url = http://www.bjnews.com.cn/news/2013/03/14/252947.html |accessdate=2013-03-14 |website = 新京报网 |date=2013年3月14日 }}</ref>，但取消了《[[五四宪法|五四宪法]]》中赋予国家主席的两项职权，包括召开[[最高国务会议|最高国务会议]]和担任[[中华人民共和国国防委员会|国防委员会]]主席，失去了宪法赋予的行政参与权、立法提案权和名义上的军事统帅权，也不再设立作为国家主席办事机构的[[中华人民共和国主席办公厅|中华人民共和国主席办公厅]]<ref>{{cite book|url=http://cpc.people.com.cn/BIG5/218984/219004/219031/222847/14778322.html |title=關於1982年憲法的起草過程|chapter=憲法與國家政治體制（二）——王漢斌訪談錄|work=百年潮|date=2011年05月30日|volume=2011年第6期|publisher=中共中央黨史研究室|language=zh-hk|accessdate=2018年4月27日}}</ref><ref>{{cite web|url=http://cpc.people.com.cn/BIG5/218984/219004/219031/222846/14811512.html |title=百年潮2011年第6期目錄|date=2011年06月02日|publisher=中國共產黨歷史網|language=zh-hk|accessdate=2018年4月27日}}</ref>。在八二宪法审议过程中，[[邓小平|邓小平]]指出：“还是要设国家主席，有国家主席代表国家比较好，但是对国家主席的职权可以规定得虚一点，不要管具体工作，不要干涉具体政务。”<ref>[http://dangshi.people.com.cn/GB/85040/12643529.html 虚与实：中国国家主席的地位及作用的演变--中国共产党新闻-人民网]</ref>

国家主席一職並沒有實際權力，但历任国家主席均由[[中国共产党中央政治局|中共中央政治局委员]]以上级别的高级党员担任，其实权来自同时担任的其他职务。自[[江泽民|江泽民]]开始，国家主席与[[中国共产党中央委员会总书记|中共中央总书记]]和[[中央军委主席|中央军委主席]]三职由同一人擔任而握有實權，[[胡锦涛|胡锦涛]]、[[习近平|习近平]]也承此慣例。虽然宪法上国家主席没有权力，但作为[[中华人民共和国国家元首列表|中华人民共和国国家元首]]，对外的最高代表，纵观历任人选，即使不是由[[中共中央主要负责人|中国共产党最高领导人]]兼任，也是由党内举足轻重的人物担任。

1983年6月，中共元老[[李先念|李先念]]和[[乌兰夫|乌兰夫]]分别当选国家主席和副主席。1988年，中共元老[[杨尚昆|杨尚昆]]和[[王震|王震]]接任国家主席和副主席的职务。1982年《宪法》雖于1988年4月、1993年3月、1999年3月、2004年3月和2018年3月经过了五次修正，但每次都继续維持国家主席、副主席的设置，从而使国家主席制度得到确立。

随着中国进行[[改革开放|改革开放]]，对外交流不断增加，领导人外访和接待外宾的次数也越来越多，在[[杨尚昆|杨尚昆]]的任期届满后，当时的[[中国共产党中央委员会总书记|中共中央总书记]]、中央军委主席[[江泽民|江泽民]]接任国家主席一职，从此共产党的[[中华人民共和国最高领导人|最高领导人]]兼任国家主席以获得[[國家元首|國家元首]]的身份开始制度化，现任中共中央总书记[[习近平|习近平]]在胡锦涛卸任国家主席后也承继这“[[党和国家最高领导人|三位一体]]”的惯例。国家主席职务让最高领导人有了代表国家的身份，在国际上进行元首外交、[[国事访问|国事访问]]和出席国际会议等国事活动。<ref>[http://cpc.people.com.cn/GB/64184/64185/180139/10818589.html 江泽民文选第三卷] 江泽民说：“党的总书记、国家主席、军委主席三位一体这样的领导体制和领导形式，对我们这样一个大党、大国来说，不仅是必要的，而且是最妥当的办法。”</ref>

2018年3月[[第十三届全国人民代表大会|第十三届全国人民代表大会]]第一次会议通过的[[中华人民共和国宪法修正案_(2018年)|宪法修正案]]刪除了国家主席的连任限制，不过没有增加任何权力，其“虚位元首”的性质不变<ref>{{cite news |title = 「中國夢」與「完全極權」：歐美專家談習近平修憲 |url = http://www.bbc.com/zhongwen/trad/world-43340584 |date = 2018-03-11 |website = BBC中文网 |author = 林祖偉 |quote = 南加州大學美中學院主任杜克雷（Clayton Dube）接受BBC中文採訪時認為，今次修憲並「不是這麼大的一件事」，他解釋說：「黨總書記、軍委主席的職務更為重要，但沒有任期限制，今次取消任期限制的是國家主席，相當於一個儀式頭銜。」}}</ref>。

== 规章制度 ==
=== 选举和任免 ===
依据《中华人民共和国宪法》，中华人民共和国主席由[[全国人民代表大会会议主席团|全国人民代表大会会议主席团]]提名，由全国人民代表大会根据《宪法》第六十二条选举产生，任期五年，最多可连任一次。《宪法》第七十九条同时规定：国家主席必须是年满45岁并有选举权及被选举权的[[中華人民共和國國籍法|中华人民共和国公民]]。

[[全国人民代表大会|全国人民代表大会]]有权选举和罢免国家主席<ref>{{cite news|title = 中华人民共和国宪法|url = http://www.gov.cn/gongbao/content/2004/content_62714.htm|accessdate = 2014-01-18|publisher = 中华人民共和国国务院}}</ref>。选举方式是先由[[全国人民代表大会会议主席团|主席团]]提名，然后全国人大投票同意。由于是[[等额选举|等额选举]]，候选人只有一个，选举票分为赞成、反对、弃权三个选项。

《憲法》第八十二至八十四條規定，[[中华人民共和国副主席|中华人民共和国副主席]]协助主席工作。國家副主席受主席的委托，可以代行主席的部分職權。如果国家主席出缺，由副主席继任；国家副主席缺位时，由全国人民代表大会补选。在国家主席、副主席都缺位时，由全国人民代表大会补选；在补选以前，由[[全国人民代表大会常务委员会委员长|全国人民代表大会常务委员会委员长]]暂时代理主席职位。

2018年2月25日，中共中央向[[全国人民代表大会|全国人民代表大会]]提出《中共中央关于修改宪法部分内容的建议》，建议取消宪法中“国家主席与副主席连续任职不得超过两届”的规定<ref name="废除任期限制" />。3月11日，[[第十三届全国人民代表大会|第十三届全国人民代表大会]]第一次会议通过《[[2018年中华人民共和国宪法修正案|中华人民共和国宪法修正案]]》，决定取消对国家主席、副主席连任次数的限制<ref>[http://www.people.com.cn/GB/shehui/1060/2391835.html 中华人民共和国宪法修正案 - 人民网]</ref><ref name="废除任期限制">{{Cite web |url = http://www.xinhuanet.com/2018-02/25/c_1122451187.htm |title=（受权发布）中国共产党中央委员会关于修改宪法部分内容的建议 |website = [[新华网|新华网]] |date=2018-02-25 }}</ref>。

=== 政治排名 ===
{{main|党和国家领导人#政治排名}}
由于身兼[[中国共产党中央委员会主席|中国共产党中央委员会主席]]，所以[[毛泽东|毛泽东]]担任国家主席时在党内的排名同样处于首位。1959年[[刘少奇|刘少奇]]接任国家主席时，他是排名第一的[[中國共產黨中央委員會副主席|中共中央副主席]]，党内地位仅次于党主席[[毛泽东|毛泽东]]名列第二。[[文化大革命|文化大革命]]爆发后，刘少奇失去了党中央副主席的头衔，虽然保留了政治局常委职务，但排名降至第八位，随后被开除出党。[[李先念|李先念]]作为相隔15年后的另一位国家主席，在[[中国共产党中央政治局常务委员会|中央政治局常委]]中排名第五，在国务院总理之后。到了[[杨尚昆|杨尚昆]]担任国家主席时，他没有进政治局常委会，但与其他领导人同时列席时排名第三，仅次于总书记[[赵紫阳|赵紫阳]]和军委主席[[邓小平|邓小平]]。从1993年[[江泽民|江泽民]]以[[中国共产党中央委员会总书记|中共中央总书记]]身份兼任国家主席开始，国家主席在时隔34年后再次排名党内首位。不过根据中国官方的政治排名，中共中央总书记的地位还是在国家主席之上，领导人的头衔依官方次序为“[[中共中央总书记|中共中央总书记]]、[[国家主席|国家主席]]、[[中央军委主席|中央军委主席]]”。<ref>{{cite web|title=中共中央总书记、国家主席、中央军委主席习近平和中央政治局常委集体瞻仰中共一大会址|url=http://www.xinhuanet.com/politics/2017-10/31/c_1121883710.htm|website=新华网|date=2017-10-31}}</ref>
