'''中央区'''（{{jpn|j=中央区|hg=ちゅうおうく|rm=Chūō ku|f=y}}）是[[日本|日本]][[東京都|東京都]]內的23個[[東京23區|特別區]]之一，由於其位於23區的正中央位置之故，因而得名。

中央區不仅地理位置处于東京的中心地，更是日本经济、信息、商业等的中心。著名的[[日本銀行|日本銀行]]、[[东京证券交易所|东京证券交易所]]等日本經濟核心都位於中央区。除此之外，包括[[銀座|銀座]]、大型布莊與[[百貨公司|百貨公司]]總社聚集的[[日本橋|日本橋]]、以及以魚貨市場闻名的[[築地|築地]]皆位於本區範圍之內。

== 概要 ==
大約位於[[東京23區|東京23區]]的[[中央|中央]]。區内有[[日本橋_(東京都中央區)|日本橋]]、[[八重洲|八重洲]]・[[築地|築地]]、[[月島|月島]]、[[銀座|銀座]]等街區。

面積為10.094km²，是東京23區中第二小的特別區，僅次於[[台東區|台東區]]。

人口130,483人，次於[[千代田區|千代田區]]（2013年6月1日）。（2013年4月定居人口達13萬人，是1967年以來的新高）區內有日本橋與銀座等大型商業區域，讓日間人口達60.6萬人。此外，本區商業強烈，高樓林立，住宅方面以高層公寓與團地等[[集合住宅|集合住宅]]為主，獨戶低層住宅或低層分租公寓較為少見（多在南部的[[佃_(東京都中央區)|佃]]和[[月島|月島]]附近）。

道路呈現棋盤狀，與他區相比較為整齊。

區域西側在江戶時代是日本橋與京橋等繁榮的下町地區，東側是同時代填海造陸形成的土地。現在，中央區在行政上可分為[[日本橋_(東京都中央區)|日本橋]]、[[京橋|京橋]]（銀座、築地等）與[[月島|月島]]三個地區。

中央區在[[第二次世界大戰|第二次世界大戰]]結束以前是運河與水運發達的地區。區内有[[京橋川_(東京都)|京橋川]]、[[櫻川_(東京都)|櫻川]]、[[築地川|築地川]]、[[汐留川|汐留川]]、[[三十間堀川|三十間堀川]]、[[鐵砲洲川|鐵砲洲川]]、[[箱崎川|箱崎川]]、濱町川、龍閑川等河船或運河。這些河川在戰後陸續填埋。雖然已看不到過去的景色，但許多町界與區界都是沿河川所劃定。

區北端與區界為龍閑川遺跡，西端是外堀遺與[[日本橋川|日本橋川]]，南端有[[汐留川|汐留川]]遺。區的東端是[[隅田川|隅田川]]。隅田川下游分為兩條水路（西側為本流，東側為「派流晴海運河」），河中的中州是[[佃_(東京都中央區)|佃]]、月島，勝鬨，[[豐海町|豐海町]]，[[晴海|晴海]]。區東南部面[[東京灣|東京灣]]。

== 歷史 ==
; 近世以前
* 大約是[[江戶|江戶]]鄉内前嶋的位置。
; 近世
* 相當於江戶町。因[[寬政改革|寬政改革]]的雇用政策，石川島（現在的大川端）設置人足寄場。
* 1603年4月14日 - 初代日本橋完工。
; 近・現代
* 1869年 - 築地鐵砲洲（今日明石町附近）設置外國人居留地（～1899年）。
* 1953年 - 戰後復興，定居人口創下172,183人的記錄。
* 1967年 - 高度經濟成長，人口脫離都心，定居人口跌破13萬人。
* 1997年4月 - 定居人口創下71,806人的記錄。
* 2007年4月4日 - 定居人口達10萬人，31年來首度。
* 2011年 - 日本橋（第十九代）架橋100周年。
* 2013年4月4日 - 定住人口破13萬人，46年來首度。
; 沿革
* 1878年11月2日施行[[郡區町村編制法|郡區町村編制法]]，日本橋區與京橋區誕生。相當於現在的中央區。
* 1947年3月15日 - 兩區合併為現在的中央區。舊日本橋區域的町名冠上「日本橋」。
* 1947年5月3日 - [[地方自治法|地方自治法]]施行，中央區成為[[特別區|特別區]]。
* 1997年3月15日 - 中央區制施行50年。

=== 地名的由來 ===
* 因大約在[[東京市|東京市]]（現[[東京都區部|東京都區部]]）中央位置而得名。

==區立圖書館==
中央區立圖書館共有3館，區民每人可分配的區立圖書館藏書數為5.5冊，是23區第二高，僅次於千代田區。23區平均為3.4冊（2012年）。
