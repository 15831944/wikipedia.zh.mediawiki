'''中央·總武緩行線'''（{{jpn|j=中央・総武緩行線|hg=ちゅうおう・そうぶかんこうせん|rm=Chūō-Sōbu Kankōsen|f=y}}）是[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）的通勤列車運行系統，由[[千葉縣|千葉縣]][[千葉市|千葉市]][[中央區_(千葉市)|中央區]][[千葉站|千葉站]]開始，經[[東京都|東京都]][[千代田區|千代田區]][[御茶之水站|御茶之水站]]，至東京都[[三鷹市|三鷹市]][[三鷹站|三鷹站]]為止。千葉站－御茶之水站之間屬於[[總武本線|總武本線]]的一部分，御茶之水－三鷹站之間為[[中央本線|中央本線]]的一部分。

旅客所見的資訊則為「中央·總武線」「中央·總武線（各站停車）」<ref name="rf201005-020">{{Cite_journal|和書||title=特集・短絡線ミステリー10 都心を貫く直通運転をさぐる|date=2010-05|publisher=[[交友社|交友社]]|journal=[[鉄道ファン_(雑誌)|鉄道ファン]]|volume=50|number=5|pages=20-24}}</ref>「中央線·總武線各站停車」<ref>{{Cite_web|url=http://www.jreast.co.jp/map/pdf/map_tokyo.pdf|title=路線ネットワーク（東京近郊路線図）|publisher=東日本旅客鉄道|format=PDF|date=2012-03-17|accessdate=2012-07-15}}</ref>「中央線（各站停車）」「總武線（各站停車）」等。車站編號使用的路線記號為'''JB'''。

中央·總武緩行線在路線圖的代表顏色及列車塗裝顏色均為黃色，故也被稱為「黃色電車」。

== 概要 ==
中央·總武緩行線是指經由中央線與總武線，以[[御茶之水站|御茶之水站]]為連接點的各站停車電車直通運轉路線。本線所使用的列車（[[JR東日本E231系電車|E231系]]和[[JR東日本209系電車|209系500番台]]之黃色電車）之終點顯示器並不顯示運行區間與方向，以「中央・總武線」顯示（包括在清晨與深夜時段，不進入中央線而僅行駛在御茶之水站以東的總武線的列車也是如此）。在三鷹站～御茶之水站之間，車站稱此路線為「中央・總武線各站停車」或「中央・總武線」；[[秋葉原站|秋葉原站]]～千葉站方面則稱為「總武線」，而總武快速線的停車站則稱為「總武線（各站停車）」；東京近郊路線圖及鐵道運行情報則把此線路線稱為「中央・總武線各站停車」，在稱呼上有很大的不同。

於部分時間（清早與深夜），以及實施大型工程而導致運行障礙時（尤其是夜間），部分中央線與總武線的直通運轉列車會在[[東京站|東京站]]開出。在三鷹站～[[中野站|中野站]]間，於星期六、假日運行圖中，中央快速線所有列車會通過[[高圓寺站|高圓寺站]]、[[阿佐谷站|阿佐谷站]]與[[西荻窪站|西荻窪站]]（本區間之日中1小時運轉班次含[[東京地下鐵東西線|東京地下鐵東西線]]直通列車為平日8班、星期六、假日12班）。

在清早與深夜，部分列車不會在中央線與總武線直通運轉，而改在中央線的東京站（三鷹以西運轉）及總武線御茶之水站折返運轉。中央線午間時使用快速列車([[JR東日本E233系電車|E233系]]）。中央線各站停車運轉時間帶前後之朝夜間中央線[[武藏小金井站|武藏小金井站]]、[[國分寺站|國分寺站]]、[[立川站|立川站]]始發、終着以中央·總武線直通的列車存在（黃色之209系、E231系運用，早朝的東行及夜間的西行）。

至於與東京地下鐵東西線的相互直通運轉方面，在東西線的列車會在中央線側的中野站，與三鷹站和總武線側的[[西船橋站|西船橋站]]與[[津田沼站|津田沼站]]進入JR區間。總武線側平日清晨與黃昏以外時段及假日，中央線側的中野站與三鷹站緩行線的清晨、深夜時間帶不會進行直通運轉外，其餘時間約一半的列車會與東西線直通。[[東葉高速線|東葉高速線]]的車輛會直接進入東西線，午間東西線的快速列車於中野站折返。本區間緩行線月台，若有東西線直通列車同時行駛，月台上之顯示版會以「總武線」標示經御茶之水站的列車，而東西線直通列車就會以「東西線」表示（東京行之中央線各站停車部分為空白表示）。[[車站自動播音|車站自動播音]]（[[ATOS|ATOS]]）則以「總武線直通 各站停車…」「地下鐵東西線直通 快速…」來標示運轉路線名稱。

總武線區間嚴重擁擠，根據2011年國土交通省的調査，錦糸町→ 兩國站混雜率高達201%，超越山手線及京濱東北線混雜率(上野至御徒町站)200%，成為全國最擁擠的路線。相反，中央線區間東行因比較多乘客使用快速線，令中央緩行線成為東京各通勤路線中最鬆動的路線。據國土交通省之調查，2011年度繁忙時間的乘車率，中央快速線為194%，中央緩行線則為89%（代代木站→[[千駄谷站|千駄谷站]]），而繁忙時間，中央緩行線東行的乘車率亦未滿100%。不過在平日中央線西行由[[御茶之水|御茶之水]]開始因有來自大量總武線乘客，乘客往往繼續利用緩行線前往[[新宿|新宿]]一帶，令緩行線西行出現較擁擠的情況。

本路線每日的末班車在1:01a.m.(UTC +9)由[[新宿站|新宿站]]開往[[三鷹站|三鷹站]]，此時間亦為[[山手線|山手線]][[車站|車站]]的最遲開車時間。

[[淺草橋站|淺草橋站]]以東的月台方面標示，會顯示直通「[[中野站|中野]]」。站中月台以快速線「東京」顯示。

== 沿革 ==
1923年（[[大正|大正]]12年）的[[關東大地震|關東大地震]]前的總武本線，起點為[[兩國橋站|兩國橋站]]（現在的[[兩國站|兩國站]]）。同時，計劃設立中央本線，並把兩國站～御茶之水站間的高架路線，以及御茶之水站～中野站間的軌道增至複線，前者在1932年（[[昭和|昭和]]7年）完成，後者在1933年（昭和8年）完成。

根據[[通勤五方面作戰|首都圈5方面通勤輸送改善作戰]]的計劃，兩國站～御茶之水站的高架線與御茶之水站～中野站的複線，將於線外兩側各增設一線，是為「雙複線化」。中央本線的中野站至荻窪站區間的雙複線化於1966年（昭和41年）完成，1969年（昭和44年）荻窪站至三鷹站區的雙複線化完成，中央·總武線的運行範圍由中野站延長至三鷹，奠定現在三鷹站～千葉站間的中央·總武線的基礎。1963年（昭和38年）[[山手線|山手線]]投入新製的[[日本國鐵103系電聯車|103系列車]]後，原來使用的[[日本國鐵101系電聯車|101系列車]]轉屬中央·總武緩行線，1969年（昭和44年），本線全線車輛轉為101系。

1972年（昭和47年）7月15日，總武本線側的東京站的「總武地下月台」啟用，[[總武快速線|總武線快速]]列車由錦糸町站延長行駛至東京站，並和錦糸町站至津田沼站間的中央·總武緩行線分離行駛，同時廢止每小時1～2班於中野站發着的總武線快速列車。不過，津田沼站至千葉站的複線，仍由各站停車與快速列車共用。1981年（昭和56年），津田沼站與千葉站的雙複線化完成，中央·總武緩行線與總武線快速列車分離行駛，並一直沿用至今。1982年（昭和57年），101系列車退役，由[[日本國鐵201系電車|201系列車]]取代。

1990年代末期，103系列車故障頻生，急速老化的列車受人批評。有見及此，[[JR東日本E231系電聯車|E231系0番台列車]]由2000年（[[平成|平成]]12年）起導入，並於短期內完成撤換所有103系列車。同時，本線使用的201系和[[日本國鐵205系電車|205系列車]]亦出現老化的問題，由[[JR東日本209系電車|209系500番台列車]]及E231系列車取代，被汰換的201系及205系列車則向其他各線轉出。

2006年11月20日，開始在本線列車上增設[[#女性專用車|女性專用車厢]]。

== 女性專用車廂 ==
中央·總武緩行線於2006年11月20日實行[[女性專用車廂|女性專用車廂]]制度，是JR東日本繼[[埼京線|埼京線]]、[[中央線快速|中央線快速]]及[[常磐緩行線|常磐緩行線]]後，第四條實施女性專用車廂制度的路線。

中央·總武緩行線的女性專用車廂制度只適用於西行（往三鷹方向）列車：所有自早上7時20分（日本時間，UTC+9）至9時20分由[[錦糸町站|錦糸町站]]開出的班次自列車起點站至御茶之水站均設有女性專用車廂。至於東行列車及西行中央本線路段均不設女性專用車廂。

至於總武緩行線與東西線直通的列車，均按東西線的安排而設有女性專用車廂制度。東西線初期全線實施女性車廂制度，現時則縮短為西船橋站至[[大手町站_(東京都)|大手町站]]。
