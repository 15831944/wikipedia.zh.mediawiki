'''中共中央文献研究室'''是[[中共中央文献编辑委员会|中共中央文献编辑委员会]]已不存在的办事机构，是[[中共中央|中共中央]]的工作部门，也是具有研究机构性质的部门，是[[中共中央直属事业单位|中共中央直属事业单位]]<ref name=jj>{{Cite web |url =https://www.wxyjs.org.cn/zywxyjsjj_485/  |title = 中央文献研究室简介 |author =  |date =  |publisher = 中共中央文献研究室 |language =  |accessdate = 2017-10-13 |quote =  }}</ref><ref name=pp>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1282903 |title = 金冲及：胡乔木、胡绳怎样编撰中共党史？ |author =  |date =2014-12-03  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

==沿革==
1950年，中共中央成立由[[刘少奇|刘少奇]]任主任的'''中共中央[[毛泽东选集|毛泽东选集]]出版委员会'''。1977年3月，经中共中央批准，成立'''中共中央毛泽东主席著作编辑出版委员会办公室'''。1980年5月，中共中央决定，将中共中央毛泽东主席著作编辑出版委员会办公室改组为'''中共中央文献研究室'''。<ref name=jj/><ref>[http://book.ifeng.com/special/lingdaochushu/list/200911/1121_8722_1445580.shtml 中国出版中共领袖人物著作相关部门简介]，凤凰网，2009年11月21日</ref>

2018年3月[[中共中央|中共中央]]印发的《[[深化党和国家机构改革|深化党和国家机构改革]]方案》称，不再保留中共中央文献研究室，相关职责并入[[中共中央党史和文献研究院|中共中央党史和文献研究院]]<ref>{{Cite web |url = http://www.xinhuanet.com/2018-03/21/c_1122570517.htm |title = 中共中央印发《深化党和国家机构改革方案》 |author =  |date =2018-03-21  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。
==职责==
*编辑[[毛泽东|毛泽东]]、[[周恩来|周恩来]]、[[刘少奇|刘少奇]]、[[朱德|朱德]]、[[任弼时|任弼时]]、[[邓小平|邓小平]]、[[陈云|陈云]]等老一辈革命家及现任中央主要领导人的著作，编辑党和国家及军队重要的当代文献和历史文献；
*撰写毛泽东、周恩来、刘少奇、朱德、任弼时、邓小平、陈云等老一辈革命家和其他中央主要领导人的年谱、传记及传记性作品，研究他们的著作、生平和思想；
*研究和宣传[[毛泽东思想|毛泽东思想]]、[[邓小平理论|邓小平理论]]；
*收集国内外有关的研究动态，与国内外学术界进行必要的交流；
*审核或协助审核、审查准备公开发表或出版的毛泽东、周恩来、刘少奇、朱德、任弼时、邓小平、陈云和现任中央常委未发表过的历史文稿、有关音像制品，有关[[党和国家领导人|党和国家主要领导人]]工作和生活情况的重要图书、文章及音像制品等，以及重大革命历史题材的电影和电视作品；
*负责联系中央指定由中央文献研究室联系的其他老一辈革命家和历任中央主要领导人著作的编辑和年谱、传记的撰写工作；
*承担党中央交办的其他任务和[[中共中央文献编辑委员会|中共中央文献编辑委员会]]的具体事物<ref>{{Cite web |url =http://cpc.people.com.cn/GB/64114/76117/5229866.html |title = 中共中央文献研究室：主要职能 |author =  |date =  |publisher = 中国共产党新闻网 |language =  |accessdate = 2017-10-13 |quote =  }}</ref>。

==机构设置==
*办公厅
*科研管理部
*宣传外事办公室
*第一编研部（编辑[[毛泽东|毛泽东]]的著作，撰写其年谱、传记，研究其思想、生平）
*第二编研部（编辑[[周恩来|周恩来]]、[[刘少奇|刘少奇]]、[[朱德|朱德]]、[[任弼时|任弼时]]的著作，撰写他们的年谱、传记，研究他们的思想、生平）
*第三编研部（编辑[[邓小平|邓小平]]、[[陈云|陈云]]的著作，撰写其年谱、传记，研究他们的思想、生平）
*第四编研部（编辑、研究[[江泽民|江泽民]]的著作和以江泽民同志为核心的第三代中央领导集体其他领导成员的著作）
*第五编研部（编辑研究以[[胡锦涛|胡锦涛]]同志为总书记的党中央的重要文献等）
*第六编研部（编辑研究[[习近平|习近平]]总书记的著作和[[中共十八大|十八大]]以来党中央的重要文献等）
*机关党委
*《党的文献》杂志社
*中央文献出版社<ref>{{Cite web |url =https://www.wxyjs.org.cn/jgsz_486/  |title =  机构设置|author =  |date =  |publisher =中共中央文献研究室  |language =  |accessdate =2017-10-13  |quote =  }}</ref>
