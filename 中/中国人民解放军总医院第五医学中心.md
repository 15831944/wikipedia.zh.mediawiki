'''中国人民解放军总医院第五医学中心'''，位于[[北京市|北京市]][[丰台区|丰台区]][[北京四环路|西四环中路]]100号，是[[中央军委后勤保障部|中央军委后勤保障部]]直属单位，[[三级甲等医院|三级甲等医院]]，主治传染病。是全国最大、全军唯一的三级甲等传染病医院<ref name=lv/><ref name=tsinghua>[http://www.sce.tsinghua.edu.cn/news/detail.jsp?id1=2440 携手清华，共塑学习型医院，清华大学继续教育学院，2015-10-22] {{webarchive|url=https://web.archive.org/web/20161119183302/http://www.sce.tsinghua.edu.cn/news/detail.jsp?id1=2440 |date=2016-11-19 }}</ref>。

== 简介 ==
中国人民解放军总医院第五医学中心的前身是[[抗日战争|抗日战争]]时期的'''延安中央医院'''。[[中华人民共和国|中华人民共和国]]成立前夕，'''中共中央直属机关医院'''（又称'''朱豪医院'''）随中央机关从[[河北省|河北省]][[平山县|平山县]][[西柏坡|西柏坡]]迁至[[北平|北平]]。1952年12月，和浙江军区直属医院上调北京的部分合并组成'''[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]直属机关医院'''。1954年7月2日，正式命名为'''中国人民解放军第三〇二医院'''。2003年，中国人民解放军第三〇二医院防疫大队被合并到[[中国人民解放军军事医学科学院|中国人民解放军军事医学科学院]]，组建了“中国人民解放军军事医学科学院疾病预防控制所”<ref name="jm">[http://www.302hospital.com/i.php?id=376 简明院史]，中国人民解放军第三〇二医院，于2016-11-19查阅</ref>。2010年，中国人民解放军第三〇二医院新药临床试验中心成立<ref>王霞，打造国内一流Ⅰ期临床试验中心——访解放军第三〇二医院新药临床研究中心主任魏振满教授，中国当代医药2015年03期</ref>。2010年，该医院党委被表彰为“全军先进[[党委|党委]]”<ref name=lv>[http://news.xinhuanet.com/mil/2011-08/05/c_121818418.htm 302医院院长吕吉云：一切为了战争的胜利]，新华网，2011-08-05</ref>。

按照[[中央军委|中央军委]]、总部命令，该医院“担负战时机动卫勤保障、平时防生反恐及突发公共卫生事件的应急处置任务，承担军委、总部及驻京部队数百个单位约四十万官兵的传染病收治和全军疑难重症传染病会诊、转诊工作，是全军传染病医疗、教学、科研、预防、保健和信息中心，是军队传染病保健专科医院和全军传染病防治技术临床培训基地”。同时，该医院还是[[国家中医药管理局|国家中医药管理局]]重点[[中西医结合|中西医结合]]医院，国家药物临床试验机构，国家紧急救援网络医院，[[全国妇联|全国妇联]]全国青少年[[乙肝|乙肝]]诊疗中心，全国肝炎基金会全国乙肝病人教育基地，全军小儿肝病专科中心，美国心脏学会心血管急救技能培训中心。此外，该医院还是军队中医药技能培训基地，是[[北京大学|北京大学]]、[[北京中医药大学|北京中医药大学]]、[[中国人民解放军第四军医大学|中国人民解放军第四军医大学]]等10余所军地高等医学院校的教学实习医院。该医院还加入了中西医结合防治重大传染病军民融合式发展战略联盟<ref name="jm" />。

中国人民解放军第三〇二医院原先隶属[[中国人民解放军总后勤部|中国人民解放军总后勤部]]。在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月中国人民解放军总后勤部撤销，组建[[中央军委后勤保障部|中央军委后勤保障部]]。该医院转隶中央军委后勤保障部<ref name=ifeng>[http://news.ifeng.com/a/20160808/49738227_0.shtml 原总后政治部副主任苏林波任军委后勤保障部政治工作局副主任]，凤凰网，2016-08-08</ref>。

2016年4月16日，中国人民解放军第三〇二医院与[[新沂市人民医院|新沂市人民医院]]协作医院签约暨揭牌仪式举行<ref name="xy" />。

2002年，中国人民解放军第三〇二医院成为[[北京大学|北京大学]]教学医院，承担北京大学本科临床医学、预防医学、护理学3个学科5个层次的教学任务，并建立了传染病学博士、硕士培养点。2017年1月20日，中国人民解放军第三〇二医院与[[北京大学医学部|北京大学医学部]]签署军民融合共建协议，正式成立'''北京大学三〇二临床医学院'''<ref>{{Cite web |url = http://news.163.com/17/0120/18/CB89T96N000187V5.html |title = 北京大学302临床医学院成立 |author =  |date = 2017-01-20 |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。

2018年，[[中国人民解放军陆军总医院|原陆军总医院]]、[[中国人民解放军海军总医院|原海军总医院]]、原第三〇二医院、[[中国人民解放军第三〇九医院|原第三〇九医院]]、[[中国人民解放军第三〇七医院|军事科学院军事医学研究院原附属医院]]、[[中国人民武装警察部队总医院|原武警总医院]]均已由[[中国人民解放军总医院|中国人民解放军总医院]]暂管<ref>{{Cite web |url = https://www.thepaper.cn/newsDetail_forward_2334813 |title = 官方披露：原陆军总医院等单位已由解放军总医院暂管 |author =  |date =2018-08-10  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。现已并入军事科学院军事医学研究院原附属医院，并更名为'''中国人民解放军总医院第五医学中心'''<ref>{{Cite web |url = http://www.sohu.com/a/273558387_267160 |title = 军队医院更名继续，解放军总医院八个医学中心亮相 |author =  |date =2018-11-06  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
