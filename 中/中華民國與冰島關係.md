'''中華民國與冰島關係'''是指[[中華民國|中華民國]]與[[冰島共和國|冰島共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對冰島的相關事務由'''[[駐丹麥台北代表處|駐丹麥台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-215-b2780-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref><ref>{{Cite web |url = http://www.taiwanembassy.org/dk/post/4.html |title = 代表處簡介 |publisher = 中華民國駐外單位聯合網站}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[冰島|冰島]]。]]
{{see also|中華民國外交|{{link-en|冰島外交|Foreign relations of Iceland}}}}
1993年2月，[[駐丹麥台北代表處|駐丹麥台北代表處]]代表許智偉赴冰島訪問，會見冰島總理[[戴維·奧德森|歐德生]]，這是中華民國官員首次見冰島總理。<ref name="86年外交年鑑"/>

1994年7月，駐丹麥台北代表處代表陳毓駒續訪冰島，會見冰島總統[[維格迪絲·芬博阿多蒂爾|芬柏嘉朵蒂]]及總理歐德生。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-4.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1997年1月，[[冰島國會|冰島國會]]外交委員會主席[[盖尔·哈尔德|赫德]]抵台訪問。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/662/153703/%e5%85%83%e6%9c%88.pdf |author = 公眾外交協調會 |date = 1997年1月 |title = 外交部中華民國八十六年元月份新聞稿 |publisher = 中華民國外交部}}</ref>10月，中華民國副總統[[連戰|連戰]]訪問冰島，在機場獲得禮遇通關，此行與總理歐德生餐敘。對於連戰的到訪，[[中華人民共和國|中華人民共和國]]向[[冰島政治|冰島政府]]提出嚴重抗議。<ref>{{Cite web |url = http://news.cts.com.tw/cts/general/199710/199710070008650.html |author = [[華視新聞|華視新聞]] |date = 1997年10月7日 |title = 連戰一行抵達冰島 |publisher = [[中華電視公司|中華電視公司]]}}</ref>

1999年，冰島國會副議長{{link-en|阿納德|Ragnar Arnalds}}、首都[[雷克雅維克|雷克雅維克]]市長{{link-en|季絲拉朵蒂爾|Ingibjörg Sólrún Gísladóttir}}抵台訪問。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter1-4.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2000年，冰島國會外交委員會主席{{link-en|歐瑞希|Tómas Ingi Olrich}}抵台訪問。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_8.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2005年3月14日，冰島國會外交委員會主席（於當年8月兼任國會[[議長|議長]]）{{link-en|比德斯朵蒂|Sólveig Pétursdóttir}}、教育暨文化委員會主席{{link-en|畢爾吉生|Gunnar Birgisson}}、環境委員會主席{{link-en|索達生|Guðlaugur Þór Þórðarson}}應外交部邀請抵台訪問6天。期間會見[[中華民國總統|中華民國總統]][[陳水扁|陳水扁]]，以及拜會[[立法院|立法院]]、[[中華民國外交部|外交部]]、[[行政院大陸委員會|行政院大陸委員會]]、[[行政院國家科學委員會|行政院國家科學委員會]]、[[行政院新聞局|行政院新聞局]]、[[經濟部工業局|經濟部工業局]]等政府單位。此行也參訪了台灣的文教設施。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=F34A74C3324B2868 |author = 公眾外交協調會 |date = 2005年3月14日 |title = 冰島國會外交委員會主席比德斯朵蒂率國會領袖團應邀訪台 |publisher = 中華民國外交部}}</ref>

2005年11月，冰島國會經濟委員會主席{{link-en|布朗戈|Pétur Blöndal}}抵台訪問。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2005/WebPages/MICO/08/08_01_04.htm |title = 《中華民國94年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2007年，冰島國會總務事務暨憲政事務特別委員會主席{{link-en|艾曼生|Birgir Ármannsson}}、工業委員會主席{{link-en|朱麗絲朵蒂|Katrín Júlíusdóttir}}抵台訪問。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2007/html/08-1-4.htm |title = 《中華民國96年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2016年8月，一名台灣女[[交換學生|交換學生]]赴冰島求學，在申請居留證時國籍被登記為「中國」而不是該名學生希望的「台灣」，中華民國外交部也指示鄰近的駐丹麥台北代表處協助處理。與當地政府斡旋三個多月後得到「[[無國籍|無國籍]]」的註記結果，冰島移民署的主管表示：「台灣不是[[外交#外交上的承認|被承認的國家]]。」<ref>{{Cite web |url = http://news.ltn.com.tw/news/focus/paper/1053735 |author = 呂伊萱 |date = 2016年11月20日 |title = 爭取中國改台灣 冰島竟發無國籍居留證 |publisher = [[自由時報|自由時報]]}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以「[[歐洲聯盟簽證政策|申根區免簽證]]」的方式入境冰島。]]
[[File:Visa_requirements_for_Icelandic_citizens.png|thumb]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|歐洲聯盟簽證政策|{{link-en|冰島公民簽證要求|Visa requirements for Icelandic citizens}}}}
冰島尚未加入[[歐洲聯盟|歐洲聯盟]]，但以「北歐護照聯盟」成員國的身份加入[[申根區|申根區]]，因此實行與申根區相同的簽證政策。歐洲[[申根區|申根區]]給予持有載明[[中華民國國民身份證|身份證]]字號的[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以[[歐洲聯盟簽證政策|申根區免簽證]]入境，停留日數與申根區合併計算，每6個月期間內總計可停留90天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-215-b2780-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_FAQ.aspx?c=AD6908DFDDB62656&n=C1B19EF0ACDD3C91&sms=867F5398A49F758A |title = 歐盟免申根簽證常見問答集 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|冰島護照|Icelandic passport}}的[[冰島人口|冰島公民]]則可用[[歐洲自由貿易聯盟|歐洲自由貿易聯盟]]的身分以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將冰島歸類為'''灰色警示'''：提醒注意（2017年11月27日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-76-659e7-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2010年3月，冰島貿易[[理事會|理事會]][[理事長|理事長]]抵台參訪，期間拜會[[中華民國外交部|外交部]]、[[經濟部國際貿易局|經濟部國際貿易局]]、[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）等政府機構，並參觀[[台塑關係企業|台塑集團]]、台灣國際[[蘭科|蘭]]展、慶富[[造船廠|造船廠]]等地。<ref>{{Cite web |url = http://www.trademag.org.tw/content02.asp?id=530353&type=22&url=/NewsSearch.asp |author = 楊璧慧 |date = 2010年3月24日 |title = 貿協拓銷團 保加利亞小市場大商機 |publisher = 經貿透視雙周刊}}</ref>

2017年，冰島是中華民國第120大貿易夥伴、第101大[[進口|進口]]夥伴、第137大[[出口贸易|出口]]夥伴。出口至冰島的金額為647萬5,467[[美元|美元]]，年增6.727%；自冰島進口的金額為1,951萬8,723美元，年減21.694%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）1,304萬3,256美元，年減30.838%。※2016年，冰島是中華民國第112大貿易夥伴、第94大進口夥伴、第137大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
