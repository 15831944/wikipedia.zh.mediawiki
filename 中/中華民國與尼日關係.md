'''中華民國與尼日關係'''是指[[中華民國|中華民國]]與[[尼日共和國|尼日共和國]]之間的關係。兩國於1963－1996年有官方[[外交#外交上的承認|外交關係]]但曾經中斷，二次斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對尼日的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-237-c449b-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[尼日|尼日]]。]]
{{see also|中華民國外交|{{link-en|尼日外交|Foreign relations of Niger}}}}
1963年7月22日，中华民国与尼日[[外交#外交上的承認|建交]]。

1965年，於首都[[尼阿美|尼阿美]]設立'''中華民國駐尼日共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1974年7月29日，两国断交。

1992年6月19日，中华民国与尼日复交。7月22日，{{link-en|尼日政府|Politics of Niger}}發表聲明，重申兩國邦交不變，中華民國駐尼日共和國大使館於同日[[中華民國國旗|升旗]]掛牌，正式開館。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=6F0DC0072EE86B53 |author = 公眾外交協調會 |date = 1992年7月23日 |title = 尼日共和國政府於七月二十二日下午發表聲明 |publisher = 中華民國外交部}}</ref>7月30日，[[中華人民共和國|中華人民共和國]]與尼日斷交。

1996年1月，[[中華民國外交部|中華民國外交部]]常務次長黃秀日訪問尼日，參加中尼合作混合[[委員會|委員會]]第一次會議，期間並會見尼日總統[[马哈曼·奥斯曼|烏斯曼]]、總理[[哈马·阿马杜|阿馬杜]]、國會議長[[马哈马杜·伊素福|伊索福]]。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=F205D9DCD92E35AE |author = 公眾外交協調會 |date = 1996年1月22日 |title = 由中華民國外交部次長黃秀日所率領久中華民國政府代表團於元月十六日下午抵達尼日首都尼阿美作友好訪問 |publisher = 中華民國外交部}}</ref>

1996年8月19日，两国再次断交<ref>{{cite book|title=《台灣歷史辭典》（附錄）|author=許雪姬|publisher=[[中華民國文化部|行政院文化建設委員會]]|chapter=中華民國歷任駐外大使(遷臺後，1949~2003.10)}}</ref>。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境尼日。]]
[[File:Visa_requirements_for_Nigerien_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|尼日簽證政策|{{link-en|尼日公民簽證要求|Visa requirements for Nigerien citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往尼日駐[[法國|法國]][[外交代表機構#大使馆|大使館]]申請簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-237-c449b-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|尼日護照|Nigerien passport}}的{{link-en|尼日人口|Demographics of Niger|尼日公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將尼日歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2016年10月27日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-63-2bbb3-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|尼日經濟|Economy of Niger}}}}
===貿易===
2017年，尼日是中華民國第176大貿易夥伴、第176大[[進口|進口]]夥伴、第163大[[出口贸易|出口]]夥伴。出口至尼日的金額為182萬0,421[[美元|美元]]，年減53.611%；自尼日進口的金額為20萬4,706美元，年增118.053%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）161萬5,715美元，年減57.819%。※2016年，尼日是中華民國第168大貿易夥伴、第189大進口夥伴、第149大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
2013年8月23日，中華民國的[[台灣中油公司|中油公司]]與[[中華人民共和國|中華人民共和國]]的[[中国石油|中國石油天然氣勘探開發公司]](CNODC)合作[[天然气田|油氣]]探採開發，臺灣中油公司[[交割|交割]]取得該公司在[[尼日|尼日]]境內Agadem礦區23.53%探勘權、20%開發生產權及20%內陸[[输油管道|管線]]權益資產。至於開發生產合作公司的權益比分別為CNODC 65%、尼日政府15%、以及中油公司20%。

===事件===
兩國在邦交期間，[[中國輸出入銀行|中國輸出入銀行]]（簡稱輸銀）提供[[貸款|貸款]]資助尼日。斷交後，輸銀向[[美國紐約南區聯邦地區法院|美國紐約南區聯邦地區法院]]提告{{noteTag|中國輸出入銀行在[[紐約|紐約]]設有辦公室，過去貸款給[[中華民國外交#邦交國|中華民國的邦交國]]時由輸銀出面與對方國家銀行簽約，合約可以適用{{link-en|紐約州法律|Law of New York}}，對中華民國比較有保障。}}，求償1億8,300萬美元。2007年，[[紐約上訴法院|紐約上訴法院]]判決尼日須給付台灣7,200萬美元欠款。在尼日殺價2,000萬美元後，索賠成功。<ref>{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20170401/37603375/ |author = 綜合報導 |date = 2017年4月1日 |title = 非洲2斷交國 判還我64億 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref><ref>{{Cite web |url = https://newtalk.tw/news/view/2017-03-31/83803 |author = 洪翠蓮 |date = 2017年3月31日 |title = 斷交就還錢！台灣求償2非洲國家63.5億勝訴 |publisher = [[新頭殼|新頭殼]]}}</ref>
