'''中国人民解放军空军太平川靶场'''，位于[[吉林省|吉林省]][[松原市|松原市]][[长岭县|长岭县]][[太平川镇_(长岭县)|太平川镇]]，隶属[[中国人民解放军北部战区空军|中国人民解放军北部战区空军]]。

==简介==
空军太平川靶场是[[中国人民解放军空军航空兵|中国人民解放军空军航空兵]]轰炸射击靶场，位于[[辽宁省|辽宁省]]、[[吉林省|吉林省]]、[[内蒙古自治区|内蒙古自治区]]交界处，属于平原地形，可组成宽大对抗空域。空军太平川靶场具备优良的战场环境，靶场呈椭圆形布局，净空条件好，电磁干扰少。空军太平川靶场配套设施齐全，有指挥塔台、视频监控系统、评估系统，靶区内有永固式轰炸靶、激光靶、火箭靶以及多型实装靶标，曾经多次担负多型制导炸弹、常规炸弹、火箭弹等的投射任务<ref>{{Cite web |url = http://news.163.com/17/0809/05/CRCH0M5J00018AOP.html |title = 选址天然空中“练兵场”供中俄数十战机精准投靶 |author =  |date =2017-08-09  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。空军太平川靶场内地形地貌丰富，含水泡子、[[盐碱地|盐碱地]]、[[草原|草原]]、树林等多种地形地貌<ref name=feibiao/>。

2017年，空军太平川靶场完成了改扩建，建设了“航空飞镖”靶场，随后在2017年8月承办了由[[俄罗斯联邦国防部|俄罗斯联邦国防部]]发起的“国际军事比赛-2017”中的“航空飞镖-2017”项目，这是中国人民解放军空军首次承办“国际军事比赛”中的“航空飞镖”项目比赛<ref name=feibiao>{{Cite web |url =  http://news.xinhuanet.com/mil/2017-08/08/c_129675756.htm?from=singlemessage&isappinstalled=1|title =  “航空飞镖－２０１７”：从盐碱地到国际化空军靶场的逆袭|author =  |date = 2017-08-08 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.jlcl.gov.cn/ywdt/clyw/201707/t20170724_167212.html |title =高材林到我县视察“航空飞镖”靶场建设情况  |author =  |date = 2017-07-24 |publisher = 长岭县人民政府 |language =  |accessdate =  |quote =  }}</ref>。其间中国人民解放军空军首次设立实战化新闻中心<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1764540 |title = 中国空军首设实战化新闻中心，新闻发布向“战时模式”聚焦 |author =  |date = 2017-08-16 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。2017年8月4日，空军太平川靶场举行“航空飞镖-2017”靶场开放日活动，这是中国人民解放军空军首次举行靶场开放活动，中国人民解放军空军和[[俄罗斯空天军|俄罗斯空天军]]进行了飞行表演、实弹飞行，现场上万名观众观看<ref>{{Cite web |url =http://www.mod.gov.cn/diplomacy/2017-08/05/content_4787854.htm  |title =“航空飞镖-2017”靶场开放日活动在松原举行  |author =  |date =2017-08-05  |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
