'''中華民國與塞爾維亞關係'''是指[[中華民國|中華民國]]與[[塞爾維亞共和國|塞爾維亞共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對塞爾維亞的相關事務由'''駐[[匈牙利|匈牙利]][[台北经济文化代表处|台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-202-6687c-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
{{see also|中華民國外交|{{link-en|塞爾維亞外交|Foreign relations of Serbia}}}}
2017年10月16日，塞爾維亞地方經濟發展聯盟[[執行長|執行長]]尤芳諾維奇（Violeta Jovanovic）與塞爾維亞地方政府5位市長抵台訪問。21日，在[[中華民國外交部|中華民國外交部]]人員陪同赴[[宜蘭市|宜蘭市區]]餐廳用餐時，[[克魯舍瓦茨|克魯舍瓦茨]]市長奈斯托羅維奇（Dragi Nestorovic）突然臉色發紫倒地，送醫不治。22日，[[宜蘭地方法院|宜蘭地方法院]]檢察署[[檢察官|檢察官]]與[[法醫|法醫]]解剖，初步認定奈斯托羅維奇是因為用餐時[[心律不整|心律不整]]昏厥跌倒，因撞到頭部導致[[腦出血|腦出血]]死亡。<ref>{{Cite web |url = https://udn.com/news/story/7315/2771730 |author = 張芮瑜 |date = 2017年10月22日 |title = 塞爾維亞市長在台猝死案 檢警初判腦出血 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|塞爾維亞簽證政策|Visa policy of Serbia}}|{{link-en|塞爾維亞公民簽證要求|Visa requirements for Serbian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往塞爾維亞駐[[日本|日本]][[外交代表機構#大使馆|大使館]]辦理簽證。因塞爾維亞與[[科索沃|科索沃]]仍存有主權爭議，由塞國前往科國，須備妥入境相關證明文件，以免在兩國邊境海關受阻。<ref name="簽證及入境須知">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-202-6687c-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>[[中華民國與科索沃關係#簽證|中華民國公民前往科索沃可免簽證]]。

持有[[塞爾維亞護照|塞爾維亞護照]]的[[塞爾維亞人口|塞爾維亞公民]]抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將塞爾維亞歸類為'''灰色警示'''：提醒注意（2017年5月19日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-329-96256-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將塞爾維亞歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年11月6日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=91F0941F403EE107 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>


==經濟==
{{see also|臺灣經濟|塞爾維亞經濟}}
===貿易===
2012年10月27日，[[經濟部國際貿易局|經濟部國際貿易局]]委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）與46家台灣業者籌組「2012年[[中歐|中]][[東歐|東歐]]暨[[巴爾幹半島|巴爾幹]]潛力市場拓銷團」，是台灣睽違塞爾維亞市場20年後的首發團。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=596192&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2012年10月26日 |title = 經部助廠商拓銷中東歐 睽違20年挺進塞爾維亞 |publisher = 經貿透視網}}</ref>在貿易洽談會中有超過120家當地業者與[[台灣商人|台商]]進行217場次洽談，商機約540萬美元。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=597614&url=/NewsSearch.asp |author = 中華民國對外貿易發展協會 |date = 2012年11月13日 |title = 台商赴獨立國協及巴爾幹市場 拓銷商機達11.65億 |publisher = 經貿透視網}}</ref>

2017年，塞爾維亞是中華民國第106大貿易夥伴、第100大[[進口|進口]]夥伴、第100大[[出口贸易|出口]]夥伴。出口至塞爾維亞的金額為2,856萬0,288[[美元|美元]]，年增86.780%；自塞爾維亞進口的金額為2,086萬8,173美元，年增56.612%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）769萬2,115美元，年增291.237%。※2016年，塞爾維亞是中華民國第114大貿易夥伴、第104大進口夥伴、第110大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
