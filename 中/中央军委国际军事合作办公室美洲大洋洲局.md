'''中央军委国际军事合作办公室美洲大洋洲局'''，位于[[北京市|北京市]]，是[[中央军委国际军事合作办公室|中央军委国际军事合作办公室]]下属局，负责该办公室[[美洲|美洲]]、[[大洋洲|大洋洲]]业务。

==沿革==
原先[[中华人民共和国国防部|国防部]]外事办公室设有'''国防部外事办公室美洲大洋洲局'''<ref name=haigui>{{Cite web |url = http://www.ce.cn/xwzx/gnsz/gdxw/201507/12/t20150712_5906214.shtml |title = 国防部新闻发言人首现“双海归”搭档 |author =  |date = 2015-07-12 |publisher = 中国经济网 |language =  |accessdate =  |quote =  }}</ref>。

在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月组建[[中央军委国际军事合作办公室|中央军委国际军事合作办公室]]。中央军委国际军事合作办公室新设'''中央军委国际军事合作办公室美洲大洋洲局'''<ref name=gjfw>{{Cite web |url =http://www.cdsndu.org/html_ch/to_articleContent_article.id=40288a8553c564450154f12e8e0e0474.html  |title = 张仕波校长出席第12期拉美高级军官研讨班结业典礼 |author =  |date =2016-05-25  |publisher = 中国人民解放军国防大学国际防务学院 |language =  |accessdate =  |quote =  }}</ref>。主管中国与[[美洲|美洲]]、[[大洋洲|大洋洲]]国家的军事外交事务。
