'''中国共产党中国人民政治协商会议全国委员会机关党组'''（简称'''中共政协全国委员会机关党组'''，或'''中共全国政协机关党组'''），由[[中国共产党中央委员会|中国共产党中央委员会]]批准成立<ref name=zgzxw>{{Cite web |url =http://www.cppcc.gov.cn/2011/09/26/ARTI1317001118843749.shtml  |title =中国共产党中国人民政治协商会议全国委员会机关党组  |author =  |date = 2011-09-26 |publisher = 中国人民政治协商会议全国委员会 |language =  |accessdate =  |quote =  }}</ref>。设在[[中国人民政治协商会议全国委员会|中国人民政治协商会议全国委员会]]机关。

==沿革==
中国共产党中国人民政治协商会议全国委员会机关党组自[[中国人民政治协商会议第三届全国委员会|中国人民政治协商会议第三届全国委员会]]开始建立，以后各届也都设立<ref name=zgzxw/>。

==职责==
中国共产党中国人民政治协商会议全国委员会机关党组的任务主要是负责在中国人民政治协商会议全国委员会机关实现[[中国共产党|中国共产党]]的路线、方针、政策，讨论和决定中国人民政治协商会议全国委员会机关的重大问题，团结非党干部和群众，完成党和国家交给的任务，指导机关和直属单位中国共产党的组织工作<ref name=zgzxw/>。

自[[中国共产党中国人民政治协商会议全国委员会党组|中国共产党中国人民政治协商会议第七届全国委员会党组]]建立后，中国共产党中国人民政治协商会议全国委员会机关党组在[[中国共产党中国人民政治协商会议全国委员会党组|中国共产党中国人民政治协商会议全国委员会党组]]领导下工作<ref name=zgzxw/>。
