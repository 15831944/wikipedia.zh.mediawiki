'''中国出口信用保险公司'''（简称“中国信保”），是[[中华人民共和国|中华人民共和国]]“由国家出资设立、支持中国对外经济贸易发展与合作、具有独立法人地位的国有政策性保险公司”<ref name=sinosure>{{Cite web|url =http://www.sinosure.com.cn/sinosure/gywm/gsjj/gsjj.html|title =公司简介|author =|date =|publisher =中国出口信用保险公司|language =|accessdate =2017-04-06|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170702040658/http://www.sinosure.com.cn/sinosure/gywm/gsjj/gsjj.html|archivedate =2017年7月2日|df =}}</ref>。

==简介==
2001年12月18日，中国出口信用保险公司成立，当时是中国唯一承办出口信用保险业务的政策性保险公司。公司资本金约300亿元人民币，以出口信用保险风险基金作为资本来源，由国家财政预算安排<ref name=sohu/><ref name=sinosure/>。
 
2011年5月14日，[[中华人民共和国国务院|中华人民共和国国务院]]批复了中国出口信用保险公司改革实施总体方案和章程修订草案，进一步明确了中国出口信用保险公司的政策性定位，大幅度补充公司资本金。2011年6月底，[[中央汇金投资有限责任公司|中央汇金投资有限责任公司]]的200亿元人民币注资到位<ref name=sohu/>。
 
2011年11月，经[[中共中央政治局常委会|中共中央政治局常委会]]批准，中国出口信用保险公司领导班子列入中央管理。2012年3月17日，中国出口信用保险公司升级为副部级[[央企|央企]]<ref name=sohu>{{Cite web |url = http://business.sohu.com/20140605/n400470145.shtml |title = 中国出口信用保险公司原副总经理被开除党籍 |author =  |date =2014-06-05  |publisher =  搜狐|language =  |accessdate =  |quote =  }}</ref>。

中国出口信用保险公司由[[中华人民共和国财政部|中华人民共和国财政部]]直接领导，同时接受[[中华人民共和国商务部|中华人民共和国商务部]]和[[中华人民共和国外交部|中华人民共和国外交部]]的业务指导，并且接受[[中国保险监督管理委员会|中国保险监督管理委员会]]的监管<ref>卢越，中国出口信用保险发展对策研究，现代经济信息2015(17):140-141</ref>。
