'''中国人民解放军战略支援部队兴城疗养院'''，位于[[辽宁省|辽宁省]][[兴城市|兴城市]]，隶属[[中国人民解放军战略支援部队参谋部|中国人民解放军战略支援部队参谋部]]。

==简介==
2004年12月，由原“'''中国人民解放军总装备部兴城疗养院'''”、原“'''中国人民解放军空军兴城疗养院'''”合并组建“'''中国人民解放军沈阳军区兴城疗养院'''”，隶属[[中国人民解放军沈阳军区|中国人民解放军沈阳军区]]，下辖原“中国人民解放军海军第三一三医院”<ref>曾祥文、孙博、刘兆强，解放军沈阳军区兴城疗养院剪影，解放军医院管理杂志2014年第12期</ref>。沈阳军区兴城疗养院位于辽宁省兴城市海滨风景区东南角，兴海南街191号。主要承担军队人员疗养任务以及军队正骨按摩专业骨干培训任务。沈阳军区兴城疗养院是“全军颈腰痛康复中心”。并且是全军特色疗法“师带徒”单位、中国软组织疼痛研究会第五临床诊疗中心、沈阳军区理疗医学专科中心及正骨按摩中心；[[黑龙江中医药大学|黑龙江中医药大学]]及[[沈阳医学院|沈阳医学院]]临床实习基地；[[长春中医药大学|长春中医药大学]]研究生培养基地及针灸推拿学院临床教学基地<ref>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536754.htm |title = 沈阳军区兴城疗养院 |author =  |date = 2014-09-12 |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，沈阳军区兴城疗养院的上级单位沈阳军区撤销，沈阳军区兴城疗养院转隶新成立的[[中国人民解放军战略支援部队参谋部|中国人民解放军战略支援部队参谋部]]，更名为'''中国人民解放军战略支援部队兴城疗养院'''<ref>{{Cite web |url = http://gov.163.com/16/1124/11/C6KQD6KQ00238BUC.html |title = 军改就是要寻找“痛点”并“止痛” |author =  |date =2016-11-24  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。
