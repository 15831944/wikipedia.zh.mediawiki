'''中国人民政治协商会议第一届全体会议宣言'''于1949年9月30日由[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]通过，宣布了[[中华人民共和国|中华人民共和国]]的成立，并介绍了此次大会的成果，以及新成立的[[中华人民共和国中央人民政府_(1949–1954)|中华人民共和国中央人民政府]]的施政纲领，号召全国人民组织起来，“拥护[[人民政府|人民政府]]和[[中国人民解放军|人民解放军]]，建立独立、民主、和平、统一、富强的[[新中国|新中国]]。”

==制定==
1949年6月15日，[[新政治协商会议筹备会|新政治协商会议筹备会]]在[[北平|北平]]成立并开始举行第一次全体会议。6月16日，新政治协商会议筹备会常务委员会第一次会议决定成立六个小组，由新政治协商会议筹备会常务委员会领导<ref name=dyxz>{{Cite web |url = http://www.cppcc.gov.cn/2011/09/26/ARTI1317001118843681.shtml |title = 新政治协商会议筹备会第一小组 |author =  |date =2011-09-26  |publisher =中国政协网  |language =  |accessdate =  |quote =  }}</ref>。其中，第五小组负责起草新政治协商会议的大会宣言（后定名《中国人民政治协商会议第一届全体会议宣言》）。第五小组第一次全体会议推定[[郭沫若|郭沫若]]、[[胡愈之|胡愈之]]、[[胡乔木|胡乔木]]草拟大会宣言初稿。第五小组第二次全体会议对郭沫若等提出的大会宣言草案进行修改<ref name=dwxz/>。

由于起草大会宣言的工作未按时完成，1949年9月16日，新政治协商会议筹备会常务委员会第六次会议同意将起草《中国人民政治协商会议第一届全体会议宣言》草案及拟订国旗、国徽图案的工作移由中国人民政治协商会议主席团决定<ref name=dashiji>{{Cite web |url =http://www.rmzxb.com.cn/zt/zx65zn/zzx/yjzx/379443.shtml  |title =第一届政协大事记  |author =  |date = 2014-09-17 |publisher = 人民政协网 |language =  |accessdate =  |quote =  }}</ref>。9月17日，新政治协商会议筹备会第二次全体会议决定将起草《中国人民政治协商会议第一届全体会议宣言》的工作移交中国人民政治协商会议第一届全体会议，并由原负责该工作的第五小组向中国人民政治协商会议第一届全体会议主席团提出报告。9月22日，中国人民政治协商会议第一届全体会议决定设立宣言起草委员会，筹备会第五小组的工作结束<ref name=dwxz>{{Cite web |url =http://www.cppcc.gov.cn/2011/09/26/ARTI1317001118843767.shtml  |title = 新政治协商会议筹备会第五小组 |author =  |date = 2011-09-26 |publisher = 中国政协网 |language =  |accessdate =  |quote =  }}</ref>。

担任新政治协商会议筹备会常务委员会主任的[[毛泽东|毛泽东]]，不仅亲自领导了《[[中国人民政治协商会议共同纲领|中国人民政治协商会议共同纲领]]》等文献的制定，还接受中国人民政治协商会议第一届全体会议委托，亲自起草了《中国人民政治协商会议第一届全体会议宣言》。该手稿共6页，现存[[中央档案馆|中央档案馆]]<ref name=mao>{{Cite web|url = http://gddazx.southcn.com/a/2015-04/27/content_123084043.htm|title = 毛泽东《中国人民政治协商会议第一届全体会议宣言（草案）》|author = |date = |publisher = 中国档案报2014年3月3日总第2577期，第三版|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20151016234958/http://gddazx.southcn.com/a/2015-04/27/content_123084043.htm|archive-date = 2015-10-16|dead-url = yes}}</ref>。宣言中“在人民领袖毛泽东主席领导之下”一句，是会议通过宣言时由代表们提议增加的<ref name=ws>{{Cite web |url =http://www.people.com.cn/GB/shizheng/8198/30446/30452/2199905.html  |title = 中国人民大团结万岁 |author =  |date =  |publisher = 人民网 |language =  |accessdate = 2018-02-14 |quote =  }}</ref>。

1949年9月30日，出席中国人民政治协商会议第一届全体会议的全体代表选举了[[中国人民政治协商会议第一届全国委员会|中国人民政治协商会议第一届全国委员会]]委员以及[[中华人民共和国中央人民政府_(1949–1954)|中华人民共和国中央人民政府]]主席、副主席、委员。下午6时，在大会统计选票期间，全体代表到[[天安门|天安门]]前，为[[人民英雄纪念碑|人民英雄纪念碑]]举行奠基典礼。随后代表们回到[[中南海|中南海]][[怀仁堂|怀仁堂]]继续开会。大会执行主席宣布了政协第一届全国委员会和中央人民政府委员会的选举结果。接着大会通过了《中国人民政治协商会议第一届全体会议宣言》和《慰问中国人民解放军电》，最后由中央人民政府副主席[[朱德|朱德]]致闭幕词。大会至此结束<ref name=dashiji/>。

1949年10月1日[[中华人民共和国开国大典|中华人民共和国开国大典]]当天，《中国人民政治协商会议第一届全体会议宣言》在《[[人民日报|人民日报]]》全文发表<ref name=ws/>。

==内容简介==
《中国人民政治协商会议第一届全体会议宣言》以“全国同胞们”开头，正文共分六段。前两段简要介绍了中国人民政治协商会议第一届全体会议胜利闭幕及代表情况<ref name=ws/>。

第三段回顾了中国一百多年以来中国人民在先进分子如领导[[辛亥革命|辛亥革命]]的伟大革命家[[孙中山|孙中山]]的领导下，“为了推翻[[帝国主义|帝国主义]]和中国反动政府的压迫”进行的不断的斗争，指出这一斗争如今由中国人民和[[中国人民解放军|人民解放军]]在[[中国共产党|中国共产党]]领导下取得了胜利。“当着我们举行会议的时候，中国人民已经战胜了自己的敌人，改变了中国的面貌，建立了中华人民共和国。我们四万万七千五百万中国人现在是站立起来了，我们民族的前途是无限光明的。”<ref name=ws/>此前9月21日毛泽东在中国人民政治协商会议第一届全体会议上即发表了《中国人民站起来了》的著名讲话<ref>{{Cite web |url =  http://www.china.com.cn/aboutchina/txt/2009-09/17/content_18545188.htm|title =1949年9月21日 毛泽东发表《中国人民站起来了》的著名讲话  |author =  |date =  2009-09-17|publisher = 中国网 |language =  |accessdate =  |quote =  }}</ref>。

第四段介绍了本次大会的主要成果，“在人民领袖毛泽东主席领导之下，我们的会议齐心一志，按照[[新民主主义|新民主主义]]的原则”完成了如下工作：
#制定了[[中国人民政治协商会议章程|中国人民政治协商会议组织法]]
#制定了[[中华人民共和国中央人民政府_(1949–1954)|中华人民共和国中央人民政府组织法]]
#制定了[[中国人民政治协商会议共同纲领|中国人民政治协商会议共同纲领]]
#决定了中华人民共和国定都于[[北京|北京]]
#制定了[[中华人民共和国国旗|中华人民共和国的国旗]]为[[五星红旗|五星红旗]]
#采用了[[义勇军进行曲|义勇军进行曲]]为[[中华人民共和国国歌|现时的国歌]]
#决定了中华人民共和国的[[纪年|纪年]]采用世界[[公元|公元]]
#选举了[[中国人民政治协商会议全国委员会|中国人民政治协商会议全国委员会]]
#选举了[[中华人民共和国中央人民政府委员会|中华人民共和国中央人民政府委员会]]<ref name=ws/><ref name=mao/>。

第五段宣布，“全国同胞们，[[中华人民共和国|中华人民共和国]]现已宣告成立，中国人民业已有了自己的中央政府。”并宣布了新成立的中华人民共和国中央人民政府的主要施政纲领：
#将遵照[[中国人民政治协商会议共同纲领|共同纲领]]在全中国境内实施[[人民民主专政|人民民主专政]]。
#将指挥[[中国人民解放军|人民解放军]]将[[第二次国共内战|革命战争]]进行到底，消灭残余敌军，解放[[中华人民共和国领土|全国领土]]，完成统一中国的伟大事业。
#将领导全国人民克服一切困难，进行大规模的经济建设和文化建设，扫除旧中国所留下来的贫困和愚昧，逐步地改善人民的物质生活和提高人民的文化生活。
#将保卫人民的利益，镇压一切[[反革命|反革命]]分子的阴谋活动。
#将加强人民的陆海空军，巩固国防，保卫领土主权完整，反对任何帝国主义国家的侵略。
#将联合一切爱好和平自由的国家、民族和人民，首先是联合[[苏联|苏联]]和[[共产主义阵营|各新民主国家]]，以为自己的盟友，共同反对帝国主义者挑拨战争的阴谋，争取[[世界和平|世界的持久和平]]<ref name=ws/>。

第六段号召全国同胞们进一步组织起来，用人民群众的集体力量，“拥护人民政府和人民解放军，建设独立民主和平统一富强的新中国。”<ref name=ws/>
