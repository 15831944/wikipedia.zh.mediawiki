'''中華民國與吉布地關係'''是指[[中華民國|中華民國]]與[[吉布地共和國|吉布地共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對吉布地的相關事務由'''{{link-en|駐沙烏地阿拉伯王國台北經濟文化代表處|Taipei Economic and Cultural Representative Office in the Kingdom of Saudi Arabia}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-191-a2058-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|吉布地簽證政策|Visa policy of Djibouti}}|{{link-en|吉布地公民簽證要求|Visa requirements for Djiboutian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。{{link-en|吉布地政府|Politics of Djibouti}}沒有給予[[世界政區索引|任何地區]]的公民[[簽證#签证优惠待遇|免簽證]]待遇，持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可在首都的[[吉布地－安布利國際機場|吉布地－安布利國際機場]]辦理[[簽證#落地签证|落地簽證]]入境，可停留1個月以上。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-191-a2058-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|吉布地護照|Djibouti passport}}的{{link-en|吉布地人口|Demographics of Djibouti|吉布地公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將吉布地歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2015年1月19日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-97-78835-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|吉布地經濟|Economy of Djibouti}}}}
===貿易===
2017年，吉布地是中華民國第114大貿易夥伴、第227大[[進口|進口]]夥伴（在[[非洲国家和地区列表|非洲國家]]排名倒數第2；在[[世界政區索引|世界經濟體]]倒數第10）、第93大[[出口贸易|出口]]夥伴。出口至吉布地的金額為3,528萬2,064[[美元|美元]]，年增58.516%；自吉布地進口的金額為1,162美元，年減98.022%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）3,528萬0,902美元，年增58.930%。※2016年，吉布地是中華民國第123大貿易夥伴、第194大進口夥伴、第106大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
