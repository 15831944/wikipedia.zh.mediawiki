'''中国人民解放军国防科技大学航天科学与工程学院'''，位于[[湖南省|湖南省]][[长沙市|长沙市]][[开福区|开福区]]德雅路109号，隶属[[中国人民解放军国防科技大学|中国人民解放军国防科技大学]]。

==沿革==
学院的前身是'''[[中国人民解放军军事工程学院|中国人民解放军军事工程学院]]导弹工程系'''。1956年7月，军事工程学院在全院选调两个班的学员转入导弹工程专业学习，1959年正式组建导弹工程系。1970年，导弹工程系随军事工程学院南迁[[长沙|长沙]]<ref name=benke/><ref name=senlin/><ref>航天与材料工程学院，高等教育研究学报2008(1)</ref>。1978年[[中国人民解放军国防科学技术大学|中国人民解放军国防科学技术大学]]成立后，设'''中国人民解放军国防科学技术大学应用力学系'''。1984年12月，经中央军委批准，应用力学系更名为'''中国人民解放军国防科学技术大学航天技术系'''。1986年12月，经中央军委批准，航天技术系兼研究所<ref name=gfkjdxshq>{{Cite web |url = http://www.nudt.edu.cn/about/history2.htm |title = 国防科技大学时期 |author =  |date =  |publisher = 国防科学技术大学 |language =  |accessdate =2017-06-03  |quote =  }}</ref>。1999年，航天技术系扩建为'''中国人民解放军国防科学技术大学航天与材料工程学院'''<ref>{{Cite web |url = http://zsb.nwpu.edu.cn/index.php?s=/home/article/detail/id/87.html |title =于起峰院士  |author =  |date = 2014-04-17  |publisher = 西北工业大学本科招生信息网 |language =  |accessdate =  |quote =  }}</ref><ref name=senlin/>。2013年，更名为'''中国人民解放军国防科学技术大学航天科学与工程学院'''<ref>谭怀鹰，江南工业集团和国防科大航天科学与工程学院签订战略合作框架协议成立飞行器联合研究中心，中国军转民2013(8):11</ref>。

至2017年改革前，学院已形成以3个一级学科（航空宇航科学与技术、材料科学与工程、力学）为主体的学科布局。2017年改革前，学院有3个[[博士后|博士后]]科研流动站，10个博士学科授权点，15个硕士学科授权点，4个技术类本科专业、4个指挥类本科专业，研究生培养横跨6个一级学科。学院拥有航空宇航推进理论与工程国家级重点学科，飞行器设计国家重点（培育）学科，航空宇航科学与技术、材料科学与工程、力学3个湖南省重点一级学科。在2012年全国学科评估中，航空宇航科学与技术学科位居全国第二名。学院主要承担航空航天和新材料领域高素质人才培养、基础研究、关键技术攻关、新型航天装备研制任务<ref name=benke>{{Cite web |url =http://www.gotonudt.cn/site/gfkdbkzsxxw/htkxygcxy/index.html  |title = 航天科学与工程学院 |author =  |date =  |publisher = 国防科学技术大学本科招生网 |language =  |accessdate = 2017-06-03 |quote =  }}</ref>。

2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，调整组建[[中国人民解放军国防科技大学|中国人民解放军国防科技大学]]，下设'''中国人民解放军国防科技大学航天科学与工程学院'''<ref>{{Cite web |url = http://www.nudt.edu.cn/Articleshow.asp?id=7836 |title = 国防科技大学航天科学与工程学院科研人员招聘启事 |author =  |date =2017-09-15  |publisher =  中国人民解放军国防科技大学|language =  |accessdate =  |quote =  }}</ref>。
