'''中華民國與莫三比克關係'''是指[[中華民國|中華民國]]與[[莫三比克共和國|莫三比克共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對莫三比克的相關事務由'''[[中華民國駐史瓦帝尼王國大使館|中華民國駐史瓦帝尼王國大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-214-3f38b-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
[[File:Voting_res_2758.png|thumb]]，[[莫三比克|莫三比克]]當時屬於[[葡萄牙|葡萄牙]]的海外省（[[葡屬東非|葡屬東非]]），因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境莫三比克。]]
[[File:Visa_requirements_for_Mozambican_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
===外交===
{{see also|中華民國外交|{{link-en|莫三比克外交|Foreign relations of Mozambique}}}}
1990年9月1日，簽署《[[中華民國交通部郵政總局|中華民國臺灣郵政總局]]與{{link-en|莫三比克郵政|Correios de Moçambique|莫三比克（共和國）郵政總局}}間國際[[特快专递|快捷郵件]]服務協定》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e8%8e%ab%e4%b8%89%e6%af%94%e5%85%8b&tycountry_e=%e8%8e%ab%e4%b8%89%e6%af%94%e5%85%8b&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|莫三比克簽證政策|{{link-en|莫三比克公民簽證要求|Visa requirements for Mozambican citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可在首都的[[馬布多國際機場|馬布多國際機場]]辦理觀光單次[[簽證#落地签证|落地簽證]]入境，停留最多30天。經由[[南非機場列表|南非國際機場]]轉機入出境莫三比克，不需申辦南非[[過境簽證|過境簽證]]；若出機場再轉搭陸上交通工具前往莫三比克，因已實際入境南非，故須申辦[[南非簽證政策|南非簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-214-3f38b-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|莫三比克護照|Mozambican passport}}的{{link-en|莫三比克人口|Demographics of Mozambique|莫三比克公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將莫三比克歸類為'''灰色警示'''：提醒注意（2017年4月28日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-268-ca85b-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）「2008年[[非洲|東南非]]原料採購暨貿易訪問團」於7月3日訪問[[莫三比克|莫三比克]]舉辦貿易洽談會。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/WebArchive/2039/97%E5%B9%B4%E4%BA%8B%E4%BE%8B.pdf |author = 公眾外交協調會 |title = 中國大陸阻撓我國際空間事例（97年事例）|publisher = 中華民國外交部}}</ref>

2013年，[[經濟部國際貿易局|經濟部國際貿易局]]挑選最具拓銷潛力的22國26城市推動執行「貿易尖兵任務」，其中[[非洲国家和地区列表|非洲]]3國的[[新興市場|新興市場]]就包括莫三比克。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622045&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2013年10月30日 |title = 貿易尖兵勇闖22國26城市 新興市場商機無窮 |publisher = 經貿透視網}}</ref>

2017年，莫三比克是中華民國第113大貿易夥伴、第92大[[進口|進口]]夥伴、第128大[[出口贸易|出口]]夥伴。出口至莫三比克的金額為811萬8,511[[美元|美元]]，年減10.056%；自莫三比克進口的金額為2,771萬0,792美元，年減28.714%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）1,959萬2,281美元，年減34.356%。※2016年，莫三比克是中華民國第102大貿易夥伴、第82大進口夥伴、第123大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
