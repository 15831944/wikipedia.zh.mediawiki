'''中華民國與布吉納法索關係'''是指[[中華民國|中華民國]]與[[布吉納法索|布吉納法索]]（1984年之前稱'''[[上伏塔共和國|上伏塔共和國]]'''）之間的關係。兩國於1961－1973年、1994－2018年有正式[[外交#外交上的承認|外交關係]]，第二次斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對布吉納法索的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-39-b3061-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===

{{see also|2018年中華民國邦交國斷交與代表機構更名事件}}
1961年12月14日，[[中華民國|中華民國]]與[[上伏塔共和国|上伏塔共和国]]建立大使級[[外交#外交上的承認|外交關係]]。<ref name="105年外交年鑑"/>於首都[[瓦加杜古|瓦加杜古]]設立'''中華民國駐上伏塔共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[大使|大使]]。

1973年9月15日，上伏塔與[[中華人民共和國|中華人民共和國]]建交，10月23日，中華民國與上伏塔斷交。<ref name="105年外交年鑑">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf|title=《中華民國105年外交年鑑》〈第三章 對外關係〉|publisher=中華民國外交部|date=2017-05|accessdate=2018-02-28|deadurl=no|archiveurl=https://web.archive.org/web/20180129004249/http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf|archivedate=2018-01-29}}</ref>

1984年8月4日，上伏塔更名為[[布吉納法索|布吉納法索]]。

1994年2月2日，兩國恢復中斷20年的大使級外交關係。於首都瓦加杜古設立'''中華民國駐布吉納法索大使館'''；於首都[[台北市|台北]]設立'''布吉納法索駐中華民國大使館'''，並互派大使。<ref name="105年外交年鑑"/>

2017年1月，傳出兩國的邦交不穩，布吉納法索外交部長巴里（Alpha Barry）主動釋出風聲，指[[中華人民共和國|北京]]開出500億[[美元|美元]]甚至更多補助做為復交的條件，但「布吉納法索拒絕了，因為[[台灣|台灣]]是我們的朋友和合作夥伴。我們合作得非常愉快，沒有理由重新考慮雙邊關係。」對此，[[中華人民共和國外交部|中華人民共和國外交部]]發言人[[華春瑩|華春瑩]]立即反擊，強調布吉納法索的說法是天方夜譚。<ref>{{Cite web |url = https://udn.com/news/story/7331/3160725 |author = 戴瑞芬 |date = 2018年5月24日 |title = 布吉納法索與台斷交 立場曾搖擺 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref>

====斷交====
2018年5月24日，布吉納法索與中華民國第二次斷交。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=8742DCE7A2A28761&s=9E27EE1F74D49767 |author = 公眾外交協調會 |date = 2018年5月24日 |title = 中華民國政府基於維護國家尊嚴，決定自即日起終止與布吉納法索的外交關係 |publisher = 中華民國外交部}}</ref>5月26日，與中華人民共和國復交。<ref>{{Cite web|url=http://www.fmprc.gov.cn/web/zyxw/t1562759.shtml|title=中国与布基纳法索复交|accessdate=2018-05-26|author=|date=2018-05-26|work=[[中华人民共和国外交部|中华人民共和国外交部]]|publisher=|language=zh-CN}}</ref>

2018年5月25日，{{link-en|美國國務院發言人|Spokesperson for the United States Department of State}}表示，「中國正在改變[[臺海現狀|台海的現狀]]，破壞數十年來得以維持和平、繁榮與發展的架構。」國務院並對布吉納法索的決定表示失望。<ref>{{Cite web |url = http://news.ltn.com.tw/news/world/breakingnews/2437824 |author = 曹郁芬 |date = 2018年5月25日 |title =台布斷交 美國務院批中國改變台海現狀破壞和平 |publisher = [[自由時報|自由時報]]}}</ref>

由於兩國已斷交，當時有一批送往布吉納法索價值約45萬美元的低階軍事物資在海上航運中，[[中華民國外交部|中華民國外交部]]與[[國防部軍備局|國防部軍備局]]下令，要求這批物資在抵達[[非洲|非洲]]某港口時不准卸貨，直接送回台灣。<ref>{{Cite web |url = https://www.ettoday.net/news/20180525/1177166.htm |author = 翁嫆琄 |date = 2018年5月25日 |title = 送布吉納法索45萬美元軍資 政府下令直接送回台灣 |publisher = [[ETtoday新聞雲|ETtoday新聞雲]]}}</ref>

布吉納法索是繼2018年5月1日[[中華民國與多明尼加關係|中華民國與多明尼加斷交]]後單月內出現的第二個斷交國，為20年首例，而一年內淨減少之邦交國數亦創20年來之紀錄；同時該年亦是繼2005年以來首度在一年內與兩個友邦終止外交關係。

==經濟==

===貿易===
2017年，布吉納法索是中華民國第151大貿易夥伴、第157大[[進口|進口]]夥伴、第127大[[出口贸易|出口]]夥伴。出口至布吉納法索的金額為822萬1,574[[美元|美元]]，年增18.627%；自布吉納法索進口的金額為70萬5,442美元，年增18.469%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）751萬6,132美元，年增18.642%。※2016年，布吉納法索是中華民國第155大貿易夥伴、第161大進口夥伴、第131大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]|date = 2018-02-26 |accessdate = 2018-02-28 |deadurl = no |archiveurl = https://web.archive.org/web/20160309193715/http://cus93.trade.gov.tw/FSCI/ |archivedate = 2016-03-09 }}</ref>。

出口至布吉納法索的主要項目：[[氰化鈉|氰化鈉]]、[[燈具|燈具]]、[[人工肾|人工肾]]裝置、[[智慧卡|智慧卡]]、其他靜電式[[逆變器|逆變器]]等；自布吉納法索進口的主要項目：[[芝麻|芝麻]]、[[棉花|棉花]]。<ref name="國家檔">{{Cite web |url = https://www.trade.gov.tw/World/ListArea.aspx?code=7020&nodeID=977&areaID=3&country=b645biD5ZCJ6YKj5rOV57Si&pw=3|title = 布吉納法索國家檔(Burkina Faso) |publisher =經濟部國際貿易局|date = 2016-07-21|accessdate = 2018-03-03}}</ref>

===投資===
根據[[中華民國經濟部|中華民國經濟部]]投資審議委員會統計，截至2017年，布吉納法索在台投資共4件，總投資額192萬美元。投資的產業包括：[[金融服務|金融保險業]]、支援[[服務業|服務業]]、[[住宿|住宿]]及[[餐飲業|餐飲業]]等；台灣對布吉納法索投資共4件。<ref name="國家檔"/>

==援助==
=== 农业 ===
兩國建交後，由於自1963年底起，[[中華人民共和國|中華人民共和國]]積極拓展與[[法國|法國]]、[[非洲国家和地区列表|非洲各國]]的外交關係，[[中華民國政府|中華民國政府]]便致力於穩固對非洲國家的邦交，主要方式即提供[[農業技術|農業技術]]援助，而[[上伏塔|上伏塔]]（[[布吉納法索|布吉納法索]]舊名）也是提供農技援助的對象之一。中華民國農耕隊在上伏塔開墾了波碧（Boulbi）、鹿達（Louda）與姑河（Kou-Bobo-Dioulasso）三個墾區，推廣[[水稻|水稻]]耕種。由於頗有成效，上伏塔成為中華民國在西非拓展農技援助的示範區，然而在1972年9月，上伏塔與中華人民共和國簽署了一份《經濟技術合作協定》，並在中華人民共和國承諾接手中華民國農耕隊仍在進行中的姑河計畫與修建Tombao鐵路之下，於1973年9月與中華人民共和國建交，中華民國於該年10月23日宣布中止兩國邦交，且撤回農耕隊。1994年，布吉納法索總統[[布莱兹·孔帕奥埃|布莱兹·孔帕奥埃]]因事前獲悉[[西非法郎|西非法郎]]即將[[貶值|貶值]]，恐引發經濟失衡，便迅速轉換承認，與中華民國復交，而中華民國方面則派出5人前鋒考察團。<ref>{{cite journal|author=王文隆 |title=〈鋤犁拓邦誼：臺灣農耕隊在上伏塔的國民外交〉|journal=《臺灣文獻》 |volume=63 |issue=3|date=2012-09-30 |publisher=[[國史館臺灣文獻館|國史館臺灣文獻館]] |issn=1016-457X |pages=125－150頁}}</ref>

1994年5月31日，成立駐布吉納法索農技團。<ref name="86年外交年鑑"/>歷年來，除了開墾[[巴格雷縣|巴格雷]]墾區[[白沃爾特河|那干貝河]]右岸1,200[[公頃|公頃]]水稻田外，並進行第永格列地區陸稻6,000公頃開發計畫，以達到糧食增產、自給自足。巴格雷墾區已於2009年完成{{link-en|技術移轉|Technology transfer}}，並將墾區業務與資材交由布吉納法索自行經營管理。另[[高粱酒|高粱酒]]生產計畫（酒廠設備可利用當地[[樹薯|樹薯]]生產藥用酒精）亦於2009年結束並移交；2004年7月起，協助執行發展養魚計畫。<ref name="93年外交年鑑"/><ref name="98年外交年鑑"/>

===醫療===
1994年6月21日，成立駐布吉納法索醫療團，負責支援該國第3大城[[庫杜古|古都古]]友誼醫院的門診及住院工作，以及每年提供醫療援款供該國購買[[医疗器械|醫療器材]]及[[藥品|藥品]]、選送醫事人員赴國外受訓、改善醫療基礎建設、舉辦短期醫學研究及講習班等。<ref name="86年外交年鑑"/><ref name="93年外交年鑑"/>由臺灣廠商興建有600床的「龔保雷國家醫院」於2011年9月1日啟用，台灣[[埔里基督教醫院|埔里基督教醫院]]協助開院與管理工作，<ref name="100年外交年鑑"/>2013年12月，完成[[醫院資訊系統|醫院管理系統]]架設，是布吉納法索首次引進臺灣的醫管系統。<ref name="102年外交年鑑"/>

===電力===
自2011年起，援助布吉納法索國民教育暨識字部執行「非洲一盞燈計畫」，以公開[[招標|招標]]方式採購自主型[[太陽能|太陽能]]燈，提升該國的[[基礎教育|基礎教育]]。另援助布吉納法索國營電力公司（SONABEL）在{{link-en|濟加地區|Ziga Department}}興建一座1.1[[瓦特#百萬瓦|百萬瓦]]的[[太陽能發電|太陽能發電廠]]，以協助該國解決電力不足問題，於2017年5月11日啟用。<ref name="106年外交年鑑"/>

2017年，經駐布吉納法索大使館遴選，有17名臺灣獎學金受獎生、4名國合會獎學金受獎生赴台就學；另選派24名該國學員赴台參加各項研習班（含[[遠朋班|遠朋班]]、國際[[人力資源|人力資源]]培訓研習班、[[國際土地政策研究訓練中心|國際土地政策研究訓練中心]]研習班）。<ref name="106年外交年鑑"/>

中華民國協助布吉納法索推動職業訓練體系可分為兩階段：2006年6月－2013年6月為第1階段，由[[行政院勞工委員會職業訓練局|行政院勞工委員會職業訓練局]]成立專案小組，協助布吉納法索推動「加強職業訓練計畫」，2008年已代訓首批高階主管及種籽師資。<ref name="97年外交年鑑"/>另在{{link-en|古都古大學|University of Koudougou}}設立工教系，以強化自行培養[[技職教育|技職教育]]及職業訓練師資的能力。<ref name="98年外交年鑑"/>[[濟尼亞雷|金亞雷]]職訓示範中心也於2010年12月21日竣工；<ref name="99年外交年鑑"/>2013年7月－2016年12月為第2階段，由國合會承辦，協助興建4所高職、2個職訓示範中心等。<ref name="102年外交年鑑"/>

====客運====
兩國無直航班機，可經由（不計航點遠近，截至2018年11月29日）：

{{ROC}}
*[[臺灣桃園國際機場|台北]]→[[阿塔蒂尔克国际机场|伊斯坦堡]]、[[戴高樂國際機場|巴黎]]，再轉機前往布吉納法索的[[瓦加杜古國際機場|瓦加杜古國際機場]]。

===駕車===
持有中華民國[[國際駕駛執照|國際駕駛執照]]或申請布吉納法索駕駛執照，並投保[[車險|車險]]，可在布吉納法索駕車不超過6個月。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-39-b3061-05-1.html |title = 生活資訊 |publisher = 外交部領事事務局}}</ref>
