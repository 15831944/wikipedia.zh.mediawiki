'''中国铁路济南局集团有限公司'''<ref>{{Cite web |url =  http://money.163.com/17/1023/15/D1EO750V002581PP.html|title = 中铁总公司制大进展:11家地方铁路集团新掌门亮相 |author =  |date = 2017-10-23  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>，原名'''济南铁路局'''，是[[中國鐵路總公司|中國鐵路總公司]]下属公司，管轄範圍大致為[[山東省|山東省]]境内及周边的铁路网，包括[[京滬鐵路|京滬鐵路]]和[[京九鐵路|京九鐵路]]於山東省內一段，以及省内的[[胶济铁路|胶济铁路]]、[[胶新铁路|胶新铁路]]、[[兖石铁路|兖石铁路]]、[[蓝烟铁路|蓝烟铁路]]等干线，分别连接[[青岛|青岛]]、[[烟台|烟台]]、[[日照|日照]]三大[[港口|港口]]。濟南鐵路局管理的线路总延展长度為8283.9公里，复线里程2125.8公里。下轄车站有310个（2007年），其中[[特等站_(中鐵)|特等站]]4个、[[一等站_(中鐵)|一等站]]10个、[[二等站_(中鐵)|二等站]]40个、三等及以下车站241个。

==沿革==
1993年，济南铁路局进行了工商登记<ref name=guapai/>。

2008年初，原济南铁路局徐州枢纽和[[陇海线|陇海线]]等山东省外的线路划归[[上海铁路局|上海铁路局]]，而[[烟大铁路轮渡|烟大铁路轮渡]]（包括旅顺西站）、[[邯济铁路|邯济铁路]]党政班子和威海地方铁路局机务段划归济南铁路局。济局在2007年度中国企业500强排名中名列第155，在全国各路局中居于中上。

2012年3月30日[[任纪善|任纪善]]就任济南铁路局局长。前任局长为[[瞿建明|瞿建明]]，于2008年10月27日就任，接替被免职的[[耿志修|耿志修]]。2008年间济南铁路局局长两度换人，都是因发生事故而被免职。2008年4月28日[[膠濟鐵路|膠濟鐵路]]发生[[2008年膠濟鐵路列車相撞事故|列車相撞事故]]，4月29日铁道部免去济南铁路局局长陈功的职务，由耿志修接替<ref>[http://news.sina.com.cn/c/2008-04-29/163315451929.shtml 铁道部免除原济南铁路局领导班子主要成员职务]</ref>。2008年10月13日，由济南铁路局济南机务段司机负责带道的[[CRH5|CRH5]]型0号综合检测车执行青岛至[[徐州|徐州]]的DJ5506次时，因司机的错误操作，列车在膠濟鐵路超速运行3分钟23秒。事故发生后耿志修被免职<ref>[http://www.ynet.com/view.jsp?oid=45336815 列车超速致济南铁路局局长换人]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，由瞿建明接替<ref>[http://www.ourail.com/viewthread.php?tid=34245 济局换新局长]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2017年11月4日，经[[国家工商行政管理总局|国家工商行政管理总局]]核准，济南铁路局名称变更为'''中国铁路济南局集团有限公司'''（(国)名称变核内字[2017]第5277号）<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1851596  |title = 15家铁路局将集体变身集团公司：名称变更已通过核准 |author =  |date =2017-11-06   |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月15日，在国家企业信用信息公示系统中，中国铁路济南局集团有限公司已完成工商登记变更。工商登记变更后，中国铁路济南局集团有限公司仍为中国铁路总公司100%控股的公司，公司由原先的全民所有制企业变成有限责任公司（非自然人投资或控股的法人独资）企业，注册资本由原先的519.34144亿元变更为1416.9059亿元人民币，经营范围增加铁路专用设备及相关工业设备的制造、安装、维修、销售、租赁，保险代理服务，工程总承包及设计、施工、监理和管理服务，旅游服务，广告业，煤炭、矿石批发零售，房地产开发经营和中介等<ref name=wu>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1865510|title =北京济南等14家铁路局正式变身路局集团：完成工商登记变更|author =  |date = 2017-11-15 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月19日，中国铁路济南局集团有限公司正式挂牌<ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1870782|title = 18个铁路局已改制为铁路局集团有限公司，今日正式挂牌 |author =  |date = 2017-11-19 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref name=guapai>{{Cite web |url = http://finance.ifeng.com/a/20171119/15803099_0.shtml |title = 18家铁路局集团改制后集中挂牌！附主要领导人简历 |author =  |date = 2017-11-19 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

==机务段==
机务段负责存放、维护、检修铁路机车，工作通常包括清洗、加油、加水、检修、除尘等。济南铁路局下辖以下机务段：
*济南机务段（济局济段）
*济南西机务段（济局西段）：下设济南西车间、聊城车间、兖州车间
*青岛机务段（济局青段）：下设青岛西车间、青岛车间、临淄车间
*威海市地方铁路威海机务段（威局威段）

'''现已撤销的机务段'''
*淄博机务段（济局淄段）
*兖州机务段（济局兖段）
*莱芜机务段（济局莱段）

==客运段==
[[File:201610_Metal_Plates_of_Jinan_Passenger_Service_Department.jpg|thumb]]
客运段负责管理客运列车乘务人员，担当本局管内的旅客列车的服务（包括内旅客列车乘务工作和餐饮服务）。济南铁路局下辖以下客运段：
*济南客运段
*青岛客运段

==车务段==
[[File:Linyi_Department_of_Train_Operation_(20170206163338).jpg|thumb]]
车务段负责列车运营控制指挥，管辖辖区内的各大小车站、货运和客运的计划和收入、列车的运行监控。济南铁路局下辖以下车务段：
*临沂车务段：[[胶新铁路|胶新铁路]][[五莲站|五莲站]]至[[红花站|红花站]]；[[兖石铁路|兖石铁路]][[阮家村站|阮家村站]]至[[石臼所站|石臼所站]]；东平铁路放城至平邑站；[[枣临铁路|枣临铁路]][[新兴站|新兴站]]至[[朱保站|朱保站]]。
*青岛西车务段
*兖州车务段

==列车事故==
*2018年1月25日，配属济南局青岛北动车所的CRH380BL-5522（长客产，定员1005）担当的G284/281次2车厢（ZY 552202，一等座车，定员56）因电气故障而冒烟失火，乘务组将乘客疏散至其他车厢。列车在京沪高铁定远站停车后，乘客从列车两侧车门撤离。事故未造成人员伤亡。
