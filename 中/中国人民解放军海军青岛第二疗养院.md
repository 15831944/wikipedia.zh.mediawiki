'''中国人民解放军海军青岛第二疗养院'''，位于[[山东省|山东省]][[青岛市|青岛市]]，隶属[[中国人民解放军海军后勤部|中国人民解放军海军后勤部]]。

==简介==
1977年6月，由济南军区后勤部第九分部的部分干部以及[[山东省军区|山东省军区]]独立第二师医院组建'''中国人民解放军济南军区青岛第二疗养院'''，隶属[[中国人民解放军济南军区|中国人民解放军济南军区]]后勤部建制。院长史国柱，政治委员唐德福。驻湛流干路8号<ref name=sdsqw>{{Cite web |url = http://lib.sdsqw.cn/bin/mse.exe?seachword=&K=b2&A=12&rec=20&run=13 |title =青岛市志·军事志 第一篇 驻军与军事机构  第四章 人民军队 第七节 医疗单位  |author =  |date =  |publisher = 山东省情网 |language =  |accessdate =2017-05-20  |quote =  }}</ref>。

1951年7月7日，'''中国人民解放军海军青岛干部疗养院'''成立。院长甄陶民。1954年7月21日，改成'''中国人民解放军第一五一疗养院'''。1969年11月，改成'''中国人民解放军海军青岛疗养院'''。驻芝泉路3号<ref name=sdsqw/>。

1951年8月，组建'''中国人民解放军空军青岛疗养院'''，隶属[[中国人民解放军空军后勤部|中国人民解放军空军后勤部]]建制。院长汤庭兴。驻芝泉路10号<ref name=sdsqw/>。

2005年3月28日，由原海军青岛疗养院、空军青岛疗养院合并整编成'''中国人民解放军济南军区青岛第二疗养院'''，院党委书记单守勤<ref>{{Cite web |url = http://news.163.com/09/0625/21/5CME2BS3000120GR.html |title = 济南军区青岛第二疗养院改革发展 |author =  |date = 2009-06-25 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，济南军区撤销，济南军区青岛第二疗养院转隶[[中国人民解放军海军后勤部|中国人民解放军海军后勤部]]，改称'''中国人民解放军海军青岛第二疗养院'''。海军青岛第二疗养院、[[中国人民解放军海军青岛第一疗养院|海军青岛第一疗养院]]由此成为海军唯一直属的两个疗养机构<ref>{{Cite web |url = http://mt.sohu.com/20170320/n483950791.shtml |title = 这不是一般的疗养院,在这里水兵可“满血复活”重返战场 |author =  |date = 2017-03-20 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。
