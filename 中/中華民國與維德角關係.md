'''中華民國與維德角關係'''是指[[中華民國|中華民國]]與[[維德角共和國|維德角共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對維德角的相關事務由'''{{link-en|駐葡萄牙台北經濟文化中心|Taipei Economic and Cultural Centre, Lisbon, Portugal}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-249-3fa7d-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|維德角簽證政策|{{link-en|維德角公民簽證要求|Visa requirements for Cape Verdean citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可在{{link-en|維德角機場列表|List of airports in Cape Verde|維德角的國際機場}}辦理[[簽證#落地签证|落地簽證]]入境，停留最多30天。經維德角轉機，若持有第三國機票與所需文件，可免申請[[過境簽證|過境簽證]]，但須在機場轉機區等待。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-249-3fa7d-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>該國是[[霍亂|霍亂]]及[[黃熱病|黃熱病]]疫區，入境須出示黃熱病[[疫苗接種|疫苗接種]]證明。<ref name="生活資訊">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-249-3fa7d-05-1.html |title = 生活資訊 |publisher = 外交部領事事務局}}</ref>

持{{link-en|維德角護照|Cape Verdean passport}}的{{link-en|維德角人口|Demographics of Cape Verde|維德角公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將維德角歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2017年6月6日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-355-1f56b-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將維德角歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=B59BDA63945E9BB8 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
2017年，維德角是中華民國第219大貿易夥伴、第205大[[進口|進口]]夥伴、第213大[[出口贸易|出口]]夥伴，在[[非洲国家和地区列表|非洲國家]]排名倒數第3、5、6；在[[世界政區索引|世界經濟體]]倒數第20、32、18。出口至維德角的金額為6萬3,893[[美元|美元]]，年增3,915.902%；自維德角進口的金額為2萬6,539美元，年增6,357.178%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）3萬7,354美元，年增3,065.593%。※2016年，維德角是中華民國第229大貿易夥伴、第224大進口夥伴、第226大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
