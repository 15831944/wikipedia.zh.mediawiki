'''中国人民解放军航天员大队'''，对外又称'''中国航天员大队'''，现为[[中国人民解放军战略支援部队航天系统部|中国人民解放军战略支援部队航天系统部]]直属单位，负责中国[[航天员|航天员]]工作。

== 沿革 ==
1995年8月，[[中国载人航天工程指挥部|中国载人航天工程指挥部]]向[[中央军委|中央军委]]呈报了关于选拔航天员的请示，建议从空军现役飞行员中选拔预备航天员，同年9月批复同意。10月，根据中央军委指示，由原[[国防科工委|国防科工委]]和[[中国人民解放军空军|空军]]联合组成预备航天员选拔领导小组，从符合基本条件的空军飞行员中进行航天员初选。经反复研究和挑选，共有12人录取为预备航天员<ref name=rmw/>。

1997年12月，经中央军委批准，中国人民解放军航天员大队正式组建<ref name=rmw/>。1998年1月5日，中国人民解放军航天员大队在[[北京航天城|北京航天城]]成立，[[李庆龙|李庆龙]]、[[吴杰_(航天员)|吴杰]]、[[杨利伟|杨利伟]]、[[翟志刚|翟志刚]]、[[聂海胜|聂海胜]]、[[费俊龙|费俊龙]]、[[刘伯明_(航天员)|刘伯明]]、[[潘占春|潘占春]]、[[赵传东|赵传东]]、[[景海鹏|景海鹏]]、[[刘旺|刘旺]]、[[张晓光_(航天员)|张晓光]]、[[邓清明|邓清明]]、[[陈全_(航天员)|陈全]]共14人成为首批航天员，[[申行运|申行运]]任大队长<ref>[http://www.people.com.cn/GB/shizheng/1026/2308137.html 记中国人民解放军航天员大队航天员兼教练李庆龙]，人民网，2004-01-23</ref><ref>[http://mil.news.sina.com.cn/2003-11-14/162841.html 航天员是这样炼成的：登上“58个阶梯”上太空]，新浪，2003-11-14</ref><ref>[http://news.sina.com.cn/c/2008-09-23/173816342153.shtml 揭秘神七航天员梯队成员趣闻]，新浪，2008-09-23</ref>。同年8月，时任中央军委主席[[江泽民|江泽民]]签署《中国人民解放军航天员条例》，对航天员的训练工作、生活待遇和安全保卫等作出了规定<ref name=rmw/>。后来航天员大队定名“中国航天员大队”。2014年[[费俊龙|费俊龙]]参加活动时，媒体报道其职务是“中国人民解放军航天员大队大队长”；2016年2月[[聂海胜|聂海胜]]在公开报道里则被称为“中国航天员大队大队长”<ref>[http://www.dqdaily.com/guonei/2016-09/01/content_3943166.htm 刘洋当上妇联副主席 “神舟战队”6人跻身将军]，大庆网，2016-09-01</ref>。

中国航天员大队成立以来，共有14人次航天员执行6次载人飞行任务，共计巡游太空68日，绕地飞行1089圈，行程4600余万千米，完成100多项空间科学试验。而先后选拔的两批21名航天员，均通过了30多门基础理论知识和100多个科目训练的考核，全部具备独立执行载人航天飞行任务的能力，创造了世界航天员训练“零淘汰率”的纪录<ref name=zxw>{{cite web |url =http://www.chinanews.com/gn/2018/01-21/8429520.shtml|title=问鼎苍穹 胸怀九天——记中国人民解放军航天员大队  |website = 中国新闻网|date=2018-01-21 |accessdate=2018-01-23}}</ref>。

2003年11月，[[中央军委|中央军委]]授予航天员大队“英雄航天员大队”[[荣誉称号|荣誉称号]]<ref name="sina" />。2017年7月28日，中央军委主席[[习近平|习近平]]签署通令，给在完成任务中作出突出贡献的航天员大队记一等功<ref>{{cite web |url =http://www.thepaper.cn/newsDetail_forward_1745137|title=中央军委主席习近平签署通令，给2个单位、10名个人记功|website = 新华网|date=2017-07-28|accessdate=2018-01-23}}</ref>。

2018年1月5日，为纪念航天员大队成立20周年，11名曾经出征太空的航天员在内的全体航天员在北京航天城重温入队誓词<ref name=zxw/>。1月25日，[[中共中央宣传部|中共中央宣传部]]授予航天员大队（航天员群体）“[[时代楷模|时代楷模]]”荣誉称号<ref>{{cite web |url =http://www.xinhuanet.com/2018-01/25/c_1122314858.htm|title=中宣部授予航天员群体“时代楷模”荣誉称号 |website = 新华网|date=2018-01-25|accessdate=2018-01-25}}</ref>。

中国航天员大队原先隶属[[中国人民解放军总装备部|中国人民解放军总装备部]]。在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月撤销解放军总装备部，组建[[中央军委装备发展部|中央军委装备发展部]]，航天员大队随即成为装备发展部直属单位{{NoteTag|2016年3月全国“两会”，航天员[[刘洋_(航天员)|刘洋]]佩戴[[中央军委装备发展部|中央军委装备发展部]]直属单位臂章亮相。}}。2017年，中国航天员大队转隶[[中国人民解放军战略支援部队航天系统部|中国人民解放军战略支援部队航天系统部]]{{NoteTag|2017年8月，航天员[[景海鹏|景海鹏]]被授予“[[八一勋章_(2017年)|八一勋章]]”时佩戴[[战略支援部队|战略支援部队]]胸标；此后，航天员[[刘洋_(航天员)|刘洋]]佩戴战略支援部队臂章出席[[全国妇联|全国妇联]]会议。}}。
