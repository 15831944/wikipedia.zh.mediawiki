'''中华文明探源工程'''是继[[夏商周断代工程|夏商周断代工程]]之后又一项由中国官方立项的大规模多学科古史研究项目，主要目的是为了探索中华文明的起源问题。该项目的预研究项目于2002年列入[[中华人民共和国|中华人民共和国]]“[[十五计划|十五计划]]”<ref>{{cite article|url=http://www.arc.sdu.edu.cn/xueshuhd/qianyan/tanyuan.htm|title=“中华文明探源工程预研究”进展顺利|publisher=中国社会科学院院报|year=2002|author=王巍|access-date=2016-04-28|archive-url=https://web.archive.org/web/20161130131205/http://www.arc.sdu.edu.cn/xueshuhd/qianyan/tanyuan.htm|archive-date=2016-11-30|dead-url=yes}}</ref>，至2015年已进行第四期。

== 缘起 ==
中国史书常把[[炎黄|炎黄]]二帝作为中华文明的发端，并将5000年作为中华文明的历史跨度。但史书上对[[炎帝|炎帝]]、[[黄帝|黄帝]]至[[尧|尧]][[舜|舜]][[禹|禹]]的记载多带有神话色彩，因而无法作为信史。如果扣除仍然存疑的[[夏朝|夏朝]]，则剩余的时长仅剩3500餘年。自20世纪20年代以来，中国考古学的发展为研究早期中华文明提供了大量的资料。20世纪80年代以来，随着考古资料的丰富，学者确信夏朝时期，中国已经存在初步发展的文明，且[[三皇五帝|三皇五帝]]的传说也反应了一部分当时社会发展状况的信息。“九五”期间，实施的国家重点科技攻关计划项目“夏商周断代工程”开创了自然科学与人文科学相结合，研究人文社会科学问题的模式。而中华文明探源工程的目的就是在之前工作的基础上，探寻中华文明的起源、早期发展历程、特点以及发展状况<ref name="wangwei-intro">{{cite article|title=关于中华文明探源工程|url=http://kaogu.cn/cn/zhongdaketi/2013/1025/31399.html|publisher=中国文物报|author=王巍|author2=赵辉|date=2012-07-13|deadurl=yes|archiveurl=https://web.archive.org/web/20160530130126/http://kaogu.cn/cn/zhongdaketi/2013/1025/31399.html|archivedate=2016-05-30}}</ref>。

== 分期 ==
在正式工程开始前，2002至2003年，中华文明探源工程首先进行了预研究，涉及的时间范围为考古上的[[龙山文化|龙山时期]]，即公元前3000年至2000年，对应[[五帝|五帝]]时期。研究地域为中原地区。工程第一期于2004至2005年展开，涉及的时间范围为五帝至[[商代|商代]]初年，即公元前2500年至公元前1500年，研究地域仍为预研究阶段确定的中原地区。第二期于2006至2008年展开，将研究的时间跨度扩展到公元前3500年至公元前1500年，而地域也扩展至整个黄河流域以及长江中、下游和辽河流域<ref name="wangwei-intro" />。第三期于2009年至2013年进行。第四期于2013年至2015年进行<ref>{{cite web|url=http://cul.china.com.cn/lishi/2013-03/05/content_5773148.htm|title=中华文明探源第四阶段 比较研究世界古老文明|publisher=中国新闻网|date=2013-03-05|accessdate=2016-04-28}}</ref>。

2018年5月，中华文明探源成果公布<ref>[https://www.youtube.com/watch?v=Q7Y_O8cPEiA 央視官方頻道-中华文明探源成果公布]</ref>：在距今5000年前，中国已经进入了古国文明的阶段。<ref>{{Cite web|title=探源工程成果证实中华文明始于5000余年前|url=https://www.scio.gov.cn/xwfbh/xwbfbh/wqfbh/37601/38374/zy38378/Document/1630144/1630144.htm|work=中华人民共和国国务院新闻办公室|accessdate=2019-03-03|author=|date=2018-05-28|publisher=}}</ref>
