'''中華民國與斐濟關係'''是指[[中華民國|中華民國]]與[[斐濟共和國|斐濟共和國]]之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，但在1996年相互承認國家地位，也於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。<ref name="代表處簡介"/><ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-1.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>後來斐濟方面於2017年5月10日撤館。<ref name="撤館"/>

==交流==
===援助===
1978年，中華民國開始在斐濟派駐農技團，雙方多年來以政府名義簽署《[[農業技術|農業技術]]合作協定》，協助斐濟改良[[農作物|農作物]]及推廣熱帶蔬果生產；<ref name="駐館與駐地關係"/>依據簽署的《[[製糖|製糖]]技術合作協定》，中華民國協助斐濟發展糖業，派遣專家配合斐濟糖業研究中心從事[[水土流失|水土保持]]、[[水資源|水資源]]管理、植物營養、肥料、{{link-en|雜草控制|Weed control}}等多項研究計畫，以及考察發展製糖工業的可行性。<ref name="斐濟總理"/>此外，[[財團法人國際合作發展基金會|財團法人國際合作發展基金會]]（國合會）亦派遣技師協助斐濟規劃[[水產養殖|水產養殖]]發展計畫，並傳授養殖技術。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

[[中華民國外交#邦交國|太平洋友邦]]行動醫療團計畫包括互相承認但未建交的斐濟，並連續5年派遣[[馬偕紀念醫院|馬偕醫院]]行動醫療團在斐濟第1大島[[維提島|維提島]]及北島[[瓦努阿島|瓦努阿島]]義診。<ref name="駐館與駐地關係"/>2017年，[[國泰醫療財團法人|國泰綜合醫院]]規劃派遣行動醫療團2梯次（每團8人）至斐濟巡迴義診。本計畫是透過診療服務的[[臨床醫學|臨床]]交流達到{{link-en|技術轉移|Technology transfer}}目的，先收集特殊病例，再藉由醫療團隊與當地醫師共同診療，以提升當地醫師對特殊及複雜疾病的診療能力。<ref>{{Cite web |url = https://www.mofa.gov.tw/medicine/ProjectDetail.aspx?n=DC0982B4CACC386B&msn=24275AA207E835AC&csn=3A0809E259677F5D&news=8B3A7BE94CDDBF52&mednews=1237C1B39FE383C4&photosn=89E9C6BB0DCA761C |title = 國際醫療合作計畫 |publisher = 中華民國外交部}}</ref>

斐濟於2016年2月20日遭受[[南半球|南半球]]有紀錄以來最強烈的[[熱帶氣旋等級#澳洲與斐濟|五級熱帶氣旋]]－[[氣旋溫斯頓|溫斯頓]]侵襲，中華民國駐斐濟商務代表團於25日代表[[中華民國政府|中華民國政府]]捐贈{{link-en|斐濟政府|Politics of Fiji}}[[人道援助|人道援助]]款項，由{{link-en|政府大樓 (斐濟)|Government Buildings, Suva|斐濟總理府}}代表政府接受。<ref name="105年外交年鑑"/>

===學術===
中華民國每年提供國合會及[[中華民國教育部|教育部]]獎學金名額各1名，提供斐濟學生前來攻讀[[學士|學士]]及[[碩士|碩]]、[[博士|博士]]。斐濟學生亦可申請參與中華民國提供給[[太平洋島國論壇|太平洋島國論壇]]（PIF）秘書處的台灣獎學金計畫，就近申請就讀[[南太平洋大學|南太平洋大學]]。<ref name="駐館與駐地關係"/>
