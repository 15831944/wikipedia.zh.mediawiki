'''中国人民解放军空军航空医学研究所'''，位于[[北京市|北京市]]，隶属[[中国人民解放军空军后勤部|中国人民解放军空军后勤部]]。

==简介==
1954年8月30日，中国人民解放军空军航空医学研究所成立。当时该所是在[[苏联|苏联]]专家的建议下成立的<ref>{{Cite web |url = http://kj.81.cn/node_57718.htm |title = 专访中国工程院院士俞梦孙 |author =  |date = 2013-08-12 |publisher = 中国空军网 |language =  |accessdate =  |quote =  }}</ref><ref>硕果累累庆华诞 众志成城攀高峰——空军航空医学研究所建所40周年巡礼，航空军医1994年03期</ref>。1994年，在空军航空医学研究所建所40周年时，[[中央军事委员会副主席|中央军委副主席]][[刘华清|刘华清]]、[[中华人民共和国国防部|中华人民共和国国防部]]部长[[迟浩田|迟浩田]]、[[中华人民共和国国家科学技术委员会|中华人民共和国国家科学技术委员会]]主任[[宋健|宋健]]、[[中华人民共和国卫生部|中华人民共和国卫生部]]部长[[陈敏章|陈敏章]]、[[中国人民解放军总后勤部|中国人民解放军总后勤部]]副部长[[刘明璞|刘明璞]]等领导分别为该所题词<ref>空军副司令员杨振玉中将在庆祝空军航空医学研究所成立四十周年大会上的讲话，航空军医1994年03期</ref>。

2009年7月，都江堰、杭州、大连、青岛、临潼五所疗养院统一由各大军区联勤部转隶各大军区空军后勤部，成立5个航空医学鉴定训练中心，构建为“[[中国人民解放军空军总医院|空军总医院]]、空军航空医学研究所——区域性航空医学鉴定训练中心——航空兵部队卫生机构”三级航空医学鉴定训练体系<ref>何强、彭天国、董文，空军航空医学鉴定训练中心的发展与思考 ，全国航空航天医学学术会议暨云南省航空航天医学学术会议2010,27(3):180-181</ref><ref>{{Cite web |url = http://news.xinhuanet.com/mil/2011-12/25/c_111295752.htm |title = 一场从陆地打响的空战 |author =  |date = 2011-12-25 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

2013年9月22日，空军航空医学研究所、[[北京大学工学院|北京大学工学院]]共同成立[[中华人民共和国|中华人民共和国]]第一个健康系统工程研究所<ref>李晓龙，空军航空医学研究所与北京大学共同建立我国首个健康系统工程研究所，人人健康2013年22期</ref>。

==附属医院==
中国人民解放军空军航空医学研究所附属医院的前身是“'''华北军区空军医院'''”，于1951年3月8日由步兵九十五师、二〇五师、二〇六师医务人员组成。最初院址在[[北京市|北京市]][[德胜门内大街|德胜门内大街]][[新街口东街|蒋养房胡同]]。1952年更名为“'''第二十六空军医院'''”。1953年，该医院迁至今北京市[[海淀区|海淀区]]昌运宫15号。1954年更名为“'''中国人民解放军第四六六医院'''”。1959年更名为“'''中国人民解放军空军北京医院'''”。1993年复名“'''中国人民解放军第四六六医院'''”。2006年1月正式更名为“'''中国人民解放军空军航空医学研究所附属医院'''”。该医院是[[三级甲等医院|三级甲等综合性医院]]<ref>{{Cite web |url = http://baobao.sohu.com/20120516/n343236673.shtml |title = 北京466医院简介 |author =  |date = 2012-05-16 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
