'''中華民國與薩摩亞關係'''是指[[中華民國|中華民國]]與[[薩摩亞獨立國|薩摩亞獨立國]]（1997年之前稱'''西薩摩亞'''）之間的關係。兩國於1972－1975年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對薩摩亞的相關事務由'''中華民國駐[[斐濟|斐濟]][[台北经济文化代表处|商務代表團]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = http://www.roc-taiwan.org/fj/post/7.html |title = 代表處簡介 |publisher = 中華民國駐外單位聯合網站}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，[[薩摩亞|薩摩亞]]當時雖已獨立但未加入，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
{{see also|中華民國外交|{{link-en|薩摩亞外交|Foreign relations of Samoa}}}}
1972年5月29日，兩國建立大使級[[外交#外交上的承認|外交關係]]，但中華民國未在西薩摩亞設立[[外交代表機構#大使馆|大使館]]，由[[中国驻澳大利亚大使列表|駐澳大利亞大使]]兼任。

1974年5月3日，簽署《[[中華民國政府|中華民國政府]]與[[萨摩亚政治|西薩摩亞政府]][[漁業|漁業]]技術合作協定》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e8%a5%bf%e8%96%a9%e6%91%a9%e4%ba%9e&tycountry_e=%e8%a5%bf%e8%96%a9%e6%91%a9%e4%ba%9e&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>

1975年11月6日，西薩摩亞與中華民國斷交。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境薩摩亞。]]
[[File:Visa_requirements_for_Samoan_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|薩摩亞簽證政策|{{link-en|薩摩亞公民簽證要求|Visa requirements for Samoan citizens}}}}
目前[[国家列表_(按洲排列)|全世界]]的外籍公民都可以[[簽證#签证优惠待遇|免簽證]]的方式入境薩摩亞，持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可停留最多60天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-265-8c7c9-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&sms=B75E93A3DAB23A99 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|薩摩亞護照|Samoan passport}}的{{link-en|薩摩亞人口|Demographics of Samoa|薩摩亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將薩摩亞歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2016年12月8日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-416-9d3fb-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將薩摩亞歸類為'''第二級警示'''：對當地採取加強防護（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=13E834A970C6BDF2 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|薩摩亞經濟}}
===貿易===
2018年，薩摩亞是中華民國第166大貿易夥伴、第179大[[進口|進口]]夥伴、第156大[[出口贸易|出口]]夥伴。出口至薩摩亞的金額為284萬0,459[[美元|美元]]，年增33.769%；自薩摩亞進口的金額為22萬9,556美元，年增36.286%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）261萬0,903美元，年增33.552%。2017年，薩摩亞是中華民國第175大貿易夥伴、第180大進口夥伴、第159大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
