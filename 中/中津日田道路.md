'''中津日田道路'''（{{lang-ja|中津日田道路}}、假名：{{lang|ja|なかつひたどうろ}}），是一條連接[[定留IC|定留IC]]與[[耶馬溪山移IC|耶馬溪山移IC]]的[[公路|公路]]（地域高規格道路）。在1994年12月成為計畫路線。現時只有部分區間開通。


== 道路區間 ==
=== 中津港線1 ===
* 調査區間 2.5km
由起點[[中津港|中津港]]至[[大分縣道23號中津高田線|大分縣道23號中津高田線]]之間。

=== 中津港線2 ===
* 起點 : [[大分縣|大分縣]][[中津市|中津市]]大字定留
* 終點 : 大分縣中津市大字犬丸
* 長度 : 1.5km
* 規格 : 第1種3級
* 設計速度 : 80km/h
* [[行車線|行車線]] : 4條（[[暫定2車線|暫定2車線]]）
* 道路寬道 : 20.5m（暫定14.0m）
由中津高田線至[[國道213號|國道213號]]之間。在2000年開始興建。在2009年3月20日，以[[自動車専用道路|自動車専用道路]]暫定2車線型式啟用。

=== 中津道路 ===
* 起點 : 大分縣中津市大字犬丸
* 終點 : 大分縣中津市大字伊藤田
* 長度 : 2.1km
* 規格 : 第1種3級
* 設計速度 : 80km/h
* 行車線 : 4條（暫定2車線）
* 道路寬道 : 20.5m（暫定14.0m）
由國道213號至[[國道10號|國道10號]]之間。在2009年3月20日，以自動車専用道路暫定2車線型式啟用。

=== 中津三光道路 ===
* 起點 : 大分縣中津市大字伊藤田
* 終點 : 大分縣中津市三光西秣
* 長度 : 2.1km
* 行車線 : 2條
* 道路寬道 : 10.5m
在2006年開始興建。此區間會連接[[東九州自動車道|東九州自動車道]]，預計在2013年開通。

=== 三光本耶馬溪道路 ===
* 起點 : 大分縣中津市三光西秣
* 終點 : 大分縣中津市[[本耶馬溪町|本耶馬溪町]]落合
* 長度 : 12.8km
* 規格 : 第1種第3級
* 設計速度 : 80km/h
* 行車線 : 2條
* 道路寬道 : 12.0m
在2007年開始興建。

=== 本耶馬渓耶馬溪道路 ===
* 起點 : 中津市本耶馬渓町落合
* 終點 : 中津市[[耶馬溪町|耶馬溪町]]大字山移
* 長度 : 5.0km
* 規格 : 第1種第3級
* 設計速度:80km/h
* 行車線 : 4條（暫定2車線）
* 道路寬道 : 20.5m（暫定14.0m）
在1996年開始興建。在2012年3月31日開通。

=== 耶馬溪道路、耶馬溪山國道路、山國日田道路 ===
* 調査區間 5km
* 計畫路線 10km
* 調査區間 10km
