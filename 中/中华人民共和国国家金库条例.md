《'''中华人民共和国国家金库条例'''》是一部中华人民共和国为统一组织国家财政收支、健全[[国库|国家金库]]制度而制定和颁布实施的法律<ref name="law">{{Cite web |url=http://law.npc.gov.cn/FLFG/flfgByID.action?txtid=2&flfgID=3815&showDetailType=QW |title=中华人民共和国国家金库条例 |accessdate=2018-07-24 |author=国务院 |date=1985-07-27 |publisher=全国人大法律法规库}}</ref>。该法共6章23条，于1985年7月27日由[[中华人民共和国国务院|中华人民共和国国务院]]以国发[1985]96号文颁布，并于同日开始施行。

== 立法背景 ==
《中华人民共和国国家金库条例》颁布实施前，行使其规范[[中华人民共和国国家金库|国库]]管理职能的是1950年3月由[[中央人民政府政务院|中央人民政府政务院]]第22次会议通过并公布的《中央金库管理条例》<ref name="bg1">{{Cite journal |title=中央金库条例 |author=中央人民政府政务院 |url=http://www.cnki.com.cn/Article/CJFDTotal-FJZB195003005.htm |journal=福建政报 |issue=3 |year=1950 |volume=2 |page=6-7}}</ref>，《中央金库管理条例》是在中华人民共和国建国初期制定的，没有成熟的金库管理经验，仅简要地以十二条约500字的篇幅对国库管理制度进行规定。随着国家经济发展，《中央金库管理条例实施细则》曾多次修订，但《中央金库管理条例》始终未作修订，条例已经不能满足国库管理的实际需要。在当时国家金库的管理过程中，已经出现了由于法规本身规定不明确所直接或间接引起的职责不明和人员编制短缺等问题<ref name="bg2">{{Cite journal |title=关于国库条例的修订 |author=徐树勋, 宫迎春 |url=http://www.cnki.net/kcms/doi/10.14115/j.cnki.zgcz.1985.09.032.html |journal=财政 |accessdate=2018-07-24 |issue=9 |doi=10.14115/j.cnki.zgcz.1985.09.032 |year=1985|page=23-24}}</ref>。在这种情况下，国务院意图通过制定和颁布新的法规，指导解决国库管理中遇到的困难，同时为国库管理工作提供更为完备的法律保证<ref name="bg3">{{Cite journal |title=我国国家金库的建立与发展 |author=蔡玉治 |url=http://www.cnki.net/kcms/doi/10.14115/j.cnki.zgcz.1985.09.017.html |journal=财政 |issue=9 |doi=10.14115/j.cnki.zgcz.1985.09.017 |year=1985 |page=22-23}}</ref>。
== 内容 ==
《中华人民共和国国家金库条例》共分6章23条，包括了总则、组织机构、职责权限、库款的收纳与退付、库款的支拨等章节<ref name="law"/>。与《中央金库管理条例》在法理上只能规制中央的国库不同，条例首次将地方金库纳入规制范围，并明确了各级的地位和领导关系，确认了各级金库之间的垂直领导关系。针对国库管理工作中人员编制短缺的问题，条例对国库管理机构的设置、人员配备进行了明确，指出国库由[[中国人民银行|中国人民银行]]具体经理国库，并配备编制单列的机构和人员。针对权责不明的情况，条例对国库收纳、划分、留解和拨付等各项职责和权限作出了明确规定<ref name="law"/><ref name="bg2"/>。

== 与其他法律法规关系 ==
=== 上位法 ===
《中华人民共和国国家金库条例》颁布之初，条例并没有上位法，在《国家金库条例》颁布后陆续颁布施行的《中华人民共和国银行管理暂行条例》和《国家预算管理条例》在内容上对国家金库条例有一定指导意义<ref name="law2">{{Cite web |url=http://law.npc.gov.cn/FLFG/flfgByID.action?flfgID=11928&showDetailType=QW&keyword=&zlsxid=02 |title=中华人民共和国银行管理暂行条例 |accessdate=2018-07-24 |author=国务院 |date=1986-01-07 |publisher=全国人大法律法规库}}</ref><ref name="law3">{{Cite web |url=http://law.npc.gov.cn/FLFG/flfgByID.action?flfgID=11998&showDetailType=QW&keyword=&zlsxid=02 |title=国家预算管理条例 |accessdate=2018-07-24 |author=国务院 |date=1991-10-21 |publisher=全国人大法律法规库}}</ref>。但由于上述两部法规晚于《国家金库条例》颁布且三部法规均为行政法规，故在法理和实质内容上，《国家金库条例》的内容没有清晰反映上述两部法规的内容<ref name="comment1">{{Cite journal |title=国库法规的建设与完善 |author=黄润, 周海涛 |url=http://kns.cnki.net/KCMS/detail/detail.aspx?dbcode=CJFQ&dbname=CJFD2007&filename=JRKJ200706017&uid=WEEvREcwSlJHSldRa1FhdkJkVWI3Nkh0MDI3V2pOODlHWi9STE4wREJOZz0=$9A4hF_YAuvQ5obgVAqNKPCYcEjKensW4ggI8Fm4gTkoUKaID8j8gFw!!&v=MTI5MTZZNFI4ZVgxTHV4WVM3RGgxVDNxVHJXTTFGckNVUkxLZllPUnJGeXptVkx6TEx6L0FaTEc0SHRiTXFZOUU= |journal=金融会计 |issue=6 |year=2007 |page=71-74}}</ref>。

1994和1995年，《[[中华人民共和国预算法|中华人民共和国预算法]]》和《[[中华人民共和国中国人民银行法|中华人民共和国中国人民银行法]]》相继经[[全国人民代表大会|全国人大]]通过并颁布施行<ref>{{Cite web |url=http://law.npc.gov.cn/FLFG/flfgByID.action?flfgID=34548362&showDetailType=QW&keyword=&zlsxid=01 |title=中华人民共和国预算法 |accessdate=2018-07-25 |author=全国人大 |date=1994-03-22 |publisher=全国人大法律法规库}}</ref><ref>{{Cite web |url=http://law.npc.gov.cn/FLFG/flfgByID.action?flfgID=198&zlsxid=01&showDetailType=QW |title=中华人民共和国中国人民银行法 |accessdate=2018-07-25 |author=全国人大 |date=1995-03-18 |publisher=全国人大法律法规库}}</ref>，《国家金库条例》有了真正意义上的上位法，国库设立的必要性等问题在法律体系中也得到了确认<ref name="comment1"/>。
=== 下位法 ===
《中华人民共和国国家金库条例》第二十一条规定“本条例实施细则，由[[中华人民共和国财政部|财政部]]和[[中国人民银行|中国人民银行]]总行共同制定”<ref name="law"/>。根据第二十一条的规定，财政部和中国人民银行于1986年1月3日制定并颁布实施了《中华人民共和国国家金库条例实施细则（试行）》<ref>{{Cite web |url=http://law.npc.gov.cn/FLFG/flfgByID.action?flfgID=25572&keyword=&zlsxid=04 |title=关于公布废止金融规章目录（第三批）的通知（银发[1993]349号） |accessdate=2018-07-25 |author=中国人民银行 |date=1993-12-10 |publisher=全国人大法律法规库}}</ref>，随后又于1990年根据试行的情况调整颁布了《中华人民共和国国家金库条例实施细则》<ref>{{Cite web |url=http://www.law-lib.com/law/law_view.asp?id=47210 |title=中华人民共和国国家金库条例实施细则（试行） |accessdate=2018-07-24 |author=中国人民银行, 财政部 |publisher=法律图书馆}}</ref>。《国家金库条例实施细则》对国库的机构人员管理和工作规范等方面进行了详细说明，作为《国家金库条例》的下位法具体地明确了条例实施过程中的操作规范和指导了国家金库实际管理工作<ref>{{Cite web |url=http://www.nbcs.gov.cn/art/2004/9/7/art_1780_24.html |title=中华人民共和国国家金库条例实施细则 |accessdate=2018-07-25 |author=中国人民银行, 财政部 |date=2004-09-07 |publisher=宁波市财政局}}</ref>。

== 法规实施与评价 ==
=== 实施 ===
在《国家金库条例》颁布施行前，《中央金库条例》没有规定应设立专门机构办理国库业务，故在相当长的历史时期内并无专设国库管理机构。《国家金库条例》颁布实施后，专设办理国库业务的机构开始逐步建立，至1998年各级国库机构陆续对外公开挂牌并负责国库管理和业务办理工作。此后随着《中华人民共和国预算法》等上位法的实施，国库也构建起了相对较为健全的管理机制<ref name="comment1"/>。
=== 评价 ===
《国家金库条例》颁布施行在实施上改善了国家金库的管理状况，但众多研究指出在当前状况下，条例仍然存在诸多问题。在条例的实施细则和《国库会计管理规定》、《国库会计核算
业务操作规程》等一系列规定多次修订颁布的情况下，1985年颁布的《国家金库条例》本身并没有进行修订<ref name="comment2">{{Cite journal |title=中国国库监管制度研究 |author=陈伟, 周应恒 |url=http://www.cnki.net/kcms/doi/10.16528/j.cnki.22-1054/f.2008.12.039.html |journal=经济纵横 |accessdate=2018-07-24 |issue=12 |doi=10.16528/j.cnki.22-1054/f.2008.12.039 |year=2008 |page=95-97}}</ref>。这造成了法律体系的法律法规存在一定的冲突缺位：《银行管理暂行条例》、《中国人民银行法》颁布实施后，人民银行的性质、职能、权限发生了很大的变化，而《国家金库条例》并未就这些变化跟进修订；《国家金库条例》规定有权查究处理，但其上位法《预算法》并未就此项权力进行确认；《国家金库条例》对1995年以来实行的[[中国1994年分税制改革|分税制]]也没有充分反映，一系列冲突造成《国家金库条例》实际管理的需要有较为明显的差异<ref name="comment1"/>。

除了长时间未修订造成的法规短板外，亦有研究指出法规构建的管理体系本身也值得改革。现有的《国家金库条例》构建的中国人民银行经理国库、政府财政部门负责使用国库资金的多头共管局面，淡化了经营主体，使得国库经营管理过程中权责不统一，国库资金所有者缺乏提高国库资金运行效率的意愿和运作国库的必要权力<ref name="comment3">{{Cite journal |title=国库现金管理法规背景的思考 |author=王瑛 |url=http://www.cnki.net/kcms/doi/10.19477/j.cnki.11-1077/f.2005.05.016.html |journal=财政研究 |accessdate=2018-07-24 |issue=5 |doi=10.19477/j.cnki.11-1077/f.2005.05.016 |year=2005 |page=40-43}}</ref>。
