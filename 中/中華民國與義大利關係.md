'''中華民國與義大利關係'''是指[[中華民國|中華民國]]與[[義大利共和國|義大利共和國]]之間的關係。1913－1970年，兩國有官方[[外交#外交上的承認|外交關係]]但曾經中斷，二次斷交後直至1990年代，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
{{see also|中華民國外交|義大利外交}}
{{see also|中国驻意大利大使列表{{!}}中華民國駐義大利王國／義大利共和國外交官列表|意大利驻华大使列表{{!}}義大利王國／義大利共和國駐中華民國外交官列表}}
1912年，中華民國成立後，向義大利派駐外交代表。

1913年10月6日，義大利[[外交#外交上的承認|外交承認]]中華民國[[北洋政府|北洋政府]]。於首都[[羅馬|羅馬]]設立'''中華民國駐[[意大利王國_(1861年–1946年)|義大利王國]][[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1928年11月27日，隨著[[國民革命軍北伐|國民革命軍北伐]]接近尾聲，義大利外交承認中華民國[[國民政府|國民政府]]。

1934年10月17日，建立大使級外交關係。公使館升格為'''中華民國駐義大利王國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1936年11月28日，義大利與[[大日本帝国|日本]]簽署協定，承認[[滿洲國|滿洲國]]。

1941年7月1日，義大利承認南京[[汪精衛政權|汪精衛政權]]，中華民國與意大利斷交。

1944年10月26日，中華民國承認義大利新政府（[[義大利與盟國停戰|義大利與盟國停戰]]），並恢復大使級外交關係。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1970年11月6日，與中華民國斷交。<ref name="103年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/09.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>9日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[義大利|義大利]]。]]
1971年10月25日，義大利與絕大多數歐洲國家支持[[中華人民共和國|中華人民共和國]]取得[[聯合國|聯合國]]裡的「[[中國與聯合國#代表權問題|中國席位]]」（中國代表權問題）。

1994年6月，[[義大利國會|義大利國會]]成立「[[國會|國會]]友台協會」，是推動雙方關係的重要平台；2008年11月，[[中華民國立法院|中華民國立法院]]則成立「中華民國－義大利[[國會議員|國會議員]]友好協會」。<ref name="駐館與駐地關係">{{Cite web |url = http://www.taiwanembassy.org/it/post/75.html |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref>

自1997年2月15日，義大利將其簽證直接蓋於[[中華民國護照|中華民國護照]]內頁，解決多年來以「通行證」替代正式簽證的不便。自同年11月3日，進一步核發《[[申根公約|申根公約]]》共同簽證。<ref name="87年外交年鑑"/>自2011年1月11日，與《申根公約》成員國同步給予持中華民國護照的旅客[[欧洲联盟签证政策|免簽證]]待遇。

2002年11月、2003年1月，兩國分別公告相互承認[[駕駛執照|駕駛執照]]並免試換照。<ref name="91年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__04.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1980年2月，[[意大利众议院|義大利眾議員]]史德嘉禮尼（Bruno Stegagnini）等人，於義大利成立'''義大利與[[中華民國在台灣|在臺灣之中華民國]]經濟文化友好協會'''（Association for the Friendship, Cultural and Economic Relations between Italy and the R.O.C. in Taiwan），作為兩國政府溝通的橋樑。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-4.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1989年9月，義大利外貿協會於首都[[台北市|台北]]設立[[實質大使館|具大使館性質]]的'''義大利貿易推廣[[辦事處|辦事處]]'''（Italian Trade Promotion Office）。1995年，另設立'''義大利經濟文化推廣辦事處'''，並自1996年7月起受理並核發[[簽證|簽證]]。2006年9月，將兩機構合併為'''義大利經濟貿易文化推廣辦事處'''（[[義大利文|義大利文]]：{{lang|it|Ufficio Italiano di Promozione Economica, Commerciale e Culturale}}）。<ref name="86年外交年鑑"/><ref name="103年外交年鑑"/><ref name="駐華外國機構">{{Cite web |url = http://www.mofa.gov.tw/OfficesInROC_Content.aspx?n=8CEB2B5F5436B997&sms=8EBFADC1592C7BFE&s=4ACFA38B877F185F&os=DB1708E9D8084B23 |title = 駐華外國機構 |publisher = 中華民國外交部}}</ref>

1990年8月7日，於首都[[羅馬|羅馬]]設立同等性質的'''台北文化經濟協會'''（義大利文：{{lang|it|Istituto Culturale ed Economico di Taipei}}）。1995年6月14日，更名為'''駐義大利臺北經濟文化辦事處'''（義大利文：{{lang|it|Ufficio Economico e Culturale di Taipei in Roma, Italia}}）。1996年7月2日，再更名為'''駐義大利[[台北经济文化代表处|台北代表處]]'''（義大利文：{{lang|it|Ufficio di Rappresentanza di Taipei in Italia}}）。<ref name="86年外交年鑑"/><ref name="103年外交年鑑"/><ref name="駐外館處">{{Cite web |url = http://www.mofa.gov.tw/OverseasOffice_Content.aspx?n=6357834932B83C83&sms=5A961ED2E4BA25E8&s=4ACFA38B877F185F&os=1E77F44E3255EFB0 |title = 駐外館處 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於義大利第2大城[[米蘭|米蘭]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#EUROPE |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]則在首都羅馬設立'''駐義大利代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2017年，義大利是中華民國第20大貿易夥伴、第21大[[進口|進口]]夥伴、第19大[[出口贸易|出口]]夥伴。出口至義大利的金額為21億3,340萬2,773[[美元|美元]]，年增14.663%；自義大利進口的金額為25億2,665萬6,467美元，年增14.611%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）3億9,325萬3,694美元，年增14.325%。※2016年，義大利是中華民國第19大貿易夥伴、第19大進口夥伴、第19大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資<small><ref>{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/it/pdf/104-48Italy.pdf |date = 2015年8月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref></small>===
根據經濟部投資審議委員會統計，1998年－2014年，[[臺灣商人|臺商]]對義大利的總投資金額為1億0,093萬美元，計有23件。多以[[航運|航運]]、[[航空|空運]]為主，其餘有[[電腦|電腦]]資訊產品、[[機車_(公路)|機車]]與零組件、[[服裝|服飾]]貿易、[[餐館|餐館]]及[[零售|零售]]等。

義商在台灣投資約有20餘家，總投資金額約7,800萬美元。主要為[[承包|承包]]大型[[建築工程|建築工程]]或與台商[[合資公司|合資]]建設公司、機車或[[汽車零組件列表|汽車零組件]]、化學[[制药工程|製藥]]及服飾時尚公司，主要仍在技術合作、貿易及售後服務等業務。

1995年起，[[中華航空|中華航空]]與[[義大利航空|義大利航空]]以[[班號共用|班號共用]]方式，開闢[[臺灣桃園國際機場|臺北]]↔[[羅馬-菲烏米奇諾機場|羅馬]]定期客運航線；1998年，[[長榮海運|長榮海運]]併購LLOYD TRIESTINO並改名{{link-en|ITALIA MARITTIMA S.p.A.|Italia Marittima}}，經營[[地中海|地中海]]及[[北部非洲|北非地區]]航運業務。2001年，在義大利南部[[塔蘭托|塔蘭托]]港設立[[貨櫃碼頭|貨櫃碼頭]]，成為地中海[[貨櫃|貨櫃]]集散中心。2006年，在[[利佛諾|利佛諾]]成立[[分公司|分公司]]，負責承攬業務；2003年，[[陽明海運|陽明海運]]則在[[熱那亞|熱那亞]]設立陽明義大利公司。

[[鴻海科技集團|鴻海科技集團]]、[[華碩電腦|華碩電腦]]、保銳科技等在[[米蘭|米蘭]]設立分公司或[[市場行銷|行銷]]據點；機車及零組件業者如[[三陽工業|三陽工業]]設立分公司、[[光陽工業|光陽工業]]等設代理商；2011年友嘉集團以1億5,000萬[[歐元|歐元]]併購當地3家[[工具機|工具機]]廠，2012年5月，又與一家義大利[[磨床|磨床]]廠簽訂合作[[備忘錄|備忘錄]]。

2013年2月，[[雲朗觀光集團|雲朗觀光集團]]分別購入[[義大利行政區劃|義大利北部與中部]]郊區的百年歷史[[莊園|莊園]][[酒店|飯店]]，正式進軍國際[[旅遊|旅遊]]業以發展[[連鎖店|連鎖產業]]（成立義大利飯店品牌：LDC Italian Hotels）。2015年10月，位於羅馬近郊的A ROMA飯店開幕；另有台商從事[[国际贸易|進出口貿易]]，以及在[[米蘭|米蘭]]、[[波隆那|波隆那]]、[[威尼斯|威尼斯]]及[[羅馬|羅馬]]等經營[[中國菜|中餐]]館。

目前成立的台商組織有：義大利台灣商會。
