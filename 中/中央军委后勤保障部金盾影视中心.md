'''中央军委后勤保障部金盾影视中心'''，位于[[北京市|北京市]]，隶属[[中央军委后勤保障部办公厅|中央军委后勤保障部办公厅]]。

==简介==
2012年3月，'''中国人民解放军总后勤部金盾影视中心'''成立，隶属中国人民解放军总后勤部司令部。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，改为'''中央军委后勤保障部金盾影视中心'''，隶属[[中央军委后勤保障部办公厅|中央军委后勤保障部办公厅]]<ref name=msh>{{Cite web |url =  http://www.msweekly.com/news/wenhua/2017/0222/80871.html|title = 金盾影视:弘杨主旋律传承红色文化 |author =  |date = 2017-02-22 |publisher = 民生网 |language =  |accessdate =  |quote =  }}</ref><ref name=gczw>{{Cite web |url = http://www.guancha.cn/Celebrity/2017_04_09_402684.shtml |title = 《人民的名义》里，竟藏真正的中央军委官员 |author =  |date =2017-04-09  |publisher =观察者网  |language =  |accessdate =  |quote =  }}</ref>。

该中心具有[[国家广电总局|国家广电总局]]颁发的甲等电视剧制作资质、广播电视节目制作经营许可证、互联网视听节目传播证，以及国家广电总局批准的制作播出网络剧、网络片、微电影的资质<ref name=msh/>。

该中心的作品有：电影《[[百团大战_(电影)|百团大战]]》、《[[开罗宣言_(电影)|开罗宣言]]》、《勇士》、《我不是王毛》、《特种兵王》等，电视剧《[[人民的名义|人民的名义]]》、《麻雀》、《二炮手》等<ref name=gczw/>。
