'''中国人民解放军临潼疗养院'''，位于[[陕西省|陕西省]][[西安市|西安市]][[临潼区|临潼区]]，隶属[[郑州联勤保障中心|郑州联勤保障中心]]。

==简介==
原先设有'''中国人民解放军兰州军区临潼第一疗养院'''、'''中国人民解放军兰州军区临潼第二疗养院'''<ref>王绍林、刘荣辉、李文章、郝海水，矿泉微量元素对人体的生理作用，中国疗养医学2003年03期</ref>。2004年至2005年间合并成立'''中国人民解放军兰州军区临潼疗养院'''，隶属[[中国人民解放军兰州军区|中国人民解放军兰州军区]]。其地点毗邻[[华清宫遗址|华清宫]]，东临[[秦始皇兵马俑博物馆|秦始皇兵马俑博物馆]]。疗养院拥有和[[华清池|华清池]]同出一源的温泉。2016年改革前，疗养院内建有健康管理中心、骨伤康复中心、文体活动中心，并先后被定为国家级健康管理示范基地、军队创新人才带教导师疗养基地、[[国家中医药管理局|国家中医药管理局]]“治未病”试点单位、[[中国人民解放军第四军医大学|中国人民解放军第四军医大学]]疗养康复专业本科生教学基地<ref>{{Cite web |url =http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536826.htm  |title = 兰州军区临潼疗养院 |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，该疗养院由[[中国人民解放军兰州军区|中国人民解放军兰州军区]]转隶[[郑州联勤保障中心|郑州联勤保障中心]]，更名'''中国人民解放军临潼疗养院'''<ref name=zhzh>{{Cite web |url =  http://zb.81.cn/content/2017-06/08/content_7632431.htm?from=singlemessage|title = 临潼疗养院在服务部队保障打赢上聚焦用力 |author =  |date =2017-06-08  |publisher =军报记者  |language =  |accessdate =  |quote =  }}</ref>。

2016年改革前，该疗养院设有第一疗养区<ref>{{Cite web |url =http://xblj.81.cn/jrwz/content/2016-10/13/content_7299119.htm  |title = 西部战区陆军安排部署年度干部集体疗养工作 |author =  |date = 2016-10-13  |publisher = 军报记者 |language =  |accessdate =  |quote =  }}</ref>、第二疗养区<ref>{{Cite web |url =http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536722.htm  |title = 兰州军区临潼疗养院第二疗养区 |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>、附属医院<ref>张建祥、白伍泉、赵双宝，中国人民解放军兰州军区临潼疗养院温泉水质分析，中国疗养医学2006,15(1):9-12</ref>。
