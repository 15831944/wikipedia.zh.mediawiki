'''中央商务区'''（{{Lang-en|Central Business District}}，'''CBD'''）是指一个大[[都市|都市]]裡[[商业|商业]]活动的集中地，這些區域可能同時是[[市中心|市中心]]，也有可能是市中心的一部分，甚至在市中心以外。一般來説，在歷史較悠久的城市，商務區經常是市中心的一部分，或者是市中心以外的新建城區，而在新興城市中商業中心區經常就是[[市中心|市中心]]。

[[日語|日語]]和[[朝鮮語|朝鮮語]]有「'''都心'''」或「'''繁華街'''」的用詞，意指都市或[[都會區|都會區]]的中心部分，兩者分別相當於[[中文|中文]]裏的“[[市中心|市中心]]”、以及“[[大街|大街]]”或「[[鬧區|鬧區]]」。「都心」一詞也逐漸於華文世界使用，尤其是[[臺灣|臺灣]]的新興都市計畫區，如[[新莊副都心|新莊副都心]]等。

在新興城市中，由於缺少歷史中心區，商業中心區和[[市中心|市中心]]經常吻合。此類情況在[[北美洲|北美洲]]特別普遍，在北美洲這種混合商業中心區暨市中心稱爲「{{lang|en|Downtown}}」（原意指「下城」，但衍生泛指城市中心的繁華商業區或鬧區），原因是[[紐約市|紐約市]]較早期發展的區域為[[曼哈頓島|曼哈頓島]]的南端，在地圖上剛好是城市的下方「Downtown」，後來紐約市逐漸向北發展，相對的就形成了「上城」（{{lang|en|Uptown}}）。因此在北美洲{{lang|en|Uptown}}可以泛指遠離市中心的市郊或是住宅區，{{lang|en|Downtown}}則成為市中心的代名詞。而在同樣是新興城市的[[澳大利亞|澳大利亞]]、[[新西蘭|新西蘭]]等地，某些城市將此類混合商業中心區暨市中心稱爲“{{lang|en|CBD}}”，而其他城市則將其稱爲“{{lang|en|City Centre}}”。

== 概述 ==

商業中心區起源于20世纪20年代的美国，意为商业会聚之地。商業中心區大多設於國家或地區的主要核心地帶，它集中了大量金融、商业、贸易、信息及中介服务机构，拥有大量商务办公、酒店、公寓等配套设施，具备完善的市政交通与通讯条件，是适应工业社会人们需要就近办理商务、提高效率的需要而产生的<ref>{{cite web |url = http://business.sohu.com/87/01/article214700187.shtml |title = 我们为什么要建CBD  我们要建多少CBD}}</ref>。區內的各種完善設施通常包括甲級商業大廈、大型購物中心、政府及公共機構、康樂文娛設施等。此外，區內的可達度極高，[[公路|公路]][[幹線|幹線]]、[[鐵路|鐵路]]、[[港口|港口]]均設於區內的便利位置，方便市民由各區往來。近年來，一些大城市繁華地段的用地已變得極為緊張，以至於商業中心區要向地下發展<ref>{{cite web |url = http://bj.house.sina.com.cn/news/2016-03-11/11106113893690517009087.shtml?wt_source=news_dbxw_bt02 |title = 探秘CBD地下商业王国}}</ref>。

另一方面，亦從都心发展出次一级商業中心或副都心地区。例如，[[东京|东京]]的都心地区在[[銀座|銀座]]、[[丸之內|丸之內]]、[[日本橋|日本橋]]、[[新宿|新宿]]一带，副都心地区在[[池袋|池袋]]、[[澁谷|澁谷]]、[[赤坂|赤坂]]、[[台場|台場]]一带；[[首爾|首爾]]的都心在[[中區_(首爾)|中区]]及[[鍾路區|鍾路區]]一带，副都心在[[江南区|江南区]]及[[瑞草区|瑞草区]]一带。[[香港|香港]]的商業中心區為[[中環|中環]]及[[銅鑼灣|銅鑼灣]]一帶。
