===貿易===
2013年5月21日，[[中華民國外交部|外交部]]委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）與11家台灣廠商籌組參加「第21屆[[維爾紐斯|維爾紐斯]]波海科技展（BALTTECHNIKA 2013）」。該展是立陶宛及[[波羅的海|波羅的海]]鄰近國家規模最大的工業科技展，[[台灣|台灣]]以國家館形象展出，並為該展最大的海外參展國。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=611494&url=/NewsSearch.asp |author = 鄭美麗 |date = 2013年5月27日 |title = 外貿協會首次組團參加立陶宛最大科技展 |publisher = 經貿透視網}}</ref>

2017年，立陶宛是中華民國第82大貿易夥伴、第96大[[進口|進口]]夥伴、第67大[[出口贸易|出口]]夥伴。出口至立陶宛的金額為1億1,981萬2,438[[美元|美元]]，年增5.509%；自立陶宛進口的金額為2,243萬1,276美元，年增15.334%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）9,738萬1,162美元，年增3.478%。※2016年，立陶宛是中華民國第81大貿易夥伴、第99大進口夥伴、第66大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
