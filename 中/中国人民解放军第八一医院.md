'''中国人民解放军第八一医院'''，位于[[江苏省|江苏省]][[南京市|南京市]][[秦淮区|秦淮区]]杨公井34标34号，隶属[[无锡联勤保障中心|无锡联勤保障中心]]，为[[三级甲等医院|三级甲等医院]]<ref>{{Cite book|title=南京卫生年鉴.2011|last=《南京卫生年鉴》编辑委员会编|first=|publisher=扬州：广陵书社|year=2011.12|isbn=7-80694-791-4|location=|pages=293-295}}</ref>。

==简介==
1947年1月，在[[山东|山东]][[郯城|郯城]]成立'''华东野战军卫生部直属医院'''。此后随作战部队转战[[江苏省|江苏省]]、[[山东省|山东省]]、[[安徽省|安徽省]]、[[上海市|上海市]]和[[沂蒙山|沂蒙山]]区，参加了[[孟良崮战役|孟良崮战役]]、[[豫东战役|豫东战役]]、[[淮海战役|淮海战役]]、[[渡江战役|渡江战役]]、[[上海战役|上海战役]]等。1949年3月更名为'''华东军区第三野战军后勤部卫生部直属医院'''，同年8月进驻[[南京|南京]]。1950年4月迁到杨公井34标34号（此处1941年为[[汪精卫政权|汪精卫政权]]的中央医院，1945年[[抗日战争|抗日战争]]胜利后为陆海空三军联勤总医院），同时从野战医院调整为驻军医院，取消野战医院的分队编制，建成以科室编制的分科医院，并更名为'''华东军区后勤部卫生部直属医院'''。1954年3月18日命名为'''中国人民解放军第八一医院'''。1983年到1986年，是南京军区军医学校教学医院。1978年被定为全军肿瘤专科中心<ref>{{Cite web |url = http://www.jssdfz.com/book/wsz2/DEFAULT.html |title = 江苏省志·卫生志（下） |author = 江苏省地方志编纂委员会 |date =  2000年|publisher = 江苏人民出版社  |language =  |accessdate =  |quote = 第十一章 西医 第六节 主要医院简介 }}</ref><ref>{{Cite web|url = http://www.81yy.com/page.php?cate=217183|title = 第八一医院概况|author = |date = |publisher = 中国人民解放军第八一医院|language = |accessdate = 2017-06-16|quote = |archive-url = https://web.archive.org/web/20170629095558/http://www.81yy.com/page.php?cate=217183|archive-date = 2017-06-29|dead-url = yes}}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第八一医院由[[中国人民解放军南京军区|中国人民解放军南京军区]]转隶[[无锡联勤保障中心|无锡联勤保障中心]]<ref>{{Cite web |url = http://www.huaxia.com/thjq/jsxw/dl/2017/04/5279880_2.html |title = 追踪沈阳联勤保障中心联勤干部王桂相的一天 |author =  |date =  2017-04-18 |publisher =  华夏经纬网|language =  |accessdate =  |quote =  }}</ref>。
