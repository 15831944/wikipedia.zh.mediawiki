'''中国共产党江苏省委员会'''（简称'''中共江苏省委'''）是[[中国共产党|中国共产党]]在[[江苏省|江苏省]]的领导机关，由中国共产党江苏省代表大会选举产生。在代表大会闭会期间，执行[[中国共产党中央委员会|中国共产党中央委员会]]的指示和中国共产党江苏省代表大会的决议，领导江苏省的工作，定期向[[中国共产党中央委员会|中国共产党中央委员会]]报告工作<ref>[http://news.xinhuanet.com/18cpcnc/2012-11/18/c_113714762_5.htm 中国共产党章程：党的地方组织]</ref>。现任中共江苏[[省委书记|省委书记]]是[[娄勤俭|娄勤俭]]。

== 历史沿革 ==
中共江苏省委最初是在[[四一二政变|四一二政变]]后成立于1927年6月，在[[国共内战|国共内战]]及[[抗战|抗战]]时期长期处于地下活动状态，至1943年1月被撤销改为中共中央华中局敌区工作部<ref>[http://www.jsdsw.org.cn/news_view.asp?newsid=132 江苏党史网：中共江苏省委（1927.6-1943.1）]</ref>。[[中华人民共和国|中华人民共和国]]建立后，江苏省恢复省治前夕于1952年11月重新成立中共江苏省委<ref>[http://dangshi.people.com.cn/n/2012/1122/c85037-19661566-2.html 中国共产党新闻网：1952年中共江苏省委重建]</ref>，成为江苏省地方政权的最高领导机构。[[文化大革命|文化大革命]]开始后，1967年1月，中共江苏省委被造反夺权，工作中断。1967年3月，江苏省实行[[军管|军管]]。1968年3月，[[江苏省革命委员会|江苏省革命委员会]]成立。1970年12月，重组中共江苏省委，与江苏省革委会合署。1974年11月，中共江苏省委与江苏省革委会开始分设机构，江苏省委恢复工作<ref>[http://www.jsdsw.org.cn/news_view.asp?newsid=136 江苏党史网：中共江苏省第五次党代会]</ref>。

== 工作职能 ==
中华人民共和国成立以前，中共江苏省委处于地下活动状态，主要职能是领导苏南、苏北和南京、上海等城市的工人运动、秘密党组织的斗争和统战工作。

中华人民共和国成立以后，中共成为执政党，江苏省委在名义上是中共在江苏省的各级地方组织的领导机关，负责管理江苏省的党务，但在“[[党领导一切|党领导一切]]”的体制下，与其它省级行政区一样，除了文革动乱时期以外，江苏省委是江苏省地方政权的实际掌控者和领导者，省委常委会作为其常设机构是江苏省事实上的最高行政决策机构。

== 机构驻地 ==
中华人民共和国成立以前，中共江苏省委驻地在上海。

中华人民共和国成立以后，南京成为江苏省会，中共江苏省委驻于南京市，初期办公地点在西康路33号[[美国驻中华民国大使馆旧址|美国驻中华民国大使馆旧址]]（现为西康宾馆），1960年之后迁入位于西侧马鞍山上的古林寺遗址，形成现在的北京西路70号省委大院。
