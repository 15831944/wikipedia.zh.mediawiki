'''中国人民解放军军乐团'''<ref name="gfb">[http://news.mod.gov.cn/big5/headlines/2012-07/16/content_4385117.htm 寫在解放軍軍樂團擴編成立60周年之際，中华人民共和国国防部，2012-07-16]</ref>，位于[[北京市|北京市]][[海淀区|海淀区]]车道沟8号，为[[中央军委政治工作部|中央军委政治工作部]]直属单位，是[[中华人民共和国|中华人民共和国]]的大型[[管樂器|管樂]]藝術表演團體之一，也是[[中华人民共和国|中华人民共和国]]三大[[军乐团|军乐团]]之一（其他两个分别为[[中国人民解放军海军军乐团|中国人民解放军海军军乐团]]和[[武警军乐团|武警军乐团]]<ref>[http://news.ifeng.com/a/20160507/48714843_0.shtml 三大军乐团走进上海高校 专场演出受师生热捧，凤凰网，2016-05-07]</ref>）。

== 简介 ==
[[File:Military_orchestras_of_the_Western_MD_and_the_People’s_Liberation_Army_of_China_03.jpg|thumb]]
中國人民解放軍軍樂團的前身是1948年成立的華北軍區軍樂隊。1947年10月，[[晋察冀边区|晋察冀军区]]野战兵团在[[清风店战役|清风店]]歼灭[[国民革命军第三军|国民革命军第三军]]主力，俘虏该军军长[[罗历戎|罗历戎]]，并收容了该军军乐队。在该军军乐队及晋察冀军区抗敌剧社乐队的基础上，成立了華北軍區軍樂隊。后来，华北军区军乐队又收编了[[北平市警察局|北平市警察局]]军乐队等等。

1949年8月，由华北军区军乐队、二十兵团军乐队、坦克师军乐队、北平公安总队军乐队组成了联合军乐队，在北苑开始了集训。1949年10月1日，在[[中华人民共和国开国大典|中華人民共和國開國大典]]上，聯合軍樂队第一次亮相，演奏了代國歌《[[义勇军进行曲|义勇军进行曲]]》。

1950年，在中華人民共和國建國一周年慶典上，軍樂隊游行經過[[天安门|天安门]]前時，[[毛泽东|毛泽东]]對代總參謀長[[聂荣臻|聂荣臻]]說：“中國人口這麼多，[[天安门广场|天安门广场]]這麼大，要成立千人軍樂團。”遵照毛澤東的指示，在聶榮臻的關懷下，具体筹建工作由华北军区政治部负责，罗浪、王建中、白森林、朱为流、靳明启5人组成筹备小组。

1952年7月10日，中國人民解放軍軍樂團在北京馬駒橋擴編成立，罗浪任团长，王建中任副团长，白森林任政委，朱为流任副政委。<ref name="gfb" /><ref>[http://news.xinhuanet.com/ttgg/2014-09/30/c_1112695549.htm 为国家奏乐，为民族喝彩——走近解放军军乐团]，新华网，2014-09-30</ref><ref>[http://www.xtrb.cn/epaper/xtrb/html/2013-11/30/content_456925.htm 中国人民解放军军乐团的光辉历程]，邢台日报，2013年11月30日</ref>

1957年5月1日，[[苏联元帅列表|蘇聯元帥]][[克利缅特·叶夫列莫维奇·伏罗希洛夫|克利缅特·叶夫列莫维奇·伏罗希洛夫]]應邀出席中華人民共和國慶祝“五一”國際勞動節活動。在天安門城樓上，伏羅希洛夫看到天安門廣場上已連續演奏数小時的軍樂團時，對毛澤東說︰“只有像中國這樣有紀律的人民才能做到這一點，他們簡直像從地底下長出來的樹一樣，站得那麼穩，幾個鐘頭動也不動。”<ref name="gfb" />

1972年2月，[[美国总统|美国总统]][[1972年尼克松访华|尼克松访华]]，歡迎儀式在[[北京首都国际机场|首都機場]]舉行。軍樂團在首都機場站立一個多小時。当晚，在[[周恩來|周恩來]]為尼克松舉行的歡迎宴會上，軍樂團演奏了美国的《牧場上的家》、《火雞在草堆里》等樂曲。<ref name="gfb" />

截至2012年，中國人民解放軍軍樂團先後完成了中華人民共和國成立以來的14次國慶閱兵以及歷次[[中国共产党全国代表大会|中国共产党全国代表大会]]，[[全国人民代表大会|全國人大]]、[[中国人民政治协商会议全国委员会|全國政協]]會議開幕式、閉幕式，香港、澳門回歸祖國政權交接儀式，[[2008年夏季奥林匹克运动会|北京奧運會]]、[[中國2010年上海世界博覽會|上海世博會]]等演奏任務7000多次，迎送过将近200多個國家的上千位國家元首、政府首腦、軍事代表團。<ref name="gfb" />

中國人民解放軍軍樂團的主要任务是为国家和军队的重大典礼、重要会议、迎送国宾等仪式举行音乐演奏，在艺术上着意追求中国气派。

1950年代，中國人民解放軍軍樂團創辦上海軍樂訓練班，開辦中华人民共和国第一所軍樂學校，后来又創建軍樂大專班。2006年，完善了軍樂專業的本科教育體系。<ref name="gfb" />

2000年代，中国人民解放军军乐团建成了[[解放军军乐厅|解放军军乐厅]]。该音乐厅隶属解放军军乐团，位于北京市[[海淀区|海淀区]][[紫竹院路|紫竹院路]]100号东侧长智路。<ref>[http://www.china.com.cn/info/2009-03/19/content_17470303.htm 解放军军乐厅，中国网，2009-03-19] {{webarchive|url=https://web.archive.org/web/20160831085256/http://www.china.com.cn/info/2009-03/19/content_17470303.htm |date=2016-08-31 }}</ref>

在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月，中国人民解放军军乐团原上级单位[[中国人民解放军总政治部|中国人民解放军总政治部]]撤销，中国人民解放军军乐团转隶新成立的[[中央军委政治工作部|中央军委政治工作部]]<ref>苏莉，军乐人生 不忘初心——记众廉军乐演奏艺术培训中心和其创始人冯鹤，时代报告·中国报告文学2016年11月号</ref>。

经[[中共中央|中共中央]]批准，自2018年1月1日起由[[中国人民解放军|中国人民解放军]]担负国旗护卫和礼炮鸣放任务。中国人民解放军从而取代了[[中国人民武装警察部队国旗护卫队|中国人民武装警察部队国旗护卫队]]、[[中国人民武装警察部队礼炮队|中国人民武装警察部队礼炮队]]。2018年1月1日7时36分，[[天安门广场|天安门广场]]升国旗仪式，首次由[[中国人民解放军陆海空三军仪仗队|中国人民解放军陆海空三军仪仗队]]和军乐团执行<ref>{{cite web |url = http://news.ifeng.com/a/20171230/54676388_0.shtml  |title =  元旦起由解放军担负国旗护卫任务|author =  |date = 2017-12-30 |website = 凤凰网 |language = zh-hans |accessdate =  |quote =  }}</ref>。中国人民解放军仪仗队自此开始担负天安门广场升降旗任务。中国人民武装警察部队国旗护卫队不再负责升降旗任务，番号同时撤销<ref>{{cite web|url =http://www.sohu.com/a/213325119_359863|title =武警名分已定：归中央军委领导 但不属于人民解放军序列|author =|date =2017-12-28|website =搜狐|language =zh-cn|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20171230225932/http://www.sohu.com/a/213325119_359863|archivedate =2017年12月30日|df =}}</ref>。

2018年，在[[深化国防和军队改革|深化国防和军队改革]]中，原八一电影制片厂、原总政歌舞团、原总政歌剧团、原总政话剧团、原总政军乐团合并成立[[中国人民解放军文化艺术中心|中国人民解放军文化艺术中心]]<ref>{{Cite web |url = http://news.sina.com.cn/c/nd/2018-02-02/doc-ifyrcsrw7757091.shtml |title =八一电影制片厂被曝遭裁撤 曾拍摄芳华士兵突击等  |author =  |date =2018-02-02  |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref name=changbiao>{{Cite web |url =http://3g.163.com/dy/article/DKQRNFE7051780GA.html  |title =“解放军文化艺术中心电影电视制作部”成立 八一厂厂标不会摘  |author =  |date =2018-06-21  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

== 主要礼仪活动及曲目 ==
=== 旗礼 ===
* 出国旗：《[[歌唱祖国|歌唱祖国]]》
* 升降国旗：《[[义勇军进行曲|中华人民共和国国歌]]》

=== 国宾欢迎仪式 ===
* 各国[[国歌列表|国歌]]、《[[义勇军进行曲|中华人民共和国国歌]]》
* 检阅及分列式：《[[检阅进行曲|检阅进行曲]]》、《[[分列式进行曲|分列式进行曲]]》、《[[中國人民解放軍進行曲|中国人民解放军军歌]]》

=== 会议 ===
* 迎送[[党和国家领导人|党和国家领导人]]和贵宾：《[[欢迎进行曲|欢迎进行曲]]》、《[[欢送进行曲|欢送进行曲]]》、《[[辉煌时刻|辉煌时刻]]》
* 开闭会：《[[中华人民共和国国歌|中华人民共和国国歌]]》、《[[国际歌|国际歌]]》
* 党和国家领导人接见、颁奖：《[[团结友谊进行曲|团结友谊进行曲]]》、《[[辉煌时刻|辉煌时刻]]》

=== 悼念活动 ===
* [[追悼会|追悼会]]或遗体告别仪式：《[[哀乐_(乐曲作品)|哀乐]]》（[[罗浪|罗浪]]作曲）、《[[葬礼进行曲|葬礼进行曲]]》（[[李桐树|李桐树]]作曲）
* 献花圈、花篮：《[[献花曲|献花曲]]》（罗浪作曲）
