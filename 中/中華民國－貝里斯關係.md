'''中華民國－貝里斯關係'''是指[[中華民國|中華民國]]與[[貝里斯|貝里斯]]之間的外交關係。

==建交歷程==
1981年貝里斯獨立之後，一位具有中国血统的貝里斯商人[[伍永泉_(外交官)|伍永泉]]（William Quinto）與貝里斯人民联合党的支持者，开始透过{{link-en|赛义德·穆萨|Said Musa}}（Said Wilbert Musa）游说当时的貝里斯总理[[喬治·卡德爾·普賴斯|-{zh-cn:乔治·卡德尔·普赖斯;zh-tw:喬治·卡德爾·蒲萊士}-]]（George Cadle Price），与中華民國建立关系。<ref name="NPF">{{cite news|url=http://www.npf.org.tw/post/1/4283|title=送別伍永泉大使 憶當年建交經過|work=[[財團法人國家政策研究基金會|財團法人國家政策研究基金會]]|date=2008-05-26|accessdate=2012-04-22}}</ref>

1984年5月，当时的[[中華民國|中華民國]]驻[[危地马拉|危地马拉]]的大使[[陸以正|陸以正]]来到貝里斯，会见了伍永泉和-{zh-cn:普赖斯;zh-tw:蒲萊士}-總理，以讨论雙方建立关系的可能性，也得到-{zh-cn:普赖斯;zh-tw:蒲萊士}-總理同意，但陸以正大使返瓜國後，當時因瓜地馬拉主張貝里斯屬於危地马拉國土，因此中華民國與貝里斯建交案遭到當時危地马拉副總統{{link-en|鲁道夫·罗伯斯·萨莫拉|Rodolfo Lobos Zamora}}（Rodolfo Lobos Zamora）反對，且在瓜國內閣會議中主張，如中華民國承認貝里斯，危地马拉就與中華民國斷交，陸以正大使急電中華民國政府，強調不能因小失大，又加上-{zh-cn:普赖斯;zh-tw:蒲萊士}-總理所屬人民聯合黨在當年國會大選落敗，由在野黨聯合民主黨執政，中華民國政府只好把建交案暫時擱置。<ref name="NPF">{{cite news|url=http://www.npf.org.tw/post/1/4283|title=送別伍永泉大使 憶當年建交經過|work=[[財團法人國家政策研究基金會|財團法人國家政策研究基金會]]|date=2008-05-26|accessdate=2012-04-22}}</ref>

1989年9月，-{zh-cn:普赖斯;zh-tw:蒲萊士}-重任總理，任命-{zh-cn:赛义德·穆萨;zh-tw:賽義德·穆沙}-擔任外交部長，雙方重新接觸，並在1989年10月13日正式建交。<ref name="NPF">{{cite news|url=http://www.npf.org.tw/post/1/4283|title=送別伍永泉大使 憶當年建交經過|work=[[財團法人國家政策研究基金會|財團法人國家政策研究基金會]]|date=2008-05-26|accessdate=2012-04-22}}</ref>

== 歷史 ==
*1989年10月13日[[中華民國|中華民國]]與[[貝里斯|貝里斯]]建立大使級外交關係。<ref name="ROC Foreign Relation Yearbook 1997-3">{{cite book|title=中華民國外交部86年外交年鑑-第三章中外關係|publisher=[[中華民國外交部|中華民國外交部]]|date = 1998-06 |accessdate=2017-12-16}}</ref>

*1989年10月25日中華民國政府在貝國前首府[[貝里斯市|貝里斯市]]設立大使館。<ref name="ROC Foreign Relation Yearbook 1997-3">{{cite book|title=中華民國外交部86年外交年鑑-第三章中外關係|publisher=[[中華民國外交部|中華民國外交部]]|date = 1998-06 |accessdate=2017-12-16}}</ref>

*1992年9月21日貝里斯政府在[[台北|台北]]設立大使館，旋因財政拮据關閉。<ref name="ROC Foreign Relation Yearbook 1997-3">{{cite book|title=中華民國外交部86年外交年鑑-第三章中外關係|publisher=[[中華民國外交部|中華民國外交部]]|date = 1998-06 |accessdate=2017-12-16}}</ref>

*1999年8月由該國駐華大使[[伍永泉_(外交官)|伍永泉]]大使在台北復館。<ref name="ROC Foreign Relation Yearbook 1999">{{cite book|title=中華民國外交部88年外交年鑑-第三章中外關係|publisher=[[中華民國外交部|中華民國外交部]]|date = 2000-06 |accessdate=2017-12-21}}</ref>

==兩國貿易==
2018年貝里斯是中華民國第149名貿易伙伴，兩國貿易額8,662,113美元，其中中華民國對貝里斯出口6,987,615美元，是第132大出口國，進口1,674,498美元，是第147大進口國，與2017年貿易額相比貿易總額約增加7.00%、出口約增加13.09%、進口約減少12.64%<ref>{{cite book |url = http://cus93.trade.gov.tw/FSCI/ |title = 經濟部國際貿易局貿易統計系統 |publisher = [[經濟部國際貿易局|經濟部國際貿易局]] |date = 2019-02-23 |accessdate = 2019-02-27}}</ref>。
*中華民國－貝里斯主要出口項目：冷氣機、發電機、積體電路、腳踏車、塑膠製品、鋁製品等。<ref name=bft>{{Cite web |url = https://www.trade.gov.tw/World/ListArea.aspx?code=7020&nodeID=977&areaID=1&&country=b646LKd6YeM5pav|title = 各國經貿資訊 |publisher =[[經濟部國際貿易局|經濟部國際貿易局]]|date = 2018-02-08|accessdate = 2018-03-10}}</ref>
*中華民國－貝里斯主要進口項目：水產品、光碟片、敲擊樂器、空氣壓縮機、木材等。<ref name=bft>{{Cite web |url = https://www.trade.gov.tw/World/ListArea.aspx?code=7020&nodeID=977&areaID=1&&country=b646LKd6YeM5pav|title = 各國經貿資訊 |publisher =[[經濟部國際貿易局|經濟部國際貿易局]]|date = 2018-02-08|accessdate = 2018-03-10}}</ref>

===航空客運===
兩國目前無直航班機，來往貝里斯之客運航線有以下方式：

1.搭乘飛往{{flagicon|USA}}[[洛杉磯國際機場|洛杉磯]]、{{flagicon|USA}}[[歐海爾國際機場|芝加哥-歐海爾]]、{{flagicon|USA}}[[喬治布希洲際機場|休士頓-喬治·布希]]班機。

2.在各機場轉機可到達{{flagicon|BLZ}}[[貝里斯市|貝里斯市]]-[[菲利普戈尔德森国际机场|菲利浦·高森國際機場]]。
*上述目的地國際機場為鄰近該國首都或該國最主要樞紐機場，若要前往該國其他國際機場或國內線機場詳見{{link-en|貝里斯機場列表|List of airports in Belize|貝里斯機場}}。
