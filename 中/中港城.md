'''中港城'''（{{lang-en|'''China Hong Kong City'''}}）位於[[香港|香港]][[九龍|九龍]][[尖沙咀|尖沙咀]][[廣東道|廣東道]]33號，是一項綜合性物業，於1988年落成啟用，其獨特的金色幕牆及[[中國客運碼頭|中國客運碼頭]]及旅客設施，令它享有「通往中國的黃金之門」的美譽；<ref>{{cite web |language=zh-hk |url=http://hk.ulifestyle.com.hk/spot/detail.html?id=ABdGD1ozBwhZEgNu |title=中港城 |publisher=[[HK.ULifestyle.com.hk|HK.ULifestyle.com.hk]] }}</ref>並且榮獲[[健力士世界紀錄大全|健力士世界紀錄大全]]最大的金色玻璃幕牆建築物{{fact}}。中港城包括五座海景高智能寬頻主幹寫字摟、購物商場、皇家太平洋酒店、中國客運碼頭、巴士和的士總站及大型停車場。中港城臨近[[九龍公園|九龍公園]]，主要的交通工具均能直達中港城，步行往來[[尖沙咀天星碼頭|天星碼頭]]和[[尖沙咀站|尖沙咀站]]、[[柯士甸站|柯士甸站]]只需數分鐘。

==特色==
中港城商場內開設不同知名時裝品牌的特賣場（outlet）如：[[GEOX|GEOX]] Outlet<ref>{{cite web |language=zh-hk |url=http://hk.ulifestyle.com.hk/spot/detail.html?id=ABdGDVo3BwlZFANq   |title=GEOX Outlet (中港城) |publisher=[[HK.ULifestyle.com.hk|HK.ULifestyle.com.hk]] }}</ref>、Marathon Sport Outlet<ref>{{cite web |language=zh-hk |url=http://hk.ulifestyle.com.hk/spot/detail.html?id=ABdGDVo3BwlZFANi   |title=Marathon Outlet 馬拉松特賣場 |publisher=[[HK.ULifestyle.com.hk|HK.ULifestyle.com.hk]] }}</ref>、J Journey Outlet<ref>{{cite web |language=zh-hk |url=http://hk.ulifestyle.com.hk/spot/detail.html?id=ABdGDVo3BwlZFANp   |title=J Journey Outlet (中港城) |publisher=[[HK.ULifestyle.com.hk|HK.ULifestyle.com.hk]] }}</ref>、BeLLE Outlet、ALDO Outlet、Catalog Outlet、[[Esprit|Esprit]] Outlet、ISA Outlet等。<ref>{{cite web |language=zh-hk |url=http://www.discoverhongkong.com/tc/shop/where-to-shop/malls-and-department-stores/china-hong-kong-city.jsp   |title=中港城 |publisher=[[香港旅遊發展局|香港旅遊發展局]] }}</ref>

然而中港城商場規模上則遠遠不及毗鄰的[[海港城|海港城]]，人流亦較疏落。
