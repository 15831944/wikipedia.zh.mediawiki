'''中華民國與馬拉威關係'''是指[[中華民國|中華民國]]與[[馬拉威共和國|馬拉威共和國]]之間的關係。兩國於1966－2008年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對馬拉威的相關事務由'''{{link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-174-eaee3-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[馬拉威|馬拉威]]。]]
{{see also|中華民國外交|{{link-en|馬拉威外交|Foreign relations of Malawi}}}}
{{see also|中華民國駐馬拉威大使列表{{!}}中華民國駐馬拉威外交官列表|馬拉威駐中華民國大使列表{{!}}馬拉威駐中華民國外交官列表}}
馬拉威於1964年獨立時，即邀請中華民國參加獨立慶典。<ref name="86年外交年鑑"/>

1966年7月6日，馬拉威獨立兩週年暨改制共和國紀念，中華民國再次獲邀前往慶賀。7月11日，兩國[[外交#外交上的承認|建交]]。9月，於首都[[里朗威|里朗威]]設立'''中華民國駐馬拉威共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-3.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1967年8月，馬拉威總統[[海斯廷斯·卡穆祖·班達|班達]]訪問中華民國，並會晤總統[[蔣中正|蔣中正]]。

1999年12月，於首都[[台北市|台北]]設立'''馬拉威共和國駐中華民國大使館'''，並派駐大使。<ref name="89年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_7.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2007年12月28日，馬拉威與[[中华人民共和国|中华人民共和国]]建交。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=AA9409D48C1AD221&sms=EDAED6A15453D64B&s=A3B0A847C6476DAF |author = 公眾外交協調會 |date = 2011年4月10日 |title = 中國大陸阻撓我國際空間事例（96年10月至12月份事例）|publisher = 中華民國外交部}}</ref>2008年1月14日，中華民國與馬拉威斷交。<ref>{{Cite web |url =https://www.mofa.gov.tw/News_Content_M_2.aspx?n=70BCE89F4594745D&sms=700DE7A3F880BAE6&s=4E7DF422F4E73EB6 |author = 公眾外交協調會 |date = 2008年1月14日 |title = 楊政務次長宣布我中止與馬拉威共和國外交關係記者會答詢紀要 |publisher = 中華民國外交部}}</ref>

2009年2月，馬拉威總統[[賓古·瓦·穆塔里卡|莫泰加]]坦言馬拉威與中國建交後獲利未如預期，甚至比不上過去與台灣有邦交時所獲得的幫助，馬拉威當初與台灣斷交並與中國建交是錯誤的決定，自責「無法向馬拉威人民交代」。<ref>[http://news.ltn.com.tw/news/politics/paper/277259 馬拉威 後悔與台灣斷交]</ref>

===貿易===
2017年，馬拉威是中華民國第168大貿易夥伴、第142大[[進口|進口]]夥伴、第165大[[出口贸易|出口]]夥伴。出口至馬拉威的金額為172萬9,718[[美元|美元]]，年增20.905%；自馬拉威進口的金額為245萬5,314美元，年減20.253%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）72萬5,596美元，年減55.978%。※2016年，馬拉威是中華民國第167大貿易夥伴、第132大進口夥伴、第166大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===台商組織===
除「馬拉威台灣商會」外，另於2007年成立「馬拉威台灣商會首都分會」。<ref name="96年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2007/html/08-1-3.htm |title = 《中華民國96年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

==援助==
===農業===
1965年12月，中華民國派駐馬拉威農耕隊。1967年8月，馬拉威總統[[海斯廷斯·卡穆祖·班達|班達]]抵台訪問，要求擴大[[農業技術|農業技術]]合作。<ref name="86年外交年鑑"/>斷交前，中華民國技術團本部設於首都[[里朗威|里朗威]]，另設有[[糧食|糧食]]、[[園藝|園藝]]發展訓練暨推廣中心，以及[[蔬菜|蔬菜]][[種子|種子]]繁殖中心。協助發展園藝、糧食作物（以[[玉米|玉米]]為主，[[稻|稻作]]為輔）、小型農田水利等3項計畫。<ref name="90年外交年鑑"/>

===醫療===
2000年2月，中華民國派駐馬拉威[[醫療|醫療]]團，並委託[[長庚醫院|長庚醫院]]代訓[[專科醫師|專科醫師]]。此外，協助興建的[[姆祖祖|姆祖祖]]中央醫院於11月啟用，主要進行[[愛滋病|愛滋病]]研究與防治。<ref name="89年外交年鑑"/>

===教育===
2000年4月，派駐馬拉威職業訓練顧問團，協助里朗威技術學院發展[[技職教育|技職教育]]。<ref name="89年外交年鑑"/>除每年提供馬拉威官員短期職訓課程外，2004年起，提供「臺灣[[獎學金|獎學金]]」，以促進雙邊教育交流。財團法人國際合作發展基金會（國合會）亦於2007年提供「[[高等教育|高等教育]]獎學金」，給予馬拉威官員或教師赴台取得[[學位|學位]]。<ref name="96年外交年鑑"/>

===其他===
1991年3月，[[南部区_(马拉维)|馬拉威南部地區]]因連日豪雨導致山洪爆發，造成數百人死傷，[[中華民國政府|中華民國政府]]捐贈5萬[[美元|美元]]協助賑災救難工作。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=7F17B25FB5166711 |author = 公眾外交協調會 |date = 1991年3月20日 |title = 我友邦馬拉威共和國南部地區因連日豪雨於三月十日山洪爆發 |publisher = 中華民國外交部}}</ref>

1992年初，[[非洲南部|非洲南部]]嚴重缺雨造成大規模[[乾旱|乾旱]]，4月22日，中華民國政府捐贈300萬美元予馬拉威協助賑災救難之用。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=A4A345D25A3A034F |author = 公眾外交協調會 |date = 1992年4月24日 |title = 我撥款五百萬美元援贈非洲南部三友邦賑災 |publisher = 中華民國外交部}}</ref>
