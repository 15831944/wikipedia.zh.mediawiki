'''中華民國與匈牙利關係'''是指[[中華民國|中華民國]]與[[匈牙利|匈牙利]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，自1990年代，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
{{see also|中華民國外交|{{link-en|匈牙利外交|Foreign relations of Hungary}}}}
1912年，中華民國成立後，向[[匈牙利|匈牙利]]與[[奧地利|奧地利]]在[[1867年奧匈折衷方案|1867年合組]]的「[[奧匈帝國|奧匈帝國]]」派駐外交代表。

1913年10月6日，奧匈帝國[[外交#外交上的承認|外交承認]]中華民國。於[[維也納|維也納]]設立'''中華民國駐奧匈帝國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1917年8月14日，[[第一次世界大戰|第一次世界大戰]]爆發，中華民國對奧匈帝國[[宣戰|宣戰]]並斷交。（1918年，[[奧匈帝國#解體後的國家|奧匈帝國解體]]成奧地利、匈牙利與其他國家。）

1949年，[[中華人民共和國開國大典|中華人民共和國成立]]，加上1950－1990年代東西方陣營的「[[冷戰|冷戰]]」期間，中華民國在[[意識形態|意識形態]]和[[政體列表|制度]]上均偏向以[[美國|美國]]、[[英國|英國]]為首的[[資本主義|資本主義]]陣營，復加上以[[蘇聯|蘇聯]]為首的[[社會主義國家|共產集團國家]]的封閉，使雙方幾乎沒有往來。

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[匈牙利人民共和國|匈牙利人民共和國]]。]]
1971年10月25日，身為[[社會主義國家|共產陣營]]成員及蘇聯[[衛星國|衛星國]]之一的匈牙利支持[[中華人民共和國|中華人民共和國]]取得[[聯合國|聯合國]]裡的「[[中國與聯合國#代表權問題|中國席位]]」（中國代表權問題）。

1989年的「[[東歐民主化|東歐劇變]]」使匈牙利轉型為民主國家並重新對外開放，雙方始恢復往來。

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於首都[[布達佩斯|布達佩斯]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#EUROPE |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]也於布達佩斯設立'''駐匈牙利代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2010年4月2日，[[行政院農業委員會|行政院農業委員會]]（農委會）公告匈牙利為[[禽流感|禽流感]]非疫區國家，並開放進口該國的[[家禽|禽肉]]製品。<ref name="投資環境簡介"/>

2017年，匈牙利是中華民國第45大貿易夥伴、第48大[[進口|進口]]夥伴、第36大[[出口贸易|出口]]夥伴。出口至匈牙利的金額為5億3,227萬5,353[[美元|美元]]，年增1.574%；自匈牙利進口的金額為2億7,714萬9,736美元，年增12.818%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）2億5,512萬5,617美元，年減8.348%。※2016年，匈牙利是中華民國第45大貿易夥伴、第48大進口夥伴、第35大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =經濟部國際貿易局}}</ref>

===投資===
根據經濟部投資審議委員會統計，1992年－2013年，[[臺灣商人|臺商]]對匈牙利的總投資金額為1,489萬美元，共8件；匈牙利對臺灣的總投資金額為19萬美元，共3件。另根據匈牙利投資暨貿易局統計，臺商對匈牙利的總投資金額為3億9,150萬美元。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/hu/pdf/103-58Hungary.pdf |date = 2014年12月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

2003年，[[鴻海科技集團|鴻海科技集團]][[子公司|子公司]]的[[富智康集團|富智康集團]]於Komaron投資設廠，最初生產[[行動電話|行動電話]]及零組件，目前主要是組裝[[華為|華為]]的[[無線通訊|無線通訊]]系統；2008年，富智康於匈牙利[[塞克什白堡|塞克什白堡]]工業區設立[[電腦|電腦]]組裝廠。<ref name="投資環境簡介"/>

2005年，[[合勤科技|合勤科技]]在匈牙利設立[[分公司|分公司]]，從事網路通訊產品銷售；2007年，[[華碩電腦|華碩電腦]]設立分公司，從事[[筆記型電腦|筆記型電腦]]及行動電話銷售。<ref name="投資環境簡介"/>

目前在匈牙利的台商約16家，主要經營項目包括[[資訊及通訊科技|資通訊產品]]、[[鞋|鞋類]]、[[家具|家具]]、[[紡織品|紡織品]]及食品的[[批發|批發]]與[[零售|零售]]，以及[[食品產業|餐飲]]、[[旅遊|旅遊業]]等。1996年2月，台商成立「匈牙利台灣商會」；在匈牙利工作或居住的[[臺灣裔|僑民]]約80人。<ref name="投資環境簡介"/>
