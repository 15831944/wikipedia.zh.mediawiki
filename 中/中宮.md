'''中宮'''，只見於古代[[中國|中國]]及部分[[東亞|東亞]]地區，指的是[[紫微垣|紫微垣]]<ref>《宋史》卷四十九 志第二 天文二；《步天歌》</ref>或天極星<ref>《史記．卷二七．天官書》：「中宮天極星，其一明者，太一常居也。」 </ref>，「洛書軌跡」等風水或陰陽學，稱土為中宮<ref>《白虎通德論·五行》</ref>，或九宮、洛書九星中間為中宮，[[秦漢|秦漢]]以後[[王后|王后]]演變成[[皇后|皇后]]，則稱皇后居住的宮室，或稱皇后本身。原來王后的宮室「[[後宮|後宮]]」，反而不在中宮之列。

== 辭源 ==
君主時代在建設宮城時，大多將皇后的宮室建於子午線上，位於後宮的中心，因而稱之中宮。

== 中國 ==
中宮，或稱皇后<ref>《新唐書．卷一一二．馮元常傳》：「嘗密諫中宮權重，宜少抑，帝雖置其計，而內然之，由是為武后所惡。」</ref><ref>《封神演義》:	中宮元配皇后姜氏，西宮妃黃氏，馨慶宮妃楊氏。 </ref>，也象徵皇后居住的宮殿<ref>《漢書．卷一一．哀帝紀》：「春秋母以子貴，尊定陶太后曰恭皇太后，丁姬曰恭皇后，各置左右詹事，食邑如長信宮、中宮。」</ref>
，如[[漢朝|漢朝]][[長樂宮|長樂宮]]中的長秋宮、[[紫禁城|紫禁城]][[坤寧宮|坤寧宮]]等等，而坤寧宮雖自[[雍正|雍正]]以後不再作為皇后寢宮，但仍是實質意義上的中宮。

== 日本 ==
[[日本|日本]]的中宮，最早是皇后、[[皇太后|皇太后]]、[[太皇太后|太皇太后]]的總稱。自[[村上天皇|村上天皇]]立[[女御|女御]][[藤原安子|藤原安子]]為皇后起，中宮成為皇后別稱。[[一條天皇|一條天皇]]時，權臣[[藤原道長|藤原道長]]迫使一條天皇冊立兩位皇后，並為此將中宮作為獨立、等同於皇后的正式封號，從此天皇能同時擁有兩正配，造成'''一帝二-{后}-'''的[[平妻|平妻]]現象。此後諸多天皇在冊立皇后時，多先立為中宮，日後再立第二位正配時，便將先前的中宮改號為名義上較高的皇后，新立的第二位正配則是中宮。

自[[平安時代|平安時代]]前期開始，中務省轄下有所謂[[中宮職|中宮職]]，是專職服侍與負責與中宮相關的事物。在上下任天皇交替時，若前代天皇中宮仍享有中宮職的承職，則新任天皇皇后只能稱皇后。[[二戰|二戰]]以後，[[日本皇室|日本皇室]]實行[[一夫一妻制|一夫一妻制]]，中宮一號遂廢。最後一位中宮是[[光格天皇|光格天皇]]的中宮[[欣子內親王|欣子內親王]]。

== 朝鮮王朝 ==
[[朝鮮王朝|朝鮮王朝]]是[[中國|中國]]的[[朝貢國|朝貢國]]，其君主只能稱地位較次的[[國王|國王]]而非[[皇帝|皇帝]]，因此國王的正室於生前亦不得稱[[皇后|皇后]]，稱較次的[[王妃|王妃]]，死後方能[[諡號|追諡]]為[[王后|王后]]。

王妃在口語上的稱謂是'''中殿'''、'''坤殿'''或'''中宮'''，其宮室[[交泰殿|交泰殿]]亦稱做'''中宮殿'''。國王換代後，升為[[王大妃|王大妃]]或[[王大妃|大王大妃]]的王妃，口語上的稱謂則是'''上殿'''、'''慈殿'''。宮中僕役、臣民和[[嬪御|嬪御]]稱王妃為'''中殿娘娘'''（[[韓語|韓語]]：{{lang|ko|중전마마}}），{{lang|ko|마마}}是[[敬語|敬語]]，音譯「媽媽」，也通用於尊稱男性，相當於[[漢語|漢語]]的「[[娘娘|娘娘]]」、「爺」。
