'''中国翰园碑林'''位于[[河南省|河南省]][[开封市|开封市]][[龙亭公园|龙亭公园]]西侧，是[[国家4A级旅游景区|国家4A级旅游景区]]。

== 简介 ==
碑林是由开封市退休干部[[李公涛|李公涛]]节衣缩食，于1985年以只许投入、不许索取为原则发起创建的，20年来他全家共为碑林建设无偿投入1500多万元。李公涛先生因此被誉为“当代文化[[愚公|愚公]]”<ref name="愚公">{{cite web |url=http://www.hanyuan.cc/html/66/1/1925/1.htm |language= |title=当代文化愚公李公涛 |publisher=中国翰园碑林 |date=2009-05-22 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20120126024006/http://www.hanyuan.cc/html/66/1/1925/1.htm |archivedate=2012-01-26 }}</ref>。碑林被[[河南省|河南省]]教委以及20多家大、中、小[[学校|学校]]定为德育教育基地。<ref>{{cite web |url=http://www.tripdv.com/view/v/zl_32910.html |language= |title=中国翰园碑林 |publisher=旅游视频网 |date= |accessdate=2012年12月 |deadurl=yes |archiveurl=https://web.archive.org/web/20121118060051/http://www.tripdv.com/view/v/zl_32910.html |archivedate=2012-11-18 }}</ref>

碑林占地120亩，有当代书画巨匠名篇碑刻3500块，碑廊长3公里，与[[西安碑林|西安碑林]]、[[曲阜碑林|曲阜碑林]]齐名。碑刻以[[书法|书法]]艺术为主，集诗、书、画、印之大成，融[[碑刻|碑刻]]艺术和[[古典园林|古典园林]][[建筑|建筑]]艺术于一体。按表现内容和表现对象设中心碑廊、现代碑廊、[[宋代|宋代]]碑廊、历代帝王名臣碑廊、[[绘画|绘画]]碑廊、[[篆刻|篆刻]]碑廊、[[硬笔书法|硬笔书法]]碑廊、[[少数民族|少数民族]][[文字|文字]][[书法|书法]]碑廊、中年书法碑廊和国际友谊碑廊。<ref>{{cite web |url=http://www.kfly.com.cn/meijingkaifeng/china-hanyuanbeilin/ |language= |title=中国翰园碑林 |publisher=开放旅游网 |date=2009-05-16 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20090926202356/http://www.kfly.com.cn/meijingkaifeng/china-hanyuanbeilin/ |archivedate=2009-09-26 }}</ref>碑林被新闻界誉为“中国历史上第一座民办碑林”、“中国最大碑林”、“世界之最”。<ref name= "愚公"/>
