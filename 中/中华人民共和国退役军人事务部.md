'''中华人民共和国退役军人事务部'''，简称'''退役军人部'''，是[[中华人民共和国国务院|中华人民共和国国务院]]主管退役军人服务管理事务的[[国务院组成部门|组成部门]]<ref name="官方简称">{{cite web |author1=国务院办公厅秘书局 |title=关于转发国务院机构简称的通知 |url=http://www.gdllw.org/tzgg/201806/W020180622515099385181.pdf |website=广东老龄网 |accessdate=2018-11-25 |location=北京 |language=zh-cn}}</ref>。

退役军人事务部主要负责拟订退役军人思想政治、管理保障等工作政策法规并组织实施，褒扬彰显退役军人为[[中国共产党|党]]、[[中华人民共和国|国家]]和人民牺牲奉献的精神风范和价值导向，负责军队转业干部、复员干部、退休干部、退役士兵的移交安置工作和自主择业退役军人服务管理、待遇保障工作，组织开展退役军人教育培训、优待抚恤等，指导全国拥军优属工作，负责[[烈士_(中华人民共和国)|烈士]]及退役军人荣誉奖励、军人公墓维护以及纪念活动等工作<ref>{{cite web|url=http://www.xinhuanet.com/politics/2018lh/2018-03/13/c_137035436.htm|title=王勇：组建退役军人事务部|date=2018-03-13|publisher=新华网}}</ref>。

== 沿革 ==
2018年3月17日[[第十三届全国人民代表大会#第一次会议|第十三届全国人民代表大会第一次会议]]通过《第十三届全国人民代表大会第一次会议关于国务院机构改革方案的决定》，批准《国务院机构改革方案》。方案规定：“组建退役军人事务部。将[[中华人民共和国民政部|民政部]]的退役军人优抚安置职责，[[中华人民共和国人力资源和社会保障部|人力资源和社会保障部]]的军官转业安置职责，以及[[中央军委政治工作部|中央军委政治工作部]]、[[中央军委后勤保障部|后勤保障部]]有关职责整合，组建退役军人事务部，作为国务院组成部门。”<ref>{{Cite web|url=http://www.npc.gov.cn/npc/dbdhhy/13_1/2018-03/14/content_2048552.htm|title=关于国务院机构改革方案的说明——2018年3月13日在第十三届全国人民代表大会第一次会议上|author=王勇|date=2018-03-13|publisher=中国人大网}}</ref><ref>{{Cite web|url=http://www.npc.gov.cn/npc/xinwen/2018-03/17/content_2049996.htm|title=第十三届全国人民代表大会第一次会议关于国务院机构改革方案的决定|date=2018-03-17|publisher=中国人大网}}</ref><ref>{{Cite web|url=http://www.npc.gov.cn/npc/xinwen/2018-03/18/content_2050371.htm|title=国务院机构改革方案|date=2018-03-17|publisher=中国人大网}}</ref>

2018年4月16日，退役军人事务部正式在北京北[[北京五环路|五环]]外的办公楼挂牌成立<ref>{{Cite web|url=http://www.xinhuanet.com/politics/2018-04/16/c_1122688886.htm|title=中华人民共和国退役军人事务部正式在北京挂牌|date=2018-04-16|publisher=新华网}}</ref>。

2018年7月31日，中华人民共和国退役军人事务部门户网站于7月31日上线运行。<ref>{{Cite web|url=http://www.mod.gov.cn/topnews/2018-07/31/content_4821300.htm|title=中华人民共和国退役军人事务部门户网站上线运行公告 - 中华人民共和国国防部|accessdate=2018-07-31|last=杨一楠|work=www.mod.gov.cn}}</ref>
== 职责 ==
退役军人事务部的主要职责为：拟订退役军人思想政治、管理保障等工作政策法规并组织实施，褒扬彰显退役军人为党、国家和人民牺牲奉献的精神风范和价值导向，负责军队转业干部、复员干部、退休干部、退役士兵的移交安置工作和自主择业退役军人服务管理、待遇保障工作，组织开展退役军人教育培训、优待抚恤等，指导全国拥军优属工作，负责烈士及退役军人荣誉奖励、军人公墓维护以及纪念活动等<ref>{{Cite web|url=http://www.ce.cn/xwzx/gnsz/gdxw/201805/03/t20180503_29016635.shtml|title=退役军人事务部公布办公地址等信息 已于4月16日挂牌|date=2018-05-03|publisher=中国新闻网}}</ref>。
