'''中国人民解放军三一一基地'''，位于[[福建省|福建省]][[福州市|福州市]]，隶属[[中国人民解放军战略支援部队|中国人民解放军战略支援部队]]。

==沿革==
2003年12月5日，中共中央、中央军委批准发布《中国人民解放军政治工作条例》，“舆论战、心理战、法律战”被明确写进该条例，此后中国军队普遍开展了“三战”研究和训练<ref>{{Cite web|url =http://news.xinhuanet.com/mil/2004-06/21/content_1538252.htm|title =中国军队开展舆论战、心理战、法律战研究和训练|author =|date =2004-06-21|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20150930012002/http://news.xinhuanet.com/mil/2004-06/21/content_1538252.htm|archivedate =2015-09-30}}</ref>。

2005年左右，'''中国人民解放军总政治部三一一基地'''在福建成立，隶属[[中国人民解放军总政治部|中国人民解放军总政治部]]，开展“舆论战、心理战、法律战”。2011年，该基地被指定为所有对[[台湾|台]]心理战工作的焦点，包括协助转播[[海峡之声|海峡之声]]广播电台的节目<ref>{{Cite web |url = http://mil.cankaoxiaoxi.com/2013/0817/257086.shtml |title = 日媒称“高新七号”大幅提升解放军心理战能力 |author =  |date =2013-08-17  |publisher = 参考消息 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，其上级单位[[中国人民解放军总政治部|中国人民解放军总政治部]]撤销，该基地转隶[[中国人民解放军战略支援部队|中国人民解放军战略支援部队]]<ref name=huodong>{{Cite web |url = http://news.tsinghua.edu.cn/publish/thunews/10303/2016/20161220204556768409076/20161220204556768409076_.html |title =  陈旭参加新法3班主题团日活动|author =  |date = 2016-12-20 |publisher = 清华大学新闻网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1584288  |title = 梅华波少将调任战略支援部队某基地政委，原任空降兵学院政委 |author =  |date =2016-12-21  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>，称'''中国人民解放军三一一基地'''<ref>彭靖棠、陈志莉、郑宏飞、马兴龙、于涛、姚洋，新型太阳能融雪净水系统研究，水处理技术2017年01期</ref>。
