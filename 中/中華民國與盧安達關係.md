'''中華民國與盧安達關係'''是指[[中華民國|中華民國]]與[[盧安達共和國|盧安達共和國]]之間的關係。兩國於1962－1972年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對盧安達的相關事務由'''{{link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''、'''[[駐法國台北代表處|駐法國台北代表處]]'''、'''[[駐歐盟兼駐比利時代表處|駐歐盟兼駐比利時代表處]]'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-165-5a18a-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[盧安達|盧安達]]。]]

==政治==
===外交===
{{see also|中華民國外交|{{link-en|盧安達外交|Foreign relations of Rwanda}}}}
1962年7月1日，建立大使級[[外交#外交上的承認|外交關係]]。於首都[[吉佳利|吉佳利]]設立'''中華民國駐盧安達共和國[[外交代表機構#大使馆|大使館]]'''，但只派駐[[外交官#其他外交官员|一等秘書]]暫代館務。

1965年9月15日，派駐[[特命全權大使|大使]]。

1972年5月13日，與中華民國斷交。

1994年7月，[[中華民國政府|中華民國政府]]捐款200萬[[美元|美元]]援助[[卢旺达大屠杀|盧安達難民]]，並於8月送交聯合國兒童基金會美國委員會（The United States Committee for the United Nations Children’s fund），再由該會上交給[[聯合國兒童基金會|聯合國兒童基金會]]（UNICFF）全權處理。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=849C383A98F01EDB |author = 公眾外交協調會 |date = 1994年8月17日 |title = 中華民國基於人溺己溺之人道精神並響應聯合國蓋里秘書長要求全世界各國向盧安達難民施予援手之呼籲 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#电子签证|電子簽證]]／[[簽證#落地签证|落地簽證]]的方式入境盧安達。]]
[[File:Visa_requirements_for_Rwandan_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|盧安達簽證政策|Visa policy of Rwanda}}|{{link-en|盧安達公民簽證要求|Visa requirements for Rwandan citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可至盧安達移民總署網頁申辦[[簽證#电子签证|電子簽證]]，或於抵達首都的[[基加利国际机场|吉加利國際機場]]時申辦[[簽證#落地签证|落地簽證]]，停留最多30天。持[[肯亞|肯亞]]或[[烏干達|烏干達]]簽發效期90天多次入境的「東非旅遊簽證」，亦可入境盧安達。入境須出示[[黃熱病|黃熱病]][[疫苗接種|疫苗接種]]證明（黃皮書）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-165-5a18a-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/ElectronicVisa.aspx?n=6F246645F10311D7&AreaSN=D33B55D537402BAA |title = 電子簽證資訊 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|盧安達護照|Rwandan passport}}的{{link-en|盧安達人口|Demographics of Rwanda|盧安達公民}}抵台參加國際會議、運動賽事、商展活動，也可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將盧安達歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2016年12月1日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-394-5e4da-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2017年，盧安達是中華民國第196大貿易夥伴、第173大[[進口|進口]]夥伴、第198大[[出口贸易|出口]]夥伴。出口至盧安達的金額為25萬8,115[[美元|美元]]，年減49.143%；自盧安達進口的金額為30萬7,459美元，年減5.894%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）4萬9,344美元，年減127.289%。※2016年，盧安達是中華民國第191大貿易夥伴、第169大進口夥伴、第191大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
