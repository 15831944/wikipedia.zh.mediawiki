'''中国人民解放军白求恩国际和平医院'''，简称'''白求恩国际和平医院'''、'''和平医院'''，位于[[河北省|河北省]][[石家庄市|石家庄市]][[桥西区_(石家庄市)|桥西区]]中山西路398号，隶属[[郑州联勤保障中心|郑州联勤保障中心]]，为综合性[[三级甲等医院|三级甲等医院]]。

==简介==
医院的前身是晋察冀军区白求恩国际和平医院。1937年9月底，12位医务人员在[[山西省|山西省]][[五台县|五台县]]耿镇组成'''耿镇休养所'''。1937年11月7日，[[晋察冀军区|晋察冀军区]]成立'''晋察冀军区后方医院'''。1938年6月，[[加拿大|加拿大]]胸外科专家[[白求恩|白求恩]]来到这家医院，先后在医院的两个休养所的基础上成立了模范医院和特种外科医院<ref name=jj/>。其中，1938年9月15日在山西五台县松岩口村举行“模范医院”成立大会，时值在[[伦敦|伦敦]]召开的国际和平会议闭幕，故“模范医院”又称“国际和平医院”<ref>{{Cite web |url = http://mil.news.sina.com.cn/2006-08-29/2012393772.html  |title = 白求恩与白求恩国际和平医院 |author =  |date =  2006-08-29|publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。1939年11月12日，白求恩殉职。为了纪念白求恩，1940年1月5日晋察冀军区将该医院更名为'''白求恩国际和平医院'''<ref name=yingxiong>{{Cite web |url = http://www.81.cn/lj/2015-08/03/content_6611400.htm |title = 英雄部队：北京军区白求恩国际和平医院 |author =  |date = 2015-08-03 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref><ref name=shi>{{Cite web |url =  http://www.bqejsyjh.com/newslook.asp?NewID=4210&FID=90&page=0|title =十件往事  |author =胡文彬  |date =  |publisher = 白求恩精神研究会 |language =  |accessdate =2017-06-16  |quote =  }}</ref>。

[[抗日战争|抗日战争]]期间，晋察冀军区白求恩国际和平医院随部队转战[[太行山|太行山]]区，曾先后驻[[河北省|河北省]][[阜平县|阜平县]]大台、温塘、城南庄等地。医院经多次改建和扩编，与[[晋察冀军区|晋察冀军区]]预备医院组成多个野战分院，成为晋察冀军区及其后[[华北军区|华北军区]]的医疗技术中心。[[日本投降|日本投降]]后，1945年9月进驻[[张家口市|张家口市]]，接收了伪蒙疆医院<ref name=shi/>。1946年6月，医院被上级命名为'''[[白求恩医科大学|白求恩医科大学]]附属医院'''<ref name=dong/>。[[第二次国共内战|第二次国共内战]]爆发后，1946年9月[[中国国民党|中国国民党]]军队进攻[[中共|中共]][[解放区|解放区]]，白求恩医科大学及其附属医院奉命撤出张家口，1946年11月进驻河北省[[唐县|唐县]]，白求恩医科大学驻唐县葛公村，医院驻唐县张合庄，相距3里<ref name=shi/>。

1947年11月[[中国人民解放军|中国人民解放军]]取得[[石家庄战役|石家庄战役]]胜利。1948年2月底，晋察冀军区白求恩医科大学附属医院接到进驻石家庄的命令，1948年3月启程迁往石家庄。1948年7月1日，晋察冀军区白求恩医科大学与晋冀鲁豫军区北方大学医学院合并，定名为“华北医科大学”。1948年10月，白求恩医科大学附属医院与北方大学医学院附属医院合并，定名为“'''中国人民解放军白求恩国际和平医院'''”。1948年10月28日，在医院小礼堂举行开院典礼。次日即1948年10月29日，因国民党[[傅作义|傅作义]]部偷袭石家庄，医院奉命撤往[[衡水地区|衡水地区]]<ref name=shi/>。一个月后迁回石家庄<ref>{{Cite web |url = http://www.peacehospital.com/articlehtm/2009-8-4/20098493823.html |title = 二十五、向衡水转移 |author =  |date =  2009-08-04|publisher =  白求恩国际和平医院|language =  |accessdate =  |quote =  }}</ref>。

1974年12月，中央军委决定中国人民解放军白求恩国际和平医院正式对外开放<ref name=jj/>。1975年在[[周恩来|周恩来]]关怀下，医院建成了“白求恩纪念馆”和“柯隶华纪念馆”。1979年[[邓小平|邓小平]]为医院题词“做白求恩式的革命者，做白求恩式的科学家”<ref name=wrkong/>。1980年，[[江一真|江一真]]（1937年创办晋察冀军区卫生学校的首任校长）出任[[河北省人大常委会|河北省人大常委会]]主任，他以个人名义向[[聂荣臻|聂荣臻]]写了情况报告，希望聂荣臻关怀这家医院，并建议解决医院设备更新、基本建设、体制改革问题。聂荣臻随即在该报告上作出批示。不久，[[中国人民解放军总后勤部|中国人民解放军总后勤部]]一次性批给白求恩国际和平医院650万元人民币购置仪器设备，此后批准建设门诊大楼、干部病房楼等，医院建制由团级单位升格为副师级单位，直属北京军区后勤部领导<ref name=dong/>。1995年，医院被[[中国人民解放军总后勤部卫生部|中国人民解放军总后勤部卫生部]]评定为[[三级甲等医院|三级甲等医院]]<ref name=wrkong>吴渭虹、任玉光、孔令占，在白求恩塑像下——白求恩国际和平医院护理工作简介，解放军护理杂志1997年03期</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军白求恩国际和平医院由[[中国人民解放军北京军区|中国人民解放军北京军区]]转隶[[郑州联勤保障中心|郑州联勤保障中心]]<ref>王玉平、何子安、高斌，紧盯使命任务 聚焦保障打赢——郑州联勤保障中心所属医院扎实做好卫勤保障工作掠影，解放军健康2017年第2期</ref>。
