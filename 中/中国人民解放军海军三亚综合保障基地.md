'''中国人民解放军海军三亚综合保障基地'''，位于[[海南省|海南省]][[三亚市|三亚市]]，隶属[[中国人民解放军南部战区海军|中国人民解放军南部战区海军]]，是中国继[[中国人民解放军海军青岛综合保障基地|海军青岛综合保障基地]]之后第二个[[航空母舰|航空母舰]]综合保障基地。

==沿革==
2012年，该基地开工建设<ref name=sina/>。2013年11月29日，航母[[辽宁舰|辽宁舰]]靠泊该军港，在该军港短暂停靠<ref name=sina/><ref>{{Cite web |url = http://news.qq.com/a/20131130/000913.htm |title = 专家：启用三亚基地意味着中国将有不止1艘航母 |author =  |date = 2013-11-30 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.xinhuanet.com/photo/2013-12/31/c_125936184.htm |title = 辽宁舰2013年多方面突破 |author =  |date = 2013-12-31 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。2014年11月，该基地基本建成。该基地拥有可以双向系泊大型船只的船坞（码头），这显示中国海军的两座航母综合保障基地均有能力容纳两艘航母。该基地的船坞（码头）长700米，建成后成为世界上最长的航母码头。（相比之下，[[美国|美国]]在[[弗吉尼亚州|弗吉尼亚州]][[诺福克海军基地|诺福克海军基地]]的航母码头以及在[[日本|日本]]的航母码头长度都在400米至430米之间。）航母的主要陆上保障设施包括航母驻泊港口、舰载机驻屯机场、训练保障设施等<ref name=sina>{{Cite web |url =  http://mil.news.sina.com.cn/2015-08-02/0907836228.html|title =中国在三亚建全球最大航母码头 与潜艇基地相连  |author =  |date = 2015-08-02 |publisher =  新浪|language =  |accessdate =  |quote =  }}</ref>。

该基地自组建以来至2015年，先后完成首次保障航母靠泊、援潜装备技术保障、远航后装保障、海上集训舰艇伴随保障等任务<ref>{{Cite web |url = http://www.81.cn/jfjbmap/content/2015-08/09/content_119564.htm |title = 扬帆起航驶向深蓝——南海舰队某综合保障基地破解难题谋发展纪实 |author =  |date =  |publisher = 解放军报2015年8月9日第07版专题新闻 |language =  |accessdate =  |quote =  }}</ref>。
