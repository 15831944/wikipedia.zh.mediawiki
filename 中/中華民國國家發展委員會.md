'''國家發展委員會'''（簡稱'''國發會'''）是[[中華民國|中華民國]]有關國家發展的[[中華民國政府|最高主管機關]]，隸屬於[[行政院|行政院]]，前身為「行政院經濟建設委員會」與「行政院研究發展考核委員會」。主要負責[[規劃|規劃]][[國家|國家]][[發展|發展]][[策略|策略]][[計畫|計畫]]、促進經濟與社會整體發展、以及政府治理等工作，因其範圍涉及跨部會工作而有「小行政院」的別稱。<ref>[http://n.yam.com/cnyes/fn/20140122/20140122068827.html 國發會等同「小行政院」江揆盼1+1大於2 交出亮麗成績單]，蕃薯藤新聞網，2014年01月22日</ref>

== 沿革 ==
; <big>行政院經濟建設委員會歷史</big>
* 1948年7月，[[中華民國政府|中華民國政府]]與[[美國政府|美國政府]]在[[南京市_(中華民國)|南京市]]簽定《[[美援|中美經濟援助協定]]》，中華民國政府設立'''行政院美援運用委員會'''（簡稱美援會）。
* 1963年9月，美援會改組為'''行政院國際經濟合作發展委員會'''（簡稱經合會）。
* 1973年8月，行政院為加強經濟設計、研究工作，將經合會改組為'''行政院經濟設計委員會'''（簡稱經設會）。
* 1977年12月，行政院為加強國家經濟建設之有效推動、促進國家經濟之整體發展，將經設會與行政院財經小組合併改組為'''[[行政院經濟建設委員會|行政院經濟建設委員會]]'''（簡稱經建會）。

; <big>行政院研究發展考核委員會歷史</big>
* 1966年，行政院成立'''行政改革研究會'''。<ref>[http://theme.archives.gov.tw/govreform/guide_01-7.html 行政院組織改造檔案展 解嚴之前的行政院>> 行政改革研究會－行政機關檢討改進]，行政院研究發展考核委員會</ref>
* 1969年3月，正式成立'''[[行政院研究發展考核委員會|行政院研究發展考核委員會]]'''。
* 1987年1月，《行政院研究發展考核委員會組織條例》完成立法程序，正式成為行政院的常設機關。
* 2012年5月20日，[[中華民國文化部|文化部]]成立後，研考會政府出版品管理處併入文化部。

; <big>國家發展委員會成立後</big>
* 2013年8月6日，[[立法院|立法院]]第8屆第3會期第2次臨時會[[三讀通過|三讀通過]]《國家發展委員會組織法》與《國家發展委員會檔案管理局組織法》，並授權行政院訂立施行日期。<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20130806/237808/ 組織再造　經建會轉成國家發展委員會 - 台灣《蘋果日報》 2013年8月06日]</ref>
* 2013年8月21日，《國家發展委員會組織法》與《國家發展委員會檔案管理局組織法》制定公布，並由行政院訂立2014年1月22日起施行。<ref>[http://www.president.gov.tw/Default.aspx?tabid=84&lctl=view&itemid=10016&ctid=96 制定國家發展委員會組織法 - 總統令 中華民國102年8月21日 華總一義字第10200156121號]</ref><ref>[http://www.president.gov.tw/Default.aspx?tabid=84&lctl=view&itemid=10017&ctid=96 制定國家發展委員會檔案管理局組織法 - 總統令 中華民國102年8月21日 華總一義字第10200156131號]</ref>
* 2014年1月22日，[[行政院經濟建設委員會|行政院經濟建設委員會]]、[[行政院研究發展考核委員會|行政院研究發展考核委員會]]、[[行政院主計總處|行政院主計總處]]電子處理資料中心、以及[[行政院公共工程委員會|行政院公共工程委員會]]的工程管制考核業務正式整合為「'''國家發展委員會'''」<ref>[https://www.ndc.gov.tw/m1.aspx?sNo=0055075 國發會揭牌成立 邁向國家發展新紀元 2014-1-22] {{webarchive|url=https://web.archive.org/web/20140127114035/http://www.ndc.gov.tw/m1.aspx?sNo=0055075 |date=2014-01-27 }}</ref>，會本部設於經建會舊址。未來計畫在臺北市中山區[[南京東路_(台北市)|南京東路]]及伊通街口新建「國發大樓」做為新總部大樓，總工程費用約新台幣8億多元，工期約2至3年。
* 2018年7月1日，承接[[臺灣省政府|臺灣省政府]]和[[福建省政府|福建省政府]]的員額與業務。<ref>[https://money.udn.com/money/story/7307/3224504 省級機關明年起預算歸零]</ref>
