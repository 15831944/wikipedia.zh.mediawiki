'''中国人民解放军第三医院'''，北院位于[[陕西省|陕西省]][[宝鸡市|宝鸡市]][[金台区|金台区]]东风路45号，隶属[[郑州联勤保障中心|郑州联勤保障中心]]，为[[三级甲等医院|三级甲等医院]]。

==简介==
1951年7月，创建'''西北军区第五陆军医院'''。1955年改称'''中国人民解放军第三医院'''<ref name=sqyw>{{Cite web |url = http://www.sanqinyou.com/lishi/info/1373111220531705_2.html |title = 宝鸡兵防空防历史（2） |author =  |date = 2013-07-31 |publisher = 三秦游网 |language =  |accessdate =  |quote =  }}</ref><ref name=xhw>{{Cite web|url =http://news.xinhuanet.com/mil/2008-03/04/content_7713591.htm|title =解放军第三医院|author =|date =2008-03-04|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20130807112004/http://news.xinhuanet.com/mil/2008-03/04/content_7713591.htm|archivedate =2013-08-07}}</ref>。医院隶属[[中国人民解放军兰州军区|中国人民解放军兰州军区]]<ref name=xhw/>。

1996年，医院被[[中国人民解放军总后勤部|中国人民解放军总后勤部]]评为[[三级甲等医院|三级甲等医院]]。1997年被[[中华人民共和国卫生部|中华人民共和国卫生部]]评为“爱婴医院”，获军委总部“白求恩杯”优质服务竞赛三连冠。1997年被军委总部评为“为部队服务先进单位”。截至2008年，第三医院承担[[中国人民解放军第四军医大学|中国人民解放军第四军医大学]]、[[西安交通大学|西安交通大学]]医学院、[[延安医学院|延安医学院]]、[[西安医学院|西安医学院]]等院校的教学任务，第三医院神经外科研究所被西安交通大学医学院定为硕士研究生培训点，第三医院神经内科、精神病科被中国人民解放军第四军医大学定为临床科研教学基地<ref name=xhw/>。

1970年，由'''中国人民解放军第一八四野战医院'''组建'''中国人民解放军第三十一医院'''，1978年从三合村的宝鸡卫生学校迁到石坝河，同年7月由野战医院改成胸科专科医院。主要接收部队胸肺疾病及传染疾病患者<ref name=sqyw/>。第三十一医院是兰州军区唯一以收治结核病、传染病为主，并设有内外科的带专科性质的“二级甲等医院”。1990年代，第三十一医院撤销编制，改为第三医院石坝河医疗所<ref>张克荣、周荣、张振玉、白小京，医疗所重点专科特色建设初探，解放军医院管理杂志1999年05期</ref>。后来发展为第三医院南院，为宝鸡市级结核病定点医院<ref>{{Cite web|url = http://www.bjws.gov.cn/Info01-bde242a8-99fd-42b6-b41f-f7f690a857b2.html|title = 宝鸡市结核病定点诊疗医院名单|author = |date = 2016-12-28|publisher = 宝鸡市卫生和计划生育局|language = |accessdate = |quote = }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第三医院由[[中国人民解放军兰州军区|中国人民解放军兰州军区]]联勤部转隶[[郑州联勤保障中心|郑州联勤保障中心]]<ref>{{Cite web |url =http://army.81.cn/content/2017-02/22/content_7497254.htm|title =郑州联保第三医院心理服务队心理健康辅导惠及万余官兵 |author =  |date =  2017-02-22 |publisher = 中国陆军网|language =  |accessdate =  |quote =  }}</ref>。
