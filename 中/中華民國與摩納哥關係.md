'''中華民國與摩納哥關係'''是指[[中華民國|中華民國]]與[[摩納哥公國|摩納哥公國]]（又稱'''摩納哥[[親王國|親王國]]'''）之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對摩納哥的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-130-c2ed3-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
{{see also|中華民國外交|{{link-en|摩納哥外交|Foreign relations of Monaco}}}}
1982年4月18日，[[摩納哥親王|摩納哥親王]][[蘭尼埃三世|蘭尼埃三世]]與王妃[[格蕾丝·凯利|葛麗絲·凱莉]]抵台訪問。

2001年，摩納哥委任具有[[中華民國國籍|中華民國國籍]]的[[徐楓|徐楓]]為該國駐中華人民共和國[[上海|上海]][[名譽領事|名譽領事]]。<ref>{{Cite web |url = http://www.fmprc.gov.cn/web/gjhdq_676201/gj_676203/oz_678770/1206_679522/1206x1_679536/ggzwwjjgxx_679540/t576541.shtml |date = 2017年2月 |title = 该国驻我外交机构信息 |publisher = [[中华人民共和国外交部|中华人民共和国外交部]]}}</ref>

2009年5月18－27日，行政院衛生署（今[[衛生福利部|衛生福利部]]，衛福部）署長[[葉金川|葉金川]]首度以「[[中華台北|中華台北]]觀察員」身分出席[[世界衛生組織|世界衛生組織]]（WHO）的[[世界衛生大會|世界衛生大會]]（WHA）。[[摩納哥|摩納哥]]、[[美國|美國]]、[[歐盟|歐盟]]、[[日本|日本]]、[[布吉納法索|布吉納法索]]、[[聖多美普林西比|聖多美普林西比]]等國於大會中發言歡迎中華台北與會。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=1D726227D1804F76 |author = 公眾外交協調會 |date = 2011年9月16日 |title = 政府推動有意義參與聯合國體系的具體進展 |publisher = 中華民國外交部}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|歐洲聯盟簽證政策|{{link-en|摩納哥公民簽證要求|Visa requirements for Monégasque citizens}}}}
摩納哥沒有加入[[歐洲聯盟|歐洲聯盟]]與《[[申根公約|申根公約]]》，但根據1999年生效的《[[阿姆斯特丹條約|阿姆斯特丹條約]]》使得《申根既有規範》（Schengen acquis）正式成為[[歐盟法律|歐盟法律]]的一部分。再者，摩納哥與[[歐洲聯盟成員國|歐盟國家]]關係密切並開放[[邊界|邊界]]，在[[申根區|申根區]]實際執行過程中將摩納哥視為[[法國|法國]]的一部分。歐洲申根區給予持有載明[[中華民國國民身份證|身份證]]字號的[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以[[歐洲聯盟簽證政策|申根區免簽證]]入境，停留日數與申根區合併計算，每6個月期間內總計可停留90天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-130-c2ed3-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_FAQ.aspx?c=AD6908DFDDB62656&n=C1B19EF0ACDD3C91&sms=867F5398A49F758A |title = 歐盟免申根簽證常見問答集 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

{{link-en|摩納哥人口|Demographics of Monaco|摩納哥公民}}持有{{link-en|摩納哥護照|Monégasque passport}}者也可以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將摩納哥歸類為'''灰色警示'''：提醒注意（2017年4月22日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-368-7eca0-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|摩納哥經濟|Economy of Monaco}}}}
===貿易===
2017年，摩納哥是中華民國第198大貿易夥伴、第171大[[進口|進口]]夥伴、第203大[[出口贸易|出口]]夥伴，在[[歐洲國家和地區列表|歐洲國家]]排名倒數第3、4、3；在[[世界政區索引|世界經濟體]]倒數第41、66、28。出口至摩納哥的金額為16萬0,208[[美元|美元]]，年減42.197%；自摩納哥進口的金額為35萬9,380美元，年增21.549%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）19萬9,172美元，年增976.373%。※2016年，摩納哥是中華民國第201大貿易夥伴、第172大進口夥伴、第199大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
