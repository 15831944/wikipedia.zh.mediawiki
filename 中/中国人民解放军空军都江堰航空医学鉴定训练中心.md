'''中国人民解放军空军都江堰航空医学鉴定训练中心'''，位于[[四川省|四川省]][[都江堰市|都江堰市]][[翠月湖镇|翠月湖镇]]聚青路18号，隶属[[中国人民解放军西部战区空军|中国人民解放军西部战区空军]]后勤部。

==简介==
1952年始建，前身是[[西南军区|西南军区]]空军第一七二医院。后来先后调整改为陆军第五十二医院、成都军区干部训练队、第十四疗养队、空军灌县疗养院、空军都江堰疗养院。2004年5月26日，转隶成都军区联勤部，同年10月整编为'''中国人民解放军成都军区峨眉疗养院都江堰疗养区'''。2009年7月，都江堰、杭州、大连、青岛、临潼五所疗养院统一由各大军区联勤部转隶各大军区空军后勤部，成立5个航空医学鉴定训练中心，构建为“[[中国人民解放军空军总医院|空军总医院]]、[[中国人民解放军空军航空医学研究所|空军航空医学研究所]]——区域性航空医学鉴定训练中心——航空兵部队卫生机构”三级航空医学鉴定训练体系。其中，2009年7月20日，成都军区峨眉疗养院都江堰疗养区转隶[[成都军区空军|成都军区空军]]后勤部，整编为'''中国人民解放军空军都江堰航空医学鉴定训练中心'''，同年9月15日该中心正式组建。该中心担负飞行人员健康疗养、航空生理心理训练、航空医学鉴定保障任务，并且承担航空医学基础研究及师军职干部疗养工作<ref>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536782.htm |title =  空军都江堰航空医学鉴定训练中心|author =  |date = 2014-09-12 |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref><ref>何强、彭天国、董文，空军航空医学鉴定训练中心的发展与思考 ，全国航空航天医学学术会议暨云南省航空航天医学学术会议2010,27(3):180-181</ref><ref>{{Cite web |url = http://news.xinhuanet.com/mil/2011-12/25/c_111295752.htm |title = 一场从陆地打响的空战 |author =  |date = 2011-12-25 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url = http://www.zhb.gov.cn/gkml/hbb/bgth/201705/t20170511_413924.htm|title = 关于举办2017年全国“12369”环保举报岗位培训班的通知|author = |date = 2017-05-08|publisher = 中华人民共和国环境保护部|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170808194051/http://www.zhb.gov.cn/gkml/hbb/bgth/201705/t20170511_413924.htm|archive-date = 2017-08-08|dead-url = yes}}</ref>。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，转隶[[中国人民解放军西部战区空军|中国人民解放军西部战区空军]]后勤部<ref>{{Cite web |url = http://kj.81.cn/content/2016-09/30/content_7284495.htm |title = 严把新兵复检关 确保为部队输送合格兵源 |author =  |date =2016-09-30  |publisher =中国空军网  |language =  |accessdate =  |quote =  }}</ref>。
