'''中国人民解放军历史'''即[[中国人民解放军|中国人民解放军]]的军史，始于1927年8月1日的[[南昌起义|南昌起义]]。历史上，这支军队在创建之初曾称为“'''[[中国工农革命军|中国工农革命军]]'''”，1928年至1936年间称为“'''[[中国工农红军|中国工农红军]]'''”；1937至1945年间其主力编入[[国民革命军|国民革命军]]序列，称为“'''[[八路军|八路军]]'''”及“'''[[新四军|新四军]]'''”；[[解放战争|解放战争]]期间，各部陆续改称“'''中国人民解放军'''”，沿用至今。[[朝鲜战争|朝鲜战争]]期间，解放军部分部队曾以'''[[中国人民志愿军|中国人民志愿军]]'''名义入朝作战。中国人民解放军作为[[中国共产党|中国共产党]]所掌握的武装力量的主体，其军史贯穿了[[第一次国共内战|第一次国共内战]]、[[中国抗日战争|抗日战争]]、[[第二次国共内战|第二次国共内战]]、[[中华人民共和国历史|中华人民共和国历史]]。

== 建军历程及战史 ==
中国人民解放军以1927年8月1日作为建军日，不过在之前的[[国共合作#第一次国共合作_(1923-1927)|国共第一次合作]]的[[国民革命军北伐|北伐战争]]中，已有完全由中国共产党掌握的部队存在，如[[國民革命軍第四軍|“铁军”国民革命军第四军]]的[[叶挺独立团|叶挺独立团]]等。

1927年4月12日[[蒋介石|蒋介石]]领导的国民党的部分势力於国共合作的國民革命軍北伐期间在[[上海|上海]]开始[[四一二事件|清党]]（或称“[[四一二政变|四一二政变]]），捕杀中共成员和[[国民党左派|国民党左派]]，致使[[国共合作#國共分裂|第一次国共合作破裂]]<ref>{{cite web | url=http://history.sina.com.cn/his/zl/2013-02-25/155347543.shtml | title=國民黨的清黨運動 | author=王奇生 | date=2013年2月25日 | quote=}}</ref>；是年秋至1928年春，中国共产党为了反抗和推翻他们认为是反动和非法的[[國民政府#南京（1927年4月18日～1937年11月17日）|南京国民政府]]而先后发动了南昌起义、[[秋收起义|秋收起义]]、[[广州起义|广州起义]]、[[湘南起义|湘南起义]]和湖北东部等地区的起义。这些地区起义後保留下来的部队，命名为'''中国工农革命军'''。

1928年4月，南昌起义后保留下来的一部分军队和湘南起义的农民军，由[[朱德|朱德]]、陈毅等率领到达[[井冈山革命根据地|井冈山革命根据地]]，同[[毛泽东|毛泽东]]领导的秋收起义队伍[[井冈山会师|实现会师]]，组成了“中国工农革命军第四军”，简称[[红四军|红四军]]，含有发扬原国民革命军第四军英勇作战的光荣传统之意，后改称为“中国工农红军第四军”<ref name="称谓演变">{{cite web |author1=潘泽庆 |title=中国人民解放军称谓的演变 |url=http://cpc.people.com.cn/GB/64162/64172/85037/85039/6170009.html |website=[[中国共产党新闻网|中国共产党新闻网]] |publisher=[[人民日报社|人民日报社]] |accessdate=2019-01-15 |location=北京 |date=2007年8月}}</ref><ref name="红四军">
{{cite web |author1=王积建 |title= “朱毛红军”缘何始称“红四军”|url= http://dangshi.people.com.cn/GB/8658091.html|website=[[中国共产党新闻网|中国共产党新闻网]] |publisher=[[人民日报社|人民日报社]] |accessdate=2019-01-15 |location=北京 |date=}}</ref>。 5月後各地起义部队陆续改称'''中国工农红军'''，简称“'''红军'''”。

第一次国共内战时期，红军曾发展到30万人，建立了以江西[[中央苏区|中央苏区]]为首、总人口一千多万的多个[[中国共产党革命根据地|革命根据地]]，并击破了[[国军|国军]]四次大规模[[中央苏区反围剿战爭|围剿]]。后来红军未能打破国军[[中央苏区第五次反围剿战争|第五次围剿]]，[[中央红军|中央红军]]被迫于1934年10月退出根据地进行[[长征|长征]]。1935年1月[[遵义会议|遵义会议]]后，红军的指挥权回到毛泽东手中。1936年10月，红军三大主力在甘肃会师，长征结束。此时红军总共有数万人，根据地则仅存[[陕甘宁边区|陕甘宁根据地]]一处。

1936年12月[[西安事变|西安事变]]之后，国共两党停止内战，开始[[第二次国共合作|第二次国共合作]]。1937年7月7日抗日战争全面爆发之后，8月25日红军的主力部队暂时改编为[[国民革命军|国民革命军]]第八路军，简称“'''八路军'''”，9月11日改称'''国民革命军第十八集团军'''。活动在江西、福建、广东、湖南、湖北、河南、浙江、安徽八[[省_(中華民國)#國民政府時期|省]]十四个地区的红军游击队集中起来，10月2日改编为“国民革命军陆军新编第四军”，简称“'''新四军'''”，该番号系由[[叶挺|叶挺]]提议，意在表示继承北伐战争“铁军”国民革命军第四军的优良传统和国共两党的再次合作<ref name="新四军">{{cite web |author1=李友唐 |title=新四军军部的十个旧址 |url=http://cpc.people.com.cn/GB/85037/85038/7733647.html |website=[[中国共产党新闻网|中国共产党新闻网]] |publisher=[[人民日报社|人民日报社]] |accessdate=2019-01-15 |location=北京}}</ref>。抗战中，八路军和新四军深入敌后开辟多个[[抗日根据地|抗日根据地]]，逐渐扩充了自身实力，至抗战结束时，拥有部队120万，民兵200万。

1931年[[九一八事变|九一八事变]]后，中国共产党在[[中国东北|中国东北]]地区建立了抗日游击队，这些部队使用过工农革命军、东北人民革命军等名称，后中共满洲省委将其整编为'''[[东北抗日联军|东北抗日联军]]'''，人数最多时达到3万。1940年东北抗联遭受严重失败后基本停止军事行动，余部退入[[苏联|苏联]]。抗日战争胜利後，1945年中共领导下的关内各解放区部队大批进入东北地区，10月31日组成[[东北人民自治军|东北人民自治军]]，1946年1月14日，东北人民自治军改称'''[[东北民主联军|东北民主联军]]'''。

早在1944年8月20日召开的中共中央军委高干会议上，[[刘少奇|刘少奇]]说：“正规军、游击队合起来叫‘解放军’，或加之为国民革命军解放军。”这是迄今所见档案资料中首次出现“解放军”称谓。1945年，八路军总司令朱德批下七道抗日反攻命令，并在其中第四道命令中首次提出“'''人民解放军'''”一词。1945年8月12日出版的延安《[[解放日报|解放日报]]》（第1543号）第一版全文公布[[朱德|朱德]]总司令发布的6项命令，总标题为《延安总部命令各路解放军向辽吉热察绥等省挺进》。1945年8月29日至10月10日，国共[[重庆谈判|重庆谈判]]中，焦点是中共军队整编和[[解放区|解放区]]政权存在，当时中共中央及中共中央军委未继续对自己领导下的军队提出和使用“解放军”称谓。1946年6月，第二次國共內戰爆發，中国共产党领导下的解放区各部队由八路军、新四军、东北民主联军等陆续改称“人民解放军”，在各地区，“解放军”称谓在使用时，多冠以地名，如“南路人民抗日解放军”、“广东人民抗日解放军”、“山东解放军”等。1946年9月12日，《解放日报》在社论中率先使用“人民解放军”称谓。随后，在[[毛泽东|毛泽东]]及[[新华社|新华社]]发表的文章中，又多次公开出现“人民解放军”称谓。1946年10月3日，《解放日报》在《为实现一月停战协定及政协决议而斗争》的社论中，首次正式提出“中国人民解放军”称谓。1947年10月10日，《[[中国人民解放军宣言|中国人民解放军宣言]]》发表，这是全军性改称“'''中国人民解放军'''”的标志。至1948年初，全军各部队均改称中国人民解放军。1948年11月1日，中共中央军委作出《关于统一全军组织及部队番号的规定》，团以上各部队均冠以“中国人民解放军”称谓<ref>{{cite journal |title = “解放军”称号何时出现何时使用 |journal = 四川统一战线 |year = 2000 |author = 庞小兵 |issue = 8 |accessdate = 2012-08-15}}</ref><ref>{{Cite web|url =http://www.guancha.cn/military-affairs/2017_07_06_416780.shtml|title =1946年，第一次使用“中国人民解放军”称谓|author =|date =2017-07-06|publisher =解放军报|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20170709094507/http://www.guancha.cn/military-affairs/2017_07_06_416780.shtml|archivedate =2017-07-09}}</ref>。

经过数年内战，解放军消灭{{NoteTag|包括毙伤、俘虏、遣散、改编}}了数百万[[中华民国国军|国军]]，迫使[[中华民国政府|中华民国政府]]于1949年迁往[[台湾|台湾]]。1949年10月1日，[[中华人民共和国中央人民政府|中华人民共和国中央人民政府]]成立。1950年春，解放军[[海南島戰役|攻占海南岛]]，守岛国军退往台湾；10月，解放军进军[[西藏|西藏]]；1951年，[[西藏和平解放|西藏和平解放]]。至此，中共完成了对[[中国大陆|中国大陆]]地区的军事控制。

1949年，中国人民解放军成立了[[中国人民解放军海军|海军]]和[[中国人民解放军空军|空军]]。1966年，成立了[[第二炮兵|第二炮兵]]。

[[朝鲜战争|朝鲜战争]]期间，[[中华人民共和国|中华人民共和国]]组建'''[[中国人民志愿军|中国人民志愿军]]'''，于1950年10月19日进入朝鲜，与[[联合国军|联合国军]]作战。1953年7月27日，[[朝鲜停战协定|朝鲜停战]]。1958年，志愿军从朝鲜撤回国内。

中华人民共和国与周边国家的较大规模边境冲突还包括1962年[[中印战争|中印战争]]及1979年[[对越自卫反击战|对越自卫反击战]]等。

== 建国后的军队改革历程 ==
1949年10月1日，中华人民共和国成立时，解放军总员额达550万人。1950年4月，中央决定将全军总员额从550万人减至400万人。1950年5月，全军参谋工作会议在北京召开，确定精简整编的原则是：陆军统编为国防军及公安部队；国防军分成战时和平时两种编制，平时一般是“[[三三制|三三制]]”；野战军及兵团机构除了参加[[解放台湾|攻台]]的部队外均撤销；成立公安部队领导机构。

1950年下半年开始大规模复员，到1951年初共缩减94万余人<ref>{{Cite web|url =http://www.thepaper.cn/newsDetail_forward_1728064|title =新华社刊文回顾我军首次大规模精简整编：计划复员150万人|author =|date =2017-07-08|publisher =澎湃新闻|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20170928235448/http://www.thepaper.cn/newsDetail_forward_1728064|archivedate =2017-09-28}}</ref>。随后由于[[抗美援朝战争|抗美援朝战争]]爆发，军队扩充至627万。1952年全军总人数降为400余万，1955年底精简至320多万，1958年底降为240万。但随后[[中苏交恶|中苏交恶]]，解放军人数再次大幅度上升，至1975年达610万之多。改革開放後以改善國民經濟為要務，1982年军队总员额减至400余万。后经1985年裁军100万、1997年裁军50万、2003年裁军20万、2015年裁军30万，至2017年底，解放军员额降至200万。

改革开放后，[[军队经商|军队曾被允许经商]]，对促进经济起到一定作用，也带来了贪污等问题，后被限停。

1997年9月12日，[[中国共产党第十五次全国代表大会|中国共产党第十五次全国代表大会]]在北京召开，[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军委主席|中央军委主席]][[江泽民|江泽民]]在大会报告中宣布：“在八十年代裁减军队员额一百万的基础上，我国将在今后三年内再裁减军队员额五十万。”<ref>刘标谦，为什么要裁减军队员额五十万?，内部文稿 1997年24期</ref>2000年3月9日，江泽民在[[第九届全国人民代表大会|第九届全国人民代表大会]]第三次会议解放军代表团全体会议上宣布，到1999年底，中共十五大提出的裁减军队员额五十万的任务已完成<ref>{{Cite web |url =http://www.chinanews.com/2000-3-10/26/21367.html  |title = 中国已完成裁军五十万任务|author =  |date =2000-03-10  |publisher =中国新闻网  |language =  |accessdate =  |quote =  }}</ref>。

2003年9月1日，中央军委主席江泽民在出席[[中国人民解放军国防科学技术大学|中国人民解放军国防科学技术大学]]五十周年庆典活动时正式宣布了中共中央、中央军委的决定：“我军将在2005年前再裁减军队员额二十万。”<ref>{{Cite web |url = http://news.sina.com.cn/c/2003-09-01/1323670341s.shtml |title =江泽民出席国防科大50年庆典 宣告我军裁员20万  |author =  |date = 2003-09-01 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>2005年前军队体制编制调整改革的任务是：压缩规模，改革体制，优化结构，调整编组，完善制度，从编成结构上提升军队战斗力<ref>{{Cite web |url=http://www.huaxia.com/xw/zdxw/2005/03/375047.html  |title =中国军队改革 优化结构提升战斗力  |author =  |date = 2005-03-03 |publisher = 华夏经纬网 |language =  |accessdate =  |quote =  }}</ref>。截至2005年12月31日，军队体制编制调整改革方案确定的任务完成，如期裁减员额20万，其中全军精简干部17万，军队总员额下降为230万，陆军部队占全军总员额的比例下降至历史最低点<ref>解放军如期裁员20万——总员额降至230万 作战部队比例明显上升，武警工程大学学报2006(2):59-59</ref>。

2006年，解放军全面启动非现役公勤人员制度，军级以上单位的机关和非作战部队中由[[士官|士官]]和[[义务兵|义务兵]]承担的炊事、驾驶等部分公勤岗位，逐步使用社会聘用人员。<ref>[http://news.xinhuanet.com/politics/2006-02/15/content_4183604.htm]</ref>2007年7月3日，全军换發[[07式军服|07式军服]]工作正式开始<ref>[http://news.xinhuanet.com/mil/2007-07/03/content_6323318.htm]</ref>。2006年冬季征兵开始，心理检测正式成为解放军征兵体检项目。这一检测包括智力检测和人格检测2项内容，应征公民如有任意一项结论不合格，则不能应征入伍。<ref>[http://news.xinhuanet.com/mil/2006-11/01/content_5275769.htm]</ref>

2012年[[中共十八大|中共十八大]]后[[习近平|习近平]]就任中共中央总书记和[[中共中央军委主席|中共中央军委主席]]，开始推动[[深化国防和军队改革|深化国防和军队改革]]，是中共执政以来规模最大的军事改革。

[[中华人民共和国|中华人民共和国]]成立后，截至2014年，中国人民解放军共进行13次体制编制调整改革，逐步建立和完善了军队领导管理和作战指挥体制，完成了从单一陆军向诸军种合成军队的转变<ref>{{Cite web |url =  http://theory.people.com.cn/n/2014/0224/c40531-24444312.html|title = 曲爱国：改革开放以来我军体制编制调整改革 |author =  |date =2014-02-24  |publisher =  中国共产党新闻网|language =  |accessdate =  |quote =  }}</ref>。

2015年9月3日，中共中央总书记兼中央军委主席[[习近平|习近平]]宣布[[裁军|裁军]]30万人，至2017年前军队规模将降至200万人<ref>熊建.[http://politics.people.com.cn/n/2015/0903/c1001-27543904.html 中国军队历次裁军数据一览] {{webarchive|url=https://web.archive.org/web/20160404024823/http://politics.people.com.cn/n/2015/0903/c1001-27543904.html |date=2016-04-04 }}人民网.2015年09月03日</ref>。2015年12月31日，成立陸軍領導機構、戰略支援部隊，第二炮兵更名為火箭軍<ref>[http://news.wenweipo.com/2016/01/01/IN1601010031.htm 陸軍領導機構、火箭軍、戰略支援部隊成立] {{webarchive|url=https://web.archive.org/web/20160304061155/http://news.wenweipo.com/2016/01/01/IN1601010031.htm |date=2016-03-04 }}</ref>。

2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，陆军现役员额首次降到百万以下；海军、火箭军、战略支援部队现役员额有所增加，空军现役员额不变，这些军兵种总员额同陆军大体相当<ref>{{Cite web|url =http://www.thepaper.cn/newsDetail_forward_1729808|title =解放军打破“大陆军”结构：陆军现役员额首次降到百万以下|author =|date =2017-07-11|publisher =澎湃新闻|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20170711181952/http://www.thepaper.cn/newsDetail_forward_1729808|archivedate =2017-07-11}}</ref>。

=== 中国人民解放军参与的战争与战役 ===
*1946年－1950年：'''第二次國共內戰'''，交战方[[中华民国国军|中华民国国军]]。
**[[三大战役|三大战役]]
**[[渡江战役|渡江战役]]
*1950年－1953年：'''朝鮮战争'''（以中国人民志愿军名义），交战方联合国军、[[大韩民国国军|大韩民国国军]]。
**[[朝鲜战争#中国人民志愿军参战|五次战役]]
**[[上甘岭战役|上甘岭战役]]
*1955年：[[一江山岛战役|一江山岛战役]]，交战方中华民国国军。
*1958年：[[金门炮战|-{zh-hans:金门炮战; zh-hant:八二三炮戰;}-]]（第二次台湾海峡危机），交战方中华民国国军。
*1959年：平息[[1959年藏区骚乱|藏区骚乱]]。
*1962年：[[中印边界自卫反击战|中印边界自卫反击战]]，交战方[[印度武装部队|印度武装部队]]。
*1965年：[[八六海战|八六海战]]、[[东引海战|东引海战]]、[[崇武以东海战|崇武以东海战]]，交战方中华民国国军。
*1965年：[[越南战争|越南战争]](抗美援越战争)，交战方[[反共主义|反共主义]]阵营。
*1969年：[[珍宝岛事件|珍宝岛自卫反击战]]、[[铁列克提|铁列克提事件]]，交战方[[苏联武装力量|苏联武装力量]]。
*1974年：[[西沙自卫反击战|西沙自卫反击战]]，交战方[[越南共和国军|南越军队]]。
*1979年－1990年：[[1979年－1990年中越边境冲突|中越边境冲突]]（主要为1979年[[对越自卫反击战|对越自卫反击战]]、1984年[[两山战役|两山战役]]），交战方[[越南人民军|越南人民军]]。
*1987年：[[1987年中印边境冲突|中印边境冲突]]，交战方[[印度武装部队|印度武装部队]]。
*1988年：[[赤瓜礁海战|赤瓜礁海战]]，交战方越南人民军。

==其他事件与行动==
中国人民解放军参与的著名事件和其他重要的行动有：
*1949年至今：中华人民共和国历次[[阅兵#_中华人民共和国|阅兵]]。
*1976年：[[唐山大地震|唐山大地震]]救援。
*1989年：[[六四事件|北京戒严任务]]。
*1995年－1996年：[[台湾海峡导弹危机|台湾海峡导弹危机]]。
*1997年：[[中国人民解放军驻香港部队|中国人民解放军驻香港部队]]进驻[[香港|香港]]。
*1998年：[[1998年中国水灾|中国长江水灾]]救援。
*1999年：[[中国人民解放军驻澳门部队|中国人民解放军驻澳门部队]]进驻[[澳门|澳门]]。
*2001年：[[中美撞机事件|中美撞机事件]]。
*2008年：[[2008年中国雪灾|中国南方雪灾]]救援。
*2008年：[[汶川大地震|汶川大地震]]救援。
*2008年至今：[[中国海军索马里护航行动|中国海军索马里护航行动]]。
*2010年：[[玉树地震|玉树地震]]救援、[[2010年舟曲泥石流災害|舟曲泥石流]]救援。
*2011年：[[利比亚内战|利比亚]]撤侨。
*2013年：[[芦山地震|芦山地震]]救援。
*2014年：[[馬來西亞航空370號班機空難|馬來西亞航空370號班機空難]]搜救。
*2015年：[[果断风暴行动|也门]]撤侨。
