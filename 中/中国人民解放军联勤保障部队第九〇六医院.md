'''中国人民解放军联勤保障部队第九〇六医院'''，前称'''中国人民解放军第一一三医院'''，位于[[浙江省|浙江省]][[宁波市|宁波市]][[鄞州区|鄞州区]]中山东路377号，隶属[[中国人民解放军东部战区海军|中国人民解放军东部战区海军]]后勤部。

==简介==
第九〇六医院的前身是'''中国人民解放军陆军第二十二军医院'''，1951年9月创建于[[浙江省|浙江省]][[定海县|定海县]]，1953年改称'''中国人民解放军华东军区第六野战医院'''，1954年9月改称'''中国人民解放军第十四预备医院'''，1956年5月改称'''中国人民解放军第一一三医院'''。1987年移驻[[浙江省|浙江省]][[宁波市|宁波市]]<ref name=zjsjsz> {{Cite book | author =浙江省军事志编纂委员会编  | title = 浙江省军事志 | location =  | publisher =方志出版社  | date = 1999年 | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote = 第三篇 驻军 第三章 中华人民共和国成立后驻军 第五节 医院(疗养院) }} </ref>。

中国人民解放军第四一二医院：1954年6月，以东北军区第十三医院第一分院及华东军区海军淞沪基地医院部分工作人员为基础，组建'''中国人民解放军海军宁波医院'''。同年9月，改称'''中国人民解放军第四一二医院'''，隶属华东军区海军后勤部建制领导，院址位于浙江省[[鄞县|鄞县]]。1962年，划归[[中国人民解放军海军东海舰队|中国人民解放军海军东海舰队]]后勤部建制领导，1985年11月改属[[中国人民解放军海军舟山水警区|中国人民解放军海军舟山基地]]（中国人民解放军37502部队）后勤部建制领导<ref name=zjsjsz/>。2000年，第四一二医院并入第一一三医院。

2014年，经[[中国人民解放军总后勤部卫生部|中国人民解放军总后勤部卫生部]]审核批准，第一一三医院生殖医学中心正式揭牌成立<ref>{{Cite web |url = http://news.cnnb.com.cn/system/2014/01/29/007976109.shtml |title = 113医院生殖医学中心近日正式揭牌成立 |author =  |date = 2014-01-29 |publisher =  中国宁波网|language =  |accessdate =  |quote =  }}</ref>。

第一一三医院原先隶属[[中国人民解放军南京军区|中国人民解放军南京军区]]联勤部<ref name=nanjing>{{Cite web|url =http://news.xinhuanet.com/mil/2014-06/23/c_126648752.htm|title =南京军区第113医院提升战伤救治能力|author =|date =2014-06-23|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20140706022754/http://news.xinhuanet.com/mil/2014-06/23/c_126648752.htm|archivedate =2014-07-06}}</ref>。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军南京军区|中国人民解放军南京军区]]联勤部转隶[[中国人民解放军东部战区海军|中国人民解放军东部战区海军]]后勤部<ref>{{Cite web |url =  http://navy.81.cn/content/2016-11/08/content_7348289.htm|title = 解放军第113医院组织90余名新聘人员集训 |author =  |date =2016-11-08  |publisher =  中国海军网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://db.81.cn/content/2016-06/03/content_7085671.htm |title =一位人民军医的妙手丹心  |author =  |date =2016-06-03  |publisher =军报记者  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.81.cn/zghjy/2016-11/08/content_7348326.htm|title =  113医院组织新聘人员集训夯实“兵之初”思想根基|author =  |date = 2016-11-08 |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref>。
