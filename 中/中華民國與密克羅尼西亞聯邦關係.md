'''中華民國與密克羅尼西亞聯邦關係'''是指[[中華民國|中華民國]]與[[密克羅尼西亞聯邦|密克羅尼西亞聯邦]]之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對密克羅尼西亞的相關事務由'''中華民國駐[[帛琉共和國|帛琉共和國]][[外交代表機構#大使馆|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-135-1ddbe-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|密克羅尼西亞簽證政策|Visa policy of Micronesia}}|{{link-en|密克羅尼西亞公民簽證要求|Visa requirements for Micronesian citizens}}}}
目前[[国家列表_(按洲排列)|全世界]]的外籍公民都可以[[簽證#签证优惠待遇|免簽證]]的方式入境密克羅尼西亞，依國籍可停留30天、90天、1年。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可停留最多30天，若停留超過30天，須事先申請[[簽證|簽證]]，可至密克羅尼西亞駐[[關島|關島]][[總領事館|總領事館]]辦理。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-135-1ddbe-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&sms=B75E93A3DAB23A99 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|密克羅尼西亞護照|Micronesian passport}}的{{link-en|密克羅尼西亞人口|Demographics of the Federated States of Micronesia|密克羅尼西亞公民|}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|密克羅尼西亞經濟|Economy of the Federated States of Micronesia}}}}
===貿易===
2018年，密克羅尼西亞是中華民國第158大貿易夥伴、第190大[[進口|進口]]夥伴、第145大[[出口贸易|出口]]夥伴。出口至密克羅尼西亞的金額為569萬4,360[[美元|美元]]，年減81.554%；自密克羅尼西亞進口的金額為9萬8,956美元，年增31.342%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）559萬5,404美元，年減81.830%。2017年，密克羅尼西亞是中華民國第117大貿易夥伴、第194大進口夥伴、第99大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
