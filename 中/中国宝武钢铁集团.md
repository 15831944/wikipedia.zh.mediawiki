'''中国宝武钢铁集团有限公司'''（简称'''宝武集团'''、'''宝武'''，'''Baowu'''）由原宝钢集团有限公司和[[武钢集团|武汉钢铁（集团）公司]]联合重组而成，为国有独资公司，是中国最具竞争力的钢铁联合企业，总部位于上海。宝武已涉足包括钢铁主业和资源开发及物流、钢材延伸加工、工程技术服务、煤化工、金融投资、生产服务等七大相关产业，其钢铁主业已形成普碳钢、不锈钢、特钢三大产品系列，广泛应用于汽车、家电、石油化工、机械制造、能源交通、建筑装潢、金属制品、航天航空、核电、电子仪表等行业。宝钢产品通过遍布全球的销售网络，畅销国内外市场，不仅保持国内板材市场的主导地位，而且将钢铁精品出口至日本、韩国、欧美等四十多个国家和地区。

宝钢为美国《财富》杂志评选的世界500强企业之一，旗下子公司宝山钢铁股份有限公司(SH:600019)是宝钢集团在上海证券交易所的上市公司。另集团持有[[新華保險|新华人寿]]15.11%股权。<ref>[http://www.hkexnews.hk/listedco/listconews/sehk/2016/0329/LTN20160329943_C.pdf 新华人寿 2015年年報]</ref>

== 历史 ==
[[File:1967-01_1966年上海第三钢铁厂.jpg|thumb]]
1977年1月，上海市和冶金工业部鉴于上海地方钢铁工业长期缺铁，重新提出建设现代化铁厂的建议，拟在现址附近筹建炼铁厂，这是宝钢工程的最初设想<ref name=bgz1>{{cite news|url=http://www.shtong.gov.cn/Newsite/node2/node2245/node74532/node74536/index.html|title=宝钢志→大事记|publisher=  上海市地方志办公室}}</ref><ref name=bgz2>{{cite news|url=http://www.shtong.gov.cn/Newsite/node2/node2245/node74532/node74537/index.html|title=宝钢志→工程建设篇|publisher=  上海市地方志办公室}}</ref>。1977年9月，[[中华人民共和国冶金工业部|冶金部]]副部长[[叶志强|叶志强]]率领包括[[中华人民共和国国家计划委员会|国家计委]]在内的“中国金属学会”代表团访日，回国后向国务院建议从日本引进冶金技术新设备。1977年10月，[[邓小平|邓小平]]访问日本，参观[[新日鐵|新日铁]]{{link-ja|新日鐵住金君津製鐵所|新日鐵住金君津製鐵所|君津制铁所}}，对陪同的新日铁会长[[稻山嘉宽|稻山嘉宽]]和社长[[斋藤英四郎|斋藤英四郎]]说“就照这个工厂的样子帮我们建设一个”。1977年11月，[[中国共产党中央政治局|中共中央政治局]]决定从“新日铁”引进成套设备，在上海建厂。1977年11月下旬，国家计委、冶金部、外贸部联合向国务院呈上了《关于引进新技术和装备，加速钢铁工业的报告》，建议“抢建上海炼铁厂”，“拟引进两座4000立方米高炉”，“拟在1980年建成第一座高炉”。1977年11月29日，李先念接见[[日中长期贸易协议委员会|日中长期贸易协议委员会]]委员长、新日铁会长稻山嘉宽，谈及双方合作建设大型钢铁厂时，稻山积极回应。1977年底，[[中国共产党上海市委员会|中共上海市委]]第三书记[[彭冲|彭冲]]主持会议，确定新建钢铁厂的厂名定为上海宝山钢铁总厂<ref name=bgz1/>。

在[[文化大革命|文化大革命]]之后，1978年2月[[第五届全国人大|第五届全国人大]]第一次会议通过《1976年到1985年发展国民经济十年规划纲要》规定：到1985年，钢产量要达到6000万吨，原油产量要达到2.5亿吨；从1978年至1985年，建成14个大型重工业基地。

1978年3月9日，国家计委、经委、建委、上海市、冶金部向国务院报送《关于上海宝山钢铁总厂的厂址选择、建设规模和有关问题的请示报告》，提出厂址宜选在宝山[[月浦镇|月浦]]，建设规模为年产钢、铁各600万吨<ref name=bgz1/>。3月30日，中共中央正式批准了宝钢建设项目，4月6日，宝钢建设用地被中共上海市委批覆同意，一次征用土地11581亩<ref name=bgz1/>。上海郊区之一[[宝山区_(上海市)|宝山区]]被选中成为建设地址。1978年12月21日[[中国技术进出口总公司|中国技术进出口总公司]]和日本新日铁公司在上海签订了关于订购上海宝山钢铁总厂成套设备的总协议书。1978年12月23日，宝山钢铁总厂正式开工建设，预定分两期、用7年时间建成<ref name=bgz1/>。

1979年6月16日开始，国务院财经委员会连续召开四次会议，专题讨论宝钢工程问题。副总理[[李先念|李先念]]在会议上说：“宝钢放在上海是为了解决调铁的问题。开始并不想搞这样大的规模，由于高炉很大，上海现有钢厂吃不了这么多铁，发生了‘大茶壶小茶碗’的问题，于是增加了炼钢、轧钢设备，这样就形成了一个完整的钢厂。同年9月，邓小平在一次会议上说：“历史将证明，建设宝钢是正确的<ref name=bgz1/>”。

1980年夏季，正在打桩的宝钢初轧工地发生了桩基位移。宝钢顾问委员会的专家，经过现场踏勘，认为“大面积土建位移属于正常情况”，“位移量并未超过上限”。1980年11月4至5日，[[中央财经委员会|中央财经领导小组]]全体会议决定：宝钢工程一号高炉系统(一期工程建设)推迟到1984年建成；二期工程全部停、缓建。1981年1月国家计委、建委主持召开宝钢一期工程建设论证会。1981年8月7日，根据国务院的指示，国家计委、国家建委联合发出通知：宝钢一期改列为续建项目。1985年9月15日，经过长达7年多时间的筹建，宝钢一期建成投产<ref name=bgz1/>。
[[File:上海宝钢南京梅山公司_-_panoramio.jpg|thumb]]
宝钢在中国经济飞速增长带来的钢铁缺口中获益。然而，随着经济的进一步自由化，宝钢在国内外有了新的竞争对手。该公司决定新增一个进出口贸易营销部门，以保持竞争力，这项计策在[[大韩民国|韩国]]获得了极大成功。在[[亚洲金融危机|亚洲金融危机]]中，宝钢本身虽然遭受损失，但设法保持盈利，并吞并其它亏损的国有企业。

1987年8月，宝钢总厂和宝钢工程指挥部成立改革研究小组，草拟组建以上海宝山钢铁总厂为主体的横向经济联合组织——宝山钢铁联合（集团）公司的方案，这一方案于1988年6月获[[国家体制改革委员会|国家体制改革委员会]]批准。1988年8月8日，宝钢集团在宝山宾馆隆重举行成立大会。1991年12月，国务院下达国发（1991）71号文《批转国家计委、国家体改委、国务院生产公室关于选择一批大型企业集团进行试点请示的通知》，将宝钢集团列入第一批试点企业集团<ref name=bgz3>{{cite news|url=http://www.shtong.gov.cn/Newsite/node2/node2245/node74532/node74543/index.html|title=宝钢志→宝钢集团篇|publisher=  上海市地方志办公室}}</ref>。1993年7月15日，经国家体制改革委员会、国家经济贸易委员会批准，[[中华人民共和国国家工商行政管理总局|国家工商行政管理局]]注册登记，上海宝山钢铁总厂改名为宝山钢铁（集团）公司<ref name=bgz4>{{cite news|url=http://www.shtong.gov.cn/Newsite/node2/node2245/node74532/node74535/index.html|title=宝钢志→总述|publisher=  上海市地方志办公室}}</ref>。1998年11月17日，宝山钢铁（集团）公司吞并了[[上海冶金控股（集团）公司|上海冶金控股（集团）公司]](上鋼)和[[上海梅山有限公司|上海梅山（集团）公司]](梅鋼)，重命名为上海宝钢集团公司。新公司成为中国最大钢铁企业，年产量接近两千万吨。{{citation needed}}

1997年上海冶金控股旗下的[[上海浦东钢铁有限公司|上海浦东钢铁有限公司]](原上海第三钢铁厂)與德國[[蒂森克虜伯|克虏伯蒂森不锈钢有限公司]]建立中德合營公司[[上海克虏伯不锈钢有限公司|上海克虏伯不锈钢有限公司]]，<ref>{{cite web |url=http://www.skschina.com/cn/company.html |title=存档副本 |accessdate=2016-10-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20160306104830/http://www.skschina.com/cn/company.html |archivedate=2016-03-06 }}</ref>1998年隨母公司上海冶金控股成為寶鋼集團子公司。

2000年寶鋼集團子公司宝钢股份於[[上海证券交易所|上海证券交易所]]上市。<ref>[http://www.sse.com.cn/disclosure/listedinfo/announcement/c/600019_2001_n.pdf 宝钢股份 2001年年報]</ref>2004年宝钢股份向宝钢铁集团收購[[上海梅山钢铁股份有限公司|上海梅山钢铁股份有限公司]] (梅钢)。

2008 年寶鋼集團成立[[廣東鋼鐵集團|廣東鋼鐵集團]]，計劃於[[广东省|广东省]][[湛江市|湛江市]]建筑新厂。原計劃[[韶關鋼鐵|韶關鋼鐵]]集团(韶鋼集团)與[[广州钢铁企业集团|广州钢铁企业集团]](广钢集团)同時成為廣東鋼鐵集團成員，廣東、广州國資委成為廣東鋼鐵集團小股東(20%)，2011年廣東、广州國資委退出廣東鋼鐵集團，韶鋼集团、广钢集团跟隨退出。同年，51%韶關鋼鐵集团權益由廣東國資委無償划转寶鋼集團。韶鋼集团為[[韶鋼松山|韶鋼松山]]母公司 (53.37%, {{szse|000717|nocat}})。

2012年10月[[宝钢股份|宝钢股份]]向广州國資委收購71.8032%[[湛江钢铁|湛江钢铁]]股權，作價49.76億，由宝钢股份及湛江钢铁出資興建湛江新厂。<ref>[http://static.sse.com.cn/disclosure/listedinfo/announcement/c/2012-10-30/600019_20121030_3.pdf 钢股份关于收购湛江钢铁股权的公告] 2011-10-30</ref>宝钢已於同年5月得到[[国家发展改革委|国家发展改革委]]批准興建該項目。<ref>[http://money.163.com/12/0528/15/82JQCTTL002524SO.html 发改委关于湛江钢铁基地项目的批文]</ref>

2011年9月1日由山西[[太原鋼鐵|太原鋼鐵]]集團與寶鋼集團、[[中信股份|中信]]金屬公司、[[鞍山鋼鐵集團|鞍鋼]]和[[首钢集团|首鋼]]共同出資成立的[[中國鈮業投資控股有限公司|中國鈮業投資控股有限公司]]成功以19.5億美元收購世界最大鈮公司巴西礦冶（CBMM）15%的股權。CBMM由巴西Moreira Salles家族控制，五大企業完成入股權後，該家族仍持有七成控股權。{{citation needed}}

2012年12月20日[[寶鋼股份|寶鋼股份]]將投資80億元，參與中石油集團西氣東輸三線工程，佔合資公司股權比例約12.8%。<ref name=宝鋼股份2015年報>[http://www.cninfo.com.cn/cninfo-new/disclosure/sse/bulletin_detail/true/1202115798?announceTime=2016-03-31 寶鋼股份 2015年年報]</ref>初步估算，該項目總投資約1160億元，合資公司註冊資本擬為625億元。{{citation needed}}寶鋼集團下屬全資子公司華寶投資共同參股西三線管道專案，其中華寶投資計劃投資金額20億元，佔合資公司股權比例約3.2%。{{citation needed}}

== 宝钢地区办事处 ==
1978年，宝山钢铁总厂开始建设。<ref name="上海民政志解放后区划管理">[http://shtong.gov.cn/newsite/node2/node2245/node65977/node65987/node66011/userobject1ai61577.html 《上海民政志》>>第二章 行政区划管理>>第二节 解放后区划管理] {{webarchive|url=https://web.archive.org/web/20150924122646/http://www.shtong.gov.cn/newsite/node2/node2245/node65977/node65987/node66011/userobject1ai61577.html |date=2015-09-24 }}</ref>为加强地区对宝钢建设的服务，<ref name="吴淞区志建置沿革">[http://shtong.gov.cn/newsite/node2/node4/node2249/node2685/node2690/node2709/node60996/userobject1ai50195.html 《吴淞区志》>>第一篇 地理建置>>二、建置>>建置沿革] {{webarchive|url=https://web.archive.org/web/20160119145011/http://www.shtong.gov.cn/newsite/node2/node4/node2249/node2685/node2690/node2709/node60996/userobject1ai50195.html |date=2016-01-19 }}</ref>同年12月，设立[[上海市革命委员会|上海市革命委员会]]'''宝钢地区办事处'''，作为上海市革命委员会的派出机关，属区级，行使区政权的某些职能。<ref name="上海民政志解放后区划管理" />

宝钢地区办事处辖有与宝钢建设有关的盛桥、月浦等居民点。1979年11月，又将[[杨浦区|杨浦区]]的吴淞地区（包括[[吴淞街道|吴淞]]、泗塘两街道）划入宝钢地区。宝钢地区办事处的管辖范围约25平方公里。<ref name="上海民政志解放后区划管理" />

由于宝钢地区办事处不是区一级行政建制，不能设立人民代表大会和人民政府，直接影响这个地区居民行使公民权利和进行政权建设；同时，由于没有划定明确的区域界线与[[宝山县|宝山县]]的某些地段犬牙交错，在市政建设、商品供应、市场管理、社会治安等方面都有许多矛盾。为了全面规划地区建设，加强行政管理，保证宝山钢铁总厂重点工程建设顺利进行。1980年10月，经[[中华人民共和国国务院|国务院]]批准，恢复[[吴淞区|吴淞区]]建制。将宝钢地区办事处的辖区和宝山县城厢镇（0.6平方公里）以及宝钢与吴淞之间的农田空地（24.4平方公里），共计50平方公里的地区作为吴淞区的行政区域。<ref name="上海民政志解放后区划管理" />
