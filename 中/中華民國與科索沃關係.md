'''中華民國與科索沃關係'''是指[[中華民國|中華民國]]與[[科索沃共和國|科索沃共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對科索沃的相關事務由'''駐[[匈牙利|匈牙利]][[台北经济文化代表处|台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-204-4dec2-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，[[科索沃|科索沃]]當時屬於[[南斯拉夫社會主義聯邦共和國|南斯拉夫]]的[[塞爾維亞社會主義共和國|塞爾維亞社會主義共和國]]，由南斯拉夫作為[[主權國家|主權國家]]參與投票。]]
{{see also|中華民國外交|科索沃外交}}
1999年6月7日，在[[科索沃战争|科索沃战争]]后，[[中華民國總統|中華民國總統]][[李登辉|李登辉]]宣布将以3亿[[美元|美元]]援助科索沃<ref>{{cite web|title=我決金援科索沃三億美元 |url=http://forums.chinatimes.com/report/diplomacy/88060801.htm |publisher=[[中國時報|中國時報]] |date=1999-06-08 |accessdate=2016-10-19}}</ref>，[[中華人民共和國外交部发言人|中華人民共和國外交部发言人]][[章啟月|章啟月]]对这项援助案表示想要制造「[[两个中国|两个中国]]」或「[[一中一台|一中一台]]」；[[行政院大陸委員會|行政院大陸委員會]]副主委[[许柯生|许柯生]]对媒体表示只是纯粹的[[人道援助|人道考量]]，北京方面不必有过度的政治联想<ref>{{cite web|title=過境紐約 蕭揆兩度與美官員溝通援外案 |url=http://forums.chinatimes.com/report/diplomacy/88061007.htm |publisher=中國時報 |date=1999-06-10 |accessdate=2016-10-19}}</ref>。除了金援外，也捐贈上百輛[[公共汽車|公共汽車]]，上面寫著「Love from Taiwan」。<ref name="拒絕"/>

2008年2月19日，[[科索沃宣佈獨立事件|科索沃宣佈獨立]]兩天後，[[中華民國外交部|中華民國外交部]]正式[[外交#外交上的承認|承認]]科索沃共和國。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=5028B03CED127255&s=3C269AA9ACAB4478 |author = 公眾外交協調會 |date = 2008年2月19日 |title = 中華民國（台灣）自即日起正式承認科索沃共和國 |publisher = 中華民國外交部}}</ref>擔任科索沃總理[[哈辛·塔奇|塔奇]]（Hashim Thaçi）外交政策[[幕僚|幕僚]]與[[科索沃民主黨|科索沃民主黨]]國會議員的{{link-en|拉斯尼奇|Memli Krasniqi}}（Memli Krasniqi）說：「真的很謝謝[[台灣|台灣]]，但是我們要加入[[聯合國|聯合國]]，我們需要[[中國|中國]]的支持」。<ref name="拒絕">{{Cite web |url = http://alexandroslee.pixnet.net/blog/post/17863807-%E7%8D%A8%E7%AB%8B%E5%BE%8C%E7%9A%84%E7%A7%91%E7%B4%A2%E6%B2%83-%E7%82%BA%E5%85%A5%E8%81%AF%E5%90%88%E5%9C%8B-%E6%8B%92%E7%B5%95%E8%88%87%E5%8F%B0%E7%81%A3%E5%BB%BA%E4%BA%A4%28 |author = 外電報導 |date = 2008年3月3日 |title = 獨立後的科索沃 為入聯合國 拒絕台灣 |publisher = [[痞客邦|痞客邦]]}}</ref>名為「Who Recognized Kosova as an independent state? The Kosovar people thank you!」網站上的各國承認列表也無[[中華民國|中華民國]]或台灣的標示<ref>{{Cite web |url = http://kosovothanksyou.com/ |title = Who Recognized Kosova as an independent state? The Kosovar people thank you! |publisher = kosovothanksyou}}</ref>。然而[[中華民國國旗|中華民國國旗]]一度出現在科索沃政府的感謝名單上，引發[[中華人民共和國|中華人民共和國]]的不滿<ref>[https://www.thenewslens.com/article/40881 跟台灣一樣「不被國際承認」的科索沃，終於成功加入國際足總]</ref>，現今{{link-en|科索沃外交部|Ministry of Foreign Affairs (Kosovo)}}對[[台灣|台灣]]的官方稱謂為「台灣－[[中華台北|中華台北]]（台灣）」。<ref>http://www.mfa-ks.net/?page=2,102</ref>

===科索沃與台灣獨立問題===
2016年12月18日，台灣憲法學會[[秘書長|副秘書長]]、[[國立清華大學|國立清華大學]]科技法律研究所教授[[黃居正|黃居正]]表示，國際間有科索沃的先例，透過聯合國的臨時治理當局下的議會發表宣言宣佈獨立，因此在[[中華民國政府|中華民國體制]]下「透過[[立法院|立法院]]來[[台灣憲法|制憲]]、來[[臺灣獨立運動|宣佈獨立]]，並不是不能做」。<ref>{{Cite web |url = http://www.peoplenews.tw/news/9799dc80-ffd2-41cd-aa3f-7d9b8c8fbe87 |author = 唐詩 |date = 2016年12月18日 |title = 歐巴馬沒錯！黃居正：台灣應突破現狀，制憲可採「科索沃模式」 |publisher = [[民報_(2014年)|民報]]}}</ref>

持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境科索沃，於6個月期限內停留最多90天。該國法律規定行前須至科索沃駐[[土耳其|土耳其]][[伊斯坦堡|伊斯坦堡]][[領事館|領事館]]通報入境事宜。<ref name="簽證及入境須知">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-204-4dec2-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持有[[科索沃護照|科索沃護照]]的{{link-en|科索沃公民|Demographics of Kosovo}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將科索沃歸類為'''灰色警示'''：提醒注意（2017年2月13日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-181-8cd1a-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>
