'''中和車站'''位於[[臺灣|臺灣]][[新北市|新北市]][[中和區|中和區]]，曾為[[臺灣鐵路管理局|臺灣鐵路管理局]][[中和線_(台鐵)|中和線]]（現已廢止）的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*目前無完整資料，僅有照片。
*依照片中資料顯示，應有一座島式月台與側式月台。<ref>{{cite web|language=zh-tw|url=http://ticket.myweb.hinet.net/station/a28,01.htm|title=中和線|author=|publisher=|date=|accessdate=2013-10-23}}</ref>

== 利用狀況 ==
*目前已廢除，廢除前為貨運站。
*今址為[[台北捷運|台北捷運]][[南勢角站|南勢角站]]及[[中和機廠|中和機廠]]。

== 車站週邊 ==
*中和國中
*興南國小
*華夏科技大學

== 歷史 ==
*1965年3月25日：設站。為三等站。
*1980年1月16日：改為簡易站，由板橋車站管理。
*1990年9月23日：廢站。
