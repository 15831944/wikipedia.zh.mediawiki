中華人民共和國的[[中华人民共和国婚姻法|法律]]規定婚姻为一夫一妻、男女平等的婚姻制度。對同性結婚法無明文規定，法律制定時，未考慮到同性[[婚姻|婚姻]]以及其他形式的[[民事結合|民事結合]]的需求和可能性。2009年的调查显示超过30%的[[北京|北京]]人支持“[[同性婚姻|同性婚姻]]”，其余持不确定或反对态度<ref>{{cite news|url=http://www.guardian.co.uk/world/2009/feb/25/gay-rights-china-beijing|title=Beijing's 'happy couples' launch campaign for same-sex marriages(同性婚姻"幸福俪偶"活动在北京举行)|date=2009-02-25|language=en}}</ref>。2007年[[上海|上海]]的调查显示30%的受访者支持同性婚姻。

== 法律提案 ==
《[[s:中华人民共和国婚姻法|中华人民共和国婚姻法]]》第二条和第五条<ref>{{cite web |url = http://www.gov.cn/banshi/2005-08/21/content_25037.htm |title=中华人民共和国婚姻法（修正）|language=zh-cn |publisher=中国政府门户网站 www.gov.cn |date=2005-08-21 |id=全国人大法规库}}</ref>中規定“婚姻”為一夫一妻、男女平等的婚姻制度，而未考慮到同性[[婚姻|婚姻]]以及其他形式的[[民事結合|民事結合]]的需求和可能性。同性之间“结婚”，在[[中華人民共和國|中華人民共和國]]不具法律效力，也不受法律保护和拘束。

在中国同性恋社群里广为人知的社会学家[[李銀河_(社會學家)|李银河]]教授分别在2003年、2005年、2006年和2008年向[[中国人民政治协商会议|中国人民政治协商会议]]提交了作为婚姻法修正案的《[[中国同性婚姻提案|中国同性婚姻提案]]》。因为未能得到足够多的附议人，她的所有的四次提案均未得以进入大会审定议题。然而，李银河于2008年表示“虽然今年几乎不可能成功，但是应当一年接一年地提下去，直到它通过为止。”<ref>{{cite news |url = http://it.sohu.com/20060305/n242135417.shtml |title=李银河同性婚姻立法再受挫 此观点在中国太超前|date=2006-03-05}}</ref>2008年，同性恋权利支持机构发起了一项运动，意图收集足够的签名来支持对同性婚姻的认可。<ref>{{cite web |url = http://www.ny.fx120.net/eden/xwh/zgxdxwh/200807/19566.htm |title = 中国同性婚姻大事记 |date = 2008-07 |deadurl = yes |archiveurl = https://web.archive.org/web/20100420074022/http://www.ny.fx120.net/eden/xwh/zgxdxwh/200807/19566.htm |archivedate = 2010-04-20 }}</ref>2010年月1月3日，一对分别为46岁及27岁的同性恋男子在中国[[成都|成都]]的一个酒吧内举行了[[婚礼|婚礼]]。国内媒体均有报道，但是他们的婚姻在中国大陆并不能获得任何法律形式上的认可及保护。<ref>{{cite news |url = http://news.sohu.com/20100126/n269822277.shtml |title = 国内首例男同性恋公开结婚 吁社会平等对待 |date=2010-01-26}}</ref><ref>{{cite news |url = http://news.163.com/photonew/00AP0001/7654.html |title=成都两位男同性恋公开结婚 |date=2010-01-26}}</ref><ref>{{cite news |url = http://news.xinhuanet.com/society/2010-01/26/content_12874445.htm|title=国内首例男同性恋者公开结婚 每天遭数十人围观|date=2010-01-26}}</ref>

2013年，百餘同性戀者父母致信人大籲婚姻合法化<ref>{{cite news|title=百餘同性戀者父母致信人大籲婚姻合法化|url=http://www.bbc.com/zhongwen/trad/china/2013/02/130227_china_gay_open_letter.shtml|agency=BBC|date=2013-02-27}}</ref>。

2015年12月16日，孫文麟和胡明亮起訴[[湖南省|湖南省]]長沙市芙蓉區民政局要求登記結婚或為「同性戀婚姻維權第一案」，他主張《婚姻法》原文並不是「一男一女」，而是「一夫一妻」。「在我的理解裡，這個『一夫一妻』並不是指要一男一女的異性戀才能結婚，而是男男、女女、男女都可以結婚，這樣的法律才算是沒有歧視的。」<ref>{{cite news|title=湖南現「同性戀婚姻維權第一案」：登記結婚遭拒後起訴民政局|url=http://www.thepaper.cn/newsDetail_forward_1410583|agency=澎湃新聞|date=2015-12-18}}</ref><ref>{{cite news|title=中國同志婚姻權第一案，民政局被告上法庭|url=http://cn.nytimes.com/china/20160128/c28chinagay/|agency=紐約時報}}</ref>。4月13日，長沙芙蓉區中級人民法院宣判原告敗訴，社會學家[[李銀河_(社會學家)|李银河]]表示：「他們做這件事是很有勇氣的事情，即使失敗也要堅持，也要爭取自己的權利。勝訴的可能性非常小，因為它只是按照現行的婚姻法做另一種解釋，這東西不足以推翻原來的條文，所以還是象徵意義比實質意義大」<ref>{{cite news|title=中國 「同性戀維權第一案」敗訴|url=http://news.uschinapress.com/2016/0413/1060881.shtml|agency=僑報網|date=2016-04-13}}</ref>。

== 官方态度 ==
[[中华人民共和国政府|中华人民共和国政府]]对于同性恋的态度可以归结为“三个不”：“不支持；不反对；不提倡。”<ref>{{cite web |url = http://www.dw.de/%E5%87%BA%E6%9F%9C%E5%A4%A7%E5%AD%A6%E7%94%9F%E8%87%B4%E4%BF%A1%E4%B8%A4%E4%BC%9A%E4%BF%83%E5%90%8C%E6%80%A7%E5%A9%9A%E5%A7%BB%E5%90%88%E6%B3%95/a-17458094 |work=德国之声 |title=“出柜”大学生致信两会，促同性婚姻合法 |date=2014-02-26}}</ref><ref>{{cite web |url = http://helanonline.cn/article/3507 |title=中国两对同性恋伴侣公开结婚吁社会平等对待 |work=荷兰在线 |date=2012-08-22}}</ref>但同时一直视同性恋议题为“敏感话题”：2010年时北京警方[[牡丹园事件|突查同性恋聚集地牡丹园]]并带走大批同性恋者讯问；2013年时任[[冰岛|冰岛]][[冰岛总理|女总理]][[约翰娜·西于尔扎多蒂|约翰娜·西于尔扎多蒂]]携同性伴侣约妮娜·列奥斯多蒂访华时，中国官方媒体也刻意在报道中忽略了其夫人的消息。<ref>{{cite web |url = http://www.dw.de/%E5%86%B0%E5%B2%9B%E5%A5%B3%E6%80%BB%E7%90%86%E5%88%B0%E8%AE%BF%E5%AE%98%E5%AA%92%E5%92%8C%E8%B0%90%E5%A4%AB%E4%BA%BA/a-16749787 |work=德国之声 |title=冰岛女总理到访，官媒“和谐”夫人 |date=2013-04-17}}</ref><ref>{{cite web |url = http://www.bbc.co.uk/zhongwen/trad/china/2010/09/100928_chinabeijing_gay.shtml |title = 北京警方國慶前突查同性戀聚集地牡丹園 |work=BBC |date=2010-09-28}}</ref><ref>{{cite web |url = http://news.sina.com.cn/w/gc/2013-04-15/162740.shtml |title = 外媒：八成中国同性恋者迫于压力选择结婚 |date=2013-04-15 |work=新浪}}</ref>

1997年，同性性行为在中国被去罪化。2001年，[[中华人民共和国卫生部|卫生部]]正式将同性恋从[[精神疾病|精神疾病]]名单中移除，但对同性婚姻未作考虑。[[全国政协|全国政协]]新闻发言人[[吴建民_(中国)|吴建民]]在被问及李银河的同性婚姻提案时表示，同性婚姻在中国“仍太超前”。他强调即使在许多被认为比中国更加自由开放的西方国家，同性婚姻也未被认可<ref>{{cite news |url = http://it.sohu.com/20060305/n242135417.shtml |title = 李银河同性婚姻立法再受挫 此观点在中国太超前 |date=2006-03-05}}</ref>。这一表态可以解读为暗示中国政府也许在更远的将来考虑同性婚姻，而不会在最近考虑此事。

此外，中国政府声称“为保护儿童”，所以禁止外国同性恋者收养中国儿童。<ref>{{cite web |url = http://adoption.state.gov/country_information/country_specific_info.php?country-select=china |title = Intercountry Adoption - China - Who can Adopt? - Marriage Requirements |accessdate = 2011-12-07 |deadurl = yes |archiveurl = https://web.archive.org/web/20111222171510/http://adoption.state.gov/country_information/country_specific_info.php?country-select=china |archivedate = 2011-12-22 }}</ref>
