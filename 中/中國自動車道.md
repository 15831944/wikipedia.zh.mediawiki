'''中國自動車道'''（{{jpn|j=中国自動車道|hg=ちゅうごくじどうしゃどう|rm=Chūgoku jidōshadō|f=y}}），是起於[[日本|日本]][[大阪府|大阪府]][[吹田市|吹田市]]經由[[兵庫縣|兵庫縣]]、[[岡山縣|岡山縣]]、[[廣島縣|廣島縣]]、[[島根縣|島根縣]]到達[[山口縣|山口縣]][[下關市|下關市]]的[[日本高速公路|高速公路]]（[[高速自動車國道|高速自動車國道]]）。略稱為'''中國道'''（{{lang|ja|中国道／ちゅうごくどう，CHUGOKU EXPWY}}），法定路線名為[[中國縱貫自動車道|中國縱貫自動車道]]，當初的道路名就有使用這樣的法定路線名（道路名改為現在的「中國自動車道」時期不詳）<ref>『高速道路と自動車』1970年3月版</ref>另外，現在的地圖仍然有使用「中國縱貫道」的表記方式。

[[吹田系統交流道|吹田系統交流道]]至[[神戶系統交流道|神戶系統交流道]]間及[[山口系統交流道|山口系統交流道]]→[[下關交流道|下關交流道]]間被指定為{{AHN-AH|1}}[[泛亞公路1號線|泛亞公路1號線]]（AH1）的一部分。高速公路路線編號與[[關門橋|關門橋（關門自動車道）]]共同被編為 {{Ja Exp Route Sign|E2A}} <ref>[[山崎系統交流道|山崎系統交流道]] - [[佐用系統交流道|佐用系統交流道]]間與 {{Ja Exp Route Sign|E29}} 、[[落合系統交流道|落合系統交流道]] - [[北房系統交流道|北房系統交流道]]間與 {{Ja Exp Route Sign|E73}} 、[[千代田系統交流道|千代田系統交流道]] - [[廣島北交流道|廣島北交流道]]間與{{Ja Exp Route Sign|E74}}分別重複編號。</ref><ref>{{Cite web |url=http://www.mlit.go.jp/road/sign/numbering/list/index.html |title=高速道路ナンバリング一覧 |publisher=国土交通省 |accessdate=2017-05-05 }}</ref>。

==概要==
中國自動車道為日本最早規劃的[[高速公路|高速公路]]之一，為[[近畿|近畿]]地區貫穿整個[[中國地方|中國地方]]的東西向高速公路，其距離離[[山陽地區|山陽地區]]和[[山陰地區|山陰地區]]等距離。自1970年開始陸續通車，全線通車於1983年，總距離長達540.1KM，在日本國內僅次於[[東北自動車道|東北自動車道]]為第二長的高速公路。
由於中國自動車道在貫穿中國山區地形時大量使用[[隧道|隧道]]及[[橋梁|橋梁]]，以當時的土木技術來講沒辦法達到極為有效的貫通，導致路線極為迂迴，大半路線的速限在80km/h以下，1997年[[山陽自動車道|山陽自動車道]]全線通車後，中國自動車道的車輛移轉到山陽自動車道上，現今中國自動車道山區路段的車輛已極為稀少。

== 歷史 ==
* [[1970年|1970年]]（[[昭和|昭和]]45年）
**[[3月1日|3月1日]] : '''中国自動車道'''<ref>前年（[[1969年|1969年]]）に開通した'''中央高速道路'''（現・[[中央自動車道|中央自動車道]]）の[[暫定2車線|暫定2車線]][[対面通行|対面通行]]区間の[[中央線_(道路)|中央線]]を破線（追い越し可能区間）で供用して正面衝突事故が相次いだため、[[警察庁|警察庁]]から[[日本道路公団|日本道路公団]]に営業路線名で「高速道路」の名称使用を自粛するよう要請があり、暫定措置として[[1972年|1972年]]まで法定路線名をそのまま使用した。尚、開通時には、現在と同じく中国自動車道の名称が使われることになった。『交通工学』1970年春季増刊号・田中敬一・田口二朗「万国博関連道路計画について」写真より</ref>中國吹田交流道 - 中國豐中交流道間以[[2車道高速公路|暫定2車道]]方式通車<ref>同時期に開催されていた[[日本万国博覧会|日本万国博覧会]]会場へのアクセス路線として[[北大阪急行電鉄|北大阪急行電鉄]]が'''会場線'''という臨時線を[[1970年|1970年]][[2月24日|2月24日]]から同年[[9月14日|9月14日]]まで現上り線の敷地を利用して万国会場前まで供用していたため、現下り線部分を暫定2車線対面通行にして開通させた。尚、前述の中央高速での反省を踏まえ、中国吹田IC - 中国豊中IC間は[[追い越し|追い越し]]禁止区間（中央線を黄色の実線で供用）となった。</ref>。
**[[7月23日|7月23日]] : 中國豐中交流道 - 寶塚交流道間通車。
* [[1973年|1973年]]（昭和48年）[[11月14日|11月14日]] : 小月交流道 - 下關交流道間通車。同時與關門橋（關門自動車道）連接。
* [[1974年|1974年]]（昭和49年）
**[[6月4日|6月4日]] : 西宮北交流道 - 福崎交流道間通車。
**[[7月31日|7月31日]] : 小郡交流道 - 小月交流道間通車。
**[[12月21日|12月21日]] : 美作交流道 - 落合交流道間通車。
