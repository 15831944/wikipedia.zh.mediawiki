'''中央號誌站'''位於[[台灣|台灣]][[屏東縣|屏東縣]][[獅子鄉_(台灣)|獅子鄉]]，位於[[枋野車站|枋野車站]]至[[古莊號誌站|古莊號誌站]]間，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[南迴線|南迴線]]的鐵路[[臺灣鐵路管理局車站等級#號誌站|號誌站]]。站址位於[[中央隧道|中央隧道]]西口外，為南迴線唯一雙線區間的西側端點。

== 車站構造 ==
*無站體設施，僅有一混凝土建築，作為繼電室與電源室使用。
*無月台，站內僅有一組16號[[轉轍器|雙動轉轍器]]。
*東正線北側設有[[安全側線|安全側線]]，可避免順行列車因車輛韌機失效而對撞。

== 利用狀況 ==
*無人[[臺灣鐵路管理局車站等級#號誌站|號誌站]]，就地控制設備設於[[枋野車站|枋野站]]。
*僅供列車交會用，無站員派駐、亦無為交會及工程以外目的之列車停靠。

== 車站週邊 ==
*位於南迴線深山裡的[[秘境車站|秘境車站]]，附近無住家。
*兩端緊臨枋野三號隧道東口及[[中央隧道|中央隧道]]西口。
*山區多毒蛇及蚊蟲，前往該站須注意安全。

== 聯外交通 ==
*地處鄉道[[屏東縣鄉道列表|屏147-1線]]（已解編）的東端，距[[枋野車站|枋野站]]的公路距離約5公里，距林務局檢查哨約4.5公里，需經過林務局檢查哨。
*行動通訊死角地帶，進出需注意自身安全，建議攜伴同行。

== 歷史 ==
*1992年10月5日：設站。
