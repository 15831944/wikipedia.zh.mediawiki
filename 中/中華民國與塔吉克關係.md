==政治==
===外交===
{{see also|中華民國外交|{{link-en|塔吉克外交|Foreign relations of Tajikistan}}}}
兩國存在著[[法理|法理上]]的領土爭議。[[De_facto|實際]]被塔吉克控制的[[帕米爾高原|帕米爾高原]]，由[[中華民國疆域#領土爭議|中華民國聲索主權]]，[[戈爾諾—巴達赫尚自治州|戈爾諾—巴達赫尚自治州]]大部分都涵蓋其中。2010年4月27日，{{link-en|塔吉克政府|Politics of Tajikistan}}將帕米尔高原以东1,158[[平方公里|平方公里]]土地劃歸[[中華人民共和國|中華人民共和國]]。<ref>{{Cite web |url = http://news.ifeng.com/mainland/detail_2011_10/01/9614237_0.shtml |author = 陈思强 |date = 2011年10月1日 |title = 中塔新划定国界交接 1158平方公里国土回归(图) |publisher = [[鳳凰衛視#鳳凰網|鳳凰網]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|塔吉克簽證政策|Visa policy of Tajikistan}}|{{link-en|塔吉克公民簽證要求|Visa requirements for Tajik citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以在首都的[[杜尚別國際機場|杜尚別國際機場]]辦理[[簽證#落地签证|落地簽證]]入境塔吉克，停留最多45天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-140-12370-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=7F220D7E656BE749 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|塔吉克護照|Tajik passport}}的{{link-en|塔吉克人口|Demographics of Tajikistan|塔吉克公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將塔吉克歸類為'''灰色警示'''：提醒注意（2017年5月23日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-330-24ada-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|塔吉克經濟}}
===貿易===
2018年，塔吉克是中華民國第124大貿易夥伴、第93大[[進口|進口]]夥伴、第196大[[出口贸易|出口]]夥伴，在[[亞洲國家和地區列表|亞洲國家]]排名倒數第12、17、1；在[[世界政區索引|世界經濟體]]倒數第121、148、41。出口至塔吉克的金額為42萬2,459[[美元|美元]]，年增7.666%；自塔吉克進口的金額為2,647萬5,478美元，年增67.139%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）2,605萬3,019美元，年增68.650%。2017年，塔吉克是中華民國第133大貿易夥伴、第106大進口夥伴、第194大出口夥伴，在亞洲國家排名倒數第12、17、4；在世界經濟體倒數第106、131、37。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
