'''中華民國與烏干達關係'''是指[[中華民國|中華民國]]與[[烏干達共和國|烏干達共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對烏干達的相關事務由'''{{Link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-160-68526-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
{{see also|中華民國外交|{{link-en|烏干達外交|Foreign relations of Uganda}}}}
2017年8月，烏干達體育界人士揭露{{Link-en|烏干達外交部|Ministry of Foreign Affairs (Uganda)|該國外交部}}公文，指因為遵守「[[一個中國|一個中國]]」原則，建議{{Link-en|烏干達教育暨體育部|Ministry of Education and Sports (Uganda)|烏干達教育部}}不派出官方代表團參加[[2017年夏季世界大學運動會|台北世界大學運動會]]。但最後烏干達代表團還是如期參賽，<ref>{{Cite web |url = https://udn.com/news/story/6656/2647762 |author = 徐偉真 |date = 2017年8月17日 |title = 烏干達可如期參賽世大運 外交部表示歡迎 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref>並獲得[[2017年夏季世界大學運動會#獎牌榜|1金1銀1銅]]成績。

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|烏干達簽證政策|Visa policy of Uganda}}|{{link-en|烏干達公民簽證要求|Visa requirements for Ugandan citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。{{link-en|烏干達政府|Politics of Uganda}}對[[世界政區索引|所有地區]]的公民皆給予簽證優惠待遇，持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可至烏干達移民局網站申請入境或轉機[[簽證#电子签证|電子簽證]]，核准後以[[電子郵件|電子郵件]]傳至申請人。入境須出示[[黃熱病|黃熱病]][[疫苗接種|疫苗接種]]證明（黃皮書）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-160-68526-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/ElectronicVisa.aspx?n=6F246645F10311D7&AreaSN=D33B55D537402BAA |title = 電子簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|烏干達護照|Ugandan passport}}的{{link-en|烏干達人口|Demographics of Uganda|烏干達公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將烏干達歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2018年8月24日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-242-5f5e6-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將烏干達歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=FBA47617D573F4FA |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|烏干達經濟}}
===貿易===
2012年6月16日，[[經濟部國際貿易局|經濟部國際貿易局]]委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）與54家台灣業者執行「游龍專案」，籌組「2012年[[非洲国家和地区列表|非洲南撒哈拉]]拓銷團」。共前往[[烏干達|烏干達]]、[[坦尚尼亞|坦尚尼亞]]、[[安哥拉|安哥拉]]及[[南非|南非]]，是台灣歷來拓銷非洲最大團。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=585707&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2012年6月14日 |title = 貿協游龍專案出擊非洲，史上最大非洲拓銷團開拔！ |publisher = 經貿透視網}}</ref>

2017年，烏干達是中華民國第169大貿易夥伴、第158大[[進口|進口]]夥伴、第155大[[出口贸易|出口]]夥伴。出口至烏干達的金額為288萬2,924[[美元|美元]]，年增6.824%；自烏干達進口的金額為70萬1,106美元，年減7.725%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）218萬1,818美元，年增12.525%。※2016年，烏干達是中華民國第172大貿易夥伴、第158大進口夥伴、第157大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

==司法==
===犯罪事件===
2016年4月23日，[[中華人民共和國|中華人民共和國]]（以下簡稱對岸）[[公安机关#公安机关层级|貴州省公安廳]]破獲大型[[電信詐騙|電信詐騙]]案，詐騙據點在非洲烏干達境內，涉及金額1億1,700萬[[人民幣|人民幣]]，共逮捕62人，其中有10名[[中華民國籍|中華民國籍]]。此案件涉及對岸[[中華人民共和國行政區劃|20多個省市]]、超過180起電信詐騙案，也是對岸近年來最大的單筆電信詐騙案。<ref>{{Cite web |url = http://www.ettoday.net/news/20160423/685903.htm |author = 綜合報導 |date = 2016年4月23日 |title = 抓了10台灣人！5.9億烏干達詐騙集團分工嚴密 |publisher = [[ETtoday_東森新聞雲|ETtoday 東森新聞雲]]}}</ref>此案發生在2015年12月，貴州警方深入追查後發現主嫌為台灣人，並將詐騙機房設於非洲烏干達，全案依《兩岸共同打擊犯罪及[[法律互助協定|司法互助協議]]》偵辦。對岸於2016年1月5日派工作小組抵台會商、提供情資，2個月內陸續在[[桃園|桃園]]、[[新竹|新竹]]、[[台中|台中]]等地逮捕17名[[車手|車手]]、38名嫌犯（不包含原先逮捕的62人），查扣[[銀聯卡|銀聯卡]]1,983張、[[贓款|贓款]]668萬，分別移送[[臺灣桃園地方法院檢察署|臺灣桃園地方法院檢察署]]及[[臺灣高等法院檢察署#組織|各地檢署]]偵辦。<ref>{{Cite web |url = http://www.setn.com/News.aspx?NewsID=140931 |author = [[中央通訊社|中央通訊社]] |date = 2016年4月24日 |title = 烏干達詐騙案　刑事局在台已逮捕55名嫌犯 |publisher = [[三立新聞網|三立新聞網]]}}</ref>2017年1月20日，又逮捕躲藏台灣的旗下機房成員共23人，依[[詐欺罪|詐欺罪]]移送地檢署偵辦。<ref>{{Cite web |url = http://news.ltn.com.tw/news/society/breakingnews/1953635 |author = 邱俊福 |date = 2017年1月20日 |title = 非洲單筆詐5億得手 詐騙機房成員返國避風頭 |publisher = [[自由時報|自由時報]]}}</ref>
