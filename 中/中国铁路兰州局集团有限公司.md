'''中国铁路兰州局集团有限公司'''<ref>{{Cite web |url =  http://money.163.com/17/1023/15/D1EO750V002581PP.html|title = 中铁总公司制大进展:11家地方铁路集团新掌门亮相 |author =  |date = 2017-10-23  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>，原名'''兰州铁路局'''，是[[中國鐵路總公司|中國鐵路總公司]]下属公司。管理线路总延展长度為9390[[公里|公里]]，营业里程3851公里。其管辖范围为[[甘肃省|甘肃省]]、[[宁夏回族自治区|宁夏回族自治区]]的铁路网。

==沿革==
1956年3月1日，由原天水铁路局迁址[[兰州|兰州]]而成'''兰州铁路局'''<ref name="lanjubsc">{{cite book|author=刘皋|coauthors= 李冰; 杨鹏; 高卫国; 梁广奎|title=兰州铁路分局志|year=2002|publisher=中国铁道出版社|isbn=7-113-04667-3|pages=64-65}}</ref>。后来以“吃苦、創業、團結、奉獻”為[[企業|企業]][[標語|標語]]。

1993年，兰州铁路局进行了工商登记<ref name=guapai/>。

2017年11月7日，太原、兰州、广州铁路（集团）公司等3家铁路局（公司）的名称变更通过[[国家工商行政管理总局|国家工商行政管理总局]]核准，兰州铁路局名称变更为'''中国铁路兰州局集团有限公司'''（(国)名称变核内字[2017]第5344号）<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1854064  |title = 18家铁路局将集体变身集团公司：名称变更已全部通过核准 |author =  |date =2017-11-08   |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月13日，在国家企业信用信息公示系统中，中国铁路兰州局集团有限公司已完成工商登记变更。工商登记变更后，中国铁路兰州局集团有限公司仍为中国铁路总公司100%控股的公司，公司由原先的全民所有制企业变成一人有限责任公司，注册资本由原先中华人民共和国铁道部出资的42亿人民币，变更为由中国铁路总公司出资的1546.7144亿人民币；经营范围增加铁路专用设备及相关工业设备的制造、安装、维修、销售、租赁，经营进出口业务，物流金融等<ref name=si>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1861836 |title = 4家铁路局率先变更工商登记：广州西安局集团增房产开发功能 |author =  |date = 2017-11-13 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月19日，中国铁路兰州局集团有限公司正式挂牌<ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1870782|title = 18个铁路局已改制为铁路局集团有限公司，今日正式挂牌 |author =  |date = 2017-11-19 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref name=guapai>{{Cite web |url = http://finance.ifeng.com/a/20171119/15803099_0.shtml |title = 18家铁路局集团改制后集中挂牌！附主要领导人简历 |author =  |date = 2017-11-19 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

==机务段==

机务段负责存放、维护、检修铁路机车，工作通常包括清洗、加油、加水、检修、除尘等。兰州铁路局下辖以下机务段：
*兰州西机务段（兰局兰段）
*嘉峪关机务段（兰局嘉段）
*迎水桥机务段（兰局迎段）

==工务段==
