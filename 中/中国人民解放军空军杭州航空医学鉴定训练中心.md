'''中国人民解放军空军杭州航空医学鉴定训练中心'''，位于[[浙江省|浙江省]][[杭州市|杭州市]][[西湖区_(杭州市)|西湖区]]杨公堤15号，隶属[[中国人民解放军东部战区空军|中国人民解放军东部战区空军]]后勤部。

==简介==
'''中国人民解放军空军杭州疗养院'''于1950年5月始建<ref name=ahlg/>。2009年7月，都江堰、杭州、大连、青岛、临潼五所疗养院统一由各大军区联勤部转隶各大军区空军后勤部，成立5个航空医学鉴定训练中心，构建为“[[中国人民解放军空军总医院|空军总医院]]、[[中国人民解放军空军航空医学研究所|空军航空医学研究所]]——区域性航空医学鉴定训练中心——航空兵部队卫生机构”三级航空医学鉴定训练体系<ref>何强、彭天国、董文，空军航空医学鉴定训练中心的发展与思考 ，全国航空航天医学学术会议暨云南省航空航天医学学术会议2010,27(3):180-181</ref><ref>{{Cite web |url = http://news.xinhuanet.com/mil/2011-12/25/c_111295752.htm |title = 一场从陆地打响的空战 |author =  |date = 2011-12-25 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。其中，2009年8月31日，组建成立“'''中国人民解放军空军杭州航空医学鉴定训练中心'''”，同时保留对外名称“'''中国人民解放军空军杭州疗养院'''”，隶属[[南京军区空军|南京军区空军]]后勤部。空军杭州疗养院位于[[杭州市|杭州市]][[西湖风景名胜区|西湖风景名胜区]]中心的[[杨公堤|杨公堤]]15号，占地面积235亩，是中国人民解放军最大的飞行人员健康保健基地，承担中国人民解放军全军飞行人员体检、鉴定、训练、评估等医疗保健任务<ref name=ahlg>{{Cite web |url = http://job.aust.edu.cn/qycontent.jsp?urltype=tree.TreeTempUrl&wbtreeid=1051&userid=2660 |title = 空军杭州疗养院 |author =  |date =  |publisher = 安徽理工大学 |language =  |accessdate = 2017-05-20 |quote =  }}</ref><ref>{{Cite web |url =http://news.sina.com.cn/o/2016-11-08/doc-ifxxmyuk6191656.shtml  |title = 随军家属专场招聘 |author =  |date = 2016-11-08 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，转隶[[中国人民解放军东部战区空军|中国人民解放军东部战区空军]]后勤部。
