'''中国人民解放军军事科学院军事科学信息研究中心'''，位于[[北京市|北京市]]，是[[中国人民解放军军事科学院|中国人民解放军军事科学院]]的研究中心。

==沿革==
1947年3月，[[胡宗南|胡宗南]]率军[[重点进攻|重点进攻]][[陕甘宁边区|陕甘宁边区]]首府[[延安|延安]]。[[周恩来|周恩来]]让[[童陆生|童陆生]]负责将中共中央及中央军委多年收集的近百部兵书古籍及其他图书资料从王家坪东渡[[黄河|黄河]]向晋西的三交镇转移。几经辗转，直到1948年，[[中共中央|中共中央]]和中央军委转移到[[西柏坡|西柏坡]]后，童陆生才将这批军事图书和档案资料交给当时负责全军军事训练的[[叶剑英|叶剑英]]<ref name=tong>{{Cite web |url =http://www.81.cn/jsdj/2016-05/23/content_7067914.htm  |title = 昔日周恩来军事参谋童陆生：逛废品站淘兵书古籍 |author =  |date =2016-05-23  |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref>。

1948年6月，在军委四局属下正式建立'''军委四局图书资料室'''，以收集[[中国人民解放军|中国人民解放军]]及其友军和敌军的军事图书资料为特色<ref name=tantan/><ref name=zengchen/>。该图书资料室由叶剑英提议建立，童陆生任首任主任<ref name=tong/>。1949年随军委机关迁入北京<ref name=zhaochen/>。1950年10月，在该室基础上成立'''中央人民政府人民革命军事委员会军事训练部军事出版局军事图书馆'''，转隶[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]军事训练部军事出版局<ref name=zengchen>曾庆洋、陈庆平，军事科学院军事图书馆，文献工作研究1989(2):41-44</ref><ref name=zhaochen>赵宝利、陈庆平，军事科学院军事图书资料馆古籍保护工作报告，中国国家图书馆</ref><ref name=tongshishi/>。到1951年，已有四万多册藏书，其中66%是军事图书<ref name=tantan>李士顺，谈谈中国人民解放军的图书馆事业，图书馆学研究1983(4):30-32</ref><ref>{{Cite web |url =http://theory.people.com.cn/GB/136458/17761934.html  |title =大力加强新形势下军队网络思想政治教育——全军加强网络思想政治教育座谈会经验交流材料摘登  |author =  |date =2012-04-27  |publisher =中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。该图书馆后转隶[[中国人民解放军训练总监部|中国人民解放军训练总监部]]军事出版部，更名'''中国人民解放军训练总监部军事出版部军事图书馆'''<ref>中国历代兵书目录，中国人民解放军训练总监部军事出版部军事图书馆编印，1957年</ref><ref name=zhaochen/><ref name=tongshishi/>。

1958年3月，中国人民解放军军事科学院成立，该图书馆转隶军事科学院<ref name=zengchen/><ref name=jiepai>中国国家数字图书馆军事科学院分馆揭牌，数字图书馆论坛2009(12):83</ref><ref name=jiepaie>{{Cite web |url =http://news.163.com/09/1211/20/5Q9F5SA7000120GU.html  |title = 中国国家数字图书馆军事科学院分馆揭牌暨网站开通运行 |author =  |date =  2009-12-11 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref><ref name=tantan/>，更名'''中国人民解放军军事科学院军事图书馆'''<ref name=tongshishi/>。至1959年，该馆藏书已达40多万册，集中了大批军事图书及革命文物资料<ref name=tantan/>。1964年，军事图书馆和资料处合并为图书资料处<ref name=zengchen/>。名称先后为'''中国人民解放军军事科学院办公室图书资料处'''<ref>第二次国内革命战争时期书刊资料索引（续编），军事科学院院办图书资料处，1977年2月</ref>、'''中国人民解放军军事科学院计划指导部图书资料处'''<ref>第二次世界大战论著索引，军事科学院计划指导部图书资料处，1985年8月</ref>。1986年9月，图书资料处更名'''中国人民解放军军事科学院军事图书馆'''，隶属军事科学院计划组织部<ref name=zengchen/><ref>军事科学院军事图书馆编，中国人民解放军组织沿革和各级领导成员名录，军事科学出版社，1987年</ref>。1993年至1994年间更名'''中国人民解放军军事科学院军事图书资料馆'''<ref name=jiepai/><ref>宋晓军，全军图书资料工作一体化管理设想，图书馆论坛1993(5):60+17</ref><ref>陈浩良、周健，加强文献数据库建设 为军事科研服务，国家图书馆学刊1994(1):73-76</ref>。

军事科学院军事图书资料馆是全军军事科研信息中心，是为军事科学研究提供信息服务的学术性机构<ref name=jiepaie/>。2007年9月被[[中华人民共和国文化部|文化部]]批准为全国古籍保护试点单位<ref name=zhaochen/>。2009年6月被[[中华人民共和国国务院|国务院]]、文化部批准为全国古籍重点保护单位<ref name=jiepaie/><ref name=zhaochen/>。2009年，[[中国国家图书馆|中国国家图书馆]]与军事科学院军事图书资料馆达成合作协议，共同建设“[[中国国家数字图书馆|中国国家数字图书馆]]军事科学院分馆”。2009年12月11日，在军事科学院军事图书资料馆举行了分馆揭牌暨网站开通仪式，文化部副部长[[周和平|周和平]]、军事科学院院长[[刘成军|刘成军]]、政委[[刘源|刘源]]和副院长[[刘继贤|刘继贤]]、国家图书馆馆长[[詹福瑞|詹福瑞]]等出席并共同为“中国国家数字图书馆军事科学院分馆”揭牌，开通了“中国国家数字图书馆军事科学院分馆网站”<ref name=jiepai/>。

2017年7月19日，新调整组建的军事科学院、国防大学、国防科技大学成立大会暨军队院校、科研机构、训练机构主要领导座谈会在北京[[八一大楼|八一大楼]]举行，[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军委主席|中央军委主席]][[习近平|习近平]]向军事科学院、国防大学、国防科技大学授军旗、致训词，出席座谈会并发表讲话<ref name=xunci>{{Cite web |url =http://news.xinhuanet.com/politics/2017-07/19/c_1121347127.htm  |title = 习近平向军事科学院、国防大学、国防科技大学授军旗致训词 |author =  |date = 2017-07-19 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。新调整组建的军事科学院成立了'''中国人民解放军军事科学院军事科学信息研究中心'''<ref name=weidong>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1789598 |title =新组建的军事科学院下属八个研究院全部亮相  |author =  |date = 2017-09-09 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.sina.com.cn/o/2018-01-13/doc-ifyqqciz6325611.shtml|title = 用好军民融合大数据 |author =  |date = 2018-01-13 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。
