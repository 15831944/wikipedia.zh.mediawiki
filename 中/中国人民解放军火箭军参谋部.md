'''中国人民解放军火箭军参谋部'''，位于[[北京市|北京市]]，是[[中国人民解放军火箭军|中国人民解放军火箭军]]机关职能部门。

==沿革==
1966年7月，[[中国人民解放军第二炮兵部队|中国人民解放军第二炮兵部队]]正式成立。机关设'''中国人民解放军第二炮兵司令部'''、[[中国人民解放军第二炮兵政治部|中国人民解放军第二炮兵政治部]]、[[中国人民解放军第二炮兵后勤部|中国人民解放军第二炮兵后勤部]]。1975年12月增设中国人民解放军第二炮兵科技部（后改称技术装备部，1998年改设[[中国人民解放军第二炮兵装备部|装备部]]）<ref>{{Cite web |url =http://www.81.cn/jmywyl/2016-02/29/content_6933950.htm?1456760091721  |title = 火箭军，为第二炮兵正名 |author =  |date = 2016-02-29 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

2015年12月，在[[深化国防和军队改革|深化国防和军队改革]]中，成立[[中国人民解放军火箭军|中国人民解放军火箭军]]<ref>{{Cite web |url = http://news.xinhuanet.com/politics/2016-01/01/c_1117646667.htm  |title =陆军领导机构火箭军战略支援部队成立大会在京举行 习近平向中国人民解放军陆军火箭军战略支援部队授予军旗并致训词  |author =  |date =2016-01-01  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。中国人民解放军火箭军设'''中国人民解放军火箭军参谋部'''。
