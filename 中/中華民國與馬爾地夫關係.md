'''中華民國與馬爾代夫關係'''是指[[中華民國|中華民國]]與[[馬爾代夫共和國|馬爾代夫共和國]]之間的關係。兩國於1966－1972年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對馬爾地夫的相關事務由'''[[駐印度台北經濟文化中心|駐印度台北經濟文化中心]]－[[清奈|清奈]]辦事處'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-183-3e14c-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]一案缺席，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[馬爾地夫|馬爾地夫]]。]]
{{see also|中華民國外交|馬爾地夫外交}}
1966年7月7日，兩國建立大使級[[外交#外交上的承認|外交關係]]，於首都[[馬累|馬利]]設立'''中華民國駐馬爾地夫蘇丹國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1968年11月11日，该国改称马尔代夫共和国。

1972年4月15日，马尔代夫与中华民国断交。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境馬爾地夫。]]
[[File:Visa_requirements_for_Maldivian_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|馬爾地夫簽證政策|{{link-en|馬爾地夫公民簽證要求|Visa requirements for Maldivian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。馬爾地夫對[[世界政區索引|世界各地]]的[[公民|公民]]，皆給予[[簽證#签证优惠待遇|免簽證]]或[[簽證#落地签证|落地簽證]]的待遇。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可在抵達[[易卜拉欣·納西爾國際機場|易卜拉欣·納西爾國際機場]]後以落地簽證的方式入境馬爾地夫，免簽證費，停留最多30天但可延長。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-183-3e14c-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&sms=B75E93A3DAB23A99 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|馬爾地夫護照|Maldivian passport}}的{{link-en|馬爾地夫人口|Demographics of the Maldives|馬爾地夫公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將馬爾地夫歸類為'''灰色警示'''：提醒注意（2018年3月23日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-461-c1296-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將馬爾地夫歸類為'''第二級警示'''：對當地採取加強防護（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=BE519904483395E3 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|馬爾地夫經濟|Economy of the Maldives}}}}
===貿易===
2018年，馬爾地夫是中華民國第148大貿易夥伴、第133大[[進口|進口]]夥伴、第144大[[出口贸易|出口]]夥伴。出口至馬爾地夫的金額為572萬1,585[[美元|美元]]，年減5.401%；自馬爾地夫進口的金額為303萬9,191美元，年減4.935%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）268萬2,394美元，年減5.924%。2017年，馬爾地夫是中華民國第147大貿易夥伴、第136大進口夥伴、第141大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===觀光===
馬爾地夫的自然景色受到外國遊客的喜愛，每年有許多[[中華民國國民|台灣遊客]]前往度假，也有不少是專程到馬爾地夫拍[[婚紗|婚紗]]、度[[蜜月|蜜月]]。目前無直航班機前往馬爾地夫，須至第三地轉機。<ref>{{Cite web |url = http://www.skyscanner.com.tw/flights-to/mv/airlines-that-fly-to-maldives.html |title = 飛往馬爾地夫(MV)的航空公司 |publisher = Skyscanner}}</ref><ref>{{Cite web |url = http://www.startravel.com.tw/StarTravel.Web.IAIR/AChoiceTicketPrice_Project.aspx?PROD_NO=&comp=&forward=2&s1=SCASIA&CTKT=&PROJECT=&s2=MD&CFIT=&PS=QS&from=&C2FIT=&REGION=&s3=MLE&sdate=&edate=&downdate=&trans_port= |title = 馬爾地夫機票 |publisher = startravel}}</ref>
