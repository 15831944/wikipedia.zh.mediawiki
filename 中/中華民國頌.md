《'''中華民國頌'''》<ref>{{youtube|8KJHDlns1Ow|中華民國頌}}.lamourdeDieu.2008-10-12</ref>是由[[音樂人|音樂人]]兼[[導演|導演]][[劉家昌|劉家昌]]所創作，歌詞歌頌了[[中國歷史|中國歷史]]與壯麗山川。最早的音樂專輯版本由[[費玉清|費玉清]]所唱，收錄在其1978年12月1日發表的《[[凝望_(專輯)|凝望]]》音樂專輯中。此曲也曾在[[香港|香港]][[橙天嘉禾|嘉禾公司]]的[[電影|電影]]《[[黃飛鴻之二：男兒當自強|黃飛鴻之二：男兒當自強]]》中作為[[背景音樂|背景音樂]]出現。

== 開播曲歷史 ==
1980年12月至1981年1月5日，[[中國電視公司|中國電視公司]]於每日中午播放[[中華民國國歌|中華民國國歌]]之後與每日晚間節目正式開始之前播放的「開播曲」改為[[費玉清|費玉清]]（當時藝名為費玉青）主唱的《中華民國頌》，取代原開播曲《[[美好的今天|美好的今天]]》。

1981年1月6日，中視主控室接獲指示，停用已播了一個多月的《中華民國頌》，恢復以《美好的今天》為開播曲；原因是中視主控室在該日接獲指示稱，由於劉家昌在[[美國|美國]]宣稱《中華民國頌》是他拍攝的電影《[[揹國旗的人|揹國旗的人]]》的主題曲，故應停播《中華民國頌》以免違規。<ref>本報訊，〈「中華民國頌」突被撤下 中視開播曲恢復舊調〉，《民生報》1981年1月7日第9版。</ref>對此，1981年1月6日，中視對外表示，他們早先並不知道《中華民國頌》是《揹國旗的人》的主題曲，所以才把《中華民國頌》選為開播曲；但是現在他們知道這是劉家昌為了宣傳《揹國旗的人》而打歌，所以立刻決定恢復以《美好的今天》為開播曲。<ref>台北訊，〈中視昨換開播曲 不再替人打歌了〉，《[[聯合報|聯合報]]》1981年1月7日第3版。</ref>費玉清得知中視停播《中華民國頌》，只回應「真可惜」；中視節目部對外表示，綜藝節目裡仍然能唱《中華民國頌》。<ref>金琳，〈中視突換開播曲 費玉青感到尷尬〉，《民生報》1981年1月8日第9版。</ref>

1987年12月時，中視已改用《中華民國頌》為開播曲。<ref>舞伶，〈費玉清慘遭胡瓜修理？〉，《[[掃描線周刊|掃描線周刊]]》第103期（[[中視文化公司|中視文化公司]]1987年12月18日出版）第24頁。</ref>

1988年11月，由於劉家昌曾經要求中視支付[[版稅|版稅]]，中視主動改以上檔的國語連續劇主題曲為開播曲與收播曲，不再以《美好的今天》與《[[晚安曲|晚安曲]]》分別作為開播曲與收播曲，以避開[[版權|版權]]爭議，同時兼收廣告之效<ref>熊迺康 台北報導，〈中視‘晚安曲’ 不再道晚安〉，《[[聯合晚報|聯合晚報]]》1988年11月3日第5版。</ref>；可見中視在此之前已不使用《中華民國頌》為開播曲。

《中华民国颂》的歌詞很短，一共四句。歌词第一句歌頌了[[中國大陆|中國大陆]]的壯麗山川：一望无垠的青海大草原和绵延至天边的喜瑪拉雅山。第二句讲[[中国|中国]]有五千年的历史，先辈在不畏困难险阻在这里建设自己的家园。第三句和第四句抒发感慨：中华民国（中华民族）能够经得起任何考验，只要黃河和長江的水不断流，中华民国（中华民族）将世世代代永远延续下去。

== 政治敏感及中華民族頌 ==
<!--「中華民族頌」重定向至此章節，故請勿隨意修改該章節標題-->
由於[[中華人民共和國政府|中華人民共和國政府]]一方認為，[[中華民國政府|中華民國政府]]於1949年撤退到臺灣之後已經名存實亡。所以歌手在[[中國大陸|中國大陸]]及[[港澳地區|港澳地區]]演唱此歌曲時，為了避免觸及敏感[[政治|政治]]問題，常會將歌名和歌詞中的「中華民國」改為「[[中華民族|中華民族]]」，歌词中除了「中華民國」改成「中華民族」之外，其他完全不变，即《中華民族頌》。例如2008年劉家昌在[[香港|香港]]的個人[[演唱會|演唱會]]中，將歌詞中的「中華民國」改成「中華民族」<ref>{{youtube|g-rR-a5MpqA|劉家昌要選中華民國總統，經考驗，「中華民國頌」怎麼沒直到永遠？}}.care4here.2011-08-20</ref>。

[[香港|香港]]歌手[[張明敏|張明敏]]發表的最初專輯版本歌曲中的歌詞就是將「中華民國」改成「中華民族」，其演唱的此歌曲的名稱常被稱作《中華民族》或《中華民族頌》，但是他從來沒有演唱過原版的《中華民國頌》的歌詞。

2019年1月10日，台灣的韓國服飾業者吳茈仁說，當她看到[[台灣獨立運動|台灣獨立運動]]大老[[史明|史明]][[元旦|元旦]]出來升旗、揮舞[[中華民國國旗|中華民國國旗]]時，她就知道《中華民國頌》末句歌詞「中華民國，中華民國，千秋萬世，直到永遠」，[[台灣獨立|台灣獨立]]只是[[南柯太守傳|南柯一夢]]<ref>{{cite news|author=吳茈仁|title=民進黨應廢台獨黨綱|url=https://www.dopost.com/articles-4/20190109-3|language=zh-tw|date=2019-01-10|newspaper=[[讀報|讀報]]|accessdate=2019-02-27}}</ref>。

== 被[[惡搞|惡搞]] ==
[[太陽花學運|反反服貿抗議]]之中，[[王炳忠|王炳忠]]因在2014年4月1日的講話之中演唱了《中華民國頌》<ref>{{youtube|LXT4M50YFVc|0401中華民國頌 by新頭殼newtalk}}</ref>，但是由於其演唱之時跑調走音十分嚴重，尤其到了最後“千秋萬世 直到永遠”的時候，嗓子已經筋疲力盡。導致衆多臺灣網友稱爲“中華破音頌”<ref>{{youtube|mA4fr9-QaSE|東森新聞唱破音遭調侃！ 王炳忠再開嗓雪恥}}</ref>，同時也有臺灣網友將王炳忠演唱的部分與[[Imagine_Dragons|Imagine Dragons]]演唱的《It's Time》結合在一起[[惡搞|惡搞]]爲《It's 腫花皿鍋》<ref>{{youtube|S2px1zB7Nsk|王炳忠- "It's 腫花皿鍋" ft. Imagine Dragons}}</ref>，甚至亦有人特意將其演唱的跑調走音部分截取下來做爲手機鈴音使用<ref>{{youtube|WGqTzc7GGW8|東森新聞王炳忠「破音頌」 暴紅 KUSO鈴聲高人氣}}</ref>。
