'''中国人民解放军北戴河疗养院'''，位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[北戴河区|北戴河区]]，隶属[[郑州联勤保障中心|郑州联勤保障中心]]。

==简介==
'''[[中国人民解放军北京军区|中国人民解放军北京军区]]北戴河疗养院'''始建于1948年<ref name=shen/>。2005年10月，由'''中国人民解放军北京军区北戴河疗养院'''（'''中国人民解放军第二八一医院'''）、'''[[中国人民解放军海军|中国人民解放军海军]]北戴河疗养院'''、'''[[中国人民解放军空军|中国人民解放军空军]]北戴河疗养院'''（'''中国人民解放军第一七八医院'''）三所团级疗养院整合组建为新的'''中国人民解放军北京军区北戴河疗养院'''，设有两个临床部，担负驻秦皇岛地区体系单位官兵、职工、家属的医疗卫勤保障任务<ref>关世奎、孟德仲、张福，不断提高疗治服务水平、努力提升卫勤保障能力——北京军区北戴河疗养院为兵服务工作剪影，解放军医院管理杂志2014(8)</ref><ref name=qianjin>陈洪伟、张志强，前进中的北京军区北戴河疗养院，人民军医2013(6)</ref><ref name=ming>{{Cite web |url =  http://news.163.com/14/1121/11/ABIPMBLK00014AED.html|title = 北戴河一医院发生命案7死1伤 |author =  |date = 2014-11-21 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。其中，原北京军区北戴河疗养院改为北京军区北戴河疗养院第一疗养区，位于西海滩路4号；原海军北戴河疗养院改为第二疗养区，位于金山嘴路10号；原空军北戴河疗养院改为第三疗养区<ref>张法胜、 王宏伟、罗永起，沟通在特勤疗养服务保障工作中的现实意义，中华医学会航海医学分会航海康复医学专业委员会学术会议，2014年</ref>，位于驼峰路1号。

该疗养院是一所集医疗、疗养、体检于一体的大型综合型疗养院<ref name=gfb/>。截至2013年，该疗养院拥有11个营区、5个海水浴场，总占地面积1634亩；展开疗养床位1100多张，医疗床位500多张；拥有卫生技术人员700多名（包括高中级技术职务人员400多名）。“[[十一五规划|十一五]]”以来直到2013年，疗养院先后获军队医疗成果二等奖2项、三等奖16项，在国内核心期刊发表1000余篇论文，中标全军“[[十二五规划|十二五]]”计划课题3项<ref name=qianjin/>。急救中心2000年被北京军区确定为“北戴河急救中心”；肾脏病科2006年被[[中国人民解放军总后勤部卫生部|中国人民解放军总后勤部卫生部]]确定为“全军肾脏病中西结合治疗中心”，主任周柱亮享受[[政府特殊津贴|政府特殊津贴]]，被北京军区表彰为“育才有功专家”；康复医学科2007年被[[国家中医药管理局|国家中医药管理局]]和中国人民解放军总后勤部卫生部确定为“军队重点建设专科”<ref name=gfb>{{Cite web |url = http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536804.htm |title =北京军区北戴河疗养院  |author =  |date = 2014-09-12 |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref>。

2014年11月20日，在该疗养院[[2014年北戴河医院砍杀案|发生一起杀人案]]，造成7死1伤<ref name=ming/>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，该疗养院由北京军区转隶[[郑州联勤保障中心|郑州联勤保障中心]]<ref>{{Cite web |url = http://www.81.cn/kj/2017-08/30/content_7736572.htm |title = 驻训官兵的体检服务可以量身订制了！你可不要错过哦 |author =  |date =  2017-08-30|publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。此后称'''中国人民解放军北戴河疗养院'''。
