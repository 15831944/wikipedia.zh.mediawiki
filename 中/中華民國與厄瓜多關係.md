'''中華民國與厄瓜多關係'''是指[[中華民國|中華民國]]與[[厄瓜多共和國|厄瓜多共和國]]之間的關係。1946－1971年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，[[中華民國政府|中華民國政府]]於厄瓜多[[首都|首都]][[基多|基多]]設立[[實質大使館|具大使館性質]]的代表機構。

1946年1月6日，中華民國與厄瓜多建立公使級[[外交#外交上的承認|外交關係]]。

1947年7月11日，派駐厄瓜多[[特命全權公使|公使]]（由[[中国驻哥伦比亚大使列表|駐哥倫比亞公使]]兼任。1955年，兼使改為專使）。

1948年1月，於首都[[基多|基多]]設立'''中華民國駐厄瓜多共和國[[外交代表機構#公使馆|公使館]]'''。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1950年11月30日，厄瓜多在[[聯合國安全理事會|聯合國安全理事會]]上否決中共代表[[伍修權|伍修權]]等提出之「美國侵略台灣案」<ref>{{cite conference |title=聯合國安全理事會正式紀錄 第七十二號 第五三0次會議 |date=1950-11-30|publisher=[[聯合國安全理事會|聯合國安全理事會]] |location=[[紐約|紐約]]發拉星草場 |url = http://www.un.org/en/ga/search/view_doc.asp?symbol=S/PV.530&referer=/english/&Lang=C |accessdate=2015-08-24 |language=zh-hant}}</ref>。

1957年1月15日，升格為大使級外交關係。7月，公使館升格為'''中華民國駐厄瓜多共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。<ref name="86年外交年鑑"/>

1971年10月25日，厄瓜多投票支持[[中華人民共和國|中華人民共和國]]取代中華民國的「[[中國與聯合國#代表權問題|中國席位]]」，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。11月17日，厄瓜多與中華民國斷交。<ref name="駐外"/>（12日，先行關閉大使館<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>）

===代表機構===
{{see also|中華民國駐外機構列表|駐臺灣外交機構列表}}

1977年4月12日，於首都基多設立[[實質大使館|具大使館性質]]的'''中華民國[[台北经济文化代表处|駐厄瓜多商務處]]'''（[[西班牙文|西班牙文]]：{{lang|es|Oficina Comercial de La República de China (Taiwán)}}），是首個在非[[中華民國外交#邦交國|邦交國]]使用中華民國國號的[[中華民國駐外機構列表|駐外機構]]{{noteTag|截至2018年7月，駐非邦交國機構使用中華民國國號的有：斐濟；非邦交國駐臺灣機構使用駐「華」名稱的有：奈及利亞、阿曼。}}。5月1日，台灣領務代理處則更名為'''駐惠夜基分處'''，之後復更名為'''駐惠夜基辦事處'''（於1998年10月31日關閉）。<ref name="外交年鑑"/><ref name="86年外交年鑑"/><ref name="駐外">{{Cite web |url = http://www.taiwanembassy.org/ec/post/2393.html |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref>

1983年3月25日，厄瓜多於首都[[台北市|台北]]設立同等性質的商務處，但於1988年7月因財政困難而關閉。2005年2月，厄瓜多派員赴台辦理恢復設處事宜，但該員返國後車禍身亡，且{{link-en|2006年厄瓜多大選|Ecuadorian general election, 2006|厄瓜多政權輪替}}以及經費問題，擱置至今。<ref name="外交年鑑"/><ref name="94年外交年鑑"/>

2017年6月27日，[[中華民國外交部|中華民國外交部]]表示受{{link-en|厄瓜多政府|Politics of Ecuador}}片面要求，原中華民國駐厄瓜多商務處已更名為'''臺北駐厄瓜多商務處'''（西班牙文：{{lang|es|Oficina Comercial de Taipei}}）。<ref>[http://www.cna.com.tw/news/firstnews/201706275008-1.aspx 駐厄瓜多代表處更名 中華民國改台北].中央社</ref><ref name="駐外館處">{{Cite web |url = http://www.mofa.gov.tw/OverseasOffice_Content.aspx?n=6357834932B83C83&sms=5A961ED2E4BA25E8&s=106241E966C563C0&os=9223A12B5B31CB37 |title = 駐外館處 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>

===貿易===
2018年，厄瓜多是中華民國第75大貿易夥伴、第115大[[進口|進口]]夥伴、第59大[[出口贸易|出口]]夥伴。出口至厄瓜多的金額為1億6,906萬1,300[[美元|美元]]，年增18.934%；自厄瓜多進口的金額為759萬1,949美元，年減39.205%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1億6,146萬9,351美元，年增24.533%。2017年，厄瓜多是中華民國第77大貿易夥伴、第113大進口夥伴、第60大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===

1978年12月6日，恢復派駐厄瓜多[[農業技術|農業技術]]團，提供[[稻米|稻米]]、[[蔬菜|蔬菜]]、[[猪生产学|養豬]]、[[蘭科|蘭花]]、小型[[農業機械|農機]]運用、[[資訊|資訊]]教育、[[水產養殖|水產養殖]]、[[食品加工|食品加工]]、[[穀物|穀物]][[生產|生產]]與[[行銷|行銷]]等方面合作。因應國際合作趨勢和情勢變遷、多邊和雙邊的合作需求、以成果及符合民生為導向，技術合作業務自2013年12月31日完成轉型，並自2014年起採行計畫經理人新制與厄瓜多技術合作。目前執行及推動的計畫包括：[[竹|竹]]產業合作、[[牡蠣|牡蠣]]繁養殖、農業食品加工、蔬菜產銷能力提升、[[鞋|製鞋]][[工業園區|工業園區]]等。<ref name="駐外"/><ref name="104年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2014年3月20日，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）「2014年[[拉丁美洲|中南美洲]]利基產業拓銷團」率領53家廠商前往第1大城[[瓜亞基爾|惠夜基]]，與250家買主約500多人進行貿易洽談。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=631735&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2014年3月20日 |title = 暌違2年貿協中南美拓銷團再訪厄瓜多締造近千萬美元商機 |publisher = 經貿透視網}}</ref>

2015年7月2日，[[中華民國外交部|中華民國外交部]]委託外貿協會首次組團參加於首都[[基多|基多]]展覽中心舉行的「2015年厄瓜多國際[[鞋類|鞋類]]及組件展」，為期3天。臺灣是唯一以國家館參展的參展國。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=668494&url=/NewsSearch.asp |author = 姚碧蓮 |date = 2015年7月8日 |title = 搶攻中南美洲商機，外貿協會首度率台灣廠商參加厄瓜多國際鞋類及組件展 |publisher = 經貿透視網}}</ref>

2017年9月13日，臺北駐厄瓜多商務處以設立「臺灣形象館」方式參加厄瓜多基多商會（[[西班牙文|西班牙文]]：{{lang|es|Cámara Comercial de Quito}}）首次舉辦的「[[B2B|B2B]]厄瓜多進出口、貿易洽談、座談及展覽會」。<ref name="106年外交年鑑"/>

===僑民===
{{see also|臺灣裔{{!}}海外臺僑|海外華人|{{link-es|厄瓜多華人|Inmigración china en Ecuador}}}}
截至2017年，在厄瓜多的台僑約800人。主要僑居地在[[基多|基多]]、[[瓜亞基爾|惠夜基]]、[[馬查拉|馬恰拉]]等城市。<ref name="106年外交年鑑"/>先後設立的僑團有：[[基多|厄京]]華僑公會、惠夜基自由華僑協會、馬恰拉自由華僑協會、[[孫中山|孫中山]]協會、厄瓜多台灣商會、厄瓜多惠夜基台灣商會、厄瓜多台灣同鄉會、[[孔子|孔]][[孟子|孟]]文化交流協會、中華民國基金會、厄瓜多福爾摩沙基金會、厄瓜多台灣民主進步協會、中厄文化中心等。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2007/html/08-1-6.htm |title = 《中華民國96年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>2002年，中華民國駐厄瓜多商務處協助僑界成立全僑民主和平聯盟厄瓜多支盟。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__06.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

===宗教===
{{see also|台灣宗教|{{link-en|厄瓜多宗教|Religion in Ecuador}}}}
1995年10月，台灣高雄市[[元亨寺|元亨寺]]菩妙老和尚在惠夜基設立厄瓜多第一個[[佛堂|佛堂]]。2000年5月，在第3大城[[昆卡_(厄瓜多尔)|昆卡]]建造元亨寺精舍（靜修[[道場|道場]]）。至今，在基多、惠夜基及第5大城[[馬查拉|馬查拉]]均已成立元亨國際學佛會分會。<ref name="生活資訊"/>

===學術===
2002年1月，[[國際合作發展基金會|國際合作發展基金會]]（國合會）邀請厄瓜多工商部派員參與「科技產業政策與管理研習班」。厄瓜多原擬派主管[[工業化|工業化]][[次長|次長]]參加，但因[[中華人民共和國|中華人民共和國]]駐厄瓜多[[外交代表機構#大使馆|大使館]]向該國外交部及工商部施壓，以致次長無法成行。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=4D1E240AB4C7F7AF&sms=10AEB7B38956F518&s=3F0C9D3263939F42 |author = 公眾外交協調會 |date = 2002年5月2日 |title = 中國大陸阻撓我國際空間事例（91年1月至3月事例）|publisher = 中華民國外交部}}</ref>

中華民國政府提供厄瓜多學生台灣[[獎學金|獎學金]]、[[中華民國國語|華語文]]獎學金，國合會也提供高等人力培訓獎學金，至2016年獲頒獎學金學生有70人。2015年，厄瓜多成立留台[[校友会|校友會]]。<ref name="駐外"/><ref name="106年外交年鑑"/>
