'''中華民國與瑞士關係'''是指[[中華民國|中華民國]]與[[瑞士聯邦|瑞士聯邦]]之間的關係。1913－1950年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交直至1970年代後，在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，[[瑞士|瑞士]]當時尚未加入，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
{{see also|中華民國外交|{{link-en|瑞士外交|Foreign relations of Switzerland}}}}
{{see also|中国驻瑞士大使列表{{!}}中華民國駐瑞士外交官列表|瑞士驻华大使列表{{!}}瑞士駐中華民國外交官列表}}
1911年[[辛亥革命|辛亥革命]]後，瑞士和中國的經貿關係受制於[[大日本帝国|日本]]在中國的霸權、以及瑞士對歐洲[[列強|列強]]的附庸；這樣的情況一直持續到[[二戰|二戰]]結束，瑞士和其它歐洲國家放棄了殖民貿易、並將注意力從[[食物|食品]]和[[紡織品|紡織品]]貿易轉向[[手錶|手錶]]和[[機械|機械]]貿易<ref name="cw">{{cite conference|publisher=chinaweb.de |url=http://www.chinaweb.de/china_politik/china_schweiz/beziehungen_wirtschaft_china_schweiz.htm |archiveurl=https://web.archive.org/web/20150216133611/http://www.chinaweb.de/china_politik/china_schweiz/beziehungen_wirtschaft_china_schweiz.htm |archivedate=2015-02-16 |author=Annette Ryser |title=Sino-Swiss Economic Relations |date=2007年2月 |deadurl=yes }}</ref>。

1913年10月8日，瑞士[[外交#外交上的承認|外交承認]]中華民國。

1918年6月13日，建立公使級外交關係。

1919年1月，於首都[[伯恩|伯恩]]設立'''中華民國駐瑞士聯邦[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1930年代，[[中國抗日戰爭|中國抗日戰爭]]爆發後，两名瑞士[[传教士|传教士]]遭到[[日軍|日軍]]飞机轰炸而喪生；当时的瑞士駐[[广州|广州]]名誉[[领事|领事]]受[[英国|英]]、[[美国|美]]、[[荷兰|荷兰]]等国家所托，負責保障这些国家在广州的[[侨民|侨民]]的利益<ref name=Guang>{{cite web|title=瑞士驻广州总领事馆 |url=http://www.switzerland65china.com/cn/swissinsititution-13.html |accessdate=2015-02-19 |archiveurl=https://web.archive.org/web/20150219015234/http://www.switzerland65china.com/cn/swissinsititution-13.html |archivedate=2015-02-19 |deadurl=yes }}</ref>。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國維持短暫的邦交關係。

1950年1月20日，與中華民國斷交、關閉大使館<ref name="103年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/09.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>。

1999年9月21日，[[九二一大地震|台灣發生大地震]]，瑞士派出40人的搜救隊抵台救災。

瑞士尚未加入[[歐洲聯盟|歐洲聯盟]]，但有簽署《[[申根公約|申根公約]]》，因此實行與[[申根區|申根區]]相同的[[簽證|簽證]]政策。歐洲申根區給予持有載明[[中華民國國民身份證|身份證]]字號的[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以[[歐洲聯盟簽證政策|申根區免簽證]]入境，停留日數與申根區合併計算，每6個月期間內總計可停留90天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-71-9a9e2-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_FAQ.aspx?c=AD6908DFDDB62656&n=C1B19EF0ACDD3C91&sms=867F5398A49F758A |title = 歐盟免申根簽證常見問答集 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

[[瑞士人口|瑞士公民]]持有[[瑞士護照|瑞士護照]]者則可用[[歐洲自由貿易聯盟|歐洲自由貿易聯盟]]的身分以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將瑞士歸類為'''灰色警示'''：提醒注意（2017年5月10日）。<ref>{{Cite web |url =https://www.boca.gov.tw/sp-trwa-content-339-1d052-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
[[經濟部國際貿易局|經濟部國際貿易局]]於首都[[伯恩|伯恩]]設立'''駐瑞士代表處經濟組'''。2002年1月1日，中華民國成為[[世界貿易組織|世界貿易組織]]（WTO）會員，於第2大城[[日內瓦|日內瓦]]設立'''中華民國（[[臺澎金馬個別關稅領域|臺澎金馬個別關稅領域]]）常駐世界貿易組織代表團'''（英文：Permanent Mission of the Separate Customs Territory of Taiwan, Penghu, Kinmen and Matsu to the World Trade Organization），常任代表為[[朱敬一|朱敬一]]。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/OverseasOffice_Content.aspx?n=6357834932B83C83&sms=5A961ED2E4BA25E8&s=31218F193CB4FD6E&os=163B8937FBE0F186 |title = 駐外館處 |publisher = 中華民國外交部}}</ref>

2017年，瑞士是中華民國第27大貿易夥伴、第25大[[進口|進口]]夥伴、第35大[[出口贸易|出口]]夥伴。出口至瑞士的金額為5億3,406萬4,852[[美元|美元]]，年增17.263%；自瑞士進口的金額為15億6,996萬4,045美元，年增0.782%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）10億3,589萬9,193美元，年減6.027%。※2016年，瑞士是中華民國第27大貿易夥伴、第24大進口夥伴、第36大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資===
根據經濟部投資審議委員會統計，截至2015年1月，[[臺灣商人|臺商]]對瑞士的總投資金額為1億3,393萬美元，計有23件；瑞商在台灣總投資金額為8億0,296萬美元，計有281件。<ref name="投資環境簡介"/>

在瑞士有宏碁、[[華碩電腦|華碩電腦]]、[[台達電子|台達電子]]、[[長榮海運|長榮海運]]及[[百略醫學科技|百略醫學科技]]等台商直接設立營業處所並派有員工經營，另有少數[[高科技|高科技]]集團則由當地瑞商[[代理|代為經銷]]。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/ch/pdf/104-49Switzerland.pdf |date = 2015年8月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

目前已成立瑞士台灣商會、瑞士中華學人暨專業學會等組織。<ref name="投資環境簡介"/>
