'''中國－巴拿馬關係'''，是指[[中国|中国]]与[[巴拿马|巴拿马]]的双边关系，现时是指[[中華人民共和國|中華人民共和國]]與[[巴拿马共和国|巴拿马共和国]]的雙邊關係。

巴拿马自从清末就与中国建立了外交关系，随后的中华民国延续了这一外交关系。1949年中华人民共和国成立后，巴拿马仍与[[中华民国|中华民国]]保持外交关系。尽管巴拿马政府曾提出与中华人民共和国建交，但其直到2017年才与中华人民共和国建交，并同时与中华民国断交。

== 清朝 ==
[[中國|中國]]與[[巴拿馬|巴拿馬]]雙方關係可追溯至[[清朝|清朝]]，雙方於1910年1月16日（清[[宣統|宣統]]元年十二月六日）建立領事級外交關係。1910年5月9日（宣统二年四月一日），中国驻巴拿马总领事馆建立，首任总领事为[[欧阳庚|欧阳庚]]。<ref>{{cite book |title=中国和拉丁美洲关系简史 |authors = 沙丁、杨典求、焦震衡、孙桂荣 |publisher=河南人民出版社 |date=1986年6月 |edition = 第1版 |quote=宣统二年四月一日,欧阳庚偕同通译、书记等官员抵达巴拿马, "即日正式开办总领事馆,启用官防任事" }}</ref>

== 中华民国 ==
{{Main|中華民國與巴拿馬關係}}

1912年中華民國成立，承續两国關係。1954年5月1日，双方的公使馆升格為大使館等級。

2017年6月12日晚間8時（[[UTC−05:00|UTC−05:00]]），[[巴拿马|巴拿马]]宣告与中华人民共和国政府建交，随即[[台北时间|台北时间]]第二天，中華民國正式宣布与之断交。

== 中华人民共和国 ==
=== 建交前 ===
在中華人民共和國成立後至2017年，巴拿馬一直維持與[[中華民國|中華民國]]的外交關係，基於[[一個中國|一個中國]]的原則，中華人民共和國長期不與巴拿馬建交，但於1996年在巴拿馬設立[[中华人民共和国驻巴拿马贸易代表处|貿易代表處]]和貿易發展辦事處經商室。
[[File:1964-03_1964年_中国抗议美国入侵巴拿马.jpg|thumb]]
