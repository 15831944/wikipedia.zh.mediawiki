'''中華民國與安哥拉關係'''是指[[中華民國|中華民國]]與[[安哥拉共和國|安哥拉共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對安哥拉的相關事務由'''{{link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-166-36b2e-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
[[File:Voting_res_2758.png|thumb]]，[[安哥拉|安哥拉]]當時尚未獨立，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]

===外交===
{{see also|中華民國外交|{{link-en|安哥拉外交|Foreign relations of Angola}}}}
1992年3月，兩國簽署《聯合聲明》及《特別協定》。9月，在首都[[羅安達|羅安達]]設立[[實質大使館|具大使館性質]]的'''中華民國駐安哥拉[[臺北經濟文化代表處|特別代表團]]'''。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1998/section_3/page3-1-3.htm |title = 《中華民國87年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>目前已關閉。

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|安哥拉簽證政策|Visa policy of Angola}}|{{link-en|安哥拉公民簽證要求|Visa requirements for Angolan citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往安哥拉駐[[新加坡|新加坡]][[外交代表機構#大使馆|大使館]]申請簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-166-36b2e-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>入境須出示[[黃熱病|黃熱病]]的[[疫苗接種|疫苗接種]]證明（黃皮書）。<ref name="國外旅遊警示分級表"/>

持{{link-en|安哥拉護照|Angolan passport}}的{{link-en|安哥拉人口|Demographics of Angola|安哥拉公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將安哥拉歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2017年2月6日）；<ref name="國外旅遊警示分級表">{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-103-7e643-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將安哥拉歸類為'''第二級警示'''：對當地採取加強防護（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=4B84885EB6795AC7 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
2012年6月16日，[[經濟部國際貿易局|經濟部國際貿易局]]委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）與54家台灣業者執行「游龍專案」，籌組「2012年[[非洲国家和地区列表|非洲南撒哈拉]]拓銷團」。前往[[安哥拉|安哥拉]]、[[南非|南非]]、[[坦尚尼亞|坦尚尼亞]]及[[烏干達|烏干達]]，是台灣歷來拓銷非洲最大團。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=585707&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2012年6月14日 |title = 貿協游龍專案出擊非洲，史上最大非洲拓銷團開拔！ |publisher = 經貿透視網}}</ref>

2017年，安哥拉是中華民國第36大貿易夥伴、第27大[[進口|進口]]夥伴、第101大[[出口贸易|出口]]夥伴，在[[非洲国家和地区列表|非洲經濟體]]排名第2、1、11。出口至安哥拉的金額為2,807萬8,174[[美元|美元]]，年增219.958%；自安哥拉進口的金額為14億8,125萬1,626美元，年增12.964%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）14億5,317萬3,452美元，年增11.569%。※2016年，安哥拉是中華民國第34大貿易夥伴、第26大進口夥伴、第124大出口夥伴，是中華民國在非洲的最大貿易夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
