'''中華民國與智利關係'''是指[[中華民國|中華民國]]與[[智利共和國|智利共和國]]之間的關係。1915－1971年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[智利|智利]]。]]
{{see also|中華民國外交|{{link-en|智利外交|Foreign relations of Chile}}}}
{{see also|中国驻智利大使列表{{!}}中華民國駐智利外交官列表|智利驻华大使列表{{!}}智利駐中華民國外交官列表}}
1915年2月18日，中華民國與智利簽署《通商及友好[[條約|條約]]》，並建立公使級[[外交#外交上的承認|外交關係]]。<ref name="駐外"/>

1921年2月，中華民國派駐智利[[外交官#代办|代辦]]。

1924年，將代辦升格為[[特命全權公使|公使]]。於首都[[聖地亞哥_(智利)|聖地牙哥]]設立'''中華民國駐智利共和國[[外交代表機構#公使馆|公使館]]'''。

1946年12月3日，建立大使級外交關係，公使館升格為'''中華民國駐智利共和國[[外交代表機構#大使馆|大使館]]'''，<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>並派駐[[特命全權大使|大使]]。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1951年－1961年，中華民國未派駐智利大使。

1971年1月5日，智利與中華民國斷交（於1970年宣布）。<ref name="駐外"/>

1996年3月，智利眾議院對中華人民共和國[[台灣海峽飛彈危機|在台灣海峽試射飛彈]]，威脅台灣即將於3月23日舉行的[[1996年中華民國總統選舉|首次中華民國總統直接選舉]]一事，表達關切並通過決議案予以聲援。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=E96A84A210D9727B |author = 公眾外交協調會 |date = 1996年3月20日 |title = 智利共和國眾議院於本月十三日就中共在台灣東北角及西南角海域內進行飛彈演習 |publisher = 中華民國外交部}}</ref>

===貿易===
智利是中華民國在[[拉丁美洲|拉丁美洲]]第3大貿易夥伴，次於[[巴西|巴西]]及[[墨西哥|墨西哥]]。<ref name="與我國經貿關係"/>[[經濟部國際貿易局|經濟部國際貿易局]]在首都[[聖地亞哥_(智利)|聖地牙哥]]設立'''駐智利代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=215&pid=305863 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2018年，智利是中華民國第34大貿易夥伴、第29大[[進口|進口]]夥伴、第53大[[出口贸易|出口]]夥伴。出口至智利的金額為2億4,648萬3,056[[美元|美元]]，年增1.776%；自智利進口的金額為13億6,777萬6,342美元，年減0.477%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）11億2,129萬3,286美元，年減0.959%。2017年，智利是中華民國第33大貿易夥伴、第29大進口夥伴、第50大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資===
[[臺灣商人|台商]]在智利的實際投資金額與件數統計差異頗大。根據經濟部投資審議委員會統計，首都聖地牙哥地區台商約140家，智利北部[[伊基克|伊基克]][[自由貿易區|自由貿易區]]約70家，其餘各地約35家，總投資金額約4億6,000萬美元；另根據智利官方統計，1974年－2013年，台商總投資金額為1,410萬美元。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/cl/pdf/103-39Chile.pdf |date = 2014年12月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>在伊基克自由貿易區經營的台商，亦掌握及經營鄰近的[[玻利維亞|玻利維亞]]、[[巴拉圭|巴拉圭]]、[[秘魯|秘魯]]、[[厄瓜多|厄瓜多]]、[[阿根廷|阿根廷]]、[[巴西|巴西]]等[[製成品|消費品]][[轉口貿易|轉口貿易]]市場，成為[[南美洲#南美洲国家和地区|南美市場]]重要樞紐。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

主要投資產業：禮品百貨進口[[批發|批發]][[零售|零售]]、[[成衣|成衣]]進口批發零售、[[資訊及通訊科技|資訊通訊產品]]進口、[[鞋|鞋類]]、[[紡織|紡織]][[布|布料]]、[[襪子|織襪]]、[[塑膠|塑膠]]包材、[[成人紙尿片|成人紙尿片]]、[[三聚氰胺-甲醛树脂|美耐皿]]、[[食品加工|食品加工]]、[[航運|海運]][[航空|空運]]承攬、[[旅遊業|旅遊業]]等。[[友訊科技|友訊科技]]、[[昆盈企業|昆盈企業]]、[[冠捷科技|冠捷科技]]等在智利設立南美洲區域營運中心；[[長榮海運|長榮海運]]則在智利設立據點及[[代理|代理]]行。<ref name="投資環境簡介"/>

===組織===

==僑民==
