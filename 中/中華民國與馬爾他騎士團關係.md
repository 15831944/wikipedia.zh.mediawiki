'''中華民國與馬爾他騎士團關係'''是指[[中華民國|中華民國]]與[[馬爾他騎士團|耶路撒冷、羅得島和馬爾他聖約翰主權軍事醫院騎士團]]（通稱'''馬爾他騎士團'''）之間的關係。雙方無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構{{noteTag|一般而言，各國皆由駐[[聖座|教廷]]大使兼任駐該團[[特命全權大使|大使]]。<ref>{{Cite web |url = http://www.mofa.gov.tw/CountryInfo.aspx?CASN=FB01D469347C76A7&n=9C9CC6640661FEBA&sms=26470E539B6FA395&s=CC62364F57537537 |title = 國家相關資訊 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>}}。雙方在[[越南|越南]]、[[保加利亞|保加利亞]]、[[薩爾瓦多|薩爾瓦多]]和[[塞爾維亞|塞爾維亞]]等國均提供[[人道救援|人道救援]]計畫，雖無邦交，但高層曾經互訪。<ref name="可能與台建交"/>

===外交===
[[File:Voting_res_2758.png|thumb]]，[[馬爾他騎士團|馬爾他騎士團]]始於1994年成為[[聯合國大會觀察員列表|觀察員實體]]，因此不能對當時[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
{{see also|中華民國外交|马耳他骑士团建交列表{{!}}馬爾他騎士團外交}}
2012年9月9日，馬爾他騎士團總理兼外交部長{{link-en|馬哲立|Jean-Pierre Mazery}}（Jean-Pierre Mazery）應[[中華民國外交部|中華民國外交部]]邀請率團抵台訪問6天。期間會見[[中華民國總統|總統]][[馬英九|馬英九]]、[[中華民國外交部#歷任首長|外交部長]][[楊進添|楊進添]]並晚宴款待、[[中華民國衛生福利部|行政院衛生署]]署長[[邱文達|邱文達]]、[[行政院大陸委員會|行政院大陸委員會]]副主任委員[[劉德勳|劉德勳]]、[[天主教臺灣地區主教團|天主教臺灣地區主教團]]主席與[[台北總教區|台北總教區]]總主教[[洪山川|洪山川]]、財團法人國際合作發展基金會秘書長[[陶文隆|陶文隆]]及[[教廷駐華大使列表|教廷駐華代辦]][[陸思道|陸思道]]，接受[[輔仁大學|輔仁大學]]頒贈[[名誉学位|榮譽博士學位]]並發表演講。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=5E24116A3DB71515 |author = 公眾外交協調會 |date = 2012年9月7日 |title = 馬爾他騎士團總理兼外長馬哲立閣下訪華 |publisher = 中華民國外交部}}</ref>

2014年10月，中華民國外交部長[[林永樂|林永樂]]訪問馬爾他騎士團。<ref>{{Cite web |url = https://www.orderofmalta.int/2015/11/09/first-visit-of-grand-master-to-the-republic-of-china-taiwan/ |date = 09/11/2015 |title = First visit of a Grand Master to the Republic of China (Taiwan) |publisher = orderofmalta.int}}</ref>

2015年11月4日，{{link-en|馬爾他騎士團元首與大教長列表|List of Princes and Grand Masters of the Sovereign Military Order of Malta|馬爾他騎士團元首與大教長}}[[马修·费斯汀|費斯汀]]（Fra′ Matthew Festing）、衛生部長蒙特貝（Rochefoucauld-Montbel）應[[中華民國政府|中華民國政府]]邀請率團抵台訪問5天。期間會見總統馬英九並相互[[勳章|贈勳]]與款宴、外交部長林永樂並款宴，接受輔仁大學頒贈名譽[[哲學博士|哲學博士]]學位並發表演講。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=8742DCE7A2A28761&s=40B00EC083426D28 |author = 公眾外交協調會 |date = 2015年11月4日 |title = 馬爾他騎士團元首費斯汀大教長閣下應邀率團訪華 |publisher = 中華民國外交部}}</ref>這是該團元首首次訪台。<ref name="可能與台建交"/>12月2日，費斯汀於[[馬爾他宮|該團總部]]頒贈{{link-en|馬爾他騎士團大綬大十字功績勳章|Order pro Merito Melitensi|大綬大十字功績勳章}}予中華民國駐[[聖座|教廷]]大使[[王豫元|王豫元]]，是該團首次頒贈此等級之勳章予無邦交國家的官員。<ref>{{Cite web |url = http://www.roc-taiwan.org/va/post/650.html |title = 王豫元大使獲馬爾他騎士團贈大綬勳章 |publisher = 中華民國駐外單位聯合網站}}</ref>

2017年3月14日，《[[富比世|富比世]]》雜誌刊登[[專欄作家|專欄作家]]詹寧斯（Ralph Jennings）專文指出，[[臺灣戰後時期|台灣]]與[[中華人民共和國|中國]]對峙70年，外交空間長期遭到壓縮，目前僅有[[中華民國外交#邦交国|21個邦交國]]{{noteTag|最新為17個。}}，但歐洲馬爾他騎士團將有可能成為台灣新盟友。文中指出，台灣與馬爾他騎士團關係逐年穩固，雖然沒有官方承諾，但馬爾他騎士團對外聯絡主任狄羅比亞特（Eugenio Ajroldi di Robbiate）曾向《富比世》透露，「基於雙方的合作，我們有信心能與中華民國發展出更深厚的關係」。狄羅比亞特雖不願明說該團是否將和中華民國簽署正式[[外交#外交上的承認|建交]]文件，但該團並未與中華人民共和國建交，且近年來雙邊高層積極互訪，確實有可能成為新的外交盟友。<ref name="可能與台建交">{{Cite web |url = http://news.ltn.com.tw/news/focus/paper/1085901 |author = 呂伊萱 |date = 2017年3月15日 |title = 富比世︰馬爾他騎士團 可能與台建交 |publisher = 《[[自由時報|自由時報]]》}}</ref><ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20170314/1075881/ |author = 桂家齊 |date = 2017年3月14日 |title = 台灣有新邦交國？《富比士》：馬爾他騎士團與台關係密切 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》}}</ref>中華民國外交部表示，未來與馬爾他騎士團將在現有友好合作基礎上，積極加強國際[[人道援助|人道援助]]夥伴關係，並深化彼此在國際人道援助及慈善醫療領域合作。<ref>{{Cite web |url = http://www.chinatimes.com/realtimenews/20170314005099-260407 |author = 陳建瑜 |date = 2017年3月14日 |title = 馬爾他騎士團與台建交？外交部：深化合作 |publisher = 《[[中時電子報|中時電子報]]》}}</ref>對於建交消息則沒有著墨。

2018年2月5日，中華民國[[立法院外交及國防委員會|立法院外交及國防委員會]]組團參訪馬爾他騎士團總部，由該團駐[[保加利亞|保加利亞]][[特命全權大使|大使]]朱柯里（Camillo Zuccoli）接待。<ref>{{Cite web |url = https://money.udn.com/money/story/5641/2969771 |author = 黃雅詩 |date = 2018年2月5日 |title = 立委訪梵 籲教廷關注中國宗教自由 |publisher = 《[[經濟日報_(臺灣)|經濟日報]]》}}</ref>

2018年5月，馬爾他騎士團駐[[聯合國|聯合國]]大使 Marie-Therese Pictet-Althann 在[[世界衛生大會|世界衛生大會]]（WHA）的發言中，肯定台灣的[[医疗卫生|醫療衛生]]實力，並提及台灣和馬爾他騎士團是人道合作夥伴。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/2435361 |author = 呂伊萱 |date = 2018年5月23日 |title = WHA大會24代表挺台 馬爾他騎士團意外獻聲 |publisher = 《自由時報》}}</ref>

馬爾他騎士團人民持有[[歐盟護照|歐盟護照]]者也可以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將義大利歸類為'''灰色警示'''：提醒注意（2018年8月23日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-345-8d407-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將義大利歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年11月6日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=CFB77829C7ABD761 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>
