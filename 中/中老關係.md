'''中老關係'''，是指[[中國|歷史上的中國]]、以至現在的[[中華人民共和國|中華人民共和國]]和[[老撾人民民主共和國|老撾人民民主共和國]]的關係。[[中國|中國]][[國家通訊社|國家通訊社]][[中国新闻网|中国新闻网]]曾經以「山水相连的友好邻邦」來形容中國和老撾兩國關係，又指老挝政府支持[[一个中国|一个中国]]<ref name="Vlad">{{cite news|title=中国同老挝的关系|url=http://www.chinanews.com/gj/zlk/2014/01-15/177_2.shtml|newspaper=中国新闻网}}</ref>，[[国务院直属事业单位|国务院直属事业单位]][[新華社|新華社]]亦形容中国和老挝擁有「深厚的传统友谊」<ref>{{cite web|title=老挝概况|url=http://news.xinhuanet.com/ziliao/2002-06/18/content_445536_2.htm|publisher=新华网}}</ref>。《[[环球时报|环球时报]]》指出，中國[[记者|记者]]走访老挝时獲悉当地官员称該國需要成为中国的“好邻居、好朋友、好[[革命同志|同志]]、好伙伴”<ref name=all>{{cite news|title=老挝要当中国的“四好”邻居 |url=http://world.huanqiu.com/depth_report/2012-12/3378843.html |date=2012-12-14 |archiveurl=https://web.archive.org/web/20130122175105/http://world.huanqiu.com/depth_report/2012-12/3378843.html |archivedate=2013-01-22 |deadurl=yes }}</ref>。[[法國國際廣播電台|法國國際廣播電台]]就報道過中国对老挝的影响深重，引起[[美国|美国]]的警惕与防范，及[[老挝人|老挝人]]對[[華人|華人]]反感<ref>{{cite news|title=中国与老挝之间既需要又仇恨|url=http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20120713-%E4%B8%AD%E5%9B%BD%E4%B8%8E%E8%80%81%E6%8C%9D%E4%B9%8B%E9%97%B4%E6%97%A2%E9%9C%80%E8%A6%81%E5%8F%88%E4%BB%87%E6%81%A8|newspaper=法國國際廣播電台|date=2012-07-13}}</ref>。

中国和老挝的边界長860公里，位於中國[[云南省|云南省]]一帶<ref name=thanks/>。

== 歷史 ==
中國和老撾两国之間的交往始於[[三国時期|三国時期]]，當時老挝近[[云南|云南]]的地区被称為“堂明”；东吴黄武六年（即227年），堂明遣派使者到[[东吴|东吴]]進贡<ref name=thanks/>。[[唐高宗|唐高宗]]时，[[真腊|真腊]]分為南北两国，北部名為[[文单国|文单国]]，国都建於文单城（即今天的[[万象|万象]]）；文单国和[[唐朝|唐朝]]關係緊密且友好，曾經先后四次向唐遣使<ref name=thanks>{{cite web|author=吴平|title=古代云南与老挝的佛教文化交流|url=http://www.mzb.com.cn/html/report/411664-1.htm|publisher=中国民族报社|date=2013-06-14|deadurl=yes|archiveurl=https://web.archive.org/web/20141006085401/http://www.mzb.com.cn/html/report/411664-1.htm|archivedate=2014-10-06}}</ref>。

历史上，中國和老撾兩國边民之间的贸易十分活跃，中国商人會到老挝境内从事贸易，老挝边民亦會到中国；[[明朝|明朝]]時已有中国人移居老撾，[[清朝|清朝]][[雍正|雍正]]时期，[[琅勃拉邦王国|南掌]]（即琅勃拉邦王国，属今老挝）归化中国，成為中国的属国，但琅勃拉邦於1893年成为[[法国|法国]][[保护国|保护国]]，终结與中国的关系<ref>{{cite news|author=费正清|title=清朝12属国：琉球国成冲绳县 暹罗成泰国|url=http://sd.ifeng.com/chinese/shihaigouchen/detail_2013_05/15/802917_0.shtml?_from_ralated|newspaper=凤凰新媒体|date=2013-05-15}}</ref>。18至19世紀時，華僑貿易在老撾經濟上已有相當大的影響力<ref name=book>{{cite book|author=张明|title=中外文化交流史|year=2013|publisher=青苹果数据中心|coauthors=于井尧}}</ref>。華僑和老挝人共同生活，使老挝傳入了大量中國文化，例如不少[[老挝語|老挝語]]用詞的發音和[[汉語|汉語]]有相似之處<ref name=book/>。
[[File:Welcoming_Prince_Souvanna.jpg|thumb]]
中华人民共和国和[[老撾王國|老撾王國]]在1961年4月25日正式建交<ref name="Vlad"/>；1970年代末至80年代中，[[越南|越南]]对老挝的影响日益加深，老挝曾把与越南的关系写进[[宪法|宪法]]，随着[[中越关系|中越关系]]恶化，中國和老撾兩國之間关系变得冷淡，及至[[中越自卫反击战|中越自卫反击战]]時，老挝支持越南，中断与中国的官方贸易<ref name=all/>。後來的1989年，中老兩國关系正常化，得到全面的恢复和发展，自此後兩國在不同领域的交流合作不断深化<ref name="Vlad"/>。2009年，兩國建立全面战略合作伙伴关系<ref name=inter>{{cite news|title=专访中国驻老挝大使关华兵：中老关系进入历史最好时期|url=http://news.xinhuanet.com/2013-09/21/c_125419470.htm|newspaper=新华国际|date=2013-09-21}}</ref>。

== 經貿關係 ==
2014年1月，中国与老挝兩國的双边贸易總额为5.26亿[[美元|美元]]：其中中国到老挝的出口额為3.04亿美元，主要出口商品為[[矿砂|矿砂]]、[[矿渣|矿渣]]及[[矿灰|矿灰]]；老挝到中国的出口额則為2.22亿美元，主要出口商品為[[铜|铜]]與铜制品、[[木|木]]制品、矿砂、矿渣、矿灰、[[谷物|谷物]]及金属制品<ref>{{cite news|title=2014年1月中国与老挝双边贸易额环比下降25.9%|url=http://customs.caexpo.com/data/country/2014/04/14/3620409.html|newspaper=南博网|date=2014-04-14|access-date=2014-10-04|archive-url=https://web.archive.org/web/20150602054542/http://customs.caexpo.com/data/country/2014/04/14/3620409.html|archive-date=2015-06-02|dead-url=yes}}</ref>。

近年來，中國和老撾兩國的官方和民间经贸合作不断扩大<ref name=DNK/>，越来越多中国企业和个人到老挝投资及贸易<ref name=bank>{{cite news|title=老挝首家中老合资银行成立，助力在老中资客户发展|url=http://la.mofcom.gov.cn/article/sqfb/201406/20140600643042.shtml|newspaper=中华人民共和国驻老挝人民民主共和国大使馆经济商务参赞处|date=2014-06-27}}</ref>。2005年，老挝中国企业商会于老挝首都[[万象|万象]]登记注册<ref name=DNK>{{cite news|title=老挝中国企业商会日前正式成立|url=http://la.mofcom.gov.cn/article/zxhz/zzjg/200512/20051201101495.shtml|newspaper=中华人民共和国驻老挝人民民主共和国大使馆经济商务参赞处|date=2005-12-20}}</ref>；2013年初，[[东方电气集团|东方电气集团]]分別与老挝政府和[[老挝国家电力公司|老挝国家电力公司]]签訂合作协议字仪式，东方电气在老撾特许经营[[水电站|水电站]]25年，总投资為9949万美元，另一份則為购电合作协议<ref>{{cite news|author=马佳|title=东方电气投资的南芒河1水电站项目签署特许经营协议及购电合作协议|url=http://la.mofcom.gov.cn/article/zxhz/hzjj/201301/20130108517929.shtml|newspaper=中华人民共和国驻老挝人民民主共和国大使馆经济商务参赞处|date=2013-01-10}}</ref>。約一年後，老挝成立首家中老合资[[银行|银行]]<ref name=bank/>。

== 文化關係 ==
2014年1月末，[[中国云南广播电视台|中国云南广播电视台]]和[[老挝国家电视台|老挝国家电视台]]在老挝首都[[万象|万象]]联合举办名為「中老情·合家欢」的春节电视晚会，晚会上有帶两国传统和特色的文艺表演<ref>{{cite news|author=周楚卿|title=中国老挝联合春晚：中老混搭 精彩纷呈|url=http://news.xinhuanet.com/overseas/2014-01/26/c_119136880.htm|newspaper=新华国际|date=2014-01-26}}</ref>。同年9月，为庆祝兩國建交53周年，[[中华全国归国华侨联合会|中华全国归国华侨联合会]]应老挝中国商会湖南分会的邀请，到老挝進行名為“亲情中华·魅力湖南·美丽常德”的文化交流活動<ref>{{cite news|title=“亲情中华·魅力湖南·美丽常德”文化交流团在老挝受到热烈欢迎|url=http://www.chinaql.org/sites/ql/c83/d_20140922103403647477.html|newspaper=中华全国归国华侨联合会|date=2014-09-22|deadurl=yes|archiveurl=https://web.archive.org/web/20160304200545/http://www.chinaql.org/sites/ql/c83/d_20140922103403647477.html|archivedate=2016-03-04}}</ref>。

據老撾新聞、文化與旅遊部的於2013年的統計數據，最多中國遊客到老撾旅遊的月份為2月<ref>{{cite news|title=春節帶旺老撾琅勃拉邦旅遊:酒店全滿 路邊停滿中國牌照汽車|url=http://finance.takungpao.com.hk/q/2014/0130/2253247.html|newspaper=大公網|date=2014-01-30}}</ref>。

== 軍事關係 ==
2011年末，時任老撾國防部部長[[隆再·皮吉|隆再·皮吉]][[中將|中將]]指該國感谢中方对老挝国防军队的支持和帮助，老挝军队高度重视与中国发展关系，希望能进一步推动两军关系的发展；[[中国人民解放军副总参谋长|中国人民解放军副总参谋长]][[蔡英挺|蔡英挺]]指中方愿意与老挝推动务实合作，为促进两军关系发展與维护地区和平稳定而「做出应有的贡献」<ref>{{cite news|title=老挝国家和军队领导人会见中国军事代表团|url=http://www.chinanews.com/gn/2011/12-05/3509766.shtml|newspaper=中新网|date=2011-12-05}}</ref>。該年，[[中国人民解放军国防大学防务学院|中国人民解放军国防大学防务学院]]组织老撾「军事外交培训班」学员到[[武汉|武汉]]、[[长沙|长沙]]进行参观，他們参观了[[海军工程大学|海军工程大学]]、[[国防科技大学|国防科技大学]]、[[毛泽东|毛泽东]]故居等地；有军事外交培训班人員表示在参观中了解到中国军情，並感受到中国军队对老挝的热情<ref>{{cite news|title=老挝军事外交培训班赴武汉、长沙参观见学|url=http://www.cdsndu.org/cn/news/htmlnews/201312161105430.htm|newspaper=国防大学防务学院|date=2013-11-30|deadurl=yes|archiveurl=https://archive.is/20141004051230/http://www.cdsndu.org/cn/news/htmlnews/201312161105430.htm|archivedate=2014-10-04}}</ref>；[[中国人民解放军|中国人民解放军]]文化交流代表团於同年在老挝举办「中国军队文化活动周」，為中国军方首次在国外举办此類的文化活动<ref>{{cite news|title=中国军队将首次赴国外举办文化活动周|url=http://chn.chinamil.com.cn/2011whz/2011-12/26/content_4753444.htm|newspaper=解放军报社|date=2011-12-26|deadurl=yes|archiveurl=https://archive.is/20141004051448/http://chn.chinamil.com.cn/2011whz/2011-12/26/content_4753444.htm|archivedate=2014-10-04}}</ref>。
