'''中国铁路太原局集团有限公司'''，原名'''太原铁路局'''，是[[中國鐵路總公司|中國鐵路總公司]]下属公司。管理的线路总延展长度為3623.4[[公里|公里]]，营业里程1595.8公里，下辖有176个车站，其管辖范围基本上为原北京铁路局管辖的[[山西省|山西省]]境内的铁路网。

==沿革==
2005年3月18日0时，铁道部将全国15个铁路分局所下辖的41个铁路分局全部裁撤。其中[[北京铁路局|北京铁路局]]下辖的5个铁路分局：北京铁路分局、天津铁路分局、石家庄铁路分局、大同铁路分局、太原铁路分局被裁撤。经铁道部批准，将原北京铁路局下属的大同铁路分局与太原铁路分局合并成立新的'''太原铁路局'''。[[太焦铁路|太焦铁路]][[大平站|大平站]]以南、[[邯长铁路|邯长铁路]]、[[侯月铁路|侯月铁路]][[端氏站|端氏站]]以南归[[郑州铁路局|郑州铁路局]]管辖；[[石太铁路|石太铁路]][[坡头站|坡头站]]以东、[[京原铁路|京原铁路]][[灵丘站|灵丘站]]以东、[[阳涉铁路|阳涉铁路]]归北京铁路局管辖。

2005年，太原铁路局进行了工商登记<ref name=guapai/>。

2017年11月7日，太原、兰州、广州铁路（集团）公司等3家铁路局（公司）的名称变更通过[[国家工商行政管理总局|国家工商行政管理总局]]核准，太原铁路局名称变更为'''中国铁路太原局集团有限公司'''（(国)名称变核内字[2017]第5343号）<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1854064  |title = 18家铁路局将集体变身集团公司：名称变更已全部通过核准 |author =  |date =2017-11-08   |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月14日，在国家企业信用信息公示系统中，中国铁路太原局集团有限公司已完成工商登记变更。工商登记变更后，中国铁路太原局集团有限公司仍为中国铁路总公司100%控股的公司，公司由原先的全民所有制企业变成有限责任公司（非自然人投资或控股的法人独资）企业，由原先铁道部出资变更为中国铁路总公司出资，注册资本由原先的483.00741亿元变更为585.10856亿元人民币，经营范围扩大，增加铁路专用设备及相关工业设备的制造、安装、维修、销售、租赁，建筑施工，运输生产资料购销，房地产开发经营和租赁经营，物业服务，建筑装饰装修工程，矿产资源开采，花卉苗木种植，食品经营等<ref name=jiu>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1863934 |title = 9家铁路局变更工商登记：全民所有制企业变为有限责任公司 |author =  |date = 2017-11-14 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月19日，中国铁路太原局集团有限公司正式挂牌<ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1870782|title = 18个铁路局已改制为铁路局集团有限公司，今日正式挂牌 |author =  |date = 2017-11-19 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref name=guapai>{{Cite web |url = http://finance.ifeng.com/a/20171119/15803099_0.shtml |title = 18家铁路局集团改制后集中挂牌！附主要领导人简历 |author =  |date = 2017-11-19 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。
