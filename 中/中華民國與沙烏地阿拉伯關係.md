'''中華民國與沙烏地阿拉伯關係'''是指[[中華民國|中華民國]]與[[沙烏地阿拉伯王國|沙烏地阿拉伯王國]]（通稱'''沙烏地阿拉伯'''、'''沙烏地'''）之間的關係。兩國於1946－1990年有正式[[外交#外交上的承認|外交關係]]，斷交後，在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

1946年11月15日，簽訂《中國沙-{地}-阿拉伯友好[[條約|條約]]》{{noteTag|條約稱呼沙-{地}-阿拉伯，非台灣慣稱的-{沙烏地}-阿拉伯。}}，建立[[領事|領事級]][[外交#外交上的承認|外交關係]]。<ref name="駐館與駐地關係">{{Cite web |url = http://www.taiwanembassy.org/sa/post/30.html |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref><ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/07.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，由於沙烏地阿拉伯為[[伊斯蘭教|伊斯蘭教]][[政教合一|政教合一]]國家、反對[[共產主義|共產主義]]，因此雙方仍有外交關係。

1957年5月30日，升格為大使級外交關係。於首都[[利雅德|利雅德]]設立'''中華民國駐沙烏地阿拉伯王國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1957年8月，[[中華民國總統|總統]][[蔣中正|蔣中正]]任命[[馬步芳|馬步芳]]為首任驻沙烏地阿拉伯大使。

1961年8月，馬步芳因逼侄女马月兰为[[妾|妾]]的[[醜聞|醜聞]]，須召回法辦；但馬步芳拒絕回國而入籍沙烏地阿拉伯。1975年7月31日，馬步芳在沙烏地阿拉伯病死。<ref>{{cite news|title=荒淫無道的軍閥馬步芳 連外孫女都慘遭其蹂躪|url=http://history.dwnews.com/big5/news/2011-01-09/57277936-4.html|publisher=[[多維新聞網|多維新聞網]]|date=2011-01-09|accessdate=2015-03-07}}</ref>

1971年5月17至20日，[[沙烏地阿拉伯國王|沙烏地阿拉伯國王]][[费萨尔·本·阿卜杜勒-阿齐兹·阿勒沙特|費瑟]]抵台訪問，與總統蔣中正會晤；雙方於同月20日發布《聯合公報》，費瑟邀請蔣介石訪問沙烏地阿拉伯，蔣同意訪問。10月25日，沙烏地阿拉伯在[[聯合國大會|聯合國大會]]上支持中華民國繼續保有[[中國與聯合國#代表權問題|中國代表權]]，但[[聯合國2758號決議|表決失敗]]。沙國是唯一支持中華民國的[[中東#國家及地區|中東國家]]。但同時沙特阿拉伯也曾主張由[[中華人民共和國|中華人民共和國]]代表[[中國|中國]]，中華民國則以「[[台灣|台灣]]」的名義在[[聯合國|聯合國]]佔有一席，此為「雙重代表權案」；又建議聯合國主持[[公民投票|公民投票]]，由臺灣人民[[自決|自決]]臺灣的未來。<ref>{{cite journal|author=[[陳隆志|陳隆志]]|title=台灣與聯合國—回顧與展望|journal=新世紀智庫論壇|date=2001-06-30|issue=14}}</ref>然而後來卻因為沙烏地阿拉伯強硬的態度，間接導致中華民國退出聯合國。<ref>[http://bbs3.nsysu.edu.tw/txtVersion/treasure/sis/M.1053575941.A/M.1053576038.KM.html 回憶一九七一年聯合國席位的最後一戰]</ref>

1970年代，[[中華民國政府|中華民國政府]]在台灣推行[[十大建設|十大建設]]，其中經費有許多[[貸款|貸款]]自沙烏地阿拉伯。故將[[中山高速公路|中山高速公路]]上橫跨[[彰化縣|彰化縣]]與[[雲林縣|雲林縣]]的公路橋命名為[[中沙大橋|中沙大橋]]以資紀念。<ref>http://ktli.sinica.edu.tw/people/%BB%AF%ACJ%A9%F7.html</ref><ref>{{cite web |url=http://taipei.tzuchi.org.tw/rhythms/magazine/content/58/2/taiwan_m.htm |title=存档副本 |accessdate=2007-09-04 |deadurl=yes |archiveurl=https://archive.is/20120714181217/http://taipei.tzuchi.org.tw/rhythms/magazine/content/58/2/taiwan_m.htm |archivedate=2012-07-14 }}</ref><ref>【白嘉莉傳奇】《大社會·會大咖》[[東森電視|東森傳播]] 2009.10.02播出</ref>

1977年7月，總統[[嚴家淦|嚴家淦]]訪問沙烏地阿拉伯，是[[臺灣戰後時期|中華民國政府遷台]]後首次出訪的總統。同年，中華民國[[榮民工程事業管理處|榮民工程事業管理處]]承建穿越阿拉伯高原西緣斷崖的[[夏爾降坡公路|夏爾降坡公路]]，以及興建{{link-ar|吉達港|ميناء جدة الإسلامي}}開工。

1990年7月22日，沙烏地阿拉伯與中華民國斷交。<ref name="駐館與駐地關係"/><ref name="外交年鑑"/>中華民國外交部長[[錢復|錢復]]於當日召見[[沙特阿拉伯驻华大使列表|沙烏地駐華大使]]舒海爾（Asaad Abdul Aziz Al-Zuhair），並向{{link-en|沙烏地阿拉伯政府|Politics of Saudi Arabia|沙烏地政府}}提出4點要求：「對沙國政府此一損害中沙兩國利益之決定，正式向其政府提出嚴重抗議、要求沙國政府對於續留沙國境內之[[中華民國國民|中華民國國民]]，依照國際慣例給予合法保障及公平待遇、要求沙國政府確保我駐沙大使館及駐[[吉達|吉達]]總領事館之所有館產、請沙國政府儘速與我就未來中沙關係之調整進行磋商。」<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=0BC4C3C2E7178A71 |author = 公眾外交協調會 |date = 1990年7月22日 |title = 外交部部長錢復就沙烏地阿拉伯王國與中共建交並與我中止外交關係事 |publisher = 中華民國外交部}}</ref>

2005年8月3日，[[中華民國外交部#歷任首長|中華民國外交部長]][[陳唐山|陳唐山]]訪問沙烏地阿拉伯，出席[[法赫德|法赫德]]國王之喪禮，並祝賀新王[[阿卜杜拉·賓·阿卜杜勒-阿齊兹·阿紹德|阿布杜拉]]登基。<ref name="駐館與駐地關係"/>

2015年1月24日，中華民國外交部[[政务官|政務次長]]柯森耀率領特使團前往沙國，向新王薩爾曼呈遞中華民國之弔唁及祝賀[[國書|國書]]；外交部長[[林永樂|林永樂]]亦於同月26日前往沙烏地阿拉伯駐華商務辦事處，就沙王阿布都拉日前辭世事，代表中華民國政府及人民向沙國政府及人民表達衷心弔唁之意，並在代表塔勒陪同下簽名致哀。<ref>[http://www.mofa.gov.tw/News_Content_M_2.aspx?n=8742DCE7A2A28761&sms=491D0E5BF5F4BC36&s=340415DC94519E3D 外交部部長林永樂代表中華民國政府及人民赴沙烏地阿拉伯駐華商務辦事處為沙王阿布都拉辭世簽名弔唁]，中華民國外交部</ref>

持[[沙烏地阿拉伯護照|沙烏地阿拉伯護照]]的{{link-en|沙烏地阿拉伯人口|Demographics of Saudi Arabia|沙烏地公民}}則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將沙烏地按地區歸類為'''橙色警戒'''：避免非必要旅行，與'''紅色警戒'''：不宜前往，宜儘速離境（以上截至2015年11月24日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-119-6e05d-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將沙烏地歸類為'''第二級警示'''：對當地採取加強防護（2015年6月9日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=FE259AF4995F9A97 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
中華民國[[經濟部國際貿易局|經濟部國際貿易局]]在首都[[利雅德|利雅德]]設立'''駐沙烏地阿拉伯王國代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=217&pid=305864 |title = 本部駐外單位 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

沙烏地阿拉伯是中華民國在[[中東|中東]]的最大貿易夥伴及最大[[石油|原油]]供應國，自2011年起，每年自沙烏地進口原油均超過100億[[美元|美元]]，佔石油進口總額的3分之1強，也是自沙烏地進口總額的80%以上，多年來對沙烏地的貿易均呈現巨額[[淨出口|入超]]。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

2014年，銷往沙烏地的[[轎車|小客車]]（不含[[汽車零組件列表|零件]]）持續快速成長，金額為6億9,376萬美元，年增27%，已成為台製汽車最大的[[出口贸易|出口]]市場；[[柴油|柴油]]近3年來也是對沙烏地出口的重要項目第2位（約占出口比重的15%－30%），年增8.3%，但沙烏地與外商合資興建2座[[煉油廠|煉油廠]]後，也已具備自給自足的煉油產能，未來自中華民國[[進口|進口]]柴油是否將大幅減少，值得觀察。<ref name="與我國經貿關係"/><ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=671486&url=/NewsSearch.asp |author = 經濟部駐沙烏地阿拉伯王國代表處經濟組 |date = 2015年8月20日 |title = 本年1至5月台灣對沙國出口大幅衰退29.2% |publisher = 經貿透視網}}</ref>

2018年，沙烏地阿拉伯是中華民國第14大貿易夥伴、第8大進口夥伴、第30大出口夥伴。出口至沙烏地的金額為7億8,110萬4,950美元，年減28.490%；自沙烏地進口的金額為86億3,218萬5,037美元，年增25.734%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）78億5,108萬0,087美元，年增35.993%。2017年，沙烏地阿拉伯是中華民國第15大貿易夥伴、第9大進口夥伴、第25大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===技術團===
====農漁業====
1979年2月，在沙國農業與水務部（農水部）正式成立。[[中華民國政府|中華民國政府]]派遣34位農業及水利專業技術人員，以應聘方式分派到農水部有關單位服務，以配合沙國經濟建設之發展。首任團長為蔡海塗博士，迄今共歷7位團長。由於多數團員任勞任怨的工作態度以及優異的技術表現，團員人數曾經達到50人。

漁技團係依據1979年9月中、沙兩國簽訂的《[[淡水魚|淡水魚]][[養殖漁業|養殖]]合作計畫協議書》，在1980年11月成立於[[沙特國王大學|紹德國王大學]]農學院第拉普農場。當時的執行機構為[[台灣省|台灣省]][[行政院農業委員會水產試驗所|水產試驗所]]及沙國國家科技中心（1986年改稱為阿布都阿澤國王科技城或簡稱老王科技城）。

1988年元月第六屆中、沙經濟技術合作會議時，簽署《中沙科技合作協定》，自此漁業計畫亦包括在該合作協定。

第九屆中、沙經濟技術合作會議，雙方同意將養殖技術擴及濱海地區，惟因雙方斷交而尚無進展，經農、漁團之居間聯絡，[[中華民國科技部|國科會]]同意以專家短期互訪之方式開啟技術合作計畫。

漁技團及農服團於1997年7月1日合併更名為駐沙烏地阿拉伯技術團。

2012年，由{{link-en|沙烏地阿拉伯政治|Politics of Saudi Arabia|沙烏地阿拉伯政府}}出資，委請「國家土地政策訓練中心」代訓其農業相關人員，計畫項目包括「[[休閒農業|休閒農業]]」、「[[海水魚|海水魚]]繁養殖」及「[[畜牧业|畜牧]]生產－[[乳牛|乳牛]]及[[家禽|家禽]]」。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/f0a43734-c84a-436d-bd3a-060d93fbbb19.pdf |author = 國際合作及經濟事務司 |date = 2013年4月26日 |title = 國際合作發展事務101年度報告（參、101年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref>

====其他====

====客運====
1980年代，[[中華航空|中華航空]]曾有經營台北[[中正國際機場|中正國際機場]]（現名臺灣桃園國際機場）↔沙烏地[[吉達國際機場|吉達國際機場]]、{{link-en|達蘭國際機場|King Abdulaziz Air Base}}的航線，但在1990年斷交後廢除，迄今不曾恢復。

兩國無直航班機，可經由（不計航點遠近，截至2018年11月16日）<ref name="生活資訊">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-42-190d2-05-1.html |title = 生活資訊 |publisher = 外交部領事事務局}}</ref>：

{{ROC}}
*[[臺灣桃園國際機場|臺北]]→[[北京首都國際機場|北京]]（*飛沙烏地[[朝觐_(伊斯兰教)|朝覲]]專機）、[[廣州白雲國際機場|廣州]]、[[兰州中川国际机场|蘭州]]*、[[烏魯木齊地窩堡國際機場|烏魯木齊]]*、[[银川河东国际机场|銀川]]*、[[尼諾伊·阿基諾國際機場|馬尼拉]]、[[新加坡樟宜機場|新加坡]]、[[吉隆坡國際機場|吉隆坡]]、[[檳城國際機場|檳城]]*、[[蘇凡納布機場|曼谷]]*、[[蘇加諾-哈達國際機場|雅加達]]、[[朱安达国际机场|泗水]]、[[英迪拉·甘地國際機場|德里]]、[[杜拜國際機場|杜拜]]、[[阿塔蒂尔克国际机场|伊斯坦堡]]、[[倫敦希斯洛機場|倫敦]]、[[戴高樂國際機場|巴黎]]、[[羅馬-菲烏米奇諾機場|羅馬]]、[[法蘭克福國際機場|法蘭克福]]、[[阿姆斯特丹史基浦機場|阿姆斯特丹]]、[[維也納國際機場|維也納]]、[[約翰·甘迺迪國際機場|紐約]]、[[洛杉磯國際機場|洛杉磯]]，再轉機前往{{link-en|沙烏地阿拉伯機場列表|List of airports in Saudi Arabia|沙烏地阿拉伯的國際機場}}。
