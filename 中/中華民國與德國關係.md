'''中華民國與德國關係'''是指[[中華民國|中華民國]]與[[德意志聯邦共和國|德意志聯邦共和國]]之間的關係。1913－1941年，兩國有官方[[外交#外交上的承認|外交關係]]但曾經中斷，二次斷交後由於[[同盟國軍事佔領德國|德國分治]]與[[冷戰|冷戰]]緣故，中華民國只與德意志聯邦共和國（西德）保持關係，直到[[兩德統一|兩德統一]]。目前均在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
{{see also|中華民國外交|德國外交|中德關係 (1911年－1941年){{!}}中德合作}}
{{see also|中華民國駐德國大使列表{{!}}中華民國駐德國外交官列表|德國駐中華民國大使列表{{!}}德國駐中華民國外交官列表}}
1912年，中華民國成立後，派駐[[德意志國|德意志國]]外交代表。

1913年10月6日，德國[[外交#外交上的承認|外交承認]]中華民國。在首都[[柏林|柏林]]設立'''中華民國駐德意志國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1917年3月14日，[[第一次世界大戰|第一次世界大戰]]爆發後，中華民國與德國斷交，德國在上海的全部6艘船艦遭扣留<ref>[http://cn.nytimes.com/china/20170316/iht-retrospective-1917-china/zh-hant/ 時報歷史上的今天：中國與德國斷交]，[[紐約時報中文網|紐約時報中文網]]，2017年3月16日</ref>。3月16日，直隶交涉员[[黄荣良|黄荣良]]陪同中華民國政府代表、直隶全省警务处处长[[杨以德|杨以德]]前往德国驻津领事署、[[天津德租界|天津德租界]]工部局办理接收事宜。3月17日，[[天津德租界|天津德租界]]收回后改名为天津特别一區，并设管理局行使行政管理职能<ref name="地方志">{{cite web | url=http://www.tjdfz.org.cn/tjtz/dsj/minguo/ | title=天津地方志 > 天津通志 > 大事记——中华民国（1912.1－1949.9） | publisher=天津地方志 | accessdate=2012-08-25 }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。7月1日，撤館。8月14日，對德國[[宣戰|宣戰]]。

1921年5月20日，中華民國與德國在[[北京|北京]]簽署《[[中德協約|中德協約]]》，此為中華民國簽署的第一個平等條約，<ref name="史料">{{cite web |url=http://www.npm.gov.tw/exh100/diplomatic/page_ch03.html |date=2011年 |title=《百年傳承　走出活路》－中華民國外交史料特展：力爭平等 |publisher=[[國立故宮博物院|國立故宮博物院]]}}</ref>兩國恢復邦交。

1927年，總統[[蔣中正|蔣中正]]聘請[[馬克斯·鮑爾|馬克斯·鮑爾]]等德國軍官為軍事顧問和教官。

1934年，前[[德国总参谋部|德國國防部部隊局]]長[[漢斯·馮·塞克特|漢斯·馮·塞克特]]任[[國民政府|國民政府]]軍事顧問團資深顧問。

1935年5月18日，駐德國公使館升格為'''中華民國駐德意志國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1938年2月，[[納粹德國|納粹德國]]承認[[滿洲國|滿洲國]]。4月對中國[[禁運|禁運]]軍火、6月召回駐中國大使、7月撤出軍事顧問團。

1941年7月1日，納粹德國承認[[汪精衛政權|汪精衛政權]]。7月2日，與納粹德國斷交、10日撤館。

1949年10月1日，[[中華人民共和國|中華人民共和國]]成立。年底，[[中華民國政府|中華民國政府]]遷至[[台灣|台灣]]。同年，[[德意志地區|德意志地區]]亦分別成立[[德意志民主共和國|德意志民主共和國]]（東德）和[[德意志聯邦共和國|德意志聯邦共和國]]（西德）。

1950年，東德与中华人民共和国建交。<ref name="中德建交"/>之後由於東西[[冷戰|冷戰]]，中華民國在[[意識形態|意識形態]]和[[政體列表|制度]]上均偏向與[[美國|美國]]為首的[[資本主義|資本主義]]陣營，復加上共產集團國家的封閉，使雙方幾乎沒有往來。在過往中華民國政府更曾禁止國民前往包括東德在內的東歐共產集團國家。

1955年10月，中華民國政府宣佈終止對德戰爭狀態。<ref name="年表">[[陳布雷|陳-{布}-雷]]等編著：《蔣介石先生年表》，台北：[[傳記文學|傳記文學]]出版社，1978年6月1日</ref>{{rp|83}}

1963年7月2日，中華民國政府成立簡稱「[[德國軍事顧問團|明德小組]]」（{{lang-de|Ming-teh-Gruppe}}）的「明德專案連絡人室」，以推動與西德培訓在台之中華民國國軍事宜。與[[白團|白團]]及[[美軍顧問團_(中華民國)|美軍顧問團]]並列為戰後台灣三大外國軍事顧問團。另有派軍官赴德培訓的「培德案」，前後合計共培訓了25位軍官{{noteTag|前[[國家安全會議_(中華民國)|總統府國家安全會議]]秘書長[[殷宗文|殷宗文]]與前[[中華民國陸軍司令|陸軍總司令]][[李楨林|李楨林]]、孟澤爾將軍侍從官[[王玉麒_(臺灣軍人)|王玉麒]]等<ref>{{cite news| url =http://la.worldjournal.com/view/full_sf/22580056/article-| title =蔣介石 曾聘德軍顧問訓練國軍| work =世界新聞網-北美華人社區新聞| author =王慶偉| location =美國[[史丹福|史丹福]]| year =2013| date =May 15, 2013| language =zh| publisher =[[北美世界日報|北美世界日報]]| deadurl =yes| archiveurl =https://archive.is/20140522151602/http://la.worldjournal.com/view/full_sf/22580056/article-| archivedate =2014年5月22日}}</ref>}}。

[[File:Voting_res_2758.png|thumb]]，[[兩德統一|統一]]前的[[東德|東德]]與[[西德|西德]]當時尚未加入，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
1971年10月25日，由於當時兩德皆非[[聯合國會員國列表|聯合國會員國]]，因此無法就[[中國與聯合國#代表權問題|中國代表權]]問題表態。

1972年，西德与中华人民共和国建交。<ref name="中德建交">《[http://www.fmprc.gov.cn/chn/pds/ziliao/wjs/2159/t9015.htm 中德建交背景情况]》[[中华人民共和国外交部|中华人民共和国外交部]]</ref>之前，西德與中华民国没有正式外交关系，是當時少數與[[海峽兩岸關係|兩岸]]均沒有建交的[[西方世界|西方國家]]。

1979年1月4日，總統[[蔣經國|蔣經國]]接受德國《[[明鏡週刊|明鏡週刊]]》記者德薩尼訪談時嚴正表示，中華民國政府絕對不會與北平偽政權發生接觸，因為與共產黨談和，無異「與虎謀皮」。

1999年9月21日，臺灣中部發生[[九二一大地震|九二一大地震]]，[[德國總理|總理]][[格哈特·施羅德|施洛德]]對災民及家屬表達最深沉的哀悼<!-- <ref name="worldleaders"/>引用錯誤：沒有為"worldleaders"的參考文獻提供內容 -->。德國[[紅十字會與紅新月會國際聯合會|紅十字會]]救難協會成員11人，22日帶著7隻受過專業訓練的[[搜索及拯救犬|救難犬]]，以及1500公斤的各式器材抵台，救難犬經過緊急[[隔離檢疫|檢疫]]通關後，立刻投入救災。<ref>[各國救援隊到美、英、德等13國500多人抵台]【1999-09-23/[[聯合報|聯合報]]/10版/集集大震　特別報導】</ref>另有德國殯儀協會專家團15人抵台協助[[台中市|台中市]]。<ref>[21個國家、38個國際救援組織來台，其中大部分均具官方身分][[中國時報|中時新聞]]資料庫，1999/09/25</ref>

2019年1月16日，對於中國共產黨中央委員會總書記[[習近平|習近平]]在《[[告臺灣同胞書|告臺灣同胞書]]》發表40周年紀念會的[[习五条|有關談話]]，德國外交部長[[海科·馬斯|馬斯]]在[[德国联邦议院|國會]]答覆議員質詢時表示，[[德國政府|德國政府]]將秉持過去立場，向中方清楚表達「德國無法接受[[解放台湾|用武力威脅台灣]]的作法」。並說，在[[歐洲聯盟外交事務委員會|歐洲聯盟外交事務委員會]]，中國用了不可不謂狡猾的方式影響了[[欧洲联盟成员国|個別成員國]]，至少一次成功阻止針對台灣的決議案，因此期盼歐盟將來能採[[多數決|多數決]]，而非[[共識決|共識決]]，對[[台灣問題|台灣問題]]採取一致立場。馬斯是近年針對台灣問題，清楚表達立場的最高層級德國官員；另一方面，德國外交部次長{{link-en|麥可·羅思|Michael Roth (politician)|羅思}}以書面答詢時則表示，德國一向認為台灣問題只能用和平的方式解決，在雙方互相尊重的基礎上進行對話。唯有在雙方都同意的情況下，[[臺海現狀|兩岸的現狀]]才可以改變。<ref>{{Cite web |url = https://tw.news.appledaily.com/international/realtime/20190119/1503872 |author = 綜合報導 |date = 2019年1月19日 |title = 挺台最高層級 德國外長：無法接受中國武統台灣 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref><ref>{{Cite web |url = https://www.thenewslens.com/article/112365 |author = Abby Huang |date = 2019年1月20日 |title = 德國外交部長：不接受中國武力威脅台灣、呼籲歐盟對台「立場一致」 |publisher = [[風傳媒|風傳媒]]}}</ref>

===貿易===
德國是中華民國在[[歐洲國家和地區列表|歐洲]]最大的貿易伙伴<ref name="105年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>。[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於德國第3大城[[慕尼黑|慕尼黑]]、第7大城[[杜塞道夫|杜塞道夫]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#EUROPE |title = 外貿協會全球據點 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]則在首都與第1大城[[柏林|柏林]]設立'''駐德國代表處經濟組'''、第5大城設立'''駐[[法蘭克福|法蘭克福]]辦事處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2018年，德國是中華民國第8大貿易夥伴、第5大[[進口|進口]]夥伴、第10大[[出口贸易|出口]]夥伴，在[[歐洲國家和地區列表|歐洲經濟體]]皆排名第1。出口至德國的金額為70億5,954萬7,741[[美元|美元]]，年增9.399%；自德國進口的金額為99億6,633萬4,106美元，年增8.328%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）29億0,678萬6,365美元，年增5.813%。2017年，德國是中華民國第8大貿易夥伴、第5大進口夥伴、第10大出口夥伴，在歐洲經濟體皆排名第1。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資===
根據經濟部投資審議委員會（投審會）統計，1969年－2014年，[[臺灣商人|臺商]]對德國的總投資金額為2億4,334萬美元，共179件。

臺商在[[資訊業|資訊業]]與[[軟體|軟體業]]已在德國打開[[品牌|品牌]]市場並穩定成長。例如：[[宏碁|宏碁]]、[[華碩電腦|華碩電腦]]、[[明基電通|明基電通]]、[[宇達電通|宇達電通]]、[[創見資訊|創見資訊]]、[[微星科技|微星科技]]、[[友訊科技|友訊科技]]、[[天瀚科技|天瀚科技]]、[[曜越科技|曜越科技]]、[[昆盈企業|昆盈企業]]等。在[[航運|海運]]運輸上，[[長榮海運|長榮海運]]、[[陽明海運|陽明海運]]及[[萬海航運|萬海航運]]等公司群聚於德國[[漢堡港|漢堡港]]；[[中華航空|中華航空]]、[[長榮航空|長榮航空]]在德國經營已久。中華航空除貨運外，也提供[[臺灣桃園國際機場|臺北]]－[[法蘭克福國際機場|法蘭克福]]直航客運服務，而長榮航空則以貨運為主要經營業務，其德國[[分公司|分公司]]也出售客運[[機票|機票]]。<ref name="投資環境簡介"/>

德商在臺從事[[化學|化學]]、[[機械|機械]]、[[金融|金融]]、[[通訊|通訊]]、[[物流|物流]]、[[生物技術|生物技術]]、[[醫藥|醫藥]]、[[半導體|半導體]]、[[清洁能源|綠能]]及面板等產業。例如：[[安聯|安聯]]、[[默克集團|默克集團]]、[[羅德史瓦茲|羅德史瓦茲]]、[[伊諾力克工業股份|伊諾力克工業股份]]、[[拜耳|拜耳]]、[[蒂森克虜伯|蒂森克虜伯]]、[[漢高|漢高]]、{{link-en|創浦|Trumpf}}、{{link-en|飛斯妥|Festo}}、{{link-en|曼茲自動化|Manz}}、[[DHL|DHL]]、英華威、應薄、皕德等公司；中華民國經濟部[[工業技術研究院|工業技術研究院]]（工研院）與德國ExtraEnergy合作，建立[[純電動車|電動車]][[蓄電池|電池]]共通[[介面|介面]]與安全標準；默克化學製藥與生技中心共同宣佈在臺灣建立亞洲技術訓練中心；{{link-en|阿托科技|Atotech}}進駐[[高雄市|高雄]]環保科技園區，並成立研發中心；[[西門子公司|西門子公司]]投資設立研發中心，協助業界開發高階[[工具機|工具機]]。<ref name="投資環境簡介"/>2017年10月，西門子在臺灣成立[[離岸風力發電|離岸風力發電]]亞太營運中心。<ref name="106年外交年鑑"/>

2013年3－5月，德國研究船{{link-en|太陽號|RV Sonne}}前往台灣海域探勘，由台灣能源國家型科技計畫辦公室與德國[[基爾亥姆霍茲海洋研究中心|基爾亥姆霍茲海洋研究中心]]及{{Link-de|海洋環境科學研究中心|MARUM – The Ocean in the Earth System}}合作，進行[[天然氣水合物|天然氣水合物]]（可燃冰）、[[遙控潛水器|遙控潛水器]]的科學應用，結合[[二氧化碳|二氧化碳]]海底地層儲存地質探勘及[[黑潮|黑潮]]發電海洋地質鑽勘。<ref name="102年外交年鑑"/>雙方於2009年在台北首次舉行關於可燃冰議題的雙邊會議，第四屆「可燃冰雙邊會議（TaiGer IV）」於2014年2月13－14日在德國[[不萊梅大學|不萊梅大學]]海洋環境科學研究中心舉行，會中並簽署合作[[備忘錄|備忘錄]]。<ref name="可燃冰">{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=BD3B450373053909&sms=02499EFF01F32DED&s=21B125B0A792A40B |author = 公眾外交協調會 |date = 2014年2月18日 |title = 台德可燃冰雙邊會議於德國不萊梅舉行(駐德國代表處) |publisher = 中華民國外交部}}</ref>

[[中華民國外交部|中華民國外交部]]提供臺灣[[獎學金|獎學金]]、教育部提供[[中華民國國語|華語文]]獎學金給予德國留學生，科技部則提供德國學者及研究人員赴台研究獎補助；德國各相關機構也提供台灣學者及研究人員赴德獎補助。2017年，台灣留德學生有2,013人，德國留台學生則有1,174人。<ref name="106年外交年鑑"/>

2016年8月，[[德國聯邦教育及研究部|德國聯邦教育及研究部]]次長{{Link-de|舒特|Georg Schütte (Staatssekretär)}}與中華民國教育部政務次長[[陳良基|陳良基]]於台北召開「臺德[[職業教育|職業教育]]論壇」。11月，於柏林舉辦首屆「臺德[[高等教育|高等教育]]論壇」，研討大學合作、教學、培養[[博士|博士]]人才及高等教育政策研究交流等，會中並簽署合作備忘錄。<ref name="105年外交年鑑"/>
