'''中国人民解放军北部战区联合作战指挥中心'''，位于[[辽宁省|辽宁省]][[沈阳市|沈阳市]]，隶属[[中国人民解放军北部战区联合参谋部|中国人民解放军北部战区联合参谋部]]，是[[中国人民解放军北部战区|中国人民解放军北部战区]]的联合作战指挥机构。

==简介==
2015年11月27日，[[中华人民共和国国防部|中华人民共和国国防部]]新闻发言人[[杨宇军|杨宇军]]就[[深化国防和军队改革|深化国防和军队改革]]在健全联合作战指挥体制上有何举措时称，“主要是适应打赢信息化战争、有效履行使命任务的要求，组建战区联合作战指挥机构，健全军委联合作战指挥机构，构建完善精干高效的战略战役指挥体系。”<ref>{{Cite web |url =http://military.china.com.cn/2015-11/27/content_37177664.htm  |title =国防部谈军队改革：组建战区联合作战指挥机构  |author =  |date =2015-11-27  |publisher =  中国网|language =  |accessdate =  |quote =  }}</ref>2016年2月1日，中华人民共和国国防部新闻局局长、国防部新闻发言人杨宇军在国防部专题新闻发布会上表示，战区是“本战略方向的唯一最高联合作战指挥机构”<ref>{{Cite web |url = http://news.163.com/16/0202/06/BEQ5BF2500014AED.html|title = 战区是唯一最高联合作战指挥机构|author =  |date =  2016-02-02 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2016年2月前后，新成立的五个战区分别设立战区联合作战指挥中心。战区联合作战指挥中心是在此前各[[大军区|大军区]]作战指挥中心的基础上建成，增加了空军、火箭军等军兵种，增设了测绘导航、空域管理等作战要素，调整了指挥席位的设置<ref>{{Cite web |url = http://mil.huanqiu.com/china/2016-02/8623038.html |title =西部战区砍原军区数十个指挥席 30分钟发7方案  |author =  |date = 2016-02-29 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref>。

2016年1月29日起，北部战区联合作战指挥中心集中1个月开展“联合作战值班人员培训”<ref>{{Cite web |url = http://news.xinhuanet.com/mil/2016-02/21/c_128737381.htm |title =北部战区扎实推进联合作战指挥中心运行建设  |author =  |date = 2016-02-21 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。2016年2月2日夜12时，北部战区联合作战指挥中心正式启动试运行值班<ref name=mohe>{{Cite web |url =  http://news.ifeng.com/a/20160225/47575569_0.shtml|title =  解放军新战区在磨合中走向联合|author =  |date = 2016-02-25 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。2016年4月初，北部战区首批44项军事规章汇编为《北部战区军事规章汇编》公布，北部战区联合作战指挥中心根据其中的《联合作战值班规定》要求，几乎每日组织一场演练<ref>{{Cite web |url = http://news.xinhuanet.com/mil/2016-04/10/c_128880130.htm |title = 北部战区建章立制推动主战职能落地生根 |author =  |date = 2016-04-10 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。2016年4月19日，中共中央总书记、国家主席、中央军委主席、军委联指总指挥[[习近平|习近平]]在[[中央军委联合作战指挥中心|军委联指中心]]大楼总指挥席上就座不久，通过视频向北部战区司令员[[宋普选|宋普选]]了解北部战区联指中心运转情况<ref>{{Cite web |url =  http://www.chinawriter.com.cn/n1/2016/0928/c405175-28745454.html|title =写在主阵地上的优秀答卷  |author =  |date = 2016-09-28 |publisher = 中国作家网 |language =  |accessdate =  |quote =  }}</ref>。
