'''中国人民武装警察部队研究院'''，位于[[北京市|北京市]]，是[[中国人民武装警察部队|中国人民武装警察部队]]科研机构。

==简介==
2017年12月31日，中国人民武装警察部队研究院组建，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[奥林匹克森林公园|奥林匹克森林公园]]东侧，“是理技合一的军事科学研究机构，是武警部队党委首长决策的咨询智库、理论研究的主要阵地、科技创新的孵化中心、法规制度的完善平台、专业人才的储备基地。”中国人民武装警察部队研究院设有战略规划、系统、法制、装备、工程等领域的研究部门<ref>{{Cite web |url =https://www.thepaper.cn/newsDetail_forward_2272973  |title = 武警部队研究院去年底组建，系武警党委首长决策的咨询智库 |author =  |date = 2018-07-18 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.81.cn/jfjbmap/content/2018-02/24/content_200183.htm |title =武警研究院围绕部队遂行任务需求破题立项 科技创新注重研战一体  |author =  |date = |publisher = 解放军报2018年2月24日第02版：要闻 |language =  |accessdate =  |quote =  }}</ref>。
