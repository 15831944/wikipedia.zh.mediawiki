'''中国气象局'''（{{lang-en|China Meteorological Administration}}，[[缩写|缩写]]作 {{lang|en|CMA}}）是[[中华人民共和国国务院|中华人民共和国国务院]]的副部级[[国务院直属事业单位|直属事业单位]]，主要承担全国[[气象|气象]]工作的政府行政管理及气象工作的组织管理。其下属[[国家气象中心|国家气象中心]]则主要负责[[天气预报|天气预报]]、海象预报等工作。

== 沿革 ==
1949年12月8日，'''[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]气象局'''（中央军委气象局）成立，统一领导全国气象事业。1950年3月，成立了第一个直属业务机构[[中央气象台|中央气象台]]<ref name="zgtqw">[http://www.weather.com.cn/static/html/article/20100224/196218.shtml 中央气象台的变迁，中国天气网，2010-02-24]</ref><ref name="zgqxj">[http://2011.cma.gov.cn/ztbd/ggkfssn/jiedu/200812/t20081212_22713.html 兴利除弊话改革——气象部门双重领导管理体制实施回顾，中国气象局，2008-12-12]</ref>。1953年8月1日，为配合国家开展大规模经济建设的需要，根据[[中央人民政府主席|中央人民政府主席]][[毛泽东|毛泽东]]、[[政务院总理|政务院总理]][[周恩来|周恩来]]的命令，中央军委气象局自[[中国人民解放军|中国人民解放军]]建制转归[[中央人民政府政务院|政务院]]建制，成为政务院[[国务院直属机构|直属机构]]，更名为'''中央气象局'''；全国各地的气象机构也均从军队建制转归地方[[人民政府|人民政府]]建制<ref name="tjsz">[http://www.tj.gov.cn/tblm/tj_sz/200709/t20070921_22653.htm 第一节 机构沿革，天津史志，2007-09-21] {{webarchive|url=https://web.archive.org/web/20080607093250/http://www.tj.gov.cn/tblm/tj_sz/200709/t20070921_22653.htm |date=2008-06-07 }}</ref>。

1969年12月4日，国务院、[[中共中央军委|中共中央军委]]发出《关于总参军事气象局与中央气象局合并问题的通知》，决定[[中国人民解放军总参谋部气象水文局|总参谋部军事气象局]]与中央气象局于1970年1月2日正式合并，撤销总参谋部军事气象局，称'''中央气象局'''，并保留总参谋部军事气象局的名称，归[[中国人民解放军总参谋部|总参谋部]]建制。各级气象部门由当地军事部门和地方人民政府双重领导。<ref>[http://www.wyzxwk.com/Article/lishi/2009/09/86994.html 《文革简论》第三篇第一章第九节改革规章制度，乌有之乡网刊，2009-07-06] {{webarchive|url=https://web.archive.org/web/20141214210837/http://www.wyzxwk.com/Article/lishi/2009/09/86994.html |date=2014-12-14 }}</ref><ref name="tjsz" />

1973年3月6日，[[中共中央|中共中央]]同意国务院、中共中央军委《关于调整测绘、气象、邮电部门体制问题的请示》，决定：中央气象局与总参谋部分开，中央气象局划归国务院建制，另行组建[[中国人民解放军总参谋部气象水文局|总参气象局]]；省（自治区、直辖市）以下气象部门划归同级[[革命委员会|革命委员会]]领导<ref name="tjsz" />。1973年5月23日，国务院、中共中央军委发出《关于调整气象部门体制的通知》，正式将中央气象局与总参气象局分开，单独组建总参气象局。<ref name=zgqxj/>

1982年8月23日，中央气象局改称'''中国气象局'''。1994年，中国气象局由[[国务院直属机构|国务院直属机构]]改为[[国务院直属事业单位|国务院直属事业单位]]。

2017年5月，在[[世界气象组织|世界气象组织]]（WMO）执行理事会第69次届会上，中国气象局被正式认定为[[世界气象中心|世界气象中心]]（WMC）<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1702270 |title =世界气象组织执行理事会：正式认定中国气象局为世界气象中心  |author =  |date =  2017-06-06|publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

== 机构编制　==
[[File:1963-05_1963年_中央气象局观测站.jpg|thumb]]
实行“气象部门与地方[[人民政府|人民政府]]双重领导、以气象部门为主”的领导管理体制，国家、省、地、县四级管理。现有31个[[中国一级行政区|省级行政区]]（省、自治区、直辖市）气象局、329个[[地级行政区|地级行政区]]（地级市、地区、自治州、盟）气象局、2155个[[县级行政区|县级行政区]]（县、县级市、旗）气象局,382个国家、省、地气象台,2424个气象观测站（其中艰苦气象站987个，占41%）、120个高空气象观测站、323个天气雷达站、5个气象卫星地面接收站、7个全球和区域大气成分本底观测站等。气象部门现有职工107 372人，国家、省、地、县分别占6.2%、24.1%、32.4%、37.3%,其中在职职工73 176人，离退休人员34 196人。
