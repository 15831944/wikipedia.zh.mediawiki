'''中华人民共和国行政区划代码'''是[[中华人民共和国|中华人民共和国]]用于[[行政区划|行政区划]]上的识別码，同时是一项由国家标准机构——[[中国国家标准化管理委员会|中国国家标准化管理委员会]]通过并公开发布的[[国家标准|国家标准]]，国标号为GB/T 2260，并加上后缀发布年份，如GB/T 2260-2007<ref>{{cite web |title=关于征求GB/T 2260-2007《中华人民共和国行政区划代码》国家标准第1号修改单（报批稿）意见的通知 |url=http://www.sac.gov.cn/szhywb/yjzj/201610/t20161028_218932.htm |website=中国国家标准化管理委员会}}</ref>。

此国家标准于1980年首次发布，并在1982年、1984年、1986年、1988年、1991年、1995年、1999年、2002年及2007年发布更新版本，现时最新版本是GB/T 2260-2007第1号修改单<ref>{{cite web |author1=中国国家标准化管理委员会 |title=关于批准发布GB/T 2260-2007《中华人民共和国行政区划代码》国家标准第1号修改单的公告 |url=http://www.whsis.cn/news/show-5337.html}}</ref>。

== 基本应用 ==
此标准规定了县级或以上行政区划的数字代码（由6位数字组成）和字母代码（省级行政区由2位大写字母组成，地级或县级行政区由3位大写字母组成），适用于对行政区划进行标识、信息处理和交换等<ref>{{cite web |title=中华人民共和国行政区划代码GB/T 2260-2007 |url=http://www.gb688.cn/bzgk/gb/newGbInfo?hcno=C9C488FD717AFDCD52157F41C3302C6D |website=中国国家标准化管理委员会}}</ref>。然而，在一般应用上，数字代码更为普及，例如有以下范畴：
*[[中华人民共和国公安部|公安部]]用于编配[[中华人民共和国公民身份号码|公民身份号码]]（头6位数字的地址码）；
*[[中华人民共和国民政部|民政部]]用于行政区划的标识；
*[[中华人民共和国国家统计局|国家统计局]]用于开展统计调查上的行政区划划分。

数字代码采用三层六位层次码结构，代码从左至右的含义是：
* 第一层（第一、二位）表示[[省级行政区|省级行政区]]（[[省|省]]、[[自治区|自治区]]、[[直辖市|直辖市]]、[[特别行政区|特别行政区]]），其中第一位代表[[大行政区|大区]]。
* 第二层（第三、四位）表示[[地级行政区|地级行政区]]（[[地级市|地级市]]、[[地区_(行政区划)|地区]]、[[自治州|自治州]]、[[盟|盟]]），直辖市的市辖区、县的汇总码，及[[省直辖县级行政区|省（自治区）直辖县级行政区]]的汇总码。
* 第三层（第五、六位）表示[[县级行政区|县级行政区]]（[[县|县]]、[[自治县|自治县]]、[[市辖区|市辖区]]、[[县级市|县级市]]、[[旗_(行政区划)|旗]]、[[自治旗|自治旗]]、[[林区|林区]]、[[特区_(中国县级行政区)|特区]]）。

== 代码的排序、更改与撤销 ==
一般来说，区划代码是按照政区设立的日期{{efn|不适用于第一层（省级）代码，第一层代码的顺序是按照大区分列，以及建国以来国家规定的省级行政区排序而定。}}，依数字的顺序排列的。但早期部分地级市因应市辖区的性质，分为城区与郊区两个类型，而区划代码排列的开始位置亦有所不同。城区代码由“02”开始排列（“01”是地级市市辖区的汇总码），郊区则由“11”开始排列。例如[[武汉市|武汉市]][[江汉区|江汉区]]等6个城区由“420102”排起，[[洪山区|洪山区]]等7个郊区则由“420111”排起。这种排序方式近年已不再使用，但现存此情况的地级市在新设市辖区时，尚未有准则应在城区还是郊区的代码后排下去，造成代码疑问。

单纯的政区更名并不会更改代码。管辖范围有变化，或政区合并且同时沿用原有政区之一名称，一般会沿用原有政区的代码，例如[[上海市|上海市]][[黄浦区|黄浦区]]与[[卢湾区|卢湾区]]合并成新的黄浦区，就沿用了原黄浦区的代码{{efn|合并政区时不沿用原有名称一般不会沿用代码，如2018年[[长治市|长治市]][[城区_(长治市)|城区]]（“140402”）与[[郊区_(长治市)|郊区]]（“140411”）合并组建[[潞州区|潞州区]]（“140403”），潞州区未采用原有两个政区的代码，再如2007年[[乌鲁木齐市|乌鲁木齐市]][[东山区|东山区]]（“650108”）与[[昌吉回族自治州|昌吉回族自治州]][[米泉市|米泉市]]（“652303”）合并组建乌鲁木齐市[[米东区|米东区]]（“650109”），米东区并未采用原有两个政区的代码。}}，但不同类型的政区合并沿用原有政区之一名称，代码可能沿用{{efn|例如2011年[[重庆市|重庆市]][[万盛区|万盛区]]（“500110”）与[[綦江县|綦江县]]（“500222”）合并组建重庆市[[綦江区|綦江区]]，沿用原万盛区代码“500110”。}}，也可能不沿用{{efn|例如2014年[[广东省|广东省]][[茂名市|茂名市]][[茂港区|茂港区]]（“440903”）与[[电白县|电白县]]（“440923”）合并组建茂名市[[电白区|电白区]]（“440904”），电白区并未采用原有两个政区的代码。}}。大部分政区类型改变{{efn|有少数政区类型改变代码不会更改，包括县改设自治县（例如2003年四川省撤销北川县设立[[北川羌族自治县|北川羌族自治县]]时仍保持行政区划代码不变）。}}，例如撤县设区、撤县设市等，代码则需要更改。上级行政区发生变化，例如所属政区撤地区设市，代码也需要更改。上级行政区发生变化，下级行政区间的代码虽改变，但除非下级行政区类型改变，排序一般不变{{efn|例如2013年[[青海省|青海省]]撤销[[海东地区|海东地区]]设立[[海东市|海东市]]，[[乐都县|乐都县]]（“632123”）代码顺序本在[[平安县|平安县]]（“632121”）之后，但由于调整同时乐都县被撤销并设立为[[乐都区|乐都区]]（“630202”），其代码顺序移动至平安县（“630221”）之前。}}{{efn|其中一个例外为直辖的[[重庆市|重庆市]]，[[万县区|万县区]]（万州区）（“500101”）、[[涪陵区|涪陵区]]（“500102”）排在原四川省重庆市下级行政区第一位的[[渝中区|渝中区]]（“500103”）之前，原因可能是考虑到前者二区的前身[[万县市|万县市]]、[[涪陵市|涪陵市]]在重庆直辖前为地级市，而[[黔江土家族苗族自治县|黔江土家族苗族自治县]]（黔江区）虽为[[黔江地区|黔江地区]]行政公署驻地，但不是地级市，因此没有此等地位，只能按顺序排在原涪陵市下属县之后。}}。为保证代码的唯一性，因行政区划变更而撤销的代码一般不会再赋予其他政区使用{{efn|2000年[[四川省|四川省]]撤销[[眉山地区|眉山地区]]设立地级[[眉山市|眉山市]]时，因用作地级市的码段“01−20”不敷应用，因此重新启用了“511400”这个原本赋予给了1995年成立、1997年划归直辖[[重庆市|重庆市]]的地级[[涪陵市|涪陵市]]的代码（《对GB/T 2260-95的补充变更对照表》中已明确将“511400”赋予涪陵市），而未有按照广东省潮州等市的先例，依从规则启用“515100”代码。民政部在整理历史资料时为免混乱，将1995至1997间短暂存在的地级涪陵市给予代码“517000”<ref>{{cite web |title=1996年中华人民共和国行政区划代码（截止1996年12月31日） |url=http://www.mca.gov.cn/article/sj/tjbz/a/201713/201708220914.html |publisher=[[中华人民共和国民政部|中华人民共和国民政部]] |accessdate=2019-01-23}}</ref>。}}。

== 代码的发布与维护 ==
虽然理论上行政区划代码应经由国家标准化管理委员会通过后才公开发布并生效，但实际上行政区划经常有变更，而标准化管理委员会并不是恒常地发布最新的代码，而且审核县级或以上行政区划的设立、命名和变更等是由民政部区划地名司负责。因此，当有县级或以上的区划变更时，民政部会按照上述规则赋予变更后行政区新的区划代码。民政部公布的代码几乎与标准化管理委员会公布的一模一样，但亦有例外。例如2017年第2号修改单中[[海南省|海南省]][[儋州市|儋州市]]与[[三沙市|三沙市]]的代码分別是“460300”、“460400”，正好与民政部公布的互换了排序<ref>{{cite web |author1=全国信息分类与编码标准化技术委员会 |title=关于对《中华人民共和国行政区划代码》国家标准第2号修改单（征求意见稿）征求意见的通知 |url=http://www.cnis.gov.cn/wzgg/201707/t20170706_23032.shtml |website=中国标准化研究院}}</ref>。

在《中华人民共和国行政区划代码》（GB/T 2260）的基础上，国家标准化管理委员会亦发布《县级以下行政区划代码编制规则》作为前者扩充与延拓的配套标准，与GB/T 2260配合使用，国标号为GB/T 10114，于1988年12月首次发布，2003年作第一次修订
<ref>{{cite web |title=县级以下行政区划代码编制规则GB/T 10114-2003 |url=http://www.gb688.cn/bzgk/gb/newGbInfo?hcno=9B7102FB0564BAF69E77C0D4EDE9D5B3 |website=中国国家标准化管理委员会}}</ref>。

== 非正式行政区划的代码编制 ==
理论上，虽然根据上述方法编制的行政区划代码可以涵盖全国所有行政区划，但实际上由于种种原因，各地存在各种“非正式”的行政区划。所谓“非正式”，即是民政部未予承认的行政区划{{efn|民政部鉴于各种“开发区”（或工业园区）不是实际的行政区划，因此，各类“开发区”均不编制县及县以上的行政区划代码。“开发区”中的县以下行政区划，可依据“开发区”范围内的街道、镇、乡和居民委员会、村民委员会的建制编制代码<ref>{{cite web |title=统计上使用的县以下行政区划代码编制规则 |url=http://www.mca.gov.cn/article/sj/xzqh/1980/201507/20150715854849.shtml |website=中华人民共和国民政部}}</ref>。}}，例如各地方政府普遍设立的[[经济开发区|经济开发区]]、[[高新区|高新区]]、[[城市新区|城市新区]]，以及部分因历史原因形成的[[政企合一|政企合一]]单位、[[农垦|农垦]]单位、[[林业|林业]]单位、[[兵团|兵团]]单位等。这些非正式行政区划不会被列入国家标准的行政区划代码中，然而个別部门，例如国家统计局，仍会因应实际需要而赋予这些单位独立的代码。例如2017年版的《统计用区划代码》中列明“为满足统计调查工作组织和数据汇总的需要，国家统计局对一些符合条件的开发区编制了统计汇总识别码”<ref>{{cite web |title=统计用区划代码和城乡划分代码发布说明 |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2017/index.html |website=国家统计局}}</ref>。

=== 县级或副县级 ===
非正式县级或副县级开发区在《统计用区划代码》的县级码段为71−80，唯应同时满足以下条件：
# 国家和省人民政府正式批准成立的开发区；
# 开发区的管理等同于县级人民政府，行使县级人民政府的管理职能，即管理开发区内的社会公共事务；
# 开发区至少管理一个乡级单位；
# 开发区管委会成立并运作两年及以上。

较典型的例子有：
* 江苏省[[苏州市|苏州市]][[苏州工业园区|苏州工业园区]]，统计局赋码320571；
* 湖南省[[怀化市|怀化市]][[洪江管理区|洪江管理区]]，统计局赋码431271；
* 甘肃省[[兰州市|兰州市]][[兰州新区|兰州新区]]，统计局赋码620171。

还有一些较特殊的非正式县级或副县级行政区，例如：
* 黑龙江省[[大兴安岭地区|大兴安岭地区]][[加格达奇区|加格达奇区]]、[[松岭区|松岭区]]、[[新林区|新林区]]、[[呼中区|呼中区]]，统计局分別赋码232761−232764；
* 青海省[[海西蒙古族藏族自治州|海西蒙古族藏族自治州]][[大柴旦行政委员会|大柴旦行政委员会]]，统计局赋码632857。

=== 乡镇级 ===
民政部未确认的开发区、工矿区、农场等“类似乡级单位”，乡级代码为400−599。

较典型的例子有：
* 河北省[[宁晋县|宁晋县]][[大曹庄管理区|大曹庄管理区]]，统计局赋码130528401；
* 安徽省[[安庆市|安庆市]][[皖河农场|皖河农场]]，统计局赋码340803500；
* 湖北省[[沙洋监狱管理局|沙洋监狱管理局]]（实为副厅级单位），统计局赋码420822450；
* [[新疆生产建设兵团|新疆生产建设兵团]]的各个团场，统计局一般赋予乡级代码501−599。

另外，“198”是用来赋给虚拟乡镇级单位的代码，例如部分已由市辖区直管社区的城市，如安徽省[[铜陵市|铜陵市]][[铜官区|铜官区]]，统计局赋予虚拟乡镇代码340705198代表直管社区的汇总。

=== 村级 ===
民政部未确认的园区、工矿区、农场等“类似村级单位”，村级代码为400−599（498、598除外）。
* 400−499表示类似居民委员会（不含498）；
* 500−599表示类似村民委员会（不含598）；
* 498表示虚拟居民委员会；
* 598表示虚拟村民委员会。

另外，“398”是用来赋给乡级单位直接管辖村民小组单位的代码。

== 部门编码的差异 ==
在绝大部分的情况下，县级或以上的行政区划代码在不同部门的系统上都是统一的。出现差异的情况往往是县级以下的行政区划的编码，以及对非正式行政区划的编码。例如[[国家统计局|国家统计局]]赋予黑龙江省[[大兴安岭地区|大兴安岭地区]][[加格达奇区|加格达奇区]]“232701”这个代码，然而当[[漠河县|漠河县]]在2018年撤县设市时，民政部则将“232701”这个代码赋给了[[漠河市|漠河市]]，国家统计局为避免引起混乱，给加格达奇区另赋代码“232761”<ref>{{Cite web|title=2018年统计用区划代码 - 大兴安岭地区|url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2018/23/2327.html|work=www.stats.gov.cn|accessdate=2019-02-14|author=|date=|publisher=}}</ref>。

民政部与统计局在乡级行政区划的代码编制有较多差异，例如以内蒙古自治区[[包头市|包头市]][[达尔罕茂明安联合旗|达尔罕茂明安联合旗]]为例，可以看出两个部门的编码差异。全旗12个乡、镇、苏木中，只有2个镇是两个部门都赋给了同一代码。
