'''中国人民解放军第一八〇医院'''，又名'''华侨大学附属海峡医院'''，位于[[福建省|福建省]][[泉州市|泉州市]][[丰泽区|丰泽区]]花园路180号，隶属[[无锡联勤保障中心|无锡联勤保障中心]]，为[[三级甲等医院|三级甲等医院]]。

==简介==
中国人民解放军第一八〇医院的前身是[[晋冀鲁豫野战军|晋冀鲁豫野战军]]第一纵队休养所，1945年诞生在[[邯郸战役|邯郸战役]]中。1956年，奉命自[[黑龙江省|黑龙江省]][[佳木斯市|佳木斯市]]南迁到福建省泉州市，1963年整编为'''中国人民解放军第一八〇医院'''<ref name=zndx>{{Cite web |url = http://jobsky.csu.edu.cn/Home/NewsDetails/712 |title = 中国人民解放军第一八〇医院 |author =  |date = 2015-12-16 |publisher = 中南大学毕业生就业指导服务中心 |language =  |accessdate =  |quote =  }}</ref>。
[[File:中国人民解放军第一八〇医院.jpg|thumb]]
医院位于泉州市[[清源山|清源山]]南麓，是综合性[[三级甲等医院|三级甲等医院]]。截至2015年，医院拥有1个全军中心（全军眼科创伤救治中心），6个军区专科专病中心（南京军区肝病中心、肝胆外科中心、烧伤中心、口腔颌面外科专病中心、颅脑创伤救治中心、部队卫勤信息化建设支持中心）。医院是[[南方医科大学|南方医科大学]]非直属附属医院，[[中国人民解放军第四军医大学|中国人民解放军第四军医大学]]、[[福建医科大学|福建医科大学]]等军内外医科大学的教学医院。2016年10月20日，“[[华侨大学|华侨大学]]附属海峡医院”揭牌仪式在该医院病房大楼广场前举行，该医院正式成为'''华侨大学附属海峡医院'''<ref name=zndx/>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第一八〇医院由[[中国人民解放军南京军区|中国人民解放军南京军区]]转隶[[无锡联勤保障中心|无锡联勤保障中心]]<ref>{{Cite web |url = http://news.xinhuanet.com/mil/2017-05/18/c_129606950.htm|title =联勤保障部队所属医院改善护理队伍结构 |author =  |date =  2017-05-18 |publisher = 新华网|language =  |accessdate =  |quote =  }}</ref>。
