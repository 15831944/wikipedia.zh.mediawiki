中華民國與甘比亞的接觸始於1960年。<ref name="89年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_7.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1968年11月12日，甘比亞和中華民國[[外交#外交上的承認|建交]]。<ref>{{Cite web |url = http://210.241.75.208/scripts/newsnote/tornado/searcher.exe?p=%A5%CC%A4%F1%A8%C8%B0%EA&property=1%3B576%2C&h=0&l=0&a=15&f=0&z=1&t=0&v=root |title = 民國重要史事檢索 |publisher = [[國史館|國史館]]}}</ref>{{noteTag|《中華民國甘比亞國政府聯合公報》（1968年11月12日發表於台北，載《外交部公報》第33卷（民國五十七年）第4號第42頁）}}

1969年11月5日，於首都[[班竹|班竹]]設立'''中華民國駐甘比亞共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1974年12月14日，甘比亞與[[中華人民共和國|中華人民共和國]]簽署《建交公報》。12月28日，中華民國與甘比亞斷交。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-3.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>12月30日，駐甘比亞大使館關閉。

1995年7月13日，在{{link-en|1994年甘比亞政變|1994 Gambian coup d'état}}後，[[葉海亞·賈梅|葉海亞·賈梅]]成為執政者，與中華民國復交。7月21日，駐甘比亞大使館復館；'''甘比亞共和國駐中華民國大使館'''亦於1996年6月在首都[[台北市|台北]]開館。<ref name="86年外交年鑑"/>

2013年11月14日，[[冈比亚国家领导人列表|甘比亞總統]]賈梅單方面宣佈基於「國家戰略利益」決定和中華民國斷交，並表示「覺得台灣已經不需要我們了」。11月18日，[[中華民國政府|中華民國政府]]也基於維護國家尊嚴及[[活路外交|活路外交]]原則，決定終止與甘比亞之外交關係，並撤離大使館、技術團人員及停止一切雙邊合作計畫<ref name="終止">{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=5028B03CED127255&sms=5ED24855AD8E6C58&s=5FA466F8A2407F09 |author = 公眾外交協調會 |date = 2013年11月18日 |title = 中華民國政府基於維護國家尊嚴及活路外交原則，決定自即日起終止與甘比亞共和國之外交關係 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = https://www.ettoday.net/news/20131115/296372.htm |author = 綜合報導 |date = 2013年11月15日 |title = 邦交國剩22個！甘比亞宣布斷交：覺得台灣不需要我們了 |publisher = [[ETtoday新聞雲|ETtoday新聞雲]]}}</ref>。

2016年3月17日，[[中華人民共和國外交部|中華人民共和國外交部]]發表聲明，與甘比亞伊斯蘭共和國復交{{noteTag|該國曾於2015年12月11日更改國號。2017年1月，改回甘比亞共和國。}}<ref>{{Cite web |url = http://www.chinatimes.com/newspapers/20160318001437-260102 |author =  藍孝威 |date = 2016年3月18日 |title = 北京示警！大陸、甘比亞建交 |publisher = [[中時電子報|中時電子報]]}}</ref>。

邦交期間，中華民國政府曾積極協助甘比亞進行有益國家發展及國計民生之合作計畫，包括[[教育|教育]]、[[農業|農業]]、[[醫療|醫療]]、[[軍事|軍事]]及[[基礎建設|基礎建設]]等項目。<ref name="終止"/>

==經濟==
{{see also|臺灣經濟|{{link-en|甘比亞經濟|Economy of the Gambia}}}}
===貿易===
2017年，甘比亞是中華民國第188大貿易夥伴、第161大[[進口|進口]]夥伴、第191大[[出口贸易|出口]]夥伴。出口至甘比亞的金額為45萬9,046[[美元|美元]]，年增6.164%；自甘比亞進口的金額為65萬1,567美元，年增80.826%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）19萬2,521美元，年減367.149%。※2016年，甘比亞是中華民國第193大貿易夥伴、第168大進口夥伴、第194大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
1999年，有[[台灣商人|台灣廠商]]在甘比亞投資3萬美元設立進出口貿易公司，[[承包|承包]]工程。<ref name="88年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter1-3.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

==援助==
===農業===
1995年11月25日，中華民國派駐甘比亞[[農業技術|農業技術]]團。工作項目包括[[稻米|稻米]]增產、設立[[蔬菜|蔬菜]]生產區、設立[[農業機械|農業機械]]中心以支援稻米及蔬菜生產、指導農民種植稻米及蔬菜，尤其以{{link-en|潮汐灌溉|Tidal irrigation}}大幅降低稻米生產成本，另成立蔬菜產銷合作社，輔導蔬菜外銷。<ref name="88年外交年鑑"/>

資助甘比亞「稻米擴展、蔬菜生產班輔導及[[吳郭魚|吳郭魚]]養殖{{link-en|技術移轉|Technology transfer}}計畫」。其中在甘比亞進行「陸稻擴展計畫」運用台灣農業技術優勢，以4年期（2009－2012年）、3萬2,000[[公頃|公頃]]陸稻推廣、稻穀單位面積產量提高至2.2[[公噸|公噸]]為目標，搭配各類講習與訓練，建立陸稻推廣標準作業流程。{{link-en|甘比亞政治|Politics of the Gambia|甘比亞政府}}也主動提出接續陸稻擴展計畫的「陸稻鞏固計畫」，預定2013－2017年執行，以持續改善該國[[糧食自給率|糧食自給率]]偏低的現況，但因2013年兩國斷交而中止。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/f0a43734-c84a-436d-bd3a-060d93fbbb19.pdf |author = 國際合作及經濟事務司 |date = 2013年4月26日 |title = 國際合作發展事務101年度報告（參、101年度我國國際合作發展事務執行概況；肆、呼應聯合國千禧年發展目標）|publisher = 中華民國外交部}}</ref>

===醫療===
1995年10月，簽署《[[醫療|醫療]]合作協定》，中華民國承諾每年提供醫療援助金給予甘比亞用以採購[[醫療設備|醫療設備]]、[[藥品|藥品]]、聘僱外國醫事人員、派遣醫事人員赴國外研習。<ref name="89年外交年鑑"/>

財團法人國際合作發展基金會（國合會）在甘比亞成立「[[孕婦|孕產婦]]保健功能提升計畫」。2012年，[[馬偕紀念醫院|馬偕紀念醫院]]義診團兩度赴該國實施醫療服務。<ref name="101年外交年鑑"/>
