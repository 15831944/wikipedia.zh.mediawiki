'''中華民國與希臘關係'''是指[[中華民國|中華民國]]與[[希臘共和國|希臘共和國]]之間的關係。1929－1972年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，[[中華民國政府|中華民國政府]]於[[首都|首都]][[雅典|雅典]]設立[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]一案棄權，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[希臘王國|希臘王國]]。]]
{{see also|中華民國外交|{{link-en|希臘外交|Foreign relations of Greece}}}}
{{see also|中国驻希腊大使列表{{!}}中華民國駐希臘外交官列表|希腊驻华大使列表{{!}}希臘駐中華民國外交官列表}}
1929年9月30日，希臘與中華民國建立公使級[[外交#外交上的承認|外交關係]]。於首都[[雅典|雅典]]設立'''中華民國駐[[希臘第二共和國|希臘第二共和國]][[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1947年11月19日，建立大使級外交關係。公使館升格為'''中華民國駐[[希臘王國|希臘王國]][[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1957年3月22日，總統[[蔣中正|蔣中正]]獲[[希臘國王|希臘國王]][[保羅一世_(希臘)|保羅一世]]頒授{{link-en|大十字救世主勳章|Order of the Redeemer}}。

1960年1月14日，希臘恢復在臺大使館。11月18日，首任[[希臘駐華大使列表|駐華大使]]顧思達呈遞到任[[國書|國書]]。<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|654}}。

1972年6月5日，與中華民國斷交。<ref name="103年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/09.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>8日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

===投資===
