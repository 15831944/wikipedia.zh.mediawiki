'''中華民國與喀麥隆關係'''是指[[中華民國|中華民國]]與[[喀麥隆共和國|喀麥隆共和國]]之間的關係。兩國於1960－1971年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對喀麥隆的相關事務由'''{{Link-en|駐奈及利亞聯邦共和國臺北貿易辦事處|Taipei Trade Office in the Federal Republic of Nigeria}}'''、'''[[駐法國台北代表處|駐法國台北代表處]]'''、'''[[駐歐盟兼駐比利時代表處|駐歐盟兼駐比利時代表處]]'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-206-196e3-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[喀麥隆|喀麥隆聯邦共和國]]。]]
{{see also|中華民國外交|喀麥隆外交}}
1960年2月19日，建立大使級[[外交#外交上的承認|外交關係]]。於首都[[雅溫德|雅溫德]]設立'''中華民國駐喀麥隆聯邦共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1971年4月3日，與中華民國斷交。5日，關閉大使館，<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>至今邦交未曾恢復。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境喀麥隆。]]
[[File:Visa_requirements_for_Cameroonian_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|喀麥隆簽證政策|Visa policy of Cameroon}}|{{link-en|喀麥隆公民簽證要求|Visa requirements for Cameroonian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]與[[黃熱病|黃熱病]][[疫苗接種|疫苗接種]]證明的[[中華民國國民|中華民國公民]]可委託當地人、[[關係人|關係人]]或[[旅行社|旅行社]]，向該國國家安全總署邊境警察局（Delegation Generale a la Surete Nationale，Direction de la Police des Frontieres）申請「[[簽證#落地签证|落地簽證]]函」（Visa d′Entree au Debarquement），持函在抵達{{link-en|喀麥隆機場列表|List of airports in Cameroon|該國機場}}時辦理落地簽證入境喀麥隆，停留最多30天，不可延長。簽證相關事宜，可至喀麥隆駐[[日本|日本]][[外交代表機構#大使馆|大使館]]詢問。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-206-196e3-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|喀麥隆護照|Cameroonian passport}}的{{link-en|喀麥隆人口|Demographics of Cameroon|喀麥隆公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將喀麥隆按地區歸類為'''橙色警戒'''：避免非必要旅行，與'''紅色警戒'''：不宜前往，宜儘速離境（以上截至2018年10月15日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-274-33bc6-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將喀麥隆歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=796E421196566EF1 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|喀麥隆經濟|Economy of Cameroon}}}}
===貿易===
2017年，喀麥隆是中華民國第149大貿易夥伴、第137大[[進口|進口]]夥伴、第139大[[出口贸易|出口]]夥伴。出口至喀麥隆的金額為616萬7,507[[美元|美元]]，年增23.094%；自喀麥隆進口的金額為297萬0,068美元，年減8.157%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）319萬7,439美元，年增79.979%。※2016年，喀麥隆是中華民國第150大貿易夥伴、第131大進口夥伴、第142大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
