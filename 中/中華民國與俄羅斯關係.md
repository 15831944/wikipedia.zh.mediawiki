'''中華民國與俄羅斯關係'''是指[[中華民國|中華民國]]與[[俄羅斯聯邦|俄羅斯聯邦]]（通稱'''俄羅斯'''、'''俄國'''）之間的關係。1913－1949年，以俄羅斯為主體的[[俄羅斯帝國|俄羅斯帝國]]／[[蘇聯|蘇聯]]與中華民國有官方[[外交#外交上的承認|外交關係]]，期間多次斷交與復交。1949年10月3日，苏联是第一個與[[中华人民共和国|中华人民共和国]]建交，並與中华民国斷交的國家。1991年[[苏联解体|苏联解体]]后，俄罗斯联邦繼承苏联只承认中华人民共和国的立场，但和中华民国於[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構，雙邊在經貿等非官方的往來也如常。

==中华民国与帝俄／苏联关系==
{{see also|中蘇關係|中華民國外交|{{link-en|蘇聯外交|Foreign relations of the Soviet Union}}}}

1912年，中華民國成立後，向[[俄羅斯帝國|俄羅斯帝國]]派駐外交代表。

1913年10月6日，俄羅斯帝國[[外交#外交上的承認|外交承認]]中華民國[[北洋政府|北洋政府]]。於首都[[聖彼得堡|聖彼得堡]]設立'''中華民國駐俄羅斯帝國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1918年2月26日，[[十月革命|十月革命]]及其後的[[俄國內戰|俄國內戰]]，使得與[[中華民國|中華民國]]邦交中斷。

1924年5月31日，新成立不久的[[蘇聯|蘇聯]]與中華民國恢復邦交並升格為大使級外交關係，但依舊派駐蘇聯公使。

1929年7月18日，[[中東路事件|中東路事件]]導致蘇聯與中華民國第二次斷交。

1932年12月12日，恢復大使級外交關係。於首都[[莫斯科|莫斯科]]設立'''中華民國駐蘇維埃社會主義共和國聯盟[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

自1930年代起，苏联势力就已染指新疆地区，当时主政[[新疆省_(中華民國)|新疆]]的[[盛世才|盛世才]]在苏联政府和民国政府之间摇摆。最终在苏联支持下，1944年在新疆多地爆发武装叛乱，而其中[[伊宁武装暴动|伊宁武装暴动]]又成为[[三区革命|三区革命]]开始的标志。

1949年10月1日，[[中華人民共和國開國大典|中华人民共和国成立]]，3日，苏联與[[中华人民共和国|中华人民共和国]]建交（是最早承認並建交的國家），與中华民国第三次斷交。11日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>12月，[[臺灣戰後時期|中華民國政府遷台]]。

1950年11月30日，蘇聯在[[聯合國安全理事會|聯合國安全理事會]]上支持中共代表[[伍修權|伍修權]]等提出之「[[美國|美國]]侵略[[台灣|台灣]]案」，為該案唯一的贊成國。<ref>{{cite conference|title=聯合國安全理事會正式紀錄 第七十二號 第五三0次會議|date=1950-11-30|publisher=[[聯合國安全理事會|聯合國安全理事會]]|location=[[紐約|紐約]]發拉星草場|url=http://www.un.org/en/ga/search/view_doc.asp?symbol=S/PV.530&referer=/english/&Lang=C|accessdate=2015-08-24|language=zh-hant}}</ref>

1952年，[[中華民國政府|中華民國政府]]向[[聯合國|聯合國]]提出《[[聯合國大會505號決議|聯合國大會505號決議]]》（控蘇案）獲得通過，次年經[[立法院|立法院]]通過廢除《[[中蘇友好同盟條約|中蘇友好同盟條約]]》。

1950－1990年代東西方陣營的「[[冷戰|冷戰]]」期間，中華民國在[[意識形態|意識形態]]和[[政體列表|制度]]上均偏向與[[美國|美國]]為首的[[資本主義|資本主義]]陣營，復加上[[蘇聯|蘇聯]]本身的封閉，使雙方幾乎沒有往來。以及中華民國政府普遍認為在[[第二次國共內戰|國共內戰]]中敗退且失去中國大陸的原因，就在於蘇聯對[[中國共產黨|中國共產黨]]的長期資助和扶持，遂在台灣長期宣揚「[[反共抗俄|反共抗俄]]」的立場，不斷在教育及社會各層面貶抑蘇聯及[[共產主義|共產主義]]的形象，並嚴禁相關事物在臺灣的傳播。[[蔣中正|蔣中正]]本人也親自擴充《[[蘇俄在中國|蘇俄在中國]]》一書，詳述俄國自[[清朝#近代開啟|清末]]以來對華之侵略歷史，以做為對蘇聯的警惕和提防。此一時期的教育宣傳至今仍影響台灣社會對俄國的觀感。

==中華民國與俄羅斯聯邦關係==
===外交===
{{see also|{{link-en|俄羅斯外交|Foreign relations of Russia}}}}
1991年，[[苏联解体|苏联解体]]、[[獨立國家國協|獨立國家國協]]（獨立國協）成立，[[俄罗斯联邦|俄罗斯联邦]]繼承[[苏联|苏联]]只承认[[中华人民共和国|中华人民共和国]]的立场。

1999年9月21日，[[九二一大地震|台灣發生大地震]]，[[俄羅斯聯邦政府|俄羅斯聯邦政府]]派遣緊急救援小組70人，搭乘另有10名機員的軍用[[伊留申76|伊留申76]]專機運送重型車3輛和30噸搜救裝備於9月22日下午抵達[[台中市|台中]][[清泉崗機場|清泉崗機場]]協助救災，一降落之後馬上趕往[[東勢區|東勢鎮]]東勢王朝大樓的倒塌現場，成為首度降落台灣的俄國軍機。<ref>[各國救援隊到俄軍機首降台灣美、英、德等十三國五百多人抵台]【1999-09-23/[[聯合報|聯合報]]/10版/集集大震　特別報導】</ref>俄羅斯救難隊一行共計83人，實際投入救援工作的有73人。

2000年8月12日，俄羅斯奧斯卡級[[核子動力潛艇|核子動力潛艇]]「[[庫斯科號|庫斯科號]]」在[[巴倫支海|巴倫支海]]沉沒，艦上118名官兵全數罹難。29日，[[中華民國外交部|中華民國外交部]]致電俄羅斯聯邦政府表達哀悼及慰問之意；總統[[陳水扁|陳水扁]]代表政府及人民捐贈20萬美元給予罹難官兵的家屬。<ref name="89年外交年鑑"/>

2002年7月，以增進台俄雙方人民的友誼和合作，並促進台灣與俄羅斯及其他獨立國協國家在經貿、投資、科技、文化、教育、觀光等領域的交流之全國性之[[法人#社團法人|社團法人]]「[[臺俄協會|臺俄協會]]」成立<ref>[http://www.tra.org.tw/03-about/about_01.htm 台俄協會簡介]</ref>。

持有[[中華民國護照|中華民國護照]]與邀請函的[[中華民國國民|中華民國公民]]前往俄羅斯，觀光簽證停留期限以邀請函及訂房日數而定；商務簽證停留期限為每180天內總計不可超過90天。抵達俄羅斯後7個工作日內，必須前往或郵寄至移民局辦理「抵俄報告通知書」，停留期間應隨身攜帶以便查驗。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-43-b55c4-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref><ref>{{Cite web |url = http://www.mtc.org.tw/chinese/consular.html |title = 簽證組|publisher = [[莫斯科台北經濟文化合作協調委員會|莫斯科台北經濟文化協調委員會]]駐台北代表處}}</ref>此外，若持有[[中华人民共和国|中华人民共和国]]签发的[[中华人民共和国护照|一次性临时护照]]赴俄，中華民國[[内政部移民署|内政部移民署]]将按照《[[两岸人民关系条例|两岸人民关系条例]]》，依法注销中華民國户籍<ref>{{cite web|title=持中國大陸臨時護照赴俄令台男失戶籍 事主批條例過嚴 |url=http://www.bbc.com/zhongwen/trad/chinese-news-41897486 |publisher=[[BBC中文网|BBC中文网]] |date=2017-11-07 |accessdate=2017-11-08}}</ref>。

持有[[俄羅斯護照|俄羅斯護照]]的[[俄罗斯人口|俄羅斯公民]]則可以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多14天，期限至2019年7月31日。<ref>{{Cite web |url = https://www.chinatimes.com/realtimenews/20180906001700-260407 |author =  楊孟立 |date = 2018年9月6日 |title = 拚觀光 外交部：給予俄羅斯14天免簽 |publisher = 中時電子報}}</ref><ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於首都兼第1大城[[莫斯科|莫斯科]]、第2大城[[聖彼得堡|聖彼得堡]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#EUROPE |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]也於莫斯科設立'''駐俄羅斯代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2014年，俄羅斯[[個人電腦|個人電腦]]市場主要[[品牌|品牌]]依序為[[聯想集團|聯想集團]]（Lenovo，21.6%）、[[華碩電腦|華碩電腦]]（ASUS，15.6%）、[[惠普|惠普]]（HP，14%）、[[宏碁|宏碁]]（Acer，12.2%）。其中[[桌上型電腦|桌上型電腦]]品牌依序為HP、Lenovo、DNS、DEPO、Acer；[[筆記型電腦|筆記型電腦]]品牌依序為Lenovo、ASUS、Acer、HP、[[戴爾|戴爾]]（DELL）。當年度俄羅斯的品牌[[知名度|知名度]]中，ASUS排名第14。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/ru/pdf/104-61Russia.pdf |date = 2015年8月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

2017年，俄羅斯是中華民國第21大貿易夥伴、第16大[[進口|進口]]夥伴、第26大[[出口贸易|出口]]夥伴。出口至俄羅斯的金額為10億9,012萬9,115[[美元|美元]]，年增25.760%；自俄羅斯進口的金額為33億0,674萬9,607美元，年增27.060%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）22億1,662萬0,492美元，年增27.710%。※2016年，俄羅斯是中華民國第21大貿易夥伴、第16大進口夥伴、第28大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =經濟部國際貿易局}}</ref>

貿易過程中，中華民國未向俄羅斯進口[[石油|石油]]與[[天然氣|天然氣]]。

===投資===
根據經濟部投資審議委員會統計，1993年－2014年，[[台灣商人|台商]]對俄羅斯的總投資金額為2,711萬美元，計有9件（與實際有差異，是因雙方貿易仍有透過第3國）。<ref name="投資環境簡介"/>

目前常駐俄羅斯的台商約20餘家。在[[聖彼得堡|聖彼得堡]]有2家：[[長榮海運|長榮海運]]、[[陽明海運|陽明海運]]從事[[船運|船運]]服務；[[遠東聯邦管區|遠東地區]]（[[海參崴|海參崴]]）有1－2家；首都[[莫斯科|莫斯科]]則佔絕大多數，主要從事[[貿易|貿易]]、[[旅遊|旅遊]]或[[市場行銷|行銷]]。台灣資訊電腦廠商如宏碁、華碩、[[富智康集團|富智康集團]]（[[鴻海科技集團|鴻海科技集團]]的[[子公司|子公司]]）、[[宏達國際電子|宏達國際電子]]、[[明基電通|明基電通]]、[[技嘉科技|技嘉科技]]、[[微星科技|微星科技]]、[[威剛科技|威剛科技]]、[[訊舟科技|訊舟科技]]、[[研華科技|研華科技]]、[[全漢|全漢]]等公司均在莫斯科設立代表[[辦事處|辦事處]]或分支機構。[[裕隆汽車|裕隆汽車]]設立據點以組裝與銷售旗下品牌[[納智捷汽車|納智捷汽車]]，拓展俄國汽車市場。此外，台商也參與[[賭博|博弈]]特區第一家外商投資[[賭場|賭場]]計畫。<ref name="投資環境簡介"/>

目前已成立俄羅斯台灣商會。<ref name="投資環境簡介"/>

雙邊舉辦的會議有：台俄經濟合作會議、台俄雙邊企業[[執行長|執行長]]論壇、台俄合作論壇。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref><ref name="95年外交年鑑"/>

===學術===
{{see also|臺灣教育|俄羅斯教育}}
[[中華民國教育部|中華民國教育部]]自1991年7月設立「[[獨立國家國協|獨立國家國協]]及東歐地區國家來華留學[[獎學金|獎學金]]」，其中包括俄羅斯。後改為教育部的「[[中華民國國語|華語文]]獎學金」，以及[[中華民國外交部|外交部]]的「台灣獎學金」，提供俄羅斯學生申請赴台研習中文或攻讀[[碩士|碩]][[博士|博士]]。截至2016年，台灣留俄學生有178人，主要集中在[[莫斯科|莫斯科]]及[[聖彼得堡|聖彼得堡]]；在臺的俄羅斯留學生則有375人，攻讀學位及研修語言者各佔半數。此外，有多位華語文教師在[[俄罗斯大学列表|俄羅斯各大學]]任教，推動[[正體字|正體中文]]教學。<ref name="105年外交年鑑"/>

1999年12月，俄羅斯[[聖彼得堡國立大學|聖彼得堡國立大學]]校長薇兒碧芝卡亞（Verbitskaya）與副校長抵台訪問，期間頒贈中華民國副總統[[連戰|連戰]][[榮譽博士學位|榮譽博士學位]]，並參加[[中國文化大學|中國文化大學]]舉辦的國際學術會議。<ref name="88年外交年鑑"/>

2006年11月，[[國立政治大學|國立政治大學]]舉辦「[[俄羅斯文學|俄羅斯文學]]傳統與創新」國際[[學術研討會|學術研討會]]、「台北莫斯科[[論壇|論壇]]：俄羅斯在[[東北亞|東北亞]]的角色」（2008年11月續辦<ref name="97年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2008/html/08-1-2.htm |title = 《中華民國97年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>）<ref name="95年外交年鑑"/>。2007年5月，舉辦「獨立國協國家民主發展的挑戰與展望」學術研討會。<ref name="95年外交年鑑"/>

2002年4月設立'''駐俄羅斯代表處科技組'''後，便積極拓展臺俄[[科技|科技]]合作、共同執行各項協定與[[備忘錄|備忘錄]]，除了科技合作之外，還包括[[人文|人文]]及[[社會科學|社會科學]]領域。自2008年起，[[行政院國家科學委員會|行政院國家科學委員會]]（後升格為科技部）與俄羅斯基礎研究基金會、人文科學基金會、科學基金會，以及[[俄羅斯科學院|俄羅斯科學院]]西伯利亞分院、遠東分院共同補助多年期研究計畫案，以及舉行研討會。<ref name="97年外交年鑑"/><ref name="105年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

===航空===
1993年7月，雙方[[航權|航權]]代表團於莫斯科磋商台俄空運通航事宜，並簽署《台俄[[民航|民航]]合作協定》。1997年9月3日，透過台北莫斯科經濟文化協調委員會（北莫協）與莫斯科台北經濟文化協調委員會（莫北協）的架構下完成換函生效。9月4日，由雙方外交部同時宣布台北桃園↔莫斯科正式通航，雙方指定的航空公司為台灣的[[中華航空|中華航空]]與俄羅斯的[[全祿航空|全祿航空]]。<ref name="86年外交年鑑"/>但俄羅斯要求兩家航空公司須先簽署商業協定（Commercial Agreement），並由民航主管機關核准，以致通航計畫延宕。<ref name="89年外交年鑑"/>2001年4月，中華航空和[[長榮航空|長榮航空]]獲飛越西伯利亞[[領空|領空]]權。<ref name="90年外交年鑑"/>

====客運====
2001年4月，長榮航空與[[海參崴航空|海參崴航空]]首度以[[包機|包機]]直飛[[臺灣桃園國際機場|台北桃園]]↔[[海參崴國際機場|海參崴]]。<ref name="90年外交年鑑"/>2002年8月－2003年底，全祿航空與中華航空開辦台北桃園↔[[伏努科沃國際機場|莫斯科伏努科沃]]包機航線。<ref name="91年外交年鑑"/>2014年、2015年，全祿和華航（[[班號共用|共用全祿班號]]）再度開辦台北桃園↔莫斯科伏努科沃短期定期航班。
