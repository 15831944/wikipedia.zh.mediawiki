'''中国[[移动电话|移动电话]]产业'''是目前世界上发展最快的行业之一。[[中国移动|中国移动]]、[[中国联通|中国联通]]和[[中国电信|中国电信]]是中国三大移动电话网络服务提供商。

==发展过程==
1988年，中国[[GSM|GSM]]網絡建成<ref>[http://baike.baidu.com/view/9113.htm?fr=ala0_1_1 Baidu Encyclopedia: China Mobile Communications Corporation's History] (in Chinese)</ref> 。

联想移动成立于2002年。2005年10月，联想手机市场份额升至中国手机整体市场第四、国产手机厂商第一。2007年3月，推出首款中文版Windows Mobile 6电脑手机ET600。2014年1月30日，[[联想集团|联想集团]]以29亿美元收购[[摩托罗拉移动|摩托罗拉移动]]<ref>[http://investor.google.com/releases/2014/0129.html Lenovo to Acquire Motorola Mobility from Google]</ref><ref>[http://finance.huanqiu.com/data/2014-01/4803197.html 联想宣布29亿美元收购摩托罗拉移动]</ref>。2015年8月27日，联想移动业务并入摩托罗拉<ref>[http://www.ce.cn/cysc/tech/gd2012/201508/28/t20150828_6344347.shtml 联想品牌手机退出江湖 国产智能机放弃机海战术]</ref>。

[[华为技术有限公司|华为技术有限公司]]于2003年7月成立手机业务部。2009年2月，华为展示了首款Android智能手机。2013年三季度华为手机出货量1270万部，以4.8%的市场份额跻身全球第三。

[[小米科技|小米科技]]于2010年4月6日成立<ref name=":0">{{Cite web|url = http://money.163.com/14/1031/09/A9SHL81G00253G87.html|title = 小米魏来：关于小米的趣史和八大创始人|accessdate = 2016-02-18|author = |date = 2014-10-31|publisher = 网易财经}}</ref>。2011年，發佈第一款「小米手機」<ref>{{Cite web|url = http://company.cnstock.com/company/scp_dsy/tcsy_tt/201309/2729536.htm|title = 公司估值达100亿美元 雷军独家解密小米定位|accessdate = 2016-02-18|author = 温婷|date = 2013-09-06|publisher = 上海证券报}}</ref>。2015年，小米成為中國前三大智能手機廠商之一和全球最大智能手機廠商之一<ref>{{Cite web|url = http://www.thepaper.cn/newsDetail_forward_1432469|title = 小米、华为、苹果夺去年中国手机市场前三，三星跌出前五|accessdate = 2016-02-20|author = 周玲|date = 2016-02-16|publisher = 澎湃新闻}}</ref>。

[[2008年中國电信业重組|2008年中國电信业重組]]将6家电信业[[国有企业|国有企业]]合併为3家，并为其发放三种制式的[[3G|3G]]牌照<ref>{{cite web|author1=何君|title=中国电信业十年历经四次重组|url=http://www.cww.net.cn/news/html/2009/11/4/2009114835379064.htm|website=证券日报|accessdate=2016-06-21|date=2009年11月4日}}</ref>。

==专利纠纷==
*2016年5月[[华为|华为]]在美国起诉[[三星|三星]]，称其侵犯了它的11项手机专利权<ref>[http://tech.qq.com/a/20160627/056755.htm 中国手机制造商大举获取专利权 争夺市场优势]</ref>。
*2016年深圳市佰利营销服务有限公司状告苹果公司的〝[[iPhone_6|iPhone 6]]〞和〝[[iPhone_6_Plus|iPhone 6 Plus]]〞的外观设计侵犯了佰利公司的100C手机。5月10日，北京市知识产权局认定苹果公司的两款手机侵权，并责令其停止销售<ref>[http://www.ntdtv.com/xtr/gb/2016/06/20/a1272267.html 被判外观抄袭中国手机 苹果称目前不受影响]</ref>。
*2016年6月[[高通|高通]]向北京知识产权法院起诉[[魅族|魅族]]<ref>[http://tech.sina.com.cn/zl/post/detail/t/2016-06-27/pid_8507821.htm 高通诉魅族背后：中国手机产业创新战略的缺失？]</ref>。
