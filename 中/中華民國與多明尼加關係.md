'''中華民國與多明尼加關係'''是指[[中華民國|中華民國]]與[[多明尼加共和國|多明尼加共和國]]之間的關係。兩國於1941－2018年有正式[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對多明尼加的相關事務由'''中華民國駐[[瓜地馬拉|瓜地馬拉]][[中華民國駐外機構列表|大使館]]'''、'''[[駐邁阿密臺北經濟文化辦事處|駐邁阿密臺北經濟文化辦事處]]'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-91-bf4b3-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[多明尼加|多明尼加]]。]]
{{see also|中華民國駐多明尼加大使列表{{!}}中華民國駐多明尼加外交官列表|多明尼加駐中華民國大使列表{{!}}多明尼加駐中華民國外交官列表}}
{{see also|2018年中華民國邦交國斷交與代表機構更名事件}}
1932年，[[中国驻古巴大使列表|中華民國駐古巴公使]]赴多明尼加視察{{link-en|多明尼加華人|Ethnic Chinese in the Dominican Republic|僑務}}，並洽議兩國間之友好條約。1940年5月11日，駐[[古巴|古巴]][[公使|公使]]與多明尼加政府簽署《中多友好條約》。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1941年4月9日，中華民國與多明尼加建立公使級[[外交#外交上的承認|外交關係]]。<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

1943年7月12日，在首都[[聖多明哥|聖多明哥]]設立'''中華民國駐多明尼加共和國[[外交代表機構#公使馆|公使館]]'''。1944年4月20日，正式開館。<ref name="外交年鑑"/>先由駐古巴公使兼任，<ref name="86年外交年鑑"/>1947年，派駐專任公使。

1946年10月，於首都[[南京市_(中華民國)|南京]]設立'''多明尼加共和國駐中華民國公使館'''。1953年3月8日，在[[臺北市|臺北]]復館。<ref name="外交年鑑"/>

1957年6月8日，多明尼加派駐首任駐華[[大使|大使]]。8月20日，正式升格為大使級外交關係，兩國公使館升格為'''[[外交代表機構#大使馆|大使館]]'''。<ref name="外交年鑑"/>

1995年8月，{{link-en|多明尼加參議院|Senate of the Dominican Republic}}通過決議，支持[[中國與聯合國#代表權問題|中華民國參與聯合國]]。9月，{{link-en|多明尼加眾議院|Chamber of Deputies (Dominican Republic)|眾議院}}亦通過有關決議。<ref name="86年外交年鑑"/>

1997年9月，{{link-en|多明尼加外交部長列表|Foreign Ministers of the Dominican Republic|多明尼加外交部長}}拉多雷（Eduardo Lattorre）於[[聯合國大會|聯合國大會]]發言支持中華民國參與聯合國。<ref name="86年外交年鑑"/>

2002年12月11日，多明尼加眾議院通過決議，呼籲[[中共|中共]]撤除[[中國人民解放軍對臺灣飛彈|對台灣的飛彈]]。另決議支持[[歐洲議會|歐洲議會]]有關要求[[世界衛生組織|世界衛生組織]]（WHO）接納台灣為{{link-en|觀察員身分|Observer status|觀察員}}之立場。<ref name="91年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__06.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2005年5月，多明尼加在[[世界衛生大會|世界衛生大會]]（WHA）提案邀請中華民國以觀察員身分與會，並於WHA總務委員會發言支持。同年底，多明尼加外交部長{{link-en|莫拉雷斯|Carlos Morales Troncoso}}（Carlos Morales Troncoso）在第60屆聯合國大會發言「要求[[台海兩岸|台海兩岸]]以對話代替衝突，以利雙方人民發展」。<ref name="94年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2005/WebPages/MICO/08/08_01_06.htm |title = 《中華民國94年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2018年5月1日，多明尼加與中華民國斷交，結束長達77年的外交關係。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=8742DCE7A2A28761&s=6A57DEE2B1115B9E |author = 公眾外交協調會 |date = 2018年5月1日 |title = 中華民國政府基於維護國家尊嚴，決定自即日起終止與多明尼加的外交關係 |publisher = 中華民國外交部}}</ref>當日，駐多明尼加大使館舉行[[中華民國國旗|降旗]]儀式。<ref>{{Cite web |url = https://udn.com/news/story/6656/3121282 |author = 陳煜彬 |date = 2018年5月3日 |title = 影／多明尼加大使館降旗 外交部：環教扎根不撤 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref>

兩國邦交不穩傳聞已久，2017年底，中華民國援贈多明尼加價值3,500萬[[美元|美元]]的[[中華民國國軍|國軍]]除役裝備，包括90輛[[悍馬_(軍用汽車)|悍馬車]]、2架[[UH-1直升機|UH-1H直升機]]等。2018年初，首批50輛悍馬車運抵多明尼加。相關人士指出，多明尼加玩兩面手法，先把武裝設備騙到手才宣布斷交，目的就是為了取得[[中華人民共和國|中華人民共和國]]支持該國爭取[[聯合國安全理事會成員國列表|聯合國非常任理事國]]的席次。多明尼加曾兩度爭取非常任理事國皆失敗，2018年捲土重來，中華人民共和國的支持十分關鍵，也確實成功當選。<ref>{{Cite web |url = https://tw.appledaily.com/headline/daily/20180502/38001369 |author = 綜合報導 |date = 2018年5月2日 |title = 多明尼加斷交 我10億軍援敗給中國920億利誘 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref>

2018年9月7日，{{Link-en|美國國務院發言人|Spokesperson for the United States Department of State}}發布聲明，國務院已召回{{Link-en|美國駐多明尼加大使列表|List of ambassadors of the United States to the Dominican Republic|駐多明尼加大使}}{{noteTag|以及2018年與中華民國斷交的{{Link-en|美國駐薩爾瓦多大使列表|List of ambassadors of the United States to El Salvador|駐薩爾瓦多大使}}、2017年斷交的{{Link-en|美國駐巴拿馬大使列表|List of ambassadors of the United States to Panama|駐巴拿馬臨時代辦}}。}}，商討該國最近與台灣斷交的決定。<ref>{{Cite web |url = https://udn.com/news/story/6809/3355682 |author = 張加 |date = 2018年9月8日 |title = 不滿與台斷交 美召回駐巴國、多國、薩國使節 |publisher = 聯合新聞網}}</ref><ref>{{Cite web |url = http://www.peoplenews.tw/news/65607481-4591-46b9-a501-806823d8f1a8 |author = 綜合報導 |date = 2018年9月8日 |title = 挺台灣！美國下令召回與台斷交三國大使 |publisher = [[民報_(2014年)|民報]]}}</ref>

持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境多明尼加，停留最多30天。<ref name="簽證及入境須知">{{Cite web|url = https://www.boca.gov.tw/sp-foof-countrycp-03-91-bf4b3-02-1.html|title = 簽證及入境須知|publisher = 外交部領事事務局|accessdate = 2018-01-10|deadurl = no|archiveurl = https://web.archive.org/web/20180128132453/https://www.boca.gov.tw/sp-foof-countrycp-03-91-bf4b3-02-1.html|archivedate = 2018-01-28}}</ref><ref>{{Cite web|url =http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=ABBF62618F53F8DE|title =免簽證資訊|publisher =中華民國外交部}}</ref>

==經濟==
{{see also|臺灣經濟|多明尼加經濟}}
===貿易===
2018年，多明尼加是中華民國第68大貿易夥伴、第63大[[進口|進口]]夥伴、第61大[[出口贸易|出口]]夥伴。出口至多明尼加的金額為1億4,735萬9,205[[美元|美元]]，年增18.709%；自多明尼加進口的金額為1億1,315萬0,738美元，年增54.375%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）3,420萬8,467美元，年減32.711%。2017年，多明尼加是中華民國第70大貿易夥伴、第72大進口夥伴、第65大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
根據前駐多明尼加[[外交代表機構#大使馆|大使館]]經濟[[參事|參事]]處統計，截至2017年，[[台商|台商]]在多明尼加投資共57家，總投資額約1億7,000萬美元。投資的產業包括：[[太陽能發電|太陽能發電]]、[[鞋|製鞋業]]、[[塑膠瓶|塑膠瓶]]及[[塑膠袋|塑膠袋]]、[[衛生紙|衛生紙]]及[[建材|建材]]等之加工及製造，[[汽車零組件列表|汽機車零配件]]、[[電腦|電腦]]及周邊設備等產品之進口及銷售；根據[[中華民國經濟部|中華民國經濟部]]投資審議委員會統計，截至2018年3月，多明尼加在臺投資共6件，總投資額314萬美元。投資的產業包括：[[電子元件|電子零組件]]製造業、[[批發|批發]][[零售業|零售業]]、[[住宿|住宿]]及[[餐飲業|餐飲業]]。<ref>{{Cite web|url=https://www.trade.gov.tw/App_Ashx/File.ashx?FilePath=../Files/Doc/8dca1dfa-a859-4f9b-8735-6e2e06adc576.pdf|title=多明尼加共和國(Dominican Republic) 國家檔 |publisher=經濟部國際貿易局}}</ref>

===合作===
斷交前，中華民國派駐技術團協助多明尼加進行[[水稻|水稻]]改良、[[竹|竹]]栽培、竹屋及[[竹織工藝品|竹工藝]]訓練、[[園藝栽培|園藝栽培]]、[[淡水魚|淡水魚]]苗培育暨箱網[[養殖漁業|養殖]]、以及[[海蝦養殖|海蝦養殖]]等，另派駐工業服務團協助輔導該國[[中小企業|中小企業]]發展。為協助該國縮短[[數位落差|數位落差]]，並培訓數位科技人才，於2004年在多明尼加美洲技術學院設立職業訓練中心、提供現代化數位科技設備、定期派遣專家赴該國講授[[数控机床|CNC]]機械加工、[[模具|模具]]製造、[[自動控制|自動控制]]、[[工业电子系统|工業電子]]、電腦輔助[[机械制图|机械制图]]、[[電腦輔助設計|電腦輔助設計]]與[[電腦輔助製造|製造]]等課程。<ref name="94年外交年鑑"/>駐團與相關計畫已於2013年底結束。<ref name="102年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2013/11.pdf |title = 《中華民國102年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>另於2015年6月起執行「多明尼加竹產業發展計畫」、2016年1月起執行「多明尼加微中小型企業輔導體系能力建構計畫」。<ref name="外交年鑑"/>
