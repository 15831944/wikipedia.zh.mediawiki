'''中国省级行政区人类发展指数列表'''列出[[中國|中國]]各[[中国一级行政区|省级行政区]]（省、自治区、直辖市与特别行政区）根据[[人类发展指数|人类发展指数]]的排名表。数据来源为[[联合国开发计划署|联合国开发计划署]]出版的《2016中国人类发展报告》{{RefTag|name=16年|1={{Cite web |url = https://issuu.com/undp-china/docs/undp-ch-_nhdr_2016_ch|format=PDF |title=《2016中国人类发展报告》 |year=2016 |publisher=[[联合国开发计划署|联合国开发计划署]]中国分部 |language=zh-cn |accessdate=2016年9月1日 |page=第97页}}}}，根据2014年份的数据整理。该报告未包含[[香港|香港特别行政区]]以及[[澳門|澳门特别行政区]]，也不包括中华人民共和国未实际统治的[[台灣|台灣]]。上述三地的数据分別参照自[[联合国开发计划署|联合国开发计划署]]和[[澳门特区政府|澳门特区政府]][[统计暨普查局|统计暨普查局]]、[[中華民國內政部|中華民國內政部]]在2014年的统计数据（由于联合国于2010年采用[[人类发展指数#计算方法|新的计算方法]]，各地区数据普遍有明显的降低，并不意味着发展倒退{{NoteTag|1=[http://hdr.undp.org/en/media/HDR_2013_EN_TechNotes.pdf HDR_2013_EN_TechNotes]. 第2页. 另见[[人类发展指数#2010年开始的新计算方法|人类发展指数#2010年开始的新计算方法]]}}）。
中国各[[中国一级行政区|省级行政区]]大体可分为三个梯次{{RefTag|1={{Cite web |url = http://www.usc.cuhk.edu.hk/webmanager/wkfiles/4998_1_paper.pdf |title = 中国人类发展的地区差距和不协调：历史视角下的“一个中国，四个世界” |format = PDF |author = 杨永恒、[[胡鞍钢|胡鞍钢]]、[[张宁|张宁]] |publisher=[[清华大学|清华大学]]国情研究中心 |date=2005年5月15日 |language=zh-cn |accessdate=2010年7月8日}}}}：
: <font size="+1" font color="#003C00">■</font><font size="+1" font color="#007F00">■</font><font size="+1" font color="#00C400">■</font> 指数在0.8以上（[[北京市|京]]、[[上海市|沪]]、[[天津市|津]]、[[香港|港]]、[[澳門|澳]]、[[台湾|臺]]）为极高的人类发展指数，已处于相当于[[发达国家|发达国家]]的发展水平，譬如[[新加坡|新加坡]]、[[芬蘭|芬蘭]]、[[捷克|捷克]]、[[波蘭|波蘭]]等。
: <font size="+1" font color="#00F900">■</font><font size="+1" font color="#D3FF00">■</font> 指数0.700至0.799（[[中国大陆|中国大陆地区]]多数[[中国东部经济区|东部沿海省市]]、[[中国东北地区|东三省]]、[[内蒙古自治区|内蒙]]、[[中国中部经济区|中部经济区]]拥有高人类发展指数，相当于中、高阶[[发展中国家|发展中国家]]，譬如[[土耳其|土耳其]]、[[巴西|巴西]]、[[墨西哥|墨西哥]]、[[泰国|泰国]]等。
: <font size="+1" font color="#FFFF00">■</font><font size="+1" font color="#FFD215">■</font> [[中国西部|中国西部]]内陆地区相对落后，有5个省、市、自治区的发展指数在0.600至0.699之间，属于中人类发展指数，相当于低阶发展中国家，譬如[[埃及|埃及]]、[[菲律宾|菲律宾]]、[[南非|南非]]、[[不丹|不丹]]等。

相较先前的《中国人类发展报告》以及国际的《人类发展报告》，[[大中华地区|大中华地区]]各地均有明显的提升。自1999年[[西藏自治区|西藏]]摆脱低人类发展指数后境内未再出现低指数，2014年中国从中指数变为高指数。

根据《2016年中国人类发展报告》推算[[中国大陆|中国大陆]]地区平均人类发展指数：2017年为0.753；2023年为0.803；2028年为0.845；2035年为0.902{{RefTag|{{cite journal |title=undp-ch-_nhdr_2016_en |author=UNDP |url = http://hdr.undp.org/sites/default/files/reports/2783/undp-ch-_nhdr_2016_en.pdf |journal=undp-ch-_nhdr_2016_en |year=2016 }}}}。
