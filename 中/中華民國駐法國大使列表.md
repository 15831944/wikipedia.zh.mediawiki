=== {{ROC}}驻法国大使（1936年－1949年） ===
自1936年2月[[顾维钧|顾维钧]]由驻法国公使升任为首任大使起，至1949年[[钱泰|钱泰]]卸任为止，凡三任全权大使（其中[[魏道明|魏道明]]未到任）。<ref name="ml" />1949年后至1964年中华民国与法国断交为止，未再派遣大使。

==斷交後中華民國駐法國外交官==
斷交後，僅設'''法華經濟貿易觀光促進會'''（{{lang|fr|ASPECT}}）及'''亞洲貿易促進會駐巴黎辦事處'''維持兩國互動。1995年5月15日，法國同意將兩機構合署更名為[[實質大使館|具大使館性質]]'''[[駐法國台北代表處|駐法國台北代表處]]'''（法文：{{lang|fr|Bureau de Représentation de Taipei en France}}）。<ref name="103年">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/09.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref><ref name="代表處簡介">{{Cite web |url = http://www.taiwanembassy.org/fr/post/13.html |title = 代表處簡介 |publisher = 中華民國駐外單位聯合網站}}</ref><ref name="駐館與駐地關係">{{Cite web |url = http://www.taiwanembassy.org/fr/post/2936.html |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/OverseasOffice_Content.aspx?n=6357834932B83C83&sms=5A961ED2E4BA25E8&s=E6436B9A96D5AEE5&os=D7B9F1DAF6192D32 |title = 駐外館處 |publisher = 中華民國外交部}}</ref>
