'''中華民國與芬蘭關係'''是指[[中華民國|中華民國]]與[[芬蘭共和國|芬蘭共和國]]之間的關係。兩國於1926－1932年有官方[[外交#外交上的承認|外交關係]]，斷交後直至1990年代，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。芬蘭在近代歷史上受[[俄羅斯帝國|俄羅斯帝國]]、[[蘇聯|蘇聯]]與[[冷戰|冷戰]]影響頗深，與中華民國的官方關係並不密切。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[芬蘭|芬蘭]]。]]
{{see also|中華民國外交|{{link-en|芬蘭外交|Foreign relations of Finland}}}}
{{see also|中国驻芬兰大使列表{{!}}中華民國駐芬蘭外交官列表|芬兰驻华大使列表{{!}}芬蘭駐中華民國外交官列表}}
1923年，中華民國派駐芬蘭[[代辦|代辦]]（由駐[[瑞典|瑞典]]公使館[[外交官#其他外交官员|一等秘書]]兼任代辦使事）。

1925年，開始派駐芬蘭[[特命全權公使|公使]]。

1926年10月29日，芬蘭與中華民國建立公使級[[外交#外交上的承認|外交關係]]。於首都[[赫爾辛基|赫爾辛基]]設立'''中華民國駐芬蘭共和國[[外交代表機構#公使馆|公使館]]'''；芬蘭則在[[上海市_(中華民國)|上海]]派駐公使，並於[[南京市_(中華民國)|南京]]、上海、[[廣州市_(中華民國)|廣州]]、[[漢口市_(中華民國)|漢口]]、[[天津|天津]]、[[煙臺|煙臺]]、[[青島|青島]]設立[[領事館|領事館]]。{{noteTag|《芬蘭一瞥》〈外交〉，芬蘭駐華公使館於1937年12月在上海發行。}}

1932年4月28日，中華民國駐芬蘭公使館「暫行停辦」，館務由駐瑞典公使館辦理。{{noteTag|1933年、1938年中華民國外交部出版的《駐外使領各館職員錄》列有駐芬蘭國公使館，只是標註「暫停」、「暫行停辦，由駐瑞典使館兼理」。而1946年的《駐外使領各館職員錄》中已經不再列駐芬蘭國公使館；1937年5月1日，有[[芬蘭駐華大使列表|芬蘭駐華公使]]（駐[[大日本帝国|日本]]公使兼任）{{link-en|華爾槐納|Hugo Valvanne}}到中國遞交到任[[國書|國書]]，至1939年仍有駐華公使館秘書代辦使事。}}

1933年7月20日，[[芬兰外交部|芬蘭外交部]]照會[[中華民國外交部|中華民國外交部]]，聲明在華利益委託[[丹麥|丹麥]]駐華公使館代理。

其後，芬蘭承認[[日本帝國|日本帝國]]扶植的[[滿洲國|滿洲國]]，滿洲國並派駐芬蘭公使（駐[[納粹德國|納粹德國]]公使兼任）。隨著1941年芬蘭倒向納粹德國，中華民國與芬蘭的外交關係不久後終止。

2002年12月18日，[[芬蘭政治|芬蘭政府]]發佈公報承認[[中華民國汽車駕駛執照|中華民國汽車駕駛執照]]並實施免試互換駕照。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2004/08/08_01_04.htm |title = 《中華民國93年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2015年4月18日，國貿局委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）辦理的「2015年新[[北歐|北歐]]利基產業拓銷團」與36家廠商前往[[俄羅斯|俄羅斯]][[聖彼得堡|聖彼得堡]]、[[芬蘭|芬蘭]]、[[愛沙尼亞|愛沙尼亞]]、[[瑞典|瑞典]]及[[波蘭|波蘭]]等國爭取商機，該團也是[[台灣|台灣]]近期赴北歐地區規模最大的拓銷團。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=661953&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date =  2015年4月17日 |title = 貿協規模最大北歐拓銷團 4月18日啟程波海網商機 |publisher = 經貿透視網}}</ref>

2018年，芬蘭是中華民國第49大貿易夥伴、第42大[[進口|進口]]夥伴、第51大[[出口贸易|出口]]夥伴。出口至芬蘭的金額為2億6,144萬1,418[[美元|美元]]，年增1.644%；自芬蘭進口的金額為3億9,908萬9,783美元，年增44.080%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）1億3,764萬8,365美元，年增595.947%。2017年，芬蘭是中華民國第53大貿易夥伴、第49大進口夥伴、第49大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
2003年，[[鴻海科技集團|鴻海科技集團]]透過香港[[富智康集團|富智康集團]][[併購|併購]]芬蘭{{link-fi|Eimo|Eimo}}，投資金額為6,700萬[[歐元|歐元]]。2007年，[[光寶科技|光寶科技]]併購芬蘭Perolos，投資金額為2億7,700萬歐元。此外，[[宏碁|宏碁]]、[[合勤科技|合勤科技]]、[[友訊科技|友訊科技]]、[[趨勢科技|趨勢科技]]等則透過其他歐洲分支機構在芬蘭成立銷售或服務據點，負責人均為外籍人士；<ref name="投資環境簡介"/>根據經濟部投資審議委員會統計，截至2017年，芬蘭對台灣投資共27件，總投資金額約533萬美元。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

===教育===
{{see also|台灣教育|芬蘭教育}}
1998年，[[國立台灣大學|國立台灣大學]]校長[[陳維昭|陳維昭]]訪問芬蘭。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1998/section_3/page3-1-4.htm |title = 《中華民國87年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>4月4日，{{link-en|芬蘭科學院|Academy of Finland}}與[[行政院國家科學委員會|行政院國家科學委員會]]簽署[[科學方法|科研]]交流合作協定，芬蘭科學院歷任院長亦應邀訪台。<ref name="駐館與駐地"/><ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 國家基本資料表 |publisher = 中華民國對外貿易發展協會}}</ref>

2003年9月起，[[中華民國教育部|中華民國教育部]]提供「台灣[[獎學金|獎學金]]」及「[[中華民國國語|華語文]]獎學金」予芬蘭學生赴台[[留學|留學]]。<ref name="駐館與駐地"/>

2004年9月26日，[[赫爾辛基大學|赫爾辛基大學]][[校監|總校長]]（Chancellor）萊維歐（Kari Olavi Raivio）博士夫婦應[[中華民國外交部|外交部]]及[[國立台灣大學|國立台灣大學]]邀請抵台訪問8天。期間拜會[[中央研究院|中央研究院]]、外交部、教育部、[[行政院衛生署|行政院衛生署]]及國立台灣大學等部會機構。 <ref>{{Cite web |url = http://www.mofa.gov.tw/Mobile/News_Content.aspx?s=DC105045B3030AB0 |author = 公眾外交協調會 |title = 芬蘭赫爾辛基大學總校長萊維歐博士夫婦應邀訪台 |publisher = 中華民國外交部}}</ref>

2010年起，中華民國外交部提供「台灣獎助金」予芬蘭學者赴台從事[[漢學|漢學]]或台灣研究。2011年1月起辦理「教育部獎補助外國人士來華短期研究或研習華語文要點」予芬蘭學生赴台留學。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2011/2.1.4.pdf |title = 《中華民國100年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

截至2017年，共有[[台灣大專院校列表|台灣28所大學]]與[[芬兰大学列表|芬蘭26所大學]]進行各項研究合作、[[訪問學者|交換教授]]及學生或締結姊妹校等計畫。台灣前往芬蘭留學的[[碩士|碩]][[博士生|博士生]]與交換學生近百人。<ref name="106年外交年鑑"/>
