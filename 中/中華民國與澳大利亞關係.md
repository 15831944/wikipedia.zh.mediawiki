'''中華民國與澳大利亞關係'''是指[[中華民國|中華民國]]與[[澳大利亞聯邦|澳大利亞聯邦]]（[[台灣|台]]、[[香港|港]][[澳門|澳]]多通稱'''澳-{}-洲'''）之間的關係。兩國於1941－1972年有官方[[外交#外交上的承認|外交關係]]，斷交後，在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

=== 在大陸時期 ===
1912年，中華民國成立後，繼承[[清朝政府|清朝政府]]於[[光緒|光緒]]卅四年（1908年）在[[大英帝國#太平洋地区|英屬澳洲]]設置的[[領事館|總領事館]]，更名為'''中華民國駐澳洲總領事館'''。

1941年7月15日，兩國建立公使級[[外交#外交上的承認|外交關係]]，設立'''中華民國駐澳洲聯邦[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1945年9月，[[澳大利亞國防軍|澳大利亚军]]{{link-en|澳大利亞陸軍第11師|11th Division (Australia)|第11师}}占领[[新不列颠岛|新不列颠岛]]。9月17日中国[[战俘营|战俘营]]的官兵在KUMUIA YAMA营区门口列队相迎[[同盟國_(第二次世界大戰)|盟軍]]<ref>{{cite web|url=http://blog.wenxuecity.com/blogview.php?date=200710&postID=26464| title=流放在热带丛林的中国抗日将士(2)|publisher= wenxuecity.com|accessdate=2008年12月24日}}</ref>。當時中国戰俘营的最高指挥官[[國民革命軍|國民革命軍]]吴炎[[中校|中校]]及軍官陈国樑[[少校|少校]]、刘伟宾[[侍從官|副官]]和秦家麟[[中尉|中尉]]。中國戰俘獲得盟軍的援助逐漸的得到軍服與武器。9月28日，在吴炎的陪同下，澳军[[参谋总长|参谋总长]]{{link-en|J. Northcott|John Northcott}}[[中将|中将]]和澳军第11师师长{{link-en|K. W. Eather|Kenneth Eather}}少将等检阅了中国军营部队<ref>{{cite web|url=http://blog.wenxuecity.com/blogview.php?date=200710&postID=26516| title=流放在热带丛林的中国抗日将士(3)|publisher= wenxuecity.com|accessdate=2008年12月24日}}</ref>。

1948年2月16日，升格為'''中華民國駐澳洲聯邦[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

===在臺灣時期===
1949年中華民國政府遷台後，兩國的關係在1949至1971年間的政治環境主要著重於「中國問題」。<ref name="IvanShearer">Ivan Shearer, ‘International Legal relations between Australia and Taiwan: Behind the Façade.’ Australia Year Book of International Law 28.113 (2000)</ref>

1955年9月6日，{{en-link|澳洲外交部長|Minister for Foreign Affairs (Australia)}}稱，澳洲在[[聯合國大會|聯合國大會]]將繼續尋求擱置准許[[中共|中共]]進入聯合國，除主權發生爭執之國家外，澳洲將竭力推動接受現有向大會申請入會之國家，以便擴大聯合國之範圍。澳外長認為中共要求入會，不能與申請入會者相提並論。蓋認國民政府中國早為聯合國之會員國，目前問題之焦點不在承認新會員國問題，而係應決定兩個政府以何者有資格代表中國之問題。<ref>[http://catalog.digitalarchives.tw/item/00/47/b5/d9.html 典藏臺灣：駐美爾鉢領事館電外交部有關澳外長將繼續尋求擱置准許中共進入聯合國事]</ref>

1965年7月10日，外交部長[[沈昌煥|沈昌煥]]赴澳大利亞訪問一週，14日發表聯合公報，18日離開澳洲赴[[菲律賓|菲律賓]]訪問。<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|715}}。

1966年6月11日，在[[澳洲自由黨|自由黨]]籍的[[澳大利亞總理|澳大利亞總理]][[哈罗德·霍尔特|哈洛德·荷特]]執政時，澳洲在台北設立大使館。<ref name="ThomasNicholasReOrientating">Thomas,Nicholas, Re-Orientating Australia-Taiwan Relations 1972 to present (England: Ashgate Publishing, 2004)</ref>

1967年4月4日，荷特訪問臺灣3天；4月6日，與蔣中正兩度會談，發表聯合公報。<ref name="記" />{{rp|736}}。

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[澳洲|澳洲]]。]]
1971年10月25日，澳洲支持中華民國繼續保有[[聯合國|聯合國]]裡的[[中國與聯合國#代表權問題|中國席次]]（即'''中國代表權'''，《[[聯合國大會2758號決議|聯合國大會2758號決議]]》），但最終表決失敗。

1972年12月22日，[[澳洲|澳洲]]宣布與[[中華民國|中華民國]]斷交，並承認[[中華人民共和國|中華人民共和國]]且與其[[外交#外交上的承認|建交]]，<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/06.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>中華民國與澳洲的雙方關係陷入低潮。

1975年4月，中華民國總統[[蔣介石逝世|蔣介石逝世]]，[[澳洲政府|澳洲政府]]稱：「澳洲人民永遠感激蔣中正。由於他的勇敢、領導與犧牲，才使澳洲在二次大戰時免於被日本-{征}-服」。

1994年8月，中華民國外交部政務次長房金炎前往澳洲[[黃金海岸_(昆士蘭州)|黃金海岸市]]，出席[[南太平洋論壇|南太平洋論壇]]會後對話會議，並主持外交部第三屆[[大洋洲|南太平洋地區]]區域會報。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=2900DA1EDD0345A6 |author = 公眾外交協調會 |date = 1994年8月5日 |title = 外交部房政務次長金炎於本（八）月二日偕同相關人員前往澳洲黃金海岸市 |publisher = 中華民國外交部}}</ref>

2017年5月1日，《[[雪梨晨鋒報|雪梨晨鋒報]]》（SMH）報導，澳大利亞（以下簡稱澳方）是該年《[[金伯利進程國際證書制度|金伯利進程國際證書制度]]》（Kimberley Process，簡稱金伯利進程）{{link-en|政府間組織|Intergovernmental organization}}的輪值主辦國，於1－4日在[[伯斯|伯斯]]舉行中期會議（Intersessional Meeting），由{{link-en|澳洲外交部長|Minister for Foreign Affairs (Australia)}}[[朱莉·毕晓普|畢曉普]]（Julie Bishop）主持。首日的開幕式，澳方的1名官員開始介紹外長畢曉普及歡迎所有代表團成員時，中華人民共和國代表團（以下簡稱中方）突然開啟麥克風提出程序問題，要求確認在場的會員是否有「正式」邀請，想將台灣代表團趕出會場、拒絕台灣參與、阻擾會議進行，引發部分國家出席代表嚴厲譴責，澳方高層官員也以「舉止異常」、「令人作嘔」等用詞形容中方所為非常不尊重他人。由於中方不斷地咆哮鼓噪，澳方最終被迫將程序問題列入會議的首個議程，隨後中斷會議，澳、中官員密室協商。最後，在中方的堅持下，澳方不得不請台灣代表團離場，澳洲外交部發言人表示：「令人遺憾的是，中方持續地阻擾議程，澳方已向[[中国驻澳大利亚大使列表|中方駐澳大使]]表達對此等行為的關切。」{{noteTag|由於中華人民共和國的反對，以致中華民國無法以正式會員身分出席，但在友好國家的斡旋下，2007年以{{link-en|觀察員身分|Observer status}}參加年度會議。根據[https://www.kimberleyprocess.com/en/2017-kp-participants-list 金伯利進程網站]，參與國名單下方特別註明「[[中華台北|中華台北]]{{link-en|鑽石原石|Rough diamond}}貿易實體已符合金伯利進程的最低要求」。今年依例以「鑽石原石貿易實體」受邀出席。}}<ref>{{Cite web |url = http://www.epochtimes.com/b5/17/5/2/n9099748.htm |author = 吳英 |date = 2017年5月3日 |title = 澳媒：國際會議上中共官員舉止「令人作嘔」|publisher = [[大紀元時報|大紀元時報]]}}</ref><ref>{{Cite web |url = http://www.ettoday.net/news/20170504/917009.htm |author = 楊絡懸 |date = 2017年5月4日 |title = 台灣參與國際會議杜絕「血鑽石」遭中國代表趕出會場 |publisher = [[ETtoday新聞雲|ETtoday新聞雲]]}}</ref>

2018年6月22日，《雪梨晨鋒報》報導，澳洲與台灣在2017年9月秘密簽署[[備忘錄|備忘錄]]，允許澳洲把安置在[[諾魯共和國|諾魯共和國]]，且有緊急醫療需求的[[難民|難民]]送到台灣接受治療，其主要目的是避免難民在澳洲接受治療後趁機滯留。[[中華民國外交部|中華民國外交部]]證實消息，指簽署備忘錄以來，已安排10多名安置在諾魯的[[重症醫學|急症或重症]]難民抵台接受治療，醫療費用由澳洲支付。官員強調並非「秘密簽署」，是因外交協定不會簽署後都對外發布，也因為擔心中華人民共和國打壓，希望低調進行。報導並指出，澳洲也向[[世界政區索引|其他國家]]尋求協助，但只有台灣與之簽署。<ref>{{Cite web |url = https://tw.appledaily.com/headline/daily/20180625/38052889 |author = 綜合報導 |date = 2018年6月25日 |title = 台澳密約曝光 外交突圍 我接收難民就醫 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref>

持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以[[簽證#电子签证|電子簽證]]（Electronic Travel Authority, ETA）的方式入境澳洲，有效期1年可多次入境，每次停留最多3個月。目前臺灣無法在{{link-en|澳洲移民暨邊界保護部|Department of Immigration and Border Protection}}網站線上申辦ETA，現居臺灣並設有戶籍者，可直接持護照前往[[澳洲辦事處|澳洲駐臺辦事處]]所指定的37家[[旅行社|旅行社]]申辦ETA，絕大多數的申請案都可以立即獲得核發。申請ETA免費，但代辦的旅行社則會收取數額不同的服務費。經澳洲轉機，若符合要件，可[[簽證#签证优惠待遇|免簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-17-c471c-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ElectronicVisa.aspx?n=6F246645F10311D7&sms=B75E93A3DAB23A99 |title = 電子簽證資訊 |publisher = 中華民國外交部}}</ref>

持有[[澳洲護照|澳洲護照]]的[[澳大利亚人口|澳洲公民]]則可以免簽證的方式入境中華民國，停留最多90天，期限至2019年12月31日。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

此外，兩國皆提供無名額限制的18－30歲青年為期1年的[[工作假期签证|度假打工簽證]]（皆必須未曾取得此種簽證）。<ref>{{Cite web |url = http://www.youthtaiwan.net/WorkingHoliday/Cus_WHNation.aspx?n=4892E8B8F5C0E174&s=E4CFAE92581A7EB7 |title = 青年度假打工 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-82-44-0023d-1.html |title = 澳大利亞籍人士申請中華民國度假打工事由簽證相關手續說明 |publisher = 外交部領事事務局}}</ref>目前[[中華民國國民|中華民國公民]]以度假打工名義在澳洲工作者達到9,000人左右，而在澳洲的留學生共約10,000人左右。<ref name="經濟文化"/>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將澳洲歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2018年10月3日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-393-7b82a-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===背景===
1991年，中華民國與澳洲建立直飛航線後，經貿關係穩定增加。但近年受[[經濟大衰退|全球金融風暴]]、[[澳洲元|澳幣]]大幅[[貨幣升值|升值]]、[[煤|煤]][[鐵|鐵]]等[[能源|能源]][[礦物|礦物]]價格大幅波動等因素影響，使雙邊[[国际贸易|進出口貿易]]變化頗大。其中澳幣持續升值迫使移轉採購來源，例如煤礦增加自[[印尼|印尼]]進口量，廢鐵改自[[俄羅斯|俄羅斯]]進口，農牧產品也有轉向[[紐西蘭|紐西蘭]]及[[瑞士|瑞士]]的趨勢。<ref name="投資環境簡介"/>

雙邊貿易自1991年30億[[美元|美元]]到1997年成長至50億美元，2008年更突破100億美元。2009年衰退至83.2億美元，2010年回升至120億美元，2011年達145億美元高峰。2012年滑落至129億美元，2013年減少至117億美元，2014年再減至109億美元（佔中華民國貿易比重1.85%）。<ref name="投資環境簡介"/>

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於澳洲第1大城[[悉尼|悉尼]]設立'''臺灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#OCEANIA |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]在首都[[坎培拉|坎培拉]]設立'''駐澳大利亞代表處經濟組'''；<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=214&pid=305862 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>澳洲則在雪梨設立'''澳台工商委員會'''。<ref>{{Cite web |url = http://atbc.asn.au/home |title = Australia-Taiwan Business Council|publisher = 澳台工商委員會}}</ref>

2018年，澳洲是中華民國第10大貿易夥伴、第6大[[進口|進口]]夥伴、第15大[[出口贸易|出口]]夥伴，在[[大洋洲国家和地区列表|大洋洲經濟體]]皆排名第1。出口至澳洲的金額為35億9,556萬5,637美元，年增15.907%；自澳洲進口的金額為95億5,603萬3,575美元，年增16.268%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）59億6,046萬7,938美元，年增16.486%。2017年，澳洲是中華民國第11大貿易夥伴、第7大進口夥伴、第16大出口夥伴，在大洋洲經濟體皆排名第1。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

{{link-en|澳洲教育部|Department of Education and Training (Australia)}}於1995年5月在[[台北市|台北]]設立「澳大利亞國際教育[[基金會|基金會]]」，用以推廣國際教育合作計畫及促進雙邊文教交流。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1998/section_3/page3-1-1.htm |title = 《中華民國87年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

首屆台澳[[高等教育|高等教育]]會議於2002年4月12日在[[南澳大利亞|南澳大利亞]]首府[[阿得雷德|阿得雷德]]舉行。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__01.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2009年，首次核定臺澳[[研究生|研究生]]導航計畫，補助台灣[[博士生|博士生]]赴澳洲醫學等相關單位研究。[[中華民國教育部|中華民國教育部]]與[[澳洲國立大學|澳洲國立大學]]、[[蒙納許大學|蒙納許大學]]合作設立「臺灣研究[[講座|講座]]」。[[台灣大專院校列表|台灣多所大學]]也與[[澳洲大學列表|澳洲的大學]]簽署合作[[備忘錄|備忘錄]]。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2009/html/08-1-1.htm |title = 《中華民國98年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>2015年，澳洲在台灣留學人數為371人<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>；2017年，台灣在澳洲留學人數為1萬6,573人。{{link-en|澳洲外交暨貿易部|Department of Foreign Affairs and Trade (Australia)|澳洲外交部}}核定的「{{link-en|新可倫坡計畫|New Colombo Plan}}」（New Colombo Plan）也獎助121名澳洲大專學生赴台短期研習與實習。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>
