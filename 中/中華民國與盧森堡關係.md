==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]一案棄權，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[盧森堡|盧森堡]]。]]
{{see also|中華民國外交|{{link-en|盧森堡外交|Foreign relations of Luxembourg}}}}
1949年1月1日，盧森堡與中華民國建立公使級[[外交#外交上的承認|外交關係]]，並派駐[[特命全權公使|公使]]（由[[中国驻比利时大使列表|駐比利時王國大使]]兼任）。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1964年12月16日，升格為大使級外交關係，並派駐[[特命全權大使|大使]]（由駐比利時王國大使兼任）。

1971年9月，參謀總長[[賴名湯|賴名湯]]訪問盧森堡。<ref>[http://catalog.digitalarchives.tw/item/00/3a/18/f0.html 賴總長訪問盟邦歸國]，典藏台灣</ref>

1972年11月14日，與中華民國斷交。

==經濟==
{{see also|臺灣經濟|盧森堡經濟}}
===貿易===
2017年，盧森堡是中華民國第71大貿易夥伴、第97大[[進口|進口]]夥伴、第56大[[出口贸易|出口]]夥伴。出口至盧森堡的金額為1億7,153萬8,411[[美元|美元]]，年增180.427%；自盧森堡進口的金額為2,240萬6,259美元，年增16.903%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1億4,913萬2,152美元，年增255.043%。※2016年，盧森堡是中華民國第94大貿易夥伴、第100大進口夥伴、第80大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
截至2017年，[[臺商|臺商]]赴盧森堡總投資金額約3,148萬美元；盧森堡在台灣總投資金額約4億6,866萬美元。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

[[中華航空|中華航空]]與[[盧森堡國際貨運航空|盧森堡國際貨運航空]]均有[[貨機|貨機]]飛航兩地。根據盧森堡商會統計，目前[[台灣|台灣]]對歐洲空運貨物50%的出口是經由盧森堡轉運，主要為面板等[[高科技|高科技]]產品。1989年，華航以盧森堡作為[[歐洲國家和地區列表|歐洲]]貨運中心。2014年，雙邊每週平均有20班貨機往返；<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 主要產業概況 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[光陽工業|光陽機車]]在盧森堡設立據點，主要市場包括[[比利時|比利時]]、[[法國|法國]]。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref>

盧森堡前往台灣投資，例如貨運[[物流|物流]]公司的盧森堡貨運及工程科技公司的Paul Wurth；2010年，盧森堡{{link-en|森拉天時集團|Ceratizit}}與台灣春保集團合資成立「春保森拉天時[[鎢|鎢]][[鋼|鋼]][[合金|合金]]公司」，專門製造高硬度鎢鋼合金工具，是為雙方[[合資公司|合資]]成功案例。<ref name="與我國經貿關係"/>

台灣是盧森堡{{link-en|UCITS基金|Undertakings for Collective Investment in Transferable Securities Directives}}在[[亞洲國家和地區列表|亞洲]]銷售的主要市場。盧森堡UCITS基金在[[歐洲聯盟|歐盟]]以外的主要銷售市場有[[瑞士|瑞士]]、[[挪威|挪威]]、[[新加坡|新加坡]]、[[香港|香港]]、[[巴林|巴林]]、[[列支敦斯登|列支敦斯登]]及[[台灣|台灣]]等。台灣是該基金在亞洲第3大市場，次於新加坡及香港。<ref name="與我國經貿關係"/>

根據[[盧森堡證券交易所|盧森堡證券交易所]]統計，截至2015年1月，已有48家台灣公司發行56檔[[存託憑證|全球存託憑證]]（GDR）。包括[[宏達國際電子|宏達國際電子]]、[[中國鋼鐵|中國鋼鐵]]（中鋼）、[[歌林公司|歌林公司]]、[[力成科技|力成科技]]、華亞科技等，位居外國公司在該證交所存託憑證掛牌第2名（次於[[印度|印度]]171家公司，186檔）。2009年1月1日，盧森堡證券交易所為台灣[[上市公司|上市公司]]設立「台灣指數」（The Lux GDRs Taiwan）。<ref name="與我國經貿關係"/>

===會議===
2007年9月，雙方在聯合商業協會架構下，於[[盧森堡_(城市)|盧森堡]]召開「第1屆台盧經濟合作會議」，此後每屆會議於兩國首都輪流舉辦。<ref name="與我國經貿關係"/>2017年12月，於[[臺北市|臺北]]舉辦「第10屆臺盧經濟合作會議」，雙方就[[循環經濟|循環經濟]]、{{link-en|數位經濟|Digital economy}}等議題交換意見。盧森堡代表團並參觀台灣的[[育成中心|育成中心]]、[[初创企业|新創公司]]，尋求合作商機。<ref name="106年外交年鑑"/>

===航空===
1978年5月，[[中華航空|中華航空]]與[[盧森堡國際貨運航空|盧森堡國際貨運航空]]簽署航空協定，盧森堡貨運自同年7月15日飛航臺北，華航自1982年5月11日飛航盧森堡，華航日後即在盧森堡設立歐洲貨運中心。1984年12月13日，兩國民航局簽署[[航權|航權]]換文，除原有貨運外，另增加客運，但因市場需求不大未開闢客運航班。<ref name="86年外交年鑑"/><ref name="97年外交年鑑"/>
