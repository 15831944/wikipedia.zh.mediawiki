'''中華民國與巴布亞紐幾內亞關係'''是指[[中華民國|中華民國]]與[[巴布亞紐幾內亞獨立國|巴布亞紐幾內亞獨立國]]（通稱'''巴布亞紐幾內亞'''、'''巴紐'''）之間的關係。兩國於1999年有正式[[外交#外交上的承認|外交關係]]，但只維持17天即斷交，目前在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==歷史==
{{see also|拉包爾戰俘營}}
[[太平洋戰爭|太平洋戰爭]]期間，日軍攻佔新幾內亞[[拉包爾|拉包爾]]，在拉包爾建立海空軍事基地，並從[[中國大陸|中國大陸]]與[[新加坡|新加坡]]等佔領區運送[[盟軍|盟軍]]戰俘來此做工，其中有大量中國與[[印度|印度]]戰俘。曾有一千多位[[中華民國|中華民國]][[國民革命軍|國民革命軍]]軍人被俘後被送到此，其中包括[[中國抗日戰爭|抗戰]]初期堅守[[四行倉庫|四行倉庫]]的「[[八百壯士|八百壯士]]」中的36名官兵。期間有一部分死在營中。[[日本投降|日本投降]]後，[[澳大利亚|澳大利亚]]軍隊進駐拉包爾，生還者曾為死難者修建一座公墓，但已被夷為平地<ref name="聯合報2">{{cite news | last =李志德| first =| title =八百壯士淪戰俘 倖存者稀| publisher =[[聯合報|聯合報]]| date = 2008年12月22日  | url = http://udn.com/NEWS/NATIONAL/NATS5/4654088.shtml| accessdate = 2008-12-23 }} </ref><ref name="北京青年報">{{cite news| last =余瑋 吳志菲| first =| title =抗日"八百壯士"倖存者的血淚回憶| publisher =[[北京青年報|北京青年報]]| date =2005年5月| url =http://hk.huaxia.com/thjq/wzzdlj/2005/05/20974.html| accessdate =2008-12-23}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref><ref>{{cite web|url=http://blog.wenxuecity.com/blogview.php?date=200710&postID=26464| title=流放在热带丛林的中国抗日将士(2)|publisher= wenxuecity.com|accessdate=2008年12月24日}}</ref>。

===外交===
[[File:Voting_res_2758.png|thumb]]，[[巴布亞紐幾內亞|巴布亞紐幾內亞]]當時屬於[[巴布亞紐幾內亞領地|澳洲領地]]，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
{{see also|中華民國外交|{{link-en|巴布亞紐幾內亞外交|Foreign relations of Papua New Guinea}}}}
{{see also|2018年中華民國邦交國斷交與代表機構更名事件}}
1994年10月，巴布亞紐幾內亞副總理兼財政暨計畫部長{{link-en|克里斯·海維達|Chris Haiveta|海維達}}抵台參加[[中華民國國慶|中華民國國慶]]活動。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=84EBBA1B00228CDF |author = 公眾外交協調會 |date = 1994年10月7日 |title = 欣逢中華民國八十三年雙十國慶 |publisher = 中華民國外交部}}</ref>

1995年5月22日，海維達與中華民國外交部長[[錢復|錢復]]代表雙方政府在[[台北市|台北]]簽署《相互承認聯合公報》，決定依據[[國際法|國際法]]，尤其是關於經濟、貿易、技術及國際合作之原則相互給予完全承認。<ref name="86年外交年鑑"/><ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=998B9E46D51C50B6 |author = 公眾外交協調會 |date = 1995年5月26日 |title = 中華民國與巴布亞紐幾內亞兩國政府有鑒於冷戰結束後 |publisher = 中華民國外交部}}</ref>9月，在巴布亞紐幾內亞舉行第3屆「臺灣／中華民國－論壇國家對話會議」。參與國家有[[東加|東加]]、[[諾魯|諾魯]]、[[吐瓦魯|吐瓦魯]]、[[索羅門群島|索羅門群島]]、[[斐濟|斐濟]]、[[巴布亞紐幾內亞|巴布亞紐幾內亞]]等[[大洋洲|南太平洋國家]]。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=405E97A156F4058C |author = 公眾外交協調會 |date = 1995年9月19日 |title = 外交部政務次長房金炎於本（九）月十六日率團前往巴布亞紐幾內亞 |publisher = 中華民國外交部}}</ref>巴布亞紐幾內亞[[常駐聯合國代表|常駐聯合國代表]]在第50屆[[聯合國大會|聯合國大會]]的總辯論中，提出「會籍普遍化原則」或[[海峽兩岸關係|兩岸關係]]的方式，間接呼應[[中國與聯合國#台灣人民代表權問題及台灣入聯運動|中華民國參與聯合國]]。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=76ADD08F602C49F8 |author = 公眾外交協調會 |date = 1995年10月13日 |title = 聯合國大會第五十屆常會總辯論自九月二十五日至十月十一日召開期間 |publisher = 中華民國外交部}}</ref>

1999年7月初，巴紐總理[[比爾·史凱特|史凱特]]依據{{link-en|巴布亞紐幾內亞內閣|Cabinet of Papua New Guinea|內閣會議}}授權，率領相關閣員至台北洽商建交事宜。7月5日，雙方外交部長共同簽署《建交公報》、[[外交#外交上的承認|建立外交關係]]。7月21日，巴紐新任總理[[梅克雷·莫劳塔|莫羅塔]]與新內閣以「前政府建交之方式不符合巴紐與其他國家建立正式外交關係之正常程序」為由，否認與中華民國建交事實。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=5028B03CED127255&sms=5ED24855AD8E6C58&s=02B397F5B7FB29ED |author = 非政府組織國際事務會 |date = 2003年7月5日 |title = 中華民國政府與巴布亞紐幾內亞政府依照國際法原則，尤其是關於國家平等、相互尊重主權及互利之原則，為加強兩國間之友好及合作關係，爰經決定自一九九九年七月五日起建立全面外交關係 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2002年至2008年，巴紐總理[[迈克尔·索马雷|索馬利]]藉由[[亞太經濟合作會議|亞太經濟合作會議]]（APEC）領導人非正式會議之便，與出席的[[亞太經濟合作會議中華臺北領袖代表列表|中華台北領袖代表]]進行雙邊會談。<ref name="91年外交年鑑"/><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2004/08/08_01_01.htm |title = 《中華民國93年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2005/WebPages/MICO/08/08_01_01.htm |title = 《中華民國94年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2006/html/08-1-1.htm |title = 《中華民國95年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2007/html/08-1-1.htm |title = 《中華民國96年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2008/html/08-1-1.htm |title = 《中華民國97年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2006年至2008年，兩國洽談建立全面外交關係，但雙方官員商議多次後，巴紐無意建交。

===代表機構===

1988年12月27日，巴布亞紐幾內亞外交部長[[迈克尔·索马雷|索馬利]]與中華民國外交部長[[連戰|連戰]]簽署《建立關係[[備忘錄|備忘錄]]》。依據該備忘錄，中華民國及巴紐分別在[[莫士比港|莫士比港]]及[[臺北市|臺北]]設立商務代表團，作為兩國官方代表機構，雙方機構正式人員均享有[[外交豁免權|外交豁免權]]。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-1.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1990年2月12日，在首都莫士比港設立[[實質大使館|具大使館性質]]的'''中華民國（臺灣）[[台北经济文化代表处|駐巴布亞紐幾內亞商務代表團]]'''（Trade Mission of the Republic of China (Taiwan) in Papua New Guinea），是少數在非[[中華民國外交#中华民国的邦交国|邦交國]]使用中華民國國號的[[中華民國駐外機構列表|駐外機構]]{{noteTag|截至2018年7月，駐非邦交國機構使用中華民國國號的有：斐濟；非邦交國駐臺灣機構使用駐「華」名稱的有：奈及利亞、阿曼。}}<ref name="86年外交年鑑"/>。
