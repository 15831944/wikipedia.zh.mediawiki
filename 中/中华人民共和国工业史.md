自从1949年10月1日以来，[[中华人民共和国工业|中华人民共和国的工业]]有了巨大的进步，从大部分东西都不能自己制造到可以独立自主的生产[[汽车|汽车]]、[[飞机|飞机]]、核武器、[[火箭|火箭]]等，但是自從開始市場經濟改革後，现在也着面临數量大卻不精緻、求精無價格競爭力等等许多问题。

==概述==
===1949年-1978年===
从1949年[[中华人民共和国|中华人民共和国]]成立到1953年，[[中华人民共和国|中华人民共和国]]实行的是过渡时期总路线。从1953年开始<ref>[http://news.qq.com/a/20090702/000699.htm 1953年：对[[资本主义|资本主义]]工商业的社会主义改造开始]</ref>中华人民共和国实行了社会主义改造，1956年改造完成后[[中华人民共和国|中华人民共和国]]实行的是苏联式的高度集中的计划经济体制。1953年时，中华人民共和国政府开始实施[[第一个五年计划_(中国)|第一个五年计划]]，确立了'''以重工业优先发展为主导的新中国工业化发展的主要方向'''。<ref name="AS">{{Cite web |url=http://military.china.com/zh_cn/history4/news2/11078475/20100118/15781236.html |title=苏联援助中国的“156项”：新中国工业城市发展(1) |access-date=2012-06-08 |archive-url=https://web.archive.org/web/20100210045303/http://military.china.com/zh_cn/history4/news2/11078475/20100118/15781236.html |archive-date=2010-02-10 |dead-url=yes }}</ref>。同时，[[苏联|苏联]]对[[中华人民共和国|中华人民共和国]]展开了大规模的援助。[[第一个五年计划_(中国)|第一个五年计划]]时期[[苏联|苏联]]援助[[中华人民共和国|中华人民共和国]]的[[156项工程|156项工程]]中，进入实际施工的共有150项。基本上涵盖了[[中华人民共和国|中华人民共和国]]的各个方面，这对[[中华人民共和国|中华人民共和国]]冷战期间重工业体系的形成有着巨大的帮助作用<ref name="AS" />。

1957年，[[第一个五年计划_(中国)|第一个五年计划]]结束，[[中华人民共和国|中华人民共和国]]人民的努力再加上[[苏联|苏联]]政府的援助，[[第一个五年计划_(中国)|第一个五年计划]]取得了巨大的成就，在此期间，[[中华人民共和国|中华人民共和国]]生产了自己的第一辆汽车、第一辆飞机等。

然而，[[第一个五年计划_(中国)|第一个五年计划]]所取得的巨大成功使得[[中国共产党中央委员会|中共中央]]的高层领导人产生了冒进情绪，1958年[[中国共产党中央委员会|中共中央]]发动了[[大跃进|大跃进运动]]，生产计划“以钢为纲”、“元帅升账”。为实现钢铁翻番，全国大量基础建设上马。1958年到1960年每年新增基本建设投资都在百亿元以上，1960年达384亿元，比上年增长11%，比1957年增加1.8倍。在基本建设投资总额中，生产性建设投资所占比重都在86%以上（1958年为88%，1959年为86.8%，1960年为86.4%），挤掉了非生产性的建设<ref>{{cite book |title =当代中国财政 |author = |date = |language =zh-hans |publisher =|pages =158|isbn= }}</ref>。工业上单纯追求产量带来诸多负面影响。为保证钢铁生产达标，钢铁工业本身的基本建设规模扩大，与钢有关的煤、电、运输等行业建设也随之增加。导致工业内部失调，全国职工人数猛增，超出国民经济特别是农业负担能力，加剧了社会商品的供需矛盾。

在公布完成的1108万吨钢中，合格钢只有800万吨；1369万吨生铁中，合格品只有953万吨<ref>{{cite book|title = 《中共党史参考资料》第二十三册 |accessdate =2009-06-07 |author =国防大学党史党建政工教研室编 |language =zh-hans|pages = 131-132 }}</ref><ref>[[周恩来|周恩来]]，《关于调整一九五九年国民经济计划主要指标和进一步开展增产节约运动的报告》，1959年8月26日，第二届[[全国人民代表大会常务委员会|全国人民代表大会常务委员会]]第五次会议</ref>。国家统计局估算，1958年土法炼钢亏损达50亿元，财政上花费了大量补贴<ref>{{cite book|title = 《大跃进亲历记》|accessdate =2009-06-07 |date =1996年|author =[[李锐_(1917年)|李锐]] |language =zh-hans|pages = 234  |publisher=上海远东出版社|isbn=9787806132180}}</ref>。森林等自然资源遭到过度开发甚至严重破坏。

中共八届六中全会，根据[[毛泽东|毛泽东]]“压缩空气”的精神，将北戴河提出的1959年钢产量3000万吨的指标降低为1800万吨左右。据新华社《内参》报道，从1959年1月开始，钢铁生产的指标，月月都没有完成计划。记者报道说：
{{Cquote|
开年以来，全国钢铁生产计划完成的不好。1月份全国产钢77万吨，仅为去年12月份产量的47.7%；产铁142万吨，为去年12月份的51.5%；产钢材52万吨，为去年12月份的83.9%。按今年第一季度钢铁生产计划的要求，全季平均每天应该生产钢33000吨，铁57000吨，钢材22000吨；而实际上1月份每天只生产钢25000吨，铁46000吨，钢材17000吨，比计划欠了很多。<ref>{{cite journal|url=|title =当前全国钢铁生产为什么没有完成计划？ |author =  |date =1959-02-21|language = zh|publisher =[[内部参考|内部参考]]}}</ref>|}}

1961年1月14日的中国共产党第八届九中全会制定了“调整、巩固、充实、提高”方针，大跃进运动终告结束。

文化大革命发生后，[[中华人民共和国|中华人民共和国]]的工业生产受到了巨大的破坏，但是也仍然取得了巨大的成就，如第一颗原子弹、第一颗氢弹、第一颗人造卫星等。

===1978年-现在===
1978年年底开始了艰难地改革进程，当时苏联基础的低效率且老化的工业，在市场竞争下面临严重资金危机，透过引进西方技术加以汰换，80年代现代化工业也基本上重新建立了起来。在1980年代，也有一些令[[中华人民共和国政府|中华人民共和国政府]]苦恼的地方，价格体系是双轨制，同时国有企业由于实行的是“铁饭碗”制度导致生产效率不高，经济的快速增长受到能源短缺的制约。

直到在1980年代中期时工业在一些领域取得了实质性的成功。工业产值为1952年的25倍。同时在[[煤炭|煤炭]]，[[纺织|纺织]]，[[自行车|自行车]]领域是世界上的主要生产国。在1980年代中期时大力的将制造业引入南方欠发达地区和农村地区。轻工业产品的产量相比改革开放以前已经大大增加。此时，企业的管理人员能够行使更大的自主权，实施技术创新，以提高效率，降低运营成本。

从1980年中期开始，中华人民共和国政府开始高度重视[[科技|科技]]作用，1995年5月，[[中国共产党中央委员会|中国共产党中央委员会]]、[[中华人民共和国国务院|中华人民共和国国务院]]发表了《加速科学技术进步的决定》，正式提出了科教兴国战略。

目前，[[中华人民共和国|中华人民共和国]]在[[核能|核工业]]、[[航天|航天工业]]、[[激光|激光工业]]、[[生物工程|生物工程]]等方面处于世界领先地位，在数控车床等核心工业技术上劣势明显，在汽车和机车制造、民用飞机制造等重工业方面缺乏核心竞争力。
