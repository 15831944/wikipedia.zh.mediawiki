'''中国共产党地方组织'''，是[[中国共产党|中国共产党]]对各地方工作进行领导的组织机构，包括各地方的代表大会及由其产生的委员会、[[纪律检查委员会|纪律检查委员会]]。各地方组织在《[[中国共产党章程|中国共产党章程]]》中称为'''党的地方组织'''，其中对地方组织的地位、职权和任务作了规定。<ref name="jjh">{{cite book|title=《中华人民共和国资料手册》|editor=蒋建华等主编|publisher=社会科学文献出版社|chapter=党的地方组织}}</ref>

== 创立初期（1921年－1927年） ==
1921年7月，中国共产党成立。1921年11月，[[中国共产党中央委员会|中国共产党中央委员会]]通知上海、北京、广州、武汉、长沙分别成立“区执行委员会”<ref name="jjh" />。1922年7月，在[[中共二大|中共二大]]上通过了《关于共产党的组织章程决议案》和《中国共产党章程》，其中规定：如果一个地区有两个支部以上，可组织成立地方执行委员会；各地区有两个地方执行委员会以上，可组织成立区执行委员会。直至1927年[[中共五大|中共五大]]召开时，中央直属地方组织共有湖南、湖北、上海、广东、北方、山东、江西、豫区、陕甘9个区执委；福州、大连、北满、安徽、重庆5个地执委；奉天、芜湖、六安、南陵5个特别支部；濉溪、寿县堰口集支部。1927年中共五大修订的党章，撤销了各区执委、地执委的建制。<ref name="vol1">{{cite book|title=《中国共产党组织史资料》第一卷：党的创建和大革命时期（1921.7-1927.7）|author=中共中央组织部、中共党史研究室、中央档案馆|publisher=中共党史出版社|year=2000}}</ref>

== 第一次国共内战时期（1927年－1937年） ==
1927年中共五大党章对地方组织的建制进行了调整，此后中共地方组织形成了省委、市或县委、区委的建制。1928年[[中共六大|中共六大]]又新增了“特别区”一级，包括几县或省之一部分。党章分为城乡区的组织、县或市的组织、省之组织三章，对各级地方组织的地位、职权和任务作了规定。<ref name="jjh" /><ref>{{cite web|url=http://news.xinhuanet.com/ziliao/2002-03/04/content_696164.htm|title=中国共产党党章（1928年7月10日中共六大通过）|publisher=新华网}}</ref>至1937年7月[[抗日战争|抗日战争]]爆发前，省委或省工委组织在[[陕甘宁革命根据地|陕甘宁革命根据地]]有中共陕北、陕甘宁省委，在国民党统治区有中共河北、山东、陕西省委，山西、绥远、河南、广西省工委，云南临时工委，山西公开工委等。<ref name="vol2">{{cite book|title=《中国共产党组织史资料》第二卷：土地革命战争时期（1927.8-1937.7）|author=中共中央组织部、中共党史研究室、中央档案馆|publisher=中共党史出版社|year=2000}}</ref>

== 抗日战争时期（1937年－1945年） ==
自1937年[[七七事变|七七事变]]前夕至1945年[[抗日战争|抗日战争]]结束，这一时期中国共产党地方组织的规模有了很大的发展，省（区）委由17个增至30多个，地、县委遍布中国大部分地区。<ref name="vol3">{{cite book|title=《中国共产党组织史资料》第三卷：抗日战争时期（1937.7-1945.8）|author=中共中央组织部、中共党史研究室、中央档案馆|publisher=中共党史出版社|year=2000}}</ref>

== 第二次国共内战时期（1945年－1949年） ==
1945年[[中共七大|中共七大]]党章修订了中共的组织系统，规定了省/边区/地方党委、县/市委、区委的建制。<ref>{{cite web|url=http://news.xinhuanet.com/ziliao/2002-03/04/content_696452.htm|title=中国共产党党章（1945年6月11日中共七大通过）|publisher=新华网}}</ref>此后，随着[[第二次国共内战|第二次国共内战]]的进行，地方组织不断进行调整，至1949年9月共有中共省委21个、省（区）工委3个、区党委16个、直辖市委10个、特委或工委6个。<ref name="vol4">{{cite book|title=《中国共产党组织史资料》第四卷：全国解放战争时期（1945.8-1949.9）|author=中共中央组织部、中共党史研究室、中央档案馆|publisher=中共党史出版社|year=2000}}</ref>

== 中华人民共和国成立至文革时期（1949年－1976年） ==
1949年10月1日[[中华人民共和国|中华人民共和国]]成立，中国共产党成为执政党。1956年[[中共八大|中共八大]]党章规定了地方党组的职权，并规定了省、自治区、直辖市和自治州委员会每届任期为3年，县、自治县、市委员会每届任期为2年。<ref name="jjh" />1967年[[文化大革命|文化大革命]]发动后，各地党委纷纷被夺权，[[革命委员会|革命委员会]]相继成立。<ref name="vol5">{{cite book|title=《中国共产党组织史资料》第五卷：过渡时期和社会主义建设时期（1949.10-1966.5）|author=中共中央组织部、中共党史研究室、中央档案馆|publisher=中共党史出版社|year=2000}}</ref><ref name="vol6">{{cite book|title=《中国共产党组织史资料》第六卷：“文化大革命”时期（1966.5-1976.10）|author=中共中央组织部、中共党史研究室、中央档案馆|publisher=中共党史出版社|year=2000}}</ref>

== 文革结束至今（1976年至今） ==
1982年[[中共十二大|中共十二大]]修订党章，对的地方组织体作了许多调整，专门列有“党的地方组织”一章，规定了地方组织的地位、职权及产生办法等。其中规定，省、自治区、直辖市、设区的市和自治州代表大会每5年举行一次，县（旗）、自治县、不设区的市和市辖区代表大会每3年举行一次；省、自治区、直辖市、设区的市和自治州委员会每届任期为5年，县（旗）、自治县、不设区的市和市辖区委员会每届任期为3年。十二大党章还规定设立各级纪律检查委员会，同时在各省、自治区、直辖市设立[[顾问委员会|顾问委员会]]。顾问委员会是中共干部新老交替的过渡机构，1992年[[中共十四大|中共十四大]]后不再设立。十四大党章还将县（旗）、自治县、不设区的市和市辖区代表大会改为每5年举行一次，相应的委员会每届任期亦改为5年。<ref name="jjh" /><ref>{{cite web|url=http://cpc.people.com.cn/GB/64162/64168/64567/65446/6415720.html|title=中国共产党章程（中国共产党第十四次全国代表大会部分修改，1992年10月18日通过）|publisher=人民网}}</ref>
