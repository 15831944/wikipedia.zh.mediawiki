'''中華民國與英國關係'''，亦稱為'''臺英關係'''，是指[[中華民國|中華民國]]與[[大不列顛及北愛爾蘭聯合王國|大不列顛及北愛爾蘭聯合王國]]（通稱'''英國'''、'''[[不列顛|不列顛]]'''、'''聯合王國'''）之間的關係。1913－1950年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

[[英國政府|英國政府]]高度評價[[臺灣民主化|台灣的政治民主化]]成就，並列為強化經貿關係的[[新興市場|新興市場]]，除了增派駐臺人員，也積極發展與台灣的[[經濟|經貿]]、[[科技|科技]]、[[教育|教育]]、[[能源|能源]]、[[運輸|運輸]]、[[文創|文創]]、[[金融|金融]]及[[智慧城市|智慧城市]]等領域的交流與合作。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

=== 北洋政府 ===
1911年，[[辛亥革命|辛亥革命]]时期，英国公使[[朱尔典|朱尔典]]支持[[袁世凯|袁世凯]]东山再起。

1912年，中華民國成立後，向英國派駐外交代表。

1913年10月6日，英國[[外交#外交上的承認|外交承認]]中華民國[[北洋政府|北洋政府]]，於首都[[倫敦|倫敦]]設立'''中華民國駐[[大不列顛及愛爾蘭聯合王國|大不列顛及愛爾蘭聯合王國]][[外交代表機構#公使馆|公使館]]'''（1927年，變更國號為[[大不列顛及北愛爾蘭聯合王國|大不列顛及北愛爾蘭聯合王國]]），並派駐[[特命全權公使|公使]]。

1925年，發生英軍射殺中國人的[[沙基慘案|沙基慘案]]，廣東軍政府號召粵港兩地逕行罷工示威，並對香港進行封鎖，是為[[省港大罷工|省港大罷工]]。

=== 國民政府 ===
[[File:INF3-334_Unity_of_Strength_American,_Chinese_and_British_soldiers_with_flags_of_their_countries.jpg|thumb]]中華民國及英美[[同盟國_(第二次世界大戰)|同盟]][[海報|海報]]]]
{{see also|英國駐中華民國大使館舊址}}
1927年，在[[国民革命军北伐|国民革命军北伐]]中，[[国民政府|国民政府]]收回[[汉口|汉口]]、[[九江|九江]]的[[租界#各国在华租界数目|英租界]]<ref>中文本原存於中華民國外交部，現寄存於臺北外雙溪[[國立故宮博物院|國立故宮博物院]]恆溫恆濕的庫房保存</ref>。

1928年12月20日，隨著[[國民革命軍北伐|國民革命軍北伐]]幾近尾聲，英國外交承認中華民國[[國民政府|國民政府]]。

[[中國抗日戰爭|中國抗日戰爭]]初期，中華民國曾多次向英國求助被拒，[[蔣介石|蔣介石]]委員長甚至親自向當時的英國大使卡爾表示，中華民國政府可以提供20萬人替英國保衛香港，依舊被英國拒絕<ref name="援助">{{cite news|title=〈蘇聯、英國、美國對華援助〉|work=《明報》|date=2015-08-09 |pages = 新聞專題A12-A13版}}</ref>。

1935年5月23日，建立大使級外交關係。公使館升格為'''中華民國駐大不列顛及北愛爾蘭聯合王國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1938年10月，華南、華中相繼失陷，英國在華勢力範圍基本為日本控制；英國開始援華，直到1941年12月[[太平洋戰爭|太平洋戰爭]]爆發，英國對華提供財政援助貨款共1,550萬[[英鎊|英鎊]]，其中三分之二是1940年後提供，對華軍援微乎其微<ref name="援助"/>。

1943年1月11日，《中英新约》签字，除[[香港|香港]][[九龙|九龙]]外，英国放弃在中国的特殊权利<ref>中文本原存於中華民國外交部，現寄存於臺北外雙溪國立故宮博物院恆溫恆濕的庫房保存 </ref>。

1946年3月，英國在[[淡水區|淡水]][[紅毛城|紅毛城]]旁重新復館，作為英國[[領事館|領事館]]台灣分館的辦公地點，一直租用至1972年。

1947年5月9日，中英雙方聯合發表新聞，即將簽訂中英空中運輸協定<ref name="民國史大事記">{{cite book |editors = 李新 總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室 編，韓信夫、姜克夫 主編 |title=《中華民國史大事記》|location=[[北京|北京]] |publisher=[[中華書局|中華書局]] |date=2011年7月}}</ref>{{rp|8351}}。5月16日，英國太平洋艦隊總司令鮑毅德率官兵130餘人訪問福州<ref name="民國史大事記" />{{rp|8355}}。7月23日，《中英航空協定》在南京簽字<ref name="民國史大事記"/>{{rp|8388}}。

1949年5月25日，[[美國國務院|美國國務院]]機密檔案紀錄：提到一名[[英國外交部|英國外交部]]的助理秘書{{link-en|埃斯勒·德寧|Esler Dening}}以個人的名義表示：「萬一有中國[[亡命政府|亡命政府]]或中國[[流亡政府|流亡政府]]設在臺灣──其目前還不是中國的合法[[領土|領土]]──[[英國政府|英國政府]]很可能就在[[淡水郡|淡水]]設立一間[[領事館|領事館]]作為英國在中國的[[大使館|大使館]]辦公室。」並提到德寧認為：「任何設在臺灣的中國政府都會處於一個非常模糊不清的處境，並且會給世界各國的政府以及尤其是[[聯合國|聯合國]]帶來難題。」<ref>[http://digicoll.library.wisc.edu/cgi-bin/FRUS/FRUS-idx?type=turn&entity=FRUS.FRUS1949v09.p0352&id=FRUS.FRUS1949v09&isize=M United States Department of State / Foreign relations of the United States, 1949. The Far East: China(1949)], University of Wisconsin Digital Collections</ref>

=== 中華民國政府遷台後 ===
1950年1月6日，英國承認[[中華人民共和國|中華人民共和國]]（未建立外交關係），中華民國外交部發表聲明，對英斷絕邦交，撤回駐英大使<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|528}}。14日，外交部宣布關閉駐英國大使館。但英國於[[台北|台北]][[淡水區|淡水]]設置的領事館（[[紅毛城|紅毛城]]）仍然維持其領務及商務功能，並實際上仍具有領事館的功能。<ref name="台英關係"/><ref name="104年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>同年7月26日，[[英國外交部|英國外交部]]官員{{link-en|肯尼斯·楊格|Kenneth Younger}}發表書面聲明稱：「[[英國政府|英國政府]]在法律上承認中共為中國之[[合法政府|合法政府]]……台灣在法律上迄為日本領土，故無所謂台灣政府。[[日本投降|日本投降]]後當時之[[中華民國_(大陸時期)|中國政府]]，經其餘[[同盟國_(第二次世界大戰)|盟國]]之同意，取得台灣之臨時治理權，但仍須俟[[和約|和約]]對其地位作最後之決定。」<ref>{{Cite book |author = 酆邰 |title = 《台灣法律地位問題的研究》 |publisher = [[黎明文化事業公司|黎明文化事業公司]] |date = 1985 |page = 第60頁}}</ref>11月30日，英國在[[聯合國安全理事會|聯合國安全理事會]]上否決中共代表[[伍修權|伍修權]]等提出之「美國侵略台灣案」<ref>{{cite conference |title=聯合國安全理事會正式紀錄 第七十二號 第五三0次會議 |date=1950-11-30|publisher=[[聯合國安全理事會|聯合國安全理事會]] |location=[[紐約|紐約]]發拉星草場 |url = http://www.un.org/en/ga/search/view_doc.asp?symbol=S/PV.530&referer=/english/&Lang=C |accessdate=2015-08-24 |language=zh-hant}}</ref>。

1951年9月8日，英國簽署《[[舊金山和約|舊金山和約]]》，該約規定日本放棄台灣與澎湖，而[[台灣地位未定論|未明定其歸屬]]。

1958年9月11日，[[八二三炮戰|八二三炮戰]]期間，[[英國外相|英國外相]][[塞爾文·勞埃|塞爾文·勞埃]]表示，美國協防外島乃至使用[[核武器#武器的投送|戰術性核武]]顯然會有連鎖反應的危險，而中華民國自外島撤軍則可加強其國際地位。<ref>[[美國國防部|美國國防部]]助理部長辦公室委託[[蘭德公司|蘭德公司]]研究著作、謝永湉譯，1985年，《一九五八年台灣海峽危機》，[[中華民國國防部|中華民國國防部]]史政編譯局</ref>{{rp|227-236，266}}

1970年5月8日，新任英國駐淡水領事戴斐（Thomas Dussy）抵達臺灣履任<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|787}}。為迄今最後一位正式派駐台灣的英國籍外交官。

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括英國。]]

===背景===
[[英國貿易投資總署|英國貿易暨投資署]]自1998年起，將台灣列為全球14個加強出口拓銷重點市場之一，以參展、組團及邀訪等方式，開拓台灣市場；[[台灣商人|台灣廠商]]則以訪問拓銷團、貿易洽談說明會、參展或在英國設立據點等方式，開拓英國市場。雙邊實質合作近年成長迅速，並建立數個對話機制，包括台英[[次長|次長]]級經貿諮商會議（第20屆）、[[可再生能源|可再生能源]]圓桌論壇（第12屆）、[[智慧城市|智慧城市]]交流論壇（第1屆）、[[鐵道|鐵道]]工業論壇等。2017年舉辦的臺英經貿諮商會議，進一步成立[[農業|農業]]、[[金融服務|金融服務]]等對話機制，促進雙邊關係發展。<ref name="106年外交年鑑"/>

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於首都[[倫敦|倫敦]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#EUROPE |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]亦在倫敦設立'''駐英國代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2017年，英國是中華民國第17大貿易夥伴、第23大[[進口|進口]]夥伴、第13大[[出口贸易|出口]]夥伴，在[[歐洲國家和地區列表|歐洲經濟體]]排名第3、6、3。出口至英國的金額為37億8,548萬2,119美元，年增3.911%；自英國進口的金額為19億4,493萬3,686美元，年增5.469%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）18億4,054萬8,433美元，年增2.315%。※2016年，英國是中華民國第16大貿易夥伴、第23大進口夥伴、第13大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

中華民國多出口[[電子產品|電子]][[機械|機械]]、[[車輛|車輛]]、[[自動資料收集|自動資料處理機]]、[[積體電路|積體電路]]微組件等產品至英國，而英國多出口電子機械、[[汽油|油料]]產品、車輛、[[藥品|藥品]]、渦輪[[噴射引擎|噴射引擎]]及[[有機化學|有機化學]]品等產品至中華民國<ref>[http://www.cieca.org.tw/DB/tjybb_detail.asp?Item=8&name=U.K.%AD^%B0%EA 中華民國國際經濟合作協會 國家基本檔：英國]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

===投資===
中華民國於英國投資之製造業佔中華民國投資[[歐洲國家和地區列表|歐洲]]總額的70%，[[製造業|製造業]]廠商以[[電腦|電腦]]電子相關產品之製造、組裝、維修、[[行銷|行銷]]及[[配送|配送]]為主，投資廠商包括[[宏碁|宏碁]]、[[明基|明基]]、[[華碩|華碩]]、[[仁寶電腦|仁寶]]、[[鴻海|鴻海]]，[[友訊|友訊]]、[[英業達|英業達]]、[[錸德|錸德]]、[[東元|東元]]及[[大同公司|大同]]等，而其次投資為[[金融服務|金融服務]]與[[海運|海]][[空運|空]]運輸，其中包含[[臺灣銀行|臺灣銀行]]、[[第一銀行|第一銀行]]、[[彰化銀行|彰化銀行]]、[[華南銀行|華南銀行]]、[[兆豐銀行|兆豐銀行]]、[[中國信託|中國信託]]、[[長榮海運|長榮海運]]、[[中華航空|中華航空]]、[[陽明海運|陽明海運]]等企業。

1995年11月，[[中華映管|中華映管]]在英國[[蘇格蘭|蘇格蘭]]投資2億6,000萬[[英鎊|英鎊]]（約105億新台幣）設廠生產[[映像管|映像管]]，為當地帶來3,300個工作機會，是英國史上最大單筆投資案。該廠第1期工程於1997年11月舉行開幕典禮，英國女王[[伊莉莎白二世|伊莉莎白二世]]及[[蘇格蘭事務大臣|蘇格蘭事務大臣]]均應邀出席。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-4.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1998年，台灣廠商不受[[亞洲金融風暴|亞洲金融風暴]]影響，在英國有多筆投資案，其中[[宏碁集團|宏碁集團]]所屬[[明基電通|明基電通]]即於[[威爾斯|威爾斯]]投資4,000多萬美元，設廠生產[[監視器|監視器]]等電腦周邊設備產品。1998年7月，英國王儲[[威爾斯親王查爾斯|查爾斯]]以晚宴歡迎宏碁集團董事長[[施振榮|施振榮]]，肯定其在英國投資的貢獻。<ref name="87年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1998/section_3/page3-1-4.htm |title = 《中華民國87年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

==交流==
===教育===
{{see also|臺灣教育|英國教育}}
英國於1989年在臺灣成立英國教育中心（Anglo-Taiwan Education Centre），用以推展文教業務及提供台灣學生赴英國留學、進修等資訊服務，該中心於1993年與英國貿易促進會合併擴編為英國貿易文化辦事處，即今日的[[英國在台辦事處|英國在台辦事處]]。<ref name="86年外交年鑑"/>

近年來，台灣留學英國的學生持續增加，包括就讀正規學校及短期進修在內，每年約20,000人，而英國也是台灣赴歐洲留學的首選。<ref name="105年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

===文化===
{{see also|臺灣文化|英國文化}}
自2013年起，推動「[[臺灣文化|臺灣文化]]光點計畫」，並與英國[[索立大學|索立大學]]、[[愛丁堡大學|愛丁堡大學]]、[[倫敦大學亞非學院|倫敦大學亞非學院]]、[[倫敦大學柏貝克學院|倫敦大學柏貝克學院]]、{{link-en|曼徹斯特華人當代藝術中心|Centre for Chinese Contemporary Art}}等機構合作推出系列文化活動，將臺灣文化推廣至英國社會；2014年起，舉辦「[[愛丁堡國際藝穗節|愛丁堡國際藝穗節]]－臺灣季」活動。<ref name="106年外交年鑑"/>

===社運===
2007年，台灣旅英僑學界人士在倫敦舉行支持[[1971年至今中華民國與世界衛生組織的關係|台灣加入世界衛生組織]]、支持[[2008年臺灣入聯公民投票|台灣加入聯合國]]等活動，英國國會台英國會小組共同主席{{link-en|理查·福克納|Richard Faulkner, Baron Faulkner of Worcester|福克納}}等議員到場支持。<ref name="96年外交年鑑"/>
