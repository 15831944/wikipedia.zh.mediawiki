'''中和區'''是[[台灣|台灣]][[新北市|新北市]]下轄的[[市轄區|市轄區]]之一，位於[[台北盆地|台北盆地]]西南側，絕對位置約東經121.26度至121.31度，北緯24.57度至25.01度之間。與[[永和區|永和區]]關係密切，為[[雙城|雙城]]區關係，常合稱為[[雙和|雙和]]地區。

中和區為[[臺北都會區|臺北都會區]]的衛星城市之一，人口密集，總人口數多達41.3萬人，為台灣[[直轄市|直轄市]]中人口數第五多的[[市轄區|市轄區]]，人口密度高達每平方公里2.05萬人，為新北市人口密度第六高的市轄區。<ref>[[:en:List_of_cities_proper_by_population_density|List of cities by population density]]{{en}}</ref>

== 政區 ==

由於目前人口眾多，習慣上於其下劃分[[次分區|次分區]]。有東側[[秀山|秀山]]、西側[[積穗|積穗]]、南側[[南勢角|南勢角]]、北側[[漳和|漳和]]等四個次分區<ref>[http://www.sinyi.com.tw/knowledge/search_news.php/155/0/12/kmw5xy 新北市中和區成屋市場專題報導]</ref>。唯此劃分僅是為行政便利所作的權宜措施，無獨立的次分區機關，次分區亦非法律上的公法人。同樣的情況亦出現於台灣各個其他高度[[都市化|都市化]]的[[市轄區|市轄區]]，如[[板橋區_(台灣)|板橋區]]、[[新莊區|新莊區]]、[[三重區|三重區]]、[[桃園市|桃園市]][[桃園區|桃園區]]、[[高雄市|高雄市]][[鳳山區|鳳山區]]。

== 地理與氣候 ==
中和區位於[[臺北市|臺北市]]的西南方，東北與[[永和區|永和區]]交界；西北與[[板橋區_(台灣)|板橋區]]接攘；北隔[[新店溪|新店溪]]與[[臺北市|臺北市]][[萬華區|萬華區]]相望，東南依[[新店溪|新店溪]]及鹿寮山地與[[新店區|新店區]]為界；西與[[土城區|土城區]]相鄰；面積為20.294平方公里 。

地形輪廓呈南尖北寬之形狀。南端地勢較高，然高度最高僅海拔二百餘公尺，為丘陵及礫石台地地形。而北部地形則為[[台北盆地|台北盆地]]的沖積平原，海拔高度即降於不到十餘公尺。

新店溪為東端與北端界河，也是[[淡水河|淡水河]]水系的最大支流，其上游地區雨量豐沛且季節分配均勻，故終年流量大穩定。雖然名稱為「溪」，但實際上具「河」的型態。早年頗富航運之利，往上游可達[[景美|景尾]]、[[新店區|新店]]等港口，往下游可達淡水河的[[艋舺|艋舺]]、[[大稻埕|大稻埕]]等港口。後來因泥沙汙積、河川污染及陸運發達等因素之衝擊，航運早已終止。

[[File:20180519廟仔尾溝台灣魚木.jpg|thumb]]
[[File:胖葫蘆池_20100328.jpg|thumb]]

[[南勢角溪|南勢角溪]]又名南山溝，為新店溪的支流，也是一條貫穿中和的細長河川，發源於橫路里山區，全長約8公里<ref>[https://www.zhonghe.ntpc.gov.tw/content/?parent_id=10037&type_id=10006 地理環境]，中和區公所</ref>。[[二八張溝|二八張溝]]是南勢角溪西側支流，在勝利橋附近匯入南勢角溪<ref>{{Cite web |url=http://publish.ntpc.gov.tw/Book/laborbook/no.11/right2/right_2_02_04.htm |title=中和排水路主流段開闢工程 |access-date=2011-06-17 |archive-url=https://web.archive.org/web/20150119144624/http://publish.ntpc.gov.tw/Book/laborbook/no.11/right2/right_2_02_04.htm |archive-date=2015-01-19 |dead-url=yes }}</ref>。[[瓦磘溝|瓦磘溝]]又名潭墘溝，為南勢角溪的主要支流之一，其中、下游及上游一小段為本區與永和區的界河。早年南勢角溪、瓦磘溝下游均可通行小舟，現已都被整治成為典型的大排水溝，其中二八張溝下游因水患工程整治為排水路主線，改由中和抽水站直接排入新店溪。

[[中原溝|中原溝]]也是新店溪的支流，流經中原里，現為工業排水溝，由中原抽水站排入新店溪。

中和區沒有大型湖泊，在中和國中附近曾有一大型灌溉水塘，名為南勢角大陂，今已填平<ref>[http://www1.geo.ntnu.edu.tw/download.php?filename=38_5910fe4e.pdf&dir=publish&title=%E5%85%A8%E6%96%87%E4%B8%8B%E8%BC%89 台北地區漢人聚落拓墾與地形之相關研究]</ref>。在灰磘里的軍營則尚存一埤塘，湖面面積約0.5公頃，地主亦已將水放乾。

中和區氣候基本上與台北盆地相似。台北盆地每年平均降雨量為2111毫米，平均溫度為攝氏21.7度。但本區位置較偏於山地邊緣，所以山地降雨量較盆地之平原中心略高。

== 歷史 ==
中和區有兩處史前遺跡，分別是東側秀山區的[[尖山_(新北市)|尖山]]及西側員山區的[[外員山|員山子]]兩處，根據推測前者應是屬於圓山前期文化的系統，距今約3000-4200年間；後者文化橫跨了繩紋陶系統，距今約2000-4200年間。由此兩處史前文化遺址可得知，中和遠從遠古時期即有人在此定居。

文字記載中和區早期為[[台灣原住民|原住民族]][[凱達格蘭人|凱達格蘭人]]（屬[[平埔族|平埔族]]）所居住，分為「擺接」及「秀朗」（繡朗）兩社<ref>《諸羅縣志》</ref><ref>郁永和，《裨海記遊》</ref>。據考證，擺接社即今板橋區一帶及本區枋寮的西北地區；秀朗社即今秀朗、秀山兩里附近地區。至1642年（[[明|明]][[崇禎|崇禎]]十五年）荷蘭人所製的戶口名冊，始有Chiron（秀朗）之記載，是為中和最早見諸文字的紀錄，但如何由圓山文化與繩紋陶文化的人種族群演變到凱達格蘭人則因資料不足而難以得知<ref name="中和的史前遺跡">[http://blog.roodo.com/hotown/archives/4126571.html 《中和的史前遺跡》，中和庄文史研究協會]</ref>。

1661年（南明[[永曆|永曆]]十五年，[[清|清]][[順治|順治]]十八年），鄭成功佔領台灣後規置郡縣，北部台灣皆隸[[天興縣|天興縣]]。1664年升天興縣為州，中和地區亦改隸[[天興州|天興州]]。1683年（永曆三十七年，清[[康熙|康熙]]二十二年），鄭氏政權滅亡，清廷重新規劃郡縣，以天興州舊北界，置[[諸羅縣|諸羅縣]]，所以中和地區又改屬諸羅縣。在此之後，中和地區仍屬荒地，不見有漢人居住的記載；而後隨著郡縣幾次重劃，亦分別改隸過淡水廳淡水堡、擺接堡。直到1741年（清乾隆六年）[[閩|閩]]、[[粵|粵]]來此開墾的人與日俱多，始有漢人村莊之記載。<ref>劉良璧，《臺灣府志》</ref>

1859年（[[咸豐|咸豐]]九年）九月七日[[漳泉械鬥|漳泉械鬥]]，當日瓦嗂為械鬥爆發地，枋寮街大火，造成兩地村莊皆被毀。此事件乃因造成此地泉、漳兩族群之語言、信仰、習慣不同與[[利益衝突|利益衝突]]所造成的不合，械鬥之後影響甚久，曾有很長一段時間兩族群彼此視為世仇，互不來往、互不通婚、互不為友。

1895年（[[光緒|光緒]]二十一年）五月六日，[[日本|日本]]依[[馬關條約|馬關條約]]登陸臺灣。十一月十六日[[南勢角|南勢角]][[呂潮棟|呂潮棟]]、[[呂元典|呂元典]]兩兄弟及鄉民約兩千餘人，聯絡台北地區各反日軍勢，約定在觀音山舉火為暗號，進攻台北城；但在網尾渡（今永和區[[中正橋|中正橋]]下）為日軍所截擊，呂潮棟被捕後成仁，呂元典亦憂憤而亡死。日本為了報復，將南勢角一帶村莊悉數焚毀。

1920年，地方制度再次變革，取'''中'''坑-{庄}-之首字、漳'''和'''-{庄}-之次字，合稱[[中和庄|中和-{庄}-]]，隸屬於[[臺北州|臺北州]][[海山郡|海山郡]]，相當於現今的中永和地區。1945年日本戰敗歸還台灣後，當時中華民國國民政府為避免接收時期行政可能會有銜接不上的現象，所以暫時依循原制以維持政務。

1946年地方制度改制，廢[[街庄|街庄]]為鄉鎮，當時本區人口數不及3萬人，依制稱「中和鄉」，隸台北縣海山區。1950年8月16日廢區，由台北縣直轄。自中華民國政府遷台之後，因中和鄉鄰近中央政府所在地──台北市，又將中和鄉劃入台北市民防空疏散地區，且此地房地產價格相對較為低廉，以致人口急速增加。其中尤以靠近台北市的龜崙蘭溪洲地區人口增加最為快速，於是分鄉設鎮之議遂起。

1958年4月1日，[[中和鄉|中和鄉]]分割出龜崙蘭溪洲地區的全部六個村及潭墘、秀朗地區各一個村，正式設鎮。初擬鎮名為中興鎮，但因與當時台灣省政府所在地之[[中興新村|中興新村]]有同名之嫌，所以定名為[[永和區|永和鎮]]。不過由於某些村莊對於要屬中和鄉還是[[永和區|永和鎮]]的意見一直無法統合，以致於中和與永和之邊界直至1960年十月才確定，由縣政府派員會同雙方鄉鎮長及民政課長，於沿鄉鎮分界處豎立水泥樑柱以為界碑；自此中和、永和正式分治。<ref>盛清沂，《重修中和鄉誌》</ref>

分鄉設鎮後，中和鄉仍轄有枋寮、中原、平河、瓦磘、廟美、牛埔、灰磘、瑞穗、積穗、外南、內南、橫路、頂南、力行、安樂、秀山等十六村，總面積為19.71平方公里，共轄176鄰、4,619戶、23,312人<ref>[https://www.zhonghe.ntpc.gov.tw/content/?parent_id=10038&type_id=10038 昨日中和]，中和區公所</ref>。

其後因台灣經濟結構改變，農村人口逐漸向都市遷移，中和鄉距離台北市較近，人口因而快速增加，遂繼永和之後，變成台北市的[[衛星城市|衛星城市]]。1978年底，人口數已突破17萬，隔年一月一日依「台灣省各縣市實施地方自治綱要」改制為[[縣轄市|縣轄市]]。此後人口持續快速成長，至2001年底，人口數已超過40萬。2010年因為[[五都改制|台北縣升格為新北市]]，改制為[[市轄區|市轄區]]。

== 行政組織 ==
中和區公所是中和區為新北市在此地的行政機關，依組織自治條例，依法辦理自治事項並執行上級政府委辦事項，置區長由市長派任為期4年、副區長1人、主任祕書1人、秘書1人、視導4人。設民政課、社會課、經建課、工務課、役政災防課、人文課、秘書室、會計室、人事室、政風室等單位。

== 經濟 ==
中和區早期以[[農業|農業]]、[[礦業|礦業]]為主。1961年左右，開始發展工商業，以[[紡織|紡織]]與[[印刷|印刷]]為主。近期由於一些科技園區的陸續興建，再次轉型為發展高科技產業，主要以[[福爾摩沙高速公路|國道三號]][[中和交流道|中和交流道]]附近為主要聚集地。著名的高科技公司如[[電腦|電腦]][[主機板|主機板]]製造商[[微星|微星科技]]、[[顯示卡|顯示卡]]製造商[[麗台科技|麗台科技]]等均在此地設廠，[[記憶體|記憶體]]製造商[[威剛科技|威剛科技]]與軟體業如[[大宇資訊|大宇資訊]]、[[遊戲橘子|遊戲橘子]]、[[宇峻奧汀|宇峻奧汀]]等企業亦在此設立總部。附近也因工商企業聚集而有31家[[銀行|銀行]]（共61家分行）及其他多家[[金融業|金融業]]在此設立分行據點。

==== 主要道路 ====
*[[市道106號|景平路]]-為[[市道106號|市道106號]]。與[[台64線|台64線]]共線，貫穿中和區全境，是中和區最重要也是最繁忙的道路；本路段頭尾設有秀朗橋與[[華中橋|華中橋]]，可分別通往本市[[新店區|新店區]]、[[臺北市|台北市]][[萬華區|萬華區]]，[[台北捷運|台北捷運]][[環狀線_(台北捷運)|環狀線]]沿著本路段興建，將以高架方式通過。
*[[中正路|中正路]]-連接板橋區和中和區，上方有[[台64線|台64線]]快速道路通過，交通繁忙，也是[[中和交流道|中和交流道]]必經道路；在中正路與連城路交叉口北側，辦公大樓林立，許多企業總公司位於此街廓。
*[[中山路|中山路]]-連接[[板橋區_(台灣)|板橋區]]和[[永和區|永和區]]主要道路，亦為[[市道106號|市道106號]]。
*中和路-中和區舊聚落的道路，也是中和區舊商圈核心道路。後段有[[捷運永安市場站|捷運永安市場站]]，與[[永和區|永和區]]交接(東邊為中和區中和路；西邊為[[永和區|永和區]]中和路)，以[[中和公園|中和公園]]為中心，發展出獨立於舊聚落之生活圈。
*連城路-[[雙和|雙和]]地區通往[[土城區|土城區]]最主要道路，[[台北捷運|台北捷運]][[萬大樹林線|萬大樹林線]]目前沿本路段興建中。
*[[北95線|興南路]]-貫穿南勢角地區的主要道路，沿途觀光旅遊資源豐富，興南夜市、華新街（[[緬甸|緬甸]]街）以及[[烘爐地|烘爐地]]皆位於本路段沿線。
*[[市道111號|景新街]]-南勢角主要道路，連接到[[新店區|新店區]]安坑地區的重要道路，亦為[[市道111號|市道111號]]。
*[[成功路|成功路]]-秀朗地區主要道路，貼近新店溪，連接[[永和區|永和區]]。2008年5月9日前舊名為「成功南路」。
*[[北91線|員山路]]-員山積穗地區主要道路。
*莒光路-[[外員山|外員山地區]]主要道路之一，可聯絡[[板橋區_(台灣)|板橋區]]與[[土城區|土城區]]；117巷後,也就是[[民享街(中和)|民享街]]116巷[[土地使用分區|土地使用分區]]為農業區，尚未開發。[[台北捷運|台北捷運]][[萬大樹林線|萬大樹林線]]莒光站預計設置於本路段。
*[[北90線|圓通路]]-牛埔地區(以錦中里為核心的生活圈)的主要道路，可通往[[圓通禪寺|圓通禪寺]]，[[雙和醫院|衛生福利部雙和醫院]]的院區位於本路段與中正路之間。
*自立路、自強路、更生街(橫跨[[永和區|永和區]])-秀山地區的主要道路。
*板南路-連接板橋區和中和區南勢角地區，為[[中和線_(台鐵)|台鐵中和線]]舊路廊，也是中和工業區的主要幹道之一；未來部分路段預計有[[環狀線_(台北捷運)|台北捷運環狀線]]高架段通過。

==== 趣談 ====
由於[[雙和|雙和]]地區路名錯綜複雜，常讓不熟悉[[雙和|雙和]]地區的人迷路，故[[天下雜誌|天下雜誌]]在2001年的「319鄉向前行」特刊中，稱中和為「迷宮城市」；網路也流傳一首[[打油詩|打油詩]]，說明[[雙和地區|雙和地區]]路名複雜的程度：

{{Cquote|
永和有永和路，中和也有永和路，<br>
中和有中和路，永和也有中和路；<br>
中和的中和路有接永和的中和路，<br> 
永和的永和路沒接中和的永和路；<br> 
永和的中和路有接永和的永和路，<br>
中和的永和路沒接中和的中和路。<br>

永和有中正路，中和也有中正路，<br>  
永和的中正路用景平路接中和的中正路；<br>
永和有中山路，中和也有中山路，<br>
永和的中山路直接接上了中和的中山路。<br>
永和的中正路接上了永和的中山路，<br>
中和的中正路卻不接中和的中山路。<br>

中正橋下來不是中正路，但永和有中正路；<br>
秀朗橋下來也不是秀朗路，但永和也有秀朗路。<br>
永福橋下來不是永福路，永和沒有永福路；<br>
福和橋下來也不是福和路，但福和路接的卻是永福橋。}}
