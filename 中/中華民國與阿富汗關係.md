'''中華民國與阿富汗關係'''是指[[中華民國|中華民國]]與[[阿富汗伊斯蘭共和國|阿富汗伊斯蘭共和國]]之間的關係。兩國於1944－1950年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對阿富汗的相關事務由'''{{link-en|駐沙烏地阿拉伯王國台北經濟文化代表處|Taipei Economic and Cultural Representative Office in the Kingdom of Saudi Arabia}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-190-fde3f-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[阿富汗王國|阿富汗王國]]。]]
{{see also|中華民國外交|{{link-en|阿富汗外交|Foreign relations of Afghanistan}}}}
{{see also|中国驻阿富汗大使列表{{!}}中華民國駐阿富汗外交官列表|阿富汗驻华大使列表{{!}}阿富汗駐中華民國外交官列表}}
1944年3月2日，兩國駐[[土耳其|土耳其]]使節簽訂《中國阿富汗友好[[條約|條約]]》，建立公使級[[外交#外交上的承認|外交關係]]。<ref name="條約協定">{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e9%98%bf%e5%af%8c%e6%b1%97&tycountry_e=%e9%98%bf%e5%af%8c%e6%b1%97&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

1945年，於首都[[喀布爾|喀布爾]]設立'''中華民國駐[[阿富汗王國|阿富汗王國]][[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1950年1月12日，阿富汗外交承認[[中華人民共和國|中華人民共和國]]。1月14日，中華民國關閉公使館。1955年1月20日，與中華人民共和國建立外交關係，並派駐[[特命全權大使|大使]]。

1971年10月25日，阿富汗反對[[中國與聯合國#代表權問題|中華民國續留聯合國]]，支持中華人民共和國取代成為[[聯合國會員國列表|聯合國會員國]]的「[[中國|中國]]」席次。

2002年11月14日，外傳{{link-en|阿富汗臨時政府|Afghan Interim Administration}}主席[[卡爾札伊|卡爾札伊]]、副主席與外交、軍事官員於年初搭乘[[中華航空|中華航空]]經[[中正國際機場|中正國際機場]]轉赴[[東京|東京]]參加[[反恐战争|反恐]]與重建會議。[[中華民國外交部|中華民國外交部]]對此不置可否，但根據當天參與的人士透露，外交部長[[田弘茂|田弘茂]]曾前往[[機場貴賓室|機場貴賓室]]會晤卡爾札伊，表達[[臺灣|臺灣]]對阿富汗的[[人道援助|人道支援]]，卡爾札伊並向臺灣關懷阿富汗重建工作表達誠摯的謝意。<ref>{{Cite web |url = http://old.ltn.com.tw/2002/new/nov/15/today-p12.htm |author = 鄒景雯 |date = 2002年11月15日 |title = 我民航機載運阿富汗官員 八月已見報 |publisher = [[自由時報|自由電子新聞網]]}}</ref>

2016年10月19日，簽訂《[[中華民國法務部調查局|中華民國法務部調查局]]洗錢防制處與阿富汗伊斯蘭共和國{{link-en|阿富汗銀行|Da Afghanistan Bank}}阿富汗金融交易及報告分析中心關於涉及[[洗錢|洗錢]]、相關前置犯罪及資助[[恐怖主義|恐怖主義]]{{link-en|金融情報|Financial intelligence|金融情資}}交換合作瞭解[[備忘錄|備忘錄]]》。<ref name="條約協定"/>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境阿富汗。]]
[[Image:Visa_requirements_for_Afghan_citizens.png|thumb]]的[[阿富汗人口|阿富汗公民]]必須申請簽證方可入境[[中華民國|中華民國]]。]]

{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|阿富汗簽證政策|阿富汗公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]若需辦理阿富汗簽證，可前往阿富汗駐[[日本|日本]]、駐[[韓國|韓國]][[外交代表機構#大使馆|大使館]]聯繫事宜。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-190-fde3f-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>若前往{{link-en|阿富汗駐巴基斯坦大使館|Embassy of Afghanistan, Islamabad|駐巴基斯坦大使館}}申請簽證，則必需出具[[中華人民共和國|中華人民共和國]]大使館的「[[無異議證明|無異議證明]]」。<ref>{{Cite web |url = https://udn.com/news/story/6656/3338965 |author = 徐偉真 |date = 2018年8月30日 |title = 李昂登阿富汗受阻 外交部：需出具中國使館「無異議證明」 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref>

持[[阿富汗護照|阿富汗護照]]的[[阿富汗人口|阿富汗公民]]抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將阿富汗歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2018年1月23日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-166-b115f-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將阿富汗歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2015年11月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=C4DA81F55DFF6D8E |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

{{see also|臺灣經濟|阿富汗經濟}}
===貿易===
2018年，阿富汗是中華民國第176大貿易夥伴、第201大[[進口|進口]]夥伴、第164大[[出口贸易|出口]]夥伴，在[[亞洲國家和地區列表|亞洲國家]]排名倒數第5、4、6；在[[世界政區索引|世界經濟體]]倒數第69、40、73。出口至阿富汗的金額為174萬8,647[[美元|美元]]，年增3.795%；自阿富汗進口的金額為6萬6,038美元，年增216.198%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）168萬2,609美元，年增1.129%。2017年，阿富汗是中華民國第178大貿易夥伴、第208大進口夥伴、第167大出口夥伴，在亞洲國家排名倒數第6、3、8；在世界經濟體倒數第61、29、64。<ref>{{Cite web |url = https://cus93.trade.gov.tw/|title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
