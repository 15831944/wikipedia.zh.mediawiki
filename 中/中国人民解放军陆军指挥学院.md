'''中国人民解放军陆军指挥学院'''，位于[[江苏省|江苏省]][[南京市|南京市]]，隶属[[中国人民解放军陆军|中国人民解放军陆军]]，是培养陆军指挥人才的陆军指挥院校。
[[File:PLA_Nanjing_Army_Command_College.jpg|thumb]]

== 沿革 ==
*1938年11月26日，在[[河南省|河南省]][[杞县|杞县]]傅集成立'''新四军游击支队随营学校'''<ref name=mod>{{Cite web |url =http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536842.htm  |title =南京陆军指挥学院  |author =  |date = 2014-09-12 |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref>。新四军游击支队司令员兼政治委员[[彭雪枫|彭雪枫]]兼任校长。
*1940年3月18日，扩建为'''中国人民抗日军政大学第四分校'''，是新四军成立最早、历时最长的学校。1944年9月11日，校长彭雪枫在战斗中牺牲。
*1945年3月24日，为纪念彭雪枫，将抗日军政大学第四分校命名为'''雪枫军政大学'''，由新四军第四师师长[[张爱萍|张爱萍]]兼任校长。
*1946年4月，雪枫军政大学、苏中公学、华中野战军随营干部学校合并组建'''华中雪枫大学'''，由[[粟裕|粟裕]]兼任校长。
*1946年11月25日，由华中雪枫大学、山东军区军政学校、东江纵队、淮南随营学校、山东军区通信学校等5个单位统一改编为'''华东军事政治大学'''，在山东省[[莒南县|莒南县]]大店举行学校成立大会暨第一期开学典礼。
*1949年9月1日，经总政治部同意，华东军区、第三野战军前委决定，以在山东（后来在江苏[[苏州|苏州]]）办学的原华东军事政治大学、在江苏[[常熟|常熟]]办学的第三野战军军政干校、第八兵团司政供卫机构为基础，组成新的'''华东军事政治大学'''。设校务部、训练部、政治部。设第二总队（第六至第九团，驻苏州）、第三总队（第十一至第十五团）、工人子弟团（驻[[上海|上海]]）、高干团（第一团，1949年9月15日在南京成立）。1949年10月18日，在南京举行第一期开学典礼。1949年11月，伞兵总团拨归华东军事政治大学，华东军事政治大学警卫团拨归南京警备司令部。1950年1月22日，第二总队奉命从苏州移驻南京[[孝陵卫|孝陵卫]]。1950年1月28日，工人子弟团从上海迁南京，转归伞兵总团领导。1950年5月，工人子弟团拆散补入第九兵团。1950年6月，学校精简原第二、第三总队以及团一级机构，新组建7个总队<ref name=jsszjsz> {{Cite book | author = 江苏省地方志编纂委员会编著 | title =  江苏省志·军事志| location =北京  | publisher = 军事科学出版社 | date = 2000年 | pages = 196-197, 199-200 | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。
*1950年11月28日，奉令撤销华东军事政治大学。1950年12月，第一总队及第一医院调归[[中国人民解放军军事学院|中国人民解放军军事学院]]，第五、第七总队与华东军区教导总队合并<ref name=jsszjsz/>。
*1951年1月，以原华东军事政治大学直属队和第二、第三、第四、第六总队为基础，改编组成'''华东军区高级步兵学校'''，驻南京孝陵卫，直属华东军区建制领导<ref name=jsszjsz/>。
*1951年3月5日，华东军区高级步兵学校改称'''中国人民解放军第三高级步兵学校'''<ref name=jsszjsz/>。
*1952年4月28日，第三高级步兵学校调归中央军委，组建'''中国人民解放军总高级步兵学校'''<ref name=jsszjsz/>。1952年5月2日正式组建<ref name=zgjw/>。1953年1月10日，中国人民解放军总高级步兵学校在南京正式开学。1958年11月14日，中央军委决定将总高级步兵学校并入中国人民解放军军事学院<ref name=jsszjsz/>。
<br>
*1974年1月27日，[[南京军区|南京军区]]决定筹建'''中国人民解放军南京军区军政干部学校'''。1974年12月成立。驻南京浦口花旗营。1976年4月19日举行开学典礼。设4个大队（第一、第二大队是军事大队，第三大队是政治大队，各编3个学员队。外训大队编2个学员队）<ref name=jsszjsz/>。
*1978年1月，中央军委命令以位于花旗营工地的南京军区军政干部学校为基础，升格改建为'''中国人民解放军南京高级步兵学校'''，隶属中央军委<ref name=jsszjsz/>。
*1980年12月，更名为'''中国人民解放军南京高级陆军学校'''<ref name=jsszjsz/>。
*1986年7月，更名为'''中国人民解放军陆军指挥学院'''，是培养合成军队军官的中级指挥院校，归[[中国人民解放军总参谋部|中国人民解放军总参谋部]]建制，仍驻南京浦口花旗营<ref name=jsszjsz/>。
*1999年，更名为'''中国人民解放军南京陆军指挥学院'''<ref name=mod/><ref name=zgjw>{{Cite web |url =http://www.81.cn/jwzl/2015-11/25/content_6787091.htm  |title =中国人民解放军南京陆军指挥学院  |author =  |date =2015-11-25  |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref>。正军级。学院是军队院校“2110工程”三期重点建设院校<ref name=mod/>。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，该校由[[中国人民解放军总参谋部|中国人民解放军总参谋部]]转隶[[中国人民解放军陆军|中国人民解放军陆军]]<ref>{{Cite web |url = http://www.81.cn/lj/2016-01/31/content_6882208.htm  |title = 南京陆军指挥学院：融入改革“大棋盘”走好自己“小棋盘” |author =  |date = 2016-01-31 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，该校更名为'''中国人民解放军陆军指挥学院'''<ref>{{Cite web |url = http://edu.people.com.cn/GB/n1/2017/0630/c1053-29373203.html |title =  国防部公布调整改革后军队院校名称|author =  |date =2017-06-30  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
