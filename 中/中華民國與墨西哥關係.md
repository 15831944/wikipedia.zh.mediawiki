'''中華民國與墨西哥關係'''是指[[中華民國|中華民國]]與[[墨西哥合眾國|墨西哥合眾國]]之間的關係。1913－1972年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後直至1990年代，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[墨西哥|墨西哥]]。]]
{{see also|中華民國外交|{{link-en|墨西哥外交|Foreign relations of Mexico}}}}
{{see also|中国驻墨西哥大使列表{{!}}中華民國駐墨西哥外交官列表|墨西哥驻华大使列表{{!}}墨西哥駐中華民國外交官列表}}
1912年，中華民國成立後，向墨西哥派駐外交代表。

1913年5月2日，墨西哥[[外交#外交上的承認|外交承認]]中華民國，於首都[[墨西哥城|墨西哥城]]設立'''中華民國駐墨西哥[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。<ref name="99年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2010/2.1.6.pdf |title = 《中華民國99年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

1944年7月23日，公使館升格為'''中華民國駐墨西哥[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

中華民國除了在首都墨西哥城設有大使館，並分別在[[下加利福尼亞州|下加利福尼亞州]]首府[[墨西加利|墨西加利]]及墨西哥第2大城[[瓜達拉哈拉_(墨西哥)|瓜達拉哈拉]]設有總領事館及[[領事館|領事館]]，並派駐[[農業技術|農業技術]]人員。<ref name="86年外交年鑑"/>

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1971年10月25日，墨西哥投票反對[[中國與聯合國#代表權問題|中華民國續留聯合國]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。11月16日，墨西哥與中華民國斷交，<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/11.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>11月19日，大使館關閉。

====中華民國總統專機繞道====
2007年1月，中華民國總統[[陳水扁|陳水扁]]出訪[[中華民國外交#邦交國|中南美州友邦]]（嘉欣專案）返程過境[[美國|美國]][[洛杉磯|洛杉磯]]前，[[中華人民共和國|中華人民共和國]]向墨西哥施壓，臨時取消總統專機飛越墨西哥[[飛航情報區|飛航情報區]]之許可，被迫繞道以致抵達時間延後數小時，各項原訂行程亦大受影響。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/out/2604/report_1.htm |author = 中南美司 |date = 2007年3月 |title =「嘉欣專案」-攜手共榮、邦誼永續之旅 |publisher = 《外交部通訊》第廿六卷第四期}}</ref>

2018年8月，中華民國總統[[蔡英文|蔡英文]]出訪中南美州友邦（同慶之旅）的專機在過境美國洛杉磯後，準備飛往[[巴拉圭|巴拉圭]]，原本最直接的飛行路線，是沿著墨西哥的太平洋海岸線南下前往南美洲，但據了解，[[中華航空|中華航空]]在接洽時「注意到墨西哥的態度一度猶疑不定」，為避免不必要的飛航風險，最後選擇繞開墨西哥飛航情報區再直飛巴拉圭。<ref>{{Cite web |url = http://news.ltn.com.tw/news/focus/paper/1224670 |author = 蘇永耀、呂伊萱 |date = 2018年8月15日 |title = 墨西哥態度猶疑 總統專機繞道 |publisher = 《[[自由時報|自由時報]]》}}</ref>

===代表機構===
{{see also|中華民國駐外機構列表|駐臺灣外交機構列表}}
{{see also|{{link-en|墨西哥駐外機構列表|List of diplomatic missions of Mexico}}|{{link-en|駐墨西哥外交機構列表|List of diplomatic missions in Mexico}}}}
1972年1月，於首都[[墨西哥城|墨西哥城]]設立商務[[辦事處|辦事處]]。1974年10月，因中華人民共和國的壓力，墨西哥強迫辦事處人員離境，兩國關係隨即中斷。1982－1986年，[[中華民國外交部|中華民國外交部]]與[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）人員陸續前往墨西哥，促成兩國關係開始解凍。1987年4月，外貿協會與墨西哥企業家國際事務協會簽署合作協議書、互設辦事處。1989年4月19日，於墨西哥城設立'''遠東貿易服務中心駐墨西哥辦事處'''。1993年7月14日，更名為'''{{link-en|駐墨西哥臺北經濟文化辦事處|Taipei Economic and Cultural Office in Mexico}}'''（[[西班牙文|西班牙文]]：{{lang|es|Oficina Económica y Cultural de Taipei en México}}），隸屬於中華民國外交部，墨西哥並給予部分[[外交豁免權|外交禮遇]]。<ref name="86年外交年鑑"/><ref name="外交年鑑"/><ref name="駐外館處">{{Cite web |url = http://www.mofa.gov.tw/OverseasOffice_Content.aspx?n=6357834932B83C83&sms=5A961ED2E4BA25E8&s=85E1E406503C665B&os=12AED0688767E7CF |title = 駐外館處 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref><ref name="駐館與駐地關係">{{Cite web |url = http://www.roc-taiwan.org/mx/post/13.html |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref>

1990年6月，墨西哥負責對外貿易投資事務的國營外貿銀行於首都[[臺北市|臺北]]設立'''墨西哥駐華商務辦事處'''（Mexican Trade Services）。1997年，{{link-en|墨西哥經濟部|Secretariat of Economy (Mexico)}}成立{{link-en|墨西哥貿易投資局|ProMéxico|貿易投資局}}（ProMéxico）取代遭裁撤的原外貿銀行，並續指派代表主持商務辦事處業務。1991年6月，另由{{Link-en|墨西哥外交部|Secretariat of Foreign Affairs (Mexico)}}於台北設立'''{{Link-en|墨西哥商務簽證文件暨文化辦事處|Mexican Trade Services Documentation and Cultural Office}}'''（西班牙文：{{lang|es|Oficina de Enlace de México en Taiwán}}），以處理[[領事|領事]]事務，並直接核發[[簽證|簽證]]，是墨西哥駐台官方代表機構。2011年9月，有鑑於兩國在[[美墨邊界|美墨邊界]]的[[下加州|下加州]]經貿投資關係密切，因此[[中華民國政府|中華民國政府]]同意在[[台灣|台灣]]設立'''下加州政府駐全[[亞洲|亞洲]]代表處'''。<ref name="外交年鑑"/><ref name="駐館與駐地關係"/><ref name="駐華外國機構">{{Cite web |url = http://www.mofa.gov.tw/OfficesInROC_Content.aspx?n=8CEB2B5F5436B997&sms=8EBFADC1592C7BFE&s=85E1E406503C665B&os=5A7290C85CBCD419 |title = 駐華外國機構 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境墨西哥。]]
[[File:Visa_requirements_for_Mexico_citizens.png|thumb]]必須申請簽證方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|墨西哥簽證政策|墨西哥公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]及[[美國簽證政策|美國簽證]]的[[中華民國國民|中華民國公民]]，以觀光、商務或過境為目的，可[[簽證#签证优惠待遇|免簽證]]入境墨西哥。必須注意的是，墨西哥是給予持有「傳統實體有效美簽」的旅客得以免簽證入境，而中華民國成為[[美國|美國]]「[[免簽證計畫|免簽證計畫]]」的參與國後，透過「[[旅遊許可電子系統|旅行授權電子系統]]」（ESTA）所取得的赴美授權許可，並非「傳統實體有效美簽」，因此仍需事先申請美國簽證或墨西哥簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-98-dc6ec-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref>

持{{link-en|墨西哥護照|Mexican passport}}的[[墨西哥人口|墨西哥公民]]抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

===貿易===
墨西哥是中華民國在[[中南美洲|中南美洲]]的最大貿易夥伴。

1989年，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於墨西哥首都[[墨西哥城|墨西哥城]]設立'''台灣貿易中心'''。<ref name="外貿協會全球據點">{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#C-AMERICA |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref><ref name="駐館與駐地關係"/>[[經濟部國際貿易局|經濟部國際貿易局]]也在墨西哥城設立'''駐墨西哥代表處經濟組'''。<ref name="本部駐外單位">{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=215&pid=305863 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2018年，墨西哥是中華民國第24大貿易夥伴、第34大[[進口|進口]]夥伴、第18大[[出口贸易|出口]]夥伴，在[[美洲|美洲]]經濟體排名第3、5、3。出口至墨西哥的金額為24億8,402萬2,104[[美元|美元]]，年增8.742%；自墨西哥進口的金額為7億9,152萬7,110美元，年增16.465%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）16億9,249萬4,994美元，年增5.472%。2017年，墨西哥是中華民國第25大貿易夥伴、第38大進口夥伴、第17大出口夥伴，在美洲經濟體排名第4、5、2。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =經濟部國際貿易局}}</ref>

資通訊向來是出口至墨西哥的第1大類別，尤其以零組件為主。資通訊產品中的[[發光二極體|發光二極體]]（LED）／[[液晶顯示器|液晶顯示器]]（LCD）面板是主力產品，2013年，分別位居出口至墨西哥產品的第2及第4名；[[印刷電路板|印刷電路板]]也位居第3名；鋼鐵產品類，自2011年，其他鍍或塗[[鋅|鋅]]之鐵或非[[合成鋼|合金鋼]]扁軋製品（非波浪化者），更連續3年位居第1名；台灣製的[[螺絲|螺絲]][[螺帽|螺帽]]是墨西哥第2大供應來源（僅次於[[美國|美國]]），是在[[中南美洲|中南美洲]]的最大市場，勝過[[巴西|巴西]]。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/mx/pdf/103-36Mexico.pdf |date = 2014年12月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>
