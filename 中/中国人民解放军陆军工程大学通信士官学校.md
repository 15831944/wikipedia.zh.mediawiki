'''中国人民解放军陆军工程大学通信士官学校'''，位于[[重庆市|重庆市]][[沙坪坝区|沙坪坝区]][[重庆林园|林园]]甲1号，是中国人民解放军陆军工程大学培养通信士官的学校，隶属[[中国人民解放军陆军工程大学|中国人民解放军陆军工程大学]]。

== 沿革 ==
;中国人民解放军通信兵工程学院
*1955年，建立中国人民解放军重庆通信兵技术学校。
*1961年，中国人民解放军西安军事电信工程学院（雷达导航系）、中国人民解放军重庆通信兵技术学校合并组建中国人民解放军雷达工程学院
1963年，中国人民解放军雷达工程学院更名中国人民解放军通信兵工程学院。

;中国人民解放军汉口通信技术学校
*1949年，建立中国人民解放军中南军区通信学校。
*1951年，更名中国人民解放军第四通信学校。
*1953年，更名中国人民解放军通信机务学校。
*1955年，更名中国人民解放军汉口通信技术学校。

;中国人民解放军宣化通信学校

;中国人民解放军沈阳通信兵学院

;中国人民解放军重庆通信学院
*1969年，中国人民解放军通信兵工程学院、中国人民解放军汉口通信技术学校、中国人民解放军宣化通信学校、中国人民解放军沈阳通信兵学院等6所院校合并，成立'''中国人民解放军通信兵工程技术学校'''。
*1975年，更名'''中国人民解放军通信技术学校'''。
*1973年，通信兵工程技术学校（部分）分出，成立中国人民解放军通信工程学院，后参与组建[[中国人民解放军理工大学|中国人民解放军理工大学]]。
*1978年，中国人民解放军通信技术学校更名'''中国人民解放军无线电通信技术学校'''。
**1978年，中国人民解放军通信技术学校（有线通信专业）分出，成立中国人民解放军有线电通信技术学校，后发展为[[中国人民解放军西安通信学院|中国人民解放军西安通信学院]]。
*1986年，中国人民解放军无线电通信技术学校升格建立'''中国人民解放军重庆通信学院'''。
*1998年，获硕士学位授予权<ref name=yjs/>。
*2015年12月，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军总参谋部|中国人民解放军总参谋部]]转隶[[中国人民解放军陆军|中国人民解放军陆军]]<ref name=yjs>{{Cite web |url = http://www.mod.gov.cn/services/2016-09/20/content_4733411.htm |title = 中国人民解放军重庆通信学院招收硕士研究生 |author =  |date = 2016-09-20 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

;中国人民解放军陆军工程大学通信士官学校
*2017年，以原中国人民解放军理工大学、中国人民解放军军械工程学院主体为基础，组建[[中国人民解放军陆军工程大学|中国人民解放军陆军工程大学]]，重庆通信学院改建为'''中国人民解放军陆军工程大学通信士官学校'''<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1700729 |title =军媒披露：陆军工程大学由解放军理工大学和军械工程学院组建  |author =  |date = 2017-06-04 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。2017年8月3日，中国人民解放军陆军工程大学通信士官学校揭牌仪式举行<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1751534 |title = 陆军工程大学通信士官学校揭牌：陆军加快转型建设的重要举措 |author =  |date =2017-08-03  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
