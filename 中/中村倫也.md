'''中村倫也'''（{{jpn|j=中村　倫也|hg=なかむら ともや|rm=Nakamura Tomoya}}；{{bd|1986年|12月24日|||catIdx=Nakamura Tomoya}}），[[日本|日本]]男[[演員|演員]]，原藝名為「中村友也」，[[東京都|東京都]]出身，所屬經紀公司為{{link-ja|TOP COAT|トップコート}}，身高170cm，血型A型。

== 經歷 ==
中村高中一年級時被經紀公司發掘<ref name="CREA WEB 2012.10.05">{{Cite web |author=くれい響 |date=2012-10-05 |url=http://crea.bunshun.jp/articles/-/2146 |title=ミュージカル「RENT」でメインキャストに選ばれた中村倫也 |work=CREA WEB |publisher=[[文藝春秋|文藝春秋]] |accessdate=2013-03-09}}</ref>，而進入所屬經紀公司的培訓所「Artist☆Artist」學習演戲。2005年，因電影《七人の弔》正式以演員身分出道<ref name="CREA WEB 2012.10.05" />。

2014年，初次主演舞台劇《{{link-ja|The History Boys|ヒストリーボーイズ}}》，並以此劇獲得第22回讀賣演劇大賞優秀男優賞。中村曾擔任許多戲劇的反派角色，如在電視劇《[[黑金丑島君_(電視劇)|黑金丑島君 第三季]]》的「神堂」一角亦引起注目，為正反派色彩皆詮釋得游刃有餘的演員。

2018年，yahoo檢索大賞俳優部受賞。

== 人物 ==
* 中村為家中的次男，有一個哥哥。
* 喜好[[古典樂|古典樂]]及[[爵士樂|爵士樂]]
* 興趣是散步、拍照和踢[[足球|足球]]等。
* 很喜歡動物。家裡養著[[倉鼠|倉鼠]]、90cm的水槽養著[[多鰭魚|多鰭魚]]等動物。
* 現在最想飼養的動物是[[樹懶|樹懶]]。
* 因為變換自在的演技被稱為「變色龍俳優」。
* 喜歡日本酒。
* 喜歡的食物：[[蕎麥麵|蕎麥麵]]、[[鱈魚|鱈魚]]、味噌剁碎魚[[刺身|刺身]]（日語：なめろう）、[[明太子|明太子]]、[[香雅飯|香雅飯]]、[[炸雞|炸雞]]、雞麵。
* 討厭的食物：[[香菇|香菇]]、[[豌豆|豌豆]]、[[青花菜|青花菜]]、[[香蕉|香蕉]]、[[貝類|貝類]]、[[蕈類|蕈類]]、[[青椒|青椒]]。
* 開始吃辣之後，對於辣是沒有極限的。
