{{中国共产党项目}}
'''中国共产主义青年团'''，簡稱'''中国共青團'''，在[[中国大陆|中国大陆]]常简称为'''团'''，是[[中国共产党|中国共产党]]的[[青年组织|青年组织]]，是[[中华人民共和国|中华人民共和国]]的[[人民团体|人民团体]]之一。<ref name = "Lawrence R. Sullivan 582">{{cite book | author = Lawrence R. Sullivan | title = Historical Dictionary of the People's Republic of China | location = 美國拉納姆 | publisher = {{Link-en|羅曼和利特爾菲爾德出版集團公司|Rowman & Littlefield}} | date = 2007年5月23日 | page = 第582頁 | ISBN = 978-0810864436 | accessdate = 2015年6月1日 | url = https://books.google.com.tw/books?id=UQKyAAAAQBAJ&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false | language = en | quote =  }}</ref>

中国共青团的前身[[上海社会主义青年团|上海社会主义青年团]]在[[陳獨秀|陳獨秀]]和[[李大釗|李大釗]]響應[[國際共產主義運動|國際共產主義運動]]下於1920年8月成立，其全国性组织“'''中国社会主义青年团'''”则于1922年成立，初期接受[[第三國際|第三國際]]的指揮和援助。1940年代起，中國共產黨及其领导的中国共青团逐渐摆脱外國勢力的影響，改由以本土派掌权。在[[抗日战争|抗日战争]]期间，与国民黨进行了[[第二次国共合作|第二次国共合作]]，共青团被解散并改组为[[青年救国会|青年救国会]]。中共在[[第二次國共內戰|第二次國共內戰]]中取得胜利后，中共中央又宣布复设“'''中国新民主主义青年团'''”并在1957年改为现名。<ref name="简介">{{cite web|title=中国共青团简介|url=http://www.gqt.org.cn/695/gqt_tuanshi/gqt_ghlc/gqt_introduce/introduce/|website=中国共产主义青年团中央委员会|accessdate=2017-06-05}}</ref>

中國共青团的組織架構主要基於[[民主集中制|民主集中制]]，而主要架构則源於[[苏联|苏联]]的[[全联盟列宁主义青年共产主义联盟|全联盟列宁主义青年共产主义联盟]]。中國共青团最高機構是每5年召開一次的[[中國共產主义青年团全國代表大會|全國代表大會]]，而當全國代表大會閉會期間則由[[中國共產主义青年团中央委員會|中央委員會]]作為核心權力機構。但因為中央委員會通常1年只會召開1次會議，使得多數職權和工作都由[[中國共產主义青年团中央委员会常务委员会|中央委员会常务委员会]]和[[中国共产主义青年团中央书记处|中央书记处]]掌握，[[中国共产主义青年团中央书记处第一书记|共青团中央书记处第一书记]]则成为团中央主要負責人。

中國共產主義青年團申請人的加入年龄必須介於14歲至28歲間<ref>{{Cite web|url=http://qnzz.youth.cn/gzdt/201807/t20180702_11657414.htm|title=中国共产主义青年团章程_青年组织__中国青年网|accessdate=2018-07-25|last=青网共青团刘喆|work=qnzz.youth.cn}}</ref>，共青团同时还主導和監督以14歲以下兒童組成的先锋組織[[中國少年先鋒隊|中國少年先鋒隊]]<ref name = "Lawrence R. Sullivan 582" />。截至2017年年底，全国共有共青团员8124.6万名，其中学生团员5795.1万名；共有基层团组织357.9万个，其中，基层团委20.4万个，基层团工委1.6万个，团总支16.5万个，团支部319.4万个，<ref>{{Cite web|url=http://qnzz.youth.cn/gzdt/201805/t20180531_11632923.htm|title=最新数据统计：全国共青团员8124.6万|accessdate=2018年7月25日|author=中国青年网|date=2018年5月31日|publisher=中国青年网|language=cn|quote=}}</ref>成为世界上规模最大的青年组织之一。

中國共青团始終宣稱其理想與目標是實現[[共產主義|共產主義]]，將自身視為代表最广大中国青年的青年团体。其中《[[中國共產主义青年团章程|中國共產主义青年团章程]]》指出中國共青团堅持[[馬克思列寧主義|馬克思列寧主義]]、[[毛澤東思想|毛澤東思想]]、[[中國特色社會主義|中國特色社會主義]]、[[鄧小平理論|鄧小平理論]]、[[三個代表|三個代表]]和[[科學發展觀|科學發展觀]]、[[习近平新时代中国特色社会主义思想|習近平新時代中國特色社會主義思想]]等思想，並將中國共青团表述為：「是中国共产党领导的先进青年的群团组织，是广大青年在实践中学习中国特色社会主义和共产主义的学校，是中国共产党的助手和后备军」。<ref name="团章">{{cite web|author1=中国共产主义青年团全国代表大会|title=中国共青团章程|url=http://www.gqt.org.cn/ccylmaterial/regulation/|website=中国共产主义青年团中央委员会|accessdate=2017-06-05}}</ref>此外，由共青团中央出身的官员一向在[[中华人民共和国政治|中国政治]]中占据不可或缺的角色，被广泛称为“[[团派|团派]]”，具有较大影响力。

== 名称标识 ==
=== 名称 ===
1920年5月，中国共产党在上海率先组织了中国共产主义青年团的前身'''中国社会主义青年团'''。1921年7月，[[中国共产党|中国共产党]]正式成立，中国社会主义青年团在党的组织下于各地筹建。1922年5月，中国社会主义青年团第一届全国代表大会在共产党的领导下召开，成立了全国统一的组织。1925年1月，第三届全国代表大会召开，中国社会主义青年团更名为'''中国共产主义青年团'''。[[抗日战争|抗日战争]]结束后，为适应新形势新任务的需要，1949年4月正式更名为'''中国新民主主义青年团'''。1957年5月，现名恢复，解放前后的中国社会主义青年团、共产主义青年团和新民主主义青年团三大团体正式结合。

=== 团旗 ===
[[File:Tuanyida_Square.jpg|thumb]]的[[团一大广场|团一大广场]]，中央有共青团团旗图案。]]

中国共产主义青年团团旗，是[[毛泽东|毛泽东]]、[[周恩来|周恩来]]等亲自审定，并经中共中央批准，于1950年5月4日由中国共产主义青年团中央委员会公布的。旗面为红色，象征革命胜利；左上角缀黄色[[五角星|五角星]]，周围环绕黄色圆圈，象征着中国青年一代紧密团结在中国共产党周围。

宽高比3:2，旗面对分为4等分长方形，左上长方形上下12等分，以此长方形中心点为圆心，3等分及4等分为半径画两圆周，两圆周之间就是黄色圆圈。再在内圆周上定出5个等距离的点，其中一点位于圆周正上方。将此5点中各相隔的两点相联成直线，此5直线之外轮廓线就是黄色五角星之外沿。<ref name="团章"/>

=== 团徽 ===
中国共产主义青年团团徽，是经中共中央审定批准，于1959年5月4日由共青团中央颁布的。团徽的内容为团旗、齿轮、麦穗，初升的太阳及其光芒，写有“中国共青团”五字的绶带。团徽涂色为金红二色。团旗的旗面和绶带为红色，团旗上的五角星和环绕它的圆圈、旗边、旗杆、齿轮、麦穗、初升的太阳及其光芒、“中国共青团”五个字都为金色。

中国共产主义青年团团徽，是中国共产主义青年团的标志：象征着共青团在[[马克思列宁主义|马克思列宁主义]]、[[毛泽东思想|毛泽东思想]]的光辉照耀下，团结各族青年，朝着中共所指引的方向奋勇前进。<ref name="团章"/>

=== 团歌 ===
中國共產主義青年團團歌是《[[光榮啊，中國共青團|光榮啊，中國共青團]]》，[[胡宏伟|胡宏伟]]作词，[[雷雨声|雷雨声]]作曲，1988年中国共产主义青年团第十二次全國代表大會確認為代團歌。2003年7月，中国共产主义青年团第十五次全国代表大会通过了关于《中国共产主义青年团章程》（修正案）的决议，将《光荣啊，中国共青团》确定为团歌。<ref name="团章"/>

=== 团员证 ===
中国共产主义青年团团员证封皮原为仿皮革材质，封面为墨绿色，封面上方有红色烫金团徽，下方书写“团员证”（但并未标注“中国共产主义青年团”字样）。<ref name="团章"/>

=== 团章 ===
中国共产主义青年团团章，是团的理想目标和最高意志的体现，也规范了团的组织纪律和各项工作开展的纲领。最新版的团章由中国共产主义青年团第十八次全国代表大会部分修改，2018年6月29日通过。<ref name="团章" />

== 历史发展 ==
[[File:1953_huyaobang.jpg|thumb]]、[[廖承志|廖承志]]（右）出席全国第二次青年团代表大会时在主席台上。左为[[胡耀邦|胡耀邦]]。]]

1922年5月，中国社会主义青年团成立，1925年改称中国共产主义青年团；1936年11月，为吸收广大青年参加[[抗日救亡运动|抗日救亡运动]]，[[中共中央|中共中央]]决定将共青团组织改造为“广大群众的非党的青年组织”。1937年1月，共青团中央宣布结束工作。曾先后出现像[[中华民族解放先锋队|中华民族解放先锋队]]、青年救国会、青年抗日先锋队等抗日救国团体；[[抗战|抗战]]胜利后，于1946年试建新民主主义青年团的工作，1949年4月成立中国新民主主义青年团，1957年改称中国共产主义青年团；1982年12月举行第十一次全国代表大会，修改通过了《中国共产主义青年团章程》。<ref name="简介"/>

== 组织机构 ==
{{Main|中國共青团組織}}
[[File:Central_committee_of_the_Communist_Youth_League_of_China_(20170314111509).jpg|thumb]]的办公大楼。]]
中國共青团創建之初便以[[蘇聯共青团|蘇聯共青团]]為榜樣，採取了蘇聯共青团的組織模式，也就是設有共產黨的組織的机关都设立了共青团组织。共青团的組織原則採取[[民主集中制|民主集中制]]，並且基於[[民主|民主]]和{{Link-en|中央集權政府|Centralized government|集中}}兩個原則而成其中。<ref name = "王传志">{{Cite web |url = http://www.qstheory.cn/zxdk/2013/201310/201305/t20130513_230301.htm |title = 民主集中制： 我国政治制度的核心机制 |author = 王传志 |date = 2013年5月16日 |publisher = 《[[求是|求是]]》 |language = zh-Hans |accessdate = 2015年6月1日 |quote =  }}</ref>在《中國共青团章程》中提到：「民主集中制是共青团根本的组织原则。要充分发扬民主，尊重团员主体地位，切实保障团员的民主权利。要实行正确的集中，加强组织性和纪律性，保证团的决议得到有效的贯彻执行。」<ref name="团章"/>而毛澤東曾經表示民主集中制是「將民主和集中兩個似乎相衝突的東西，在一定形式上統一起來」，認為藉此結合能夠處理民主和集中之間的內部矛盾，同時民主集中制在[[自由|自由]]和{{Link-en|紀律|Discipline}}協調上更具優越性<ref name = "王传志"/>。

=== 非常設機關 ===
与中国共产党的组织形式类似，根據《[[中国共青团章程|中国共青团章程]]》確立之體制，其中[[中國共产主义青年团全國代表大會|中國共产主义青年团全國代表大會]]是中國共青团的最高機構。<ref name="团章"/>在1988年[[中國共产主义青年团第十二次全國代表大會|中國共产主义青年团第十二次全國代表大會]]之前全國代表大會經常不定期召開<ref name="代表大会">{{cite web|title=团的历次代表大会|url=http://www.gqt.org.cn/695/gqt_tuanshi/gqt_ghlc/lcdbdh/|website=中国共产主义青年团中央委员会|accessdate=2017-06-06}}</ref>，之後確認每五年舉行一次全國代表大會的制度，而每次大會將會持續數天。而根據《中国共青团章程》內容，除非在特別情況下全國代表大會不得延期<ref name="团章"/>。在《中國共青团章程》中賦予了全國代表大會4項責任：

# 审查和批准中央委员会的工作报告；
# 讨论和决定全团的工作方针、任务和有关重大事项；
# 修改团的章程；
# 选举中央委员会。

而在全國代表大會閉會後，則是由全國代表大會所產生的中國共产主义青年团中央委員會擔任最高決策機構；在這期間由中央委員會執行全國代表大會的決議並領導中國共青团全部工作，對外則是代表共青团<ref name="团章"/>。

新當選的中國共青团中央委員會委員除了之後會在不同政府部門、地方团组织、学校工作外，在第一次全體會議上會分別從[[中國共青团中央委员会|中國共青团中央委员会]]委員中選出作為主要負責人的[[中國共青团中央书记处第一书记|中國共青团中央书记处第一书记]]，並且由其擔任中國共青团的領導人。另外還會選出[[中国共青团中央委员会常务委员会|中国共青团中央委员会常务委员会]]、[[中国共青团中央书记处|中央书记处]]。中國共青团中央委員會全體会议一般每年舉行一次，為期2天至3天。在中央委員會全體會議閉會期間，則是由中央委员会常务委员会行使其职权。全国代表大会、中央委员会、中央委员会常务委员会都闭会时，由中央书记处行使中央委員會的職權<ref name="团章"/>。

在當前组织體系中，中國共青团中央书记处作為中國共青团決策機關以及最高領導機關，其中重大的政治、思想、政策和組織問題都必須在中央书记处會議上討論通過。同時基於在組織上採取民主集中制作為基本原則，中國共青团要求下級機關必須服從上級機關，而所有中國共青团組織都必須服從中央領導集團指示，并最终服从于[[中国共产党集体领导制度|中国共产党领导集体]]的指示；這使得中國共青团一切工作都是由中央集中領導，各級委員會的委員必須無條件的執行中共中央和共青团中央的任務<ref name="团章"/><ref>{{cite book | author = Lawrence R. Sullivan | title = Historical Dictionary of the People's Republic of China | location = 美國拉納姆 | publisher = {{Link-en|羅曼和利特爾菲爾德出版集團公司|Rowman & Littlefield}} | date = 2007年5月23日 | pages = 第583頁 | ISBN = 978-0810864436 | accessdate = 2015年6月1日 | url = https://books.google.com.tw/books?id=UQKyAAAAQBAJ&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false | language = en | quote =  }}</ref>。

=== 常設機關 ===
在中國共青团全國代表大會、中央委員會全體會議和中央委员会常务委员会全體會議閉會期間，代行中央委员会職權的中國共青团中央书记处即為實質上的最高決策機構。与中国共产党类似，共青团藉由閉會制度使得中國共青团的權力集中至中央书记处上，儘管理論上全國代表大會有權駁回中央委員會或中央书记处的決定，不過實際上在非動亂時期從未發生過這一情況。根據《中国共青团章程》，中國共青团中央委員會常务委员会負責召集中央委员会的會議。<ref name="团章"/>

中國共青团中央委員會第一次全體會議上，還會選舉幾個工作部門、局處和其他機構成員，並且全國代表大會閉會的5年期間從事各項工作<ref name="团章"/>。[[共青团中央辦公廳|共青团中央辦公廳]]是中國共青团的的核心聯繫部門，負責包括日常通訊、協議、安排會議議程等行政工作<ref>{{Cite book | author = Lawrence R. Sullivan | title = Historical Dictionary of the Chinese Communist Party | location = 美國拉納姆 | publisher = {{Link-en|羅曼和利特爾菲爾德出版集團公司|Rowman & Littlefield}} | date = 2011年11月4日 | pages = 第212頁 | ISBN = 978-0810872257 | accessdate = 2015年6月1日 | url = https://books.google.com.tw/books?id=g0PjFe5i0iEC&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false | language = en | quote =  }}</ref>。中國共青团目前轄下還擁有多個主要核心部門，這包括有負責監督職位安排和審查幹部工作發展的[[共青团中央組織部|共青团中央組織部]]、負責監督团中央所属新聞媒體並且向团员和青年提供爱国主义宣传的[[共青团中央宣傳部|共青团中央宣傳部]]、負責中國共青团與其他海外組織和國際機構聯繫的[[共青团中央国际联络部|共青团中央国际联络部]]、负责辅导青少年工作的[[共青团中央城市青年工作部|共青团中央城市青年工作部]]、[[共青团中央农村青年工作部|共青团中央农村青年工作部]]、[[共青团中央学校部|共青团中央学校部]]、[[共青团中央少年部|共青团中央少年部]]以及負責監督全国学联等群眾组织的[[共青团中央統一戰線工作部|共青团中央統一戰線工作部]]。

在教育層面，中國共青团還設立有提供高級幹部和準幹部政治訓練和共產主義思想教導的[[中央團校|中央團校]]，而在新聞媒體對外傳播方面，中國共青团中央委員會直接掌握中國共青团發行的机关報《[[中国青年報|中国青年報]]》。

=== 地方組織 ===
中國共青团在各個[[中國一級行政區|省]]、[[中華人民共和國行政區劃|自治區]]、[[直轄市|直轄市]]；[[設區的市|設區的市]]、[[自治州|自治州]]；[[縣級行政區|縣]]（包括[[旗_(行政區劃)|旗]]）、[[自治縣|自治縣]]、[[不設區的市|不設區的市]]和[[市轄區|市轄區]]成立地方代表大會<ref name="团章"/>，並且由這些代表大會自行選出該級單位的委員會<ref name="团章"/>。此外，还在[[中国共产主义青年团新疆生产建设兵团委员会|新疆兵团]]、[[中国共产主义青年团中央直属机关工作委员会|中央直属机关]]、[[中国共产主义青年团中央国家机关工作委员会|中央国家机关]]、[[中国共产主义青年团全国铁道委员会|全国铁道]]、[[中国共产主义青年团全国民航委员会|全国民航]]、[[中国共产主义青年团中央金融工作委员会|中央金融企业]]和[[中国共产主义青年团中央企业工作委员会|中央企业]]中设立了相应的地方组织。中國共青团地方各級代表大會應該每3年舉行一次，但是在特殊情況下可能會決定提前或者延後進行，不過這項決議必須由當地更上一層的地方代表大會委員會批准<ref name="团章"/>。委員會的人數和選舉程序由各地地方代表大會決定，同樣必須經由上級地方代表大會批准才得以施行<ref name="团章"/>。地方各級代表大會許多職責與中國共產黨全國代表大會類似，主要負責审查和批准同级委员会的工作报告、讨论和决定本地区团的工作任务和有关重要事项、选举同级委员会<ref name="团章"/>。

共青团地方代表大會代表當選後有3年任期<ref name="团章"/>。而如果一個地方代表大會提前或者延後召開會議，代表大會委員的任期相應地縮短或加長<ref name="团章"/>。不過除了各級地方代表大會委員會負責人必須向上一級的地方代表大會負責外<ref name="团章"/>。各級地方委員會全體會議每年至少要召開1次全體會議，在其主導地方团的工作期間應該執行上級中國共產黨組織的指示、以及和同級代表大會的決議<ref name="团章"/>。共青团的省级委员会的行政级别一般为[[正厅级|正厅级]]，地级和县级委员会则为[[正处级|正处级]]和[[正科级|正科级]]。

不過在地方委員會部分也採取類似中國共青团中央委员会的架構，實際上在地方代表大會和地方委員會閉會時便是由地方常務委員會主持政務和負責相關事項<ref name="团章"/>。其中地方常務委員會的人選是在地方委員會第一次全體會議選舉審議產生，之後由其負責人提報給上一級的地方委員會批准<ref name="团章"/>。

=== 基层组织 ===
共青团還在地方國家機關、人民團體、經濟組織、文化組織、学校、[[中国人民解放军|中国人民解放军]]、[[中国人民武装警察部队|中国人民武装警察部队]]和其他非黨組織的領導機關中成立基层组织。

共青团在全国所有大学、中学中设置共青团组织。在学校中，共青团组织往往领导[[学生会|学生会]]，并与学生会并称为“团委学生会”。共青团在基层委员会、总支部下建立支部。具体而言，其在学校内所有[[班级|班级]]都设立了[[团支部委员会|团支部委员会]]，与班级委员会共同管理班级，[[班长|班长]]和[[团支部书记|团支部书记]]负责班级的全面工作，学校团支部一般设书记、组织委员、宣传委员等职位，负责具体事宜。

=== 共青团员 ===
{{Quote box| width = 300px | align = right | quoted = true | bgcolor = #FFFFF0 | salign = right | quote = 我志愿加入中国共产主义青年团，坚决拥护中国共产党的领导，遵守团的章程，执行团的决议，履行团员义务，严守团的纪律，勤奋学习，积极工作，吃苦在前，享受在后，为共产主义事业而奋斗。 | source= 中國共產主义青年团入团誓詞<ref name="团章"/>}}

申請加入中國共青团者年齡必須滿十四周歲<ref name="团章"/>，并有两名入团介绍人。申請人必須在中國共青团团旗前舉行入团宣誓才能夠成為共青团团員。根據《中國共青团章程》，全體团員必須服從命令、遵守紀律、堅持團結、服務於中國共青团和人民，接受中国共产党的领导，並且促進社會主義道路的發展<ref name="团章"/>。

在加入中國共青团將會獲得許多權限，這包括有参加团的有关会议和团组织开展的各类活动，接受团组织的教育和培训；在团内有选举权、被选举权和表决权；在团的会议和团的媒体上，参加关于团的工作和青年关心的问题的讨论，对团的工作提出建议，监督、批评团的领导机关和团的工作人员；对团的决议如有不同意见，在坚决执行的前提下，可以保留，并且可以向团的上级组织提出；参加团组织讨论对自己处分的会议，并且可以申辩，其他团员可以为其作证和辩护；向团的任何一级组织直至中央委员会提出请求、申诉和控告，并要求有关组织给以负责的答复<ref name="团章"/>。同時包括中國共青团中央委員會等組織，都沒有權限得以剝奪中國共青团团員的這些權力<ref name="团章"/>。

由共青团中央基层组织建设部公布的全国团内统计最新数据显示，截至2017年年底，全国共有共青团员8124.6万名，其中学生团员5795.1万名；共有基层团组织357.9万个，其中，基层团委20.4万个，基层团工委1.6万个，团总支16.5万个，团支部319.4万个。<ref>{{Cite web|url=http://qnzz.youth.cn/gzdt/201805/t20180531_11632923.htm|title=最新数据统计：全国共青团员8124.6万_青年组织__中国青年网|accessdate=2018-07-25|last=青网共青团李彦龙|work=qnzz.youth.cn}}</ref>

其退出方式与少先队相似，均为超龄自动退出。多数团员是超过28岁后自动退团，亦有人士因为毕业、工作或者居住地变更的原因，没有进行组织关系转接导致团籍遗失，或因没有缴纳团费、不参加组织活动，按照团章“团员没有正当理由，连续六个月不交纳团费、不过团的组织生活，或连续六个月不做团组织分配的工作，均被认为是自行脱团。”的规定，自动脱离共青团。“团员自行脱团，应由支部大会决定除名，并报上级委员会批准。”<ref name="团章"/>

但是，担任团的干部，即使超过28岁，也依然具有共青团员身份。根据团章和[[中共中央组织部|中共中央组织部]]的规定，团员加入共产党后，依然保留团籍，28岁后未担任团的职务者，则不再保留团籍。<ref name="团章"/>

== 政策和影响 ==
=== 青年政策 ===

中国共青团是中国政府开展青年工作的主要机构，共青团着力在互联网新媒体等新兴传媒层面开展工作。共青团中央在[[微博|微博]]、[[知乎|知乎]]、[[bilibili|bilibili]]等青年人接触较多的网站开设了官方帐号，以争夺舆论主导权。共青团中央经常就社会舆情事件发表评论，其政治立场以[[中国民族主义|中国民族主义]]、[[爱国主义|爱国主义]]和[[中国反西方情绪|反西方情绪]]为主，倾向于支持中国政府的立场，但也反对其所称的“非理性爱国”行为。在[[电影《没有别的爱》演员撤换事件|电影《没有别的爱》演员撤换事件]]、[[2016年中国大陆网民“出征”Facebook事件|2016年中国大陆网民“出征”Facebook事件]]等重要舆情事件中多次引导舆论和事件发展。<ref name = "青年政策甲">{{Cite web |url = https://www.zhihu.com/question/48235350/answer/134062967 |title = 微博上的共青团中央是怎样的一种存在？ |author = 知乎机构帐号“共青团中央” |date = 2016年12月3日 |publisher = 知乎 |language = zh-cn |accessdate = 2017年7月22日 |quote =  }}</ref>

=== 政治影响 ===
共青团系统出身的干部在[[中华人民共和国政治|中华人民共和国政治]]中具有较大影响，被广泛称为“[[团派|团派]]”。2012年[[中共十八大|中共十八大]]以后，在[[中共中央政治局|中共中央政治局]]委员之中曾有共青团中央书记处任职经历者就有有[[李克强|李克强]]、[[李源潮|李源潮]]、[[刘延东|刘延东]]、[[刘奇葆|刘奇葆]]和[[胡春华|胡春华]]等五人。

共青团系统往往被视作青年官员晋升的一条捷径，各地团委书记任期届满后多能直接出任下级行政区的党政主官等要职。「团派」一词最早在[[胡耀邦|胡耀邦]]年代即有提及。前中共中央总书记胡耀邦担任中共高层领导职务时，大量曾在其担任第九届中国共青团中央书记处第一书记期间有过共青团职务背景的官员获得提升<ref>{{cite book|author=國立政治大學國際關係研究中心|title=《中國大陸研究》（44卷）|year=2001年|pages=21页}}</ref>，其中包括时任[[中央办公厅|中央办公厅]]主任的[[胡启立|胡启立]]、共青团中央第一书记的[[胡锦涛|胡锦涛]]等。

团派人物特点为大学本科以上学历，谨慎且不张扬。在被认为是团派的成员中，除少数具有高干子女背景或其他显赫背景外，大多出身平民阶层，因而比较贴近普通民众<ref>[http://gb.chinareviewnews.com/doc/1014/0/7/5/101407527.html?coluid=7&kindid=0&docid=101407527 汪洋与外来工分享工人经历：睡地铺、盼涨薪] 中评电讯 2010年8月8日</ref>。团派通过组织内部选拔推荐的形式，以取代及对抗在[[中国共产党|中国共产党]]党内长期存在的世系（如[[太子党|太子党]]）与地域保护体系（如[[上海帮|上海帮]]等）。

就团派的定义尚未在官方场合进行公示或评论，就此是否形成派系仍有反对意见。譬如[[鍾沛璋|鍾沛璋]]、[[何家棟|何家棟]]等，称这些人物恰恰代表中国共产党党内干部年轻化的自然趋势，而非派系使然。政治作家[[何清涟|何清涟]]认为“任职于共青团中央的官员之间既无共同的利益纽带，也无一个愿意维系帮派利益的领袖，更无互为奥援的愿望，将其称之为政治帮派，实在有点勉强。”<ref>{{cite web |url = http://www.voachinese.com/a/communist-youth-league-not-faction-20160807/3454725.html |language= |title = 何清涟：官员多有共青团干出身，但无“团派”——分析中共的“共青团政治”(2) |publisher=美国之音 |date=2016-08-08 |accessdate= }}</ref>

内部消息称，[[中共中央总书记|中共中央总书记]][[习近平|习近平]]曾于2017年严厉批评共青团“空喊口号”，“形同虚设”，“四肢麻痺”，“说科技说不上、说文艺说不通、说工作说不来、说生活说不对路。说来说去就是那几句官话、老话、套话，同广大青年没有共同语言、没有共同爱好，那当然就会话不投机半句多<ref>{{cite web |url = http://news.dwnews.com/china/news/2018-06-05/60062526.html |language= |title = 共青团十八大将召开 习近平曾严厉批评|publisher=多维新闻网 |date=2018-06-05 |accessdate= 2018-06-24}}</ref>。”
