'''中華民國與關島關係'''是指[[中華民國|中華民國]]與[[美利堅合眾國|美利堅合眾國]]的[[屬地|屬地]]－[[關島|關島]]之間的關係。

==政治==
===外交===
1973年1月12日，關島與[[台北市|台北市]]締結為[[臺北市友好城市或姐妹城市列表|姐妹市]]至今。<ref name="卡瑪丘"/>

[[关岛议会|關島議會]]對中華民國亦友好，多次通過決議案支持台灣參與[[世界衛生組織|世界衛生組織]]（WHO）、[[國際民航組織|國際民航組織]]（ICAO）等國際組織。<ref name="卡瑪丘"/><ref name="過境"/>

中華民國總統[[馬英九|馬英九]]、[[蔡英文|蔡英文]]出訪[[中華民國外交#邦交國|南太平洋邦交國]]時，以過境名義訪問關島，並由總督親自接待。<ref>{{Cite web |url = http://hk.crntt.com/doc/1012/7/0/9/101270934.html?coluid=9&kindid=5790&docid=101270934&mdate=0326143039 |author = 康子仁 |date = 2010年3月26日 |title = 馬英九過境關島 薄瑞光接機 |publisher = [[中國評論通訊社|中國評論通訊社]]}}</ref><ref name="過境">{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/2242965 |author = 鍾麗華 |date = 2017年11月4日 |title = 蔡總統過境 關島總督高規格接待 |publisher = [[自由時報|自由時報]]}}</ref>

目前沒有在對方[[首府|首府]]互設[[外交代表機構|代表機構]]，對關島的相關事務由'''中華民國駐[[帛琉共和國|帛琉共和國]][[外交代表機構#大使馆|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref name="駐館位置及聯絡方式">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-112-c8ef7-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref>

===簽證===
持有載明[[中華民國國民身份證|身份證]]字號的新版普通晶片[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以美國「[[免簽證計劃|免簽證計劃]]」的方式入境關島，停留最多90天。事先上網申請「[[旅遊許可電子系統|旅行授權電子系統]]（ESTA）」取得授權許可，且無其他特殊限制而無法適用者，即可免除預先申請[[B签证_(美国)|B1/B2簽證]]而直接前往關島；另可申請關島及[[北馬里亞納群島|北馬里亞納群島]]實施的免簽證計畫，停留最多45天，但規定較多，例如須從台灣直飛關島、填寫相關入出境表、不得接續前往美國其他地區等。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-112-c8ef7-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=0984A85A3A9A6677 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>前往關島可利用美國的「[[全球入境|全球入境計畫]]（Global Entry）」，只需事先申請、完成安全查核再通過面試後，入出境關島的[[安東尼奧·汪帕特國際機場|安東尼奧·汪帕特國際機場]]就可以使用專用通道，省去排隊查驗通關時間。

持有[[美國護照|美國護照]]的{{Link-en|關島人口|Demographics of Guam|關島公民}}也可以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

中華民國外交部的[[國外旅遊警示分級表|國外旅遊警示分級表]]將關島歸類為'''灰色警示'''：提醒注意（2017年6月2日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-195-1b256-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
===貿易===
2018年，關島是中華民國第96大貿易夥伴、第121大[[進口|進口]]夥伴、第80大[[出口贸易|出口]]夥伴，在[[大洋洲国家和地区列表|大洋洲經濟體]]排名第5、6、3。出口至關島的金額為6,914萬4,205[[美元|美元]]，年增89.688%；自關島進口的金額為582萬9,517美元，年增31.392%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）6,331萬4,688美元，年增97.767%。2017年，關島是中華民國第110大貿易夥伴、第129大進口夥伴、第91大出口夥伴，在大洋洲經濟體排名第7、7、6。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
[[關島總督|關島總督]]抵台訪問時，除官式拜會外，也在台灣招商赴關島投資。<ref name="卡瑪丘"/><ref name="卡佛"/><ref name="卡佛閣下"/>

===觀光===
關島是台灣遊客喜愛的旅遊地點之一，台灣也數年排名關島第3大觀光客來源地，僅次於[[日本|日本]]、[[韓國|韓國]]。<ref name="卡瑪丘"/><ref name="卡佛閣下"/><ref name="過境"/>
