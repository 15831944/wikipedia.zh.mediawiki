'''中国人民武装警察部队新疆生产建设兵团总队'''，机关驻[[新疆维吾尔自治区|新疆维吾尔自治区]][[乌鲁木齐市|乌鲁木齐市]]。

==沿革==
中国人民武装警察部队新疆生产建设兵团指挥部的前身是新疆生产建设兵团的内卫部队。1984年，根据[[中华人民共和国国务院|中华人民共和国国务院]]要求，新疆生产建设兵团的内卫部队改编为“'''中国人民武装警察部队新疆维吾尔自治区总队生产建设兵团支队'''”，继续隶属新疆生产建设兵团。1988年，经国务院批准，[[中华人民共和国农业部|中华人民共和国农业部]]和[[中华人民共和国公安部|中华人民共和国公安部]]将中国人民武装警察部队新疆维吾尔自治区总队生产建设兵团支队调整为“'''中国人民武装警察部队新疆维吾尔自治区总队生产建设兵团指挥所'''”，继续隶属新疆生产建设兵团<ref name=shengge/><ref name=weihu>{{Cite web|url =http://www.xj.xinhuanet.com/bt/2006-02/20/content_6276701.htm|title =新疆生产建设兵团的奋进之维护稳定(二九二)|author =|date =2006-02-20|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20070118011023/http://www.xj.xinhuanet.com/bt/2006-02/20/content_6276701.htm|archivedate =2007-01-18}}</ref><ref name=zujian/>。

经国务院、[[中央军事委员会|中央军委]]同意，1996年4月17日，根据国函〔1996〕9号文件和〔1996〕武司字第58号文件精神，自1996年5月1日起，中国人民武装警察部队新疆维吾尔自治区总队生产建设兵团指挥所改称为“'''中国人民武装警察部队新疆生产建设兵团指挥部'''”，由副师级调整为正师级，正式纳入武警内卫部队序列，接受新疆生产建设兵团和中国人民武装警察部队新疆维吾尔自治区总队的双重领导，中国人民武装警察部队总部负责该部队的军事、政治、后勤工作的全面领导<ref> {{Cite book | author = 新疆生产建设兵团史志编纂委员会 编、何绍慈 主编 | title =  新疆生产建设兵团年鉴1997年| location =  | publisher =新疆人民出版社  | date =1997年10月  | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote = 武警新疆兵团指挥部 }} </ref><ref name=weihu/><ref name=zujian/>。中国人民武装警察部队新疆生产建设兵团指挥部下设各支队、大队、中队<ref name=shengge/>。

2012年，[[中华人民共和国国务院|国务院]]、[[中央军事委员会|中央军委]]批复：中国人民武装警察部队新疆生产建设兵团指挥部升格为副军级；任命邸明为司令员，吴玉科为政治委员。2012年10月25日，中国人民武装警察部队新疆生产建设兵团指挥部宣布升格命令大会在乌鲁木齐举行，武警部队副政治委员[[吴云峰|吴云峰]]中将宣布了国务院、中央军委《关于武警新疆生产建设兵团指挥部编制等级调整的批复》，以及国务院、中央军委和武警部队关于邸明、吴玉科二人的任职命令<ref name=shengge>{{Cite web |url = http://china.caixin.com/2012-10-26/100452776.html |title = 武警新疆兵团指挥部升格副军级 |author =  |date = 2012-10-26 |publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref><ref name=zhbian>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1996363 |title =武警新疆生产建设兵团指挥部于去年10月整编成立为兵团总队  |author =  |date = 2018-02-12 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

中国人民武装警察部队新疆生产建设兵团指挥部及其前身，积极支援[[新疆维吾尔自治区|新疆维吾尔自治区]]、新疆生产建设兵团的经济建设、扶贫开发、[[社会治安综合治理|社会治安综合治理]]等工作，参与处置乌鲁木齐“5·19”反革命骚乱事件、“8·15”反恐战斗、“2·24”抗震救灾、“1·21”抗洪抢险、2004年“5·18”空难救援、2009年[[乌鲁木齐七五事件|乌鲁木齐七五事件]]、2014年沙漠营救民工等任务，涌现出第四届“中国杰出青年卫士”杨厚强等先进模范<ref name=shengge/><ref name=zujian>{{Cite web|url =http://www.xj.xinhuanet.com/2004-08/03/content_2613575.htm|title =武警新疆兵团指挥部组建20周年|author =|date =2004-08-03|publisher =新华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170313220850/http://www.xj.xinhuanet.com/2004-08/03/content_2613575.htm|archivedate =2017-03-13}}</ref>。

2017年10月，在武警部队改革调整中，中国人民武装警察部队新疆生产建设兵团指挥部整编成立为'''中国人民武装警察部队新疆生产建设兵团总队'''。整编后的兵团总队主要担负兵团辖区看押、看守、警卫、守卫、城市武装巡逻和自治区疆内反恐维稳任务<ref name=zhbian/><ref>{{Cite web |url =http://www.china.com.cn/newphoto/2017-10/10/content_41709010.htm  |title = 高清大图围观武警新疆兵团指挥部新训生活 |author =  |date =2017-10-10  |publisher =中国网  |language =  |accessdate =  |quote =  }}</ref>。
