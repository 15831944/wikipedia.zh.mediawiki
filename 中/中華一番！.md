《'''中華一番！'''》（{{lang-ja|中華一番！}}）是日本漫畫家[[小川悦司_(漫畫家)|小川悦司]]的作品，在連載後期改為《'''真・中華一番！'''》（{{lang|ja|真・中華一番！}}），自1995年起至1999年連載於《[[週刊少年Magazine|週刊少年Magazine]]》，[[單行本|單行本]]共17本，[[東立出版社|東立出版社]]以《中華小廚師》為名發行中文版。此外有改編成[[動畫|動畫]]版和真人電視劇版《[[中華小當家|中華小當家]]》。

作者小川悅司表示，扣除主角劉昴星外，他個人最喜歡的故事三要角依序是四郎、趙瑜與向恩。

續集《'''中華一番！極'''》（{{lang|ja|中華一番！極}}）於2017年11月開始在講談社的網路漫畫連載平台《Magazine Pocket》隔周連載<ref>[https://qooah.com/2017/11/02/chuka-ichiban/amp/ 日漫《中華一番》開新篇，名為《中華一番！極》]</ref>，半月刊於每隔周五更新。中文版將於2017年12月29日開始在《東立電子書城》之《新少年快報B》連載。

==概要==
在作品中除了如其他料理漫畫、以超現實的誇張手法和扭曲的神情彰顯食物的美味以外，其世界觀龐大，更創造出異想天開的烹飪方式，例如「一瞬間支解水牛的刀法」、「用燃燒的[[長江|長江]]烤魚」、「在砧板上切好的肉自動飛到鍋中」等不合常理的調理方式。廚師會使用如格鬥漫畫般的必殺技來烹飪，把菜刀等廚具拿來互相砍殺的武打場景也非常多，在料理漫畫中並不多見。

== 故事大綱 ==
[[清朝|清朝]]末年[[四川|四川]]國營餐館菊下樓將舉行新任大廚繼位之爭，主角劉昴星是已逝的前任大廚阿貝的兒子，他的對手是阿貝叛逃的徒弟紹安，在麻婆豆腐對決中，劉昴星擊敗紹安贏得大廚之位，但他按照比賽評審李提督的建言前往[[廣州|廣州]]南鮮酒家持續精進。劉昴星拜該餐館的副料理長丁油為師，認識師傅的獨生女梅麗、並結交師兄唐三傑。在劉昴星協助三傑克服過去的障礙後，三傑回到故鄉[[上海|上海]]。

一日，法國大廚巴傑魯造訪廣州並貶低中國料理，憤怒的劉昴星與他進行炒飯對決並擊敗了他，巴傑魯心有不甘、縱火燒燬南鮮酒家。為重建餐館、劉昴星參加廣州餃子大賽取得優勝及獎金，餐館重建更名為陽泉酒家。劉昴星依據餃子大賽評審朗文大師的建議參加特級廚師測驗，他在測驗中結交了與自己不相上下的好對手蘭飛鴻，與他一起通過測驗，成為中國史上最年輕的特級廚師。在測驗中發現自己實力尚淺的劉昴星展開修行，他造訪[[廣西|廣西]]、[[貴州|貴州]]及[[雲南|雲南]]的城鎮精進廚藝，旅途中收了四郎為徒，之後回到陽泉酒家。

又一日，「鋼棍」解師傅前來陽泉酒家踢館，與劉昴星不打不相識，進行燒賣對決後結為好友；不久後陽泉酒家接到「面具廚師」李嚴寄來邀請函，李嚴企圖殺丁油報仇，在敗給劉昴星之後向他透露企圖以料理統治中國的「黑暗料理界」之事；數日後、劉昴星與曾是陽泉酒家學徒的「七星刀」雷恩進行鯛魚對決，並在比賽中得知「傳說中的廚具」之事。這批藏於中國各地的廚具各個具有神奇力量，傳說只要能吃上以所有廚具做出的料理便能[[長生不死|長生不死]]。為了不讓該廚具落入黑暗料理界手中，劉昴星、梅麗、雷恩、解師傅和四郎組成「光明廚師團」展開搜集廚具的旅程。

眾人造訪[[上海|上海]]遇見三傑，他們接到黑暗料理界的挑戰，雙方各派出四名廚師、以爭奪記有「傳說中的廚具」地點的藏寶圖為目標展開筵席料理對決，劉昴星在第四回合對決中與墮入黑暗的師兄紹安再會，並在豆腐對決中再度取勝，紹安悔過後投海自盡。接著、廚師團接連取得幾項廚具，但劉昴星卻在長江一戰中險敗黑暗料理界的幹部「五虎星」之一的亞刊，並因此與解、雷兩人分離。

劉昴星、四郎和梅麗與蘭飛鴻重逢，回到四川菊下樓的劉昴星與趁隙滲透的五虎星朱七對決，擊敗他後與他成為好友，之後、他主動進攻黑暗料理界總部「梁山泊」擊敗亞刊。另一方面、在[[敦煌|敦煌]]的解師傅和雷恩也為了傳說廚具與五虎星之一的密拉對決。為爭奪最後兩項傳說廚具，眾人齊聚北京紫禁城展開對決，五虎星之首凱由企圖以毒膳謀害擔任評審的[[慈禧太后|慈禧太后]]，但遭劉昴星識破並擊敗，並因謀反而遭亂箭射殺，而另一名五虎星顏先也自知不敵蘭飛鴻而陣前投降，黑暗料理界勢力因而瓦解，但是傳說廚具因沾上凱由的血而盡數失效。在那之後劉昴星等人再度展開旅程，用料理帶給大家幸福。

《中華一番！極》的故事中，劉昴星回到故鄉四川參加他父親瑪琉的祭禮，然後菊下樓就被神秘勢力所炸毀，同為參加祭禮的龍廚師兼瑪琉的弟子路以認為這個事件是與阿昴的父親有關，認為瑪琉仍然活著，所以劉昴星決定展開尋找父親的旅程，並解開黑暗料理界的謎團。

==特殊用語==
; 傳說中的廚具
: 是作品中虛構的八樣特別廚具，由[[唐代|唐代]]一位名為許林的廚師以天上掉下來的隕石打造而成，只有足夠資格的廚師才能成為廚具的正統繼承人，並完全發揮廚具的靈力，傳說若有人能吃到超能廚師用八樣傳說中的廚具所做的料理，那個人便會「[[長生不老|長生不老]]」。
: 一旦廚具被用於料理以外的用途（例如殺人、沾到人血）便會失去靈力，之後得用山上湧出的清泉淨化，持續不斷地清潔廚具，才能讓廚具恢復原本的靈力。
*'''第一廚具「永靈刀」'''
:使用者：劉昴星、雷恩
:能使切過的海鮮類恢復鮮度，正統繼承人會使其浮現霸龍紋。藏於[[廣州|廣州]]。最終第二度被血封印。
*'''第二廚具「轉龍壺」'''
:使用者：劉昴星、上海龍鎮酒家
:放入食材後就能在瞬間讓湯汁高速成熟發酵，正統繼承人會使其發出強光。藏於[[上海|上海]]。最終被血封印。
*'''第三廚具「魔聖銅器」'''
:使用者：翁聲譽→劉昴星→凱由
:能使乾貨以數百倍的速度泡水恢復成原狀，正統繼承人會使其發出強光。藏於[[安徽|安徽]]。最終被血封印。
*'''第四廚具「[[迦樓羅|迦樓羅]]刀」'''
:使用者：陳建→蘭飛鴻
:能在切割之間淨化一切鳥獸之肉，正統繼承人會使其浮現瑞鳳紋。藏於[[四川|四川]]。最終第二度被蘭飛鴻丟入谷底。
*'''第五廚具「[[貪狼|貪狼]]壺」'''
:使用者：顏先及其助手
:只要把材料全放進去，無論要剁碎或切絲都能立刻辦到。藏於[[山東|山東]][[泰山|泰山]]。最終被血封印。
*'''第六廚具「[[冰箱|靈藏庫]]」'''
:使用者：雷恩、解師傅
:任何食材只要放進這個廚具，就可以永遠保持新鮮。藏於[[敦煌|敦煌]]。最終被血封印。
*'''第七廚具與第八廚具「玉龍鍋」'''
:一組雙鍋，相傳「唯有持有此物之人，才能持有全部八樣廚具」，效果不明。藏於[[北京|北京]]御膳房，為全國料理決戰的獎品。最終被血封印。

== 電視動畫 ==
《中華一番》動畫版是由[[日本動畫公司|日本動畫公司]]製作，[[富士電視台|富士電視台]][[聯播網|聯播網]]（[[FNS|FNS]]）首播，首播期間為1997年4月27日至1998年9月13日，首播時間為[[日本時間|日本時間]]每週日19：30～20：00，全52話。第一首片頭曲中使用的片頭動畫，其藍黃交替的閃爍畫面因為受到[[3D龍事件|3D龍事件]]的影響而在之後做了修正。

動畫結局並非完整改編原著，而是改為取得魔聖銅器後，於北京紫禁城與被黑暗料理界控制心智的阿飛與兩名麟廚師進行「御前料理對決」，雖然中途一再遭受皇帝的秘書官龍總管的惡意阻撓，但還是逆轉勝，正式站上中國料理界的頂點，成為名副其實的中國第一廚師。動畫結束前，僅擁有的傳說廚具為“永靈刀”、“轉龍壺”、“魔聖銅器”三件，其餘五件尚未找齊，也無五虎星。此外原著中的西南修行期間，劉昴星的隨行夥伴僅有四郎，但動畫版讓梅麗同行。

===譯名問題===
1999年6月28日至2000年1月9日，[[台灣電視公司|台灣電視公司]]首播《中華一番》。2000年2月12日至2000年11月18日，台視以《'''中華一番之滿漢傳奇'''》為名重播《中華一番》；重播時的[[贊助商|贊助商]]為[[統一企業|統一企業]]，因此動畫中許多人物在重播時都冠以該公司產品名稱，如主角劉昴星被翻譯成「小當家」（脆麵類品牌）、「阿Q」（泡麵類品牌「阿Q桶麵」），還有「及第師父」（冷凍水餃品牌「及第水餃」）、「嘟嘟」（真空包裝食品，為「滿漢香腸」小香腸的名稱）等。此外，由於字型相似，劉昴星在動畫版中被誤譯為劉昂星。

== 電視劇 ==
{{主|中華小當家}}
中國大陸製作的電視劇，由李力持執導，蕭正楠、何潤東、羅海瓊和何美鈿聯袂主演。

==跨界合作==
2016年時、與台灣美食品牌日芳珍饌合作推出料理，在外包裝有作者繪製之角色、內容為日芳珍饌之冷凍食品及作者訪談書。<ref>[https://www.topseafood.com.tw/node/404 中華一番！X日芳珍饌 官方合作企劃]</ref>
