'''中華民國與捷克關係'''是指[[中華民國|中華民國]]與[[捷克共和國|捷克共和國]]之間的關係。兩國於1930－1949年有官方[[外交#外交上的承認|外交關係]]，斷交後直至1990年代，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。{{link-en|捷克政府|Government of the Czech Republic}}採行「[[一個中國|一個中國]]」政策，但大致維持與[[海峽兩岸關係|兩岸政府]]平衡接觸交往，並堅持以和平方式解決兩岸爭議。<ref name="駐在國國情簡介">{{Cite web|url =http://www.taiwanembassy.org/CZ/ct.asp?xItem=586177&CtNode=1091&mp=146&xp1=|title =駐在國國情簡介|publisher =中華民國駐外單位聯合網站|deadurl =yes|archiveurl =https://web.archive.org/web/20160304104920/http://www.taiwanembassy.org/CZ/ct.asp?xItem=586177&CtNode=1091&mp=146&xp1=|archivedate =2016-03-04}}</ref>近年來，在經貿、文教、科技、觀光等領域交流頻繁，雙邊實質關係穩定。<ref name="104年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>[[哈維爾|哈維爾]]在擔任[[捷克總統|捷克總統]]期間及卸任後，對中華民國採取友好態度，雙方政府官員與國會議員至今亦常互訪。

==政治==
===外交===
{{see also|中華民國外交|{{link-en|捷克外交|Foreign relations of the Czech Republic}}}}
{{see also|中国驻捷克大使列表{{!}}中華民國駐捷克斯洛伐克外交官列表|捷克驻华大使列表{{!}}捷克斯洛伐克駐中華民國外交官列表}}
1930年2月12日，[[中華民國|中華民國]]與[[捷克斯洛伐克共和國|捷克斯洛伐克共和國]]建立公使級[[外交#外交上的承認|外交關係]]。

1931年，於首都[[布拉格|布拉格]]設立'''中華民國駐捷克斯洛伐克共和國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]，初期由[[中国驻波兰大使列表|駐波蘭公使]]兼任。

1939年，[[納粹德國|納粹德國]]佔領捷克斯洛伐克後（[[蘇台德區問題|蘇台德區問題]]），原公使停派，改調[[中国驻罗马尼亚大使列表|駐羅馬尼亞]]。兩國中斷外交關係期間，捷克斯洛伐克外交承認[[滿州國|滿州國]]，滿州國並派駐公使。

1941年8月26日，中華民國外交承認位於[[英國|英國]]首都[[倫敦|倫敦]]的[[捷克斯洛伐克流亡政府|捷克斯洛伐克流亡政府]]，並在英國設立公使館，由[[中国驻荷兰大使列表|駐荷蘭公使]]兼任。

1944年7月25日，公使館升格為'''中華民國駐捷克斯洛伐克共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]，初期由駐荷蘭大使兼任。1945年7月30日，大使館正式復館。

1949年10月1日，[[中華人民共和國開國大典|中華人民共和國成立]]於[[北京|北京]]；10月3日，[[捷克斯洛伐克社會主義共和國|捷克斯洛伐克社會主義共和國]]予以承認並改與其建交。中華民國便宣布斷交閉館。當時中華民國尚能有效統治[[中國大陸|中國大陸]]部分地區，因此[[臺灣戰後時期|國民政府遷台]]後雙方就無官方關係。

1950－1990年代的[[冷戰|冷戰]]期間，中華民國在[[意識形態|意識形態]]和[[政體列表|制度]]上均偏向以[[美國|美國]]、[[英國|英國]]為首的[[資本主義|資本主義]]陣營，復加上以[[蘇聯|蘇聯]]為首的[[社會主義國家|共產集團國家]]的封閉，使雙方幾乎沒有往來。

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[捷克斯洛伐克社會主義共和國|捷克斯洛伐克社會主義共和國]]。]]
1971年10月25日，身為共產陣營成員及蘇聯[[衛星國|衛星國]]之一的捷克斯洛伐克社會主義共和國投票支持中華人民共和國加入[[聯合國|聯合國]]。（即[[中國與聯合國#代表權問題|中國代表權問題]]、《[[聯合國大會2758號決議|聯合國大會2758號決議]]》）

1989年11月，[[天鵝絨革命|天鵝絨革命]]後，促進了捷克斯洛伐克社會主義共和國的[[民主化|民主化]]與政權和平移轉，並改國號為[[捷克斯洛伐克聯邦共和國|捷克斯洛伐克聯邦共和國]]。1993年1月1日，[[天鵝絨分離|斯洛伐克宣布獨立]]，[[聯邦制|聯邦體制]]瓦解。在捷克轉型為[[民主#民主國家|民主國家]]並重新對外開放後，雙方始恢復往來。

1994年，捷克高層官員在第49屆[[聯合國大會|聯合國大會]]的總辯論中，提出「會籍普遍化原則」，間接呼應[[中國與聯合國#台灣人民代表權問題及台灣入聯運動|中華民國參與聯合國]]。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=4F8B8E40F8AEE541 |author = 公眾外交協調會 |date = 1994年10月14日 |title = 聯合國大會第四十九屆常會總辯論自九月二十六日至十月十二日召開 |publisher = 中華民國外交部}}</ref>

1995年10月，[[捷克總統|捷克總統]][[哈維爾|哈維爾]]於聯合國大會記者會中發言支持中華民國加入聯合國。<ref name="哈維爾"/>

2009年5月8日，擔任[[欧洲理事会主席|歐盟輪值主席國]]的捷克，發表聲明歡迎[[中華台北|中華台北]]受邀以[[世界衛生大會觀察員|觀察員身分]]參加第62屆[[世界衛生大會|世界衛生大會]]（WHA）。<ref name="駐在國國情簡介"/>

===交流===
中華民國與捷克的科技交流是[[東歐|東歐]]國家中最早的，自1991年[[蘇聯解體|蘇聯解體]]後即開始接觸。<ref name="99年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2010/2.1.4.pdf |title = 《中華民國99年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>2006年10月，設立'''駐捷克代表處科技組'''，並{{Link-en|多重委派|Dual accreditation|兼轄}}數個[[歐洲國家和地區列表|中東歐國家]]。<ref name="96年外交年鑑"/>2007年，舉行首屆台捷科技日<ref name="96年外交年鑑"/>，至2017年已舉行第五屆。<ref>{{Cite web |url = http://www.chinatimes.com/realtimenews/20170515001848-260407 |author =  陳建瑜 |date = 2017年5月15日 |title = 捷克高官訪台 揭台捷科技日活動 |publisher = [[中時電子報|中時電子報]]}}</ref>

2014年6月5日，第13屆台捷經濟合作會議於布拉格舉行。會議由捷克工業總會[[董事|董事]]及中華民國國際經濟合作協會（國經協會）副[[理事長|理事長]]共同主持，並邀請{{link-en|捷克工業暨貿易部|Ministry of Industry and Trade (Czech Republic)}}主管貿易及歐盟事務次長及中華民國駐捷克代表致詞。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref>

每年3月，[[台灣區電機電子工業同業公會|台灣區電機電子工業同業公會]]（電電公會）均組團前往捷克第2大城[[布爾諾|布爾諾]]參加捷克國際電機電子展，該展是[[中歐|中]]、[[東歐|東歐]]最大商展。<ref name="投資環境簡介"/>
