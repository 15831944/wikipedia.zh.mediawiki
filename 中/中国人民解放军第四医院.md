'''中国人民解放军第四医院'''，位于[[青海省|青海省]][[西宁市|西宁市]][[城东区_(西宁市)|城东区]]八一东路65号，隶属[[西宁联勤保障中心|西宁联勤保障中心]]，为[[三级甲等医院|三级甲等医院]]<ref>{{Cite web |url =  http://www.xncb.gov.cn/t?t=89276  |title = 市级定点医疗机构 |author =  |date = 2015-05-26 |publisher = 西宁·城北 |language =  |accessdate =  |quote =  }}</ref>。

==简介==
中国人民解放军第四医院的前身是中国人民解放军第一军卫生部所属的三个卫生休养所，曾随[[西北野战军|西北野战军]]转战[[陕甘宁边区|陕甘宁边区]]，参加[[延安保卫战|延安保卫战]]、解放大西北。1951年4月，组建为'''中国人民解放军第四医院'''。同时成立中国人民解放军第四医院高原医疗队，到[[青藏高原|青藏高原]]各地巡诊<ref>{{Cite web |url =http://www.qhnews.com/2009zt/system/2009/04/07/002720641.shtml |title = 忠诚——记兰州军区高原医疗队 |author =  |date = 2009-04-07 |publisher = 青海新闻网 |language =  |accessdate =  |quote =  }}</ref>。

2010年，医院参加了[[玉树地震|玉树地震]]抗震救灾工作<ref>{{Cite web |url =http://news.enorth.com.cn/system/2010/05/26/004717062.shtml  |title =解放军四医院累计救治地震伤病员1.2万人次  |author =  |date = 2010-05-26 |publisher = 北方网 |language =  |accessdate =  |quote =  }}</ref>。2011年4月18日，由中国人民解放军第四医院和[[塔尔寺|塔尔寺]]藏医院联手建设的中国人民解放军第四医院藏医科在青海西宁正式成立并接诊病人，这是中国首家军地共建的藏医门诊<ref>{{Cite web |url = http://news.163.com/11/0418/18/71UMAVT100014JB5.html |title = 首个军地共建的藏医门诊在解放军第四医院成立 |author =  |date = 2011-04-18  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，医院由[[中国人民解放军兰州军区|中国人民解放军兰州军区]]联勤部转隶[[西宁联勤保障中心|西宁联勤保障中心]]<ref>{{Cite web |url =http://www.81.cn/jwgz/2017-06/02/content_7625444.htm|title =生病不用怕！“五个零”硬标准保障看病就医 |author =  |date = 2017-06-02 |publisher =中国军网 |language =  |accessdate =  |quote =  }}</ref>。
