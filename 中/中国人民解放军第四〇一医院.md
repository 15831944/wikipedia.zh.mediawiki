'''中国人民解放军第四〇一医院'''，位于[[山东省|山东省]][[青岛市|青岛市]]，隶属[[中国人民解放军北部战区海军|中国人民解放军北部战区海军]]，是综合性[[三级甲等医院|三级甲等医院]]<ref>{{Cite news|url=http://bddsb.bandao.cn/data/20110315/html/75/content_1.html|title=解放军第401医院特色科室诚信为民|author=|date=2011-03-15|work=半岛都市报|accessdate=}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==简介==
1950年5月27日正式开院，其组织基础是从湖南北上的第四野战军后勤二分部第二兵站医院和第二野战军军部人员及第十师，院址设于青岛[[浮山所|浮山所]]西北侧建于1938年的日本陆军病院旧址。1954年9月，命名为“'''中国人民解放军第四〇一医院'''”。1994年，被军委总部评为全军首批“[[三级甲等医院|三级甲等医院]]”。1995年，被授予国家级“爱婴医院”称号。1998年，被军委总部评为“为部队服务先进单位”，同年被指定为“海军计划生育优生优育技术中心”<ref>于文学、陈洪坤，第401医院简介，解放军医院管理杂志2000年第05期</ref>。2006年医院成立远程会诊中心，可与其他军队医院共享医疗资源<ref>{{Cite news |url = http://news.bandao.cn/news_html/201503/20150313/news_20150313_2510743.shtml |title=不少医院设远程会诊中心 不出青岛看国外名医|author=王鑫鑫|date=2015-03-13|work=城市信报|accessdate=}}</ref>。2012年，医院曾派员参与[[南苏丹|南苏丹]][[维和|维和]]任务<ref>{{Cite news |url = http://news.bandao.cn/news_html/201211/20121130/news_20121130_2030561.shtml?i|181598:1 |title=401医院7医援人员回国 维和10个月瘦40斤(图) |author=宋涛 |date=2012-11-30 |work=半岛都市报 |accessdate=2017-05-01}}</ref>。

医院原隶属[[中国人民解放军海军|中国人民解放军海军]]，2004年9月转隶[[中国人民解放军济南军区|中国人民解放军济南军区]]联勤部<ref name=ysy/><ref name=jinan>{{Cite web |url =http://www.81.cn/lj/2014-07/04/content_6034086.htm  |title =济南军区401医院 主题教育人人登台  |author =  |date =2014-07-04   |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://epaper.qingdaonews.com/html/qdrb/20120118/qdrb368203.html  |title =瞄准一流强内功 聚焦科技谋发展  |author =  |date = 2012-01-18 |publisher = 青岛日报 |language =  |accessdate =  |quote =  }}</ref>。2004年9月，原中国人民解放军第一四一医院并入第四〇一医院，成为第四〇一医院北院区，位于青岛市李沧区<ref name=fanggr/><ref name=ysy>{{Cite web |url =http://www.qingdaonews.com/gb/content/2004-10/28/content_3822675.htm  |title = 原解放军141医院更名为401医院北院区401医院开始南北连锁 |author =  |date = 2004-10-28 |publisher = 青岛新闻网 |language =  |accessdate =  |quote =  }}</ref>；原中国人民解放军第四〇九医院（始建于1973年）转隶第四〇一医院，成为第四〇一医院崂山分院<ref>刘光焰、张侃，着眼学科特色 推动创新发展——解放军第四0一医院崂山分院(原第四0九医院)建设发展掠影，实用医药杂志2009,26(7):97-98</ref>。2016年在[[深化国防和军队改革|深化国防和军队改革]]中，第四〇一医院转隶[[中国人民解放军北部战区海军|中国人民解放军北部战区海军]]后勤部<ref>{{Cite web |url = http://www.wenming.cn/sbhr_pd/zghrb/zrwl/201606/t20160628_3476567.shtml |title = 翟瑞仁 |author =  |date =2016-06-28  |publisher = 中国文明网|language =  |accessdate =  |quote =  }}</ref>。

截至2014年，医院有博士后工作站联合培养专业2个，特种医学博士培养点、硕士联合培养专业26个；拥有7个青岛市重点特色学科，9个济南军区专科中心，2个全军专科中心（全军手外科医学专科中心、 全军航海病医学专科中心），1个国家实验机构（国家药理实验机构），1个全军临床基地（全军临床药理基地），是国家首批器官移植准入医院<ref name=mod>{{Cite web |url =http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536900.htm  |title = 第四〇一医院 |author =  |date =2014-09-12  |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url =http://bddsb.bandao.cn/data/20110315/html/75/content_1.html|title =解放军第401医院特色科室诚信为民|author =|date =2011-03-15|publisher =半岛网|language =|accessdate =|quote =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name=qdzaobao/>。
