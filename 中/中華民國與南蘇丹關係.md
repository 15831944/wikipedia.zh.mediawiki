'''中華民國與南蘇丹關係'''是指[[中華民國|中華民國]]與[[南蘇丹共和國|南蘇丹共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對南蘇丹的相關事務由'''{{link-en|駐沙烏地阿拉伯王國台北經濟文化代表處|Taipei Economic and Cultural Representative Office in the Kingdom of Saudi Arabia}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-193-a60d1-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
{{see also|中華民國外交|南蘇丹外交}}
1993年7月，[[中華民國政府|中華民國政府]]撥款30萬[[美元|美元]]予[[紅十字國際委員會|紅十字國際委員會]]，採購醫療藥品援助蘇丹南部災民（[[南苏丹独立公投|時為蘇丹共和國的領土]]）。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=9AFB65552BF57A67 |author = 公眾外交協調會 |date = 1993年9月24日 |title = 中華民國政府基於人道考慮 |publisher = 中華民國外交部}}</ref>

2011年7月9日，中華民國[[外交#外交上的承認|外交承認]]南蘇丹共和國。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=8742DCE7A2A28761&sms=491D0E5BF5F4BC36&s=E7268D6ECAE0C18B |author = 公眾外交協調會 |date = 2011年7月9日 |title = 中華民國政府正式承認南蘇丹共和國 |publisher = 中華民國外交部}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|南蘇丹簽證政策|Visa policy of South Sudan}}|{{link-en|南蘇丹公民簽證要求|Visa requirements for South Sudanese citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]與邀請函或聲明書的[[中華民國國民|中華民國公民]]可附回郵信封將相關文件寄至南蘇丹駐[[美國|美國]][[外交代表機構#大使馆|大使館]]申請「旅行許可」（Travel Permit）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-193-a60d1-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|南蘇丹護照|South Sudanese passport}}的{{link-en|南蘇丹人口|Demographics of South Sudan|南蘇丹公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將南蘇丹歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2015年11月11日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-172-0dc13-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|南蘇丹經濟|Economy of South Sudan}}}}
===貿易===
2015年以前，中華民國與南蘇丹無[[国际贸易|進出口貿易]]往來。

2017年，南蘇丹是中華民國第233大貿易夥伴、第236大[[進口|進口]]夥伴、第225大[[出口贸易|出口]]夥伴，在[[非洲国家和地区列表|非洲國家]]排名皆倒數第1；在[[世界政區索引|世界經濟體]]倒數第6、1、6。出口至南蘇丹的金額為2,349[[美元|美元]]（只10月有出口額），年減99.695%；自南蘇丹進口的金額為無資料（2016年為1,392美元）。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）2,349美元，年減99.695%。※2016年，南蘇丹是中華民國第196大貿易夥伴、第216大進口夥伴、第181大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===援助===
2011年，國際合作發展基金會（國合會）與國際[[慈善机构|慈善机构]]{{link-en|美慈組織|Mercy Corps}}（Mercy  Corps）合作進行「南蘇丹[[難民|難民]][[農業|農業]]支援計畫」。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/9652c596-1cea-4268-ae5c-7832ece3c1a2.pdf |author = 國際合作及經濟事務司 |date = 2012年4月30日 |title = 國際合作發展事務100年度報告（參、100年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref>2013年，再次合作進行「南蘇丹[[艾卜耶地區|艾卜耶地區]]難民[[糧食安全|糧食安全]]支援計畫」，協助南蘇丹及[[蘇丹|蘇丹]]邊界處艾卜耶地區返鄉難民提升生產[[園藝植物|園藝作物]]之能力，並增加當地糧食安全程度。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/133bf67f-6547-4c62-8bfd-ef26e383f644.pdf |author = 國際合作及經濟事務司 |date = 2014年4月23日 |title = 國際合作發展事務102年度報告（參、102年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref>
