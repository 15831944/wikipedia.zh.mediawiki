'''中華民國與約旦關係'''是指[[中華民國|中華民國]]與[[約旦哈希姆王國|約旦哈希姆王國]]之間的關係。兩國於1957－1977年有官方[[外交#外交上的承認|外交關係]]，斷交後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。兩國於人員代訓、教育、醫療、[[人道援助|人道援助]]、經貿等各方面交流及合作在[[中華民國外交#中华民国的邦交国|無邦交國]]中仍屬密切。<ref name="駐館與駐地關係">{{Cite web |url = http://www.roc-taiwan.org/jo/post/828.html |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref>

===軍事===
{{see also|大漠計畫}}
1966年，約旦提出了請[[中華民國空軍|中華民國空軍]]訓練約旦[[飛行員|飛行員]]的要求，並開始派遣軍事人員赴台灣[[軍用機場|空軍基地]]秘密培訓。1972年5月，前[[中華民國空軍司令|空軍總司令]][[王叔銘|王叔銘]]任[[中華民國駐約旦大使列表|駐約旦大使]]。1974年，中華民國空軍派出兩名[[上校|上校]]軍官前往約旦擔任軍事顧問和高級戰術教官。1975年，前空軍總司令[[陳衣凡|陳衣凡]]任駐約旦大使。此外，中華民國派遣一個軍事教練小組赴約旦，進行[[戰鬥機|戰鬥機]]基本戰術訓練教學，同時[[機場地勤|地勤人員]]也跟進。1977年，約旦與中華民國[[外交#外交上的承認|斷交]]，但之後兩國仍有軍事交流。

===教育===
{{see also|臺灣教育|{{link-en|約旦教育|Education in Jordan}}}}
中華民國[[中華民國教育部|教育部]]、[[中華民國外交部|外交部]]、財團法人國際合作發展基金會（國合會）每年提供[[獎學金|獎學金]]名額，給約旦學生前往台灣研讀[[中華民國國語|華語文]]，取得[[學士|學士]]或[[碩士|碩]]、[[博士|博士]]學位；約旦每年亦提供獎學金，給台灣學生赴約旦研讀[[阿拉伯文|阿拉伯文]]、取得碩士或博士學位。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>約旦留學生並成立「安曼留臺[[校友會|校友會]]」。<ref name="102年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2013/07.pdf |title = 《中華民國102年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>國合會也派遣華語文教師在約旦部分大學與機構開設[[正體字|正體中文]]課程班並宣揚[[台灣文化|台灣文化]]。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

===援助===
1999年9月21日，[[921大地震|台灣發生大地震]]，約旦派遣專機運送[[帳篷|帳篷]]、[[睡袋|睡袋]]、[[毛毯|毛毯]]、[[嬰兒食品|嬰兒食品]]及[[藥品|藥品]]等救濟物資共15[[公噸|公噸]]賑災。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_6.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

國合會與{{link-en|美慈組織|Mercy Corps}}（Mercy Corps）合作於2014年12月執行「北約旦[[水井|水井]]修復計畫」，規劃修復現有2口井之可用性和供水能力，提供當地[[敘利亞內戰難民|敘利亞難民]]清潔用水，直接受益人數達43,500人。也支持約旦境內敘利亞難民營執行[[孩童|孩童]]及[[青少年|青少年]]之心理重建工作、照護敘利亞罹[[癌|癌]]難童、約旦邊境河谷掃[[地雷|雷]]、約旦[[拉尼婭王后_(約旦)|拉尼婭王后]]家庭及孩童照護中心等計畫。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/f9783c18-6276-407e-8458-51dbc00060e7.pdf |author = 國際合作及經濟事務司 |date = 2016年4月22日 |title = 國際合作發展事務104年度報告（參、104年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref>此外，[[慈濟基金會|慈濟基金會]]、[[台灣兒童暨家庭扶助基金會|台灣兒童暨家庭扶助基金會]]（家扶基金會）等機構也在約旦推動慈善工作，定期對偏鄉地區及弱勢家庭發放救濟物資、提供義診服務及照顧幼童生活與教育。<ref name="106年外交年鑑"/>

===會展===
2018年4月28日，台灣代表團出席約旦「第14屆國際城市節」（International City Festival）文化活動，展覽開幕後，約旦方面受中華人民共和國施壓，派人撤下懸掛的[[中華民國國旗|中華民國國旗]]，但台灣代表團堅決保留桌上型小幅國旗。30日，[[中華民國外交部|中華民國外交部]]對此嚴厲譴責，表示「這不會讓中國變偉大；要讓中國偉大，只有認真看待中華民國獨立存在的事實」。除國旗外，攤位還有一張用[[阿拉伯文|阿拉伯文]]書寫的告示「台灣使館」。<ref>{{Cite web |url = https://www.nownews.com/news/20180430/2745366 |author = 林仕祥 |date = 2018年4月30日 |title = 約旦文化活動撤我國旗 外交部痛批：這不會讓中國變偉大 |publisher = [[NOWnews今日新聞|NOWnews今日新聞]]}}</ref>
