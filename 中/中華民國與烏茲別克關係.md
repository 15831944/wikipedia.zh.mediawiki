===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|烏茲別克簽證政策|Visa policy of Uzbekistan}}|{{link-en|烏茲別克公民簽證要求|Visa requirements for Uzbekistani citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]與{{link-en|烏茲別克外交部|Ministry of Foreign Affairs (Uzbekistan)}}核准的邀請函之[[中華民國國民|中華民國公民]]可以在首都的[[塔什干國際機場|塔什干國際機場]]辦理單次觀光[[簽證#落地签证|落地簽證]]入境。停留3日以上，須向警察機關或住宿旅館辦理登記。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-142-11541-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=7F220D7E656BE749 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|烏茲別克護照|Uzbekistani passport}}的{{link-en|烏茲別克人口|Demographics of Uzbekistan|烏茲別克公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將烏茲別克歸類為'''灰色警示'''：提醒注意（2017年6月4日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-245-df64a-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2018年，烏茲別克是中華民國第138大貿易夥伴、第136大[[進口|進口]]夥伴、第120大[[出口貿易|出口]]夥伴。出口至烏茲別克的金額為1,130萬6,289[[美元|美元]]，年增10.509%；自烏茲別克進口的金額為273萬6,516美元，年增47.830%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）856萬9,773美元，年增2.265%。2017年，烏茲別克是中華民國第138大貿易夥伴、第147大進口夥伴、第120大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
2007年2月1日，哈薩克KazDipper [[有限公司#香港|Ltd.]][[總裁|總裁]]有意與台灣廠商合作[[承包|承包]]{{link-en|烏茲別克政府|Politics of Uzbekistan}}[[銅礦|銅礦]]標案。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=454595&url=/NewsSearch.asp |author = 賴秀華 |date = 2007年2月6日 |title = 烏茲別克政府銅礦採購標案 |publisher = 經貿透視網}}</ref>

2014年10月13日，[[經濟部國際貿易局|經濟部國際貿易局]]委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）執行的「前進[[中亞#國家及地區|中亞]][[新兴市场|新市場]]－[[哈薩克|哈薩克]]、[[烏茲別克|烏茲別克]]商機研討會」在[[台北市|台北]]舉辦。目的在協助[[臺灣商人|台商]]開發中亞市場、掌握全球[[清真|清真]]商機。<ref>{{Cite web |url = http://www.trademag.org.tw/content02.asp?id=647635&type=49&url=/NewsSearch.asp |author = 陳姿君 |date = 2014年10月29日 |title = 掌握一手資訊 前進中亞非難事 |publisher = 經貿透視雙周刊第404期}}</ref>
