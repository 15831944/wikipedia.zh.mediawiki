'''[[中华人民共和国|中华人民共和国]]高等教育'''包括[[普通高等教育|普通高等教育]]和[[成人高等教育|成人高等教育]]，屬於[[中华人民共和国教育|中华人民共和国教育]]体系中的重要一環。教育部数据显示，中国高等教育毛入学率为45.7%。<ref>{{Cite web |url=http://edu.sina.com.cn/gaokao/2018-07-20/doc-ihfqtahh4390486.shtml|title=教育部：我国高等教育毛入学率达到45.7%|accessdate=2018-11-30 |author= |date=2018-11-30 |work= |publisher=新浪网|language=Zh-hans }}</ref>中国政府预测2019年高等教育毕业生834万，1997年出生1445万，占比57%。<ref>{{Cite web |url=http://www.gov.cn/xinwen/2018-11/29/content_5344559.htm|title=2019届全国普通高校毕业生预计达834万人|accessdate=2018-11-30 |author= |date=2018-11-30 |work= |publisher=中华人民共和国中央人民政府|language=Zh-hans }}</ref>

== 分級 ==
普通高等教育分为[[专科|专科]]教育（学制二至三年）、[[本科|本科]]教育（学制四年）和[[研究生|研究生]]教育（包括[[硕士|硕士]]和[[博士|博士]]学位）三个阶段。

而成人高等教育经[[中華人民共和國教育部|教育部]]审定核准，學校包括以下类型：[[广播电视大学|广播电视大学]]、[[职工高等学校|职工高等学校]]、[[职业技术学院|职业技术学院]]、[[农民高等学校|农民高等学校]]、[[管理干部学院|管理干部学院]]、[[教育学院|教育学院]]、独立设置的[[函授学院|函授学院]]、普通高校所属的成（继）教院，实行[[成人高等学校招生全国统一考试|成人高等学校招生全国统一考试]]，全国统一招生，然而學生畢業後所發畢業證書會註明「成人高等教育」字樣。

招生类型和學程一般如下(特殊科系各校另計)：
*高中升专科（高起专）
**脱产學程：两年拿專科畢業證、無學位
**业余學程：兩年半拿專科畢業證、無學位
**函授學程：兩年半拿專科畢業證、無學位
*高中升本科（高起本）
**脱产學程：四年拿本科畢業證、附學士學位證
**业余學程：五年拿本科畢業證、附學士學位證
**函授學程：五年拿本科畢業證、附學士學位證
*[[专升本|专科升本科]]（[[专升本|专升本]]）
**脱产學程：两年拿本科畢業證、附學士學位證
**业余學程：兩年半拿本科畢業證、附學士學位證
**函授學程：兩年半拿本科畢業證、附學士學位證
*研究所（須取得學士）
**碩士脱产學程：两年到三年拿研究所畢業證、附碩士學位證
**博士脱产學程：碩士後四至六年拿研究所畢業證、附博士學位證
**直博脱产學程：學士後五年拿研究所畢業證、附博士學位證

== 争议 ==

=== 国际化 ===

进入21世纪，国际化是中国各校高校追求的目标。外籍学生的人数是评价高校教育国际化的重要因素。但一味追求扩大外籍学生人数，亦在中国高校中产生一系列问题，即使是中国最顶尖的高校亦不可避免<ref>{{Cite web |url=http://www.sohu.com/a/136857295_100928|title=以“国际化”的名义，中国高校扩招留学生的低效与不公|accessdate=2018-06-03 |author= |date=2017-04-28 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。

外籍学生在招生、教育、[[奖学金|奖学金]]等各方面享受[[超国民待遇|超国民待遇]]，本国学生则沦为[[二等公民|二等公民]]，两者待遇悬殊。以奖学金为例，本国籍学生每年仅有少数人能获得数千元的奖学金。而[[江苏省教育厅|江苏省教育厅]]在2014年发布《留学江苏行动计划》中，给予每名外籍学生5万至9万的政府奖学金。2020年时，江苏省将有5万名外籍学生<ref>{{Cite web |url=http://news.163.com/14/0829/07/A4Q3INEE00014Q4P.html|title=凭啥给留学生超国民待遇|accessdate=2018-06-03 |author=晏扬 |date=2014-08-29 |work= |publisher=网易，来源：大洋网-信息时报(广州)|language=Zh-hans }}</ref>。[[中国省级行政区人均地区生产总值列表|在2014年时，江苏省人均GDP不过8.1万元]]，人均[[可支配收入|可支配收入]]仅2.7万元<ref>{{Cite web |url=http://js.people.com.cn/n/2015/0203/c360302-23779158.html|title=江苏2014年居民人均收入27173元 城乡差距趋小|accessdate=2018-06-03 |author=朱殿平 |date=2015-02-03 |work= |publisher=人民网|language=Zh-hans }}</ref>。此举引发争议后，江苏省教育厅确认奖学金数额确实如此，但不是每名外籍学生均享有<ref>{{Cite web |url=http://js.people.com.cn/n/2014/0906/c360307-22227969.html|title=外国留学生来江苏求学每年可获5万至9万资助|accessdate=2018-06-17 |author=蒋廷玉、耿联、杨频萍 |date=2014-09-06 |work= |publisher=人民网|language=Zh-hans }}</ref>。
