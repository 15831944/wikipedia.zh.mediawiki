'''中正紀念堂站'''位於[[台北市|台北市]][[中正區_(臺北市)|中正區]]，為[[台北捷運|台北捷運]][[淡水信義線|淡水信義線]]與[[松山新店線|松山新店線]]交會的[[捷運車站|捷運車站]]。在小南門線尚未與新店線整併為[[松山新店線|松山新店線]]前，該站曾是台北捷運第一個三線[[換乘站|交會車站]]，而興建中的[[萬大-中和-樹林線|萬大樹林線]]端點站亦設於此，待萬大樹林線第一期路線通車後將再次成為三線交會車站。

== 車站概要 ==
站名取自所在地東北側著名名勝及明顯公共建物「[[中正紀念堂|中正紀念堂]]」。車站位於[[羅斯福路_(台北市)|羅斯福路一段]]中正區龍福里與新營里交界處，與南海路、[[林森南路|林森南路]]的交叉口以及愛國東西路口間，代碼為'''R08／G10／LG01'''；萬大線車站則預定設於南海路地下<ref>[http://www.dorts.gov.tw/ct.asp?xItem=1125290&CtNode=32181&mp=115001 台北市政府捷運工程局 ─ 路線說明] {{webarchive|url=https://web.archive.org/web/20101208184230/http://www.dorts.gov.tw/ct.asp?xItem=1125290&CtNode=32181&mp=115001 |date=2010-12-08 }}</ref>。

本站臨近[[臺北府城南門|臺北府城南門]]（麗正門），附近亦有南門市場，故在以括號加註「南門」副站名。本站出入口也加註副站名，開台北捷運加註副站名之先例。

2018年夏季台北捷運於部分車站新增日本語到站廣播，中正紀念堂站以音讀「ちゅうせいきねんどう(Chūsei kinen dō)」廣播。

== 車站構造 ==
紅線（淡水信義線）：採取地下三層車站，月台型式採[[島式疊式月台|島式疊式月台]]及[[島式月台|島式月台]]，為[[跨月台轉車站|跨月台轉車站]]<ref>{{Cite web |url=http://www2.sdpo.dorts.gov.tw/paper_gln/3th/stuff2.htm |title=第一章 路線規劃 |access-date=2010-11-28 |archive-url=https://web.archive.org/web/20131113020222/http://www2.sdpo.dorts.gov.tw/paper_gln/3th/stuff2.htm |archive-date=2013-11-13 |dead-url=yes }}</ref>，兩個[[島式月台|島式-{月台}-]]分別位於地下第二層與第三層兩月台層間上下平行重疊，七個出入口，本站設有[[半高式月台門|半高式月台門]]。<ref>{{cite web|url=http://www2.sdpo.dorts.gov.tw/paper_gln/3th/stuff2.htm|title=第二章 土木，建築，水環、軌道|publisher=Department of Rapid Transit Systems|accessdate=2010-11-27|archive-url=https://web.archive.org/web/20131113020222/http://www2.sdpo.dorts.gov.tw/paper_gln/3th/stuff2.htm|archive-date=2013-11-13|dead-url=yes}}</ref> 由於車站地下一樓（穿堂層）至二樓（月台層）採整體挑高設計，因此穿堂層被分為南北兩部分，兩側大廳及南北兩側出入口（出口1～4、出口5～7）無法經由穿堂層相通。

萬大線：地下二層車站，二個[[島式月台|島式月台]]，七個出入口。


設於該站南端的-{zh-tw:橫渡線; zh-hk:渡線; zh-hans:渡线;}-除了聯絡淡水線及新店線之外，還可以使其餘高運量路線列車在經新店線或小南門線抵達該站後，得以換軌至淡水線，前往[[北投機廠|北投機廠]]進行大型維修作業。

=== 車站月台 ===
*一月台為淡水信義線「[[淡水站|淡水]]－[[象山站|象山]]」及「[[北投站|北投]]－[[象山站|象山]]／[[大安站_(臺北市)|大安]]」往[[淡水站|淡水]]、[[北投站|北投]]方向的上行月台；三月台則為往[[象山站|象山]]、[[大安站|大安]]方向的下行月台。
*二月台為松山新店線「[[松山車站_(台灣)|松山]]－[[新店站|新店]]」及「[[松山車站_(台灣)|松山]]－[[台電大樓站|台電大樓]]」往[[松山站|松山]]方向的上行月台，四月台則為往[[新店站|新店]]、[[台電大樓站|台電大樓]]方向的下行月台。

本站一、三月台的LED跑馬燈隨信義線通車後，使用與信義線路段相同的LED跑馬燈與電子看板。
本站二、四月台的跑馬燈則使用新店線路段的新型跑馬燈與電子看板。

=== 車站出口 ===

出口1～4位於車站南端、出口5～7位於車站北端，[[無障礙|無障礙]]電梯位於出口1、出口5。其中有兩個電梯於出口1位於南端，出口5位於北端。
* 出口1：勞工保險局總局暨台北業務組（羅斯福路西側、南海路北側）[[File:Accessibility-directory.svg|20px]]
* 出口2：[[南門市場_(臺北)|南門市場]]、南海路（羅斯福路西側、南海路南側）
* 出口3：金華街頭與羅斯福路一段的交界處
* 出口4：林森南路（羅斯福路東側、林森南路口）
* 出口5：[[中正紀念堂|中正紀念堂]]（愛國東路北側、中山南路口），設有藝廊 [[File:Accessibility-directory.svg|20px]]
* 出口6：國家圖書館、中山南路（中山南路西側、愛國西路口）[[臺北市立大學附設實驗國民小學|臺北市立大學附設實驗國民小學]]的後門
* 出口7：愛國西路、中央業務局、中央銀行、國庫署、菸酒公司

出口1為了配合萬大線施工於2013年3月24日進行改建。由於未來南海路下方的萬大線站體啓用後，爲了與紅線、綠線進行站內轉乘，其付費區將透過通道連接至現有車站B1南側大廳。屆時橫越南海路下方的出口1必須移位，並加建電梯和手扶梯連接地面和車站<ref>[http://web.pcc.gov.tw/tps/tps/tp/main/pms/tps/tp/InitialDocumentPublicRead.do?pMenu=common&method=getTpReadFormal&tpReadSeq=50011018 捷運中正紀念堂1號(A2)出口改建工程-政府電子採購網]</ref> ，暫時封閉出口一及二號月台上方部分空間，穿堂層靠近出口一的付費區未來預留通道通往萬大樹林線，一號出口也設置獨立進出站閘門。此工程已經完工於2014年10月8日重新啟用開放。

== 營運狀況 ==
[[File:A_train_of_Xiaonanmen_Line_approaching_CKS_Meml_Hall.JPG|250px]]
[[File:中正紀念堂站近況.jpg|250px]]
本處除捷運站外，車站周圍的[[羅斯福路_(台北市)|羅斯福路]]、愛國東路與中山南路皆設有「捷運中正紀念堂站」[[公車|公車]]站牌，有多條公車路線行經。週邊也有不少行政機關與公共設施，愛國東路上則有許多婚紗店。

根據{{TRTS ridership-date}}資料，中正紀念堂站每日進出站客量約為{{TRTS ridership|station=中正紀念堂站}}人次，在台北捷運各站中排行第{{TRTS ridership-rank|station=中正紀念堂站}}名。由於本站兩線列車幾乎同時抵達，且為[[跨月台轉車站|跨月台轉車站]]的緣故，經常有於兩線之間換車的乘客在月台上穿梭，繁忙程度亦較其他轉車站高，是系統內轉乘人次最高的車站之一。此外也為了防止人員蓄意跳軌等因素，本站亦於2013年增設[[半高式月台門|半高式月台門]]來保護民眾與行車的安全<ref>[http://udn.com/NEWS/SOCIETY/BREAKINGNEWS2/5872039.shtml 防跳軌 北捷15站設月台門]</ref>。

另外由於本站是淡水信義線與松山新店線的轉乘站關係，再加上月台設計又是採跨月台平行轉乘，導致出現兩線之間換車的乘客於月台上奔跑的現象，因此基於乘客安全上的考量，在松山新店線往松山方向的列車停靠本站後，除了本站轉乘資訊後再增加「換車時請勿奔跑」的廣播來提醒乘客。

此外，[[臺北市政府|臺北市政府]]、[[臺北市政府捷運工程局|臺北市政府捷運工程局]]、[[國家兩廳院|國家兩廳院]]曾於2008年1月推出以地下通道聯絡捷運站及兩廳院的計畫，原先預定於10月完工，不過卻由於[[鋼鐵|鋼筋]][[水泥|水泥]]等原物料價格的飆漲而宣告暫緩。<ref>聯合新聞網 工程費飆漲捷運連通兩廳院跳票</ref>

== 紀念章 ==
本站的專屬紀念章是以「中正紀念堂」主體為主，結合廣場兩旁造景，增添中式建築景觀之美。

== 內裝與公共藝術 ==
中正紀念堂站因為是國家戲劇院與音樂廳等重要文化機構的主要進出玄關，再加上車站內部空間廣闊且有加高挑高高度的月台區，因此擁有與臺北捷運其他車站迥異的內裝佈局並擺設有許多公共藝術品。車站內部皆是以石材牆面裝飾，包括上層月台正上方的天花板到車站出口手扶梯處與中間連結的走道，都可以看到各種不同的公共藝術品。

;非想、想飛
:往金華街之出口3電扶梯側牆有一組由林書民製作、玻璃材質的[[全像攝影|全像攝影]]公共藝術作品─。此全像攝影描述了一架[[紙飛機|紙飛機]]的機身，由一張白紙變成包括畢業證書、聯考考卷、火車時刻表乃至於罰單等各種一個人的生命中會遇到的紙張狀物件，並在最後化成一隻[[鴿子|鴿子]]的過程{{R|PublicArt1}}。

;舞台、月台
:在車站月台層之間的上方樓梯設置、以烤漆不鏽鋼板與鋼管的二組垂簾造型公共藝術作，作者為黃承令。

;輕鬆的雲、走路的樂
:「輕鬆的雲、走路的樂」是一組設置在往財政部之出入口7通道壁面（音符圖案）及天花（燈箱60件）處的公共藝術作品，由莊普與楊岸創作。在燈箱中展示的圖片，是全台灣在不同時期、不同場景的各種畫面。

== 燈光 ==
本站的燈光曾經過調整，較營運之初來得亮<ref>[http://www.chinaelectric.com.tw/mrt.htm 台北捷運G11中正紀念堂站 照明改善計畫 ‧ 中國電器版權所有 ‧] {{webarchive|url=https://web.archive.org/web/20101123155842/http://www.chinaelectric.com.tw/mrt.htm |date=2010-11-23 }}</ref>。                                  

之前的主要光源是現在天花板上的日光燈，光源照到天花板上再反射到地面上，以照亮整個車站。次要光源是在站名看板上有鹵素燈或燈箱。因為主要光源是間接照明，距離又比較遠（挑高的關係），所以整個車站的燈光顯得較暗。                                  

後來，在各個柱子上方加裝燈光成為新主要光源，利用反射的裝置反射到牆上與地面，同時部分也直接射到天花板上，整個站體也就明亮得多。原本天花板的日光燈一部份不再使用，繼續使用的部份也減半。

== 本站歷史 ==
* 1998年12月24日：紅線側（一月台、三月台）月台隨著淡水線「[[台北車站|-{台北}-車站]]－中正紀念堂」段、新店線「中正紀念堂－[[古亭站|古亭]]」段與[[台北捷運中和線|中和線]]正式通車而啟用。
* 2000年8月31日：綠線側（二月台、四月台）月台隨著小南門線「[[西門站|西門]]－中正紀念堂」正式通車而加入營運。
* 2010年11月：萬大線前置工程動工。
* 2013年3月24日：1號出口配合萬大線與紅線、綠線站內轉乘工程封閉施工。
* 2013年10月13日：辦理信義線初勘作業。
* 2013年11月8日：交通部辦理信義線履勘作業。
* 2013年11月24日：紅線側（一月台、三月台）月台之路軌隨信義線「中正紀念堂—[[象山站|象山]]」正式通車而延伸至象山站。另因應信義線通車，小南門線自本站延伸至新店線[[台電大樓站|台電大樓站]]。至此，本站四線月台全數進入營運。<ref>{{cite web|title=政院經建會討論通過台北市政府所報「臺北都會區大眾捷運系統內湖線、南港線東延段、新莊線暨蘆洲支線、信義線及松山線通車時程調整或訂定」一案，同意依據台北市政府實際工程進度，重新訂定營運通車時程（行政院經濟建設委員會）|url=http://www.cepd.gov.tw/m1.aspx?sNo=0010579&key=&ex=%20&ic=&cd=|date=2008年8月25日|accessdate=2008年9月28日|deadurl=yes|archiveurl=https://web.archive.org/web/20090120141859/http://www.cepd.gov.tw/m1.aspx?sNo=0010579&key=&ex=%20&ic=&cd=|archivedate=2009年1月20日}}</ref> 
*2014年3月30日：反服貿遊行期間，本站與台大醫院站一度湧入大量人潮，當日下午2點至2點40分啟動列車過站不停模式，淡水線當日有八班列車過站不停。<ref>[http://udn.com/NEWS/NATIONAL/NAT4/8582163.shtml 北捷首度因遊行 機動過站不停]</ref>
*2014年7月1日：[[台北捷運淡水線|淡水線]]、[[台北捷運新店線|新店線]]半高式月台門正式啟用。。
*2014年10月8日：重置新建後的1號出口開放使用。
*2014年11月15日：隨松山線「松山－西門」通車，新店端發出的列車將透過原小南門線開往松山站，象山端發出的列車將直通至淡水站，本站四側月台回復原本營運設計。
