'''中華民國與伊拉克關係'''是指[[中華民國|中華民國]]與[[伊拉克共和國|伊拉克共和國]]之間的關係。伊拉克於1942－1958年，是[[中華民國外交#中华民国的邦交国|中華民國的邦交國]]，[[外交#外交上的承認|斷交]]後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對伊拉克的相關事務由'''駐[[約旦|約旦]][[臺北經濟文化辦事處|臺北經濟文化辦事處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-247-7d3d1-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[伊拉克|伊拉克]]。]]
{{see also|中華民國外交|{{link-en|伊拉克外交|Foreign relations of Iraq}}}}
{{see also|中国驻伊拉克大使列表{{!}}中華民國駐伊拉克外交官列表|伊拉克驻华大使列表{{!}}伊拉克駐中華民國外交官列表}}
1942年3月16日，兩國簽署《中華民國[[伊拉克王國|伊拉克王國]]友好條約》，建立公使級[[外交#外交上的承認|外交關係]]。<ref name="條約協定">{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e4%bc%8a%e6%8b%89%e5%85%8b&tycountry_e=%e4%bc%8a%e6%8b%89%e5%85%8b&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

1943年，中华民国开始派驻伊拉克[[特命全權公使|公使]]，由[[中国驻伊朗大使列表|驻伊朗公使]]兼任。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1950年，於首都[[巴格達|巴格達]]設立'''中華民國駐伊拉克王國[[外交代表機構#公使馆|公使館]]'''。

1956年3月，公使館升格為'''中華民國駐伊拉克王國[[外交代表機構#大使馆|大使馆]]'''，並派駐[[特命全權大使|大使]]。

1957年8月14日，簽署《中華民國與伊拉克王國間文化專約》。<ref name="條約協定"/>

1958年，伊拉克[[1958年伊拉克革命|发生政变]]，[[阿拉伯复兴社会党|阿拉伯复兴社会党]]推翻[[哈希姆王朝|哈希姆王朝]]后，伊拉克政府承认[[中华人民共和国|中华人民共和国]]。7月18日，与中华民国断交。

1991年3月，{{link-en|1991年伊拉克暴動|1991 uprisings in Iraq|伊拉克爆發內戰}}後，其國內上百萬[[庫德族|庫德族]]難民湧入[[伊朗|伊朗]]及[[土耳其|土耳其]]，[[中華民國政府|中華民國政府]]提供1,000萬[[美元|美元]]援助庫德族難民，其中，有120萬美元捐贈予{{link-en|伊拉克紅新月會|Iraqi Red Crescent Society}}。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=2DA8A151B2D9BC0E |author = 公眾外交協調會 |date = 1991年9月18日 |title = 中華民國政府鑒於伊拉克自本年三月爆發內戰後 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境伊拉克。]]
[[File:Visa_requirements_for_Iraqi_citizens.png|thumb]]的{{link-en|伊拉克人口|Demographics of Iraq|伊拉克公民}}必須申請簽證方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|伊拉克簽證政策|{{link-en|伊拉克公民簽證要求|Visa requirements for Iraqi citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。因伊拉克政局不穩以及與[[約旦|約旦]]邊境未有簽證查驗單位，[[中華民國外交部|中華民國外交部]]呼籲勿前往。由{{link-en|庫德斯坦自治政府|Kurdistan Regional Government|伊拉克庫德族自治政府}}所核發簽證，僅可前往[[伊拉克庫德斯坦|自治政府轄區]]，並不具前往{{link-en|伊拉克政府|Politics of Iraq}}管轄地區之效力，但由{{link-en|伊拉克駐外機構列表|List of diplomatic missions of Iraq|駐外大使館}}核發的伊拉克簽證，一般能通行伊拉克全境。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-247-7d3d1-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持有[[伊拉克護照|伊拉克護照]]的{{link-en|伊拉克人口|Demographics of Iraq|伊拉克公民}}抵台參加國際會議、運動賽事、商展活動，可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

中華民國外交部的[[國外旅遊警示分級表|國外旅遊警示分級表]]將伊拉克歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2017年11月27日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-73-86853-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|伊拉克經濟|Economy of Iraq}}}}
2009年，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）籌組16家包括[[建筑材料|建材]]、[[手工具|手工具]]、機械、零組件、[[汽車零組件列表|汽車零配件]]及[[日用品|家用品]]等[[產業|產業]]的參展團，於10月12－15日前往[[伊拉克庫德斯坦|伊拉克庫德族自治區]]的首府[[艾比爾|艾比爾]]參加「2009年伊拉克艾比爾國際商展」，這是中華民國首次進入伊拉克參展的紀錄。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=521713&url=/NewsSearch.asp |title = 臺灣首發參展團成功叩關伊拉克市場 |author = 李哲賢 |date = 2009年11月3日 |publisher = 經貿透視網}}</ref>

2012年底，[[經濟部國際貿易局|經濟部國際貿易局]]（國貿局）委託外貿協會發起「伊拉克貿易尖兵計畫」。<ref name="貿協伊拉克破冰團臺灣商品驚艷伊國 締造商機800萬美元"/>成功邀請伊拉克大型國營紡織事業Hilla Textile公司前來採購[[不織布|不織布]]機械；伊拉克庫德族自治區總商會與進出口[[工會|工會]]也在8月21－31日派員前來訪問。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622045&url=/NewsSearch.asp |title = 貿易尖兵勇闖22國26城市 新興市場商機無窮 |author = 外貿協會 |date = 2013年10月30日 |publisher = 經貿透視網}}</ref>

2013年，國貿局委託外貿協會籌組「2013年[[中東|中東]]貿易訪問團」於11月1－4日首度前往伊拉克拓銷。該團34家廠商近50位團員，於伊拉克庫德族自治區首府艾比爾舉行貿易洽談會。最受買主歡迎的產品包括汽車零配件、整廠設備、[[太陽能|太陽能]]仿古[[油燈|油燈]]、[[五金件|五金]]建材、[[医疗器械|醫療器材]]、[[陶瓷|陶瓷]]藝品及[[膠帶|膠帶]]等。<ref name="貿協伊拉克破冰團臺灣商品驚艷伊國 締造商機800萬美元">{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622569&url=/NewsSearch.asp |title = 貿協伊拉克破冰團臺灣商品驚艷伊國 締造商機800萬美元 |author = 詹雅晴 |date = 2013年11月5日 |publisher = 經貿透視網}}</ref>

2018年，伊拉克是中華民國第31大貿易夥伴、第26大[[進口|進口]]夥伴、第88大[[出口贸易|出口]]夥伴。出口至伊拉克的金額為5,044萬2,051[[美元|美元]]，年增11.490%；自伊拉克進口的金額為17億6,413萬6,729美元，年增44.149%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）17億1,369萬4,678美元，年增45.403%。2017年，伊拉克是中華民國第38大貿易夥伴、第30大進口夥伴、第88大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
