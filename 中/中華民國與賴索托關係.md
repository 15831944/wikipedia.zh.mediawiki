'''中華民國與賴索托關係'''是指[[中華民國|中華民國]]與[[賴索托王國|賴索托王國]]之間的關係。兩國於1966－1993年有官方[[外交#外交上的承認|外交關係]]但曾經中斷，二次斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對賴索托的相關事務由'''{{link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-172-0ebf8-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[賴索托|賴索托]]。]]
{{see also|中華民國外交|{{link-en|賴索托外交|Foreign relations of Lesotho}}}}
{{see also|中国驻莱索托大使列表{{!}}中華民國駐賴索托王國外交官列表}}
1966年10月31日，賴索托與中華民國[[外交#外交上的承認|建立外交關係]]。於首都[[馬塞盧|馬塞盧]]設立'''中華民國駐賴索托王國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。[[中華民國政府|中華民國政府]]曾因萊索托[[旱災|旱災]]，而向萊索托捐助1,000[[公噸|公噸]][[玉米|玉米]]<ref name="gsg">{{cite book|title=中華民國史事紀要（初稿）|publisher=[[中華民國國史館|中華民國國史館]]|year=1983}}</ref>，並派出農耕隊協助種植玉米，因產量滿意而被[[萊索托首相|萊索托首相]]{{link-en|喬納森|Leabua Jonathan}}透過[[廣播|廣播]]向中華民國政府致謝<ref name="gsg" />。

1971年10月25日，賴索托在[[聯合國大會|聯合國大會]]上支持中華民國[[中國與聯合國#代表權問題|保有「中國」席次]]。

1983年5月14日，雙方斷交，賴索托改與[[中華人民共和國|中華人民共和國]]建交。

1986年，萊索托發生軍事[[政變|政變]]，軍方接掌政權。中華民國政府於是在1989年3月向萊索托軍政府[[貸款|貸款]]3,000萬[[美元|美元]]用作興建[[水壩|水壩]]，並派出時任[[中華民國外交部|外交部]]次長[[章孝嚴|章孝嚴]]和外交部非洲司副司長杜靈兩度訪問萊索托，以期望萊索托軍政府與中華民國重新建交<ref name=journal>{{cite journal|author=佟心平|title=台湾当局“经援外交”在非洲的发生、发展及其趋势|year=2002|publisher=[[外交學院|外交學院]]}}</ref>。其後在一名當地經營餐館的[[華僑|華僑]]和一名前[[行政院農業委員會|行政院農業委員會]]官員的協助之下，中華民國於同年年底與萊索托軍政府達成建交協定<ref name=journal />，並於翌年4月5日生效。中華人民共和國遂於同一天與萊索托斷交<ref name=mfa>{{cite web|url=http://www.fmprc.gov.cn/web/gjhdq_676201/gj_676203/fz_677316/1206_677970/sbgx_677974/|title=中国同莱索托的关系|date=2015-07|accessdate=2015-10-31|publisher=[[中華人民共和國外交部|中華人民共和國外交部]]}}</ref>。

1990年4月5日，賴索托與中華民國復交，中華人民共和國與其斷交。

1991年2月24日，賴索托軍事委員會委員賴洛托里[[准將|准將]]抵台訪問。期間會見外交部長[[錢復|錢復]]、政務次長房金炎、行政院衛生署長[[張博雅|張博雅]]、[[行政院農業委員會|行政院農業委員會]]副主任委員邱茂英、[[榮民工程事業管理處|榮民工程事業管理處]]長陳豫等人士。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=DD2609E7B736E4F7 |author = 公眾外交協調會 |date = 1991年2月23日 |title = 賴索托王國軍事委員會會委員賴洛托里准將伉儷偕隨員等一行六人 |publisher = 中華民國外交部}}</ref>10月8日，軍事委員會委員加訥[[上校|上校]]亦抵台訪問，並參加[[中華民國國慶|中華民國國慶]]活動。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=724676506F55FA1A |author = 公眾外交協調會 |date = 1991年10月7日 |title = 賴索托王國軍事委員會委員加訥上校夫婦等一行四人訂於本（十）月八日抵華訪問五天。 |publisher = 中華民國外交部}}</ref>12月，政務次長房金炎訪問賴索托。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=F332341DCBDA9C12 |author = 公眾外交協調會 |date = 1991年11月25日 |title = 外交部房次長金炎暨夫人定於本月廿六日偕同非洲司主管官員啟程前往南部非洲 |publisher = 中華民國外交部}}</ref>

1991年5月21日，賴索托國內發生暴亂後，中華民國旅賴[[僑民|僑民]]撤至[[南非|南非]]勒迪布蘭德市救難中心者已達400人，其中[[中華人民共和國國籍法|中華人民共和國公民]]有84名。並由中華民國駐賴索托[[外交代表機構#大使馆|大使館]]、駐南非大使館及駐[[約翰尼斯堡|約翰尼斯堡]][[領事館|總領事館]]全力配合下，已提供難僑妥善之照顧；政府基於保僑、護僑之立場，業由[[中華民國外交部|外交部]]和[[僑務委員會|僑務委員會]]分別急匯5萬[[美元|美元]]，另[[中華救助總會|中國大陸災胞救濟總會]]亦透過外交部電匯5萬美元，配合進行濟助受難僑胞事宜。當時之濟助行動由駐賴索托大使館統籌協調辦理。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=9DF6C445AEFC2C84 |author = 公眾外交協調會 |date = 1991年5月30日 |title = 賴索托王國自本月廿一日晚發生暴亂後 |publisher = 中華民國外交部}}</ref>

1993年12月24日，賴索托與中華民國再度斷交。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#电子签证|電子簽證]]的方式入境賴索托。]]
[[File:Visa_requirements_for_Lesotho_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|賴索托簽證政策|Visa policy of Lesotho}}|{{link-en|賴索托公民簽證要求|Visa requirements for Lesotho citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]與邀請函的[[中華民國國民|中華民國公民]]可至賴索托網站付費申辦觀光、商務及學生簽證，申請成功後，會寄發核准通知至申請人[[電子信箱|電子信箱]]。經由[[南非機場列表|南非國際機場]]轉機入出境賴索托，不需申辦南非[[過境簽證|過境簽證]]；若出機場再轉搭陸上交通工具前往賴索托，因已實際入境南非，故須申辦[[南非簽證政策|南非簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-172-0ebf8-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ElectronicVisa.aspx?n=6F246645F10311D7&AreaSN=D33B55D537402BAA |title = 電子簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|賴索托護照|Lesotho passport}}的{{link-en|賴索托人口|Demographics of Lesotho|賴索托公民}}抵台參加國際會議、運動賽事、商展活動，也可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將賴索托歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2017年6月3日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-400-00773-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|賴索托經濟|Economy of Lesotho}}}}
===貿易===
2017年，賴索托是中華民國第101大貿易夥伴、第175大[[進口|進口]]夥伴、第86大[[出口贸易|出口]]夥伴。出口至賴索托的金額為5,563萬2,993[[美元|美元]]，年增3.002%；自賴索托進口的金額為24萬8,702美元，年減32.631%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）5,538萬4,291美元，年增3.247%。※2016年，賴索托是中華民國第99大貿易夥伴、第167大進口夥伴、第83大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

==交流==
===援助===
1992年初，[[非洲南部|非洲南部]]嚴重缺雨造成大規模[[乾旱|乾旱]]，[[中華民國政府|中華民國政府]]捐贈100萬[[美元|美元]]予賴索托，用以購買[[玉米|玉米]]賑濟災民。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=A4A345D25A3A034F |author = 公眾外交協調會 |date = 1992年4月24日 |title = 我撥款五百萬美元援贈非洲南部三友邦賑災 |publisher = 中華民國外交部}}</ref>

[[台灣世界展望會|台灣世界展望會]]從1999年開始，每年投入1,300多萬[[新台幣|新台幣]]，幫助5,800名兒童，並教育賴索托人[[耕種|耕種]]，增加[[糧食|糧食]]和[[收入|收入]]。2012年，賴索托宣布全國[[紧急状态|進入糧食危機狀態]]，[[行政院農業委員會農糧署|行政院農業委員會農糧署]]透過台灣世界展望會，捐贈2,000[[公噸|噸]][[白米|白米]]應急。<ref>{{Cite web |url = http://news.tvbs.com.tw/local/4130 |author =  林秉儀 |date = 2013年1月30日 |title = 【拯救賴索托】賴國1成愛滋遺孤 台送米年捐千萬 |publisher = [[TVBS新聞台|TVBS新聞台]]}}</ref>

2002年2月，[[非政府組織|非政府組織]]的醫療團[[臺灣路竹醫療和平會|台灣路竹會]]赴賴索托義診。抵達當地後，由於[[中華人民共和國|中華人民共和國]]向{{Link-en|賴索托外交暨國際關係部|Ministry of Foreign Affairs and International Relations (Lesotho)|賴索托外交部}}施壓，使得該國衛生部不同意醫療團在當地義診，為了避免醫療資源浪費，路竹會轉往[[史瓦濟蘭|史瓦濟蘭]]義診。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=4D1E240AB4C7F7AF&sms=10AEB7B38956F518&s=3F0C9D3263939F42 |author = 公眾外交協調會 |date = 2002年5月2日 |title = 中國大陸阻撓我國際空間事例（91年1月至3月事例）|publisher = 中華民國外交部}}</ref>

2009年，[[財團法人中華民國佛教慈濟慈善事業基金會|慈濟]]志工發放向賴索托農民發放耐旱的玉米種子，幫助當地居民自力更生。

2010年7月13日，慈濟的志工們計畫收購這些玉米，當做冬令發放的物資，今年賴索托的慈濟冬令發放，總計發放四場，幫助超過八百個家庭<ref>{{Cite web |url=http://www.tzuchi.org.tw/index.php?option=com_content&view=article&id=3918:2010-07-30-07-19-05&catid=106:2009-02-23-07-34-34&Itemid=558 |title=慈濟賴索托援糧 玉米慶收成-週二, 13 七月 2010 慈濟基金會 |access-date=2014-08-31 |archive-url=https://web.archive.org/web/20160305045550/http://www.tzuchi.org.tw/index.php?option=com_content&view=article&id=3918:2010-07-30-07-19-05&catid=106:2009-02-23-07-34-34&Itemid=558 |archive-date=2016-03-05 |dead-url=yes }}</ref>。
