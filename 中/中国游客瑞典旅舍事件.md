'''中国游客瑞典旅舍事件'''，中文媒体也称之为'''瑞典游客事件'''或'''瑞典警方粗暴对待中国游客事件'''，是指在瑞典當地時間2018年9月2日，[[中国大陆|中国大陆]]游客曾某<ref name=sina1>{{cite news |title=瑞典事件当事人 天士力尼日利亚经理曾骥被扒 |url=http://finance.sina.com.cn/stock/jhzx/2018-09-19/doc-ihkhfqnt0018130.shtml |accessdate=2018-09-19 |publisher=[[新浪|新浪]]财经 |date=2018-09-19 |archive-url=https://web.archive.org/web/20180919133200/http://finance.sina.com.cn/stock/jhzx/2018-09-19/doc-ihkhfqnt0018130.shtml |archive-date=2018-09-19 |dead-url=no }}</ref>携其父母在[[瑞典|瑞典]]旅游时提前抵达了[[斯德哥爾摩|斯德哥爾摩]]一家旅舍，因住宿问题与旅舍产生了冲突，之后被当地警方带离旅社。曾某指责其在整个事件中受到了虐待，并要求瑞典方面道歉赔偿。<ref name=globaltimes1>{{cite news |author1=赵觉珵 |title=中国游客遭瑞典警方粗暴对待，一家三口被扔坟场，外交部严正交涉！ |url=http://world.huanqiu.com/exclusive/2018-09/13021185.html |accessdate=2018-09-19 |publisher=[[环球时报|环球时报]] |date=2018-09-15 |archive-url=https://web.archive.org/web/20180914211131/http://world.huanqiu.com/exclusive/2018-09/13021185.html |archive-date=2018-09-14 |dead-url=no }}</ref>9月15日媒体曝光后，此事件迅速发酵，引发了各种关注与争论，同时也是[[中华人民共和国|中国]]与[[瑞典|瑞典]]之间的一场[[外交|外交]]风波。<ref name=nytimes1>{{cite news |author1=Christina Anderson |author2=Echo Hui |title=中国游客瑞典旅社事件考验两国关系 |url=https://cn.nytimes.com/world/20180918/chinese-tourists-sweden-vacation/ |accessdate=2018-09-19 |publisher=[[纽约时报|纽约时报]]中文网 |date=2018-09-18 |archive-url=https://web.archive.org/web/20180919130951/https://cn.nytimes.com/world/20180918/chinese-tourists-sweden-vacation/ |archive-date=2018-09-19 |dead-url=no }}</ref>

== 事件经过 ==
2018年9月2日凌晨1点，曾某携其父母<ref name="很大"/>抵达了瑞典[[斯德哥爾摩|斯德哥爾摩]][[國王島區|國王島區]]的一家名为“发电机斯德哥尔摩”（Generator Stockholm）的连锁旅社，但预订的房间需当天白天才能入住，曾某要求旅店让他们在大厅椅子上休息一段时间，但遭到旅店拒绝<ref name=chinadaily1>{{cite news |author1=曹德胜 |title=中国游客被瑞典警察粗暴对待事件当事人：“碰瓷”不合逻辑，严惩涉事警察 |url=http://cn.chinadaily.com.cn/2018-09/17/content_36929279.htm |accessdate=2018-09-19 |publisher=[[中国日报|中国日报]] |date=2018-09-17 |archive-url=https://web.archive.org/web/20180917102827/http://cn.chinadaily.com.cn/2018-09/17/content_36929279.htm |archive-date=2018-09-17 |dead-url=no }}</ref><ref>http://www.epochtimes.com/gb/18/9/15/n10716821.htm</ref><ref name=thepaper1/>。

随后，店方报警，两名女警到场处理，警方记录显示出警人员在当地时间2日凌晨1点43分到达涉事旅店<ref name=thepaper1/>。警方到场后强行将曾父抬出酒店，据曾某称，警方曾将其父亲从座位上拉倒，并拖行至门口后才抬出酒店，期间曾父犯病抽搐，警方未给予任何人道帮助<ref name=globaltimes1/>。由于店内并无监控摄像，店方与曾某的争执过程，以及警方在店内的处置过程，均未留下影像资料<ref name="dianfang"/>。根据后来在网上曝光的曾某及路人拍摄的涉事视频，曾某在店外有假摔及大喊呼喊的过激行为<ref name=hkej1>{{cite news |title=社評 - 茶杯裏的外交風波 中國遊客涉嫌假摔 |url=http://www1.hkej.com/dailynews/article/id/1947855/ |accessdate=2018-09-19 |publisher=[[信报|信报]] |date=2018-09-19 |archiveurl=https://cn.reuters.com/article/%E6%B8%AF%E6%8A%A5%E7%A4%BE%E8%AF%84%EF%BC%9A%E8%8C%B6%E6%9D%AF%E9%87%8C%E7%9A%84%E5%A4%96%E4%BA%A4%E9%A3%8E%E6%B3%A2%EF%BC%8C%E4%B8%AD%E5%9B%BD%E6%B8%B8%E5%AE%A2%E6%B6%89%E5%AB%8C%E5%81%87%E6%91%94-%E4%BF%A1%E6%8A%A59%E6%9C%8819%E6%97%A5-idCNL3S1W509F |archivedate=2018-09-19 |location=[[路透|路透]]中文网}}</ref>。曾某对此的解释是为了呼救，引来行人围观，并称曾有路人要求警方给予救援。不久另外两辆警车抵达现场，曾某称警察持枪驱散人群，将曾某一家强行带上警车，曾某称期间其父母均受到警方的殴打<ref name=globaltimes1/>。但香港《[[信报|信报]]》文章称，据公开目击者的多个视频显示，当地警察在抬走曾某一家时未有暴力迹象，也没有持枪驱散民众的行为；曾某在没有警察触碰的情况之下，自己扑倒在地，高呼“杀人了”（{{lang-en|This is killing!}}）。

曾某称大概半小时后，警方将三人扔在“斯德哥尔摩市区几十公里以外的一座坟场”离开<ref name=globaltimes1/>，感觉受到了警方的侮辱和戏弄；[[法国国际广播电台|法国国际广播电台]]则引述网友言论为当地警方辩护称，曾某所称的坟场其实是瑞典著名景点，联合国[[瑞典世界遺產列表|世界文化遗产]][[林地公墓|林地公墓]]，并非“荒郊野外”，而且百米外还设有地铁站<ref name=rfi1>{{cite news |author1=安德烈 |title=瑞典警方对中国一家三口施暴案--真相与陷阱 |url=http://cn.rfi.fr/20180916-%E7%91%9E%E5%85%B8%E8%AD%A6%E6%96%B9%E5%AF%B9%E4%B8%AD%E5%9B%BD%E4%B8%80%E5%AE%B6%E4%B8%89%E5%8F%A3%E6%96%BD%E6%9A%B4%E6%A1%88-%E7%9C%9F%E7%9B%B8%E4%B8%8E%E9%99%B7%E9%98%B1 |accessdate=2018-09-19 |publisher=[[法国国际广播电台|法国国际广播电台]] |date=2018-09-18}}</ref>；据《晚报》9月17日报道，瑞典检方认为“把人放在林地公墓车站是瑞典警方惯用的处理方式”<ref name="荔枝网">{{cite web |title=瑞典事件已看不懂：国外节目辱华 国内又来谣言 |url=http://news.jstv.com/a/20180923/1537670234765.shtml |website=荔枝网-环球时报 |date=2018-09-23}}</ref>。其后曾某一家在路人的帮助下返回城区；到达市区后，曾某向中国驻瑞典大使馆通报此事，并向当地警方报案；之后三人赶赴机场，离开瑞典<ref name=globaltimes1/>。

9月2日凌晨6时，游客曾某向中国驻瑞典大使馆进行了事件的口头报告。由于技术原因，大使馆在9月5日上午收到曾某的书面详细报告和照片，并向涉事酒店了解情况。

== 发展 ==
9月5日，中国驻瑞典大使馆向[[瑞典外交部|瑞典外交部]]通报了情况并提出严正交涉，[[中国外交部|中国外交部]]也在北京向[[瑞典驻华大使馆|瑞典驻华大使]]提出了严正交涉。但此后的近2周时间里，瑞方未对中方的交涉进行回应，中方将事件在媒体上曝光<ref name="sina0918"/>。而瑞典方面首席檢察官埃里克森（Mats Ericsson）表示，調查早在9月7日就已經結束，評估結果是瑞典警方沒有任何過錯<ref>{{Cite web |url = http://www.ntdtv.com/xtr/b5/2018/09/20/a1392154.html |title = 風波再起 中國遊客大鬧瑞典被列入「上訴」  |author = 新唐人 |date = 2018年9月19日  |language = 中文  }}</ref>。

9月14日，中国驻瑞典大使馆在其网站上发布提醒，提到“近期还有中国游客遭到瑞方公务人员粗暴对待”，除需提高安全意识外，可在發生意外時留下证据，並即時报警并來向大使館請求协助<ref>{{cite web |author1=中国驻瑞典使馆 |title=中国驻瑞典使馆再次提醒在瑞中国公民务必提高安全意识、加强安全防范 |url=https://www.fmprc.gov.cn/ce/cese/chn/lsfw/lsxxyfw/t1595196.htm |website=中华人民共和国驻瑞典大使馆 |date=2018-09-14}}</ref>。9月15日，中国大陆媒体《[[环球时报|环球时报]]》以「中國遊客遭瑞典警方粗暴對待，一家三口被扔墳場，外交部嚴正交涉」標題最早<ref name=thepaper1>{{cite news |author1=汪伦宇 |title=“瑞典警方粗暴对待中国游客”续：出警记录显示双方争执原因 |url=https://www.thepaper.cn/newsDetail_forward_2445316 |accessdate=2018-09-19 |publisher=[[澎湃新闻|澎湃新闻]] |date=2018-09-16}}</ref>对事件进行了报道<ref name=globaltimes1/>。之后，其他媒体对事件作出了更加详尽的报道。

9月15日，《环球时报》采访事发旅店时，前台工作人员表示需周一经理上班后才能回应。9月17日，执行店长回复《环球时报》称，因事件发酵，所有评论需要由总公司统一做出回应<ref name="dianfang">{{cite web |author1=冬瓜侠 |title=瑞典事件更新：事发旅店称当晚无监控，瑞典检方说对警察不调查 |url=http://world.huanqiu.com/exclusive/2018-09/13044047.html |website=环球网 |date=2018-09-17 |access-date=2018-09-20 |archive-url=https://web.archive.org/web/20180917165147/http://world.huanqiu.com/exclusive/2018-09/13044047.html |archive-date=2018-09-17 |dead-url=no }}</ref>。瑞典當地新聞網站《晚報》對此事件則提出，預訂了房間仍會有規定的入住時間，提前報到旅館一方有理據不對顧客做出接待，並且此為包含中國在內全世界旅館會採用的營運方式<ref>{{Cite web |url = http://www.ntdtv.com/xtr/b5/2018/09/20/a1392154.html |title = 風波再起 中國遊客大鬧瑞典被列入「上訴」  |author = 新唐人 |date = 2018年9月19日  |language = 中文  }}</ref>。

9月17日中午，就该事件，事发地警察局负责人向中国大使馆确认三名中国游客未违反瑞典法律，但同时表明瑞典警察的行为也没有违反瑞典法律<ref name="sina0918">{{cite web |title=瑞媒称中国游客事件或由中方故意导演 中使馆驳斥 |url=http://news.sina.com.cn/c/nd/2018-09-18/doc-ifxeuwwr5405283.shtml |website=新浪网 |date=2018-09-18 |access-date=2018-09-20 |archive-url=https://web.archive.org/web/20180918021920/http://news.sina.com.cn/c/nd/2018-09-18/doc-ifxeuwwr5405283.shtml |archive-date=2018-09-18 |dead-url=no }}</ref>。9月17日，桂从友大使在使馆接受瑞典《快报》记者专访，认为“三名中国游客被瑞典警察粗暴对待，生命安全和尊严都受到了威胁和伤害”，质疑瑞典法律对人权和人的尊严的保护，同时指出瑞典西南部城市[[布罗斯市|布罗斯]]曾于2011年11月发生过类似事件，当时肇事警察被认定为[[渎职|渎职]]<ref name="sina0918"/>。

== 后续发展 ==
[[File:Jesper_Rönndahl_2011-05-18_001.jpg|缩略图]]
当地时间9月21日晚，[[瑞典电视台|瑞典电视台]]政治讽刺节目《[[瑞典新闻|瑞典新闻]]》主持人{{le|耶斯佩尔·罗恩达尔|Jesper Rönndahl}}在节目中拿曾某事件作为调侃对象，将曾母坐地大喊的“救命啊”说成“Kill me now”。他在节目开头阶段引用该节目两名工作人员在龙门歧视华人的事件反问观众，如果瑞典对萨米尔人、阿拉伯人、非洲人说这样的话，早就被大量投诉，但是对俄罗斯人和中国人却不会。又再次反问观众，瑞典人痛恨歧视，但却不包括俄罗斯人和中国人。并要求观众不要将重庆的名字混淆成有种族歧视意思的tjingtjiong。<ref name='video'></ref>

节目最后播放了在[[优酷|优酷]]上发布的一段时长1分33秒的视频，视频中的女主持人称欢迎中国游客到瑞典，称瑞典人不会在历史建筑外拉屎，并展示了一张带有中文“禁止大便”的警示牌，警示标志中的人物为头戴[[斗笠|斗笠]]的形象，正在一边用[[筷子|筷子]]吃饭，一边大便；女主持人还称中国人是[[种族主义|种族主义者]]，“[[平等原則|平等的原则]]不适用中国人”<ref>{{cite web |title=这就是辱华的瑞典节目：污蔑中国人是种族主义者 |url=http://news.163.com/18/0923/01/DSBO65DJ0001899N.html |website=网易-观察者网 |date=2018-09-23}}</ref><ref>{{cite web |title=央视评论瑞典辱华视频：趣味低级令人震惊 |url=http://www.dzwww.com/xinwen/guojixinwen/201809/t20180923_17874095.htm |website=大众网-中央电视台 |date=2018-09-23}}</ref>。中国驻瑞典大使馆发言人9月22日就该节目发表谈话，指节目主持人的言论恶毒侮辱攻击中国和中国人，予以强烈谴责，且已向瑞典电视台提出强烈抗议<ref name="fmprc922">{{cite web |title=中国驻瑞典大使馆发言人就瑞典电视台辱华节目发表谈话 |url=https://www.fmprc.gov.cn/ce/cese/chn/gdxws/t1597892.htm |website=中国驻瑞典大使馆 |date=2018-09-22}}</ref>。此外，中國大使館亦表示，節目中使用的地圖沒有台灣與西藏部分地區，侵犯了中國主權。<ref name="BBC">{{cite web |title=瑞典電視節目被指「辱華」：種族歧視還是誤會 |url=https://www.bbc.com/zhongwen/trad/world-45623776 |website=BBC中文网 |accessdate=2018-09-25 |date=2018-09-24}}</ref>
