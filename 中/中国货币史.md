中国[[货币|货币]]有着悠久的历史，早在[[先秦时期|先秦时期]]就出现了多种通用货币。中国於公元九世纪发行了人类历史上第一种[[纸币|纸币]]——'''[[交子|交子]]'''。到了元代，中国发行了世界上第一种法定货币——'''[[钞|钞]]'''。

== 先秦時期 ==
[[贝币|贝币]]被确信是中原一帶最早的货币形式，它出现在大约三千到四千五百年前。[[汉字|汉字]]中的“貨”、“買”、“賣”、“販”，以及许多与产品交换有关的字，都是以“貝”作为[[部首|部首]]。 然而，贝作为货币的流通程度仍旧是未知的，但它应该在市场中占有了主导位置。而由骨、木、石、铅和铜为材料制作的贝的仿制品则在当时的贸易体系中十分常见。

=== 商代 ===
在[[中国|中国]]的[[商代|商代]]，已经开始以[[贝壳|贝壳]]作为货币使用了，随着商品经济的发展，天然的贝壳作为货币渐渐供不应求了，于是出现[[人工贝币|人工贝币]]，如石贝币、骨贝币、蚌贝币等。到了商代晚期，出现了用铜质的[[金属贝币|金属贝币]]。

铜贝在[[商朝|商朝]]<small>（约前1500 - 约前1046）</small>旧都[[殷墟|殷墟]]的遗址中被发现，而铜成为了[[周朝|周朝]]的通用货币。

根据目前的发现，中国人在人类历史上最早开始使用金属货币。目前世界上现存最早的金属铸币——铸造铜贝，出土于中国[[河南省|河南省]][[安阳市|安阳市]]大司空村商代晚期<small>（公元前14世纪-公元前11世纪）</small>的墓穴中。<ref>[http://www.henan.gov.cn/hngk/system/2006/08/02/010000219.shtml 河南省政府：中国最早金属铸币 商代晚期铸造铜贝]{{zh-cn}}</ref><ref>http://books.google.com/books?id=RGu62yiVF8wC&pg=PT186&dq=shang+coin&lr=#PPT187,M1</ref>当时，铸币与更早使用的贝壳相似，所以它被命名为贝币。<ref>[http://chinesechinese.net/HistoryofChina.html YK Kwan：中国历史概览] {{webarchive|url=https://web.archive.org/web/20120503202130/http://www.chinesechinese.net/HistoryofChina.html |date=2012-05-03 }}{{en}}</ref><ref>[http://www.travelchinaguide.com/intro/focus/currency.htm 秦朝以前的贝币]{{en}}</ref><ref>[http://encarta.msn.com/encyclopedia_681500371_2/shang_dynasty.html 商代经济百科全书]{{en}}[http://www.webcitation.org/5kwQvkSkI 存档] 2009-10-31.</ref>

=== 周代 ===
在[[戰國|战国时期]]<small>（前476年，一說前453年或前403年～前221年）</small>，中国使用的铜质铸币主要有三种形态。 [[周朝|两周]]、[[魏国_(战国)|魏国]]、[[韩国_(战国)|韩国]]和[[秦国|秦国]]使用铲形的[[布币|布币]]；[[田齐|齐国]]使用刀形的[[刀币|刀币]]；[[赵国|赵国]]和[[燕国|燕国]]在战国时期先使用布币，后使用刀币；[[楚国|楚国]]的[[蚁鼻钱|蚁鼻钱]]。

== 秦至兩漢 ==
[[File:S-106_W_Han_banliang,_Wudi,_140-87_BC,_lead,_prob_private_mint,_22-23mm.jpg|thumb]]，[[漢武帝|漢武帝]]時期（前140年-前87年）鑄]]
[[File:Wuzhuqian.JPG|thumb]]，[[西汉|西汉]]時期铸，藏[[陕西历史博物馆|陕西历史博物馆]]]]
作为统一中国的一部分，[[秦始皇|秦始皇]]<small>（前259年－前210年8月11日）</small>取消了原来六国流通的货币，代之以根据秦国以前使用的铸币设计的全国统一的铜钱——[[半两|秦半两]]。这种铜钱为圆形，中部有方孔。这一式样直到20世纪初的中国仍在使用。由于单个硬币的价值低，中国人习惯上将一千个铜钱串在一起。然而，政府税收同时以铜钱和产品（比如丝绸卷）的形式征收。薪金在[[秦朝|秦朝]]和[[汉朝|两汉]]则以数[[石_(质量单位)|石]]粮食的形式支付。

== 晋至隋 ==
[[三国|三国]][[魏|魏]][[晋|晋]][[南北朝|南北朝]]时期的社会动乱，金属货币的流通范围减小，且形制多样，币值不一，出现了重物轻币的现象。

三国时期的[[曹魏|曹魏]]实行的实物货币政策，[[魏明帝|魏明帝]]时恢复铸行[[五铢|五铢]]钱，形制与东汉时期五铢相似。[[蜀汉|蜀汉]]和[[东吴|东吴]]多实行大钱。蜀币主要有：[[直百五铢|直百五铢]]、[[直百|直百]]等。吴币主要有：[[大泉五百|大泉五百]]、[[大泉当千|大泉当千]]、[[大泉二千|大泉二千]]等。

[[西晋|西晋]]成立后主要沿用[[汉|汉]][[魏|魏]]旧钱，兼用谷帛等实物；[[东晋|东晋]]成立之初则沿用用吴国旧钱，后来出现了五铢小钱，相传是吴兴[[沈充|沈充]]所铸，所以又称：“[[沈郎五铢|沈郎五铢]]”。

[[十六国|十六国]]期间的[[成汉|成汉]][[李寿|李寿]]铸行了中国最早的[[年号钱|年号钱]]“[[汉兴|汉兴]]”钱；[[南北朝|南北朝]]时期的社会十分动荡，币值混乱，私铸现象严重。[[北朝|北朝]]从[[北魏|北魏]]开始，钱文逐渐摆脱纪重局限，逐步向年号钱制过渡。

[[隋朝|隋朝]]的建立，使[[中国|中国]]混乱的货币趋向于统一，[[隋文帝|隋文帝]][[开皇|开皇]]三年铸行了一种合乎标准的[[五铢钱|五铢钱]]，并禁止旧钱的流通。

== 唐代 ==
[[唐|唐]][[武德|武德]]四年铸行的[[铜钱|铜钱]]——[[开元通宝|开元通宝]]，以前的纪值纪重钱币一去不复返，代之的是[[宝文币制|宝文币制]]（主要是通宝、元宝和重宝）。开元通宝是唐朝三百年的主要铸币，另外还铸有[[乾封重宝|乾封重宝]]、[[乾元重宝|乾元重宝]]、[[大历元宝|大历元宝]]、[[建中通宝|建中通宝]]、[[咸通玄宝|咸通玄宝]]及史思明所铸[[顺天元宝|顺天元宝]]、[[得壹元宝|得壹元宝]]等。

後來出現了[[飛錢|飛錢]]，是[[唐憲宗|唐憲宗]]時代產生的一種匯兌制度，商人在京城把錢交給諸軍、諸[[節度使|節度使]]、諸道[[進奏院|進奏院]]或者富豪門閥等單位，單位開立票券，商人可攜券到其它地區的辦事處取錢。這種匯兌方式減低了銅錢的需求，緩和錢幣的不足，同時減輕了商人貿易時攜帶[[貴金屬|貴金屬]]或[[銅錢|銅錢]]的不便。由於飛錢輕便，又有一些官員或富豪的保證，在大額交易時，有些商人會直接使用飛錢，故飛錢是迄今為止已知最早的[[紙幣|紙幣]]雛形。

== 五代十国 ==
[[五代十国|五代十国]]政治分裂割据，改朝换代像走马灯一样，各国以铸恶钱来增强自身实力，以达到削弱他国力量的目的，故钱币甚多，但质量不高。

[[宋朝|宋朝]]<small>（960－1279）</small>早期，中国再次在五代十国的基础上统一了币制。宋朝之前的硬币中，北部各国更喜欢使用铜铸币。南方各国则使用铅或铁铸币，[[四川|四川]]使用自己的[[铁币|铁币]]。它们在宋朝初年还继续流通了一段时期。公元1000年，国家统一完成，中国进入一个经济快速增长的时期，这主要表现在铸币量的增长。1073年是北宋铸造硬币的高峰年，政府大约生产了600万串铜钱，每串铜钱包含1000枚铜币。北宋还铸造了超过200万串硬币，出口到[[亚洲内陆|亚洲内陆]]、[[日本|日本]]和[[东南亚|东南亚]]，成为當地的主导硬币。四川出现的被称为「[[交子|交子]]」的纸质[[票據|票據]]，被宋朝商人很快采用，成为最初的纸币。这被证明十分有用，所以国家开始接手这种形式的纸币，并在1024年首次推出。12世纪，许多形式的纸币成为中国的主流货币，包括如下：

== 金元 ==
[[File:Yuan_dynasty_banknote_with_its_printing_plate_1287.jpg|thumb]]

蒙古灭[[金國|金國]]后，当时[[蒙古人|蒙古人]]也发行過纸币，但只是暂时性的，流通领域有局限，发行量也很少，反而是地方割据势力多发行纸币。忽必烈即位后，力求填补财政空缺，统一货币。他认为华北地區應繼續沿用历代铜钱貨幣體系，但當時华北的铜礦供应量其实是不足以铸钱的。有鑑於當時已流通金國纸币成本低廉，[[忽必烈|忽必烈]]于是决定发行纸币，稱之為'''[[中统钞|中统钞]]'''，並以[[金|金]]、[[银|银]]、[[绢|绢]]为储备金，是世界上第一种[[法定货币|法定货币]]。忽必烈原意是先在华北地區发行纸币，日后待铜供应量增加後，才彷效历朝铸造铜钱，与纸币并用。後來由於這種纸币體系並沒有出現重大的信貸危机，信誉很好，因此忽必烈時期一直没有铸造铜钱。直到至大(1308-1311)及至正年间，纸币因为被過量印製而大幅贬值，才铸造铜钱作为辅助性货币。

蒙古灭南宋時，南宋的[[会子|会子]]是东南、四川，和湖南地區的主要货币，用于财政收支和交易。南宋滅亡後，元朝以中统钞取代[[会子|会子]]。在宋代会子流行的地区成立了[[平准库|平准库]]，汇集金银以备兑换，又设立[[行用库|行用库]]，加强回收金银。和华北不同，宋末時南中国部分地区惯用铜钱，因此蒙古灭南宋后，仍然容許這些地區以铜钱作為流通貨幣。纸币在元朝末年、至正后期以前，一直信誉良好。元代的纸币发行量和发行地域均大大超过前代。元朝先后在首都[[大都|大都]]和各行省设置了负责管理纸币印造、发行、兑换、检验伪钞、回收昏钞等之机构，其中中央的印钞机构称[[印造局|印造局]]，负责印造交钞库和定钞库等。元朝民間私有的金银均要上交朝廷。

1260年发生的通货膨胀，使元朝在1287年用新的纸币代替了舊有纸币，但是由于无节制的纸币印刷导致的[[通货膨胀|通货膨胀]]直到元朝末期依然存在。

== 明代 ==

[[File:Old_metal_money_coins_from_China.jpg|缩略图]]
[[明朝|明朝]]<small>（1368–1644）</small>初期，国家在重新统一的早期同样尝试使用纸币。这种货币同样经历了迅速的通货膨胀，它在1450年暂停发行，但是直到1573年仍在流通。直到明朝晚期[[李自成|李自成]]威胁北京时，这种纸币才在1643年和1644年重新印刷。在明朝大部分时期，中国有一个包括所有重要交易的纯私人货币体系。从海外流入的白银, 开始在南部省[[广东|广东]]作为货币使用，并在1423年传到长江下游地区成为纳税的法定货币。各省税收自1465年起以白银的形式上交首都，[[灶户|灶户]]从1475年起开始使用白银支付，[[徭役|徭役]]豁免费从1485年起使用白银支付。中国对白银的需求部分通过西班牙人从美洲的进口得到满足，特别是[[秘鲁|秘鲁]]的[[波托西|波托西]]和[[墨西哥|墨西哥]]，在西班牙人1571年建立[[马尼拉|马尼拉]]之后。但这时的白银还没有被铸造為[[銀圓|銀圓]]。它们以重量为一个标准[[两|两]]（约36克）的银（被称为[[元宝|元宝]]）流通，尽管其纯度和重量在地区与地区间略有不同。

== 清代 ==
{{main|清朝貨幣}}

[[File:Qing_Dynasty-2000_wen-1859.jpg|thumb]]玖年製的[[大清宝钞|大清宝钞]]]]
[[清朝|清朝]]仍然保持着[[银|银]][[铜|铜]]共存的货币体系。铜质货币系统以[[文_(貨幣)|文]]为基础。银质货币系统有几个单位，在[[清朝|清朝]]时的关系是：1[[两|两]] = 10[[錢_(單位)|钱]] = 100[[分_(質量單位)|分]] = 1000[[文_(貨幣)|厘]]（银币）。

1889年，单位“[[元|元]]”被引入，与当时的[[墨西哥比索|墨西哥比索]]等价，并且细分为10角，100分，和1000文。1元相当于7钱2分（或0.72两）银，并且在那时硬币被这样用英语标记。

清朝最早发行的银币在[[广东|广东]]的铸造局铸造，面额为5分，1角、2角、5角和1元。其他地区的铸造局在19世纪90年代开设并生产与之相似的铸币。面值为的1文、2文、5文、10文和20文的铜币也得到发行。清政府于1903年起，开始发行自己的银元；以元为单位的纸币从十九世纪九十年代起，由众多地方银行、私人银行，以及清政府成立的[[中国通商银行|中国通商银行]]和[[大清户部银行|大清户部银行]]（即后来的[[大清银行|大清银行]]）发行。

==近現代==

===中华民国===
====银元及关金券====
{{main|银元|关金券}}
[[File:Kwangtung_Provincial_Bank_Flowery_Pagoda.jpg|缩略图]]
[[辛亥革命|辛亥革命]]推翻[[清朝|清朝]]后，[[中华民国|中华民国]]成立。[[南京临时政府|南京临时政府]]迫切需要发行军用钞票取代之前清朝的货币。成功地，各省宣布脱离清政府独立并发行自己的军用钞票。1914年，《[[国币条例|国币条例]]》颁布并规定银元为中华民国[[国币|国币]]。尽管与封建王朝时期比，铸币的设计发生了改变，但是其规格和铸币用的材料大多保持不变，直到20世纪30年代。在20世纪20年代到30年代间，大部分地区的铸币局关闭，剩下的有些则一直开到1949年。

1931年5月中央银行发行关金兑换券，供缴纳关税。初时的关金券流通量有限。海关金单位兑换券由[[中央銀行_(中華民國)|中央银行]]发行用以方便支付进口商品的关税。关金券与美元挂钩，比价为关金一元對四十美分。这种联系汇率在一九三五年被废除，银行允许投放关金券进行流通。市场中已经充斥着过多的纸币，关金券只能加速恶性通货膨胀。

1936年起，国民政府发行含铜量達五十個百份比的一分和二分硬币，五分、十分和二十分以及半圆的[[镍币|镍币]]（一段时间后材质改为铜镍合金）。1940年，发行铝制一分与五分硬币。

1920年代末至1930年代初，国际银价出现大幅波动，影响中国经济。特别是美国在1934年推出的《购银法案》令国际银价上升，导致中国银根短缺、利率急升。中国的白银大量外流，本国货币濒临崩溃。中国被迫放弃[[银本位|银本位]]。这种情况加剧了国内众多商业、省立和外资银行发行不同价值货币的情况。

====一九三五年币制改革====
{{main|法币}}
1935年，国民政府进行币制改革，将货币发行权限制在四大国有银行：[[中国银行|中国银行]]、[[中央銀行_(中華民國)|中央银行]]、[[交通银行|交通银行]]和稍后加入的[[中国农民银行|中国农民银行]]。银元的流通被禁止，查禁私有白银。 发行新货币“[[法币|法币]]”为[[法定货币|法定货币]]。

1942年2月，关金提高含金量至0.88861克纯金，即一关金等于一美元。同年四月，定关金为一元折合法币二十元，与法币自由兑换，并且同时在市面流通。之后法币快速贬值，政府亦发行大额关金券，成为比法币面额更高的货币。

====东北券====
{{main|东北券}}
1945年[[抗日战争|抗日战争]]结束后，[[中央銀行_(中華民國)|中央银行]]发行了"[[东北九省流通券|东北九省流通券]]"，簡稱'''東北券'''。用以取代[[滿洲國|滿洲國]]发行的纸币，它的价值大约是其他地区流通的法币的10倍。1948年，东北券被[[金圆券|金圆券]]取代。东北券的发行使[[中国东北|中国东北]]没有发生像法币流通区一样的[[惡性通貨膨脹|惡性通貨膨脹]]。

====金圆券及银圆券====
{{main|金圆券|银圆券}}

[[第二次世界大战|第二次世界大战]]爆发，政府大量印鈔以滿足军费开支，导致法币急剧贬值。戰后，[[中華民國政府|中華民國政府]]在1948年8月对币制进行进一步改革以消除[[恶性通胀|恶性通胀]]。政府发行金圆券回收法币，比价为一元金圆券對三百万元法币對二十五美分。一元金圆券名义上价值0.22217克黄金。然而，这种货币并没有真正的由黄金支持，所以恶性通货膨胀并没有消失。

最终在1949年，中華民國政府再次宣布了一项改革，发行银圆券，恢复[[银本位|银本位]]。银圆券与金圆券的比率为一元银元券= 一百万元金圆券，以[[中央造币厂|中央造币厂]]铸造的银元作为发行支持。但是这种货币仅流通了很短的时间，[[中国共产党|中国共产党]]很快以武力奪取了中國大陸省份的控制权。它们被[[中国人民银行|中国人民银行]]发行的[[人民幣|人民幣]]取代。

====台湾货币====
{{main|台湾银行券|旧台币|新台币}}

[[台湾银行|台湾银行]]最初在日治时期的1899年由日本人建立。银行发行[[台湾银行券|台湾银行券]]，与[[日元|日元]]挂钩。[[中華民國政府|中華民國政府]]接管台湾后，改組新設的[[台湾银行|台湾银行]]被许可继续发行自己的货币——台币，与台湾银行券等价。这是[[國民政府|國民政府]]将台湾与流通法币的地区相隔离，使其远离恶性通货膨胀的尝试。但是，由于[[國共內戰|國共內戰]]超量調度台灣物資，台币持續贬值。

1949年的货币改革之后，[[新台币|新台币]]诞生，法定比率为1银元 = 3元新台币。1949年旧台币以40000:1的比率被[[新台币|新台币]]取代。国民党撤退到[[台湾|台湾]]后，[[按照法律的|法律所规定的]][[中华民国|中华民国]]货币仍然为银元。由[[台湾银行|台湾银行]]发行的新台币只在中华民国[[台澎金馬|台澎金馬]]與[[大陳島|大陳島]]流通。2000年7月1日订颁的《中央银行发行新台币办法》将[[新台币|新台币]]正式订为中华民国国币。

===日治政府货币===
[[大日本帝国|大日本帝国]]政府於1930年代至[[二戰|二戰]]期間先後佔領過許多地區及設立傀儡政權，同時亦发行货币。

====满洲====
{{main|满洲国圆}}
1931年[[滿洲事變|滿洲事變]]，[[日軍|日軍]]入侵[[滿洲|滿洲]]（[[中国东北|中国东北]]）时，这里有多种货币在流通，包括各省银行发行的货币、国民政府发行的法币以及[[朝鲜银行|朝鲜银行]]与[[台湾银行|台湾银行]]发行的纸币。

[[满洲国|满洲国]]建立后，日本人1932年7月1日在[[新京|新京]]（今[[长春|长春]]）成立了[[满洲中央银行|满洲中央银行]]。银行除了行使商业职能，还同时作为[[中央银行|中央银行]]发行货币。[[满洲国圆|满洲国圆]]最初规定1圆即23.91克[[白银|白银]]，相當於1[[銀圓|銀圓]]，但是在1935年它脱离了银本位并开始与[[日圆|日圆]]按1:1的[[匯率|匯率]]绑定。[[二次大战|二次大战]][[日本投降|日本投降]]时，满洲国圆被[[國民政府|國民政府]][[中央銀行_(中華民國)|中央银行]]发行的[[东北九省流通券|东北九省流通券]]收兑。

====内蒙古====
[[File:MENG_CHIANG_BANK_5_CHIAO_1938.jpg|thumb]]
在被日军占领之前，中国北方省份（包括[[绥远|绥远]]，[[察哈尔|察哈尔]]，[[山西|山西]]）的主要银行[[察哈尔商业钱局|察哈尔商业钱局]]。日本侵略后，银行撤离占领区并带走其全部资本和未发行的货币。日本军政府于1937年9月27日迅速成立了[[察南银行|察南银行]]，行使货币发行权。

傀儡政权[[蒙疆联合自治政府|蒙疆联合自治政府]]成立后，当局将[[察南银行|察南银行]]与[[察南丰业银行|察南丰业银行]]、[[绥远平市官钱局|绥远平市官钱局]]、[[冀东银行|冀东银行]]等三家较小的地方银行合并，成立[[蒙疆银行|蒙疆银行]]。蒙疆银行从1937年起发行[[蒙疆圆|蒙疆圆]]，与日本军用手票和日元等值挂钩。

====傀儡政权====

日本占领中国期间建立两个傀儡政权。在北方，设立在[[北京|北京]]的"[[中华民国临时政府_(北京)|中华民国临时政府]]"成立了[[中国联合准备银行|中国联合准备银行]]。联合准备银行1938年发行的纸币与国民政府发行的法币等值。虽然最初两者等值，但是1939年日本人开始禁止使用国币并任意设定联合准备银行券的汇率。 [[联合准备银行券|联合准备银行券]]在1945年抗日战争胜利后以5:1的比率被法币取代。

1938年[[中华民国维新政府|南京維新政府]]成立，並設立[[華興商業銀行|華興商業銀行]]，由其發行貨幣，稱華興券。后1940年，該政權同中华民国临时政府合併並改組為[[汪精卫政权|南京国民政府]]。他们成立[[中央储备银行|中央储备银行]]并在1941年开始发行[[中储券|中储券]]。尽管最初中储券与法币等值，但是它也被任意改变为与0.18元日本军票等值。1945年，中储券以200:1的比率被法币取代。

====日本军用手票====
{{main|日本军用手票}}
[[日本军用手票|日本军用手票]]在东亚的日占区广泛分布。起初，它们的发行是用于支付士兵支出。这样做的目的是无限量的军票不能兑换为日元，这样就不会使日本发生通货膨胀。然而，它对东亚当地经济的破坏性影响却不可估量。

1937年起这种货币成为了中国日占区的“法定货币”。一段时间后它改由傀儡政权银行发行。然而在1941年到1945年间，军票在[[香港|香港]]被强制使用。最初2元港币折合1元日本军票，但是[[港币|港币]]更受当地人欢迎并被大量囤积。为了解决这个问题，日军更于1943年6月起硬性规定军票为香港唯一合法的交易媒介并将军票面值贬至1元兑4元港币。

===中华人民共和国===
====人民币====
{{Main|人民币}}
[[File:Renminbi1ban_200yuan.jpg|thumb]]

[[中国共产党|中国共产党]]在1948年和1949年取得中国大东北地区的控制权。1948年12月，共产党在[[华北银行|华北银行]]、[[北海银行|北海银行]]、[[西北农民银行|西北农民银行]]的基础上，在[[河北省|河北省]][[石家庄市|石家庄市]]组建了[[中国人民银行|中国人民银行]]，接管共产党控制地区的货币发行权。[[中华人民共和国|中华人民共和国]]公告，在一段短期时间内，十萬元金圆券可以兑换一元[[人民币|人民币]]。1957年12月，中华人民共和国开始发行面值为1分、2分、5分的铝硬币。自1961年起，请苏联代印人民币三元、五元、十元券。

[[人民币|人民币]]是[[中华人民共和国|中华人民共和国]]的官方[[通貨|货币]]。它在[[中国大陆|中国大陆]]是[[有限法偿|法偿货币]]，但在[[香港|香港]]和[[澳门|澳门]]并不是。人民币的单位是[[元_(貨幣)|元]]、[[角_(貨幣)|角]]、[[分_(貨幣)|分]]；一元對十角對一百分。分已经基本不再使用，所以流通的硬币中主要为一角、五角、一元硬币。纸币的面额从一角到一百元，它们在大小和颜色上都有区别。元是人民币的主要单位，用来计量货币，是[[貨幣#價值尺度|货币的尺度]]。人民币一共发行了12种面额：1元、5元、10元、20元、50元、100元、200元、500元、1000元、5000元、10000元和50000元。这些面额一共有62种版别。1955年人民币以1:10000调整币值，第二套人民币发行了12种面额，包括1分、2分、5分、1角、2角、5角、1元、2元、3元、5元和10元。

目前最新的第五套人民币于1999年10月1日起发行。货币有8种面值：1角、5角、1元、5元、10元、20元、50元和100元。2004年，带有毛泽东头像的1元纸币开始发行。从1999年起，流通的硬币共有1分、2分、5 分、1角、5角和1元面值。

[[香港|香港]]和[[澳门|澳门]]有自己的货币政策和货币，[[港币|港币]]和[[澳门币|澳门币]]，它们与人民币并不等同。

====外汇兑换券====
{{main|外汇兑换券}}
大陆的[[中国银行|中国银行]]是特许的的主要外贸和外汇银行。在1979年到1994年间，外国游客到中国被要求使用中国银行发行的[[外汇兑换券|外汇兑换券]]进行交易。这种兑换券已经取消使用，取而代之的是人民币。
