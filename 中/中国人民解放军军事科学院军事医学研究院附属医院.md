'''中国人民解放军军事科学院军事医学研究院附属医院'''，位于[[北京市|北京市]][[丰台区|丰台区]][[东大街_(丰台)|东大街]]8号，隶属[[中国人民解放军军事科学院军事医学研究院|中国人民解放军军事科学院军事医学研究院]]，是[[三级甲等医院|三级甲等医院]]。

== 简介 ==
1957年8月，[[中华人民共和国卫生部|中华人民共和国卫生部]]筹建'''卫生部同位素医院'''，地址为位于北京市[[海淀区|海淀区]][[北太平路_(北京)|北太平路]]2号<ref name="lishiyange" />。该医院是[[苏联|苏联]]援建中国的第48个项目，故又俗称“48医院”<ref name=liuli/>。1958年12月10日，中华人民共和国卫生部、[[中国人民解放军总后勤部|中国人民解放军总后勤部]]联合发出通知，将该医院移交军队建制，转隶[[中国人民解放军军事医学科学院|中国人民解放军军事医学科学院]]，命名为'''中国人民解放军第三〇七医院'''。1981年9月，更名为'''中国人民解放军军事医学科学院附属医院'''，对外仍称中国人民解放军第三〇七医院。1983年6月，升为[[正师级|正师级]]单位，医院所属的处依次改为医务部、政治部、院务部。2005年9月，医院整体搬迁到[[北京市|北京市]][[丰台区|丰台区]][[东大街_(丰台)|东大街]]8号（原[[北京军医学院|北京军医学院]]营院）。2012年，医院通过[[三级甲等|三级甲等医院]]综合医院评审<ref name="lishiyange">[http://www.307hospital.com/lsyg.jhtml# 历史沿革]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}，中国人民解放军第三〇七医院网站，于2016-11-27查阅</ref>。

2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，其上级单位中国人民解放军军事医学科学院调整组建为[[中国人民解放军军事科学院军事医学研究院|中国人民解放军军事科学院军事医学研究院]]，该院更名为'''中国人民解放军军事科学院军事医学研究院附属医院'''。

中国人民解放军第三〇七医院是医疗、教学、科研相结合的三级甲等综合医院，是北京市首批医疗保险定点医疗机构，[[卫生部核事故医学应急中心|卫生部核事故医学应急中心]]第三临床部，[[中国疾病预防控制中心|中国疾病预防控制中心]]临床医院，国家反恐处突医学救援基地<ref>[http://www.307hospital.com/yyjjnew.jhtml#location 医院简介] {{webarchive|url=https://archive.is/20160117033408/http://www.307hospital.com/yyjjnew.jhtml |date=2016-01-17 }}，中国人民解放军第三〇七医院网站，于2016-11-27查阅</ref>。

2018年，[[中国人民解放军陆军总医院|原陆军总医院]]、[[中国人民解放军海军总医院|原海军总医院]]、[[中国人民解放军第三〇二医院|原第三〇二医院]]、[[中国人民解放军第三〇九医院|原第三〇九医院]]、军事科学院军事医学研究院原附属医院、[[中国人民武装警察部队总医院|原武警总医院]]均已由[[中国人民解放军总医院|中国人民解放军总医院]]暂管<ref>{{Cite web |url = https://www.thepaper.cn/newsDetail_forward_2334813 |title = 官方披露：原陆军总医院等单位已由解放军总医院暂管 |author =  |date =2018-08-10  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。现已合并入原第三〇二医院，并更名为'''中国人民解放军总医院第五医学中心'''<ref>{{Cite web |url = http://www.sohu.com/a/273558387_267160 |title = 军队医院更名继续，解放军总医院八个医学中心亮相 |author =  |date =2018-11-06  |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
