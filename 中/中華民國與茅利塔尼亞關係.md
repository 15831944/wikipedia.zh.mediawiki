'''中華民國與茅利塔尼亞關係'''是指[[中華民國|中華民國]]與[[茅利塔尼亞伊斯蘭共和國|茅利塔尼亞伊斯蘭共和國]]之間的關係。兩國於1960－1965年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對茅利塔尼亞的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-131-66cfa-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[茅利塔尼亞|茅利塔尼亞]]。]]
{{see also|中華民國外交|{{link-en|茅利塔尼亞外交|Foreign relations of Mauritania}}}}
1960年11月28日，茅利塔尼亞獨立當日與中華民國建立大使級[[外交#外交上的承認|外交關係]]。

1965年6月，於首都[[諾克少|諾克少]]設立'''中華民國駐茅利塔尼亞伊斯蘭共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1965年9月11日，與中華民國斷交。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境茅利塔尼亞。]]
[[File:Visa_requirements_for_Mauritanian_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|茅利塔尼亞簽證政策|Visa policy of Mauritania}}|{{link-en|茅利塔尼亞公民簽證要求|Visa requirements for Mauritanian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]前往茅利塔尼亞觀光或商務考察，可於抵達首都的[[諾克少國際機場|諾克少國際機場]]時，向[[海關|海關]]申請[[簽證#落地签证|落地簽證]]入境，停留最多30天並不得延長。若由[[黃熱病|黃熱病]]疫區國家入境，須出示黃熱病[[疫苗接種|疫苗接種]]證明（黃皮書）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-131-66cfa-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|茅利塔尼亞護照|Mauritanian passport}}的{{link-en|茅利塔尼亞人口|Demographics of Mauritania|茅利塔尼亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將茅利塔尼亞歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2017年4月22日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-196-dd307-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|茅利塔尼亞經濟|Economy of Mauritania}}}}
===貿易===
2017年，茅利塔尼亞是中華民國第165大貿易夥伴、第133大[[進口|進口]]夥伴、第171大[[出口贸易|出口]]夥伴。出口至茅利塔尼亞的金額為128萬6,470[[美元|美元]]，年增8.339%；自茅利塔尼亞進口的金額為355萬6,032美元，年減38.882%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）226萬9,562美元，年減50.990%。※2016年，茅利塔尼亞是中華民國第158大貿易夥伴、第123大進口夥伴、第169大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
