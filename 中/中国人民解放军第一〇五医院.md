'''中国人民解放军第一〇五医院'''，位于[[安徽省|安徽省]][[合肥市|合肥市]][[蜀山区|蜀山区]]长江西路424号，隶属[[无锡联勤保障中心|无锡联勤保障中心]]，为综合性[[三级甲等医院|三级甲等医院]]。

==简介==
1951年10月，由[[中国人民志愿军|中国人民志愿军]]后勤第二、三、四分部的第十、十一、十二、十三医院（中南军区），第十六、十七、十八医院（西南军区），第十九医院（华东军区），第二十兵团第一、二总医院（华北军区）抽组成立'''中国人民志愿军后勤第二基地医院'''，配属中国人民志愿军后勤部第二分部<ref name=ahszjsz/>。1952年12月组建于[[朝鲜战争|朝鲜战场]]<ref name=jj/>。1954年，院部并入中国人民志愿军后勤部第二分部卫生处（下属医院体制未动）。1956年改称'''中国人民志愿军第五一一医院'''<ref name=ahszjsz/>。1957年8月，与驻[[安徽省|安徽省]][[合肥市|合肥市]]的第一〇五医院对调换防并互换番号（第五一一医院改为'''中国人民解放军第一〇五医院'''；原第一〇五医院赴朝后改为中国人民志愿军第五一一医院）<ref name=ahszjsz>{{Cite book |url =http://60.166.6.242:8080/was40/index_sz.jsp?rootid=42818&channelid=43861  |title = 安徽省志·军事志 |author =安徽省地方志编纂委员会  |date = 1995年 |publisher =安徽人民出版社 |page=467 |language =  |accessdate =  |quote =  }}</ref><ref name=jj>{{Cite web |url = http://www.105yy.cn/yygk/ |title =医院简介  |author =  |date =  |publisher = 中国人民解放军第一〇五医院 |language =  |accessdate =2017-06-16  |quote =  }}</ref>。此后逐步发展成为集医疗、教学、科研、预防、保健为一体的综合性[[三级甲等医院|三级甲等医院]]<ref name=jj/>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第一〇五医院由[[中国人民解放军南京军区|中国人民解放军南京军区]]转隶[[无锡联勤保障中心|无锡联勤保障中心]]<ref>{{Cite web |url = http://jz.81.cn/n2014/tp/content_7586188.htm |title = 无锡联勤保障中心第105医院狠抓战备建设不放松 |author =  |date = 2017-05-03 |publisher =  军报记者|language =  |accessdate =  |quote =  }}</ref>。
