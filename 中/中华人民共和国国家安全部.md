'''中华人民共和国国家安全部'''（{{lang-en|Ministry of State Security of the People's Republic of China}}，[[缩写|缩写]]为 {{lang|en|MSS}}），标准简称'''安全部'''，亦简称'''国家安全部'''、'''国安部'''，是[[中华人民共和国国务院|中华人民共和国国务院]]主管[[国家安全|国家安全]]的[[国务院组成部门|组成部门]]<ref name="官方简称">{{cite web|url=http://www.haikou.gov.cn/xxgk/szfbjxxgk/zfgb/2008zfgb/2008nd12q/201103/t20110306_187419.html|title=关于转发国务院机构简称的通知|accessdate=2018-10-03|author=海口市档案局|publisher=海口市人民政府}}</ref>。

国家安全部垂直领导全国[[国家安全机关|国家安全机关]]。主要职能包括[[信息|情报]]收集分析、[[反情報|反间谍]]、[[政治保卫局|政治保卫]]等，是中国最大和最活跃的对外情报机构，也部分参与[[國內安全保衛|国内安全]]事务。1983年7月，原[[中共中央调查部|中共中央调查部]]整体、[[公安部国内安全保卫局|公安部政治保卫局]]部分、[[中共中央统一战线工作部|中共中央統戰部]]部分单位、[[中华人民共和国国防科学技术工业委员会|国防科工委]]部分单位合并成立国家安全部。国家安全部总部设在[[北京市|北京]]。

== 历史 ==
中华人民共和国情报系统源自中国共产党延安时期的“[[中共中央调查部|中央社会部]]”，由副部长[[李克农|李克农]]负责情报业务。中华人民共和国成立后，社会部分为政务院情报总署、军委作战部情报局、军委总情报部三个部分。

1950年12月10日，[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]总情报部成立，管理情报部、技术部、联络部三个部，归中央军委[[中国人民解放军总参谋部|总参谋长]]直接领导。1953年2月29日，军委总情报部撤销，各部划归[[中国人民解放军总参谋部|总参谋部]]建制。1955年6月20日，[[中央军事委员会|中共中央军委]]决定，将总参谋部联络部改归[[中国共产党中央书记处|中共中央书记处]]直接领导，定名为[[中共中央调查部|中共中央调查部]]，其部长依次为李克农、[[孔原|孔原]]和[[罗青长|罗青长]]，主管工作的是[[中华人民共和国国务院总理|国务院总理]][[周恩来|周恩来]]和[[中共中央办公厅|中共中央办公厅]]主任[[杨尚昆|杨尚昆]]。随着“[[文化大革命|文化大革命]]”爆发，中央调查部卷入派系斗争而一度被取消。

1975年，[[康生|康生]]的去世使得[[罗青长|罗青长]]重新接管中央调查部。1983年，经[[中共中央政法委员会|中共中央政法委员会]]秘书长、公安部部长[[刘复之|刘复之]]申请，中央决定以中共中央调查部整体、公安部政治保卫局以及中央统战部部分单位、国防科工委部分单位为基础，合并组建国家安全部<ref name="报道1118A">{{cite web||author1=搜狐网编辑 url=http://news.sohu.com/20131112/n390024538.shtml|title=资料:中华人民共和国国家安全部|accessdate=2018-11-17|date=2013-11-12|publisher=搜狐网}}</ref>。由[[凌云_(国家安全部部长)|凌云]]出任国家安全部首任部长<ref name="组织史资料F1">{{cite book|title=中国共产党组织史资料（附卷一）|last=中国共产党组织史资料编审委员会|accessdate=2018-12-27|year=1998|location=北京}}</ref>。

1983年7月，中共中央批准设立中国共产党国家安全部党组（简称安全部党组），凌云任国家安全部首任党组书记<ref name="组织史资料">{{cite book|title=中国共产党组织史资料（第七卷）|last=中国共产党组织史资料编审委员会|year=1998|location=北京}}</ref>。

1993年2月22日，[[第七届全国人民代表大会常务委员会|第七届全国人民代表大会常务委员会]]第三十次会议通过《[[中華人民共和國國家安全法_(1993年)|中华人民共和国国家安全法]]》<ref name="法规1"/>。该法第二条规定，“国家安全机关是本法规定的国家安全工作的主管机关<ref name="法规1">{{cite web|author1=中华人民共和国全国人民代表大会常务委员会 |url=http://www.law-lib.com/law/law_view.asp?id=9333|title=中华人民共和国国家安全法|accessdate=2018-11-17|author=全国人民代表大会常务委员会|publisher=法律图书馆}}</ref>。”1994年国务院常务会议通过《{{s|中华人民共和国国家安全法实施细则}}》，规定：“间谍组织和间谍组织代理人由中华人民共和国国家安全部确认”，“敌对组织由国家安全部或者公安部确认<ref name="法规1X">{{cite web|author1=中华人民共和国国务院 |url=http://www.mod.gov.cn/policy/2009-07/14/content_3100981.htm|title=中华人民共和国国家安全法实施细则|accessdate=2018-11-17|date=2009-07-14|publisher=中华人民共和国国防部}}</ref>。”

1989年7月，经中共中央批准，组建中国共产党国家安全部委员会（简称安全部党委），不再保留国家安全部党组<ref name="组织史资料" />。

2014年11月1日，[[第十二届全国人民代表大会常务委员会|第十二届全国人民代表大会常务委员会]]第十一次会议通过《[[中华人民共和国反间谍法|中华人民共和国反间谍法]]》，并废止1993年通过的《国家安全法》<ref name="法规2">{{cite web|author1=中华人民共和国全国人民代表大会常务委员会 |url=http://www.npc.gov.cn/npc/xinwen/2014-11/02/content_1884660.htm|title=中华人民共和国反间谍法|accessdate=2018-11-17|date=2014-11-02|publisher=中国人大网}}</ref>。《反间谍法》规定，“国家安全机关是反间谍工作的主管机关<ref name="法规2"/>。”2017年11月22日，国务院总理[[李克强|李克强]]签署第692号[[中华人民共和国国务院令|国务院令]]，公布《{{s|中华人民共和国反间谍法实施细则}}》，细则规定：“间谍组织和间谍组织代理人由国务院国家安全主管部门确认”，“敌对组织由国务院国家安全主管部门或者国务院公安部门确认<ref name="法规2X">{{cite web|author1=中华人民共和国国务院 |url=http://www.gov.cn/zhengce/content/2017-12/06/content_5244819.htm|title=中华人民共和国反间谍法实施细则|accessdate=2018-11-17|date=2017-12-06|publisher=中国政府网}}</ref>。”

2017年6月27日，[[第十二届全国人民代表大会常务委员会|第十二届全国人民代表大会常务委员会]]第二十八次会议通过《{{s|中华人民共和国国家情报法}}》<ref name="法规3"/>。《国家情报法》规定，“[[中央国家安全委员会|中央国家安全领导机构]]对国家情报工作实行统一领导”，“国家安全机关和公安机关情报机构、军队情报机构按照职责分工，相互配合，做好情报工作、开展情报行动<ref name="法规3">{{cite web|author1=中华人民共和国全国人民代表大会常务委员会 |url=http://www.npc.gov.cn/npc/xinwen/2017-06/27/content_2024529.htm|title=中华人民共和国国家情报法|accessdate=2019-03-18|date=2017-
06-27|publisher=中国人大网}}</ref>。”

2018年4月15日，国家安全部宣布开通互联网举报受理平台网站，接受组织和个人的举报信息<ref name="报道1118B">{{cite web|author1=澎湃新闻编辑 |url=https://www.thepaper.cn/newsDetail_forward_2077508|title=国家安全部开通互联网举报受理平台，接受组织和个人举报信息|accessdate=2018-11-17|date=2018-04-15|publisher=澎湃新闻}}</ref>。

== 职权 ==
根据《{{s|中华人民共和国国家情报法}}》，国家安全机关在国家情报工作中行使下列职权<ref name="法规3"/>：
{{columns-list|2|
# 国家情报工作机构根据工作需要，依法使用必要的方式、手段和渠道，在境内外开展情报工作。
# 国家情报工作机构应当依法搜集和处理境外机构、组织、个人实施或者指使、资助他人实施的，或者境内外机构、组织、个人相勾结实施的危害中华人民共和国国家安全和利益行为的相关情报，为防范、制止和惩治上述行为提供情报依据或者参考。
# 国家情报工作机构可以按照国家有关规定，与有关个人和组织建立合作关系，委托开展相关工作。
# 国家情报工作机构可以按照国家有关规定，开展对外交流与合作。
# 国家情报工作机构依法开展情报工作，可以要求有关机关、组织和公民提供必要的支持、协助和配合。
# 国家情报工作机构根据工作需要，按照国家有关规定，经过严格的批准手续，可以采取技术侦察措施和身份保护措施。
# 国家情报工作机构工作人员依法执行任务时，按照国家有关规定，经过批准，出示相应证件，可以进入限制进入的有关区域、场所，可以向有关机关、组织和个人了解、询问有关情况，可以查阅或者调取有关的档案、资料、物品。
# 国家情报工作机构工作人员因执行紧急任务需要，经出示相应证件，可以享受通行便利。国家情报工作机构工作人员根据工作需要，按照国家有关规定，可以优先使用或者依法征用有关机关、组织和个人的交通工具、通信工具、场地和建筑物，必要时，可以设置相关工作场所和设备、设施，任务完成后应当及时归还或者恢复原状，并依照规定支付相应费用；造成损失的，应当补偿。
# 国家情报工作机构根据工作需要，按照国家有关规定，可以提请海关、出入境边防检查等机关提供免检等便利。
# 国家情报工作机构及其工作人员应当严格依法办事，不得超越职权、滥用职权，不得侵犯公民和组织的合法权益，不得利用职务便利为自己或者他人谋取私利，不得泄露国家秘密、商业秘密和个人信息。
}}

== 争议事件 ==
1985年，中華人民共和國国家安全部某局局长（准确职务不明）[[俞强声|俞强声]]叛逃美国，导致潜伏在[[美国中央情报局|美国中央情报局]]的中国元老级间谍[[金无怠|金无怠]]被捕，中美两国爆发间谍战<ref>{{cite news|url=http://news.creaders.net/headline/newsViewer.php?nid=529018&id=1181682&dcid=33|title=政法委书记灰头土脸检讨 国安部长黯然下台！|date=2012-08-24|work=外参|accessdate=2012-08-25}}</ref>，国安部部长凌云被撤职。

2009年，曾在中華人民共和國國家安全部工作的[[特務|特工]][[李鳳智|李鳳智]]在美國[[华盛顿哥伦比亚特区|华盛顿]]提出[[庇護權|政治庇護]]申請<ref>{{cite news|url=http://news.bbc.co.uk/chinese/trad/hi/newsid_7950000/newsid_7954100/7954192.stm|title=叛逃中國“間諜”促美國對華施壓|date=2009-03-20|publisher=BBC中文网|accessdate=2014-01-25}}</ref>。

2012年，中華人民共和國国家安全部副部长[[陆忠伟|陆忠伟]]的下属因为向美国传递有关中国“政治、经济以及战略领域”的情报而被逮捕<ref>{{cite news|url=http://www.voachinese.com/content/china-spy-reuters/1211679.html|title=胡锦涛下令严查国安部＂间谍门＂|author=燕青|date=2012-06-15|publisher=[[美国之音|美国之音]]|accessdate=2012-06-21}}</ref>。

香港《内幕》杂志称，国家安全部通常以[[新華社|新華社]]、[[中国人民对外友好协会|中国人民对外友好协会]]、[[中国国际交流协会|中国国际交流协会]]、[[中国国际友谊促进会|中国国际友谊促进会]]、[[中国国际人才交流协会|中国国际人才交流协会]]、[[中国国际旅行社|中国国际旅行社]]、[[中华贸易公司|中华贸易公司]]、[[深圳振华宾馆|深圳振华宾馆]]作为其掩护机构<ref>{{cite book|title=习总手握十三大权杖|author=《内幕》第30期|isbn=1-6303-2159-1}}</ref>。
