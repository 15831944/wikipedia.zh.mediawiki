'''中華民國與蓋亞那關係'''是指[[中華民國|中華民國]]與[[蓋亞那共和國|蓋亞那共和國]]之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對蓋亞那的相關事務由'''中華民國駐[[聖文森及格瑞那丁|聖文森國]][[中華民國駐外機構列表|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-261-69055-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
{{see also|中華民國外交|{{link-en|蓋亞那外交|Foreign relations of Guyana}}}}
1970年8月24日，簽署《[[中華民國政府|中華民國政府]]與{{link-en|蓋亞那政府|Politics of Guyana|蓋亞那合作共和國政府}}間[[農業技術|農業技術]]合作協定》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e8%93%8b%e4%ba%9e%e9%82%a3&tycountry_e=%e8%93%8b%e4%ba%9e%e9%82%a3&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

2008年5月，[[蓋亞那|蓋亞那]]、[[巴基斯坦|巴基斯坦]]與[[中華人民共和國|中華人民共和國]]在[[世界衛生組織|世界衛生組織]]（WHO）的[[世界衛生大會|世界衛生大會]]（WHA）發言反對將[[中華民國外交#中华民国的邦交国|中華民國的邦交國]]向秘書處提出名為「邀請[[臺灣|臺灣]]以[[世界衛生大會觀察員|觀察員]]身份參與世界衛生大會」（Inviting Taiwan to participate in the World Health Assembly as an observer）的提案納入議程。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=E50FCF98DE71F5BB&sms=8C1B5DF8A167A082&s=AAAB437F2374D7AD |author = 公眾外交協調會 |date = 2011年8月10日 |title = 中國大陸阻撓我國際空間事例 |publisher = 中華民國外交部}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|蓋亞那簽證政策|蓋亞那公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]須前往{{link-en|蓋亞那駐外機構列表|List of diplomatic missions of Guyana|蓋亞那駐外館處}}或郵寄至蓋亞那國安部申辦觀光或商務簽證。經蓋亞那轉機，亦須申請簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-261-69055-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|蓋亞那護照|Guyanese passport}}的{{link-en|蓋亞那人口|Demographics of Guyana|蓋亞那公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將蓋亞那歸類為'''灰色警示'''：提醒注意（2017年5月1日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-359-06b90-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將蓋亞那歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=4E79822E96B2FFE9 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|蓋亞那經濟|Economy of Guyana}}}}
===貿易===
2012年10月18日，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）邀請16家[[美國|美國]]、[[墨西哥|墨西哥]]與[[蓋亞那|蓋亞那]]廠商前往[[臺灣|臺灣]]參加「反[[傾銷|傾銷]]商機說明會暨採購媒合會」。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=594968&url=/NewsSearch.asp |author = 詹惟甯 |date = 2012年10月10日 |title = 重量級美墨買主來台採購反傾銷商機說明會暨採購媒合會 |publisher = 經貿透視網}}</ref>

2018年，蓋亞那是中華民國第152大貿易夥伴、第142大[[進口|進口]]夥伴、第147大[[出口贸易|出口]]夥伴。出口至蓋亞那的金額為463萬2,125[[美元|美元]]，年增32.567%；自蓋亞那進口的金額為242萬1,039美元，年增28.834%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）221萬1,086美元，年增36.911%。2017年，蓋亞那是中華民國第163大貿易夥伴、第145大進口夥伴、第151大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
