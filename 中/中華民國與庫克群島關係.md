{{Infobox Bilateral relations|中華民國與庫克群島|ROC|COK|filetype=png}}

'''中華民國與庫克群島關係'''是指[[中華民國|中華民國]]與[[庫克群島|庫克群島]]之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對庫克群島的相關事務由'''{{link-en|駐紐西蘭台北經濟文化辦事處|Taipei Economic and Cultural Office in New Zealand}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-184-32281-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|庫克群島經濟|Economy of the Cook Islands}}}}
===貿易===
2018年，庫克群島是中華民國第214大貿易夥伴、第214大[[進口|進口]]夥伴、第208大[[出口贸易|出口]]夥伴，在[[大洋洲國家和地區列表|大洋洲國家]]排名倒數第3、2、3；在[[世界政區索引|世界經濟體]]倒數第31、27、29。出口至庫克群島的金額為14萬1,249[[美元|美元]]，年減31.563%；自庫克群島進口的金額為1萬5,353美元，年減66.116%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）12萬5,896美元，年減21.844%。2017年，庫克群島是中華民國第209大貿易夥伴、第199大進口夥伴、第200大出口夥伴，在大洋洲國家排名倒數第3、6、2；在世界經濟體倒數第30、38、31。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===觀光===
2018年8月29日，[[中國國民黨|中國國民黨]]前發言人、[[不當黨產處理委員會|不當黨產處理委員會]]委員[[楊偉中|楊偉中]]和家人前往庫克群島度假，在{{link-en|愛圖塔基島|Aitutaki}}與女兒進行[[泛舟|泛舟]]活動時發生意外，楊偉中為了救泛舟溺水的女兒過程出事，雖然經過搶救，最後仍不治。庫克群島警方表示楊偉中的遺體已從愛圖塔基島移至[[拉羅湯加島|拉羅湯加島]]，由[[法醫|法醫]]相驗，警方並與{{link-en|駐紐西蘭臺北經濟文化辦事處|Taipei Economic and Cultural Office in New Zealand|台灣駐紐西蘭辦事處}}聯絡後續事宜。<ref>{{Cite web |url = https://udn.com/news/story/6813/3342992 |author = 范振光 |date = 2018年9月1日 |title = 楊偉中證實救女溺斃 庫克群島警方提醒水上安全措施 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref>9月9日，遺體經由紐西蘭奧克蘭運抵台灣桃園。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/2545260 |author = 綜合報導 |date = 2018年9月8日 |title = 楊偉中遺體要回家了！明天清晨可抵桃園機場 |publisher = [[自由時報|自由時報]]}}</ref>

===駕車===
持有[[中華民國汽車駕駛執照|中華民國汽車駕駛執照]]必須取得英文譯本才可在庫克群島駕車；若在庫克群島騎[[機車_(公路)|機車]]，則須在首都通過筆試與路考，並付清相關費用，以取得庫克群島駕駛執照。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-184-32281-05-1.html |title = 生活資訊 |publisher = 外交部領事事務局}}</ref>
