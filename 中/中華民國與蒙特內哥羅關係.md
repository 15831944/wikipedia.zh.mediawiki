'''中華民國與蒙特內哥羅關係'''是指[[中華民國|中華民國]]與[[蒙特內哥羅|蒙特內哥羅]]（[[音譯|音譯]]'''-{zh-hans:蒙特内哥罗; zh-hant:蒙特內哥羅}-'''；[[意譯|意譯]]'''-{zh-hans:黑山; zh-hant:黑山}-'''）之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對蒙特內哥羅的相關事務由'''駐[[匈牙利|匈牙利]][[台北经济文化代表处|台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-203-b44bd-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|蒙特內哥羅簽證政策|Visa policy of Montenegro}}|{{link-en|蒙特內哥羅公民簽證要求|Visa requirements for Montenegrin citizens}}}}
持有載明[[中華民國國民身份證|身份證]]字號的[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境蒙特內哥羅，停留最多90天。<ref name="簽證及入境須知">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-203-b44bd-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|蒙特內哥羅護照|Montenegrin passport}}的[[蒙特內哥羅人口|蒙特內哥羅公民]]則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將蒙特內哥羅歸類為'''灰色警示'''：提醒注意（2017年4月17日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-356-8f863-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|蒙特內哥羅經濟}}
===貿易===
2017年，蒙特內哥羅是中華民國第170大貿易夥伴、第215大[[進口|進口]]夥伴、第152大[[出口贸易|出口]]夥伴，在[[歐洲國家和地區列表|歐洲國家]]排名倒數第5、2、7；在[[世界政區索引|世界經濟體]]倒數第69、22、79。出口至蒙特內哥羅的金額為340萬4,765[[美元|美元]]，年減35.868%；自蒙特內哥羅進口的金額為1萬1,157美元，年減99.536%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）339萬3,608美元，年增16.763%。※2016年，蒙特內哥羅是中華民國第151大貿易夥伴、第139大進口夥伴、第141大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
