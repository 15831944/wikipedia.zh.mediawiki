'''中華民國與馬來西亞關係'''（{{lang-ms|Hubungan Malaysia-Taiwan}}）是指[[中華民國|中華民國]]與[[馬來西亞聯邦|馬來西亞聯邦]]（'''馬來西亞'''、'''大馬'''）之間的關係。兩國於1964－1974年有領事級[[外交#外交上的承認|外交關係]]，終止關係後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

中華民國是僅次於[[日本|日本]]、[[美國|美國]]以及[[新加坡|新加坡]]，馬來西亞第四大的[[外商直接投資|外資來源國]]，累計投資金額達到124.17億[[美元|美元]]<ref name="駐館與駐地關係"/>。另一方面，大馬為臺灣「正式修讀學位外籍留學生」最大來源國，這源自於中華民國長期、穩定的「僑教政策」，作為僑務外交策略一環，中華民國將海外[[華裔|華裔]]學生視為[[僑生|僑生]]（僑胞）予以特殊優惠待遇，鼓勵全球華人到中華民國留學，而[[馬來西亞華人|大馬數百萬計的華人]]正是中華民國歡迎的對象<ref>「2017年Contact TAIWAN攬才共識營」鏈結在臺僑外生，協助企業找人才 http://info.taiwantrade.com/CH/bizsearchdetail/8455958/C</ref>。

1974年5月31日，[[馬來西亞政府|馬來西亞政府]]與[[中華民國政府|中華民國政府]]終止領事關係，並與[[中華人民共和國政府|中華人民共和國政府]]建立外交關係。<ref name="駐館與駐地關係"/>

1974年8月3日，中華民國設立'''駐吉隆坡遠東貿易旅遊中心'''，1988年升格為[[實質大使館|具大使館性質]]的'''駐馬來西亞台北經濟文化中心'''，駐館人員享有若干[[外交豁免權|外交豁免權]]，1992年7月13日改為現名：'''[[駐馬來西亞臺北經濟文化辦事處|駐馬來西亞臺北經濟文化辦事處]]'''（{{lang-ms|Pejabat Ekonomi dan Kebudayaan Taipei}}）。<ref name="駐館與駐地關係"/><ref name="駐外館處">{{Cite web |url = http://www.mofa.gov.tw/OverseasOffice_Content.aspx?n=6357834932B83C83&sms=5A961ED2E4BA25E8&s=07880B28C8E3EAEA&os=D0322AD5E2C38096 |title = 駐外館處 |publisher = 中華民國外交部}}</ref><ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/06.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

馬來西亞於1977年開始派移民官駐中華民國，並以'''[[馬來西亞航空|馬來西亞航空公司]]台北分公司'''名義辦理[[簽證|簽證]]業務與其他事項。1982年於台北成立'''馬航行政部'''，1983年增設'''文化貿易交流中心'''。1987年將原本的行政、商務機構合併更名為'''[[馬來西亞友誼及貿易中心|馬來西亞友誼及貿易中心]]'''（馬來語：{{lang|ms|Persahabatan Malaysia dan Pusat Perdagangan, Taipei}}）。1989年11月，於該中心之下增設'''馬來西亞旅遊資訊中心'''，以推展各項業務。<ref name="駐館與駐地關係"/><ref>[http://www.kln.gov.my/web/twn_taipei/home 馬來西亞友誼及貿易中心]</ref><ref name="駐華外國機構">{{Cite web |url = http://www.mofa.gov.tw/OfficesInROC_Content.aspx?n=8CEB2B5F5436B997&sms=8EBFADC1592C7BFE&s=07880B28C8E3EAEA&os=9BB844180349DC88 |title = 駐華外國機構 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將馬來西亞按地區歸類為'''灰色警示'''：提醒注意，與'''橙色警戒'''：避免非必要旅行（以上截至2017年12月5日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-253-95de0-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將馬來西亞歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=919502C2C9A44B19 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於[[馬來西亞|馬來西亞]]首都[[吉隆坡|吉隆坡]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#ASIA |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]也在吉隆坡設立'''駐馬來西亞代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=214&pid=305862 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

1997年5月，首屆台馬經貿部長級會議在[[台北市|台北]]舉行。<ref name="86年外交年鑑"/>

2012年，中華民國是馬來西亞第8大貿易夥伴（次於[[中华人民共和国|中國大陸]]、[[新加坡|新加坡]]、[[日本|日本]]、[[美國|美國]]、[[泰國|泰國]]、[[印尼|印尼]]與[[韓國|韓國]]）、第7大供應來源與第11大出口市場。<ref name="駐館與駐地關係">{{Cite web |url = http://www.roc-taiwan.org/my/post/3505.html |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref>

2018年，馬來西亞是中華民國第7大貿易夥伴、第7大[[進口|進口]]夥伴、第8大[[出口贸易|出口]]夥伴。出口至馬來西亞的金額為106億0,258萬7,943[[美元|美元]]，年增2.252%；自馬來西亞進口的金額為92億4,558萬4,372美元，年增28.731%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）13億5,700萬3,571美元，年減57.420%。2017年，馬來西亞是中華民國第7大貿易夥伴、第8大進口夥伴、第8大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資===
馬來西亞於1987年開始放寬[[外商直接投資|外人投資]]並採取獎勵措施後，[[臺灣商人|臺商]]投資案件明顯增加，並於1990年代初達到高峰，1990年及1994年是馬來西亞最大外資來源國。之後中國大陸、[[越南|越南]]與印尼等競相爭取外資，同時馬來西亞又面臨勞工短缺、工資上漲等問題，因此1990年代後期的投資便逐漸衰退。<ref name="駐館與駐地關係"/>

至2017年12月底，臺商的總投資金額為124.17億美元，次於日本、美國、新加坡，是第4大外資來源國。2013年，在馬來西亞的投資案件有18件，投資金額為3,995萬美元，位居馬來西亞第16大投資國。投資較多的行業依序為[[化學品列表|化學及其製品]]、[[非金屬礦產|非金屬礦]]產品、[[電子產品|電機電子]]、基本金屬、食品、[[非鐵金屬|非鐵金屬]]及交通設備等。<ref name="駐館與駐地關係"/><ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/my/pdf/103-17Malaysia%20.pdf |title = 投資環境簡介 |publisher = 經濟部全球臺商服務網}}</ref>

===經貿問題===
{{see also|2014年台灣劣質油品事件}}
2012年6月26日起，馬來西亞政府對從[[中華民國|中華民國]]、[[韓國|韓國]]、[[印尼|印尼]]、[[中華人民共和國|中華人民共和國]]及[[土耳其|土耳其]]進口的鋼線材（Steel Wire Rods）產品啟動反傾銷調查，{{link-en|馬來西亞貿易工業部|Ministry of International Trade and Industry (Malaysia)}}（貿工部）決定自2013年2月20日起，對鋼線材的進口課徵[[反傾銷稅|反傾銷稅]]3.03%－25.2%，為期5年，以保護國內鋼鐵產業。[[中國鋼鐵|中國鋼鐵公司]]（中鋼）的鋼線材產品將被課徵反傾銷稅10.98%（原進口稅為5%），豐興鋼鐵公司被課徵9.04%稅率，其他公司則課徵25.2%稅率。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref>

2012年7月27日起，馬來西亞對從中華民國進口的[[轧制|熱軋]]鋼產品啟動反傾銷調查，經濟部國際貿易局（國貿局）駐馬來西亞代表處經濟組接洽馬來西亞貿工部及提案廠商說明，促使熱軋鋼提告廠商－美佳鋼鐵於11月9日正式函告馬來西亞貿工部撤銷對其控告案，並終止調查。<ref name="與我國經貿關係"/>

2012年7月27日起，馬來西亞對從中華民國、[[泰國|泰國]]、中華人民共和國、印尼及[[越南|越南]]進口的雙向延伸[[聚丙烯|聚丙烯]]薄膜（BOPP）啟動反傾銷調查，國貿局駐馬來西亞代表處經濟組爭取參與調查及調降反傾銷稅率。根據馬來西亞貿工部於2013年3月18日發布的事實揭露報告（NOEF），中華民國BOPP出口廠商的反傾銷稅率已自初判報告的20.42%降為12.37%。<ref name="與我國經貿關係"/>

2014年10月起，馬來西亞對從中華民國[[衛生福利部食品藥物管理署|衛生福利部食品藥物管理署]]（食藥署）網站所列涉案廠商進口含[[豬油|豬油]]食品執行進口管制措施。國貿局駐馬來西亞代表處經濟組經過逾2個月的爭取與說明受污染產品已全面回收銷毀，並已更換油品配方重新上架，食品安全無虞，同年12月23日獲得[[馬來西亞衛生部|馬來西亞衛生部]]通知，同意重新開放涉案食品進口。<ref name="與我國經貿關係"/>

===臺商組織===

===觀光===
台灣遊客赴馬來西亞的人數近年來約在20萬人次，於2000年首度達到30萬人次。但2003年因受[[嚴重急性呼吸道症候群|SARS]]、[[伊拉克戰爭|伊拉克戰爭]]等影響，驟降至12萬人次。之後旅客人數逐漸回升，至2017年為33萬人次；而前來台灣的馬來西亞旅客也逐年增加，2000年僅為5萬8千餘人次，2011年突破30萬人次，至2017年已達到約53萬人次。<ref name="駐館與駐地關係"/><ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>為發展觀光，台灣觀光協會在吉隆坡設有辦事處。<ref name="投資環境簡介"/>

===教育===
[[檳吉臺灣學校|檳吉臺灣學校]]，簡稱檳吉臺校，前身為檳城臺灣僑校，由[[英業達集團|英業達集團]]創辦人[[溫世仁|溫世仁]]於1991年成立，為[[海外臺灣學校|海外第一所臺灣學校]]。校名「檳吉」取自「[[檳城州|檳城州]]」、「[[吉打州|吉打州]]」兩州州名的首字結合而成。

[[吉隆坡臺灣學校|吉隆坡臺灣學校]]，原名吉隆坡中華台北學校，是馬來西亞[[台商|台商]]總會於1991年授權中馬區聯誼會籌辦，成立目的是為解決台商子女教育問題。

中華民國[[高等教育|高等教育]]評鑑中心與{{link-en|馬來西亞高等教育部|Ministry of Higher Education (Malaysia)}}學術資格鑑定機構（MQA）於2012年7月20日簽署《雙方高教評鑑品質互信聲明》，MQA並在其網站公布中華民國157所受承認的校院，兩國的相關機構已相互承認彼此高等教育學歷。<ref name="駐館與駐地關係"/>

2017年，馬來西亞在台灣的[[留學生|留學生]]有1萬7千餘人，歷年留臺校友總人數超過6萬人，還設立了「馬來西亞留台校友會聯合總會」；同年，台灣在馬來西亞的留學生約80人。<ref name="106年外交年鑑"/><ref name="投資環境簡介"/>

===航空===
1990年10月22日，簽署《台馬交換航權協定》，雙方可指定多家航空公司飛行台馬航線。<ref name="駐館與駐地關係"/>

1995年9月5日，在[[吉隆坡|吉隆坡]]簽署《台馬航權協定附約》，新增至23個總係數點，以因應空運市場需要。<ref name="駐館與駐地關係"/>

1997年10月7日，在[[台北市|台北]]簽署《台馬交換航權協定附約二號》，規定雙方航線完全自由化，兩國航空公司可自本國任何點，飛往對方境內任何點；洲際延遠[[航權#中间点权或延远权|第五航權]]班次也逐年遞增。<ref name="駐館與駐地關係"/>
