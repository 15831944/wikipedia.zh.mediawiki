'''中山七里 '''（なかやま しちり、1961年 - ）是一位[[日本|日本]][[小說家|小說家]]、[[推理作家|推理作家]]，出生於[[岐阜縣|岐阜縣]]<ref>{{Cite news| title = 小説家・推理作家の中山七里氏 岐阜が舞台の作品書く 県出身作家の活躍、秘密語る| newspaper = [[岐阜新聞|岐阜新聞Web]]| date = 2013-01-24| url = http://www.gifu-np.co.jp/tokusyu/2013/seikon/sei20130124.shtml| accessdate = 2014-12-12| deadurl = yes| archiveurl = https://web.archive.org/web/20141215091439/http://www.gifu-np.co.jp/tokusyu/2013/seikon/sei20130124.shtml| archivedate = 2014-12-15}}</ref><ref name=sony>{{Cite web |url = http://ebookstore.sony.jp/stc/special/author/nakayama/ |title = 『スタート!』インタビュー｜中山七里さん「映画を作るつもりで、この本を書きました」 |publisher = [[リーダーストア|リーダーストア]] |accessdate = 2013-02-19 |deadurl = yes |archiveurl = https://web.archive.org/web/20130102220615/http://ebookstore.sony.jp/stc/special/author/nakayama/ |archivedate = 2013-01-02 }}</ref>。

== 經歷 ==
1961年、中山七里出生於岐阜縣<ref name=inpocket201311>{{Cite |title = 読者を挑発する新社会派 中山七里スペシャルインタビュー |date = 2013-11|publisher = [[講談社|講談社]]  |author = [[IN★POCKET|IN★POCKET]] |isbn = 9784060607133 |pages = 172-191 |ref = harv}}</ref>。幼稚園時期經常閱讀書籍<ref name=inpocket201311 />，小學生時期接觸[[英國|英國]]作家[[阿瑟·柯南·道爾|阿瑟·柯南·道爾]]《[[福爾摩斯探案|福爾摩斯探案]]》、[[法國|法國]]作家[[莫理斯·盧布朗|莫理斯·盧布朗]][[亞森·羅賓|亞森·羅賓]]系列。中學生時期接觸[[阿嘉莎·克莉絲蒂|阿嘉莎·克莉絲蒂]]與[[艾勒里·昆恩|艾勒里·昆恩]]作品<ref name="subaru201411">{{Cite journal|author = 中山七里 |date = 2014-10-17 |title = インタビュー 「ミステリは最高のエンタテイメントなんです」|journal = [[小説すばる|小説すばる]] |issue = 2014年11月号|pages = 80-83 |publisher = [[集英社|集英社]]}}</ref>。1970年代、中山七里觀看電影『[[犬神家一族|犬神家一族]]』，開始對[[橫溝正史|橫溝正史]]、[[江戶川亂歩|江戶川亂歩]]產生興趣<ref name=inpocket201311 />。中山七里於高中時代開始創作<ref name=inpocket201311 />，並參加小說新人賞<ref name=sony/>、大學時代、中山七里以「謝罪」入選[[江戶川亂步獎|江戶川亂步獎]]<ref name=inpocket201311 />。

中山七里畢業於[[花園大學|花園大學]][[文學部|文學部]]國文科<ref name=honyaclub>{{Cite web| title = 2013年12月号 創作の現場 中山七里| publisher = [[日本出版販売|Honya Club.com]]| url = http://www.honyaclub.com/shop/contents2/tenbo28_sousaku.aspx| accessdate = 2014-01-17}}</ref>，之後停止創作。

2006年、中山七里參加[[島田莊司|島田莊司]]見面會後、興起創作小說的念頭<ref name=otonafami201312>{{Cite |title = 音楽ミステリーの名手が放つ家族と記憶、その罪の物語 |date = 2013-12|publisher = [[KADOKAWA|KADOKAWA]]  |author = [[オトナファミ|オトナファミ]] |isbn = 4910022631235 |page = 10 |ref = harv}}</ref>。中山七里以『[[魔女甦醒|魔女甦醒]]』參加[[「這本推理小說了不起！」大獎|「這本推理小說了不起！」大獎]]、最終審査落選。2009年、『[[再見，德布西|再見，德布西]]』獲得第8回「這本推理小說了不起！」大獎<ref name=sony/>、48歳以小説家出道<ref name=davinch201301>{{Cite |title = めくるめく仕掛けに喝采必至! 中山七里版"キネマの天地" |date = 2013-01|publisher = [[メディアファクトリー|メディアファクトリー]]  |author = [[ダ・ヴィンチ|ダ・ヴィンチ]] |volume = 225 |isbn = 4910059470135 |page = 70 |ref = harv}}</ref>。

==家族==
中山七里妻子是[[電子琴|電子琴]]教師<ref name=rakuten>{{Cite web| author = 宇田夏苗| title = 著者インタビュー － 中山七里『さよならドビュッシー』| publisher = [[楽天ブックス|楽天ブックス]]| date = 2013-02-07| url = http://books.rakuten.co.jp/event/book/interview/20130207-shichiri-nakayama/| accessdate = 2013-08-17}}</ref><ref name=honyaclub />，目前育有一子。
