'''中華民國與委內瑞拉關係'''是指[[中華民國|中華民國]]與[[委內瑞拉玻利瓦爾共和國|委內瑞拉玻利瓦爾共和國]]之間的關係。1941－1974年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，中華民國在首都[[卡拉卡斯|卡拉卡斯]]設立[[實質大使館|具大使館性質]]的代表機構，但於2007年被迫關閉。且自2000年[[烏戈·查韋斯|查維茲政權]]建立後，便多次於公開場合聲明委內瑞拉拒絕承認中華民國的[[合法性|合法性]]。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[委內瑞拉|委內瑞拉]]。]]
{{see also|中華民國外交|{{link-en|委內瑞拉外交|Foreign relations of Venezuela}}}}
{{see also|中国驻委内瑞拉大使列表{{!}}中華民國駐委內瑞拉外交官列表|委内瑞拉驻华大使列表{{!}}委內瑞拉駐中華民國外交官列表}}
1941年4月9日，中華民國開始派駐委內瑞拉[[特命全權公使|公使]]（由[[中国驻古巴大使列表|駐古巴公使]]兼任）。<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2007/html/08-1-6.htm |title = 《中華民國96年外交年鑑》〈第二章 對外關係〉 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>

1944年8月4日，於委內瑞拉首都[[卡拉卡斯|卡拉卡斯]]設立'''中華民國駐委內瑞拉共和國[[外交代表機構#公使馆|公使館]]'''；於中華民國[[中國抗日戰爭|抗日戰爭]]時期首都[[重慶|重慶]]設立'''委內瑞拉共和國駐中華民國公使館'''。兩國並互派公使。<ref name="外交年鑑"/>

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，委內瑞拉駐華公使館也搬遷至[[臺北市|臺北]]。<ref>''Taipei Economic and Cultural Representative Office in Venezuela'' [http://www.roc-taiwan.org.ve/about.htm   Relaciones entre la República de China y la República de Venezuela] {{webarchive|url=https://web.archive.org/web/20071213040557/http://www.roc-taiwan.org.ve/about.htm |date=2007-12-13 }} Accessed 18 December 2007 {{es icon}}</ref>

1966年7月1日，兩國公使館同時升格為'''[[外交代表機構#大使馆|大使館]]'''。<ref name="外交年鑑"/>2月，委內瑞拉先行派駐[[特命全權大使|大使]]，10月，中華民國亦派駐大使。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1974年6月28日，委內瑞拉與[[中華人民共和國|中華人民共和國]][[外交#外交上的承認|建交]]，29日，中華民國與委內瑞拉斷交。8月1日，大使館關閉，兩國關係降至非官方的代表機構層級。<ref> ''Taipei Economic and Cultural Representative Office in Venezuela'' [http://www.roc-taiwan.org.ve/about.htm   Relaciones entre la República de China y la República de Venezuela] {{webarchive|url=https://web.archive.org/web/20071213040557/http://www.roc-taiwan.org.ve/about.htm |date=2007-12-13 }} Accessed 18 December 2007 {{es icon}}</ref><ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>同年8月，於首都卡拉卡斯設立[[實質大使館|具大使館性質]]的'''駐委內瑞拉台灣商務辦事處'''。1992年6月22日，更名為'''駐委內瑞拉[[台北經濟文化辦事處|台北經濟文化辦事處]]'''。<ref name="外交年鑑"/>

1988年1月15日，簽署《中華民國與委內瑞拉（共和國）間國際[[特快专递|快捷郵件]]服務協定》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e5%a7%94%e5%85%a7%e7%91%9e%e6%8b%89&tycountry_e=%e5%a7%94%e5%85%a7%e7%91%9e%e6%8b%89&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

1991年4月22日，委內瑞拉前總統{{link-en|艾雷拉|Luis Herrera Campins}}夫婦抵台訪問。期間會見立法院長[[梁肅戎|梁肅戎]]、外交部長[[錢復|錢復]]、經濟部長[[蕭萬長|蕭萬長]]、[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]副董事長武冠雄等。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=B3A67AD6ADE8CB43 |author = 公眾外交協調會 |date = 1991年4月21日 |title = 委內瑞拉共和國前總統艾雷拉伉儷由該國前國家投資委員會主席 |publisher = 中華民國外交部}}</ref>

2002年，[[委內瑞拉全國代表大會|委內瑞拉全國代表大會]]外交委員會主席{{link-es|孟多亞|Julio Montoya}}等人抵台訪問。<ref name="91年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__06.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2006年，{{link-en|委內瑞拉中央大學|Central University of Venezuela}}校長巴瑞斯（Antonio París Pantalon）抵台訪問。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2006/html/08-1-6.htm |title = 《中華民國95年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2019年1月，委內瑞拉[[2019年委內瑞拉總統地位危機|發生總統雙胞危機]]後，2月14日，中華民國駐美國代表[[高碩泰|高碩泰]]做為唯一受邀的[[亚洲国家和地区列表|亞洲]]代表，出席在美國[[华盛顿哥伦比亚特区|華盛頓]]舉行的「委內瑞拉人道危機全球會議（Global Conference on the Humanitarian Crisis in Venezuela）」時，代表[[中華民國政府|中華民國政府]]提供價值50萬[[美元|美元]]的[[人道援助|人道援助]]予委內瑞拉。委內瑞拉反對派高層並在[[Twitter|Twitter]]發文感謝台灣政府的捐款。<ref>{{Cite web |url = https://udn.com/news/story/6656/3646472 |author = 林麒瑋 |date = 2019年2月15日 |title = 我國捐50萬美金給委國反對派 外交部：必要人道救援 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref>

持有[[委內瑞拉護照|委內瑞拉護照]]的{{link-en|委內瑞拉人口|Demographics of Venezuela|委內瑞拉公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|委內瑞拉經濟|Economy of Venezuela}}}}
2004年3月，[[中華民國經濟部|中華民國經濟部]]政務次長[[陳瑞隆|陳瑞隆]]率領「2004年[[拉丁美洲|拉丁美洲]]貿易訪問團」訪問[[墨西哥|墨西哥]]、[[委內瑞拉|委內瑞拉]]、[[哥倫比亞|哥倫比亞]]、[[阿根廷|阿根廷]]、[[秘魯|秘魯]]。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2004/08/08_01_06.htm |title = 《中華民國93年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2018年，委內瑞拉是中華民國第134大貿易夥伴、第112大[[進口|進口]]夥伴、第131大[[出口贸易|出口]]夥伴。出口至委內瑞拉的金額為700萬1,571[[美元|美元]]，年減35.980%；自委內瑞拉進口的金額為951萬6,762美元，年減34.529%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）251萬5,191美元，年減30.123%。2017年，委內瑞拉是中華民國第122大貿易夥伴、第108大進口夥伴、第119大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
