'''中華民國總統官邸'''是[[中華民國|中華民國]][[中華民國總統|總統]]居住之[[住宅|寓所]]，由[[中華民國政府|中華民國政府]]提供予總統[[第一家庭|及其家人]]於其任職期間入住。現址位於[[臺北市|臺北市]][[中正區_(臺北市)|中正區]][[龍福里_(台北市)|龍福里]][[重慶南路_(臺北市)|重慶南路]]二段與[[愛國西路|愛國西路]]的交會處，鄰近[[臺北捷運|捷運]][[小南門站_(臺北市)|小南門站]]，自[[李登輝|李登輝]]就任總統後開始使用，現任總統[[蔡英文|蔡英文]]入住後取名為「'''永和寓所'''」。

== 概述 ==
中華民國總統的官邸與辦公室並不位於同一地方。1948年開始設置總統職位時，先後擔任正任總統與[[代總統|代總統]]的[[蔣中正|蔣中正]]與[[李宗仁|李宗仁]]均各自居住於原有在[[南京市_(中華民國)|南京市]]的寓所，分別位於'''[[憩廬|黃埔路3號]]'''與'''[[南京李宗仁公馆旧址|傅厚崗68號]]'''。1949年[[國府遷台|國府遷台]]後，歷任總統均居住於台北市，但官邸位置並不固定，例如蔣中正居於'''[[士林官邸|士林官邸]]'''，[[嚴家淦|嚴家淦]]居於'''[[嚴家淦先生故居|重慶官邸]]'''，[[蔣經國|蔣經國]]居於'''[[七海寓所|七海寓所]]'''。[[李登輝|李登輝]]擔任總統後，沿用其擔任[[中華民國副總統|副總統]]時的官邸，此處即今之總統官邸。

另外，[[中華民國副總統|副總統]]並不與總統住於同一官邸，而是於他處居住（如[[陳誠|陳誠]]已拆除之官邸位於[[信義路|信義路]]二段20巷1號之1、[[嚴家淦|嚴家淦]]之[[嚴家淦先生故居|重慶官邸]]、[[謝東閔|謝東閔]]及[[李元簇|李元簇]]之官邸皆位於[[大直|大直]]、[[李登輝|李登輝]]任副總統時之官邸位於[[大安區_(臺北市)|大安區]]大安路二段等）；自[[呂秀蓮|呂秀蓮]]以後，大都固定於由前副總統[[陳誠|陳誠]]家族宅邸改裝之台北市[[仁愛路_(台北市)|仁愛路三段]]的[[中華民國副總統官邸|副總統官邸]]，但前副總統[[吳敦義|吳敦義]]未搬入該官邸。<ref>{{cite news|url=http://news.ltn.com.tw/news/focus/paper/962694|title=副總統官邸空四年 陳建仁遷入機率高|author=|publisher=自由時報|date=2016-02-28}}</ref>

== 官邸的別名 ==
總統官邸依[[中華民國國家安全局|國安局]]維安代號的改變，而有不同的別稱。而其警衛室代號取意不同，往往展現其當時國情、執政權輪替與歷任總統其治國風格與理念。

== 官邸的維安 ==
中華民國總統官邸的維安工作由國家安全局特種勤務指揮中心統合指揮侍衛編組、便衣編組、武裝編組、警察編組並區分侍衛區、內衛區、中衛區、外衛區共同維護警衛對象、家屬及其住所之安全與安寧（法源依據：《[[s:特種勤務條例|特種勤務條例]]》-中華民國101年1月4日生效）。
