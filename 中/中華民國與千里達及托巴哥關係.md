'''中華民國與千里達及托巴哥關係'''是指[[中華民國|中華民國]]與[[千里達及托巴哥共和國|千里達及托巴哥共和國]]（通稱'''千里達'''）之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對千里達的相關事務由'''中華民國駐[[聖文森及格瑞那丁|聖文森國]][[中華民國駐外機構列表|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-258-c51de-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|千里達及托巴哥簽證政策|千里達及托巴哥公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可至千里達駐[[香港|香港]][[領事館|名譽領事館]]申請簽證，或由該國人士向其移民單位申辦[[簽證#落地签证|落地簽證]]。經千里達轉機，若已取得下段航程登機證、轉機時間少於4小時、於管制區內轉機，則無須簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-258-c51de-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持有{{link-en|千里達及托巴哥護照|Trinidad and Tobago passport}}的{{link-en|千里達及托巴哥人口|Demographics of Trinidad and Tobago|千里達公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將千里達及托巴哥歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2017年5月22日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-3-3c0f4-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將千里達歸類為'''第二級警示'''：對當地採取加強防護（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=E3CA6C67AEF2B001 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|千里達及托巴哥經濟|Economy of Trinidad and Tobago}}}}
===貿易===
2018年，千里達及托巴哥是中華民國第74大貿易夥伴、第58大[[進口|進口]]夥伴、第97大[[出口贸易|出口]]夥伴。出口至千里達的金額為3,246萬7,779[[美元|美元]]，年增3.834%；自千里達進口的金額為1億4,875萬5,666美元，年增18.414%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）1億1,628萬7,887美元，年增23.245%。2017年，千里達及托巴哥是中華民國第76大貿易夥伴、第59大進口夥伴、第98大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
