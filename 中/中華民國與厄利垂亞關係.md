'''中華民國與厄利垂亞關係'''是指[[中華民國|中華民國]]與[[厄利垂亞國|厄利垂亞國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對厄利垂亞的相關事務由'''{{link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''、'''駐[[杜拜|杜拜]][[臺北經濟文化代表處|臺北商務辦事處]]'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-197-056fb-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|厄利垂亞簽證政策|Visa policy of Eritrea}}|厄利垂亞公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往厄利垂亞駐[[日本|日本]][[外交代表機構#大使馆|大使館]]申請簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-197-056fb-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持[[厄利垂亞護照|厄利垂亞護照]]的{{link-en|厄利垂亞人口|Demographics of Eritrea|厄利垂亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將厄利垂亞歸類為'''橙色警戒'''：避免非必要旅行（2017年4月19日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-18-3ef09-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2017年，厄利垂亞是中華民國第213大貿易夥伴、第192大[[進口|進口]]夥伴、第209大[[出口贸易|出口]]夥伴。出口至厄利垂亞的金額為12萬0,917[[美元|美元]]，年減19.505%；自厄利垂亞進口的金額為7萬5,725美元，年增11.490%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）4萬5,192美元，年減45.086%。※2016年，厄利垂亞是中華民國第214大貿易夥伴、第193大進口夥伴、第210大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
