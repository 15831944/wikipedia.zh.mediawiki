'''中央国家机关后勤干部培训中心'''，是[[国家机关事务管理局|国家机关事务管理局]]直属[[事业单位|事业单位]]，与'''全国机关事务管理研究会'''[[合署办公|合署办公]]<ref name=ggj>{{Cite web |url = http://www.ggj.gov.cn/zzjg/jgsz/sydw/200610/t20061016_21894.htm |title = 中央国家机关后勤干部培训中心 |author =  |date =2017-12-18   |publisher = 国家机关事务管理局 |language =  |accessdate =  |quote =  }}</ref>

==沿革==
1991年，[[国务院机关事务管理局|国务院机关事务管理局]]和[[中华人民共和国人事部|人事部]]下发《关于加强中央国家机关后勤干部岗位培训工作的通知》（〔91〕国管培训字第229号）。1993年4月22日，[[国务院机关事务管理局|国务院机关事务管理局]]办公室以〔93〕局办字第11号印发《关于中央国家机关后勤干部岗位培训工作的实施意见》，内称“培训工作的日常管理和组织实施由我局中央国家机关后勤干部培训中心负责”，并附“中央国家机关后勤干部培训中心1993年——1995年培训工作重点”<ref>{{Cite web |url =http://gkml.ggj.gov.cn/zfxxgk/zhxxml/xxgkzcfg/201103/t20110325_569.htm  |title =关于中央国家机关后勤干部岗位培训工作的实施意见  |author =  |date =1993-04-22  |publisher =国家机关事务管理局  |language =  |accessdate =  |quote =  }}</ref>。

==职责==
#贯彻党和国家有关干部教育的方针、政策，结合中央国家机关后勤部门的具体情况，制订后勤干部岗位培训的中、长期规划和年度计划,并组织实施；
#制订中央国家机关后勤部门主要岗位的基本要求及教学大纲；
#组织实施后勤干部的规范化岗位培训和短期适应性培训，颁发培训证书，进行学籍和证书管理；
#负责收集、整理后勤干部培训信息资料，组织编写与审定后勤干部培训教材和参考资料；
#对中央国家机关各单位后勤部门组织的岗位培训进行业务指导；
#对中央国家机关后勤干部培训情况进行调查、统计和分析；组织进行后勤干部培训工作的理论研究与经验交流<ref name=jieshao>{{Cite web |url = http://www.pxzx.ggj.gov.cn/zxjs/201106/t20110613_1371.htm |title = 中心介绍 |author =  |date = 2011-06-13 |publisher = 中央国家机关后勤干部培训中心 |language =  |accessdate =  |quote =  }}</ref>。

==全国机关事务管理研究会==
1995年2月21日，'''全国机关事务工作协会'''召开成立大会<ref name=guo>郭济，开拓进取 务实创新──在全国机关事务工作协会第二次会员代表大会暨机关后勤改革座谈会上的讲话，中国机关后勤1999年S1期</ref><ref>李芬莉，继往开来努力开创协会工作新局面——机关事务工作协会发展历程回顾，中国机关后勤2008(12):35-37</ref>。'''全国机关事务管理研究会'''在全国机关事务工作协会基础上更名成立。2014年，为贯彻实施《机关事务管理条例》，经国管局党组同意并报民政部批准，全国机关事务工作协会更名为全国机关事务管理研究会<ref>{{Cite web |url =http://www.ggj.gov.cn/qx/qxjhzj/201502/t20150206_14723.htm  |title =全国机关事务工作协会2014年工作总结  |author =  |date =  2015-02-06|publisher = 全国机关事务管理研究会 |language =  |accessdate =  |quote =  }}</ref>。2016年12月9日，召开全国机关事务管理研究会第一次会员大会暨推进机关事务管理创新与理论建设座谈会。全国机关事务管理研究会的主要职责是：“开展机关事务管理理论与实践问题的研究，为推进管理科学化、保障法制化、服务社会化，建立和完善机关事务法治体系，发挥机关事务管理在国家治理中的作用服务。”<ref>{{Cite web |url = http://www.ggj.gov.cn/qx/qxyjhgk/qxyjhjj/201704/t20170407_14644.htm |title =全国机关事务管理研究会简介  |author =  |date =  2016-12-22|publisher = 全国机关事务管理研究会 |language =  |accessdate =  |quote =  }}</ref>
