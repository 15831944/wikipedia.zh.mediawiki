'''中華民國與黎巴嫩關係'''是指[[中華民國|中華民國]]與[[黎巴嫩共和國|黎巴嫩共和國]]之間的關係。黎巴嫩於1954－1971年，是[[中華民國外交#中华民国的邦交国|中華民國的邦交國]]，[[外交#外交上的承認|斷交]]後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對黎巴嫩的相關事務由'''駐[[約旦|約旦]][[臺北經濟文化代表處|臺北經濟文化辦事處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-248-44b59-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]一案棄權，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[黎巴嫩|黎巴嫩]]。]]
{{see also|中華民國外交|{{link-en|黎巴嫩外交|Foreign relations of Lebanon}}}}
{{see also|中国驻黎巴嫩大使列表{{!}}中華民國駐黎巴嫩外交官列表|黎巴嫩驻华大使列表{{!}}黎巴嫩駐中華民國外交官列表}}
1954年11月11日，兩國建立公使級[[外交#外交上的承認|外交關係]]。於首都[[貝魯特|貝魯特]]設立'''中華民國駐黎巴嫩共和國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1957年10月19日，升格為大使級外交關係，公使館升格为'''中華民國駐黎巴嫩共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1971年10月25日，黎巴嫩對於[[中國與聯合國#代表權問題|中華民國續留聯合國]]採棄權態度，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。

1971年11月9日，黎巴嫩与中华民国断交<ref name="lscd">{{cite book|title=《台灣歷史辭典》（附錄）|author=許雪姬|publisher=[[中華民國文化部|行政院文化建設委員會]]|chapter=中華民國歷任駐外大使(遷臺後，1949~2003.10)}}</ref>。

1998年2月，中華民國副總統[[連戰|連戰]]前往黎巴嫩訪問。<ref>{{Cite web |url = http://210.241.75.208/scripts/newsnote/tornado/searcher.exe?s=1&z=1&k=&m=0&p=&b=129706&v=root |title = 民國重要史事檢索 |publisher = [[國史館|國史館]]}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境黎巴嫩。]]
[[File:DEC18_Visa_requirements_for_Lebanese_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|黎巴嫩簽證政策|黎巴嫩公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以[[電子郵件|電子郵件]]向中華民國駐[[約旦|約旦]]商務辦事處申請代辦單次入境黎巴嫩、停留最多30天的預審制觀光[[簽證#落地签证|落地簽證]]。經黎巴嫩轉機，若持有第三國簽證及後段機票，無需申辦[[過境簽證|過境簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-248-44b59-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=7F220D7E656BE749 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|黎巴嫩護照|Lebanese passport}}的{{link-en|黎巴嫩人口|Demographics of Lebanon|黎巴嫩公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將黎巴嫩歸類為'''橙色警戒'''：避免非必要旅行（2017年11月27日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-379-049ae-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|黎巴嫩經濟|Economy of Lebanon}}}}
2010年11月3日，睽違7年之後，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）帶領由22家廠商組成的貿易訪問團造訪黎巴嫩，在首都[[貝魯特|貝魯特]]舉行貿易洽談會。其中以[[電機|電機]]、[[汽車零組件列表|汽車零配件]]、[[節能|節能]][[照明|照明]]等產品最受當地買主青睞。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=547837&url=/NewsSearch.asp |author = 李東森 |date = 2010年11月5日 |title = 睽違7年，台灣優質產品再度於黎巴嫩發光! |publisher = 經貿透視網}}</ref>

2011年10月24日，外貿協會帶領由24家廠商組成的[[中東|中東]]貿易訪問團抵達黎巴嫩，10月25日在首都貝魯特舉行貿易洽談會。其中以電動工具、[[不鏽鋼|不鏽鋼]][[建材|建材]]、汽車零配件、[[發光二極體|發光二極體]]（LED）等產品最受當地買主青睞。<ref name="貿協帶領廠商開發黎巴嫩商機">{{Cite web |url = http://www.trademag.org.tw/News.asp?id=571570&url=/NewsSearch.asp |author = 王裕宏 |date = 2011年10月26日 |title = 貿協帶領廠商開發黎巴嫩商機 |publisher = 經貿透視網}}</ref>

2018年，黎巴嫩是中華民國第92大貿易夥伴、第128大[[進口|進口]]夥伴、第78大[[出口贸易|出口]]夥伴。出口至黎巴嫩的金額為8,260萬7,891[[美元|美元]]，年減9.641%；自黎巴嫩進口的金額為488萬6,553美元，年增17.346%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）7,772萬1,338美元，年減10.929%。2017年，黎巴嫩是中華民國第92大貿易夥伴、第130大進口夥伴、第78大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

目前中華民國在黎巴嫩沒有[[中華民國駐外機構列表|駐外單位]]，雙邊貿易額也不高，還有很大開拓空間。廠商要取得[[黎巴嫩簽證政策|黎巴嫩簽證]]相當困難，外貿協會自2010年起連續兩年帶領廠商進入黎巴嫩市場，都是由外貿協會駐[[杜拜|杜拜]]主任前往協助辦理[[簽證|簽證]]，透過與黎巴嫩[[商會|商會]]合作，成功辦理貿易洽談會及參訪活動。<ref name="貿協帶領廠商開發黎巴嫩商機"/>
