1947年8月14日，中華民國與[[巴基斯坦自治領|巴基斯坦自治領]]建立大使級[[外交#外交上的承認|外交關係]]。{{noteTag|《[[申報|申報]]》報導「【[[中央社|中央社]][[南京市_(中華民國)|南京]]十四日電】[[中華民國外交部|外部]]今公布：中國政府決定與巴基斯坦互換[[特命全權大使|大使]]，按巴基斯坦為[[穆斯林世界|回教新邦]]，而[[省_(中華民國)|我國西北諸省]]信奉回教者亦甚眾，故兩國間使節之互派，將使兩國間之友誼日趨敦睦。」}}但中華民國沒有派駐巴大使，而巴基斯坦有派駐華使節。

1948年8月，{{link-en|巴基斯坦政府|Politics of Pakistan}}準備在[[南京市_(中華民國)|南京]]設立[[外交代表機構#大使馆|大使館]]。10月，巴基斯坦代表塔楚丁{{notetag|name=另譯Tag1|另譯達菊中、達朱丁、達菊丁。}}訪華處理在南京籌設大使館，並在[[上海市_(中華民國)|上海]]籌設[[領事館|領事館]]事宜。{{noteTag|《寧波日報》報導，民國三十七年八月二十六日第二版《巴基斯坦將設駐華使館》：「【中央社五日[[新德里|新德里]]電】可靠方面消息，巴基斯坦政府將採取步驟準備在南京建立巴基斯坦大使館，巴政府之負責此項職任之特派員塔楚丁，將於九月一日自[[喀拉蚩|喀拉蚩]]飛南京，準備一切，但巴政府迄未決定其首屆駐華大使之人選，按塔氏曾任印度駐華辦事員。」<br>《申報》報導，民國三十七年十月五日第一版《巴基斯坦代表昨訪[[劉師舜|劉師舜]]》：「【本報南京四日電】巴基斯坦代表達菊中教授，四日午十一時半訪我外次代理部務劉師舜，並面交出使之證明文件，達氏除在京籌設大使館外，並將在滬籌設領事館。據達氏稱，巴基斯坦首任駐華大使館人選尚在遴選中。」}}

1949年4月，[[中華民國政府|中華民國政府]]同意巴基斯坦在新疆[[喀什市|喀什]]開設領事館。{{noteTag|《申報》報導「【本報[[重慶市_(中華民國)|重慶]]廿八日電】巴基斯坦政府以新疆南部哈什與其毘鄰，關係密切，請准在哈什設領館，聞我外交部已同意。」}}{{noteTag|《[[中华人民共和国建交列表|中華人民共和國建交]]檔案：1949-1955》巴基斯坦篇提到，1950年1月29日，{{link-en|巴基斯坦駐蘇聯大使|Category:Ambassadors of Pakistan to the Soviet Union}}庫萊西回電新中國駐蘇大使[[王稼祥|王稼祥]]，希望中國政府將巴駐新疆喀什噶爾總領事穆罕默德·沙迪克當作可信任的巴方駐新疆代表，並允許他以此身份辦公。王稼祥大使回電說，在正式建交前，將把沙迪克當作外國僑民看待。}}

1950年1月4日，巴基斯坦與中華民國斷交。

1971年7月，[[美國總統|美國總統]][[尼克森|尼克森]]的[[總統國家安全事務助理|總統國家安全事務助理]][[季辛吉|季辛吉]]在訪問巴基斯坦的途中[[基辛格秘密访华|秘密訪問了北京]]（藉口[[腹痛|肚子痛]]）。在北京，季辛吉會見[[中華人民共和國|中華人民共和國]][[國務院總理|國務院總理]][[周恩來|周恩來]]。7月15日，尼克森宣布次年[[1972年尼克森訪華|訪問中華人民共和國]]。此舉為日後關係正常化奠定基礎，[[海峽兩岸關係|兩岸政府]]的國際地位也開始逆轉。<ref>{{Cite web |url = http://www.backchina.com/news/2011/05/18/141838.html |author = 《[[蘋果日報|蘋果日報]]》 |date = 2011年5月18日 |title = 基辛格装肚子痛助中美破冰 35年来“靠中吃饭” |publisher = BackChina.com}}</ref><ref>{{Cite web |url = http://www.books.com.tw/products/0010069633 |author = 傅建中 |date = 1999年6月22日 |title = 《季辛吉祕錄》 |publisher = [[博客來數位科技股份有限公司|博客來數位科技股份有限公司]]}}</ref>

持{{link-en|巴基斯坦護照|Pakistani passport}}的[[巴基斯坦人口|巴基斯坦公民]]抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將巴基斯坦歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2017年2月18日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-26-8c4ea-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將巴基斯坦歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2015年11月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=57E50F1E23B7006C |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

2013年，[[經濟部國際貿易局|經濟部國際貿易局]]挑選最具拓銷潛力的22國26城市推動執行「貿易尖兵任務」，其中[[亞洲國家和地區列表|亞洲6國]]的[[新興市場|新興市場]]就包括[[巴基斯坦|巴基斯坦]]。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622045&url=/NewsSearch.asp |author = 中華民國對外貿易發展協會 |date = 2013年10月30日 |title = 貿易尖兵勇闖22國26城市 新興市場商機無窮 |publisher = 經貿透視網}}</ref>

2013年11月26日，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）副[[秘書長|秘書長]]與巴基斯坦第2大城也是最大[[軟體|軟體]]產業重鎮的[[拉合爾|拉合爾]]商工總會（Lahore Chamber of Commerce and Industry，LCCI）[[會長|會長]]共同簽署合作協議，加強兩會合作，促進兩國廠商經貿往來。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=624128&url=/NewsSearch.asp |author = 中華民國對外貿易發展協會 |date = 2013年11月27日 |title = 外貿協會與巴基斯坦拉合爾商工總會簽署合作備忘錄 |publisher = 經貿透視網}}</ref>

2018年，巴基斯坦是中華民國第44大貿易夥伴、第55大[[進口|進口]]夥伴、第31大[[出口贸易|出口]]夥伴。出口至巴基斯坦的金額為7億2,980萬2,795[[美元|美元]]，年增40.847%；自巴基斯坦進口的金額為1億8,249萬6,451美元，年增32.767%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）5億4,730萬6,344美元，年增43.765%。2017年，巴基斯坦是中華民國第49大貿易夥伴、第57大進口夥伴、第37大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
