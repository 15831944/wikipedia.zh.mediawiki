'''中国中车股份有限公司'''（{{lang|en|'''CRRC Corporation Limited'''}}<ref>{{Cite web |url = http://www.hkexnews.hk/listedco/listconews/sehk/2014/1230/LTN20141230922_C.pdf |title = 中國北車股份有限公司第二屆監事會第二十四次會議決議公告}}</ref>，{{sehk|1766}}，{{sse|601766}}），简称'''中国中车'''或'''CRRC'''，是[[中国|中国]]一家从事[[铁路机车|铁路机车]]、[[铁路车辆|铁路车辆]]、[[动车组|动车组]]、[[地铁|地铁]]及其零部件的研发、制造、厂修及[[IGBT|IGBT]]<ref>{{Cite web |url = http://scitech.people.com.cn/n/2013/1028/c1057-23342054.html |title=中国南车IGBT销售破6万只}}</ref>、公交车<ref>{{Cite web |url = http://hn.people.com.cn/n/2014/1212/c195194-23197020.html |title = 株洲南车研发出“纯电驱动城市公交”已接订单}}</ref>等周边产业的大型[[中央企业|中央企业]]，不但是中国最大的[[铁路|轨道交通设备]]制造商和解决方案供应商，在全球同样具有领导地位。

2015年6月1日，[[中国南车股份有限公司|中国南车股份有限公司]]（CSR）吸收合并[[中国北车股份有限公司|中国北车股份有限公司]]（CNR）成立中国中车，成为全球最大铁路车辆及设备制造商。

== 历史沿革 ==
[[中国南车|中国南车]]与[[中国北车|中国北车]]成立之后出现了重复投资、海外恶性竞争等问题，并随之出现合并的呼声与方案<ref>{{Cite web | url=http://finance.eastmoney.com/news/1354,20130718307470585.html | title=南车和北车恶性竞争 整合存争议}}</ref>。2014年9月，网络上出现[[中国南车|中国南车]]、[[中国北车|中国北车]]即将合并的传言<ref>{{Cite web|url=http://finance.people.com.cn/stock/GB/n/2014/0915/c67815-25658391.html| title=传南车北车合并 人民日报：巨无霸火车厂没必要}}</ref>。2014年10月27日，[[中国南车股份有限公司|中国南车股份有限公司]]、[[中国北车股份有限公司|中国北车股份有限公司]]在上交所同时停牌<ref>{{Cite web|url=http://finance.people.com.cn/stock/n/2014/1028/c67815-25920708.html | title=南北车同时停牌再起整合猜想 高铁出海引发大戏}}</ref>。2014年10月29日，两家公司已经成立合并工作组，筹划合并事项<ref>{{Cite web|url=http://news.xinhuanet.com/fortune/2014-10/29/c_1113032696.htm |title= 北车南车将合并　中国高铁装备制造业将“一个声音”对外说话}}</ref>。2014年12月1日，两家公司发布公告继续停牌<ref>{{Cite web|url=http://finance.people.com.cn/n/2014/1204/c1004-26145025.html|title=南车北车合体初步方案曝光}}</ref>。

2014年12月30日晚间，南车、北车合并方案出台，中國南車将吸收合併中國北車，组成新的中国中车集团，合併後新公司承繼及承接中國南車與中國北車的全部資產、負債、業務、人員、合同、資質及其他一切權利與義務，從而實現雙方對等合併的行為<ref>{{Cite web | url=http://legal.people.com.cn/n/2014/1231/c188502-26304311.html | title=中国南车北车合并方案出炉 总资产有望超3000亿元}}</ref><ref>{{Cite web|url=http://politics.people.com.cn/n/2015/0102/c70731-26312384.html |title=南北车合并，能否实现“1+1>2”}}</ref><ref>{{Cite Web | url=http://ccnews.people.com.cn/n/2014/1231/c141677-26308299.html | title=南北车合并为中车 3000亿级“巨无霸”诞生}}</ref>。

2015年6月1日'''中国南车股份有限公司'''完成工商变更登记手续，变更后的公司中文名称为'''中国中车股份有限公司'''，简称中国中车。英文名称为CRRC Corporation Limited，简称CRRC。<ref>http://weibo.com/5618105325/CkzxQxm99</ref><ref>[http://www.csrgc.com.cn/g725/s2429/t253952.aspx 中国中车官方微博正式开通] {{webarchive|url=https://web.archive.org/web/20150610014655/http://www.csrgc.com.cn/g725/s2429/t253952.aspx |date=2015-06-10 }}</ref>

合併換股完成後，中國中車總股本達272.89億股，其中A股總股本約229.18億股、H股43.71億股H股。

2016年7月20日（[[美国東部時間|美国東部時間]]），美國《[[財富雜誌|財富雜誌]]》公布，中國中車截至2016年度[[財富世界500强|財富世界500强]]，排名為266位<ref>[http://beta.fortune.com/global500/crrc-266 中國中車股份有限公司-2016年度美國財富雜誌全球500大企業（266）（按國際會計準則編製，政府持有50%股權或以上，於2015年5月28日，已被中国北方机车车辆工业集团公司合併，以及於2015年6月1日更名前為中国南车集团公司）][[財富雜誌|財富雜誌]]</ref>。

2016年10月21日，中國中車在北京举行启动会，將研發時速600公里的高速磁浮列車等首批3個專案重點計畫。這3個專案是「軌道交通系統安全保障技術」、「時速400公里及以上高速客運裝備關鍵技術」和「磁浮交通系統關鍵技術」。這是中國大陸首個由企業牽頭組織實施的國家重點專項。

2017年，中車青島四方在[[芝加哥|芝加哥]]舉行建廠動工儀式。總投資1億美元占地275畝，此為中標芝加哥846輛、金額總計13億美元的地鐵標案配套計畫，在美國生產美國用的[[地鐵|地鐵]]車輛。車體總組焊是於青島廠，之後運往芝加哥做轉向架組裝和車輛總裝<ref>[http://www.chinatimes.com/newspapers/20170328000895-260309 中時 - 中車芝加哥建廠 為美打造地鐵車廂]</ref>。

2017年5月份中國中車集團發佈一種導入[[人工智能|人工智能]]化的新世代轨道交通系統，稱為「智慧軌道快運」即虛擬軌道電車（Autonomous rail Rapid Transit, ART），以解決[[BRT|BRT]]的一些缺點，<ref>[http://mt.sohu.com/it/d20170602/145429290_682294.shtml 颠覆式创新！中国中车全球首列虚拟轨道列车今日首发]</ref>該系統車輛大量運用[[物聯網|物聯網]]技術與綠能科技。車輛能源來自快充钛酸锂电池<ref>[https://udn.com/news/story/7332/2499611 全球首創虛擬軌道列車亮相]，</ref>只要在首末兩站充電就能完成25公里的路線，不須路面電網。其所行驶的道路只需简易喷涂轨迹标线，通过车载的多具攝影機[[軟體|軟體]]进行图像捕捉分析后控制车辆行驶于由标线构成的虚拟轨道上。車輛乍看與公路客车無異但搭乘感與轨道客车類似，而轉彎半徑小。其創新車輪制動系統全電腦化控制，其路網管理系統運用[[大數據|大數據]]與人工智能，在道路意外發生時能重新規劃替代路線脱离虚拟轨道繞道而行。

2018年5月，中国中车宣布实施货车业务重组，建立两个子集团。以[[中车齐齐哈尔交通装备|中车齐齐哈尔交通装备有限公司]]为主体成立[[中车齐车集团|中车齐车集团]]，下设中车齐齐哈尔车辆有限公司、中车沈阳机车车辆有限公司、中车石家庄车辆有限公司、中车山东机车车辆有限公司、中车风电（锡林郭勒）有限公司5家全资子公司；以[[中车长江车辆|中车长江车辆有限公司]]为主体成立[[中车长江集团|中车长江集团]]，下设中车长江公司、中车贵阳公司、中车眉山公司、中车太原公司、中车西安公司5家全资子公司<ref>{{Cite web|url=http://www.xinhuanet.com/fortune/2018-05/09/c_1122805858.htm|title=中国中车重组货车业务成立齐车集团 }}</ref><ref>{{Cite Web | url=http://finance.sina.com.cn/stock/s/2018-05-23/doc-ihawmauc1034976.shtml| title=中车长江集团在武汉成立 中国中车完成货车业务重组}}</ref>。
