'''中華民國與亞塞拜然關係'''是指[[中華民國|中華民國]]與[[亞塞拜然共和國|亞塞拜然共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對亞塞拜然的相關事務由'''[[臺北市|臺北]]－[[莫斯科|莫斯科]]經濟文化協調委員會[[臺北經濟文化代表處|駐莫斯科代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-144-9faf1-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|亞塞拜然簽證政策|亞塞拜然公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]須先至[[亞塞拜然的駐外機構|亞塞拜然的駐外機構]]取得簽證才能入境該國。入境簽證停留最多90天；[[過境簽證|過境簽證]]停留最多5天。但[[阿塞拜疆政治|亞塞拜然政府]]對[[中華民國政府|中華民國政府]]並非友善，取得該國簽證具相當難度。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-144-9faf1-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持有[[亞塞拜然護照|亞塞拜然護照]]的{{link-en|亞塞拜然人口|Demographics of Azerbaijan|亞塞拜然公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

===貿易===
2013年，[[經濟部國際貿易局|經濟部國際貿易局]]挑選最具拓銷潛力的22國26城市推動執行「貿易尖兵任務」，其中[[東歐|東歐]]7國的[[新興市場|新興市場]]就包括[[亞塞拜然|亞塞拜然]]。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622045&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2013年10月30日 |title = 貿易尖兵勇闖22國26城市 新興市場商機無窮 |publisher = 經貿透視網}}</ref>

2018年，亞塞拜然是中華民國第41大貿易夥伴、第30大[[進口|進口]]夥伴、第152大[[出口贸易|出口]]夥伴。出口至亞塞拜然的金額為365萬8,894[[美元|美元]]，年增115.397%；自亞塞拜然進口的金額為12億1,473萬1,752美元，年增72.776%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）12億1,107萬2,858美元，年增72.672%。2017年，亞塞拜然是中華民國第48大貿易夥伴、第36大進口夥伴、第166大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
