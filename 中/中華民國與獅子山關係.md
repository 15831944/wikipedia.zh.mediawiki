'''中華民國與獅子山關係'''是指[[中華民國|中華民國]]與[[獅子山共和國|獅子山共和國]]（[[意譯|意譯]]'''-{zh-hans:狮子山; zh-hant:獅子山}-'''；[[音譯|音譯]]'''-{zh-hans:塞拉利昂; zh-hant:塞拉利昂}-'''）之間的關係。兩國於1963－1971年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對獅子山的相關事務由'''{{Link-en|駐奈及利亞聯邦共和國臺北貿易辦事處|Taipei Trade Office in the Federal Republic of Nigeria}}'''、'''[[駐英國台北代表處|駐英國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-210-4fbe9-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[塞拉利昂|獅子山]]。]]
{{see also|中華民國外交|{{link-en|獅子山外交|Foreign relations of Sierra Leone}}}}
1963年9月28日，建立大使級[[外交#外交上的承認|外交關係]]。於首都[[自由城|自由城]]設立'''中華民國駐獅子山共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1971年8月20日，與中華民國斷交。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境獅子山。]]
[[File:Visa_requirements_for_Sierra_Leonean_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|獅子山簽證政策|{{link-en|獅子山公民簽證要求|Visa requirements for Sierra Leonean citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]與[[黃熱病|黃熱病]][[疫苗接種|疫苗接種]]證明（黃皮書）的[[中華民國國民|中華民國公民]]可前往獅子山駐[[美國|美國]][[外交代表機構#大使馆|大使館]]、駐[[甘比亞|甘比亞]][[高級專員|高級專員公署]]（大使館）申請簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-210-4fbe9-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|獅子山護照|Sierra Leonean passport}}的{{link-en|獅子山人口|Demographics of Sierra Leone|獅子山公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將獅子山歸類為'''橙色警戒'''：避免非必要旅行（2017年8月17日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-336-ae5d7-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將獅子山歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年11月6日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=46969DA565D3C9DC |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|獅子山經濟|Economy of Sierra Leone}}}}
===貿易===
2017年，獅子山是中華民國第145大貿易夥伴、第127大[[進口|進口]]夥伴、第142大[[出口贸易|出口]]夥伴。出口至獅子山的金額為572萬3,098[[美元|美元]]，年增64.101%；自獅子山進口的金額為449萬5,025美元，年增1,682.629%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）122萬8,073美元，年減62.043%。※2016年，獅子山是中華民國第170大貿易夥伴、第175大進口夥伴、第154大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
