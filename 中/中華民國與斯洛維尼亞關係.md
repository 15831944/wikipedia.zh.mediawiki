'''中華民國與斯洛維尼亞關係'''是指[[中華民國|中華民國]]與[[斯洛維尼亞共和國|斯洛維尼亞共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對斯洛維尼亞的相關事務由'''[[駐奧地利台北經濟文化代表處|駐奧地利台北經濟文化代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-152-61175-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，[[斯洛維尼亞|斯洛維尼亞]]當時屬於[[南斯拉夫社會主義聯邦共和國|南斯拉夫]]的[[斯洛維尼亞社會主義共和國|斯洛維尼亞社會主義共和國]]，由南斯拉夫作為[[主權國家|主權國家]]參與投票。]]
{{see also|中華民國外交|{{link-en|斯洛維尼亞外交|Foreign relations of Slovenia}}}}
斯洛維尼亞自1990年起，透過[[中華民國駐外機構列表|中華民國駐外館處]]表達與台灣發展關係的意願。斯洛維尼亞外交部長顧問德拉岡（Zvone Dragan）及科技部長{{link-sl|湯西奇|Peter Tancig}}分別於1991年12月及1992年6月抵台訪問；駐匈牙利台北代表處人員亦於1995－1997年多次前往斯洛維尼亞，商討加強雙邊關係。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-4.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1992年11月，簽署《[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]與{{link-en|斯洛維尼亞商工總會|Chamber of Commerce and Industry of Slovenia}}貿易合作協議書》。<ref name="86年外交年鑑"/>

1998年5月，[[斯洛維尼亞國會|斯洛維尼亞國會]]議員團由史畢雷迪奇議員（Bogomir Spiletic）率領首次抵台訪問。10月，中華民國亦首次在該國首都[[盧比安納|盧比安納]]舉辦[[國慶日_(中華民國)|國慶]]酒會。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/mo3/chapter1-4.htm |title = 《中華民國88年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2000年8月，斯洛維尼亞經濟部長薩國顯（Jose Zagozan）抵台訪問。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_8.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2011年2月14日，[[斯洛文尼亚总理列表|斯洛維尼亞前總理]][[亚内兹·扬沙|楊薩]]夫婦及台斯友好協會主席{{link-en|Mihael Brejc|Mihael Brejc|布雷茲}}夫婦應[[中華民國外交部|中華民國外交部]]邀請抵台訪問6天。期間會見[[中華民國總統|總統]][[馬英九|馬英九]]、[[中華民國外交部#歷任首長|外交部長]][[楊進添|楊進添]]，以及拜會[[中華民國國防部|國防部]]、[[行政院經濟建設委員會|行政院經濟建設委員會]]、[[行政院大陸委員會|行政院大陸委員會]]等政府單位。並參訪[[台灣民主基金會|台灣民主基金會]]、中華民國對外貿易發展協會、[[中山科學研究院|中山科學研究院]]、[[新竹科學工業園區|新竹科學工業園區]]，以及台灣的文教設施。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=8742DCE7A2A28761&sms=491D0E5BF5F4BC36&s=98AEB23D9799FED9 |author = 公眾外交協調會 |date = 2011年2月14日 |title = 斯洛維尼亞共和國前總理楊薩伉儷一行4人訪華 |publisher = 中華民國外交部}}</ref>

2016年，臺北市議會議長[[吳碧珠|吳碧珠]]訪問斯洛維尼亞。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

[[斯洛維尼亞人口|斯洛維尼亞公民]]持有[[歐盟護照|歐盟護照]]者也可以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將斯洛維尼亞歸類為'''灰色警示'''：提醒注意（2017年4月24日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-285-67476-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
[[File:Container_ship_in_Koper_2013.jpg|thumb]][[長榮海運|長榮海運]]貨輪由[[斯洛維尼亞|斯洛維尼亞]]商港[[科佩爾|科佩爾]]出港]]
2013年，[[經濟部國際貿易局|經濟部國際貿易局]]挑選最具拓銷潛力的22國26城市推動執行「貿易尖兵任務」，其中[[東歐|東歐]]7國的[[新興市場|新興市場]]就包括[[斯洛維尼亞|斯洛維尼亞]]。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622045&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2013年10月30日 |title = 貿易尖兵勇闖22國26城市 新興市場商機無窮 |publisher = 經貿透視網}}</ref>

2017年，斯洛維尼亞是中華民國第72大貿易夥伴、第89大[[進口|進口]]夥伴、第57大[[出口贸易|出口]]夥伴。出口至斯洛維尼亞的金額為1億5,620萬0,119[[美元|美元]]，年增0.830%；自斯洛維尼亞進口的金額為3,330萬1,320美元，年增3.338%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1億2,289萬8,799美元，年增0.172%。※2016年，斯洛維尼亞是中華民國第67大貿易夥伴、第87大進口夥伴、第57大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===學術===
{{see also|臺灣教育|{{link-en|斯洛維尼亞教育|Education in Slovenia}}}}
[[中華民國教育部|中華民國教育部]]自1991年7月設立「[[獨立國家國協|獨立國家國協]]及東歐地區國家來華留學[[獎學金|獎學金]]」，其中包括斯洛維尼亞。<ref name="86年外交年鑑"/>2004年起，改為教育部的「[[中華民國國語|華語文]]獎學金」，以及[[中華民國外交部|外交部]]的「台灣獎學金」，提供斯洛維尼亞學生申請赴台研習中文或攻讀[[碩士|碩]][[博士|博士]]。

斯洛維尼亞[[盧比安納大學|盧比安納大學]]東亞所設有「臺灣研究中心」，並開設台灣研究課程。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2011/2.1.4.pdf |title = 《中華民國100年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>自2012年，教育部補助選派華語文教師至該大學任教。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2012/2.1.4.pdf |title = 《中華民國101年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2015年4月，[[駐奧地利台北經濟文化代表處|駐奧地利台北經濟文化代表處]]與[[國立斯洛維尼亞博物館|國立斯洛維尼亞博物館]]、盧比安納大學漢學系、盧比安納大學臺灣研究中心、斯洛維尼亞太極拳協會等共同舉辦「[[臺灣文化|臺灣文化]]月」活動。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

自1993年，財團法人國際合作發展基金會（國合會）等機構提供名額，給斯洛維尼亞派員赴臺參加國際技術合作訓練。<ref name="86年外交年鑑"/>
