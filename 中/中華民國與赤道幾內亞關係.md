'''中華民國與赤道幾內亞關係'''是指[[中華民國|中華民國]]與[[赤道幾內亞共和國|赤道幾內亞共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對赤道幾內亞的相關事務由'''{{Link-en|駐西班牙台北經濟文化辦事處|Taipei Economic and Cultural Office in Spain}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-148-f84ff-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|赤道幾內亞簽證政策|Visa policy of Equatorial Guinea}}|{{link-en|赤道幾內亞公民簽證要求|Visa requirements for Equatorial Guinean citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]與邀請函的[[中華民國國民|中華民國公民]]可至赤道幾內亞駐[[西班牙|西班牙]]、[[英國|英國]]、[[法國|法國]]與[[美國|美國]]的[[外交代表機構|使領館]]申請觀光和商務簽證。<ref name="簽證及入境須知">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-148-f84ff-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|赤道幾內亞護照|Equatorial Guinean passport}}的{{link-en|赤道幾內亞人口|Demographics of Equatorial Guinea|赤道幾內亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將赤道幾內亞歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2017年5月19日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-124-654cf-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2017年，赤道幾內亞是中華民國第200大貿易夥伴、第172大[[進口|進口]]夥伴、第206大[[出口贸易|出口]]夥伴。出口至赤道幾內亞的金額為14萬2,187[[美元|美元]]，年減5.544%；自赤道幾內亞進口的金額為34萬6,029美元，年減98.089%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）20萬3,842美元，年減98.865%。※2016年，赤道幾內亞是中華民國第128大貿易夥伴、第102大進口夥伴、第209大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
