'''中央军委人民武装委员会'''，是[[中央军事委员会|中央军委]]议事协调机构，负责人民武装工作。

==沿革==
[[抗日战争|抗日战争]]时期，1940年[[晋察冀边区|晋察冀边区]]召开人民武装代表大会，经民主选举产生了晋察冀边区人民武装委员会，以加强对民兵、自卫队的领导。1941年11月7日，[[中共中央军事委员会|中共中央军事委员会]]《关于抗日根据地军事建设的指示》中规定：“自卫队与民兵应由各级（由边区到县）人民武装委员会统一领导。”1942年后，多数地区的人民武装委员会陆续改成人民武装部<ref name=miyun>{{Cite web|url = http://mfj.bjmy.gov.cn/gk/3/2014/0621/1717.html|title = 人民武装委员会|author = |date = 2014-06-21|publisher = 密云防空防灾信息网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20171010155532/http://mfj.bjmy.gov.cn/gk/3/2014/0621/1717.html|archive-date = 2017-10-10|dead-url = yes}}</ref>。

[[中华人民共和国|中华人民共和国]]成立后，根据[[中共中央|中共中央]]1952年12月4日《关于建立各级人民武装委员会的决定》，从中央到区均建立各级党的人民武装委员会，主任由同级[[党委书记|党委书记]]兼任，办事机构设在各该级人民武装部门。1959年12月14日，中共中央决定，取消人民武装委员会，成立中央军委和省、地区、县党委民兵工作组<ref name=miyun/>。

1958年，[[毛泽东|毛泽东]]针对[[美帝国主义|美帝国主义]]在[[台湾海峡|台湾海峡]]制造紧张局势及[[台湾|台湾]][[中国国民党|国民党]]当局声称要“反攻大陆”，向全党、全国发出“大办民兵师”的号召，在[[北戴河|北戴河]]召开的[[中共中央政治局|中共中央政治局]]会议上，通过了《中共中央关于民兵问题的决定》，作出了“[[全民皆兵|全民皆兵]]”的决策。1959年11月，中央军委决定成立'''中央军委民兵工作组'''，[[罗荣桓|罗荣桓]]任组长，[[张爱萍|张爱萍]]、[[甘泗淇|甘泗淇]]任副组长，以加强对民兵工作的领导。1960年4月，在北京召开全国民兵代表会议，到会代表6000多人<ref name=ben>{{Cite web |url =http://cpc.people.com.cn/GB/64162/82819/88994/89980/6023809.html  |title = 第二十一章 第四节  “兵民是胜利之本” |author =  |date =  2007-07-24|publisher = 中国共产党新闻网|language =  |accessdate =  |quote =  }}</ref>。

1961年7月，[[中共中央军委|中央军委]]决定将中央军委民兵工作组扩大为'''中央军委人民武装委员会'''，罗荣桓任主任，张爱萍、甘泗淇、[[傅秋涛|傅秋涛]]任副主任，日常工作由[[中国人民解放军总参谋部动员部|中国人民解放军总参谋部动员部]]承办<ref>{{Cite web |url =http://www.gfdy.gov.cn/org/2012-07/20/content_4962717.htm  |title =国家人民武装动员办公室  |author =  |date =2012-07-20   |publisher = 中国国防动员网 |language =  |accessdate =  |quote =  }}</ref><ref name=ben/>。1961年12月11日，中共中央、国务院决定将各级民兵工作组改为人民武装委员会<ref name=miyun/>。

1966年至1976年“[[文化大革命|文化大革命]]”期间，各级人民武装委员会的工作一度中断<ref name=miyun/>。

1978年11月7日，中共中央发出《关于恢复中央军委和各级地方党委人民武装委员会的通知》，决定恢复中央军委和各级地方党委的人民武装委员会；中央军委人民武装委员会由[[国家计委|国家计委]]、[[中华人民共和国公安部|公安部]]、[[中华人民共和国财政部|财政部]]、[[中华人民共和国农林部|农林部]]、[[中华人民共和国教育部|教育部]]、[[中华人民共和国民政部|民政部]]、[[国家体委|国家体委]]、[[全国总工会|全国总工会]]、[[全国妇联|全国妇联]]、[[共青团中央|共青团中央]]等国家机关、[[人民团体|人民团体]]和[[中国人民解放军总参谋部|中国人民解放军总参谋部]]、[[中国人民解放军总政治部|总政治部]]、[[中国人民解放军总后勤部|总后勤部]]的负责人组成，主任由中央军委副主席兼任，办事机构设在[[中国人民解放军总参谋部动员部|总参谋部动员部]]；省、地区、县（市）的人民武装委员会分别由同级地方党委、政府有关部门、人民团体和省军区、军分区、县（市）人民武装部的负责人组成，主任由同级党委书记（或副书记）兼任，办事机构设在省军区、军分区和县（市）人民武装部，负责日常工作<ref name=miyun/>。

1994年11月29日，《国务院、中央军委关于成立国家国防动员委员会的通知》（国发〔1994〕63号）发出，国务院、中央军委决定成立[[国家国防动员委员会|国家国防动员委员会]]。“国家国防动员委员会是在国务院、中央军委领导下，主管全国国防动员工作的议事协调机构。”“国家国防动员委员会成立后，[[国家人民防空委员会|国家人民防空委员会]]，[[国务院、中央军委交通战备领导小组|国务院、中央军委交通战备领导小组]]，中央军委人民武装委员会即自行撤销。”<ref name=chengli>{{Cite web |url = http://www.gov.cn/xxgk/pub/govpublic/mrlm/201108/t20110812_63985.html |title =国务院、中央军委关于成立国家国防动员委员会的通知  |author =  |date = 1994-11-29 |publisher =中国政府网  |language =  |accessdate =  |quote =  }}</ref>。

==办事机构==
中央军委人民武装委员会的办事机构是'''中央军委人民武装委员会办公室'''，设在[[中国人民解放军总参谋部动员部|中国人民解放军总参谋部动员部]]<ref>严高荣，丰城市开展向“关心国防后备力量建设的好书记”学习活动，中国民兵1991(12):19-19</ref>。历任办公室主任是：

*[[曹宇光|曹宇光]]（1978年—1982年，总参谋部动员部部长兼）
*[[孟平|孟平]]（1982年—1985年，总参谋部动员部部长兼）
*[[陈超|陈超]]（1985年—1990年，总参谋部动员部部长兼）<ref>杨文斗，刊授是提高民兵军政素质的重要途径 陈超部长谈民兵工作改革的形势和刊授教育的发展，中国民兵1987(9):3</ref>
*[[谭冬生|谭冬生]]（1990年—1994年，总参谋部动员部部长兼）
