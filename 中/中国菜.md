{{中国菜}}
'''中国菜'''又叫'''中餐'''，發源於[[中國|中國]]，对[[东亚|东亚]]地区的[[飲食文化|飲食文化]]带来深远影响，也同時常見於多華人之地區。[[韓國|韓國]]和[[日本|日本]]稱爲'''中華料理'''或'''中国料理'''。中国菜的特点为：'''色'''、'''香'''、'''味'''、'''意'''、'''形'''，被称为「国菜五品」。中国菜有強烈的香氣，并根据各地风味采用不同的調味。按烹饪特点又可分为：'''选料'''、'''刀工'''、'''火候'''和'''调味'''。

中國菜品系之多是非常驚人的，這是由于中國地区廣大，從南向北的[[熱帶|熱帶]]、[[副熱帶|副熱帶]]一路到[[溫帶|溫帶]]，由[[南海|南海]]島嶼氣候至內陸[[大陸性氣候|大陸性氣候]]，還有世界屋脊的[[高原|高原]][[山脈|山脈]]與無數源遠流長的[[河流|河流]]、[[窪地|窪地]]與[[湖泊|湖泊]]，提供了不可想像的多元食材。加上漢唐時[[絲路|絲路]]貿易交流、宋元時漠北民族的文化衝擊、明清時期[[大航海時代|大航海時代]]的新大陸物種，也沒有明顯宗教忌口的風氣，擁有持續發展達三千年的農漁傳承與廚藝創新。清代中期，确立了主要[[调味料|调味料]]——[[辣椒|辣椒]]和[[蔗糖|蔗糖]]的运用。奠定了当代中国菜的[[口味|口味]]基础<ref name="a1">{{Cite web |url=http://news.ifeng.com/a/20180224/56228765_0.shtml |title=八大菜系的历史，比春晚长不了几年 |accessdate=2018-05-04 |author=吴余 |date=2018-02-24 |work= |publisher=凤凰网|language=Zh-hans }}</ref>。近代以来，加工食材的技術、刀具也是最多樣的，使各个地方菜肴差异极大。由於無法一概而論，光代表[[菜系|菜系]]就有[[鲁菜|鲁菜]]、[[川菜|川菜]]、[[粤菜|粤菜]]、[[淮扬菜|淮扬菜]]等四大菜系，深入研究可多達八到十種以上，這還不包含漢族和中國境內少數民族與東亞其他國家的融合，多樣化程度之高在全世界幾乎沒有類似的案例，因而中國菜式可以說是覆蓋範圍非常廣大的飲食霸主。

中国菜是目前[[中國文化|中國文化]]中最主要的代表。

===选料===
中国菜的食材非常丰富，所有能吃的东西，都可以做为中国菜的食材，但食材的选择关系到菜品的质量。

中国菜中常选择名贵的食材，如[[燕窝|燕窝]]、[[鱼翅|鱼翅]]、[[熊掌|熊掌]]、[[鹿尾|鹿尾]]、[[虎骨|虎骨]]、[[猴脑|猴脑]]等。但是部分食材取自[[保护动物|保护动物]]，加上衛生防疫等考量，所以某些名菜在現代无法烹製，但也有厨师使用替代品进行烹饪。

====食材选取====
* 时令：食材根据动植物的成熟时间不同，品质也不同。比如[[淮扬菜|淮扬菜]]有“[[刀鱼|刀鱼]]不过[[清明|清明]]，[[鲟鱼|鲟鱼]]不过[[中華端午節|端午]]”的说法。
* 区域：不同区域生长的食材品质亦不同。比如[[蟹|蟹]]，以[[阳澄湖|阳澄湖]]和[[盘锦|盘锦]]产为佳。
* 品种：做一些菜肴，需要不同品种的食材。比如，[[北京烤鸭|北京烤鸭]]是用北京特有的[[填鸭|填鸭]]做成；白斩鸡则用[[三黄鸡|三黄鸡]]做成。
* 部位：食材不同的部位被制作成不同的菜。比如家常菜“肉段”应用里脊制作。
* 因材制宜：大部分菜肴对食材很讲究。例如生蠔和醉虾要求新鲜的[[蠔|蠔]]和[[虾|虾]]。而一些菜肴，例如[[炒饭|炒饭]]和[[回锅肉|回锅肉]]，则最好用隔夜的米饭和猪肉。
* 时代变化：例如[[北京烤鸭|北京烤鸭]]，以前是越肥越好，现代则倾向用瘦肉鸭。现代也不再利用保护动物和珍贵植物做食材。另外开始引进其他外国食材，例如[[法国蜗牛|法国蜗牛]]等。

====食材六品====
; 色 : 有些食材是为了装饰用的。比如在[[鱼翅羹|鱼翅羹]]里加一点[[藏红花|藏红花]]。在中国菜中，任何点缀的东西都可以食用。
; 香 : 增加香味的食材。[[胡椒|胡椒]]，[[茴香|茴香]]一些[[香辛料|香辛料]]主要是调味去除食材的腥味。也有一些高级菜肴如[[叫化鸡|叫化鸡]]用荷叶包裹，闻起来有一股清香。[[鲜花饼|鲜花饼]]加入鲜花，吃起来有临花圃之中。
; 味 : 食材的选择也可以增强食物的味道。比如[[虾饺|虾饺]]中加入的[[虾脑|虾脑]]。使[[饺子|饺子]]更加鲜美。
; 意 : 食材的名称、形状可代表菜肴意境。古时候，举子们在赶考的时候要吃条红色鲤鱼，意为「鲤鱼跳龙门」。
; 形 : 即使是冷盘，对食材的要求也是相當講究的。
; 养 : 一些食材本身就有丰富的营养价值，甚至还有药用性。像[[红枣猪手汤|红枣猪手汤]]富含[[动物胶|动物胶]]，对孕妇非常有帮助。

====食材对菜系的影响====
* [[鲁菜|鲁菜]]：发端于春秋战国时的齐国和鲁国，形成于秦汉。宋代后，鲁菜就成为「北食」的代表。分为胶东菜,济南菜以及孔府菜等三大流派。鲁菜讲究调味纯正，口味偏于鹹鲜，具有鲜、嫩、香、脆的特色。十分讲究清汤和奶汤的调制，清汤色清而鲜，奶汤色白而醇。
* [[川菜|川菜]]：四川离海较远，因此以河鲜，菌类，干货，山珍为特色。主要以“辣”为特色,对西南、华南各地区有很大影响。
* [[粤菜|粤菜]]：廣東位于亞熱帶地区，瀕臨南海，雨量充沛，物產富饒。海鲜以“生猛、鲜”为主，对原料的新鲜程度要求很严。
* [[苏菜|苏菜]]：处于江淮水乡，水产丰富，菜肴制作精细，对[[长江|长江]]下游区域有很大影响。

===刀工===
中国菜的原料大多加工成小块尺寸，不像[[西餐|西餐]]在食用时进行切割。中国菜对刀工非常讲究，工具主要是菜刀和砧板，可将原料切成片、丝、条、块、丁、粒、米茸等形状，并要求其大小、厚薄、粗细均匀。有些原料经厨师的刀工后可拼成[[栩栩如生|栩栩如生]]的美丽图案。

====原料成形====
* [[马牙段|马牙段]]：把葱从中间冲开，切2厘米长的段。 
* [[骰子丁|骰子丁]]：呈四方形，面积有1厘米立方。 
* [[象眼块|象眼块]]：两头尖、中间宽，一般是4厘米长、宽1.5厘米、厚1.5厘米。其大小根据主料、盛器的大小而定。 
* [[菱形块|菱形块]]：和象眼块相仿，但没象眼块那么规则。 
* [[骨牌块|骨牌块]]：呈长方形。一般为5厘米长、2.5厘米宽、7毫米厚。大小可根据情况而定。 
* [[滚刀块|滚刀块]]：采用原料滚动、斜立刀的方法，将原料切成基本相同的块，如[[烧茄子|烧茄子]]等。 
* [[绿豆丁|绿豆丁]]：一种类似绿豆子的小方丁，约5-7毫米立方。 
* [[劈柴块|劈柴块]]：不规则的原料经过加工,形状基本一致，一般长5厘米，宽、厚各1厘米。 
* [[雪花片|雪花片]]：一般1.5毫米厚，长、宽各1.5厘米。 
* [[柳叶片|柳叶片]]：大部分用笋尖、[[茭白|茭白]]尖制作，一般长5厘米、厚1.5毫米，一头尖，一头宽。 
* [[松里|松里]]：熟原料的面不动，里边的大块肉撕成小片后放平。如[[锅烧鸭|锅烧鸭]]、[[清蒸整面鸭|清蒸整面鸭]]等。 
* [[卧刀片|卧刀片]]：又称坡刀片。操作方法是左手按原料，右手执刀，刀背向外，刀背略高于刀刃，使刀身呈坡斜状，成品呈斜片状，如腰片、海参、肚片等。

===烹调方法与火候===
[[File:20090518_Shanghai_Shikumen_Museum_6860.jpg|thumb]]的家庭厨房]]
中国菜烹调方法非常多，有[[凉拌|凉拌]]、[[炒|炒]]、[[爆|爆]]、[[溜|溜]]、[[煸|煸]]、[[蒸|蒸]]、[[熬|熬]]、[[煮|煮]]、[[炖|炖]]、[[煨|煨]]、[[烩|烩]]、[[氽|氽]]、[[涮|涮]]、[[烧|烧]]、[[焯|焯]]、[[卤|卤]]、[[酱|酱]]、[[煎|煎]]、[[炸|炸]]、[[焖|焖]]、[[烤|烤]]、[[焗|焗]]、[[熏|熏]]等几十种，每一种又可分为好几类。中国菜在制作过程中还十分讲究火候，以蒸排骨为例，蒸的时间长了，肉就老了，时间短了，则没熟透。

===调味===
{{main|香草和调味料列表}}
[[File:Sucre_blanc_cassonade_complet_rapadura.jpg|缩略图]]，是烹饪中常见的甜味剂。背后则是跨越千年发展的[[蔗糖|蔗糖]]制造技术。]]
中国菜的调味料也非常多，是影响菜肴的主要原因之一。[[苦_(味覺)|苦]]、[[咸味|咸]]、[[甜味|甜]]、[[酸|酸]]、[[鲜味|鲜]]是基本[[味道|味道]]，它们与[[辣|辣]]（[[麻辣|麻辣]]），即是中国文化中所称的酸甜苦辣。除苦味外，都是菜肴烹饪时所追求的味道。

[[食盐|食盐]]是中国，亦是全世界烹饪时，最主要的咸味剂，而[[酱油|酱油]]、[[蚝油|蚝油]]、[[豆豉|豆豉]]等是中国常见的咸鲜口味[[调味剂|调味剂]]。西周时，中国烹饪即使用[[醋|醋]]制造[[酸味|酸味]]。[[先秦|先秦]]以来，烹饪的[[甜味剂|甜味剂]]主要是[[蜂蜜|蜂蜜]]和[[饴糖|饴糖]]，饴糖以[[麦芽糖|麦芽糖]]最为常见。发源于印度的[[蔗糖|蔗糖]]，虽然在前1世纪或之前即传入中国，但中国制糖业的发达和技术成熟，则是在唐代以后。明代，中国制糖技术更为成熟，采用黄泥水淋脱色法制造出[[白糖|白糖]]。随着[[地理大发现|地理大发现]]，[[辣椒|辣椒]]在明朝末年传入中国。此前，则用[[花椒|花椒]]制造[[辣味|辣味]]。清代中期<ref name="a1"/>，辣椒和蔗糖在中国获得普遍的应用，而在食盐匮乏的[[贵州|贵州]]，辣椒则起到替代食盐的作用。或称之为穷人的[[食用油|油]]盐<ref>{{Cite web |url=http://ah.people.com.cn/n2/2015/1227/c358327-27410588.html |title=趣味历史：辣椒明朝才进入中国 先秦人“重口味”爱用花椒 |accessdate=2018-05-04 |author= |date=2015-12-27 |work= |publisher=人民网，来源：中国青年网|language=Zh-hans }}</ref>。1908年，日本化学家[[池田菊苗|池田菊苗]]发明了的[[味精|味精]]，它与[[鸡精|鸡精]]是当代普遍使用的[[鲜味剂|鲜味剂]]。

除甜、辣、酸、鲜等调味剂外，常用的调味料依据效用有以下几类：
# 调色：[[酱油|酱油]]、[[红曲米|红曲米]]。
# 调香：[[麻油|麻油]]、[[麻酱|麻酱]]、[[黄酱|黄酱]]、[[葱|葱]]、[[蒜|蒜]]、八角/茴香、[[桂皮|桂皮]]、[[丁香|丁香]]、[[肉豆蔻|肉豆蔻]]、[[白豆蔻|白豆蔻]]、[[草果|草果]]、[[香菜|香菜]]、[[香菇|香菇]]、[[桂花|桂花]]、[[薄荷|薄荷]]、[[三奈|三奈]]、[[月桂|香叶]]、[[砂仁|砂仁]]、[[甘草|甘草]]、[[木香|木香]]、[[良姜|良姜]]、[[厚朴|厚朴]]。
# 调型：[[生粉|生粉]]、[[菱角|菱粉]]、[[冰糖|冰糖]]。
# 滋补：[[枸杞|枸杞]]、[[桂圆|桂圆]]、[[党参|党参]]、[[西洋参|西洋参]]、[[黄芪|黄芪]]、[[当归|当归]]、[[天麻|天麻]]、[[冬虫夏草|冬虫夏草]]、[[川芎|川芎]]、[[茯苓|茯苓]]、[[薏米|薏米]]、[[三七|三七]]、[[莲子|莲子]]。
# 其他添加剂：[[高汤|高汤]]。

===用餐礼仪===
{{main|中式餐桌禮儀}}
因中国地域辽阔，[[礼仪|礼仪]]繁多而且不一致。但有些用餐礼仪还是通用的。
* 所有用餐者、特别是长辈未到时，不可以入座；主人长辈先食；如果有长辈，应该让菜或斟酒。
* 一般来讲餐桌上正对门的位置是主人的位置，主人左右两侧应为第一、第二贵宾。妥善的做法是等待主人为你指定座位。
* 吃飯時不能發出咀嚼的聲音；不狼吞虎咽，不用嘴送食他人的让菜。不用舌头舔食餐具；不伏碗吃饭，应端碗吃饭；拿饭碗时，不可以端碗底；吐出的骨头鱼刺等饭渣，应放到指定的地方。如果要咳嗽、打喷嚏，要用手或手帕捂住嘴，并把头向后方转。
* 使用匙饮汤，不举碗海喝（然而這一般只適用於街外用膳；在家用膳舉碗海喝亦無傷大雅），不用[[筷子|筷子]]搅拌热汤，不呡汤而饮；夹菜时候最好一次夹走。不可以用筷子在菜盘子中乱翻。。
* 等上餐时候，不可以用筷子互相敲打，或者拿筷子敲打餐具；筷子不可对夹，如果用餐时一双筷子因为夹菜而夹到另一双筷子，称为筷子打架。这是非常不礼貌的事情；如果饭没吃完，中途想放下碗筷休息片刻，筷子要平放在碗上，绝对不可以垂直插在饭里，因为这被认为是凭吊死人的；任何餐具反扣的行为被视为不礼貌。比如碗或酒具。
* 如果多人入席，桌子很大，又不是旋转桌子，菜离你比较远，最好不要唐突地站起来去夹食，相反，如果观察到其他人想吃自己身边的菜，可以站起来帮助对方，这样符合礼节，也显得亲切；如果盘中的菜已不多，你又想把它“打扫”干净，应征询一下同桌人的意见。
* 用餐时如果餐具失落到地上而损坏是非常不好的。有些地方新年打碎餐具有说“岁岁平安”的习惯。

===艺术===
[[File:BCfood12.JPG|thumb]]]]
中國菜很早就講究色、香、味俱佳，《[[后汉书|后汉书]]·边让传》：“函牛之鼎以烹鸡，多汁则淡而不可食，少汁则熬而不可熟。”中国菜的取名多彩多姿，以写意手法命名的有“龙虎会”、“凤爪龙衣”、“狮子头”、“佛跳墙”，以人物命名的如“东坡肉”、“宋嫂鱼羹”、“宫保鸡丁”。唐朝皇帝[[唐懿宗|唐懿宗]]命皇宮內準備各種美味給[[同昌公主|同昌公主]]食用，其中“靈消炙”這道菜，是用喜鵲舌、羊心尖等材料調制而成。

===文学===
中国菜在文学作品中记录了许多不同历史时期中的著名菜肴，例如《[[红楼梦|红楼梦]]》、《[[金瓶梅|金瓶梅]]》等，现代美食家经常从这些作品中可以找到创新和恢复古代菜肴的方法。《红楼梦》第六十回【牛奶茯苓霜】：“……只有昨儿有粤东的官儿来拜，送了上头两小篓子茯苓霜。……这地方千年松柏最多，所以单取了这茯苓的精液和了药，不知怎么弄出这怪俊的白霜儿来。说第一用人乳和着，每日早起吃一钟，最补人的；第二用牛奶子；万不得，滚白水也好。我们想着，正宜外甥女儿吃。……”。

===哲学===
中国的饮食文化在各个方面体现出来，古代的[[哲学家|哲学家]]经常用烹饪方式来进行比喻，如《[[吕氏春秋|吕氏春秋]]·察今》載：“尝一脟肉而知一镬之味，一鼎之调。”[[老子|老子]]在《[[道德经|道德经]]》中也提到：“治大国，若烹小鲜”。[[孔子|孔子]]曾提到：“食不厌精，烩不厌细”和“肉不正不食”以比喻身正和严格要求自己。[[班固|班固]]在《[[汉书|汉书]]》中也写道：“王者以民为本，而民以食为天”，将饮食看成是治理国家首要的问题。

“庖丁解牛”出自《[[庄子|庄子]]》，讲的是顺其自然的养生之道。“君子远庖厨”，来源于《[[礼记|礼记]]》。意为心胸广大的人应该仁慈，不應進入廚房宰殺動物以備食用。中医学家认为，饮食是健康之本。药王[[孙思邈|孙思邈]]说过：“救急之道在于药，安身之本在于食。” 而《[[本草纲目|本草纲目]]》作者[[李时珍|李时珍]]也有“药补不如食补”的说法。
