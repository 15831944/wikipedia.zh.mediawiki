'''中華民國與斯洛伐克關係'''是指[[中華民國|中華民國]]與[[斯洛伐克共和國|斯洛伐克共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，但於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，[[斯洛伐克|斯洛伐克]]當時屬於[[捷克斯洛伐克社會主義共和國|捷克斯洛伐克社會主義共和國]]，以其作為[[主權國家|主權國家]]參與投票。]]
{{see also|中華民國外交|{{link-en|斯洛伐克外交|Foreign relations of Slovakia}}}}
{{see also|中国驻捷克大使列表{{!}}中華民國駐捷克斯洛伐克外交官列表|捷克驻华大使列表{{!}}捷克斯洛伐克駐中華民國外交官列表}}
1930年2月12日，[[捷克斯洛伐克共和國|捷克斯洛伐克共和國]]（第一共和）與中華民國建立公使級[[外交#外交上的承認|外交關係]]。

1931年，於首都[[布拉格|布拉格]]設立'''中華民國駐捷克斯洛伐克共和國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]（[[中国驻波兰大使列表|駐波蘭公使]]兼任）。

1939年，因上年[[蘇台德區問題|納粹德國佔領捷克斯洛伐克]]，於是關閉駐在公使館，原公使調駐[[羅馬尼亞|羅馬尼亞]]。

1941年8月26日，中華民國外交承認位於[[英國|英國]]首都[[倫敦|倫敦]]的[[捷克斯洛伐克流亡政府|捷克斯洛伐克流亡政府]]，並在當地設立公使館。

1944年7月25日，升格為大使級外交關係。公使館升格為'''中華民國駐捷克斯洛伐克共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1945年5月，[[第二次世界大戰歐洲戰場的結束|納粹德國投降]]後，大使館由英國遷回捷克斯洛伐克（第三共和）。

1949年10月5日，與中華民國斷交，至今邦交未曾恢復。

1950－1990年代東西方陣營的「[[冷戰|冷戰]]」期間，中華民國在[[意識形態|意識形態]]和[[政體列表|制度]]上均偏向以[[美國|美國]]為首的[[資本主義|資本主義]]陣營，復加上以[[蘇聯|蘇聯]]為首的[[社會主義國家|共產集團國家]]的封閉，導致從[[臺灣戰後時期|中華民國政府遷台]]後幾乎沒有官方往來。

1971年10月25日，身為共產陣營成員及蘇聯[[衛星國|衛星國]]之一的[[捷克斯洛伐克社會主義共和國|捷克斯洛伐克社會主義共和國]]支持[[中華人民共和國|中華人民共和國]]取得[[聯合國|聯合國]]的「[[中國與聯合國#代表權問題|中國席位]]」（即中國代表權問題、《[[聯合國大會2758號決議|聯合國大會2758號決議]]》）。

1993年1月1日，[[天鵝絨分離|斯洛伐克宣布獨立]]，[[捷克斯洛伐克聯邦共和國|捷克斯洛伐克聯邦共和國]]瓦解。在斯洛伐克轉型為[[民主#民主國家|民主國家]]並重新對外開放後，雙方始恢復往來。

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將斯洛伐克歸類為'''灰色警示'''：提醒注意（2017年4月21日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-284-40af1-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|斯洛伐克經濟}}
===貿易===
[[經濟部國際貿易局|經濟部國際貿易局]]於首都[[布拉提斯拉瓦|布拉提斯拉瓦]]設立'''駐斯洛伐克代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2017年，斯洛伐克是中華民國第61大貿易夥伴、第75大[[進口|進口]]夥伴、第48大[[出口贸易|出口]]夥伴。出口至斯洛伐克的金額為2億8,359萬4,919[[美元|美元]]，年減21.676%；自斯洛伐克進口的金額為6,145萬1,066美元，年增0.796%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）2億2,214萬3,853美元，年減26.226%。※2016年，斯洛伐克是中華民國第56大貿易夥伴、第73大進口夥伴、第42大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資===
目前在斯洛伐克投資或設有營運據點的[[臺灣商人|台商]]計有3家，投資概況如下：

[[台達電子|台達電子]]，項目包括：[[通訊技術|通訊]]、[[資訊科技|資訊]]、[[電子產品|電子]]、[[自動化技術|工業自動化]]及[[医疗器械|醫療設備]]使用之[[電源供應|電源系統]]、[[太陽能電池|太陽能電池]]及[[風能|風力]][[涡轮发动机|渦輪機]]之[[逆變器|變流器]]、大型[[顯示器|顯示器]]（或稱[[LED|LED]]牆，可供[[医疗卫生|醫療]]用途或[[監視|監控]]室使用）。總投資金額約5,350萬[[歐元|歐元]]，僱用員工約700人。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/sk/pdf/104-66Slovak%20Republic.pdf |date = 2015年7月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

[[鴻海科技集團|鴻海科技集團]]，項目包括[[液晶顯示器|液晶電視]]及[[數位視訊轉換盒|電視機上盒]]。2010年3月，鴻海以4,500萬歐元[[併購|併購]]日商[[索尼公司|索尼公司]]（SONY）位於斯洛伐克[[尼特拉|尼特拉]]廠及歐洲配銷中心（鴻海[[持份者|持股]]90%；索尼持股10%），[[ODM|代工生產]]SONY品牌液晶電視，深化鴻海在[[歐洲國家和地區列表|歐洲]]的投產布局。2013年雇用員工1,054人。<ref name="投資環境簡介"/>

[[友達光電|友達光電]]，項目包括液晶電視及液晶顯示器[[前端和后端|後端]][[模組|模組]]。總投資金額約40億[[新台幣|新台幣]]，雇用員工約200人。<ref name="投資環境簡介"/>

===會展===
2013年9月，中華民國經濟部常務次長[[卓士昭|卓士昭]]訪問斯洛伐克，與該國經濟部政務次長{{link-en|帕福里斯|Pavol Pavlis}}舉行首次對話會議，就貿易、投資及旅遊等領域合作交換意見。<ref name="102年外交年鑑"/>

2013年10月，斯洛伐克總統[[伊万·加什帕罗维奇|蓋斯巴洛維奇]]於該國電子電機展（ELO SYS）為「臺灣館」剪綵。<ref name="102年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2013/09.pdf |title = 《中華民國102年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>
