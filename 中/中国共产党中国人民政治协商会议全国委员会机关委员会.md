'''中国共产党中国人民政治协商会议全国委员会机关委员会'''（简称'''全国政协机关党委'''），是[[中国人民政治协商会议全国委员会|中国人民政治协商会议全国委员会]]机关的[[中国共产党|中国共产党]]基层组织。

==沿革==
1983年，[[全国政协办公厅|全国政协办公厅]]被明确为正部级机构，下设8个局级机构以及机关党委<ref name=yongan>{{Cite web |url = http://www.yaszx.gov.cn/list/articlelist.asp?id=709 |title = 政协知识汇编 |author =  |date = 2011-11-11 |publisher = 福建省永安市政协 |language =  |accessdate =  |quote =  |archive-url = https://web.archive.org/web/20171007021325/http://www.yaszx.gov.cn/list/articlelist.asp?id=709# |archive-date = 2017-10-07 |dead-url = yes }}</ref>。

1994年机构改革后，全国政协办公厅下设11个行政局级机构以及机关党委<ref name=yongan/>。

2002年机构改革后，办公厅下设1个副部级机构，15个行政局级机构（含机关党委）<ref name=yongan/><ref>{{Cite web |url =  http://www.chinadaily.com.cn/zgzx/zx60nian/2009-09/14/content_8688482.htm |title = 中国人民政治协商会议全国委员会的工作机构组织结构图 |author =  |date =  2009-09-14|publisher =中国网  |language =  |accessdate =  |quote =  }}</ref>。

2014年12月，全国政协机关党委作为中直机关开展述职试点工作的5家试点单位之一，向[[中共中央直属机关工作委员会|中直工委]]领导班子述职。2015年2月13日，全国政协机关召开机关基层党组织向机关党委述职试点工作会议，全国政协机关党组书记、常务副秘书长[[孙怀山|孙怀山]]肯定了机关党委和基层党组织述职试点工作，会上[[全国政协办公厅秘书局|秘书局]]、[[全国政协经济委员会办公室|经济委员会办公室]]、[[人民政协报|人民政协报社]]党组织负责人作为代表向全国政协机关党委述职<ref name=shuzhi>{{Cite web |url =  http://www.xinhuanet.com/politics/2015-02/14/c_127494779.htm |title =全国政协机关举行基层党组织 向机关党委述职试点工作会议  |date =2015-02-14  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

2017年11月17日至18日，中国共产党全国政协机关第十一次代表大会召开，[[中共中央直属机关工作委员会|中直工委]]副书记[[李勇|李勇]]出席闭幕会，[[全国政协机关党组|全国政协机关党组]]成员、副秘书长、机关党委书记、[[全国政协办公厅研究室|办公厅研究室]]主任[[舒启明|舒启明]]主持闭幕会并在开幕会上代表第十届机关党委作工作报告，大会选举出第十一届全国政协机关党委和机关纪委<ref name=shiyi>{{Cite web |url = http://www.china.com.cn/cppcc/2017-11/20/content_41914727.htm |title =  中国共产党全国政协机关第十一次代表大会召开|author =  |date =2017-11-20  |publisher =  中国网|language =  |accessdate =  |quote =  }}</ref>。
