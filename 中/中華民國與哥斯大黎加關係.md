'''中華民國與哥斯大黎加關係'''是指[[中華民國|中華民國]]與[[哥斯大黎加共和國|哥斯大黎加共和國]]之間的關係。1941－2007年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對哥斯大黎加的相關事務由'''中華民國駐[[尼加拉瓜共和國|尼加拉瓜共和國]][[中華民國駐外機構列表|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-153-d7ffa-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===

{{see also|中華民國駐哥斯大黎加大使列表{{!}}中華民國駐哥斯大黎加外交官列表|哥斯大黎加駐中華民國大使列表{{!}}哥斯大黎加駐中華民國外交官列表}}
1941年4月9日，中華民國與哥斯大黎加建立公使級[[外交#外交上的承認|外交關係]]，於首都[[聖荷西_(哥斯大黎加)|聖荷西]]設立'''中華民國駐哥斯大黎加共和國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]（由[[中国驻巴拿马大使列表|駐巴拿馬公使]]兼任）。<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2007/html/08-1-6.htm |title = 《中華民國96年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1954年，於首都[[台北市|台北]]派任'''哥斯大黎加駐中華民國名譽[[領事|領事]]'''。1959年，設立'''哥斯大黎加駐中華民國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。<ref name="外交年鑑"/>

1959年5月5日，公使館升格為'''中華民國駐哥斯大黎加共和國大使館'''，並派駐大使。<ref name="外交年鑑"/>

1996年1月，哥斯大黎加前總統{{link-en|孟赫|Luis Alberto Monge}}、{{link-en|艾張第|Mario Echandi Jiménez}}、{{link-en|多瑞荷|José Joaquín Trejos Fernández}}、{{link-en|喀德戎|Rafael Ángel Calderón Fournier}}聯名致函聯合國秘書長[[布特羅斯·布特羅斯-加利|蓋里]]支持[[中國與聯合國#台灣人民代表權問題及台灣入聯運動|中華民國參與聯合國]]，由哥斯大黎加[[常駐聯合國代表|常駐聯合國代表]]轉交蓋里。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=F2F18E0B80A28CC1 |author = 公眾外交協調會 |date = 1996年2月19日 |title = 哥斯大黎加前總統孟赫（Luis Alberto Monge）聯合哥國前總統艾張第（Mario Echandi） |publisher = 中華民國外交部}}</ref>

1996年3月，{{link-en|哥斯大黎加政府|Politics of Costa Rica}}與[[哥斯达黎加共和国立法大会|國會]]對中華人民共和國[[台灣海峽飛彈危機|在台灣海峽試射飛彈]]，威脅中華民國即將於3月23日舉行的[[1996年中華民國總統選舉|首次總統直接選舉]]一事，深表憤慨並發表公報譴責。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=C326C156F43A2893 |author = 公眾外交協調會 |date = 1996年3月11日 |title = 哥斯大黎加政府對中共自本月八日起在台灣近海試射飛彈深表憤慨並於同日發表公報譴責中共以武力脅迫我國 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=657916241843B29E |author = 公眾外交協調會 |date = 1996年3月14日 |title = 哥斯大黎加國會副議長及其他二十一位議員於三月十一日針對中共在台灣海峽舉行軍事演習對我進行武嚇之舉動共同提案譴責中共 |publisher = 中華民國外交部}}</ref>哥斯大黎加並聯合[[薩爾瓦多|薩爾瓦多]]、[[瓜地馬拉|瓜地馬拉]]、[[宏都拉斯|宏都拉斯]]、[[尼加拉瓜|尼加拉瓜]]總統及[[貝里斯|貝里斯]]總理共同發表公報予以聲援。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=5B6A37274C59E4B3 |author = 公眾外交協調會 |date = 1996年3月20日 |title = 哥斯大黎加、薩爾多瓦、瓜地馬拉、宏都拉斯、尼加拉瓜總統及貝里斯總理 |publisher = 中華民國外交部}}</ref>

2007年6月7日，在建交66年後，哥斯大黎加與中華民國斷交。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=70BCE89F4594745D&sms=700DE7A3F880BAE6&s=EDB3FDC03972FC58 |author = 公眾外交協調會 |date = 2007年6月7日 |title = 中華民國（台灣）政府宣布與哥斯大黎加共和國終止外交關係新聞說明會紀要 |publisher = 中華民國外交部}}</ref>當時有9名哥斯大黎加學生在台灣學習[[中華民國國語|中文]]並攻讀[[學士學位|學士學位]]，中華民國政府在兩國斷交後仍繼續提供[[獎學金|獎學金]]以完成學業。<ref name="外交年鑑"/>

===貿易===
2018年，哥斯大黎加是中華民國第73大貿易夥伴、第68大[[進口|進口]]夥伴、第76大[[出口贸易|出口]]夥伴。出口至哥斯大黎加的金額為8,775萬7,448[[美元|美元]]，年增9.246%；自哥斯大黎加進口的金額為9,437萬8,068美元，年增33.240%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）662萬0,620美元，年減169.711%。2017年，哥斯大黎加是中華民國第78大貿易夥伴、第73大進口夥伴、第80大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
[[臺灣商人|台商]]在哥斯大黎加總投資金額為9,489萬美元，僱用員工人數約743人。主要投資項目有：[[加工出口區|加工區]]經營、[[漁業|漁撈]]、[[塑膠|塑膠]]、[[家具|家具]]、[[食物|食品]]、[[汽車零組件列表|汽車零件]]、[[蘭科|蘭花]]栽培、[[旅館|旅館]]、[[廚房#設備|廚具設備]]、[[电动门|鐵捲門]]、[[貿易|貿易]]等，大部分屬[[中小型企業|中小型企業]]。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/cr/pdf/103-35CostaRica.pdf |date = 2014年12月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

較大型投資包括：中華加工出口區（BES）設立於1991年，投資金額約1,200萬美元，目前有廠商20家，其中1家為台商，在該國已建立相當知名度；台糖蘭園（[[台灣糖業公司|台灣糖業公司]]，TAISUCO）設立於1998年，投資金額600萬美元，位於中華加工出口區內，目前擁有[[中美洲|中美洲]]規模最大、最現代化[[蝴蝶蘭屬|蝴蝶蘭]][[溫室|溫室]]，培育蝴蝶蘭外銷[[北美洲国家和地区列表|北美洲]]、[[拉丁美洲|中南美洲]]、[[歐洲國家和地區列表|歐洲]]市場；以及由旅哥[[海外華人|華人]]及哥國人士共同集資設立的國泰商業銀行（並非[[國泰世華商業銀行|國泰世華商業銀行]]海外分行），資本額1,200萬美元。<ref name="投資環境簡介"/>

目前成立的組織有：哥斯大黎加台灣商會、哥斯大黎加台灣協會、旅哥斯大黎加中華總會。<ref name="投資環境簡介"/>
