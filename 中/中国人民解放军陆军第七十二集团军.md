'''中国人民解放军陆军第七十二集团军'''，隶属[[中国人民解放军东部战区陆军|东部战区陆军]]，军部驻地[[浙江省|浙江省]][[湖州市|湖州市]]。

==沿革==
1930年，[[贺龙|贺龙]]、[[周逸群|周逸群]]等人在[[湘鄂西苏区|湘鄂西]]创建[[红二军团|红二军团]]。[[第一次国共内战|土地革命战争]]时期，红二军团在[[中国共产党|中国共产党]]的领导下，创立了以洪湖为中心的[[湘鄂西苏区|湘鄂西苏区]]，并在战争中总结出在河湖港汊地区开展游击战的经验，受到[[毛泽东|毛泽东]]赞扬。在[[二万五千里长征|二万五千里长征]]中，红二军团自[[湖南省|湖南省]][[桑植县|桑植县]]刘家坪出发，突破了敌军130余个团的包围，先后经湖南、[[贵州|贵州]]、[[云南|云南]]、[[四川|四川]]，于1936年7月与[[红四方面军|红四方面军]]在[[四川省|四川省]][[甘孜县|甘孜县]]会师。随后部队继续北上，跨过草地，翻越[[六盘山|六盘山]]，于同年10月与[[中央红军|中央红军]]在[[甘肃省|甘肃省]][[会宁县|会宁县]]会师<ref name=jianshi>{{Cite web |url =  http://history.news.163.com/07/0327/08/3AJ15PJU00011243.html|title = 中国人民解放军陆军集团军简史 |author =  |date =2007-03-27  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref name=guan>{{Cite web |url = http://zjnews.zjol.com.cn/05zjnews/system/2011/08/15/017768385.shtml |title =  73011部队军史陈列馆|author =  |date = 2011-08-15 |publisher = 浙江在线 |language =  |accessdate =  |quote =  }}</ref>。

1937年7月[[抗日战争|抗日战争]]全面爆发，根据[[中共中央|中共中央]]命令，该部队改编为[[八路军|八路军]][[120师|一二〇师]]三五八旅，开赴华北抗日前线，先后开辟了[[晋西北抗日根据地|晋西北抗日根据地]]、[[大青山抗日根据地|大青山抗日根据地]]。[[晋绥边区|晋绥边区]]便是由该部队开创。1939年11月，贺龙特别团在[[黄土岭围歼战|黄土岭伏击战]]中，击毙了日军中将旅团长[[阿部规秀|阿部规秀]]。此后部队回师晋西北保卫[[陕甘宁边区|陕甘宁边区]]，并参加了抗日战争末期的大反攻，取得了抗日战争的胜利<ref name=jianshi/><ref name=guan/>。

[[第二次国共内战|第二次国共内战]]中，该部队大部被编入[[张宗逊|张宗逊]]、[[廖汉生|廖汉生]]领导的'''[[西北野战军|西北野战军]]第一纵队'''和[[许光达|许光达]]、[[孙志远|孙志远]]领导的西北野战军第三纵队。在转战[[陕北|陕北]]的过程中，先后参加了[[延安保卫战|延安保卫战]]、[[青化砭战役|青化砭战役]]、[[羊马河战役|羊马河战役]]、[[蟠龙战役|蟠龙战役]]、[[陇东战役|陇东战役]]、[[榆林战役|榆林战役]]等战役。1948年春在[[宜川战役|宜川战役]]中，第一纵队在瓦子街与敌军白刃格斗，涌现出拼刺英雄[[刘四虎|刘四虎]]（第三五八旅第七一四团第六连第二班班长）。1949年2月1日，西北野战军第一纵队整编为'''中国人民解放军第一军'''，军长[[贺炳炎|贺炳炎]]，政委[[廖汉生|廖汉生]]。下属第三五八旅（旅长[[黄新廷|黄新廷]]，政委[[余秋里|余秋里]]）、独立第一旅（旅长[[王尚荣|王尚荣]]）、第七旅分别改称第一、第二、第三师。全军共2.2万人。随后第一军参加了[[陕中战役|陕中战役]]、[[扶眉战役|扶眉战役]]、[[陇东追击战|陇东追击战]]、攻占青海。1949年10月1日，第一军军部奉命兼[[青海省军区|青海军区]]领导机关。在解放战争期间，第一军共歼敌104227人，攻占县以上城市51座<ref name=jianshi/><ref name=guan/>。

[[中华人民共和国|中华人民共和国]]成立后，部队驻防[[青海省|青海省]]、[[甘肃省|甘肃省]]，剿匪肃特，抢修[[青藏公路|青藏公路]]、宁张公路及飞机场，参加[[昌都战役|昌都战役]]，促进了[[西藏和平解放|西藏和平解放]]。1952年6月，第一军与第三军合编为重装军，番号仍为第一军，第一、第三师合编为第一师，第二、第八师合编为第二师，第七、九师合编为第七师。军长黄新廷<ref name=jianshi/><ref name=guan/>。

1953年1月，整编后的第一军分两部跨过[[鸭绿江|鸭绿江]]，加入[[中国人民志愿军|中国人民志愿军]]，参加[[抗美援朝战争|抗美援朝战争]]，番号'''中国人民志愿军第一军'''。停战后留守朝鲜，直到1958年撤军回国，驻防[[河南省|河南省]][[开封市|开封市]]，恢复'''中国人民解放军第一军'''番号，隶属[[武汉军区|武汉军区]]<ref name=jianshi/><ref name=guan/>。1960年4月19日，[[中国人民解放军总参谋部|中国人民解放军总参谋部]]下发《关于修改某些机构、职务名称事的通知》，改称'''中国人民解放军陆军第一军'''。1962年，第一军曾开赴[[福建省|福建省]]执行紧急战备任务。1964年1月，[[中华人民共和国国防部|中华人民共和国国防部]]授予第一军第一师第一团第六连“硬骨头六连”称号。1969年全军重排番号，第七师改称第三师。1975年第一军与隶属[[南京军区|南京军区]]的[[中国人民解放军陆军第二十集团军|第二十军]]对调，换防至[[浙江省|浙江省]]。1984年7月，军长[[傅全有|傅全有]]率领第一军参加[[两山轮战|老山地区防御作战]]，1985年6月,“硬骨头六连”又被[[中央军事委员会|中央军委]]授予“英雄硬六连”称号<ref name=jianshi/><ref name=guan/>。

1985年，在[[百万大裁军|百万大裁军]]中，陆军第一军改编为'''中国人民解放军陆军第一集团军'''（原代号为中国人民解放军83010部队，后代号改为中国人民解放军73011部队），隶属[[南京军区|南京军区]]，下辖第一、二、三师及原第六十军第一八一师共计4个步兵师，此外还编入坦克第十师、炮兵第九师（1992年并入该集团军）和高炮旅。1987年该军组建了中国人民解放军全军第一个“蓝军团”。1990年代中期以后，全军编制体制重大调整。原属第一集团军的步兵第二、第一八一师于1996年改为[[武警|武警]]机动师。1998年后，步兵第三师改为摩步旅，步兵第一师改编为中国人民解放军全军第一个水陆两栖机械化步兵师，坦克第十师改编为装甲师，高炮旅改编为防空旅<ref name=jianshi/>。

2016年，中国人民解放军七大军区撤销，成立五大战区，陆军第一集团军划归[[中国人民解放军东部战区陆军|中国人民解放军东部战区陆军]]<ref>{{cite news|title=媒体:五大战区成立一月 18个集团军分属哪里|url=http://news.sina.com.cn/c/nd/2016-02-28/doc-ifxpvysv4979300.shtml|accessdate=2016-02-29|agency=法制晚报|date=2016-02-28}}</ref>。

==编制==
中国人民解放军陆军第一集团军在1996年前辖有第一、第二、第三师及原第六十军一八一师共计4个步兵师，此外还有坦克第十师、炮兵第九师和高炮旅。1987年，该集团军组建了中国人民解放军的第一个“蓝军团”。1990年代中期后，中国人民解放军的编制体制进行了重大调整。1996年，原属陆军第一集团军的步兵第二、第一八一师被改编为[[中国人民武装警察部队|中国人民武装警察部队]]机动师（武警第二师、武警第一八一师）。1998年后，所辖步兵第三师被改编为[[摩托化步兵|摩托化步兵]]旅，第一师被改编为中国人民解放军第一个水陆[[海軍陸戰隊|两栖]]机械化步兵师，坦克第十师与高炮旅也分别被改编为两栖装甲旅、机步旅和防空旅。全集团军约5万人左右。2016年，摩托化步兵第三旅开始向空中骑兵旅转型<ref name=zhuanxing/>。
