'''中国人民解放军空军青岛航空医学鉴定训练中心'''，位于[[山东省|山东省]][[青岛市|青岛市]][[市南区|市南区]]岳阳路18号，隶属[[中国人民解放军北部战区空军|中国人民解放军北部战区空军]]后勤部。

==简介==
空军青岛航空医学鉴定训练中心的前身是始建于1950年2月的'''中国人民解放军空军青岛疗养院'''，这是[[中国人民解放军空军|中国人民解放军空军]]第一所疗养院，担负全军航天员、飞行员及军职以上首长疗养、体检鉴定、训练任务<ref name=zhongxin>使命在肩——空军青岛航空医学鉴定训练中心，青岛画报2014(10)</ref><ref>{{Cite web |url =http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536777.htm  |title =空军青岛航空医学鉴定训练中心  |author =  |date = 2014-09-12 |publisher =  中华人民共和国国防部|language =  |accessdate =  |quote =  }}</ref>。

2009年7月，都江堰、杭州、大连、青岛、临潼五所疗养院统一由各大军区联勤部转隶各大军区空军后勤部，成立5个航空医学鉴定训练中心，构建为“[[中国人民解放军空军总医院|空军总医院]]、[[中国人民解放军空军航空医学研究所|空军航空医学研究所]]——区域性航空医学鉴定训练中心——航空兵部队卫生机构”三级航空医学鉴定训练体系<ref>何强、彭天国、董文，空军航空医学鉴定训练中心的发展与思考 ，全国航空航天医学学术会议暨云南省航空航天医学学术会议2010,27(3):180-181</ref><ref>{{Cite web |url = http://news.xinhuanet.com/mil/2011-12/25/c_111295752.htm |title = 一场从陆地打响的空战 |author =  |date = 2011-12-25 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。其中，2009年9月29日，'''中国人民解放军空军青岛航空医学鉴定训练中心'''正式组建。该中心拥有岳阳路、荣成路、第二海水浴场、第三海水浴场四个区域<ref name=zhongxin/>。该中心隶属[[济南军区空军|济南军区空军]]后勤部。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，转隶[[中国人民解放军北部战区空军|中国人民解放军北部战区空军]]后勤部。
