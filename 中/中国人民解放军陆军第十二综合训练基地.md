'''中国人民解放军陆军第十二综合训练基地'''，位于[[重庆市|重庆市]][[铜梁区|铜梁区]]，隶属[[中国人民解放军西藏军区|中国人民解放军西藏军区]]。

==沿革==
2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，调整组建'''中国人民解放军陆军第十二综合训练基地'''，位于[[重庆市|重庆市]][[铜梁区|铜梁区]]<ref>{{Cite web |url = http://www.yidianzixun.com/article/0I86zAYw |title = 重庆市铜梁区第十七届人民代表大会第二次会议举行第二次全体会议 |author =  |date =2018-01-13 |publisher = 一点资讯 |language =  |accessdate =  |quote =  }}</ref>。根据[[中国人民解放军陆军|中国人民解放军陆军]]《2017年新兵训练总体方案》和[[中国人民解放军西藏军区|中国人民解放军西藏军区]]《2017年度新兵训练指示》，自2017年起，该训练基地承担西藏军区部分单位和基地本级数千名新兵的新训任务<ref name=xinxun>{{Cite web |url = https://www.sohu.com/a/212538384_600540 |title =陆军某部新兵“基地化”训练成果怎么样？答案全都在这里……  |author =  |date =2017-12-24   |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
