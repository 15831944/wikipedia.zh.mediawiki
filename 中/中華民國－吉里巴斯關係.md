'''中華民國－吉里巴斯關係'''是指[[中華民國|中華民國]]與[[吉里巴斯共和國|吉里巴斯共和國]]之間的外交關係。

==歷史==
*2003年11月7日 中華民國與吉里巴斯建立大使級外交關係<ref>{{cite news|url=http://www.mofa.gov.tw/News_Content_M_2.aspx?n=5028B03CED127255&sms=5ED24855AD8E6C58&s=99E543473035938F|title=中華民國政府與吉里巴斯共和國政府基於平等互惠之原則暨發展兩國友好關係之共同意願，決定自二○○三年十一月七日起建立大使級外交關係|publisher=[[中華民國外交部|中華民國外交部]]|date=2003-11-07|accessdate=2015-11-02}}</ref>。

*2004年1月10日 中華民國在[[塔拉瓦|塔拉瓦]]百里基(Bairiki)設置大使館。<ref name="ROC Foreign Relation Yearbook 2016">{{cite book|title=中華民國外交部105年外交年鑑-第三章對外關係|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-11 |accessdate=2018-04-17}}</ref>

*2013年5月31日 吉里巴斯在[[臺北|臺北]]設立大使館。<ref name="ROC Foreign Relation Yearbook 2013">{{cite book|title=中華民國外交部102年外交年鑑-第三章對外關係|publisher=[[中華民國外交部|中華民國外交部]]|date = 2014-12 |accessdate=2018-04-17}}</ref>

==兩國貿易==
2018年吉里巴斯是中華民國第177名貿易伙伴，兩國貿易額1,746,435美元，其中中華民國對吉里巴斯出口1,721,854美元，是第165大出口國，進口24,581美元，是第211大進口國，與2017年貿易額相比貿易總額約減少41.79%、出口約減少42.28%、進口約增加45.97%。<ref>{{cite book |url = http://cus93.trade.gov.tw/FSCI/ |title = 經濟部國際貿易局貿易統計系統 |publisher = [[經濟部國際貿易局|經濟部國際貿易局]] |date = 2019-02-23 |accessdate = 2019-03-05}}</ref>
*中華民國－吉里巴斯主要出口項目：燈具及照明配件、二極體、電晶體及類似半導體裝置、光敏半導體裝置、腳踏車、其他鋼鐵製品等。<ref name=bft>{{Cite web |url = https://www.trade.gov.tw/World/Detail.aspx?nodeID=45&pid=581418|title = 各國經貿資訊 |publisher =[[經濟部國際貿易局|經濟部國際貿易局]]|date = 2018-02-08|accessdate = 2018-03-04}}</ref>
*中華民國－吉里巴斯主要進口項目：冷凍魚、捲筒或平版紙及紙板等。<ref name=bft>{{Cite web |url = https://www.trade.gov.tw/World/Detail.aspx?nodeID=45&pid=581418|title = 各國經貿資訊 |publisher =[[經濟部國際貿易局|經濟部國際貿易局]]|date = 2018-02-08|accessdate = 2018-03-04}}</ref>

===航空客運===
兩國目前無直航班機，來往吉里巴斯之客運航線有以下方式：

1.可搭乘飛往{{flagicon|AUS}}[[布里斯班机场|布里斯本]]班機。

2.轉機即可到達{{flagicon|KIR}}[[南塔拉瓦|南塔拉瓦]]-[[邦里基國際機場|邦里基國際機場]]。<ref>{{cite web|url=http://www.airkiribati.com.ki/tarawa2brisbane |title=Air Kiribati Brisbane to Tarawa |publisher=[[吉里巴斯航空|吉里巴斯航空]] |date= |accessdate=2017-11-05}}</ref>

＊上述目的地國際機場為鄰近該國首都或該國最主要樞紐機場，若要前往該國其他國際機場或國內線機場詳見{{link-en|吉里巴斯機場列表|List of airports in Kiribati|吉里巴斯機場}}。

＊航線會因航空公司營運狀況更改或取消，出發前應與負責該航線的航空公司確認航線及航班，以免影響權益。
