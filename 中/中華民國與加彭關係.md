'''中華民國與加彭關係'''是指[[中華民國|中華民國]]與[[加彭共和國|加彭共和國]]之間的關係。兩國於1960－1974年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對加彭的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''、'''[[駐歐盟兼駐比利時代表處|駐歐盟兼駐比利時代表處]]'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-239-d0062-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
{{see also|中華民國外交|{{link-en|加彭外交|Foreign relations of Gabon}}}}
{{see also|中国驻加蓬大使列表{{!}}中華民國駐加彭外交官列表|加蓬驻华大使列表{{!}}加彭駐中華民國外交官列表}}
1960年12月9日，建立大使級[[外交#外交上的承認|外交關係]]。

1965年5月，於首都[[自由市|自由市]]設立'''中華民國駐[[加彭共和國|加彭共和國]][[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1974年3月30日，與中華民國斷交。

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|加彭簽證政策|Visa policy of Gabon}}|{{link-en|加彭公民簽證要求|Visa requirements for Gabonese citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可至該國移民局的[[簽證#电子签证|電子簽證]]申請系統取得入境許可後，於抵達首都的[[利伯維爾國際機場|利伯維爾國際機場]]辦理[[簽證#落地签证|落地簽證]]入境，單次簽證停留最多1－3個月；多次簽證停留最多6個月。入境須出示[[黃熱病|黃熱病]][[疫苗接種|疫苗接種]]證明（黃皮書）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-239-d0062-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ElectronicVisa.aspx?n=6F246645F10311D7&AreaSN=D33B55D537402BAA |title = 電子簽證資訊 |publisher = 中華民國外交部}}</ref>

持[[加彭護照|加彭護照]]的{{link-en|加彭人口|Demographics of Gabon|加彭公民}}抵台參加國際會議、運動賽事、商展活動，也可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將加彭歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2016年11月4日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-55-50981-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將加彭歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=BE2E7DD3CCF9DC68 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

{{see also|臺灣經濟|{{link-en|加彭經濟|Economy of Gabon}}}}
===貿易===
2017年，加彭是中華民國第160大貿易夥伴、第131大[[進口|進口]]夥伴、第161大[[出口贸易|出口]]夥伴。出口至加彭的金額為189萬4,681[[美元|美元]]，年增40.676%；自加彭進口的金額為358萬9,200美元，年增219.134%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）169萬4,519美元，年減862.702%。※2016年，加彭是中華民國第175大貿易夥伴、第156大進口夥伴、第167大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
