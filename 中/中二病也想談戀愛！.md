== 概要 ==
本作品為第一屆[[京都動畫大賞|京都動畫大賞]]輕小說部門之優秀作品，獲獎的評語是「文中的著眼點和想法都令人覺得很有趣」。得獎作品於2011年6月1日出版，隨後亦有推出續集，全4卷。

作者在後記表示，全書有不少[[劇情透露|捏他]]是向《[[涼宮春日的憂鬱|涼宮春日的憂鬱]]》[[致敬|致敬]]。

動畫版除了最基本的角色設定外，其餘部分與原著小说的劇情和人物背景并不相同。

== 故事簡介 ==
;原作
:'''富樫勇太'''於[[初級中學|初中]]畢業的時候，決定同時為自己舉辦[[中二病|中二病]]畢業的儀式。打算順利度過高中生活的勇太，在開學日以最後的中二語錄完全封鎖這段[[黑历史|黑历史]]。可惜天意弄人，勇太遇上了現任[[中二病|中二病患者]]暨同班同學'''小鸟遊六花'''，更被六花看出[[中二病|中二病]]的殘餘症狀。六花於是以此威逼勇太與她「立下契约」。六花的行為舉止惹來了勇太的不少同學、家人、甚至[[青梅竹馬|青梅竹馬]]的注目，使勇太在享受[[現充|現充]]生活時充滿着難言之隱……

;動畫第二季《中二病也想談戀愛！戀》
:因為發生些問題失去住所的六花開始尋找新家，在一直都不順利的情況之下，只好搬到目前只有勇太一個人居住的富樫家，但此時樟葉為了迎考而回到了富樫家。總之，勇太與六花最後還是連同樟葉一起展開了有點奇妙的共同生活。另一方面，原本的小鳥遊家搬來了一位新的鄰居，她就是過去勇太所憧憬的少女——自稱魔法魔王少女「蘇菲婭寧·SP·[[薩圖爾努斯|薩圖爾努斯]]7世」的七宮智音。就這樣，仍然完全稱不上普通的勇太高二第一學期開始了。

== 電視動畫 ==
2011年12月28日，京都動畫在中二病也想談戀愛！的官方網頁上公開宣佈動畫化的宣傳片。

2012年7月10日，正式公開動畫版網頁，並宣佈為TV動畫及其製作人員，播放時間在10月3日至12月19日在[[東京都會電視台|TOKYO MX]]、[[SUN電視台|SUN電視台]]、[[京都放送|京都放送]]、[[愛知電視台|愛知電視台]]、[[日本BS放送|BS11]]等[[全國獨立放送協議會|獨立電視網]]。全12話+電視未播放1話，由[[大塚芳忠|大塚芳忠]]擔任第1話至第12話的旁白。每話的下回預告於逢星期一16時00分（日本時間）在[[YouTube|YouTube]]的京都动画官方频道及动画官方网站上配信。

2013年3月28日，在[[玉子市場|玉子市場]]最後一話放送完畢後，發表作品「契約更新」的廣告。在同日也更新[https://twitter.com/anime_chu_2/ 官方Twitter]和[http://www.anime-chu-2.com/tv 官方網頁]。

2013年5月8日，在官方Twitter中宣布5月9日的深夜，官方網頁預訂將會有更新。而在日本著名討論區2CH的[http://ikura.2ch.net/test/read.cgi/anime2/1366814203/ 該動畫的討論串]，以及一[https://twitter.com/RikkaTakanashi/status/332064959271952387 非官方Twitter]放出可能是2013年6月號的動畫雜誌[[Newtype_(雜誌)|Newtype]]中的一張該動畫的海報，指出此更新可能是宣布第二季製作決定。

2013年5月10日，[[日本|日本]]時間當天凌晨零時正<ref group="注">在日本亦可寫作2013年5月9日24:00。</ref>，[http://www.anime-chu-2.com/tv 官方網頁]更新，正式宣布第二季製作決定。官方網頁的背景被替換成二季製作決定的圖片。

2013年9月14日，於日本上映最新電影《小鳥遊六花・改 ～劇場版 中二病也想談戀愛！～》。

==劇場版==
===小鳥遊六花·改 ～劇場版 中二病也想談戀愛！～===
2013年6月9日，在官方Twitter宣布將於9月14日，在全日本範圍內的27個戲院公映其劇場版「'''小鳥遊六花·改 ～劇場版 中二病也想談戀愛！～'''」，內容主要是第一季的總集篇，而劇場版三分之一為新作劇情，包含涉及到第2期的過渡劇情<ref>{{cite web|url=http://www.cinematoday.jp/page/N0053677 |title=アニメ「中二病でも恋がしたい!」が映画化！新作も含む総集編に - シネマトゥデイ|publisher=Cinematoday|accessdate=2013年12月8日|language=ja}}</ref>。2014年2月19日，發行劇場版的[[藍光光碟|BD]]及[[DVD|DVD]]。

====製作人員====
* 原作：虎虎「中二病也想談戀愛！」（KA Esuma文庫／京都動畫）
* 監督：[[石原立也|石原立也]]
* 劇本：[[花田十輝|花田十輝]]
* 人物設定、總作画監督：[[池田和美|池田和美]]
* 角色原案：逢坂望美
* 美術監督：篠原睦雄
* 色彩設定：竹田明代
* 攝影監督：山本倫
* 設定：高橋博行
* 音響監督：[[鶴岡陽太|鶴岡陽太]]
* 音樂：[[虹音|虹音]]
* 動畫製作：[[京都動畫|京都動畫]]
* 製作：中二病也想製作委員會
* 发行：[[松竹|松竹]]

==== 主題曲 ====
; 片頭曲「－Across the line－」
: 作詞、作曲、編曲：ZAQ，主唱：小鳥遊六花（[[内田真禮|内田真禮]]）
; 片尾曲「Secret Survivor」
: 作詞、作曲、編曲：ZAQ，主唱：Black Raison d'être（小鳥遊六花、丹生谷森夏、五月七日茴香、凸守早苗）
; 片尾曲「INSIDE IDENTITY」
: 作詞、作曲、編曲：ZAQ，主唱：Black Raison d'être（小鳥遊六花、丹生谷森夏、五月七日茴香、凸守早苗）
; 劇場版完全新作Lite主題曲「{{lang|ja|深淵に舞う戦慄謝肉祭}}」
: 主唱：ZAQ
