'''中華民國與以色列關係'''是指[[中華民國|中華民國]]與[[以色列國|以色列國]]雙邊的關係。兩國至今並未有正式[[外交#外交上的承認|外交關係]]，但於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
{{see also|中華民國外交|以色列外交}}
[[中華民國政府|中華民國政府]]對[[錫安主義|猶太復國主義]]一直持同情和支持態度，<ref name="The Republic of China and Israel, 1911-2003"/>1949年3月初，中華民國政府正式[[外交#外交上的承認|承認]]以色列，在[[聯合國大會|聯合國大會]]上投票贊成接納以色列為[[聯合國成員國列表|會員國]]，允許以色列向[[中華民國|中國]]派出一名[[領事|領事]]駐節[[上海|上海]]，處理[[猶太人|猶太人]][[難民|難民]]向以色列的轉移問題。<ref name="Note of the Israel Embassy">{{Cite book |author = Note of the Israel Embassy in Washington DC to Republic of China Embassy, 16 November 1948 |title = Israel State Archiv es (ISA), Foreign Ministry (FM) file 2554/13, 1948-1965}}</ref>但不久，[[國府遷台|國民政府遷往臺灣]]，以色列駐上海領事{{link-en|伊爾坦|Walter Eytan}}提出：「根據我國的承認政策，我們的承認必須建立在務實的基礎之上.....毫無疑問，[[中華人民共和國政府|共和國政府]]是控制[[中國大陸|大陸]]的有效政府。」<ref name="The First Ten Years: A Diplomatic History of Israel">{{Cite book |author = Walter Eytan |title = '''''The First Ten Years: A Diplomatic History of Israel''''' | location = New York |publisher = Simon and Schuster |date = 1958}}</ref>這一思想奠定了以色列對華政策的大方向。

1950年1月9日，以色列正式承認了新建立的[[中華人民共和國|中華人民共和國]]，成為[[中東|中東]]地區第一個承認[[中國共産黨|中共]]政權的國家。<ref name="The Republic of China and Israel, 1911-2003">{{Cite book |author = Jonathan Goldstein |title = The Republic of China and Israel, 1911-2003 |publisher = Frank Cass |date = 2003}}</ref>。1959年，以色列向該國駐[[聯合國|聯合國]]大使指示，「儘管我們與[[红色中国|共產黨中國]]的交往不順，但也要盡量避免與[[台灣|台灣]]的[[中國國民黨|國民黨]]政權來往。」<ref name="Israel's China Policy 1950-1992">{{Cite book | author = E.Zev Sufott |title = Israel's China Policy 1950-1992. '''''Israel Affairs''''' |date = 2000 |volume = 7 |issue = 1}}</ref>不過，中華人民共和國並未與以色列建立外交關係，在[[冷戰|冷戰]]時期，基於反對[[帝國主義|帝國主義]]及[[輸出革命|輸出革命]]的路線，中共一直支持[[阿拉伯國家|阿拉伯國家]]對抗以色列。在一段長期間內，中共和以色列維持敵對關係，直至1992年中共和以色列才建立外交關係。但即使如此，以色列並無與中華民國建立外交關係。基於兩國都是美國的盟友，美國一直對兩國提供軍事和經濟支持。

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[以色列|以色列]]。]]
1971年10月25日，以色列投票反對[[中國與聯合國#代表權問題|中華民國續留聯合國]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。

1992年1月24日，以色列與中華人民共和國建立外交關係，中華民國政府表示關切並申明「以色列與中共在《建交公報》中所提及任何涉及我國主權及各權利事項，將一概不予承認。對於未來中以兩國之關係，我將基於國家利益慎加衡慮，再作定奪。」<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=340F42E1360F9916 |author = 公眾外交協調會 |date = 1992年1月24日 |title = 對於以色列與中共於八十一年元月廿四日正式建立外交關係事 |publisher = 中華民國外交部}}</ref>

1995年11月4日，以色列總理[[伊扎克·拉宾|拉賓]]遇刺身亡。5日，[[中華民國外交部|中華民國外交部]]發表聲明表示極感震悼。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=BD4579AC07D69D83 |author = 公眾外交協調會 |date = 1995年11月5日 |title = 中華民國外交部對以色列拉賓總理被刺身亡極感震悼 |publisher = 中華民國外交部}}</ref>

1999年9月21日，[[921大地震|台灣發生921大地震]]，以色列總統[[埃泽尔·魏茨曼|魏茨曼]]發表聲明表達關切，並於30日援贈醫療與嬰兒用品，供災民使用。<ref name="88年外交年鑑"/>

2002年3月，第二屆[[台灣|台灣]]、[[美國|美國]]、[[以色列|以色列]]三邊國防安全研討會在以色列舉行，邀集三方學者就[[外交|外交]]、[[國防|國防]]及[[戰略|戰略]]等共同關切之議題進行研討，並建立第二軌道對話。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__02.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2012年6月13日，台灣[[雲門舞集|雲門舞集]]在以色列[[特拉維夫表演藝術中心|特拉維夫歌劇院]]演出《屋漏痕》，榮譽團長、中華民國第一夫人[[周美青|周美青]]出席並參與演後酒會。<ref>{{Cite web |url = http://www.cloudgate.org.tw/news/show/220 |author = 何定照 |date = 2012年6月15日 |title = 雲門以國演出「屋漏痕」全場驚艷 |publisher = [[雲門舞集|雲門舞集]]}}</ref>

2019年2月，台北市長[[柯文哲|柯文哲]]前往以色列[[耶路撒冷|耶路撒冷]]，除了做為唯一受邀的華人市長參加「全球市長會議」外，並參訪自駕車公司[[Mobileye|Mobileye]]、[[哭牆|哭牆]]、[[以色列猶太大屠殺紀念館|猶太人大屠殺紀念館]]、{{link-en|以色列創新局|Israel Innovation Authority}}、[[魏茨曼科學研究學院|魏茨曼科學研究學院]]，以及會見臺拉維夫市長{{link-en|胡爾代|Ron Huldai}}，兩位市長相見時，桌上並放著[[中華民國國旗|中華民國國旗]]與[[以色列國旗|以色列國旗]]。原訂和其他參加會議的市長，一起會見以色列總理[[本雅明·内塔尼亚胡|納坦尼雅胡]]，因該國總理變更行程而取消。<ref>{{Cite web |url = https://tw.sports.appledaily.com/life/realtime/20190222/1521323/ |author = 綜合報導 |date = 2019年2月22日 |title = 柯Ｐ明訪以國 今年唯一華人市長參加全球市長會議 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》}}</ref><ref>{{Cite web |url = https://tw.appledaily.com/new/realtime/20190225/1523182/ |author = 張博亭 |date = 2019年2月25日 |title = 【以色列取經】被以國總理放鴿子 柯P：對方急飛莫斯科見普丁 |publisher = 《蘋果日報》}}</ref><ref>{{Cite web |url = https://tw.appledaily.com/new/realtime/20190226/1523763/ |author = 張博亭 |date = 2019年2月26日 |title = 【以色列花絮】尷尬！柯Ｐ見台拉維夫市長 被禁止獨照 |publisher = 《蘋果日報》}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境以色列。]]
[[File:Visa_requirements_for_Israeli_citizens.png|thumb]]的[[以色列人口|以色列公民]]可以免簽證的方式入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|以色列簽證政策|{{link-en|以色列公民簽證要求|Visa requirements for Israeli citizens}}}}
持有載明[[中華民國國民身分證|身分證]]字號的[[中華民國護照|中華民國普通護照]]之[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境以色列，每180天內停留時間不得超過90天。以色列對於免簽證入境的外國國民實施「入境卡」制度（entry card），為身分證大小的硬質紙卡，效力與入境章戳相同，於出境時由移民官收回。停留以國期間若前往[[巴勒斯坦|巴勒斯坦]]地區，要再返回以色列時，[[以色列國防軍|以國軍方]]檢查哨會查驗該入境卡與[[護照|護照]]後才能放行。經以色列過境轉機亦可免簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-48-9bc3c-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=7F220D7E656BE749 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持有[[以色列護照|以色列護照]]的[[以色列人口|以色列公民]]也可以免簽證入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將以色列按地區歸類為'''灰色警示'''：提醒注意、'''黃色警示'''：特別注意旅遊安全並檢討應否前往、'''橙色警戒'''：避免非必要旅行，與'''紅色警戒'''：不宜前往，宜儘速離境（以上截至2018年5月11日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-454-2b393-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===投資===
中華民國投資以色列的總金額約2億8千萬美元，[[臺灣商人|臺商]]的投資主要以入股投資，比較少介入實際經營。臺商來以色列除採購[[鑽石|鑽石]]外，[[高科技產業|高科技產業]]也是主要合作項目。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/il/pdf/103-70Israel.pdf |title = 投資環境簡介 |publisher = 經濟部全球臺商服務網}}</ref>

在以色列的重大投資案件有，旺宏電子投資以色列[[Tower半導體|Tower半導體]]1億美元設置[[晶圓代工|晶圓廠]]、華邦電子以6,500萬美元併購[[美國國家半導體|美國國家半導體]]在以色列的研發據點成立華邦電子以色列[[分公司|分公司]]並增資3,500萬美元、[[行政院|行政院]]國家發展基金投資Giza[[創業投資|創投]]基金1,300萬美元，且續增投資3,000萬美元加強雙方創投合作。另根據[[中華民國經濟部|經濟部]]投資審議委員會統計資料顯示目前在以色列投資的，包括中華開發工業銀行投資數十家創投公司產品，如[[虛擬攝影棚|虛擬攝影棚]]系統、[[電子商務|電子商務]]小額付款系統、[[网络安全|網路安全]]及[[虛擬私人網路|VPN]]系統、[[行動電話|行動通訊]]影像傳輸、行動通訊平台、[[伺服器|伺服器]]輸入[[積體電路|晶片]]等。此外，[[廣達電腦|廣達電腦]]投資無線頻寛晶片設計、新育創投投資[[壓電效應#壓電多晶體（壓電陶瓷）|壓電陶瓷]]超音波馬達、宏遠科技投資彩色[[顯示器|顯示器]]、益鼎及利鼎創投投資[[無線射頻辨識|無線射頻]]元件及[[積體電路|積體電路]]、惠華創投投資[[圖像傳感器|影像感測器]]，以及上銀科技[[購併|購併]]以色列Mega F Motion Systems Ltd.等。其餘創投公司的投資額從數百萬至1,000萬美元不等，而部分創投公司則是透過[[美國|美國]]據點轉投資以色列的高科技產業。<ref name="投資環境分析"/>

===軍售===
[[以色列航太工業|以色列航太工業]]和中華民國的軍事合作關係，可追溯到1971年[[聯合國第2758號決議|中華民國退出聯合國]]後，[[中華民國國軍|中華民國國軍]]追求武器的自力更生，分散採購來源。以色列剛好在這個時期大力擴軍，希望藉軍品外銷，分攤武器研發成本。但顧慮到中華民國和[[阿拉伯國家|阿拉伯國家]]的關係，一切都在機密情況下進行。<ref name="以台秘密軍事合作">{{Cite web |url = http://www.gvm.com.tw/Boardcontent_3402.html |title = 以臺秘密軍事合作 |author = 文現深 |date = 1990年3月 |publisher = 《[[遠見雜誌|遠見雜誌]]》}}</ref>

1975年，當[[美國|美國]]迫於[[中华人民共和国|中华人民共和国]]壓力而拒絕向中華民國出售[[AIM-9響尾蛇飛彈|響尾蛇飛彈]]時，以色列卻出售了[[地对空导弹|防空飛彈]]，並特許就地生產[[天使飛彈#衍生型|天使二型反艦飛彈]]（中華民國國軍改名為「雄風」，並在此基礎上研製了[[雄風飛彈|雄風系列飛彈]]）和發射架。這是雙方第一筆軍火交易。到1989年，總共生產了523枚[[飛彈|飛彈]]和77座發射架。其餘特許生產的還有50艘塔瓦爾（Dvora）快艇，以及大批槍械和[[迫擊炮|迫擊炮]]，包括127公釐蟒蛇[[火箭炮|火箭炮]]，[[IMI_Galil突擊步槍|加利爾步槍]]、[[烏茲衝鋒槍#衍生型|烏茲微型衝鋒槍]]，以及其他各種電子設備和武器。<ref name="Israel's military transfers to China and Taiwan">{{Cite book |author = Yitzhak Shichor |title = Israel's military transfers to China and Taiwan, '''''Survival''''' |date = 1998 |volume = 40 |issue = 1}}</ref>

此後雙方的軍售無論是在量還是在質都迅速上升。中華民國的[[F-CK-1經國號戰鬥機|經國號戰鬥機]]和300架[[F-5戰鬥機|F-5戰鬥機]]全部由以色列升級換代，裝上了以色列特許在中華民國生產的雷達，以及[[中華民國海軍|中華民國海軍]]的[[陽字號驅逐艦|陽字號驅逐艦]]安裝了天使反艦飛彈、以色列航太工業出售的羅塞特（Reshet）火控和指揮系統，其他公司則翻新了13艘[[中華民國海軍艦艇列表|舊軍艦]]。此外，地下飛機[[防御工事|掩體]]系統、武器和燃料儲存系統據信都採用了以色列模式。<ref name="Chung-Shan: Taiwan's Military Research Institute">{{Cite book | author = Robert Karniol | title = Chung-Shan: Taiwan's Military Research Institute, '''''International Defense Review''''' | date = 1990 |volume = 23 |issue = 9}}</ref>

===飛彈===
英國《[[詹氏年鑑|詹氏年鑑]]》於1980年代曾經推測過[[青鋒飛彈|青鋒飛彈]]的研發過程受到以色列大量的技術支援。2001年，[[國家中山科學研究院|國家中山科學研究院]]也曾經在內部刊物中指明青鋒飛彈有來自以色列的協助。該飛彈已於1991年10月服役。

===潛艇===
在[[中华人民共和国|中國大陸]]購買[[俄羅斯|俄羅斯]]的[[基洛級潛艇|基洛級潛艇]]後，為了維持[[臺灣海峽|台海]]平衡，中華民國軍方尋求購買新式常規[[潛艇|潛艇]]。因為美國以發展[[核子動力潛艇|核子動力潛艇]]為主，已不再生產常規潛艇，故將目光再一次投向以色列。考慮到中國大陸的反應，故由美國出面協調，制定了以色列設計、[[德國|德國]]生產、美國購買交貨的方式。<ref>{{cite news |author = 高路 |coauthors = 李雨桐 |url = http://tw.people.com.cn/GB/14810/4187293.html |title = 出售部分尖端武器售臺灣，美默許以色列對臺軍售 |publisher = 《[[人民網|人民網]]》 |date = 2006年3月10日}}</ref>以色列的[[海豚級潛艇|海豚級潛艇]]是軍方多年來的目標，為此，曾任[[中華民國國家安全會議|中華民國國家安全會議]]副秘書長的[[江春男|江春男]]、諮詢委員柯承亨等人曾赴美國、以色列實地考察。<ref>{{cite news |author = 丘冰 |title = 臺灣'軍事外交'全面出擊 |publisher = 《世界報》 |date = 2005年9月7日}}</ref>

===衛星===
2001年9月10日，《[[聯合報|聯合報]]》披露，中華民國軍方對中國大陸進行監控的方式有「重大突破」，雙方達成一項秘密協議，正式租用以色列的{{link-en|地球資源觀測一號|EROS A}}（EROS A）商用[[間諜衛星|間諜衛星]]以及另外三顆即將發射的同類型[[人造衛星|衛星]]，在為期6年的時間內，對[[中國人民解放軍|中國人民解放軍]][[軍事基地|設施]]和部隊動態實施偵蒐。依靠這顆衛星，偵測目標將深入到中國內陸，偵查對象包括所有「對中華民國具有威脅的飛彈發射基地」。根據衛星提供的情報，可為軍方「突襲」中國大陸提供精確資料。在這筆交易中，以色列獲得的利潤將高達20億美元。<ref>{{cite news |title = Taiwan "Borrows" Israeli Satellite During Overpasses |publisher = '''''SpaceDaily''''' |date = 12 Aug 2001}}</ref><ref>{{cite news |author = [[李凌_(記者)|李淩]] |title = [[中國人民解放軍|解放軍]]模擬攻[[澎湖群島|澎湖]] |publisher = 《[[世界新闻报_(中国)|世界新聞報]]》 |date = 2001年8月16日}}</ref><ref>{{cite news |author = 宗樹 |title = 臺灣發展[[間諜衛星|間諜衛星]] |publisher = 《[[中國國防報|中國國防報]]》 |date = 2001年9月11日}}</ref>此外，雙方還合作一項高度機密的「天貴」計劃，這項電子定向偵測計劃目的是提高軍方對[[雷達|雷達]]定位和[[無線電|無線電]][[通訊監察|監聽]]的能力。

==核技術==
{{see also|中華民國與大規模殺傷性武器}}
{{see also|船帆座事件|擁有核武器的國家列表#可能擁有{{!}}擁有核武器的國家列表}}
中華民國很早就與以色列有[[核技術|核技術]]來往，雙方經常進行核科學家的互訪，並提出派遣核研究人員赴以色列相關研究機構參加培訓，認為「這不僅將提高我們研究人員的水準，也有利於增進我們之間的瞭解。」<ref>{{Cite book |author = Robert Karniol |title = Cheng Chen-hwa（Institute of Nuclear Science, National Tsing Hua University, Taiwan）to Ernst D.Bergmann（Israel Atomic Energy Commission）| date = 27 November 1963}}</ref>

1964年10月16日，中華人民共和國[[596工程|成功試爆原子彈]]後，中華民國便決心發展[[核武器|核武器]]，建立自主核能力。1965年7月，時任總統[[蔣中正|蔣中正]]親自命[[中華民國國防部|國防部]]次長[[唐君鉑|唐君鉑]]負責籌建[[中山科學研究院|中山科學研究院]]與核能研究所，並撥專款1.4億[[美元|美元]]，加快核武器研製和生產。但由於1955年7月18日就與美國簽有《和平使用[[核结合能|原子能]]合作協議》，1965年5月25日又與[[國際原子能總署|國際原子能總署]]簽定《原子能安全保障檢查三邊協定》，因此只能借民用之名，從當時世界上核能研究較為先進的國家獲得技術援助。<ref>{{cite news |author = 王衛星 |url = http://news.xinhuanet.com/world/2003-10/23/content_2767773.htm |title = 以武拒統賊心不死，臺當局企圖搞核武器 |publisher = 《[[新華網|新華網]]》 |date = 2003年10月23日}}</ref>

1965年初，{{link-en|以色列原子能委員會|Israel Atomic Energy Commission}}主席{{link-en|博格曼|Ernst David Bergmann}}受邀訪台，會見了[[中華民國國防部長|國防部長]][[蔣經國|蔣經國]]。<ref>{{cite news |title = Jiang Jingguo（[[蔣經國|蔣經國]]） to E.D.Bergmann |date = 23 November 1965}}</ref>根據《[[紐約時報|紐約時報]]》的報導，1981年，雙方的科學家聯合參與了[[南非|南非]]核計劃。<ref>{{cite news |title = [[南非|南非]]核計畫 |publisher = 《[[紐約時報|紐約時報]]》 |date = 1981年6月28日}}</ref>

與以色列的核往來推動了中華民國核技術的快速發展，但擁有核武器不符合美國的[[東亞|東亞]]戰略，因此防止發展獨立核力量是美國的一貫政策。70年代早期，美國認為中華民國在五年內將具備製造核武器的能力，國際原子能總署的例行檢查也發現了異樣，於是在美國和國際的壓力下收手。<ref>{{Cite book |author = 居克飛 |title = 台灣核能利用走向成熟 《海峽科技與產業》 |date = 1995 |issue = 2}}</ref><ref>{{Cite book | author = 陳朝柱 |title = 台灣核能發電簡況 《福建能源開發與節約》 |date = 1996 |issue = 1}}</ref>
