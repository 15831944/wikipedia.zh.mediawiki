'''中華民國－吐瓦魯關係'''是指[[中華民國|中華民國]]與[[吐瓦魯|吐瓦魯]]之間的外交關係。

==歷史==

*1979年9月19日，[[中華民國|中華民國]]與[[吐瓦魯|吐瓦魯]]建立大使級外交關係。<ref name="ROC Foreign Relation Yearbook 2016">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf|title=中華民國外交部105年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-05 |accessdate=2018-03-01}}</ref>

*1979年-1998年，吐瓦魯事務由駐[[東加|東加]]王國大使兼轄。<ref name="ROC Foreign Relation Yearbook 2016">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf|title=中華民國外交部105年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-05 |accessdate=2018-03-01}}</ref>

*1998年11月2日，因中華民國與東加王國中止外交關係，吐瓦魯事務改由駐[[索羅門群島|索羅門群島]]大使兼轄。<ref name="ROC Foreign Relation Yearbook 2016">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf|title=中華民國外交部105年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-05 |accessdate=2018-03-01}}</ref>

*1998年12月25日，中華民國政府在[[富納富提|富納富提]]設立大使館。<ref name="ROC Foreign Relation Yearbook 2016">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf|title=中華民國外交部105年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-05 |accessdate=2018-03-01}}</ref>。

*2004年4月1日，中華民國政府派任專任大使。<ref name="ROC Foreign Relation Yearbook 2016">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf|title=中華民國外交部105年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-05 |accessdate=2018-03-01}}</ref>

*2009年8月8日，臺灣遭逢莫拉克風災，吐瓦魯捐助中華民國21萬美元，相當於該國一年國內生產毛額的1%。<ref>{{cite news|author=劉玉秋|title=〈總統感念吐瓦魯在國際支持台灣〉|newspaper=《中央廣播電臺》|date=2014-11-12}}</ref>

*2013年3月14日，吐瓦魯政府在[[臺北|臺北]]設立大使館。<ref name="ROC Foreign Relation Yearbook 2013">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2013/06.pdf|title=中華民國外交部102年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date = 2014-04 |accessdate=2018-03-01}}</ref>

==兩國貿易==
2018年吐瓦魯是中華民國第217名貿易伙伴，兩國貿易額123,716美元，其中中華民國對吐瓦魯出口86,014美元，是第210大出口國，進口37,702美元，是第203大進口國，與2017年貿易額相比貿易總額約減少49.15%、出口約減少64.30%、進口約增加1464.40%。<ref>{{cite book |url = http://cus93.trade.gov.tw/FSCI/ |title = 經濟部國際貿易局貿易統計系統 |publisher = [[經濟部國際貿易局|經濟部國際貿易局]] |date = 2019-02-23 |accessdate = 2019-03-05}}</ref>
*中華民國－吐瓦魯主要出口項目：燈具及照明配件、空氣泵或真空泵及風扇、撚線、繩、索及纜、機器腳踏車、其他鋼鐵製品等。<ref name=bft>{{Cite web |url = https://www.trade.gov.tw/World/Detail.aspx?nodeID=45&pid=581418|title = 各國經貿資訊 |publisher =[[經濟部國際貿易局|經濟部國際貿易局]]|date = 2018-02-08|accessdate = 2018-03-04}}</ref>
*中華民國－吐瓦魯主要進口項目：冷凍魚。<ref name=bft>{{Cite web |url = https://www.trade.gov.tw/World/Detail.aspx?nodeID=45&pid=581418|title = 各國經貿資訊 |publisher =[[經濟部國際貿易局|經濟部國際貿易局]]|date = 2018-02-08|accessdate = 2018-03-04}}</ref>

===航空客運===
兩國目前無直航班機，來往吐瓦魯之客運航線方式：

1.可搭乘飛往{{flagicon|AUS}}[[悉尼机场|悉尼]]、{{flagicon|NZL}}[[奥克兰国际机场|奧克蘭]]、{{flagicon|HKG}}[[香港國際機場|香港]]、{{flagicon|KOR}}[[仁川國際機場|首爾]]、{{flagicon|AUS}}[[布里斯班机场|布里斯本]]、{{flagicon|SIN}}[[新加坡樟宜機場|新加坡]]、{{flagicon|USA}}[[丹尼爾·井上國際機場|檀香山]]班機。

2.在各機場轉機前往{{flagicon|FIJ}}[[楠迪國際機場|楠迪]]。

3.轉機即可抵達{{flagicon|TUV}}[[富納富提|富納富提]]-[[富納富提國際機場|富納富提國際機場]]。

＊上述目的地國際機場為鄰近該國首都或該國最主要樞紐機場。

＊航線會因航空公司營運狀況更改或取消，出發前應與負責該航線的航空公司確認航線及航班，以免影響權益。
