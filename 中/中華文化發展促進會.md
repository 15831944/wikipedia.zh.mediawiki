'''中華文化發展促進會'''（簡稱'''文促會'''），位于中国[[北京市|北京市]][[朝阳区|朝阳区]][[辛店路|辛店路]]1号亚运新新家园朗月园1栋8号9号，是[[中華人民共和國|中華人民共和國]]的全国性文化学术团体。<ref name=jj>{{Cite web |url = http://www.chinaapc.org/wencuhuijianjie/2013-06-28/52.html |title = 中华文化发展促进会 |author =  |date = 2013-06-28 |publisher = 中华文化发展促进会 |language =  |accessdate =  |quote =  }}</ref>

==简介==
中华文化发展促进会成立於2001年6月。是由文艺界、学术界人士组成的全国性文化学术团体。中华文化发展促进会下设理事会、办公室、文化交流部、学术研究部、项目企划部，拥有中国华艺音像实业有限公司、[[华艺出版社|华艺出版社]]及影视、广播、互联网站等传媒。中华文化发展促进会的宗旨是“弘扬[[中华文化|中华文化]]、凝聚[[中华民族|民族]]意识、推进祖国统一”。通过各种形式和渠道，向海内外介绍中国文明史、传播中华优秀传统文化艺术，出品介绍中国[[改革开放|改革开放]]成就和[[精神文明建设|精神文明建设]]成果的图书、画册、影视作品，每年在台港澳地区及海外举办学术交流及文艺演出、书展、影展、画展等文化交流活动。<ref name=jj/>

中华文化发展促进会與台灣方面的[[兩岸統合學會|兩岸統合學會]]間有密切互動，經常合辦學術研討會。<ref>{{Cite web |url = http://www.chinanews.com/tw/2013/06-26/4970482.shtml |title = 两岸“筑信研讨会”共识:一中架构处理军事互信 |author =  |date =2013-06-26  |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>因為该会第一届常務副會長[[邢運明|邢運明]]，副會長[[辛旗|辛旗]]，秘書長鄭劍、副秘書長權月明等人，皆來自[[中國人民解放軍總政治部聯絡部|中國人民解放軍總政治部聯絡部]]，因此该会也被視為總政治部聯絡部下轄單位。

2011年，中华文化发展促进会聘請[[許世詮|許世詮]]等22人任该会自成立后的首批理事<ref>{{cite news|author1=刘舒凌|title=中华文化发展促进会聘请许世铨等22人为首批理事|url=http://www.chinanews.com/cul/2011/11-09/3449533.shtml|date=2011-11-09|agency=中國新聞網}}</ref>。
