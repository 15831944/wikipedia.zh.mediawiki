'''中国人民解放军陆军第九综合训练基地'''，机关位于[[河北省|河北省]][[张家口市|张家口市]][[宣化区|宣化区]]，隶属[[中国人民解放军中部战区陆军|中国人民解放军中部战区陆军]]。

==沿革==
中国人民解放军陆军第九综合训练基地宣化营区的所在地，原是[[中国人民解放军炮兵指挥学院|中国人民解放军炮兵指挥学院]]。2011年，根据中央军委有关文件，炮兵指挥学院迁址[[南京|南京]]，在原炮兵指挥学院旧址利用其原有部分教育资源建立'''中国人民解放军总参谋部炮兵训练基地'''<ref>{{Cite web |url =  http://www.yifatong.com/Posts/view/4798571 |title =中国人民解放军总参谋部炮兵训练基地与丁某某、杨某某租赁合同纠纷一审民事判决书  |author =  |date =2016-07-27  |publisher = 易法通 |language =  |accessdate =  |quote =  }}</ref>。2011年11月5日成立，隶属[[中国人民解放军总参谋部|中国人民解放军总参谋部]]。这是全军第一个炮兵训练基地，担负培养炮兵专业学兵和基层训练骨干的任务<ref>{{Cite web |url =http://mil.news.sina.com.cn/2011-11-06/0508672432.html  |title =  全军首个炮兵训练基地成立|author =  |date = 2011-11-06 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.81.cn/jwzb/2014-11/07/content_6216447.htm   |title =  总参炮兵训练基地实战化教学缩短人才成长周期|author =  |date =2014-11-07  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，转隶[[中国人民解放军陆军|中国人民解放军陆军]]，更名'''中国人民解放军陆军炮兵训练基地'''<ref>{{Cite web |url =  http://www.bidchance.com/calggnew/2017/04/11/19306672.html|title =中国人民解放军陆军炮兵训练基地清煤工程项目设计|author =  |date = 2017-04-11 |publisher = 中国招标网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.zjkxuanhua.gov.cn/article/20160824/795473214-2016-00889.html |title =宣化森林公安分局与炮兵训练基地练习团积极开展军警联谊活动  |author =  |date =2016-08-24 |publisher = 张家口市宣化区人民政府 |language =  |accessdate =  |quote =  }}</ref>。

2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，该基地与其他单位合并，调整组建'''中国人民解放军陆军第九综合训练基地'''，隶属[[中国人民解放军中部战区陆军|中国人民解放军中部战区陆军]]，基地机关位于[[河北省|河北省]][[张家口市|张家口市]][[宣化区|宣化区]]，另设怀安营区（位于[[河北省|河北省]][[张家口市|张家口市]][[怀安县|怀安县]]）、北戴河营区（位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[抚宁区|抚宁区]]）<ref>{{Cite web |url =http://www.bidcenter.com.cn/newscontent-42157076-1.html  |title = 陆军第九综合训练基地五营清煤工程施工招标公告 |author =  |date = 2017-09-20 |publisher =中国采招网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.bidcenter.com.cn/newscontent-40759947-1.html |title =  第九综合训练基地应急营房工程施工招标公告|author =  |date = 2017-08-15 |publisher = 中国采招网 |language =  |accessdate =  |quote =  }}</ref>。
