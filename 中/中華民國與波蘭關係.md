'''中華民國與波蘭關係'''是指[[中華民國|中華民國]]與[[波蘭共和國|波蘭共和國]]之間的關係。兩國於1929－1949年有官方[[外交#外交上的承認|外交關係]]，斷交後直至1990年代，在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[波蘭人民共和國|波蘭人民共和國]]。]]
{{see also|中華民國外交|{{link-en|波蘭外交|Foreign relations of Poland}}}}
{{see also|中国驻波兰大使列表{{!}}中華民國駐波蘭外交官列表|波兰驻华大使列表{{!}}波蘭駐中華民國外交官列表}}
1929年9月18日，波蘭與中華民國建立公使級[[外交#外交上的承認|外交關係]]。

1930年9月22日，商談互派使節。1931年，於首都[[華沙|華沙]]設立'''中華民國駐[[波蘭共和國|波蘭共和國]][[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1940年8月18日，因上年{{link-en|佔領波蘭(1939–45)|Occupation of Poland (1939–45)|納粹德國與蘇聯佔領波蘭}}，於是關閉駐波蘭公使館。

1942年6月24日，升格為大使級外交關係。公使館升格為'''中華民國駐波蘭共和國[[外交代表機構#大使馆|大使館]]'''（原設於[[英國|英國]]首都[[倫敦|倫敦]]，1945年5月，[[第二次世界大戰歐洲戰場的結束|納粹德國投降]]後遷回波蘭），並派駐[[特命全權大使|大使]]（[[中国驻比利时大使列表|駐比利時大使]]兼任）。

1949年10月5日，波蘭與中華民國斷交。<ref name="103年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/09.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>當時中華民國尚能有效統治[[中國大陸|中國大陸]]部分地區，加上1950－1990年代東西方陣營的「[[冷戰|冷戰]]」期間，中華民國在[[意識形態|意識形態]]和[[政體列表|制度]]上均偏向以[[美國|美國]]為首的[[資本主義|資本主義]]陣營，復加上以[[蘇聯|蘇聯]]為首的[[社會主義國家|共產集團國家]]的封閉，導致從[[臺灣戰後時期|中央政府遷台]]後幾乎沒有官方往來。

1971年10月25日，身為共產陣營成員及蘇聯[[衛星國|衛星國]]之一的波蘭支持[[中華人民共和國|中華人民共和國]]取得[[聯合國|聯合國]]裡的「[[中國與聯合國#代表權問題|中國席位]]」（中國代表權問題）。

1989年的「[[東歐民主化|東歐劇變]]」使波蘭轉型為民主國家並重新對外開放，雙方始恢復往來。

2013年11月，《[[聯合國氣候變化綱要公約|聯合國氣候變遷綱要公約]]》[[2013年联合国气候变化大会|締約國第19次大會]]（UNFCCC COP19）在華沙舉行，台灣派出代表團，以及[[立法院|立法院]]與[[非政府組織|非政府組織]]團體等80餘人共同參與。<ref name="102年外交年鑑"/>

2018年10月，波蘭「{{lang|pl|Idź Pod Prąd TV}}」的電視節目公開支持[[台灣獨立|台灣獨立]]，表示「2018年是[[波蘭第二共和國|波蘭獨立100週年]]，我們波蘭人希望台灣能夠獨立，對中國說NO，對台灣說YES」，此舉引起[[中華人民共和國媒體|中華人民共和國媒體]]不滿，波蘭駐中華人民共和國大使館30日發表聲明，提到[[波蘭政府|波蘭政府]]於政策面始終遵照[[一個中國|中國規定的原則]]，但也說明{{link-en|波蘭憲法|Constitution of Poland}}保障所有公民的[[言論自由|言論自由]]，對於電視節目內容並不會給予評論或建議。中華民國行政院長[[賴清德|賴清德]]30日在立法院則說，這代表「[[論語|德不孤必有鄰]]」，認同台灣為[[主權獨立國家|主權獨立國家]]，人民有權決定自己的生活方式。<ref>{{Cite web |url = http://news.ltn.com.tw/news/world/breakingnews/2597085 |author = 綜合報導 |date = 2018年10月31日 |title = 電視節目挺台獨中國氣噗噗 波蘭回應：國民有言論自由 |publisher = [[自由時報|自由時報]]}}</ref>

==經濟==
[[File:SYM_Fiddle_II_20120823.jpg|240px]][[盧布斯卡省|盧布斯卡省]]省會[[綠山城|綠山城]]街頭的[[臺灣|臺灣]][[三陽工業|三陽工業]][[機車|機車]]]]
{{see also|臺灣經濟|波蘭經濟}}
波蘭是中華民國在[[東歐|東歐]]第1大貿易夥伴（不包括俄羅斯）。

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於首都[[華沙|華沙]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#EUROPE |title = 外貿協會全球據點|publisher = 中華民國對外貿易發展協會}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]也於華沙設立'''駐波蘭代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2017年，波蘭是中華民國第40大貿易夥伴、第50大[[進口|進口]]夥伴、第29大[[出口贸易|出口]]夥伴。出口至波蘭的金額為8億8,963萬9,566[[美元|美元]]，年增13.976%；自波蘭進口的金額為2億7,664萬7,122美元，年增28.791%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）6億1,299萬2,444美元，年增8.351%。※2016年，波蘭是中華民國第41大貿易夥伴、第50大進口夥伴、第29大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

波蘭[[奢侈品|高端]][[腳踏車|腳踏車]]以進口為主，前5大[[品牌|品牌]]中，台灣就佔前2名，依序為[[捷安特|捷安特]]、[[美利達工業|美利達工業]]，排名第5的{{link-en|SPECIALIZED|Specialized Bicycle Components}}，美利達也是主要[[股東|股東]]。台灣腳踏車[[外包|代工]]大廠愛地雅工業在[[波蘭行政區劃|波蘭中部]]設有工廠，為歐洲客戶代工；SPECTEC公司，則對波蘭當地業者提供[[ODM|貼牌代工]]服務。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/pl/pdf/104-59Poland.pdf |date = 2015年8月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

===投資<small><ref name="投資環境簡介"/></small>===
根據經濟部投資審議委員會統計，[[臺灣商人|臺商]]對波蘭的總投資金額為2億0,302萬美元，前往波蘭設點有27家公司，僱用4,600人，其中20家[[控股公司|母公司]]皆在台灣且有17家為[[上市公司|上市公司]]。台商波蘭[[子公司|子公司]]大部分經第三地轉投資，以銷售自家公司[[電子產品|電子產品]]為主，例如：[[宏碁|宏碁]]、[[華碩電腦|華碩電腦]]、[[台達電子|台達電子]]（台達電）、[[技嘉科技|技嘉科技]]、[[訊舟科技|訊舟科技]]、[[研華科技|研華科技]]等，僱用員工皆在30人以下；而僱用員工較多的是以組裝[[製造業|製造業]]為主，有：[[仁寶電腦|仁寶電腦]]（[[電視機|電視]]）、[[冠捷科技|冠捷科技]]（電視）、晶元光電（[[發光二極體|發光二極體]]）、飛磁材料（[[磁珠|鐵芯]]）、昶虹貿易（電視[[印刷電路板|電路板]]）、愛地雅工業（腳踏車）等，其次是[[筆記型電腦|筆記型電腦]]維修業，有仁寶電腦及[[微星科技|微星科技]]等；以資金在波蘭自創公司的有8家，僱用員工皆在30人以下。

目前已成立波蘭台灣商會。

[[中華民國教育部|中華民國教育部]]自1991年7月設立「[[獨立國家國協|獨立國家國協]]及東歐地區國家來華留學[[獎學金|獎學金]]」，其中包括波蘭。後改為教育部的「[[中華民國國語|華語文]]獎學金」，以及[[中華民國外交部|外交部]]的「台灣獎學金」，提供波蘭學生申請赴台研習中文或攻讀[[碩士|碩]][[博士|博士]]。教育部並派任數名華語文教師至[[波兰大学列表|波蘭的大學]]任教，協助推廣[[正體字|正體中文]]；自1997年，波蘭亦提供獎學金名額給台灣學生赴波蘭進修，例如「[[波蘭政府|波蘭政府]]獎學金」與「暑期[[波蘭文|波蘭文]]研習獎學金」。<ref name="86年外交年鑑"/><ref name="106年外交年鑑"/>
