'''中華民國與秘魯關係'''是指[[中華民國|中華民國]]與[[秘魯共和國|秘魯共和國]]之間的關係。1913－1971年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[秘魯|秘魯]]。]]
{{see also|中華民國外交|{{link-en|秘魯外交|Foreign relations of Peru}}}}
{{see also|中国驻秘鲁大使列表{{!}}中華民國駐秘魯外交官列表|秘鲁驻华大使列表{{!}}秘魯駐中華民國外交官列表}}
中華民國與秘魯的[[外交#外交上的承認|外交關係]]可追溯於[[清朝|清朝]][[光緒|光緒]]元年，1875年12月11日，兩國簽署《中秘友好通商行船[[條約|條約]]》，相互設立外交代表機構、派駐使節（清朝由[[中国驻美国大使列表|出使美國欽差大臣]]兼任）。<ref name="外交年鑑"/>

1912年2月12日，中華民國繼承清朝與秘魯的外交關係，並向秘魯派駐外交代表。

1913年5月5日，秘魯外交承認中華民國，並建立公使級外交關係。於首都[[利馬|利馬]]設立'''中華民國駐秘魯共和國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]（[[中国驻巴西大使列表|駐巴西公使]]兼任，由[[代辦|代辦]]使事，後兼使改為專使）。

1944年9月1日，建立大使級外交關係，公使館升格為'''中華民國駐秘魯共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。<ref name="駐館與駐地"/>

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1961年5月22日，[[秘魯總統|秘魯總統]][[曼努埃爾·普拉多·烏加特切|浦樂多]]抵台訪問，[[中華民國總統|總統]][[蔣中正|蔣中正]]親至[[台北松山機場|松山機場]]以軍禮相迎。

1970年4月28日，[[秘魯總理|秘魯總理]]{{link-es|孟達略|Ernesto Montagne Sánchez}}抵台訪問。<ref>[http://catalog.digitalarchives.tw/item/00/3a/18/53.html 秘魯總理訪華]，典藏臺灣</ref>

1971年11月2日，秘魯與中華民國斷交。3日，關閉大使館。<ref name="駐館與駐地"/>

===貿易===

2018年，秘魯是中華民國第51大貿易夥伴、第51大[[進口|進口]]夥伴、第49大[[出口贸易|出口]]夥伴。出口至秘魯的金額為2億8,562萬3,711[[美元|美元]]，年增22.507%；自秘魯進口的金額為2億8,770萬6,553美元，年減13.455%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）208萬2,842美元，年減97.902%。2017年，秘魯是中華民國第51大貿易夥伴、第44大進口夥伴、第53大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資===
截至2014年，在秘魯的[[臺灣商人|台商]]約30多家，總投資金額為1億5,000萬美元。多自[[美國|美國]]、[[智利|智利]]及[[厄瓜多|厄瓜多]]輾轉投資，且多為個人獨資。產業類別涵蓋：[[貿易業|貿易業]]（舊[[引擎|引擎]]、[[機車_(公路)|機車]]、[[腳踏車|腳踏車]]、[[汽車零組件列表|汽車零組件]]、工業用[[縫紉機|縫紉機]]及零配件、[[資訊科技|資訊產品]]、[[餐具|餐具]]、[[玩具|玩具]]及[[禮品|禮品]]、[[木材|木材]]及[[海產|海產]]加工等）、[[第三產業|服務業]]（[[航運|船務]]代理、[[旅行社|旅行社]]、[[旅館|旅館]]、[[休閒|休閒]][[娛樂|娛樂]]等），以及[[房地產|房地產]]開發等。台商分布以首都[[利馬|利馬]]佔大多數，第2大城[[特魯希略_(秘魯)|特魯希略]]、第3大城[[阿雷基帕|阿雷基帕]]、第4大城[[奇克拉約|奇克拉約]]、南部[[塔克納|塔克納]]的[[自由貿易區|自由貿易區]]、東部林產區[[普卡爾帕|普卡爾帕]]等地也有少數台商。<ref name="投資環境簡介"/>

臺灣科技廠如[[宏碁|宏碁]]、[[華碩電腦|華碩電腦]]、[[技嘉科技|技嘉科技]]等品牌皆已在秘魯行銷，並以「[[臺灣|臺灣]][[接單生產|接單]]、[[中华人民共和国|中國大陸]]出貨」方式，透過[[代理|代理]]商推廣銷售。華碩電腦與技嘉科技也在秘魯設立[[行銷|行銷]]據點兼維修站。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/pe/pdf/104-40Peru.pdf |date = 2015年7月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

目前成立的組織：秘魯臺灣商會，會員約30人。<ref name="投資環境簡介"/>

==交流==
===僑民===
{{see also|臺灣裔{{!}}海外臺僑|海外華人|秘魯華人}}
華人移民秘魯始於1849年（清朝[[道光|道光]]二十九年），目前華裔人口超過2百萬，但臺僑僅約300人。<ref name="106年外交年鑑"/>

2002年8月10日，成立全僑民主和平聯盟秘魯支盟。<ref name="91年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__06.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

===援助===
2007年8月15日，[[2007年秘魯大地震|秘魯發生大地震]]，[[中華民國政府|中華民國政府]]除捐贈10萬[[美元|美元]]外，另派遣5人醫療團赴災區協助救援，成為首個抵達災區的外國救援隊伍，{{link-en|秘魯衛生部|Ministry of Health (Peru)}}部長並公開表達感謝之意。<ref name="96年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2007/html/08-1-6.htm |title = 《中華民國96年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

===教育===
{{see also|臺灣教育|{{link-en|秘魯教育|Education in Peru}}}}
台灣[[國立政治大學|國立政治大學]]與秘魯[[國立聖馬爾科斯大學|國立聖馬爾科斯大學]]簽有[[交換學生|交換學生]]協議，自2008年起，每年有1－3位政大學生前往秘魯就學6－12個月；聖馬爾科斯大學則透過臺灣[[獎學金|獎學金]]徵選，推薦學生前往政大就學。<ref name="105年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>截至2017年，秘魯學生已有151人獲發臺灣獎學金或[[中華民國國語|華語文]]獎學金赴台就讀。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

每年亦有5－6名兩國青少年透過[[國際扶輪|國際扶輪]]青少年交換計畫至台灣或秘魯研習一年，此計畫至2016年已實施9年。<ref name="105年外交年鑑"/>
