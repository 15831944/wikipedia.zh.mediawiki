'''中国人民解放军空军漳州指挥所'''，位于[[福建省|福建省]][[漳州市|漳州市]]，隶属[[中国人民解放军东部战区空军|中国人民解放军东部战区空军]]。

== 沿革 ==
1960年2月，中国人民解放军空军福州指挥所在福建省[[福州市|福州市]]组建。同年6月，福州指挥所与驻晋江的[[中国人民解放军福州军区空军|中国人民解放军福州军区空军]]机关对调，福州军区空军机关移防福州，原福州指挥所移防晋江并改称'''中国人民解放军空军晋江指挥所'''。1962年6月21日，空军晋江指挥所改组为[[中国人民解放军空军福州基地|中国人民解放军空军第八军]]，隶属福州军区空军建制。后移驻福建漳州。1970年5月，空军第八军军部调驻[[江西|江西]][[南昌|南昌]]，由福州军区空军抽调干部在漳州组建福州军区空军辅助指挥所，为福州军区空军派出机构，以接替空军第八军的防务。1973年8月，空军第八军调回漳州。1976年4月，驻漳州的空军第八军撤销。暂由一个精干的指挥班子作为福州军区空军派出的福州军区空军辅助指挥所，担负闽南地区空军部队作战指挥任务。1978年8月，漳州的福州军区空军辅助指挥所正式列编，称'''中国人民解放军空军漳州指挥所'''。1985年9月，在[[百万大裁军|百万大裁军]]中，空军漳州指挥所撤销<ref>{{Cite web |url =http://news.sohu.com/2004/05/28/04/news220300490.shtml  |title =空军历史上的十四个军  |author =  |date = 2004-05-28 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>叶青松，皮定均空难事件背后的历史细节，党史博览2011(11):53-56</ref>。1990年代末，重新组建'''中国人民解放军空军漳州指挥所'''，副师级<ref>{{Cite web |url = http://news.sina.com.cn/c/2002-01-30/461253.html |title = 感天动地英雄曲 来自漳州特大山火现场的报告 |author =  |date = 2002-01-30 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。在2016年[[深化国防和军队改革|深化国防和军队改革]]前，空军漳州指挥所隶属[[中国人民解放军南京军区空军|中国人民解放军南京军区空军]]。2016年转隶[[中国人民解放军东部战区空军|中国人民解放军东部战区空军]]<ref>{{Cite web |url = http://www.longyan.gov.cn/zwpd/gzdt/lyyw/jrly/201701/t20170125_671263.htm |title = 龙岩市领导赴漳慰问空军漳州指挥所 |author =  |date = 2017-01-25 |publisher =中国龙岩  |language =  |accessdate =  |quote =  }}</ref>。
