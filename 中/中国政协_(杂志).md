《'''中国政协'''》杂志由[[中国人民政治协商会议全国委员会|中国人民政治协商会议全国委员会]]主管、[[中国人民政治协商会议全国委员会办公厅|中国人民政治协商会议全国委员会办公厅]]主办，面向[[中华人民共和国|中华人民共和国]]境内外公开发行。中国政协杂志社是中国人民政治协商会议全国委员会办公厅直属[[事业单位|事业单位]]<ref>{{Cite web |url = http://www.gjsy.gov.cn/cxzl/ |title = 中国政协杂志社 |author =  |date =  |publisher =事业单位在线  |language =  |accessdate = 2018-02-14 |quote =  }}</ref>。

==简介==
1951年4月，[[周恩来|周恩来]]批准创办《[[中国人民政治协商会议全国委员会|中国人民政治协商会议全国委员会]]会刊》，是中国人民政治协商会议全国委员会机关主办的内部刊物<ref name=dshj/>。[[文革|文革]]期间被迫停刊。[[粉碎四人帮|粉碎“四人帮”]]后，于1978年5月复刊，定名《政协会刊》，性质是中国人民政治协商会议全国委员会内部的公报性刊物<ref>薄茹，政协报刊新闻宣传工作的发展历程，新闻世界2010(12):38-39</ref>。 

1980年11月，创办《学习参考资料》。1994年12月，创办《地方政协工作通讯》。2000年1月，经全国政协办公厅批准，上述三种刊物合并为《中国政协》杂志，向国内外发行，[[李瑞环|李瑞环]]题写刊名。2002年9月，[[中央编办|中央编办]]批复成立《中国政协》杂志社。同年12月，《中国政协》杂志社成立大会召开。2006年7月6日，社委会成立。2009年3月，中央编办等部委批复《中国政协》杂志社转为财政补助事业单位。2011年1月，《中国政协》由月刊改为半月刊，《中国政协》（半月刊）正式出版发行<ref name=dshj>{{Cite web |url = http://www.zgzx.com.cn/node_44638.htm |title = 大事记 |author =  |date =  |publisher = 中国政协传媒网 |language =  |accessdate = 2018-02-09 |quote =  }}</ref>。此外，还先后出版了各种专刊、会刊<ref name=shi>孙芸辉，携手同行 十年跨越——记《中国政协》杂志社成立十周年座谈会，中国政协2012(24):46-48</ref>。办刊方针为“立足政协，服务统战，面向社会”。主要读者是全国各级政协委员和政协工作者，各[[民主党派|民主党派]]、工商联、各族各界精英，内容以宣传[[中国共产党领导的多党合作和政治协商制度|中国共产党领导的多党合作和政治协商制度]]，人民政协的性质、地位及作用，各级政协组织和政协委员履职实践为主，属于时政类综合性新闻期刊<ref name=jj/>。

中国政协传媒网2012年由中国政协杂志社创办，2014年1月28日新版正式上线。该网站的理念是“网站向杂志借力，杂志向网站延伸”，设有新闻、理论、履职、人物、健康、文化、书画七个频道和视频库、资料库、图片库<ref name=jj>{{Cite web |url = http://www.zgzx.com.cn/node_44638.htm |title =《中国政协》杂志  |author =  |date =  |publisher = 中国政协传媒网 |language =  |accessdate = 2018-02-09 |quote =  }}</ref>。
