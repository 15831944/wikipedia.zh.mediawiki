'''中国人民解放军第一八四医院'''，位于[[江西省|江西省]][[鹰潭市|鹰潭市]][[月湖区|月湖区]]湖东路4号，隶属[[无锡联勤保障中心|无锡联勤保障中心]]，为三级乙等医院。

==简介==
中国人民解放军第一八四医院创建于1967年，位于江西省鹰潭市，是以创伤急救、脑外伤、骨与关节损伤、胸腹联合伤等多发伤救治及消化内镜诊疗、外科微创手术、心血管介入治疗、肿瘤精确放疗等技术为特色的综合性三级乙等医院<ref>文化建院 人才兴院 科技强院——解放军第184医院科学发展打造一流品牌，铁军2012年第10期</ref>。截至2010年，医院已成为[[中国人民解放军福州总医院|中国人民解放军南京军区福州总医院]]协作医院，[[南方医科大学|南方医科大学]]（原[[中国人民解放军第一军医大学|中国人民解放军第一军医大学]]）、[[南昌大学医学院|南昌大学医学院]]等医学院校的教学医院<ref>解放军第184医院，价格月刊2010年06期</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第一八四医院由[[中国人民解放军南京军区|中国人民解放军南京军区]]转隶[[无锡联勤保障中心|无锡联勤保障中心]]<ref name=dazao>{{Cite web |url = http://www.sohu.com/a/141270104_612648|title = 无锡联保中心第184医院全力救治“5.15”交通事故伤员 |author =  |date =  2017-05-17 |publisher =  搜狐|language =  |accessdate =  |quote =  }}</ref>。
