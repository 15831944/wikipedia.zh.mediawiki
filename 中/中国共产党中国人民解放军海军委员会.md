'''中国共产党中国人民解放军海军委员会'''，简称'''海军党委'''，是[[中国共产党|中国共产党]]在[[中国人民解放军海军|中国人民解放军海军]]的委员会。

==沿革==
1950年6月，[[中共中央组织部|中共中央组织部]]批复，同意中国共产党中国人民解放军海军临时委员会（海军临时党委）成立。1953年，设海军临时党委常务委员会<ref name=gswshang>{{Cite web |url =http://www.hprc.org.cn/gsw/detail_zzslk.jsp?channelid=75034&record=3149 |title =  附卷二  中国人民解放军组织(1949.10—1997.9) 上部　过渡时期和社会主义建设时期(1949.10—1966.5) 第四章　中国人民解放军各军兵种 第一节　海军及海军党委|author =  |date =  |publisher = 中华人民共和国国史网 |language =  |accessdate =  2017-09-16 |quote =  }}</ref>。

1956年6月，海军第一届党代表大会召开，选举产生并经[[中共中央|中共中央]]批准，成立第一届海军党委，由11人组成党委常务委员会<ref name=gswshang/>。

1960年7月，海军第二届党代表大会召开，选举产生并经中共中央批准，成立第二届海军党委，由11人组成党委常务委员会<ref name=gswshang/>。
 
1964年1月，海军第三届党代表大会召开，选举产生并经中共中央批准，成立第三届海军党委，由10人组成党委常务委员会<ref name=gswshang/>。 

“[[文化大革命|文化大革命]]”爆发后，海军直属机关及院校开展“[[四大_(文化大革命)|四大]]”，海军党委第二书记[[苏振华|苏振华]]等人遭[[批斗|批斗]]，海军党委常务委员会委员[[张学思|张学思]]被迫害致死<ref name=gswzhong>{{Cite web |url =http://www.hprc.org.cn/gsw/detail_zzslk.jsp?channelid=75034&record=3187 |title =  附卷二  中国人民解放军组织(1949.10—1997.9) 中部　“文化大革命”时期(1966.5—1976.10) 第四章　中国人民解放军各军兵种 第一节　海军及海军党委 |author =  |date =  |publisher = 中华人民共和国国史网 |language =  |accessdate =  2017-09-16 |quote =  }}</ref>。

1969年5月16日至6月6日，海军第四次党代表大会召开，选举产生并经中共中央批准，成立第四届海军党委，由10人组成党委常务委员会<ref name=gswzhong/>。 

1972年5月，经中共中央批准，海军党委由[[肖劲光|肖劲光]]任党委第一书记，[[王宏坤|王宏坤]]任党委第二书记，[[苏振华|苏振华]]任党委第三书记。1973年3月又决定苏振华任党委第一书记，肖劲光任党委第二书记，[[王宏坤|王宏坤]]任党委第三书记。1973年12月，因海军领导班子调整，海军党委成员也相应调整<ref name=gswzhong/>。 

1977年12月，海军第五次党代表大会召开，选举产生第五届海军党委。五届一次党委全会选举并经中共中央、中央军委批准，由10人组成党委常务委员会。1979年2月，[[叶飞|叶飞]]任党委第一书记。1980年12月，[[李耀文|李耀文]]任党委第二书记。1982年11月，李耀文任党委第一书记，[[刘华清|刘华清]]任党委第二书记。1985年7月，经中共中央、中央军委批准，海军党委常务委员会成员调整<ref name=gswxia>{{Cite web |url =http://www.hprc.org.cn/gsw/detail_zzslk.jsp?channelid=75034&record=3220  |title = 附卷二  中国人民解放军组织(1949.10—1997.9) 下部　社会主义事业发展新时期(1976.10—1997.9) 第四章　中国人民解放军各军兵种 第一节　海军及海军党委 |author =  |date =  |publisher = 中华人民共和国国史网 |language =  |accessdate = 2017-09-16 |quote =  }}</ref>。

1986年12月，中共中国人民解放军海军第六次代表大会召开，选举产生第六届海军党委。六届一次党委全会选举并经中共中央、中央军委批准，由10人组成党委常务委员会<ref name=gswxia/>。
 
1991年12月，中共中国人民解放军海军第七次代表大会召开，选举产生第七届海军党委。七届一次党委全会选举并经中共中央、中央军委批准，由10人组成党委常务委员会<ref name=gswxia/>。 

1996年12月，中共中国人民解放军海军第八次代表大会召开，选举产生第八届海军党委。八届一次党委全会选举并经中共中央、中央军委批准，由9人组成党委常务委员会<ref name=gswxia/>。 

2001年12月，中共中国人民解放军海军第九次代表大会召开，选举产生第九届海军党委<ref>{{Cite web |url =http://news.sohu.com/06/25/news147432506.shtml  |title =军委领导强调建有强大综合作战能力现代化海军  |author =  |date = 2001-12-15 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。九届一次党委全会选举并经中共中央、中央军委批准，由？人组成党委常务委员会。
