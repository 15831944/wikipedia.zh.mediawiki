'''中华人民共和国最高人民法院'''，官方简称'''最高法'''、'''高法院'''，亦简称'''最高人民法院'''、'''最高法院'''、'''最高院'''，是[[中华人民共和国|中华人民共和国]]的最高[[审判机关|审判机关]]。最高人民法院的裁判为终审裁判，不得[[上诉|上诉]]。

最高人民法院由[[全国人民代表大会|全国人民代表大会]]选举产生的[[中华人民共和国最高人民法院院长|最高人民法院院长]]和[[全国人民代表大会常务委员会|全国人民代表大会常务委员会]]决定任命的[[中华人民共和国最高人民法院副院长|副院长]]、[[最高人民法院审判委员会|审判委员会]]委员、庭长、副庭长和审判员组成，对全国人民代表大会及其常务委员会负责。

==沿革==
1949年10月22日，[[中央人民政府最高人民法院|中央人民政府最高人民法院]]正式成立。

1954年9月，组建'''中华人民共和国最高人民法院'''，作为最高国家审判机关。1956年4月至6月期间，最高人民法院根据全国人大常委会的决定，设置[[中华人民共和国最高人民法院特别军事法庭|特别军事法庭]]，在辽宁沈阳和山西太原开庭，审判45名侵华日军战犯。1958年，最高人民法院设置派出机构——[[最高人民法院西藏分院|最高人民法院西藏分院]]，依法审理西藏地区刑事、民商事案件，最高法西藏分院后于1965年撤销并改为[[西藏自治区高级人民法院|西藏自治区高级人民法院]]。

1980年9月29日至1981年[[林彪、江青反革命集团案|林彪、江青反革命集团案]]审理期间，最高人民法院根据全国人大常委会的决议，设立[[中华人民共和国最高人民法院特别法庭|特别法庭]]，在北京市[[东城区|东城区]]正义路1号开庭，审判该案主犯。

2000年，最高人民法院进行机构改革。最高人民法院、最高人民检察院机关加强并完善了法律赋予的审判检察职能，调整规范了党务、政工机构。最高人民法院新设了立案庭和审判监督庭，实现了立审、审执、审监三个分离<ref name=quanshan>{{Cite web|url = http://www.xzqs.gov.cn/Article/Show?id=36232|title = 2000年全国各级人大、政协机关，法院、检察院机关，群众团体机关机构改革的情况|author = |date = 2012-02-06|publisher = 徐州市泉山区人民政府办公室|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180317231805/http://www.xzqs.gov.cn/Article/Show?id=36232|archive-date = 2018-03-17|dead-url = yes}}</ref>。

2014年12月，最高人民法院设置第一、二巡回法庭。2016年12月，全国人大常委会批准设立最高人民法院第三、四、五、六巡回法庭<ref>{{cite web |url = http://www.court.gov.cn/zixun-xiangqing-33751.html |language = zh |title = 全国人大常委会任命最高法第三、四、五、六巡回法庭庭长 |agency = 新华社 |date = 2016-12-25 |accessdate = }}</ref>。至此，全国除[[北京|北京]]、[[天津|天津]]、[[河北|河北]]、[[山东|山东]]和[[内蒙古|内蒙古]]以外，均被最高法巡回法庭巡回区覆盖，由最高人民法院各巡回法庭跨行政区审理特定案件。

2018年6月，在[[广东省|广东省]][[深圳市|深圳市]]、[[陕西省|陕西省]][[西安市|西安市]]分别设立最高人民法院第一、二国际商事法庭<ref>{{cite web |url = http://www.xinhuanet.com/legal/2018-06/29/c_1123057535.htm|language = zh |title =最高人民法院第一、第二国际商事法庭开始正式办公 |agency = 新华社 |date = 2018-06-29 |accessdate = }}</ref>。

== 职责 ==
根据《[[中华人民共和国宪法|中华人民共和国宪法]]》、《{{s|中华人民共和国人民法院组织法}}》和相关法律，以及《{{s|最高人民法院主要职责内设机构和人员编制规定}}》，作为国家最高审判机关，最高人民法院承担下列职责<ref name="报道0122">{{cite web |author1=最高人民法院办公厅 |title=最高人民法院2018年部门预算 |url=http://www.court.gov.cn/zixun-xiangqing-39382.html |website=中华人民共和国最高人民法院门户网站 |accessdate=2019-01-22 |location=北京 |language=zh-cn}}</ref>：
{{columns-list|2|
# 对全国人民代表大会和全国人民代表大会常务委员会负责并报告工作，接受全国人民代表大会及其常务委员会的监督。
# 依法向全国人民代表大会和全国人民代表大会常务委员会提出议案。
# 依法审判法律规定由最高人民法院管辖的和其认为应当由自己审判的刑事、民事、行政、海事等第一审案件。
# 依法审判法律规定由最高人民法院管辖的刑事、民事、行政、海事等第二审案件。
# 依法核准死刑案件。
# 受理各类申诉和再审申请，对其中确有错误的，提审或指令下级人民法院再审。
# 依法审判由最高人民检察院按照审判监督程序提出的抗诉案件。
# 依法对下级人民法院行使指定管辖权。
# 监督地方各级人民法院和专门人民法院的审判工作。
# 对审判过程中具体应用法律的问题作出司法解释。
# 依法行使司法执行权和司法决定权。
# 依法决定国家赔偿。
# 组织、指导地方各级人民法院和专门人民法院办理司法协助事项。
# 对法律、法规、规章等草案提出意见；针对案件审理中发现的问题提出司法建议。
# 对最高人民法院的法官和其他工作人员进行思想政治教育、组织专业培训；指导下级人民法院的思想政治工作和教育培训工作。
# 按照权限管理法官和其他工作人员；协助主管部门管理地方人民法院和专门人民法院的机构、编制工作。
# 领导人民法院的监察工作。
# 组织各级人民法院同外国和港澳台地区司法界、国际组织的司法交流活动。
# 在审判工作中宣传法制，教育公民自觉遵守宪法、法律。
# 管理最高人民法院直属单位和社会团体。
# 承办其他应由最高人民法院负责的工作。
}}

== 规章制度 ==
=== 司法解释 ===
{{Wikisource|中华人民共和国最高人民法院司法解释}}
[[审判解释|审判解释]]，是[[中华人民共和国法律|中华人民共和国法律体系]]中的一种[[法律渊源|法律渊源]]。《[[s:全国人民代表大会常务委员会关于加强法律解释工作的决议|全国人民代表大会常务委员会关于加强法律解释工作的决议]]》中规定：“凡属于法院审判工作中具体应用法律、法令的问题，由最高人民法院进行解释”。[[中国内地|中国内地]]的[[司法解释|司法解释]]还有[[最高人民检察院|最高人民检察院]]的“[[检察解释|检察解释]]”。而狭义的“[[法律解释|法律解释]]”等于[[全国人大常委会|全国人大常委会]]的“[[立法解释|立法解释]]”。

=== 法官等级 ===
{{Main|中华人民共和国大法官}}
2001年修改的《[[中华人民共和国法官法|中华人民共和国法官法]]》第十八条规定，“法官的级别分为十二级。”“[[中华人民共和国最高人民法院院长|最高人民法院院长]]为首席大法官，二至十二级法官分为大法官、高级法官、法官。”第十九条规定，“法官的等级的确定，以法官所任职务、德才表现、业务水平、审判工作实绩和工作年限为依据。”第二十条规定，“法官的等级编制、评定和晋升办法，由国家另行规定。”
