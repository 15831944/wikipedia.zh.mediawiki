'''中華民國與菲律賓關係'''是指[[中華民國|中華民國]]與[[菲律賓共和國|菲律賓共和國]]之間的關係。兩國於1946－1975年有正式[[外交#外交上的承認|外交關係]]，斷交後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。雖無外交關係，但雙方在經貿、投資、觀光、勞工關係密切、互補性高。<ref name="雙邊關係">{{Cite web |url = http://www.taiwanembassy.org/ph/post/292.html |title = 中華民國臺灣與菲律賓共和國雙邊關係 |publisher = 中華民國駐外單位聯合網站}}</ref>

==政治==
===外交===
{{see also|中華民國外交|{{link-en|菲律賓外交|Foreign relations of Philippines}}}}

1947年4月18日，簽署《中華民國菲律賓共和國友好條約》，建立公使級[[外交#外交上的承認|外交關係]]。<ref name="105年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>於首都[[馬尼拉|馬尼拉]]設立'''中華民國駐菲律賓共和國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1949年7月10日，中華民國總統[[蔣中正|蔣中正]]應邀由臺灣扺[[菲律賓|菲律賓]]訪問，與[[菲律賓總統|菲律賓總統]][[季里諾|季里諾]]在[[碧瑤|碧瑤]]會談，並發表聯合聲明，號召東亞各國組織「太平洋聯盟」，以扼制共產主義之威脅。8月3日，公使館升格為'''中華民國駐菲律賓共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1950年1月，當[[臺灣戰後時期|中華民國政府剛遷至台灣]]後，蔣中正在台灣恢復行使[[中華民國總統|總統]]職權前夕，以[[中國國民黨主席|中國國民黨總裁]]的身份再度訪問菲律賓，和總統[[季里諾|季里諾]]會晤。

1960年，菲律賓總統[[卡洛斯·加西亞|賈西亞]]抵台訪問並與蔣中正晤談。

1961年1月17日，外交部長[[沈昌煥|沈昌煥]]赴[[菲律賓|菲律賓]]，參加中華民國、菲律賓、[[大韓民國|大韓民國]]、[[越南共和國|越南共和國]]四國外長會議；1月18日，會議於馬尼拉舉行，協商團結，應付共黨。<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|666}}

1963年3月20日，副總統[[陳誠|陳誠]]代表蔣中正訪問菲律賓，23日，發表聯合公報，加強經濟技術合作與文化交流。<ref name="記" />{{rp|689}}。

1966年3月25日，總統蔣中正與菲律賓總統[[馬可仕|馬可仕]]分別宣布：1966年為「中、菲友誼年」。<ref name="記" />{{rp|724}}8月7日，菲律賓外交部長{{link-en|納西索·羅慕斯|Narciso Ramos|羅慕斯}}（Narciso Ramos）訪臺；8月9日返國前提出亞洲和平運動主要步驟三項。

1968年8月29日，駐菲律賓大使[[杭立武|杭立武]]與菲外長羅慕斯為處理逾期[[華僑|華僑]]問題達成協議，並簽署聯合聲明<ref name="記" />{{rp|757}}。

1970年5月2日，[[菲律賓副總統|菲律賓副總統]]{{link-en|費南多·羅培士|Fernando Lopez|羅培士}}（Fernando Lopez）訪臺；5月6日，簽訂《中、菲[[農學|農技]]協定協議[[備忘錄|備忘錄]]》。<ref name="記" />{{rp|726}}。

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[菲律賓|菲律賓]]。]]

1971年7月29日，原由中華民國控制的[[中業島|中業島]]遭[[菲律賓|菲律賓]]併吞，菲方并將島嶼改名为“派格阿萨岛”（Pag-Asa）。<ref>[http://udn.com/NEWS/WORLD/WOR3/6476176.shtml 1971年菲律賓攻佔中业岛]</ref> 中华民国海军曾打算抵抗，但因當時接獲中華民国政府急电：不挑战，故並未作出抵抗。<ref>《陆战薪传》，王懿辉，台湾海军陆战队司令部</ref>同年10月25日，菲律賓投票支持[[中國與聯合國#代表權問題|中華民國續留聯合國]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。

1975年6月9日，兩國[[外交#外交上的承認|斷交]]。<ref name="105年外交年鑑"/>

獲准到[[美國|美國]]就醫的[[反對黨|反對黨]]領袖[[班尼格諾·艾奎諾|班尼格諾·艾奎諾]]經長期協調抗爭後，在時任總統[[馬可仕|馬可仕]]保證其安全的情況下在1983年獲准回菲律賓參加[[菲律宾国会|國會]]選舉，並於8月21日經[[臺灣桃園國際機場|中正機場]]搭乘[[中華航空公司|中華航空公司]]班機回馬尼拉，卻於步下飛機階梯時遭人開槍擊中頭部身亡，雖然機場警察隨即槍斃兇手，並拘捕了數名疑兇，但一般認為該謀殺為馬可仕政府暗中策劃。馬可仕政府立即中止與中華民國間的航約，約一個月後恢復。艾奎諾遺孀[[柯拉蓉·艾奎諾|柯拉蓉·艾奎諾]]於1986年領導「[[人民力量革命_(1986年)|人民力量革命]]」就任總統，推翻馬可仕政權。

1987年10月13日，菲律賓最高法院首席大法官{{link-en|鄭漢淇|Claudio Teehankee|鄭建祥}}夫婦抵台訪問。期間會見司法院長[[林洋港|林洋港]]、外交部長[[丁懋時|丁懋時]]及其他政府官員，並接受[[輔仁大學|輔仁大學]]頒贈[[榮譽博士學位|榮譽博士學位]]。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=014732DC0E0BAD62 |author = 公眾外交協調會 |date = 1987年10月13日 |title = 菲律賓共和國最高法院首席大法官鄭建祥 |publisher = 中華民國外交部}}</ref>

1993年6月，甫卸任總統的艾奎諾夫人抵台訪問，主持菲律賓駐台辦事處揭幕典禮，並與[[台北市長|台北市長]][[黃大洲|黃大洲]]和總統[[李登輝|李登輝]]會晤。<ref>[http://catalog.digitalarchives.tw/item/00/3a/a1/a0.html 黃大洲與菲律賓艾奎諾夫人主持菲國駐華辦公室啟用]，典藏台灣</ref><ref>[http://catalog.digitalarchives.tw/item/00/60/86/e5.html 李登輝活動輯要—民國八十二年六月(007-040101-00061-009)]，典藏台灣</ref>

===援助===
1990年7月16日，菲律賓呂宋島{{link-en|1990年呂宋地震|1990 Luzon earthquake|發生強烈地震}}後，[[中華民國政府|中華民國政府]]自「國際災難人道基金」中撥款20萬[[美元|美元]]協助賑災救難工作。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=C39FC98A294B5D4F |author = 公眾外交協調會 |date = 1990年7月18日 |title = 菲律賓於本月十六日發生強烈地震 |publisher = 中華民國外交部}}</ref>

1991年6月，菲律賓[[皮納土波火山|皮納土波火山]]爆發造成呂宋島慘重災情，中華民國政府捐贈20萬美元及避難用[[帳篷|帳篷]]200頂。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=36F70B8870373107 |author = 公眾外交協調會 |date = 1991年6月20日 |title = 對於菲律賓品納土玻火山連續爆發造成該國呂宋島慘重災情事 |publisher = 中華民國外交部}}</ref>11月，菲律賓遭受[[熱帶風暴黛瑪_(1991年)|熱帶風暴賽洛瑪]]侵襲，造成數千人死亡，中華民國政府再撥款20萬美元。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=E8DF122F71C12B64 |author = 公眾外交協調會 |date = 1991年11月18日 |title = 菲律賓日前遭受強烈颱風侵襲 |publisher = 中華民國外交部}}</ref>

2013年11月，[[颱風海燕_(2013年)|強烈颱風海燕]]襲擊菲律賓中部省分，導致災情、死傷慘重。中華民國政府除第一時間宣佈捐贈[[菲律賓政府|菲律賓政府]]20萬美元協助賑災外，[[中華民國外交部|外交部]]也積極洽請社會各界踴躍捐輸，在10日內即募得680[[公噸|公噸]]賑災物資，並協調由[[中華民國國防部|國防部]]分別派遣空軍18架次[[C-130運輸機|C-130運輸機]]及海軍[[新港級戰車登陸艦|中和艦]]裝載物資馳援。至2014年2月，政府及社會各界捐款及捐獻物資合計約1,270萬美元。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/133bf67f-6547-4c62-8bfd-ef26e383f644.pdf |author = 國際合作及經濟事務司 |date = 2014年4月23日 |title = 國際合作發展事務102年度報告（參、102年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref>

===觀光／勞務===
2017年，台灣遊客前往菲律賓有23萬6,000人次；菲律賓旅客赴台則約29萬人次，比起2016年的17萬人次有大幅成長。<ref name="106年外交年鑑"/><ref name="105年外交年鑑"/>

近年，台灣前往菲律賓經商、應聘或投資入境的技術、工商等從業人員約6,000人；2017年，[[在臺菲律賓人|在臺灣的菲籍勞工]]約15萬人。<ref name="106年外交年鑑"/>
