[[中华人民共和国铁路客运|中华人民共和国铁路客运]]经过多年的发展，车票样式与票价体系经过多次调整，且车票也有多种。目前，中国大陆的铁路客票包括普通票、通票、联程票、返程票、异地票、儿童票、学生票、[[站台票|站台票]]等票种。

==相关法律规定==
依据《[[:s:铁路旅客运输规程_(2010年10月13日)|铁路旅客运输规程]]》规定，[[车票|车票]]是铁路旅客运输合同的基本凭证，属于承运人与旅客之间权利义务关系的协议，自售出车票时起成立，至按票面规定运输结束旅客出站时止。该《规程》同时规定，车票中包括客票和附加票两部分。

另外，車票亦作为旅客進站上車的憑證，限'''單次'''使用。同時，旅客必須保留車票用作列车上查验车票及出站之用，否則需要'''補票'''。

==车票种类==
在中华人民共和国铁路客运历史上，车票曾出现多种介质（包括纸质票和电子车票），而特殊车票亦出现多种。

===普通纸质车票===
普通纸质车票为中国铁路客运车票常见的车票介质，历史上曾先后出现过硬板票、软纸票和磁介质车票三种。现时仍在使用的车票介质为软纸票和磁介质车票，而硬板票目前已停止使用。

====硬板票====
[[File:1993_Zhengzhou_Beijing_Railway_Ticket.jpg|缩略图]]至[[北京站|北京]]252次（即今[[K179/180次列车|K180次]]）硬臥車票正反面，使用硬板票版式]]
[[埃德蒙森式鐵路車票|硬板票]]是中国铁路客运车票中连续使用时间最长的版式，自中华人民共和国成立前的[[中华民国大陆时期|民国时期]]便开始使用。此种车票为事先印刷好的车票，票面只有车站、票价等信息。在未实行对号入座时，车票未标明座号，而在铁路实行对号入座制度后，车票后面需要由售票员加贴一张印有车次、发车时间和座位号的小纸条。另外每张车票还会用符号标注列车的级别，其中绿色一道杠为普速快车，绿色两道杠为特快列车，红色两道杠为软座特快。在硬板车票时代，实行“以无座票为基础，有座票作为补充”的制度<ref>{{Cite web |url=http://www.chinanews.com/cul/2013/02-06/4552566.shtml |title=民国铁路客票售票一律"无座" 订座需额外花钱 |accessdate=2018-12-09 |publisher=新晚报 |date=2013-02-06}}</ref><ref name=yzwb>{{Cite web |url=http://www.chinanews.com/cj/2018/10-23/8657395.shtml|title=三十张火车票上演满满的“回忆杀” 老铁路人讲述火车票变迁史 |accessdate=2018-12-09 |publisher=扬子晚报|author=记者 徐媛园 通讯员 祖韬 |date=2018-10-23}}</ref><ref name=xlzh>{{Cite web |url=http://finance.sina.com.cn/consume/xiaofei/2018-02-09/doc-ifyrkzqr0561991.shtml|title=春运30年:从“硬板儿票”到磁制票 卖票速度快了30倍|accessdate=2018-12-09 |publisher=新浪综合 |date=2018-02-09}}</ref>。

硬板票的出售过程相当繁琐，用时需三到五分钟，甚至更长时间。售票员需要将不同的票按类分到不同的格子柜中，然后根据旅客的目的地，从柜子中找到所需要的票，用[[针孔打印机|针孔打印机]]打印日期，再用胶水在车票背面粘贴小纸条，最后盖完章才能将票卖出。此外，车票上有沿线主要站点，但如果旅客需要买的到站车票上没有，则售票员需要手写一张“区段票”若火车上需要购票或者补票，列车员则要手写一张“代用票”<ref name=yzwb/><ref name=xlzh/>。当年售票员的入职要求严格，需要在客运段工作满三年以上才能调入售票车间。在售票车间，师傅还需要培训徒弟卖票三个月才能出徒。售票员还需要使用[[算盘|算盘]]计算票款和找零，报出正确的票价，才能将车票和款项交到旅客手里<ref name=yzwb/><ref name=cctv>{{Cite web |url=http://www.chinanews.com/cj/2018/02-13/8448019.shtml|title=中国火车票务系统每天1500亿浏览量 1秒钟卖票700张|accessdate=2018-12-09 |publisher=央视财经  |date=2018-02-13}}</ref>。

自1990年代后期防伪技术较高的计算机打印的软纸票出现后，软纸票开始逐步取代硬板票的地位，但在一些不具备电脑售票的车站仍适度保留硬板票<ref>{{Cite web |url=http://finance.people.com.cn/GB/1038/4718804.html|title=第二百亿张硬板火车票下线  见证铁路发展|accessdate=2018-12-09 |publisher=北京青年报  |date=2006-08-18}}</ref>。2007年6月30日，[[北京铁路局|北京铁路局]]辖内的[[青龙桥站|青龙桥站]]售出最后一张硬板票，至此硬板票全面停止使用<ref name=cctv/>。

====机打票====
=====软纸票=====
[[File:Rail_ticket_of_a_sleeper_in_northern_China.jpg|缩略图]]
[[File:一张哈尔滨站至宋站的火车票.jpg|缩略图]]]]
1980年代，[[深圳站|深圳站]]率先使用计算机售票，车票也改为软纸式车票<ref>{{Cite web |url=http://news.cnnb.com.cn/system/2018/11/25/030005420.shtml|title=告别纸质火车票！铁总：电子客票将于明年全国推广|accessdate=2018-12-09 |publisher=央视财经  |date=2018-11-25}}</ref>。

1996年初，[[上海铁路局|上海]]、[[北京铁路局|北京]]等铁路局开始试点计算机打印的软纸票。同年末，原[[中华人民共和国铁道部|铁道部]]确立了软纸客票统一样式标准，正式开始使用软纸车票。起初软纸票底色为蓝色，之后改为淡粉色。此种车票为售票时现场打印，且售票时间大大缩短。至2000年时，软纸票版式进行了一定的调整。此后铁路部门再次对车票板式进行调整，将列车车次调整至出发站与到达站之间，同时原“全价”字样改为人民币符号“￥”<ref name=ysxw>{{Cite web |url=http://news.sina.com.cn/o/2018-01-20/doc-ifyquixe5235026.shtml|title=一张火车票：厚厚的历史 满满的回忆|accessdate=2018-12-09 |publisher=央视新闻  |date=2018-01-20}}</ref>。

由于互联网售票的发展，加上近年来[[客运专线|客运专线]]的开通、车站[[验票闸门|自助检票闸机]]的投入使用，以及对防伪安全性要求的提高，软纸票开始逐步退出历史舞台。2017年6月，北京铁路局管内车站和代售点停止发售软纸票，统一发售磁介质票<ref name=tzrzp>{{Cite web |url=http://www.chinanews.com/sh/2017/06-16/8252121.shtml|title=北京终结20年粉色软纸火车票 不适应自助检票等需求|accessdate=2018-12-09 |publisher=北京青年报  |date=2017-06-16}}</ref>。

=====磁介质票=====
[[File:Train_tickets_blue.jpg|缩略图]]
2007年4月18日，中国铁路首次开行动车组列车，同时在全国范围发售磁介质票。初期磁介质票的底色为粉红色，四角为弧型，背景图案为一列动车组列车（车型为[[CRH2A|CRH2A]]）从右向左快速行驶的画面。此后底色改为浅蓝色。此种车票可在车站的自助检票闸机上使用，提高了检票速度和准确率<ref name=ysxw/><ref name=tzrzp/>。

====学生票====
[[File:CRH_Student_Ticket.jpg|thumb]]

为方便学生在家校之间往返，减轻没有收入的学生的经济负担，在[[大学|大学]]、[[大专院校|大专院校]]，[[中等专业学校|中等专业学校]]、[[技工学校|技工学校]]和中、小学就读，家校不在同一地的没有[[工资|工资收入]]的[[学生|学生]]、[[研究生|研究生]]，可以购买学生票。学生证上贴有[[中华人民共和国教育部|教育部]]和[[中华人民共和国铁道部|铁道部]]监制的火车票学生优惠卡，并填有优惠的区间，持有该种证件的学生每年可以购买四次往返于家校之间的单程车票，当年没有用完的，不能转移到下一年。<ref name="pax_regulation"/>学生票硬座为全票票价的一半，购买硬卧在同种席位全价基础上减去同区间硬座票价的一半，软席不售学生票；[[动车组|动车组]]只发售二等座学生票，价格为公布运价全价票的75%。<ref>[http://www.12306.cn/mormhweb/tlcs/201001/t20100121_1525.html 铁路客户服务中心--哪些学生可以购买学生票？]</ref>购买学生票时需要出示学生证，且在上车后验票时遇学生票亦须检查学生证，若未携带学生证或者区间不对应则可能需要补交车票差价或全价。

国际联运列车的学生票票价减成25%，即同中国内地动车组列车学生票折扣相同，为车票全价的75%。另外，在中国内地大专院校就读，没有工资收入的香港籍学生，凭附有加盖院校公章的减价优待证的学生证和火车票学生优惠卡（新生凭录取通知书，毕业生凭学校书面证明可购买一次学生票），可购买家庭至学校所在地（或实习地点）至[[香港西九龙站|香港西九龙站]]的学生票<ref name=gsg/>。

==票面版式与制票机==
中国大陆地区的铁路车票，国内与国际列车版式完全不同。

根据现行规定，中国大陆境内的铁路客运车票正面必须载明以下内容：
*出票地点：标示于车票右上角，2015年6月25日起改为车票最下方21位售票码的右边（右上角改为检票口地点），代表出票的车站或代售点所属车站。
*发站和到站站名：除了以[[中文|中文]]显示，下方附有站名的[[汉语拼音|汉语拼音]]或英文，2015年6月25日起在始发站和到达站的站名后均标注上了“站”字。
*车次：参见[[中国铁路列车车次|中国铁路列车车次]]。
*运行径路（仅限通票）：标示于发站站名下方，代表列车的运行经由和运价里程接算车站。
*通票等级（仅限通票）：标示于到站站名下方，以“普客/普快/特快到<到站>”方式表示，代表通票等级。
*乘车日期及发车时间：時間為[[北京時間|北京時間]]，以[[24小时制|24小时制]]表示。
*车厢号及座号或卧铺号：以“XX车XXX号”方式表示，部分城际铁路（采用CRH6型）则标注不对号入座。
*票价及票种或支付方式简称。
*列车等级，及座席或卧席等级：参见[[中华人民共和国铁路客运#旅客列车等级及车次|旅客列车等级及车次]]。
*乘车人身份证号和姓名：上排为身份证号（为保护隐私其中四位数用*代替），下排为乘车人姓名，2015年6月25日起改为并排标注。
*标注“{{Lang|zh-hans|买票请到12306 发货请到95306 中国铁路祝您旅途愉快}}”字样，2015年6月25日起新增（往返[[香港西九龙站|香港西九龙站]]的车票则会标注“{{Lang|zh-hans|跨境旅客须知详见旅客相关运输组织规则和车站公告}}”）
*车票原则上限乘当日当次车，中途下车前程失效，运输合同即行终止。

为提高火车票的防伪性能，铁道部决定自2009年12月10日起，采用票面样式经过重新设计的车票（包括纸质和磁卡车票）。新版车票的最大特点是原来在车票下方的防伪[[条码|条码]]换成[[QR码|二维图案]]<ref>[http://www.17u.net/news/newsinfo_233715.html 12月10日起火车票将改版]</ref>。中国铁路的车票版式于2015年6月25日再次发生变动，并于8月1日全部替代旧有版式。<ref>{{cite news|url=http://jj.jxnews.com.cn/system/2015/06/25/013988189.shtml|title=新版火车票8月1日起全面启用 车站名检票口更醒目|publisher=[[九江晨报|九江晨报]]|date=2015-06-25|accessdate=2015-07-01}}</ref>

==购票方式==
在12306网站上线前，旅客通常需要前往车站购票窗口和代售点购买车票。随着自动售票机和网络售票的普及，绝大多数旅客开始通过网络售票，而车站购票窗口和代售点的排队情况也得到了缓解。据相关部门统计，2018年[[春运|春运]]售出的3.5亿张车票中，通过12306平台销售的达2.8亿张<ref name=cctv/>。

从2011年6月1日起，全国所有铁路列车实行购票[[中国大陆火车票实名制|实名制]]。乘客需携带有效身份证件方可购票，有效证件包括：[[中华人民共和国居民身份证|居民身份证]]、按规定可使用的有效[[护照|护照]]、[[港澳居民来往内地通行证|回鄉證]]、[[往来港澳通行证|港澳通行证]]、[[台湾居民来往大陆通行证|台胞證]]、[[港澳台居民居住证|港澳台居民居住证]]等<ref>{{cite news|title=配合实名制京津城际快通卡升级 |url=http://www.peoplerail.com/rail/show-456-47766-1.html |publisher=人民铁道报 |date=2011-05-21 |accessdate=2016年3月28日 }}</ref><ref>{{cite news|url = http://news.xinhuanet.com/fortune/2010-01/10/content_12783988.htm|title =铁路部门详解春运火车实名购票流程|accessdate =2010-01-10 |author =新华闻综合|date =  2010-01-10|language = zh-hans|publisher =[[新华网|新华网]]}}</ref>。

==退票、改签与补票==
中国大陆地区境内运输的铁路列车与国际/香港联运列车的实行规定完全不同。

中国大陆境内运输的旅客列车的铁路客票可在票面标明的开车时间后最迟两小时之前进行一次改签。在原票开车前进行改签所得新票，在其开车时间之前仍可进行退票；原票开车后改签的车票，则无法退票，票面标注'开车后改签不予退票'字样。改签不收取手续费。<ref name="pax_regulation"/>

改签网购车票时，若新票票价高于原票票价，须用银行卡支付新票全额票款，原票款将在银行规定时间内退回购票时所使用的银行卡；若新票票价低于原票票价，应退票款将在银行规定时间内退回购票时所使用的银行卡或支付宝（微信支付）账户（办理退票时同理）；若改签后的新票票价等于原票票价的，则无需办理支付手续。

中国大陆境内运输列车的团体旅客办理车票退改签业务须在开车时间48小时以前办理。如购票时一并支付过代售点服务费，此项费用不退。

2013年8月，中国内地铁路运输企业再度修改客票退改签政策，实行在大陆地区“通签通退”，即内地车站发售的铁路车票可于规定时间（改签：原则上最迟开车后2小时之前；退票：开车前）在全路任一有条件的铁路车站办理退改签业务。此前，退改签业务办理范围仅限票面始发站、票面右上角出票地车站及二者的同城车站。退票费的收取由2011年9月25日<ref>{{cite web| url = http://news.sohu.com/20110916/n319500147.shtml| title = 火车票退票费由20%降至5% “霸王条款”终低头| publisher = 京华时报| date = 2011-09-16| accessdate = 2013-08-29}}</ref>开始实行的单一标准“票面数额的5%，最低2元”，转变为随退票时间而变的梯度化模式。退票时间距票面开车时间前48小时以上，退票费用仍为5%；距开车时间24小时-48小时，退票费为票价的10%；距开车时间24小时直到开车前，退票费为票价的20%。最低标准仍为2元。<ref>{{cite web| url = http://www.12306.cn/mormhweb/zxdt_ggxx/201308/t20130826_892.html| title = 
9月1日起火车票全国通退通签并实施梯次退票| publisher = 中国铁路总公司| date = 2013-08-26| accessdate = 2013-08-29}}</ref>

对于[[香港西九龙站|香港西九龙站]]始发终到的列车，则以《廣深港高速鐵路跨境旅客運輸組織規則》的规定为准。根据《规则》，旅客仅可办理一次变更乘车日期、车次、席别的改签手续，且改签后不能退票。办理改签的时间不晚于票面指定的日期、车次开车前30分钟，始发站为香港西九龙站的车票应不晚于开车前60分钟。且改签后的车票不得退票<ref>{{Cite web|url=http://www.xinhuanet.com/2018-09/23/c_1123472744.htm|title=广铁集团详解广深港高铁搭乘须知ABC|accessdate=2018-09-24|work=新华网|language=zh-hans|archive-url=https://web.archive.org/web/20180923105510/http://www.xinhuanet.com/2018-09/23/c_1123472744.htm|archive-date=2018-09-23|dead-url=no}}</ref>。

国际联运旅客列车的境内回转车车票退票，与上述国内车票相同，但是其跨境和境外段车票，退票规定与内地车票不同：国际列车发车的72小时前办理退票时，核收票价的5%，购票手续费不退还。国际列车发车的72小时内办理退票或改签，核收客票票价的5%以及卧铺票票价的80%作为退票费，购票手续费不退还。国际联运列车开车后3小时内退票或改签，卧铺票作废，核收客票票价的5%，购票手续费不退还。改签需重新购买卧铺票。

国际团体旅客在发车前5日内不予退票或改签。国际列车发车3小时后或已托运行包后，不再办理退票，不得改签。购买团体乘车票据，不对其中一部分旅客办理退票或改签。票据丢失按无票处理，继续乘车需重新购票。

如旅客未赶上国际联运旅客列车，则自列车出发时起3小时以内，有权向发售乘车票据的机构申明此事并改乘下次列车，但卧铺票除外。

===补票===
列车长办公席、车站出站口补票处均可办理此项业务。若补票行为是自发的，则只在应付票额基础上多收2元手续费；若被站车工作人员勒令补票，则视当事列车/车站的不同，有可能会被加罚应付票额的50%并核收手续费<ref name="pax_regulation"/>。2012年5月起，旅客若遗失车票，可于不晚于票面载明的开车时间前30分钟在车站窗口凭原票乘车人有效身份证件办理车票挂失补办手续，凭再次缴纳票款后得到的新票进站上车，新票票面上注明“挂失补”字样。之后旅客须主动联系所乘列车车长确认席位使用正常并开具客运记录，到站时在列车长陪同下与车站值班员进行站车交接。出站后，旅客凭新票、带有列车长和值班员签字的客运记录、票面乘车人有效身份证件进行退票，全程收取手续费2元。

自2017年1月1日起，铁路部门开始实施新的补票规定。根据新规，如果旅客在检票进站前丢失火车票，需在车票票面发站停止售票前到车站售票厅指定窗口办理挂失补办手续，按原车票车次、席位、票价重新购买一张新车票。旅客上车后，需向列车工作人员声明，经列车核验“挂失补车票”、购票时所使用的有效身份证件原件与乘车人一致，并在到站前确认未发现原车票被他人使用后，可开具客运记录，与“挂失补车票”一并作为退票的凭证。到站后向车站出站口工作人员声明并配合查验，可于24小时内办理退票手续，退回“挂失补车票”票款，不收退票费，仅收取2元手续费。但对于在列车上丢失火车票的旅客，需向列车工作人员声明，经列车查验乘车人、购票时所使用的有效身份证件原件、购票信息一致后，可办理挂失补办手续，不需要缴纳全额票款，仅收取2元手续费，票面标注“车票丢失”字样。到站前核验席位使用正常后，开具客运记录。到站后向车站出站口工作人员声明并配合查验，凭票面标注“车票丢失”字样车票、客运记录和购票时所使用的有效身份证件原件办理出站检票手续<ref>{{cite web|url=http://e.hznews.com/paper/djsb/20170116/A06/1/|title=火车票丢失 花2元可补办|date=2017-01-16}}{{Dead link|date=2019年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

国际联运列车的进站验票，身份验证以及车上补票等规定，由[[铁路合作组织|铁组]]部门规章约束之，不受上述规定限制。

==票价==
=== 基准票价 ===
中国铁路'''基础'''客运票价在1989年、1995年分别从0.01855元/公里提价到0.03861元/公里和0.05861元/公里。此后一直未涨。

中国大陆目前的铁路客运票价是由客票票价与附加票票价组成。基本票价以每人每公里的票价率为基础，加上按各种列车等级、空调、席别的上浮比率，以“递远递减”的方式计算。其中列车运行公里数以《铁路客运运价里程表》上公布的为准。硬座（含无座）为起始基准票价率，其他席别如硬卧、软座、软卧等均以此基准票价率上浮。卧铺的上铺价格低于中铺，中铺低于下铺。而列车等级按非空调普客为基准标价，以上等级按一定幅度上浮。其中，快速、特快和直达特快上浮比率相同，而普快上浮较低一些。如果为空调列车，则按车辆种类（依[[列车供电|列车供电]]方式）以两种上浮比率计算，分别为车辆自供电的“空调客车”及发电车集中供电的“新型空调客车”（新空调），其中“新型空调客车”的上浮率也比“空调客车”高，一般新空调客车票价是在普通车基础上上浮50%。列車的无座票價與硬座相同。

2013年起，不再加收以基本票价的2%计算的旅客意外伤害强制保险费<ref>《铁路旅客意外伤害强制保险条例》，1951年4月24日，政务院财经委</ref>。

目前普通线路的動車組列车公布票价是鐵道部按照原[[國家計委|國家計委]]在1997年发布的《鐵路客運運價規則》制定，其中二等座以当时的高等級軟座快速列車作基准价格<ref>[http://www.southcn.com/news/china/zgkx/200704120501.htm 動車組票價執行高等級軟座快速列車價格]</ref>，上浮幅度與硬卧相若，而一等座上浮幅度與软卧相若，但不实行“递远递减”原则。然而各鐵路局可視市場情況執行打折票價<ref>[http://travel.people.com.cn/BIG5/41636/41644/5640804.html 中國鐵路高速動車組列車票價得到靈活執行]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。长距离动车组通常会按客流情况在公布票价的基础上执行折扣。動車組站票票價則與二等座相同。

新建高速铁路和客运专线的动车组票价由中国铁路总公司（原铁道部）制定。[[2011年甬台温铁路列车追尾事故|2011年甬台温铁路列车追尾事故]]发生后，所有动车组票价统一下调5%。现行动车组票价計價公式如下：

*普通动车组（D字头車次）一等座票价：0.37×里程
*普通动车组（D字头車次）二等座票价：0.3×里程
*高铁动车组（G字头車次）一等座票价：0.74×里程
*高铁动车组（G字头車次）二等座票价：0.46×里程

动车组列车票价执行“递远递减”的计价原则，分段实行折扣。其中500公里以内执行原价，500—1000公里约执行9折，1000公里以上约执行8折（此为大致标准，各线路的具体票价计算方法，请以国家发改委的文件为准）。而对于运行于涉及200公里线路的高速动车组列车而言，其票价则以200公里线路票价计算。例如，北京南至青岛的高速动车组列车，北京南至济南段票价按照300公里标准计算，济南至青岛段则按照200公里标准计算<ref>{{cite news|title=高铁票是如何定价的？为啥有时比飞机票还贵？有人算清楚了 |url=http://society.cnr.cn/js/20170209/t20170209_523577925.html|publisher=央广网|date=2017-02-09 |accessdate=2017-02-14 }}</ref>。

===票价浮动===
对于地方铁路及特殊区段的线路而言，在票价方面有一定的上浮标准，不执行标准国铁运价的国铁线路称特殊运价区段，如[[青藏铁路|青藏铁路]][[格尔木|格尔木]]至[[拉萨|拉萨]]段实行上浮，每公里票价上浮：硬座不上浮，软座0.09元，硬卧0.10元，软卧0.16元。而因[[廣深鐵路|廣深鐵路]]由[[广深铁路股份有限公司|广深铁路股份有限公司]]营运，是地方合资上市铁路，{{Fact|其上浮率也是全国最高的|time=2014-06-25T12:59:31+00:00}}，1994年允許廣深線准高速铁路软座票价在全國同一運價基礎上浮50%，1996年原广深铁路公司实行股份制改造，票价再次上浮50%，并可在规定浮动幅度内，由企业自主定价<ref>[http://view.news.qq.com/a/20070108/000033_1.htm 中国首次为铁路客运举行价格听证会]</ref>，此特殊运价政策一直实行至今，总上浮率达225%。

在动车组列车方面，从2015年起，铁路开始基于[[大数据|大数据]]的分析，试点浮动票价机制<ref name=ysxw2>{{cite news |author1=|title=铁路将逐步实行“一日一价” 以后坐火车出行也要挑日子啦！  |url = http://www.xinhuanet.com/fortune/2018-04/20/c_1122712527.htm|accessdate=2018-12-23 |newspaper = 央视新闻微信公众号|date=2018-04-20}}</ref>。2015年12月6日开通的[[宁安客运专线|宁安客运专线]]开始，中国铁路新开通的200—250KM级别高速铁路（如南百、丹大、金丽温）的二等座票价均较标准上浮20%，约为0.37元/公里。

根据[[国家发改委|国家发改委]]发布的《[[:s:关于改革完善高铁动车组旅客票价政策的通知|关于改革完善高铁动车组旅客票价政策的通知]]》，从2016年1月1日起，动车组列车票价改由中国铁路总公司自行定价<ref>{{cite news|title=热门线路或有提价空间 铁总获高铁票价定价权 |url=http://news.xinhuanet.com/finance/2016-02/20/c_128736003.htm |publisher=中国经济网 |date=2016年2月20日 |accessdate=2016年2月20日 }}</ref>。

自2017年4月21日起，运行于[[杭深线|杭深线]]的时速200-250公里的动车组车票票价将作出调整，这是铁总取得高鐵車票自主定價權後，首次上調票价的线路<ref>{{cite news|title=12306客服：东南沿海高铁动车组票价自4月21日调整 |url=http://finance.sina.com.cn/roll/2017-02-14/doc-ifyamkpy9319259.shtml|publisher= 新华网 |date=2017-02-14 |accessdate=2017-02-14 }}</ref><ref>{{cite news|title=大陸部分高鐵調漲 一等座漲幅近7成 |url=http://www.chinatimes.com/realtimenews/20170213005147-260409|publisher= 中央社 |date=2017-02-13 |accessdate=2017-02-14 }}</ref>。根据铁总官方发布的公告，杭深线宁波至厦门段大部分动车组列车一等座涨幅最高超50%，二等座涨幅16%-20%，同时有少量车次下调票价。此次票价优化调整后，旅客乘坐动车出行在旅行时间和票价方面仍具有一定优势，而且有利于改善公司的经营状况，不断更新改进服务设施和条件，为旅客创造更好的旅行环境<ref>{{cite news|title=中国高铁迎首次跨省调价 一等座最高涨幅超50% |url=http://news.sina.com.cn/o/2017-04-12/doc-ifyecfnu8251838.shtml|publisher= 一财网 |date=2017-04-12|accessdate=2017-04-12 }}</ref>。

*2017年底，铁路部门对14条动车组列车运营线路动车组列车票价开展浮动试点。
*2018年4月28日起至年底，铁路部门下调28条城际铁路（包括[[广珠城际铁路|广珠城际铁路]]、[[海南环岛高速铁路|海南环岛高速铁路]]、[[宁安城际铁路|宁安城际铁路]]、[[丹大城际铁路|丹大城际铁路]]、[[青荣城际铁路|青荣城际铁路]]、[[郑开城际铁路|郑开城际铁路]]、[[武孝城际铁路|武孝城际铁路]]等）的部分动车组列车票价，部分线路的折扣达到20%<ref>{{cite news|title=部分动车票价优惠幅度提高至20%|url=http://www.gov.cn/xinwen/2018-03/30/content_5278422.htm|publisher= 经济日报|date=2018-03-30|accessdate=2018-12-23 }}</ref>。同时对[[京沪卧铺动车组列车|京沪卧铺动车组列车]]的票价实行浮动机制<ref name=ysxw2/>。
*2018年5月27日起，[[京津城际铁路|京津城际列车]]一等座、特等座票价涨价，其中一等座票价由65.5元上涨到88元，特等座票价由93.5元上涨到99元<ref>{{cite news |author1=裴剑飞 |title=京津城际一等座票价上调22.5元 二等座票价保持不变  |url = http://www.xinhuanet.com/fortune/2018-05/27/c_1122893048.htm|accessdate=2018-06-01 |newspaper = [[新京报|新京报]] |date=2018-05-27}}</ref>。
*2018年7月5日起，铁路部门对运行于[[沪宁城际铁路|沪宁城际铁路]]、[[宁杭客运专线|宁杭客运专线]]、[[宁合铁路|宁合铁路]]、[[汉宜铁路|汉宜铁路]]、[[贵广客运专线|贵广客运专线]]和[[柳南城际铁路|柳南城际铁路]]，且时速为200至250公里的动车组列车，分季节、分时段、分席别、分区段在限价内实行票价下浮，最大折扣幅度6.5折<ref>{{cite news |author1=刘伟娟|title=南京至武汉高铁时速将达250公里 |url = http://dz.xdkb.net/html/2018-05/06/content_486239.htm|accessdate=2018-06-01 |newspaper = [[现代快报|现代快报]] |date=2018-05-06}}</ref>。
