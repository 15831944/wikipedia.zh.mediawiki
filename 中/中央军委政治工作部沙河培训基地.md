'''中央军委政治工作部沙河培训基地'''，位于[[北京市|北京市]][[昌平区|昌平区]]七里渠小沙河村328号，隶属[[中央军委政治工作部|中央军委政治工作部]]。

==简介==
该基地原名[[中国人民解放军总政治部|中国人民解放军总政治部]]沙河培训基地。其中设有中国人民解放军总政治部沙河招待所。2016年，中国人民解放军总政治部沙河招待所获评为中央和北京市的北京地区（城六区）2016-2018年度会议定点单位<ref name=diyi>{{Cite web |url = http://www.ccgp.gov.cn/cggg/dfgg/zbgg/201701/t20170117_7867097.htm |title = 北京地区（城六区）2016-2018年度会议定点单位协议采购项目中标公告：006中标公告附件1：第一包中标供应商名单 |author =  |date = 2017-01-17 |publisher =中国政府采购网  |language =  |accessdate =  |quote =  }}</ref><ref name=caigou>{{Cite web |url = http://hy.bgpc.gov.cn:82/MEETING_2009/platform/organ/selectTInfHotel.action?ofooter=P1&tibhPk=20161115JJ768801075 |title = 中国人民解放军总政治部沙河招待所 |author =  |date =  |publisher = 北京市政府采购会议定点综合查询系统 |language =  |accessdate = 2017-03-28|quote =  }}</ref>。

2016年，中国人民解放军总政治部撤销，该基地更名为中央军委政治工作部沙河培训基地<ref>{{Cite web |url = http://www.chinanews.com/cul/2017/01-04/8113511.shtml |title =画家刘飞出席当代书画艺术名家邮品首发式  |author =  |date = 2017-01-04 |publisher =  中国新闻网|language =  |accessdate =  |quote =  }}</ref>。
