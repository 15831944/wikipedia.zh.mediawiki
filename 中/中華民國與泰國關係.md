==歷史==
===冲突===
1942年－1943年，中华民国与泰国爆发[[中泰邊境戰爭|中泰邊境戰爭]]。

===泰北孤軍===
{{see also|泰北孤軍}}
1949年後期由於[[第二次國共內戰|國共內戰]]失利，部分[[中華民國國軍|中華民國國軍]]及其眷屬從雲南向緬甸撤退，其中兩支最終停留在泰國，改編為「[[泰北志願自衛隊|泰北志願自衛隊]]」，中華民國與泰國邦交期間，泰國默許中華民國軍隊駐守其北部邊境。1975年由於協助泰國政府平定[[泰國共產黨|泰國共產黨]]游擊隊有功，其參戰者全數給予泰國[[公民|公民]]身份，其眷屬亦給予[[永久居留權|永久居留權]]，使其得以合法居留泰國。後來該軍後裔逐漸融入泰國社會，部分後裔亦前往[[台灣|台灣]]求學就業，而中華民國政府亦提供資源協助其生活。

===外交===
{{see also|中華民國外交|{{link-en|泰國外交|Foreign relations of Thailand}}}}
{{see also|中国驻泰国大使列表{{!}}中華民國駐暹羅／泰國外交官列表|泰国驻华大使列表{{!}}暹羅／泰國駐中華民國外交官列表}}
1946年1月23日，簽署《中華民國暹羅王國友好條約》{{noteTag|1945年9月8日，泰王國更名為暹羅王國。1949年5月11日，暹羅王國復名為泰王國。}}，建立大使級[[外交#外交上的承認|外交關係]]，並互相設立[[外交代表機構#大使馆|大使館]]、派駐[[特命全權大使|大使]]。<ref name="條約協定查詢系統"/>

1949年，[[臺灣戰後時期|中華民國政府遷台]]之後，兩國仍維持邦交關係。

1963年6月5日－8日，泰王[[蒲美蓬|蒲美蓬]]及[[詩麗吉王后|詩麗吉王后]]訪問臺灣4日，[[中華民國總統|總統]][[蔣中正|蔣中正]]與[[中華民國第一夫人|第一夫人]][[宋美齡|宋美齡]]夫婦親至[[臺北松山機場|松山軍用機場]]以隆重[[軍禮|軍禮]]歡迎。<ref>[http://www.taiwanembassy.org/TH/ct.asp?xItem=79018&ctNode=1760&mp=231&nowPage=3&pagesize=15 泰國蒲美蓬國王及詩麗吉王后1963年6月5日至8日訪問中華民國珍貴歷史照片紀錄] {{webarchive|url=https://web.archive.org/web/20160304084146/http://www.taiwanembassy.org/TH/ct.asp?xItem=79018&ctNode=1760&mp=231&nowPage=3&pagesize=15 |date=2016年3月4日 }}, 駐泰國臺北經濟文化辦事處</ref><ref>[http://nrch.cca.gov.tw/ccahome/photo/photo_meta.jsp?xml_id=0006457279&collectionname=%E5%AE%8B%E5%AE%B6%E4%B8%89%E5%A7%8A%E5%A6%B9 泰國國王蒲美蓬陛下設宴答謝總統伉儷]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}, 國家文化資料庫, [[中華民國文化部|中華民國文化部]]</ref><ref>[http://catalog.digitalarchives.tw/item/00/3a/19/55.html 泰國國王暨王后訪問中華民國], [[數位典藏|數位典藏]]與數位學習國家型科技計畫 Taiwan e-Learning & Digital Archives Program</ref>訪問期間至桃園石門水庫和龜山鄉農家參觀水利設施和農村建設，並與農民親切互動，也參觀土地改革館和台中農業改良場，以及台中空軍基地和國軍眷村等地。<ref>[http://www.cna.com.tw/news/aopl/201610130460-1.aspx 泰王蒲美蓬曾說與中華民國感情無法撕開]</ref>

1967年3月27日，[[泰国首相列表|泰國首相]][[他儂·吉滴卡宗|他儂·吉滴卡宗]]抵台訪問，並與蔣中正總統晤談。<ref>[http://catalog.digitalarchives.tw/item/00/31/9f/fb.html 泰國他儂元帥訪華]，典藏台灣</ref>7月4日，外交部長[[魏道明|魏道明]]抵達[[曼谷|曼谷]]出席亞太理事會第二屆部長級會議<ref name="記" />{{rp|739}}。
 
1968年1月4日，副總統[[嚴家淦|嚴家淦]]訪問[[泰國|泰國]]<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|748}}。

1969年5月14日，[[中華民國國防部長|國防部長]][[蔣經國|蔣經國]]以總統特使身份訪問泰國，晉見國王蒲美蓬，並與首相他儂·吉滴卡宗商討[[亞太地區|亞太地區]]七全及雙方合作問題。<ref name="列傳(4)">{{cite book|editor=[[王成斌|王成斌]]等主編|title=《民國高級將領列傳》(4)|location=[[北京|北京]]|publisher=[[解放軍出版社|解放軍出版社]]|year=1998年}}</ref>{{rp|490}}同年，[[沈昌煥|沈昌煥]]任[[中国驻泰国大使列表|駐泰國大使]]，在大使任內，泰國對中國政策立場不定，[[泰國華僑|泰國華僑]]界形勢不安，沈曾多次召集會議、穩定士氣。為鞏固邦交關係，上任即邀蔣經國訪問泰國，竭力呈請政府捐贈[[亞洲理工學院|亞洲理工學院]]研究基金，以爭取成為該學院[[董事|董事]]，台灣每年可選派在職人員及學者前往攻讀[[碩士|碩士]]或[[博士|博士]]。此一方案不僅敦睦了兩國邦誼，也造就了不少台灣經貿科技人才，對日後[[台灣經濟|台灣經濟]]起飛，也發揮了一些作用。後又與[[泰國皇家軍隊|泰國軍方]]合作協助推行[[泰王山地計劃|泰王山地計劃]]，成效卓著。此計劃執行至今仍未停止，對後來中止外交關係情況下，甚多政要訪泰，包括[[李登輝|李登輝]]以總統地位赴泰國，均受到泰王接見禮遇，有所穩定了與泰國的邦交關係。

1971年10月25日，泰國對於[[中國與聯合國#代表權問題|中華民國是否續留聯合國]]的投票採棄權方式，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。

1975年7月1日，泰國與中華民國斷交，同日與[[中華人民共和國|中華人民共和國]]建立正式外交關係。

1994年，前總統[[李登輝|李登輝]]於[[春節|春節]]期間訪問[[菲律賓|菲律賓]]、[[印尼|印尼]]及[[泰國|泰國]]。<ref>[http://www.taiwan.gov.tw/friendship100/story_content.asp?code=A&citem=A05&lang=%20 總統李登輝訪美國康乃爾大學，臺灣民主受推崇]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}, [[行政院新聞局|行政院新聞局]]</ref>

===事件===

2017年7月，前中國國民黨主席[[洪秀柱|洪秀柱]]原訂22日訪問泰國，卻在出訪當天因簽證問題而取消行程。洪秀柱辦公室當天發表聲明指出，[[泰國貿易經濟辦事處|泰國駐台代表處]]的確有所刁難，也批判「[[新南向政策|新南向政策]]」只是欺騙人民的虛幻口號，而政府相關部門日前均以「不評論個人簽證」為理由回應外界詢問。據了解，[[泰國外交部|泰國外交部]]已向台方說明原因，指洪秀柱早在7月初或更早就決定出訪，卻遲至出發前幾天才遞出簽證申請，包括[[中國國民黨|中國國民黨]]批評「新南向政策」等立場，都讓泰國擔心她的到訪只會傳達對泰國「負面、不友善」的訊息，的確多所考慮，但泰國也還沒決定拒發簽證，最終就是趕辦不及；<ref>{{Cite web |url = http://www.storm.mg/article/305048 |author = 石秀娟 |date = 2017年7月26日 |title = 幕後》洪秀柱去不了泰國是新南向政策害的嗎？簽證申請內情首度曝光 |publisher = [[風傳媒|風傳媒]]}}</ref>前中華民國副總統[[呂秀蓮|呂秀蓮]]也在2016、17年兩度被泰國拒發簽證，對此，呂秀蓮說「對我來講，泰國也不是什麼了不起的國家」、「世界上很多國家都歡迎我去，我也不太在乎」。<ref>{{Cite web |url = https://www.nownews.com/news/20170327/2442548 |author = 邱明玉 |date = 2017年3月27日 |title = 兩度簽證遭拒 呂秀蓮：泰國也不是什麼了不起的國家 |publisher = [[NOWnews_今日新聞|NOWnews 今日新聞]]}}</ref>

===批評===
2016年開放對泰國免簽證後，泰國赴台[[賣淫|賣淫]]人數激增，形成治安漏洞，根據[[中華民國警察|中華民國警方]]統計，2015年查獲赴台賣淫者159人中有1個是[[泰國人|泰國籍]]，2016年有301人赴台賣淫，泰國籍增加至15人，2017年到5月已經有296人，泰國籍就佔99人。[[外交部領事事務局|外交部領事事務局]]副局長表示，[[行政院會議|行政院跨部會會議]]已討論此議題，[[內政部移民署|內政部移民署]]將針對特定對象，加強入境審查。<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20171016/1223237/ |author = 陳郁仁 |date = 2017年10月16日 |title = 開放泰國免簽來台後賣淫人數激增 外交部：加強入境審查 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref>

[[蔡英文政府|蔡英文政府]]為推行[[新南向政策|新南向政策]]，給予泰國試辦免簽證，但泰國並未給予對等待遇。2017年8月，泰國貿易經濟辦事處代表畢倫（Piroon Laismit）在接受專訪時首度鬆口「有可能給台灣免簽證」，{{link-en|泰國政府|Politics of Thailand}}正考量各項因素及合適時機宣布。[[駐泰國台北經濟文化辦事處|駐泰國台北經濟文化辦事處]]代表[[童振源|童振源]]也已發出[[漢語|中]][[英語|英]][[泰文|泰文]]版說帖，積極遊說泰國政府。<ref>{{Cite web |url = http://www.chinatimes.com/newspapers/20170831000414-260118 |author = 陳建瑜 |date = 2017年8月31日 |title = 泰「可能」對台灣開放免簽證 |publisher = 中時電子報}}</ref>

2018年8月28日，第一位會說中文的泰國貿易經濟辦事處代表通才（Thongchai Chasawath）召開記者會，說明泰國簽證自10月起團簽委外辦理的「代辦費」增加[[新台幣|新台幣]]470元，並拿出改良後的包裝與過去的紅白[[塑膠袋|塑膠袋]]比較，引喻失義、模糊焦點，被誤以為是「簽證費」要漲價，旅行商業同業公會全國聯合會表示不排除發動[[抵制|抵制]]，拒絕前往泰國旅遊。29日，辦事處收回團簽增收代辦費的決定，自行吸收470元的代辦費。另外，關於對等給予台灣免簽證一事，通才表示「家家有本難念的經」、「泰國對很多國家開放免簽證，但也沒有得到互惠，請體諒我們不是拒絕」、「我們是朋友不是做買賣，朋友體諒朋友，這樣過得去啊。」<ref>{{Cite web |url = https://www.chinatimes.com/realtimenews/20180829001401-260407 |author = 邱文秀 |date = 2018年8月29日 |title = 泰簽漲被揚言抵制 泰代表：我們是朋友不是做買賣 請體諒 |publisher = 中時電子報}}</ref><ref>{{Cite web |url = https://www.cw.com.tw/article/article.action?id=5091897 |author = 賀桂芬 |date = 2018年8月29日 |title = 泰簽風波 泰代表會中文反砸鍋 |publisher = [[天下雜誌|天下雜誌]]}}</ref>

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於[[曼谷|曼谷]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#ASIA |title = 外貿協會全球據點|publisher = 中華民國對外貿易發展協會}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]也設立'''駐泰國代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=214&pid=305862 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2018年，泰國是中華民國第12大貿易夥伴、第13大[[進口|進口]]夥伴、第11大[[出口贸易|出口]]夥伴。出口至泰國的金額為61億6,841萬0,755[[美元|美元]]，年減3.346%；自泰國進口的金額為45億8,187萬6,181美元，年增5.125%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）15億8,653萬4,574美元，年減21.593%。2017年，泰國是中華民國第12大貿易夥伴、第11大進口夥伴、第11大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資===
1990年代，[[臺灣商人|台商]]響應中華民國政府的[[南向政策|南向政策]]，大舉來泰國投資設廠，在[[亞洲金融風暴|亞洲金融風暴]]之前達到高峰，近年來因泰國政局不穩、勞工短缺、[[2011年泰國水災|2011年泰國水災]]、[[最低工資|最低工資]]調高，以及[[中華人民共和國|中華人民共和國]]與[[越南|越南]]市場的快速崛起等因素，根據泰國統計資料，台商在泰國投資雖有減緩現象，但累積投資金額仍高居[[外商直接投資|外人投資]]第3位。台商在泰國投資活躍，且泰國也是進入[[東南亞|東南亞]]市場的重要伙伴。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/th/pdf/103-15Thailand.pdf |title = 投資環境簡介 |date = 2014年12月 |publisher = 經濟部投資業務處}}</ref><ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 投資環境簡介 |publisher = 中華民國對外貿易發展協會}}</ref>

根據{{link-en|泰國投資促進委員會|Thailand Board of Investment}}的資料顯示，2017年台商在泰國投資件數有49件，投資金額為2.67億美元。歷年累計投資件數有2,369件，累計投資金額為144億8,000萬美元。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>但許多台商公司或工廠是以泰籍夥伴名義登記，或認股低於50%，而未能以台商身份列入統計，故實際家數應遠超於此。另根據泰國15個台商聯誼會調查結果，其轄下登記的台商會員總數已超過3,000家，且仍有相當數量的台商未加入台商聯誼會，保守估計泰國台商家數至少在5,000家以上。<ref name="投資環境簡介"/>

台商在泰國的投資，除了[[消費電子產品|電子]]、[[橡膠|橡膠]]、[[鋼|鋼鐵]]及[[石油化學|石化]]等投資金額較大的產業外，其他大部分均為傳統[[中小企業|中小企業]]的[[製造業|製造業]]，投資項目包含甚廣，從魚蝦[[水產業|養殖]]、[[紡織|紡織]]、[[機械|機械]]、[[進口|進]][[出口贸易|出口]]、[[珠寶|珠寶]]、[[农作物|農產品]]加工、運動器材、[[家具|家具]]、到[[陶瓷材料|陶瓷]]、[[建材|建材]]、[[人力仲介|人力仲介]]、[[房地產|房地產]]開發、[[證券|證券]]、[[保全|保全]]、[[珍珠奶茶|珍珠奶茶]]、[[旅行社|旅行社]]等都有。<ref name="投資環境簡介"/>

===合作===
國際合作發展基金會（國合會）與{{link-en|泰國皇家計畫基金會|Royal Project Foundation}}的[[農業技術|農業技術]]合作自1970年至今。<ref name="105年外交年鑑"/>

2014－2016年，國合會協助泰國皇家計畫基金會執行「[[柑橘|柑橘]]及[[百香果|百香果]]植病防治計畫」和「澀[[柿|柿]]及[[菇類|菇類]]生產計畫」，提升其植病防治及相關生產技術。<ref>{{Cite web |url = http://www.icdf.org.tw/ct.asp?xItem=21279&ctNode=29917&mp=1  |date = 2014年5月2日 |title = 各國合作計畫 |publisher = 國際合作發展基金會}}</ref><ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/7fb2e392-9e9e-4bf7-a52e-68579f6bfca4.pdf |author = 國際合作及經濟事務司 |date = 2015年4月14日 |title = 國際合作發展事務103年度報告（參、103年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref>

===臺商組織===
泰國的台商組織是依照不同地區成立，目前已成立的有[[北柳府|北柳]]、[[北欖府|北欖]]、拉加邦、[[春武里府|春武里]]、[[吞武里|吞武里]]、[[曼谷|曼谷]]、萬磅、是隆、亞速、北區、[[呵叻府|呵叻]]、泰南、泰北、[[羅勇府|羅勇]]、[[普吉府|普吉]]等15個台商聯誼會，這些台商聯誼會各有組織並各自推選會長。為了統合台商力量，這15個台商聯誼會共同成立了「泰國臺灣商會聯合總會」，並為該會設立固定辦公會所，聘僱專職人員辦理各項會務，並經常辦理研討會、商展及聯誼等活動。<ref name="投資環境簡介"/>

===臺資銀行===
目前設立的有[[兆豐國際商業銀行|兆豐國際商業銀行]]（子行，另有3家分行），而[[國泰世華商業銀行|國泰世華商業銀行]]、[[中國信託商業銀行|中國信託商業銀行]]、[[第一商業銀行|第一商業銀行]]、[[上海商業儲蓄銀行|上海商業儲蓄銀行]]等則設有辦事處。<ref name="投資環境簡介"/>

===犯罪事件===

1994年1月，[[泰国皇家警察|泰國警察]]為謀財殺害[[中華民國籍|中華民國籍]]（以下簡稱台灣籍）旅客，中華民國政府除採取各種措施與行動，[[中華民國外交部|外交部]]於4月召見[[泰國貿易經濟辦事處|泰國駐台代表]]，重申對此案的嚴重關切，並促請就所提各項要求給予應有交待。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=32508FD33FE2470A |author = 公眾外交協調會 |date = 1994年3月18日 |title = 泰國警察為謀財殺害我國籍旅客案自元月底遭揭以來 |publisher = 中華民國外交部}}</ref>

2017年2月，中華民國[[刑事警察局|刑事警察局]]與[[台中市警察局|台中市警察局]]接獲情資共組專案小組追查跨國[[詐騙集團|詐騙集團]]，先查獲境內的話務機房，再與泰國警方聯手破獲設在泰國曼谷的詐騙機房，共逮捕台灣籍與[[中華人民共和國國籍法|中華人民共和國籍]]（以下簡稱中國大陸籍）犯嫌10人。<ref>{{Cite web |url = https://udn.com/news/story/2/2313561 |author = 游振昇 |date = 2017年3月1日 |title = 詐騙集團不到一年騙上千萬 台灣泰國逮10人 |publisher = 聯合新聞網}}</ref>

2017年3月29日，2年前在[[印尼|印尼]]涉及電信詐騙案的3名台灣籍男子，在押解回台由[[桃園地方法院|桃園地方法院]]判刑定讞發監執行前逃至泰國，經[[桃園地方法院檢察署|桃園地方法院檢察署]]發布[[通緝|通緝]]，在台灣與泰國警方合作逮捕後押解回台。<ref>{{Cite web |url = http://www.epochtimes.com/b5/17/3/29/n8978666.htm |author = 鍾元 |date = 2017年3月29日 |title = 3詐騙犯逃至泰國 台泰合作逮捕押解回台 |publisher = 大紀元時報}}</ref>

2017年7月6日，刑事警察局與泰國警方聯手破獲以台灣籍男子為首的詐騙集團，共20名嫌犯被逮捕，除2名中國大陸籍外，其餘的18名台灣籍嫌犯在26日由泰國遣返台灣，移送刑事警察局接受偵訊。<ref>{{Cite web |url = http://www.storm.mg/article/304904 |author = 施孝衡 |date = 2017年7月26日 |title = 專騙大陸人 18名台籍詐騙犯由泰國遣返回台 |publisher = 風傳媒}}</ref>

2017年7月19日，泰國警方與[[中华人民共和国人民警察|中華人民共和國警方]]（以下簡稱對岸、中國大陸）聯手破獲在泰國的兩岸電信詐騙集團，共逮捕44名嫌犯，25名為台灣籍，19名是中國大陸籍，對岸已要求泰國將全數嫌犯遣送至中國大陸受審。泰國移民總局長納塔通（Nathathorn Prousoontorn）在[[記者會|記者會]]上表示，嫌犯先在泰國定罪，待案件結束後再遣送至對岸。這將是台灣與泰國於2013年簽訂《共同打擊跨國經濟及相關犯罪協議》以來，首度未將台灣籍嫌犯遣返台灣。<ref>{{Cite web |url = http://www.ettoday.net/news/20170723/973026.htm?t=%E6%B3%B0%E5%9C%8B%E7%A7%BB%E6%B0%91%E7%B8%BD%E5%B1%80%E5%AE%A3%E5%B8%83%E3%80%8025%E5%90%8D%E5%8F%B0%E7%81%A3%E8%A9%90%E9%A8%99%E5%AB%8C%E3%80%8C%E5%85%A8%E9%83%A8%E9%81%A3%E9%80%81%E4%B8%AD%E5%9C%8B%E3%80%8D |author = 綜合報導 |date = 2017年7月23日 |title = 泰國移民總局宣布 25名台灣詐騙嫌「全部遣送中國 |publisher = [[ETNEWS新聞雲|ETNEWS新聞雲]]}}</ref><ref>{{Cite web |url = http://www.chinatimes.com/newspapers/20170724000316-260108 |author = 連雋偉、蕭博文、陳建瑜 |date = 2017年7月24日 |title = 25台籍詐騙犯 泰將遣陸 |publisher = 中時電子報}}</ref>

===駕車===
中華民國與泰國同為1949年《[[维也纳道路交通会议|日內瓦道路交通公約]]》締約國，泰國承認締約國依據1949年公約版本所核發的[[國際駕駛執照|國際駕駛執照]]，因此持有[[中華民國交通部|中華民國交通部]]核發的國際駕駛執照可以合法在泰國駕車，期限1年；長期在泰國居留者，可持[[中華民國汽車駕駛執照|中華民國汽車駕駛執照]]申辦{{link-en|泰國駕駛執照|Driving licence in Thailand}}。首次申辦可獲核發1年期駕照，期滿後可申辦5年期駕照。<ref name="生活資訊"/>
