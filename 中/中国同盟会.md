'''中国同盟会'''（简称'''同盟会'''），亦稱為'''中國革命同盟會'''，是[[清朝|清朝]]末年由[[兴中会|兴中会]]、[[华兴会|华兴会]]、[[光复会|光复会]]等多个團體集合而成的[[帮会|革命組織]]，1905年成立於日本[[東京|東京]]。其組織網絡遍布中國與世界各地的[[華人社區|華人社區]]。其發動多次推翻清朝的[[起義|起義]]活動，包括成功推翻清朝建立共和政體的[[辛亥革命|辛亥革命]]。

同盟会主要由[[孙中山|孙中山]]、[[宋教仁|宋教仁]]、[[黄兴|黄兴]]、[[黄元秀|黄元秀]]、[[蔡元培|蔡元培]]等人领导，由於其成员其包羅之眾，自1906年以後就不斷發生內爭，包括個人的衝突、理論的不同與革命方略的歧見，特別是由於一連串武裝起义的失敗，更增加了內爭的激烈性。辛亥革命後，同盟会改組為[[國民黨_(1912–1913)|國民黨]]，成為之後[[中華革命黨|中華革命黨]]與[[中國國民黨|中國國民黨]]之前身。<ref>{{cite web|url=http://ap6.pccu.edu.tw/encyclopedia/data.asp?id=342&htm=02-058-0678中國同盟會.htm|title=中國同盟會|publisher=|date= |language =zh-hant }}，《中華百科全書》典藏版</ref>

== 歷史 ==
===成立===
[[File:同盟會成立.jpg|thumb]]
[[File:中國同盟會證書.jpg|thumb]]
[[File:李鐵夫孫中山12345.jpg|缩略图]] 畫布油畫 93×71.7cm|右]]
1905年7月，在[[黑龍會|黑龍會]]領袖[[內田良平|內田良平]]的牽線下，孫中山返回[[日本|日本]][[東京|東京]]。孫中山在東京倡導籌備成立中國同盟會。

1905年8月20日，在[[東京市|東京市]][[赤坂區|赤坂區]][[頭山滿|頭山滿]]提供的民宅二樓[[榻榻米|榻榻米]]房，中國革命同盟會成立（後為避免[[日本政府|日本政府]]反對，改名為中國同盟會），孫中山被推舉為[[總理|總理]]，[[黃興|黃興]]等任庶務；除制定了《軍政府宣言》、《中國同盟會總章》和《革命方略》等文件，並決定在國內外建立支部和分會，聯絡[[華僑|華僑]]、[[會黨|會黨]]和[[新軍|新軍]]，成為全國性的革命組織。

===方针政策===
同盟会的前身是[[湖南|湖南]][[华兴会|华兴会]]（[[黄兴|黄兴]]、[[宋教仁|宋教仁]]、[[陈天华|陈天华]]等）和[[广东|广东]][[兴中会|兴中会]]（孙中山、[[胡汉民|胡汉民]]、[[汪精卫|汪精卫]]等），除此之外还有[[江苏|江]][[浙江|浙]][[光复会|光复会]]（[[章太炎|章太炎]]、[[陶成章|陶成章]]、[[蔡元培|蔡元培]]、[[秋瑾|秋瑾]]、[[徐錫麟|徐錫麟]]等）、[[科学补习所|科学补习所]]、[[青幫|青幫]]、[[洪門|洪門]]等多个组织参與加入。

同盟会确认其政纲是孙中山提出的：“驱除鞑虏，恢复中华，创立民国，平均地权”十六字纲领；发行《[[民报|民报]]》作为机关刊物（原名《二十世纪之支那》，为[[华兴会|华兴会]]机关刊物，同盟会成立后在1905年易名《民报》）。

中国同盟会与孙中山设想的一个[[中华民国|中华民国]]的政府组织一致：在[[总理|总理]]下设行政部、立法部和司法部，这实际上是[[三权分立|三权分立]]的原则。《民报》在[[章炳麟|章炳麟]]、[[陶成章|陶成章]]等主编下，由[[胡汉民|胡汉民]]、[[汪精卫|汪精卫]]等执笔，与主张保皇、由[[康有为|康有为]]、[[梁启超|梁启超]]执笔的《[[新民丛报|新民丛报]]》展开激烈论战，成为革命思想的重要阵地。

===分裂===
同盟会在成立後不到二年即分裂，1906年，日本政府決定取諦中國留學生參加的同盟會，遂引起同盟會的留学生歸國或繼續在日本發展的爭論，雙方激辯不已，其中一名成員[[陳天華|陳天華]]甚至以自殺明志，最後許多成員回到中國，但有的成員則選擇繼續留在日本，造成了第一次的分裂。

隔年，黃興與孫中山因為革命旗帜问题而产生纠葛。宋教仁則對同盟會中許多事感到心寒，萌生退意，向孫中山請辭庶務幹事長一職。宋教仁在日記裡寫到他對孫中山與同盟會的不滿，並且有另謀發展之心。

同盟會的第三次重大分裂<ref>{{Cite web|url = http://sunology.culture.tw/cgi-bin/gs32/s1gsweb.cgi?o=dcorpus&s=id=%22TG0000000968%22.&searchmode=basic|title = 復張繼論辭同盟會總理職事函|accessdate = 1909-11-12|author = |date = 1909-11-12|publisher = 台灣，國父全集，第四冊，第99-100頁.}}</ref>，是因光復會的'''[[陶成章|陶成章]]、'''[[章太炎|章太炎]]等人指控孫中山挪用會款的贪腐事件、私下接受日本政府政治獻金6,000元與日本商人{{link-ja|鈴木久五郎|鈴木久五郎}}政治獻金10,000元。孫文收款後只留2,000元作《民報》經費，陶成章懷疑孫中山暗中搗鬼，遂指稱河口起義所用軍費不過一千多元，孫中山將各地同志的捐款攫為己有，家中發了大財。接着，陶成章糾集五六人在新加坡《南洋總匯新報》發表《同盟會七省意見書》，即所謂的《孫文罪狀》。指責孫中山「謊騙營私」，在滙豐銀行有巨額存款，貪污兩萬革命經費，有「殘賊同志」、 「蒙蔽同志」、「敗壞全體名譽」等三種十二項「罪狀」，提出九條善後辦法，要求「開除孫文總理之名，發表罪狀，遍告海內外」，但是孫中山與其支持者則堅決反對，雙方僵持不下，互鬥激烈，章太炎與光復會等人选择退出同盟会，孙中山与汪精卫、胡汉民等于[[南洋|南洋]]另组总部，在1907年同盟會幾乎瓦解。所以1907年後的起義究竟要算在哪個組織上，至今仍是爭論不休。

1907年，日本政府受清廷壓力，以一萬五千日元請孫中山離開日本。總和臨行前日本政府和財團向他贈款2.3萬日元。當時日幣200元日幣相當於現在一千萬。<ref>{{Cite web|url = https://books.google.com.tw/books?id=nS07BAAAQBAJ&lpg=PT270&ots=W4nSnfAgXn&dq=%E5%AD%AB%E6%96%87%20%E9%85%92&hl=zh-TW&pg=PT273#v=onepage&q=%E5%AD%AB%E6%96%87%20%E9%85%92&f=false|title = 日本近現代史|accessdate = 2013-08-01|author = 田原総一朗|date = 2013-08-01|publisher = }}</ref>

相當於現今11億5千萬日幣與2億8千萬新臺幣。

1912年1月[[陳其美|陳其美]]命令部下[[蔣介石|蔣介石]]對[[陶成章|陶成章]]執行暗殺。蔣介石對刺殺陶成章一事欣然受命，並將刺陶之事載入自己日記。<ref>{{Cite book|title = 《蔣主席》第八章《槍殺陶成章》|last = |first = |publisher = 鄧文儀|year = 1945|isbn = |location = 上海|pages = }}</ref>

===辛亥革命後===
武昌起义爆发了全国规模的[[辛亥革命|辛亥革命]]，同盟会本部由日本东京迁至[[上海|上海]]；[[中華民國臨時政府_(南京)|南京临时政府]]成立后，再迁南京。
{{cn|有一些人并不赞同孙中山的[[三民主义|三民主义]]，对孙提出的平均土地更有人反对。|time=2017-09-04}}1912年8月7日，在[[宋教仁|宋教仁]]的组织下，同盟会、[[统一共和党|统一共和党]]、[[国民公党|国民公党]]、[[国民共进会|国民共进会]]和[[共和实进会|共和实进会]]联合在[[北京|北京]]成立[[國民黨_(1912年-1913年)|国民党]]，孙中山为理事长、[[宋教仁|宋教仁]]为代理事长。而[[章炳麟|章炳麟]]等人則与[[黎元洪|黎元洪]]组建[[共和党_(中国)|共和党]]。

===入会仪式===
入會人手寫盟書並舉手宣誓：“聯盟人某省某府某縣人某某某，當天發誓：驅除韃虜，恢復中華，創立民國，平均地權，矢信矢忠，有始有卒。如或渝此，任眾處罰，天運乙巳年七月某日，中國同盟會會員某某某。”之後，由孫文帶入一小室，秘密傳授暗號：“問何處人，答為漢人。問何物，答為中國物。問何事，答為天下事。”禮畢，孫中山祝賀道：“自今日起，君等已非清朝人矣。”
