'''中国人民解放军火箭军综合训练基地'''，位于[[河北省|河北省]][[张家口市|张家口市]][[宣化区|宣化区]]，是[[中国人民解放军火箭军|中国人民解放军火箭军]]专设的后勤训练机构<ref>{{Cite web |url = http://www.zjkxuanhua.gov.cn/article/20130606/zjkxhqjyj-2013-01826.html |title = 宣化一中隆重举办2013年高三学生毕业庆典 |author =  |date = 2013-06-06 |publisher =张家口市宣化区人民政府  |language =  |accessdate =  |quote =  }}</ref>。

==简介==
该基地原名'''[[中国人民解放军第二炮兵部队|中国人民解放军第二炮兵部队]]综合训练基地'''，是第二炮兵唯一的后勤专业技术骨干教学训练单位，承担财务、军需、物资、油料、营房、后勤装备、军交运输等9个门类14个不同专业的士官教学培训任务。2013年，基地有高级工程师7名。截至2013年，基地有4名教员入选第二炮兵人才库，1名士官教练员获全军优秀士官，有3项科研成果获军队科技进步奖<ref name=lz>[http://roll.sohu.com/20130226/n367107841.shtml 塞外劲吹科研风 第二炮兵综合训练基地以科研促发展纪实，搜狐，2013-02-26]</ref>。

2013年，该基地与[[中国人民解放军防化学院|中国人民解放军防化学院]]展开联合办学。2013年首批接受联合办学培训的79名防化专业学兵，90%以上提前了一个周期获得专业技术合格证书<ref name=zxw>[http://www.chinanews.com/mil/2014/05-20/6190137.shtml 二炮某综合训练基地携手院校建立联合办学机制，中国新闻网，2014-05-20]</ref>。

2014年2月20日，该基地后勤部供应科正营职助理员王中昌罹患肺癌病逝。王中昌生前对每项经费支出都精打细算，被誉为“红管家”，第二炮兵后勤部党委为王中昌记二等功，并作出了向王中昌学习的决定<ref>{{Cite web |url=http://www.chinadaily.com.cn/hqgj/jryw/2014-02-28/content_11305478.html |title=二炮某训练基地助理员精打细算 被誉“红管家”，中国日报网，2014-02-28 |access-date=2016-10-21 |archive-url=https://web.archive.org/web/20161022025155/http://www.chinadaily.com.cn/hqgj/jryw/2014-02-28/content_11305478.html |archive-date=2016-10-22 |dead-url=yes }}</ref><ref>[http://news.xinhuanet.com/2013-11/29/c_118355729.htm 一身正气的“红管家”——记第二炮兵某综合训练基地后勤部助理员王中昌，新华网，2013-11-29]</ref><ref name=ddq>[http://epaper.gmw.cn/gmrb/html/2013-12/22/nw.D110000gmrb_20131222_1-03.htm?div=-1 “服务没有等待区”，光明日报2013年12月22日，第03版]</ref>。

2015年，来自各地职业技术院校的118名士兵作为第二炮兵首批定向培养直招士官，在该综合训练基地经过3个多月培训后，走上第二炮兵气象、测绘、通信等保障专业岗位<ref>[http://news.ifeng.com/a/20150410/43526078_0.shtml 118名直招士官走进二炮 来自各地职业技校，凤凰网，2015-04-10]</ref>此后，该综合训练基地继续从各地职业技术院校招收定向培养直招士官。<ref>[http://news.163.com/15/0617/22/ASBKRS3D00014AEE.html 第二炮兵定向培养直招士官军地联席会在长沙举行，网易，2015-06-17]</ref>。

2015年12月，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第二炮兵升格为[[中国人民解放军火箭军|中国人民解放军火箭军]]，该基地改称'''中国人民解放军火箭军综合训练基地'''<ref>[http://www.81.cn/depb/2016-08/11/content_7202265.htm 课堂即战场：火箭军某综合训练基地创新培养测绘人才，中国军网，2016-08-11]</ref><ref>{{Cite web |url = http://www.dky.bjedu.cn/xxxw_1956/201703/t20170331_44262.html |title = 我校组织开展首批定向培养士官生入伍实习调研 |author =  |date =2017-03-31  |publisher =北京电子科技职业学院  |language =  |accessdate =  |quote =  }}</ref>。
