'''中奧關係'''，是指[[中國歷史|歷史上的中國]]與[[奧地利歷史|奧地利]]，以至現在[[中華人民共和國|中華人民共和國]]與[[奧地利共和國|奧地利共和國]]之間的雙邊關係。

== 歷史 ==
=== 中華人民共和國與奧地利建交前 ===
奧地利與中國的首次貿易交往始於1781年。於1869年，[[奧匈帝國|奧匈帝國]]和[[清朝|清朝]]締結條約，建立外交關係，並於1871年正式執行<ref name="bbouc">{{Cite web|url=http://www.bmeia.gv.at/cn/botschaft/peking/bilaterale-beziehungen/bilaterale-beziehungen-oesterreich-und-china.html|title=奥地利中国关系|accessdate=2015-03-11|work=|publisher=奧地利駐華大使館|date=}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1881年，清朝開始向奧匈帝國派駐欽差大臣，首任為[[李凤苞|李凤苞]]，於1881年4月5日獲任命。起初，駐奧匈帝國欽差大臣一職為兼職，至1902年起改為專使<ref>{{cite book|title=《清季中外使领年表》|author=中国第一历史档案馆|publisher=[[中华书局|中华书局]]}}</ref>。而奧匈帝國則在1896年起於[[北京|北京]]設立公使館，公使館原址現為[[中国国际问题研究院|中国国际问题研究院]]的所在地<ref name="bbouc" />。1900年[[八國聯軍之役|八國聯軍之役]]期間，奧匈帝國為參戰國之一。

奧匈帝國於1913年10月6日承認[[中華民國|中華民國]]，中華民國當時派出之唯一一任駐奧匈帝國之公使名為[[沈瑞麟|沈瑞麟]]<ref name="ml">{{cite web |url = http://202.196.33.227/CRFDHTML/r200607067/r200607067.1107476e.html |title = 十六　中國駐國外使節名録 (1911——1949) |deadurl = yes |archiveurl = https://www.webcitation.org/67EZGedK5?url=http://202.196.33.227/CRFDHTML/r200607067/r200607067.1107476e.html |archivedate = 2012-04-27 }}</ref>。1917年，因中華民國在第一次世界大戰中加入[[協約國|協約國]]，向身為[[同盟國_(第一次世界大戰)|同盟國]]的奧匈帝國宣戰，中奧斷交，直至[[奧地利第一共和國|奧地利第一共和國]]於1920年建國後兩國方重新建立外交關係<ref name="ml" />。

1938年，[[納粹德國|納粹德國]]吞併奧地利，中華民國與奧地利再度斷交。第二次世界大戰後，中華民國和奧地利曾短暫互設公使館。1949年中華人民共和國建國後，奧地利駐中華民國公使館閉館<ref name="bbouc" />。翌年，中華民國駐奧地利公使館亦閉館<ref name="ml" />。但中華民國與奧地利之外交關係在此時並沒有中斷，直至1971年5月28日中華人民共和國和奧地利建交為止。

1964年，中華人民共和國和奧地利簽訂換文，互設商务代表处<ref name="adl" />。

=== 中華人民共和國與奧地利建交後 ===
1971年5月28日，中華人民共和國和奧地利共和國建立大使級外交關係。兩國在同年九月起開始互派大使。<ref name="adl">{{Cite web |url = http://www.chinaembassy.at/chn/adl/t1029790.htm |title=中奥双边关系 |accessdate=2015-03-11 |work= |publisher=中華人民共和國駐奧地利共和國大使館 |date=2013-04-10 }}</ref>

自中華人民共和國和奧地利建交以來，两国关系呈现良好发展势头，各领域合作发展加快<ref name="adl" />。1985年9月，時任[[奥地利总统|奥地利总统]][[鲁道夫·基希施莱格|鲁道夫·基希施莱格]]成為首位訪華的奧地利國家元首<ref name="adl" />。在1993年4月，時任[[奧地利總理|奧地利總理]][[弗朗茨·弗拉尼茨基|弗朗茨·弗拉尼茨基]]成為首位訪華的奧地利政府首腦<ref name="adl" />。中國方面，1994年6月，時任[[中華人民共和國國務院總理|中華人民共和國國務院總理]]的[[李鵬|李鵬]]訪問奧地利，成為首位訪奧的中國[[政府首腦|政府首腦]]<ref name="adl" />。1999年3月，時任[[中華人民共和國主席|中華人民共和國主席]]的[[江澤民|江澤民]]對奧地利進行[[國事訪問|國事訪問]]，成為首位訪問奧地利的[[中華人民共和國國家元首|中國國家元首]]<ref name="adl" />。

2007年9月20日，時任奧地利總理[[阿爾弗雷德·古森鮑爾|阿爾弗雷德·古森鮑爾]]會見[[第十四世達賴喇嘛|達賴喇嘛]]<ref>{{Cite web|url=http://www.topiso.net/tibet.org.tw/news_detail.php?news_id=206|title=達賴喇嘛訪問奧地利 與總理舉行會晤|accessdate=2015-03-11|work=|publisher=財團法人達賴喇嘛西藏宗教基金會|date=}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，以致中奧關係一度緊張。在2008年5月2日，古森鮑爾總理會見時任中國駐奧地利大使[[吴恳|吳懇]]時表示，奧地利重視發展對華關係，坚持一个中国原则，承认台湾和西藏都是中国领土的一部分<ref>{{Cite web|url = http://www.fmprc.gov.cn/mfa_chn/gjhdq_603914/gj_603916/oz_606480/1206_606578/fyrygth_606586/t451023.shtml|title = 外交部发言人秦刚就中奥关系答记者问|accessdate = 2015-03-11}}</ref>，兩國關係才回復正常。但其後[[維爾納·法伊曼|維爾納·法伊曼]]總理亦在2012年5月26日會見達賴喇嘛<ref>{{Cite web |url = http://www.vot.org/%E5%A5%A7%E5%9C%B0%E5%88%A9%E7%B8%BD%E7%90%86%E6%9C%83%E6%99%A4%E9%81%94%E8%B3%B4%E5%96%87%E5%98%9B-%E7%B6%AD%E4%B9%9F%E7%B4%8D%E8%88%89%E8%A1%8C%E6%AD%90%E6%B4%B2%E8%81%B2%E6%8F%B4%E8%A5%BF%E8%97%8F/ |title=奧地利總理會晤達賴喇嘛 維也納舉行歐洲聲援西藏大型集會 |accessdate=2015-03-11 |work= |publisher= 西藏之聲 |date= }}</ref>，再次令中奧關係呈負面影響<ref name="adl" />。

1972年10月，中華人民共和國和奧地利共和國簽訂《贸易和支付协定》，將兩國之民間貿易關係提升至政府间贸易关系<ref name="adl" />。

於2013年，中奧進出口總額為70.6849億美元，其中中國出口奧地利為20.3818億美元，奧地利出口中國為50.3031億美元<ref>{{Cite web |url = http://at.mofcom.gov.cn/article/zxhz/tjsj/201406/20140600619590.shtml |title=近十年中奥双边贸易统计 |accessdate=2015-03-11 |work= |publisher=中華人民共和國駐奧地利共和國大使館經濟商務參贊處 |date=2014-06-10 }}</ref>。

==文化關係==
自1998年起，中國[[中央民族樂團|中央民族樂團]]每年都在维也纳[[金色大厅|金色大厅]]举办“新春音乐会”，[[中国交响乐团|中国交响乐团]]亦曾經應邀赴奧演出。而{{link-en|维也纳施特劳斯乐团|Wiener Johann Strauss Orchester|维也纳施特劳斯乐团}}、[[維也納少年合唱團|維也納少年合唱團]]、[[維也納爱樂樂團|維也納爱樂樂團]]和[[维也纳国家歌剧院|维也纳国家歌剧院]]等亦曾經访华<ref name="adl" />。1999年，[[中國中央電視台|中國中央電視台]]及[[奧地利廣播集團|奧地利廣播集團]]在北京[[故宫博物院|故宫博物院]][[午门|午门]]联合举办音乐家舞台文艺晚会，中國、奧地利、德國和瑞士近千名藝術家參加演出，由四國電視台實況轉播<ref name="adl" />。

2001年11月，中奧兩國簽訂文化合作協議<ref name="adl" />。

奧地利曾兩度將2006年和2011年定為中國年<ref name="adl" />。

中國在奧地利設立了兩間[[孔子學院|孔子學院]]，分別位處[[维也纳大学|维也纳大学]]及[[格拉茨大学|格拉茨大学]]<ref name="adl" />。
