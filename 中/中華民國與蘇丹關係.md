'''中華民國與蘇丹關係'''是指[[中華民國|中華民國]]與[[蘇丹共和國|蘇丹共和國]]（通稱'''蘇丹'''、'''北蘇丹'''）之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對蘇丹的相關事務由'''{{link-en|駐沙烏地阿拉伯王國台北經濟文化代表處|Taipei Economic and Cultural Representative Office in the Kingdom of Saudi Arabia}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-194-3f925-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|蘇丹簽證政策|蘇丹公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往蘇丹駐[[馬來西亞|馬來西亞]][[外交代表機構#大使馆|大使館]]、駐[[泰國|泰國]][[領事館|領事館]]申請簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-194-3f925-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持[[蘇丹護照|蘇丹護照]]的{{link-en|蘇丹人口|Demographics of Sudan|蘇丹公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將蘇丹歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2015年11月11日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-418-a939f-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

== 經濟 ==
{{see also|臺灣經濟|蘇丹經濟}}
=== 貿易 ===
2017年，蘇丹是中華民國第129大貿易夥伴、第162大[[進口|進口]]夥伴、第109大[[出口贸易|出口]]夥伴。出口至蘇丹的金額為1,874萬1,376[[美元|美元]]，年減44.208%；自蘇丹進口的金額為64萬6,303美元，年增31.532%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1,809萬5,073美元，年減45.333%。※2016年，蘇丹是中華民國第109大貿易夥伴、第163大進口夥伴、第92大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
