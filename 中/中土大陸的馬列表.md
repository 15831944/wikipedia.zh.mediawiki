== 比翁的小馬 ==
《[[哈比人歷險記|哈比人歷險記]]》中，當[[比爾博·巴金斯|比爾博·巴金斯]]和十三名[[矮人_(中土大陸)|矮人]]離開換皮人[[比翁|比翁]]的居所時，比翁送了一批小馬可使他們快速抵達[[幽暗密林|幽暗密林]]，但他們得在進入森林前將小馬送還，因為比翁非常熱愛他的動物。起初矮人們並不想遵守約定，但[[甘道夫|甘道夫]]說服了他們，不要試圖跟比翁對抗<ref>{{Cite book|title=哈比人|last=[[托爾金|托爾金]]|publisher=聯經|year=2014年1月初版第四刷|isbn=978-957-08-4103-9|location=臺北}}</ref>。

== 鄉巴佬 ==
鄉巴佬是[[湯姆·龐巴迪|湯姆·龐巴迪]]的馬，身材較胖，十分聰明機智。平時龐巴迪很少騎牠，而讓牠在山野間自由行動<ref name=":0">{{Cite book|title=魔戒首部曲：魔戒現身|last=托爾金|publisher=聯經|year=2012年12月|isbn=978-957-08-4100-8|location=臺北}}</ref>。

== 梅里的小馬 ==
[[梅里雅達克·烈酒鹿|梅里]]擁有的五匹小馬隨著哈比人們自[[夏爾_(中土大陸)|夏爾]]一路來到[[布理|布理]]，但牠們在馬廄中被[[戒靈|戒靈]]給嚇走；隨後牠們一起行動，去找之前認識的鄉巴佬，並被湯姆·龐巴迪照顧了一段時間。之後龐巴迪將牠們送回到[[躍馬旅店|躍馬旅店]]老闆[[巴力曼·奶油伯|奶油伯]]那邊去，讓牠們在旅店工作<ref name=":0" />。

== 阿斯法洛斯 ==
當佛羅多在[[風雲頂|風雲頂]]被[[安格馬巫王|巫王]]刺傷後，徒步行走的哈比人們很難及時趕到[[布魯南渡口|布魯南渡口]]，而戒靈仍不斷地追尋哈比人們。但所幸從[[瑞文戴爾|瑞文戴爾]]趕來的[[精靈_(中土大陸)|精靈]][[葛羅芬戴爾|葛羅芬戴爾]]（在電影中則是[[亞玟|亞玟]]）找到了哈比人們，他讓受傷的佛羅多騎乘精靈駿馬阿斯法洛斯（Asfaloth）。在之後值遇到戒靈時，阿斯法洛斯載著佛羅多飛速前往布魯南渡口，其速度連戒靈的坐騎都追趕不上<ref name=":0" />。

== 比爾 ==
當佛羅多等四個哈比人在布理躍馬旅店住宿期間，戒靈入侵並嚇走了馬廄裡所有的馬，包括哈比人們的五匹小馬在內。次日早晨，哈比人們好不容易才找到一匹出售的小馬，其主人[[比爾·羊齒蕨|比爾·羊齒蕨]]（後來才知道他是[[薩魯曼|薩魯曼]]手下的間諜）以高價將將這匹小馬比爾賣出。哈比人發現比爾受到嚴重的虐待，但牠在[[山姆·詹吉|山姆·詹吉]]細心的照料下逐漸得到恢復，一路隨著隊伍抵達瑞文戴爾。山姆對牠有著濃厚的情感。

後來小馬比爾成為了幫[[魔戒遠征隊|魔戒遠征隊]]背負行李的馬，在進入摩瑞亞前，小馬比爾因受到[[水中監視者|水中監視者]]的驚嚇而離去<ref name=":0" />。但聰明的牠最後找到回家的路，而被奶油伯收養；魔戒聖戰結束後，小馬比爾與山姆重聚，並被其帶到夏爾<ref name=":1">{{Cite book|title=魔戒三部曲：王者再臨|last=托爾金|publisher=聯經|year=2012年12月|isbn=978-957-08-4102-2|location=臺北}}</ref><ref>{{Cite news|url=https://www.tor.com/2017/09/22/middle-earth-cage-match-bill-the-pony-vs-shadowfax/|title=Middle-earth Cage Match: Bill the Pony vs. Shadowfax|date=2017-09-22|newspaper=Tor.com|accessdate=2017-12-27|language=en-US}}</ref>。

== 布里哥 ==
{{see also|魔戒電影三部曲原創人物列表#布里哥}}布里哥是一匹棕色的馬，只出現在[[彼得·傑克森|彼得·傑克森]]的電影改編中<ref>{{Cite web|url=http://www.brego.net/lotr/brego.php|title=Brego in Lord of the Rings: Brego the Builder and Brego the Horse|accessdate=2017-12-27|author=|publisher=}}</ref>。布里哥的主人[[希優德|希優德]]在[[艾辛河|艾辛河]]渡口戰役中被殺害，之後[[亞拉岡_(魔戒)|亞拉岡]]在[[伊多拉斯|伊多拉斯]]馬廄中安撫躁動不安的牠，自此兩人成為親密的朋友。在亞拉岡被座狼拖下懸崖後，布里哥前來救了他。

== 羅赫林 ==
羅赫林是亞拉岡在北方時的坐騎。魔戒聖戰時期，[[賀爾巴拉|賀爾巴拉]]等[[北方遊俠|北方遊俠]]將牠帶來洛汗交給亞拉岡，此後亞拉岡騎著牠通過[[亡者之道|亡者之道]]，並參與了在[[剛鐸|剛鐸]]的戰役。羅赫林和牠的同胞（其餘北方遊俠的坐騎）一樣勇敢，能夠克服恐懼跟隨主人進入亡者之道<ref name=":1" />。

== 影疾 ==
{{main|影疾}}影疾屬於米亞拉斯神駒種族的馬，在魔戒聖戰期間是[[巫師_(中土大陸)|巫師]]甘道夫的座騎，牠陪伴甘道夫經歷了所有的旅程，直到前往[[阿門洲|海外仙境]]<ref>{{Cite news|url=http://lotr.wikia.com/wiki/Shadowfax|title=Shadowfax|newspaper=The One Wiki to Rule Them All|accessdate=2017-12-27|language=en}}</ref>。

== 哈蘇風與阿羅德 ==
亞拉岡、[[勒苟拉斯|勒苟拉斯]]與[[金靂|金靂]]在追尋[[強獸人|強獸人]]途中於洛汗平原上遇見了[[伊歐墨|伊歐墨]]的部隊，在經過長時間的討論，伊歐墨知道他們並非洛汗的敵人，便送給他們兩匹馬——哈蘇風（Hasufel）與阿羅德（Arod ），牠們此前的主人在與強獸人戰鬥時遇害。亞拉岡騎乘的是哈蘇風，勒苟拉斯與金靂則騎乘阿羅德，這兩匹馬跟隨了他們很長一段時間<ref name=":2">{{Cite book|title=魔戒二部曲：雙城奇謀|last=托爾金|publisher=聯經|year=2012年12月|isbn=978-957-08-4101-5|location=臺北}}</ref>。

== 史戴巴 ==
史戴巴是[[洛汗_(地名)|洛汗]]的一匹灰色小馬。在[[聖盔谷之戰|聖盔谷之戰]]後，梅里成為[[希優頓|希優頓]]的隨扈，他騎乘史戴巴共路經了[[登哈洛|登哈洛]]與伊多拉斯<ref name=":1" />。

== 雪鬃 ==
[[File:Flag of the Kingdom of Rohan No Border.svg|缩略图|192x192像素|
洛汗國的旗幟就是一匹正在奔跑的白色駿馬。
]]
雪鬃是[[洛汗國王|洛汗國王]]希優頓騎乘的白色駿馬，可能也是屬於米亞拉斯一種，牠奔跑的速度非常快速，比其他騎士還要快很多。然而牠在[[帕蘭諾平原戰役|帕蘭諾平原戰役]]上因中了巫王的毒箭，而將其主人希優頓壓在其身下。戰後雪鬃的屍體被埋在平原上<ref name=":1" />。

== 火蹄 ==
為洛汗國第三元帥伊歐墨的坐騎，曾於前往[[聖盔谷|聖盔谷]]的途中載過[[矮人_(中土大陸)|矮人]]金靂一程。目前還不清楚牠是否為米亞拉斯種族<ref name=":2" />。

== 溫佛拉 ==
在洛汗軍隊於登哈洛集結後，希優頓開始拒絕讓梅里參與戰鬥，因為身為小哈比人的他對騎士而言只是一種負擔。然而梅里隨後在伊多拉斯遇見了一位騎士德海姆（實際上是[[伊歐玟|伊歐玟]]），後者讓梅里騎乘他的坐騎溫佛拉。在帕蘭諾平原戰役時，溫佛拉被巫王嚇到，而在平原上四處奔跑<ref name=":1" />。

== 戒靈的馬 ==
甘道夫曾表示過九名戒靈騎乘的黑馬，從一生下來就是被訓練成服侍[[索倫_(魔戒)|索倫]]的，所以牠們不會對戒靈產生恐懼<ref name=":0" />。有可能索倫派出底下的[[半獸人|半獸人]]至洛汗國搶劫黑色的馬匹<ref name=":2" />，而送到[[魔多|魔多]]來訓練牠們成為戒靈坐騎的。

== 索倫之口的馬 ==
僅在《[[王者再臨|王者再臨]]》的「黑門開啟」一章中出場，小說描述，[[索倫之口|索倫之口]]騎的這匹馬十分巨大，臉部更像是個可怕的骷髏面具<ref name=":1" />。

== 羅哈洛 ==
羅哈洛是[[第一紀元|第一紀元]]精靈國王[[芬國盼|芬國盼]]的坐騎。芬國盼騎著牠前往[[安格班|安格班]]迎戰魔王[[魔苟斯|魔苟斯]]<ref>{{Cite book|title=精靈寶鑽|last=托爾金|publisher=聯經|year=2015年1月初版第二十一刷|isbn=978-957-08-2524-4|location=臺北}}</ref>。

== 費樂羅夫 ==
費樂羅夫（Felaróf）是洛汗國王[[伊歐|伊歐]]給那匹殺死他父親的馬取的名字。當時伊歐之父[[李歐德|李歐德]]在試圖馴服費樂羅夫時，牠變得狂野並將李歐德摔死。伊歐追尋了這匹馬好長一段時間，當他找到費樂羅夫並沒有殺死牠，而是將其制服，以不用[[馬鞍|馬鞍]]的方式駕馭牠。自此費樂羅夫的後代子孫便屬於洛汗國王，這個馬種被稱作「米亞拉斯」，除了國王的子孫外不讓任何人騎乘牠們；唯一的例外是影疾<ref name=":1" />。

== 米亞拉斯神駒==
米亞拉斯是中土大陸的一個馬種，其祖先來自於西方的海外仙境，牠們可以理解人類的語言，不喜歡馬鞍，只允許特定對象（洛汗王族及後來的甘道夫）騎乘牠們。已知名字的米亞拉斯神駒有費樂羅夫和影疾<ref name=":1" />。
