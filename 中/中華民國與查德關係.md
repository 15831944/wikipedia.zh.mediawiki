'''中華民國與查德關係'''是指[[中華民國|中華民國]]與[[查德共和國|查德共和國]]之間的關係。兩國於1962－2006年有官方[[外交#外交上的承認|外交關係]]但曾經中斷，二次斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對查德的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-234-a88f4-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[查德|查德]]。]]
{{see also|中華民國外交|{{link-en|查德外交|Foreign relations of Chad}}}}
{{see also|中華民國駐查德大使列表{{!}}中華民國駐查德外交官列表|查德駐中華民國大使列表{{!}}查德駐中華民國外交官列表}}
1962年1月31日，兩國[[外交#外交上的承認|建立外交關係]]。於首都[[恩將納|恩將納]]設立'''中華民國駐查德共和國[[外交代表機構#大使馆|大使館]]'''，1966年派駐[[特命全權大使|大使]]。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-3.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1972年11月28日，查德與[[中華人民共和國|中華人民共和國]]建交，中華民國與之斷交。<ref name="86年外交年鑑"/>

1997年8月12日，查德外交部長{{link-en|阿納狄夫|Mahamat Saleh Annadif}}抵台，與中華民國外交部長[[章孝嚴|章孝嚴]]在[[台北市|台北]]簽署《復交聯合公報》，恢復外交關係；15日，中華人民共和國與查德斷交。其後，兩國'''互設大使館'''、互派大使。<ref name="86年外交年鑑"/>

2006年8月5日，兩國再次斷交，查德與中華人民共和國復交。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=8742DCE7A2A28761&sms=491D0E5BF5F4BC36&s=2C11BF7FD929A3AD |author = 公眾外交協調會 |date = 2006年8月9日 |title = 外交部說明處理與查德斷交情形及政府立場記者會 |publisher = 中華民國外交部}}</ref>行政院长[[苏贞昌|苏贞昌]]被迫取消参加[[乍得总统|乍得总统]]就职典礼的行程<ref>[http://www.voachinese.com/a/a-21-w2006-08-06-voa1-58399242/1081360.html 乍得在苏贞昌来访前夕对台湾断交]</ref>。

===貿易===
2017年，查德是中華民國第118大貿易夥伴、第93大[[進口|進口]]夥伴、第214大[[出口贸易|出口]]夥伴。出口至查德的金額為5萬3,364[[美元|美元]]，年減74.677%；自查德進口的金額為2,672萬1,766美元，年減6.122%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）2,666萬8,402美元，年減5.611%。※2016年，查德是中華民國第113大貿易夥伴、第89大進口夥伴、第204大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
[[台灣中油|臺灣中油公司]]赴查德探勘石油，自2006年1月25日起，透過轉投資[[子公司|子公司]]OPIC（海外石油及投資公司）與[[乍得政治|查德政府]]正式簽署礦區[[合約|合約]]及聯合經營合約，取得查德BLTI、BCSII、BCOIII共3礦區的探勘[[專營權|專屬權]]，並由中油擔任礦區經營人，查德政府則為[[合伙企业|合夥人]]。2011年1月，中油在查德礦區發現重大油藏，是近40年來日產量最多的油井<ref name="中油探勘查德油井成功">{{Cite web |url = http://www.moneydj.com/KMDJ/News/NewsViewer.aspx?a=dd0880c4-6e8e-4ab2-9ad1-ad3c7d75b9f6 |title = 中油探勘查德油井成功！可日產原油9800桶 |date = 2011年2月8日 |publisher = MoneyDJ理財網}}</ref><ref name="有查德油田能賺錢">{{Cite web |url = http://www.ettoday.net/news/20130104/148384.htm |title = 有查德油田能賺錢　中油虧損怪罪「政策性補貼」 |date = 2013年1月4日 |publisher = 東森新聞雲}}</ref>。2016年9月，中油宣布為了分攤投資風險將與[[中國大陸|中國大陸]]的[[華信能源公司|華信能源公司]]合作，轉讓三成五的權益給陸方<ref name="中油部分讓利">{{Cite web |url = http://a.udn.com/focus/2016/09/04/24335/index.html?from=udn-referralnews_ch2artbottom |title = 10年查德採油戰收手 中油部分讓利陸企 |date = 2016年9月4日 |publisher = 聯合報}}</ref>。預計2019年底產出原油。

==交流==
===援助===
1964年，兩國簽署《[[農學|農業發展技術]]合作協定》，1965年，中華民國派遣[[農耕|農耕]]隊、[[獸醫|獸醫]]隊、{{link-en|搾油|Expeller pressing}}技術隊前往查德，其中農耕隊至1972年兩國斷交後撤離。1997年8月12日，兩國恢復外交關係後，根據所簽署《[[經濟|經濟]]暨[[技術|技術]]合作協定》、《[[醫療|醫療]]合作協定》立即展開合作計畫。農業技術考察團赴查德考察，並決定派遣農技團，協助該國發展[[稻米|稻米]]與[[蔬菜|蔬菜]]作物，以增加農業產量；另派遣醫療人員進駐首都[[恩將納|恩將納]]的自由醫院，展開技術援助工作。<ref name="86年外交年鑑"/><ref name="87年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1998/section_3/page3-1-3.htm |title = 《中華民國87年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>
