'''中华全国新闻工作者协会'''，简称'''中国记协'''，是[[中华人民共和国|中华人民共和国]]新闻界最大、影响力最为广泛的[[社会团体|社会团体]]，同时受[[中国共产党|中国共产党]]领导，亦是参加[[中国人民政治协商会议|中国人民政治协商会议]]的团体之一。前身为在[[周恩来|周恩来]]倡导下于1937年11月8日在[[上海|上海]]成立的'''[[中国青年新闻记者学会|中国青年新闻记者学会]]'''。协会现有团体会员214个，覆盖全国70多万新闻工作者，并与国外新闻界积极展开交流合作。<ref>[http://www.xinhuanet.com/zgjx/staticcontent/jxjj.html 中华全国新闻工作者协会简介]中国记协网 2010年05月20日</ref>

== 沿革 ==
* 1937年11月8日，中国青年新闻记者学会在[[上海|上海]]成立。
* 1949年7月，中华全国新闻工作者协会筹备会在[[北平|北平]]成立。
* 1957年3月14日，中华全国新闻工作者协会在[[北京|北京]]正式成立。
* 1966年，因为[[文化大革命|文化大革命]]，协会被迫停止活动。
* 1980年，协会恢复活动。
* 1997年11月8日，记协成立60周年，正式推出会徽。<ref>[http://china.qianlong.com/4352/2006/11/02/2502@3489801.htm 中华全国新闻工作者协会] {{webarchive|url=https://web.archive.org/web/20150320153251/http://china.qianlong.com/4352/2006/11/02/2502%403489801.htm |date=2015-03-20 }}2006-11-02</ref>

== 组织 ==
协会的最高权力机关为全国理事会，五年一届。理事由各下属团体单位推选产生，另设常务理事会承担理事会闭会间的工作。常务理事会选举产生主席、副主席并任命书记处书记，后者负责理事会的日常工作。

== 主办奖项 ==

* [[长江韬奋奖|长江韬奋奖]]（范长江新闻奖、韬奋新闻奖于2005年合并而成）
