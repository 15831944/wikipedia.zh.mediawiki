'''中華民國與聖馬利諾關係'''是指[[中華民國|中華民國]]與[[聖馬利諾共和國|聖馬利諾共和國]]之間的關係。兩國從未有官方[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對聖馬利諾的相關事務由'''駐[[義大利|義大利]][[台北经济文化代表处|台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-156-f5318-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|歐洲聯盟簽證政策|{{link-en|聖馬利諾公民簽證要求|Visa requirements for San Marino citizens}}}}
聖馬利諾沒有加入[[歐洲聯盟|歐洲聯盟]]與《[[申根公約|申根公約]]》，但根據1999年生效的《[[阿姆斯特丹條約|阿姆斯特丹條約]]》使得《申根既有規範》（Schengen acquis）正式成為[[歐盟法律|歐盟法律]]的一部分，且該國全境在[[義大利行政區劃|義大利國土]]內，並開放[[邊界|邊界]]，因此[[De_facto|實質上]]可被認作屬於申根區，能適用歐盟的簽證政策。歐洲申根區給予持有載明[[中華民國國民身份證|身份證]]字號的[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以[[歐洲聯盟簽證政策|申根區免簽證]]入境，停留日數與申根區合併計算，每6個月期間內總計可停留90天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-156-f5318-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_FAQ.aspx?c=AD6908DFDDB62656&n=C1B19EF0ACDD3C91&sms=867F5398A49F758A |title = 歐盟免申根簽證常見問答集 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

{{link-en|聖馬利諾人口|Demographics of San Marino|聖馬利諾公民}}持有{{link-en|聖馬利諾護照|San Marino passport}}者也可以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將聖馬利諾歸類為'''灰色警示'''：提醒注意（2016年11月4日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-349-94d25-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2017年，聖馬利諾是中華民國第192大貿易夥伴、第203大[[進口|進口]]夥伴、第183大[[出口贸易|出口]]夥伴。出口至聖馬利諾的金額為83萬0,588[[美元|美元]]，年增169.032%；自聖馬利諾進口的金額為3萬0,534美元，年增38.163%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）80萬0,054美元，年增179.122%。※2016年，聖馬利諾是中華民國第208大貿易夥伴、第203大進口夥伴、第198大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
