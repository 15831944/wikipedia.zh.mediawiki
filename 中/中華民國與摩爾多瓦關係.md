'''中華民國與摩爾多瓦關係'''是指[[中華民國|中華民國]]與[[摩爾多瓦共和國|摩爾多瓦共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對摩爾多瓦的相關事務由'''[[臺北市|臺北]]－[[莫斯科|莫斯科]]經濟文化協調委員會[[台北经济文化代表处|駐莫斯科代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-147-ba17a-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，[[摩爾多瓦|摩爾多瓦]]當時屬於[[蘇聯|蘇聯]]的[[加盟共和國|加盟共和國]]（[[摩爾達維亞蘇維埃社會主義共和國|摩爾達維亞蘇維埃社會主義共和國]]），由蘇聯作為[[主權國家|主權國家]]參與投票。]]

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|摩爾多瓦簽證政策|Visa policy of Moldova}}|{{link-en|摩爾多瓦公民簽證要求|Visa requirements for Moldovan citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]與邀請函的[[中華民國國民|中華民國公民]]可至摩爾多瓦[[簽證#电子签证|電子簽證]]申請系統辦理簽證，但[[摩爾多瓦政治|摩爾多瓦政府]]對[[中華民國政府|中華民國政府]]並非友善，取得該國簽證具相當難度，[[中華民國外交部|中華民國外交部]]也建議避免在該國轉機。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-147-ba17a-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持有{{link-en|摩爾多瓦護照|Moldovan passport}}的[[摩爾多瓦人口|摩爾多瓦公民]]抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|摩爾多瓦經濟}}
===貿易===
2017年，摩爾多瓦是中華民國第152大貿易夥伴、第123大[[進口|進口]]夥伴、第145大[[出口贸易|出口]]夥伴。出口至摩爾多瓦的金額為385萬3,833[[美元|美元]]，年增3.838%；自摩爾多瓦進口的金額為493萬5,347美元，年減29.952%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）108萬1,514美元，年減67.563%。※2016年，摩爾多瓦是中華民國第141大貿易夥伴、第118大進口夥伴、第152大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
