'''中華民國與古巴關係'''是指[[中華民國|中華民國]]與[[古巴共和國|古巴共和國]]之間的關係。1913－1960年兩國有正式[[外交#外交上的承認|外交關係]]，因[[古巴革命|古巴共產革命]]導致斷交後，與中華民國關係極不友好。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-200-b9999-06-1.html |title = 特殊資訊 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構，對古巴的相關事務由'''{{link-en|駐哥倫比亞台北商務辦事處|Taipei Commercial Office in Bogotá, Colombia}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-200-b9999-03-1.html |title = 駐館位置及聯絡方式 |publisher = 外交部領事事務局}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[古巴|古巴]]。]]
{{see also|中華民國外交|{{link-en|古巴外交|Foreign relations of Cuba}}}}
{{see also|中国驻古巴大使列表{{!}}中華民國駐古巴外交官列表|古巴驻华大使列表{{!}}古巴駐中華民國外交官列表}}
1912年，中華民國成立後，向古巴派駐外交代表。

1913年5月4日，中華民國與古巴建立公使級[[外交#外交上的承認|外交關係]]，由[[中國駐美國大使列表|駐美國]]、[[中国驻墨西哥大使列表|墨西哥公使]]兼任。

1921年，於首都[[哈瓦那|哈瓦那]]設立'''中華民國駐古巴共和國[[外交代表機構#公使馆|公使館]]'''，兼使改為[[特命全權公使|專使]]。

1942年11月12日，[[中国驻古巴大使列表|駐古巴公使]]與{{link-en|古巴外交部|Ministry of Foreign Affairs (Cuba)}}長簽署《中華民國古巴共和國友好條約》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e5%8f%a4%e5%b7%b4&tycountry_e=%e5%8f%a4%e5%b7%b4&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1957年8月23日，公使館升格為'''中華民國駐古巴共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1959年[[古巴革命|古巴革命]]，親美的[[巴蒂斯塔|巴蒂斯塔]]政權被推翻。

1960年9月3日，古巴[[菲德爾·卡斯楚|卡斯楚]]政府與中華民國斷交。

2002年1月14日，[[塞內加爾|塞內加爾]]、[[格瑞那達|格瑞那達]]、[[尼加拉瓜|尼加拉瓜]]、[[瓜地馬拉|瓜地馬拉]]及[[查德|查德]]於2001年11月14－16日致函[[世界衛生組織|世界衛生組織]]（WHO）幹事長[[格罗·哈莱姆·布伦特兰|布琅蘭]]（Gro Harlem Brundtland），要求將「邀請中華民國以[[世界衛生大會觀察員|觀察員]]身分參加[[世界衛生大會|世界衛生大會]]」之提案列入2002年5月的第五十五屆世界衛生大會（WHA）討論，並盼WHO執行委員會能將該提案列入WHA之臨時議程。古巴執委發言要求執委會不應討論上述提案，此發言引發激烈辯論。最後古巴提出[[程序動議|程序動議]]，建議不將邀請台灣以觀察員身分參加WHA之提案納入該執委會議程討論，並要求交付[[投票|表決]]。會中隨即表決該項動議，計有[[北韓|北韓]]等20國贊成，[[中華民國外交#中华民国的邦交国|友邦]]3國反對，[[英國|英國]]等8國棄權，[[哈薩克|哈薩克]]缺席未參與投票。主席依表決結果宣布此案將不列入執委會討論。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=311EA34F5B667B52 |author = 公眾外交協調會 |date = 2002年1月15日 |title = WHO執委會決議不將我參與世衛組織提案列入討論，我深感遺憾 |publisher = 中華民國外交部}}</ref>

2016年4月14日，中華民國法務部長[[羅瑩雪|羅瑩雪]]到[[立法院司法及法制委員會|立法院司法及法制委員會]]就[[2016年台灣詐騙集團成員涉嫌電信詐騙案|肯亞遣送台灣詐騙嫌犯至中國大陸事件]]進行專案報告，時代力量立委[[黃國昌|黃國昌]]就一名在[[巴拿馬|巴拿馬]]被中華人民共和國發布[[紅色通緝令|紅色通緝令]]的台灣人，被送古巴途中[[跳樓|跳樓]]自殺之事進行質詢。由於羅瑩雪提及時說出「然後他就死掉了」，其用詞讓黃國昌暴怒。羅瑩雪指出法務部接獲通報後，請示前行政院長[[毛治國|毛治國]]、總統[[馬英九|馬英九]]，馬英九指示[[中華民國法務部|法務部]]跟[[中華民國外交部|外交部]]、[[陸委會|陸委會]]協商，尚未協商，「人就上飛機了」。黃國昌批評因為政府的遲延，人才會被送到古巴；羅瑩雪則反問哪邊遲延？雙方你來我往、互不示弱。<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20160414/838742/ |author = 法庭中心 |date = 2016年4月14日 |title = 【更新】台人遭中國紅色通緝 羅瑩雪：「他就死掉了」 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》}}</ref><ref>{{Cite web |url = http://www.ettoday.net/news/20160414/680509.htm?feature=ponta&tab_id=21 |author = 徐政璿 |date = 2016年4月14日 |title = 台人逃亡古巴！羅瑩雪一句「他就死掉了」黃國昌震怒 |publisher = 《[[ETtoday_東森新聞雲|ETtoday 東森新聞雲]]》}}</ref>

2017年5月22日，世界衛生大會（WHA）開幕，在進入議程前，大會先行討論由總務委員會提交的議程建議，其中包括由中華民國11個友邦聯合提出的補充提案項目，邀請台灣以觀察員身分出席WHA。為了讓各方意見能夠充分表達，大會主席裁決進行[[辯論|二對二辯論]]，由邦交國[[聖文森及格瑞那丁|聖文森及格瑞那丁]]與[[帛琉|帛琉]]，以及中華人民共和國與古巴，分別代表正反方發言。古巴代表指出，《[[聯合國大會2758號決議|聯合國大會2758號決議]]》以及《WHA 25.1號決議》，都已處理過中華人民共和國做為[[中國與聯合國#代表權問題|唯一中國代表]]，參加WHA等相關會議的問題，因此大會應該將精力集中在重要的問題上。<ref>{{Cite web |url = http://www.upmedia.mg/news_info.php?SerialNo=17611%2F%3F%3Dfb |author = 仇佩芬 |date = 2017年5月22日 |title = WHA慷慨陳詞5分鐘　我友邦挺台：政治問題不該在世衛解決 |publisher = 《[[上報|上報]]》}}</ref>最後，總務委員會決定不排入議程。<ref>{{Cite web |url = http://news.ftv.com.tw/NewsContent.aspx?ntype=air&sno=2017523P05M1 |author = 顏辰州、楊柏玨 |date = 2017年5月23日 |title = 挺台提案不列議程 陳時中:中國汙染人權 |publisher = 《[[民視新聞台|民視新聞台]]》}}</ref>

2018年5月21日，古巴依舊在世界衛生大會（WHA）總務委員會的二對二辯論擔任反方，封殺「邀請台灣以觀察員身分參與WHA」的提案列入議程。<ref>{{Cite web |url = https://udn.com/news/story/6656/3155235 |author = 徐偉真 |date = 2018年5月22日 |title = 友邦替我執言 WHA和中國二對二辯論 |publisher = 《[[聯合新聞網|聯合新聞網]]》}}</ref>

{{see also|古巴簽證政策|{{link-en|古巴公民簽證要求|Visa requirements for Cuban citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可於出發所在地[[旅行社|旅行社]]購買古巴「觀光卡」（台灣本地旅行社目前無代售），或於抵達古巴前一站所在地辦理[[登機報到|登機手續]]時，向[[航空公司|航空公司]]購買觀光卡，停留最多30天。<ref name="簽證及入境須知">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-200-b9999-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持有{{link-en|古巴護照|Cuban passport}}的[[古巴公民|古巴公民]]抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將古巴歸類為'''灰色警示'''：提醒注意（2018年1月16日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-60-12bd3-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將古巴歸類為'''第二級警示'''：對當地採取加強防護（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=BF8212C8B091475E |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
2018年，古巴是中華民國第139大貿易夥伴、第114大[[進口|進口]]夥伴、第143大[[出口贸易|出口]]夥伴。出口至古巴的金額為586萬7,547[[美元|美元]]，年增55.744%；自古巴進口的金額為817萬2,398美元，年增2.240%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）230萬4,851美元，年減45.459%。2017年，古巴是中華民國第139大貿易夥伴、第118大進口夥伴、第147大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
