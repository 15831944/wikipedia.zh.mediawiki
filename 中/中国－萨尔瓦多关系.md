'''中國－薩爾瓦多關係'''是指[[中国|中国]]与[[萨尔瓦多|萨尔瓦多]]的双边关系，现时指[[中華人民共和國|中華人民共和國]]與[[萨尔瓦多共和国|萨尔瓦多共和国]]的外交關係。
== 历史 ==
=== 中华民国 ===
{{seealso|中華民國與薩爾瓦多關係|满洲国外交}}
1933年6月15日，中华民国政府与[[危地马拉|危地马拉]]建立领事级外交关系，由[[中華民國駐瓜地馬拉大使列表|驻危地马拉总领事]]兼办萨尔瓦多侨务<ref name="ROC Foreign Relation Yearbook 1997"/>。10月7日，中华民国驻危地马拉总领事馆正式开馆。

1934年5月21日，萨尔瓦多驻日本东京总领事葛伦沙通知满洲国驻日本公使[[丁士源|丁士源]]，该国已在3月3日宣布承认满洲国。萨尔瓦多是继[[日本|日本]]之后第二个承认满洲国的国家。消息传开，中国各界一片抗议，[[中华民国外交部|中华民国外交部]]为此致电[[国际联盟|国际联盟]]，要求针对萨尔瓦多这一行为予以制裁<ref>{{cite web |title=伪满洲国成立后溥仪曾想逃离东北 |url=http://www.jnocnews.jp/news/show.aspx?id=86136 |website=日本新华侨报网 |accessdate=2018-08-24 |date=2016-05-30}}</ref>。该行为违反了国际联盟在1933年2月通过的不承认满洲国的合法性决议和1933年6月发布的《不承认满洲国办法》。1934年6月萨尔瓦多受灾，满洲国皇帝[[溥仪|溥仪]]特别拨出1万元“内帑金”捐助给萨方。萨尔瓦多承认满洲国后一直未派驻外使节，后委托满洲国代聘领事。1940年，满洲国外交部委派[[王荆山|王荆山]]出任萨尔瓦多驻满洲国名誉领事<ref>{{cite web |author1=张贤达 |title=王荆山耐人寻味的双色人生 |url=http://ccwb.1news.cc/html/2010-11/29/content_111304.htm |publisher=长春晚报 |accessdate=2018-08-24 |date=2010-11-29}}</ref>。虽然在1941年12月[[珍珠港事变|珍珠港事变]]后，萨尔瓦多就追随[[美国|美国]]对日本宣战，但直到1943年1月，萨尔瓦多才宣布取消对满洲国的承认<ref>{{cite web |author1=司马戡 |title=萨尔瓦多外交风云:溥仪逃亡目标,台军留学首选 |url=http://news.163.com/18/0823/09/DPSRQR92000187UE.html |website=网易历史 |accessdate=2018-08-24 |date=2018-08-23}}</ref>。

1941年4月9日，[[中华民国|中华民国]]与萨尔瓦多建立公使级外交关系，由[[中国驻巴拿马大使列表|驻巴拿马公使]]兼任驻萨尔瓦多公使<ref name="ROC Foreign Relation Yearbook 1997">{{cite web|url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html|title=中華民國外交部86年外交年鑑|publisher=[[中華民國外交部|中華民國外交部]]|date=1998-06|accessdate=2017-12-16|archive-url=https://web.archive.org/web/20130604074247/http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html|archive-date=2013-06-04|dead-url=no}}</ref>。1949年10月1日[[中华人民共和国|中华人民共和国]]成立后，萨尔瓦多继续维持与中华民国政府的外交关系，且于1961年6月1日升格大使級外交關係<ref name="105年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部 |deadurl = no |archiveurl = https://web.archive.org/web/20180129004249/http://multilingual.mofa.gov.tw/web/web_UTF-8/YearBook/105/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |archivedate = 2018-01-29 }}</ref>。

2018年8月21日，萨尔瓦多与中华民国断交。[[中華民國外交部長列表|中华民国外交部长]][[吳釗燮|吳釗燮]]宣布，停止雙邊合作並撤離大使館、技術團，并称薩國政府自2017年以来曾不斷向中华民国要求大量資金，援助该国联合港的开发案，但台方认为「這是不合適的開發案」，可能导致台湾和萨尔瓦多政府都陷入高度的债务风险，因此拒绝<ref>{{cite web |author1=曹宇帆 |title=中國挹注薩爾瓦多聯合港開發 美國曾示警 |url=https://www.nownews.com/news/20180821/2804028 |website=今日新闻 |accessdate=2018-08-24 |date=2018-08-21}}</ref>。對此，薩爾瓦多總統發言人羅倫沙納（Roberto Lorenzana）于8月21日反駁称，吳釗燮的指控不但「卑鄙惡劣」，且「完全錯誤」，称薩國轉與中国大陸建交的主因是當局與北京貿易所帶來的龐大優勢，以及國際社會上多数國家和大陸建立商業與外交關係的现状<ref>{{cite web |author1=詹雅婷 |title=台薩斷交！吳釗燮稱要錢　薩爾瓦多怒駁：台指控錯誤、卑鄙惡劣 |url=https://www.ettoday.net/news/20180822/1241256.htm |website=ETtoday新聞雲 |accessdate=2018-08-24 |date=2018-08-22}}</ref>。

=== 中华人民共和国 ===
==== 建交前 ====
1950年11月，[[中國人民解放軍進軍西藏|中国人民解放军进入西藏]]后，萨尔瓦多在联合国提议“外国军队进入西藏”案列入大会议程<ref>{{Cite book|title=美洲列国志|last=世界知识社编辑|first=|publisher=[[世界知识出版社|世界知识社]]|year=1953.10|isbn=|location=|pages=92}}</ref><ref>{{Cite book|title=世界知识手册 1954|last=世界知识手册编辑委员会|first=|publisher=世界知识社|year=1954.02|isbn=|location=|pages=510}}</ref>。[[昌都战役|昌都战役]]后，当时西藏代表团团长[[黎吉生|黎吉生]]要求[[土丹旦达|土丹旦达]]等人，向联合国致信呼吁。在[[里查逊|里查逊]]修改定稿、译成英文后，决定派遣[[嘉乐顿珠|嘉乐顿珠]]和[[夏格巴·旺秋德丹|夏格巴·旺秋德丹]]二人去联合国呼吁，请求支持。1950年11月8日，其控诉书交给联合国后，英、美国均不敢公开出面支持，而指使联合国大会轮值主席、萨尔瓦多代表团团长{{link-es|赫克托·戴维·卡斯特罗|Héctor David Castro}}<ref>{{Cite book|title=喇嘛王国的覆灭|last=（美）梅·戈尔斯坦著；杜永彬译|first=|publisher=北京：中国藏学出版社|year=2005|isbn=7-80057-725-2|location=|pages=621}}</ref>在11月15日，建议联合国大会讨论西藏问题<ref>{{Cite book|title=东西之间的“西藏问题” 外二篇|last=汪晖著|first=|publisher=北京：[[生活·读书·新知三联书店|生活·读书·新知三联书店]]|year=2014.08|isbn=978-7-108-05036-6|location=|pages=63}}</ref>，“设立委员会授权研讨联大对此事可以采取的适当步骤”<ref>{{Cite book|title=中美关系史 修订本 第2卷 1949-1972|last=陶文钊著|first=|publisher=上海：[[上海人民出版社|上海人民出版社]]|year=2016.08|isbn=978-7-208-13895-7|location=|pages=99-100}}</ref>。[[苏联|苏联]]代表[[雅科夫·馬利克|雅科夫·馬利克]]<ref>{{Cite book|title=和平解放西藏与执行协议的历史记录 上|last=中共西藏自治区委员会党史研究室编著|first=|publisher=北京：[[中共党史出版社|中共党史出版社]]|year=2014.09|isbn=978-7-5098-2793-2|location=|pages=55}}</ref>和[[中华民国|中华民国]]代表均发言反对讨论这个问题，“认为西藏是中国不可分割的一部分”。英国代表则称对实际情况不了解，建议延期讨论。印度代表支持英国的建议；美国代表之后同意搁置，没有举行辩论<ref>{{Cite book|title=班禅额尔德尼传|last=牙含章著|first=|publisher=北京：[[华文出版社|华文出版社]]|year=2015.02|isbn=978-7-5075-4293-6|location=|pages=261}}</ref>。

1986年6月以[[刘绍山|刘绍山]]为首的贸促会代表团访萨。1986年10月萨尔瓦多地震后，[[中国红十字会|中国红十字会]]致电慰问并赠款3万美元<ref>{{Cite book|title=各国概况 美国和大洋洲部分|last=马武业主编|first=|publisher=北京：世界知识出版社|year=1991.11|isbn=7-5012-0296-6|location=|pages=251}}</ref>。1995年5月，中国企业家代表团访萨；9月，萨总统夫人伊丽莎白·德卡尔德隆（{{lang-en|Elizabeth Aguirre de Calderón}}）率团出席在北京举行的[[联合国世界妇女大会|联合国第四次世界妇女大会]]<ref>{{Cite book|title=各国概况 中美|last=孟淑贤主编|first=|publisher=北京：世界知识出版社|year=1997.01|isbn=7-5012-0835-2|location=|pages=86}}</ref>。

2000年6月，[[中国驻墨西哥大使列表|中国驻墨西哥大使]][[沈允熬|沈允熬]]访萨。8月，以萨尔瓦多共和国市政委员会主席、新圣萨尔瓦多市市长奥尔蒂斯为首的萨尔瓦多市长代表团防华。10月，以副总协调员阿玛亚为首的萨尔瓦多马蒂阵线代表团访华。11月，以议会领导委员会秘书长、全国协和党领导成员马丘卡为首的多党议员团访华。2000年联合国大会会议上，萨未作为提案国支持中华民国重返联合国，亦末联署。在2000年联合国人权会上，萨尔瓦多也对中华人民共和国动议投了反对票<ref>{{Cite book|title=世界知识年鉴 2001-2002|last=世界知识年鉴编辑委员会编|first=|publisher=北京：世界知识出版社|year=2001.12|isbn=7-5012-0444-6|location=|pages=1004}}</ref>。2002年5月，[[法拉本多·马蒂民族解放阵线|法拉本多·马蒂民族解放阵线]]副总协调员格瓦拉访华。7月，政协外委会副主任委员李北海和[[秦华孙|秦华孙]]访萨。2003年7月，马蒂阵线政治委员会成员[[布兰蒂诺|布兰蒂诺]]访华。12月，马蒂阵线副总统候选人马塔访华。2004年10月，马蒂阵线总协调员桑切斯访华。12月，中联部工作组出席马蒂阵线第19届全国代表大会。2005年3月，外交学会副会长[[王珍_(外交官)|王珍]]、中联部拉美局副局长[[康学同|康学同]]率团访萨。9月，马蒂阵线政治委员会成员弗洛尔应中联部邀请访华。2006年6月，马蒂阵线议员马丁内斯访华。7月，马蒂阵线总协调员冈萨雷斯访华。9月，萨外长莱内斯参加[[李肇星|李肇星]]外长与中美洲外长的集体会晤。12月，侨联副主席[[李祖沛|李祖沛]]访萨<ref name=2008c>{{Cite book|title=世界知识年鉴 2008/2009|last=世界知识出版社编|first=|publisher=北京：[[世界知识出版社|世界知识出版社]]|year=2009.03|isbn=978-7-5012-3526-1|location=|pages=909}}</ref>。2008年5月，以[[达戈维托|达戈维托]]为团长的萨尔瓦多议员团一行四人访华。2009年3月，应[[马蒂阵线|马蒂阵线]]邀请，[[中共中央对外联络部|中共中央对外联络部]]工作组以观察员身份赴萨观察大选。

2011年9月，应中联部邀请，马蒂阵线派员参加中美洲加勒比国家政党干部考察团。2012年6月，[[中国外交学会|中国外交学会]]副会长[[黄星原|黄星原]]率团访萨<ref name=2011c />。

自2010年开始，由[[中国国际贸易促进委员会|中国国际贸易促进委员会]]主办，[[中国国际商会|中国国际商会]]承办的“中华人民共和国贸易展览会”分别于2010年、2012年、2015年和2017年等年份在萨尔瓦多举办<ref>{{cite web |author1=喻虹霞 |title=中国－萨尔瓦多经贸合作论坛成功举办 |url=http://news.163.com/10/1125/12/6MB8CVTR00014JB5.html |website=网易-新华网 |date=2010-11-25}}</ref><ref>{{cite web |author1=中国国际商会 |title=张伟副会长出席2015年（萨尔瓦多）中华人民共和国贸易展览会开幕式并致辞 |url=http://www.ccpit.org/Contents/Channel_3894/2015/1127/506898/content_506898.htm |website=中国国际贸易促进委员会 |accessdate=2018-08-21 |date=2015-11-27}}</ref><ref>{{cite web |author1=许雷 |title=萨尔瓦多中国贸易展览会促两国经贸往来 |url=http://www.xinhuanet.com/world/2017-07/24/c_1121370451.htm |website=新华网 |date=2017-07-24}}</ref>。

2001年12月，中华人民共和国加入[[世贸组织|世贸组织]]时，萨尔瓦多援引世贸组织相关规定，与中国互不适用《世贸组织协定》。2014年7月1日，萨方撤销对中国的互不适用。2014年11月26日，中国驻世贸组织代表团临时代办朱洪约见萨尔瓦多驻世贸组织大使利马，向其通报中国政府决定自2014年7月2日起对原产于萨尔瓦多的产品适用最惠国税率，并将退还对7月2日后到港货物多征的税款等情况<ref>{{cite news |author1=姚文 |title=中国决定对萨尔瓦多实行最惠国待遇 |url=http://intl.ce.cn/sjjj/qy/201412/02/t20141202_4024839.shtml |work=国际商报 |date=2014-12-02 }}{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。<ref>{{cite web |author1=商务部新闻办公室 |title=中国决定对萨尔瓦多实行最惠国待遇 |url=http://www.mofcom.gov.cn/article/ae/ai/201412/20141200820763.shtml |website=中华人民共和国商务部 |accessdate=2018-08-21 |date=2014-12-04}}</ref>2014年12月8日，中华人民共和国海关总署正式发布公告<ref>{{cite web |title=中华人民共和国海关总署公告2014年第88号 |url=http://www.mofcom.gov.cn/article/b/g/201503/20150300908103.shtml |website=中华人民共和国商务部 |accessdate=2018-08-21 |date=2015-03-05}}</ref>。

2015年，萨尔瓦多政府与中华人民共和国政府在[[世界贸易组织|世界贸易组织]]内达成协议，中国给予萨尔瓦多等同于其他世贸组织成员的税收优惠。同年，萨尔瓦多蔗糖企业首次获得法律许可向中国大陆出口[[蔗糖|蔗糖]]{{efn|双方达成协议后，从萨尔瓦多出口到中国大陆的蔗糖商品支付中国海关50%的税收，而不是之前的125%。<ref>{{cite web |title=萨尔瓦多首次向中国大陆卖糖 还未与北京建交 |url=http://mil.news.sina.com.cn/2015-03-09/1635823727.html |website=新浪网-环球网 |date=2015-03-09 |access-date=2018-08-21 |archive-url=https://web.archive.org/web/20150318140026/http://mil.news.sina.com.cn/2015-03-09/1635823727.html |archive-date=2015-03-18 |dead-url=no }}</ref>}}<ref name="hubei">{{cite news |title=与台当局“断交”的萨尔瓦多是什么样的国家 |url=http://zy.cnhubei.com/dongxiang/uHkbJn1100 |accessdate=2018-08-21 |work=湖北日报-澎湃新闻}}</ref>。

双方建交之前，中华人民共和国驻[[哥斯达黎加|哥斯达黎加]]使馆经商参处曾负责推动中国与萨尔瓦多的双边经贸合作<ref>{{cite web |author1=驻哥斯达黎加经商参处 |title=驻哥斯达黎加使馆经商参处主要职能 |url=http://cr.mofcom.gov.cn/article/about/intruduce/201801/20180102698969.shtml |website=中华人民共和国商务部 |date=2018-01-16}}</ref>。

中国主要向萨尔瓦多出口纺织品、鞋类、车辆、机械设备、化学品等。萨尔瓦多主要向中国出口电力机械、金属矿砂等<ref name=fmprc/>。
