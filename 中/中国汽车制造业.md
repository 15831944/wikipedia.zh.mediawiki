'''中国汽车制造业'''增势迅猛，中国于2011年汽车销量，达1850万辆；2009年，中国生产的汽车销量突破1300万辆，超越美国成为全球最大的汽车市场<ref>{{Cite web |url=http://www.gov.cn/jrzg/2010-01/11/content_1507693.htm |title=汽车工业协会：去年中国汽车产销量跃居世界第一 |accessdate=2015-11-08 |author=记者：张毅 |date=2010-01-11 |work= |publisher=中华人民共和国中央人民政府网站，来源：新华社 |language=Zh-hans }}</ref>。

== 历史 ==
[[File:Jie_Fang,July_1956_in_Changchun.jpg|thumb]]

1930年代初，当时中国汽车保有量在万辆左右，均为外国品牌。在[[张学良|张学良]]支持下，位于[[沈阳|沈阳]]的辽宁迫击炮厂仿美国瑞雪牌[[货车|货车]]，于1931年6月制造出75型货车，取名“民生牌”<ref>{{Cite web |url=http://epaper.syd.com.cn/sywb/html/2013-11/26/content_960880.htm |title=汽车在沈阳的一百年 |accessdate=2015-11-08 |author= |date=2013-11-26 |work= |publisher=沈阳晚报数字报纸 |language=Zh-hans |archive-url=https://web.archive.org/web/20180513011259/http://epaper.syd.com.cn/sywb/html/2013-11/26/content_960880.htm# |archive-date=2018-05-13 |dead-url=yes }}</ref><ref name="a1"/>。数月后，即爆发[[九一八事變|九一八事变]]，辽宁迫击炮厂被[[日军|日军]]侵占。1934年，伪[[满洲国政府|满洲国政府]]在辽宁迫击炮厂基础上，引入日本[[同和自动车工业|同和自动车工业株式会社]]（翻译为中文即日本同和汽车股份公司），进行汽车组装和维修。1938年，已能年产4100辆[[轿车|轿车]]和[[卡车|卡车]]。此后，[[新京|新京]]（今[[长春市|长春市]]）、[[哈尔滨|哈尔滨]]两地设立卡车工厂。同时，日本在[[满洲国|满洲国]]地区的私营汽车工厂年产量亦有1800辆。而在[[中华民国政府|中华民国政府]]控制下的[[山西省_(中華民國)|山西]]和[[湖南省_(中華民國)|湖南]]两地，分别于1933年1936年试制出数辆货车<ref name="a1">{{Cite web |url=http://news.mydrivers.com/1/322/322170.htm |title=中国第一辆国产车 83年前张学良在沈阳制造 |accessdate=2015-11-08 |author= |date=2014-09-24 |work= |publisher=|language=Zh-hans }}</ref>。但随着[[抗日战争|抗日战争]]全面爆发，中国时局动荡不安。至1949年[[国府迁台|国府迁台]]为止，中国始终未能建立起自己的汽车工业体系。

1949年，[[中华人民共和国|中华人民共和国]]成立后，在[[苏联|苏联]]的援助下，[[中华人民共和国政府|中华人民共和国政府]]开始建立自己的汽车产业。1953年开工建设的[[中国第一汽车制造厂|中国第一汽车制造厂]]（简称中国一汽，位于[[吉林省|吉林省]]长春市），即是从苏联引进的[[156项重点工程|156项重点工程]]之一。1956年7月13日，第一辆[[解放牌汽车|解放牌汽车]]完成组装，实现量产。1958年，研制成功[[东风轿车|东风]]和[[红旗轿车|红旗轿车]]。中国一汽也因此被视为中国汽车工业的肇始<ref>{{Cite web |url=http://jlrbszb.chinajilin.com.cn/html/2009-09/29/content_559025.htm |title=中国一汽：共和国汽车工业的旗帜 |accessdate=2015-11-08 |author=记者：王建议 |date=2009-09-29 |work= |publisher=吉林日报网站 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160304053505/http://jlrbszb.chinajilin.com.cn/html/2009-09/29/content_559025.htm |archivedate=2016年3月4日 |df= }}</ref>。1958年，全国各地开始兴建汽车制造厂，至1960年共有16家，而维修改装车厂由16家发展为28家。但同时，中国的汽车工业在初期就形成了“缺重少轻”的结构特点<ref>{{Cite web |url=http://auto.ifeng.com/news/domesticindustry/20090901/95334.shtml |title=半个多世纪汽车档案 见证60年大变迁(组图)  |accessdate=2015-11-08 |author= |date=2009-09-01 |work= |publisher=凤凰网，来源：三秦都市报|language=Zh-hans }}</ref>。

1980年[[中国|中国]]生产22万辆汽车，厂家达到200家左右，每家平均生产量只有年产1000辆左右。最大的中国一汽的生产能力年产6万辆。主要产品是货车，也生产中国最有名的红旗牌轿车。1984年[[上海汽车|上海汽车]]和[[大众公司|大众公司]]设立合作汽车厂，开始生产轿车。1990年代其他外资企业投资中国，设立合作汽车厂。当时买车的大部分是[[出租车|出租车]]公司。2002年前后以来，中国人民个人所得增长，一般民众开始购买私人用车，从此汽车销售量大幅增长[[奇瑞|奇瑞]]，[[吉利|吉利]]等民族资本的厂家也进入市场。

2009年中国汽车生产量更超过1000万辆，达到1379万辆，超越[[日本|日本]]成为世界最大汽车生产国。<ref>{{Cite web|url=http://auto.ifeng.com/roll/20100205/206870.shtml|title=2009年中国汽车产量超过日本 跃居世界第一|publisher=凤凰网（来源：财经时报）|date=2010-02-05|accessdate=2016-12-18}}</ref>广州在数年间成为中国重要的汽车（尤其是[[轿车|轿车]]）生产基地之一，江南最大的[[广州市陈田汽配市场|广州市陈田汽配市场]]位於市郊，广州同时亦是国内出产日本汽车最多的城市。

2015年中国汽車生產量達2450萬輛，销量达2460万辆。<ref>{{Cite web|url=http://www.caam.org.cn/xiehuidongtai/20160112/1705183569.html|title=2015年汽车工业经济运行情况|publisher=中国汽车工业协会网站|date=2016-01-12|accessdate=2016-12-18}}</ref>2017年中国汽车销量达2887.9万辆，连续9年位居全球第一。<ref>{{Cite news |url=http://www.xinhuanet.com/politics/2018-01/21/c_129795500.htm |title=2017年我国汽车销量2887.9万辆 连续9年全球第一 |date=2018-01-21 |work=新华网}}</ref>
