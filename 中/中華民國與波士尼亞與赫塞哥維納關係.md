'''中華民國與波士尼亞與赫塞哥維納關係'''是指[[中華民國|中華民國]]與[[波士尼亞與赫塞哥維納|波士尼亞與赫塞哥維納]]（通稱'''波士尼亞'''、'''波赫'''）之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對波赫的相關事務由'''駐[[匈牙利|匈牙利]][[台北经济文化代表处|台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-201-34a82-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
{{see also|中華民國外交|波士尼亞與赫塞哥維納外交}}
2012年6月20日，[[波士尼亞與赫塞哥維納政府|波士尼亞與赫塞哥維納政府]]於{{link-en|波士尼亞與赫塞哥維納部長會議|Council of Ministers of Bosnia and Herzegovina|部長會議}}決定給予[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]入境，於完成政府公報刊登及後續行政程序後正式實施。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=965B7E3E5A7C81F1 |author = 公眾外交協調會 |date = 2012年7月5日 |title = 波士尼亞與赫塞哥維納予我國人免簽證待遇 |publisher = 中華民國外交部}}</ref>

2014年，援贈波士尼亞與赫塞哥維納之[[波赫聯邦|波赫聯邦]]8萬[[美元|美元]]協助[[2014年欧洲东南部洪灾|洪水災害重建]]。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/7fb2e392-9e9e-4bf7-a52e-68579f6bfca4.pdf |author = 國際合作及經濟事務司 |date = 2015年4月14日 |title = 國際合作發展事務103年度報告（參、103年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|波士尼亞與赫塞哥維納簽證政策|{{link-en|波士尼亞與赫塞哥維納公民簽證要求|Visa requirements for Bosnia and Herzegovina citizens|波赫公民簽證要求}}}}
持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境波士尼亞與赫塞哥維納，每6個月期間內總計可停留90天。<ref name="簽證及入境須知">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-201-34a82-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|波士尼亞與赫塞哥維納護照|Bosnia and Herzegovina passport|波赫護照}}的[[波赫公民|波赫公民]]抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將波赫歸類為'''灰色警示'''：提醒注意（2017年5月5日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-144-3e1cb-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2013年，[[經濟部國際貿易局|經濟部國際貿易局]]挑選最具拓銷潛力的22國26城市推動執行「貿易尖兵任務」，其中[[東歐|東歐]]7國的[[新興市場|新興市場]]就包括[[波士尼亞與赫塞哥維納|波士尼亞與赫塞哥維納]]。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622045&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2013年10月30日 |title = 貿易尖兵勇闖22國26城市 新興市場商機無窮 |publisher = 經貿透視網}}</ref>

2015年4月14日，[[中華民國外交部|外交部]]委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）組團參加「2015年波赫[[莫斯塔爾|莫斯塔爾]]國際商展」，該展是波赫規模最大商展。外貿協會是第4次組團參展，也是首度接受主辦單位邀請擔任夥伴國（Partner Country），[[中華民國國旗|中華民國國旗]]與[[國旗列表|其他參展國國旗]]同時出現。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=661898&url=/NewsSearch.asp |author = 中華民國對外貿易發展協會 |date = 2015年4月16日 |title = 2015年波赫Mostar國際商展隆重開幕臺灣館展現夥伴國氣勢 |publisher = 經貿透視網}}</ref>

2018年，波士尼亞與赫塞哥維納是中華民國第150大貿易夥伴、第131大[[進口|進口]]夥伴、第149大[[出口贸易|出口]]夥伴。出口至波赫的金額為399萬1,976[[美元|美元]]，年增47.919%；自波赫進口的金額為346萬5,463美元，年增40.649%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）52萬6,513美元，年增124.202%。2017年，波士尼亞與赫塞哥維納是中華民國第164大貿易夥伴、第141大進口夥伴、第156大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
