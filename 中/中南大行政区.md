'''中南行政区'''，简称'''中南大区'''、'''中南区'''，原名'''中原行政区'''，是[[中华人民共和国政府|中华人民共和国政府]]在1949年成立的一个[[大行政区|大行政区]]<ref>[http://www.gov.cn/test/2007-03/23/content_558707.htm 中华人民共和国行政区划（1949年）]</ref>，1954年6月19日撤销，所属[[中国一级行政区|省级行政区]]由中央直辖<ref>[http://www.gov.cn/test/2007-03/23/content_558796.htm 中华人民共和国行政区划（1954年）]</ref><ref>{{cite book |title = 中华法学大辞典: 宪法学卷 |year = 1995 |publisher = 中国检察出版社 |page = 123}}</ref>。

中南行政区本级设[[中共中央中南局|中共中央中南局]]、[[中南行政委员会|中南行政委员会]]、[[中南军区|中南军区]]三套班子<ref name="kuailelaoren" />，大区治所设于[[武汉|武汉]]。

== 历史 ==
1954年6月19日，中央人民政府决定撤销中南行政委员会，中南行政区所辖各省市改为中央直辖<ref>[http://www.gov.cn/test/2007-03/23/content_558796.htm 中华人民共和国行政区划（1954年）]</ref>。中南行政区机关原办公用楼改为[[中国共产党武汉市委员会|中共武汉市委]]办公楼。<ref name="changjiangribao">{{cite web | url=http://news.ifeng.com/gundong/detail_2011_06/11/6951975_0.shtml | title=当年的中共中央中南局所在地 | accessdate=2011-06-11}}}</ref>

首任中南局第一书记、中南军政委员会主席、中南军区司令员为[[林彪|林彪]]<ref name="kuailelaoren">{{cite web|url=http://www.laorenbao.net/html/2010-02/25/content_2264353.htm |title=新中国六大行政区设立与撤销始末 |accessdate=2011-11-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20121210000213/http://www.laorenbao.net/html/2010-02/25/content_2264353.htm |archivedate=2012-12-10 }}</ref>。

中南行政区（原中原行政区）本级机关演变情况如下：
*[[中共中央中原局|中共中央中原局]]{{to}}中共中央华中局{{to}}[[中共中央中南局|中共中央中南局]]
*[[中原临时人民政府|中原临时人民政府]]{{to}}[[中南军政委员会|中南军政委员会]]{{to}}[[中南行政委员会|中南行政委员会]]
*[[中原军区|中原军区]]、[[华中军区|华中军区]]{{to}}[[中南军区|中南军区]]

== 行政区划 ==
1954年6月建制撤销前，中南行政区管辖2个直辖市、6个省（括注省会）<ref>{{cite web|title=中华人民共和国行政区划（1952年）|url=http://www.gov.cn/test/2007-03/23/content_558770.htm|website=中央政府门户网站}}</ref>：
*直辖市：[[武汉市|武汉市]]、[[广州市|广州市]]
*省：[[河南省|河南省]]（[[开封市|开封市]]）、[[湖北省|湖北省]]（[[武汉市|武汉市]]）、[[湖南省|湖南省]]（[[长沙市|长沙市]]）、[[江西省|江西省]]（[[南昌市|南昌市]]）、[[广西省|广西省]]（[[南宁市|南宁市]]）、[[广东省|广东省]]（[[广州市|广州市]]）
