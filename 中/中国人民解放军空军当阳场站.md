'''中国人民解放军空军当阳场站'''，位于[[湖北省|湖北省]][[宜昌市|宜昌市]][[当阳市|当阳市]]，隶属中国人民解放军空军航空兵第十三师。

==沿革==
1937年至1938年间，[[国民政府|国民政府]]军队在铁路坝（今夷陵广场）修建军用机场用于飞行训练。1940年，[[日军|日军]]攻占宜昌，铁路坝机场沦为废墟。1941年，日军强征民夫万名，先后在宜昌县土门垭以及[[当阳县|当阳县]]修建机场，用于轰炸中国大后方。1945年8月[[日本投降|日本投降]]，机场关闭<ref name=bianqian>{{Cite web|url = http://www.ycjt.gov.cn/content-22511-281555-1.html|title = 宜昌交通之变迁|author = 黄传林|date = 2006-03-27|publisher = 宜昌市交通运输局|language = |accessdate = |quote = }}{{Dead link|date=2019年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[中华人民共和国|中华人民共和国]]成立后，中央军委决定在宜昌土门修建机场，1953年开工，1954年通航。随后又在当阳县修建军用机场<ref name=bianqian/>。曾设'''中国人民解放军空军当阳基地'''<ref>{{Cite web |url =http://yc.58xw.net/zwgk/2014-07-24/145669.html   |title = 这么远，那么近 |author =  |date =  2014-07-24|publisher = 宜昌新闻网 |language =  |accessdate =  |quote =  }}</ref>。后改为'''中国人民解放军空军当阳场站'''<ref>{{Cite web |url =  http://www.jingmen.gov.cn/zwgk/xwfbh/lcxwfbhsl/szfxwfbh/201609/t20160908_192293.shtml|title = 2016荆门爱飞客飞行大会新闻发布会 |author =  |date =2016-09-08  |publisher = 荆门市人民政府 |language =  |accessdate =  |quote =  }}</ref>。

空军当阳场站原隶属[[中国人民解放军广州军区空军|中国人民解放军广州军区空军]]空军航空兵第十三师，为该师第三十七团驻地。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，仍隶[[中国人民解放军中部战区空军|中国人民解放军中部战区空军]]空军航空兵第十三师。
