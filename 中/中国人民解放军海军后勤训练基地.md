'''中国人民解放军海军后勤训练基地'''，位于[[河北省|河北省]][[邯郸市|邯郸市]]，是[[中国人民解放军海军后勤部|中国人民解放军海军后勤部]]直属单位。

==沿革==
海军后勤训练基地的前身成立于1970年<ref name=zhanyou>战友!还记得邯郸、邢台的新兵连吗？，当代海军杂志社微信号，2016-09-29</ref>。2000年9月，在原海军后勤部邯郸办事处和海军后勤部教导大队的基础上，组建成立'''中国人民解放军海军后勤技术兵训练基地'''<ref name=shi>{{Cite web|url = http://www.hd.gov.cn/zwgk/dtxw/bddt/201207/t20120727_445651.shtml|title = 市领导参加军事日活动|author = |date = 2012-07-27|publisher = 邯郸市政府门户网站|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20120830042416/http://www.hd.gov.cn/zwgk/dtxw/bddt/201207/t20120727_445651.shtml|archive-date = 2012-08-30|dead-url = yes}}</ref><ref name=zhanyou/>。2005年3月，更名为'''中国人民解放军海军后勤训练基地'''，主要担负海军舰艇航空兵军需勤务、物资油料勤务、卫生勤务以及海军军港机场工程勤务等7大类35个专业的初级预选士官培训任务<ref name=shi/><ref name=zhanyou/><ref>{{Cite web|url = http://www.handannews.com.cn/zhengwu/c/2017-07/28/content_335276.htm?WebShieldDRSessionVerify=jNFiWOIcqWAjSuuEMBYA|title = 市领导慰问驻邯部队官兵|author = |date = 2017-07-28|publisher = 邯郸日报|language = |accessdate = |quote = }}{{Dead link|date=2019年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
