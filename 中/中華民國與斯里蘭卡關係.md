'''中華民國與斯里蘭卡關係'''是指[[中華民國|中華民國]]與[[斯里蘭卡民主社會主義共和國|斯里蘭卡民主社會主義共和國]]（1972年之前稱'''錫蘭'''）之間的關係。兩國於1945－1950年有領事級[[外交#外交上的承認|外交關係]]，終止關係後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對斯里蘭卡的相關事務由'''[[駐印度台北經濟文化中心|駐印度台北經濟文化中心]]－[[清奈|清奈]]辦事處'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-182-70658-03-1.html |title =  駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[斯里蘭卡|錫蘭]]。]]
{{see also|中華民國外交|{{link-en|斯里蘭卡外交|Foreign relations of Sri Lanka}}}}
1945年4月30日，中華民國向[[英屬錫蘭|時為英國殖民地的錫蘭]]派駐[[可倫坡|可倫坡]]領事。

1948年2月4日，錫蘭獨立，與中華民國繼續保持[[領事|領事]]關係。

1950年1月7日，錫蘭與[[中華人民共和國|中華人民共和國]]建立[[特命全權大使|大使級]][[外交#外交上的承認|外交關係]]。同日，與中華民國終止領事關係。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#电子签证|電子簽證]]的方式入境斯里蘭卡。]]
[[File:Visa_requirements_for_Sri_Lankan_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|斯里蘭卡簽證政策|{{link-en|斯里蘭卡公民簽證要求|Visa requirements for Sri Lankan citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以[[簽證#电子签证|電子簽證]]（Electronic Travel Authorization, ETA）的方式入境斯里蘭卡，停留最多30天。或在第1大城[[可倫坡|可倫坡]]的[[班達拉奈克國際機場|班達拉奈克國際機場]]辦理[[簽證#落地签证|落地簽證]]。入境斯里蘭卡後，可在入出境移民署申請簽證延期，停留最長不可超過6個月。經斯里蘭卡轉機亦須取得電子簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-182-70658-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ElectronicVisa.aspx?n=6F246645F10311D7&sms=B75E93A3DAB23A99 |title = 電子簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|斯里蘭卡護照|Sri Lankan passport}}的{{link-en|斯里蘭卡人口|Demographics of Sri Lanka|斯里蘭卡公民}}抵台參加國際會議、運動賽事、商展活動，也可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將斯里蘭卡歸類為'''灰色警示'''：提醒注意（2018年3月20日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-283-05ec6-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將斯里蘭卡歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2016年8月16日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=6A5A1AA61A9A6C29 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於斯里蘭卡第1大城[[可倫坡|可倫坡]]設立'''[[台北世界貿易中心|台北世界貿易中心]]辦事處'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#ASIA |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

2011年12月6日，外貿協會舉辦的「2011年[[印度|印度]]、[[孟加拉|孟加拉]]及[[斯里蘭卡|斯里蘭卡]]拓銷團」，於可倫坡進行貿易洽談會。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=574701&url=/NewsSearch.asp |author = 外貿協會 |date = 2011年12月7日 |title = 貿協率團赴可倫坡，外銷接單逾450萬美元 |publisher = 經貿透視網}}</ref>

2018年，斯里蘭卡是中華民國第54大貿易夥伴、第79大[[進口|進口]]夥伴、第44大[[出口贸易|出口]]夥伴。出口至斯里蘭卡的金額為4億0,531萬2,800美元，年減0.586%；自斯里蘭卡進口的金額為6,681萬5,447美元，年增15.496%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）3億3,849萬7,353美元，年減3.245%。2017年，斯里蘭卡是中華民國第55大貿易夥伴、第78大進口夥伴、第42大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

==醫療==
{{see also|苟亞博}}
1992年10月8日，斯里蘭卡的國際眼庫顧問、[[馬來西亞|馬來西亞]][[沙巴州|沙巴州]]眼庫創辦人魏偉傑訪問[[財團法人中華民國佛教慈濟慈善事業基金會|財團法人中華民國佛教慈濟慈善事業基金會]]。魏偉傑在斯里蘭卡創辦了眼角膜眼庫20多年，讓3萬多人重見光明，光是[[台灣|台灣]]就送了2,700多枚的眼角膜。訪問[[佛教慈濟醫療財團法人|慈濟醫院]]也是希望能成為[[東臺灣|台灣東部]]眼角膜捐贈移植的據點。因為在斯里蘭卡的[[佛教#佛弟子|佛教徒]]，認為[[布施|布施]]食物會得[[壽命|長壽]]，布施眼角膜會得[[智慧|智慧]]，因此，佛弟子[[轉世|往生]]後，大多願意將眼角膜捐出。魏偉傑在可倫坡市中心成立眼庫中心，只要有人捐贈，就由專人取下來，透過各聯絡網直接送到世界各地。<ref>{{Cite web |url = http://enquarterly.tzuchiculture.org.tw/monthly/312/312c4-6.htm |date = 1992年10月8日 |title = 斯里蘭卡國際眼庫顧問魏偉傑先生來訪 |publisher = 慈濟人文志業中心 慈濟月刊312期 }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

但也因為斯里蘭卡醫療資源有限，環境衛生也有待加強，不少進口的眼角膜，都容易產生[[感染|感染]]。曾經使用過斯里蘭卡眼角膜的臺灣[[中國醫藥大學|中國醫藥大學]]附設醫院眼科部主任表示，[[人眼|人體眼睛]]很脆弱，自我防衛機制差，「只要整個眼睛一感染[[細菌|病菌]]，再也沒有挽回空間。」至於為何容易感染，「可能是保存眼角膜的保養液也有問題。」<ref>{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20130830/35258429/ |date = 2013年8月30日 |title = 蘋果調查：台灣600病患 捐贈不到一半 |publisher = 臺灣[[蘋果日報_(台灣)|蘋果日報]]}}</ref>
