'''中華民國與西班牙關係'''是指[[中華民國|中華民國]]與[[西班牙王國|西班牙王國]]之間的關係。1913－1973年，兩國有官方[[外交#外交上的承認|外交關係]]但曾經中斷，二次斷交後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]一案棄權，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[西班牙|西班牙]]。]]
{{see also|中華民國外交|{{link-en|西班牙外交|Foreign relations of Spain}}}}
{{see also|中国驻西班牙大使列表{{!}}中華民國駐西班牙王國、西班牙共和國、西班牙國、西班牙王國外交官列表|西班牙驻华大使列表{{!}}西班牙駐中華民國外交官列表}}
1913年10月6日，西班牙[[外交#外交上的承認|外交承認]]中華民國[[北洋政府|北洋政府]]，並派駐[[特命全權公使|公使]]（皆為兼任或[[代辦|代辦]]使事）。

1928年12月27日，隨著[[國民革命軍北伐|國民革命軍北伐]]幾近尾聲，西班牙外交承認中華民國[[國民政府|國民政府]]。

1941年7月2日，因西班牙外交承認[[滿洲國|滿洲國]]和[[汪精衛政權|汪精衛政權]]，中華民國與之斷交。

1952年6月27日，恢復邦交並升格為大使級外交關係。於首都[[馬德里|馬德里]]設立'''中華民國駐西班牙國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。由於當時西班牙為[[反共主义|反共]]的[[佛朗哥時期|佛朗哥政權]]一黨執政，故仍與中華民國保持外交關係，但早期由{{link-es|西班牙駐菲律賓大使列表|Anexo:Embajadores de España en Filipinas|駐菲律賓大使}}兼任駐華大使，而駐華大使館僅派秘書、參事擔任[[代辦|代辦]]。1960年才任命長期擔任駐華代辦的[[胡國材|胡國材]]為駐華大使。<ref>[http://catalog.digitalarchives.tw/item/00/31/99/36.html 西班牙駐華代辦酒會]，1954-03-27，典藏台灣</ref>

1971年10月25日，西班牙在[[聯合國大會|聯合國大會]]上就[[中國與聯合國#代表權問題|中國代表權]]問題棄權，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》（其他棄權的[[歐洲國家和地區列表|歐洲國家]]為[[希臘|希臘]]、[[賽普勒斯|賽普勒斯]]與[[盧森堡|盧森堡]]）。

1973年3月10日，與中華民國斷交。4月9日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

1979年4月15日，總統[[蔣經國|蔣經國]]接受西班牙《[[雅報|雅報]]》記者柯契洛訪談表示，中共所謂「[[國共合作|國共合作]]」，就是要以「國共合作」來達到其消滅國民黨的陰謀。

2019年1月2日，對於[[中國共產黨中央委員會總書記|中國共產黨中央委員會總書記]][[習近平|習近平]]在《[[告台灣同胞書|告台灣同胞書]]》發表40周年紀念會的[[習五條|有關談話]]，被中華民國總統[[蔡英文|蔡英文]]拒絕，並在[[Facebook|Facebook]]分享「[[台灣共識|台灣共識]]」各國語言版本。5日，追求[[2017－2018年西班牙憲政危機|從西班牙獨立]]的加泰隆尼亞政府前主席[[卡萊斯·普吉德蒙|普伊格蒙特]]，在[[Twitter|Twitter]]轉貼蔡英文的貼文，並以[[正體中文|正體中文]]表示，「對台灣總統蔡英文女士在她發表的聲明裡使用[[加泰隆尼亞語|加泰隆尼亞語]]，我深表感謝。我們支持台灣爭取自由和自決權。」<ref>{{Cite web |url = https://www.nownews.com/news/20190107/3159798/ |author = 綜合報導 |date = 2019年1月7日 |title = 加泰隆尼亞主席同挺台灣 支持蔡英文總統台灣共識主張 |publisher = [[NOWnews今日新聞|NOWnews今日新聞]]}}</ref>

==經濟==
{{see also|臺灣經濟|西班牙經濟}}
===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於第2大城[[巴塞隆納|巴塞隆納]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#EUROPE |title = 外貿協會全球據點 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]則於首都兼第1大城[[馬德里|馬德里]]設立'''駐西班牙代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2017年，西班牙是中華民國第31大貿易夥伴、第37大[[進口|進口]]夥伴、第27大[[出口贸易|出口]]夥伴。出口至西班牙的金額為10億4,634萬0,069[[美元|美元]]，年增19.390%；自西班牙進口的金額為6億9,910萬3,186美元，年增4.313%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）3億4,723萬6,883美元，年增68.393%。※2016年，西班牙是中華民國第30大貿易夥伴、第36大進口夥伴、第27大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

===投資===
[[台灣商人|台商]]在西班牙投資以[[国际贸易|貿易]]及[[批發|批發]]（[[電腦|電腦]]及周邊設備、[[通訊|通訊]]及[[电脑网络|網路設備]]、[[家用電器|家用電器]]、[[食物|食品]]、[[機車_(公路)|機車]]、[[健身房|運動器材]]、[[腳踏車|腳踏車]]）、[[相片|照片]][[沖洗|沖洗]]、[[禮物|禮品]][[零售|零售]]、[[食品產業|餐飲]]等為主，多屬中小型投資。台灣[[上市公司|上市]][[柜台买卖|上櫃公司]]如[[宏碁|宏碁]]、[[華碩電腦|華碩電腦]]、[[美利達工業|美利達工業]]、[[長榮海運|長榮海運]]、[[微星科技|微星科技]]、[[友訊科技|友訊科技]]、[[圓剛科技|圓剛科技]]、[[訊舟科技|訊舟科技]]、[[智邦科技|智邦科技]]等亦在西班牙設有[[子公司|子公司]]或[[分公司|分公司]]，並處理[[葡萄牙|葡萄牙]]市場開發業務。<ref name="投資環境簡介"/>

目前已成立西班牙台灣商會（1997年5月10日成立）、西班牙台灣青年商會（2010年3月27日成立）。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/es/pdf/104-56Spain.pdf |date = 2015年8月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

===會議===
[[中華民國全國商業總會|中華民國全國商業總會]]與西班牙全國工商航運總會（[[西班牙文|西班牙文]]：{{lang|es|Consejo Superior de Cámaras de Comercio, Industria y Navegación de España}}）於1982年成立「中西、西中經濟商務合作委員會」（[[英文|英文]]：{{lang|en|R.O.C.-Spain & Spain-R.O.C. Economic and Business Cooperation Committee）}}，其後每年輪流在臺北及馬德里舉辦聯席會議，研商促進雙邊經貿合作事宜。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-4.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>但1995年第13屆聯席會議召開後，即因會議成效不彰而停辦。2000年恢復舉辦第14屆聯席會議，台灣主辦單位改為中歐貿易促進會。<ref name="89年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2000/mofa_8.htm |title = 《中華民國89年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2000年1月，為增進雙邊實質交流，兩國自1973年斷交後首度舉辦官方經貿諮商會議（總司長級），首屆在馬德里舉行，並於2001年7月在台北舉辦第2屆官方經貿諮商會議。<ref name="90年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2001/Chap3/3-1-4.htm |title = 《中華民國90年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

==交流==
===僑民===
{{see also|台灣裔{{!}}海外臺僑|西班牙華人}}
西班牙境內來自中華民國台灣的僑民人數約1,500人，使用語言主要為[[中華民國國語|國語]]及[[台灣話|台語]]。主要僑團有台灣協會、台灣商會、[[中華民國婦女聯合會|中華民國婦女聯合會]]西班牙分會、巴塞隆納台灣[[同鄉|同鄉]]聯誼會等，並設有兩所僑校（[[馬德里|馬德里]]、[[巴塞隆納|巴塞隆納]]）。旅居馬德里及巴塞隆納的僑民主要從事[[国际贸易|貿易業]]、[[資訊業|電腦業]]、[[電子產品|電子]]及[[腳踏車|腳踏車]]零件進出口、[[旅遊|旅行業]]等，其他城市的僑民仍以從事[[餐飲業|餐飲業]]為主。<ref name="生活資訊">{{Cite web |url = http://www.boca.gov.tw/content.asp?CuItem=512&BaseDSD=13&CtUnit=16&mp=1 |title = 生活資訊 |publisher = 外交部領事事務局}}</ref>

===教育===
{{see also|台灣教育|西班牙教育}}
自1960年代即有台灣學生赴西班牙留學，主要進修[[西班牙語|語言學]]、[[西班牙文學|文學]]、{{Link-en|西班牙藝術|Spanish art|美術}}、{{Link-en|西班牙音樂|Music of Spain|音樂}}、{{Link-en|西班牙建築|Spanish architecture|建築}}、工商業設計等，亦有少數從事{{Link-en|西班牙醫療保健|Health care in Spain|醫學}}、[[西班牙歷史|歷史]]、{{Link-en|西班牙媒體|Media of Spain|傳播}}、企業管理、[[西班牙觀光|觀光旅遊]]、[[西班牙政治|政治科學]]等方面的研究；另有短期進修或[[遊學|遊學]]，進修項目包括[[佛朗明哥|佛朗明哥舞蹈]]、[[古典吉他|西班牙吉他]]等。目前在西班牙的台灣學生以短期遊學居多，攻讀正式[[學位|學位]]的較少。<ref name="96年外交年鑑"/>近年來，台灣學生赴西班牙留學未見增多，主因是兩國教育體制及學歷認證制度不同，形成留學生申請學校的障礙。<ref name="93年外交年鑑"/>

兩國學術交流仍以台灣學生赴西班牙的單向為主。台灣的[[輔仁大學|輔仁大學]]、[[淡江大學|淡江大學]]、[[靜宜大學|靜宜大學]]與[[西班牙大学列表|西班牙姊妹校大學]]簽署大三學生赴西班牙進修1年的計畫；[[文藻外語大學|文藻外語大學]]則利用暑期安排學生至西班牙合作大學從事語文進修2個月。這4所大學皆設有西班牙語文相關系所。<ref name="96年外交年鑑"/>

台灣則分別提供[[中華民國教育部|教育部]]的「[[中華民國國語|華語文]]獎學金」，以及[[中華民國外交部|外交部]]的「台灣獎學金」，給予西班牙學生赴台進修學位或語文。<ref name="96年外交年鑑"/>

截至2017年，在西班牙的台灣學生約300餘名；在台灣的西班牙學生則約260名。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>

===文化===
{{see also|台灣文化|西班牙文化}}
2015年1月，設立'''駐西班牙代表處文化組'''，並就[[電影|電影]]、[[視覺藝術|視覺藝術]]、[[表演藝術|表演藝術]]、[[音樂|音樂]]及[[舞蹈|舞蹈]]、[[陶藝|陶藝]]等方面，與{{link-en|西班牙國家電影資料館|Filmoteca Española}}、{{link-en|西班牙國立陶瓷博物館|González Martí National Museum of Ceramics and Decorative Arts|國立陶瓷博物館}}及國際偶戲藝術節等文化機構及藝術展演單位合作，推動一系列文化交流活動。<ref name="104年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>7月9日，台灣[[鶯歌陶瓷博物館|鶯歌陶瓷博物館]]與西班牙國立陶瓷博物館締結為姐妹館，並簽署文化交流協議。<ref name="姐妹館">{{Cite web |url = http://news.ltn.com.tw/news/life/breakingnews/1375744 |author = 翁聿煌 |date = 2015年7月10日 |title = 台灣與西班牙兩大陶博館 締結姐妹館 |publisher = [[自由時報|自由時報]]}}</ref>

2017年，與{{link-en|西班牙國立裝飾藝術博物館|Museo Nacional de Artes Decorativas}}合作，展出[[國立歷史博物館|國立歷史博物館]]的[[刺繡|刺繡]]館藏；{{Link-en|駐西班牙台北經濟文化辦事處|Taipei Economic and Cultural Office in Spain}}連續4年推動建立臺灣文化品牌計畫，與{{link-en|圓環藝術中心|Círculo de Bellas Artes}}共同舉辦「臺灣製造－臺灣藝術節」（Made in Taiwan-Taiwan en el Círculo）活動。<ref name="106年外交年鑑"/>

===犯罪事件===
2016年12月，{{link-en|西班牙警察|National Police Corps|西班牙警方}}在[[馬德里|馬德里]]、[[巴塞隆納|巴塞隆納]]、[[阿利坎特|阿利坎特]]等地同時破獲[[台湾诈骗集团|電信詐騙集團]]機房，[[中華民國外交部|中華民國外交部]]證實，經馬德里警方告知{{Link-en|駐西班牙台北經濟文化辦事處|Taipei Economic and Cultural Office in Spain|駐西班牙台北代表處}}，於馬德里涉案的[[台灣人|台灣人]]有47人。<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20161220/1016372/ |author = 陳培煌 |date = 2016年12月20日 |title = 西班牙詐騙案 外交部：確認47台人涉案 |publisher = [[蘋果日報_(台灣)|台灣蘋果日報]]}}</ref>2017年2月17日，{{link-en|西班牙內閣部長會議|Council of Ministers (Spain)}}同意[[中華人民共和國|中華人民共和國]]（以下簡稱對岸）對涉及[[電信詐騙|電信詐騙]]案的269名嫌犯，包括218名台灣人[[引渡|引渡]]至對岸的請求。對此，中華民國外交部再度召見西班牙駐台代表，要求西班牙在完成司法程序後，依據「國籍管轄原則」將涉案台灣人遣送回台。並對於[[西班牙政府|西班牙政府]]的不友善決定，[[中華民國政府|中華民國政府]]將檢討現行的雙邊交流合作，以反映嚴重關切。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/paper/1079937 |author = 呂伊萱 |date = 2017年2月21日 |title = 西班牙詐騙案》我再召西代表要求遣返218台嫌 |publisher = 自由時報}}</ref>12月15日，{{Link-en|西班牙司法制度|Judiciary of Spain|西班牙法院}}同意將121名嫌犯引渡至對岸，23日再度同意引渡93人到對岸。辯護律師對將人引渡至對岸提出抗議，但被西班牙法官以「[[中国台湾|台灣是中國的一部份]]」為由駁回。<ref>{{Cite web |url = https://hk.news.appledaily.com/china/realtime/article/20171223/57618891 |author = 海外網 |date = 2017年12月23日 |title =【涉中港詐騙案】西班牙再遣返93台犯到大陸 法官：台灣係中國一部份 |publisher = [[蘋果日報_(香港)|香港蘋果日報]]}}</ref>

*[[台南機場|台南]]→香港→西班牙。
