'''中華民國與尚比亞關係'''是指[[中華民國|中華民國]]與[[尚比亞共和國|尚比亞共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對尚比亞的相關事務由'''{{Link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-163-5317f-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|尚比亞簽證政策|Visa policy of Zambia}}|{{link-en|尚比亞公民簽證要求|Visa requirements for Zambian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]與說明函或邀請函的[[中華民國國民|中華民國公民]]須至尚比亞[[簽證#电子签证|電子簽證]]網站申請「簽證同意函」（Visa Approved Letter），獲准後以[[電子郵件|電子郵件]]寄發，於抵達{{link-en|尚比亞機場列表|List of airports in Zambia|該國指定機場}}時繳交簽證費並領證。觀光簽證停留最多90天；商務簽證停留最多30天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-163-5317f-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ElectronicVisa.aspx?n=6F246645F10311D7&AreaSN=D33B55D537402BAA |title = 電子簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|尚比亞護照|Zambian passport}}的{{link-en|尚比亞人口|Demographics of Zambia|尚比亞公民}}抵台參加國際會議、運動賽事、商展活動，也可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將尚比亞歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2018年11月1日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-139-16c0b-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|尚比亞經濟|Economy of Zambia}}}}
===貿易===
2017年，尚比亞是中華民國第94大貿易夥伴、第68大[[進口|進口]]夥伴、第162大[[出口贸易|出口]]夥伴。出口至尚比亞的金額為182萬9,526[[美元|美元]]，年增13.055%；自尚比亞進口的金額為8,873萬4,571美元，年增0.363%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）8,690萬5,045美元，年增0.126%。※2016年，尚比亞是中華民國第90大貿易夥伴、第61大進口夥伴、第164大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
