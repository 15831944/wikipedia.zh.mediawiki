'''中華民國與列支敦斯登關係'''是指[[中華民國|中華民國]]與[[列支敦斯登公國|列支敦斯登公國]]（又稱'''列支敦斯登[[親王國|親王國]]'''）之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對列支敦斯登的相關事務由'''駐[[瑞士|瑞士]][[台北经济文化代表处|台北文化經濟代表團]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-154-17ab3-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以「[[歐洲聯盟簽證政策|申根區免簽證]]」的方式入境列支敦斯登。]]
[[File:Visa_requirements_for_Liechtenstein_citizens.png|thumb]]的{{link-en|列支敦斯登人口|Demographics of Liechtenstein|列支敦斯登公民}}可以[[簽證#签证优惠待遇|免簽證]]的方式入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|歐洲聯盟簽證政策|{{link-en|列支敦斯登公民簽證要求|Visa requirements for Liechtenstein citizens}}}}
列支敦斯登尚未加入[[歐洲聯盟|歐洲聯盟]]，但有簽署《[[申根公約|申根公約]]》，因此實行與[[申根區|申根區]]相同的簽證政策。歐洲申根區給予持有載明[[中華民國國民身份證|身份證]]字號的[[中華民國護照|中華民國護照]]之[[中華民國國民|中華民國公民]]可以[[歐洲聯盟簽證政策|申根區免簽證]]入境，停留日數與申根區合併計算，每6個月期間內總計可停留90天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-154-17ab3-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/News_FAQ.aspx?c=AD6908DFDDB62656&n=C1B19EF0ACDD3C91&sms=867F5398A49F758A |title = 歐盟免申根簽證常見問答集 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&AreaSN=FB01D469347C76A7 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持[[列支敦斯登護照|列支敦斯登護照]]的{{link-en|列支敦斯登人口|Demographics of Liechtenstein|列支敦斯登公民}}則可用[[歐洲自由貿易聯盟|歐洲自由貿易聯盟]]的身分以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將列支敦斯登歸類為'''灰色警示'''：提醒注意（2016年10月28日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-78-10178-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==

===貿易===
2017年，列支敦斯登是中華民國第134大貿易夥伴、第111大[[進口|進口]]夥伴、第158大[[出口贸易|出口]]夥伴。出口至列支敦斯登的金額為247萬5,230[[美元|美元]]，年增622.838%；自列支敦斯登進口的金額為1,293萬1,202美元，年增18.419%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）1,045萬5,972美元，年減1.148%。※2016年，列支敦斯登是中華民國第138大貿易夥伴、第109大進口夥伴、第197大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
