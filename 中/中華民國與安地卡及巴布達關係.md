'''中華民國與安地卡及巴布達關係'''是指[[中華民國|中華民國]]與[[安地卡及巴布達|安地卡及巴布達]]（通稱'''安巴'''、'''安地卡'''）之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構，對安巴的相關事務由'''中華民國駐[[聖克里斯多福及尼維斯|聖克里斯多福及尼維斯]][[中華民國駐外機構列表|大使館]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-216-16335-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
[[File:Voting_res_2758.png|thumb]]，[[安地卡及巴布達|安地卡及巴布達]]當時尚未獨立，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#电子签证|電子簽證]]的方式入境安地卡及巴布達。]]
[[File:Visa_requirements_for_Antigua_and_Barbuda_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|安地卡及巴布達簽證政策|{{link-en|安地卡及巴布達公民簽證要求|Visa requirements for Antigua and Barbuda citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可至安地卡及巴布達移民局網站填寫申請表並以[[信用卡|信用卡]]線上付款後，將單次入境觀光[[簽證#电子签证|電子簽證]]以[[電郵|電郵]]至申請人。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-216-16335-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ElectronicVisa.aspx?n=6F246645F10311D7&AreaSN=ABBF62618F53F8DE |title = 電子簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|安地卡及巴布達護照|Antigua and Barbuda passport}}的{{link-en|安地卡及巴布達人口|Demographics of Antigua and Barbuda|安巴公民}}抵台參加國際會議、運動賽事、商展活動，也可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國衛生福利部|中華民國衛生福利部]]的國際間旅遊疫情建議等級表將安地卡及巴布達歸類為'''第二級警示'''：對當地採取加強防護（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=F2BFEB3C42A60183 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
2018年，安地卡及巴布達是中華民國第202大貿易夥伴、第178大[[進口|進口]]夥伴、第198大[[出口贸易|出口]]夥伴，在[[美洲|美洲]]國家排名倒數第3、3、2；在[[世界政區索引|世界經濟體]]倒數第43、63、39。出口至安巴的金額為37萬9,499[[美元|美元]]，年增263.588%；自安巴進口的金額為22萬9,915美元，年增50.253%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）14萬9,584美元，年減407.514%。2017年，安地卡及巴布達是中華民國第208大貿易夥伴、第185大進口夥伴、第211大出口夥伴，在美洲國家排名倒數第1、5、1；在世界經濟體倒數第31、52、20。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
