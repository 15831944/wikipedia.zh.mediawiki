== 媒体管制 ==
虽然一般认为[[中国共产党|中国共产党]]领导下的[[中华人民共和国政府|中华人民共和国政府]]牢牢[[中华人民共和国言论审查|掌控中国的主流媒体]]，但中国官方对保守的伊斯兰服饰可能呈现相反的状态。

2015年，在录制电视节目《[[花兒與少年|花儿与少年]]》时，女演员[[宁静|宁静]]虽非[[穆斯林|穆斯林]]，由於攝製節目的地方在土耳其，而且在清真寺內[http://ent.ifeng.com/a/20150528/42412708_0.shtml]，為了給予尊重，攝製節目時穿著以示尊重的頭巾(希賈布)，卻被說成是全身罩蓋的「[[吉里巴甫服|吉里巴甫服]]」。[[湖南卫视|湖南卫视]]在播出节目时，对涉及宁静的电视画面采取消音，直至完全删除<ref>{{Cite web|url=http://www.vdfly.com/star/20150531/44883.html|title=《花儿与少年》宁静镜头被删除 原因竟然是这样的|accessdate=2016-04-23|author=|date=2015-05-31|work=|publisher=青春娱乐网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20151029044045/http://www.vdfly.com/star/20150531/44883.html|archivedate=2015-10-29}}</ref>。同年，由中国共产党临夏州委宣传部、临夏州民族事务委员会等单位联合<ref name="a2" />为[[临夏回族自治州|临夏回族自治州]]建州60周年拍摄的[[献礼电影|献礼电影]]《[[情定河州|情定河州]]》中，女演员身穿[[里切克|里切克]]<ref>{{Cite web |url = http://ent.163.com/15/1215/10/BASD25JH000300B1.html |title=电影《情定河州》顺利杀青 吴佳尼造型照曝光 |accessdate=2016-04-23 |author= |date=2015-12-15 |work= |publisher=网易 |language = zh-hans }}</ref>{{NoteTag|新闻报道原文将[[里切克|里切克]]称为少数民族服饰，当指[[回族|回族]]传统民族服饰。}}。2016年4月[[北京国际电影节|北京国际电影节]]中，剧中女演员[[阎青妤|阎青妤]]、[[霍秋慧|霍秋慧]]穿着里切克为电影宣传<ref name="a2">{{Cite web|url=http://www.nxnews.net/hz/system/2016/04/19/011197075.shtml|title=王姬率《情定河州》亮相北京国际电影节 服饰刮起民族风|accessdate=2016-04-23|editor=责任编辑：杨春晖|date=2016-04-20|work=|publisher=宁夏新闻网，来源：中穆网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160505123931/http://www.nxnews.net/hz/system/2016/04/19/011197075.shtml|archivedate=2016-05-05}}</ref>。虽然阎青妤与宁静同样非[[穆斯林|穆斯林]]，但官方并未阻止她的这种行为。2017年3月，甘肃省全国人大代表[[白忠华|白忠华]]佩戴[[希贾布|希贾布]]出席全国人大全体会议，也未被阻止。<ref>[https://www.letscorp.net/archives/116607 敌在主席台？全国“两会”惊现新疆警方明令禁止的宗教极端服饰]</ref>
