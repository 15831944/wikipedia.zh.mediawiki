'''中国人民解放军第二一〇医院'''，主院区位于[[辽宁省|辽宁省]][[大连市|大连市]][[西岗区|西岗区]]胜利路80号，隶属[[沈阳联勤保障中心|沈阳联勤保障中心]]，为[[三级甲等医院|三级甲等医院]]。

==简介==
1945年11月，组建'''吉辽军区后方医院'''。此后名称经多次变化，1954年改称'''中国人民解放军第二一〇医院'''。1964年6月迁至胜利路现址。2005年1月，原中国人民解放军第四〇三医院整编为第二一〇医院临床部<ref>李泽、栾振涛、于润泽，紧贴实战 锐意创新——解放军第210医院装备技术革新纪实，医疗卫生装备2011年12期</ref>。医院先后被评为“[[三级甲等医院|三级甲等医院]]”、“全国百佳医院”、“全国爱婴医院”等<ref name=mod>{{Cite web |url =http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536713.htm   |title = 第二一〇医院 |author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

截至2014年，医院设有42个科室、61个专业，展开床位1050张；拥有6个军区级以上重点学科。中西医结合治疗[[白血病|白血病]]达到[[中华人民共和国|中华人民共和国]]国内领先水平；消化内镜诊断、心脏病介入治疗、人工关节置换、超声诊断等技术在[[沈阳军区|沈阳军区]]及驻地优势较强<ref name=mod/>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第二一〇医院由[[中国人民解放军沈阳军区|中国人民解放军沈阳军区]]转隶[[沈阳联勤保障中心|沈阳联勤保障中心]]<ref>{{Cite web |url =http://www.81.cn/zghjy/2016-12/20/content_7415799.htm |title = 第210医院：向部队官兵学习，做为兵服务标兵 |author =  |date = 2016-12-20 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。
