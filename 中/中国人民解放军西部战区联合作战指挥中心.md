'''中国人民解放军西部战区联合作战指挥中心'''，位于[[四川省|四川省]][[成都市|成都市]]，隶属[[中国人民解放军西部战区联合参谋部|中国人民解放军西部战区联合参谋部]]，是[[中国人民解放军西部战区|中国人民解放军西部战区]]的联合作战指挥机构。

==简介==
2015年11月27日，[[中华人民共和国国防部|中华人民共和国国防部]]新闻发言人[[杨宇军|杨宇军]]就[[深化国防和军队改革|深化国防和军队改革]]在健全联合作战指挥体制上有何举措时称，“主要是适应打赢信息化战争、有效履行使命任务的要求，组建战区联合作战指挥机构，健全军委联合作战指挥机构，构建完善精干高效的战略战役指挥体系。”<ref>{{Cite web |url =http://military.china.com.cn/2015-11/27/content_37177664.htm  |title =国防部谈军队改革：组建战区联合作战指挥机构  |author =  |date =2015-11-27  |publisher =  中国网 |language =  |accessdate =  |quote =  }}</ref>2016年2月1日，中华人民共和国国防部新闻局局长、国防部新闻发言人杨宇军在国防部专题新闻发布会上表示，战区是“本战略方向的唯一最高联合作战指挥机构”<ref>{{cite web |url = http://news.163.com/16/0202/06/BEQ5BF2500014AED.html |title = 战区是唯一最高联合作战指挥机构 |author =  |date =  2016-02-02 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2016年2月前后，新成立的五个战区分别设立战区联合作战指挥中心。战区联合作战指挥中心是在此前各[[大军区|大军区]]作战指挥中心的基础上建成，增加了空军、火箭军等军兵种，增设了测绘导航、空域管理等作战要素，调整了指挥席位的设置。其中，西部战区联合作战指挥中心设置了联合作战计划、战场预警、综合态势、数据资源、领航引导、国防动员、装备保障协调等数十个指挥席位<ref>{{Cite web |url = http://mil.huanqiu.com/china/2016-02/8623038.html |title =西部战区砍原军区数十个指挥席 30分钟发7方案  |author =  |date = 2016-02-29 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref><ref>{{cite web|url = http://news.hexun.com/2016-04-21/183441426.html |title =军委联指中心座位分布复杂 挂两巨幅地图 |author =|date =2016-04-21 |publisher =和讯|language =|accessdate =|quote =}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2016年2月，西部战区联合作战指挥中心正式投入运行<ref>{{cite web |url = http://www.81.cn/jmywyl/2016-03/03/content_6938737.htm |title = 西部战区联合作战指挥中心投入运行 |author =  |date = 2016-03-03 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。
