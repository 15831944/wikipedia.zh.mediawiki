'''中國染廠大廈'''（英文：'''CDW Building'''），簡稱'''中染大廈'''，是一座位於[[香港|香港]][[荃灣區|荃灣區]][[青山公路|青山公路]]388號的大廈，1982年落成，樓高27層，總樓面面積982,777平方呎，由[[香港興業|香港興業]]名下的前香港上市公司中國染廠集團有限公司擁有。大廈原為工廠大廈，由於1980年代開始工業北移，中國染廠於2000年代初期把大廈改為辦公大樓，其後進行多次翻新。目前大廈低層為商場「[[8咪半|8咪半]]」，现設有最大租户[[宜家傢居|宜家傢居]]。<ref>{{Cite news|url=http://202.76.19.41/HKRI/Interim%20Report%202017-18/|title=Interim Report 2017/18|accessdate=2018-04-06}}</ref>現由[[香港興業|香港興業國際集團]]有限公司持有。

== 商場 ==
中染大廈於2004年完成翻新工程，並且把地下及一樓原用的工業樓面改為總樓面面積達8萬多呎的中染新城<ref>{{cite web|url=http://www.centaline-cis.com/Files/Industrial/Press%20Conf/CDW_PROJECT/CDW.pdf|title=荃灣工廈展現全新面貌　工商樓面翻新增值，「中染新城」矚目登場|last=  |first=  |coauthors=|date=2004-08-26|publisher= 中原(工商舖)網站  |accessdate=2011-08-20}}</ref>，商舖包括食肆、診所、便利店、[[自動櫃員機|自動櫃員機]]及[[補習班|補習學校]]等，而[[薩莉亞|薩莉亞]]香港首家分店亦在中染新城開業。

自2014年起，商場再進行翻新，亦將部分寫字樓樓面轉為商場部分，商場總面積大增至27萬平方呎，樓高8層，於2016年分期落成。翻新後商場改名為「8咪半」<ref>{{cite web| url=http://www.hkri.com/icms2/template?series=56&article=10722| title=香港興業荃灣全新商場命名為「8咪半」 商場租務理想 獲主要租戶預租| publisher=[[香港興業國際集團有限公司|香港興業國際集團有限公司]]| date=2016-01-18}}</ref>。

「8咪半」的名稱源於青山公路1910年代開路初期，是以「咪」（英文“mile”的中譯）標示該處與尖沙咀碼頭的距離，而中染大廈現址為荃灣青山公路388號，剛好在距離九龍起點8咪及9咪之間，所以稱為「8咪半」。<ref>{{Cite web|url=https://www.eightandahalf.com.hk/en/About-8-half/8-half?sc_lang=zh-tw|title=「8咪半」|accessdate=2018-04-06|work=www.eightandahalf.com.hk|language=zh-hk}}</ref>

2017年，[[宜家家居|宜家家居]]在此商場開設香港第4間分店，樓高4層，佔地90,000平方呎，在10月11日開業，也是[[宜家家居|宜家家居]]重新進駐荃灣地區（[[宜家家居|宜家家居]]曾在[[荃灣廣場|荃灣廣場]]開設分店）。<ref>{{Cite news|url=https://www.weekendhk.com/lifestyle/ikea-%E5%AE%9C%E5%AE%B6%E5%AE%B6%E5%B1%85-8%E5%92%AA%E5%8D%8A-%E8%8D%83%E7%81%A3/|title=期間限定！IKEA pop-up體驗站～為荃灣新分店打響頭炮 {{!}} 生活 {{!}} 新假期|last=新假期|date=2017-05-29|newspaper=新假期|accessdate=2018-04-06|language=en-US}}</ref>

現時其他主要商戶包括[[Market_Place_by_Jasons|Market Place by Jasons]]、[[恬愉堂中醫中心|恬愉堂中醫中心]]、[[薩莉亞|薩莉亞意式餐廳]]、[[Pacific_Coffee|Pacific Coffee]]、同仁四季火鍋店、7樓[[海港酒家|海港酒家]]和8樓[[舒適堡|舒適堡]]等。

== 租戶 ==
中國染廠大廈前身'''中國染廠'''，其他租戶包括[[Sunny_Idea|意晴]]（Sunny Idea，16樓）。

== 交通 ==
中染大廈提供185個車位，並可經有蓋空調行人天橋連接港鐵荃灣站，同時鄰近荃灣巴士總站和過境巴士站。
