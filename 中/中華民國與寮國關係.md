'''中華民國與寮國關係'''是指[[中華民國|中華民國]]與[[寮人民民主共和國|寮人民民主共和國]]之間的關係。寮國於1958－1962年，是[[中華民國外交#中华民国的邦交国|中華民國的邦交國]]，[[外交#外交上的承認|斷交]]後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構，對寮國的相關事務由'''{{link-en|駐越南台北經濟文化辦事處|Taipei Economic and Cultural Office, Hanoi, Vietnam}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}<ref>[https://www.boca.gov.tw/sp-foof-countrycp-03-117-2e720-03-1.html 中華民國外交部 領事事務局：寮國]</ref><ref>[http://www.roc-taiwan.org/VN/mp.asp?mp=227 駐越南台北經濟文化辦事處]</ref>。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[寮王國|寮王國]]。]]
{{see also|中華民國外交|{{link-en|寮國外交|Foreign relations of Laos}}}}
{{see also|中国驻老挝大使列表{{!}}中華民國駐寮國外交官列表|老挝驻华大使列表{{!}}寮國駐中華民國外交官列表}}
1958年11月17日，中华民国与老撾建立领事級[[外交#外交上的承認|外交关系]]。

1959年1月5日，中华民国驻[[永珍|永珍]][[领事馆|领事馆]]設立，並派駐[[領事|領事]]。

1960年[[聖誕節|聖誕節]]，[[民航空運公司|民航空運公司]]有一班至[[緬甸|緬甸]]北部的[[包機|包機]]轉包給[[中華航空|華航]]，空軍情報署長[[衣復恩|衣復恩]]請這班飛機於回程時去寮王國永珍探一探陸路交通不便的寮王國的航空交通市場。執行任務的任歌樵在當地接了幾筆包機生意，三天內帶回三萬多[[美元|美元]]<ref>林玉萍 臺灣航空工業史——戰爭羽翼下的1935年~1979年 新銳文創(秀威代理)ISBN：9789868681545</ref>。1961年1月1日，駐寮空軍[[駐外武官|武官]]林乃順拍[[電報|電報]]給衣復恩報告華航取得兩條航線的合同。衣復恩立即告訴楊道古，並派烏鉞率領兩架[[C-47運輸機|C-47]]飛機，23名空地勤人員進駐永珍，任務代號是「北辰小組」。除了將飛機與機組人員租給飛霞航空公司（Cie Veha Akat）提供寮王國境內的空中交通之外，也替[[美國|美國]][[中央情報局|中情局]]運送[[特種部隊|特種部隊]]。1962年，{{link-en|寮國皇家航空公司|Royal Air Lao}}（Royal Air Lao）也開始向華航租機組人員。這些生意一直持續到1975年寮王國滅亡。

1962年2月9日，领事馆升格为总领事馆。

1962年5月16日，老挝政府親美的[[文翁亲王|文翁亲王]]在台湾宣布与中华民国建立大使級外交關係，並設立[[外交代表機構#大使馆|大使館]]、派駐[[特命全權大使|大使]]，但是遭到了政府内部[[梭发那·富马亲王|梭发那·富马亲王]]的抵制。

1962年9月2日，老挝联合政府宣布承认[[中华人民共和国|中华人民共和国]]。9月7日，与中华人民共和国建交，10日，與中華民國断交，兩國的大使級外交關係僅維持三個多月，而後有關老挝事務，由'''{{link-en|駐越南台北經濟文化辦事處|Taipei Economic and Cultural Office, Hanoi, Vietnam}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。

1971年10月25日，寮國投票反對[[中國與聯合國#代表權問題|中華民國續留聯合國]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。

1994年10月7日，簽署《[[中華民國交通部郵政總局|中華民國台灣郵政總局]]與寮國郵政總局間關於國際[[特快專遞|快捷郵件]]瞭解[[備忘錄|備忘錄]]》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=%e5%af%ae%e5%9c%8b&tysubject_e=%e5%af%ae%e5%9c%8b&tysubject_o=%e5%af%ae%e5%9c%8b&tycountry_c=&tycountry_e=&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>

[[File:Visa_requirements_for_Laotian_citizens.png|thumb]]必須申請[[簽證|簽證]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|寮國簽證政策|寮國公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可至{{link-en|寮國駐外機構列表|List of diplomatic missions of Laos|寮國駐外館處}}申請入境簽證，或至[[老挝机场列表|寮國機場]]的[[海關|海關]]申請[[簽證#落地签证|落地簽證]]，停留最多30天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-117-2e720-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&sms=B75E93A3DAB23A99 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|寮國護照|Laotian passport}}的[[老挝人口|寮國公民]]符合特定要求者，可透過[[中華民國政府|中華民國政府]]的[[中華民國簽證政策#東南亞七國網路申辦入境中華民國許可|東南亞七國網路申辦入台許可]]申請[[簽證#签证优惠待遇|免簽證]]，停留最多30天；抵台參加團客旅遊、國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

隨著雙邊關係逐步進展，赴寮國的商務及旅遊人數不斷增加，對於有意前往寮國從事經貿投資活動的[[台灣商人|台商]]而言，每次均需辦理簽證相當不便，雖然[[中華民國政府|中華民國政府]]已多次商請寮國政府，予以[[簽證#签证优惠待遇|免簽證]]待遇，但均未獲回應。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將寮國歸類為'''灰色警示'''：提醒注意（2017年12月5日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-360-2d0eb-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將寮國歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=8816227A9637D1EC |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|寮國經濟}}
===背景===
寮國北鄰[[中華人民共和國|中華人民共和國]]，又是[[內陸國家|內陸國家]]，國家安全需仰賴其幫助。[[經濟地理學|有此地緣關係]]，所以中華人民共和國是寮國第2大貿易夥伴，僅次於[[泰國|泰國]]；是寮國第3大[[外商直接投資|外資]]來源，僅次於[[越南|越南]]及泰國。中華人民共和國也是主要政府發展援助（Official Development Assistance，ODA）的提供國家之一，經濟發展要倚靠中華人民共和國，因此對與[[中華民國|中華民國]]拓展關係戒慎恐懼、心態保守，對中華民國並不友好。寮國雖已成為[[世界貿易組織|世界貿易組織]]（WTO）會員國，但對與中華民國發展關係仍持審慎態度，以免碰觸寮國與中華人民共和國的敏感關係。也因此，兩國互訪交流活動，特別是官方層面仍屬有限，因而雙方迄今未互設辦事機構。<ref name="與我國經貿關係"/>

===貿易===
近年來，以[[經濟部國際貿易局|經濟部國際貿易局]]（國貿局）舉辦的「台寮WTO入會雙邊諮商會議」較為重要。此外，國貿局駐越南代表處經濟組於2014年10月5日前往寮國，會晤{{Link-en|寮國工業暨商業部|Ministry of Industry and Commerce (Laos)|工商部}}對外貿易政策司高層官員，就加強雙邊經貿往來及在多邊[[國際組織|國際組織]]架構下互動及交流事宜交換意見，雙方都希望進一步促成彼此業者的商業合作契機。<ref name="與我國經貿關係"/>

在[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）的邀請下，寮國重要商會及大型企業首度組團訪問，進行破冰之旅。訪問團是由寮國商工總會（LNCCI）副會長帶領，於2014年3月26－29日訪問4天，參加在[[台北世界貿易中心南港展覽館|南港展覽館]]舉行的「全球採購夥伴大會」與「前進寮國拓展市場研討會」，並與外貿協會簽署合作[[備忘錄|備忘錄]]，以加強雙邊經貿合作關係。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=632228&url=/NewsSearch.asp |author = 黃榮俊 |date = 2014年3月26日 |title = 寮國首支訪問團26日抵台，展開破冰之旅 |publisher = 經貿透視網}}</ref>

2018年，寮國是中華民國第130大貿易夥伴、第104大[[進口|進口]]夥伴、第141大[[出口贸易|出口]]夥伴。出口至寮國的金額為593萬7,794[[美元|美元]]，年減11.937%；自寮國進口的金額為1,463萬1,106美元，年減6.023%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）869萬3,312美元，年減1.504%。2017年，寮國是中華民國第124大貿易夥伴、第107大進口夥伴、第134大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
自1989年起，中華民國開放寮國為直接貿易國家，目前在寮國從事投資或貿易的[[臺灣商人|臺商]]約30人，大部分集中於首都[[永珍|永珍]]地區。投資以[[中小企業|中小企業]]為主，投資金額不大，主要以[[成衣|成衣]]、[[紡織|紡織]]、[[木材|木材]]開發、[[家用电器|家電]]販售、[[塑膠|塑膠]]製造、[[造紙廠|造紙]]、[[家具|家具]]、[[飯店|飯店]]、及[[矿产资源|礦產]]開發為主。2000年－2012年12月，累計投資金額為2,388萬美元。<ref>[http://www.trade.gov.tw/World/ListArea.aspx?code=7020&nodeID=977&areaID=4&country=b645a-u5ZyL&pw=3 中華民國經濟部 國際貿易局 寮國國家檔案]</ref><ref name="市場環境分析">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 市場環境分析 |publisher = 中華民國對外貿易發展協會}}</ref>
