'''中山國中站'''位於[[中華民國|中華民國]][[台北市|台北市]][[中山區_(臺北市)|中山區]]、[[松山區_(臺灣)|松山區]]交界處，為[[台北捷運|台北捷運]][[文湖線|文湖線]]的[[捷運車站|捷運車站]]。本站往松山機場站方向即為高架段至地下段之過渡帶。

== 車站概要 ==
車站位於[[復興北路|復興北路]]上，接近[[民權東路|民權東路]]口、南以西起中山區錦州街、東起松山區富錦街口，原訂站名為'''[[下埤頭_(臺北市)|下埤頭]]'''，後因[[臺北市立中山國民中學|中山國中]]即位於鄰近巷內而得名；代碼為'''BR12'''。

本站往松山機場站方向為高架段至地下段之過渡帶。另外，列車由本站往松山機場站方向行經至民族東路與復興北路口時，有一個V字型的急轉彎，此段正巧又是下坡段，為確保安全，行車速度相當緩慢。

此外，為了便利理解其所在位置，某些臺北市的市區[[公車|公車]]會在公車路線圖或轉乘指引中，將此站稱為「民權復興路口」。

== 車站構造 ==
高架車站，兩個[[側式月台|側式-{月台}-]]，一個出入口；另有與站體共構的聯合開發辦公大樓。

===車站月台===
在[[台北捷運內湖線|內湖線]]通車以前，中山國中站為棕線的北方端點站，因此站內的1號月台（上行，北向）僅作為供乘客下車專用的下車月台，不開放上車。車站北側留有一段尾軌，一方面預備銜接內湖線軌道，一方面可作為儲車調度用，另外因留有[[橫渡線|橫渡線]]，[[動力分散式|電聯車]]得以轉向並換軌2號月台（下行，南向）繼續載客運行。內湖線通車後，與文山線直通運行，1號月台並開始開放乘客上車。

此外，中山國中站只有一座天橋供雙向通行，此設計有別於文湖線其他車站的動線，有兩座跨越軌道的天橋，因此在實施到站人潮與離站人潮的分流上較為困難。

== 營運狀況 ==
[[File:Overrun_of_MRT_Jhongshan_Jr_Hi_Sch_Sta.JPG|thumb]]
車站附近的復興北路、民權東路與民生東路為辦公大樓與金融機構的聚集地帶，平日上班族通勤人口眾多；民權東路上並有[[公車|公車]][[公車專用道|專用道]]與站台，有許多公車路線行經，可轉乘前往[[內湖區|內湖]]、[[三重區|三重]]、[[新莊區|新莊]]、[[蘆洲區|蘆洲]]、[[士林區|士林]]、[[天母|天母]]等地；也有前往[[龜山區|龜山]]（[[長庚醫療財團法人|林口長庚醫院]]）、[[桃園區|桃園區]]及往返於[[臺北松山機場|臺北松山機場]]與[[臺灣桃園國際機場|臺灣桃園國際機場]]間的國道（[[高速公路|高速公路]]）客運路線；另外車站旁還有往返於本站與[[美麗華百樂園|美麗華百樂園]]間的免費接駁車站牌。除此之外，在捷運[[松山機場站|松山機場站]]於2009年通車前，中山國中站是離松山機場最近的捷運車站。

== 歷史 ==
* 1996年3月28日：隨著[[台北捷運文山線|木柵線]]（今文山線）正式通車而啟用（當時即於此站舉行通車剪綵典禮）。
* 1997年，香港電影《[[春光乍洩|春光乍洩]]》利用[[台北捷運文山線|木柵線]]（今文山線）取景，做為電影最終喬段，並以列車進入本站做為電影的結束。
* 2009年7月4日：於當日凌晨舉行「再見中山國中站儀式」，感謝中山國中站員工及馬特拉系統對木柵線（今文山線）13年來的付出，市長[[郝龍斌|郝龍斌]]壓下按鈕後兩束[[雷射|雷射]]光直射出去，象徵中山國中站隨內湖線正式通車而停止作為棕線的端點站。
* 2013年12月12日：上午8時許在本站與松山機場站之間的高架路段，一輛故障列車卡在往松山機場方向，因適逢上班時間，有多名乘客困在車廂內，已緊急排除該狀況。
