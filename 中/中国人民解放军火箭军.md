'''中国人民解放军火箭军'''，简称'''解放军火箭军'''、'''中国火箭军'''，是[[中国人民解放军|中国人民解放军]]的五大[[中国人民解放军军兵种列表|军兵种]]之一，由于其装备的导弹大多以东风开头，所以被俗称为'''东风快递'''<ref>{{Cite web|title=中国人民解放军火箭军官方微博|url=https://weibo.com/u/6969972165}}</ref>。由[[中央军事委员会|中央军事委员会]]直接领导指挥，以地对地常规及[[战略导弹|战略导弹]]为主要装备，担负常规导弹目标打击及核反击战略作战等任务<ref>{{cite web |url = http://news.mod.gov.cn/headlines/2016-01/01/content_4634909.htm |title = 国防部新闻发言人就深化国防和军队改革有关问题接受媒体专访 |accessdate = 2016年1月1日 |website = 国防部网 }}</ref>。

火箭军的前身为存在于1966年至2015年的战略性独立[[兵种|兵种]]「'''中国人民解放军第二炮兵部队'''」（简称'''二炮'''）。在[[深化国防和军队改革|深化国防和军队改革]]中的2015年12月31日升格为“火箭军”。此举参考了[[苏联战略火箭军|苏联战略火箭军]]的有益经验，目的是强化[[核三位一体|陆海空三位一体的立体核威慑力量]]<ref>{{cite web |url = http://mil.huanqiu.com/observation/2016-01/8302064.html |title = 解读火箭军：或统合陆海空战略打击力量 实力剧增 |accessdate = 2016年1月1日 |website = 环球网 }}</ref>。

中国火箭军的主要武器为搭载[[核武器|核弹头]]的[[战略导弹|战略弹道导弹]]及搭载常规战斗部的弹道导弹和巡航导弹，装备[[短程弹道导弹|短程]]、[[中程弹道导弹|中程]]、[[洲际弹道导弹|洲际]][[弹道导弹|弹道导弹]]，远程[[巡航导弹|巡航导弹]]等多类型武器。

== 历史 ==
[[File:DF-5B_intercontinental_ballistic_missiles_during_2015_China_Victory_Day_parade.jpg|缩略图]]
[[File:China_Announces_Troop_Cuts_at_WWII_Parade_(screenshot)_20159180736.JPG|缩略图]]
1956年元旦下午三点，[[空气动力学|空气动力学]]家、中國導彈之父[[钱学森|钱学森]]给解放军高级將領讲课时在[[黑板|黑板]]上写下“[[火箭|火箭]]军”三字，並預言導彈戰將是未來戰爭必定出現的場景<ref>[https://www.thepaper.cn/newsDetail_forward_1415968 “火箭军”之名从哪儿来：1956年元旦由钱学森提出]</ref>。1956年，[[中国共产党中央委员会|中共中央]]、[[中央军事委员会|中共中央军委]]作出“研製导弹、原子弹，创建中国战略核力量、打破核垄断”的决策。1958年起，逐步组建了战略导弹的科研、训练、教学机构，隶属军委炮兵领导。1959年，中国第一支[[地对地导弹|地对地导弹]]部队正式组建。

解放軍导弹部队成立于1966年7月1日，是在极其秘密的情况下组建的，国务院总理[[周恩来|周恩来]]命名其为“第二炮兵”，简称“二炮”。作为直属[[中央军事委员会|中共中央军委]]掌控的战略部队，二炮除了身穿[[中国人民解放军陆军|中国人民解放军陆军]][[中国人民解放军军服|制服]]外，实际上和陆军各部没有关联，使用“二炮”名称主要是成立之初考虑到[[保密|保密]]等问题的需要。

1984年10月1日，中华人民共和国成立35周年国庆阅兵，第二炮兵部队的装备、人员首次公开展示。

2015年12月31日，作为[[深化国防和军队改革|深化国防和军队改革]]的措施之一，第二炮兵部队改名为“中国人民解放军火箭军”，由陸軍附属兵种成为与[[中国人民解放军陆军|陆]][[中国人民解放军海军|海]][[中国人民解放军空军|空]]三军并列的第四个独立[[军种|军种]]。2015年12月31日，[[中国人民解放军陆军|中国人民解放军陆军]]领导机构、中国人民解放军火箭军、[[中国人民解放军战略支援部队|中国人民解放军战略支援部队]]成立大会在[[北京|北京]][[八一大楼|八一大楼]]举行，[[中国共产党中央委员会总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军委主席|中央军委主席]][[习近平|习近平]]向陆军、火箭军、战略支援部队授军旗并致训词。成立大会上，[[中国共产党中央政治局|中共中央政治局委员]]、[[中央军委副主席|中央军委副主席]][[范长龙|范长龙]]宣读习近平签发的中央军委关于组建陆军领导机构、火箭军、战略支援部队及其领导班子成员任职命令和决定。大会由中共中央政治局委员、中央军委副主席[[许其亮|许其亮]]主持。<ref name="火箭军成立" /><ref>[http://news.xinhuanet.com/politics/2016-01/01/c_1117646667.htm 陆军领导机构火箭军战略支援部队成立大会在京举行 习近平向中国人民解放军陆军火箭军战略支援部队授予军旗并致训词，新华网，2016-01-01]</ref>

2016年7月1日，火箭军启用新式礼服与常服。颜色采用墨绿色搭配卡其色，礼服、春秋常服、冬常服、夏常服裤、大衣等为[[墨绿色|墨绿色]]，与火箭军主战装备的外观颜色相近；衬衣为[[卡其色|卡其色]]，与其他军种服装颜色区别明显。换发的服装包括军帽、礼服、常服、毛衣、绒衣裤和肩章、领章等151个品种。<ref>[http://finance.ifeng.com/a/20160630/14545494_0.shtml 火箭军官兵7月1日启用穿着新式礼服常服，凤凰网，2016-06-30]</ref>

火箭军由[[核導彈|核導彈]]部隊、[[常規導彈|常規導彈]]部隊及作戰保障部隊組成，約15萬兵員（一说10万人左右<ref>{{Cite web |url=http://www.armystar.com/wqzb/2016-11-14_46220_4.html |title=发生何事？中国突然公开洲际导弹数量(4) |access-date=2017-08-13 |archive-url=https://web.archive.org/web/20170731070648/http://www.armystar.com/wqzb/2016-11-14_46220_4.html |archive-date=2017-07-31 |dead-url=yes }}</ref>）。2016年11月，共青团中央在其官方微博上披露：火箭军装备各型近程弹道导弹（东风-11，东风-15及东风-16系列）1150枚，各型中程弹道导弹300枚（东风-21系列和东风-26型），各型远程/洲际弹道导弹200枚（东风-5系列，东风-31系列及东风-41型），此外还装备各类巡航导弹3000枚<ref>{{Cite web |url=http://www.armystar.com/wqzb/2016-11-14_46220_4.html |title=发生何事？中国突然公开洲际导弹数量(4) |access-date=2017-08-13 |archive-url=https://web.archive.org/web/20170731070648/http://www.armystar.com/wqzb/2016-11-14_46220_4.html |archive-date=2017-07-31 |dead-url=yes }}</ref>。{{clear}}
