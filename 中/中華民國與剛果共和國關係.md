'''中華民國與剛果共和國關係'''是指[[中華民國|中華民國]]與[[剛果共和國|剛果共和國]]（通稱'''剛果'''、'''剛果（[[布拉薩市|布]]）'''）之間的關係。兩國於1960－1964年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對剛果的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''、'''[[駐歐盟兼駐比利時代表處|駐歐盟兼駐比利時代表處]]'''{{Link-en|多重委派|Dual accreditation|共同管轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-280-d1072-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[剛果人民共和國|剛果人民共和國]]。]]
{{see also|中華民國外交|{{link-en|剛果共和國外交|Foreign relations of the Republic of the Congo}}}}
1960年9月10日，建立大使級[[外交#外交上的承認|外交關係]]。

1961年6月，於首都[[布拉薩|布拉薩]]設立'''中華民國駐剛果共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1964年4月17日，與中華民國斷交。20日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境剛果共和國。]]
[[File:Visa_requirements_for_R_Congo_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|剛果共和國簽證政策|Visa policy of the Republic of the Congo}}|{{link-en|剛果共和國公民簽證要求|Visa requirements for Republic of the Congo citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往剛果共和國駐[[日本|日本]][[外交代表機構#大使馆|大使館]]申請簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-280-d1072-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|剛果共和國護照|Republic of the Congo passport}}的{{link-en|剛果共和國人口|Demographics of the Republic of the Congo|剛果公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將剛果歸類為'''橙色警戒'''：避免非必要旅行（2017年4月27日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-208-df22d-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|剛果共和國經濟|Economy of the Republic of the Congo}}}}
===貿易===
2017年，剛果共和國是中華民國第111大貿易夥伴、第85大[[進口|進口]]夥伴、第185大[[出口贸易|出口]]夥伴。出口至剛果的金額為76萬5,613[[美元|美元]]，年減39.531%；自剛果進口的金額為3,959萬4,803美元，年減55.349%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）3,882萬9,190美元，年減55.578%。※2016年，剛果共和國是中華民國第91大貿易夥伴、第60大進口夥伴、第168大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
