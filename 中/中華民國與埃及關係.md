'''中華民國與埃及關係'''是指[[中華民國|中華民國]]與[[埃及阿拉伯共和國|埃及阿拉伯共和國]]之間的關係。兩國於1942－1956年有官方[[外交#外交上的承認|外交關係]]，斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對埃及的相關事務由'''駐[[約旦|約旦]][[臺北經濟文化辦事處|臺北經濟文化辦事處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref name="駐館位置及聯絡方式">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-244-28874-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[埃及|埃及]]。]]
{{see also|中華民國外交|{{link-en|埃及外交|Foreign relations of Egypt}}}}
{{see also|中国驻埃及大使列表{{!}}中華民國駐埃及王國／埃及共和國外交官列表|埃及驻华大使列表{{!}}埃及王國駐中華民國外交官列表}}
1930年4月23日，簽署《中埃[[关税|關稅]]臨時辦法條文》。<ref>{{Cite web |url = http://no06.mofa.gov.tw/mofatreatys/Result.aspx?tysubject_c=&tysubject_e=&tysubject_o=&tycountry_c=%e5%9f%83%e5%8f%8a&tycountry_e=%e5%9f%83%e5%8f%8a&tyeffectivedate=&tyeffectivedateE=&tysigneddate=&tysigneddateE=&start=Y&tykeyword=&Order=Signing |title = 條約協定查詢系統 |publisher = 中華民國外交部}}</ref>

1934年，中華民國於[[埃及英治時期|被英國控制下的埃及]]設立[[領事館|領事館]]。<ref name="年鑑">{{Cite web | url=http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm | title=我國與世界各國關係一覽表 | work=中華民國88年外交年鑑}}</ref>

1942年6月，於首都[[開羅|開羅]]設立'''中華民國駐[[埃及王國|埃及王國]][[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。<ref name="年鑑"/>

1948年11月14日，公使館升格為'''中華民國駐埃及王國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。<ref name="年鑑"/>

1950年11月30日，埃及在[[聯合國安全理事會|聯合國安全理事會]]上否決中共代表[[伍修權|伍修權]]等提出之「美國侵略台灣案」<ref>{{cite conference |title=聯合國安全理事會正式紀錄 第七十二號 第五三0次會議 |date=1950-11-30|publisher=[[聯合國安全理事會|聯合國安全理事會]] |location=[[紐約|紐約]]發拉星草場 |url = http://www.un.org/en/ga/search/view_doc.asp?symbol=S/PV.530&referer=/english/&Lang=C |accessdate=2015-08-24 |language=zh-hant}}</ref>。

1953年6月18日，廢除帝制，[[埃及共和國|埃及共和國]]成立。<ref name="年鑑"/>

1956年5月16日，埃及與中華民國斷交，關閉大使館。<ref name="年鑑"/>

1971年10月25日，在決定[[中國與聯合國#代表權問題|中國代表權]]的問題中，埃及連同大多數[[非洲国家和地区列表|北非國家]]，支持[[中華人民共和國|中華人民共和國]]取代中華民國的「中國」席位，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境埃及。]]
[[File:Visa_requirements_for_Egyptian_citizens.png|thumb]]的{{link-en|埃及人口|Demographics of Egypt|埃及公民}}必須申請[[簽證|簽證]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|埃及簽證政策|Visa policy of Egypt}}|{{link-en|埃及公民簽證要求|Visa requirements for Egyptian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可於抵達{{link-en|埃及機場列表|List of airports in Egypt|埃及主要國際機場}}時辦理[[簽證#落地签证|落地簽證]]入境，停留最多30天；抵達埃及主要國際港口時，也可申辦落地簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-244-28874-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持[[埃及護照|埃及護照]]的{{link-en|埃及人口|Demographics of Egypt|埃及公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將埃及按地區歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往，與'''紅色警戒'''：不宜前往，宜儘速離境（以上截至2017年11月27日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-211-28cd1-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
1990年，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於首都[[開羅|開羅]]設立'''台灣貿易中心'''。<ref name="年鑑"/><ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#AFRICA |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

2017年，埃及是中華民國第60大貿易夥伴、第91大[[進口|進口]]夥伴、第44大[[出口贸易|出口]]夥伴，在[[非洲国家和地区列表|非洲經濟體]]排名第4、13、2。出口至埃及的金額為3億6,175萬5,413[[美元|美元]]，年減18.253%；自埃及進口的金額為2,870萬2,777美元，年增23.422%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）3億3,305萬2,636美元，年減20.565%。※2016年，埃及是中華民國第54大貿易夥伴、第95大進口夥伴、第38大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===犯罪事件===
2014年7月，{{Link-en|埃及警察|Egyptian National Police|埃及警方}}查獲跨國[[詐騙集團|詐騙集團]]，共逮捕96人，當中有40名[[中華民國籍|中華民國籍]]嫌犯，其中1人是主嫌遭{{Link-en|埃及司法制度|Judiciary of Egypt|埃及法院}}判刑後仍有3年[[有期徒刑|有期徒刑]]，另39人被判刑2年已服刑完畢，但因私自用網路撥打[[國際電話|國際電話]]觸犯{{Link-en|埃及法律|Egyptian law}}，因此{{Link-en|埃及電信|Telecom Egypt|埃及國營電話公司}}對其進行[[民事诉讼|民事求償]]，法院判定每人平均支付1萬7,500[[美元|美元]]才能離境。這39人雖然已出獄，但仍被限制出境，[[中華民國外交部|中華民國外交部]]與[[中華民國駐外機構列表|駐外館處]]表示不會代為繳納罰金。若中華人民共和國方面要把嫌犯帶走，就必須替嫌犯償付這筆民事欠款。<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20170307/1070677/ |author = 陳培煌 |date = 2017年3月7日 |title = 埃及39台人詐騙犯 無力繳千萬罰金遭限制出境 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref>後來，埃及以嫌犯未繳清罰金為由，延長半年刑期。2017年1月，39名嫌犯出獄；3月，罰金繳清。並自4月起分批將嫌犯遣返回台，隨即遭拘提至[[台中地方法院檢察署|台中地方法院檢察署]]，依詐欺罪偵辦。<ref>{{Cite web |url = http://www.chinatimes.com/newspapers/20170620000324-260106 |author =  胡欣男、周毓翔 |date = 2017年6月20日 |title = 詐騙集團埃及蹲苦牢 每天僅一餐 酸奶配大餅 |publisher = [[中時電子報|中時電子報]]}}</ref>
