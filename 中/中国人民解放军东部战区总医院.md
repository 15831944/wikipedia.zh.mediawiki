'''中国人民解放军东部战区总医院'''，总院位于[[江苏省|江苏省]][[南京市|南京市]][[玄武区|玄武区]]中山东路305号，隶属[[无锡联勤保障中心|无锡联勤保障中心]]，是[[三级甲等医院|三级甲等医院]]。

==简介==
1929年1月，筹建'''中央模范军医院'''，主要收容伤兵。当时该医院占地面积50亩，建有13座活动木屋，300张病床，并拥有X线机等医疗设备和器械。1930年，[[南京国民政府|南京国民政府]][[行政院|行政院]]决定拨款改建该医院，更名为'''中央医院'''，划归南京国民政府内政部卫生部直属，由[[刘瑞恒|刘瑞恒]]兼任院长。当时[[胡文虎|胡文虎]]捐37.5万元银洋作为建筑费，所以主楼门楣上留下了胡文虎手迹。中央医院建成后，有外科、内科、眼科、妇产科、泌尿科、小儿科、传染科、牙科、五官科、电疗科、X线科等，是南京规模最大、档次最高的医院，专门为政府高级军政人员看病，并同国外医疗机构开展技术合作<ref>{{Cite web |url = http://dz.xdkb.net/html/2015-02/05/content_384179.htm |title = 孔祥熙家里人患牙病 蒋介石牙医韩文信“耍大牌” |author =  |date =2015-02-05  |publisher = 现代快报 |language =  |accessdate =  |quote =  }}</ref>。[[抗日战争|抗日战争]]时期，中央医院先后辗转[[长沙|长沙]]、[[贵阳|贵阳]]，1941年迁至[[重庆|重庆]]，重庆中央医院曾作为[[国立上海医学院|国立上海医学院]]的附属教学医院。抗日战争胜利后，重庆中央医院部分迁返南京，并将[[汪精卫政权|汪精卫政权]]的伪中央医院接收并入<ref name=jsszwsz>{{Cite book |url = http://www.jssdfz.com/book/wsz2/DEFAULT.html |title = 江苏省志·卫生志（下） |author = 江苏省地方志编纂委员会 |date = 2000年 |publisher =江苏人民出版社  |language =  |accessdate =  |quote = 第十一章 西医 第六节 主要医院简介 }}</ref>。重庆中央医院另有部分留在重庆，部分迁移至[[广州|广州]]。中央医院遂演化为中国人民解放军南京总医院（原南京中央医院）、[[重庆|重庆]][[西南医院|西南医院]]（原重庆中央医院）、[[广东省人民医院|广东省人民医院]]（原广州中央医院）等医院。

1949年4月[[中国人民解放军|中国人民解放军]]在[[渡江战役|渡江战役]]攻占南京时，由中国人民解放军接管该医院。1950年6月，更名为'''中国人民解放军华东军区医院'''。后更名为'''中国人民解放军华东军区总医院'''。1955年，更名为'''中国人民解放军南京军区总医院'''。1986年，更名为'''中国人民解放军南京军区南京总医院'''。南京军区南京总医院是[[中国人民解放军第二军医大学|中国人民解放军第二军医大学]]的教学医院<ref name=jsszwsz/>。

1993年9月，[[南京大学|南京大学]]与南京军区南京总医院签订开展教学合作的协议，南京军区南京总医院对外挂牌“'''南京大学医学院临床学院'''”，地位及作用等同[[南京大学医学院|南京大学医学院]]的附属医院。此后，医院人员在国内期刊发表论文署名单位均为“南京大学医学院临床学院”，对外参加国际学术活动、在国外期刊发表论文署名单位为“Jinling Hospital, Nanjing University, School of Medicine”。为统一对外称谓，医院向南京大学医学院提出申请，并上报南京大学批准后，2013年将“南京大学医学院临床学院”称谓统一为“'''南京大学医学院附属金陵医院'''”，英文称谓统一为“Jinling Hospital, Medical School of Nanjing University”。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，南京军区南京总医院由[[中国人民解放军南京军区|中国人民解放军南京军区]]转隶[[无锡联勤保障中心|无锡联勤保障中心]]，并更名'''中国人民解放军南京总医院'''<ref>{{Cite web |url = http://www.81.cn/jmywyl/2017-04/17/content_7564588.htm |title =  跨越2095公里，一张转诊单的“联勤之旅”|author =  |date =2017-04-17  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

2018年，更名为'''中国人民解放军东部战区总医院'''<ref>{{Cite web |url = https://news.sina.cn/2018-11-23/detail-ihmutuec3034133.d.html?vt=4 |title = 解放军五大战区总医院全部成立 |author =  |date =2018-11-23  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

南京市[[中山东路_(南京)|中山东路]]305号南京总医院内的“中央医院旧址”是南京市重点保护民国建筑，1931年由[[杨廷宝|杨廷宝]]设计，[[杨铭鼎|杨铭鼎]]监造。2002年被列为[[江苏省文物保护单位|江苏省文物保护单位]]。

==南京总医院汤山分院==
南京总医院汤山分院的地址为[[江苏省|江苏省]][[南京市|南京市]][[江宁区|江宁区]]温泉路5号。

1977年2月28日，'''中国人民解放军第一二五医院'''由南京军区后勤第十六分部划归江苏省军区领导，编床位300张。1983年10月，改为'''中国人民解放军南京疗养院'''，编床位100张，划归南京军区后勤第十六分部领导<ref>{{Cite book |url =  |title = 江苏省志·军事志|author = 江苏省地方志编纂委员会 |date =  2000年|publisher = 军事科学出版社 |page=598-601 |language =  |accessdate =  |quote = 第九章 后方勤务 第四节 卫生 六、医院简介 }}</ref>。

1938年，[[侵华日军|侵华日军]]在南京[[汤山|汤山]]建成陆军医院，[[抗日战争|抗日战争]]胜利后为国民党'''[[三军总医院|三军总医院]]汤山分院'''。1949年[[中华人民共和国|中华人民共和国]]成立后，为'''华东炮兵医院'''、'''华东军区第三陆军医院'''。其间，几度易名，1954年改称'''中国人民解放军第八三医院'''<ref name=jnshy>{{Cite web|url =http://www.jnqmzj.gov.cn/n1708c37.aspx|title =南京总医院疗养区门诊楼竣工启用|author =|date =2015-11-10|publisher =江宁双拥|language =|accessdate =|quote =}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1995年2月，中国人民解放军第八三医院与位于[[中山陵|中山陵]]的中国人民解放军南京疗养院合并为'''中国人民解放军南京军区南京疗养院'''，隶属联勤第十六分部，疗养和治疗床位并设<ref name=jnshy/>。2000年6月，南京军区南京疗养院从中山陵点调整到汤山点<ref>{{Cite web |url = http://www.nztsfy.com/index.php?case=archive&act=show&aid=33 |title =疗养一科简介  |author =  |date =2014-12-04   |publisher =南京军区南京总医院汤山分院  |language =  |accessdate =  |quote =  }}</ref>。2004年11月，转隶南京军区南京总医院。2005年9月，改称'''中国人民解放军南京军区南京总医院疗养区'''，承担疗养任务。该疗养区又称'''中国人民解放军南京军区南京总医院汤山分院'''。2015年11月7日，南京军区南京总医院疗养区举行门诊楼竣工启用暨转隶总院十周年庆祝活动<ref name=jnshy/>。2016年[[深化国防和军队改革|深化国防和军队改革]]后，改称'''中国人民解放军南京总医院疗养区'''（'''中国人民解放军南京总医院汤山分院'''）。
