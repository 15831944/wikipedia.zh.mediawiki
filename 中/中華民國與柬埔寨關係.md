'''中華民國與柬埔寨關係'''是指[[中華民國|中華民國]]與[[柬埔寨王國|柬埔寨王國]]之間的關係。兩國於1947－1958年有領事級[[外交#外交上的承認|外交關係]]，終止關係後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構，對柬埔寨的相關事務由設在[[越南|越南]]的'''{{Link-en|駐胡志明市台北經濟文化辦事處|Taipei Economic and Cultural Office, Ho Chi Minh City}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref name="駐館位置及聯絡方式">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-159-129ef-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
{{see also|中華民國外交|{{link-en|柬埔寨外交|Foreign relations of Cambodia}}}}
1947年9月3日，中華民國在[[法屬印度支那|法屬印度支那]]的柬埔寨[[金邊|金邊]]設立[[領事館|領事館]]。但1958年7月19日，柬埔寨與[[中華人民共和國政府|中華人民共和國政府]]建交，10月與[[中華民國政府|中華民國政府]]終止領事關係。

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括當時的[[高棉共和國|高棉共和國]]。]]
1970年10月，[[龍諾|龍諾]]將軍發動政變建立共和政體，改變1970年以前的親共立場，與中華民國的接觸逐漸頻繁，於是投下支持[[中國與聯合國#代表權問題|中華民國續留聯合國]]的一票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，是唯二支持中華民國的東南亞國家（另一個為菲律賓），1974年底甚至考慮恢復與中華民國邦交，然而1975年，[[紅色高棉|紅色高棉]]攻陷[[金邊|金邊]]，建立[[民主柬埔寨|柬埔寨民主共和國]]，中華民國與[[高棉共和國|高棉共和國]]的交流暫告中斷。

1990年代，[[柬埔寨王國|柬埔寨王國]]恢復後，中華民國與柬埔寨的交流日益熱絡，但仍堅持親共立場。1994年9月，在金邊由[[駐泰國台北經濟文化辦事處|駐泰國臺北經濟貿易辦事處]]代表與柬埔寨總理特使簽署《互設代表處協定》。根據該協定，分別設立「駐金邊臺北經濟文化代表處」與「駐臺北金邊經濟文化代表處」。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=5FB0AED3B1F0E49F |author = 公眾外交協調會 |date = 1994年9月10日 |title = 中華民國與柬埔寨王國為促進雙方經貿 |publisher = 中華民國外交部}}</ref>1995年11月，駐金邊代表處正式成立，但於1997年8月28日關閉。<ref name="駐館位置及聯絡方式"/>

2010年8月，柬埔寨《{{link-en|金邊郵報|The Phnom Penh Post}}》報導，[[柬埔寨政府首腦列表|柬埔寨首相]][[洪森|洪森]]禁止中華民國在柬埔寨設立[[辦事處|辦事處]]，一律嚴懲協助其設立辦事處的官員。對此中華民國政府回應，希望柬方正視中華民國政府的存在，並非[[台灣省_(中華人民共和國)|中華人民共和國之一省或一地區]]，如果持這樣的態度，可能會對爭取[[臺商|臺商]]到柬埔寨[[投資|投資]][[貿易|貿易]]都會有不良影響。<ref>[http://www.epochtimes.com/b5/10/8/12/n2993546.htm 《大紀元時報》 〈柬埔寨禁設辦事處 臺灣提嚴正抗議〉]</ref>此外，柬埔寨更禁止台商悬挂[[中华民国国旗|中华民国国旗]]，首相洪森指这是为了“维护中国主权”。<ref name="推翻"/>

2017年9月，柬埔寨亲政府网站《Fresh News》的报道指，有“台湾激进组织成员”与[[柬埔寨救國黨|柬埔寨救國黨]]“秘密会面”。报道同时刊登救国党领袖[[金速卡|金速卡]]的女儿金摩诺薇雅（Kem Monovithya）出席台湾[[民进党|民进党]]总统候選人[[蔡英文|蔡英文]]竞选活动的照片。执政的[[柬埔寨人民党|柬埔寨人民党]]称政府将就报道内容进行调查，之後，柬埔寨就拘捕了金速卡，指他合谋与[[美国|美国]]推翻首相洪森的政权。民进党也澄清，绝不可能介入它国的政治，也说已经让柬埔寨方面了解，事情已经“圆满落幕”。<ref name="推翻">{{Cite web |url = http://www.bbc.com/zhongwen/simp/chinese-news-41396250 |author = 綜合報導 |date = 2017年9月26日 |title = 台湾如何看被指试图“推翻”柬埔寨政府的指控？|publisher = 《[[BBC#BBC中文部|BBC中文網]]》}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]、[[簽證#电子签证|電子簽證]]的方式入境柬埔寨。]]
[[File:Visa_requirements_for_Cambodian_citizens.png|thumb]]必須申請[[簽證|簽證]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|柬埔寨簽證政策|柬埔寨公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]、[[簽證#电子签证|電子簽證]]的方式入境柬埔寨。觀光簽證可停留1個月並可延長1個月、一般簽證可停留1個月至1年、[[過境簽證|過境簽證]]可停留5天、電子簽證可停留1個月。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-159-129ef-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&sms=B75E93A3DAB23A99 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|柬埔寨護照|Cambodian passport}}的[[柬埔寨人口|柬埔寨公民]]符合特定要求者可透過[[中華民國政府|中華民國政府]]的[[中華民國簽證政策#東南亞七國網路申辦入境中華民國許可|東南亞七國網路申辦入台許可]]申請[[簽證#签证优惠待遇|免簽證]]，停留最多30天；抵台參加團客旅遊、國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將柬埔寨歸類為'''灰色警示'''：提醒注意（2017年12月5日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-174-95d96-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將柬埔寨歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=D4ED5728A0DBC301 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|柬埔寨經濟}}
===貿易===
2018年，柬埔寨是中華民國第46大貿易夥伴、第67大[[進口|進口]]夥伴、第32大[[出口贸易|出口]]夥伴。出口至柬埔寨的金額為7億0,061萬7,626[[美元|美元]]，年增7.824%；自柬埔寨進口的金額為9,438萬2,433美元，年增21.656%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）6億0,623萬5,193美元，年增5.948%。2017年，柬埔寨是中華民國第47大貿易夥伴、第70大進口夥伴、第32大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
中華民國投資柬埔寨的企業以[[中小型企業|中小型企業]]為主，總計約200餘家。主要投資項目為：[[房地產|房地產]]及土地開發、[[農業|農業]]開發、[[木材|木材]]加工、[[紡織|紡織]][[成衣|成衣]]、[[鞋|製鞋業]]、[[觀光|旅遊業]]，其中最主要的投資為成衣加工業，共計90件投資項目與50家廠商，約占柬埔寨製衣廠[[工會|工會]]四分之一。<ref>[http://twbusiness.nat.gov.tw/countryPage.do?id=11&country=KH 中華民國經濟部 全球臺商服務網 柬埔寨]</ref>

===金融===
最早於柬埔寨設立銀行的[[臺灣銀行列表|中華民國籍銀行]]為[[第一銀行|第一銀行]]，該公司於1998年於柬埔寨[[首都|首都]][[金邊|金邊]]設置分行。2011年[[兆豐國際商業銀行|兆豐國際商業銀行]]亦於金邊設立分行營運。後因{{link-en|柬埔寨國家銀行|National Bank of Cambodia}}停止發行銀行執照的緣故，其餘臺資銀行皆以[[併購|併購]]的方式進入柬埔寨市場，如[[玉山銀行|玉山銀行]]收購柬埔寨聯合商業銀行七成股權、[[國泰世華銀行|國泰世華銀行]]收購柬埔寨新加坡儲蓄商業銀行。<ref>[http://bank.hexun.com.tw/2013-03-22/152397244.html 和訊銀行 臺灣玉山銀行收購柬埔寨聯合商業銀行70%股權]</ref>

===臺商組織===
柬埔寨[[臺灣商人|台商]]協會於1996年9月1日成立。目前有2個台商組織，包括台商會及台商成衣紡織聯誼會，其中台商會現有廠商會員114家以及個人會員108家。會員廠商涵蓋的產業包括製衣、製鞋、旅遊、房地產、農產加工、醫療服務、木材加工等。<ref>{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/kh/pdf/103-21Cambodia.pdf |title = 投資環境簡介 |publisher = 經濟部全球臺商服務網}}</ref>

===犯罪事件===
2016年6月底，涉嫌[[台湾诈骗集团|電信詐騙]]的[[台灣人|台灣人]]遭到{{Link-en|柬埔寨警察|Law enforcement in Cambodia|柬埔寨警方}}逮捕，在北京当局介入后共有18名嫌疑犯被遣送至[[温州|温州]]。<ref>{{Cite web |url = http://gotv.ctitv.com.tw/2016/09/262539.htm |date = 2016年9月2日 |title = 台灣詐騙犯又遭逮！柬埔寨電信詐騙案 13名台人恐遭遣陸 |publisher = [[中天電視|中天電視]]}}</ref>9月20日，在柬埔寨涉嫌電信詐騙的13名台灣人又被遣送至中華人民共和國（以下簡稱對岸），中華民國外交部對此深表不滿。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/1831679 |author = 呂伊萱 |date = 2016年9月20日 |title = 柬埔寨13詐騙台嫌今遣中 外交部表達關切 |publisher = 《[[自由時報|自由時報]]》}}</ref>

2017年7月26日，柬埔寨移民總署於17日破獲跨國電信詐騙案，共逮捕32名嫌犯，其中7名是台灣人，有4名在26日遭遣送至對岸，剩下3名也將一併遣送。中華民國外交部對{{link-en|柬埔寨政府|Politics of Cambodia}}基於「[[一個中國|一個中國]]」政策又迫於對岸壓力，不僅不同意{{Link-en|駐胡志明市台北經濟文化辦事處|Taipei Economic and Cultural Office, Ho Chi Minh City}}前往探視，更將犯嫌遣送對岸，表示嚴正關切及深切遺憾。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/2144452 |author = 彭琬馨 |date = 2017年7月27日 |title = 柬埔寨7名詐騙犯強行遣中 外交部表遺憾 |publisher = 《自由時報》}}</ref>

2018年11月26日，[[中华人民共和国公安部|中华人民共和国公安部]]與柬埔寨執法單位合作，共逮捕233名電信詐騙犯，包括46名台灣人，這是近年來在[[東南亞|東南亞]]逮捕電信詐騙犯最多的一次。12月6日，對岸安排3架民航[[包機|包機]]，將其全部遣送至對岸。<ref>{{Cite web |url = https://tw.appledaily.com/new/realtime/20181206/1479552/ |author = 綜合報導 |date = 2018年12月6日 |title = 46台人在柬埔寨涉詐騙被抓 今被押解到中國廣州 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》}}</ref>
