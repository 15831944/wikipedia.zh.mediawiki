==歷史==
===泰北孤軍===
{{see also|泰北孤軍}}
1949年，[[中華民國|中華民國]]在[[國共內戰|國共內戰]]失利後撤往[[臺灣|臺灣]]，原於[[雲南|雲南]]地區作戰的[[中華民國陸軍|中華民國陸軍]]第八軍237師、第廿六軍93師，在失去後援後便經過跋山涉水，輾轉逃至[[緬甸行政區劃|緬甸北部]]。爾後因緬甸在[[中華人民共和國|中華人民共和國]]的慫恿與[[蘇聯|蘇聯]]的協助之下，向[[聯合國|聯合國]]控訴中華民國入侵，在經過一番斡旋的調節之後，俗稱「孤軍」的中華民國陸軍第八軍237師、第廿六軍93師，移往[[泰國|泰國]]北部定居。爾後「孤軍」在{{link-en|泰國共產黨暴動|Communist insurgency in Thailand|泰國內戰}}有功，因而獲得[[泰国君主列表|泰王]][[蒲美蓬|蒲美蓬]]授予功勳以及泰國居留權。部分孤軍成員選擇落地生根留在泰國，而部分選擇移民中華民國臺灣，目前大多數孤軍後裔多居住在[[新北市|新北市]][[中和區|中和區]]與[[永和區|永和區]]，中和區就因此產生俗稱「'''緬甸街'''」的華新街。

1947年9月9日，兩國建立大使級[[外交#外交上的承認|外交關係]]，並互派[[特命全權大使|大使]]。1948年1月1日，外交部次長[[葉公超|葉公超]]作為慶賀[[緬甸|緬甸]]獨立{{Link-en|特使|Diplomatic_rank#Special_envoys}}抵達[[仰光|仰光]]，參與緬甸正式獨立的儀式。1949年12月18日，兩國斷交，並於1950年1月10日關閉[[外交代表機構#大使馆|大使館]]。

1950年6月8日<ref>[http://www.fmprc.gov.cn/mfa_chn/ziliao_611306/2193_611376/ 中華人民共和國與各國建立外交關係日期簡表]</ref>，緬甸與中華人民共和國建交之後轉而不承認中華民國。1971年10月25日，緬甸投下不支持[[中國與聯合國#代表權問題|中華民國續留聯合國]]的一票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。

2012年10月，[[翁山蘇姬|翁山蘇姬]]訪問[[美國|美國]]三周，10月2日在[[洛杉磯|洛杉磯]]向數千名支持者發表公開演說時被問到緬甸期盼的民主典範時，她說緬甸從許多地方有太多東西可以學習，不只是像[[南韓|南韓]]、[[臺灣|臺灣]]、[[蒙古|蒙古]]和[[印尼|印尼]]等亞洲國家，還有在1980到1990年代從[[共產黨|共產黨]]轉型為民主的[[東歐|東歐]]國家、從[[軍政府|軍政府]]轉型的[[拉丁美洲|拉丁美洲]]國家、以及從[[獨裁政體|獨裁政體]]轉型的[[南非|南非]]。<ref>[http://news.rti.org.tw/index_newsContent.aspx?nid=381495&id=6&id2=2 結束訪美前夕 翁山蘇姬向民主轉型典範致敬] {{webarchive|url=https://archive.is/20121222175427/http://news.rti.org.tw/index_newsContent.aspx?nid=381495&id=6&id2=2 |date=2012-12-22 }}, [[中央廣播電臺|中央廣播電臺]] Radio Taiwan International, 2012/10/3</ref><ref>[http://www.google.com/hostednews/afp/article/ALeqM5g7jcSqNCiDcId7mEJs9DXzmqzGtA?docId=CNG.a49d8e2a5f23889e2d2987feb130e63f.31 Myanmar's Suu Kyi ends US trip, hailing democracy], AFP, Oct 3, 2012</ref>

===背景===
{{see also|緬甸民主改革}}
2011年緬甸文人政府執政前，因政治考量，在投資法規中雖未明文將中華民國列為禁止[[外商直接投資|直接投資]]國家，但由於兩國無外交關係，緬甸使領館對[[中華民國政府|中華民國政府]]所發文件或驗證的文件，均不予驗證，造成赴緬甸投資必須以設在第3地（[[香港|香港]]、[[新加坡|新加坡]]、[[泰國|泰國]]等）的公司名義提出。緬甸進口商申請要進口中華民國產品，所提[[發票|發票]]也必須是第3地公司所開，才可獲准核發進口許可證，並以[[美元|美元]]進行國際交易轉移，且只能經由東南亞的新加坡、[[馬來西亞|馬來西亞]]、泰國及香港等少數幾家銀行進行緬甸[[信用狀|信用狀]][[押匯|押匯]]等業務。此外，緬甸港口設施落後，使得貨船運量受限，出口至緬甸或自緬甸進口的貨物大多經由新加坡換船再運往兩國，致使兩國貿易需採間接方式，造成諸多限制與雙邊經貿交流的不順利。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

自從2011年3月緬甸政府宣示政經改革開放以來，中華民國政府多次向緬甸政府要求解除直接貿易及直接投資的限制，而在2013年1月終獲得解除對臺灣廠商經商的相關限制。<ref name="與我國經貿關係"/>

===貿易===
2013年，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於[[緬甸|緬甸]]前首都、最大城市[[仰光|仰光]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#ASIA |title = 外貿協會全球據點|publisher = 中華民國對外貿易發展協會}}</ref><ref name="105年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2016/PDF/%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91%E5%AE%8C%E6%95%B4%E6%8E%92%E7%89%88.pdf |title = 《中華民國105年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>2016年，[[經濟部國際貿易局|經濟部國際貿易局]]也在仰光設立'''駐緬甸代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=214&pid=305862 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2018年，緬甸是中華民國第64大貿易夥伴、第76大[[進口|進口]]夥伴、第50大[[出口贸易|出口]]夥伴。出口至緬甸的金額為2億7,135萬4,598[[美元|美元]]，年增13.900%；自緬甸進口的金額為7,132萬1,484美元，年增24.646%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）2億0,003萬3,114美元，年增10.504%。2017年，緬甸是中華民國第65大貿易夥伴、第79大進口夥伴、第51大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

=== 投資 ===
近年來隨著[[缅甸民主改革|緬甸的政治經濟開放]]，許多[[臺商|臺商]]因而紛紛前往投資。至2017年，有超過250家臺商在緬甸投資，投資金額為3億7,000萬美元。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>主要投資項目為[[成衣|成衣]]加工、[[日用品|民生用品]]、[[玻璃纖維|玻璃纖維]][[水塔|水塔]]、[[漁船|漁船]]、[[空氣調節|冷氣空調]]、[[包裝|包裝]][[膠帶|膠帶]]、[[瓦楞紙盒|紙箱]]、[[建築|建築]]、[[木材|木材]]、[[農業|農業]]、[[鞋|鞋]]業、[[刺繡|繡花]]、[[汽車零組件列表|汽車零件]]、[[化妝品|化妝品]]、[[旅行社|旅行社]]、[[聚乙烯|PE]][[帆布|帆布]]加工、[[飲食業|餐飲]]、[[珠寶|珠寶]][[首飾|首飾]]、[[水產|水產]]、[[航空|航空]]（[[華航|華航]]及[[長榮|長榮]]）、[[金融|金融]]（[[第一商業銀行|第一銀行]]）、[[光學|光學]][[眼鏡#鏡片|鏡片]]（[[亞洲光學|亞洲光學]]）等。<ref>[http://www.trademag.org.tw/content02.asp?id=488951&type=21 國經商情雙周刊：臺商在仰光 守著落後待騰飛]</ref><ref>[http://www.gvm.com.tw/Boardcontent_11136_2.html 遠見雜誌：光學獲利王 亞洲光學─永遠比對手看快一步]</ref>。<ref name="投資環境簡介"/>

2013年4月，中華民國的[[台灣中油|中油公司]]宣布與中華人民共和國的[[中国石油化工集团|中石化集團]]合作，共同開發位於緬甸中央盆地「緬甸陸上D區礦區」的[[石油|油]][[天然氣|氣]]。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/mm/pdf/103-16Myanmar%20.pdf |title = 投資環境簡介 |publisher = 經濟部全球臺商服務網}}</ref>

=== 農業 ===
臺灣農友種苗公司設立於1968年，該公司於1994年進入緬甸市場，並於1996年與{{link-en|緬甸農業暨灌溉部|Ministry of Agriculture and Irrigation (Myanmar)}}合作，在最大城市[[仰光|仰光]]近郊的[[伊洛瓦底省|伊洛瓦底省]]設立農業學校，教導農民種植[[瓜|瓜果類]]作物。其中以[[西瓜|西瓜]]在緬甸的推廣最為成功，臺灣農友種苗公司的西瓜種苗在緬甸的[[市場佔有率|市佔率]]高達九成，除了供應緬甸的[[供給和需求#内需|內需]]市場外，甚至外銷[[中华人民共和国|中华人民共和国]]。有[[臺灣媒體|臺灣媒體]]將之喻為「西瓜外交」<ref>[http://mag.udn.com/mag/newsstand/storypage.jsp?f_ART_ID=421628  聯合新聞網：農友種苗的西瓜外交]</ref>。

目前簽訂有《臺緬農業合作[[備忘錄|備忘錄]]》。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 國家基本資料表 |publisher = 中華民國對外貿易發展協會}}</ref>
