'''中華民國與巴哈馬關係'''是指[[中華民國|中華民國]]與[[巴哈馬聯邦|巴哈馬聯邦]]之間的關係。兩國於1989－1997年有官方[[外交#外交上的承認|外交關係]]。斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對巴哈馬的相關事務由'''[[駐邁阿密臺北經濟文化辦事處|駐邁阿密臺北經濟文化辦事處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-180-34bb4-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref><ref>{{Cite web |url = http://www.taiwanembassy.org/usmia/post/16.html |title = 辦事處簡介 |publisher = 中華民國駐外單位聯合網站}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，[[巴哈馬|巴哈馬]]當時屬於[[大英帝國#美洲与大西洋|英國殖民地]]，因此不能對[[中國與聯合國#代表權問題|中華民國續留聯合國]]一案參與投票，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。]]
{{see also|中華民國外交|{{link-en|巴哈馬外交|Foreign relations of The Bahamas}}}}
1989年1月，中華民國行政院長[[俞國華|俞國華]]率領交通部長[[郭南宏|郭南宏]]、外交部政務次長[[金樹基|金樹基]]、新聞局長[[邵玉銘|邵玉銘]]等官員，前往巴哈馬會見總理{{link-en|賓德林|Lynden Pindling}}、副總理兼外交部長梅納商討建交事宜<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=86EE0295F59D361B |author = 公眾外交協調會 |date = 1989年1月5日 |title = 行政院俞院長伉儷率團訪問巴哈馬、多明尼加及瓜地馬拉三國 |publisher = 中華民國外交部}}</ref>。1月9日，兩國建立大使級[[外交#外交上的承認|外交關係]]<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=76D2DDD4CDEFFBC5 |author = 公眾外交協調會 |date = 1989年1月9日 |title = 中華民國與巴哈馬建立全面外交關係 |publisher = 中華民國外交部}}</ref>。5月，於首都[[拿索_(巴哈馬)|拿索]]設立'''中華民國駐巴哈馬聯邦[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]<ref name="鄭重聲明"/>；於首都[[台北市|台北]]設立'''巴哈馬聯邦駐中華民國[[領事館|名譽領事館]]'''<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=618EEBD2E7992EEC |author = 公眾外交協調會 |date = 1990年4月20日 |title = 巴哈馬政府為便利國人赴該國從事旅遊及商務活動 |publisher = 中華民國外交部}}</ref>。

1991年10月8日，{{link-en|巴哈馬參議院議長列表|List of Presidents of the Senate of the Bahamas|巴哈馬參議院議長}}柯比（Edwin Coleby）夫婦抵台訪問。期間會見立法院長[[梁肅戎|梁肅戎]]、外交部常務次長[[程建人|程建人]]、[[行政院農業委員會|行政院農業委員會]]副主任委員邱茂英等政府官員，並參加[[中華民國國慶|中華民國國慶]]活動。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=11732A92DA40A9BA |author = 公眾外交協調會 |date = 1991年10月7日 |title = 巴哈馬參議院議長柯比伉儷訂於本（十）月八日抵華訪問六天 |publisher = 中華民國外交部}}</ref>

1996年1月24日，[[中華民國外交部#歷任首長|外交部長]][[錢復|錢復]]夫婦前往巴哈馬訪問，期間會見[[巴哈馬總督|巴哈馬總督]]、[[巴哈马总理|總理]]、副總理、外交部長等政府官員。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=D9BFACD2156C4C24 |author = 公眾外交協調會 |date = 1996年1月25日 |title = 外交部長錢復夫婦於廿四日下午飛抵巴哈馬作為期五天之正式訪問 |publisher = 中華民國外交部}}</ref>

1997年5月18日，兩國斷交。<ref name="鄭重聲明">{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=5028B03CED127255&sms=5ED24855AD8E6C58&s=F87C4522B08212D0 |author = 公眾外交協調會 |date = 1997年5月18日 |title = 中華民國政府茲鄭重聲明自即日起終止與巴哈馬之外交關係，同時中止中巴兩國各項協定及合作關係，並召回駐巴哈馬大使 |publisher = 中華民國外交部}}</ref>6月20日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境巴哈馬。]]
[[File:Visa_requirements_for_Bahamian_citizens.png|thumb]]的[[巴哈马人口|巴哈馬公民]]必須申請簽證方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|巴哈馬簽證政策|巴哈馬公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]申請單次巴哈馬觀光簽證可停留最多90天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-180-34bb4-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持有[[巴哈馬護照|巴哈馬護照]]的[[巴哈马人口|巴哈馬公民]]抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

====事件====
在邦交期間，巴哈馬於1992年9月30日起，曾給予中華民國公民[[簽證#签证优惠待遇|免簽證]]入境60天待遇。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=38531DEDC395B4C6 |author = 公眾外交協調會 |date = 1992年10月9日 |title = 巴哈馬（COMMONWEALTH OF BAHAMAS）為便利我國人士赴該國旅遊洽商 |publisher = 中華民國外交部}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|巴哈馬經濟|Economy of the Bahamas}}}}
===貿易===
2018年，巴哈馬是中華民國第183大貿易夥伴、第154大[[進口|進口]]夥伴、第190大[[出口贸易|出口]]夥伴。出口至巴哈馬的金額為54萬5,236[[美元|美元]]，年減42.113%；自巴哈馬進口的金額為87萬2,166美元，年增115.187%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）32萬6,930美元，年減160.927%。2017年，巴哈馬是中華民國第184大貿易夥伴、第170大進口夥伴、第179大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===合作===
兩國於1992年協議成立[[海螺|海螺]]殼加工訓練所，由中華民國提供機器及技術，協助巴哈馬利用盛產的海螺殼，加工製造成[[首飾|首飾]]或[[紀念品|紀念品]]，此計劃於1995年7月開始實施。加工訓練所於1996年1月啟用，由中華民國外交部長[[錢復|錢復]]、巴哈馬教育部長杜茂德共同剪綵。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=D015CC7CABEC323D |author = 公眾外交協調會 |date = 1996年1月26日 |title = 中華民國與巴哈馬合作成立的海螺殼加工訓練所於今（廿五）日下午（台北時間廿六日）正式揭幕 |publisher = 中華民國外交部}}</ref>
