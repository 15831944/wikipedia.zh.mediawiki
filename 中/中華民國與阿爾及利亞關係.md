'''中華民國與阿爾及利亞關係'''是指[[中華民國|中華民國]]與[[阿爾及利亞民主人民共和國|阿爾及利亞民主人民共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對阿爾及利亞的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-125-a4cce-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
{{see also|中華民國外交|{{link-en|阿爾及利亞外交|Foreign relations of Algeria}}}}
1971年10月25日，第26屆[[聯合國大會|聯合國大會]]就[[阿爾巴尼亞|阿爾巴尼亞]]、[[阿爾及利亞|阿爾及利亞]]等23國所提出的「恢復[[中華人民共和國|中華人民共和國]]在[[聯合國|聯合國]]的合法權利」議案進行[[投票|表決]]。最終以76票贊成、35票反對、17票棄權的結果通過。根據《[[聯合國憲章|聯合國憲章]]》和聯合國大會議事規則，提案通過即成為正式決議，是為《'''[[聯合國大會2758號決議|聯合國大會2758號決議]]'''》。[[中華民國|中華民國]]、[[馬爾地夫|馬爾地夫]]、[[阿曼|阿曼]]等3國未投票。當時代表'''[[中國|中國]]'''[[合法性|正統]]的[[中華民國政府|中華民國政府]]以[[中華民國外交部#歷任首長|外交部長]][[周書楷|周書楷]]為代表在表決之前數小時，於大會上主動宣布退出聯合國。日後，中華民國政府將此決議稱為「排我納[[共匪|匪]]案」。

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|阿爾及利亞簽證政策|{{link-en|阿爾及利亞公民簽證要求|Visa requirements for Algerian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往[[首爾|首爾]]的{{Link-ko|阿爾及利亞駐韓國大使館|주한 알제리 대사관}}申請簽證。經阿爾及利亞轉機，亦須申請[[過境簽證|過境簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-125-a4cce-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持[[阿爾及利亞護照|阿爾及利亞護照]]的[[阿爾及利亞人口|阿爾及利亞公民]]抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將阿爾及利亞歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2017年4月20日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-167-5803a-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|阿爾及利亞經濟}}
===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於首都[[阿爾及爾|阿爾及爾]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#AFRICA |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

阿爾及利亞在政治上與中華民國極不友好，申請簽證費時也需保人等許多障礙，除阿爾及爾台灣貿易中心專任人員外，在當地無任何[[台灣商人|台商]]及[[台灣人|台灣民眾]]。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref>

2017年，阿爾及利亞是中華民國第63大貿易夥伴、第54大[[進口|進口]]夥伴、第79大[[出口贸易|出口]]夥伴。出口至阿爾及利亞的金額為8,157萬4,534[[美元|美元]]，年減0.298%；自阿爾及利亞進口的金額為2億1,693萬3,561美元，年增34.800%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）1億3,535萬9,027美元，年增71.100%。※2016年，阿爾及利亞是中華民國第64大貿易夥伴、第54大進口夥伴、第78大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
