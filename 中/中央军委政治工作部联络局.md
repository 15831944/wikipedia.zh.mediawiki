'''中央军委政治工作部联络局'''，位于[[北京市|北京市]]，是[[中央军委政治工作部|中央军委政治工作部]]下属局，负责联络工作。

==沿革==
1955年6月20日，中央军委总参谋部联络部正式改建为[[中共中央调查部|中共中央调查部]]，划归党务系统，而联络部中的军事情报单位继续保留在中央军委领导下，后来沿革为'''中国人民解放军总政治部联络部'''<ref>{{Cite web |url =  http://epaper.chinatibetnews.com/xzsb/html/2014-04/18/content_530697.htm|title =罗青长逝世  |author =  |date =2014-04-18  |publisher = 西藏商报 |language =  |accessdate =  |quote =  }}</ref>。中国人民解放军总政治部联络部于1955年6月设立。1969年12月撤销，并入[[中国人民解放军总政治部群众工作部|中国人民解放军总政治部群众工作部]]。1975年7月恢复设立中国人民解放军总政治部联络部。

中国人民解放军总政治部联络部曾下设联络处、调研处<ref>范秀传，“边疆史地研究与边疆现状座谈会”纪要，中国边疆史地研究导报1990年第03期</ref>等机构，并曾派出广州联络局<ref>{{Cite web |url = http://www.81.cn/jfjbmap/content/2016-03/28/content_139044.htm |title =  郑锦波同志逝世|author =  |date = 2016-03-28 |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref>等机构。

2016年在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军总政治部撤销，成立[[中央军委政治工作部|中央军委政治工作部]]，下设'''中央军委政治工作部联络局'''<ref>{{Cite web |url =http://www.jledu.gov.cn/show/44537.html  |title = 吉林省高校体育专业2018届毕业生供需见面洽谈会在吉林体育学院举行 |author =  |date =  2017-12-06 |publisher = 吉林省教育信息网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://mp.weixin.qq.com/s/E-s3O6iELDWkYxggnoFwkQ  |title = 腊月里的温暖，在军属心底如花绽放 |author =  |date =2017-01-12  |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

==职责==
2010年修订的《中国人民解放军政治工作条例》第十四条规定：中国人民解放军政治工作的主要内容是：“（十四）联络工作。进行瓦解敌军、联络友军工作。开展对[[台湾|台]]有关工作。调查研究外军、敌军和民族分裂势力情况。开展[[心理战|心理战]]工作。”<ref> {{Cite book | author =  | title =  中国人民解放军政治工作条例| location =  | publisher =  中国人民解放军总政治部| date = 2010年 | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref><ref>{{Cite web |url =  http://www.guancha.cn/strategy/2013_05_27_147429.shtml|title = 外媒聚焦解放军总政治部神秘部门：可影响外国防务政决策 |author =  |date = 2013-05-27 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>
