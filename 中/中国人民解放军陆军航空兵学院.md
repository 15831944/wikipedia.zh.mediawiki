'''中国人民解放军陆军航空兵学院'''（简称'''陆军航空兵学院'''），位于[[北京市|北京市]][[通州区|通州区]][[台湖镇|台湖镇]]，隶属[[中国人民解放军陆军|中国人民解放军陆军]]。是军官与士兵同校训练的多学科、多层次的综合院校，为[[中国人民解放军陆军航空兵|中国人民解放军陆军航空兵]]部队培养各类直升机应用人才<ref name="xinhuanet">{{Cite web |url = http://news.xinhuanet.com/mil/2009-06/30/content_11624515.htm |title =解放军报：陆军航空兵学院10岁了 陆军之鹰从这里展翅  |author =  |date =2009-06-30  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

== 专业设置 ==
学院驻地分布在“一市、两省、四地”<ref name=elyw/>。以教学训练为中心。教学训练基本包含飞行教学（空勤）与地面教学（地勤）两类培训，融合指挥、飞行、机务三大专业<ref name="xinhuanet"/> 。

2006年起，该院招收地方应届高中毕业生，学制四年，全部实行“2+2”培训模式，即：被该院录取的新学员直接到[[中国人民解放军国防科学技术大学|中国人民解放军国防科学技术大学]]（湖南长沙）报到，第一、二年级在国防科学技术大学学习基础课程，第三、四年级转入陆军航空兵学院学习专业课程，各科考试成绩合格，发给陆军航空兵学院本科毕业证书及工学学士学位证书，并由陆军航空兵学院负责分配到全军陆军航空兵部队<ref name=elyw/>。

2015年时，陆军航空兵学院设有基础部、飞行理论系、指挥系、直升机机械工程系、直升机机载设备系，共设19个专业，其中包括<ref name=elyw/>：
*航空兵中级指挥<ref name=elyw/>
*航空飞行与指挥<ref name=elyw/>
*飞行器系统与工程<ref name=elyw/>
*电气工程及其自动化<ref name=elyw/>
*导航工程<ref name=elyw/>
*兵器工程<ref name=elyw/>
