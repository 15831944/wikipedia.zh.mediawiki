'''中国人民解放军驻吉布提保障基地'''（'''The Chinese PLA Support Base in Djibouti'''），位于[[吉布提共和国|吉布提共和国]][[吉布提市|吉布提市]]，是[[中国人民解放军|中国人民解放军]]的驻外保障基地，隶属[[中国人民解放军海军|中国人民解放军海军]]，也是[[中国人民解放军|中国人民解放军]]的海外[[军事基地|军事基地]]之一。

== 沿革 ==
{{See also|中国人民解放军海军索马里护航行动}}

2008年，中国海军护航编队开始在亚丁湾巡逻护航，此后其一直在吉布提港进行补给和休整。2015年3月底至4月初，因[[也门内战_(2015年至今)|也门内战]]引发中国撤侨行动，吉布提在该行动中发挥了战略中转作用<ref name="jiang" />。

2015年5月9日，[[法新社|法新社]]报道称，[[吉布提总统|吉布提总统]][[伊斯梅尔·奥马尔·盖莱|盖莱]]表示欢迎中国在吉布提建设基地<ref name="jiang">{{Cite web |url = http://news.163.com/16/0117/02/BDGFLGA300014AED.html  |title = 中国将在吉布提建后勤保障设施 |author =  |date = 2016-01-17 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。

2015年，经中国政府与吉布提政府协商，决定中国人民解放军在吉布提共和国首都吉布提市建设保障基地并派驻必要的军事人员。主要为中国人民解放军在[[非洲|非洲]]及[[西亚|西亚]]方向参与护航、维和、人道主义救援等任务提供保障，并且便于更好地执行军事合作、联演联训、应急救援、撤侨护侨等任务，同有关方面一道维护国际战略通道安全<ref name=ifeng/>。

2015年12月31日，[[中华人民共和国国防部|中华人民共和国国防部]]召开例行记者会，国防部新闻事务局局长、新闻发言人[[杨宇军|杨宇军]]称，中国和吉布提经协商，就中方在吉布提建设保障设施事宜达成一致，该设施将主要用于中国人民解放军执行[[亚丁湾|亚丁湾]]和[[索马里|索马里]]海域护航、人道主义救援等任务的休整补给保障<ref>{{Cite web |url = http://military.people.com.cn/n1/2015/1231/c1011-28001421.html |title = 国防部证实中方将在吉布提建设后勤保障设施 |author =  |date =2015-12-31  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。

中国人民解放军驻吉布提保障基地是中国人民解放军首个海外基地，属于港口型综合保障基地，主要面向陆军、海军部队补给，为海军亚丁湾护航编队、非洲维和人员提供后勤保障支持。基地联合先遣组由陆海军官兵组成<ref name=cai>{{Cite web |url = http://www.guancha.cn/military-affairs/2018_02_18_447328.shtml |title = 中国军人爱种菜！吉布提官兵种出了大西瓜 |author =  |date = 2018-02-18 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>。

2017年7月11日，在纪念[[郑和下西洋|郑和下西洋]]而设的[[中国航海日|中国航海日]]<ref>{{Cite web |url =  http://www.guancha.cn/shenshiwei/2017_07_14_418200.shtml|title = 沈诗伟：到吉布提前，先认识下隔壁“老王” |author =  |date = 2017-07-14 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>上，基地成立暨部队出征仪式在中国[[广东省|广东省]][[湛江市|湛江市]]某军港码头举行。[[中国人民解放军海军|中国人民解放军海军]]司令员[[沈金龙|沈金龙]]宣读驻吉布提保障基地组建命令并授军旗，中国人民解放军海军政治委员[[苗华|苗华]]致辞。随后中国人民解放军驻吉布提保障基地官兵搭乘中国人民解放军海军[[井冈山舰|井冈山舰]]、[[东海岛船|东海岛船]]出发前往吉布提<ref name="ifeng">{{Cite web |url = http://news.ifeng.com/a/20170711/51413962_0.shtml |title = 解放军驻吉布提保障基地成立 驻地部队今出征 |author =  |date = 2017-07-11 |website =凤凰网  |language = |accessdate =  |quote =  }}</ref>。

2017年8月1日（[[中国人民解放军建军纪念日|中国人民解放军建军纪念日]]），中国人民解放军驻吉布提保障基地部队进驻营区仪式在基地营区举行，标志着基地正式投入使用。[[中国人民解放军海军|中国人民解放军海军]]副司令员[[田中_(中将)|田中]]，中国驻吉布提大使[[符华强|符华强]]，吉布提国防部部长巴赫敦、总参谋长扎卡里亚，有关国家驻吉布提使节、驻吉布提部队代表，以及中国人民解放军海军第26批护航编队官兵代表等300多人参加了仪式<ref>{{cite news |title = 解放军驻吉布提保障基地投入使用 |url = http://news.ifeng.com/a/20170801/51542682_0.shtml |accessdate=2017-08-01 |agency=CNR国防时空 |website = 凤凰网 |date=2017-08-01 }}</ref>。

驻吉布提保障基地官兵到达驻地后，除了日常开展各类训练外<ref>{{Cite web |url = http://news.sina.com.cn/o/2017-09-23/doc-ifymenmt6424727.shtml |title = 我驻吉布提保障基地首次组织实弹射击训练 |author =  |date =2017-09-23  |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.81.cn/jmtt/2017-11/28/content_7845356.htm |title = 我驻吉布提保障基地组织火炮实弹射击训练 |author =  |date =2017-11-28  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>，还发扬中国人民解放军自产农副业产品的传统，在荒漠上种出了[[西瓜|西瓜]]等各类蔬菜水果，丰富了官兵的饮食<ref name=cai/>。
