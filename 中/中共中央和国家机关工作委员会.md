'''中国共产党中央委员会中央和国家机关工作委员会'''，通称'''中央和国家机关工作委员会'''，简称'''中央和国家机关工委'''，是[[中国共产党中央委员会|中共中央]]派出机构，负责中央和国家机关党的工作。

中央和国家机关工作委员会主要负责统一组织、规划、部署中央和国家机关党的工作，指导中央和国家机关党的政治建设、思想建设、组织建设、作风建设、纪律建设，指导中央和国家机关各级党组织实施对党员特别是党员领导干部的监督和管理，领导中央和国家机关各部门机关党的纪律检查工作，归口指导行业协会商会党建工作等工作<ref name="fangan"/>。

==沿革==
2018年3月中共中央印发的《[[深化党和国家机构改革|深化党和国家机构改革]]方案》称，“组建中央和国家机关工作委员会。为加强中央和国家机关党的建设，落实全面从严治党要求，深入推进党的建设新的伟大工程，统一部署中央和国家机关党建工作，整合资源、形成合力，将中央直属机关工作委员会和中央国家机关工作委员会的职责整合，组建中央和国家机关工作委员会，作为党中央派出机构。”“不再保留[[中共中央直属机关工作委员会|中央直属机关工作委员会]]、[[中共中央国家机关工作委员会|中央国家机关工作委员会]]。”<ref name="fangan">{{Cite web|url=http://www.xinhuanet.com/2018-03/21/c_1122570517.htm|title=中共中央印发《深化党和国家机构改革方案》|accessdate=2018-03-21|date=2018-03-21|publisher=新华网}}</ref>

2018年3月25日，[[中国共产党中央委员会|中共中央]]宣布任命[[中国共产党中央政治局|中央政治局委员]]、[[中国共产党中央书记处|中央书记处书记]]、[[中共中央办公厅|中央办公厅]]主任、原[[中共中央直属机关工作委员会|中直机关工委]]书记[[丁薛祥|丁薛祥]]为中央和国家机关工委首任书记，原[[中共中央直属机关工作委员会|中直机关工委]]常务副书记[[孟祥锋|孟祥锋]]为分管日常工作的副书记（正部长级）<ref name="shuji">{{Cite web|url=http://www.ce.cn/xwzx/gnsz/szyw/201803/25/t20180325_28599227.shtml|title=丁薛祥兼任中央和国家机关工作委员会书记|accessdate=2018-03-25|author=|date=2018-03-25|publisher=中国经济网}}</ref>。

==职责==
根据《{{s|中国共产党中央委员会中央和国家机关工作委员会职能配置、内设机构和人员编制规定}}》，中共中央和国家机关工委承担下列职能：
{{columns-list|2|
# 统一组织、规划、部署中央和国家机关党的工作，提出加强和改进党的建设的意见和建议，研究制定工作规划，并抓好组织实施。
# 指导中央和国家机关党的政治建设、思想建设、组织建设、作风建设、纪律建设，把制度建设贯穿其中，深入推进反腐败斗争。
# 指导中央和国家机关各级党组织和广大党员学习马克思列宁主义、毛泽东思想、邓小平理论、“三个代表”重要思想、科学发展观、习近平新时代中国特色社会主义思想。
# 对中央和国家机关各级党组织、党员领导干部落实党建责任制、遵守政治纪律和政治规矩情况进行监督检查，并向党中央报告。
# 指导中央和国家机关各级党组织实施对党员特别是党员领导干部的监督和管理，及时向党中央反映各部门领导班子、领导干部的情况。
# 配合中央有关部门抓好中央和国家机关各部门领导班子思想政治建设，参与对党员领导干部民主生活会和中央各部委、国家机关各部委党组（党委）理论学习中心组学习的督促检查和指导，了解掌握情况，按规定报送情况报告。
# 督促指导中央和国家机关各部门机关党委按期换届，审批关于召开党员大会或党员代表大会的请示，审批中央和国家机关各部门机关党委和机关纪委领导班子的组成及书记、副书记的任免。
# 指导各级党组织加强基层组织建设，做好党员发展、教育和管理等工作。
# 领导中央和国家机关各部门机关党的纪律检查工作。
# 了解掌握中央和国家机关工作人员的思想状况，指导中央和国家机关各级党组织加强思想政治工作和精神文明建设。
# 领导中央和国家机关工会、共青团、妇联等群团组织的工作，指导中央和国家机关各级党组织做好党的群众工作和统一战线工作。
# 归口指导行业协会商会等全国性社会组织党建工作。
# 协同有关部门指导、规划、协调、督查、检查中央和国家机关干部教育培训工作，培训轮训中央和国家机关局处级党员领导干部和党务干部。
# 协助有关部门协调中央和国家机关各部门做好维护稳定、离退休干部和老龄等工作。
# 完成党中央交办的其他任务。
}}
