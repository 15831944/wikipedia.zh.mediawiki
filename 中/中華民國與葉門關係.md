'''中華民國與葉門關係'''是指[[中華民國|中華民國]]與[[葉門共和國|葉門共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對葉門的相關事務由'''{{link-en|駐沙烏地阿拉伯王國台北經濟文化代表處|Taipei Economic and Cultural Representative Office in the Kingdom of Saudi Arabia}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-195-bc8f3-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|葉門簽證政策|Visa policy of Yemen}}|{{link-en|葉門公民簽證要求|Visa requirements for Yemeni citizens}}}}
兩國[[公民|公民]]皆須申請[[簽證|簽證]]方可入境對方國家。

持有{{link-en|葉門護照|Yemeni passport}}的[[也门人口|葉門公民]]抵台參加國際會議、運動賽事、商展活動，可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將葉門歸類為'''紅色警戒'''：不宜前往，宜儘速離境（2015年11月11日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-351-6438c-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將葉門歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2017年12月26日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=55C204B531DCC159 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==軍事==
{{see also|大漠計畫}}
1979年開始，{{link-en|沙烏地阿拉伯政府|Council of Ministers of Saudi Arabia}}為因應[[葉門人民民主共和國|南]]、[[葉門阿拉伯共和國|北葉門]]的軍事衝突，與[[中華民國政府|中華民國政府]]商議，計畫由[[中華民國空軍|中華民國空軍]]每年派遣近百名現役[[飛行員|飛行員]]和[[機場地勤|地勤人員]]以沙烏地阿拉伯軍人的身份軍援葉門阿拉伯共和國（北葉門），此祕密行動是為「[[大漠計畫|大漠計畫]]」。直到1990年，[[葉門統一|南北葉門統一]]，大漠計劃才结束，中華民國空軍共在12年中派出700多位人員。<ref name="劉忠武">{{cite book

==經濟==
{{see also|臺灣經濟|葉門經濟}}
===貿易===
2013年，[[經濟部國際貿易局|經濟部國際貿易局]]挑選最具拓銷潛力的22國26城市推動執行「貿易尖兵任務」，[[中東#國家及地區|中東3國]]的[[新興市場|新興市場]]就包括[[葉門|葉門]]。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622045&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2013年10月30日 |title = 貿易尖兵勇闖22國26城市 新興市場商機無窮 |publisher = 經貿透視網}}</ref>

2018年，葉門是中華民國第133大貿易夥伴、第122大[[進口|進口]]夥伴、第118大[[出口貿易|出口]]夥伴。出口至葉門的金額為1,240萬6,683[[美元|美元]]，年減14.276%；自葉門進口的金額為560萬0,731美元，年增57.596%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）680萬5,952美元，年減37.669%。2017年，葉門是中華民國第131大貿易夥伴、第134大進口夥伴、第114大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
