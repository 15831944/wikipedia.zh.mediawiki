'''中国人民解放军西部战区总医院'''，本部位于[[四川省|四川省]][[成都市|成都市]][[金牛区|金牛区]]蓉都大道270号，隶属[[西宁联勤保障中心|西宁联勤保障中心]]，为[[三级甲等医院|三级甲等医院]]。

==简介==
医院的前身是[[第一次国共内战|第一次国共内战]]时期的[[红二军团|红二军团]]医疗队。[[第二次国共内战|第二次国共内战]]时期，随中国人民解放军第十八兵团进入四川，曾先后改为'''川西军区卫生学校教学医院'''、'''四川军区后方勤务部甲级陆军医院'''、'''西南军区第十七陆军医院'''、'''中国人民解放军第四十二陆军医院'''，1956年3月正式改编为'''中国人民解放军成都总医院'''<ref name=js>{{Cite web |url =  http://www.xn91.com/About|title =医院介绍|author =  |date =  |publisher = 中国人民解放军成都军区总医院 |language =  |accessdate =2017-06-26  |quote =  }}</ref>。

1994年，医院被[[中华人民共和国卫生部|中华人民共和国卫生部]]评为全军首批“[[三级甲等医院|三级甲等医院]]”。1995年，医院被中华人民共和国卫生部、[[联合国儿童基金会|联合国儿童基金会]]、[[世界卫生组织|世界卫生组织]]评为“爱婴医院”。2005年，医院被成都市评为“诚信示范医院”。2008年，医院被评为“全国百强优质服务示范医院”。2010年，医院被评为“全国改革创新医院”。2012年，医院被评为“全国医院文化建设先进单位”<ref name=js/>。

2009年，成都军区总医院口腔科整体迁至红星路一段1号，成立成都军区总医院附属口腔医院<ref>{{Cite web |url = http://cd.qq.com/a/20090618/000366.htm |title = 成都军区总医院附属口腔医院成立 |author =  |date = 2009-06-18 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军成都军区总医院由[[中国人民解放军成都军区|中国人民解放军成都军区]]转隶[[西宁联勤保障中心|西宁联勤保障中心]]，并更名'''中国人民解放军成都总医院'''<ref>{{Cite web |url = http://news.ifeng.com/a/20170624/51312870_0.shtml|title =西宁联勤保障中心成都总医院第一批医护人员抵茂县 |author =  |date =  2017-06-24 |publisher = 凤凰网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1730559 |title = 解放军成都、兰州总医院已隶属于西宁联勤保障中心 |author =  |date = 2017-07-11 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.sohu.com/a/206715531_373202|title =厉害了word中医院，与中国人民解放军成都总医院签署战略合作协议并举行揭牌仪式！ |author =  |date =2017-11-26  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

2018年，更名为'''中国人民解放军西部战区总医院'''<ref>{{Cite web |url = https://news.sina.cn/2018-11-23/detail-ihmutuec3034133.d.html?vt=4 |title = 解放军五大战区总医院全部成立 |author =  |date =2018-11-23  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。
