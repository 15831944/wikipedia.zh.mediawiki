'''中華民國與孟加拉國關係'''是指[[中華民國|中華民國]]與[[孟加拉人民共和國|孟加拉人民共和國]]之間的關係。兩國從未建立[[外交#外交上的承認|外交關係]]，但中華民國曾於孟加拉首都[[達卡|達卡]]設立[[實質大使館|具大使館性質]]的[[台北经济文化代表处|代表機構]]，後於2009年關閉。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，[[孟加拉國|孟加拉國]]當時屬於[[巴基斯坦|巴基斯坦]]的一省（[[東巴基斯坦|東巴基斯坦]]），由巴基斯坦作為[[主權國家|主權國家]]參與投票。]]
{{see also|中華民國外交|{{link-en|孟加拉國外交|Foreign relations of Bangladesh}}}}
2004年3月1日，[[中華民國政府|中華民國政府]]在孟加拉首都[[達卡|達卡]]設立[[實質大使館|具大使館性質]]的'''駐孟加拉代表處'''，<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2008/html/08-1-1.htm |title = 《中華民國97年外交年鑑》〈第二章 對外關係〉 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>但因{{Link-en|孟加拉國媒體|Media of Bangladesh|孟加拉媒體}}報導{{Link-en|孟加拉國外交部長|Minister of Foreign Affairs (Bangladesh)|該國外交部長}}堅持「[[一個中國|一個中國]]」政策及否認[[台灣|台灣]]在達卡設立官方機構之談話；以及[[中国驻孟加拉国大使列表|中華人民共和國駐孟加拉大使]]曾警告稱，台灣於達卡設處將危及[[中孟關係|兩國雙邊關係]]，盼{{Link-en|孟加拉國政府|Politics of Bangladesh}}立即關閉台灣駐處，以符合「一個中國」之主張。造成駐孟加拉代表處沒有實際運作性質，在2009年經過[[中華民國外交部|中華民國外交部]]評估後，決定裁撤。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=C12FBCFC70C2596F&sms=1B9564664158991B&s=7E3688014AB1C217 |author = 公眾外交協調會 |date = 2004年4月21日 |title = 中國大陸阻撓我國際空間事例（93年1月至3月事例）|publisher = 中華民國外交部}}</ref><ref>[http://news.ltn.com.tw/news/politics/paper/280641 自由時報：中國打壓．功能幾近零／駐孟加拉代表處 撤定了]</ref>目前對孟加拉的有關事務由'''[[駐印度台北經濟文化中心|駐印度台北經濟文化中心]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-122-07ecd-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]可以[[簽證#落地签证|落地簽證]]的方式入境孟加拉。]]
[[File:Visa_requirements_for_Bangladeshi_citizens.png|thumb]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|孟加拉國簽證政策|{{link-en|孟加拉國公民簽證要求|Visa requirements for Bangladeshi citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持有[[中華民國護照|中華民國護照]]與邀請函的[[中華民國國民|中華民國公民]]可於抵達首都[[達卡|達卡]]的[[沙阿賈拉勒國際機場|沙阿賈拉勒國際機場]]後辦理[[簽證#落地签证|落地簽證]]入境孟加拉，停留最多30天。經孟加拉轉機無需簽證，亦可申請落地轉機簽證，可獲72小時停留時間。孟加拉在中華民國尚無辦理簽證機構，須至[[香港特別地區|香港]]、[[泰國|泰國]]、[[印度|印度]]等第三地的{{link-en|孟加拉國駐外機構列表|List of diplomatic missions of Bangladesh|孟加拉駐外機構}}辦理。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-122-07ecd-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&sms=B75E93A3DAB23A99 |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|孟加拉國護照|Bangladeshi passport}}的{{link-en|孟加拉人口|Demographics of Bangladesh|孟加拉公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將孟加拉按地區歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往，與'''橙色警戒'''：避免非必要旅行（以上截至2017年6月1日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-137-a1195-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將孟加拉歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=B9FB5FFD117011F5 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

===貿易===
中華民國與孟加拉目前未簽署任何[[自由貿易協定|自由貿易協定]]（FTA），因此並未提供貿易優惠措施。<ref name="投資環境分析"/>

[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於孟加拉首都[[達卡|達卡]]設立'''臺灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#ASIA |title = 外貿協會全球據點|publisher = 中華民國對外貿易發展協會}}</ref>

2018年，孟加拉是中華民國第40大貿易夥伴、第71大[[進口|進口]]夥伴、第26大[[出口贸易|出口]]夥伴。出口至孟加拉的金額為11億4,113萬9,513[[美元|美元]]，年增14.652%；自孟加拉進口的金額為8,934萬6,295美元，年增6.294%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）10億5,179萬3,218美元，年增15.422%。2017年，孟加拉是中華民國第41大貿易夥伴、第69大進口夥伴、第28大出口夥伴。<ref name="中華民國進出口貿易統計">{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
[[中華民國經濟部|中華民國經濟部]]的「協助廠商全球布局計畫」將孟加拉列為重點投資國家。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref>

目前有近50家[[臺灣商人|台商]]於孟加拉投資，主要的投資類別以[[成衣|成衣]]、製[[鞋|鞋]]業、[[水產養殖|水產養殖]]、[[家具|家具]]等[[傳統產業|傳統產業]]為主，其他如[[磁磚|磁磚]]、[[聚氯乙烯|PVC]]板、[[自行車|自行車]]、[[食品|食品]]、成衣配件、[[帳篷|帳篷]]、運動[[背包|背包]]、紡[[紗|紗]]廠等。除製鞋業集中在[[吉大港|吉大港]]的CEPZ、KEPZ與Comilla EPZ[[工業區|工業區]]外，其餘產業皆位於首都[[達卡|達卡]]近郊。除食品、磁磚及PVC板外，其它產業以[[出口贸易|外銷]]為主。<ref name = Qna>[http://dhaka.taiwantrade.com.tw/note.jsp?id=1862&lang=zh_TW 達卡臺灣貿易中心-投資孟加拉Q&A]</ref>因為製鞋產業的[[群聚效應|群聚效應]]也帶來製鞋上下游[[供應鏈|供應鏈]]的聚集，因此臺商除了設立製鞋工廠外，亦有其他臺商將製鞋設備銷予非臺資廠商。<ref>[http://www.epochtimes.com/b5/10/5/8/n2900895.htm 大紀元時報：孟加拉台商製鞋中心成形 群聚續保競爭優勢]</ref>

目前在孟加拉的臺商投資規模最大者是位於達卡[[加工區|加工區]]的潤興紡織，員工1萬餘人；其次為[[吉大港|吉大港]]加工區[[寶成集團|寶成集團]]所屬的製鞋廠，員工約5,000人。中華民國投資孟加拉以製鞋產業為多，在經過[[中华人民共和国|中华人民共和国]]工資上漲、[[越南|越南]]缺工[[罷工|罷工]]以及[[2014年越南排華暴動|排華]]的問題後，諸多製鞋廠商除了將工廠遷往[[印尼|印尼]]外，也有選擇孟加拉者。<ref name="投資環境分析"/><ref name = Bus>[http://twbusiness.nat.gov.tw/countryPage.do?id=11&country=BD 中華民國經濟部 臺商資訊網：孟加拉]</ref>因孟加拉相較於前者勞力充沛，並且沒有缺工的問題。<ref name = light>[http://www.trademag.org.tw/content02.asp?id=645626&type=50 經貿透視：孟加拉的臺灣之光]</ref>

除此之外，也有臺資企業在孟加拉的[[股票上市|股票市場上市]]，華韋紡織即是其中一例，負責人同時也擔任孟加拉台商會會長。<ref name="投資環境分析"/><ref name = light/>不過由於政治因素，目前在孟加拉設立資本的臺資背景企業，多透過[[維京群島|維京群島]]、[[香港|香港]]等第三地間接投資孟加拉。<ref name = Bus/>

2015年4月，總計目前台商對孟加拉投資金額約2億9,042萬美元。<ref name="投資環境分析">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 投資環境分析 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

===組織===
2009年3月，成立「孟加拉台灣商會」，是亞洲台灣商會聯合總會成員。該國台商、幹部及眷屬約120餘人，多居住於達卡及吉大港。<ref>{{Cite web |url = http://www.boca.gov.tw/content.asp?CuItem=852&BaseDSD=13&CtUnit=16&mp=1 |title = 生活資訊 |publisher = 外交部領事事務局}}</ref>
