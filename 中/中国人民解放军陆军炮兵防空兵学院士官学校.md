'''中国人民解放军陆军炮兵防空兵学院士官学校'''，位于[[辽宁省|辽宁省]][[沈阳市|沈阳市]]，隶属[[中国人民解放军陆军炮兵防空兵学院|中国人民解放军陆军炮兵防空兵学院]]<ref name=xunli>{{Cite web |url =  http://www.81.cn/jwgz/2017-06/11/content_7634751.htm|title =2017年军校巡礼第十八站：陆军炮兵防空兵学院（附报考指南）  |author =  |date = 2017-06-11 |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref>。

== 沿革 ==
*1977年，创办'''中国人民解放军第二地面炮兵学校'''。
*1986年，升格'''中国人民解放军沈阳炮兵学院'''。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军总参谋部|中国人民解放军总参谋部]]转隶[[中国人民解放军陆军|中国人民解放军陆军]]<ref>{{cite web|title=沈阳炮兵学院已由原总参谋部转隶陆军|url=http://www.thepaper.cn/newsDetail_forward_1464002|publisher=澎湃新闻|accessdate=2016-05-24}}</ref>。
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，以解放军陆军军官学院、南京炮兵学院、防空兵学院、沈阳炮兵学院为基础组建[[中国人民解放军陆军炮兵防空兵学院|中国人民解放军陆军炮兵防空兵学院]]。沈阳炮兵学院撤编，在沈阳原址设'''中国人民解放军陆军炮兵防空兵学院士官学校'''<ref name=xunli/>。
