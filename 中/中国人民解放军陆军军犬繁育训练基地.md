'''中国人民解放军陆军军犬繁育训练基地'''，位于[[北京市|北京市]]，隶属[[中国人民解放军陆军|中国人民解放军陆军]]。

==简介==
[[抗日战争|抗日战争]]时期，[[中国共产党|中国共产党]]领导的军队首次运用军犬作战。当时[[八路军|八路军]]组建的军犬特工队突袭[[侵华日军|侵华日军]]营地、截获日军物资。1945年5月，八路军利用被俘后投诚的敌方专业技术人员和军犬，开展军犬繁殖和训练，培训出中共军队中第一代军犬工作者和军犬。[[中华人民共和国|中华人民共和国]]成立后，军犬正式装备部队<ref name=fengzh>冯凯旋、朱民、张坤平，“军犬王国”探秘，中国摄影家2004(9):18-19</ref>。

1949年，[[中国人民解放军|中国人民解放军]]开始创建军犬繁殖培训机构，最初建在[[北京|北京]]。1950年代分为两个军犬队，迁往[[黑龙江省|黑龙江省]]和[[云南省|云南省]][[昆明|昆明]]。到1960年代初，各[[大军区|大军区]]相继建立军犬队，先后繁殖培训了大批军犬，在国防建设中起到了重要作用<ref name=dfw/>。

[[文化大革命|文化大革命]]期间，在极左思潮影响下，过分强调“人的因素”，撤销了军犬编制，机构大部分解散，军犬濒于断绝。保留下来的黑龙江和昆明两个军犬队有名无实，军犬繁殖发展停滞<ref name=dfw/>。

1979年，[[邓小平|邓小平]]访问[[美国|美国]]，美方安排了军犬训练表演，邓小平回国后指示有关部门一定要发展中国军犬事业<ref name=fengzh/>。

1980年代中期，军犬被重新重视，军犬事业开始振兴<ref name=dfw/>。1991年大规模恢复军犬编制<ref>{{Cite web |url =http://news.cri.cn/gb/42071/2014/01/16/6351s4393408.htm  |title = 我军《当代军犬》杂志正式向社会公开发行 |author =  |date = 2014-01-16 |publisher = 国际在线 |language =  |accessdate =  |quote =  }}</ref>。到2000年代初，各[[大军区|大军区]]、[[中国人民解放军海军|海军]]、[[中国人民解放军空军|空军]]和[[中国人民解放军总后勤部|总后勤部]]、[[中国人民解放军总装备部|总装备部]]、[[武警|武警]]都已建有军犬训练队，全军每年繁殖培训2000余条军犬送往部队，边防连队、特警部队、军用仓库、保卫部门大多使用了军犬，全军重点仓库基本实现以犬助哨。2000年代初期，已有近万条军犬在部队服役<ref name=dfw/>。

1985年10月，'''中国人民解放军北京军区后勤部军犬训练队'''成立，隶属[[中国人民解放军北京军区|中国人民解放军北京军区]]后勤部<ref name=jj/>。最早一批军犬是原[[内蒙古军区|内蒙古军区]]军犬班的20条军犬<ref name=dfw/>。1990年，总参谋长[[迟浩田|迟浩田]]前来视察并题词：“当人民卫士，充顽敌克星。”随后，迟浩田率中国军事代表团出访[[泰国|泰国]]，中泰双方在军犬专业技术交流上达成共识。不久，于斌（该训练队领导，后来任北京军犬繁育训练基地主任）随总参谋部军犬工作小组乘专机两次赴泰国，接回数十条种犬，填补了中国人民解放军优质进口种犬的空白<ref name=yubin/>。1990年底，北京军区后勤部军犬训练队升格为'''中国人民解放军北京军区军犬训练队'''<ref name=cuijzh/>。

1995年9月，经[[中国人民解放军总参谋部|中国人民解放军总参谋部]]批准，成立'''中国人民解放军北京军犬繁育训练基地'''，仍隶北京军区<ref name=jj/>。1995年12月成立，并从[[昌平区|昌平]]迁到[[延庆区|延庆]]<ref name=cuijzh/>。2002年军委总部赋予该基地五项新任务。到2016年转隶陆军前夕，该基地主要任务有四项：（1）为全军培训中高级层次军犬专业士官和军犬专业技术骨干；（2）为全军引进、繁育优良种犬；（3）开展军犬疾病防控和饲料装具科研工作并指导全军；（4）为[[中国人民解放军总参谋部|总参谋部]]、[[中国人民解放军总装备部|总装备部]]和北京战区部队训练、补充各类工作用犬。该基地是全军唯一集军犬教学、繁育、科研、遂行多样化军事任务于一体的综合性基地<ref name=jj>{{Cite web |url =http://jz.chinamil.com.cn/zhuanti/content/2016-07/05/content_7134523.htm  |title =  北京军犬繁育训练基地简介|author =  |date = 2016-07-05 |publisher = 军报记者 |language =  |accessdate =  |quote =  }}</ref><ref name=cuijzh>王军、李英俊，托起中国军犬事业明天的太阳——走近崔金柱主任和他的军犬基地，中国工作犬业2007(8):15-17</ref>。该基地位于[[北京市|北京市]][[延庆县|延庆县]][[八达岭长城|八达岭长城]]风景区内，占地面积近3万平方米<ref>北京军犬繁育训练基地简介，中国工作犬业2007年08期</ref>。

1998年10月，基地党委带头在全军发起“全军军犬专业训练大比武”，在历时一周的比武中，基地参赛队夺得团体冠军、1项全能冠军、5项个人单项冠军，一举奠定了在全军13个军犬队中的龙头地位。1997年9月，基地和[[中国刑警学院|中国刑警学院]]联合举办军犬专业培训班。2002年首期军犬专业士官培训中专班开课。2004年8月基地开通军犬专业自动化信息管理局域网<ref name=yubin/>。

基地成立后，开展了“爱基地、爱岗位、爱军犬”的三爱特色教育，形成了“忠诚、使命、荣誉、奉献”的军犬精神。基地先后执行[[2008年北京奥运会|北京奥运会]][[2008年北京残奥会|残奥会]]、[[首都各界庆祝中华人民共和国成立60周年大会|首都60周年国庆大会]]、北京[[2014年中国APEC峰会|APEC峰会]]、[[习近平|习近平]]等[[党和国家领导人|党和国家领导人]]视察战区部队、[[纪念中国人民抗日战争暨世界反法西斯战争胜利70周年大会|纪念抗战胜利70周年阅兵]]的场地安检任务。同军地保卫及公安部门协同执行刑事侦查及现场搜索任务百余次。为[[国家地震局|国家地震局]]培训第一批地震救援犬及训导员，受训的多条工作犬先后参与[[海地地震|海地地震]]、[[太平洋|太平洋]][[海啸|海啸]]、[[汶川地震|汶川地震]]等自然灾害的救援。2008年，基地被[[中共中央|中共中央]]、[[中华人民共和国国务院|国务院]]表彰为“北京奥运会、残奥会先进集体”。截至2016年，基地已为全军培养各类军犬专业人员3000多人，配发种犬1000多条，军事工作用犬2000多条<ref name=jj/><ref name=wuyan/>。

2001年时，基地机关设有司令部、政治部、后勤部，有训练队和教导队，还有四个研究室：军犬疾病防治研究室，军犬饲料、装具研究室，军犬训练研究室，军犬繁育研究室<ref name=dfw/>。

2010年，北京军犬繁育训练基地从[[德国|德国]]、[[英国|英国]]、[[荷兰|荷兰]]等国引进近百只优质种犬，该基地的军犬队伍实现第五次换代<ref>{{Cite web |url =http://mil.news.sina.com.cn/2010-03-21/1343587601.html  |title = 北京军犬繁育训练基地实现军犬第五次换代 |author =  |date = 2010-03-21 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。

2014年，军委总部在[[云南|云南]][[昆明|昆明]]召开全军军犬业务工作会议，提出“建设规范化、训练实战化、管理精细化、保障体系化”的目标。军犬在应对[[毒品|毒品]]犯罪、暴力恐怖、[[地震|地震]][[海啸|海啸]]等非传统安全威胁，以及在执行敌后侦查、战场搜救、战场接力传输等任务中都将进一步发挥作用<ref name=jj/>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，该训练基地转隶[[中国人民解放军陆军|中国人民解放军陆军]]<ref name=wuyan>{{Cite web |url =http://www.mod.gov.cn/power/2016-07/06/content_4688575.htm  |title =  让“无言战友”有勇有力——陆军北京军犬繁育训练基地提高军犬战斗力侧记|author =  |date = 2016-07-06 |publisher =中华人民共和国国防部  |language =  |accessdate =  |quote =  }}</ref>。随后更名'''中国人民解放军陆军军犬繁育训练基地'''<ref name=shjd>{{Cite web |url =  http://www.sohu.com/a/202114242_428290|title =陆军军犬繁育训练基地官兵 掀起学习贯彻十九大精神热潮  |author =  |date =2017-11-03   |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。

2016年，“国际军事比赛-2016”在[[俄罗斯|俄罗斯]][[德米特罗夫|德米特罗夫]]举行的“忠诚朋友”竞赛中，由陆军军犬繁育训练基地为主组建的中国参赛队，在同[[俄罗斯|俄罗斯]]、[[白俄罗斯|白俄罗斯]]、[[哈萨克斯坦|哈萨克斯坦]]参赛队的竞争中夺得军犬两项个人赛、300米障碍、防卫勤务、军犬两项接力赛4个课目的第一名，获得该项冠军<ref>{{Cite web |url = http://finance.ifeng.com/a/20160811/14726135_0.shtml |title =我军参赛官兵屡创佳绩扬威国际赛场  |author =  |date =2016-08-11  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref name=py/>。
