'''中華民國與阿根廷關係'''是指[[中華民國|中華民國]]與[[阿根廷共和國|阿根廷共和國]]之間的關係。1945－1972年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]一案棄權，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[阿根廷|阿根廷]]。]]
{{see also|中華民國外交|{{link-en|阿根廷外交|Foreign relations of Argentina}}}}
{{see also|中国驻阿根廷大使列表{{!}}中華民國駐阿根廷外交官列表|阿根廷驻华大使列表{{!}}阿根廷駐中華民國外交官列表}}
1945年5月30日，兩國建立大使級[[外交#外交上的承認|外交關係]]，<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/11.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>首任中華民國駐阿根廷[[特命全權大使|大使]]於1946年4月13日呈遞到任[[國書|國書]]。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>分別於首都[[布宜諾斯艾利斯|布宜諾斯艾利斯]]設立'''中華民國駐阿根廷共和國[[外交代表機構#大使馆|大使館]]'''；於[[國民政府|國民政府]]首都[[重慶|重慶]]設立'''阿根廷共和國駐中華民國大使館'''。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1972年2月19日，阿根廷與中華民國斷交。8月10日，駐阿根廷大使館關閉。<ref name="館務資訊">{{Cite web |url = http://www.roc-taiwan.org/ar/post/3273.html |title = 館務資訊 |publisher = 中華民國駐外單位聯合網站}}</ref>

==經濟==
{{see also|臺灣經濟|阿根廷經濟}}
===背景===
阿根廷自1989年起實施[[貿易自由化|貿易自由化]]政策，大幅開放進口，兩國間的貿易開始逐年上升。<ref name="91年外交年鑑"/>

===貿易===
[[經濟部國際貿易局|經濟部國際貿易局]]在首都[[布宜諾斯艾利斯|布宜諾斯艾利斯]]設立'''駐阿根廷代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=215&pid=305863 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2018年，阿根廷是中華民國第62大貿易夥伴、第59大[[進口|進口]]夥伴、第52大[[出口贸易|出口]]夥伴。出口至阿根廷的金額為2億5,366萬0,935[[美元|美元]]，年減15.674%；自阿根廷進口的金額為1億2,543萬7,159美元，年增7.740%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1億2,822萬3,776美元，年減30.459%。2017年，阿根廷是中華民國第59大貿易夥伴、第61大進口夥伴、第47大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = 經濟部國際貿易局}}</ref>

阿根廷對中華民國實施反傾銷的項目有：[[腳踏車|腳踏車]]車架、腳踏車及[[機車_(公路)|機車]]用[[鋼|鋼]]絲及帶帽鋼絲、[[聚酯|聚酯]]加工絲紗、[[聚對苯二甲酸乙二酯|聚對苯二甲酸乙二酯]]（PET）、<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/ar/pdf/103-38Argentina.pdf |date = 2014年12月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>手操作[[扳手|扳手]]及扳鉗（調查中）。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

===投資===
[[臺灣商人|台商]]在阿根廷的實際投資金額與件數差異頗大。根據經濟部投資審議委員會統計，截至2013年，臺商對阿根廷的總投資金額為1,860萬美元，共2件；另根據阿根廷台灣商會估計，台商總投資金額為1億5,000萬美元，共350件。<ref name="投資環境簡介"/>

主要投資產業：自助[[超級市場|超市]]、[[食品產業|餐飲]]、[[国际贸易|進出口貿易]]、[[旅遊|旅遊]]服務、[[医疗卫生|醫療]]、[[汽車維修設備|汽車保養]]、[[汽車零組件列表|汽車零配件]]進口、[[電腦|電腦]]零配件進口及電腦組裝、[[沖洗|沖洗]]相館及[[時鐘|鐘錶]][[零售|零售]]、[[漁業|漁業]]捕撈、[[航運|海運]]、[[通訊|通訊]]、[[房地產|房地產]]、[[飯店|旅館]]、[[停車場|停車場]]、[[網路咖啡廳|網咖]]及[[電話|電話]]服務店、[[食品加工|食品加工]]、清潔用紙[[製造業|製造業]]、[[手套|手套]]製造業、[[塑膠分類標誌#塑膠類別表|塑膠品]]製造業及[[農業|農]]、[[畜牧业|牧業]]。部分第2代移民從事[[律師|律師]]、[[會計師|會計師]]、[[醫師|醫師]]等專業領域行業。<ref name="投資環境簡介"/>

目前成立的組織有：阿根廷台灣商會，會員近50名；阿根廷[[臺灣裔|台灣僑民]]聯合會。<ref name="投資環境簡介"/>

===漁業===
為解決中華民國籍[[漁船|漁船]]經常涉嫌違規跨越[[專屬經濟區|阿根廷海域]]遭致炮擊、扣押及罰款等問題，近年積極輔導台灣業者爭取與阿根廷進行漁業合作，自1993年開始，首度展開「{{link-en|包船|Chartering (shipping)|訂期租船}}」型合作，1994年又增加「[[合資|合資]]」型合作。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=139B784954EB399A |author = 公眾外交協調會 |date = 1994年5月23日 |title = 有關外電報導我國籍魷釣漁船「金源興號」|publisher = 中華民國外交部}}</ref>

==僑民==
{{see also|阿根廷華人|臺灣裔{{!}}海外臺僑|海外華人}}
台灣在阿根廷的[[僑民|僑民]]早期有2萬餘人，但經過數次阿根廷經濟危機，尤其是{{link-en|1998年－2002年阿根廷大蕭條|1998–2002 Argentine great depression|2001年發生經濟恐慌及暴動}}後，目前旅居阿根廷的台灣僑民約1萬餘人，大多數居住在首都[[布宜諾斯艾利斯|布宜諾斯艾利斯]]，其他則散居[[布宜諾斯艾利斯省|布宜諾斯艾利斯省]]及[[科爾多瓦省_(阿根廷)|科爾多瓦省]]等內地省分。目前約有35個台僑團體及宗教[[慈善組織|慈善團體]]，其中「阿根廷台灣僑民聯合會」又稱「阿根廷自由華僑聯合會」是綜合性僑團，並設有[[會館|會館]]，作為僑民集會活動場所。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2007/html/08-1-6.htm |title = 《中華民國96年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref><ref name="106年外交年鑑"/>

在僑民教育方面，有華興中文學校、新興中文學校、中觀僑聯中文學校、愛育學校及佛光書苑等。<ref name="106年外交年鑑"/>

在中華民國台北市舉行的[[2017年夏季世界大學運動會閉幕式|2017年夏季世界大學運動會閉幕式]]，[[2017年夏季世界大學運動會阿根廷代表團|阿根廷代表團]]有運動員手持、身披[[中華民國國旗|中華民國國旗]]進場，此舉遭到[[國際大學運動總會|國際大學運動總會]]發函關切，強調世大運的運動員行為不能帶有政治意涵，違反大會規則。<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/sports/20170901/1195486/ |author = 綜合報導 |date = 2017年9月1日 |title =【有片】世大運閉幕式 阿根廷代表團披我國旗進場被FISU發函警告 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref>會長[[奧雷格·麥斯汀|奧雷格·麥斯汀]]接受採訪時表示：「有隊伍揮舞臺灣旗幟，我們立即寄信給阿根廷代表團，要求他們不要帶有政治意涵的物品。」[[水球|水球]]選手[[帕特里西奧·馬洛尼|帕特里西奧·馬洛尼]]則在[[Facebook|Facebook]]以[[西班牙文|西班牙文]]表示：「不管他們怎麼說，但臺灣愛我們。」文中也感謝台北市民及選手村的[[志工|志工]]，還說「我們會想你們」。<ref>{{Cite web |url = http://news.ltn.com.tw/news/focus/paper/1132126 |author = 沈佩瑤 |date = 2017年9月3日 |title = 披我國旗遭FISU警告 阿根廷選手：台灣愛我們 |publisher = [[自由時報|自由時報]]}}</ref>
