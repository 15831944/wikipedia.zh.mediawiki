'''中华民族园'''，又称'''中华民族博物院'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[民族园路|民族园路]]1号，是以[[中华人民共和国|中华人民共和国]]各民族文化、文物、社会生活为主题的公园和[[人类学|人类学]]博物馆。由民营企业北京中华民族园有限公司运作。

==简介==
中华民族园（中华民族博物院）位于[[北京中轴线|北京城市中轴线]]北段西侧。北至[[北京四环路|北四环中路]]，南至[[北土城西路|北土城西路]]，东至[[北辰路|北辰路]]，西至[[北辰西路|北辰西路]]，占地面积50公顷。分为南区、北区，两区之间有[[民族园路|民族园路]]分隔。中华民族园是[[北京奥林匹克公园|北京奥林匹克公园]]的组成部分<ref name=women/>。

中华民族园1992年开始建设，1994年6月18日北园建成开放，2001年9月29日南园建成开放<ref name=women/>。该园是由海外华侨侨领雷学金（祖籍[[福建|福建]][[南安|南安]]，[[畲族|畲族]]人）发起建设<ref>{{Cite web |url = http://news.xinhuanet.com/yzyd/overseas/20141124/c_1113375880.htm |title =  侨领雷学金的民族情：建中华民族园给老外开眼|author =  |date = 2014-11-24 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。该园的建设获得了[[中共北京市委|中共北京市委]]、[[北京市人民政府|北京市人民政府]]的支持，以及[[中华人民共和国|中华人民共和国]]各少数民族自治地方政府和民众的帮助，海外华侨的资助。园区建成开放时，由[[中共北京市委|中共北京市委]]、[[北京市人民政府|北京市人民政府]]主持，举行开幕仪式<ref name=women>{{Cite web |url =http://www.emuseum.org.cn/node/17  |title = 关于我们 |author =  |date =  |publisher = 中华民族园 中华民族博物院 |language =  |accessdate = 2017-01-17 |quote =  }}</ref>。
[[File:China_Nationalities_Museum_-_panoramio.jpg|thumb]]]]
中华民族园是北京市“八五”建设和建国50周年大庆的重点工程。先后举办了北京市人民政府外国使节招待会、[[国家旅游局|国家旅游局]]“中国民俗旅游年”开幕式、北京国际旅游文化节闭幕式等活动<ref name=zj/>。该园还是中华人民共和国建国45周年、50周年、55周年大庆的中央会场<ref name=gkx>{{Cite web|url =http://www.emuseum.org.cn/node/827|title =中华民族园（中华民族博物院）致北京市旅游委的一封公开信|author =|date =2016-12-06|publisher =中华民族园 中华民族博物院|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170118051857/http://www.emuseum.org.cn/node/827|archivedate =2017-01-18}}</ref>。1999年[[中华人民共和国国庆|中华人民共和国国庆]]期间，中华民族园举办“民族之光”主题游园活动，12万人次观众前来参观。进入21世纪，中华民族园客流量下滑。2004年中华民族园曾举办首次民族园庙会，但游客很少。作为北京市著名的老牌主题公园，同时也是一家民营企业（北京中华民族园有限公司），中华民族园在21世纪初的经营情况并不乐观。2001年，国家旅游局公布首批国家4A级旅游景区，其中包括中华民族园。但在2016年12月，国家旅游局公布旅游景区整顿结果，中华民族园因质量等级复核分数未达标，被撤销国家4A级旅游景区资质<ref name=zj>{{Cite web |url = http://money.163.com/16/1207/22/C7NDUNSK002580S6.html |title = 中华民族园4A景区之路终结 |author =  |date =2016-12-07  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2001年8月3日，[[北京市文物局|北京市文物局]]批准北京中华民族园正式更名为“北京中华民族博物院”，成为一家大型[[人类学|人类学]]博物馆<ref>{{Cite web |url = http://www.bjww.gov.cn/2004/6-28/164.html |title = 北京市文物局2001年三季度文博事业大事记 |author =  |date = 2004-06-28 |publisher = 北京文博 |language =  |accessdate =  |quote =  }}</ref>。此后中华民族园对外使用“中华民族园”和“中华民族博物院”两个名称，并逐渐由主题公园向博物馆转型。

中华民族园（中华民族博物院）内建有中国56个民族的民族特色建筑，收藏有十多万件民族文物、展品，并举办各民族文化展览和表演。中华民族园（中华民族博物院）的展馆由主展馆和分馆组成。主展馆内设有：祖宗堂、车船驿站、绣房、书房、育儿房、礼房、度量衡、箱房、纺织坊、农家院、鞋帽包、中国明清两朝床具展览、清代民国时期脸盆架展览、清代衣架展。分馆共有56个，设在中国56个民族的56组建筑中，每个民族设一个博物馆，馆内展示该民族的历史文化和民俗。这56组民族建筑是中华民族园（中华民族博物院）的独特之处，极富特色。1998年北京中华民族博物院（中华民族园）获得世界建筑师大会组委会颁发“保护民族建筑奖”<ref>{{Cite web |url =  http://www.emuseum.org.cn/node/9|title = 分馆介绍 |author =  |date =  |publisher =  中华民族园 中华民族博物院|language =  |accessdate = 2017-01-17 |quote =  }}</ref>。
