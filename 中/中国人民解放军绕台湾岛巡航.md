'''中国人民解放军绕台湾岛巡航'''，或简称'''绕台岛巡航'''<ref name="a1"/>、'''绕岛巡航'''、'''绕台'''<ref>{{Cite web |url=http://taiwan.huanqiu.com/article/2019-04/14730121.html|title=解放军海空兵力绕台震动岛内 台媒：有警告台美意味 |accessdate=2019-04-17 |author=记者：崔明轩、范凌志|date=2019-04-16 |work= |publisher=环球网|language=Zh-hans }}</ref><ref name="a3"/>，是指[[中国人民解放军海军|中国人民解放军海军]]、[[中国人民解放军空军|空军]]飞机不定期绕行[[台湾岛|台湾岛]]的飞行活动。[[中华人民共和国国防部|中华人民共和国国防部]]将之定义为针对“[[台独|台独]]”所开展的实战化军事训练<ref name="a2"/>。中華民國（台灣）方面认为，绕岛巡航是对其的军事施压与骚扰<ref name="a4"/>。

==背景==
{{参见|台海现状|海峡两岸关系|海峡两岸关系史}}

[[两岸分治|两岸分治]]后，代表中國的[[中華民國政府|中華民國政府]]（後被簡稱'''台湾'''）的海、空軍军事实力略優於中国大陆（以北京為首都的中华人民共和国），初期中华民国方面一直掌控[[台湾海峡|台湾海峡]]的[[制空权|制空权]]，甚至中華民國空軍還數度飛越海峽中線執行空投文宣以及騷擾任務，中华民国军机多次深入中国大陆地区进行高空侦察，[[黑貓中隊|足迹基本覆盖中国大陆全境]]。但随着解放军防空能力的提高，中华民国军机对中国大陆的抵近侦察亦在1990年代结束。进入21世纪後，解放军军力有大幅度提升，而台灣方面卻在領導人李登輝的經營大台灣與戒急用忍的戰略下逐漸失去空優，双方攻守逐漸互换。21世纪之初，解放军的飞行训练雖然尚局限于内陆，出海训练的内容也很简单，少有协同。2004年后，解放军开始尝试军机大编队分批出海训练，训练水平獲得快速提升<ref>{{Cite web |url=http://www.chinesepress.com/Military/60978.html |title=空軍“繞島巡航”，不再是新聞|accessdate=2019-04-18 |author= |date=2018-01-07 |work= |publisher=华侨网|language=Zh-hant }}</ref>，而台灣方面只能在其編隊接近台灣的航空識別區時緊急派軍機升空監控。

==内容==

== 中华人民共和国官方态度 ==

=== 2017年12月，解放军空军首次使用“绕岛巡航”一词 ===

2017年12月12日，[[中国人民解放军空军|中国人民解放军空军]]新闻发言人[[申进科|申进科]]首次使用“绕岛巡航”一词，他表示，空军前一天出动轰-6K、苏-30、歼-11和侦察机、预警机、加油机等“绕岛巡航”，锤炼提升了维护国家主权和领土完整的能力。此前“绕岛巡航”指[[中国关于钓鱼岛之言行#中日建交後|中国海警编队绕钓鱼岛巡航]]<ref>{{Cite web |url=http://news.ifeng.com/a/20171213/54111462_0.shtml |title=远航988天后，空军首次使用“绕岛巡航”一词|accessdate=2019-04-18 |author=记者：岳三猛 |date=2017-12-13 |work= |publisher=凤凰网，来源：法制晚报·看法新闻|language=Zh-hans }}</ref>。

=== 2018年4月，国防部首次承认绕岛巡航 ===

[[2019年4月臺灣|2018年4月]]，[[中华人民共和国国防部|中华人民共和国国防部]]例行记者会，发言人[[吴谦_(解放军)|吴谦]]首次承认解放军进行绕岛巡航训练，並稱此舉為[[中国人民解放军|中国人民解放军]]针对“台独”分裂活动所开展的海上方向实战化军事训练，内容为出动多型多架[[战机|战机]]绕[[台湾岛|台湾岛]]巡航<ref name="a2">{{Cite web|url=http://www.xinhuanet.com/politics/2018-04/26/c_1122749356.htm|title=国防部新闻发言人就空军绕岛巡航、中印两军关系等答记者问-新华网|accessdate=2018-04-26|work=www.xinhuanet.com}}</ref>。发言人吴谦表示，“采取的一系列行动，针对的是岛内‘台独’势力及其活动，为的是防止台湾民众福祉因‘台独’图谋而受损害<ref>{{Cite web|url=http://www.mod.gov.cn/shouye/2018-04/26/content_4811190_2.htm|title=2018年4月国防部例行记者会文字实录 - 中华人民共和国国防部|accessdate=2018-04-26|last=刘上靖|work=www.mod.gov.cn}}</ref>。”

同一时间，对解放军空军「绕岛巡航」是否是是针对中华民国[[行政院|行政院]]院长[[賴清德|赖清德]]的“台独”言论以及“台独”分裂活动发出的警告的提问，[[中华人民共和国国务院台湾事务办公室|中华人民共和国国务院台湾事务办公室]]新闻发言人[[马晓光|马晓光]]表示，“‘台独’分裂活动是台海和平稳定的最大现实威胁<ref>{{Cite web|url=http://news.haiwainet.cn/n/2018/0419/c3541083-31301970.html|title=“绕岛巡航”是对“台独”言论警告？国台办回应 - 资讯 -|accessdate=2018-04-26|last=L_204342|work=news.haiwainet.cn}}</ref>。”[[中国人民解放军空军|解放军空军]]则称「绕岛巡航」是“用战机航迹丈量祖国的大好河山”，锤炼提升解放军维护国家主权和领土完整的能力<ref>{{Cite web |url=https://www.voachinese.com/a/%E4%B8%AD%E5%9B%BD%E5%86%8D%E5%8A%A0%E5%8E%8B-%E7%A7%B0%E7%BB%95%E5%B2%9B%E5%B7%A1%E8%88%AA%E6%98%AF%E4%BB%A5%E6%88%98%E6%9C%BA%E4%B8%88%E9%87%8F%E5%8F%B0%E6%B9%BE-20180426/4367148.html|title=中国再加压 称绕岛巡航是以战机丈量台湾 |accessdate=2019-04-03 |author=钟辰芳 |date=2018-04-27 |work= |publisher=美国之音网站|language=Zh-hans }}</ref>。

==中华民国（台湾）态度==

2017年，[[中华民国|中华民国]][[陆委会|陆委会]]发表声明表示，“相关动态政府均全程监侦掌握并妥慎应处。中共当局刻意操作对台施压与骚扰，企图挑起两岸及区域紧张，台湾人民知之甚明、不会接受。唯有放弃武力恐吓、正视两岸分治现实、透过务实沟通对话，双方分歧的解决才有可能<ref name="a4">{{Cite web|url=https://www.mac.gov.tw/News_Content.aspx?n=05B73310C5C3A632&sms=1A40B00E4C745211&s=EE35D203F7498123&Create=1|title=中華民國行政院大陸委員會|accessdate=2018-04-26|date=2017-02-24|last=行政院大陸委員會|work=行政院大陸委員會}}</ref>。”

2019年4月15日，解放军飞机再度绕台。当日，在「[[台湾关系法|台湾关系法]]＆ AIT@40：40年友谊庆祝酒会」上，[[中华民国总统|中华民国总统]][[蔡英文|蔡英文]]表示，在充满变化的世界里，少数不变的是台美共同对印太地区和平、安全、稳定的承诺。但近期共军增加在区域内的军事活动，包括今天军机飞过[[宫古海峡|宫古海峡]]、[[巴士海峡|巴士海峡]]，都挑战台海与区域稳定。[[AIT|AIT]]主席[[莫健_(美國外交官)|莫健]]则回应，中国这样的行动的确对台湾有威胁及胁迫，也影响到台海稳定情势，但不确定这样的行动是否跟当天AIT活动有关。军机绕台的确威胁到区域稳定及两岸关系，也无法赢得台湾人民的民心<ref name="a3">{{Cite web|url=https://www.chinatimes.com/cn/realtimenews/20190415003481-260407?chdtv|title=蔡总统： 共机绕台挑战台海稳定 莫健：无法赢得台民民心|newspaper=中时电子报|accessdate=2019-04-17|date=2019-04-15|language=zh}}</ref>。
