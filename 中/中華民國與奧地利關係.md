'''中華民國與奧地利關係'''是指[[中華民國|中華民國]]與[[奧地利共和國|奧地利共和國]]之間的關係。1913－1971年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。雖無邦交，但雙方均致力維護[[民主|民主]]、[[自由|自由]]及[[人權|人權]]等價值，兩國在經貿、旅遊、藝文及學術等關係也往來密切。<ref name="駐館與駐地關係">{{Cite web |url = http://www.roc-taiwan.org/at/post/12.html |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[奧地利|奧地利]]。]]
{{see also|中華民國外交|{{link-en|奧地利外交|Foreign relations of Austria}}}}
{{see also|中国驻奥地利大使列表{{!}}中華民國駐奧匈帝國／奧地利外交官列表|奥地利驻华大使列表{{!}}奧匈帝國／奧地利駐中華民國外交官列表}}
1912年，中華民國成立後，向[[奧匈帝國|奧匈帝國]]派駐外交代表。

1913年10月6日，奧匈帝國[[外交#外交上的承認|外交承認]]中華民國，於[[維也納|維也納]]設立'''中華民國駐奧匈帝國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1917年8月14日，[[第一次世界大戰|第一次世界大戰]]爆發後，中華民國對奧匈帝國[[宣戰|宣戰]]，並召回公使。

1919年，[[巴黎和會_(1919年)|巴黎和會]]召開後，中華民國與[[奧地利第一共和國|奧地利第一共和國]]簽署[[和平条约|和約]]。9月10日，恢復於維也納設立的公使館，改名為'''中華民國駐奧地利公使館'''，並恢復派駐公使。

1938年3月13日，[[德奧合併|德國吞併奧地利]]後，中華民國撤除駐奧地利公使館，改設立'''中華民國駐維也納[[領事館|總領事館]]'''，並改派駐[[領事|總領事]]。

1946年7月6日，[[第二次世界大戰|第二次世界大戰]]結束後，中華民國恢復承認奧地利。

1947年9月2日，設立'''中華民國駐奧地利代表公署'''。

1950年，中華民國駐奧地利公使[[沈士華|沈士華]]離任後，就未再派駐公使。並以節省經費、業務量少為由裁撤駐奧地利使館，但仍視邦交存在；1965年2月，[[联邦欧洲与国际事务部|奧地利外交部]]遠東事務主管梅葉哈鼎（Anton Mayr-Harting）抵台訪問時亦認為兩國間的邦交並未斷絕，只是暫時停頓。<ref name="演變時程">{{Cite web |url = https://issuu.com/kai-chihchang/docs/1949-2017__________________1_ |author = 黃剛 陳純一 |date = 2018年1月 |title = 世界各國與臺海兩岸政府之使領或實質關係演變時程 1949~2017 |publisher = [[國立政治大學國際事務學院|國立政治大學國際事務學院]]國際法學研究中心}}</ref>

1971年5月28日，奧地利與中華人民共和國建立[[特命全權大使|大使級]]外交關係，並與中華民國斷交，<ref name="adl">{{Cite web |url = http://www.chinaembassy.at/chn/adl/t1029790.htm |title=中奥双边关系 |work= |publisher=中華人民共和國駐奧地利共和國大使館 |date=2013-04-10 }}</ref>中華民國政府表示深以為憾。<ref name="演變時程"/>

1971年10月25日，在決定[[中國與聯合國#代表權問題|中國代表權]]問題的《[[聯合國大會2758號決議|聯合國大會2758號決議]]》文中，奧地利連同大多數歐洲國家，支持中華人民共和國取代中華民國的[[聯合國會員國列表|「中國」席位]]。

1999年9月21日，[[九二一大地震|台灣發生大地震]]，奧地利派出10人搜救隊抵臺救災。<ref name="內政部消防署統計中時新聞資料庫，1999/10/5">[內政部消防署統計][[中國時報|中時新聞]]資料庫，1999/10/5</ref><ref>[國際搜索隊陸續離台]《[[工商時報|工商時報]]》，1999/09/24</ref>

===貿易===
[[經濟部國際貿易局|經濟部國際貿易局]]於首都[[維也納|維也納]]設立'''駐奧地利代表處經濟組'''；<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=216&pid=308025 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>奧地利則在首都[[台北市|台北]]設立'''商務代表辦事處'''。<ref name="商務代表辦事處">{{Cite web |url = https://www.mofa.gov.tw/OfficesInROC_Content.aspx?n=8CEB2B5F5436B997&sms=8EBFADC1592C7BFE&s=6EA825DBF40D5319&os=2421206077EBB11F |title = 奧地利商務代表辦事處 |publisher = 中華民國外交部}}</ref>

2018年，奧地利是中華民國第42大貿易夥伴、第40大[[進口|進口]]夥伴、第42大[[出口贸易|出口]]夥伴。出口至奧地利的金額為4億3,985萬8,242[[美元|美元]]，年增5.522%；自奧地利進口的金額為6億5,809萬4,302美元，年增25.978%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）2億1,823萬6,060美元，年增106.764%。2017年，奧地利是中華民國第43大貿易夥伴、第42大進口夥伴、第41大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =經濟部國際貿易局}}</ref>

===投資===
根據經濟部投資審議委員會統計，1990年－2013年，[[臺灣商人|臺商]]對奧地利的總投資金額為1,538萬美元，計有49件。<ref name="投資環境簡介"/>

在奧地利從事[[飲食|餐飲]]業的臺商有10餘家，大多將中國餐館轉型成[[日本料理|日本料理]]、[[中國菜|中華料理]]或{{link-en|東南亞料理|List of Asian cuisines#Southeast Asian cuisine}}等「{{link-en|亞洲式料理|Asian cuisine}}餐館」以區隔市場。除了餐飲業外，臺商多半於1990年以後成立，企業規模屬[[中小型企業|中小型]]或[[辦事處|辦事處]]型態，具知名度的廠商包括[[中華航空|中華航空]]、[[長榮航空|長榮航空]]、[[長榮海運|長榮海運]]、兆輝[[石油化學|石化]]、[[宏碁電腦|宏碁電腦]]、[[明基電通|明基電通]]、[[鴻海科技集團|鴻海科技集團]]等。經營產業包括[[資訊科技|資訊]]、[[通訊技術|通訊]]、[[電子產品|電子]]、[[運輸|運輸]]、[[家電|家電]]、[[食品產業|食品製造]]、[[国际贸易|進出口貿易]]、[[旅行社|旅行社]]及其他[[第三產業|服務業]]等。<ref name="投資環境簡介"/>

近年來，陸商與港商在奧地利有多起大型投資案，對臺商形成經營上的競爭與壓力。陸商大企業逐漸摒棄代理商[[通路行銷|通路]]，直接前往奧地利成立[[分公司|分公司]]，戰略思考與早期臺商在維也納設立據點大致相同，以此地作為前進[[東歐|東]]、[[西歐|西歐]]市場跳板。但不同於臺商的是，陸商在奧地利成立分公司的規模遠大於10多年前的臺商規模，其經營方式也日趨靈活與創新。<ref name="投資環境簡介"/>

目前成立的臺商組織有奧地利臺灣商會。<ref name="投資環境簡介"/>

==交流==
===僑民===
{{see also|臺灣裔{{!}}海外臺僑|海外華人}}
奧地利為[[德語#地理分布|德語系國家]]，移民管制日趨嚴格，每年僅有少數配額，且需通過[[德語|德語]]及在奧地利生活常識測驗，因此外國人移民奧地利的困難度越來越高，近年也少有中華民國新進僑民。但由於奧地利是[[西方文化|歐洲文化]]重鎮之一，前往該國學習[[西方美術史|藝術]]與[[西方音樂史|音樂]]的臺灣[[留学|留學生]]不少，學成之後也有留在當地定居者。除1970－1980年代的早期僑民外，大多數居住在奧地利的則是前往當地學習音樂的留學生，或早期臺灣廠商派駐奧地利的工作人員。根據[[中華民國僑務委員會|中華民國僑務委員會]]統計，旅居奧地利的[[華人|華人]]總數約有2萬人，而[[臺灣裔|中華民國僑民]]約2千餘人，其餘大多數為來自[[中華人民共和國|中華人民共和國]]或[[中南半島|中南半島]]國家的華人。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/at/pdf/103-64Austria.pdf |date = 2014年12月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

===文化===
{{see also|台灣文化|奧地利文化}}
2008年2－5月，[[國立故宮博物院|國立故宮博物院]]與奧地利[[藝術史博物館|藝術史博物館]]於[[維也納|維也納]]合辦《物華天寶展》。<ref>{{Cite web |url = https://www.npm.gov.tw/exh97/masterpiece/index1_ch.html |author =  |date = 2008年2月 |title = 物華天寶—台灣國立故宮博物院精品展 |publisher = [[國立故宮博物院|國立故宮博物院]]}}</ref>

奧地利以其藝文及自然風景著稱，是[[中華民國國民|台灣遊客]]喜愛的旅遊國家之一。2010年，前往奧地利有26,081人次，赴台則有5,806人次。2015年，赴該國觀光、商務、遊學超過10萬人。截至2016年，[[中華航空|中華航空]]、[[長榮航空|長榮航空]]每週共有7航班[[臺灣桃園國際機場|臺北]]↔[[維也納國際機場|維也納]]。<ref name="駐館與駐地關係"/>奧地利的[[管弦樂團|交響樂團]]、[[合唱|合唱團]]亦曾多次來台演出。
