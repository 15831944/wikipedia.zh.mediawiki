'''中国关心下一代工作委员会'''，简称'''中国关工委'''<ref name="条例"/>、'''中关工委'''，是[[中華人民共和国|中国大陆]]一个主張关心、教育、培养全国各族青少年健康成长、以[[离休|离]][[退休|退休]]老干部为主体全国群众性工作组织。工作上与各地区、各部门关工委是指导关系。<ref>{{cite web |title=中国关心下一代工作委员会简介 |url=http://www.zgggw.gov.cn/zuzhijiegou/1914.html |publisher=中国关工委 |accessdate=2018-11-24 |date=2015-06-06}}</ref>经费由财政全额拨付<ref name="条例"/>。中国关工委党总支部委员会被列入[[国家机关事务管理局|国家机关事务管理局]]直属党组织<ref name="a">{{cite news |title=我局各直属党组织认真组织收听收看党的十九大开幕会 |url=http://www.ggj.gov.cn/xwzx/ggjxw/201710/t20171020_8301.htm |accessdate=2018-11-24 |work=[[国家机关事务管理局|国家机关事务管理局]] |date=2017-10-20}}</ref>。

==历史==
1990年2月经[[中共中央|中共中央]]、[[中华人民共和国国务院|国务院]]批准成立。

2008年3月6日，经[[中共中央组织部|中共中央组织部]]研究并上报中共中央国务院，同意[[王丙乾|王丙乾]]不再担任中国关工委主任职务，由[[顾秀莲|顾秀莲]]接任中国关工委主任<ref>{{cite web |title=关于转发中关工委《关于顾秀莲同志任 中国关心下一代工作委员会主任的通知》的通知 |url=http://www.qsn365.com/qsn365/articles/43965 |publisher=浙江省关心下一代工作委员会 |accessdate=2018-11-24}}</ref>。

==组织==
中国关工委设名誉主任、顾问若干名，主任一名，常务副主任、副主任若干名，特邀副主任若干名，秘书长一名，副秘书长若干名，实行主任负责制。领导成员由主任、常务副主任、副主任、秘书长组成，每年由主任主持召开一次领导成员全体会议。主任办公会议是中国关工委的议事机构，成员由主任、常务副主任、秘书长、副秘书长、办公室主任组成，每半个月召开一次。名誉主任、顾问、特邀副主任会议、全国关心下一代工作会议、有关部委联席工作会议每年召开一次，全国关心下一代工作先进集体、先进个人的表彰大会每五年召开一次。<ref name="条例">{{cite web |title=中国关心下一代工作委员会工作条例 |url=http://www.zgggw.gov.cn/zdjs/1920.html |publisher=中国关工委 |accessdate=2018-11-24}}</ref>

各省、市、县和国务院各部委办一般都设立自己的关工委，例如教育部关心下一代工作委员会、北京市关心下一代工作委员会等，一般都以“五老”（老干部、老战士、老专家、老教师、老模范）为主体<ref>{{cite web |title=关心下一代工作 |url=http://www.gd.gov.cn/govinc/nj2010/08shsh/080310.htm |publisher=[[广东省人民政府|广东省人民政府]] |accessdate=2018-11-24}}</ref>。地方关工委主任多由当地退休正职领导担任。
