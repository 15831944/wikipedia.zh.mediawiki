'''中華民國與中非共和國關係'''是指[[中華民國|中華民國]]與[[中非共和國|中非共和國]]（通稱'''中非'''）之間的關係。兩國於1962－1998年有官方[[外交#外交上的承認|外交關係]]，期間多次斷交與復交。1998年第3次斷交後，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對中非的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-233-7998a-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[中非共和國|中非共和國]]。]]
{{see also|中華民國外交|{{link-en|中非共和國外交|Foreign relations of the Central African Republic}}}}
{{see also|中華民國駐中非共和國大使列表{{!}}中華民國駐中非共和國外交官列表|中非共和國駐中華民國大使列表{{!}}中非共和國駐中華民國外交官列表}}
1962年4月13日，建立大使級[[外交#外交上的承認|外交關係]]。於首都[[班基|班基]]設立'''中華民國駐中非共和國[[外交代表機構#大使馆|大使館]]'''，但沒有派駐[[特命全權大使|大使]]，而是[[代辦|代辦]]。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-3.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1964年11月5日，與中華民國首次斷交。

1968年5月8日，恢復大使級外交關係，並派駐大使。<ref name="86年外交年鑑"/>

1976年8月23日，與中華民國第二次斷交。

1991年7月8日，再度恢復大使級外交關係，並'''互設大使館'''、互派大使。<ref name="86年外交年鑑"/>

1992年5月，中非總統[[安德烈·科林巴|科林巴]]抵台訪問。<ref name="86年外交年鑑"/>

1994年10月，中非總統[[安热-费利克斯·帕塔塞|巴達塞]]率領外交暨法語國家事務部長{{link-fr|貝達雅-恩加羅|Simon Bédaya-Ngaro}}、財政計畫暨國際合作部長竇古納（Emmanuel Dokouna）、國會關係部長賈巴（Geraro Gaba）等官員抵台參加[[中華民國國慶|中華民國國慶]]活動；<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=84EBBA1B00228CDF |author = 公眾外交協調會 |date = 1994年10月7日 |title = 欣逢中華民國八十三年雙十國慶 |publisher = 中華民國外交部}}</ref>12月，中華民國外交部長[[錢復|錢復]]訪問中非。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=E253BF8E1FAF978D |author = 公眾外交協調會 |date = 1994年12月16日 |title = 外交部長錢復已於本（十二）月十四日抵達中非共和國作為期四天之訪問 |publisher = 中華民國外交部}}</ref>

1997年10月21－23日，[[中国常驻联合国代表列表|中華人民共和國駐聯合國常任代表]][[秦華孫|秦華孫]]，以[[聯合國安全理事會|聯合國安全理事會]]輪值主席身分，率團赴中非查訪[[維和部隊|維和部隊]]任務執行情形。[[中華民國外交部|中華民國外交部]]指示大使館隨時掌握情勢演變，並提醒{{link-en|中非共和國政府|Politics of the Central African Republic|中非政府}}與之接觸，不得損及與中華民國之關係，外交部常務次長鄭文華也前往中非訪問。<ref name="86年外交年鑑"/>

1997年11月，中非財政部長[[阿尼塞-喬治·多洛蓋萊|道魯格]]、外交部次長兼[[常駐聯合國代表|常駐聯合國代表]]費南德斯（Fernández）抵台訪問。<ref name="86年外交年鑑"/>

1998年1月29日，與中華民國第三次斷交、關閉大使館。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content_M_2.aspx?n=5028B03CED127255&sms=5ED24855AD8E6C58&s=D7D460B5F6F6630F |author = 公眾外交協調會 |date = 1998年1月29日 |title = 中華民國外交部對於中非共和國於本（八十七）年元月廿九日與中共簽署所謂「復交」聯合公報發表聲明 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境中非共和國。]]
[[File:Visa_requirements_for_Central_African_Republic_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|{{link-en|中非共和國簽證政策|Visa policy of Central African Republic}}|{{link-en|中非共和國公民簽證要求|Visa requirements for Central African Republic citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往中非駐[[南非|南非]][[外交代表機構#大使馆|大使館]]申請簽證。因該國有重大[[流行病|流行疾病]]，入境須出示國際[[疫苗接種|疫苗接種]]證明（黃皮書）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-233-7998a-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref name="生活資訊">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-233-7998a-05-1.html |title = 生活資訊 |publisher = 外交部領事事務局}}</ref>

持{{link-en|中非共和國護照|Central African Republic passport}}的{{link-en|中非共和國人口|Demographics of the Central African Republic|中非公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國衛生福利部|中華民國衛生福利部]]的國際間旅遊疫情建議等級表將中非共和國歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=1E939BD289DD3375 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|中非共和國經濟}}
===背景===
中非共和國交通建設落後、運費昂貴，不利對外貿易，且國際收支全賴[[外援|外援]]，兩國幾無貿易可言。<ref name="86年外交年鑑"/>

===貿易===
2017年，中非共和國是中華民國第207大貿易夥伴、第184大[[進口|進口]]夥伴、第208大[[出口贸易|出口]]夥伴。出口至中非的金額為13萬2,231[[美元|美元]]，年減24.718%；自中非進口的金額為15萬3,289美元，年減62.747%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）2萬1,058美元，年減91.071%。※2016年，中非共和國是中華民國第200大貿易夥伴、第165大進口夥伴、第208大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

==援助==
===醫療===
1991年11月29日，成立駐中非醫療團，主要任務為負責友誼醫院及總統府醫療所的門診工作。<ref name="86年外交年鑑"/>

===農業===
1992年2月，中華民國派出農業考察團前往中非，8月23日，派駐[[農業技術|農業技術]]團，下設5個分團，以協助[[水稻|水稻]]、[[蔬菜|蔬菜]]、[[葫蘆科|瓜果]]、旱作栽培推廣。為加強合作關係，於1994年5月設立[[翁貝拉-姆波科省|姆波科]]農漁牧示範訓練推廣中心，增加[[水產養殖|水產養殖]]、[[畜牧|畜牧]]等計畫。1996年5月，{{link-en|班基協議|Bangui Agreements|中非發生兵變}}，農技團設備及財務損失嚴重。<ref name="86年外交年鑑"/>

===事件===
兩國在邦交期間，[[中國輸出入銀行|中國輸出入銀行]]（以下簡稱輸銀）提供[[貸款|貸款]]資助中非共和國。斷交後，輸銀於2015年12月向[[美國紐約南區聯邦地區法院|美國紐約南區聯邦地區法院]]提告{{noteTag|中國輸出入銀行在[[紐約|紐約]]設有辦公室，過去貸款給[[中華民國外交#邦交國|中華民國的邦交國]]時由輸銀出面與對方國家銀行簽約，合約可以適用{{link-en|紐約州法律|Law of New York}}，對中華民國比較有保障。}}，求償包括未償還貸款與[[利息|利息]]。法院於2017年1月裁定中非共和國須償還1億5,490萬美元給輸銀。<ref>{{Cite web |url = https://udn.com/news/story/5/2375838 |author = 田思怡 |date = 2017年3月31日 |title = 與台灣斷交的非洲國家小心了 銀行打官司討回64億貸款 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref><ref>{{Cite web |url = http://www.cw.com.tw/article/article.action?id=5081742 |author = [[德國之聲|德國之聲]] |date = 2017年4月2日 |title = 與台灣斷交 要付出代價！ |publisher = [[天下雜誌|天下雜誌]]}}</ref>
