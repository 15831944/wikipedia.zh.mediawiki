'''中国人民解放军联勤保障部队第九〇四医院'''，现为[[无锡联勤保障中心|无锡联勤保障中心]]属下的一所医院，由原中国人民解放军第一〇〇医院（苏州长城医院）、中国人民解放军第一〇一医院（无锡市太湖医院）和中国人民解放军第一〇二医院（常州和平医院）合并组建而成，成立于2018年11月5日，以原一〇一医院为本部<ref name=merge1>{{Cite web |url = http://www.wxrb.com/news/wxxw/201811/t20181106_1434698.shtml|title = 再见！解放军101医院 陪伴了无锡人64年,合并后更大更强 |author =  |date =  2018-11-06|publisher = 无锡新传媒网|language =  |accessdate =  |quote =  }}</ref>。

==沿革==
===原一〇〇医院===
[[File:Gate_of_the_No.100_Hospital_of_the_PLA.jpg|缩略图]]
[[File:In-Patient_Building_of_the_No.100_Hospital_of_the_PLA.jpg|缩略图]]题写），大楼顶部的“100医院”字样亦被拆除]]
1949年3月，在江苏[[泰州|泰州]]组建'''苏南军区卫生部直属医疗队'''。1950年10月移驻苏州市，改编并扩建成'''苏南军区直属医院'''。1951年3月24日，接收苏州公立医院并改编为'''苏南军区医院'''。1952年11月，改称'''江苏军区医院'''。1954年9月16日，改称'''中国人民解放军第一〇〇医院'''，编床位300张，是[[中国人民解放军江苏省军区|江苏军区]]中心医院。1973年7月16日，[[南京军区|南京军区]]司令部颁发该医院编制表，编床位200张。1983年整编，床位减少到150张。1986年11月18日，改为驻军医院，编床位180张（含疗养床位30张）<ref name=jsszjsz>{{Cite book |url =  |title = 江苏省志·军事志|author = 江苏省地方志编纂委员会 |date =  2000年|publisher = 军事科学出版社 |page=598-601 |language =  |accessdate =  |quote = 第九章 后方勤务 第四节 卫生 六、医院简介 }}</ref>。

1965年3月10日，从中国人民解放军第一〇〇医院、[[中国人民解放军第一〇二医院|中国人民解放军第一〇二医院]]、中国人民解放军第一二〇医院（即后来的[[中国人民武装警察部队江苏省总队医院|中国人民武装警察部队江苏省总队医院]]）抽调人员及床位，加上江苏省军区机关及从地方转入部队的医师、护士19人，在江苏省[[海安县|海安县]]组建成立中国人民解放军第一二四医院，隶属江苏省军区。1967年编184人，床位150张。1970年该医院编床位200张。1973年7月16日，该医院增编床位50张。1983年整编，床位减至100张。1988年12月，该医院撤销，改成中国人民解放军第一〇〇医院第二门诊部<ref name=jsszjsz/>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第一〇〇医院由[[中国人民解放军南京军区|中国人民解放军南京军区]]转隶[[无锡联勤保障中心|无锡联勤保障中心]]<ref>{{Cite web |url = http://www.81.cn/zghjy/2016-11/06/content_7344803.htm|title = 100医院开展“稳思想、正秩序、保安全、促改革”活动 |author =  |date =  2016-11-06|publisher = 中国军网|language =  |accessdate =  |quote =  }}</ref>。

===原一〇一医院===
中国人民解放军第一〇一医院的前身成立于1948年<ref name=zgjsw>{{Cite web |url = http://jsnews2.jschina.com.cn/system/2014/12/17/022968534.shtml |title =101医院成南医大教学医院 提升整体科研水平  |author =  |date =2014-12-17  |publisher =  中国江苏网|language =  |accessdate =  |quote =  }}</ref>。1954年9月16日，由'''中国人民解放军华东军区第五陆军医院'''改称'''中国人民解放军第一〇一医院'''，归[[中国人民解放军江苏省军区|江苏军区]]领导，驻无锡市。1960年1月，调归陆军第二十七军建制<ref>{{Cite book |url =  |title = 江苏省志·军事志|author = 江苏省地方志编纂委员会 |date =  2000年|publisher = 军事科学出版社 |page=598-601 |language =  |accessdate =  |quote = 第九章 后方勤务 第四节 卫生 六、医院简介 }}</ref>。

医院是综合性[[三级甲等医院|三级甲等医院]]，2013年通过了[[中国人民解放军总后勤部卫生部|中国人民解放军总后勤部卫生部]]组织的等级医院复评。截至2014年，医院已成为[[江南大学|江南大学]]附属医院，是[[中国人民解放军第二军医大学|中国人民解放军第二军医大学]]等十多所医学院校的临床学院和教学医院。2014年12月16日，医院成为[[南京医科大学|南京医科大学]]教学医院并举行揭牌仪式。截至2014年，医院拥有全军颅脑损伤救治中心、南京军区军事训练医学研究所、南京军区口腔正畸中心、南京军区腹腔镜外科中心、南京军区临床麻醉中心以及9个地区性重点学科<ref name=zgjsw/><ref>{{Cite web |url =http://www.wxrb.com/node/news_dushi/2014-12-17/I0C8JDGEC650431.html  |title =第101医院成为南京医科大学教学医院  |author =  |date =  2014-12-17|publisher =无锡新传媒  |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第一〇一医院由[[中国人民解放军南京军区|中国人民解放军南京军区]]转隶[[无锡联勤保障中心|无锡联勤保障中心]]<ref name=dazao>{{Cite web |url = http://military.china.com/news/568/20161230/30129461.html|title = 无锡联勤保障中心：打造军民陆海空联勤保障新网 |author =  |date =  2016-12-30 |publisher =  中华网|language =  |accessdate =  |quote =  }}</ref>。

2016年9月28日，环[[太湖|太湖]]区域第一家[[直升机|直升机]]医疗救援基地正式在该医院落成使用，同时中国胸痛中心、高级卒中中心也在该医院揭牌<ref>{{Cite web |url =http://news.163.com/16/0930/08/C26QN52G00014AED.html  |title = 解放军第101医院 建军民融合救援基地 |author =  |date = 2016-09-30 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

===原一〇二医院===
1947年3月，由[[华东野战军|华东野战军]]第六纵队后方医院的1个医疗队和第一、第十纵队后方留守处1个医疗队在[[山东|山东]][[临沂|临沂]]建院，番号是'''华东军区第三野战军第十后方医院'''，隶属华东野战军东线兵团卫生部领导。1948年6月，在[[济南战役|济南战役]]之前，医院在[[利津|利津]]地区渡过[[黄河|黄河]]，经[[周村|周村]]抵达[[章丘|章丘]]，驻[[历城|历城]]接受收治任务。1948年底，因[[淮海战役|淮海战役]]需要，医院进驻山东[[济南|济南]]辛庄。1949年1月，医院从济南南迁，经[[蚌埠|蚌埠]]、[[浦口|浦口]]、[[无锡|无锡]]抵达[[南通|南通]]刘桥、平湖地区驻扎。其间，医院接收部分国民党军战俘。1950年3月，医院迁到无锡玉祁，整编成'''华东军区第五野战医院'''。此后继续南迁，从[[浙江|浙江]]抵达[[江西|江西]][[玉山县|玉山]]、[[上饶|上饶]]，1950年底迁至江苏[[宜兴|宜兴]][[丁蜀镇|丁蜀镇]]<ref name=jsszjsz>{{Cite book |url =  |title = 江苏省志·军事志|author = 江苏省地方志编纂委员会 |date =  2000年|publisher = 军事科学出版社 |page=598-601 |language =  |accessdate =  |quote = 第九章 后方勤务 第四节 卫生 六、医院简介 }}</ref><ref name=ghzj>{{Cite web |url = http://www.pla102.com/a/yiyuangaikuo/guanghuizuji/ |title = 光辉足迹 |author =  |date =  |publisher =中国人民解放军第一〇二医院  |language =  |accessdate = 2017-06-23 |quote =  }}</ref>。

1951年6月，医院迁到江苏常州，担负接收[[抗美援朝战争|抗美援朝战争]]伤员的任务。到1952年上半年，医院整编为'''中国人民解放军第一野战医院'''，下设四个分院，分别驻现院址（常州市和平北路55号）、原体育场、[[天宁寺_(常州)|天宁寺]]、[[清凉寺_(常州)|清凉寺]]。当时该医院隶属华东军区后勤部卫生部领导<ref name=jsszjsz/><ref name=ghzj/>。

1954年9月16日，根据华东军区的命令，医院分编为'''中国人民解放军第一〇二医院'''和中国人民解放军第一〇三医院（调至[[福建|福建]][[永安市|永安]]。后来迁至[[广州|广州]]，发展为[[珠江医院|珠江医院]]），1954年10月正式分编。其中第一〇二医院隶属[[中国人民解放军江苏省军区|江苏军区]]后方勤务部领导，编床位500张<ref name=jsszjsz/><ref name=ghzj/>。1959年5月，第一〇二医院组建手术队参加[[1959年藏区骚乱|西藏平叛]]<ref name=ghzj/>。1962年1月15日，根据南京军区命令，改编为南京军区康复医院，编床位400张<ref name=jsszjsz/>。1962年6月，因为中国东南沿海地区的战备需要，医院调出一百多人以及90张床位，组建中国人民解放军第六九六野战医院（后发展为[[中国人民武装警察部队安徽省总队医院|中国人民武装警察部队安徽省总队医院]]）。至1965年调出100多人及100张床位，组建中国人民解放军第一二四医院（1988年撤销）。1969年，为组建[[中国人民解放军第三五九医院|中国人民解放军第三五九医院]]，又调出部分工作人员及床位。至此，医院床位缩编至300张<ref name=ghzj/>。

1976年8月，第一〇二医院完成了收治部分[[唐山大地震|唐山大地震]]伤员的任务。1978年1月，医院全面对外开放，在满足体系部队及老干部医疗的前提之下，向社会开放全日制门诊。1979年6月，医院组织医疗队到江苏省[[溧阳县|溧阳县]]参加[[1979年溧阳地震|抗震救灾]]。1979年2月，医院派出脑外科手术队参加[[对越自卫还击作战|对越自卫还击作战]]<ref name=ghzj/>。

1983年12月，第一〇二医院整编，床位缩减至150张，并且撤销了传染科及肺结核科，科室不再设政工干部。1984年底，医院将原招待所扩大到200张床位。1986年11月18日，医院精减整编，改为驻军医院。1986年12月15日，划归[[南京军区|南京军区]]后勤第十三分部建制领导<ref name=jsszjsz/><ref name=ghzj/>。1991年5月，[[1991年華東水災|中国部分地区特大洪涝灾害]]发生，医院赴灾区送医送药。1994年1月，中国人民解放军精神医院奠基人、第一〇二医院精神神经科主任、南京军区精神医学研究所所长[[高柏良|高柏良]]被南京军区政治部表彰为“优秀共产党员”，《[[人民日报|人民日报]]》、《[[解放军报|解放军报]]》头版头条及《人民前线》、[[中央人民广播电台|中央人民广播电台]]等媒体作报道<ref name=ghzj/>。

1995年7月，[[中国人民解放军总后勤部卫生部|中国人民解放军总后勤部卫生部]]批准第一〇二医院成立“全军精神医学中心”。1996年11月，中国人民解放军总后勤部批准第一〇二医院为[[三级甲等医院|三级甲等医院]]。2000年5月，经中国人民解放军总后勤部卫生部批准，第一〇二医院成立“全军心理疾病防治中心”。2003年3月，经南京军区批准，第一〇二医院成立南京军区心理康复中心<ref name=ghzj/>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军第一〇二医院由[[中国人民解放军南京军区|中国人民解放军南京军区]]转隶[[无锡联勤保障中心|无锡联勤保障中心]]<ref>{{Cite web |url = http://military.people.com.cn/n1/2016/1122/c1011-28885899.html|title =战时心理伤救护 10年调研10万官兵 |author =  |date =  2016-11-22 |publisher =  人民网|language =  |accessdate =  |quote =  }}</ref>。

2017年11月4日，东部战区联勤心理危机干预中心在第一〇二医院成立。这也是全军首个联勤心理危机干预中心<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1850769 |title = 全军首家：东部战区联勤心理危机干预中心在第102医院成立 |author =  |date = 2017-11-05  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

===九〇四医院===
2018年11月5日，原中国人民解放军第一〇〇医院（苏州长城医院）、中国人民解放军第一〇一医院（无锡市太湖医院）和中国人民解放军第一〇二医院（常州和平医院）合并组建为新的'''中国人民解放军联勤保障部队第九〇四医院'''，无锡联勤保障中心陈明德参谋长宣读医院组建命令、医院领导任职命令及党委常委组成通知<ref name=merge2>{{Cite web |url = http://www.wxrb.com/news/own/201811/t20181105_1434499.shtml|title = 中国人民解放军无锡联勤保障中心第904医院成立大会胜利召开 |author =顾明  |date =  2018-11-05|publisher = 无锡新传媒网|language =  |accessdate =  |quote =  }}</ref>。重组后的九〇四医院以原一〇一医院为本部，将原有的三家医院的医疗资源加以整合。其中原一〇〇医院在烧伤、骨科等救治方面具有优势；原一〇一医院以创伤外科和急救为特色，拥有全军颅脑损伤救治中心、国家高级卒中中心、中国胸痛中心等单位；原一〇二医院则拥有全军精神医学中心、全军心理疾病防治中心等<ref name=merge1/>。
