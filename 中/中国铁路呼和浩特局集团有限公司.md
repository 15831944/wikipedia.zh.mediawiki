'''中国铁路呼和浩特局集团有限公司'''，原名'''呼和浩特铁路局'''，是[[中國鐵路總公司|中國鐵路總公司]]下属公司，管辖范围包括[[内蒙古自治区|内蒙古自治区]]的中部西部地区。原呼和浩特铁路局辖区内共有135个车站。

内蒙古自治区内，集宁南以东[[通辽市|通辽市]]以西归呼和浩特铁路局管内合资铁路公司-集通铁路公司管辖，赤峰市以南通辽市以东的铁路线归[[沈阳铁路局|沈阳铁路局]]管辖，[[兴安盟|兴安盟]]以北地区的铁路线归[[哈尔滨铁路局|哈尔滨铁路局]]管辖。[[敦煌铁路|敦煌铁路]]额济纳旗支线归[[兰州铁路局|兰州铁路局]]管辖。

==沿革==
1995年，呼和浩特铁路局进行了工商登记<ref name=guapai/>。

2017年11月4日，济南、乌鲁木齐、南昌、北京、哈尔滨、武汉、郑州、上海、呼和浩特、昆明、青藏铁路公司等11家铁路局（公司）的名称变更通过[[国家工商行政管理总局|国家工商行政管理总局]]核准，呼和浩特铁路局名称变更为'''中国铁路呼和浩特局集团有限公司'''（(国)名称变核内字[2017]第5290号）<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1851596  |title = 15家铁路局将集体变身集团公司：名称变更已通过核准 |author =  |date =2017-11-06   |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月17日，在国家企业信用信息公示系统中，中国铁路呼和浩特局集团有限公司已完成工商登记变更。工商登记变更后，中国铁路呼和浩特局集团有限公司仍为中国铁路总公司100%控股的公司，公司由原先的全民所有制企业变成有限责任公司（非自然人投资或控股的法人独资）企业，注册资本由原先的544.87856亿元变更为482.599亿元人民币，经营范围增加房地产开发经营等<ref name=yi>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1868305|title =告别地方铁路局！18家路局集体变身集团公司，铁总全资控股|author =  |date = 2017-11-17 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月19日，中国铁路呼和浩特局集团有限公司正式挂牌<ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1870782|title = 18个铁路局已改制为铁路局集团有限公司，今日正式挂牌 |author =  |date = 2017-11-19 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref name=guapai>{{Cite web |url = http://finance.ifeng.com/a/20171119/15803099_0.shtml |title = 18家铁路局集团改制后集中挂牌！附主要领导人简历 |author =  |date = 2017-11-19 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。
