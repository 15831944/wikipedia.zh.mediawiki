'''C919'''是由[[中国商用飞机|中国商用飞机有限责任公司]]研制的一款168-190座级[[窄体飞机|窄体干线客机]]，专为短程到中程的航线设计，属于单通道150座级，标配168个座位，最多可容纳190个座位，學名為「中短程雙發動機窄體民用運輸機」<ref>{{Cite web|url = http://www.chinaequip.gov.cn/2014-10/30/c_133754175.htm|title = C919飞起来|accessdate = 2015-11-07|author = 装备制造|date = 2014-10-30|publisher = 中国技术装备网|deadurl = yes|archiveurl = https://web.archive.org/web/20160323045604/http://www.chinaequip.gov.cn/2014-10/30/c_133754175.htm|archivedate = 2016-03-23}}</ref><ref>{{cite news|title = C919飞起来|url = http://www.chinaequip.gov.cn/2014-10/30/c_133754175.htm|accessdate = 2015-11-07|date = 2014-10-30|newspaper = 中国技术装备网|deadurl = yes|archiveurl = https://web.archive.org/web/20160323045604/http://www.chinaequip.gov.cn/2014-10/30/c_133754175.htm|archivedate = 2016-03-23}}</ref>。

C919项目于2008年11月启动。2009年9月8日，C919外形样机在[[香港|香港]]举行的{{tsl|en|Asian Aerospace|亚洲国际航空展}}上首次公开亮相。2015年11月2日，C919首架總裝完畢，原型机尾號為B-001A，於中国商飞公司总装制造中心浦东基地厂房内正式下线<ref name=":2">{{cite news|author=贾远琨 钱春弦|title=国产大型客机C919首架机在上海总装下线|url=http://news.xinhuanet.com/fortune/2015-11/02/c_1117008884.htm|accessdate=2015-11-02|newspaper=新华网|date=2015-11-02}}</ref><ref>{{cite news|url=http://news.ifeng.com/a/20151102/46077249_0.shtml|title=实拍国产大飞机C919总装下线 布局168座位|date=2015-11-02|agency=鳳凰網}}</ref>。

C919的首架原型机本计划于2014年首飞，2016年交付航企使用。其后经过两次延期，原型机于2016年12月25日送交試飛中心；2017年4月，高速滑行試驗完成；并于2017年5月5日下午兩點在[[上海浦东国际机场|上海浦东国际机场]]順利完成首飛。[[中国共产党中央委员会|中共中央]]和[[中华人民共和国国务院|国务院]]在飞机首飞成功后发出贺电<ref>{{cite news|url=http://news.sina.com.cn/c/nd/2017-05-03/doc-ifyeycfp9143497.shtml|title=C919飞机首飞倒计时2天 客户已达23家订单570架|date=2017-05-03|agency=新浪新聞中心 國內新聞}}</ref>。

== 机名由来 ==
C919全称COMAC-C9-19，C取自[[中华人民共和国|中国]]（China）和[[中国商用飞机|中国商飞]]（COMAC）的首字母，并寓意将与[[空中巴士|空中巴士]]（Airbus）及[[波音|波音]]（Boeing）形成“ABC”并行的局面；飞机型号选择9，是因为九与“久”谐音，在中国传统文化中代表永恒，象征“天长地久”，后面的19指最大载客量为190个座位<ref>{{cite news|author=孙闻|title=吴光辉委员：首型国产大型客机将命名为“Ｃ919”|url=http://www.gov.cn/2009lh/content_1252509.htm|accessdate=2013-03-03|newspaper=中国政府网|date=2009-03-06}}</ref><ref>[http://media.china.com.cn/cmyw/2017-05-05/1038172.html 国产大飞机C919拟今日首飞 “全民网红”长啥样?]{{Dead link|date=2019年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==發展==
=== 背景 ===
C919并非[[中华人民共和国|中华人民共和国]]的第一个大型客机项目，终止于1986年的[[運-10|運-10]]有相似的载客量和更高的国产化率，但C919是第一个市场化的全新的窄体客机计划，也是中國[[改革开放|改革开放]]後的首個大型幹線客机項目。製造大飛機是中國產業發展的重要策略，因此C919是中國國家經濟戰略的要角之一。首架机於2015年11月在[[浦东|浦东]]基地总装下线<ref>{{cite news|title=C919總裝下線! 習近平李克強作重要批示|url=http://news.wenweipo.com/2015/11/02/IN1511020019.htm|newspaper=文匯網|date=2015-11-02}}</ref>。

=== 測試情況 ===
*与飞机总装平行推进的系统测试仍在紧张进行之中，包括机械系统测试、发电和配电等供电系统测试、航电系统测试试验，按照飛機真實的構型狀態，實現机械、供电、航电系统交聯互通，開展飛機級全系統交聯試驗。C919大型客機是中國第一個进行机械、供电、航电共同进行的“三系统联试”项目，是系统集成试验的自主创新。
*2016年4月10日，C919試驗機（10001架機）已從中國商飛公司總裝製造中心浦東基地，移轉至中國飛機強度研究所上海分部，正式啟動全機靜力試驗。所謂地面靜力試驗，就是檢測飛機主體結構夠不夠結實，是一種破壞性試驗。專門有「靜力試驗機」，用力壓它、拉它，看飛機的結構強度極限，直到它破損為止。以檢驗飛機是否具備飛行所需的結構強度，是保證飛機飛行安全的重要手段<ref>{{cite web|url=http://www.chinatimes.com/realtimenews/20160413003019-260417| title=陸C919大飛機啟動靜力試驗 年內首飛|date=2016-04-13| publisher=[[中國時報|中國時報]]| language=zh-hant}}</ref>。
*2016年11月8日，C919悄悄地完成了首飞前两项重要的试验——全机2.5g限制载荷静力试验圆满顺利、左右发动机均一次点火成功。而在静力试验完成后，飞机就可以进入调试阶段并进行地面飞控、燃油系统等的完整测试，测试无误后，飞机就可以进入滑跑阶段。如果滑跑阶段没有问题，则就可以进入首飞<ref>{{cite web|url=http://mil.news.sina.com.cn/jssd/2016-11-12/doc-ifxxsmif2858126.shtml| title=深度：中国国产大客机将首飞 或替代运20成特种平台|publisher=[[新浪|新浪]] |date=2016-11-12}}{{zh-hans}}</ref>。
*2016年12月21日，C919飛機發動機供應商，[[CFM国际|CFM國際公司]][[CFM_International_LEAP|LEAP-1C]]集成推進系統，同時被[[欧洲航空安全局|歐洲航空安全局（EASA）]]和[[美国联邦航空管理局|美國聯邦航空局（FAA）]]授予了型號合格證，為中國商飛C919飛機投入商業運營鋪平了道路。
*2016年12月25日，C919飛機首架機交付試飛中心，首飛在即。C919大型客机项目正在按计划有序推进：自C919首架机总装下线后，项目在系统集成试验、静力试验、机上试验、试飞准备等几条主线稳步推进，現已進入「首飛」前衝刺階段「全機地面共振試驗」和「模態耦合試驗」<ref>{{cite web|url=http://www.comac.cc/xwzx/gsxw/201612/29/t20161229_4701299.shtml| title=C919飞机首架机交付试飞中心|publisher=[[中國商飛|中國商飛]]新聞 |date=2016-12-30}}{{zh-hans}}</ref><ref>{{cite web|url=http://www.cnbeta.com/articles/572087.htm|title=C919大飞机首架机交付试飞中心 首飞在即|publisher=[[CnBeta|CnBeta]]新聞|date=2016-12-30|deadurl=yes|archiveurl=https://web.archive.org/web/20161231182216/http://www.cnbeta.com/articles/572087.htm|archivedate=2016-12-31}}{{zh-hans}}</ref>。
*2017年3月1日，C919飛機在上海[[浦東機場|浦東機場]]完成第一次全過程低速、中速、高速滑行試驗。對於民用飛機而言，低速滑行是指將飛機的速度控制在每小時55公里以內的滑行；中速滑行是指速度為每小時55至170公里的滑行；高速滑行則是指每小時超過170公里的滑行。向首飛又邁進一步，進入首飛準備狀態<ref>{{cite web|url=http://www.chinatimes.com/realtimenews/20170305003129-260409| title=完成滑跑實驗陸C919客機今年有望首飛|date=2017-03-05| publisher=[[旺報|旺報]]| language=zh-hant}}</ref><ref>{{cite web| url=http://news.sina.com.tw/article/20170306/20940616.html| title=中國商飛副總經理吳光輝：C919進入首飛準備狀態| date=2017-03-06| publisher=[[新浪|新浪]]| language=zh-hant| deadurl=yes| archiveurl=https://web.archive.org/web/20170306211916/http://news.sina.com.tw/article/20170306/20940616.html| archivedate=2017-03-06}}</ref>。
*2017年3月20日，中國63名院士和專家組成的評審委員會，分為總體技術、機體結構、飛控機械、航電電氣、推進燃油、製造和綜合7個組，一致同意通過國產大型客機C919首飛技術評審，建議在完成電磁兼容等試驗及滑行試驗驗證後可提請首飛放飛評審，這標誌着C919向着首飛目標又邁出了堅實的一步<ref>{{cite web|url=http://paper.wenweipo.com/2017/03/26/CH1703260017.htm| title=國產客機C919通過首飛技術評審|publisher=[[文匯報_(香港)|文匯報]] |date=2017-03-26}}{{zh-hant}}</ref>。
*2017年4月16日上午，C919飛機在上海浦東機場再次進行高速滑行試驗，最高滑行速度達到248km/h（約134節）<ref>{{cite web|author1=IT之家|title=国产大飞机C919首次完成跑道高速滑行测试|url=http://www.ithome.com/html/it/304553.htm}}</ref><ref>{{cite news|url=http://hk.on.cc/cn/bkn/cnt/news/20170416/bkncn-20170416163017376-0416_05011_001.html| title=首飛前最後一關　國產客機完成高速滑行測試|publisher=[[東網|東網]]|date=2017-04-16}}{{zh-hant}}</ref>，因飛機為空載，預計已達到[[V速率|起飛決斷速度]]（V1）。
*2017年4月18日，中国科学院、中国工程院、中国民航局、中航工业、中国商飞等单位的25名专家分为飞机组和空勤组两个专业组。经过严格评审，评审委员会一致同意，C919大型客机首架机通过首飞放飞评审，待完成全部高速滑行测试后，可择机开展首飞。这也是继3月20日通过了首飞技术评审后，C919大型客机完成的首飞前最后一项评审任务<ref>{{cite web|url=http://mil.news.sina.com.cn/china/2017-04-19/doc-ifyeimqy2584779.shtml| title=中国商飞：C919首飞不赶进度 绝不带任何隐患上天|publisher=[[新浪|新浪]] |date=2017-04-19}}{{zh-hant}}</ref>。
*2017年4月23日上午9時左右，C919大型客機在浦東機場第四跑道進行了第四次高速滑行試驗，飛機兩次在滑跑中機頭昂起，前輪離地，高滑抬前輪試驗獲得圓滿成功。高速滑行抬前輪試驗是最接近首飛狀態的一次試驗，主要目的是為了對飛機抬前輪過程中的操縱響應，以及飛行性能進行感受。今天的試驗進一步測試了飛機各項設備在地面高速運動時的性能，以及驗證[[起飛|「抬前輪速度(VR)」]]－－ [[轉速|轉速]]不能小于起飞决断速度（V1）或1.05倍的[[V速率|空中最小操纵速度]]（Vmca）這一重要指標。兩次抬前輪試驗，C919的最大速度約113節，前輪離地時間分別達到11秒和14秒。中國商飛公司副總經理、C919客機總設計師吳光輝告訴記者，經過五場低速、九場中速、四場高速滑行試驗，試驗結果數據有效，C919首飛前地面滑行試驗全部結束<ref>{{cite web|url=http://news.wenweipo.com/2017/04/23/IN1704230018.htm| title=C919完成最後一項地面滑行試驗 首飛在望|publisher=[[文匯報_(香港)|文匯報]] |date=2017-04-19}}{{zh-hant}}</ref>。

== 特色 ==
中国商飞表示，由于已知销量（绝大多数为意向订单）远小于同样大小的[[波音|波音]]和[[空中客车|空中客车]]机款，该公司商用客机单价更贵，但随着订单数的攀升将会逐步降低。其使用了与[[波音737_MAX|波音737 MAX]]和[[空中客车A320neo系列|空中巴士A320 neo]]相同的[[CFM_International_LEAP|CFM国际 LEAP家族]]中-1C引擎，将比目前[[波音737|波音737]]和[[空中客车A320系列|空中巴士A320]]节省15%的燃料。除此之外减碳排放量50%，载客數多8到10人，且座位空间更宽敞，可以與對手開發中的下一代飛機媲美，預期這款產品有競爭力的部分主要是瞄準在經濟與舒適性<ref>[http://www.voanews.com/chinese/2009-09-08-voa45.cfm 亚洲航展：中国展出大飞机模型关健技术仍靠进口]</ref>。

C919客机的自主创新，反映在五个方面：一，总体方案来自于中方；二，自行主持气动外形设计并进行相关试验；三，机体从设计到制造完全由中国完成；四，中国进行各种来自不同供应商的系统的集成；五，由中国进行全面管理<ref>{{Cite web |url=http://m.airforceworld.com/a/20151206/2854.html |title=C919中国客机创新分析 |access-date=2015-12-06 |archive-url=https://web.archive.org/web/20151208133744/http://m.airforceworld.com/a/20151206/2854.html |archive-date=2015-12-08 |dead-url=yes }}</ref>。

=== 部件製造 ===
C919将首先配备[[CFM国际|CFM国际]]生产的[[CFM_International_LEAP|CFM LEAP-X 系列发动机]]，型号为LEAP-1C。其中{{le|引擎吊舱|Nacelle}}和[[推力反向器|推力反向器]]来自{{link-en|Nexcelle|Nexcelle}}，采用大量复合材料，并进行了减噪处理，具有高级进气道调控功能和电传控制的推力反向器，轮胎则采用[[米其林|米其林]]公司的Air X子午线轮胎。

机头雷达由中航工业天津制造。机头由中航工业成都制造。机尾及垂尾由中航工业哈尔滨制造。中心翼盒、外翼盒、翼段、副翼由中航工业西安制造。前机身，中机身段以及后机身等值段由中航工业洪都制造。前缘襟翼与后缘襟翼由中航工业昌河制造。中航工业上海为第一生产线和总装厂，中航工业洪都为拟定的第二生产线和第二总装厂。为配套C919的生产及总装，南昌航空工业城（即中航工业洪都新址）已于2009年开始建造，计划于2012年第四季度开始生产C919的主要部件。

飞机将在上海完成设计和总装，國產化率約三成，估计首航时国产化率提升至约50%，2025年后超过90%。
