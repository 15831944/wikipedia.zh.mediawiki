'''中華民國臨時政府'''是[[中國|中國]][[辛亥革命|辛亥革命]]期間，由独立各省和[[革命党|革命党]]人於1912年1月1日在[[江苏省_(中华民国)|江苏省]][[南京府|南京府]]成立的政府機構<ref>{{cite web |url = http://news.ifeng.com/history/zhongguojindaishi/detail_2011_07/11/7610418_2.shtml |title = 黎元洪谈孙中山：名声是虚构 对革命没有实质贡献 |website = 凤凰网 |date=2011年7月11日 }}</ref>，也是[[中華民國歷史|中華民國歷史]]上第一個[[中央政府|中央政府]]機構，又稱'''南京臨時政府'''<ref>http://gaz.ncl.edu.tw/guide.jsp?jid=94001163</ref>，簡稱'''臨政'''。其存在時間約3個月。

==沿革==
1912年1月，中華民國臨時政府成立，1月5日舉行内阁第一次會議<ref>{{cite web |url=http://past.people.com.cn/BIG5/shizheng/252/6534/6564/20010928/572437.html |title=存档副本 |accessdate=2009-07-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20050827193518/http://past.people.com.cn/BIG5/shizheng/252/6534/6564/20010928/572437.html |archivedate=2005-08-27 }}</ref>，其組織方法依照《[[中華民國臨時政府組織大綱|中華民國臨時政府組織大綱]]》規定。1912年3月8日[[南京臨時參議院|南京臨時參議院]]通过《[[中華民國臨時約法|中華民國臨時約法]]》，3月11日公布实施，取代《[[中華民國臨時政府組織大綱|中華民國臨時政府組織大綱]]》。《[[中華民國臨時約法|中華民國臨時約法]]》公布实施后，南京的中華民國臨時政府解散，由[[北京|北京]][[中华民国临时政府_(袁世凯)|中华民国临时政府]]继承，權力歸入不同機關。

1911年12月29日，[[各省都督府代表联合会|各省都督府代表联合会]]代表在南京召开[[1911年中华民国临时大总统选举|中华民国临时大总统选举会]]，选举[[孙文|孙文]]为[[中华民国临时大总统|中华民国临时大总统]]。1912年1月1日，孙文在南京就任临时大总统。同年1月3日，[[各省都督府代表联合会|各省都督府代表联合会]]代表在南京召开[[1911年中华民国临时大总统选举|中华民国临时副总统选举会]]，选举[[黎元洪|黎元洪]]为中华民国临时副总统。<ref name=lsl>刘寿林等编，民国职官年表，北京：中华书局，1995年</ref>

1912年2月15日，[[南京临时参议院|中华民国临时参议院]]在南京召开[[1912年中华民国临时大总统选举|中华民国临时大总统选举会]]，选举[[袁世凯|袁世凯]]为[[中华民国临时大总统|中华民国临时大总统]]。同年2月20日，[[南京临时参议院|中华民国临时参议院]]在南京召开[[1912年中华民国临时大总统选举|中华民国临时副总统选举会]]，选举[[黎元洪|黎元洪]]为中华民国临时副总统。3月10日，袁世凯在[[北京|北京]]就任临时大总统。4月1日，孙文在南京向[[南京临时参议院|中华民国临时参议院]]辞去临时大总统职务。<ref name=lsl/>

===参议院===
[[File:Nanjinglinshicanyiyuan2.jpg|200px]]。該樓原為[[江蘇諮議局|江蘇諮議局]]大樓，於1912年1月28日改爲[[中華民國臨時參議院|中華民國臨時參議院]]院址。<ref>[http://www.jllib.cn/njmgjz.cn/xzjz/b219 中国国民党中央党部，南京民国建筑网，于2011-07-02查阅] {{webarchive|url=https://archive.is/20120908054629/http://www.jllib.cn/njmgjz.cn/xzjz/b219 |date=2012-09-08 }}</ref>]]
{{Main|南京临时参议院}}

1912年1月2日，[[各省都督府代表联合会|各省都督府代表联合会]]开始代行参议院职权。1912年1月28日，临时参议院于[[南洋劝业场|南洋劝业场]]正式开幕，[[各省都督府代表联合会|各省都督府代表联合会]]自行解散。4月2日临时参议院议决临时政府迁至北京，4月4日议决该院迁至北京，4月8日休会。
