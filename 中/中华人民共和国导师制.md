== 争议 ==

中国高校对教师的考评体系，是以[[论文|论文]]数量和课题项目为核心。办学的初衷——教学，反成为边缘工作，学生亦沦为导师控制下的廉价人力资源。2005年，[[中华人民共和国教育部|中国教育部]]公布《教育部关于进一步加强高等学校本科教学工作的若干意见》，提出“有条件的高校要积极推行导师”。或认为，以[[本科生|本科生]]为廉价劳动力资源方是推广本科生导师制的动力<ref name="a1">{{Cite web |url=http://www.sohu.com/a/228240553_176673 |title=观察//中国式导师制：“权力的滥用”与“绝对的服从”|accessdate=2018-05-05 |author=陆慢慢 |date=2018-04-14 |work= |publisher=|language=Zh-hans }}</ref>。 

在[[研究生|研究生]]、[[博士生|博士生]]阶段，导师个人主宰学生学业，决定学生命运，形成人身依附。而在理工科类高校实验室中，其工作团队通常以企业的方式运作。导师作为负责人，职责与企业老板相同<ref>{{Cite web |url=http://view.news.qq.com/a/20080107/000038.htm|title=博士导师反目呼唤平衡机制|accessdate=2018-05-05 |author=李隼 |date=2008-01-07 |work= |publisher=腾讯网，来源：羊城晚报|language=Zh-hans }}</ref>。学生的学习过程实质变为工作，双方形成隐形的[[雇佣|雇佣]]关系，故学生多将导师称为[[老板|老板]]。但此种劳动关系不被中国法律承认，亦不受保护。与普通劳动者相比，甚至在校外、休息时间，学生仍需为导师无休止的劳动<ref name="a1"/>。对于导师压榨学生，高校可能会因经济利益而不愿干涉，进而纵容<ref>{{Cite web |url=http://view.news.qq.com/a/20080107/000048.htm|title=导师是“老板”？ 看研究生给导师“打工”|accessdate=2018-05-05 |author= |date=2008-01-07 |work= |publisher=腾讯网，来源：现代教育报|language=Zh-hans }}</ref>。全国各地高校，因导师压榨学生，学生以自杀寻求解脱的个案，比比皆是。2018年暴光的[[陶崇园事件|陶崇园事件]]即使是此类情况中的典型。
