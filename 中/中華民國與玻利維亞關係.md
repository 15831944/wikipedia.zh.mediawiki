'''中華民國與玻利維亞關係'''是指[[中華民國|中華民國]]與[[多民族玻利維亞國|多民族玻利維亞國]]之間的關係。1919－1985年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，曾在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構，但相繼關閉，目前對玻利維亞的有關事務由'''駐[[秘魯|秘魯]][[台北经济文化代表处|台北經濟文化辦事處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-196-8ebb7-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref><ref name="代表處簡介">{{Cite web|url =https://www.taiwanembassy.org/pe/post/1328.html|title =代表處簡介|publisher =中華民國駐外單位聯合網站}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[玻利維亞|玻利維亞]]。]]
{{see also|中華民國外交|{{link-en|玻利維亞外交|Foreign relations of Bolivia}}}}
{{see also|中国驻玻利维亚大使列表{{!}}中華民國駐玻利維亞外交官列表|玻利维亚驻华大使列表{{!}}玻利維亞駐中華民國外交官列表}}
1919年12月3日，中華民國與玻利維亞建立公使級[[外交#外交上的承認|外交關係]]。於[[De_facto|實際上]]的首都[[拉巴斯|拉巴斯]]設立'''中華民國駐玻利維亞共和國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]。

1947年5月10日，公使館升格為'''中華民國駐玻利維亞共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1985年7月11日，玻利維亞與中華民國斷交。<ref name="外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2008/html/08-1-6.htm |title = 《中華民國97年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

兩國斷交後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。分別為1990年12月26日，'''中華民國駐玻利維亞[[台北经济文化代表处|商務及領務辦事處]]'''（[[西班牙文|西班牙文]]：{{lang|es|Oficina Comercial Consular de la Republica de China}}），享有與[[國際組織|國際組織]]駐玻利維亞機構同等的地位與待遇。2004年11月11日，更名為'''駐玻利維亞臺灣商務辦事處'''（西班牙文：{{lang|es|Oficina Comercial de Taiwan}}）。已關閉，自2009年7月28日起改由駐[[秘魯|秘魯]]台北經濟文化辦事處{{Link-en|多重委派|Dual accreditation|兼轄}}；<ref name="外交年鑑"/><ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=431EEA37186C6DC5 |author = 公眾外交協調會 |date = 1990年10月30日 |title = 新聞稿 |publisher = 中華民國外交部}}</ref><ref name="代表處簡介"/>以及1992年4月29日，'''玻利維亞駐華商務暨金融代表處'''（[[英文|英文]]：{{lang|en|Bolivian Commercial and Financial Representation}}）。自2002年9月起未再派代表，也已關閉。<ref name="外交年鑑"/><ref>{{Cite web |url = http://www.mofa.gov.tw/OfficesInROC_Content.aspx?n=8CEB2B5F5436B997&sms=8EBFADC1592C7BFE&s=DCD4BE1106ED5B01&os=8A33013523400F35 |title = 駐華外國機構 |publisher = 中華民國外交部}}</ref>

1991年5月9日，{{Link-en|玻利維亞參議院議長|President of the Senate of Bolivia}}巴爾達（Gonzalo Valda Cárdenas）率領多位議員抵台訪問。期間會見立法院長[[梁肅戎|梁肅戎]]、外交部長[[錢復|錢復]]、政務次長房金炎、[[行政院農業委員會|行政院農業委員會]]副主任委員林享能等人士。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=9B8C6736AE1D9141 |author = 公眾外交協調會 |date = 1991年5月8日 |title = 玻利維亞共和國國會議員訪華團一行八人由參議院議長巴爾達率領 |publisher = 中華民國外交部}}</ref>

===簽證===
[[File:Visa_requirements_for_Taiwanese_citizens_(6).png|thumb]]的[[中華民國國民|中華民國公民]]必須申請[[簽證|簽證]]方可入境玻利維亞。]]
[[File:Visa_requirements_for_Bolivian_citizens.png|thumb]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|玻利維亞簽證政策|玻利維亞公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]以觀光或探訪為由，若無法在{{link-en|玻利維亞駐外機構列表|List of diplomatic missions of Bolivia|玻利維亞駐外館處}}取得簽證，可於該國邊境或機場移民局取得單次入境30天簽證，可延長至90天。入境須出示[[黃熱病|黃熱病]]的[[疫苗接種|疫苗接種]]證明（前往首都[[拉巴斯|拉巴斯]]則免）。經玻利維亞轉機亦須辦理簽證。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-196-8ebb7-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持{{link-en|玻利維亞護照|Bolivian passport}}的{{link-en|玻利維亞人口|Demographics of Bolivia|玻利維亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將玻利維亞歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2016年12月30日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-177-c0dd2-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將玻利維亞歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=437796D4B280BA84 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|玻利維亞經濟|Economy of Bolivia}}}}
===貿易===
2018年，玻利維亞是中華民國第142大貿易夥伴、第130大[[進口|進口]]夥伴、第127大[[出口贸易|出口]]夥伴。出口至玻利維亞的金額為851萬4,748[[美元|美元]]，年增15.782%；自玻利維亞進口的金額為366萬3,348美元，年增99.624%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）485萬1,400美元，年減12.096%。2017年，玻利維亞是中華民國第148大貿易夥伴、第148大進口夥伴、第133大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

==僑民==
{{see also|臺灣裔{{!}}海外臺僑|海外華人}}
在玻利維亞的[[僑民|僑民]]於1980年前後曾達1,300人左右，90%以上來自[[台灣|台灣]]，後因該國政經情況不佳，相繼轉赴他國發展。目前持[[中華民國護照|中華民國護照]]的僑民約200人，其中50%集中定居在第1大城[[圣克鲁斯_(玻利维亚)|聖克魯斯]]，其餘則分居[[De_facto|實際上]]的首都[[拉巴斯|拉巴斯]]及第3大城[[科恰班巴|科恰班巴]]，各約40人及60人。這三座城市也先後成立「中華總會」、「中華商會」及「華夏促進會」。僑民多經營[[飲食業|餐飲業]]、[[商店|商店]]、[[旅館|旅館]]、[[旅行社|旅行社]]等，也有開設[[塑膠|塑膠]]工廠及[[麵粉|麵粉]]廠者。聖克魯斯目前設有[[華人|華人]]的[[基督教|基督]][[教會|教會]]與[[教堂|教堂]]，另有民間集資興建的祐海宮以供奉[[媽祖|媽祖]]。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2007/html/08-1-6.htm |title = 《中華民國96年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref><ref>{{Cite web |url = http://www.boca.gov.tw/content.asp?CuItem=990&BaseDSD=13&CtUnit=14&mp=1 |title = 國家基本資料 |publisher = 外交部領事事務局}}</ref>
