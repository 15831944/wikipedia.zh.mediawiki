'''中国福利彩票'''简称'''福利彩票'''或'''福彩'''，由[[中华人民共和国民政部|中国民政部]]的直属事业单位[[中国福利彩票发行管理中心|中国福利彩票发行管理中心]]负责发行销售的[[彩票|彩票]]。

福彩与[[体彩|体彩]]同为中国（大陆）主要的两大彩票发行机构，由其福彩双色球更为主要的彩票购买对象，每期销售额均过亿计。

== 宗旨 ==
福利彩票的发行的宗旨是：'''向社会募集资金用于国家福利事业。'''

== 彩票类型 ==
目前福彩发行的主要彩票类型有：[[双色球|双色球]]、[[北京赛车|北京赛车]]、[[福彩3D|福彩3D]]、[[东方6+1|东方6+1]]、[[刮刮乐|刮刮乐]]、[[乐透型15选5|乐透型15选5]]、[[时时彩|时时彩]]等。

双色球、福彩3D、七乐彩为主力销售品种，尤其'''双色球'''更为人所知，并为当下中国最热门的彩票品种。

== 历史 ==
* 1986年6月18日，[[中华人民共和国民政部|民政部]]向[[中华人民共和国国务院|国务院]]正式报送《关于开展社会福利有奖募捐活动的请示》。同年12月20日获得通过。
* 1987年2月5日，[[中国共产党中央书记处|中国共产党中央书记处]]原则通过，并明确'''唯一发行机关为民政部。'''
* 1987年6月3日，中国社会福利有奖募捐委员会（简称中募委）在[[北京市|北京]]成立。
* 1987年7月27日，第一批8000万张“中国社会福利有奖募捐奖券”（面值一元），在[[河北省|河北]]、[[江苏省|江苏]]、[[浙江省|浙江]]、[[山东省|山东]]、[[上海市|上海]]等10个省、市试点发行，
* 1987年7月28日在[[河北省|河北省]][[石家庄市|石家庄市]]售出了第一张彩票。
* 1993年10月和1994年5月，中国社会福利有奖募捐券发行中心分别被国际彩票组织INTERTOTO和AILE接纳为正式会员和临时委员。并改名为“中国福利彩票”，发行机构正式定名为“中国福利彩票发行中心”。
* 1997年，国际彩票组织（INTERTOTO）和（AILE）合并为“[[世界彩票协会|世界彩票协会]]”（WLA），“中国福利彩票发行中心”成为其正式会员。
* 2003年2月16日，第一个统一游戏规则、统一开奖、共享奖池的全国联合销售的电脑彩票游戏“[[双色球|双色球]]”乐透型福利彩票在全国22个省、自治区和直辖市发行，标志着电脑福利彩票品牌时代的到来。 
* 2004年4月1日，中国福利彩票双色球开奖节目在[[中国教育电视台|中国教育电视台]]一套直播。
*2015年底第十二屆全國人大常委審議大會上，[[尹中卿|尹中卿]]常委問責表示658億人民幣的年度彩票資金中約490億是用於社福，但有169億是非法用途和去向不明，其中有5億用於非法購買和維修公務長官車，其餘有發放員工非法獎金、非法出國旅遊等用途，其餘大筆是以培訓中心名義興建樓堂管所，興建發包存在貪汙，落成後大樓許多閒置另有一些非法廉價出租給特定人經商，隱含利益輸送。<ref>[http://news.cntv.cn/2015/12/30/VIDE1451478775739863.shtml 央視-問責之监督的力量]</ref>
