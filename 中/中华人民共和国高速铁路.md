'''中国高速铁路'''是目前世界上最大规模的[[高速铁路|高速铁路]]网，始建設於2004年，2008年中國大陸第一条高速铁路[[京津城际铁路|京津城际高速铁路]]建成通车<ref>[http://silkroad.news.cn/2018/0208/83787.shtml 高铁时代 迎面驶来].新華絲路.</ref>。截至2018年12月總里程达[[中国高铁线路列表|2.9万公里]]，其中運營時速可達300公里的路線，總里程超1萬公里<ref group="注">若採"營運時速每小時200公里以上"的國際定義，不含提速改造的既有線，僅計算新建路線，包括客貨混合線，總長度計27,077公里，本資料是依[[中國高鐵路線列表|中國高鐵路線列表]]所示各路線，將開通時即達到時速200km/h以上各路線長度加總而成，包括新建的客貨混合線。</ref>，占世界2/3以上<ref>{{cite news|title=中国高铁营业里程达2.9万公里 超世界总数三分之二|url=http://www.chinanews.com/gn/2018/12-24/8710957.shtml}}</ref>，有包括[[香港特別行政區|香港特別行政區]]的30个省级行政区開通高鐵，官方於「铁路安全管理条例」中定义為：“设计开行时速250公里以上（含预留），并且初期运营时速200公里以上的客运列车专线铁路”<ref name = "中华人民共和国国务院令第639号">{{cite web | author = 国务院办公厅 | title = 中华人民共和国国务院令第639号 | publisher = 中央政府门户网站 | date = 2013-09-06 | language = zh | accessdate = 2014-11-02 | url = http://www.gov.cn/zwgk/2013-09/06/content_2482594.htm | quote = "本条例所称高速铁路，是指设计开行时速250公里以上（含预留），并且初期运营时速200公里以上的客运列车专线铁路。"}}</ref>，因此一部分的新建路線雖然符合國際定義（运营速度不小于200公里/小时<ref>蘇昭旭；世界高速鐵路百科{{ISBN|9789864610471}}</ref>）的高鐵，但官方並不認為是高鐵（如[[福廈鐵路|福廈鐵路]]、[[廈深鐵路|廈深鐵路]]等），而中国大陆新建的高铁路线仍持續兴建中，預計2025年，中国大陆将建成约38,000公里的高速铁路网<ref name=length>{{cite news|title=Chinese high speed network to double in latest master plan|url=http://www.railwaygazette.com/news/infrastructure/single-view/view/chinese-high-speed-network-to-double-in-latest-master-plan.html|publisher=Railway Gazette|date=2016-07-21|accessdate=2016-11-05}}</ref>。

中国人口众多和流动频繁，对作为大众化交通工具的高速铁路需求巨大。而中国各地气候和地形不一，也为高铁建设带来挑战。因此，中国将特有的线路建设、运营调度系统、通信和网络系统、机械、材料进行相互配合，建成了针对不同气候、不同地形的高铁，例如经过零下40度高寒地带的[[哈大高铁|哈大高铁]]、全球第一条环岛高铁和最南端的高铁、穿越热带地区和台风频发环境的[[海南环岛高铁|海南环岛高铁]]、经过大风区和戈壁沙漠的[[兰新高铁|兰新高铁]]和隧道长度超过总长度一半的[[贵广高铁|贵广高铁]]<ref name="gtsh">{{cite web |url = http://news.xinhuanet.com/mrdx/2015-01/26/c_133945792.htm|title=短短10年,中国悄然迈向“高铁社会” |publisher=新华每日电讯|date=2015-01-26|accessdate=2016-11-05}}</ref>。

中國高鐵使人们的生活和工作方式发生了重大变化，促進沿路地區的經濟發展，也使得中国进一步迈向“高铁社会”<ref name="gtsh"/><ref>{{cite web |url = http://www.china.com.cn/international/txt/2010-12/29/content_21637803.htm |title=华尔街日报：高铁改变中国人生活方式|publisher=中国网|date=2010-12-29|accessdate=2016-11-05}}</ref>，使相鄰省份成為一日生活圈。目前無运营時速200公里以上的鐵路运营的區域僅剩正在建設高鐵的[[宁夏回族自治区|宁夏回族自治区]]<ref>{{cite web |url = http://news.wenweipo.com/2015/10/11/IN1510110005.htm |title=寧夏首條高鐵開工 銀川2小時抵蘭州|publisher=文汇网|date=2015-10-11|accessdate=2016-11-05|language=zh-hk}}</ref>，以及暫無高鐵計劃的[[澳門|澳门特别行政区]]、[[西藏自治区|西藏自治区]]。在中国大陸，高铁路线包括客运专线、城际线、国铁一级线，以及既有改造线<ref name=globaltimes>{{cite web |url = http://www.globaltimes.cn/content/896819.shtml |title=High-speed rail a vital social good|publisher=环球时报|date=2014-12-15|accessdate=2016-11-05}}</ref>。而目前利用建設經驗所新研製的技術並能開始進軍海外，參與輸出高鐵的承建競爭，例如土耳其高速列車、印尼高鐵、美國西部快線等，但也有部分項目被擱置。

2013年2月1日实行的「[[铁路主要技术政策|铁路主要技术政策]]」从高速铁路中移除了[[既有线提速线路|既有线提速线路]]，[[高速铁路|高速铁路]]仅仅指新建设计开行250km/h（含预留）及以上[[普通动车组列车|动车]]组列车，初期运营速度不小于200km/h的客运专线铁路<ref>{{cite web |url =http://www.gov.cn/flfg/2013-02/20/content_2334582.htm |title=铁路主要技术政策（铁道部令第34号）|publisher=「中华人民共和国政府」网 |date=2013-02-20 |accessdate=2016-11-05}}</ref>。2014年1月1日施行的「[[铁路安全管理条例|铁路安全管理条例]]」中，高速铁路是指设计开行时速250公里以上（含预留），且初期运营时速200公里以上的客运专线（比[[I級鐵路|I級鐵路]]更高一級）。未能符合其中一項者，皆為[[快速鐵路|快速鐵路]]<ref>{{cite web |url = http://www.gov.cn/zwgk/2013-09/06/content_2482594.htm |title=铁路安全管理条例|publisher=「中华人民共和国政府」网 |date=2013-09-06 |accessdate=2016-11-05}}</ref>。

==发展历史 ==
中国大陸在[[高速铁路|高速铁路]]领域的发展较世界上部分[[发达国家|发达国家]]晚，起步较其晚了20至30年，但自21世纪以来，取得了迅速的发展<ref name=世界高铁看中国>{{cite web |url = http://world.people.com.cn/GB/57507/13496288.html |title=世界高铁看中国|publisher=人民网}}</ref>。

===早期规划===
[[File:N595ncd80.JPG|thumb]]运行的普速列车。]]
1964年，日本[[新干线|新干线]]通車后，中国便开始关注境外高铁的发展。1978年10月，时任[[中国共产党中央委员会副主席|中共中央副主席]]兼[[中华人民共和国国务院副总理|国务院副总理]][[邓小平|邓小平]]访问日本，在日方安排下乘坐[[東海道新幹線|東海道新幹線]]由东京赴京都，途中表示“像风一样快，我们现在很需要跑”，“真快，我们也需要快”。当时，中国铁路运输里程为5.2万公里，没有高铁线路，客运列车最高运行时速为每小时120公里，且因停站过多和经常晚点造成平均速度仅有每小时43公里<ref name=中国高铁技术6年跨越发达国家30年历程>{{cite web |url = http://news.sina.com.cn/c/sd/2009-12-22/041019308581.shtml |title=中国高铁技术6年跨越发达国家30年历程|publisher=广州日报}}</ref><ref>{{cite web |url = http://www.zhoushan.cn/newscenter/yzsp/201607/t20160725_793350.shtml |title=算一算高铁的经济账与政治账|publisher=浙江日报}}</ref>，而邓小平当时乘坐的[[新幹線0系電聯車|新干线0系电力动车组]]营运最高速度为每小时210公里。自此以后，在1980年代初，中国的铁路官员开始着手规划高铁建设。当时在日本研习期间的铁路系统官员[[张曙光_(铁道部)|张曙光]]观看了邓小平考察新干线的影片后，便有了着手规划高铁的念头，表示要让中国人也能坐上高铁<ref>{{cite web |url = http://news.takungpao.com/mainland/focus/2013-09/1897801.html |title=张曙光庭审披露：行贿者80万元购温家宝画像送张|publisher=大公网}}</ref>。而[[党和国家领导人|中国领导人]]以后每次访日，双方必定会谈到新干线，日方也会必定安排乘坐和参观新干线，同时将中日高速铁路建设方面的合作上升到了“日中友好的象征”的高度<ref name=qianlong>{{cite web|url=http://www.qianlong.com/2955/2003/08/06/53@1525197_1.htm|title=日本等待中国“求婚”|publisher=千龙网|deadurl=yes|archiveurl=https://web.archive.org/web/20120227040918/http://www.qianlong.com/2955/2003/08/06/53@1525197_1.htm|archivedate=2012-02-27}}</ref>。1984年，中国的一家出版社出版了《高速铁路》一书，是中国第一本高铁书籍，全书整理了日本、法国、德国等国高铁发展情况，书中前言中也写道：“曾经被称为衰老化的客运工具——铁路，因为高铁将获得新的生命契机”。这本书也为早期中国建设高铁的研究起到了一定作用<ref name=寻访/>。

中国大陸对高速铁路的研究实际始于1990年代初，当时[[京沪高速铁路|京沪高速铁路]]正处于构思阶段，而[[京沪铁路|既有线]]长期处于极度饱和状态。2007年，京沪既有线平均每公里客运密度为全国的5.2倍<ref>{{cite web |url = http://news.xinhuanet.com/life/2008-04/19/content_8007272.htm|title=京沪高铁开工计划5年建成 北京到上海只需5小时|publisher=北京晨报}}</ref>。1990年[[中华人民共和国铁道部|铁道部]]完成了《京沪高速铁路线路方案构想报告》并提交[[全国人大|全国人大]]会议讨论，这是中国大陸首次正式提出兴建高速铁路<ref name=温家宝宣布京沪高速铁路全线开工>{{cite web|url=http://www.gov.cn/jrzg/2008-04/18/content_947868.htm|title=温家宝宣布京沪高速铁路全线开工 并为铁路奠基|publisher=}}</ref>。1994年，当时的[[国家科委|国家科委]]、[[国家计委|国家计委]]、[[国家经贸委|国家经贸委]]、[[国家体改委|国家体改委]]和铁道部课题组完成了“京沪高速铁路重大技术经济问题前期研究报告”的深化研究，后成立京沪高速铁路预可行性研究办公室<ref name=温家宝宣布京沪高速铁路全线开工/>。

1995年，时任[[中国共产党中央政治局常务委员会|中共中央政治局常委]]、[[中华人民共和国国务院总理|国务院總理]][[李鵬|李鵬]]曾表示，[[九五計劃|九五計劃]]期間，中国將開始修建連接北京和上海的高速鐵路。由于龐大的建設費用，加上高速列車造价昂贵，当时他沒有提及具體的時間<ref name=高鐵王>{{cite web|url=http://news.wenweipo.com/2011/06/14/IN1106140032.htm|title=「高鐵王」:零配件國產化90%|publisher=香港文汇网}}</ref>。1995年4月5日李鹏在“建设全国统一的综合交通运输网络体系” 的报告中再次指出，京沪高速铁路在“九五”期间只能作前期准备工作<ref>{{cite web|url=|title=《建设全国统一的综合交通运输网络体系》|publisher=《中国交通报》|date=1996年12月23日}}</ref>。1996年3月，由李鹏发表、[[全国人大|全国人大]]批准的《国民经济和社会发展“九五”计划和2010年远景目标纲要》中，明确表示，“下个世纪前10年，集中力量建设一批对国民经济和社会发展具有全局性、关键性作用的工程。……着手建京沪高速铁路，形成大客运量的现代化运输通道。”京沪高速铁路的动工计划正式被推迟到21世纪<ref>{{cite web|url=http://www.nmgfgw.gov.cn/Content1.aspx?id=16252&columnid=126|title=中华人民共和国国民经济和社会发展“九五”计划和2010年远景目标纲要(1)|deadurl=yes|archiveurl=https://web.archive.org/web/20080102033215/http://www.nmgfgw.gov.cn/Content1.aspx?id=16252&columnid=126|archivedate=2008-01-02}}</ref>。

由于高速铁路相对具有运载能力大、运行[[速度|速度]]快、运输效率高等特点，而中国铁路此时面临的主要问题是[[客运|客运]]速度慢、运输能力严重不足，因此高速铁路越来越受到重视<ref name=中国速度/>。在中国大陸[[第九个五年计划|第九个五年计划]]（1996年－2000年）期间进行的三次[[中国大陸|中国大陸]][[铁路|铁路]]大提速的基础上，铁道部随后制定了《“十五”期间铁路提速规划》正式将高速铁路建设列入规划，《规划》提出：到“十五”末期，初步建成以[[北京|北京]]、[[上海|上海]]、[[广州|广州]]为中心，连接中國大陸主要[[城市|城市]]的全路快速客运网，总里程达16000公里<ref>{{cite web|url=http://www.chnrailway.com/news/2008516/2008516173294075730.shtml|title=中国需要高速铁路|publisher=}}</ref>；客运专线旅客列车最高时速达到200公里及以上，实现高速铁路、部分繁忙干线客货分线<ref>[http://jtyss.ndrc.gov.cn/fzgh/t20050726_37884.htm “十五”交通规划简介]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；而用于高速铁路车辆的[[交流电|交流电]]传动、[[动车组|动车组]]技术研究也同步进行，并开展时速270km高速动车组（[[中国铁路DJJ2型电力动车组|DJJ2]]）的研制<ref>[http://www.grandexh.com/railway2003/5-Y-Plan(Chinese).doc 铁路科技发展“十五”计划和2015年长期规划纲要 ] {{Wayback|url=http://www.grandexh.com/railway2003/5-Y-Plan(Chinese).doc |date=20060511060751 }}，中华人民共和国铁道部</ref>。

===铁路提速===
{{main|中国铁路大提速}}
[[File:Xinshisu.jpg|缩略图]]于1998年至2007年开行的[[SJ2000列车|新时速列车]]]]
[[File:0B6X8339_(8272950378).jpg|140px]]
1993年，中国大陆铁路旅客列车平均运行速度仅为48.1公里。为此，在1997年4月1日，中国大陆的铁路实施第一次提速调图，将列车最高运行时速提高到140公里，平均运行速度也提高到时速54.9公里，之后中国实行了四次铁路大提速<ref name=中国铁路的前六次大提速>{{cite web|url=http://news.xinhuanet.com/newscenter/2009-03/31/content_11107242.htm|title=新闻背景：中国铁路的前六次大提速|publisher=新华网}}</ref>。

最著名的提速案例为[[广深铁路|广深铁路]]的电气化提速改造。1980年代末，当时的铁道部决定从国情实际出发，以时速160公里的准高速铁路作为突破口，为未来发展高速铁路进行探索和试验。选定试验区段对既有线路进行技术改造，可以用较少的投资、较短的时间将既有线路改造成能够开行时速160公里旅客列车的准高速铁路。而当时的广深铁路具有其它铁路线所没有的优越条件，包括不断增长的运输需求、地形条件较好、长度适中等，使其成为首要的试验目标<ref>{{cite journal | author = 龚深弟 | title = 铁道部通过广深线旅客列车最高时速160km的实施方案 | journal = 《中国铁路》| pages =  | publisher = 铁道部科学技术信息研究所 | location = 北京 | issn = 1001-683X |date=1990年9月}}</ref>。1989年，铁道部成立了由[[中国铁道科学研究院|中国铁道科学研究院]]和[[广州铁路局|广州铁路局]]组成的联合专家组，对广深线旅客列车的最高速度提高到160km/h进行了前期可行性研究<ref name=tdzs1995>{{cite journal | author = 葛闻安 | title = 广深准高速铁路——中国铁路高速化的起点 | journal = 《铁道知识》 |  pages = 8-9 | publisher = 中国铁道学会 | location =北京 | issn = 1000-0372|date=1995年3月}}</ref>。但由于1990年代初进行准高速改造时资金不足，故推迟了电气化的实施，在过渡阶段使用内燃机车牵引。至1996年，广深铁路总公司改制为广深铁路股份有限公司，广深铁路运用发行[[H股|H股]]募集资金及自有资金持续投入建设准高速铁路、高速电气化及配套工程<ref>{{Cite web |url=http://www.gsrc.com/big5/article.php?article_id=102 |title=发展历程 |accessdate=2011-05-27 |publisher=广深铁路股份有限公司 |date= }}</ref>。1997年2月，总投资8亿元人民币的广深线高速电气化工程全面开工；1998年5月28日，广深线高速电气化工程竣工，同年8月28日正式投入运营。广深铁路率先使用由[[瑞典|瑞典]]租賃、最高營運速度达200km/h、被命名为「[[新時速|新時速]]」的[[X2000列車|X2000擺式高速动车组]]运行广深城际列车<ref name=shenzhenjtz>{{cite book |author=閻志禎、李漢君 |title=深圳市交通運輸志 |year=2001 |publisher=方志出版社 |location=北京 |isbn=7-80122-699-2 |page=}}</ref>。由于全线采用了众多达到1990年代国际先进水平的技术和设备，因此当时广深铁路被视为中国由既有线改造踏入高速铁路的开端<ref>{{cite web|url=http://3g.ifeng.com/news/sharenews.f?vt=5&aid=91795241&mid=|title=高铁改变中国|publisher=凤凰网}}</ref>。1998年6月，[[韶山8型|韶山8型]]电力机车于[[京广铁路|京广铁路]]的区段试验中达到了时速240公里的速度，创下了当时的“中国铁路第一速”，是中国大陸第一种高速[[铁路机车|铁路机车]]<ref name=中国速度>{{cite web|url=http://cjb.newssc.org/html/2012-05/11/content_1578411.htm|title=中国速度|publisher=经理日报|deadurl=yes|archiveurl=https://web.archive.org/web/20161031155235/http://cjb.newssc.org/html/2012-05/11/content_1578411.htm|archivedate=2016-10-31}}</ref>。

2007年4月18日，中国大陆铁路开始实施第六次铁路提速，涉及京哈、京沪、京广、陇海、沪昆、胶济、广深、京九、兰新等铁路干线。提速后，线路允许速度达时速120公里以上的线路延展里程达2.2万公里，时速200公里及以上的线路延展里程达6003公里，时速250公里的线路延展里程达846公里<ref name=中国铁路的前六次大提速/>。此次提速使中国铁路开始大规模投入使用动车组列车，也使中国铁路跨入高速时代。从2007年4月18日起，中国大陆铁路陆续开行140对[[普通动车组列车|普通动车组列车]]，陆续使用158列[[CRH1|CRH1]]、[[CRH2|CRH2]]和[[CRH5|CRH5]]电力动车组。当时动车组列车票价参照的是1997年原[[国家计委|国家计委]]公布的高等级软座快速列车票价水平，一等车客票基准价为每公里0.3366元（人民币，下同）；二等车客票基准价为每公里0.28元<ref>{{cite web|url=http://news.qq.com/a/20070413/000749.htm|title=铁路18日第六次大提速 中国铁路跨入高速时代|publisher=新华网}}</ref><ref>{{cite web|url=http://www.gov.cn/jrzg/2007-04/12/content_580527.htm|title=铁路第6次提速4大亮点：更快、更多、更好、更省|publisher=新华社}}</ref>。

===技术之争===
1998年6月2日，在[[中国科学院|中国科学院]]和[[中国工程院|中国工程院]]两院院士大会上，时任国务院总理[[朱镕基|朱镕基]]向在座的一千多位院士提出“京沪高速铁路是否可以采用磁悬浮技术？”的问题<ref>{{cite web|url=http://www.china.com.cn/chinese/2003/Mar/288673.htm|title=磁悬浮的中国困惑}}</ref>。此后，以[[何祚庥|何祚庥]]为代表的一批科学家开始支持磁悬浮，而铁道部则支持采用轮轨技术。京沪高铁项目论证也出现了“磁悬浮派”和“轮轨派”。当时[[中共中央|中共中央]]提出的“跨越式发展”口号，也是这一争论的背景。为了证明轮轨高速铁路的优越性，铁道部于1998年开始设计生产200公里电力动车组。而科技部也在2000年初成立了磁悬浮预可行性研究小组，同年年底磁悬浮项目获得立项<ref name=dzwww>{{cite web|url=http://www.dzwww.com/special/ts/jdf/200912/t20091230_5293551_2.htm|title=中国高铁“和谐号”前传|publisher=大众网}}</ref>。2000年6月，上海市政府决定在上海建设高速磁悬浮<ref>{{cite web|url=http://www.smtdc.com/cn/gycf4.html|title=立项背景-SMT-上海磁浮|access-date=2016-10-31|archive-url=https://web.archive.org/web/20151127003554/http://www.smtdc.com/cn/gycf4.html|archive-date=2015-11-27|dead-url=yes}}</ref>，尽管磁悬浮速度有无与伦比的优势，但磁悬浮还没有在中国的高速铁路网络得到广泛使用。因此，上海磁浮的盈亏情况、磁悬浮的[[电磁辐射污染|电磁辐射]]、[[噪音污染|噪音]]等对人体健康的影响均是专家和民众关注和争议的焦点<ref>{{cite web|title=上海磁浮优化案仍引争议 有公众称健康已受影响|url=http://news.xinhuanet.com/life/2008-01/17/content_7437917.htm|publisher=新华网、法制日报|date=2008年1月17日|accessdate=2013-10-01}}</ref><ref>{{cite web|title=8号线磁悬浮的4年之争|url=http://epaper.nfdaily.cn/html/2013-06/20/content_7199654.htm|publisher=南方日报|date=2013年6月20日|accessdate=2013-10-01|archive-url=https://web.archive.org/web/20131004215950/http://epaper.nfdaily.cn/html/2013-06/20/content_7199654.htm|archive-date=2013-10-04|dead-url=yes}}</ref><ref>{{cite news|url=http://www.reuters.com/article/2008/01/12/us-china-maglev-protest-idUSPEK32757920080112|title= 
Hundreds protest Shanghai maglev rail extension|date=2008-01-12|journal=Reuters}}</ref>。

1999年，[[秦沈客运专线|秦沈客运专线]]开始建设。该线于2003年建成运营，有铁道部官员认为，[[秦沈客运专线|秦沈客运专线]]是中国第一条真正意义上的高速铁路，但中国[[国家铁路局|国家铁路局]]官网未予以承认<ref>[http://www.nra.gov.cn/ztzl/hyjc/gstl_/zggstL/201602/t20160215_21046.html 中国高速铁路].國家鐵路局.</ref><ref>[http://www.nra.gov.cn/ztzl/hyjc/gstl_/zggstL/gstllc/201602/t20160216_21067.html 中国高速铁路发展历程].國家鐵路局.</ref>。为了挑战外资主导地位，在2002年，铁道部推出了完全利用自主知识产权、花费1.4亿元人民币研发的动车组列车“中华之星”。[[中华之星|中华之星]]在秦沈客运专线创造了每小时321.5公里的最快速度纪录<ref name=中国速度/><ref>{{cite web|title=中国高铁断代史： “第一高铁”秦沈客专如何被湮没|url=http://finance.sina.com.cn/roll/20110611/00069974415.shtml|publisher=21世纪经济报道|date=2013年6月20日|accessdate=2013-10-01}}</ref><ref name=ft>{{cite web|title=中国是怎样“消化”高铁技术的？（上）|url=http://www.ftchinese.com/story/001034854?full=y|publisher=FT中文网|date=2010-10-08|accessdate=2013-10-01}}</ref>。中国也先后研发了[[DDJ1型电力动车组|DDJ1型电力动车组]]、[[先锋号电力动车组|先锋号电力动车组]]、[[DJF3型电力动车组|DJF3型电力动车组]]等电力动车组，但最终以失败告终<ref name=dzwww/>。

2003年，[[刘志军|刘志军]]担任铁道部长，提出“铁路跨越式发展”并大力推进高速铁路技术进步，甚至不惜大规模负债建设、非法挪用资金及牺牲铁路系统职工待遇，中国高速铁路因此开始进入高速发展时期，而高速铁路技术争论以铁道部胜出告终<ref name=dzwww/>。2004年1月，国务院批准中国大陸第一个《中长期铁路网规划》<ref name=2004中长期铁路网规划>{{cite web|url=http://www.gov.cn/ztzl/2005-09/16/content_64413.htm|title=国家《中长期铁路网规划》内容简介|publisher=}}</ref>，正式宣布规划建设里程超过1.2万公里的客运专线，客车速度目标值达到每小时200公里及以上<ref name=2004中长期铁路网规划/>。同时《规划》也提出，对京沪高速铁路和其他三条南北向高速铁路线采用轮轨技术，为中国高速铁路快速建设铺平了道路<ref>{{cite web|url=http://english.peopledaily.com.cn/200401/16/eng20040116_132743.shtml |title=Rail track beats Maglev in Beijing–Shanghai High Speed Railway |work=People's Daily  |date=2004-01-18 |accessdate=2011-10-17}}</ref><ref name = China2>{{cite web|url=http://www.railway-technology.com/projects/beijing/ |title=Beijing–Shanghai High-Speed Line, China |publisher=Railway-technology.com |date=2011-06-15 |accessdate=2011-10-17}}</ref>。中国政府认为，磁悬浮和既有线路的制式不统一，无法联网。其次造价也要比轮轨昂贵很多。另外磁悬浮目前投入商用的只有上海磁悬浮，经验不足。而已有的轮轨技术则已经十分成熟，在对环境带来的影响上，也比磁悬浮更有优势<ref name=中国高铁技术6年跨越发达国家30年历程/>。

===技术引进===
[[File:CRH1_EMU_Head.jpg|缩略图]]型电力动车组]]
[[File:CRH2-011A.JPG|缩略图]]型电力动车组]]
尽管“中华之星”等自主研发的电力动车组的最高时速创造了历史记录，但由于当时中国高铁的设计要求低于国外高速铁路标准，整体核心技术不成熟，距离商业运营还有很大距离，同时一些核心关键技术和设备可靠性、稳定性不足，列车的工艺水平比较差，在试验不到兩年後就放棄了商業化，因此当时自主研发的电力动车组很快隕落。于是，中国政府从2004年起，改從外國企業引進高速列車<ref name=高鐵王/><ref name=诞生与成长>{{cite web|title=中国式高铁的诞生与成长|url=http://news.xinhuanet.com/globe/2010-03/04/content_13096289.htm|publisher=《环球》杂志|date=2010-03-04|accessdate=2016-10-16}}</ref>。

早在1998年11月，时任中共中央总书记兼国家主席[[江泽民|江泽民]]访日期间，日方安排江泽民乘坐[[东北新干线|东北新干线]]。时任日本内阁总理大臣[[小渊惠三|小渊惠三]]对江泽民说，中国的高速铁路项目，日本将“官民并举，倾力协作”。2000年4月，日本运输省官员曾表示中国建设京沪高铁项目时，日本不仅可以向中国出口车辆，而且要向中国转移新干线系统的技术<ref name=qianlong/>。2003年4月，据《[[日本经济新闻|日本经济新闻]]》报道，中国高层曾向日本铁路业者表示“中国选定新干线的可能性为95%”。此消息发布后，引发中国[[民族主义|民族主义]]者的反响，他们表示反对采用这一技术，并在网上发起签名活动<ref name=qianlong/>。

2004年起，中国政府开始对铁路动车组进行公开招标。2004年6月17日，《[[人民铁道|人民铁道]]》和中国采购与招标网同时发布招标公告，铁道部拟采购时速200公里的动车组，共计10包200列。公告明确投标主体是国内企业，但它必须取得国外先进技术的支持。最初铁道部与拥有新干线[[新干线700系电力动车组|700系]]及[[新干线800系电力动车组|800系]]技术的[[日本车辆制造|日本车辆制造]]及[[日立製作所|日立製作所]]洽商，但日车及日立均表明拒绝向中国出售车辆及技术转移。其后中方改向[[川崎重工业|川崎重工业]]洽商，当时川崎重工业销售业绩并未如预期理想，便出售3组[[新干线E2系电力动车组|E2系]]动车组及其车辆技术予中国。此举最初仍遭到[[JR东日本|JR东日本]]、日车及日立反对，后经一轮谈判，川崎重工业在其他日本公司不反对的情况下，向中国出售E2系车辆及技术。中国铁道部和有关部门做了大量的引导工作，保证了招标工作按计划推进<ref name=诞生与成长/>。

2004年10月，川崎代表“日本企业联合体”与中国铁道部签订出口铁路车辆、转让技术的合同。中国方面向日本川崎重工订购为数60列时速200公里级别的动车组，总价值93亿元人民币。其中3列在日本完成，另有6列以散件形式付运，由中方负责组装，其余51列将通过技术转让，由[[中车四方|青岛四方机车车辆厂]]建造，当时一些高技术部件仍然依赖进口。2005年8月，铁道部向四方机车、川崎重工、三菱电机、[[株洲所|株洲所]]、[[株洲南车电机|株洲南车电机]]及石家庄国祥运输设备六方签订51列CRH2A型动车组机电产品技术转让合同。首列CRH2A于2006年3月1日从神户港装船，3月8日运抵青岛。而中国利用龐大的市場，通过招标和技术引进，开始在国内生产[[CRH1|CRH1]]、[[CRH3|CRH3]]和[[CRH5|CRH5]]。其中，西门子公司向[[长春轨道客车|长春轨道客车]]招标，提出引进“维拉罗E”的技术，但西门子因为提出高昂的转让技术、车辆造价费用而无法在第一轮招标获得任何订单。2005年，西门子参加第二轮竞标。当时中方给出更严格的条件。最后西门子完全接受中方的技术转让方案和价格方案，和[[唐山轨道客车|唐山轨道客车]]进行合作，生产出[[CRH3|CRH3]]列车。这两次招标，使得中国高铁同时引进了四家的先进技术，为日后的国产化道路打下了基础<ref name=高鐵王/><ref name=诞生与成长/>。

中国的高速铁路网络建设，均由政府和国有企业管理、规划和资助。

===建设意义===
中国和境外的批评者一直质疑在大部分发展中国家建设高昂的高速铁路系统的必要性，而大多数务工人员则无力承担高铁的车票费用<ref name=NYT3>{{cite news|last=Forsythe |first=Michael |url=http://www.nytimes.com/2009/12/23/world/asia/23iht-letter.html |title=Michael Forsythe "Letter from China: Is China's Economy Speeding Off the Rails?" |location=China |publisher=Nytimes.com |date=2009-12-22 |accessdate=2011-08-17}}</ref>。然而，中国政府已经表明，虽然高铁建设造价昂贵，但这些建设项目也会推动一些政策目标。由于中国幅员辽阔，人口众多和流动频繁，因此对作为大众化交通工具的高速铁路需求巨大，也为旅客提供快速，可靠和舒适的出行方式<ref name="gtsh"/>。

虽然中国高铁的运营时间比发达国家要短，但“高铁经济效应”已明显显现，对沿线产业带和城市现代服务业的培育，以及沿线地区人口流动速度提升和人口聚集，均具有重要促进作用。在高铁规模快速扩张的时期，高铁的空间效应将进一步显现<ref name=sdpc/>，也成为推动国家经济社会发展的强大引擎<ref name=50亿人次>{{cite news|url=http://www.gov.cn/shuju/2016-07/21/content_5093413.htm |title=中国高铁动车组发送旅客突破50亿人次 旅客发送量年均增长30%以上|publisher=央广网 |date=2016-07-21 |accessdate=2016-07-21}}</ref>。具体体现在以下方面：

*通过提升铁路的运输能力，与就业市场实现无缝对接，以提高经济[[生产率|生产力]]和长期[[竞争力|竞争力]]<ref name=NYT3/><ref name=NYT1>{{cite news|last=Bradsher |first=Keith |url=http://www.nytimes.com/2009/01/23/business/worldbusiness/23yuan.html?_r=1&pagewanted=all |title=Keith Bradsher, "China's Route Forward" |location=China |publisher=Nytimes.com |date=2009-01-22 |accessdate=2011-08-17}}</ref>。另外，高铁开通后，一些既有线释放了运能，缓解了货运能力长期紧张的局面，全社会人流、物流周转明显加快，成本有效降低，这对于铁路更有利可图<ref name=50亿人次/><ref name=autogenerated5>{{cite news|last=Freeman |first=Will |url=http://online.wsj.com/article/SB10001424052748704025304575283953879199386.html |title=Freeman & Kroeber, "Opinion: China's Fast Track to Development" |publisher=Wall Street Journal |date=2010-06-02 |accessdate=2011-08-17}}</ref>。
*在经济衰退时期，由于高铁建设创造就业机会，也推动了对建筑、钢铁和水泥行业的需求，短期刺激经济。其中，京沪高铁共聘用了11万工人施工<ref name=NYT1/><ref name=NYT2>{{cite news|last=Bradsher |first=Keith |url=http://www.nytimes.com/2010/02/13/business/global/13rail.html?pagewanted=all |title=Keith Bradsher, "China Sees Growth Engine in a Web of Fast Trains"|location=China; United States |publisher=Nytimes.com |date=2010-02-12 |accessdate=2011-08-17}}</ref><ref name=amazing>{{cite news|url=http://money.cnn.com/2009/08/03/news/international/china_high_speed_bullet_train.fortune/|title=China's amazing new bullet train|location=|publisher=CNN}}</ref>。
*促进跨城市经济一体化及[[二线城市|二线城市]]的增长。高铁的引入，使得二线城市的市场潜力增加59％，楼价也比预期平均实际增长4.5％<ref>{{cite news | url=http://journalistsresource.org/studies/international/china/china-high-speed-rail-economic-development | title=China’s high-speed-rail network and the development of second-tier cities}} </ref>。高铁的开通也产生“同城效应”，实现区域资源共享，加快产业梯度转移，有效推动区域内产业优化分工，围绕构建高铁沿线产业链条，形成比较优势，促进沿线地区的产业协调互补发展。例如随着长三角地区高速铁路的不断开通，带动了长三角地区协同分工、错位发展、有序的产业体系逐步完善，有效支撑并在很大程度上引导了各城市不同的产业体系等的发展。此外，高铁也使中国人的生活半径和活动范围出现明显扩大和拓宽的现象，生活方式和节奏逐渐发生变化，催生出“星期天工程师”“假日专家”等新职业，形成了“高铁社会”。同时使老年人异地养老变成现实，如廊坊、昆山、德州等地的养老院开始吸引北京、上海的老年人，以更低的成本享受更舒适的养老生活<ref name="gtsh"/><ref name=sdpc/><ref name=50亿人次/>。
*支持环境可持续发展。电力动车组能源使用量比其他交通工具要少，并且可以从更多种能源（包括[[可再生能源|可再生能源]]）获取电力。而汽车和飞机没有这些特点，因为这些能源更依赖于[[各国石油进口量列表|进口石油]]<ref name=autogenerated5/>。
*发展高铁设备产业。高铁的建设，也将中国发展成为高速铁路建设技术的主要来源国<ref name=NYT2/>。中国的轨道装备制造商引进进口技术后，直接进入本地化生产过程，之后进行了技术转化，甚至开始准备出口海外，与外国供应商竞争。 在中车四方取得[[川崎重工业|川崎重工业]]新干线E2型授权生产权的六年后，中车四方可以自行生产CRH2A列车，而川崎重工业也在高铁上结束与中车四方的合作<ref>{{cite web|url=http://www.ft.com/cms/s/0/60da2222-8ab5-11df-8e17-00144feab49a.html |title=Japan Inc shoots itself in foot on bullet train|publisher=Ft.com |date=2010-07-08 |accessdate=2011-08-17}}</ref>。如今，中国拥有世界上系统技术最全的高速铁路技术，可以承担从通信信号、工务工程、牵引供电、机车客车制造乃至运营管理等领域的“一揽子”出口<ref name=走向世界的中国技术>{{cite web|url=http://news.xinhuanet.com/science/2015-12/24/c_134946932.htm |title=走向世界的中国技术——高铁 （下）|publisher=科普中国 |date=2015-12-24 |accessdate=2015-12-24}}</ref>。
*拉动产业的发展。高铁的建成，带动了冶金、机械、建筑、橡胶、电力、信息、计算机、精密仪器等第二产业的快速发展，也拉动了沿线城市旅游、餐饮、商贸等第三产业的发展。例如[[京津城际铁路|京津城际铁路]]开通后，使两个直辖市及周边地区的经济交流日益增多；京沪高铁开通运营后，沿线城市成为承接长三角和环渤海两大经济区产业转移的新平台<ref name=50亿人次/>。
*带动新型[[城镇化|城镇化]]发展。随着一些高铁线路客流量的增长，充分说明了高速铁路对人口流动具有显著的诱增效应，使原先鲜为人知或知名度高但交通不便的中小城镇成为吸纳人口的热点<ref name=sdpc/>。

中國自从經濟步入“[[新常態|新常態]]”后，高铁建设从之前的勞動密集型，向資本技術密集型轉化升級<ref>{{cite web|url=http://www.hkcna.hk/content/2015/0503/362643.shtml |title=中國高鐵開闢陸權時代新絲路|publisher=《紫荊》 |date=2015-05-03 |accessdate=2015-12-24}}</ref>。在中国全部投入建设高铁的款项中，有约40-50％是由中央政府通过国有银行和金融机构贷款提供，而另外40％由原铁道部发行的债券提供，而其余10-20％则由省级和地方政府提供<ref name=autogenerated20111026/><ref name=autogenerated5/>。

高铁建设完毕后，除了需要通过票款偿还贷款外，还需要通过车站及列车服务、线路使用、广告获得大量收入来偿还<ref>{{cite web|url=http://finance.sina.com.cn/roll/20110408/23389660922.shtml|title=京沪高铁资金成本每日2630万 能否盈利有待观察}}</ref>。尽管载客量令人瞩目，但几乎每条高铁线路在其运营的第一年都遭受了亏损。以京津城铁为例，该路线开通两年来，累计开行列车10.4万列，运送旅客4100万人。而京津城铁的修建投资额达204.2亿元，每年的运营总支出约18亿元，其中包括银行贷款100亿和每年6亿多的还贷利息，年利息利率在6.3%至6.8%之间，贷款周期为5-10年<ref name=autogenerated1>{{cite web|url=http://finance.ifeng.com/news/20100918/2634606.shtml|title=不计建设投资 京津高铁今年持平}}</ref>。在2008年8月1日至2009年7月底，京津城铁共运送旅客达1870万人，平均每天运送旅客5.5万人，营收11亿元，亏损7000万；而在开通运营的第二年，京津城铁客运量增加到2230万人，收入增加到14亿元，亏损5000万。根据京津城际高铁项目建议书，京津高铁预期的设计年运量约3000万人次，营业额接近18.4亿。但在投入运营时，出现盈亏持平，甚至略有盈余的迹象。据华建管理公司的会计师文清路分析，京津城铁在年运营费用支出达18亿的基础上，需要实现年运送旅客量在4000万人次以上，才能实现“算小账”中的略有盈余<ref name=autogenerated1/>。2013年，京津城铁客运量上升到2585万人次<ref>{{cite web|url=http://news.hexun.com/2014-12-20/171610757.html?from=rss|title=京津城际高铁二线拟明年开工 或通过三河、香河、大厂}}</ref>。该线路每年最大载客量可达1亿人次<ref name=autogenerated2>{{cite web|url=http://finance.ifeng.com/roll/20100930/2675126.shtml|title=4万公里快速铁路网冲刺}}</ref>，初期成本的回收期为16年<ref name=autogenerated1/>。

在其他线路方面，石太客运专线开通运营后的第一年，总亏损8亿元，到2010年，当年预计亏损9亿元；2009年9月开通的沿海铁路，当年亏损达3.77亿<ref name=打包高铁资产/>。郑西客运专线自2010年2月开通运营以来，预计2010年全年收入为6亿，但必须支付11亿元的利息。在2012年前三季度共亏损18.7亿元<ref>{{cite web|url=http://www.yicai.com/news/2014/12/4055850.html|title=火爆城际铁路的上座率考验|access-date=2014-12-27|archive-url=https://web.archive.org/web/20141227062723/http://www.yicai.com/news/2014/12/4055850.html|archive-date=2014-12-27|dead-url=yes}}</ref>。虽然这些线路所造成的亏损由铁路部门承担，但一些地方政府也会采取补贴的方式（如成灌铁路），此举可以提高该线路的利用率<ref name=autogenerated2/>。2014年12月，河南省政府将对郑开城际铁路、郑焦城际铁路、郑机城际铁路实行暂定5年的运营亏损补贴机制<ref>{{cite web|url=http://hn.ifeng.com/jingji/zhongyuanjingji/detail_2014_12/11/3272664_0.shtml|title=河南对城际铁路实行运营亏损补贴 补亏期暂定5年}}</ref>。

根据2010年预测，从2014年开始，铁道部（现中国铁路总公司）将迎来还债高峰<ref name=打包高铁资产/>。一些经济学家也建议高铁线路需要通过进一步政府补贴的方式，以降低票价，提高乘客量和最终收入<ref name=autogenerated2/>。

由于中国高速铁路网络不断扩大，因此乘客人数也在不断增加。而高铁票价日后也会更加便宜，同时工资也会出现大幅增长的迹象<ref name="Speedy Trains Transform China">{{cite news|last=Bradsher|first=Keith|url=http://www.nytimes.com/2013/09/24/business/global/high-speed-train-system-is-huge-success-for-china.html|accessdate=2013-09-26|newspaper=纽约时报|date=2013-09-24|title=Speedy Trains Transform China}}</ref>。

===车站规划===
根据2009年版的《高速铁路设计规范》规定，车站名称应与车站所在地地名一致，且在全国范围内不应有相同的车站名称。位于城市城区或近郊区的，应使用该城市的名称命名。当车站位于县、乡（镇）或村所在地时，应以当地县、乡（镇）或村庄的名称命名。而在2014年版《高速铁路设计规范》则删除了这些内容。因此在中国大陆，有一些高铁站的实际位置，与站名标识的位置相距较远<ref>{{cite web|title=此武夷山站到不了武夷山 高铁站名中的“李逵”与“李鬼”|url=http://www.infzm.com/content/119668|publisher=南方周末|accessdate=2016-09-19}}</ref>。

2018年5月8日，[[国家发改委|国家发改委]]、[[中华人民共和国自然资源部|自然资源部]]、[[住建部|住建部]]与[[中国铁路总公司|中国铁路总公司]]联合发布《关于推进高铁站周边区域合理开发建设的指导意见》，明确新建铁路选线应尽量减少对城市的分割，新建车站选址尽可能在中心城区或靠近城市建成区<ref>{{cite web|title=高铁车站选址尽可能在中心城区|url=http://news.ifeng.com/a/20180508/58193930_0.shtml|publisher=凤凰网|accessdate=2018-05-09}}</ref>。

===线路命名===
中国高速铁路的命名比较混乱，一些民众、专业媒体，甚至铁路部门都没有完全对所有的名词概念有一个全面、准确的理解。主要路线命名有<ref>{{cite web|url=http://news.mydrivers.com/1/409/409052.htm|title=高铁与动车有什么区别？|publisher=}}</ref><ref>{{cite web|url=http://society.people.com.cn/n/2014/1021/c372093-25876513.html|title=这次彻底明白：白话说说高铁站选址的科学|publisher=}}</ref>：
#在2009年12月前发布《高速铁路设计规范》之前，与既有线路走向平行的路线以“XX客运专线”命名。
#在2009年12月后发布的《高速铁路设计规范》规定，与既有线路走向平行的路线以“XX高速铁路”取代“客运专线”这一称谓命名<ref>GB/T 10621-2009《高速铁路设计规范》 "高速铁路：新建铁路旅客列车设计最高行车速度达到250km/h及以上的铁路。</ref>。
#在城市之间开行旅客列车的铁路线，线路长度相对较短，站间密度较大，设计旅客发送量大，车站位置位于市中心的路线，以“XX城际铁路”命名。
#新建的电气化线路，或对既有线路实行电气化改造的线路，如[[杭深线|杭深线]]、[[宁蓉线|宁蓉线]]<ref>{{cite web|url=http://view.news.qq.com/zt2011/DGC/index.html|title=动车与高铁 区别何在|publisher=腾讯新闻|deadurl=yes|archiveurl=https://archive.is/20161030080845/http://view.news.qq.com/zt2011/DGC/index.html|archivedate=2016年10月30日|df=}}</ref>。

===售票方式===
{{seealso|中国铁路客运车票与票价}}

从2011年6月1日起，全国所有动车组列车将实行购票[[中国大陆火车票实名制|实名制]]。乘客需携带有效身份证件方可购票，有效证件包括：[[居民身份证|居民身份证]]、按规定可使用的有效[[护照|护照]]、[[港澳居民来往内地通行证|回鄉證]]、[[中华人民共和国来往港澳通行证|港澳通行证]]、[[台湾居民来往大陆通行证|台胞證]]、[[港澳台居民居住证|港澳台居民居住证]]等<ref>{{cite news|title=配合实名制京津城际快通卡升级 |url=http://www.peoplerail.com/rail/show-456-47766-1.html |publisher=人民铁道报 |date=2011-05-21 |accessdate=2016年3月28日 }}</ref><ref>{{cite news|url = http://news.xinhuanet.com/fortune/2010-01/10/content_12783988.htm|title =铁路部门详解春运火车实名购票流程|accessdate =2010-01-10 |author =新华闻综合|date =  2010-01-10|language = zh-hans|publisher =[[新华网|新华网]]}}</ref>。

对于[[自动售票机|自动售票机]]的购票者而言，由于自2011年6月1日起[[中國大陸|中國大陸]]全面实行购票[[火车票实名制|实名制]]，因此全国各地[[火车站|火车站]]自动售票机也需要使用有效证件购票，但目前绝大多数火车站售票机能读取[[中华人民共和国居民身份证|中华人民共和国居民身份证]]、[[港澳居民来往内地通行证|回鄉證]](2012版)、[[台湾居民来往大陆通行证|台胞證]](只要有晶片)、[[港澳台居民居住证|港澳台居民居住证]]、[[中華人民共和國外國人永久居留身份證|外國人永久居留身份證]]。若持其他身份证明文件如：[[护照|护照]]、[[港澳居民来往内地通行证|回鄉證]]、[[中华人民共和国来往港澳通行证|港澳通行证]]、[[台湾居民来往大陆通行证|台胞證]]等乘客需到人工售票窗口购票<ref>{{cite web|url=http://www.legaldaily.com.cn/index/content/2012-01/02/content_3259944.htm?node=20908|title=哈尔滨火车站自动售票机只认二代身份证|publisher=哈尔滨日报}}</ref>（不过[[广深动车组列车|广深动车组列车]]沿线车站的自动售票机则可以使用护照、港澳通行证和回鄉證购票<ref>{{cite web|url=http://hk.apple.nextmedia.com/international/art/20110601/15302532|title=港人要用回鄉證 廣深線實名制售車票|publisher=香港苹果日报}}</ref>）。之后，铁路部门逐步扩大加装卡式回乡证和台胞证专用识读设备自动售（取）票机的范围和数量，实现省级城市所在地车站、广东省境内高铁车站、京广、京沪、杭深线所有高铁车站的全覆盖，方便持回乡证和台胞证的旅客购票。截至2018年12月，已有500多个车站的2000多台自动售（取）票机提供该项自助服务<ref>{{cite web|title=12306试点候补购票服务 新功能将终结抢票软件混战|url=http://www.xinhuanet.com/fortune/2018-12/22/c_1123889861.htm|publisher=北京晚报|author=李博|accessdate=2018-12-22}}</ref>。

持[[中华人民共和国居民身份证|第二代居民身份证]]于网络购票的旅客可选择前往各[[火车站|火车站]]、代售点领取传统纸质票，或直接使用第二代居民身份证入闸上车（需上车站和下车站均具备居民身份证检票条件）<ref>{{cite web|url=https://kyfw.12306.cn/otn/gonggao/enterAndExit.html|title=常见问题|publisher=中国铁路客户服务中心}}</ref>。

====购票限制====
至今为止，目前仅被人民法院限制消费的[[失信被执行人|被执行人]]（老赖）在购买动车组列车车票时，将会被受到限制。2015年8月25日起，所有被[[中华人民共和国法院列表|各级人民法院]]限制消费的被执行人及单位被执行人的法定代表人、主要负责人、影响债务履行的直接责任人员、实际控制人，均不能购买G字头列车（[[高速动车组列车|高速动车组列车]]）全部座位及其他动车组列车一等座以上座位。此行为被人民法院定位为“高消费行为”<ref>{{cite news|title=25日起“老赖”不得乘坐g字头动车组列车 |url=http://difang.gmw.cn/newspaper/2015-08/27/content_108851168.htm |publisher=城市晚报电子版|date=2015-08-27  }}</ref>。

另外，在2016年8月，有媒体传闻称从2016年8月15日起所有在动车组列车吸烟的乘客，除接受罚款外，其动车组乘坐权也被暂停，旅客需持本人身份证件到铁路客户服务中心签订协议书方可再次购买。若旅客签订协议书后，再次在动车组上违法吸烟的，将终身禁止购买动车组列车车票<ref>{{cite news|title=动车组最严禁烟新规:车上吸烟被抓两次终身禁乘 |url=http://news.qq.com/a/20160823/030572.htm |publisher=贵阳日报|date=2015-08-23  }}</ref>。但在2016年9月28日，中国铁路总公司回应称，铁总从未出台过这种规定，此政策属于误传<ref>{{cite news|title=中国铁总：“动车组吸烟将被终身禁乘”是误传|url=http://news.qq.com/a/20160930/017869.htm |publisher=法制网|date=2016-09-30  }}</ref>。

2018年3月16日，[[国家发改委|国家发改委]]发布《关于一定期限内适当限制特定严重失信人乘坐火车和飞机的意见》，规定对扰乱铁路站车运输秩序且危及铁路安全、造成严重社会不良影响的、在动车组列车上吸烟或者在其他列车的禁烟区域吸烟的、查处的倒卖车票、制贩假票的，以及依据相关法律法规应予以行政处罚的旅客，各铁路运输企业将限制涉事旅客购买动车组列车车票，为期180天。另外，对冒用优惠（待）身份证件、使用伪造或无效优惠（待）身份证件购票乘车的、持伪造、过期等无效车票或冒用挂失补车票乘车的、无票乘车、越站（席）乘车且拒不补票的旅客，各铁路运输企业将限制涉事旅客购买动车组列车车票。涉事旅客补齐所欠票款后（自补票次日算起），铁路运输企业恢复对其发售车票；涉事旅客补齐第一次所欠票款一年内，三次发生上述行为的，行为责任人补齐所欠票款90天后（含90天），铁路运输企业恢复发售车票，不补齐所欠票款，铁路运输企业不对其恢复发售车票。意见同时规定，对其他领域的严重违法失信行为有关责任人，也将被限制乘坐动车组列车席位（有效期为一年，一年期满自动移除；在有效期内其法定义务履行完毕的，有关部门应当在7个工作日内通知铁路总公司移除名单），包括G字头动车组列车全部座位以及其他动车组列车一等座以上座位，适用旅客包括：
# 有履行能力但拒不履行的重大税收违法案件当事人；
# 在财政性资金管理使用领域中存在弄虚作假、虚报冒领、骗取套取、截留挪用、拖欠国际金融组织和外国政府到期债务的严重失信行为责任人；
# 在社会保险领域中存在以下情形的严重失信行为责任人：用人单位未按相关规定参加社会保险且拒不整改的；用人单位未如实申报社会保险缴费基数且拒不整改的；应缴纳社会保险费且具备缴纳能力但拒不缴纳的；隐匿、转移、侵占、挪用社会保险基金或者违规投资运营的；以欺诈、伪造证明材料或者其他手段骗取社会保险待遇的；社会保险服务机构违反服务协议或相关规定的；拒绝协助社会保险行政部门对事故和问题进行调查核实的；
# 证券、期货违法被处以罚没款，逾期未缴纳的；上市公司相关责任主体逾期不履行公开承诺的；
# 被人民法院按照有关规定依法采取限制消费措施，或依法纳入失信被执行名单的；
# 相关部门认定的其他限制乘坐火车高级别席位的严重失信行为责任人。

以上规定自2018年5月1日起施行<ref>{{cite news|title=多部门联合发文：适当限制特定严重失信人乘坐民航、火车|url=http://www.jiemian.com/article/1995081.html|publisher=界面新闻|date=2018-03-16 }}</ref>。

===票价===
据西班牙媒体统计，中国高铁的票价与西班牙、日本、法国、德国和意大利等发达国家相比，为全世界最低，每公里的票价为0.04欧元，远低于西班牙的0.19欧元，以及法国的0.22欧元、德国的0.27欧元、意大利的0.25欧元和日本的0.22欧元<ref name=最低>{{cite news|title=中国高铁票价全世界最低 优势被收入水平抵消|url=http://finance.people.com.cn/n/2013/0115/c1004-20210543.html|publisher=齐鲁晚报   |date=2013-01-15  }}</ref>。但由于中国的个人收入差距很大，许多人的收入远低于平均值（2011年中国私营单位年均工资仅24556元，而全国城镇人均可支配收入更只有21810元），因此高铁票价对于一般大众而言相对要贵<ref name=最低/><ref>{{cite news|title=京广高铁865元票价贵吗|url=http://view.news.qq.com/zt2012/jg/index.htm|publisher=腾讯|date= }}</ref>。

2018年4月28日起，铁路运输企业将依据价格法律法规，对部分线路高等级席别票价进行调整，部分线路动车一等座票价上涨约30%<ref>{{cite web|author1=搜狐|title=京津城际一等、特等座将涨价 一等座票价涨至88元 2018-04-04 22:54|url=http://www.sohu.com/a/227324262_362254}}</ref>。2018年7月调图后，部分线路动车二等座票价也将实行优化调整，如[[贵广客运专线|贵广客运专线]]全程二等座票价除部分车次打折外将全面上涨，由267.5元涨至323元。

===航空竞争===
[[File:201610_Air–Rail_transferring_notice_at_SHQ.jpg|thumb]]到达厅往[[虹桥机场|虹桥机场]]的通道口处设有[[中国东方航空公司|东航]]、[[上海航空公司|上航]]与[[上海铁路局|上海铁路局]]推出的空铁联运产品的取票提示]]
由于近年来中国已有多条高铁建成，因此航空业危机已然显现。据武汉天河机场2011年提供的统计数据显示，600公里以下航程的航班已经有六七成停飞；当年春运期间，天河机场发送旅客量同比下降8.52%，为23年来首次下滑，而武广高铁旅客的平均上座率比2010年同期增长60%以上<ref name=nfzh2011>{{cite web|url=http://www.infzm.com/content/57266 |title=高铁分流民航客源：多条短程航班停飞 |publisher=Infzm.com |date= |accessdate=2018-12-11}}</ref>。截至2013年10月，中国高速铁路每月载客量是航空的两倍<ref name="Speedy Trains Transform China"/>。

为了应对高铁挑战，航空公司和机场已经开始转变战略，一些航空公司和机场也开始筹划“空铁联运”，把高铁车次、时间和飞机航线进行搭配，旅客只需要买一张套票就可以实现飞机和火车的换乘<ref name=nfzh2011/>。此外，自2016年11月1日起，[[中国民用航空局|中国民用航空局]]和[[国家发展改革委|国家发展改革委]]开始实施《关于深化民航国内航空旅客运输票价改革有关问题的通知》，规定800公里以下航线、800公里以上与高铁动车组列车形成竞争航线的旅客运输票价交由航空公司依法自主制定<ref>{{cite news|title=800公里以下航线票价将放开 “白菜价”机票会更多|url=http://news.sohu.com/20161025/n471213138.shtml|publisher=现代金报|date=2016-10-25 }}</ref>。

===运行速度===
2011年7月1日后中国所有高铁进行第一次降速，运营时速从350公里降低为300公里，原设计时速250公里则只以200公里运行。由于此次大降速和后面的降速都是在[[盛光祖|盛光祖]]任期内执行，时任铁道部部长的盛光祖被国内车迷戏称为“盛高阻”（取高铁阻碍者之意）。之后在23天发生[[2011年甬台温铁路列车追尾事故|7.23追尾事故]]后进行第二次降速。2015年两会期间[[政协委员|政协委员]]、中铁建（中铁建设集团有限公司）总裁赵广发認為现在不少高铁以时速200公里的速度运营是巨大浪费，建议高铁恢复350公里时速<ref>{{cite news|url=http://news.ifeng.com/a/20150307/43290063_0.shtml#_zbs_2015lh|title=政协委员：高铁应恢复350公里时速 降速是浪费|publisher=凤凰网}}</ref>。2016年6月，在国家“十二五”科技成就展上，[[中国共产党中央委员会总书记|中国共产党中央委员会总书记]][[习近平|习近平]]与[[北京交通大学|北京交通大学]][[贾利民|贾利民]]教授就中国高铁是否有条件恢复350公里时速进行了交流<ref>{{cite news|url=http://news.china.com/domestic/945/20160606/22815477_all.html|title=习近平之问：高铁能否全面恢复350公里时速？2017年4月1日，中國所有高鐵恢復速度，運營時速300公里恢復350公里，原設計時速250公里恢復250公里|publisher=中华网|deadurl=yes|archiveurl=https://archive.is/20160606061242/http://news.china.com/domestic/945/20160606/22815477_all.html|archivedate=2016年6月6日}}</ref>。2017年9月21日，中国铁路实行新的运行图，在京沪高铁开行数对由[[复兴号电力动车组|复兴号电力动车组]]担当、最高运营时速为350公里的高速列车<ref>{{cite news|url=http://news.xinhuanet.com/local/2017-08/21/c_1121513479.htm|title=京沪高铁9月起时速提至350公里 两地4.5小时可到达|publisher=新华网}}</ref>。2018年8月8日，京津城际铁路也实现达速，恢复2011年降速前的350km/h的时速运行。<ref>{{cite news|url=http://www.xinhuanet.com/politics/2018-08/08/c_129928935.htm|title=京津城际复兴号动车组实现时速350公里运行|publisher=新华网}}</ref>

自2017年1月5日起延长运行的[[京昆高速动车组列车|京昆高速动车组列车]]（G403/4次、G405/6次，运行里程2760公里，运行时间约12-13小时），是至今为止世界上运行长度最长的动力分散式动车组运行车次<ref name=bjqnb>{{cite news|url=http://epaper.ynet.com/html/2017-01/06/content_234729.htm?div=0|title=高铁首次直通昆明 13小时抵达|publisher=北京青年报|date=2017-01-06|accessdate=2017-01-06|deadurl=yes|archiveurl=https://web.archive.org/web/20170107171614/http://epaper.ynet.com/html/2017-01/06/content_234729.htm?div=0|archivedate=2017-01-07}}</ref>，超越了[[G529/530次列车|G529/530次列车]]（{{站|北京西}}↔[[北海站_(广西)|北海]]）2697公里的纪录<ref>{{cite web|url=http://news.sina.com.cn/c/2016-06-15/doc-ifxszfak3901559.shtml|title=北京西至北海将开通全国运行里程最长动车|publisher=中国青年报|date=2016-06-15|accessdate=2016-07-01}}</ref>。

===车次运行時間===
成都東至福州动车组列车（D2241/2244次，运行里程2076公里，运行时间16小時2分），是至今为止國內运行時間最长的动力分散式动车组运行车次。
