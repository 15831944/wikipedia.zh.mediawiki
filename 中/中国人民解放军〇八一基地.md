'''中国人民解放军〇八一基地'''，位于[[湖北省|湖北省]][[枝江市|枝江市]]，隶属[[中国人民解放军陆军|中国人民解放军陆军]]。

==简介==
该基地设在[[湖北省|湖北省]][[宜昌市|宜昌市]]东部的[[枝江市|枝江市]]<ref name=sansha>{{Cite web|url = http://www.hbrf.gov.cn/info/wcm/61220.htm|title = “三傻”主任——记湖北省枝江市人防办原主任杨东海|author = |date = 2014-11-15|publisher = 湖北省人民防空办公室（省民防局）|language = |accessdate = |quote = }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。前身是'''中国人民解放军总参谋部兵种部〇八一基地'''，隶属[[中国人民解放军总参谋部|中国人民解放军总参谋部]]兵种部，下设防化兵部、行政管理科等部门<ref>{{Cite web |url = http://www.jingmen.cn/govinfo/szbmdw_4799/sczj/201412/t20141208_124796.shtml |title =领导之窗 |author =  |date =  2014-10-22|publisher =荆门市人民政府  |language =  |accessdate =  |quote =  }}</ref>。2003年12月中国人民解放军总参谋部兵种部撤销，并入新组建的[[中国人民解放军总参谋部军训和兵种部|中国人民解放军总参谋部军训和兵种部]]；2011年12月21日，中国人民解放军总参谋部军训和兵种部改编成立[[中国人民解放军总参谋部军训部|中国人民解放军总参谋部军训部]]<ref>{{Cite web |url = http://mil.news.sina.com.cn/2011-12-21/1507678194.html |title = 解放军总参谋部军训和兵种部改编为总参军训部 |author =  |date = 2011-12-21 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。该基地随之改称'''中国人民解放军总参谋部〇八一基地'''<ref name=sansha/>，下设总参谋部〇八一基地医院等部门及单位<ref>{{Cite web |url =http://www.bidcenter.com.cn/newscontent-18486203-4.html  |title =规划档案馆监理预中标结果  |author =  |date = 2014-12-05 |publisher =中国采招网  |language =  |accessdate =  |quote =  }}</ref>。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，'''中国人民解放军〇八一基地'''转隶[[中国人民解放军陆军|中国人民解放军陆军]]<ref>{{Cite web |url = http://politics.cntv.cn/2017/06/02/VIDEyRCligBhuSRp2bkr1N6A170602.shtml |title = 蒋超良就贯彻落实国务院中央军委电视电话会精神强调 加快推进驻鄂部队全面停止有偿服务工作 |author =  |date = 2017-06-02 |publisher = 央视网 |language =  |accessdate =  |quote =  }}</ref>。
