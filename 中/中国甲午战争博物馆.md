'''中国甲午战争博物馆'''，位于[[中華人民共和國|中國]][[山东省|山东省]][[威海市|威海市]]，是为纪念[[甲午战争|甲午战争]]而设立的博物馆。

==历史==
1984年，国家决定对外开放14个沿海城市。1985年4月1日，威海港和[[刘公岛|刘公岛]]正式对外开放。[[中华人民共和国国务院|国务院]]和[[中央军事委员会|中央军委]]决定，[[北洋海军提督署|北洋海军提督署]]及其附属建筑由驻军分批移交地方管理<ref name=dshj>{{Cite web |url = http://www.jiawuzhanzheng.org/menu/about-us/dashiji/ |title = 大事记 |author =  |date =  |publisher = 中国甲午战争博物馆 |language =  |accessdate =2018-03-01  |quote =  }}</ref>。

1985年3月21日，威海市人民政府批准成立'''威海市北洋海军提督署文物管理所'''，为全民文化[[事业单位|事业单位]]，编制4人，隶属威海市（现[[环翠区|环翠区]]）文化局。1987年10月，国务院批准地级[[威海市|威海市]]成立。同年12月9日，威海市北洋海军提督署文物管理所上划为市直文化单位，隶属威海市教育文化体育委员会。同时威海市编制委员会决定，威海市北洋海军提督署文物管理所同时挂“威海市文物管理所”的牌子，编制20人，在履行原职责同时，还对威海市所属的[[荣成县|荣成县]]、[[文登县|文登县]]、[[乳山县|乳山县]]、环翠区的文物工作进行宏观管理和业务指导<ref name=dshj/>。
[[File:中国甲午战争博物馆远景20120728.JPG|thumb]]
1992年3月1日，该所划归威海市文化委员会。同年4月25日，经国家文物主管部门批准，威海市北洋海军提督署文物管理所更名'''中国甲午战争博物馆'''。2003年6月30日，中共威海市委、威海市人民政府决定将该馆划归刘公岛管理委员会领导，仍是市直文化事业单位。2008年该馆被[[国家文物局|国家文物局]]公布为首批[[国家一级博物馆|国家一级博物馆]]。2009年9月经威海市机构编制委员会批准，更名'''中国甲午战争博物院'''。但该院对外仍在同时使用中国甲午战争博物馆的名称<ref name=dshj/>。

1985年，设办公室、陈列业务部、宣传教育部。1992年，增设安全保卫部、财务经营部、旅游服务社、文物科技保管部。1995年成立图书资料研究中心。2004年增设水师学堂管理部，对水师学堂范围内的文物遗址进行管理保护并对外开放<ref name=yuxm/>。

中国甲午战争博物馆陈列馆在[[中共中央政治局常委|中共中央政治局常委]][[李长春|李长春]]的关怀下，于2005年3月启动建设，2008年正式对外开放<ref>{{Cite web |url = http://www.cssn.cn/zt/zt_xkzt/zt_lsxzt/lsxzt_jwj/jw_jsp/jstp/201403/t20140306_1020662.shtml |title =中国甲午战争博物馆陈列馆简介  |author =  |date =2014-03-06  |publisher = 中国社会科学网 |language =  |accessdate =  |quote =  }}</ref>。

中国甲午战争博物馆被[[共青团中央|共青团中央]]公布为“全国青少年教育基地”，被[[中华人民共和国教育部|教育部]]、[[中华人民共和国民政部|民政部]]、[[中华人民共和国文化部|文化部]]、[[国家文物局|国家文物局]]、共青团中央、[[中国人民解放军总政治部|解放军总政治部]]联合公布为全国中小学爱国主义教育基地，是[[中共中央宣传部|中共中央宣传部]]公布的全国一百个爱国主义教育示范基地之一，国家文物局公布的全国十大优秀社会教育基地之一。1994年7月6日，[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]][[江泽民|江泽民]]题写馆名“中国甲午战争博物馆”。[[李鹏|李鹏]]、[[李岚清|李岚清]]、[[乔石|乔石]]、[[刘华清|刘华清]]、[[田纪云|田纪云]]等[[党和国家领导人|党和国家领导人]]也曾先后为该馆题词<ref name=wy>{{Cite web |url =http://news.163.com/40922/1/10RM44UI0001127S.html  |title = 观海祭英魂，参观中国甲午战争博物馆 |author =  |date = 2004-09-22 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

==展览==
[[File:Liugongisland_warmemorialhall.jpg|thumb]]
中国甲午战争博物馆是纪念遗址性博物馆，以[[北洋海军|北洋海军]]及[[甲午战争|甲午战争]]为主题。其保护管理的“[[刘公岛甲午战争纪念地|刘公岛甲午战争纪念地]]”包括北洋海军提督署、龙王庙、丁汝昌寓所、水师学堂、铁码头、黄岛炮台、旗顶山炮台、东泓炮台、日岛炮台等28处遗址，均为[[全国重点文物保护单位|全国重点文物保护单位]]。开放参观的有北洋海军提督署、龙王庙、丁汝昌纪念馆、黄岛炮台与兵器馆、东泓炮台、旗顶山炮台等等。馆藏历史照片1000余幅，文物资料200余件，打捞舰船文物标本300余件，其中包括[[济远舰|济远舰]]双主炮<ref name=wy/>。

该馆利用北洋海军提督署等清代建筑作为展览场所，以图片、文物、蜡像、模型、沙盘、影视等形式，展现了北洋海军及甲午战争的历史。展览主要包括北洋海军陈列、礼仪厅复原陈列、[[威海卫之战|威海卫之战]]陈列、北洋海军将领会议蜡像厅、[[丁汝昌|丁汝昌]]殉难处、复制[[蒂森克虏伯|克虏伯]]大炮、济远舰打捞文物展、丁汝昌生平事迹展、北洋海军将士名录墙、北洋海军将士事迹展、甲午战争史研究成果展等<ref name=wy/>。

'''刘公岛甲午战争纪念地'''，位于[[中华人民共和国|中国]][[山东省|山东省]][[威海市|威海市]]，由中国甲午战争博物馆管理。1988年1月13日，被[[中华人民共和国国务院|中华人民共和国国务院]]公布为[[第三批全国重点文物保护单位|第三批全国重点文物保护单位]]<ref>{{Citeweb |title=国务院关于公布第三批全国重点文物保护单位的通知 |url=http://www.sach.gov.cn/col/col1646/index.html |publisher=国家文物局 |date=1988-01-13 |access-date=2018-03-01 |archive-url=https://web.archive.org/web/20141008131520/http://www.sach.gov.cn/col/col1646/index.html |archive-date=2014-10-08 |dead-url=yes }}</ref>。2000年6月，刘公岛甲午战争纪念地被列为山东省历史优秀建筑。2000年9月，[[山东省人民政府|山东省人民政府]]公布了刘公岛甲午战争纪念地的保护范围和保护内容（其中北洋海军提督署、威海水师学堂内包含部分稍晚增建的英式建筑），包括刘公岛、威海湾南北两岸及日岛的[[北洋海军|北洋海军]]和[[甲午战争|甲午战争]]纪念建筑物及遗址。2000年10月26日，[[山东省人民代表大会常务委员会|山东省第九届人民代表大会常务委员会]]第十七次会议通过并公布施行了《山东省刘公岛甲午战争纪念地保护管理规定》<ref>{{Citeweb|title=刘公岛甲午战争纪念地|url=http://whwh.gov.cn/art/2012/9/21/art_13687_424661.html|publisher=威海市文化广电新闻出版局|date=2012-09-21|deadurl=yes|archiveurl=https://web.archive.org/web/20160304233309/http://whwh.gov.cn/art/2012/9/21/art_13687_424661.html|archivedate=2016-03-04}}</ref><ref>{{Citeweb|title=山东省刘公岛甲午战争纪念地保护管理规定|url=http://www.sdrd.gov.cn/communiqueComm+9+17/19700101/4282.html|publisher=山东人大信息网|date=2000-10-26|deadurl=yes|archiveurl=https://web.archive.org/web/20130425014549/http://www.sdrd.gov.cn/communiqueComm+9+17/19700101/4282.html|archivedate=2013-04-25}}</ref><ref>{{Cite web |url = http://www.whwh.gov.cn/art/2017/7/7/art_13687_945916.html |title =威海市各级文物保护单位  |author =  |date =  2017-07-07|publisher =威海市文化广电新闻出版局   |language =  |accessdate =  |quote =  }}</ref>。

;威海湾北岸范围
*祭祀台炮台：海岸炮台
*老母顶炮台：永久性陆地炮台
*合庆滩炮台（柏顶炮台）：永久性陆地炮台
*九峰顶炮台：永久性炮台。位于西钦村以东。西南约两公里是丰柴顶炮台。
*遥了墩炮台：临时炮台。位于今环翠区委党校后山，金海滩东北方。1894年12月3日后开工抢修。绥军利用明代的遥了墩烟墩设置。
*远遥墩炮台：临时炮台。位于今[[哈工大|哈工大]]（威海）西北。1894年12月3日后开工抢修。
*丰柴顶炮台：临时炮台。位于今高技区柴峰小区北侧的丰柴顶。1894年12月3日后开工抢修。
*东里夼炮台：临时炮台。位于今环翠区万福山庄小区附近。1894年12月3日后开工抢修。
*棉花山炮台：临时炮台。1894年12月3日后开工抢修。
*佛顶山炮台：临时炮台。1894年12月3日后开工抢修。
