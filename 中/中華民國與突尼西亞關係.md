'''中華民國與突尼西亞關係'''是指[[中華民國|中華民國]]與[[突尼西亞共和國|突尼西亞共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對突尼西亞的相關事務由'''[[駐法國台北代表處|駐法國台北代表處]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-128-a5910-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|突尼西亞簽證政策|突尼西亞公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可前往突尼西亞駐[[日本|日本]][[外交代表機構#大使馆|大使館]]申請簽證。經突尼西亞轉機亦需申請[[過境簽證|過境簽證]]。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-128-a5910-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref>

持[[突尼西亞護照|突尼西亞護照]]的{{link-en|突尼西亞人口|Demographics of Tunisia|突尼西亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將突尼西亞歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2017年4月22日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-183-f966a-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|{{link-en|突尼西亞經濟|Economy of Tunisia}}}}
===貿易===
2017年，突尼西亞是中華民國第100大貿易夥伴、第110大[[進口|進口]]夥伴、第90大[[出口贸易|出口]]夥伴。出口至突尼西亞的金額為4,368萬8,354[[美元|美元]]，年減3.272%；自突尼西亞進口的金額為1,356萬0,100美元，年增5.195%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）3,012萬8,254美元，年減6.654%。※2016年，突尼西亞是中華民國第98大貿易夥伴、第105大進口夥伴、第87大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
截至2015年，尚未有[[台灣商人|台商]]在突尼西亞設立[[公司|公司]]。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 投資環境分析 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>
