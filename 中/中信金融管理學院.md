'''中信金融管理學院'''，簡稱'''中金院'''，是一所位於[[臺南市|臺南市]][[安南區|安南區]]的私立[[大專院校|大專院校]]，前身為成立於2000年的'''興國管理學院'''，2015年因獲[[中信金控|中信金控]]捐資而重組更名<ref name=zxjrglxy>{{Cite news| author=黃宣翰| title=興國改名中信學院 高階經理人來上課| url=http://udn.com/news/story/6886/869399| newspaper=[[聯合報|聯合報]]| date=2015-04-29}}</ref>。

== 歷史沿革 ==
前身'''興國管理學院'''是2000年由曾創辦[[臺南市私立興國高級中學|臺南市私立興國高級中學]]的[[趙景霖|趙景霖]]向台糖租地創立；成立之初設有資訊管理學系、企業管理學系、應用日語學系、財務金融學系、財稅學系、會計學系六個學系，後陸續增至21個學系。

但由於後續學校財務困難，在[[施光訓|施光訓]]擔任校長後，於2015年2月[[中國信託金融控股|中信金控]]計畫透過子公司[[台灣彩券|台灣彩券]]捐資並接手經營權，但也在同時興國管理學院因師資不足，遭教育部以[[行政命令|行政命令]]勒令停止招生一年<ref>{{cite news|title =興國遭勒令停招 百名學生將轉學| url = http://newtalk.tw/news/view/2015-02-10/56812 | work = 新頭殼 | author =  柯昱安 | language = zh-tw | date = 2015-02-10}}</ref><ref>{{cite news|title =「買下」興國管理學院？中信金：是「捐贈」| url = http://www.ettoday.net/news/20150211/466638.htm | work = 東森新聞雲 | author = 蔡怡杼 | language = zh-tw | date = 2015-02-11}}</ref>；之後學校提出改善計畫，優退原校全部師資，同時申請更名為「'''中信金融管理學院'''」，獲得教育部同意並恢復招生<ref>{{cite news|title = 教部同意興國管理學院　更名「中信金融管理學院」| url = http://www.nownews.com/n/2015/05/05/1676581 | work = 今日新聞 | author = 葉立斌 | language = zh-tw | date = 2015-05-05}}</ref>。

重組後的中信金融管理學院，縮小規模僅設立財務金融學系、財經法律學系、企業管理學系三個學系，並採單獨招生，每系招生50人，在2015年全校只招收150人<ref>{{cite news|url=http://www.chinatimes.com/newspapers/20141203000376-260102| author=新聞稿| title=教育部原則同意康寧大學與康寧醫護暨管理專科學校兩校合併及興國管理學院申請變更學校名稱為「中信金融管理學院」| newspaper=教育部高等教育司| date=2015-04-24| accessdate=2015-05-10}}</ref><ref name=zxjrglxy/><ref>{{cite web|title=中信學校財團法人中信金融管理學院改名|url=http://www.kh.edu.tw/publicInfo/eduReport/20150326|website=高雄市政府教育局全球資訊網|publisher=中信金融管理學院|accessdate=2015-10-01|date=2015-06-29}}</ref>。2016年起陸續增設金融管理學院學士班、人工智慧學系、金融管理研究所碩士班及法律研究所碩士班。

重組後，學校於2016年起先後獲得[[Cheers快樂工作人雜誌|Cheers快樂工作人雜誌]]評選為「國內Top 20辦學卓越的大學」<ref>{{Cite news| url=http://www.appledaily.com.tw/realtimenews/article/new/20160217/797787/| title=中信接手興國　搖身變辦學卓越大學| newspaper=[[蘋果日報_(台灣)|蘋果日報]]| date=2016-02-17| author=劉榮輝}}</ref>及「最佳大學」。

2017年[[中信集團|中信集團]]利用學校空間設立了與高級中學教育同等學力的的[[在家自學|在家自學]]教育機構－[[中信國際實驗教育機構|中信國際實驗教育機構]]。

== 保障就業 ==
該校與中信金控進行產學合作，在校成績優良、考取9張金融證照、及多益750分可媒合至中信金控上班。
2019年4月，公布保障就業職缺，分別為櫃檯服務專員、電話客服人員、法金服務人員、會計人員、保單服務人員、核保理賠人員、客服人員、商品精算人員、營業員。

== 社團活動 ==
* 證券研究社：簡稱證研社，宗旨是讓所有人都能瞭解證券市場的運作原理與操作方式，並且不定期開設社課與討論活動。
* 管弦樂團：由在音樂上有一定素養的同學組成，並開放初階同學參與學習，擔任各項學校及校外的重要演出工作。
* 勤學社：設置於校內勤學大樓5樓，宗旨為「自治互助，扶持成長」，負責宿舍之自主管理。
* 網球社：為全校主要體育性社團。
* 棒球社：學院除了招收甲組棒球隊球員之外，對棒球運動有興趣的一般生也組成棒球社，並積極參加各項棒球賽事。
