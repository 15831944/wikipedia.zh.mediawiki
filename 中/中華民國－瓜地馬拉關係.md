'''中華民國－瓜地馬拉關係'''是指[[中華民國|中華民國]]與[[瓜地馬拉|瓜地馬拉]]之間的外交關係。

==歷史==
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[瓜地馬拉|瓜地馬拉]]。]]
*1933年6月15日：[[中華民國|中華民國]]與[[瓜地馬拉|瓜地馬拉]]建立領事級外交關係。<ref name="ROC Foreign Relation Yearbook 2016-3">{{cite book|title=中華民國外交部105年外交年鑑-第三章對外關係|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-11 |accessdate=2018-03-01}}</ref>

*1935年：中華民國政府在[[瓜地馬拉城|瓜地馬拉城]]設置[[總領事館|總領事館]]。<ref name="ROC Foreign Relation Yearbook 1997-3">{{cite book|title=中華民國外交部86年外交年鑑-第三章中外關係|publisher=[[中華民國外交部|中華民國外交部]]|date = 1998-06 |accessdate=2017-12-16}}</ref>

*1954年12月22日：中華民國政府將總領事館升格為[[公使館|公使館]]<ref name="ROC Foreign Relation Yearbook 1997-3">{{cite book|title=中華民國外交部86年外交年鑑-第三章中外關係|publisher=[[中華民國外交部|中華民國外交部]]|date = 1998-06 |accessdate=2017-12-16}}</ref>，瓜地馬拉政府在[[台北|台北]]設立駐華公使館。<ref name="ROC Foreign Relation Yearbook 2016-3">{{cite book|title=中華民國外交部105年外交年鑑-第三章對外關係|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-11 |accessdate=2018-03-01}}</ref>

*1960年9月1日：瓜地馬拉政府將公使館升格為[[大使館|大使館]]。<ref name="ROC Foreign Relation Yearbook 2016-3">{{cite book|title=中華民國外交部105年外交年鑑-第三章對外關係|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-11 |accessdate=2018-03-01}}</ref>

*1973年2月14日：中華民國政府向瓜地馬拉派駐技術團。<ref name="ROC Foreign Relation Yearbook 2016-3">{{cite book|title=中華民國外交部105年外交年鑑-第三章對外關係|publisher=[[中華民國外交部|中華民國外交部]]|date = 2017-11 |accessdate=2018-03-01}}</ref>

==兩國貿易==
2018年瓜地馬拉是中華民國第72名貿易伙伴，兩國貿易額185,433,979美元，其中中華民國對瓜地馬拉出口129,329,142美元，是第64大出口國，進口56,104,837美元，是第81大進口國，與2017年貿易額相比貿易總額約減少19.40%、出口約增加9.11%、進口約減少49.69%<ref>{{cite book |url = http://cus93.trade.gov.tw/FSCI/ |title = 經濟部國際貿易局貿易統計系統 |publisher = [[經濟部國際貿易局|經濟部國際貿易局]] |date = 2019-02-23 |accessdate = 2019-02-27}}</ref>。

*中華民國－瓜地馬拉主要出口項目：混合烷基苯、汽車零配件、聚縮醛、機車零配件等。<ref name=bft>{{Cite web |url = https://www.trade.gov.tw/World/ListArea.aspx?code=7020&nodeID=977&areaID=1&country=b6455Oc5Zyw6aas5ouJ&pw=3|title = 各國經貿資訊 |publisher =[[經濟部國際貿易局|經濟部國際貿易局]]|date = 2017-10-19|accessdate = 2018-03-10}}</ref>
*中華民國－瓜地馬拉主要進口項目：蔗糖、咖啡、回收銅、回收紙板等。<ref name=bft>{{Cite web |url = https://www.trade.gov.tw/World/ListArea.aspx?code=7020&nodeID=977&areaID=1&country=b6455Oc5Zyw6aas5ouJ&pw=3|title = 各國經貿資訊 |publisher =[[經濟部國際貿易局|經濟部國際貿易局]]|date = 2017-10-19|accessdate = 2018-03-10}}</ref>

中美洲薩爾瓦多、瓜地馬拉、尼加拉瓜及宏都拉斯四國簽署之單一簽證協定自2007年10月25日生效，即由該四國任何一國所核發之簽證將適用於自由進出該四國。<ref name="簽證及入境須知">{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-80-8d3d1-02-1.html|title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]|accessdate = 2018-01-10}}</ref><ref>{{Cite web|url=https://www.boca.gov.tw/cp-37-220-9f130-1.html|title=中華民國國民可以免簽證、落地簽證及電子簽證前往之國家與地區|publisher=[[外交部領事事務局|外交部領事事務局]]|accessdate=2018-01-09}}</ref>

=== 航空客運 ===
兩國目前無直航班機，來往瓜地馬拉之客運航線有以下方式：

1.搭乘飛往{{flagicon|USA}}[[洛杉磯國際機場|洛杉磯]]、{{flagicon|USA}}[[喬治布希洲際機場|休士頓-喬治·布希]]班機。

2.在各機場轉機即可到達{{flagicon|GUA}}[[瓜地馬拉城|瓜地馬拉城]]-[[拉奧羅拉國際機場|拉奧羅拉國際機場]]。

＊上述目的地國際機場為鄰近該國首都或該國最主要樞紐機場，若要前往該國其他國際機場或國內線機場詳見{{link-en|瓜地馬拉機場列表|List of airports in Guatemala|瓜地馬拉機場}}。

＊航線會因航空公司營運狀況更改或取消，出發前應與負責該航線的航空公司確認航線及航班，以免影響權益。
