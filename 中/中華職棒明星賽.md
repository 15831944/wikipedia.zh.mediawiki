'''中華職棒明星賽'''（''CPBL All-Star Game''，又稱為'''中華職棒紅白明星對抗賽'''）於1990年開始舉行。原意為仿效美日職棒於球季中擇定一週末舉行賽事，選手陣容則由球迷票選心目中的明星球員，組成兩支明星隊伍分庭抗禮，讓棒球選手能夠盡情表現與球迷同樂。1992年起加入全壘打大賽，1993年起則將對抗賽改為一場並開始頒發「單場最有價值球員」，1998年則納入跑壘大賽。

== 紅白明星對抗賽 ==
明星賽分為對抗賽、全壘打大賽及跑壘大賽，[[中華職棒全壘打大賽|全壘打大賽]]則是於1992年開始舉行，而跑壘大賽則於1998年納入明星賽中。
在賽事表現上，自2002年起，只要比賽隊伍陣容有[[興農牛|興農牛]]，該隊就不曾贏球，而被稱為中華職棒明星賽的「興農魔咒」<ref>{{cite web |url=http://www.espnstar.com.tw/news/baseball/2011/0726/210261.htm |title=興農魔咒十年未破 台銀紅抱走勝利 |author=cpbl |work=ESPN STAR.com |language=zh-tw |deadurl=yes |archiveurl=https://archive.is/20130418213934/http://www.espnstar.com.tw/news/baseball/2011/0726/210261.htm |archivedate=2013-04-18 }}</ref>。不過，2012年的明星賽興農與統一的白隊以一分之差氣走了紅隊，中止長達十年的興農魔咒<ref>{{cite web |url=http://www.espnstar.com.tw/news/baseball/2012/0722/242924.htm |title=CPBL明星賽:基哥打破興農魔咒 白隊力退紅隊 |author=TSNA |work=ESPN STAR.com |language=zh-tw |deadurl=yes |archiveurl=https://web.archive.org/web/20120804055019/http://www.espnstar.com.tw/news/baseball/2012/0722/242924.htm |archivedate=2012-08-04 }}</ref>。同年，紅白兩隊也上演首場的球員交易戲碼，原屬於紅隊的[[林智勝|林智勝]]與原屬於白隊的[[林益全|林益全]]兩者互換隊伍，帶給球迷驚喜<ref>{{cite web |url=http://www.espnstar.com.tw/news/baseball/2012/0722/242925.htm |title=CPBL明星賽:雙林互換 林益全砲轟隊友林彥峰 |author=TSNA |work=ESPN STAR.com |language=zh-tw |deadurl=yes |archiveurl=https://web.archive.org/web/20120804055024/http://www.espnstar.com.tw/news/baseball/2012/0722/242925.htm |archivedate=2012-08-04 }}</ref>。另外，該年的全壘打大賽由紅隊[[鍾承佑|鍾承佑]]獲得冠軍，成為中華職棒史上首位完成全壘打大賽二連霸的球員<ref>{{cite web |url=http://www.espnstar.com.tw/news/baseball/2012/0722/242842.htm |title=全壘打大賽 鍾承佑史無前例二連霸！峰 |author=CPBL |work=ESPN STAR.com |language=zh-tw |deadurl=yes |archiveurl=https://web.archive.org/web/20120727043937/http://espnstar.com.tw/news/baseball/2012/0722/242842.htm |archivedate=2012-07-27 }}</ref>。

=== 賽制 ===
;1990年（職棒元年）
*以「上半季排名」分組
:一、四名組成明星紅隊，二、三名則組成明星白隊
;1991年～1998年、2004年、2016年（職棒二年～職棒九年、職棒十五年、職棒二十七年）
*以「前一年度成績」分組
:四球團時期：一、四名組成明星紅隊，二、三名則組成明星白隊
:六球團時期：一、三、五名組成明星紅隊，二、四、六名則組成明星白隊
;2001年（職棒十二年）
*以「球齡」作為分野，組成老將新秀世紀對抗賽
;1999、2000、2002、2003年、2005年～2012年（職棒十、十一、十三、十四年、十六年～二十三年）
*採用「南北區域對抗」形式分組
:四球團時期：屬地在最北的兩支球團組成明星紅隊，剩下的兩支球團組成明星白隊
:六球團時期：屬地在最北的三支球團組成明星紅隊，剩下的三支球團組成明星白隊
;2013~2014年（職棒二十四～二十五年）
*採用「明星賽特別選秀會」
:總教練：以抽籤決定教練團組成方式
:選手：各守備位置人氣統計前兩名(外野手前六名)之選手參加選秀
;2015年（職棒二十六年）
*首創以「守備位置平均體重」區分「力量」、「速度」兩組
:總教練：以抽籤決定教練團組成方式

=== 競賽規則 ===
*先發球員由各隊總教練於明星賽特別選秀會中選出，其餘人選由各隊教練團評選推薦。 
*單場比賽局數以九局為限，比數相同不進行延長賽。 
*每任投手出賽最多只能投三局。
