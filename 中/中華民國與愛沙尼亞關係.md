'''中華民國與愛沙尼亞關係'''是指[[中華民國|中華民國]]與[[愛沙尼亞共和國|愛沙尼亞共和國]]之間的關係。兩國於1937－1940年有官方[[外交#外交上的承認|外交關係]]，但因[[蘇聯佔領波羅的海國家|蘇聯併吞愛沙尼亞]]而中止。自1991年[[蘇聯解體|從蘇聯獨立]]以來，目前沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對愛沙尼亞的相關事務由'''駐[[拉脫維亞|拉脫維亞]][[臺北經濟文化代表處|臺北代表團]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-227-cf8ce-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref><ref>{{Cite web |url = http://www.roc-taiwan.org/lv/post/20.html |title = 代表處簡介 |publisher = 中華民國駐外單位聯合網站}}</ref>

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，[[愛沙尼亞|愛沙尼亞]]當時屬於[[蘇聯|蘇聯]]的[[加盟共和國|加盟共和國]]（[[愛沙尼亞蘇維埃社會主義共和國|愛沙尼亞蘇維埃社會主義共和國]]），由蘇聯作為[[主權國家|主權國家]]參與投票。]]
{{see also|中華民國外交|{{link-en|愛沙尼亞外交|Foreign relations of Estonia}}}}
1937年12月21日，愛沙尼亞與中華民國建立[[特命全權公使|公使級]]的[[外交#外交上的承認|外交關係]]。

1940年8月6日，因[[蘇聯佔領波羅的海國家|蘇聯吞併愛沙尼亞]]而中止。

1991年11月8日，中華民國外交部次長[[章孝嚴|章孝嚴]]訪問愛沙尼亞，與對外經濟關係部長{{link-et|皮爾夫|Mehis Pilv}}簽署互設[[實質大使館|貿易代表團]]協定及經濟合作[[備忘錄|備忘錄]]。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-4.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1995年6月，中華民國經濟部長[[江丙坤|江丙坤]]率團赴[[瑞士|瑞士]]參加蒙太納[[論壇|論壇]]，期間會見愛沙尼亞總統[[伦纳特·梅里|梅里]]，並就雙邊經貿關係交換意見。8月，愛沙尼亞前總理[[马尔特·拉尔|拉爾]]國會議員抵台訪問。<ref name="86年外交年鑑"/>

2012年3月28日，[[愛沙尼亞國會|愛沙尼亞國會]]議員及{{link-en|愛沙尼亞政府|Government of Estonia}}官員訪問團應[[中華民國政府|中華民國政府]]邀請抵台訪問。訪問團成員包括{{link-en|愛沙尼亞國會議員列表|List of members of the Parliament of Estonia|國會議員}}{{link-en|翰森|Margus Hanson}}、{{link-en|愛沙尼亞外交部|Minister of Foreign Affairs (Estonia)}}經濟發展司總司長{{link-et|彭安德|Andre Pung}}，亞太司負責[[亞洲國家和地區列表|亞]]、[[澳洲|澳]]、[[大洋洲國家及地區列表|大洋洲]]事務政務司副司長康思博（Siiri Königsberg），以及{{link-en|愛沙尼亞經濟暨交通部|Ministry of Economic Affairs and Communications|經濟暨交通部}}歐盟與國際合作司司長{{link-et|盧畢|Viljar Lubi}}。期間拜會[[中華民國外交部|外交部]][[政务官|政務次長]]並接受款宴，以及[[行政院國家科學委員會|行政院國家科學委員會]]、[[經濟部國際貿易局|經濟部國際貿易局]]、[[交通部觀光局|交通部觀光局]]、[[新竹科學園區|新竹科學園區]]、[[中華經濟研究院|中華經濟研究院]]及中華民國國際經濟合作協會等部會機構。此行也參訪了台灣的文教設施。<ref>{{Cite web |url = http://www.mofa.gov.tw/mobile/News_Content.aspx?s=D7ED3CD644C83997 |author = 公眾外交協調會 |title = 愛沙尼亞國會議員及政府官員訪問團一行應邀訪華 |publisher = 中華民國外交部}}</ref>

2014年7月15日，中華民國[[第一夫人|總統夫人]][[周美青|周美青]]以「臺北愛樂[[管弦樂團|管弦樂團]]」榮譽會長身份訪問愛沙尼亞第2大城[[塔爾圖|塔爾圖]]。當晚「臺北愛樂管弦樂團」應該市「{{link-en|珠戲音樂節|Glasperlenspiel Music Festival}}」主辦單位邀請在{{link-en|塔爾圖聖約翰教堂|St. John's Church, Tartu}}演出。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=BD3B450373053909&sms=02499EFF01F32DED&s=1532405DAB7E1856 |author = 公眾外交協調會 |date = 2014年7月15日 |title = 中華民國總統夫人周美青女士訪愛沙尼亞 |publisher = 中華民國外交部}}</ref>

2016年5月，中華民國前副總統[[呂秀蓮|呂秀蓮]]訪問愛沙尼亞。<ref name="105年外交年鑑"/><ref>{{Cite web |url = http://annettelu.pixnet.net/blog/post/63367492-day-7-5-14-2016-riga----%E6%84%9B%E6%B2%99%E5%B0%BC%E4%BA%9E%E9%A6%96%E9%83%BDtallinn |author = 陳正翰 |date = 2016年5月20日 |title = Day 7 5/14/2016 Riga -- 愛沙尼亞首都Tallinn |publisher = [[痞客邦|痞客邦]]}}</ref>

對愛沙尼亞的政務及領務由'''駐[[拉脫維亞|拉脫維亞]][[臺北經濟文化代表處|臺北代表團]]'''{{Link-en|多重委派|Dual accreditation|兼轄}}、商務由'''駐[[芬蘭|芬蘭]]臺北代表處'''兼轄、新聞業務由'''{{Link-en|駐瑞典台北代表團|Taipei Mission in Sweden}}'''兼轄。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/09.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

[[愛沙尼亞公民|愛沙尼亞公民]]持有[[歐盟護照|歐盟護照]]者也可以[[簽證#签证优惠待遇|免簽證]]的方式入境中華民國，停留最多90天。<ref>{{Cite web|url =https://www.boca.gov.tw/cp-10-4485-149c7-1.html|title =外籍人士來臺免簽證適用國家名單|publisher =外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將愛沙尼亞歸類為'''灰色警示'''：提醒注意（2017年4月19日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-332-443d9-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2015年4月18日，[[經濟部國際貿易局|經濟部國際貿易局]]委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）辦理的「2015年新[[北歐|北歐]]利基產業拓銷團」與36家廠商前往[[俄羅斯|俄羅斯]][[聖彼得堡|聖彼得堡]]、[[芬蘭|芬蘭]]、[[愛沙尼亞|愛沙尼亞]]、[[瑞典|瑞典]]及[[波蘭|波蘭]]等國爭取商機，該團也是[[台灣|台灣]]近期赴北歐地區規模最大的拓銷團。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=661953&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date =  2015年4月17日 |title = 貿協規模最大北歐拓銷團 4月18日啟程波海網商機 |publisher = 經貿透視網}}</ref>

2017年11月，[[工業技術研究院|工業技術研究院]]董事長[[李世光|李世光]]率領經濟部「2017年歐洲[[初创企业|新創]]展會參訪及招商團」赴愛沙尼亞考察商機。<ref name="106年外交年鑑"/>

2018年，愛沙尼亞是中華民國第86大貿易夥伴、第100大[[進口|進口]]夥伴、第74大[[出口贸易|出口]]夥伴。出口至愛沙尼亞的金額為1億0,158萬2,923[[美元|美元]]，年增5.384%；自愛沙尼亞進口的金額為1,999萬7,702美元，年減14.986%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）8,158萬5,221美元，年增11.960%。2017年，愛沙尼亞是中華民國第89大貿易夥伴、第95大進口夥伴、第76大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
