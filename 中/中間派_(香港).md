'''中間派'''（{{lang-en|Moderate groups}}）是[[香港|香港]]的[[中间派|中间派]]政治派系，政治光譜介乎於[[建制派|建制派]]和[[民主派_(香港)|民主派]]之間，主要政治團體包括：[[專業動力|專業動力]]、[[新思維_(香港)|新思維]]、[[民主思路|民主思路]]等，它們沒有統一的政治思想，但都主張理性和務實。

==政治思想==
中間派各黨派對不同政治事件的立場都有著一定程度的差異，例如專業動力政治色彩較淡，主張務實為民生、面對不公義事情可作適度抗爭但不會損害他人利益。<ref>{{Cite web| url =https://www.hkcnews.com/article/10753/%E6%96%B9%E5%9C%8B%E7%8F%8A-%E7%AB%8B%E6%B3%95%E6%9C%83%E8%A3%9C%E9%81%B8-%E9%84%A7%E5%AE%B6%E5%BD%AA-10757/%E6%96%B9%E5%9C%8B%E7%8F%8A| title =【數說選戰】方國珊新界東得票翻一番　10大增長區吸中產建制票| work =眾新聞| date = 2018-03-14}}</ref><ref>{{Cite web| url =https://www.hkcnews.com/article/10856/%E6%96%B9%E5%9C%8B%E7%8F%8A-%E9%9B%A8%E5%82%98%E9%81%8B%E5%8B%95-%E4%B8%80%E5%9C%B0%E5%85%A9%E6%AA%A2-10856/%E9%9B%A8%E5%82%98%E9%81%8B%E5%8B%95| title =方國珊有何立場？佔中「關心學生」不滿戴耀廷　一地兩檢「米已成炊」曾投棄權票| work =眾新聞| date = 2018-03-24}}</ref>民主思路則主張與中央政府溝通<ref name="pape_「民主思路」">{{Cite web| url = http://paper.wenweipo.com/2015/10/28/WA1510280005.htm
|title = 「民主思路」：將着力與中央溝通|author = |work =香港文匯報|date = 2015-10-28}}</ref>、支持「[[袋住先|袋住先]]」、反對2014年[[讓愛與和平佔領中環|佔領中環]]期間的[[雨傘革命|暴力事件]]<ref>{{Cite web| url =http://news.mingpao.com/pns/%E6%9D%8E%E7%AB%8B%E5%B3%AF%EF%B9%95%E3%80%8C%E4%B8%AD%E9%96%93%E6%B4%BE%E3%80%8D%E5%B8%82%E6%B0%91%E5%B0%8D%E6%94%BF%E6%94%B9%E7%9A%84%E6%85%8B%E5%BA%A6/web_tc/article/20140926/s00012/1411669322819| title =李立峯﹕「中間派」市民對政改的態度| work =香港明報| date = 2014-09-26}}</ref>及[[2016年農曆新年旺角騷亂|2016年農曆新年旺角騷亂]]。新思維就說過支持民建聯成為香港的執政黨。在而立法會唯一中間派代表[[陳沛然|陳沛然]]雖支持設立[[西九龙内地口岸区|西九龙内地口岸区]]，但又反對修改《[[議事規則|議事規則]]》，他曾形容自己為「中間偏黃（民主派）」。

==歷史==
經歷過連串的[[雨傘運動佔領區|佔領中環行動]]、[[2015年香港立法會政改表決事件|否決政改]]等政治事件後，造成社會分化、泛民主派陣營進一步分裂，部分組成更為激進及強調香港本土意識及前途[[自決|自決]]的[[本土派_(香港)|本土派]]。

2015年，原屬[[公民黨_(香港)|公民黨]]的立法會議員[[湯家驊|湯家驊]]，在跟隨公民黨立場投票否決政改後，退出公民黨並辭去議席，改而成立中間派組織[[民主思路|民主思路]]。同時，支持政改的[[民主黨_(香港)|民主黨]]的前立法局議員[[狄志遠|狄志遠]]退黨，聯同被民主黨開除黨籍的前立法會議員[[黃成智|黃成智]]，成立中間派組織新思維。

根據2015年[[香港大學民意研究計劃|香港大學民意研究計劃]]民調數據顯示：中間派支持者佔41.9%，民主派支持者佔28.4%，建制派支持者佔11.4%。<ref >{{Cite web|url=http://www.hkcd.com/content/2015-10/27/content_965551.html|title=七成民意認同 港需溫和中間派| work =香港商報| date = 2015-10-07}}</ref>

[[2016年香港立法會新界東地方選區補選|2016年香港立法會新界東地方選區補選]]屬於中間派的[[黃成智|黃成智]]及[[方國珊|方國珊]]敗於泛民主派[[楊岳橋|楊岳橋]]。方國珊得33,424票、黃成智得17,295票，中間派在新界東整體得票約為12%。

[[新思維_(香港)|新思維]]主席[[狄志遠|狄志遠]]認為在選舉上由於「歸邊效應」，所以選民才放棄中間路線，投票給民主派和建制派。他認為只要中間派的政團合作，中間派在[[2016年香港立法會選舉|2016年香港立法會選舉]]大有可能在[[新界東|新界東]]取得一席。<ref>{{Cite web|url =http://the-sun.on.cc/cnt/news/20160301/00407_023.html|title =得票無增長 中間派感失望 |publisher=香港太陽報|date =2016-03-01}}</ref>

[[2016年香港立法會選舉|2016年香港立法會選舉]]，黃成智退出新思維，參選[[社會福利界功能界別|社會福利界功能界別]]。狄志遠參選九龍西。[[方國珊|方國珊]]在新界東得34,544票，第10位落選。<ref>{{Cite web|url =http://hk.apple.nextmedia.com/news/art/20160906/19761793|title =險勝方國珊　梁國雄幸保尾席 |publisher=香港蘋果日報|date =2016-09-06}}</ref>民主思路亦派出兩條名單出選香港島和新界東，可是兩者最終分別以10,028票及8,084票低票落選並被沒收選舉按金。

2017年7月1日，[[湯家驊|湯家驊]]加入[[香港行政會議|香港行政會議]]。

2018年1月，民主派決定舉行[[2018年香港立法會補選民主派初選|初選]]以選出[[2018年3月香港立法會補選|三月立法會補選]]的候選人，當時仍然是民協黨員的[[馮檢基|馮檢基]]成為後備候選人（Plan B），但民主派勸喻馮檢基放棄擔任Plan B。多方壓力下馮檢基宣布放棄，同時埋下馮檢基脫離民主派的伏線。

2018年7月，馮檢基退出民協，9月表示有意參加[[2018年11月香港立法會九龍西地方選區補選|補選]]，以示反對民主派欽點（其實是[[劉小麗|劉小麗]]的意願）李卓人擔任Plan B。馮檢基與民主派決裂後，民協隨即表態支持李卓人。補選拉票期間，馮檢基被李卓人批評馮檢基「界票」，對民主派造成負面影響，形容他是「建制B隊」。最後李卓人和馮檢基雙方票數相加都比建制派代表[[陳凱欣|陳凱欣]]少而落敗，民主派自此不認可馮是民主派中人。

==與政府的關係==
雖則中間派獨立於建制民主兩派之間，但是與中國，以及香港政府有不少互動，例如：湯家驊晉身行政會議、新思維友好[[周奕希|周奕希]]是中國的肇慶市政協<ref>[https://forum.hkej.com/node/127220 葵青同道投建制 泛民難翻盤]</ref>，同時又在建制派幫助之下成為葵青區議會副主席、前成員[[黃成智|黃成智]]被政府委任為[[房委會|房委會]]成員、就連[[專業動力|專業動力]]的[[方國珊|方國珊]]在區議會選舉時都有親中社團推薦<ref>[http://indepth.mingpao.com/php/passage.php?seriesno=12&titleno=1448128839437 愛國社團推薦 民建聯優先 194人名單 自由黨僅佔2]</ref>，因此民主派人士認為中間派人士與政府過從甚密，特別是民主派出身的會被指是「投共」、「華麗轉身」。<ref>[https://www.hk01.com/01%E5%8D%9A%E8%A9%95-%E6%94%BF%E7%B6%93%E7%A4%BE/157097/%E5%91%A8%E5%BA%AD%E8%A2%ABdq-%E5%8D%9A%E8%A9%95-%E8%8F%AF%E9%BA%97%E8%BD%89%E8%BA%AB%E4%B8%8D%E6%98%AF%E7%BD%AA-%E5%8F%AA%E6%98%AF%E6%B9%AF%E5%A4%A7%E7%8B%80%E8%BD%89%E5%BE%97%E5%A4%AA%E7%84%A1%E6%8A%80%E5%B7%A7 華麗轉身不是罪，只是湯大狀轉得太無技巧]</ref><ref>[https://www.vjmedia.com.hk/articles/2018/06/01/181041/%E6%96%B0%E6%99%89%E9%A6%99%E6%B8%AF%E9%9B%BB%E5%BD%B1%E4%BA%BA-%E6%B9%AF%E5%AE%B6%E9%A9%8A 新晉香港電影人——湯家驊]</ref><ref>[https://martinoei.com/article/13822/%E6%B9%AF%E5%AE%B6%E9%A9%8A-%E4%B8%8B%E4%B8%80%E5%80%8B%E5%8A%89%E6%B1%9F%E8%8F%AF 湯家驊——下一個劉江華]</ref>
