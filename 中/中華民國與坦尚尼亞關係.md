'''中華民國與坦尚尼亞關係'''是指[[中華民國|中華民國]]與[[坦尚尼亞聯合共和國|坦尚尼亞聯合共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對坦尚尼亞的相關事務由'''{{Link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-167-84af7-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===外交===
{{see also|中華民國外交|{{link-en|坦尚尼亞外交|Foreign relations of Tanzania}}}}
2014年3月28日，[[阿曼|阿曼]]籍「Tawariq 1」[[漁船|漁船]]船長許清泰於2009年疑涉在坦尚尼亞[[專屬經濟區|經濟海域]]非法捕漁被逮，經{{link-en|坦尚尼亞司法暨憲法部|Ministry of Justice and Constitution Affairs|坦尚尼亞司法單位}}判刑20年[[有期徒刑|有期徒刑]]確定。後經船長本人及家屬、[[中華民國外交部|外交部]]以及{{Link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa|駐南非代表處}}持續進行上訴，最終該國上訴法庭宣判無罪獲釋。<ref>{{Cite web |url = http://www.mofa.gov.tw/News_Content.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=B095BDA67F669EDC |author = 公眾外交協調會 |date = 2014年3月28日 |title = 外交部營救繫獄坦尚尼亞國人許清泰船長獲釋 |publisher = 中華民國外交部}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|坦尚尼亞簽證政策|{{link-en|坦尚尼亞公民簽證要求|Visa requirements for Tanzanian citizens}}}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可在{{link-en|坦尚尼亞機場列表|List of airports in Tanzania|坦尚尼亞的國際機場}}辦理[[簽證#落地签证|落地簽證]]入境，停留最多90天。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-167-84af7-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ArrivalVisa.aspx?n=B58866EDD169513F&AreaSN=D33B55D537402BAA |title = 落地簽證資訊 |publisher = 中華民國外交部}}</ref>

持{{link-en|坦尚尼亞護照|Tanzanian passport}}的{{link-en|坦尚尼亞人口|Demographics of Tanzania|坦尚尼亞公民}}抵台參加國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將坦尚尼亞歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2017年6月23日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-127-e8ab0-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

===貿易===
2012年6月16日，[[經濟部國際貿易局|經濟部國際貿易局]]委託[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）與54家台灣業者執行「游龍專案」，籌組「2012年[[非洲国家和地区列表|非洲南撒哈拉]]拓銷團」。共前往[[坦尚尼亞|坦尚尼亞]]、[[烏干達|烏干達]]、[[安哥拉|安哥拉]]及[[南非|南非]]，是台灣歷來拓銷非洲最大團。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=585707&url=/NewsSearch.asp |author = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]] |date = 2012年6月14日 |title = 貿協游龍專案出擊非洲，史上最大非洲拓銷團開拔！ |publisher = 經貿透視網}}</ref>

2017年，坦尚尼亞是中華民國第109大貿易夥伴、第109大[[進口|進口]]夥伴、第102大[[出口贸易|出口]]夥伴。出口至坦尚尼亞的金額為2,762萬6,891[[美元|美元]]，年減6.324%；自坦尚尼亞進口的金額為1,436萬2,232美元，年增40.671%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）1,326萬4,659美元，年減31.208%。※2016年，坦尚尼亞是中華民國第107大貿易夥伴、第113大進口夥伴、第94大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>
