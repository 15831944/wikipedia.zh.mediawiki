'''中国铁路武汉局集团有限公司'''<ref>{{Cite web |url =  http://money.163.com/17/1023/15/D1EO750V002581PP.html|title = 中铁总公司制大进展:11家地方铁路集团新掌门亮相 |author =  |date = 2017-10-23  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>，原名'''武汉铁路局'''，于2017年铁路局公司化改制而成，现为[[中国铁路总公司|中国铁路总公司]]（简称铁总）旗下的18个全资子公司之一。中铁武汉局集团公司管辖范围基本上在[[湖北省|湖北省]]全境以及[[河南省|河南省]]南部、[[安徽省|安徽省]]的部分地区。共管理线路89条，其中有[[京广高铁|京广高铁]]、[[沪汉蓉客运专线|沪汉蓉客专]]（[[合武铁路|合武]]+[[汉宜铁路|汉宜]]）、[[武九客专|武九客专]]和[[武咸城际铁路|武咸]]、[[武冈城际铁路|武冈]]、[[武孝城际|武孝城际]]等6条[[高速铁路|高速铁路]]，[[京广|京广]]、[[京九|京九]]、[[宜万铁路|宜万]]、[[焦柳铁路|焦柳]]、[[武九铁路|武九]]、[[汉丹铁路|汉丹]]、[[襄渝铁路|襄渝]]、[[宁西铁路|宁西]]、[[漯阜铁路|漯阜]]9条既有普速干线，以及74条支线和联络线。至2017年9月，管理的线路总延展长度为11883.2公里，营业里程5075.5公里，高速铁路营业里程1345.91公里。下辖有车站268个（不含合资铁路所辖车站），其中共有车站269个（不含合资铁路所辖车站），其中[[中華人民共和國鐵路車站等級#.E7.89.B9.E7.AD.89.E7.AB.99|特等站]]3个、[[中華人民共和國鐵路車站等級#.E4.B8.80.E7.AD.89.E7.AB.99|一等站]]14个、[[中華人民共和國鐵路車站等級#.E4.BA.8C.E7.AD.89.E7.AB.99|二等站]]29个、[[中華人民共和國鐵路車站等級#.E4.B8.89.E7.AD.89.E7.AB.99|三等站]]68个、[[四等站|四等站]]92个、[[五等站|五等站]]63个。<ref>{{Cite web |url=http://www.whrailway.cn/html/2017/lujugaikuang_1214/51751.html |title=武汉铁路局概况 |access-date=2018-02-09 |archive-url=https://web.archive.org/web/20180210062105/http://www.whrailway.cn/html/2017/lujugaikuang_1214/51751.html |archive-date=2018-02-10 |dead-url=yes }}</ref>其中，局管内[[武汉铁路枢纽|武汉铁路枢纽]]为国家规划的[[中国铁路枢纽|综合铁路枢纽]]。

== 沿革 ==
[[File:Wuhan_Railway_Bureau_headquarters_4233.jpg|thumb]]
武汉铁路局所辖区域内最早的铁路起源于1893年5月[[张之洞|张之洞]]在[[湖北省|湖北省]]修建的[[大冶铁路|大冶铁路]]。1949年以前，[[汉口|汉口]]和[[武昌|武昌]]曾先后8次设立铁路局（或相当于局级的机构）。1949年以后，武汉地区铁路以[[长江|长江]]为界，以北为汉口铁路分局，管辖界限北至[[许昌|许昌]]，隶属于[[郑州|郑州]]铁路管理局；长江以南设立武昌铁路分局，管辖到长沙以北，隶属于[[衡阳|衡阳]]铁路管理局。1953年1月，武昌铁路分局并入汉口铁路分局，并更名为汉口铁路运输分局，仍隶属于郑州铁路管理局。

1958年10月1日，武汉第一次设局，称武汉铁路管理局。1961年8月，长沙铁路管理局并入武汉铁路管理局。武汉局管辖范围内包括[[汉口|汉口]]、[[信阳|信阳]]和[[长沙|长沙]]等3个办事处。1963年4月1日，武汉铁路管理局被撤销，同时设立武汉铁路分局，管辖范围包括湖北省境内铁路，并被划归为郑州铁路管理局的子机构。1966年6月1日，设立襄樊铁路分局，同样被划归郑州铁路管理局。     

1971年7月1日，武汉第二次设局，称武汉铁路局。主要管辖湖北省境内铁路，下设武汉、襄樊2个铁路分局。

1983年3月1日，武汉铁路局再次被撤销，武汉、襄樊2个铁路分局再度被划给郑州铁路局。1987年3月，[[京广线|京广线]]孟庙以南至东篁店段和[[孟宝铁路|孟宝线]]由郑州铁路局交给武汉铁路分局管辖。    2005年3月18日，[[铁道部|铁道部]]决定撤销武汉、襄樊铁路分局，重新成立'''武汉铁路局'''，实行路局直管站段体制。同年5月15日，[[南昌铁路局|南昌铁路局]][[京九线|京九线]]淮滨至蔡山段、[[宁西线|宁西线]]潢川北站至祖师庙站划归武汉铁路局管辖。

2005年，武汉铁路局进行了工商登记<ref name=guapai/>。

2017年11月4日，济南、乌鲁木齐、南昌、北京、哈尔滨、武汉、郑州、上海、呼和浩特、昆明、青藏铁路公司等11家铁路局（公司）的名称变更通过[[国家工商行政管理总局|国家工商行政管理总局]]核准，武汉铁路局名称变更为'''中国铁路武汉局集团有限公司'''（(国)名称变核内字[2017]第5282号）<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1851596  |title = 15家铁路局将集体变身集团公司：名称变更已通过核准 |author =  |date =2017-11-06   |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月15日，在国家企业信用信息公示系统中，中国铁路武汉局集团有限公司已完成工商登记变更。工商登记变更后，中国铁路武汉局集团有限公司仍为中国铁路总公司100%控股的公司，公司由原先的全民所有制企业变成有限责任公司（非自然人投资或控股的法人独资）企业，注册资本由原先的1517.59691亿元变更为2246.1591亿元人民币，经营范围由原先仅能从事铁路客货运输、运输设施修理与制造、物资采购与供销（国家限制物资除外）、生活服务、承办陆运进出口业务的国际货物运输代理业务等，扩大到囊括通信、信号、供电、供水、工务设备的研制、生产、安装与修理，工务线路、桥涵、隧道修理，装卸机械、机械动力、仪器仪表及货运安保设备的研制、生产、安装与修理，铁路线路、站场设备技术开发与应用，基本建设，对外经济贸易，卫生保健、文化设施服务，仓储，房地产开发和经营，信息及通讯网络技术系统集成、研发和服务等<ref name=wu>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1865510|title =北京济南等14家铁路局正式变身路局集团：完成工商登记变更|author =  |date = 2017-11-15 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

== 机务段 ==
*[[武昌南机务段|武昌南机务段]] (武局南段)
武昌南机务段建成于1976年。位于武汉市洪山区青菱乡李家桥柴湖。2003年，武南机务段兼并具有100年历史的武昌机务段。目前武南机务段主要负责武汉局大部分客车和武九线部分货车牵引工作以及武汉枢纽长江以南地区调车工作，担当本段配属和局管内部分内燃机车的大、中、小、辅修和电力机车中、小、辅修及整备任务，也担当部分动车组机务乘务工作。下辖武昌机务折返段和汉口折返段。
*江岸机务段 (武局江段)
江岸机务段始建于1898年。位于武汉市黄陂区[[武汉北编组站|武汉北编组站]]内。主要担当武汉局管内货运列车、东西向部分旅客列车牵引和武汉枢纽长江以北的调车作业任务。
*襄阳北机务段 (武局襄段)

==车辆段==
*武汉客车车辆段：武汉车辆段始建于1963年。位于武汉市武昌区莲溪寺晒湖路36号。占地面积29公顷，生产房屋面积达76,114平方米，段内客车整备场有整备线13条、存车线8条、临修线1条段内有3线12台位的段修库和2线6台位的空调检修库各1座。担负着武汉地区武汉局普速客运列车的整备工作。汉口客车整备所始建于1992年，位于武汉市江汉区复兴村。汉口客车整备所有整备线12条、临修线2条、存车线4条，主要负责江北地区武汉局担当的客运列车整备检修工作。

== 動車段 ==
