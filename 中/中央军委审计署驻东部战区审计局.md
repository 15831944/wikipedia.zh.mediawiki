'''中央军委审计署驻东部战区审计局'''，位于[[江苏省|江苏省]][[南京市|南京市]]，是[[中央军委审计署|中央军委审计署]]下属局，负责[[中国人民解放军东部战区|中国人民解放军东部战区]]审计工作。

==沿革==
在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月组建[[中央军委审计署|中央军委审计署]]。中央军委审计署新设'''中央军委审计署驻东部战区审计局'''，驻[[中国人民解放军东部战区|中国人民解放军东部战区]]<ref name=tian>{{Cite web |url = http://k.sina.com.cn/article_5044281310_12ca99fde02000iqmo.html?cre=tianyi&mod=pcpager_mil&loc=1&r=9&doct=0&rfunc=8&tj=none&tr=9 |title =军委审计署审计长田义祥：军队审计干部目前近一半人员是文职  |author =  |date = 2018-07-16 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。该审计局所属单位分布在5个城市<ref>{{Cite web |url = http://fanfu.people.com.cn/n1/2017/0716/c64371-29407724.html |title = 军委审计署驻东部战区审计局将主题教育活动融入实践 |author =  |date =2017-07-16  |publisher =  中国共产党新闻网|language =  |accessdate =  |quote =  }}</ref>。
