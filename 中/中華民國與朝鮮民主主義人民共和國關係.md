==政治==
===外交===

[[File:ROC1979-05_National_Assembly_Secretariat_China_Map.pdf|thumb]]第六次會議實錄》所附的《中華民國全圖》，一旁的[[朝鮮半島|朝鮮半島]]部分標注僅有「[[韓國|韓國]]」，並將漢城（今首爾）作為唯一的首都。也未畫出[[三八線|三八線]]。]]
[[第二次世界大戰|第二次世界大戰]]結束之後，[[朝鮮半島|朝鮮半島]]南北分由美蘇佔領託管，與[[中華民國政府|中華民國政府]]關係密切之[[大韓民國臨時政府|大韓民國臨時政府]]，在美國支持下於南方成立[[大韓民國|大韓民國]]，而中華民國隨即予以承認，並視其為代表朝鮮半島唯一的合法政府。在[[冷戰|冷戰]]期間，中華民國一直只承認大韓民國並不承認朝鮮民主主義人民共和國的存在。[[臺灣省戒嚴令|戒嚴時代]]官方發行之世界地圖上朝鮮半島部分曾經長期只標記「[[韓國_(消歧義)|韓國]]」或「大韓民國」字樣<ref>[[立法院國會圖書館|立法院國會圖書館]]電子藏書：[http://lis.ly.gov.tw/lgcgi/ttsweb?@0:0:1:/disk1/lg/lymeeting/lymeetingdb 國民大會歷次會議實錄]：《第一屆國民大會第六次會議實錄》，1979年50月；《第一屆[[國民大會|國民大會]]第七次會議實錄》，1985年05月；《第一屆國民大會第八次會議實錄》，1991年04月；《第一屆國民大會第二次臨時會實錄》，1991年10月 ：《中華民國全圖》。」</ref>。意識形態上的對立加上北韓自身的封閉，因此在冷戰時期雙方自然幾乎沒有任何往來。

[[浦項工科大學|浦項工科大學]]教授朴宣英研究，1948年7月10日中华民国国防部第二厅发给韩国外交部文件报告，1948年2月苏联和北韩在平壤商定，将间岛、安东及吉林定为韩人自治区；苏联欲将延吉、牡丹江、穆陵及其邻近地区划入北韩；这些地区由北韩正规军驻扎，朝-{}-鲜人主管地方行政，并另附了一张地图<ref>[http://old.chnarchive.chosun.com/site/data/html_dir/2006/06/27/20060627000013.html 台湾政府文件：苏联和北韩曾试图将间岛归属于北韩] {{webarchive|url=https://web.archive.org/web/20161221092828/http://old.chnarchive.chosun.com/site/data/html_dir/2006/06/27/20060627000013.html |date=2016-12-21 }}</ref>。

===禁運===
為防止北韓及[[伊朗|伊朗]]可能透過中華民國獲取發展核武器及[[大規模毀滅性武器|大規模毀滅性武器]]計畫所需的設備，在國際[[禁運|禁運]]部份北韓商品情勢下，自2006年6月起，中華民國也制定《輸往北韓及伊朗之敏感貨品清單》，有398項貨品禁止出口。依「戰略性高科技貨品輸出入管理辦法」規定，應向[[經濟部國際貿易局|經濟部國際貿易局]]或委任、委託的機關（構）申請「戰略性高科技貨品輸出許可證」，獲准後便可報關出口。但即使有相關規定，也仍有違法出口而被判刑的案例。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref><ref>{{cite web|url=http://news.ltn.com.tw/news/politics/paper/677296|title= 涉賣北韓武器機具 美逮臺商父子 |publisher= 《[[自由時報|自由時報]]》|author=陳慧萍、侯柏青|date=2013-05-08}}</ref><ref>{{cite web|url=http://news.chinatimes.com/focus/501013397/112013050800093.html|title=我392項產品 禁輸往北韓伊朗|publisher=《[[中時電子報|中時電子報]]》|author=蒼弘慈|date=2013-05-08|deadurl=yes|archiveurl=https://web.archive.org/web/20131004221508/http://news.chinatimes.com/focus/501013397/112013050800093.html|archivedate=2013-10-04}}</ref>

2017年9月22日，[[中華民國行政院|中華民國行政院]]宣佈配合國際對北韓接連發射飛彈及[[2017年朝鲜核试验|核試驗]]的[[联合国安全理事会第2375号决议|制裁]]，全面停止與北韓的雙邊貿易。

==經濟==
{{see also|臺灣經濟|北韓經濟}}
===背景===
中華民國與北韓無外交關係，雙方亦未互設官方性質的政治或經濟交流管道，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）曾計畫前往北韓設立據點，推動雙邊貿易，因客觀環境而未能實現，但民間工商界及旅遊業方面，仍有交流及接觸。[[中華民國政府|中華民國政府]]於1990年3月開放對[[蘇聯|前蘇聯]]及[[阿爾巴尼亞|阿爾巴尼亞]]等前[[社會主義國家|社會主義國家]]的進出口貨品以直接貿易方式辦理，其[[簽證|簽證]]規定也按一般進出口規定辦理，後又於1991年11月核定開放對北韓的直接貿易。<ref name="與我國經貿關係"/>

===貿易===
2008年，首度組團參加「2008年平壤國際商品展覽會」，分別由外貿協會、華悅國際公司及臺灣朝鮮民間經濟技術交流協會租用30個攤位，為當時第2大參展方。<ref name="與我國經貿關係"/>

2009年，雖引起[[美國在臺協會|美國在臺協會]]和[[駐臺北韓國代表部|駐臺北-{}-韓國代表部]]的關切，但'''臺灣朝鮮經貿協會'''由外貿協會在「幕後」協助下成功創立，並於該年1月15日舉辦第一次成立大會。<ref>{{cite web|title=臺灣朝鮮經貿協會成軍|url=http://reader.chinatimes.com/forum_35799.html|author=龔俊榮|date=2009-01-16|publisher=《[[工商時報|工商時報]]》|deadurl=yes|archiveurl=https://web.archive.org/web/20131004224955/http://reader.chinatimes.com/forum_35799.html|archivedate=2013-10-04}}</ref>

中華民國與北韓的貿易往來自2005年起逐漸增加，根據{{link-en|韓國貿易振興公社|KOTRA}}統計，自2011年後一直是北韓第6大貿易夥伴、第20大[[進口|進口]]夥伴、第2大[[出口贸易|出口]]夥伴。2013年，佔北韓總貿易比重的0.55%。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 經濟回顧與展望 |publisher = 中華民國對外貿易發展協會}}</ref><ref name="市場環境分析">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 市場環境分析 |publisher = 中華民國對外貿易發展協會}}</ref>

由於北韓[[联合国安全理事会第2375号决议|遭受國際制裁]]，2018年，北韓與中華民國無官方的貿易統計資料。2017年，北韓是中華民國第173大貿易夥伴、第139大進口夥伴、第216大出口夥伴，在[[亞洲國家和地區列表|亞洲國家]]排名倒數第7、11、1；在[[世界政區索引|世界經濟體]]倒數第66、98、15。出口至北韓的金額為4萬5,495[[美元|美元]]，年減91.874%；自北韓進口的金額為270萬4,225美元，年減77.725%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）265萬8,730美元，年減77.041%。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =[[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
2018年10月1日，前[[中華開發|中華開發]]董事長、[[臺灣綜合研究院|臺灣綜合研究院]]創辦人[[劉泰英|劉泰英]]，在台北宴請北韓[[新義州特別行政區|新義州特別行政區]]長官[[杨斌_(商人)|楊斌]]，在場人士包括台灣企業家，以及韓國[[樂天_(Lotte)|樂天集團]]與馬來西亞[[雲頂集團|雲頂集團]]，舉行一場跨國性的北韓投資會議。早在2002年，楊斌曾詢問劉泰英投資機會，盼能複製[[台灣奇蹟|台灣經濟成功經驗]]，共同合作開發新義州。劉泰英當時建議新義州除了發展國防外交外，還需要行政、立法及司法權，但後來楊斌涉嫌經濟犯罪被中國大陸監禁，加上當時美國亦不樂見台灣協助開發新義州，於是合作案無疾而終。至2018年，北韓領導人[[金正恩|金正恩]]分別[[2018年4月南北韓高峰會|與韓國總統文在寅]]、[[2018年朝美首脑会晤|美國總統川普會面]]，使得[[朝鲜核问题|朝鮮半島緊張局勢]]降溫，才讓台灣與北韓合作開發新義州一案重新提及。<ref>{{Cite web |url = https://udn.com/news/story/7238/3400408 |author = 綜合報導 |date = 2018年10月3日 |title = 劉泰英擺龍蝦宴會北韓新義州特首 謝金河、陳敏薰皆列席 |publisher = 《聯合新聞網》}}</ref>
