[[中山高速公路|中山高速公路]]為[[中華民國|中華民國]]（[[臺灣|臺灣]]）第一條[[中華民國國道|國道]]。1971年開工，1978年通車。全長374.3[[公里|公里]]（[[汐五高架|汐五高架]]則為20.7公里，不包含其中），共有34個[[交流道|交流道]]（含端點）。隨著[[福爾摩沙高速公路|國道三號]]及各地[[快速道路|快速公路]]通車，2017年10月時共有85座交流道（含高架道路交流道與端點），其中包括62座一般性交流道、16座[[系統交流道|系統交流道]]、3座[[匝道|出口匝道]]、5座高架轉接道與端點。

本表共分為交流道距離（里程）、交流道名稱、交流道形式、通車日、出口道路（包含聯絡地區）。交流道形式分為直接式、上下匝道、Y型、單葉型、多葉型、四葉型、菱形型、喇叭型及複合型等。

== 北區 ==
北區定義為[[交通部高速公路局|交通部高速公路局]]北區養護工程分局之轄區，即基隆端－新竹系統交流道（南端）間<ref>{{cite web |title=轄區 |url=https://www.freeway.gov.tw/northern/Publish.aspx?cnid=728 |website=交通部高速公路局北區養護工程分局 |publisher=交通部高速公路局 |accessdate=2019-04-17 |date=2018-04-03}}</ref>。
