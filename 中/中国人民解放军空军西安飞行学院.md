'''中国人民解放军空军西安飞行学院'''，简称'''空军西安飞行学院'''，是空军初级指挥院校，隶属[[中国人民解放军西部战区空军|中国人民解放军西部战区空军]]。

== 沿革==
;空军第二飞行学院
*1949年11月，组建'''中国人民解放军空军第二轰炸学校'''<ref name=dier>{{Cite web |url =http://www.mod.gov.cn/service/2009-09/09/content_4086520.htm  |title =  飞行学院介绍：空军第二飞行学院|author =  |date = 2009-09-09 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
*1949年12月，更名'''中国人民解放军第二航空学校'''<ref name=dier/>。
*1966年3月至1969年4月，先后由[[吉林|吉林]][[长春|长春]]迁址[[四川|四川]][[夹江|夹江]]<ref name=dier/><ref name=scszjsz/>。
*1974年4月，由空军建制划归[[成都军区空军|成都军区空军]]指挥所建制<ref name=scszjsz/>。
*1976年，更名'''中国人民解放军空军第二航空学校'''<ref name=dier/>。
*1985年9月，改属[[成都军区空军|成都军区空军]]建制<ref name=scszjsz/>。
*1986年5月，更名'''中国人民解放军空军第二飞行学院'''<ref name=dier/><ref name=scszjsz> {{Cite book | author =  四川省地方志编纂委员会编| title =  四川省志·军事志| location =  | publisher = 四川人民出版社 | date = 1999年 | pages =98  | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。执行师级权限<ref name=scszjsz/>。
*1992年9月，更名'''中国人民解放军空军第十六飞行学院分院'''。
*1993年6月，恢复'''中国人民解放军空军第二飞行学院'''。
*1999年7月26日，奉中央军委命令，空军第二飞行学院与空军第十六飞行学院合并，组建新的'''中国人民解放军空军第二飞行学院'''，隶属成都军区空军。原空军第十六飞行学院改为'''中国人民解放军空军第二飞行学院西安分院'''。
*2004年6月10日，根据中央军委命令，空军第二飞行学院院部从四川夹江迁至陕西户县，撤销空军第二飞行学院西安分院番号。

;空军第十六飞行学院
*1958年6月22日，根据中央军委决定，空军在[[陕西省|陕西省]][[户县|户县]]组建培训空中领航员和空中通信射击员的专门学校。1958年9月12日，[[中华人民共和国国防部|国防部]]授予部队番号。1958年9月29日，由空军司令部命令为'''中国人民解放军第十六航空学校'''。学校以[[中国人民解放军第三航空预备学校|中国人民解放军第三航空预备学校]]和[[中国人民解放军第一航空学校|中国人民解放军第一航空学校]]、[[中国人民解放军第二航空学校|中国人民解放军第二航空学校]]的部分人员、[[兰州军区空军|兰州军区空军]]户县场站为基础组建。
*1976年6月，更名'''中国人民解放军空军第十六航空学校'''。
*1986年6月23日，空军院校整编，更名'''中国人民解放军空军领航学院'''。
*1992年8月，更名'''中国人民解放军空军第十六飞行学院'''。
*1999年7月26日，奉中央军委命令，空军第二飞行学院与空军第十六飞行学院合并，组建新的'''中国人民解放军空军第二飞行学院'''，隶属成都军区空军。原空军第十六飞行学院改为'''中国人民解放军空军第二飞行学院西安分院'''。
*2004年6月10日，根据中央军委命令，空军第二飞行学院院部从四川夹江迁至陕西户县，撤销空军第二飞行学院西安分院番号。

;空军第五飞行学院
*1949年12月1日，在[[山东省|山东省]][[济南市|济南市]]组建'''中国人民解放军第三驱逐机学校'''<ref name=diwu>{{Cite web |url =http://www.mod.gov.cn/service/2009-09/09/content_4086525.htm  |title = 飞行学院介绍：空军第五飞行学院 |author =  |date = 2009-09-09 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
*1949年12月24日，更名'''中国人民解放军第五航空学校'''<ref name=diwu/>。
*1966年4月，由山东济南迁至[[甘肃|甘肃]][[武威|武威]]<ref name=diwu/>。
*1976年6月，更名'''中国人民解放军空军第五航空学校'''<ref name=diwu/>。
*1986年7月，更名'''中国人民解放军空军第五飞行学院'''<ref name=diwu/>。
*1992年8月，空军第五飞行学院与空军第八飞行学院合并为新的'''中国人民解放军空军第五飞行学院'''<ref name=diwu/>。 

;空军第八飞行学院
*1967年7月1日，经[[中国人民解放军总参谋部|中国人民解放军总参谋部]]批准，以[[中国人民解放军第三航空学校|中国人民解放军第三航空学校]]第二训练团为基础，在[[新疆|新疆]][[哈密市|哈密]]柳树泉组建'''中国人民解放军第八航空学校'''<ref name=wanmi>{{Cite web |url =http://news.ifeng.com/a/20140606/40610371_0.shtml  |title = 万米高空可见哈密戈壁滩巨幅地标 每个字半个足球场大 |author =  |date = 2014-06-06 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref> {{Cite book | author =新疆通志编纂委员会  | title = 新疆通志·军事志 | location =  | publisher =新疆人民出版社  | date =1997年  | pages =  | ISBN =  | accessdate =  | url =  | language =  | quote = 第二篇  军队组织体制 第五章  人民解放军驻疆部队组织体制 第五节  学校、教导（训练）队 }} </ref>。组建初期，辖3个训练团，以原第三航空学校第二训练团为基础组建第八航空学校第一、二训练团，以原[[中国人民解放军第五航空学校|第五航空学校]]部分人员和哈密场站为主组建第八航空学校第三训练团<ref name=bashang/>。
*1968年4月，第二训练团在骆驼圈机场正式开训，因机场周围没有明显地标，第二训练团飞行人员苦战8天，在5个空域下方的戈壁滩上扒开石子露出黄土，制作了由“向斗争中学习”、“只争朝夕”、“排除万难去争取胜利”、“毛主席万万岁！”、“为人民服务”共30个大字组成的5个地标，总面积75000平方米，地标由第二训练团领航主任季臣业设计<ref name=wanmi/><ref name=bashang/>。
*1972年10月，原第五航空学校第一训练团划归第八航空学校，编为第八航空学校第四训练团<ref name=bashang>{{Cite web |url =http://www.bhxzy.cn/index.php?m=home&c=article&v=detail&detailid=10323&model=0  |title =展翅天山——对原空军第八航空学校的历史回顾（上）  |author =李黎光  |date =2017-08-23   |publisher = 八航校战友联谊网 |language =  |accessdate =  |quote =  }}</ref>。
*1976年6月，更名'''中国人民解放军空军第八航空学校'''<ref name=bashang/>。
*1982年5月，因安全飞行15年，中央军委给该校记集体一等功<ref name=baxia/>。
*1986年6月，更名'''中国人民解放军空军第八飞行学院'''<ref name=bashang/>。
*1987年7月，学院安全飞行20年，空军在学院召开空军院校飞行安全经验交流会议，空军党委发来贺信。在[[中国人民解放军|中国人民解放军]]建军60周年前夕，时任院长代表学院出席了全军英模代表会议。1987年10月4日，中央军委主席[[邓小平|邓小平]]签发命令，为学院记集体一等功<ref name=baxia>{{Cite web |url = http://www.bhxzy.cn/index.php?m=home&c=article&v=detail&detailid=10324&model=0 |title =展翅天山——对原空军第八航空学校的历史回顾（下）  |author = 李黎光 |date = 2017-08-28 |publisher = 八航校战友联谊网 |language =  |accessdate =  |quote =  }}</ref>。
*1992年8月,中央军委颁发《全军院校体制编制调整改革方案》。1992年9月，空军在北京召开空军院校工作会议，传达中央军委颁发的《全军院校体制编制整编改革方案》和全军院校工作会议精神，部署空军院校体制改革整编工作。按照该《方案》，空军第五飞行学院和空军第八飞行学院合并整编为空军第五飞行学院，撤销空军第八飞行学院番号。空军第八飞行学院第一训练团改编为空军第五飞行学院第一训练团，空军第八飞行学院第三训练团改编为空军第五飞行学院第二训练团。1992年10月1日，空军院校新番号正式启用<ref name=diwu/><ref name=wanmi/><ref name=bashang/><ref name=baxia/>。

;空军西安飞行学院
*2011年，解放军四总部联合召开第十六次全军院校会议，决定将空军6所飞行学院整合组建为空军哈尔滨、石家庄、西安3所飞行学院<ref>{{Cite web |url =http://theory.people.com.cn/GB/16812836.html  |title =中国军校改革背后的战略指向  |author =  邱蜀林|date = 2012-01-06 |publisher =中国共产党新闻网  |language =  |accessdate =  |quote =  }}</ref>。
*2012年，由空军第二飞行学院、空军第五飞行学院等单位合并组建'''中国人民解放军空军西安飞行学院'''<ref>{{Cite web |url =http://www.mod.gov.cn/reports/201403/wzry/2014-09/12/content_4536775.htm  |title =  空军西安飞行学院|author =  |date = 2014-09-12 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。副军级<ref>{{Cite web |url =http://news.sina.com.cn/o/2012-01-21/112623835432.shtml  |title = 港媒:空军院校体系趋成熟折射空军发展新走向 |author =  |date = 2012-01-21 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。隶属[[中国人民解放军成都军区空军|中国人民解放军成都军区空军]]。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军成都军区空军|中国人民解放军成都军区空军]]转隶[[中国人民解放军西部战区空军|中国人民解放军西部战区空军]]<ref>{{Cite web |url =  http://mt.sohu.com/20160702/n457438206.shtml |title =西部战区空军培育“四有”新一代革命军人传统教育基地挂牌仪式举行  |author =  |date = 2016-07-02 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，调整组建新的'''中国人民解放军空军西安飞行学院'''<ref>{{Cite web |url = http://news.ifeng.com/a/20170530/51183107_0.shtml |title =  2017年军队招生单位名录发布 军改后院校新名称曝光|author =  |date = 2017-05-30 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.mod.gov.cn/shouye/2017-06/29/content_4783975.htm  |title = 国防部公布调整改革后军队院校名称 |author =  |date =2017-06-29  |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
