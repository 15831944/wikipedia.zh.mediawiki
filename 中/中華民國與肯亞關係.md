'''中華民國與肯亞關係'''是指[[中華民國|中華民國]]與[[肯亞|肯亞共和國]]之間的關係。兩國無正式[[外交#外交上的承認|外交關係]]，目前也沒有在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。對肯亞的相關事務由'''{{link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa}}'''{{Link-en|多重委派|Dual accreditation|兼轄}}。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-175-34af8-03-1.html |title = 駐館位置及聯絡方式 |publisher = [[中華民國外交部|中華民國外交部]] [[外交部領事事務局|領事事務局]]}}</ref>

===簽證===
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|肯亞簽證政策|肯亞公民簽證要求}}
兩國公民皆須申請[[簽證|簽證]]方可入境對方國家。持[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以[[簽證#电子签证|電子簽證]]的方式入境肯亞，先至{{link-en|肯亞政府|Politics of Kenya}}網站註冊個人帳號、填寫資料及繳費，列印後於入境時出示。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-175-34af8-02-1.html |title = 簽證及入境須知 |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/ElectronicVisa.aspx?n=6F246645F10311D7&AreaSN=D33B55D537402BAA |title = 電子簽證資訊 |publisher = 中華民國外交部}}</ref>

持[[肯亞護照|肯亞護照]]的{{link-en|肯亞人口|Demographics of Kenya|肯亞公民}}抵台參加國際會議、運動賽事、商展活動，也可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將肯亞歸類為'''橙色警戒'''：避免非必要旅行（2017年12月1日）。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-160-cab20-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>

==經濟==
{{see also|臺灣經濟|肯尼亞經濟{{!}}肯亞經濟}}
===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於首都[[奈洛比|奈洛比]]設立'''台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#AFRICA |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>

2017年，肯亞是中華民國第87大貿易夥伴、第87大[[進口|進口]]夥伴、第77大[[出口贸易|出口]]夥伴。出口至肯亞的金額為9,304萬4,757[[美元|美元]]，年減9.564%；自肯亞進口的金額為3,751萬0,475美元，年增0.672%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）5,553萬4,282美元，年減15.376%。※2016年，肯亞是中華民國第78大貿易夥伴、第84大進口夥伴、第72大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===投資===
在肯亞的[[台灣商人|台商]]有4家，且均為[[成衣|成衣]]加工廠。台商是運用[[美國|美國]]為[[非洲国家和地区列表|非洲國家]]所提供的《{{link-en|非洲成長暨貿易機會法|African Growth and Opportunity Act}}》免[[關稅|關稅]]、免配額的優惠待遇及當地較便宜的工資在肯亞設廠。自[[台灣|台灣]]或[[中华人民共和国|中國大陸]]等地輸入原料（如[[牛仔布|牛仔布]]、[[合成纖維|合成纖維]]針織布或[[鉤針編織|鉤針織布]]）及附配件（[[鈕釦|鈕釦]]、標籤）等，在加工縫製成衣後，輸往美國及[[歐洲國家和地區列表|歐洲]]。

2011年3月17日，成立肯亞台灣商會。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 國家基本資料表 |publisher = 中華民國對外貿易發展協會}}</ref>

==司法==

===犯罪事件===
{{see also|台湾诈骗集团|2016年台灣詐騙集團成員涉嫌電信詐騙案}}
2014年11月30日，{{link-en|肯亞警察|Kenya Police|肯亞警方}}破獲非法入境且未經許可使用[[無線通訊|無線電設備]]的[[電信詐騙|電信詐騙]]案，逮捕的77名嫌犯有28名為[[中華民國籍|中華民國籍]]（以下簡稱[[台灣人|台灣人]]），其餘是[[中華人民共和國國籍法|中華人民共和國籍]]。2016年4月5日，{{link-en|肯亞司法制度|Judiciary of Kenya|肯亞法院}}宣判其中37名嫌犯無罪，23名為台灣人，當庭釋放但限制住居並於21日內離境，判決書沒有指出須遣返回台灣。7日，{{Link-en|駐南非共和國台北聯絡代表處|Taipei Liaison Office in the Republic of South Africa|駐南非台北代表處}}派員抵達肯亞探視。8日，在中華人民共和國（以下簡稱對岸、中方）施壓下，肯亞將其中8名台灣人遣送至對岸，其餘15名被拘留警局，同日，肯亞警方破獲另起詐騙案，逮捕22名台灣人。11日，肯亞將拘留警局的15名台灣人，及8日逮捕的22名台灣人，全部遣送至對岸。最後5名台灣人因[[海峽兩岸關係|兩岸政府]]爭奪管轄權，5度延審至8月23日。但到8月5日，中方在經濟、貿易上不斷向肯亞施壓，強烈要求{{link-en|肯亞政府|Politics of Kenya}}要依據「[[一個中國原則|一個中國原則]]」，將嫌犯遣送至對岸，讓5度延審的審判庭提前。開庭前，中方人員包圍法庭入口，更在[[乔莫·肯雅塔国际机场|奈洛比國際機場]]備妥飛機，準備在宣判後搶人；駐南非台北代表處則聯絡[[半島電視台|半島電視台]]、[[英國廣播公司|英國廣播公司]]、[[法新社|法新社]]、[[美聯社|美聯社]]、[[路透社|路透社]]及[[國際特赦組織|國際特赦組織]]到場，營造輿論壓力。法院最後宣判5名台灣人無罪，法官並明確指出須「遣返回台灣」。<ref>{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20160806/37337817/ |author = 陳培煌 |date = 2016年8月6日 |title = 讚 肯亞案 5台人判無罪遣台 |publisher = [[蘋果日報_(台灣)|蘋果日報]]}}</ref><ref>{{Cite web |url = http://www.bbc.com/zhongwen/trad/china/2016/04/160411_kenya_taiwan_china |author = 劉子維 |date = 2016年4月11日 |title = 肯尼亞涉案獲判無罪台灣人遭遣返至中國 |publisher = [[BBC#BBC中文部|BBC中文網]]}}</ref><ref name="自由">{{Cite web |url = http://news.ltn.com.tw/news/politics/paper/1018605 |author = 陳鈺馥、項程鎮 |date = 2016年8月6日 |title = 肯亞案5台嫌無罪 法官判「遣返台灣」 |publisher = [[自由時報|自由時報]]}}</ref>

45名被遣送至對岸的台灣人，都被拘留在北京市[[海淀區|海淀區]]的[[看守所|看守所]]。[[中華民國法務部|中華民國法務部]]表示，全案由[[中華人民共和國公安部|中華人民共和國公安部]]偵辦，已有家屬透過[[海峽交流基金會|海峽交流基金會]]協助到看守所探視被拘留的親人。<ref name="自由" />2017年12月21日，[[北京市第二中级人民法院|北京市第二中級人民法院]]一審宣判2名台灣主嫌[[有期徒刑|有期徒刑]]15年，其餘被告有期徒刑1年9個月－14年。<ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/paper/1162265 |author = 鍾麗華、林良昇、項程鎮、張文川 |date = 2017年12月22日 |title = 肯亞詐騙案44台嫌 中國全判刑 |publisher = 自由時報}}</ref>

肯亞案引發兩岸關係緊張，台灣批評中方強押人的行徑「鴨霸」（蠻橫），中方官媒《[[環球時報|環球時報]]》發表評論還擊，標題名為「台灣，請給自己留點臉」的文章指出，台灣人常自誇「台灣最美的風景是人」，就應該用行動證實，把電信詐騙犯法辦，不要讓政治綁架法治正義。文中還列舉詐騙受害案例，並提及過去中方曾把海外逮捕的台灣詐騙犯交給[[台灣當局|台灣當局]]處理，結果台灣「把他們放了。」<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20160414/838596/ |author = 綜合報導 |date = 2016年4月14日 |title = 陸官媒談肯亞案 要台灣「給自己留點臉」 |publisher = 蘋果日報}}</ref>

===觀光===
