'''中国人民政治协商会议全国委员会办公厅新闻局'''（简称'''全国政协办公厅新闻局'''），是[[中国人民政治协商会议全国委员会|中国人民政治协商会议全国委员会]]机关的工作部门之一，隶属[[中国人民政治协商会议全国委员会办公厅|中国人民政治协商会议全国委员会办公厅]]。

==沿革==
1992年，设立'''中国人民政治协商会议全国委员会新闻办公室'''（简称'''全国政协新闻办公室'''），协调管理全国政协的新闻宣传工作<ref name=boru>薄茹，政协报刊新闻宣传工作的发展历程，新闻世界2010(12):38-39</ref>。后与[[中国人民政治协商会议全国委员会办公厅研究室|中国人民政治协商会议全国委员会研究室]]（简称全国政协研究室）设在一起<ref name=liuzhd>{{Cite web |url =http://renshi.people.com.cn/n/2014/0120/c139617-24167831-3.html  |title = 刘志勇、刘正东、磨长英增选为广西壮族自治区政协副主席 |author =  |date =  |publisher =中国共产党新闻网  |language =  |accessdate =  |quote =  }}</ref>。

2002年，全国政协机关机构改革后，新组建了[[中国人民政治协商会议全国委员会办公厅研究室|中国人民政治协商会议全国委员会办公厅研究室]]（简称全国政协办公厅研究室）。2002年9月24日，全国政协办公厅研究室成立，升格为副部级机构，下设理论局、信息局、新闻局，均为副局级。新闻局即'''中国人民政治协商会议全国委员会办公厅研究室新闻局'''（简称'''全国政协办公厅研究室新闻局'''）<ref>{{Cite web |url =  http://www.cppcc.gov.cn/2011/09/24/ARTI1316857436015617.shtml  |title = 全国政协办公厅研究室成立大会 |author =  |date = 2002-09-24   |publisher =中国政协网  |language =  |accessdate =  |quote =  }}</ref>，对外又称'''中国人民政治协商会议全国委员会办公厅新闻办公室'''（简称'''全国政协办公厅新闻办公室'''）<ref>{{Cite web |url = http://www.people.com.cn/GB/34948/34951/34955/3142196.html |title = 党政领导干部直系亲属出国申报备案制度试点 |author =  |date =2005-01-25  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。

2006年出台的《中共中央关于加强人民政协工作的意见》提出，“各级党委要积极组织并大力推动关于人民政协的理论研究、宣传和教育工作。”2007年1月，全国政协主席[[贾庆林|贾庆林]]在视察[[人民政协报|人民政协报]]社时，就做好政协新闻宣传工作作出指示。2007年11月28日，全国政协新闻宣传工作座谈会在[[北京|北京]]召开，这是历史上首次召开全国政协新闻宣传工作座谈会，会议以[[中共十七大|中共十七大]]精神为指导，依照照《中共中央关于加强人民政协工作的意见》的要求，讨论修改《全国政协关于加强和改进新闻宣传工作的意见》（讨论稿）<ref name=boru/>。

2000年代末，全国政协办公厅新闻办公室升格为'''中国人民政治协商会议全国委员会办公厅新闻局'''（简称'''全国政协办公厅新闻局'''）<ref name=zhangja>{{Cite web |url = http://news.163.com/17/0301/13/CEER8R6700018AOQ.html |title =张敬安任长春市副市长  |author =  |date = 2017-03-01 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://cppcc.people.com.cn/GB/34952/13521104.html  |title =  全国政协办公厅组织中央媒体赴四川采访|author =  |date = 2010-12-19 |publisher = 中国政协新闻网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.cppcc.gov.cn/2011/11/23/ARTI1322013701379833.shtml |title =中国政协的构成  |author =  |date = 2013-03-14  |publisher =  中国政协网|language =  |accessdate =  |quote =  }}</ref>。
