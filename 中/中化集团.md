'''中国中化集团公司'''（英语：'''SINOCHEM Group'''），简称'''中化集团'''，是国有大型骨干企业。其主营业务涉及[[农业|农业]]、[[能源|能源]]、[[化工|化工]]、[[房地产|房地产]]、[[金融|金融]]等领域，2018年，中化集团在[[财富|财富]][[世界500强|世界500强]]中排名第98位。作为[[中国中央企业|中国中央企业]]之一，中化在国内外有200多家分支机构，旗下有[[中化国际|中化国际]]，[[中华化肥|中华化肥]]，[[方兴地产|方兴地产]]，[[中国种子集团公司|中国种子集团公司]]等子公司。
旗下上市公司有[[中化國際|中化國際]]、[[中化化肥|中化化肥]]（297）、[[方興地產|方興地產]]（817）、[[遠東宏信|遠東宏信]]（3360）、[[GMG環球|GMG環球]]（5IM）等。

截至2016年10月，中化集团旗下共有8家上市公司：<br />
[[A股|A股]]公司：中化国际、扬农化工、江山股份、英特集团；<br />
[[港股|港股]]公司：包括中国金茂、中化化肥、远东宏信；新三板公司金茂投资。

2005年12月，由中国中化集团公司和[[法国|法国]][[道达尔|道达尔]]集团共同投资成立的合资企业[[中化道达尔|中化道达尔]]在[[北京|北京]]成立。

== 历史 ==
1950年，中化集团的前身中国进出口公司成立，在60年代更名为中国化工进出口总公司，70年代时中化打开了中国原油的出口渠道，同时大量出口化工产品和进口[[化肥|化肥]]、[[农药|农药]]等物资<ref>{{Cite web |url = http://www.sinochem.com/1187.html |title= 企业概况-公司简介|language= 中文 |accessdate= 2017年6月11日}}</ref>。

80年代中化开始进行国际化经营试点，并且开始拓展业务领域，1989年，中化集团首次进入[[世界500强|世界500强]]。90年代随着[[改革开放|改革开放]]的持续深入，中化开始推行市场化发展战略，实施战略转型，成为一家多元化集团化的综合贸易公司。2000年，中化集团成为中央直接管理的国有重点骨干企业之一，[[2003年11月10日|2003年11月10日]]，中化集团全称更名为中国中化集团公司，2007年中化通过资产重组收购了[[中国种子集团公司|中国种子集团公司]]和沈阳化工研究院，同年中化旗下的[[方兴地产|方兴地产]]在[[香港|香港]]上市成功。2011年3月30日，中化旗下的[[远东宏信|远东宏信]]在[[香港|香港]]上市成功。

== 业务范围 ==
中化集团的主要业务范围涉及农业、能源、化工、地产、金融等领域。

在能源领域，中化集团已有数十年从事石油业务的历史， 近年来，在增强国际贸易和仓储物流等传统优势的同时，实现了上游勘探开发业务的历史性突破，成为在行业内具有重要影响力的国家石油公司。中化是中国第四大石油公司，业务范围包括石油[[勘探|勘探]]、[[原油|原油]]、燃料油、轻质油、仓储物流和炼化等。在石油贸易方面，中化在改革开放之后中化成为了非常重要的[[石油|石油]]进口企业，除原油进口之外还大规模进行转口贸易。

在农业领域，中化集团是中国最大的全肥种、产供销一体化运作的化肥企业，并在农药研发、生产、销售和种子制种、加工领域保持行业领先。中化收购了中国种子集团公司，同时中化还是中国最大的[[化肥|化肥]]供应分销商，在[[农药|农药]]的研发和生产也一直是中化的优势业务。

在化工领域，中化集团在氟化工、天然橡胶、化工物流等细分领域保持领先优势，并在染料、医药研发和营销方面有着重要影响，是中国领先的化工产品综合服务商。

在金融领域，中化集团向市场提供多种金融产品和服务，旗下的[[外贸信托|外贸信托]]、[[远东租赁|远东租赁]]、[[诺安基金|诺安基金]]、[[中宏人寿|中宏人寿]]等企业均在各自领域具有较强的市场地位和行业影响力 。

在地产领域，中化集团致力于高端酒店和商用物业的专业化开发运营 ，投资控股的[[金茂大厦|金茂大厦]]和[[凯晨世贸中心|凯晨世贸中心]]已经成为上海[[浦东|浦东]]和北京[[西长安街|西长安街]]的地标性建筑。

在油頁岩领域，美國先鋒自然資源公司（Pioneer Natural Resources）2013年1月30日宣布，將向中国中化集团公司（Sinochem Group）出售美國德州Wolfcamp頁岩油氣田40%股權，交易總價為17億美元。
