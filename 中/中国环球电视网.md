'''中国环球电视网'''（{{lang-en|China Global Television Network}}，缩写作 {{lang|la|CGTN}}），[[一个机构两块牌子|又称]]'''中国国际电视台'''，是[[中央广播电视总台|中央广播电视总台]]拥有的一个全球性电视网。其前身为央视的外语国际频道{{noteTag|包括央视[[中国环球电视网纪录频道|英语纪录频道]]、[[中国环球电视网_(频道)|英语新闻频道]]以及[[中国环球电视网西班牙语频道|西班牙语]]、[[中国环球电视网法语频道|法语]]、[[中国环球电视网阿拉伯语频道|阿拉伯语]]、[[中国环球电视网俄语频道|俄语]]国际频道。}}，於[[北京时间|北京时间]]2016年12月31日12:00开播。

==背景==
如不计入外语教学节目，央视的外语节目始于1986年，名称为“CCTV External Service”。1986年12月30日，中国中央电视台《英语新闻》（CCTV NEWS）节目开播，当时每天20:30在[[中国中央电视台财经频道|中央电视台第三套节目]]{{noteTag|1986年开播，与当时的央视二套共用一个频道分时段播出，与现在的CCTV-3无关（现在的CCTV-3在1986年至1992年期间是第四套节目）<ref>赵群. 中央电视台节目有新变化[J]. 新闻实践, 1987(3).</ref>。}}播出。此后中央电视台对外部开办了一系列外语节目，主要面向在华外国人播出。1992年1月1日，央视的英语节目开始通过卫星覆盖[[北美|北美]]地区。[[中国中央电视台中文国际频道|中央电视台第四套节目]]开播后，央视的外语节目开始在CCTV-4播出，直到2002年CCTV-4改为全中文频道为止；而CCTV-2的外语节目在CCTV-4开播后仍保留了一段时间<ref>孙宝国, 吴瑕. 栏目化时期的中国电视对外宣传[J]. 视听, 2013(7):103-105.</ref>。

在CGTN开播之前，央视共有六套外语频道，分别为1997年开播的英语频道{{noteTag|1997年开播时名称为“英语传送频道”，2000年更名为“英语频道”，2010年又更名为“英语新闻频道”。}}、2007年分频播出的西班牙语频道和法语频道{{noteTag|前身为2004年开播的西班牙语·法语频道。}}、2009年开播的阿拉伯语频道和俄语频道、以及2011年开播的纪录频道（CCTV-9）的英文版{{noteTag|中国环球电视网（原央视英语新闻频道）也曾使用“CCTV-9”呼号，时称“英语国际频道（CCTV International），但与2011年开播的央视纪录频道并无关联。纪录频道只是顺延使用了当时空缺的第九频道序号。}}。2016年2月19日，[[中共中央总书记|中共中央总书记]]、[[中国国家主席|中国国家主席]][[习近平|习近平]]在央视、[[人民日报社|人民日报社]]和[[新华社|新华社]]视察时提出“要加强国际传播能力建设，增强国际话语权，集中讲好中国故事，同时优化战略布局，着力打造具有较强国际影响力的外宣旗舰媒体”；央视成立CGTN的举动被认为是受到了习近平讲话的影响，也被认为是中国媒体增强国际影响力、争夺国际话语权的又一举动<ref name=gm1226>{{cite web |url = http://www.guanmedia.com/news/detail_4917.html |title = CCTV→CGTN 中国首个环球电视网元旦上线 |publisher=观媒 |author=一鸣 |accessdate=2016-12-27 |date=2016-12-26 |archiveurl=https://web.archive.org/web/20161227142650/http://www.guanmedia.com/news/detail_4917.html |archivedate=2016-12-27 |deadurl=yes }}</ref><ref name=ap1230>{{cite web |url = http://www.standard.net/Business/2016/12/31/China-state-broadcaster-rebrands-international-outlets.html |title=China state broadcaster rebrands international outlets |publisher=Standard-Examiner |agency=AP |accessdate=2016-12-31 |date=2016-12-30 |archiveurl=https://web.archive.org/web/20161231081639/http://www.standard.net/Business/2016/12/31/China-state-broadcaster-rebrands-international-outlets.html |archivedate=2016-12-31 |deadurl=yes }}</ref><ref name=voa1231>{{cite web |url = http://www.voachinese.com/a/news-china-cgtn-20161232/3658005.html |title=大外宣又一大动作，中国一新国际电视台开播 |publisher=VOA |accessdate=2017-01-10 |date=2016-12-31 |archiveurl=https://web.archive.org/web/20170101152051/http://www.voachinese.com/a/news-china-cgtn-20161232/3658005.html |archivedate=2017-01-01 |deadurl=yes }}</ref>。

==筹备及发展==
2016年12月底，中央电视台特约评论员、[[中央财经大学|中央财经大学]]副教授[[刘春生|刘春生]]在[[新浪微博|新浪微博]][[爆料|爆料]]，中央电视台外语频道将更名为“中国环球电视网”，并且正在进行改版演练、样片制作工作；但该条微博不久后被删除。随后有媒体查阅[[央视网|央视网]]发布的节目单，发现中央电视台俄语频道北京时间2016年12月31日中午12:00将会播出《中国环球电视网俄语频道开播特别新闻节目》；另外中国环球电视网官网（{{URL|www.cgtn.com}}）也已上线。

2016年12月30日，中央电视台正式发布了中国环球电视网筹建的消息，中央电视台外语频道将于2017年新年前夕统一更名为“中国环球电视网”。据中央电视台官方介绍，中国环球电视网将是一个多语种、多平台媒体集群，它将由6个电视频道、1个视频发稿通讯社和新媒体机构等组成，同时拥有北美和非洲两个分台，欧洲分台也在筹建之中。6个电视频道中，原中央电视台英语新闻频道更名为中国环球电视网，中央电视台的西班牙语、法语、阿拉伯语和俄语频道更名为中国环球电视网的西班牙语、法语、阿拉伯语和俄语频道，中央电视台纪录频道英文版更名为中国环球电视网纪录频道<ref name=cnews1230>{{cite web|url=http://news.cctv.com/2016/12/30/ARTIGxTaqwOw4frYiJSfCLNO161230.shtml |title=央视正在筹建中国国际电视台（中国环球电视网） |publisher=央视网 |accessdate=2016-12-30 |date=2016-12-30 |archiveurl=https://web.archive.org/web/20161230113205/http://news.cctv.com/2016/12/30/ARTIGxTaqwOw4frYiJSfCLNO161230.shtml |archivedate=2016-12-30 |deadurl=yes }}</ref>。

2016年12月31日[[北京时间|北京时间]]12:00（[[格林尼治时间|格林尼治时间]]4:00），中国环球电视网正式上线，习近平代表[[中共中央|中共中央]]致信祝贺，[[中共中央政治局常委|中共中央政治局常委]]、[[中共中央书记处书记|中共中央书记处书记]][[刘云山|刘云山]]出席开播仪式并讲话，[[中共中央宣传部|中共中央宣传部]]部长[[刘奇葆|刘奇葆]]在开播仪式上宣读习近平的贺信<ref name=cnews1231>{{cite web|url=http://news.xinhuanet.com/politics/2016-12/31/c_1120226953.htm |title=习近平致信祝贺中国国际电视台（中国环球电视网）开播 |publisher=新华网 |accessdate=2016-12-31 |date=2016-12-31 |archiveurl=https://web.archive.org/web/20161231150612/http://news.xinhuanet.com/politics/2016-12/31/c_1120226953.htm |archivedate=2016-12-31 |deadurl=yes }}</ref>。

中国环球电视网各频道开播时节目如下：
* 中国环球电视网（CGTN主频）：《{{lang|en|Global Watch}}》（环球瞭望）
* 中国环球电视网西班牙语频道：《{{lang|es|Noticias}}》（新闻）
* 中国环球电视网法语频道：《{{lang|fr|Le Journal}}》（新闻）
* 中国环球电视网阿拉伯语频道：《{{lang|ar|الأخبار}}》（新闻）
* 中国环球电视网俄语频道：《{{lang|ru|Новости}}》（新闻）
* 中国环球电视网纪录频道：纪录片《一带一路》（英语配音版）

2017年3月9日，为中国环球电视网运行设立的“中国环球广播电视有限公司”在北京市工商局完成注册手续，正式成立。该公司为央视全资子公司，由央视副台长孙玉胜担任公司董事长兼法定代表人<ref>{{cite news |title=中国环球广播电视有限公司成立 |url = http://www.sapprft.gov.cn/sapprft/contents/6582/318296.shtml |accessdate=2017-03-16 |publisher=国家新闻出版广电总局 |date=2017-03-16 |archiveurl=https://web.archive.org/web/20170316123205/http://www.sapprft.gov.cn/sapprft/contents/6582/318296.shtml |archivedate=2017-03-16 |deadurl=yes }}</ref><ref>{{cite web|title=中国环球广播电视有限公司 |url = http://www.gsxt.gov.cn/%7BVfFU9_YOpown1L_duZGa5bypqY8g6mN-77zQQg_Qr85YTzoUSa37dD-OuHN_jHgEe03HPWq9cBE3MXf8-eFH2zPZRaztx1eFWEMkAmE2KAWMfGAuIdUUr5MwFKubFWtgymYx_nLFvaFMNymVLCY_zQ-1490063658788%7D |publisher=国家企业信用信息公示系统 |accessdate=2017-03-21 |archiveurl = https://web.archive.org/web/20170321034621/http://www.gsxt.gov.cn/%7BVfFU9_YOpown1L_duZGa5bypqY8g6mN-77zQQg_Qr85YTzoUSa37dD-OuHN_jHgEe03HPWq9cBE3MXf8-eFH2zPZRaztx1eFWEMkAmE2KAWMfGAuIdUUr5MwFKubFWtgymYx_nLFvaFMNymVLCY_zQ-1490063658788%7D |archivedate=2017-03-21 |deadurl=yes }}</ref>。

2018年3月21日，根据[[中共中央|中共中央]]印发的《深化党和国家机构改革方案》，原中央电视台（中国国际电视台）、[[中央人民广播电台|中央人民广播电台]]、[[中国国际广播电台|中国国际广播电台]]合并，组建新的[[中央广播电视总台|中央广播电视总台]]，作为[[国务院直属事业单位|国务院直属事业单位]]，归口[[中共中央宣传部|中共中央宣传部]]领导。原中央电视台（中国国际电视台）、中央人民广播电台、中国国际广播电台建制撤销，但对内保留原呼号，对外统一呼号为“中国之声”<ref>{{cite web|url=http://tech.ifeng.com/a/20180321/44915086_0.shtml|title=我国组建中央广播电视总台：中央电视台等撤销建制|author=驱动之家|website=驱动之家|deadurl=no|archiveurl=https://web.archive.org/web/20180322015129/http://tech.ifeng.com/a/20180321/44915086_0.shtml|archivedate=2018-03-22}}</ref>。同年，[[美国司法部|美国司法部]]援引美国《[[外国代理人登记法|外国代理人登记法]]》要求中国环球电视网注册为“外国代理人”<ref>{{cite news |title=新华社在美成“外国代理人” 中美之争延伸 |url=https://www.bbc.com/zhongwen/simp/world-45570571 |accessdate=2019-01-10 |agency=BBC |date=2018年 9月 19日}}</ref>。2019年2月1日，中国国际电视台美国分部正式在美国司法部注册为外国代理人。其注册文件显示，美国分部在2018年12月1日至2019年1月31日从中国中央电视台获得了约800万美元。美国分台台长麻静的年收入为22.1万美元。

==争议==
2017年5月25日，在[[中国乌镇围棋峰会|中国乌镇围棋峰会]][[柯洁|柯洁]]与[[AlphaGo|AlphaGo]]三番棋第二局赛后发布会上，CGTN记者用英语向柯洁提问，并且语速非常快，现场翻译甚至无法及时准确翻译，柯洁以“如果你是中国记者，而我是中国棋手，我认为你应该用中文向我提问”回应而拒绝回答问题。随后CGTN记者不得不又用中文向柯洁重复了问题，他也解释称，CGTN是一个以英语为工作语言的平台，所以提问题需要用英语。同日晚，CGTN官方微博发文表示，英语是CGTN的工作语言，同时也向柯洁和现场翻译表达了歉意。该争议发生后，随即成为了网络热议话题<ref>{{cite web |url = http://business.sohu.com/20170526/n494610598.shtml|title=央视因记者英文提问向柯洁道歉，但这次中国人就别太苛责CCTV了 |publisher=搜狐 |agency=蓝鲸财经记者工作平台 |accessdate=2017-05-28|date=2017-05-26|archiveurl=https://web.archive.org/web/20170818120808/http://www.sohu.com/a/143771310_250147|archivedate=20170-8-18}}</ref>。
