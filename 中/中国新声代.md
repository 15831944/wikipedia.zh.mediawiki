《'''中国新声代'''》是[[湖南电视台|湖南电视台]][[金鹰卡通|金鹰卡通]]頻道于2013年3月开始打造，2013年5月11日起播出第一季的[[儿童|儿童]][[歌唱|歌唱]][[选秀|选秀]]節目<ref>{{Cite web |url=http://www.hunantv.com/c/20130409/1059212119.html |title=《中国新声代》夹缝求生 差异化打造儿童好声音 |accessdate=2014-06-15 |archiveurl=https://web.archive.org/web/20140714190815/http://www.hunantv.com/c/20130409/1059212119.html |archivedate=2014-07-14 |deadurl=no }}</ref>，主要选拔对象为14岁以下的[[青少年|青少年]]儿童，打造“[[中国梦|中国梦]]”式的儿童音乐<ref>{{Cite web |url=http://www.hunantv.com/c/20130409/1105361927.html |title=《中国新声代》打造“中国梦”式儿童音乐 |accessdate=2014-06-15 |archiveurl=https://web.archive.org/web/20140714145308/http://www.hunantv.com/c/20130409/1105361927.html |archivedate=2014-07-14 |deadurl=no }}</ref>。第一季于世界儿歌日开始启动<ref>{{Cite web |url=http://www.hunantv.com/c/20130321/1545393554.html |title=《中国新声代》唱响新童声 世界儿歌日启动报名 |accessdate=2014-06-15 |archiveurl=https://web.archive.org/web/20150705181329/http://www.hunantv.com/c/20130321/1545393554.html |archivedate=2015-07-05 |deadurl=no }}</ref>，于2013年7月28日结束，第二季于2014年8月23日结束，第三季于2015年10月3日結束，第四季于2016年9月17日結束，第五季於2018年10月13日結束。

==具体内容==
===第一季===
<!--*第一季详见：[[2013中国新声代|2013中国新声代]]-->
*第一季中国新声代总决赛于2013年7月28日结束，胡海泉班获得“最佳班级”<ref>{{Cite web |url=http://ent.huanqiu.com/music/yinle-neidi/2013-06/4054675.html |title=《中国新声代》周日单播 超长上演班级对抗 |accessdate=2014-06-15 |archiveurl=https://web.archive.org/web/20160304190554/http://ent.huanqiu.com/music/yinle-neidi/2013-06/4054675.html |archivedate=2016-03-04 |deadurl=no }}</ref>，2013年8月4日举行的“毕业典礼”中，胡海泉班级学员最终角逐前三强，最终“最受欢迎小歌手”三甲为冠军王睿卓，亚军简丹，季军肖宇涛<ref>{{Cite web |url=http://news.7k7k.com/zt/zgxsd/news/597281.html |title=《中国新声代》提前收官 最受欢迎小歌手三甲决出 |accessdate=2014-06-15 |archiveurl=https://web.archive.org/web/20180720091432/http://news.7k7k.com/zt/zgxsd/news/597281.html |archivedate=2018-07-20 |deadurl=no }}</ref>。

===第二季===
<!--*第二季详见：[[2014中国新声代|2014中国新声代]]-->
*第二季中国新声代已經完結，但其中，來自《[[变形计_(第九季)|變形計]]》第九季的山村孩童次仁旺堆也為了圓夢，被節目組邀請成為表演嘉賓，而他的幕後推手，是他的朋友龍龍和其父——歌唱老師和評審——山河老師。

最终，冠軍兼最佳女歌星為譚芷昀，亞軍兼最佳男歌星為余家輝，季軍為封迪。

==后续成长==
*[[新声一班|新声一班]]，由巨匠娱乐、金鹰卡通联手护航，著名音乐人镰田俊哉亲自打造，首张专辑《中国新声代-新声一班》已发行。成员由从《中国新声代》里脱颖而出的新声代儿童偶像爱新觉罗媚、拜尔娜、王睿卓、蹇佳成、姜琪航、曲婉婷、张钰琪等7人组成。
