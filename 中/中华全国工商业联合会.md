'''中华全国工商业联合会'''，简称'''全国工商联'''，是[[中华人民共和国|中华人民共和国]]的一个[[人民团体|人民团体]]，是中华人民共和国最大的民间[[商会|商会]]，是工商界的[[统一战线|统一战线]]组织，在[[中国人民政治协商会议|中国人民政治协商会议]]内设有[[功能界别|界别]]。

== 沿革 ==
全国工商联成立于1953年，其中央组织是中华全国工商业联合会执行委员会。当前，其全国工商联共有会员471万多个，其中企业会员260万多个，团体会员6万多个，个人会员205万多个。行业商会和地方商会覆盖全国。

2017年11月22日，全国工商联十一届六次执委会议在北京举行，通过了全国工商联十一届执行委员会向中国工商业联合会第十二次全国代表大会的报告和《中国工商业联合会章程（修正案）》，决定将这2份文件提请中国工商业联合会第十二次全国代表大会审议。会议还通过了《关于修改中国工商业联合会会徽的决议》。根据此决议，从即日起启用全国工商联新会徽，原会徽同时废止<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1876856 |title = 中华全国工商业联合会启用新会徽，老会徽“光荣退休” |author =  |date =2017-11-23  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.rmzxb.com.cn/c/2017-11-23/1878725.shtml  |title = 全国工商联十一届六次执委会议在京举行 |author =  |date =2017-11-23   |publisher =人民政协网  |language =  |accessdate =  |quote =  }}</ref>。
