'''中国人民解放军陆军勤务学院'''，简称'''陆军勤务学院'''，位于[[重庆市|重庆市]][[沙坪坝区|沙坪坝区]][[重庆大学城|大学城]]，隶属[[中国人民解放军陆军|中国人民解放军陆军]]，是培养[[中国人民解放军|中国人民解放军]][[后勤|后勤]]通用专业军事人才的高等教育院校<ref name=xunli>{{Cite web |url = http://www.81.cn/jwgz/2017-06/10/content_7634426.htm |title = 2017年军校巡礼第十六站：陆军勤务学院（附报考指南） |author =  |date = 2017-06-10|publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。

==沿革==
;中国人民解放军后勤工程学院
*1961年1月，以[[河南省|河南省]][[新乡市|新乡]]的中国人民解放军第三军械预备学校为基础，组建'''中国人民解放军后勤工程学院'''，校址在[[四川省|四川省]][[重庆市|重庆市]]<ref name=xunli/><ref name=cnrgengming>{{Cite web |url =  http://news.cnr.cn/native/city/20170703/t20170703_523830584.shtml|title = 在渝两所部队院校更名 |author =  |date =2017-07-03  |publisher = 央广网 |language =  |accessdate =  |quote =  }}</ref><ref name=scszjsz> {{Cite book | author = 四川省地方志编纂委员会 | title = 四川省志·军事志 | location =  | publisher =四川人民出版社  | date = 1999年 | pages = 99 | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。
**中国人民解放军第三军械预备学校：1954年，创建'''中国人民解放军炮兵第八预备学校'''，隶属[[军委炮兵|军委炮兵]]。1956年，更名为'''中国人民解放军第三军械预备学校'''，转隶[[中国人民解放军总军械部|中国人民解放军总军械部]]，校址在河南省新乡。
*1962年初，[[中国人民解放军后勤学院|中国人民解放军后勤学院]]营房系、中国人民解放军后勤学院油料系、[[中国人民解放军装甲兵工程学院|中国人民解放军装甲兵工程学院]]汽车系、[[中国人民解放军炮兵工程学院|中国人民解放军炮兵工程学院]]步兵武器设计科的人员并入<ref name=scszjsz/><ref name=cnrgengming/>。
*1969年2月19日，[[军委办事组|军委办事组]]以军委〔69〕1号文件发出关于军队院校调整方案的通知，后勤工程学院改称'''中国人民解放军后勤高级专科学校'''<ref name=scszjsz/>，通县军械学校、南京军械学校、后勤技术兵学校并入，并与原后勤工程学院轻武器系共同组建成为后勤高级专科学校军械训练大队。
**中国人民解放军通县军械学校：1949年初，军委后方勤务部为适应[[第二次国共内战|第二次国共内战]]的需要，根据[[周恩来|周恩来]]副主席“火速训练一批军械人员”的指示，在[[北平市|北平市]][[通州区_(北京市)|通县]]组建'''[[中国人民革命军事委员会|中国人民革命军事委员会]]后方勤务部军械训练大队'''，并从[[华北军政大学|华北军政大学]]第一总队选调200名学员到大队学习，大队长为[[杨立三|杨立三]]<ref>{{Cite web |url = http://yzlnb.sjzdaily.com.cn/html/2016-10/19/content_1281473.htm |title =军械训练学习证书   |author =  |date =  2016-10-19|publisher =  燕赵老年报|language =  |accessdate =  |quote =  }}</ref><ref name=yaolan/>。1949年6月，通县军械训练大队改建为'''中国人民解放军军委后勤部军械学校'''，隶属军委后勤部<ref>{{Cite web |url = http://www.mod.gov.cn/service/2008-06/02/content_4085921.htm  |title =军械工程学院  |author =  |date = 2008-06-02  |publisher =中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。不久，更名'''中国人民解放军华北军械学校'''。1951年，中央军委决定，更名'''中国人民解放军第一军械学校'''。1952年10月，更名'''中国人民解放军第一炮兵技术学校'''。1956年2月，中华人民共和国国防部电令，更名'''中国人民解放军通县军械技术学校'''。1960年4月，更名'''中国人民解放军通县军械学校'''。
**中国人民解放军南京军械学校：1947年3月，[[华东野战军|华东野战军]]后勤部在[[山东省|山东省]][[诸城县|诸城县]]创建了华东军区后勤部军械训练队<ref name=yaolan>{{Cite web|url =http://zgsc.china.com.cn/zxun/zhzx/2016-11-21/551458.html|title =军械人才的摇篮 中国人民解放军军械工程学院|author =|date =2016-11-21|publisher =中国网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170809171806/http://zgsc.china.com.cn/zxun/zhzx/2016-11-21/551458.html|archivedate =2017-08-09}}</ref>。1949年5月，由[[徐州市|徐州市]]北郊迁至南京市通济门外红花村开办的华东军区军械训练大队，隶属总后勤部军械部，是培养军械技师、军械所长的初级专业技术学校。1951年春，华东军区军械训练大队与华东军区军械部研究所合并，定名'''中国人民解放军第三军械学校'''，仍驻南京<ref name=nanjing/>。1951年6月9日，华东军区命令，以华东军区后勤军械部军械教导大队扩编组成中国人民解放军第三军械学校<ref name=jsds/>。1952年3月，改为'''中国人民解放军第三炮兵技术学校'''，隶属华东军区炮兵部领导<ref name=nanjing/>。1952年10月28日，中央军委命令，改为中国人民解放军第三炮兵技术学校，归军委炮兵司令部领导。1956年2月4日，[[中华人民共和国国防部|中华人民共和国国防部]]电令，改为'''中国人民解放军南京军械技术学校'''，归[[中国人民解放军总后方勤务部|中国人民解放军总后方勤务部]]军械部建制<ref name=jsjishu>{{Cite web |url = http://www.jssdfz.gov.cn/book/jsz1/D3/D4J3.html |title = 三、人民解放军专业技术学校 |author =  |date =  |publisher =江苏省地方志  |language =  |accessdate =2017-08-09  |quote =  }}</ref>。1956年8月，正式改为中国人民解放军南京军械技术学校<ref name=jsds>{{Cite web |url = http://www.jssdfz.gov.cn/book/jsz2/FL/FL.html |title =大事年表 |author =  |date =  |publisher =江苏省地方志  |language =  |accessdate = 2017-08-09 |quote =  }}</ref>。1960年4月，更名'''中国人民解放军南京军械学校'''。1964年，通县军械学校与南京军械学校合并。招收部队连、排职军官和初中、高中毕业生，学制2至3年。1969年秋，南京军械学校撤销<ref name=nanjing>{{Cite web|url = http://app.nanjing.gov.cn/njnj/sz8_12/8/html/08Noname374.html|title = 第三节 南京解放后军事院校|author = |date = |publisher = 中国南京|language = |accessdate = 2017-08-09|quote = }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
**中国人民解放军后勤技术兵学校：1953年9月，中央军委指示，在[[东北军区|东北军区]]筹建'''中国人民解放军沈阳修械技工学校'''，校址在[[沈阳市|沈阳市]][[铁西区|铁西区]]。1962年8月[[中国人民解放军总后勤部|中国人民解放军总后勤部]]通知，根据1962年6月22日中央军委决定，武汉第三汽车修理工学校与沈阳修械技工学校合并组建'''中国人民解放军后勤技术兵学校'''，校址在沈阳市。
*1973年12月8日，中央军委转发[[中共中央|中共中央]]批准的全军院校领导小组《关于全军恢复和增建41所院校的报告》，决定恢复中国人民解放军军械技术学校。1974年7月，后勤高级专科学校军械训练大队分出，划归石家庄的中国人民解放军军械技术学校（后成为[[中国人民解放军军械工程学院|中国人民解放军军械工程学院]]）。
*1973年，后勤高级专科学校汽车系分出，参与组建中国人民解放军运输技术学校（后成为[[中国人民解放军军事交通学院|中国人民解放军军事交通学院]]）。
*1975年，升格'''中国人民解放军后勤工程学院'''<ref name=xunli/><ref name=scszjsz/>。
*2006年10月，中央军委批准后勤工程学院整体搬迁[[重庆大学城|重庆大学城]]<ref>{{Cite web |url = http://edu.people.com.cn/GB/8216/47717/47721/9260689.html |title = 后勤工程学院院长：以科学发展观指导新校区建设 |author = 李建 |date = 2009-05-07 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军总后勤部|中国人民解放军总后勤部]]转隶[[中央军事委员会训练管理部|中央军事委员会训练管理部]]<ref>{{Cite web |url =http://gaokao.chsi.com.cn/zsgs/zhangcheng/listVerifedZszc--page-7,method-index,lb-1,start-2600.dhtml  |title = 2016年招生章程 |author =  |date =  |publisher = 教育部阳光高考信息公开平台 |language =  |accessdate = 2016-09-15 |quote =  }}</ref>。

;中国人民解放军军事经济学院
*中国人民解放军军需学校：1946年7月，在[[合江省|合江省]][[佳木斯市|佳木斯市]]（今属[[黑龙江省|黑龙江省]]）创建'''东北民主联军供给学校'''，隶属[[东北民主联军|东北民主联军]]总后勤部<ref name=xunli/>，首任校长[[高志洁|高志洁]]，首任政委[[张庆孚|张庆孚]]。1947年2月，更名'''中国人民解放军东北军需学校'''。1949年9月随军南下[[湖南省|湖南省]][[株州|株州]]，更名'''中国人民解放军中南军需学校'''。1951年1月迁至[[汉口|汉口]]，改编为'''中国人民解放军中南军区后勤干部学校'''，为师级单位，隶属中南军区后勤部，校址在汉口罗家墩。同年10月更名为'''中国人民解放军第四后方勤务学校'''。1952年9月，改建'''中国人民解放军军需学校'''，隶属总后勤部<ref name=dwhfzw>{{Cite web|url =http://www.whfz.org.cn/html/2010/cat_309_0126/19148.html|title =武汉解放和新中国时期|author =|date =2010-01-26|publisher =大武汉方志网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170803213015/http://www.whfz.org.cn/html/2010/cat_309_0126/19148.html|archivedate =2017年8月3日|df =}}</ref>。
*中国人民解放军财务学校：1950年，创建'''中国人民解放军华北军区后勤学校'''。1951年，更名'''中国人民解放军第六后方勤务学校'''。1952年，更名'''中国人民解放军财务学校'''。
*中国人民解放军后方工程学校：1955年，[[中国人民解放军军需学校|中国人民解放军军需学校]]（营房管理系）分出，建立'''中国人民解放军后方工程学校'''。
*中国人民解放军军需工业干部学校：总后勤部军需生产部军需工业干部训练班分出，建立'''中国人民解放军军需工业干部学校'''。
*1958年6月以后，以中国人民解放军军需学校为基础，合并中国人民解放军财务学校、中国人民解放军后方工程学校、中国人民解放军军需工业干部学校及[[中国人民解放军后勤学院|中国人民解放军后勤学院]]油料中级班、[[青岛|青岛]]的[[中国人民解放军海军高级专科学校|中国人民解放军海军高级专科学校]]后勤系，组建成面向全军、多科性的'''中国人民解放军后勤学校'''。1961年2月改编为军级单位<ref name=dwhfzw/>。
*1969年10月，中国人民解放军后勤学校奉命停办<ref name=dwhfzw/>。
*1974年2月，复校，更名'''中国人民解放军后勤军政干部学校'''，为军级单位，隶属总后勤部<ref name=dwhfzw/>。
*1978年1月，更名'''中国人民解放军后勤学校'''<ref name=dwhfzw/>。
*1983年6月，升格'''中国人民解放军高级后勤学校'''<ref name=dwhfzw/>。
*1984年，[[中国人民解放军铁道兵工程学院|中国人民解放军铁道兵工程学院]]军事交通系并入。
*1986年6月，中国人民解放军高级后勤学校、襄樊后勤学校合并，组建'''中国人民解放军军事经济学院'''<ref name=xunli/>。
**中国人民解放军襄樊后勤学校：1983年，创建'''中国人民解放军襄樊后勤学校'''。
*1993年，军事经济学院军事交通专业分出，并入中国人民解放军运输工程学院（后为[[中国人民解放军军事交通学院|中国人民解放军军事交通学院]]）。
*2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，由[[中国人民解放军总后勤部|中国人民解放军总后勤部]]转隶[[中央军事委员会训练管理部|中央军事委员会训练管理部]]<ref>{{cite web|title=原总参、总后、总装所辖装备学院等多所军校转隶军委训管部|url=http://www.thepaper.cn/newsDetail_forward_1483043|publisher=澎湃新闻|accessdate=2016-06-13}}</ref>。

;中国人民解放军陆军勤务学院
*2017年，在[[深化国防和军队改革|深化国防和军队改革]]中，以后勤工程学院和军事经济学院为基础组建'''中国人民解放军陆军勤务学院'''，是培养全军后勤通用专业军事人才的高等教育院校，主要承担后勤综合业务、后勤基层管理、油料勤务、军需勤务、军事设施勤务、军事物流管理、军事物资采购管理、财务审计等领域的本科教育、任职培训、研究生教育、士官培训、文职人员培训、外军留学人员培训及国家国民经济动员干部培训等任务。主校区位于重庆市大学城<ref name=xunli/>。2017年8月17日，陆军勤务学院举行揭牌仪式<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1766640 |title = 陆军勤务学院揭牌：以后勤工程学院和军事经济学院为基础组建 |author =  |date = 2017-08-17 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
