'''中華民國與比利時關係'''是指[[中華民國|中華民國]]與[[比利時王國|比利時王國]]之間的關係。1913－1971年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，在對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。比利時在中華民國參與[[國際組織|國際組織]]上採取與[[歐洲聯盟|歐洲聯盟]]一致立場，支持中華民國「有意義參與」。<ref name="駐館與駐地">{{Cite web |url = http://www.taiwanembassy.org/be/post/19.html |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref>

==政治==
===外交===
{{see also|中華民國外交|{{link-en|比利時外交|Foreign relations of Belgium}}}}
{{see also|中国驻比利时大使列表{{!}}中華民國駐比利時外交官列表|比利时驻华大使列表{{!}}比利時駐中華民國外交官列表}}
1912年，中華民國成立後，向比利時派駐外交代表。

1913年10月6日，比利時[[外交#外交上的承認|外交承認]]中華民國[[北洋政府|北洋政府]]，建立公使級外交關係。於首都[[布魯塞爾|布魯塞爾]]設立'''中華民國駐比利時王國[[外交代表機構#公使馆|公使館]]'''，並派駐[[特命全權公使|公使]]（期間派駐[[外交官#代办|代辦]]、[[外交官#其他外交官员|一等秘書]]、二等秘書兼任）。

1927年1月17日，比利時公使洛恩宣布，[[比利時聯邦政府|比利時政府]]願意將[[天津比租界|天津租界地]]交還給中國，以示友好。

1928年11月22日，隨著[[國民革命軍北伐|國民革命軍北伐]]接近尾聲，比利時外交承認中華民國[[國民政府|國民政府]]。

1929年8月31日，兩國簽訂交還[[天津|天津]]的[[租界|租界]]地約章。規定該租界的[[行政管理|行政管理]]權、公產，移交給中國政府；而比租界[[工部局|工部局]]所負的9萬3,000兩[[中国古代银质钱币|白銀]]（包括[[利息|利息]]）由中國政府償還。

1931年1月15日，舉行交接典禮，中華民國收回天津的比利時租界地。

1937年6月1日，建立大使級外交關係。公使館升格為'''中華民國駐比利時王國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。

1943年，旅比學人[[錢秀玲|錢秀玲]]獲悉一名比國青年因從事反[[納粹|納粹]]活動遭[[德意志國防軍|德軍]]宣判[[絞刑|絞刑]]，即積極展開營救。錢女士的堂兄[[錢卓倫|錢卓倫]]為國軍中將，與當時德國駐比利時和法國北部戰區的最高行政長官[[亞歷山大·馮·法肯豪森|亞歷山大·馮·法肯豪森]]將軍為舊識，錢女士透過向法肯豪森將軍陳情，促成該青年順利獲釋，其後錢女士陸續成功營救近百名[[二戰|二戰]]期間被德軍判刑之比利時愛國青年。二戰結束後，為表彰錢秀玲女士的義舉，比利時政府特授予[[國家感謝勳章|國家感謝勳章]]，比國[[瓦隆大區|瓦隆大區]]{{link-en|艾克興市|Écaussinnes}}亦將市中心之一條街命名為「錢夫人路」（Rue Madame Perlinghi）以資紀念。<ref>[http://www.mofa.gov.tw/News_Content_M_2.aspx?n=8742DCE7A2A28761&sms=491D0E5BF5F4BC36&s=E6FC6EC1B1B4571B 比利時僑界重要人士錢秀玲女士家屬應邀來華參加「抗戰勝利暨臺灣光復70週年」紀念活動]，中華民國外交部</ref>

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[比利時|比利時]]。]]
1971年9月，參謀總長[[賴名湯|賴名湯]]訪問比利時。<ref>[http://catalog.digitalarchives.tw/item/00/3a/18/f0.html 賴總長訪問盟邦歸國]，典藏台灣</ref>10月25日，比利時與中華民國斷交。<ref name="外交年鑑"/>

1993年12月，比利時[[比利時參議院|參]]、[[比利時眾議院|眾]]兩院成立「比利時[[台灣|台灣]]之友[[國會議員|國會議員]]聯誼會」，推動兩國[[國會|國會]]交流，曾多次通過友台決議案，最近1次為2015年11月。<ref name="駐館與駐地"/>

比利時眾、參議院分別於2003年2月及4月通過呼籲中國撤除[[中國人民解放軍對臺灣飛彈|對台飛彈]]的決議案，亦要求政府[[欧洲联盟签证政策|核發簽證]]給予赴比利時作私人訪問的台灣高階政府人員。<ref name="93年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2004/08/08_01_04.htm |title = 《中華民國93年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

2007年4月26日，比利時眾議院全會通過《台灣在[[世界衛生組織|世界衛生組織]]之地位》決議案，要求比利時聯邦政府促請世界衛生組織（WHO）儘速研究適合台灣參與該組織方式（比照台灣參與[[世界貿易組織|世界貿易組織]]，WTO模式），並提出一項正式且恆久解決WHO與台灣間科學、技術暨設備合作之議定書。並將本決議案轉知[[歐盟部長理事會|歐盟部長理事會]]、[[歐洲執委會|歐洲執委會]]、[[歐洲聯盟#會員國|歐盟會員國]]、[[中华人民共和国政府|中國政府]]、[[中華民國政府|台灣政府]]、[[各国议会联盟|國際國會議員聯盟]]、世界貿易組織及世界衛生組織。<ref>{{Cite web |url = http://www.mofa.gov.tw/igo/News_Content.aspx?n=1548B29AE7A54B88&sms=F2E48E6A8FF685DC&s=D01BF0A005BCBEEC |author = 公眾外交協調會 |date = 2007年4月30日 |title = 比利時眾議院全會通過「台灣在世界衛生組織之地位」決議案比利時眾議院全會通過「台灣在世界衛生組織之地位」決議案 |publisher = 中華民國外交部}}</ref>

2008年5月，比利時外交部長{{link-en|德古奇|Karel De Gucht}}頒贈{{link-en|比利時王冠司令勳章|Order of the Crown (Belgium)|王冠司令勳章}}給予長榮集團董事長[[張榮發|張榮發]]，表彰其對促進雙邊經貿關係的貢獻。<ref name="97年外交年鑑"/>

2012年2月16日，比利時眾議院全會通過決議，並於2月20日對外公布，支持[[海峽兩岸關係|兩岸]]持續對話交流及台灣參與[[國際組織|國際組織]]並與歐盟洽簽[[綜合經濟合作協議|經濟合作協定]]（ECA）。該決議文中籲請比利時政府：協同歐盟其他會員國採取一致有效行動，鼓勵[[中华人民共和国|中國大陸]]與[[台灣|台灣]]繼續對話及擴大交流，以便和平解決歧見；重申支持台灣有意義參與國際組織，尤其以{{link-en|觀察員地位|Observer status|觀察員身分}}參與[[聯合國氣候變化綱要公約|聯合國氣候變化綱要公約]]（UNFCCC）及[[國際民航組織|國際民航組織]]（ICAO）；支持歐盟與台灣擴大經濟合作，尤其是洽簽經濟合作協定。<ref>{{Cite web |url = http://www.mofa.gov.tw/igo/News_Content.aspx?n=F39864F689523588&sms=7D1F8B934FF18A3D&s=6A06E33B5DF46498 |author = 公眾外交協調會 |date = 2012年2月24日 |title = 我政府歡迎比利時眾議院通過決議案，支持我參與國際組織及與歐盟洽簽「經濟合作協定（ECA）」 |publisher = 中華民國外交部}}</ref>

2016年4月18日，台灣代表團（以下簡稱台方）出席[[經濟合作暨發展組織|經濟合作暨發展組織]]（OECD）與比利時政府聯合舉辦的「鋼鐵部門產能過剩及產業結構調整高階論壇」及「鋼鐵委員會第八十次例會」，比利時副總理辦公室國際事務主任對台方表示中華人民共和國代表團（以下簡稱中方）認為在高階官員對話會議中，台方團長層級不足，要求不得與會{{noteTag|[[中華民國經濟部|中華民國經濟部]]強調，我方當場提出抗議，表示近半數代表團團長位階與我相仿，理由不足，但抗議無效後即離場，並持續向主辦單位抗議。官員表示，當初此會議為{{link-en|比利時外交部長列表|List of Foreign Ministers of Belgium|比利時外交暨經濟部長}}致函邀請我方組團，當時均未說明有任何限制，對我方層級（組長級）也沒有任何意見。}}。由於中方施壓，於主辦單位要求下，台方退出會場。在向比利時抗議後，台方在19日重新回到會場開會，[[駐歐盟兼駐比利時代表處|駐歐盟兼駐比利時代表處]]表示，台方以「Taiwan」名義與會，座位安排在[[土耳其|土耳其]]與[[瑞典|瑞典]]兩國中間。比利時也派出代表團團長、參事向台方表示歉意，坦言承受相當大壓力。此事件中，[[駐法國台北代表處|駐法國台北代表處]]同步向OECD表達嚴正立場、[[行政院大陸委員會|行政院大陸委員會]]也向中方表達抗議。<ref>{{Cite web |url = http://news.ltn.com.tw/news/focus/paper/981004 |author = 黃佩君、歐祥義、陳伃軒 |date = 2016年4月20日 |title = 比利時國際鋼鐵會議 中國逼我退場 |publisher = [[自由時報|自由時報]]}}</ref><ref>{{Cite web |url = http://www.chinatimes.com/realtimenews/20160419005602-260401 |author = 陳建瑜 |date = 2016年4月19日 |title = 國際鋼鐵會議逼台灣團退席 比利時官方道歉 |publisher = [[中時電子報|中時電子報]]}}</ref>

===學術===
[[行政院國家科學委員會|行政院國家科學委員會]]（國科會，後升格[[中華民國科技部|科技部]]）與{{link-en|比利時國家科學基金會|National Fund for Scientific Research}}、荷語區[[校际微电子中心|校際微電子研究中心]]（IMEC）、{{link-en|法蘭德斯生物科技中心|Vlaams Instituut voor Biotechnologie}}（VIB）皆簽有雙邊合作協議。就[[基礎科學|基礎科學]]研究、[[微電子學|微電子]]及[[半導體|半導體]]科技、[[生物技术|生物科技]]等領域推動學術合作與研究人員交流；[[國立交通大學|國立交通大學]]與比利時[[荷语天主教鲁汶大学|荷語魯汶大學]]及IMEC分別簽有合作協議，推動[[碩士|碩]]、[[博士|博士]]研究生及研究人員合作研習；[[中央研究院|中央研究院]]農業生物科技研究中心則與VIB、[[根特大學|根特大學]]有頻繁學術交流。<ref name="駐館與駐地"/>

2015年12月，[[中華民國國家圖書館|中華民國國家圖書館]]在根特大學設立比利時第一個「臺灣[[漢學|漢學]]資源中心」。<ref name="104年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2017年9月，為推動[[綠能|綠能]]發展，中華民國科技部政務次長蘇芳慶率團訪問比利時荷語區[[亨克|亨克]]能源城（EnergyVille）的協力機構，即荷語魯汶大學ELECTA電力團隊、校際微電子研究中心[[電池|電池]]及[[節能|節能]]團隊、{{link-en|弗拉芒技術研究所|Flemish Institute for Technological Research}}（VITO）綠能應用團隊，以瞭解該地區節能城市、[[智慧電網|智慧電網]]的相關計畫。<ref name="106年外交年鑑"/>

===會展===
2010年，[[工業技術研究院|工業技術研究院]]（工研院）和[[歐洲聯盟|歐盟]]駐台經貿辦事處舉辦「2010 Galileo Executive Forum」，並邀請歐盟官員及產業專家赴台與會，建立雙方接觸平台。<ref name="投資環境簡介"/>

2012年6月，比利時於「台北國際食品展覽會」中的國家館設置20個攤位。知名食品商（包含[[啤酒|啤酒]]、GALLER[[巧克力|巧克力]]、{{link-en|SPA礦泉水|Spa (mineral water)}}等）均赴台參展。<ref name="投資環境簡介"/>

2013年1月，比利時是當年度「[[台北國際書展|台北國際書展]]」的主題國，並設立「比利時館」，於[[書展列表|書展]]中推廣比利時[[法語|法文]]書籍、[[漫畫|漫畫]]及[[薩克斯風|薩克斯風]]等。<ref name="投資環境簡介"/>
