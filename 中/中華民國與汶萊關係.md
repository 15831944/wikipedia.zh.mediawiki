'''中華民國與汶萊關係'''是指[[中華民國|中華民國]]與[[汶萊達魯薩蘭國|汶萊達魯薩蘭國]]（又稱'''汶萊和平之國'''）之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，但於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構。

===代表機構===
{{see also|中華民國駐外機構列表|駐臺灣外交機構列表}}
{{see also|{{link-en|汶萊駐外機構列表|List of diplomatic missions of Brunei}}|{{link-en|駐汶萊外交機構列表|List of diplomatic missions in Brunei}}}}
1978年9月27日，中華民國在首都[[斯里巴卡旺市|斯里巴卡旺]]設立'''駐汶萊[[遠東|遠東]]貿易文化中心'''。1996年7月1日，更名為'''駐汶萊[[台北经济文化代表处|台北經濟文化辦事處]]'''（Taipei Economic and Cultural Office in Brunei Darussalam）。2006年3月17日，辦事處暫停運作。2007年3月1日，正式復館，全面恢復辦理各項[[領事|領事]]業務（包括[[簽證|簽證]]、[[護照|護照]]及文件證明等）及僑務等事務。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/06.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref><ref name="駐外館處">{{Cite web |url = http://www.mofa.gov.tw/OverseasOffice_Content.aspx?n=6357834932B83C83&sms=5A961ED2E4BA25E8&s=E0588283EFAA02AD&os=21595FA41A9EE70A |title = 駐外館處 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>

2002年9月12日，簽署汶萊在台設處協定。10月8日，在首都[[台北市|台北]]設立'''汶萊貿易旅遊代表處'''（Brunei Darussalam Trade and Tourism Office）。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__01.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref><ref name="駐華外國機構">{{Cite web |url = http://www.mofa.gov.tw/OfficesInROC_Content.aspx?n=8CEB2B5F5436B997&sms=8EBFADC1592C7BFE&s=E0588283EFAA02AD&os=F29A02A9D36C47F0 |title = 駐華外國機構 |publisher = 中華民國外交部}}</ref>

===貿易===
2018年，汶萊是中華民國第60大貿易夥伴、第43大[[進口|進口]]夥伴、第116大[[出口贸易|出口]]夥伴。出口至汶萊的金額為1,287萬3,117[[美元|美元]]，年增7.270%；自汶萊進口的金額為3億8,477萬0,247美元，年增72.378%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）3億7,189萬7,130美元，年增76.078%。2017年，汶萊是中華民國第66大貿易夥伴、第53大進口夥伴、第116大出口夥伴。<ref>{{Cite web |url = https://cus93.trade.gov.tw |title = 中華民國進出口貿易統計 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

===招商===
2013年10月26日，[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）帶領[[台灣商人|台灣廠商]]參加汶萊首屆「2013年[[東南亞國家協會|東協]]博覽會」，有[[馬來西亞|馬來西亞]]、[[日本|日本]]、[[韓國|韓國]]、[[杜拜|杜拜]]（[[阿拉伯聯合大公國|阿拉伯聯合大公國]]）、[[菲律賓|菲律賓]]、[[泰國|泰國]]、[[緬甸|緬甸]]、[[柬埔寨|柬埔寨]]、[[中華人民共和國|中華人民共和國]]及當地業者參展，中華民國為最大參展國，共計有30家廠商展出。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=622046&url=/NewsSearch.asp |title = 汶萊「2013年東協博覽會」台灣參展團亮麗登場 |date = 2013年10月30日 |publisher = 經貿透視網}}</ref>期間，[[中國駐汶萊大使館|中華人民共和國駐汶萊大使館]]要求主辦單位將中華民國展館名稱從「[[臺灣|臺灣]]館」（Taiwan Pavilion）改為「臺灣企業館」（Taiwan Enterprises Pavilion）；另要求{{link-en|汶萊媒體|Media of Brunei|汶萊英文報紙}}將刊登在其廣告頁的「Taiwan Pavilion」更名為「Taiwan Booth」（臺灣攤位）。但中華民國最終仍堅持保有展館名稱。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/WebArchive/2151/%E4%B8%AD%E5%9C%8B%E5%A4%A7%E9%99%B8%E9%98%BB%E6%92%93%E6%88%91%E5%9C%8B%E9%9A%9B%E7%A9%BA%E9%96%93%E4%BA%8B%E4%BE%8B%EF%BC%88102%E5%B9%B4%EF%BC%89.pdf |author = 公眾外交協調會 |title = 中國大陸阻撓我國際空間事例（102年事例）|publisher = 中華民國外交部}}</ref>

2014年6月25日，[[中華民國外交部|中華民國外交部]]委託外貿協會組團參加汶萊年度最大規模商展「2014年東協消費展」，台灣館為最大參展國，共計有17家廠商展出。<ref>{{Cite web |url = http://www.trademag.org.tw/News.asp?id=639295&url=/NewsSearch.asp |title = 貿協組團赴汶萊參展，積極行銷MIT及開拓穆斯林市場 |date = 2014年6月30日 |publisher = 經貿透視網}}</ref>

==汶萊華人==

{{see also|汶萊華人|臺灣裔{{!}}海外臺僑}}
