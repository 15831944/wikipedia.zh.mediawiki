'''中華民國與哥倫比亞關係'''是指[[中華民國|中華民國]]與[[哥倫比亞共和國|哥倫比亞共和國]]之間的關係。1941－1980年，兩國有官方[[外交#外交上的承認|外交關係]]，斷交後，於對方[[首都|首都]]互設[[實質大使館|具大使館性質]]的代表機構，但哥倫比亞於2002年關閉。

==政治==
===外交===
[[File:Voting_res_2758.png|thumb]]一案棄權，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括[[哥倫比亞|哥倫比亞]]。]]
{{see also|中華民國外交|{{link-en|哥倫比亞外交|Foreign relations of Colombia}}}}
{{see also|中国驻哥伦比亚大使列表{{!}}中華民國駐哥倫比亞外交官列表|哥伦比亚驻华大使列表{{!}}哥倫比亞駐中華民國外交官列表}}
1941年4月9日，中華民國與哥倫比亞建立公使級[[外交#外交上的承認|外交關係]]，並派駐[[特命全權公使|公使]]（由[[中国驻巴西大使列表|駐巴西公使、大使]]兼任）。

1947年5月8日，於首都[[波哥大|波哥大]]設立'''中華民國駐哥倫比亞共和國[[外交代表機構#公使馆|公使館]]'''，11月6日正式運作。<ref name="駐地">{{Cite web |url = http://www.roc-taiwan.org/co/post/1993.html |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref>

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，兩國繼續維持邦交。

1961年4月6日，公使館升格為'''中華民國駐哥倫比亞共和國[[外交代表機構#大使馆|大使館]]'''，並派駐[[特命全權大使|大使]]。<ref name="駐地"/>

哥倫比亞亦於1961年派駐中華民國大使（由駐[[日本|日本]]大使兼任），後於1965年在[[台北市|台北]]設立'''哥倫比亞共和國駐中華民國大使館'''，由[[外交官#代办|代辦]]主持館務。<ref name="86年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1997/3-1-6.html |title = 《中華民國86年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

1972年3月28日，設立'''駐[[巴蘭幾亞|巴蘭幾亞]][[領事館|領事館]]'''，5月20日正式運作。1979年11月19日，升格為'''總領事館'''。<ref name="駐地"/>

1980年2月8日，哥倫比亞與中華民國斷交。<ref name="駐地"/>

2013年5月14日，{{link-en|哥倫比亞參議院|Senate of Colombia}}外交委員會無異議通過支持[[台灣|台灣]]參與[[國際民航組織|國際民航組織]]（ICAO）。6月20日，參議院全會通過由外委會[[親台派|友台]]議員共同聯名支持台灣以{{link-en|觀察員身分|Observer status}}參與ICAO案之決議案。該案後由參議院秘書長致函給ICAO秘書長。<ref name="102年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2013/11.pdf |title = 《中華民國102年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2014年3月25日，哥倫比亞參議院全會無異議通過支持台灣以觀察員身分參與[[聯合國氣候變化綱要公約|聯合國氣候變遷綱要公約]]（UNFCCC）建議案。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2014/11.pdf |title = 《中華民國103年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>2015年11月18日，參議院環境委員會再度通過有關建議案。<ref name="104年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2015/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B104%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國104年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

===貿易===
哥倫比亞是中華民國在[[拉丁美洲|拉丁美洲]]第3大出口市場，次於[[墨西哥|墨西哥]]及[[巴西|巴西]]。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]在首都[[波哥大|波哥大]]設立'''駐哥倫比亞代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=215&pid=305863 |title = 本部駐外單位|publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

2018年，哥倫比亞是中華民國第56大貿易夥伴、第64大[[進口|進口]]夥伴、第46大[[出口贸易|出口]]夥伴。出口至哥倫比亞的金額為3億3,930萬8,917[[美元|美元]]，年增5.031%；自哥倫比亞進口的金額為1億1,073萬2,591美元，年減12.105%。貿易呈現[[淨出口|出超]]（[[盈餘|盈餘]]）2億2,857萬6,326美元，年增15.986%。2017年，哥倫比亞是中華民國第57大貿易夥伴、第58大進口夥伴、第46大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =經濟部國際貿易局}}</ref>

台灣商品在[[筆記型電腦|筆記型電腦]]方面，有[[宏碁|宏碁]]、[[華碩電腦|華碩電腦]]；[[主機板|主機板]]有宏碁、[[技嘉科技|技嘉科技]]、[[微星科技|微星科技]]；[[記憶體|記憶體]]有[[金士頓科技|金士頓科技]]、[[創見資訊|創見資訊]]；[[滑鼠|滑鼠]]、[[電腦鍵盤|電腦鍵盤]]、[[揚聲器|電腦喇叭]]等有技嘉科技、[[昆盈企業|昆盈企業]]及[[路由器#无线网络路由器|無線路由器]]的[[友訊科技|友訊科技]]等在哥倫比亞佔有相當市場。<ref name="投資環境簡介"/>

===投資===
[[臺灣商人|台商]]主要投資產業：[[製造業|製造業]]的投資金額約7,000萬美元，主要在首都[[波哥大|波哥大]]（[[醬油|醬油]]、[[醬|調味醬]]、[[食用蕈|食用菇種]]、袋包類）、第2大城[[麥德林|麥德林]]（[[運動鞋|運動鞋]]、運動服、[[牛仔褲|牛仔褲]]）、第3大城[[卡利|卡利]]（[[藤|藤]]製[[家具|家具]]、[[機車_(公路)|機車]]組裝）及[[庫庫塔|庫庫塔]]（[[塑膠袋|塑膠袋]]）等。[[第三產業|服務業]]則包括：[[貿易|貿易]]服務業（[[電腦|電腦]]及周邊設備、電工及[[電子產品|電子器材]]、[[照明|照明]]設備、[[汽車零組件列表|汽車零配件及零組件]]、[[食物|食品]]、[[禮物|禮品]]、[[簾|窗簾]]及[[咖啡|咖啡]]等），以及[[賭場|賭場]]、[[飲食業|餐飲業]]、[[翻譯|翻譯]]、[[語言教育|語言教學]]、[[法律|法律]][[顧問|顧問]]等。<ref name="投資環境簡介">{{Cite web |url = http://twbusiness.nat.gov.tw/countryfiles/co/pdf/103-41Colombia.pdf |date = 2014年12月 |title = 投資環境簡介 |publisher = 經濟部投資業務處}}</ref>

目前成立的組織有：哥倫比亞台商協會，現有會員廠商25家。<ref name="投資環境簡介"/>

===僑民===

旅居哥倫比亞的臺僑約400多人，大多集中在首都[[波哥大|波哥大]]，以及第3大城[[卡利|卡利]]與第4大城[[巴蘭幾亞|巴蘭幾亞]]。<ref name="駐地"/>

2002年6月15日，全僑民主和平聯盟哥倫比亞支盟在波哥大舉行成立大會。<ref name="91年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2002/03/03__01__06.htm |title = 《中華民國91年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

===援助===
2010年，財團法人國際合作發展基金會（國合會）與美國慈善組織合作「哥倫比亞{{link-en|索阿查省|Soacha Province}}[[土石流|土石流]]災害防治計畫」。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2010/2.1.6.pdf |title = 《中華民國99年外交年鑑》〈第二章 對外關係〉 |publisher = 中華民國外交部}}</ref>

2013年，協助哥倫比亞弱勢婦女電腦職業訓練、改善貧困鄉村學童文具及育樂設備、因公殘障警察人員醫護設備等計畫。<ref>{{Cite web |url = https://www.mofa.gov.tw/Upload/RelFile/17/262/133bf67f-6547-4c62-8bfd-ef26e383f644.pdf |author = 國際合作及經濟事務司 |date = 2014年4月23日 |title = 國際合作發展事務102年度報告（參、102年度我國國際合作發展事務執行概況）|publisher = 中華民國外交部}}</ref>
