《'''中原突围'''》，中国大陆电视剧，导演[[石伟|石伟]]，主演[[徐平|徐平]]、[[刘永生|刘永生]]、[[郭伟华|郭伟华]]、[[徐成峰|徐成峰]]、[[高田昊|高田昊]]、[[席与立|席与立]]。

== 剧情 ==
{{See alse|中原突围}}
该剧讲述了[[抗日战争|抗日战争]]结束之后[[国共内战|国共内战]]时期[[中国共产党|中国共产党]]和[[中国国民党|中国国民党]]为争夺中国大陆的政权的一场战役“中原突围”。<ref>{{cite web |url=http://www.people.com.cn/GB/paper1787/6525/640771.html |title=献礼片也要讲故事《中原突围》24日央视首播 |accessdate= 2002年6月22日|publisher=人民网 |work= |language =zh-cn}}</ref>

==花絮==
该剧是“[[中共十六大|中共十六大]]”和“中国共产党建党80周年”的献礼片。<ref>{{cite web |url=http://news.66wz.com/system/2004/08/10/057600335.shtml |title= 《中原突围》今晚荧屏突围|accessdate=2004年8月10日 |publisher=温州网 |work= |language =zh-cn}}</ref><ref>{{cite web|url=http://news.china.com/zh_cn/focus/jiandangzn/beijiziliaoaa/10000622/20010606/10039780.html|title=重大革命历史题材电视剧《中原突围》开拍|accessdate=2001|publisher=中华网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160304195641/http://news.china.com/zh_cn/focus/jiandangzn/beijiziliaoaa/10000622/20010606/10039780.html|archivedate=2016年3月4日|df=}}</ref><ref>{{cite web |url= http://www.people.com.cn/GB/wenyu/223/5724/5727/20010628/499328.html|title=献礼剧：《中原突围》 |accessdate=2001 |publisher=人民网 |work= |language =zh-cn}}</ref>首播式在[[人民大会堂|人民大会堂]]河南厅举行。<ref>{{cite web |url=http://news.xinhuanet.com/newscenter/2002-06/29/content_462548.htm |title=中原突围首播式 |accessdate= |publisher=新华网 |work= |language =zh-cn}}</ref>

该剧由[[湖北电视经济频道|湖北电视经济频道]]、[[鄂豫边区|鄂豫边区]]革命史编辑部和[[八一电影制片厂|八一电影制片厂]]制作，首映是在[[中国中央电视台综合频道|中国中央电视台综合频道]]。<ref>{{cite web |url= http://news.xinhuanet.com/newscenter/2002-06/20/content_450186.htm|title=重大革命历史题材电视剧《中原突围》即将播出  |accessdate= 2002|publisher=新华网 |work= |language =zh-cn}}</ref>

该剧是主角是中原突围的领导人物李先念和王震。<ref name=" cat" >{{cite web |url=http://www.cnhubei.com/200605/ca1098358.htm |title=[影视点击]气势磅礴的史诗 电视剧《中原突围》热播 |accessdate=2006 |publisher=荆楚网 |work= |language =zh-cn}}</ref>
