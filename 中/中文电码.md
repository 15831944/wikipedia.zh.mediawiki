'''中文电码'''，又称'''中文商用电码'''、'''中文电报码'''或'''中文电报明码'''，原本是於[[电报|电报]]之中传送中文信息的方法。它是第一个把汉字化作电子讯号的编码表。

自[[摩尔斯电码|摩尔斯电码]]在1835年发明后，一直只能用来传送[[英语|英语]]或以[[拉丁字母|拉丁字母]]拼写的文字。1873年，法国驻华人员威基杰（S·A·Viguer）参照《[[康熙字典|康熙字典]]》的部首排列方法，挑选了常用汉字6800多个，编成了第一部汉字电码本《电报新书》。<ref>{{cite web |title=1868-2004年中国通信业发展之路 |url=http://tech.sina.com.cn/it/t/2004-05-20/1440364686.shtml |website=新浪网 |publisher=通信信息报 |date=2004-05-20 |accessdate=2018-12-01 |archive-url=https://web.archive.org/web/20181201062455/http://tech.sina.com.cn/it/t/2004-05-20/1440364686.shtml |archive-date=2018-12-01 |dead-url=no }}</ref><ref>{{cite web |title=《电报新书》 |url=http://museum.chinatelecom.com.cn/relics/file/list/index.html?i=1 |accessdate=2018-12-01 |archive-url=https://web.archive.org/web/20181201070007/http://museum.chinatelecom.com.cn/relics/file/list/index.html?i=1 |archive-date=2018-12-01 |dead-url=no }}</ref>后由任[[上海电报局|上海电报局]]首任总办的[[郑观应|郑观应]]将其改编成为《中国电报新编》。<ref>{{cite web |author1=徐祖哲 |author2=梁斌 |title=世界第一台电子式中文快速译码机的诞生 |url=http://dangshi.people.com.cn/GB/16461571.html |website=中国共产党新闻网 |publisher=人民政协报 |accessdate=2018-12-01 |date=2011-12-01 |archive-date=2012-01-30 |archive-url=https://web.archive.org/web/20120130033359/http://dangshi.people.com.cn/GB/16461571.html |dead-url=no }}</ref><ref>{{cite web |title=《中国电报新编》 |url=http://museum.chinatelecom.com.cn/relics/file/list/index.html?i=2 |website=中国电信博物馆 |accessdate=2018-12-01 |archive-date=2018-12-01 |archive-url=https://web.archive.org/web/20181201065501/http://museum.chinatelecom.com.cn/relics/file/list/index.html?i=2 |dead-url=no }}</ref>

中文电码表采用了四位阿拉伯数字作代号，从0001到9999按四位数顺序排列，用四位数字-{}-表示最多一万个汉字、字母和符号。汉字先按部首，后按笔划排列。字母和符号放到电码表的最尾。後來由於一萬個漢字不足以應付戶籍管理的要求，又有第二字面漢字的出現。在香港，兩個字面都採用同一編碼，由輸入員人手選擇字面；在台灣，第二字面的漢字會在開首補上“1”字，變成5個數字的編碼。

==应用==
中文电码可用作电脑裡的[[中文输入法|中文输入法]]，但因中文电码是“无理码”，记忆困难，一般用户几乎无法熟练地掌握使用。

在[[香港|港]][[澳門|澳]]，每个有中文姓名的市民的[[身份证|身份证]]上，均会在他的姓名下面，印有中文电码。在[[香港|香港]]，外國人取得的入港[[簽證|簽證]]亦有印上中文电码。在香港很多政府或商业机构的表格中，都会要求填写者填写他的中文电码，以便輸入電腦。
