'''中華民國與印度尼西亞關係'''是指[[中華民國|中華民國]]與[[印度尼西亞共和國|印度尼西亞共和國]]（通稱'''印尼'''）之間的關係。兩國無官方[[外交#外交上的承認|外交關係]]，但在對方[[首都|首都]]互設具政治、經濟、文化、[[簽證|簽證]]、僑務等功能的[[實質大使館|具大使館性質]]之代表機構，雙方民間商務交流亦屬密切。<ref name="外交年鑑"/>

===外交===
{{see also|中華民國外交|{{link-en|印度尼西亞外交|Foreign relations of Indonesia}}}}

1912年中華民國建立後，在[[荷屬東印度|時為荷蘭殖民地的印尼]]的巴達維亞、泗水等地設置領事館。

1930年，赴印尼華校任教的女教師蕭信庵，遭荷蘭船「渣華號」船員[[性侵|性侵]]的事件，引發反荷蘭的騷動，是為[[反荷運動|反荷運動]]。

1949年12月24日，特派[[吳鐵城|吳鐵城]]為中華民國慶賀[[荷蘭王國|荷蘭王國]]向[[印度尼西亞聯邦共和國|印尼聯邦]]移交主權及[[印度尼西亚|印尼共和國]]獨立典禮特使；派[[蔣家棟|蔣家棟]]為副使。<ref>{{Cite web |url = http://web.drnh.gov.tw/scripts/newsnote/tornado/searcher.exe?s=1&z=1&k=&m=0&p=&b=68161&v=root |title = 民國重要史事檢索系統 |publisher = [[國史館|國史館]]}}</ref>

1950年4月13日，印尼[[蘇卡諾|蘇卡諾]]政府與[[中華民國政府|中華民國政府]]斷交、與[[中華人民共和國|中華人民共和國]]建交。

1962年8月24日，印尼拒絕中華民國代表團參加於雅加達舉行的[[1962年亞洲運動會|亞洲運動會]]（另一被拒絕的是[[以色列|以色列]]）。

1967年10月27日，[[反共|反共]]的[[蘇哈托|蘇哈托]]政府與中華人民共和國斷交。

1971年10月25日，印尼對於[[中國與聯合國#代表權問題|中華民國是否續留聯合國]]的投票採棄權方式，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》。

1981年，[[行政院長|行政院長]][[孫運璿|孫運璿]]訪問印尼，並會晤[[蘇哈托|蘇哈托]]總統<ref>[http://www.sunyunsuan.org.tw/g1.asp 孫運璿年表], 財團法人[[孫運璿學術基金會|孫運璿學術基金會]]</ref>。

1994年2月春節期間，[[中華民國總統|總統]][[李登輝|李登輝]]訪問印尼，會晤總統蘇哈托與科技部長[[優素福·哈比比|哈比比]]等<ref>[http://www.president.gov.tw/Default.aspx?tabid=74 李總統登輝先生], [[中華民國總統府|中華民國總統府]]</ref><ref>[http://catalog.digitalarchives.tw/item/00/60/99/28.html 李登輝總統訪問印尼菲律賓與泰國], [[數位典藏|數位典藏]]與數位學習國家型科技計畫</ref>。11月，行政院經濟建設委員會主任委員[[蕭萬長|蕭萬長]]前往印尼雅加達，[[亞太經濟合作會議中華臺北領袖代表列表|代表]]總統李登輝參與[[APEC峰会|亞太經濟合作會議領導人非正式會議]]。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=FAEEE2F9798A98FD&sms=6DC19D8F09484C89&s=B5D03468BEF833CA |author = 公眾外交協調會 |date = 1994年12月1日 |title = 外交部錢部長於昨（三十）日邀請行政院經濟建設委員會蕭主任委員萬長 |publisher = 中華民國外交部}}</ref>

1998年1月，[[行政院長|行政院長]]蕭萬長訪問印尼。<ref>{{Cite web |url = https://www.president.gov.tw/NEWS/4485 |date = 1998年01月23日 |title = 總統下午聽取行院長蕭萬長訪問菲、印後的專題報告 |publisher = 中華民國總統府}}</ref>

1998年5月，在[[黑色五月暴動|印尼排華事件]]中，[[中華民國政府|中華民國政府]]多次緊急指派專機飛往印尼疏散受難者，另一方面[[中華民國外交部#歷任首長|外交部長]][[胡志強|胡志強]]於同年7月29日公開召見印尼駐華代表，當面向印尼提出了「最嚴正抗議」。

2002年8月，[[中華民國副總統|副總統]][[呂秀蓮|呂秀蓮]]訪問印尼<ref>[http://www.president.gov.tw/Default.aspx?tabid=131&itemid=1270 副總統結束印尼訪問行程返抵國門], 中華民國總統府</ref>。

2006年5月，總統[[陳水扁|陳水扁]]訪問印尼<ref>[http://www.libertytimes.com.tw/2006/new/may/12/today-fo1.htm 扁轉赴巴譚島 考察投資環境] {{webarchive|url=https://web.archive.org/web/20110223172548/http://www.libertytimes.com.tw/2006/new/may/12/today-fo1.htm |date=2011-02-23 }}, 《[[自由時報|自由時報]]》, 2006年5月12日</ref>。

2006年10月，印尼前總統[[優素福·哈比比|哈比比]]預定抵台出席「亞太商工總會40週年慶祝大會暨21屆大會」，但在中華人民共和國的施壓下取消行程。<ref>{{Cite web |url = https://www.mofa.gov.tw/News_Content.aspx?n=5127B62318146444&sms=22E3BC8A4FF53F8A&s=405D07ED3FEFA41A |author = 公眾外交協調會 |date = 2007年7月23日 |title = 中國大陸阻撓我國際空間事例（95年10月至12月份事例）|publisher = 中華民國外交部}}</ref>

2013年5月9日，在[[廣大興28號事件|廣大興28號事件]]中，雖然中華民國與印尼沒有正式的外交關係，但印尼官方支持[[中華民國政府|中華民國政府]]向[[國際法庭|國際法庭]]申訴。<ref>[http://www.ettoday.net/news/20130517/208911.htm 東森新聞：嗆菲律賓說謊！　印尼：臺灣應告上國際法庭]</ref>

===僑民===
{{see also|臺灣裔{{!}}海外臺僑|印度尼西亞華人}}
臺灣在印尼之[[僑民|僑民]]有20萬9,000人（2012年底）<ref>[http://www.ocac.gov.tw/public/public.asp?selno=9429&no=9429&level=C 海外臺灣僑民人數] {{webarchive|url=https://web.archive.org/web/20120916092810/http://www.ocac.gov.tw/public/public.asp?selno=9429&no=9429&level=C |date=2012-09-16 }}, [[僑務委員會|僑務委員會]]</ref>，知名人士包括[[白嘉莉|白嘉莉]]。印尼在臺灣的僑民則有18萬2,597人（2013年5月）<ref>[http://www.immigration.gov.tw/ct.asp?xItem=1210695&ctNode=29699&mp=1 〈外僑居留人數統計表〉], [[內政部入出國及移民署|內政部入出國及移民署]]</ref>。出身印尼而已[[歸化|歸化]]為[[中華民國國民|中華民國國民]]的知名人物包括：[[王宇婕|王宇婕]]與[[莫愛芳|莫愛芳]]等。

===跨國移工===
{{see also|外籍勞工#臺灣的外籍勞工{{!}}在臺灣的外籍勞工|外籍配偶 (臺灣){{!}}在臺灣的外籍配偶}}
在台灣工作的東南亞移工，印尼人是人數最多的，截至2017年12月，共有25萬8,048人；職業類別上大多數是屬於「醫療保健及社會工作服務業、其他服務業」。<ref name="106年外交年鑑">{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac2017/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B106%E5%B9%B4%E5%A4%96%E4%BA%A4%E5%B9%B4%E9%91%91.pdf |title = 《中華民國106年外交年鑑》〈第三章 對外關係〉 |publisher = 中華民國外交部}}</ref><ref>[http://statdb.cla.gov.tw/statis/stmain.jsp?sys=220&ym=10201&ymt=10206&kind=21&type=1&funid=q13018&cycle=41&outmode=0&&compmode=0&outkind=1&fldspc=0,3,28,2,&codspc0=0,7,&rdm=etiUbiq0 產業及社福外籍勞工人數－按行業及國籍分]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }},勞動統計, [[中華民國勞動部|行政院勞工委員會]]</ref>

2014年10月就任[[印尼總統|印尼總統]][[佐科威|佐科威]]表示他知道在臺印尼勞工獲得良好對待，他很感謝臺灣善待這些印尼勞工，印尼將繼續維持這種合作關係，並呼籲印尼勞工應遵守臺灣的法律並尊重臺灣習俗。他確信臺灣給予這些印尼勞工好的薪水，如果臺灣還需要更多的印尼勞工，他們也會推薦親友到臺灣工作，「這是我們（臺灣和印尼）關係的基礎」。佐科威也感謝多數在臺印尼勞工在[[2014年印度尼西亞總統選舉|總統大選]]中支持他，「我希望我們能夠在臺灣見面」。印尼國會副議長{{link-id|帕利約|Priyo Budi Santoso}}也表示，「臺灣善待印尼勞工的經驗足以成為中東及世界其他國家對待印尼勞工的最佳範例」。臺灣雇主與印尼幫傭互動融洽的真實案例也曾登上社會新聞，讓印尼官員深受感動。{{link-en|安塔拉 (新聞通訊社)|Antara (news agency)|印尼國家通訊社安塔拉}}社長{{link-id|賽佛|Saiful Hadi}}亦稱「臺灣對印傭好大家都知道」。<ref>{{citation |url=http://www.cna.com.tw/news/aopl/201412060123-1.aspx |title=台灣善待印傭 印尼總統掛保證 |archiveurl=https://web.archive.org/web/20160218110607/http://www.cna.com.tw/news/aopl/201412060123-1.aspx |date=2014-12-06 |archivedate=2016-02-18 |newspaper=《中央通訊社》 |deadurl=yes }}</ref><ref name="總統佐科威">{{citation|url=http://www.cna.com.tw/news/aopl/201410190050-1.aspx |title=重台印關係 佐科威對台有期待 |archiveurl=https://web.archive.org/web/20141219201536/http://www.cna.com.tw/news/aopl/201410190050-1.aspx |date=2014-10-19 |archivedate=2014-12-19 |newspaper=《中央通訊社》 |deadurl=yes }}</ref>

[[File:Visa_requirements_for_Indonesian_citizens.png|thumb]]必須申請[[簽證|簽證]]方可入境[[中華民國|中華民國]]。]]
{{see also|中華民國簽證政策|中華民國公民簽證要求}}
{{see also|印度尼西亞簽證政策|{{link-en|印度尼西亞公民簽證要求|Visa requirements for Indonesian citizens}}}}
持有[[中華民國護照|中華民國護照]]的[[中華民國國民|中華民國公民]]可以[[簽證#签证优惠待遇|免簽證]]的方式入境印尼，停留最多30天不得延長。[[簽證#落地签证|落地簽證]]停留30天可延長1次、商務簽證停留最多60天。經印尼過境轉機，也可以落地免簽。<ref>{{Cite web |url = https://www.boca.gov.tw/sp-foof-countrycp-03-12-c23b0-02-1.html |title = 簽證及入境須知 |publisher = [[外交部領事事務局|外交部領事事務局]]}}</ref><ref>{{Cite web |url = http://www.mofa.gov.tw/FreeVisa.aspx?n=2D82B2CE7F34FD09&sms=B75E93A3DAB23A99 |title = 免簽證資訊 |publisher = 中華民國外交部}}</ref>

持有{{link-en|印尼護照|Indonesian passport}}的[[印尼人口|印尼公民]]符合特定要求者，可透過[[中華民國政府|中華民國政府]]的[[中華民國簽證政策#東南亞七國網路申辦入境中華民國許可|東南亞七國網路申辦入臺許可]]申請免簽證，停留最多30天；抵台參加團客旅遊、國際會議、運動賽事、商展活動，則可以[[中華民國簽證政策#电子签证|電子簽證入境中華民國]]，停留最多30天並不得延期。<ref>{{Cite web |url = https://www.boca.gov.tw/cp-228-4488-8208a-1.html |title = 誰具備申請電子簽證的資格? |publisher = 外交部領事事務局}}</ref><ref>{{Cite web |url = https://www.boca.gov.tw/cp-215-643-5e740-1.html |title = 我的電子簽證效期多久？|publisher = 外交部領事事務局}}</ref>

[[中華民國外交部|中華民國外交部]]的[[國外旅遊警示分級表|國外旅遊警示分級表]]將印尼歸類為'''黃色警示'''：特別注意旅遊安全並檢討應否前往（2018年10月16日）；<ref>{{Cite web |url = https://www.boca.gov.tw/sp-trwa-content-80-15073-1.html |title = 國外旅遊警示分級表 |publisher = 外交部領事事務局}}</ref>[[中華民國衛生福利部|衛生福利部]]的國際間旅遊疫情建議等級表將印尼歸類為'''第一級注意'''：提醒遵守當地的一般預防措施（2018年10月30日）。<ref>{{Cite web |url = http://www.cdc.gov.tw/InternationalTravel/CountryInfo.aspx?treeid=aa2d4b06c27690e6&nowtreeid=fb6a405b1d67aa69&tid=510C4EEDEF6A4F26 |title = 國際重要疫情資訊 |publisher = [[中華民國衛生福利部|中華民國衛生福利部]]}}</ref>

==經濟==

===貿易===
[[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]（外貿協會）於[[印尼|印尼]]首都設立'''[[雅加達|雅加達]]台灣貿易中心'''。<ref>{{Cite web |url = http://taipei.taiwantrade.com.tw/flash/big/unitList.jsp?lang=zh_TW#ASIA |title = 外貿協會全球據點|publisher = [[中華民國對外貿易發展協會|中華民國對外貿易發展協會]]}}</ref>[[經濟部國際貿易局|經濟部國際貿易局]]也在雅加達設立'''駐印尼代表處經濟組'''。<ref>{{Cite web |url = http://www.trade.gov.tw/Pages/Detail.aspx?nodeID=214&pid=305862 |title = 本部駐外單位 |publisher = [[中華民國經濟部|中華民國經濟部]] [[經濟部國際貿易局|國際貿易局]]}}</ref>

國際貿易局將印尼列為重點拓銷市場，從2010年第2季開始委託外貿協會執行「[[台灣國際品牌|台灣國際品牌]]整合行銷傳播及產業廣宣專案」，以各種密集的B2C（business to customer）即[[零售|零售]]活動介紹臺灣的[[IT|IT]]產品。<ref name="主要產業概況">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 主要產業概況 |publisher = 中華民國對外貿易發展協會}}</ref>

2014年，印尼是[[宏碁|宏碁]]在[[東南亞|東南亞]]最大的市場，[[個人電腦|個人電腦]]銷售量過去6年居印尼之冠，2013年被[[華碩|華碩]]取代，市場第一、二名分別由華碩及宏碁佔據；[[明基電通|明基]]的[[投影機|投影機]]與[[友訊|友訊]]的無線[[路由器|路由器]]也是印尼第一品牌；BenQ電腦液晶螢幕居第3名，僅次於[[韓國|韓國]]的[[三星顯示|Samsung]]與[[LG|LG]]；[[自行車|自行車]]與零配件是印尼第3大進口來源國，但僅佔印尼該產業總進口額5%以下；[[塑膠|塑膠]]機械則佔印尼該產業進口額的40%。<ref name="主要產業概況"/>

印尼政府常用貿易救濟措施如反傾銷及防衛措施來[[貿易壁壘|限制進口產品]]，至2013年，共5種產品遭印尼政府採行反傾銷或防衛措施調查，產品涵蓋[[不鏽鋼|不鏽鋼]]及紡織[[原物料|原物料]]等。<ref name="與我國經貿關係">{{Cite web |url = http://www.taitraesource.com/default.asp |title = 與我國經貿關係 |publisher = 中華民國對外貿易發展協會}}</ref>

2017年11月，在印尼舉辨首屆臺印[[鋼鐵|鋼鐵]]對話會議。<ref name="106年外交年鑑"/>

2018年，印尼是中華民國第15大貿易夥伴、第10大[[進口|進口]]夥伴、第16大[[出口贸易|出口]]夥伴。出口至印尼的金額為33億3,113萬7,501[[美元|美元]]，年增4.314%；自印尼進口的金額為55億0,089萬1,046美元，年增12.290%。貿易呈現[[淨出口|入超]]（[[赤字|赤字]]）21億6,975萬3,545美元，年增27.225%。2017年，印尼是中華民國第14大貿易夥伴、第10大進口夥伴、第15大出口夥伴。<ref>{{Cite web|url =https://cus93.trade.gov.tw|title =中華民國進出口貿易統計|publisher =經濟部國際貿易局}}</ref>

===投資===
2017年，中華民國對印尼投資3.97億美元，為印尼第14大[[外商直接投資|外資來源國]]。在印尼的[[臺灣商人|台商]]及技術人員約2萬餘人，估計台商家數在2,000家以上。2001－2017年，在印尼總投資件數共1,946件，總投資金額為48.69億美元。臺商投資最多的行業為[[造紙業|造紙業]]、[[橡膠|橡]]塑膠業、運輸機械及[[非金屬礦產|非金屬礦]]業等；其他投資如[[家具|家具]]業、[[紡織|紡織]]業、[[鞋|鞋]]業、[[金屬|金屬]]製品業、[[貿易|貿易]][[服務業|服務業]]及[[農業|農業]]等。並在[[雅加達|雅加達]]、[[萬隆|萬隆]]、[[井裡汶|井裡汶]]、[[三寶瓏|三寶瓏]]、[[泗水|泗水]]、[[峇里島|峇里島]]、[[巴淡|巴淡]]、[[北苏门答腊省|蘇北]]、[[中爪哇|中爪哇]]等地設有「臺灣工商聯誼會」，為整合印尼各地台商會，於是成立「印尼台灣工商聯誼會總會」，2014年總會長由泗水台灣工商聯誼會理事長兼任。<ref name="106年外交年鑑"/><ref name="與我國經貿關係"/><ref>[http://twbusiness.nat.gov.tw/countryPage.do?id=11&country=ID 中華民國經濟部 臺商服務網：印尼-外商現況及投資機會]</ref>

除了傳統[[製造業|製造業]]外，臺商近幾年亦開始於印尼投資服務業，以六角國際為例，該公司於2011年至印尼開設販售[[珍珠奶茶|珍珠奶茶]]等的臺式茶飲店。<ref>[http://www.cna.com.tw/postwrite/Detail/77835.aspx#.UekAlI1gemA 《中央通訊社》：日出茶太看準伊斯蘭商機 賀印尼雅加達2店開幕]</ref>另外，[[中華民國政府|中華民國政府]]近幾年亦與當地業者合作，時常舉辦臺灣商品的展示會，並於2013年7月開設常駐型的展示中心：雅加達臺灣精品形象店。<ref>{{Cite web |url=http://www.cdnews.com.tw/cdnews_site/docDetail.jsp?coluid=112&docid=102387371 |title=中央日報：臺精品印尼展店　女星愛不釋手 |access-date=2016-02-21 |archive-url=https://web.archive.org/web/20160304100420/http://www.cdnews.com.tw/cdnews_site/docDetail.jsp?coluid=112&docid=102387371 |archive-date=2016-03-04 |dead-url=yes }}</ref>[[臺灣|臺灣]]最大、亦是[[全球|全球]]最大[[電子專業製造服務|電子代工]]企業[[富士康|鴻海富士康集團]]於近期評估在印尼設廠，以生產電子[[行動電話|通訊設備]]以及[[平板電腦|平板電腦]]等產品。<ref>[http://www.tzuchi.org.tw/index.php?option=com_content&view=article&id=11529:2013-04-17-03-58-30&catid=108:2009-02-23-07-34-57&Itemid=553 《中央通訊社》：印尼部長：8月洽簽鴻海投資案]{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>[[中國信託商業銀行|中國信託商業銀行]]在雅加達設有分行。<ref>{{Cite web |url = http://www.taitraesource.com/default.asp |title = 國家基本資料表 |publisher = 中華民國對外貿易發展協會}}</ref>

===農業合作===
自1996年起，兩國每年輪流主辦[[農業部|農業部]]部長級會議。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1998/section_3/page3-1-1.htm |title = 《中華民國87年外交年鑑》〈第三章 中外關係〉 |publisher = 中華民國外交部}}</ref>

兩國於2010年簽署《一鄉一特產合作協議》，透過臺灣駐印尼農業技術團進行推廣，在[[西爪哇省|西爪哇省]]、[[中爪哇省|中爪哇省]]、[[東爪哇省|東爪哇省]]以及[[峇里省|峇里省]]從事推廣事業，協助當地農民進行產業振興。<ref>[http://www.roc-taiwan.org/ID/ct.asp?xItem=142330&CtNode=4792&mp=291&xp1=%20 中華民國外交部 駐印尼臺北經濟貿易代表處：駐館與駐地關係] {{webarchive|url=https://web.archive.org/web/20160305010959/http://www.roc-taiwan.org/ID/ct.asp?xItem=142330&CtNode=4792&mp=291&xp1= |date=2016-03-05 }}</ref>

===佐科威與中華民國===
於2014年當選、就任[[印尼總統|印尼總統]]的[[佐科威|佐科威]]於從政前經營[[家具|家具]]公司，和[[中華民國|中華民國]]、[[新加坡|新加坡]]、[[南韓|南韓]]與[[日本|日本]]都有往來，常前往台灣洽商業務的他對台灣熟悉且友善。[[木匠|木匠]]之子佐科威表示他以前常對友人說，假日時如果找不到他，他就是在台灣處理生意。他盛讚台灣消費力高、是很好的家具市場。他至今仍記得[[台灣飲食|台灣的美食]]、人情和[[西門町|西門町]]的繁華景象，他還能憑記憶用精準的[[中華民國國語|國語]]唸出「西門町很熱鬧」的中文。佐科威表示，台灣的製造業、企業能力強，也有良好的管理制度和處理投資的優秀人才，[[基礎建設|基礎建設]]良好，印尼和台灣一定要繼續合作。他指出，2013年台灣投資印尼的金額為4億美元，這是龐大的金額，他期盼台灣對印尼的投資能持續增加，「我想邀請你們造訪印尼，來印尼投資」。時任[[雅加達省|雅加達省]]長的佐科威也是最終落實[[鴻海科技集團|鴻海科技集團]]（Foxconn）印尼投資案的關鍵人物。<ref name="總統佐科威"/><ref>{{citation|url=http://www.cna.com.tw/news/aopl/201407220590-1.aspx |title=佐科威勝選 台印尼經貿料升溫 |archiveurl=https://web.archive.org/web/20141120213242/http://www.cna.com.tw/news/aopl/201407220590-1.aspx |date=2014-07-22 |archivedate=2014-11-20 |newspaper=《中央通訊社》 |deadurl=yes }}</ref><ref>{{citation|url=http://www.cna.com.tw/news/firstnews/201408240010-1.aspx |title=佐科威：印尼台灣必須持續合作 |archiveurl=https://web.archive.org/web/20140826080434/http://www.cna.com.tw/news/firstnews/201408240010-1.aspx |date=2014-08-24 |archivedate=2014-08-26 |newspaper=《中央通訊社》 |deadurl=yes }}</ref>
