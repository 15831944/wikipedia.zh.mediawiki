'''中央军委政治工作部直属工作局'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[黄寺大街|黄寺大街]]丙8号<ref>{{Cite web |url = http://www.ccgp.gov.cn/cggg/dfgg/zbgg/201701/t20170103_7820212.htm |title = 中央军委政治工作部直工局军委政治工作部系统委托地方造价咨询公司结算审计信息库入围采购中标公告 |author =  |date = 2017-01-03 |publisher = 中国政府采购网 |language =  |accessdate =  |quote =  }}</ref>，是[[中央军委政治工作部|中央军委政治工作部]]下属局，负责该部直属工作。

==沿革==
1962年前后，[[中国人民解放军总政治部|中国人民解放军总政治部]]整编，增设中国人民解放军总政治部国防科学工作部、'''中国人民解放军总政治部直属政治部'''、'''中国人民解放军总政治部管理局'''。<ref name=xhw>[http://news.xinhuanet.com/mil_new/2007-11/24/content_7138315.htm 中国人民解放军总政治部，新华网，2007-11-24]</ref>1969年11月28日，直属政治部改为'''中国人民解放军总政治部直属政治处'''，归中国人民解放军总政治部办公室领导。1971年6月，扩编为'''中国人民解放军总政治部直属政治部'''。

[[中共十一届三中全会|中共十一届三中全会]]以后，总政治部根据中央军委的决定调整了机构，设有：组织部、干部部、宣传部、文化部、保卫部、联络部、群工部、军事检察院、军事法院、办公厅、直属政治部、管理局。<ref name=xhw/>

1985年，总政治部管理局和直属政治部合并为'''中国人民解放军总政治部直属工作部'''。<ref name=xhw/>

2016年改革前，中国人民解放军总政治部直属工作部下设有：综合局<ref>刘巍，做好内勤工作的几点体会，办公室业务2013(12):30-31</ref>、管理保障局<ref>伍鑫，总政直工部管理保障局党委 扎实开展“三互”活动 提升自我教育管理质量，军队政工理论研究2008年第02期</ref>、组织纪检局<ref>[http://news.takungpao.com/mainland/zgzq/2014-11/2811097_print.html 大公特稿：中央军委纪委机构单设可能性不断走高，大公网，2014-11-03]</ref>、干部局<ref>[http://hdjxb.bjhd.gov.cn/jxfw/inform/getInform.do?method=preEditInform&from=view&informId=3941 中共北京市海淀区军队离退休干部第七次代表大会胜利召开，海淀军休信息网，2015-05-29]</ref>、财务物资局<ref>[http://mil.news.sina.com.cn/2009-11-20/0654574502.html 总政直工部财务物资局探索军人保险基金规范化管理新路 倾心尽力保障军人安心服役，新浪，2009-11-20]</ref>、基建营房局<ref>[http://www.86gc.net/article/html/254838.html 北京总政机关军官公寓楼工程( 中国人民解放军总政治部直属工作部基建营房局)，亿美建筑资讯网，2015-07-22]</ref>、军需运输装备局<ref>[http://mil.news.sina.com.cn/2007-12-28/0613478683.html 解放军表彰新军服装备工作先进单位和个人，新浪，2007-12-28]</ref>、卫生局<ref>陈超，总政卫生系统举办心理服务骨干培训班，解放军预防医学杂志2011(6):I0004-I0004</ref>等机构，并负责中国人民解放军总政治部黄寺老干部服务管理局<ref>[http://www.js7tv.cn/news/201507_13959.html 《军营大舞台》公益行动走进总政黄寺老干部服务管理局，中国军视网，2015-07-04] {{webarchive|url=https://web.archive.org/web/20161009182353/http://www.js7tv.cn/news/201507_13959.html |date=2016-10-09 }}</ref>（下设：黄寺第一离职干部休养所<ref>[http://tech.gmw.cn/newspaper/2014-09/15/content_100862065.htm 总政黄寺第一离职干部休养所所长张澎—— 像对亲人一样对待老干部，光明网，2014-09-15]</ref>、黄寺第二离职干部休养所<ref>[http://www.chinanews.com/mil/2014/06-05/6245609.shtml 总政黄寺第二干休所：老干部满意就是最高标准，中国新闻网，2014-06-05]</ref>、魏公村离职干部休养所等等）<ref>[http://mil.news.sina.com.cn/2011-04-16/0400642808.html 总政车道沟干休所精心组织注重细节 加装电梯工程实现无障碍通，新浪，2011-04-16]</ref>、白石桥老干部服务管理局<ref>[http://military.cnr.cn/zgjq/20160701/t20160701_522556990.html 白石桥老干部服务管理局老干部学习党章纪实，央广军事，2016-07-01]</ref>、玉泉路老干部服务管理局<ref>[http://news.xinhuanet.com/politics/2012-12/25/c_124141727.htm 总政玉泉路老干部大学采访见闻，新华网，2012-12-25]</ref>、[[西直门宾馆|西直门宾馆]]等全部总政治部直属单位的行政管理工作。<ref>[http://military.cnr.cn/zgjq/201409/t20140917_516452835.html 总政直属单位群星锻造工程全面启动实施，央广军事，2014-09-17]</ref>

在[[深化国防和军队改革|深化国防和军队改革]]中，2016年1月，中国人民解放军总政治部撤销，成立[[中央军委政治工作部|中央军委政治工作部]]。随之成立'''中央军委政治工作部直属工作局'''。<ref name=xiao/>
