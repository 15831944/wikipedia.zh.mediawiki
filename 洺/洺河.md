'''洺河'''是[[子牙河|子牙河]]系[[滏阳河|滏阳河]]的一条主要支流，又称'''沙河'''，古称“'''浸水'''”、“'''洺水'''”，元代改称“洺河”<ref>《汉书.地理志》：魏郡武安有浸水。三国时易称洺水。洺水之目不知谁改，俗谓山之下，地名，故曰洺水。元代始称洺河。</ref>。发源于[[太行山|太行山]]东麓河北[[武安市|武安市]]西北部，流经[[永年|永年]]、[[鸡泽|鸡泽]]和邢台境内[[南和县|南和县]]，于[[任县|任县]]环水村入[[北澧河|北澧河]]。

== 河流源头 ==
洺河发源于[[河北|河北]][[武安市|武安市]]西北部[[太行山|太行山]]余脉，上游分[[南河|南河]]、[[北河|北河]]两支，以[[小摩天岭|小摩天岭]]为天然分水岭。北河上游有三道主川（门道川、常社川、白云川）汇入。南河上游称管陶川，因川中管陶村而得名。南河全长95.01公里，北河全长62.3公里。南北两支在武安永和村汇合后称洺河，自汇合处经永年区至[[鸡泽县|鸡泽县]]沙阳村北出邯郸境，境内长76.4公里，流域面积2488平方公里。

== 流域面积 ==
洺河流域地势西高东低，以[[京广铁路|京广铁路]]为界，铁路以西为山区和丘陵区，面积1810平方公里，占邯郸境内流域面积的73％，一般海拔500到1000米，最高武安市青崖寨海拔1898.7米；铁路以东为冲积平原，面积678平方公里，占邯郸境内流域面积的27％，海拔30到70米。据临洺关水文站资料，洺河多年平均径流量1.13亿立方米，平均含沙量3.72千克/立方米，年输沙量40万吨。

洺河自武安至永年[[临洺关|临洺关]]段，河道纵坡1/300到1/1000，河宽200至400米，深2到4米，过水能力约1500立方米/秒。临洺关以下河段有高2米右堤，河宽由200米逐渐缩至80米，槽深2一3米，平均纵坡1/1800，过水流量800一300立方米/秒。鸡泽县段经[[1977年|1977年]]整治后深度为2到2.5米，滩地宽约60米，堤高2米，平均纵坡1/3000，引洪能力约200立方米/秒。因洺河最大引洪能力为1500立方米/秒，并且上宽下窄，到鸡泽段河道行洪能力仅200立方米/秒，所以如果一旦遇洪水就会泛滥成灾。由于年降雨量大部分集中在每年夏天汛期，其他季节，特别是冬春缺雨少雪时节，流域内也经常有过干旱的威胁。根据文献记载，500多年中，较大旱灾就有49次，几乎平均每10年多一次<ref>2015年版《邯郸市志》。</ref>。

== 蓄洪发电 ==
[[1949年|1949年]]后，在南北河上建成[[青塔水库|青塔]]、[[车谷水库|车谷]]、[[口上水库|口上]]、[[四里岩水库|四里岩]]、[[大远水库|大远]]等5座中型水库，总库容为12721万立方米，小（一）型水库8座，总库容2919万立方米，小（二）型水库43座，总库容1172万立方米。兴建和改建了口上、车谷、青塔、贾庄等4处万亩以上灌区，控制灌溉面积37.1万亩。还建成口上、活水、四里岩、车谷等4处小型水电站，装机容量达4187千瓦，年发电量达595.4万千瓦时。[[2005年|2005年]]，武安市开始对洺河进行河道综合治理，使河道防洪标准由原来的不足五5年一遇，提高至20年一遇。[[枣尖渠|枣尖渠]]是[[武安|武安]]历史上一条人工开凿的古老的引河水灌溉农田的渠道。历经明、清、民国时期，灌溉面积曾一度达4000余亩，至近代成为武安灌区一个渠系的组成部分。

== 河患之祸 ==
根据[[光绪|光绪]]年间《永年县志》记载，历史上洺河在[[永年|永年]]、[[鸡泽|鸡泽]]曾多次溢堤泛滥，决口改道，改道于[[滏阳河|滏阳河]]以北，[[小沙河|小沙河]]以南、永年中部地区留下许多故道。洺河在历史上曾有过几次大的变迁，洺河在[[唐|唐]]、[[宋|宋]]、[[金|金]]三代，曾从永年裴坡庄附近向东南流，经北李固、苏固、赵固、北碾头、周村、贾村一带，到西大慈村转向东流，大致是沿着现在[[滏阳河|滏阳河]]的流向流入[[曲周县|曲周县]]境<ref>参考《华北平原北部历史时期河道变迁图》、《河北平原黑龙港地区古河道图》等资料。</ref>。民国二十六年（[[1937年|1937年]]），洺河在永年席庄村西北决口，改向东北流，经席庄、杨庄、豆庄村北，至韩庄村北入[[鸡泽县|鸡泽县]]，形成今河道。原经席庄、豆庄村南向东，至辛庄堡入[[鸡泽县|鸡泽县]]的河道湮废为故道。是洺河最年轻的一条故道。
