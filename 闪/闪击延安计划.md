'''闪击延安计划'''是指1943年6月间，[[蒋介石|蒋中正]]命令第八战区副司令长官[[胡宗南|胡宗南]]率领50万军队突袭[[延安|延安]]的军事计划。后因[[熊向晖|熊向晖]]向中国共产党泄密，中共中央通知驻重庆的[[董必武|董必武]]向外界公开蒋的命令，作战计划最终作罢<ref name="a1">{{Cite web |url=http://news.ifeng.com/history/phtv/dsy/201001/0125_5712_1524513.shtml |title=胡宗南闪击延安前五天 中共窃得情报挽救危局 |accessdate=2018-07-13 |author= |date=2010-01-25 |work= |publisher=凤凰网|language=Zh-hans }}</ref>。

== 背景 ==

=== 国际共运 ===

=== 第三届第二次国民参政会大会通过决议 ===

== 过程 ==

此项军事计划由蒋介石审定，第八战区司令长官[[朱绍良|朱绍良]]向胡宗南下达。于是，胡宗南召集国军第九十军、暂编第二十五师、新编第三十七师师长以上人员于[[洛川县|洛川县]]开会。6月18日，胡宗南在洛川军事会议上进行军事部署，预计于7月9日闪击延安，攻占[[陕甘宁边区|陕甘宁边区]]<ref name="a2">{{Cite web |url=http://www.mm111.net/mmwb/data/20111128/html/19/content_1.html |title=胡宗南闪击延安遭遇空城计 |accessdate=2018-07-13 |author= |date=2011-11-28 |work= |publisher=茂名晚报电子版|language=Zh-hans }}</ref>。

此项作战计划不久被中共得知，多年后得知泄密者是胡宗南亲信[[熊向晖|熊向晖]]<ref name="a1"/>。当时，中共方面驻守延安及周边地区仅三、四万人的部队。于是，中共中央决定公布己方获得的国军进攻陕甘宁和延安的电报、书面命令，以阻击国军的军事行动。7月4日，[[朱德|朱德]]致电胡宗南，表明已知胡宗南部队“已将河防大军向西调动，弹粮运输络绎于途，内战危机，有一触即发之势”，同时警告“当此抗战艰虞之际，力谋团结，犹恐不及，若遂发动内战，必致兵连祸结，破坏抗战团结之大业，而使日寇坐收渔利 。”7月5日，朱德、[[萧劲光|萧劲光]]分别致电蒋介石、[[何应钦|何应钦]]、[[徐永昌|徐永昌]]和陕甘宁边区周围驻军将领，呼吁团结，避免内战<ref name="a2"/>。同时，在重庆的董必武将朱德电文交送美、英、苏等国驻华大使馆及新闻界<ref name="a1"/>。

7月6日，胡宗南致电蒋介石，建议“停止行动”<ref name="a2"/>。7日，蒋介石复电胡宗南同意罢兵<ref name="a1"/>。当日，胡宗南部在前线已开始[[炮击关中|炮击关中]]。
