{{作品名稱|name=''The Shining''|hk=閃靈|tw=鬼店|cn=hk}}是一部发行于1980年的英美合拍[[恐怖片|恐怖电影]]，根据[[斯蒂芬·金|斯蒂芬·金]]的1977年[[闪灵_(小说)|同名小说]]改编，导演是[[斯坦利·库布里克|斯坦利·库布里克]]<ref>{{Cite web |url = http://www.tcm.com/tcmdb/title/89937/Shining-The/ |title = The Shining |publisher = Turner Classic Movies |language = en}}</ref>。[[杰克·尼科尔森|杰克·尼科尔森]]在片中饰演作家{{link-en|杰克·托伦斯|Jack Torrance}}，{{link-en|雪莉·杜瓦|Shelley Duvall}}饰演妻子温蒂，{{link-en|丹尼·洛依德|Danny Lloyd}}饰演他们的儿子{{link-en|丹尼·托倫斯|Danny Torrance}}。本片有不少部分和原著不同。

美國導演[[馬田·史高西斯|馬田·史高西斯]]把本片列為史上最恐怖的11部恐怖電影之一。<ref>{{Cite web |url = https://www.thedailybeast.com/martin-scorseses-scariest-movies-of-all-time |title = Martin Scorsese's Scariest Movies of All Time |author = Martin Scorsese |date = 2015-10-31 |publisher = The Daily Beast |language = en}}</ref>

== 故事概要 ==
{{link-en|杰克·托伦斯|Jack Torrance}}是一名作家，为了摆脱工作上的失意，接受了一份在冬天维护[[科罗拉多州|科罗拉多州]]一家奢华的山间饭店的工作。他年幼的儿子拥有[[通灵|通灵]]能力，能够看到过去和未来的东西，如栖息在[[酒店|酒店]]的[[鬼魂|鬼魂]]。到酒店安顿好后不久，这家人就被暴风雪困住了，杰克逐渐受到一个[[超自然|超自然]]的存在的影响；他陷入疯狂并企图谋杀他的妻子和儿子。
