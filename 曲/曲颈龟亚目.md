'''曲頸龜亞目'''（[[学名|学名]]：{{lang|la|Cryptodira}}）是[[龟鳖目|龟鳖目]]下的一个亚目，又称'''潜颈龟亞目'''、'''隐颈龟亞目'''，分布广泛，陆地、淡水和海洋均有分布，包含了大多数龟鳖类，其中现存的物种大都生活在淡水中。曲頸龜亞目与[[侧颈龟亚目|侧颈龟亚目]]的不同之处在于，曲頸龜亞目回缩头部时是S形弯折颈部後直接缩回壳内，而侧颈龟亚目则是将颈部侧向折回壳内。

== 形态 ==
曲颈龟亚目生物的体型相差巨大，体型最大的[[棱皮龟|棱皮龟]]甲长可达256.5厘米，是龟鳖目中体型最大的物种，而体型最小的[[斑点鹰嘴珍陆龟|斑点鹰嘴珍陆龟]]（''Homopus signatus''）的最大甲长只有9.8厘米。

在身体结构方面，有的种类在背甲和腹甲结合处有下缘盾，[[大鳄龟|大鳄龟]]（''Macroclemys temminckii''）在肋盾和缘盾之间还有上缘盾。除[[海龟科|海龟科]]中某些物种外，本亚目物种腹甲与喉盾之间不存在间喉盾。本亚目物种的间喉盾、上缘盾和下缘盾在龟鳖目内均为最原始的形态<ref>{{ja}}{{cite book |author=安川雄一郎 |title=HER・PET・OLOGY Vol．3《水棲ガメの世界》 |pages=18－21 |year=2005 |publisher=誠文堂新光社 |location=東京 |isbn=978-4416705438 }}</ref>。

在食性方面，本亚目内包含[[肉食|肉食]]、[[草食|草食]]和[[杂食|杂食]]性物种。

== 分类和演化 ==
曲頸龜亞目的[[演化|演化]]历程主要在[[侏罗纪|侏罗纪]]时期进行，到侏罗纪末期时几乎已完全取代了侧颈龟亚目在河水和湖泊中的地位，这时陆生物种开始发展。 

曲頸龜亞目下共现存有3个[[总科|总科]]，分别是[[海龟总科|海龟总科]]（Chelonioidea）、[[陆龟总科|陆龟总科]]（Testudinoidea）和[[鳖总科|鳖总科]]（Trionychoidea）。過去曾有“[[动胸龟总科|动胸龟总科]]”（Kinosternoidea），现在被认为是最原始鳖总科的[[並系群|並系]]集合，因为它们不能构成一个自然类群<ref name=haaramo2008>{{cite web|url=http://www.helsinki.fi/~mhaaramo/metazoa/deuterostoma/chordata/reptilia/parareptilia/cryptodira.html |author={{aut|Mikko Haaramo}} |title=Cryptodira |accessdate=2008-05-07 |date=008-03-11 |publisher=Mikko's Phylogeny Archive |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090214071814/http://www.helsinki.fi/~mhaaramo/metazoa/deuterostoma/chordata/reptilia/parareptilia/cryptodira.html |archivedate=2009-02-14 }}</ref>。

目前对于曲頸龜亞目的界定普遍有两种说法，其一：曲頸龜亞目包含[[真曲颈龟下目|真曲颈龟下目]]，以及數個从[[化石|化石]]得知的若干原始的已灭绝[[演化支|演化支]]。這個演化支範圍是由一些非常早分化出来的基礎类群、以及Centrocryptodira演化支（包含现存曲頸龜亞目，與現存物種的史前近亲，也包含[[近代曲颈龟类|近代曲颈龟类]]）组成<ref name = haaramo2008 />。

另一个说法将“曲頸龜亞目”一词限定为{{link-en|冠群|crown clade}}演化支，範圍相當於近代曲颈龟类（Polycryptodira），按照这种版本的分類法，这时的曲頸龜亞目应被称作'''Cryptodiramorpha'''。依照這個版本，侧颈龟亚目和曲頸龜亞目就不再是[[支序分類學|姐妹分类单元]]了<ref name = haaramo2008 />。
