'''渝北区'''是[[重慶市|重慶市]]的市辖区，原为[[江北县|江北县]]，南端比邻[[江北区_(重庆市)|江北区]]，北端与[[四川省|四川省]][[华蓥市|华蓥市]]接壤，区人民政府驻[[双凤桥街道|双凤桥街道]]。渝北区幅员面积1452平方公里，其中耕地4.13万公顷，绿地2万公顷，2009年常住人口97.62万人。渝北区依华蓥，挟两江，素有山城北大门之称。渝北区是全国文明城区和国家级卫生城区，亦是国内驰名的山水园林城区和优秀旅游城区，是全国首批命名的“武术之乡”。
== 歷史沿革<ref>重慶市渝北區地方志編纂委員會.《江北縣誌》.重慶出版社,1996年12月第1版.第67頁</ref> ==
=== 江北廳 ===
清初為巴縣江北鎮。乾隆十九年（1754年）置江北廳隸[[重慶府|重慶府]]，廳治設江北城（今重慶市江北區江北嘴江北舊城）。乾隆二十四年擴大廳域，將巴縣仁里上六甲、義里十甲、禮里十甲劃歸江北廳，隸屬四川省[[川東道|川東道]]重慶府。中華民國元年（1912年）3月，袁世凱就任臨時大總統，以府州廳縣直隸省政。民國二年廢府廳州，統改為縣，以道統縣，江北廳更名江北縣，隸四川省川東道。

=== 江北縣 ===
* 民國24年6月全川劃為18個行政督察區，[[江北縣|江北縣]]隸於第三行政督察區，專員公署駐永川縣永昌鎮。
* 1949年11月30日，屬[[巴縣專區|巴縣專區]]轄，12月20日專署移駐璧山縣，更名[[璧山專區|璧山專區]]，仍轄江北縣。
* 1951年4月20日，專署移駐江津縣，更名[[江津專區|江津專區]]，仍轄江北縣，駐水土坨。
* 1952年，劃歸重慶直轄市<ref>四川省的北碚市及綦江、長壽、江北3縣劃歸重慶市直轄。1952年9月27日西南軍政委員會【會民令字第0033號】命令，1953年1月16日內政部【內民字第32號】批復</ref>。
* 1953年，江北縣劃歸四川省[[江津專區|江津專區]]<ref>重慶市的巴縣、綦江縣、江北縣劃歸四川省江津專區；長壽縣劃歸四川省涪陵專區。1953年1月16日內務部【內民字第32號】批准</ref>。
* 1965年11月19日，江北縣人民委員會駐地由水土鎮遷至兩路公社<ref>江北縣人民委員會駐地由水土鎮遷至兩路公社。1965年11月19日四川省人民委員會批准</ref>。
* 1977年9月24日，[[江津地區|江津地區]]的江北縣劃歸重慶市<ref>★1977年9月24日國務院【國發104號】批復：江津地區的江北縣劃歸重慶市。</ref>。
* 1994年12月17日，撤銷江北縣，設立渝北區<ref>1994年12月17日國務院【國函138號】批復：同意撤銷江北縣、巴縣，設立渝北區、巴南區。</ref>。

*街道：龙溪街道、龙山街道、龙塔街道、双凤桥街道、两路街道、王家街道、双龙湖街道、回兴街道、悦来街道、鸳鸯街道、人和街道、天宫殿街道、翠云街道、大竹林街道、康美街道；
*镇：石船镇、洛碛镇、龙兴镇、大塆镇、古路镇、统景镇、大盛镇、木耳镇、兴隆镇、茨竹镇、玉峰山镇、礼嘉镇。

其中，鸳鸯街道、人和街道、天宫殿街道、翠云街道、大竹林街道、礼嘉镇由重庆市[[北部新区|北部新区]]代管。
=== 縣域沿革<ref>重慶市渝北區地方志編纂委員會.《江北縣誌》.重慶出版社,1996年12月第1版.第74-75頁</ref> ===
* 民國十年（1921年），江北城內外7廂(城內弋陽、石樑、寶蓋、金沙、落伽5雁；城外鶯花、上關2廂)，及附近密集居民區，劃歸重慶商務督辦公署管轄。
* 民國二十二年，沿嘉陵江、長江岸上自香國寺起，經沙灣、陳家館、劉家台、簡家台、廖家台、江北城至同德堂、溉瀾溪劃歸重慶市區，面積約15平方公里。
* 民國二十五年4月1日，北碚成立嘉陵江三峽鄉村建設實驗區，江北縣西部南段二岩、東陽鎮、黃葛、文星4鄉鎮劃歸北碚，但國稅、省稅、地方稅的征解催收，仍由江北縣巡督各鄉辦理；縣立教育機關，亦未經明令划出，仍由縣政府管理，北碚實驗區負監督之責，民國30年始正式劃界。
* 民國二十九～三十年，先後將石馬河、貓兒石、觀音橋、寸灘及恆興鎮（唐家沱）2個保，人和鎮4個保，回龍鎮8個保，石馬鄉8個保，龍溪鄉14個保劃歸重慶市。
* 1950年3月19日，江北縣唐家沱鄉鐵山村和郭家沱魚嘴鄉第二十一村劃歸重慶市<ref>1950年3月19日川東行署【署民行字62號】批復</ref>。
* 1950年4月18日，川東區行政公署同意，將江北縣所屬的郭家沱自藥鋪經十字埡口、牌坊、梅子嵐埡至賈家溝一帶西南地區劃歸重慶市第六區管轄。
* 1951年3月1日，川東行署、重慶市人民政府聯合發出命令，將江北縣唐家沱、郭家沱均劃歸重慶市。3月16日舉行交接會議，對界線作出決議：唐家沱建設村屬江北縣，鐵山村歸重慶市；郭家沱、魚嘴鄉第二十一村全部劃入重慶市<ref>1951年6月2日內務部【年民字第74號】批准：4.江北縣郭家沱地區、唐家沱地區分別劃歸重慶市第六區、第七區。</ref>。
* 1951年7月23日，江北縣西部第七（八？）區天府礦區附近大田坎一地劃歸[[北碚市|北碚市]]<ref>江北縣第八區大田坎一地劃歸北碚市。1951年7月23日川東行署民政廳【民行字第138號】批復</ref>。
* 1951年12月7日，江北縣紅土地鄉居民劃歸重慶市<ref>1951年12月7日川東行署民政廳【民政字第304號】批復</ref>。
* 1952年10月，大石、人和、龍溪3個鄉劃歸重慶市第二區（今[[江北区_(重庆市)|江北區]]）。1953年5月，原屬江北縣第一區的大石、人和、龍溪3個鄉，由重慶市劃還江北縣，但留有龍溪鄉東南部嘉樂溪（今龍塔村南）地片未劃還。
* 1953年2月2日，四川省人民政府決定：將[[鄰水縣|鄰水縣]][[高興鄉|高興鄉]]第五村（大面村）劃歸江北縣茨竹鄉和中興鄉各管一部分<ref>鄰水縣第四區[[高興鄉|高興鄉]]第五村（大面村）全部劃歸江北縣第十三區茨竹、中興2兩分管。1953年2月2日四川省人民政府【府民字第0207號】批復</ref>；6月6日，省府批覆：江北縣洛磧鎮杜家村（第五村）劃歸長壽縣沙溪鄉管轄<ref>江北縣第五區洛磧鎮杜家村劃歸長壽縣。1953年7月21日四川省人民政府【府民字第0957號】批復</ref>；11月10日省府批准：將長壽縣稱沱鄉第八村四、五、六、七4個組及九村一組，永興鄉第七村的八、九組，分別劃歸江北縣大壪鄉、東山鄉、明月鄉管轄<ref>1953年11月9日四川省人民政府【府民字第1192號】批復：2．長壽縣稱沱鄉八村的四、五、六、七組及九村一組，永興鄉的七、八、九組劃歸江北縣。</ref>。

== 区域经济 ==
渝北区历史上为重庆的两个母县之一的江北县，随着重庆的快速发展和城市扩张，1991年改为渝北区。渝北区南部的部分区域，如鸳鸯街道等逐渐成为[[重庆主城区|重庆主城区]]9区组成部分之一，但渝北区原驻地两路镇距离主城区较远，依靠着[[重庆江北国际机场|重庆江北国际机场]]和[[长安集团|长安集团]]两个巨大的消费群体，一直保持着类似县城一样相对独立的经济环境，走上了自成体系的产业发展道路。

渝北区自2001年以来经济发展迅速。在设立北部新区早期的2003年，全年GDP67.52亿元。<ref>[http://yb.cq.gov.cn/Item/73.aspx 重庆市渝北区人民政府2004年政府工作报告]</ref>此后，渝北区掀起了大规模发展工业的大潮，2010年渝北区GDP达573.8亿元，在短短六年时间内，经济产值翻了近7倍。2011年突破670亿元人民币大关。<ref>[http://yb.cq.gov.cn/Item/8414.aspx 重庆市渝北区人民政府2010年政府工作报告]</ref>渝北区在2004年基本实现[[小康|小康]]，并提出2014年进入全民社会保障统筹的全面小康目标。2010年，中国国务院批复设立中国内陆唯一的国家级开发开放新区[[两江新区|两江新区]]，渝北区是其重要的组成部分。

== 自然与旅游资源 ==
渝北区不仅工业发达，自然和旅游资源也很丰富，名山胜水、古迹园林、民俗风情汇萃而成独具魅力的旅游胜景。东部有盛名远播的“武陵仙境”统景风景区，是重庆“十佳”风景名胜之一，以“统景峡猿”居古巴渝十二景之首，统揽山、水、林、泉、峡、洞、瀑，被誉为“自然博物馆”，同时也是[[中国西南地区|西南]]最大的亚健康温泉理疗保健中心。邻近的张关大型溶洞群体，星罗棋布，其河恋河2000米的地下阴河如海底潜龙，与御临洞相连而形成全长5000多米的地下宫殿。白岩乡境内的天险洞内起伏跌宕、钟乳丛生，附近有茂密的原始森林和[[白莲教|白莲教]]遗址、悬棺、栈道等古迹；排花洞又称“水晶龙宫”，以飞珠溅玉的排花飞瀑幽深怪异的钟乳洞天而著称。北部的两岔湖尤如水上曲廊，绮丽如画。西部嘉陵江畔有南宋抗元名城——多功城，与合川[[钓鱼城|钓鱼城]]为姊妹名城。城区附近有花果如山的鹿山观光农业示范区、玉峰山森林公园及“农家乐”旅游热点。城区有展示[[川东|川东]]婚、丧、嫁、娶系列民俗的巴渝民俗文化村，城南有休闲旅游的碧津公园、双龙湖公园、宝圣湖和九龙湖、金龟岛度假村。

== 人口 ==
2009年末，渝北区常住人口97.62万人，其中城镇人口67.43万人，城镇化率69.1%；户籍人口98.5万人，其中非农业人口48.39万人，农业人口50.08万人。<ref>[http://www.cqybq.gov.cn/news/201003/news_20100311144740.html 2009年渝北区国民经济和社会发展统计公报]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 交通 ==
渝北区交通便捷。重慶市域的两个民用機場之一[[江北国际机场|江北国际机场]]座落于此。[[渝怀铁路|渝怀铁路]]横贯全区，有客运一等站[[重慶北站|重慶北站]]。国道210线与319线在渝北区交汇，内环高速G5001，绕城高速G65，外环绕城G75，机场高速G210，渝邻高速G319，渝宜高速，渝武高速经过渝北区。区政府所在地距重庆朝天门码头和铁路[[重庆站|重庆站]]20公里，有公交快车608路直达往返，到各旅游景点均有旅游专线车接送。区内通讯发达，全境于2007年早于全國近2年首先开通国内外程控[[3G|3G]]直拨无线电话。旅行社、宾馆、娱乐场所、购物商场配套，形成吃、住、行、游、购、娱“一条龙”优质服务。

== 工业 ==
渝北区工业非常发达，第二产业总产值占全区经济产值60%以上。坐落有世界500强企业91家，其中投资最大的即为[[长安汽车|长安汽车]]与[[福特|福特]]合资组成的长安[[福特|福特]][[马自达|马自达]]汽车有限公司，总投资超过80亿美元。除此之外，渝北区投资超过100亿人民币的企业还有[[富士康|富士康科技集团]]、仁宝电脑、和舰科技、[[惠普|惠普]]、[[力帆|力帆工业]]、[[三星电子|三星电子]]、[[ABB|ABB]]、廣達電腦、[[中興通訊|中興通訊]]、[[华为通信|华为通信]]、宗申控股、隆鑫集团、嘉陵集团等。
