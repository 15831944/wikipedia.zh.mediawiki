《'''凶间雪山'''》（{{lang-en|'''The Demon in the Mountain''' 或 '''Bleeding Mountain'''}}）2012年上映的中国大陆惊悚片和恐怖片，改编自网络小说《相信谁》，导演[[周耀武|周耀武]]、周文涛、王俊翔，主演[[吴卓羲|吴卓羲]]、邓丽欣、TAE、孙祖杨、邓紫衣。<ref>{{cite web |url= http://lady.163.com/12/1224/11/8JG4QOKF00264M4F.html|title=《凶间雪山》上映首周表现不俗 放长线向元旦开战 |accessdate= 2012-12-24 |publisher= 网易|work= |language =zh-cn}}</ref>

== 剧情 ==
该片讲述5人组成的“飞鸟登山队”在[[贡嘎山|贡嘎山]]遭遇到的神秘故事。<ref>{{cite web |url=http://news.ifeng.com/gundong/detail_2013_01/11/21071472_0.shtml |title=《凶间雪山》 |accessdate=2013年1月11日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>
