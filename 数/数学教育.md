'''数学教育'''是研究[[数学|数学]]教学的实践和方法的学科。而且，数学教育工作者也关注促进这种实践的工具及其研究的发展。数学教育是现代社会激烈争论的主题之一。这个术语有个歧义，它既指各地的教室里的实践，也指新生的一个学科，它有自己的期刊，会议，等等。这方面最重要的国际组织是{{le|数学教育国际委员会|International Commission on Mathematical Instruction}}。<ref>{{cite web |url=http://www.kurims.kyoto-u.ac.jp/IMU/ICMI/ |title=存档副本 |accessdate=2005-07-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20050503002601/http://www.kurims.kyoto-u.ac.jp/IMU/ICMI/ |archivedate=2005-05-03 }}</ref>

== 历史沿革 ==
基础数学是多数古文明的教育系统的一部分，包括[[古希腊|古希腊]]，[[罗马帝国|罗马帝国]]，[[吠陀文明|吠陀社会]]和[[古埃及|古埃及]]。在多数情况下，只有足够高地位，财富或等级的[[男性|男性]]孩童才能接受正规教育。

在[[柏拉图|柏拉图]]把文科分成[[三学科|三学科]]和[[四学科|四学科]]的划分中，四学科包括数学的[[算术|算术]]和[[几何|几何]]领域。这个结构在[[中世纪|中世纪]]欧洲所发展的[[经典教育|经典教育]]的体系得到了延续。几何的教育基于[[欧几里得|欧几里得]]的''原本''。商业的学徒，如石匠，商人和借贷者需要学习和他们的行业相关的这种实用数学。

第一本英语的数学教科书由[[Robert_Recorde|Robert Recorde]]出版，从1540年的''艺术的基础''(''The Grounde of Artes'')开始。

在[[文艺复兴|文艺复兴]]时期，数学的学术地位下降了，因为它和手工业和贸易紧密相关。虽然在欧洲的大学里继续教授数学，它被视为[[自然哲学|自然哲学]]，[[形而上学|形而上学]]和[[道德哲学|道德哲学]]的辅助。

这个趋势在[[十七世纪|十七世纪]]得到某种逆转，[[阿伯丁大学|阿伯丁大学]]在1613年建立数学主席职位，随后有[[牛津大学|牛津大学]]在1619年建立几何主席职位和[[剑桥大学|剑桥大学]]在1662年设立的[[卢卡逊教授|卢卡逊教授]]。但是，数学一般不在大学之外教授。例如[[艾萨克·牛顿|牛顿]]在他在1661年进入[[剑桥三一学院|剑桥三一学院]]之前没有受过正规数学教育。

在[[十八世纪|十八世纪]]和[[十九世纪|十九世纪]]，[[工业革命|工业革命]]导致[[城市|城市]]人口大量增加。基本的数字技能，如描述时间，数钱和简单[[算术|算术]]，称为新的城市生活的基本能力。在新的[[公共教育|公共教育]]系统中，数学成了从幼年开始的课程的中心部分。

到[[二十世纪|二十世纪]]，数学成了所有[[发达国家|发达国家]]的核心课程的一部分。但是，多样和变化着的关于数学教育的目的的思想导致所采用的内容和方法几乎没有任何整体上的一致性。

== 目的 ==
在不同的时期在不同的文化和国家中，数学教育试图达到不同的目标。这些目标包括:
* 教授给所有学生的[[数字|数字]]技巧。
* 教授给大部分学生的实用数学（[[算术|算术]]，[[基础代数|基础代数]]，平面和立体[[几何|几何]]，[[三角学|三角学]]），使得他们有能力从事贸易或手工业。
* 早期的抽象代数概念教育（例如[[集合_(数学)|集合]]和[[函数|函数]]）
* 选择性的数学领域的教育（例如[[欧式几何|欧式几何]]）作为[[公理化体系|公理化体系]]的实例和[[演绎推理|演绎推理]]的一个模型
* 选择性的数学领域的教育（例如[[微积分|微积分]]）作为现代社会的智力成就的一个实例
* 教授给希望以[[科学|科学]]为职业的学生的高等数学

数学教育的方式和变化的目标一致。

== 标准 ==
绝大部分的历史时期，数学教育的标准是地域性的，由不同的学校或教师根据学生的水平和兴趣来设置。

在现代，有一种趋势是建立地区或国家标准，通常隶属于更广泛的学校教学大纲。例如在[[英国|英国]]，数学教育的标准是[[英国国家教育大纲|英国国家教育大纲]]的一部分[https://web.archive.org/web/20130424131549/http://www.nc.uk.net/webdav/servlet/XRM?Page%2F%40id=6004&Subject%2F%40id=22]。在[[美国|美国]]，[[美国数学教师国家委员会|美国数学教师国家委员会]][http://nctm.org]制定了一系列文档，最近的有[[学校数学的原则和标准|学校数学的原则和标准]][http://standards.nctm.org/]，为学校数学的总体目标达成了一致。更具体的教学标准一般在州一级制定 - 譬如在[[加利福尼亚|加利福尼亚]]，加州教育理事会为数学教育制定了标准[https://web.archive.org/web/20051228115904/http://www.cde.ca.gov/be/st/ss/mthmain.asp].

== 水平 ==
不同水准的数学教授给不同年龄的学生。一个大致的对算术和代数的子课题的教学年龄的参考如下:
* 加法：5-7岁;更多的位数8-9岁
* 减法：5-7岁;更多的位数8-9岁
* 乘法：7-8岁;更多的位数9-10岁
* 除法：8岁;更多的位数9-10岁
* 简单代数：11-12岁
* 代数：13岁以上

== 方法 ==
任何特定环境下的方法很大程度上由相关的教育系统所设定的目标所决定。教授数学的方法包括:
* '''经典教育''' - [[中世纪|中世纪]]的经典教育大纲中的数学教育通常基于[[欧几里得|欧几里得]]的《[[幾何原本|幾何原本]]》，它被作为[[演绎推理|演绎推理]]的[[范式|范式]]来教授。
* '''[[死记硬背|死记硬背]]''' - 通过重复和记忆来教授数学结果，定义和概念。通常用于[[乘法表|乘法表]]。
* '''习题''' - 通过完成大量同类的练习来传授数学技巧，例如加[[带分数|带分数]]或者解[[二次方程|二次方程]]。例如，[[古氏积木|古氏积木]]（Cuisenaire rods）来教授分数。
* '''[[问题求解|问题求解]]''' - 通过给学生无标准答案，不同寻常的，和有时候无解的问题来培养数学的智力，创造力和[[启發式|启發式]]思考。问题的范围可以从{{le|應用題 (數學)|Word problem (mathematics education)|應用題}}到像[[国际数学奥林匹克竞赛|国际数学奥林匹克竞赛]]这样的国际数学竞赛问题。
* '''[[新数学|新数学]]''' - 一种专注于[[集合论|集合论]]这样的抽象概念而不是实际应用的教授数学的方法。
* '''历史方法''' - 教授在一个历史，社会和文化背景下数学的发展过程。比纯粹抽象的方式提供了更多的[[人文乐趣|人文乐趣]]。
这些方法不是所有的，而且任何数学教育系统很可能包含很多不同的方法。

== 数学教师 ==
这些人曾在一生中某一阶段教授数学，但他们在其他方面更为著名：
* [[Lewis_Carroll|Lewis Carroll]], [[英国|英国]]作家Charles Dodgson的笔名，曾在[[牛津基督教堂|牛津基督教堂]]讲授数学
* [[道尔顿|道尔顿]], 英国化学家和物理学家，曾在[[曼彻斯特|曼彻斯特]]，[[牛津|牛津]]和[[约克|约克]]的学校和大学教数学。
* [[Tom_Lehrer|Tom Lehrer]], [[美国|美国]]歌曲作家和讽刺作家，曾在[[哈佛|哈佛]]和[[麻省理工学院|麻省理工学院]]教数学。
* [[Georg_Joachim_Rheticus|Georg Joachim Rheticus]], [[奥地利|奥地利]]绘图家，[[哥白尼|哥白尼]]的学生，曾在[[Wittenberg大学|Wittenberg大学]]教数学。
* [[Edmund_Rich|Edmund Rich]], 13世纪[[坎特伯雷大主教|坎特伯雷大主教]]，在[[牛津|牛津]]和[[巴黎|巴黎]]的大学教过数学。
* [[Archie_Williams|Archie Williams]], [[美国|美国]]运动员，奥运金牌得主，在[[加里福尼亚|加里福尼亚]]高中教过数学。
