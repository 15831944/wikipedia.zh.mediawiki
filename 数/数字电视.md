'''數位電視'''（{{lang-en|'''Digital television'''}}）是指採編、播出、傳輸、接收等環節中全面採用[[数字信号|數位訊號]]的[[電視|電視]]系統，與[[類比電視|類比電視]]相對。數位電視系統可以傳送多種業務，如[[高畫質電視|高畫質電視]]、[[標準畫質電視|標準畫質電視]]、[[智慧型電視|智慧型電視]]及數位業務等等。

== 标准 ==
[[File:Digital_broadcast_standards.svg|thumb]]
数字电视涉及很多领域的标准。其中传输标准分为：地面传输（无线）、有线传输、卫星传输、手持设备传输四个体系。

# 美國标准[[ATSC|ATSC]] 8-VSB用於6MHz電視频道。
#:使用地區：[[美國|美國]]、[[加拿大|加拿大]]、[[墨西哥|墨西哥]]、[[大韓民國|南韓]]、[[中美洲|中美洲]]部份國家。
# 欧洲标准[[DVB-T|DVB-T]] COFDM用於6/7/8 MHz電視频道。
#:使用地區：[[地球|全球]]大部份國家和地區，包括[[歐洲|歐洲]]及[[大洋洲|大洋洲]]各國、[[亞洲|亞洲]]多國，[[非洲|非洲]]及[[中東|中東]]大部份地區亦已在2006年承諾採用<ref>[http://www.dvb.org/about_dvb/dvb_worldwide/index.xml DVB-T Worldwide - DVB-T Status Overview]</ref>，目前[[台灣|台灣]]採用的是歐規DVB-T 6MHz。
# 日本标准[[ISDB|ISDB-T]] COFDM用於6MHz電視频道（屬欧洲标准改良型）。
#:使用地區：[[日本|日本]]、中美洲部份國家、[[南美洲|南美洲]]大部份國家（僅[[哥倫比亞|哥倫比亞]]及[[法屬圭亞納|法屬圭亞納]]使用DVB-T）
# 中國大陆标准[[DTMB|DTMB]]（[[中華人民共和國|中華人民共和國]]於2006年8月自訂標準）
#:使用地區：[[中國大陸|中國大陸]]、[[香港|香港]]、[[澳門|澳門]]和[[古巴|古巴]]

* 有线传输标准
# 美國标准为[[ATSC-C|ATSC-C]]。（其他地区待补充）
# 欧洲标准为[[DVB-C|DVB-C]]。
# 日本标准为[[ISDB-C|ISDB-C]]。
# 台灣的有線電視標準不一，但以[[DVB-C|DVB-C]]為主，另有系統採[[ATSC-C|ATSC-C]]。
# 中国大陆的有线电视网络一般采用的是欧洲[[DVB-C|DVB-C]]标准。

* 卫星传输标准
# 欧洲：欧洲标准为[[DVB-S|DVB-S]]、[[DVB-S2|DVB-S2]]。这两个标准也是事实上的国际标准。
# 日本：放送卫星（BS）采用[[ISDB|ISDB-S]]；通信卫星（CS）分为两类，东经110度CS采用ISDB-S，东经124/128度CS采用DVB-S。
# 中国大陆：中国主要采用[[DVB-S|DVB-S]]及[[DVB-S2|DVB-S2]]作为卫星直播电视标准。

* 手持设备传输标准
# 欧洲标准为[[DVB-H|DVB-H]]。

# 国际上还有[[DVB-SH|DVB-SH]]标准及[[MediaFLO|MediaFLO]]标准。
# 日本采用ISDB-T标准中[[1seg|1seg]]方式。
# 中国大陆采用[[CMMB|CMMB]]标准。

== 數位電視系統 ==
根据信号的空间传播方式，数字电视主要包括4种形式：数字电视地面传输系统、数字[[有线电视|有线电视]]系统、数字[[卫星电视|卫星电视]]系统、[[IPTV|IP]]數位電視系統。

數位電視系統包括前端系统、传输网络和接收终端。

== 数字电视机 ==
能够直接接收并处理外来数字电视信号的电视机，或已內置數碼電視[[解碼器|解碼器]]的電視機，一般称为“数字电视机”（IDTV）。

接收[[模拟电视|模拟电视]]信号、仅在图像还原过程中采用数字处理技术的电视机，经常被称为“数字电视”，但这并不是真正意义上的数字电视。

目前能够直接接收并处理外来数字电视信号的电视机已逐漸普及，但大部份家庭仍使用只能接收模拟电视信号的电视机。如要继续使用这些电视机，可以利用“[[數位視訊轉換盒|数字电视-{zh-hans:机顶盒;zh-hk:機頂盒;zh-tw:機上盒;}-]]”（{{lang|en|Set-top Box}}）把数字电视信号转换成模拟电视信号，就能以模拟电视机接收数字电视。

== 與模擬電視比較 ==
一條[[模擬電視|模擬電視]]频道本來只能傳送一個模擬節目，但經在數碼化壓縮訊號後可同時播出3至4個[[標準畫質電視|標準畫質]]（{{lang|en|Standard-definition television, SDTV}}）的節目，因此計算下來單一频道或節目的傳輸成本會降低；而觀眾則有更多的節目可供選擇，另外訊號經數碼化處理，可消除[[雜訊|雜訊]]及干擾，觀眾便能看到更清晰細緻的畫面。

=== 中国大陆 ===
中国大陆于近年来大力推行由电视模拟信号向数字信号的转换。计划于2015年前在全国范围关闭有线电视模拟信号。但是在推行过程中存在许多问题。

地面无线信号将在数字化覆盖水平与现有模拟覆盖率相当后，逐步关闭模拟信号。2020年前有望实现全面数字化，届时计划将全面关闭模拟信号。<ref>{{Cite web|url=http://www.qlwb.com.cn/2015/1207/508765.shtml|title=2018年主要电视节目无线数字化覆盖，2020年地面模拟电视信号停用 －泰安－齐鲁晚报网|accessdate=2017-09-03|last=邓金易|work=www.qlwb.com.cn}}</ref>
* 中国大陆的数字电视在标准方面较为混乱。1998年开始数字电视试验及研究，历时6年投入超过10亿元[[人民币|人民币]]，仍未形成统一标准，不同领域形成了几个差别很大的独立体系。
* 有线电视系统是地方独立运营的诸侯割据状态。数字化开始比较早，多采用欧洲标准DVB-C。2003年至2004年已有[[北京|北京]]、[[上海|上海]]、[[青岛|青岛]]、[[江苏|江苏]]、[[杭州|杭州]]、[[佛山|佛山]]、[[深圳|深圳]]、[[广州|广州]]、[[大连|大连]]等地开通了数字有线电视播出。2005年，又有[[福州|福州]]、[[厦门|厦门]]等地开通了数字电视的播放。
* 无线电视受制于标准争论，[[清华大学|清华大学]]的DMB-T标准和[[上海交通大学|上海交通大学]]的ADTB-T标准是最为热门的竞争对手。2006年8月，[[國家標準化管理委員會|國家標準化管理委員會]]公佈了中國大陸的數碼廣播标准，是為[[DMB-T/H|DMB-T/H]]（GB 20600-2006）。<ref>[http://www1.chinesenewsnet.com/MainNews/SocDigest/Technology/xhw_2006_09_01_01_42_01_436.html]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://www.cnetnews.com.cn/news/electron/story/0,3800060195,39524023,00.htm]</ref><ref>{{cite web |url=http://www.sac.gov.cn/cgi-bin/bulletin/showbulgb.asp?id=904&year=2006&num=8&tnums=95&cnt=161&pdate=2006-8-24 |title=存档副本 |accessdate=2007-01-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20070115061353/http://www.sac.gov.cn/cgi-bin/bulletin/showbulgb.asp?id=904&year=2006&num=8&tnums=95&cnt=161&pdate=2006-8-24 |archivedate=2007-01-15 }}</ref>此标准是上述两标准的融合产物。
* 手持设备方面则更为混乱。在电信领域，[[信息产业部|信-{}-息产业部]]和[[中国移动|中国移动]]等支持的是[[新岸线公司|新岸线公司]]的T-MMB标准（普遍认为其源于南韩的T-DMB标准，但新岸线方面予以否认）；在广电领域，2006年10月24日，[[国家广播电影电视总局|国家广播电影电视总局]]颁布了中国移动多媒体广播行业标准为STiMi标准，此后一般称之为CMMB。两方面互不认可，各自都在试运营阶段。因为入网许可证需要电信管理部门的审批，搭载CMMB模块的手机难以获得入网许可证，目前CMMB更多被在MP4等个人便携终端采用。此外，还有清华大学的DMB-H标准、[[华为|华为]]的CMB、[[中国标准化协会|中国标准化协会]]多媒体通信广播标准化技术委员会CDMB等。据报道，经过半年多的“选秀”，2008年4月3日的国家标准化管理委员会的工作会议决定，手机电视的国家标准采用T-MMB标准<ref>[http://tech.sina.com.cn/t/2008-04-04/09442120072.shtml]</ref>；而广电方面起初参加“选秀”，而后中途退出、并宣布不会采用该国家标准<ref>[http://tech.sina.com.cn/t/2008-04-04/22052120223.shtml]</ref>。
* 标准的混乱使众多开发商不得不放慢跟进的脚步。这在事实上远远落后于欧洲DVB标准的产业化进程，造成大量厂商需要进行多套技术方案的准备，且没有成形的市场环境。
* 中国的数字电视收费模式没有形成：
** 中国大陆原有电视体系中，节目免费收看，电视体系通过插播广告来获得收入。各地一般都有几十套免费的模拟电视节目播出，这是建立数字电视节目的收费模式的一个重要难题；
** 中国大陆的广电体系是地方独立运营的诸侯割据状态，数字电视的收费分成是各方利益争夺的焦点。
*目前，[[湖南|湖南]]、[[山东|山东]]等省市[[无线数字电视|无线数字电视]][[地面波|地面波]]传送的央视和地方台十全十美，发展鼎盛，乃至县城农村地区遍地开花。

==== 數碼地面電視廣播 ====
{{Main|香港數碼地面電視廣播}}

2009年底數據顯示，數碼地面電視覆蓋全港85%人口，現時覆蓋全港18區超過96%的人口。而接收數碼電視的家庭（滲透率）則由2008年的32.3%增至46.5%。2011年3月，一項公眾調查顯示數碼廣播滲透率增至63%，但同時增幅開始放緩。2013年調查顯示數碼廣播滲透率增至80%，2015年6月，根據無綫的2015年中期報告指數碼廣播的滲透率由全港80%的住戶微升至全港約86%的住戶，通訊事務管理局的主要的通訊業統計數字直到2015年12月，數碼地面電視滲透率接近85%。

政府原先訂定在2012年底前終止模擬廣播，但考慮到數碼地面電視服務在香港的實際推行情況和市場狀況，政府在2011年6月表示現階段不會取消原有的模擬廣播直至2015年<ref name="hknewsgov2015" /><ref>[http://www.info.gov.hk/gia/general/201106/22/P201106220282.htm 終止模擬電視廣播工作目標日期延至二○一五年年底]（[http://www.cedb.gov.hk/ctb/chi/legco/pdf/LegCo_Brief_ASO.pdf 附件]）商務及經濟發展局，2011年6月22日</ref><ref>[http://www.youtube.com/watch?v=9m_8k4wf0hQ 政府押後三年後才終止電視模擬廣播] 香港無綫電視六點半新聞，2011年6月22日</ref>，然而於2014年12月9日再把模擬電視廣播進一步推後2020年年底才正式取消<ref>[http://www.info.gov.hk/gia/general/201412/09/P201412090668.htm 終止模擬電視廣播工作目標修訂至二○二○年]</ref>。
2019年2月11日，政府正式宣佈於2020年11月30日晚上11時59分起終止模擬電視廣播，並於0時0分起全面實行數碼廣播；並指出由於本港與內地均使用同一頻譜，故為免出現互相干擾情況而決定押後實施此決定至2020年11月30日才實行<ref>[https://hk.on.cc/hk/bkn/cnt/news/20190211/bkn-20190211150242241-0211_00822_001.html 2020年終止模擬廣播 港府資助16萬戶買數碼電視]，on.cc東網，2019年2月11日</ref><ref>[https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/293423/%E6%A8%A1%E6%93%AC%E9%9B%BB%E8%A6%96%E5%BB%A3%E6%92%AD2020%E5%B9%B411%E6%9C%88%E7%B5%82%E6%AD%A2-%E7%BF%8C%E6%97%A5%E5%85%A8%E9%9D%A2%E8%BD%89%E7%94%A8%E6%95%B8%E7%A2%BC%E9%9B%BB%E8%A6%96%E5%BB%A3%E6%92%AD 模擬電視廣播2020年11月30日終止 關愛基金助合資格住戶置新設備]，香港01，2019年2月11日</ref><ref>[https://hk.news.appledaily.com/local/daily/article/20190212/20611116 模擬電視廣播明年12月「熄機」]，蘋果日報，2019年2月12日</ref>。

=== 馬來西亞 ===
馬來西亞[[Astro|Astro]]亦於2009年12月18日正式啟播高清服務。率先於2009年和2010年推出的高清频道有：Astro SuperSport HD、National Geographic Channel HD、History HD、HBO HD、ESPN HD、[[Astro至尊HD|Astro至尊HD]]、AXN HD、FX HD等等。直到2012年，Astro已有21个高清频道。

一旦數碼無線電視系統（DTT）完全取代現有的類比電視系統（Anolog TV），馬來西亞人將可在2017年或2018年享受80個免費電視频道。Puncak Semangat有限公司數碼無線電視系統計劃副總監莫哈末利茲萬說，在這80個免費频道中，有24個是高清频道，及56個標準清晰度频道（SD）。他說，這些電視频道將覆蓋整個區域，以便資訊能夠有效傳達，尤其是一些國內特定地區，包括政府的資訊频道。“現有的類比電視系統將會在2016年6月暫停，目前我們正籌備30個频道，即9個高清频道和21個標準清晰度频道。”報導《每日新聞》報導，莫哈末利茲萬說，通過數碼無線電視系統，消費者將獲更多好處，包括更有素質的服務。

=== 日本 ===
{{main|日本地面数字电视}}
日本從2000年12月1日起開始以[[廣播衛星|廣播衛星]]（[[通訊衛星|Broadcasting satellite]]）轉發數位電視訊號，稱為「BS Digital」。2003年12月1日起在三大都會區（[[日本首都圈|首都圈]]、[[京阪神|京阪神]]、[[名古屋|名古屋]]）開始地面發射站發射數位電視訊號，並且逐步擴展到全日本。2006年4月1日起，開播針對行動收視者的[[1seg|1seg]]數位電視。除了[[岩手縣|岩手縣]]、[[宮城縣|宮城縣]]及[[福島縣|福島縣]]因[[東日本大地震|東日本大地震]]受災嚴重，日本[[總務省|總務省]]決定將此三縣的停播時間延至2012年3月31日，日本其餘44都道府縣的無線及衛星類比電視訊號已於2011年7月24日結束。

===美國===
美國採用[[ATSC|ATSC]]系統，已經於2009年6月12日起停止無線類比電視訊號播送，僅留下少部份地方有小功率類比電視訊號播送，訊號亦已於2015年全面關閉。

===泰國===
泰國原計畫於2010年開始播放地面無線數位電視，並在2015年完成將類比訊號轉換為數位訊播放階段，但由於政治問題，被多次延遲。目前於2014年已正式開播地面無線數位電視，但擴及到全國必須到2016年，並計畫於2020年將類比訊號轉換為數位訊號完成。

其系統規格採用DVB-T2，六個電視频道，但DVB-T2可讓觀眾收看多達48個免費電視台節目。其中公眾商業台有24台，政府商業台有12台，以及12個地方社區電視台，其中有10個電視台節目為HD高畫質節目。

===新加坡===
[[新傳媒私人有限公司#簡史|新傳媒]]轄下各頻道於2006年12月22日起開展數碼廣播，並已於2018年1月2日凌晨零時起終止各頻道正常模擬廣播（一直播放宣導字卡至2月1日凌晨全部模擬版本頻道正式停波），全面過渡至數碼電視。
