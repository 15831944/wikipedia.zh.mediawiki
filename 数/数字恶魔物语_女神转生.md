{{vgname|数字恶魔物语 女神转生|ja=デジタル・デビル物語 女神転生}}是一款由[[Atlus|Atlus]]制作、[[南梦宫|南梦宫]]发行的[[角色扮演类游戏|角色扮演类游戏]]<ref name="mobygames">[http://www.mobygames.com/game/nes/digital-devil-story-megami-tensei ''tensei''] game overview at {{tsl|en|MobyGames||MobyGames}}</ref>。本游戏于1987年9月11日在日本地区[[FC游戏机|FC游戏机]]发行<ref name="gamefaqs">{{cite web | url = http://www.gamefaqs.com/nes/578475-digital-devil-monogatari-megami-tensei/data| title = Release information | publisher = [[GameFAQs|GameFAQs]]| date = | accessdate = 2015-11-19}}</ref>。本游戏根据西谷史的小说《[[数字恶魔物语|数字恶魔物语]]》改编。

本游戏设有多个元素，这些成为[[女神转生系列|女神转生系列]]以后作品的标准。其中包括第一视角的地牢和捕捉恶魔。

== 玩法 ==
FC版的《数字恶魔无语 女神转生》是一款传统的角色扮演类游戏。玩家控制一组2个人类和一些恶魔的队伍。这群队伍使用第一人称视角探索地下城。

战役是回合制，当探索地下城时，会随机发生。

== 制作 ==
《数字恶魔物语 女神转生》源于日本作家西谷谷所写的科幻小说三部曲。小说由[[德間書店|德間書店]]出版，在日本获得巨大成功。小说第一次引入了[[原創動畫錄影帶|原創動畫錄影帶]]（OVA）。其重新叙述了最初小说的事件。随着OVA在1987年完成，德间书店决定将系列引入电子游戏。

== 评价 ==
《Fami通》将Altus的3D地下城游戏与《Deep Dungeon系列》进行对比。对于本游戏的玩法和设定进行了赞扬。
