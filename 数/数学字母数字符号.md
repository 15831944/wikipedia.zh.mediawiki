{{Hatnote|本文关注Unicode[[码位|码位]]区段（Unicode block），着眼于Unicode的字母编码与排版等。不关注拉丁字母与希腊字母在数学中的使用与含义。}}
'''数学字母数字符号'''（Mathematical Alphanumeric Symbols）是Unicode的一个[[Unicode字符列表|码位区段]]（{{lang|en|Unicode block}}）的正式名称。包含不同字型风格的[[拉丁字母|拉丁字母]]、[[希腊字母|希腊字母]]与十进制[[数字|数字]]在数学中用作不同指代含义的符号。例如，黑板粗体（blackboard bold），Unicode中使用的对应术语是双线体（double-struck）。

[[Unicode|Unicode]]的[[Unicode字符平面映射#第一辅助平面|第一辅助平面]]的码位区段1D000-1DFFF是“记号系统”(Notational systems)，在<code>U+1D400</code> ... <code>U+1D7FF</code>包含了称之为数学字母数字符号的[[字形|字形]]). 这使得可以专门设计与使用特殊的数学字符字型，不同于正文中使用的普通字母、数字。例如，数学中使用的''斜体'' "A"与''正体''字母"A"有不同的含义. 这些数学字母数字符号仅用于数学公式，不用于普通的文本中.

不同字型的这些符号在不同的数学领域往往具有特定的、固定的含义。

还有一种情况，例如U+1D4C1: 𝓁({{smallcaps|{{lc:MATHEMATICAL SCRIPT SMALL L}}}})，还有一个类似外观的字符U+2113: ℓ ({{smallcaps|{{lc:SCRIPT SMALL L}}}}).<ref>[http://www.unicode.org/charts/PDF/U1D400.pdf Unicode code chart U1D400 (PDF)]</ref>
