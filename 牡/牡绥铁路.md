'''牡绥铁路'''是[[中华人民共和国|中华人民共和国]][[黑龙江省|黑龙江省]][[牡丹江市|牡丹江市]]的一条铁路，以“[[滨绥铁路|滨绥铁路]]牡丹江至绥芬河段扩能改造工程”名义报批。铁路由[[牡丹江市|牡丹江市]]至[[绥芬河市|绥芬河市]]，全长138.8公里，设计时速200公里/小时<ref>{{cite web| url=http://www.suifenhe.gov.cn/contents/1581/3865.html| title=牡绥线扩能改造项目| publisher=绥芬河市人民政府| accessdate=2013-11-29| archiveurl=https://web.archive.org/web/20131203041131/http://www.suifenhe.gov.cn/contents/1581/3865.html| archivedate=2013-12-03| deadurl=yes}}</ref>。

== 概況 ==
[[滨绥铁路|滨绥铁路]]牡丹江至绥芬河段，于1901年建成，由于建设年代久远，技术条件差，旅客列车运行时速只有46.5公里。2008年末，[[中华人民共和国铁道部|原铁道部]]决定对滨绥线牡丹江至绥芬河段进行扩能改造，并新建二线，届时旅客列车时速将达到200公里以上<ref>{{cite web | url=http://news.163.com/10/0514/09/66KQ2KPA00014AEE.html | title=牡绥线扩能改造3年后完工 哈尔滨2小时即达绥芬河 | accessdate=2013-11-29}}</ref>。

2009年10月，国家发展改革委批复了滨绥铁路牡丹江至绥芬河段扩能改造工程可行性研究报告<ref>{{cite web | url=http://www.sdpc.gov.cn/xmsphz/t20091119_314102.htm | title=滨绥铁路牡丹江至绥芬河段扩能改造工程可行性研究报告通过批复 | publisher=中华人民共和国国家发展和改革委员会 | accessdate=2013-11-29}}</ref>。2010年7月，牡绥铁路於開工建設，于2015年12月28日竣工通车<ref>{{cite web | url=http://news.xinhuanet.com/local/2015-12/28/c_1117604730.htm | title=牡绥铁路扩能改造工程竣工通车  | accessdate=2015-12-28}}</ref>。

线路自既有滨绥铁路[[牡丹江站|牡丹江站]]引出，经[[穆棱市|穆棱]]、[[下城子|下城子]]、[[绥阳县|绥阳]]至边境口岸城市绥芬河。總長度138公里，投資108億元[[人民币|人民币]]。设计速度为每小时200公里，全线为双线[[電氣化鐵路|電氣化鐵路]]。规划运力客车40对/日、货运5000万吨/年。铁路设[[牡丹江站|牡丹江]]、[[爱河站|爱河]]、[[磨刀石站|磨刀石]]、[[穆棱站|穆棱]]、[[细鳞河站|细鳞河]]、[[绥阳站|绥阳]]、[[绥芬河站|绥芬河]]七个车站，工程投资106.2亿元。牡丹江至绥芬河间单程耗时将由近4个小时缩至1个小时<ref name="fugong">{{Cite web |url=http://mudanjiang.dbw.cn/system/2012/05/23/053895161.shtml |title=牡绥铁路扩能改造项目全线11个工程开复工 |access-date=2013-11-29 |archive-url=https://web.archive.org/web/20130502151929/http://mudanjiang.dbw.cn/system/2012/05/23/053895161.shtml |archive-date=2013-05-02 |dead-url=yes }}</ref>。

工程于2015年底完工，并于2016年初开通运营。<ref>{{cite news|url=http://www.andatielu.com/html/4769155337.html|title=最近中俄电气化铁路正式开通|date=2016-01-07}}</ref>
