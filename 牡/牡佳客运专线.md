'''牡佳客运专线'''，是[[中国|中国]]一条建设中连接[[黑龙江省|黑龙江省]][[牡丹江市|牡丹江市]]与[[佳木斯市|佳木斯市]]的客运专线，设计时速为250公里。线路从既有[[滨绥铁路|滨绥铁路]][[牡丹江站|牡丹江站]]引出，途经[[鸡西市|鸡西市]]、[[七台河市|七台河市]]、[[双鸭山市|双鸭山市]]，引入既有[[牡佳铁路|牡佳铁路]][[佳木斯站|佳木斯站]]，全长374.922公里，其中新建线路370.890公里<ref name="tsdig">{{cite web|url=http://www.tsdig.com/NewsDetail.aspx?Id=846&FileName=info&Language=1|title=新建牡丹江至佳木斯铁路客运专线工程环境影响评价第二次信息公告|publisher=铁道第三勘察设计院集团有限公司}}</ref>。

== 概述 ==
牡佳客运专线由[[中国铁路总公司|中国铁路总公司]]和黑龙江省共同出资修建，项目可研报告于2016年9月30日获得[[中华人民共和国国家发展和改革委员会|中国国家发展和改革委员会]]的批准<ref>{{cite news|url=https://wx.abbao.cn/a/1957-d411d5941e28f96b.html|title=【关注】牡佳客运专线项目可研报告通过国家发改委审查|publisher=好文网}}</ref>，已于2016年11月8日开始建设，预计于2020年开通。<ref>{{cite web |url=http://news.163.com/16/1109/08/C5DTD6GM00014AED.html |title=牡佳客专昨日开工建设打造我省东部环线“两小时”经济圈 |publisher=News.163.com |date=2016-11-09 |accessdate=2017-08-14 |deadurl=no |archiveurl=https://web.archive.org/web/20170815023343/http://news.163.com/16/1109/08/C5DTD6GM00014AED.html |archivedate=2017-08-15 |df= }}</ref>牡佳客运专线建成后，将與[[哈牡客运专线|哈牡客运专线]]、[[哈佳铁路|哈佳铁路]]形成黑龙江省内快速铁路环线，缩短黑龙江省东部城市间的距离<ref>{{cite news|url=http://hljmdj.wenming.cn/wmjj/201610/t20161009_2863342.html|title=牡佳客运专线项目可研报告通过国家发改委审查|publisher=中国文明网}}</ref>。

== 线路走向 ==
牡佳客运专线全线设牡丹江、[[桦林东站|桦林东]]、[[林口南站|林口南]]、[[鸡西西站|鸡西西]]、[[七台河西站|七台河西]]、[[桦南东站|桦南东]]、[[双鸭山西站|双鸭山西]]、佳木斯共8座车站，并预留孟家岗北站和四马架北站<ref name="tsdig"/>。
