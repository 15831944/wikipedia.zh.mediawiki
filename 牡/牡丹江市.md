'''牡丹江市'''是[[中华人民共和国|中华人民共和国]][[黑龙江省|黑龙江省]]下辖的[[地级市|地级市]]，位于黑龙江省东南部，中国大陸最大的边贸城市之一，是該省最南部的一座城市；亦是北方的一個著名旅遊城市。南接[[吉林省|吉林省]]，東与[[俄罗斯|俄罗斯]]接壤，境内的[[绥芬河|绥芬河]]和[[东宁|东宁]]是[[中国|中国]]与俄罗斯进行贸易的重要口岸。因[[松花江|松花江]]上最大支流之一的[[牡丹江|牡丹江]]橫跨市區因而得名。[[牡丹江海浪机场|牡丹江海浪机场]]拥有六条国际航线和数条国内航线，是中国[[东北地区|东北地区]]的七大航空港之一。牡丹江市是黑龙江省第三大城市，亦是黑龙江省东部最大的城市和[[政治|政治]]、[[文化|文化]]、[[交通|交通]]、[[科技|科技]]、[[经济|经济]]中心。总面积3.88万平方公里，人口262.04万，其中市区人口88.32万。市人民政府驻江南新區。

== 历史 ==
牡丹江是由[[满语|满语]]“穆丹乌拉”转变为[[汉语|汉语]]的，“穆丹”在满语中是“弯曲”的意思，“[[乌拉|乌拉]]”是江的意思。牡丹江的名字实际上是“弯弯曲曲的江”。

牡丹江市的历史悠久，自商朝直到隋朝的二千三百多年间一直是[[满族|满族]]人的先祖[[肃慎|肃慎]]、[[挹娄|挹娄]]、[[勿吉|勿吉]]的世居地。[[唐代|唐代]]时期该地属于[[渤海国|渤海国]]五京之首[[上京龙泉府|上京龙泉府]]，亦是全國首都，城市整體佈局仿造鄰國大唐[[長安|長安]]而建，占地亦達到了十六平方公里，分外城、內城和宮城。外城為土牆，內城為石牆，宮城為玄武岩牆。五座迴廊相連的大殿排列在南北軸線上。渤海國曾被稱之為“海東勝國”，經濟鼎盛一時，是曾與[[大唐|大唐]]關係，交流最為密切的國家之一。當時渤海國與朝鮮半島關係緊密，[[渤海人|渤海人]]自稱自己為[[高句麗|高句麗]]的繼承人，而渤海國與朝鮮半島則被外間稱之為“南北國”。

926年渤海國被[[契丹|契丹]]所滅後，其北部領土被契丹吞併，部分南部領土則被[[高麗|高麗]]吞併。

[[遼代|遼代]]属于上京胡里改路，金代和元代则属万户府直辖，明代和清代是宁古塔将军和吉林将军所管辖地方。由于清代朝廷一直以“祖宗发源之圣地”为由实行禁止垦荒政策，直到1898年沙俄修筑中东铁路，设立了牡丹江火车站，才开始发展。
=== 俄治時期 ===
1914年，牡丹江彙聚了一批來自[[日本|日本]]，[[俄國|俄國]]，[[英國|英國]]，[[法國|法國]]和[[丹麥|丹麥]]等地的商人，並開始出現外國銀行以及洋行，其中法國利豐公司，英國華英公司，俄国宝隆洋行，丹麦宝隆洋行分別在牡丹江設置代銷處，令城市商業蓬勃發展<ref>《牡丹江市誌》黑龍江人民出版社出版 上卷第15頁</ref>1923年2月，[[五四運動|五四運動]]組織者之一的[[馬駿|馬駿]]回到牡丹江組織地下傳媒開始傳播新文化，對當時牡丹江市的青年一代影響至深。1925年，日本佐佐木公司在牡丹江開設[[鴉片|鴉片館]]，是牡丹江市出現的第一個鴉片館。

1929年7月，“[[中東路事件|中東路事件]]”爆發，中華民國政府於7月7日對[[蘇聯|蘇聯]]宣戰，調國民革命軍師長郑泽生在牡丹江設防，并在新立屯建設飛機場，有飛機三架，是牡丹江市的第一個機場，但該年10月即被蘇聯空軍炸毀，全部飛機均報廢<ref>《牡丹江市誌》黑龍江人民出版社出版 上卷第18頁</ref>。

=== 日占时期 ===
[[满洲国|满洲国]]时期，牡丹江一带是抗日联军将领[[周保中|周保中]]领导的抗联第2路军活动的地方，著名的抗联女战士“[[八女投江|八女投江]]”就发生在牡丹江境内[[林口县|林口县]]。在[[九一八事变|九一八事变]]后，[[周保中|周保中]]率部戰敗，牡丹江成為日本的殖民地，為掠奪牡丹江市周邊的大量木材煤炭等资源，进一步加强铁路建设，牡丹江站因位于“满州里—绥芬河”“图们—佳木斯”绥满图佳两条铁路线的交叉点，成为重要的交通枢纽，幷於1937年12月建市，成为[[满洲国|满洲国]]的一个省会。

1938年10月15日，日本在牡丹江市設立[[領事館|領事館]]，將牡丹江市的級別提升爲滿洲國的特別市，并設立满洲纸浆株式会社，满洲真空株式会社和鏡泊湖水利發電所，牡丹江的政治和經濟地位在此後大為提升。1940年3月1日，[[日文|日文]][[報紙|報紙]]《东满日日新闻》在牡丹江市創刊發行，成為牡丹江市歷史上的第一份報紙，同年8月1日，中文版發行。1943年7月，滿洲國政府在牡丹江市設立滑翔機研究所及訓練場，即現在的[[牡丹江海浪機場|海浪機場]]雛形。

1945年8月8日，蘇聯對日宣戰，并迅速突破過境，佔領綏芬河、東寧等地，直逼牡丹江市中心。8月10日夜間，日本在牡僑民大量撤退，造成牡丹江火車站混亂，政府出動軍隊維持秩序，但依舊造成多人死傷。8月9日-8月13日蘇聯空軍多次對市區進行轟炸，大量政府機構被炸毀，其中包括日本駐華領事館；并造成市民的大量傷亡。8月14日蘇聯紅軍進入牡丹江市區，並在16日完全佔領牡丹江市。

牡丹江在日治滿洲國期間經濟發展迅速，目前的大多數城市基礎建設，包括自來水淨水管道，城區電力系統佈局，工廠雛形均為日本規劃設計，在1930年代逐漸成為東北地區東北部的政治，經濟和文化中心。

=== 第二次国共内战时期 ===
1945年8月，[[苏联军队|苏联军队]]发起[[八月风暴行动|八月风暴行动]]，随后攻入[[中国东北地区|东北地区]]。8月15日，[[日本无条件投降|日本无条件投降]]。8月16日，苏军攻占牡丹江市。这是苏军在东北攻占的第一个城市<ref name="a1"/>。此后，随着[[满洲国政府|满洲国政府]]垮台，[[国民政府|国民政府]]和[[中国共产党|中国共产党]]开始对东北地区的争夺。中共军队的[[李荆璞|李荆璞]]、[[金光侠|金光侠]]部随苏军进入牡丹江市。国民政府试图接管牡丹江市、成立牡丹江市政府被中共阻止，未能成功。10月14日，成立牡丹市人民政府，李荆璞任市长<ref name="a1">{{Cite web |url=http://dangshi.people.com.cn/n1/2016/0303/c85037-28169463-4.html |title=东北根据地建立人民民主政权侧记 |accessdate=2017-11-04 |author=于佰春 |date=2016-03-03 |work= |publisher=中国共产党新闻网|language=Zh-hans }}</ref>。

1946年3月，苏军撤回苏联，国民政府增兵东北。4月，中共政权成立了[[绥宁省|绥宁省]]，后改[[牡丹江省|牡丹江省]]，省会设在牡丹江市。5月15日，国民政府方面发动[[牡丹江暴动|牡丹江暴动]]，暴动被中共所挫败。1947年8月15日，牡丹江廣播電臺成立并開始向全境播音，後於1949年併入牡丹江人民廣播電臺。1948年10月牡丹江市并入[[松江省_(中华人民共和国)|松江省]]，牡丹江市亦成為松江省的省会。1949年，[[中华民国政府|中华民国政府]]在[[第二次国共内战|第二次国共内战]]中失败撤退至台灣，共产党在中国大陆建立[[中华人民共和国|中华人民共和国]]政府，牡丹江作为最早的中共占领地之一，其松江省的省會行政地位并没有发生变化。

=== 中华人民共和国时期 ===
1954年6月19日，松江省被調整并入黑龙江省，牡丹江被撤銷省會行政級別后成為黑龍江省的一個[[省轄市|省轄市]]。1956年3月5日，國務院设立了牡丹江市专员公署，專屬機關駐牡丹江市。1967年1月16日市委和市政府被紅衛兵造反派接管，該月18日地位和專屬亦被奪權，牡丹江市陷入無政府狀態。1968年2月6日，省革委决定，牡丹江地革委与牡丹江市革委机构合并，撤消牡丹江市建制。称牡丹江地区「[[革命委员会|革命委员会]]」，辖1个市11个县一个区。2月21日，地市[[公安局|公安局]]、[[检察院|检察院]]、[[法院|法院]]机关開始实行军管，成立“牡丹江地区公安机关军事管制委员会”，城市司法系統被徹底打亂。

1969年3月，[[珍寶島事件|珍寶島事件]]爆發，牡丹江市作為中方最前線城市，被派駐大量解放軍陸軍以及裝甲師駐紮，[[陳錫聯|陳錫聯]]部署瀋陽軍區空軍部隊將[[海浪機場|海浪機場]]先後兩次進行擴建，並派駐[[解放軍空軍|解放軍空軍]]第21殲擊師以及空軍雷達第六旅駐紮，海浪機場至此成為中華人民共和國最重要的邊防空軍基地之一。1972年5月，牡丹江電視臺開始試播，并首次開始自製節目，是黑龍江省的第一個市級電視臺。1973年9月1日，地市再次被分设，恢復牡丹江市建制，地革委會成為最高權力機構。1983年10月根据中华人民共和国[[国务院|国务院]]决定，地市再次合并，撤销地区行政公署，实行市带县的领导体制。而这一套体制至今仍在延续。

== 地理 ==
牡丹江市位于[[黑龍江省|黑龍江省]]的東南部，亦是全省最靠近南部的一座城市。西側是[[哈爾濱市|哈爾濱市]]，南側是[[吉林省|吉林省]]，而東側則直接與[[俄羅斯|俄羅斯]]接壤。

全市平均海拔230米，地形則是以[[山地|山地]]，[[丘陵|丘陵]]爲主，呈現出中山、低山、丘陵、[[盆地|河谷盆地]]四種地質形態，城市東部爲屬于[[長白山|長白山]]系的[[老爺嶺|老爺嶺]]和[[張廣才岭|張廣才岭]]，中部是牡丹江河谷盆地，整個地區凸現山勢幷且連綿起伏，亦縱橫河流，被稱爲「九分山水一分田」。
 
牡丹江地區的海拔最高處位于張廣才岭的白突山，其海拔高度爲1686.9米；而海拔最低地區則是位于[[綏芬河市|綏芬河市]]與俄羅斯的邊境地區，爲86.5米。

=== 氣候 ===
牡丹江市屬中[[温带大陆性季风气候|温带大陆性季风气候]]，年平均溫度4.5度，城市氣候整體四季分明，但冬季時間長度明顯多于夏季。夏季溫度亦明顯低于中國南方其它部分城市，所以處于其境內的[[镜泊湖|镜泊湖]]等自然度假區域亦在中華人民共和國境內爲小有名氣。冬季氣溫偏低，但是明顯高于黑龍江其它城市，幷且由于盆地原因風速亦偏低，境內的一些地方譬如位于[[海林市|海林市]]的[[雪鄉|雪鄉]]因爲獨特的地理位置和氣候環境亦聞名于外。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为2798723人<ref>{{cite book |author=牡丹江市统计局|title=《牡丹江市2010年第六次全国人口普查主要数据公报》}}</ref>。其中，男性为1414185人，占50.53%；女性为1384538人，占49.47%。性别比（以女性为100）为102.14。0－14岁的人口为314038人，占11.22%；15－64岁的人口为2214550人，占79.13%；65岁及以上的人口为270135人，占9.65%。

=== 民族 ===
牡丹江市大部分人口以及後代均是來自1930年代至1960年代，「[[闖關東|闖關東]]」時期的[[山東省|山東省]]以及[[河北省|河北省]]民眾以及其後代。原住民在1930年代前多為[[滿族|滿族]]。城市现有少数民族38个，其中人口多達24万人，占全市总人口的8.7%。其中以[[朝鮮族|朝鮮族]]市民居多，大部分分布于[[西安區|西安區]]，并建設有「朝鮮民族風情街」等民族氣息很濃的設施。

== 经济 ==
[[File:Mdj-taiping-road.jpg|thumb]]一角。拍摄于2008年2月]]
牡丹江的经济收入主要来自旅游业和轻工业。近年来在边贸领域拓展了新的经济增长点，经济对外贸易占黑龙江全省贸易总额的75%以上，并成为了全国最大的对俄罗斯贸易城市。牡丹江市拥有全国四大橡胶制造工厂之一的桦林橡胶厂，而北方工具厂则是以生产兵工产品著称的大型国有企业，位于市区的大宇造纸企业集团是黑龙江省内最大的外资工厂之一，另外还有一处大型的水利发电站莲花发电站建立在郊区。所辖穆棱市拥有中国大陆最大的风力发电基地之一的[[黑龙江华富穆棱风电项目|黑龙江华富穆棱风电项目]]。

目前經濟主要發展地區爲規劃中的江南新區，預計在2009年末城市人口將達到100萬，并且設立新的區域，即江南新區。

2009年國內生產總值603.4亿元，在黑龙江省排在[[哈爾濱市|哈爾濱市]]、[[大慶市|大慶市]]、[[齊齊哈爾市|齊齊哈爾市]]和[[綏化市|綏化市]]后，但人均國內生產總值则達到22416元人民幣，排在大慶市和哈爾濱市后名列全省第三位。

=== 主要商业 ===
牡丹江市主要的商業區位於東安區與西安區交界處的[[太平路商業區|太平路商業區]]，以及東安區的[[東一條路步行街|東一條路步行街]]。包括大商集團牡丹江百貨大樓，[[大商集團|大商集團]]新瑪特購物廣場，波斯特購物廣場，牡丹江勸業場在內的多個大型商場位於太平路；而在東一條路步行街上則興建了多個服裝品牌的專賣店，城市最大的超級市場[[大潤發|大潤發]]牡丹江店亦位於該處。另外在太平路以及東一條路步行街上亦興建了多個四星級以上的酒店以及如[[肯德基|肯德基]]在內的快餐連鎖店。

主要大型品牌商場有：[[大潤發|大潤發]]、[[國美電器|國美電器]]、[[蘇寧電器|蘇寧電器]]、[[百盛|百盛]]。

== 交通 ==
牡丹江市处于[[哈尔滨|哈尔滨]]经[[俄罗斯|俄罗斯]][[海参崴|海参崴]]到[[日本|日本]][[新潟|新潟]]的国际大通道中段，境内有3个县（市）与俄罗斯接壤，[[國境|边境线]]长221公里，与俄罗斯远东地区有上百年的通商历史。牡丹江市与海参崴直线距离248公里，与[[纳霍德卡|纳霍德卡]]直线距离331公里，距黑龙江省进入[[日本海|日本海]]和[[太平洋|太平洋]]最近的出海口俄罗斯海参崴市340公里，是黑龙江省东出海参崴、南下[[图们江|图们江]]的交通枢纽，从这里出海到日本比之[[大连港|大连港]]运距要短1600公里。境内有众多口岸 辖区内有牡丹江航空港、[[绥芬河铁路|绥芬河铁路]]、公路和[[东宁公路|东宁公路]]4个国家一类口岸，年货运能力700万吨、客运能力200万人次，是[[东北亚|东北亚]]区域经济合作中的一个重要的人员流动、物资集散、信息传递和交通中心。

== 文化 ==
=== 樣板戲 ===
1950年代開始后，「一本书」、「一台戏」使牡丹江市扬名全國。作家[[曲波|曲波]]以自己“解放战争”初期在牡丹江周边“剿匪”的经历为题材，写出了长篇小说《[[林海雪原|林海雪原]]》，此书成为当时的名著， 脍炙人口，也使人们在知道了[[杨子荣|杨子荣]]、[[高波|高波]]等剿匪英雄的同时，初识了牡丹江这块神奇的土地。[[文化大革命|文化大革命]]期间，[[江青|江青]]親自提名的8个样板戏唱响全国，以《林海雪原》改编的现代京剧《[[智取威虎山|智取威虎山]]》，更让“打虎上山”的[[杨子荣|杨子荣]]家喻户晓，牡丹江亦因此在中國大陸地區知名度大增。
=== 東北文化 ===
牡丹江的社會整體文化分布大約在1940年代初步形成，但是對其影響至今社會階層文化則出現于1950年代后。由于大量來自[[河北省|河北省]]以及[[山東省|山東省]]的移民出現，牡丹江市與其它東北城市在同一個時期內一樣形成了一種結合當地民俗的獨特异樣文化氛圍。這些主要體現在[[飲食|飲食]]，[[衣著|衣著]]，[[生活方式|生活方式]]，包括[[方言|方言]]方面。

城市的方言為[[東北話|東北話]]，{{fact|偏向于[[哈爾濱話|哈爾濱話]]}}，由於1920年代至1950年代城市曾在“闖關東”時期吸引大量華北，華東北部移民，對於年長的市民日常交流亦存在[[山東話|山東話]]，[[膠東話|膠東話]]以及[[河北話|河北話]]。

=== 飲食 ===
城市的飲食體系主要以來自[[山東省|山東省]]的[[魯菜|魯菜]]爲主，形成了[[東北菜|東北菜]]中的[[炖菜|炖菜]]、[[蘸醬菜|蘸醬菜]]等。但是牡丹江市在飲食方面又不同于其它中國東北城市，由于其擁有大量的[[朝鮮族|朝鮮族]]居民，[[韓國菜|韓國菜]]亦是整個城市的主要飲食文化的組成部分之一。
[[File:Juecai.JPG|thumb]]「[[蕨菜|蕨菜]]」]]

在市民的主要飲食中，[[火鍋|火鍋]]亦占據主導地位，这被认为是長期生活在[[寒冷|寒冷]]而又漫长的[[冬季|冬季]]所致。[[狗肉火锅|狗肉火锅]]是牡丹江市的特色饮食之一，人们在冬季喜欢與朋友或家人围坐在一起用已經煮熟的[[狗肉|狗肉]]放置在早已沸騰的火鍋內食用。通常在狗肉火鍋內還要加上[[枸杞子|枸杞子]]、[[豆腐|豆腐]]、[[生菜|生菜]]、[[白菜|白菜]]等。另外狗肉由于大多數的[[朝鮮族|朝鮮族]]市民食用而逐漸被[[漢族|漢族]]市民接受，并流行于整個城市。狗肉的加工程序亦會依照[[朝鮮半島|朝鮮半島]]的製作方法，用[[辣椒醬|辣椒醬]]和[[狗油|狗油]]加工成為「[[狗醬|狗醬]]」，在配合[[狗肉|狗肉]]、狗排、狗[[內臟|內臟]]等蘸食。

另外，由於城市位於山區中，故亦以[[蘑菇|蘑菇]]、[[木耳|木耳]]、[[榛子|榛子]]、[[松子|松子]]而出名，如蘑菇，木耳亦出口到全國各地以及海外。像東北菜中流行的燉菜中，就包括了一些蘑菇類食品，如[[小雞燉蘑菇|小雞燉蘑菇]]等。

其它的當地特色飲食還有[[朝鮮冷面|朝鮮冷面]]、東北[[酸菜|酸菜]]等其它[[東北菜|東北菜]]。

== 宗教 ==
牡丹江市與[[中國大陸|中國大陸]]其他地區相同，是一個缺乏整體[[宗教信仰|宗教信仰]]的世俗社會。大部分民眾并沒有正統的宗教信仰，但是亦有相當一部分人是[[佛教|佛教]]和[[基督教|基督教]]的信徒，在每年的[[耶誕節|耶誕節]]以及一些其他的紀念節日中，一些基督教信徒亦會進行一些集會來進行他們的信仰活動。另外，一些[[回族|回族]]市民亦是[[伊斯蘭教|伊斯蘭教]]的信仰者。[[牡丹江市圓通講寺|牡丹江市圓通講寺]]亦是黑龍江省面積和影響力最大的佛教寺廟之一。

== 體育 ==
1980年代前期主要以[[冰雪運動|冰雪運動]]爲主，亦是國家冰雪運動的訓練基地之一。1980年代后已然以冰雪運動爲主，主要爲[[冰球|冰球]]，[[速度滑冰|速度滑冰]]爲主，1990年代開始后[[滑雪|滑雪]]運動盛行，這個期間牡丹江市爲國家培養了一批世界級的運動員，包括短道速滑名将李琰（现中国短道速滑队总教练），世界冠軍[[王曼丽|王曼丽]]。

2006年舉辦[[黑龍江省第十一届運動會|黑龍江省第十一届運動會]]，是牡丹江市有史以來承辦最大型的綜合性質運動會。

主要的體育設施包括[[牡丹江北山體育場|牡丹江北山體育場]]，[[牡丹江東四體育館|牡丹江東四體育館]]，牡丹江市師範學院游泳中心，牡丹江火车头体育场，牡丹江市室內滑冰場等。
