'''蛙'''，是[[先锋派|先锋派]]作家[[莫言|莫言]]的[[长篇小说|长篇小说]]。最早由[[上海文艺出版社|上海文艺出版社]]于2009年出版发行。

== 故事概述 ==
本书通过写信和戏剧回忆过去和描绘现在的方式来叙述故事。书籍构成是剧作家蝌蚪写给[[日本|日本]]作家杉谷义人的四封长信和一部[[话剧|话剧]]。讲述了“我的姑姑”一位平凡的乡村妇产科医生六十年来落实[[计划生育|计划生育]]的故事，描绘了一幅新中国波澜壮阔的生育史。<ref name="txlj " />

== 创作过程 ==
该书由莫言构思酝酿了十几年，写了四年，三次删改，莫言的第十一部长篇小说。<ref name="txlj " />

== 创作风格 ==
该书由四封信一个剧本构成，属于一种新的艺术形式。同时极富有[[萨特|萨特]]式样的[[荒诞派|荒诞派]]色彩，融合了诙谐、戏谑、调侃、反讽、嬉闹、灵魂独白、戏中戏等文体风格。<ref name="txlj ">{{cite web|url=http://book.qq.com/a/20110818/000069_2.htm|title=莫言《蛙》简介|accessdate=2011年8月18日|work=|publisher=腾讯网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20121015060637/http://book.qq.com/a/20110818/000069_2.htm|archivedate=2012年10月15日|df=}}</ref>

== 人物原型 ==
书中姑姑万心的原型是莫言的小姑管贻兰，生于1937年，1956年开始在[[高密县|高密县]][[大栏乡|大栏乡]]医院做医生，1996年退休，其生活和书中姑姑的生活经历大致一样，只是晚年没有精神恍惚状态。<ref>{{cite web |url=http://news.qq.com/a/20121014/000075.htm |title=莫言反对政府修缮其旧居 认为这样做劳民伤财 |accessdate= 2012年10月14日|publisher=腾讯网 |work= |language =zh-cn}}</ref>

== 作者简介 ==
[[莫言|莫言]]，原名管谟业，山东高密县人，中国当代著名作家。

== 奖项和评价 ==
2011年8月，莫言凭借《蛙》获得第八届[[茅盾文学奖|茅盾文学奖]]。<ref>{{cite web |url=http://news.ifeng.com/gundong/detail_2011_08/21/8564174_0.shtml|title=《张炜莫言拿到久违的茅盾文学奖》|accessdate=2011年8月21日|work=蒋庆|publisher=凤凰网|language= zh-hans}}</ref>颁奖辞中说：“莫言的《蛙》以乡村医生别无选择的命运，折射着我们民族伟大生存斗争中经历的困难和考验。小说以多端视角呈现历史和现实的复杂苍茫，表达了对生命伦理的思考。叙述和戏剧多文本的结构方式建构宽阔的对话空间，从容自由、机智幽默，体现作者强大的叙事能力和执著的创新精神。”<ref>{{cite web |url=http://book.sina.com.cn/news/c/2011-09-20/1517291229.shtml|title=《茅盾文学奖颁奖词公布 对获奖作品给予好评》|accessdate=2011年9月20日|work=喵喵|publisher=新浪网|language= zh-hans}}</ref>

[[莫言|莫言]]自谓：“《蛙》是裹在皮袍里的小我。《蛙》的这部小说也是我自己对人生几十年的回顾，在很多冠冕堂皇的借口之下掩藏着很多个人的私念和私心。《蛙》有一个重要的意义，这个重要意义是面对自我，是面对自己的内心进行挖掘，是一种自我批评的精神。”<ref>{{cite web |url= http://book.sina.com.cn/news/a/2011-09-18/2330291059.shtml|title=实录：莫言称《蛙》是自己对人生几十年的回顾 |accessdate=2011年9月18日 |work=|publisher=新浪网 |language= zh-hans}}</ref>
