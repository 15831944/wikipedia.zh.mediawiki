'''佳冬車站'''位於[[屏東縣|屏東縣]][[佳冬鄉|佳冬鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[屏東線|屏東線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[島式月台|島式月台]]一座。

== 利用狀況 ==
*目前為甲種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，為[[屏東線|屏東線]]之重要交會車站。
*停靠狀態：
**[[臺鐵區間車|區間車]]：皆停靠（[[復興號列車|復興號]]車廂編組或是單組[[臺鐵柴聯車|柴聯車]]編組）。
**[[臺鐵區間快車|區間快]]：皆停靠(2組柴聯車編組)
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1940年7月1日：設站。
*1944年：[[林邊車站|林邊站]]＝[[枋寮車站|枋寮站]]間因戰時需要而拆除。
*1953年1月16日：復舊。
*2003年7月16日：由三等站降為甲種簡易站，並指定由[[枋寮車站|枋寮站]]管理。
*2009年8月：[[八八水災|八八水災]]造成林邊溪橋受損，[[林邊車站|林邊車站]]遭厚實的泥沙覆蓋，導致列車只能行駛至[[南州車站|南州車站]]。
*2009年12月30日：恢復停駛路段的鐵路運輸。
*2016年1月1日：啟用多卡通刷卡機。
