'''佳冬神社'''是[[臺灣日治時期|臺灣日治時期]]設於[[高雄州|高雄州]][[東港郡|東港郡]][[佳冬庄|佳冬庄]]（今[[屏東縣|屏東縣]][[佳冬鄉|佳冬鄉]]）的[[神社|神社]]。

== 歷史 ==
佳冬神社選址於[[高雄州|高雄州]][[東港郡|東港郡]][[佳冬庄|佳冬庄]]佳冬{{r|金2}}{{r|蔡}}（今[[屏東縣|屏東縣]][[佳冬鄉|佳冬鄉]]佳和路與兩旁民宅）。耗資9800圓，於昭和11年（1936年）動工，同年4月13日鎮座{{r|蔡}}。[[近代社格制度|社格]]為無格社{{r|金2}}。祭祀[[天照大神|天照大神]]、[[開拓三神|開拓三神]]（[[大國主命|大國主命]]、[[大己貴命|大己貴命]]、[[少彥名命|少彥名命]]）與[[北白川宮能久親王|北白川宮能久親王]]，例祭日11月3日{{r|蔡}}{{r|金2}}。

戰後神社廢止。參道兩側蓋起民宅，原神社建築亦逐漸消失{{r|金3}}。2012年以"佳冬鄉神社"之名公告為歷史建築<ref>{{Cite web |url =https://nchdb.boch.gov.tw/assets/overview/historicalBuilding/20120924000001|title = 歷史建築 佳冬鄉神社|author =國家文化資產網|publisher =文化部文化資產局|language =中文（台灣）}}</ref>。

== [[神社建築|神社建築]] ==
佳冬神社由四座[[鳥居|鳥居]]、神橋、[[石燈籠|石燈籠]]數座、手水舍、[[石獅子|狛犬]]一對以及拜殿、本殿、祭器庫、社務所等建物所組成，從第一鳥居一直到最深處，大約長140公尺{{r|蔡}}。其中鳥居為神明系{{r|蔡}}之靖國鳥居，並與神橋、燈籠等同為混凝土製。拜殿為神明造{{r|蔡}}，本殿也是<ref>{{Cite web |url = http://justabalcony.blogspot.tw/2017/09/just-old_15.html |title = 屏東-佳冬 佳冬神社，內文第3張圖可見神明造拜殿與本殿|author =JUST A BALCONY|date = 2017-09-15 |language = 中文（台灣）}}</ref>。

== 現況 ==
神社原址目前仍留有鳥居、正殿基座等遺跡在{{r|蔡}}。包含成為佳和路的原參道路徑（其中原位於二之鳥居前的一小段階梯也隨之消失<ref>{{Cite web |url = http://justabalcony.blogspot.tw/2017/09/just-old_15.html |title = 屏東-佳冬 佳冬神社，內文第2、3張圖，皆可見於二之鳥居前曾有一段樓梯|author = JUST A BALCONY |date = 2017-09-15 |language = 中文（台灣）}}</ref>）。以及與民宅相溶的一之鳥居、神橋、僅存一邊柱子的二之鳥居、僅存一邊龜腹（鳥居底座）的三之鳥居<ref>{{Cite web |url = https://www.google.com.tw/maps/@22.4115847,120.5624106,3a,15y,128.46h,74.81t/data=!3m6!1e1!3m4!1sgpzu0zEJZ8jY12GC-dWu4A!2e0!7i13312!8i6656?hl=zh-TW |title = 街景中的三之鳥居龜腹|date = 2015-07 |publisher = google地圖 |language = 中文（台灣） }}</ref>、僅存基壇的[[石燈籠|燈籠]]數基（另有[[石燈籠|燈籠]]構件數枚散落於參道兩旁民宅內）、部分隱沒入柏油路面的拜殿基壇<ref>{{Cite web |url = https://www.google.com.tw/maps/@22.411495,120.5624705,3a,15y,346.49h,79.09t/data=!3m6!1e1!3m4!1sWYZGasDNQP4a-6MnmEXRDw!2e0!7i13312!8i6656?hl=zh-TW |title =街景中的拜殿磨石子基壇 |date = 2015-07|publisher =google地圖 |language = 中文（台灣） }}</ref>、部分玉垣、本殿基壇和駁坎等{{r|金1}}。另有狛犬1對移到[[國立佳冬高級農業職業學校|國立佳冬高級農業職業學校]]{{r|金1}}<ref>{{Cite web |url = http://www.sinkasiraya.com/2015-everyday-shrine-jiadong-kintama/ |title = #22 日常 神社X金玉滿堂，內文第2、3張圖是現位於佳冬高農的狛犬近照 |author = Siraya |date =2015-08-01 |publisher = 半月城津 |language = 中文（台灣） }}</ref><ref>{{Cite web |url = https://www.google.com.tw/maps/@22.4126435,120.5639285,3a,15y,3.86h,94.01t/data=!3m6!1e1!3m4!1sa7MYhxwoLkHQ5rl5gDq6Xg!2e0!7i13312!8i6656?hl=zh-TW|title = google地圖街景中現位於佳冬高農人文科技大樓1樓教務處與實習處間川堂的原佳冬神社狛犬|date =2015-07|language =中文（台灣）}}</ref>。
