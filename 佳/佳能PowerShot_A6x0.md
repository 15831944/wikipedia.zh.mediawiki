'''佳能 PowerShot A6x0'''指的是[[佳能|佳能公司]]于2005年至到2007年之间推出的5款[[数码相机|小型数码相机]]。

这5款相机分别是：
*'''Canon PowerShot A610'''
*'''Canon PowerShot A620'''
*'''Canon PowerShot A630'''
*'''Canon PowerShot A640'''
*'''Canon PowerShot A650 IS'''

A6x0在其市场存活期内，扮演[[Canon_PowerShot|佳能PowerShot]]A系列数码相机中的高端产品，在参数上比较接近同为PowerShot 的G系列相机（此系列定位准专业应用）。
== Canon PowerShot A610 / A620 ==
[[File:Canon_PowerShot_A610.jpg|right]]

'''Canon PowerShot A610'''与'''Canon PowerShot A620'''于[[2005年8月|2005年8月]]由佳能公司一同发布。

A610与A620功能基本一致，像素比A620略低。

'''PowerShot A610''' / '''PowerShot A620'''是[[Canon_PowerShot_A95|Canon PowerShot A95]]的继任者。'''A610 / A620'''的继任者是'''Canon PowerShot A630 / A640'''。

== Canon PowerShot A630 / A640 ==
[[File:Canon_PowerShot_A640_img_0782.jpg|right]]

'''Canon PowerShot A630 / A640'''是[[佳能|佳能]]出品的[[数码相机|数码相机]]，于[[2006年8月|2006年8月]]发布。

'''A630 / A640'''的前任是'''A610 / A620'''。

佳能在A640上，第一次将千万[[像素|像素]]引入[[Canon_PowerShot|PowerShot]]系列[[数码相机|数码相机]]。虽然相机成像质量好坏不能单纯凭借像素指标，但是“千万像素”确实是一个足够吸引人的“广告数字”。

== Canon PowerShot A650 IS ==
==破解／改机==
PowerShot A6x0系列拥有较大尺寸的CCD，为优质成像提供了保证。有爱好者研究出针对[[DIGIC|DIGIC II]]的（涵盖 A6x0 及S2 IS与S3 IS）的破解而使其能输出RAW格式图像并拥有更多功能如实时直方图等。这使A6x0变得更加具有吸引力。
