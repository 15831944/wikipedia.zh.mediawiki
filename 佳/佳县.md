'''佳县'''（旧作“'''葭县'''”，1964年改为“佳县”<ref>http://dmw.gov.cn/newslist.asp?id=3552{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>）是[[中華人民共和國|中華人民共和國]][[陕西省|陕西省]][[榆林市|榆林市]]所辖的一个[[县|县]]。总面积为2144平方公里，2002年人口为25万人。

== 历史 ==
[[北宋|北宋]]置[[晋宁军|晋宁军]]。[[金朝|金朝]][[大定|大定]]二十四年（1184年）改[[葭州|葭州]]，辖[[神木|神木]]、[[府谷|府谷]]、[[吴堡|吴堡]]、葭县四县。[[民国|民国]]二年（1913年）由州改为县。1964年，因“葭”字生僻，改为佳县。

1947年，[[第二次国共内战|第二次国共内战]]期间，[[毛泽东|毛泽东]]率领[[中共中央|中共中央]]及[[中国人民解放军|中国人民解放军]]总部，转战[[陕北|陕北]]来到葭县，生活、战斗了98天。期间，毛泽东在[[梁家岔|梁家岔]]的土[[窑洞|窑洞]]里指挥了著名的[[沙家店战役|沙家店战役]]。

== 地理 ==
佳县县城建在[[黄河|黄河]]与[[佳芦河|佳芦河]]半包围起来的一个山巅之上，隔黄河与山西相望，县城三面均为峭壁，旧城池临崖而建，以石为墙，险峻无比，是一座著名的石头城。

== 文化 ==

佳县方言属于[[晋语|晋语]][[吕梁片|吕梁片]][[汾州小片|汾州小片]]。

== 古迹 ==
佳县[[白云山庙|白云山庙]]是[[中国西北地区|中国西北地区]]最大的[[道教|道教]]建筑群，2001年公布为[[全国重点文物保护单位|全国重点文物保护单位]]，每年农历四月上旬在此举办的庙会吸引附近的晋蒙陕十多个县群众参加。
