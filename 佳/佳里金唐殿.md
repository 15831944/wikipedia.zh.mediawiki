'''玉勅皇勅金唐殿'''，俗稱佳里金唐殿，初稱「代天府」，承蒙[[乾隆皇帝|清朝乾隆皇帝]]勅封金唐殿，位於[[佳里區|臺灣臺南市佳里區]]，主祀玉勅代天巡狩'''朱府千歲'''、'''雷府千歲'''、'''殷府千歲''' <ref name=" 南瀛古廟誌 ">{{cite book|author= 南瀛古廟誌 |coauthors= |title=凃順從|pages= |year=1994年|publisher=臺南縣立文化中心 }}</ref>，民間尊稱三老爺、蕭壠王，為[[臺灣|臺灣]]民間盛行的[[王爺千歲信仰|王爺千歲信仰]]代表之一，為朱府大千歲、雷府二千歲、殷府三千歲三老爺系統祖廟。

逢[[子_(地支)|子]]、[[卯|卯]]、[[午|午]]、[[酉|酉]]年[[農曆|農曆]][[正月|正月]]中旬行主辦[[蕭壠香|蕭壠香]]( 南瀛三大古香、南瀛七大香)。
[[佳里區|蕭壠地區]]聚落、經濟、信仰中心，為[[佳里金唐殿#轄境廟宇|十七角頭廿四村-{庄}-]]大廟。經核定為直轄市定古蹟<ref name=" 南瀛古廟誌 ">{{cite book|author= 南瀛古廟誌 |coauthors= |title=凃順從|pages= |year=1994年|publisher=臺南縣立文化中心 }}</ref><ref name=" 南瀛刈香誌 ">{{cite book|author= 南瀛刈香誌|coauthors= |title=黃文博|pages= |year=1993年|publisher=臺南縣立文化中心 |location=臺南市 |isbn=}}</ref>，名列南瀛百景之一。與[[保西代天府|歸仁保西代天府]]、[[南鯤鯓代天府|北門南鯤鯓代天府]]，稱為南瀛古三大代天府。


==沿革==
===初建時期===

金唐殿位於今[[佳里區|佳里區]]中山路旁，兩邊商家林立，對面即為「東市場」或俗稱「舊市場」的佳里菜市場，是[[佳里區|蕭壠地區]]繁榮地帶。[[明朝|明朝]]末年間(西元16世紀末期)，來此移民墾殖或經商者日益擴增，為祈求合境平安及驅除瘟疫，於東勢角一帶，建一草寮安奉隨船帶來家鄉神明，為金唐殿前身，初名「東安宮」，與[[西拉雅族|西拉雅族]][[阿立祖|阿立祖]]為[[佳里區|蕭壠社]]共同信仰，[[永曆_(南明)|明朝永曆]]十八年(西元1664年)[[明鄭時期|明鄭時期]]施行寓兵於農的政策，現今[[金唐殿|金唐殿]]廟前中山路到新生路一帶，曾有一條由北而南轉西流入大海之小溪，溪流兩側一帶即為[[明鄭時期|明鄭]]鎮兵[[林可棟|林可棟]]的屯墾地。[[康熙|清朝康熙]]三十七年（西元1698年），[[林可棟|林可棟]]捐獻現所在風水寶地興建新廟，將神尊移至新廟奉祀，廟名為「護庇宮」，並獻上匾額「宏文求莫」<ref name=" 南瀛古匾誌 ">{{cite book|author= 南瀛古匾誌|coauthors= |title=曾曉馨、曾絮敏|pages= |year=2009年|publisher=臺南縣政府文化局 }}</ref> ，廟前為市場，廟後為學堂，至此成為[[佳里區|蕭壠地區]]經濟貿易及文教中心，逐漸興盛起來。

===玉勅代天府===

早期[[金唐殿|金唐殿]]建廟之初，原主祀為金元帥(金府千歲)、方娘娘(方府娘娘)、觀音佛祖、蕭王爺（蕭府千歲）。[[康熙|清朝康熙]]年間，載有玉勅代天巡狩的王船從[[福建|中國福建]]漂來，停著於蕭壠地區而被撿到，-{庄}-民逐恭請玉勅代天巡狩，巡按人間，監察善惡，掃除瘟疫邪煞，考核轄境神明，訪查民瘼，燮理陰陽，按察出巡祈求合境平安。從此為了防止天災地變及各式危害以保護安全，舉辦盛大刈香醮典活動。並指示-{庄}-民，有朱王（大王）、雷王（二王）、殷王（三王）三尊天神[[玉皇上帝|奉旨]]下凡鎮守蕭壠，並前往到當時規模最大的[[臺灣縣_(1684年-1887年)|臺灣縣]][[保大西里|保大西里]]'''大人廟代天府'''(今[[歸仁區|歸仁]][[保西代天府|保西代天府]])迎請回[[金唐殿|金唐殿]]奉祀，成為鎮殿主神。

===皇勅金唐殿===

[[乾隆|清朝乾隆]]年間，於[[林爽文事件|林爽文事件]]中，協助[[欽差大臣|欽差大臣]][[福康安|福康安]]官軍平亂有功，[[乾隆皇帝|乾隆皇帝]][[詔|恩旨]]御賜「旌義」匾，賜名蕭壠為「旌義」<ref>《臺案彙錄庚集》，「廣東、泉州等莊義民...業經降旨賞給「褒忠」、「旌義」里名匾額，以示獎勵。」</ref>，以示獎勵。興建「義民亭」，勒石立「旌義碑」，以垂不朽。賜「下馬碑」，文官下轎、武官下馬，以示尊崇。勅封為「金唐殿」，以顯神威。因受[[玉皇上帝|玉帝]]及[[乾隆皇帝|皇帝]]勅封，於三川殿、正殿廟脊及正殿神龕使用[[五爪金龍|五爪金龍]]之[[王爺廟|王爺廟]]<ref name=" 南瀛古廟誌 ">{{cite book|author= 南瀛古廟誌 |coauthors= |title=凃順從|pages= |year=1994年|publisher=臺南縣立文化中心 }}</ref><ref name=" 南瀛刈香誌 ">{{cite book|author= 南瀛刈香誌|coauthors= |title=黃文博|pages= |year=1993年|publisher=臺南縣立文化中心 |location=臺南市 |isbn=}}</ref>，豎立大旗杆，以示仕宦之身。

===日治時期===

[[台灣日治時期|日本明治三十年]](1897年)十月一日 創蕭壠公學校(今[[佳里國小|佳里國小]])，於金唐殿後殿做臨時教室，招收六年制學生一班。

[[台灣日治時期|日本明治]]年間在經歷[[蕭壟事件|蕭壠事件]]後，時任[[北門郡|北門郡守]]酒井正之，追荐死難靖安，配合地方耆老及保正，以及金唐殿神明指示-{庄}-民下，前往迎請[[北港朝天宮|北港朝天宮天上聖母]](北港媽祖)及[[南鯤鯓代天府|南鯤鯓代天府吳府三千歲]](南鯤鯓吳三王)來[[佳里區|佳里-{庄}-]]協助金唐殿朱府、雷府、殷府千歲安撫邪煞，在經觀四輦的儀式，代天巡狩，撫慰亡靈，並整併集中各地有應公祠，部分安置於第十五角廣安宮將軍府(民間忠烈祠)，並奏請[[玉皇上帝|玉旨]]勅封孔德尊王(樹王公)坐鎮廣安宮，統領這些有應公，以供人參拜致祭，部份安葬於廟旁約兩公里處的群塚裡，約在[[大正|大正]]十一年(1922年)間完成，自此[[佳里區|佳里-{庄}-]]才又繁榮起來。此後，[[佳里區|佳里-{庄}-民]]為感念[[北港朝天宮|北港朝天宮天上聖母]]及[[南鯤鯓代天府|南鯤鯓代天府吳府三千歲]]當時前來協助[[佳里區|佳里-{庄}-]]的恩情，每逢金唐殿主神聖誕之際，由金唐殿神明乘座八抬轎前往迎請[[北港朝天宮|北港朝天宮天上聖母]]及[[南鯤鯓代天府|南鯤鯓代天府吳府三千歲]]來作客看戲，以答謝當時協助之恩。

===民國時期===

[[民國|民國]]七十四年（西元1985年），金唐殿因保存[[台灣日治時期|日本昭和]]時期聘汕頭[[剪瓷雕|剪粘名匠]][[何金龍|何金龍]]之完整作品等，極具有歷史、文化、藝術保存價值 ，經內政部評定將金唐殿列入[[台灣古蹟列表|國定三級古蹟]]<ref name=" 南瀛古廟誌 ">{{cite book|author= 南瀛古廟誌 |coauthors= |title=凃順從|pages= |year=1994年|publisher=臺南縣立文化中心 }}</ref><ref name=" 南瀛刈香誌 ">{{cite book|author= 南瀛刈香誌|coauthors= |title=黃文博|pages= |year=1993年|publisher=臺南縣立文化中心 |location=臺南市 |isbn=}}</ref>，縣市合併後為直轄市定古蹟。

==建築特色==

*建築格局為三開間三進型式，正面三川殿，並配以左右護龍，左廂為義民亭(現為辦公室)，右廂為慈善會辦公室。護龍正面作西洋式牌樓，三川殿屋脊屬潮州風格，其特色在燕尾尖倒勾作捲螺，屋脊及規帶線腳繁複，背身曲度和緩。整體建築構造屬於傳統式木結構及承重磚牆組合，廟埕前立木製大旗竿一對。

*金唐殿現今廟貌規模於[[咸豐_(年號)|清朝咸豐五年]]年間(1855年)奠定，當時狹隘殿堂己不敷日益增多的信徒，由總理(現今主任委員)楊成美、陳文秀、[[舉人|舉人]]陳德言等首倡重建，禮聘嘉義[[交趾陶|交趾陶]]名匠[[葉王|葉王]]重建[[金唐殿|金唐殿]]。[[昭和|日本昭和]]三年（1928年）時任[[佳里區|佳里-{庄}-長]]黃深淵復倡修廟，禮聘府城名畫家[[潘春源|潘春源]]([[潘麗水|潘麗水]]之父)來彩繪及畫門神，經[[潘春源|潘春源]]介紹下，禮聘[[剪瓷雕|汕頭剪黏名匠]][[何金龍|何金龍]]來主持[[金唐殿|金唐殿]]尪仔堵及屋頂裝飾的整修工作。[[何金龍|何金龍]]在黃深淵的首肯下，將[[葉王|葉王]]的交趾陶盡數汰換，從內外壁堵、屋脊到筏仔頭，從三川步口、正殿到後牆，全部改成剪黏。其[[剪瓷雕|剪黏]]作品人物比例精準、造型變化生動、姿勢表情豐富、色彩華美、作工細緻、用材自由，將剪黏技術發揮得淋漓盡致，並留下豐富完整的剪黏珍品留傳至今，然昔日交趾陶遺蹟，已不復見。[[何金龍|何金龍]]更造一[[孫中山|孫中山先生]]塑像剪粘於右壁前上方。
*民國45年（1956年）金唐殿重修，禮聘[[何金龍|何金龍]]得意門生王石發（何金龍傳人）、王保原（國寶剪黏藝師、何金龍傳人）父子補修剪黏、[[潘春源|潘春源]]彩繪油畫、[[施天福|施天福]](任[[三峽祖師廟|三峽祖師廟]]石雕總工頭)雕刻龍柱石壁，所有匠師均一時之選，作品雋永絕世，曾任[[立法院|立法院長]][[黃國書|黃國書]]、[[考試院|考試院長]][[賈景德|賈景德]]及[[司法院|司法院長]][[王寵惠|王寵惠]]均楹聯題書。

== 轄境廟宇 ==
*'''十七角頭廿四-{庄}-'''<ref name=" 南瀛古廟誌 ">{{cite book|author= 南瀛古廟誌 |coauthors= |title=凃順從|pages= |year=1994年|publisher=臺南縣立文化中心 }}</ref><ref name=" 金唐殿之美 ">{{cite book|author= 金唐殿之美|coauthors= |title=張明忠|pages= |year=2008年|publisher=佳里玉勅皇勅金唐殿 |location=臺南市 |isbn=}}</ref>

===十七角頭===

蕭壠角頭起源說法有數種，其第一種是是蕭壠因撿拾到王船，而舉辦王船醮典，境內的十七角頭便依抽籤順序來決定遶境的順序；其第二種[[乾隆|清朝乾隆]]年間，[[林爽文事件|林爽文事件]]時，蕭壠-{庄}-的義民以角頭的名義協助清朝官軍平亂，以出兵的先後順序來排定地方角頭的順序。在管理委員會尚未成立之前，境內的十七角頭在過去則負起金唐殿的祭典工作<ref name="三1">{{cite book |title= 三級古蹟佳里金唐殿  |author=刈香轄境 }}</ref>。
