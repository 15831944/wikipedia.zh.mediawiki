'''佳里區'''位於[[臺灣|臺灣]][[臺南市|臺南市]]西部，北鄰[[學甲區|學甲區]]，西鄰[[將軍區|將軍區]]，西南連[[七股區|七股區]]，東鄰[[麻豆區|麻豆區]]，南接[[西港區|西港區]]。[[2010年中華民國縣市改制直轄市|縣市合併]]前為全縣人口最多的[[鎮_(中華民國)|鎮]]。全境地勢平坦，海拔約5.5公尺，境內有[[將軍溪|將軍溪]]與七股溪流經。

本區是[[北門郡|北門地區]]鹽分地帶的門戶及政經中心。[[台灣日治時期|日治時期]]以來出了許多[[鹽分地帶|鹽分地帶]]文學家及詩人，被視為臺灣文學重鎮。此外也是[[西拉雅|西拉雅]]系平埔族蕭壠社的聚居地，為臺灣最早的西拉雅文化發源地之一。代表作物為[[牛蒡|牛蒡]]，有「牛蒡的故鄉」之美譽。本區足球風氣鼎盛，有四級足球，均曾獲全國冠軍。

== 歷史 ==
佳里區舊稱「蕭壠」或「蕭壟」，為[[平埔族|平埔]]原住民[[西拉雅族|西拉雅族]]社名「Saulang」之音譯，意為「契約之地」。昔日蕭壠社與新港社（今新市）、目加溜灣社（今善化）、麻荳社（今麻豆）共稱平埔族四大社。<ref>[http://web2.tainan.gov.tw/Jiali/CP/10667/history-1.aspx 《關於佳里鎮》，臺南縣佳里鎮公所官網] {{webarchive|url=https://web.archive.org/web/20100429075849/http://web2.tainan.gov.tw/Jiali/CP/10667/history-1.aspx |date=2010-04-29 }}</ref>「蕭壠（Soulang）」一語有「河川曲流處」的意思，剛好可以說明這個社的特色：位於[[曾文溪|曾文溪]]下游，瀕臨河岸亦靠近海口，是各社中最依賴魚產的村子。<ref>[http://www.books.com.tw/exep/prod/booksfile.php?item=0010391773 《舊文獻.新發現 台灣原住民歷史文獻解讀》]{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>[[荷治時期|荷治時期]]荷蘭人曾授予蕭壠社民牛隻使用，為第一批引進台灣的牛隻。另外荷蘭人為了在地的基督教信仰教育，1659年也在此設立東印度地區第一間以當地人為招募對象的「蕭壠神學院」，這也成為了臺灣第一所神學院。<ref>[http://culture.tainan.gov.tw/form/index-1.php?m2=252&id=1597 《蕭壟神學院》 - 臺灣第一在臺南]，臺南市政府文化局</ref>

明鄭時期，鄭成功死亡之後，鄭襲和鄭經因為繼位問題爆發戰爭，雙方軍隊就此在佳里區交戰。

[[乾隆|清朝乾隆]]年間，[[林爽文事件|林爽文事件]]中，蕭壠居民協助[[清朝|清朝]]官兵平亂有功，[[乾隆皇帝|乾隆皇帝]][[詔|恩旨]]，賜名蕭壠為'''「旌義」'''，以示獎勵。<ref>《臺案彙錄庚集》，「廣東、泉州等莊義民...業經降旨賞給「褒忠」、「旌義」里名匾額，以示獎勵。」</ref>

古代的佳里區位於濱海的[[蕭壠半島|蕭壠半島]]，西南靠[[臺江內海|臺江內海]]，北臨古曾文溪（今將軍溪），當時的蕭壠市街與[[佳里興|佳里興]]尚隔溪分處南北。十九世紀中葉古曾文溪改道、臺江陸浮後，佳里區才位於內陸至今。

本區在[[明鄭時期|明鄭時期]]隸屬[[天興縣|天興縣（州）]]下之[[永定里_(堡里)|永定里]]及[[開化里_(堡里)|開化里]]，到了[[台灣清治時期|清治時期]]則隸屬[[臺灣府|臺灣府]][[諸羅縣|諸羅縣]]，1704年以前一直為縣治所在；1787年後諸羅改名嘉義，繼續隸[[嘉義縣_(清朝)|嘉義縣]]管轄；清末1887年改隸[[臺南府|臺南府]]。1920年臺灣地方改制，因古縣治所在地[[佳里興|佳里興]]，而將蕭壠改名為「佳里」，並設置「佳里-{庄}-」，劃歸[[臺南州|臺南州]][[北門郡|北門郡]]管轄，為郡治；1933年12月升格為「[[佳里街|佳里街]]」。戰後改設為[[臺南縣|臺南縣]]佳里鎮，2010年12月25日改稱佳里區。

=== 蕭壟屠殺事件 ===
{{Unreferencedsect|time=2012-11-12T05:22:37+00:00}}
{{main|蕭壟事件}}
因日本某親王（據說是[[北白川宮能久|北白川宮能久]]）於臺南[[鹽水區|鹽水]]被暗殺，當時軍隊行經蕭壟，而導致於1895年農曆9月3日蕭壠居民被血腥屠殺，死亡人數據達兩千人。當時的生還者回憶：「事後，-{庄}-民前來收屍，整整裝十八輛牛車之多，由於死亡人數太多，連棺木都一棺難求。」因此佳里區在每年農曆九月三日，有祭拜因走番仔反而犧牲的祖先習俗。

1895年10月10日，日軍混成第四旅團登陸布袋嘴（嘉義地區），當地義軍領袖林昆岡，以敢死隊之勢捍衛區裡。然而武器窳劣不敵，退據蕭壟街（今臺南市佳里區）。於是日軍大肆搜索，民兵躲到溪邊雜樹林的天然溝壑中，因嬰兒哭聲而被發現後，日兵分別截住長坑的頭尾兩端，然後亂槍齊放，對著坑內猛烈射擊了近2小時，躲到坑裡避難的臺灣人無一幸免。

關於佳里的舊地名「蕭壠」，有傳說即認為是得自當地在事件中消失了很多人後而取的「消人」。由於早在1699～1704繪製之[[康熙臺灣輿圖|康熙臺灣輿圖]]即有出現蕭壟社的地名，因此此傳說不足採信。

== 行政區劃與主要聚落 ==
[[File:佳里中山路街屋.JPG|thumb]]旁）]]
本區共轄16里，因歷史與地緣關係可劃分為五個地區，每個地區設有一警察派出所，隸屬於[[臺南市政府警察局|佳里分局]]。1963年下廍里改名為嘉福里、1969年中元里改名為忠仁里。2018年鄰里整併，營頂里、溪州里合併為營溪里；禮化里、興化里合併為佳興里；頂廍里、嘉福里合併為下營里；通興里、龍安里、蚶寮里合併為塭內里；三協里併入子龍里；漳州里併入海澄里；安西里拆成安西里與文新里，且光復路以東部分併入東寧里；建南里拆成建南里與南勢里。

== 經濟 ==
本區在[[2010年中華民國縣市改制直轄市|縣市合併]]前為臺南縣5個五萬人口以上的鄉鎮市之一，僅次於北方的[[新營區|新營]]與南方的[[永康區|永康]]、[[仁德區|仁德]]、[[歸仁區|歸仁]]<ref>[http://ngis.nat.gov.tw/statistic/Document/pdf/99.pdf 《都市及區域發展統計彙編99年版》]，頁156。（國家發展委員會）</ref>，如此的人口也造就了商業機能的發達，市區約以文化路、延平路、新生路、光復路所包圍的區塊為核心區域，而中山路、光復路、延平路等主要的南北向街道沿新生路以南也有相當的繁榮度，金融機構等亦不在少數。鄰近學區的安西路一帶近年來則因黃昏市場興起、連鎖超市的進駐等因素帶動人潮的往來。近郊另有連鎖大型賣場[[大潤發|大潤發]]進駐，是臺南市中部地帶唯一的一間量販店<ref>自2014年[[全買大賣場|全買大賣場]]麻豆店關閉後，目前中臺南一帶僅有大潤發佳里店一間大賣場。</ref>；其他商店業者，如[[全聯福利中心|全聯福利中心]]、[[寶雅生活館|寶雅生活館]]、[[麥當勞|麥當勞]]、[[85度C|85度C]]、[[丹丹漢堡|丹丹漢堡]]、[[屈臣氏|屈臣氏]]、[[康是美|康是美]]、[[大樹藥局|大樹藥局]]等亦在本區設有連鎖分店。

本區的紡織業曾風光一時，但隨著產業外移等因素已趨近沒落。海澄里境內設有[[佳里工業區|佳里工業區]]（萊竿寮工業區），以輕工業為主。而臺南沿海地區的虱目魚產業在本區還設有加工工廠，虱目魚罐頭也被視為佳里的特產之一。農業部分除稻作等糧食作物之外，以胡蘿蔔、花卉、牛蒡為主要的經濟作物。

* [[高文瑞|高文瑞]]：戰後曾出任臺南北門家政學校、[[國立北門高級農工職業學校|北門農校]]、[[國立北門高級中學|北門中學]]校長、實施地方自治後當選[[臺南縣|臺南縣]]第一、二屆[[台南縣縣長|縣長]]，對地方建樹頗多。
* 戴再生：早年經營戴竹營造及康乃馨公司，事業有成後步入政壇，曾任縣議員、副縣長、議長等職，為民喉舌，熱心公益，積極爭取經費建設鄉里，深受民眾肯定。
* [[郭水潭|郭水潭]]：筆名郭千尺，1906年生，1995年去世，為[[鹽分地帶|鹽分地帶]]前輩作家，其作詩見長，有「世紀之歌」、「三等病室」、「向棺木慟哭」、「斑鳩與廟守」、「故區之歌」等，小說「某男人的手記」、「福爾摩沙」等。
* 林清文：1919年生，1978年去世，鹽分地帶前輩詩人及劇作家。著有新劇劇本「廖添丁」，重要詩作有「新生之歌」、「疼愛生命之日」、「給規矩男不二男」，其長篇自傳體小說「愚者自傳」曾在自之晚報副刊連載。 
* [[林芳年|林芳年]]：本名林精鏐，筆名李秋華，1941年生，佳里區子良廟人，為抗日宿儒林芹春長男。林芳年是橫跨兩代的臺灣新文學作家，也是戰後鹽分地帶創作力最旺盛的作家，能詩、小說及散文，亦能寫評論，算是全能作家。著有小說「失去歡樂的春天」、「情脈」、「森走的自由」、「瘦猿的人生」等。
* 楊順明：筆名羊子喬，1951年生，[[東吳大學_(台灣)|東吳大學]]中文系畢業，出版詩集「月浴」、「收成」、散文集「太陽手記」、「輪迴」，評論集「蓬萊文章台灣詩」、主編「鹽分地帶文學選」、「光復前臺灣文學」、新詩四卷。
* [[黃勁連|黃勁連]]：1947年生，[[國立嘉義大學|嘉義師專]]、[[中國文化大學|中國文化大學]]中系文藝組畢業。現任「蕃薯詩社」、總編輯，臺南市「傳統文教基金會」、「區城生活學苑」、閩南語班教席。編有<台譯千家詩>、<台譯唐詩三百首>等。
* [[林佛兒|林佛兒]]：筆名林白，1941年生專事小說、散文的創作，創立國內知名的「林白出版社」發行書籍數千種，著有「尋找春格里拉」、「南方的果樹園」、「北回歸線」等書。
* 王榮周：東吳大學法律系畢、[[國立中興大學|國立中興大學]]法律研究所碩士，曾任[[中華民國財政部|財政部]][[金融監督管理委員會證券期貨局|證管會]]主秘、財政部專門委員、[[財政部國庫署|國庫署]]長、臺北市國稅局長、[[法務部調查局|調查局]]長、財政部常務次長。
* [[蘇新|蘇新]]：日治時期[[臺灣共產黨|臺灣共產黨]]幹部，曾為臺灣共產黨「改革同盟」成員、臺灣共產黨中央委員。
* [[高育仁|高育仁]]：曾任第四屆[[台灣省議會|台灣省議會]]議員、第七任[[臺南縣|臺南縣]]縣長、第七、八屆台灣省議會議長、[[立法委員|立法委員]]，前任[[中國電視公司|中國電視公司]]董事長，兼任[[三三會|三三會]]常務監事。
* 陳秉輝：[[鄭成功|鄭成功]]的軍人。
* [[吳清基|吳清基]]：[[臺灣|臺灣]]學者，臺南市佳里區人，[[國立臺灣師範大學|國立臺灣師範大學]]教育博士，[[臺灣師範大學|臺灣師範大學]]教育系兼任教授。曾擔任教育部技職司長、常務次長、政務次長、[[臺北市|臺北市]]副市長、教育部部長等職。現任[[臺灣教育大學系統|臺灣教育大學系統]]總校長、[[龍華科技大學|龍華科技大學]]校務諮詢顧問。
* [[許強|許強]]：佳里[[佳里興|佳里興]]人，日治時期著名醫師，為內科學權威。
* [[吳耀庭|吳耀庭]]：[[大统集團|大统集團]]創辦人。

== 交通 ==
=== 公路 ===
[[佳里區|佳里區]]為[[臺南市|臺南市]]西側最發達熱鬧的市鎮之一，市區的交通路網甚為發達。原本由[[台19線|台19線]]（中山路、延平路）及[[市道176號|市道176號]]（進學路、新生路）在市區內構成主要十字路網中心，但隨市區的發展及交通流量需求，兩公路皆已改道行經市區東側的「佳東路」。「佳東路」為[[佳里區|佳里]]環市區道路中的一段，與「佳南路」、「佳西路」及「佳北路」構成完整的外環道路線。

除由外環道路及十字路線構成的道路基本架構外，[[佳里區|佳里]]由外環道路輻散出多條聯外公路，計有[[南19線|南19線]]（忠孝路）、[[南32線|南32線]]（文化路）、[[南37線|南37線]]（光復路、成功路）及[[南41線|南41線]]（中山路、勝利路）等四條，再加上外圍的[[南26線|南26線]]、[[南27線|南27線]]、[[南28線|南28線]]、[[南29線|南29線]]及[[南51線|南51線]]等五條轉接道路，讓[[佳里區|佳里]]無論往北到[[學甲區|學甲]]，往西北通達[[將軍區|將軍]]，或是連繫西南側的[[七股區|七股]]及東南側的[[西港區|西港]]、東側的[[麻豆區|麻豆]]，各個方向都有便利的公路可以到達<ref name="motc2018">{{cite web |title=公告調整臺南市區道公路路線。 |url=https://odmdoc.motc.gov.tw/IFDEWebBBS_MOTC/ExternalBBS.aspx?ThirdDocId=107RD05954 |publisher=中華民國交通部 |website=中華民國交通部公告發布令 |series=交路字第1070413619號 |accessdate=2019-04-12 |date=2018-11-21}}</ref>

==目前近況==
佳里區是近代最早發展的鄉鎮區之一，居住在此的以閩南人和部分外省人為主。不過由於人口外移加上人口老化的影響，所以現在居住在佳里區的輕中壯年齡層(20~50歲之間)的人口不算太多。加之以在佳里區要謀生實屬不大容易，導致年輕的人口往外遷移，這也造成了在臺灣各地的旅外臺南同鄉會成員年年增加。雖然在地方政府政策和觀光兩路安排下，在佳里區和附近的[[西港區|西港區]]、[[七股區|七股區]]、[[將軍區|將軍區]]及[[學甲區|學甲區]]的返鄉的人口雖略有增加，但補不上流失的速度。在佳里區主要的大姓是[[黃姓|黃姓]](六安里)和[[洪姓|洪姓]](六安里)，再次為[[陳姓|陳姓]]和[[翁姓|翁姓]]([[洪姓|洪姓]]和[[翁姓|翁姓]]以'''六桂堂'''為堂號)。
