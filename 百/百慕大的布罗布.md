'''百慕大的布罗布'''（英语：Bermuda Blob）是指1988年、1995年和1997年在[[百慕大|百慕大]]发现的[[大型神秘海洋生物屍體|神秘海洋生物尸体]]。

== 1988 ==
1988年，渔夫特迪·塔克（Teddy Tucker）在百慕大{{link-en|红树湾|Mangrove Bay}}发现一具白的纤维状神秘生物遗体，这是有记载的第一具百慕大地区的[[大型神秘海洋生物屍體|布罗布]]。1995年，一支研究团队的测试报告表明这可能是[[鲨鱼|鲨鱼]]的遗体。<ref>{{Cite web|url=http://www.lchr.org/a/36/gf/dubiousglobsters.html|title=Dubious Globsters|accessdate=2017年11月11日|author=|date=|publisher=lchr}}</ref>

== 1995 ==
1995年，百慕大海滩再度发现一具神秘海洋生物尸体，它被命名为“Bermuda Blob2”。在2004年的分析结论中，“Blob 2”被证实是[[鯨|鲸鱼]]的脂肪组织。<ref>{{Cite web|url=http://www.cryptopia.us/site/2010/01/bermuda-blob-bermuda/|title=BERMUDA BLOB: (BERMUDA)|accessdate=2017年11月1日|author=Rob Morphy|date=2010年1月13日|publisher=cryptopia}}</ref>

== 1997 ==
1997年1月，百慕大海滩发现第三具神秘海洋生物尸体。在2004年，“Bermuda Blob 3” 同样被证实是鲸鱼的脂肪组织。<ref>{{Cite web|url=https://forteanzoology.blogspot.co.uk/2011/03/michael-newton-blob-that-never-was.html|title=MICHAEL NEWTON: A “Blob” That Never Was|accessdate=2017年11月11日|author=Jon Downes|date=2011年3月20日|publisher=blogspot}}</ref>
