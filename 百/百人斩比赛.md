'''百人斬比賽'''是指由1937年11月底至12月10日，兩名日本軍官[[向井敏明|向井敏明]]少尉和[[野田毅_(陸軍軍人)|野田毅]]少尉在日軍從上海進攻南京直至[[南京大屠殺|南京大屠殺]]前夕途中，以先殺滿100個中国士兵者為勝的競賽。[[野田毅_(陸軍軍人)|野田毅]]殺了105人，[[向井敏明|向井敏明]]殺了106人。又因不確定是誰先達到殺100人之數，決定這次比賽不分勝負，重新比賽誰殺滿150名中國軍。

1945年日本投降後，參加東京[[遠東國際軍事法庭|遠東國際軍事法庭]]對日戰犯審判的中國代表[[高文彬|高文彬]]在已被盟军封存的日军档案中無意間發現了這個報導，於是立即通知中國[[南京|南京]]，兩名日本軍官在1947年（昭和22年）9月2日、被[[盟軍最高司令官總司令部|盟軍最高司令官總司令部]]（GHQ）逮捕。後被引渡回中國，二人经[[南京军事法庭|南京军事法庭]]查明後被判處死刑，于1948年1月28日在南京中华门外雨花台刑场执行枪决，两人留下的[[遺書|遺書]]中稱只有戰場殺敵，絕无杀害[[俘虜|俘虜]]和[[平民|平民]]的行爲，并写有“中国万岁，日本万岁”，祈願[[中日關係|中日關係]]永遠和平的字样。

其他類似的百人斬事件還有日军[[第6师团|第6师团]][[上尉|上尉]][[田中军吉|田中军吉]]的一人砍杀300人事件。

== 1937年的報導 ==
<!--[[File:Contest_To_Cut_Down_100_People.jpg|thumb]]》有關殺人競賽的報導标题为：“百人斩，大接战，勇壮向井、野田两少尉”，“百人斩，超纪录，向井106——野田105，两少尉延长战”，照片中從右起為[[野田毅|野田毅]]少尉與[[向井敏明|向井敏明]]少尉]]-->
1937年11月30日至12月11日，日军[[第16师团|第16师团]]步兵19旅团第9联队第3大队的两个少尉军官[[野田毅_(陸軍軍人)|野田毅]]、[[向井敏明|向井敏明]]，在从上海向南京进攻的白刃战战斗中斩杀了国军士兵。《[[东京日日新闻|东京日日新闻]]》（即现在的《[[每日新闻|每日新闻]]》），连续刊登该报四名随军记者浅海、光本、安田、铃木分别从中国江苏省常州、丹阳、句容、南京等地发回的现场报道，详细报道了此二人在无锡横林镇、常州车站、丹阳奔牛镇、吕城镇、陵口镇、句容县城、南京紫金山等地刀劈百余人的经过。这些报道不仅仅时间、地点明确，杀人过程及数字清楚，而且同时还配发了照片。 

《[[东京日日新闻|东京日日新闻]]》于1937年11月30日首次报道了[[向井敏明|向井敏明]]少尉和[[野田毅_(陸軍軍人)|野田毅]]少尉進行百人斩竞赛的消息：

{{quote||『东京日日新闻』报道记事

1937年11月30日的日报（第1报）

百人斩竞争！两少尉，很快已经达到80人

*[廿九日浅海、光本、安田特派员发于常州] 用了六日时间踏破常熟、无锡间四十公里的○○部队在同一距离的无锡、常州之间也只用了三日时间就快速突破了。真正的神速，快攻。在最前线的片桐部队裏有两名青年将校发起「百人斩竞争」。在从无锡出发后很快就一人斩了五十六人，另一人斩了二十五人。一人是富山部队向井敏明少尉（二十六岁）＝山口县玖珂郡神代村籍贯。另一人是同部队的野田毅少尉（二十五岁）＝鹿儿岛县肝属郡田代村籍贯。在剑道三段向井少尉腰间的是一把名为「関の孫六」的名刀，野田少尉的刀虽无名但却是一把祖上传下的宝刀。

*在向无锡进发后向井少尉随着二十六、七公里的铁道线做大移动，野田少尉沿着铁道线前进。两个人暂时分开，在出发后的第二天早上野田少尉在距无锡八公里的无名村处冲进敌人的碉堡斩了4名敌人先扬威名。听到了这个消息后向井少尉奋然而起，在当天晚上冲进横林镇的敌阵和部下一起斩了五十五名。

*在这以后野田少尉在横林镇九名，威关镇六名、廿九日常州车站六名、合計斩了二十五名、向井少尉在这之后常州车站附近斩了四名。记者等在到车站的时候看到他们两人在站头会面的光景。

*向井少尉：照这么下去别说去南京了，到丹阳的时候我就可能斩了100名左右了，是野田的失败呀，我的刀砍了五十六人只有一个缺口。

*野田少尉：我们两人都不砍逃跑的人，我又是个副官所以成绩上不去，到丹阳之前一定创下大记录给你看。

}}

==审判取证背景==
当时国际形势急剧变化，昔日的[[反法西斯|反法西斯]]盟友[[美國|美國]]、[[蘇聯|蘇聯]]已渐形成对峙，[[美國政府|美國政府]]已不大願意整肅舊日本軍官，且[[美國人|美國人]]尊重法治，特別重視[[證據|證據]]，因此在法庭聽證十分嚴謹，國民政府苦於[[国共内战|国共内战]]，粗心大意，胡亂[[举证|举证]]，法庭召問国民政府[[军政部|军政部]]次长[[秦德纯|秦德纯]]，要求秦提出南京大屠殺的証言，秦卻空泛地說，日军“到处杀人放火，无恶不作”，被[[远东国际军事法庭|远东国际军事法庭]]斥为空洞无据，胡說八道，[[莫須有|莫須有]]，几乎被轰下证人台，这也为战后這兩位涉案軍官的遺族翻案提供了机会。

== 1971年的報導 ==
《[[朝日新聞|朝日新聞]]》記者[[本多勝一|本多勝一]]於1971年多次赴中國實地調查，完成報導《中國之旅》，並於當年8月至12月在《朝日新聞》上連載。這篇報導記錄了[[南京大屠殺|南京大屠殺]]及「百人斬」等屠殺事件。

===《中国》月刊的报道===
[[志志目彰|志志目彰]]（1971年时任职于中央劳保组织推进部）在杂志《中国》1971年12月刊的文章中回顾，百人斩报道刊登出来的一年又四个月后，野田回到故乡小学时，曾经直接对他说过
<ref>月刊《中国》1971年12月号，转引自本多勝一著《南京への道》“百人斬り‘超記録’”，東京，朝日新聞社1987年4月30日第4次印刷版，第130页</ref>：
{{Quote box |align = center| width = 80%| quote =乡土出身的勇士啦，百人斩竞赛的勇士啦，报上写得都是我。……实际突击中杀的只有四、五人，……对着占领了的敌人的战壕，“你来，你来”叫着，支那兵都是傻瓜，渐渐的都出来过来了。让他们排着，然后左一个，右一个斩……得到“百人斩”的评价，实际几乎都是这样斩的……|salign= right |sstyle = font-size:85% |source = ——《中国》月刊1971年12月号}}

大屠杀虚构派人士据此认为，野田为了所谓“大勇武”而进行杀人比赛是虚构的。大屠杀实证派人士则据此认为，杀俘虏以达到百人更符合真实情况，说明杀人比赛是真实的；而且证明杀人比赛并不是在战争行为中杀死敌军士兵，而是杀死已经投降的战俘，更加有违伦理。<ref name="hzq">{{cite book | title = 《再论“百人斩” 》| author =程兆奇 | url = http://www.historicalchina.net/admin/webedit/uploadfile/hundredzq.pdf | journal = 《江苏社会科学》| year = 2002 | issue = 6}}</ref>

==2003年的调查==
[[侵华日军南京大屠杀遇难同胞纪念馆|侵华日军南京大屠杀遇难同胞纪念馆]]馆长[[朱成山|朱成山]]2003年8月在[[大阪国立图书馆|大阪国立图书馆]]查阅到，当年《[[东京日日新闻|东京日日新闻]]》《[[大阪日日新闻|大阪日日新闻]]》《[[大阪朝日新闻|大阪朝日新闻]]》等许多报纸都刊发了百人斩的消息。

==日本刀的物理局限问题==
大屠杀虚构派人士山本七平等人提出，“日本刀的消耗非常快，在实际的战斗中，用了一次，就几乎成了废品”，已经“不被作为战场
的兵器使用”，而几乎是作为“仪仗”使用，是“工艺美术品”。大屠杀实证派人士则引用任日本刀剑博物馆副馆长[[佐藤寒山|佐藤寒山]]在《日本刀概说》的见解，认为“日本刀的特色被公认为实用是因为（1）不会折断，（2）不会弯曲，（3）而且特别锐利”，而尽管[[明治维新|明治维新]]后曾有“废刀令”，日本刀一时成了“无用的长物”，但昭和之后又出现了所谓“{{lang|ja|軍刀ブ—ム}}”（军刀高潮/軍刀boom），并在中日战争爆发后达到最盛。<ref>{{cite book | author = 佐藤寒山 | title =《原色日本の美術》第25卷《甲胄と刀剣·日本刀概説》|pages = 第239页}}</ref>1945年后，“武器制造禁止令”明确规定禁止制造日本刀。1950年后，日本“文化财保护委员会”作为“传统文化”重新允许制造日本刀，并向工艺品转化。因此1937年战时的日本刀斩百人是可能的。<ref name="hzq"/>

==2003年日军遗属的起诉==
2003年4月，战犯向井敏明、野田毅的遗属向井（田所）千惠子、野田马萨等人上诉日本[[东京地方法院|东京地方法院]]，控告《[[朝日新闻|朝日新闻]]》、[[毎日新聞|毎日新聞]]、柏書房、[[本多勝一|本多勝一]]等当年的相关报道是屬於毀謗名譽行為、企圖為向井敏明翻案，其理由是一把日本军刀砍一个人或几个人就会卷刃，不可能连续砍一百多人。而且原報導是根據戰地上「開玩笑」而編撰的，從原報導來看，也只是在進行白刃戰，並無殺害非武裝人員的記述。

对此，[[南京大屠杀遇难同胞纪念馆|南京大屠杀遇难同胞纪念馆]]馆长[[朱成山|朱成山]]的解释是<ref>{{cite web | url=http://news.sina.com.cn/c/2005-08-11/17326668717s.shtml | title=南京大屠杀史料与事实：日军百人斩杀人比赛 | publisher=新浪网 | accessdate=2013-07-27}}</ref>：
{{cquote|“百人斩杀人比赛”的实质，并不在于用一把刀还是几把刀，或是刀枪并用，杀死了100多位中国人，而是在罪恶的侵略战争中，以虐杀俘虏与平民为乐，并惨无人道地进行比赛。这种罪恶行径充分暴露侵略者的本质，理所当然受到正义的谴责。}}
 	
2005年8月23日，日本东京地方法院宣判不支持原告诉讼，驳回了原告方的赔偿请求。<ref>[http://news.bbc.co.uk/chinese/simp/hi/newsid_4170000/newsid_4177000/4177088.stm 东京法院驳回“百人斩”诽谤状]</ref>理由是無法證明百人斬殺人比賽是根據戰地上「開玩笑」而編撰的，駁回了原告方的賠償請求。

原告再上訴日本東京高等法院。2006年5月24日，日本[[東京高等法院|東京高等法院]]二審駁回向井（田所）千惠子、野田馬薩等人的上訴要求。原告再上訴最高法院。2006年12月22日，[[日本最高法院|日本最高法院]]駁回向井千惠子等人的上訴要求。確定原告敗訴。

==“百人斩”物证==
参与此杀人竞赛的日军使用的部分日本军刀有名称：向井（至少杀害106人）使用的军刀是“关之孫六”，野田（至少杀害105人）使用的是家传军刀，田中军吉（至少杀害300人）使用的是“助广”。目前在[[台北|台北]]的[[国军历史文物馆|国军历史文物馆]]中陈列有刀柄上镶嵌的铜块上面用日文刻写头版“南京之役杀107人”字样的一把军刀被认为是其中之一，也有观点认为这把军刀不是那三把军刀之一，后面隐藏着逃脱法网的另一个凶手，但也有留日學者李茂生對此軍刀的來源表示存疑 (因“南京之役杀107人”字样的一把军刀為98式军刀,但此刀在百人斩比赛中還沒生產)<ref>楊德宜，[https://udn.com/news/story/11431/2651521 砍憲兵軍刀來自南京大屠殺？李茂生點出關鍵]，udn聯合新聞網，2017-08-19</ref>。
