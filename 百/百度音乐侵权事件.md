'''百度音乐侵权事件'''，又称'''百度MP3侵权事件'''，是指[[百度公司|百度公司]]旗下[[百度MP3|百度MP3]]未经音乐版权持有人许可私自将其作品置于互联网使用的事件。百度MP3侵权事件次数较多，自2005年开始至今，仍不断面临法律诉讼。

== 事件 ==
=== 事件经过 ===
2005年9月26日，[[环球唱片|环球唱片有限公司]]、[[华纳唱片|华纳唱片有限公司]]、[[金牌娱乐事业有限公司|金牌娱乐事业有限公司]]、[[EMI_GROUP|EMI GROUP香港公司]]、[[索尼BMG|索尼BMG]]、[[新艺宝|新艺宝]]和[[正东唱片|正东唱片]]一共七家公司向[[北京市第一中级人民法院|北京市第一中级人民法院]]提出诉讼，控诉[[百度MP3|百度MP3]]侵犯其著作版权。七大公司要求百度公司停止侵权行径，在百度官方网站和《[[法制日报|法制日报]]》公开赔礼道歉，并且赔偿损失共计167万人民币；后百度公司败诉。<ref>{{cite web |url=http://tech.sina.com.cn/i/2005-09-26/1411729161.shtml |title=七大唱片公司状告百度MP3侵权案今日开庭 |publisher=新浪网 |accessdate=2005年 |work= |language=zh-cn |deadurl=no |archiveurl=https://web.archive.org/web/20140315143513/http://tech.sina.com.cn/i/2005-09-26/1411729161.shtml |archivedate=2014-03-15 |df= }}</ref>

2008年1月17日，[[中国音乐著作权协会|中国音乐著作权协会]]向[[北京市|北京市]][[海淀区|海淀区]]人民法院提起诉讼，就[[百度MP3|百度MP3]]未经版权许可使用《[[爱我中华|爱我中华]]》等共计50首歌曲，索赔106万人民币，2010年，[[中国音乐著作权协会|中国音乐著作权协会]]胜诉。<ref>{{cite web|url= http://tech.163.com/10/0224/05/6090TMI4000915BF.html|title= 百度MP3歌词搜索再度被判侵权|publisher= 网易|accessdate= 2010年|work= |language= zh-cn|deadurl= no|archiveurl= https://web.archive.org/web/20100228071625/http://tech.163.com/10/0224/05/6090TMI4000915BF.html|archivedate= 2010-02-28|df= }}</ref><ref>{{cite web|url= http://www.chinanews.com/it/it-qydt/news/2010/02-24/2135352.shtml|title= 百度MP3歌词搜索再度被判侵权|publisher= 中国新闻网|accessdate= 2010年|work= |language= zh-cn|deadurl= no|archiveurl= https://web.archive.org/web/20160305083449/http://www.chinanews.com/it/it-qydt/news/2010/02-24/2135352.shtml|archivedate= 2016-03-05|df= }}</ref><ref>{{cite web |url=http://finance.qq.com/a/20100224/003718.htm |title=百度MP3歌词搜索再度被判侵权 |publisher=腾讯网 |accessdate=2010年 |work= |language=zh-cn |deadurl=no |archiveurl=https://web.archive.org/web/20180706104159/https://finance.qq.com/a/20100224/003718.htm |archivedate=2018-07-06 |df= }}</ref>

2008年6月10日，[[数字音乐网站娱乐基地|数字音乐网站娱乐基地]]向[[美国|美国]]联邦南纽约地区法院提出诉讼，控诉百度MP3未经著作权人授权使用其作品，这是百度公司面临的一次海外法律事件。<ref>{{cite web|url= http://business.sohu.com/20080613/n257472513.shtml|title= 百度MP3搜索在美遭侵权起诉|publisher= 搜狐网|accessdate= 2008年|work= |language= zh-cn|deadurl= no|archiveurl= https://web.archive.org/web/20160304203956/http://business.sohu.com/20080613/n257472513.shtml|archivedate= 2016-03-04|df= }}</ref>

2011年1月20日，[[川子|川子]]、[[马条|马条]]、[[山人乐队|山人乐队]]、[[周云蓬|周云蓬]]、[[李志|李志]]、[[秋野|秋野]]、[[洛兵|洛兵]]、[[沈庆|沈庆]]、[[吴梦奇|吴梦奇]]等27个艺人连同[[十三月唱片公司|十三月唱片公司]]向百度公司发出律师函，指出百度MP3未经版权许可使用了其137首原创歌曲，要求百度公司赔偿其作品损失费用共计685万人民币。<ref name="wya ">{{cite web |url=http://tech.163.com/11/0121/15/6QUDNLNR000915BF.html |title=27位音乐人向百度发律师函诉MP3侵权索赔685万 |publisher=网易 |accessdate=2011年 |work= |language=zh-cn |deadurl=no |archiveurl=https://web.archive.org/web/20121030204829/http://tech.163.com/11/0121/15/6QUDNLNR000915BF.html |archivedate=2012-10-30 |df= }}</ref>

2011年4月7日，[[谷建芬|谷建芬]]、[[高晓松|高晓松]]、[[小柯|小柯]]、[[小虫|小虫]]、[[李思菘|李思菘]]等领导的[[华语音乐词曲作者维权联盟|华语音乐词曲作者维权联盟]]向百度公司发出公开信，提出与百度公司谈判解决百度MP3侵权问题，谈判条件为：“下线、道歉、赔偿、共谋发展”。<ref>{{cite web|url=http://tech.ifeng.com/internet/special/baiduwenku/content-1/detail_2011_04/11/5655100_0.shtml|title=音乐人讨伐百度再升级 谷建芬任首席谈判代表|publisher=凤凰网|accessdate=2011年|work=|language=zh-cn|deadurl=no|archiveurl=https://web.archive.org/web/20140315221229/http://tech.ifeng.com/internet/special/baiduwenku/content-1/detail_2011_04/11/5655100_0.shtml|archivedate=2014-03-15|df=}}</ref>

== 社会影响 ==
音乐人[[高晓松|高晓松]]说：“百度是音乐圈最切齿的仇人，全球最大的盗版音乐销赃地，用偷来的音乐大卖广告赚钱的无耻商人，奴役[[音乐家|音乐家]]的[[吸血鬼|吸血鬼]]。可昨儿百度颁奖，竟然有那么多歌手去捧场，某公司旗下艺人甚至倾巢出动。自己没骨气，活该被人欺！”<ref name="wya " />

《[[新浪网|新浪网]]》评论家亦子剑的文章“百度被诉及败诉的积极意义”指出，此事有助于网民认识到版权的重要性，促进了企业和公民尊重的版权意识。<ref>{{cite web|url=http://tech.sina.com.cn/i/2005-09-29/1102732209.shtml|title=亦子剑：百度被诉及败诉的积极意义|publisher=新浪网|accessdate=2005年|work=|language=zh-cn|deadurl=no|archiveurl=https://web.archive.org/web/20140315143300/http://tech.sina.com.cn/i/2005-09-29/1102732209.shtml|archivedate=2014-03-15|df=}}</ref>
