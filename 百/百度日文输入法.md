'''百度日文日入法'''（'''Baidu IME'''）是由[[百度|百度]]开发的[[日文输入法|日文输入法]]。

2009年12月16日，Baidu Type Beta版发布。2010年5月11日，正式版发布。2011年3月8日更改为现在的名称。

== 概要 ==
'''Baidu IME'''是在[[搜索引擎|搜索引擎]]和其他网页信息基础上用[[语音识别|统计性语言模式]]开发的日文输入法<ref>{{cite news|url=http://www.itmedia.co.jp/news/articles/0912/16/news081.html|title=バイドゥ、日本語入力システム『Baidu Type』公開 『ケータイ世代に使いやすく』|publisher=ITmedia ニュース|date=2009年12月16日|language=ja}}</ref>。当初Beta版发布时的理念是「让日文打字更有趣（{{lang|ja|楽しい日本語入力}}）」<ref>{{cite web|url=http://staffblog.baidu.jp/2009/12/16/baidu-type-001/|title=バイドゥが文字入力システム『BaiduType（バイドゥタイプ）』Betaを公開します|publisher=Baidu Japan Blog|date=2009年12月16日|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20131011073528/http://staffblog.baidu.jp/2009/12/16/baidu-type-001/|archivedate=2013年10月11日}}</ref>，而现在更强调输入法的个性与风格<ref>{{cite web|url=http://staffblog.baidu.jp/2011/05/19/baidu-ime-05/|title=【Baidu IME】記者発表会を実施！ - 『I ♥ ME – わたしスタイルIME』サイトとTwitterアカウントも開設しました|publisher=Baidu Japan Blog|date=2011年5月19日|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20130725071030/http://staffblog.baidu.jp/2011/05/19/baidu-ime-05/|archivedate=2013年7月25日}}</ref>。

该输入法主要面向喜爱流行文化的年轻人，内置大量[[御宅族|宅]]文化词汇和[[流行语|流行语]]，以及[[2ch|2ch]]常用的[[颜文字|颜文字]]和[[ASCII艺术|ASCII Art]]。并以[[二次元|二次元]][[萌|萌]]系角色「愛美」（IME的諧音）作为其[[代言人|代言人]]<ref>{{cite web|url=http://ime.baidu.jp/type/lp/nikko20111219/|title=愛美のIME体験レポート Baidu IMEで、ブログ書くのがもっと楽しくなっちゃった！|language=ja}}</ref><ref>[http://comic.sina.com.cn/n/2011-12-23/07533474.shtml 百度也玩拟人化 度娘爱美（Ime）登场！]</ref>，一度在中文[[御宅族|御宅族]]中成为话题<ref>{{cite web|url=http://news.searchina.ne.jp/disp.cgi?y=2011&d=1230&f=column_1230_005.shtml|title=百度が日本で公式萌えキャラを出したと中国オタクの間で話題に|language=ja}}</ref>。卸載百度IME時出現的可愛女孩，也获得了用戶好評<ref>{{cite news|url=http://blog.esuteru.com/archives/7155337.html|title=『Baidu IME』のアンインストール画面に現れる美少女について公式がコメント！「感情を揺さぶるような仕掛けを心がけてます」|publisher=はちま起稿|date=2013-06-03}}</ref>。

=== 开发的原委 ===
百度搜索引擎在讨论亲和性新服务时提出了日文输入法的开发。由于与手机相比，在[[PC|PC]]上的日文输入法有些欠缺的考虑与公司产品事业部的意识相结合，2009年初开始策划、同年夏天开始开发<ref>{{cite news|url=http://www.itmedia.co.jp/news/articles/0912/21/news020.html|title=『日本語を楽しく』 27歳ケータイ女子が企画したIME『Baidu Type』|publisher=ITmedia ニュース|date=2009年12月21日}}</ref>。

== 功能 ==
百度日文输入法有对网页信息的开发、知名人士和话题关键词的转换、表情符号、预测变换、输入学习等功能<ref>{{cite news|url=http://r25.yahoo.co.jp/fushigi/jikenbo_detail/?id=20100223-00001539-r25|title=中国の検索エンジン・百度の日本語入力システムもけっこう便利|publisher=Web R25|date=2010年2月23日|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20130708095735/http://r25.yahoo.co.jp/fushigi/jikenbo_detail/?id=20100223-00001539-r25|archivedate=2013年7月8日}}</ref>。此外，皮肤可以改变外观，用户也可以根据需要自定义設定<ref>{{cite news|url=http://pc.nikkeibp.co.jp/article/news/20110518/1031859/|title=バイドゥがBaidu IMEの最新機能を紹介|publisher=PC Online ニュース|date=2011年5月18日|language=ja}}</ref>。后来添加了[[截图|截图]]功能<ref>{{cite news|url=http://www.rbbtoday.com/article/2011/03/09/75010.html|title=バイドゥ、リニューアルした日本語入力システム『Baidu IME』公開|publisher=RBB TODAY|date=2011年3月9日|language=ja}}</ref>。
以下是官网介绍的功能：<ref>{{cite web|url=http://ime.baidu.jp/type/about/index.html|title=Baidu IME 機能と特徴|publisher=Baidu Japan|language=ja}}</ref>
* 话题关键字输入
* 表情符号输入
* [[ASCII艺术|AA]]输入
* 云输入
* 潜在能见度转换，预测学习功能
* 皮肤定制
* 输入方式选择
* 个人字典选项
* 截图功能
* 手写功能

== 隐私争议 ==
最初默认将用户使用输入法的信息加密后发送到百度的[[服务器|服务器]]，隨後在默認設定中將此項改爲關閉<ref>{{cite web|url=http://www.sourcenext.com/titles/free/baidutype/agreement.html|script-title=ja:利用ガイドライン（Baiduでは修正前の情報が削除されたため、ソースネクスト同梱のものを示す）|language=ja}}</ref><ref>{{cite web|url=http://staffblog.baidu.jp/2009/12/18/baidu-type-002/|script-title=ja:【Baidu Type】ログ情報の送信設定を『オフ』に変更しました|publisher={{lang|ja|バイドゥ公式スタッフブログ}}|date=2009年12月18日|deadurl=yes|archiveurl=https://web.archive.org/web/20131005103437/http://staffblog.baidu.jp/2009/12/18/baidu-type-002/|archivedate=2013年10月5日}}</ref>。

2013年12月26日，{{link-ja|日本內閣官房情報資訊安全中心|内閣官房情報セキュリティセンター}}（NISC）提醒政府部門、大學和研究機構避免使用該程式，理由是使用者的輸入紀錄會被傳送到國外伺服器，有泄露敏感資訊的風險<ref>{{cite news|url=http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS5/8386095.shtml|title=日示警 日語輸入法存資安風險|publisher=中央社|date=2013-12-26|archiveurl=<!-- 存檔連結-->https://web.archive.org/web/20131227210804/http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS5/8386095.shtml |archivedate=<!-- 存檔日期-->2013-12-27}}</ref><ref>{{cite news|url=http://headlines.yahoo.co.jp/hl?a=20131226-00000017-bloom_st-bus_all|script-title=ja:外務省：中国「百度」製入力ソフトを削除、情報漏えい対策|publisher=Bloomberg|date=2013-12-26 |language =<!--语言-->ja|archiveurl=<!-- 存檔連結-->https://web.archive.org/web/20131228122719/http://headlines.yahoo.co.jp/hl?a=20131226-00000017-bloom_st-bus_all |archivedate=<!-- 存檔日期-->2013-12-28}}</ref>。
百度随即予以回应，称百度日文输入法所采用的云输入技术，会将用户输入的部分数据加密上传至服务器，但不涉及个人隐私内容。而且，相关服务器及数据也全部放在日本境内接受管理，不存在非法上传数据问题和泄露风险<ref>{{cite news|url=http://tech.sina.com.cn/i/2013-12-26/18179044129.shtml|title=百度否认日语输入法数据泄漏：未上传个人隐私|publisher=新浪科技 |date=2013-12-26}}</ref>。
[[中央电视台|中央电视台]]评论则认为日本政府此举乃别有用心<ref>{{cite web | url=http://tech.sina.com.cn/i/2013-12-29/13269051036.shtml | title=央视：日本政府炒作禁用百度输入法别有用心 | publisher=大公科技 | accessdate=2013-12-30}}</ref>。
12月30日，百度對軟件進行了升級，雲輸入（{{lang|ja|クラウド入力}}）被改爲默認關閉，用戶需要在設置中手動打開。<ref>[http://prtimes.jp/main/html/rd/p/000000021.000006410.html {{lang|ja|日本語入力システム「Baidu IME」に関する 弊社の対応について。}}] PR TIMES</ref>。

時值2014年初，日本內閣官房情報資訊安全中心在其官方網站上僅僅提示隱私流失的危險性，未直接指出何種輸入法。<ref name="会社で[国民を守る情報セキュリティサイト] ">{{Cite web
