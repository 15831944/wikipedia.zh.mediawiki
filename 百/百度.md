'''百度公司'''（簡稱'''百度'''，{{lang-en|Baidu}}）是一家主要經營[[搜索引擎|搜索引擎]]服務的[[互联网|互联网]]公司，于2000年1月1日由[[李彦宏|李彦宏]]、[[徐勇_(企业家)|徐勇]]两人创立于[[北京|北京]][[中关村|中关村]]<ref>[http://home.baidu.com/about/about.html 百度公司官方介绍] {{webarchive|url=https://web.archive.org/web/20140920225846/http://home.baidu.com/about/about.html |date=2014-09-20 }}</ref>。“百度”源自中国[[南宋|南宋]]词人[[辛弃疾|辛弃疾]]的《[[青玉案|青玉案]]·元夕》的一句词：「众里寻他千'''百度'''，蓦然回首，那人却在，灯火阑珊处」<ref>{{Wikisource-inline|青玉案 (辛棄疾)|青玉案·元夕|zh}}</ref>，企業標誌則是一個「熊掌」，來自於“猎人以熊掌來尋跡”的意象。百度是中国三大互联网企业巨頭BAT中的「B」（「A」與「T」分別是[[阿里巴巴集團|阿里巴巴集團]]（{{lang|en|Alibaba Group}}）和[[騰訊|騰訊]]（{{lang|en|Tencent}}））。

截至2018年5月，百度在[[Alexa_Internet|Alexa]]的排名上位於第四，中国互联网排名第一<ref>{{Cite web|url =http://www.alexa.com/topsites|title =Alexa 全球网站排名|accessdate =|author =|date =|publisher =|deadurl =no|archiveurl =https://web.archive.org/web/20090520045403/http://www.alexa.com/topsites|archivedate =2009-05-20}}</ref>，佔全世界搜索引擎市場份额12.3％（居於第二位，仅次于[[谷歌|谷歌]]<ref>{{Cite web|url = https://www.netmarketshare.com/search-engine-market-share.aspx?qprid=4&qpcustomd=0&qpsp=64&qpnp=4&qptimeframe=Q|title = 全球搜索引擎份额排名|deadurl = no|archiveurl = https://web.archive.org/web/20160821030435/https://www.netmarketshare.com/search-engine-market-share.aspx?qprid=4&qpcustomd=0&qpsp=64&qpnp=4&qptimeframe=Q|archivedate = 2016-08-21}}</ref>）。

== 资本与架构 ==
2000年1月1日，[[李彦宏|李彦宏]]在北京市中关村创建了百度在线网络技术（北京）有限公司（Baidu Online Network Technology (Beijing) Co.,Ltd.，简称“百度在线”）。<ref>{{Cite web|url=http://www.jizhuomi.com/internet/692.html|title=百度成长史：从2000年的异军突起到如今的焦虑与变革-IT互联网-鸡啄米|accessdate=2018-05-27|work=www.jizhuomi.com|language=zh-CN|deadurl=no|archiveurl=https://web.archive.org/web/20170719162606/http://www.jizhuomi.com/internet/692.html|archivedate=2017-07-19}}</ref>

2005年6月6日，百度控股有限公司在[[上海|上海]]成立了第二个全资子公司——百度中国有限公司（Baidu (China) Co.,Ltd.，一般简称“百度中国”）。

==百度国际化==
===百度日本===
{{main|百度日本}}
2006年，百度日本公司成立。2015年3月，由于未能在日本市场引起反响，百度日本正式停止在日本的搜索业务。百度一名发言人表示：并不排除，未来百度日本公司再次推出日本地区的搜索服务，或者通过日本其他平台推出搜索服务的可能性。

2011年12月13日，百度日本公司宣布收购知名日文手机端输入法“Simeji”，并宣布继续负责该款产品的规划与开发。

== 争议 ==
{{Section OR}}
{{main|对百度的争议}}
多年以来，百度在众多事件上存有争议：
* 百度是否为真正的“民族企业”。
* 倾向性审查政策。对政治敏感词汇打击力度较强，而对于诸如贴吧等内容的淫秽色情图片，商业、欺诈广告等则审查较轻。
* 产品内容的过度审查及造假，对用户内容的随意处理与控制（旗下百度贴吧删帖严重，并有“以70万出卖吧主位置”的恶性事件<ref>[http://zh.moegirl.org/%E7%9B%97%E7%89%88%E8%88%B0%E5%A8%98%E6%8A%B5%E5%88%B6%E4%BA%8B%E4%BB%B6 盗版舰娘抵制事件-萌娘百科] {{webarchive|url=https://web.archive.org/web/20150615062007/http://zh.moegirl.org/%E7%9B%97%E7%89%88%E8%88%B0%E5%A8%98%E6%8A%B5%E5%88%B6%E4%BA%8B%E4%BB%B6 |date=2015-06-15 }}</ref>）。与此同时却大肆利用色情内容作为营销手段（百度云盘使用BT下载，在线看片作为卖点，在百度贴吧大量植入色情游戏广告）吸引流量，[[百度网盘|百度网盘]]推出早期以“[[色情片|爱情片，动作片]]，这里都有”作为宣传广告。
* 多年来在数次针对Google的网络攻击中存有嫌疑，并被指参与2002年Google被政府封锁的事件。
* “百度全家桶”垃圾软件爭議
* 侵犯版权（旗下百度MP3及百度文库，都曾引起轩然大波及各类官司，作家[[韩寒|韩寒]]曾写过多篇博文批评百度的侵权行为），2013年11月13日，优酷土豆集团、搜狐视频、腾讯视频等十余家公司联合发布“中国网络视频反盗版联合行动宣言”，表示将联合对抗百度日益严重的网络视频盗版和盗链行为。并已向法院起诉百度的盗版侵权案件共立案百余起，涉及百度盗链、盗播移动视频版权的影视作品逾万部，向百度索赔由此带来的损失3亿元。同时，优酷土豆表示，在此前已经设置了robots协议阻止百度视频搜索爬虫抓取，但百度依然强行抓取。
* 对[[百度竞价排名事件|竞价排名]]商户审查不够（收取各类贩售假药，三流医院，钓鱼网站等灰色行业的高额广告费，在搜索结果中将用户导向该类网站。遭中国官媒CCTV的数次曝光性报道后承诺改进，随后推出凤巢系统，却再次被报道此系统并非改善而是变本加厉）<ref>{{cite web |url=http://news.ifeng.com/a/20160926/50024213_0.shtml |title=央视再揭百度竞价排名模式；新华网IPO获核准；锤子科技资产负债率高达99% |publisher=36氪、凤凰资讯 |date=2016-09-26 |deadurl=no |archiveurl=https://web.archive.org/web/20161001205402/http://news.ifeng.com/a/20160926/50024213_0.shtml |archivedate=2016-10-01 }}</ref><ref>{{cite web|url=http://news.cctv.com/2016/09/27/VIDEz3l64d8sutorQdUhiAHO160927.shtml|title=“网络消费陷阱”揭秘·变了味的百度推广|publisher=CCTV央视网|date=2016-09-27|deadurl=no|archiveurl=https://web.archive.org/web/20161003080211/http://news.cctv.com/2016/09/27/VIDEz3l64d8sutorQdUhiAHO160927.shtml|archivedate=2016-10-03}}</ref>
