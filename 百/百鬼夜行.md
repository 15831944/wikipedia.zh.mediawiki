'''百鬼夜行'''（ひゃっきやぎょう、ひゃっきやこう）是流傳在日本[[民間傳說|民間傳說]]中出現在夏日夜晚的[[妖怪|妖怪]]大遊行，也是著名妖怪繪師[[鳥山石燕|鳥山石燕]]一系列的[[妖怪繪卷|妖怪繪卷]]（参见[[画图百鬼夜行|画图百鬼夜行]]<ref name=toy>{{cite book|author=葉怡君|title=妖怪玩物誌|date=2006|publisher=遠流|isbn=978-957-32-5861-2}}</ref>{{rp|115}}）。{{Fact|当代因日本奇幻作家[[夢枕獏|夢枕獏]]的小說《[[陰陽師_(小說)|陰陽師]]》系列以及日本漫畫家[[今市子|今市子]]的漫畫《[[百鬼夜行抄|百鬼夜行抄]]》，使得這個名詞開始在華文地區流行|time=2018-06-08T06:24:20+00:00}}，衍生出許多[[同人誌|同人誌]]以及奇幻創作，如：[[今市子|今市子]]的[[百鬼夜行抄|百鬼夜行抄]]、[[绿川幸|绿川幸]]的[[夏目友人帐|夏目友人帐]]和[[椎橋寬|椎橋寬]]的[[妖怪少爺|妖怪少爺]]、[[妖怪手錶|妖怪手錶]]中的百鬼姬一角命名來於此。

'''百鬼'''一词，最早起源于中国，[[东汉|东汉]]·[[张衡|张衡]]（78年-139年）的《[[东京赋|东京赋]]》中，“度朔山上有桃树，下常简阅百鬼，鬼无道理者，[[神荼|神荼]]与[[郁垒|郁垒]]，持以苇索，执以饲虎。” <ref>[[东汉|东汉]]·[[张衡|张衡]]（78年-139年）的《[[东京赋|东京赋]]》中，“度朔山上有桃树，下常简阅百鬼，鬼无道理者，[[神荼|神荼]]与[[郁垒|郁垒]]，持以苇索，执以饲虎。”</ref>[[东晋|东晋]]（317年－420年）的《[[搜神记|搜神记]]》·卷二“'''寿光侯'''者，汉章帝时人也。能劾百鬼众魅，令自缚见形。”<ref>[[东晋|东晋]]（317年－420年）的《[[搜神记|搜神记]]》·卷二“'''寿光侯'''者，汉章帝时人也。能劾百鬼众魅，令自缚见形。”</ref>；《[[抱朴子|抱朴子]]》·内篇“若忽偶忘'''守一'''，而为百鬼所害。或卧而魇者，即出中庭视辅星，握固守一，鬼即去矣。”<ref>《[[抱朴子|抱朴子]]》·内篇“若忽偶忘'''守一'''，而为百鬼所害。或卧而魇者，即出中庭视辅星，握固守一，鬼即去矣。”</ref>

== 百鬼夜行略說 ==
日本的[[平安時代|平安時代]]<ref name=ele>{{cite book|author=天野行雄、宇田川新聞|title=日本妖怪物語|date=2018-04|publisher=LaVie|isbn=978-986-408-364-0}}</ref>{{rp|54}}，是一個幽暗未明，存在著人類和妖怪兩者的時代，妖怪住的地方，和人類所住的地方，其實空間上是重疊的，只是人類在白天活動，妖怪們則是在晚間出現。<ref name=toy/>{{rp|68}}這樣的感覺，可以在[[宮崎駿|宮崎駿]]電影《[[千与千寻|千与千寻]]》、[[椎桥宽|椎桥宽]]作品《[[妖怪少爺|妖怪少爺]]》中看到，白天沓無人跡的[[主題樂園|主題樂園]]，雖已成為荒墟，但一到了夜晚，華燈初上，妖怪們紛紛出現，飲食街，溫泉鄉，妖怪們出沒的夜晚，過著如同人類一般的夜生活，好不熱鬧。那個時代裡，在[[京都|京都]]，到了夜晚來臨，整條路空無一人，這時候會出現許多奇形怪狀的妖怪，像是[[廟會|廟會]]的行列一般，帶著猙獰的面孔，走在大路上，人稱「'''百鬼夜行'''」，有的像是破掉的茶壺、鍋碗瓢盆成群結隊地在夜晚的街道遊行，據說親眼目睹的人會遭受[[詛咒|詛咒]]無緣無故地喪命。<ref name=toy/>{{rp|109}}

== 付喪神的說法<ref name=toy/>{{rp|104}} ==
百鬼夜行的現象，多少反映了[[鎖國時期|鎖國時期]]以前日本的社會狀況，由於海外貿易興盛，從大陸運來的手工藝品充斥，因此人們開始不懂得愛惜東西的社會現況，老一輩的人因為物資缺乏，總是抱持著惜物的觀念，並且認為百鬼夜行的現象，是'''[[付喪神|付喪神]]'''的來到人間搗亂，一直到[[室町時代|室町時代]]付喪神的概念才漸漸地成形。

[[付喪神|付喪神]]是藉由環境的[[陰陽|陰陽]]之變，使器物化身成自然界的鬼魅到人間來作亂。儘管人們在一開始，對於[[付喪神|付喪神]]感到無比的恐懼，但是不久之後，人們卻將[[付喪神|付喪神]]當做是一種茶餘飯後消遣的話題，大大地降低了原先所擁有之令人敬畏不已的神秘感。

== 辟邪對策 ==
不過，當時的人們，很害怕看到這些妖怪們，所以都緊閉門窗，安然地進入夢鄉。只有那倒楣的不幸的夜歸人，會遇上這群妖怪的行列，如果口中唸誦經文<ref name=ele/>{{rp|54}}或是咒語，用松枝在黑暗中探路，據說妖怪就不會欺身害人，即使妖怪已經接近你，也不會讓你看見他們的身影，不過，擁有[[通靈|通靈]]能力的貴人或是少年，才會看得見百鬼夜行的景象，而且也不致遭受到禍害。

== 撒豆驅鬼的習俗 ==
為了驅邪納福，因此在每年立春的前一天。這一天叫做「[[節分|節分]]」。長年使用不要的器物必須丟到屋外，用火燒掉它，以免有妖怪作祟。所以才會有[[除夕|除夕]]大掃除的習慣流傳下來，這個儀式就是「除舊佈新」<ref name=ele/>{{rp|38}}。不論在寺廟還是在每個家庭都舉行一種叫做「撒豆驅鬼」的活動。它可能起源於中國古代的[[追儺|追儺]]儀式，即一種[[過年|過年]]活動。現在舉行撒豆驅鬼活動時，在寺廟中，一般都是處於[[本命年|本命年]]的男人擔當撒豆驅邪的任務。人們相信吃了和自己的年齡一樣多的豆子就能在這一年平安無病，因此人們爭先恐後地撿這種豆子。而在每個家庭裏，沒有專門規定撒豆子的人，往往由父親戴上鬼的面具擔當鬼的角色，孩子們向「鬼」撒豆子，這時候人們異口同聲地喊「鬼出去，福進來！日本發大財！」場面顯得十分熱鬧。雖然這時候，外頭的氣溫還很冷，但是從這種熱情洋溢的吶喊聲中，人們還是能感受到春天即將到來的氣氛。<ref name=toy/>{{rp|35}}
