'''百福車站'''位於[[臺灣|臺灣]][[基隆市|基隆市]][[七堵區|七堵區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(北段)|縱貫線]]的[[鐵路車站|鐵路車站]]，因鄰近[[五堵|百福社區]]而得名。

== 車站構造 ==
*[[側式月臺|側式月臺]]、[[島式月臺|島式月臺]]各一座，第一月臺為側式月臺，第二月臺為島式月臺。但第一月臺目前多為讓過站不停的列車（主要是[[貨運列車|貨運列車]]）通過；比較特別的是，第二月臺的一月臺側為2B，反側才為2A。
*台鐵預定第一個設置[[月台門|月台門]]的車站<ref>[http://news.ltn.com.tw/news/life/breakingnews/2026121 參考日本經驗 台鐵也要裝月台門了 (自由時報)]</ref>
*全站為通過線配置（不設側線）。
*軌道配置圖：
[[File:TRATRACK095.JPG|LEFT]]

== 營運狀況 ==
*因[[臺鐵捷運化|臺鐵捷運化]]政策而新設，2007年5月8日落成啟用。
*目前為[[臺灣鐵路管理局車站等級#簡易站|簡易站]]。以[[七堵車站|七堵站]]為管理車站，僅停靠[[臺鐵區間車|區間車]]。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費<ref>[http://www.railway.gov.tw/tw/CP.aspx?sn=1312&n=1333 臺鐵多卡通電子票證乘車營運規則]</ref>。

== 歷史 ==
*2005年6月23日：正式開工。
*2007年5月8日：正式啟用。
*2008年8月1日：開放使用[[悠遊卡|悠遊卡]]等電子票證。
*2012年8月9日：[[基隆客運|基隆客運]]開闢[[臺鐵捷運化|臺鐵捷運化]][[接駁公車|接駁公車]]R88線（最初行駛百福社區－百福車站）。
*2013年12月：完成月台加高工程。
*2017年10月：延伸長安街天橋啟用。
