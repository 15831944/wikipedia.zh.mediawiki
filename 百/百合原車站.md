'''百合原站'''（{{jpn|j=百合が原駅|hg=ゆりがはらえき|rm=Yurigahara eki|f=y}}）是位於[[北海道|北海道]][[札幌市|札幌市]][[北區_(札幌市)|北區]]百合原5丁目，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[札沼線|札沼線]]（學園都市線）的[[鐵路車站|鐵路車站]]。車站編號為'''G07'''。電報略號為'''{{lang|ja|ユリ}}'''。

== 概要 ==
1986年，百合原公園舉行全國都市綠化博覽會「86'札幌花與綠之博覽會」（6月28日至8月31日），為了方便參加博覽會的市民而設置臨時乘降場。博覽會完結後，當地居民希望將車站恆常化，亦因為預計會有一定數量的乘客，因此決定繼續營運此車站。[[國鐵分割民營化|國鐵分割民營化]]後，正式升格為一般車站。

== 歷史 ==


* 1986年（昭和61年）6月28日 - [[日本國有鐵道|日本國有鐵道]]（國鐵）[[札沼線|札沼線]]'''百合原臨時乘降場'''開始營運，處理旅客服務<ref name=school>{{cite web|url=http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |language=ja |title=札沼線（学園都市線）の電化について |publisher=JR北海道 | date=2009-09-09 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20091006234745/http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |archive-date=2009-10-06 }}</ref><ref name=school2>{{cite web |url=http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |language=ja |title=札沼線（学園都市線）の電化開業時期について |publisher=JR北海道 |date=2011-10-13 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20111205160350/http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |archive-date=2011-12-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20111205160350/http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |archivedate=2011-12-05 }}</ref>。
* 1987年（昭和62年）4月1日 - 由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理。同時升格為'''百合原站'''。
* 1991年（平成3年）3月16日 - 札沼線的暱稱設定為「'''學園都市線'''」<ref name=school/><ref name=school2/>。
* 1995年（平成7年）3月16日 - 札沼線（學園都市線）[[太平車站_(日本)|太平車站]]至[[篠路車站|篠路車站]]路段[[複線鐵路|複線化]]。
* 2000年（平成12年） - 札沼線[[桑園車站|桑園車站]]至[[石狩月形車站|石狩月形車站]]路段引入自動進路制御裝置（PRC）。
* 2007年（平成19年）10月1日 - 設定車站編號（'''G07'''）<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |language=ja |title=駅番号表示（駅ナンバリング）を実施します |publisher=JR北海道 |date=2007-09-12 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20070930015220/http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |archive-date=2007-09-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20070930015220/http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |archivedate=2007-09-30 }}</ref>。
* 2008年（平成20年）10月25日 - 開始使用[[儲值車票|智能車票]]「[[Kitaca|Kitaca]]」<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |language=ja |title=Kitacaサービス開始日決定について |publisher=JR北海道 |date=2008-09-10 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20080913103748/http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |archive-date=2008-09-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20080913103748/http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |archivedate=2008-09-13 }}</ref>。
* 2012年（平成24年）6月1日 - 札沼線桑園站至[[北海道醫療大學車站|北海道醫療大學車站]]路段正式[[電氣化鐵路|電氣化]]（[[交流電|交流電]]20,000[[伏特|V]]、50[[赫兹|Hz]]）<ref name=school3>{{cite web|url=http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |language=ja |title=学園都市線電化開業に伴う電車の投入（第一次）について |publisher=JR北海道 | date=2012-03-14 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20120925203437/http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |archive-date=2012-09-25 }}</ref>。

== 車站構造 ==
現時為[[側式月台|側式月台]]2面2線（月台長度：135米<ref name=sapporojr>{{cite web|url=http://www.city.sapporo.jp/sogokotsu/kotsutaikei/documents/databook_3.pdf |language=ja |title=札幌の都市交通データ（3. JR） |publisher=札幌市公式ホームページ | date=2017-04-20 |accessdate=2017-08-25 |archive-url=https://web.archive.org/web/20170818094412/http://www.city.sapporo.jp/sogokotsu/kotsutaikei/documents/databook_3.pdf |archive-date=2017-08-18 }}</ref>）的地面車站。

無人車站。設有簡易[[自動售票機|自動售票機]]及[[驗票閘門|車票用簡易自動閘機]]及[[Kitaca|簡易Kitaca自動閘機]]。因此能夠使用[[Kitaca|Kitaca]]但無法購買。

== 車站周邊 ==
西側的太平地區是1960年代開始形成住宅區。而東側的百合原地區在過去主要是農地，1980年代本車站開始營運後亦慢慢形成住宅區，主要是低層房屋。位於車站東側的土地亦整理成車站前廣場，但並沒有任何巴士路線停靠及商業設施，主要供列車乘客停泊汽車之用。

* 北區太平百合原城市規劃中心
* 北警察署太平交番
* 札幌太平郵局
* 百合原公園
* 北海道中央巴士「太平9條6丁目」、「百合原公園前」站牌
