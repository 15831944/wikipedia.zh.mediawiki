'''百度网盘'''，曾用名'''百度云'''（Baidu Cloud），是[[百度|百度]]推出的一项个人[[云存储|云存储]]服务，覆盖中国大陆各大主流[[電腦|電腦]]和[[手机|手机]]操作系统，包含Web版、[[Windows|Windows]]版、[[Mac|Mac]]版、[[Android|Android]]版和[[iPhone|iPhone]]版。用户可以将自己的文件上传到网盘上，并可跨终端查看和分享。在百度建立[[百度开放服务平台|开放云]]后，合并为平台分布式私人储存服务。2013年8月22日，百度云推出“百度云用户破亿，基础服务震撼升级”活动，宣布提供2[[太字节|TB]]永久免费容量和无限制离线下载服务，以此吸引了大量的初期用户使用，成为了中国大陆最大的网盘服务商之一<ref name=2T></ref>。2016年，百度网盘总用户数突破4亿。2016年10月11日，百度云改名为百度网盘，此后会更加专注发展个人存储、备份功能。

== 历史 ==
* 2012年3月23日，百度网盘Web版本上线，同时还推出了PC、[[Android|Android]]客户端。
* 在2012年3月份的百度开发者大会上，百度正式发布百度云战略，推出百度开发者中心网站，并建立开发服务、运营服务、渠道推广以及变现四大服务体系，为开发者构建平等、开放、共赢的生态系统。
* 在2012年9月份的百度世界大会上，百度发布了面向开发者的“七种武器”，包括个人云存储PCS、多屏幕Screen X技术、云应用生成服务Site App、LBS·云、移动云测试MTC、百度应用引擎BAE和浏览内核Engine。
* 2012年10月，百度宣布推出仅两个月的百度云个人用户量已突破1000万。
* 2012年11月，百度宣布将与美国高通公司合作，实现并推广针对由高通骁龙处理器支持的Android终端的百度云服务。
* 2013年9月，百度宣布百度云用户破亿，并推出主题活动，宣布提供2T永久免费容量和无限制离线下载服务。只要登录百度云移动端，就能立即领取2048G永久免费容量。
* 2014年7月9日，百度云管家推出版本V4.7.0，从这个版本起，有用户反映百度云已经开始限制普通用户的下载速度<ref name=zhihu>{{zhihu |id=28333225|title=百度网盘下载限速如何破解？ - 百度云·网盘 - 知乎|accessdate=2016-06-21}}</ref>。与此同时，百度云开始推广其会员功能，推出一系列特权来吸引用户购买会员<ref>{{Cite web|url=http://yun.baidu.com/buy/center?tag=1&from=homepage|title=首页-会员中心-百度云|accessdate=2016-06-21|work=yun.baidu.com}}</ref>。
* 2016年3月17日，百度云管家推出超级会员功能<ref>{{Cite web|url=https://yun.baidu.com/disk/version|title=百度云 版本更新|accessdate=2016-06-21|work=yun.baidu.com}}</ref>，用户需要购买超级会员才能享有提速特权<ref>{{Cite web|url=http://yun.baidu.com/buy/center?tag=8|title=首页-会员中心-百度云|accessdate=2016-06-21|work=yun.baidu.com}}</ref><ref>{{Cite web|url=http://yun.baidu.com/buy/center?tag=4#question=vip-speed|title=帮助中心-会员中心-百度云|accessdate=2016-06-21|work=yun.baidu.com}}</ref>。
* 2016年10月11日，「百度云」正式改名为「百度网盘」，而面向企業的[[雲端運算|雲端運算]]服務「百度開放雲」更名为「百度云」。<ref>{{cite web|url=http://tech.sina.com.cn/i/2016-10-11/doc-ifxwrhpm2910380.shtml|title=百度云计算更名：2C的叫百度网盘，2B的称百度云|publisher=新浪科技|date=2016-10-11|accessdate=2016-12-10}}</ref>
* 2016年12月27日，百度网盘发布官方公告，要求用户在2016年12月31日前将客户端升级至最新版本。升级后，旧版本客户端已无法登录，因而普通用户已经不可能通过使用旧版本绕过限速。同时，移动客户端不再提供Windows Phone版的客户端服务，PC客户端也不再支持Windows XP SP3以下的系统。<ref>{{cite web|url=http://tech.sina.com.cn/n/k/2016-12-28/doc-ifxyxury8898988.shtml|title=百度网盘公告：12月31日前用户务必升级|publisher=新浪数码|date=2016-12-28|accessdate=2017-02-06}}</ref>
* 2018年1月初，百度网盘官方严打网盘第三方下载工具，用户频繁使用第三方客户端下载可能会被封号<ref>{{Cite web|url=https://www.ithome.com/html/it/344225.htm|title=百度网盘下手：频繁使用第三方客户端下载可能会封号 - 百度网盘 - IT之家|accessdate=2018-04-18|work=www.ithome.com}}</ref><ref>{{Cite web|url=https://www.ithome.com/html/soft/354081.htm|title=曝百度网盘封杀第三方下载插件：主页已不支持生成直链 - 百度网盘 - IT之家|accessdate=2018-04-18|work=www.ithome.com}}</ref><ref>{{Cite web|url=https://www.ithome.com/html/it/355168.htm|title=百度网盘：使用非官方版本或将锁定账号 - 百度网盘 - IT之家|accessdate=2018-04-18|work=www.ithome.com}}</ref>。
* 2018年3月初，有来自台湾的用户发现，台灣IP疑似被百度網盤封鎖，原本可以正常访问的页面会显示“啊哦，你所访问的页面不存在了”的消息，台灣用戶也無法向其他地區用戶正常分享資料，必须使用[[VPN|VPN]]才能正常使用<ref>{{Cite web |url=https://ck-joker.club/20180310/|title=百度網盤怎麼都變頁面不存在？台灣IP被封鎖，教你如何破解！| accessdate=2018-03-11}}</ref><ref name = "中時電子報"/>。

====用户隐私问题====
2017年7月18日，自媒体用户发文称，因为用户通过百度网盘公开分享的文件可以被所有人任意阅览与下载，所以在一些第三方网盘搜索网站上也可以搜索到这些文件，其中就有网盘用户通过公开方式分享的个人照片、通讯录等涉及个人信息的文件，这也造成了一定程度上用户隐私泄露的隐患<ref>{{cite news |title=百度网盘公开分享资料泄露 搜索网站可查个人信息 |url=http://www.xinhuanet.com/zgjx/2017-07/20/c_136457775.htm |accessdate=2018-06-08 |work=www.xinhuanet.com |publisher=中国青年报}}</ref>。
