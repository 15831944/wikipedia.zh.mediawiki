== 戊戌变法 ==
1895年[[北洋水師|北洋舰队]]在中日[[黄海海战_(1894年)|甲午海战]]中全军覆没，中日签订《[[马关条约|马关条约]]》，导致云集北京的科举举人联名上书，[[康有为|康有为]]，[[梁启超|梁启超]]作万言书，提出“拒和、迁都、练兵、变法”的主张，史称“[[公车上书|公车上书]]”，此举催生近代中国议政团体。1898年2月康有为第七次上书，建议光绪效法俄国[[彼得大帝|彼得]]，日本[[明治天皇|明治天皇]]变法维新，著《日本变政考》《俄彼得变政考》。1898年6月6日，康有为代拟，[[徐致靖|徐致靖]]上书《[[请明定国是疏|请明定国是疏]]》。1898年6月10日，光绪帝令[[翁同龢|翁同龢]]起草《[[明定国是诏|明定国是诏]]》，戊戌变法就此展开。后因为光绪及维新派与[[慈禧太后|慈禧]]及保守派之间摩擦不断，导致1898年9月19日的[[戊戌政变记|戊戌政变]]，戊戌变法就此告终。

=== 变法内容 ===
戊戌变法在教育、经济、政治、军事体制上均有改革，教育上引进[[西學|西学]]，采取中西兼修的模式。经济上鼓励私人创办[[企业|企业]]，主张兴办实业。政治上最终目标为建立[[君主立宪制|君主立宪制]]。军事上采用西式训练，鼓励私人创办军工企业。

==== 教育改革 ====
* 创办[[京师大学堂|京师大学堂]]，兴办翻译馆，设译书局。
* 学习内容中西兼修
* 废[[八股文|八股]]，改为考历史 、政治及[[四书五经|四书五经]]，且定期举行经济特科。

==== 经济改革 ====
* 鼓励实业，鼓励私人办厂
* 设[[铁路矿务总局|铁路矿务总局]]、[[农工商总局|农工商总局]]，并且在各省设分局
* 开放[[八旗|八旗]]经商的禁令，命其学习士农工商自谋生计

==== 政治改革 ====
* 解除[[报禁|报禁]]
* 裁撤冗官和重叠机构
* 准许地方官与士民上书

==== 军事改革 ====
* 采用西式训练方法
* 允许私人办军事工厂
* [[武举|武科]]停试弓、骑、剑，改试枪炮
* 遣散老弱残兵，削减军饷须支，实行[[團練|团练]]，裁减[[綠營|绿营]]，兴办[[民兵|民兵]]
* 筹办[[武备学堂|武备大学堂]]
康有为尚有许多未达述的目标，如定[[國教|国教]]、立教部及[[教会|教会]]、以[[孔子紀年|孔子纪年]]、定[[憲法|宪法]]、开[[國會|国会]]、君民合治、满汉平等、军队对皇帝负责，断发易服及迁都[[上海|上海]]等。

== 预备立宪 ==
清政府在[[义和团运动|义和团]]乱及西方干涉后实施[[清末新政|庚子后新政]]，是满清自[[洋务运动|洋务运动]]、[[戊戌变法|戊戌变法]]后第三次大型改革，目的在于使清朝建立[[君主立宪|君主立宪]]政体。[[慈禧太后|慈禧太后]]根据清[[宗室|宗室]][[载泽|载泽]]、[[戴鴻慈|戴鸿慈]]、[[徐世昌|徐世昌]]、[[端方|端方]]和[[紹英|绍英]]五大臣的意见，于[[光緒|光绪]]三十二年（1906年）下诏[[立憲運動|预备立宪]]，阐述“'''[[立憲運動|仿行宪政]]'''”的涵义为“'''大权统于朝廷，庶政公诸舆论，以立国家万年有道之基'''”。

1907年，清政府在中央设[[资政院|资政院]]，在地方设[[諮議局|谘议局]]，各地自治组织、立宪公会纷纷建立，在《百年宪政》中，1907年被成为“中国自治年”。

1908年，清政府颁布《[[钦定宪法大纲|钦定宪法大纲]]》，规定大清帝国万世一系，宣布十年后实行立宪。《[[钦定宪法大纲|钦定宪法大纲]]》规定皇权的至高无上性，将皇权宪法化，引起朝野不满，[[立憲運動|立宪派]]大失所望。[[梁启超|梁启超]]认为：这个[[钦定宪法大纲|宪法大纲]]是“吐饰耳目，敷衍门面”。

1909年（宣统元年）6月，各省进行[[諮議局|谘议局]]选举。

1910（宣统二年）九月初一，[[资政院|资政院]]举行第一次开院典礼，[[国会请愿运动|国会请愿运动]]进入高潮。各省出现数千人集会，要求各省[[督撫|督抚]]代奏请愿呈稿。湖北集会倡议“不开国会，不纳新捐”。[[北京|北京]]方面，国会请愿代表团迭向资政院和[[攝政王|摄政王]]上书，资政院除代奏请愿外并通过了[[陈请速开国会专折|陈请速开国会专折]]。在请愿运动压力下，十八个[[總督|督]][[巡撫|抚]]、[[將軍_(中國古代)|将军]]、[[都统|都统]]由[[東三省總督|东三省总督]][[锡良|锡良]]领衔联名奏请立即组织[[内阁|内阁]]、翌年开设[[國會|国会]]。清朝廷震惊之下，于11月14日宣布缩短预备立宪期限为五年，国会开设之前先设[[责任内阁|责任内阁]]。

1911年5月8日（宣统三年四月十日），[[清|清]]政府废除[[軍機處|军机处]]，宣布实行[[內閣|内阁]][[官制|官制]]，任命[[總理|内阁总理]]、内阁[[大臣|大臣]]。因成员名单中过半数为清[[宗室|宗室]]（[[皇族|皇族]]）与[[满人|满人]]，被讥嘲为“[[慶親王內閣|皇族内阁]]”。此举令国内有识之士震惊，令立宪派极其失望，认清清政府的保守顽固，无意实行宪政，相当一部分转而投身革命。

1911年发生[[辛亥革命|辛亥革命]]，南方诸省宣布独立，于1912年1月1日成立[[中華民國|中华民国]]。[[南北議和|南北议和]]后，即1912年2月12日，[[隆裕太后|隆裕太后]]发布《[[宣統帝退位詔書|退位诏书]]》，宣布清[[溥仪|宣统帝]]退位，清朝就此灭亡。

== 议会政治 ==
1912年1月1日，[[中華民國臨時政府|中华民国临时政府]]成立，孙文就职[[中華民國大總統|临时大总统]]。1月5日举行内阁第一次会议，其组织方法依照《[[中華民國臨時政府組織大綱|中华民国临时政府组织大纲]]》规定。1912年3月8日[[南京临时参议院|南京临时参议院]]通过《[[中華民國臨時約法|中华民国临时约法]]》，3月11日公布实施，取代《[[中華民國臨時政府組織大綱|中华民国临时政府组织大纲]]》。《[[中華民國臨時約法|临时约法]]》规定中华民国为[[议会共和制|议会共和制]]政体。

《[[中華民國臨時約法|中华民国临时约法]]》中规定“中华民国之[[立法权|立法权]]以参议院行之。参议院以第十八条所定各地方选派之参议员组织之。参议员每行省、内蒙古、[[外蒙古|外蒙古]]、西藏各选派五人；靑海选派一人。其选派方法由各地方自定之参议院会议时每参议员有一[[投票|表决权]]”。

[[南北議和|南北议和]]后，1912年3月10日，[[袁世凯|袁世凯]]在北京就任民国[[大總統|大总统]]。

1912年3月，[[宋教仁|宋教仁]]在上海被刺杀，[[国务总理|国务总理]][[赵秉钧|赵秉钧]]卷入刺杀案，临时参议院[[弹劾|弹劾]]国务总理赵秉钧，不就赵辞职。

《[[中華民國臨時約法|临时约法]]》实施后，中华民国实际上已经建立起一套行之有效的[[議會制|议会制度]]。
