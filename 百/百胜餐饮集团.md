'''百胜餐饮集团'''（英文：{{lang|en|'''Yum! Brands, Inc.'''}}）是一家总部位于[[美国|美国]][[肯塔基州|肯塔基州]][[路易維爾_(肯塔基州)|路易斯维尔]]的餐饮集团，旗下拥有以[[肯德基|肯德基]]、[[必胜客|必胜客]]、[[塔可鐘|塔可鐘]]（Taco Bell）等為主力的多個餐飲连锁品牌，在超过130個國家和地區擁有近40,000間餐廳，是全球门市最多的[[快餐|快餐]]公司，並被《[[财富_(杂志)|-{zh-tw:財星;zh-cn:财富;}-]]》雜誌列入[[財富世界500強|全球500-{zh-tw:大;zh-cn:强;}-]]企業名單內。

作為全球佈局的一環百勝集團在中国大陆也進行積極的市場拓展，除了旗下原本就擁有如肯德基、必胜客、塔可鐘等國際品牌外，也擁有[[东方既白|东方既白]]、[[小肥羊|小肥羊]]等中國當地品牌的經營權。

== 历史 ==
百胜公司前身泰康全球餐饮公司（Tricon Global Restaurant,Inc.），于1997年10月7日从[[百事可乐|百事可乐公司]]分离，成为独立的股票公開上市公司。它被授权在全球范围内使用肯德基、必胜客、塔可钟等品牌。

大卫·C·诺瓦克（David C. Novak）分别于2000年1月1日和2001年1月1日开始担任[[董事长|董事长]]和[[首席执行官|首席执行官]]。他同时也是[[摩根大通|摩根大通]][[董事|董事]]。

2002年3月，百胜宣布收购总部位于美国肯塔基州[[列克星敦_(肯塔基州)|列-{克}-星敦]]的Yorkshire Global Restaurants旗下的两个品牌：艾德熊和海滋客。同时公司更名为百胜。

2002年5月，百胜全球餐饮集团仅以3.2亿美元的低价就收购了美国艾德熊公司的全部股份。

2009年12月7日，與[[香港|香港]][[怡和集團|怡和集團]]正式簽約，售予肯德基台灣區經營權。

2011年11月7日，[[中华人民共和国商务部|中华人民共和国商务部]]批准了百胜收购小肥羊的申请。

== 百胜中国 ==
[[File:Picture_2149LouisvilleArena.jpg|缩略图]]
[[File:Tacobellkfcrestaurants.jpg|缩略图]]

百胜餐饮集团中国事业部是百胜全球餐饮集团中国总部，于1993年在上海成立。它为中国大陆直营、合资和特许经营的肯德基、必胜客、必胜宅急送、东方既白餐厅提供营运、开发、企划、财务、人事、法律及公共事务等方面的服务。  

截至2016年6月底，中国百胜旗下肯德基已成功地在中国大陆的1100多个城市和乡镇开出了近5039家餐厅，必胜客在超过130个城市拥有超过1610家欢乐餐厅，百胜还拥有100余家必胜宅急送和15家东方既白餐厅，员工人数37万。中国百胜目前是百胜在全球业务发展最快的市场。  

2012年2月，百胜餐饮集团以协议计划方式私有化小肥羊的交易顺利完成。小肥羊从2月2日起正式在香港联交所除牌，并成为百胜餐饮集团旗下一个新的餐饮品牌。截止2011年12月底，小肥羊在中国大陆拥有469家连锁餐厅。

2016年11月1日，百勝中國从百胜餐饮集团中分拆，在纽约证券交易所上市，股票代码 YUMC。

2017年1月9日，全球最大的墨西哥风味连锁餐厅运营商[[塔可鐘|塔可鐘]]（Taco Bell） 今日在上海陆家嘴世纪大道55号東方浮庭1F-1室开设中国第一家餐厅。

===速生鸡滥用药物事件===
2010年至2011年，百胜中国发现其鸡类产品发现多次检测[[抗生素|抗生素]]残留超标，原因是供应商在养殖[[速生鸡|速生鸡]]时滥用抗生素和多种违禁药物。百胜将相关检测结果掩盖两年多，直到被[[中央电视台|央视]]曝光后才向公众道歉。<ref>{{Cite web|url=http://news.cntv.cn/2013/01/10/VIDE1357817222608236.shtml|title=[视频]百胜餐饮集团就速生鸡滥用药物事件向消费者致歉_新闻频道_央视网|accessdate=2018-10-05|work=news.cntv.cn}}</ref>
