《'''百千家的妖怪王子'''》，日文原名《{{lang|ja|百千さん家のあやかし王子}}》，是日本漫畫家[[硝音綾|硝音綾]]的[[少女漫畫|少女漫畫]]作品。於2013年在[[角川集團|角川集團]]的日本漫畫月刊《[[月刊Asuka|月刊Asuka]]》4月號上連載。目前單行本共10卷。

== 故事簡介 ==
16歲的少女百千向陽的父母因事故去世，在生日那天她收到一封遺書，裡面寫有父母留給自己的一份遺產——位於深山老林中的一棟宅邸。以為自己即將住進一個又大​​又豪華的的宅子的日萬里興高采烈背起行囊，可是當她跋山涉水去到那裡時，卻發現整個房子破爛不堪如同鬼屋，而且剛一進門就被一個突然出現的裸男給嚇個半死……這個名叫七守葵的男子和名叫紫和伊勢的兩名友人不僅早就擅自住在這裡，還神秘兮兮地叫日萬里不要靠近屋子，被日萬里當場拒絕。事後她發現這三人毫無生活自理能力不算還經常搞破壞，不過最讓她頭疼的不是這些，而是她意外中發現這三人似乎並不是人類……

== 登場角色 ==
; {{nihongo|百千向陽|百千ひまり（ももちひまり）}}
: [[配音員|聲]] - [[野水伊織|野水伊織]]
: 16歳。性格開朗樂觀，對家人之間的'''羈絆'''懷有憧憬。父母因事故去世，在16歳生日那天收到一封遺書，並繼承了留下來作為遺產的'''豪宅'''。

; {{nihongo|七守葵／鵺|七守葵／鵺（ななもりあおい／ぬえ）}}
: 聲 - [[KENN|KENN]]
: 多年前，迷途破壞了封印之門誤入百千家，被指名為「'''守護大人'''」。根據紫所說，葵從十歲開始就被禁錮在百千家，好像在逃避著甚麼而進入遠離人煙的百千家。不久之後隱約知道自己無法離開百千家，一開始拼命嘗試各種方法想要離開百千家，但漸漸明白自己絕對無法用蠻力離開百千家。但葵與紫結下契約的時候，已經沒有再提要離開。被灌注了強大的妖力、體質特殊、能變身成為「'''鵺'''」。

; {{nihongo|紫|紫（むらさき）}}
: 聲 - [[遊佐浩二|遊佐浩二]]
: 水蛇，葵的式神，在葵誤入百千家之前一直居住在百千家。

; {{nihongo|伊勢|伊勢（いせ）}}
: 聲 - [[小野友樹|小野友樹]]
: 猩猩，葵的式神。
