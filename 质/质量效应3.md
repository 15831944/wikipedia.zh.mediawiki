{{vgname|质量效应3|en=Mass Effect 3}}是一款由[[BioWare|BioWare]]开发，[[艺电|艺电]]发行的[[动作角色扮演游戏|动作角色扮演游戏]]，于2012年3月6日在[[Microsoft_Windows|Microsoft Windows]]、[[Xbox_360|Xbox 360]]和[[PlayStation_3|PlayStation 3]]平台发行。名为《质量效应3：特别版》（''Mass Effect 3: Special Edition''）的[[Wii_U|Wii U]]版本于2012年11月18日发行。《质量效应3》是[[质量效应系列|质量效应]]三部曲的最后一部，《[[质量效应_(游戏)|质量效应]]》以及《[[质量效应2|质量效应2]]》的续作。游戏设定于2186年的银河系，银河系文明遭到由合成[[碳基生物|有机]]星舰组成的高度先进的机械种族“收割者”的侵略。主人公{{le|薛帕德指挥官|Commander Shepard}}是一名精锐的人类士兵，他/她的任务是建立种族联盟，为战争作准备。

与《质量效应2》相似，玩家可以将通关的[[游戏存档|游戏存档]]导入《质量效应3》，通过之前的决策影响情节。总体上，《质量效应3》围绕军事实力进行，玩家通过完成[[任务_(电子游戏)|任务]]和收集资源来增加军事实力。由于BioWare设计的游戏同时适用于新老玩家，因此系列的许多传统游戏元素仍然存在，如基于掩体系统的[[第三人称射击游戏|第三人称射击]]，但游戏也引入了新元素，例如[[多人电子游戏|多人部分]]。游戏的[[遊戲音樂|配音]]由风格不同的作曲家撰写，他们希望在《质量效应2》的[[管弦樂團|管弦乐]]声和《质量效应》的合成器声音之间取得平衡。

《质量效应3》获得了游戏媒体的一致好评。游戏在美术设计、角色、情感故事、改进的战斗系统、配乐和配音方面受到称赞。然而，游戏结局却难以被粉丝接受，他们认为这不符合预期。为了回应这一争议，BioWare发布了[[可下载内容|可下载内容]]《扩展剪辑》（''Extended Cut''），扩展了原来的结局。游戏荣获多个年终奖项，包括[[斯派克电子游戏大奖|斯派克电子游戏大奖]]的最佳[[角色扮演游戏|RPG]]和[[Game_Informer|Game Informer]]的年底游戏奖。2017年3月，续作《[[质量效应：仙女座|质量效应：仙女座]]》发行。

===战斗系统===
[[File:Mass_Effect_3_Combat.jpg|250px]]包含[[弹药|弹药]]，[[生命值|生命条]]和技能。]]

《质量效应3》中的战斗是实时战斗，但也允许暂停以便玩家精确瞄准、更换武器或使用菜单中的技能<ref>{{cite web|last1=Phillips|first1=Tom|title=Why BioWare ditched the power wheel from Mass Effect Andromeda|url=https://www.eurogamer.net/articles/2017-02-23-why-bioware-ditched-the-power-wheel-from-mass-effect-andromeda|website=[[Eurogamer|Eurogamer]]|accessdate=2018-04-22|date=2017-02-23|archiveurl=https://www.webcitation.org/71QPvWzsR?url=https://www.eurogamer.net/articles/2017-02-23-why-bioware-ditched-the-power-wheel-from-mass-effect-andromeda|archivedate=2018-08-04|deadurl=no}}</ref>。在战斗中，玩家以过肩视角直接控制薛帕德指挥官，通过攀爬梯子、战斗滚动或短跑来移动<ref>{{cite web |last1=Porter |first1=Will |title=Mass Effect 3 preview |url=https://www.pcgamer.com/mass-effect-3-preview-3/ |website=[[PC_Gamer|PC Gamer]] |accessdate=2018-08-25 |date=2012-02-18}}</ref>。通常有两名由[[人工智能|人工智能]]控制的小队成员跟随薛帕德战斗。然而，玩家仍然可以命令他们就位或选择他们的技能<ref name="Prima Combat">{{cite book|last1=Musa|first1=Alexander|last2=Bueno|first2=Fernando|last3=Lierop|first3=Raphael Van|title=Mass Effect 3: Prima Official Game Guide|date=2012|publisher=[[普莉瑪遊戲|Prima Games]]|location=Roseville, CA|isbn=978-0-307-89148-8|pages=30–43}}</ref>。Kinect用户可以口头发出这些命令<ref name=Kin/>。如果小队成员死亡，玩家可用群体复活（Unity）技能复活。如果玩家距离他们很近，则可根据屏幕提示复活<ref name="Prima Combat" />。

玩家通常会遭遇协调战斗并有不同角色互相支持的敌军部队<ref name="Cover" />。例如，典型的对抗涉及击败从远处射击的压制型敌人，同时还需对付不断进攻的更具侵略性的敌人<ref name="gspot-review" />。为了避免受到伤害，玩家可以利用战场上的物体作掩护，但是更大的困难是玩家也必须保持移动<ref name="Cover">{{cite web|last1=Purchese|first1=Robert|title=Mass Effect 3 cover-camping a no-no on Hardcore, Insanity|url=https://www.eurogamer.net/articles/2012-01-31-mass-effect-3-cover-camping-a-no-no-on-hardcore-insanity|website=[[Eurogamer|Eurogamer]]|accessdate=2018-04-22|date=2012-01-31|archiveurl=https://www.webcitation.org/71QqbRRGG?url=https://www.eurogamer.net/articles/2012-01-31-mass-effect-3-cover-camping-a-no-no-on-hardcore-insanity|archivedate=2018-08-05|deadurl=no}}</ref>。薛帕德或小队成员受到攻击时会损耗他们的护盾，护盾损耗完毕后，最后一层生命值会开始消耗，直至个体死亡、[[生命值|生命值]]恢复或护盾再生结束<ref name="Prima Combat"/>。

敌对势力可受到枪炮、近战攻击或战斗无人机等专业技能的伤害<ref name="Prima Combat" />。如果需要的话，近战系统可以是游戏玩法的重要层面<ref name=IGNCombat>{{cite web|first=Arthur|last=Gies|url=http://www.ign.com/articles/2011/06/02/e3-2011-mass-effect-3-from-cover-to-combat|title=E3 2011: Mass Effect 3 - From Cover to Combat - PlayStation 3 Preview at IGN|publisher=[[IGN|IGN]]|date=2011-06-02|accessdate=2016-07-06}}</ref>。例如，玩家可以执行强力近战攻击，具体取决于角色职业<ref>{{cite web|last1=Senior|first1=Tom|title=Mass Effect 3 video shows new combat skills: heavy melee, turrets, Krogan shotgun|url=https://www.pcgamer.com/mass-effect-3-video-shows-new-combat-skills-heavy-melee-turrets-krogan-shotgun/|publisher=''[[PC_Gamer|PC Gamer]]''|accessdate=2018-04-28|date=2011-12-02|archiveurl=https://www.webcitation.org/71a0mX4MT?url=https://www.pcgamer.com/mass-effect-3-video-shows-new-combat-skills-heavy-melee-turrets-krogan-shotgun/|archivedate=2018-08-11|deadurl=no}}</ref>。所有敌人都有某种形式的保护措施，但易受特定攻击的影响，例如盔甲对于焚烧技能和低[[射速|射速]]武器的抵抗较弱<ref name="Prima Combat" />。有时，玩家可以使用其他的进攻方法。例如，如果玩家遇到阿特拉斯机甲（Atlas Mech）并且在不破坏机甲的情况下击败驾驶员，薛帕德可以爬进去并控制其重型武器<ref>{{cite web|last1=Yalburgi|first1=Vinod|title='Mass Effect 3': How to Unlock Hijacker Trophy or Hijack an Atlas Mech [VIDEO]|url=https://www.ibtimes.co.uk/mass-effect-3-unlock-hijacker-trophy-atlas-333462|publisher=[[国际财经时报|International Business Times]]|accessdate=2018-04-28|date=2012-04-26|archiveurl=https://www.webcitation.org/71WwZ7Fjm?url=https://www.ibtimes.co.uk/mass-effect-3-unlock-hijacker-trophy-atlas-333462|archivedate=2018-08-09|deadurl=no}}</ref>。

===对话与道德系统===

玩家通过使用径向控制菜单与游戏中的[[非玩家角色|非玩家角色]]互动，对话选项位于对话轮的各个方向，但游戏的动作模式没有对话选项<ref name="Three Modes" />。大多数对话通过选择楷模（外交）或叛逆（恐吓）选项进一步加深，这两个选项分别位于对话轮的右上方和右下方<ref name="Prima Character" />。通过与诺曼底号上的队友交谈，薛帕德指挥官可以与他们建立友谊，在某些情况下可与他们建立恋爱关系<ref>{{cite web|last1=Hughes|first1=Matt|title=Mass Effect 3 romance guide|url=http://www.gamesradar.com/mass-effect-3-romance-guide/|publisher=''[[GamesRadar|GamesRadar+]]''|accessdate=2018-01-01|date=2012-03-09|archiveurl=https://www.webcitation.org/71KlLmOcu?url=https://www.gamesradar.com/mass-effect-3-romance-guide/|archivedate=2018-08-01|deadurl=no}}</ref>。男性或女性薛帕德都可发展同性关系<ref name=IGNSS>{{cite web|first=Daniel|last=Krupa|url=http://www.ign.com/articles/2011/05/16/mass-effect-3-to-offer-same-sex-relationships|title=Mass Effect 3 to Offer Same-Sex Relationships|publisher=[[IGN|IGN]]|date=2011-05-16|accessdate=2016-07-06}}</ref>。如果薛帕德从《质量效应2》中导入，且在游戏一二代分别有一个爱人，那么这两个角色都会在《质量效应3》中争取薛帕德的注意<ref name=IGNFate>{{cite web|url=http://www.ign.com/articles/2011/06/02/e3-2011-mass-effect-3-the-galaxys-fate-is-in-your-hands|title=E3 2011: Mass Effect 3 -- The Galaxy's Fate is in Your Hands|date=2011-06-02|accessdate=2016-07-06|publisher=[[IGN|IGN]]}}</ref>。在某些对话过程中，游戏会为玩家提示上下文敏感中断选项，该选项是对话轮的临时替代选项<ref name="Prima Character" />。

在一些情况下，玩家必须做出影响游戏叙事的决定<ref name="Decisions">{{cite web|last1=Weekes|first1=Patrick|title=Reputation in Mass Effect 3|url=http://blog.bioware.com/2012/03/01/reputation-in-mass-effect-3/|publisher=[[BioWare|BioWare]] Blog|accessdate=2018-01-01|date=2012-03-01|archiveurl=https://www.webcitation.org/71NqtOb24?url=http://blog.bioware.com/2012/03/01/reputation-in-mass-effect-3/|archivedate=2018-08-03|deadurl=no}}</ref>。例如，一个任务要求玩家支持两个种族中的一个，这可能导致另一个种族被消灭<ref>{{cite web|last1=Neltz|first1=András|title=The Most Defining Decisions You Have To Make In Mass Effect|url=https://kotaku.com/the-most-defining-decisions-you-have-to-make-in-mass-ef-471612781|website=[[Kotaku|Kotaku]]|accessdate=2018-01-01|date=2013-04-12|archiveurl=https://www.webcitation.org/71NqyIzIV?url=https://kotaku.com/the-most-defining-decisions-you-have-to-make-in-mass-ef-471612781|archivedate=2018-08-03|deadurl=no}}</ref>。在故事发展过程中，对话和叙事决定会产生“声望点”，积累后解锁进一步的决定和对话选项<ref name="Decisions" />。游戏的结局受到所获成就和游戏整个系列决定的影响<ref name="Prima Ending" />。

==开发==

《质量效应3》由BioWare开发并由[[艺电|艺电]]（EA）发行<ref>{{cite web |title=BioWare Games |url=http://www.bioware.com/en/games/ |publisher=[[BioWare|BioWare]] |accessdate=2018-06-10 |archiveurl=https://www.webcitation.org/71a1avweD?url=http://www.bioware.com/en/games/|archivedate=2018-08-11|deadurl=no}}</ref>。与系列的前几作相似，游戏由凯西·哈德森（Casey Hudson）监制，他最初与BioWare创始人{{le|葛雷格·杰斯曲克|Greg Zeschuk}}和{{le|雷·穆吉卡|Ray Muzyka}}一起开创了该系列<ref name="Drs">{{cite web |last1=Vore |first1=Bryan |title=Exclusive: Bioware's Doctors Talk Mass Effect 3 |url=https://www.gameinformer.com/b/news/archive/2011/04/13/exclusive-bioware-s-doctors-talk-mass-effect-3.aspx |publisher=''[[Game_Informer|Game Informer]]'' |accessdate=2018-06-17 |date=2011-04-13 |archiveurl=https://www.webcitation.org/71cn7VLq2?url=https://www.gameinformer.com/b/news/archive/2011/04/13/exclusive-bioware-s-doctors-talk-mass-effect-3.aspx|archivedate=2018-08-12|deadurl=no}}</ref>。首席编剧为马克·沃尔特斯（Mac Walters），他与{{le|德鲁·卡宾森|Drew Karpyshyn}}一起担任《质量效应2》的共同编剧<ref name="Drew">{{cite web |last1=Purchese |first1=Robert |title=KOTOR, Mass Effect lead writer Drew Karpyshyn leaves BioWare |url=https://www.eurogamer.net/articles/2012-02-17-kotor-mass-effect-lead-writer-drew-karpyshyn-leaves-bioware |website=[[Eurogamer|Eurogamer]] |accessdate=2018-06-10 |date=2012-02-17 |archiveurl=https://www.webcitation.org/71cnc2F8d?url=https://www.eurogamer.net/articles/2012-02-17-kotor-mass-effect-lead-writer-drew-karpyshyn-leaves-bioware|archivedate=2018-08-12|deadurl=no}}</ref>。卡宾森在《质量效应2》发行后离职，开始为[[星球大战：旧共和国|星球大战：旧共和国]]工作<ref name="Dropped Ideas" />。

===游戏设计===
早在《质量效应2》发行前，《质量效应3》的早期开发阶段就已经开始了<ref>{{cite web|title=CES 2010: BioWare Talks Mass Effect 3|author=Matt Casamassina|url=http://www.ign.com/articles/2010/01/08/ces-2010-bioware-talks-mass-effect-3|publisher=[[IGN|IGN]]|date=2010-01-08|accessdate=2016-07-06}}</ref>。因为《质量效应3》是为即将寿终正寝的[[第七世代遊戲機|游戏主机]]设计，所以BioWare更注重优化游戏玩法和故事，而不是推动技术发展<ref name="FinalH">{{cite book |last1=Keighley |first1=Geoff |title=The Final Hours of Mass Effect 3 |date=2012-03-15}}</ref>。在整个设计过程中，公司使用批评反馈指导他们的创意决策，这也是他们用在《质量效应2》的方法<ref>{{cite web|url=http://www.destructoid.com/fans-are-making-mass-effect-2-better-150282.phtml|title=Fans are making Mass Effect 2 better|last=Nicholson|first=Brad|website=Destructoid|date=2009-09-28|accessdate=2012-04-04|archiveurl=https://www.webcitation.org/71n1hPGOH?url=http://www.destructoid.com/fans-are-making-mass-effect-2-better-150282.phtml|archivedate=2018-08-19|deadurl=no}}</ref>。例如，许多粉丝抱怨《质量效应2》中的角色扮演系统过于简单，因此BioWare增加了更多角色定制选项<ref name=Designing>{{cite news|last=Schreier|first=Jason|url=https://www.wired.com/gamelife/2011/06/mass-effect-3/|title=Designing Sci-Fi RPG Mass Effect 3, BioWare Learns From Its Mistakes|website=Wired|date=2011-06-30|accessdate=2012-04-04|archiveurl=https://www.webcitation.org/71n1kPD4X?url=https://www.wired.com/gamelife/2011/06/mass-effect-3/|archivedate=2018-08-19|deadurl=no}}</ref>。另一个主要策略是让《质量效应3》对新玩家更加友好，BioWare认为这一点他们在《质量效应2》中做得不好<ref>{{cite web|author=Destin Legarie and Jack DeVries|url=http://www.ign.com/articles/2012/01/23/bioware-didnt-do-a-good-job-for-new-players-in-me2|title=BioWare 'Didn't do a Good Job' for New Players in ME2|publisher=[[IGN|IGN]]|date=2012-01-23|accessdate=2016-07-06}}</ref>。

哈德森对于单人战役的主要目标之一是改善战斗系统，希望让《质量效应3》成为市面上最好的射击游戏之一<ref name="Telegraph">{{cite web |last1=Hoggins |first1=Tom |title=Mass Effect 3 interview: Casey Hudson |url=https://www.telegraph.co.uk/technology/video-games/video-game-news/8850868/Mass-Effect-3-interview-Casey-Hudson.html |publisher=''[[每日电讯报|The Telegraph]]'' |accessdate=2018-06-17 |date=2011-10-27 |archiveurl=https://www.webcitation.org/71nbsqiqB?url=https://www.telegraph.co.uk/technology/video-games/video-game-news/8850868/Mass-Effect-3-interview-Casey-Hudson.html |archivedate=2018-08-19 |deadurl=no }}</ref>。BioWare试图通过改进敌人的动画和差异来实现这一目标，包括创造需要玩家根据角色职业使用不同战术的特殊敌人<ref>{{cite web |title=BioWare Pulse - Mass Effect 3 Combat |url=http://masseffect.bioware.com/videos/video/9757232f1c606fac575c2cf052eff5a1 |publisher=Mass Effect 3 Official Website |accessdate=2018-06-17 |date=2011-12-01}}</ref>。公司还通过引入意外事件和重新传入来降低战场的可预测性，例如地板陷落，玩家掉入完全不同的环境中，这样敌人遭遇战就不会总是像游戏简报里说的那样<ref>{{cite web |last1=Hanson |first1=Ben |title=Casey Hudson Interview: Mass Effect's Feedback Loop |url=https://www.gameinformer.com/b/features/archive/2011/04/27/casey-hudson-interview-mass-effect-s-feedback-loop.aspx |publisher=''[[Game_Informer|Game Informer]]'' |accessdate=2018-06-17 |date=2011-04-27 |archiveurl=https://www.webcitation.org/71nc4dhhV?url=https://www.gameinformer.com/b/features/archive/2011/04/27/casey-hudson-interview-mass-effect-s-feedback-loop.aspx |archivedate=2018-08-19 |deadurl=no }}</ref>。

===制作===
哈德森和沃尔特斯合作编写了一份简短的故事文档，概述了所有主要情节点，自此开始创作剧本<ref name="Writing">{{cite web |title=BioWare Pulse - Writing for Mass Effect 3 |url=http://masseffect.bioware.com/videos/video/ee64b72912e1e855d87ea6d510c3ad42 |publisher=Mass Effect 3 Official Website |accessdate=2018-06-17 |date=2012-01-12}}</ref>。然后，这些高级人员的想法被分配给团队的其他成员，每个人负责特定的角色和故事情节<ref name="Best Mission">{{cite web |last1=Phillips |first1=Tom |title=Mass Effect's best mission |url=https://www.eurogamer.net/articles/2017-03-15-mass-effects-best-mission |website=[[Eurogamer|Eurogamer]] |accessdate=2018-06-10 |date=2017-03-17}}</ref>。工作人员还负责审查彼此的工作<ref name="Writing" />。因为先前的游戏允许玩家从不同的结果中进行选择，所以编剧必须考虑大量可能的变量。但预料到这一挑战后，他们抢先锁定某些故事情节，确保《质量效应3》的所有流程至少具有一些共性<ref>{{cite web |last1=Hanson |first1=Ben |title=Casey Hudson Interview: All About Mass Effect 3 |url=https://www.gameinformer.com/b/features/archive/2011/04/28/casey-hudson-interview-mass-effect-3.aspx |publisher=''[[Game_Informer|Game Informer]]'' |accessdate=2018-06-17 |date=2011-04-28}}</ref>。整体叙事和结局在最终确定前曾多次重写。一个最初的想法是收割者试图阻止有机种族利用[[暗能量|暗能量]]<ref name="Dropped Ideas">{{cite web|url=http://www.eurogamer.net/articles/2013-06-19-ex-bioware-writer-discusses-dropped-ideas-for-mass-effect-trilogy-ending|title=Ex-BioWare writer discusses dropped ideas for Mass Effect trilogy ending|author=Philips, Tom|website=[[Eurogamer|Eurogamer]]|date=2013-06-19|accessdate=2017-05-14|archiveurl=https://www.webcitation.org/71cnoxBH8?url=https://www.eurogamer.net/articles/2013-06-19-ex-bioware-writer-discusses-dropped-ideas-for-mass-effect-trilogy-ending|archivedate=2018-08-12|deadurl=no}}</ref>。

《质量效应3》需要4万多句对话<ref name="DialogPCG">{{cite web |last1=Purslow |first1=Matt |title=Mass Effect 3 contains twice as much dialogue as the first game |url=https://www.pcgamer.com/mass-effect-3-contains-twice-as-much-dialogue-as-the-first-game/ |website=[[PC_Gamer|PC Gamer]] |accessdate=2018-08-11 |date=2011-10-13}}</ref>。相比之下，初代《质量效应》大约有2万句，《质量效应2》则有2.5万句<ref name="DialogPCG" />。由于游戏配音阵容庞大，逻辑上BioWare不可能同时让每个演员担任同一个位置<ref name="Recording">{{cite web |title=BioWare Pulse - Voice Over Recording |url=http://masseffect.bioware.com/videos/video/1ebd676a117f893ecdf71eded065820b |publisher=''Mass Effect 3'' Official Website |accessdate=2018-08-11 |date=2012-01-05}}</ref>。担任画外音监制的卡罗琳·利文斯通（Caroline Livingstone）协调来自[[埃德蒙顿|埃德蒙顿]]的录制会话，而演员们则远程工作<ref name="Recording" />。BioWare注意到，这种方法结合了大量的对话，使得其难以持续捕获每句对话正确的[[语调|语调]]和[[响度|响度]]<ref name="DialogPCG" />。为了减少这些错误，他们开发了自己的技术，例如“强度响度矩阵”（Intensity Volume Matrix），它列出了每一句话以及与之相关的适当音调<ref name="Recording" />。

最初《质量效应3》计划在两年内完成<ref name="FinalH" />。然而到2011年3月，游戏明显需要更多时间开发，因此BioWare将游戏的发行日期从2011年第四季度推迟到2012年第一季度<ref name="Release" />。为了满足最后期限，公司还裁剪了一些故事，例如从游戏本体移除新的普洛仙队友<ref name="FinalH" />。11月，《质量效应3》的私人[[软件测试|beta测试]]泄露到[[Xbox_Live|Xbox Live]]，玩家可玩到未发行的故事情节，BioWare被迫临时修改故事的某些方面<ref name="EuroLeak">{{cite web |last1=Dutton |first1=Fred |title=Mass Effect 3 story files leaked |url=https://www.eurogamer.net/articles/2011-11-07-mass-effect-3-story-files-leaked |website=[[Eurogamer|Eurogamer]] |accessdate=2018-08-25 |date=2011-11-07}}</ref>。《质量效应3》已于2012年2月正式批准分发<ref name="FinalH" />。

===音乐===
《质量效应3》的音乐由萨沙·迪克西安（Sasha Dikicyan）、山姆·哈利克、{{le|克里斯托弗·楞内茨|Christopher Lennertz}}、[[克林特·曼塞爾|克林特·曼塞爾]]和克里斯·贝拉斯科创作<ref name="Composers">{{cite web|title=These Are The People Composing Mass Effect 3's Soundtrack|url=http://www.gameinformer.com/b/news/archive/2011/12/19/these-are-the-people-composing-mass-effect-3s-soundtrack.aspx|publisher=''[[Game_Informer|Game Informer]]''|date=2011-12-19|accessdate=2011-12-19}}</ref>。迪克西安和贝拉斯科之前为《质量效应2》的一些可下载内容包创作音乐<ref name="ScoreInt">{{cite web |last1=Rougeau |first1=Michael |title=Interview: "Mass Effect 3" Composers On Bringing The Music Back To Its Roots And More |url=https://www.complex.com/pop-culture/2012/03/interview-mass-effect-3-composers-on-bringing-the-music-back-to-its-roots-and-more |magazine=[[Complex_(magazine)|Complex]] |accessdate=2018-07-27 |date=2012-03-12}}</ref>。但《质量效应》和《质量效应2》的首席作曲家{{le|杰克·沃尔|Jack Wall}}并没有参与《质量效应3》的制作<ref>{{cite web |last1=Kietzmann |first1=Ludwig |title=Mass Effect 2 composer Jack Wall explains his departure from the franchise |url=https://www.engadget.com/2011/03/04/mass-effect-2-composer-jack-wall-explains-his-departure-from-the/ |publisher=[[Engadget|Engadget]] |accessdate=2018-06-10 |date=2011-03-04}}</ref>。

每个作曲家都被分配了一个特定的游戏区域，总共有90分钟的[[游戏音乐|配乐]]<ref name="ScoreInt"/>。根据哈利克的说法，团队旨在于《质量效应2》的[[管弦樂團|管弦乐]]音与《质量效应》的[[合成器|合成]]音之间取得平衡<ref name="ScoreInt" />。[[金球奖|金球奖]]提名作曲家曼塞尔（Mansell）把他担任的角色比作[[唱片骑师|唱片骑师]]，负责在合适的时间选择合适的音乐<ref>{{cite web |last1=Acharya |first1=Kiran |title=Unspoilt By Progress: A Conversation With Black Swan Composer Clint Mansell |url=http://thequietus.com/articles/05670-clint-mansell-interview-black-swan |publisher=''[[The_Quietus|The Quietus]]'' |accessdate=2018-06-10 |date=2011-02-09}}</ref>。[[Kotaku|Kotaku]]将配乐评为2012年最佳电子游戏音乐之一<ref>{{cite web |accessdate=2013-05-21 |title=The Best Video Game Music of 2012 |first=Kirk |last=Hamilton |date=2012-12-13 |website=[[Kotaku|Kotaku]] |url=http://kotaku.com/5968289/the-best-video-game-music-of-2012 |archiveurl=https://archive.today/20130521094426/http://kotaku.com/5968289/the-best-video-game-music-of-2012 |archivedate=2013-05-21 |deadurl=no}}</ref>。

评论员认为，对质量效应三部曲来说，《质量效应3》是一个令人满意的终章。在[[Eurogamer|Eurogamer]]的一篇满分评测中，丹·怀特黑德（Dan Whitehead）称游戏讲述了一个接近传奇的完整故事，而不是更模糊或者开放式的故事<ref name="eurog-review" />。[[Game_Informer|Game Informer]]的安德鲁·雷纳（Andrew Reiner）称BioWare创造了电子游戏史上最“精雕细琢”的故事之一<ref name="gi-review" />。[[PC_Gamer|PC Gamer美国]]的汤姆·弗朗西斯（Tom Francis）评论道结局是一个“大杂烩”，但最终赞扬了它的规模和情感影响，他认为这比其主要情节更令人满意<ref name="pcgus-review">{{cite web|url=http://www.pcgamer.com/review/mass-effect-3-review/|title=Mass Effect 3 review|last=Francis|first=Tom|date=2012-03-06|magazine=[[PC_Gamer|PC Gamer]]|accessdate=2012-03-06}}</ref>。

评论员强调了使用通关存档玩《质量效应3》的体验。[[Polygon|Polygon]]的阿瑟·吉斯（Arthur Gies）认为，虽然游戏没有任何前情提要，但它对那些也玩过《质量效应》和《质量效应2》的玩家来说更有意义<ref>{{cite web |last1=Gies |first1=Arthur |title=Mass Effect 3 Review: The Sharp Edge of Hope |url=https://www.polygon.com/2012/10/2/3443570/mass-effect-3-review-the-sharp-edge-of-hope |website=[[Polygon|Polygon]] |accessdate=2018-07-28 |date=2012-03-06}}</ref>。[[影音俱樂部|影音俱樂部]]的古斯·马斯特拉帕（Gus Mastrapa）赞同这些观点，并称赞《质量效应3》为每个玩家分配他们的选择和叙事方向的灵活性<ref>{{cite web |last1=Mastrapa |first1=Gus |title=Reviews: Mass Effect 3 |url=https://games.avclub.com/mass-effect-3-1798230284 |publisher=[[影音俱樂部|The A.V. Club]] |accessdate=2018-07-29 |date=2012-03-12}}</ref>。相反的，《[[Edge_(杂志)|Edge]]》杂志认为《质量效应3》过于依赖前作的事件，无法让它作为独立作品树立声誉<ref name="edge-review">{{cite web|url=http://www.edge-online.com/features/mass-effect-3-review|title=Mass Effect 3 review|date=2012-03-06|magazine=[[Edge_(杂志)|Edge]]|accessdate=2012-03-06}}</ref>。

《质量效应3》的战斗系统和多人游戏模式收获好评。[[IGN|IGN]]的科林·莫里亚蒂（Colin Moriarty）认为战斗系统与前两作相比虽然变化很小，但仍很明显，并且预计多人模式会增加游戏的重玩价值<ref name="ign-review" />。而Game Informer指出战斗系统并不完美，但他们仍持正面看法并得出结论，敌人遭遇战比《质量效应2》更加激烈有趣，部分原因是环境和敌人类型的多样<ref name="gi-review" />。另一方面，[[GameSpot|GameSpot]]的凯文·范奥德（Kevin VanOrd）讲述了战斗中遇到的无数小毛病，比如小队成员爬上板条箱把自己置于火线之中和敌人卡在墙上<ref name="gspot-review" />。[[Machinima公司|Machinima]]还注意到了许多错误，但认为《质量效应3》仍是一次令人难忘的体验，游戏的强大角色足以弥补其缺陷<ref name="MCXbox" />。

《质量效应3》其他受到好评的方面包括美术设计、音乐和[[配音|配音]]。GameSpot称为“梦幻般”的美术设计，并特别关注游戏对颜色和构图的使用<ref name="gspot-review" />。在同一篇评测中，网站赞扬了游戏的配音，它让主要人物变得栩栩如生<ref name="gspot-review" />。PC Gamer美国特别提到了游戏“华丽”的音乐，认为其显著增加了游戏的情感影响<ref name="pcgus-review" />。[[The_Verge|The Verge]]更进一步，称这是有史以来为电子游戏打造的最“强大和令人难忘”的配乐之一<ref>{{cite web |title=Sam Hulick - Biography |url=https://samhulick.com/#bio |publisher=Sam Hulick Official Website |accessdate=2018-07-28}}</ref>。

==续作==
{{Main|质量效应：仙女座}}
2017年3月21日，续作《[[质量效应：仙女座|质量效应：仙女座]]》发行<ref>{{cite web |title=Mass Effect: Andromeda IGN Page |url=http://www.ign.com/games/mass-effect-andromeda |publisher=[[IGN|IGN]] |accessdate=2018-07-07}}</ref>。游戏故事开始于22世纪的银河系，但主要剧情发生在634年后的[[仙女座星系|仙女座星系]]<ref>{{cite web |title=Everything We Know About Mass Effect: Andromeda |url=http://www.escapistmagazine.com/articles/view/video-games/columns/everything-we-know-about/17480-Everything-You-Need-to-Know-About-Mass-Effect-Andromeda#&gid=gallery_7055&pid=1 |publisher=''[[The_Escapist|The Escapist]]'' |accessdate=2018-07-07 |date=2017-03-17 |author=The Escapist Staff}}</ref>，讲述了新主角——莎拉.莱德（Sara Ryder）或斯科特 莱德（Scott Ryder）——为人类寻找新的家园的故事<ref>{{cite web |last1=Williams |first1=Hayley |title=What I Learned From Three Hours With Mass Effect: Andromeda |url=https://www.kotaku.com.au/2017/02/what-i-learned-from-three-hours-with-mass-effect-andromeda/ |website=[[Kotaku|Kotaku]] |accessdate=2018-07-07 |date=2017-02-24}}</ref>。游戏发行后受到了媒体褒贬不一的评价<ref>{{cite web |title=Mass Effect: Andromeda for PlayStation 4 Reviews |url=http://www.metacritic.com/game/playstation-4/mass-effect-andromeda |publisher=[[Metacritic|Metacritic]] |accessdate=2018-07-07}}</ref>。截至2018年6月，BioWare宣布尚未有正式的续作计划，但凯西·哈德森提到工作室最终打算重返该系列<ref>{{cite web |last1=Roberts |first1=Samuel |title=BioWare is 'figuring out' Dragon Age 4, intends to 'get back to' Mass Effect |url=https://www.pcgamer.com/bioware-has-people-figuring-out-dragon-age-4-intends-to-get-back-to-mass-effect/ |publisher=''[[PC_Gamer|PC Gamer]]'' |accessdate=2018-07-07 |date=June 2018}}</ref>。
