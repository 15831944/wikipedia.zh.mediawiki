'''質量百分濃度'''又稱'''重量百分濃度'''，縮寫wt%，是一種表示[[混合物|混合物]]中特定物質[[濃度|濃度]]的方法，是特定物質的[[質量|質量]]<math>m_i</math>相對於所有物質總質量<math>m_{tot}</math>的比例，定義為<ref>{{GoldBookRef | file = M03722 | title = mass fraction}}</ref>

:<math>w_i = \frac {m_i}{m_{tot}}</math>

混合物中所有物質的質量百分濃度總和為1：

:<math>\sum_{i=1}^{N} m_i = m_{tot} ; \sum_{i=1}^{N} w_i = 1</math>

質量百分濃度經常會用[[百分比|百分比]]表示，是用[[無量綱|無量綱]]表示混合物中各成份濃度的方法，其他無量綱的濃度有[[摩尔分数|摩尔分数]]（[[摩尔|摩尔]]數相對總摩尔數的百分比，符號mol%）及[[体积分数|体积分数]]（[[體積|體積]]相對總體積的百分比，符號vol%）。

在[[元素分析|元素分析]]中，質量百分濃度也可以指在[[化合物|化合物]]中某一元素質量佔的比例，可以用來計算化合物的[[實驗式|實驗式]]<ref>{{cite web|url=http://khanexercises.appspot.com/video?v=xatVrAh2U0E|title=Khan Academy|work=khanacademy.org|deadurl=yes|archiveurl=https://web.archive.org/web/20100717123508/http://khanexercises.appspot.com/video?v=xatVrAh2U0E|archivedate=2010-07-17}}</ref>或[[化學式|化學式]]<ref>{{cite web|url=http://chemistry.about.com/od/workedchemistryproblems/a/mass-percent-worked-problem.htm|title=How to Calculate Mass Percent Composition|author=Anne Marie Helmenstine, Ph.D.|work=About.com Education}}</ref>。

溶液的質量百分濃度一般是指[[溶質|溶質]]的質量百分濃度，可以表示為為[[溶質|溶質]]重量除以[[溶劑|溶劑]]和[[溶質|溶質]]總重量之[[百分比|百分比]]值。

==詞語及相關詞語==
「百分浓度」不一定是指此物理量，不過這個不正確的名稱仍然存在，尤其在一些小學的教科書中。在生物學中，單位"%"有時會用來表示濃度，稱為「質量／體積濃度」，若溶液中有一克的溶質，總體積為100 mL，有時會標示為"1 %"或"1 % m/v"（質量／體積），這是不正確的，因為單位"%"只能用來標示無因次量。其實，此濃度的單位應該是g/mL。「溶液百分比」比較適合改為「質量百分比濃度」（m/m = m% = 溶質質量/混合後總質量）或「體積百分比濃度」（v/v = v% = 溶質體積/混合後總體積），不過常常還會出現「溶液百分比」一詞。

在{{link-en|熱工學|thermal engineering}}中會用[[蒸氣量|蒸氣量]]來描述蒸氣質量佔總質量的比例。合金，特別是貴金屬，會以[[成色|成色]]表示貴金屬在合金中的質量比例。

==性質==
質量百分濃度不受溫度的影響。

==和其他濃度單位的關係==

===質量濃度===
溶液中成份的質量百分濃度是其成份i在溶液中的[[質量濃度|質量濃度]]<math>\rho_i</math>和溶液[[密度|密度]] <math>\rho</math>的比值：

:<math>w_i = \frac {\rho_i}{\rho}</math>

===體積莫耳濃度===
質量百分濃度和[[體積莫耳濃度|體積莫耳濃度]]的關係類似質量百分濃度和質量濃度的關係，將上式用體積莫耳濃度取代即可：

:<math>w_i = \frac {\rho_i}{\rho}=\frac {c_i M_i}{\rho}</math>

其中<math>M_i</math>是成份<math>i</math>的[[分子量|分子量]]。
===摩尔分数===
[[摩尔分数|摩尔分数]]<math>x_i</math>可以用下式計算：

:<math>x_i = w_i \cdot \frac {M}{M_i}</math>

其中
:<math>M_i</math>為成份<math>i</math>的分子量
:<math>M</math>為混合物的平均分子量。

將上式的平均分子量用各成份的分子量取代，可得：

:<math>x_i = \frac {\frac{w_i}{M_i}}{\sum_i \frac{w_i}{M_i}}</math>

==空間濃度差異及擴散==
若混合物在不同位置的質量百分濃度不同，其質量百分濃度的梯度會產生[[擴散|擴散]]作用。
