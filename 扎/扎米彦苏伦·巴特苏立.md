'''扎米彦苏伦·巴特苏立'''（Jamiyansuren Batsuuri，{{bd|？|||}}）[[蒙古国|蒙古国]]人，蒙古国政治人物。

==生平==
巴特苏立曾任[[东戈壁省|东戈壁省]]省长，并曾两次当选[[国家大呼拉尔|国家大呼拉尔]]委员<ref>{{Cite web |url = https://www.infomongol.mn/content/read/70930.htm |title = President disagrees with PM on new ministers |author =  |date =2016-07-22  |publisher =  infomongol|language =  |accessdate =  |quote =  }}</ref>。他是[[蒙古人民党|蒙古人民党]]党员。2016年，出任[[额尔登巴特内阁|额尔登巴特内阁]]教育文化科学与体育部部长<ref>[http://mn.china-embassy.org/chn/mgdt/t1387596.htm 蒙古新一届内阁部长名单，中华人民共和国驻蒙古国大使馆，2016-08-01]</ref><ref>{{Cite web |url =https://www.cia.gov/library/publications/world-leaders-1/MG.html  |title =Chiefs of State and Cabinet Members of Foreign Governments: Mongolia  |author =  |date =  |publisher =  CIA|language =  |accessdate =2017-06-26  |quote =  }}</ref><ref>{{Cite web |url = http://www.fmprc.gov.cn/ce/cemn/chn/tpxw/t1390733.htm |title = 邢海明大使会见蒙古教育科学文化与体育部长巴特苏立 |author =  |date =2016-08-22  |publisher =中华人民共和国驻蒙古国大使馆  |language =  |accessdate =  |quote =  }}</ref>。
