{{engname|《庐山恋》|''Romance on Lushan Mountain''}}由[[上海电影制片厂|上海电影制片厂]]1980年出品，著名[[导演|导演]][[黄祖模|黄祖模]]的作品。1981年获第四届电影[[百花奖|百花奖]]最佳故事片和最佳女演员奖，[[张瑜_(演员)|張瑜]]更凭借本片当选第一届[[金鸡奖|金鸡奖]]最佳女主角。

本片是“[[文革|文革]]”后中国首部爱情为主题的电影，并且出现了当时罕见的吻戏，张瑜也凭之成为1980年代观众心中的“梦中情人”。电影也大大提升了庐山在人们心目中的地位。

根據[[吉尼斯世界纪录大全|吉尼斯世界纪录大全]]，本電影是在單一電影院上映最久的電影，它于1980年7月12日在[[江西省|江西省]][[廬山|廬山]]電影院首映后，一直每天播放到現在。
[[File:Love_Story_in_Lushan_Cinema.jpg|300px]]

==情节==
以风景优美的[[庐山|庐山]]为背景，讲述两个年轻人的爱情故事。

1972年[[上海公报|上海公报]]签署后，侨居[[美国|美国]]的[[国民党|国民党]]将军周振武的女儿周筠回祖国观光。在庐山游玩的途中她巧遇有志青年耿桦。两个对祖国怀有无限热爱且充满远大志向的青年走到一起，经过短短几天的相处，他们渐渐产生了感情。然而很快耿桦因与周筠亲密接触受到审查，周筠也郁郁离开。

[[文革|文革]]结束后，已是[[清华大学|清华大学]]研究生的耿桦在庐山和周筠重逢，这对相爱之人不愿再被分开，彼此约定婚期。而当耿桦的父亲得知周筠的身世时，两人的婚事受到了严峻的考验。

==庐山恋电影院==
庐山恋电影院的前身是庐山基督教协和礼拜堂，始建于1897年，由英国传教士[[李德立|李德立]]主持兴建。在20世纪30年代，[[蒋介石|蒋介石]][[宋美龄|宋美龄]]夫妇常来到这里做礼拜。1960年，礼拜堂被改造为“东谷电影院”。1980年7月12日，电影《庐山恋》在此首映，此后该电影院只播放这一部电影从未间断。20世纪90年代，这个电影院又更名为“庐山恋电影院”（又称“庐山恋影院”）。2002年12月12日，庐山恋电影院获得[[吉尼斯世界纪录|吉尼斯世界纪录]]。<ref>{{cite web |author1=程迪 |title=一吻38年的“庐山恋歌” |url=http://www.xinhuanet.com/politics/2018-10/12/c_1123549303.htm |website=新华网 |publisher=新华每日电讯 |accessdate=2018-12-12 |archiveurl=https://web.archive.org/web/20181019194330/http://www.xinhuanet.com/politics/2018-10/12/c_1123549303.htm |archivedate=2018-10-19 |date=2018-10-12}}</ref><ref>{{cite web |author1=郭翠潇 |title=《庐山恋曲》 |url=http://www.cctv.com/program/zbzg/topic/geography/C18966/20070718/105052.shtml |website=央视网 |accessdate=2012-07-14 |archiveurl=https://web.archive.org/web/20120714111107/http://www.cctv.com/program/zbzg/topic/geography/C18966/20070718/105052.shtml |archivedate=2018-10-19 |date=2007-07-18}}</ref><ref>{{cite web |title=庐山恋 |url=http://www.china-lushan.com/lushanlian/index.php |website=中国庐山网 |accessdate=2018-12-12 |archiveurl=https://web.archive.org/web/20180828124355/http://www.china-lushan.com/lushanlian/index.php |archivedate=2018-08-28}}</ref><ref>{{cite web |author1=何夕 |title=庐山基督教礼拜堂：庐山上唯一开放的教堂，宋美龄曾在此做礼拜 |url=http://www.gospeltimes.cn/index.php/portal/article/index/id/40459 |website=福音时报 |accessdate=2018-12-12 |archiveurl=https://web.archive.org/web/20170705044901/http://www.gospeltimes.cn/index.php/portal/article/index/id/40459 |archivedate=2017-07-05 |date=2017-04-12}}</ref>
