'''特提斯洋'''（Tethys Ocean）又名'''古地中海'''，是個[[中生代|中生代]]時期的[[海洋|海洋]]，位於[[勞亞大陸|勞亞大陸]]與[[岡瓦納大陸|岡瓦納大陸]]之間。

== 過去理論 ==
在1893年，[[奧地利|奧地利]]地質學家[[愛德華·蘇斯|愛德華·休斯]]（Eduard Suess）根據[[阿爾卑斯山脈|阿爾卑斯山脈]]與[[非洲|非洲]]的[[化石紀錄|化石紀錄]]，
提出過去在[[勞亞大陸|勞亞大陸]]與[[岡瓦納大陸|岡瓦納大陸]]之間，曾有個淺[[內海|內海]]存在者。休斯將這個史前海域命名為'''特提斯海'''（Tethys Sea），以[[希臘神話|希臘神話]]中的海神[[忒堤斯|忒堤斯]]為名。後來提出的[[板塊構造論|板塊構造論]]，推翻了休斯的許多假設，並將特提斯海改為'''特提斯洋'''（Tethys Ocean）。但是，休斯的特提斯海概念仍有其正確處，並在當時引起相當大的關注，因此休斯仍被認為是特提斯海與特提斯洋的發現者。

== 現代理論 ==
約在2億5000萬年前的晚[[二疊紀|二疊紀]]，[[盤古大陸|盤古大陸]]南部（後來的岡瓦納大陸）的北緣出現一道裂谷，[[辛梅利亞大陸|辛梅利亞大陸]]開始分裂，[[古特提斯洋|古特提斯洋]]的南邊開始出現新的海洋。在接下來的6,000萬年間，[[辛梅利亞板塊|辛梅利亞板塊]]與盤古大陸分離，往北方移動，使古特提斯洋往盤古大陸北部（勞亞大陸）的東南緣縮小。盤古大陸南部與辛梅利亞大陸（現今的[[土耳其|土耳其]]、[[伊朗|伊朗]]、[[阿富汗|阿富汗]]、[[西藏|西藏]]、[[中南半島|中南半島]]、[[馬來亞|馬來亞]]）之間出現新海洋－特提斯洋，取代原本古特提斯洋的位置。

在1億5000萬年前的[[侏羅紀|侏羅紀]]，辛梅利亞大陸與盤古大陸北部（後來的勞亞大陸）碰撞、接合，並形成[[隱沒帶|隱沒帶]]，稱為[[特提斯海溝|特提斯海溝]]。[[海平面|海平面]]上升，特提斯洋延伸入西邊的[[歐洲|歐洲]]。同一時期盤古大陸分裂為勞亞大陸與岡瓦納大陸，[[大西洋|大西洋]]開始出現。在侏儸紀與[[白堊紀|白堊紀]]期間（約一億年前），岡瓦納大陸開始分裂，[[非洲|非洲]]與[[印度|印度]]往北離開，穿越特提斯洋，[[印度洋|印度洋]]開始出現。特提斯洋的四周都有陸塊推進着，在1500萬年前的[[中新世|中新世]]晚期，特提斯洋縮減為特提斯海道（Tethys Seaway），或稱第二特提斯海。

現今[[印度|印度]]、[[印尼|印尼]]、印度洋等區域，過去曾被特提斯洋覆蓋者。現今的[[地中海|地中海]]是西特提斯洋的殘餘部份，而[[黑海|黑海]]、[[裏海|裏海]]、與[[鹹海|鹹海]]則是[[副特提斯海|副特提斯海]]的殘餘部份。特提斯洋的海底，大部分隱沒到辛梅利亞大陸與勞亞大陸之下。休斯等[[地理學家|地理學家]]在[[喜馬拉雅山脈|喜馬拉雅山脈]]的岩層中發現海洋生物的化石，這個地區過去是特提斯洋的海底，直到印度大陸與[[基梅里大陸|基梅里大陸]]碰撞，使海底上升。歐洲的[[阿爾卑斯造山運動|阿爾卑斯造山運動]]也有類似的證據，顯示[[非洲板塊|非洲板塊]]造成[[阿爾卑斯山脈|阿爾卑斯山脈]]。

對[[古生物學家|古生物學家]]而言，特提斯洋非常重要，因為特提斯洋的周圍有許多陸棚，可在這些昔日陸棚發現棲息於海洋、沼澤、河口的許多生物化石，可研究長時間的生物變化。

== 相關名詞與海域 ==
許多網路上的特提斯海，其實指的是特提斯洋，反之亦然。但有些網站將侏儸紀時期出現的大西洋，與特提斯海混淆。

特提斯洋的西部，稱為西特提斯洋（Western Tethys Ocean）、阿爾卑特提斯海（Alpine Tethys Ocean）、特提斯海。黑海、裏海、鹹海被認為是它們的殘餘部份，但黑海有可能是更古老的古特提斯洋的遺跡。但西特提斯洋並非單一個開放性海洋，而是分佈者[[火山島|火山島]]與[[微大陸|微大陸]]。西特提斯洋的東部分布者[[瓦莱洋|瓦莱洋]]、[[皮埃蒙特-利古里亞洋|皮埃蒙特-利古里亞洋]]，以[[亞伯蘭板塊|亞伯蘭板塊]]、[[伊比利亞板塊|伊比利亞板塊]]、[[阿普利亞板塊|阿普利亞板塊]]上的陸塊為界。[[中生代|中生代]]時期的海平面很高，各大陸的許多部分成為淺海。在[[新生代|新生代]]時期，歐洲中部與東部被[[副特提斯洋|副特提斯洋]]（Paratethys）覆蓋着。副特提斯海又稱[[潘諾尼亞海|潘諾尼亞海]]（Pannonian Sea），這個海域在中新世末期消失，成為獨立的[[內海|內海]]，並在[[更新世|更新世]]乾涸，成為陸地。

特提斯洋的東部，則被稱為東特提斯洋（Eastern Tethys Ocean）。

隨者相關理論的提出，科學家們將特提斯一詞，用在更古老的相似海洋上。古特提斯洋，出現在4億4000萬年前的[[志留紀|志留紀]]，位在Hunic岩層與盤古大陸南部（之後的南界先後為岡瓦納大陸、基梅里大陸）之間，在侏儸紀時期消失。[[原特提斯洋|原特提斯洋]]出現在6億年前的[[埃迪卡拉紀|埃迪卡拉紀]]，在3億6000萬年前的[[泥盆紀|泥盆紀]]消失，位在[[波羅地大陸|波羅地大陸]]與盤古大陸北部、南部之間。原特提斯洋、古特提斯洋有時會與[[瑞克洋|瑞克洋]]產生混淆，瑞克洋存在於志留紀，在上述兩個海洋的西方，波羅地大陸、[[阿瓦隆尼亞大陸|阿瓦隆尼亞大陸]]、盤古大陸南部之間。
