'''特伦蒂诺-上阿迪杰'''或'''特伦蒂诺-南蒂罗尔'''（{{lang-it|Trentino-Alto Adige}}，{{IPA-it|trenˈtiːno ˈalto ˈaːdidʒe|pron}}，{{lang-de|link=no|Trentino-Südtirol}};<ref>{{cite web |url= http://www.regione.taa.it/normativa/statuto_speciale.pdf |title= Trentino-Alto Adige/Südtirol Region |accessdate=20 February 2009 |work= Official website of the Trentino-Alto Adige/Südtirol Region|year=2009}}</ref> {{lang-lld|Trentin-Südtirol}}<ref>[http://www.pensplan.com/ld/information/index.asp PensPlan Project of the Region]</ref>）是意大利北部的一个自治区。自治区由两个分区组成，分别是多数人说意大利语的[[特伦托省|特伦托省]]及多数人说德语的[[南蒂罗尔省|南蒂罗尔省]]。该区在直至1919年被意大利吞并前是[[奥匈帝国|奥匈帝国]]的一部分。在1919年至1947年间该区被称为威尼西亚—特伦托天拿（{{lang|it|Venezia Tridentina}}）。

== 地理及经济 ==

自治区北界与[[奥地利|奥地利]]相邻，西邻[[伦巴第|伦巴第]]大区及南邻[[威尼托|威尼托]]大区。自治区面积13,607平方公里（5,256 [[平方英里|平方英里]]）。自治区环境极为多山，覆盖[[多洛米蒂山|多洛米蒂山]]大部及南[[阿尔卑斯山|阿尔卑斯山]]。横过阿尔卑斯山最低的关口，[[布伦纳罗山口|布伦纳罗山口]]，是位于自治区北面与奥地利边界之上。

特伦蒂诺-上阿迪杰肥沃的山谷生产酒、生果、奶类制品及木材，并包括制纸、化工及金属工业。该区是[[水力发电|水力发电]]能源一个主要的来源地。旅游业是区内重要的收入来源，并以其能在冬季于该地滑雪而闻名。

== 人口统计资料 ==

特伦蒂诺-上阿迪杰的人口大约950,000人（460,000人居住于[[波尔查诺|波尔查诺]]及480,000人居住于特伦托）。该区主要的种族是说意大利语的种族（总数大约百分之六十）及说德语的种族总数（大约百分之三十五），另外还有少数人说[[拉迪恩语|拉迪恩语]]（总数约百分之五）。于波尔查诺地区或南蒂罗尔，主要通行的语言是德语（人口大约百分之六十八），但是在波尔查诺本市则有四分之三的人口是说意大利语的。在特伦托省则只有很少讲德语的人。不同于南蒂罗尔，对说[[少数民族语言|少数民族语言]]的人的保护，于特伦托省会较少。

== 历史 ==
由11世纪开始，区内的一部分由[[特伦托|特伦托]]及[[布里申|布里申]]的[[贵族主教|贵族主教]]统治，被[[神圣罗马帝国|神圣罗马帝国皇帝]]给予大量的世俗权力，并多于其本身的职权。该区是[[蒂罗尔郡|蒂罗尔郡]]的一部分，由14世纪开始是[[哈布斯堡王室|哈布斯堡王室]]的领地。这个状况于19世纪初神圣罗马帝国解散后完结。主教的职位于1803年被政府取缔，并将该区重新被给予哈布斯堡王室。两年后，当奥地利在[[奥斯特里茨战役|奥斯特-{里}-茨战役]]中溃败之后，整个地区被[[拿破仑|拿破仑]]的傀儡[[意大利王国_(拿破仑)|意大利王国]]吞并；当拿破仑被打败后，1815年，该区重归奥地利统治。

在[[第一次世界大战|第一次世界大战]]期间，[[奥匈帝国|奥匈帝国]]军队与意大利军队主要的战役于阿尔卑斯山及多洛米蒂山发生，以控制南蒂罗尔这个战略目标。奥匈帝国于战争中的崩溃令意大利军队可以于1918年占领该区，而对该区的统治亦于战后的和约得到确定，于[[圣日耳曼条约|圣日耳曼条约]]之下意大利被授予[[特伦蒂诺|特伦蒂诺]]及[[南蒂罗尔|南蒂罗尔]]。
