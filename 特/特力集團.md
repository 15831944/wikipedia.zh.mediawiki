'''特力股份有限公司'''（Test Rite International Co., Ltd.），簡稱：'''特力集團'''，是[[台灣|台灣]]一個以[[貿易|貿易]]與[[零售|零售]]通路為事業重心的[[企業集團|企業集團]]，由[[何湯雄|何湯雄]]、[[李麗秋|李麗秋]]夫婦創立。

== 沿革 ==
1974年，以從事[[五金|五金]]工具進出口代理在台灣[[貿易|貿易]]業界著名的[[李麗秋|李麗秋]]創立「秋記貿易」，後來在1978年將秋記貿易改組為特力公司，以從事進出口貿易為主。之後，李麗秋與身為自己下屬的[[何湯雄|何湯雄]]結婚，李麗秋與何湯雄兩人一內一外，除了使特力公司的事業版圖邁向國際，並跨足[[零售|零售]]領域<ref>[http://www.businessweekly.com.tw/article.php?id=46391 特力集團總裁何湯雄：我就是錯在自以為是經營之神！ - 《商業周刊》第1273期 2012.04.16]</ref>。

1995年，特力公司與[[英國|英國]][[翠豐集團|翠豐集團]]（Kingfisher Group）合作，將[[DIY|DIY]]器材量販店[[B&Q|B&Q]]引進台灣。之後，又自行成立HOLA特力和樂、HOMY特力和家等品牌，大力發展以[[生活用品|生活用品]]為中心的[[量販店|量販]]事業。特力公司除了供貨予旗下的零售通路之外，同時也是[[沃爾瑪|沃爾瑪]]（Wal-Mart）、[[家得寶|家得寶]]（Home Depot）等跨國通路商的供應商。此外，特力公司在2007年至2012年5月間擁有工具製造商[[東隆五金|東隆五金]]的多數[[股權|股權]]<ref>[http://www.appledaily.com.tw/appledaily/article/finance/20120522/34244688/ 每股41.05元 特力擬賣東隆五金股權 - 台灣《蘋果日報》 2012.05.22]</ref>。

== 旗下事業 ==
* [[特力貿易|特力貿易]] -特力貿易事業群有三大業務模式，分別是：貿易、代理、進口。佔比最大為貿易，主要買家在歐美地區
* [[特力屋|特力屋]] -工具材料專賣店；原為特力集團與翠豐集團合資的「B&Q特力屋」，2007年特力集團收回自營後改為現名
* [[特力和樂|特力和樂]]（HOLA） - [[家居用品|家居用品]]專賣店，以自家代理的進口品牌為主和其他賣場區隔
* [[和樂名品傢俱|和樂名品傢俱]]（HOLA CASA） - [[高階床具|高階床具]]專賣店，前身為HOMY特力和家
* [[特家(股)公司|特家(股)公司]] - 經營國際品牌台灣總代理業務，共有收納，餐廚，家電，寢具四大類別，知名品牌如WMF，KitchenAid，ELLE，iris，Lodge，Ecovacs等等
* [[中欣實業|中欣實業]] - 建築[[營建|營建]]與[[材料|材料]]商
* [[鴻駿訊息|鴻駿訊息]]（B&S Link） - [[電子商務|電子商務]]商
