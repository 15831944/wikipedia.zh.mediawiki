'''麥可·特倫特·雷澤諾'''（{{lang-en|Michael Trent Reznor}}，{{bd|1965年|5月17日}}）是美國[[創作歌手|創作歌手]]、[[作曲家|作曲家]]和[[唱片製作人|唱片製作人]]。雷澤諾自1988年組成[[工業搖滾|工業搖滾]]樂團[[九吋釘|九吋釘]]，在團內擔任主唱，也演奏多種樂器；他在2007年離開[[Interscope唱片|Interscope唱片]]，現為獨立音樂家。2010年，他和他的妻子[[瑪麗科·曼迪格|瑪麗科·曼迪格]]，以及他的音樂夥伴[[阿提喀斯·羅斯|阿提喀斯·羅斯]]，合組[[後工業音樂|後工業]]三人樂團「[[如何消滅天使|如何消滅天使]]」<ref name = "HF1">{{cite web |author=HitFix |date=2011年12月8日 |url=http://www.hitfix.com/articles/trent-reznor-pegs-early-2012-release-date-for-first-how-to-destroy-angels-album-will |title=Trent Reznor pegs early 2012 release date for How to Destroy Angels full-length |work=[http://www.hitfix.com/ ''HitFix''] |accessdate=2011年12月29日 |language=en}}</ref><ref name = "NBHAP1">{{cite web |author=Helbig R |date=2010年6月3日 |url=http://nothingbuthopeandpassion.com/136/how-to-destroy-angels/ |title=How to Destroy Angels |work=[http://nothingbuthopeandpassion.com/ ''Nothing But Hope and Passion''] |accessdate=2011年12月29日 |language=en}}</ref><ref name="TMN1">{{cite web|author=Martinelli M |date=2011年12月8日 |url=http://thenewmontreal.com/how-to-destroy-angels/ |title=How to Destroy Angels |work=[http://thenewmontreal.com/ ''The New Montreal''] |accessdate=2011年12月29日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20120426085604/http://thenewmontreal.com/how-to-destroy-angels/ |archivedate=2012年4月26日 }}</ref>。雷澤諾和羅斯曾合作為[[大衛·芬奇|大衛·芬奇]]電影《[[社群網戰|社群網戰]]》和《[[龍紋身的女孩_(2011年電影)|龍紋身的女孩]]》編寫[[電影配樂|配樂]]，並憑前者獲得[[奧斯卡最佳原創音樂|奧斯卡最佳原創音樂]]和[[金球獎最佳原創音樂|金球獎最佳原創音樂]]。
