'''特別行動突擊隊'''（{{lang-de|Spezialeinsatzkommando}}，[[英文|英文]]：Special Operations Command，[[縮寫|縮寫]]：'''SEK'''）是隸屬[[德國|德國]]各邦警察的[[特種警察部隊|特種警察部隊]]，其主要責任與[[第九國境守備隊|第九國境守備隊]]相似，但是前者接受各邦直屬調查，後者只是接受[[德國聯邦警察|德國聯邦警察]]指揮。

== 組織 ==
特別行動突擊隊的編制由各邦決定，因此每邦至少都會在其首府設立一個特別行動突擊隊，但是如[[北莱茵-威斯特法伦|北莱茵-威斯特法伦]]與[[萊因蘭-法爾茲|萊因蘭-法爾茲]]兩邦以幾個犯罪率較高的地方為中心設立，而[[巴伐利亞|巴伐利亞]]與[[黑森|黑森]]邦則擁有兩支部隊，各自負責該邦的北部與南部。

通常該單位之地位可以等同於該邦的鎮暴部隊（{{lang-de|Bereitschaftspolizei}}）或是一個區域警局，然而通常狀況是該單位被置於各邦警察的刑案調查局（{{lang-de|Landeskriminalamt}}）之下。

由於各邦的地理環境不同，例如駐守於南巴伐利亞的部隊擁有一支可以應對高山事件的小組；而[[不萊梅|不萊梅]]與[[漢堡|漢堡]]邦則擁有可以應付海事的小組，部份特別行動突擊隊設有談判小組以應付人質挾持或自殺事件等。

特別行動突擊隊隊員經常跟MEK特遣隊進行聯絡。機動刑事特遣隊 ({{lang-de|Mobile Einsatzkommandos}}，[[英文|英文]]：Mobile Task Force，[[縮寫|縮寫]]'''MEK''') 的主要責任為處理[[拯救|拯救]][[人質|人質]]、作出策略性的快速拘捕、[[要員保護|要員保護]]等，其入職要求也比SEK低。

== 裝備 ==
特別行動突擊隊隊員不會身穿制服，但會使頭套來保密其身分。為求節約時間，因此會看到特別行動突擊隊人員身穿便服，套上戰術背心和防彈衣，以及頭套來執行任務的特殊裝扮。

一名特別行動突擊隊隊員在行動時通常會攜帶一支[[手槍|手槍]]和一支[[衝鋒槍|衝鋒槍]]；隊員更可以自行購買個人認為對任務最有利的裝備而不受約束。
