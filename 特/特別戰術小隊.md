'''特別戰術小隊'''（暫定名稱；俗稱'''速龍小隊'''；[[英文|英文]]：'''S'''pecial '''T'''actical '''S'''quad，[[縮寫|縮寫]]：'''STS'''，俗稱：'''Elite Team'''）於2014年6月成立，於大型衝突時會被召喚增援，隸屬於[[香港警務處|香港警務處]][[行動處|行動處]][[行動部|行動部]][[警察機動部隊總部|警察機動部隊總部]]，是[[準軍事化|準軍事化]]的[[特遣隊|特遣]][[防暴警察|防暴警察]]，主要責任為大型事件中[[防暴|防暴]]、維持[[內部保安|內部保安]]，於情況被長久拖延以至失控，及[[警察機動部隊|警察機動部隊]]行動被嚴重阻礙時才出動，以迅速的手法移除障礙及執行[[鎖定|鎖定]]及[[拘捕|拘捕]]行動，防微杜漸<ref>[http://www.singpao.com/xw/gat/201412/t20141210_540496.html 警明清場Elite Team為主力 若示威者不衝擊不動亂 不會用武力] {{webarchive|url=https://web.archive.org/web/20141210102933/http://www.singpao.com/xw/gat/201412/t20141210_540496.html |date=2014-12-10 }} 《成報》 2014年12月10日</ref>，並且提供[[緊急醫療服務|緊急醫療服務]]等。此外，特別戰術小隊於行動現場亦擔負觀察及即時指導等任務，確保警察機動部隊所使用的[[武力|武力]]水平合適以及[[戰術|戰術]]套用方式正當，並且於行動完成後作出檢討，以及提交改善措施建議等等。

== 組織 ==
特別戰術小隊為一支[[特遣隊|特遣隊]]，編制為一[[排|排]]（即小隊），主要由[[警察機動部隊總部|警察機動部隊總部]][[教官|教官]]（擅長大型事件中[[防暴|防暴]]及[[內部保安|內部保安]]中的[[戰術|戰術]]及[[武力|武力]]使用等）和[[香港警察學院|香港警察學院]]教官團隊（主要為[[槍械訓練科|槍械訓練科]]武力使用教官，擅長武力使用及射擊等）所組成，以及[[衝鋒隊_(香港)#組織|切割戰術隊]]（擅長處理突發事件及切割戰術等）等，另外亦有小部份來自[[機場特警組|機場特警組]]、[[反恐特勤隊|反恐特勤隊]]<ref>[http://news.singtao.ca/toronto/2014-11-27/hongkong1417068782d5343701.html 「藍衫軍」皆警隊精英] 《星島日報》 2014年11月27日</ref><ref>[http://orientaldaily.on.cc/cnt/news/20141127/00176_010.html 警150精兵組移除隊清障] 《東方日報》 2014年11月27日</ref><ref>[http://hk.apple.nextmedia.com/news/art/20141201/18953865 飛虎　機場特警　辣手清場] 《蘋果日報》 2014年12月1日</ref>及[[特別任務連|特別任務連]]（於2014年12月11日加入）<ref>[http://www.singpao.com/xw/yw/201412/t20141212_540751.html 警方清場鬥智鬥力 首設「財物大隊」 排出「銀行櫃位」] {{webarchive|url=https://web.archive.org/web/20150924124228/http://www.singpao.com/xw/yw/201412/t20141212_540751.html |date=2015-09-24 }} 《成報》 2014年12月12日</ref>等；由於特別戰術小隊為特遣隊，而且尚未正式成立，故此每次出動之人數從數十至數百不定<ref>[http://orientaldaily.on.cc/cnt/news/20141211/00176_002.html 飛虎加入速龍小隊 加強清障能力] 《東方日報》 2014年12月11日</ref>。絕大部份人員曾經駐守於[[警察機動部隊|警察機動部隊]]，擁有一定年資，富有行動經驗。特別戰術小隊僅在有需要時才會集合，崗位屬於兼任職務。

== 歷史 ==
為了應對[[讓愛與和平佔領中環|佔領中環]]，[[警察機動部隊|警察機動部隊]][[校長|校長]][[歐陽照剛|歐陽照剛]][[總警司|總警司]]於2014年6月月底組織了一支[[特遣隊|特遣隊]]，作為警察機動部隊的一道中堅力量，確保[[防暴|防暴]]任務順利。然而此特遣隊一直未有官方名稱，[[香港警務處架構|警務處內部]]一直稱呼其為'''移除隊'''（[[英文|英文]]：'''Removal Team'''），[[香港傳媒|香港傳媒]]則稱呼其為'''專業移除隊'''。隨著佔領中環發生，以至演變為後來的[[雨傘革命|雨傘革命]]，特遣隊持續地曝光；至同年11月30日，歐陽照剛總警司將其暫定名稱為特別戰術小隊<ref>[http://the-sun.on.cc/cnt/news/20141201/00407_026.html 藍衣軍正名「特別戰術小隊」] 《太陽報》 2014年12月1日</ref>。

特別戰術小隊在[[香港警務處|警務處]]處理佔領中環的日峰行動中表現出色，備受[[香港警務處憲委級職級|警務處高層人員]]所認同，因而有意將其[[特遣隊|特遣]]地位改為正式[[香港警務處架構|單位]]，歸納為[[香港警務處架構|警務處架構]]中<ref>[http://www.metrohk.com.hk/?cmd=detail&id=257922 NEWS力場 藍衫小隊改咩名？] 《都市日報》 2014年12月12日</ref>；至雨傘革命完結後，再作決定<ref>[http://hk.apple.nextmedia.com/news/art/20141208/18961877 高層指示前線　警棍只打腳] 《蘋果日報》 2014年12月8日</ref>。

2016年2月9日凌晨，旺角因小販被禁止擺檔而引發[[2016年農曆新年旺角騷亂|警民衝突]]，大量過去未被示威者使用的新戰術湧現，結果觸發警方無法招架，而需要再度召喚特別戰術小隊到場處理，首次清場於凌晨4時30分在旺角[[豉油街|豉油街]]與[[西洋菜南街|西洋菜南街]]交界，成功於30秒內驅散示威者。但於短短五分鐘後，因防備不足而全線敗退，更有一名成員受傷。

== 制服 ==
*[[貝雷帽|貝雷帽]]：新款於2002年4月引進，由[[警察機動部隊總部|警察機動部隊總部]]研製，採用柔軟、透氣、防褪色和防脫毛的絨料製造<ref>[http://www.police.gov.hk/offbeat/724/chi/n04.htm 運動短褲及貝雷帽] 《警聲》第724期</ref>。
*工作服：[[深藍色|深藍色]]<ref>[http://orientaldaily.on.cc/cnt/news/20141201/00176_006.html 藍衣警獲正名「特別戰術小隊」] 《東方日報》 2014年12月1日</ref>，由於與其他多個[[香港警務處架構|警察單位]]所穿著的制服（包括[[警察機動部隊總部|警察機動部隊總部]]工作服、[[水警總區|水警總區]]工作服、[[警犬隊_(香港)|警犬隊]]行動服、[[警察搜查隊|警察搜查隊]]工作服及[[機場特警組|機場特警組]]工作服等）雷同，為避免[[香港傳媒|傳媒]]混淆以及方便公眾識別，約11月月底至12月月初，特別戰術小隊工作服右胸位置被縫上「Police Tactical Unit Special Tactical Squad」字樣（為繼[[爆炸品處理課|爆炸品處理課]]後第二個縫有單位名稱的警察單位的單位工作服或者行動服）。此外，人員亦於日間在[[盔|盔]]頂貼上顏色貼紙、於夜間扣上[[螢光棒|螢光棒]]（包括[[黃色|黃色]]及[[紅色|紅色]]，根據分隊而定），以方便指揮官識別。
