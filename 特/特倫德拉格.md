'''特倫德拉格'''（{{lang|no|Trøndelag}}）位於[[挪威|挪威]]中部，依[[ISO_3166-2:NO|ISO 3166-2:NO]]標準編號順序，包括[[南特倫德拉格|南特倫德拉格]]與[[北特倫德拉格|北特倫德拉格]]兩個[[挪威行政區劃#郡|郡]]，總面積達{{convert|41260|sqkm|sqmi|0}}<ref>{{cite web |url=http://www.statkart.no/?module=Files;action=File.getFile;ID=24831 |work=Statens kartverk |title=Arealstatistikk for 2006 |date=2006-01-01 |language=no |accessdate=2008-08-28}}</ref>，依2008年7月所公告統計結果<ref>{{cite web |url= http://www.ssb.no/folkendrkv/ |work=Statistisk Sentralbyrå |title=Kvartalsvise befolkningsendringer |language=no |accessdate=2008-08-28}}</ref>，人口共有414,965人。也有習慣將特倫特區域兩個郡與[[默勒-魯姆斯達爾|默勒-魯姆斯達爾]]，合稱中挪威。

特倫德拉格的名稱，出自於當地古老的'''特倫德'''部落（{{lang|no|Trønder}}），結合挪威語中的'''法律'''（{{lang|no|lag}}）一詞，意思是「依法是特倫德的領域」。該區域內，最大的城市是[[特隆赫姆|特隆赫姆]]（{{lang|no|Trondheim}}），曾經是挪威歷史上的第一個首都，以及文化上被視為特倫德拉格的首都，來自於特倫德拉格的人，稱為特倫德人（{{lang|no|Trønder}}），方言特徵是在結尾處常常省略了[[元音|元音]]。

默勒-魯姆斯達爾郡的[[北默勒|北默勒]]（{{lang|no|[[:no:Nordmøre|Nordmøre]]}}）與[[魯姆斯達爾|魯姆斯達爾]]（{{lang|no|[[:no:Romsdal|Romsdal]]}}）兩地域，以及[[諾爾蘭郡|諾爾蘭]]的[[賓達爾區|賓達爾區]]（{{lang|no|[[:no:Bindal|Bindal]]}}），傳統上也屬於特倫德拉格的一部份，當地居民的方言與特倫特拉格非常接近。

鄰國[[瑞典|瑞典]]北部[[耶姆特蘭省|耶姆特蘭省]]內的[[耶姆特蘭|耶姆特蘭]]與[[海里耶達倫|海里耶達倫]]，歷史上為特倫德拉格的一部份，因此兩地又被稱為[[東特倫德拉格|東特倫德拉格]]（{{lang|sv|[[:sv:Öst-Tröndelag|Öst-Tröndelag]]}}）。

== 歷史 ==
數千年前，特倫德拉格就有人類生活，早在[[日耳曼|日耳曼]][[鐵器時代|鐵器時代]]（4世紀至8世紀），散布著許多小王國（被稱為{{lang|no|fylki}}），幾個小國間會訂定共同的法律，有早期的[[議會|議會]]與[[聯合政府|聯合政府]]（{{lang|no|thing}}），建立在[[夫羅斯塔|夫羅斯塔]]（{{lang|no|[[:no:Frosta|Frosta]]}}），稱為[[夫羅斯塔庭|夫羅斯塔庭]]（{{lang|no|[[:no:Frostating|Frostating]]}}），被視為是人類史上第一個[[民主|民主政體]]。

9世紀中葉，[[拉德|拉德]]（{{lang|no|[[:no:Lade|Lade]]}}，今特隆赫姆東邊）出現一位領主（{{lang|no|Jarl}}）[[賀孔·格約噶爾松|賀孔·格約噶爾松]]（{{lang|no|[[:no:Håkon_Grjotgardsson|Håkon Grjotgardsson]]}}），拉德領主（{{lang|no|Ladejarl}}，又譯作拉德伯爵）的政治影響，一直延續到1030年。

拉德領主包括有：
* '''賀孔·格約噶爾松'''（{{lang|no|[[:no:Håkon_Grjotgardsson|Håkon Grjotgardsson]]}}），首位拉德領主。
* '''西于爾·賀孔松'''（{{lang|no|[[:no:Sigurd_Håkonsson|Sigurd Håkonsson]]}}），賀孔·格約噶爾松之子，被挪威王[[哈拉尔二世_(挪威)|哈拉爾二世]]（{{lang-no|[[:no:Harald_Gråfell|Harald II (Eiriksson) Gråfell]]}} ，“灰袍”）所殺。
* '''賀孔·西于爾松'''（{{lang|no|[[:no:Håkon_Sigurdsson|Håkon Sigurdsson]]}}），西于爾·賀孔松之子，與丹麥'''藍牙王'''[[蓝牙王哈拉尔|哈拉爾一世]]（{{lang-da|[[:da:Harald_Blåtandll|Harald Blåtand]]}}）聯手對抗哈拉爾二世，後附庸於藍牙王，當挪威王[[奧拉夫一世_(挪威)|奧拉夫一世]]興起，賀孔迅速失勢，995年被自己的奴隸所殺。
* '''埃里克·賀孔松'''（{{lang|no|[[:no:Eirik_Håkonsson|Eirik Håkonsson]]}}），賀孔·西于爾松之子，與其胞兄弟'''斯維·賀孔松'''（{{lang|no|[[:no:Svein_Håkonsson|Svein Håkonsson]]}}），在丹麥王[[斯萬一世|斯萬一世]]（{{lang-da|[[:da:Svend_Tveskæg|Svend Tveskæg]]}}）授權下，從1000年至1012年，統治挪威。
* '''賀孔·埃里克松'''（{{lang|no|[[:no:Håkon_Eiriksson|Håkon Eiriksson]]}}），埃里克·賀孔松之子，在斯萬一世授權下，從1012年至1015年，統治挪威。

17世紀中，[[瑞典國王|瑞典國王]][[卡爾十世·古斯塔夫|卡爾十世·古斯塔夫]]實行擴張主義，與[[丹麥-挪威|丹麥-挪威]]爭雄，1658年，瑞典佔領挪威中部的[[特隆赫姆|特隆赫姆]]，雙方因此簽訂[[羅斯基勒條約|羅斯基勒條約]]，丹麥-挪威將特倫德拉格割讓給瑞典王國，兩年後再由丹麥-挪威收復。

瑞典國王卡爾十世擔心特倫德人會起而反抗，曾於特倫德拉格[[徵兵|徵兵]]，約2,000名年齡15歲以上的年青男孩加入瑞典軍隊，以對抗[[波蘭|波蘭]]與[[勃蘭登堡|勃蘭登堡]]，最後僅有三分之一的特倫德人回到故鄉，其中部份被逼安排定居於當時瑞典的國土[[愛沙尼亞|愛沙尼亞]]，以利管治。

1804年，特倫德拉格分為[[南特倫德拉格|南特倫德拉格]]與[[北特倫德拉格|北特倫德拉格]]兩個郡。

== 文化 ==
特隆赫姆有著名的'''[[特倫德拉格劇院|特倫德拉格劇院]]'''（{{lang|no|[[:no:Trøndelag_Teater|Trøndelag Teater]]}}），那是特倫德拉格官方提供的大型演出場地；另外在[[斯蒂克萊斯塔德|斯蒂克萊斯塔德]]（{{lang|no|[[:no:Stiklestad|Stiklestad]]}}，又譯作'''史狄克斯達德'''），從1954年開始，每年都會演出描述[[聖奧拉夫二世|聖奧拉夫二世]]逝世故事的地方戲劇，成為本區的特色。

本區常為人熟悉的是[[私釀酒|私釀酒]]'''[[卡斯克|卡斯克]]'''（{{lang|no|[[:no:karsk|karsk]]}}），地方美食有'''[[索德|索德]]'''（{{lang|no|[[:no:sodd|sodd]]}}），是一種水煮[[羊肉|羊肉]]與肉球，拌以[[馬鈴薯|馬鈴薯]]、[[紅蘿蔔|紅蘿蔔]]等球莖植物的主食。

本區有一種著名的[[搖滾樂|搖滾樂]]，融入了地區方言，稱為'''特倫德搖滾'''；在特隆赫姆另有非常出色的[[爵士樂|爵士樂]]。

挪威一種頻危的'''灰特倫德羊'''（{{lang-no|Grå trøndersau}}、{{lang-en|[[:en:Grey_Troender_sheep|Grey Troender]]}}），18世紀晚期在特倫德拉格被馴化畜牧，目前僅剩約50頭，有關單位正積極復育中。
