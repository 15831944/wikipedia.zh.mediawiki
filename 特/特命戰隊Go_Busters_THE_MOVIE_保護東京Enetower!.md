『'''特命戰隊Go Busters THE MOVIE 保護東京Enetower!'''』（日文：{{lang|ja|特命戦隊ゴーバスターズTHE MOVIE 東京エネタワーを守れ!}}）是2012年8月4日 [[東映|東映]]公開預定上映的『[[超級戰隊系列|超級戰隊系列]]』的『[[特命戰隊Go_Busters|特命戰隊Go Busters]]』劇場版作品，與『[[假面騎士系列|假面騎士系列]] 』的『[[假面騎士Fourze|假面騎士Fourze]] 』劇場版『[[假面騎士Fourze_THE_MOVIE_大家的宇宙來了!|假面騎士Fourze THE MOVIE 大家的宇宙來了!]] 』同時上映。

== 概要 ==
: 日本人氣歌手[[辻希美|辻希美]]也將在本作舉場版中，現聲擔任配音
: 同時本作劇場版，是在[[東京鐵塔|東京鐵塔]]的全力協助下完成拍攝。

== 故事概要ー ==
: Busters在被稱作是『'''東京Enetower'''』的Enetower旅行時，巧遇宿敵・Vaglass的Enter製造出來的新Metaloids並且進行了戰鬥，為了保護『'''東京Enetower'''』，Busters展開了新的戰鬥。

== 登場人物 ==
=== 特命戰隊Go Busters  ===
{{See|特命戰隊Go Busters #登場角色}}
; 櫻田 弘（{{lang|ja|桜田 ヒロム}}） / Red Buster（{{lang|ja|レッドバスター}}）
: 特命部的ID為555・913・315。有話就直說，經常是一針見血，也因為這樣有時會被Nick責罵。在疫苗程式的影響下獲得了瞬間加速能力。

; 岩崎龍次（{{lang|ja|岩崎 リュウジ}}） / Blue Buster（{{lang|ja|ブルーバスター}}）
: 特命部的ID為222・458・804。在疫苗程式的影響下獲得了臂力強化能力。

; 宇佐見陽子（{{lang|ja|宇佐見 ヨーコ}}） / Yellow Buster（{{lang|ja|イエローバスター}}）
: 特命部的ID為777・117・464。很討厭念書，特別是完全不懂英文。在疫苗程式的影響下獲得了腿力強化能力。

; 陣雅人（{{lang|ja|陣 マサト}}） / Beet Buster（{{lang|ja|ビートバスター}}）
: 特命部的ID為198・662・125。是個天才，但性格輕率。

===  Buddyroid  ===
; 豹田・Nick（{{lang|ja|チダ・ニック}}）
: 弘的Buddyroid，雖然有最先進的導航卻是個路癡，像是弘的大哥一樣。

; 猩崎・Banana（{{lang|ja|ゴリサキ・バナナ}}）
: 龍次的Buddyroid，容易情緒低落，極度愛操心，總是擔心著龍次。

; 兔田・Lettuce（{{lang|ja|ウサダ・レタス}}）
: 陽子的Buddyroid，頑固任性，講話句句帶刺，平時兼任陽子的老師。

; Beet・J・Stag（{{lang|ja|ビート・J・スタッグ}}）/ Stag Buster（{{lang|ja|スタッグバスター}}）
: 雅人的Buddyroid，由雅人製作，是唯一一台能變身成Go Busters的Buddyroid。

; Ene-Tan（{{lang|ja|エネたん}}）
: 劇場版的原創角色。[[青蛙|青蛙]]型的小型Buddyroid。
: FS-0O的駕駛導航，雖然是Go Busters的支援但是很討厭對方在自己毫無準備時接觸它。

=== 司令室 ===
; 黑木 猛（{{lang|ja|黒木 タケシ}}）
: 能量管理局特命部的司令官。

; 仲村美穗（{{lang|ja|仲村 ミホ}}）
: 能量管理局特命部的美女操作員。

; 森下徹（{{lang|ja|森下 トオル}}）
: 能量管理局特命部的帥哥操作員。

=== Vaglass ===
; Enter
: Vaglass中唯一的幹部，以人類的身姿進行間諜活動的怪人，可以隨意變換造型。
