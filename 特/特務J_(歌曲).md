《'''特務J'''》（{{Lang-en|'''Agent J'''}}）是臺灣歌手[[蔡依林|蔡依林]]的一支單曲，由[[國會音樂集團|國會音樂]]於2007年8月29日發行，收錄於蔡依林的專輯《[[特務J|特務J]]》；該歌曲於[[Hit_FM聯播網|Hit FM]]年度百首單曲排行榜位居第14名<ref>{{Cite web|url=http://www.hitoradio.com/newweb/chart_2.php?ch_year=2007|title=Hitoradio‧Hit Fm --華人音樂入口指標|accessdate=2017-12-01|work=www.hitoradio.com}}</ref>。

== 背景與發行 ==
臺灣歌手[[蔡依林|蔡依林]]於2006年5月12日發行專輯《[[舞孃_(專輯)|舞孃]]》，該專輯於[[兩岸三地|兩岸三地]]大獲成功，於臺灣的銷量高達30萬，並奪得臺灣年度唱片銷量排行榜總冠軍<ref>{{Cite web|url=http://ent.qq.com/a/20090323/000439_1.htm|title=蔡依林十年音乐路③：EMI时期“流行教主”_娱乐_腾讯网|accessdate=2017-12-01|work=ent.qq.com}}</ref><ref>{{Cite web|url=http://ent.sina.com.cn/y/2007-11-18/16321797061.shtml|title=2007网络盛典音乐类年度港台女歌手：蔡依林_影音娱乐_新浪网|accessdate=2017-12-01|work=ent.sina.com.cn}}</ref>。蔡依林於2007年9月21日發行第九張錄音室專輯《[[特務J|特務J]]》，該專輯以[[歌舞片|音樂電影]]為概念，並拍攝同名音樂電影《[[特務J_(電影)|特務J]]》，總製作成本共5千萬新臺幣<ref>{{Cite web|url=http://music.yule.sohu.com/20070913/n252124541.shtml|title=蔡依林烧5千万赢张惠妹 阿妹粉丝炮轰EMI不公平-搜狐音乐|accessdate=2017-12-01|work=music.yule.sohu.com}}</ref>。專輯的第一主打歌曲《特務J》於同年8月29日發布，該歌曲戲劇性的編曲搭配強勁的節奏，蔡依林亦於該歌曲中展現冷酷的唱腔<ref>{{Citation|title=蔡依林:爱情任务(CD)|date=2007|url=https://www.amazon.cn/%E8%94%A1%E4%BE%9D%E6%9E%97-%E7%88%B1%E6%83%85%E4%BB%BB%E5%8A%A1/dp/B00118OZTM|publisher=国际文化交流音像出版社|isbn=9787799627786|accessdate=2017-12-01}}</ref>。該歌曲由臺灣音樂人李宗恩、[[嚴云農|嚴云農]]與梁錦興共同填詞，臺灣音樂人黃晟峰譜曲，臺灣音樂人[[小安|小安]]編曲，臺灣音樂人[[阿弟仔|阿弟仔]]製作<ref>{{Cite web|url=https://www.mymusic.net.tw/ux/w/song/show/p000007-a0037305-s000120-t001-c2|title=特務 J{{!}}特務 J｜蔡依林-懂你想聽的-MyMusic|accessdate=2017-12-01|work=國語-懂你想聽的-MyMusic|language=zh-TW}}</ref>；該歌曲編曲人小安憑藉該歌曲獲得[[第19屆金曲獎|第19屆臺灣金曲獎]][[最佳編曲人獎|最佳編曲人獎]]，製作人阿弟仔亦憑藉該歌曲獲得[[最佳單曲製作人獎_(金曲獎)|最佳單曲製作人獎]]<ref>{{Cite web|url=http://ent.sina.com.cn/f/y/GMA2008list/|title=第19届金曲奖获奖名单_新浪网|accessdate=2017-12-01|work=ent.sina.com.cn}}</ref>。蔡依林亦為該歌曲MV學習新的舞蹈，包括飛赴英國向澳大利亞籍舞蹈老師Ari Lavanae學習[[空中絲帶|空中絲帶]]舞與向英國舞蹈學校Pineapple Dance Studios的舞蹈老師Adam Jay學習[[钢管舞|鋼管舞]]<ref>{{Cite web|url=http://ent.sina.com.cn/y/d/2007-09-03/11421700414.shtml|title=专辑：蔡依林--《特务J》(预购版)_影音娱乐_新浪网|accessdate=2017-12-01|work=ent.sina.com.cn}}</ref>。該歌曲MV由臺灣導演張時霖執導，於同年9月3日發布，該歌曲MV與韓國演員[[金載沅|金載沅]]於法國巴黎拍攝<ref>{{Cite web|url=http://enjoy.eastday.com/e/20070906/u1a3090856.html|title=蔡依林《特务J》MV首映 KISS要帅哥不要色狼-蔡依林-东方娱乐-东方网|accessdate=2017-12-01|work=enjoy.eastday.com}}</ref>，蔡依林於MV中的「特務」造型由日本造型師河部菜津子打造<ref>{{Cite web|url=http://music.yule.sohu.com/20070824/n251753713.shtml|title=蔡依林“特务J”造型大曝光 短发特务惊艳万分-搜狐音乐|accessdate=2017-12-01|work=music.yule.sohu.com}}</ref>。

== 爭議事件 ==
因為「特務」二字於中國大陸屬於敏感字眼，因此歌曲名稱於中國大陸發行的專輯中由《-{zh-cn:特务J; zh-hant:特務J}-》改為《-{zh-cn:爱情任务; zh-hant:愛情任務}-》；此外，該歌曲歌詞中的「完美-{zh-cn:特务J; zh-hant:特務J}-」與「我是愛情派來的間諜」亦於歌詞本與MV中被改成發音相近的「完美的舞J」與「我是愛情派來的茧蝶」<ref>{{cite web|url=http://politics.caijing.com.cn/2012-10-31/112246059.html|title=文化部称近日禁播带“死”字歌曲传言有误|author=吕强|date=2012年10月31日|work=财经网}}</ref>。
