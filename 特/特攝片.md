「'''特攝'''」，是指使用[[特殊效果|特殊效果]]的電影特技手法（SFX）拍攝的[[電視劇|電視劇]]及[[電影|電影]]。<ref>{{Cite news|url=http://ent.sina.com.cn/v/m/2006-02-17/0946988788.html|title=广电总局下发《通知》 叫停“真人动画”(附图)|language=zh-cn|publisher=新浪网|author=|date=2006年2月17日|accessdate=2015-03-29}}</ref><ref>{{Cite news|url=http://info.toys.hc360.com/2006/02/21082833534.shtml|title=真人动画《变身战士阿龙》可能将被封杀|language=zh-cn|publisher=慧聪网玩具行业|author=|date=2006年2月21日|accessdate=2015-03-29}}</ref>或是專指[[日本|日本]]制作的使用電影特技的影片，可以稱為'''狹義特攝'''的'''日本特摄影片'''。
“'''特攝'''”英語即為“'''Tokusatsu'''”，專指日本特攝片，但對應原意其實是指特殊效果的電影特技手法（SFX）。

== 中義的特攝影片 ==
就一般所理解特技效果只限於[[幻想|幻想]]故事或[[超現實|超現實]]題材，即使是在普通的[[動作片|動作片]]中使用，也不過是拍攝真實世界常發生的事情。

== 狹義的日本特摄影片 ==

[[日本語|日本語]]“'''{{lang|ja|特撮}}'''（{{lang|ja|とくさつ}}）”是[[和製漢語|和製漢語]]，原本為“特殊摄影技術（SFX）”的略称（相當於[[特殊效果|特殊效果]]，'''特效'''）。

“{{lang|ja|特撮}}”在[[日本|日本]]指一種大量使用[[電影特技|電影特技]]的[[影片|影片]]，專指是一些超現實或[[科幻|科幻]]題材的作品，自成一類。相對即使是[[警匪片|警匪片]]等[[動作片|動作片]]雖然也使用特效，但題材是拍在真實世界經常性發生的事件。

在日本雖然有先進的電影特技，但因為日語的影片在世界市場較狹。而且電視台和電影公司常有把超現實題材拍攝成[[動畫|動畫]]的傾向，而日本[[ACG|ACG]]又有使用很多電影的鏡頭手法，吸引了很多喜歡超現實和幻想題材的人的市場，造成較少人觀看由真人演出的電影特技影片。

結果在日本較少像[[美國|美國]]那樣把動作片以電影特技拍攝，也鮮有像[[香港|香港]]把[[武俠|武俠]]片大量使用特技，在日本所見的動作和武俠片多以寫實見稱的。在日本的武俠片概念比較接近漢語的[[功夫片|功夫片]]，而動作片更少見充滿著對'''穿著便服'''普通人的特務和殺手，作神話英雄式描述成[[魔幻寫實主義|魔幻寫實主義]]風格的（相對下[[傑克·鮑爾|-{zh-hans:杰克·鲍尔;zh-hk:包智傑;zh-tw:傑克·鮑爾;}-]]和[[007|007]]跟[[美國隊長|美國隊長]]的分別主要是服裝而不是能力和場面）。

變成不得不使用特技的場合才使用的“專門特摄影片”出現了。

通常就是[[科幻片|科幻片]]和[[恐怖電影|恐怖電影]]，在日本壓倒性多是以真實現代日本做背景的[[超級英雄|超級英雄]]和'''[[怪獸|怪獸]]'''與[[怪人|怪人]]故事，以希望吸引一些不全接受動畫中完全虛構世界觀，又希望看到超現實的事物的觀眾。而且使用真實的背景也意味佈景和服裝的需求較低，成本比完全虛構背景的嚴肅科幻低，在日本常有以動畫的方式來拍攝壯大的純虛構背景的[[奇幻|奇幻]]和科幻故事。

特攝電影：[[哥吉拉|哥吉拉]]

日本目前有三大特攝電視影集系列：
*[[超人力霸王系列|超人力霸王系列]]：是巨大化的超人，像《[[Ultraman|Ultraman]]》、《大鐵人17》
*[[假面騎士系列|假面騎士系列]]：是身大的單體英雄，像《幪面超人》、《電腦奇俠》
*[[超級戰隊系列|超級戰隊系列]]：是一般為五個的戰隊英雄，像《秘密戰隊五連者》、《JAKQ電擊隊》
