'''特雷利烏'''（{{lang|es|Twelew}}{{IPA-es|treˈleu̯|Spanish pronunciation}}）<ref>[http://www.trelew.gov.ar/ Video presentation of the town on the official website]</ref>是[[阿根廷|阿根廷]]的城市，位於該國中部，[[丘布特省|丘布特省]]東北部的丘布特河河畔，距離首都[[布宜諾斯艾利斯|布宜諾斯艾利斯]]有1451公里，距離省會的羅森有20公里<ref name="welcomeargentina">{{cite web |title=Trelew |url=https://www.welcomeargentina.com/trelew/ |website=Welcome Argentina |accessdate=2018-07-04 |language=es}}</ref>，面積249平方公里，海拔高度11米，每年平均降雨量約222毫米。此市是[[丘布特河|丘布特河]]下游最大的城市，阿根廷[[巴塔哥尼亞|巴塔哥尼亞]]地區人口第四多的城市，2010年人口99,201。<ref name="histórica">{{cite web |title=Ciudad histórica |url=https://www.trelew.gov.ar/ciudad-de-trelew/trelew/ |publisher=特雷利烏市政府 |accessdate=2018-07-04 |language=es}}</ref><ref>{{cite web |title=Dirección General de Estadística y Censos de la provincia de Chubut, Cantidad de habitantes por localidad |url=http://www.estadistica.chubut.gov.ar/sitio/inicio |publisher=丘布特省統計暨普查局 |accessdate=2018-07-04 |language=es}}</ref><ref name="geografia">{{cite web |title=Geografía |url=https://www.trelew.gov.ar/ciudad-de-trelew/geografia/ |publisher=特雷利烏市政府 |accessdate=2018-07-04 |language=es}}</ref>

特雷利烏是該地區重要的商業和工業中心，也是羊毛加工的主要中心，物品主要由[[德塞阿多港|德塞阿多港]]及[[馬德林港|馬德林港]]進出口。<ref name="histórica"/>

== 歷史 ==
[[File:Ex_Estación_Ferroviaria_de_Trelew_(I).JPG|thumb]]
特雷利烏是[[阿根廷的威爾斯人定居點|阿根廷的威爾斯人定居點]]<ref name="geografia"/>，{{link-en|Love Jones-Parry|Sir Love Jones-Parry, 1st Baronet}}爵士及{{link-en|Lewis Jones|Lewis Jones (Patagonia)}}這兩個人是阿根廷拓荒的威爾士移民首領。城市之所以命名為特雷利烏是為了紀念Lewis Jones，在[[威爾士語|威爾士語]]裡，'''tre'''是城鎮的意思，而'''Lew'''則是Lewis字尾音消失後的詞根，合在一起就成了特雷利烏（Trelew）。<ref name="vistasdelvalle">{{cite web |title=Bienvenidos a Trelew - Chubut - Patagonia Argentina |url=http://www.vistasdelvalle.com.ar/trelew/index.htm |accessdate=2018-07-05 |language=es}}</ref><ref>{{cite news|title=Trelew: a tour of the city|url=http://www.patagonia.com.ar/Trelew/116E_Trelew%3A+a+tour+of+the+city.html|accessdate=2015-05-08|publisher=Patagonia.com.ar|language=en}}</ref>特雷利烏為{{link-en|丘布特中央鐵路|Central Chubut Railway}}的起點連接丘布特河谷下游與馬德林港，該城市於1886年10月20日建立<ref name="histórica"/>，而鐵路建設設施與400名移民被「維斯塔」號火車於同年7月28日運送到這裡。<ref>{{cite book| title=Y Wladfa Gymrei| first=Lewis| last=Jones| year=1898}}</ref>1888年鐵路建成通車，以後鐵路又從特雷利烏延伸至[[蓋曼_(丘布特省)|蓋曼]]、{{link-en|多拉豐|Dolavon}}，最後延伸至[[拉斯普盧馬斯|拉斯普盧馬斯]]，1961年該線路被關閉。<ref>{{cite web| url=http://www.ferrocarrilesenelconosur.co.uk/07Sx15report1961.html| title=Cancelación definitiva de servicios| website=Ferrocarriles Cono Sur| language=es}}</ref>
=== 特雷利烏大屠殺 ===
特雷利烏是1972年[[骯髒戰爭|骯髒戰爭]]的一個場地，當時，在這裡的聯邦監獄內約100名政治犯試圖越獄逃跑，導致一名警衛死亡，他們主要是對阿根廷軍政府不滿並支持被軍政府趕下台的阿根廷前總統{{link-en|阿爾圖羅·伊利亞|Arturo Umberto Illia}}。其中少數人搶奪了一架飛機並飛往智利，其餘未能登上飛機的人則被軍政府抓獲並送回軍事監獄，19人被行刑隊槍決，剩下3人則倖存活了下來。

之後，特雷利烏被軍方搜查，當地的政治犯被扣押並被帶到布宜諾斯艾利斯的Villa Devoto監獄。而特雷利烏市民進行了罷工，並成功釋放了Villa Devoto的囚犯。這些事件記錄在{{link-en|Tomás Eloy Martínez|Tomás Eloy Martínez}}的''La Pasión según Trelew''一書中，該書最初於1973年出版，但後來被{{Link-en |國家重組進程| National Reorganization Process}}獨裁統治壓制，並於1997年重新發行。<ref>{{cite news|title="La pasión según Trelew", de Tomás Eloy Martinez|url=http://entretenimiento.terra.com.ar/terra-libros/la-pasion-segun-trelew-de-tomas-eloy-martinez,b2ab90af847ad310VgnVCM3000009acceb0aRCRD.html|accessdate=2015-05-08|work=Terra Argentina|date=013-03-26|language=es}}</ref>

== 旅遊 ==
特雷利烏是馬塞塔中部地區的旅遊中心，每年遊客人數達到80,000人，該市的旅遊旺季從9月到3月，而南美州最大的企鵝保護區──{{link-en|通博角|Punta Tombo}}也可以由這座城市到達。<ref>{{cite web |title=Producción y Turismo |url=http://www.trelew.gov.ar/web/produccion-turismo.php |publisher=特雷利烏市政府 |archiveurl=https://web.archive.org/web/20111021004951/http://www.trelew.gov.ar/web/produccion-turismo.php |archivedate=2011-10-21 |language=es}}</ref><ref name="vistasdelvalle"/>

== 人口 ==
特雷利烏據1980年的人口普查有52372名居民，1991年人口普查顯示有79340居民，而2001年的人口普查共有89547居民（女性佔50.9％，男性佔49.1％），到了2010年的人口普查顯示已達到99430人，使得特雷利烏成為丘布特省僅次於[[里瓦達維亞海軍准將城|里瓦達維亞海軍准將城]]的第二大城市。<ref>{{cite web

== 體育 ==
{{link-en|特雷利烏競技隊|Racing de Trelew}}與特雷利烏颶風隊是當地最有名的兩個足球俱樂部，而帕托魯甦橄欖球俱樂部與特雷利烏橄欖球俱樂部則是當地最著名的橄欖球俱樂部。著名的拳擊手{{link-en|Lucas Matthysse|Lucas Matthysse}}和{{link-en|Walter Matthysse|Walter Matthysse}}也來自特雷利烏。
