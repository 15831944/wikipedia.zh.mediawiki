'''睇魚岩頂'''（'''{{lang-en|Tai Yue Ngam Teng}}'''）是[[香港|香港]]的一座[[山|山峰]]，位於[[新界|新界]]東部的[[西貢區|西貢區]]，是[[西貢東郊野公園|西貢東郊野公園]]的一部分，[[海拔|海拔]]233米，西接[[西灣山|西灣山]]，南至[[罾棚角|罾棚角]]。睇魚岩頂與[[蚺蛇尖|蚺蛇尖]] ( 位於[[新界|新界]][[西貢區|西貢區]]的[[西貢東郊野公園|西貢東郊野公園]]北部 ) 及[[釣魚翁|釣魚翁山]] ( 位於[[新界|新界]][[西貢區|西貢區]]的[[清水灣郊野公園|清水灣郊野公園]]東部 ) 合稱為[[西貢三尖|「西貢三尖」]]。<ref>{{cite web|title=蘋果日報 - 短線遊：登西貢三尖睇魚岩頂眺望罾棚角|url=http://hk.apple.nextmedia.com/supplement/travel/art/20040610/4100996}}</ref><ref>{{cite web|title=蘋果日報 - 西貢三尖之一﹕釣魚翁  風景優美|url=http://hk.apple.nextmedia.com/news/art/20150319/19081926}}</ref>

==四周的風景==
在睇魚岩頂上向[[西|西方]]觀望，可看到鄰近的[[西灣山|西灣山]]；向[[東|東方]]遠眺，可以見到一望無際的大[[海|海]] ( [[南海|南海]] ) ；向[[南|南方]]觀看，可以看到[[罾棚角|罾棚角]]和[[浪茄|浪茄]]；向[[北|北方]]遠望，可見到[[大浪灣_(西貢)|大浪灣]]，以及[[東灣山|東灣山]]和被稱為「[[香港三尖|香港三尖]]之首」及「[[香港|香港]]第一險峰」的[[蚺蛇尖|蚺蛇尖]]。
