'''范綱祥'''（{{bd|1966年|4月17日||}}），臺灣桃園市人，[[國立臺灣大學|國立臺灣大學]]土木工程系學士、[[國立臺灣大學|國立臺灣大學]]環境工程研究所碩士。有律師資格及環工技師證照，曾擔任執業律師及環工技師。

2014年12月25日[[桃園市|桃園市]]升格[[六都|六都]]，范綱祥當選為第一屆「桃園市市議員」。

== 求學 ==
畢業於師大附中、台灣大學土木系，就學時期參與觀音鄉大潭村鎘米污染事件調查，開始關注環境問題。1988年考取台大環工所，1990年取得碩士文憑。

== 經歷 ==
退伍後通過[http://wwwc.moex.gov.tw/main/ExamQual/wfrmExamQualMap.aspx?kind=2&menu_id=1490&examqual_id=685 環境工程技師]專業高考， 1994年返鄉開設環工技師事務所。同時間也參與社會、政治運動，加入
[[民主進步黨|民主進步黨]]，並於1995年與陳王琨教授等人創立桃園縣環保協會（桃園環保聯盟）。先後為[[彭紹瑾|彭紹瑾]]（立法委員）及[[呂秀蓮|呂秀蓮]]（桃園縣長）助選，也兼任[[彭紹瑾|彭紹瑾]]立委桃園服務處主任（義工職）。同時間自修通過律師專業高考，開設法律事務所。

1998年，參加桃園縣縣議員選舉，以三百票之差高票落選。
2009年，接受民進黨團徵召，參選桃園市市長，最後以九千票之差落選。
2014年，桃園縣升格桃園直轄市。參加市議員選舉，當選為桃園市升格後第一屆桃園區市議員。

== 聘任職 ==
曾受聘擔任[[財團法人廣播電視事業發展基金|財團法人廣播電視事業發展基金]]監事、桃園縣政府法規審查委員、立委[[彭紹瑾|彭紹瑾]]服務處主任、桃園縣環保協會會長、桃園縣公害糾紛調解委員、[[清雲科技大學|清雲科技大學]]兼任講師、[[景文科技大學|景文科技大學]]兼任講師、[[台灣環境保護聯盟|台灣環境保護聯盟]]中央執行委員、桃園市公所社區改造委員、[[行政院公共工程委員會|行政院公共工程委員會]]第七屆技師懲戒委員會之技師工會代表委員。
