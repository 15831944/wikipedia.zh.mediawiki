'''范丞丞'''（{{lang-en|'''Adam Fan Chengcheng '''}}，{{bd|2000年|6月16日|catIdx=Fan Chengcheng}}），[[中國|中國]][[男歌手|男歌手]]，[[樂華娛樂|樂華娛樂]]旗下[[藝人|藝人]]。2018年1月，參加[[爱奇艺|愛奇藝]]偶像男團競演養成類真人秀《[[偶像練習生|偶像練習生]]》，以第三名作為九人男團[[NINE_PERCENT|NINE PERCENT]]的成員正式出道，2018年6月21日以[[樂華七子|樂華七子]]的成員再次出道，其粉絲名為「丞星」。個人應援色為橙金色。

== 個人经历 ==
* 2007年，范丞丞与父母随其姐姐[[范冰冰|范冰冰]]出席了电影节的活动，首度在媒体前曝光 。
* 2018年1月，以练习生的身份参加选秀节目《[[偶像练习生|偶像练习生]]》  ；同年2月，参加[[湖南卫视|湖南衛視]]《[[快乐大本营|快乐大本营]]》 。
* 同年4月6日，以總票數第三名作為[[NINE_PERCENT|NINE PERCENT]]組合成員正式出道。
* 同年6月21日，以[[樂華七子|樂華七子]]的成員再次出道 
* 同年11月22日，发布首张单曲《I'm here》
* 2019年2月，《青春的花路》常驻MC
* 2019年3月，《女兒們的男朋友》常駐MC

== 個人生活 ==
* 自小就读北京貴族學校私立匯佳中學，15岁时赴英留学，之后转到美国蒙特沃德學院。
*由于在美国看到艺人在台上唱歌跳舞的模样，于是也希望等在台上发光发热，然后赴韩训练，之后和队员们从韩国回来，参加《偶像练习生》最终以第三名出道。

== 人物爭議 ==
*2007年，范丞丞在范冰冰主演的电影首映礼上首度曝光，但是随后却遭网友猜测是范冰冰的私生子，因为范冰冰1981年出生，范丞丞2000年出生，两人年龄相差19岁。2018年9月25日，北京市[[海淀区|海淀区]]法院作出判决，认定范丞丞与范冰冰系姐弟关系<ref>{{Cite web|url=https://www.thepaper.cn/newsDetail_forward_2804249|title=范冰冰弟弟打赢名誉权官司，法院：“私生子”传言构成侵权_一号专案_澎湃新闻-The Paper|accessdate=2019-01-06|work=www.thepaper.cn}}</ref>。
