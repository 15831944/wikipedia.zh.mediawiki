'''范寬'''（{{bd|約950年||約1032年||catIdx=范}}），[[表字|字]]'''中立'''，（另說名'''中正'''，字'''仲立'''）生於[[五代|五代]]末，[[北宋|北宋]]著名[[画家|画家]]，京兆華原（今[[陕西|陕西]][[耀縣|耀縣]]）人。因为性情宽厚豁達，不拘成礼，嗜酒好求道，时人呼之为「宽」，遂以范寬自名<ref name="multiple1"></ref>。

范寬擅長[[山水畫|山水畫]]，初與[[關仝|關仝]]、[[李成|李成]]同為北方山水畫派三大主流，並列「北宋三大家」{{efn|name=cancel|1=「唯營丘李成、長安關仝、華原范寬，……三家鼎峙，百代標程。」<ref name="multiple1">{{cite book |author=北宋 郭若虛 |title=《圖畫見聞誌·卷一·論三家山水》}}</ref>}}{{efn|name=cance2|1=「天下皆稱寬善與山傳神，宜其與關、李並馳方駕也。」<ref name="multiple2">{{cite book |title=北宋《[[宣和畫譜|宣和畫譜]]·卷十一·山水二·范寬》}}</ref>}}，後又與[[董源|董源]]{{efn|name=cance6|1=也作「董元」。}}、李成並稱「宋三家」{{efn|name=cancel4|1=「宋畫家山水超絶唐世者，李成、董元、范寬三人而已，……故三家照耀古今，為百代師法。」<ref name="multiple6">{{cite book |author=元朝 湯垕 |title=《畫鑒·宋畫·范寬》}}</ref>}}；藝術成就使得范寬與同為華原人的[[柳公權|柳公權]]合稱「柳范」<ref name="multiple7">{{Cite news|url=http://nrch.culture.tw/twpedia.aspx?id=28651 |title=《臺灣大百科全書·范寬》 |publisher=[[中華民國文化部|中華民國文化部]] |date=2011-11-29}}</ref>。

2004年，[[美国|美国]]《[[生活_(杂志)|生活]]》杂志将范寬评为「[[生活杂志评选的千年百大人物|上一千年对人类最有影响的百大人物]]」第59位。

== 風格 ==
范寬早年師從[[荊浩|荊浩]]及李成，後來「外師造化，中得心源」而自成一家。

善與山傳神<ref name="multiple2"></ref>的范寬長年隱居[[終南山|終南山]]及[[太華山|太華山]]，<ref name="multiple2"></ref>著重寫生與用心領會，題材多取自家鄉陝西[[關中|關中]]一帶雄闊壯美的山嶽，採用全景式高遠構圖{{efn|name=cance7|1=部分西方學者稱之「巨碑式山水」或「巨碑風格山水畫」。}}。其畫作主山堂堂，顯顯如[[恆山|恆]][[泰山|岱]]，遠山多正面，折落有勢<ref name="multiple5">{{cite book |author=北宋 [[米芾|米芾]] |title=《畫史》}}</ref>；峰巒渾厚，勢壯雄強<ref name="multiple1"></ref>；溪山深虛，水若有聲<ref name="multiple5"></ref>，水際作突兀大石<ref name="multiple3">{{cite book |author=[[元朝|元朝]] 夏文彥 |title=《圖繪寶鑑·卷三·宋》}}</ref>；真石老樹挺生筆下<ref name="multiple4">{{cite book |author=北宋 劉道醇 |title=《聖朝名畫評·卷二·山水林木門第二·范寬》}}</ref>，石瀾煙林深<ref name="multiple9">{{cite book |author=北宋 [[沈括|沈括]] |title=《圖畫歌》}}</ref>，讓觀畫者有遠望不離座外<ref name="multiple4"></ref>之感。

范寬筆力老健，槍筆俱勻，人屋皆質<ref name="multiple1"></ref>，不取繁飾，寫山真骨<ref name="multiple4"></ref>；善用[[雨點皴|雨點皴]]{{efn|name=cance3|1=也作「點皴」、「芝麻皴」或「豆瓣皴」。}}和積墨法，山頂好作密林<ref name="multiple3"></ref>以致如行夜山<ref name="multiple1"></ref>般的沉鬱效果，襯托出山勢的險峻硬朗。[[米芾|米芾]]認為范寬用墨太多是一大特色，卻也落得土石不分<ref name="multiple5"></ref>。相較於李成畫作「如對面千里，秀氣可掬」，[[王詵|王詵]]形容李成與范寬的畫風正好一文一武，互為對比。

范寬也是一位畫[[雪|雪]]景聖手，故生平畫作泰半為獨門的雪景山水，闊景甚偉{{efn|name=cancel5|1=「范寬雪景五幅，闊景甚偉。」<ref name="multiple10">{{cite book |author=[[南宋|南宋]] [[周密|周密]] |title=《雲煙過眼錄·卷一·張受益謙號古齋所藏》}}</ref>}}，好畫冒雪出雲之勢，尤有氣骨<ref name="multiple4"></ref>，見之使人凜凜<ref name="multiple6"></ref>，被譽為「畫山畫骨更畫魂」{{efn|name=cancel8|1=「畫山畫骨更畫魂，范寬此中高出群。」<ref>{{cite book |author=北宋 馮時行 |title=《[[紹興_(南宋)|紹興]]六年十月六日同信可舜弼進道謁隱甫值渠》}}</ref>}}。

== 作品 ==
北宋晚期的《[[宣和畫譜|宣和畫譜]]》收錄有范寬畫作58件，書畫家[[米芾|米芾]]也在其《畫史》中記載稱見過真跡30件。

目前其流傳的作品僅有[[臺北|臺北]][[國立故宮博物院|國立故宮博物院]]收藏的〈[[谿山行旅圖|谿山行旅圖]]〉為現今藝術史學家所公認。<ref>{{cite book |author=林柏亭 主編 |title=《大觀—北宋書畫特展》 |location=臺北 |publisher=國立故宮博物院 |date=2006-12 |ISBN=957-562-504-8}}</ref>

== 影响 ==
范寬不僅是馳名當代的繪畫大師，畫作在[[汴京|汴京]]與[[洛陽|洛陽]]<ref name="multiple1"></ref>間獲得極高好評，自[[宋朝|宋朝]]开始，就是画家们学习和摹仿的对象，北宋後期的[[李唐|李唐]]好学范寬，其后又有[[南宋|南宋]][[马远|马远]]及[[夏圭|夏圭]]等人学习李唐，使得整个南宋时期的山水画几乎系出范寬一脈，此后的「[[元四家|元四家]]」、[[明朝|明朝]]的[[唐寅|唐寅]]、[[浙派|浙派]]的[[戴進|戴進]]、[[清朝|清朝]]的「金陵画派」，以至[[近代|近代]]的[[黄宾虹|黄宾虹]]等歷代大师，都受到范寬画风的影响。

== 評價 ==
*劉道醇：「宋有天下，為山水者，惟（范）中正與（李）成稱絕，至今無及之者。……范寬以山水知名，為天下所重。……自古無法，創意自我，功期造化亦列神品。……李成之筆近視如千里之遠，范寬之筆遠望不離坐外。皆所謂造乎神者也。」<ref name="multiple4"></ref>故與李成齊名，並稱「李范」。
*郭若虛：「唯營丘李成、長安關仝、華原范寬，智妙入神，才高出類，三家鼎峙，百代標程。」<ref name="multiple1"></ref>，是為「北宋三大家」，謂之「三家山水」。
*[[郭熙|郭熙]]：「今齊魯之士，惟摹營丘；關陝之士，惟摹范寬。」<ref name="multiple12">{{cite book |author=北宋 郭思 |title=《林泉高致集·山水訓》}}</ref>
*後世將同為北方山水畫派的荊浩、關仝、李成、范寬並稱「荊、關、李、范」。
*[[苏轼|苏轼]]虽然非常推崇范寬，却觉得其画「微有俗气」，与當時文人淡雅风格略有不符。
*《宣和畫譜》：「天下皆稱（范）寬善與山傳神，宜其與關（仝）、李（成）並馳方駕也。」<ref name="multiple2"></ref>
*米芾：「物象之幽雅，品固在李成上，本朝自無人出其右。」<ref name="multiple5"></ref>
*後世將李成、范寬、郭熙、米芾並列「北宋四大家」，世稱「李、郭、范、米」。
*[[赵孟頫|赵孟頫]]：「畫山，皆寫秦隴峻拔之勢。大圖闊幅，山勢逼人，真古今絕筆也。」
*湯垕：「宋畫家山水超絶唐世者，李成、董元、范寬三人而已，嘗評之董元得山之神氣，李成得體貌，范寬得骨法，故三家照耀古今，為百代師法。」<ref name="multiple6"></ref>是為「宋三家」。
*後世將同為華原人的范寬與[[楷書四大家|楷書四大家]]之一的[[唐朝|唐朝]][[書法家|書法家]]柳公權合稱「柳范」，以表彰二人藝術上的成就。<ref name="multiple7"></ref>
*[[董其昌|董其昌]]评价范寬的《谿山行旅圖》為「宋画第一」。
*[[王鐸_(天啟進士)|王鐸]]：「博大奇奧，氣骨玄邈，用荊、關、董、[[巨然|巨]]運之一機，而靈韻雄邁允為古今第一」。
*[[徐悲鴻|徐悲鴻]]：「中國所有之寶，故宮有二，吾所最傾倒者，則為范中立〈谿山行旅圖〉，大氣磅礡，沉雄高古，誠辟易萬人之作。此幅既系巨幀，而一山頭，幾占全幅面積三分之二，章法突兀，使人咋舌。全幅整寫，無一敗筆，北宋人治藝之精，真令人傾倒。」<ref name="multiple11">{{cite book |author=徐悲鴻 |title=《故宮所藏繪畫之寶》 |date=1935}}</ref>
