'''范长龙'''（{{bd|1947年|5月|}}），[[辽宁省|辽宁]][[丹东市|丹东]]人，[[中国人民解放军上将|中国人民解放军上将]]。曾任[[中國共產黨中央軍事委員會副主席|中國共產黨中央軍事委員會副主席]]，[[中華人民共和國中央軍事委員會副主席|中華人民共和國中央軍事委員會副主席]]，[[中国人民解放军上将|中国人民解放军上将]]军衔。[[中国共产党第十六届中央委员会|中国共产党第十六届中央委员会]]候补委员，第十七、十八届中央委员，第十八届中央政治局委员<ref name="zlk">{{Cite web |url = http://cpc.people.com.cn/gbzl/html/121100045.html |title = 范长龙 |author =  |date =  |website = 人民网 |language =  |accessdate = 2017-04-27 |quote =  }}</ref>。

== 生平 ==
=== 早年 ===
范长龙出生于[[安東縣_(奉天省)|安东县]]<ref>1965年更名为东沟县。</ref>[[孤山镇_(东港市)|孤山镇]]西关。在孤山镇读完小学、初中。

1963年考入[[丹东市第二中学|丹东市第二中学]]读高中。1968年10月在[[东港市|东沟县]]下乡[[插队|插队]]，参加工作。

1969年1月，范长龙以[[知识青年|知识青年]]身份在东沟县参加[[中国人民解放军|中国人民解放军]]，任[[中国人民解放军陆军第七十八集团军|陆军第十六军]]炮兵团一二二毫米[[加农炮|加农炮]]第二营第六连炮手，新兵训练结束后在[[双阳区|双阳县]][[劝农山镇|东风公社]]同心大队参加军宣队的“支农”任务。1969年9月加入[[中国共产党|中国共产党]]。1969年底被评为“五好战士”，升任第六连指挥排侦察班班长。

1971年10月6日，作为该团1969年入伍战士中首批“提干”的6人之一，担任陆军第十六军炮兵团一二二毫米加农炮营第三连指挥排排长。1972年2月调任团政治处宣传股见习宣传干事（排职）。1973年8月，转正为宣传干事，不久调任团政治处组织股组织干事。1974年，任该团一五二毫米[[榴弹炮|榴弹炮]]营一连政治指导员。其间，1975年2月至1975年12月在宣化炮兵学院学习。

1976年初，借调到陆军第十六军教导大队任战术教员。1976年5月16日从一连政治指导员岗位直接升任陆军第十六军炮兵团副团长。1979年1月，改任陆军第十六军炮兵团副团长兼参谋长。1979年2月随部队参加“南打北防”任务。其间，1980年9月至1982年8月在[[中国人民解放军军事学院|中国人民解放军军事学院]]完成班住校学习。

1982年8月，升任陆军第十六军炮兵团团长。1985年9月升任[[中国人民解放军陆军第七十八集团军|陆军第十六集团军]]步兵第四十八师参谋长。1990年，升任陆军第十六集团军步兵第四十六师师长<ref>{{cite web | language = zh-hans | publisher = 鴨綠江晚報 | title = 我和上将范长龙 | url = http://www.ddrb.cn/wb/Articles/2009-02-06/20090206170131.html | author = 赵洪富 | date = 2009-02-06 | accessdate = 2010-05-16 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1993年，任陆军第十六集团军参谋长。其间，1992年至1994年在[[中共中央党校|中共中央党校]]领导干部函授班经济管理专业学习，获得中共中央党校大学[[本科|本科]]学历。1995年3月，升任陆军第十六集团军军长、晋升少将军衔。

2000年，调任[[中国人民解放军沈阳军区|沈阳军区]]参谋长，2002年晋升中將軍銜。2003年12月，任[[中国人民解放军总参谋长助理|中国人民解放军总参谋长助理]]。2004年9月，转任[[中国人民解放军济南军区|濟南軍區]]司令員。2008年7月15日晋升[[中国人民解放军上将|上將]]軍銜<ref>{{cite web |language = zh-hans |website = 北方网 |title = 刘振起黄献中范长龙晋升上将 胡锦涛颁发命令状 |url = http://news.enorth.com.cn/system/2008/07/15/003551013.shtml |author = 马欣 |date = 2008-07-15 |accessdate = 2010-06-16 |deadurl = yes |archiveurl = https://web.archive.org/web/20080804224658/http://news.enorth.com.cn/system/2008/07/15/003551013.shtml |archivedate = 2008-08-04 }}</ref>。

=== 军委副主席 ===
2012年11月4日，[[中国共产党第十七届中央委员会|中共十七届七中全会]]增补范长龙为[[中國共產黨中央軍事委員會副主席|中國共產黨中央軍事委員會副主席]]<ref>{{cite news |title = 中国共产党十七届七中全会公报 |url = http://news.163.com/12/1104/18/8FG27L6V00014JB5.html |accessdate=2012-11-05 |website = 网易 }}</ref>。[[中国共产党第十八届中央委员会第一次全体会议|中共十八届一中全会]]继续任中共中央军事委员会副主席。2013年，[[第十二届全国人民代表大会|第十二届全国人民代表大会第一次会议]]任命为[[中華人民共和國中央軍事委員會副主席|中華人民共和國中央軍事委員會副主席]]。他是中共第十六届中央候补委员，十七届、十八届中央委员，十八届中央政治局委员。

2017年10月，屆齡退休，其[[中国共产党中央军事委员会副主席|中国共产党中央军事委员会副主席]]职务由[[中央軍委裝備發展部|中央軍委裝備發展部]]部長[[張又俠|張又俠]]接替。

2018年1月14日，[[香港|香港]]《[[星岛日报|星岛日报]]》率先报道称，[[房峰辉|房峰辉]]被移送军事检察机关后，中央决定拿下范长龙<ref>{{Cite web |url = http://std.stheadline.com/daily/news-content.php?id=1731428&target=2 |title = 中央追打「軍虎」 軍委副主席范長龍傳被查  |author =  |date = 2018-01-14 |publisher = 星島日報 |language = zh |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.zaobao.com/news/china/story20180115-826983 |title = 第三名落马中共军委副主席？ 传范长龙被立案审查 |author =  |date = 2018-01-15 |website = 联合早报网 |language = zh |accessdate =  |quote =  }}</ref>。但1月18日《[[解放军报|解放军报]]》第二版刊登的报道《用练兵热潮作答胜战之问》写道：“2016年6月26日解放军报报道，全军实战化军事训练座谈会在京召开，中央军委副主席范长龙、[[许其亮|许其亮]]出席并讲话。”<ref>{{cite news |author=李宣良 |title=用练兵热潮作答胜战之问 |url = http://www.81.cn/jfjbmap/content/2018-01/18/content_197236.htm |accessdate=2018-01-18 |work=解放军报 |date=2018-01-18 }}</ref>1月25日，国防部新闻发言人[[吴谦|吴谦]]在例行记者会上称范长龙被调查的报道是“谣言”，他还建议提问记者去看看《解放军报》1月18日刊登的《用练兵热潮作答胜战之问》一文<ref>{{Cite web |url = https://in.reuters.com/article/china-defence-corruption/china-denies-former-senior-officer-being-investigated-idINL4N1PK3MF  |title = China denies former senior officer being investigated |author =  |date = 2018-01-25 |publisher = Reuters |language = en |accessdate =  |quote = }}</ref>。2月2日中共中央军委新春文艺晚会上，范长龙到场观看并被新华社点名报道，“用身体语言辟谣”<ref>{{cite web|title=范長龍央視露臉 為落馬闢謠|url=http://www.chinatimes.com/newspapers/20180204000086-260302|website=中时电子报|accessdate=2018-02-04|date=2018-02-02}}</ref>。

2018年3月18日，十三届全国人大一次会议第六次全体会议，选举中华人民共和国中央军事委员会副主席，范长龙不再担任。<ref>{{Cite web|url=http://www.xinhuanet.com/politics/2018lh/2018-03/18/c_1122553342.htm|title=快讯：大会经投票表决，决定许其亮、张又侠为中央军委副主席-新华网|accessdate=2018-11-23|work=www.xinhuanet.com}}</ref>
