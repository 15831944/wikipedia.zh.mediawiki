{{About|數學中的範疇理論|範疇論中的範疇概念|範疇 (数学)|範疇的其他意思|範疇}}

'''範疇論'''是[[數學|數學]]的一門學科，以抽象的方法來處理數學概念，將這些概念形式化成一組組的「物件」及「[[態射|態射]]」。數學中許多重要的領域可以形式化成範疇，並且使用範疇論，令在這些領域中許多難理解、難捉摸的數學結論可以比沒有使用範疇還會更容易敘述及證明。

範疇最容易理解的一個例子為[[集合範疇|集合範疇]]，其物件為集合，態射為集合間的函數。但需注意，範疇的物件不一定要是集合，態射也不一定要是函數；一個數學概念若可以找到一種方法，以符合物件及態射的定義，則可形成一個有效的範疇，且所有在範疇論中導出的結論都可應用在這個數學概念之上。

範疇最簡單的例子之一為[[广群|广群]]，其態射皆為可逆的。群胚的概念在[[拓撲學|拓撲學]]中很重要。[[範疇_(數學)|範疇]]現在在大部分的數學分支中都有出現，在[[理論電腦科學|理論電腦科學]]的某些領域中用于對應[[資料型別|資料型別]]，而在[[數學物理|數學物理]]中被用來描述[[向量空間|向量空間]]。

範疇論不只是對研究範疇論的人有意義，對其他數學家而言也有著其他的意思。一個可追溯至1940年代的述語「[[抽象廢話|一般化的抽象廢話]]」，即被用來指範疇論那相對於其他傳統的數學分支更高階的抽象化。

== 背景 ==

研究[[範疇_(數學)|範疇]]就是試圖以「公理化」的方法抓住在各種相關連的「數學結構」中的共同特性，並以結構間的「結構保持函數」將這些結構相關起來。因此，對範疇論系統化的研究將允許任何一個此類數學結構的普遍結論由範疇的公理中證出。

考慮下面的例子：由[[群|群]]組成的[[類_(數學)|類]]'''Grp''' 包含了所有具有「群結構」的物件。要[[證明|證明]]有關群的[[定理|定理]]，即可由此套公理進行邏輯的推導。例如，由公理中可立即證明出，群的[[單位元素|單位元素]]是唯一的。

不是只專注在有特定結構的個別物件（如群）上，範疇論會著重在這些物件的[[態射|態射]]（結構保持映射）上；經由研究這些態射，可以學到更多關於這些物件的結構。以群為例，其態射為[[群同態|群同態]]。兩個群間的群同態會嚴格地「保持群的結構」，這是個以將一個群中有關結構的訊息運到另一個群的方法，使這個群可以看做是另一個群的「過程」。因此，對群同態的研究提供了一個得以研究群的普遍特性及群公理的推論的工具。

類似的研究也出現在其他許多的數學理論中，如在[[拓撲學|拓撲學]]中對[[拓撲空間|拓撲空間]]的[[連續函數_(拓撲學)|連續]]映射的研究（相關範疇稱為'''Top'''），及對[[流形|流形]]的[[光滑函數|光滑函數]]的研究等。

=== 函子 ===
{{Main|函子}}

再抽象化一次，範疇自身亦為數學結構的一種，因此可以尋找在某一意義下會保持其結構的「過程」；此一過程即稱之為[[函子|函子]]。函子將一個範疇的每個物件和另一個範疇的物件相關連起來，並將第一個範疇的每個態射和第二個範疇的態射相關連起來。

實際上，即是定義了一個「範疇和函子」的範疇，其元件為範疇，（範疇間的）態射為函子。

經由研究範疇和函子，不只是學習了一類數學結構，及在其之間的態射；還學習了「在不同類型的數學結構之間的關係」。此一基本概念首次出現於[[代數拓撲|代數拓撲]]之中。不同的「拓撲」問題可以轉換至通常較易解答的「代數」問題之上。在[[拓撲空間|拓撲空間]]上如[[基本群|基本群]]或基本群胚等基本的架構，可以表示成由[[群胚|群胚]]所組成的範疇之間的基本函子，而這個概念在代數及其應用之中是很普遍的。

=== 自然變換 ===
{{Main|自然變換}}

再抽象化一次，架構通常會「自然地相關連」，這個第一眼會覺得很曖昧的概念，產生了[[自然變換|自然變換]]（將一個函子映射至另一函子的方法）此一清楚的概念。許多數學上的重要架構可以從此一角度來研究。

== 历史注记 ==

范畴、函子和自然变换是由[[塞缪尔·艾伦伯格|塞缪尔·艾伦伯格]]和[[桑德斯·麦克兰恩|桑德斯·麦克兰恩]]在1945年引进的。这些概念最初出现在[[拓扑学|拓扑学]]，尤其是[[代数拓扑学|代数拓扑学]]里，在[[同态|同态]]（具有几何直观）转化成[[同调论|同调论]]（[[公理化|公理化]]方法）的过程中起了重要作用。[[乌拉姆|乌拉姆]]说，在1930年代的后期，波兰学派中曾出现类似的想法。

艾伦堡和麦克兰说，他们的目的在于理解自然映射；为此，必须定义函子；为了定义函子，就自然地要引进范畴。

[[同调代数|同调代数]]由于计算上的需要而使用范畴论，这对范畴论起到了推进作用；此后范畴论又在[[代数几何|代数几何]]的公理化过程中得到发展。代数几何与[[罗素-怀特海德|罗素-怀特海德]]的关于数学统一性基础的观点相抵触。广义范畴论随后产生，且更容纳了语意灵活性和[[高阶逻辑|高阶逻辑]]等多种新特征的[[泛代数|泛代数]]，现在被运用到数学的所有分支。

特殊范畴[[拓扑斯|拓扑斯]]甚至可以代替[[公理集合论|公理集合论]]作为数学的基础。然而范畴论对这些范围广泛的基础应用还是有争议的；但作为[[构造性数学|构造性数学]]的基础或注释，范畴论被研究的相当透彻。尽管如此，公理集合论至今仍然是数学家们的通用语言，并没有被范畴论的注释所取代。将范畴论引入大学程度的教学（在《伯克霍夫-麦克兰》和《麦克兰-伯克霍夫》这两本[[抽象代数|抽象代数]]的教科书的区别上可以印证）还是遭到了相当的反对。

[[范畴逻辑|范畴逻辑]]是[[直觉逻辑|直觉逻辑]]中[[直觉类型论|类型论]]的一个被明确定义的分支，在计算机学科的[[函数式编程|函数式编程]]和[[域理论|域理论]]中均有应用，并且都是在[[笛卡尔闭范畴|笛卡尔闭范畴]]中对[[λ演算|λ演算]]的非句法性描述。至少，用范畴论可以精确地描述在这些相关的领域里什么是共同的（在[[抽象化_(數學)|抽象]]的意义上）。

== 范畴 ==
{{Main|範疇 (數學)|態射}}

一个“范畴” <math>C </math> 是由如下3個數學實體所組成的：
#一個[[類_(數學)|類]] <math>ob(C) </math>，其元素稱為「物件」；
#一個類 <math>hom(C) </math>，其元素稱為「[[態射|態射]]」或「箭號」。每個態射 <math>f </math> 都只有一個「源物件」 <math>a </math> 及一個「目標物件」 <math>b </math>（其中 <math>a </math> 和 <math>b </math> 都在 <math>ob(C) </math> 內），稱之為「從 <math>a </math> 至 <math>b </math> 的態射」，標記為 <math>f : a \to b </math>。<br />所有從 <math>a </math> 至 <math>b </math> 的態射所組成的類稱之為「態射類」，標記為 <math>hom(a, b) </math>、 <math>hom_C(a, b) </math>或 <math>mor(a, b) </math>。
#一個[[二元運算|二元運算]]，稱為「態射複合」，使得對任意三個物件 <math>a </math>、 <math>b </math> 及 <math>c </math>，都會有 <math>\circ : hom(b, c) \times hom(a, b) \to hom(a, c) </math>。兩個態射 <math>f : a \to b </math> 及 <math>g : b \to c </math> 的複合寫做 <math>g \circ f </math> 或 <math>gf </math><ref>有些作者會以不同的次序做複合，將g ∘ f 寫做fg 或f ∘ g。研究電腦科學的學者在使用範疇論時經常將g ∘ f寫做f ; g</ref>，並會符合下列兩個公理：
#*[[結合律|結合律]]：若 <math>f : a \to b </math>、 <math>g : b \to c </math>及 <math>h : c \to d </math>，則 <math>h \circ  (g \circ f) = (h \circ g) \circ f </math>；
#*[[單位元|單位元]]：對任意物件 <math>x </math>，總存在一個態射 <math>1_x : x \to x </math>（稱為 <math>x </math> 的[[單位態射|單位態射]]），使得對每個態射 <math>f : a \to b </math>，都會有 <math>1_b \circ f = f = f \circ 1_a </math>。

由以上公理可證得，每個物件都只存在一個單位态射。有些作者将物件本身用單位态射来定义，这在本质上是相同的。

如果对象的类确实是个集合，那么这种范畴就被称为'''“小范畴”'''。许多重要的范畴不是小范畴。

范畴中的态射有时又称为'''“箭號”'''，这种叫法来自于[[交换图|交换图]]。

=== 范畴举例 ===

每一范畴都由其对象，态射，和复合态射来表述。为了方便起见，以下的“函数”即是指态射，不再一一说明。

* [[集合范畴|'''Set''']] 是所有[[集合论|集合]]和它们彼此之间的[[全函数_(数学)|全函数]]构成的范畴
* [[预序范畴|'''Ord''']] 是所有[[预序|预序集]]和其间的[[单调函数|单调函数]]构成的范畴
* [[广群范畴|'''Mag''']] 是所有[[广群（代数）|广群]]和其间的[[同态映射|同态映射]]构成的范畴
* [[对换范畴|'''Med''']] 是所有[[对换|对换广群]]和其间的[[同态映射|同态映射]]构成的范畴
* [[群范畴|'''Grp''']] 是所有[[群|群]]和其间的[[群同态|群同态]]构成的范畴
* [[阿贝尔群范畴|'''Ab''']] 是所有[[阿贝尔群|阿贝尔群]]和其间的[[群同态|群同态]]构成的范畴
* [[K-向量|'''Vect'''<sub>''K''</sub>]] 是所有[[域_(數學)|体]] <math>K</math>（<math>K</math>固定）上的[[向量空间|向量空间]]和其间的<math>K-</math>[[线性映射|线性映射]]构成的范畴
* [[拓扑空间范畴|'''Top''']] 是所有[[拓扑空间|拓扑空间]]和其间的[[连续|连续]]函数构成的范畴
* [[度量空间范畴|'''Met''']] 是所有[[度量空间|度量空间]]和其间的[[测地映射|测地映射]]构成的范畴
* [[一致空间范畴|'''Uni''']] 是所有[[一致空间|一致空间]]和其间的[[一致连续|一致连续函数]]构成的范畴
* 任何[[偏序集|偏序集]]<math>(P,\leq)</math>构成一个小范畴，其对象是<math>P</math>的元素，其态射是从<math>x</math>指向<math>y</math>的箭头，其中<math>x\leq y</math>。
* 任何以单一对象<math>x</math>（''<math>x</math>''为任意固定集合）为基础的[[幺半群|独异点]]构成一个小范畴。独异点的任意元素通过二元运算给出一个从<math>x</math>到<math>x</math>的映射，所有这些映射恰好是范畴的所有态射；范畴的复合态射也正好是独异点的二元运算。事实上，范畴可以看成独异点的推广；关于独异点的定义和定理有一些可以推广到范畴。
* 任何[[有向图|有向图]]对应于一个小范畴：其对象是图的[[顶点|顶点]]，其态射是图的路径，其复合态射是路径的连接。称此范畴为有向图的“自由范畴”。
* 设<math>I</math>是个[[集合_(數學)|集合]]，“<math>I</math>上的[[离散范畴|离散范畴]]”是一个小范畴，以<math>I</math>的元素为对象，以<math>I</math>的恒等映射为其唯一的态射。
* 任何范畴<math>C</math>可以在另一种看法下成为一个新的范畴：它具有相同的对象，然而所有态射都是反方向的。称此为[[对偶_(范畴论)|“对偶”或者“反范畴”]]，记作 <math>C^{op}</math>（<math>op</math>来自英文的 opposite）。
* 设<math>C</math>和<math>D</math>是范畴，则它们的“直积范畴”<math>C\times D</math> 被定义为：其对象为取自<math>C</math>的一个对象和取自<math>D</math>的一个对象的有序对，其态射亦为取自<math>C</math>的一个态射和取自<math>D</math>的一个态射的有序对，其复合态射则由其分量分别复合。

=== 态射分类 ===

[[态射|态射]] <math>f:A\rightarrow B</math>称为
* [[单态射|单态射]]，如果 <math>fg_1=fg_2</math>，则有<math>g_1=g_2</math>，此关系对所有态射<math>g_1,g_2:X\rightarrow A</math>成立。
* [[满态射|满态射]]，如果<math>g_1f=g_2f</math>， 则有<math>g_1=g_2</math>，此关系对所有态射<math>g_1,g_2:B\rightarrow X</math>成立。
* [[同构|同构]]，如果存在逆态射<math>G:B\rightarrow A</math>使得<math>fg=\mathrm{id}_B</math>并且<math>gf=\mathrm{id}_A</math>。
* [[自同构|自同构]]，如果<math>f</math>是同构态射，并且有<math>A=B</math>。
* [[自同态|自同态]]，如果<math>A=B</math>。

映射之间的关系（比如<math>fg=h</math>）在大多数情形下可用更直观的[[交换图|交换图]]来表示，在此图中对象被表示成顶点，态射被表示为箭头。

== 函子 ==

函子是范畴之间保持结构的映射。它们可以被看成以所有（小）范畴为成员的范畴中的态射。

一个从范畴<math>C</math>到范畴<math>D</math>的（[[协变与反变|协变]]）函子<math>F</math>被定义为：

* 对<math>C</math>中任意对象<math>X</math>，都有一个<math>D</math>中相应的对象<math>F(X)</math>与其对应；
* 对<math>C</math>中任意态射 <math>f:X\rightarrow Y</math>，都有一个<math>D</math>中相应的态射<math>F(f):F(X)\rightarrow F(Y)</math>与其对应；

并使下列性质成立：

* 对<math>C</math>中任意的对象<math>X</math>，都有<math>F(\mathrm{id}_x)=\mathrm{id}_{F(X)}</math>。
* 对<math>C</math>中任意两个态射<math>f:X\rightarrow Y</math>和<math>g:Y\rightarrow Z</math>，都有 <math>F(g\cdot f)=F(g)\cdot F(f)</math>。

一个从范畴<math>C</math>到范畴<math>D</math>的[[协变与反变|反变]]函子<math>F</math>不同于函子的地方仅在于将<math>D</math>中的映射箭头倒过来。比如说<math>f:X\rightarrow Y</math>是<math>C</math>中任一态射，则有<math>F(f):F(Y)\rightarrow F(X)</math>。定义反变函子的最简捷的方法是作为<math>C</math>的反范畴<math>C^{op}</math>到<math>D</math>上的函子。

有关函子的具体例子和性质请详见[[函子|函子]]条目。

== 自然和自然同构 ==

详细请见[[自然变换|自然变换]]条目。

一个“自然变换”是两个函子之间的一个关系。函子通常用来描述“自然构造”，而自然变换则用来描述两个构造之间的“自然同态”。有时候，两个截然不同的构造具有“相同的”结果；这正可以用两个函子之间的自然关系来表述。

=== 定义 ===

如果<math>F</math>和<math>G</math>是从范畴<math>C</math>到范畴<math>D</math>的（协变）函子，则从<math>F</math>到<math>G</math>的一个自然变换对于<math>C</math>中的任何对象<math>X</math>，都有一个<math>D</math>中相应的态射<math>\eta_X:F(X)\rightarrow G(X)</math>，使得对<math>C</math>中的任何态射<math>f:X\rightarrow Y</math>，都有<math>\eta_Y\cdot F(f)=G(f)\cdot \eta_X</math>；这也就是说下列图表是[[交换图|可交换的]]：

:[[File:NaturalTransformation-01.png|Diagram defining natural transformations]]

两个函子<math>F</math>和<math>G</math>称为“自然同构”，如果存在一从<math>F</math>到<math>G</math>的自然变换，使得对所有<math>C</math>中的对象<math>X</math>，<math>\eta_X</math>是一个同构。

=== 举例 ===

设<math>K</math>是[[体_(数学)|体]]，<math>V</math>是<math>K</math>上的任意向量空间，则有从向量空间到其[[对偶空间|二重对偶]]的一个“自然”[[内射|内射]]型[[线性映射|线性映射]]<math>V\rightarrow V^{**}</math> 。这些映射在以下意义上是“自然”的：二重对偶运算是一个函子，这些映射正好构成了从恒等函子到二重对偶函子的自然变换。如果向量空间的维数是有限的，我们就得到一个自然同构；因为“有限向量空间自然同构于其二重对偶”。

考虑阿贝尔群及其同态构成的范畴<math>\mathrm{Ab}</math>。对任意阿贝尔群<math>X</math>、<math>Y</math>和<math>Z</math>，我们得到群同构

:<math>\mathrm{Mor}\left ( X,\mathrm{Mor}\left ( Y,Z \right ) \right )\rightarrow \mathrm{Mor}\left ( X\otimes Y,Z \right )</math> 。

这些同构是“自然”的，因为它们定义了两个函子间的一种自然变换：<math>\mathrm{Ab}^{op}\times\mathrm{Ab}^{op}\times\mathrm{Ab}\rightarrow\mathrm{Ab}</math>。

== 泛结构、极限和上极限 ==

{{main|泛性质|极限 (范畴论)}}

运用范畴论的语言，许多数学研究领域都可以归结成一些恰当的范畴，例如所有集合的范畴，所有群的范畴，所有拓扑的范畴，等等。这些范畴里的确有一些“特殊的”对象，例如[[空集|空集]]或者[[拓扑直积|两个拓扑的直积]]。然而，在范畴的定义里，对象是原子性的，那就是说，我们无法知道一个对象到底是集合，是拓扑，还是其它抽象概念。有必要定义特殊对象而不涉及对象的内在结构，这是一个挑战。那么到底怎样不用元素而定义空集，不用开集而定义拓扑积呢？

解决这个问题的途径是借用对象和对象之间的关系，而这些关系由相应范畴中的态射给出。现在问题转化为寻找[[泛性质|泛性质]]，这些泛性质可以唯一地决定我们所感兴趣的对象。事实上，为数众多的重要结构都可用纯范畴论的方法来描述。在定义泛性质时，我们要用到一个非常关键的概念：[[范畴性|范畴性]]“极限”和其“上极限”。

== 等价范畴 ==

{{main|范畴的等价|范畴同构}}

人们很自然地要问，在什么样的情形下，两个范畴“在本质上是相同”的，换一句话来说，对其中一个范畴成立的定理，可以既定地转换成另一个范畴的定理。用来描述这种情形的主要方法是“范畴的等价性”，由函子给出。范畴的等价性在数学中有很多的应用。

== 进一步的概念和结果 ==

范畴和函子的定义只是范畴代数中最基本的部分。除此之外的重要部分如下列所述。基本上是以阅读顺序排列，尽管它们彼此之间有着内在的联系。

* [[函子范畴|函子范畴]] <math>D^C</math>以从<math>C</math>到<math>D</math>的函子为对象，以这些函子间的自然映射为泛射。[[米田引理|米田引理]]刻划了函子范畴中可表示的函子，是范畴论最著名的基本结果之一。
* [[对偶_(范畴论)|对偶原则]]：范畴论中，每一陈述，定理，或定义都有其“对偶”，实质上可以通过“反转所有箭头”来得到。如果一个陈述在范畴<math>C</math>中成立，那么它的对偶将在其对偶范畴<math>C^{op}</math>中成立。这一对偶性在范畴论的任何层次都是普适的，由于它经常不是很清晰，对偶性的应用可以揭示惊人的关联性。
* [[伴随函子|伴随函子]]：两个映射方向相反的函子对称为伴随函子，随着结合的顺序不同，分别为左伴随和右伴随。通常来自于由泛性质所定义的结构；也可以作为泛性质的一种更加抽象和更加强有力的看法。

== 范畴分类 ==

* 在许多范畴中，态射集合<math>\mathrm{Mor}(A,B)</math>不仅仅是集合，实际上是[[阿贝尔群|阿贝尔群]]，态射的复合具有群结构，也就是说是[[双线性|双线性]]的。这种范畴被称为[[预加性范畴|预加性的]]。如果这种范畴还具有所有有限的[[积与上积|积和上积]]，则称为[[加性范畴|加性范畴]]。如果所有具有一个[[核_(范畴论)|核]]和一个[[上核|上核]]，那么所有满射都是上核，所有单射都是核，我们称此为[[阿贝尔范畴|阿贝尔范畴]]。阿贝尔范畴的一个典型的例子是阿贝尔群所组成的范畴。
* 一个范畴被称为是[[极限_(范畴论)|完备]]的，如果所有[[极限_(范畴论)|极限]]存在。集合，阿贝尔群和拓扑空间的范畴是完备的。
* 一个范畴被称为是[[笛卡儿闭范畴|笛卡儿闭性]]的，如果它具有有限直积，并且一个定义在有限乘积上的态射总是可以表示成定义在其中一个因子上的态射。
* 一个[[拓扑斯|拓扑斯]]是一种特殊的笛卡儿闭范畴，在其中可表述（公理化）所有的数学结构（就象传统上使用集合论可以表示所有数学结构）。一个拓扑斯也可以用来表述一个逻辑理论。
* 一个[[群胚|群胚]]是这样一种范畴，其中每一个映射都是一个同构。群胚是群、[[群作用|群作用]]和[[等价关系|等价关系]]的推广。
