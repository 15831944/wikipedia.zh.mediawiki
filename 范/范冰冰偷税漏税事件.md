'''范冰冰偷税漏税事件'''，或称'''范冰冰偷税漏税案'''、'''范冰冰阴阳合同案'''，是指2018年中国著名女演员[[范冰冰|范冰冰]]及其担任法定代表人的企业因[[逃稅|偷逃税款]]遭调查及处罚的事件。

== 事发背景 ==
主持人[[崔永元|崔永元]]与导演[[冯小刚|冯小刚]]因冯于2003年执导的电影《[[手机_(电影)|手机]]》中情节映射的问题产生矛盾。据崔永元称，编剧[[刘震云|刘震云]]曾因此三次向其道歉<ref>{{cite web |author1=崔永元 |title=刘震云的家训 |url=https://mp.weixin.qq.com/s/guKglqf_ydgw0B0KL07ZcQ |website=骚客文艺 |date=2018-05-31}}</ref>。2018年5月10日晚，冯小刚在其微博公布《[[手机2|手机2]]》概念海报<ref>{{cite web |author1=苏姗 |title=冯小刚《手机2》开拍 葛优范冰冰等原班人马回归 |url=http://www.xinhuanet.com/ent/2018-05/11/c_1122815644.htm |website=新华网 |date=2018-05-11}}</ref>。5月11日，崔永元在其微博公布其4月7日与编剧[[刘震云|刘震云]]的短信记录，短信中崔表示不希望刘使用“手机2”命名，以避免引起不必要的联想和议论，刘称新作品将命名为“朋友圈”，而冯小刚公布的海报中却仍旧使用了“手机2”的命名，崔在这条微博评论称“冯小刚是渣子大家都知道，刘震云变成渣子速度偏快了一些<ref>{{Cite web|url=https://news.dahe.cn/2018/06-01/318520.html|title=冯小刚宣布《手机2》开拍 崔永元得知后大骂"渣子"|accessdate=2018-10-04|work=大河网}}</ref>。”

5月25日，崔永元在其微博晒出一张演艺合同的截图，约定片酬为税后1000万元，崔评论称“一个真敢要，一个真敢给”<ref>{{Cite web|url=http://life.chinanews.com/yl/2018/06-01/8528198.shtml|title=崔永元接受采访重提当年与冯小刚的恩怨纠葛|accessdate=2018-10-04|last=chinanews|work=中新网}}</ref>。5月28日，崔永元晒出多张演艺合同照片，为5月25日截图的较完整版，虽然部分内容进行了模糊处理，但可看到合同涉及演员为[[范冰冰|范冰冰]]。5月29日，崔永元又曝光称一个演员（“这哥们儿”）演一出戏，只在片场演了4天，却签一小一大双合同{{efn|所谓「大小合同」「阴阳合同」是指交易双方签订金额不同的两份合同，一份金额较小的「阳合同」用于向主管机关备案登记纳税；另一份金额较高的「阴合同」则实际约定双方交易价格，彼此对其秘而不宣，目的就是逃避纳税这一法定义务<ref>{{cite web |author1=蓝孝威 |title=崔永元点火爆料 曾受死亡威胁 |url=http://www.chinatimes.com/cn/newspapers/20181004000553-260112 |website=中时电子报 |date=2018-10-04}}</ref>。}}，小的片酬1千万，大的5千万，共拿走6千万元人民币。这些微博引起网上的广泛讨论，所涉及的合同则被外界猜测是电影《[[大轰炸|大轰炸]]》的合同，对此崔永元于5月30日接受新浪娱乐采访时并未肯定合同所涉及作品是《大轰炸》，更表示自己有一抽屉类似的合同。对于为何公布范冰冰的合同，崔表示是最初公布合同图片时因为图片模糊处理的操作不熟练，漏掉范冰冰名字的处理了，开始时是有点抱歉的，但在范冰冰在其微博称演武月很快乐后就感觉愤怒了<ref>{{cite web |author1=苏三 |title=崔永元详解和《手机2》仇怨 又为何针对范冰冰 |url=http://ent.sina.com.cn/m/c/2018-05-30/doc-ihcffhsv9352805.shtml |website=新浪娱乐 |date=2018-05-30 |access-date=2018-10-04 |archive-url=https://web.archive.org/web/20180612140037/http://ent.sina.com.cn/m/c/2018-05-30/doc-ihcffhsv9352805.shtml |archive-date=2018-06-12 |dead-url=no }}</ref>。

5月29日，崔永元曝光阴阳合同后，范冰冰工作室进行否认，并称崔永元的行为，既破坏了商业原则，又涉嫌侵犯合法权益，并称相关网络用户及媒体未经核实散布谣言的行为已经构成[[诽谤|诽谤]]<ref>{{Cite web|url=http://www.mnw.cn/news/ent/mx/2005011.html|title=范冰冰工作室声明反击崔永元：破坏商业规则 涉嫌诽谤|accessdate=2018-10-04|work=闽南网|language=zh-CN}}</ref>。6月3日，[[国家税务总局|国家税务总局]]公开表示，针对网上有关影视从业人员签订“阴阳合同”中的涉税问题，已责成[[江苏|江苏]]等地税务机关依法开展调查核实<ref>{{Cite web|url=http://news.sina.com.cn/o/2018-10-03/doc-ifxeuwws0593751.shtml|title=税务部门依法查处范冰冰“阴阳合同”等偷逃税问题|accessdate=2018-10-04|work=新浪网}}</ref>。与此同时，范冰冰工作室所在地江苏[[无锡|无锡]]的无锡市[[滨湖区|滨湖区]]地税局称已经介入此事的调查取证。随后，崔永元表示，愿意配合税务部门展开调查，愿主动将电话提供给无锡相关部门<ref>{{cite web |author1=TF0328 |title=崔永元致歉范冰冰 国税调查阴阳合同 税务盯“大腕儿”应成常态？ |url=http://www.takefoto.cn/viewnews-1486138.html |website=北晚新视觉 |date=2018-06-04}}</ref>。

== 调查及处罚 ==
6月1日，国务院副总理[[韩正|韩正]]出席国税地税征管体制改革座谈会，表示要贯彻[[习近平|习近平]]总书记的重要思想和落实[[李克强|李克强]]总理的批示要求，对国税地税征管体制改革进行动员部署。<ref>{{Cite news|url=http://www.chinatax.gov.cn/n810219/n810724/c3489796/content.html|title=韩正出席国税地税征管体制改革座谈会并讲话|work=国家税务总局|date=2018-06-02}}</ref>

6月3日，国家税务总局下令彻查明星“大小合同”。无锡市滨湖区税务局{{efn|name=ab1|相关报道皆称“地税局”。在2018年3月，[[国税地税机构合并|国税地税机构合并]]后，“地税局”已改为“税务局”。}}随后宣布介入调查取证<ref>{{cite web |title=崔永元抗议手机2事件最新进展：不想伤害范冰冰徐帆 |url=http://www.qlmoney.com/content/20180604-323164.html |accessdate=2018-10-12|author= |date=2018-06-04 |website=齐鲁财富网|language=Zh-hans}}</ref>。其上级单位──江苏省税务局{{efn|name=ab1}}、无锡市税务局{{efn|name=ab1}}亦在同日先后回应，将对相关问题进行核查。6月4日，江苏省税务局负责人表示：已经按照国家税务总局的要求，责成无锡市税务局对相关公司进行核查，因为核查需要时间，所以目前并没有可以公布的信息；同时会联合北京有关部门联系[[崔永元|崔永元]]，就公布的材料进行沟通<ref>{{Cite web |url=http://js.ifeng.com/a/20180605/6629579_0.shtml |title=范冰冰3家无锡公司难找到 江苏地税局：会联系崔永元|accessdate=2018-10-09 |author= |date=2018-08-15 |work=责任编辑：胥大伟 |publisher=凤凰网，来源：每日经济新闻|language=Zh-hans }}</ref>。此后，范冰冰再未公开露面。关于她的各类传闻，甚嚣尘上，成为公众舆论关注的焦点<ref name="a1">{{Cite web |url=http://www.sohu.com/a/257489650_821124 |title=范冰冰“消失”百余天后：代言被瓜分、电影被撤档、海报被除名？|accessdate=2018-10-09|author= |date=2018-10-03 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。其中，[[罗昌平|罗昌平]]于7月26日在新浪微博曝露的案情日后获得了证实<ref name="b1">{{Cite web |url=http://ent.sina.com.cn/m/c/2018-08-15/doc-ihhtfwqr8193791.shtml |title=70多天未发声，范冰冰跑哪儿去了？|accessdate=2018-10-12 |author=记者：丁舟洋、白芸、毕媛媛，每经编辑：温梦华 |date=2018-08-15 |work= |publisher=新浪网，来源：每日经济新闻|language=Zh-hans }}</ref>。9月15日{{efn|新浪网提及胡锡进发微博的时间是9月16日，参见上下文及其它<ref>{{Cite web |url=https://baijiahao.baidu.com/s?id=1611687097474837888&wfr=spider&for=pc |title=胡锡进：应公布范冰冰事件最新进展，公众有知情权|accessdate=2018-10-12 |author=|date=2018-09-15 |work= |publisher=|language=Zh-hans }}</ref>应是9月15日。}}，中国官方媒体《[[环球时报|环球时报]]》的总编辑[[胡锡进|胡锡进]]在新浪微博表示，范冰冰作为公众人物，范冰冰工作室和调查机构应对公众公开她的近况，公布调查情况，满足公众[[知情權|知情权]]。公开透明，是法制社会应有之义。并称，不管崔永元出于什么动机，此番爆料都属勇敢之举<ref>{{Cite web |url=http://k.sina.com.cn/article_6657898114_18cd7768200100ascp.html?cre=tianyi&mod=pcpager_focus&loc=15&r=9&doct=0&rfunc=100&tj=none&tr=9 |title=胡锡进：应公布范冰冰事件最新进展，公众有知情权|accessdate=2018-10-12 |author=|date=2018-09-16 |work= |publisher=新浪网|language=Zh-hans }}</ref>。至9月底，范冰冰工作室对其行踪仍未公开说明<ref name="a1"/>。进行调查的江苏省和无锡市的税务部门，同样未公开说明。

10月3日，中国官方媒体[[新华社|新华社]]公布调查结果，指出范冰冰拍摄电影《[[大轰炸|大轰炸]]》时取得片酬3000万元，其中1000万元申报纳税，其余2000万元以拆分合同方式偷逃个人所得税618万元，少缴营业税及附加112万元，合计730万元；此外，还查出范冰冰及其担任法定代表人的企业少缴税款2.48亿元，其中偷逃税款1.34亿元<ref name="新华网1003">{{cite news |title=税务部门依法查处范冰冰“阴阳合同”等偷逃税问题 |url=http://www.xinhuanet.com/legal/2018-10/03/c_129965300.htm |agency=新华社 |publisher=新华网 |date=2018-10-03 |access-date=2018-10-04 |archive-url=https://web.archive.org/web/20181003203810/http://www.xinhuanet.com/legal/2018-10/03/c_129965300.htm |archive-date=2018-10-03 |dead-url=no }}</ref>。

10月3日下午，[[崔永元|崔永元]]在[[新浪微博|新浪微博]]发文“事实再一次证明，就是大欺诈，参加欺诈的目前已有一人受到处罚”，同时配以电影《[[大轟炸|大轰炸]]》的宣传图。外界认为“一人受到处罚”即指范冰冰<ref>{{Cite web |url=http://ent.163.com/18/1003/14/DT6V0FS500038FO9.html#f=post1603_tab_news |title=崔永元曝《大轰炸》就是"大欺诈":已有一人被处罚|accessdate=2018-10-30 |author= |date=2018-10-03 |work= |publisher=网易|language=Zh-hans }}</ref>。此前，崔永元与电影《[[大轟炸|大轰炸]]》制片人、原[[快鹿集团|快鹿集团]]董事长[[施建祥|施建祥]]过往甚密。他所列举的明星偷税漏税的合同皆与施建祥相关<ref>{{Cite web |url=http://finance.jrj.com.cn/2018/06/06195024647054.shtml |title=“搅局者”崔永元——揭开被遗忘的“快鹿资本局”|accessdate=2018-10-13 |author=缪凌云 |date=2018-06-06 |work= |publisher=金融网站，来源：野马财经|language=Zh-hans }}</ref>。2016年，快鹿集团系列非法集资案发，由[[上海市公安局|上海市公安局]]长宁公安分局经侦支队侦查。施建祥现为中国政府通辑犯。2018年7月间，崔永元曾指快鹿集团在投资拍摄电影《大轰炸》过程中“欺诈”、“洗钱”。《大轰炸》剧组则回应，电影票房收益将回款给快鹿集团系列非法集资案的受害人<ref>{{Cite web |url=http://news.mtime.com/2018/07/04/1582116.html |title=崔永元炮轰《大轰炸》剧组"大欺诈" 剧组反驳洗钱"我们也想知道15亿谁弄走了"|accessdate=2018-10-30 |author= |date=2018-07-04 |work=编辑：隐饮 |publisher=时光网|language=Zh-hans }}</ref>。

10月7日，崔永元在[[新浪微博|新浪微博]]透过“小崔读书汇”的帐号发表文章《一声长叹一声雷<ref>{{Cite web |url=http://www.sohu.com/a/258200524_532789 |title=崔永元最新发文：一声长叹一声雷|accessdate=2018-10-12 |author= |date=2018-10-08 |work= |publisher=搜狐网|language=Zh-hans }}</ref>》，透露自己在配合国家税务总局调查范冰冰案件过程中的遭遇。称自己在第三次配合调查中，被两位未表明身份、推测为警察的人员审问，同时漠视他补充的证据。文末，明确指出[[上海市公安局|上海市公安局]]经侦大队对其参与的公司进行了彻底侦查，警方人员有违法违纪之举。10日，上海市公安局在新浪微博回应，已成立调查小组，但无法联系崔永元。崔永元回应称，上海警方已对他进行彻底地调查，两名未表明身份人员即是警察冒充税务人员在国家税务总局对他进行盘问。指原上海市公安局长宁公安分局经侦副支队长涉及其中<ref>{{Cite web |url=http://www.zhongworld.com/zh-cn/2018/10/10/%E5%B4%94%E6%B0%B8%E5%85%83%E8%88%89%E5%A0%B1%E6%B0%91%E8%AD%A6%E6%B6%89%E9%81%95%E6%B3%95%E9%81%95%E7%B4%80%E4%B8%8A%E6%B5%B7%E8%AD%A6%E6%96%B9%EF%BC%9A%E8%8B%A5%E6%9F%A5%E5%AF%A6%E5%B0%87%E4%BE%9D/ |title=崔永元举报民警涉违法违纪上海警方：若查实将依法处理|accessdate=2018-10-12 |author= |date= |work= |publisher=中文世界|language=Zh-hans }}</ref>。8月的报道中，记者曾指，范冰冰一案已由税务部门移交警方调查。依中国现有司法程序，税务局对偷税、漏税仅能进行行政处罚，涉及逃税罪，应由公安机关的经侦支队进行侦查<ref name="b1"/>。

同年12月，一批文化影视产业的代表，以匿名方式致公开信国务院总理[[李克强|李克强]]，对国家税务总局向文化影视行业征税的做法表示质疑。<ref>{{Cite news|url=http://cn.rfi.fr/中国/20181218-影视文化界名人疑集体匿名致信李克强质疑范冰冰案引发大罚税|title=影视文化界名人疑集体匿名致信李克强质疑范冰冰案引发大罚税|work=法国国际广播电台|date=2018-12-18}}</ref>
