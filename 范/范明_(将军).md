'''范明'''（原名'''郝克勇'''，{{bd|1914年|12月4日|2010年|2月23日|catIdx=Fan范明}}），[[陕西|陕西]][[西安|西安]]人。曾先后在[[复旦大学|复旦大学]]、[[国民政府|国民政府]]盐务税警官佐学校学习，后毕业于[[国立东北大学|国立东北大学]]政经系。1932年5月，加入[[中国共产主义青年团|中国共产主义青年团]]；1938年初，转为[[中国共产党|中国共产党]]党员。1950年8月受中共指派，护送[[十世班禅|十世班禅]]返回[[塔尔寺|塔尔寺]]，并出任西北军政委员会驻班禅行辕代表。曾历任西北人民解放军进藏部队司令员、兼政治委员，中共西藏工委副书记，[[西藏军区|西藏军区]]副政治委员；1955年，授[[少将|少将]][[军衔|军衔]]。

1958年被打成“[[右派|右派]]”，撤销了党政军一切职务，同时还被开除党籍、军籍；和妻子一起，被发配到[[长白山|长白山]]地区劳动改造；后又涉入“[[彭德怀反党集团|彭德怀反党集团]]”，被关押在[[北京|北京]][[秦城监狱|秦城监狱]]。1980年得到平反后，被任命为陕西省政协常务副主席。2010年2月23日，在[[西安|西安]]病逝，享年95岁。<ref>[http://news.hsw.cn/system/2010/02/26/050443243.shtml 陕籍开国少将范明逝世享年96岁] 原载《华商报》2010年2月26日</ref>

== 早年经历 ==
郝克勇于1914年12月4日出生[[陕西|陕西]][[临潼|临潼]]栎阳镇郝邢村的一个耕读世家。父亲[[郝鹏程|郝鹏程]]是[[西北军|西北军]]的创始人之一，曾在[[杨虎城|杨虎城]]军中担任特种兵营长；与杨虎城、[[赵寿山|赵寿山]]是世交的弟兄。而郝克勇自幼接受良好教育，除熟读《[[四书|四书]]》、《[[五经|五经]]》和唐诗宋词数百首外，能撰写叙事明理的短文，还特别对中医感兴趣。

郝克勇的大哥[[郝克俊|郝克俊]]是经[[邓小平|邓小平]]介绍加入中共的；1928年由[[杨虎城|杨虎城]]送到[[上海|上海]]读大学；次年，郝克勇也跟随哥哥来到上海，入建国中学上学。1932年，[[一·二八事變|“一·二八”淞沪事变]]爆发后，克俊、克勇都加入了[[国民政府军第十九路军|国民第十九军]]的抗日义勇军；当年5月，17岁的郝克勇加入了[[中国共产主义青年团|中国共产主义青年团]]。

1933年，郝克勇考入[[复旦大学|复旦大学]]，主修中文和哲学；1935年，又进入[[国民政府|国民政府]]盐务税警官佐学校学习；毕业后，被分配到[[贵州|贵州]][[天柱县|天柱县]]，当盐务税警分队长。后因扣留[[湖南省|湖南省]][[主席|主席]][[何键|何键]]的走私盐船，而得罪了何键；在“[[西安事变|西安事变]]”爆发后，受到打压，被迫返回[[西安|西安]]。1937年，考入[[国立东北大学|国立东北大学]]西安分校政经系，继续学习。在此期间，组织了抗日救亡团体“夏艺学会”，自任会长；并沟通了与[[共青团|共青团]]组织的关系。1938年初，正式加入中国共产党。

== 革命生涯 ==
在[[抗日战争|抗日战争]]时期，受中共指派，郝克勇在[[国民革命军|国民革命军]]第十七路军三十八军任中共工委委员、书记，三十八军教导队队长，直属搜索连连长；在[[周恩来|周恩来]]、[[李维汉|李维汉]]的直接领导下，从事[[统战部|统战]]、兵运工作。郝克勇利用《[[新华日报|新华日报]]》、《[[大公报|大公报]]》以及[[新华社|新华社]]的消息、社论在军中进行宣传活动；并最终完成了对三十八军的统战工作。他的活动早已受到[[国民政府|国民政府]]怀疑。1942年7月，[[毛泽东|毛泽东]]致电三十八军军长[[赵寿山|赵寿山]]，要郝克勇到[[延安|延安]]向中央汇报工作。因为长期从事秘密工作，在一次交谈时，毛泽东建议郝克勇改名；并用郝的母姓，为他取名“范明”。<ref>[http://cpc.people.com.cn/GB/68742/73841/73842/5499246.html 毛泽东为他取名 传奇的“六多”将军范明] {{webarchive|url=https://web.archive.org/web/20100407210037/http://cpc.people.com.cn/GB/68742/73841/73842/5499246.html |date=2010-04-07 }} 原载《炎黄春秋》 作者 李敏杰、赵仕枢</ref>

[[国共内战|国共内战]]时期，范明担任中共中央西北局统战部处长，负责进行西北五省、以及[[河南|河南]]和[[四川|川北]][[国民党|国民党]]控制区中共地下组织的布点任务。1949年，出任[[中国人民解放军第一野战军|中国人民解放军第一野战军]]政治部秘书长、兼联络部部长；参加了[[延安|延安]]保卫战及青化砭、羊马河、蟠龙、沙家店、宜川、荔北、扶眉、[[兰州|兰州]]等战役；是[[彭德怀|彭德怀]]的助手。

== 进藏司令 ==
[[中华人民共和国|中华人民共和国]]建国前夕，受中共指派，范明赴[[西宁|西宁]]，争取当时还只有11岁的[[十世班禅|十世班禅]]对新政府的支持；并在1950年8月25日，帮助十世班禅回到[[塔尔寺|塔尔寺]]。[[彭德怀|彭德怀]]曾打算让范明出任中共在[[乌鲁木齐|乌鲁木齐]]的负责人；后中共又拟派他赴[[朝鲜|朝鲜]]，参加[[抗美援朝|抗美援朝]]；但最终还是确定让他负责进军[[西藏|西藏]]的任务。不久，范明即出任中共西北局西藏工委书记、西北军区进藏部队司令员兼政委，西北军政委员会驻班禅行辕代表等职。

1951年4月，范明陪同十世班禅进京；5月，“[[十七条协议|十七条协议]]”签署；8月，范明率部队由[[青海|青海]]进驻[[西藏|西藏]]。1952年，[[西藏军区|西藏军区]]成立后，范明出任中共西藏地区工作委员会副书记、西藏军区(第一)副政治委员。1955年，41岁的范明授[[少将|少将]][[军衔|军衔]]。

== 牢狱之苦 ==
1958年4月，范明被打成了“[[右派|右派]]”，随即被撤销了党政军的一切职务，并被开除党籍、军籍；和妻子梁枫，被押到东北长白山劳动改造，后因病被转到[[陕西|陕西]][[大荔|大荔]]的劳改农场。1962年9月，又因“[[彭德怀反党集团|彭德怀反党集团]]”的牵连，被关押到[[秦城监狱|秦城监狱]]。

== 钻研中医 ==
范明早年就对中医有所研究。曾经为其父亲医治癌症，还经常为中共的领导和家属看病。在范明被强制劳动改造和关押监狱的二十多年里，他所有的东西都被没收了；仅剩下当年[[毛泽东|毛泽东]]赠送给他的那些车医书，没人敢收。范明在这期间，就潜心研究[[中医|中医]]。在获得了上山采药、行医看病的特权后，他不仅医治了大量疑难杂症，还把中药方面的知识用于[[小麦|小麦]]、[[棉花|棉花]]的增产，以及对牲畜的疾病医治上。<ref>[http://www.china.com.cn/military/txt/2007-08/07/content_8640391_4.htm 毛泽东与功臣们故事多] 原载《解放军报》</ref>

== 恢复自由 ==
1980年，[[中共中央组织部|中共中央组织部]]对范明进行复查，最终给予平反。此后，范明历任第六、第七届[[全国政协|全国政协]]委员，政协陕西省委员会第四、第五届副主席。

2001年7月，87岁的范明被邀请前往[[拉萨|拉萨]]，参加“庆祝西藏和平解放50周年盛典”等纪念活动，受到时任[[中共中央政治局常委|中共中央政治局常委]]、[[中华人民共和国副主席|国家副主席]]、[[中央军委|中央军委]]副主席[[胡锦涛|胡锦涛]]的接见。2008年年初，当范明在电视上看到[[拉萨|拉萨]]发生[[2008年西藏骚乱|暴力衝突]]时，气愤地摔碎了手中的玻璃杯。<ref>[http://www.tibet.cn/news/rwxw/jlrw/200905/t20090525_478310.htm 老将军范明的进藏传奇]</ref>
 
2010年2月23日18时，95岁的范明在[[西安|西安]]病逝。[[中共中央政治局常委|中共中央政治局常委]]、[[中华人民共和国副主席|国家副主席]]、[[中央军委|中央军委]]副主席[[习近平|习近平]]送了花圈。
