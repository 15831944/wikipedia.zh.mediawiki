'''范姜泰基'''（{{bd|1970年|9月12日||}}），生於台灣[[桃園市|桃園縣]]（今桃園市），[[臺灣客家人|臺灣客家人]]，現任電視演員，曾任[[中時晚報|中時晚報]]、[[中國時報|中國時報]]、[[自由時報|自由時報]][[記者|記者]]，專跑[[政治|政治]]。[[中國國民黨|中國國民黨]]籍，曾任[[郝龙斌|郝龙斌]]競選臺北市長辦公室[[发言人|发言人]]與[[中华民国总统府发言人|中华民国总统府发言人]]<ref>[http://www.tvbs.com.tw/news/news_list.asp?no=yehmin20110117205721&&dd=2013/2/4%20%A4W%A4%C8%2010:00:21 換血！　范姜泰基任國民黨發言人], [[TVBS|TVBS]], 2011/1/17</ref><ref name="xdw ">{{cite web|url=http://news.stnn.cc/hk_taiwan/201105/t20110504_1557782.html|title=范姜泰基任总统府发言人:表现沉稳 滴水不漏|publisher=星岛环球网|accessdate=2011-05-04|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20131016014111/http://news.stnn.cc/hk_taiwan/201105/t20110504_1557782.html|archivedate=2013-10-16}}</ref><ref name="CNA">{{cite web |url=http://www.cna.com.tw/News/FirstNews/201105040024-1.aspx |title=范姜泰基接府發言人 曝光客家 |publisher=中央通訊社 |date=2011-05-04 |accessdate=2011-01-07 |work= |language =zh-tw}}</ref>。

== 生平 ==
范姜泰基出身[[台灣客家人|台灣客家人]]，[[范姜姓|范姜]]為[[雙姓|雙姓]]。畢業於[[國立臺灣大學|國立臺灣大學]]政治學研究所、[[國立政治大學|國立政治大學]][[EMBA|EMBA]]<ref>{{cite web |url=http://www.president.gov.tw/Default.aspx?tabid=131&itemid=24157 |title=本府發言人由范姜泰基接任 |publisher=中華民國總統府 |accessdate=2011-05-04 |language =zh-tw}}</ref>。曾任[[郝龙斌|郝龙斌]]竞选臺北市長办公室发言人，2011年5月4日担任中华民国总统府发言人。<ref name="CNA"/>

2013年2月，范姜泰基接任[[江宜樺內閣|江宜樺內閣]]之[[行政院研究發展考核委員會|行政院研究發展考核委員會]]副主委<ref>[http://www.bcc.com.tw/news/newsview.asp?cde=2000088 簡太郎升政院副祕 內政部政次蕭家淇接]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}，中廣新聞網，2013/02/06</ref><ref>[http://www.rdec.gov.tw/ct.asp?xItem=4532177&CtNode=11955&mp=100 范姜泰基]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}，[[行政院研究發展考核委員會|行政院研究發展考核委員會]]</ref>。2013年9月17日，總統府副祕書長[[羅智強|羅智強]]因[[九月政爭|九月政爭]]去職，[[中國國民黨文化傳播委員會|中國國民黨文化傳播委員會]]主委[[蕭旭岑|蕭旭岑]]接任總統府副祕書長，范姜泰基轉任[[中國國民黨文化傳播委員會|中國國民黨文化傳播委員會]]主委至2015年1月20日。離開政治圈後，現以演員的身分演出電視劇。
