'''范长江'''（{{bd|1909年||1970年|10月23日|catIdx=F范长江}}），原名'''希天'''，[[四川|四川]][[内江|内江]]人。新闻[[记者|记者]]、编辑。民国时期《[[大公报|大公报]]》著名通讯记者，后转向[[共产党|共产党]]阵营，[[中华人民共和国|中华人民共和国]]成立后担任《[[解放日报|解放日报]]》社长、新闻总署副署长、《[[人民日报|人民日报]]》社长、国家科委副主任、全国科协副主席兼党组书记等职。[[文革|文革]]时期遭到监禁并受到迫害，1970年在其下放劳改地[[河南|河南]][[确山|确山]]一口枯井中发现其遗体，懷疑为自杀身亡。1978年12月27日得平反。

==生平==
1928年考入[[南京中央政治学校|南京中央政治学校]]乡村行政系，[[九一八事变|九一八事变]]后脱离该校，转入北大哲学系，积极参加抗日救亡运动。

1933年起为北平《[[晨报|晨报]]》、《[[世界日报|世界日报]]》和[[天津|天津]]《[[大公报|大公报]]》、《[[益世报|益世报]]》等撰稿<ref name="范长江青年时代的传奇经历">{{cite news |url = http://www.people.com.cn/GB/14677/21965/22070/2449237.html|title = 范长江青年时代的传奇经历 |accessdate =2009-06-09 |author = 赵华能|date =2004年4月15日 |language =zh-hans |publisher =人民网}}</ref>。

1935年7月，以《大公报》旅行记者的身份，深入西北地区考察旅行，历时10个月，行程6000余公里，所写旅行通讯在《大公报》上连载，公开报道了[[中国红军|红军]][[长征|长征]]和西北状况，引起轰动<ref>{{cite web |url =http://www.zgdazxw.com.cn/NewsView.asp?ID=603|title = 范长江与他的《中国的西北角》|accessdate =2009-06-09 |author = 王 鹏|date =2007年11月2日 |language =zh-hans |publisher =中国档案资讯网}}</ref>。一举成为闻名全国的新闻记者。这些通讯后来辑为《中国的西北角》一书出版，半年内重印7次。1936年8月赴[[綏遠|綏遠]]西部采访，所写通讯辑为《[[塞上行|塞上行]]》一书出版。

[[西安事变|西安事变]]后赴[[延安|延安]]采访，受到[[毛泽东|毛泽东]]等中共领导人接见<ref name="范长江青年时代的传奇经历"/>。后写有《陕北之行》、《动荡中之西北大局》等通讯。[[抗战|抗战]]爆发后，采写大量战地通讯。1938年3月，与[[恽逸群|恽逸群]]等人发起成立[[中国青年新闻记者学会|中国青年新闻记者学会]]，网罗了众多红色背景和亲共的新闻工作者。

1938年10月脱离《大公报》，脱离原因存在争议<ref name="范长江的悲剧"/>，后与[[胡愈之|胡愈之]]、[[孟秋江|孟秋江]]等人创办国际新闻社。

1939年5月经周恩来介绍在重庆加入中国共产党<ref name="范长江青年时代的传奇经历"/>，后与[[沈钧儒|沈钧儒]]的女儿[[沈谱|沈谱]]相识，结为伉俪<ref>{{cite web |url =http://d.wanfangdata.com.cn/Periodical_wsjh200109008.aspx|title = 范长江与沈谱的婚礼|accessdate =2009-06-09 |author = 冯大彪 |date =2001年9期 |language =zh-hans |publisher =文史精华 }}</ref><ref name="刘竞文"/>。

1941年5月在[[香港|香港]]与[[邹韬奋|邹韬奋]]等创办《[[华商报_(香港)|华商报]]》。

1942年后进入苏北解放区，先后任[[新华社|新华社]]华中分社社长、《[[新华日报|新华日报]]》华中版社长、华中新闻专科学校校长。

1947年任新华社“四大队”队长，随党中央转战[[陕北|陕北]]，负责新闻宣传工作。

1949年任新华社副社长。1月，作为新闻先遣队负责人之一，从[[西柏坡|西柏坡]]出发到北平接管新闻报业。1月31日进去北平，任[[中共北平市委|中共北平市委]]机关报《[[人民日报北平版|人民日报北平版]]》总编辑。3月15日《人民日报北平版》停刊，范长江随同[[第三野战军|第三野战军]]领导机关南下上海，担任新创刊的[[中共上海市委|中共上海市委]]机关报《[[解放日报|解放日报]]》社长。<ref>范长江为什么离开《人民日报》，钱江，百年潮2009年第六期</ref>

1950年1月，调往北京，任[[人民日报社|人民日报社]]社长。任社长期间，因对一些老干部提出过于严厉的批评，产生很多矛盾。

1952年初春，临时调到中国人民大学领导“三反”、“五反”运动；同时人民日报社内部运动也达到高潮，一些干部对范长江提出很多尖锐的批评。[[胡乔木|胡乔木]]得知消息后派[[黎澎|黎澎]]到人民日报报社调查，后让范长江在编委会做了两次检查。因矛盾依然过于尖锐，1952年6月将范长江调离人民日报社。<ref>范长江为什么离开《人民日报》，钱江，百年潮2009年第六期</ref>

1952年任[[政務院文化教育委員會|政務院文化教育委員會]]副秘書長。1954年，他轉任國務院第二辦公室副主任。1956年任國家科委副主任，兩年後任全國科協副主席兼黨組書記。

[[文革|文革]]爆发后遭到迫害和流放。

1970年10月23日在[[河南|河南]][[确山|确山]]一口枯井中发现范长江的遗体，死因未明<ref>{{cite web |url =http://www.gmw.cn/content/2004-11/30/content_137949.htm|title = 新闻巨子范长江|accessdate =2009-06-09 |author = 王玲|date =2004年11月30日|language =zh-hans |publisher =[[光明日报|光明日报]]}}</ref>。

==评价及影响==
* 范長江被认为是第一位報導[[中國工農紅軍|紅軍]]并引起轰动的國統區記者。[[抗日戰爭|抗日戰爭]]中他是活躍在各戰區的[[戰地記者|戰地記者]]，共产党及其宣传机构在中华民国大陆统治时期和范长江身后均给予了范长江极高的评价和肯定。[[周恩來|周恩來]]曾在給他的信中称赞道：「聽到你飽載著前線上英勇的戰訊，並帶著光榮的傷痕歸來，不僅使人興奮，而且使人感念。」

* 在范长江老家[[四川|四川]][[内江|内江]]建有范长江纪念馆。为范长江故居，位于东兴区田家镇<ref>{{cite web |url = http://www.sichuandaily.com.cn/2009/03/18/20090318350404170932.htm|title = 范长江纪念馆修复工程开工 |accessdate = 2009-06-09|author = 许静|coauthors =熊永志 |date = 2009年3月18日|language =zh-hans |publisher =四川日报}}</ref>。

* 1991年，[[中国记者协会|中国记者协会]]与范长江新闻奖基金会联合设立了“[[范长江新闻奖|范长江新闻奖]]”，成为表彰奖励[[中华人民共和国|中华人民共和国]]中青年新闻记者的全国性高层次新闻奖，2005年与[[鄒韜奮|韬奋]]奖合并为[[长江韬奋奖|长江韬奋奖]]。

* 当代学者[[傅国涌|傅国涌]]认为范长江的一生看似光鲜，却实属悲剧，只是一枚政治棋子而已。在他投身共产党后才华便随之消失，成为了共产党的工具，丧失了新闻人的独立人格。而且当政党不需要的时候，就将其遗弃，成为历史的悲剧人物<ref name="范长江的悲剧">{{cite web|url =http://www.edubridge.com/erxiantang/library/fanchangjiang.htm|title =范长江的悲剧|accessdate =2009-06-09|author =傅国涌|date =2003年3月16日|language =zh-hans|publisher =二闲堂|deadurl =yes|archiveurl =https://web.archive.org/web/20041019011354/http://www.edubridge.com/erxiantang/library/fanchangjiang.htm|archivedate =2004年10月19日}}</ref>。
