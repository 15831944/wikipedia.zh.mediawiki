'''范宅'''旧称'''西湖范氏'''和'''察院前范家'''，位于[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[海曙区|海曙区]]迎凤街130号。范宅建于明[[万历|万历]]、[[天启_(明朝)|天启]]年间，是[[范仲淹|范仲淹]]第十七世孙[[范亿|范亿]]所修建，现为宁波市规模最大，保存最完整的明代住宅建筑、著名旅游景点。1981年12月，范宅被公布为海曙区文物保护单位。

==历史==
范宅建于明万历、天启年间，是[[范仲淹|范仲淹]]第十七世孙范亿所修建的私人住宅，后住户逐渐增加到70多家。<ref>{{cite web |url= http://www.nbwb.net/pd_wwbh/info.aspx?Id=3343|title=甬城瑰宝话范宅 |last=广华 |first=王|date=2007-06-25  |website= |publisher=宁波市文物保护管理所 |access-date= |quote=}}</ref><ref>{{cite web |url= http://202.107.212.146:838/nbwshj/index.php?option=com_content&view=article&id=654:2011-09-15-02-13-07&catid=53:m&Itemid=72|title=范宅整修与开放 |last=杏荣 |first=刘 |date= |website= |publisher=宁波市政协文史资料委员会 |access-date= |quote=范宅在人间沧桑400余年，而聚居者日众，达70余户}}</ref>1981年12月，范宅被公布为海曙区文物保护单位。因为年久失修，1995年海曙区政府启动了范宅的拆迁改造工程。在这次维修改造中，有关部门先后提出了两个方案，一为拆除当时被误认作清时所建的两侧及后进建筑而保留主体，二为保留整体建筑并恢复至明时建筑风格。经过考察研究最终采取了方案二。<ref>{{cite web |url= http://202.107.212.146:838/nbwshj/index.php?option=com_content&view=article&id=654:2011-09-15-02-13-07&catid=53:m&Itemid=72|title=范宅整修与开放 |last=杏荣 |first=刘 |date= |website= |publisher=宁波市政协文史资料委员会 |access-date= |quote=1996年1月16日上午，陈守义、宋小六两位副市长再次召集市规划局、文化局、房地局、市政公用局和海曙区政府的有关负责同志，就范宅改造总体规划方案进行了专题研究。会议同意海曙区政府提出的范宅改造总体规划第二方案，即保留整体建筑，恢复明代建筑风格，正门设在南面朝东方向；拆除周围建筑，以园林绿地布置与范宅相协调。}}</ref>1997年7月，范宅维修改造完成向社会开放，成为了一个古玩市场。<ref>{{cite web|url=http://whnb7000.com/list.aspx?cat=8&id=609|title=范宅的前世今生|last=|first=|date=2017-04-14|website=|publisher=宁波市文化广电新闻出版局|access-date=|quote=　范宅是现今宁波一个很有名的古玩市场。|deadurl=yes|archiveurl=https://web.archive.org/web/20170627142508/http://whnb7000.com/list.aspx?cat=8|archivedate=2017-06-27}}</ref>2017年3月，海曙区政府对[[中山路_(宁波)|中山路]]沿线文物保护单位进行整修，范宅包含在内。<ref>{{cite web |url= http://www.nbwb.net/info.aspx?Id=38847|title=范宅修缮整治工程即将启动 |last=之润 |first=章 |date=2017-03-24 |website= |publisher=宁波市文物保护管理所 |access-date= |quote=}}</ref>
==结构==
范宅整体坐北朝南，为木结构建筑，现存建筑占地约2100平方米。位于建筑中轴线上的建筑有影壁、前进、中楼、后楼，两侧建有厢房，自北向南屋脊依次升高，整体成一日字形。建筑大门坐西朝东，门上有“大夫第”横匾。<ref>{{cite web |url=http://news.cnnb.com.cn/system/2013/08/25/007826402.shtml |title=范宅的前世今生:现今宁波一个很有名的古玩市场|last=冠夏 |first=吴 |date= 2013-08-25|website= |publisher=中国宁波网 |access-date= |quote=现存的范宅，主体建筑坐北朝南，中轴线上依次为影壁、前进、中楼、后楼，两侧是厢房，大门、仪门在前进东侧，大门坐西朝东，三开间，门前上方置“大夫第”横匾一方；大夫即指范廷谋。该宅整体建筑呈“日”字形，但从左右厢房的后走廊与第三进的后走廊相接，又与第二、三进的前走廊相连来看，又呈“月”字形，暗寓一“明”字，独具匠心。}}</ref>建筑总体共三进，第一进为面宽三间二弄带左右挟屋的单檐硬山式，细节较为精致，月梁、雀替等有阴刻，照壁上下边有卷草砖雕。第二进主体和一进类似，唯南北延伸和一进挟屋相连，北与后进相连区别于一进。后进为重檐歇山顶建筑，其二层设有透光、通风、防盗之用带直条木栏杆的窗户。<ref>{{cite web |url=http://www.nbwb.net/pd_wwly/info.aspx?Id=323&type=2|title=范宅|last= |first=|date=2006-02-09|website= |publisher=宁波市文物保护管理所 |access-date= |quote=该宅共三进，整体结构与部分装饰保存尚好。第一进主体为单檐硬山式，面宽三间二弄，带左右挟屋，其中明间拾梁式，前后檐柱柱头卷杀，上承十字斗拱，下坐鼓形柱础；次间为穿斗式，前后檐柱同明间，两挟屋隔山墙紧贴左右弄，用材较堂厅小，在梁架加工装饰上，该进较其它两进更为精致细腻，如月梁、雀替等都有线条流畅明快的阴刻，照壁上下边均有卷草砖雕。第二进主体结构基本与前进相同，而左右各布置东西方向的楼房，不同于前进的挟屋，楼房向南北延伸至与前进挟屋相连，北方后进相接，使其形成一个封闭式建筑群。后进为重檐歇山顶建筑，檐角略起翘，上层外围窗前，有直条木栅栏，以起到透光、通风及防盗作用。 }}</ref>
