'''范敬宜'''{{BD|1931年||2010年|11月13日|catIdx=Fan范敬宜}}，[[江苏|江苏]][[苏州|苏州]]人，原《[[人民日报|人民日报]]》总编辑。

范敬宜出生名门，是[[范仲淹|范仲淹]]28世孙。祖父[[范端信|范端信]]是[[范氏義莊|范氏義莊]]和[[文正書院|文正書院]]的主奉。父亲[[范承達|范承達]]毕业于[[交通大学|交通大学]]，母亲蔡佩秋也出身于书香门第。<ref name="wenweipo">{{Cite web |url=http://paper.wenweipo.com/2010/11/14/CH1011140014.htm |title=人民日報社前總編輯 范敬宜昨在京病逝 |accessdate=2010-11-14 |publisher=文汇报}}</ref><ref>{{Cite web |url=http://dailynews.sina.com/bg/chn/overseamedia/chinapress/20101114/00031994413.html |title=《人民日報》原總編范敬宜病逝 任內常以讀者來信當報紙頭版頭條 |accessdate=2010-11-14 |publisher=新浪网}}</ref>

幼年曾师从[[吴门画派|吴门画派]][[樊伯炎|樊伯炎]]<ref name="wenweipo"/>。1949年毕业于[[无锡国学专修学校|无锡国学专修学校]]，1951年毕业于[[上海|上海]][[圣约翰大学|圣约翰大学]]中文系。后任《[[东北日报|东北日报]]》（后改为《[[辽宁日报|辽宁日报]]》）编辑。1957年被打成右派，到农村劳改。1978年在辽宁[[建昌县|建昌县]]以“右派”身份入党。1979年回到《辽宁日报》，历任《辽宁日报》农村部副主任、主任、副总编辑，1984年任[[文化部|文化部]][[外文局|外文局]]局长，1986年任《[[经济日报|经济日报]]》总编辑，1993年任《人民日报》总编辑。<ref>{{Cite web |url=http://news.tsinghua.edu.cn/new/news.php?id=2850 |title=范敬宜简历 |accessdate=2010-11-14 |publisher=清华大学 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304195700/http://news.tsinghua.edu.cn/new/news.php?id=2850 |archivedate=2016-03-04 }}</ref><ref name="范敬宜的新闻人生">{{Cite web |url=http://www.infzm.com/content/33106 |title=范敬宜的新闻人生 |accessdate=2010-11-14 |publisher=南方周末}}</ref>

1998年起任[[第九届全国人民代表大会|第九届全国人民代表大会]]常务委员会委员、[[全国人民代表大会教科文卫委员会|人大教科文卫委员会]]副主任委员。2002年4月被[[清华大学|清华大学]]聘为教授、[[清华大学新闻与传播学院|新闻与传播学院]]院长。兼任[[中国新闻摄影学会|中国新闻摄影学会]]会长、[[中国新闻文化促进会|中国新闻文化促进会]]会长、[[中国人民大学|中国人民大学]]新闻学院和[[武汉大学|武汉大学]]新闻学院兼职教授、[[中国社会科学院|中国社会科学院]]研究生院新闻系博士生导师。<ref name="范敬宜的新闻人生"/>

2010年11月13日13时42分在[[北京医院|北京医院]]去世，享年79岁。<ref>{{Cite web |url=http://www.infzm.com/content/52432 |title=老报人范敬宜逝世，享年79岁 |accessdate=2010-11-14 |publisher=南方周末 }}</ref>

主要著作有《总编辑手记》、《敬宜笔记》、《范敬宜诗书画》、《范敬宜文集·新闻作品选》等。<ref name="wenweipo"/><ref name="范敬宜的新闻人生"/>
