'''珍·玛奇'''（{{lang-en|'''Jane March'''}}，{{bd|1973年|3月20日||catIdx=M}}）是一名[[英国|英国]]电影电视演员和前时尚模特。

== 生平 ==
1973年，玛奇出生在[[英格兰|英格兰]][[伦敦|伦敦]]埃奇韦尔，<ref name="Evening Standard">{{cite web |url=http://www.thisislondon.co.uk/film/article-8793288-the-original-sinner.do |title=The original sinner |first=Grace |last=Bradberry |work=[[Evening_Standard|Evening Standard]] |location=London |date=23 January 2004 |accessdate=21 April 2010}}</ref><ref name="Daily Mail">{{cite web |url=http://findarticles.com/p/news-articles/daily-mail-london-england-the/mi_8002/is_2004_March_20/sinner-pinner-teenager-jane-march/ai_n37169555/ |title=What became of the Sinner from Pinner |first=Rebecca |last=Hardy |work=[[Daily_Mail|Daily Mail]] |location=London |date=20 March 2004 |accessdate=21 April 2010}}</ref>她的父亲伯纳德·霍尔伍德，是一名中学教师，有[[英格兰人|英格兰人]]和[[西班牙人|西班牙人]]的血统；<ref>{{cite web|title=Jane March|url=http://www.imdb.com/name/nm0001506/bio|accessdate=July 15, 2012}}</ref>她的母亲珍妮，有[[越南人|越南人]]和[[中国人|中国人]]的血统，他们家还有一个哥哥。

14岁时，玛奇参加模特大赛，荣获冠军，后加盟风暴模特公司（Storm Model Management），开始了模特儿生涯。<ref name="Daily Mail"/>

玛奇的电影处女作是[[让·雅克·阿诺|让·雅克·阿诺]]执导的电影《[[情人_(1992年电影)|情人]]》，合作演员是[[梁家辉|梁家辉]]，后来陆续出演多部电影电视作品。<ref>{{cite web |url=http://www.people.com.cn/h/2011/0704/c25408-2748195308.html |title=珍·玛奇“重温亚洲” |accessdate= 2011年7月4日|publisher= 人民网|archiveurl=http://www.people.com.cn/h/2011/0704/c25408-2748195308.html |archivedate= 2012年12月15日|work= |language =zh-cn}}</ref>

2011年，她和[[邓超|邓超]]、[[伊能静|伊能静]]等合作的《情人2：巴黎宝贝》上映。<ref>{{cite web |url= http://ent.ifeng.com/movie/news/mainland/detail_2010_10/25/2885773_0.shtml|title=被《情人》导演看中 邓超巴黎牵手珍-玛奇上演姐弟恋  |accessdate=2010年10月25日 |publisher=凤凰网 |archiveurl=http://ent.ifeng.com/movie/news/mainland/detail_2010_10/25/2885773_0.shtml |archivedate=2012年12月15日 |work= |language =zh-cn}}</ref>
