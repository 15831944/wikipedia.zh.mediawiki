'''珍妮特·卡罗尔·沃尔芬巴格'''（{{lang-en|'''Janet Carol Wolfenbarger'''}}，{{bd|1958年|||CatIdx=Janet Carol Wolfenbarger}}），美国空军上将，2012年6月5日担任[[美国空军装备司令部|装备司令部]]第8任司令，是美国乃至全世界极少数女性上将之一。<ref name="wolfenbargerwrightpatterson">{{cite web|url=http://www.af.mil/news/story.asp?id=123180614|title=Senate confirms new AFMC vice commander|date=2009-12-04|publisher=af.mil|accessdate=2009-12-05|archiveurl=https://archive.is/20121212035820/http://www.af.mil/news/story.asp?id=123180614|archivedate=2012-12-12|deadurl=yes}}</ref> <ref name="AFMC">{{cite web |url=http://www.hanscom.af.mil/news/story.asp?id=123304811 |title=Gen. Wolfenbarger receives fourth star, assumes leadership of AFMC |author=Kim Dawley |date=June 5, 2012 |work= |publisher=Air Force Materiel Command Public Affairs |accessdate=June 5, 2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20130224051202/http://www.hanscom.af.mil/news/story.asp?id=123304811 |archivedate=2013年2月24日 }}</ref>

==简介==
沃尔芬巴格1976年进入[[美国空军学院|美国空军学院]]学习，是1975年10月[[约翰逊总统|约翰逊总统]]批准军校招收女学员后，首批进入美国空军学院157名女学员之一，也是96名顺利毕业的女学员之一。她于1980年从毕业，获得工程科学学士学位，被委任为空军军官，在[[佛罗里达州|佛罗里达州]]的[[埃格林空军基地|埃格林空军基地]]从事情报分析，在空军装备部门服役多年，并在[[麻省理工学院|麻省理工学院]]获得航空航天理学硕士学位，在[[武装部队产业学院|武装部队产业学院]]获得国家资源战略硕士学位。

沃尔芬巴格是当今美国空军最高级别的女性军官，2010年1月1日，[[特里·加布雷斯基|特里·加布雷斯基]]中将退役后，沃尔芬巴格成为空军现役最高级别的女性军官。

2012年2月6日，奥巴马总统提名沃尔芬巴格为[[美国空军装备司令部|装备司令部]]司令，并晋升上将，2012年3月26日该项任命在[[参议院|参议院]]通过，2012年6月5日，她就任新职，成为[[美国空军|美国空军]]史上第一位女性上将。

沃尔芬巴格在[[俄亥俄|俄亥俄]][[比弗克里克|比弗克里克]]就读比弗克里克高中，于1976年毕业。沃尔芬巴格出生于军人家庭，她的父亲埃尔登是空军退役少校，他的丈夫克雷格是空军退役上校。

*1983年，[[空军中队学校|空军中队学校]]，函授
*1985年，[[麻省理工学院|麻省理工学院]]，获得航空航天理学硕士学位
*1988年，[[弗吉尼亚州|弗吉尼亚州]]，[[贝尔沃堡|贝尔沃堡]]，[[国防系统管理学院|国防系统管理学院]]，项目管理课程
*1991年，[[空军指挥与参谋学院|空军指挥与参谋学院]]，函授
*1994年，[[华盛顿特区|华盛顿特区]]，[[国防大学|国防大学]]，[[武装部队产业学院|武装部队产业学院]]，获国家资源战略理学硕士学位
