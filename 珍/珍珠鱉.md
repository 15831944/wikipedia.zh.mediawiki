'''珍珠鱉'''（[[学名|学名]]：{{lang|la|'''''Apalone ferox'''''}}，{{lang-en|'''Florida softshell turtle'''}}）亦稱'''佛羅里達鱉'''，为[[鱉科|鱉科]][[滑鱉屬|滑鱉屬]]下的一个种。

== 分佈 ==

主要分佈於美國[[佛羅里達州|佛羅里達州]]，也見於[[佐治亞州|佐治亞州]]、[[南卡羅來納州|南卡羅來納州]]和[[阿拉巴馬州|阿拉巴馬州]]。<ref name=usgs>[http://nas.er.usgs.gov/queries/FactSheet.asp?speciesID=1273 ''Apalone ferox'' (Schneider, 1783).] USGS.</ref>

== 描述==

珍珠鱉背殼呈深褐色到橄欖綠色，革製背殼下部為白色，<ref name=apalone/> 用以隱藏自身躲避天敵。<ref>[http://www.telegraph.co.uk/earth/wildlife/6838752/Heron-tries-to-swallow-turtle-for-dinner.html Heron tries to swallow turtle for dinner.] [[Daily_Telegraph|Daily Telegraph]], 18 Dec 2009.</ref> 珍珠鱉有很長的脖頸和頭部，鼻子類似[[浮潛|浮潛呼吸器]]。是滑鱉屬最大的物種，長度可達{{convert|15|-|76|cm|in}}。<ref name=apalone/> 雌性個體比雄性大，雄性一般長約{{convert|35|cm|in}}。雌性一般長約60cm，重達{{convert|20|kg|lb}}，最高紀錄是{{convert|43.5|kg|lb}}。<ref name=iucn/> 幼年體呈橄欖黃色，有灰色斑點和黃色條紋，頭部也有黃色紋路，腹甲為灰色。成年之後消失。<ref name=apalone>[http://www.tortoise.org/archives/apalone.html ''Apalone''.] www.tortoise.org</ref>

== 習性==
珍珠龜是一種完全的水生鱉，只有在曬太陽和產卵是會離開水面。<ref name=apalone/> 不過無論是水中還是陸地上，其移動速度都很快。<ref name=apalone/> 珍珠龜是[[食肉動物|食肉動物]](主食魚類、兩棲類、昆虫類、甲殻類、貝類)，有時也[[食腐動物|食腐]]，飼養時也食用人工飼料。<ref name=iucn/>

繁殖形態為卵生，在水邊地面挖洞，一次產2-14枚卵。

== 生態學==
[[短吻鱷|短吻鱷]]是這種鱉的天敵，一些鳥類則會捕捉其幼年體。其他天敵還包括狐狸、浣熊和臭鼬。<ref name=iucn/>
