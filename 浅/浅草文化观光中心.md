'''浅草文化观光中心'''（{{lang-ja|浅草文化観光センター}}）是位于日本[[東京都|東京都]][[台東區|台東區]][[雷門_(台東區)|雷門]]二丁目的一处旅游信息咨询中心，2012年4月开业，由[[隈研吾|隈研吾]]建設都市設計事務所设计。

== 歴史 ==
淺草文化觀光中心現址原本有一座二层银行建筑，1985年台东区将其买下，开办浅草文化观光中心。<ref name="kotobank">{{cite web |script-title=ja:浅草文化観光センター |url=https://kotobank.jp/word/浅草文化観光センター-890047 |website=コトバンク |accessdate=2018-11-06  |language=ja}}</ref>由于建成已超过50年，建筑老化。2008年，台东区征集新建筑的设计方案，<ref>{{cite web |script-title=ja:浅草文化観光センター設計案コンペが開催 |url=http://architecturephoto.net/21451/ |website=architecturephoto.net |accessdate=2018-11-06 |date=2008-09-04 |language=ja |archive-url=https://web.archive.org/web/20150401153933/http://architecturephoto.net/21451/ |archive-date=2015-04-01 |dead-url=no }}</ref>[[隈研吾|隈研吾]]的方案入选。建设工程总费用为16億日元，<ref name="kotobank"/>2012年完工。

== 建築 ==
浅草文化观光中心外形为竖直堆叠的平房，每层的高度、屋顶角度、内部装修各不相同。每层均装有不燃人造杉木板制成的外遮陽百葉。<ref>{{cite web |script-title=ja:浅草文化観光センター |url=https://emachu.co.jp/case/浅草文化観光センター.html |website=Emachu |archiveurl=https://web.archive.org/web/20170617155344/https://emachu.co.jp/case/%E6%B5%85%E8%8D%89%E6%96%87%E5%8C%96%E8%A6%B3%E5%85%89%E3%82%BB%E3%83%B3%E3%82%BF%E3%83%BC.html |archivedate=2017-06-17 |language=ja |access-date=2018-11-06 |dead-url=no }}</ref>隈研吾解释称，供建筑使用的土地不够宽敞，没有将它设计为细长的{{link-ja|铅笔楼|ペンシルビル}}，而是采用平房堆叠的设计以适应周边环境。<ref>{{Cite web|date=2012-04-18|url=http://kensetsunewspickup.blogspot.jp/2012/04/20.html|script-title=ja:平屋建てを積み重ねた建築　隈研吾の「浅草文化観光センター」が20日オープン|publisher={{lang|ja|建設通信新聞公式ブログ}}|accessdate=2016-06-24|language=ja|archive-url=https://web.archive.org/web/20120713084917/http://kensetsunewspickup.blogspot.jp/2012/04/20.html|archive-date=2012-07-13|dead-url=no}}</ref>浅草文化观光中心结合了日本传统建筑的木文化和现代建筑的体积感。<ref>{{cite journal |title=东京 浅草的新视野 |journal=缤纷 |date=2013 |issue=5 |page=20-21 |issn=1005-0914}}</ref>有评论认为建筑传承了浅草地区的传统风格，有江户时代的美感。浅草文化观光中心为钢筋混凝土结构，但大量使用木材和玻璃，通风良好，视野开阔，充分利用自然光。<ref>{{cite journal |author1=Jacob |title=浅草文化中心 |journal=设计 |date=2013 |issue=9 |page=52-53 |issn=1003-0069}}</ref><ref>{{cite journal |author1=张慧超 |title=以东京浅草文化观光中心为例浅谈对东方建筑文化的理解 |journal=建筑工程技术与设计 |date=2017-02 |issue=4 |page=287 |issn=2095-6630}}</ref><ref>{{cite web |title=浅草文化旅游信息中心 / Kengo Kuma & Associates |url=https://www.archdaily.cn/cn/755117/qian-cao-wen-hua-lu-you-xin-xi-zhong-xin |website=archdaily |accessdate=2018-11-06 |date=2014-10-15 |archive-url=https://web.archive.org/web/20180816054102/https://www.archdaily.cn/cn/755117/qian-cao-wen-hua-lu-you-xin-xi-zhong-xin |archive-date=2018-08-16 |dead-url=no }}</ref>

浅草文化观光中心一层设有日语、英语、韩语、汉语的旅游咨询台及[[钱币兑换店|钱币兑换店]]。二层设有旅游信息角。六层设有阶梯的多功能空间，可与七层的展示空间一起用于举办活动。顶层为免费开放的观景台。<ref>{{cite web |script-title=ja:浅草文化観光センター　施設概要 |url=http://www.city.taito.lg.jp/index/bunka_kanko/oyakudachi/kankocenter/a-tic-gaiyo.html |website={{lang|ja|台東区役所}} |accessdate=2018-11-06 |date=2017-04-17 |language=ja |archive-url=https://web.archive.org/web/20180929154802/http://www.city.taito.lg.jp/index/bunka_kanko/oyakudachi/kankocenter/a-tic-gaiyo.html |archive-date=2018-09-29 |dead-url=no }}</ref>

起初浅草文化观光中心的设计高度是41.25米。[[淺草寺|淺草寺]]和当地商店认为浅草文化观光中心建筑过高、与周边建筑不协调，提出反对意见。<ref>{{cite web |script-title=ja:陳情22-12　「浅草文化観光センター」改築計画設計変更を求めることについての陳情 |url=http://taito.gijiroku.com/voices/g07_ChinjoView.asp?SrchID=326&kword1=&kword2= |website={{lang|ja|台東区議会　議会文書管理システム}} |accessdate=2018-11-06 |language=ja |date=2010-12-07}}</ref>最终，建筑高度削减约2米，改为38.9米。<ref name="miyamoto">{{Cite book|author = {{lang|ja|宮元健次}}|year = 2013-11-02|script-title=ja:世界に誇れる東京のビル100|publisher = {{lang|ja|エクスナレッジ}}|ISBN = 978-4-7678-1671-5|pages = 60-61 |language=ja }}</ref>

2012年，浅草文化观光中心获得[[好設計獎|好設計獎]]。<ref>{{cite web |title=Good Design Award |url=http://www.g-mark.org/award/describe/39399 |publisher=Japan Institute of Design Promotion |accessdate=2018-11-06 |language=en |archive-url=https://web.archive.org/web/20130513032321/http://www.g-mark.org/award/describe/39399 |archive-date=2013-05-13 |dead-url=no }}</ref>
