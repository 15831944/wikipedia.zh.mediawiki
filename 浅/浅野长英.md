'''浅野长英'''{{BD|1979年|8月22日|||catIdx=Q}}中文名'''耿长军'''，出生于[[日本|日本]][[长崎县|长崎县]]，中日混血儿，毕业于[[上海戏剧学院|上海戏剧学院]]，为华人影视演员。

== 生平 ==
1979年，浅野长英出生在[[日本|日本]][[长崎县|长崎县]]，父亲是中国[[东北|东北]][[黑龙江省|黑龙江省]][[哈尔滨市|哈尔滨市]]人，母亲是日本长崎县人，2000年来到[[中国大陆|中国大陆]]，曾就读于[[上海戏剧学院|上海戏剧学院]]。<ref>{{cite web |url=http://www.fawan.com/Article/yl/zy/2012/12/16/114147179978.html |title=浅野长英：不想只演日本兵 |accessdate=2012年12月16日 |publisher= 法制晚报|archiveurl=http://www.fawan.com/Article/yl/zy/2012/12/16/114147179978.html |archivedate=2012年12月26日 |work= |language=zh-cn}}</ref><ref>{{cite web |url=http://paper.wenweipo.com/2009/11/15/EN0911150024.htm |title=從被嫌棄到受尊重 淺野長英見曙光 |accessdate=2009-11-15 |publisher= 香港文匪报|archiveurl=http://paper.wenweipo.com/2009/11/15/EN0911150024.htm |archivedate= 2012年12月26日|work= |language=zh-hk}}</ref>
