{{For|物理上的「量」|量 (物理)}}
'''量'''是非[[負数|負]][[實數|實數]]，更簡單來說是其[[長度|長度]]。

== 實數 ==
實數的量通常稱為'''[[絕對值|絕對值]]'''或'''模'''。它寫作 | ''x'' |，並以此定義：
: | ''x'' | = ''x'' , 若 ''x'' ≥ 0
: | ''x'' | = -''x'' , 若 ''x'' < 0

這給出在實[[數線|數線]]中從零開始的距離。例如-5的模就是|-5|=5。

== 複數 ==
相似地，複數的量稱為'''模'''，給出在[[阿根圖|阿根圖]]從零開始的距離。這條給出複數的模的公式和[[勾股定理|勾股定理]]一樣：

:<math> \left| x + iy \right| = \sqrt{x^2 + y^2 }</math>

例如-3 + 4i的模為5。

== 歐幾里得向量 ==
在[[歐幾里得空間|歐幾里得空間]]中，[[向量|向量]]的實數量'''''x'''''最常為[[歐幾里得範數|歐幾里得範數]]，這是由[[歐幾里得距離|歐幾里得距離]]引伸過來的：[[向量|向量]]自己的[[內積|內積]]的[[平方根|平方根]]：

:<math>||x||=\sqrt{u^2+v^2+w^2}</math>
在此''u''、''v''和''w''是分量（用'''''x'''''來作表記法亦可）。

例如，[4, 5, 6]的量為√(4<sup>2</sup> + 5<sup>2</sup> + 6<sup>2</sup>) = √77，即約8.775。

== 一般向量空間==
一般來說，量的概念可以應用到向量空間，稱為[[範數向量空間|範數向量空間]]。將物件對應到其量的函數稱為[[範數|範數]]。

== 應用 ==
量永遠非負。比較的大小時，使用[[對數|對數]]為尺度很有幫助。生活中的例子有[[聲音|聲音]]的[[音量|音量]]（分貝）和[[恆星|恆星]]的[[亮度|亮度]]。
