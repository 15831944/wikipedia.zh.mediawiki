'''蕹菜'''（[[學名|學名]]：{{lang|la|''Ipomoea aquatica''}}），又称'''空心菜'''、'''通菜'''、'''通心菜'''、'''壅菜'''、'''甕菜'''、'''應菜'''、'''藤菜'''、'''瓮菜'''及'''葛菜'''，为一年生[[草本植物|草本植物]]，[[茎|茎]]蔓性，中空，节上能生不定根。

== 別名 ==
應菜，蕹菜(南方草木狀、嘉祐本草、植物名實圖考)，空心菜(福建、廣西、貴州、四川)，通菜蓊、蓊菜(福建)，藤藤菜(江蘇、四川)，通菜(廣東)，甕蔡、無心菜、藤菜、竹葉菜、葛菜、草菜、蕻菜、蓊菜(因有不同區域之品種而有不同)。

== 臺灣分布 ==
台灣本島各地，生長於水田、溝渠、溪畔及沼澤地。

== 特徵 ==
其莖長{{convert|2|-|3|m|ft}}，中空。葉披針形，長{{convert|5|-|15|cm}}，寬{{convert|2|-|8|cm}}。開白色喇叭狀花，直徑{{convert|3|-|5|cm}}，可用扦插或播種的方式繁殖。<ref>{{cite web|url=http://www.curiousgardener.com/2011/09/15/the-new-kangkong-experiment/|title=Growing kangkong in water|work=curiousgardener.com}}</ref>

蕹菜原产[[東南亞|東南亞]]，但目前主要分布于[[中国|中国]][[长江|长江]]以南[[温热带|温热带]]地区。在水田、池沼种的称为「'''水蕹菜'''」，葉、藤管比旱地种植的大。

蕹菜的[[叶|叶]]呈[[心脏|心]]形，叶柄很长；夏秋季开白色或淡紫色喇叭狀[[花|花]]；种子淡褐或黑色；蕹菜性喜温暖湿润，生长势强，最大特点是耐涝耐炎热。在15℃—40℃条件下均能生长，耐连作。对土壤要求不严，适应性广，无论旱地水田，沟边地角都可栽植。夏季炎热高温仍能生长，但不耐寒，遇霜茎叶枯死，高温无霜地区可终年栽培。

蕹菜按照繁殖方式分为[[子菜|子菜]]和藤蕹两类。栽培方式依地势不同分为旱地栽培、水生栽培和浮生栽培（或称深水栽培）三种。

== 利用 ==
=== 食用蔬菜 ===
[[File:Ipomoea_aquatica_cooked.jpg|thumb]]
以嫩[[莖|莖]]、綠[[葉|葉]]來[[炒|炒]]食或[[煮|煮]][[湯|湯]]，作為[[夏天|夏]][[秋天|秋]]高溫季節的主要[[菜餚|菜餚]]之一。普通家庭或[[餐館|餐館]]炒蕹菜，常常以[[蒜頭|蒜頭]]或[[鹽|鹽]]和[[醬油|醬油]]進行調味，偶爾加入[[牛肉|牛肉]]。當中常見菜色包括盛行於[[廣東|粵]][[香港|港]][[澳門|澳]]地區的[[椒絲腐乳通菜|椒絲腐乳通菜]]以及流行於[[新加坡|星]][[馬來西亞|馬]]兩地的「馬來風光」（即[[辣椒|辣]]炒通心菜）。
