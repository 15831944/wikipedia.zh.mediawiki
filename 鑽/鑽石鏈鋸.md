'''鑽石鏈鋸'''（英語：Diamond Wire Saws）為[[鏈鋸|鏈鋸]]的一種，在鋼線上以分節的方式鑲嵌環狀的[[人工鑽石|人工鑽石]]，然後以輪具高速轉動，以切割硬物。

主要特色為幾乎不會產生震動，同時噪音產生相較其他器具小，且因為[[鑽石|鑽石]]堅硬，適用的範圍相當廣。

並分為「乾鏈式切割」（Dry wire saw cutting）<ref>{{cite web |title = 鑽石鏈切割(Diamond Wire Saw Cutting) - Dry| url = https://www.youtube.com/watch?v=wcVujBzoA_k}}</ref>以及「濕鏈式切割」（Wet wire saw cutting）<ref>{{cite web |title = 鑽石鏈切割(Diamond Wire Saw Cutting) - Wet| url = https://www.youtube.com/watch?v=gtORolPcWlE}}</ref>。

大多用來切割[[混泥土|混泥土]]等材質。

== 實際使用例子 ==

多用於切割混泥土，常被使用於房子的牆壁切割以及道路切割，道路切割則多用濕鏈式切割工法。

例如[[北門高架橋|北門高架橋]]的拆除作業上，為了防止[[台北府城北門|台北府城北門]]受到拆除作業的影響導致受損，則在北門附近的區塊使用了鑽石鏈鋸工法，也被記錄到了[[國家地理頻道|國家地理頻道]]的《透視內幕：台北舊城區復興運動》中<ref>{{cite web |title = 忠孝橋拆除過程全都錄！　國家地理頻道「紀錄北門」搶先看｜三立新聞網SETN.com
| url = https://www.youtube.com/watch?v=YtW_-stimSc}}</ref>。

以及捷運新生站的建造工程，亦有使用鑽石切割工法<ref>{{cite web |title = 無震動低噪音鑽石鏈鋸工法應用於營運介面連續壁破除| url = http://www.file.cloud.taipei.gov.tw/publication2015/265CC6C3-E430-4599-8B5C-C47D3C96DE74/chapter/%E6%8D%B7%E9%81%8B%E6%8A%80%E8%A1%93-16_018.pdf}}</ref>。
