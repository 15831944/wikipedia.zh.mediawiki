'''音频编解码器'''是指能[[編碼器|编码]]或[[解码|解码]]音频[[數碼|数字]]数据流的设备或[[计算机程序|计算机程序]]。<ref>{{Template:Cite web|url=https://www.microsoft.com/windows/windowsmedia/player/faq/codec.mspx|title=Using codecs|accessdate=2009-12-21|publisher=Microsoft}}</ref><ref>{{Template:Cite web|url=http://desktopvideo.about.com/od/glossary/g/codec.htm|title=About.com - Codec|accessdate=2009-12-21|publisher=About.com}}</ref><ref>{{Template:Cite web|url=http://www.afterdawn.com/glossary/terms/codec.cfm|title=Glossary - Codec|accessdate=2009-12-21|publisher=Afterdawn.com}}</ref><ref>{{Template:Cite web|url=https://help.ubuntu.com/8.04/musicvideophotos/C/codecs.html|title=Ubuntu Documentation - What is a codec?|accessdate=2009-12-21|publisher=Ubuntu Documentation Team}}</ref>

在软件层面，音频编解码器是一个执行[[算法|算法]]的计算机程序，能[[数据压缩|压缩与解压缩]]数字音频数据到[[音频文件格式|音频文件]]或[[流媒体|流媒体]]音频编码格式。该算法的目的是保证质量的前提下使用最少的[[比特|比特]]表示[[高保真|高保真]]音频信号。这可以有效地减少存储空间和传输已存储[[音频文件格式|音频文件]]所需的[[带宽|带宽]]。大多数[[编解码器|编解码器]]是实现为一个具有接口的[[函式庫|函式庫]]供一个或多个[[媒体播放器|媒体播放器]]使用。

在硬件层面，音频编解码器指一个能编码模拟音频到数字音频和解码数字音频到模拟音频的独立设备。换种说法，它包含运行在同样时钟的[[類比數位轉換器|模数转换器]]（ADC）和[[數位類比轉換器|数模转换器]]（DAC）。这在[[声卡|声卡]]中被使用以支持音频输入和输出。
