'''音街ウナ'''是动画公司MTK与音乐公司[[Internet_(公司)|Internet]]合作推出的虚拟歌手，同时具备[[Vocaloid|Vocaloid]]和[[AITalk|AITalk]][[语音合成|语音合成]]技术。

== 人物设定 ==
*性别：女
*年龄：11岁
*生日：7月30日（亦是发售日）
*身高：148.5厘米（连带头上的巨型鳗鱼帽）
*出生地：音乐之街
*名字：「ウナ」代表了其角色形象的创作理念来自于[[鳗鱼|鳗鱼]]<ref>{{Cite web |url = http://www.vocaloidnews.net/zh-hans/v家新人sugurspicy形象「音街鰻」亮相/ |title = V家新人Sugar&Spicy形象「音街鰻」亮相 - VNN |author = bajucy |date = 2016年7月1日 |publisher = VNN |language = zh-hans }}</ref>
*外貌：头戴蝌蚪型鳗鱼帽，留有蓝色的双马尾和一根呆毛，拥有蓝色的[[瞳孔|瞳孔]]和一对小虎牙。身穿以[[日本校服|日本校服]]为基础修改的服装。以下列出不同人设的差异：
**Sugar：整体风格为[[仪乐队|仪乐队]]员设计。发色到发尾变为白色。
**Spicy：整体风格为流行歌手设计。帽子上为长有一对黄色犄角、右眼下有黄色闪电状刺青的鳗鱼人面。发色到发尾变为绿混黄色，刘海上有黄色挑染。左脸上有音符状的刺青。
**Talk：整体风格为优等生设计。帽子上还戴有[[四方帽|博士帽]]。发色到发尾变为粉色，马尾上缠有蓝粉色的丝带。戴着红色框的眼镜。

=== 历史 ===
2016年6月15日，官方YouTube频道上传了一篇新产品的预告视频，当中公开了制作商，新声库的部分名称，形象剪影和发售日期等信息。<ref>{{Cite web |url = https://www.youtube.com/watch?v=Ed6u-CGRoPk |title = 2016.7.30 VOCALOID4新ライブラリー誕生！ - YouTube |author = 未知 |date = 2016年6月15日 |publisher = [[YouTube|YouTube]] |language = ja }}</ref>6月29日，官方频道上传了第二篇视频，当中公布了新产品信息，包括其完整立绘图像和原声提供者信息等。<ref>{{Cite web |url = https://www.youtube.com/watch?v=rogu99LRN4k& |title = 【VOCALOID4 新ライブラリ 音街ウナ】 Sugar & Spicy - YouTube |author = 未知 |date = 2016年6月29日 |publisher = [[YouTube|YouTube]] |language = ja }}</ref>'''7月30日'''，《VOCALOID4 Library 音街ウナ V4》下载版开始发售，实体版于8月4日发售。<ref>{{Cite web |url = http://www.vocaloidnews.net/zh-hans/vocaloid「音街鳗」于日本正式发售！/ |title = VOCALOID「音街鳗」于日本正式发售！ - VNN |author = Louis |date = 2017年3月8日 |publisher = VNN |language = zh-hans }}</ref>2017年3月8日，官方频道上传了第三篇视频，当中预告了即将发布的新产品和其形象剪影。<ref>{{Cite web |url = https://www.youtube.com/watch?v=jxWNTcYmrdM |title = 【音街ウナ】メンバー紹介 / Member introductions【otomachi una】 - YouTube |author = 未知 |date = 2017年3月8日 |publisher = [[YouTube|YouTube]] |language = ja }}</ref>3月14日，官方公布新产品《音街ウナTalk Ex》，并在官方网站上释放更多详细信息。<ref>{{Cite web |url = https://twitter.com/INTERNETjp/status/841489440467955712 |title = INTERNET Co., Ltd. 的 Twitter |author = [[Internet_(公司)|Internet]] |date = 2017年3月14日 |publisher = [[Twitter|Twitter]] |language = ja }}</ref>'''4月20日'''，《音街ウナTalk Ex》的下载版和实体版同步发售。
