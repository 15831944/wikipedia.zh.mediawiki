== 評選過程 ==
=== 初選過程 ===
第54屆金馬獎於2017年8月22日公佈初選名單，未獲特別獎青睞的[[豬哥亮|豬哥亮]]主演之《[[大釣哥|大釣哥]]》於第一關就被淘汰，[[劉德華|劉德華]]主演的《[[拆弹专家_(电影)|拆彈專家]]》雖然場面震撼，但也遺憾落榜。獲得香港電影金像獎及亞太影展雙料最佳女主角獎的《[[黃金花|黃金花]]》，亦未獲青睞。暑期國片《[[報告老師！怪怪怪怪物！|報告老師！怪怪怪怪物！]]》、《[[痴情男子漢_(電影)|痴情男子漢]]》與《[[紅衣小女孩_(電影)|紅衣小女孩2]]》皆確定通過初選，今年[[台北電影節|台北電影節]]的大贏家《[[大佛普拉斯|大佛普拉斯]]》，還有[[金城武|金城武]]主演的《[[喜欢·你|喜歡你]]》和《[[擺渡人_(2016年電影)|擺渡人]]》以及[[彭于晏|彭于晏]]的《[[悟空传|悟空傳]]》和《[[乘风破浪_(电影)|乘風破浪]]》都在入選名單之內。華語電影的女力亦不容小覷，包含[[紀培慧|紀培慧]]、[[陳湘琪|陳湘琪]]主演的《[[接線員_(電影)|接線員]]》，[[張艾嘉|張艾嘉]]自導自演的《[[相爱相亲|相愛相親]]》都通過初選，台北電影節影后[[許瑋甯|許瑋甯]]則以《紅衣小女孩2》和《[[目擊者_(2017年電影)|目擊者]]》2片通過初審力抗，9月在台上映的港片《[[殺破狼·貪狼|殺破狼貪狼]]》也過初選門檻。以8400萬台幣票房暫居該年國片票房冠軍的《[[吃吃的愛|吃吃的愛]]》也收到入選好消息，金馬執委會將在10月1日公布本屆金馬獎入圍名單。<ref>{{Cite web |url = https://news.tvbs.com.tw/entertainment/759984 |author = 劉苡青 |date = 2017年8月23日 |title = 金馬初選名單公布！豬哥亮、華仔無緣晉級 |publisher = [[TVBS新聞台|TVBS新聞台]]}}</ref>

=== 入圍過程 ===
10月1日公佈入圍名單，由[[臺灣電視台|台視]]現場LIVE直播記者會，主持人為[[楊千霈|楊千霈]]，上屆[[金馬獎最佳男配角|金馬獎最佳男配角]]得主[[林柏宏|林柏宏]]宣布提名名單，金馬執委會執行長[[聞天祥|聞天祥]]揭曉「年度台灣傑出電影工作者」得主。

聞天祥表示新導演總共有100多人報名角逐，其中《[[喜欢·你|喜歡你]]》[[許宏宇|許宏宇]]、《[[擺渡人_(2016年電影)|擺渡人]]》[[張嘉佳|張嘉佳]]以及《[[接線員_(電影)|接線員]]》[[盧謹明|盧謹明]]皆被點名為遺珠。在同樣競爭激烈的女配角獎中，[[許瑋甯|許瑋甯]]同時以《[[紅衣小女孩2|紅衣小女孩2]]》與《[[目擊者_(2017年電影)|目擊者]]》兩部作品角逐，而《紅衣小女孩2》另有[[高慧君|高慧君]]競爭，名單中多有自己人競爭的情形。除了高慧君外，其他遺珠有《[[阿莉芙|阿莉芙]]》[[趙逸嵐|趙逸嵐]]、《[[血觀音|血觀音]]》的[[吳可熙|吳可熙]]以及《[[刀背藏身|刀背藏身]]》[[許晴|許晴]]等，甚至以《相愛相親》入圍最佳導演、最佳劇情片、最佳女主角的[[張艾嘉|張艾嘉]]，也差點以《[[分貝人生|分貝人生]]》入圍該獎項。<ref name="死亡之組"/>

影帝的遺珠有《擺渡人》[[梁朝偉|梁朝偉]]、《[[大佛普拉斯|大佛普拉斯]]》[[陳竹昇|陳竹昇]]與《阿莉芙》[[舞炯恩|舞炯恩]]，及該屆以《阿莉芙》、《[[自畫像_(電影)|自畫像]]》、《[[上岸的魚|上岸的魚]]》和《[[川流之島|川流之島]]》4片角逐卻全數槓龜的[[鄭人碩|鄭人碩]]<ref>{{cite news|title=鄭人碩4片槓龜大遺珠|url=http://www.chinatimes.com/newspapers/20171002000426-260112|publisher=中國時報|date=2017-10-02|author=鍾志均|accessdate=2018-01-11|language=zh-hant}}</ref>。-{影后}-方面，上一屆的金馬獎-{影后}-[[周冬雨|周冬雨]]在《喜歡你》中以「很難得的強大生活力」角逐到最後，其他遺珠尚有《[[笨鳥|笨鳥]]》[[姚紅貴|姚紅貴]]與《[[路過未來|路過未來]]》[[杨子姗|杨子姗]]等。新人遺珠則有《大佛普拉斯》[[莊益增|莊益增]]、《阿莉芙》舞炯恩、《[[嘉年華_(電影)|嘉年華]]》[[周美君|周美君]]與《自畫像》[[張喬翔|Kiwebaby]]等。<ref name="死亡之組">{{Cite web |url = https://udn.com/news/plus/9732/2733849 |author = 童一寧 |date = 2017年10月1日 |title = 金馬女配角「死亡之組」同片廝殺或一人兩片競爭激烈 |publisher = [[聯合新聞網|聯合新聞網]]}}</ref>

=== 決選過程 ===
評審團主席[[吳念真|吳念真]]透露，在影帝之爭中，《[[老獸|老獸]]》[[涂們|涂們]]在第一輪投票便獲得過半數，《[[擺渡人_(2016年電影)|擺渡人]]》[[金城武|金城武]]則名列第2；最佳男配角《[[阿莉芙|阿莉芙]]》[[陳竹昇|陳竹昇]]同樣在首輪便獲得評審過半票數的共識；最佳新演員《[[強尼·凱克|強尼·凱克]]》[[瑞瑪席丹|瑞瑪席丹]]以自然、生動又不青澀的表現，打敗第二高票的《[[痴情男子漢|痴情男子漢]]》[[蔡凡熙|蔡凡熙]]。-{影后}-之爭經過4輪廝殺，在競爭最激烈的《[[血觀音|血觀音]]》[[惠英紅|惠英紅]]、《[[相愛相親|相愛相親]]》[[張艾嘉|張艾嘉]]與《[[川流之島|川流之島]]》[[尹馨|尹馨]]中，由惠英紅勝出，張艾嘉則位居第2名。被喻為「死亡之組」的最佳女配角獎，同樣是廝殺到第4輪，最終由《血觀音》[[文淇|文淇]]以9票驚險勝出以8票緊追在後的《相愛相親》[[吳彥姝|吳彥姝]]，最佳導演獎《[[嘉年華_(電影)|嘉年華]]》[[文晏|文晏]]也是以9比8的票數勝過《相愛相親》張艾嘉。新導演獎方面，《[[分貝人生|分貝人生]]》[[陳勝吉|陳勝吉]]則是一度緊追《[[大佛普拉斯|大佛普拉斯]]》[[黃信堯|黃信堯]]。
《相愛相親》提名七項皆落馬，但在項目最佳導演獎、最佳女主角跟最佳女配角三項大獎上皆是纏鬥到最後一輪飲恨敗北，只獲得一項伯爵年度優秀獎。<ref>{{Cite web|url=http://ent.ltn.com.tw/news/breakingnews/2264927|title=金馬》涂們擒影帝 吳念真曝評審內幕「金城武是第2」|publisher=[[自由時報|自由時報]]|author=陳立妍|date=2017-11-25|accessdate=2017-11-28}}</ref><ref>{{Cite web|url=https://stars.udn.com/star/story/10090/2840137|title=金馬54／最難產！張艾嘉、文晏纏鬥四輪才分高下|publisher=[[聯合報|聯合報]]|author=蘇詠智|date=2017-11-25|accessdate=2017-11-28}}</ref>
