《'''降B大調第四交響曲'''》，'''[[作品號|作品]]60'''，是[[路德維希·范·貝多芬|路德維希·范·貝多芬]]于1806年創作的四[[樂章|樂章]][[交響曲|交響曲]]。該作品篇幅不長，較爲輕巧優美，是同類作品中常被忽視的一部。[[舒曼|舒曼]]評價之為“兩座[[北歐神話|北歐]]大神之間的溫柔[[希臘神話|希臘]]少女”，以和前後的《[[第3號交響曲_(貝多芬)|第三交響曲“英雄”]]》和《[[第5號交響曲_(貝多芬)|第五交響曲“命運”]]》作對比。但該作品中亦不乏充沛有力的樂段。

== 背景 ==

=== 創作歷程 ===

[[File:Beethoven_3.jpg|thumb]]。]]

1806年，貝多芬在他當時的贊助人{{link-en|李赫諾夫斯基親王|Karl Alois Johann-Nepomuk Vinzenz, Fürst Lichnowsky}}家度暑假時，{{link-en|歐珀斯多夫伯爵|Franz von Oppersdorff}}來訪，後者是親王的親戚。伯爵聆聽了貝多芬的《[[第2號交響曲_(貝多芬)|第二交響曲]]》。他對之十分喜愛，並要求貝多芬給他創作一部新交響曲，答應給出可觀的稿費。<ref>Paul Netl (1976) ''Beethoven Handbook''. New York: Frederick Ungar Publishing Co., p. 262</ref>貝多芬於是暫停了《[[第5號交響曲_(貝多芬)|第五交響曲]]》的創作，在歡快的情緒下完成了該作品。貝多芬與同時期創作的作品除了上面提到的作品之外，還包括其《[[小提琴協奏曲_(貝多芬)|小提琴協奏曲]]》和歌劇《[[費德里奧|費德里奧]]》的草稿。<ref name="註釋4">{{cite web |url=http://www.bso.org/images/program_notes/beethoven_symphony4.pdf |title=存档副本 |accessdate=2010-04-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20100827223427/http://www.bso.org/images/program_notes/beethoven_symphony4.pdf |archivedate=2010-08-27 }}</ref>

=== 首演 ===

作品首演于1807年3月，[[洛布科维茨亲王|弗朗茨·約瑟夫·洛科維茨親王]]家舉辦的一場私人音樂會上。同時也首演了貝多芬的《[[科里奧蘭序曲|科里奧蘭序曲]]》和《[[第4號鋼琴協奏曲_(貝多芬)|第四鋼琴協奏曲]]》。第一場公衆首演則于1808年4月13日在[[維也納|維也納]]舉行，由貝多芬親自指揮。<ref name="註釋5">http://www.statemaster.com/encyclopedia/Symphony-No.-4-(Beethoven)</ref>

=== 題獻 ===

該作品即題獻給{{link-en|歐珀斯多夫伯爵|Franz von Oppersdorff}}伯爵，獻辭稱“獻給[[西里西亞|西里西亞]]貴族弗朗茨·馮·歐珀斯多夫伯爵”。<ref name="stein">Steinberg, Michael. "The Symphony: a listeners guide". p. 19-24. Oxford University Press, 1995.</ref>

== 結構 ==
該作品共四個樂章：
# 柔板—活潑的快板
# 柔板
# 活潑的快板
# 不太快的快板
全長約30至35分鐘。

首樂章（柔板—活潑的快板），二二拍，帶引子的[[奏鳴曲式|奏鳴曲式]]。引子暗示了該樂章的内容，從有著朦朧的神秘感的降b小調弱奏開始，先上升到B大調，再經過降G，由定音鼓和小號引回降B大調。同樣的引回法在從[[展開部|展開部]]到[[再現部|再現部]]的過程中也出現，定音鼓滾奏20小節之長。第一主題極其歡快，充滿活力和喜悅，木管樂器的旋律更加強了這一效果。在發展部中出現了弦樂器與管樂器的對話，一唱一和，旋律溫馨。再現部較爲短小，並直接引入花俏的尾聲。

 |description  = 斯基德摩學院管弦樂團演奏。<!--感謝[[Musopen|Musopen]]提供。-->
 |format      = [[Ogg|Ogg]]

第二樂章（柔板），[[降E大調|降E大調]]，四三拍，慢樂章，[[奏鳴曲式|奏鳴曲式]]。該樂章主要由[[小提琴|小提琴]]奏出平靜、如歌的第一主題，伴奏則有著放鬆但持續的律動感。第二個主題是[[單簧管|單簧管]]獨奏，有些憂鬱的情調；在整個樂隊重申此主題時，情感愈發強烈。法國作曲家[[埃克托·白遼士|埃克托·白遼士]]亦形容此樂章是「純潔如天使般，難以抗拒的柔情蜜意，然而奇妙的藝術在加工期間完全消失」。

第三樂章（活潑的快板），四三拍，[[諧謔曲|諧謔曲]]。該樂章是ABABA形式，即諧謔曲和[[三聲中部|三聲中部]]在出現第一次之後，又重復了一遍。貝多芬的《[[第7號交響曲_(貝多芬)|第七交響曲]]》中同樣運用了這一結構。全曲幽默而充滿能量，從開頭就可以聽出主題的元素。三聲中段效果則仿傚一個業餘管樂隊演奏時，被一眾醉醺醺的小提琴手打擾之情況。在諧謔曲第三次出現時，由圓號強烈地切斷。

末樂章（不太快的快板），四二拍，[[奏鳴曲式|奏鳴曲式]]。該樂章雖然速度標記不算極快，但多[[十六分音符|十六分音符]]，聽起來依然迅猛生動。再現部之前有一段巴松管獨奏的柔和樂段，但隨即又進入了急促的尾聲。尾聲規模較大，變出各種花樣，直到快活地結束。[[卡爾·馬利亞·馮·韋伯|卡爾·馬利亞·馮·韋伯]]並不喜歡這一交響曲，他評論說：“低音提琴樂手一定要抱怨，‘作曲家竟讓我像野山羊一樣瘋瘋癲癲。’”<ref name="註釋4" />

== 評價與解讀 ==
該作品是貝多芬交響曲中較爲次要的一部，與《[[第1號交響曲_(貝多芬)|第一]]》和《[[第8號交響曲_(貝多芬)|第八]]》都經常被忽視。

對該作品最著名的評價，莫過於[[羅伯特·舒曼|羅伯特·舒曼]]的比喻，稱之被夾在《[[第3號交響曲_(貝多芬)|第三交響曲“英雄”]]》和《[[第5號交響曲_(貝多芬)|第五交響曲“命運”]]》，如同“兩座[[北歐神話|北歐]]大神之間的溫柔[[希臘神話|希臘]]少女”。法國作家[[羅曼·羅蘭|羅曼·羅蘭]]也曾稱此作品「保存了他人生中最明朗日子之芬芳」。這種評價有其中肯之處，因爲該作品相對而言較爲柔美、清新，沒有悲傷的動機和宏大的建構；但實際上該作品樂章的主題大多都音量大、情緒激烈，有著“雄性”的感覺。即使是慢樂章中也有貝多芬式的強有力特強樂段。因此，《第四交響曲》綜合了如歌的、優美的與狂歡的、衝動的兩种元素。<ref name="註釋5" />

貝多芬在該作品中回歸了[[古典樂派|古典樂派]]在首樂章前加緩慢引子的傳統，但也有著形式上的革新，賦予了它更深的内容。<ref>{{cite web |url=http://www.oldandsold.com/articles06/sy13.shtml |title=存档副本 |accessdate=2010-04-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20101117024514/http://www.oldandsold.com/articles06/sy13.shtml |archivedate=2010-11-17 }}</ref>該作品的調性，降B大調，在貝多芬的作品中有著放鬆、溫暖的意味。<ref name="註釋4" />

== 影響 ==
* 貝多芬的《[[第7號交響曲_(貝多芬)|第七交響曲]]》[[諧謔曲|諧謔曲]]與該作品第三樂章結構類似。
* [[門德爾松|門德爾松]]將該作品作爲其在[[萊比錫|萊比錫]][[布商大廈音樂廳|布商大廈音樂廳]]首次音樂會的曲目。
