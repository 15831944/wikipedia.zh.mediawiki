1940年至1945年的大奖赛发生在[[第二次世界大战|第二次世界大战]]期间，因此比赛的数量非常少。这些年里没有组织锦标赛。<ref>{{cite web| title = Grand Prix Winners 1895–1949 : Part 3 (1934–1949)| work = The Golden Era of Grand Prix Racing| url = http://www.kolumbus.fi/leif.snellman/gpw3.htm| first = Hans| last = Etzrodt| accessdate = 2007-08-05}}</ref><ref>{{cite web| title = 1940–1941| work = The Golden Era of Grand Prix Racing| url = http://www.kolumbus.fi/leif.snellman/gp40.htm| author =  Leif Snellman| accessdate = 2007-08-05}}</ref>这一时期的大奖赛主要在[[美国|美国]]进行。战争结束一周后，1945年大奖赛在[[巴黎|巴黎]]举行。<ref>{{cite web| title = 1940 Grands Prix| work = The GEL Motorsport Information Page| url = http://www.teamdan.com/archive/gen/1940.html| first = Darren| last = Galpin| accessdate = 2007-08-05 }}</ref><ref>{{cite web| title = 1941 Grands Prix| work = The GEL Motorsport Information Page| url = http://www.teamdan.com/archive/gen/1941.html| first = Darren| last = Galpin| accessdate = 2007-08-05 }}</ref>
