'''第44屆日本眾議院議員總選舉'''，在2005年9月11日进行，选出480名[[日本众议院|日本众议院]][[议员|议员]]。

== 背景 ==
[[日本內閣總理大臣|日本內閣總理大臣]]（首相）[[小泉纯一郎|小泉純一郎]]自2001年上台以後，推行小泉改革。在[[2003年日本大選|2003年日本大選]]中，面對著與在野的[[自由黨_(日本1998年)|自由黨]]合併的[[民主黨_(日本)|民主黨]]強大威脅下，[[自由民主黨_(日本)|自民黨]]保住了執政權。但在翌年的參議院選舉中，自民黨卻以一席之差落後在民主黨之後。小泉被迫改組內閣，與舊勢力妥協。

小泉改革的[[日本郵政民營化|日本郵政民營化]]法案，在眾議院以些微多數通過，但在參議院舉步維艱。8月8日下午，参议院举行全体会议，以125票反对、108票赞成否决了[[邮政民营化相关法案|邮政民营化相关法案]]。为挽回局面，小泉纯一郎决定解散众议院，于9月11日举行大选；他在參議院表決前曾警告，如參議院否決法案，他將解散眾議院。

8月17日，前众议院议长[[绵贯民辅|绵贯民辅]]在[[东京都|东京都]]举行[[记者招待会|记者招待会]]，宣布脱离自民党，并成立以他为首的[[國民新黨|国民新党]]参加众议院选举。小泉纯一郎当晚在回答记者提问时表示，他不会和国民新党进行联合。

8月21日，反对“邮政民营化改革相关法案”的自民党前众议院议员[[小林兴起|小林兴起]]等4人和[[长野县知事|长野县知事]]{{tsl|en|田中康夫||田中康夫}}在东京都宣布组建新的政党“[[新黨日本|新党日本]]”，新党代表由田中康夫担任。

這次解散國會，稱為[[郵政解散|郵政解散]]。

=== 內閣紛擾 ===
小泉純一郎解散國會的舉動，[[自由民主黨_(日本)|自民黨]]內部認為，由於今次[[反郵政派|反郵政派]]一定會脫離自民黨參選，在以往的分裂選舉中，自民黨勝出的次數其實不多。所以在決定解散國會的臨時內閣會議中有大臣因反對解散而不願意在解散詔書中副署（解散國會的詔書一定要有首相以及內閣各國務大臣的副署才能行效）。

反對的大臣包括[[總務大臣|總務大臣]][[麻生太郎|麻生太郎]]、[[經濟產業大臣|經濟產業大臣]][[中川昭一|中川昭一]]以及[[農林水產大臣|農林水產大臣]][[島村宜伸|島村宜伸]]。臨時內閣會議因而中斷超過兩小時。最後麻生太郎及中川昭一被小泉勸服副署，但由於島村宜伸仍然拒絕副署被小泉罷免，小泉自行兼任[[農林水產大臣|農林水產大臣]]（後來由副大臣岩永峰一繼任）。同時在臨時內閣會議中，亦決定罷免在參議院會議中對郵政民營化法案投反對票的防衛政務官柏村武昭。

[[小泉纯一郎|小泉純一郎]]在記者會上說：“今次的解散是「郵政解散」，贊成反對與否直接訴諸國民”他並表示，如果自民黨與公明黨的自公聯盟在國會選舉中得不到過半他會下台。亦有人把這此解散稱為「自爆襲擊解散」（因為國會選舉日期於9月11日，在2001年在美國受到恐怖襲擊。）{{Fact|time=2007-08-11T11:23:53Z}}

== 刺客 ==
對於在眾議院投票反對郵政民營化法案的37名議員，自民黨黨部宣佈不會提名他們參選，並在他們的選舉提名對立的候選人。當時由於被認為分裂選舉自民黨必定大敗，故坊間對自民黨這一些有意針對的行為作出責難，被稱為刺客。

對於在眾議院對郵政民營化法案投棄權票的議員，自民黨要他們簽訂誓言，在當選後對再次提出的郵政民營化法案投贊成票，這些人才能得到自民黨的提名。

對於在眾議院投票反對郵政民營化法案的37名議員，這一批議員不會再獲得自民黨的提名，所以只有以下的選擇：

#被新成立的新政黨提名
#以自民黨黨員身份，受到自民黨地方組織的支持參選
#離開自民黨以無黨籍參選
#不參選

[[自由民主党_(日本)|自民黨]]的地方組織，如[[東京都|東京都]]支部決定支援黨所提名的候選人，而[[岐阜縣|岐阜縣]]支部則對個別提名投反對票的候選人。

在這次選舉中，自民黨提名不少女性刺客，

離開了自民黨的[[龜井靜香|龜井靜香]]最先評這些針對的候選人為刺客，其中一名刺客[[小池百合子|小池百合子]]被人稱為「自民黨的[[上戶彩|上戶彩]]（因為上戶彩在某電影中飾演女刺客）。8月28日、自民黨決定使用刺客的名稱。{{Fact|time=2007-08-11T11:23:53Z}}

== 各党竞选纲领 ==

最大的在野党[[民主黨_(日本)|民主黨]]8月16日在公布的竞选大纲中说，如果该党在大选中获胜，将在12月前撤出驻扎在[[伊拉克|伊拉克]]的所有[[日本自卫队|日本自卫队]]人员，并将改善同[[亚洲|亚洲]]邻国的关系。

自民党8月19日发表多达120条目的选举纲领，以通过邮政民营化作为政治改革首要目标。

日本首相小泉纯一郎8月29日暗示，如果他能继续担任首相，将会不顾[[中华人民共和国|中]][[韩国|韩]]等国的反对，前往参拜[[靖国神社|靖国神社]]。

== 民调 ==

日本《[[读卖新闻|读卖新闻]]》8月27日公布的最新民意调查显示，大约38%的选民表示将支持自民党的众议员候选人，有16.3%的选民决定投票支持[[日本社民党|日本社民党]]候选人。

日本《[[东京新闻|东京新闻]]》8月30日公布的一份民调结果显示，日本首相小泉纯一郎所领导的自民党支持率为最大在野党民主党的近两倍。

《[[日本经济新闻|日本经济新闻]]》9月9日在其网站上公布的最新民调结果显示，大约有52%的选民表示将投票支持小泉领导的自民党，而投票支持民主党的选民则只有34%。

9月11日，据[[共同通訊社|共同社]]实施的投票所[[出口民調|出口民調]]结果显示，执政党在众院选中很可能获得半数以上议席，小泉将继续掌握政权。

== 大選結果 ==
[[File:Japanese_Election_2005.png|thumb]]

本來今次大選，對民主黨來說是天跌下來的禮物。就像1993年的選舉中，自民黨內部分裂，以數十席之差變成在野黨。

但民主黨認為這是自民黨的內鬥，不與自民黨反郵改派合流，在每一選區派出候選人。結果有的選區變成自民黨VS民主黨VS反郵政派的局面，給自民黨與反郵政派把餅做大。

而小泉把反對郵政改革的打成反改革派，眾在野黨跌入了自民黨單一政策的陷阱。而且，小泉在反郵改派的議員參選的選區中派出高知名度的刺客來狙擊這些人。

結果在小泉效應下，自民黨除了在地方選區大勝以外，還在民主黨上次大有斬獲的比例代表選區壓勝。令自民黨獲得十五年來罕見高席次。

=== 总计 ===
投票率：小選舉區67.51%（上次選舉為59.86%）、比例代表為67.46%（上次選舉為59.81%）。

開票結果：
*執政聯盟得到327席（自民黨296席・公明黨31席）
*在野黨：民主黨得到113席比選舉前177席大幅滑落而慘敗。
*日本共產黨與社會民主黨保持現有議席。
*除了離開自民黨的反郵政派以外，無黨籍當選2席。
*反郵政派當選18人（無黨籍者13人）
