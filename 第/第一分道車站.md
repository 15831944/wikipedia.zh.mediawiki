== 車站構造 ==
石建站房，牆壁上都長滿了[[青苔|青苔]]，站房中有一戶住家。

== 利用狀況 ==
*阿里山線（主線）的車站。
*因受[[八八風災|八八風災]]影響，曾於2008.8~2017.7期間停駛。
*2017年7月奮起湖～十字路及神木～第一分道路段復駛，第一分道車站復站

== 歷史 ==
