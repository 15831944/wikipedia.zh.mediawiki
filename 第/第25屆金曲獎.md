'''第25屆[[金曲獎|金曲獎]]'''，亦稱「金曲25」<ref>[https://www.youtube.com/channel/UCRlH_rhJ3175va88JW7mzSg YouTube官方頻道]</ref>，由[[文化部影視及流行音樂產業局|文化部影視及流行音樂產業局]]與[[國立傳統藝術中心|國立傳統藝術中心]]共同主辦，[[台北市政府文化局|台北市政府文化局]]協辦，[[臺灣電視公司|臺灣電視公司]]、[[超級電視台|超級電視台]]和[[台灣唱片出版事業基金會|台灣唱片出版事業基金會]]共同承辦。參賽作品限於2013年以[[臺灣地區|臺灣地區]]為首次發行地區之鐳射唱片（[[CD|CD]]）或是正式公開的戲曲表演<ref>{{Cite web |url=http://www.bamid.gov.tw/bamid/Code/NewListContent.aspx?id=db244907-2ed2-4e34-9cc3-4ffd6e226a15 |title=第25屆金曲獎報名開始，歡迎音樂界踴躍報名！ |publisher=文化部影視及流行音樂產業局 |date=2013-12-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20141017063441/http://www.bamid.gov.tw/bamid/Code/NewListContent.aspx?id=db244907-2ed2-4e34-9cc3-4ffd6e226a15 |archivedate=2014-10-17 }}</ref><ref>{{Cite web |url=http://www.ncfta.gov.tw/ncfta_ce/c05/c05020510.aspx?E=SWRlbnRpdHlJRD04MzU= |title=第25屆傳藝金曲獎開始報名 歡迎各界踴躍推薦與報名參賽！ |publisher=國立傳統藝術中心 |date=2014-01-29}}</ref>。

本屆流行音樂類獎項評審團總召集人為倪重華<ref>{{Cite web |author=林士傑|url=http://www.nownews.com/n/2014/05/21/1243085|title=金曲獎／台音樂退步？　總召倪重華：名單多元更接軌世界 |date=2014-05-21|publisher=NOWnews}}</ref>。2014年5月20日公布入圍名單，6月28日公布得獎名單<ref>[http://www.cna.com.tw/news/firstnews/201405205007-1.aspx 第25屆金曲獎入圍名單]，中央社，2014年5月20日</ref>。頒獎典禮由[[臺灣電視公司|台視]]主辦，亦打算創下全數頒獎者僅為一人之先例，希望將典禮重點回歸到得獎者身上<ref>{{Cite web |author=林士傑|url=http://www.nownews.com/n/2014/05/19/1240512|title=金曲獎／主持典禮拚不超時　哈林宣布：頒獎者全一位 |date=2014-05-19 |publisher=NOWnews}}</ref><ref name="ttv">{{cite web|url=http://www.ttv.com.tw/GoldenMelody25/default.asp|title=第25屆金曲獎頒獎典禮|publisher=台視|accessdate=2015-04-02}}</ref>。典禮主持人為[[庾澄慶|庾澄慶]]，[[星光大道|星光大道]]主持人為[[楊千霈|楊千霈]]及[[區耕祥|區耕祥]]（DJ Dennis），[[黃子佼|黃子佼]]則擔任《金曲之夜》電視特輯節目主持人。本屆金曲獎開始，「入圍影片」、「舞台」視覺上突破以往的傳統方式，讓金曲獎更加好看，更是讓設計師們發揮的平台。

傳統暨藝術音樂類獎項自本屆起移由[[國立傳統藝術中心|國立傳統藝術中心]]專責辦理，原本的出版類17個獎項重新調整為出版類、戲曲表演類共13個獎項，總獎金也從原本的[[新臺幣|新臺幣]]215萬元提高到新臺幣250萬元，其中最佳年度演出獎獎金達新臺幣50萬元<ref>{{Cite web |url=http://www.chinatimes.com/newspapers/20140305000545-260115 |title=傳藝金曲獎拒流行 新增3類獎項 |author=管婺媛 |publisher=中國時報 |date=2014-03-05}}</ref>。葉綠娜、[[柯基良|柯基良]]分別擔任出版類和戲曲表演類評審團總召集人<ref name=
"文化部0630">{{cite web|url=http://www.moc.gov.tw/artnews.do?method=findById&id=1222852536388|title=第25屆傳藝金曲獎入圍名單公佈記者會|publisher=文化部|date=2014-06-30|deadurl=yes|archiveurl=https://web.archive.org/web/20150403121730/http://www.moc.gov.tw/artnews.do?method=findById&id=1222852536388|archivedate=2015-04-03}}</ref>。2014年6月30日公布入圍名單，8月9日於[[中山堂_(臺北市)|中山堂]]舉行頒獎典禮，由[[超級電視台|超視]]直播，[[曾寶儀|曾寶儀]]、[[李小平|李小平]]擔任頒獎典禮主持。

== 流行音樂類 ==
=== 系列活動 ===
[[金曲國際音樂節|金曲國際音樂節]]於2014年6月20日至6月29日在[[華山1914文化創意產業園區|華山1914文化創意產業園區]]舉行，活動項目包含金曲25特展、大師講堂、國際論壇、商展交易與媒合中心、工作坊及金曲@Live House 等系列活動，呈現金曲獎25年的成長脈絡與軌跡，探討音樂產業的未來發展<ref>{{cite web|url=http://www.ttv.com.tw/GoldenMelody25/MusicFestival/default.asp|title=2014金曲國際音樂節|publisher=台視|accessdate=2015-04-02}}</ref>。

=== 報名資格與統計 ===
本屆流行音樂類計有171家報名單位、398張專輯、11,982件作品參賽，包括「最佳音樂錄影帶獎」623件及「最佳專輯包裝獎」302件<ref>{{cite web|url=http://www.moc.gov.tw/artnews.do?method=findById&id=1222852536209|title=第25屆流行音樂類金曲獎入圍名單揭曉！|publisher=文化部|date=2014-05-20|deadurl=yes|archiveurl=https://web.archive.org/web/20150403134221/http://www.moc.gov.tw/artnews.do?method=findById&id=1222852536209|archivedate=2015-04-03}}</ref>。由81人次評審委員經初審及複審會議後，決議本年入圍名單，計有123件作品入圍角逐24個獎項。本年入圍名單中，報名單位以[[滾石國際音樂股份有限公司|滾石國際音樂股份有限公司]]入圍8項表現最亮眼，專輯作品則以[[李榮浩|李榮浩]]《模特》及[[李健_(歌手)|李健]]《拾光》專輯入圍5項奪冠。

=== 頒獎典禮流程 ===
本屆頒獎典禮主題為「金曲25」，流行音樂類金曲獎暨音樂節目幕後人員由計畫主持人[[劉麗惠_(台視副總)|劉麗惠]]和張耕宇、典禮總策劃陳鎮川、音樂顧問[[陳子鴻|陳子鴻]]、音樂總監[[洪敬堯|洪敬堯]]組成，[[奧斯卡獎|奧斯卡獎]]和[[葛萊美獎|葛萊美獎]]國際成音大師Laurence H. Estrin則受邀協助規劃典禮<ref name="ttv"/>。

2014年6月28日星期六晚間5時在[[臺北小巨蛋|臺北小巨蛋]]舉行星光大道，主持人為[[楊千霈|楊千霈]]及[[區耕祥|區耕祥]]（DJ Dennis）。晚間7時舉行頒獎典禮，主持人為[[庾澄慶|庾澄慶]]，由[[台視|台視]]轉播。[[中華電信MOD|中華電信MOD]]、Hichannel、[[愛奇藝|愛奇藝]]、日本[[Rakuten|樂天]]VIKI、香港TVB[[J2|J2]]等也於網路及各地區轉播。

==== 直播收視率 ====
本屆流行音樂類金曲獎頒獎典禮由台視轉播，平均收視率5.98創歷史新高、總收視人口為511萬3000人<ref>{{cite news|author=王靖怡|title=金曲獎收視創新高 江蕙最吸睛|url=http://www.cna.com.tw/news/firstnews/201406305004-1.aspx|agency=中央社|date=2014-06-30}}</ref>。
*最高點：[[江蕙|江蕙]]頒發「最佳台語專輯獎」時，收視率為8.27。
*第二高點：[[江蕙|江蕙]]和[[庾澄慶|庾澄慶]]即將宣佈「最佳台語專輯獎」時，收視率有8.04。
*第三高點：[[江蕙|江蕙]]揭曉「最佳台語女歌手獎」為[[黃乙玲|黃乙玲]]時，以及[[蕭敬騰|蕭敬騰]]介紹「最佳國語男歌手獎」入圍名單時，收視率皆達7.94。
*第五高點：[[蔡依林|蔡依林]]表演「[[大藝術家_(歌曲)|大藝術家]]」時，以及[[蕭敬騰|蕭敬騰]]公佈「最佳國語男歌手獎」為[[林俊傑|林俊傑]]時，收視率皆達7.93。

=== 頒獎典禮流程 ===
2014年8月9日下午3點至6點於臺北市[[中山堂_(臺北市)|中山堂]]廣場舉行音樂嘉年華，共安排六段表演節目<ref>{{cite web|url=http://www.ncfta.gov.tw/events/Final/page4.html|title=頒獎典禮|work=第25屆傳藝金曲獎頒獎典禮|accessdate=2015-04-02|deadurl=yes|archiveurl=https://web.archive.org/web/20150403090641/http://www.ncfta.gov.tw/events/Final/page4.html|archivedate=2015-04-03}}</ref><ref>{{cite web|url=http://www.ncfta.gov.tw/events/Final/page5.html|title=表演節目|work=第25屆傳藝金曲獎頒獎典禮|accessdate=2015-04-02|deadurl=yes|archiveurl=https://web.archive.org/web/20150403114901/http://www.ncfta.gov.tw/events/Final/page5.html|archivedate=2015-04-03}}</ref>。當日7點起在[[中山堂_(臺北市)|中山堂]]中正廳舉行本屆傳藝金曲獎頒獎典禮，[[曾寶儀|曾寶儀]]、[[李小平|李小平]]擔任主持。[[王嘉明_(台灣)|王嘉明]]擔任頒獎典禮策展人，典禮共穿插六段演出。
