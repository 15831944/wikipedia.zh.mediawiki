《'''第七軍團：最後戰役'''》（{{lang-en|'''''Last Knights'''''}}）是一部2015年的動作電影，由[[紀里谷和明|紀里谷和明]]執導，Michael Konyves和Dove Sussman編劇，改編自[[元祿赤穗事件|47名赤穗浪士]]的傳說<ref>{{Cite web |url = https://nypost.com/2015/04/01/freeman-owen-casualties-of-bloody-bad-last-knights/ |title = Freeman, Owen casualties of bloody bad ‘Last Knights’ |author = Sara Stewart |date = 2015-04-01 |publisher = New York Post |language = en}}</ref><ref>{{Cite web |url = https://www.tbnweekly.com/diversions/movie_news/article_6e7f7c48-cac4-550d-9fe0-9bac37331bde.html |title = Kazuaki Kiriya's Last Knights falls short of expectations |author = Lee Clark Zumpe |date = 2015-04-05 |publisher = Tampa Bay Newspapers |language = en}}</ref>。主演是[[佳夫·奧雲|佳夫·奧雲]]和[[摩根·費曼|摩根·費曼]]。本片故事集中在一群戰士身上，他們在腐敗的部長掌控中為失去他們的主人而伺機報復。<ref>{{Cite web |url = https://www.screendaily.com/home/afm/clive-owen-morgan-freeman-to-star-in-the-last-knights/5048358.article |title = Clive Owen, Morgan Freeman to star in The Last Knights |author = Jeremy Kay |date = 2012-10-31 |publisher = Screen International |language = en}}</ref>

本片在片尾有紀念演員[[戴夫·雷傑諾|戴夫·雷傑諾]]，他在2014年去世。
