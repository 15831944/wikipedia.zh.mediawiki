'''第一次印度支那戰爭'''（{{lang-fr|'''Guerre d'Indochine'''}}），或稱'''法越戰爭'''，[[越南|越南]]稱'''東洋戰爭'''（{{vie |hn = 戰爭東洋 |qn = Chiến tranh Đông Dương}}）、'''反法抗戰'''（{{vie |hn = 抗戰𢶢法 |qn = Kháng chiến chống Pháp}}），是1945年9月至1954年7月间[[越南獨立同盟會|越南獨立同盟會]]为使[[越南|越南]]独立与[[法國|法國]]进行的一場[[戰爭|戰爭]]，战争以[[越南獨立同盟會|越南獨立同盟會]]胜利，法國被迫簽訂[[日内瓦会议_(1954年)|日内瓦协定]]告终。

== 战争起因 ==
1883年，[[中南半島|中南半島]]（即印度支那）成為[[法蘭西殖民地|法屬殖民地]]。[[二戰|二戰]]期間（1940-1945），[[日軍入侵法屬印度支那|大日本帝國佔領中南半島]]，1945年8月15日，[[日本投降|日本投降]]。1945年8月18日，[[中华民国国民革命军|中华民国国民革命军]]和[[英国军队|英国军队]]以北纬16度线为界，分别接管了越南的北部与南部。9月2日[[胡志明|胡志明]]在[[河内|河内]]发表《[[越南独立宣言|独立宣言]]》，宣布越南从日本手中[[独立|独立]]，废除[[君主制|君主制]]、与[[法国|法国]]脱离关系，建立[[越南民主共和国|越南民主共和国]]<ref name="独立宣言">{{citation |author = [[胡志明|胡志明]]  |chapter = 《[[越南独立宣言|独立宣言]]》 |date = 1945年9月2日 |title = 《胡志明选集》 |volume = 第二卷 |publisher = 越南外文出版社 |page = 第3页 |quote = 事实上从1940年秋起，我国已经成了日本而不是法国的殖民地了。而且在日本投降同盟国时，我们全国人民已经起来夺取了政权，建立了越南民主共和国。事实上，我国人民是从日寇的手里而不是从法国殖民主义者的手里夺回政权的。法国人逃跑、日本投降，保大皇帝逊位，我们人民已经粉碎了近百年来的殖民枷锁而建立了独立的越南，我们的人民还推翻了几十个世纪以来的君主制度，从而缔造了民主共和制度。因此，我们──代表越南全体人民的新越南临时政府──宣布完全同法国脱离关系，废除法国与越南签订的一切条约，取消法国在越南的一切特权。}}</ref>。

1945年6月15日戴高乐任命亲信[[菲利普·勒克萊爾|菲利普·勒克萊爾]]中将为法国远东军总司令。8月17日任命海军中将[[达尚礼|达尚礼]]为印度支那高级专员，同时任命[[菲利普·勒克萊爾|菲利普·勒克萊爾]]中将为印度支那最高军事司令官。8月18日,勒克莱尔离开法国麾军远东。

1945年8月19、23、25日，[[越盟|越盟]]发动了越南总起义（即“[[八月革命|八月革命]]”）。8月25日越盟占领了西贡，建立了自己的权力机关。8月27日组成[[越南民主共和国|越南民主共和国]]临时政府，9月2日宣告越南独立及越南民主共和国成立。

根据1943年[[开罗会议|开罗会议]]期间[[蒋介石|蒋介石]]与[[美國總統|美國總統]][[小罗斯福|小罗斯福]]之间达成的有关托管印度支那的共识，以及盟军最高统帅部有关印度支那受降令，蒋介石命陆军第一方面军司令[[卢汉|卢汉]]统部进入越南，在河内设占领军司令部，接受[[日本投降|日本投降]]。陆军总部为受降规定:“一、接受日军的投降,解除武装,遣送回日;二、组织军政府管理民政;三、驻云南的法越军在原地待命,不准入越,如有个人志愿返回越南者,必须解除武装。”<ref>陈修和:《抗战胜利后国民党军入越受降纪略》,载《文史资料选辑》第七辑,第16-17页。</ref>8月31日，卢汉通知驻越的日军第38军团军团长土桥勇逸派代表并携带驻越日军人员及驻越16度纬线以北所有日军人员、武器弹药及其它装备的名录清单前来云南[[蒙自|蒙自]]洽降。9月1日，土桥勇逸派出第38军团参谋长酒井千城和随员、译员共5人，乘日机飞抵云南蒙自，改乘中国飞机往[[开远|开远]]。9月2日在开远的中国第一方面军司令部礼堂内举行了洽降仪式。随即第一方面军副参谋长尹继勋随机抵达河内，设立前进指挥所，商洽与筹备入越受降事宜。国民政府从外交部、财政部、军政部、交通部、粮食部等六部各派出一名代表，组成受降顾问团。[[卢汉|卢汉]]鉴于越南人仇法情绪激昂，为了防止发生越法冲突乃至中越冲突，卢汉下令阻止[[三九事件|三九事件]]后逃入云南避难的法国官兵返回越南，解除从日军集中营释放出来的法国官兵的武装，禁止街巷及公共场所悬挂法国国旗。<ref>陈修和:《抗战胜利后国民党军入越受降纪略》,载《文史资料选辑》第七辑,第23页。</ref>坚持长期占领越南、托管越南和扶助越南走向独立的原则。<ref>罗敏:《中国关于战后越南问题的认知与实践(1942-1946)》,《历史研究》2003年第5期,第78页。</ref>受降部队总兵力约20万人：
* 第六十军[[曾泽生|曾泽生]]部
* 第九十三军[[卢浚泉|卢浚泉]]部
* 暂编十九师[[龙绳增|龙绳增]]部
* 暂编二十三师[[潘朔端|潘朔端]]部
* 第六十二军[[黄涛|黄涛]]部
* 第五十二军[[赵公武|赵公武]]部
* 第五十三军[[周福成|周福成]]部
* 第九十三师[[吕国权|吕国权]]部
* 荣誉第一师[[戴坚|戴坚]]部

1945年8月24日，杜鲁门在会晤戴高乐时，明确表示:“关于印度支那问题,美国政府对于法国重返印度支那一事决不反对”。<ref>戴高乐:《战争回忆录》第三卷《拯救(1944-1946)》,第160、565页。</ref>这是美国改变了罗斯福总统的政策，首次允许法国恢复在印度支那的殖民统治。

格雷西（Gracey）将军统帅的英国军队在9月10日进入越南南部，他们的“使命”是按照波茨坦公告的约定，接受驻越十六度线以南的日军投降。格雷西立即关闭了越南的新闻媒体，宣布国家进入紧急状态，还施行了宵禁。禁止集会结社，禁止持有武器。9月22日，英国人重新武装了被日军关押的法国殖民军。法军立即开始在西贡逮捕、杀害越南独立份子，占领了大部分的警察局、中央银行和市政府大楼，几乎没有遇到激烈抵抗。1945年9月23日，在[[美國|美]]、[[英國|英]]支持下，[[法國|法國]]派遣遠征軍接管越南南部；1946年1月[[法国|法国]]的接管完成。

1945年9月14日卢汉抵达河内。9月27日，受降顾问团外交代表[[凌其翰|凌其翰]]在昆明接到蒋介石电话指示，转告[[卢汉|卢汉]]三点指示:<ref>凌其翰:《我的外交官生涯———凌其翰回忆录》,第75-78页。</ref>
# 牢牢掌握[[老街|老街]]-河内-[[海防|海防]]铁路运输线
# 对越南当局采取不管态度
# 让法军开入越南,不得予以阻挡
蒋介石随即与卢汉通电下达五项补充指示：
# 对法越纠纷严守中立 
# 越军进入我防区即须解除武装 
# 对越党不干涉 
# 对越党亦不警戒 
# 对法人生命财产尽力保护 
9月28日,卢汉按计划在原法越总督府举行接受日本投降仪式。出席受降仪式盟国各国代表甚至包括[[越南民主共和国|越南民主共和国]]代表，以及来宾约有五六百人。惟独没有法国代表出席。<ref>朱翧:《越南受降日记》,商务印书馆1946年版,第19页。</ref>虽然法国驻华大使多次向中国外交部交涉在云南流亡的法军返回越南及法国派代表参加北越受降仪式，行政院与外交部也向卢汉发出指示，但陆军总部与卢汉仍坚持自己主张。法国驻印度支那高级专员[[达尚礼|达尚礼]]为河内受降仪式上中国不给法国位置，要求法国驻华大使提出抗议。<ref>戴高乐著:《战争回忆录》第三卷《拯救(1944-1946)》,第582页。</ref>

1945年10月5日[[菲利普·勒克萊爾|菲利普·勒克萊爾]]中将率领远东军到达越南西贡。此后3个月法国远东军登陆并平定了柬埔寨和[[交趾支那|交趾支那]]([[南圻|南圻]])。

1946年2月28日中国外交部部长[[王世杰|王世杰]]和法国驻华大使{{tsl|en|Jacques Meyrier|梅理蔼}}在重庆签署并互换了《关于中越关系之协定》、《关于中国驻越北军队由法国军队接防之换文》、《关于法国供给中国驻越北军队越币之换文》等外交协议。其中《关于中国驻越北军队由法国军队接防之换文》规定“中国军队交防,于三月一日至十五日期间开始,至迟应于三月三十一日完毕”。3月4日下午，中法双方军事代表在中国受降军队司令部召开交接防务会议,，法方要求中方准许法国军队于3月6日在海防港登陆。中方拒绝法军有关3月6日登陆海防的要求，并且强调法越谈判签字以前不让法国单方面接防。3月6日上午，法国远东军舰队司令阿巴努中将（Auboyneau）根据远东军司令勒克莱尔中将的指示，不顾中国军队警告，率领海陆军强行登陆海防港，并对中国军队发动突然袭击。最后这场武装冲突以法国舰队司令阿巴努签署协定承担一切责任、赔偿一切损失并保证今后不再袭击中国占领军守地而告结束。 <ref>王理寰:《抗战胜利后海防痛击法军纪实》,载中国人民政治协商会议全国委员会文史资料研究委员会编:《文史资料选辑》第十二辑,中华书局1960年版,第145-154页。</ref>
   
1946年3月6日，胡志明和法国签署了一份协定，允许法国军队返回北越，而法国承认越南是“法国[[印度支那联邦|印度支那联邦]]下的一个自由国家，可以拥有自己的下政府、议会、军队和税收系统。”

1946年春，法国政府任命著名将领[[勒克莱尔|勒克莱尔]]为[[东方远征军|东方远征军]]司令，决心以武力恢复殖民秩序。1946年5月初，[[中华民国国民革命军|中华民国国民革命军]]全部撤出越南，[[法国军队|法国军队]]开始接管越南北方。1946年7月6日，[[越南独立同盟会|越南独立同盟会]]与[[法国政府|法国政府]]在[[法国|法国]][[枫丹白露|枫丹白露]]开始谈判<ref>[http://history.people.com.cn/GB/205396/16890022.html 中越战争谁是赢家：苏联帝国梦碎，美国深陷泥潭(4)]</ref>。9月14日，双方签订临时协定，但是法国无意守约。

1946年11月26日，法国舰队轰炸了海防，造成2000人死亡，4000多伤。12月法国又向越南民主共和国宣战。

==战争过程==
1946年11月20日，[[法国军队|法国军队]]向[[越南独立同盟会|越南独立同盟会]]发出[[最后通牒|最后通牒]]，要求[[越南人民军|越盟军]]撤出[[海防市|海防]]。1946年11月24日，法国军舰和炮兵开始炮轰[[海防市|海防]]，随后越盟同意了停火协议并撤出了海防，但不久又派出军队骚扰[[海防市|海防]]。随后越盟与法国开始了[[游击战|游击战]]。越盟军兵员素质、作战经验和武器装备等均差，处处被法军压制，原本与越盟军并肩作战的[[高台教|高台教]]、[[和好教|和好教]]等宗教团体也纷纷转向投法。1947年，法军攻陷河内，越盟军重返丛林，坚持游击战。1947年，[[法国军队|法国军队]]包围了[[越盟|越盟]]位于[[北越|北越]]的基地，当时[[胡志明|胡志明]]藏匿在地洞内，得以避过法军的生擒。
[[File:Charles_DeGaulle_and_Ho_Chi_Minh_are_hanged_in_effigy_during_the_National_Shame_Day_celebration_in_Saigon,_July_1964.jpg|thumb]]以及[[戴高樂|戴高樂]]人偶（1964年[[胡志明市|西貢]]）]]

1949年6月，[[法国|法国]]决定靠成立[[傀儡政权|傀儡政权]]的方式改善局面，法国成立了[[越南国|越南国]]并挑选了当时[[流亡|流亡]]在[[香港|香港]]的[[保大帝|保大]]来擔任[[国家元首|国家元首]]，[[越南国|越南国]]随后成立了自己武装力量——[[越南国军|越南国军]]，[[越南国军|越南国军]]与[[法国军队|法国军队]]一起对越盟进行围剿。

1950年12月，[[法國|法國]]和[[美國|美國]]簽訂了《[[共同防禦協定|共同防禦協定]]》，[[法國|法國]]获得了[[美國|美國]]提供的大批武器裝備。[[越南共产党|越南共产党]]则从[[中华人民共和国|中华人民共和国]]、[[苏联|苏联]]等社会主义国家也获得大批军事援助。1950年7月至8月间，中华人民共和国军事顾问团应邀赴越。1950年9月16日，[[越南人民军|越南人民军]]发动边界战役，攻占东溪、七溪等重要据点，至10月23日时共歼敌8000人，收复了高平、谅山、太原、老街、和平等市镇，巩固并扩大了越北根据地，打通了中越边境交通线，掌握了北部战场主动权。1950年4月和8月，柬埔寨抗战政府和老挝抗战政府先后成立，积极开展抗法武装斗争。

随后经历了平原战役、西北战役。

1954年3月，[[奠邊府戰役|奠邊府戰役]]开始，5月，法国战败，被迫與[[越南獨立同盟會|越盟]]就和平问题進行谈判，1954年7月21日，[[柬埔寨|柬埔寨]]、[[北越|北越]]、[[法国|法国]]、[[老挝|老挝]]、[[中华人民共和国|中华人民共和国]]、[[南越|南越]]、[[苏联|苏联]]、[[英国|英国]]、[[美国|美国]]九个国家的外交部长在[[日內瓦會議_(1954年)|日內瓦會議]]中达成協定，[[法國|法國]]撤出[[越南|越南]]，並承認[[越南|越南]]、[[老挝|老挝]]、[[柬埔寨|柬埔寨]]为獨立國家，以[[北纬17度|北纬17度]]作为北越（当时国号为“[[越南民主共和国|越南民主共和国]]”）与南越（当时国号为“[[越南国|越南国]]”）的分治线，在1956年7月前进行普选，根据普选结果合并南北越。

==战争罪行与劳改营==
* Boudarel事件。[[Georges_Boudarel|Georges Boudarel]]是一名参加战争的法共成员，他负责在劳改营裡折磨和洗脑被俘的法国人。<ref>[http://www.anapi.asso.fr/index.php?langue=en Boudarel affair in the ANAPI official website]</ref>法国全国战俘协会后来以战争罪将他告上法庭。很多法国和越南国军的战俘都再未找到下落。
* [[自由之路行動|自由之路行動]]是法美合作的一次撤离难民的行动，被撤走的是保皇派和天主教徒为主的越南人。<ref name="operationpassagetofreedom">{{cite web

* 1957, 法国参谋长Raoul Salan利用越盟劳改营中的经验创立了两个"[[绥靖与反叛乱训练中心|绥靖与反叛乱训练中心]]"（''Centre d'Instruction à la Pacification et à la Contre-Guérilla''）并在阿尔及利亚战争中培养了几千军官。

==后续==
1955年10月，[[吳廷琰|吳廷琰]]通过[[賄賂|賄賂]]、[[恐嚇|恐嚇]]赢得了[[1955年越南公民投票|1955年越南公民投票]]，[[越南国|越南国]]由[[君主制|君主制]]变为[[共和制|共和制]]，国名更改为[[越南共和国|越南共和国]]，吳廷琰出任总统，[[保大帝|保大帝]][[流亡|流亡]][[巴黎|巴黎]]。1956年6月，[[法軍|法軍]]完全撤出越南。美国从1954年开始介入越南事务，1965年，美国开始派兵直接与[[越共|越共]]作战，1973年，[[驻越美军|美军]]撤出越南，1975年，[[越共|越共]]将[[越南共和国|越南共和国]]推翻，越南統一。
