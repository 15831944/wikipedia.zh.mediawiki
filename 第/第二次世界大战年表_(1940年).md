以下是1940年的[[第二次世界大战年表|第二次世界大战年表]]：

== 1月 ==

: 1日：一万日军在中国山西东部发动反击，以援助几被包围的日军第36师<ref name=":0">{{Cite web|url=http://ww2db.com/event/timeline/1940/|title=1940 Timeline {{!}} World War II Database|accessdate=2018-12-04|last=Chen|first=C. Peter|work=ww2db.com}}</ref>。
: 2日：苏联进攻芬兰受挫，苏军大量坦克被毁。
: 7日：英国实行基本食品配给制<ref name=":1">{{Cite book|title=''The Times Atlas of the Second World War''|last=Keegan|first=John|publisher=Times Books|year=1989|isbn=|location=|pages=16-17}}</ref>。<br />据报，[[蘇奧穆斯薩爾米戰役|蘇奧穆斯薩爾米戰役]]芬兰获胜，歼灭苏军一整个师，缴获大量军车。
: 7日：[[谢苗·康斯坦丁诺维奇·铁木辛哥|谢苗·康斯坦丁诺维奇·铁木辛哥]]将军获苏联对芬军队指挥权<ref name=":0" />。
: 10日：{{tsl|de|Mechelen-Zwischenfall|梅赫倫事件}}发生，一架携带有[[黄色方案|黄色方案]]的德国飞机坠毁于中立国[[比利时|比利时]]。
: 16日：截获文件表明[[希特勒|希特勒]]计划入侵[[斯堪的纳维亚|斯堪的纳维亚]]，并将对法国和[[低地国家|低地国家]]的入侵行动推迟至春季，因春季天气更有利于入侵。
: 17日：苏联重又进攻芬兰，以猛烈空袭作为报复。
: 20日：4时15分，德国U-44潜艇在葡萄牙海域以鱼雷击沉希腊汽船''Ekatontarchos Dracoulis''，致6人死亡。此前6小时U-44一直在搜寻''Ekatontarchos Dracoulis''的位置<ref name=":0" />。
: 21日：一艘[[U型潜艇|U型潜艇]]击沉英国[[陛下之舰|陛下之舰]]“埃克斯茅斯”号[[驱逐舰|驱逐舰]]，135名船员全部遇难。
: 24日：[[赫尔曼·戈林|赫尔曼·戈林]]指定[[萊因哈德·海德里希|萊因哈德·海德里希]]解决“猶太人問題”。
: 27日：德国制定入侵丹麦和挪威的最终计划。

== 2月 ==
[[File:The_War_in_Finland,_1940_HU55566.jpg|右]]

: 1日：[[日本国会|日本国会]]公布预算，军事开支过半，创历史新高。
: 5日：英国和法国预料到德国的占领，决定干预挪威，以援助芬兰的名义断绝铁矿石贸易。计划定于3月20日左右执行。
: 9日：[[埃里希·冯·曼施坦因|埃里希·冯·曼施坦因]]被任命为德国第38装甲兵指挥官，不再参与策划对法国的入侵。
: 10日：苏联同意在新贸易条约中向德国提供粮食和原材料。
: 14日：英国政府要求志愿军赴芬兰战斗。
: 15日：苏军攻陷芬兰重要防御点苏马，突破[[曼纳海姆防线|曼纳海姆防线]]。
: 15日：希特勒下令进行[[無限制潛艇戰|無限制潛艇戰]]。
: 16日：英国[[陛下之舰|陛下之舰]]“哥薩克”号[[驱逐舰|驱逐舰]]在中立挪威领海从德国“阿尔特马克”号运输舰强行带走303名英国战俘，引发[[阿爾特馬克號事件|阿爾特馬克號事件]]。
: 17日：芬兰继续从[[曼纳海姆防线|曼纳海姆防线]]撤退。<br />曼施坦因向希特勒提出经由[[阿登|阿登森林]]入侵法国的计划。
: 21日：{{tsl|de|Nikolaus von Falkenhorst|尼古拉斯·馮·法爾肯霍斯特}}将军被任命指挥不久后对挪威的入侵。

== 3月 ==

: 1日：希特勒指导部下入侵丹麦和挪威。
: 3日：苏联开始进攻芬兰第二大城市[[维堡|维堡]]。
: 5日：芬兰告知苏联，同意其停战条件。次日，芬兰大使赴[[莫斯科|莫斯科]]就和约进行谈判。
: 11日：英国开始实行肉类配给制<ref name=":1" />。
: 12日：芬兰经过105天战斗后于莫斯科与苏联签订和约，被迫放弃重要领土换取和平。
: 16日：德国空袭[[斯卡帕湾|斯卡帕湾]]，二战期间首次造成英国人员伤亡。
: 18日：希特勒和[[墨索里尼|墨索里尼]]在[[奥地利|奥地利]]边境[[布伦纳山口|布伦纳山口]]会晤<ref name=":1" />。希特勒认为意大利将在适当时机参战，墨索里尼表示同意。
: 20日：[[爱德华·达拉第|爱德华·达拉第]]辞任法国总理。
: 21日：[[保罗·雷诺|保罗·雷诺]]就任法国总理。
: 28日：英法达成正式协议，双方均不会单独与德国寻求媾和。
: 29日：苏联想要更多领土。[[维亚切斯拉夫·米哈伊洛维奇·莫洛托夫|维亚切斯拉夫·米哈伊洛维奇·莫洛托夫]]对[[苏联最高苏维埃|苏联最高苏维埃]]提及[[罗马尼亚|罗马尼亚]][[比薩拉比亞|比薩拉比亞]]问题。
: 30日：日本在中国南京建立[[汪精卫|汪精卫]]傀儡政权。
: 30日：英国进行秘密侦察飞行，采用{{tsl|en|Sidney Cotton|西德尼·柯頓}}开创的高空高速立体摄像技术，对苏联境内目标区域拍照，为“[[派克行动|派克行动]]”做准备。

== 4月 ==

: 4月：[[卡廷大屠殺|卡廷大屠殺]]发生，苏联[[内务人民委员部|内务人民委员部]]屠杀22000名波兰官员和警察等。
: 3日：{{tsl|en|Minister for Co-ordination of Defence|国防协调部长}}{{tsl|en|Ernle Chatfield, 1st Baron Chatfield|恩爾·查特菲爾德}}的部级地位为[[第一海軍大臣|第一海軍大臣]][[温斯顿·丘吉尔|温斯顿·丘吉尔]]的部长级国防委员会所取代。
: 9日：德军从挪威数个港口登陆，拿下[[奥斯陆|奥斯陆]]；[[威悉演习行动|德國入侵丹麥]]，6小时后丹麦投降。
: 10日：德国成立挪威政府，由德国前国防部长[[维德孔·吉斯林|维德孔·吉斯林]]领导。
: 11日：发生第一次{{tsl|en|Battles of Narvik|纳尔维克战役}}，英国驱逐舰和飞机突袭德国海军获胜。
: 12日：英军占领丹麦[[法罗群岛|法罗群岛]]。
: 13日：发生第二次纳尔维克战役，英军再次获胜。
: 14日：英法军队开始登陆挪威[[特隆赫姆|特隆赫姆]]北部的[[納姆索斯|納姆索斯]]。
: 15日：英军登陆挪威[[纳尔维克|纳尔维克]]附近的[[哈尔斯塔|哈尔斯塔]]。
: 16日：更多英军登陆挪威，特隆赫姆南部和北部尤甚；对特隆赫姆的争夺持续到22日。
: 27日：英国开始从挪威中部、特隆赫姆北部和南部撤军。

== 5月 ==

: 1日：盟军开始撤离挪威港口；撤离工作将持续到6月。
: 5日：挪威在伦敦成立流亡政府<ref name=":1" />。
: 8日：[[内维尔·张伯伦|内维尔·张伯伦]]首相在[[英国下议院|英国下议院]]{{tsl|en|Norway Debate|挪威辩论}}中险胜。
: 9日：英国征兵年龄限制扩大至36岁。 
: 10日：德国入侵比利时、法国、[[卢森堡|卢森堡]]和荷兰；内维尔·张伯伦辞任英国首相；[[温斯顿·丘吉尔|温斯顿·丘吉尔]]就任英国首相。英国入侵冰岛。
: 比利时宣布进入紧急状态。丘吉尔被要求组建一个战时联合政府<ref name=":1" />。
: 德国开始大规模进攻[[西方战线_(第二次世界大战)|西方战线]]卢森堡、比利时、荷兰和法国。德国空降部队[[埃本-埃美爾要塞戰役|拿下埃本-埃美爾要塞]]。
: 10日：[[海牙戰役|海牙戰役]]荷兰迅速击败侵略者，伞兵进攻有史以来首次失败。
: 11日：卢森堡沦陷。
: 丘吉尔向居于荷兰的德国前皇帝[[威廉二世_(德国)|威廉二世]]提供英国政治庇护，被拒绝。
: 12日：比利时炸毁[[默兹河|默兹河]]所有桥梁，以阻止德军前进。
: 12日：{{tsl|de|Schlacht bei Hannut|汉努特之战}}在比利时爆发。
: 13日：{{tsl|nl|Londens kabinet|荷兰流亡政府}}在[[伦敦|伦敦]]成立。 
: [[海因茨·古德里安|海因茨·古德里安]]将军的装甲兵团攻破法国[[色當_(阿登省)|色當]]。
: [[威廉明娜_(荷兰)|威廉明娜]]逃往英国寻求庇护。
: 丘吉尔在下议院发表“[[鲜血、辛劳、眼泪和汗水|鲜血、辛劳、眼泪和汗水]]”演讲。
: 13日：{{tsl|nl|Slag om de Grebbeberg|赫雷貝貝爾赫戰鬥}}荷兰战败于德国。
: 14日：新任战争部长[[安東尼·艾登|安東尼·艾登]]宣布成立{{tsl|en|Home Guard (United Kingdom)|英國國土警衛隊}}，主要由老人和退休人员组成。
: 丘吉尔向[[富兰克林·德拉诺·罗斯福|富兰克林·德拉诺·罗斯福]]总统和加拿大请求援助。公布了包括[[工党_(英国)|工党]]、[[自由党_(英国)|自由党]]和[[保守黨_(英國)|保守黨]]党员在内的新英国联合政府的纲要。
: 14日：[[荷蘭堤防戰役|荷蘭堤防戰役]]荷兰击败德国。
: 14日：[[鹿特丹轟炸|鹿特丹轟炸]]促成德国在{{tsl|nl|Strijd om de Maasbruggen in Rotterdam|鹿特丹战役}}的胜利，同时造成巨大破坏，大量平民死亡。
: 荷兰决定投降，唯[[泽兰省|泽兰省]]除外。
: 15日：荷军投降<ref name=":1" />。 
: 为报复德军对鹿特丹的轟炸，英国进行了[[二戰期間的戰略轟炸|二戰期間首次大规模戰略轟炸]]，袭击了[[盖尔森基兴|盖尔森基兴]]。随后几天又轰炸了[[汉堡|汉堡]]、[[不来梅|不来梅]]、[[科隆|科隆]]、[[埃森|埃森]]、[[杜伊斯堡|杜伊斯堡]]、[[杜塞尔多夫|杜塞尔多夫]]和[[汉诺威|汉诺威]]。
: 德军越过默兹河<ref name=":1" />。
: 16日：丘吉尔造访巴黎，听闻法国战争几近结束。
: 16日：比利时军撤退，比利时政府迁往法国[[波尔多|波尔多]]。后又迁至伦敦<ref>{{Cite book|title=''Dictionnaire de la Seconde Guerre mondiale en Belgique''|last=Aron|first=Paul|last2=Gotovitch|first2=José|publisher=|year=2008|isbn=|location=|pages=408}}</ref>。
: 17日：德军进驻[[布鲁塞尔|布鲁塞尔]]，拿下[[安特卫普|安特卫普]]。
: [[保罗·雷诺|保罗·雷诺]]和84岁[[一战|一战]]英雄[[菲利普·贝当|菲利普·贝当]]组建新法国政府。
: 18日：[[馬克西姆·魏剛|馬克西姆·魏剛]]取代[[毛利斯·甘莫林|毛利斯·甘莫林]]成为法国武装部队指挥官<ref name=":1" />。
: 安特卫普沦陷。
: 18日：德国赢得{{tsl|nl|Strijd in Zeeland|泽兰省之战}}。
: 19日：法国[[亚眠|亚眠]]遭德军围困；[[阿拉斯|阿拉斯]]遭[[埃尔温·隆美尔|埃尔温·隆美尔]]部队围困；另有德军到达[[滨海努瓦耶勒|滨海努瓦耶勒]]。
: 19日：英国完全入侵冰岛。
: 20日：[[海因茨·古德里安|海因茨·古德里安]]将军的装甲兵团拿下[[阿布维勒|阿布维勒]]，威胁到该地区的同盟国军。
: 23日：战前英国法西斯领导人{{tsl|en|Oswald Mosley|奧斯瓦爾德·莫斯利}}入狱；他和妻子将在狱中度过余生。 
: 24日：英国最终决定停止在挪威的行动。
: 25日：同盟国军撤至[[敦刻尔克|敦刻尔克]]<ref name=":1" />。希特勒下令德军停止向同盟国军滩头推进，允许[[赫尔曼·戈林|赫尔曼·戈林]]用空军展开进攻。[[英國皇家空軍|英國皇家空軍]]守住滩头。
: 纳粹德国空军对英格兰零星轰炸。
: [[滨海布洛涅|滨海布洛涅]]向德军投降。
: 25日：苏联准备完全接管波罗的海国家。 苏联政府指责立陶宛绑架苏联士兵。
: 25-28日：86名比利时平民在芬克特村被德军杀害。
: 26日：“A4”号[[巡邏艦|巡邏艦]]到达[[普利茅斯|普利茅斯]]，将最后40吨国家[[黄金储备|黄金储备]]运出比利时。
: 26日：[[加来|加来]]向德军投降。
: [[鄧扣克大撤退|鄧扣克大撤退]]开始，同盟国军需要从[[敦刻尔克|敦刻尔克]]撤出34万人。这一行动将在纳粹德国空军猛烈轰炸下持续到6月3日。
: 28日：比利时向德国投降；[[利奥波德三世_(比利时)|利奥波德三世]]投降并被拘留。
: 30日：[[英國內閣|英國內閣]]会议，尽管[[E·F·L·伍德，第一代哈利法克斯伯爵|E·F·L·伍德]]和张伯伦反驳有力，丘吉尔仍然赢得投票，继续作战。
: 31日：日军猛烈轰炸长江上游[[国民政府|国民政府]]首都重庆。  <br />

== 6月 ==

: 3日：[[敦克爾克大撤退|敦克爾克大撤退]]最后一天，224,686英军和121,445法军和比利时军全部撤离完毕。<br />德军轰炸[[巴黎|巴黎]]。
: 4日：温斯顿·丘吉尔向下议院发表“我们誓不投降”演讲。
: 7日：德国[[格奈森瑙號戰艦|格奈森瑙號戰艦]]和[[沙恩霍斯特號戰艦|沙恩霍斯特號戰艦]]在挪威海域击沉[[陛下之舰|陛下之舰]]“光榮”號[[航空母舰|航空母舰]]和两艘驱逐舰；英国舰只无空中掩护。
: 9日：[[苏联红军|苏联红军]]在[[拉脱维亚|拉脱维亚]]边境挑起冲突。
: 10日：意大利向法国和英国宣战。挪威投降。[[哈康七世|哈康七世]]及其政府三天前已撤至英国。
: 11日：法国政府迁至[[图尔|图尔]]。馬耳他戰役爆发。
: 12日：{{tsl|en|51st (Highland) Division|第51高地师}}一万多英军在[[索姆河畔聖瓦萊里|索姆河畔聖瓦萊里]]被俘。
: 13日：德军占领巴黎；法国政府迁至[[波尔多|波尔多]]<ref name=":1" />。
: 14日：驻扎于[[土伦|土伦]]的[[法國海軍|法國海軍]]对[[利古里亚大区|利古里亚海岸]]的意大利目标发动进攻作战。<br />苏联[[波羅的海艦隊|波羅的海艦隊]]对[[波罗的海国家|波罗的海国家]]实行全面军事封锁。波罗的海沿线苏军已预备在波罗的海国家组织[[共产主义|共产主义]]政变。苏联轰炸机击落一架自[[塔林|塔林]]飞往[[赫尔辛基|赫尔辛基]]的“{{tsl|fi|Matkustajakone Kalevan pudotus|卡勒瓦}}”号芬兰客机，机上载有美国驻塔林、[[里加|里加]]和赫尔辛基[[公使馆|公使馆]]的三个外交邮袋。
: 15日：苏联向[[立陶宛|立陶宛]]发出[[最后通牒|最后通牒]]，限八小时投降。[[安塔纳斯·斯梅托纳|安塔纳斯·斯梅托纳]]总统逃离立陶宛，因此不可能以合法方式接替政权。苏军进驻立陶宛，攻击拉脱维亚边防军。<br />英军开始从法国西部港口撤离，称“{{tsl|en|Operation Ariel|艾莉尔行动}}”。
: 16日：[[保罗·雷诺|保罗·雷诺]]辞任法国总理，[[菲利普·贝当|菲利普·贝当]]就任法国总理。<br />法国''La Curieuse''号[[单桅纵帆船|单桅纵帆船]]迫使意''Provano''号潜艇浮出水面，后将其撞沉。<br />苏联向拉脱维亚和[[爱沙尼亚|爱沙尼亚]]发出投降的最后通牒。
: 17日：[[兰开斯特里亚号|兰开斯特里亚号]]班轮在[[圣纳泽尔|圣纳泽尔]]海域沉没。此次事件是英国最严重的一次海难，至少有3000人丧生。
: 苏军进驻拉脱维亚和[[爱沙尼亚|爱沙尼亚]]。
: 18日：[[夏爾·戴高樂|夏爾·戴高樂]]将军成立法国流亡政府“法兰西民族解放委员会”；爱沙尼亚、拉脱维亚和立陶宛[[蘇聯佔領波羅的海國家|被苏联占领]]。

: 28日：[[维德孔·吉斯林|维德孔·吉斯林]]出任挪威元首。

== 10月 ==

: 1日：[[国共两党|国共两党]]在[[华南|华南]]内战；日军在[[长沙|长沙]]受挫。<br />
: 2日：[[伦敦大轰炸|伦敦大轰炸]]本月继续。
: 3日：德国命令[[华沙|华沙]]犹太人迁往[[華沙猶太區|華沙猶太區]]。
: 4日：希特勒和墨索里尼在[[布伦纳山口|布伦纳山口]]会面，讨论战争前景。
: 7日：应罗马尼亚9月7日提出的要求，德国[[入侵南斯拉夫之戰|向罗马尼亚派遣一军事特派团]]，以期为罗军提供训练并守卫罗马尼亚油田。
: 9日：[[内维尔·张伯伦|内维尔·张伯伦]]因健康原因辞去[[英国下议院|英国下议院]]职务；[[温斯顿·丘吉尔|温斯顿·丘吉尔]]当选[[保守黨_(英國)|保守黨]]领袖。
: 12日：德国对英国的所有入侵行动最早都要推迟至次年春。
: 12日：[[Battle_of_Cape_Passero_(1940)|帕塞罗角海战]]发生，英国皇家海军执行赴马耳他执行护航任务后，遭遇数艘意舰，英军获胜。
: 13日：德国轰炸英国次数大幅减少，仍有平民伤亡。
: 14日：[[巴勒姆站|巴勒姆站]]灾难发生，德国炸弹穿透地下32英尺，造成66人丧生<ref>{{Cite web|url=http://ww2today.com/14th-october-1940-disaster-at-balham-tube-station|title=14th October 1940: Disaster at Balham Tube station|accessdate=2018-12-04|work=ww2today.com|language=en-US}}</ref>。
: 15日：[[Clarence_Addison_Dykstra|克拉伦斯·艾迪生·戴克斯特拉]]任[[美國兵役登記|美国兵役登记处]]主任。
: 15日：墨索里尼及其最亲密的顾问决定[[希臘戰役|入侵希腊]]。
: 16日：美国开始征兵登记。
: 19日：意大利轰炸[[巴林|巴林]]。
: 20日：意大利飞机轰炸[[埃及|埃及]][[开罗|开罗]]和位于[[巴林|巴林]]英国保护区的美国炼油厂。
: 21日：[[利物浦|利物浦]]遭受第200次轰炸。
: 23日：希特勒在西班牙法国边境的[[昂代伊|昂代伊]]会见[[弗朗西斯科·佛朗哥|佛朗哥]]；希特勒想说服佛朗哥加入轴心国一方，未果。
: 24日：希特勒赴[[卢瓦河畔蒙图瓦尔|卢瓦河畔蒙图瓦尔]]会见[[菲利普·贝当|菲利普·贝当]]，意味着法国与纳粹政权合作的开始。
: 24日：意大利空军参加[[不列顛戰役|不列顛戰役]]。
: 25日：[[柏林|柏林]]和[[汉堡|汉堡]]遭受猛烈轰炸。
: 28日：大约凌晨3时，意大利驻希腊大使向希腊发出最后通牒，希腊总理[[扬尼斯·梅塔克萨斯|扬尼斯·梅塔克萨斯]]答复道：“那就打仗吧。”[[皇家意大利軍|皇家意大利軍]]从[[阿尔巴尼亚王国_(1939年－1943年)|意控阿尔巴尼亚]]向希腊发动攻击，[[希意戰爭|希意戰爭]]开始。希特勒对盟友的这一主动行为感到愤怒。
: 29日：U型潜艇数量增加，护航队损失惨重。
: 29日：美国《选择性服务法》第一批征兵入伍。
: 30日：罗斯福总统竞选时承诺，不会把“我们的孩子”送上战场。
: 31日：华沙地区政府将住在华沙的所有犹太人转移至犹太区。

== 11月 ==

: 1日：[[土耳其|土耳其]]宣布就[[希意戰爭|希意戰爭]]保持中立<ref name=":0" />。
: 2日：意军继续进军希腊。[[沃武薩|沃武薩]]沦陷，意军轰炸[[塞萨洛尼基|塞萨洛尼基]]。
: 5日：罗斯福总统赢得第三任期，英国认为可望从美国获得更多帮助。
: 5日：[[陛下之舰|陛下之舰]]“傑維斯灣”号[[武装商船|武装商船]]执行护航任务时沉没，但其所护舰只许多都幸免遇难。
: 7日：很明显，爱尔兰将拒绝联合王国将其港口用作海军基地。
: 8日：[[Battle_of_Elaia–Kalamas|埃莱阿-卡拉玛战役]]结束，意军进攻希腊无果，宣告结束。
: 9日：[[内维尔·张伯伦|内维尔·张伯伦]]去世。
: 11日：[[塔蘭托戰役|塔蘭托戰役]]发生，英国海军向意大利海军发起攻击，[[光辉号航空母舰_(87)|光辉号航空母舰]]的[[剑鱼式鱼雷轰炸机|剑鱼式鱼雷轰炸机]]重创3艘战列舰、2艘巡洋舰及多艘辅助舰。此事件确保了英国地中海补给线的安全。日本已在策划[[珍珠港事件|珍珠港事件]]，不久后将研究英军的这次成功。
: 12日：莫洛托夫在柏林会见希特勒和里宾特洛甫，其间讨论了国际新秩序。莫洛托夫表示苏联对芬兰、保加利亚、罗马尼亚、[[達達尼爾海峽|達達尼爾海峽]]和[[伊斯坦布尔海峡|伊斯坦布尔海峡]]有兴趣，希特勒则大谈特谈俄罗斯、德国、意大利和日本在全球的势力范围。
: 12日：[[加彭戰役|加彭戰役]]结束，英军夺取非洲中部。
: 13日：莫洛托夫再次会见希特勒，要求再次攻击芬兰。希特勒拒绝苏联扩大欧洲势力的企图。他认为不列颠已经被打败了，把印度给苏联。
: 13日：[[班都斯戰役|班都斯戰役]]结束，希腊获胜。
: 14日：[[考文垂大轟炸|考文垂大轟炸]]发生。[[考文垂座堂|考文垂座堂]]被毁，城市中世纪中心夷为平地。
: 14日：希腊开始[[反攻|反攻]]意大利。
: 15日：苏联受邀加入《[[三国同盟条约|三国同盟条约]]》，瓜分大英帝国战利品。华沙犹太区被与城市其它地方隔离开来。
: 16日：丘吉尔要求派遣部分驻北非英军至希腊，尽管其军事领袖认为北非需要他们对抗意军。
: 19日：希腊继续进军，将意军逐出希腊。[[File:Greek_Offensive_1940_41_in_Northern_Epirus.svg|缩略图]]
: 20日：[[匈牙利|匈牙利]]签署《三国同盟条约》。
: 21日：比利时在英国的流亡政府向意大利宣战。
: 22日：希腊拿下[[科尔察|科尔察]]。
: 23日：[[羅馬尼亞|羅馬尼亞]]签署《三国同盟条约》。
: 24日：[[斯洛伐克共和國_(1939年－1945年)|斯洛伐克共和國]]签署《三国同盟条约》。
: 25日：苏联给出其加入《三国同盟条约》的条件，其中提到新的领土要求。
: 29日：[[利物浦|利物浦]]彻夜遭到大规模轰炸。
: 30日：英格兰南部城市[[南安普敦|南安普敦]]遭到大规模轰炸；次日晚，南安普敦再次受到袭击；随后[[布里斯托尔|布里斯托尔]]在12月2日、[[伯明翰|伯明翰]]在12月3日又遭袭。

== 12月 ==
[[File:Ww2_allied_axis_1940_dec.png|缩略图]]

: 1-8日：希军继续击退意军，拿下[[波格拉德茨|波格拉德茨]]、[[萨兰达|萨兰达]]和[[吉诺卡斯特|吉诺卡斯特]]。
: 1日：德英两国整个月互相进行空袭，德国先轰炸英国，后英国轰炸德国。
: 1日：罗斯福总统要求美国驻英大使[[老约瑟夫·P·肯尼迪|老约瑟夫·P·肯尼迪]]引咎辞职，因他接受一家报纸采访时说“民主在英国已经结束”。
: 5日：英国皇家空军轰炸[[杜塞尔多夫|杜塞尔多夫]]和[[都灵|都灵]]。
: 6-9日：{{tsl|en|Western Desert Force|西部沙漠部队}}印度和英国军队发动[[羅盤行動|羅盤行動]]，进攻驻埃及意军。意军有七个步兵师和强化防守阵地的{{tsl|it|Raggruppamento sahariano "Maletti"|马莱蒂组织}}。最初攻击目标为意军营地附近以及{{tsl|ar|سيدي براني|西迪拜拉尼}}南部。意军营地迅速被攻占，意军[[彼得羅·馬萊蒂|彼得羅·馬萊蒂]]将军死亡，马莱蒂组织、利比亚第1师、利比亚第2师和第4黑衫师几乎全军覆没。余下意军被迫向利比亚撤退。
: 8日：佛朗哥排除西班牙参战的可能，直接导致希特勒被迫取消对直布罗陀的进攻。
: 12日：埃及有39,000多名意军死亡或被俘。
: 16日：英国皇家空军首次进行夜间空袭，目标为德国[[曼海姆|曼海姆]]。
: 16日：非洲北部英军在{{tsl|ar|السلوم|塞卢姆}}指挥占领了利比亚{{tsl|it|Forte Capuzzo|卡普佐堡}}。<br />
: 18日：希特勒下达指令，开始策划[[巴巴羅薩行動|巴巴羅薩行動]]。
: 22-24日：[[曼彻斯特|曼彻斯特]]受到轰炸。
: 28日：[[希意戰爭|希意戰爭]]局势持续对意军不利；希军大约占领了阿尔巴尼亚的四分之一。<br />28日：意大利向德国请求援助对抗希腊。
: 29日：德军大规模空袭伦敦；[[圣保罗座堂|圣保罗座堂]]遭到破坏。
