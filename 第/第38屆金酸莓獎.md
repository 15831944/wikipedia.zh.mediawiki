'''第38屆金酸莓獎'''（{{lang-en|'''38th Golden Raspberry Awards'''}}）是於2017年頒發給表現最差的電影工作者。入圍名單於2018年1月22日公佈<ref>{{cite news|title=第38屆金酸莓獎公布入圍名單|url=http://www.hypesphere.com/archives/97399|accessdate=2018-01-23|work=狂熱球電影資訊網|date=2018-01-22}}</ref>，獲獎者於同年3月3日公佈<ref>{{cite web|url=http://razzies.com/blogz.html|title=BLOGZ|website=The Razzies|deadurl=yes|archiveurl=https://web.archive.org/web/20170911155113/http://www.razzies.com/blogz.html|archivedate=2017-09-11}}</ref>。

提名公告後，該組織收到觀眾和評論家的反彈，獲得三項提名電影《[[母親！|母親！]]》及演員[[珍妮佛·勞倫斯|珍妮佛·勞倫斯]]，被認為是今年最好的電影作品，也是她職業生涯中最好的作品之一<ref>{{cite web |url=http://comicbook.com/horror/2018/01/22/jennifer-lawrence-razzies-golden-raspberries-mother/ |title=Razzies Face Backlash Over Nominating Jennifer Lawrence |website=comicbook.com |access-date=2018-01-25}}</ref><ref>{{cite web |url=https://www.avclub.com/the-razzies-waste-everyones-time-by-nominating-mother-1822304238 |title=The Razzies waste everyone’s time by nominating ''Mother!'' and not ''The Snowman'' |website=''[[The_A.V._Club|The A.V. Club]]'' |access-date=2018-01-25}}</ref><ref>{{cite web |url=http://www.nzherald.co.nz/entertainment/news/article.cfm?c_id=1501119&objectid=11980108 |title=Worst actor? Fans rage over Jennifer Lawrence Razzie award nomination |website=''[[The_New_Zealand_Herald|The New Zealand Herald]]'' |access-date=2018-01-25}}</ref><ref>{{cite web |url=http://www.complex.com/pop-culture/2018/01/mother-fans-jennifer-lawrence-razzies-nomination |title='Mother' Fans Are Pissed About Jennifer Lawrence's Razzies Nomination |website=''[[Complex_(magazine)|Complex.com]]'' |access-date=2018-01-25}}</ref>。
