'''第39屆日本眾議院議員總選舉'''（{{jpn|j=第39回衆議院議員総選挙|hg=だい39かいしゅうぎいんぎいんそうせんきょ|rm=|f=h}}）於1990年2月18日舉行，改選[[日本眾議院|日本眾議院]]全部512個議席。这次选举是[[平成時代|平成時代]]的首次眾議院選舉、也是日本引入[[消费税|消费税]]后的首次选举，也是一次由[[解散众议院|解散众议院]]引发的提前选举（众议院被解散的原因也是关于消费税的争议）。投票日当天，登记选民人数是90,322,908人。这次选举的投票率是73.31%。选举后，[[日本自由民主党|自由民主党]]损失部分议席，但仍为众议院第一大党，维持众议院的{{link-ja|稳定多数|安定多数}}，由党首[[海部俊树|海部俊树]][[第2次海部內閣|再次组建内阁]]。[[日本社会党|日本社会党]]、[[公明党|公明党]]分别为众议院第二大党、第三大党。在野党除日本社会党外，获得的席次普遍下降<ref>{{Cite web |author= 佐藤令|date= 2005-12|url= http://www.ndl.go.jp/jp/diet/publication/refer/200512_659/065905.pdf|title= 戦後の補欠選挙|format=PDF |work= |publisher= 国立国会図書館|accessdate=2016-5-26}}</ref>。

==奥姆真理教==
邪教团体[[奥姆真理教|奥姆真理教]]成员组成的“真理党”也参加了这次选举，但以无人当选的惨败告终。这次选举的失败，使奥姆真理教开始执行暴力夺权的方针<ref name=AsahiNiimi>{{cite web |title=「グル（麻原彰晃）の指示なら、人を殺すことも喜び」＜教団エリートの「罪と罰」（2）＞ |url=https://dot.asahi.com/wa/2018070600058.html?page=2 |website=週刊朝日 |accessdate=2018-12-18}}</ref>。
