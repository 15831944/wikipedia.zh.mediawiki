'''第66届托尼奖颁奖典礼'''于2012年6月10日举办，旨在表彰肯定2011年-2012年演出季中百老汇舞台剧的艺术成就。举办地于[[纽约|纽约]]灯光剧院，由哥伦比亚广播公司（CBS）现场直播<ref name=play1>{{cite web| last = Gans| first = Andrew|title = 66th Annual Tony Awards Set for June 10|work = Playbill| date = 2012年1月17日| url = http://www.playbill.com/news/article/158655-66th-Annual-Tony-Awards-Set-for-June-10}}</ref>，[[尼尔·帕特里克·哈里斯|尼尔·帕特里克·哈里斯]]奖第三次担纲主持人，这是他自2011年之后第三次主持托尼奖。2012年4月30日公布各大奖项的提名名单。
