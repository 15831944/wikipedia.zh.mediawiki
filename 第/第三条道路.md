'''第三条道路'''（{{lang-en|Third Way}}），又称'''新中间路线'''（Middle Way），是一种走在[[自由放任|自由放任资本主义]]和传统[[社会主义|社会主义]]中间的一种政治经济理念的概称<ref name = political1>[http://news.bbc.co.uk/chinese/trad/hi/newsid_4460000/newsid_4465200/4465211.stm BBC中文网-布莱尔与“第三条道路”]</ref>。它由[[中间派|中间派]]所倡导，是[[社会民主主义|社会民主主义]]的一个流派<ref name = political1></ref>，[[英国工党|英国工党]]称其为「现代化的社会民主主义」。它的中心思想是既不主张纯粹的[[自由市场|自由市场]]，也不主张纯粹的社會主義，主张在两者之间取折衷方案。

==特色==
第三条道路不只单单是走在中间，或只是一种妥协或混合出来的东西，第三条道路的提倡者看到了社会主义和资本主义互有不足之处，所以偏向某一极端也不是一件好事，第三条道路正正是揉合了双方主义的优点，互补不足而成的政治哲学。<ref name = political1></ref>

第三条道路主要内容按照捷克經濟學家[[奧塔·希克|奧塔·希克]]总结为<ref name = political1></ref>：“人文關懷”和“經濟效率”的結合，也是“國家計劃干預”和“合作社式自主企業市場競爭”（市场经济）的結合，是一种“混合經濟”的道路。

[[托尼·布莱尔|托尼·布莱尔]]在任[[英国首相|英国首相]]期间（1997－2007），放松对各种企业的监管内容，并为了鼓励企业创新和降低成本，工黨政府多次降低企业税，包括对中小企业实行税额减免等政策，大力鼓励失业者接受各种职业培训使和督促其重新就业，规定失业者领取社会补贴必须以“寻找工作”作为必要条件，并惩罚怠工者。另外一方面，布莱尔政府不断加大对交通、通讯、社会服务等基础设施的公共投资，通过改革金融机制提高服务效率。<ref>[http://news.xinhuanet.com/world/2007-06/26/content_6292888.htm 新华网-2007-06-26-布莱尔与“第三条道路”盘点] {{webarchive|url=https://web.archive.org/web/20131024083430/http://news.xinhuanet.com/world/2007-06/26/content_6292888.htm |date=2013-10-24 }}</ref>

[[比尔·克林顿|比尔·克林顿]]在任[[美國總統|美國總統]]期间（1993－2001），领导下的[[民主党_(美国)|民主黨]]由於在1994年的國會選舉慘敗，失去國會兩院的控制權，由於[[共和党_(美国)|共和黨]]控制國會兩院，克林頓政府為了希望在兩院得到共和党溫和派的支持通過法案，決定推行[[中間派|中間派]]路線，强调政府在财政上的保守、用一系列所谓“工作福利”的政策来代替旧有的“高福利”政策以及对于自由市场的维护功能。但是同时，这些政治家避免将自身与纯粹的“[[自由放任主义|自由放任主义]]”经济学和其他[[自由主义|自由主义]]相混同。

一般认为，第三条道路不是一种意识形态，因为其不涉及到政治运动。试图将第三条道路政治化的努力被称为激进中间派。<ref>[http://www.the-vibe.co.uk/2010/11/13/convergence-of-parties-to-the-centre-ground-fosters-voter-apathy/ 'Convergence of parties to the centre-ground fosters voter apathy'] 'The Vibe' article</ref>

==起源==
这个名称最早于1900年代[[庇护十一世|庇护十一世]]提出，他认为[[资本主义|资本主义]]和[[社会主义|社会主义]]都不合适，希望找到走在资本主义和社会主义中间的“第三条道路”，其后不断有人开始在这条道路上不停钻研，但当时这个概念还是相当模糊。

1938年，[[英国|英国]][[保守党_(英国)|保守党]]代表[[麦克米伦|麦克米伦]]写了一本名为《中间路线》（The Middle Way）<ref>MacMillan H (1978) ''The Middle Way'', EP Publishing Ltd. ISBN 0-7158-1333-1</ref>的著作，其后他在1957年至1963年就任英国首相也是套用这个政治理念来管治英国。

1950年代，这种观点被[[德国|德国]]秩序自由主义经济学家威廉汉姆·勒普克所发展完善为[[社会市场经济|社会市场经济]]。

1960年代，著名的捷克经济学家[[奧塔·希克|奧塔·希克]]对理论化整套第三条道路作出重大的贡献。

到了1980年代，[[右翼|右翼]]的[[英國首相|英國首相]][[撒切尔夫人|撒切尔夫人]]政府和[[美國總統|美國總統]][[雷根|雷根]]政府，将经济上缓和管制、[[个人主义|个人主义]]以及[[全球化|全球化]]等理念融合进主流的[[左翼|左翼]]思想，许多政治家倾向于用“第三条道路”来形容这种新政。而这个名称也经常被作为[[新自由主义|新自由主义]]社会经济政策的别称。就这样，这个理念成为现代欧洲民主社会中一种非常重要的思想。尤其对于一些歐美[[社会民主党|社会民主党]]派和[[美国民主党|美国民主党]]成员来说，这个理念是其核心思想。

曾任英国首相的[[托尼·布莱尔|托尼·布莱尔]]受到[[英国|英国]][[社会学|社会学]]家[[安東尼·紀登斯|安東尼·紀登斯]]影响，[[1997年英國大选|1997年英國大选]]，他在竞选期间以“第三条道路”作出竞选口号，工党放棄過往激進的[[左翼|左翼]]路線，重新定位，结果使工党經歷在野18年后，终于赢得了大选，重新上台，故他被视为体现“第三条道路”理念的代表。布萊爾認為，傳統[[左派|左派]]主打國家統治和高稅率，[[新右派|新右派]]主打個人主義和市場決定論，否定社會團結，所以他主張「第三條道路」，包涵「大膽的民營化」、「降低社會福利標準」和加大政府对公共设施的投资。有些人却误以为“第三条道路”是由托尼·布莱尔提倡或创造的，其实是错误的。<ref name = political1></ref>

1998年9月，[[德國|德國]]的[[格哈特·施罗德|格哈特·施罗德]]所領導的[[德國社會民主黨|社會民主黨]]在野16年後終於赢得了大选，和[[德國綠黨|綠黨]]組成[[聯合政府|聯合政府]]。

當時[[歐盟|歐盟]]15個會員國中，除了[[西班牙|西班牙]]及[[愛爾蘭|愛爾蘭]]外，[[執政黨|執政黨]]全部是[[中間偏左|中間偏左]]的社会民主主義政黨所領導的聯合政府。

2000年，[[薩帕特羅|薩帕特羅]]成為[[西班牙工人社會黨|西班牙工人社會黨]]領袖後開始推行類似路線的政策，稱之為「Nueva Vía」，2004年至2011年出任[[西班牙首相|西班牙首相]]。

==批评==
第三条道路也经常被评论为原有的“带有少数社会主义性质的资本主义取代社会主义制度”的社会民主主义观念，以及[[社會民主黨|社會民主黨]]派为了赢得选举，放棄過去激進的[[左翼|左翼]]道路而选择的一种手段。不少[[左派|左派]]的支持者认为第三条道路理论最终是以勞動阶层和穷人的牺牲来换取大企业的支持，獲取利益和社会福利，甚至認為社會民主黨派的第三條道路實際上是把黨[[右傾|右傾]]化，以免讓右翼的[[保守派|保守派]]政黨可以長期取得選舉優勢。由於社會民主黨派走第三条道路，因而在近二十年的選舉中，增加了不少[[中產|中產]]選民的支持，但又沒有失去基層的基本支持基礎。前[[英国首相|英国首相]][[托尼·布莱尔|托尼·布莱尔]]和前[[德国總理|德国總理]][[格哈特·施罗德|格哈特·施罗德]]等都是实行这种政策的典型中間派政治家代表。

无论在欧美或其他地区的国家，其结论都是一致的，即人民都希望有一个真正全力为人民服务，保护人民，以人民为中心的政府。但是必须留意的是，这种情况在过去是不可能的。以[[欧洲|欧洲]]为例，[[东欧|东欧]]企图避开一个由政府主导的社会；[[西欧|西欧]]则试着躲避一个由市场主导的社会。所以无论东欧或西欧彼此都在寻求一个最有活力和能力的政府，一个最可以平衡两者，支持人民生活最大福祉的政府。[[冷戰|冷戰]]后过去二十年的经验裡，[[东欧|东欧]]及[[拉丁美洲|拉丁美洲]]各国虽然拥有了自由思想与舆论，却仍无法压抑或摒弃官僚积习。这正是“第三条道路”主要的讨论问题。
