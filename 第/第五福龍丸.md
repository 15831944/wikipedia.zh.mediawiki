'''第五福龍丸'''是[[日本|日本]]一艘远洋[[鮪魚)|鮪魚]]渔船的名字，受到1954年3月1日[[美國|美國]]在[[比基尼環礁|比基尼環礁]]試爆[[氫彈|氫彈]]所產生的高能[[輻射|輻射]]感染。在試爆後的數星期，船上船員皆為[[急性辐射综合症|急性辐射综合症]]所苦，並在治療過程中，不慎經由輸血而得到了[[C型肝炎|C型肝炎]]。除了無線通信長久保山愛吉以外，其它船員全數痊癒，久保山愛吉則在半年以后的9月23日死於C型肝炎導致的[[肝硬化|肝硬化]]。久保山愛吉被視為第一個死於氫彈的受害者。

== 辐射污染意外事件 ==
1954年3月1日，在[[马绍尔群岛|马绍尔群岛]]附近海域捕鱼的第五福龍丸，受到美國在比基尼環礁進行的水下氢弹[[核子試爆|試爆]]「[[城堡行動|城堡行動]]」所產生之辐射影響，渔船上23名船员和船上的漁獲全受到核污染。雖然在試爆时，第五福龍丸在美国设定的危险水域以外捕捞，但美方對這次核子試爆的威力估算錯誤，爆炸當量超出預期的2.5倍，致使第五福龍丸的船員在接下來的數小时内一直暴露在[[輻射落塵|-{zh-hans:放射性降尘; zh-hant:輻射落塵;}-]]（「死灰」）的照射下而受害。之後美国将危险海域扩大，除了第五福龙丸外，当时在危险海域内作业的渔船數目不詳。据估计，當時受到氫彈試爆影響而被放射性物質污染的漁船可能有數百艘。

第五福龍丸遭受到氫彈試爆的辐射所污染的事件，引起當時日本國內激烈的[[反核運動|反核運動]]。由於擔心反核運動转变为反美運動，美國方面緊急與日本政府進行辐射被害者的補償交涉，提出總計200萬[[美元|美元]]的補償金額。附帶条款是「日本政府不要再追究美方責任」的担保書，以解決此事。

== 簡史 ==
[[File:Daigo_Fukuryū_Maru_01.JPG|thumb]]
第五福龍丸最初是在1947年時，在[[和歌山縣|和歌山縣]][[古座町|古座町]]以[[鰹魚|鰹魚]]漁船'''第七事代丸'''（<span lang="ja">だいななことしろまる</span>）的身份下水啟用，之後在[[靜岡縣|靜岡縣]][[燒津市|燒津市]]進行改造以後成為一艘鮪魚漁船，改名第五福龍丸。1954年3月1日在[[太平洋|太平洋]]上進行漁撈作業時遭受到核爆間接傷害，一直到3月14日返回母港燒津港時，鐵絲漁網都還處於張開的狀態沒有收回。在經過科學家的檢查之後，第五福龍丸被日本[[文部省|文部省]]收購，隔年8月移至[[東京水產大學|東京水產大學]]所屬的品川[[岸壁|岸壁]]泊靠。

之後，第五福龍丸旋即接受檢查並且去除[[放射線|放射線]]，並轉送至[[三重縣|三重縣]][[伊勢市|伊勢市]]的強力造船所進行改造，成為東京水產大學的練習船'''隼丸'''（<span lang="ja">はやぶさまる</span>），以[[千葉縣|千葉縣]][[館山市|館山市]]為母港。

1967年，由於船體老朽，擁有單位決定將隼丸報廢，在拆除掉一些還可以繼續使用的零件之後，將廢船身送到[[東京都|東京都]][[江東區_(東京)|江東區]][[夢之島|夢之島]]旁的第15號[[掩埋場|掩埋場]]拆解拋棄。但在同年，東京都的一些政府職員發現到它的存在而發起保存舊船的運動，將它安置在夢之島的[[第五福龍丸展示館|第五福龍丸展示館]]永久保存展示，直到今日。

== 以第五福龍丸为主題的作品 ==

1959年，第五福龍丸事件發生後的第5年，日本導演[[新藤兼人|新藤兼人]]拍攝了一部同樣名為《[[第五福龍丸_(電影)|第五福龍丸]]》的電影，由[[宇野重吉|宇野重吉]]飾演在災難發生半年後過世的無線電長久保山愛吉。

美籍画家[[班·夏恩|班·夏恩]]（Ben Shahn）以第五福龍丸為主題，創作了連環画《福龍》（<span lang="ja">ラッキードラゴン</span>，Lucky Dragon）。

著名日本画家、雕刻家[[岡本太郎|岡本太郎]]也以第五福龍丸為主題，創作了大型壁画《明日的神話》（<span lang="ja">明日の神話</span>），現收藏於東京都南青山的[[岡本太郎紀念館|岡本太郎紀念館]]。

日本著名的怪獸電影系列《[[哥吉拉|哥吉拉]]》（<span lang="ja">ゴジラ</span>）曾影射第五福龍丸事件，在故事中敘述因為核子試驗造成的污染，導致生物[[突變|突變]]而產生了巨大的怪獸。

日本紀錄片導演伊東英朗在2012年執導了《放射線を浴びた[Ｘ年後]》。該紀錄片主要揭露了第五福龍丸事件的秘辛。
