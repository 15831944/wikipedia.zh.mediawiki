'''第一島鏈'''是指東亞的海岸線往東向太平洋島嶼北起[[日本群島|日本群島]]、[[琉球群島|琉球群島]]，中接[[台灣|台灣]]，南至[[菲律賓|菲律賓]]、[[大巽他群島|大巽他群島]]的[[鏈形島嶼帶|鏈形島嶼帶]]之間的廣泛海域。位於[[朝鮮半島|朝鮮半島]]南方的[[大韓民國|大韓民國]]有的時候也會被視為第一島鏈的一部份。在1996年台海危機后兩岸關係相對平和的時期里，這種描述在中國大陸比較少見，而在台灣媒體中則一直佔有相對重要的意義。第一島鏈海域對於美軍防守中國人民解放軍是最有效益的區域。<ref name = "Defend the First Island Chain Proceedings Magazine - April 2014 Vol. 140/4/1,334_2018"> {{ cite web | title = Defend the First Island Chain Proceedings Magazine - April 2014 Vol. 140/4/1,334 | url = https://www.usni.org/magazines/proceedings/2014-04/defend-first-island-chain | publisher = The United States Naval Institute, 美國海軍學會 | author = By James R. Holmes | language = en | accessdate = 2018-07-28 }} </ref>

美國國防部對於第一島鏈的定義為，北從[[千島群島|千島群島]]開始，往南經過[[台灣|台灣]]，到[[婆羅洲|婆羅洲]]島，包括[[黃海|黃海]]、[[東海|東海]]、與[[南海|南海]]的西太平洋海域。<ref name = '2018年中國人民解放軍軍力報告'> {{ cite web | title = 2018年中國人民解放軍軍力報告 | url = https://media.defense.gov/2018/Aug/16/2001955282/-1/-1/1/2018-CHINA-MILITARY-POWER-REPORT.PDF | author = 美國國防部 | language = en | date = 2018-07-30 }} </ref>

== 地理位置 ==
第一島鏈的定義大約是指在[[中華人民共和國|中華人民共和國]]所主張及擁有的領海主權（海上邊界線），包括[[南海|南海]]的[[九段線|九段線]]及[[東海|東海]]的[[沖繩海槽|沖繩海槽]]（{{lang|ja|沖縄トラフ}}）。沖繩與中國的界溝在界乎[[日本|日本]]和[[台灣|台灣]]之間。

==戰略價值==
美軍將領[[道格拉斯·麥克阿瑟|道格拉斯·麥克阿瑟]]指出，美國在[[第二次世界大戰|第二次世界大戰]]前以從[[夏威夷|夏威夷]]、[[關島|關島]]到[[菲律賓|菲律賓]]的防線保護美國西海岸，但此防線易受攻擊，而二戰中也的確遭日本攻擊（台灣當時為日本領土，不在日本攻擊範圍內，而是遭到美國攻擊。可參見：[[台北大空襲|台北大空襲]]），二戰勝利使得美國可以把防線移到亞洲沿海，由美國與盟國控制的第一島鏈。<ref name="DuffyCarpenter1997">{{cite book|author1=Bernard K. Duffy|author2=Ronald H. Carpenter|title=Douglas MacArthur: Warrior as Wordsmith|url=https://books.google.com/books?id=PavSkTgefEMC&pg=PA178|year=1997|publisher=Greenwood Publishing Group|isbn=978-0-313-29148-7|pages=178–179}}{{en}}</ref>在第二次世界大戰與[[韓戰|-{zh-cn:朝鲜战争; zh-tw:韓戰; zh-hk:韓戰}-]]期間，他將位於第一島鏈中點的台灣譽為「[[不沉的航空母艦|不沉的航空母艦]]」<ref>{{Cite magazine | url=http://thediplomat.com/2011/03/why-taiwan-matters/ | title=Why Taiwan Matters | date=2011-03-08 | magazine=《[[外交家_(雜誌)|外交家]]》雜誌}}{{en}}</ref>，而[[中國大陸|中國大陸]]方面[[毛澤東|毛澤東]]也發動針對台灣的一連串作戰，但鎩羽而歸。

1950年代[[中華人民共和國|中華人民共和國]]方面試圖突破美軍在第一島鏈上的包圍網，但由于美国[[第七艦隊|第七艦隊]]和中華民國方面所占据的军事优势，中华人民共和国籍船长达三十年无法在大陆南北港口间通行。1968年，中华人民共和国方面以绕行[[太平洋|太平洋]]的方式，[[打通南北航线|打通南北航线]]。1979年，才在[[台湾海峡复航|台湾海峡复航]]。

[[冷戰|冷戰]]期間，美國的艾森豪總統於[[朝鲜战争|朝鲜战争]]後，與中華民國簽訂《中美共同防禦條約》（現已中止）正式將台灣納入第一島鏈中，[[美國|美國]]為阻止[[共產黨|共產黨]]勢力擴張，而在[[亞洲|亞洲]]東部島嶼協助建立[[反共|反共]]的政權，同時提供軍力以及其他各種形式上的援助。

台灣與菲律賓的國防在第一島鏈上最為脆弱（台灣没有正式的国际地位，菲律宾军事力量较弱），加上一連串的政治經濟外交因素，近20年來國防疲弱不振，加上其他東南亞地區經濟依賴中國越來越深，使得中國方面得以找到突破口滲透，引來[[美國|美國]]、日韓澳等國顧忌，因此美中兩國在第一島鏈上仍維持著「恐怖平衡」的狀態。

2014年4月{{tsl|en|United States Naval Institute|美國海軍學會}}評估在第一島鏈是美國防守與反擊中國人民解放軍侵略其他國家最有效益的戰略位置，不僅可以倚靠第一島鏈位置阻隔人民解放軍輕易進入西太平洋，也可以由第一島鏈各國的防守讓美軍可預測中國人民解放軍的兵力配置，美軍與第一島鏈國家陸地上的防空與反艦軍力配置合作在於美軍自由航行於第一島鏈區塊的周邊海域，美軍可以藉此有效防守並反擊中國人民解放軍的襲擊，如果第一島鏈國家也發展海中的水雷、[[潛水艇|潛水艇]]、快速移動的軍艦，那中國人民解放軍就更沒有勝算，並且對於中國人民解放軍來說要對付這些反制措施需要付出更昂貴的艦艇機械與維護費用人力。<ref name = "Defend the First Island Chain Proceedings Magazine - April 2014 Vol. 140/4/1,334_2018" />

2016年12月9日，新浪新聞網列出美軍在第一島鏈相關國家的軍事武力配置，包括在[[日本|日本]]的航空母艦艦隊停靠的海軍基地與偵察機群，美軍最先進的F-35、F-22與F-15、F-16戰鬥機群共計達110個基地、[[南韓|南韓]]因為受北韓導彈威脅已部署偵測範圍達到3,500公里導彈的[[薩德反飛彈系統|薩德反飛彈系統]]並進入作戰狀態、美軍與[[菲律賓|菲律賓]]及[[泰國|泰國]]的年度常態軍事演習與合作，支援第一島鏈的關島第二島鏈美軍擁有[[薩德反飛彈系統|薩德反飛彈系統]]與持續擴增中的轟炸機群<ref name = "中国还需更多航母突破岛链 美竟有400个基地对华包围_2016"> {{ cite web | title = 中国还需更多航母突破岛链 美竟有400个基地对华包围 | url = http://mil.news.sina.com.cn/jssd/2016-12-29/doc-ifxzczff3390427.shtml | publisher = 新浪新闻 | language = zh-cn | date = 2016-12-29 }} </ref>。

===台灣===
在第一島鏈的「封鎖鏈條」中，最為關鍵的是[[台灣島|台灣島]]。它位於第一島鏈的的中間，具有極特殊的戰略地位，掌握台灣島就能有效地遏止東海與南海的咽喉戰略通道，也有與[[第二島鏈|第二島鏈]]內海域的有利航道及走向遠洋的便捷之路。

===中國大陸===
對於[[日本|日本]]警惕中國突破「第一島鏈」，[[中國人民解放軍國防大學|中國人民解放軍國防大學]]戰略研究所孟祥青教授接受《[[環球時報|環球時報]]》採訪時表示：「日本對中國走向海洋確有不適感，因為日本是傳統[[島國|島國]]，狹隘的島民心態在這種不適中表現得淋漓盡致。」<ref>[http://hk.crntt.com/doc/1010/2/6/5/101026578.html?coluid=7&kindid=0&docid=101026578 專家：日本擔憂中國突破第一島鏈] 中評社 2009年7月19日，孟祥青教授認為，「日本民眾認為日本歷史上是海洋國家，而中國是大陸國家，這種觀點本身就是錯誤，因為中國是陸海兼備的國家——地面上，中國背陸面海；歷史上，[[鄭和下西洋|鄭和下西洋]]，以及2500年前中國漁民遠赴[[南海|南海]]都證明中國是陸海兼備國家。只是由於[[明朝|封建政權]]閉關鎖國，以及[[帝國主義|帝國主義]]長期封鎖，所以才造成中國貌似「陸地國家」。中國走向海洋，恢復陸海兼備國家其實是歷史必然。」</ref>

日本問題專家[[高洪|高洪]]認為，所謂島鏈是美日同盟體制下圍堵中國的一種戰略謀劃，第一島鏈就是希望阻止中國走向深水。這只是一種傳統安全下的考慮，日本​​更多是藉這種安排增加一些信心。但事實上中國有很合理的理由來突破這一島鏈，中國完全可以按照國際慣例、[[海洋法公約|海洋法公約]]中的相關規定“無害通過”。

中國人民解放軍對第一島鏈的威脅主要靠陸基導彈與增建航空母艦海軍艦隊。<ref name = "中国还需更多航母突破岛链 美竟有400个基地对华包围_2016" />

2013年9月中國人民解放軍海軍的海上轟炸機訓練，配備短距離反艦飛彈的轟6G轟炸機首次到第一島鏈的巴士海峽演習，同年人民解放軍空軍開始改用飛行距離較長，配備長距離導彈的轟6K轟炸機，2015年開始轟6K轟炸機開始越過第一島鏈的巴士海峽與宮古海峽到達西太平洋，2016年轟6K轟炸機開始由空中預警機以及戰鬥機伴飛獲得空中防禦能力，並且環繞台灣一週到2017年人民解放軍增加轟6K轟炸機環繞台灣島的次數，人民解放軍的轟炸機已經有鎖定台灣島作戰，以及突破第一島鏈進行作戰能力，威脅在西太平洋的美國關島與美國友邦等國家基地的安全。<ref name = '2018年中國人民解放軍軍力報告' />

=== 日本 ===
[[日本內閣|日本內閣]]批准的[[防衛白皮書|防衛白皮書]]強調「[[中国人民解放军海军|中國海軍]]威脅」。在日本有關中国人民解放军海军活動頻繁的報導中，不少都提到太平洋「第一島鏈」和「第二島鏈」，中国人民解放军海军的動向也往往與突破島鏈聯繫起來。<ref>在日本著名雜誌《世界的艦船》中，每期都刊登經過日本海域到第二島鏈以外活動的中國船隻照片。在日本防務論壇的讀者留言中，大量日本讀者表達自己的不滿。一個叫「國政鳥瞰所」的[[博客|博客]]上寫道：「不管怎麼看中國都是一個大陸國家，非要自己定義成海洋國家，這不是開玩笑吧。對於東亞局勢的緊迫，日本國家的行動緩慢，這真讓人感到煩躁不安。」</ref>

[[香港|香港]]《[[亞洲周刊|亞洲周刊]]》2009年7月發表文章稱，日本通過在[[與那國島|與那國島]]駐軍的方式將軍力向中國推進500公里，不但強化對[[釣魚台列嶼|釣魚台列嶼]]的實際控制，完善對中國大陸軍事圍堵的第一島鏈，而且也為在與中國大陸可能的衝突中「先發製人」做好準備。<ref>文章摘要如下:日本防衛省表示，日本將向處於最西端的與那國島派駐一支陸軍部隊，以加強對被喻為日本西南邊防“重鎮”及其釣魚島等西南島嶼的保衛。這項派兵屯守計劃，作為“國家西南防禦部署的重要一環”，將被列入未來五年的“新防衛計劃大綱”。據日本軍方消息人士透露，首批派駐部隊隸屬於駐守在沖繩那霸的陸上自衛隊第一混合旅團，規模不是很大，官兵人數大概有50人左右。作為一支離島特殊連隊主要分為雷達偵察和音聲監聽兩個班，其主要職責是加強對尖閣諸島及其附近海域的監視與情報收集。而選擇駐軍與那國島，除了它與釣魚島近，只有約170公里外，還有島內具有跑道長2000米的機場，遇到“釣魚島有事”等緊急情況，可以迅速空運支持軍力和隨時起降被稱為“潛艇殺手”的新銳P3C反潛攻擊機。另外，與那國島正在擴建的港口將來也能停靠軍艦等，而鄰近的先島群島業已完成補給基地建設。因此，與那國島事實上將成為日本進一步控制釣魚島，應對西南島嶼有事的第一前哨陣地，也意味著日本在軍事戰略南移中，進一步強化從九州島島南線至琉球群島西南線的第一島鏈的駐軍布防。</ref>

日本在第一島鏈戰略位置歷史從美日聯合防堵蘇聯擴張開始，中國人民解放軍目前海軍戰略在模仿冷戰時期蘇聯海軍的防禦思維，蘇聯海軍在冷戰時就劃下[[鄂霍次克海|鄂霍次克海]]紅線警告周邊國家禁止進入，中國人民解放軍的戰略主要想模仿此方式，控制南海、東海與黃海等位於在第一島鏈與第二島鏈相關聯海域；也因此中國人民解放軍使用很多蘇聯冷戰時期使用的設備與武器。日本自衛隊在面對中國時主要是扮演保護美軍基地與保存戰力的角色，對於日本國土保護自衛隊以日本南部鄰近黃海與東海的島嶼為主，日本有較強的反潛、防空與水雷技術等軍事優勢，之前與蘇聯競爭的經驗讓日本有較清楚的方向與美軍合作面對中國人民解放軍的海空軍。<ref name = "The Future of Sino-Japanese Competition at Sea_2012"> {{ cite web | title = The Future of Sino-Japanese Competition at Sea | url = https://www.nippon.com/en/in-depth/a00504/ | publisher = nippon.com | language = en | date = 2012-03-23 }} </ref>
