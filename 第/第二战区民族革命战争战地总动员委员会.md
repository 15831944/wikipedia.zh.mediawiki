'''第二战区民族革命战争战地总动员委员会'''（简称“战动总会”）是经[[阎锡山|阎锡山]]同意，由[[中国共产党|中国共产党]]、阎锡山方面及其他方面联合组成的[[抗日战争第二战区|第二战区]]抗日动员组织<ref name=zuzhi>{{Cite web |url = http://www.hprc.org.cn/gsw/detail_zzslk.jsp?channelid=75034&record=572   |title = 中国共产党组织史资料  |author =  |date = 2000年 |publisher = 中共党史出版社 |language =  |accessdate =  |quote = 第三卷 第三编　地方组织机构  西北地区 第十六章　中共中央北方局直属晋绥地区各特委、工委 }}</ref><ref name=su>苏振兰，华北敌后模范的统一战线组织——记第二战区民族革命战争战地总动员委员会，湘潮2008年第7期</ref>。

==历史==
1937年7月[[七七事变|七七事变]]发生，[[抗日战争|抗日战争]]全面爆发。[[中共中央|中共中央]]派出[[周恩来|周恩来]]、[[刘少奇|刘少奇]]、[[朱德|朱德]]、[[彭德怀|彭德怀]]、[[邓小平|邓小平]]、[[徐向前|徐向前]]、[[彭雪枫|彭雪枫]]、[[程子华|程子华]]、[[南汉宸|南汉宸]]赴[[山西省|山西省]]同阎锡山商谈合作抗日事宜<ref name=su/>。1937年9月，根据[[中共中央|中共中央]]决定，[[周恩来|周恩来]]赴[[山西省|山西省]]同阎锡山多次会谈[[八路军|八路军]]开入山西省配合作战事项，先在太和岭前线谈，后在太原谈<ref name=zuzhi/><ref name=su/>。会谈期间，周恩来向阎锡山提议在第二战区范围内成立第二战区民族革命战争战地总动员委员会。阎锡山同意在日本占领地区开展全民总动员，以便配合正面战场展开[[游击战|游击战]]，并成立抗日根据地。双方协商确定战动总会的活动范围是：[[雁门关|雁门关]]外18县（[[天镇县|天镇县]]、[[阳高县|阳高县]]、[[大同县|大同县]]、[[怀仁县|怀仁县]]、[[广灵县|广灵县]]、[[灵邱县|灵邱县]]、[[浑源县|浑源县]]、[[应县|应县]]、[[山阴县|山阴县]]、[[朔县|朔县]]、[[平鲁县|平鲁县]]、[[左云县|左云县]]、[[右玉县|右玉县]]、[[繁峙县|繁峙县]]、[[代县|代县]]、[[宁武县|宁武县]]、[[神池县|神池县]]、[[偏关县|偏关县]]）、察南5县（[[蔚县|蔚县]]、[[阳原县|阳原县]]、[[怀安县|怀安县]]、[[宣化县|宣化县]]、[[涿鹿县|涿鹿县]]）以及[[绥远省_(中华民国)|绥远省]]。后来由于日军不断进攻，战区扩大至忻口，故又增加晋北12县（[[五台县|五台县]]、[[定襄县|定襄县]]、[[静乐县|静乐县]]、[[岢岚县|岢岚县]]、[[岚县|岚县]]、[[忻县|忻县]]、[[兴县|兴县]]、[[保德县|保德县]]、[[河曲县|河曲县]]、[[临县|临县]]、[[五寨县|五寨县]]、[[崞县|崞县]]）<ref name=zuzhi/>。

1937年9月20日，战动总会在太原成立<ref name=su/>。根据双方的协议，战动总会由25位委员组成。其中[[八路军|八路军]]代表4人，阎锡山方面代表3人，社会名流5人，东北救亡总会学联代表1人，[[中国国民党|中国国民党]][[改组派|改组派]]人士2人，[[察哈尔省政府|察哈尔省政府]]主席的代表2人，绥远省代表2人，[[国民革命军第三十五军|国民革命军第三十五军]]代表1人，山西第二行政区专员1人，山西第四行政区专员1人，[[牺牲救国同盟会|牺牲救国同盟会]]代表1人，[[主张公道团|主张公道团]]代表2人<ref name=zuzhi/>。

中共中央派出[[邓小平|邓小平]]、[[彭雪枫|彭雪枫]]、[[程子华|程子华]]、[[南汉宸|南汉宸]]作为八路军代表，另外有许多中共党员出任战动总会各部部长以及各地动委会负责人。中共在战动总会内部秘密成立了领导机构——战动总会党团，书记为程子华（1938年底程子华调离，南汉宸继任书记<ref name=su/>），委员为南汉宸、[[武新宇|武新宇]]。战动总会党团领导中共党员执行中共的[[抗日民族统一战线|抗日民族统一战线]]政策及战动总会工作纲领。战动总会工作纲领的要点为：（一）组织并武装民众，开展游击战；（二）实行[[减租减息|减租减息]]，合理负担，改善人民生活；（三）实行民主政治，扶植抗日言论、出版、集会之自由<ref name=zuzhi/>。

战动总会成立后，受到八路军主力支持，不断扩大活动范围，在晋西北、晋东北、晋中、晋东南、察南、雁北、冀西、绥中、绥东、绥西的大约70个县内，普遍成立了县、区、村动委会，并且设立[[晋绥边区|晋绥边区]]办事处、晋察绥边区工作委员会、察南雁北敌区工作团。各地动委会帮助中共地方党组织在上述各地改造旧政权，成立抗日政权，并且成立工人救国会、农民救国会、青年救国会、妇女救国会、[[抗日儿童团|抗日儿童团]]等抗日组织<ref name=zuzhi/>。

在战动总会武装部领导下，还成立了20余支地方游击队，约2万多人，在晋中盆地、北[[同蒲铁路|同蒲铁路]]两侧、[[大青山_(内蒙古)|大青山]]、察南地区活动。1937年9月，晋中的游击支队组成抗日游击第一路纵队。1938年3月，阎锡山命令将战动总会下辖的游击队编成保安二区司令部下辖的8个游击支队及政治保卫队。1939年夏，撤销保安二区司令部，将其下属部队改编成陆军暂编第一师<ref name=zuzhi/>。

日军占领同蒲铁路全线之后，不但开展军事进攻，还同时向[[蒋介石|蒋介石]]、阎锡山开展政治诱降。蒋介石、阎锡山此时也对中共领导下的民众抗日感到警觉，开始压制战动总会的抗日活动及其领导下的抗日武装。1939年7月，阎锡山以国民政府命令取消一切名义的动员机关为借口，下令取消了战动总会及各级动委会。中共中央也同意结束战动总会的工作。战动总会结束后，1940年4月到1942年12月，战动总会下辖的陆军暂编第一师先后经三次改编，最终编入八路军序列；在察南、雁北的察绥游击军先后改编成察绥游击支队、雁北支队，最后并入八路军晋察冀军区第五军分区。在大青山的太原成成中学师生游击队（四支队）参与开辟了[[大青山抗日根据地|大青山抗日根据地]]<ref name=zuzhi/>。

===下辖察绥游击队=== 
*察绥游击军（1937年9月—1938年10月）
察绥游击军1937年9月在太原成立，10月随战动总会至[[离石县|离石县]]，组成第一支队。1937年12月，第一支队开赴晋西北，归[[宋时轮|宋时轮]]支队领导。后来又陆续组成第二、四、五、十五支队，开赴[[晋察冀边区|晋察冀边区]]。 
**军长：张砺生 
**政治部长：阮慕韩→[[陈凤桐|陈凤桐]]

*察绥游击纵队（1938年10月—1939年1月）
因察绥游击军四支队骑兵二大队叛变，军部及四支队、五支队遭袭溃散。经八路军[[120师|一二○师]][[359旅|三五九旅]]帮助，残余部队改编为察绥游击纵队，下辖第一支队、二支队、十五支队。 
**纵队长：[[刘苏|刘苏]]

*察绥独立游击支队（1939年1月—9月）
因察绥游击纵队二支队二大队长叛变，打死二支队支队长[[张克恭|张克恭]]，第一支队、二支队、十五支队等部队合编成察绥独立游击支队。 
**支队长：刘苏
**政治委员：[[陈凤桐|陈凤桐]] 

*雁北支队（1939年9月—1940年5月）
1939年9月，察绥游击支队和八路军一二○师三五九旅雁北支队合编成雁北支队。1940年5月间，雁北支队内部原属三五九旅所部奉调回[[延安|延安]]，雁北支队余部划归晋察冀军区第五军分区。 
**司令员：刘苏
**政治委员：陈凤桐
**副司令员：[[徐国贤|徐国贤]]
**副政治委员：[[谭文邦|谭文邦]]<ref name=zuzhi/>
