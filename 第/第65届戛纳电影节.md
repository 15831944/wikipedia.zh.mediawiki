'''第65届[[戛纳电影节|戛纳电影节]]'''于2012年5月16日至5月27日在[[戛纳|戛纳]]举办<ref name="festival-cannes.com">{{cite web |url=http://www.festival-cannes.com/en.html |title=Festival de Cannes 2012 |accessdate=2011-12-11|work=festival-cannes.com}}</ref><ref name="marchedufilm">{{cite web |url=http://www.marchedufilm.com/ |title=Festival 2012 |accessdate=2011-12-11|work=marchedufilm}}</ref>。[[意大利|意大利]]电影导演[[南尼·莫莱蒂|南尼·莫莱蒂]]担任主竞赛单元评审团主席<ref name="President">{{cite web |url=http://latimesblogs.latimes.com/movies/2012/01/cannes-jury-president-nanni-moretti.html?utm_medium=twitter&utm_source=twitterfeed |title=Nanni Moretti named 2012 Cannes jury president |accessdate=2012-01-21|work=LA Times}}</ref>。英国男演员[[蒂姆·罗斯|蒂姆·罗斯]]担任一種注目单元评审主席<ref name="RothRegard">{{cite web |url=http://www.bbc.co.uk/news/entertainment-arts-17699698 |title=Tim Roth to lead Cannes Un Certain Regard jury |accessdate=2012-04-13|work=BBC News}}</ref>。法国女演员[[贝热尼丝·贝乔|贝热尼丝·贝乔]]主持开幕与闭幕典礼<ref name="Bejo">{{cite web |url=http://www.telegraph.co.uk/culture/film/cannes-film-festival/9211616/The-Artist-star-to-host-Cannes-film-ceremonies.html |title=''The Artist'' star to host Cannes film ceremonies |accessdate=2012-04-19 |work=The Telegraph}}</ref>。

[[奥地利|奥地利]]导演[[迈克尔·哈内克|迈克尔·哈内克]]的影片《[[爱_(法语电影)|爱]]》继2009年的《[[白色緞带|白色緞带]]》后再度斩获[[金棕榈奖|金棕榈奖]]；意大利导演[[马提欧·加洛尼|马提欧·加洛尼]]执导的《[[现实_(电影)|现实]]》获评审团大奖；执导《[[柳暗花明_(2012年电影)|柳暗花明]]》的[[墨西哥|墨西哥]]导演[[卡洛斯·雷加达斯|卡洛斯·雷加达斯]]夺得最佳导演奖；[[罗马尼亚|罗马尼亚]]导演[[克里斯蒂安·蒙吉|克里斯蒂安·蒙吉]]编剧、执导的《[[山之外|山之外]]》获得最佳编剧和最佳女演员（两位女主演分享）两个奖；[[丹麦|丹麦]]演员[[麦德斯·米科尔森|麦德斯·米科尔森]]凭借《[[捕猎_(电影)|捕猎]]》获得最佳男演员奖。
