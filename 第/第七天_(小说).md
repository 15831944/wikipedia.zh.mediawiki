《'''第七天'''》，作家[[余华|余华]]在2013年出版的[[长篇小说|长篇小说]]。

== 内容 ==
该书描写1个走向火葬场的死人（杨飞）在死后7天中的人生经历。

== 风格 ==
该书采用西方创作技巧，开篇即引用[[基督教|基督教]]《[[圣经|圣经]]》的《旧约·创世记》，通过“荒诞”写法描述现实生活。<ref name=" fish" >{{cite web |url=http://news.gmw.cn/newspaper/2013-07/27/content_1857155.htm |title=《第七天》：荒诞的绝望？ |accessdate=2013 |publisher=光明网 |work= |language=zh-cn}}</ref>

== 评价 ==
对于有读者把该书比作《[[百年孤独|百年孤独]]》，余华回复说：“[[加西亚·马尔克斯|加西亚·马尔克斯]]的《百年孤独》里写了很多当时[[哥伦比亚|哥伦比亚]]报纸上的事件和话题，他说他走到街上，就有读者对他说：你写得太真实了。《第七天》不能和《百年孤独》比，人家写下的是一百年的孤独，我只是写下七天的孤独，而且人家的一百年只用了二十多万字，我的七天花费了十三万字。我深感惭愧。”<ref name=" fish" />

也有读者批评该书是“《第七天》就不过是近年来的社会新闻集锦或什锦或拼盘”。<ref>{{cite web |url=http://finance.sina.com.cn/roll/20130725/023016232813.shtml |title=写在《第七天》之后  |accessdate=2013年7月29日 |publisher=新浪网 |work= |language=zh-cn}}</ref>

[[中国作家网|中国作家网]]的专文称该书“[[强拆|强拆]]、卖肾、[[火灾|火灾]]、瞒报、[[车祸|车祸]]、地陷、冤狱、杀警、“伪娘”卖淫、死婴等”是“新闻串烧”、“微博汇编”，并且“恶评如潮”。<ref>{{cite web|url=http://news.xinhuanet.com/book/2013-07/17/c_125023501.htm|title=《第七天》为何遭遇“恶评如潮”？|accessdate=2013年7月29日|publisher=新华网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20130721183509/http://news.xinhuanet.com/book/2013-07/17/c_125023501.htm|archivedate=2013年7月21日}}</ref>
