'''美國第四艦隊'''（U. S. 4th Fleet），隸屬於[[美國南方司令部|美國南方司令部]]的艦隊，於2008年7月1日再次成立。重新組建的第四艦隊將以在[[佛羅里達州|佛羅里達州]]的南方海軍司令部所在地[[傑克遜維爾_(佛羅里達州)|傑克遜維爾]][[梅波特港|梅波特港]]為基地。

== 歷史 ==
曾是[[美國海軍|美國海軍]]的艦隊之一。在1943年3月時成立，[[第二次世界大戰|第二次世界大戰]]時主要在[[南大西洋|南大西洋]]執行封鎖和防衛敵軍潛艇的任務。1950年時解散，併入[[第二艦隊_(美國海軍)|第二艦隊]]之中。

58年後，為了因應日益左傾的[[拉丁美洲|拉丁美洲]]及加大美國對拉美國家的海上威懾力<ref>主要依據[[聯合報|聯合報]]2008年5月11日的新聞，網址見[http://udn.com/NEWS/WORLD/WOR3/4335782.shtml]。</ref>，美國的海軍作戰部長[[羅格海德|羅格海德]]上將（Gary Roughead）在2008年4月24日，宣布重建第四艦隊。第四艦隊主要由美國南方司令部（SOUTHCOM）的海軍部隊所構成，2008年7月1日正式成立。以[[佛州|佛州]][[梅波特港|梅波特港]]為基地。不同於其他艦隊由海軍中將指揮，第四艦隊則是由海軍少將指揮。

美國媒體認為，重建第四艦隊的舉措是為了加強美國對加勒比和拉丁美洲地區國家的影響力。而[[古巴|古巴]]精神領袖[[卡斯楚|卡斯楚]]、[[玻利維亞|玻利維亞]]總統[[埃沃·莫拉萊斯|埃沃·莫拉萊斯]]對此均強烈反對。卡斯楚抨擊此為「砲艦外交」（gun-boat diplomacy）死灰復燃，莫拉萊斯則批評其為「第四干預艦隊」。

== 實力簡介 ==
[[File:USN_Fleets_(2009).png|thumb]]
第四艦隊目前併同[[美國特種作戰司令部|美國特種作戰司令部]]行動。其任務以提供海上保安、人道援助、救災、緝毒等為止，並不配備攻擊戰力，因此沒有[[航空母艦|航空母艦]]與大型戰艦，更無固定配置的軍艦，均自其他艦隊輪流抽調。

== 轄區範圍 ==
任務區範圍從美國佛州一直延伸到[[南美洲|南美洲]]最南端的[[合恩角|合恩角]]的中南美洲水域，涉及30多個國家，其中的重點區域是[[加勒比海|加勒比海]]、[[墨西哥灣|墨西哥灣]]、[[中美洲|中美洲]]及[[南美洲|南美洲]][[大西洋|大西洋]]側海域。
