'''美国第七舰队'''（{{lang-en|United States Seventh Fleet}}）是[[美國海軍|美國海軍]]旗下的遠洋艦隊之一，隶属于[[美國印太司令部|美國印太司令部]]属下的[[美國太平洋艦隊|太平洋舰队]]。第七艦隊司令部设在[[日本|日本]]的[[橫須賀市|橫須賀市]][[橫須賀海軍設施|横须贺港]]，驻地包括[[日本|日本]][[佐世保基地|佐世保基地]]、[[冲绳|冲绳]]和[[韩国|韩国]][[釜山|釜山]]、[[浦项|浦项]]、[[鎮海海軍基地|镇海基地]]、[[新加坡|新加坡]]及[[菲律賓|菲律賓]]等地，是目前美国最大的海外前線投送部队。第七艦隊的主要任務包括威嚇敵國、與盟國合作保护海上通道、协防美國友邦領土以及在必要时动用武力，也经常充当美国对外進行军事行動時的機動艦隊。<ref> {{Cite book | author = 施孝瑋 | title = 《西太平洋自由之盾：美國海軍第七艦隊》全球防衛雜誌 | location = 台灣 | language = zh-hant }} </ref>

== 组建历史 ==
1941年[[二戰|二戰]][[太平洋戰爭|太平洋戰爭]]爆發，1943年3月美國將西南太平洋的艦隊改編為第七艦隊。第七艦隊在1944年秋的[[雷伊泰灣海戰|雷伊泰灣海戰]]中歸屬於[[麥克阿瑟將軍|麥克阿瑟將軍]]指揮的西南太平洋方面軍，當時的艦隊司令是[[托馬斯·金凱德|托馬斯·金凱德]]中將。1947年1月，這支部隊又被改稱為西太平洋海軍部隊，1949年8月被稱為第七特遣艦隊，直到1950年2月才又正式恢復第七艦隊的舊稱至今，參加過1950年至1953年的[[朝鮮戰爭|朝鮮戰爭]]和1965年至1975年的[[越南戰爭|越南戰爭]]，1955年的[[大陳島撤退|大陳島撤退]]中和[[中华民国军事|國軍]]合力進行浙江省的大陳島上軍民的撤退行動，主要為大陳島的居民撤退護航，1958年的福建省金門縣[[八二三砲戰|八二三砲戰]]中曾為[[中華民國海軍|中華民國海軍]]補給船護航，1991年參加[[波斯灣戰爭|波斯灣戰爭]]等。

[[冷戰時期|冷戰時期]]，隨著《[[中美共同防禦條約|中美共同防禦條約]]》在華盛頓正式簽字，中美軍事協調會議確認第七艦隊在台北設立「台灣聯絡中心」，第七艦隊在台灣的分遣隊（{{lang|en|7th Fleet Detachment}}）也利用[[左營海軍基地|左營海軍基地]]作為駐點，並透過[[高雄港|高雄港]]運補物提供物資{{R|港史館}}，直至1970年代後期[[中華民國－美國關係|斷交]]為止。1996年，中華民國舉行[[1996年中華民國總統選舉|首次總統民選]]，[[中国大陆|中国大陆]]方面對台灣本島[[基隆|基隆]]、[[高雄|高雄]]的外海發射短程彈道飛彈，[[台湾地区|台湾地区]]均陷入戰爭爆發的恐慌中，導致[[美國總統|美國總統]][[比爾·柯林頓|比爾·柯林頓]]下令第七艦隊開進台灣海峽。往後，台灣每屆總統大選，美國為了防止中国大陆以武力干涉台灣選舉，會在每屆中華民國總統選舉到就職典禮那一段期間，常態性的巡弋台灣海峽；[[2012年中華民國總統選舉|2012年中華民國總統選舉]]後，台灣海峽情勢較往年緩和，美軍第七艦隊並未在台灣沿海有大規模活動，仍以日韓兩地當作主基地，但一部分兵力改於[[菲律賓|菲律賓]]和澳洲北部的[[達爾文港|達爾文港]]駐紮。

2010年，朝鮮半島情勢緊張，爆發了[[天安艦事件|天安艦事件]]與[[延坪島砲擊事件|延坪島砲擊事件]]，南韓蒙受其害；加上20世紀末以來的[[朝核問題|朝核問題]]，日韓兩國對北韓的核生化武器感到高度恐慌，因此造成美國高度關切，經常在[[黃海|黃海]]一帶進行美日韓三國聯合演習，藉此警告[[朝鲜|朝鲜]]。

== 編制 ==
[[File:USS_blueridge.jpg|thumbnail]]]]
第七舰队大约拥有50至60艘[[军舰|军舰]]、350架[[战斗机|战机]]，舰队满员编制6万人，其中包括38,000名[[海军|海军]]官兵和22,000名[[海軍陸戰隊|海军陆战队]]员，平时总兵力约2万人。一般由CVN-76隆納·雷根號[[航空母舰|航空母舰]]组成的[[航空母艦戰鬥群|航空母艦戰鬥群]]为主要作战部队，辅以陆基航空兵、两栖部队。必要时可以从第三舰队或其他部队抽调舰艇补充。现辖旗舰一艘（[[藍嶺號兩棲指揮艦_(LCC-19)|蓝岭号]]）、核动力航空母艦一艘（[[隆納·雷根號航空母艦|隆納·雷根號]]），3至4艘配备导弹的巡洋舰、18至20艘导弹驱逐舰及护卫舰、5至6艘攻击型潜艇以及5至8艘登陆舰、18艘后勤保障船和若干汽垫船、辅助船只等。现任艦隊司令為小羅勃·L·湯瑪斯中將（Robert L. Thomas, Jr.）{{R|Commander}}。 

第七艦隊下轄以下的特遣指揮编制{{R|Structure}}{{R|C7F}}：
* 第七舰队战斗指揮部／第70特遣隊（Commander, Battle Force, 7th Fleet / CTF-70）：第七艦隊的主幹，其組成包含兩部分，分別是以隆納·雷根號為中心的第五航空母艦打擊群（CSG 5）與其所搭載的第五艦載機聯隊（CVW-5），以及由驅逐艦組成的{{link-en|第15驅逐艦戰隊|Destroyer Squadron 15}}（CDS-15）。驻地日本横须贺{{R|CTF-70}}。
* 第一海軍特戰單位指揮部／第71特遣隊（Commander, Navy Special Warfare Unit One / CTF-71）：CTF-71主要是由第一海軍特戰部隊（Commander, Navy Special Warfare, Unit One，NSWU-1）與{{link-en|美國海軍爆破物處理機動隊|United States Navy EOD|第5海軍爆破物處理機動隊}}（EODMU 5）所組成，駐地{{link-en|關島海軍基地|Naval Base Guam|關島}}。
* 第七舰队巡邏偵察指揮部／第72特遣隊（Commander, Patrol & Reconnaissance Force, 7th Fleet / CTF-72）：由數支陸基的反潛與偵察部隊所組成，駐地包括日本沖繩的[[嘉手纳空军基地|嘉手納]]與[[青森縣|青森縣]][[三澤飛行場|三澤海軍基地]]，司令部設於[[神奈川县|神奈川县]][[厚木海軍飛行場|厚木海軍基地]]。
* 西太平洋後勤群指揮部／第73特遣隊（Commander, Logistics Group, Western Pacific, Singapore Area Coordinator / CTF-73）
* 第七艦隊潛艇指揮部／第74特遣隊（Commander, Submarine Force, 7th Fleet / CTF-74）：驻地日本横须贺。
* 海軍遠征部隊太平洋指揮部／第75特遣隊（Commander, Navy Expeditionary Forces Command Pacific, 7th Fleet / CTF-75）
* {{link-en|第七艦隊兩棲作戰指揮部|Task Force 76}}／第76特遣隊（Commander, Amphibious Force, 7th Fleet / CTF-76）：驻地日本[[沖繩縣|沖繩]]
* {{link-en|美國海軍駐韓指揮部|Commander Naval Forces Korea}}／第78特遣隊（Commander, U.S. Naval Forces, Korea / CTF-78）：又常縮寫為CNFK，CTF-78是美國海軍在[[大韓民國|大韓民國]]的駐紮代表，其總部設於[[首爾|首爾]]市郊的[[龍山基地|龍山基地]]，並以[[慶尚南道|慶尚南道]]的[[鎮海海軍基地|鎮海海軍基地]]作為主要駐地。
* 第七艦隊登陸部隊指揮部／第79特遣隊（Commander, Landing Force, 7th Fleet / CTF-79）：驻地日本冲绳。

第七艦隊另有數艘[[派里級巡防艦|派里級巡防艦]]、數量不明的機動[[飛彈快艇|飛彈快艇]]等，駐紮於[[關島|關島]]、[[中途島|中途島]]等美軍基地，用以平時巡衛用，如遇到盟國或友軍船隻、軍艦會協航甚至保護之。

== 辖区范围 ==
[[File:USN_Fleets_(2007).png|thumbnail]]
[[File:USN_Fleets_(2009).png|thumb]]
第七舰队的作战区域东起[[國際換日線|国际日期变更线]]以西的[[太平洋|太平洋]]和[[印度洋|印度洋]]，西至非洲东岸[[红海|红海]]（不包括[[波斯湾|波斯湾]]），南达印度洋及[[南极|南极]]，北至[[白令海峡|白令海峡]]。整个作战区域1.3468億平方公里，比美国本土大14倍。这一区域内居住着全世界大约一半的人口，而这其中的80%又都居住在离海岸线800公里以内的陆地上。

== 其他 ==
雖然是美國海軍麾下戰力最強大的艦隊之一，但第七艦隊卻是美國最後一個只有配備傳統動力航空母艦的艦隊。之所以如此配置，主要的原因是艦隊主要駐紮地[[日本|日本]]，乃是世界上唯一曾遭受[[美國|美國]]核子武器攻擊的國家，日本人民普遍對於核子動力的軍事武器帶有負面的觀感，為了尊重日本人民的感觸，因此美方特別配置了傳統動力航空母艦[[小鷹號|小鷹號]]作為第七艦隊的核心。然而，由於小鷹號已在2008年時除役，因此接任小鷹號除役後空缺的[[喬治·華盛頓號航空母艦|喬治·華盛頓號航空母艦]]，也順勢成為第一艘泊靠並駐紮於日本本土的核子動力航空母艦{{R|BBC20051028}}。喬治·華盛頓號於2008年至2015年駐紮於日本，之後返回美國歲修，由[[隆納·雷根號航空母艦|隆納·雷根號航空母艦]]接替。
