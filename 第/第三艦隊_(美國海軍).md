'''美國第三艦隊'''（'''United States Third Fleet'''）是[[美國海軍|美國海軍]]六大艦隊之一。轄區範圍在東部及北[[太平洋|太平洋]]海域一帶面積約五千萬平方公里（包含[[白令海|白令海]]、[[阿拉斯加|阿拉斯加]]、[[阿留申群島|阿留申群島]]及部分北極），司令部設置在[[美國西部|美國西部]][[加利福尼亞州|加利福尼亞州]][[聖地牙哥_(加利福尼亞州)|聖地牙哥]]的{{Link-en|洛瑪角海軍基地|Naval Base Point Loma}}。第三艦隊在確保美國與友邦在整個環太平洋的海上貿易及石油交通路線的順暢。過去二次大戰著名艦隊司令官為[[威廉·海爾賽|威廉·海爾賽]]，現任艦隊司令官為亞歷山大中將。<ref>[http://www.globalsecurity.org/military/agency/navy/c3f.htm 2006年 第三艦隊探索]</ref>

美軍高層有意取消第三艦隊和[[第七艦隊|第七艦隊]]的行政劃界，把第三艦隊執勤範圍擴到[[西太平洋|西太平洋]]。<ref>{{cite news|author1=星島日報|title=美第三艦隊執勤 擬擴到西太平洋|url=https://hk.news.yahoo.com/%E7%BE%8E%E7%AC%AC%E4%B8%89%E8%89%A6%E9%9A%8A%E5%9F%B7%E5%8B%A4-%E6%93%AC%E6%93%B4%E5%88%B0%E8%A5%BF%E5%A4%AA%E5%B9%B3%E6%B4%8B-215637909.html|accessdate=2015-09-28|date=2015-09-28}}</ref>目前，第七艦隊司令部設在[[日本|日本]][[橫須賀港|橫須賀港]]。

== 歷史 ==
[[File:USN_Fleets_(2007).png|thumb]]
[[File:USN_Fleets_(2009).png|thumb]]
第三艦隊在[[第二次世界大戰|第二次世界大戰]]期間的1943年3月15日，由美國海軍上將[[威廉·海爾賽|威廉·海爾賽]]宣布成立，其岸上總部則於1944年6月15日在[[夏威夷|夏威夷]]的[[珍珠港|珍珠港]]內成立。艦隊管轄在[[索羅門群島|索羅門群島]]、[[菲律賓|菲律賓]]、[[台灣|台灣]]、[[琉球群島|琉球群島]]到[[美國|美國]][[新澤西州|新澤西州]]之間的整個[[太平洋|太平洋]]海域。從1945年至戰爭結束，[[密蘇里號戰艦|密蘇里號戰艦]]一直是該艦隊的旗艦。
艦隊司令官[[威廉·海爾賽|威廉·海爾賽]]在二次大戰結束後，率領他的艦隊至[[日本|日本]][[東京灣|東京灣]]，並在[[密蘇里號戰艦|密蘇里號戰艦]]的甲板上接受大日本帝國簽署的投降書。
戰後的1986年7月，第三艦隊以科羅納多號兩棲運輸艦作為該隊新的旗艦，1991年8月，岸上總部由[[珍珠港|珍珠港]]轉移至[[聖地牙哥_(加利福尼亞州)|聖地牙哥]]現址。<ref>[http://www.c3f.navy.mil/History_Page.html 美國第三艦隊官方網站] {{webarchive|url=https://web.archive.org/web/20120728155254/http://www.c3f.navy.mil/History_Page.html |date=2012-07-28 }}</ref>

== 管轄區域 ==
東部及北部太平洋海域，包含[[白令海|白令海]]、[[阿拉斯加|阿拉斯加]]、[[阿留申群島|阿留申群島]]及部分北極等。

== 戰力 ==
第三艦隊有超過100艘艦艇，包括4艘[[尼米茲級核動力航空母艦|尼米茲級核動力航空母艦]]：
*[[尼米茲號航空母艦|尼米茲號航空母艦]]（CVN-68）和航母第十一打擊群
*[[卡爾·文森號航空母艦|卡爾·文森號航空母艦]] （CVN-70）和航母第一打擊群
*[[西奧多·羅斯福號航空母艦|西奧多·羅斯福號航空母艦]]（CVN-71）和航母第十二打擊群
*[[約翰·C·史坦尼斯號航空母艦|約翰·C·史坦尼斯號航空母艦]]（CVN-74）和航母第三打擊群
