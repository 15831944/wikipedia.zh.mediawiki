'''第一次摩哈赤战役'''（{{lang-hu|mohácsi csata 或 mohácsi vész}}、{{lang-tr|Mohaç Savaşı 或 Mohaç Meydan Savaşı}}）发生于1526年8月29日，是土耳其第一次入侵匈牙利的战役。战役双方为[[路易二世_(匈牙利)|拉约什二世]]率领的[[匈牙利王国_(1301年–1526年)|匈牙利王國]]军队及由[[苏莱曼一世_(奥斯曼帝国)|苏莱曼一世]]率领的[[奥斯曼帝国|奥斯曼]]军队。

== 准备工作 ==
匈牙利已经长期对抗奥斯曼帝国于南欧的扩张。匈牙利国王拉约什二世透过于1522年与[[奥地利的玛利亚|奥地利的玛利亚]]联姻令王国与[[哈布斯堡王朝|哈布斯堡王朝]]关系更加密切，而奥斯曼帝国也知道必需打破这个初期的联盟；在拉约什二世拒绝维持和平的提议之后，奥斯曼帝国决意出兵干预。在1526年6月，一支奥斯曼远征军越过[[多瑙河|多瑙河]]攻击匈牙利本土。 

匈牙利军队被分成三个主要部分:川西凡尼亚军团负责守卫喀尔巴阡山脉南部的的关口，而军队主力由国王拉约什二世亲自指挥，而其它较小规模的军团则由克罗地亚伯爵[[克里斯托夫·法兰高宾|克里斯托夫·法兰高宾]](Christopher Frankopan)指挥。由于王国地理位置的关系，土耳其军队直至在度过巴尔干山脉的时候才确定最后的攻击目标是王国首都。但在这时候川西凡尼亚军团比土耳其军队离开布达（现布达佩斯市西部城区，当时匈牙利王国首都）更远。之后有文献指出因特兰西瓦尼亚总督[[扎波尧伊·亚诺什|扎波尧伊·亚诺什]]的军队未能及时抵达，所以必须承担部分失利的责任。

匈牙利军队所选择的战场是多瑙河下游一片广阔但崎岖的平原，包括一些沼泽地。奥斯曼军队之前几乎没有遇上任何抵抗。当拉约什二世于布达等待开战的时候，奥斯曼军队已经围攻数个城镇并越过萨瓦河及[[德拉瓦河|德拉瓦河]]。拉约什二世集合了大约26,000名士兵，而奥斯曼军队则有大约50,000–60,000人。匈牙利军队列好阵势，以取得地理上的优势希望将奥斯曼军队逐个击破。

== 战斗 ==
真正的战斗只进行了两小时。起初苏莱曼一世的军队先锋，[[鲁米利亚|鲁米利亚]]军团，行进到战场之内，被由[[柏尔·杜蒙尼|柏尔·杜蒙尼]](Pál Tomori)所率领的匈牙利军队伏击，战局并向匈牙利军队预计的情况发展。但当奥斯曼军队主力于中午抵达后，战局迅速逆转。有一段时间苏莱曼一世自己非常危险，匈牙利军队的箭击中了他的[[盔甲|盔甲]]。由于未有及时增援，所以延误战机，匈牙利军队的进攻变成不能挽回的溃散。他们未能持续进攻，那些未有逃走的士兵被杀和俘虏。拉约什二世能逃离战场，但他随后从马上跌下来，跌到一条河之中淹死。多于16,000名匈牙利士兵于最初的战斗中被杀，奥斯曼军队也死傷慘重。多于1000名匈牙利贵族被杀。

== 战后 ==
[[File:Partition_of_Hungary.png|thumb]]
[[File:2lajos.jpg|thumb]]
苏莱曼一世是一个殘忍的人，他指令不留下任何一个战俘的活口。两日后他于他的日记中写到: “苏丹得到[[大臣|大臣]]及[[酋长|酋长]]们的效忠，屠杀2,000名战俘，大雨如注。”

这次胜利并未令奥斯曼帝国得到想要的自身安全。战争意味着独立的[[匈牙利王国_(1301年–1526年)|匈牙利王國]]灭亡，但奥斯曼军队于九月撤退，匈牙利王国剩余领土被拉约什二世的妻舅，[[哈布斯堡王朝|哈布斯堡王朝]]的[[斐迪南一世_(神圣罗马帝国)|奥地利大公斐迪南]]夺取，并透过与上任匈牙利及波希米亚国王[[乌拉斯洛二世|乌拉斯洛二世]]签订的条约继承这片土地。奥地利控制匈牙利北部三分一土地和现今的[[克罗地亚|克罗地亚]]北部，而奥斯曼帝国则得到匈牙利西南部和半独立的，由原总督[[扎波尧伊·亚诺什|扎波尧伊·亚诺什]]（他受到国内小贵族支持加冕为王，称亚诺什一世）统治[[特兰西瓦尼亚|特兰西瓦尼亚]]的宗主权，利用这些据点入侵东面仍然独立的匈牙利贵族及西北面的奥地利领地，后来更导致1529年奥斯曼军队围攻维也纳，是为[[维也纳之围|维也纳之围]]。

[[File:Mohacs_Monument_at_the_Battlefield_2004.JPG|thumb]]

这场战役有时会与14世纪发生的[[尼科波利斯战役|尼科波利斯战役]]和[[克雷西会战|克雷西会战]]比较，协同作战能力相对较差的匈牙利军队普遍受到较大损失，但纪律性较强的奥斯曼军队则较少损失。

对于在东欧得到这个稳当的基地，奥斯曼军队强大的重骑兵、有效率的轻[[骑兵|骑兵]]及火炮将能不断对中欧进攻整整一个世纪。它们对匈牙利的影响，始于1529年奥斯曼帝国支持[[扎波尧伊·亚诺什|扎波尧伊·亚诺什]]（擁有匈國東部的[[特兰西瓦尼亚|特兰西瓦尼亚]]）对抗[[斐迪南一世_(神圣罗马帝国)|斐迪南]]（擁有匈國西部），直至1699年奥斯曼帝国對奧地利签订[[卡爾洛夫奇條約|卡爾洛夫奇條約]]，割让匈牙利为止。

于摩哈赤的失败对很多匈牙利人来说是决定性的，是匈牙利日趋没落的历史转折点，並成为了匈牙利民族的一个创伤。对于这个日子，直至400年后的今日都仍然提醒匈牙利民族他们对于任何困境都仍然坚毅不屈。现时匈牙利人一旦遇上困境，他们都会说「在摩哈赤所失去的远比现时的多」，藉此勉励自己面对困难。
