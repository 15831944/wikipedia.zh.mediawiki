'''第倫桃'''（[[學名|學名]]：''{{lang|la|Dillenia indica}}''），（[[英文|英文]]名稱：{{lang|en|Elephant Apple、Indian dillenia}}）<ref name = "樹木谷">[http://www.hktree.com/tree/Dillenia%20indica.htm 第倫桃]樹木谷</ref>，別稱'''擬琵琶'''、'''五椏果'''等<ref name = "台北花博">[http://beta.2010taipeiexpo.tw/ct.asp?xItem=86024&ctNode=5719&mp=3 第倫桃]2010臺北國際花卉博覽會</ref><ref name="台灣樹木解說">[http://www.envi.org.tw/twtrees/book1/130.htm 台灣樹木解說]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}環境品質文教基金會</ref><ref name = "台灣行道樹圖鑑">{{cite book|title=《台灣行道樹圖鑑》|author= |series=陳俊雄、高瑞卿|publisher=貓頭鷹出版|page=第106-107頁|isbn= 978-986-6651-02-1 |date=2008年3月}}</ref>，為[[五椏果科|五椏果科]][[五椏果屬|五椏果屬]]植物<ref name = "樹木谷"/><ref>[http://www.cfh.ac.cn/13372.sp CFH 物种信息卡]中國自然標本館</ref>。果肉含粘液[[果膠|果膠]]帶酸味，可製酸味飲料、果汁或[[果醬|果醬]]<ref name = "台北花博"/><ref name = "台灣樹木解說"/><ref name = "台灣行道樹圖鑑"/><ref name = "網絡植物園">[http://www.floral.com.tw/floral/floristbypermalink/032430611c2768ef48257026001412e7 第倫桃]網絡植物園</ref>。在[[印度|印度]]，第倫桃果實的萼片可作蔬菜食用；在[[馬來西亞|馬來西亞]]，其果實可加入[[咖喱|咖喱]]，果肉則用於洗髮<ref name = "藥用植物">{{cite book|title=《藥用植物 全世界700多種藥用植物的彩色圖鑒》|author= |series=萊斯莉。布雷姆尼斯|publisher=中國友誼出版|page=第52頁|isbn= 978-7-5057-1557-8|date=2007年2月}}</ref><ref name = "世界藥用植物圖鑑">{{cite book|title=《世界藥用植物圖鑑》|author= |series=布倫尼斯|publisher=貓頭鷹出版|page=第52頁|isbn= 978-986-7001-99-3|date=2008年3月}}</ref>。果肉亦有止咳、解熱之效；[[葉|葉]]及[[樹皮|樹皮]]可作[[收斂劑|收斂劑]]<ref name = "台灣樹木解說"/> ，葉的汁液可治禿頭，樹皮可治[[鵝口瘡|鵝口瘡]]<ref name = "藥用植物"/><ref name = "世界藥用植物圖鑑"/>。第倫桃的葉因與[[枇杷|枇杷叶]]相似，故而有拟枇杷的別稱<ref>[http://nature.edu.tw/rmedias/show/19413?page=1 第倫桃]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}台灣教育部自然生態學習網</ref>。第倫桃的果實外層包著肥厚的萼片，看似綠色的[[桃子|桃子]]，故而得名<ref name = "台北花博"/><ref name = "台灣樹木解說"/>。

==分佈及生境==
第倫桃原產於[[中國|中國]]、[[印度|印度]]、[[馬來西亞|馬來西亞]]、[[夏威夷|夏威夷]]等地，現分佈於[[中國|中國]]、[[台灣|台灣]]、[[印度|印度]]、[[斯里蘭卡|斯里蘭卡]]、[[馬來西亞|馬來西亞]]、[[中南半島|中南半島]]、[[菲律賓|菲律賓]]及[[印尼|印尼]][[爪哇|爪哇]]等地，中國國內分佈於[[雲南|雲南]]南部，[[福建|福建]][[福州|福州]]及[[廈門|廈門]]也有引種。喜生於[[山谷|山谷]]溪旁水濕地帶，土壤需排水好，喜高溫多濕環境，喜全日或半日照，耐旱，耐強風吹襲，不耐寒，生長適溫23－32[[攝氏度|攝氏度]]，[[板根|板根]]具有抓地之效，不適合在[[鹼性土壤|鹼性土壤]]中生長，以肥沃[[壤土|壤土]]為佳，播種法繁殖。<ref name = "台灣行道樹圖鑑"/><ref name = "網絡植物園"/><ref name="中國植物誌">[http://www.cvh.org.cn/zhiwuzhi/page/49(2)/194a.PDF 中國植物誌第49(2)冊]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}第194－195頁</ref><ref name="福建植物誌">[http://www.cvh.org.cn/difangzhi/fujian/page/3/450a.pdf 福建植物誌第3冊]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}第450－451頁</ref><ref name = "台北植物園">[http://tpbg.tfri.gov.tw/plants/plants_info.php?rid=60 植物資料庫]台北植物園</ref>

==形態特徵==
第倫桃是一種常綠半落葉中[[喬木|喬木]]植物，胸徑寬約1[[米|米]]，高約20－25米。[[樹皮|樹皮]]平滑，紅褐色，呈大塊薄片狀脫落；嫩枝粗壯，披褐色柔毛；老枝無毛，有明顯的[[脫葉痕|脫葉痕]]；小枝密生於主枝條末端。[[葉|葉]]倒卵形或矩圓形，互生，薄革質，頂端近圓形，有長約1厘米的短尖頭，基部圓形或廣楔形，不等側，[[葉緣|葉緣]]粗鋸齒或疏齒狀，表面皺摺，表面及背面初披柔毛，其後柔毛脫落，僅在背脈上有毛，長約15－40厘米，寬約6－14厘米；葉側脈平行，25－56對，脈間相隔5－8毫米，整齊而明顯，乾後表面及背面均突起；葉柄有狹窄的翅，基部稍擴大，長約5－7厘米；無[[托葉|托葉]]。[[花|花]]單生於近枝頂的[[葉腋|葉腋]]內，[[直徑|直徑]]約12－20厘米；[[花梗|花梗]]粗壯披毛；[[萼片|萼片]]5枚，近圓形，肥厚肉質，外側披柔毛，直徑約3－6厘米，有時基部厚達1厘米；[[花瓣|花瓣]]5枚，倒卵形，白色，長約7－9厘米；[[雄蕊|雄蕊]]發育完全，無退化雄蕊，外輪數目較多，內輪較少及比外輪長；[[花藥|花藥]]頂孔裂開，長於[[花絲|花絲]]；[[花柱|花柱]]線形，頂端向外彎；[[心皮|心皮]]16－20枚，[[胚珠|胚珠]]於每心皮多個。[[果|果]]為[[漿果|漿果]]，圓球形，宿存萼片肥厚，稍增大，不開裂，甚為堅實，成熟時黃褐色，果肉含透明粘液，直徑約10－15厘米。[[種子|種子]]腎臟形，壓扁，邊緣有毛，藏於透明粘液之果肉中。<ref name = "台灣樹木解說"/><ref name = "台灣行道樹圖鑑"/><ref name = "網絡植物園"/><ref name = "中國植物誌"/><ref name = "福建植物誌"/><ref name = "台北植物園"/>
