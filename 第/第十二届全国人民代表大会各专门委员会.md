'''第十二届全国人民代表大会各专门委员会'''是[[中华人民共和国|中华人民共和国]][[第十二届全国人民代表大会|第十二届全国人民代表大会]]设立的[[全国人民代表大会专门委员会|专门委员会]]，任期由2013年3月至2018年3月。

1982年公布施行的《[[中华人民共和国宪法|中华人民共和国宪法]]》第七十条规定：“全国人民代表大会设立民族委员会、法律委员会、财政经济委员会、教育科学文化卫生委员会、外事委员会、华侨委员会和其他需要设立的专门委员会。在全国人民代表大会闭会期间，各专门委员会受[[全国人民代表大会常务委员会|全国人民代表大会常务委员会]]的领导。”<ref>{{Cite web |url =http://www.npc.gov.cn/npc/zt/qt/gjxfz/2014-12/03/content_1888093.htm  |title = 中华人民共和国宪法（1982年12月4日） |author =  |date = 2014-12-03 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>第十二届全国人民代表大会设立9个专门委员会：[[全国人民代表大会民族委员会|民族委员会]]、[[全国人民代表大会法律委员会|法律委员会]]、[[全国人民代表大会内务司法委员会|内务司法委员会]]、[[全国人民代表大会财政经济委员会|财政经济委员会]]、[[全国人民代表大会教育科学文化卫生委员会|教育科学文化卫生委员会]]、[[全国人民代表大会外事委员会|外事委员会]]、[[全国人民代表大会华侨委员会|华侨委员会]]、[[全国人民代表大会环境与资源保护委员会|环境与资源保护委员会]]、[[全国人民代表大会农业与农村委员会|农业与农村委员会]]。
