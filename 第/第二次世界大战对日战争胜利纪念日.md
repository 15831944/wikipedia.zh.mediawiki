'''第二次世界大战对日战争胜利纪念日'''（{{lang-en|Victory over Japan Day}}，一般簡稱「{{lang|en|V-J Day}}」）是[[第二次世界大战|第二次世界大战]][[太平洋战争|太平洋戰場]]中，[[日本投降|日本投降]]從而戰爭結束的日期及其後每年的紀念日。由於戰事在各國終止的日期不等，此日在各國也有不同的紀念日期。美東戰時[[夏时制|日光時間]]（[[UTC-4|UTC-4]]）1945年8月14日凌晨，[[美國|美國]]接獲[[日本|日本]]向《[[波茨坦公告|波茨坦公告]]》各國傳遞的投降決定，當日[[美國總統|美國總統]][[哈利·S·杜鲁门|哈利·S·杜鲁门]]向全國宣佈了這一消息；美國[[羅德島州|羅德島州]]的「勝利日」（8月第2個星期一），即紀念8月14日。[[日本標準時間|日本時間]]8月15日正午，日本[[昭和天皇|昭和天皇]]向全國廣播了《[[終戰詔書|終戰詔書]]》；同日[[隴蜀時區|重慶時間]]上午10時，[[中華民國國民政府|中華民國國民政府]][[國民政府軍事委員會|軍事委員會]]委員長[[蔣中正|蔣中正]]特級上將發表了《[[:s:抗戰勝利告全國軍民及全世界人士書|抗戰勝利告全國軍民及全世界人士書]]》。日本的「{{link-ja|終戰日|終戦の日}}」、[[大韓民國|韓國]]的「[[韩国光复节|光復節]]」、[[朝鮮民主主義人民共和國|朝鮮]]的「[[光復節_(朝鮮半島)|光复节]]」，即紀念8月15日。日本時間1945年9月2日上午，日本在停泊在[[東京灣|東京灣]]的美國軍艦[[密蘇里號戰艦_(BB-63)|密蘇里號]]上與[[同盟國_(第二次世界大戰)|同盟國]]簽訂《[[降伏文書|降伏文書]]》；美國的「對日戰爭勝利紀念日」为9月2日。[[中華民國|中華民國]]將翌日9月3日定為[[軍人節|軍人節]]。而[[中華人民共和國|中華人民共和國]]的[[第二次世界大战对日战争胜利纪念日#中華人民共和國|中國人民抗日戰爭勝利紀念日]]定於[[中華民國|中華民國]]簽訂《[[降伏文書|降伏文書]]》的翌日9月3日

與「對日戰爭勝利紀念日」相對的是1945年5月8日（部分国家定在5月9日）的「[[第二次世界大戰歐戰勝利紀念日|歐戰勝利紀念日]]」（{{lang|en|Victory in Europe Day}}，一般簡稱「[[V-E_Day|V-E Day]]」）。

== 概要 ==
{{Listen|type=sound|filename=Imperial Rescript on the Termination of the War.ogg|title=“玉音放送”}}
[[File:President_Truman_announces_Japan's_surrender.jpg|thumb]]
1945年8月15日重庆时间早晨7时整，从当时[[同盟國_(第二次世界大戰)|同盟國]]主要四国的首都：[[重庆|重庆]]、[[华盛顿哥伦比亚特区|华盛顿]]、[[伦敦|伦敦]]、[[莫斯科|莫斯科]]，用[[华语|华语]]、[[英语|英语]]、[[俄语|俄语]]向战场上双方海陆空军队、世界各国，通过无线电广播，播发内容一致的公告：[[日本政府|日本政府]]已正式[[无条件投降|无条件投降]]。

日本則稱此日為'''終戰日'''（{{lang-ja|終戦の日}}）或'''終戰紀念日'''（{{lang-ja|終戦記念日}}）。日本标准时间当天中午12时整，天皇[[昭和天皇|裕仁]]通过电台广播向日本民众宣读了日本接受[[波茨坦公告|波茨坦宣言]]的《[[终战诏书|终战诏书]]》（这一广播被称为“玉音放送”）。日本政府在當日較早前，就已经透过在[[华盛顿哥伦比亚特区|华盛顿]]的[[瑞士|瑞士]]使团，告知[[美国总统|美国总统]][[杜鲁门|杜鲁门]]，日本愿意向同盟国无条件投降。

由于日本是[[轴心国|轴心国]]中最后一个投降的国家，因此，在[[V-E_Day|欧战胜利日]]后三个月的V-J日，就标志著整个[[第二次世界大战|第二次世界大战]]的終结。不过，日本政府和军队代表直到1945年9月2日（日本時區），才在停泊於[[東京灣|東京灣]]的美舰[[密蘇里號|密蘇里號]]甲板上，正式[[降伏文書|签字向同盟国投降]]。

由於“V-E Day”一名並無提及“[[納粹德國|德國]]”，而“V-J Day”卻包含日本国名，因此也有人称V-J日為'''太平洋战争胜利日'''（'''V-P Day'''，'''Victory in the Pacific Day'''）。但實際上整个[[太平洋|太平洋]]地区只有日本一个轴心国，故大部分人对後者並不太在意。

== 抗日戰爭勝利紀念日 ==
1945年9月2日，[[日本天皇|日本天皇]]和政府代表、外务大臣[[重光葵|重光葵]]及军方代表、参谋总长[[梅津美治郎|梅津美治郎]]，在停泊於东京湾的美国海军战列舰密蘇里號的甲板上，签字向中華民國等同盟国无条件投降，並由[[徐永昌|徐永昌]]代表中國在日本投降书上签字确认。

=== 中華民國 ===
*[[中華民國|中華民國]][[國民政府|國民政府]]规定自9月3日（日本簽訂降伏文書翌日）起放假一天，懸旗慶祝三天<ref name>「全國慶祝勝利 明日勝利日放假一天」[[大公報|大公報]]，1945年9月2日</ref><ref>[http://epaper.tianjinwe.com/mrxb/mrxb/2015-09/03/content_7344246.htm 70年前的今天胜利之日 举城狂欢] {{Wayback|url=http://epaper.tianjinwe.com/mrxb/mrxb/2015-09/03/content_7344246.htm |date=20151007044710 }}，每日新報，2015年9月3日</ref>。并通过决议，规定从次年开始，每年的9月3日为「'''抗日战争胜利纪念日'''」。

*1991年，[[中华人民共和国国务院|中华人民共和国国务院]]修订颁布的新办法中沿用了这一修改后的纪念日。1999年，中华人民共和国国务院令第270号《全国年节及纪念日放假办法》（1949年12月23日政务院发布，1999年9月18日国务院修订发布）中发布的有关抗日的纪念日有“七七”抗战纪念日、“九三”抗战胜利纪念日、“九一八”纪念日，以法规的形式确定了抗日战争纪念日。

===日本===
日本政府將8月15日作為“追悼戰爭死亡者、祈願和平”之日，開展全國戰爭死亡者追悼儀式。該日一般也被稱作終戰紀念日或終戰之日。政治團體和[[非政府組織|非政府組織]]也於此日舉行和平集會。

日本對於第二次世界大戰（太平洋戰爭、大東亞戰爭）終結之日有諸多說法，主要為以下所列：
# 1945年（昭和20年）8月14日：日本政府接受波茨坦宣言并通告同盟國各國。
# 1945年（昭和20年）8月15日：播放“玉音放送”（昭和天皇朗讀終戰詔書的錄音）以向日本國民公示日本投降。
# 1945年（昭和20年）9月2日：日本政府簽發決定執行波茨坦宣言的投降文书。在同盟國軍佔領下的1952年（昭和27年）4月27日的報紙上，9月2日亦被稱作“投降日”、“投降紀念日”或“戰敗紀念日”。
# 1952年（昭和27年）4月28日：依據與日本國簽訂的和平條約（舊金山條約），在國際法上，同盟國各國（蘇聯等共產主義國家除外）與日本的戰爭狀態終結。日本因舊金山和平條約恢復完全獨立，而稱4月28日為“主權恢復日”或“舊金山條約生效紀念日”。

==== 日本法律 ====
日本制訂的《引揚者給付金等支給法》（昭和32年［[[1957年|1957年]]］法律第109号）以[[8月15日|8月15日]]為起點，對引揚者（自日本佔領地撤回之日本國民）支付特別補助金，後續的相關法律（昭和42年［[[1967年|1967年]]］法律第114号）亦稱[[8月15日|8月15日]]為“'''終戦日”'''。[[1963年|1963年]][[5月14日|5月14日]]，日本內閣會議決定自同年[[8月15日|8月15日]]，由政府主辦全国戰爭死亡者追悼儀式，並在1965年之後以[[東京都|東京都]][[千代田區|千代田區]]的[[日本武道館|日本武道館]]作為舉行地點。1982年4月13日，內閣會議定8月15日為「追悼戰爭死亡者、祈願和平日」並繼續此一做法。[[日本盂蘭盆節|盂蘭盆節]]推遲一月使之和該紀念日日期一致為8月15日。

'''關於“追悼戰爭死亡者與和平祈願日”（昭和57年［1982年］4月13日閣議決定）'''
# 主旨：為了追悼在本次戰爭中死亡的人們以及祈念和平，設立“追悼戰爭死亡者與和平祈願日”。
# 日期：每年8月15日。
# 活動：政府在“追悼戰爭死亡者與和平祈願日”，于昭和38年（1963年）後，每年按附件規定繼續實施全國戰爭死亡者追悼儀式。
附件：關於全國戰爭死亡者追悼儀式的實施
# 于每年8月15日，天皇及皇后出席在日本武道館舉行的全國戰爭死亡者追悼儀式。
# 本儀式所涉及戰爭死亡者的範圍以及儀式的形式與昭和56年（1981年）之儀式相同。
# 本儀式中，來自全國的遺族代表由國費負擔参列。
# 儀式当日，國家機關設施降半旗，並建議地方公共團体等採取同様措施；在指定時刻全國國民一齊默哀。

[[全國高等學校野球選手權大會|全國高等學校野球選手權大會]]的会場也在正午報時之前中斷比賽，選手和觀眾們默哀一分鐘。2010年，在第一場比賽完成後，於正午清空場地，並在12時10分進行默哀。2014年，在第三場比賽開始前的正午清空整理場地，于12時15分進行默哀。

在進行過地面戰鬥的[[沖繩|沖繩]]，日軍在1945年[[6月23日|6月23日]]停止了有組織的抵抗，因此現在沖繩縣在6月23日設立慰靈日休假一天。雖然美軍在[[7月2日|7月2日]]宣布[[沖繩戰役|沖繩戰役]]完結，並記錄6月30日為征服沖繩的日子，但之後與日軍的零星戰鬥仍然持續。直到9月7日，南西諸島守備軍代表在嘉手納的美軍第10軍司令部簽署投降文書，沖繩戰役才正式終結。因此在沖繩縣的正式慰靈設施“[[沖繩戰跡國定公園|和平之礎]]”，標記著沖繩戰役終結日為9月7日。

==== 日本中小學教科書 ====
日本小学社会科教科書以及中学社会科教科書（歷史部分），大多把終戰日標記為[[8月14日|8月14日]]或[[8月15日|8月15日]]，有的甚至沒有提及[[9月2日|9月2日]]。另外，雖然教科書有提及舊金山和約的簽訂日期[[1951年|1951年]][[9月8日|9月8日]]，但大多同樣沒提到和約生效日期（1952年[[4月28日|4月28日]]）。

高等院校的日本史教科書大多會把終战日標記為9月2日，更多教科書則分別把8月14日及15日記載為「通告同盟國接受波茨坦宣言日」以及「向國民廣播播放通知戰爭結束之日」（如山川出版社的《詳説日本史》）。

====其他記載====
日本紀錄片等節目曾多次使用[[8月11日|8月11日]]拍攝的“世界上慶祝終戰的人們”的影像，但當時實際上是日本以“維持國體”作條件，發出電報接受波茨坦[[波茨坦公告|宣言]]，消息經報紙以獨家新聞報道並錯認為“正式投降”，使不少人因此產生“騷動”而以訛傳訛。根據当時在中立国瑞典避難的日本特派員的手記，日本大使館亦對有關日本於8月10日投降的報道發表了同樣的聲明。

== V-J日的经典镜头 ==
[[File:Vj_day_kiss.jpg|thumb]]》]]

V-J日当天，最广为流传的一个经典镜头就是在[[纽约|纽约]][[时报广场|时报广场]]上，一个年轻水兵抓住从身边经过的一位女子热吻的照片《时代广场的胜利日》（又译《[[胜利之吻|胜利之吻]]》,《时代广场的胜利日之吻》）。后来，这幅由[[阿尔弗雷德·艾森士塔特|阿尔弗雷德·艾森士塔特]]（Alfred Eisenstaedt）拍摄的照片被[[生活雜誌|生活雜誌]]刊登并广为流传。但也有人称此照片为后来摆拍而非当天实景。不过无论如何，这幅照片都给世人留下了深刻的印象，其后每年的对日战争胜利纪念日均有情侣在时代广场及其他地方模仿这一亲吻动作。<ref>{{cite news|url=http://news.china.com.cn/2015-09/01/content_36470820.htm?source=bdxsy|title=上海70对情侣再现二战“胜利日之吻”|publisher=中国网|date=2015-09-01|accessdate=2015-09-01}}</ref>
