== 表演嘉賓 ==
[[香港四大天王|香港四大天王]]清唱剛於4月1日辭世影壇巨星[[張國榮|張國榮]]生前作品《當年情》（電影《[[英雄本色_(1986年電影)|英雄本色]]》主題曲）。

== 媒體播放 ==
當晚的頒獎典禮，由[[香港無線電視|香港無線電視]][[翡翠台|翡翠台]]、[[香港電台第二台|香港電台第二台]]、[[中國中央電視台電影頻道|中國中央電視台電影頻道]]等頻道於1小時後轉播。

== 特別安排 ==
本屆金像獎舉行期間發生了張國榮辭世和[[SARS事件|SARS事件]]，頒獎禮因此取消了紅地毯，出席人員因受SARS事件影響被戴上口罩進場。
