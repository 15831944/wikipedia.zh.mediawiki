'''吉尊降白阿旺洛桑益喜丹增嘉措師松旺覺聰巴密白德貝桑布'''（{{bd|1935年|7月6日|catIdx=Dainzin Gyaco}}），簡稱'''丹增嘉措'''，又被稱作'''第十四世達賴喇嘛'''，是西藏政治和宗教的象徵人物。作為[[藏傳佛教|藏傳佛教]][[格魯派|格魯派]]的最高領袖，[[達賴喇嘛|達賴喇嘛]]被視為是[[聖觀音|聖觀音]]的化身。他提倡[[大西藏地區|大西藏地區]]高度自治、民主選舉、藏族人權、[[婦女權利|婦女權利]]、[[環境保護|環境保護]]、[[非暴力|非暴力]]、{{Link-en|宗教對話|Interfaith dialogue}}、{{Link-en|佛教和科學|Buddhism and science}}等。[[中華人民共和國|中華人民共和國]]認定其是[[西藏獨立運動|西藏獨立運動]]的分裂勢力領袖之一。

丹增嘉措生於[[中華民國|中華民國]][[青海省_(中華民國)|青海省]][[湟中縣|西寧縣]][[紅崖村|塔澤村]]（屬[[安多地區|安多地區]]）的農民家庭，原名'''拉莫頓珠'''。1937年，經過尋訪團遴選後，他被認定是[[第十三世達賴喇嘛|第十三世達賴喇嘛]]的[[轉世靈童|轉世靈童]]。隔年，西藏政府正式公開承認其為第十四世達賴喇嘛，[[國民政府|國民政府]]則批准免於「[[金瓶掣籤|金瓶掣籤]]」。1940年2月22日，其舉行[[十四世达赖喇嘛坐床典礼|坐床典禮]]，繼位成為第十四世達賴喇嘛，並由[[熱振·圖旦絳白益西丹巴堅贊|熱振·圖旦絳白益西丹巴堅贊]]、[[達扎·阿旺松繞圖多旦巴傑增|達扎·阿旺松繞圖多旦巴傑增]]先後出任[[西藏攝政列表|西藏攝政]]。

1950年11月17日，隨著[[中国人民解放军|中国人民解放军]]在[[昌都戰役|昌都戰役]]中擊敗[[藏军|藏军]]，丹增嘉措提前亲政，執掌西藏的政教大權。在1954年至1959年間，他擔任第一屆、第二屆[[全國人民代表大會|全國人民代表大會]][[全國人民代表大會常務委員會|常務委員會]][[全國人民代表大會常務委員會副委員長|副委員長]]，也是[[中國人民政治協商會議全國委員會|中國人民政治協商會議全國委員會]]第一屆委員、第二屆常務委員，以及[[西藏自治區籌備委員會|西藏自治區籌備委員會]]主任委員。但在1959年3月17日，由於藏區爆發[[1959年藏區騷亂|騷亂]]，他[[十四世達賴喇嘛出走印度|流亡]]至印度北部 。

隨後丹增嘉措在印度[[達蘭薩拉|達蘭薩拉]]建立[[藏人行政中央|藏人行政中央]]，並有大量藏族追隨他而流亡海外。他長期在各地演講，就西藏地位問題提出「{{Link-fr|西藏五點和平計劃|Plan de paix en cinq points pour le Tibet|五點和平計劃}}」。由於主張以非暴力方式磋商，他在1989年獲得[[1989年諾貝爾和平獎|諾貝爾和平獎]]。2011年，他宣布卸下政治職務，將權力移交給民主選舉產生的藏人行政中央，結束達賴喇嘛長期的政教合一傳統。爾後他減少碰觸西藏政治議題。

== 達賴喇嘛 ==
{{Main|達賴喇嘛}}
[[File:13.DalaiLama_imExil1909.jpg|thumb]]（圖中人物）的轉世靈童。]]

作為第十四世[[達賴喇嘛|達賴喇嘛]]<ref>{{Cite journal|author= R. N. Rahul Sheel |url= |title= The Institution of the Dalai Lama |journal= The Tibet Journal |date= 1989年 |volume= 第14卷 |issue= 第3期 |pages= 第19頁至第32頁 |issn=  |doi= |language= en |accessdate = 2019年1月17日 }}</ref>，丹增嘉措的[[法名和法號|法名]]為「吉尊降白阿旺洛桑益喜丹增嘉措師松旺覺聰巴密白德貝桑布」（{{藏|རྗེ་བཙུན་འཇམ་དཔལ་ངག་དབང་བློ་བཟང་ཡེ་ཤེས་བསྟན་འཛིན་རྒྱ་མཚོ་སྲིད་གསུམ་དབང་བསྒྱུར་མཚུངས་པ་མེད་པའི་སྡེ་དཔལ་བཟང་པོ་}}），意譯是「文殊尊者語自在睿智良慧持法海總攝三界無敵軍具足吉祥賢德尊」<ref>{{Cite web |url = https://web.archive.org/web/20160304185316/http://www.capdtc.org/lsmy/xzlw/201110/t20111031_1151060.htm |title = 十四世达赖喇嘛坐床典礼（三） |author = 周伟洲 |date = 2011年10月31日 |publisher = 西藏文化網 |language = Zh-hans |accessdate = 2019年1月17日 |quote = }}</ref>{{Sfnm|王貴|1991年|1loc=第30頁}}。在過去，達賴喇嘛是[[藏傳佛教|藏傳佛教]][[格魯派|格魯派]]的重要[[出家眾|僧侶]]，也是該派的實質領袖{{Sfnm|田中公明|2000年|1loc=第68頁|山姆·范·沙克|2011年|2loc=第129頁}}。達賴喇嘛被視為是[[聖觀音|聖觀音]]的化身<ref name = "民視新聞台 01">{{Cite web |url = https://www.youtube.com/watch?v=1UJFzYhD7Ns |title = 【台灣演義】專訪西藏精神領袖 達賴喇嘛 2019.02.17 |author = [[民視新聞台|民視新聞台]] |date = 2019年2月17日 |publisher = [[YouTube|YouTube]] |language = en |accessdate = 2019年1月17日 |quote =  }}</ref>。而在當前，除了作為格魯派的最高指導者外，對於其他藏傳佛教教派來說，達賴喇嘛也是藏傳佛教最高領導者，甚至被稱作藏傳佛教的「[[法王|法王]]」{{Sfnm|第十四世達賴喇嘛|1990年|佩瑪·傑布·甲日|1998年|2loc=第87頁|田中公明|2000年|3loc=第68頁|梅爾文·戈爾茨坦|2005年|4loc=第3頁}}。例如[[寧瑪派|寧瑪派]]的[[流亡|流亡]][[藏族|藏人]][[索甲仁波切|索甲仁波切]]，便認為丹增嘉措是當前藏傳佛教的最高領袖{{Sfnm|第十四世達賴喇嘛|2008年}}。

在歷史上，除了格魯派外，其他藏傳佛教的教派並沒有確定、統一的組織架構<ref name = "Portal Buddyjski CyberSangha 01">{{Cite web |url = http://www.buddyzm.edu.pl/download/geoffreysamuel.pdf |title = IN THE HIGH COURT OF NEW ZEALAND AUCKLAND REGISTRY |author =  |date = 2004年11月11日 |publisher = Portal Buddyjski CyberSangha |language = en |accessdate = 2019年1月17日 |quote =  }}</ref>。而即便是在格魯派內部，達賴喇嘛和寺院之間的權力關係，也隨著時間變化有所不同，並不完全是由達賴喇嘛所組織領導的<ref name = "Portal Buddyjski CyberSangha 01"/>。最初幾世的達賴喇嘛並非格魯派的正式領袖，不過隨著格魯派的發展、以及與其他教派發生摩擦後，[[第二世達賴喇嘛|第二世達賴喇嘛]]首先成為教派的領袖，後來的[[第三世達賴喇嘛|第三世達賴喇嘛]]更追認[[轉世|轉世]]系譜，確立格魯派的團結象徵{{Sfnm|洛朗·德賽|2005年|1loc=第135頁|田中公明|2000年|2loc=第97頁至第100頁|田中公明|2012年|3loc=第55頁至第57頁}}。而從1642年（[[第五世達賴喇嘛|第五世達賴喇嘛]]）到1959年期間，達賴喇嘛的傳統世俗職責還包括在[[甘丹頗章|甘丹頗章]]主持西藏政府<ref>{{Cite web |url = https://en.oxforddictionaries.com/definition/Dalai_Lama |title = Dalai Lama |author =  |date =  |publisher = 《[[牛津英語詞典|牛津英語詞典]]》 |language = en |accessdate = 2019年1月17日 |quote =  }}</ref>{{sfnm|河口慧海|1978年|青木文教|1990年|2loc=第284頁|田中公明|2000年|3loc=第113頁至第114頁|多田等觀|2008年|4loc=第9頁、第13頁至第16頁、第41頁至第45頁}}。

也因此，在西藏、[[蒙古高原|蒙古高原]]在內的藏傳佛教地區，達賴喇嘛同時是具有影響力的宗教權威，以及統治西藏的政治[[國家元首|領導人]]<ref name = "Brief Biography 01">{{Cite web |url = https://www.dalailama.com/the-dalai-lama/biography-and-daily-life/brief-biography |title = Brief Biography |author =  |date =  |publisher = His Holiness the 14th Dalai Lama |language = en |accessdate = 2019年1月17日 |quote =  }}</ref>{{Sfnm|河口慧海|1978年|青木文教|1990年|2loc=第284頁|田中公明|2000年|3loc=第113頁至第114頁|多田等觀|2008年|4loc=第9頁、第13頁至第16頁、第41頁至第45頁|大衛·斯內爾格洛夫等|1998年|山姆·范·沙克|2011年|小羅伯特·布斯威爾等|2013年}}。雖然過去達賴喇嘛的統治，經常受到[[蒙古帝國|蒙古帝國]]（1642年至1720年）和[[清朝|清朝]]（1720年至1912年）的統治影響，不過[[第十三世達賴喇嘛|第十三世達賴喇嘛]]時期便宣布與中國方面為「{{Link-en|檀越關係|Patron and priest relationship}}」{{Sfnm|華倫·W·史密斯|1997年|1loc=第107頁至第149頁|梅爾文·戈爾茨坦|2005年|2loc=第3頁至第23頁|林孝庭|2018年|3loc=第2頁至第8頁|張博樹|2014年|4oc=第20頁至第53頁、第63頁至第67頁|楊海英|2017年|5loc=第211頁|朱麗雙|2016年|6loc=第1頁至第9頁、第20頁至第21頁、第57頁至第61頁}}。1933年，第十三世達賴喇嘛圓寂後，西藏政府派出[[尋訪第十四世達賴喇嘛|尋訪團]]打探其[[祖古|化身]]{{Sfnm|第十四世達賴喇嘛|1990年|梅爾文·戈爾茨坦|2005年|2loc=第3頁、第243頁至第251頁|林孝庭|2018年|3loc=第143頁|張博樹|2014年|4oc=第53頁}}。最終他們被許多跡象引導，來到[[塔爾寺|塔爾寺]]附近，並找到丹增嘉措{{Sfnm|第十四世達賴喇嘛|1990年|梅爾文·戈爾茨坦|2005年|2loc=第243頁至第251頁|林孝庭|2018年|3loc=第143頁}}。在經過幾道針對可能人選的驗證後，西藏政府認定丹增嘉措便是第十三世達賴喇嘛的轉世化身{{Sfnm|第十四世達賴喇嘛|1990年|梅爾文·戈爾茨坦|2005年|2loc=第255頁至第256頁|張博樹|2014年|3oc=第55頁}}。

丹增嘉措原名為「拉莫頓珠」（{{藏|ལྷ་མོ་དོན་འགྲུབ།}}）<ref name = "官方網站 01"/><ref name = "Cosmic Harmony 01"/>，意思是「期盼充滿神性」{{Sfnm|第十四世達賴喇嘛|1990年|嘉樂頓珠和石文安|2015年|2loc=第20頁}}。其母親後來告訴他，他從小時候就未曾顯現出怯生的樣子{{Sfnm|第十四世達賴喇嘛|1990年}}。他是16個孩子中的第九個孩子，最大的孩子是姐姐[[達拉·次仁卓瑪|達拉·次仁卓瑪]]，比他年長18歲{{Sfnm|第十四世達賴喇嘛|1990年|李鐵錚|1956年|2loc=第179頁|查爾斯·阿爾弗雷德·貝爾|1987年|3loc=第399頁}}。其大哥[[土登晉美諾布|土登晉美諾布]]很早就被認定是高級喇嘛[[當彩活佛|當彩活佛]]，被迎請到著名的塔爾寺{{Sfnm|第十四世達賴喇嘛|1990年}}。後來拉莫頓珠被限制在塔爾寺時，其兩個兄弟因為是「[[轉世|承願轉世者]]」，已經在那裡學習成為[[喇嘛|喇嘛]]{{Sfnm|湯瑪斯·萊爾德|2007年|1loc=第267頁}}。在拉莫頓珠2歲多時，西藏[[噶廈|噶廈]]政府根據事前[[異象|預兆]]，派出3個尋訪小組前往東北方、東方和東南方，尋訪第十三世達賴喇嘛的[[轉世靈童|轉世靈童]]<ref name = "官方網站 01"/><ref name = "官方網站 02">{{Cite web |url = http://www.dalailamaworld.com/topic.php?t=193 |title = 第十四世達賴喇嘛大事紀錄年表 |author =  |date =  |publisher = 第十四世達賴喇嘛官方國際華文網站 |language = Zh-hant |accessdate = 2019年1月17日 |quote =  }}</ref><ref name = "任善炯 01">{{Cite web |url = http://www.xzass.org/html/news917.html |title = 达赖抵台，马英九最应该对达赖说的一句话 |author = 任善炯 |date = 2009年9月14日 |publisher = 西藏自治區社會科學院 |language = Zh-hans |accessdate = 2019年1月17日 |quote =  }}</ref>{{Sfnm|查爾斯·阿爾弗雷德·貝爾|1987年|1loc=第451頁|朱麗雙|2016年|2loc=第238頁至第239頁}}。

其中一個預兆是，在第十三世達賴喇嘛逝世後，遺體頭部原先朝向南方，後來卻轉往東北方，被視為是暗示轉世靈童的出生方向<ref name = "Cosmic Harmony 01"/>{{Sfnm|第十四世達賴喇嘛|1990年|查爾斯·阿爾弗雷德·貝爾|1987年|2loc=第451頁|梅爾文·戈爾茨坦|2005年|3loc=第247頁至第248頁}}。後來高級喇嘛、[[西藏攝政列表|西藏攝政]][[熱振·圖旦絳白益西丹巴堅贊|熱振·圖旦絳白益西丹巴堅贊]]至聖湖[[拉姆拉錯|拉姆拉錯]]觀湖，清晰「看見」水裡浮現「Ah」（{{藏|ཨཿ}}）、「Ka」（{{藏|ཀ}}）和「Ma」（{{藏|མ}}）三個藏文字母，並出現一座有著綠藍色屋瓦與金色屋頂的三層樓寺院，再經過一條曲折小徑通到東面山丘，對面有一棟怪異造型「導水槽」、綠松石色屋簷的小屋<ref name = "官方網站 01"/>{{Sfnm|第十四世達賴喇嘛|1990年|查爾斯·阿爾弗雷德·貝爾|1987年|2loc=第450頁至第451頁|梅爾文·戈爾茨坦|2005年|3loc=第243頁至第247頁|朱麗雙|2016年|4loc=第239頁}}。熱振和大多數人確信「Ah」暗示著尋訪地點位在安多地區，因而派遣尋訪團至東北方{{noteTag|對此，英國[[駐錫金政務官|駐錫金政務官]][[巴茲爾·古德|巴茲爾·古德]]曾向藏學家[[查爾斯·阿爾弗雷德·貝爾|查爾斯·阿爾弗雷德·貝爾]]講述關於東北方尋訪小組的經歷{{Sfnm|查爾斯·阿爾弗雷德·貝爾|1987年|1loc=第451頁}}。}}{{Sfnm|第十四世達賴喇嘛|1990年|查爾斯·阿爾弗雷德·貝爾|1987年|2loc=第450頁至第451頁|梅爾文·戈爾茨坦|2005年|3loc=第247頁至第248頁}}。

1937年，尋訪團抵達[[結古街道|結古]]，與[[第九世班禪額爾德尼|第九世班禪額爾德尼]]會面{{Sfnm|查爾斯·阿爾弗雷德·貝爾|1987年|1loc=第451頁|梅爾文·戈爾茨坦|2005年|2loc=第248頁至第249頁}}。第九世班禪額爾德尼提出3名候選男孩的姓名，但有兩位後來被排除{{noteTag|自從第十三世達賴喇嘛逝世後，第九世班禪額爾德尼就一直調查該地區兒童的出生預兆{{Sfnm|湯瑪斯·萊爾德|2007年|1loc=第265頁|梅爾文·戈爾茨坦|2005年|2loc=第248頁至第249頁}}。}}{{Sfnm|查爾斯·阿爾弗雷德·貝爾|1987年|1loc=第451頁|梅爾文·戈爾茨坦|2005年|2loc=第248頁至第249頁}}。在拜訪第三位人選時，尋訪團開始覺得「Ka」指的是三層樓高、綠藍色屋頂的[[塔爾寺|塔爾寺]]；在尋訪附近村落時，亦發現與觀湖結果相符合、屋頂上有結瘤{{Link-en|杜松 (植物)|Juniperus rigida|杜松}}木幹的房子<ref name = "官方網站 01"/><ref name = "任善炯 01"/>{{Sfnm|第十四世達賴喇嘛|1990年|查爾斯·阿爾弗雷德·貝爾|1987年|2loc=第451頁至第452頁|湯瑪斯·萊爾德|2007年|3loc=第265頁|梅爾文·戈爾茨坦|2005年|4loc=第248頁至第249頁}}。尋訪團領袖{{Link-en|結昌仁波切|Kwetsang Rinpoche}}先是扮作僕人，卻被拉莫頓珠稱呼為「色拉喇嘛」{{Sfnm|第十四世達賴喇嘛|1990年|梅爾文·戈爾茨坦|2005年|2loc=第249頁}}。而在攜帶一些相似物件拜訪時，拉莫頓珠於任何情況下，都能正確無誤地認出第十三世達賴喇嘛的個人用品，並拒絕選擇其他物品<ref name = "官方網站 01"/><ref name = "Cosmic Harmony 01"/>{{Sfnm|第十四世達賴喇嘛|1990年|湯瑪斯·萊爾德|2007年|2loc=第265頁至第266頁|梅爾文·戈爾茨坦|2005年|3loc=第249頁至第251頁|朱麗雙|2016年|4loc=第239頁}}。

=== 前往拉薩 ===

最初他與洛桑三旦一起接受閱讀、書法、背誦佛經等基本教育{{Sfnm|第十四世達賴喇嘛|1990年}}。大約在8歲時，洛桑三旦被送到私立學校，丹增嘉措只能趁著探訪與母親、姐姐和哥哥會面{{Sfnm|第十四世達賴喇嘛|1990年}}。這時他開始接受[[僧伽|僧伽]]的[[佛教研究|教育]]課程，包括[[五明|大五明]]及小五明<ref name = "官方網站 01"/>{{sfnm|李江琳|2010年|1loc=第76頁}}。大五明是[[梵語|梵語]]、{{Link-en|佛教哲學|Buddhist philosophy|哲學}}、工藝、[[藏醫學|藏醫學]]及佛學，佛學包括[[慧_(佛教)|般若]]、[[中觀|中觀]]、[[阿毘達磨|俱舍]]、[[佛教戒律|律儀學]]、[[因明學|釋量論]]<ref name = "官方網站 01"/>；小五明是[[詩歌|詩學]]、戲劇學、[[占星術|占星學]]、[[修辭學|詞藻學]]、聲律學<ref name = "官方網站 01"/>。除了接受成為藏族精神領袖的教育外，他還被培植成為世俗領袖，但仍保有許多遊戲時間{{Sfnm|第十四世達賴喇嘛|1990年|李江琳|2010年|2loc=第76頁}}。其每年早春搬到羅布林卡，而在6個月後的冬天開始前，搬回布達拉宮，直到20歲出頭才開始永久住在羅布林卡{{sfnm|第十四世達賴喇嘛|1990年}}。
