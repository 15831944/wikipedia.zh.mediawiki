'''第一屆[[澳門國際影展|澳門國際影展]]'''于2016年12月8-13日在[[澳門文化中心|澳門文化中心]]及[[澳門科學館|澳門科學館]]舉行。

== 籌備工作 ==
首屆[[澳門國際影展|澳門國際影展]]于2016年年初啟動籌備工作，並于2016年5月27日舉行記者會，介紹籌組委員會及國際專業團隊的組成以及籌備工作<ref>{{Cite web|url=https://www2016.iffamacao.com/zh/1st-international-film-festival-awards%e2%80%a7macao-will-grandly-commence-in-december-international-film-festival-awards-%e2%80%a2-macao/|title=第一屆澳門國際影展暨頒獎典禮12月隆重舉行 籌委會開展籌備 國際專業團隊參與|accessdate=2018-12-13|work=iffamacao.com|language=zh-CN}}</ref>。同年6月13日在[[上海|上海]]舉辦招待酒會<ref>{{Cite web|url=https://www2016.iffamacao.com/zh/former-venice-film-festival-director-marco-mueller-to-become-festival-director-for-the-debut-edition-of-international-film-festival-awards%ef%bc%8emacau-from-8-to-13-december-with-johnnie-to-and-an/|title=首屆澳門國際影展年底舉辦 黃建新、甯浩等名導齊力助陣|accessdate=2018-12-13|work=iffamacao.com|language=zh-CN}}</ref>。

* 籌組委員會主席：[[文綺華|文綺華]]（澳門特區政府旅遊局局長）
* 影展總監：[[馬可·穆勒|馬可·穆勒]]
* 影展經理：[[鄭碧雪|鄭碧雪]]
* 影展大使：[[杜琪峯|杜琪峯]]、[[許鞍華|許鞍華]]、[[崔東勛|崔東勛]]、[[Clemens_Klopfenstein|Clemens Klopfenstein]]
* 國際顧問：[[黃建新|黃建新]]（中國導演及監製）、[[吳姃琓|吳姃琓]]（韓國電影監製）、Olivier Père（法國電影公司總經理）、[[詹姆斯·夏慕斯|占士舒哈密士]]（美國電影監製、導演及編劇）、[[施南生|施南生]]（香港電影監製）、Luís Urbano（葡萄牙電影監製、影展總監）、Michael J. Werner（美國電影監製）

== 開幕典禮 ==
開幕典禮及開幕電影首映禮於2016年12月8日假[[澳門文化中心|澳門文化中心]]舉行，並由[[曾志偉|曾志偉]]、[[曾寶儀|曾寶儀]]、[[曾國祥|曾國祥]]先生擔任開幕典禮的嘉賓司儀。[[澳廣視|澳廣視]]及[[FOX_TV|FOX TV]] 錄播紅地毯及開幕典禮。出席紅地毯的除了開幕電影團隊及競賽單元評審團成員，還包括影展大使[[崔東勳|崔東勳]]、Clemens Klopfenstein，影展形象大使[[章子怡|章子怡]]，影展明星大使[[張根碩|張根碩]]，演員[[鄭秀妍|鄭秀妍]]、[[關之琳|關之琳]]、[[張晉|張晉]]、[[惠英紅|惠英紅]]、[[楊崢|楊崢]]、[[林路迪|林路迪]]、[[盧靖姍|盧靖姍]]、[[羅蘭_(香港)|盧燕]]、[[櫻庭奈奈美|櫻庭奈奈美]]，導演[[金知雲|金知雲]]、[[陸川|陸川]]<ref>{{Cite web|url=https://www2016.iffamacao.com/zh/the-1st-international-film-festival-awards%e2%80%a7macao-opened-today-with-angelin-preljocaj-and-valerie-mu%cc%88llers-polina/|title=第一屆澳門國際影展暨頒獎典禮 今日正式揭幕 開幕電影《翩翩人生波麗娜》(POLINA)|accessdate=2018-12-13|work=iffamacao.com|language=zh-CN}}</ref>。

== 頒獎典禮 ==
第一屆澳門國際影展暨頒獎典禮(IFFAM)于2016年12月13日假澳門文化中心舉行，由[[陳芷菁|陳芷菁]]及[[林盛斌|林盛斌]]擔任頒獎典禮的嘉賓司儀。頒獎典禮由[[澳廣視|澳廣視]]現場直播，並由FOX TV進行錄播<ref>{{Cite web|url=https://www2016.iffamacao.com/zh/the-1st-international-film-festival-awards%e2%80%a7macao-closes-with-star-studded-awards-ceremony-the-winter-wins-inaugural-best-film-award/|title=第一屆澳門國際影展暨頒獎典禮 群星雲集 星光熠熠 《冬日將至》勇奪最佳電影|accessdate=2018-12-13|work=iffamacao.com|language=zh-CN}}</ref>。
