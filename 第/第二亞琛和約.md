{{dablink|《第二亞琛和約》不同于《[[第一亞琛和約|第一亞琛和約]](1668)》。}}
'''《第二亞琛和約》'''，又称'''《爱克斯·拉夏贝尔和约》'''，由[[法國|法國]]、[[英國|英國]]、[[荷蘭|荷蘭]]和[[奧地利|奧地利]]在第二次亞琛和會討論後於1748年10月18日簽定，亦是[[奧地利王位繼承戰爭|奧地利王位繼承戰爭]]的終結。签署地点在[[神圣罗马帝国|神圣罗马帝国]][[亞琛|亞琛]]，由[[英國|英國]]和[[法國|法國]]代表主持。

不過條約並無解決[[英國|英國]]和[[法國|法國]]在[[西印度群島|西印度群島]]、[[非洲|非洲]]和[[印度|印度]]的貿易糾紛，為英法之间在[[七年戰爭|七年戰爭]]中发生冲突埋下導火線。而[[英國|英國]]與[[西班牙|西班牙]]的奴隸貿易糾紛，直至1750年簽定《[[馬德里和約|馬德里和約]]》才得以完全解決。[[英國|英國]]在條約中放棄了奴隸貿易的優先權，而[[西班牙|西班牙]]政府需賠償英國10萬[[英鎊|英鎊]]。

《[[第二亞琛和約|第二亞琛和約]]》使[[哈布斯堡王朝|哈布斯堡王朝]]得以延續，直至1918年[[奥匈帝国|奥匈帝国]]的覆亡。[[德国|德国]]著名音乐家[[亨德尔|亨德尔]]曾於1749年为[[英国王室|英国王室]]创作樂曲以紀念战争的终结。

==條款內容==
條約主要條款如下:

# [[法屬|法屬]][[北美|北美]][[路易斯堡|路易斯堡]]以及附近的[[新斯科舍省|新斯科舍省]]地區歸還[[法國|法國]]
# [[法屬印度|法屬印度]]的[[馬德拉斯|馬德拉斯]]割與[[英國|英國]]
# [[法國|法國]]將[[荷蘭|荷蘭]]部分邊境城鎮割與[[荷蘭|荷蘭]]
# 有條件地承認[[神圣罗马帝国皇帝|神圣罗马帝国皇帝]][[弗朗茨一世_(神圣罗马帝国)|弗朗茨一世]]，皇后[[玛丽亚·特蕾西亚|玛丽亚·特蕾西亚]]及他們的後裔對[[哈布斯堡王朝|哈布斯堡家族]]領地的繼承權
# [[哈布斯堡|哈布斯堡]]領地[[帕爾马|帕爾马]]，[[皮亞琴察|皮亞琴察]]和[[瓜斯塔拉|瓜斯塔拉]]等公國割與[[西班牙|西班牙]]的[[唐·菲利浦|唐·菲利浦]]王子
# 恢復[[摩德納公國|摩德納公國]]和[[熱那亞共和國|熱那亞共和國]]的獨立和疆域
# 重申1713年[[英國|英國]]與[[西班牙|西班牙]]簽定關於[[黑奴貿易|黑奴貿易]]的協定，並重申[[英國|英國]]的奴隸貿易優先權和向[[西屬|西屬]]殖民地運送奴隸的義務
# [[普魯士|普魯士]]獲得[[西里西亞公國|西里西亞公國]]和[[格拉茨伯國|格拉茨伯國]]
