== 劇情 ==
電影改篇自真人真事，以英國《[[星期日泰晤士報|星期日泰晤士報]]》獨眼女戰地記者[[瑪麗·科爾文|瑪麗·科爾文]]最後十年的事蹟作藍本<ref name="sina@181219">{{Cite web |url = https://ipop.sina.com.tw/posts/366336 |title = 戰火下最無畏而純粹的靈魂！「獨眼記者」撼動人心！ |author = 新浪 |date = 2018-12-19 |language = zh-tw |accessdate = 2019-02-04}}</ref>。描述她為報導真相，深入斯里蘭卡、伊拉克、阿富汗、利比亞、敘利亞等戰區，揭發戰爭暴行<ref>{{Cite web |url = https://www.thelocationguide.com/2018/11/jordan-doubled-as-multiple-locations-for-matthew-heinemans-a-private-war/ |title = Jordan doubled as multiple locations for Matthew Heineman’s A Private War |author = Shona Smith |date = 2018-11-15 |publisher = The Location Guide |language = en |accessdate = 2019-02-04 |archive-url = https://web.archive.org/web/20181115142116/http://www.thelocationguide.com/2018/11/jordan-doubled-as-multiple-locations-for-matthew-heinemans-a-private-war/ |archive-date = 2018-11-15 |dead-url = no }}</ref>。

== 製作 ==
電影由紀錄片導演{{tsl|en|Matthew Heineman|馬修·海涅曼}}執導<ref name="PBM@171222"/>。海涅曼表示他受到{{tsl|en|Marie Brenner|瑪麗·布倫納}}所寫的文章《瑪麗科爾文的-{私人戰爭}-》所啟發，希望讓世人知道各地衝突的情況，而這也是他作為紀錄片導演的願望<ref name="sina@181219"/><ref name="am730@190118">{{cite news |author = AnnLo |url = https://www.am730.com.hk/news/%E5%A8%9B%E6%A8%82/%E7%84%A1%E6%87%BC%E7%82%AE%E7%81%AB%E7%9A%84%E7%8D%A8%E7%9C%BC%E8%8B%B1%E9%9B%8C-158405 |title = 無懼炮火的獨眼英雌 |publisher = am730 |pages = B16 |date = 2019-01-18 |language = zh-hk |accessdate = 2019-02-04}}</ref><ref>{{Cite web |url = https://skypost.ulifestyle.com.hk/article/2259484/%E3%80%8A%E8%81%B7%E5%A0%B4%E5%A5%B3%E7%8E%8B%E6%94%BB%E7%95%A5%E3%80%8B%E3%80%8A%E7%AC%AC%E4%B8%80%E7%9C%BC%E6%88%B0%E7%B7%9A%E3%80%8B%20%E8%81%B7%E5%A0%B4%E5%A6%82%E6%88%B0%E5%A0%B4 |title = 《職場女王攻略》《第一眼戰線》 職場如戰場 |author = 林惠珊 |date = 2019-01-25 |publisher = 晴報 |language = zh-hk |accessdate = 2019-02-04}}</ref>。在電影籌備初期，海涅曼花了大約一年時間進行資料搜集，例如研究科爾文所製作的每篇報導與訪問，前往她在倫敦的住所，與其朋友和同事進行訪談等<ref name="am730@190118"/><ref>{{Cite web |url = https://soundandpicture.com/2019/01/director-matthew-heineman-a-private-war-post-sound/ |title = How Director Matthew Heineman and the ‘A Private War’ Post Sound Team Focused on Authenticity to Tell a Courageous True Story |author = Jennifer Walden |date = 2019-01-14 |publisher = Sound & Picture |language = en |accessdate = 2019-02-04}}</ref>。海涅曼更表示這部電影是「給新聞界的一封情書，也是向勇敢、強悍、求真的瑪麗致意」<ref>{{Cite web |url = http://seligfilmnews.com/a-private-war-interview-with-filmmaker-matthew-heineman/ |title = A PRIVATE WAR - Interview with Filmmaker Matthew Heineman |author =  Gadi Elkon |date = 2018-11-09 |publisher = Selig Film News |language = en |accessdate = 2019-02-04}}</ref><ref name="mpw@190204">{{Cite web |url = https://www.mpweekly.com/entertainment/europe/20190126-146880 |title = 【露莎蒙化身獨眼戰地記者】《第一眼戰線》使命感與心癮交戰 |author = 張一明 |date = 2019-01-26 |publisher = 明報周刊 |language = zh-hk |accessdate = 2019-02-04}}</ref>。

== 反響 ==
=== 評價 ===
《第一眼戰線》整體獲得了大多正面的評價。爛番茄根據91篇評論，該片持有89%的新鮮度，平均評分7.3／10。該網站的共識為「電影以冷靜評價前線記者所需的犧牲及露莎蒙·碧姬職業生涯最佳作品來紀念它的現實主題」<ref>{{Cite web |url = https://www.rottentomatoes.com/m/a_private_war |title = A Private War |author = Rotten Tomatoes |language = en |accessdate = 2019-02-04 |archive-url = https://web.archive.org/web/20190102024122/https://www.rottentomatoes.com/m/a_private_war |archive-date = 2019-01-02 |dead-url = no }}</ref>。[[Metacritic|Metacritic]]根據28篇評論，該片得分76分（滿分為100分），獲得「普遍有利」的評論<ref>{{Cite web |url = https://www.metacritic.com/movie/a-private-war |title = A Private War |author = Metacritic |language = en |accessdate = 2019-02-04 |archive-url = https://web.archive.org/web/20181013142340/https://www.metacritic.com/movie/a-private-war |archive-date = 2018-10-13 |dead-url = no }}</ref>。

美國娛樂雜誌《[[綜藝_(雜誌)|綜藝]]》的彼得·戴布魯基（Peter Debruge）稱該片為一部「非常複雜，心理上身臨其境」的電影<ref>{{Cite web |url = https://variety.com/2018/film/reviews/a-private-war-review-rosamund-pike-1202962491/ |title = Film Review: ‘A Private War’ |author = Peter Debruge |date = 2018-09-29 |publisher = Variety |language = en |accessdate = 2019-02-04 |archive-url = https://web.archive.org/web/20181116000930/https://variety.com/2018/film/reviews/a-private-war-review-rosamund-pike-1202962491/ |archive-date = 2018-11-16 |dead-url = no }}</ref>。另一方面，雜誌讚揚露莎蒙·碧姬真實地傳遞出戰地記者的堅定信念，又指出這部電影是她的演技巔峰之作<ref name="sina@181219"/>。

曾經在《星期日泰晤士報》擔任編輯、並與瑪麗合作無間的尚恩·萊恩（Sean Ryan）也稱讚露莎蒙·碧姬的演技，並感嘆在看完她的演出後花了數天才能平伏下來<ref name="HK01@190118"/>。此外，曾與瑪麗出生入死的英國攝影記者保羅·康羅伊更表示，當他第一次看見碧姬的造型時我他嚇了一跳；而當聽到碧姬說台詞時，保羅更認為「她就是瑪麗了」<ref name="am730@190118"/>。
