《'''第22年的告白：我是殺人犯'''》（{{lang-ja|22年目の告白 -私が殺人犯です-}}、{{lang-en|Memoirs of a Murderer}}）是2017年上映的[[日本電影|日本電影]]，導演兼編劇{{tsl|ja|入江悠|入江悠}}以犯罪心理及驚悚為元素拍攝，翻拍2012年韓國電影《[[殺人告白|殺人告白]]》<REF>[https://www.nownews.com/news/20170907/2604673 名家論壇》膝關節／第22年的告白：我是殺人犯] 2017.9.7 今日新聞</REF>。

== 劇情 ==
1995年的[[東京|東京]]，發生了轟動全[[日本|日本]]的連環殺人案，兇手逍遙法外，15年{{link-ko|起訴期限|公訴時効}}亦告屆滿。案發22年後，自稱兇手的'''曾根崎雅人'''（[[藤原龍也|藤原龍也]] 飾）高調現身記者會，並出版自己殺人的[[自傳|自傳]]，隨後又調侃一直未能破案的刑警'''牧村航'''（[[伊藤英明|伊藤英明]] 飾），甚至光明正大的向受害者家屬親口承認罪行，完全視道德如無物。如此誇張行徑居然瘋魔全國，引來大批媒體報導與粉絲朝聖，衝擊日本社會倫理。對案件耿耿於懷的牧村航其後應邀與曾根崎亮相電視節目，竟發現另一個更出人意表的真相……。<ref name="yahoo_hk">*《[https://movie.yahoo-leisure.hk/movie/details/DvBJ0-22%E5%B9%B4%E5%BE%8C%E3%81%AE%E5%91%8A%E7%99%BD%E2%80%93%E6%88%91%E6%98%AF%E6%AE%BA%E4%BA%BA%E7%8A%AF 22年後の告白 –我是殺人犯]》在香港[[雅虎|雅虎]]的電影頁面{{zh-hk}}
</ref>

== 演員 ==
*'''曾根崎雅人'''（2017年：44歲）：'''[[藤原龍也|藤原龍也]]'''
: 自稱是22年前東京連續殺人事件的兇手。
*'''牧村航'''（1995年：23歲 → 2010年：38歲 → 2017年：45歲）：'''[[伊藤英明|伊藤英明]]'''
: 負責調查連續殺人事件的刑警。曾因讓兇手逃走而使臉上殘留當年的傷疤。
*岸美晴（2017年：27歲）：[[夏帆|夏帆]]
: 第二殺人事件受害者遺孤暨目撃者。書店店員。
*小野寺拓巳（1995年：22歲 → 2010年：37歲）：[[野村周平|野村周平]]
: 牧村里香的未婚夫。22年前[[神戶大地震|神戶大地震]]後偕未婚妻往東京避難。
*牧村里香（1995年：20歲）：{{link-jp|石橋杏奈|石橋杏奈}}
: 牧村航的妹妹。前[[神戶市|神戶市]]某醫院護士，22年前神戶大地震後偕未婚夫往東京避難並投靠哥哥，因連續殺人事件而失蹤。
*春日部信司（2017年：28歲）：[[龍星涼|龍星涼]]
: 牧村航的後輩刑警。
*川北未南子（2017年：30歲）：{{link-jp|松本まりか|松本まりか}}
: 《我是殺人犯》的編輯。
*若松義生（2017年：57歲）：{{link-jp|矢島健一|矢島健一}}
: 組織犯罪對策課課長。
*戶田丈（2017年：26歲）：[[早乙女太一|早乙女太一]]
: 橘組成員。第三殺人事件受害者遺孤。
*瀧幸宏（1995年：享年45歲）：{{link-jp|平田滿|平田満}}
: 22年前牧村航上司。連續殺人事件的最後被害者。
*山縣明寬（1995年：38歲 → 2017年：60歲）：[[岩松了|岩松了]]
: 第四殺人事件的目撃者。山縣醫院院長 → 共立中央醫院院長。
*橘大祐（1995年：42歲 → 2017年：64歲）：{{link-jp|岩城滉一|岩城滉一}}
: 第三殺人事件的目撃者。橘組組長。
*仙堂俊雄（2017年：50歲）：[[仲村亨|仲村亨]]
: 新聞節目《NEWS EYES》的主播，緊追事件的真相。
