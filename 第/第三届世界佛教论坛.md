'''第三届世界佛教论坛'''是2012年4月25日至4月27日在[[香港|香港]]举行的[[世界佛教论坛|世界佛教论坛]]，主题是“和谐世界，同愿同行”，主题歌是《同愿同行》，一共有1000多为高僧大德、政要专家、社会名人等参加，开闭幕在[[香港体育馆|香港体育馆]]举行。<ref name="lion " >{{cite web |url=http://fo.ifeng.com/special/disanjieshijiefojiaoluntan/news/detail_2012_04/20/14039684_0.shtml |title=第三届世界佛教论坛日程安排 |publisher=凤凰网 |accessdate= 2012年4月20日|work= |language =}}</ref> [[南京|南京]]佛顶骨舍利2012年4月25日至4月30日在香港展出。<ref>{{cite web |url=http://fo.ifeng.com/news/detail_2012_04/20/14032263_0.shtml?_from_ralated |title=第三界世界佛教论坛 香江将现千僧万众诵经盛况 |publisher=凤凰网 |accessdate= 2012年4月20日|work= |language =}}</ref>

== 历程 ==
=== 4月26日 ===
==== 前奏 ====
第三届世界佛教论坛开幕，609位僧人在座，[[传印长老|传印长老]]、[[觉光长老|觉光长老]]、[[十一世班禅|十一世班禅]]、[[祜巴龙庄勐|祜巴龙庄勐]]四位高僧大德拈香三拜，随后僧侣齐唱《[[心经|心经]]》，接着[[朱维群|朱维群]]、觉光长老、[[曾荫权|曾荫权]]、传印长老、[[董建华|董建华]]宣读贺词。<ref name="lion " />

==== 开坛仪式 ====
===== 上午 =====
董建华、[[许嘉璐|许嘉璐]]、曾荫权、传印长老、觉光长老、[[彭清华|彭清华]]、[[王作安|王作安]]一共7人领导开幕仪式。<ref name="lion " /> 随后[[王菲|王菲]]演唱《[[心经|心经]]》，[[张明敏|张明敏]]、[[关淑怡|关淑怡]]、[[刘德华|刘德华]]等人演唱主题歌《同愿同行》。<ref>{{cite web |url= http://fo.ifeng.com/special/disanjieshijiefojiaoluntan/chuanzhen/detail_2012_04/26/14165280_0.shtml|title=张明敏演唱佛教论坛主题歌《同愿同行》 |publisher=凤凰网 |accessdate= 2012年4月26日|work= |language =}}</ref><ref name="lion " /> 接着高僧大德发言，由[[释宽运|宽运法师]]、[[释衍空|衍空法师]]主持，[[释惟觉|惟觉长老]]、[[泰国僧王|泰国代僧王]]、许嘉璐、外国政要、[[韩国|韩国]]佛教宗团协会议会会长、[[日本|日本]]佛教界代表、[[斯里兰卡|斯里兰卡]]佛教界代表依次发言，紧接着进行瞻礼佛顶舍利，随后午餐。<ref name="lion " />
