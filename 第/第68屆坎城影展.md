'''第68屆[[坎城影展|坎城影展]]'''於2015年5月13日至5月24日在[[法國|法國]][[坎城|坎城]][[節慶宮|節慶宮]]舉辦<ref name="Cannes2015">{{cite web |url=http://www.festival-cannes.com/en.html |title=Cannes Film Festival |accessdate=5 October 2014 |work=Cannes}}</ref>，[[美國|美國]][[導演|導演]][[喬爾·柯恩|喬爾·柯恩]]和[[伊桑·柯恩|伊桑·柯恩]]擔任評審團主席<ref name="Coen">{{cite web |url=http://www.festival-cannes.fr/en/article/61245.html |title=Joel and Ethan Coen to preside over the Jury of the 68th Festival de Cannes |accessdate=20 January 2015|work=Cannes}}</ref>，這是坎城影展首次有兩人共同担任评审主席<ref name="CoenBBC">{{cite web |url=http://www.bbc.co.uk/news/entertainment-arts-30894984 |title=Coen brothers to jointly head Cannes jury |accessdate=20 January 2015|work=BBC News}}</ref>。57歲法國演技派男星[[蘭伯特·威爾森|蘭伯特·威爾森]]將擔任2015年坎城影展主持工作，從坎城影展開幕起，一直到頒獎典禮，將一肩挑起跨越12天的主持大任<ref>{{cite web |url=http://ent.ltn.com.tw/news/paper/862318 |title=藍柏威爾森 將主持坎城影展 |work=自由時報 |date=2015-03-24 |accessdate=2015-03-31}}</ref>。2015年的影展海報主視覺，是在向百歲冥誕的瑞典女演員[[英格丽·褒曼|英格丽·褒曼]]致敬<ref>{{cite web |url=http://ent.ltn.com.tw/news/breakingnews/1267080 |title=英格麗褒曼百歲冥誕 登坎城主視覺海報 |work=自由時報 |date=2015-03-24 |accessdate=2015-03-31}}</ref>。坎城影展方面表示，她具現代象徵性與自由的女性，是大膽的演員，是好萊塢明星，也是新寫實主義要角，依熱情選擇角色和移居的國家，從未失去她的優雅和單純。上述這些都是影展年復一年透過獲獎藝術家和電影所強調的價值，而曾於1973年擔任影展評審團主席的英格麗·褒曼，也在這條路上做出了貢獻。<ref>{{cite web |url=http://times.hinet.net/news/16148755 |title=坎城影展海報 緬懷英格麗褒曼 |accessdate=2015-03-24|work=中央社}}</ref>
