
第1到第4钢琴协奏曲是[[莫扎特|莫扎特]]在他11岁时（1767年）于[[萨尔茨堡|萨尔茨堡]]完成的四部作品，尽管长期以来被认为是原创作品，这四部协奏曲实际上改编自多位德国音乐家的奏鸣曲。

== F大调第1钢琴协奏曲 (K.37) ==
共3个乐章：
# [[快板|快板]]（Allegro）：改编自[[劳帕赫|劳帕赫]]（Hermann_Raupach）1762年的的小提琴伴奏的键盘奏鸣曲作品（Op.1, No.5）
# [[行板|行板]]（Andante）：第二乐章由来不明，第五版[[新格罗夫音乐与音乐家辞典|新格罗夫音乐与音乐家辞典]]的作者[[埃里克·布洛姆|埃里克·布洛姆]]（Eric Blom）认为是莫扎特的原创
# 快板（Allegro）：改编自[[奥诺埃尔|奥诺埃尔]]（Leontzi_Honauer）1763年的的作品（Op.2, No.3） <ref name="imslp_k37">{{Cite web |url =http://imslp.org/wiki/Piano_Concerto_No.1_in_F_major,_K.37_(Mozart,_Wolfgang_Amadeus)|title = Piano Concerto No.1 in F major, K.37 | language = en |accessdate = 2018/06/15}}</ref>

# 朝气蓬勃的快板（Allegro spiritoso）：改编自劳帕赫的作品
# 行板（Andante）：改编自[[肖伯特|肖伯特]]（Johann Schobert）的作品
# 很快的快板（Molto allegro）：改编自劳帕赫的作品<ref name="imslp_k39">{{Cite web |url = http://imslp.org/wiki/Piano_Concerto_No.2_in_B-flat_major,_K.39_(Mozart,_Wolfgang_Amadeus)|title = Piano Concerto No.2 in B-Flat major, K.39 | language = en |accessdate = 2018/06/15}}</ref> 

== D大调第3钢琴协奏曲 (K.40) ==
共3个乐章：
# 庄严的快板（Allegro maestoso）：改编自奥诺埃尔1763年的的[[羽管键琴|羽管键琴]]奏鸣曲（Op.2, No.1）
# 行板（Andante）：改编自[[埃卡特|埃卡特]]（Johann Gottfried Eckard）1763年的小提琴奏鸣曲（Op.1, No.4）
# 急板（Presto）：改编自[[C.P.E.巴赫|C.P.E.巴赫]]1756年的的一首键盘作品 La Boehmer (H.81) <ref name="imslp_k40">{{Cite web |url = http://imslp.org/wiki/Piano_Concerto_No.3_in_D_major,_K.40_(Mozart,_Wolfgang_Amadeus)|title = Piano Concerto No.3 in D major, K.40 | language = en |accessdate = 2018/06/15}} </ref>
