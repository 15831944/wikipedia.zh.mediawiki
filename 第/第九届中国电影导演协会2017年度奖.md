'''第九届中国电影导演协会2017年度奖'''是[[中国电影导演协会|中国电影导演协会]]对2017年中国电影给予的表彰，于2018年2月27日公布初评入围名单<ref>{{cite news|url=https://www.weibo.com/ttarticle/p/show?id=2309404212080508039194|title=第九届中国电影导演协会2017年度初评入围名单揭晓|date=2018-02-27|publisher=新浪微博|accessdate=2018-03-23}}</ref>，3月22日在北京举行提名晚宴，公布提名名单<ref>{{cite news|url=http://ent.ynet.com/2018/03/23/1050081t1254_1.html|title=中国电影导演协会2017年度奖提名揭晓|date=2018-03-23}|publisher=北青网|accessdate=2018-03-23}}</ref>，4月21日在北京举行颁奖典礼<ref>{{cite news|url=http://ent.sina.com.cn/m/c/2018-04-22/doc-ifznefkh4788182.shtml|title=导演协会年度表彰《嘉年华》《芳华》 周冬雨影后|date=2018-04-22 |publisher=新浪|accessdate=2018-04-22}}</ref>，于4月22日晚22点在[[东方卫视|东方卫视]]播出。

== 表彰内容 ==
;评委会奖
* 年度导演
* 年度影片
* 年度男演员
* 年度女演员
* 年度青年导演
* 年度编剧
* 评委会特别奖（颁发给音乐、摄影指导、美术指导、剪辑、音响设计、特效设计及在年度有特殊贡献的工作人员）
;执委会表彰
* 杰出贡献导演
* 年度港台导演

参评资格{{noteTag|name=b|1=已获得电影局颁发发行放映许可证，并于2017年1月1日至2017年12月31日期间在中国内地影院或国内外影展上公开放映过的剧情片、动画片、纪录片、舞台艺术片等都具备参评资格。
年度影片的参评资格为符合上述要求的国产电影作品；
年度导演的参评资格为中国内地导演；
年度青年导演的参评资格为中国内地导演，拍摄参评作品时年龄不满38周岁，独立完成作品不超过三部；
年度男演员和年度女演员的参评资格为出演参评影片的所有华裔演员；
年度编剧的参评资格为中国内地编剧。}}

== 其它 ==
本次评委会对第五代电影人进行特别表彰，“在20世纪承上启下的历史节点，第五代电影人从题材和样式到主题和意义，从电影的造型语言、叙事构成直到美学本体，全方位地将中国电影带入现代，并走向世界<ref>{{cite news|url=http://news.sina.com.cn/o/2018-04-21/doc-ifznefkh3754498.shtml|title=第五代电影人获中国电影导演协会特别表彰|date=2018-04-21}|publisher=新浪|accessdate=2018-04-21}}</ref>。

本次颁奖典礼同时产生第三届青青葱五强名单：高临阳《团圆》，梁鸣《对岸是否艳阳》，毛泽翔（团队）《野马》，王猛《春心荡漾》，姚文逸《野摩托》，将会获得[[国家电影局|国家电影局]]颁发的一百万扶持金。
