==花絮==
该片是为[[辛亥革命|辛亥革命]]一百周年献礼片。首映是在[[北京市|北京市]]。<ref>{{cite web |url=http://ent.qq.com/a/20110926/000231.htm |title=《第一大总统》北京首映 全景展现孙中山一生 |accessdate=2011年9月26日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

该片汇集了老中青三代演员。<ref name=" cat" />

该片制片商是[[中国电影家协会|中国电影家协会]]、[[中国电影股份有限公司|中国电影股份有限公司]]，三横一竖影视文化传媒有限公司、江苏华红影视投资有限公司、奥山影视文化有限公司。<ref name="bull " >{{cite web |url=http://china.cnr.cn/ygxw/201109/t20110930_508568959.shtml |title= 电影《第一大总统》全国公映 韩庚等80后演员参演|accessdate=2011-09-30  |publisher=中国广播网 |work= |language =zh-cn}}</ref>

该片中加入了[[爱情|爱情]]元素。<ref name="bull " />

该片中演员的跨界发展没有达到预想效果的成功。<ref>{{cite web |url=http://ent.qq.com/a/20110930/000025.htm |title=《第一大总统》：跨界尚未成功 粉丝仍须努力 |accessdate=2011年9月30日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

[[田亮|田亮]]曾经是中国跳水运动员，获得过[[奥运会|奥运会]]的金牌，现在转换当演员。<ref>{{cite web |url= http://news.xinhuanet.com/sports/2012-09/02/c_123660814.htm|title=田亮谈转型路一度哽咽：娱乐圈付出回报不成正比 |accessdate=2012年9月2日 |publisher=新华网 |work= |language =zh-cn}}</ref>
