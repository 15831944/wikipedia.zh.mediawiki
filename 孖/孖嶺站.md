'''孖岭站'''是[[深圳地铁|深圳地铁]][[深圳地铁9号线|9号线]]与[[深圳地铁10号线|10号线]]的一个换乘站，9号线站台位于彩梅立交东侧的梅林路上，西接上梅林站，东接银湖站，呈东西向布置；10号线站台位于[[北环大道|北环]][[彩田路|彩田]]立交西北象限工业区内，沿[[彩田路|彩田路]]呈南北布置。车站由于处于孖岭社区内而得名。9号线车站东端设1条联络线连接[[笔架山停车场|笔架山停车场]]，车站装修以白色为主色调，柱体采用亮黄色搪瓷钢板，柱子上方加入了客家围屋古建筑的元素符号，为车站赋予了浓厚的文化气息。<ref name="hp"/><ref>[http://www.szmc.net/page/plan/build.html?code=3120 深圳地铁9号线车站简介]深圳地铁网</ref>

===商店及自助服務===

由于场地限制，站内仅在D出口附近设有一家[[便利店|便利店]]和[[中国银行|中国银行]]的自動櫃員機。

== 設施及藝術牆 ==
本站設有洗手間，位於C出口旁。

本站藝術牆名為《南國瓷話》，為高溫彩釉壁畫。

== 车站历史 ==
<li>2007年，深圳市规划国土委公布了《轨道交通近期建设线路线站位初步方案 》，当时规划图中的福田党校站就是现在的孖岭站，该站与[[深圳地铁16号线|深圳地铁16号线]]换乘。<ref>[http://www.szpl.gov.cn/main/zsgg/200707090211045.shtml 轨道交通近期建设线路线站位初步方案] {{webarchive|url=https://web.archive.org/web/20150924112716/http://www.szpl.gov.cn/main/zsgg/200707090211045.shtml |date=2015-09-24 }}深圳市规划国土委，2007-07-09 </ref></li>
<li>2012年12月，[[深圳地铁9号线|深圳地铁9号线]]正式开工，而[[深圳地铁16号线|深圳地铁16号线]]作为远期规划将在远期实施。<ref>[http://www.sznews.com/news/content/2012-12/06/content_7469796.htm 深圳交委：地铁7、9、11号线2016年通车] {{Wayback|url=http://www.sznews.com/news/content/2012-12/06/content_7469796.htm |date=20150924112917 }}深圳新闻网， 2012-12-06</ref></li>
<li>2013年，深圳地铁16号线更名为'''深圳地铁10号线'''，建设时序将提前。<ref>[http://barb.sznews.com/html/2013-10/14/content_2649130.htm# 地铁16号线将更名为10号线，纳入轨道三期规划提前开建,原地铁10号线将何去何从？] {{webarchive|url=https://web.archive.org/web/20160803185206/http://barb.sznews.com/html/2013-10/14/content_2649130.htm |date=2016-08-03 }}宝安日报，2013-10-14</ref></li>
<li>2015年10月，10号线正式动工，根据当时的设计，梅林东站施工将会拆除彩田立交转北环大道匝道，但为了不影响交通出行，施工单位最终将梅林东站整体西移至旁边的工业区内。<ref>[http://www.cutv.com/v2/2015-10-16/G14efeeegfiiinngimgdg7.shtml 地铁10号线拟于本月动工 深圳东部将增添轨道大动脉]CUTV深圳台，2015-10-16</ref></li>
*2014年8月25日，孖岭站-[[笔架山停车场|笔架山停车场]]入场线顺利贯通。<ref>[http://www.szmc.net/page/news/index.html?code=2002&id=110909 深圳地铁9号线停车场入场线顺利贯通]深圳地铁网，2014-08-25</ref>
*2014年12月17日，孖岭站-[[银湖站|银湖站]]区间中段暗挖隧道左线贯通。
== 地质 ==
孖岭站周边属于[[丘陵|丘陵]]地貌，地面起伏较大，山体岩石较多，采用[[矿山法|矿山法]]、[[TBM|TBM（岩石隧道盾构法）]]分段施工<ref name="hp">[http://www.tsdig.com/admin/Upload/file/深圳市城市轨道交通10号线工程环境影响评价第二次公示.pdf 深圳市城市轨道交通10号线工程环境影响评价第二次公示] {{webarchive|url=https://web.archive.org/web/20160306044905/http://www.tsdig.com/admin/Upload/file/%E6%B7%B1%E5%9C%B3%E5%B8%82%E5%9F%8E%E5%B8%82%E8%BD%A8%E9%81%93%E4%BA%A4%E9%80%9A10%E5%8F%B7%E7%BA%BF%E5%B7%A5%E7%A8%8B%E7%8E%AF%E5%A2%83%E5%BD%B1%E5%93%8D%E8%AF%84%E4%BB%B7%E7%AC%AC%E4%BA%8C%E6%AC%A1%E5%85%AC%E7%A4%BA.pdf |date=2016-03-06 }}铁三院，2015-10-26</ref>
