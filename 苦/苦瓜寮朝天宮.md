'''苦瓜寮朝天宮'''位於臺灣[[臺南市|臺南市]][[山上區|山上區]]，是主祀天上聖母的廟宇{{r|媽祖}}{{r|鹿耳門志}}{{r|地}}。該廟原有前往[[北港朝天宮|北港朝天宮]]「割火」之習俗，後來改為前往[[鹿耳門天后宮|鹿耳門天后宮]]進香請水{{r|媽祖}}{{r|鹿耳門志}}。

== 沿革 ==
苦瓜寮的名稱由來，據說是過去先民{{NoteTag|最早來此為郭姓家族，後來從泉州來的田、涂二姓也到此居住，並成為當地主要姓氏{{r|鹿耳門志}}。田姓多住在南140線道路北邊，涂姓則大多住在道路南邊{{r|鹿耳門志}}。}}在此搭寮種植[[苦瓜|苦瓜]]之故{{r|媽祖}}{{r|鹿耳門志}}。最初當地先民搭建草寮供奉土地公，之後在乾隆初年加入神明會供奉天上聖母{{r|媽祖}}。清嘉慶三年（1798年）建立「聖母廟」，並從祀玄天上帝{{r|媽祖}}{{r|鹿耳門志}}{{r|地}}。之後於同治元年（1862年）整修，並自[[赤山龍湖巖|赤山龍湖巖]]迎來觀音佛祖與天上聖母（三媽）{{r|媽祖}}{{r|鹿耳門志}}{{r|地}}。光緒十九年（1893年）再修{{r|媽祖}}。

日治時期，該廟在大正十年（1921年）重建，並將廟名改為朝天宮{{r|鹿耳門志}}。此外日治時期，公厝管理人許頭因緣際會前往北港朝天宮分香（二媽），並提供土地為廟產{{r|媽祖}}。而由於此一緣故，苦瓜寮朝天宮開始有每逢祭典前往北港朝天宮進香割火，並在東勢寮曾文溪橋接駕、請水返境繞-{庄}-的習俗{{r|媽祖}}{{r|鹿耳門志}}。但此習俗何時中斷則不得而知{{r|鹿耳門志}}。

二次大戰後，該廟因為民國53年（1964年）的[[1964年白河地震|白河大地震]]而重創，後來在民國56、57年（1967、1968年）重建{{r|媽祖}}{{r|鹿耳門志}}。民國92年（1993年），該廟因為媽祖指示，前往[[曾文溪|曾文溪]]國姓橋（[[台17線|臺17線]]）請水，因為受到顯宮（媽祖宮-{庄}-）盛情接待，後來（1997年）將請水地點改為[[鹿耳門溪|鹿耳門溪]]口，定制4年1科{{r|媽祖}}{{r|鹿耳門志}}。另外此事也有受到[[山上天后宮|山上天后宮]]影響，山上天后宮跟這兩間廟都有交陪關係，居中協助兩廟結緣{{r|媽祖}}。

== 進香請水 ==
苦瓜寮朝天宮的進香請水活動於媽祖誕辰前的農曆三月廿一或三月廿二舉行，第一天進香由朝天宮大轎與宋江陣自行前往，第二天請水則會有友宮神轎與陣頭參與{{r|媽祖}}。進香請水隊伍約在傍晚出發，之後多在晚上10點前抵達鹿耳門天后宮進香（會香），午夜前抵達鹿耳門溪口請水{{r|媽祖}}{{r|鹿耳門志}}。之後在返回苦瓜寮的路上，會沿途經過善化區[[土虱堀聖興宮|土虱堀聖興宮]]、[[東勢寮慶濟宮|東勢寮慶濟宮]]、[[三塊寮保安宮|三塊寮保安宮]]、[[茄拔天后宮|茄拔天后宮]]，山上區[[山上天后宮|山上天后宮]]、[[南洲開靈宮|南洲開靈宮]]等廟參香拜旗，最後在苦瓜寮遶境後入廟安座{{r|媽祖}}{{r|鹿耳門志}}。
