'''苦根'''<ref>{{cite book|title=美国|date=1981|publisher=上海辞书出版社|url=https://books.google.com/books?id=kahHAAAAMAAJ&q=%E8%8B%A6%E6%A0%B9+%E8%92%99%E5%A4%A7%E6%8B%BF|language=zh}}</ref>（{{lang-xm|Lewisia rediviva}}，{{lang-en|Bitterroot}}），亦音译作'''比特鲁特'''<ref>{{cite web|title=美国爱达荷州钼矿之路与珍稀植物|url=http://theglobalmining.com/index.php?route=news/article&cpath=1&article_id=5356|website=theglobalmining.com|accessdate=2018-04-12|language=zh|deadurl=no|archiveurl=https://web.archive.org/web/20180412092725/http://theglobalmining.com/index.php?route=news%2Farticle&cpath=1&article_id=5356|archivedate=2018-04-12}}</ref>，是一种小型[[多年生植物|多年生]][[香草|香草]]，属于{{tsl|en|Montiaceae|水卷耳科}}{{tsl|en| Lewisia|露薇花属}}。

== 描述 ==
[[File:Lewisia_rediviva_2.jpg|left]]
苦根是一种生长缓慢的[[多年生植物|多年生植物]]，直根，叶呈张开状。花茎本身无叶，约{{Convert|1|-|3|cm|1}}高。其苞片有5-6枚，呈细长状，约5 - 10毫米长。每根花茎只开一朵花，并伴有5 - 9片椭圆形的[[花萼|萼片]]。<ref name="Klinkenberg2014"/>它们颜色各异，从白色到薰衣草色或者深粉色都有。花季在4月到7月之间。<ref name="Sullivan2015"/>苦根通常有15片椭圆形花瓣，每片约18 - 35毫米长。<ref name="Klinkenberg2014"/>花成熟后会结出一个蛋形的囊，内有6～20颗近乎圆形的种子。<ref name="Klinkenberg2014"/>

== 历史与文化 ==
法国的猎人将这种植物称为“{{lang|fr|racine amère}}”，即“苦根”。<ref name = "bitterrootheaven" />美洲土著则称此植物为
“spetlum/sp̓eƛ̓m̓”或“spetlem”（“手剥的”）、“nakamtcu”（{{tsl|en|Kutenai language|库特奈语}}：''naqam¢u''）<ref>{{Cite web
| title = FirstVoices: Ktunaxa words
| accessdate = 2012-07-08
| url = http://www.firstvoices.ca/en/Ktunaxa/word/63b51dec5cf5793e/Bitterroot
}}</ref>、”mo'ôtáa-heséeo'ôtse“（{{tsl|en|Cheyenne language|夏延语}}：“黑药”）<ref>[http://www.cdkc.edu/cheyennedictionary/index-english/main.htmm Cheyenne Dictionary]{{dead link|date=July 2017 |bot=InternetArchiveBot |fix-attempted=yes }} by Fisher, Leman, Pine, Sanchez.</ref>。

[[休休尼人|肖松尼人]]、{{tsl|en|Confederated Salish and Kootenai Tribes of the Flathead Nation|弗拉特海德民族萨利士与库特奈联合部落|弗拉特海德土著}}会偶尔食用苦根的根。传统上，{{tsl|en|Ktunaxa|库特尼}}人会将[[松鸡科|松鸡]]和苦根放在一起烹制，并放入糖；其他的部落则更喜欢就着盐食用。<ref>{{Cite web

 |accessdate   = 2012-07-08
 |url          = http://www.aqam.net/clc/nutrition/articles/traditional_diet.htm
 |archive-url  = https://web.archive.org/web/20091112173307/http://www.aqam.net/clc/nutrition/articles/traditional_diet.htm
 |dead-url     = yes
 |archive-date = 2009-11-12
 |archiveurl   = https://web.archive.org/web/20091213212444/http://www.aqam.net/clc/nutrition/articles/traditional_diet.htm
 |archivedate  = 2009-12-13
 |deadurl      = yes
}}</ref>{{tsl|en|Lemhi Shoshone|莱姆哈伊肖松尼}}人相信，苦根直根顶部的红色根芯有着特殊的力量，能够挡下[[熊科|熊]]的攻击。<ref name = "bitterrootheaven" />

在[[刘易斯与克拉克远征|刘易斯与克拉克远征]]期间，1805年至1806年，[[梅里韦瑟·刘易斯|梅里韦瑟·刘易斯]]食用了苦根。为了纪念他们，刘易斯带回的植物样本在经过确认后，被德裔美国植物学家{{tsl|en|Frederick Traugott Pursh|费德里克·特拉戈特·普尔士|费德里克·普尔士}}命名为''{{lang|la|Lewisia rediviva}}''。<ref name="bitterrootheaven">

1895年2月27日，苦根花被选为蒙大拿州[[美國州花列表|州花]]。

三个大型的地理特征——{{tsl|en|Bitterroot Mountains|苦根山}}、{{tsl|en|Bitterroot Valley|苦根谷}}和{{tsl|en|Bitterroot River|苦根河}}——均以此植物命名。<ref name = "bitterrootheaven" />

2009年，依照[[被子植物APG_III分类法|被子植物APG III分类法]]，苦根所属的{{tsl|en|Lewisia|露薇花属}}被从[[马齿苋科|马齿苋科]]移到了新设立的{{tsl|en|Montiaceae|水卷耳科}}。<ref>{{cite web|last1=LLC|first1=Revolvy,|title="Montiaceae" on Revolvy.com|url=https://www.revolvy.com/main/index.php?s=Montiaceae&item_type=topic|website=www.revolvy.com|accessdate=2018-04-16|language=en}}</ref>
