苦茶油，又稱山茶油，是由[[山茶屬|山茶屬]]植物種子透過物理壓榨或化學浸出所得到的[[油脂|油脂]]。在日本稱為[[椿油|椿油]]。主要使用樹種為[[油茶|油茶]]、[[短柱茶|短柱茶]]。

<br />

== 用途 ==
常用的功能可以作為食用油、化妝品（保養頭髮）、藥用、工業用、刀具保養油等。

食用歷史：苦茶油在中國的飲食紀錄超過千年，是華人喜愛的高級食用油。茶油素有東方橄欖油之稱，但其發煙點超過200度C，更適合東方料理使用。<ref>{{Cite web|url=https://www.medpartner.club/bitter-tea-oil-introduction/|title=苦茶油的發煙點|accessdate=|author=|date=|publisher=}}</ref>

== 製油方式 ==
主要可區分為壓榨法、溶劑萃取法以及水酶法等。壓榨法主要是利用物理方式，可分為冷壓與熱壓。一般認為冷壓方式可避免來自植物中的生物活性與抗氧化能力物質喪失<ref>{{Cite web|url=https://kmweb.coa.gov.tw/subject/ct.asp?xItem=834234&ctNode=8664&mp=362&kpi=0&hashid=|title=苦茶油的製造與利用, 農委會|accessdate=|author=|date=|publisher=}}</ref>，且風味清淡，適合剛入門或偏好清淡風味的人。<ref>{{Cite web|url=苦茶油如何正確選擇|title=https://www.medpartner.club/bitter-tea-oil-introduction/|accessdate=|author=|date=|publisher=}}</ref>

然而經研究指出，冷壓苦茶油的氧化安定性不佳，容易[[酸敗|變質]]；熱壓製程可以顯著增加安定性，且在120–130°C炒焙十分鐘下，油脂的組成並無變化，顯示並無必要因健康功效追求冷壓。<ref>{{Cite web|title=苦茶籽炒焙處理後的壓榨油更安定、貯藏期更長！|url=https://www.tari.gov.tw/form/index-1.asp?Parser=2,6,1115,1106,,,4119|work=行政院農業委員會農業試驗所|date=2014-02-17|accessdate=2019-03-26|last=行政院農業委員會農業試驗所}}</ref>

== 健康功效 ==
苦茶油之[[單元不飽和脂肪酸|單元不飽和脂肪酸]]含量最高達80%以上，單元不飽和脂肪酸含有Omega-9，遇熱仍可保持穩定狀態、不易變質。

[[單元不飽和脂肪酸|單元不飽和脂肪酸]]帶給人諸多健康功效，是現代人需要的油脂。<ref>{{Cite web|url=https://zh.wikipedia.org/wiki/%E5%96%AE%E5%85%83%E4%B8%8D%E9%A3%BD%E5%92%8C%E8%84%82%E8%82%AA|title=單元不飽和脂肪酸, 維基百科|accessdate=|author=|date=|publisher=}}</ref>
