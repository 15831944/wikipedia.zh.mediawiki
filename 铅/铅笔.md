'''铅笔'''（{{Lang-de|'''Bleistift'''}}，{{Lang-fr|'''crayon'''}}，{{Lang-en|'''pencil'''}}，{{Lang-es|'''lápiz'''}}），是一种用于在纸上书写、绘画的手持式筆、工具，它是用[[石墨|石墨]]為筆芯（除了[[彩色铅笔|彩色铅笔]]以外）以及木杆為外包層而製作的*，铅笔的尾端也可以带有一个[[橡皮擦|橡皮擦]]，铅笔与其他笔（除了[[可擦笔|可擦笔]]以外）的不同之处在于它的笔迹很容易被擦淡至不明显，以放大镜仍能看到遗留痕迹。黑色的铅笔叫'''木黑'''，顏色鉛筆叫'''木顏色'''。

*本條目特指'''木鉛筆'''，其他類似書寫工具見條目的下方。

[[File:2B_pencil.jpg|thumb]]
[[File:4B_pencil.jpg|thumb]]
現代鉛筆以石墨和[[黏土|黏土]]來製造，石墨添加得越多，筆芯愈軟，顏色愈黑；而黏土添加得愈多，筆芯則愈硬，顏色愈淺。鉛筆現在仍稱作鉛筆，是因為最早的極不成形或似鉛筆的筆為[[鉛|鉛]]金屬所製造，而後則又因古時候歐洲[[中世紀|中世紀]]在化學發展僅剛啟蒙時，人們將石墨誤以為是鉛的一種，因此'''鉛筆'''一詞在諸多語言及[[東亞語言|東亞語言]]中流傳下來、廣泛使用而未修正<ref>劍橋百科2005－鉛筆</ref>。中國古時也有「鉛筆」，是以鉛粉做成的筆，用來塗改絹帛或紙上的錯字。《文選．任昉．為范始興作求立太宰碑表》：「人蓄油素，家懷鉛筆。」<ref>教育部《重編國語辭典修訂本》,http://dict.revised.moe.edu.tw/cgi-bin/cbdic/gsweb.cgi?o=dcbdic&searchid=Z00000100662</ref>

而鉛筆和其相似原理的[[自動鉛筆|自動鉛筆]]和[[免削鉛筆|免削鉛筆]]，因為其消粍性的筆尖和容易被擦掉的字跡，不需要像[[鋼筆|鋼筆]]、[[原子筆|原子筆]]、[[寶珠筆|寶珠筆]]、[[麥克筆|麥克筆]]等，時常擔心過分用力或被碰撞跌下而損壞，也不用擔心寫錯字，所以經常是學校教低年級的學生寫字的主要教學工具。而因木鉛筆可以透過控制運筆的角度劃出粗線條，所以常被用作繪畫猶素描的工具，後來更出現仿照其原理構造的[[木顏色|木顏色]]。

但因為其易被控掉的字跡，一般避免在較正式的文件上使用。而且鉛筆筆尖因為書寫過程中變純和縮短，便需要定時刨或削掉，這需要使用[[鉛筆刨|鉛筆刨]]或小刀。而刨筆所產生的碎屑即使筆刨附有收集處，仍然需要定時拋棄，所以仍不方便使用。虽然鉛筆的在應用的便利性上不如原子筆，但在野外活動刀必隨身而其不怕油不怕水的特性仍是最可靠的紀錄工具。

== 历史 ==
鉛筆的原型可以追溯至古[[羅馬|羅馬]]時代。古[[羅馬|羅馬]]人用[[紙莎草|紙莎草]]紙包裹一塊[[鉛|鉛]]來書寫。经过百年的时间发展，现在的铅笔不仅有了木制的笔杆，还有了铅笔帽。英文為「''pencil''」，來自拉丁語「''penicillus''」。

1564年，[[英格兰|英格兰]]岛受飓风袭击，[[昆布兰|昆布兰]]地区的许多大树被连根拔起，人们在树根下发现了一种黑色的矿物质——[[石墨|石墨]]。當石墨擦過物體時，可以在物体上留下黑色的石墨薄層；石墨以[[范德华力|范德华力]]附著在紙上。因当时[[化学|化学]]刚开始发展，故石墨被误认为是「[[铅|铅]]」，特别是[[方铅矿|方铅矿]]，在矿石形态上与石墨十分相似。当地牧羊人常用它在羊身上画记号，以便于辨认。石墨这种特殊的矿物质非常纯净结实，可以很容易把它制成棒形。石墨的价值不久之后就变得越来越大，主要是因为它可用于在加农炮模具上画线，并且這種物矿已被皇室所擁有。但石墨仍會被私运出去制作铅笔。因为石墨比较柔软，必须制成一定形状。石墨棒起初是用细[[绳|绳]]或[[羊皮纸|羊皮纸]]包装。铅笔的好用性被广为传播，吸引了全世界著名艺术家的注意。首先是意大利人想到用木头来固定它，刚开始的时候是在木头中挖出棒形的洞。不久以后，发明了一种更方便的技巧：用两块雕刻过的木头夹住石墨棒，然后将两块木头用胶水黏住——从本质上来说，这个方法与现在所使用的是一致的。

虽然在世界的其他地方也发现过石墨，但是[[纯度|纯度]]和[[品質|品質]]却与英格兰岛发现的不一样，必须碾碎它以去除杂质，留下石墨粉。英格兰继续垄断铅笔的生产，直到出现一种新方法可以重新组合石墨粉。这种直接从原始石墨中切取出来的方形英国铅笔一直生产到19世纪80年代。现在，在[[凯斯维克|凯斯维克]]镇（靠近原来发现石墨块的地方）有一个铅笔[[博物馆|博物馆]]。

1662年，在[[德国|德国]]的[[纽伦堡|纽伦堡]]人们首次开始尝试用粉碎的石墨制作石墨棒。他们使用了石墨混合剂——[[硫磺|硫磺]]和[[锑|锑]]，虽然可以使用，但还是不如英国铅笔。

在[[拿破仑|拿破仑]]战争的时候英国和德国铅笔不再供应给法国。拿破仑部队里的一个军官试图改变这种情况。1795年[[孔德|孔德]]发现可以把石墨粉用黏土混合，将其做成棒形放入窑中烘烤。通过改变石墨对黏土混合的比率，可以改变石墨棒的[[硬度|硬度]]（黏土越多，铅笔就越硬，笔迹颜色越浅）。这个制作方法一直延用到今天。

這時現代木鉛筆大致定型，因為不同[[粉筆|粉筆]]或[[毛筆|毛筆]]等沒有規格化的傳統書筆工具，需要高精度的機器和準確的形狀大小，才能把芯放進木桿中，所以木鉛筆是最初的現代化書寫工具。

== 鉛筆的製作過程 ==
[[File:Pencil_manufacture.svg|thumb]]
先把細微的石墨粉、黏土（运用其可塑性和黏结性）和水混合而成的液體擠進一支細金屬管子，形成像麪條一樣的長條形。把長條晾干、切割，放在窰中烘烤，再放進熱油或蜡中（运用石蜡的防水性和较光滑的性质）。接着把木塊（通常使用容易削磨的雪松木）鋸成像鉛筆一半厚度的板條，將板條刨平，刻上細長凹糟。再把長條狀的石墨條放進凹槽中，之後在另一塊板條上塗上膠水，然後壓在第一塊木條上。當膠水乾了之後，便可切割成獨立的鉛筆，製成不同的形狀，經過磨平、塗色，最後蓋上生産商的商標和其他資料，一支無縫的鉛筆便制成了。有時候，生産商還會在鉛筆的一端裝上橡皮擦。<ref>本制作方法由德国Faber-Castell AG公司提供。</ref>另有所谓“环保铅笔”，即将涂胶的废纸（通常是旧报纸）使用机器卷在石墨笔芯上制成，使用方法与传统木质铅笔相同。

== 特殊用途 ==
铅笔除了書寫，其中最重要的可以算是鉴别钻石的真伪，将钻石用水润湿，用铅笔在上面刻劃一下，如果钻石是真的，其-{表面}-不会留下铅笔劃过的痕迹。如果是用玻璃、水晶、电气石等制作的假钻石则会留下痕跡。{{citation needed}}
