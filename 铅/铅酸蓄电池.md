'''鉛酸蓄電池'''，又稱'''鉛蓄電池'''，是[[蓄電池|蓄電池]]的一種，[[電極|電極]]主要由[[鉛|鉛]]製成，[[電解液|電解液]]是[[硫酸|硫酸]]溶液的一種蓄電池。一般分為開口型[[電池|電池]]及閥控型電池兩種。前者需要定期注酸維護，後者為免維護型蓄電池。按電池型號可分為小密、中密及大密。俗称的'''电瓶'''主要指的是铅酸蓄电池。1859年，法国物理學家[[雷蒙德-路易斯·加斯頓·普蘭特|普兰特]]发明。

== 結構 ==
鉛酸蓄電池一般由[[正極|正極]]板、[[負極|負極]]板、[[隔板|隔板]]、[[電池槽|電池槽]]、電解液和接線[[端子|端子]]等部分組成。

正極板為[[二氧化鉛|二氧化鉛]]板（PbO<sub>2</sub>），負極板為鉛板（Pb）。

== 原理 ==
鉛蓄電池的原理是通過將[[化學|化學]]能和[[直流電|直流電]]能相互轉化，在[[放電|放電]]後經充電後能復原，從而達到重複使用效果。鉛蓄電池的電壓為2的倍數。

=== 放電化學反應 ===
鉛酸蓄電池中的正極活性[[物質|物質]]（二氧化鉛）與負極活性物質（[[海綿鉛|海綿鉛]]）和電解液（30%-40%的稀硫酸溶液），反應生成[[硫酸鉛|硫酸鉛]]和水。

[[化學方程式|化學方程式]]為：
:<math>\rm PbO_{2 (s)} + Pb_{(s)} + 2{H_2SO_4}_{(aq)} \rightarrow 2{PbSO_4}_{(s)} + 2{H_2O}_{(l)}</math>

負極反應：<math>\rm Pb + SO_4^{2-}  \rightarrow PbSO_4 +2e^-</math>

正極反應：<math>\rm PbO_2 + 4H^+ +SO_4^{2-} + 2e^-  \rightarrow 2H_2O + PbSO_4</math>

=== 充電化學反應 ===
硫酸鉛和水轉化為二氧化鉛、海綿鉛與稀硫酸。

化學方程式為：
:<math>\rm 2{PbSO_4}_{(s)} + 2{H_2O}_{(l)} \rightarrow PbO_{2 (s)} + Pb_{(s)} + 2{H_2SO_4}_{(aq)}</math>

=== 通用的化學反應 ===
在[[放電反應|放電反應]]及[[充電反應|充電反應]]中，沒有額外物質減少或增加，由於兩個反應條件相同，所以是[[可逆反應|可逆反應]]，但實際環境下仍有許多變因。

化學方程式為：
:<math>\rm PbO_{2 (s)} + Pb_{(s)} + 2{H_2SO_4}_{(aq)} \iff 2{PbSO_4}_{(aq)} + 2{H_2O}_{(l)}</math>

== 應用 ==
铅酸电池[[内電阻|内電阻]]小，可应对大电流放电需要。

中型及小密[[電池|電池]]應可廣泛用於[[不斷電系統|UPS]]不間斷[[電源|電源]]、控制[[開關|開關]]、[[報警器|報警器]]、[[汽車|汽車]]牽引[[電源|電源]]、电动自行车等领域。
大密電池主要應用於大型[[基站|基站]]的[[通訊|通訊]]後備電源。

使用時應注意硫酸液面高度，以免發生意外。铅酸电池充电电压过高并过充时可能产生可燃的氢气，应当避免。

充電時應該將電源正極接到電池正極(二氧化鉛)負極接到電池負極(鉛)上

== 壽命 ==
鉛酸電池壽命大約 2~4 年，不過長期處於低電量的鉛酸電池壽命會縮短，大部分廠商提供 6~12 個月的保固期。鉛酸電池最常見故障是硫酸鉛結晶過多，因此也有一些商品用來處理或避免這個問題發生，主要有這些方式：
* 加水
* 小電流充電
* 脈衝式充電
