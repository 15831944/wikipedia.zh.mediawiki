'''{{ruby-py|铅|-{zh-cn:yán;zh-tw:ㄧㄢˊ;zh-hk:jyun<sup>4</sup>}-}}山县'''<ref>{{cite news|url=http://culture.people.com.cn/n/2015/0513/c87423-26991500.html|title=“铅山”的两种读音：土话在汉语词典开创了特例|publisher=[[人民日报|人民日报]]|date=2015-05-13|accessdate=2015-05-21}}</ref>位于[[中华人民共和国|中华人民共和国]][[江西省|江西省]]东北部、[[信江|信江]]中游，[[武夷山脉|武夷山脉]]北麓，是[[上饶市|上饶市]]所辖的一个[[县|县]]。

== 历史沿革 ==
[[南唐|南唐]][[保大|保大]]十一年（953年），析[[弋阳|弋阳]]、[[上饶|上饶]]置铅山县，因[[永平镇|永平]]西四里有铅山,遂以山名县。治所在永平，隶[[信州|信州]]。 

[[元|元]][[至元|至元]]二十九年(1292年)，升县为州，名铅山州（一名永平州），隶属[[浙江行中书省|浙江行中书省]]。

[[明|明]][[洪武|洪武]]二年(1360年)复为县，隶[[广信府|广信府]]。

1914年属豫章道，1931年属第六行政区，1949年5月以后隶属上饶地(专)区。

自建县起至1949年6月，县治一直设在[[永平镇|永平镇]]。1949年7月，县治迁[[河口镇_(铅山县)|河口镇]]。<ref name="铅山县历史沿革">{{cite web |title=铅山县历史沿革 |url=http://www.jxyanshan.gov.cn/?do=detail&ids=560 |publisher=铅山县人民政府网 }}</ref>

== 行政区划 ==
铅山县7个镇、10个乡（其中2个民族乡）。县人民政府驻河口镇，永平镇为第二大镇。
*[[镇|镇]]：[[河口镇_(铅山县)|河口镇]]、[[永平镇_(铅山县)|永平镇]]、[[石塘镇_(铅山县)|石塘镇]]、[[鹅湖镇_(铅山县)|鹅湖镇]]、[[湖坊镇_(铅山县)|湖坊镇]]、[[武夷山镇|武夷山镇]]、[[汪二镇|汪二镇]]
*[[乡|乡]]：陈坊乡、虹桥乡、新滩乡、葛仙山乡、稼轩乡、英将乡、紫溪乡、太源畲族乡、天柱山乡、篁碧畲族乡

== 地理 ==
=== 区位 ===
铅山县位于江西东北部、武夷山脉北麓，东为[[上饶县|上饶县]]，北为[[横峰县|横峰县]]，西为[[弋阳县|弋阳县]]和[[贵溪市|贵溪市]]，南邻福建省的[[武夷山市|武夷山市]]和[[光泽县|光泽县]]。<ref name="江西省地图册">{{cite book |title=江西省地图册 |year=2008 |publisher=星球地图出版社 |isbn=978-7-80212-369-4 }}</ref>

=== 地形地貌 ===
北部信江沿岸有少量[[平原|平原]]，中部多[[丘陵|丘陵]]，南部为[[山地|山地]]。基本地貌特征呈南高北低。

=== 山脉山峰 ===
[[武夷山脉|武夷山脉]]主峰[[黄岗山|黄岗山]]位于铅山与福建[[武夷山市|武夷山市]]交界处，海拔2157米，为[[中国|中国]]大陆东南最高峰。其它较高的山峰有[[独竖尖|独竖尖]]（海拔2128米，江西第二高峰）、[[过风坳|过风坳]]（海拔1887米）。	

=== 水系 ===
全县均属[[信江|信江]]水系，信江干流横贯，在县内主要有两大[[支流|支流]]：[[铅山河|铅山河]]和[[陈坊河|陈坊河]]，铅山河在[[河口镇_(铅山县)|河口镇]]东汇入信江，陈坊河在[[汪二镇|汪二镇]]北汇入信江。建有中型[[水库|水库]]两座：[[丰产水库|丰产水库]]、[[铁炉水库|铁炉水库]]。<ref name="江西省地图册"/>

== 人口 ==
绝大部分居民为[[汉族|汉族]]，少数民族包括[[畲族|畲族]]（总人口为3700余人）和少量的蒙、回、藏、苗、壮、满、黎族。<ref name="铅山县历史沿革"/><ref>{{cite web |title=铅山县畲族变迁基本情况 |url=http://www.jxyanshan.gov.cn/?do=detail&ids=1304 |publisher=铅山县人民政府网 }}</ref>

== 文化 ==
[[宋|宋]][[淳熙|淳熙]]二年（1175年）四月，[[朱熹|朱熹]]、[[吕祖谦|吕祖谦]]、[[陆九龄|陆九龄]]、[[陆九渊|陆九渊]]在鹅湖寺聚会，史称“[[鹅湖之会|鹅湖之会]]”。后在寺西建有“四贤祠”。

[[淳熙|淳熙]]十五年（1188年）冬，[[辛弃疾|辛弃疾]]与[[陈亮|陈亮]]会于鹅湖寺。[[庆元|庆元]]二年（1196年），辛弃疾在上饶[[带湖|带湖]]的住宅失火，遂移居铅山瓢泉。[[开禧|开禧]]三年（1207年）九月十日，辛弃疾病逝，终年68岁，葬虎头门彭家湾洋源山。

[[淳祐|淳祐]]十年（1250年），宋理宗为鹅湖“四贤祠”改名“文宗书院”，现称[[鹅湖书院|鹅湖书院]]。

蒋士铨（1725—1784）清代戏曲家，文学家。字心馀、苕生，号藏园，又号清容居士，晚号定甫。铅山（今属江西）人。乾隆二十二年进士，官翰林院编修。乾隆二十九年辞官后主持蕺山、崇文、安定三书院讲席。精通戏曲，工诗古文，与袁枚、赵翼合称江右三大家。士铨所著《忠雅堂诗集》存诗二千五百六十九首，存于稿本的未刊诗达数千首，其戏曲创作存《红雪楼九种曲》等四十九种。 

古代以[[連四紙|連四紙]]產地聞名。
