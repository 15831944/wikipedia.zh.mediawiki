'''亳州市第一中学'''，简称'''亳州一中'''，创建于1910年，是[[安徽省|安徽省]][[亳州市|亳州市]]的一所[[高级中学|高级中学]]，为安徽省示范高中。

== 校史 ==
亳州一中创建于1910年。1959年，被列入安徽省属重点中学。1999年，被评为安徽省级示范高中。<ref name=jj>{{cite web|url=http://www.bzyzh.com/Category_24/Index.aspx|title=安徽省亳州市第一中学简介|publisher=亳州一中校园网|accessdate=2018-07-03}}</ref>

== 现状 ==
学校现有两个校区，在职教职工402人，其中特级教师5人，高级教师151人。现有教学班92个，在校学生近6000人。<ref name=jj/>

== 知名校友 ==
* [[卢秉恒|卢秉恒]]：中国工程院院士。
* [[张军_(经济学家)|张军]]：复旦大学长江特聘教授。
* [[邵正元|邵正元]]：天文学家。
* [[仲星火|仲星火]]：电影演员。
