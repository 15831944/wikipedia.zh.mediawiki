'''[[臺北監獄|臺北監獄]]圍牆遺蹟'''位於[[臺北市|臺北市]][[大安區_(臺北市)|大安區]][[金山南路|金山南路]]二段44巷（愛國東路底、金山南路交叉口附近），也就是今[[中華電信|中華電信]]台北南區營運處（電信金山大樓）的周圍。分倨中華電信台北南區營運處南北兩端各百公尺的兩石材邊牆，不但是[[臺北府城|臺北府城]]城牆僅存的遺蹟，也是[[臺灣日治時期|臺灣日治時期]][[臺北刑務所|臺北刑務所]]的四周圍牆。

== 背景 ==
1895年，台灣邁入[[臺灣日治時期|日治時期]]。[[臺灣總督府|臺灣總督府]]為了維持社會治安與鎮壓[[台灣抗日運動|反日運動]]，研擬興建臨時性監獄。在台灣北部方面，總督府特地將位於台北城內的[[清朝|清朝]]參府[[衙門|衙門]]略加修改，成為台灣第一座西方式監獄。名稱就定為台灣總督府台北刑務所。

== 拆城與監獄 ==
在城內興建刑務所同時，入台北城後的日本統治者，於1899年公告第一次市區改正計畫後，開始以街地整理與貫通道路為由，動手拆除[[台北府城|台北府城]]城牆及城內[[文廟|文廟]]、[[武廟|武廟]]。台灣總督府並將台北石城城牆的[[安山岩|安山岩]]與[[唭哩岸|唭哩岸]]石堆積於城外四處。不久，1904年底，除了刻意留存的四座城門之外，台灣總督府幾乎已將台北城所有城牆拆除。從南城牆拆除的石材，則大多運往南門附近。

1910年代左右，以台灣總督府主導的維護治安與鎮壓行動日漸興盛，台籍犯人增多。為此，總督府特別使用城牆遗留下的石材興建新的台北刑務所，而此刑務所即為戰後的台北監獄。

== 戰後 ==
充當犯人牢獄的該刑務所，於[[第二次世界大戰|第二次世界大戰]]戰後仍被[[中華民國政府|中華民國政府]]拿來當作關押犯人的「臺北監獄」，設於圍牆內的該監獄直至1963年才漸次拆遷移監，僅存四周圍的邊牆。

今日建物全部拆除殆盡的臺北監獄，只剩南北兩邊，高約三公尺，長約百餘公尺的石牆。而該兩堵牆所用石材，大多取自1900年臺灣總督府所拆除的臺北城東與南城牆所遺留下來的[[安山岩|安山岩]]及唭哩岸石。因為有著雙重且二維的歷史空間標定，因此於1998年被[[台北市政府|台北市政府]]指定為市定古蹟。

== 其他 ==
[[File:Remains_of_Taipei_Prison_Gate_and_Yamaha_Jog_20091229.jpg|thumb]]
臺北刑務所於日治時期為總督府對死刑犯的絞刑行刑所在，抗日人士[[羅福星|羅福星]]即於此被日人絞殺。圍牆遺址有處石造拱門，即是舊時的「運屍門」，現已用紅磚封死。

2017年，[[臺北捷運|臺北捷運]][[萬大線|萬大線]]人行道側溝工程挖到清代台北城城牆被日本政府拆除後的石材移到[[專賣局|前公賣局]]後方做的排水溝遺址，開挖部分並無涉及捷運站體施工，[[臺北市政府文化局|文化局]]將協助[[台北市捷運局|捷運局]]原地保留回填處理，挖到的遺跡也考慮以[[捷運北門站|捷運北門站]]現有的文物展示方式處理。<ref>[http://news.ltn.com.tw/news/politics/breakingnews/2091016 獨家》捷運萬大線挖到古城牆？議員籲捷運局停工]，自由時報</ref>
