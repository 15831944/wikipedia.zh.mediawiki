{{台灣交通}}
{{台灣歷史}}

本條目講述'''[[臺灣|臺灣]][[交通|交通]]'''概況及其歷史，包括各種[[運輸|運輸]]系統、[[載具|載具]]、[[公共交通|大眾運輸工具]]、[[管道運輸|管道運輸]]等。目前臺灣交通的最高主管機構是[[中華民國交通部|中華民國交通部]]。

==歷史==
台灣島由於地形關係，東西部交通被[[中央山脈|中央山脈]]所阻擋，南北部交通由[[臺灣河流列表|河流]]所切斷。因此早期臺灣交通極度依賴海運，甚至島嶼南北部連絡常常需要乘船至中國大陸轉乘。「[[一府二鹿三艋舺|一府二鹿三艋舺]]」的稱謂，更是說明[[臺南市|臺南]]、[[彰化縣|彰化]][[鹿港鎮|鹿港]]、[[臺北市|臺北]][[萬華區|萬華]]在當時的[[航運|航運]]地位。

[[清朝|清朝]][[臺灣清治時期|治理臺灣]]的末期時間，才開始大規模整理南北向的陸上交通。至於東西部的連結，[[沈葆楨|沈葆楨]]、[[吳沙|吳沙]]等人分別開闢[[淡蘭古道|淡蘭古道]]、[[八通關古道|八通關古道]]等橫越[[雪山山脈|雪山山脈]]、中央山脈的山間小徑，做為東西部往來的捷徑。時任[[巡撫|巡撫]][[劉銘傳|劉銘傳]]更計畫興建[[臺灣鐵路_(清朝)|臺灣鐵路]]，從[[基隆市|基隆]]至抵臺南。但後繼官員為減輕財政負擔，雖然這條鐵路已測量至[[大甲溪|大甲溪]]，僅完成基隆至臺北、臺北至新竹兩條路線，但已領先當時[[中國|中國]]其他地區的鐵路建設。

在[[馬關條約|馬關條約]]簽訂後，[[日本|日本]][[台灣日治時期|取得台灣主權]]，而今日臺灣交通建設的基礎，也是奠定於此時期。不但[[縱貫鐵路|西部鐵路]]由[[新竹市|新竹]]延伸至[[屏東縣|屏東縣]][[枋寮鄉|枋寮]]，東部的[[花蓮市|花蓮]]、[[臺東市|臺東]]之間，也建有窄軌鐵路。整個西部平原，[[糖業鐵路|糖業]]、鹽業等專用[[輕便鐵路|輕便鐵路]]更是密佈。至於空中交通，亦是奠定於此時期，但多以軍事機場為優先。[[台灣日治時期|日本殖民時期]]，臺灣的公路及鐵路，遵循日本[[道路通行方向|靠左行駛]]的道路規則；1945年，[[中華民國|中華民國]]接手統治臺灣之後，除了鐵路系統沿用舊制，公路皆改為靠右行駛。

[[中華民國政府|中華民國政府]]於[[國共內戰|國共內戰]]遷臺後，由於面對[[海峽兩岸關係|兩岸]]情勢的危機，因此早期的交通建設以[[軍事|軍事]]防衛目的為主，例如「[[南北平行預備線|糖鐵南北平行線]]」及[[西螺大橋|西螺大橋]]的修築，或[[中部橫貫公路|中部橫貫公路]]的開闢等。近數十年來，[[十大建設|十大建設]]、[[鐵路電氣化|鐵路電氣化]]、環島鐵路計畫、[[高速公路|高速公路]]路網一一完工，加上國內航空路網的建構，以及各大城市[[捷運|軌道交通系統]]、南北[[高速鐵路|高速鐵路]]的興築，今日臺灣交通的面貌逐漸完成。

====公路====
{{Main|台灣公路|臺灣機車}}
公路系統遍及台灣島及澎湖群島各鄉鎮，公路系統依公路法規定共分為五級: '''[[中華民國國道|國道]]1,050公里'''、'''[[台灣省道|省道]]5,262公里'''、'''[[台灣縣道|縣道]]([[台灣市道|市道]])3,602公里'''、'''[[台灣鄉道|鄉道]]([[台灣區道|區道]])11,383公里'''、'''[[台灣專用公路|專用公路]]413公里'''，總長度43,206-{[[公里|公里]]}-（2017年止）<ref>{{Cite web|url=https://www.motc.gov.tw/ch/home.jsp?id=64&parentpath=0,6|title=道路長度及橋梁座數|accessdate=2019-01-18|work=www.motc.gov.tw|language=zh-TW}}</ref>。除以上五級公路外，其餘為普通道路，道路又分由水土保持局管理的農路、由林務局管理的林道、位於都市計劃區內的市區道路（含市區快速道路、巷弄）和其它普通產業道路及私人道路。

汽車運輸業依公路法規定可分為公路汽車客運業、市區汽車客運業、遊覽車客運業、計程車客運業、小客車租賃業、小貨車租賃業，以及汽車貨運業等。

====軌道運輸====
{{Main|台灣鐵路運輸}}

軌道運輸服務主要分為傳統鐵路（[[臺灣鐵路管理局|台鐵]]）、[[高速鐵路|高速鐵路]]（[[台灣高速鐵路|台灣高鐵]]）、專用鐵路（如[[阿里山森林鐵路|阿里山林鐵]]）以及大眾捷運法所規定的[[捷運|大眾捷運系統]]、（[[輕軌|輕軌]]、[[纜車|纜車]]）等系統。

台灣島在[[台灣日治時期|日治時期]]，主要發展鐵路運輸，各種產業鐵路曾密如蛛網遍佈全島。[[第二次世界大戰|第二次世界大戰]]結束進入[[台灣戰後時期|民國時期]]後，[[台灣政府|台灣政府]]受到[[美國交通|美式公路主義]]影響，改以公路為主要交通方式，例如[[臺北市|臺北市]]發展[[城市軌道交通系統|城市軌道交通系統]]時程，就遲上[[亞洲|亞洲]]其他新興[[首都|首都]]許多；公路建設的興盛，也造成鐵路運量降低，許多鐵路線因而走入歷史。直至1990年代以後，隨著全台快速、高速公路建設逐漸完備，以及都市面臨雍塞、污染問題，政府方又重視軌道運輸，推動如[[臺鐵捷運化|臺鐵捷運化]]等政策。

2006年8月22日，[[中華民國國家發展委員會|行政院經濟建設委員會]]正式宣佈未來以軌道運輸取代公路建設，以解決環境問題。2007年起，由於大量軌道運輸路線的興築，以及公路建設已近完備的原因，軌道與公路的建設預算比例已達到6:4{{Fact|time=2009-04-23T12:45:17+00:00}}<!-- 本模板調用由 {{subst:Fact/auto}} 自動產生 -->。2011年，臺鐵先後開闢[[沙崙線|沙崙線]]、[[六家線|六家線]]，為臺鐵19年以來新線。

===海上交通===
{{Main|台灣港口}}
台灣島四面環海，海上交通在國共內戰後期的背景下長期遭受管制，國際物流仰賴政府特許航運公司透過商船運輸，也造就台灣有兩大對外的港口，如：基隆港、高雄港。且由於台灣島四面環海，漁港數量多達231處[[漁港|漁港]]，其中本島有155處（占三分之二），離島76處（占三分之一）。
====國際商港====
[[File:KeelungHarbor_west_002.JPG|250px]]
台灣島的國際商港、國內商港的管理業務，由[[中華民國交通部|交通部]][[交通部航港局|航港局]]負責管理商港（分為國際商港、國內商港）所管轄。

交通部以國營之[[臺灣港務公司|臺灣港務公司]]經營各商港，並劃定各港務分公司轄區；離島方面，交通部委託[[金門縣政府|金門縣政府]]、[[連江縣政府|連江縣政府]]成立港務處，管理縣內國內商港。目前有'''4座國際商港'''、'''4座國際商港輔助港口'''、'''5座國內商港'''。

== 現況問題 ==
# 計程車沒有落實總量管制，造成計程車車輛供過於求。<ref>[https://www.thenewslens.com/article/57647 修法提高Uber罰金的荒謬（上）：空車率近40％的牌照濫發才是計程車殺手]</ref>
# 不夠完善的[[大眾運輸系統|大眾運輸系統]]，造成私人交通運具使用率遠於大眾運輸使用率。
# 汽機車分流之道路規劃未符合實際需求，[[摩托車|機車]]使用者沒有擁有足夠之路權，且直接影響行車安全，使得民怨四起。<ref>[http://trocmra.blogspot.tw/2015/02/1189.html 摩托族行不安 年死1189人-台灣摩托車友協會]</ref>
# 私有運具持有率極高，造成許多停車問題。例如路邊違規停車、併排停車。造成車道數減少、車速降低、公車無法停靠車站，且違規停車之車輛駕駛人於上下車時也常造成後方車輛之行車安全。
# {{Fact|縣市政府未針對道路的發展改善鄉道路線，亦不公佈其路線的詳盡路線表，使得鄉道系統凌亂不堪。|time=2014-07-21T05:17:52+00:00}}
# 部分駕駛人不遵守交通規則，造成許多交通問題。例如汽車逆向超車、汽機車不禮讓行人、機車在車陣中亂鑽、公車靠站時常發生機車在乘客上下車之處行駛經過而發生危險的現象。
