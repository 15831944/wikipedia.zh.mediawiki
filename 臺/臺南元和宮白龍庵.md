'''臺南元和宮白龍庵'''，全稱'''臺南大銃街元和宮全臺白龍庵'''，位在臺灣[[臺南市|臺南市]][[北區_(台南市)|北區]]，主祀[[保生大帝|保生大帝]]與[[五福大帝|五福大帝]]{{r|大道公}}。該廟舊稱「水仔尾真人廟」，後改稱「大銃街元和宮」，白龍庵毀於日治時期後，部分文物與神像安置到元和宮內，所以後來廟名才又加上「白龍庵」{{r|大道公}}。

== 沿革 ==
根據《[[臺灣縣志|臺灣縣志]]》的記載，該廟是在康熙卅五年（1696年）由里人建於[[臺灣府城|臺灣府城]][[鎮北坊|鎮北坊]]水仔尾，原稱稱「水仔尾真人廟」，後改稱「大銃街元和宮」{{r|大道公}}{{r|傅}}。後來在乾隆年間與道光三十年（1850年）曾進行重修{{r|大道公}}。

白龍庵之由來則是因為總鎮標署{{NoteTag|在今臺南公園對面的停車場{{r|大道公}}。}}的[[福州|福州]][[籍貫|籍]][[軍人|官兵]]因思鄉與水土不符等原因，而迎奉[[福州白龍庵|福州白龍庵]][[五福王爺|五福王爺]][[軟身神像|軟身神像]]來臺，於總鎮標署衙門內建白龍庵{{r|大道公}}。

日治時期，元和宮在明治廿九年（1896年）進行擴建{{r|大道公}}。兩年後（1898年）總鎮標署被改建成日本陸軍經理部廳舍，署衙內的白龍庵也被拆毀，後來部分文物與神像被安置到元和宮內{{r|大道公}}。而元和宮此後在大正三年（1914年）、昭和十一年（1936年）也發起重修{{r|大道公}}。

二次大戰後，該廟在民國卅九年（1950年）在廟後成立保生慈善會，聘請醫師低價看診{{r|大道公}}。而後到了民國四十九年（1960年）再次重修，三年後（1963年）落成，隔年（1964年）舉辦慶成祈安建醮{{r|大道公}}。民國七十九年（1990年）再次重修，此次為防淹水，特別在請示神明後將地板升高一尺半{{r|大道公}}。民國八十一年（1992年）完工後，於民國八十七年（1998年）舉辦慶成祈安三朝建醮{{r|大道公}}。

== 祀神 ==
元和宮內除了原本所供奉的保生大帝外，還有白龍庵的五福大帝，此外尚有後殿的觀音菩薩、福德正神、註生娘娘，中軍府的白龍庵畢中軍，眾壽堂的陰陽都總管，如意増壽堂的顯靈公駕前什家將等神{{r|傅}}。

== 家將團 ==
[[File:白龍庵如意增壽堂.jpg|thumb]]
白龍庵如意增壽堂什家將為臺灣最早的家將團，其陣法曾流傳至屏東東港共善堂與嘉義古桃城拱吉堂等地{{r|南文}}{{r|王}}{{r|大道公}}。過去五福大帝底下各有一家將團（如下表）{{r|如性}}{{r|述}}，張部顯靈公十將、鍾部應靈公八將、劉部宣靈公八將、史部揚靈公六將、趙部振靈公八將{{r|大道公}}，但今只剩張公爺的如意增壽堂什家將{{r|南文}}。其完整配置分別是甘柳謝范四大將軍、春夏秋冬四季大神，以及文差、武差、文判、武判、什役所組成{{r|南文}}。該堂仍保有1903年時的出團照片{{r|王}}。

如意增壽堂什家將曾獲得中華民國教育部頒發薪傳獎，但後來教練陳欽明認為臺灣的廟會、家將文化變質，所以停止家將團活動也不收徒{{r|王}}。直到2015年經過臺南市政府與北區區公所兩年來的拜訪下，才同意收徒並配合文化局協助保存什家將的文化資產{{r|王}}。

== 建築 ==
元和宮的建築在主軸線上依序是三川殿、拜殿、正殿、次拜殿與後殿，左側殿為中軍府，右側殿為眾壽堂，右廂房為如意増壽堂{{r|傅}}。
