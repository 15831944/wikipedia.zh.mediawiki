'''臺北行旅廣場'''（'''Taipei Travel Plaza'''，原名'''交六廣場'''）為[[臺北市|臺北市]][[中正區_(臺北市)|中正區]][[忠孝西路_(臺北市)|忠孝西路]]一段與[[重慶南路_(臺北市)|重慶南路]]一段路口東北側之綠地，為[[台北市市長|台北市市長]][[柯文哲|柯文哲]]提出的'''西區門戶計畫'''之一環，總占地約1.6公頃，於2017年1月26日完工開放（廣場南方之'''交六公車站'''則於同年4月29日開放使用）。廣場位於車水馬龍的[[臺北車站|臺北車站]]旁邊，使往返台北的旅客能在擁擠的車站旁，找到一塊整齊開闊的綠地；也使台北居民能在嘈雜的都市中，擁有一處可以安靜放鬆的秘境。

臺北行旅廣場之前身為臺北西站[[臺北西站A棟|A棟]]、[[臺北西站B棟|B棟]]，在[[臺北市政府|臺北市政府]]西區門戶計畫中遭到拆除，並與原站後方之綠地合併為現在之臺北行旅廣場。廣場南方新建了'''交六公車站'''，將原本停靠於交七公車站（位於臺北車站南側廣場）之[[臺北市聯營公車|臺北市聯營公車]]部分路線（14、39（含夜間公車）、232副線、274、299（含區間車）、539、652、藍1及忠孝新幹線）遷移至此停靠<ref>{{Cite web |url = http://www.pto.gov.taipei/ct.asp?xItem=288448041&ctNode=22734&mp=117041 |title = 臺北車站交6公車站區啟用與停靠位置調整 | publisher=台北市公共運輸處 |date = 2017-04-20|accessdate = 2017-04-20}}</ref>，成功紓解交七公車站因停靠區長度不足所造成的公車壅塞問題。而北側廣場之總綠地面積達5,056平方公尺，可容納1,500至2,000人，可以舉辦各種活動並供民眾休憩使用<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20170115/1034784/ |title = 【展望片】台北西站綠化大改造　「台北行旅廣場」變身國門客廳 | publisher=蘋果日報 |date = 2017-01-15|accessdate = 2017-01-15}}</ref>。

==命名==
臺北行旅廣場原名為交六廣場，台北市政府於2016年12月26日至2017年1月1日舉辦網路投票（i-Voting）徵求新名稱，最後「臺北行旅廣場」以863票勝出第二名「川流之心」（639票）及第三名「臺北綠驛」（578票）<ref>{{Cite web |author=林媛玲、陳思豪|url = http://www.appledaily.com.tw/realtimenews/article/new/20170103/1026720 |title = 交六廣場命名結果出爐　以後改叫臺北行旅廣場 | publisher=蘋果日報 |date = 2017-01-03|accessdate = 2017-01-03}}</ref>。臺北行旅廣場的命名概念為：'''期望國內外旅客可在此閱讀不一樣的臺北，外鄉遊子也可以便捷地返鄉，並從這個廣場開啟臺北故事新的序幕'''<ref>{{Cite web |url = http://www.gov.taipei/ct.asp?xItem=259829162&ctNode=5158&mp=100001 |title = 交六正名「臺北行旅廣場」 1/26開放使用 |publisher=台北市政府新聞稿 | date = 2017-01-25| accessdate = 2017-01-25}}</ref>。
