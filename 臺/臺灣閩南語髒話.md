'''[[臺灣話|臺灣閩南語]][[髒話|髒話]]'''，或'''臺灣髒話'''、'''台語髒話'''，臺灣話稱之為'''垃圾話'''（{{lang-zh-min-nan|lah-sap-uē}}），指的是以臺灣閩南語為媒介所說出的辱人之語。1921年出版的《[[台灣風俗誌|台灣風俗誌]]》，是[[台灣日治時期|台灣日治時期]]日本政府派出大量的學者研究、訪查的髒話記錄<ref>{{cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20150707/643429/|date=2015-07-07|title=這是真的！　日本派人研究台語髒話|author=劉人豪|publisher=蘋果日報（台灣）}}</ref>。

== 用字==
=== 動作 ===
==== 姦 ====
{{main|幹你娘}}
'''姦'''（{{lang|zh-min-nan|kàn}}）為臺語中最為常見之髒話、情緒語言，被喻為「臺灣國罵」。其意義為動詞「性交」，借用為「強姦」，是具有極度侵犯性的詞語。其用法相當於[[現代標準漢語|華語]]（俗稱[[北京官话|北京話]]<!--[[北京話|北京話]]會連結到[[官话|官話]][[京師片|京師片]]下的北平話-->）的「肏」以及[[粤语|粵語]]中的「[[屌|屌]]」。由於[[中華民國|中華民國]]於戰後接收[[臺灣|臺灣]]後積極推動[[國語運動#中華民國時期|國語運動]]，造成[[臺灣人|臺灣人]]對臺語的文書並不熟悉，因此臺灣人經常以華語之借音字來書寫臺語以達義。<!--在髒話的書寫方面，亦存在著這種情形。-->「'''幹'''」是最常見的華語借音字，另外「淦」、「贛」、「-{干}-」等字的用法亦常見。現代臺語單唸一個字時，則多讀作高降的[kan˥˧]，係原陰去聲調（第3聲）的變調，而其後的受詞省略，此為閩南語常見的意猶未盡的言談標誌。

「姦」字另有許多不同的[[幹你娘#延伸用詞|衍伸用法]]：
* [[幹你娘|姦恁娘]]（{{lang|zh-min-nan|kàn lín-niâ}}）
:字面意義為「性侵你母親」，用法同國語「[[草泥马|肏你媽]]」與粵語「[[屌你老母|屌你老母]]」，常俗寫成「幹你娘」或「幹您娘」，又稱為「三字經」。罵此語者通常沒有字面上的這種念頭，純粹只是用來抒發憤怒情緒之語。
* 姦恁老母（{{lang|zh-min-nan|kàn lín lāu-bú}}）
* 姦恁祖媽（{{lang|zh-min-nan|kàn lín tsóo-má}}）
* 姦恁老師（{{lang|zh-min-nan|kàn lín lāu-su}}）
:「師」字的讀音有時為華語發音。而偏泉腔中，「師」字也能讀為sir，近似於華語。
* 姦恁娘[[阴道|膣]][[屄|屄]]（{{lang|zh-min-nan|kàn lín-niâ tsi-bai}}）
:亦作「姦恁娘膣毴」。
* 姦你真爽（{{lang|zh-min-nan|kàn lí tsin-són}}）
:字面意義為「性侵你真爽」。豬哥亮在《[[大囍臨門|大囍臨門]]》的角色叫做李金爽，用台語就叫做「哩金宋」。
* 姦甲會牽絲（{{lang|zh-min-nan|kàn kah ē khan-si}}）
:形容[[性交|性交]]時或之後[[生殖器|性器官]]上的[[體液|體液]]，如[[陰道分泌液|陰道分泌液]]、[[精液|精液]]，或兩者之混合液。
* 恁娘（[[恁娘卡好|恁娘較好]]）（{{lang|zh-min-nan|lín-niâ（lín-niâ ka-hó）}}）
:「姦恁娘（較好）」的簡稱及衍生語。亦作「恁娘咧」（{{lang|zh-min-nan|lín-niâ--leh}}）。
* 相姦 （{{lang|zh-min-nan|sio-kàn}}）
: [[性交|性交]]之意
:狗姦懟  （gaò gań doiû)
:被狗強姦（性侵）之意，亦衍生為，「互狗姦姦咧」  （hô gaò gàn gân leh)
:

==== 使 / 駛 ====

'''使'''（{{lang|zh-min-nan| sái}}) 亦為台灣閩南語中常見之髒話、情緒語言，起源於[[泉州話|泉州話]]。 同'''姦'''（{{lang|zh-min-nan|kàn}}）一樣，相當於國語的「肏」， 義為動詞「性交」，具有極度侵犯性的詞語。亦可寫成「'''駛'''」。因此，泉系閩南話常忌說「駛」（sái）字，不說「駛車」（sái-tshia）而說「開車」（khui-tshia）；不說「駛牛車」（sái-gû-tshia）而說「牽牛車」（khan-gû-tshia）；不說「駛田」（sái-tshân）而說「犁田」（luê-tshân」；不說「順風好駛船」（sái-tsûn）而說「順風好行船」（kiânn-tsûn）。

* 駛恁娘 / 使恁娘（{{lang|zh-min-nan|sái lín-niâ}}）
: 以「駛」而言其字面意義為「駕駛」；以「使」而言其字面義為「使勁」（形容性交時的動作）。 
* 駛恁老母（{{lang|zh-min-nan|sái lín lāu-bú}}） / 使恁母（{{lang|zh-min-nan|sái lín bú}}）
* 駛恁殼 （{{lang|zh-min-nan|sái-lín-khak}}） / 使𪜶殼的 （{{lang|zh-min-nan|sái-lín-khak}}--ê）
: 以「殼」言其字面義為女性生殖器

==== 躄 ====
'''躄'''（{{lang|zh-min-nan|phih}}) 同'''姦'''（{{lang|zh-min-nan|kàn}}）一樣，起源於[[漳州話|漳州話]]， 「躄」（phih）的本義為「臉朝下仆倒」，以此動作引申性交的動作，相當於國語的「肏」，具有極度侵犯性的詞語。此用法起源於[[漳州話|漳州話]]。從「躄」字面意思為「有行進不止的樣子」（形容性交時的動作）。

* 躄恁娘（{{lang|zh-min-nan|phih lín-niâ}}）
* 去予人躄（予人 {{lang|zh-min-nan|phőng}} / {lang|zh-min-nan|hông}}）

==== 肏 ====
'''肏'''（{{lang|zh-min-nan|tshoh}}) 同'''姦'''（{{lang|zh-min-nan|kàn}}）一樣， 義為動詞「性交」，具有極度侵犯性的詞語。

* 肏恁娘（{{lang|zh-min-nan|tshoh lín-niâ}}）

==== 箍 ====
'''箍'''（{{lang|zh-min-nan|khoo}}) 同'''姦'''（{{lang|zh-min-nan|kàn}}）一樣，相當於國語的「[[草泥馬|肏]]」, 義為動詞「性交」，具有極度侵犯性的詞語, 主要使用於[[同安|同安]]

* 箍恁娘（{{lang|zh-min-nan|khoo lín-niâ}}）

==== 哭 ====
'''哭'''（{{lang|zh-min-nan|khàu}}）亦為閩南語中常見之髒話、情緒語言。由於過去[[國語運動#中華民國時期|國語政策]]之影響，今多以[[中華民國國語|國語]]（[[現代標準漢語|現代標準漢語]]）書寫為「靠」，有時會與[[河南话|河南话]]的「尻」（意同「姦」、「肏」）<ref>{{cite web|url=http://book.sina.com.cn/nzt/ele/city1000/101.shtml|title=粗口叫“我尻”|work=《中国城市的1000个细节》|publisher=新浪读书|date=2005-07-19|accessdate=2015-07-12|archiveurl=https://web.archive.org/web/20141220110226/http://book.sina.com.cn/nzt/ele/city1000/101.shtml|archivedate=2014-12-20|deadurl=yes}}</ref>混淆做「我靠」。

「哭」字的衍伸用法有：
* 哭枵（{{lang|zh-min-nan|khàu-iau}}）
:字義是當小孩或嬰兒「因肚子餓而哭嚎」，以人「因飢餓而吵鬧」之比喻，來罵對方無理取鬧。屬於閩南語中不禮貌之粗話，用來表示糟糕、不滿、驚訝或遺憾。常俗寫成「靠-{么}-」、「靠夭」、「靠腰」。
* 哭爸（{{lang|zh-min-nan|khàu-pē}}）／哭爸哭母（{{lang|zh-min-nan|khàu-pē-khàu-bú}}）
:字義為喪父、喪母時，在喪禮中哭喪的行為，用以比喻他人的叫苦或抱怨。用來表示糟糕、不滿、驚訝或遺憾。亦屬於閩南語中不禮貌之粗話，衍伸用詞包括「哭恁爸」。常俗寫成「靠盃」或「靠北」。
:亦作「考妣」，指「家長雙亡」。
* 哀爸叫母（{{lang|zh-min-nan|ai-pē-kiò-bú}}）
:不具粗俗性，屬「哭爸」的衍伸詞，描述人如「哭爹喊娘」般吵吵嚷嚷、呼天搶地

=== 屄 ===
'''[[屄|屄]]'''（{{lang|zh-min-nan|bai}}），為女子的外生殖器官，作髒話時通常與「膣」（{{lang|zh-min-nan|tsi}}）連用。
* 膣屄（{{lang|zh-min-nan|tsi-bai}}）
:亦作「膣毴」，字面意思為[[女性生殖系統|女性生殖器]]，衍伸為喻人言行令人厭惡。常俗寫成「雞掰」或「機掰」，也因此衍生出「雞歪」或「機車」的用法。
* 臭膣屄（{{lang|zh-min-nan|tshàu tsi-bai}}）

=== 𡳞 ===
'''𡳞'''（{{lang|zh-min-nan|lān}}）, 為男子的生殖器，亦可寫成「膦」。
*𡳞鳥（{{lang|zh-min-nan|lān-tsiáu}}）
:亦作 「膦鳥」, 字面意思為[[男性生殖系統|男性生殖器]]
*𡳞脬 （{{lang|zh-min-nan|lān-pha}}）
:亦作 「膦脬」,字面意思為[[陰囊|陰囊]]
*扶𡳞脬（{{lang|zh-min-nan|phôo-lān-pha}}）
:為動詞「拍馬屁」、「諂媚」、「阿諛奉承」之義

=== 姦撟 ===
「撟」（kiāu）的本義係一動詞，係「以工具作為槓桿以增加力道，而把鎖住或卡住的門、鎖撬開，或把石頭等重物舉起。」以此動作隱喻男性的性交動作「'''姦撟'''」（{{lang|zh-min-nan|kàn-kiāu}}），本義亦為性交動作，如今多引申作「詈罵」、「罵三字經」等以粗俗的話語惡言怒罵。俗寫亦作「訐譙」。

*肏姦撟（{{lang|zh-min-nan|tshoh-kàn-kiāu}}）、肏姦搦撟（{{lang|zh-min-nan|tshoh-kàn-la̍k-kiāu}}）
:亦為「辱罵」之義
*使姦撟 （{{lang|zh-min-nan|sái kàn-kiāu}}）
:亦為「辱罵」之義
*媾姦撟（{{lang|zh-min-nan|koo-kàn-kiāu}}）
:亦為「辱罵」之義

=== 其他 ===
* 白目（{{lang|zh-min-nan|pe̍h-ba̍k}}）
:喻人言行令人厭惡、明知故犯。（常不被認定為髒話）
* 白痴（{{lang|zh-min-nan|pe̍h-tshi}}）
:罵人智力低下，意同華語「[[智能障礙|智障]]」，現代華話常以火星文借音寫作「北七」，亦能用阿拉伯數字「87」表示。
