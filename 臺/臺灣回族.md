'''臺灣回族'''，是一個相當稀少的族群，屬於[[外省|外省]]族群。
==知名人物==
*[[白崇禧|白崇禧]]：[[中國國民黨|中國國民黨]]黨員，[[新桂系|新桂系]]將領，中華民國抗日名將，前[[中華民國國防部部長|國防部部長]]（1946 - 1948年）
*[[白先勇|白先勇]]：白崇禧兒子，中華民國旅美作家<ref>{{Cite web |url = http://forums.chinatimes.com/report/people_story/20081004/inside1-1.html |title = 叛逆 白崇禧白先勇的基因密碼 |accessdate = 2018/09/09  }}</ref>
*[[馬步青|馬步青]]：[[馬家軍|馬家軍]]將領，[[國民革命軍|國民革命軍]]中將
*[[馬緒援|馬緒援]]：馬步青兒子，中華民國國軍中將，前國防部常務次長（任期：1986 - 1991年）
*[[馬繼援|馬繼援]]：馬步芳兒子，馬家軍將領，[[中華民國國軍|中華民國國軍]]中將，前[[國民大會代表|國民大會代表]]
*[[常東昇|常東昇]]：中華民國摔跤選手，中華民國國軍上校<ref>{{Cite web |url = http://www.changshuaijiao.org/index.php?includePath=read&pageSet=essay&rssRead=5,102,102 |title = 常東昇宗師練功與生平傳奇 |accessdate = 2018/09/09  }}</ref>
*[[劉文雄_(基隆市)|劉文雄]]：曾任中國國民黨籍[[臺灣省議員|臺灣省議員]]、[[親民黨|親民黨]]籍[[立法委員|立法委員]]、親民黨副秘書長
*[[哈遠儀|哈遠儀]]：中華民國知名主持人，前[[中天新聞|中天新聞]]主持人，現為[[中視新聞|中視新聞]]主播
