'''臺北大學'''指：
* '''[[國立臺北大學|國立臺北大學]]'''，慣稱為'''臺北大學'''。

== 公立大專院校 ==
* [[國立臺北教育大學|國立臺北教育大學]]，簡稱北教、北教大，一所位於臺灣臺北的教育大學。
* [[國立臺北藝術大學|國立臺北藝術大學]]，簡稱北藝、北藝大，位於臺北市北投區關渡地區，以藝術表演、人文創作等科系為主的公立大學。
== 公立技職專院校 ==
* [[國立臺北科技大學|國立臺北科技大學]]，簡稱北科、北科大，問路時很容易遇到和台北大學混淆的一所研究型科技大學。
* [[國立臺北商業大學|國立臺北商業大學]]，簡稱北商、北商大，一所位於臺北中正區的科技大學。
* [[國立臺北護理健康大學|國立臺北護理健康大學]]，簡稱北護、北護大，是一所位於臺北的國立科技大學，以教授健康科學課程為主。

== 私立大專院校 ==
* [[臺北醫學大學|臺北醫學大學]]，簡稱北醫、臺北醫大，位於臺北市信義區的醫學專門大學，前身為臺北醫學院。
== 私立技職專院校 ==
* [[臺北城市科技大學|臺北城市科技大學]]，簡稱城科大、城科，是一所位於臺北市北投區的科技大學，前身為光武工專。
* [[台北海洋科技大學|台北海洋科技大學]]，簡稱北海科大，為一所私立科技大學，校址位於臺北市士林區和新北市淡水區，前身為[[台北海院|海院]]、中國海專，2017年8月1號改名。
