'''台鐵地下化'''，為[[交通部臺灣鐵路管理局|交通部臺灣鐵路管理局]]（簡稱台鐵）[[台灣鐵路管理局路線列表|營運路線]]對於[[臺灣西部|臺灣西部]]各[[都市|都市]]的阻隔及對地面交通的影響提出的改善方案中的地下[[鐵路|鐵路]]（簡稱地鐵，或稱[[鐵路地下化|鐵路地下化]]）方案。

此方案的實施範圍為[[縱貫線|縱貫線]]全線各路段，以「專案」作為各工程的計算單位。工程專案由[[交通部鐵道局|交通部鐵道局]]及交通部臺灣鐵路管理局工務處臺北工務段（僅辦理桃園鐵路地下化）專責辦理，部分工程已經列入[[新十大建設|新十大建設]]的[[臺鐵捷運化|臺鐵捷運化]]計畫當中。

== 地下鐵路初期專案 ==
包括第一期「[[台北鐵路地下化專案#台北車站地下化|台北車站鐵路地下化工程]]」、第二期「[[台北鐵路地下化專案#鐵路地下化東延松山工程專案〈萬板專案〉|萬板專案]]」、第三期「[[松山專案|松山專案]]」、第四期「[[南港專案|南港專案]]」之工程，及「高雄專案」規劃皆是地鐵處時期辦理的。

===臺北鐵路地下化專案===
{{main|臺北鐵路地下化專案}}
[[臺北鐵路地下化專案|臺北鐵路地下化專案]]係將[[南港車站|南港]]、[[松山車站_(台灣)|松山]]、[[臺北車站|臺北]]、[[萬華車站|萬華]]、[[板橋車站_(台灣)|板橋]]五站改建為地下化車站，並同步施作高鐵臺北段路廊。由[[交通部臺北市區地下鐵路工程處|交通部臺北市區地下鐵路工程處]]。

該專案範圍北起[[汐科車站|汐科車站]]南方，南至[[浮洲車站|浮洲車站]]北方止，施工路段總長度約22.672公里。目前臺北鐵路地下化專案所有工程均已完工，並於2011年10月24日全面通車，僅剩高鐵南港—汐止基地間路段擱置中。

== 地下鐵路中期專案 ==
===高雄市區鐵路地下化計畫===
{{main|高雄市區鐵路地下化計劃}}
[[高雄市區鐵路地下化計畫|高雄市區鐵路地下化計畫]]由[[交通部鐵路改建工程局|交通部鐵路改建工程局]]辦理，計畫範圍北起[[新左營車站|新左營車站]]南方，南至[[鳳山車站|鳳山車站]]南方止，施工路段總長度約15.37公里。該計畫復設[[內惟車站|內惟]]、[[鼓山車站|鼓山]]、[[三塊厝車站|三塊厝]]三座車站；增設[[美術館車站|美術館]]、[[民族車站|民族]]、[[科工館車站|科工館]]、[[正義車站|正義]]等四座通勤車站；並將現有[[左營車站|左營]]、[[高雄車站|高雄]]、[[鳳山車站|鳳山]]三站改為地下化車站，高雄市區鐵路地下化計劃的第一階段工程已於2018年10月14日第一階段切換通車，而第二階段工程則預計於2023年完工。

== 地下鐵路後期計畫 ==
===臺南市區鐵路地下化計畫===
{{main|臺南市區鐵路地下化計劃}}
[[臺南市區鐵路地下化計畫|臺南市區鐵路地下化計畫]]由交通部鐵路改建工程局辦理，計畫範圍北起[[大橋車站|大橋車站]]南方，南至[[保安車站|保安車站]]北方止，施工路段總長度約8.6公里，並計畫延伸至[[永康車站|永康車站]]。

該將現有[[臺南車站|臺南車站]]改建為地下化車站設置二座島式月台及四股道；增設[[南臺南車站|南臺南]]、[[林森車站|林森]]兩站，後期將增設[[康橋車站|康橋車站]]三座通勤車站，預計在2024年6月完工。

===臺鐵都會區捷運化桃園段地下化計畫===
{{main|臺鐵都會區捷運化桃園段地下化計劃}}
[[臺鐵都會區捷運化桃園段地下化計畫|臺鐵都會區捷運化桃園段地下化計畫]]全長17.15公里。該計畫將現有[[桃園車站|桃園]]、[[內壢車站|內壢]]、[[中壢車站|中壢]]改建為地下車站；增設[[鳳鳴車站_(新北市)|鳳鳴]]、[[中路車站|中路]]、[[永豐路車站|永豐]]、[[中原大學車站|中原]]、[[平鎮車站|平鎮]]等五座通勤車站。

為配合地下化工程施工，整條路線闢建為25公尺，預計徵收土地5萬3千多平方公尺及拆除建物2萬7千多平方公尺，目前此計劃正在規劃中。

==其他計劃==
*因應基隆都市更新，將基隆車站站體南移、月台向北延伸，以騰空原車站區域內之土地，提供都更開發使用，新的[[基隆車站|基隆車站]]站體為半地下化。
*原先計畫的臺鐵都會區捷運化桃園段高架化計畫出現變化，[[桃園市長|桃園市長]][[鄭文燦|鄭文燦]]以民眾意向及效益可行性為基礎提出應將原高架化計畫改為地下化設計，並於2015年2月開始進行可行性研究的招標工作，台鐵都會區捷運化桃園段高架化計畫改為地下化設計。
