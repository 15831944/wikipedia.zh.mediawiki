'''文賢國中'''為[[台灣|台灣]][[台南市|台南市]]一所[[國民中學|國民中學]]，创始自1984年初，台南市政府指示民德國中負責編定此校1985年會計年度單位預算起；又於1985年5月20日，函委當時金城國中校長張齡材先生擔任此校文賢國中籌備處主任，進行設校，
校址經選定台南市北區文元里文賢一路二號，舊時明末[[鄭成功|鄭成功]]複台的台江地區中心任置---文元寮，

== 校史 ==

1987年8月5日[[台灣省政府|省府]]七六府教四字第一五三一六四號函；准此校自1987年8月1日正式成立。首任校長王茂德於同月廿五日奉台南市市長林文雄先生親頒印信，接掌校務，進行第一年新生入學上課籌備工作。

* 成立第一年此校招收十二班(後增一班)一年級新生
* 第二年又收二十班新生(後又增一班)
* 第三年再招收二十四班一年級新生入學，是時全校已達五十八班。
* 第十三年(1999年九月)全校已達六十八班，二、三九六西學生之數。
* 第一屆畢業生已分別在1990年暑假勞燕分飛邁向萬里鵬程，1990年六月應屆(第十一屆)畢業生九○九名也將完成九年國教而畢業。
* 累計第一至整十一屆已有畢業生計一一、八八○名，更上層樓求發展。

民國七十六年，此校成立之初，全校以二十五名教師辦起文賢國中，至2007年有二○○名教職員為文賢學子服務，期作育國家社會之英才、棟樑。1999年王校長因屆齡退休，同年八月市政府遴聘陳校長炯明接掌校務至2007年七月。

民國九十六年八月一日第三任鄭校長英建接任，學校正面臨少子化的社會問題衝擊，已逐年減班，鄭校長以身作則，以教育為志業，深信教育愛是辦學的根源，堅持以學生為主體，營造優質之校園文化，接納家長為學校夥伴關係，藉以整合校外資源，發展學校特色及重質也重量的文賢國中得以永續經營。

== 校舍 ==

文賢國中首期校舍於1986年11月11日動土興建

* 第一期工程包括普通教室九間、辦公室十六間、自然科實驗室六間，合計三十一間。
* 第二期校舍工程又興建普通教室二十三間，地科教室三間、運動場鋪設優麗龍跑道。
* 更配合改進及發展國教首期六年計畫第六年增建家政、工藝、音樂、美術等九間專科教室。
* 1989年再蓋第三棟普通教室二十八間等校舍新建工程。
* 又為因應該校學區到處大興土木、新社區林立所需，1980年度單位預算，繼續興建普通教室九間及圖書館等工程。
* 1991年又再度興建教室、辦公室十六間等。
* 1994年發包興建總工程款逾億的學生活動中心亦已於1996年十一月落成，學校硬體建設即將全部完成。

== 學校教育目標 ==

1、 培育自尊尊人，勤勞負責的態度<br />
2、 培養愛家、愛鄉、愛國的情操<br />
3、 培養五育均衡發展樂觀進取心青少年<br />
4、 啟迪創造思考與價值判斷的能力<br />
5、 增進解決問題、適應社會變遷並養成終身學習的態度<br />
6、 鍛鍊強健體魄及堅忍毅力，培養從事正當休閒活動的知能<br />
7、 培養互助合作與民主法治的精神，增進群己和諧的關係<br />
8、 增進審美與創作能力<br />
9、 培養熱愛生命與維護自然環境的態度<br />
10、以生活教育、品德教育為中心，培養明禮尚義的美德
