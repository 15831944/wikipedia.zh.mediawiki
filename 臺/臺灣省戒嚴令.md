《'''臺灣省戒嚴令'''》，正式名稱為《'''臺灣省政府、臺灣省警備總司令部佈告戒字第壹號'''》，是[[中華民國|中華民國]][[臺灣省政府主席|臺灣省政府主席]]兼[[臺灣省警備總司令|臺灣省警備總司令]][[陳誠|陳誠]]於1949年5月19日頒佈的[[戒嚴|戒嚴令]]，宣告自同年5月20日零時（[[中原標準時間|中原標準時間]]）起在[[臺灣省|臺灣省]]全境實施戒嚴，至1987年由[[中華民國總統|中華民國總統]][[蔣經國|蔣經國]]宣佈同年7月15日解嚴為止，共持續38年又56天。此戒嚴令頒布時的臺灣省轄區包含[[臺灣|臺灣]]本島與周邊附屬島嶼、以及[[澎湖群島|澎湖群島]]。

而[[金門縣|金門]]、[[馬祖列島|馬祖]]，直到1992年11月7日，[[中華民國國防部|中華民國國防部]]才宣布金門縣解除戒嚴，正式結束長達四十三年之久的戒嚴與實施三十六年的戰地政務實驗。

這是臺灣第二次實施戒嚴，第一次戒嚴是[[二二八事件|二二八事件]]發生時，由[[臺灣省行政長官|臺灣省行政長官]]兼警備總司令[[陳儀|陳儀]]所發布。在[[臺灣歷史|臺灣歷史]]上，該戒嚴令實行的時期又被稱為「'''戒嚴時代'''」或「'''戒嚴時期'''」<ref>{{Cite episode |url=http://news.ftv.com.tw/newsprog/taiwan/promo.aspx?id=167|title=台灣演義 台灣戒嚴史|credits=民間全民電視公司|network=民視新聞台|accessdate=2010-10-17|language=zh-tw|airdate=2010-10-17}}</ref>。

雖然《'''臺灣省戒嚴令'''》於1987年7月15日「解除」。但[[中華民國|中華民國]]([[台灣|台灣]])直到2013年8月6日[[立法院|立法院]]三讀通過修正《[[軍事審判法|軍事審判法]]》第1條、第34條和第237條，都未實際脫離[[戒嚴|戒嚴狀態]]。

== 背景 ==
由於[[第二次國共內戰|第二次國共內戰]]情勢對[[中國國民黨|中國國民黨]]所執政的[[中華民國政府|中華民國政府]]趨於不利，1948年（民國37年）12月10日，[[中華民國總統|中華民國總統]][[蔣中正|蔣中正]]發布戒嚴令，史稱第一次全國戒嚴令，而與戰場較遠未受到影響的[[新疆省_(中華民國)|新疆省]]、[[西康省|西康省]]、[[青海省_(中華民國)|青海省]]、[[臺灣省|臺灣省]]、[[西藏地方|西藏地方]]則不在範圍之內。直到1949年（民國38年）5月19日，臺灣省全境宣佈自20日起戒嚴。

《臺灣省戒嚴令》頒佈後，中華民國政府在[[中國大陸|中國大陸]]的统治情勢持續惡化，1949年（民國38年）7月7日，代理總統[[李宗仁|李宗仁]]發佈第二次全國戒嚴令。[[中華民國政府|中華民國政府]]於1949年，（民國38年）12月遷抵[[臺北市|臺北市]]，而中國大陸大部分領土則被[[中國共產黨|中國共產黨]]控制，[[海峽兩岸|海峽兩岸]]開始進入長期對峙狀態，此戒嚴令開始成為中華民國政府在臺灣穩固統治的重要[[法律|法律]]，並等同宣布臺灣處於如[[戰爭|戰爭]]般的[[緊急狀態|緊急狀態]]。

===戒嚴區的劃分===
[[臺灣警備總司令部|臺灣警備總司令部]]將[[臺灣|臺灣]]全境分為5個戒嚴區：
*臺北市戒嚴區：以警備旅旅長[[任世桂|任世桂]]兼任戒嚴司令。
*北部戒嚴區：包括[[新北市|臺北縣]]、[[新竹縣|新竹縣]]和[[基隆市|基隆市]]、[[新竹市|新竹市]]，以第六軍軍長[[戴樸|戴樸]]兼任戒嚴司令。
*中南部戒嚴區：包括[[臺中|臺中]]、[[臺南|臺南]]、[[高雄|高雄]]和[[彰化市|彰化市]]、[[嘉義市|嘉義市]]、[[屏東市|屏東市]]，以第八十軍軍長[[唐守治|唐守治]]兼任戒嚴司令。
*東部戒嚴區：包括[[花蓮縣|花蓮縣]]、[[臺東縣|臺東縣]]，以憲兵教導第一團團長[[歐廷昌|歐廷昌]]兼任戒嚴司令。
*澎湖戒嚴區：包括[[澎湖群島|澎湖群島]]全部，以馬公要塞司令[[李振清|李振清]]兼任戒嚴司令。

===其他相關法令===
從戒嚴令頒佈直到1949年（民國38年）底，中華民國政府陸續頒佈了一些相關管制法令。較爲重要的有：《[[戒嚴期間防止非法集會結社遊行請願罷課罷工罷市罷業等規定實施辦法|戒嚴期間防止非法集會結社遊行請願罷課罷工罷市罷業等規定實施辦法]]》、《[[臺灣省戒嚴期間新聞紙雜誌圖書管制辦法|臺灣省戒嚴期間新聞紙雜誌圖書管制辦法]]》、《[[懲治叛亂條例|懲治叛亂條例]]》等。

===連坐保證制度===
[[臺灣省政府|臺灣省政府]]在1949年（民國38年）7月9日開始全面實行省政府員工的[[連坐|連坐]]保證制度，未有保證人具保者不予雇用。在此之前，最早由[[臺中市|臺中市]]於5月16日對轄內公務員與教師實施的連坐保證制度。從公務人員開始，該制度逐步擴及到臺灣社會幾乎所有公私機構單位，成爲戒嚴時期遍及臺灣絕大多數人口的基本政治審查制度之一。部分內容至今仍存在於臺灣一些公司行號的人事作業流程中。

1950年（民國39年）4月3日，臺灣省政府頒布「[[臺灣省反共保民委員會組織辦法|臺灣省反共保民委員會組織辦法]]」<ref name="紀錄">{{cite book|editor=[[張之傑|張之傑]]等|title=《20世紀臺灣全紀錄》|location=台北|publisher=錦繡出版社|year=1991}}</ref>{{rp|310}}，並令各縣市剋日成立<ref name="紀錄"/>{{rp|310}}。

== 戒嚴令的解除 ==
[[File:The-lifting-of-martial-law_news_express_text_card_of_CTV_News_19870714.jpg|缩略图]]的[[政令宣導字卡|政令宣導字卡]]]]
[[File:台湾解严令.jpg|thumb]]

1980年代中期開始，臺灣開始出現要求徹底解嚴的運動，尤其以1986年5月19日[[黨外運動|黨外運動]]人士於[[中山堂_(臺北市)|臺北市中山堂]]舉行的[[519綠色行動|519綠色行動]]最為重要，示威抗議的民眾高舉「只要解嚴、不要國安法」、「百分之百解嚴」等標語。

1987年7月14日，總統[[蔣經國|蔣經國]]頒佈總統令，宣告[[臺灣地區|臺灣地區]]自同年7月15日凌晨零時起解嚴，解除在臺灣本島、澎湖與其它附屬島嶼實施的戒嚴令<ref>[http://law.moj.gov.tw/LawClass/LawAll.aspx?PCode=A0000016 （中華民國）全國法規資料庫入口網站 - 臺灣地區解嚴令]</ref>（簡稱「解嚴」），在臺灣實施達38年又2個月的戒嚴令自此走入歷史。總統令同時還宣佈廢止戒嚴期間依據《[[s:戒嚴法_(中華民國)|戒嚴法]]》制定的30項相關法令，而[[中華民國國防部|中華民國國防部]]也對237位於戒嚴時期遭軍法審判的民眾予以減刑或釋放。

而接近[[中國大陸|中國大陸]]的[[福建省_(中華民國)|福建省]][[金門縣|金門縣]]、[[連江縣_(中華民國)|連江縣]]依照1956年（民國45年）6月23日行政院令頒布之《[[金門、馬祖地區戰地政務實驗辦法|金門、馬祖地區戰地政務實驗辦法]]》實施戰地政務<ref>[http://www.rootlaw.com.tw/LawContent.aspx?LawID=A040060030000900-0811105 金門、馬祖地區戰地政務實驗辦法]</ref><ref>[https://sites.google.com/a/kmip.org/kinmen/Wiki/History/Military-Government 金門百科beta 戰地政務時期]</ref>。而國防部於1991年《[[動員戡亂時期臨時條款|動員戡亂時期臨時條款]]》廢止同年5月1日以金馬位處前線，與敵相鄰，且在共軍火砲射程之內，隨時有遭受攻擊之虞，在中共尚未放棄武力犯臺之前仍屬接戰地域，外島指揮官在戒嚴情勢仍存在的情況下實施臨時戒嚴（二度戒嚴）；直到1992年11月7日同時解除臨時戒嚴與戰地政務，歷時近43年。<ref>[http://www.matsu.idv.tw/gov/council/historica/data05/03.html 連江縣議會 - 解除二度戒嚴]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>立法院於2017年12月5日通過制定的《促進轉型正義條例》即以1992年11月7日界定「[[威權統治|威權統治]]時期」之迄日。<ref>[http://lis.ly.gov.tw/lydata/upload/lynewbill/000007463/250000a/1061205%E7%AC%AC9%E5%B1%864%E6%9C%83%E6%9C%9F%E7%AC%AC11%E6%AC%A1%E6%9C%83%E8%AD%B0.docx 〈制定促進轉型正義條例〉，第9屆第4會期第11次會議，立法院國會圖書館]</ref>

== 戒嚴令的影響 ==
[[File:臺灣綠島人權紀念碑Monument_for_Human_Rights_on_Lyudao_(GreenIsland)_of_Taiwan.jpg|right]][[綠島|綠島]]人權紀念碑：一道長約十來公尺的石牆上，刻滿當年在[[臺灣白色恐怖時期|白色恐怖]]下的犧牲者名單]]
頒佈戒嚴令是影響臺灣社會發展的重要歷史事件。依據《戒嚴法》規定：「在宣布戒嚴期間，由戒嚴地域的最高司令官掌管行政事務及司法事務」，政府為便利戰時管理而在[[國共內戰|國共內戰]]期間執行，人民自由與基本人權，包括集會、結社、言論、出版、旅遊等權利被限縮，即所謂[[黨禁|黨禁]]、[[報禁|報禁]]、海禁、出口旅遊禁等，在此段時期言論自由受到普遍限制。政府用相關法令條文對[[中國共產黨|共產黨]]人、政治上持異議人士（多為黨外人士）進行逮捕、軍法審判、關押或處決．負責執行的[[台灣警備總司令部|臺灣警備總司令部]]在時任的中國國民黨[[蔣中正|委員長]]指示下徹底執行，此間臺灣常有人突然失蹤，不時傳出冤獄，俗稱「[[臺灣白色恐怖時期|白色恐怖]]」。

戒嚴的實施為臺灣的政治、[[經濟|經濟]]發展提供了相對穩定的中央集權與現今社會基礎。戰爭驅使國民政府在戒嚴時期推行[[耕者有其田|耕者有其田]]與[[三七五減租|三七五減租]]、幣制（戰爭導致奔馳型物價膨脹，原先的「[[舊臺幣|臺幣]]」改制為「[[新台幣|新臺幣]]」沿用至今）等戰時政策、及[[十大建設|十大建設]]等一系列重大經濟政策與基礎設施建設，臺灣產業形態由農漁業爲主轉為以[[輕工業|輕工業]]製造與商業服務業為主，人民所得與生活水準均顯著提升。

== 程序不合法之爭議 ==
2009年，前[[總統府國策顧問|總統府國策顧問]][[謝聰敏|謝聰敏]]（1964年與[[彭明敏|彭明敏]]、[[魏廷朝|魏廷朝]]起草《[[台灣自救運動宣言|臺灣人民自救宣言]]》而被判10年、又在1971年[[花旗銀行|花旗銀行]]爆炸案遭誣陷被捕入獄15年）與白色恐怖政治受難者團體指臺灣省警備總司令部當年發布的戒嚴令未依法按級呈報代總統[[李宗仁|李宗仁]]，再由總統提交[[立法院|立法院]]追認，因此，依法無效。中國國民黨政府依無效的戒嚴令侵害部分人士的人身自由及財產權，但司法院大法官卻不受理釋憲，他們要求[[監察院|監察院]]彈劾大法官、糾舉大法官失職。<ref>[http://www.twtimes.com.tw/index.php?page=news&nid=36881 白色恐怖受難者　控訴大法官失職], [[臺灣時報|臺灣時報]], 2009-4-6</ref><ref name="公視">[http://web.pts.org.tw/php/news/pts_news/detail.php?NEENO=156283 監院報告:38年戒嚴令 發布有瑕疵], [[公視新聞|公視新聞]], 2010-8-11</ref>

2010年，監察委員[[黃煌雄|黃煌雄]]等提出調查報告指出1948至1949年有三次戒嚴令，1949年5月20號發佈的第二次戒嚴令是否有依戒嚴法第3條送立法院追認無從考究，1949年11月22日第三次的戒嚴令如未經總統宣告發布（李宗仁代總統當時並不在中華民國境內，不可能簽字公告，不符憲法第39條規定），形式要件不完整，法定程序有瑕疵，則戒嚴令因欠缺形式法效而失效，軍事審判機關的審判權就有瑕疵，戒嚴時期因案被宣告沒收之財產「恐須重新審酌」，當然也會發生其後的救濟問題。但最終相關法律的認定是[[司法院大法官會議|司法院大法官會議]]。<ref>[http://www.cna.com.tw/news/FirstNews/201008110019-1.aspx 監院調查38年戒嚴令有爭議], [[中央通訊社|中央通訊社]], 2010-8-11</ref><ref name="公視"/><ref>[http://www.cy.gov.tw/sp.asp?xdURL=./di/Message/message_1.asp&ctNode=903&msg_id=3108 臺灣發布戒嚴是否符合法定程序 監察院提調查報告], [[監察院|監察院]], 2010-8-11</ref><ref>[http://www.cy.gov.tw/dl.asp?fileName=1101817141997.pdf 2010年監察院人權工作實錄：第1冊 公民與政治權利], [[監察院|監察院]], 2010</ref>

謝聰敏等人對監院的調查結果感到振奮，他們表示自己因為在戒嚴時期遭到軍法審判而坐牢，如果監委認定戒嚴令有瑕疵，那麼當時的審判就站不住腳，他們將根據這項調查報告向大法官聲請[[釋憲|釋憲]]、爭取[[平反|平反]]及[[冤獄|冤獄]]賠償。<ref name="公視"/>[[民主進步黨|民主進步黨]][[立法委員|立法委員]][[管碧玲|管碧玲]]表示沒有依照憲法程序實施戒嚴令是違法、不合法的，「不合法的戒嚴令實施後，使多少人民的生命、自由、財產因此受到剝奪、被軍事審判，這些怎麼辦？應該要慎重研議國家補救的體制！」被列入[[黑名單_(臺灣)|黑名單]]的立委[[蔡同榮|蔡同榮]]也說：臺灣人很無辜，實施戒嚴令讓海內外的鄉親受害很大，他因此在美國30年卻有家歸不得，他要求政府應該追查相關法律責任、補償。<ref>[http://www.nownews.com/n/2010/08/12/668124 監院稱38年戒嚴令實施有瑕疵　綠委：慎重研議補救體制], [[Nownews|Nownews]], 2010-8-12</ref>

== 解除戒嚴後 ==
解除戒嚴對臺灣社會帶來以下改變：
* 出入境及出版物的管理由[[中華民國國防部|國防部]]移交[[內政部警政署|內政部警政署]]及[[行政院新聞局|行政院新聞局]]負責。1987年12月1日，臺灣新聞局宣布，自1988年1月1日起，開放[[報禁|報禁]]<ref name="演變">{{cite book|author=茅家琦|title=《蔣經國的一生與他的思想演變》|location=台北|publisher=[[臺灣商務印書館|臺灣商務印書館]]|year=2003}}</ref>{{rp|429}}。解除報紙限證在29家、每份限三大張、限印於發行地點內之束縛。
* 1987年11月5日，[[行政院會議|行政院會議]]通過《[[人民團體組織法草案|人民團體組織法草案]]》，將政治團體列為人民團體之一，受該法約束。新籌組政黨有二十多個；直到1989年1月20日，[[立法院|立法院]]三讀通過《[[動員戡亂時期人民團體法|動員戡亂時期人民團體法]]》，各政治團體均得依法自由成立，「並從事選舉自由活動」<ref name="演變"/>{{rp|429}}。開放民眾登記政治團體，人民可依法組黨結社、組織參加[[遊行示威|集會遊行]]及從事政治活動。
