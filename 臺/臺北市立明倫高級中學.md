'''臺北市立明倫高級中學，英文縮寫為MLSH'''，位於[[台北市|台北市]][[大同區_(臺灣)|大同區]][[承德路|承德路]]三段 336 號。前瞻孔子廟，後比基隆河，左銜承德橋回收場，右鄰[[臺北市立重慶國民中學|重慶國中]]，籃、排球場上有中山高速公路、五股汐止高架覆蓋。

== 校史 ==
*1968年(民國57年)，「臺北市立明倫國民中學」創立，6月設校，7月先假[[臺北市大同區大龍國民小學|大龍國小]]從事籌備工作，8月1日學校成立，9月借[[臺北市立陽明高級中學|陽明國中]]校舍上課。
*1970年(民國59年)，校舍竣工完成，遷入現址。
*1971年(民國60年)，增設益智教育特殊班，辦理啟智教育。
*1981年(民國70年)，增設美術資優實驗班。
*1993年(民國82年)，改制為「臺北市立明倫高級中學」，並停招國中生。
*1998年(民國87年)，成立資源班。
*2004年(民國93年)，成立語文資優班。
*2009年(民國98年)，最後一屆語文資優班學生畢業。
*2010年(民國99年)，第一屆自然實驗班及語文實驗班成立。

== 校名由來 ==
因鄰近孔子廟，依孔子廟「明倫堂」而得名。<br />
校舍大樓也依孔子所言：「志於道，據於德，依於仁，游於藝」而命名，希望老師能以此做為立身處世的座右銘，為教育與文化傳承而奉獻心力。

== 校地面積 ==
明倫高中是台北市公立高中裡數一數二小的學校之一，它的校地面積僅比[[西松高中|西松高中]]及[[成功高中|成功高中]]的校地面積大一點而已(但是在[[中山高|中山高]][[汐五高架|汐五高架]]正下方有一小塊地可供學校使用)，但是由於現階段校地施工中，所以目前可用面積是全台北市最小的公立高中，僅有一棟8字形的教學大樓以及後操場(位於[[中山高|中山高]][[汐五高架|汐五高架]]正下方)；正因如此，該校有一句勉勵學生的順口溜："小小的明倫，可以有大大的天空"。
