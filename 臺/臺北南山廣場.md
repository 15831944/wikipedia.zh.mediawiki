'''臺北南山廣場'''是位於[[臺灣|臺灣]][[臺北市|臺北市]][[信義計畫區|信義計畫區]]的[[商辦|商辦]][[摩天大樓|摩天大樓]]，坐落於松智路與松仁路之間，土地約面積5300坪左右，興建基地原為[[台北世界貿易中心|台北世貿]]二館。樓高272公尺，地上樓層48樓，地下樓層5樓，總樓地板面積為60,340坪，於2017年10月26日竣工，並於2018年6月11日落成開幕。目前為全球第219名的高樓，
臺北市第二高樓（僅次於[[臺北101|臺北101]]）以及臺灣第三高樓（僅次於[[台北101|臺北101]]及[[高雄85大樓|高雄85大樓]]），商場部分由[[微風廣場|微風廣場]]經營，於2019年1月10日正式開幕。

由於會影響到臺北101舉辦的[[跨年|跨年]][[煙火|煙火]]施放，未來可能合作舉辦跨年煙火活動。<ref>[http://www.ettoday.net/news/20131127/301196.htm: 南山廣場估2017年完工　尹衍樑：將與101辦跨年煙火秀]</ref>

== 發展歷史 ==

[[File:Taipei_Nan_Shan_Plaza_20160723.jpg|thumb]]
[[File:Taipei_Nan_Shan_Plaza_construction_site_20170715b.jpg|thumb]]
在[[台北世界貿易中心南港展覽館|世貿南港展覽館]]開始營運後，台北世貿二館已功成身退；[[台北市政府|台北市政府]]為活化資產提出以設定[[地上權|地上權]]的方式招商，於2012年由[[南山人壽|南山人壽]]取得50年地上權<ref>{{zh}}{{cite news|title=挑戰101租金 世貿二館改建南山廣場|author=林巧雁|publisher=[[台灣蘋果日報|台灣蘋果日報]]|url=http://www.appledaily.com.tw/appledaily/article/finance/20121013/34570873|date=2012-10-13|accessdate=2013-11-15}}</ref>，預計總投資金額超過[[新臺幣|新臺幣]]450億元，興建購物廣場、辦公空間及博物館等公共展覽空間。臺北市政府獲得標金收入近新臺幣270億元，為目前臺北市土地開發金額最高的開發案，超越台北101決標金額新臺幣206億8889萬元。<ref>[http://www.appledaily.com.tw/appledaily/article/finance/20121013/34570873 挑戰101租金 世貿二館改建南山廣場]</ref>此外，臺北市政府未來每年至少有新臺幣兩億元左右的地租收入。

2013年世貿二館舉辦之臺北國際工具機展結束後，開始進行世貿二館拆除作業；同年11月27日，南山人壽舉行南山廣場動土典禮<ref>{{zh}}{{cite news|title=台北南山廣場 打造親善商圈|author=王玉樹|publisher=[[中國時報|中國時報]]|url=http://www.chinatimes.com/newspapers/20131128000436-260114|date=2013-11-28|accessdate=2016-06-17}}</ref><ref>[http://www.mygonews.com/news/detail/news_id/107862 重大建設！台北南山廣場開工動土，將成台北新地標]</ref>。

== 辦公大樓 ==

== 微風南山 ==
裙樓商場樓高7層，共1.6萬坪樓面，由[[微風廣場|微風置地]]承租20年，商場名稱定為[[微風南山|微風南山]]（Breeze NAN SHAN），並且在2樓至4樓引入[[JR東日本|JR東日本]]旗下的百貨[[atré|atré]]。微風南山內設51間店舖，其中16間為首次進軍台灣。微風南山於2019年1月10日正式開幕。<ref>{{cite news |author1=凃盈如 |title=1月10日終於要開了 微風南山公布獨家品牌 |url=http://ent.ltn.com.tw/news/breakingnews/2642803 |accessdate=2018-12-18 |work=自由時報 |agency=自由時報}}</ref>

== 餐廳 ==
46至48樓為餐廳，已承租給微風南山。
