== 電視台及類型 ==
{{主條目|臺灣電視台列表}}
[[臺灣|臺灣]]的[[電視台|電視台]]，可分為[[地上數位電視|無線數位電視台]]（無線數位電視事業）、[[衛星廣播電視節目供應者|衛星電視台]]（衛星廣播電視節目供應者）、有線電視業者、[[中華電信MOD|MOD]]、有線電視數位頻道、網路電視、等六大類型。

# 無線電視台共有6家事業機構、22個頻道，以[[大氣電波|大氣電波]][[UHF|UHF]]頻段傳送到地上衛星接收站以（[[DVB-T|DVB-T]]）發送[[數位訊號|數位訊號]]提供到裝有數位訊號接收盒的用戶觀看。
# 衛星電視台則是將節目訊號發射至軌道衛星，裝有為衛星訊號接收器的用戶則透過直接接收衛星訊號，來收看電視。
# 有線電視業者：[[台灣有線電視業者列表|有線電視系統業者]]接受衛星訊號後，傳送至用戶端收看。依據《有線廣播電視法》之規定，無線電視台必須載於[[有線電視|有線電視]]上作為基本頻道提供用戶收看。
# [[中華電信MOD|MOD]]平台：自2005年[[中華電信|中華電信]]的[[IPTV|IPTV]]服務─[[中華電信MOD|MOD]]平台開播後，更提供了閱聽人收看電視節目的新途徑─[[寬頻電視|寬頻電視]]，許多舊與新的衛星電視台陸續在該平台播出。
# 有線電視數位頻道：與中華電信競爭，大同小異。如：2012年9月5日凱擘大寬頻正式推出SUPER MOD的平台服務。
# 網路電視：通常指觀眾直接透過境外視頻網站收看或VPN渠道觀看，如：[[YouTube|YouTube]]、[[FC2|FC2]]、[[伊莉討論區|伊莉]]等等。

==== 廣播制式和相關技術 ====
{{Expand|time=2018-03-03T15:53:06+00:00}}
2001年，台灣無線電視業界選定以[[DVB-T|DVB-T]]制式為數位電視的製播規格。2004年7月開播時，五家無線電視台所屬的14個電視頻道均採[[MPEG|MPEG]]串流編碼。

2008年，公視在測試高畫質電視時，為了有效使用頻寬，採用[[H.264|H.264/MPEG-4 AVC High Profile @ Level 4]]壓縮格式進行影像壓縮，採用[[AAC|AAC]]壓縮技術進行音訊壓縮，透過[[UHF|UHF]]頻段地面波播送1920×1080i 59.94Hz格式的[[DVB-T|DVB-T]]節目。使日後其他四家無線電視台於2012年以高畫質電視轉播奧運時，也比照相同規格播出。但由於第一代數位電視機上盒或電視內建調諧器只支援MPEG串流編碼，收看高畫質電視必須改用支援H.264編碼格式的機上盒或電視機方能收看。自2017年2月3日，華視有效運用頻寬以播出5個頻道，也將標準畫質電視頻道改採H.264編碼。到2018年3月底為止，內建第一代規格數位電視調諧器的電視或機上盒，只能收看4個以MPEG-2編碼電視頻道。
