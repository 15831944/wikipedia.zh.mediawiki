'''臺北府城西門'''，正式名稱為'''寶成門'''，是[[臺北府城|臺北府城]]的五座[[城門|城門]]之一。西門由於連結[[艋舺|艋舺]]商圈，因此取「'''寶物成就'''」之意，盼維持艋舺的流通興旺而得名。

西門擁有臺北府城城門中最華麗的雙層樓閣，[[泉州三邑|泉惠南]]商人捐建，也是台北城五個城門裡面最華麗的城門。1905年[[台灣日治時代|台灣日治時代]]市街改正的拆牆過程中拆除此門造成民意反彈，致使[[台灣總督府|台灣總督府]]拆除其他四座城門的計劃叫停，西門成為台北城五個城門中唯一被拆除殆盡的城門。舊址旁設有「西門印象」裝置藝術，供民眾認識西門的歷史點滴。

== 歷史 ==
*1879年正月開始建造。
*1882年三月竣工。
*1905年因實施[[市區改正|市區改正]]遭到拆除。
*1906年原址改建為橢圓公園。
*1983年寶成門舊址石碑設立。
*1999年配合[[台北捷運|台北捷運]]啟用，原址改為十字路口，圓環空間移到路口四角作為捷運西門站出口用地。
*2014年配合台北建城130年，建造「西門印象」裝置藝術，坐落在西門舊址北邊，捷運西門站4號出口旁。
*2017年寶成門舊址石碑遭到損毀<ref>{{cite web|title=【破壞片】35年歷史記事碑遭毀損　嫌犯落網道歉|url=https://tw.appledaily.com/new/realtime/20170828/1191757/}}</ref>。
*2018年底寶成門舊址石碑重立。

== 地理位置 ==
寶成門舊址位於今[[台灣|台灣]][[台北市|台北市]][[中正區_(台北市)|中正區]]、[[萬華區|萬華區]]區界處，大約在衡陽路、寶慶路、中華路及成都路等數條道路交會路口東北角。

2014年（台北建城130周年），台北市政府在捷運西門站4號出口設置鋼構裝置藝術品「西門印象」，高2層樓，長7公尺，寬4.9公尺，保有原西門的輪廓，上方由右到左標示「寶成門」三字，是配合著過去中文由右到左的書寫習慣，其中一面鋼柱上面有該藝術品的說明與西門舊址的相對位置圖。周邊利用地面設置了台北府城的示意圖，標有五座城門與城牆的相對位置與整座城的軸線方向以及城門的原貌銘版。配合新建城門藝術品，西門站出口四改裝成城牆的風格，從米色改為城牆狀的外皮。

2018年[[台北燈節|台北燈節]]中，已被拆除的西門及城牆的舊址附近（今[[中山堂_(臺北市)|中山堂]]後方鄰中華路一段的區域，[[臺北市立國樂團|臺北市立國樂團]]前）設置由華山站貨場與調皮工作室設計的祈福燈區作品「西門城垛」。

1983年4月，舊址東方設有一座寶成門舊址紀念石碑，位於衡陽路、寶慶路交口處，即捷運西門站出口三與出口四中間分隔島上。

2017年8月27日早上7時許，一名疑為精神病患之蕭姓男子，散步時突然使盡力氣向石碑推去，導致石碑斷成三截損毀。

2018年11月寶成門舊址石碑重立。
距中華路中心約五十米，又中華路為八十米寬道路，碑上記載：“...門之舊址在碑正前方二十公尺處...”。

== 周圍現況 ==
此處除臨近[[博愛特區|博愛特區]]外，周圍大部分屬於西區商圈的範圍：包括東邊的衡陽路、[[博愛路|博愛路]]、[[城中市場|城中市場]]等傳統商圈與西邊的青少年流行商圈—[[西門町徒步區|西門町徒步區]]，因此經常有許多休閒或購物人群經過此一帶。

周邊中華路有[[公車專用道|公車專用道]]，與附近道路的站牌皆有許多[[公車|公車]]路線行經，為台北重要公車轉乘站。便於轉乘前往[[板橋區_(新北市)|板橋]]、[[土城區|土城]]、[[三峽區|三峽]]、[[新莊區|新莊]]、[[三重區|三重]]、[[五股區|五股]]、[[萬華區|萬華]]、[[公館_(臺北市)|公館]]、[[景美|景美]]、[[松山區_(台北市)|松山]]、[[南港區|南港]]、[[建成圓環|圓環]]、[[中山北路_(台北市)|中山北路]]、[[天母|天母]]、[[國立故宮博物院|故宮]]等地。

此外，地下有[[台灣高速鐵路|高鐵]]、[[臺灣鐵路管理局|台鐵]]、[[台北捷運|捷運]]通過，其中捷運[[板南線|板南線]]、[[松山新店線|松山新店線]]設有[[西門站|西門站]]，十分便利。
