'''臺灣'''是位於[[亞洲大陸|亞洲大陸]]東南海岸的一個[[島嶼|島嶼]]，目前被[[中華民國政府|中華民國政府]]管轄。「臺灣」也可以指[[地理|地理]]範圍、[[文化|文化]]領域或[[行政區劃|行政區劃]]。

== 地理範圍 ==
* [[臺灣|臺灣島]]（舊稱[[福爾摩沙|福爾摩沙]]），其附屬島嶼有[[蘭嶼|蘭嶼]]、[[綠島|綠島]]、[[小琉球|小琉球]]、[[龜山島|龜山島]]等。
* [[台湾地理|臺灣島及附屬島嶼、澎湖群島，屬於广义上的“台湾”]]。
*[[臺灣_(新西伯利亞州)|臺灣 (新西伯利亞州)]]（{{lang|ru|Тайвань}}）：[[俄羅斯|俄羅斯]][[新西伯利亚州|新西伯利亞州]][[新西伯利亞水庫|新西伯利亞水庫]]上的一座[[無人島|無人島]]。

== 政治實體 ==
* {{ROC}}：一個[[未被國際普遍承認的國家|未被國際普遍承認的國家]]，曾为“中国”的唯一合法政府，但在1971年後，其在聯合國的“中国”代表權被[[中华人民共和国|中华人民共和国]][[联合国大会2758号决议|取代]]，有效管轄[[領土|領土]]為臺灣、[[澎湖|澎湖]]、[[金門|金門]]、[[馬祖|馬祖]]、[[東沙群島|東沙群島]]與部分[[南沙群島|南沙群島]]。因目前台灣本島佔中華民國現今實際領土的98%，故「臺灣」一詞被廣泛用作中華民國的通稱。

== 行政區劃 ==
*[[台灣省_(消歧義)|臺灣省]]：
**[[臺灣省_(中華民國)|臺灣省 (中華民國)]]：是中華民國的一個[[省_(中華民國)|省份]]，1999年後被[[中華民國政府|中華民國政府]]予以[[臺灣省政府功能業務與組織調整|虛級化]]的[[行政區劃|行政區劃]]。
**[[台湾省_(中华人民共和国)|台湾省 (中华人民共和国)]]：是[[中华人民共和国|中华人民共和国]]在其[[中华人民共和国行政区划|行政區划]]中所宣稱但未實際管轄的一个[[省_(行政区划)|省级行政区]]。

== 外交或法律称呼 ==
*[[中華民國（台灣）|中華民國（台灣）]]：[[中華民國（臺灣）|中華民國（臺灣）]]是[[陳水扁政府|陳水扁政府]]時期在政府文書中使用的名稱。目前則僅運用於[[英文|英文]]等[[外語|外語]]文書。
*[[臺灣地區|臺灣地區]]：指[[臺澎金馬|臺澎金馬]]。
**[[中華民國憲法|中華民國憲法]]增修條文正式名稱為「[[中華民國自由地區|中華民國自由地區]]」，與[[大陸地區|大陸地區]]相對，用以代稱[[國共內戰|國共內戰]]後國府遷台實際管治之領域。
**[[中华人民共和国|中華人民共和國]]以「臺灣地區」代稱[[中華民國|中華民國]]實際統治地區。
* {{TPE}}：中華民國參與某些[[國際組織|國際組織]]和國際運動賽事时使用的名稱，如[[奧林匹克運動會|奧林匹克運動會]]。
* [[中國臺灣|中國臺灣]]：与[[中国台灣省|中國台灣省]]并为[[联合国|联合国]]和[[国际标准化组织|国际标准化组织]]对台湾的稱呼。
* [[中国台北|中国台北]]：[[亚洲开发银行|亚洲开发银行]]和[[欧洲复兴开发银行|欧洲复兴开发银行]]对台湾的称呼。
* [[台澎金马个别关税领域|台澎金马个别关税领域]]：[[世界贸易组织|世界贸易组织]]对台湾的称呼。
*[[台灣國|台灣國]]（或稱[[台灣共和國|台灣共和國]]）：当前[[臺灣獨立運動|臺灣獨立運動]]支持者對於臺灣及其附屬島嶼建立的國家名稱之一。目前尚無國家承認或官方公開支持。

== 历史名称 ==
* [[臺灣府|臺灣府]]：[[臺灣清治時期|臺灣清治時期]]時設於臺灣的行政區名稱，1875年設[[臺北府|臺北府]]後，臺灣府縮小管轄範圍至臺灣中南部。1887年設[[臺南府|臺南府]]後，臺灣府縮小管轄範圍至臺灣中部。
* [[福建臺灣省|福建臺灣省]]：[[中國|中國]]在1885年至1895年間存在的一個[[行省|行省]]，由[[臺灣道|臺灣道]]升格而來，一般以「[[臺灣省|臺灣省]]」俗稱，史稱「[[臺灣建省|臺灣建省]]」。這是第一個專為[[臺灣|臺灣]]設置的[[省級行政區|省級行政區]]。
* [[臺灣民主國|臺灣民主國]]：1895年短暫出現於臺灣的抗日挺清政權。
* [[日治台灣|日治台灣]] (1895-1945)：[[日本帝國|日本帝國]]統治下的一個海外屬地。
* [[台灣共和國臨時政府|台灣共和國臨時政府]]（1956-1977）：學者[[廖文毅|廖文毅]]建立的政權，1956年於日本[[東京|東京]]成立。

==景點==
*[[小臺灣|小臺灣]]：位於[[澎湖縣|澎湖縣]][[七美鄉_(臺灣)|七美鄉]][[七美嶼|七美嶼]]的一個天然[[海蝕平台|海蝕平台]]。

== 人物 ==
* [[台灣·瓊斯|台灣·瓊斯]]：[[美國|美國]]的一名現役[[美式足球|美式足球]]員。
