'''臺灣總督府立醫院'''，是指[[臺灣總督府|台灣總督府]]直轄的醫院，由中央出資建設與管理。其包含一般醫院，以及針對特殊疾病，如[[结核病|結核病]]、[[麻风病|痲瘋病]]，和[[精神疾患|精神病]]所設的療養院所。

== 醫院 ==
最初於1895年6月在台北大稻埕千秋街設置台灣病院，隔年5月，台灣病院改稱台北病院，並在台中廳、台南廳設立病院。之後因地方首長陳情，於同年6月在淡水、基隆、新竹、宜蘭、鹿港、苗栗、雲林、埔里、嘉義、鳳山、澎湖設立病院，並在恆春、台東設立診療所。1898年將醫院由地方縣廳改隸於總督府之下，由於經費限制，將部分醫院裁撤。1910年設立花蓮港醫院。1911阿緱醫院設立後，裁撤鳳山醫院。1914年設立打狗醫院。台北醫院於1938年4月移交給台北帝國大學醫部附屬醫院。1940年成立花蓮港醫院玉里分院<ref>{{Cite web|url=http://stfb.ntl.edu.tw/cgi-bin/gs32/gsweb.cgi?o=dbook&s=id=%22jpli2007-bk-sxt_0767_60_1939%22.&searchmode=basic|title=臺灣の衛生|accessdate=|author=臺灣總督府警務局衛生課|date=1939-12-04|publisher=}}</ref><ref name=":0">{{Cite web|url=http://etds.lib.ncku.edu.tw/etdservice/view_metadata?etdun=U0026-0812200910243727&query_field1=&query_word1=%E9%86%AB%E9%99%A2%E5%BB%BA%E7%AF%89|title=日治時期台灣總督府醫院建築研究|accessdate=|author=王淳熙|date=2002|publisher=成功大學建築學系碩士論文}}</ref>。

府立醫院設立當時，皆借用既有的民宅、寺廟或官署使用。至1904年，醫院才全數新築及搬遷。<ref name=":0" />

== 結核病院 (松山療養所) ==
[[File:松山療養所宿舍1.jpg|缩略图]]
'''松山療養所'''設立於1914年，位於[[臺北市|台北市]][[南港區|南港區]]昆陽街，為台灣第一個結核病療養院。最初稱為'''錫口養生院'''，原由台北州管轄，於1924年3月由總督府直轄。戰後作為省立結核病防治院。

== 痲瘋病院 (樂生院) ==
[[File:The_former_Losheng_Sanatorium_Administration_Center_20070322.jpg|缩略图]]
{{See|衛福部樂生療養院}}
'''樂生院'''於1930年10月開院，位於[[新北市|新北市]][[新莊區|新莊區]]中正路794號。現在為衛生福利部樂生療養院舊院區。

== 精神病院 (養神院) ==
'''養神院'''於1934年10月完工，位於台北市虎林街，原址為住宅。收容人數為100人。戰後為省立[[五分埔|五分埔]]療養院，於1970年代遷至[[衛生福利部桃園療養院|衛福部桃園療養院]]現址`<ref>{{Cite web|url=http://linchunsheng.blogspot.com/2010/08/blog-post.html|title=養神院|accessdate=|author=林小昇之米克斯拼盤|date=2010-08-04|publisher=}}</ref>。
