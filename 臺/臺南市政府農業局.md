'''臺南市政府農業局'''（簡稱'''臺南市農業局'''），是[[臺南市政府|臺南市政府]]所屬的一級機關，本局除秘書室、人事室、會計室、政風室等幕僚單位外，分設農務科、森林及自然保育科、漁業科、畜產科、農產行銷科、農會輔導科、農地管理工程科等七業務科，另設有臺南市動物防疫保護處及臺南市漁港及近海管理所二附屬機關，專責推動本市各項農漁林牧業政策及業務<ref>{{cite web |author1=吳政修 |title=影╱台南推動食農教育亮麗 農業局：期提供更安全食材 |url=https://udn.com/news/story/7326/3390466 |publisher=聯合報 |accessdate=2018-09-27}}</ref><ref>{{cite web |author1=遠見雜誌整合傳播部企劃製作 |title=永續農業在臺南 新三農政策奏效 |url=https://www.gvm.com.tw/article.html?id=41366 |publisher=遠見雜誌 |accessdate=2017-12-08}}</ref>。
