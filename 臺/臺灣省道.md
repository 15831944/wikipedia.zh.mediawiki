'''臺灣省道'''（[[英語|英語]]譯名：{{lang|en|Provincial Highway}}）是[[臺灣|臺灣]]主要的普通[[公路|公路]]系統，目前共有98條路線（含支線），其中16條為[[快速公路|快速公路]]，其餘82條為一般公路。新建與養護機關為[[交通部公路總局|交通部公路總局]]。

== 簡介 ==
依據[[中華民國|中華民國]]《公路法》第二條規定，[[省道|省道]]係指聯絡二[[縣|縣]]（[[市|市]]）以上、省際交通及重要政治、經濟中心之主要道路。原本「臺灣省道」是指[[臺灣省|臺灣省]]境內的省道，隨著臺灣境內陸續成立數個[[直轄市_(中華民國)|直轄市]]，臺灣省道已不僅限於臺灣省境內，而成為臺灣除[[臺灣高速公路|高速公路]]之外主要的公路系統。各路線均由[[中華民國交通部|中華民國交通部]]擬訂，報請[[行政院|行政院]]核定公告。

省級道路的路線編號標誌包含'''一般省道'''及'''省道快速公路'''兩種，一般省道為盾形藍底雙白框中間標記白色阿拉伯數字及文字，而省道快速公路為紅底藍框中間標記白色阿拉伯數字及文字。主要是用於區別一般省道與省道快速公路的不同。

附帶一提的是在過去的省道第一代標誌以黑色圓形「台」字為底，上半圓放上「臺」字，下半圓標記編號之黑色阿拉伯數字及文字；第二代標誌則為圓形白底黑框，中間標記黑色阿拉伯數字及文字。

依據《道路交通標誌標線號誌設置規則》第89條規定，路線編號標誌需設於已編號之省道路線及其與各級道路之交岔路口上。並於省道每隔一公里設置一面且標示里程。

== 編號 ==
===主線編號===
[[File:TW_PHW1.svg|left]]
為便於公路主管機關管理及用路人辨識，中華民國各級道路依其聯絡方向不同分為奇數及偶數，通常北南向及東北、西南向道路使用奇數編號；東西向及西北、東南向使用偶數編號。而每條公路的里程數亦依此方式自起點至終點標示里程。{{clear}}
===支線編號===
[[File:TW_PHW1e.svg|left]]
依據該省道支線起點訂定支線編號，除以主線為數字編號外，另以十[[天干|天干]]依序予以文字編號。{{clear}}
== 省道公路系統 ==
===快速公路===
{{see also|台灣快速道路}}
由於臺灣為一狹長之海島，主要長程運輸多以南北向為主；為了有效分散[[高速公路|高速公路]]的車流，同時方便城鄉聯絡而建造的快速公路系統；用路人能夠利用這些快速公路銜接[[臺61線|西濱快速公路]]、[[中山高速公路|中山高速公路]]及[[福爾摩沙高速公路|福爾摩沙高速公路]]分散車流，彌補東西向高速公路的不足。

省道等級的快速公路包含[[台61線|台61線]]、[[台62線|台62線]]、[[台64線|台64線]]、[[台65線|台65線]]、[[台66線|台66線]]、[[台68線|台68線]]、[[台72線|台72線]]、[[台74線|台74線]]、[[台76線|台76線]]、[[台78線|台78線]]、[[台82線|台82線]]、[[台84線|台84線]]、[[台86線|台86線]]及[[台88線|台88線]]。

===環島公路===
{{see also|環島公路}}
包括省道[[台1線|台1線]]（繞經台灣西部）及[[台9線|台9線]]（繞經臺灣東部），共長約935.8公里，為環繞東西部地區主要幹道。

===橫貫公路===
{{see also|橫貫公路}}
包括[[台7線|台7線]][[北部橫貫公路|北部橫貫公路]]、[[台8線|台8線]][[中部橫貫公路|中部橫貫公路]]、[[台20線|台20線]][[南部橫貫公路|南部橫貫公路]]等省道，用於橫向連貫東西部地區的幹道。

===縱貫公路===
{{see also|縱貫公路}}
包括[[台3線|台3線]]、[[台13線|台13線]]、[[台19線|台19線]]、[[台21線|台21線]]、[[台29線|台29線]]等五條省道，規劃總長約962.1公里，為西部平原輔助幹道。

===濱海公路===
{{see also|濱海公路}}
為環繞台灣本島行經濱海地區的主要幹道，主要包含[[台2線|台2線]]、[[台11線|台11線]]、[[台15線|台15線]]、[[台17線|台17線]]及[[台26線|台26線]]。

===聯絡公路===
{{see also|聯絡公路}}
聯絡公路系統主要為聯絡環島、橫貫、縱貫及濱海公路系統，著名遊憩區與機場、高速鐵路車站、港埠等重要交通公共建設的聯外道路等。使省道路線形成綿密路網，其中包含[[台4線|台4線]]、[[台5線|台5線北基公路]]、[[台6線|台6線]]、[[台10線|台10線]]、[[台12線|台12線]]、[[台14線|台14線]]、[[台16線|台16線]]、[[台18線|台18線阿里山公路]]、[[台22線|台22線旗楠公路]]、[[台23線|台23線]]、[[台24線|台24線霧台公路]]、[[台25線|台25線鳳林公路]]、[[台27線|台27線]]、[[台28線|台28線]]、[[台30線|台30線玉長公路]]、[[台31線|台31線高鐵桃園站聯外道路]]、[[台37線|台37線高鐵嘉義站聯外道路]]、[[台39線|台39線高鐵台南站聯外道路]]及[[台63線|台63線中投公路]]。

== 省道共線 ==
為達到一條公路路線起讫點間的聯絡功能，而無法每條路線均各自形成獨立路線系統時，則會產生局部路線共同使用之部份。此一共同使用路線部份稱為'''[[共線_(道路)|共線路段]]'''。
* 依公路等級分類：有同等級之道路共線及不同等級之道路共線。
* 依起讫方向分類：有同一起讫方向之共線路段（同向共線、順椿共線）及相反起讫方向之共線路段（逆向共線、逆椿共線）。

== 省道長度 ==
*台灣地區省道總長度為5160公里，高雄市所占比例最高為513公里。{{RefTag|1=[http://stat.motc.gov.tw/mocdb/stmain.jsp?sys=210&funid=b340101&type=1 台灣道路長度與面積查詢]}}
*快速公路總長度最長的縣市為台南市約85.1公里。

==養護機關==
省道依法由公路總局養護，根據[http://law.moj.gov.tw/LawClass/LawContent.aspx?PCODE=K0040002 公路修建養護管理規則]第七條及[http://law.moj.gov.tw/LawClass/LawAll.aspx?PCode=K0040043 公路委託管理辦法]規定，省道經過直轄市政府、市政府行政區域部分，其修建、養護及管理除快速公路由交通部之專設機構辦理外，由交通部與當地直轄市政府、市政府協商定之。各直轄市、市省道路段養護機關大部分由公路總局養護<ref>{{cite web |url=https://www.thb.gov.tw/file.ashx?id=e8192aca-779e-46bf-b437-f40572b78c02|title=本局各養護工程處轄管路段里程階梯圖表|date=2018-07-24|work=交通部公路總局}}</ref>。
