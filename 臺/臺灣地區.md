'''臺灣地區'''，亦称'''臺澎金馬地區'''、'''臺澎金馬'''、'''臺閩地區'''，《[[中華民國憲法增修條文|中華民國憲法增修條文]]》歷次版本中稱為「'''自由地區'''」，是指自1955年[[大陳島撤退|大陳島撤退]]之後迄今，[[中華民國|中華民國]]有效管理的[[領土|領土]]<ref>{{cite web|url=https://law.moj.gov.tw/LawClass/LawAll_print.aspx?PCode=A0000002|title=中華民國憲法增修條文|date=2005-06-10|work=法務部}}</ref>，與1949年[[國共內戰|國共內戰]]後由[[中華人民共和國|中華人民共和國]]實際統治的「[[中国大陆|大陸地區]]」相對。涵蓋範圍包括[[臺灣|臺灣]]及附屬島嶼、[[澎湖群島|澎湖群島]]、[[金門縣|金門群島]]、[[馬祖列島|馬祖列島]]、[[烏坵鄉|烏坵列嶼]]、[[東沙群島|東沙群島]]、[[南沙群島|南沙群島]]之[[太平島|太平島]]與[[中洲礁|中洲礁]]<ref name="入出國及移民法">{{Cite web |url =http://www.immigration.gov.tw/immigr-law/cp.jsp?displayLaw=true&lawId=2c9e8a814c0a504d014c0bbf5b3d0011 |title = 入出國及移民法 |date = |publisher = 移民署法規查詢系統 |language = zh-tw |accessdate = 2015-06-19 |quote = 臺灣地區：指臺灣、澎湖、金門、馬祖及政府統治權所及之其他地區。|archiveurl = https://web.archive.org/web/20150619135622/http://www.immigration.gov.tw/immigr-law/cp.jsp?displayLaw=true&lawId=2c9e8a814c0a504d014c0bbf5b3d0011 |archivedate= 2015-06-19 }}</ref><ref name="臺灣地區與大陸地區人民關係條例">{{Cite web |url = http://www.mac.gov.tw/ct.asp?xItem=87948&CtNode=5659&mp=1 |title = 臺灣地區與大陸地區人民關係條例 |publisher = 行政院大陸委員會全球網 |language =zh-tw |accessdate = 2015-06-19 |format = |quote = 臺灣地區：指臺灣、澎湖、金門、馬祖及政府統治權所及之其他地區。|archiveurl =https://web.archive.org/web/20150619140513/http://www.mac.gov.tw/ct.asp?xItem=87948&CtNode=5659&mp=1 |archivedate= 2015-06-19 }}</ref>，全由[[中華民國島嶼列表|島嶼]]構成。由於臺灣為最主要的組成領土，因此得名。

== 使用 ==
目前[[中華民國政府|中華民國政府]]對於表示其有效統治領土的用詞不一，但以使用「臺灣地區」為主：

* '''臺灣地區''' - 主要使用於《[[臺灣地區與大陸地區人民關係條例|兩岸人民關係條例]]》等[[兩岸關係|兩岸關係]]相關法律。
* '''臺澎金馬（地區）''' - 部分政府文件使用。主要用於表示中華民國現有統治範圍的場合。<ref>[http://www.mofa.gov.tw/News_Content.aspx?n=8742DCE7A2A28761&s=ABCE429A177E6037 中華民國政府再次重申對臺澎金馬的主權地位] - [[中華民國外交部|中華民國外交部]]</ref>
* '''臺閩地區''' - 主要在政府的[[統計|統計]]刊物中使用，使用情形與「[[臺澎金馬|臺澎金馬]]」類似。在統計刊物中，臺灣地區僅指臺灣本島與澎湖群島。<ref>[http://www.fia.gov.tw/public/Attachment/47316113571.pdf 臺閩地區各縣市最近五年營利事業家數增長情形統計表] - [[財政部財政資訊中心|財政部財政資訊中心]]</ref>
* '''自由地區''' - 最早出現於《[[動員戡亂時期臨時條款|動員戡亂時期臨時條款]]》，相對於由中國共產黨所佔領的「淪陷地區」。1991年《[[動員戡亂時期臨時條款|動員戡亂時期臨時條款]]》廢止後所實施的《[[中華民國憲法增修條文|中華民國憲法增修條文]]》<ref>[http://www.president.gov.tw/Default.aspx?tabid=65 中華民國總統府 中華民國憲法增修條文]</ref>沿用之，指中華民國政府目前的治權範圍，相對於「大陸地區」以及疆域內中國共產黨統治區域以外非屬於中華民國政府的統治區域。

== 認知差異 ==
「臺灣地區」一詞的意義，依各方立場解釋不同而有差異。

|{{ROC}}||兩大政治陣營——[[泛藍|泛藍]]及[[泛綠|泛綠]]各有不同的解讀。惟目前共識是同意在中華民國一中基礎上以「臺灣地區」處理涉及「大陸地區」的事務。[[泛藍|泛藍]]陣營認為該用法避談國號，在《[[中華民國憲法|中華民國憲法]]》中，在中華民國固有領土之內不允許有其他國家的出現，而是以「大陸地區」和「自由地區」的地區之間的關係去面對此問題。「自由地區」指的是台澎金馬地區。基於此原則，大陸地區及臺灣地區同屬[[一個中國|一個中國]]，即中華民國[[一國兩區|一國兩區]]，依照《[[兩岸人民關係條例|兩岸人民關係條例]]》處理兩岸關係。||《[[中華民國憲法|中華民國憲法]]》<br>《[[憲法增修條文|憲法增修條文]]》
||[[泛綠|泛綠]]陣營傾向認為「中華民國（臺灣）」和「中華人民共和國（大陸）」間是[[一邊一國|一邊一國]]或[[特殊兩國論|兩國關係]]，認為中華民國的領土僅及於臺澎金馬及其附屬島嶼，主張[[中華民國是台灣|中華民國是台灣]]，且現在[[中華民國總統|總統]]、[[中華民國副總統|副總統]]與[[立法院|國會]]均於臺澎金馬選出，再加上「地區」字樣是對[[臺灣|臺灣]]的不尊重與自我矮化。然而[[民進黨政府|民進黨政府]]執政時期仍依照《[[兩岸人民關係條例|兩岸人民關係條例]]》處理兩岸關係。
