'''臺南市立東原國民中學'''（簡稱「東原國中」）是一所位於[[臺灣|臺灣]][[臺南市|臺南市]][[東山區_(臺南市)|東山區]]的[[國民中學|國民中學]]，建立於1970年。全校班級數為49班，學生人數共790人<ref>[http://163.26.2.28/sch_data/ 臺南市教育局資訊中心]</ref>。

== 行政組織 ==
*校長室 
*教務處
*總務處
*訓導處
*輔導室
*主計室

== 學校沿革 ==
一、自98年起年年增班,101學年度全校已由四班增為七班。

二、98,99學年度扶助弱勢-夜光天使計畫連兩年特優。

三、99學年度攜手計畫課後扶助全國績優學校。

四、101年6月1日教育部安排關懷之旅，教育部陳益興次長、賴清德市長與賴美惠議長蒞校指導。

五、101學年度，跨學區學生人數比率突破1：1。
        籃球隊教育部乙級聯賽殿軍；舉重隊參加全國中等教育聯合運動會囊括2金
        輔導室蔡惠婷主任及教務處陳一銘主任，結合在地物產開發精油及龍鳳酥

六、102學年度增2班，全校9班。

七、103學年度增2班，全校11班，學生人數逾240人。

== 學校特色 ==
以弱勢扶助與專長培訓發展為主要經營方針。

一.提供住宿,三餐,夜間自習看護予弱勢家庭學童(包括單親,新住民,隔代教養,中低收入戶,寄養家庭等),以弭補其家庭功能不足之處,讓學生可以致力於學業,不受家庭困境干擾，建構沒有孩子被遺棄的愛與關懷社區。

二.發展籃球,跆拳,舉重,高爾夫,大鼓等體育重點發展運動以及烹飪,園藝,資訊,直笛等第二專長技藝活動。提供學生除學業外的其他選擇,依據個人興趣,專長擴展自己更多的可能性，並讓每位孩子有高峰成長經驗，建構全是贏家的希望校園。
