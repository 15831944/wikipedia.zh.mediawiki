'''台灣與日本關係史'''是指[[台灣歷代|台灣歷代]]與[[日本歷史|日本歷代]]在歷史上不同階段的關係。雙方自十六世紀即有相互來往，并建構了各式各样的关系<ref>{{cite journal|author1=鄭瑞明|title=清領初期的台日貿易關係（1684-1722）|journal=臺灣師大歷史學報|issue=第32期|url=http://www.his.ntnu.edu.tw/files/publish/181_b3e6c86e.pdf|accessdate=2016-11-03}}</ref>。台日關係可分为几个阶段：1895年前[[明鄭時期|明鄭]]與[[台灣清治時期|清朝]]统治下台湾与日本的关系、1895至1945年臺灣做為日本殖民地与[[內地_(大日本帝國)|日本内地]]之间的[[台灣日治時期|关系]]、1945年後[[中華民國|中華民國]]统治下的[[中華民國與日本關係|關係]]。

==1683年以前==
{{see also|台湾历史|日本历史}}
日本从[[室町時代|室町時代]]（1336年至1573年）開始，以[[高砂|高砂]]、高砂國、[[高山國|高山國]]稱呼台灣。進入[[安土桃山時代|安土桃山時代]]（1568年至1603年）以後，开始寻求官方往来。日本[[後陽成天皇|後陽成天皇]][[文祿|文祿]]2年（1593年），[[豐臣秀吉|豐臣秀吉]]派遣使者{{link-ja|原田孫七郎|原田孫七郎}}[[豐臣秀吉高山國招諭文書|諭令高山國納貢]]，但是因使者找不到可以傳遞文書給高山國的人而無功而返<ref>{{Cite journal|author=岩生成一|script-title=ja:〈[[豊臣秀吉|豊臣秀吉]]の[[臺灣島|臺灣島]]招諭計畫書〉|journal=《[[臺北帝國大學|臺北帝國大學]]文政學部史學科研究[[年報|年報]]》|publisher=臺北帝國大學|location=[[臺北州|臺北州]]|language=ja|accessdate=2011-06-07}}</ref>。

日本進入[[江戶時代|江戶時代]]（1603年至1867年）以後，日台开始有官方往来。日本[[慶長|慶長]]14年（1609年），[[德川家康|德川家康]]任命[[有馬晴信|有馬晴信]]曉諭台湾原住民納貢，原住民不予理睬。[[元和|元和]]2年（1616年），德川家康任命長崎代官[[村山等安|村山等安]]征服台灣，等安以次子村山秋安率領2、3千人征台，但因遭遇風災無功而返。[[寬永|寬永]]5年（1628年），荷兰驻台灣的[[荷蘭東印度公司|荷蘭東印度公司]]與日本商人因貿易糾紛而發生「[[濱田彌兵衛事件|濱田彌兵衛事件]]」。寬永15年（1638年），日本鎖國，荷蘭商館遷移到出島。寬文二年(1662年)，[[鄭成功|鄭成功]]代替荷蘭東印度公司[[明鄭時期|統治台灣]]，重建台湾政權，結束[[臺灣荷蘭統治時期|荷蘭人在台湾的統治]]。鄭成功死後，由於[[清朝|清朝]]對鄭氏（東寧）勢力进行封鎖，[[鄭經|鄭經]]加强與日本的经贸往來。當時台湾對日本主要輸出品為稻米、鹿皮、蔗糖及來自中國國內的絲織品，从日本进口[[銀|銀]]、[[銅|銅]]、[[鉛|鉛]]、[[盔甲|盔甲]]、[[棉布|棉布]]、[[瓷器|瓷器]]等。為了加強雙邊的貿易關係，鄭經允許日本商人住在[[基隆|基隆]]<ref name=鄭日/>，台日雙方的貿易量在1665年到1672年達到高峰<ref>《[[臺灣歷史|台灣史]]101問》，頁109</ref><ref name="鄭日">《[[臺灣政治史|臺灣政治史]]》，頁62-63</ref>。郑氏三世之货币，以制钱[[永历通宝|永历通宝]]是委托日本铸造，也因為通商頻繁，東寧亦流通當時日本的貨幣——{{link-ja|寬永通寶|寛永通宝}}<!-- <ref>[http://nrch.culture.tw/twpedia.aspx?id=1225 鄭氏時期總論]</ref>{{dead link}} --><ref>{{Cite web |url = http://www.zhishi5.com/shrw/2011/1202/article_3603.html |title = 郑氏时期总论 |date = 2011-12-02 |publisher = 中国知识网}}</ref>。

==清朝统治时期==
{{see also|臺灣清治時期}}
1683年，清廷派遣[[施琅|施琅]]率軍攻台，結束明鄭時期統治，大清將台灣納入版圖。之後再度行[[渡台禁令|渡台禁令]]限制漢人抵台，渡台者須向官府申報等；而同時期日本幕府亦厲行鎖國政策，雙方的往來遂中斷長達近兩百年。1871年，[[第二尚氏王朝|琉球國]]船隊在[[臺灣|臺灣]]東南部遇害，是為「[[八瑤灣事件|八瑤灣事件]]」。1874年，[[大日本帝國|大日本帝國]]因八瑤灣事件出兵攻打[[臺灣原住民族|台灣南部原住民]]部落的軍事行動，是為「[[牡丹社事件|牡丹社事件]]」。日本方面則稱為“台湾出兵”或是“征台之役”。

== 日本統治時期 ==
{{see also|臺灣日治時期}}
1895年4月17日，清日[[甲午戰爭|甲午戰爭]]結束，[[李鴻章|李鴻章]]和[[伊藤博文|伊藤博文]]在日本馬關（今下關市）簽署《[[馬關條約|馬關條約]]》，[[台灣|台灣]]及[[澎湖|澎湖]]被割讓給[[大日本帝國|日本]]。1895年5月23日，[[唐景崧|唐景崧]]發表《[[台灣民主國|台灣民主國]]獨立宣言》以拒日本。1895年5月26日，台灣民主國國會議長[[林維源|林維源]]棄職逃亡福建廈門。1895年5月29日，日軍登陸[[澳底|澳底]]。1895年6月4日，台灣民主國總統唐景崧未戰陣前棄職逃亡中國廈門。1895年6月14日，日軍轉進台北，雖有[[辜顯榮|辜顯榮]]協助指引進城，但遭到人民頑抗，戰事之發展對於具有武力優勢及擁有正規軍之日軍而言則出乎意料之外。

== 中華民國統治時期 ==
{{see also|中華民國與日本關係|中日關係史|臺灣戰後時期|白團}}
===1940年代===
[[File:Ando_Rikichi_surrender.jpg|thumb]]，[[安藤利吉|安藤利吉]]將軍在[[陳儀|陳儀]]將軍發布署部第一號命令的受領證上簽名蓋章後，經由[[諫山春樹|諫山春樹]]將軍（左）轉交給[[陳儀|陳儀]]將軍（右）。]]
1945年10月，[[蔣中正|蔣中正]]接見[[美國|美國]]記者，告以[[日本天皇|日本天皇]]命運應由日本人民決定<ref name="年表" >{{cite book |authors=[[陳布雷|陳-{布}-雷]]等編著 |title=《蔣介石先生年表》 |location=[[台北市|台北]] |publisher=[[傳記文學|傳記文學]]出版社 |date=1978-06-01 }}</ref>{{rp|47}}。

[[第二次世界大戰|第二次世界大戰]]後，[[中華民國|中華民國]]代表[[同盟國軍事佔領日本|同盟國接管臺灣]]並派駐[[臺灣省行政長官|行政長官]]管轄。

1946年7月19日，在日本的[[台灣人|台灣人]]與日本警方和[[暴力團|暴力團]]在東京澀谷發生衝突，史稱「[[澀谷事件|澀谷事件]]」。

1947年2月27日至5月16日，在台灣爆發「[[二二八事件|二二八事件]]」。

1949年中華民國政府遷台後，[[冷戰|冷戰]]態勢的形成促成同為[[資本主義|資本主義]]陣營的中華民國與日本國開始謀求建立同盟關係。

===1950年代===
1950年3月31日，[[中華民國政府|中華民國政府]]通過「台日貿易計畫」<ref name="臺灣全紀錄" >{{cite book|editors=張之傑等|title=《20世紀臺灣全紀錄》|location=台北|publisher=錦繡出版社|year=1991}}</ref>{{rp|309}}。6月1日，臺灣、日本定期航線開航<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|531}}。6月11日，蔣中正在《[[革命實踐研究院|革命實踐研究院]]軍官訓練團成立之意義》講話中指出“中日兩國必須親睦合作，才能達到共存共榮的目的”<ref>{{cite book|title=《總統蔣公思想言論總集》卷二十三|location=台北|quote=
「經過我們[[第二次中日戰爭|這一次抗戰]]之後，大部份日本人就都感悟中國真是他們一個兄弟之邦，而一般有識之士，更深切瞭解，亞洲如果沒有一個獨立自由的中國，日本決不能單獨存在。我們只要日本軍民真正覺悟到這一點，中日之間就有了合作的基礎；同時我們自己也要認清，如果亞洲沒有一個獨立自由的日本，中國也是不能單獨存在的。現在事實擺在眼前：日本經過八年戰爭，已經徹底失敗了，我們雖然獲得一時的勝利，但是因為[[國共內戰|蘇俄指使共匪作亂]]，到如今也是徹底失敗了。這兩個國家現狀，實在都等於亡國，過去所謂「同歸於盡」的話，不幸而中了。現在中日兩國既已明白日本不能侵略中國，中國亦不可敵視日本，兩國必須親睦合作，才能達到共存共榮的目的，至此 [[孫文|總理]]的外交政策，亦才有實行的可能了。當前國際上沒有真正幫助我們的國家，我們重新研讀 總理遺教，越發覺得中日合作的重要。當然目前日本亦沒有力量來幫助我們，但就技術人員一項而論，日本的數量比我們多，程度比我們高。日本人作戰的經驗與能力的優良，是人人所共知的，尤其是他[[日本军|陸海空軍]]軍人視死如歸的精神，和他協同一致的行動，除了[[德國|德國]]以外，世界上沒有其他的國家可與相比，所以我這一次決定請日本教官來擔任我們軍事訓練。這些日本教官自到臺灣以來，對我們各方面的貢獻，十分誠摯，他們都知道中國今日的災禍，無異是日本的災禍，復興中國的工作，就是復興日本的工作，所以他們在這裏擔任教官，和在日本訓練他本國學生一樣的精誠，因此我們腦筋裏再不可留有過去的敵意，更不能存一種輕視的心理，以為他們是打敗仗的，不值得我們的尊重。我們要反省在抗戰中我們究竟憑什麼來打敗日本？老實說我們抗戰的勝利，一半是靠著總理的[[三民主義|主義]]和正確的國策，一半是靠著[[美援|友邦美國的援助]]，纔有此徼倖的勝利。難道日本真是被我們打敗的麼？現在我們國家在這樣存亡危急，到處受人欺凌，被人侮辱的時候，而日本教官反肯冒險來台，且能以其一片至誠，來幫助我們[[反共抗俄|反共抗俄]]，教授我們作戰的精神技術，以及其他各國所不能學得的學問，願與我們共患難、同甘苦、同仇敵愾、同舟共濟，那我們更應該特別優禮他們，尊敬他們。」}}</ref>{{rp|257-259}}。

1951年5月，蔣中正接見日本記者發表談話，聲明合作始能安定[[東亞|東亞]]，有助世界和平<ref name="年表" />{{rp|68}}。6月，蔣中正就參加對日和約問題發表鄭重聲明，指出其抗日最早、犧牲最重、貢獻最大，對日和約如無法參加，不獨對其不公平，抑且使對日和約失其真實，並將加深[[遠東|遠東]]局勢混亂，不接受任何含歧視條件<ref name="年表" />{{rp|68}}。6月30日，《中日貿易協定》期滿，雙方同意不定期延長<ref name="臺灣全紀錄" />{{rp|327}}。12月，[[日本首相|日本首相]][[吉田茂|吉田茂]]向美國保證不承認「[[中共|中共]]」，並願依《[[舊金山和約|舊金山和約]]》之原則與台簽訂一項恢復正常關係之條約<ref name="年表" />{{rp|69}}。

1952年4月28日，《[[中華民國與日本國間和平條約|中華民國與日本國間和平條約]]》在台北簽字<ref name="年表" />{{rp|70}}，條約共計十四條，中華民國不索賠償<ref name="現代史" >{{cite book |author=李守孔 |title=《中國現代史》 |location=台北 |publisher=[[三民書局|三民書局]] |year=1973 |isbn=9571406635 }}</ref>{{rp|194}}，條約第二條承認日本已在《[[舊金山和約|舊金山和約]]》中放棄對於台灣、澎湖以及[[南沙群島|南沙群島]]、[[西沙群島|西沙群島]]之一切權利與要求<ref name="現代史" />{{rp|194}}；蔣中正接見日本締結和平條約全權代表[[河田烈|河田烈]]，寄語日本朝野，共建東亞和平<ref name="年表" />{{rp|70}}。8月，蔣中正正式批准簽署该條約<ref name="年表" />{{rp|71}}。裁撤駐日代表團，台、日於[[台北市|台北]]、[[東京都|東京]]互設[[外交代表機構#大使馆|大使館]]<ref name="年表" />{{rp|71}}。10月，日本新任駐台大使[[芳澤謙吉|芳澤謙吉]]呈遞[[國書|國書]]<ref name="年表" />{{rp|71}}。

1955年3月5日，「中日空運臨時協定」在東京簽字<ref name="記" />{{rp|594}}。4月22日，中日貿易協定簽字換文<ref name="記" />{{rp|595}}。

1956年2月28日，在二二八事件九周年時，[[台灣共和國臨時政府|台灣共和國臨時政府]]在東京成立。5月，台日簽訂貿易協定<ref name="年表" />{{rp|85}}。8月，蔣中正接見日本各界親善訪問團，團長[[石井光次郎|石井光次郎]]親呈[[日本首相|首相]][[鳩山一郎|鳩山一郎]]手函，並謂日本人莫不感激蔣於日本戰敗後仍維護日本政體及對日[[以德報怨_(蔣中正)|以德報怨]]之寬大政策<ref name="年表" />{{rp|86}}。12月，日本政府正式否認鳩山一郎首相調解「[[二個中國|二個中國]]」<ref name="年表" />{{rp|87}}。[[聯合國安理會|聯合國安理會]]通過日本加入[[聯合國|聯合國]]<ref name="年表" />{{rp|87}}。

1957年3月，蔣中正書面答覆日本記者，期望台日對付[[蘇聯共產黨|蘇共]]侵略，永久精誠合作<ref name="年表" />{{rp|88}}。4月，「中日合作策進委員會」在日本東京成立<ref name="年表" />{{rp|88}}。6月，日本內閣總理大臣[[岸信介|岸信介]]訪問台灣，與蔣中正伉儷商談台日有關問題<ref name="年表" />{{rp|88}}。9月，蔣中正派特使[[張群|張群]]飛日本報聘<ref name="年表" />{{rp|89}}。

1958年5月，發生「[[長崎國旗事件|長崎國旗事件]]」。10月，蔣中正答覆日本記者詢問時，指斥中共每藉貿易以達其政治目的，盼望日本堅定立場免遭顛覆滲透<ref name="年表" />{{rp|92}}。

1959年7月，[[岸信介|岸信介]]稱，「台灣如在中國共產黨統治下，對日本的安全保障是重大威脅」<ref>{{cite web|url=http://news.ltn.com.tw/news/world/paper/847844|publisher=[[自由時報|自由時報]]|title=日外交文件解密／岸信介︰中若治台 對日威脅大}}</ref>。12月，日本前首相[[吉田茂|吉田茂]]訪台，謁見蔣中正致敬，並就當時世局交換意見<ref name="年表" />{{rp|95}}。

===1960年代===
進入1960年代後，日本執政[[自由民主黨_(日本)|自由民主黨]]對中共雖仍不正式承認，但雙方自1962年後分別設立「[[中國中日備忘錄貿易辦事處駐東京聯絡處|備忘錄貿易辦事處]]」，從事商品之交易<ref name="現代史" />{{rp|210}}。日本左翼份子屢次訪問中國大陸；中共亦經常派遣所謂「文化團體」赴日本從事滲透活動<ref name="現代史" />{{rp|210}}。

1963年5月，蔣中正命[[總統府秘書長|總統府秘書長]]張群訪問日本；8月，日本商人以維尼龍紗廠售出予中共，而其政府之進出口銀行居間擔保，台灣方面極力表示反對<ref name="年表" />{{rp|104}}。[[第2次池田內閣_(第3次改造)|池田勇人內閣]]批准倉敷[[嫘縈|螺縈]]公司對中國大陸輸出分期付款的維尼龍工廠成套設備，並以輸出入銀行給予200萬美元的融資週轉。輸出入銀行乃是日本國家金融機構，利用到了這個機構，被視為超越了民間貿易範圍的一種「經濟援助」。日本首相[[池田勇人|池田勇人]]則以「這是依循政經分離原則的貿易」為藉口，一意孤行。9月19日，池田勇人在招待美國[[赫斯特国际集团|赫斯特報系]][[總編輯|總編輯]]等人時表示：「中共在三、五年內不會有變化；臺灣的反攻大陸政策，沒有依據，近乎幻想。」9月21日，蔣中正接見由東京來到台北的同一批人士時，嚴斥池田首相媚共言論，指出《[[中蘇友好同盟互助條約|中蘇友好同盟互助條約]]》係防[[日本軍國主義|日本軍國主義]]，並警告日本人勿忘歷史教訓<ref name="年表" />{{rp|105}}。10月，中華人民共和國派出中國油壓機械訪日代表團前往日本參訪，團員周鴻慶前往中華民國大使館尋求[[政治庇護|政治庇護]]，誤入[[蘇聯|蘇聯]]大使館。池田勇人內閣將欲投奔台灣之周鴻慶遣返中國大陸<ref name="年表" />{{rp|106}}。日本種種不友好行動，使台日關係瀕於破裂<ref name="年表" />{{rp|106}}。中華民國政府不滿，一度召回駐日大使[[張厲生|張厲生]]表示抗議。史稱「[[周鴻慶事件|周鴻慶事件]]」<ref>{{cite web|url=http://www.epochtimes.com/b5/4/10/31/n704932.htm|author=林思雲|title=蔣介石的眼淚（六）|deadurl=yes|archiveurl=https://web.archive.org/web/20150701234217/http://www.epochtimes.com/b5/4/10/31/n704932.htm|archivedate=2015-07-01}}</ref>。

1964年2月，日本前首相吉田茂訪台，蔣中正與之多次會談，檢討亞洲反共局勢，認為欲謀求東亞和平與安定，必須增加瞭解真誠合作<ref name="年表" />{{rp|106}}。蔣中正以吉田茂為日本元老，乃促其負責共同挽救此一危機<ref name="年表" />{{rp|106}}。吉田茂與蔣中正共同發表《吉田書簡》支援國府，始緩解雙方關係。3月，蔣中正接見日本記者訪問團，期望日本國民勿受中國大陸迫誘，而危害台日傳統友誼<ref name="年表" />{{rp|106}}。3月5日，[[日本外務省|日本外務省]]次官{{Tsl|ja|毛利松平}}訪臺，闡明日本對中共政經分離政策<ref name="記" />{{rp|699}}。7月，[[日本外相|日本外相]][[大平正芳|大平正芳]]抵台，兩度晉見蔣中正，陳述台日關係，並表示日本「非常希望中華民國能反攻復國成功」<ref name="年表" />{{rp|107}}。10月，[[1964年夏季奧林匹克運動會|第18屆奧運會在東京舉行]]，其[[1964年夏季奧林匹克運動會火炬接力|聖火傳遞路線]]包含台北市，成為迄今唯一一次進入台灣的[[奧運聖火|奧運聖火]]。舉行期間發生中華民國代表團之運動員[[馬晴山|馬晴山]]投靠中共事件。

1965年1月13日，[[日本首相|日本首相]][[佐藤榮作|佐藤榮作]]與[[美國總統|美國總統]][[林登·约翰逊|詹森]]發表聯合聲明，強調支持臺灣，並與臺灣保持友好及正常外交關係<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|710}}。2月8日，佐藤榮作在日本國會表示，吉田茂致蔣中正書簡，對日本政府具有約束力<ref name="年表" />{{rp|109}}。3月17日，日本政府批准1億5,000萬美元貸款計畫，協助臺灣推行四期經建計畫；4月26日，貸款協定在臺北簽署協文；9月22日，1億600萬美元貸款發展臺灣經濟之技術協定在日本東京簽署。<ref name="記" />{{rp|712}}。12月1日，[[總統府秘書長|總統府秘書長]][[張群|張群]]拜會佐藤榮作，代表蔣中正賀日、韓復交<ref name="記" />{{rp|718}}。[[文化大革命|文化大革命]]後，中共與日本貿易總額略有增加，因之日本投機政客積極從事與中共建交之活動<ref name="現代史" />{{rp|210}}。由於[[周恩來|周恩來]]敵視日本佐藤內閣，建交談判遲遲未獲具體結果<ref name="現代史" />{{rp|210}}。

1967年7月，中華民國、美國、日本、韓國領袖在[[漢城|漢城]]舉行非正式會議，時任副總統[[嚴家淦|嚴家淦]]銜命代表參加<ref name="年表" />{{rp|116}}。9月7日，日本首相佐藤榮作訪台，兩度晉見蔣中正商討台日共同問題，並就當時世局及中國大陸中共情勢交換意見<ref name="年表" />{{rp|116-117}}。9月12日，蔣接見日本記者發表談話，指出中共內部必陷長期混亂，盼日本人尊重東方傳統，不使共產主義有滲透之機會，唯有台日加強合作，始能達到此一目的<ref name="年表" />{{rp|117}}。11月27日，[[中華民國國防部部長|中華民國國防部部長]]蔣經國訪問日本，[[日本天皇|日本天皇]][[裕仁|裕仁]]接見時表示永不忘懷蔣中正寬大德意<ref name="年表" />{{rp|117}}。[[蔣經國|蔣經國]]並與佐藤首相、[[眾議院_(日本)|眾議院]]議長石井光次郎、[[參議院議長_(日本)|參議院議長]][[重宗雄三|重宗雄三]]、[[外務大臣_(日本)|外務大臣]][[三木武夫|三木武夫]]、[[防衛大臣|防衛廳長官]][[增田甲子七|增田甲子七]]等人會面，謁吉田茂之墓，參訪[[日產|日產]]和[[東芝|東芝]]等主要企業工廠，以及[[日本放送協會|NHK]]與每日新聞社，12月2日返台。1萬5千餘名日本國民在東京舉行感謝蔣中正大會<ref name="年表" />{{rp|117}}。12月，日本前首相岸信介率團謁見蔣中正<ref name="年表" />{{rp|118}}。

1968年2月27日，駐日大使[[陳之邁|陳之邁]]向[[日本政府|日本政府]]嚴重抗議日本欲與[[蒙古人民共和国|外蒙]]建交企圖<ref name="記" />{{rp|750}}。6月8日，蔣中正接見日本新聞編輯訪台團，重申堅決反對「兩個中國」之一貫立場，並強調日本如果廢除《吉田書簡》，無異廢棄《中日和約》；蔣中正指出中共為[[東亞|東亞]]災禍之源，所謂中立共存，將貽無窮禍害，日本應合作消弭危機<ref name="年表" />{{rp|119}}。

1970年1月，蔣中正接見日本外籍記者訪問團時鄭重指出：不論任何國家的政策如何轉變，我保衛自由與安全的立場將永不改變<ref name="年表" />{{rp|123}}。並正告日本人民，中共圖赤化日本，必須分清敵友<ref name="年表" />{{rp|123}}。7月，嚴家淦代表蔣中正訪問日本<ref name="年表" />{{rp|123}}。

=== 斷交後至蔣經國時期（－1988年1月13日） ===
[[File:Voting_res_2758.png|thumb]]，即《[[聯合國大會2758號決議|聯合國大會2758號決議]]》，其中包括日本。]]
1971年10月，日本在[[聯合國大會2758號決議|聯合國大會2758號決議]]表決時投反对票，反對中華人民共和國取代[[中國與聯合國#代表權問題|中華民國在联合国的席次]]，但最终失敗。

1972年7月，立場親台的日本首相[[佐藤榮作|佐藤榮作]]宣佈退休<ref name="現代史"/>{{rp|194}}。周恩來乃利用時機主動發表雙方儘速建交之聲明<ref name="現代史" />{{rp|210}}。7月6日，[[日本國會|日本國會]]通過親中的[[田中角榮|田中角榮]]繼任首相，不顧日本民意，於9月25日訪問[[中華人民共和國|中華人民共和國]]，9月29日雙方發表《[[中華人民共和國政府和日本國政府聯合聲明|聯合公報]]》，[[外交#外交上的承認|建立外交關係]]<ref name="現代史"/>{{rp|194}}。日本政府同時與中華民國政府斷絕邦交，當時日本政府之苦衷曾由大平正芳外相率直表示為「斷腸之痛」，不僅是政府而已，甚至多數日本國民也有同樣感慨<ref>{{cite book|title=《總統蔣公哀思錄》第二編|location=台北|publisher=總統蔣公哀思錄編纂小組|date=1975-10-31|editor=總統蔣公哀思錄編纂小組|chapter=〈以德報怨的人〉——日本東京《每日新聞》一九七五年四月七日社論|quote=……日本政府於一九七二年九月與中共「建交」，同時與蔣先生的中華民國政府斷絕了邦交。當時日本政府的苦衷曾由大平外相率直的表示為「斷腸之痛」，不僅是政府而已，甚至多數的國民也有同樣感慨。……}}</ref>{{rp|7}}。同日，中華民國外交部發表聲明，宣佈對日絕交。12月28日，關閉[[外交代表機構#大使馆|大使館]]。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = [[中華民國外交部|中華民國外交部]]}}</ref>惟與日本民間及其反共人士仍維持友好之關係，雙方貿易及文化交流繼續發展<ref name="現代史"/>{{rp|194}}。1973年日本國會議員成立[[日華議員懇談會|日華議員懇談會]]，以維持雙方交流。

1975年4月5日，[[蔣中正逝世|蔣中正逝世]]。日本[[內閣官房長官|內閣官房長官]][[井出一太郎|井出一太郎]]代表[[日本政府|日本政府]]發表一個「非官方」談話稱讚蔣：「在激動歷史的轉變中的日本國民之間所熟知而且親密的人士，他是現代中國歷史上的人物……確信必將留住人們的記憶裏……蔣總統是再造日本的大恩人，蔣中正以德報怨的對日政策，加速了日本戰後重建與復興。蔣總統的逝世對日本國民有如晴天霹靂，有良知的日本國民莫不悲痛萬分」<ref>{{cite book|title=《蔣介石的台灣時代》|location=台北|publisher=風雲時代出版社|year=1993|author=李松林|page=487}}</ref>，「……蔣總統在本世紀初曾留學日本，並有很多日本友人，為深受日本國民欽佩、敬愛之人物。渠（他）一生與現代中國歷史息息相關，亦為一部中國現代史，深信其豐功偉績將永遠留存人類之記憶中。……」<ref name="錄二">{{cite book|title=《總統蔣公哀思錄》第二編|location=台北|publisher=總統蔣公哀思錄編纂小組|date=1975-10-31|editor=總統蔣公哀思錄編纂小組}}</ref>{{rp|7}}同日，前首相[[佐藤榮作|佐藤榮作]]談話稱：「日本將永遠難忘蔣總統於第二次世界大戰後給予日本的大恩大德。最令日本人難忘的一件事就是，蔣總統於第二次世界大戰後對日本以德報怨的恩惠。」<ref name="錄二"/>{{rp|7}}《[[日本經濟新聞|日本經濟新聞]]》說：「蔣氏之死是反攻象徵的消失和國共對立抗爭解決的好機會」<ref>{{cite web |url=http://junshi.xilu.com/2011/0421/news_345_154108.html |title=讓人意想不到：蔣介石逝世時日本人的反應 |publisher=西陸東方軍事 |author=要搞清楚 |date=2011-04-21 }}</ref>。4月7日，自民黨幹事會決定派遣前日本首相佐藤榮作為“自民黨總裁代理”名義前往台灣弔喪。消息發表後，立刻引起中國大陸方面強烈譴責。中國大陸對日本政府兩面討好作風反應極大，認為“在此《[[中日和平友好條約|中日和平友好條約]]》正在交涉締約的階段，日本政府此舉，實是給日後的中日友好關係予以惡劣的影響”。三木聽到中國聲明後，“為了不加深對中國方面的刺激”，改變主張，認為最好以私人名義前往弔唁。結果，在自民黨會議上，佐藤榮作以自民黨「友人代表」出席蔣中正喪禮。其餘出席者尚有其兄[[岸信介|岸信介]]、眾議員[[石原慎太郎|石原慎太郎]]等人。

1978年6月12日，《中華週報》刊出[[蔣經國|蔣經國]]答記者問，表示不能接受採取日本模式，中止美國外交關係，但要廣泛地維持貿易關係；[[中華民國與美國關係|美台關係]]和[[中華民國與日本關係|日台關係]]完全不同<ref>{{cite news |quote=記者問[[蔣經國|蔣經國]]：「有人建議，[[美國|美國]]和中華民國的關係，採取日本模式，中止外交關係，但要廣泛地維持貿易關係。對此總統覺得怎樣？」 |title=蔣總統答記者指摘 |publisher=中華週報 |date=1978-06-12 }}</ref>。

1984年1月3日，[[行政院新聞局|新聞局]]宣佈專案准許四部日本影片進口，為自1973年以來的首次解禁。

=== 李登辉时期（1988年1月13日－2000年5月20日） ===
