'''臺南市政府衛生局'''（簡稱'''臺南市衛生局'''）<ref>[https://health.tainan.gov.tw//page.asp?nsub=A0A300 臺南市政府衛生局組織架構]</ref><ref>{{cite web |author1=杜忠聰 |title=臺南市政府衛生局辦理107年「廉政診療室」社會參與暨機密及安全維護宣導活動 |url=https://times.hinet.net/news/21924158 |website=hinet.net |accessdate=2018-08-26}}</ref><ref>{{cite web |author1=杜忠聰 |title=臺南市政府衛生局企業誠信系列專題107年第3季宣導活動 |url=https://news.sina.com.tw/article/20180817/27883804.html |publisher=新浪台灣 |accessdate=2018-08-26}}</ref>，2010年12月25年成立，是[[臺南市政府|臺南市政府]]所屬的一級機關，為臺南市衛生最高主管機關。

== 歷史 ==
2018年7月，臺南市政府衛生局前食藥科許姓科長等3名官員，因私人交往關係涉及圖利及洩密，[[台南地檢署|台南地檢署]]今天偵結起訴請求法官從重量刑<ref>{{cite news|author=張榮祥   |title=涉圖利洩密 台南市衛生局3官員遭訴|url=http://www.cna.com.tw/news/asoc/201807030317-1.aspx|language=zh-hant|accessdate=2018-07-26|newspaper=《[[中央社|中央社]]》 |date=2018-07-03}}</ref><ref>{{cite news|author=洪榮志   |title=涉圖利特定廠商 南市衛生局3官員被訴|url=http://www.chinatimes.com/realtimenews/20180703002983-260402|language=zh-hant|accessdate=2018-07-26|newspaper=《[[中國時報|中國時報]]》 |date=2018-07-03}}</ref>。
