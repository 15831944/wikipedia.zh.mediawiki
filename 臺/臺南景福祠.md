'''臺南景福祠'''位於[[臺南市|臺南市]][[中西區_(臺灣)|中西區]]，又稱'''佛頭港景福祠'''，於民國七十四年（1985年）11月27日公告為三級古蹟{{r|文}}。該廟主祀[[福德正神|福德正神]]，為臺南市唯一被列為古蹟的土地公廟{{r|傅}}，該地過去是[[臺灣府城|臺灣府城]]西定坊的五條港之一的佛頭港{{r|石}}，而佛頭港港道會在這裡分成「王宮港」、「媽祖港」與「關帝港」三條港道延伸進府城中的[[大天后宮|大天后宮]]與[[開基武廟原正殿|開基武廟]]一帶{{r|遠}}，亦即是景福祠為佛頭港上游的匯合處{{r|傅}}。現在港道雖已不存，但該廟所在地形成的永樂市場依然十分熱鬧。

== 沿革 ==
[[File:臺南景福祠三川門.JPG|230px]]
景福祠是在清乾隆十五年（1750年）時興建，當時只有拜亭與正殿，格局顯得狹窄{{r|石}}。到了乾隆四十三年（1778年）時，該廟信徒購置店屋出租，將租金作為該廟的香火{{r|石}}。當時該廟頗受重視，乾隆四十五年（1780年）所立的「佛頭港福德祠碑記」便明文規定廟前的店屋不可以高過此廟{{r|遠}}。然而到了嘉慶十二年（1807年），該廟發生火災，於三年後集資大修{{r|石}}，留下了嘉慶十六年（1811年）的「重建景福祠碑記」和廟中「萬物資生」匾作為此事見證{{r|遠}}。之後該廟有150年未曾大修，只有小規模的修補{{r|石}}。

二次大戰後，該廟在民國五十三年（1964年）整建，而七十年（1981年）時再修，而成今貌{{r|傅}}。
[[File:臺南景福祠萬物資生匾.JPG|230px]]

== 建築特色 ==
該廟的建築只由正殿與拜亭組成，而在正殿兩側的山牆有人物泥塑，裡頭空間不大{{r|傅}}。而在廟前左側立有清朝石碑兩座與民國七十年重修時的石碑一座，外面有玻璃保護著。廟中掛著的嘉慶古匾「萬物資生」則為該廟最重要的古匾{{r|傅}}。

== 其他 ==
佛頭港昔日運送的貨物以從[[福州|福州]]來的杉木為大宗，有很多杉行在此{{r|傅}}，因此景福祠前的普濟街古名為「杉行街」{{NoteTag|1=杉行街在二次大戰後是先改名為「人和街」，再改名為今天的「普濟街」{{r|傅}}。}}{{r|遠}}。而當時在碼頭有「大崙蔡」與「前埔蔡」兩個苦力集團，時常因搶著搬運貨物而發生衝突，再加上兩邊剛好同姓，而有了「蔡堵蔡，神主摃摃破」的俗諺{{r|遠}}。

除此之外由於佛頭港的水道較為寬長，昔日船夫與碼頭苦力會在這裡賽龍舟，為臺灣府城的一大盛事{{r|遠}}。
