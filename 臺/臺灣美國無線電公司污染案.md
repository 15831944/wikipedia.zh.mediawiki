'''臺灣美國無線電公司污染案'''，又稱RCA事件、RCA污染事件或RCA污染案，是一件發生於[[台灣|台灣]]桃園縣桃園市（今[[桃園市|桃園市]][[桃園區|桃園區]]）的[[土壤污染|土壤污染]]及[[地下水污染|地下水污染]]公害事件。

== 背景 ==
[[File:RCA_Taiwan_Limited_recruitment_on_Central_Daily_News_1969-12.gif|thumb]]》1969年12月27日第6版刊登的[[招聘|招聘]]廣告。]]

[[美國無線電公司|美國無線電公司]]，簡稱RCA，曾經是[[美國|美國]][[家電|家電]]第一品牌，生產[[電視機|電視機]]、[[映像管|映像管]]、[[錄放影機|錄放影機]]、[[音響|音響]]等產品。在[[台灣奇蹟|台灣經濟起飛時期]]的1970年至1992年期間，RCA在台灣設立[[子公司|子公司]]「台灣美國無線電股份有限公司」（RCA Taiwan Limited），並在[[桃園區|桃園]]、[[竹北市|竹北]]、[[宜蘭市|宜蘭]]等地設廠。1970年，RCA於[[桃園市|桃園縣]][[桃園區|桃園市]]中山路、[[文中路|文中路]]與富裕街之間設置桃園廠為總廠，生產電子產品、電器產品、[[電視機|電視機]]之電腦選擇器，廠房占地約7.2[[公頃|公頃]]{{r|thb}}。1986年，美國無線電公司被美國[[奇異公司|奇異公司]][[併購|併購]]，繼續生產電視機之電腦選擇器<ref name="國內外整治成功案例介紹">{{Cite web |url= http://www2.nsysu.edu.tw/IEE/lou/part_3/lesson_9/ch9_con.htm | author=|language=zh-tw ||title=國內外整治成功案例介紹|publisher=  }}</ref>。1988年，[[法國|法國]]{{tsl|fr|Technicolor (entreprise française)|湯姆笙公司}}（Thomson Consumer Electronics，TCE）從奇異公司取得RCA桃園廠之[[產權|產權]]。1991年，湯姆笙公司發現，RCA桃園廠[[有機化學|有機化學]]廢料排入廠區造成污染。1992年，湯姆笙將RCA桃園廠關廠。1992年3月，湯姆笙將RCA桃園廠廠區土地[[所有權|所有權]]出售予[[宏億建設|宏億建設]]，宏億建設準備將此地開發成[[購物中心|購物中心]]。

== 污染事件曝光 ==
1994年6月1日，當時的[[立法委員|立法委員]]、前[[行政院環境保護署|行政院環境保護署]]（環保署）署長[[趙少康|趙少康]]召開[[記者會|記者會]]，舉發RCA桃園廠長期挖[[井|井]]傾倒[[有機溶劑|有機溶劑]]等有毒廢料，導致廠區之土壤及地下水遭受嚴重污染。環保署隨即於6月2日委由具有美國[[超級基金|超級基金]]法律實務經驗之[[丁力行|丁力行]]教授率隊進入場址配合業者進行調查蒐證工作，經由現場具體證據證實該廠土壤及地下水確實存在污染之疑慮。環保署隨後成立調查專案小組，函知[[中華民國內政部|內政部]]暫停該廠址之土地變更用途。6月2日，丁力行要求、並取得業者代表Rip Dyer之同意，自隔日起由業者緊急供應居民[[瓶裝水|瓶裝水]]及接裝[[自來水|自來水]]，請當地里長及主管單位向居民持續宣導自即日起切勿使用地下水，以降低可能的健康風險。環保署同時成立監督小組，要求RCA、奇異公司及湯姆笙公司儘速移除污染源{{r|thb}}，並監督污染調查工作。環保署委託[[工業技術研究院|工業技術研究院]]調查RCA桃園廠附近民井地下水質，發現主要之污染物為[[1,1-二氯乙烷|1,1-二氯乙烷]]、[[1,1-二氯乙烯|1,1-二氯乙烯]]、[[四氯乙烯|四氯乙烯]]、[[1,1,1-三氯乙烷|1,1,1-三氯乙烷]]、[[三氯乙烯|三氯乙烯]]等當時電子業常使用的具有[[揮發性|揮發性]]之含氯[[有機化合物|有機化合物]]{{r|thb}}。

== 土壤及地下水污染控制與整治 ==
1995年5月29日，宏億建設表示，已準備妥當充分資料，將於[[美國時區|美國時間]]本月30日委託律師在湯姆笙公司註冊所在地的美國[[德拉威爾州|德拉威爾州]]提起訴訟，控告湯姆笙公司[[詐欺|詐欺]]等罪名。

1996年，在環保署的壓力下，奇異公司與湯姆笙公司進行RCA桃園廠廠址污染調查，花費[[新台幣|新台幣]]2億多元進行土壤整治。1997年，RCA桃園廠廠址完成土壤整治，然而奇異公司表示無法整治已遭受污染的地下水。根據奇異公司於1998年提出的報告，RCA桃園廠廠址污染整治達到核定之基準有困難<ref name="國內外整治成功案例介紹"/>，意即地下水污染仍未解決。由於RCA當年故意將桃園廠生產過程中產生之具汙染性的有機溶劑以非法秘密方式直接打入自行開挖的水井中，並且在水井內鋪上沙土以吸收污染物，使環保署當初稽查時無法發現污染；而且由於上述污染物與地下水並不互溶，因此會隨著水流緩慢擴散，造成廠址之地下水永久性污染而難以復原<ref>{{Cite web |url= http://ecofriendlchi.blogspot.tw/2012/05/59-rca.html | author=|language=zh-tw ||title=5.9 我說桃園的RCA|publisher= 種樹在方寸}}</ref>。

1998年，宏億建設提出RCA桃園廠廠址變更開發申請，環保署環境影響評估審查委員會審查結論公告「有條件通過環境影響評估審查」，結論包括「本基地之污染地區污染物未清除（完成整治）前，不得申請建造執照」、「本基地污染地區不得興建建築物」等9項決議。

1998年10月，[[監察院|監察院]]提出的糾正案指出：1975年至1991年間，[[行政院勞工委員會|行政院勞工委員會]]（勞委會）曾對RCA進行8次勞動檢查，發現RCA違反《有機溶劑中毒預防規則》、《勞工健康管理規則》和《勞工安全衛生規則》，但僅以公文要求RCA改善，既無罰則、也沒有追蹤列管，未善盡保護勞工之責；此外，勞委會、環保署、[[行政院衛生署|行政院衛生署]]基於分工，各自為政，未積極就化學物質影響、罹癌員工醫療救助與法律補償協助等方面建立處理與補償機制，確有不當。

1999年，行政院成立「RCA行政院跨部會專案小組」，由[[行政院政務委員|行政院政務委員]][[黃大洲|黃大洲]]主持，整合勞委會、衛生署、環保署、法務部、經濟部等相關單位；並持續邀請受害者參與，協助處理[[全民健康保險|全民健康保險]]給付、健康檢查追蹤、RCA桃園廠廠址污染等事項。但2000年[[第一次張俊雄內閣|第一次張俊雄內閣]]期間，行政院撤銷此專案小組。<ref>{{Cite web|url=http://www.coolloud.org.tw/node/3699|author=RCA員工自救會、台灣人權促進會、工作傷害受害人協會、民間司法改革基金會|language=zh-tw|title=4億飛了，跨國企業毒害工人不必賠？ 前RCA罹癌員工要求恢復「RCA行政院跨部會專案小組」|publisher=苦勞網|date=2007-06-14}}</ref>

2000年1月，宏億建設提出RCA桃園廠廠址地下水污染來源控制、應變及管理計畫，擬對汙染源區四周施作[[連續壁|連續壁]]工程阻絕擴散，並以加強生物處理方式進行整治；該計畫獲得環保署審核通過，但宏億建設因財務狀況不佳而未進行整治工作。

2002年4月26日，[[桃園縣政府|桃園縣政府]]依照《[[:s:土壤及地下水污染整治法|土壤及地下水污染整治法]]》，將RCA原桃園廠公告為「污染控制場址」。2004年3月19日，環保署依據桃園縣政府初步評估結果，確認RCA原桃園廠地下水已達地下水污染管制標準後，公告RCA原桃園廠為「地下水污染整治場址」<ref>{{Cite web |url= http://sgw.epa.gov.tw/Public/Download_Data/AnnualReport/96%E7%AC%AC%E5%8D%81%E7%AB%A0.pdf | author= 行政院環境保護署|language=zh-tw ||title=重大污染場址|publisher= 環保署96年度土壤及地下水污染整治年報}}</ref>。

2005年7月5日，環保署表示，確認RCA原桃園廠汙染案的法定「污染行為人」是最初在台灣設廠的RCA，RCA應負起整治責任。

== 受害員工 ==
===受害原因===
經過台灣多位學者研究，RCA桃園廠污染場址之各種污染物，確實有相當高的機率導致[[癌症|癌症]]或影響健康。由於RCA桃園廠許多受僱[[勞工|勞工]]在工作時吸入或[[皮膚|皮膚]]接觸有機溶劑，員工長期暴露在高度致癌風險之工作環境。另外，RCA桃園廠生產線勞工在廠區喝的[[飲水機|飲水機]]是接地下水而非自來水，而外地受僱勞工住在廠外附近的員工[[宿舍|宿舍]]，宿舍內使用的水也是地下水；因此員工洗澡或飲水皆容易大量暴露在高致癌性之有機溶劑<ref>《司法改革雜誌》，RCA污染事件始末，2001年，第59至60頁</ref>。

===受害症狀===
1998年6月，[[台灣媒體|台灣媒體]]報導，原RCA桃園廠員工及廠址附近居民罹患癌症或死亡{{r|thb}}。RCA在台灣設廠之期間，雇用員工高達2萬至3萬人。根據2001年統計，在RCA桃園廠工作多年的員工，至少已經有1,375人罹患[[癌症|癌症]]，包括[[乳癌|乳癌]]、[[子宮頸癌|子宮頸癌]]、[[肝癌|肝癌]]、[[大腸癌|大腸癌]]、[[鼻咽癌|鼻咽癌]]等各式[[惡性腫瘤|惡性腫瘤]]，其中216人已過世<ref>{{Cite web|url=http://www.taiwan-panorama.com/show_issue.php?id=2007109610075C.TXT|author=滕淑芬|language=zh-tw|title=台灣版「永不妥協」 ──RCA職災事件|publisher=《台灣光華雜誌》|date=2007-10}}</ref>。許多受害員工經由媒體報導才得知，原來自己罹癌與以前同事因癌症過世，皆源於RCA桃園廠土地被污染之故。

然而，裁判書中引用的WHO IARC目前為止的科學證據中，也只有三氯乙烯為一級致癌物，以及其對導致腎臟癌是有充分證據的狀況；其他成分的風險層級、以及與其他癌症的關聯，其實並沒有對應。特別是部分致癌物是廣泛存在的、且對應於多種癌症的狀況下，比如為一級致癌物，且為多種癌症有充分證據的菸、酒、B型肝炎病毒、C型肝炎病毒等。<ref>[http://monographs.iarc.fr/ENG/Classification/index.php　AGENTS CLASSIFIED BY THE IARC MONOGRAPHS, VOLUMES 1–120] WHO IARC</ref>

===法律救濟===
1998年7月，RCA桃園廠受害員工宣布籌組「RCA污染受害者自救會」，並打算對RCA提告求償；自救會籌備完成後，正式登記為「桃園縣原台灣美國無線電公司員工關懷協會」，簡稱「RCA員工關懷協會」，但對外行動仍簡稱「RCA員工自救會」<ref>{{Cite web|url=http://www.coolloud.org.tw/node/75762|author=黃小陵|language=zh-tw|title=經濟奇蹟背後，永不妥協的RCA工人|publisher=苦勞網|date=2013-09-29}}</ref>。2001年5月，80位[[律師|律師]]組成義務律師團。2002年7月，義務律師團向[[台北地方法院|台北地方法院]]聲請[[假扣押|假扣押]]RCA在台灣的資金，並查詢RCA在台灣的資金流向，但[[經濟部投資審議委員會|經濟部投資審議委員會]]以[[機密|機密]]為由拒絕提供；義務律師團轉向[[國稅局|國稅局]]調閱資料，發現RCA於2000年度的利息所得只剩美金33萬元，換算成[[本金|本金]]只有新台幣一千多萬元。2003年，RCA員工關懷協會解除對義務律師團的訴訟委任。

2004年4月23日，三十多名前RCA桃園廠員工與RCA桃園廠廠址附近居民到[[台北地檢署|台北地檢署]]及台北地方法院提起團體訴訟，並請求[[民事|民事]]賠償新台幣24億元。2005年3月24日，針對RCA員工關懷協會提起的此次團體訴訟，台北地方法院認為，RCA員工關懷協會未依法登記為「公益社團法人」，不具備團體訴訟的當事人資格，故將全案程序駁回。後來RCA員工關懷協會上訴[[中華民國最高法院|中華民國最高法院]]，最高法院廢棄先前判決、發回[[臺灣高等法院|臺灣高等法院]]，再由臺灣高等法院發回台北地方法院審理<ref name="RCA污染案 民事求償15年來首次進入實體訴訟">{{Cite web |url= http://e-info.org.tw/node/49251|author=黃-{彗}-珊、莫聞|title=RCA污染案 民事求償15年來首次進入實體訴訟|publisher= [[台灣環境資訊協會|台灣環境資訊協會]]環境資訊中心|date=2009-11-12}}</ref>。

2007年，由[[法律扶助基金會|法律扶助基金會]]、[[台北律師公會|台北律師公會]]、[[民間司法改革基金會|民間司法改革基金會]]、[[台灣人權促進會|台灣人權促進會]]及[[林永頌|林永頌]]、[[魏千峰|魏千峰]]、[[李秉宏|李秉宏]]等多位律師共同籌組義務律師團，除以RCA為[[被告|被告]]外，並將奇異公司及湯普笙公司列為被告，透過法律訴訟爭取賠償<ref>{{Cite web |url= http://www.jrf.org.tw/newjrf/Layer2/aboutjrf_2-1.asp?id=936 | author=|language=zh-tw |title=RCA職業災害－－台灣司法史上史無前例的案件|publisher= 民間司法改革基金會}}</ref><ref name="蘋果日報">{{Cite web|url=http://www.appledaily.com.tw/appledaily/article/headline/20131124/35460082/|title=人物特寫：盲律師聽讀打字 接案拼勝|language=zh-tw|publisher=《蘋果日報》|author=許麗珍 |date=2013-11月-24|accessdate=2015-04-26}}</ref>。2009年11月11日，台北地方法院首次傳喚受害人出庭作證，RCA案正式進入訴訟程序<ref name="RCA污染案 民事求償15年來首次進入實體訴訟"/>，至2013年持續訴訟中<ref>{{Cite web|url=http://www.coolloud.org.tw/node/74653|author=林岳德|language=zh-tw|title=「因為RCA，我們一家都受害」──2013年6月13日RCA工傷案開庭側記|publisher=苦勞網|date=2013-06-17}}</ref>。

2013年9月13日，台北地方法院再次開庭審理RCA桃園廠汙染案，汙染案受害者及家屬共同發表[[口述歷史|口述歷史]]書《拒絕被遺忘的聲音：RCA工殤口述史》並當庭交給法官<ref>{{Cite web|url=http://life.chinatimes.com/LifeContent/1401/20130914000867.html|author=林欣誼|language=zh-tw|title=RCA工殤抗爭 19年仍在進行式|publisher=[[中國時報|中國時報]]|date=2013-09-14}}</ref>；此書由RCA員工關懷協會與中華民國工作傷害受害人協會共同製作、[[行人文化實驗室|行人文化實驗室]]出版（ISBN 978-9868965249），記載汙染案12位受害者的真實故事<ref>{{Cite web|url=http://www.coolloud.org.tw/node/75718|author=陳逸婷|language=zh-tw|title=拒絕被遺忘的RCA工殤口述史|publisher=苦勞網|date=2013-09-27}}</ref>。

2015年4月17日，台北地方法院一審宣判RCA員工關懷協會勝訴，RCA、湯姆笙公司須賠償新臺幣5億6445萬元。<ref>{{Cite web |url=http://news.ltn.com.tw/news/society/breakingnews/1289752|author=|language=zh-tw|title=史上最大RCA工傷案 一審判賠5.6億|publisher=自由時報}}</ref><ref>{{Cite web |url= http://www.cnabc.com/news/aall/201504170301.aspx|author=|language=zh-tw |title=RCA案相關大事記|publisher=中央社商情網}}</ref>而一審判決採納勞工方請求的原因，大致可歸納為三點。第一、[[民事訴訟|民事訴訟]]上[[因果關係|因果關係]]的[[舉證責任|舉證責任]]，原則上由原告負擔。但[[法官|法官]]考量勞工跟RCA的知識與財力落差，加上本案相關資訊均位於RCA控制中，所以RCA所排放的污染與勞工的健康損害間有無因果關係，應由RCA負擔舉證責任證明勞工的傷害與RCA的汙染物無關。在分配完因果關係的舉證責任後，法院進一步強調，由於汙染物與疾病之間本來就有一定程度的不確定性，而本案呈現的科學證據已能證明汙染物與疾病有特定關聯，兩者顯然有因果關係。第二、法院在本案中延後了請求權時效的起算時間點。儘管RCA抗辯許多員工從離職到發病中間已相隔超過法定的十年[[時效|時效]]，但法院仍認為，既然[[侵權行為|侵權行為]]的要件之一包含損害發生，自然必須從發病後開始起算時效才是合理的解釋方法；至於還沒有發病的勞工，也同樣受到健康風險上升的威脅，一樣可以在知悉RCA汙染事件後十年內向RCA求償。第三、因RCA已大幅脫產，縱使勞工勝訴也未必能獲得足額賠償，法院採計勞方律師的立場，認為RCA對勞工應負侵權行為的損害賠償責任；依照[[美國|美國]]法院見解，RCA的控制公司如湯姆笙公司等，亦應該依據[[揭穿公司面紗原則|揭穿公司面紗原則]]對勞工負擔賠償責任，以避免這些控制公司透過公司制度逃避損害賠償責任。<ref>{{cite web| url=http://itdels.digital.ntu.edu.tw/Item.php?ID=A_0004_0013_0002 | title=臺北地方法院95年度重訴更一字第4號民事判決| publisher=台灣法實證研究資料庫 | date=2015-11-09 | accessdate=2016-07-24}}</ref><ref>{{cite web| url=http://tadels.law.ntu.edu.tw/upload/edm_file/Tadels_Communication_16_20151117.pdf | title=RCA 工傷案報導及「霸王條款」介紹 | publisher=基礎法學與人權研究通訊第16期 | date=2015-11-17 | accessdate=2016-07-24}}</ref>

儘管台北地方法院一審判決RCA員工關懷協會勝訴，但由於RCA竹北廠勞工、以及部分在提起訴訟前即死亡的勞工仍未能獲得賠償，RCA員工關懷協會仍強調會上訴，繼續與RCA對簿公堂以爭取權利。<ref>{{Cite web |url=https://tw.news.yahoo.com/rca%E5%86%8D%E4%B8%8A%E8%A8%B4-%E8%87%AA%E6%95%91%E6%9C%83%E7%B1%B2%E8%90%AC%E5%90%8D%E5%8F%97%E5%AE%B3%E8%80%85%E5%87%BA%E9%9D%A2-032100597.html|author=沈雅雯|title=RCA再上訴 自救會籲萬名受害者出面|publisher=中央廣播電台|date=2015-05-26}}</ref>
2017年10月27日，台灣高等法院經兩年審理，宣判RCA等四家公司須連帶賠償486人，賠償金額增加到新台幣7億1840萬元。
