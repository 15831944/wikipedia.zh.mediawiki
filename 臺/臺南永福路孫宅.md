'''臺南永福路孫宅'''，為[[臺南市|台南市]][[中西區|中西區]]永福路的一棟木造建築，興建於1940年代，由住吉組承造。於2016年指定為[[臺南市文化資產|台南市歷史建築]]。

== 沿革 ==
原為勸業協會的辦公室，其主要工作是配合其對面之台南市役所（今 [[臺南市中西區永福國民小學|永福國小]]）的政令，開發不動產、勸導ㄔ企業家，以及做金融融資、擔保等工作。1945年戰後，由[[葉南輝|葉南輝]]醫師取得，作為別墅使用。1950年，轉讓予孫金寬。之後曾一度借給[[連震東|連震東]]，作為[[國民大會|國民大會]]代表選舉事務所。<ref>{{Cite web|title=文化部文化資產局--國家文化資產網|url=https://nchdb.boch.gov.tw/assets/overview/historicalBuilding/20160225000001|work=nchdb.boch.gov.tw|accessdate=2019-04-07}}</ref>

2014年10月，其所有權人曾有意將此建築拆除，經[[臺南市政府文化局|台南市文化局]]協調後，於隔月將其列為暫定古蹟；並在2016年指定為歷史建築。<ref>{{Cite web|title=〈南部〉老屋即刻救援 台南文化局擋拆奏功 - 地方 - 自由時報電子報|url=https://news.ltn.com.tw/news/local/paper/883149|work=自由電子報|accessdate=2019-04-07|language=zh-Hant-TW}}</ref>

== 建築特色 ==
永福路孫宅為一獨棟兩層樓建築，四周設有圍牆，並留設庭院，通風採光良好。其各部開窗、角隅凸窗作法、屋頂、屋簷作法特別。大門柱與溝面磚、石宮燈等保存良好，具建築史研究價值。其用途因為由辦公室轉為住宅使用，後來在北側另設置廚房及浴室。<ref>{{Cite web|title=文化部文化資產局--國家文化資產網|url=https://nchdb.boch.gov.tw/assets/overview/historicalBuilding/20160225000001|work=nchdb.boch.gov.tw|accessdate=2019-04-07}}</ref>

== 其他 ==
其南側為[[葉石濤|葉石濤]]筆下的蝸牛巷。
