'''臺北市立聯合醫院企業工會'''，簡稱'''北市聯醫工會'''，是由[[臺北市立聯合醫院|臺北市立聯合醫院]]受僱勞工組成的[[工會|工會]]。會址於[[台北市|台北市]][[萬華區|萬華區]]萬大路184號2樓，與[[台北市產業總工會|台北市產業總工會]]共用辦公空間。

== 成立起源 ==
[[衛生福利部健康保險署|衛生福利部健康保險署]]於2015年年底公佈最賺錢的10大醫院，臺北市立聯合醫院盈餘[[新臺幣|新臺幣]]6.1億元、位居全台第4名<ref>2015-12-30 聯合新聞網，[http://www.cw.com.tw/article/article.action?id=5073540 〈最賺錢、最賠錢的健保醫院〉]</ref>。但[[臺北市政府勞動局|臺北市政府勞動局]]於2016年1月11日公佈全北市醫療院所勞動條件的檢查結果，不合格的醫院共計18所，臺北市立聯合醫院共8個院所，就高達5個院所違反《勞動基準法》，而仁愛院區、松德院區更違反高達5項<ref>2016-1-12 自由時報，[http://news.ltn.com.tw/news/life/paper/948226 〈北市勞檢大醫院 18家統統不合格〉]</ref>，其他3個院區陽明院區、中興院區、忠孝院區皆榜上有名，醫療院所違反《勞基法》第24條的未給付加班費尤為嚴重<ref>2016-1-12 自由時報，[http://health.ettoday.net/news/629378 〈爆！北市聯醫去年盈餘高達6.1億　勞工加薪不到300元〉]</ref>；同年7月護理人員出面控述，申請加班費44次，均遭駁回<ref>2016-7-2 蘋果日報，[http://www.appledaily.com.tw/appledaily/article/headline/20160702/37292636/ 〈申請44次加班費均駁回而被迫離職〉]</ref>。故特此由基層[[護理師|護理師]]自行組織，並由臺北市勞動局登記立案。

== 籌備期間 ==
*2015年11月基層護理師從臉書成立「北市聯醫工會籌備會」到[https://www.facebook.com/taipeicity.union/ 「臺北市立聯合醫院企業工會」]，2016年北市聯醫工會籌備會成員針對工作超時、未給加班費，從年初開始奔波北市聯醫8個院區，協助第一線醫護人員處理勞動問題，逐步由下而上的組織團結。

== 訴求 ==
2016年5月1日，北市聯醫工會籌備會參與[[五一大遊行|五一大遊行]]，同時力邀各院區的同仁加入北市聯醫工會籌備行動，共同爭取勞工福利、並保障勞工權益，並上街頭訴求[[蔡英文政府|蔡英文政府]]落實工會籌備會五大訴求<ref>2016-5-1 苦勞網，[http://www.coolloud.org.tw/node/85141 〈【行動】五一大遊行──捍衛基層勞工權益、還我合法勞動環境〉]</ref>:
*一、立法增訂三班護病比，補足第一線照護人力。
*二、推動醫師納入勞動基準法。
*三、減少外包人力，增加正職雇用。
*四、提高勞工基本工資。
*五、勞檢結果納入醫院評鑑。
