'''臺中市私立天主教育仁國民小學'''('''Yu-Jen Catholic Elementary School''')，簡稱'''育仁國小'''或'''育仁小學'''，是一所位於[[中華民國|中華民國]][[台中市|台中市]][[北區_(臺中市)|北區]][[雙十路|雙十路]]二段的一所[[天主教|天主教]][[私立學校|私立學校]]，是在[[1967年|1967年]]由[[比利時|比利時]][[聖母聖心會|聖母聖心會]]的[[神父|神父]]所創立，後來由[[彰化|彰化]][[田中|田中]]耶穌聖心修女會接掌校務<ref>{{Cite web |url = http://www.yjes.tc.edu.tw/yuspec1a.htm |title = 歡迎來到台中市私立育仁小學!!|language = 中文  }}</ref>。

==校史<ref>{{Cite web |url = http://www.yjes.tc.edu.tw/moral/about/about.html  |title = 育仁小學品德教育網 |language = 中文  }}</ref>==

1967年創校，訂名為「私立育仁小學」，宋稚青神父為第一任校長

*1971年設立啟智班

*1977年完成校徽設計

*1983年更名為「私立育仁國民小學」

*1998年「新校舍第一期工程」開工典禮

*2000年「新校舍第一期工程」落成啟用

*2004年7月「新教學大樓第二期工程」開始籌建

*2004年12月「新教學大樓第二期工程」落成

*2017年12月 創校50周年校慶<ref>{{Cite web |url = https://udn.com/news/story/7325/2891184  |title = 育仁小學50周年校慶 師生校友同歡 |language = 中文  }}</ref><ref>{{Cite web |url = https://udn.com/news/story/7325/2446411 |title = 育仁小學50周年慶 師生、家長獻愛心 |language = 中文  }}</ref>
