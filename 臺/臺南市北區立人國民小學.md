'''臺南市北區立人國民小學'''是位於臺灣臺南市[[北區_(臺南市)|北區]]的國民小學，其歷史可追溯到[[臺灣日治時期|日治時期]]創立的「臺南第二公學校」{{r|寶|page=15}}{{r|小學|page=20}}{{r|校史}}。該校也是臺南傳統棒球名校，有陳智源、黃清輝、陳振雄等校友{{r|小學|page=43、45}}。

== 沿革 ==
立人國小的歷史可追溯到日治時期明治卅一年（1898年）創立的「臺南第二公學校」{{NoteTag|1=第一公學校於1897年設於[[臺南孔廟|臺南孔廟]]內，為今天國立臺南大學附設實驗國民小學前身{{r|石}}{{r|寶|page=15}}，目前該校已完全沒有日治時期的校舍保留下來<ref>[http://fuxiao.ps.nutn.edu.tw/?page_id=17 學校簡介].國立台南大學附設實驗國民小學</ref>。}}，該校最初借用[[臺南水仙宮|臺南水仙宮]]附近的[[府城三郊|府城三郊]]總部三益堂（今臺南永樂市場）上課{{r|王}}{{r|石}}{{r|寶|page=15}}{{r|小學|page=20}}{{r|傅}}。臺南第二公學校是在該年（1898年）9月30日核准設立，於10月22日舉行開校典禮{{r|寶|page=15}}。開校當日有臺南縣知事[[磯貝靜藏|磯貝靜藏]]、臺南辨務署署長川田久喜等首長到場，並有當地紳商、書房教師等60多人到場{{r|寶|page=15}}。首任校長是平田忠四郎，教職員有日籍教員近藤森、臺籍雇員郭君盤和葉朗初等3人，共招收了57名學童{{r|寶|page=15}}。

而由於三益堂空間不夠使用，明治卅三年（1900年）5月28日將男學童遷到設在外新街舊育嬰堂{{NoteTag|該地現為郭綜合醫院{{r|寶|page=15}}。}}的校舍，該處有3間教室，並有辦公室、浴室、廁所、宿舍、運動場等設施{{r|寶|page=15}}。

後來由於學生逐漸增多，便在[[德慶溪|德慶溪]]北邊原清軍北廠用地（今址）建立新校舍，於1909年3月完工，共有本館1棟8間，教室3棟各3間，而隔年2月又因校舍被充作臺灣南部共進會展示會場而加蓋了禮堂{{r|石}}。完工之後，男學生便移到這裡上課，女生則仍留在三益堂，不久又移到外新街育嬰堂，後來在1912年於[[赤崁樓|赤崁樓]]北邊的臺灣縣衙上建女生部校舍，獨立成「臺南市女子公學校」（今臺南市成功國小前身）{{r|石}}。

在男生移到新校舍之後，學生仍然持續增加，於是又將鄰近的[[臺南三山國王廟|臺南三山國王廟]]借作校舍，之後於1935年開始分期拆除本館與教室，於1938年完成保留至今的校舍，且於同年4月改名為「寶公學校」{{r|石}}。1942年則因推行國民教育，又改名為「寶國民學校」{{r|王}}。

進入[[臺灣戰後時期|臺灣戰後時期]]之初，該校仍因襲舊名，但不久便在1946年2月改名為「臺南市北區第一國民學校」，隔年2月又以[[孫立人|孫立人]]將軍之名改為「立人國民學校」，直到1968年推行[[台灣九年國民義務教育|九年義務教育]]才改為「立人國民小學」至今{{r|石}}。

== 相關學校 ==
寶公學校（當時仍為第二公學校）在1913年4月增加了二年制實業科，六年後改為簡易商業學校，後來在1922年3月獨立，為今天臺南市「[[國立臺南高級商業職業學校|國立臺南高商]]」前身{{r|石}}<ref>[http://www.tncvs.tn.edu.tw/editor_model/u_editor_v1.asp?id={D4BD5078-4627-400A-A959-6C0F7091DC19} 校史簡介].國立臺南高級商業職業學校</ref>，同年4月增設二年制高等科並持續到戰後的1946年8月{{r|石}}。而1927年4月1日又分出了第四公學校到清朝鎮海營所在地，為今天臺南市協進國小前身{{r|石}}<ref>[http://120.115.16.5:8080/chinese2008/aboutsjps/index.jsp 協進簡介].臺南市協進國小</ref>。1942年4月1日分設出伏見國民學校於今天臺南市延平國中校址，為現在臺南市公園國小前身{{NoteTag|1=伏見國民學校在二次大戰後改為第二國民學校，於1948年2月1日遷入了[[原花園尋常小學校本館|花園尋常小學校校舍]]並改為公園國小，即今天校址。原校址後來便交由今延平國中使用。}}{{r|石}}<ref>[http://www.gyes.tn.edu.tw 臺南市公園國小]</ref><ref>[https://web.archive.org/web/20100617113956/http://www.ypjh.tn.edu.tw:8082/modules/tinyd0/index.php?id=3 校史].臺南市延平國中</ref>。

== 校友 ==
*陳智源：1969年威廉波特世界少棒賽冠軍金龍隊主力投手，後來留學美國[[新墨西哥州立大學|新墨西哥州立大學]]取得機械工程博士學位{{r|小學|page=45}}。
*黃清輝：1971年世界少棒賽冠軍巨人隊成員，1977年考上臺灣大學社會系{{r|小學|page=45}}。
*陳振雄：1973年世界少棒賽冠軍巨人隊成員，後來擔任1993年世界青少棒賽國家隊總教練{{r|小學|page=45}}。
