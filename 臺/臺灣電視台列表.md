[[臺灣|臺灣]]的[[電視台|電視台]]，主管機關為[[國家通訊傳播委員會|國家通訊傳播委員會]]（簡稱NCC）。可分為[[地上數位電視|無線數位電視台]]（無線數位電視事業）、[[衛星廣播電視節目供應者|衛星電視台]]（衛星廣播電視節目供應者）、有線電視業者、[[中華電信MOD|MOD]]、有線電視數位頻道、網路電視六大類型。

# 無線電視台共有6家事業機構、22個頻道，將節目透過衛星傳送到地上衛星接收站，再將其[[DVB-T|DVB-T]][[數位訊號|數位訊號]]透過[[大氣電波|大氣電波]][[UHF|UHF]]頻段發送到有裝數位訊號接收盒（含天線）的用戶觀看。
# 衛星電視台則是將節目訊號發射至軌道衛星，裝有為衛星訊號接收器的用戶則透過直接接收衛星訊號，來收看電視。
# 有線電視業者：[[台灣有線電視業者列表|有線電視系統業者]]接受衛星訊號後，傳送至用戶端收看。依據《有線廣播電視法》之規定，無線電視台必須載於[[有線電視|有線電視]]上作為基本頻道提供用戶收看。
# [[中華電信MOD|中華電信MOD]]：2005年中華電信MOD開播，更提供了閱聽人收看電視節目的新途徑─[[寬頻電視|寬頻電視]]，許多舊與新的衛星電視台陸續在該平台播出。
# 有線電視數位頻道：與中華電信MOD競爭，大同小異。如：2012年9月5日[[凱擘大寬頻|凱擘大寬頻]]推出[[Super_MOD|Super MOD]]。
# 網路電視：通常指觀眾直接透過境外[[視訊網站|視訊網站]]收看或[[VPN|VPN]]管道觀看，如：[[YouTube|YouTube]]等。

== 無線電視台 ==
{{main|臺灣數位無線電視}}
台灣無線電視自2012年7月全面停止[[類比電視|類比訊號]]發送，僅提供DVB-T訊號之播出。目前台灣共有6個無線電視事業機構，合計在台灣播送22個無線數位頻道。

《[[s:有線廣播電視法|有線廣播電視法]]》第33條規定，「系統經營者應同時轉播依法設立無線電視電台之節目及廣告，不得變更其形式、內容及頻道，並應列為基本頻道」；但依NCC函釋及實務，以各家公司主頻道與[[公共電視文化事業基金會|公共電視文化事業基金會]]所屬的[[公視3台|公視3台]]、[[客家電視台|客家電視台]]及[[原住民族電視台|原住民族電視台]]為限。台灣有線電視系統已於2017年底全面數位化 ，NCC已在檢討修正該函釋。

== 衛星及寬頻電視台 ==
依據中華民國《[[衛星廣播電視法|衛星廣播電視法]]》，[[衛星廣播電視節目供應者|衛星廣播電視節目供應者]]（[[衛星電視台|衛星電視台]]），係指「自有或向[[衛星轉頻器|衛星轉頻器]]經營者租賃轉頻器或頻道，將節目或[[廣告|廣告]]經由衛星傳送給服務[[經營者|經營者]]、有線廣播電視系統經營者（包括[[有線|有線]][[電視節目|電視節目]]播送系統）或無線[[廣播電視|廣播電視]][[電台|電台]]者。」可分為「境內衛星廣播電視節目供應者」及「境外衛星廣播電視節目供應者」。

[[中華電信MOD|中華電信MOD]]開播前，閱聽人多透過有線電視系統收看衛星電視台。MOD開播後，舊有衛星電視台基於各種考量，僅少部分至該平台上架，而MOD上許多新成立的衛星電視台亦未向有線電視系統商申請上架，因此目前[[台灣|台灣]]的衛星電視台集團可劃分為：僅於有線電視系統上架者、僅於中華電信MOD上架者以及於多[[平台|平台]]上架者。

== 戶外電視媒體 ==
行動電視頻道，在[[公車|公車]]、[[捷運站|捷運站]]、[[廣場|廣場]]、[[餐飲店|餐飲店]]及[[超商|超商]]等公眾場所使用顯示器的方式提供流動多媒體資訊廣播服務，未獲政府的節目執照，不向普通家庭用戶傳送節目。
* [[BeeTV|BeeTV]]（原為[[柏泓媒體|柏泓媒體]]經營的公車及捷運廣告頻道，2012年起由「台灣摩菲爾國際」經營，全天候僅播放商業托播廣告及當日[[氣象|氣象]]）
* [[Fami_Channel|Fami Channel]]（為[[全家便利商店|全家便利商店]]裡面播放的頻道，全天候播放商業托播廣告、《TVBS創富新聞》一分鐘片段及當日氣象）
