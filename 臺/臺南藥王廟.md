'''臺南藥王廟'''是昔日[[臺灣府城|臺灣府城]][[七寺八廟|七寺八廟]]之一，也是全臺藥王廟的開基祖廟，主祀[[藥王|藥王]]大帝，位於[[台南市|台南市]][[中西區_(臺灣)|中西區]]。不過現在的建築是重建後的鋼筋水泥建築，不過仍保留了舊廟的一些木構建以及[[道光|道光]]十一年（1831年）的楹聯、香爐及[[同治|同治]]八年（1869年）的籤桶。

另外在藥王廟的右前方有祭祀一尊榕松公，據說是一顆已有三百年歷史的老樹。

== 沿革 ==
[[File:福壽我民匾.JPG|thumb]]所書之「福壽我民」匾。]]
藥王廟的創建年代一說是清[[康熙|康熙]]二十四年（1685年）；另一說是康熙五十七年（1718年），由道標千總[[姚廣|姚廣]]提議建造。當時的藥王廟是在西定坊[[北勢街|北勢街]]尾。後來因為原廟狹窄而信徒日眾，便另擇新地於乾隆二十九年（1764年）建廟。時任[[福建分巡台灣道|臺灣道]]的[[蔣允焄|蔣允焄]]亦有參與修廟事宜，故在廟中供奉有其長生祿位。當時廟深有三進，規模比今日宏大。之後清道光十八年（1838年）時有重修過，歷代亦有數次重修。

而到了民國五十八年（1969年）時，因為要建設協進街而拆毀了廟宇後方，只剩下前殿。後來於七十五年（1986年）便在前殿之地上再次重建，七十九年（1990年）安座後遂成今貌。

== 特色 ==
與一般寺廟的坐北朝南方位相比，藥王廟是少見的坐西朝東。有一說因為該廟主祀藥王大帝，面向東方有迎向生命之意。臺南有「五馬朝江一回頭」<ref>五馬指的是臺南五間坐東朝西的廟宇：小媽祖廟、廣安宮、小關帝廟、[[臺南水仙宮|水仙宮]]與開山宮。這五座廟朝向位在臺南西邊的[[臺江內海|臺江內海]]以保佑海上行船的平安。</ref>之說，那「一回頭」即是指藥王廟。

== 主祀神之爭議 ==
該廟主祀藥王大帝，廟方認為乃[[神農氏|神農氏]]，但《臺南市市區史蹟調查報告書》一書認為應為古代名醫[[扁鵲|扁鵲]]。另外還有其他說法說主神是[[韓康|韓康]]、[[孫思邈|孫思邈]]、[[韋慈藏|韋慈藏]]、[[章善俊|章善俊]]等。不過《臺南市市區史蹟調查報告書》認為因該廟例行祭典是農曆[[四月廿八日|四月廿八日]]，又[[高士奇|高士奇]]《扈從西巡日錄》中有寫說「藥王廟專祀扁鵲」，且若專祀神農應稱「藥'''皇'''廟」而非「藥'''王'''廟」，故主祀之神應為扁鵲而非神農。<ref>出自《臺南市市區史蹟調查報告書》（臺灣省文獻委員會編印，1979/06/30），164頁</ref>
