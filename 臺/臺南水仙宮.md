'''臺南水仙宮'''位於[[臺南市|臺南市]][[中西區_(臺灣)|中西區]]，是中華民國直轄市定古蹟，也是昔日[[臺灣府城|臺灣府城]][[七寺八廟|七寺八廟]]之一。廟內主祀[[水仙尊王|水仙尊王]]，即「一帝兩王二大夫」的[[大禹|大禹]]、[[寒奡|寒奡]]、[[項羽|項羽]]、[[伍子胥|伍子胥]]和[[屈原|屈原]]。該廟所在之地過去是臺灣府城[[五條港|五條港]]之一的[[五條港_(臺南)|南勢港]]，是[[府城三郊|三郊]]的總部所在地，該廟亦由當地商人合資興建。

== 沿革 ==
該廟的前身最初是在清[[康熙|康熙]]二十三年（1684年）時由該地商人集資創建供奉水仙王之廟，當時只是間茅頂竹柱篾壁的質樸廟宇。到了康熙四十年（1701年），才改建為磚壁瓦頂。而四年後，[[臺灣縣|臺灣縣]]知縣[[王仕俊|王仕俊]]將一間大店與四間小店的租金做為該廟香火來源。後來於康熙五十四年（1715年），原任臺灣水師左營遊擊的[[卓爾壇|卓爾壇]]號召泉漳商旅集資重建，並聘請潮州工匠前來主持建築工作，耗費四年方完工。完工後，臺灣府知府[[王珍|王珍]]捐獻了「著靈鰲柱」匾給該廟。
[[File:臺南水仙宮三郊旗.JPG|thumb]]
到了[[乾隆|乾隆]]六年（1741年），臺廈商旅陳逢春等人重修廟宇，並填平廟前的南勢港道以為廟埕，又購置店鋪以供香火、立碑於廟前埕覺津寺、建造後殿及獻「萬水朝宗」匾。而乾隆二十八年（1763年），北郊商人蘇萬利等人集資修廟，為郊商參與廟宇維修之始。次年，由於水仙宮廟前小港淤塞，被附近居民侵占，於是當時的臺灣府知府[[蔣允焄|蔣允焄]]下令拆除違建並增廣廟埕，事後立「水仙宮清界勒石記」石碑在廟中三川門內。之後在[[嘉慶|嘉慶]]元年（1796年），[[府城三郊|三郊]]大修水仙宮，並改廟北十三間店鋪為「三益堂」，為總理三郊事務之辦公室；同年原奉祀在後殿的福德正神移到佛頭港[[景福祠|景福祠]]合祀。之後[[道光|道光]]十三年（1833年）、[[光緒|光緒]]三年（1877年）時，亦有整修。

進入[[臺灣日治時期|日治時期]]後，由於三郊資助過[[劉永福|劉永福]]抗日，以及與中國大陸內地間的貿易之中斷，導致三郊商號不是被查抄就是倒閉，連帶影響到為三郊所管理的水仙宮。日本[[大正|大正]]五年（1916年）時曾進行整修，但在[[昭和|昭和]]十六年（1941年）時，臺南州知事[[一番瀨佳雄|一番瀨佳雄]]（{{lang|ja|一番ヶ瀨佳雄}}）宣布解散原三郊與洋行買辦合組的「台南三郊組合」，併入「台南商工會議所」，並拍賣包含水仙宮在內的三郊產業<ref>除水仙宮外，還有[[大天后宮|大天后宮]]、義民祠與溫陵媽廟等廟宇。</ref>，不過水仙宮與大天后宮因為[[臺灣總督府|總督府]]官員[[宮本延人|宮本延人]]及其他官員介入而暫緩處理。然而在[[第二次世界大戰|第二次世界大戰]]時，水仙宮被盟軍轟炸，導致中、後殿毀損。後來在中後殿蓋了防空洞<ref>據說宮本延人力爭阻止此計畫，並得總督府首肯，但當時該廟兩名執事已經先自行拆除。</ref>，而水仙宮只剩下三川門殿。而原祀奉於正殿內的五尊水仙尊王鎮殿金身，在水仙宮正殿拆除後，據傳金身被移往同屬三郊管理之大天后宮寄祀。<ref>毛紹周，〈水仙尊王信仰精神的建構：以臺南大天后宮之陪祀水神為探討中心〉《文史臺灣學報》第2期（2010.12），頁289～331。</ref>

民國四十三年（1954年），由郭炳輝發起在僅剩的三川門殿及部分中庭重建水仙宮，僅剩一進。之後民國七十四年（1985年）又整修過，約三年後完工，並於七十九年（1990年）為此建醮。現民國一百零四年(2015)正重修中，神像移奉至旁邊水仙里活動中心。

== 建築特色 ==
[[File:臺南水仙宮門.JPG|thumb]]
今天的水仙宮只剩下一進，已不復當初為七寺八廟之一時的恢弘模樣，廟宇周圍並成為了市場。不過廟內仍留有了一些古物，如[[蔣允焄|蔣允焄]]所立「水仙宮清界勒石記」石碑和日本大正六年時的「重修水仙宮碑記」。

另外由於廟中有供奉大禹的關係，所以廟門上沒有門神，而以門釘裝飾。

== 其他 ==
*臺南俗諺「上帝廟坅墘，水仙宮簷前」，指上帝廟（[[臺南北極殿|臺南北極殿]]）因位於[[鷲嶺|鷲嶺]]之上，其廟埁比水仙宮的屋簷還高。
*過去在水仙宮前立有兩根旗杆，用來指揮三郊與五條港區自組的義民團抵禦外敵。然而據說右邊的旗杆沖到了對面[[臺南藥王廟|藥王廟]]的藥王大帝，以致神像的眼睛受傷未癒。
