'''臺南市議會'''是[[中華民國|中華民國]][[臺南市|臺南市]]的最高[[立法機關|立法機關]]，分設兩座議事廳，民治議事廳位在[[新營區|新營區]]民治路的原省轄縣臺南縣議會大樓。永華議事廳位於[[安平區|安平區]]永華路的原省轄市臺南市議會大樓。本屆議長及副議長分別由退出民進黨的郭信良（{{nonp}}）及林炳利（{{nonp}}）拿下，終結民進黨長達8年的完全執政。現任議員[[民進黨|民進黨]]21席、[[國民黨|國民黨]]16席、[[時代力量|時代力量]]1席、[[台聯|台聯]]1席及無黨籍18席，總共57席。

== 簡介 ==
=== 原縣議會 ===
1946年3月24日縣議會的前身縣參議會舉行第1次參議員選舉，產生77位縣參議員，並在同年4月15日臺南縣參議會在新營鎮（現新營區）成立，首屆正副會長為[[陳華宗|陳華宗]]、楊群英先生。1950年[[臺灣省政府|臺灣省政府]]宣告各縣市實施實行[[地方自治|地方自治]]後同年1月28日舉行第1屆縣議員選舉，2月12日台南縣議會成立，4月15日舉行首屆民選縣長選舉<ref>{{cite web |url=http://tyart.tnc.gov.tw/01_about/ |title=關於總爺-縣長的話 |publisher=總爺藝文中心 |deadurl=yes |archiveurl=https://web.archive.org/web/20150106141847/http://tyart.tnc.gov.tw/01_about/ |archivedate=2015-01-06 }}</ref>，陳宗華與楊群英先生當選首屆正副議長<ref>{{cite book |author=陳延輝，蕭晉源/著 |title= 《台南縣派系興起與政黨政治的確立》 |pages = 第5頁 |publisher= 秀威資訊 | date = 2005-05 |isbn= 986726326X }}</ref>，開啟地方自治的新頁。

=== 原市議會 ===
1946年4月15日市議會的前身第一屆臺南市參議會於臺南市中正路三號成立，1950年實行[[地方自治|地方自治]]，旋於1950年10月18日正式成立臺南市第一屆議會，首屆議員人數有23席，首任正副議長為[[葉禾田|葉禾田]]、[[辛文炳|辛文炳]]，最後一任省轄台南市議會為第16屆任期為2005年至2010年。

=== 直轄市議會 ===
臺南市議會議員為公民直選，任期四年得連選連任，共劃分為18個選區（其中兩個選區為平地原住民族選區和山地原住民族選區）。
第一屆議長為賴美惠，副議長為郭信良黨籍皆為民主進步黨。
目前為第三屆直轄市議會，自2018年12月25日就任，現任議長為郭信良，副議長為林炳利皆為無黨籍。

== 編制 ==
=== 行政單位 ===
* 秘書室：負責機要業務，研究發展之管制與考核，年度工作計畫，行政單位各種會報，圖書資料蒐集與管理，議會史料之蒐集、編輯、陳列與管理及其他不屬各組室之事項。
* 議事組：負責議會內各項選舉，程序委員會業務，議事日程之編擬，會議之準備與通知，提案、決議案與審查報告之整理，大會各種議案資料、公報與議事錄之編輯與印發，期刊之編輯與發行。聽證會與跨委員會專案小組活動及人民請願案之處理。
* 行政組：負責各項費用支給事項、議員健保、議員國內、外經建考察，黨團業務，議員暨其公費助理人事資料。
* 總務組：負責出納、採購、廳舍與會議場所營繕與管理，車輛、技工、工友及駐衛警管理。
* 公共事務組：市民服務事項，姐妹會業務，來賓及外賓接待，本會組團出國考察訪問，新聞發布與媒體之聯繫，記者會、座談會與各項活動之辦理，及其他公共關係事項。
* 法規研究室：法制研究，預算分析，紀律委員會業務及大會議事法制。
* 資訊文書室：資訊系統與設備之建置與維護，資訊訓練與發展，電子作業制度制定，影音系統執行與管理，文書之收發與繕校，檔案管理及印信典守等事項。
* 人事室：辦理人事管理事項。
* 會計室：依法辦理歲計、會計及統計事項。

=== 議事編制 ===
* 程序及紀律委員會：除依臺南市議會組織自治條例第26條將議員移付懲戒案外，經由本會議員4人以上提議，8人以上之連署，對違反議事規則，妨礙議場秩序之議員亦得提請懲戒案件。
* 民政委員會：審查民政局、社會局、勞工局、秘書處、人事處、政風處、法制處、民族事務委員會、研究發展考核委員會、各區公所等所屬單位相關事項。
* 財經委員會：審查主計處、財政稅務局等所屬單位相關事項。
* 教育委員會：審查教育局、文化局、新聞及國際關係處等所屬單位相關事項。
* 保安委員會：審查警察局、消防局、交通局、觀光旅遊局等所屬單位相關事項。
* 工務委員會：審查工務局、水利局、地政局、都市發展局等所屬單位相關事項。
* 建設委員會：審查經濟發展局、農業局、衛生局、環境保護局等所屬單位相關事項。
* 法規委員會：審查本市法規等相關事項。除程序及紀律委員會外各委員會，得於每會期開始時，邀請相關局處會作業務報告，並至議會備詢。

== 議會大樓整建 ==
=== 原縣議會 ===
台南縣參議會成立之初的議事堂是借用新營鎮（現新營區）中山堂。推行地方自冶後，參議會改制為縣議會，借用縣府大禮堂行使職權。1954年1月10日，台南縣議會舊址中山路議事大樓落成，縣議員不必再寄人籬下行使問政權。1980年5月13日民治路的議會大樓興建落成縣議會由新營市中山路舊址正式遷入，八卦形的屋頂為當時少見的建築設計<ref>{{cite web|url=http://xinying-culture.tainan.gov.tw/archives/index-1.php?m2=26&id=29 | author= 吳南圖 | title=縣議會春秋 |publisher=新營文化中心}}</ref>。

=== 原市議會 ===
舊台南市議會（中西區中正路三號，現為[[臺南市議政史料館|臺南市議政史料館]]）與舊市政府（現為[[國立臺灣文學館|國立臺灣文學館]]）為鄰，原本是[[日治時代|日治時代]]（昭和10年，西元1935年）興建之[[台南州|台南州]]廳附屬建物<ref>{{cite web |url=http://www.boch.gov.tw/boch/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&caseId=DA09602001333&version=1&assetsClassifyId=1.1 |title=文化資產個案導覽原臺南州會 |publisher=文化部 |deadurl=yes |archiveurl=https://web.archive.org/web/20150106141838/http://www.boch.gov.tw/boch/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&caseId=DA09602001333&version=1&assetsClassifyId=1.1 |archivedate=2015-01-06 }}</ref>，在[[二次世界大戰|二次世界大戰]]後成為臺南市參議會會址，1950年改制為台南市議會後繼續沿用，1965年增修建議會大樓，由於舊議會大樓位處舊市區中心，有交通擁塞、停車不易與辦公廳舍老舊與議事設備不敷使用之缺點，市議會及市政府於1984年起研議遷建新市政中心。

最後決議市議會及市政府一併遷建於五期重劃區「機三十七」機關用地（安平區永華路二段二號）即為現在的[[臺南市政府|永華市政中心]]。議會新建工程分為三個階段：第一階段為1986年1月18日完成結構體發包，1993年5月19日完工並驗收。第二期土木、水電裝修工程於1994年8月發包，在1997年9月6日完工。第三期大樓室內裝修工程1998年6月20日完工，於1998年7月27-28日搬遷工作，1998年8月8日市會議廳舍落成，正式啟用。
