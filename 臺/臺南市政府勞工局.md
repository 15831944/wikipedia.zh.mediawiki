'''臺南市政府勞工局'''（簡稱'''臺南市勞工局'''），是[[臺南市政府|臺南市政府]]所屬的一級機關，本局除秘書室、人事室、會計室、政風室等幕僚單位外，分設就業促進科、勞動條件科、勞資關係科、勞安福利科、勞動檢查中心等四業務科一中心，及職訓就服中心一附屬機關，另設有關子嶺勞工育樂中心、南門勞工育樂中心，專責推動本市各項勞工政策及就業服務業務<ref>{{cite web |author1=杜忠聰 |title=臺南市政府勞工局107年共辦理6場次「青少年職涯探索就業講座」 |url=https://n.yam.com/Article/20181114524858 |publisher=大成報 |accessdate=2019-01-03}}</ref><ref>{{cite web |author1=吳政修 |title=身心障礙者不用擔心頭路 台南勞工局準備免費課程 |url=https://udn.com/news/story/7326/3105881 |publisher=聯合報 |accessdate=2018-04-25}}</ref>。
