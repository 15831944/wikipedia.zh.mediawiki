'''臺北市立永建國民小學'''(Taipei Municipal Yongjian Elementary School)，簡稱'''永建國小'''，位於臺北市文山區試院路2號，鄰近[[世新大學|世新大學]]、[[考試院|考試院]]。

== 校史 ==
永建國小創立於民國41年，原名'''考試院附設中興小學'''，由[[考試院院長|考試院院長]][[鈕永建|鈕永建]]創辦。當時為了考慮院內員工子女就學安全及方便，聯合[[考選部|考選部]]、[[銓敘部|銓敘部]]、軍醫屬及[[中華民國經濟部|經濟部]]成立[[董事會|董事會]]，以私立學校型態設校，學生人數僅63人。

因學校經費不足，經營困難，故於民國57年經董事會研議並徵得[[臺北市政府|臺北市政府]]同意後，移交市政府管轄，改為公立學校，更名「建興國民小學」。爾後於民國58年，為紀念鈕永建對學校的貢獻，更名為「永建國民小學」。<ref name="測試">{{Cite web |url=http://www.yjps.tp.edu.tw/chinese/modules/tinyd/index.php?id=1 |title=沿革 |access-date=2014-01-12 |archive-url=https://web.archive.org/web/20140112051235/http://www.yjps.tp.edu.tw/chinese/modules/tinyd/index.php?id=1 |archive-date=2014-01-12 |dead-url=yes }}</ref>

== 遷建校舍 ==
由於校地的不足，位在試院路二段的永建國小即將遷校到木柵路一段[[國家發展研究院|國家發展研究院]]舊址一部分。土地由買主[[元利建設|元利建設]]捐贈，面積約2.27公頃。土地大部分屬於小學校地，其餘為住宅用地。並附建[[臺北市停車管理工程處|臺北市停車管理工程處]]公有地下停車場工程，地下2樓，地上4樓，汽車216位，機車85位，解決國家考場、仙跡岩登山者停車問題。現在正在發包新建，預計2017年遷校。
