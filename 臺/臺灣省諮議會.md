'''臺灣省諮議會'''，簡稱'''省諮議會'''{{NoteTag|依照現行[[中華民國法律|中華民國法律]]，[[省_(中華民國)|省]]設置諮議會作為諮詢機關，由於中華民國目前僅有臺灣省設置此單位，故多直接簡稱為'''省諮議會'''。}}，是[[中華民國|中華民國]][[臺灣省|臺灣省]]的省政諮詢機關，位於[[臺中市|臺中市]][[霧峰區|霧峰區]]，其前身為「臺灣省議會」，乃臺灣實施[[臺灣省政府功能業務與組織調整|精省]]前的最高[[议会|民意機關]]。在[[立法院|立法院]]於[[1992年中華民國立法委員選舉|1992年全面改選]]前，臺灣省議會直選是[[臺灣|臺灣]]民眾可以參與到的[[立法机构|民意機關]]常態選舉中位階最高者。1998年，臺灣省議會改為現名，因[[臺灣省政府功能業務與組織調整|精省]]後[[臺灣省|臺灣省]]已非地方自治實體，其主要任務改以諮詢省政及歷史文獻保存為主。2018年7月1日，臺灣省諮議會與[[臺灣省政府|臺灣省政府]]同步於[[de_facto|實質上]]裁撤，行政組織及業務移交[[中華民國政府|中央政府]]其他機關辦理，檔案、公報、議事錄等文獻則移交[[國史館臺灣文獻館|國史館臺灣文獻館]]保存。但因憲法條文並未刪修，故名義上組織依然存在，僅保留諮議會名稱與諮議長職位<ref>[https://udn.com/news/story/6656/3223156 賴清德：108年省級機關預算全歸零]</ref><ref>[https://udn.com/news/story/11311/3224504 賴揆拍板：省級機關明年起預算歸零]</ref>。

== 沿革 ==
1945年，[[台灣日治時期|台灣日治時期]]結束，進入[[臺灣戰後時期|中華民國時期]]。

1946年4月15日，[[中華民國國民政府|中華民國國民政府]]依《[[:s:省參議員選舉條例|省參議員選舉條例]]》及《[[:s:省參議會組織條例|省參議會組織條例]]》，以[[間接選舉|間接選舉]]方式由全台各縣市參議會選出省參議員三十名。1946年5月1日，在[[臺北市|臺北市]][[南海路|南海路]][[原臺灣教育會館|原臺灣教育會館]]正式成立「'''[[臺灣省參議會|臺灣省參議會]]'''」（臺北市當時為臺灣省[[省會|省會]]）。1948年，因[[二二八事件|二二八事件]]爆發後時局不穩，中央延長參議員任期。

1951年，臺灣省調整縣市行政區域，連帶各縣市先後普選議員成立縣、市議會，使原省參議員因[[選區|選區]]全部變更而失其代表性。9月，[[行政院|行政院]]頒《臺灣省臨時省議會組織規程》及《臺灣省臨時省議會選舉罷免規程》。12月，'''臺灣省臨時省議會'''成立，[[黃朝琴|黃朝琴]]、[[林頂立|林頂立]]當選正副議長。<ref name="年表，第69頁">[[陳布雷|陳-{布}-雷]]等編著，《蔣介石先生年表》，台北：[[傳記文學_(雜誌)|傳記文學]]出版社，1978年6月1日，第69頁</ref>臺灣省參議會結束。

1954年6月2日成立第二屆臺灣省臨時省議會，1957年6月成立第三屆。當時[[台海兩岸|台海兩岸]]處於對峙局面，隨著[[臺灣省政府|臺灣省政府]]疏遷至[[中臺灣|臺灣中部]]，臺灣省臨時省議會在1958年5月移往[[臺中縣|臺中縣]][[霧峰鄉|霧峰鄉]]（今[[臺中市|臺中市]][[霧峰區|霧峰區]]）新落成的議事大樓。

1959年6月，行政院以[[行政命令|行政命令]]將臺灣省臨時省議會改稱'''臺灣省議會'''，將原臨時省議會第三屆第五次大會改稱為臺灣省議會第一屆第一次大會，將成立之法令改為《臺灣省議會暫行組織規程》。

1998年12月20日，[[李登輝政府|李登輝政府]]實施[[臺灣省政府功能業務與組織調整|精省]]，臺灣省政府成為行政院的派出機關、失去原有的[[地方分权|地方自治]]功能；臺灣省議會也同步裁撤，並於同日依《[[中華民國憲法增修條文|中華民國憲法增修條文]]》第九條第一項第二款及精省條例暨行政院令頒《臺灣省諮議會組織規程》之規定，於原址成立「'''臺灣省諮議會'''」，成為臺灣省政府的諮詢機構；省諮議會依法置諮議員至多23員，任期三年，由[[行政院院長|行政院院長]]提請總統任命之<ref>{{cite web |publisher= |title=臺灣省諮議會，歷史沿革 |url=http://www.tpa.gov.tw/big5/Content/Content.asp?cid=12&urlID=12 |accessdate=2007-09-18}}</ref>。此時期省諮議會主要工作為管理臺灣省參議會、臨時省議會、省議會時期之檔案、公報、議事錄以及地方自治事務研究發展<ref>{{cite web |publisher= |title=臺灣省諮議會，本會願景 |url = http://www.tpa.gov.tw/big5/Content/Content.asp?cid=621&urlID=621 |accessdate=2007-09-18 }}</ref>。

2018年7月1日，[[行政院|行政院]]宣佈省及機關[[預算|預算]]歸零與去任務化，不再編列員額與預算，行政組織員額移交[[立法院|立法院]]。<ref>[http://www.ncyu.edu.tw/files/bulletin/secretary/函轉臺灣省諮議會去任務化部分人員業務移撥立法院承接函_1.pdf 臺灣省諮議會去任務化部分人員業務移撥立法院承接函]</ref>議會相關檔案、公報、議事錄以及地方自治事務研究發展等任務則移交[[國史館臺灣文獻館|國史館臺灣文獻館]]。<ref>[https://www.tpa.gov.tw/opencms/news/news1/News0027.html 公告本會史料總庫、地方議會議事錄總庫明年起移由國史館臺灣文獻館管理]</ref>臺灣省諮議會原址命名為「臺灣省議會紀念園區」，交由[[臺中市文化資產處|臺中市文化資產處]]保存。除了臺灣省諮議會之外，[[立法院|立法院]]中部辦公室與[[立法院議政博物館|議政博物館]]亦位於園區內。

== 功能與影響 ==
[[台灣戒嚴|台灣戒嚴]]時期，由於[[中華民國政府|中華民國政府]]為維繫其「[[法統|法統]]」象徵，[[立法院|立法院]]、[[國民大會|國民大會]]等國會中央民意機構長期無法改選，臺灣省議會的[[代議民主制|代議]]功能因而突顯，也成為當時台灣最高代表性的民意代表機構。

許多擔任[[臺灣省議員列表|省議員]]者都在[[臺灣民主化|臺灣民主運動]]的進程中扮演過重要角色，如[[李萬居|李萬居]]、[[郭國基|郭國基]]、[[郭雨新|郭雨新]]、[[吳三連|吳三連]]、[[李源棧|李源棧]]等五名[[黨外運動|黨外]]勢力議員合稱「省議會五虎將」，又與[[許世賢|許世賢]]合稱「五龍一鳳」。<ref>蘇瑞鏘, [http://www.tpa.gov.tw/upfile/www/Pdf/%BD%D7%A4%E5%B6%B0/%B2%60%A4%C6%A5x%C6W%A5%C1%A5D/d06.pdf 台灣（臨時）省議會「五龍一鳳」對結社權的態度——以「中國地方自治研究會」為中心]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}, 臺灣省諮議會編, 2004</ref><ref>賴昭呈, [http://ndltd.ncl.edu.tw/cgi-bin/gs32/gsweb.cgi?o=dnclcdr&s=id=%22093NTNU5227021%22.%20&searchmode=basic 台灣政治反對運動:歷史與組織分析(1947-1986)], [[國立臺灣師範大學|國立臺灣師範大學]]政治學研究所博士論文, 2005</ref><ref>[[陳菊|陳菊]], [http://www.twcenter.org.tw/a01/a01_04.htm 郭雨新與台灣戰後民主運動] {{webarchive|url=https://web.archive.org/web/20111227111803/http://www.twcenter.org.tw/a01/a01_04.htm |date=2011-12-27 }}, [[吳三連|吳三連]]台灣史料基金會</ref><ref>莊雅茹, [http://ndltd.ncl.edu.tw/cgi-bin/gs32/gsweb.cgi/login?o=dnclcdr&s=id=%22091NTNU0493026%22.%20&searchmode=basic 戰後臺灣女性參政之先驅:許世賢(1908-1983)的政治生涯], [[國立臺灣師範大學|國立臺灣師範大學]]歷史研究所碩士論文, 2003</ref>

== 空間與建築 ==
=== 議會會址 ===
* 1946年5月1日，「[[臺灣省參議會|臺灣省參議會]]」於位在[[臺北市|臺北市]][[古亭區|古亭區]]（今[[臺北市|臺北市]][[中正區_(臺北市)|中正區]]）的[[台灣日治時期|台灣日治時期]]之[[原臺灣教育會館|原臺灣教育會館]]成立。1951年12月，後繼機關「臺灣省臨時省議會」繼續使用此為集會場所。
* 1958年5月1日，「臺灣省臨時省議會」在移往[[臺中縣|臺中縣]][[霧峰區|霧峰鄉]]（今[[臺中市|臺中市]][[霧峰區|霧峰區]]）新落成的議事大樓。1959年6月，後繼機關「臺灣省議會」繼續使用此為集會場所。1998年12月20日，後繼機關「臺灣省諮議會」繼續使用此為集會場所。
* 2018年7月，「臺灣省諮議會」組織移交[[中華民國政府|中央政府]]。臺灣省諮議會大樓交由[[臺中市文化資產處|臺中市文化資產處]]保存為其[[直轄市_(中華民國)|直轄市]]市定古蹟「臺灣省議會紀念園區」。

目前臺灣省諮議會無任何實體集會場所。
