'''臺灣經濟'''在[[二次大戰|二次大戰]]結束後，以[[臺灣日治時期|臺灣日治時期]]的農業與輕工業為基礎，逐漸發展起來以中小企業引導投資的小型[[資本主義|資本主義]]經濟體，輔以[[中華民國政府|中華民國政府]]主管投資大部分的公營事業體、以及眾多[[外商直接投資|外商投資]]所引導的[[資本主義|資本主義]]經濟體。至今，臺灣的多方面經濟還正在持續增長中。

在1951年至1988年间，台灣年均[[经济增长|經濟增長率]]達到9.57%，1988年至2008年间为6.1%，2008年至2015年间为2.8%<ref>{{cite web|title=经济成长率|url=http://www.stat.gov.tw/point.asp?index=1|website=中华民国统计资讯网|accessdate=2016-08-24|location=100-65台北市廣州街2號}}</ref>。[[農業|農業]]在[[GDP|GDP]]中的比重從1952年的35%下降到5%。傳統[[勞力密集工業|人力密集型工業]]已經漸漸由電子產業替代。臺灣的[[電子工業|電子工業]]對世界經濟舉足輕重，一定數量的[[電腦|電腦]]或電子零部件都是在臺灣所生產的。對外的[[貿易|貿易]]是台灣重要的經濟命脈，在過去[[美國|美國]]和[[日本|日本]]長久以來一直都是台灣的前兩大貿易夥伴，現在則是將市場與人才的重心轉移至[[中國大陸|中國大陸]]，在中國大陸有五萬多個[[台資|台資]]企業，長期居留的[[台商|台商]]及眷屬則有上百萬人之多。另有部分轉向[[歐洲|歐洲]]和[[東南亞|東南亞]]市場，其中以[[泰國|泰國]]、[[印度尼西亞|印度尼西亞]]、[[菲律賓|菲律賓]]、[[馬來西亞|馬來西亞]]和[[越南|越南]]為主要投資國家。

在2008年，隨著中華民國二次[[政黨輪替|政黨輪替]]，重新執政的[[中國國民黨|中國國民黨]]在政策上緩和兩岸關係並加強與中國大陸的經貿交流關係，簽訂[[海峽兩岸經濟合作架構協議|海峽兩岸經濟合作架構協議]]，根据中華民國財政部統計自2000年以來对中國大陸貿易出口总额增加了約25%，而對美日的出口總額則減少了約18%，現今中華人民共和國佔了中華民國贸易出口總額的40%，在2013年時啟動兩岸貨幣清算機制，金融業在新台幣與人民幣的匯兌轉換上，不再透過美元做為中間貨幣，台灣對中國大陸地區貸總額達2.7兆台幣，是高風險貸款，這顯示台灣經貿已相當依賴中國大陸。<ref>{{ cite web |title=表九：臺灣對東協六國之進出口狀況 |url=http://www.cier.edu.tw/cef/CEF-index-4.pdf |publisher=[[中華經濟研究院|中華經濟研究院]] |date=2014-03-07 |accessdate=2014-03-30 |language=zh-tw}}</ref><ref>{{Cite web|url=https://webcache.googleusercontent.com/search?q=cache:MucHtXByd9EJ:https://www.dgbas.gov.tw/public/Attachment/56395647CA7FAVC2.pdf+&cd=8&hl=zh-TW&ct=clnk&gl=tw|title=台灣地區產業結構與勞動市場變化之研究|accessdate=2017-03-14|last=yilun|work=webcache.googleusercontent.com}}</ref>

台灣是[[亚洲开发银行|亚洲开发银行]]（亚行）、[[亞太經合會|亞太經合會]]（APEC）、[[世界贸易组织|世界贸易组织]]（WTO）和東南亞國家中央銀行總裁聯合會（SEACEN）的成员，也是[[经济合作与发展组织|经济合作与发展组织]]（OECD）观察员。臺灣在2010年排名前五位的贸易伙伴分別為大陸、日本、美国、[[欧盟|欧盟]]和[[香港|香港]]。<ref>{{cite web |url=http://cus93.trade.gov.tw/fsci/ |title=中華民國進出口貿易統計 |language=zh-tw |accessdate=2014-03-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20160309193715/http://cus93.trade.gov.tw/FSCI/ |archivedate=2016-03-09 }}</ref>台灣的[[外匯儲備|外匯儲備]]目前為止是世界第五。<ref>http://www.stockq.org/economy/reserve.php</ref><ref>[https://www.imf.org/external/pubs/ft/weo/2016/02/weodata/weorept.aspx?pr.x=84&pr.y=7&sy=2011&ey=2015&scsm=1&ssd=1&sort=country&ds=.%20&br=1&c=193%2C156%2C456%2C132%2C184%2C134%2C528%2C136%2C158%2C112%2C111%2C542&s=NGDPDPC%2CLP&grp=0&a=%20 World Economic Outlook Database, October 2016] IMF 2017年2月3日閱讀。</ref>

2011年台灣人均GDP达到2萬美元，<ref>[https://www.imf.org/external/pubs/ft/weo/2016/02/weodata/weorept.aspx?pr.x=84&pr.y=7&sy=2011&ey=2015&scsm=1&ssd=1&sort=country&ds=.%20&br=1&c=193%2C156%2C456%2C132%2C184%2C134%2C528%2C136%2C158%2C112%2C111%2C542&s=NGDPDPC%2CLP&grp=0&a=%20 World Economic Outlook Database, October 2016] IMF 2018年11月7日閱讀。</ref>[[國際貨幣基金|國際貨幣基金]]（IMF）預測，2022年台灣人均將達到3萬美元。<ref>[https://www.imf.org/external/pubs/ft/weo/2018/02/weodata/weorept.aspx?pr.x=75&pr.y=17&sy=2017&ey=2023&scsm=1&ssd=1&sort=country&ds=.%20&br=1&c=193%2C124%2C156%2C138%2C935%2C132%2C134%2C184%2C136%2C528%2C158%2C112%2C542%2C111&s=NGDPDPC%2CLP&grp=0&a=%20 World Economic Outlook Database, October 2018] IMF 2018年11月7日閱讀。</ref>

台灣正積極推動FTA（[[自由貿易協定|自由貿易協定]]），目前台灣已生效的是[[巴拿馬|巴拿馬]]、[[瓜地馬拉|瓜地馬拉]]、[[尼加拉瓜|尼加拉瓜]]、[[薩爾瓦多|薩爾瓦多]]等國的FTA，台灣優先推動和[[日本|日本]]、[[美國|美國]]、[[歐盟|歐盟]]等國家和聯盟的FTA。<ref>張道宜等，《圖解簡明世界局勢：2017年版》。台北：易博士，2016</ref>

==發展过程==
{{main|台灣經濟史}}
1945年日本战败，[[中華民國國民政府|中華民國國民政府]]於10月25日接收臺灣。戰後臺灣工業[[資源|資源]]極為缺乏，日本人遭到遣返，管理階層被替代。由于遭到國共內戰牽連，政府軍事支出居高不下，幾乎無力從事投資性支出。1949年[[陳誠|陳誠]]就任臺灣省主席改革幣制，舊臺幣以四萬比一的比率兌換新臺幣。
[[File:Taiwan_yoy_%.png|缩略图]]）<ref>[http://www.stat.gov.tw/point.asp?index=1 中华民国统计咨询网]</ref>]]
1949年12月7日中華民國中央政府遷至臺北市，实行《[[臺灣省戒嚴令|臺灣省戒嚴令]]》，政治上虽然相對穩定，但經濟上卻面臨困局。其主要原因為，戰後中國[[惡性通貨膨脹|惡性通貨膨脹]]的影響，與貨幣濫發等因素。除此，隨中華民國政府自中國大陸撤退來台後的200餘萬爆增人口，也對台灣經濟造成影響。面對此沉重壓力，中華民國政府採取適度干涉的計劃經濟制度，加上[[美援|美援]]（达14.8億美元）和戰前日本人奠定的工業基礎，得以即時重建台灣經濟。農業方面，戰後政府支持的台灣小農生產與資本農業雙軌並置。一連串經濟新政，例如：發行[[新台幣|新台幣]]的幣制改革、[[三七五減租|三七五減租]]、[[公地放領|公地放領]]及[[耕者有其田|耕者有其田]]等土地改革政策，增加了農業生產。在工業方面，台灣於1950年代中期實施第一期經濟建設計劃，發展[[勞力密集工業|勞力密集輕工業]]，以替代進口商品，另外，也採取提高[[關稅|關稅]]、限制進口、外匯管制等方式，扶植以紡織業為主的民營企業。50年代，美國等先進國家將勞力密集產業外移，[[尹仲容|尹仲容]]、[[嚴家淦|嚴家淦]]等[[技術官僚|技術官僚]]為吸引外人投資，決定採取自由開放、鼓勵出口等政策，由出口帶動生產，並發動了「19點財經改革措施」、「[[獎勵投資條例|獎勵投資條例]]」、成立[[加工出口區|加工出口區]]等政策，讓台灣成為美、日兩國的加工基地並使台灣經濟活動從由[[農業社會|農業社會]]轉為[[工業社會|工業社會]]。

1970年代，尽管台灣經濟受到[[石油危機|石油危機]]等因素影響，在中華民國總統[[蔣經國|蔣經國]]推動[[十大建設|十大建設]]下，石化業與重工業打下了良好基礎。1980年代後，資金漸趨穩定的台灣陸續放寬進出口與投資限制，並於1987年實施新外匯條例實施，人民得以自由持有及運用外匯。中華民國政府發展高科技工業，1978-1988年间，中華民國年均[[经济增长|經濟增長率]]達到8%，90年代平均經濟增長率約在4%-6%，2000年之後的經濟增長率主要要由科技、石化兩大產業，以及通貨膨脹所共同促成，增長率在1-3％不等。台灣成為亞洲的主要經濟體之一，[[外匯存底|外匯存底]]世界第五<ref>http://www.stockq.org/economy/reserve.php</ref>。<ref>{{Cite web|url=http://rcted.ncu.edu.tw/digest_view.php?serial=171|title=歡迎光臨--台灣經濟發展研究中心資料網|accessdate=2017-03-14|work=rcted.ncu.edu.tw}}</ref>

== 货币 ==
自中華民國三十八年[[中央政府|中央政府]]遷台後，為了穩定金融而新生一個貨幣兌換機制演變至今。現今的法定流通貨幣為[[新臺幣|新臺幣]]，货币代码為NTD，或稱TWD。

=== 發行史 ===
*1949年以前，台灣銀行發行台幣，史稱[[舊台幣|舊台幣]]，最終被以四萬塊舊台幣兌換一元[[新台幣|新台幣]]走入歷史。
*1949年6月，實施幣改制革，發行新台幣，規定一美元可兌換5元新台幣，並實施單一匯率。
*1979年2月　外匯市場成立，廢除與美元掛鉤的[[聯繫匯率制|固定匯率]]，改採機動匯率。
*1982年8月，中央銀行改變外匯市場操作方式，匯率基礎從銀行與顧客間交易改為銀行間交易，每日匯率調整仍保持以不超過中心匯率上下2.25%為限。
*1982年11月，外匯交易中心開始逐日公佈新台幣實質有效匯率指數。新台幣仍繼續貶值。
*1985年9月，新台幣貶值至1美元兌換新台幣40元左右。由於國際美元走弱及台美貿易逆差擴大，美國開始壓迫台湾，放任新台幣升值，中央銀行乃採緩慢升值方式，因而導致海外熱錢大量流入台灣。<ref>{{cite isbn|9789571346403|noedit|ref=harv}}[http://www.books.com.tw/web/sys_serialtext/?item=0010358129 ]</ref>
*1990年5月16日　因台灣政治情況不穩因素，創新台幣一日內大貶1元的紀錄。

=== 匯率 ===
新台幣最高時為1949年6月，一元新台幣兌換20美分。最低為1959年－1961年，一元新台幣兌換2.50美分。
*1949年6月，新台幣兌美元，一元新台幣可兌換20美分，實施單一匯率。
*1950年2月，[[臺灣省政府|臺灣省政府]]公布省屬公營事業外匯處理辦法，新台幣兌美元，一元新台幣可兌換13.33美分，實施二元複式匯率。（USD1=NTD7.5）
*1950年7月，結束台灣省公布省屬公營外匯處理辦法，新台幣兌美元，一元新台幣可兌換9.71美分，回復實施單一匯率。（USD1=NTD10.3）
*1951年4月，[[臺灣銀行|臺灣銀行]]公佈外匯結匯證買賣辦法，新台幣匯率則成為所謂「官價匯率及結匯證買賣價格」並行。
*1958年11月，新台幣兌美元，一元新台幣可兌換2.77美分，行政院再度實施單一匯率。（USD1=NTD36.08）
*1959年－1961年，新台幣兌美元，一元新台幣可兌換2.50美分，新台幣匯率大致維持40元左右。（USD1=NTD40）
*1973年2月，新台幣兌美元，一元新台幣可兌換2.63美分，實施單一匯率。（USD1=NTD38）
*1981年8月，新台幣兌美元，一元新台幣可兌換2.63美分，機動匯率。（USD1=NTD38）
*1992年7月9日，新台幣兌美元，一元新台幣可兌換4.08美分，機動匯率。新台幣升值至最高價位。（USD1=NTD24.5070）
*1995年3月8日，新台幣兌美元，一元新台幣可兌換3.84美分，機動匯率。因國際美元弱勢。（USD1=NTD25.9910）
*2012年，新台幣兌美元，一元新台幣可兌換3.44美分，機動匯率。（USD1=NTD29.09）

==== 能源産业 ====
[[File:Kuosheng_Nuclear_Power_Plant-P1020606.JPG|220px]]
{{main|台灣能源}}
{{seealso|臺灣發電廠列表|台灣電力公司}}

[[台灣電力公司|台灣電力公司]]為中華民國的國營電力公司，負責台灣本島、澎湖、金門和馬祖的電力供應，是唯一的壟斷行公用事業。由台電負責所有的發電和輸電、配電，而民間發電廠所發出的電力必須交由台電統籌購入運用。
澎湖部分地區並有設立電力合作社提供偏離島地區的電力，是唯一未有台電直接介入營運的供電系統 。
截至民國99年（2010年）為止，發電裝置總容量為4,081萬瓩，電力生產來源總計有11座水力發電廠、11座火力發電廠、3座核能發電廠，而核四發電廠目前仍在興建中。

2004年，台湾已探明石油储量290万桶，已探明天然气储量382.3亿立方米，日产石油500[[桶_(單位)|桶]]。2001年日消耗石油988,000桶。2001年，台湾生产天然气75000万立方米，输出41000万立方米。消耗天然气66.4亿立方米，输入天然气63亿立方米。
