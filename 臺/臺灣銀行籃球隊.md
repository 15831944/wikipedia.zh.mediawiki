'''臺灣銀行籃球隊'''是一支[[台灣|台灣]]的業餘甲組或半職業[[男性|男子]][[籃球|籃球]]隊。1973年[[臺灣銀行|臺灣銀行]]在[[台灣省政府主席|臺灣省政府主席]][[謝東閔|謝東閔]]裁定下成立籃球隊，於1974年投入甲組籃球聯賽；並於2003成為[[超級籃球聯賽|超級籃球聯賽]]創始球隊之一。台銀籃球隊對於台灣籃球的影響功不可沒，雖隊史只有一年拿下例行賽前四名，但是細數他們歷年的球員陣容，還是培養出不少好手，當年有「紅色旋風」或「紅色閃電」稱號。

== 著名人物 ==
*[[陳信安|陳信安]]　－　短暫參與過台銀。臺灣飛人，台灣挑戰美國職籃第一人，曾效力於[[裕隆恐龍|裕隆恐龍]]、[[中國|中國]][[東莞烈豹籃球俱樂部|東莞烈豹籃球隊]]和[[璞園建築籃球隊|璞園建築/台中璞園]]，現今已退休轉到璞園建築公司。
*[[鍾枝萌|鍾枝萌]]　－　為緯來獵人隊前身--[[緯來獵人籃球隊|中廣戰神隊]]前任總教練。
*[[簡明富|簡明富]]　－　有「小巨人」的稱號，職業生涯都效力於臺灣銀行籃球隊。
*[[李雲翔|李雲翔]]　－　前東森羚羊隊總教練，曾為[[緯來體育台|緯來體育台]]球評，現為[[中華民國籃球協會|中華民國籃球協會]]副理事長。
*[[韋陳明|韋陳明]]　－　台銀行員，原台銀球員、曾任臺灣銀行籃球隊總教練自2003年至2007年，2012年又回鍋執掌兵符至2014年，2017年三度回鍋執掌兵符。打造「紅色旋風之稱」的總教練
*[[洪濬正|洪濬正]]　－　歷年<!--1977~1984嗎?-->[[威廉瓊斯盃國際籃球邀請賽|威廉瓊斯盃國際籃球邀請賽]]選手與國手，亦是達欣工程籃球隊前身中興電工的首任總教練，長年旅居美國，2014-15回國執掌臺灣銀行籃球隊總教練兵符，為台銀歷年第一位非行員教練。
*[[陳順詳|陳順詳]]　－　職業生涯曾效力於[[金門酒廠籃球隊|九太科技/東森羚羊]]，目前都效力於臺灣銀行籃球隊。

== 退休背號 ==
*'''01號：[[簡明富|簡明富]]'''
:1997年加入台銀，為正式行員、台銀SBL初始成員，亦為台銀第四季打進SBL四強之主力班底。2009年一度退休，當專任行員，2012年(第十季)復出，2013年(第十一季)配合台銀換血，不再上場，於2014年3月第十一季SBL最後一場賽事舉辦正式退休儀式，轉任教練。
