'''臺灣綜合大學系統'''（{{lang-en|'''Taiwan Comprehensive University System'''，'''TCUS'''}}），簡稱'''臺綜大'''，舊稱「臺灣T4大學聯盟」，是一個由[[國立成功大學|國立成功大學]]、[[國立中興大學|國立中興大學]]、[[國立中山大學|國立中山大學]]、[[國立中正大學|國立中正大學]]等4所[[綜合大學|綜合大學]]合作成立的「[[大學系統|大學系統]]」，[[朱經武|朱經武]][[院士|院士]]擔任首屆總校長。2016年與[[德国理工大学九校联盟|德國理工大學九校聯盟]]（TU9）締結姐妹關係<ref>{{Cite news|url=https://www.chinatimes.com/newspapers/20170506000760-260301|title=新聞分析－亞洲大學結盟屬罕見 或瞄準帶路|last=中時電子報|newspaper=中時電子報|accessdate=2018-11-01|language=zh-TW}}</ref>。

== 概要 ==
2012年，國立成功大學校長[[黃煌煇|黃煌煇]]指出：
{{Cquote|...過去[[臺聯大|臺聯大]]成效未彰顯，主因在於彼此間同質性太高，臺綜大四校中成大以[[醫學|醫學]][[機械|機械]]見長、中興以[[農業|農業]][[生技|生技]]聞名、中山則在[[海洋科學|海洋]][[生物學|生物]]有許多領先研究，中正則培育出不少[[社會科學|社會科學]]人才<ref>{{Cite web |url=http://news.secr.ncku.edu.tw/files/15-1054-99437,c9485-1.php?Lang=zh-tw |title=台綜大系統成立 推研究所聯招 |access-date=2017-07-27 |archive-url=https://web.archive.org/web/20170811143750/http://news.secr.ncku.edu.tw/files/15-1054-99437,c9485-1.php?Lang=zh-tw |archive-date=2017-08-11 |dead-url=yes }}</ref>。}}

;合作
:臺綜大4校共同籌組跨校、跨領域的研究中心。初期掛牌成立「公共政策研究中心」、「微奈米研究中心」及「海洋政策研究中心」等三個跨校整合的研究單位，並以[[生命科學|生命科學]]、[[海洋科學|海洋科學]]、[[能源|能源]]與前瞻科技（Advanced Research）為主要合作項目。
:臺綜大4校學生可到盟校擔任[[交換學生|交換學生]]，各校每學年推薦10名學生至他校，同時也接受10名交換生入學。
:臺綜大4校成立[[學生|學生]]跨校選課、[[教師|教師]]巡迴上課的「領袖[[通識|通識]]課程」。
:臺綜大4校著手打通跨校轉系管道，教育部初步同意保留10%到20%的名額，供學生轉換校系。
::2013年，臺綜大4校試辦[[運動|運動]]績優[[學生|學生]]聯合招生。
::2013年，臺綜大「國際工作圈」整合4校資源，實際以[[英語|英語]]授課（International Summer School）共同招收海內外學生來台修習專業課程，體驗[[台灣文化|臺灣文化]]、精進[[華語|華語]]學習<ref>{{cite web|url=http://summer.taiwanstudy.asia/|title=Hi, Taiwan!|website=summer.taiwanstudy.asia}}</ref>。
::2016年，臺綜大轉學生考試將進行跨校聯招。 
:由於T3聯盟的成立，原屬於國立中山大學、國立成功大學之間校際競賽的「城灣盃」也在2008年納入了國立中興大學，更名為「興城灣盃」。2011年因批准了國立中正大學的加入，更名為「[[正興城灣盃|正興城灣盃]]」。

== 評價 ==
*[[中興大學|中興大學]]以[[農學|農學]]立校，是[[中台灣|中台灣]]最大的綜合大學。
*[[成功大學|成功大學]]以[[工學|工學]]立校，是[[南台灣|南台灣]]最大的綜合大學。
*[[國立中山大學|中山大學]]以[[幕僚部門|智庫學者]]、[[商學|商學]]、[[海洋学|海洋學科]]立校，是[[南台灣|南台灣]]最初的國立綜合大學<REF>1980年創立時，成大尚為「省立大學」。</REF>，[[中山大學_(消歧義)|中山大學系列]]之一。
*[[中正大學|中正大學]]以[[學科列表|基礎學科]]立校，是台灣最初的郊區[[大學城|大學城]]大學<REF>http://www.lib.ccu.edu.tw/history/history_CCU.pdf</REF>。

== 沿革 ==
*2008年，[[國立中山大學|國立中山大學]]發起，與[[國立成功大學|國立成功大學]]、[[國立中興大學|國立中興大學]]共組'''臺灣T3大學聯盟'''。原計畫邀請[[高雄醫學大學|高雄醫學大學]]加入，但未能實現。「T3」為「TOP3」之簡稱，意謂[[中臺灣|中臺灣]]、[[南臺灣|南臺灣]]的頂尖[[國立大學|國立]][[綜合大學|綜合大學]]。
*2010年，{{Fact|經三校開會後以2 : 1(中山大學、中興大學同意，成功大學反對)的比數|time=2018-11-08T18:56:26+00:00}}批准[[國立中正大學|國立中正大學]]加入，因此更名為'''臺灣T4大學聯盟'''。
*2011年，[[中華民國教育部|中華民國教育部]]核定成立'''臺灣綜合大學系統'''。
*2012年11月5日，[[香港科技大學|香港科技大學]]前校長、[[中央研究院院士|中央研究院院士]][[朱經武|朱經武]]（[[成大|成大]]校友）就任首任系統校長。

=== 歷史淵源 ===

;中興大學－成功大學
:興大、成大2校皆創建於[[台灣日治時期|台灣日治時期]]。[[臺灣戰後時期|中華民國接管臺灣後]]，「中興」、「成功」成為校名合稱「中興成功」的「省立大學」。
:興大、成大的第4任校長皆為[[羅雲平|羅雲平]]博士，也因此2校校園都有一棟「雲平（大）樓」。

;中興大學－中山大學
:[[台灣省議會|台灣省議會]]曾於1971年提議「省立中興大學」升格改制為「國立中山大學」，但未成功。

;中山大學－中正大學
:中山、中正2校皆創建於[[台灣1980年代|台灣1980年代]]，皆蘊含「平衡南北[[高等教育|高等教育]]」的政治使命，也皆以[[中華民國總統|中華民國總統]]名諱定名。
:[[嘉義縣|嘉義縣]]民代曾戮力爭取'''中山'''在該地復校。12年後，'''中正'''成為嘉義縣民雄鄉第一所大學。
