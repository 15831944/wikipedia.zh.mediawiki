'''臺北市立美術館'''，簡稱'''北美館'''，是一座位於[[台灣|台灣]][[臺北市|臺北市]]的[[美術館|美術館]]，成立於1983年12月24日<ref>{{Cite web |url=http://taiwan.net.tw/m1.aspx?sNo=0001090&id=9989 |title=臺北市立美術館 |accessdate=2014-03-02|publisher=中華民國交通部觀光局}}</ref>，典藏及展示的作品類型相當廣泛，但主要以台灣的當代藝術為主。

== 簡介 ==
[[File:Taipei_Art_Park_and_Taipei_Fine_Arts_Museum_View_From_Overpass_in_2010_TIFE.jpg|thumb]]
[[File:Interior-Taipei-fine-arts-museum.jpg|thumb]]

本館舊址曾為[[美軍協防台灣司令部|美軍協-{防台}-灣司令部]]（USTDC）總部所在地<ref>{{cite news|title=《找尋舊時代的回憶》：在台美軍的故事(二)|url=http://www.formosamedia.com.tw/?page_id=1326|publisher=玉山周報|date=2009-08-17|accessdate=2014-03-02|deadurl=yes|archiveurl=https://web.archive.org/web/20100627051557/http://www.formosamedia.com.tw/?page_id=1326|archivedate=2010-06-27}}</ref>。是台灣第一座現代美術館，館區約有6,200坪<ref>{{Cite web |url=http://www.icid.ncku.edu.tw/cie/2007b-Fall/%E5%8F%B0%E5%8C%97%E5%B8%82%E7%AB%8B%E7%BE%8E%E8%A1%93%E9%A4%A8-%E5%BC%B5%E5%A9%B7%E5%AA%9B.pdf|title= 台北市立美術館參訪報告|accessdate=2014-03-02|publisher=國立成功大學創意產業設計研究所}}</ref>，外觀屬於理性的當代建築，以疊狀的白色方型為主。由建築師[[高而潘|高而潘]]依據中國傳統的[[四合院|四合院]]概念所設計，仿擬殷商時期的斗拱粗獷的表現。建築師以管狀預力管壁系統與大廳預力格子梁，呈現現代化東方旨趣的空間語彙。其管徑7.5公尺是參考科比意的無限成長（Endless growth）概念的美術館。共分為六層，地上三層，地下三層，主要的展示空間集中在地上三樓與地下一樓。目前在主建築旁亦有雕塑花園。

== 位置 ==
該館位於臺灣臺北市[[中山區_(臺北市)|中山區]][[花博公園|花博公園]]美術園區內<ref>{{Cite web|url=http://pkl.taipei.gov.tw/ct.asp?xItem=9833937&CtNode=46925&mp=106011|title=花博公園美術園區|accessdate=2014-03-02|publisher=臺北市政府工務局公園路燈工程管理處|date=2013-01-02|deadurl=yes|archiveurl=https://web.archive.org/web/20141006085327/http://pkl.taipei.gov.tw/ct.asp?xItem=9833937&CtNode=46925&mp=106011|archivedate=2014-10-06}}</ref>，北邊隔美術廣場有[[臺北故事館|臺北故事館]]與故事茶坊，再北邊為[[大佳河濱公園|大佳河濱公園]]與[[基隆河|基隆河]]，西邊隔中山北路與[[圓山公園|圓山園區]]花海區相望，東邊為停車場與[[計程車|計程車]]休息站，南面為寰宇庭園區，在[[風味館|風味館]]的北邊。

== 組織架構 ==
[[File:Lin_Ping_P1250334.jpg|thumb]]

北美館隸屬於臺北市政府文化局，為二級機關，置館長、副館長各1人，下設有[[研究|研究]]組、[[展覽|展覽]]組、[[典藏|典藏]]組、[[教育|教育]][[推廣|推廣]]組、[[總務|總務]]組、[[會計|會計]]室、[[人事|人事]]室、[[政風|政風]]室，共五個組三個室，預算員額144人，組織架構如下：
 
館長及副館長室包含館長、副館長、組員及技工共5人。各單位職掌（預算員額數）如下：
 
'''研究組（9人）'''
* 掌理美術學術[[研究|研究]][[出版|出版]]、專題研究、本館及美術相關[[文獻|文獻]]資料彙集與整理、學術[[論壇|論壇]]規劃及充實並管理[[圖書|圖書]]相關資料。
* 其下設有美術圖書室，對外提供藝術專業書籍閱覽服務。
* 學術研究出版、館務發展政策擬訂、專案研究。
 
'''展覽組（24人）'''
* 掌理國、內外美術作品之[[展覽|展覽]]、展覽[[專輯|專輯]]出版、[[雙年展|雙年展]]事務、美術競賽及申請展、國際美術機構之交流業務等事項。
* 其下設有美工設計室、攝影室、佈展小組。
* 國內當代、現代藝術展覽策展、國際展覽引介交流。
* 臺北雙年展、[[威尼斯雙年展|威尼斯雙年展]]臺灣館策劃、重要國際展覽專案計劃。
 
'''典藏組（7人）'''
* 掌理美術作品之徵集蒐藏、分類登記、整理[[考據|考據]]、[[裱裝|裱裝]]修護等事項。
* 其下設有作品維護室、典藏[[庫房|庫房]]。
* 美術作品之徵集蒐藏、分類登記、整理考據、裱裝修護。
 
'''教育推廣組（27人）'''
* 掌理[[美術教育|美術教育]]之[[推廣|推廣]]、國際學術交流、[[節慶|節慶]]與周末夜間活動、[[義工|義工]]培訓等事項。
* 其下設有義工隊、視聽室。
* 美術教育推廣、夜間開放活動規劃執行、國際學術研討會策辦。
 
'''總務組（63人）'''
* 掌理[[文書|文書]]、[[事務|事務]]、[[印信|印信]]、[[出納|出納]]、[[財產管理|財產管理]]、管制[[考核|考核]]、機電管理、[[警衛|警衛]]及其他不屬各組之事項。
* 其下設有機電室、總機室、駐衛警察隊。
* 行政文書、採購事務、出納保管、財產管理、管制考核、機電管理、警衛安全。

'''會計室（4人）'''
* 依法辦理歲計、[[會計|會計]]事務並兼辦[[統計|統計]]事務。
 
'''人事室（4人）'''
* 依法辦理[[人事|人事]]管理事項。
 
'''政風室（1人）'''
* 依法辦理[[政風|政風]]事項。

== 特色 ==
北美館截至2016年12月底為止，有4,923件典藏作品，<ref>{{Cite web |url=https://www.tfam.museum/Common/editor.aspx?id=11&ddlLang=zh-tw |title=典藏>關於典藏 |date=2016-12-31 |publisher=臺北市立美術館}}</ref>多為1940年代之後的臺灣當代藝術作品，共分為13類，包括[[雕塑|雕塑]]、[[版畫|版畫]]、[[油畫|油畫]]、[[素描|素描]]等等。此外，北美館在1998年規劃了「典藏常設區」，設在二樓。1980年代北美館的展覽以當代[[水墨畫|水墨畫]]為主軸，1990年代之後，[[版畫|版畫]]與[[媒體藝術|媒體藝術]]成了主要的展覽重心，2000年代則多有[[數位電子藝術|數位電子藝術]]與當代經典[[產品設計|產品設計]]的展覽。此外，北美館亦是臺灣藝術界主辦世界性藝術展覽的先驅，在1980年代就陸續舉辦了相當多的國際展覽。1998年開始了「[[台北雙年展|臺北雙年展]]」，第一屆「臺北雙年展：欲望場域」邀請國際策展人[[南條史生|南條史生]]（Fumio Nanjo）來舉辦。

=== 臺灣精品館 ===
館門朝北，於[[2010年臺北國際花卉博覽會|2010年臺北國際花卉博覽會]]時期規劃了一億四千萬元開譬了南面的新門─北美館南進門，目前做為臺灣精品館使用（已於2017年10月11日結束營業）。<ref>[http://www.taiwanexcellence.org/index.php/about/docs/index/1000000523/1/1%20%E5%8F%B0%E7%81%A3%E7%B2%BE%E5%93%81%E9%A4%A8 臺灣精品館]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url=http://www.taipei-expopark.tw/archive.aspx?uid=107 |title=花博公園- 台灣精品館 |accessdate=2014-03-02 |publisher=財團法人臺北市會展產業發展基金會 |date=2011-07-26 |archiveurl=https://web.archive.org/web/20140305183917/http://www.taipei-expopark.tw/archive.aspx?uid=107 |archivedate=2014-03-05 |deadurl=yes }}</ref>

=== 建築特色 ===
具有中國木結構建築之接榫意象，並且富有結構表現主義的特色，即建築結構本身就具有美感，無須額外裝飾即能表現出建築的結構美。建築的另一項特色為管狀結構，呈現向外延伸的狀態，隱含了[[日本|日本]][[代謝建築|代謝建築]]的意義，賦予了建築生命。

=== 其它館舍 ===
原本館旁有興建臺北水墨館之計劃，後因故未能施行。後利用[[臺北市政府舊廈|臺北市政府舊廈]]，做為臺北市第二美術館，即今日的[[臺北當代藝術館|臺北當代藝術館]]。

== 交通 ==
===臺北捷運===
自捷運{{淡水線}}[[圓山站|圓山站]]下車，1號出口出站後，步行穿越花博公園圓山園區，再左轉行進至[[中山北路|中山北路]]三段，於右前方即可抵達美術館。

===公車===
搭乘21、42、203、208、218、220、247、260、277、279、287、310、612、677、1717、2022、9006、紅2、中山幹線至「臺北市立美術館」站下車。

* 公車站牌位置：美術館前（往北，[[士林|士林]][[大直|大直]]方向）、花博公園圓山園區小徑與中山北路三段交叉口旁（往南，[[臺北車站|臺北車站]]方向）。
* 203、277、279、612僅往北方向設站。
* 紅2、21、220為無障礙公車。
