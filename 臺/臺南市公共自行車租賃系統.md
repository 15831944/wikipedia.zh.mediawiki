'''臺南市公共自行車租賃系統'''（英文：'''T-Bike'''）是[[臺灣|臺灣]][[臺南市|臺南市]]由[[臺南市政府|臺南市政府]]交通局負責建置的[[公共自行車|公共自行車]]租賃系統，2016年8月8日正式營運<ref>{{cite web |url=http://tbike.tainan.gov.tw/Portal/News/Detail/12 |title=臺南市T-Bike 8月8日正式上路 |date=2016-08-08 |publisher=臺南市政府T-Bike官網}}</ref>。截至2019年3月29日，T-Bike已在臺南市建置61個站點。
[[File:TBike_Tainan_2016-08-10.jpg|thumb]]

== 歷史 ==
*2014年，臺南市交通局開始規劃臺南市公共自行車租賃系統。<ref>{{cite web |url=http://udn.com/news/story/7326/1318793-%E9%A8%8ET-bike%E9%81%8A%E5%8F%B0%E5%8D%97-%E6%98%8E%E5%B9%B4%E8%A8%AD10%E7%AB%99 |title=騎T-bike遊台南 明年設10站] |date=2015-11-17 |publisher=聯合報}}</ref>
*2016年8月8日，上線啟用，首波在台南都會公園、奇美博物館文化園區及南科等地設10站。<ref>{{cite web |url=http://news.ltn.com.tw/news/life/breakingnews/1784603 |title=南市T-Bike 八八節上線 |date=2016-08-04 |publisher=自由時報}}</ref><ref>{{cite web |url=http://news.ltn.com.tw/news/life/breakingnews/1787165 |title=台南市T-Bike 8日上路 賴清德親自來體驗 |date=2016-08-06 |publisher=自由時報}}</ref>
*2017年1月26日，站點擴增至中西區、南區、北區、安平區及東區等市轄區，共12個地點開始提供服務。<ref>{{cite web |url=http://news.ltn.com.tw/news/life/breakingnews/1959361 |title=低碳慢遊府城 南市T-Bike除夕啟用 |date=2017-01-26 |publisher=自由時報}}</ref>
*2017年2月17日，[[嘉南藥理大學|嘉南藥理大學]]捐贈台南市政府T-Bike，當日嘉南藥理大學站啟用 。<ref>{{cite web |url=http://m.ltn.com.tw/news/life/breakingnews/1978285 |title=台南首座民間捐贈T-Bike租賃站 嘉藥大學站落成 |date=2017-02-17 |publisher=自由時報}}</ref>
*2017年4月8日，新增10站租賃站，包括[[臺南車站|臺南車站]]、公路客運與國道客運的上下車處，方便外地旅客轉乘使用。<ref>{{cite web |url=https://udn.com/news/story/3/2389829?from=udn-relatednews_ch1 |title=台南T-Bike又添10站！今年底前免費騎半小時 |date=2017-04-07 |publisher=聯合報}}</ref>
*2017年6月20日，再新增19個租賃站，包含臺南火車站後站、花園夜市、林默娘公園等，今年52站已全數完成。<ref>{{Cite news|url=http://www.taiwanhot.net/?p=466053|title=台南市52站T-Bike建置完成　近日通電啟用 {{!}} 台灣好新聞 TaiwanHot.net|newspaper=台灣好新聞 TaiwanHot.net|accessdate=2017-06-20|language=zh-TW}}</ref>
*2017年9月22日，[[崑山科技大學|崑山科技大學]]捐贈台南市政府T-Bike，當日崑山科大站啟用，亦為永康區第一個站點。<ref>{{cite web |url=https://udn.com/news/story/7326/2716510 |title=台南T-Bike租賃站擴點 市府盼企業捐贈 |date=2017-09-22 |publisher=聯合報}}</ref>
*2018年6月22日，台南市交通局為台南市公共自行車系統投保第三人責任險。<ref>{{cite web |url=http://news.ltn.com.tw/news/life/breakingnews/2466482 |title=台南T-Bike也有「第3人責任險」 今天起上路 |date=2018-06-22 |publisher=自由時報}}</ref>
*2018年6月30日，再新增海安正興及高鐵台南2處租賃站。
*2018年8月1日，台南市交通局再為台南市公共自行車系統投保個人傷害險，成為全國第一雙保險之公共自行車系統。<ref>{{cite web |url=http://news.ltn.com.tw/news/life/breakingnews/2505909 |title=台南T-Bike今起投保個人傷害險 成全國第一雙保險公共自行車系統 |date=2018-08-01 |publisher=自由時報}}</ref>
*2019年2月1日：新增5處租賃站，其中包含新市火車站及善化火車站。
*2019年3月29日：新增交通大學(臺南校區)租賃站。

==租賃方式==
[[File:T-Bike車體.jpg|thumb]]
*採用無人管理自助式服務，可甲地租乙地還，各站皆可使用[[一卡通_(臺灣)|一卡通]]（含市民卡）、[[悠遊卡|悠遊卡]]、[[信用卡|信用卡]]（[[VISA|VISA]]／[[萬事達卡|Master Card]]／[[JCB|JCB]]）進行租賃。
*一張卡片最多可租借1輛腳踏車。
*使用一卡通（含[[臺南市市民卡|臺南市市民卡]]）、悠遊卡的民眾須先於[http://tbike.tainan.gov.tw/Portal 公共自行車資訊網]、各租賃站中控台或官方APP辦理會員註冊才可使用；信用卡則無須記名作業。
*使用一卡通、悠遊卡租借時，卡片儲值餘額需新台幣10元以上才可租借使用。
*24小時客服專線：0800-527527、06-2436668

==收費方式==
*信用卡及一卡通、悠遊卡：每30分鐘10元，每日最高100元
*一卡通臺南市民卡：每30分鐘5元，每日最高100元
*2019年12月31日前，持一卡通(含臺南市民卡)、悠遊卡及信用卡 前30分鐘免費
*租賃費用按租賃時間計收，不足30分鐘，以30分鐘計

== 租賃站位置 ==
{{main|臺南市公共自行車租賃系統站點列表}}
截至2019年3月29日，T-Bike已經在臺南市建置61個站點，設點的行政區包括[[仁德區|仁德區]]（7個）、[[新市區|新市區]]（4個）、[[善化區|善化區]]（5個）、[[中西區_(臺南市)|中西區]]（9個）、[[南區_(臺南市)|南區]]（8個）、[[北區_(臺南市)|北區]]（8個）、[[東區_(臺南市)|東區]]（8個）、[[安平區|安平區]]（7個）、[[安南區|安南區]]（2個）、[[永康區|永康區]]（1個）、[[歸仁區|歸仁區]]（2個）<ref>{{cite web |url=http://tbike.tainan.gov.tw/Portal/Station/List  |title=租賃站資訊}}</ref>。租賃站依鎖孔數量分為2種，大站有32個鎖孔、小站有24個鎖孔，皆設有中控台(Kiosk)1座可自助操作借還車、加入會員、租還車記錄查詢、延長租車及租賃站查詢等服務，除中文外並提供英文及日文操作介面。服務中心設在臺南市仁德區文華路二段上的[[臺南都會公園|臺南都會公園]]管理中心，提供會員申請等協助。
