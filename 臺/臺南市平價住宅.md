'''臺南市平價住宅'''，為[[臺南市政府|台南市政府]]為安置低收入戶，所建的連棟住宅，土地及建物所有權皆屬於市政府。平價住宅共計兩處，分別為[[安南區|安南區]]的'''大同平價住宅'''，和[[東區_(臺南市)|東區]]的'''大林平價住宅'''，興建於1967年至1969年間<ref name=":0">{{Cite web|url=https://news.ltn.com.tw/news/life/breakingnews/2310539|title=台南「貧民窟」將走入歷史 月底清除完畢|accessdate=|author=|date=2018-01-12|publisher=自由時報}}</ref>。由於居住環境及衛生不佳、被當地人稱為貧民窟<ref name=":0" />。

[[921大地震|921大地震]]後，這兩處平價住宅經安全鑑定，建議拆除重建，以及居住環境不佳等因素，台南市政府於1999年下令禁止遷入新戶籍，並計畫將其拆除。但因為轉售、轉租、佔用情形嚴重，安置補償費用談不成，且當地民眾找市議員出面<ref>{{Cite web|url=https://news.ltn.com.tw/news/local/paper/1062433|title=大林、大同危險住宅 賴清德：明年拆除|accessdate=|author=|date=2016-12-16|publisher=}}</ref>，造成拆遷計畫一直延宕。[[2016年高雄美濃地震|2016年高雄美濃地震]]後，拆遷計畫才加速進行，並由社會局輔導原住戶搬遷。兩者於2017年開始進行拆除作業，目前均已拆除，並作為公共設施。

== 大林平價住宅 ==
'''大林平價住宅'''，又稱'''忠孝新村'''、'''大林平價國宅'''、'''大林國宅'''，為台南市首個平價住宅。位於台南市東區[[大福里_(臺南市)|大福里]]立德六、七、八路，為三層樓的連棟建築，工程分為兩期，共建有226戶。1968年3月16日落成，時任[[臺灣省政府主席|台灣省主席]][[黃杰_(將軍)|黃杰]]到場剪綵<ref>{{Cite web|url=http://catalog.digitalarchives.tw/item/00/31/a0/93.html|title=黃杰主席主持台南市平價住宅落成典禮|accessdate=|author=|date=1968-03-16|publisher=台灣電影文化公司}}</ref>。當初一、二樓提供低收入戶資格者免費入住，三樓則是以五萬元購買<ref>{{Cite web|url=http://www.cdns.com.tw/news.php?n_id=1&nc_id=150479|title=拆大林平價宅 齊頭補助不公|accessdate=|author=|date=2017-03-14|publisher=}}</ref>。921大地震後，有6戶被列為危樓而拆除。2018年拆遷前，約有400人居住於此。

大林平價住宅於2018年1月20日拆除，拆除後規劃為「立德臨時停車場」，可支援未來東側[[臺南市區鐵路地下化計畫|台南鐵路地下化]]所增設[[林森車站|林森車站]]的停車需求。

== 大同平價住宅 ==
'''大同平價住宅'''，又稱'''大同新村'''，位於台南市安南區[[塭南里|塭南里]]安昌街，為二層樓的加強磚造連棟建築，共有344戶。每戶室內空間約8坪，陽台及樓梯等空間約2坪<ref>{{Cite web|url=https://land.tainan.gov.tw/FileDownLoad/FileUploadList/269.pdf|title=土地信託制度運用於平價住宅社區更新管理之可行性研究|accessdate=|author=陳定國|date=2006-08-11|publisher=台南市安南地政事務所}}</ref>。1971年5月29日落成，時任台灣省主席[[陳大慶|陳大慶]]到場剪綵<ref>{{Cite web|url=https://www.ttv.com.tw/news/tdcm/viewnews.asp?news=0062814|title=台南市平價住宅落成剪綵|accessdate=|author=台視新聞|date=1971-05-29|publisher=}}</ref>。

大同平價住宅於2017年11月25日開始拆除作業<ref>{{Cite web|url=https://www.youtube.com/watch?v=wUyTxU1-Goc|title=南市大同平價住宅住戶全數搬離 開始拆除|accessdate=|author=大代誌臺南|date=2017-11-26|publisher=Youtube}}</ref>，拆除後規劃為公園。
