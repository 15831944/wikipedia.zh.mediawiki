'''鈴木Cappuccino'''（{{lang-ja|スズキ・カプチーノ}}）是1990年代[[日本|日本]][[鈴木公司|鈴木公司]]開發製造的[[輕型車|輕型]][[雙座敞篷車|雙座敞篷車]]。車名來自[[義大利|義大利]]泡沫[[咖啡|咖啡]]「[[卡布奇諾|卡布奇諾]]」，以顯現小巧精緻敞篷車的形象<ref>{{ja}}[http://www.suzuki.co.jp/about/faq/dom4_name.html スズキ株式会社：スズキ四輪車 車名の由来]。</ref>。

== 背景 ==
在[[日本泡沫經濟|日本泡沫經濟]]時期國民普遍富裕，於是各車廠紛紛開發外觀帥氣又兼具駕駛樂趣的車款，以吸引消費者掏腰包購買。此款車與同時期發售的[[Autozam_AZ-1|Autozam AZ-1]]、[[本田Beat|本田Beat]]等被[[消費者|消費者]]截取車名字首，合稱為「{{lang|ja|平成ABCトリオ}}」。

== 概要與歷史 ==
原廠在1987年[[東京車展|東京車展]]上發表鈴木Cappuccino[[概念車|概念車]]，兩年後在同一個展覽會中公開量產車，1991年10月開始生產，同年11月正式上市銷售（原廠代號EA11R）。此部車最大的特色就是以可隱藏的後車窗搭配三片收納式車頂篷，可分成硬頂全閉、[[敞篷車|全開篷]]、{{link-en|半開篷車|Targa top|半開篷}}、{{link-en|T字形車頂|T-top}}等四種形態。同級競爭對手中唯一採用[[前置後驅|前置後驅]]的佈局，且車身前後配重為51：49。動力來源為657c.c.直列三缸DOHC [[鈴木F族引擎#F6A型（DOHC_4汽門渦輪增壓附中冷器）|F6A型]][[渦輪增壓器|渦輪增壓]]引擎（附中冷器），最大馬力64hp / 6,500rpm、扭力峰值8.7kg·m / 4,000rpm，搭配五速手排變速箱。由於引擎採縱置式，引擎室左右兩側具有足夠空間容納輕型車首創的四輪[[雙橫臂懸架|雙A臂懸吊系統]]，搭配四輪碟煞制動系統。雖然車身小巧，標準配備卻包含空調系統、電動窗、置物空間等，另可加價選購ABS[[防鎖死煞車系統|防鎖死煞車系統]]、托森式[[限滑差速器|限滑差速器]]、雙前座SRS[[安全氣囊|安全氣囊]]等配備。

同為右駕的緣故，香港代理商曾引入Cappuccino，1992年10月亦現身於[[英國國際車展|英國國際車展]]，1993年10月正式外銷[[英國|英國]]市場，並配合當地法律規範變更方向燈、後霧燈、最高[[時速|時速]]140lm/hr限速功能等。至於外銷[[德國|德國]]的版本，雖然該國靠右行駛，仍維持右駕設定。其他[[歐洲|歐洲]]國家如[[法國|法國]]、[[荷蘭|荷蘭]]等，也可見到鈴木Cappuccino的蹤影。

1995年5月實施小改款（原廠代號EA21R），改以[[鋁合金|鋁合金]]製成的658c.c.直列三缸DOHC [[鈴木K族引擎#K6A型|K6A型]][[渦輪增壓器|渦輪增壓]]引擎（附中冷器），搭配16[[位元|位元]][[发动机控制器|行車電腦]]，最大馬力沒有變動，但峰值扭力變成10.5kg·m / 3,500rpm。因K6A型引擎而減重10[[公斤|公斤]]，且新增三速自排變速箱。不過由於[[欧洲联盟委员会|歐盟執委會]]頒布更嚴格的汽車廢氣排放標準，原廠評估銷售規模不足以修改K6A型引擎，故放棄外銷機會。

受到[[日本泡沫经济|泡沫經濟]]崩解的影響，此類特殊車輛市場出現低迷；加上輕型車法規變更、[[运动型多用途车|SUV]]風潮興起等諸多因素，遂於1998年10月停產。在7年之間，總共生產26,583輛。
