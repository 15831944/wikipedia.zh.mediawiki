'''鈴鹿線'''（{{jpn|j=鈴鹿線|hg=すずかせん|rm=Suzuka sen|f=y}}）是自日本[[三重縣|三重縣]][[鈴鹿市|鈴鹿市]]的{{link-ja|伊勢若松車站|伊勢若松駅}}至{{link-ja|平田町車站|平田町駅}}的[[近畿日本鐵道|近畿日本鐵道]]（近鐵）的鐵路線。這條路線通過鈴鹿市市區，是前往平田町車站附近工廠的通勤路線。

== 路線資料 ==
*路線距離（{{link-ja|營業距離|営業キロ}}）：8.2公里
*[[軌距|軌距]]：1435毫米
*站數：5個（包括起終點站）
*複線路段：沒有（全線単線）
*電氣化路段：全線電氣化（直流1500V）
*[[閉塞_(鐵路)|閉塞方式]]：自動閉塞式
*最高速度：80公里/小時
全線由名古屋統括部（舊名古屋營業局）管轄。

== 運行形態 ==
在早晚每小時運行三班至四班列車。白天、清晨和深夜則每小時運行兩班列車。
== 車站列表 ==
*所有車站都位於[[三重縣|三重縣]][[鈴鹿市|鈴鹿市]]。
*普通列車省略。急行、普通列車都鈴鹿線內各站停車。急行只限往平田町運行。
*急行的名古屋線內的停靠站參見[[名古屋線|名古屋線]]。
