'''鈴木真夕'''（{{lang-ja|鈴木真夕}}，{{bd|1994年|3月16日|||}}），[[千葉縣|千葉縣]]出身，[[日本|日本]]的[[寫真偶像|寫真偶像]]、[[AV女優|AV女優]]。曾是[[E-BODY|E-BODY]]的專屬女優<ref name="e-body">{{Cite web |url=http://www.av-e-body.com/actress/detail/50050/1/ |title=公式プロフィール |work=AV女優：鈴木真夕 |publisher=[[E-BODY|E-BODY]] |accessdate=2016-08-07}}</ref>。

== 簡歷 ==
2011年10月，在「純粋少女〜17歳、初めてのTバック〜」中[[初登場|初登場]]<ref name="zakzak">{{Cite web |url=http://www.zakzak.co.jp/gravure/idol/news/20111121/idl1111211233000-n1.htm |title=17歳・鈴木真夕、アレをハメられちゃった〜! |publisher=ZAKZAK |date=2011-11-21 |accessdate=2015-08-09}}</ref><ref>{{cite web|url=http://www.playno1.com/article-10850-1.html|title=E-Body最強大物、出陣! |publisher=AVNo.1-PLAYNO.1|date=2015-04-13}}</ref>。

2014年11月，在[[週刊Playboy|週刊Playboy]]上刊登了個人的裸體寫真<ref>{{Cite web |url=http://wpb.shueisha.co.jp/2014/11/17/39161/ |title=週刊プレイボーイNo.48 |publisher=[[週刊Playboy|週刊プレイボーイ]]|date=2014-11-17 |accessdate=2016-10-15}}</ref>。

2015年5月，成為[[E-BODY|E-BODY]]的專屬女優，在「現役グラビアアイドルAV解禁」中AV初登場<ref>{{Cite web |url=http://sankei-sports.com/adult/sugao/20150527.html |title=現役グラドル解禁気持ちＥ～ＢＯＤＹ　鈴木真夕 |publisher=サンケイスポーツ |date=2015-05-27 |accessdate=2016-08-07 |archive-url=https://web.archive.org/web/20160923152338/http://sankei-sports.com/adult/sugao/20150527.html |archive-date=2016-09-23 |dead-url=yes }}</ref>。

2016年，被日本成人網站[[DMM.com|DMM.com]]選為AV界「最優秀的新人AV女優」第5名<ref>{{Cite web |url=http://www.dmm.co.jp/adultaward/vominate_html/=/navi=none/ |title=DMM.R18アダルトアワード2016 最優秀新人女優賞ノミネート女優発表！！ |publisher=[[DMM.com|DMM.com]] |date=2016-02-12 |accessdate=2016-10-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304111703/http://www.dmm.co.jp/adultaward/vominate_html/=/navi=none/ |archivedate=2016-03-04 }}</ref><ref>{{Cite web |url=http://m.nownews.com/news/2004313 |title=熱騰騰！2016最優秀的新人AV女優名單出爐 |publisher=[[今日新聞網|今日新聞網]] |date=2016-02-23 |accessdate=2016-10-15}}</ref>。

興趣是[[購物|購物]]、做塑膠模型<ref name="e-body"/>。最喜歡的作品是[[機動戰士GUNDAM_SEED|機動戰士GUNDAM SEED]]。專長是彈[[鋼琴|鋼琴]]、[[烹飪|烹飪]]<ref name="e-body"/>。
