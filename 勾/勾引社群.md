'''勾引社群'''（{{lang-en|'''Pickup artist'''}}  简称：'''PUA'''），也叫'''泡学'''、'''把妹达人'''、'''搭讪艺术家'''，起初指的是一群受过系统化学习、实践、和不断自我完善情商的男性。后来泛指很会吸引异性，让异性着迷的男女们。但随着PUA文化的变迁，PUA的定义已从简单的搭讪扩展到整个两性交往流程。主要涉及：[[搭讪|搭讪]]（初识）、吸引（互动）、建立联系，升级关系、直到发生亲密接触并确定两性关系<ref>{{Cite web |url = http://cppcc.china.com.cn/2018-05/29/content_51530729.htm |title =媒体曝光情感操控骗局：如果PUA有用，还要爱情干嘛？ |author =中国网 |date = 2018-05-29 |publisher =  |language = zh-cn |accessdate =  |quote = }}</ref>。

==沿革==
泡学可以追溯到1970年，当时 Eric Weber写了一本书《泡妞攻略》（{{lang-en|《How To Pick Up Girls!》}}），被认为是泡学鼻祖。这一阶段，还有心理治疗师Albert Ellis 写了《情色诱惑的艺术》。但这一阶段，只有零星的独立作者和泡妞导师，并没有形成社群。

80年代后期，Ross Jeffries出版了关于泡妞技巧的短片小说《如何睡你想睡的女人》。1994年，Ross Jeffries的学生刘易斯· 德佩恩创立了新闻组 alt .seduction.fast（ASF），标志着勾引社群形成——一个以互联网论坛，电子邮件，博客，[[社交媒体|社交媒体]]组成的网络。

==批评与媒体报道==
在华语社会，最先把pua引入的是留学生。部分留学生翻译了Neil Strauss的《把妹达人》（{{lang-en|《The Game: Penetrating the Secret Society of Pickup Artists》}}），并将其引入华语地区。早期，该社群只是分享男性如何通过技巧和心理学应用，去接近、搭讪自己喜欢的人，但后来逐渐演化成骗色、骗财的手段。
同时大量的PUA组织自称恋爱导师，开始招生，学费高昂<ref>{{Cite web |url = http://news.sina.com.cn/s/2018-05-28/doc-ihcaquev2247454.shtml |title =起底PUA：教男生把妹女生撩凯子 单人学费数十万 |author = |date =2018年05月28日 |publisher = 新京报 |language = zh-cn |accessdate = 2019.01.17 |quote = }}</ref>。这些PUA组织不仅教授男生如何“把妹”，也会培训女生如何“撩凯子”。
