'''吊兰'''（[[学名|学名]]：''{{lang|la|Chlorophytum comosum}}''）又名'''钓兰'''、'''挂兰'''、'''兰草'''、'''折鹤兰'''，[[歐美|歐美]]國家稱'''蜘蛛草'''（{{lang|en|spider plant}}），[[日本|日本]]称折鶴蘭（{{lang|ja|折鶴蘭}}），是相當常見的垂掛式[[观叶植物|观叶植物]]，原產於[[南非|南非]]。

== 特征 ==
吊蘭為常綠[[草本|草本]]植物，具簇生的圓柱形肉質[[氣生根|鬚根]]和短根狀[[莖|莖]]。[[葉|葉]]基生，狹長柔軟條形，頂端漸尖，長30[[公分|公分]]左右、寬1－2公分；基部抱莖，著生於短莖上。成熟的植株會長出[[變態莖|走莖]]形成花梗，細長彎曲，自葉腋抽出超出葉上，長30－60公分，先端會長出有[[氣生根|氣生根]]的小植株。[[花序|總狀花序]]，單一或分枝；[[花|花]]數朵一簇，散生在花梗旁。花小，[[白色|白色]]，[[花被|花被]]2輪共6片，[[雄蕊|雄蕊]]6枚。[[子房|子房]]3室，通常有胚珠多顆；[[果實|果實]]成銳三角形，側向3裂；[[種子|種子]]扁平或碟狀。[[花序|花序]]上部有時會生出2－8公分的條形葉叢。花期一般在[[春|春]]、[[夏|夏]]，[[冬|冬]]季，室內也可開花。

== 栽培 ==
吊蘭性喜溫暖濕潤、半陰的環境。生長適溫為[[攝氏|攝氏]]15－25度，不低於5攝氏度能安全越冬。肉質根貯水組織發達，耐旱。適宜排水良好，肥沃鬆肥的[[土壤|土壤]]上生長。吊蘭一般使用分株繁殖，也可利用走莖上的小植株繁殖。吊蘭可以種子播種繁殖，但少用。

吊蘭適應性強，容易種植，且株形美觀，故受到廣泛的喜愛。經過證實，在室內種植吊蘭可有效減低室內的空氣污染物質<ref name="nasastudy">{{cite web
