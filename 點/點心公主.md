== 概要 ==
本作品在開始在[[講談社|講談社]][[少女漫畫|少女漫畫]]雜誌《[[Nakayoshi|Nakayoshi]]》（{{lang|ja|なかよし}}）2004年9月號（2004年8月發行）上開始連載，目前漫畫單行本出版了全部10冊。另外，新創刊的《[[Nakayoshi文庫|Nakayoshi文庫]]》（{{lang|ja|なかよし文庫}}）發售了小說版，內容全為漫畫版沒有的原創小說。

本作品在2006年獲得第30屆（平成18年度）[[講談社漫畫賞|講談社漫畫賞]]兒童部門獎<ref>講談社漫畫賞（日語：{{lang|ja|[[:ja:講談社漫画賞|講談社漫画賞]]}}）是日本講談社對連載於自家漫畫雜誌所評選頒發出來的獎賞，第一回獎賞成立於1977年。可以參考下列網站：{{Ja icon}}[http://www.kodansha.co.jp/award/manga.html 講談社漫畫賞] {{webarchive|url=https://web.archive.org/web/20120102010051/http://www.kodansha.co.jp/award/manga.html |date=2012-01-02 }}、{{Ja icon}}[http://www.kodansha.co.jp/award/archive/manga.html 講談社漫畫賞歷代受獎名單] {{webarchive|url=https://www.webcitation.org/68aPoPjDW?url=http://www.kodansha.co.jp/award/archive/manga.html |date=2012-06-21 }}</ref>。

[[台灣|台灣]]地區的中文版是由[[長鴻出版社|長鴻出版社]]所代理發行，[[香港|香港]]、[[澳門|澳門]]地區的中文版是由[[天下出版社|天下出版社]]所代理發行。此外[[美國|美國]]地區是由[[Del_Rey|Del Rey]]所代理發行。

== 故事簡介 ==
'''風見七虹香'''是個擅長料理的中學一年級生。雙親過世時與「布丁王子」相遇，從那個時候開始為了讓王子吃到世界第一好吃的點心，而成為點心師傅。在王子給的布丁湯匙裡的「星花學園」的標誌是唯一的線索，而轉學到星花學園……。

== 人物介紹 ==
=== 主要角色 ===
; {{nihongo|風見 七虹香|風見 七虹香（かざみ なじか）}}
:就讀中學二年級的女主角，暱稱「七虹香」。以前住在[[北海道|北海道]]的[[薰衣草|薰衣草]]之家，她為了與「布丁王子」相遇，而轉學到[[東京|東京]]的星花學園。非常喜歡做料理。七虹香被小茜誤認沒有任何專長卻轉到特別班，而開始欺負她，但是後來兩人互相和解了。以前被大地叫成[[猴子|猴子]]，現在則是直接叫她的名字。曾經參加過世界點心大賽，擁有敏銳的味覺的才能，後來由於空的[[死亡|死亡]]而喪失了味覺。因此被迫從特別班轉到普通班，並且不能居住在學校的宿舍。
; {{nihongo|北澤 大地|北沢 大地（きたざわ だいち）}}
:七虹香的同班同學。空的弟弟，但是兩人的關係非常差。擅長打[[籃球|籃球]]。雖然小茜喜歡他，但他喜歡七虹香。雖然他的個性粗魯，不過他也有善良的一面。曾經買一隻手錶打算送給七虹香，不過那隻手錶被小茜奪去了，跟小茜是青梅竹馬。空過世之後代替空的職位，成為新任的學生會長。是七虹香的布丁王子。
; {{nihongo|北澤 空|北沢 空（きたざわ そら）}}
:七虹香所憧憬的人，大地的哥哥，擅長彈鋼琴。星花學園的學生會長，個性溫柔，受到女孩子的歡迎，常常鼓勵七虹香。空的父親打算計劃建立料理學校而驅使七虹香，可是空不想傷害七虹香，而將料理學校的計劃中止。喜歡七虹香，為了幫七虹香購買材料而遇到交通事故身亡。
; {{nihongo|岸田 茜|岸田 茜（きしだ あかね）}}
:七虹香的同班同學，大地和空的青梅竹馬，從小就喜歡大地，是個個性強悍的女孩子。「Strawberry」雜誌的模特兒。當初她認為七虹香是個卑鄙的人，後來成為了朋友。以前她不當減肥時，吃了七虹香做的老奶奶的[[水蜜桃|水蜜桃]]派後，而變得和七虹香非常親近。
; {{nihongo|藤田先生|フジタさん}}
:「藤田食堂」的[[廚師|廚師]]，平常看起來像老頭子一樣。其實他以前是在[[巴黎|巴黎]]的三星級餐廳工作。
; {{nihongo|水野 星夜|水野 星夜（みずの せいや）}}
:代替七虹香的料理特優生，與空的外表相像。個性好強，跟七虹香一樣同在北海道成長，水野集團的獨生子。因在比賽時輸給七虹香而不甘，時常向她挑戰，後來漸漸喜歡七虹香，七虹香小時候吃的布丁其實是他做的，但是由大地拿給七虹香的。

=== 星花學園的人們 ===
; {{nihongo|理事長|理事長（りじちょう）}}
:空和大地的父親。
; {{nihongo|副會長|副会長（ふくかいちょう）}}
:來到藤田食堂。喜歡吃甜食，總是在咖啡加入三匙糖。
; {{nihongo|書記|書記（しょき）}}
:喜歡清淡口味。

=== 七虹香的家人 ===
; {{nihongo|風見 七瀬|風見 七瀬、かざみ ななせ}}
:七虹香的父親，因船難而死亡。
; {{nihongo|風見 香|風見 七瀬、かざみ かおり}}
:七虹香的母親，以前認識藤田先生，因船難而死亡。

=== 薰衣草之家 ===
; {{nihongo|萩尾老師|萩尾先生、はぎお}}
:薰衣草之家的老師，撫養七虹香長大的人。
; {{nihongo|風太|風太、ふうた}}
:住在薰衣草之家的少年，與七虹香同樣經歷雙親過世。喜歡吃的東西是香蕉，討厭的食物是胡蘿蔔。

=== 其他角色 ===
; {{nihongo|岸 夜梨子|岸 夜梨子、きし よりこ}}
:知名的料理記者，她所寫的書是暢銷書，介紹的店家也都門庭若市。她是全日本點心大賽的評審，對七虹香的才華感到佩服。
