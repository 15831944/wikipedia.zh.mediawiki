'''屠岸'''（{{bd|1923年||2017年|12月16日|catIdx=T屠}}），原名'''蒋壁厚'''，笔名'''叔牟'''、''' 屠岸'''（至于用‘屠岸’作为笔名，我是学[[鲁迅|鲁迅]]，用母亲的姓。‘岸’字有对当时反动政府傲岸的意思，我很喜欢）绰号“尤里卡”，<ref name="multiple1">{{cite web |url=http://www.chinanews.com/cul/2011/08-15/3259379.shtml|title=《屠岸：用生命与诗魂拥抱》|accessdate=2011年8月15日|work=|publisher=中国新闻网|language=zh-hans}}</ref>[[江苏|江苏]][[常州|常州]]人。中国当代著名诗人、翻译家，[[人民文学出版社|人民文学出版社]]原总编辑。

== 生平 ==
1923年11月22日，屠岸出生于[[江苏|江苏]][[常州|常州]]文笔塔下的一个书香世家。<ref name="multiple1">{{cite web |url=http://www.chinanews.com/cul/2011/08-15/3259379.shtml|title=《屠岸：用生命与诗魂拥抱》|accessdate=2011年8月15日|work=|publisher=中国新闻网|language=zh-hans}}</ref>

小学一年级，屠岸在女西校度过。二年级，转入冠英小学（觅渡桥小学前身），一直读到毕业。小学三年级时，母亲亲自教授他[[中国古典文化|中国古典文化]]，培养了他一生爱好诗歌的感情传统。<ref name="multiple1">{{cite web |url=http://www.chinanews.com/cul/2011/08-15/3259379.shtml|title=《屠岸：用生命与诗魂拥抱》|accessdate=2011年8月15日|work=|publisher=中国新闻网|language=zh-hans}}</ref>

屠岸于1946年在[[上海交通大学|上海交通大学]]肄业。

1941年开始发表诗歌。

1956年加入[[中国作家协会|中国作家协会]]。

1969年，[[文革|文革]]初期，军宣队宣布，屠岸必须在国庆前夕离开北京下放，屠岸下放到[[河北省|河北省]][[怀来县|怀来县]]沙城土木乡。<ref name="multiple2">{{cite web |url=http://www.chinanews.com/cul/2011/03-09/2894470.shtml|title=《屠岸的干校点滴:手里无书心中有诗》|accessdate=2011年3月9日|work=|publisher=中国新闻网|language=zh-hans}}</ref>

1969年深秋，屠岸从土木乡搬迁到宋家营。

1970年元旦后，剧协又一次迁移，转到[[河北省|河北省]][[宝坻县|宝坻县]]，分住在几个村里。屠岸分配到北清沟。屠岸和妻子的书籍都被没收，但是他依然在心中默念无字之书。<ref name="multiple2">{{cite web |url=http://www.chinanews.com/cul/2011/03-09/2894470.shtml|title=《屠岸的干校点滴:手里无书心中有诗》|accessdate=2011年3月9日|work=|publisher=中国新闻网|language=zh-hans}}</ref>

1970年9月，干校再次迁移，屠岸从宝坻搬到[[静海县|静海县]]的团泊洼。<ref name="multiple2">{{cite web |url=http://www.chinanews.com/cul/2011/03-09/2894470.shtml|title=《屠岸的干校点滴:手里无书心中有诗》|accessdate=2011年3月9日|work=|publisher=中国新闻网|language=zh-hans}}</ref>

1972年，“斗私批修”，屠岸做了两次思想检查，最终通过，恢复了[[中国共产党|中国共产党]]的组织生活。从1969年4月初步解放，到1972年7月才彻底解放。<ref name="multiple2">{{cite web |url=http://www.chinanews.com/cul/2011/03-09/2894470.shtml|title=《屠岸的干校点滴:手里无书心中有诗》|accessdate=2011年3月9日|work=|publisher=中国新闻网|language=zh-hans}}</ref>

1973年1月接到通知：屠岸的妻子妙英到出版口（后来发展成[[新闻出版总署|新闻出版总署]]）工作，屠岸分配到了[[人民文学出版社|人民文学出版社]]工作。<ref name="multiple2">{{cite web |url=http://www.chinanews.com/cul/2011/03-09/2894470.shtml|title=《屠岸的干校点滴:手里无书心中有诗》|accessdate=2011年3月9日|work=|publisher=中国新闻网|language=zh-hans}}</ref>

曾任《戏剧报》编辑、编辑部主任，[[中国戏剧家协会|中国戏剧家协会]]研究室副主任，[[人民文学出版社|人民文学出版社]]现代文学编辑室副主任、主任及副总编、总编、专家委员会副主任，编审等职务。[[中国作家协会|中国作家协会]]第四届理事，中国作家协会名誉委员。

2017年12月16日下午5时,在北京逝世，享年94岁。<ref>{{Cite web|url=http://www.thepaper.cn/newsDetail_forward_1909137|title=著名诗人、翻译家、出版家，人民文学出版社原总编辑屠岸逝世_中国政库_澎湃新闻-The Paper|accessdate=2017-12-16|work=www.thepaper.cn}}</ref>

*《哑歌人的自白》

==  奖项荣誉==
英国诗人[[济慈|济慈]]一生只活了25岁，因为22岁得了肺结核，屠岸也在22岁得了肺结核，这在当时是可怕的病，屠岸把济慈当做异国异代的冥中知己，似乎穿越了时空在生命和诗情上相遇。晚年的屠岸花了3年时间，译成《济慈诗选》，1997年由中国人民文学出版社出版，并且获得了第二届[[鲁迅文学奖|鲁迅文学奖]]翻译奖。<ref name="multiple1">{{cite web |url=http://www.chinanews.com/cul/2011/08-15/3259379.shtml|title=《屠岸：用生命与诗魂拥抱》|accessdate=2011年8月15日|work=|publisher=中国新闻网|language=zh-hans}}</ref>

== 评价 ==
*诗评家[[杨匡汉|杨匡汉]]称“《屠岸十四行诗》的问世，标志着又一位中国十四行诗人的成热”。<ref name="multiple1">{{cite web |url=http://www.chinanews.com/cul/2011/08-15/3259379.shtml|title=《屠岸：用生命与诗魂拥抱》|accessdate=2011年8月15日|work=|publisher=中国新闻网|language=zh-hans}}</ref>

*文艺评论家[[李元洛|李元洛]]对屠岸的诗大加称道，称“屠岸是一位具有自己的风格的诗人，他的风格整体美学特色可以用‘高雅典丽’一语来概括”。<ref name="multiple1">{{cite web |url=http://www.chinanews.com/cul/2011/08-15/3259379.shtml|title=《屠岸：用生命与诗魂拥抱》|accessdate=2011年8月15日|work=|publisher=中国新闻网|language=zh-hans}}</ref>
