'''昆明地铁6号线'''是[[昆明地铁|昆明地铁]]的一条线路，又被称作'''昆明长水国际机场快速公交项目'''，该线连接昆明市区至[[昆明长水国际机场|昆明长水国际机场]]。一期工程于2010年3月开工建设，2012年6月28日通车运营。全长约18公里，首末班时间为每天6:20至22:20，发车间隔时间约15分钟，单程运行时间约23分钟，票价为0～4公里内2元起步，之后按实际乘坐里程“递远递减”跳档计价，跳档里程分别为5、7、9、11、13公里等，每跳一档，票价均按1元增加，以此类推。二期工程计划延伸至昆明市中心，通過3號線與其他線路連接並實現換乘。为配合昆明地铁3号线工程的建设，昆明地铁6号线于2016年3月起暂停观光试运行。停运期间，线路与3号线开展同步调试，并实施提升改造和功能完善工作，2017年8月29日与地铁3号线同步开通试运营。

== 历史 ==
*2010年3月18日，长水机场场内段（机场中心站-机场前站）开工建设。
*2010年8月1日，6号线一期工程全线开工建设。
*2011年4月28日，6号线大板桥站特大桥第一片梁实现成功浇筑。<ref>{{cite web |url=http://news.kunming.cn/subject/content/2011-05/09/content_2519709.htm |date=2011年5月9日 |title=昆明新机场快速公交项目 第一片梁实现成功浇筑}}</ref>
*2011年6月22日，由中铁十八局集团四公司承建的昆明地铁六号线瓦角村隧道胜利贯通，这也是昆明地铁六号线工程首座贯通的隧道。<ref>{{Cite web |url=http://www.cr18g.com/Article_Show.asp?ArticleID=4137 |date=2011年6年23日 |title=昆明地铁六号线首座隧道贯通 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
*2011年12月27日，6号线大板桥站至航空港南站区间开始了冷滑试验。
*2011年12月30日，昆明地铁首列车在6号线大板桥站至航空港南站区间进行了第一次热滑试验。
*2012年2月6日，昆明地铁6号线空载试运行开始。<ref>{{cite web |url=http://kunming.yunnan.cn/html/2012-02/10/content_2037138.htm |date=2012年2月10日 |title=昆明地铁列车从2月6日已开始试运行 |publisher=云南网 |deadurl=yes |archiveurl=https://web.archive.org/web/20120317151557/http://kunming.yunnan.cn/html/2012-02/10/content_2037138.htm |archivedate=2012年3月17日 |df= }}</ref>
*2012年6月28日，昆明地铁6号线正式投入运营。<ref>{{cite web |url=http://www.kmgdgs.com/content/2012-06/27/content_2999098.htm |date=2012年6月27日 |title=长水国际机场快速公交基础设施通车 |publisher=昆明轨道交通有限公司 |deadurl=yes |archiveurl=https://web.archive.org/web/20140517152658/http://www.kmgdgs.com/content/2012-06/27/content_2999098.htm |archivedate=2014年5月17日 }}</ref>
*2016年3月5日，昆明地铁6号线暂停运营，开启提质调试工作。<ref>{{cite web |url=http://news.163.com/16/0229/06/BGVLLMP500014AEE.html |date=2016年2月29日 |title=昆明地铁6号线一期下月5日起暂停运行 预计明年上半年恢复开通 |publisher=云南网}}</ref>
*2017年8月29日，6号线东部汽车站-机场中心站段重新开通，机场前，大板桥两站投入使用，同时开通的还有[[昆明地铁3号线|昆明地铁3号线]]。<ref>{{cite web|url=http://society.yunnan.cn/html/2017-08/29/content_4923411.htm|title=昆明轨道交通3号线、6号线一期开通试运营|publisher=云南网|date=2017-08-29|accessdate=2017-08-29|archive-url=https://web.archive.org/web/20170829120214/http://society.yunnan.cn/html/2017-08/29/content_4923411.htm|archive-date=2017-08-29|dead-url=yes}}</ref>

== 线路与车站 ==
昆明地铁6号线共设6站5区间。

一期工程于东部汽车客运站附近设起点站[[东部汽车站站|东部汽车站站]]，与3号线换乘，之后沿机场高速公路北侧至新机场，并在大板桥镇附近设[[大板桥站|大板桥站]]，在过沾昆铁路之后设[[机场前站|机场前站]]，之后以小半径转至航站楼前，并平行于航站楼设终点站[[机场中心站|机场中心站]]。<ref>[http://www.kmgdgs.com/content/2010-04/24/content_2136111.htm 昆明轨道交通6号线（机场线）工程招标]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>6号线一期工程线路全长18.018公里，其中地下段长7.76公里，高架段长7.633公里。全线设站4座，其中高架车站1座，地下车站3座。

二期工程将西延伸2站2区间，西起于塘子巷（设[[塘子巷站|塘子巷站]]），经拓东路、东郊路，下穿菊华立交桥（设[[菊华站|菊华站]]）后，线路拐向北侧经归十西路、虹桥立交，在东部汽车客运站附近接入一期线路。二期段线路长度为7.616公里，其中地下线长6.514公里，高架线长1.102公里，设地下车站2座，可同在建中的轨道交通2号线换乘。<ref name=r1>[http://daily.clzg.cn/html/2011-08/09/content_221115.htm 昆明机场轨道交通示范线（6号线）工程变更环境影响评价第二次公示] {{webarchive|url=https://web.archive.org/web/20140714150038/http://daily.clzg.cn/html/2011-08/09/content_221115.htm |date=2014-07-14 }}.2011-08-09.</ref>二期工程最初规划起于[[巫家坝国际机场|巫家坝机场]]，后更改为位于昆明主城内拓东路附近的[[塘子巷站|塘子巷站]]，可同[[昆明轨道交通2号线|2号线]]换乘。<ref name=r1>[http://daily.clzg.cn/html/2011-08/09/content_221115.htm 昆明机场轨道交通示范线（6号线）工程变更环境影响评价第二次公示] {{webarchive|url=https://web.archive.org/web/20140714150038/http://daily.clzg.cn/html/2011-08/09/content_221115.htm |date=2014-07-14 }}.2011-08-09.</ref>
