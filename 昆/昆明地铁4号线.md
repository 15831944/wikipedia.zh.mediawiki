'''昆明地铁4号线'''是[[昆明地铁|昆明地铁]]在建中的一条线路，为连接昆明主城与呈贡的西北、东南方向辅助线。4号线北端起于陈家营站，南端止于[[昆明南站|昆明南站]]，串联了昆明主城西北的国家高新技术产业开发区、主城中心区、主城东南的国家经济技术开发区，并向南连接呈贡新城。<ref>{{Cite web |url=http://yn.yunnan.cn/html/2017-03/14/content_4758796.htm |title=昆明地铁4号线螺狮湾站15日封顶 |access-date=2017-05-25 |archive-url=https://web.archive.org/web/20170314135941/http://yn.yunnan.cn/html/2017-03/14/content_4758796.htm |archive-date=2017-03-14 |dead-url=yes }}</ref>线路全長43.396公里，全地下线，共设置29個地下站。<ref>{{Cite web |url=http://yn.yunnan.cn/html/2017-05/12/content_4822831.htm |title=昆明地铁4号线鲜花大道站主体结构封顶 2020年上半年试运营 |access-date=2017-05-25 |archive-url=https://web.archive.org/web/20170512142230/http://yn.yunnan.cn/html/2017-05/12/content_4822831.htm |archive-date=2017-05-12 |dead-url=yes }}</ref>

==站點==
4 号线将与1、2、3、5、6、7号线等設换乘站。位于海屯路与昆武高速高架桥交叉口西侧的大塘子站，共设置7个出入口，將成為4号线的第二大站，也是4与7号线的换乘站。<ref>{{Cite web |url=http://yn.yunnan.cn/html/2017-03/14/content_4758796.htm |title=昆明地铁4号线螺狮湾站15日封顶 |access-date=2017-05-25 |archive-url=https://web.archive.org/web/20170314135941/http://yn.yunnan.cn/html/2017-03/14/content_4758796.htm |archive-date=2017-03-14 |dead-url=yes }}</ref>

==歷史==
* 2016年12月30日，4号线开始盾构施工。3台盾构机分别从陈家营站、螺蛳湾站、赛马场西站同步顺利始发。 <ref>{{Cite web |url=http://society.yunnan.cn/html/2016-12/31/content_4676981.htm |title=昆明地铁4号线开始盾构施工 |access-date=2017-05-25 |archive-url=https://web.archive.org/web/20170101143434/http://society.yunnan.cn/html/2016-12/31/content_4676981.htm |archive-date=2017-01-01 |dead-url=yes }}</ref>
* 2017年5月12日，鲜花大道站主体结构封顶，為4号线第三个封顶的车站。<ref>{{Cite web |url=http://yn.yunnan.cn/html/2017-05/12/content_4822831.htm |title=昆明地铁4号线鲜花大道站主体结构封顶 2020年上半年试运营 |access-date=2017-05-25 |archive-url=https://web.archive.org/web/20170512142230/http://yn.yunnan.cn/html/2017-05/12/content_4822831.htm |archive-date=2017-05-12 |dead-url=yes }}</ref>
