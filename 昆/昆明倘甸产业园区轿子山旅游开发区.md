'''{{FULLPAGENAME}}'''、'''倘甸两区'''<ref>{{Cite web |url=http://www.kmcdc.cn/html/area/t5.htm |title=区县之窗》昆明倘甸产业园区轿子山旅游开发区（倘甸两区）|accessdate=2018-05-31 |author= |date=|work= |publisher=昆明市疾病预防控制中心网站|language=Zh-hans }}</ref>、'''两区'''<ref name="a1">{{Cite web |url=http://news.163.com/16/0925/11/C1Q9SE8G00014AEE.html |title=朱家健：倘甸和轿子山两区 2018年与昆明市同步实现脱贫 |accessdate=2018-05-31 |author= |date=2016-09-25|work= |publisher=网易，来源: 云南网(昆明)|language=Zh-hans }}</ref>是[[中华人民共和国|中国]][[云南省|云南省]][[昆明市|昆明市]]的一个[[县级管理区|县级管理区]]，而在行政上则定义为[[寻甸回族彝族自治县|寻甸回族彝族自治县]]下辖的一个[[类似乡级单位|类似乡级单位]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|53/01/29/405/000}}}}</ref>。实际上管辖九个乡镇：[[东川区|东川区]][[红土地镇|红土地镇]]、[[舍块乡|舍块乡]]，[[禄劝彝族苗族自治县|禄劝彝族苗族自治县]][[转龙镇_(禄劝县)|转龙镇]]、[[雪山乡_(禄劝县)|雪山乡]]、[[乌蒙乡|乌蒙乡]]，[[寻甸回族彝族自治县|寻甸回族彝族自治县]][[倘甸镇|倘甸镇]]、[[凤合镇|凤合镇]]、[[联合乡_(寻甸县)|联合乡]]、[[金源乡_(寻甸县)|金源乡]]<ref>{{Cite web |url=https://www.sohu.com/a/147449980_711512 |title=昆明倘甸产业园区轿子山旅游开发区卫生系统二〇一七年公开招聘工作人员公告 |accessdate=2018-05-31 |author= |date=|work= |publisher=搜狐网|language=Zh-hans }}</ref>。
