'''昆仑山脉'''（{{lang-en|'''Kunlun Mountains'''}}；{{bo|z=ཁུ་ནུ་རི་རྒྱུད|w=khu nu ri rgyud}}）西起[[帕米尔高原|帕米尔高原]]东部，东到[[柴达木河|柴达木河]]上游谷地，于东经97°至99°处与[[巴颜喀拉山脉|巴颜喀拉山脉]]和[[阿尼玛卿山|阿尼玛卿山]]（[[积石山|积石山]]）相接，北邻[[塔里木盆地|塔里木盆地]]与[[柴达木盆地|柴达木盆地]]。山脉全长2,500余公里，宽130至200公里，[[平均海拔|平均海拔]]5,500至6,000米，西窄东宽，总面积达50多万平方公里。一般认为最高峰是[[崑崙山|崑崙山]]（[[海拔|海拔]]6638[[米|米]]），位於[[新疆维吾尔自治区|新疆维吾尔自治区]][[和田地区|和田地区]]南面，但实际上[[公格尔山|公格尔山]]（7719米）最高。

昆仑火山群是周邊山區、高原上罕見的火山山區，有超過70座火山坐落於此，其中最高的一座火山海拔高達5808公尺（{{Coord|35.5|N|80.2|E|type:mountain}}），但因其位於高原上，因此實際上火山錐實體僅高於周邊高原約300公尺（僅海拔的5％），但若僅考量絕對高度（海拔）而不考量相對高度而言，該山為亞洲、中國第一高、東半球第二高的火山（僅次於[[吉力馬扎羅山|吉力馬扎羅山]]），也是{{link-en|七大洲最高火山|Volcanic Seven Summits}}之一，與伊朗的[[德馬峰|德馬峰]]並論，最後一次爆發時間為1951年5月27日<ref>[http://www.volcano.si.edu/world/volcano.cfm?vnum=1004-03- 全球火山作用計畫 - 昆仑火山群]</ref>。

唐時，西域來的僕從時而被稱作「崑崙奴」（唐傳奇）。

== 名称 ==
[[File:Silk-Road_course.jpg|thumb]]
[[File:Kunlun_IMG_0541.jpg|thumb]]
最早《[[禹贡|禹贡]]》即记有昆仑，《[[山海经|山海经]]·海内西经》描述说：“海内昆仑之虚，在西北、帝下之都。昆仑之虚，方八百里，高万仞。上有木禾，长五寻，大五围。面有九井，以玉为槛。面有九门，门有开明善守之，百神之所在。”《[[淮南子|淮南子]]》认为：“昆仑之丘，或上倍之，是谓凉风之山，登之而不死；或上倍之，是谓悬圃，登之乃灵，能使风雨；或上倍之，乃维上天，登之乃神，是谓太帝之居。”[[干寶|干寶]]《[[搜神記|搜神記]]》卷十三-{云}-：「崑崙之墟，……其外絕以弱水之深，又環以炎火之山。山上有鳥獸草木，皆生育滋長於炎火之中，故有火浣布。」[[王嘉|王嘉]]《[[拾遺記|拾遺記]]》卷十-{云}-：「崑崙山有昆陵之地，其高出日月之上。山有九層，每層相去萬里。有雲氣，從下望之，如城闕之象。」

《[[史记|史记]]．大宛传》中记载：“汉使穷河源，河源出于寘，其山多玉石，采来，天子案古图书，名河所出山曰崑崙云”。[[蔡元定|蔡元定]]《发微论》：“凡山皆祖昆仑，分枝分脉，愈繁愈细，此万殊而一本也。”陶公《提脉赋》亦-{云}-：“大致察脉起自昆仑”。[[崔鸿|崔鸿]]在《[[十六国春秋|十六国春秋]]》中，称昆仑山为“海上之诸山之祖”、“天下名山僧占多”。《[[古今图书集成|古今图书集成]]》载：“西王母所居宫阙，在龟山春山西那之都，昆仑之圃，阆风之苑。有城千里，玉楼十二，琼华之阙，光碧之堂，九层元室，紫翠丹房；左带瑶池，右环翠水。其山之下，弱水九重，洪涛万丈，非飙车羽轮，不可到也。”

== 地势 ==
崑崙山脉西高东低，按地势分西、中、东3段。
*西昆仑山脉位于[[帕米尔高原|帕米尔高原]]的东缘，北起中国、吉尔吉斯斯坦、塔吉克斯坦交界处的[[外阿莱山脉|外阿莱山脉]]。西北-东南走向至[[克里雅山口|克里雅山口]]与中昆仑山脉分界。位于西昆仑山海拔在7000米以上的山峰有3座，6000米以上的山峰有7座，平均海拔为5500～6000米。[[公格尔峰|公格尔峰]]海拔7649米，[[公格尔九别峰|公格尔九别峰]]海拔7530米，[[慕士塔格峰|慕士塔格峰]]7509米。昆仑山脉西侧与之平行的[[喀喇昆仑山脉|喀喇昆仑山脉]]更为高大雄峻，二者之间为海拔3100～3900米[[塔什库尔干谷地|塔什库尔干谷地]]与[[叶尔羌河|叶尔羌河谷]]。 
*中昆仑山脉西起[[克里雅山口|克里雅山口]]与西昆仑山脉分界，东至[[车尔臣河|车尔臣河]]九个大坂山与东昆仑山与[[阿尔金山脉|阿尔金山脉]]相接。即东经77°～86°。主脉东西向向南略呈弧形。中昆仑山海拔6000米以上的山峰有8座，平均海拔5000～5500米 
*东昆仑山脉从[[喀拉米兰山口|喀拉米兰山口]]自西向东略呈扇形展开，分为3支：
**北支[[祁漫塔格|祁漫塔格山]]，其南隔以[[阿牙克库木盆地|阿牙克库木盆地]]；东延为[[唐松乌拉山|唐松乌拉山]]、[[布尔汗布达山|布尔汗布达山]]；
**中支[[阿尔格山|阿尔格山]]，东延为[[博卡雷克塔格|博卡雷克塔格]]、[[唐格乌拉山|唐格乌拉山]]与[[布青山|布青山]]，与[[阿尼玛卿山|阿尼玛卿山]]相接；
**南支[[可可西里山|可可西里山]]，与[[巴颜喀拉山|巴颜喀拉山]]相接  

[[克里雅山口|克里雅山口]]位于昆仑山的西段与中段的山势走向转折点，西昆仑山自帕米尔高原自西北向东南迤逦而来，在克里雅山口转折为自西向东走势的中昆仑山。克里雅山口西侧的高峰{{tsl|en|Kunlun Goddess|昆仑神女峰}}(Kunlun Goddess Mt.)，海拔7167米，谷歌地球上的坐标是北纬35度18分51.03秒，东经80度55分00.55秒，是[[和田地区|和田地区]][[玉龙喀什河|玉龙喀什河]]的源头。如果把更高的公格尔山与慕士塔格山算作帕米尔高山群，而把昆仑神女峰视作昆仑山的最高峰。 

昆仑山[[克里雅山口|克里雅山口]]西北方向一片辽阔的[[阿什库勒盆地|阿什库勒盆地]](即乌鲁克高原盆地)无人区，平均海拔4700余米，面积约740平方公里。盆地内的阿什库勒火山群的14座火山，熔岩面积达180平方公里，其中一号火山是[[中国大陆|中国大陆]]最年轻的火山，海拔4921米，相对高度150米，坐标为北纬35度41分50.09秒，东经81度34分38.45。1951年5月27日上午9时50分曾经火山爆炸式喷发，无熔岩溢出。

中昆仑山、东昆仑山和[[阿尔金山|阿尔金山]]的分界，是[[喀拉米兰山口|喀拉米兰山口]]<ref>[http://www.igsnrr.ac.cn/kxcb/dlyzykpyd/kxkcsj/wtwrqtm/200611/t20061101_2156127.html 李明森：《羌塘无人区探秘十七——登上喀拉米兰山口》，[[中国科学院地理科学与资源研究所|中国科学院地理科学与资源研究所]]网站 ]</ref>与[[车尔臣河|车尔臣河谷]]。阿尔金山以[[元古代|元古代]]地层为主，而昆仑山以[[晚古生代|晚古生代]]为主。从[[地质图|地质图]]上看阿尔金山颜色要比东昆仑山颜色暗很多，二者的界限是[[阿尔金断裂带|阿尔金断裂带]]。

== 地质 ==
崑崙山脉与[[塔里木盆地|塔里木盆地]]和[[柴达木盆地|柴达木盆地]]间均以深大断裂相隔。属[[南亚陆间区|南亚陆间区]]与[[中轴大陆区|中轴大陆区]]交界的北缘。崑崙山地区以[[前震旦系|前震旦系]]为基底；经过[[古生代|古生代]]海域下沉及[[华力西运动|华力西运动]]褶皱上升，构成崑崙中轴和山脉中脊；经过[[中生代|中生代]][[燕山运动|燕山运动]]构成主脊两侧4000米以上的山体。

== 氣候 ==
崑崙山北坡屬[[暖溫帶|暖溫帶]]塔里木荒漠和柴達木荒漠，降水量小；隨著海拔的增高，暖溫帶荒漠過渡為高山荒漠，降水量隨之增加。雪線在海拔5600米至5900米，雪線以上為終年不化的冰川，冰川面積達到3000平方公里以上，是中國的大冰川區之一，冰川溶水是中國幾條主要大河的源頭，包括[[長江|長江]]、[[黃河|黃河]]、[[瀾滄江|瀾滄江]]（湄公河）、[[怒江|怒江]]（薩爾溫江）和[[塔里木河|塔里木河]]。

== 物产 ==
崑崙山区有100多种高等[[植物|植物]]，但一般都是低矮的灌木类。野生[[动物|动物]]都是高原特有的如[[藏羚羊|藏羚羊]]、[[野牦牛|野牦牛]]、[[野驴|野驴]]等。

新疆和田的崑崙山麓出产最高质美[[玉|玉]]——[[和阗玉|和田玉]]，从古代起就是[[中原地区|中原地区]]玉石主要来源，因此《[[千字文|千字文]]》提到“玉出崑岗”。

== 关于昆仑山的神话崇拜 ==
《史记》记载“禹兴于西羌”。《[[山海经|山海经]]》详尽描述了神话中的昆仑山与[[西王母|西王母]]，[[华夏族|华夏族]]从西北之高原，迁向中原，再到沿海。<ref>《[[山海经|山海经]]·海內西经》：海内昆仑之虚，在西北，帝之下都。昆仑之虚，方八百里，高万 仞。上有木禾，长五寻，大五围。面有九井，以玉为槛。面 有九门，门有开明兽守之，百神之所在。在八隅之岩，赤水之际，非 （仁） [夷]羿莫能上冈之岩。</ref>《[[穆天子传|穆天子传]]》记载：穆天子见西王母于昆仑丘；<ref>《穆天子传》：天子西征，至于西王母之邦，吉日甲子，天子宾于西王母。</ref>穆天子至群玉之山，天下高山尽收眼里（通说为帕米尔高原）。昆仑崇拜自古以来都是中国至高崇拜，[[汉人|汉人]]的玉石崇拜也是因为昆仑产玉，玉石在汉文化中成为高贵血统的标示。

有一種說法認為由於[[崑崙山脈|崑崙山脈]]高聳挺拔，為古代中國和西部之間的天然屏障，所以被古代中國人當作世界邊緣，加上其終年積[[雪|雪]]令中國古代以[[白色|白色]]象徵西方<ref>{{cite web|title=一起踏上寻根之旅 探秘中国龙脉潜伏之地(2)|url=http://travel.china.com/vane/happy/11119700/20121225/17599696_1.html|website=中华网旅游_用旅行传递世界之美|deadurl=yes|archiveurl=https://web.archive.org/web/20171201081006/http://travel.china.com/vane/happy/11119700/20121225/17599696_1.html|archivedate=2017年12月1日|df=}}</ref>。
