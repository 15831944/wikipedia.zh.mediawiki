'''昆明长水国际机场'''（{{airport codes|KMG|ZPPP}}）是[[中华人民共和国|中华人民共和国]][[云南省|云南省]][[昆明市|昆明市]]新啟用的机场，於'''2012年6月28日8時'''起開始營運，一般也称为'''昆明新机场'''（相应地，[[昆明巫家坝国际机场|昆明巫家坝国际机场]]则是'''老机场'''）。定位为中国面向[[东南亚|东南亚]]、[[南亚|南亚]]和连接欧亚的国家门户枢纽机场<ref>{{Cite web |url=http://www.yn.chinanews.com.cn/html/other2/20081125/77511.html |title=投资230亿 昆明将拥有国家大型门户枢纽机场 |date=2008年11月25日 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

昆明新机场场址位于[[官渡区|官渡区]][[大板桥街道|大板桥街道]]长水村与花箐村<ref>{{Cite web|url=http://www.shxb.net/html/20051116/20051116_11831.shtml|title=迎新机场 千名村民将搬迁|date=2005年11月16日|deadurl=yes|archiveurl=https://web.archive.org/web/20151118092957/http://www.shxb.net/html/20051116/20051116_11831.shtml|archivedate=2015年11月18日|df=}}</ref>，在昆明市东北方向，距市中心直线距离约24.5公里，海拔2102米。项目总投资230余亿元，本期规划为满足2020年旅客吞吐量3800万人次、货邮吞吐量95万吨、飞机起降30.3万架次的需求建设。远期规划控制用地约22.97平方公里。昆明新机场本期建有两条平行跑道，东跑道长4500米，宽60米，可满足[[空客|空客]][[A380|A380]]客机的起降；西跑道长4000米，宽45米。[[机场飞行区等级|机场飞行区等级]]为按照4F标准规划设计。

昆明长水国际机场2018年全年旅客吞吐量达47,088,140人次，位居中国大陆机场第六位；货邮吞吐量达428,292.1吨，位居大陆第八位；起降航班达360,785架次，位居大陆第四位<ref name = "caac"/>。

== 历史 ==

=== 建设背景 ===
昆明市先前使用的[[巫家坝国际机场|巫家坝国际机场]]，建于1922年，历经3次改扩建，航站楼设计容量800万人次，但仅2008年巫家坝机场的客运吞吐量就达到了1528万人次，2010年达到2019万人次<ref>{{Cite web |url=http://www.caac.gov.cn/I1/K3/201103/t20110315_38273.html |title=2010年全国机场生产统计公报 |date=2011年3月15日 |deadurl=yes |archiveurl=https://web.archive.org/web/20120221215245/http://www.caac.gov.cn/I1/K3/201103/t20110315_38273.html |archivedate=2012年2月21日 }}</ref>，远远超出了现有航站楼设计容量，机场运营压力巨大。而且巫家坝机场距离仅昆明市中心直线距离仅6.6公里，周围已被城市包围，不具备原地扩建的条件。因此，昆明市政府决定迁建一座全新的机场。

昆明新机场建设项目前期工作自1998年启动。
*2007年1月29日，[[国务院|国务院]]和[[中央军委|中央军委]]批准立项。
*2007年10月，[[中國民用航空局|中國民用航空局]]批复昆明新机场选址方案。
*2007年11月15日，[[国土资源部|国土资源部]]批复建设用地预审。
*2007年12月11日，昆明新机场正式奠基。
*2008年8月26日，[[国家发改委|国家发改委]]批复昆明新机场可行性研究报告。
*2008年12月5日，昆明新机场正式开工建设。
*2008年12月22日，中国民航局、云南省政府联合批复昆明新机场总体规划。
*2009年2月24日，中国民航局、云南省政府联合批复昆明新机场建设项目机场工程初步设计及概算。
*2009年9月13日，昆明新机场代表性建筑“七彩钢带”合龙。
*2009年12月1日，航站楼组合体混凝土结构工程完成。
*2010年6月5日，昆明新机场航站楼主体工程正式封顶。<ref>{{Cite web |url=http://tv.people.com.cn/GB/166419/11793098.html|title=昆明新机场航站楼今天封顶|date=2010年6月5日}}</ref>
*2011年7月29日，昆明新机场飞行区东跑道沥青道面工程完工。<ref>{{Cite web |url=http://www.newkma.com/a/jianshedongtai/20110823_816.html |title=昆明新机场飞行区东跑道沥青道面顺利完工 |date=2011年8月23日 }}</ref>
*2011年8月29日，昆明新机场正式命名为昆明长水国际机场。<ref>{{Cite web |url=http://www.caac.gov.cn/A3/201108/t20110829_42119.html |title=昆明新机场正式命名为昆明长水国际机场 |publisher=中国民航总局 |language=zh-cn |date=2011年8月29日 |deadurl=yes |archiveurl=https://web.archive.org/web/20111106150456/http://www.caac.gov.cn/A3/201108/t20110829_42119.html |archivedate=2011年11月6日 }}</ref>
*2011年12月17日，昆明新机场主体工程竣工验收结束。<ref>{{Cite web |url=http://www.newkma.com/a/jianshedongtai/20120111_1033.html |title=昆明新机场第二批非民航专业单位工程顺利通过竣工验收 |publisher=昆明新机场建设指挥部 |language = zh-cn |date=2012年1月11日 }}</ref>
*2012年6月28日，昆明长水国际机场正式啟用。

=== 校验调试及转场运营 ===
昆明新机场工程建设由昆明新机场建设指挥部统一协调，经过近3年的主体工程建设后，进入校验调试及转场运营阶段。机场原计划于2011年底实现转场运营<ref>{{Cite web|url=http://www.yn.xinhuanet.com/newscenter/2011-01/27/content_21961054.htm|title=李汉柏：确保昆明新机场实现年底转场运营目标|publisher=新华网云南频道|language=zh-cn|date=2011年1月27日|deadurl=yes|archiveurl=https://web.archive.org/web/20130730190424/http://www.yn.xinhuanet.com/newscenter/2011-01/27/content_21961054.htm|archivedate=2013年7月30日}}</ref>，但是由于工程建设进度比原计划滞后等多个原因，转场时间后推迟至2012年6月28日转场运行<ref>{{cite news |language = zh-cn | url =http://news.kunming.cn/km-news/content/2012-01/13/content_2809162.htm | title =昆明新机场“转场”延时 暂无校飞和试飞时间表 | publisher =昆明信息港| date =2012年1月13日}}</ref><ref>{{cite news |language = zh-cn |url = http://kunming.yunnan.cn/html/2012-03/02/content_2072619.htm |title = 昆明长水国际机场将于2012年6月28日转场运行 |publisher = 云南网 |date = 2012年3月2日 |access-date = 2012年3月3日 |archive-url = https://web.archive.org/web/20160306041655/http://kunming.yunnan.cn/html/2012-03/02/content_2072619.htm |archive-date = 2016年3月6日 |dead-url = yes }}</ref>。新机场于2012年6月27日晚正式开始转场，巫家坝机场同时关闭。设备及器材等在当晚午夜陆续从巫家坝机场搬迁往新机场，停泊在巫家坝机场的18架飞机陆续转场到新机场。早上8时，长水机场正式开始运作。首班从长水机场起飞的由昆明经停[[迪庆|迪庆]]飞往[[拉萨|拉萨]]的'''MU5939'''次航班成功飞离昆明长水国际机场，标志着昆明长水国际机场正式投入运营<ref name="r3">{{Cite web |url=http://yn.xinhuanet.com/newscenter/2012-06/28/c_131680551.htm |title=昆明长水国际机场开始正式运营 |date=2012年6月28日 |publisher=新华网 |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20160304192250/http://yn.xinhuanet.com/newscenter/2012-06/28/c_131680551.htm |archivedate=2016年3月4日 }}</ref>。

=== 运营状况 ===
2016年12月13日，昆明长水国际机场成为中国大陆第5家年旅客吞吐量突破4000万人次的机场<ref>{{cite web|url=http://yn.yunnan.cn/html/2016-12/13/content_4652009.htm|title=昆明长水机场年旅客吞吐量突破4000万人次|work=云南网|access-date=2017-01-06|archive-url=https://web.archive.org/web/20161214160751/http://yn.yunnan.cn/html/2016-12/13/content_4652009.htm|archive-date=2016-12-14|dead-url=yes}}</ref>，之前昆明长水机场在2014年突破3000万<ref name="30 million">[http://news.carnoc.com/list/300/300989.html 昆明长水国际机场年旅客吞吐量突破3000万人次]</ref>，昆明巫家坝机场于2005年旅客吞吐量突破1000万人次，2010年突破2000万人次。截止2016年12月，昆明机场共开通航线297条，其中国内航线230条，国际航线60条，地区航线7条。通航城市158个，其中国内113个，国际40个，地区5个。共有46家航空公司在昆明航空市场运营，其中国内28家，国际14家，地区4家。2017年，吞吐量达全国第六，全球第37名。

== 设施与服务 ==

=== 航站楼 ===
[[File:Kunming_new_airport_terminal_2012-9-26.jpg|thumb]]
[[File:Kunming_Changshui_Airport_interior1.JPG|thumb]]
[[File:KMG_arrivals_lounge.jpg|thumb]]
昆明新机场航站楼位于两条平行[[跑道|跑道]]之间的航站区用地南端，主要由前端主楼、前端东西两侧指廊、中央指廊、远端东西Y型指廊组成。南北总长度为855.1米，东西宽1134.8米，最高点为南侧屋脊顶点，相对标高72.91米。航站楼建筑占地15.91万平方米，总建筑面积54.83万平方米，为地上三层（局部四层）、地下三层构型。昆明新机场航站楼建筑由曾经参与过[[鸟巢|鸟巢]]、[[首都国际机场|首都国际机场]]T3航站楼设计与工程建设的英国工程顾问公司[[奥雅纳|奥雅纳]]和北京市建筑设计研究院有限公司设计<ref>{{Cite web |url=http://www.skyscrapercity.com/showthread.php?t=401560 |title=Arup signs MoU for the design of major airport hub in China |date=2006年9月13日 |language=en}}</ref>。提供站坪机位84个，其中近机位68个<ref>{{Cite web |url=http://www.newkma.com/a/xiangmugaikuang/20100430_3.html|title=昆明新机场项目简介}}</ref>。建成后超越了[[墨西哥城贝尼托·胡亚雷斯国际机场|墨西哥城机场]]成为世界{{tsl|en|List of largest buildings in the world|世界最大型建筑列表|第五大}}航站楼，同时也是仅次于[[首都国际机场|首都国际机场]]T3（98.6万平方米）的国内第二大航站楼<ref name="r2">{{Cite web|url=http://www.newkma.com/a/xiangmugaikuang/20100430_5.html|title=昆明新机场规划设计创新|date=2010年5月8日|deadurl=yes|archiveurl=https://web.archive.org/web/20120313115818/http://www.newkma.com/a/xiangmugaikuang/20100430_5.html|archivedate=2012年3月13日}}</ref>。新建3.5万平方米的货运站，1.4万平方米的航空配餐设施；配套建设供电、供水、供热、供冷、燃气、污水污物处理设施等。2013年1月1日起，机场推出免费无线上网服务，信号主要覆盖航站楼F3层出发大厅、待安检区、安检区、候机厅、B1层到达大厅等区域<ref>{{Cite web |url=http://news.kunming.cn/km-news/content/2012-12/31/content_3175039.htm |title=昆明长水国际机场推出旅客免费无线上网服务 |date=2012年12月31日 |accessdate=2013年1月2日 |publisher=昆明信息港}}</ref>。2014年10月26日起，昆明长水机场提供“手机值机”服务。旅客可无需打印登机牌，直接到安检口扫描手机二维码，打印登机凭证，完成安检、登机<ref name="2014冬春换季">{{Cite web |url=http://news.carnoc.com/list/297/297605.html |title=冬春换季　长水机场推出新航线新服务新产品 |date=2014年10月26日 |accessdate=2014年10月26日|publisher=昆明信息港}}</ref>。

航站楼主楼地上3层（局部4层），分别为F1、F2、F3、F4；地下3层，分别为B1、B2、B3。F1是国内进港通道、国际进港通道和联检区、办公区等；F2是国际出发联检区、候机区与国内候机区、到达通道、办公区机房等；F3是值机大厅和国内出发安检区、候机区，值机大厅分为A、C、D、E、F、G、H、J、K区域办理登机手续，其中A~H为国内旅客办理乘机手续处，J、K为国际旅客办理乘机手续处；F4是陆侧餐厅；B1是行李提取大厅、迎客大厅和到达车道，行李提取大厅设有行李提取转盘15个，其中国内11个、国际4个；B2是与停车楼的连接厅、地铁车站站厅；B3是地铁车站站台、设备机电用房以及服务后勤用房。

==== 钢结构 ====
昆明新机场的代表性建筑结构为航站楼中央大厅里的7条形似彩带的钢箱梁，它支撑着航站楼的屋面系统。这七条彩带状的钢箱梁连同188根锥形钢管柱、738根幕墙柱及12根T型柱组成了昆明新机场航站楼主体钢结构工程，用钢量约2.9万吨。7根彩带寓意象征“七彩云南”。弯扭箱形钢彩带不仅是航站楼重要的装饰构件，而且是屋面网架的结构<ref>{{Cite web |url=http://www.ynjzzz.com/Info.aspx?ModelId=1&Id=1839 |title=钢结构新技术领航昆明新机场 |date=2010年12月24日 }}</ref>。

==== 行李系统 ====
[[File:昆明國際機場行李輸送帶.jpeg|thumb]]
昆明新机场采用的行李系统由昆明船舶设备集团公司生产，是大型机场中首次选用国产行李自动分拣系统。整套行李处理系统由始发行李处理系统、到达行李处理系统、中转行李处理系统、早到行李储存系统、自动分拣系统、人工编码系统等子系统组成，总计2523台设备<ref>{{Cite web |url=http://www.ynstc.gov.cn/kjxx/200911230005.htm |date=2009年11月23日 |title=我国大型机场行李处理系统首次实现国产化 }}</ref>。

==== 停车场 ====
停车场位于航站楼前，设有停车楼与停车场，停车楼面积9.8万平方米，地面停车场6.2万平方米，共有3800个停车位。其中出租车停车场面积约51760平方米，设计停车位961个、入口4个、出口4个。

==== 跑道 ====
昆明新机场飞行区一期工程新建两条长度为4500米及4000米、垂直间距1950米的远距平行跑道，其中东跑道宽60米，西跑道宽45米。东跑道长4500米，宽60米，是目前中国等级最高、长度最长的沥青混凝土跑道。配置双向I类精密进近仪表着陆系统及相应的助航灯光系统。飞行区工程建设规模按照4F标准规划设计，远期规划4条跑道。

====仪表着陆系统====
昆明长水国际机场现已安装并正式启用Ⅱ类[[仪表着陆系统|仪表着陆系统]]（俗称盲降系统）。Ⅱ类系统最低运行标准为能见度400米或跑道视程不小于350米；着陆最低标准的决断高度不低于30米。

运营初期，长水机场22号跑道和03号跑道均安装有Ⅰ类仪表着陆系统，最低运行标准为能见度800米，决断高度60米。为应对大雾天气，2013年1月28日，机场申请Ⅱ类盲降系统。2014年9月5日，机场Ⅱ、Ⅲ类盲降建设项目得到民航西南地区管理局的正式批准。建设内容及规模包括对长水机场22号和03号跑道现有仪表着陆系统、助航灯光系统等设施Ⅱ、Ⅲ类改造，03号跑道入口内移540米等。2014年11月，22号跑道仪表着陆系统Ⅱ类升级改造工程顺利完成投产校飞调试。2015年1月，工程竣工验收。2015年1月13日，机场完成22号跑道Ⅱ类飞行程序的试飞与低能见度运行程序的综合演练，试飞取得圆满成功。2015年4月10日，经中国民航局批复，昆明长水机场22号跑道Ⅱ类盲降系统正式启用。至此，昆明长水机场成为继北京、上海、广州、西安、成都等城市机场之后，又一个具备二类盲降的机场<ref name="ILS2">{{cite web|url=http://society.yunnan.cn/html/2015-04/10/content_3686193.htm|title=昆明长水机场实现二类盲降运行 飞机着陆更安全|publisher=云南网|date=2015-04-10|access-date=2015-04-11|archive-url=https://web.archive.org/web/20150417101206/http://society.yunnan.cn/html/2015-04/10/content_3686193.htm|archive-date=2015-04-17|dead-url=yes}}</ref>。

预计03号跑道的Ⅱ类系统将于2015年10月建成启用。

===空中快线===
昆明长水国际机场联合各航空公司、[[西双版纳嘎洒机场|西双版纳嘎洒机场]]、[[丽江三义机场|丽江三义机场]]、云南机场汽车服务公司，整合[[昆明|昆明]]、[[西双版纳|西双版纳]]、[[丽江|丽江]]三地航空及地面运输资源，在2014年冬春季启动昆明—西双版纳、昆明—丽江空中快线试点新产品，通过提高昆明—西双版纳、昆明—丽江航班密度，合理配置航班时刻资源，优化旅客乘机流程，实现空中快线“公交式”运营。

昆明—西双版纳、昆明—丽江基本满足全天每小时至少一个航班。

云南机场汽车服务公司为昆明—西双版纳、昆明—丽江空中快线旅客提供优惠票价，旅客提供乘坐空中快线航班相关凭证，可以享受单程20元的优惠价格，云南机场汽车服务公司在城市候机楼可提供预留机上座位、登机牌等服务。

昆明机场、西双版纳机场、丽江机场新增“昆明—西双版纳、昆明—丽江空中快线”标识标牌。昆明机场出发到西双版纳、丽江的旅客到E区E01、E02、E03、E18、E19、E20号专用柜台办理值机手续，使用24号、25号安检专用通道，并在紧邻安检通道的中指廊候机厅等候登机。从丽江、西双版纳到达昆明机场的旅客到9号、10号专用行李提取转盘提取托运行李。西双版纳、丽江机场均为空中快线配备标识标牌、专用值机柜台、专用安检通道、专用候机厅、专用行李提取转盘<ref name="2014冬春换季" />。
===通程航班===
海关总署批准中国东方航空自2015年4月28日起对经昆明长水国际机场中转进、出境旅客试行“通程航班”中转联运业务。至此，昆明长水国际机场将成为继[[北京首都国际机场|北京首都国际机场]]、[[上海浦东国际机场|上海浦东国际机场]]、[[广州白云国际机场|广州白云国际机场]]之后，全国第四个试行该模式的国际机场。“通程航班”联程中转模式下，中转进出境旅客将享受在始发站一次办理乘机手续，托运行李交运，经中转地海关对旅客手提及托运行李实施一次监管，行李至目的站直接提取的“一票到底，行李直达”的高效、便捷通关服务。

首批在[[巴黎|巴黎]]、[[迪拜|迪拜]]、[[曼谷|曼谷]]、[[新加坡|新加坡]]、[[香港|香港]]五个境外城市和[[北京|北京]]、[[上海|上海]]、[[广州|广州]]、[[西安|西安]]、[[成都|成都]]、[[深圳|深圳]]、[[武汉|武汉]]、[[郑州|郑州]]、[[杭州|杭州]]、[[银川|银川]]、[[兰州|兰州]]、[[哈尔滨|哈尔滨]]、[[丽江|丽江]]、[[西双版纳|西双版纳]]等十五个境内城市至昆明的国际、国内航线施行“通程航班”联程中转业务。5月1日，昆明机场将迎来首个“通程航班”<ref>[http://news.xinhuanet.com/2015-04/22/c_134174366.htm 昆明长水机场将开通“通程航班”中转联运业务]</ref>。

=== 机场高速 ===
[[File:The_toll_station_on_the_New_Kunming_Airport_Expressway.jpg|thumb]]
昆明新机场专用高速路西起昆明东绕城线与东三环虹桥立交连接线相交的两面寺立交，东至昆明长水国际机场，经金马村、瓦角村，跨越[[贵昆铁路|贵昆铁路]]，经过省司法警官职业学院西侧，全长14.779公里，全线按双向8车道高速公路设计，设计车速100公里/小时。机场高速路全线共设置两处枢纽型互通立交桥，即两面寺立交和大板桥立交，主线收费站共设置23个收费岛，24条收费车道。收费广场总长约470米，总宽度约133米。收费标准为：小车每辆单边收费十元。机场高速已实现全线贯通投入使用。<ref name="r1">{{Cite web |url=http://www.yn.chinanews.com.cn/html/tupiantoutiao/20110329/156344.html |title=昆明新机场主体工程7月底验收 |date=2011年3月29日 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>进出新机场的公路，除了机场专用高速路外，还可走昆嵩高速路和贵昆、呈黄等城市快速公路。

== 144小时(次日零时起六天)过境免签 ==
2019年1月1日起，昆明长水国际机场口岸正式实施[[中华人民共和国签证政策#144小时免签|144小时过境免签]]政策。全球共51个国家的公民，持本人有效国际旅行证件，以及144小时内已确定日期及座位、由昆明长水国际机场出境前往第三国(地区)的联程机票，可以于机场口岸申请144小时过境免签。获准过境免签的外国人士，自入境的次日零时起，可在昆明市行政区域停留最多144小时。<ref>{{Cite web|url=http://yn.yunnan.cn/html/2014-09/30/content_3389275.htm|title=昆明长水机场对全球53个国家开通72小时过境免签|date=2014年9月30日|access-date=2015年3月31日|archive-url=https://web.archive.org/web/20141010224823/http://yn.yunnan.cn/html/2014-09/30/content_3389275.htm|archive-date=2014年10月10日|dead-url=yes}}</ref>

适用144小时过境免签的国家如下：

*欧洲[[申根公约|申根签证协议]]国家（24个）：{{Flag|奥地利}}、{{Flag|比利时}}、{{Flag|捷克}}、{{Flag|丹麦}}、{{Flag|爱沙尼亚}}、{{Flag|芬兰}}、{{Flag|法国}}、{{Flag|德国}}、{{Flag|希腊}}、{{Flag|匈牙利}}、{{Flag|冰岛}}、{{Flag|意大利}}、{{Flag|拉脱维亚}}、{{Flag|立陶宛}}、{{Flag|卢森堡}}、{{Flag|马耳他}}、{{Flag|荷兰}}、{{Flag|波兰}}、{{Flag|葡萄牙}}、{{Flag|斯洛伐克}}、{{Flag|斯洛文尼亚}}、{{Flag|西班牙}}、{{Flag|瑞典}}、{{Flag|瑞士}}。

*欧洲其他国家（13个）：{{Flag|俄罗斯}}、{{Flag|英国}}、{{Flag|爱尔兰}}、{{Flag|塞浦路斯}}、{{Flag|保加利亚}}、{{Flag|罗马尼亚}}、{{Flag|乌克兰}}、{{Flag|塞尔维亚}}、{{Flag|克罗地亚}}、{{Flag|波黑}}、{{Flag|黑山}}、{{Flag|阿尔巴尼亚}}、{{Flag|马其顿}}。

== 绿色机场 ==
昆明新机场在建造设计中实践绿色机场的概念。按照中国民航局把昆明新机场建设成为“节约型、环保型、科技型、人性化的绿色机场示范工程”的要求，根据民用机场的功能特征，按照国家《绿色建筑评价标准》星级要求，在绿色昆明新机场建设中着重突出节约、环保、科技和人性化的基本思路。主要体现在：航站楼采用自然通风设计，减少空调耗能；优化的自然采光设计，减少照明耗能；采用幕墙节能设计策略和外遮阳设计；采用新型空调系统，减少能源消耗；积极利用可再生能源等。<ref name="r2">{{Cite web|url=http://www.newkma.com/a/xiangmugaikuang/20100430_5.html|title=昆明新机场规划设计创新|date=2010年5月8日|deadurl=yes|archiveurl=https://web.archive.org/web/20120313115818/http://www.newkma.com/a/xiangmugaikuang/20100430_5.html|archivedate=2012年3月13日}}</ref>

== 争议 ==

自启用以来，已由于大雾和冰雪天气造成多次大面积延误，因而有不少对选址的质疑。根据一些当地民众反映，在机场修建以前，该地区就一直是多雾、大风、雷雨集中区域，当地“长水”名称的来源可能与其气候特征有关。一些提出质疑的人士亦建议重新启用[[昆明巫家坝国际机场|昆明巫家坝国际机场]]<ref>{{Cite web |url=http://society.yunnan.cn/html/2013-01/06/content_2563588.htm |title=昆明长水机场大雾 引发巫家坝“复航”猜想 |date=2014年1月20日 |access-date=2014年12月20日 |archive-url=https://web.archive.org/web/20160306122934/http://society.yunnan.cn/html/2013-01/06/content_2563588.htm |archive-date=2016年3月6日 |dead-url=yes }}</ref><ref>{{Cite web |url=http://news.163.com/air/14/0120/11/9J1F98RG00014P42.html |title=云南政协委员建议保留巫家坝机场为备降机场 |date=2014年1月20日}}</ref>。
对此，昆明长水国际机场负责人回应，长水机场的选址是经过国内机场建设、气象方面的专家反复论证的，在选址的时候也清楚长水的气象条件比巫家坝要复杂，但这个机场的规划吞吐量非常大，这个场址符合云南民航长远发展的选择。并且长水机场的气象条件实际上相对好于其他一些全国大型机场<ref>{{Cite web |url=http://www.chinanews.com/sh/2013/11-30/5565568.shtml |title=云南民航官员：不能因为大雾而怀疑长水机场选址 |date=2013年11月30日}}</ref>。另外，长水机场[[仪表着陆系统|Ⅱ类盲降系统]]已正式投入使用。该系统的启用，使长水机场能够在低能见度的条件下保证正常运行，因大雾天气造成大面积延误的情况将得到改观<ref name="ILS2"/>。

另外，在机场启用最初两年时间内，机场高速公路、停车场及候机大厅等曾出现一些问题，引起媒体的关注<ref>{{Cite web|url=http://www.shxb.net/html/20120814/20120814_321070.shtml|title=长水停车场现塌陷 机场建设方：天气变化所致|date=2012年8月14日|deadurl=yes|archiveurl=https://web.archive.org/web/20131230232526/http://www.shxb.net/html/20120814/20120814_321070.shtml|archivedate=2013年12月30日|df=}}</ref><ref>{{Cite web |url=http://news.dichan.sina.com.cn/2013/06/09/756060.html |title=暴雨突袭昆明机场候机厅内惊现水帘洞 |date=2013年6月9日}}</ref><ref>{{Cite web |url=http://news.carnoc.com/list/233/233313.html |title=长水机场问题不断　网友调侃：长水“很水” |date=2012年9月13日}}</ref>。但近年来，机场运营管理已逐步进入正轨，没有发生更多类似事件。
