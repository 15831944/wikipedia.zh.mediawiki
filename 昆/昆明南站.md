'''昆明南站'''，位于[[云南省|云南省]][[昆明市|昆明市]][[呈贡区|呈贡区]]东部龙潭山脚下，是一座西南地区铁路客运枢纽站。车站距离昆明市行政中心约3km、[[滇池|滇池]]约7km，距[[昆明站|昆明站]]约28km（铁路里程）。车站主要办理[[南昆客运专线|南昆客运专线]]、[[昆玉铁路|昆玉]]、[[渝昆铁路|渝昆]]、[[沪昆客运专线|沪昆客运专线]]客车作业。2011年6月16日，昆明南站正式开工建设。主体工程于2016年7月16日完工，于2016年12月28日与沪昆客运专线，南昆客运专线、昆玉河铁路同步开通运营<ref>{{Cite web |url=http://society.yunnan.cn/html/2011-06/24/content_1682755.htm |title=昆明新火车南站形似孔雀开屏 |date=2011年6月24日 |publisher=云南网 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref>{{Cite web |url=http://fj.people.com.cn/BIG5/n2/2016/1208/c234960-29433103.html |title=明年1月5日南鐵實行新運行圖　福州去昆明僅11小時|date=2016年12月8日 |publisher=福州新聞網}}</ref><ref name=yn>{{cite news|url=http://yn.yunnan.cn/html/2016-12/26/content_4669511.htm|title=要买北京、上海、深圳还需再等等|publisher=云南网|date=2016-12-26|accessdate=2016-12-26|6=|deadurl=yes|archiveurl=https://web.archive.org/web/20161226215941/http://yn.yunnan.cn/html/2016-12/26/content_4669511.htm|archivedate=2016年12月26日|df=}}</ref>。昆明南站是中国西南部规模最大的铁路客运站。

==历史==
2016年12月28日昆明南站启用首日，该站于当日10时开行往贵阳北、南宁、玉溪站方向的动车组列车，三班首发列车分别是开往贵阳北的G4136次、开往南宁的D4208次以及开往玉溪的D9109次<ref>{{cite news|url=http://news.ifeng.com/a/20161228/50488235_0.shtml|title=云南接入全国高速铁路网 铁路建设创多项世界之最 |publisher=人民网 |date=2016-12-28 |accessdate=2016-12-28 |}}</ref>。

2017年9月4日，[[南昆客运专线|南昆客运专线]]昆明南至昆明段开通运营，昆明南站与[[昆明站|昆明站]]间首次实现动车组列车的连接。<ref>{{cite news|url=http://www.ynta.gov.cn/Item/33789.aspx|title=今起昆明站至昆明南站间开通动车|publisher=云南日报|date=2017-09-04|accessdate=2017-09-04}}</ref>

==车站设计==
[[File:Kunming_South_Railway_Station_Concourse.jpg|thumb]]
[[File:Peacock_flower_bed_at_Kunming_South_Railway_Station_(20180215083808).jpg|thumb]]
[[File:Ticket_office_No.3_of_Kunming_South_Railway_Station_(20180214185022).jpg|thumb]]
[[File:Kunming_South_Railway_Station_Exit_Corridor.jpg|thumb]]
[[File:Exits_1_and_2_of_Kunming_South_Railway_Station_(20180214185353).jpg|thumb]]
车站按16台30线规模设计。沪昆客专列车布置在城市一侧，按9条到发线、9个站台面设计；渝昆、昆玉铁路列车布置在车站中间，按7条到发线、7个站台面设计；环滇列车按4条到发线、4个站台面设计；云桂铁路列车布置在靠山一侧，按10条到发线、10个站台面设计。车站旅客高峰小时客流量12054人。<ref name=r1>{{cite journal en |author = 黄智勤|year = 2012|title = 华美的现代民族建筑乐章——昆明南站设计|journal =《华中建筑》|issue = 06|pages = 64}}</ref>

站房设计采用高架站上进站下出站形式，建筑面积达12万平方米，建筑层数：地上三层；地下二层。

昆明南站由中铁第四勘察设计院设计，以“雀舞春城，美丽绽放”为主题，车站外部造型选取孔雀造型，主入口为具有云南少数民族特色的坡顶仿木构雨棚。<ref name="r1" />

=== 相关设施 ===
昆明南站配套建设昆明南动车运用所，以及昆明南车辆段、昆明南客车技术整备所、昆明南客机整备所（“一段两所”）。2016年6月22日，昆明南动车所竣工。2017年11月28日，“一段两所”工程开工。<ref>{{cite web|title=昆铁枢纽改造今日开工|publisher=昆明日报|date=2009-06-30|author=孙娅}}</ref><ref>{{cite web|title=云南高铁昆明南动车运用所竣工|date=2016-06-22|accessdate=2017-11-30|publisher=中国新闻网|url=http://www.chinanews.com/sh/2016/06-22/7913678.shtml}}</ref><ref>{{cite web|title=十一局五公司云桂铁路“一段两所”工程正式动工|author=侯利 刘娅红|publisher=中铁建|date=2017-11-30|accessdate=2017-11-30|url=http://www.crcc.cn/art/2017/11/30/art_98_663537.html}}</ref>

==旅客列车目的地==

根据《中国铁路总公司关于公布2016年底调整列车运行图的通知》，2017年1月5日铁路运行图调整后，昆明南站原计划开行列车74对。但根据沪昆高铁云南段和云桂铁路云南段开通运营新闻发布会的消息，自2017年1月5日起，昆明南站计划每天开行37对动车组列车，包括经沪昆高铁的高速动车组列车18对、经南昆客专的普通动车组列车16对，以及经昆玉河铁路的普通动车组列车3对，其中终到北京西2对、终到上海虹桥3对、终到南京南2对（一对为经沪昆高铁、[[京沪高铁|京沪高铁]]、另一对为经沪昆高铁、[[宁杭高铁|宁杭高铁]]）、终到贵阳北5对、终到长沙南1对、终到深圳北2对、终到南昌西1对、终到济南西1对、终到武汉1对、终到广州南12对、终到南宁东3对、终到北海1对、终到玉溪3对<ref name=yn/>。原定开行前往其他城市方向的列车，包括一对终到[[北京西站|北京西站]]的[[卧铺动车组列车|卧铺动车组列车]]<ref>{{Citation|date = 2016-10-29|title = 铁总运函〔2016〕836号　中国铁路总公司关于公布2016年底调整列车运行图的通知|publisher =[[中国铁路总公司|中国铁路总公司]]|pages=171}}</ref>则暂缓至2017年4月16日开行。2019年1月5日起，昆明局优化了昆明站至昆明南站间的客车开行方案，每天开行客车44对，其中包括普速列车<ref>{{cite news|url=http://www.yn.xinhuanet.com/original/2018-12/29/c_137706992.htm|title=1月5日起铁路调整列车运行图 昆明局集团公司图定客车194.5对 |publisher=新华网 |date=2018-12-29 |accessdate=2019-01-13|}}</ref>。
