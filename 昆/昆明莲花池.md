'''莲花池'''位于云南省，昆明北部，临近[[云南民族大学|云南民族大学]]。根据史书记载，唐宋时期就有泉水汇流于此，宋代大理国国王也曾在此曲水流觞。明朝初年成为「滇阳六景」之一,有「龙池跃金」的美誉<ref>{{cite journal |author1=陈伟 |title=莲花池与陈圆圆 |journal=《云南档案》 |issue=2010年12期}}</ref>。经历代修缮，元代将其列入昆明八景<ref name="文化涟漪">{{cite journal |author1=谌强 |title=让莲花池荡起文化的涟漪 |journal=[[中华文化画报|中华文化画报]] |issue=2014年6期 |accessdate=2019-03-29}}</ref>{{rp|1}}。莲花池内有五颗龙眼，百年以来一直清流涌流。池水满时满入盘龙江。“[[秦淮八艳|秦淮八艳]]”之一的陈圆圆自溺于莲花池，一代佳丽香消玉殒，从此繁花落尽。

==历史沿革==
莲花池根据据明代史书《[[云南府志|云南府志]]》记载，商山之麓“下有冷泉，名莲花池，浴之可以去风疾”。翻译过来就是说百姓相信在莲花池里洗浴可以治聊疾病。吴三桂曾经选址此地，兴建“安阜园”为宠幸妃子陈圆圆的住宅。当年吴三桂在昆明五华山办公，为了陈圆圆专门修建了一条2千米长的栈道，从他办公处，直通莲花池。
19世纪50年代开始，莲花池逐渐凋敝、水体污染加重<ref>{{cite journal|title=十亩荷花鱼世界 半城杨柳拂楼台——赏滇中胜景昆明莲花池公园|last1=许成启|url=http://www.cqvip.com/qk/96604x/2016012/671439107.html|date=|journal=防护林科技|issue=2016年第12期|doi=|year=|volume=|page=|language=zh|pmid=|website=|access-date=2019-03-29}}</ref>。

1982年，莲花池公园建成，面积15700平方米。池北有儿童公园，免费对外开放<ref>{{cite book|title=莲花池公园|url=http://xuewen.cnki.net/read-R2006090020005180.html|language=zh|access-date=2019-03-29|last=罗亚蒙|first=《中国历史文化名城大辞典·下》|publisher=人民日报出版社|year=1998年|isbn=|location=|pages=}}</ref>。2006年，昆明启动历史文化名城建设项目和城中村改造试点<ref name="文化涟漪"/>{{rp|1}}。

五华区政府于2006年12月开始重建，于2008年9月29日再次开园<ref name="植被"/>{{rp|1}}。人们于饭后步入莲花池所欣赏当年吴三桂“金屋藏娇”之处。

== 吴三桂 ==
吴三桂：（1612年—1678年）字长伯。明辽东人，祖籍江苏高邮。康熙二十年（1681年）昆明被围，吴世璠自杀，余众出降。<ref>{{cite web |author1=中国历史故事网 |title=吴三桂简介,吴三桂和陈圆圆的故事 |url=http://www.gs5000.cn/gs/lishirenwu/17731.html |website=www.gs5000.cn}}</ref>

== 陈圆圆 ==
