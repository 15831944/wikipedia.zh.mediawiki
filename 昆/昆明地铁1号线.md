'''昆明地铁1号线'''是[[昆明地铁|昆明地铁]]运营中的一条[[地铁|地铁]]线路，该线是贯穿昆明新老市区南北的骨干线路，全长约41.4千米，设站25座。试验段工程于2009年开工建设，一期工程（环城南路站-大学城南站）已于2010年5月1日开工建设。1号线一期将分段开通运营，南段（晓东村站-大学城南站）于2013年5月20日开通运营，北段（晓东村站-环城南路）2014年4月30日开通后连同[[昆明轨道交通2号线|2号线]]一期一起[[贯通运营|贯通运营]]。

== 建设 ==
===试验段===
*2009年7月13日，昆明首先在[[官渡区|官渡区]]昆洛路动工兴建[[城市轨道交通|城市轨道交通]]试验段。昆洛路试验段全长4.98公里，采用高架桥方式架设。线路起于世纪城站，沿彩云路（新昆洛路）中央隔离带布设，经体育城站、体育城南站、穿越广福立交后沿道路东侧走线，到达五腊村站，共有4站3区间。采用中岛站台车站，两侧设天桥进站。<ref>{{Cite web |url=http://news.kunming.cn/km-news/content/2009-07/13/content_1918726.htm |title=昆明城轨昆洛路试验段正式动工 1年半后可建成 |date=2009年7月13日 }}</ref>

===一期工程===
[[File:Kunming_Metro_in_Kunluo_road.jpg|thumb]]
1号线一期（环城南路—大学城南）和2号线一期（环城南路—北部汽车客运站）组成昆明地铁首期工程，全长42.1公里（其中高架段11.7公里，地下段29.4公里），全线设站31座（地下站25座，高架站6座）。首期工程线路由北市区北部汽车客运站向西转入北京路，沿北京路经主城下穿[[昆明火车站|昆明火车站]]之后向东南方向转入春城路，经会展中心、[[巫家坝国际机场|巫家坝国际机场]]后，在世纪城附近沿彩云南路路中向南延伸，最终在[[呈贡|呈贡]]大学城片区广电大学设置终点站。2011年7月，首期工程增设三站，1号线增设得胜桥站和金碧广场站，2号线增设昆明火车站站。<ref>[http://www.shxb.net/html/20110701/20110701_283667.shtml 地铁1号2号线增设三个站] {{Wayback|url=http://www.shxb.net/html/20110701/20110701_283667.shtml |date=20110923234017 }}</ref>
*2010年5月1日，1、2号线首期工程全面开工建设。<ref name=r2>[http://news.sina.com.cn/c/2010-05-02/165120194281.shtml 昆明地铁1号2号线首期工程开工]</ref>
*2010年6月3日，“春城一号”和“春城五号”两台[[盾构机|盾构机]]始发，昆明地铁开始进入隧道施工阶段。
*2011年3月28日，1号线南段开始铺轨。<ref name=r5>[http://video.yunnan.cn/html/2011-03/30/content_1550427.htm 昆明地铁首期工程铺轨施工启动]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

== 规划 ==
{{YN Future infrastructure}}
*1号线二期（西延线）：线路起于环城南路，止于西昌路，全长4.2公里，全线按地下线敷设；共设地下车站4座，建成后与1号线一期工程贯通运营。
*1号线三期（北延线）：线路起于西昌路，止于省政法学校，全长3.7公里，全线按地下线敷设；共设地下车站3座，建成后与1号线二期工程贯通运营。
*1号线支线：线路起于春融街站，止于[[昆明南站|昆明南站]]，全长5.2公里，全线按地下线敷设；共设地下车站4座，建成后与1号线一期线路同步运营。
*1号线南延线：线路起于广电大学，止于公园北路，全长7.3公里，全线按高架线敷设；共设高架车站3座，建成后与1号线二期工程贯通运营。

==事故==
*2013年1月8日上午9时9分，昆明地铁首期工程1号线南段列车在空载试运行过程中，列车行至离[[斗南站|斗南站]]约500米段时，第一节车厢脱轨，导致司机室暖风装置坠落，造成值班司机一死一轻伤。<ref>{{Cite web |publisher = 网易|url=http://news.163.com/13/0108/14/8KN2BN2T00014JB6.html|title=昆明地铁空载试运行时脱轨 致1死1伤 |date=2013年1月8日}}</ref>
