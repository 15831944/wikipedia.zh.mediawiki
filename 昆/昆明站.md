'''昆明站'''位于[[中华人民共和国|中国]][[云南省|云南省]][[昆明市|昆明市]][[官渡区|官渡区]]北京路南端，于1966年启用，为[[昆明铁路局|昆明铁路局]]管辖的[[一等站_(中铁)|一等站]]，是[[沪昆铁路|沪昆铁路]]、[[成昆铁路|成昆铁路]]、[[南昆铁路|南昆铁路]]以及[[南昆客运专线|南昆客运专线]]的终点。昆明站北距[[成都站|成都站]]1100公里，东距[[上海站|上海站]]2690公里，东南距[[南宁站|南宁站]]828公里。

==历史==
国民政府规划的[[米轨|米轨]]叙昆铁路（叙府/宜宾至昆明）于1939年开工建设，但工程受到战争影响于1942年停工，仅完成从昆明到沾益156公里的铺轨<ref name="neikun">{{Cite news|title=百年梦圆内昆线——写在内昆铁路全线铺通之际|url=http://www.people.com.cn/GB/jinji/31/179/20011004/574956.html|publisher=人民网|date=2001-10-04}}</ref>。1949年后，滇黔线和内昆线的建设被纳入[[第二个五年计划_(中国)|国民经济建设第二个五年计划]]，1958年开工。其中，昆明至沾益段利用原叙昆铁路路基，但在施工过程中改为[[准轨|准轨]]。1962年原滇黔线和内昆线的贵阳-树舍-昆明段合并为[[贵昆铁路|贵昆铁路]]。贵昆铁路全线于1966年3月建成<ref name="guikun">{{cite web|title=贵昆铁路|url=https://www.crcc.cn/art/2015/1/7/art_281_71056.html|publisher=中国铁建股份有限公司|date=2015-01-07}}</ref>，昆明火车站随即投入使用<ref name="general">{{cite web|title=昆明火车站|url=https://www.cbi360.net/hyjd/20170417/69538.html|publisher=建筑网|date=2017-04-17}}</ref>，为通过式车站。车站设有一座面积6420平方米的砖混结构站房，内设3个旅客候车室共1415平方米。车站站场三台八线，其中客车到发线5股、货车到发线及正线共2股、机走线1股<ref name="yntl">{{cite book|author=云南省地方志编纂委员会|title=云南省志 卷三十四 铁道志|year=1994|publisher=云南人民出版社|isbn=7-222-01548-5}}</ref>{{rp|92}}。

1971年7月1日，[[成昆铁路|成昆铁路]]开通运营。当日从昆明站开行始发列车，在[[西昌站|西昌站]]与成都发来的列车相遇<ref>{{cite web|title=一寸铁路一寸血 她们也是成昆铁路缔造者 却鲜有人知（组图）|url=http://society.yunnan.cn/html/2018-01/24/content_5056186.htm|publisher=云南网|date=2018-01-24}}</ref>。至1980年代初，昆明站每日开行6对列车，主要发往北京、上海、贵阳、攀枝花、中谊村、金马村方向<ref name="general"/>。1997年11月30日[[南昆铁路|南昆铁路]]建成通车。

2003年7月，昆明铁路客站开始改扩建工程，工程包括新建站房和站台改造。在车站进行改扩建工程期间，旅客使用临时候车室进出站<ref name="interior">{{cite news|title=豪华气派 耳目一新 新昆明站新年展新姿(组图)|url=http://news.sina.com.cn/o/2004-12-27/12464640433s.shtml|publisher=新浪网|date=2004-12-27}}</ref>。工程于2004年12月28日竣工，并于2005年春节前投入使用<ref name="station building">{{cite news|title=综述：昆明火车新站落成 云南铁路建设进入高潮|url=http://www.chinanews.com/news/2004/2004-12-29/26/522060.shtml|publisher=中新网|date=2014-12-29}}</ref><ref name="change">{{cite news|title=昆明铁路客站昨落成(图)|url=http://news.sohu.com/20041229/n223702673.shtml|publisher=搜狐网|date=2004-12-29}}</ref>。2007年初：昆明站南站房建成，成為中国西南部首個雙面火車站<ref>
[https://web.archive.org/web/20070220060053/http://www.yn.xinhuanet.com/newscenter/2007-01/26/content_9147978.htm 西南首個雙面火車站春運啟用]</ref>。但由于配套问题，南站房从未启用<ref>{{cite news|title=昆明站南广场建成10年未使用 待地铁2号线二期通车后可直抵火车站|url=https://new.qq.com/omn/20180108/20180108G0LXXN.html|publisher=腾讯|date=2018-01-08}}</ref>。

2017年9月4日[[南昆客运专线|南昆客运专线]]昆明南至昆明段开通运营，昆明站首次开行动车组列车<ref>{{cite news|url=http://www.ynta.gov.cn/Item/33789.aspx|title=今起昆明站至昆明南站间开通动车|publisher=云南日报|date=2017-09-04|accessdate=2017-09-04}}</ref>。同时站内开启快速换乘通道，站内换乘不需再安检<ref name="emu">{{cite news|title=昆明站至昆明南站、玉溪站正式开行动车 站内换乘不用再安检|url=http://xw.kunming.cn/a/2017-09/04/content_4760451.htm|publisher=昆明信息港|date=2017-09-04}}</ref>。昆明站在2018年[[昆楚大铁路|昆楚大铁路]]开通前夕对候车室进行了重新装修<ref>{{cite news|title=【直击】昆明站候车室换新颜|url=http://society.yunnan.cn/html/2018-06/28/content_5273457.htm|publisher=云南网|date=2018-06-28}}</ref>。

==车站设施==
===站房===
{{multiple image|align=right|direction=vertical|width=220
|image1=Kunming Railway Station Ticket Hall.jpg|caption1=售票大厅（第一售票区）
|image2=Kunming Railway Station Waiting Room 4.jpg|caption2=第四候车室
}}
昆明站由主站房、站前广场和高架平台、跨线式高架候车室、地下停车场等组成。昆明站北站房（主站房）于2005年投入使用，总面积二万七千多平方米。进站大厅屋檐为“人”字形的玻璃墙，为傣族建筑风格<ref name="interior"/>，并获得2005年[[鲁班奖|鲁班奖]]<ref>{{cite web|url=http://old.zgjzy.org/lbjsjk/proj.asp?a=2005061|title=昆明站主站房|publisher=鲁班奖数据库}}</ref>。昆明站南站房于2006年完工，高4层，建筑面积11502平方米<ref>{{cite news|title=昆明站南站房有望元旦前投用　我省建成西南地区首个“双面”火车站|url=http://news.sina.com.cn/c/2006-11-15/100110508294s.shtml|publisher=新浪网|date=2006-11-15}}</ref>。但截至2018年南站房从未被使用。

站房设有4个普通候车室、3个贵宾候车室、1个母婴候车室和1个军人候车室<ref name="change"/>，中央为候车通道。站房二层设有4个候车厅入口，通道东侧为第二候车厅，设有八九百个座位，为茶室；西侧可到残疾人、母子候车厅。高架候车区横跨站场，设有第三候车厅与第五候车厅（原第四候车室），每个设有1000多个座位。第五候车室原为“农民工候车室”，后因该名称含有歧视意味，遂恢复原名<ref>{{cite news|title=昆明火车站放弃使用“农民工候车室”标牌(图)|url=http://news.163.com/08/0123/16/42TFJQ2F0001125G.html|publisher=网易新闻|date=2008-01-23}}</ref>。2017年9月昆明站开行动车组列车后，第五候车室内的部分区域被用作动车组列车专用候车区<ref name="emu"/>。第三与第五候车厅间的中央通道两侧共植有6棵高大的棕榈树，棕榈树下还种植多种绿色植物和鲜花<ref name="interior"/>，后改为民族风情购物长廊。昆明站南站房竣工后新增第四候车室（原第五、第六候车室），位于长廊末端。候车室最高聚集人数为七千人，日乘降旅客可达十一万人。<ref name="station building"/><ref>{{cite news|title=新昆明铁路客站28日竣工|url=http://news.yntv.cn/category/10101/2004/12/29/2004-12-29_178002_10101.shtml|publisher=云南电视台|date=2004-12-29}}</ref>

昆明站候车室每天23:45关闭，不允许进站候车、购票取票，次日早晨才会开放。昆明站原在站房进站口设有一道安检<ref name="interior"/>，[[2014年昆明火车站暴力恐怖袭击事件|2014年昆明火车站暴力恐怖袭击事件]]后在站前广场增设一道安检<ref>{{cite news|title=昆明火车站广场围起了铁网，增设了安检设备|url=https://www.thepaper.cn/newsDetail_forward_1306769|publisher=澎湃新闻|date=2015-03-01}}</ref>。2017年位于站房候车厅的安检被撤除<ref>{{cite news|title=花8块钱 22分钟从昆明站坐到昆明南站|url=http://m.xinhuanet.com/2017-09/05/c_136584372.htm|publisher=新华网|date=2017-09-05}}</ref>。站前广场设有第二售票区<ref>{{cite news|title=昆明火车站增设30台自助售票机 临时候车棚没票也能进|url=http://news.ifeng.com/gundong/detail_2014_01/09/32852413_0.shtml|publisher=凤凰网|date=2014-01-09}}</ref>。

===站场===
昆明站建有6座候车站台，共有13条股道<ref>{{cite web|title=每天4000余人乘地铁到火车站 市民期待南广场开通|url=http://xw.kunming.cn/a/2014-06/24/content_3598527.htm|publisher=昆明信息港|date=2014-06-24}}</ref>。站房上方采用弧形无柱风雨棚<ref name="general"/><ref name="interior"/>，该雨棚工程荣获中国建筑钢结构金奖<ref>{{cite news|title=昆明火车站无站台柱雨棚工程|url=http://www.crec4.com/content-1160-23593-1.html|publisher=中国中铁四局集团|date=2009-05-08}}</ref>。

==事件==
===知青卧轨事件===
1978年12月28日，昆明的知青们为[[知青大返城|争取返城]]，在昆明站以东的羊方凹集体卧轨示威，导致数十对客运和货运列车受阻。请愿团与政府僵持三天之后，党中央国务院于12月31日同意知青请愿团赴京反映情况，但请愿团人数需在30人以内<ref>{{cite news|title=1978年云南知青为返城集体卧轨 数十列车受阻|url=http://news.ifeng.com/history/zhiqing/ziliao/detail_2012_04/06/13688442_0.shtml|publisher=凤凰网|date=2012-04-06}}</ref><ref>{{cite news|title=“知青之父”追忆知青大返城始末|url=http://dangshi.people.com.cn/GB/120281/9393311.html|publisher=人民网党史频道|date=2009-06-01}}</ref>。

===袭击事件===
{{Main|2014年昆明火车站暴力恐怖袭击事件}}
2014年3月1日晚21时12分，五名[[新疆|新疆]]籍[[恐怖分子|恐怖分子]]持刀冲入昆明站砍杀路人。随后大批民警赶至现场，火车站内传出枪声，当场4名暴徒被击毙，而其餘一名暴徒被擊傷，事件发生于21时12分，但直到23时还有人求救，警方起初没有估计到事件的严重性，未能及时安排警力以反恐处理事件。这次事件发生后已造成31人死亡，142人受伤。“3.01”事件定性为严重暴力恐怖事件。<ref>{{cite web |url = http://news.xinhuanet.com/2014-03/01/c_119563949.htm |language=zh-hans |title=一伙歹徒在昆明火车站袭击路人 已有数名被处置 |website = 新华网 |date=2014-03-01|accessdate=2014-03-02 }}</ref><ref>{{cite web |url = http://news.china.com.cn/rollnews/news/live/2014-03/01/content_25221068.htm |title=一伙歹徒持械冲进昆明火车站广场见人就砍 |language=zh-hans |publisher=中国网 |date=2014-03-01 |accessdate=2014-03-02 }}</ref><ref>{{cite web |url = http://news.sina.com.cn/c/2014-03-02/033829599206.shtml |title=昆明警方当场击毙5名暴徒 仍在围捕其余暴徒 |language=zh-hans |publisher=新浪新闻 |accessdate=2014-03-02 }}</ref>
