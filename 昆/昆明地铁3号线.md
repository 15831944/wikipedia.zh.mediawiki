'''昆明地铁3号线'''是[[昆明地铁|昆明地铁]]的一条线路，为昆明主城内东西方向的骨干线，线路经过昆明大部分最古老的商业街区。一期线路起点位于主城西部[[石咀站_(地铁)|石咀站]]，终点位于主城[[东部汽车站站|东部汽车站站]]，线路全长19.16公里，其中高架线1.17公里，地下线和过渡段长17.99公里。共设车站17座，全地下车站，平均站间距1.15公里。3号线二期工程將延伸到[[西山森林公園|西山森林公園]]，设西山公园站、车家壁站、普坪村站3个站，接入3号线一期工程石咀站。

3号线与[[昆明地铁1号线|1号线]]、[[昆明地铁2号线|2号线]]、[[昆明地铁4号线|4号线]]、[[昆明地铁5号线|5号线]]、[[昆明地铁6号线|6号线]]换乘，全线共设置换乘车站5座，分别为[[潘家湾站|潘家湾站]]与1号线换乘、省五一路站与5号线换乘、[[东风广场站|东风广场站]]与2号线换乘、大树营站与4号线换乘和[[东部汽车站站|东部汽车站站]]与6号线换乘。

==历史==
*2010年8月，3号线试验段开工建设。
*2011年7月起，3号线各站点分批陆续开始围挡施工。
*2012年4月14日，[[盾构机|盾构机]]从眠山站始发，3号线进入掘进阶段。<ref>{{Cite web |url=http://www.kmgdgs.com/content/2012-04/16/content_2916132.htm |title=小渔村站顺利封顶 地铁3号线进入掘进阶段 |access-date=2012-06-16 |archive-url=https://web.archive.org/web/20120626231906/http://www.kmgdgs.com/content/2012-04/16/content_2916132.htm |archive-date=2012-06-26 |dead-url=yes }}</ref>
*2016年3月19日，随着盾構機從車家壁站隧道破土而出，3號綫隧道全綫貫通。<ref>[http://news.163.com/16/0319/20/BII16JAE00014AEE.html 昆明地铁3号线全线贯通 2017年上半年将投入运营]</ref>
*2016年8月24日，石咀車輛段主體完工。<ref>{{Cite web |url=http://www.kmgdgs.com/node_30801_5.htm |title=昆明地铁3号线石咀车辆段主体完工 |access-date=2016-10-01 |archive-url=https://web.archive.org/web/20160730065210/http://kmgdgs.com/node_30801_5.htm |archive-date=2016-07-30 |dead-url=yes }}</ref>
*2016年11月14日，3号线一期17个车站主体结构全部完工；二期3个车站主体结构全部完成，铺轨工作全部完成。<ref>[http://news.163.com/16/1114/15/C5RFNKTL000187VG.html 昆明地铁3号线17个车站主体结构全部完工 明年上半年通车]</ref>
*2017年8月29日，3号线开通试运营。<ref>{{cite web|url=http://society.yunnan.cn/html/2017-08/29/content_4923411.htm|title=昆明轨道交通3号线、6号线一期开通试运营|publisher=云南网|date=2017-08-29|accessdate=2017-08-29|archive-url=https://web.archive.org/web/20170829120214/http://society.yunnan.cn/html/2017-08/29/content_4923411.htm|archive-date=2017-08-29|dead-url=yes}}</ref>
