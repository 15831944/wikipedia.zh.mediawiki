'''驚嘆號'''（'''！'''，也稱為'''-{zh-cn:感叹号; zh-tw:感嘆號; zh-hk:驚嘆號;}-'''或'''-{zh-cn:惊叹号; zh-tw:嘆號;}-'''）用於句子結尾，表示驚嘆、感歎或號嘆。[[中华民国|中华民国]][[中华民国教育部|教育部]][[国语推行委员会|国语推行委员会]]《重订标点符号手册》指明“用在感嘆、命令、祈求、勤勉等語句之後。”<ref>中華民國教育部国语推行委员会，《重订标点符号手册》，中华民国八十六年三月台湾学术网络三版。</ref>中华人民共和国的《标点符号用法》中也指出“感嘆句末尾的停頓，用嘆號。”“語氣強烈的祈使句末尾，也用嘆號。”“語氣強烈的反問句末尾，也用嘆號。”<ref>中华人民共和国国家标准，《标点符号用法》，1995年12月13日发布，1996年6月1日实施。</ref>

== 源頭 ==
一般认为「!」这一符號源自[[拉丁語|拉丁語]]的“io”，意即喜悅的表達。它可能是由於兩個字母拼成一個單音，或字母i放在[[句號|句號]]之上而成。

== 變體 ==
如同在[[西班牙文|西班牙文]]中尾端有“?”的句首需要加“¿”，在西班牙文中尾端有「!」的句子前面也必須有倒置的變體，例：¡OK!

嘆號（！）可與問號（？）結合成[[疑問驚嘆號|疑問驚嘆號]]（‽，或?!）。
== 其他用法 ==
[[File:Achtung.svg|thumbnail]]圍繞的感歎號]]

* 過量使用感歎號，就如過分的情感，[[文學|文學]]上會視為惡劣。
*在廣告中，經常會使用感歎號以吸引人的注意，有些牌子或書籍名稱甚至本身有附有感歎號，如 [[雅虎|Yahoo!]]、《[[蘿球社！|蘿球社！]]》等。
* 在[[數學|數學]]上，!是[[階乘|階乘]]的符號。對於所有正整數n，<math>n!=1\times2\times3\times...\times n</math>；而规定<math>0!=1</math>。
* 一些[[編程語言|編程語言]]使用「!=」表示不等於。這時感歎號稱為“bang”或“shriek”。
* 在[[Geek_Code|Geek Code]]，「!」表示該名[[極客|極客]]不想回答相關範疇的事。
* 在[[國際象棋|國際象棋]]的棋譜中，「!」表示一步好棋，「!!」表示極妙之着。
*世界上有部分於地名中夾帶驚嘆號的案例，如[[英國|英國]][[德文郡|德文郡]]的「[[韋斯特沃德霍！|韋斯特沃德霍！]]」和[[加拿大|加拿大]][[魁北克省|魁北克省]]的「[[聖路易士哈！哈！|聖路易士哈！哈！]]」。

== 兩性與嘆號的使用 ==
研究顯示，由於[[女性|女性]]的感情較[[男性|男性]]豐富，女性使用感歎號也較男性多<ref>{{英文}}[http://jcmc.indiana.edu/vol11/issue4/waseleski.html Gender and the Use of Exclamation Points in Computer-Mediated Communication: An Analysis of Exclamations Posted to Two Electronic Discussion Lists]</ref>。

==電腦應用==
感歎號在 Unicode 字符是 U+0021「!」。

[[全形|全形]]分號是 U+FF01：
*[[台港澳|台港澳]]和[[日本|日本]]標準為置中，如「<span style="border: 1px dotted red;">{{lang|zh-tw|！}}</span>」，[[直排|直排]]時亦然
*[[中國大陸|中國大陸]]標準為靠左下，如「<span style="border: 1px dotted red;">{{lang|zh-cn|！}}</span>」；直排時則靠右上，新版 Unicode 增加配合中國大陸寫法之「垂直感歎號（PRESENTATION FORM FOR VERTICAL EXCLAMATION MARK），U+FE15」 「<span style="border: 1px dotted red;">︕</span>」<ref>http://www.unicode.org/charts/PDF/UFE10.pdf</ref>，以兼容 [[GB_18030|GB 18030]]
*另有「小感歎號（SMALL EXCLAMATION MARK）」「<span style="border: 1px dotted red;">{{lang|zh-tw|﹗}}</span>」在 U+FE57，源自 [[CNS11643|CNS11643]] / [[Big5|Big5]]。
