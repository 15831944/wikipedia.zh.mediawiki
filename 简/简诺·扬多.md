'''简诺·扬多'''（{{lang-hu|Jandó Jenő}}，{{bd|1952年|2月1日|||catIDX = J}}），[[匈牙利|匈牙利]][[钢琴家|钢琴家]]，[[布达佩斯|布达佩斯]][[李斯特音乐学院|李斯特音乐学院]]教授。

== 青少年时期 ==
简诺·扬多在[[布达佩斯|布达佩斯]][[李斯特音乐学院|李斯特音乐学院]]师从Katalin Nemes教授以及Pál Kadosa教授，此后赢得了包括[[齐夫拉·乔治|齐夫拉]]钢琴比赛和{{le|迪诺·西亚尼|Dino Ciani|西亚尼}}钢琴比赛在内的众多国际奖项。但是，他职业生涯的真正开始于他18岁时在贝多芬钢琴比赛获得第三名。同时，他成为了1973年匈牙利钢琴比赛<!-- 尚不明确 -->的赢家，并且在1977年{{link-en|雪梨國際鋼琴比賽|Sydney International Piano Competition|悉尼国际钢琴比赛}}[[室内乐|室内乐]]组获得第一名。

== 扬多的演奏，独奏与伴奏 ==
他对独奏和伴奏一样热爱，从他的录制作品涵盖[[贝多芬|贝多芬]]全套钢琴奏鸣曲、[[舒伯特|舒伯特]]的《[[鳟鱼五重奏|鳟鱼五重奏]]》和[[贝多芬|贝多芬]]的{{link-en|钢琴三重奏|Piano trio}}《{{link-en|第5－6號鋼琴三重奏 (貝多芬)|Piano Trios, Op. 70 (Beethoven)|幽靈}}》和《{{link-en|第7號鋼琴三重奏 (貝多芬)|Piano Trio, Op. 97 (Beethoven)|大公}}》而体现出来。

作为一个伴奏演奏家，扬多与[[西崎崇子|西崎崇子]]合作演奏了贝多芬、[[塞扎尔·弗兰克|法朗克]]和[[葛利格|葛利格]]的{{link-en|小提琴奏鸣曲|Violin sonata}}、舒伯特的全套小提琴奏鸣曲，以及为[[拿索斯唱片|拿索斯唱片]]录制的[[莫扎特|莫扎特]]奏鸣曲。其中部分唱片在[[企鵝唱片指南|企鵝唱片指南]]中获得极高的排名。他在[[高大宜|高大宜]]的大提琴奏鳴曲及最近與德國[[大提琴|大提琴家]]瑪莉亞·柯利科（Maria Kliegel）合作的[[多赫南伊·埃爾諾|埃爾諾]]大提琴奏鳴曲中亦展現他的成就。

他常邊彈琴邊唱歌。為了抑制這個行為，他在嘴裡銜著未點燃的香菸<ref>{{cite web | title = Jeno Jando- Bio, Albums, Pictures – Naxos Classical Music. | url = http://www.naxos.com/person/Jeno_Jando/339.htm | accessdate = 2016-06-30 | language = en}}</ref>。

== 演奏錄音 ==
揚多專為[[拿索斯唱片|拿索斯唱片]]灌錄唱片，並已灌錄超過60張唱片，包含[[約翰·塞巴斯蒂安·巴哈|巴哈]]、[[貝多芬|貝多芬]]、[[弗朗茨·李斯特|李斯特]]、[[舒曼|舒曼]]、[[舒伯特|舒伯特]]、[[布拉姆斯|布拉姆斯]]、[[海頓|海頓]]、[[巴爾托克|巴爾托克]]、[[蕭邦|蕭邦]]及其他[[作曲家|作曲家]]的作品。
