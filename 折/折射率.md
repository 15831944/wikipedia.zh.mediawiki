某种[[光學介質|介质]]的'''折射率''' {{VarSerif|n}} 等于「[[光速|光在真空中的速度]]（{{VarSerif|c}}）」跟「光在介质中的[[相速度|相速度]]（{{VarSerif|v}}）」之比，即：

:<math>n=\frac{c}{v}</math>
比如水的折射率是1.33。表示光在真空中的传播速度是在水中传播速度的1.33倍。

历史上，折射率最早出现在[[折射定律|折射定律]]([[斯涅爾定律|斯涅爾定律]])中，<math>n_1\sin\theta_1 = n_2\sin\theta_2</math>。其中，<math>\theta_1</math>与<math>\theta_2</math>分别是光在介质界面上的入射角和折射角，两种介质的折射率分别是<math>n_1</math>与<math>n_2</math>。

== 水波的相對折射率 ==
水波的'''相對折射率'''為水波由''A''水區進入不同深度的''B''水區時，水波[[入射角|入射角]]''i''和[[折射角|折射角]]''r''的[[正弦|正弦]]的比值，稱為「''A''水區進入''B''水區的相對折射率<math>n_{AB}</math>」或「''B''水區對''A''水區的相對折射率<math>n_{AB}</math>」。

<math>n_{AB} = \frac{\sin i}{\sin r}</math>；<math> \frac{\sin i}{\sin r}=\frac{n_{B}}{n_{A}} </math>

== 雙折射率 ==
[[雙折射|雙折射]]材料的折射率，取決於光的[[偏振|偏振]]和傳播方向。

== 折射率是波长的函数 ==

在相同介質中，不同的[[波长|波长]]的光，因為行進速度不同，造成在折射過程中偏折角度不同，其折射率''n''（''λ''）也不同，这叫做[[光色散|光色散]]。折射率与[[波长|波长]]或者[[频率|频率]]的关系称为光的[[色散关系|色散关系]]。常用的折射率有：

* ''n''<sub>d</sub>是介质在[[夫朗和斐谱线|夫朗和斐谱线]]d（氦黄线587.56纳米）的折射率。
* ''n''<sub>F</sub>是介质在[[夫朗和斐谱线|夫朗和斐谱线]]F（氢蓝线486.1纳米）的折射率。
* ''n''<sub>C</sub>是介质在[[夫朗和斐谱线|夫朗和斐谱线]]C（氢红线656.3纳米）的折射率。
* ''n''<sub>e</sub>是介质在[[夫朗和斐谱线|夫朗和斐谱线]]e（汞绿线546.07纳米）的折射率。
===柯西公式===

<math>n=A+\frac{B}{\lambda^2}+\frac{C}{\lambda^4}</math><ref>Rudolf Kingslake, Lens Design Fundamentals,p12, Academic Press 1978</ref>

===康拉迪公式===

<math>n=A+\frac{B}{\lambda}+\frac{C}{\lambda^{3.5}}</math><ref>A.E.Conrady Applied Optics and Optical Design, part II,p659, Dover</ref>

===Herzberger公式===

<math>n=A+B*\lambda^2+\frac{C}{\lambda^2-\delta^2}+\frac{C}{(\lambda^2-\delta^2)^2}</math><ref>Rudolf Kingslake, Lens Design Fundamentals,p14, Academic Press 1978</ref>
===肖特玻璃厂公式===

<math>n=A+\frac{B}{\lambda}+\frac{C}{\lambda^2}+\frac{D}{\lambda^4}+\frac{E}{\lambda^6}+\frac{F}{\lambda^8}</math><ref>Rudolf Kingslake, Lens Design Fundamentals,p14, Academic Press 1978</ref>
===Sellmeier公式===

<math>N^2-1=\frac{A*\lambda^2}{\lambda^2-D}+\frac{B*\lambda^2}{\lambda^2-E}+\frac{C*\lambda^2}{\lambda^2-F}</math><ref>Milton Laikin  Lens Design p9 2006 CRC Press 978-0849382789 </ref>

== 不透明物体的折射率 ==

另外不透明的物体的折射率也是可以测量的，在图形学中，可以使用不同的折射率来渲染金属或者塑料这样的不同的反射效果。

== 複數折射率 ==
<math>\tilde{n} = n + i\kappa</math>
複折射率的实部即通常的折射率，而虚部称为消光系数（extinction coefficient），表示材料對傳播通過電磁波的衰減量。
