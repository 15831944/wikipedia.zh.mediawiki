'''折從阮'''，[[五代|五代]]時的軍事家、政治家，半獨立的[[藩鎮|藩鎮]]，歷事[[後唐|後唐]]迄[[後周|後周]]四朝。[[河東道|河東道]][[雲中|雲中]]（今[[山西省|山西省]][[大同市|大同市]]）人，生于[[唐|唐]][[大順|大順]]二年（891年），卒于後周[[顯德|顯德]]二年（955年）。初名從遠，字可久，為了避後漢高祖[[劉知遠|劉知遠]]之諱，改名從阮。

== 家族淵源 ==
[[折姓|折姓]]是[[折掘氏|折掘氏]]改為單字漢姓，出自[[党項羌|党項羌]]，更早則出自[[鮮卑族|鮮卑族]]，和[[吐谷渾|吐谷渾]]也有親戚關係。折掘氏[[漢|漢]][[晉|晉]]之際就已是西北大姓了，[[五胡十六國|五胡十六國]]時的[[南涼|南涼]]景王[[禿髮傉檀|禿髮傉檀]]的王后，就是出自鮮卑折掘氏（《[[晉書|晉書]]》南涼載記），禿髮就是拓拔氏的一支，而以遊牧民族的習慣，后族一般都來自大部落，以折掘王后來看可知折掘氏是可以和拓拔氏結為婚姻同盟的大族了。另外也有說鮮卑折掘氏是古代[[匈奴|匈奴]][[折蘭王|折蘭王]]的後代，國破後又融入鮮卑族。

== 生平功績 ==
折家世為北蕃大族，發跡則從唐末開始，河東當時是[[晉王|晉王]][[李克用|李克用]]的地盤，克用以緣河五鎮合為府谷鎮，[[折宗本|折宗本]]、[[折嗣倫|折嗣倫]]遂世為鎮將，又以功升為[[麟州|麟州]][[刺史|刺史]]。嗣倫就是從阮之父。折從阮最早仕官是擔任[[後唐莊宗|後唐莊宗]]的[[牙將|牙將]]，後來拜為[[府州|府州]]刺史。[[石敬瑭|石敬瑭]]割[[燕雲十六州|燕雲十六州]]給[[契丹|契丹]]，他在府州駐守。[[開運|開運]]元年（西元944年），[[後晉|後晉]]與契丹敗盟後，他攻取契丹城堡十余處，每次作戰必勝。不過後晉在將臣無能下累敗于契丹軍，折家在西北一隅的小勝並無法挽回[[汴京|汴京]]被破的命運。

[[後漢|後漢]]高祖[[劉知遠|劉知遠]]時為[[河東節度使|河東節度使]]，在[[郭威|郭威]]等將校支持下稱帝，擁兵坐觀[[中原|中原]]亂局，招撫後晉舊臣，終於利用契丹退軍的時機，迅速進入[[開封|開封]]、[[洛陽|洛陽]]等地。劉知遠為了酬謝折家的首附之功，升府州為[[永安軍|永安軍]]，擢從阮為永安軍[[節度使|節度使]]，及府、勝等州[[觀察處置使|觀察處置使]]。[[後漢隱帝|後漢隱帝]]即位後又加封特進、檢校太師，另授安北都護、振武軍節度使。

[[後周|後周]]太祖[[郭威|郭威]]稱帝，折從阮以北國雄鎮，國朝重臣的地位，在奉表稱藩後，後周就加封他為同平章事，等於是有使相的地位了。歷任宣義、保義、靜難、永安等四鎮節度使，太祖將崩又以為世宗[[顧命|顧命]]。因鎮守邊關有功，去世之時，後周世宗贈為[[中書令|中書令]]。

折從阮一生經歷了五代時期最混亂的時期，又遇上契丹南下覆滅石晉的危機，府州直面契丹的[[西京|西京]][[大同府|大同府]]與西南路招討司，作為河東地區抵禦契丹入侵的第一道關卡，戍守邊境、維護百姓，對於中原朝庭或華夏民族而言功不可沒。特別是後周代漢後，因為[[北漢|北漢]]割據[[太原|太原]]，隔斷府州與汴洛的聯繫，而西面又阻於世仇的[[平夏部|平夏部]]，折從阮仍奉表不絶，及至於二度入闕，其氣節令當世感佩，更重要因為折家軍的精騎無雙，牽制北漢與契丹無法全力南下，才讓後周世宗可以放手進行先南後北的統一大業。

== 忠貞將門 ==
*{{main|府州折氏世系图}}

折家雖然與西夏拓拔氏同族，但是折家地近雲代腹心，從唐初以來就開始漢化，折掘家又與拓拔家為世仇，加上從來中原天子對折家就是累世加恩，所謂的”簪纓不替，其勛業彪炳史冊，舊絶千古”（[[清|清]][[畢沅|畢沅]]《關中金石記》）。因此不論從維護折家的家業，又或是忠于王事，有[[宋|宋]]一代府州折家抗擊外族不遺餘力，從折從阮之後英才輩出，傳世九代歷二百年間，出了三個節度使、四個[[團練使|團練使]]、一個[[防禦使|防禦使]]。從阮子[[折德扆|德扆]]，孫[[折御勛|御勛]]、[[折御卿|御卿]]或其玄孫[[折克行|折克行]]皆為當世名將。從阮的孫女[[折賽花|折賽花]]嫁給[[代|代]]北名將[[楊業|楊業]]，楊業就是演義中的楊老令公，折賽花自然就是傳說中的[[佘老太君|佘老太君]]了，因為老太君的家傳祖訓，對於楊家忠心抗[[遼|遼]]具有很大影響。

折家數代北抗契丹、西禦党項，五代史稱”獨據府州，控扼西北，中國賴之”、”夏人畏之，益左廂兵，專以當折氏”、”自從阮而下，繼生名將，世篤忠貞，足為西北之捍，可謂無負於宋者矣”。正如[[折繼閔|折繼閔]]《神道碑》中所述”而折氏傳襲至今不絶，非夫世濟忠義，勤勞王家，炳丹青而不渝，盟帶礪而無愧者，孰能與于此？”
