'''哥伦比亚住宅圈'''（{{lang-en|'''Columbia Circle'''}}），俗称“外国弄堂”，是民国时期位于上海西郊的一片高级住宅区，大致位于今日[[长宁区|长宁区]][[新华路街道_(上海市)|新华路街道]]范围内。

==开辟与发展==
[[File:Map_of_Columbia_Country_Club,_Shanghai,_1933.jpg|thumb]]
哥伦比亚住宅圈所在地原是上海西郊的农田。20世纪20年代初，[[上海公共租界|上海公共租界]][[上海公共租界工部局|当局]]趁江浙两地军阀交战之机在此[[越界筑路|越界筑路]]。1925年，[[新华路_(上海)|安和寺路]]（Avenue Amherest，今新华路）和哥伦比亚路（今番禺路）被修筑。<ref name=qq>{{cite web |url= http://sh.qq.com/a/20130127/000040.htm|title=上海传奇城市设计师邬达克 寻找“哥伦比亚”  |accessdate=2014-06-30 |publisher= 腾讯网  |work= |language=zh-hans}}</ref>是年，上海美国侨民在西郊[[延安西路_(上海)|大西路]]以南，兴建占地广阔的[[美国乡村总会|美国乡村总会]]（Columbia Country Club）。与此同时，美商普益地产公司陆续购进安和寺路、哥伦比亚路一带的土地百余亩，将道路两侧统一规划用地划分为82块，每块基地根据客户需求建造不同的建筑，整体命名为“哥伦比亚住宅圈”。<ref name=qq></ref><ref>{{cite book |author= |title=《上海百年名楼名宅: 上海百年名宅》|year=2006 |publisher= 光明日报出版社|pages=57 |isbn=9787802062368 |language=zh-hans}}</ref>这些建筑以花园别墅和联体别墅为主。由于该公司经理是[[匈牙利|匈牙利]]人，故他聘请了[[匈牙利|匈牙利]]籍设计师[[邬达克|邬达克]]设计其中一部分建筑。1928年，数十幢花园住宅被兴建。虽然这些建筑都是出自同一家地产公司，但是总体上规划得体，且毎幢住宅风格、样式各异。这使得哥伦比亚住宅圈迅速成为了沪西的高档住宅区，住户以外籍侨民为主。<ref>{{cite book |author= |title=《街道背后: 海上地名寻踪》|year=2008 |publisher= 同济大学出版社|pages=146 |isbn= 9787560837369 |language=zh-hans}}</ref>此后亦有其他地产商在此投资房产。<ref name=dfz>{{cite web |url= http://www.shtong.gov.cn/newsite/node2/node71994/node81772/node81775/node81786/userobject1ai109062.html|title=特色志 >> 上海名建筑志 >> 美轮美奂的海上洋楼 >> "外国弄堂"：新华路住宅  |accessdate=2014-06-30 |publisher= 上海地方志办公室  |work= |language=zh-hans}}</ref><ref>{{cite book|title=永不飘散的风情|author =惜珍|publisher=东方出版中心|page =196-202|isbn=9787547300169}}</ref>

==现代保护==
自1989年起，这些建筑分批被列为[[上海市优秀历史建筑|上海市优秀历史建筑]]。2005年，《上海市新华路历史文化风貌区保护规划》经上海市政府批准通过，番禺路-淮海西路-安顺路-定西路-法华镇路合围的区域被划为[[新华路历史文化风貌区|新华路历史文化风貌区]]。规划对这一范围内的建筑分为五类，采取不同的保护措施。同时将[[新华路|新华路]]列为一类风貌保护道路，规划永不拓宽；新华路119弄、155弄、185弄、211弄、329弄、272弄、294弄七条弄堂列为风貌保护巷弄。<ref>{{cite web|url=http://218.242.36.250/News_Show.aspx?id=8285&type=1|title=上海市新华路历史文化风貌区保护规划|accessdate=2014-06-30|publisher=上海市规划和国土资源网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20140421064035/http://218.242.36.250/News_Show.aspx?id=8285&type=1|archivedate=2014-04-21}}</ref>

==优秀历史建筑==
===新华别墅===
即新华路211-329弄，包括29幢独立式花园住宅，为哥伦比亚住宅圈内的精华建筑群体。建筑由[[邬达克|邬达克]]设计，柴顺记营造厂建造，包括英国式、美国式、荷兰式、意大利式、西班牙式等多种样式。其中新华路211弄2号曾为传教士[[李佳白|李佳白]]住宅。新华路329弄36号周均时住宅，原为西班牙公使馆使用，也是幢全国唯一的双层圆形花园别墅，俗称“蛋糕房”。这些建筑分五批列入优秀历史建筑名单。<ref name=qq></ref><ref name=dfz></ref><ref name=df>{{cite web |url= http://finance.eastday.com/consumption/m5/20120817/u1a6792876.html|title=侬好上海：上海有条“外国弄堂”又名新华别墅  |accessdate=2014-07-01 |publisher= 东方网  |work= |language=zh-hans}}</ref>
===新华路179号住宅===
[[File:A_picture_from_China_every_day_200.jpg|thumb]]
建于1925—1935年间，为假三层砖木结构的花园住宅，建筑样式为德国式。建筑采用红瓦双坡屋面，立面材质为拉毛水泥。其南立面[[山墙|山墙]]有半露明黑色木构架，与窗框连成一体。<ref name=df></ref>
===新华路315号住宅===
[[File:A_picture_from_China_every_day_199.jpg|thumb]]
建于1930年，原为某国石油大王为其儿子在沪购的结婚用房，是一幢假三层的砖木结构的独立式花园住宅，建筑风格为英国乡村风格。<ref name=dfz></ref><ref name=df></ref>
===新华路483号住宅===
[[File:A_picture_from_China_every_day_197.jpg|thumb]]
建于1946年，原为[[金润庠|金润庠]]住宅，为三层钢筋混凝土结构的建筑，建筑风格为现代风格。该建筑由中国建筑师顾梦良设计，上海梁记营造厂营建。建筑立面水平线条流畅，装饰简洁。<ref name=dfz></ref>
===梅泉别墅===
即新华路593弄，包括20幢二层砖木结构的花园住宅，属于花园式里弄。梅泉别墅建于1933年，由房地产商吴其达投资，中国建筑师[[奚福泉|奚福泉]]设计，李顺记营造厂施工。<ref name=dfz></ref>

位于番禺路60号，原是[[邬达克|邬达克]]为自己设计的住宅，但建成后他并未入住。1929年，[[邬达克|邬达克]]因设计[[慕尔堂|慕尔堂]]时遇到麻烦，[[孙科|孙科]]出面解决，就把这幢住宅低价转让给孙科。住宅现为[[上海生物制品研究所|上海生物制品研究所]]，1989年列入[[上海市文物保护单位|上海市文物保护单位]]和第一批上海市优秀历史建筑。

这是一幢假三层砖木结构的花园住宅，具有明显的西班牙建筑特征，亦兼具其他建筑风格的特点，属于混合式建筑。<ref name=qq></ref><ref>{{cite web |url= http://www.shtong.gov.cn/newsite/node2/node71994/node81772/node81775/node81786/userobject1ai109035.html|title=特色志 >> 上海名建筑志 >> 美轮美奂的海上洋楼 >> 和谐统一的混合式建筑：孙科住宅（上海生物制品研究所）  |accessdate=2014-07-02 |publisher= 上海地方志办公室  |work= |language=zh-hans}}</ref>

===邬达克公馆===
位于番禺路129号，建筑风格为英国乡村风格。[[邬达克|邬达克]]在转让自己的住宅给[[孙科|孙科]]后，又设计建造了这幢住宅作为自己的新居。现已改造为邬达克纪念馆。<ref name=qq></ref>
