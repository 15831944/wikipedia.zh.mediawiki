'''哥伦比亚公平自由'''（{{lang-es|Colombia Justa Libres}}）是一个[[基督教|基督教]]政党，成立于2017年，由多个[[基督教派系|基督教派系]]组成。<ref>{{Cite web |url = http://palmira.extra.com.co/entrega-de-400-mil-firmas-377220/ |title = Entrega de 400 mil firmas |date = 2017-12-11 |language = es |accessdate = 2019-04-04 }}</ref>[[2018年哥伦比亚议会选举|2018年哥伦比亚议会选举]]期间，共选出了4名议员，其中包括3名参议员和1名众议员。自2018年上半年以来，和[[民主中心|民主中心]]、{{le|绝对革新独立运动|Movimiento Independiente de Renovación Absoluta}}等政党组成哥伦比亚大联盟，推选[[伊万·杜克·马尔克斯|伊万·杜克·马尔克斯]]为[[哥伦比亚总统|总统]]，并成功当选。

==意识形态==
作为一个公民、民主和[[基督教|基督教]]政党，哥伦比亚公平自由寻求在在哥伦比亚社会的各个领域建立[[道德|道德]]和公共道德原则，共同努力影响哥伦比亚的政治、公民和体制事务，并在[[经济|经济]]、[[政治|政治]]、[[社会|社会]]和[[文化|文化]]领域做出改变。<ref>{{Cite web |url = https://www.elespectador.com/node/727882/ |title = Arranca el pulso por el Senado|website=www.elespectador.com |date = 2017-12-11 |language = es |accessdate = 2019-04-05 }}</ref>

他们为人类的尊严，平等机会，加强[[民主|民主]]而努力，为一个没有腐败并为人民服务的政府，为了[[基督徒|基督徒]]的道德，为家庭，为了正义，诚实，卓越和真理而努力。推进公民行动，改变[[哥伦比亚|哥伦比亚]]所需的计划和体制措施。
