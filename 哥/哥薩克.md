'''哥萨克'''（{{lang-ua|козаки}}；{{lang-ru|каза́ки́}}）是一群生活在东欧大草原（[[乌克兰|乌克兰]]及[[俄罗斯|俄罗斯]]南部）的[[游牧|游牧]][[民系|民系]]，在历史上以驍勇善战和精湛的[[骑术|骑术]]著称，為支撑[[俄罗斯帝国|俄罗斯帝国]]于17世纪往东扩张的主要力量。[[移民|移民]][[中国|中国]]的部分哥萨克被称为[[阿尔巴津人|阿尔巴津人]]。

== 歷史 ==
13世纪开始，一些[[斯拉夫人|斯拉夫人]]为了逃避[[蒙古帝国|蒙古帝国]]中[[钦察汗国|钦察汗国]]的统治而流落到俄罗斯南部地区，包括[[頓河_(俄羅斯)|顿河]]流域、[[第聂伯河|第聂伯河]]下游和[[伏尔加河|伏尔加河]]流域。[[15世纪|15]]、[[16世纪|16世纪]]时，一些城市貧民與不愿成為[[農奴|農奴]]的俄罗斯、乌克兰农民迁徙到俄南地区。这些人被称为“哥萨克”，即[[突厥语|突厥语]]中的“[[自由人|自由人]]”。哥萨克人在俄南建立了一些地方政权。[[東歐|東歐]]因為[[平原|平原]]多，[[山地|山地]]少，因此哥薩克族群多數以河流命名，如“[[顿河|顿河]]哥萨克”，“[[烏拉爾河|烏拉爾河]]哥薩克”等。

哥萨克人以英勇善战著称。在[[俄羅斯歷史|俄羅斯歷史]]上，[[钦察汗国|钦察汗国]]被推翻之后，[[俄羅斯帝國|俄羅斯帝國]][[沙皇|沙皇]]通过收买哥萨克[[貴族|貴族]]與[[酋長|酋長]]們而控制哥萨克人。哥薩克人組成的騎兵，是沙俄的重要武力。俄国向[[西伯利亞|西伯利亞]]擴張的过程中哥薩克是俄国的主要依仗。沙俄时期，一共组建了11个哥萨克军区。分别是：[[顿河|顿河]]、[[库班|库班]]、[[捷列克|捷列克]]（Terek）、[[阿斯特拉罕|阿斯特拉罕]]、[[乌拉尔|乌拉尔]]、[[奥伦堡|奥伦堡]]、[[西伯利亚|西伯利亚]]、[[斜米列契|斜米列契]]（今[[哈萨克|哈萨克]]）、外[[貝加尔|貝加尔]]、[[阿穆尔|阿穆尔]]和[[乌苏里|乌苏里]]军区。此外还有[[叶尼塞|叶尼塞]]军区和原本定于1917年组建的[[伊尔库茨克团|伊尔库茨克团]]，因故未能实行。

[[十月革命|十月革命]]之后，哥萨克人少数参加[[布尔什维克|布尔什维克]]政府的[[苏联红军|苏联红军]]，多数参加反共產的[[白军|白军]]。[[第二次世界大战|第二次世界大战]]期间，哥萨克骑兵发挥了重要作用，在[[史達林格勒攻防戰|史達林格勒攻防戰]]中建立重大戰功，而同时也有一些人趁机反叛[[苏联|苏联]]的统治。

== 現況 ==
[[File:Novosibirsk-Karimov.jpg|thumb]]
今天的哥萨克人不被认为是一个民族，居住在俄罗斯、乌克兰等地，但他们是一个保持传统的人群共同体。2005年，俄罗斯总统[[弗拉基米尔·普京|弗拉基米尔·普京]]提出《关于俄罗斯哥萨克人的国家义务》议案，保护哥萨克人的传统。目前調查仍有約700萬人自認為是俄羅斯哥薩克人。

== 文化 ==
=== 哥薩克舞 ===
哥薩克舞是當地的傳統舞蹈。

== 相關文學作品 ==
* [[沙俄|沙俄]]作家[[列夫·托爾斯泰|列夫·托爾斯泰]]的中篇小說《[[哥薩克_(小說)|哥薩克]]》描述了俄國貴族青年到高加索山與哥薩克人相處的過程。
* [[苏联|苏联]]作家[[肖洛霍夫|肖洛霍夫]]的长篇小说《[[静静的顿河|静静的顿河]]》描述了十月革命前后顿河哥萨克的生活。
* [[苏联|苏联]]作家[[巴别尔|巴别尔]]的半叙事性小说集《[[骑兵军|骑兵军]]》，描述了[[苏波战争|苏波战争]]时期的苏俄第一骑兵军的哥萨克士兵的战斗与生活。
