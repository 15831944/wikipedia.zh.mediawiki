{{Vgname|哥萨克3|en=Cossacks 3}}是2016年由乌克兰GSC Game World公司推出的一款即时战略游戏。本作实质是系列[[哥薩克：歐洲戰爭|首作]]的重制版，游戏的时代背景同样设定为17-18世纪的欧洲。
== 游戏玩法 ==
本作中囊括了一代原作中的12个国家，70个单位，100项技术和140以上的历史建筑<ref>{{Cite web|url=http://www.cossacks3.com/index.php?lang=en#about|title=Cossacks 3|date=|publisher=GSC Game World|accessdate=2018-05-05}}</ref> 。本作支持8名玩家和8000个单位同局对战，且支持[[追加下载内容|追加下载内容]]。
