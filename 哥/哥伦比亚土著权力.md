'''哥伦比亚土著权力'''（{{lang-es|Autoridades Indígenas de Colombia}}）是哥伦比亚一个进步的{{le|本土主义 (意识形态)|Indigenism|本土主义}}政党。它是一个[[土著|土著]]政治组织，于1978年成立，1991年8月15日正式被承认为[[哥伦比亚|哥伦比亚]]政党。哥伦比亚土著权力的政治、社会和文化运动，源于争取土地和捍卫土著人民权利的斗争。自1991年《宪法》颁布以来，该党一直在参与国家的政治活动，并提出了一种模式，其功能来自于地域性，自治，身份，自己的历史，参与，多样性和跨文化原则，体现在哥伦比亚社会和土著人民之间的关系中。<ref>{{Cite web |url = https://www.mininterior.gov.co/content/aico |title = Autoridades Indígenas de Colombia|language = es |accessdate = 2019-04-04 }}</ref><ref>{{Cite web |url = https://congresovisible.uniandes.edu.co/partidos/perfil/aico-autoridades-indigenas-de-colombia/9/ |title = Partido AICO - Autoridades Indígenas de Colombia |language = es |accessdate = 2019-03-05 }}</ref>

==人物==
* 曼努埃尔·比埃尔沃·帕丘坎·钦格尔（Manuel Biervo Palchucan Chingal）：洛斯帕斯托斯镇的土著领袖，曾任卡里萨马、纳里尼奥市长。现为[[哥伦比亚|哥伦比亚]]土著选区的[[参议员|参议员]]。<ref>{{Cite web |url = https://congresovisible.uniandes.edu.co/congresistas/perfil/manuel-bitervo-palchucan-chingal/8734/|title = Senador Manuel Bitervo Palchucan Chingal |language = es |accessdate = 2019-04-04 }}</ref>
