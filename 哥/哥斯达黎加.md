'''哥斯达黎加共和國'''（{{lang-es|'''República de Costa Rica'''}}），通稱'''哥斯达黎加'''（台灣稱'''哥斯大-{}-黎加'''，中國大陸和港澳稱'''哥斯達-{}-黎加'''），意為「豐饒海岸」，是[[中美洲|中美洲]][[国家|国家]]，北邻[[尼加拉瓜|尼加拉瓜]]，南与[[巴拿马|巴拿马]]接壤。哥斯达黎加是当今世界上第一个[[沒有軍隊的國家列表|裁撤军队的国家]]<ref>另日本有实质上的军队，很多国家独立时就没设立过军队。[[列支敦士登|列支敦士登]]在1868年因为经费开销过大而废除了本国军队。[[冰岛|冰岛]]于1869年裁撤了军队。</ref>。哥斯达黎加首都在[[聖荷西_(哥斯达黎加)|聖荷西]]。

== 歷史 ==
[[File:Iglesia_Inmaculada_Concepcion_Ujarras.jpg|thumb]]
{{main|哥斯達黎加歷史}}

在[[前哥倫布時代|前哥倫布時代]]，就有[[印第安人|印第安人]]居住在[[哥斯達黎加|哥斯達黎加]]，是[[中美洲文明|中美洲文明]]和[[安第斯文明|安第斯文明]]的交匯處。

16世紀，當地的[[瑪雅人|瑪雅人]]和[[阿茲台克人|阿茲台克人]]被[[西班牙|西班牙]]人征服，於是哥斯達黎加成為西班牙領土[[新西班牙|新西班牙]]最南部的一個省，省府在[[卡塔多|卡塔多]]（Cartago）。

哥斯達黎加曾暫時地加入[[阿古斯汀一世_(墨西哥)|阿古斯汀一世]]的墨西哥帝國（''參看''[[墨西哥歷史|墨西哥歷史]]和[[墨西哥帝國|墨西哥帝國]]），後來成為[[中美洲聯合省|中美洲聯合省]]（''參看''1823年到1839年的[[中美洲歷史|中美洲歷史]]）的一員。1824年，首都遷往[[聖荷西_(哥斯大黎加)|聖荷西]]，1940年代哥斯達黎加成為獨立的國家。

哥斯達黎加逃過了許多侵擾中美洲各國的麻煩。與其一些鄰國不同，哥斯達黎加是穩定、和平和民主管理的一個例子。19世紀晚期，有兩次短暫的暴力活動迫害了民主的發展。1948年內戰之後[[哥斯達黎加總統|哥斯達黎加總統]][[何塞·菲格雷斯·費雷爾|何塞·菲格雷斯·費雷爾]]宣佈解散軍隊，並在1949年將此列入[[憲法|憲法]]。從那時起，哥斯達黎加進入罕見的無軍隊介入的民主格局。

哥斯達黎加儘管仍是[[農業國家|農業國家]]，但已經取得相對較高的生活水平，土地所有權普遍擴張，而且旅遊業蓬勃發展。

== 政治 ==
{{main|哥斯达黎加政治}}
'''哥斯达黎加'''是拥有强大的宪法制衡系统的[[民主|民主]][[共和国|共和国]]。行政责任归属于[[总统|总统]]，总统是国家的力量中心。该国有两位副总统以及由包括一名副总统在内的15人组成的内阁。总统和57名立法会议员由选举产生，以4年为任期。1969年批准的宪法修正案限制总统和立法会议员只能任一届。尽管如此，原立法议员们可以隔一届任期再去参选。目前，有人提出要修改宪法，而使总统获得第二个任期，与此同时，在立法会裡面也有人质疑禁止总统连任的法律是否违宪。2003年4月，经过一场激烈的争论后，立法会以决议案的形式，正式肯定了禁止任何一位总统寻求在任期后的任何时间再次出任的法律条文。根据这个决议案，[[奥斯卡·阿里亚斯|奥斯卡·阿里亚斯]]如果在即将到来的2006年政府选举中再度参加竞选，即属违反宪法。但是其后阿里亚斯通过其政黨修憲，並已於2006年成功再度當選總統，並於2010年完成其任期。2010年其副總統勞拉當選總統，任期到2014年。阿里亚斯是自由贸易的提倡者，他支持与美国签订的自由贸易条约，这个条约的签订具有极大的争议性，其争议的激烈程度能够酿成一场全国性的示威。

哥斯达黎加以按比例的方式选举立法院议员。

國家專營的保險和电信部門，在2009年開放予私營機構參與競爭（在此之前，要得到一個手提電話號碼也要等上幾個月）。有很多國家機構都是獨立運作和享有自治的，包括電力、國營商業銀行（須面對私營銀行競爭）和[[社會保障|社會保障]]體系，這些部門對於發展哥斯達黎加的高質素生活都很重要。

七个省的省長由總統任命，可是他們的權力不大。省內沒有獨立的立法機關。根據憲法，哥斯达黎加沒有軍隊，只有警察和安全部隊維護內部安全。

== 经济 ==
{{main|哥斯大黎加經濟}}

哥斯达黎加经济主要是旅游业，农业和电子元件出口，从1997年[[经济萧条|经济萧条]]以后，开始快速增长。哥斯达黎加位于[[中美洲|中美洲]]的狭长地带且在美洲中心，有直通欧亚的海洋，可以很方便地进入美洲市场。

哥斯达黎加经济的快速发展得益于政府的一项七年的高科技发展计划。在该国投资免税并且国民教育状况很好，成为了投资的乐土，很多跨国高科技公司已经开始出口产品。 

[[货币|货币]]单位是[[哥斯达黎加科朗|哥斯达黎加科朗]]，与[[美元|美元]]的比价为500-560，与[[欧元|欧元]]比价是600。

== 人口 ==
哥斯达黎加的人口中[[混血儿|混血儿]]和纯种欧洲人占94%，主要的移民是[[西班牙|西班牙]]人，另有不少[[德国|德国]]人，[[波兰|波兰]]人，[[意大利|意大利]]人和[[犹太人|犹太人]]，在哥斯达黎加只有很少的[[印第安人|印第安人]]从西方入侵中存活下来，现在土著居民只有29000人，不足总人口的1%。如今[[印第安人|印第安人]]的数量有所增加，主要得益于移民福利，以获得农业工人的机会。[[十九世纪|十九世纪]]从[[牙买加|牙买加]]移居来的黑人后裔组成了3%人口，约96000人的少数民族。另外1%是[[海外华人|华人]]。

哥斯达黎加有大约10%的[[尼加拉瓜|尼加拉瓜]]人，他们大多是在1979－1990年[[内战|内战]]时流亡来的。此外，[[哥伦比亚|哥伦比亚]]人，[[巴拿马|巴拿马]]人和[[秘鲁|秘鲁]]人的数量也有大量增加。

== 外交 ==
哥斯达黎加與[[美國|美國]]傳統上關係密切，一直與美國保持聯盟關係。哥斯达黎加於2007年與中華民國斷交，轉與中華人民共和國建交，並取消中華民國護照護照持有人的免簽證待遇，惟之後又予以恢復，所有持有效中華民國護照者皆可免簽證入境哥國及停留至多90天。

== 地理 ==
{{main|哥斯大黎加地理}}
哥斯达黎加位于[[中美洲|中美洲]]的狭长地带，北纬10度；西经84度。东临[[加勒比海|加勒比海]]，西靠[[太平洋|北太平洋]]，有着1290千米的海岸线（东岸加勒比海212千米，西岸太平洋1016千米）。

哥斯达黎加北部与[[尼加拉瓜|尼加拉瓜]]接壤（边境线长309千米），东南偏南与[[巴拿马|巴拿马]]接壤（边境线长639千米）。总共有51100平方公里，其中领土50660平方公里，领水440平方公里，比[[美国|美国]]的[[西弗吉尼亚州|西弗吉尼亚州]]稍小，也相当于[[爱尔兰|爱尔兰]]。

哥斯达黎加[[地形|地形]]是：海岸边是平原，而中部被崎岖的高山所隔绝。该国宣布其[[专属经济区|专属经济区]]为200[[海里|海里]]，领海为12海里。气候属于[[热带|热带]]和[[亚热带|亚热带]]，还有一部分是[[新热带|新热带]]。

== 文化 ==
[[File:Teatro_National_de_Costa_Rica_-_front.jpg|thumb]]
{{main|哥斯達黎加文化}}

哥斯達黎加擁有多樣的文化，那裡是中美洲和南美洲的文化交匯處。當16世紀[[西班牙人|西班牙人]]征服美洲後，哥斯達黎加北部地區是[[瑪雅文明|瑪雅文明]]最南部的領地，中部和南部是[[奇布查族|奇布查族]]領地，而西海岸則居住著17世紀至18世紀從非洲販賣來的黑人奴隸，也就在這個時候，許多[[中國|中國]]農民來到這裡，修築鐵路。

哥斯達黎加音樂有[[tambito|tambito]]和[[punto|punto]]，十分獨特，比如「punto guanacasteco」來自於[[瓜納卡斯特省|瓜納卡斯特省]]，而「punto sancarleno」則來自於[[阿拉輝拉省|阿拉輝拉省]]的聖卡洛斯（San Carlos）。哥斯達黎加北部（曾是[[瑪雅人|瑪雅人]]的勢力範圍）和大西洋海岸傳出了許多音樂和傳奇故事。

== 軍事治安 ==
因避免當時一觸即發的軍事政變，哥斯达黎加撤除了軍隊，安全保障由美國負責。

現今治安狀況與普遍不好的[[中南美洲|中南美洲]]國家相比稍好。

== 自然生態 ==
[[File:Tortuguero_Nationalpark.jpg|thumb]]
哥斯達黎加擁有多樣的[[動物|動]][[植物|植物]]。沒有[[陸軍|陸軍]]和[[海軍|海軍]]，是[[野生動物|野生動物]]的樂園。坊間流傳：食葉之[[螞蟻|蟻]]代陸軍，[[金剛鸚鵡|金剛鸚鵡]]代空軍，而[[鯨|鯨]]取代了艦艇。哥斯達黎加40%的領土覆蓋著原始森林。

Tortuguero國家公園裡有[[蜘蛛猿|蜘蛛猿]]、[[喉猴|喉猴]]、[[白頭猴|白頭猴]]、[[三趾樹懶|三趾樹懶]]、320多種鳥類（包括了八種鸚鵡）以及各種爬蟲，是它們共同的家園。
