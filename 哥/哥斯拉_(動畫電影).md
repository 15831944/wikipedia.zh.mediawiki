《'''GODZILLA'''》（{{lang-ja|ゴジラ}}），是一部由動畫公司{{tsl|ja|ポリゴン・ピクチュアズ|Polygon Pictures}}制作的長篇動畫電影。本作為《[[哥吉拉|哥吉拉系列]]》的第30至32作，同時是系列首部長篇動畫電影作品<ref>{{Cite web |url = http://www.shin-godzilla.jp/news/news_160819_1.html |title = 世界初製作 “ゴジラ” アニメーション映画化決定！ |author = シン・ゴジラ |date = 2016-08-19 |publisher = 東寶株式會社 |language = ja |accessdate = 2017-08-16}}</ref>。

本作由三章構成，第一章{{日本作品|哥吉拉 怪獸惑星|GODZILLA 怪獣惑星}}於2017年11月7日日本上映<ref name="oricon_170326">{{Cite web |url = http://www.oricon.co.jp/news/2088097/full/ |title = アニメ映画『GODZILLA』史上初の3部作 主人公は宮野真守 |author = oricon |date = 2017-03-26 |language = ja |accessdate = 2017-08-16}}</ref>，而在電影上映後[[Netflix|Netflix]]將於其平台上向全世界發佈本作<ref>{{Cite web |url = http://eiga.com/news/20170315/21/ |title = アニメ映画「GODZILLA」、劇場公開後にNetflixで全世界配信！ |author = 映画.com |date = 2017-03-15 |language = ja |accessdate = 2017-08-16}}</ref>。

第二章{{日本作品|哥吉拉 決戰機動增殖都市|GODZILLA 決戦機動増殖都市}}於2018年5月18日上映。

第三章{{日本作品|哥吉拉 噬星者|GODZILLA 星を喰う者}}於2019年1月11日上映。

== 劇情 ==
[[File:Godzilla(2017)_right_front_view_in_Namba_Marui_August_20,_2017.jpg|200px]]”制作的雕塑。]]
某天，由於巨大生物「怪獸」與專門驅逐這些怪獸之怪物─哥斯拉的出現，使人類發現原來地球的支配者並不是他們。

「怪獸」與「哥斯拉」持續戰鬥了半世紀，而作為夾心的人類則不斷飽受雙方戰爭波及，最後只好計畫離開地球，從而遠離這場怪獸間的戰爭。

2048年，由中央政府管理下的人工智慧所挑選出的受選者們乘搭恆星間移民船，打算移居至距離地球11.905光年的「[[鯨魚座τe|鯨魚座τe]]」。然而，經歷長達20年旅程的人類，在到達這顆星球後才發現居住環境條件相比當初預計的還要惡劣許多，難以讓人類在此安居生存。

另一方面，幼時曾目睹雙親遭哥斯拉殺害的青年晴生，在這20年的時光中不斷想著重回地球殺掉哥斯拉為父母報仇。當得知星球難以居住後，晴生隨即成為移民團中主流的「地球歸還派」的中心人物，並決定進行危險的長距離亞空間航行重回地球。然而，當他們回到地球時，地球已經經過兩萬年的歲月，成為一個以哥斯拉為地上生物系頂點的未知世界<ref>{{Cite web |url = http://godzilla-anime.com/intro/ |title = Story |author = 《GODZILLA》動畫電影官方網站 |language = ja |accessdate = 2017-08-16}}</ref>。

== 人物 ==
; {{nihongo|晴生·榊|ハルオ・サカキ|聲：[[宮野真守|宮野真守]]}}
: 本作主角。日本人，24歳，階級為大尉。
: 幼時曾目睹雙親遭哥斯拉殺害，對哥吉拉充滿恨意。
: 第一部中發現哥吉拉的弱點，成功擊敗哥吉拉，但擊敗的哥吉拉只是哥吉拉的後代（擬態生物）。隨後被真正的哥吉拉瞬間擊敗。
: 第二部中從伏斯瓦雙胞胎得知增值都市，利用增值都市重新制訂戰略，最後不想失去人性，選擇摧毀增值都市，失去擊敗哥吉拉的機會。
: 第三部被梅德菲斯選為基多拉的見證者卻不願接受，摧毀基多拉的神物，導致基多拉被哥吉拉給擊退，最後理解梅德菲斯的話語，帶著剩餘的奈米金屬（禿鷹、優子）朝向著哥吉拉自殺。

; {{nihongo|梅德菲斯|メトフィエス|聲：[[櫻井孝宏|櫻井孝宏]]}}
: 艾克西夫的軍属神官。50歳（換算至人類歳數則是25歳），階級為中佐。
: 負責觀察地球的艾克西夫神官，希望晴生能成為傳遞基多拉威嚴的傳承者。

; {{nihongo|優子·谷|ユウコ・タニ|聲：[[花澤香菜|花澤香菜]]}}
: 日本人，19歳，階級為曹長。晴生的後輩。
: 動力裝甲駕駛員，暗戀晴生。第二部中因為奈米金屬將動力裝甲強化為禿鷹而成為禿鷹的第一位駕駛員，隨後在與哥吉拉戰鬥時，被加爾古強制奈米金屬化而腦死。

; {{nihongo|馬丁|マーティン・ラッザリ|聲：[[杉田智和|杉田智和]]}}
: 意大利裔美国人，34歳，階級為少佐。軍屬的環境生物學者。
: 精通各項學術的科學家，通常充當部隊軍醫，不信任外星人。第三部成功破解奈米金屬的權限，打算利用剩下的奈米金屬重新打造增殖都市對抗哥吉拉。

; {{nihongo|亞當|アダム・ビンデバルト|聲：[[梶裕貴|梶裕貴]]}}
: 德國人，21歳，階級為少尉。登陸艇駕駛員。

; {{nihongo|加爾古|ムルエル・ガルグ|聲：[[諏訪部順一|諏訪部順一]]}}
: 畢爾薩魯多的技術軍官。60歳（換算至人類歳數則是35歳），階級為中佐。
: 第二部中為了擊敗哥吉拉，決定犧牲人性與增值都市結合達到零延遲操作，命令增值都市吞噬城內所有人來加速都市的生成，最後被晴生駕駛的禿鷹給擊殺。

== 名詞 ==

[[艾克西夫|艾克西夫]]：高智慧外星人，皮膚潔白，頭髮兩側會綁髮髻。擁有心電感應能力卻不對外透露。崇拜一個以「伽瑪托倫演算法」為中心的宗教，重大的決策都會使用伽瑪托倫演算法來決定，準確率達100%。見證過許多高度文明滅亡，自己的星球同樣也被怪物毀滅，卻從不透露此怪物的情報。

[[畢爾薩魯多|畢爾薩魯多]]：高智慧外星人，皮膚深黑，體態高壯。非常重視工業、高科技。研發出「奈米金屬」，是機械哥吉拉的重要推手。認為要戰勝哥吉拉唯有成為怪獸一途。自己的星球因為恆星滅亡只能逃離，答應幫助地球人擊敗哥吉拉換得地球的居住權。

[[伏斯瓦|伏斯瓦]]：兩萬年後的地球人，白髮，皮膚黑底但有類似線條刺青的昆蟲鱗粉遍佈全身，完全適應兩萬年後的地球。擁有心電感應能力，學習語言的能力十分卓越。崇拜巨卵（原作設定上為摩斯拉）。居住在地洞。

哥吉拉：本作主要怪獸，與原作設定有些微差異。本作據說是從植物進化而來。沒有骨骼，全身肌肉宛如金屬般堅硬。細胞擁有磁鐵性質，可蓄積電磁力發射電磁炮。初登場為50公尺高左右，兩萬年後成長為六倍之大(300公尺) 。擊敗聯合軍團與地球上的所有怪獸，達到逆生態環境發展（所有地球上的一切都順著哥吉拉的存在而進化，包括地球環境）。

增值都市：機械哥吉拉的遺體。雖然被擊毀，但主控台的破壞導致系統失控，使奈米金屬不斷的自我複製，在兩萬年後成為一座巨大的智慧體城市。

[[基多拉|基多拉]]：高維度的怪獸，僅可用肉眼觀察，任何儀器皆觀測不到，可以不遵照人類世界的物理法則。

黃金巨卵（[[摩斯拉|摩斯拉]]）：伏斯瓦人所崇拜的神，卵的所在地為禁區，只有長老、神官可以進入。可以透過觸摸巨卵達到遠距離的心電感應。

== 發行 ==
2017年3月，Netflix Japan總經理格雷格·彼得斯（Greg Peters）宣佈本作將透過Netflix平台於超過190個國家播出，並表示「與一些好像東寶一樣的優秀創作者合作，並將哥斯拉帶給190多個國家的Netflix用戶，是我們（Netflix）的一個重要里程碑」<ref>{{Cite web |url = https://media.netflix.com/en/press-releases/netflix-brings-original-anime-series-godzilla-to-fans-worldwide |title = Netflix brings original anime series, GODZILLA, to fans worldwide |author = Belle Baldoza |date = 2017-03-13 |publisher = Netflix Media Center |accessdate = 2017-08-16}}</ref>。同年6月，官方宣佈《怪獸行星》將於同年11月17日於日本上映<ref name="oricon_170613"/>。
