'''哥倫比亞救護飛機墜機事件'''是哥倫比亞Ambulancias Aéreas de Colombia公司所屬的一架救護專機， 於當地時間2014年3月12號上午6點30分從波哥大的埃爾多拉多國際機場飛往卡克塔省的索若拉市<ref>{{cite web |url=http://newstrackindia.com/newsdetails/2014/03/13/9--Six-killed-in-Colombia-ambulance-air-crash-.html|title=Six killed in Colombia ambulance air crash|publisher=NEWS TRACK India|accessdate=2014-03-13}}</ref>執行醫療任務，飛機在比亞維森西奧空域飛行員回報發生機械故障要進行緊急迫降<ref>{{cite web|url=http://www.flightweb.com/forums/index.php?showtopic=5241|title=Fatal Fixed-wing crash in Columbia|publisher=Flightweb|accessdate=2014-03-14|archive-url=https://web.archive.org/web/20140322043202/http://www.flightweb.com/forums/index.php?showtopic=5241|archive-date=2014-03-22|dead-url=yes}}</ref>。經引導警急降落在{{link-en|La Vanguardia機場|La Vanguardia Airport}}<ref>{{cite web |url=http://www.trtchinese.com/trtworld/zh/newsDetail.aspx?HaberKodu=fe5cfe0b-d63c-4f9b-a5a8-a5e4958e077b|title=哥倫比亞一救護飛機墬毀|publisher=TRT|accessdate=2014-03-14|language=zh-hans}}</ref>時不幸撞上樹木，引發爆炸<ref>{{cite web |url=http://news.cts.com.tw/cts/international/201403/201403131391000.html|title=哥國小飛機迫降撞樹 爆炸釀6死|publisher=華視 |accessdate=2014-03-13}}</ref>，機上除了正副駕駛外，搭載的醫護與傷患等全都不幸遇難。
