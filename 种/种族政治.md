'''种族政治'''是指政客利用种族问题来推动议程的政治行为。
==马来西亚==
马来西亚[[政治人物|政治人物]][[丹斯里|丹斯里]][[拿督|拿督]][[郑可扬|郑可扬]]提出：“马来西亚已经实行了51年的种族政治，我们知道这是不利于各种族团结的，因为每个党派都只代表它所代表的种族集团.......当所有种族都加入一个党派时，没有一个人将成为党内的第一.......当一个马来族、[[马来西亚华人|华裔]]或[[马来西亚印度人|印度裔]]的英雄很容易，但成为马来西亚的英雄却很难......国家正面临着经济问题，政府和政党提出马来西亚该如何团结人民并面对这些挑战的议程是非常重要的......”<ref>{{cite web |url=http://thestar.com.my/news/story.asp?file%3D%2F2008%2F8%2F11%2Fnation%2F22053356%26sec%3Dnation |title=Archived copy |accessdate=2008-08-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20080814073017/http://thestar.com.my/news/story.asp?file=%2F2008%2F8%2F11%2Fnation%2F22053356&sec=nation |archivedate=2008-08-14 |df= }}</ref>

{{tsl|en|Marina Mahathir|玛丽娜·马哈迪}}写道：“同样的事情发生在我们的国家。不幸的是，种族政治还没有真正平息下去，有些人反应就好像种族清洗刚刚发生一样......”<ref>{{cite web |url=http://thestar.com.my/news/story.asp?file%3D%2F2008%2F3%2F26%2Ffocus%2F20746652%26sec%3Dfocus |title=Archived copy |accessdate=2008-08-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20121006021344/http://thestar.com.my/news/story.asp?file=%2F2008%2F3%2F26%2Ffocus%2F20746652&sec=focus |archivedate=2012-10-06 |df= }}</ref>

有人问及[[倪可汉|倪可汉]]“你最不喜欢马来西亚人的是什么？”他回答道：“种族政治”。<ref>{{cite web |url=http://thestar.com.my/news/story.asp?file%3D%2F2008%2F5%2F1%2Fmyrepresentative%2F21085441%26sec%3Dmyrepresentative |title=Archived copy |accessdate=2008-08-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20121006021403/http://thestar.com.my/news/story.asp?file=%2F2008%2F5%2F1%2Fmyrepresentative%2F21085441&sec=myrepresentative |archivedate=2012-10-06 |df= }}</ref>

马来西亚前[[总理|总理]][[马哈迪·莫哈末|马哈迪·莫哈末]]因[[三美威魯|三美威魯]]在其网志上言论而将其视为种族主义者。

[[國際紐約時報|國際紐約時報]]的菲利普·鲍灵提到马来西亚的政治组织长期以来一直以种族为主，[[伊斯兰教|伊斯兰教]]有时成为种族政治的一种手段，这是衡量竞争对手对马来种族进步承诺的标准。<ref>[http://www.iht.com/articles/2006/08/29/opinion/edbowring.php]</ref>
