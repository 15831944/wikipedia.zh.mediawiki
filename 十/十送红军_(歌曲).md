《'''十送红军'''》是一首由[[张士燮|张士燮]]作词、[[朱正本|朱正本]]作曲于1961年<ref name="a1"/>创作的以送别[[中国工农红军|红军]][[长征|长征]]为题材的歌曲。常被认为是一首[[江西民歌|江西民歌]]，但其实此歌是由张士燮、朱正本根据[[陈裕光_(作曲家)|陈裕光]]整理的[[兴国县|兴国]][[客家山歌|客家山歌]]《送郎调》（[[采茶戏|采茶戏]]中的《长歌》）改编创作的一首江西民歌风格的歌曲<ref>[http://news.xinhuanet.com/mil/2011-04/10/c_121286905.htm 《十送红军》为何少了四“送”？ ]，新华网，2011-04-10，2011-05-06导入。</ref><ref>[http://www.jxnews.com.cn/jxfzb/system/2007/04/25/002472037.shtml 陈裕光：赣南民歌一滴忧伤的泪]</ref><ref>[http://club.china.com/data/thread/1011/2743/24/28/3_1.html 《十送红军》真相：那些被异形附身的客家民歌] {{Wayback|url=http://club.china.com/data/thread/1011/2743/24/28/3_1.html |date=20120710212748 }}，中华网，2012-07-07，2012-12-19导入。</ref>。2001年播出的电视连续剧《[[长征_(电视剧)|长征]]》曾使用此曲作为[[插曲|插曲]]和[[片尾曲|片尾曲]]。[[宋祖英|宋祖英]]演唱的片尾曲，亦是此歌最为知名的演唱版本<ref name="a1">{{Cite web |url=http://www.hinews.cn/news/system/2011/05/26/012601300.shtml |title=《十送红军》创作溯源：曲作者仅用一小时谱成|accessdate=2018-11-26 |author= |date=2011-05-26 |work= |publisher=|language=Zh-hans }}</ref>。

该曲虽名为“十送红军”，但只有“六送”（一、三、五、七、九、十），并非歌曲曾被删减，而是创作时只写了“六送”，为的是避免歌曲显得太于“絮烦”了。但即使只有“六送”该曲仍比较长（要想完整唱下来，要近15分钟）。后来创编《[[东方红_(音乐舞蹈史诗)|东方红]]》时，也是因为曲长的问题没被使用<ref name="搞音乐曾是工作需要 大胆违令成就了经典红歌">[http://culture.ifeng.com/8/detail_2011_05/03/6120144_0.shtml 搞音乐曾是工作需要 大胆“违令”成就了经典红歌]，凤凰网，2011-05-03，2011-05-06导入。</ref>。电视剧《长征》片尾曲则只有四送“四送”（一、七、九、十）<ref name="a1"/>。

== 创作背景 ==

1960年，当任的空军司令[[刘亚楼|刘亚楼]]出访[[朝鲜|朝鲜]]期间欣赏了大型歌舞史诗《[[三千里江山|三千里江山]]》，朝鲜战士的表演给他留下了深刻的印象。次年，他便给空政文工团布置了创作革命历史歌曲晚会《[[革命历史歌曲表演唱|革命历史歌曲表演唱]]》的任务。

虽然晚会歌曲搜集本着“采集革命历史歌曲，不新创作”的原则进行，但由于关于[[中国工农红军|红军]]开始[[长征|长征]]时[[中央苏区|根据地]]人民送别红军的情景的歌曲没有找到合适的，所以当时负责文字、音乐、舞蹈创作的张士燮、朱正本等人创作了《十送红军》<ref name="搞音乐曾是工作需要 大胆违令成就了经典红歌"/>。当年，随着《革命历史歌曲表演唱》公演，此曲一炮而红<ref name="a1"/>。

== 陕西版镇巴县版 ==

[[陕西省|陕西省]][[镇巴县|镇巴县]]《十送红军》版本的曲调为民歌“绣荷包”。除地名不同外，多处内容与张士燮、朱正本版本相似。而且“一送”至“十送”完整<ref name="a1"/>。
