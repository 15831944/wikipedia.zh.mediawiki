{{engname|《十二生肖》|''CZ12: Chinese Zodiac''}}是由[[成龙|成龙]]担任导演并主演的[[动作电影|动作电影]]，以一群赏金猎手追逐[[圆明园十二生肖兽首铜像|圆明园十二生肖兽首铜像]]为题材。外景地包括[[巴黎|巴黎]]、[[北京|北京]]、[[台湾|台湾]]、[[香港|香港]]、[[澳洲|澳洲]]、[[拉脱维亚|拉脱维亚]]等地。<ref name="pyw " />成龙称这将是他的最后一部大型动作片。影片于2012年12月20日在中国大陆和香港上映，本片在大陆累计票房达到8.8亿元人民币，累计观影人次突破2000万，在当年位列中国电影史上华语电影票房亚军（仅次于同档期的《泰囧》12亿元票房）。<ref>{{Cite web|url=http://movie.mtime.com/boxoffice/#CN/all|title=即将上映 - 观影指南 - Mtime时光网|accessdate=2017-08-28|work=movie.mtime.com}}</ref>全球票房累计超过1.7亿美元。<ref>{{Cite web|url=http://www.boxofficemojo.com/movies/intl/?page=&id=_fCHINESEZODIAC01|title=Chinese Zodiac (2012)|accessdate=2017-08-28|work=www.boxofficemojo.com}}</ref>

==劇情==
為了賺取國際文物販子Lawrence懸賞的巨額獎金，受古董商人所聘的國際奪寶獵人JC（成龍飾）率領奪寶精英小組四處奔波追回北京圓明園「十二生肖青銅獸首」中失散的最後六個獸首。小組成員除了以能打愛耍寶的JC為首領，當中還包括全能飆車打手加高手Simon（權相佑飾）、技術控制好手David（廖凡飾），天才鑒寶大師Coco（姚星彤飾）以及跆拳道美人Bonnie（張藍心飾）。 

JC率領奪寶小組首站前往據傳置身於法國的兩個青銅獸首，在尋寶過程中，他卻遇上中國文物專家關教授的留法女兒Coco，更沒想到關教授父女的愛國情操，卻與自己金錢至上的性格逕相違背。在機緣巧合之下JC認識落魄貴族Katherine，得知另兩個青銅獸在一座不為人知的南海熱帶島嶼，當他們前往島上的沉船遺蹟時，卻遭遇一群各國種族組成的聯合海盜團阻擾，面對一連串的危機四伏，這群擅於偷天換日的組員們是否能突破困境擁有圓滿的人生？而一向惜命如金的傑克在龐大錢財與真愛之間，又該如何抉擇... <ref>[http://app2.atmovies.com.tw/film/filmdataApp2.cfm?filmid=fccn46114336&fr=blade05 開眼電影網-十二生肖 Chinese Zodiac]</ref>

== 花絮 ==
该片于2011年6月份开拍，是[[成龙|成龙]]的第101部作品。开拍地點在[[法国|法国]]，而[[尚蒂伊城堡|尚蒂伊城堡]]和[[孔代博物馆|孔代博物馆]]是首次允许电影製作隊伍进入拍攝，并且把[[路易十六|路易十六]]座椅等国家级文物供给成龙作为电影道具使用。<ref name="pyw ">{{cite web|url=http://zzrb.zynews.com/html/2011-12/12/content_331733.htm|title=成龙《十二生肖》|accessdate=2011-12-12|work=|publisher=郑州日报|language=zh-cn}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

电影在全球各地实地取景，到2012年4月18日，已经完成在[[法国|法国]]、[[北京|北京]]和[[萬那杜|萬那杜]]等地取景，从4月开始，转战[[拉脱维亚|拉脱维亚]]取景，<ref>{{cite web |url=http://ent.huanqiu.com/moive/hk/2012-04/2629459.html |title=成龙将携影片《十二生肖》 参加戛纳电影节 |publisher=环球时报 |accessdate=2012-04-18 |work= |language =zh-cn}}</ref>于5月份杀青。<ref>{{cite web |url=http://ent.qq.com/a/20120310/000022.htm |title= 《十二生肖》5月杀青 成龙：4分钟戏份拍2个月|accessdate= 2012年|work=|publisher=腾讯网 |language = zh-cn}}</ref>

著名美国音乐人[[肯尼·基|肯尼·基]]也被成龙邀请到剧中出演角色。<ref>{{cite web |url= http://news.mtime.com/2011/07/27/1465938.html|title=肯尼基也对演戏感兴趣 成龙邀其客串《十二生肖》 |accessdate= 2012年|work=|publisher=时光网 |language = zh-cn}}</ref>

该片精彩戏幕为“枪火追逐、山地落伞、盘路追车、滑梯缠斗、多角度爆破”，武打镜头从头到尾不间断，而该片获得了美国IMAX Corporation的支持和签约。<ref>{{cite web |url=http://ent.163.com/11/1211/22/7L1DRTF8000300B1.html |title=《十二生肖》发预告 2012年12月12日正式上映 |accessdate= 2012年|work=|publisher=网易 |language = zh-cn}}</ref>

2012年12月12日，该片全球首映礼在[[北京市|北京市]][[国家体育馆|国家体育馆]]举行，歌唱家[[刘媛媛_(歌唱家)|刘媛媛]]受邀出席。<ref>{{cite web |url= http://news.xinhuanet.com/ent/2012-12/13/c_124089223.htm|title=《十二生肖》首映礼 刘媛媛成龙有望合作电影 |accessdate= 2012年12月13日|publisher= 新华网|archiveurl=http://news.xinhuanet.com/ent/2012-12/13/c_124089223.htm |archivedate=2012年12月17日 |work= |language=zh-cn}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/c/2012-12-14/16043812020.shtml |title= 刘媛媛给成龙捧场 亮相《十二生肖》首映|accessdate=2012年12月14日 |publisher= 新浪网|archiveurl=http://ent.sina.com.cn/m/c/2012-12-14/16043812020.shtml |archivedate=2012年12月17日 |work= |language=zh-cn}}</ref>

== 票房 ==
;{{CHN}}大陆
《十二生肖》于2012年12月20日在中国上映，截止2013年1月13日，本片累计票房达到8.07亿元人民币，累计观影人次突破2000万，位列中国电影史上华语电影票房亚军（仅次于同档期的《泰囧》12亿元票房，后被《[[西游·降魔篇|西游·降魔篇]]》超越）<ref>{{cite web|url=http://e.weibo.com/2304129841/zezrUoNFu| language= zh| title="十二生肖"累计票房达到8亿元人民币|publisher=中国电影报|date=2013-01-15 |accessdate= }}</ref>。

;{{HK}}
本片於[[香港|香港]]直到下映為止總票房約$1500萬港幣，依票價90港幣來算，粗估有17萬觀影人次，約香港總人口2%，票房慘澹。<ref>[https://tw.news.yahoo.com/%E6%88%90%E9%BE%8D-%E7%94%9F%E8%82%96-%E9%A6%99%E6%B8%AF%E7%A5%A8%E6%88%BF%E8%B7%8C%E5%87%BA3%E5%90%8D%E5%A4%96-213000936.html 成龍《生肖》香港票房跌出3名外　YAHOO新聞]</ref>

'''东南亚'''

《十二生肖》登陆东南亚后立刻登上周票房冠军，在跨年一周全面问鼎新、马、泰三地票房。在新加坡、马来西亚、泰国等地票房全面压制《霍比特人》等好莱坞大片，屡屡占领周票房冠军位置，成为近年来华语片海外票房之最。在新加坡上映22天时就已打破亚洲电影在新加坡票房记录。<ref>{{Cite web|url=http://ent.qq.com/a/20130114/000602.htm|title=《十二生肖》内地票房破8亿 全球票房将破10亿_娱乐_腾讯网|accessdate=2017-08-28|work=ent.qq.com|language=zh-CN}}</ref>

;北美
[[2013年|2013年]][[10月18日|10月18日]]上映，截至10月27日，累計票房僅9.4萬美元，依票價8美元計算，粗估有1萬觀影人次，為成龍作品在北美的最低票房。《洛杉矶时报》評價稱：“看电影的时候非常想提前离场，观众看起来很迷茫，并且都在不停看表。”<ref>{{cite web|url=http://news.mtime.com/2013/10/30/1519983.html| language= zh| title="十二生肖"北美票房仅9万 创成龙功夫片新低|publisher=時光網|date=2013-10-30 |accessdate= }}</ref>
