{{佛教}}
'''十大弟子'''是[[佛陀|佛陀]]在世時十個主要[[弟子|弟子]]。
==记载==
* [[北傳佛教|北傳佛教]]的十大弟子：<ref>北傳十大弟子說法來自《[[維摩詰所說經|維摩詰所說經]]弟子品》所說的十位弟子。</ref>
# [[大迦葉|大迦葉]]<ref>這裡「葉」讀作「攝」；佛陀另有弟子[[迦葉三兄弟|迦葉三兄弟]]，原为拜火教 ，與此摩訶迦葉無關。</ref>：[[頭陀|頭陀]]<ref>「頭陀」是梵語Dhuta的諧音直譯，華語的意思是「抖擻」、「清心寡欲」，同時也是佛教的修行方式之一。</ref>第一。也稱「摩訶迦葉」。
# [[目犍連|目犍連]]：[[神通|神通]]第一。也稱「摩訶<ref>「摩訶」是梵語諧音直譯，華語的意思為「大」。</ref>目犍連」、「目犍連」、「大目犍連」、「目連」。
# [[富樓那|富樓那]]：說法第一。全稱「富樓那彌多羅尼子」。
# [[須菩提|須菩提]]：[[解空|解空]]第一。
# [[舍利弗|舍利弗]]：[[智慧|智慧]]第一。也有人翻譯成「鶖鷺子」、「舍利子」。
# [[羅睺羅|羅睺羅]]：[[密行|密行]]第一。原是佛陀的獨子，「羅睺羅」有累贅、阻礙的意思。
# [[阿難陀|阿難陀]]：多聞第一。原是佛陀的堂弟，許多佛經中也直接稱為「[[阿難|阿難]]」。
# [[優婆離|優婆離]]：持戒第一。
# [[阿那律|阿那律]]：[[天眼|天眼]]第一。也稱「[[阿尼律陀|阿尼律陀]]」。
#[[迦旃延|迦旃延]]：議論第一。也稱「摩訶迦旃延」、「大迦旃延」。
