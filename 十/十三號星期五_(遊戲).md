{{vgname|十三號星期五|en=Friday the 13th: The Game}}是建基於[[十三號星期五_(電影)|同名電影系列]]，由{{link-en|IllFonic|IllFonic}}開發並由{{link-en|Gun Media|Gun Media}}發行在[[Microsoft_Windows|Microsoft Windows]]、[[PlayStation_4|PlayStation 4]]和[[Xbox_One|Xbox One]]平台的[[開放世界|半開放]]式[[恐怖遊戲|生存恐怖遊戲]]<ref name=":4" />，也是第一個{{link-en|IllFonic|IllFonic}}使用[[虚幻引擎|虚幻引擎4]]作引擎的遊戲<ref name=":02">{{Cite web |url=https://forums.unrealengine.com/showthread.php?93747-Off-Site-Project-Advena-Level-Designer |title=Contract / Paid Off-Site Level Project Advena Designer |last=Gonzalez |first=Kedhrin |date=2015-12-08 |website= |publisher=[[Unreal_Engine|Unreal Engine]]|access-date=2016-08-15}}</ref>，遊戲在2017年5月26日於全球推出<ref name=":2">{{Cite web|url=https://friday-the-13th-the-game.backerkit.com/hosted_preorders/34622|title=Friday the 13th: The Game - BackerKit|last=|first=|date=|website=|publisher=BackerKit|access-date=2016-08-13}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。該遊戲是一個不對稱的多人遊戲，遊玩時會有七名玩家控制七名水晶湖露營區的巡邏員，對抗由一位玩家控制的[[面具傑森魔|面具傑森魔]]。

遊戲最初以《亂砍者Vol.1：夏令營》（''Slasher Vol. 1: Summer Camp''）為名於2015年年中開始開發<ref name=":5">{{Cite web |url=https://www.youtube.com/watch?v=P878Ub564eA |title=Summer Camp Announcement Trailer |last= |first= |date=2014-10-31 |website= |publisher=[[YouTube|YouTube]]|access-date=2016-08-17}}</ref>，後來Gun Media為了獲得開發遊戲的許可，遂與十三號星期五系列電影的導演{{link-en|肖恩·坎寧安|Sean S. Cunningham}}進行了商談，在經過一連串會談後，IllFonic終獲得遊戲的開發權，《亂砍者Vol.1：夏令營》亦因此改名為《十三號星期五》。該遊戲在[[Kickstarter|Kickstarter]]和BackerKit籌募了大約109萬美金的資金<ref name=":2" /><ref name=":6" />。

== 遊戲玩法 ==
[[File:Friday13.jpg|left]]遊戲的截圖]]
《十三號星期五》是一個以[[十三號星期五_(電影)|同名電影系列]]中的水晶湖露營區為場景的[[開放世界|半開放]]式第三人稱[[恐怖遊戲|生存恐怖遊戲]]<ref name=":4">{{Cite news |url=http://www.gamespot.com/articles/friday-the-13th-game-revealed-dev-promises-blood-g/1100-6431388/ |title=Friday the 13th Game Revealed, Dev Promises Blood, Guts, Gore, Brutal Kills - GameSpot |last=Makuch |first=Eddie |date=October 13, 2015 |work=[[GameSpot|GameSpot]]|access-date= |via=}}</ref>，遊戲的背景時間設於20世紀80年代中期，目的是為了更好地捕捉原作的感覺。

《十三號星期五》是一個{{link-en|不對稱性 (形狀)|Asymmetry|不對稱}}的[[多人電子遊戲|多人遊戲]]，一局遊戲最多可以有八個人同時遊玩。其中一位玩家會扮演要盡可能殺死其他玩家（巡邏員）的[[面具傑森魔|面具傑森魔]]<ref name=":1">{{Cite news |url=http://www.cinemablend.com/games/Friday-13th-Video-Game-Lets-You-Become-Jason-92107.html |title=Friday The 13th Video Game Lets You Become Jason - CinemaBlend |last= |first= |date= |work=CinemaBlend|access-date=August 13, 2016 |via=}}</ref>，這名扮演傑森的玩家可以覺察鄰近自己的巡邏員和傳送至自己於地圖上任何一個想去的地方<ref name=":0">{{Cite web |url=https://www.youtube.com/watch?v=ecF59xvFZvQ |title=Friday the 13th Game Gameplay Demo (E3 2016) |last= |first= |date=June 14, 2016 |website= |publisher=[[YouTube|YouTube]]|access-date=August 13, 2016}}</ref>。其他七名玩家則扮演水晶湖露營區的巡邏員，他們的目標是完成所有任務以逃出營區或於限定時間內生存。該七名玩家也可以通過團隊合作殺死傑森以達成「史詩級的勝利」，但這極難做到<ref name=":1" />。另外，當其中一位玩家滿足某些條件時，他可以變為控制{{link-en|湯米·賈維斯|Tommy Jarvis}}。

遊戲原本計劃僅限於多人遊戲模式，但IllFonic打算如果遊戲營收達到16.25萬美金的話將會於2017年夏季添加單人遊戲模式<ref>{{Cite news |url=http://www.polygon.com/2016/10/13/13279064/friday-the-13th-game-delayed-single-player-confirmed |title=Friday the 13th game delayed to 2017, adds single-player |last=McWhertor |first=Michael |date=October 13, 2016 |work=[[Polygon_(website)|Polygon]]|access-date=April 13, 2017 |via=[[Vox_Media|Vox Media]]}}</ref>，在該模式中玩家會控制傑森，目標是殺死所有由電腦操控的巡邏員。除此之外，有一些任務將會改編自電影的情節，而其他任務則是遊戲原創<ref>{{Cite news |url=https://www.kickstarter.com/projects/613356213/friday-the-13th-the-game/posts/1707655 |title=HUGE UPDATE: Adding Single Player Campaign and Offline AI Bots, Tommy Jarvis, New Map - Packanack Lodge, and much more!!! |date=October 13, 2016 |work=[[Gun_Media|Gun Media]]|access-date=June 7, 2017 |via=[[Kickstarter|Kickstarter]]}}</ref><ref>{{Cite news |url=https://twitter.com/Friday13thGame/status/788123681021104128 |title=Friday the 13th Game on Twitter |date=October 17, 2016 |work=Friday the 13th: The Game |access-date=June 7, 2017 |via=[[Twitter|Twitter]] |deadurl=yes |archiveurl=https://web.archive.org/web/20170607152848/https://twitter.com/Friday13thGame/status/788123681021104128 |archivedate=2017年6月7日 }}</ref>。

== 開發 ==
{{quote box|肖恩（肖恩·坎寧安）在2015年年初來到我們這裡，但我們很快就發現當我們想獲得他的許可時是不會有一個正常的商務對話的。這是一個相互敬佩和尊重的對話，肖恩也很快注意到我們對《十三號星期五》的熱情，並在接下來的幾個月進行了好幾個令人難以置信的會談。我們決定改進我們的《亂砍者Vol.1：夏令營》，並欣然接受《十三號星期五》的改編遊戲許可，而肖恩也對我們製作的遊戲感到驚訝<ref name=":2" />。|width=30%}}

在IllFonic{{link-en|Nexuiz (2012年遊戲)|Nexuiz (2012 video game)|重製}}《[[Nexuiz|Nexuiz]]》和Gun Media於2015年發行{{link-en|火線突破：最終時刻|Breach & Clear: Deadline}}後，IllFonic開始製作以水晶湖營地為背景的《亂砍者Vol.1：夏令營》<ref>{{Cite news |url=http://gamerant.com/slasher-summer-camp-screenshots-details/ |title='Slasher Vol 1: Summer Camp' Reveals First Screenshots, Details |last=Cooper |first=Dalton |date= |work=GameRant|access-date=2016-08-13 |via=}}</ref>，之後《[[十三號星期五_(電影)|十三號星期五]]》的導演{{link-en|肖恩·坎寧安|Sean S. Cunningham}}與IllFonic進行了一系列的會談<ref>{{cite web |url=http://www.playstationlifestyle.net/2015/11/11/friday-the-13th-the-game-ps4/ |title=Friday the 13th: The Game Reaches Funding Goal, Licensed Horror Will Hit PS4 in 2016 |website=PlayStationLifestyle |accessdate=2015-11-11}}</ref><ref>{{cite web |url=http://www.egmnow.com/articles/news/egm-exclusive-fleshbreaking-new-friday-the-13th-videogame-featuring-horror-icon-jason-voorhees-in-development/ |title=EGM EXCLUSIVE: Fleshbreaking new Friday the 13th videogame featuring horror icon Jason Voorhees in development |website=EGMNOW |accessdate=2015-01-09}}</ref>。此外，《亂砍者Vol.1：夏令營》也是IllFonic第一個使用[[虚幻引擎|虚幻引擎4]]作引擎的遊戲<ref name=":02" />。遊戲的執行總監和製作人蘭迪·格林貝克（Randy Greenback）在BackerKit與[[Kickstarter|Kickstarter]]上開設集資網站，以籌備遊戲開發的資金。最終他們一共籌募了{{US$|126萬6570元}}<ref name=":2" /><ref name=":6">{{Cite web |url=https://www.kickstarter.com/projects/613356213/friday-the-13th-the-game |title=Friday the 13th: The Game by Randy Greenback - Gun Media - Kickstarter |last=Greenback |first=Randy |date= |website= |publisher=[[Kickstarter|Kickstarter]]|access-date=2016-08-13}}</ref><ref>{{cite web |url=http://bloody-disgusting.com/news/3369665/friday-the-13th-the-game-is-a-kickstarter-success/ |title=‘Friday the 13th: The Game’ is a Kickstarter Success |website=BloodyDisgusting |accessdate=2015-11-11}}</ref>。

2015年10月13日，Gun Media在他們的YouTube頻道上發布宣傳預告片<ref>{{Cite web |url=https://www.youtube.com/watch?v=VjNm2pthBfM |title=Friday the 13th: The Game - Official Announcement Trailer |last= |first= |date=2015-10-13 |website= |publisher=[[YouTube|YouTube]]|access-date=2016-08-14}}</ref>，並公布《亂砍者Vol.1：夏令營》已更名為《十三號星期五》<ref>{{Cite news |url=http://gamerant.com/friday-the-13th-game-confirmed/ |title='Friday the 13th' Arrives on Consoles October 2015 |last=Connolly |first=Denny |date= |work=GameRant|access-date=2016-08-13 |via=}}</ref><ref name=":3">{{cite web |url=https://www.kickstarter.com/projects/613356213/friday-the-13th-the-game |title=HUGE NEWS! Summer Camp Becomes Official Friday the 13th Game! |website=DreadCentral |accessdate=2015-10-14}}</ref>。萬聖節當天，Gun Media上傳了一段影片在其YouTube頻道上，片中可見{{link-en|哈利·曼弗雷迪尼|Harry Manfredini}}的電腦文件，暗示他正為遊戲譜寫音樂<ref>{{Cite web |url=https://www.youtube.com/watch?v=_c7-mzDwLfs&feature=share |title=Friday the 13th: The Game - Harry Manfredini Says Happy Halloween! |last= |first= |date= |website= |publisher=|access-date=}}</ref>。同年11月4日，Gun釋放一段顯示遊戲環境的[[軟件版本週期#開發期|準預覽版本]]影片<ref>{{Cite web |url=https://www.youtube.com/watch?v=rLN-6EBeh9k |title=Friday the 13th: The Game - Pre-Alpha Camp Crystal Lake Environment Fly-throughs |last= |first= |date=2015-11-04 |website= |publisher=[[YouTube|YouTube]]|access-date=2016-08-14}}</ref>。6天後（11月10日），IllFonic於Gun Media的YouTube頻道上載關於《十三號星期五》的測試和開發的影片，其中包括傑森的[[三維模型|三維模型]]<ref>{{Cite web |url=https://www.youtube.com/watch?v=t4W_VdnIK0I |title=Friday the 13th: The Game - Behind the Scenes at IllFonic |last= |first= |date=2015-11-10 |website= |publisher=[[YouTube|YouTube]]|access-date=2016-08-14}}</ref>。

2016年2月25日，Gun和IllFonic釋放一段有著{{link-en|凱恩·霍德|Kane Hodder}}、{{link-en|塔拉·佩吉|Tarah Paige}}和瑞安·斯塔茨（Ryan Staats）的錄像，前者在遊戲中飾演傑森，後兩者飾演男性和女性巡邏員，[[動作捕捉|動作捕捉]]則由[[數字王國|數字王國]]負責。在該段錄像中霍德做出許多動作，以供開發團隊製作傑森殺人的動畫<ref>{{Cite web |url=https://www.youtube.com/watch?v=4j3-Nnu7kxI |title=Friday the 13th: The Game - Behind the Scenes of the Motion Capture Shoot |last= |first= |date=2016-02-25 |website= |publisher=[[YouTube|YouTube]]|access-date=2016-08-14}}</ref>。

在2016年1月的{{link-en|PAX展|PAX (event)|PAX South}}上，IllFonic和Gun Media開發團隊向公眾展示[[軟件版本週期#開發期|預覽版本]]的影片和一系列傑森殺人的動畫<ref>{{Cite web |url=http://www.idigitaltimes.com/friday-13th-game-panel-pax-south-2016-reveals-new-kill-animations-jason-vorhees-508063 |title='Friday the 13th: The Game' Panel At Pax South Reveals New Kill Animations For Jason Voorhees |last=Mozuch |first=Mo |date=2016-01-29 |website= |publisher=iDigitalTimes |access-date=2016-08-13 |archive-url=https://web.archive.org/web/20160827075707/http://www.idigitaltimes.com/friday-13th-game-panel-pax-south-2016-reveals-new-kill-animations-jason-vorhees-508063 |archive-date=2016-08-27 |dead-url=yes }}</ref> ，之後他們亦在接下來的5月舉行的[[E3_2016|E3 2016]]中公布遊戲玩法<ref name=":03">{{Cite web |url=https://www.youtube.com/watch?v=ecF59xvFZvQ |title=Friday the 13th Game Gameplay Demo (E3 2016) |last= |first= |date=2016-06-14 |website= |publisher=[[YouTube|YouTube]]|access-date=2016-08-13}}</ref>。同年9月2日，Gun Media於{{link-en|PAX展|PAX (event)|PAX West}}釋出有著[[湯姆·薩維尼|湯姆·薩維尼]]專門設計的傑森殺人動畫片段的預告片《XIII》<ref>{{cite web |url=https://www.youtube.com/watch?feature=youtu.be&v=rUmkTGSyRA8 |title=XIII PAX West Trailer 2016 -- Friday the 13th: The Game |date=2016-09-02 |publisher=[[YouTube|YouTube]] |accessdate=2016-09-05}}</ref>。

== 原聲帶 ==
《十三號星期五》遊戲的作曲者為原電影的作曲家{{link-en|哈利·曼弗雷迪尼|Harry Manfredini}}，該原聲帶的實物版將包括遊戲的[[概念藝術|概念藝術]]和{{link-en|封套內容介紹|Liner notes|封套內容介紹文字}}<ref>{{Cite web|url=https://friday-the-13th-the-game.backerkit.com/hosted_preorders/34629|title=Friday the 13th: The Game - BackerKit|last=|first=|date=|website=|publisher=BackerKit|access-date=2016-08-13}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。Gun Media在開發《十三號星期五》期間，他們從即將發布的原聲帶中提前公布了兩首曲目，這兩首曲目後來被包含在2015年11月9日發行的《Harry Manfredini Full Track - 01》中<ref>{{Cite web |url=https://www.youtube.com/watch?v=dp5ZikcQ1uA |title=Friday the 13th: The Game -Harry Manfredini Full Track - 01 |last= |first= |date=2015-11-09 |website= |publisher=[[YouTube|YouTube]]|access-date=2016-08-14}}</ref>和2016年5月14日發行的《Harry Manfredini Full Track - 02》<ref>{{Cite web |url=https://www.youtube.com/watch?v=Hv2y7nTUKY4 |title=Friday the 13th: The Game - Harry Manfredini Full Track - 02 |last= |first= |date=2016-05-14 |website= |publisher=[[YouTube|YouTube]]|access-date=2016-08-14}}</ref>。

在2016年{{link-en|PAX展|PAX (event)|PAX West}}中，Gun Media發放了由{{link-en|Crazy Lixx|Crazy Lixx}}錄製的遊戲預告片，題為《殺手》（''Killer''）<ref>{{cite web|url=http://www.metacritic.com/game/playstation-4/friday-the-13th-the-game/trailers/7322619|title=FRIDAY THE 13TH: THE GAME - PAX WEST 2016 TRAILER|accessdate=2017-07-08}}</ref>。

《十三號星期五》發行後，[[Metacritic|Metacritic]]根據21條評論給予PC版的《十三號星期五》61/100分<ref name="MC-PC" /> ，PS4版為根據13條評論獲得59/100分<ref name="MC-PS4"/>，至於Xbox One版則只有49/100分（8條評論）<ref name="MCXONE"/>，前兩者為「好壞參半」，後者則為「普遍不受歡迎」。

[[GameSpot|GameSpot]]的彼得·布朗（Peter Brown）批評主機版的《十三號星期五》只能「斷斷續續地地遊玩」，他闡述道玩家與朋友之間的私人對決有時候是可行的，但如果是想在沙發上舒舒服服的跟陌生人在線遊玩的話，那玩家不是要等待10分鐘以上的配對，就是完全沒有運氣的找不到任何一個人跟你一起對賽<ref name="GSpot"/>。布朗還舉出遊戲的一些問題，例如地圖有瑕疵、[[幀率|幀率]]不一致、動畫效果不佳和[[碰撞偵測|碰撞偵測]]匱乏等，而這些問題都導出《十三號星期五》是在未開發完畢的情況下發行的，幸好的是遊戲的不對稱性和面具傑森魔的玩法稍微彌補了遊戲的品質<ref name="GSpot"/>。

[[IGN|IGN]]的戴蒙·哈特菲爾德（Daemon Hatfield）讚揚扮演傑森時的元素，但他亦批評扮演巡邏員時的「低級樂趣」，而這些樂趣會破壞你的心情<ref name="IGN"/>。[[Polygon|Polygon]]的奧斯汀·戈斯林（Austen Goslin）則認為《十三號星期五》有其原作的表面水平，但欠缺其精神<ref name="Poly"/>。
