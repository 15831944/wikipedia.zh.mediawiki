'''十五少年漂流記'''（英語：'''''{{lang|en|Two Years' Vacation}}'''''，[[法語|法語]]：'''''{{lang|fr|Deux ans de vacances}}'''''）是[[法國|法國]]小說家[[朱爾·凡爾納|朱爾·凡爾納]]的小說，1888年出版，多次被改編成[[動畫|動畫]]、[[電影|電影]]、[[電視劇|電視劇]]。內容描述15名少年或兒童在無人島生活了兩年的冒險故事。

==故事==
1860年3月9日，一起原因不明的事故造成了一艘名為士羅號的船發生船難，當時在船上的十五名少年漂流到一座島上，在確定了島嶼是[[無人島|無人島]]後，十五名少年在島上選出了首領、發展出制度、發現新的動植物、製作工具，過著與大自然對抗的生活。
在無人島生活兩年後，少年們終於獲救。

==登場人物==
===十五個少年與狗===
漂流到無人島上的十五個人，故事的主角。年齡以1860年時為記。

; 柏里安([[法國|法國]]人 13歲)
: 土木技師之子，聰明活潑，是第二任領袖。

; 德諾班([[英國|英國]]人 13歲)
: 大地主之子，好勝心強，天生具有領袖魅力，和格魯士、威利、維普合組了「德諾班黨」。擅長射擊和翻譯，口才相當好。

; 戈爾登(美國人 15歲)
: 十五少年中年紀最年長的，第一任領袖。

; 狄克(法國人 10歲)
: 柏里安的弟弟，個性調皮搗蛋，令人擔心。

; 格魯士 (英國人 13歲)
: 德諾班的表哥，德諾班黨的一員。

; 巴克達 (英國人 13歲)
: 擅長製作工具，負責擔任書記，寫下眾人在島上發生的事。在結局時、他的日記經過整理出版並熱銷。

; 維普 (英國人 13歲)
: 德諾班黨的一員。

; 威利 (英國人 13歲)
: 德諾班黨的一員。

; 嘉乃特 (英國人 12歲)
: 士羅號的所有人的兒子，和薩威斯關係很好。

; 薩威斯 (英國人 12歲)
: 相當有活力的少年，負責照料家畜。

; 詹金斯 (英國人 9歲)
: 紐西蘭科學協會會長之子。

; 埃佛森 (英國人 9歲)
: 牧師之子，優等生。

; 卡士達 (英國人 8歲)
: 紐西蘭陸軍將校之子，登場人物中最年少的一個。

; 杜魯 (英國人 8歲)
: 紐西蘭陸軍將校之子，比卡士達年長6個月，喜歡吃甜食。

; 莫可 (黑人 12歲)
: 見習水手，擅長做料理和駕船。

; 范恩 (狗)
: 少年們飼養的狗，在狩獵時幫了很大的忙。

==用語==
; 傑亞曼寄宿學校
: 少年們就讀的寄宿學校，是當地上流家庭的孩子就讀的學校。學生都是來自英國、美國、法國或德國的白人子弟。

; 士羅號
: 少年搭乘的船，在事故後解體。

; 傑亞曼島
: 少年在漂流後所上岸的島，以他們自己的寄宿學校命名。他們居住的洞穴命名為「法人洞」，島上的地名也都以他們故鄉的事物來命名。
