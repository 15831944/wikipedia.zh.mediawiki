'''十字分道車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]，為[[行政院農業委員會林務局|林務局]][[阿里山森林鐵路|阿里山森林鐵路]]眠月線、祝山線之[[鐵路車站|鐵路車站]]。

== 車站構造 ==
鐵皮、木造站房各一

== 利用狀況 ==
本站現時為眠月線和祝山線的停靠站。
== 歷史 ==
*1915年：設站。

*1986年：隨祝山線通車而搬至現址。
