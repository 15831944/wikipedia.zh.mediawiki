'''铁血十八星旗'''，简称'''十八星旗'''，又称'''鐵血旗'''、'''九角旗'''、'''九角十八星旗'''，是[[武昌起义|武昌起义]]后[[中華民國|中華民國]][[湖北軍政府|湖北軍政府]]宣告成立时所采用的[[旗帜|旗帜]]。<ref>{{Cite web |url=http://news.takungpao.com/mainland/tkchina/2013-05/1622134_2.html|title=中国国旗的演变史|date=2013-05-20|publisher=大公网}}</ref>原是[[湖北|湖北]]革命团体[[共进会|共进会]]的会旗。1907年8月间，[[焦达峰|焦达峰]]、[[刘公|刘公]]、[[孫武_(民國)|孙武]]等一批在[[日本|日本]]的[[同盟会|同盟会]]会员筹组共進會，任务是谋划准备在[[长江|长江]]中游的反清武装起义，议定以十八星旗为会旗。1911年9月，在[[同盟会|同盟会]]推动下策划[[武昌起义|武昌起义]]。[[文学社|文学社]]和共进会召开联席会议，组成领导起义的总指挥部，定十八星旗为旗帜。武昌起义在1911年10月10日晚7时左右爆发，起义军成功占领[[武昌|武昌]]全城。 10月11日，[[中華民國軍政府鄂軍都督府|中華民國軍政府鄂軍都督府]]（俗稱[[湖北軍政府|湖北軍政府]]）宣告成立，十八星旗为其旗帜。1912年1月10日，[[中華民國臨時參議院|中華民國臨時參議院]]通过专门决议，以[[五族|五族]]共和旗（簡称[[五色旗|五色旗]]）为[[中国国旗|中国国旗]]。以十八星旗为陆軍旗。不過[[北洋政府|北洋政府]]很快就停止使用十八星旗，以五色旗为[[國旗|國旗]]，以十九星旗为[[中華民國陸軍|中華民國陸軍]][[軍旗|軍旗]]，以[[青天白日红旗|青天白日红旗]]为[[海軍旗|海軍旗]]。1928年（民国十七年）12月17日，[[國民政府|國民政府]]立法通过[[青天白日滿地紅|青天白日滿地紅]]为中国的正式国旗，颁行全中国使用，全面取代五色旗和十八星旗。雖然今日十八星旗、十九星旗已無法律地位，不過目前仍有部分支持[[中华联邦主义|中华联邦主义]]人士常使用這兩面旗幟。

==十八星旗意义==
铁血十八星旗，由[[红|红]][[黄|黄]][[黑|黑]]三色组成：红底与黑九角象征“血”与“铁”，即革命须抱[[鐵血演說|铁血]]主义；黑九角代表《[[禹貢|禹貢]]》中记载的[[冀州|冀]]、[[兗州|兗]]、[[青州|青]]、[[徐州|徐]]、[[揚州|揚]]、[[荊州|荊]]、[[豫州|豫]]、[[梁州|梁]]、[[雍州|雍]][[九州|九州]]；黑九角内外两圈各九颗共十八颗圆星，代表关内汉族的[[十八行省|十八行省]]；星呈金黄色，表示与满清对立之汉族[[炎黄子孙|炎黄子孙]]。清朝政府直到19世紀為止只准许汉族在关内居住，不允许汉族随便迁入居住的有关外的[[满洲|满洲]]（[[东三省|东三省]]）、[[西藏|西藏]]（[[青海|青海]]）、[[蒙古|蒙古]]（内外蒙）、[[新疆|新疆]]。铁血十八星旗所能体现的人口理念，也绝对不是通过自我阉割来提高所谓的“人均资源”。所谓内九州，当然是指中国；九州就是中国的代称。所谓外九州，就是汉人在国境外的人口、商业、文化、军事力量等等涉及的所有地方{{FACT}}。

==十九星旗==

[[File:Chinese-army_Wuhan_flag_(1911-1928)_19_dots.svg|thumb]]。]]

十九星旗是十八星旗的变体。由于[[东北三省|东北三省]]的议会代表提出原有十八星旗没有包含东三省。1912年（民國元年）1月10日，[[中华民国临时参议院|中华民国临时参议院]]通过专门决议，以十八星旗为[[中華民國軍旗|陸軍旗]]。<ref>{{Cite web |url=http://www.epochtimes.com/gb/11/1/16/n3144268.htm |title=李后主：《让子弹飞》中的暗线故事大解读|date=2013-05-20|publisher=大纪元}}</ref>1912年（民國元年）5月，中华民国臨時參議院在[[北京|北京]]提出“國旗統一案”，仍然以[[五色旗|五色旗]]為国旗，而[[青天白日滿地紅旗|青天白日滿地紅旗]]為[[海軍旗|海軍旗]]。

但是否继续以十八星旗为陸軍旗则受到多數[[東三省|東三省]]籍的臨時參議院[[議員|議員]]反對，因为十八星旗并不包括[[东三省|东三省]]，也不能代表全中国。于是议员们提出意见修改十八星旗，先是有提议在旗中央加上一顆較大的黃星，以表示中国的统一；后又认为中央黃星大有側重中央、忽视地方之意，中央黄星又改成与其他黄星一般大小，成为十九星旗。十九星旗的提案被议会表决通过，取代十八星旗成为[[中華民國軍旗|陸軍旗]]。1912年6月11日，[[中華民國臨時大總統|中華民國臨時大總統]][[袁世凱|袁世凱]]公布此次國旗議案結果，成為中華民國第一個具有法律意義的國旗方案。1928年12月17日[[國民革命軍北伐|國民革命軍北伐]]成功后，国民政府全面废除鐵血旗。

==影视作品==
《[[让子弹飞|让子弹飞]]》影片中多次出现“铁血十八星旗”，但影片背景却是1920年，铁血十八星旗在正式场合下已经不再使用了，文官出身的县长[[张麻子|张麻子]]，在县衙应悬挂当时中华民国的国旗五色旗，但在影片中的[[鹅城|鹅城]]，没有悬挂一面五色旗却都是铁血十八星旗。（此处引用有待核实）
