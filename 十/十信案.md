'''十信案'''，為1985年[[臺灣|臺灣]]發生的「臺北市第十信用合作社」超貸、侵佔、背信與偽造文書案的[[經濟犯罪|經濟犯罪]]事件。

== 弊案過程 ==
「十信」即「[[臺北市第十信用合作社|臺北市第十信用合作社]]」，1911年9月成立，初名為「臺北信用組合」；[[臺灣戰後時期|二戰後]]由臺灣人接收、改組，改名為「臺北市第十信用合作社」，簡稱「臺北十信」。1957年起，臺北十信理事主席均由[[霖園集團|國泰集團]][[蔡萬春|蔡萬春]]家族擔任。

1960年，蔡萬春因大量投資土地引起臺北十信資金流動困難，於是發起「一元開戶」運動，臺北十信存款在短短數月內突破[[新台幣|新臺幣]]一億元，使臺北十信成為全國七十五家[[信用合作社|信用合作社]]中規模最大與營業量最多的單位。

1972年4月24日，[[台北市政府|台北市政府]]宣布，第四信用合作社經營有重大困難，併入第十信用合作社。<ref name = sinica_keyword_history>{{cite web | title = 戰後台灣歷史年表 十信案 | url = http://twstudy.iis.sinica.edu.tw/twht/General/SimpleTwht.asp?page=2&KeyWord=%A4Q%ABH | publisher = 中央研究院主題計畫-台灣研究 | language = zh-tw | accessdate = 2018-07-08}}</ref>

1975年8月11日，[[台北市議會|台北市議會]]決議，要求依《合作社法》辦理十六信與十信合併，並處分失職人員。<ref name = sinica_keyword_history />

1979年10月4日，財政部對十信虛列庫存現金處分：補繳利息一千餘萬元、罰鍰27萬9千元。<ref name = sinica_keyword_history />

1982年，蔡萬春之子[[蔡辰洲|蔡辰洲]]當選[[立法委員|立法委員]]，結合增額立委，與[[劉松藩|劉松藩]]、[[王金平|王金平]]、[[洪玉欽|洪玉欽]]、[[謝生富|謝生富]]、[[李宗仁_(臺南縣)|李宗仁]]、[[李友吉|李友吉]]、[[林聯輝|林聯輝]]、[[蔡勝邦|蔡勝邦]]、[[吳梓|吳梓]]及[[蕭瑞徵|蕭瑞徵]]等立法委員組成[[派系|派系]]「十三兄弟」，經常邀請財經官員，而且向官方[[遊說|遊說]]「合作社理、監事可無限制連選連任」以及「[[信託公司|信託公司]]可承辦[[銀行|銀行]]業務」等。蔡辰洲也將大量資金投入[[房地產|房地產]]事業。

1983年2月28日，立委[[金紹賢|金紹賢]]指出，在《合作社法》審議中，有14位立法委員受十信招待攜眷旅遊[[菲律賓|菲律賓]]與[[香港|香港]]。<ref name = sinica_keyword_history />

1983年，[[中華民國財政部|財政部]]進行一般金融事務檢查時發現臺北十信有不正常貸款現象，因此派員進駐輔導，並給予警告。

1984年初，臺北十信又發生不良放款跳升現象；經過財政部派員進駐輔導一年之後，臺北十信的不正常放款現象確有改善，因此財政部未採取更嚴厲措施。

1984年10月，財政部駐臺北十信人員表示，臺北十信庫存現金有偏低之趨勢；財政部督促臺北十信限期改善，然而情況並無好轉。

=== 1985年弊案爆發 ===

1985年1月底，臺北十信有18家分社、6萬多社員，存款額最高達171億元，為當時規模最大、歷史最久的信用合作社。[[臺北市政府財政局|臺北市政府財政局]]調查臺北十信營業違規的事件登上媒體後，臺北十信各分社出現擠兌提款現象；此時財政部次長[[李洪鰲|李洪鰲]]仍對新聞媒體稱，主因是臺北十信客戶準備發年終獎金給員工，此現象各大銀行皆有，並非只有臺北十信如此。<ref name="茅家琦2003">{{cite book|author=茅家琦|title=蔣經國的一生和他的思想演變|url=https://books.google.com/books?id=nNuAWMnHvyYC&pg=PA411|year=2003|publisher=臺灣商務印書館|isbn=978-957-05-1796-5|pages=411–}}</ref>

1985年2月，《[[天下雜誌|天下雜誌]]》訪問中央銀行總裁[[張繼正|張繼正]]與財政部長[[陸潤康|陸潤康]]時，兩人都曾提出當前金融紀律有問題。張繼正直說，過去臺北十信屢次嚴重違規，但皆因為利益團體力量太大，動員所有人情關說而使得主管單位僅止於糾正處分，但未揭發解決弊案。<ref name="殷允芃1987">{{cite book|author=殷允芃|title=等待英雄|url=https://books.google.com/books?id=C7RgCwAAQBAJ&pg=PT73|date=30 September 1987|publisher=天下雜誌出版|isbn=978-957-9079-19-8|pages=73–}}</ref>

1985年2月9日，由於臺北十信放款總額154億元占存款總額151億元之比率高達102%，顯示臺北十信已無放款能力；財政部為保障存戶合法權益、穩定金融秩序，明令臺北十信停止營業三天，並由[[臺灣省合作金庫|臺灣省合作金庫]]暫為接管，嚴加清查整頓。臺北十信在主管機關的督導下，舉行臨時社員大會。<ref name = sinica_keyword_history /><ref name = china.com.cn>{{cite news | title = 衝垮蔡家天下的“十信風暴” | url = http://big5.china.com.cn/chinese/TCC/haixia/19671.htm | publisher = 中國網際網路新聞中心 | language = zh-tw | accessdate = 2018-07-08}}</ref>

1985年2月10日，財政部宣布，臺北十信現任及前任理監事限制出境、禁止財產轉移。<ref name = sinica_keyword_history />

1985年2月11日，臺北十信營業廳的存款戶擠兌提款仍持續達36億元，部分分社出現無現金可提領現象，此現象造成台灣其他銀行存戶也出現擠兌提款現象。<ref name="茅家琦2003" /><ref name = china.com.cn />

1985年2月12日，臺北十信擠兌提款金額達到61.8億元，連帶影響蔡辰男的國泰信託也出現擠兌提款與退票。台北十信經理[[葉煌良|葉煌良]]捲款兩千多萬元逃亡。<ref name = sinica_keyword_history /><ref name = china.com.cn />

1985年2月13日，國泰信託公司也出現擠兌提款金額達到90億元。<ref name = china.com.cn />

1985年2月18日，臺北十信委託合作金庫代管；蔡辰洲也因利用人頭貸款、串通其部屬或假冒他人名義貸款等違反《[[票據法|票據法]]》之[[刑責|刑責]]被捕。

1985年3月1日，台北地檢處獲立法院同意，以退票171張，金額高達1億2千餘萬元...另外涉嫌侵佔、背信、偽造文書等罪拘禁時任[[立法委員|立法委員]]的蔡辰洲等人。<ref name="茅家琦2003" />

1985年3月3日，財政部與中央銀行宣布重整[[國泰信託投資公司|國泰信託投資公司]]，指示[[交通銀行|交通銀行]]、[[中國農民銀行|中國農民銀行]]與[[中央信託局|中央信託局]]組成聯合銀行團進駐國泰信託，銀行團清理債務時發現國泰信託對蔡辰洲的國泰塑膠資金融通達30億元；貸款總額達169億元，三分之二金額的貸款對象皆為國泰信託集團的關係企業。銀行團要求蔡辰男以私人財產抵押，蔡辰男營運最好的[[台北來來大飯店|台北來來大飯店]]以15億元賣給[[國產汽車股份有限公司|國產汽車]]老闆[[張秀政|張秀政]]；在國泰信託集團相關企業出售或關閉後，由[[三陽工業|三陽工業]]集團[[黃世惠|黃世惠]]接管國泰信託，國泰信託經營情況好轉。<ref name = sinica_keyword_history /><ref name = china.com.cn />

1985年3月11日，經濟部長、前財政部長[[徐立德|徐立德]]自承對十信案有道德責任請辭，8月財政部長[[陸潤康|陸潤康]]自請處分請辭，[[中國國民黨秘書長|中國國民黨秘書長]][[蔣彥士|蔣彥士]]因在選舉中主張提名蔡辰洲請辭。<ref name="茅家琦2003" />

1985年3月16日，交通部官員指出，十信、國信事件發生以來，市場游資造成郵政儲金存款增加120億元。<ref name = sinica_keyword_history />

1985年4月4日，台北地方法院依違反《票據法》判處蔡辰洲有期徒刑15年。<ref name = sinica_keyword_history />

1985年12月，台北地方法院判處蔡辰洲有期徒刑12年、褫奪公權6年，其他53名罪犯被判處1年以上至6年有期徒刑，其中30人緩刑。<ref name="茅家琦2003" />蔡辰洲在被司法判決累計670年有期徒刑，確定交監執行刑期達125年。<ref name = china.com.cn /><ref>{{cite news | title = 揭秘「十信案」》給蔣經國最後一擊 迎來台灣新時代！ | url = https://www.cmmedia.com.tw/home/articles/8636 | publisher = 信傳媒 | language = zh-tw | date = 2018-02-26}}</ref>

1986年12月5日，台北第十信用合作社結束營業，12月8日起改為隸屬[[合作金庫|合作金庫]]。<ref name = sinica_keyword_history />

1987年5月13日，[[法務部|法務部]]准許蔡辰洲因[[肝癌|肝癌]]出具保證金[[保外就醫|保外就醫]]，並限制其出境。<ref name = sinica_keyword_history />

1987年5月14日，蔡辰洲在保外就醫期間過世，十信案遺留下5,000多名受害人與150億元債務。<ref name = sinica_keyword_history /><ref name = china.com.cn />

1987年，[[蕭瑞徵|蕭瑞徵]]因錢財糾紛被人[[槍擊|槍擊]][[暗殺|暗殺]]，是臺灣唯一一件立委遭槍擊致死案件。

1988年1月4日，台北地方法院判決合作金庫要求十信理監事賠償案敗訴。<ref name = sinica_keyword_history />

1992年4月3日，台北地方法院判決十信違規放款案7名高級職員須賠償合作金庫1億餘元。<ref name = sinica_keyword_history />

==後續影響==
十信案的爆發嚴重傷害臺灣投資人的信心，臺北十信各分社都受到嚴重的[[擠兌|擠兌]]，數千存款戶一生的積蓄血本無歸。十信案受害者達十萬人以上，六十多家企業面臨破產，民眾對銀行不信任，也對國民黨政府公權力感到失望，十信案對政府權威造成打擊。<ref name="茅家琦2003" />

1985年9月2日，[[兆豐國際商業銀行|交通銀行]]常務董事[[余井塘|余井塘]]在常務董事會上痛斥十信案，激動地講到：「一百億元！你們從前聽過一百億元的弊案嗎？」言畢，余井塘[[腦溢血|腦溢血]]，送醫五日後亡故<ref>{{Cite web|url=http://www.southnews.com.tw/polit/polit_00/polit_05/00847.htm|author=盧世祥|language=zh-tw|title=國民黨重啟造神運動愚行|publisher=南方快報|date=2003-08-12|accessdate=2014-09-21}}</ref><ref>{{Cite web|url=http://old.ltn.com.tw/2003/new/aug/13/today-o1.htm |author=盧世祥|language=zh-tw|title=重啟造神運動？|publisher=[[自由時報|自由時報]]|date=2003-08-13|accessdate=2014-09-21}}</ref>。

十信案也導致國泰集團承受污名。蔡辰洲最後在判刑前夕，於1987年5月14日病故於[[國泰醫院|國泰醫院]]。經濟部長[[陸潤康|陸潤康]]、財政部長[[徐立德|徐立德]]、[[中國國民黨秘書長|中國國民黨秘書長]][[蔣彥士|蔣彥士]]、[[中國國民黨|中國國民黨]]臺北市黨部主任委員[[關中_(人物)|關中]]也因此辭職。然而十信案在當時並非特例，同期的[[華僑信託|華僑信託]]、[[國泰信託|國泰信託]]、[[亞洲信託|亞洲信託]]與[[第一信託|第一信託]]等金融機構也是岌岌可危、隨時處於崩壞邊緣，並不一定要拿蔡辰洲出氣，因此蔡家至今對此依舊耿耿於懷。
