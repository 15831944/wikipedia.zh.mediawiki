'''十八仔'''（[[白話字|白話字]]：{{lang|nan|si̍p-pat-á}}）<ref>[http://twblg.dict.edu.tw/holodict_new/result_detail.jsp 臺灣閩南語常用詞辭典]</ref><ref>[http://taiwanlanguage.wordpress.com/2011/12/31/%E5%8D%9A%E8%B3%AD%E2%80%A2%E5%8D%9A%E5%B1%80%EF%BC%88pua%CA%94%E2%8A%A6-kiau%CB%8B%EF%BC%89%E2%94%80%E2%94%80%E8%B3%AD%E5%8D%9A/ 劉建仁，〈博賭·博局（puaʔ⊦-kiauˋ）──賭博〉，《台灣語的語源與理據》]</ref>，又稱'''十八啦'''、'''洗芭樂'''、'''喜發樂、樗蒱'''，是一種在[[台灣|台灣]]常見的擲[[骰子|骰子]][[賭博|賭博]]遊戲。Sepat平埔原住民語是四，四巴投阿是用四粒投阿。使用的器材為4顆六面骰子及一個大湯[[碗|碗]]，遊戲時，玩家間會互相大喊''18啦''、''BG啦''，玩笑似的來唱衰對方、鼓舞自己，玩起來十分熱鬧。

== 道具 ==
*骰子：正方體有六面、每面刻1至6點的凹圓點。
== 歷史 ==

*《[[世說新語|世說新語]]》
**<方正篇>：王子敬數歲時， 嘗看諸門生樗蒲...(後略)。
**<任誕篇>：溫太真位未高時，屢與揚州、淮中估客樗蒱，與輒不競。
*《[[演繁露|演繁露]]》樗蒲經：「古斷木為子，一具凡五子，故名五木。」

== 一般規則 ==
* 參與者有[[莊家|莊家]]一人及[[閒家|閒家]]數人。
* 每局開玩前，閒家先各自[[下注|下注]]於自己面前。接著由莊家先擲骰子，再依序由各閒家逐一擲骰子，以所得點數大小與莊家分勝負。
* 計點方式係先扣除2顆點數相同的骰子，其餘2顆點數合計即為其所得點數。 若有兩組骰子點數相同，則以點數較大者之合計為其所得點數。有3顆骰子相同且另一顆不同，也必須重擲。
** 因此，所得點數最大為12點，稱為「十八」（{{lang|nan|si̍p-pat}}）；最小為3點，稱為「扁膣」(發音類似BG)。
** 若四顆骰子正面均相同，稱為「一色」（[[外省人|外省人]]稱'''豹子'''），不論點數多寡，均以勝計。
** 若所擲骰子正面均不相同，稱為「無面」或「無點」，應重新擲至有點數為止<ref>[http://www5.gametower.com.tw/games/Freeplay/gold/dice/index.aspx 《骰子喜發樂：遊戲介紹》，淘金館官網]</ref>。

== 特殊規則 ==
* 有些賭局規定3顆骰子點數相同且另一顆不同，可以計算。
* 有些賭局規定若閒家擲出「十八」或「一色」（豹子）時，莊家得加倍賠注。
* 有些賭局規定若莊家擲出「十八」或「一色」（豹子）時，則立即「[[通吃|通吃]]」閒家所下之注，閒家不應再擲。若莊家擲出「扁膣」，則立即「通賠」給閒家所下之注，閒家亦不應再擲。<ref>[http://www5.gametower.com.tw/games/Freeplay/gold/dice/instruction/index.aspx 《骰子喜發樂：莊家模式》，淘金館官網]</ref>
