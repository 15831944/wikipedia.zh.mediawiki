《'''十面埋伏'''》（{{lang-en|'''House of Flying Daggers'''}}）是一部2004年[[张艺谋|张艺谋]]导演的[[中国電影|中国電影]]，[[程小东|程小东]]担任动作指导，由[[刘德华|刘德华]]、[[金城武|金城武]]和[[章子怡|章子怡]]主演。该片是继2002年的《[[英雄_(电影)|英雄]]》之後，[[张艺谋|张艺谋]]执导的第二部武俠片。

== 情节 ==
故事講述[[唐代|唐代]][[大中_(唐朝)|大中]]十三年，[[朝政|朝政]]腐敗，民間出現不少反官府的組織，以飛刀門的勢力最大，朝廷深以為患，飛刀門幫主柳雲飛在一場戰鬥中犧牲，但在新任幫主領導之下，飛刀門的勢力更大。於是[[奉天|奉天]]縣兩大捕頭劉捕頭和金捕頭奉命於十日之內，將[[黑社會|黑社會]]組織「飛刀門」新任幫主緝拿歸案。劉捕頭在[[牡丹|牡丹]]坊擒下飛刀門前幫主柳雲飛之女舞妓小妹，並設下圈套，由金捕頭偽裝成隨風大俠，乘暗夜劫獄，救出小妹，成為小妹身邊的[[臥底|臥底]]，希望能查出飛刀門的總部，以便一舉剿滅。逃亡途中，隨風對小妹呵護有加，因而彼此動了真情；而此案事實上是案外有案、局中有局，除金捕頭之外，包括小妹和劉捕頭在內的很多角色都有著雙重身份。本片將“臥底”這個身份發揮到極致。

== 创作 ==
2003年9月11日开拍<ref name="艰苦">{{cite web|url=http://www.southcn.com/ent/movie/200309230505.htm | language= | title=《十面埋伏》乌克兰拍摄条件艰苦 |publisher=南方网 |date=2003-09-23 |accessdate= }}</ref>。主要拍摄地包括[[乌克兰|乌克兰]][[利沃夫|利沃夫]]<ref name= "艰苦"/>与克索沃国家森林公园[http://blog.renren.com/share/249763677/777849046]、北京和[[重庆|重庆]][[永川|永川]]。精彩的竹林武打场面是在永川的著名景点——[[茶山竹海|茶山竹海]]拍的<ref>{{cite web|url=http://ent.163.com/ent_2003/editor/news/starnews/040108/040108_213268.html | language= | title=《十面埋伏》刘德华章子怡竹林开"打" |publisher=华西都市报 |date=2004年1月8日 |accessdate= }}</ref>，而片中最后刘金两人的打斗戏份是在[[乌克兰|乌克兰]]拍摄的<ref>{{cite web|url=http://news.gd.sina.com.cn/ent/2004-07-09/546827.html | language= | title=为拍《十面埋伏》 张艺谋在乌克兰种下一片花海(图) |publisher=羊城晚报 |date=2004年7月9日 |accessdate= }}</ref>。

2004年1月14日于[[北京电影制片厂|北京电影制片厂]]正式关机。后期工作在[[澳大利亚|澳大利亚]]制作完成<ref>{{cite web|url=http://www.xiancn.com/gb/wbpaper/2004-01/16/content_110620.htm | language= | title=《十面埋伏》前日关机(图) |publisher=西安新闻网 |date= 2004-01-16 |accessdate= }}</ref><ref>{{cite web |url=http://e.jmrb.com/m/2004/01/16/09/m_14299.shtml |language= |title=《十面埋伏》关机庆功 |publisher=江门日报 |date=2004-01-16 |accessdate= }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

=== 梅艷芳辭演 ===
[[梅艷芳|梅艷芳]]生前曾獲張藝謀邀請參演《十面埋伏》，後來因為患子宮頸癌病重，加上朋友和醫生極力勸喻，最終辭演。为了表达剧组的惋惜和悼念梅艳芳，导演张艺谋最终决定临时修改剧本，将大姐一角永远留给梅艳芳，而电影片尾序幕亦打出“谨以此片缅怀梅艳芳女士”的字幕。

== 评价 ==
2004年5月亮相[[戛纳国际电影节|戛纳国际电影节]]<ref name="festival-cannes.com">{{cite web |url=http://www.festival-cannes.com/en/archives/ficheFilm/id/4200080/year/2004.html |title=Festival de Cannes: House of Flying Daggers |accessdate=2009-12-05|work=festival-cannes.com}}</ref>获得了热烈的反响。<ref>{{cite web|url=http://www.thecanadianencyclopedia.com/index.cfm?PgNm=TCE&Params=M1ARTM0012614 |title=Cannes Film Festival 2004 |publisher=''[[Maclean's_Magazine|Maclean's Magazine]]'' |accessdate=2008-07-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20110608090618/http://www.thecanadianencyclopedia.com/index.cfm?PgNm=TCE&Params=M1ARTM0012614 |archivedate=2011-06-08 }}</ref><ref>{{cite web |url = http://www.guardian.co.uk/film/2004/may/20/cannes2004.cannesfilmfestival1 |title = Cannes 2004: Review roundup |publisher = ''[[The_Guardian|The Guardian]]'' |author = [[Peter_Bradshaw|Peter Bradshaw]] |date = 2004-05-20 |accessdate = 2008-07-28}}</ref>观众看完后起立鼓掌达20分钟之久。<ref>{{cite web|url=http://archive.japantoday.com/news/jp/e/tools/print.asp?content=newsmaker&id=189 |title=Daggers of the mind |publisher=''[[Japan_Today|Japan Today]]'' |date=2004-07-29 |accessdate=2008-07-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20081220114548/http://archive.japantoday.com/news/jp/e/tools/print.asp?content=newsmaker&id=189 |archivedate=2008-12-20 }}</ref>

在国内，一般影評家對此片的評價毁誉参半。不过该片在欧美等国上映后大多评价甚高，跟当年对《英雄》的评价情况相类似：在中国争议严重，在西方则普遍获得赞美之声，海外票房也很可观。电影获得了美国影评人的广泛称赞。<ref name="meta">{{cite web |url = http://www.metacritic.com/film/titles/houseofflyingdaggers |title = House of Flying Daggers |publisher = [[Metacritic|Metacritic]] |accessdate = 2008-07-28}}</ref> 在电影评论网站[[Metacritic|Metacritic]]上，根据发表的37个评论，它收获了平均89分的高分。<ref name="meta" />在[[烂番茄|烂番茄]]上，160个专业评论中有88%的评论给予了积极正面的评价<ref>{{cite web |url = http://www.rottentomatoes.com/m/house_of_flying_daggers/ |title = House of Flying Daggers |publisher = [[Rotten_Tomatoes|Rotten Tomatoes]] |accessdate = 2008-07-28}}</ref>；[[Metacritic|Metacritic]]把它列为2004年度好评排行第六的电影<ref>{{cite web |url = http://www.metacritic.com/film/awards/ |title = Film Awards and Top Tens by Year |publisher = [[Metacritic|Metacritic]] |accessdate = 2008-07-28}}</ref>。

2005年美国《[[时代周刊|时代周刊]]》评选2004年全球十佳电影，将《英雄》和《十面埋伏》并列为第一名<ref>[http://news.bandao.cn/newsdetail.asp?id=79420 美国《时代》杂志评2004年十大最佳电影 《十面埋伏》与《英雄》并列第一]</ref>。

== 票房 ==
中国大陆票房为1.53亿人民币，2004年度票房排名第二，华语片排名第一。香港票房1500多万。

2004年12月3日在北美15家影院上映，首周获得US$397,472 (平均每厅收获$26,498)。总共在北美收获$11,050,094，是2004年北美票房第三高的外语片；北美以外地区收获US$81,751,003，因此全球票房为$92,801,097美元。
