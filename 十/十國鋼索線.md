'''十國鋼索線'''（{{jpn|j=十国鋼索線|hg=じっこくこうさくせん|rm=Jikkokukōsaku sen|f=y}}），通稱'''箱根十國峠纜索鐵路'''（{{jpn|j=箱根十国峠ケーブルカー|link=no}}），是位於[[日本|日本]][[靜岡縣|靜岡縣]][[函南町|函南町]]的[[缆索铁路|缆索铁路]]，由[[伊豆箱根鐵道|伊豆箱根鐵道]]負責營運<ref name="運賃">{{Cite news|url=http://www.izuhakone.co.jp/jukkoku-cable/fea/index.html/|script-title=ja:箱根 十国峠ケーブルカーについて（運賃・営業時間など）- 箱根 十国峠ケーブルカー|newspaper=箱根 十国峠ケーブルカー|accessdate=2018-11-07|language=ja|title=存档副本|archive-url=https://web.archive.org/web/20171231142746/http://www.izuhakone.co.jp/jukkoku-cable/fea/index.html/#|archive-date=2017-12-31|dead-url=no}}</ref>。

== 簡介 ==
本路線連接了位於[[箱根|箱根]]及[[伊豆|伊豆]]要道{{link-ja|熱海箱根峠線|静岡県道20号熱海箱根峠線}}的十國登口車站及位於日金山山頂的十國峠車站。十國峠車站位於函南町與熱海市交界，登上標高766公尺的日金山山頂「十國峠」可眺望「十國五島」（十國為[[伊豆國|伊豆]]、[[駿河國|駿河]]、[[遠江國|遠江]]、[[甲斐國|甲斐]]、[[信濃國|信濃]]、[[武藏國|武藏]]、[[上總國|上總]]、[[下總國|下總]]、[[安房國|安房]]及[[相模國|相模]]，五島則為[[伊豆大島|大島]]、{{link-ja|利島 (伊豆群島)|利島|利島}}、[[新島|新島]]、{{link-ja|神津島}}及[[三宅島|三宅島]]）<ref>{{Cite news|url=http://www.izuhakone.co.jp/jukkoku-cable/monument/index.html/|script-title=ja:十国峠山頂のご案内 - 箱根 十国峠ケーブルカー|newspaper=箱根 十国峠ケーブルカー|accessdate=2018-11-07|language=ja|title=存档副本|archive-url=https://web.archive.org/web/20181106060942/http://www.izuhakone.co.jp/jukkoku-cable/monument/index.html/#|archive-date=2018-11-06|dead-url=no}}</ref><ref>{{Cite news|url=https://www.ataminews.gr.jp/s/spot/120/|script-title=ja:十国峠|newspaper=あたみニュース - 熱海市観光協会 公式観光サイト|accessdate=2018-11-07|language=ja|title=存档副本|archive-url=https://web.archive.org/web/20160127152312/http://www.ataminews.gr.jp/s/spot/120/#|archive-date=2016-01-27|dead-url=no}}</ref>。如果天氣狀況佳，還有機會看見[[東京鐵塔|東京鐵塔]]和[[東京晴空塔|東京晴空塔]]<ref>{{Cite web|url=http://www.at-s.com/news/detail/100085898.html|script-title=ja:スカイツリー、十国峠から見えた！函南|accessdate=2018-11-07|author=|date=|publisher=[[靜岡新聞|靜岡新聞]]|dead-url=yes|title=存档副本|archive-url=https://web.archive.org/web/20111220053322/http://www.at-s.com/news/detail/100085898.html#|archive-date=2011-12-20}}</ref>。

十國峠站南側草地視野廣闊。其前方有通往山腰{{link-ja|姫之澤公園|姫の沢公園}}的步道，步道旁設有許多戶外運動設施，吸引許多親子登山客來訪。

除了纜車，也可駕車從熱海峠{{link-ja|伊豆Skyline|伊豆スカイライン}}{{link-ja|熱海峠交流道|熱海峠インターチェンジ}}前往十國峠站。上述草地旁設有停車場。

== 路線資料 ==
* 路線距離（{{link-ja|營業距離|営業キロ}}）：0.3公里（316公尺）<ref name="運賃"/>。
* [[軌距|軌距]]：1,435[[公釐|公釐]]（[[標準鐵軌|標準鐵軌]]）<ref name="運賃"/>。
* 站數：2個（即起、終點車站）<ref name="運賃"/>。
* 高低差：101公尺<ref name="運賃"/>。
包括同集團的{{link-ja|駒岳鋼索線|伊豆箱根鉄道駒ヶ岳鋼索線}}（2005年廢線）在內的日本大多數[[缆索铁路|缆索铁路]]採用1,067公釐轨距，然而本線在製作車輛時借用了當時日本國內唯一採用標準軌、現已廢線的{{link-ja|妙見森林鋼索線|能勢電鉄妙見の森ケーブル}}列車，因此成為日本少數使用標準軌的纜索鐵路<ref name="情報局">{{Cite news|url=http://shizuoka-hamamatsu-izu.com/izu/atami-city/sz543/|script-title=箱根十国峠ケーブルカー - 静岡・浜松・伊豆情報局|date=2018-08-23|newspaper=静岡・浜松・伊豆情報局|accessdate=2018-11-07|language=ja}}</ref>。

== 使用車輛 ==
本路線共有二部列車承擔路線運作，名稱分別為「日金」及「十國」，並以伊豆箱根鐵道母公司{{link-ja|西武集團|西武グループ}}旗下持有球隊[[西武獅|西武獅]]的代表色「{{link-ja|獅子色|ライオンズカラー}}」塗裝<ref name="運賃"/><ref name="情報局"/>。

== 運行形態 ==
本路線現行時間表為8:50至16:30分，每10至15分鐘開出一班列車，自起點至終點花費時間約3分鐘<ref name="運賃"/>。

== 交通接駁 ==
* 十國登山口站：[[熱海站|熱海站]]搭乘往{{link-ja|元箱根|元箱根}}的{{link-ja|伊豆箱根巴士|伊豆箱根バス}}。行駛時間約35分。
