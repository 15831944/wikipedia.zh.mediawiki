'''十字路車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]，為[[行政院農業委員會林務局|林務局]][[阿里山森林鐵路|阿里山森林鐵路]][[阿里山線|阿里山線]]之[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*車站左邊是辦公室，右邊是候車室。

== 利用狀況 ==
*阿里山線(主線)的停靠站。
*因受[[八八風災|八八風災]]影響，曾於2008.8.8~2017.9.1期間停駛。
*2017年7月5日起，每週三停靠郵輪式列車。
*2017年9月1日起,正班車開到本站,本站暫時成為林鐵本線上山終點.

== 車站週邊 ==
