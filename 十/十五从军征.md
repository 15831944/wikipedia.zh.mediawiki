'''《十五从军征》'''，是一篇出自《[[乐府诗集|乐府诗集]]》的[[乐府|乐府]]诗词<ref>{{Cite web |url=http://www.thn21.com/wen/show/36435.html|title=汉乐府《十五从军征》原文、注释、翻译、在线朗读与赏析|language=zh-cn||quote=十五从军征 （汉）《乐府诗集》}} </ref>，在《乐府诗集》中属《横吹曲辞·梁鼓角横吹曲》，又名《紫骝马歌》。

== 解题 ==
本篇描写一个老战士回乡后无家可归的悲惨情景，揭露了當時兵役制度给劳动人民造成的苦难<ref>{{Cite web|url=http://www.shicimingju.com/baidu/list/16452.html|title=《十五从军征》作品赏析|language=zh-cn|4=|quote=这首诗描绘了一个家破人亡的老兵形象，控诉了汉代兵役制给人民带来的　深重苦难。|access-date=2014-12-25|archive-url=https://web.archive.org/web/20141225060321/http://www.shicimingju.com/baidu/list/16452.html|archive-date=2014-12-25|dead-url=yes}}</ref>。

== 本文 ==
十五从军征，八十始得归。<br />

道逢乡里人：家中有阿谁？<br />

遥看是君家，松柏冢累累。<br />

兔从狗窦入，雉从梁上飞。<br />

中庭生旅谷，井上生旅葵。<br />

舂谷持作饭，采葵持作羹。<br />

羹饭一时熟，不知贻阿谁！<br />

出门东向看，泪落沾我衣。<ref>{{Cite web|url=http://www.shicimingju.com/baidu/list/16452.html|title=《十五从军征》|language=zh-cn|4=|quote=十五从军征，八十始得归。道逢乡里人：家中有阿谁？遥看是君家，松柏冢累累。兔从狗窦入，雉从梁上飞。中庭生旅谷，井上生旅葵。舂谷持作饭，采葵持作羹。羹饭一时熟，不知送阿谁！出门东向看，泪落沾我衣。|access-date=2014-12-25|archive-url=https://web.archive.org/web/20141225060321/http://www.shicimingju.com/baidu/list/16452.html|archive-date=2014-12-25|dead-url=yes}}</ref>
