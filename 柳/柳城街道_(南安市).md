'''柳城街道'''是位于[[中华人民共和国|中国]][[福建省|福建省]][[泉州市|泉州市]][[南安市|南安市]]的一个[[街道办事处|街道]]。街道辖区位于南安市区东南大门，是南安市区出入[[厦门市|厦门]]、泉州市區的重要门户。

== 名称由来 ==
因南安县治从[[丰州镇|丰州]]迁移至此，使用了丰州的别称“柳城”<ref> {{cite web |url=http://www.nanan.com/cg/news/6/2002124153036.htm |title=你知道“溪美”地名的由来吗？ |accessdate=2008-04-15 |date=2002-12-04 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20050313100946/http://www.nanan.com/cg/news/6/2002124153036.htm |archivedate=2005-03-13 }}</ref>。柳城街道1999年从原溪美镇拆出时继续沿用了这一名称。

== 概况 ==
柳城街道共有人口6.38万人，总面积71.9平方公里，市区面积6平方公里，森林覆盖率达60%。

主要产业为[[建筑|建筑]][[建材|建材]]、机械[[铸造|铸造]]、塑料[[鞋革|鞋革]]、[[房地产开发|房地产开发]]、城区[[服务业|服务业]]等<ref> {{cite web |url=http://www.fj.xinhuanet.com/dszx/2005-10/25/content_5424376.htm |title=乡镇概况 |accessdate=2008-04-15 |date=2005-10-25 |publisher=新华网南安在线 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20070811180025/http://www.fj.xinhuanet.com/dszx/2005-10/25/content_5424376.htm |archivedate=2007-08-11 }}</ref>。

== 行政区划 ==
柳城街道办事处驻金街，辖4个[[社区|社区]]、11个[[行政村|行政村]]<ref> {{cite web|url=http://xzqh.org/quhua/35fj/0583na.htm |title=南安市 |accessdate=2008-04-15 |publisher=行政区划网站 |language=zh-hans }}</ref>：

金街社区、新华社区、帽山社区、柳东社区、象山村、露江村、霞西村、霞东村、双林村、杏莲村、三堡村、下都村、祥堂村、上都村、施坪村。
