'''柳雷鸟'''（学名：Lagopus lagopus，Willow ptarmigan），或稱'''柳松雞'''，在[[松鸡|松鸡]]家族中是一种中等体型的鸟，主要分布于[[欧亚大陆|欧亚大陆]]的北部至[[蒙古|蒙古]]、[[乌苏里|乌苏里]]及[[萨哈林岛|萨哈林岛]]，但也見於北美洲地區。它已被列为[[中国国家二级重点保护野生动物名录|中国国家二级重点保护野生动物]]，此外亦是美國[[阿拉斯加州|阿拉斯加州]]的[[美國州鳥列表|州鳥]]。

== 别名 ==
柳雷鸟有许多别名：雷鸟、柳鸡、苏衣尔、雪鸡。

== 特征 ==
在[[夏季|夏季]]的时候，雄鸟身体的羽毛上半部分为黑褐色（上面有不规则的棕黄色横斑），下半部分为白色；翅膀的羽毛为白色；黑褐色的尾巴中央有一对白色[[尾羽|尾羽]]；脚上也覆盖着白色羽毛。雌鸟身体的羽毛上半部分为黑褐色（上面有淡黄色点斑），羽端为白色；下半部分污黄色。

到了[[秋季|秋季]]，雄鸟和雌鸟上体的羽色都变成了棕黄栗色，布满黑色的斑纹，胸、腹部换成了白色的羽毛。

进入[[冬季|冬季]]，雄鸟和雌鸟全身的羽毛都变为白色，仅尾羽和[[飞羽|飞羽]]的羽干为黑色。

柳雷鸟眼睛内的[[虹膜|虹膜]]为褐色；嘴黑色；爪也是黑色。

== 习性 ==
柳雷鸟生活在北极附近的冻原地带、冻原灌丛森林和多岩石的草甸地带，非常耐寒。喜欢在树林中活动，有时也到农田地带。除了繁殖期外大多成群活动，冬季甚至可达百只以上。活动范围广，冬季会进行长途迁移。

== 繁殖 ==
[[File:Lagopède_des_saules_MHNT.jpg|thumb]]
柳雷鸟的繁殖期是5—7月，配对前雄鸟会在自己占据的领地里炫耀表演并发出求偶的鸣叫声，吸引雌鸟前来交尾。

== 食物 ==
柳雷鸟是典型的植食鸟类，很少吃[[昆虫|昆虫]]。主要是各种[[桦树|桦树]]、[[柳树|柳树]]、[[杨树|杨树]]等乔木的嫩芽、嫩枝、嫩叶、花絮、[[种子|种子]]和[[果实|果实]]。冬春季吃植物的芽苞和微枝，夏季吃绿叶，秋季吃草籽和[[浆果|浆果]]。
