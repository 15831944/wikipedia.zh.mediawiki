== 生平 ==
原名杨柳。柳岩衡阳卫校毕业后，在广州珠江医院实习，后在心内科任护士。高中毕业后，考入[[湖南师范大学|湖南师范大学]]，获得[[汉语|汉语言]][[学士|学士]]。<ref>{{cite web |url=http://yule.sohu.com/20111017/n322436005.shtml|title=《柳岩：被评性感是莫大褒奖，望男友是幽默胖子》|accessdate=2011年10月17日|work=|publisher=搜狐网|language=zh-cn}}</ref>

1999年，签约[[广州电视台|广州电视台]]34频道擔任[[节目主持|节目主持]]，又拍摄多輯广告及小品剧集。

2005年，参加由[[安徽电视台|安徽电视台]]及[[光线传媒|光线传媒]]聯合舉的[[猫人超级魅力主持秀|猫人超级魅力主持秀]]，获第七名，随即加入光线传媒至今，与[[谢楠|谢楠]]及[[大左|大左]]被合称为“光线三宝”。<ref>{{cite web|url=http://data.yule.sohu.com/star/data/22/22413/|title=《明星资料库：柳岩》|accessdate=|work=|publisher=搜狐网|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20130525094033/http://data.yule.sohu.com/star/data/22/22413/|archivedate=2013-05-25}}</ref>

TC Candler在2015年中選出柳岩為「全球100張最美面孔」第89名(此為第一次進榜)。

== 个人生活 ==
柳岩自曝情史，当红女主播柳岩录制湖南卫视节目《完美释放》时，因为在节目中无法接受一位倾诉者的婚姻观点而自曝辛酸情史。情绪激动的柳岩透露，10年前自己便有过结婚的念头，但后来被“人渣”男友狠心抛弃，并感叹“不经历人渣，怎么会成为孩子他妈？”<ref>[http://www.zhiyinlady.com/news/2012-07-25/30971.html 柳岩自曝情史 想结婚却遇人渣男友]</ref>

录制湖南卫视节目《非常了得》时，自曝曾疯狂追求黄立行遭拒绝。<ref>[湖南卫视节目《非常了得》2013-12-04]</ref>

“我妈妈是老师对我管教特别严，18岁以前我对男生是没有什么感觉的，后来我谈恋爱了，交往过的男朋友都不错，只碰到过一个人渣。……可后来发生了一些事情，我就被甩了。”<ref>{{cite web |url=http://www.chinanews.com/yl/2012/07-19/4042792.shtml |title=柳岩自曝情史:10年前曾向结婚却遭人渣男友抛弃 |accessdate= 2012年7月19日|publisher= 中国新闻网|work= |language =zh-cn}}</ref>
