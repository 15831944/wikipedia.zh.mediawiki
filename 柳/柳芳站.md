'''柳芳站'''位于[[北京市|北京市]][[朝阳区|朝阳区]]，是[[北京地铁|北京地铁]][[北京地铁13号线|13号线]]的一个车站。车站编号是1315。

== 位置 ==
這個站位於東土城路和柳芳北街交汇处，站址原为[[望和铁路|望和铁路]]（[[双沙铁路|双沙铁路]]从[[望京站_(铁路)|望京]]引出的支线{{NoteTag|1=望和铁路：北起望京站，南至和平里站，全长10.8公里。1969年建成通车。该路为北京铁路枢纽东北环铁路的组成路段，是京包铁路东段的一条单线直线。主要承运北京东北郊的大宗和零担货物，并与终端附近北京造纸一厂的专用铁路线接轨。该路除起止站外无中途站。}}）[[和平里站|和平里站]]。{{NoteTag|1=和平里站：位于香河园街道境内。建于1917年，为望和线终点站，是市区东北部主要货运站。占地面积16万平方米，建筑面积2700平方米。站台呈南北走向。建有基本站台、货物线、货场、仓库、装卸机械等设施。有通往造纸一厂的专用铁路线。由该站北行与[[京包线|京包线]]接轨，折向南经望京站、[[星火站|星火站]]后转向东与[[京承线|京承线]]、[[京秦线|京秦线]]、[[京坨铁路|京坨线]]沟通，转向南经[[百子湾站_(铁路)|百子湾站]]与东南环铁路线、[[丰双铁路|双丰线]]沟通。南行至造纸一厂为专用铁路线终端。货运物资以煤炭、沙石、纸浆原料、纸张及零担散装货物为主。}}

== 出口 ==
這個站有2個出口：A西、B東。

== 命名 ==
规划建设阶段，柳芳站曾称作“和平里北街站”、“煤厂站”。{{NoteTag|1=柳芳的地名原为牛房村，1988年改为现名。}}开通前，这座车站由于靠近柳芳北里居住区，而改称柳芳站。<ref>{{cite news|title=北京轻轨铁路5座站名有所改动 新站借地扬名|date=2002-07-01|accessdate=2012-12-09|url=http://house.sina.com.cn/n/b/2002-07-01/12417.html|author=梁琦|deadurl=yes|archiveurl=https://web.archive.org/web/20150927173007/http://house.sina.com.cn/n/b/2002-07-01/12417.html|archivedate=2015-09-27}}</ref><ref>{{cite news|url=http://www.bj.xinhuanet.com/zhuanti/pandianbeijing-chengtie/chengtie-38.htm|title=东线城铁冷热滑试验顺利完成准备试运行|accessdate=2012-12-09}}</ref>柳芳站前的公交场站在13号线开通后一段时间内仍然使用“和平里火车站”一名。<ref>{{cite news|url=http://www.bjnews.com.cn/news/2013/12/29/299672.html|title=104路快车下岗 车迷送别末班车|publisher=[[新京报|新京报]]|date=2013-12-29|accessdate=2015-09-26}}</ref>[[和平里北街站|和平里北街站]]这一站名后来被用于{{北京地铁线路名|5}}的一个车站。
