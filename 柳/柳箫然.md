'''柳箫然'''（{{Kor|諺=유소연|漢=柳簫然|羅=Ryu So-yeon|f=y}}，{{bd|1990年|6月29日}}），是一名[[大韩民国|韩国]][[职业选手|职业]][[高尔夫球|高尔夫球]]运动员，参加[[LPGA巡回赛|LPGA巡回赛]]和[[韩国LPGA巡回赛|韩国LPGA巡回赛]]<ref>所谓“韩国LPGA巡回赛”，是指韩国的女子职业高尔夫协会单独组织的比赛，即KLPGA巡回赛，而非[[美国|美国]]的[[LPGA|LPGA]]所组织。</ref>。她曾获得[[2006年亚洲运动会|2006年多哈亚运会]][[亚洲运动会高尔夫比赛|高尔夫球]]金牌、2011年[[美国女子公开赛|美国女子公开赛]]冠军。

==职业生涯==
柳箫然在2007年（17岁）时转为职业球手。她在2008年2月美国的[[Cactus巡回赛|Cactus巡回赛]]，以6杆的优势赢得比赛。<ref name="cactustour2008">{{cite web |publisher=Cactus Tour web site |url=http://www.thecactustour.com/archive/results2008.html |title=Cactus Tour 2008 Results |accessdate=17 July 2011 |deadurl=yes |archiveurl=https://web.archive.org/web/20111008044923/http://www.thecactustour.com/archive/results2008.html |archivedate=2011年10月8日 }}</ref> 之后她加入了韩国LPGA，赢下了她参加的第一个赛事[[首尔体育报公开赛|首尔体育报公开赛]]（{{lang-en|Sports Seoul Open}}）。

2009年，她在延长赛中战胜[[徐希炅|徐希炅]]（{{lang-ko|서희경}}），夺得了[[中国女子公开赛|中国女子公开赛]]的冠军。<ref>{{cite web |url=http://sports.sina.com.cn/golf/2013-09-24/15516798700.shtml |title=华彬LPGA赛韩潮汹涌 崔罗莲柳箫然劲刮美女风(图) |accessdate=2013-10-07 |date=2011-09-24 |publisher=新浪体育 |language=zh-cn}}</ref>

2011年7月11日，柳箫然赢得了[[美国女子公开赛|美国女子公开赛]]。在最后一轮的第18洞也是锦标赛中最难的一洞时，她以一杆的劣势落后于领先者徐希炅，但是依靠一杆将球敲到6[[英尺|英尺]]处并成功推进[[小鸟球|小鸟球]]，她追平了徐希炅，得以将比赛拖入三洞的延长赛。最终，她在延长赛中以3杆的优势获胜，成为第五名获得[[美国女子公开赛|美国女子公开赛]]冠军的韩国人。<ref>{{cite web |url=http://v.ku6.com/show/NkiVYXDfDHodzV-8.html |title=正午体育新闻20110712 柳萧然延长赛夺冠 |accessdate=2013-10-06 |date=2011-07-12 |publisher=酷6网 |language=zh-cn }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

柳箫然的第二个LPGA巡回赛冠军是在[[2012年LPGA巡回赛|2012年]]的[[杰米·法尔托莱多精英赛|杰米·法尔托莱多精英赛]]上获得的。在赛季还剩下一场比赛时，柳箫然已然在[[LPGA年度最佳新秀|LPGA年度最佳新秀]]分数排行上拥有足够的领先优势来获得本赛季的这一奖项。<ref>{{cite web |url=http://www.lpga.com/golf/news/2012/10/ryu-clinches-rookie-of-year.aspx |title=Ryu Know It! |date=29 October 2012 |publisher=LPGA |accessdate=12 November 2012}}</ref>

2013年6月23日，柳箫然在[[沃尔玛西北阿肯色锦标赛|沃尔玛西北阿肯色锦标赛]]的一场延长驟死賽上不敌其韩国同胞[[朴仁妃|朴仁妃]]。朴仁妃在第一附加洞打出了小鸟球，力压仅打出[[平标准杆|平标准杆]]成绩的柳箫然，最终获胜。

==个人生活==
柳箫然就读于韩国[[延世大学|延世大学]]，上学期间仍全职比赛于LPGA巡回赛。<ref name="yonsei">Golf Channel, [http://www.golfchannel.com/news/randall-mell/college-and-pro-golf-its-possible/ College and pro golf? It's possible] 27 April 2012. Retrieved 8 April 2013.</ref> 2013年2月，她顺利毕业并获得[[体育管理|体育管理]]学位。<ref name="graduation">Golfcourier,com, [http://www.golfcourier.com/news/story/39125/so-yeon-ryu-graduates/full_story.html So Yeon Ryu Graduates] 25 February 2013. Retrieved 8 April 2013.</ref>
