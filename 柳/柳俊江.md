'''柳俊江'''（{{lang-en|'''Ryan Lau'''}}，{{bd|1981年|4月4日||}}），[[香港|香港]]男性[[电视新闻|电视新闻]][[记者|记者]]、[[媒体|媒体]]人、[[环保|环保]]工作人士、动物保护人士、[[香港电影|香港电影]]工作者，前[[香港|香港]]新聞從業員。2002-2010年任職[[無綫新聞|無綫新聞]]，曾担任港闻、[[中国|中国]]新闻、专题记者，也曾被派驻[[北京|北京]]和[[广州|广州]]，担任驻[[中国内地|中国内地]]记者期間亲身参与采访报道包括：[[世界贸易组织第六次部长级会议|2005年香港世贸会议]]、[[2008年中国雪灾|2008年中国雪灾]]、[[四川大地震|四川大地震]]、[[北京奥运|北京奥运]]、[[2008年金融海啸|2008年金融海啸]]、[[神舟七号|神七升空]]、[[全国两会|全国两会]]等重大新闻。2010年[[除夕|除夕]]辭任[[無綫新聞|無綫新聞]]，轉任[[香港愛護動物協會|香港愛護動物協會]][[中國|中國]]外展總監，網羅內地和香港的精英團隊，協助香港愛護動物協會建立內地辦事處，在中國內地推廣愛護動物的訊息。柳俊江於2013年辭任愛協職位，轉到[[九巴|九巴]]出任[[社區|社區]]事務經理，為九巴於[[區議會|區議會]]等[[團體|團體]]游說及說項。

== 背景 ==
=== 記者工作 ===
柳俊江於2002-2010年曾任[[無綫新聞|無綫新聞部]]港聞組、中國組高級記者，並曾被派駐[[北京|北京]]和[[廣州|廣州]]，及后转任專題記者，負責《[[星期日檔案|星期日檔案]]》等新闻特辑节目的[[編輯|編輯]]和[[採訪|採訪]]工作。

柳俊江担任[[记者|记者]]期间，参与采访报道多宗[[两岸三地|两岸三地]]的突发事件、重大事件，如[[世界贸易组织第六次部长级会议|2005年香港世贸会议]]、2006年的[[台湾|台湾]][[倒扁运动|倒扁运动]]、2007年[[九江大桥垮塌事故|广东九江大桥倒塌]]、2008年[[2008年中国雪灾|南方雪灾]]、[[四川大地震|四川汶川8级大地震]]、[[2008年中国民众抵制家乐福事件|反家乐福示威]]、[[2008年中国奶制品污染事件|毒奶粉风波]]、[[北京奥运|北京奥运]]、[[神舟七号|神七升空]]及[[金融海啸|珠三角工厂倒闭潮]]等等，是经验丰富的香港新闻和中国新闻记者。

其中，2008年[[四川大地震|四川大地震]]采访中，柳俊江和另一隊TVB採訪隊[[潘蔚林|潘蔚林]]在恶劣环境中徒步約兩天，直达震央，成为全球第一位进入震央[[映秀镇|映秀镇]]的國際媒体记者，直接面对灾情的画面报道，其拍攝的畫面為大部分國際媒体採用，包括[[NHK|NHK]]、[[CNN|CNN]]、[[BCC|BCC]]、[[TVBS|TVBS]]等。他也多次参与报道中国[[国家领导人|国家领导人]]的外访活动，如总理[[温家宝|温家宝]]出访[[朝鲜|朝鲜]]等。

=== 擔任主播及主持 ===
柳俊江於[[2008年9月|2008年9月]]開始兼任主播，起初曾任星期六、日的《[[無綫電視午間新聞|午間新聞]]》和《[[無綫電視晚間新聞|晚間新聞]]》主播，2008年9月27日起兼任星期一至五之《[[無綫電視晚間新聞|晚間新聞]]》主播，是近年來在兼任主播後最快主持以上時段的男主播。

2009年2月2日起，無綫《[[無綫電視晚間新聞|晚間新聞]]》改革後，柳俊江亦成為《[[晚間新聞|晚間新聞]]》之替更主播，在[[方健儀|方健儀]]採訪及放假時頂替。2010年4月17日及18日，他頂替正於[[美國|美國]]採訪的[[方東昇|方東昇]]，首次擔任星期六、日《[[六點半新聞報道|六點半新聞報道]]》新聞主播；另外於2009年9月24日起擔任《[[新聞透視|新聞透視]]》主持。與《晚間新聞》剛相反，當他赴外地採訪及放假時，[[方健儀|方健儀]]便頂替他主持此節目。

柳俊江在2011年10月開始擔任[[香港電台電視部|香港電台電視部]]節目《[[議事論事|議事論事]]》主持，直至2017年9月為止，其後由[[黃雅宇|黃雅宇]]接任；2015年4月起與[[黃心美|黃心美]]、[[朱汶欣|朱汶欣]]、[[駱振偉|駱振偉]]、[[劉希信|劉希信]]、[[梁文禮|梁文禮]]、[[麻利亞|麻利]]擔任now TV節目《[[Apostrophe_(電視節目)|Apostrophe]]》主持。

=== 離開新聞界 ===
2010年尾，柳俊江於其博客透露已離開TVB和從事八年的新聞行業，表示近年「記者淪為『車衣女工』，新聞部管理層對每日的新聞早已有一套『看法』，記者很多時被要求『照單執藥』」，文章流露他對新聞編輯自由以及記者的待遇抱有懷疑，因而求退。

2011年，柳俊江接受媒体采访称，自己把毕生精力投放在挚爱的{{le|动物保育|Animal ethics}}和[[环保|环保]]工作中，致力为香港爱护动物协会拓展内地事务，招募香港和内地的精英团队，筹备协会的内地动物保育项目，并把动物保育的理念带入中国大陆，寄望日渐富强的大中华地区可以逐渐重视环境保护、[[生态平衡|生态平衡]]、动物保育等可持续发展的战略<ref>{{cite web |url=http://china.spca.org.hk/tc/zui_xin_xiao_xi/zui_xin_xiao_xi.html |title=存档副本 |accessdate=2011-11-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20130420002925/http://china.spca.org.hk/tc/zui_xin_xiao_xi/zui_xin_xiao_xi.html |archivedate=2013-04-20 }}</ref>，筹建协会在[[广州|广州]]和[[北京|北京]]的办事处<ref>{{cite web |url=http://china.spca.org.hk/tc/zhong_guo_wai_zhan.html |title=存档副本 |accessdate=2011-11-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20111119025318/http://china.spca.org.hk/tc/zhong_guo_wai_zhan.html |archivedate=2011-11-19 }}</ref><ref>{{cite web |url=http://www.spca.org.hk/chi/china/china_donation.asp |title=存档副本 |accessdate=2011-11-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20111209105606/http://www.spca.org.hk/chi/china/china_donation.asp |archivedate=2011-12-09 }}</ref>。

== 趣聞 ==
柳俊江因其趣怪的做事作风，被同行昵称为“柳爷”。由於高大英俊，故被娛樂傳媒形容为「大鼻子情圣」，与[[罗晖翔|罗晖翔]]、[[潘蔚林|潘蔚林]]、[[胡天衡|胡天衡]]等新晉男記者一度成爲少女偶像，并有『新闻王子』之称。
