'''柳南城际铁路'''，也称'''柳南客运专线'''，位于[[广西壮族自治区|广西壮族自治区]]境内，北起工业重镇[[柳州市|柳州市]]，经由[[来宾市|来宾市]]到达自治区首府[[南宁市|南宁市]]，全长226千米。

==简介==
'''柳南城际铁路'''是广西境内两条[[客运专线|客运专线]]中的一条，也是湘桂铁路扩能改造工程的一个子项目。该线路于2007年12月和[[衡柳铁路|衡柳铁路]]同时开工，[[2012|2012]]年6月29日在[[宾阳县|宾阳县]][[黎塘镇|黎塘镇]]开始铺轨，同年12月开通。投入运营后，柳州至南宁区间的铁路实现客货分线，湘桂铁路既有线承担货运职能，而客运职能则主要由城际线承担。

==歷史==
*2007年12月27日：[[湘桂鐵路|湘桂鐵路]]（衡陽至南寧）及此線開始進行擴能改造工程。<ref>{{Cite web |url=http://unn.people.com.cn/BIG5/14771/6711813.html |title=柳南城際鐵路項目正式啟動 南寧桂林2小時互通 |date=2007年12月28日 |author=楊清 |publisher=人民网 |deadurl=yes |archiveurl=https://web.archive.org/web/20131224111550/http://unn.people.com.cn/BIG5/14771/6711813.html |archivedate=2013年12月24日 }}</ref>
*2013年11月18日：湘桂鐵路（衡陽至南寧）与此線對接成功。<ref>{{Cite web |url=http://scitech.people.com.cn/BIG5/n/2013/1122/c1057-23619701.html |title=廣西衡柳高鐵和柳南城際鐵路成功對接 |date=2013年11月22日 |author=石才學 |publisher=人民网}}</ref>
*2013年12月30日：線路通車。
*2018年7月1日：该线将按照最高时速250公里运营<ref>{{Cite web |url=http://dz.xdkb.net/html/2018-05/06/content_486239.htm |title=南京至武汉高铁时速将达250公里 |date=2018-05-06|author=刘伟娟 |publisher=现代快报}}</ref>。

==主要技术指标==
*铁路等级：客运专线；
*正线数目：双线；
*限制坡度：12‰；
*速度目标值：250km/h；
*最小曲线半径：5500m，柳州、南宁枢纽内路段根据限速确定；
*牵引种类：电力；
*机车类型：[[和谐号电力动车组|和谐号电力动车组]]、[[复兴号电力动车组|复兴号动车组]]、其他客运电力机车
*到发线有效长度：650m；
*列车运行方式：自动控制；
*行车指挥方式：综合调度。
