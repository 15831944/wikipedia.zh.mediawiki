'''柳惔'''（{{bd|462年||507年|catIdx=L}}），[[表字|字]]'''文通'''，河东解人。[[南朝齐|南朝齐]][[司空|司空]][[柳世隆|柳世隆]]次子，本人为[[南朝梁|南朝梁]][[官员|官员]]。

== 生平 ==
=== 南朝宋年间 ===
柳惔好学，善作文，尤其懂音律，年轻时与长兄[[柳悦|柳悦]]齐名。<ref name=NS38>[[:s:南史/卷38|《南史》卷三十八]]</ref>

[[南朝宋|南朝宋]][[昇明|昇明]]三年（479年），[[萧赜|萧赜]]为中军大将军，命柳惔为参军，转主簿。<ref name=LS12>[[:s:梁书/卷12|《梁书》卷一十二]]</ref>

=== 南朝齐年间 ===
南朝齐高官[[王俭_(南朝)|王俭]]同时也是辞赋家、目录学家，曾对人称赞柳悦、柳惔兄弟：“柳氏二龙，可谓一日千里。”王俭任[[尚书左仆射|尚书左仆射]]时，曾拜访柳世隆家，柳世隆以为拜访自己，徘徊良久。但王俭上门后，只问柳悦和柳惔，对柳世隆说：“您贤能的儿子都有盛才，一日见顾，今故报礼。若仍相造，似非本意，恐年少窥人。”<ref name=NS38/>

后萧赜为[[齐武帝|齐武帝]]，柳惔曾参加他在烽火楼的设宴，齐武帝认为他的诗写得好，对弟弟豫章王[[萧嶷|萧嶷]]说柳惔不但风韵清爽，而且作文刚健秀美。<ref name=NS38/>

后入为尚书三公郎，累迁太子中舍人，为皇子巴东王[[萧子响|萧子响]]友（幕僚），萧子响为[[荆州|荆州]][[刺史|刺史]]，柳惔随之赴任。萧子响昵近小人，柳惔知道要遭祸，称病回京。后萧子响作乱被杀，柳惔得免。<ref name=NS38/><ref name=LS12/>

历任中书侍郎、中护军长史。累迁新安太守，因无政绩被免。久后，为右军谘议参军事。[[建武_(南朝齐)|建武]]末年，为西戎校尉，[[永元_(南朝齐)|永元]]元年（499年）四月，在宁朔将军任上为梁、南秦二州刺史。<ref>[[:s:南齐书/卷7|《南齐书》卷七]]</ref>二年（500年）[[雍州|雍州]]刺史[[萧衍|萧衍]]起兵，十二月，柳惔举[[汉中|汉中]]响应。<ref name=NS38/><ref>[[:s:资治通鉴/卷143|《资治通鉴》卷一百四十三]]</ref>三年（501年）正月，皇弟南康王[[萧宝融|萧宝融]]为相国，时封梁王的萧衍屡上表劝其登基，柳惔与[[竟陵郡|竟陵]]太守[[曹景宗|曹景宗]]一同劝进。<ref>[[:s:南齐书/卷38|《南齐书》卷三十八]]</ref>二月，萧宝融发檄文到京邑，赞柳惔等人同起义兵。<ref>[[:s:梁书/卷1|《梁书》卷一]]</ref>三月萧宝融登基为[[齐和帝|齐和帝]]后，以时任征虏将军柳惔为益、宁二州刺史，<ref>[[:s:南齐书/卷8|《南齐书》卷八]]</ref>后又用为[[侍中|侍中]]，领前军将军。<ref name=LS12/>

=== 南朝梁年间 ===
萧衍登基为[[梁武帝|梁武帝]]，征柳惔为护军将军，未拜，迁太子詹事，加散骑常侍。先前萧衍任雍州刺史时，曾解茅土玉环赠柳惔。[[天监|天监]]二年（503年）元会（新年宴会），武帝问柳惔所佩玉环是否新亭所赠。柳惔答是的，一直佩带着。武帝趁机劝酒，柳惔没喝完，武帝说：“我经常将卿比作[[刘琨|刘越石]]。”罢会后，封柳惔为'''曲江县侯'''，邑千户。武帝趁宴作诗送给柳惔：“尔实冠群后，惟馀实念功。”又曾说：“[[徐元瑜|徐元瑜]]（前[[广州|广州]]刺史，谋袭广州，被诛）违命岭南，[[周朝|周朝]]的书说父子兄弟罪不相及，朕已放其诸子，何如？”柳惔说：“罚不及嗣，赏延于后，今复见之圣朝。”时人以为他知言。三年（504年）正月迁[[尚书右仆射|尚书右仆射]]。<ref name=NS38/><ref name=LS12/><ref name=LS2>[[:s:梁书/卷2|《梁书》卷二]]</ref><ref>[[:s:南史/卷6|《南史》卷六]]</ref><ref>[[:s:资治通鉴/卷145|《资治通鉴》卷一百四十五]]</ref>

天监四年（505年）十月，武帝大举北伐，以弟中军大将军临川王[[萧宏|萧宏]]都督众军，以柳惔为副。梁军起初于五年（506年）五月攻克梁城，但七月，萧宏得知魏援军在附近，畏懦不敢进军，商议班师。左卫将军[[吕僧珍|吕僧珍]]建议知难而退，萧宏同意，柳惔不同意：“自我大军所临，何城不服，何谓难乎？”众将也都不同意退军。<ref>[[:s:南史/卷51|《南史》卷五十一]]</ref>九月，梁军败，萧宏、柳惔、[[徐州|徐州]]刺史[[昌义之|昌义之]]等屯梁城，[[北魏|北魏]]中山王[[元英|元英]]又大破梁军于淮南，萧宏、柳惔、昌义之等大将五人弃梁城沿淮东逃，米三十万石为魏所获。元英追杀梁军到马头，<ref>[[:s:魏书/卷19下|《魏书》卷一十九下]]</ref><ref>[[:s:资治通鉴/卷146|《资治通鉴》卷一百四十六]]</ref>擒梁将四十二人，斩获五万有余。<ref>[[:s:魏书/卷8|《魏书》卷八]]</ref><ref>[[:s:魏书/卷19下|《魏书》卷一十九下]]</ref><ref>[[:s:北史/卷18|《北史》卷一十八]]</ref>北伐失败，柳惔复为仆射，因久病转[[金紫光禄大夫|金紫光禄大夫]]，加散骑常侍，给亲信二十人。未拜，六年（507年）六月出为使持节、安南将军、[[湘州|湘州]]刺史。<ref name=LS2/>十月，卒于州。武帝为他素服举哀，赠侍中、抚军将军，给鼓吹一部。谥'''穆'''。<ref name=NS38/><ref name=LS12/>

== 性格特点 ==
柳惔度量宽博，家人未尝见其喜怒。很敬重妻子，以至于畏惮她，虽然本性爱音乐，却不敢看漂亮的女伎。仆射[[张稷_(南朝)|张稷]]与柳惔关系亲密，得柳惔妻敬重。张稷每拜见柳惔，必先相问夫人。柳惔每欲见女妓，都通过张稷请奏妻子。其妻隔幔坐，妓女才出来，柳惔才能看到。<ref name=NS38/>

== 作品 ==
柳惔著《仁政传》及《诸诗赋》，粗有辞义。<ref name=NS38/><ref name=LS12/>有文集《柳惔集》二十卷。<ref>[[:s:隋书/卷35|《隋书》卷三十五]]</ref>

== 评价 ==
*《梁书》：“昔[[窦融|窦融]]以河右归汉，终为盛族；柳惔举南郑响从，而家声弗霣（同“殒”），时哉！”即认为柳惔举汉中起兵响应萧衍，因而光大家门，可与两汉之际归顺中央的窦融相比较。
*《南史》：“世隆文武器业，殆人望也，诸子门素所传，俱云克构。”指柳世隆诸子能传承先辈门风，完成前辈事业。
