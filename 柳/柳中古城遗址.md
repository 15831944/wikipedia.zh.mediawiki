'''柳中古城遗址'''是一座位于[[新疆维吾尔自治区|新疆维吾尔自治区]][[吐鲁番市|吐鲁番市]][[鄯善县|鄯善县]][[鲁克沁镇|鲁克沁镇]]始建上限在[[唐|唐]]或更早下限在[[宋|宋]]以后的古城<ref name="文物局">{{cite book |editor=新疆维吾尔自治区文物局 |title=新疆维吾尔自治区第三次全国文物普查成果集成·吐鲁番地区卷 |date=2011年11月 |publisher=科学出版社 |isbn=978-7-03-032615-7 |page=32 |edition=第一版 |url=http://book.sciencereading.cn/shop/book/Booksimple/onlineRead.do# |accessdate=2019-04-16}}</ref>。

== 地理信息 ==
柳中古城东距鄯善县城约43公里，西距吐鲁番市高昌区约70公里，隶属[[鄯善县|鄯善县]][[鲁克沁镇|鲁克沁镇]]的[[库尼夏村|库尼夏村]]。城址三面环山，属平原地形，北高南低，西宽东窄<ref name="初探">{{cite journal |author1=徐佑成 |author2=朱海生 |title=柳中古城形制初探 |journal=吐鲁番学研究 |date=2015 |issue=01 |pages=93-97 |doi=10.14087/j.cnki.65-1268/k.2015.01.008}}</ref>。
== 遗迹现状 ==
城池当年呈长方形，东西约1000米，南北近400米。减地留墙与夯筑方式是城墙的主要采用形式，从低到顶分三层且界限明显，底部为从地面往下挖土形成1.5-2米墙体，中部为9-10米夯筑墙体，顶部为砖砌形式<ref>{{cite web |author1=佚名 |title=城廓日日柳年年火焰山下杨柳春 不老神话之柳中古城 |url=http://www.naic.org.cn/html/2017/gcgz_0702/3944.html |website=中国民族建筑研究会 |accessdate=2019-04-16}}</ref>。

但有另一说根据对60年代的航拍的观察和估计，古城分为内外城，内城为方形，南北长约700米，东西约为800米，面积约0.6平方千米，内城西面城墙向北延伸，与外城墙相连。外城墙呈方形，南北约为1000米，东西长约1900千米，面积约1.9平方千米，城墙宽约6米<ref name="初探" />。而现在，大量残墙与住宅房屋相互掩盖，共发现城墙14处，保存较好的有6处位于双塔清真寺附近，最长城墙约140米，其它8处保存较差位于居民区及后院内<ref name="初探" />。另一处资料与之相符<ref name="吐鲁番">{{cite web |author1=吐鲁番学研究院 |title=新疆吐鲁番地区柳中古城调查新发现 |url=http://www.turfanological.com/ft/article/article.php?articleid=756 |website=吐鲁番学研究院 |accessdate=2019-04-16}}</ref>。其中墙体破坏原因主要分为三种，一是挖墙种地，二是挖墙盖房，三是改为他用<ref name="吐鲁番" />。
== 出土文物 ==
出土过具有[[唐代|唐代]]风格的陶瓷碎片，高数十厘米的鎏金铜佛像。地表有[[清代|清代]]铜钱<ref>{{cite web |author1=吐鲁番学研究院 |title=柳中城 |url=http://www.turfanological.com/ft/article/article.php?articleid=717 |website=吐鲁番学研究院 |accessdate=2019-04-16}}</ref>。
== 历史 ==
柳中城地处古代西域交通孔道。[[东汉|东汉]]延光中班勇为西域[[长史|长史]]驻此。[[十六国|十六国]][[前凉|前凉]]张骏置田地县，唐[[贞观|贞观]]中灭[[高昌|高昌]]改置柳中县，[[贞元|贞元]]中地入[[吐蕃|吐蕃]]。明称为柳城，也作鲁陈或柳陈。1930年，[[黄文弼|黄文弼]]曾考察该遗址，1988年正式调查并记录<ref>{{cite web |author1=新疆文物局 |title=柳中古城遗址 |url=http://www.silkroads.org.cn/article-13031-1.html |website=丝绸之路世界遗产 |accessdate=2019-04-16}}</ref>。
