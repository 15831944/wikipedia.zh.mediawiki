'''柳石明'''（{{bd|1938年||2017年|8月8日}}）原籍[[山东|山东]]，生于[[北京|北京]]，[[中华人民共和国|中华人民共和国]]男高音歌唱家，国家一级演员<ref name=sohu>{{Cite web|url =http://www.sohu.com/a/163472352_661726|title =缅怀！老一辈歌唱家柳石明逝世 《有一个美丽的传说》等作品广为传唱|author =|date =2017-08-09|publisher =搜狐|language =|accessdate =|quote =}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name=rmw>{{Cite web |url = http://culture.people.com.cn/n/2012/0713/c172318-18510006.html |title = 柳石明从艺50周年系列庆典活动将举行 |author =  |date =  2012-07-13|publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1938年生于北京。幼年身体孱弱，5岁才开口说话。柳石明小时候爱听[[收音机|收音机]]，有一天听收音机播放[[刘宝全|刘宝全]]的《[[大西厢|大西厢]]》，柳石明跟着唱起来。1958年，柳石明同时考上了外语学院与[[中央戏剧学院|中央戏剧学院]]，他选择进入中央戏剧学院表演系学习，从此走上文艺道路。在校期间，开始学习[[话剧|话剧]]表演，同时跟[[姜蝶|姜蝶]]学了四年声乐。1962年，自中央戏剧学院表演系毕业，被分配到[[中央实验歌剧院|中央实验歌剧院]]任歌剧演、独唱演员。后来师从[[中央音乐学院|中央音乐学院]][[姜家祥|姜家祥]]教授<ref name=sohu/><ref name=rmw/>。

1962年，歌唱家[[郭兰英|郭兰英]]要演歌剧《[[白毛女_(歌剧)|白毛女]]》，而郭兰英的原搭档年事已高，乃将柳石明调入[[中国歌剧舞剧院|中国歌剧舞剧院]]，他由此成为新一代杨白劳的扮演者。此后他和郭兰英合作二十余年，先后合作演出《[[小二黑结婚_(歌剧)|小二黑结婚]]》、《白毛女》、《[[窦娥冤_(歌剧)|窦娥冤]]》、《青山长在》、《郭兰英歌剧片断晚会》等大型民族歌剧。郭兰英告别舞台时演出的四个歌剧片断：《小二黑结婚》、《白毛女》、《窦娥冤》、《[[刘胡兰_(歌剧)|刘胡兰]]》，都是由柳石明搭档演出。其间柳石明随中央音乐学院[[王福增|王福增]]教授系统学习西洋美声唱法<ref name=sohu/><ref name=rmw/>。

随着郭兰英退出歌剧舞台，柳石明转型演唱歌曲。1985年，作曲家[[吕远|吕远]]为电视剧《木鱼石的传说》主题歌《一个美丽的传说》谱曲，吕远在谱曲时便想好请柳石明来演唱。柳石明演唱的《一个美丽的传说》随电视剧播出后，受到观众热烈欢迎，由此成为他的成名作。后来柳石明享受国务院[[政府特殊津贴|政府特殊津贴]]，是[[中国音乐家协会|中国音乐家协会]]会员、中国少数民族音乐协会会员、中国声乐家协会会员<ref name=rmw/>。他还是[[中国农工民主党|中国农工民主党]]第十二届中央委员<ref name=shishi/>。

晚年柳石明患[[前列腺癌|前列腺癌]]、[[骨癌|骨癌]]很多年，但他年过八旬依然在举办画展、坚持演出<ref name=beiwan/>。

2017年8月8日，柳石明在[[北京友谊医院|北京友谊医院]]病逝，享年80岁<ref name=beiwan>{{Cite web |url =  http://www.takefoto.cn/viewnews-1234664.html?p=1234664|title =  歌唱家柳石明病逝享年81岁 《一个美丽的传说》红了30年|author =  |date = 2017-08-09 |publisher = 北晚新视觉 |language =  |accessdate =  |quote =  }}</ref><ref name=shishi>{{Cite web |url = http://ent.sina.com.cn/y/yneidi/2017-08-10/doc-ifyixiar9257332.shtml |title =中国歌剧舞剧院沉痛哀悼著名歌唱家柳石明同志  |author =  |date =  2017-08-10|publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。

==作品==
柳石明的演唱明亮甜润，吐字清晰，韵味醇厚。他对[[京剧|京剧]]、[[京韵大鼓|京韵大鼓]]等[[戏曲|戏曲]][[曲艺|曲艺]]、各地民歌都有研究。擅长演唱陕北民歌如《有吃有穿》、《横山上下来些游击队》。他演唱的[[中国东北|东北]]民歌《丢戒指》、[[新疆|新疆]]民歌《[[达坂城的姑娘|达坂城的姑娘]]》、[[四川|四川]]民歌《好久没到这方来》等民歌也都很吸引观众。京韵大鼓宗师[[骆玉笙|骆玉笙]]曾向他传授《重整河山待后生》。[[叙利亚|叙利亚]]民歌《你呀你呀》是他的保留曲目<ref name=rmw/>。

柳石明为电视台、电台、音像公司录制过百余首歌曲及歌剧片段，曾为《木鱼石的传说》、《玉碎宫倾》、《[[爱新觉罗·浩|爱新觉罗·浩]]》等电影、电视剧配唱插曲<ref name=rmw/>。

柳石明在歌剧《窦娥冤》中饰演窦天章，获[[中华人民共和国文化部|中华人民共和国文化部]]颁发的表演奖和优秀奖。在《郭兰英歌剧片断》晚会中的表演，获中华人民共和国文化部颁发的表演一等奖。1991年在《歌剧片断》晚会中的表演，获中华人民共和国文化部颁发的特别奖。柳石明演唱并录制的《一个美丽的传说》获中国影视协会颁发的新时期影视优秀歌曲金奖及[[中华全国青年联合会|中华全国青年联合会]]评选的青年最喜爱的歌曲一等奖。他演唱的《故乡的椰子树》曾被全国十多家广播电台当作每周一歌播放<ref name=rmw/>。

[[中国唱片总公司|中国唱片总公司]]、[[上海|上海]]声像出版社、[[广州|广州]]太平洋唱片公司、广州白天鹅音像公司等音像公司曾为柳石明出版多张专辑与合辑。中国唱片总公司还将柳石明列入“20世纪中华歌坛名人百集”。[[中华人民共和国公安部|中华人民共和国公安部]]华盛音像出版社出版了柳石明主讲的《民族声乐发声法》。[[中央电视台|中央电视台]]、[[中央人民广播电台|中央人民广播电台]]、[[北京人民广播电台|北京人民广播电台]]、[[山东电视台|山东电视台]]、[[天津人民广播电台|天津人民广播电台]]、[[吉林人民广播电台|吉林人民广播电台]]等媒体为柳石明制作过多套专题节目<ref name=rmw/>。

柳石明参加过中央电视台文化部春节晚会、元旦晚会、《放歌九十年》、《歌声飘过六十年》、《激情广场》等晚会。在大型音乐舞蹈史诗《[[中国革命之歌|中国革命之歌]]》中，他和[[李谷一|李谷一]]在《北方号角》一场中合演“观灯”。他还曾到[[美国|美国]]、[[俄罗斯|俄罗斯]]、[[白俄罗斯|白俄罗斯]]、[[日本|日本]]、[[越南|越南]]等国演出<ref name=rmw/>。

柳石明还从事声乐教育工作。男高音歌唱家[[聂建华|聂建华]]、书画家[[程风子|程风子]]等人是他的学生<ref name=rmw/>。
