'''柳建伟'''{{BD|1963年|10月|||catIdx= L柳}}，[[河南省|河南省]][[镇平县|镇平县]]人，毕业于[[中国人民解放军信息工程学院|中国人民解放军信息工程学院]]、[[中国人民解放军艺术学院|中国人民解放军艺术学院]]。柳建伟是[[八一电影制片厂|中国人民解放军八一电影制片厂]]副厂长、[[中国作家协会|中国作家协会]]和[[中国电影家协会|中国电影家协会]]成员、[[河南文学院|河南文学院]]院士。<ref name="sma " >{{cite web |url=http://www.chinawriter.com.cn/fwzj/writer/53.shtml |title=柳建伟 |accessdate= |publisher=中国作家网 |work= |language=zh-hans}}</ref>柳建伟是[[军人|军人]]出身的小说家，正师级，[[大校|大校]]军衔。

==生平==
1963年，柳建伟出生在[[河南省|河南省]][[镇平县|镇平县]]。

1979年9月，柳建伟加入[[中国人民解放军|中国人民解放军]]，在[[中国人民解放军信息工程学院|中国人民解放军信息工程学院]]学习，1983年毕业。毕业后，柳建伟在[[八一电影制片厂|中国人民解放军八一电影制片厂]]工作。<ref name="fish" >{{cite web |url=http://www.zjol.com.cn/05culture/system/2006/05/17/006625105.shtml |title=柳建伟 |accessdate= 2006|publisher=浙江在线 |work= |language=zh-hans}}</ref>

1985年，柳建伟开始发表作品。<ref name="fish" />

1991年，柳建伟到[[中国人民解放军艺术学院|中国人民解放军艺术学院]]学习，1993年毕业。1993年，柳建伟进入[[鲁迅文学院|鲁迅文学院]]学习，1994年毕业。1994年开始，柳建伟进入[[北京师范大学|北京师范大学]]学习，1997年毕业。<ref name="fish" />

1997年，柳建伟到[[成都军区|成都军区]]工作，2004年，柳建伟又转回八一电影制片厂工作。<ref name="fish" />
