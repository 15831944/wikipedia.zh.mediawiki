'''鼠疫场遗址'''，位于[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[海曙区|海曙区]][[中山路_(宁波)|中山东路]]以南，东北到太平巷，南靠开明街的5000余平米区域。<ref>{{cite web |url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=925&type=2 |title=鼠疫场遗址 |date=2005-12-02|publisher=宁波市文物保护管理所|quote=1940年10月22日，日军向位于中山东路以南，东北到太平巷，南靠开明街的范围内空投蜘蛛拌和的小麦跳蚤，面粉和传单，内有鼠疫菌}}</ref>1940年10月27日[[侵华日军|侵华日军]]在这片区域投下大量被污染的[[面粉|面粉]]和麦粒，引起了一次较为严重的[[鼠疫|鼠疫]]，直接造成106人死亡、137间民居因防疫而被焚毁。<ref>{{cite web |url=http://mil.sohu.com/20150624/n415564853.shtml |author=|title=5千平方“鼠疫场”被焚 日军细菌战宁波千人罹难|date=2015-06-24|publisher=搜狐军事|quote=}}</ref>1984年10月，该遗址区域被公布为[[海曙区文物保护单位|海曙区文物保护单位]]。1995年9月3日，鼠疫场遗址纪念碑落成，后于2005年9月3日，迁建纪念碑于遗址区域内。<ref>{{cite web |url=http://news.sina.com.cn/o/2005-09-03/12286853095s.shtml |title=宁波开明街鼠疫细菌战遗址纪念碑上午迁建落成|author=魏萍|date=2005-09-03|publisher= 新浪新闻|quote=}}</ref>

==鼠疫事件始末==
1940年10月27日下午2时许，日军[[731部队|731部队]]飞机飞入宁波开明街一带，投放2公斤鼠疫菌。11月2日，《时事公报》刊登消息称宁波城内发现传染病。同月3日，浙江省卫生处吴昌丰通过抽取血液和[[淋巴结|淋巴腺]]穿刺液化验确认患者俞元德所患为[[鼠疫|鼠疫]]，这是第一例确诊病例。5日，《[[时事公报|时事公报]]》开始刊发防疫专辑。8日，鄞县第二次防疫会议要求有关人员必须注射鼠疫疫苗，后注射范围扩大至全市中小学生，共计23343人次接受注射鼠疫疫苗。28日，鄞县第十九次防疫会议决定将疫区房屋物品一并焚毁，後日晚，疫区内共115户137间民居被焚毁。<ref>{{cite web |url=http://zjnews.zjol.com.cn/system/2015/05/25/020667402.shtml |title=5000多平方“鼠疫场”被焚 日军细菌战宁波1554人罹难 |author=胡昊 |date=2015-05-26|publisher=浙江在线|quote=}}</ref>

==鼠疫场遗址纪念碑==
鼠疫场遗址纪念碑初立于1995年9月3日，因为当时遗址范围内无空地可以利用而设置于遗址外部，上刻有“毋忘国耻，励志图强”字样，2003年时曾有侵华日军“731”细菌战受害者64周年祭活动在此举行。<ref>{{cite web |url=http://news.sohu.com/24/97/news214939724.shtml |author=龚国荣|title=组图：宁波举行侵华日军“731”细菌战受害者64周年祭 |date=2003-10-28 |publisher=搜狐新闻|quote=}}</ref>

2005年9月3日，新鼠疫场遗址纪念碑落成于遗址范围内，为高8.1米的三棱锥形建筑，采用黑色大理石建造，椎体西面刻有“毋忘国耻，励志图强”、“侵华日军宁波鼠疫区遗址”及“宁波各界人士重立抗战胜利60周年纪念日”字样，南面用中、英、日三国文字刻有事件史料，东面嵌有疫区示意图及照片。离新纪念碑底座以北2.4米处地面有一钢制示意图，上有麦粒、面粉等物象征日军投下的物品。鼠疫场遗址纪念碑共计占地100余平方。<ref>{{cite web |url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=925&type=2 |title=鼠疫场遗址|date=2005-12-02 |publisher=宁波市文物保护管理所|quote=}}</ref>
