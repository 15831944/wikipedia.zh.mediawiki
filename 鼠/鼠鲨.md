'''鼠鯊'''（[[學名|學名]]：''Lamna nasus''），又名'''大西洋鯖鯊'''，是[[鼠鯊科|鼠鯊科]]的遠洋大型[[鯊魚|鯊魚]]。鼠鯊被認為是處於易危至[[滅絕|滅絕]]的狀況，[[歐洲聯盟|歐洲聯盟]]建議將鼠鯊列入[[聯合國|聯合國]]的[[瀕臨絕種野生動植物國際貿易公約|瀕臨絕種野生動植物國際貿易公約]]中。<ref>{{cite press release |title=More oceanic sharks added to the IUCN Red List |publisher=[[IUCN|IUCN]] |date=2007-02-22 |url=http://www.iucn.org/en/news/archive/2007/02/22_pr_sharks.htm |format= |language= |accessdate=2007-02-25 |quote= }}</ref>

== 分佈 ==
鼠鯊分佈自[[北美洲|北美洲]]東北部海岸（由[[新澤西州|新澤西州]]至[[格陵蘭|格陵蘭]]）及[[非洲|非洲]]西北部海岸、[[摩洛哥|摩洛哥]]或[[西撒哈拉|西撒哈拉]]及[[地中海|地中海]]，至[[冰島|冰島]]對出的海洋、[[挪威|挪威]]的北岸及[[俄羅斯|俄羅斯]]的西北海岸。在[[南半球|南半球]]，鼠鯊亦環球分佈在[[緯度|緯度]]30°S-60°S的海域。

鼠鯊在[[世界自然保護聯盟瀕危物種紅色名錄|世界自然保護聯盟瀕危物種紅色名錄]]中被列為易危。[[加拿大|加拿大]]亦將鼠鯊列為瀕危動物。近年在[[英國|英國]]海域的過份捕殺亦殺死了大批的鼠鯊。

== 棲息地 ==
鼠鯊主要是遠洋的[[鯊魚|鯊魚]]，但亦可以在[[海岸|海岸]]找到。牠們一般生活在水溫5至10℃的冷水中，但亦有在低至1℃或高至23℃的地方生活。牠們出沒於海面至水深多於700米的地方。

== 解剖及外觀 ==
鼠鯊最特別的地方是在[[背鰭|背鰭]]上有一白汶。這使牠們與[[太平洋鼠鯊|太平洋鼠鯊]]及[[大白鯊|大白鯊]]有所分別。[[尾鰭|尾鰭]]上有兩個龍骨，與太平洋鼠鯊相似。

鼠鯊的堅實及沉重的[[鯊魚|鯊魚]]，背部呈深藍色，[[腹部|腹部]]呈白色，鼻端呈錐形。鼠鯊可以生長至3.7米長，重160至250公斤。

== 行為 ==
鼠鯊是最快速的[[鯊魚|鯊魚]]之一。牠們可以完全躍出水面。

== 飲食 ==
鼠鯊是機會主義者，主要以[[硬骨魚|硬骨魚]]為食，如[[鯖魚|鯖魚]]、[[鯡魚|鯡魚]]、[[帆蜥魚|帆蜥魚]]及[[刀魚|刀魚]]。

== 生殖 ==
鼠鯊是[[卵胎生|卵胎生]]的。[[妊娠期|妊娠期]]約8至9個月。每胎最多6頭幼鯊，一般的都是4頭，有時可能有7頭。幼鯊出生時長約60至80厘米。雌性鼠鯊在約12至13年會達至性成熟，雄性則要7至8年。
