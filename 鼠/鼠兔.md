'''鼠兔科'''（学名 ''Ochotonidae''）是[[兔形目|兔形目]]下的一个[[科|科]]，有两个属，分别是[[意大利鼠兔属|意大利鼠兔属]]（''Prolagus''）和[[鼠兔属|鼠兔属]]（''Ochotona''），然而也有部分书籍将意大利鼠兔属列为[[兔形目|兔形目]]下的一个[[科|科]]。意大利鼠兔属下仅有[[意大利鼠兔|意大利鼠兔]]一种，而意大利鼠兔已灭绝[http://en.wikipedia.org/wiki/Prolagus_sardus 1]。鼠兔属下则有多达30种，统称为'''鼠兔'''，又名'''鸣声鼠'''或'''石兔'''。

== 特征 ==
鼠兔是一种[[耳朵|耳朵]]短而圆、没有尾巴的動物。据说鼠兔是因为外形酷似[[仓鼠|仓鼠]]而得名。鼠兔没有[[犬齿|犬齿]]，[[门齿|门齿]]可不断生长。

前肢五指，后肢四趾。爪子细而弯，适合掘土。毛色因种而异，由灰色、灰褐色至锈褐色，夏毛要比冬毛淡。

== 分布 ==
鼠兔主要分布于[[青藏高原|青藏高原]]附近和[[亚洲|亚洲]]中部的[[高原|高原]]或[[山地|山地]]，也分布于[[亚洲|亚洲]]东北部、[[北美洲|北美洲]]西部和[[欧洲|欧洲]]。

== 习性 ==
鼠兔能挖复杂的洞穴系统居住，而有些[[鸟类|鸟类]]（如[[褐背拟地鸦|褐背拟地鸦]]和[[雪雀|雪雀]]等）会利用鼠兔的洞穴躲避太阳的强烈[[辐射|辐射]]或[[暴风|暴风]]与[[冰雹|冰雹]]，鼠兔则可借助[[鸟|鸟]]类的惊鸣来报警。这种生物现象在[[中国|中国]]古籍被称作“'''鸟鼠同穴'''”。

鼠兔多栖息于地面植被矮小稀疏的地区。鼠兔是草食动物，通常在白天活动。鼠兔有储备食物的习惯，所以当[[冬天|冬天]]来临时，鼠兔不[[冬眠|冬眠]]，而是依赖所储备的食物过冬。
