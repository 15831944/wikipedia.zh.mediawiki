中国古代常用两横“＝”或两点“〻”作为'''重文号'''或'''省代号'''，其最早起源于[[中国|中国]][[商朝|商朝]]<ref>{{Citation| title=書誌情報用語解説| editor=戦国楚簡研究会 | url=http://www.let.osaka-u.ac.jp/chutetsu/sokankenkyukai/wording.html| accessdate=2008-06-13}}</ref><ref>{{Citation| title=Database of Selected Characters from Guodian and Mawangdui Manuscripts — Introduction| first=Matthias| last=Richter| year=2006| url=http://www.aai.uni-hamburg.de/MPC/Introduction.pdf| accessdate=2008-06-13| deadurl=yes| archiveurl=https://web.archive.org/web/20060227092842/http://www.aai.uni-hamburg.de/MPC/Introduction.pdf| archivedate=2006-02-27}}</ref>。如左图所示的[[史颂鼎|史颂鼎]]铭文中末尾的“-{子<sub>=</sub>孫<sub>=</sub>寶用}-”，便使用了“＝”表示重复，意为“子子孫孫寶用”。

现时，很多人在手写时仍会非正式的使用叠字符号“々”来表示重复的汉字，如「謝謝」写作「謝々」。

[[女書|女書]]的疊字符號是{{unichar|16FE1|NUSHU ITERATION MARK|use=script|use2=Nshu}}。

== 日文 ==
'''叠字符号'''（{{jpn|j=踊り字|hg=おどりじ}}）「々」(同の字点／どうのじてん。因為看起來像是片假名的「ノ」加上「マ」，所以也被稱為「ノマ」) 被認為是從「仝」的草書轉化而來。是在[[日语|日语]]中表示[[日本汉字|汉字]]（或[[假名|假名]]）重复的符号。例如“{{lang|ja|人々}}”中的“々”表示与前面的“{{lang|ja|人}}”重复，即“{{lang|ja|人人}}”。对于[[日本汉字|汉字]]、[[平假名|平假名]]、[[片假名|片假名]]这三种不同的[[日语书写系统|书写系统]]，分别有不同的叠字符号与之对应。

=== 汉字叠字符号 ===
日文中使用“々”（U+3005）来表示汉字的重复，如：
* {{lang|ja|時時}} → {{lang|ja|時々}}（{{lang|ja|ときどき}}）
* {{lang|ja|明明白白}} → {{lang|ja|明々白々}}（{{lang|ja|めいめいはくはく}}）
* {{lang|ja|赤裸裸}} → {{lang|ja|赤裸々}}（{{lang|ja|せきらら}}）
* {{lang|ja|複複複線}} → {{lang|ja|複々々線}}（{{lang|ja|ふくふくふくせん}}）
* {{lang|ja|小小小支川}} → {{lang|ja|小々々支川}}（{{lang|ja|しょうしょうしょうしせん}}）

但如果重复的汉字属于不同词汇，原则上不使用叠字符号，如：
* {{lang|ja|会社社長}}（{{lang|ja|かいしゃしゃちょう}}）
* {{lang|ja|民主主義}}（{{lang|ja|みんしゅしゅぎ}}）
不过也有例外，如“{{lang|ja|公演会々場}}”。特别是婚礼和葬礼相关的情形，同样的汉字直接重复的话，会让人联想到再婚或者祸不单行，所以常会见到“{{lang|ja|結婚式々場}}”“{{lang|ja|告別式々場}}”的写法。

而在[[东亚文字排列方向|竖排]]书写时，也会使用“〻”（U+303B）来作为叠字符号，但目前已不太常用。

=== 假名叠字符号 ===
对于[[平假名|平假名]]与[[片假名|片假名]]，则分别使用“ゝ”（U+309D）和“ヽ”（U+30FD）作为叠字符号。另外，还有“ゞ”（U+309E）和“ヾ”（U+30FE）用作有[[浊音符|浊点]]时的叠字符号。这些符号近年已经甚少使用。如：
* {{lang|ja|ここ}} → {{lang|ja|こゝ}} （这里）
* {{lang|ja|バナナ}} → {{lang|ja|バナヽ}} （香蕉）
* {{lang|ja|くっつける}} → {{lang|ja|くつゝける}} （粘合）（[[促音|促音]]写成“つ”的旧写法）

在一些专有名词中，现在仍然使用假名叠字符号。
* {{lang|ja|学問のすすめ}} → {{lang|ja|{{link-ja|劝学篇_(福澤諭吉)|学問のすゝめ|学問のすゝめ}}}}（[[福泽谕吉|福泽谕吉]]、{{link-ja|小幡笃次郎|小幡篤次郎}}合著书籍《劝学篇》）
* {{lang|ja|いすず}} → {{lang|ja|いすゞ}}（[[五十铃|五十铃]]）

另外，在前一假名有浊点时，仍用“ゝ”表示无浊点的假名，用“ゞ”表示有浊点的假名，如：

* {{lang|ja|づつ}} → {{lang|ja|づゝ}}（筒）
* {{lang|ja|ぶぶ漬け}} → {{lang|ja|ぶゞ漬け}}（京都[[茶渍饭|茶渍饭]]）

=== 多字符叠字符号 ===
[[File:IterationMarks600.jpg|thumb]]》中使用的“〱”叠字符号]]
在[[东亚文字排列方向|竖排]]书写时，可以使用类似于平假名{{lang|ja|[[く|く]]}}的符号“〱”（U+3031）、“〲”（U+3032），并拉长至多个字符的高度。在使用文字处理软件时，如果无法显示高度为两个字符的叠字符号，则通常将其分拆为“〳”（U+3033）与“〵”（U+3035）、“〴”（U+3034）与“〵”（U+3035）。如：
{| style="border:none;text-align:center;line-height:100%" lang="ja" xml:lang="ja"
|ま<br />あ<br />ま<br />あ
| → 
|ま<br />あ<br />〳<br />〵
|　　
|ど<br />う<br />し<br />て<br />ど<br />う<br />し<br />て
| → 
|ど<br />う<br />し<br />て<br />〳<br />〵
|　　
|見<br />る<br />見<br />る
| → 
|見<br />る<br />〳<br />〵
|}

如果重复文字因[[连浊|连浊]]而较原先增加了浊点，则使用带浊音的叠字符号：
{| style="border:none;text-align:center;line-height:100%" lang="ja" xml:lang="ja"
|し<br />か<br />じ<br />か</span>
| → 
|し<br />か<br />〴<br />〵
|　　
|離<br />れ<br />離<br />れ
| → 
|離<br />れ<br />〴<br />〵
|}

如果被重复文字以浊音开始，则不带浊音和带浊音的叠字符号都可使用：
{| style="border:none;text-align:center;line-height:100%" lang="ja" xml:lang="ja"
|ボ<br />ヤ<br />ボ<br />ヤ</span>
| → 
|ボ<br />ヤ<br />〳<br />〵
|ボ<br />ヤ<br />〴<br />〵
|　　
|ブ<br />ラ<br />ン<br />ブ<br />ラ<br />ン</span>
| → 
|ブ<br />ラ<br />ン<br />〳<br />〵
|ブ<br />ラ<br />ン<br />〴<br />〵
|}

如果文字重复3次，则连用2次叠字符号；如果文字重复4次，则被叠字符号和被重复文字交替使用：
{| style="border:none;text-align:center;line-height:100%" lang="ja" xml:lang="ja"
|ト<br />ン<br />ト<br />ン<br />ト<br />ン</span>
| → 
|ト<br />ン<br />〳<br />〵<br />〳<br />〵
|　　
|ぐ<br />ん<br />ぐ<br />ん<br />ぐ<br />ん<br />ぐ<br />ん</span>
| → 
|ぐ<br />ん<br />〴<br />〵<br />ぐ<br />ん<br />〴<br />〵
|}

== 英文 ==
{{主条目|同上符号}}
在英语裡，[[同上符号|同上符号]]（〃）代表一个重复的单词，仅用于非正式场合。

== 泰文 ==
在[[泰语|泰语]]，''mai yamok''（[[wikt:ๆ|ๆ]]）代表一个重复的单词。如同汉语，重复的单词表示强调。

== 古埃及语 ==

[[埃及语|古埃及语]]的象形文字，[[圣书体|圣书体]]，使用如下符号来表示重复前字：<hiero>O50:Z4</hiero>   拉丁转写：''sp sn''，直译：“两次”。

== 西夏文 ==
[[西夏文|西夏文]]有时使用形如“㇋”的符号做叠字符号，但不在印刷的文本中出现。Unicode收录其字于{{unichar|16FE0|TANGUT ITERATION MARK|use=script|use2=Tang}}。
