《'''相聚一刻'''》又譯《'''一刻館'''》，是[[高橋留美子|高橋留美子]]的[[漫畫|漫畫]]作品。該作於1980年在[[小學館|小學館]]青年漫画[[雜誌|雜誌]]《[[Big_Comic_Spirits|Big Comic Spirits]]》創刊號上開始連載，後來改編成電視動畫，全96話。

めぞん是法語Maison的音譯，是「屋」或「館」的意思。

《相聚一刻》主要在描述一刻館的房客－大學落榜生'''五代裕作'''，和年輕守寡的管理員音無響子以及其他房客之間的互動。

《相聚一刻》動畫版於86年三月開播後，外界反響一般，批評者多說動畫版太著重《相聚一刻》的輕鬆一面，而忽略對角色感情的深入描寫，演釋手法亦較誇張，有失原作的平實風格。

或許是不如預期，當《相聚一刻》播至第27集，製作班底便來個大變身，劇本交付由《我係小忌廉》而大放異彩的伊藤和典接手，至於人物設計方面則由森山雄治改由曾擔任[[魔法小天使|魔法小天使]] 、[[機動警察|機動警察]]及[[橙路|橙路]]的[[高田明美|高田明美]](這也是後來有人常把《相聚一刻》中的音無響子 與《[[橙路|橙路]]》中的鮎川圓 相較的原因之一) ，這也是[[高田明美|高田明美]]老師 為 [[高橋留美子|高橋留美子]]老師作品動畫化的第二彈，第一彈是[[福星小子|福星小子]]。

除人物樣貌行為有變，最令人驚喜是整套動畫的風格趨向平實，由輕快諧趣轉成著墨主角心路歷程的變化，仔細又多元化的感情描述，從而成功轉型而成為動畫經典之作。

== 登場人物 ==
主要角色的姓氏中都含有[[數字|數字]]：音無、一之瀨、二階堂、三鷹、四谷、五代、六本木、七尾、八神、九條、最後是響子的原姓－千草。

配音員（日本 / 香港TVB / 香港ATV/ 台灣）
* 五代裕作（配音員：[[二又一成|二又一成]] / [[黃健強|黃健強]] / [[翟耀輝|翟耀輝]] / [[魏伯勤|魏伯勤]]）
*:男主角，一刻館五號房房客，個性優柔寡斷，家人在故鄉新潟縣經營大眾食堂。故事開始的時候，他是個準備重考的大學落榜生，後來考上了三流私立大學。在一刻館的幾年間，他不但遇到了理想伴侶，也在大學畢業後找到真正適合自己的職業──幼稚園老師，因此本作可說是他的成長故事。
* 音無響子（[[配音員|配音員]]：[[島本須美|島本須美]] / [[林元春|林元春]] / [[王夢華|王夢華]] / [[李宛鳳|李宛鳳]]）
*:女主角，一刻館的管理員，本姓千草，年輕貌美卻是個寡婦。她不顧父母反對，與亡夫音無惣一郎結婚，展現出剛強個性。對愛情相當認真，對亡夫念念不忘，但在相處的過程中，不知不覺喜歡上五代，還會為他跟其他女性感情要好而吃醋。其「註冊商標」是小雞圍裙與掃把。最後勇敢地接受五代的愛情與他結為連理，並生下一個健康的女嬰──春香。
* 一之瀨花枝 / 一之蘭（配音員：[[青木和代|青木和代]] / [[區瑞華|區瑞華]] / [[龍德瓊|龍德瓊]]）
* 一之瀨先生 / 一之蘭（配音員：[[矢田稔|矢田稔]] / [[張鶯|張鶯]]）
* 一之瀨賢太郎（配音員：[[坂本千夏|坂本千夏]] / [[蔡惠萍|蔡惠萍]]）
*:一之瀨一家是住在一刻館一號房房客。花枝可說是非常典型的歐巴桑，不但唯恐天下不亂，而且喜歡參加別人的聚會，還會大白天喝酒。賢太郎為其子，是個頑皮的小學生，喜捉弄五代。對母親的行為相當反感，卻又無能為力。花枝的丈夫在本作中雖有出場，但存在感近乎零，也沒名字，因此一般都通稱他一之瀨氏（一之瀨先生）。
* 二階堂望（配音員：[[堀川亮|堀川亮]] ※在原作與劇場版中登場，電視版無此角）
*:一刻館二號房房客，大學生，家境殷實，起初搬來一刻館時曾與其他房客格格不入，後來逐漸融入，但在事件上的反應可謂遲鈍。
* 三鷹瞬（配音員：[[神谷明|神谷明]] / [[林國雄|林國雄]] / [[黃志成|黃志成]] / [[官志宏|官志宏]]）
*:響子的網球教練，家境優渥且體能卓越，特色是牙齒經常白得閃閃發光。熱烈追求響子，多次向對方求婚但都被拒絕，是五代的情敵。唯一弱點是怕狗，一直設法克服。後來因誤會而和九條明日菜結婚。
* 三越善三郎 （只在電視動畫版出現）（配音員：[[堀勝之祐|堀勝之祐]] / [[馮志潤|馮志潤]]）
*:一度住在一刻館三號房的房客，在電視動畫裡只出現四集。
* 四谷（配音員：[[千葉繁|千葉繁]] / [[張炳強|張炳強]] / [[梁志達|梁志達]]→[[曾秉輝|曾秉輝]]）
*:一刻館四號房房客，五代的鄰居，也是個謎樣角色，身份背景都是謎。曾用過「四谷菊千代」、「四谷劍之介」及「錢德勒四谷」等奇怪假名。平時多有騷擾五代的舉動，常從牆洞偷窺五代生活。
* 六本木朱美（配音員：[[三田友子|三田友子]] / [[姚天麗|姚天麗]] / [[譚淑英|譚淑英]]）
*:一刻館六號房房客，在附近一家酒店「茶茶丸」當服務生。個性豪放潑辣，時常穿著內衣褲在公寓裡晃蕩，喜愛喝酒，也口無遮攔。
* 七尾梢/須惠/小津江（配音員：[[富永美衣奈|富永美衣奈]] / [[袁淑珍|袁淑珍]]）
*:與五代在同一地方打工，也常邀五代回家吃飯，一直被當作五代的女友，而她似乎也不以為意，因此響子對她頗吃醋。大學畢業後，在銀行裡工作。因誤會而和學長結婚，並定居名古屋。
* 八神吹雪（配音員：[[淵崎有里子|淵崎有-{里}-子]] / [[蔡惠萍|蔡惠萍]]）
*:五代在響子母校裡當實習老師時的學生，家境敷裕，對五代抱有戀慕之心。知道響子和亡夫惣一郎是[[師生戀|師生戀]]便效法她，對五代死纏爛打。
* 九條明日菜（配音員：[[鶴弘美|鶴弘美]] / [[樂蔚|樂蔚]] / [[吳小藝|吳小藝]]）
*:舊[[華族|華族]]千金，個性靦腆。與三鷹相親認識。對他一見鍾情，甚至得了相思病，不過跟三鷹相反的是──她非常喜歡狗。後來和三鷹結婚。
* 惣一郎（犬）（配音員：[[千葉繁|千葉繁]] / [[梅欣|梅欣]]）
*:響子的愛犬，是亡夫惣一郎在路上撿回來雜種狗。因響子在呼喚其夫時，這隻狗也連帶有反應，因此後來取名叫惣一郎。
*:「惣」是「總」的異體字。
* 音無惣一郎（配音員：[[田中秀幸|田中秀幸]] / [[林國雄|林國雄]]）
*:響子的亡夫，也是她的高中老師。他與響子萌生[[師生戀|師生戀]]，在響子父母反對下與響子結婚，但是婚後沒多久就突然過世。由於故事一開始時他已過世，在回憶場面中都沒出現真正長相。響子對他始終念念不忘，因此也成五代最大的情敵。
* 音無老人（配音員：[[槐柳二|槐柳二]] / [[譚炳文|譚炳文]]→[[源家祥|源家祥]]）
*:音無惣一郎之父，一刻館的主人。
* 音無郁子（配音員：[[莊真由美|莊真由美]] / [[黃麗芳|黃麗芳]] / [[李宛鳳|李宛鳳]]）
*:音無惣一郎的姪女，曾聘請五代當家庭老師。
* 坂本（配音員：[[古川登志夫|古川登志夫]] / [[馮志潤|馮志潤]] / [[曾秉輝|曾秉輝]]）
*:五代的大學同學。
* 黑木小夜子（配音員：[[島津冴子|島津冴子]]）
*:五代大學時同年級的社團同學，畢業後在幼稚園工作，曾請五代到幼稚園幫忙，讓五代找到最適合自己走的路。
* 茶茶丸的老闆（配音員：[[若本紀昭|若本紀昭]] / [[譚炳文|譚炳文]]）
*:茶茶丸酒店老闆，據說他的名字就是茶茶丸，後來與朱美結婚。
* 千草律子（配音員：[[松島實|松島實]] / [[雷碧娜|雷碧娜]] / [[徐愛貞|徐愛貞]]）

== 電視動畫 ==
1986年3月26日－1988年3月2日在日本[[富士電視網|富士電視網]]播出通常30分鐘的電視動畫，全96話。動畫由[[Kitty_film|Kitty film]]承包（實質由[[STUDIO_DEEN|STUDIO DEEN]]製作）。還有本作是接《[[福星小子|福星小子]]》播出時段的動畫。

配音陣容除了主角五代與女主角響子的配音員2人之外，大多延續前作《福星小子》的陣容。

《相聚一刻》的[[原聲配樂|原聲配樂]]，亦是令《相聚一刻》動畫版更成功的素之一，前期的 [[杉山卓夫|杉山卓夫]] ，以致後期的 [[川井憲次|川井憲次]] ，編製的配樂都是悅耳動聽。如杉山為一刻館四季，編作的四季曲：「春」、「夏之雲」、「秋」及「冬」，一聽樂曲，便能感受一刻館的季節變化。

== OVA、總集篇 ==
=== 相聚一刻 移動中的季節 ===
OVA總集篇，以四季情由開始至終結將電視版重新輯錄而成，約一小時90分鐘如一套電影版本。

=== 相聚一刻 回憶 ===
以音無響子的聲優島本須美聲音回憶點滴。

=== 相聚一刻 番外篇 一刻島始末記 ===
OVA漫畫單元故事，電視版播映完畢後，再動畫化作品。

== 相聚一刻 完結篇 ==
相聚一刻動畫電影版，1988年2月6日在日本首映，片長65分鐘，故事描述五代與響子結婚的前一夜。
