'''相城站'''，为中国[[安徽省|安徽省]][[合肥市|合肥市]]的一座地铁车站，工程站名'''相城路站'''，位于[[相城路|相城路]]与[[淮海大道|淮海大道]]交叉口。相城站为[[合肥轨道交通3号线|合肥轨道交通3号线]]的首末站。<ref>{{cite web |title="图书馆"易误解 合肥市民为地铁3号线站名“挑刺” |url=http://www.ahwang.cn/hefei/20190125/1848031.shtml |publisher=安徽网 |accessdate=2019-01-25}}</ref>

==车站结构==
相城站位于淮海大道与相城路口西侧，沿淮海大道东西向布置，为地上三层双柱三跨矩形岛式站，共设2个出入口。<ref>{{cite web |author1=凤凰网安徽 |title=合肥地铁3号线沿线站点公布！开通运营时间定在… |url=http://wemedia.ifeng.com/44596356/wemedia.shtml |accessdate=2019-02-04}}</ref>

== 工程建设 ==
