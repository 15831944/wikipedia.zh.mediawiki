'''相關不代表因果'''（{{lang-en|'''correlation does not imply causation'''}}）是[[科學|科學]]和[[統計學|統計學]]經常強調的重要[[觀念|觀念]]，意思是若兩個事物（統計學上會用[[變數#统计学上|變數]]代表）有明顯的[[相關|相關]]時（即當一件事出現，另一件事也出現），不一定表示兩者之間有[[因果關係|因果關係]]。

例如說看到[[死刑|死刑]]判決越多，[[謀殺|謀殺]]犯罪率越高，不代表兩者間有關，尤其不代表死刑導致更多謀殺，有可能是謀殺案件增加導致更多死刑判決，或其他因素同時導致更多謀殺與更多死刑判決。

== 相關證明因果 ==
相對地，'''相關證明因果'''（{{lang-en|correlation proves causation}}）、'''與此故因此'''（{{lang-la|cum hoc ergo propter hoc}}）即是認為當兩變數相關時，必有因果關係，這是一種[[非形式謬誤|非形式謬誤]]。

舉例來說，若看到[[死刑|死刑]]判決變多的時候，也是[[謀殺|謀殺]]犯罪率變高的時候，就認為死刑導致更多謀殺；或者看到對同一種犯罪，處罰受得越重的罪犯，再犯率越高，就認為[[嚴刑峻罰|嚴刑峻罰]]導致更多犯罪；或者看到越常被父母動用[[體罰|體罰]]的孩子，有越多的偏差行為，就認為體罰導致偏差行為；或者看到夏商周三代的末代君王都寵幸美女，就認為美女會導致一個朝代覆亡（像例如[[晉朝|晉朝]][[杜預|杜預]]為《[[左傳|左傳]]》做的注裡提到說「[[夏朝|夏]]以[[妹喜|妹喜]]，[[商朝|殷]]以[[妲己|妲己]]，[[周朝|周]]以[[褒姒|褒姒]]，三代所由亡也」）等，在沒有其他資料佐證的狀況下，就有可能犯了與此故因此的謬誤。

一個實際的例子是流行病學研究曾發現，接受[[激素替代療法|激素替代療法]]的婦女，發生[[冠心病|冠心病]]的比率較一般人低，使醫界一度認為激素替代療法有預防冠心病的效果。然而後續的隨機[[對照實驗|對照實驗]]顯示，激素替代療法會使[[冠心病|冠心病]]的發生率增加。重新分析資料後發現，接受激素替代療法的婦女社会地位多半較高，她們通常有較好的飲食、運動習慣，因而比較不容易發生冠心病。<ref>{{cite journal |author = Lawlor DA, Davey Smith G, Ebrahim S |title = Commentary: the hormone replacement-coronary heart disease conundrum: is this the death of observational epidemiology? |journal = Int J Epidemiol |volume = 33 |issue = 3 |pages = 464–7 |pmid = 15166201 |doi = 10.1093/ije/dyh124|date=June 2004}}</ref>
