'''山姓'''為[[中文姓氏|中文姓氏]]之一，在《[[百家姓|百家姓]]》中排名第227位。

== 姓氏起源 ==
#出自姜氏，為烈山（又作列山〕氏的后代。據《風俗通》記載：“山氏，為古代烈山氏的后代”；相傳，[[炎帝|炎帝]]為上古時[[姜姓|姜姓]]部族的首領，是[[少典|少典]]娶蟜氏女而生，原居于[[姜水流域|姜水流域]]，后向東发展到中原地區。炎帝出生于烈山，故號烈山氏。他的后代有的就用“山”作為姓氏。望族出于[[河内|河内]]。
#源于以官為姓氏。據《名賢氏族言行类稿》载，周代有山務之官，掌管山林，后代以官為氏。[[周代|周代]]掌管山林开发管理的官員叫山師，也稱山虞。有人世襲山師，子孙便姓山，稱為山氏。
#出自以祖名為氏。[[春秋|春秋]]時[[楚國|楚國]]有个高官叫[[叔山冉|叔山冉]]，他的后代以他名字中的“叔山”的山字為姓，世代相傳。
#出自他族改姓。據《[[魏書|魏書]]·官氏志》的記载，古代有[[复姓|复姓]]吐难氏，[[北魏|北魏]]時改為山氏。
#出自[[回族|回族]]中有山姓。回族中的山姓，早期見于元朝官員山僧（晉寧路總管），其先祖為康里人，显然是人關后才使用的山姓。現今山姓回族主要分布在[[河南|河南]]、[[寧夏|寧夏]]。

== 得姓始祖 ==
烈山、叔山冉。山氏的姓源久遠，至少也有3000年的歷史，且來源不一。根據《風俗通》上記载說，山氏，是古代烈山氏的后代，望族出于河内，此支山姓奉烈山为山姓的得姓始祖。又春秋時楚國有个高官叫叔山冉，他的后代以他名字中的“山”字為姓，世代相傳。叔山冉的后代中也有以“冉”為姓的，所以叔山姓和冉姓，有同一个始祖。此支山姓恭叔山冉為山姓的得姓始祖。

== 歷史名人 ==
*[[山濤|山濤]]（公元205年─公元283年），字巨源，晉代吏部尚書。為“竹林七賢”之一。西晉河内懷縣（今河南武陟西方，據河南省武陟縣大虹橋鄉網站資料，大虹橋鄉是山濤出生地，不知詳否）人。虽居高官榮責，卻貞慎簡約，俸祿薪水，散于邻里，時人謂為“璞玉渾金”。武帝時任尚書之職，凡甄拔人物，各有題目，稱"山公啟事"。
*[[山簡|山簡]]（公元253年—公元312年），字季倫。河内懷縣（今河南武陟西方）人，乃著名文學家、“竹林七賢”之一的山濤之子。山簡因镇守襄阳時飲酒優游而聞名于古今。山簡初任為太子舍人，歷侍中尚書，领[[吏部|吏部]]，有建言。永嘉三年（公元309年）出為镇南将軍，都督荆、湘、交、廣四州（今两湖两廣一带）諸軍事，镇襄阳。
*[[山琮|山琮]]，隋代人，《隋書》記载曾作《老子傳》一部，名字與元朝山琮相同.
*[[山康|山康]]，唐代高僧，十五歲學佛，遍游四方。相傳，他云游睦州時，人們曾看見佛从他口中出來，后賜好廣道大師。
*[[山行章|山行章]]，宋代人，保眉州無羅城等處功績顯著。
*[[山說|山說]]，宋代人，登科。
*[[山悰|山悰]]，宋代人，桂陽王鑠侍讀。
*[[山褥環|山褥環]]，宋代人，領軍暢當饒將。
*[[山琮|山琮]]，元代人，大德三年状元，山東朝城山姓始祖。
*[[山錫之|山錫之]]，明代蘇州人，洪武年初知縣，改創縣治及學舍公宇焕然一新，至今賴焉。
*[[山青|山青]]，明代徐州人，古今知大義，沉毅有勇略，為燕山護衛百戶。永樂年初，以靖難功績當上右軍都督僉事。
*[[山云|山云]]，明代徐州人，永樂年中，以指揮遠征功績升為都督僉事。宣德正統元年，平息蠻將，鎮廣西，成為當地的都督。
*[[山秀|山秀]]，明代仁和人，正統年中為睢陽縣丞相。
*[[山浩|山浩]]，字文淵，明代酃縣人，天順年中考獲舉人，任為平陽通判。
*[[山禹|山禹]]，明代吳州人，嘉靖丙午年考獲舉人，任為府同知。

== 近代名人 ==
*[[山棟|山棟]]、[[山梁|山梁]]、[[山青|山青]]，民國西康省知名鄉紳，四川瀘定人（山姓樹字輩人〕。
*[[山侖|山侖]]，山東東龍口人（山姓鲁東系第十七世 即廣字輩人），中國工程院院士，公元1933年生。
*[[山紅紅|山紅紅]]，河南洛陽人，中國石油大學（華東）校長。
*[[山翀|山翀]]，浙江溫州市人，中國歌劇舞劇院青年舞蹈家，1969年生，1991年畢業於北京舞蹈學院民族舞劇表演系。中國舞蹈家協會會員，中國戲劇家協會會員，文化部青聯委員，北京市青聯委員，首都女藝術家聯誼委員。
*[[山文華|山文華]]，湖北武漢人，1946年7月生，大學文化、廣州市道路擴建工程辦公室總工程師、高級工程師。
*[[山艷春|山艷春]]，山東人，1962年12月生，大學文化、暨南大學醫學院附属醫院副主任醫師。
*[[山白|山白]]，江蘇揚州人，回族，1956年7月生。高級工程師。畢業于華東理工大學（原華東化工學院）。現任中國石化集團公司儀征化纤股份公司涤倫一厂設備高級工程師。
