'''山下亨'''是[[日本|日本]][[搖滾|搖滾]][[樂團|樂團]][[ONE_OK_ROCK|ONE OK ROCK]]的唯一[[吉他手|吉他手]]，多半以藝名'''Toru'''稱之。

==簡歷==
成立ONE OK ROCK前，曾和小浜良太（現ONE OK ROCK的[[貝斯手|貝斯手]]Ryota）在Hip-Hop團體HEADS內活動<ref>http://www.jpopasia.com/celebrity/toru/</ref>，兩人為童年好友。他們兩人於2005年就讀[[堀越高等學校|堀越高等學校]]時和學長Alex成立了ONE OK ROCK，擔任團長、第二吉他手以及[[主唱|主唱]]。並在聽到[[森內貴寬|森內貴寬]]（現ONE OK ROCK的主唱[[Taka|Taka]]）的演唱時，力邀他加入ONE OK ROCK擔任主唱<ref>http://karashuki.tumblr.com/post/71605558461/the-history-of-one-ok-rock-part-1</ref>。和Taka為樂團的主要詞曲創作者。2009年Alex因個人因素離開樂團後，擔任樂團的第一吉他手<ref>http://natalie.mu/music/news/16457 {{ja}}</ref>。
