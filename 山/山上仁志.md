'''山上仁志'''（1966年 - ）是一位[[日本|日本]]的[[游戏制作人|游戏制作人]]，现为[[任天堂|任天堂]][[任天堂企划开发本部|企划开发本部]]企划开发部第2组管理人。

== 生平 ==

出生于[[大阪府|大阪府]]，[[ABO血型系统|血型]]是B型<ref>{{cite web|url=http://www.nindori.com/interview/156FE/index.html|title=ファイアーエムブレム 暁の女神 開発スタッフインタビュー|accessdate=2011-04-07|author=|date=2007-04|publisher=ニンドリドットコム|archiveurl=https://web.archive.org/web/20070618061417/http://www.nindori.com/interview/156FE/index.html|archivedate=2007年6月18日}}</ref>。

于1989年进入任天堂工作。在[[任天堂開發第一部|开发第一部]]进行《パネルでポン》等[[益智游戏|益智游戏]]以及[[掌上遊戲機|掌机]]游戏相关的开发<ref name="BFGB">{{cite web |url=http://www.nintendo.co.jp/nom/0008/b_fight/page02.html |publisher=任天堂 |title=バルーンファイトGB／開発者インタビュー|date=2000 |accessdate=2011-04-07}}</ref>。

近年主要在企划开发部<ref name="tetris">{{cite web |url=http://www.nintendo.co.jp/nom/0606/22/index.html |publisher=ニンテンドーオンラインマガジン |title=『テトリスDS』開発スタッフインタビュー |date=2006-06 |accessdate=2011-04-07}}</ref>担当外包作品的制作人。此外，与他相关的作品有不少被[[计算机娱乐分级机构|计算机娱乐分级机构]]（CERO）归入年龄指定分类，在[[任天堂|任天堂]]可以说是少数派。

2017年2月13日，在任天堂公布的两段关于《火焰之纹章 回声 另一位英雄王》的影片中，他分别用相当不纯熟的普通话和粤语宣布该作将加入繁体和简体中文语言选项，在玩家中引起极大反响<ref>{{Cite news|url=http://www.g-cores.com/articles/23103|title=《火焰纹章 Echoes 另一位英雄王》中文化确定|accessdate=2017-10-25|archive-url=https://web.archive.org/web/20171017042248/http://www.g-cores.com/articles/23103|archive-date=2017-10-17|dead-url=yes}}</ref>。
