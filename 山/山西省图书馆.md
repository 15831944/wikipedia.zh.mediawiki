'''山西省图书馆'''（'''Shanxi Library''')是山西省规模最大的公共图书馆，始建于清宣统元年（1909年），也是国内为数不多的百年老馆之一。

== 历史 ==

=== 中华人民共和国成立前 ===
山西省图书馆始建于1909年10月9日，当时名为[[山西教育图书博物馆|山西教育图书博物馆]]，馆址位于[[太原市|太原市]]文庙（今[[山西省民俗博物馆|山西省民俗博物馆]]）。<ref>{{cite book |author=李慧智 |title= 《[[藏与用—山西省图书馆的收藏与利用|藏与用—山西省图书馆的收藏与利用]]》 |publisher= 山西人民出版社 |isbn= 7-203-05135-8 |date=2004年10月}}</ref>1933年10月更名山西省立民众教育馆。[[抗日战争|抗日战争]]期间日军占领太原后在民众教育馆原址成立了[[太原博物馆|太原博物馆]]，又于1940年更名为山西新民教育馆。1945抗日战争胜利后恢复山西省立民众教育馆的名称。1948年[[国共内战|国共内战]]陷入停顿。1949年4月24日解放军占领太原后更名山西省图书博物馆。1953年更名[[山西省博物馆|山西省博物馆]]。

=== 中华人民共和国成立后 ===
[[File:Shanxi_Library_Wenyuan_Branch.jpg|thumb]]
1957年[[山西省委|山西省委]]和[[山西省人民委员会|山西省人民委员会]]决定筹建新的山西省图书馆，将图书馆与博物馆分离。1960年8月28日，位于文源巷的山西省图书馆新馆正式对外开放。1966年“[[文化大革命|文化大革命]]”开始，图书馆遭受重大损失。1978年底被禁锢的图书基本被开放。1985年缩微第一次试拍。2007年[[山西省古籍保护中心|山西省古籍保护中心]]成立，山西省古籍保护工作启动。2013年7月1日位于长风商务区的新馆正式对外开放，原文源馆成为古籍馆。

== 规模 ==
山西省图书馆目前由文源馆和长风馆组成。文源馆位于解放南路文源巷，总建筑面积1.8万平方米，主体建筑中心四层、两翼三层，曾被列为全国图书馆建筑典范之一。包括山西省少年儿童图书馆和[[山西省古籍保护中心|山西省古籍保护中心]]。长风馆为山西省图书馆的主馆区，位于太原市[[长风商务区|长风商务区]]，是“[[十一五|十一五]]”时期全省大型公共文化设施建设项目之一，占地面积60亩，建筑面积5万平方米，设计藏书量700万册，有各类阅览室27个，可容纳3000人同时阅览，还有视听室、报告厅、多功能厅、展厅以及可供读者选择的专题阅览区、数字多媒体体验区、休闲阅读区等。<ref>{{cite web |title= 山西省图馆简介|url=http://lib.sx.cn/html/1/dzzn/33.html}}</ref>

== 馆藏 ==
山西省图书馆馆藏总量340余万册（件），其中普通文献2,554,668册，古籍近300,000册（件），地方文献120,000余册，视听文献26,465张，[[缩微制品|缩微制品]]4,950盒，电子图书503,222种。<ref>{{cite web |title= 资源介绍|url=http://lib.sx.cn/html/1/dzzn/32.html}}</ref>
