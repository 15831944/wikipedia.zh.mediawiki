《'''山海逆戰'''》是香港漫畫家[[邱福龍|邱福龍]]作品。山海之間無奇不有，天下更有各種不同奇珍異獸存活人間。智者伯益發現食用[[異獸|異獸]]之肉，飲其鮮血，便能獲得[[異獸|異獸]]之力。而這些身負奇能者，被稱為[[異士|異士]]！[[伯益|伯益]]走遍天下，筆錄所見的[[異獸|異獸]]，並寫成《[[山海經|山海經]]》，更是以此創立的[[山海派|山海派]]，宣揚食用[[異獸|異獸]]獲得異能的理念。但是當[[伯益|伯益]]消失之後，[[山海派|山海派]]卻逐漸地失去勢力平衡。

== 組織/根據地 ==
*[[山海派|山海派]] − 海角崖（總壇）。
 
== 角色介紹 ==

=== 勾徹 ===
: 本作品男主角，[[山海派|山海派]]的俘虜，髮色蒼白的少年。被三人異士小組派出消滅[[狸力|狸力]]，因緣際會吃下異獸[[視肉|視肉]]、[[狸力|狸力]]之肉，擁有'''[[特殊異能|特殊異能]]''' →  再生能力(視肉)、土之力(狸力)。蚩尤转世，后发现自己是上古魔神蚩尤转世，受上古黄帝所托，身负救世重任-击杀炎帝，阻止他把世人变成异兽的计划（神异创世）。击杀炎帝后，获取其力量，再度打开穹界之渊，把异兽迁移到别的星体，令人间永享太平。因部分逆战碎片在黄帝石躯内，所以导致先前得到的逆战不完整，后更在击杀姚星辰一战中损坏了逆战，需返回一线天拿回碎片重铸完整逆战。在重铸逆战途中因头盔神力遇见自身的前世（蚩尤）并返回蚩尤的过去与其并肩作战击杀刑天。现已获得神之躯体和重铸全新的逆战，更获得蚩尤心法传功“无法循-无常道，无极意-无限绝，无念起-无想定，心无心-我无我，逆天者胜-一往无常”

=== 于若蘭 ===
: 行道門現任掌門。

=== 殷天翔 ===
: 行道門弟子，后背叛行道会，加入山海派成为火将，食人面异兽英招获得人兽合一之力。

=== 蚩尤 ===
异兽之主，雄霸荧惑之境，好勇斗狠的不败战神，强调力量与速度，遇强越强，修习内功，轻功和护身气劲，后更铸刀创招，并感悟出终极杀招-天地寂灭，唯我一刀。

逆战刀诀（共分七式）：狂凶极逆，逆焰燃天，洪荒逆流，离刀叛道，触逆龙鳞，云耀逆闪，溯风萧萧

逆天战道（内功）共分三层：无法无常，无极无限，无念无想

轻功：流离闪空

护身气劲：金刚不破

終極殺招：天地寂灭，唯我一刀


=== 炎帝 ===
掌管人間秩序的百勝武神，修习内功，劍招，轻功和护身气劲，神劍赤霄與鎧甲組成最強攻防，并有终极杀招-天命。

神炎劍道（共分六式）：星火燎原，天火追雲，離火爍金，劫火焚城，峰火沖霄，聖火破穢

神炎玄天功（内功）共分五层：紫耀，青燁，白煉，赤炫，無明

無明天劍 (無明之境方能使用) 共三招：天照，天塹，天罰

轻功：流火無痕

护身气劲：神炎真軀

終極殺招：天命



=== 黃帝 ===
公正嚴明，宅心仁厚的太虛之神，修习内功，劍法，配合神劍天誅，能夠破除一切邪惡，并有终极杀招-九鼎封禪大法。

神皇九鼎劍法（共分十式）：太極飛雲，精明洞淵，天地陰陽，蒼梧祀天，混沌初開，山岳五神，浮光洞天，回光晃照，鎮龜伏蛇，靈光煉神

黃帝內經（内功）共分七层：宗動天，清浄天，上行天，成輪天，兜率天，太白天，光明天

終極殺招：九鼎封禪大法

== 幫派簡介 ==
=== [[山海派|山海派]] ===
*'''伯益''' → 本作品男主角之一。[[山海派|山海派]]門主。一代奇人[[伯益|伯益]]以自創神功[[『|『]][[潛無限|潛無限]][[』|』]]，屠宰了五頭霸王凶獸，五頭霸王凶獸分別是[[火鳳凰|火鳳凰]](重生)、[[化蛇|化蛇]](御水)、[[足朮踢|足朮踢]]、[[蜚|蜚]](千里通)、[[鮯鮯魚|鮯鮯魚]](極速)五頭異獸，得五大驚世[[異獸|異獸]]之力。[[伯益|伯益]]走遍天下，筆錄所見的異獸，寫成《[[山海經|山海經]]》。並以此經為名，創立[[山海派|山海派]]。鼓吹獵獸、食獸之風，領導眾[[異士|異士]]對抗凶獸之禍。其後又食[[冰鳳凰|冰鳳凰]]之肉，所食異獸達到六頭。败于姚星塵后，又食异兽燭陰，得風雨雷電之力。
*'''姚星塵''' → 山海派代任門主。自[[伯益|伯益]]失蹤後，山海派群龍無首，改由'''[[姚星塵|姚星塵]]'''執掌山海派重大基業，贊代決策和領導之責。之後[[伯益|伯益]]七年生死未卜，由'''[[姚星塵|姚星塵]]'''暫時代理門主之位，吃下四種異獸，分別為[[吡鐵|吡鐵]]、[[猾褢|猾褢]]、[[梁渠|梁渠]]、[[檮杌|檮杌]]。 在行道会分舵一战更创新以檮杌异能吸收饕餮凶魂，自创神功阎蚀虚凶。但被勾徹以新创绝学断生求道配合天地寂灭，唯我一刀击杀。
*'''夏燊''' → 山海派四將之一。黃髮藍眼，曾吃下異獸[[畢方|畢方]]之肉，擁有異獸[[畢方|畢方]]'''[[特殊異能|特殊異能]]''' → [[『|『]][[火|火]]之力[[』|』]]。在行道会分舵一战被殷天翔击杀。
*'''傲嵐''' → 山海派上士，三人異士小組之首。紫髮紅眼，為首領隊的女子，面若寒霜，外表冷酷，戴著一對鐵製的手套。曾吃下異獸[[欽原|欽原]] 之肉，雙掌晶化蘊含[[『|『]][[施毒|施毒]][[』|』]]能力，引申出三大絕技1.觸亡。2.封頂。3.觸痛。騎馬駕駛馬車離開[[山海派|山海派]]總壇−海角崖，前赴中原，執行代門主[[姚星塵|姚星塵]]委派的任務，對付異獸[[狸力|狸力]]，得到[[狸力|狸力]]的肉帶回[[山海派|山海派]]總壇。
*'''袁通''' → 山海派上士，三人異士小組之二。曾吃過異獸[[舉父|舉父]]之肉。
*'''徐九''' → 山海派上士，三人異士小組之三。戴著獨門火器 —— [[三眼銃|三眼銃]]，爆彈三管齊發，殺傷力非同小可。準備拿來對付異獸[[狸力|狸力]] 。曾吃過異獸[[犰狳|犰狳]]之肉，具有異獸[[犰狳|犰狳]]'''[[特殊異能|特殊異能]]''' →[[『|『]][[氣絕|氣絕]][[』|』]]的能力，裝死幾可亂真。

=== [[行道會|行道會]] ===
武功：修兵道、武天下

修兵道：將自身當成武器，修練至無堅不催之境，包含腿拳掌三套武學─神木腿、宿星拳、空掌。

武天下：集天下武學精髓而出，等閒不動用。包含三招─一即是全、武無第二、天下三絕（天絕、地絕、人絕）

=== [[神農氏|神農氏]] ===
武功：七輪先天炁、九矢射天、廚神道
