'''山葵'''（{{jpn|j=山葵|hg=わさび|kk=ワサビ|rm=wasabi|f=t}}，音译为“哇沙米”），學名{{lang|la|''Eutrema japonicum''}}，是一種屬於[[十字花科|十字花科]][[山萮菜屬|山萮菜屬]]的植物，又稱為「'''山萮菜'''」，為綠色植物，味道極其強烈。它的辛辣跟[[辣椒|辣椒]]不同，辣椒的辣味刺激舌頭，而山葵的辣味卻刺激[[鼻竇|鼻竇]]。山葵常見於日本[[山谷|山谷]][[河流|河流]]旁邊生長，其他品種包括{{lang|la|''W. koreana''}}及{{lang|la|''W. tetsuigi''}}；市面上普遍出售的兩種山葵是{{lang|la|''W. japonica ''}}'Daruma'（達磨）及{{lang|la|''W. japonica ''}}'Mazuma'（真妻），但山葵還有許多不同品種。因为山葵较为少见，也常用味道近似的[[辣根|辣根]]生产仿制品。

== 名稱 ==
日文「{{lang|ja|山葵}}」兩字跟「{{lang|ja|ワサビ}}」的發音並不對應，為[[日語|日語]][[熟字訓|熟字訓]]一例；「山葵」原指種在幽暗山上，葉如[[錦葵科|錦葵科]]植物的葉的植物。[[日文漢字|日文漢字]]名「{{lang|ja|和佐比}}」於918年首次出現於《{{lang|ja|本草和名}}》中，但純為發音用字。

中文有時誤稱山葵為[[芥末|芥末]]，但其實芥末是由[[芥菜|芥菜]]種子生产的，與山葵無關，兩者英文名稱也不同（山葵英文為Wasabi，芥末英文為Mustard），只是兩者都是刺激性醬料。

香港稱山葵為'''芥辣'''，但在日常口語會直接用訓讀音Wasabi。

== 食用 ==
[[File:Wasabi_by_EverJean_in_Nishiki-ichiba,_Kyoto.jpg|thumb]][[錦市場|錦市場]]销售的新鲜山葵根]]
山葵通常以莖部出售；山葵莖需要磨成細泥狀才能使用，當山葵泥研磨好後需將山葵莖包好，防止氣味揮發跟變質；因此壽司師傅製作壽司時會把山葵放在魚和飯之間。較講究的壽司師傅會在製作壽司時，才會研磨所需的山葵泥，好保持山葵泥清涼的風味跟避免變質。

山葵葉可以食用，也有山葵莖的辣味。山葵葉用鹽醋醬汁醃製過夜後可製成山葵沙拉，又可以用醬油開水烹煮，作為下飯、下酒的小菜或卷壽司的材料食用，甚至可以蘸上澱粉漿後炸成[[天婦羅|天婦羅]]。有些壽司店為了不敢吃山葵的客人，會在部分壽司端出前用山葵葉包好，等客人點餐時才端上來達到山葵泥壓制魚腥的相同效果。

市場上銷售的所謂“山葵”，也有糊狀和乾粉產品。糊狀的山葵通常用[[牙膏|牙膏]]筒狀的管子裝載。粉狀的產品有袋裝和罐裝產品，可以用水開成糊。但這些產品的大部分是使用類似[[蘿蔔|蘿蔔]]的十字花科植物[[辣根|辣根]]（''Armoracia rusticana''）和綠色食用色素生產的仿製品。部分日本生產的糊狀產品中含有真正山葵，則包裝上加以字樣區別，“本わさび使用”（使用真正山葵）是含有50％或以上山葵，“本わさび入り”（加入真正山葵）是含有50％以下的產品。但，真正山葵也實際上以葉子和莖部為主。真正的山葵根具有一定的甜味，與這些代用品的味道不同，但一旦干燥後，味道和香辣味便會消失，因此真正的山葵不適合加工成粉狀產品。在[[美國|美國]]或[[中國|中國]]幾乎所有壽司店，和日本部分[[壽司|壽司]]店，都用仿製山葵，因為真正的山葵非常昂貴。

== 用途 ==
山葵糊看起來像風行於[[美國|美國]]或[[墨西哥|墨西哥]]的[[鱷梨醬|鱷梨醬]]；被人惡作劇或因為不了解[[日本料理|日本料理]]的人，有時會一次吃太多山葵，刺激[[鼻竇|鼻竇]]而感到不適，幸好山葵的辣味感比[[辣椒|辣椒]]的辣味感維持較短。山葵可以放在豆腐上吃，或和凉的[[蕎麥麵|蕎麥麵]]一起食用。小量的山葵也適合放在[[壽司|壽司]]或[[生魚片|生魚片]]上。很多時候人們會把山葵跟[[醬油|醬油]]混合，才讓食物蘸上；但是山葵的味道於[[水|水]]中會迅速溶解，吃山葵的最好方法是把食物蘸上醬油後，才加上山葵，並要避免山葵與醬油混合。

== 化學 ==
[[File:Wasabi_071103.jpg|thumb]]
山葵的獨特味道是由[[異硫氰酸鹽|異硫氰酸鹽]]而來的，包括：

* 6-甲基硫己基異硫氰酸鹽
* 7-甲基硫庚基異硫氰酸鹽
* 8-甲基硫辛基異硫氰酸鹽

研究發現異硫氰酸鹽能抑制[[微生物|微生物]]生長。這可能解釋了為甚麼容易變壞的[[海鮮|海鮮]]通常佐以山葵來吃。可是如果對海鮮的品質有懷疑，無論是否佐以山葵都不該進食，因為山葵完全不能治療[[食物中毒|食物中毒]]。

== 栽培 ==
[[File:wasabi_crop_2006-07-29.JPG|thumb]]耕种的山葵]]
[[File:Wasabi,_Iwasaki_Kanen_1828.jpg|thumb]]於1828年绘制的山葵绘图]]
因為能夠大型栽培山葵的地方並不多，今日大部分加工「山葵」其實只是[[染|染]]成綠色的[[歐洲|歐洲]][[辣根|辣根]]（''Armoracia rusticana''）末，或是辣根末加上山葵末製成。在日本，栽培山葵的地方有：

* [[靜岡縣|靜岡縣]]
* [[長野縣|長野縣]]
* [[島根縣|島根縣]]
* [[山梨縣|山梨縣]]
* [[岩手縣|岩手縣]]

人工栽培山葵的地方北至[[北海道|北海道]]，南至[[九州_(日本)|九州]]。真正山葵的需求非常大，日本常需要從以下地方大量進口：

* [[美國|美國]][[俄勒岡州|俄勒岡州]]
* [[中國大陸|中國大陸]]
* [[紐西蘭|紐西蘭]]

山葵栽培需要[[雞|雞]][[糞|糞]]等[[肥料|肥料]]及長期流動的水，如果不管制的話，會對河流構成嚴重污染。且山葵對於[[地力|地力]]的消耗是相當大的，只要是收割完山葵的田，為使種植山葵遺留的物質能夠有時間去消耗掉，需要休耕五年才能再種植山葵或其他植物。

== 準備方法 ==
[[File:WasabiOnOroshigane.jpg|220px]]上的山葵]]
山葵通常用金屬製的[[卸鐵|卸鐵]]研磨，但有人愛用較傳統的工具，如一面光滑一面粗糙的乾[[鯊魚|鯊魚]]皮；也有人親手製出有齒的磨來準備山葵。不過新鮮的山葵相當昂貴，許多餐館會寧願使用粉末狀的代替品。
