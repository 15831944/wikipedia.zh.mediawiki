'''山菠菜'''（[[学名|学名]]：{{lang|la|'''''Prunella asiatica'''''}}）为[[唇形科|唇形科]][[夏枯草属|夏枯草属]]下的一个种。又稱為[[茄茉菜|茄茉菜]]、[[野菠菜|野菠菜]]、[[榆錢菠菜|榆錢菠菜]]、[[麝香草|麝香草]]、[[法國菠菜|法國菠菜]]和[[奶油葉|奶油葉]]。[[菠菜|菠菜]]的近親，外型與菠菜相似，卻是不同的蔬菜，但營養更勝菠菜。

== 產地分布== 

*[[青海|青海]] ( [[中國|中國]] ) 、[[新疆|新疆]]、[[內蒙古|內蒙古]]、[[河北|河北]] ( [[中國|中國]] )、[[黑龍江|黑龍江]] ( [[中國|中國]] )、[[吉林|吉林]] ( [[中國|中國]] )、[[遼寧|遼寧]] ( [[中國|中國]] )、[[山西|山西]] ( [[中國|中國]] )、[[山東|山東]] ( [[中國|中國]] )、[[江蘇|江蘇]] ( [[中國|中國]] )、[[浙江|浙江]] ( [[中國|中國]] )、[[安徽|安徽]] ( [[中國|中國]] )、[[江西|江西]] ( [[中國|中國]] )、[[台灣|台灣]]、[[日本|日本]]、[[北韓|北韓]]、[[歐洲|歐洲]]、[[西伯利亞|西伯利亞]]。

== 營養價值 ==
{{Unreferenced section}}
山菠菜擁有豐富的[[維他命A|維他命A]]、[[維他命B1|維他命B1]]、[[維他命B2|維他命B2]]、[[維他命C|維他命C]]、[[鈣質|鈣質]]、[[鐵質|鐵質]]、[[蛋白質|蛋白質]]、[[胡蘿蔔素|胡蘿蔔素]]等營養，常吃有助造血補血,是一種營養豐富的蔬菜。

若直接炒食會微帶苦澀味，屬品種特性，經熱開水燙過後再炒食，即可去除部份苦澀，口感佳。

古[[希臘人|希臘人]]和[[羅馬人|羅馬人]]廣泛用來醫治[[咽喉痛|咽喉痛]]、[[消化不良|消化不良]]和[[黃疸病|黃疸病]],也用作菜食。全草入藥，[[江蘇|江蘇]]民間用作[[利尿|利尿]]、[[降血壓|降血壓]]、治[[淋病|淋病]]及[[瘰癧|瘰癧]]，又可當[[茶飲|茶飲]]。
