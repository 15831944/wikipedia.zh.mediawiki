'''山口夕輝'''（{{bd|1993年|10月17日||}}）是[[日本|日本]]女子團體[[NMB48|NMB48]] Team N的成員。[[大阪府|大阪府]]出身。KYORAKU吉本所屬。

== 經歴 ==
; 2010年
: 9月20日、參加『NMB48一期生徵選會』合格（參加總數7,256名、最終合格者26名）。
: 2010年10月9日、在『AKB48東京秋祭』以NMB48第1期研究生26名之1人首次登場。

; 2011年
: 6月14日、接替宣布畢業的森彩華昇格到Team N。
: 9月20日、『[[AKB48第24張單曲選拔猜拳大會|AKB48第24張單曲選拔猜拳大會]]』本戰勝出得到前16，参加[[AKB48|AKB48]]的第24張單曲選拔的決定<ref>[http://ameblo.jp/akihabara48/entry-11024136395.html AKB48第24張單曲猜拳選抜決定！&1位〜8位發表！] - AKB48官方部落格〜TOKYO DOMEまでの軌跡〜 2011年9月20日</ref>。

; 2017年
: 3月14日、於Team BII公演時宣布將自NMB48畢業。
: 4月28日、於Team BII卒業公演後正式畢業。
