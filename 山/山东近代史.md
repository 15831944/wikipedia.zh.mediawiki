{{山东历史}}
'''山东近代史'''是指从1840年中英[[鸦片战争|鸦片战争]]爆发以后的[[山东历史|山东历史]]。
== 大事记 ==
;1840年
*6月，中英[[鸦片战争|鸦片战争]]爆发。8月15日，英侵略军的舰队从浙江北上经过山东海面，山东巡抚[[托浑布|托浑布]]奉命带兵赴[[登州|登州]]督防。
*9月17日，清廷命沿海督抚，洋船经过不必开放枪炮。18日，托浑布遣人馈赠英军牛羊蔬菜。
*本年，藏书家[[杨以增|杨以增]]在聊城建藏书楼“[[海源阁|海源阁]]”，为晚清四大私人藏书楼之一。
*本年，德州、聊城等62州县夏秋受[[涝灾|涝灾]]。另有20县受[[旱灾|旱灾]]。粮食减产五成以上。

[[清圣祖|清圣祖]][[康熙|康熙]]六十一年（1722年），山东耕地达到9000余万亩。[[第二次鸦片战争|第二次鸦片战争]]中，清政府被迫簽訂《[[天津條約|天津條約]]》开放[[登州|登州]]等10处为[[通商口岸|通商口岸]]<ref>{{cite web|title=抗擊英法聯軍入侵大沽之戰|publisher=中國國防資訊網|url=http://info.cndsi.com/html/20070823/28657104133.html|date=2007-08-23|accessdate=2009-12-03|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20111217184338/http://info.cndsi.com/html/20070823/28657104133.html|archivedate=2011-12-17}}</ref>，1861年改定[[烟台|烟台]]为通商口岸{{notetag|烟台是山东第一个开放口岸。}}。1888年，[[北洋水师|北洋水师]]于[[山东|山东]][[威海卫|威海卫]][[刘公岛|刘公岛]]正式成立，7年后在[[甲午战争|甲午战争]]中全军覆没，[[威海|威海]]被日军攻占<ref>[[王双印|王双印]]《甲午战后中国海军近代化建设述论》</ref>。
== 德国对山东的侵略 ==
{{main|曹州教案|胶澳租借条约}}
[[File:Lange_diercke_sachsen_asien_tsingtau.jpg|thumb]]
1867年，德国地理学家[[李希霍芬|李希霍芬]]在中国多个省份考察后认为[[胶州湾|胶州湾]]东岸有着优越的装船条件，若建一条从[[青岛|青岛]]到省府[[济南|济南]]的铁路可以方便获得沿线的矿产资源。1896年12月14日，德国驻华公使海靖(Herr Baron von Heyking)正式向大清[[总理衙门|总理衙门]]提出租借胶州湾及其周边地区五十年作为“储煤站”的要求，被总理衙门拒绝。1897年11月1日，发生[[曹州教案|曹州教案]]，两名德国传教士在曹州府（今[[菏泽市|菏泽市]]）巨野县被杀，德国趁机派军舰于11月14日占领胶州湾地区（今属[[青岛市|青岛市]]）。1898年3月6日由[[李鸿章|李鸿章]]、[[翁同和|翁同和]]代表清政府，在北京与德国驻中国公使海靖签署中德《[[胶澳租界条约|胶澳租界条约]]》。条约规定，德国租借胶澳（今[[青岛|青岛]]）九十九年，德国修筑两条从胶澳到济南的铁路并获得沿线三里内开矿权，德国获得承办山东省各项事务的优先权共三端十款，之后分别又在8月22日签订《[[胶澳租地合同|胶澳租地合同]]》，10月6日签订《[[胶澳潮平合同|胶澳潮平合同]]》和《[[胶澳边界合同|胶澳边界合同]]》，确定了德国占领整个胶州湾沿岸，并将山东省划为德国的势力范围。

1897年山东省[[曹州府|曹州府]][[巨野县|巨野县]]发生[[曹州教案|曹州教案]]，次年[[清政府|清政府]]被迫签署《[[胶澳租借条约|胶澳租借条约]]》将[[青岛|青岛]]和[[英租威海卫|威海]]分别租借给[[德国|德国]]和[[英国|英国]]<ref name="qdda1">{{citeweb|title=青岛市档案馆举办《胶澳租借条约》征集情况介绍会|url=http://www.qdda.gov.cn/front/laoqingdao/preview.jsp?ID=1810239&subjectid=12487493309361276001|author=青岛市档案馆征集处|publisher=青岛档案信息网|accessdate=2012-04-26|language=zh-cn}}</ref>。德国还修筑了从青岛通往济南的[[胶济铁路|胶济铁路]]（1905年），和[[津浦铁路|津浦铁路]]的北段（1911年），把山东当作自己的势力范围。1899年，[[义和团|义和团]]从山东兴起，攻击遍布全省的西方传教士<ref>{{cite web|author1=马执斌|title=是“扶清灭洋”，还是“助清灭洋”？|url=http://old.pep.com.cn/czls/js/tbjx/dy/8s/201008/t20100825_735604.htm|website=人民教育出版社|accessdate=2016-11-30|date=2002-11-15}}</ref>。清末，面对[[俄国|俄国]]和[[日本|日本]]的威胁，东北终于向[[汉族|汉族]]移民开放，加上民国初年[[张作霖|张作霖]]的鼓励措施，在100年间有为数上千万的山东农民[[闯关东|闯关东]]谋生，成为今天东北人口中的重要组成部分。

== 英国对山东的侵略 ==
{{main|瓜分中国|订租威海卫专条}}
[[File:British_Weihaiwei_flag.svg|thumb]]
1898年，[[英国|英国]]强租[[威海卫|威海卫]]。1900年，设威海卫行政长官署，属[[英国殖民部|英国殖民部]]。
在第一次世界大战后，德国作为战败国无条件交出[[胶州湾租借地|胶州湾租借地]]，但在英国等国的操纵下将上述地区交给日本。
==日本对山东的侵略==
1914年日本入侵，引发[[山东问题|山东问题]]。1922年[[华盛顿会议|华盛顿会议]]上经美国调停，日本将青岛和胶济铁路归还[[中华民国|中华民国]]。1928年5月3日[[國民革命軍|國民革命軍]]在[[北伐|北伐]]途中经过山东[[济南市|济南城]]时与日军发生冲突，引发[[济南惨案|济南惨案]]<ref>{{cite web|title =济南惨案中山东交涉署人员前后事|url =http://ws.jnzx.gov.cn/index.php?m=content&c=index&a=show&catid=168&id=317|author =李宗益|publisher =濟南文史網|accessdate =2010-02-25}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite book|title=《日本史話(近代篇)》|author =  汪公紀 |publisher = 聯經出版事業股份有限公司 |year = 2005 |isbn = 9570829311}}</ref>。1930年，[[国民政府|国民政府]]从英国手中收回威海。1937年[[七七事变|七七事变]]后，省内国民党政权、日伪政权、中共政权并立，其辖区犬牙交错。

==抗日战争结束后==
1945年8月，抗日战争胜利后，国民党政权、中共政权依然并存。1947年5月在山东省中部沂蒙山区的[[孟良崮战役|孟良崮战役]]中，国军最精锐的王牌[[国民革命军第七十四军|七十四师]]被[[华东野战军|华东野战军]]消灭<ref>{{cite web|title=孤注一擲的豪賭——不可想像的孟良崮戰役|url=http://big5.huaxia.com/js/gc/2007/00627698.html|website=華夏經緯網|publisher=中華網|accessdate=2016-11-30|date=2007-05-30}}</ref>。1949年6月2日，[[中華民國政府|中華民國政府]]第十一綏靖區部隊及美軍从[[青岛|青岛]]撤退后，[[中国人民解放军|解放军]]进驻，并将其并入山东管辖。8月12日，[[中国人民解放军第三野战军|第三野战军]]取得[[长山列岛战役|长山列岛战役]]胜利。至此，解放军完全占领[[山東省_(中華民國)|山东省]]全境<ref>{{Cite web |url=http://news.xinhuanet.com/mil/2012-09/19/c_123736149.htm |title=长山列岛战役：解放军大规模两栖登陆第一战 |accessdate=2014-04-20 |author=记者：石其鹏、申红，编辑：王瑶 |date=2012年9月19日 |work= |publisher=新华网，来源：人民网|language=Zh-hans }}</ref>。

1966年8月25日，[[青岛|青岛]]发生了干部、工人群众与青岛三大院校部分学生对立的“青岛事件”，为山东“[[文革|文革]]”拉开了序幕<ref>{{cite web|author1=张兆新|title=“青岛事件”开启山东文革 为造反派夺权埋下伏笔|url=http://arc.dailyqd.com/2015-04/30/content_244061.htm|website=青报网|accessdate=2016-11-30|date=2015-04-30}}</ref>。1980年代以后，山东尤其是东部沿海地区经济有较大发展，在2004年规模工业产值和利润首次超过[[广东|广东]]成为中国第一工业大省。
