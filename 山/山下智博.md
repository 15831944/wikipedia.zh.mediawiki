'''山下智博'''（{{bd|1985年|10月29日}}），在华日本人，[[上海大学|上海大学]][[留学生|留学生]]，中国大陆[[网络红人|网络红人]]。山下智博出生于[[日本|日本]][[北海道|北海道]]，2008年毕业于[[大阪艺术大学|大阪艺术大学]]艺术策划系，之后就职于[[日本|日本]][[札幌市|札幌市]]教育文化会馆，2012年9月起移居[[中华人民共和国|中国]][[上海|上海]]<ref>[https://thepage.jp/detail/20160411-00000005-wordleaf 中国で”山P”に次ぐ人気！ 日本企業も注目のクリエイター山下智博とは？ | THE PAGE（ザ・ページ）]{{ja}}</ref>。

== 经历 ==
2010年，在看到朋友在上海从事艺术工作后决定来中国发展。<ref>[https://dot.asahi.com/dot/2016040600038.html?page=1 中国ネット界で人気すぎる日本人　「山下さん」って誰だ！？ (1/4) 〈dot.〉｜AERA dot. (アエラドット)]{{ja}}</ref>

2012年9月以学习中文为契机来到上海留学。

2013年6月，山下智博开始以与人偶“薇薇”的同居生活为内容表演[[行为艺术|行为艺术]]，并多次带着人偶出现在各大漫展上，从而被中国网友戏称为“从日本来的变态”。<ref name=ifxivscc0282535>[http://news.sina.com.cn/s/wh/2015-10-21/doc-ifxivscc0282535.shtml “日本屌丝”来华记:抱充气娃娃乘地铁(图)|充气娃娃_新浪新闻]</ref>

2014年7月，山下智博开始在弹幕视频网站[[bilibili|bilibili]]上发布以实际生活为素材的自制[[微电影|微电影]]《日本屌丝》，引发网络热议。<ref>{{Cite web|url = http://www.thepaper.cn/newsDetail_forward_1255193|title = 远东志：与充气娃娃同居，“日本屌丝”在中国迎来人生第二春|accessdate = |author = |date = 2014-07-10|publisher = 澎湃新闻|language = zh-cn}}</ref>

2014年12月24日起开始在bilibili上连载个人娱乐节目《绅士大概一分钟》，每天更新一期，节目内容以介绍日本商品、新闻、美食、旅游，日本的调查排行，以及日常生活和搞怪为主。{{R|ifxivscc0282535}}

2015年到上海大学美术学院就读研究生<ref>[http://ent.sina.com.cn/hr/2017-01-17/doc-ifxzqhka3217280.shtml 他是日本老司机 却在中国网红排行榜第38|网红|视频_新浪娱乐_新浪网]</ref><ref>[https://www.rbbtoday.com/article/2016/02/01/139239.html 中国動画サイトで4億回再生の日本人！「山下智博」がすごい理由 | RBB TODAY]{{ja}}</ref>。

2016年10月2日《绅士大概一分钟》宣布停止日更，同年10月14日宣布每周一和周四更新一期。

{{as of|2017|1}}，山下智博在视频网站和[[新浪微博|微博]]上的粉丝共计约200万，视频播放超过5亿次<ref>[http://news.nanyangpost.com/2017/01/4_86.html 中国直播网红催生新经济圈 - 南洋视界]，转载《[[朝日新闻|朝日新闻]]》</ref><ref>[https://dentsu-ho.com/articles/4147 動画再生５億回！中国人が熱狂する日本人「山下智博」って誰？ | ウェブ電通報]{{ja}}</ref>。

== 特色 ==
山下智博在微博中自称为“中日润滑油”，在接受采访时表示“我的很多微博粉丝都想学日语，因为在学校里学到的日语都是非常正式的日语用语，而我当时教的是比较生活化的，粉丝们也非常喜欢，这也是我觉得我第一次起到了‘中日润滑油’的作用，当时就决定要把这个作为我在中国的奋斗的目标了”。<ref>{{Cite web|url = http://www.jpbeta.net/2014/11/nihonloser-2014-1127/|title = 专访“日本屌丝”山下智博，笑与泪与快感的华尔兹|accessdate = |author = |date = 2014-11-28|publisher = JPbeta 多元化日本文化资讯站|language = zh-cn}}</ref>
