'''山下貴司'''（{{bd|1965年|9月8日}}）是[[日本|日本]][[政治家|政治家]]、[[檢察官|檢察官]]、法務官僚、[[律師|律師]]，現為[[自由民主党_(日本)|自由民主党]]（石破派）的[[眾議院_(日本)|眾議院]]議員。曾任法務大臣政務官、内閣府大臣政務官，現任[[法務大臣|法務大臣]]。在進入政界前，他曾在[[特別捜査部|東京地檢特捜部]]<ref>{{Cite news|url = https://www.sankei.com/politics/news/181002/plt1810020017-n1.html |title = 法相に石破派若手の山下貴司氏 全ポスト固まる |newspaper = [[產經新聞|產經新聞]] |date = 2018-10-02 |accessdate = 2018-10-21}}</ref>、[[法務省|法務省]]、[[駐美日本大使館|駐美日本大使館]]等地任職，還擔任過[[慶應義塾大学|慶應義塾大学]]法学部的臨時講師<ref>[https://www.kantei.go.jp/jp/97_abe/meibo/seimukan/yamashita_takashi.html 法務大臣政務官 山下 貴司（やました たかし）│第3次安倍第3次改造内閣 大臣政務官名簿]</ref>。

山下在2018年的自由民主党總裁選舉中支持[[石破茂|石破茂]]<ref>{{Cite news|url = https://www.sankei.com/politics/news/181002/plt1810020077-n1.html |title = 安倍首相が山下貴司法相を抜擢した理由 |newspaper = [[產經新聞|產經新聞]] |date = 2018-10-02 |accessdate = 2018-10-21}}</ref>，隨後卻被[[安倍晋三|安倍晋三]]任命為[[第4次安倍内阁_(改造)|第4次安倍改造内閣]]的法務大臣<ref>{{Cite news|url = https://www.sankei.com/politics/news/181002/plt1810020030-n1.html |title = 法相に石破派の山下氏 第4次安倍改造内閣が発足へ |newspaper = [[產經新聞|產經新聞]] |date = 2018-10-02 |accessdate = 2018-10-21}}</ref>。另外，同時被任命為[[文部科学大臣|文部科学大臣]]的[[柴山昌彦|柴山昌彦]]是其在東京大學時代的同期生<ref>{{Cite news|url = http://www.chunichi.co.jp/article/front/list/CK2018100202000277.html |title = 石破派・山下氏、法相に 安倍改造内閣、初入閣が半数超 |newspaper = [[中日新聞|中日新聞]] |date = 2018-10-02 |accessdate = 2018-10-21}}</ref>。
