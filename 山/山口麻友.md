'''山口麻友'''（{{jpn|j=山口 まゆ|kn=やまぐち まゆ|rm=Yamaguchi Mayu}}，{{bd|2000年|11月20日|||catIdx=Yamaguchi Mayu}}），[[日本|日本]][[女演員|女演員]]。出身於[[日本|日本]][[東京都|東京都]]，所屬經紀公司為[[FLaMme|FLaMme]]，身高157公分<ref name="個人介紹">{{cite web|script-title=ja:經紀公司個人介紹頁面|url=http://www.flamme.co.jp/actress/profile.php?talentid=16|publisher=FLaMme|accessdate=2019-04-14|language=ja}}</ref>。

== 經歷 ==
小學4年級加入經紀公司{{link-ja|NEWS娛樂|NEWSエンターテインメント}}，主要從事舞台劇活動，2011年7月首次演出舞台劇「真夏之夜的夢〜LOVE2011」<ref>{{Cite web|url=http://ameblo.jp/gunjikikaku/entry-10930115169.html|script-title=ja:気になる女優VOL、1|accessdate=2019-04-13|date=2011-06-21|publisher=郡司企画★アプローズカンパニー|language=ja}}</ref>。上國中後因想專心學業而停止演藝活動<ref name="deview">{{Cite web |url=https://deview.co.jp/Interview?am_interview_id=363&set_cookie=2|script-title=ja:「面接で質問に答えるたびに新しい発見があって楽しかった!」/山口まゆ|work=インタビュー / 特集|publisher=Deview|date=2014-08-07|accessdate=2019-04-13|language=ja}}</ref>。

2014年母親幫忙報名了Audition情報誌「{{link-ja|De☆View}}」上的試鏡，合格後收到多家經紀公司邀請，因憧憬演員[[有村架純|有村架純]]而選擇加入經紀公司FLaMme<ref name="deview" />。同年7月首次演出電視劇《晝顏～平日午後3點的戀人們～》<ref name="アイムホーム">{{Cite news|url=https://www.oricon.co.jp/news/2051674/full/|script-title=ja:木村拓哉主演ドラマ『アイムホーム』に抜擢の新人女優・山口まゆ、オーディション秘話を明かす|publisher=[[ORICON_NEWS|ORICON NEWS]]|date=2015-04-16|accessdate=2019-04-13|language=ja}}</ref>。

2015年在電視劇《[[天才婦科醫|天才婦科醫]]》中飾演國中二年級的孕婦而受到矚目<ref name="sponichi20151111">{{Cite news|url= http://www.sponichi.co.jp/entertainment/news/2015/11/11/kiji/K20151111011487970.html |script-title=ja:山口まゆ「コウノドリ」で中２の妊婦役熱演「想像力駆使した」 |newspaper=Sponichi Annex|publisher=體育日本新聞社 |date= 2015-11-11|accessdate= 2019-04-13|language=ja}}</ref><ref name="週プレNEWS">{{Cite news |script-title=ja:ブランニューガールズ連続インタビュー（4） 14歳の妊婦役が超話題になった、若き演技派女優・山口まゆ（15歳）|newspaper=週刊Playboy|publisher=[[集英社|集英社]]|date=2016-01-07 |author=西中賢治|url=http://wpb.shueisha.co.jp/2016/01/07/59107/|accessdate=2019-04-13|language=ja}}</ref>。同年演出電影《[[再會吧！青春小鳥|再會吧！青春小鳥]]》並與演出合唱團中的7名成員<ref group="注">其他6人包括了[[恒松祐里|恒松祐里]]、[[葵若菜|葵若菜]]、[[柴田杏花|柴田杏花]]、朝倉ふゆな、植田日向、高橋奈奈。</ref>一同組成了期間限定團體「Lips!」，發行單曲「壁ドン 〜拝啓、勇気を出せない君へ〜（アカペラver.）」<ref>{{Cite web |date=2015-03-26 |url=http://news.ponycanyon.co.jp/2015/03/6081 |script-title=ja:新垣結衣主演 映画｢くちびるに歌を｣の”可愛すぎる合唱部”「Lips!」が配信デビュー |work=PONY CANYON NEWS|publisher=ポニーキャニオン|accessdate=2019-04-13|archiveurl=https://web.archive.org/web/20160810200454/https://news.ponycanyon.co.jp/2015/03/6081|archivedate=2016-08-10|language=ja}}</ref>。

== 人物 ==
* 興趣和特長是看電影、鋼琴、舞蹈、芭蕾舞。
* 受到母親影響而喜歡古典音樂，4歲時開始學習芭蕾舞、小學1年級時開始學習鋼琴<ref name="週プレNEWS" />。
