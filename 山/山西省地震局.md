'''山西省地震局'''（{{lang-en|Shanxi Earthquake Administration}}），是[[中国地震局|中国地震局]]和[[山西省人民政府|山西省人民政府]]双重领导，以中国地震局为主的事业单位。根据山西省人民政府授权，依法履行防震减灾主管机构的各项职责，承担本行政区域内防震减灾工作政府行政管理职能<ref>{{Cite book|title=中国地震年鉴 1949-1981|last=《中国地震年鉴》编辑部编|first=|publisher=北京：地震出版社|year=1990.03|isbn=7-5028-0295-9|location=|pages=766}}</ref>。

== 沿革 ==
山西是开展地震工作较早的省份。山西省最早的地震观测始于1952年。1967年，组建并成立山西省地震观测站。1968年，设山西省地震队，设太原、大同、临汾，垣曲（后改为夏县）四个地震台，太原为中心台。1970年，山西省地震领导小组与山西省科委地震队合并，成立山西省地震队。1972年，成立山西省革命委员会地震办公室和山西省地震队。1975年，成立'''山西省革命委员会地震局'''。1978年，改为山西省地震局。1984年，调整了机构，设山西省地震监测预报中心，统管六个监测预报研究室和科技资料室<ref>{{Cite book|title=中国地震年鉴 1984|last=《中国地震年鉴》编辑组|first=|publisher=北京：地震出版社|year=1986.01|isbn=|location=|pages=239}}</ref>。

== 职能 ==
山西省地震局职能为：依照《[[中华人民共和国防震减灾法|中华人民共和国防震减灾法]]》、《山西省防震减灾条例》等法规监督、检查全省的防震减灾工作；实施行业管理，指导市、县防震减灾工作，对省级以下地震台网实行统一规划，资源共享；建立健全本省范围内地震监测预报、震灾预防和紧急救援三大工作体系；管理中国地震局移交的地震监测台网；管理主要由省政府投资并主要为全省防震减灾工作服务的一般项目<ref>{{Cite web|url=http://www.shxdzj.gov.cn/n1066/n1071/index.html|title=山西省地震局简介 山西省地震局|accessdate=2018-07-25|work=山西省地震局}}</ref>。

== 事件 ==
2010年2月20-21日，关于山西一些地区要发生地震的消息通过短信、网络等渠道疯狂传播，由于听信“地震”传言，山西太原、晋中、长治、晋城、吕梁、阳泉六地几十个县、市的数百万群众2月20日凌晨开始走上街头“躲避地震”，山西地震局官网一度瘫痪。21日上午，山西省地震局发出公告辟谣<ref>{{Cite book|title=新媒体浪潮|last=黄健编|first=|publisher=南宁：[[广西教育出版社|广西教育出版社]]|year=2011.07|isbn=978-7-5435-6281-3|location=|pages=410}}</ref>。山西省公安机关立即对谣言来源展开调查，后查明造谣者共5人，均被处罚<ref>{{Cite book|title=面向网络舆情的群体性事件的预警机制研究|last=生奇志著|first=|publisher=沈阳：[[东北大学出版社|东北大学出版社]]|year=2014.09|isbn=978-7-5517-0804-3|location=|pages=113-114}}</ref>。2018年4月，山西省地震局官网针对《地震警示，山西》的谣言发布辟谣公告，称文中所说的“未来两个月内中国还将发生7级以上地震，湖北、山西等地为重点”等信息纯属谣言<ref>{{Cite web|url=http://news.sina.com.cn/o/2018-04-15/doc-ifzfkmth3783673.shtml|title=朋友圈热传“山西将发生地震” 省地震局：纯属谣言千万别信|accessdate=2018-07-25|work=新浪网}}</ref><ref>{{Cite web|url=http://news.163.com/18/0413/10/DF924IKF000187V5.html|title=中国将发生7级以上大地震？山西省地震局:纯属谣言|accessdate=2018-07-25|last=网易|work=news.163.com}}</ref>。
