'''山本鈴美香'''{{BD|1949年|6月17日||}}，[[日本|日本]][[漫畫家|女性漫畫家]]。[[山梨縣|山梨縣]][[鹽山市|鹽山市]]出身（今[[甲州市|甲州市]]）出身<!--又或者是「{{link-ja|北巨摩郡|北巨摩郡}}出身」，由於來源不明，故以《漫畫家人名事典》的內容為基礎經過修正。-->。代表作是暨多次改編動畫和電視劇的《[[網球甜心|網球甜心]]》。

== 來歷 ==
1949年出生於[[山梨縣|山梨縣]][[鹽山市|鹽山市]]（現在併入改名[[甲州市|甲州市]]），後搬到[[埼玉縣|埼玉縣]][[浦和市|浦和市]]（現在併入改名[[埼玉市|埼玉市]]）居住成長。{{link-ja|埼玉縣立浦和西高等學校|埼玉県立浦和西高等学校}}畢業。1970年從[[武藏野美術大學|武藏野美術大學]]商業設計科畢業之後<ref name="jiten407">漫畫seek&日外Associates編輯部共同編輯《漫畫家人名事典》，{{link-ja|日外Associates|日外アソシエーツ}}，2003年發行，第407頁。</ref>。翌年1971年以《{{lang|ja|その一言がいえなくて}}…》成為漫畫界的出道作。1973年，開始在[[集英社|集英社]]旗下少女漫畫雜誌《{{link-ja|週刊Margaret|マーガレット (雑誌)}}》連載以[[網球|網球]]為主題的《[[網球甜心|網球甜心]]》，獲得爆發性的人氣。後同名作品連載至1980年長達7年的這段時間多次被改編成動畫或拍成電視劇<ref group="注">漫畫評論家{{link-ja|米澤嘉博|米澤嘉博}}表示，《網球甜心》和《[[凡爾賽玫瑰|凡爾賽玫瑰]]》這兩部作品誕生之後建立了《{{link-ja|週刊Margaret|マーガレット (雑誌)}}》的黃金時期，並經過長達多年之後，依然穩固著[[少女漫畫|少女漫畫]]的代表性作品地位。米澤嘉博著《戰後少女漫畫史》，[[筑摩書房|筑摩書房]]＜{{link-ja|筑摩文庫|ちくま文庫}}＞，2007年，第237頁。</ref>。之後發表另一部長篇連載作品，內容以16世紀的[[歐洲|歐洲]]為故事舞台的《7{{lang|ja|つの黄金郷}}》。

1981年左右與家人陪同搬到鹽山市居住，成為「神山會」[[新興宗教|新興宗教]]的[[巫女|巫女]]<ref name = "jiten407"/>。至今沒有再從事漫畫執筆，而是任神山會[[祖師|教祖]]。
