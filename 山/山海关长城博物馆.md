'''山海关长城博物馆'''，位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[山海关区|山海关区]]第一关路中段，是介绍[[山海关|山海关]]长城历史文化的专题博物馆。

==简介==
山海关长城博物馆始建于1991年。2004年，被河北省列为“[[爱国主义教育基地|爱国主义教育基地]]重点建设项目”进行改陈，总投资620万元人民币，设计及施工由[[清华大学美术学院|清华大学美术学院]]承担，经半年改陈完毕，形成《华夏脊梁》基本陈列。2005年，在[[国家文物局|国家文物局]]主办的[[全国博物馆十大陈列展览精品|全国博物馆十大陈列展览精品]]评选中，山海关长城博物馆基本陈列获中国博物馆、纪念馆最高奖“全国博物馆十大陈列精品评选‘精品奖’”<ref>{{Cite web |url =http://finance.sina.com.cn/roll/20050622/1517145124.shtml  |title = 山海关长城博物馆荣膺“精品奖” |author =  |date =  2005-06-22|publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。2007年起，国家、河北省、秦皇岛市投资4200万元人民币对该馆进行改陈和扩建<ref name=ju>{{Cite web |url =http://www.qhdwhj.gov.cn/front/third.action?id=507&tid=7  |title =山海关长城博物馆  |author =  |date =2015-06-06  |publisher = 秦皇岛市文化广电新闻出版（版权）局 |language =  |accessdate =  |quote =  }}</ref>。2009年7月29日，经两年改扩建后，新馆免费对外开放<ref name=kf>{{Cite web |url =  http://news.163.com/09/0805/17/5FVHH1GH000120GR.html|title =山海关长城博物馆免费开放受欢迎 市民争相参观  |author =  |date = 2009-08-05 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。

新的山海关长城博物馆展览分成《万里长城》和《万里长城——山海关》两部分，含长城历史等8个单元，展出图片图表623幅，文物1100多件（套），雕塑艺术品36件，多媒体9组，模型景观18组，立体电动图表9组。山海关长城博物馆是国家级[[爱国主义教育基地|爱国主义教育基地]]和青少年科普知识普及基地<ref name=kf/>。
