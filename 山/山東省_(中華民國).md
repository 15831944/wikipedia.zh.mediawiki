'''山東省'''，為[[中華民國|中華民國]]延續清朝制度所設置的22省之一，為華北六省之一。簡稱為「'''魯'''」。

== 省名由來 ==
以在[[太行山|太行山]]之東而得名。金代[[大定|大定]]八年（1168年）設置了山東東路、山東西路統軍司，「山東」一詞從此做為一個正式行政區劃的名稱開始使用。

== 管轄範圍 ==
中華民國山東省繼承自清代，其管轄範圍大致与今[[大陸地區|大陸地區]]的[[山東省|山東省]]相同(不含青岛市、東明縣)，及河南省范縣、台前縣。位置東濱[[黃海|黃海]]，西邻[[河南省_(中華民國)|河南省]]，北鄰[[河北省_(中華民國)|河北省]]，南邻[[安徽省_(中華民國)|安徽省]]、[[江蘇省_(中華民國)|江蘇省]]，东南一隅为[[青岛市_(中华民国)|青岛市]]。民國三十六年（1947年），全省土地面積為146 736平方千米。

== 歷史沿革 ==
=== 民國初期 ===
==== 袁氏時期 ====
宣統三年（1911年）爆發[[辛亥革命|辛亥革命]]後，山東省各界聯合會會長[[夏繼泉|夏繼泉]]說服山東巡撫[[孫寶琦|孫寶琦]]宣布山東獨立，任山東都督，後又取消獨立，由[[張廣建|張廣建]]兼署山東巡撫兼提督。11月12日，煙台革命黨人舉行起義成功，成立煙台臨時軍政府。民國元年（1912年）2月，[[胡瑛|胡瑛]]被[[孫中山|孫中山]]任命為山東都督，在煙台組織山東軍政府。胡瑛就任後，將煙台作為山東臨時省府駐地。南北議和成功後，臨時大總統[[袁世凱|袁世凱]]於3月15日下令改原清朝的督撫為都督，原山東巡撫張廣建改任山東都督職，後來改由周自齊擔任山東都督。胡瑛於3月19日辭職，3月25日解散煙台山東軍政府。

民國二年（1913年）1月各省都督改為民政長，周自齊成為山東民政長。8月，[[田文烈|田文烈]]任山東民政長兼軍務會辦。田文烈發揮内政手腕，掃平了省内匪賊，使省内治安得以維持，財務整理和實業振興大有成果。民國三年（1914年）2月11日，[[高景祺|高景祺]]任山東民政長。5月9日，山東民政長高景祺辭職，由[[蔡儒楷|蔡儒楷]]接任。同年5月23日，各省民政長改為巡按使，蔡儒楷成為山東巡按使。9月，陸軍第五師師長[[靳雲鵬|靳雲鵬]]署理山東都督，明年督理山東軍務。民國四年（1915年），靳雲鵬與其他十三省將軍聯名擁護袁世凱稱帝，並獲封一等伯爵。年末，南方將領[[唐繼堯|唐繼堯]]、[[蔡鍔|蔡鍔]]、[[李烈鈞|李烈鈞]]等在雲南宣佈獨立，並出兵討袁。袁世凱的軍隊受挫，南方其他各省之後亦紛紛宣佈獨立。民國五年（1916年）4月他聯合各省通電勸袁辭職，被撤職。

==== 軍閥時期 ====
民國五年（1916年）6月，袁世凱去世後，蔡儒楷被免職，由皖系軍閥[[張懷芝|張懷芝]]任山東督軍，成為皖系控制的地盤。張懷芝於民國七年（1918年）被護理山東督軍[[張樹元|張樹元]]排擠出山東。同年11月張樹元署理山東督軍兼省長。

民國八年（1919年）[[沈銘昌|沈銘昌]]任山東省省長，7月離任，屈映光繼任。11月，[[田中玉|田中玉]]任山東督軍。儘管與山東省長[[屈映光|屈映光]]同屬[[皖系軍閥|皖系軍閥]]，但因權力、地盤之爭而和田中玉對立。起初，第47旅旅長[[馬良_(民國)|馬良]]站在[[屈映光|屈映光]]一邊，田中玉顯劣勢。民國九年（1920年）7月，[[直皖戰争|直皖戰争]]爆發，田中玉未赴主戰場而是留在山東。此後[[直皖戰争|直皖戰争]]皖系敗北，[[屈映光|屈映光]]失勢下野，田中玉掌握山東省實權。勝利的[[直系軍閥|直系軍閥]]因需要田中玉辦事，故他得以留任。

第二次[[直奉戰爭|直奉戰爭]]後，山東成為奉系軍閥勢力範圍內，督軍也都由奉系軍閥充任。民國十三年（1924年），段祺瑞把山東讓給奉系[[張宗昌|張宗昌]]。張被任為山東省軍務督辦，兼省主席。民國十四年（1925年），張宗昌擅自將中央政府直轄的膠澳商埠督辦公署改為山東公署管轄的膠澳商埠局，將青島納入其勢力範圍。

民國十五年（1926年）[[國民革命軍|國民革命軍]][[北伐|北伐]]，[[張作霖|張作霖]]任命張宗昌為「安國軍」副司令，南下援助孫傳芳。民國十六年（1927年）初，張宗昌部在[[南京|南京]]、上海一帶被[[國民革命軍|國民革命軍]]打敗，退回山東。[[寧漢分裂|寧漢分裂]]期間，張宗昌進攻[[馮玉祥|馮玉祥]]軍隊，又大敗。同年6月2日，北伐軍攻克重鎮徐州，直魯軍大部退入山東，江北的孫傳芳部也放棄淮陰，退往山東。民國十七年（1928年），北伐軍入山東省掃蕩奉軍，張宗昌率餘部撤退到河北。

==== 日本入侵青岛和山东 ====
民國三年（1914年），[[第一次世界大戰|第一次世界大戰]]爆發，日軍從膠東半島北部登陸，包圍[[胶州湾租借地|胶澳地区]]（今属[[青岛市|青岛市]]）。經過激戰，最終從德國手中将其奪取。民國八年（1919年）[[巴黎和會|巴黎和會]]對青島的處理方案引發了中國學生的激烈抗議——[[五四運動|五四運動]]。民國十一年（1922年）[[華盛頓會議|華盛頓會議]]後，日本將青島歸還中國。

民國十七年（1928年）5月，[[國民革命軍|國民革命軍]][[北伐|北伐]]期間，途中經過濟南城時，日本方面借口北伐軍對濟南城內的日本僑民進行搶劫、強姦、屠殺，而出動軍隊展開報復，蓄意屠殺中國軍人與民眾六千餘人，中國談判代表[[蔡公時|蔡公時]]慘遭日軍殺害，史稱「[[五三慘案|五三慘案]]」。慘案發生後，日方否認日軍屠殺中國軍民，並要南京國民政府道歉、賠償、懲凶，並於5月11日攻佔濟南。直至次年3月，南京國民政府與日本政府簽訂《中日濟案協定》之後，日軍才退出濟南。

=== 南京十年 ===
==== 蔣馮對立與中原大戰 ====
民國十七年（1928年）5月，[[孫良誠|孫良誠]]為山東省政府第一任主席。孫良誠穩定了對山東省的統治，使受濟南事件及北京政府統治混亂影響的山東省獲得一定的平穩。然而[[馮玉祥|馮玉祥]]因[[蔣中正|蔣中正]]派中央系部隊進駐，而不讓馮系孫良誠部接防，氣憤已極，使孫辭去山東省主席一職。國民政府任命[[石敬亭|石敬亭]]任山東省代主席。

民國十八年（1929年）蔣馮對立升高前夕，馮系[[韓復榘|韓復榘]]率領20師舊部依附[[蔣中正|蔣中正]]，並在[[中原大戰|中原大戰]]期間擔任國民政府第一軍團司令與[[閻錫山|閻錫山]]所率領的[[晉軍|晉軍]]在[[魯|魯]][[豫|豫]]一帶交戰，此役中韓部的表現，被中央委任為山東省主席。

民國十九年（1930年），[[國民政府|國民政府]]設立[[青島|青島]]為院轄市（独立于山东），同年通過外交談判，從英國手中收回[[威海衛|威海衛]]。

==== 韓氏據魯 ====
韓復榘在擔任山東省主席7年中，為了鞏固其勢力，[[暗殺|暗殺]]了曾統治山東的[[張宗昌|張宗昌]]，驅逐[[效忠|效忠]]於[[南京|南京]][[國民政府|國民政府]]的第17軍軍長[[劉珍年|劉珍年]]，另外國民黨山東省黨務管理委員會主任[[張葦村|張葦村]]被暗殺也有可能是其作為；在鞏固實力之餘，卻也曾邀請[[梁漱溟|梁漱溟]]前往山東進行鄉村建設與部分的基礎設施建造。主政山東期間，还曾大力加強基礎文化教育和恢复農業生產，促進了社會發展。韩舊式思想較重，常親自主持司法審判，不依據事實和民國法律，僅對人犯察顏觀色，依個人喜好認定是否有罪和量刑，造成一定數量的冤案。

=== 抗日戰爭 ===
民國二十六年（1937年）－民國三十四年（1945年）的抗日戰爭中，日軍佔領了山東的鐵路和城市，共產黨則在農村建立起沂蒙山、膠東等[[解放區|解放區]]。

==== 國軍方面 ====
[[抗戰|抗戰]]開始後，韓復榘身兼山東省主席、第五戰區副司令、第三集團軍總司令。民國二十六年（1937年）10月，日軍向山東進攻，12月底，韓未聽從[[軍事委員會|軍事委員會]]防禦命令，卻自行放棄黃河防線與[[濟南|濟南]]，率部隊後撤以保存實力，使日軍先後佔領德縣、濟南、曲阜、兗州、青島，打通了膠濟線。山東境內的其他國軍被迫南撤至隴海路附近。韓並暗中與日人談判。蔣中正接受五戰區司令長官[[李宗仁|李宗仁]]以及[[馮玉祥|馮玉祥]]的建議，決定制裁韓復榘。

民國二十七年（1938年）1月11日蔣中正在[[開封|開封]]召開將領會議，要求一、五戰區[[團長|團長]]以上軍官參加。韓復榘到[[開封站|開封火車站]]時即遭逮捕，隨即移送[[漢口|漢口]]軍事監獄審判。經過軍事審判後在1月24日遭[[槍斃|槍斃]]，

3月，日軍第10兵團沿津浦路南下，先後佔濟南、泰安和滕縣，3月24日開始進攻台兒莊。國軍集中優勢兵力，在台兒莊一舉殲滅第10師團及第5師團增援部隊一萬餘人締造[[台兒莊大捷|台兒莊大捷]]，為抗戰初期國軍對日作戰首次獲得較大地的勝利。

民國三十二年（1943年）4月，第２８集團軍[[李仙洲|李仙洲]]率部進軍山東，在魯西、魯南地區對日軍作戰。在李仙洲部隊同日軍作戰時，魯西及魯南地區的八路軍部隊與李仙洲部隊發生衝突。

==== 中共方面 ====
[[File:Luo_Ronghuan.jpg|200px]]]]
民國二十六年（1937年），中共山東省委先後組織[[徂徠山|徂徠山]]、[[天福山|天福山]]、[[黑鐵山|黑鐵山]]、[[泰西|泰西]]等地組織武裝起義，陸續開闢膠東、魯中、魯西、清河、湖西、魯南等小範圍抗日根據地。民國二十七年（1938年），成立中共中央山東分局，[[郭洪濤|郭洪濤]]任黨委書記。山東各起義部隊改編為[[八路軍|八路軍]]山東縱隊，[[張經武|張經武]]、[[王建安|王建安]]任指揮，[[黎玉|黎玉]]為政治委員。

民國二十八年（1939年），[[羅榮桓|羅榮桓]]與[[陳光|陳光]]率[[國民革命軍第十八集團軍第115師|八路軍115師]]師部和主力一部進入山東魯西地區，參與指揮[[樊壩戰鬥|樊壩戰鬥]]、[[梁山戰鬥|梁山戰鬥]]等。同年，山東縱隊正式組建為八路軍第一縱隊，[[徐向前|徐向前]]任司令員，[[朱瑞|朱瑞]]任政治委員。各小型抗日根據地合併為山東抗日根據地。

民國三十二年（1943年）5月，八路軍與國軍李仙洲部隊發生衝突，挫敗李仙洲部隊。下半年，日軍雖然繼續對華北抗日根據地進行掃蕩，但在部分地區已開始實行收縮。共軍抓住時機，向敵人舉行反攻作戰。7月至8月間，魯中區和濱海區八路軍開展了奪取沂魯山區和諸（城）日（照）莒（縣）山區戰役，殲敵3000餘人，拔除據點130餘個，基本上控制了上述地區，把魯中、濱海、膠東各區連繫起來，使山東根據地的抗日戰爭處於更加有利的地位。

民國三十四年（1945年）2月至5月間，又展開以攻取邊緣區和殘存的根據地內的以敵佔城鎮為主要目標的攻勢作戰，晉冀魯豫八路軍山東軍區羅榮桓指揮部隊向魯西、晉南地區發起進攻的夏季攻勢，對日軍發動了猛烈的攻勢，連續發動了討伐偽軍[[吳化文|吳化文]]部戰役、討伐偽軍[[榮子恒|榮子恒]]部戰役及沂水戰役、營縣戰役等15次較大規模的攻勢戰役，殲滅日偽軍6萬餘人，形成了渤海、膠東、魯中、魯南、濱海5個鞏固的根據地。控制山東境內的[[津浦鐵路|津浦鐵路]]、[[膠濟鐵路|膠濟鐵路]]、[[隴海鐵路|隴海鐵路]]，收復除濟南等少數城市之外的山東大部地區，日軍活動範圍被遏制在城市<ref>[http://www.honggushi.com/Article/ymsj/zmzy/200901/4981.html 山東軍區反攻戰役] {{webarchive|url=https://web.archive.org/web/20111212082552/http://www.honggushi.com/Article/ymsj/zmzy/200901/4981.html |date=2011-12-12 }}</ref>。山東軍區部隊也上升至27萬正規軍，佔了當時共產黨軍隊的三分之一。

8月10日，日本政府發出乞降照會。總司令[[朱德|朱德]]發佈大反攻命令，由中共山東軍區主力與基幹部隊編成山東野戰兵團，向日偽軍展開進攻，攻克22座縣城，並逼近濟南、青島、徐州、連雲等大城，山東省境內除鐵路沿線及西部幾座縣城外，均獲攻佔<ref>[http://www.cgw.cn/html/plahistory/149/152/176/2009/0326/4426.shtml 繼續奪取日偽軍控制的中小城鎮山東軍區部隊的作戰行動]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。8月13日，由7個廳局組成的中共山東省政府在莒南縣大店鎮宣告成立。

=== 抗戰勝利後 ===
民國三十四年（1945年）8月抗日戰爭勝利後，中共掌控了山東省大部分地區，並建立了省級政權。[[重慶國民政府|重慶國民政府]]命令國軍沿平綏、同蒲、平漢、津浦等鐵路，向華北推進受降。山東省接收人員則隨同國軍接收掌控青島、濟南等大城市。民國三十六年（1947年）上半年，國共雙方軍隊在[[江蘇省|江蘇省]][[漣水縣|漣水縣]]發生兩次戰役，共軍方面承受了不小的消耗，根據自己「不計一城一地得失，大量殲滅敵人有生力量」的方針，退出江蘇、安徽北部，北撤山東；國軍由陸軍總司令[[顧祝同|顧祝同]]親自指揮六十餘萬國軍，由青島、濟南、臨沂三面進攻山東。由於國軍採用了齊頭並進的戰術，將軍隊控制在一個範圍內，導致共軍的遊擊戰術無法奏效。但[[粟裕|粟裕]]提出以山東共軍主力決戰的方式粉碎圍攻的方式最終被採用。在[[孟良崮戰役|孟良崮戰役]]中，在[[陳毅|陳毅]]的華東野戰軍圍攻下，國軍精銳部隊[[整編七十四師|整編七十四師]]全軍覆沒，師長張靈甫陣亡。6月，國軍繼續自魯南向北進攻，18日，[[胡璉|胡璉]][[整編第11師|整編第11師]]攻下中共山東指揮部所在地魯中南麻鎮，喪失沂蒙山根據地。此時因劉鄧軍在魯西進攻，國軍被迫自魯中回援，陳毅粟裕部隊乘機進攻南麻整11師，但因連降大雨，共軍彈藥受潮，攻擊出現困難，久攻未克。此時國軍再度調集大軍圍攻，陳粟部因多處軍隊有遭遇合圍危險，傷亡較大，因此放棄圍攻，撤至黃河以北惠民縣；國軍在南麻、臨朐兩役勝利後，國軍由范漢傑的指揮下向[[膠東半島|膠東半島]]推進，9月16日，國軍佔領中共膠東軍區司令部所在地萊陽，10月1日攻佔最終目標煙臺，切斷山東和東北的聯繫。10月中旬，國軍基本佔領膠東解放區。蔣中正隨即將膠東兵團逐步調往其他戰區，山東局勢開始扭轉。

同年11月，中華民國國民政府於可掌控的範圍內舉行了[[1947年國民大會代表選舉|第一屆國大代表選舉]]，民國三十七年（1948年）1月又[[1948年中華民國立法委員選舉|選舉]]出第一屆立法委員。同年上半年，共軍在張店、周村、昌樂、濰縣、兗州諸戰役接連勝利，消滅國軍近20萬人，並切斷濟南與青島的交通連繫。9月24日，[[陳毅|陳毅]]的部隊攻佔濟南，消滅國軍10萬餘人，山東省主席[[王耀武|王耀武]]化裝逃往青島途中，在壽光被俘。山東省政府、第十一綏靖區部隊10萬人也奉命撤離，解放軍進駐該城。

至1949年8月，[[中国人民解放军|中国人民解放军]]已经占领除長山八島（即[[长山列岛|长山列岛]]）外的山东省全境。[[中国人民解放军第三野战军|解放军第三野战军]]在8月11日晚发起[[长山列岛战役|长山列岛战役]]，至12日占领长岛全境<ref>{{Cite web |url=http://news.xinhuanet.com/mil/2012-09/19/c_123736149.htm |title=长山列岛战役：解放军大规模两栖登陆第一战  |accessdate=2014-04-20 |author=记者：石其鹏、申红，编辑：王瑶 |date=2012年9月19日 |work= |publisher=新华网，来源：人民网|language=Zh-hans }}</ref>。8月19日，駐守長山八島的海軍撤守該地，當地居民也跟隨國軍撤遷，來到左營落腳。自此，中華民國在山東省的行政機構徹底瓦解。

== 行政區劃 ==
=== 道制 ===
民國元年初，山東行政區劃沿用清末舊制，為3道、10府、3個直隸州、8個府轄州，96縣。至民國二年（1913年）1月，廢除清朝的府、州，但沿用道制，為省、道、縣三級。2月，依據北京政府公布的《劃一現行各道地方行政官廳組織令》，山東由原「濟東泰武臨道」、「兗沂曹濟道」、「登萊青膠道」3道，改為岱北道、岱南道、濟西道、膠東道等4道觀察使，共轄107縣。民國三年（1914年）6月3日，內務部奉袁世凱申令，公佈各省、道區域名稱，將山東的岱北道改稱濟南道、岱南道改稱濟寧道、濟西道改稱東臨道，膠東道依舊。民國十四年（1925年）10月22日，山東督辦兼省長張宗昌重劃為濟南、東昌、泰安、武定、德臨、淄青、萊膠、東海、兗濟、琅琊、曹濮等11道。民國十七年（1928年）4月30日，張宗昌被北伐軍逐出山東，各道被廢除。各道轄區如下：

民國三十四年（1945年），魯西、魯北、膠東三個行署被撤銷，設立魯北、魯西北、魯西、魯南、魯中、魯東6個政府辦事處。與此同時，成立煙臺行政區（隸屬於第七區）、利廣霑棣墾區（隸屬於第五區）、臨冠丘特區（隸屬於第四區）。全省共轄6個政府辦事處。17個行政督察區，107個縣，1個行政區，2個特區。

== 政府机构 ==
=== 省會 ===
山東省會初治歷城縣（今濟南市城區）。濟南五三慘案發生後，民國十七年（1928年）6月遷治泰安縣（今泰安市）。民國十八年（1929年）5月遷回歷城縣，同年7月起治濟南市（今濟南市城區）。抗戰期間一度遷至魯中[[沂水縣|沂水縣]]南麻店，抗戰勝利後遷回濟南市。民國三十七年（1948年）9月濟南失守，省政府寄治于青島市。民國三十八年（1949年）6月，隨著國軍撤離青島而裁撤。
