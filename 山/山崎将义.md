== 音乐生涯 ==
山崎将义从读中学就开始和他当时的的同学组成他们自己的[[乐团|乐团]]，并逐渐开始进行他们乐团各种关于[[音乐|音乐]]的活动。1993年，山崎将义来到东京，并居住在了[[横滨|横滨]]「櫻木町」，起初他只是从开始创作广告歌曲和广告音乐来提高自己的音乐创作水平和增加对音乐知识的了解。之后，在1995年时，他发行了个人单曲《月光照耀》，这张[[单曲|单曲]]也是他开始音乐生涯后他的第一张个人单曲。但是，他个人的第一张单曲并未受到广泛的关注。在之后的一年里，1996年，山崎将义又发行了他个人的第三张单曲《芹菜》。这首单曲《芹菜》后来被[[SMAP|SMAP]][[翻唱|翻唱]]，由此开始受到广泛关注，成为当时的流行歌曲。

山崎将义又在1996年首次涉足[[电影|电影]]的演出，与[[导演|导演]]篠原哲雄、[[演员|演员]]田中要次、真田麻垂美共同合作，而且山崎将义亲自去日本[[群马县|群马县]]拍摄篠原哲雄的电影《月亮与高丽菜》，并在电影中饰演了主角，因为拍摄电影的原因，所以从而暂时停止了自己音乐新的创作工作，在拍摄电影期间，山崎将义为电影《月亮与高丽菜》制作了[[主题曲|主题曲]]《[[One_more_time,_One_more_chance|One more time, One more chance]]》，此首歌曲之后受到了影迷和乐迷的欢迎。从此山崎将义才开始真正的受到了社会各界人士的关注，他的歌曲《One more time, One more chance》后来被[[香港|香港]]知名歌手[[郑中基|郑中基]]翻唱，并且将山崎将义的《One more time, One more chance》更名为《one more time》。山崎将义的歌曲《One more time, One more chance》在当时排到日本国内音乐歌曲排行榜的第29名。1998年，山崎将义和[[松下由树|松下由树]]合作出演了山崎将义主演的第一部电视剧《奇迹之人》，《奇迹之人》的主题曲《这里有我》也是山崎将义本人的一首著名歌曲。

2007年，日本动画作家也是电影导演的[[新海诚|新海诚]]又一次使用山崎将义的名曲《One more time, One more chance》作为[[新海诚|新海诚]]自己最新动画电影《[[秒速五厘米|秒速五厘米]]》的主题曲。并且，山崎将义自己完成了所有的词曲的创作。<ref>[http://www.office-augusta.com/yama/index.html 山崎将义官网]</ref>

== 音乐风格 ==
山崎将义创作的歌曲曲风以民谣摇滚的风格为主。

== 擅长乐器 ==
山崎将义擅长的[[乐器|乐器]]分别是吉他、口琴、钢琴、爵士鼓、合成器。<ref>[http://www.universal-music.co.jp/yamazaki_masayoshi/index.html 山崎将义环球官网]</ref>
