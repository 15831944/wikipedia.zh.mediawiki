'''山陽本線'''（{{jpn|j=山陽本線|hg=さんようほんせん|rm=San'yō honsen|f=y}}）是一條連結[[兵庫縣|兵庫縣]][[神戶市|神戶市]][[中央區_(神戶市)|中央區]]的[[神戶站_(兵庫縣)|神戶站]]<ref name="youran-jrw" /><ref name="kobe-np20171005" />至[[福岡縣|福岡縣]][[北九州市|北九州市]][[門司區|門司區]]的[[門司站|門司站]]，沿[[瀨戶內海|瀨戶內海]]行走的鐵路線（[[幹線|幹線]]）。除本線外，也設有通稱「[[和田岬線|和田岬線]]」的{{link-ja|兵庫站|兵庫駅}}－[[和田岬站|和田岬站]]間的支線。神戶站－[[下關站|下關站]]間與和田岬線由[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）<ref name="youran-jrw" />、下關站－門司站間由[[九州旅客鐵道|九州旅客鐵道]]（JR九州）<ref name="youran-jrk" />管轄。一般略稱山陽線。

另外，廣義上[[山陽新幹線|山陽新幹線]]的[[新神戶站|新神戶站]]至[[小倉站_(福岡縣)|小倉站]]的路段也包括山陽本線，但此條目只記載[[在來線|在來線]]的山陽本線的概要與沿革等。新幹線參見「[[山陽新幹線|山陽新幹線]]」，而在來線的部分地域詳細資料參見以下條目。

* [[JR神戶線|JR神戶線]]（神戶站－姬路站間）
* [[山陽本線_(廣島地區)|山陽本線 (廣島地區)]]（白市站－岩國站間）

== 概要 ==
神戶站為終點的[[東海道本線|東海道本線]]往西延長，由[[神戶市|神戶]]經[[岡山市|岡山]]、[[福山市|福山]]、[[廣島市|廣島]]、[[山口市|山口]]、[[下關市|下關]]等瀨戶内海沿海各都市，直至[[北九州市|北九州]]的[[門司區|門司]]的鐵路線。與東海道本線間有普通、快速、新快速、特急、貨物列車直通運行之故亦統稱「'''東海道・山陽本線'''」。

原來有[[東京|東京]]和[[京阪神|京阪神]]與[[中國地方|中國地方]]、[[九州|九州]]連結的長距離旅客列車運行，但[[山陽新幹線|山陽新幹線]]開業後，長距離旅客輸送的角色交付給新幹線，並行的山陽本線改以地域輸送為中心，不設全線行走的定期旅客列車，倉敷站 - 門司站間沒有特急列車行走。另一方面，[[日本貨物鐵道|日本貨物鐵道]]（JR貨物）的[[貨物列車|貨物列車]]則全區間運行，大多更直通運行至東海道本線。

西半區間有「[[瀨野八|瀨野八]]」之稱的[[瀨野站|瀨野站]] - [[八本松站|八本松站]]間有22.6[[千分比|‰]]的陡坡和曲線半徑300m的急彎存在而成為提速的重大障害。

神戶站－姬路站間與東海道本線的大阪站－神戶站間設有'''[[JR神戶線|JR神戶線]]'''的䁥稱。另外，神戶站－相生站間與和田岬線根據{{link-ja|旅客營業規則|旅客営業規則}}位於「[[大都市近郊區間_(JR)|大阪近郊區間]]」，神戶站－西明石站間位於「{{link-ja|電車特定區間|電車特定区間}}」。電車特定區間成為[[四線鐵路|四線鐵路]]，[[新快速|新快速]]、[[特別急行列車|特急]]、貨物列車主要行走[[電車線與列車線|列車線]]，快速列車與普通電車行走[[電車線與列車線|電車線]]。

一部分路段位於JR西日本的[[IC卡|IC]][[乘車卡|乘車卡]]「[[ICOCA|ICOCA]]」的使用範圍，以及JR九州的IC乘車卡「[[SUGOCA|SUGOCA]]」的使用範圍。當中ICOCA的使用範圍，神戶站－相生站間與和田岬線都位於ICOCA近畿圈地域<ref>[http://www.jr-odekake.net/icoca/area/map/kinki.html ご利用可能エリア 近畿圏エリア｜ICOCA：JRおでかけネット] {{Wayback|url=http://www.jr-odekake.net/icoca/area/map/kinki.html |date=20110228095251 }} - 西日本旅客鉄道</ref>，和氣站－[[三原站_(日本)|三原站]]間位於岡山、廣島地域的岡山、福山地區，三原站－{{link-ja|南岩國站|南岩国駅}}間位於岡山、廣島地域的廣島地區<ref>[http://www.jr-odekake.net/icoca/area/map/okayama.html ご利用可能エリア 岡山・広島エリア｜ICOCA｜ICOCA：JRおでかけネット] {{Wayback|url=http://www.jr-odekake.net/icoca/area/map/okayama.html |date=20110404093246 }} - 西日本旅客鉄道</ref>。但是，近畿圈地域的西端相生站與岡山、廣島地域的東端有和氣站、{{link-ja|有年站|有年駅}}、[[上郡站|上郡站]]、{{link-ja|三石站|三石駅}}、{{link-ja|吉永站|吉永駅}}不位於「ICOCA」的使用範圍，與此同時該路段亦無法使用（即近畿圈地域與岡山、広島地域互相使用）（2018年夏兩地域將統合，有年站、上郡站、三石站、吉永站將可使用<ref name="westjr20171018">[http://www.westjr.co.jp/press/article/2016/01/page_8222.html ICOCAのご利用エリアが大きく広がります！] - 西日本旅客鉄道、2017年10月18日</ref>）。

另外，[[下關站|下關站]]－[[門司站|門司站]]間位於SUGOCA的福岡、佐賀、熊本、大分地域內。但是，ICOCA地域與SUGOCA地域不能使用IC乘車卡互相通行。

線路色方面，JR神戶線區間設定為「藍」，姫路站以西則沒有特別設定。岡山支社為「綠」、廣島支社為「藍」、有各支社獨自設定的線路色。

== 運行形態 ==
與[[東海道本線|東海道本線]]相同，都市間輸送由[[山陽新幹線|山陽新幹線]]負責，運行全線的列車只有[[夜行列車|夜行列車]]。

=== 地域輸送 ===
以[[姬路站|姬路站]]為界線，在界線前後的運輸系統截然不同。姬路站以東的JR神戶線路段因為要與私鐵（[[山陽電氣鐵道|山陽電氣鐵道]]．[[神戶高速鐵道|神戶高速鐵道]]）競爭，所以設有[[新快速|新快速]]列車以作都市間的高速運輸。該等新快速列車在停靠姬路站之後會轉為各站停車，所有從JR神戶線而來的列車均以[[上郡站|上郡站]]作終點站，但其實很多列車會在上郡站之前的[[相生站|相生站]]轉到[[赤穗線|赤穗線]]行走至[[播州赤穗站|播州赤穗站]]作終點站。

== 歷史 ==
=== 概略 ===
神戶站至下關站是由私鐵[[山陽鐵道|山陽鐵道]]開通。1888年兵庫站～明石站間開業、翌年神戶站、龍野站開通。以後路線順次往西延伸，1901年開通至馬關站（現在的下關站）。山陽鐵道於1906年基於[[鐵道國有法|鐵道國有法]]而國有化，成為[[官營鐵道|官營鐵道]]山陽本線。

現在的[[岩德線|岩德線]]於1934年全通。距離比較短的岩德線編入山陽本線，舊有經柳井的線路改為柳井線。但由於現岩德線有不適合當時[[蒸氣機關車|蒸氣機關車]]牽引運行的長大隧道（欽明路隧道，3149m）存在，1944年含柳井線的神戶～下關間全線複線化完成後，柳井線再被編入山陽本線，岩國～周防高森～櫛濱間再成為岩德線。

下關站至九州間於山陽鐵道時代用鐵道連絡船來連接，1942年世界最初的海底隧道[[關門鐵道隧道|關門隧道]]開通，山陽本線延長至門司站。

和田岬線以外的全線電氣化於1964年完成，把同年開通的東海道新幹線連接往西日本各地的[[特別急行列車|特急列車]]、[[急行列車|急行列車]]運行頻繁。其後，[[山陽新幹線|山陽新幹線]]於1972年至岡山站、1975年至博多站開業，山陽本線行走的部份特急被廢止。此後，除了臥舖特快和貨物列車以外，以地域內輸送為主。

2018年7月5日，[[2018年7月西日本豪雨|西日本豪雨]]後，山陽本線[[海田市車站|海田市車站]]至[[三原車站|三原車站]]及[[柳井車站|柳井車站]]至[[德山車站|德山車站]]等路段因土石流入軌道、變電所淹水及路基沖刷流失等災害受損而暫停行駛列車，據JR西日本預估需至少一個月以上才能修復通車<ref>[http://www.westjr.co.jp/press/article/2018/07/page_12665.html 「平成30年7月豪雨」にともなう長期間の運転見合わせについて]</ref>。由於山陽本線是關西地區往來九州地區之間唯一的貨物列車行駛路線，因此造成JR貨物此路段鐵路貨物運輸全面停擺，影響每日三萬噸的運量，JR貨物從7月12日起暫改以海運及公路運輸<ref>[https://newswitch.jp/p/13657 ＪＲ貨物、九州方面１日３万トンに影響。再開まで数カ月]</ref>。8月28日起，暫以經由[[伯備線|伯備線]]、[[山陰本線|山陰本線]]及[[山口線|山口線]]繞道的方式行駛一部份貨物列車。經兩個多月的維修後，於9月30日起全線恢復通車<ref>{{Cite web |url=https://www3.nhk.or.jp/hiroshima-news/20181001/0002491.html |title=ＪＲ山陽線 ３か月ぶり運転再開 |access-date=2018-10-01 |archive-url=https://web.archive.org/web/20181001070421/https://www3.nhk.or.jp/hiroshima-news/20181001/0002491.html |archive-date=2018-10-01 |dead-url=yes }}</ref>。
