'''山梨学院大学附属中学高等学校'''（{{lang-ja|'''山梨学院大学附属中学高等学校'''}}）位于[[日本|日本]][[山梨县|山梨县]][[甲府市|甲府市]][[酒折|酒折]]三丁目，为山梨学院大学的附属学校。

== 历史 ==
1956年（[[昭和|昭和]]31年），山梨学院大学附属中学高等学校创建，1995年（[[平成|平成]]7年），学校增加新校区。

== 学科 ==
*普通科
*英语科

== 社团活动 ==
*体育系：[[剑道|剑道]]、[[台球|台球]]。
*文化系：合唱团、[[围棋|围棋]]、[[将棋|将棋]]、[[茶道|茶道]]。
