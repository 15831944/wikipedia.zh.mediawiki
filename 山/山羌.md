'''山羌'''（学名：{{lang|la|''Muntiacus reevesi''}}、{{lang-en|Reeves's muntjac}}），-{zh-cn:台湾称山羌、羌仔、麂、麂子、小麂、黄麂; zh-tw:又稱羌仔、麂、麂子、黄麂; zh-hk:臺灣称山羌、羌仔、麂、麂子、臺灣小麂;}-。鹿科麂屬的草食性動物， 分布於[[臺灣|臺灣]]與中國[[華中|華中]]、[[華南|華南]]地區。該動物屬於可食用野生動物，為台灣原住民的傳統獵物之一。2019年1月9日起，台灣[[行政院農業委員會|農委會]][[行政院農業委員會林務局|林務局]]將山羌由[[臺灣保育物種列表|保育類]]調整為一般類野生動物，<ref>{{Cite web|url=https://www.forest.gov.tw/0000014/0063328|title=農委會公告修正陸域保育類野生動物名錄 自108年1月9日生效 提醒民眾注意|accessdate=2019-02-13|author=行政院農業委員會林務局|date=2019-01-09|publisher=行政院農業委員會林務局}}</ref>但仍限於地方主管機關所畫定之區域內獵捕。<ref>{{Cite news|url=https://www.chinatimes.com/newspapers/20190110000692-260107|title=保育除名 獼猴、山羌仍不得獵捕|author=莊哲權|date=2019-01-10|work=中時電子報|accessdate=2019-02-13}}</ref>

== 型態 ==
身長不及一米，雄性有短角而雌性僅有骨質隆起，在鹿科動物中公山羌的角並不特化，且保有其他鹿科動物所沒有的上犬齒，靠保護色及靈敏的行動來避敵，雄性額內緣至角基內側各有一黑色條紋，雌性則在額前有盾型班塊，白色的腹部以及尾巴下方的白色也是主要特徵，叫聲似犬吠。<ref>{{Cite web | url=https://scitechvista.nat.gov.tw/c/s2Qg.htm | title=野生動物：森林、山羌、犬吠聲|website=科技部科技大觀園}}</ref>

=== 與赤麂的分別 ===
許多人會將黃麂與[[赤麂|赤麂]]混淆，甚至誤將兩者視為同一種動物，但其實不然。事實上黃麂和赤麂是不同品種，兩者外形很相似，黃麂身形較小，膚色由黃色至棕色都有；赤麂身形較龐大，一般呈紅棕色。在2006年[[香港|香港]][[漁護署|漁護署]]做大型普查，才發現在港的所謂「黃麂」，全部是赤麂，不過現時在《野生動物保護條例》之內，受保護野生動物一項上仍寫黃麂，而非赤麂。<ref>http://hk.myblog.yahoo.com/littlematch2048/article?mid=19237&fid=-1&action=next</ref><ref>{{cite news | url=https://hk.news.appledaily.com/local/daily/article/20080102/10597369 | title=哺乳動物普查發現　港人經常張冠李戴專題報道：赤麂誤當黃麖| date=2008年1月2日}}</ref>

== 分佈 ==
黃麂自然分布於[[臺灣|臺灣]]與中國[[華中|華中]]、[[華南|華南]]地區，另外也有人為引入的族群分布於[[日本|日本]]（[[房總半島|房總半島]]、[[伊豆大島|伊豆大島]]）及[[英國|英國]]（[[德比郡|德比郡]]以南）等地；由低海拔到海拔3,000公尺都有分佈，從闊葉林及水源處也可找到牠們。

有2個亞種：
* 黃麂（''M. reevesi reevesi''），分佈於中國南部。
* 台灣山羌（''M. reevesi micrurus'', 英文名 Formosan Reeve's muntjac），台灣全島從低海拔到三千公尺都有分佈，以闊葉林及水源處常見。此亞種為臺灣特有。和[[臺灣水鹿|水鹿]]及[[臺灣梅花鹿|梅花鹿]]是臺灣僅有的三種原生鹿科動物<ref>{{cite web | title=荷蘭統治時期之村社承包制度 研究成果報告(精簡版)|url=http://ntur.lib.ntu.edu.tw/bitstream/246246/78702/1/952415H002010.pdf |publisher=國立臺灣大學經濟學系暨研究所 | author=吳聰敏 | page=7 | year=2007年}}</ref>。

== 習性 ==
黃麂多數獨居，其領域幾乎不會重疊，於一隻雄性的領域中可能會有三、四隻一齊的雌性。

== 繁殖 ==
黃麂雌性大約五個月才[[性成熟|性成熟]]，開始[[排卵|排卵]]，排卵間隔約14天，每胎生一隻，[[懷孕期|懷孕期]]七至八個月，生產後三至四天便可再受孕，不受哺乳影響，因此牠們全年均可連續繁殖。雌性懷孕會形成集體的大量懷孕週期，幼仔多數同期出生，與一般[[草食性動物|草食性動物]]的年週期繁殖不同。雄性黃麂則約九個月達到性成熟，在換角期間仍具製造[[精子|精子]]的生殖力，不同於一般鹿科動物。
