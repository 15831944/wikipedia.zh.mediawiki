'''山本二三'''（{{bd|1953年|6月27日|catIdx=Yamamoto Nizou}}）是出身於[[長崎縣|長崎縣]]的日本動畫[[美術監督|美術監督]]、導演，及[[京都造型藝術大學|京都造型藝術大學]]的客座教授。<ref>{{cite web |url=http://www.kyoto-art.ac.jp/art/character/character_teacher.html |script-title=ja:キャラクターデザインコース／教員一覧 |work=京都造型藝術大學 |accessdate=2012-10-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20110907075216/http://www.kyoto-art.ac.jp/art/character/character_teacher.html |archivedate=2011-09-07 }}</ref>

對於天空浮雲的描繪具有個人風格，日本動漫界將山本二三繪製的雲朵特別以「二三雲」來稱呼。<ref>{{cite web|url=http://tam-info.blogspot.tw/2012/07/blog-post_18.html |script-title=ja:「二三雲（にぞうぐも）　写真コンテスト」開催決定！  |work=高岡市美術館 |date=2012-07-18 |accessdate=2012-10-19}}</ref>

== 經歷 ==
於東京設計學院畢業後，山本二三進入[[日本動畫公司|日本動畫公司]]。<ref name="mononoke">{{cite book |script-title=ja:もののけ姫　ロマンアルバム |year=1997 |publisher=[[德間書店|德間書店]] |isbn=978-4-19-720026-9 |page=84頁}}</ref>之後山本二三結識了同社的[[高畑勳|高畑勳]]及[[宮崎駿|宮崎駿]]倆人，參與他們的劇場動畫《[[魯邦三世卡里奧斯特羅之城|魯邦三世卡里奧斯特羅之城]]》、或是電視動畫《[[未來少年柯南|未來少年柯南]]》、《[[清秀佳人_(動畫)|清秀佳人]]》等作品。

1985年高畑勳及宮崎駿成立了[[吉卜力工作室|吉卜力工作室]]，山本二三在吉卜力創建後十數年其間；陸續接手了《[[天空之城|天空之城]]》、《[[螢火蟲之墓|螢火蟲之墓]]》、《[[魔法公主|魔法公主]]》等吉卜力動畫作品的美術監督。<ref name="mononoke" />

山本二三個人代表執導動畫為將漫畫家[[小田秀次|小田秀次]]的作品《[[少女的奇幻森林|少女的奇幻森林]]》、於2007年改編發表的同名動畫。而個人的美術展{{nihongo|「日本動畫美術創造者 山本二三展」|日本アニメーション美術の創造者 山本二三展}}則於2011年7月在[[神戶|神戶]]發表。<ref>{{cite web |url=http://kobebiennale.blog.eonet.jp/default/2011/01/post-53f7.html |script-title=ja:せっちゃんを思い出す |work=Kobe Biennale |date=2011-01-13 |accessdate=2012-10-19 |title=存档副本 |archive-url=https://web.archive.org/web/20160304103154/http://kobebiennale.blog.eonet.jp/default/2011/01/post-53f7.html |archive-date=2016-03-04 |dead-url=yes }}</ref>
