{{engname|山东影视传媒集团有限公司|Shandong Film & TV Media Group}}，简称'''山影'''<ref name="a1">{{Cite web |url=http://www.xinhuanet.com//zgjx/2016-03/03/c_135150398.htm|title="山影出品必属精品" 1.39亿元净利润这样"拍"出来|accessdate=2018-10-06 |author= |date=2016-03-03 |work= |publisher=新华网|language=Zh-hans }}</ref>，是一家位于[[山东省|山东省]][[济南市|济南市]]的国有[[传媒|传媒]]集团。以制作多部热门电视剧而知名<ref name="a1"/>。

前身是1986年10月16日成立的'''山东电影电视剧制作中心'''，2007年相关简介称，这是一家集电影、电视剧策划制作销售、音像制品、影视报刊出版发行、电影拷贝洗印生产以及其它影视相关产业于一体的企业<ref>{{Cite web |url=http://ent.sina.com.cn/v/2007-06-28/18441617259.shtml|title=资料：山东电影电视剧制作中心简介|accessdate=2018-10-06 |author=|date=2007-06-28 |work= |publisher=新浪网|language=Zh-hans }}</ref>。

1978年，[[山东电视台|山东电视台]]拍摄山东省第一部电视剧《人民的委托》。1980年，'''山东广播电视艺术团'''成立。1986年，[[山东电影制片厂|山东电影制片厂]]、山东广播电视艺术团合并为'''山东电影电视剧制作中心'''。2008年，[[山东广播电视台|山东广播电视台]]及下属的山东电影电视剧制作中心出资组建'''山东影视集团'''。2012年6月，山东影视集团企业转制，重新组建'''山东影视传媒集团有限公司'''<ref>{{Cite web |url=http://paper.dzwww.com/dzrb/content/20160201/Articel01003MT.htm|title=“鲁剧”为什么这样红|accessdate=2018-10-06 |author=赵琳 |date=2016-02-01 |work= |publisher=大众日报数字报|language=Zh-hans }}</ref>，是山东省管的国有大型文化集团<ref name="a2">{{Cite web |url=http://www.sd-ysjt.com/about_complex.aspx?FId=n1:1:1&cur=0&cur=0|title=集团概况|accessdate=2018-10-06 |author= |date= |work= |publisher=山东影视传媒集团网站|language=Zh-hans }}</ref>。2016年时的报道称，自1982年起，已制作5000多部（集）影视作品<ref name="a1"/>。公司主营影视制作，兼有艺人经纪、电子音像出版、电视节目供片、影视基地开发、文化产业投资等<ref name="a2"/>。
