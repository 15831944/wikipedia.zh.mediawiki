'''山頂醫院總站'''（葡文：'''{{lang|pt|HOSPITAL S. JANUÁRIO/ TERMINAL}}'''、{{lang-en|'''St. Januario Hospital Terminal'''}}）位於[[澳門|澳門]]若憲馬路，鄰近[[山頂醫院|山頂醫院]]的一個終點站，[[澳巴|澳巴]]'''[[澳門巴士H1線|H1]]'''路線以該站為終點站，[[澳巴|澳巴]]'''[[澳門巴士6B路線|6B]]'''、[[澳門新福利|新福利]]'''[[澳門巴士H3路線|H3]]'''路線途經本站。因得勝斜巷行人隧道工程此站在2015年1月31日至2016年7月1日期間暫停使用。

== 沿革 ==
*因山頂醫院地理位置問題，缺乏交通，又沒有專門巴士線到達，在愕斜巷升降機建成前，若乘搭[[澳門巴士6路線|6]]或[[澳門巴士28C路線|28C]]路線巴士，只能在東望洋新街下車，再沿石梯步行前往，相當不便，因此坊間一直希望在醫院範圍增設巴士站，方便居民。

== 歷史 ==
*2006年9月3日，配合[[澳門巴士H1線|H1路線]]投入服務，本站啟用。
*2015年1月31日，因得勝斜巷行人隧道工程，本站暫停使用。
*2016年7月1日，本站恢復使用。
*2018年3月15日至4月8日，因在加思欄後新馬路進行接駁街渠，鋪設水管、電纜及管道工程，本站暫停使用。

== 設施 ==
*候車亭。

== 各車道安排 ==
