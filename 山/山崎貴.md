'''山崎貴'''（{{bd|1964年|6月12日|||Yamazaki Takashi}}），[[日本|日本]]電影[[導演|導演]]，2005年和2014年两次获得[[日本電影金像獎最佳導演獎|日本電影金像獎最佳導演獎]]。

==生平==
13歳時，接觸『[[星際大戰四部曲：曙光乍現|星際大戰四部曲：曙光乍現]]』與『[[第三類接觸_(電影)|第三類接觸 (電影)]]』、決心從事特撮工作。

山崎貴於[[長野縣松本縣丘高等学校|長野縣松本縣丘高等学校]]、[[阿佐佐谷美術専門学校|阿佐佐谷美術専門学校]]畢業。

1986年，山崎貴加入[[白組|株式会社白組]]。

2000年、山崎貴以『[[打機王|打機王]]』出道<ref>{{Cite book|language=ja|date = 2001-04-30|title = 宇宙船YEAR BOOK 2001|series = [[宇宙船_(雑誌)|宇宙船]]別冊|publisher = [[朝日ソノラマ|朝日ソノラマ]] |page = 69 |chapter = 「ジュブナイル」山崎貴監督インタビュー |id = 雑誌コード：01844-04}}</ref>。

2005年、『[[永遠的三丁目的夕陽|永遠的三丁目的夕陽]]』獲得[[報知電影獎|報知電影獎]]、[[電影旬報|電影旬報]]、[[日本電影金像獎|日本電影金像獎]]等獎項。

2013年、設立[[VFX-JAPANアワード|VFX-JAPANアワード]]、『永遠的三丁目的夕陽‘64』獲得劇場公開電影部門、『麻吉妖怪島』劇場動畫電影部門獎<ref>[http://animeanime.jp/article/2013/02/23/13100.html 「三丁目の夕日’64」、もののけ島、巨神兵など　VFX-JAPANアワード受賞 | アニメ！アニメ！]</ref>。

2013年作品《[[永远的0|永远的0]]》获得[[第38届日本电影学院赏|第38届日本电影学院赏]]最佳影片、最佳导演、最佳男主角等八项大奖。

== 人物 ==
山崎貴加入原作版『[[風之谷|風之谷]]』工作<ref>『別冊カドカワ 総力特集「崖の上のポニョ」』2008年、角川書店、61p</ref>。

2012年4月，山崎貴與[[佐藤嗣麻子|佐藤嗣麻子]]結婚<ref>{{Twitter status|satoshimako|201233684907556865|佐藤嗣麻子（@satoshimako）2012年5月12日の発言}}、2012年5月17日参照。</ref><ref>{{Cite web |author= |date=2012-06-01 |url=http://www.yomiuri.co.jp/entertainment/news/20120601-OYT1T00156.htm |title=映画監督の山崎貴さんと佐藤嗣麻子さんが結婚 |work=[[YOMIURI_ONLINE|YOMIURI ONLINE]] |publisher=[[読売新聞社|読売新聞社]] |accessdate=2012-06-03}}</ref>。
