'''山下美月'''（{{jpn|j=山下 美月|hg=やました みづき|rm=Yamashita Mizuki}}，{{bd|1999年|7月26日}}）是[[日本偶像|日本偶像藝人]]，為女子偶像團體[[乃木坂46|乃木坂46]]成員，[[東京都|東京都]]出身<ref name="3ki">{{Cite book|author=筒井優太|author2=林将勝|editor=西中賢治|script-title=ja:乃木坂46×週刊プレイボーイ2016|date=2016-11-24|publisher=集英社|page=1-5|chapter=別冊Book in Book 3期生名鑑「誌上お見立て会」開催!|language=ja}}</ref>。目前亦是潮流雜誌《[[CanCam|CanCam]]》的專屬模特兒。

== 簡歷 ==
山下美月在1999年7月26日出生於東京都<ref name="3ki" />，其名字由祖父命名<ref name="blog20170403" />，意為「出生在一個有美麗月亮的晚上」<ref name="blog20170403">{{Cite web|url=http://blog.nogizaka46.com/third/2017/04/037826.php|script-title=ja:届けたいもの( ˙꒳˙ ) 山下美月|accessdate=2018-03-14|author=山下美月|date=2017-04-03|work=乃木坂46 3期生 公式ブログ|publisher=乃木坂46LLC|language=ja|archive-url=https://web.archive.org/web/20180317233511/http://blog.nogizaka46.com/third/2017/04/037826.php|archive-date=2018-03-17|dead-url=no}}</ref>。高中時，山下曾經加入茶道部，並曾任該部的副部長<ref name="blog20161228" />。因為看見[[早安家族|早安家族]]旗下偶像團體[[Buono!|Buono!]]的演出，使山下渴望成為偶像<ref name="otokake">{{cite web|url=https://otokake.com/matome/wkMOz1|script-title=ja:山下美月（乃木坂46）の胸チラにファン大興奮！！気になる○○舐め写真の真相とは？！【画像あり】|date=2018-10-30|archiveurl=https://web.archive.org/web/20180317232733/https://otokake.com/matome/wkMOz1|archivedate=2018-03-17|dead-url=yes|language=ja}}</ref>。在加入乃木坂46之前，山下曾經在[[LesPros娛樂|LesPros娛樂]]旗下事務所LesPros Aster從事演藝活動<ref name="otokake" />，亦曾參與[[櫸坂46|櫸坂46]]的甄選，但在筆試階段落選<ref name="otokake" />。

== 人物 ==
山下的暱稱是「{{lang|ja|づっきー}}」<ref name="3ki" />、「{{lang|ja|みづき}}」<ref name="cc" />和「{{lang|ja|みーちゃん}}」<ref name="zine">{{cite web|url=https://web-zine.net/887/2|script-title=ja:大園桃子×山下美月×伊藤理々杏-金の卵の素顔:3期生は日に日に成長していく(2/2)|publisher=WebZine|date=2017-01-27|language=ja|access-date=2018-11-03|archive-url=https://web.archive.org/web/20180812150343/https://web-zine.net/887/2|archive-date=2018-08-12|dead-url=no}}</ref>，而同屬乃木坂46的[[堀未央奈|堀未央奈]]則稱其為「{{lang|ja|ムーン}}」<ref name="zine2">{{cite web|url=https://web-zine.net/724|script-title=ja:乃木坂46 堀未央奈×山下美月-HONEY MOON(1/2)|publisher=WebZine|date=2017-04-15|language=ja|access-date=2018-11-03|archive-url=https://web.archive.org/web/20180627202648/https://web-zine.net/724|archive-date=2018-06-27|dead-url=no}}</ref>。她是家中獨女<ref name="blog20170322">{{Cite web|url=http://blog.nogizaka46.com/third/2017/03/037530.php|script-title=ja:エビフライ( ˙꒳˙ ) 山下美月|accessdate=2018-03-13|author=山下美月|date=2017-03-22|work=乃木坂46 3期生 公式ブログ|publisher=乃木坂46LLC|language=ja|archive-url=https://web.archive.org/web/20180317233432/http://blog.nogizaka46.com/third/2017/03/037530.php|archive-date=2018-03-17|dead-url=no}}</ref>。

山下有過打高爾夫球、打網球、打排球和跳舞的經驗<ref>{{Cite web|url=http://realsound.jp/2017/04/post-11916.html|script-title=ja:乃木坂46 生田絵梨花のプレゼンにバナナマン日村も苦笑「接待感がすご過ぎる」|accessdate=2017-04-03|author=向原康太|date=2017-04-03|work=Real Sound|publisher=blueprint|language=ja|archive-url=https://web.archive.org/web/20170404152629/http://realsound.jp/2017/04/post-11916.html|archive-date=2017-04-04|dead-url=no}}</ref>，而她所憧憬的人物是[[藥師丸博子|藥師丸博子]]<ref name="otokake2" />。

山下在乃木坂46是衍生企劃組合「若樣軍團」的團員，為該團的勺子擔當<ref name="wakasama1" /><ref name="wakasama2" />，其應援顏色則為藍色和黃色<ref name="blog20170121">{{Cite web|url=http://blog.nogizaka46.com/third/2017/01/036588.php|script-title=ja:ぶるぶる( ˙꒳˙ ) 山下美月|accessdate=2017-01-22|author=山下美月|date=2017-01-21|work=乃木坂46 ３期生 公式ブログ|publisher=乃木坂46LLC|language=ja|archive-url=https://web.archive.org/web/20180317233047/http://blog.nogizaka46.com/third/2017/01/036588.php|archive-date=2018-03-17|dead-url=no}}</ref>。她在乃木坂46憧憬的前輩是[[西野七瀨|西野七瀨]]<ref>{{cite web|url=https://woman.infoseek.co.jp/news/entertainment/cyzo_20170227_937281|script-title=ja:乃木坂46“未来のエース”山下美月に飛び交う「前田敦子似」の声は、吉か凶か？|date=2017-02-27|publisher=infoseek WOMAN|accessdate=2018-03-17|language=ja|archive-url=https://web.archive.org/web/20180317234024/https://woman.infoseek.co.jp/news/entertainment/cyzo_20170227_937281|archive-date=2018-03-17|dead-url=no}}</ref>和[[櫻井玲香|櫻井玲香]]<ref name="rs20180212">{{Cite web|url=http://realsound.jp/2018/02/post-158337_2.html|script-title=ja:乃木坂46 山下美月の“小悪魔っぷり”で「若様軍団」解散の危機に？　若月佑美「ショックです」|accessdate=2018-03-16|author=向原康太|date=2018-02-12|work=Real Sound|publisher=blueprint|language=ja|archive-url=https://web.archive.org/web/20180317232123/http://realsound.jp/2018/02/post-158337_2.html|archive-date=2018-03-17|dead-url=no}}</ref>。山下覺得櫻井是她最憧憬前輩的原因在於她是「在背後對你說話，但不發一言的隊長」<ref name="rs20180212" />。
