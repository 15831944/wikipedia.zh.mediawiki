'''山楂'''，[[英文名|英文名]]：''Hawthorn''，[[学名|学名]]：''{{lang|la|Crataegus}}''，又称'''山裡红'''、'''大紅果'''、'''山楂肉'''、'''酸楂'''、'''赤棗'''、'''山楂炭'''、'''生山楂'''、'''焦山楂'''、'''山櫨'''、'''仙楂'''、'''仙查'''、'''山查'''，是[[蔷薇科|蔷薇科]][[苹果亚科|苹果亚科]]的一个[[属_(生物)|属]]，全世界有约200个[[种_(生物)|种]]。其学名''Crataegus''来自[[希腊语|希腊语]]中的''kratos''（硬的）。

== 特征 ==
常绿[[乔木|乔木]]。广卵形或三角状卵形叶子，羽状5～9裂，叶脉上有短柔毛。

山楂需要充沛的阳光，一般生长于农田边缘、灌木林或者阔叶林中。山楂可以达到数米高，可以活500多年。5月和6月山楂一般开白花，少数山楂开红花，花气味很浓。果实大多数是红色的。山楂的枝分叉非常多，往往枝上有刺。木头非常硬。

过去果农有将[[梨|梨]]嫁接在山楂上的做法，但由于山楂容易患火疫病，这个做法现在已被放弃。

== 用途 ==
===食用===
山楂的成熟果实可生用或炒黄焦用入药。山楂果实被广泛用于制造[[糖葫芦|糖葫芦]]、[[果丹皮|果丹皮]]、[[山楂饼|山楂饼]]、[[山楂糕|山楂糕]]等酸甜食物。
加於[洛神花]熬煮成茶飲風味佳。

===药用===
{{medical}}
在[[中国|中国]]等地山楂是药果兼用树种。

山楂主要的药物成分是[[大分子单宁|大分子单宁]]，其它药物成分有[[蔓荆子|蔓荆子]]、牡荆素-2”-鼠李糖甙等。

用於擴張冠狀血管、降低血壓和血膽固醇及恢復心肌壁。減少脂肪量的儲存，增加細胞內維生素C的含量。對於貧血、心血管和循環疾病、高血膽固醇和降低免疫力的疾病有幫助。

== 歷史 ==
關於山楂有許多傳說。由於過去歐洲人認為山楂可以阻擋惡魔和邪惡的魔鬼，山楂往往被種在院子和田野的邊上作為屏障。

在歐洲第一位紀錄山楂的藥用作用的是1世紀時期的[[迪奧斯克裡德斯|迪奧斯克裡德斯]]。在中國山楂在中醫中有使用。在[[美洲|美洲]]當地的[[印第安人|印第安人]]也知道山楂的藥用作用。

相傳英國[[理查三世_(英格蘭)|理查三世]]和後來的[[亨利七世|亨利七世]]在Boswolds決戰時，英王的[[王冠|王冠]]被此樹勾起。

[[紅背伯勞鳥|紅背伯勞鳥]]會將它捕捉到的[[昆蟲|昆蟲]]刺在山楂的刺上作為儲藏，因此它在德文中獲得了“九殺”（Neuntöter）的名字。

[[蘇聯|蘇聯]]1950年代[[流行歌曲|流行歌曲]]《[[山楂樹|山楂樹]]》講述的是一個[[三角戀|三角戀]]故事：兩名愛上同一名女子的男青年在同一棵山楂樹下等待戀人，女子感到難以抉擇。
