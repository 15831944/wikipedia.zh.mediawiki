'''山尾志櫻里'''（{{bd|1974年|7月24日|catIdx=Yamao}}）為[[日本|日本]][[政治家|政治家]]，曾任[[檢察官|檢察官]]。現為[[立憲民主党_(日本)|立憲民主党]]的[[眾議院_(日本)|眾議院]]議員，曾在[[民進党_(日本)|民進党]]中擔任政務調査会長、国民運動局長。旧姓'''菅野'''。

==經歷==
1974年7月24日出生於[[宮城縣|宮城縣]][[仙台市|仙台市]]<ref>{{Cite web|url=https://www.yomiuri.co.jp/election/shugiin/2012/profile/yc87018034.htm|script-title=ja:山尾志桜里 : 東海比例　民主党 : プロフィル : 衆院選2012 : 衆院選 :  選挙 : YOMIURI ONLINE（読売新聞）|accessdate=2018-11-21|publisher=[[讀賣新聞|讀賣新聞]]|language=ja}}</ref>。從東京大學法學部畢業後，她於2002年通過司法考試<ref>{{Cite web|url=https://www.yamaoshiori.jp/profile.html|script-title=ja:山尾しおり(山尾志桜里)：衆議院議員/愛知県第７区|accessdate=2018-11-21|work=山尾志櫻里|language=ja}}</ref>，其後先後在東京地方檢察廳、千葉地方檢察廳、名古屋地方檢察廳等地工作。她於2009年投入政界，代表[[民主党_(日本1998年)|民主黨]]出戰[[第45屆日本眾議院議員總選舉|第45屆日本眾議院議員總選舉]]並在[[愛知縣第7區|愛知縣第7區]]獲得壓倒性勝利，但她在2012年的[[第46屆日本眾議院議員總選舉|第46屆日本眾議院議員總選舉]]中未能成功連任<ref>{{Cite web|url=https://www.yomiuri.co.jp/election/shugiin/2012/kaihyou/yc87.htm|script-title=ja:東海　【比例代表】 : 開票結果 : 総選挙2012 : 衆院選 :  選挙 : YOMIURI ONLINE（読売新聞）|accessdate=2018-11-21|publisher=讀賣新聞|language=ja}}</ref>。及後在2014年的[[第47屆日本眾議院議員總選舉|第47屆日本眾議院議員總選舉]]中，她再度當選該選區眾議員重返眾議院<ref>{{Cite web|url=http://www.asahi.com/senkyo/sousenkyo47/kaihyo/A23.html#Area007|script-title=ja:愛知 - 開票速報 - 2014衆院選：朝日新聞デジタル|accessdate=2018-11-21|work=[[朝日新聞|朝日新聞電子版]]|language=ja}}</ref>。

在2010年和2015年的民主党代表選舉期間，她曾先後作為[[菅直人|菅直人]]和[[細野豪志|細野豪志]]的聯名推薦人支持二人<ref>{{Cite news|url=http://www.jiji.com/jc/zc?k=201501/2015010700415&g=pol|script-title=ja:時事ドットコム：民主代表選候補の推薦人名簿|date=2015-01-07|agency=[[時事通信社|時事通信]]|accessdate=2018-11-21|archive-url=https://archive.fo/20150107162414/http://www.jiji.com/jc/zc?k=201501/2015010700415&g=pol|archive-date=2015-01-07}}</ref>。2016年[[民进党_(日本)|民進黨]]成立後，她因在國會提出「待機兒童問題」而受到矚目<ref name=":0">{{Cite news|url=https://www.sankei.com/premium/news/171125/prm1711250011-n2.html|script-title=ja:【政界徒然草】後援会長もあきれた！　「不倫」報道相手を顧問にした山尾志桜里氏の逃避行　説明責任はいずこに？|publisher=[[產經新聞|產經新聞]]|date=2017-11-25|accessdate=2018-11-22|language=ja}}</ref>，其後出任民進黨政調会長<ref>{{Cite news|url=http://mainichi.jp/articles/20160329/ddm/002/010/065000c|script-title=ja:待機児童問題：「政府、子供に金出さず」　民進党・山尾政調会長 - 毎日新聞|agency=[[毎日新聞|毎日新聞]]|accessdate=2018-11-21|language=ja}}</ref>。同年，《{{link-ja|週刊新潮|週刊新潮}}》報道她在2012年擔任「民主党愛知縣第7区總支部」支部長時，該支部在一年中所花的汽油費高達230萬日圓，而正常駕駛情況下用這些油可繞地球5圈<ref>{{Cite news|url=https://www.sankei.com/politics/news/160401/plt1604010008-n1.html|script-title=ja:なんと地球５周分！？　民進・山尾志桜里政調会長、驚愕の“ガソリン代疑惑”|publisher=[[產經新聞|產經新聞]]|accessdate=2018-11-22|language=ja}}</ref>。年底，她在[[名古屋市|名古屋]]的記者會上澄清，這些油費是當時的一名秘書為了虛增報銷費用，用在加油站撿到的廢棄賬單等進行申請的。該秘書目前已辭職並還清費用，因此她本人並不打算提起刑事告訴<ref>{{Cite news|url=https://www.sankei.com/west/news/161227/wst1612270053-n1.html|script-title=ja:【民進・山尾氏疑惑】元秘書が不正認める、ガソリン代２１７万円返還　陳謝「政治活動に使われた事実ない」|publisher=產經WEST|accessdate=2018-11-22|language=ja}}</ref>。

2017年，她一度被民進黨代表[[前原誠司|前原誠司]]內定為幹事長人選，但因被9月7日發行的《[[週刊文春|週刊文春]]》爆出與比她年輕9歲的已婚律師倉持麟太郎有外遇而被取消內定<ref>{{cite news|script-title=ja:民進党新執行部が発足　「山尾志桜里幹事長」は週刊誌の既婚男性との交際疑惑取材で断念|url=http://www.sankei.com/politics/news/170905/plt1709050046-n1.html|accessdate=2018/11/22|publisher=[[產經新聞|產經新聞]]}}</ref>。她於同一天退出民進黨，並在離黨記者會上明確否定二人之間有男女關係<ref name=":0" /><ref>{{Cite news|url=https://www.huffingtonpost.jp/2017/09/07/yamao_a_23200093/|script-title=ja:山尾志桜里氏が離党届を提出「子育てに奮闘するお母さんに申し訳ない」 記者会見で謝罪（会見全文）|date=2017-09-07|newspaper=[[哈芬登郵報|哈芬登郵報]]|accessdate=2018-11-21|language=ja}}</ref>。在同年舉行的[[第48屆日本眾議院議員總選舉|第48屆日本眾議院議員總選舉]]中，她以[[無黨籍|無黨籍]]身分參選，並以不到1000票的微弱優勢當選<ref>{{Cite news|url=http://www.asahi.com/senkyo/senkyo2017/kaihyo/A23.html#Area007|script-title=ja:愛知-開票速報-2017衆議院選挙（衆院選）：朝日新聞デジタル|agency=朝日新聞電子版|accessdate=2018-11-21|language=ja}}</ref>。當選後，她啟用倉持律師擔任其「政策顧問」，再次引發爭議<ref name=":0" />。2017年12月，她與[[蓮舫|蓮舫]]等人加入[[立憲民主黨_(日本)|立憲民主黨]]<ref>{{Cite news|url=https://www.sankei.com/photo/story/news/171226/sty1712260019-n1.html|script-title=ja:蓮舫氏、立民へ入党届　山尾志桜里氏ら４人は決定|newspaper=産経フォト|accessdate=2018-11-21|language=ja}}</ref>。

漫畫家[[小林善紀|小林善紀]]為她的粉絲，曾在《週刊文春》的外遇報道之後在個人[[部落格|部落格]]發佈撰寫該篇報道的記者的姓名、照片等內容<ref>{{Cite news|url=https://www.sankei.com/politics/news/171120/plt1711200014-n1.html|script-title=ja:山尾志桜里氏、疑惑弁護士と大阪出張の文春砲　女性セブンでは望月記者と大放談、不倫疑惑をすり替え|date=2017-11-21|agency=產經新聞|accessdate=2018-11-21|language=ja}}</ref>。
