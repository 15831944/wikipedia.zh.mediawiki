'''山药泥'''是一种以[[日本薯蓣|日本薯蓣]]（细叶野山药）或[[薯蓣|薯蓣]]（山药）的块茎为原料，磨碎制成的食品。在日本常添加[[出汁|出汁]]、[[山葵|山葵]]酱、[[洋葱|洋葱]]等调味。山药泥常作为[[便当|便当]]的配菜、或者添加到[[乌冬面|乌冬面]]、[[荞麦面|荞麦面]]中、或配合[[刺身|刺身]]、[[豆腐|豆腐]]食用。<ref name=shiki_aki>『四季日本の料理 秋』講談社 ISBN 4-06-267453-X</ref><ref name=shun_akifuyu>『旬の食材 秋・冬の野菜』講談社 ISBN 4-06-270136-7</ref>
== 组合 ==
*{{tsl|ja|麦とろご飯|山药泥麦饭}}是在{{tsl|ja|麦飯|麦饭}}上添加山药泥制成的食物，常添加[[出汁|出汁]]、[[清酒|清酒]]、[[味醂|味醂]]、[[酱油|酱油]]、白[[味噌|味噌]]、鸡蛋的混合酱汁，再加上[[葱|葱]]、[[青海苔|青海苔]]一起食用。是[[静冈县|静冈县]]的代表食物。
*山掛（{{lang-ja|山掛け}}）指和山药泥混合制成的食物，常指切成方块的[[金枪鱼|金枪鱼]]和山药泥混合后的食物。
== 特性 ==
山药泥具有特有的粘性，是由于山药中的粘性物质在磨碎后被释放到细胞外，溶于水后生成的。这些粘性物质一般认为是[[多糖|多糖]]类和[[蛋白质|蛋白质]]<ref name="sato1">{{Cite journal|author1=佐藤利夫 |author2=水口純 |author3=鈴木周一 |author4=戸倉正利 |year=1967 |title=イチョウイモ粘質物の精製および性質 |journal=日本化学会誌 |volume=88 |issue=2 |page=106 |publisher=[[日本化学会|日本化学会]]}}</ref><ref name="sato2">{{Cite journal|author=佐藤利夫 |year=1967 |title=イチョウイモ粘質物の組成と構造（とくにマンナンの性状） |journal=日本化学会誌 |volume=88 |issue=9 |page=84 |publisher=[[日本化学会|日本化学会]]}}</ref><ref name="sato3">{{Cite journal|author=佐藤利夫 |year=1967 |title=イチョウイモ粘質物に対する各種酵素の作用 |journal=日本化学会誌 |volume=88 |issue=9 |page=91 |publisher=[[日本化学会|日本化学会]]}}</ref>，但这种物质的构造和性状尚未得到充分的解释。<ref name="tsukui">{{Cite journal|author=津久井学 |date=2003-07 |title=ヤマイモ粘質物の性状と構造の解析 |url=http://agriknowledge.affrc.go.jp/RN/2010680217.pdf |format=pdf |journal=日本食品保蔵科学会誌 |volume=29 |issue=4 |pages=229-236 |publisher=日本食品保蔵科学会 |issn=13441213}}</ref>

山药泥含有大小为100微米左右的[[草酸钙|草酸钙]]的针状结晶，接触到皮肤可能会引起瘙痒。<ref name="asahi">{{cite web|url=http://www.asahi.com/edu/nie/tamate/kiji/TKY200504110161.html |title=ヤマイモはなぜかゆいの？ |publisher=朝日新聞社 |accessdate=2018-05-29}}</ref>。
