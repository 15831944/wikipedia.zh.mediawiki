'''山葡萄'''（[[学名|学名]]：{{lang|la|''Vitis amurensis''}}）为[[葡萄科|葡萄科]][[葡萄属|葡萄属]]的植物。

== 形态 ==
落叶木质[[藤本|藤本]]。小枝嫩时疏生蛛丝状茸毛，有卷须；宽卵形大叶，3～5浅裂，有浅三角形齿牙，基部深心形，平滑或脉上有毛；夏季开花，雌雄异株，圆锥花序；黑色果实。

== 分布 ==
分布于[[中国大陆|中国大陆]]的[[黑龙江|黑龙江]]、[[安徽|安徽]]、[[辽宁|辽宁]]、[[浙江|浙江]]、[[吉林|吉林]]、[[山西|山西]]、[[山东|山东]]、[[河北|河北]]等地，生长于海拔200米至1,200米的地区，多生在山坡、沟谷林中及灌丛中，目前尚未由人工引种栽培。

== 别名 ==
阿穆尔葡萄（江苏南部种子植物手册）

== 异名 ==
* Vitis amurensis Rupr. f. hermaphrodita J. Y. Li
