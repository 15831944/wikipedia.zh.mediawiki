'''山-{里}-車站'''位於[[台灣|台灣]][[臺東縣|臺東縣]][[卑南鄉|卑南鄉]]嘉豐村山-{里}-部落，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。配合[[花東鐵路電氣化|花東鐵路電氣化]]進行改建，改建工程於2013年8月正式完工。

== 車站構造 ==
*[[岸式月台|岸式月台]]一座。
*本站無天橋及地下道，乘客於第一月台乘車。

== 利用狀況 ==
*目前因應例休試辦比照乙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，僅留站員1人值班。
*停靠狀態：僅停靠[[區間車|區間車]]。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站無發售車票之業務。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1982年6月27日：因[[臺東線|臺東線]]鐵路拓寬改線時設站。
*2004年4月29日：山-{里}-站以北號誌自動化系統啟用。
*2015年7月28日：配合仲夏寶島號停靠本站，開賣紀念月台票。<ref>{{cite news |language = zh-hant |url = https://tw.news.yahoo.com/%E5%82%B3%E8%AA%AA%E4%B8%AD%E5%88%B0%E4%B8%8D%E4%BA%86%E7%9A%84%E8%BB%8A%E7%AB%99-%E5%B1%B1%E9%87%8C%E7%AB%99%E9%A6%96%E8%B3%A3%E7%B4%80%E5%BF%B5%E6%9C%88%E5%8F%B0%E7%A5%A8-090138265.html |title = 「傳說中到不了的車站」 山里站首賣紀念月台票 |date = 2015-07-28 |accessdate = 2015-08-07}}</ref>
*2016年6月28日：啟用多卡通刷卡機。
*2017年9月16日：本站試辦比照乙種[[簡易站|簡易站]]運轉，並指定由[[臺東車站|臺東站]]管理。
