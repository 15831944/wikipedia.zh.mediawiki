'''山东半岛蓝色经济区'''范围包括[[山东|山东]]全部海域和[[青岛|青岛]]、[[东营|东营]]、[[烟台|烟台]]、[[潍坊|潍坊]]、[[威海|威海]]、[[日照|日照]]6市及[[滨州市|滨州市]]的[[无棣|无棣]]、[[沾化|沾化]]2个沿海县所属陆域，海域面积15.95万平方公里，陆域面积6.4万平方公里。2009年,区内总人口3291.8万人，人均地区生产总值50138元<ref>{{ cite web |url = http://www.sdlb.gov.cn/art/2011/3/2/art_536_29131.html |title = 山东半岛蓝色经济区区域范围 |publisher = 山东半岛蓝色经济区建设办公室 |date = 2011-03-02 |language = zh-cn |deadurl = yes |archiveurl = https://web.archive.org/web/20130511090817/http://www.sdlb.gov.cn/art/2011/3/2/art_536_29131.html |archivedate = 2013-05-11 }}</ref>。该[[经济区|经济区]]位于中国最大的[[半岛|半岛]]——[[山东半岛|山东半岛]]，是[[环渤海地区|环渤海地区]]与[[长江三角洲地区|长江三角洲地区]]的重要接合部、[[黄河流域|黄河流域]]地区最便捷的出海通道、[[东北亚|东北亚经济圈]]的重要组成部分。

== 确立过程 ==
2011年1月4日，国务院批复《山东半岛蓝色经济区发展规划》，这是“十二五”开局之年第一个获批的国家发展战略，也是国内第一个以海洋经济为主题的区域发展战略。《规划》的批复实施，是中国区域发展从陆域经济延伸到海洋经济、积极推进陆海统筹的重大战略举措，标志着全国海洋经济发展试点工作进入实施阶段，也标志着山东半岛蓝色经济区建设正式上升为国家发展战略，成为国家海洋发展战略和区域协调发展战略的重要组成部分。

== 地理区域 ==
山东半岛蓝色经济区以[[青岛|青岛]]为中心，以[[山东半岛|山东半岛]]地区（包括[[青岛|青岛]]、[[烟台|烟台]]、[[威海|威海]]、[[日照|日照]]、[[潍坊|潍坊]]）为中心区域，向西扩展延伸至[[黄河三角洲|黄河三角洲]]冲积平原的[[东营|东营]]和[[滨州|滨州]]的部分地区。整个规划范围包括[[山东|山东]]全部海域和[[青岛|青岛]]、[[东营|东营]]、[[烟台|烟台]]、[[潍坊|潍坊]]、[[威海|威海]]、[[日照|日照]]6市及[[滨州市|滨州市]]的[[无棣|无棣]]、[[沾化|沾化]]2个沿海县所属陆域，海域面积15.95万平方公里，陆域面积6.4万平方公里。

==发展规划==
=== 战略定位与规划 ===
建设具有较强国际竞争力的现代海洋产业集聚区、具有世界先进水平的海洋科技教育核心区、国家海洋经济改革开放先行区和全国重要的海洋生态文明示范区。到2015年，现代海洋产业体系基本建立，综合经济实力显著增强，海洋科技自主创新能力大幅提升，海陆生态环境质量明显改善，海洋经济对外开放格局不断完善，率先达到全面建设小康社会的总体要求；到2020年，建成海洋经济发达、产业结构优化、人与自然和谐的蓝色经济区，率先基本实现现代化。<ref>{{ cite web|url=http://www.sdlb.gov.cn/art/2011/3/2/art_536_29138.html|title=山东半岛蓝色经济区发展目标|publisher=山东半岛蓝色经济区建设办公室|date=2011-03-02|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20130511091602/http://www.sdlb.gov.cn/art/2011/3/2/art_536_29138.html|archivedate=2013-05-11}}</ref><ref>{{cite web|accessdate=2014-06-08|url=http://sd.ce.cn/xw/sd/201404/04/t20140404_1443550.shtml|title=山东半岛蓝色经济区发展规划实施3年多 进入战略视野|publisher=中国经济网|deadurl=yes|archiveurl=https://web.archive.org/web/20140829131724/http://sd.ce.cn/xw/sd/201404/04/t20140404_1443550.shtml|archivedate=2014-08-29}}</ref>

=== 空间布局 ===
根据山东半岛蓝色经济区的战略定位、资源环境承载能力、现有基础和发展潜力，按照以陆促海、以海带陆、海陆统筹的原则，优化海洋产业布局，提升[[胶东半岛高端海洋产业集聚区|胶东半岛高端海洋产业集聚区]]核心地位，壮大[[黄河三角洲高效生态海洋产业集聚区|黄河三角洲高效生态海洋产业集聚区]]和[[鲁南临港产业集聚区|鲁南临港产业集聚区]]两个增长极；优化海岸与海洋开发保护格局，构筑海岸、近海和远海三条开发保护带；优化沿海城镇布局，培育[[青岛|青岛]]—[[潍坊|潍坊]]—[[日照|日照]]、[[烟台|烟台]]—[[威海|威海]]、[[东营|东营]]—[[滨州|滨州]]三个城镇组团，形成“一核、两极、三带、三组团”的总体开发框架。<ref>{{ cite web|url=http://www.sdlb.gov.cn/art/2011/3/2/art_536_29139.html|title=山东半岛蓝色经济区空间布局|publisher=山东半岛蓝色经济区建设办公室|date=2011-03-02|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20120429042913/http://www.sdlb.gov.cn/art/2011/3/2/art_536_29139.html|archivedate=2012-04-29}}</ref>
