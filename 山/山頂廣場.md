'''山頂廣場'''（'''{{lang|en|The Peak Galleria}}'''）是[[香港|香港]]一所[[商場|商場]]，位於[[香港島|香港島]][[太平山_(香港)|太平山]][[山頂道|山頂道]]118號，佔地13萬平方呎，為山頂區內最大型的購物中心，發展商為[[恒隆地產|恒隆地產]]。

山頂廣場<ref>{{cite web |language=zh-hk |url=http://hk.ulifestyle.com.hk/spot/detail.html?id=ABdGDVowBwlZEgNs |title=山頂廣場 |publisher=[[HK.ULlifestyle.com.hk|HK.ULlifestyle.com.hk]] }}</ref>建於1992年，1993年2月5日正式開幕。商場樓高3層，原址為1873年至1938年的[[山頂酒店|山頂酒店]]，其後改為露天巴士總站。現時山頂廣場的地面層仍設有一個公共運輸交匯處，以及[[停車場|停車場]]設施。一樓則設有兒童遊樂場，廣場頂層設有一個佔地萬多呎的花園平台，於2008年曾進行翻新。平台內設有收費[[望遠鏡|望遠鏡]]（現已改為免費），遊人可以在此觀賞[[維多利亞港|維多利亞港]]兩岸及[[薄扶林|薄扶林]]一帶的景色。

2015年8月，城規會通過批准在山頂廣場天台興建全港首條空中飛索，全長約120米。 <ref>{{cite web |language=zh-hk |url=http://cablenews.i-cable.com/webapps/news_video/index.php?news_id=463784 |title=城規會通過山頂廣場建空中飛索 |publisher=有線新聞 }}</ref>空中飛索設施，局限在商場範圍內完成飛索。每次飛索時速預計可到24公里（約1秒可滑至約6米），即20秒左右就飛完，預計一小時最多可讓120人次參與。<ref>{{cite web |language=zh-hk |url=http://hk.ulifestyle.com.hk/spot/detail.html?id=ABdGDVo2BwlZHwNj&utm_source=uhk&utm_campaign=spot-list&utm_medium=title&utm_content=title3-01 |title=山頂廣場空中飛索 |publisher=HK [[港生活|港生活]] }}</ref>不過發展商到2016年9月表示終止計劃。<ref>{{cite web |language=zh-hk |url=http://hk.on.cc/hk/bkn/cnt/news/20160902/bkn-20160902125421089-0902_00822_001.html |title=山頂廣場無得玩空中飛索！發展商終止計劃|publisher=on.cc東網}}</ref>

==商場設計==
山頂廣場興建工程由Denton Corker Marshall（DCM）負責。為了加強商場特色，當年場內不同位置放置了多個藝術品。例如山頂廣場正門前露天廣場設有全港第一座跳躍噴泉，噴泉池的底部是平地面。商場內的中庭也設有「金沙漏斗」藝術品，水花由頂部傾瀉而下。此兩個藝術品由[[美國|美國]]的WET Design設計。

商場入口中庭頂部也設有由Cork Marches-chi設計的游走霓紅燈飾。此外，商場內的左右迴廊的天窗上，懸掛了許多大型全屬掛飾及色彩奪目的平面雕塑。全屬掛飾更會隨着空氣流動而產生各種不同形態。

然而，商場內所有藝術品已在2008年拆卸。

==商場店舖==
山頂廣場店舖包括多間精品店，包括[[山頂朱古力博物館|山頂朱古力博物館]]、[[香港電車文化館|香港電車文化館]]、家居用品店、時裝店及[[百佳|百佳]][[International_(超級市場)|International]]。在食肆方面，包括[[麥當勞餐廳|麥當勞餐廳]]、[[Haagen_Dazs|Haagen Dazs]]、[[Delifrance|Delifrance]]、[[星巴克咖啡|星巴克咖啡]]、[[Marion_Crepes|Marion Crepes]]、[[麥奀雲吞麵世家|麥奀雲吞麵世家]]、[[翠華餐廳|翠華餐廳]]及商場最大食肆[[峰景餐廳|峰景餐廳]]（CAFE DECO）。商場頂層曾設有娛樂中心及大型酒吧。其後在2002年酒吧結業後變更為[[幼稚園|幼稚園]]。全球最恐怖鬼屋[[Fright_Dome|Fright Dome]]，2014年9月26日至11月2日首度登陸亞洲，選址商場頂層平台，佔地15000平方米，設兩個主題區「殺人小丑」及「德州電鋸殺人狂」。

==翻新工程==
2015年，恒隆地產計劃2016年為商場進行翻新工程，由國際建築設計事務所Aedas負責，預計2019年夏天完成。翻新後3樓將設全球首個「大富翁夢想世界」主題館，佔地約二萬呎，將於2019年6月至7月試業。<ref>{{cite news |title=全球首個「大富翁夢想世界」主題館　落戶山頂廣場 |url=https://wealth.hket.com/article/2245125/%E5%85%A8%E7%90%83%E9%A6%96%E5%80%8B%E3%80%8C%E5%A4%A7%E5%AF%8C%E7%BF%81%E5%A4%A2%E6%83%B3%E4%B8%96%E7%95%8C%E3%80%8D%E4%B8%BB%E9%A1%8C%E9%A4%A8%E3%80%80%E8%90%BD%E6%88%B6%E5%B1%B1%E9%A0%82%E5%BB%A3%E5%A0%B4 |accessdate=2019-01-08 |publisher=香港經濟日報 |date=2019-01-07}}</ref>
