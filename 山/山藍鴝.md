'''山藍鴝'''（學名：''Sialia currucoides''；英語：mountain bluebird）是[[蓝鸲属|蓝鸲属]]的一種小鳥，體重平均{{Convert|30|g}}，體長{{Convert|16|-|20|cm}}。雄性除去腹部外，通體藍色。雌性體色則較為黯淡。它分佈于北美洲中西部，是美國[[愛達荷州|愛達荷州]]和[[內華達州|內華達州]]的[[美國州鳥列表|州鳥]]<ref>{{cite web | url= http://www.netstate.com/states/symb/birds/id_mountain_bluebird.htm | title= Idaho State Bird | work= Georgia State Bird Mountain Bluebird Sialia currucoides | publisher= Netstate}}</ref><ref>{{cite web | url= http://www.val-u-corp.com/articles/nevada-state-bird.html | title= Nevada State Bird | work= Nevada State Bird, mountain bluebird | publisher= Val-U-Corp Services, Inc. | access-date= 2019-03-03 | archive-url= https://web.archive.org/web/20150114023107/http://www.val-u-corp.com/articles/nevada-state-bird.html | archive-date= 2015-01-14 | dead-url= yes }}</ref>。
