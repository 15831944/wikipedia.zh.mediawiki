==簡介與特色==
本遊戲為《[[戰艦少女|戰艦少女]]》的外傳作品，除了沿用一部分來自《戰艦少女》的艦艇外，也額外加入一部分的原創艦艇或全新繪製的艦艇；另外就玩法而言，本遊戲從《戰艦少女》原本的[[隨機數生成|RNG]]自動裁決戰鬥方式，大幅改變了玩法，可讓玩家藉由[[戰略角色扮演遊戲|SRPG]]的操作方式<ref>{{Cite web |url = http://news.4399.com/cqzh/zixun/m/851007.html |title = 老牌国产舰娘手游《战舰少女R》将推战棋类外传新作 |author = 未荻未荻 |date = 2018-06-27 |publisher = 4399手機遊戲網 |language = zh-cn |accessdate = 2018-10-09 |quote = }}</ref>，投入主力作戰艦艇五艘、預備艦艇兩艘、支援艦艇一艘，移動艦艇、呼叫援軍並殲滅敵艦。
==故事大綱==
隨著不明水下物體（Unindentified Underwater Objects, UUO）在世界各地肆虐，由[[私人軍事服務公司|PMC]]總裁馬克西穆斯·希爾伯特（Maximus Hilbert）在[[挪威|挪威]][[卑爾根|卑爾根]]設立國際維安部隊「'''NeoForce'''」，統合全球海上力量、建立浮動要塞，對抗來自UUO的威脅。

而NeoForce迄今已成立四支艦隊，包含由[[胡德號戰鬥巡洋艦|胡德]]率領的第一艦隊、[[俾斯麥號戰艦|俾斯麥]]率領的第二艦隊、[[伊吹號航空母艦|伊吹]]率領的第三艦隊、[[密蘇里號戰艦|密蘇里]]率領的第四艦隊<ref>{{Cite web |url = https://gamewith.jp/gamedb/article/game/show/4043/3664?from=ios |title = 初出し情報！『蒼青のミラージュ』新隊員・イベントを大公開！ |author = GameWith編輯部 |date = 2018-10-21 |publisher = Gamewith |language = ja |accessdate = 2018-10-21 |quote = }}</ref>，在世界各地南征北討、殲滅UUO、奪回一望無際的大海……
