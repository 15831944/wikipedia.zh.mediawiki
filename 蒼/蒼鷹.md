'''蒼鷹'''（[[學名|學名]]：''{{lang|la|Accipiter gentilis}}''），別名'''牙鷹'''，是鷹科鷹屬中一種中到大型的[[猛禽|猛禽]]。與其他[[鵟屬|鵟屬]]及[[鷹|鷹]]等一樣都在日間活动。

苍鹰廣泛分布在[[北半球|北半球]]的溫帶地區。在[[北美|北美]]地区被稱为''Northern Goshawk''，是當地的[[留鳥|留鳥]]。但在較冷的地區，如[[亞洲|亞洲]]北部及[[加拿大|加拿大]]等地的蒼鷹則會迁移到南方過冬。

蒼鷹的图案也出现在[[葡萄牙|葡萄牙]]領土[[亞速爾群島|亞速爾群島]]的旗幟上。因为最初來到這個群島的葡萄牙[[探險者|探險者]]踏上這片土地時正好看到一只猛禽在上空盤旋，他们認為那是一只蒼鷹。事實上亞速爾群島上從来沒有蒼鷹出沒。經过後來的學者研究，那应该是[[普通鵟|普通鵟]]的一个亞種。

== 外觀描述 ==
[[File:AccipterGentilisJuvenileFlight1.jpg|thumb]]
蒼鷹是鷹屬鳥類中體型最大的一員<ref>{{cite web |url=http://redpath-museum.mcgill.ca/Qbp/birds/Specpages/northerngoshawk.htm|title=Northern Goshawk|accessdate=2008-06-17|publisher=Birds of Quebec}}</ref>，首先於1758年由[[卡爾·林奈|卡爾·林奈]]在其〈[[自然系統|自然系統]]〉（''{{lang|la|Systema naturae}}''）一書中介紹及命名<ref>{{la icon}} {{cite book | last=Linnaeus | first=C | authorlink=Carolus Linnaeus | title=Systema naturae per regna tria naturae, secundum classes, ordines, genera, species, cum characteribus, differentiis, synonymis, locis. Tomus I. Editio decima, reformata. | publisher=Holmiae. (Laurentii Salvii). | date=1758| quote = F. cera pedibusque flavis, corpore cinereo maculis fuscis cauda fasciis quatuor nigricantibus. |pages=89}}</ref>。其翅膀短而寬闊，並有長長的尾羽，這是林棲鳥種適應高密度樹林生活的重要特征。苍鹰有明顯的[[兩性異形|兩性異形]]現象，雄鳥上半身灰藍，下半身則為有條紋的灰，體長49至57厘米、翼展為93至105厘米。雌鳥比雄鳥體型更大，可長達58至64厘米，翼展達108至127厘米，上半身暗藍灰、下半身則為純灰。最小的雄鳥體重只有630[[克|克]]，但最大的雌鳥則可重達2.4[[公斤|公斤]]。亞成鳥上半身是有條紋的灰色，下半身則為純灰色。牠們常展現出獨有的飛行特徵──五次緩慢的拍翼後滑翔而去。

在[[歐亞大陸|歐亞大陸]]上，雄鳥常與雌性的[[北雀鷹|北雀鷹]]（[[:en:Sparrowhawk|Sparrowhawk]]）相混淆，但一般它們都有肥胖的體型和相對较長的翅膀。

在[[北美洲|北美洲]]，苍鹰幼鳥容易與小型的[[條紋鷹|條紋鷹]]及[[雞鷹|雞鷹]]相混淆，但人们还是能够通过比较體型将它們区分开来。

== 捕獵及食物 ==
蒼鷹主要捕捉其他細小的鳥類及[[哺乳動物|哺乳動物]]為食，常常集体進行掠地飛行，越过茂密的森林，並对獵物突然發起袭击。與大部分猛禽一樣，蒼鷹也是“機會主義者”，只要看到合適的獵物就會出擊。但牠們還是對鳥類情有獨鍾，特別是[[披肩雞|披肩雞]]（[[:en:ruffed_grouse|ruffed grouse]]、[[鴒形目|鴒形目]]的鳥種及[[雀形目|雀形目]]中的椋鳥等。水鳥當中，體型小於[[綠頭鴨|綠頭鴨]]的鳥類也是牠們的目標。此外，蒼鷹也偶尔捕捉一些體型較大的動物，如[[雪兔|雪兔]]及[[長耳大野兔|長耳大野兔]]等。

== 習性 ==
[[File:GOSHAWK.JPG|thumb]]
进入[[春天|春天]]的繁殖季節，雄鳥就會開始牠們如[[過山車|過山車]]般的飛行表演，而這時也是觀察這種神秘鳥类的最佳時機。成鳥會在三至四月回到牠們築巢的地方，並在四至五月期間下蛋。巢內一般有1到5顆鳥蛋在內，大小為59 x 45mm，約重60克。孵卵期在28到38日內。幼鳥則在35日起開始離巢，10日後便會尝试飛行。即使在一年後，幼鳥仍會在其父母身边活動，這與成鳥會保護整個區域有關。牠們對任何入侵者均沒有好感，即使是路過的人類也會遭到牠們激烈的驅趕。因此，即使是它们的鳥蛋或幼鳥也极少被捕掠。蒼鷹偶爾也會招到如[[雕鴞|雕鴞]]和[[鵟屬|鵟屬]]猛禽的攻擊。

== 狀況 ==
19世紀时，蒼鷹因为受到收藏家的青睞及被當地的[[獵場看守人|獵場看守人]]猎杀而在[[英國|英國]]灭绝了，但近年有不少來自歐洲大陆的蒼鷹重新移居英国。目前在英國[[諾森伯蘭|諾森伯蘭]]的[[開爾德森林|開爾德森林]]內已發現了牠們的蹤影。
