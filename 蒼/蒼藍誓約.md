《'''蒼藍誓約'''》為[[中華人民共和國|中國]][[北京巴別時代科技股份有限公司|北京巴別時代科技股份有限公司]]於2018年開發的{{Link-ja|艦船擬人化|艦船擬人化|艦娘}}主題手機遊戲。本遊戲首先於2018年11月16日展開[[封閉測試|封閉測試]]<ref>{{cite web |url=https://www.babeltime.com/site2/read?id=15187 |title=《苍蓝誓约》精英测试指挥官招募，现在开启！ |author=<!--Staff writer(s); no by-line.--> |date=2018-11-16 |website=北京巴別時代官網 |publisher=北京巴別時代 |access-date=2019-03-12 |quote= |language=zh-cn}}</ref>，接著於2019年3月1日展開[[公開測試|公開測試]]<ref>{{cite web |url=https://www.babeltime.com/site2/read?id=15783 |title=巴别时代《苍蓝誓约》将于3月1日开启安卓删档付费测试 |author=<!--Staff writer(s); no by-line.--> |date=2019-02-14 |website=北京巴別時代官網 |publisher=北京巴別時代 |access-date=2019-03-12 |quote= |language=zh-cn}}</ref>。現階段僅暫定發行簡體中文版，正式上市時程待定。

== 簡介與特色 ==
本遊戲的前身為遊戲企劃案《'''誓約之海'''》，後來在2018年底，於封閉測試開始前改名《蒼藍誓約》。<br/>
遊戲玩法跟《[[艦隊Collection|艦隊Collection]]》[[街機|街機]]版十分相似，玩家可派遣六艘艦艇投入戰鬥，並搭配不同樣式的武裝和陣形與敵軍艦隊交戰<ref>{{cite web |url=https://www.youtube.com/watch?v=XlKgyuxZjuE |title=《蒼藍誓約》手機遊戲介紹 |author=<!--Staff writer(s); no by-line.--> |date=2019-03-11 |website=Youtube |publisher=[[巴哈姆特電玩瘋|巴哈姆特電玩瘋]] |access-date=2019-03-12 |quote= |language=zh-tw}}</ref>。

== 故事大綱 ==
本遊戲故事設定在一個架空的世界中，隨著上一次大戰的結束，無數陣亡將士的意念沉入大海，並匯聚為一股無價的能量─「'''奧斯能量'''」。<br/>
然而，從異次元來襲的生命體「'''穆伯'''」，以[[蟲洞|時空縫隙]]大舉侵略，打破了上次大戰後長久的和平；於是玩家臨危受命，召集麾下的各式艦艇，除了對抗「穆伯」以外，更要守護「奧斯能量」這個無價資產……
