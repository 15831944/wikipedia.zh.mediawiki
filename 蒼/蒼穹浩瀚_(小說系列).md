《'''蒼穹浩瀚'''》（{{lang-en|'''''The Expanse'''''}}）是[[科幻|科幻]]長篇小說及短篇小說系列，由[[詹姆斯·S·A·科里|詹姆斯·S·A·科里]]創作，這是作者{{link-en|丹尼爾·亞伯拉罕|Daniel Abraham (author)}}與泰·弗蘭克（Ty Franck）的[[筆名|筆名]]。第一本小說《{{link-en|利維坦覺醒|Leviathan Wakes}}》於2012年被提名[[雨果獎最佳長篇小說獎|雨果獎最佳長篇小說獎]]。<ref>{{Cite web |url = http://www.thehugoawards.org/hugo-history/2012-hugo-awards/ |title = 2012 Hugo Awards |publisher = Hugo Awards |language = en |deadurl = yes |archiveurl = https://www.webcitation.org/66nnsGGRI?url=http://www.thehugoawards.org/hugo-history/2012-hugo-awards/ |archivedate = 2012-04-09 }}</ref>2017年，本系列被提名雨果獎「最佳系列」<ref>{{Cite web |url = http://www.thehugoawards.org/hugo-history/2017-hugo-awards/ |title = 2017 Hugo Awards |publisher = Hugo Awards |language = en}}</ref>。

至2019年為止，《蒼穹浩瀚》包括了七本主要小說和五本短篇作品（兩個前傳短篇故事、一個前傳短篇小說、一個短篇小說設定於《卡利班之戰》與《''Abaddon's Gate''》之間，以及一個圍繞《''Nemesis Games''》事件的短篇小說）。另外二本長篇小說，還有兩本短篇小說已經計劃好。本系列被[[Syfy|Syfy]]改編成電視劇，以《[[太空無垠|太空無垠]]》（''The Expanse''，又名《蒼穹浩瀚》）的標題播出。該劇於2015年11月23日在串流服務上播放，2015年12月14日在Syfy上播放。Syfy不再續訂第四季，但在2018年5月26日，在該劇的粉絲群進行協商活動後拯救了它，亞馬遜公司CEO[[傑佛瑞·貝佐斯|傑佛瑞·貝佐斯]]宣布[[亞馬遜影片|亞馬遜]]購買了該劇<ref>{{Cite web |url = https://www.theverge.com/2018/5/26/17379736/the-expanse-amazon-prime-season-four-saved |title = It's official: Amazon has saved The Expanse |author = Andrew Liptak |date = 2018-05-26 |publisher = The Verge |language = en}}</ref>。

== 劇情簡介 ==
本書系列主要講述船長Jim Holden和火星軍艦[[羅西南多|羅西南多]]號（Rocinante）上的船員無意中卷入一個人類迄今所知最大的陰謀活動，對抗貪婪的大公司，腐敗政客，以及好戰的海軍上將以獲得真相。

== 角色人物 ==
羅西南多號（Rocinante）：
一艘獨立武裝飛船，船員是冰塊搬運工坎特伯里號（Canterbury）上的生還者。
*'''James "Jim" R. Holden'''：羅西南多號的地球居民船長
*'''Naomi Nagata'''：行星帶居民的工程師及羅西南多號的執行官
*'''Amos (Timothy) Burton'''：羅西南多號的地球居民首席工程師
*'''Alex Kamal'''：羅西南多號的火星領航員
*'''Roberta "Bobbie" W. Draper'''：火星國會共和國海軍（MCRN）的槍炮軍士
*'''Clarissa "Claire" Melpomene Mao'''，又名'''Melba Alzbeta Koh'''，又名'''Peaches'''：Mao-Kwikowski貿易公司巨頭Jules-Pierre Mao的女兒，當她作為Melba時，她是一名有執照的電化學技師。在與她聯繫後，Amos給了她綽號Peaches。

外行星：
*'''Josephus "Joe" Aloisus Miller'''：行星帶居民，Star Helix Security保安公司管理的穀神星站警探
*'''Juliette "Julie" Andromeda Mao'''：地球居民富豪Jules-Pierre Mao的次女，前賽艇手，外行星聯盟皈依者
*'''Frederick "Fred" Lucius Johnson'''：前聯合國海軍，曾被譴責為安德森站的劊子手，現在是OPA的領導人
*'''Dr. Praxidike "Prax" Meng'''：木衛三（Ganymede）上的南方大豆農場計劃的首席植物學家和Mei Meng的父親
*'''Carlos "Bull" c de Baca'''：OPA的成員，擔任貝西摩斯號（Behemoth）上的首席保安官
*'''Basia "Baz" Merton'''：來自木衛三的焊工
*'''Rev. Dr. Annushka "Anna" Volovodov'''：在木衛二（Europa）上的聖約翰聯合會的[[衛理宗|衛理公會]]牧師
*'''Manéo "Néo" Jung-Espinoza'''：來自穀神星的年輕行星帶居民
*'''Filip Inaros'''：OPA的少年成員，以及Marco Inaros和Naomi Nagata的兒子

火星：
*'''Sauveterre'''：火星國會共和國海軍Barkeith號的隊長

地球：
*'''Dimitri Havelock'''：來自地球的保安承包商和Joe Miller的前伙伴
*'''Chrisjen Avasarala'''：聯合國行政監管助理副部長，後來成為[[聯合國秘書長|聯合國秘書長]]
*'''Dr. Elvi Okoye'''：來自地球的生物學家

拉科尼亞（Laconia）：
*'''Winston Duarte'''
*'''Paolo Cortázar'''
*'''Santiago Jilie Singh'''
*'''Anton Trejo'''
*'''Aliana Tanaka'''

== 衍生作品 ==
=== 電視劇 ===
{{main|太空無垠}}
2014年4月，美國電視臺[[Syfy|Syfy]]宣佈小說系列改編成電視劇，並且預訂了第一季各一小時長的10集，於2015年12月首播。劇集主演有[[湯馬士·珍|湯馬士·珍]]飾演Josephus Miller，以及[[史蒂文·斯崔特|史蒂文·斯崔特]]飾演Jim Holden。羅西南多號的其他船員演員有[[多明妮克·蒂珀|多明妮克·蒂珀]]飾演Naomi Nagata、[[凱斯·安凡|凱斯·安凡]]飾演Alex Kamal和[[韋斯·查塔姆|韋斯·查塔姆]]飾演Amos Burton。其他主要演員有[[索瑞·安達斯魯|索瑞·安達斯魯]]飾演Chrisjen Avasarala、[[查德·科爾曼|查德·科爾曼]]飾演Fred Johnson和[[佛羅倫斯·費佛瑞|佛羅倫斯·費佛瑞]]飾演Julie Mao。
