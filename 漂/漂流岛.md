'''漂流岛'''（Castaway Cay）原名'''戈尔达岛'''（Gorda Cay），是一座位于[[巴哈马|巴哈马]]北部[[阿巴科群岛|阿巴科群岛]]的私人岛屿。目前，漂流岛为[[华特迪士尼公司|华特迪士尼公司]]完全拥有，为其旗下[[迪士尼郵輪公司|迪士尼郵輪公司]]的[[迪士尼魔法號|迪士尼魔法號]]、[[迪士尼奇妙號|迪士尼奇妙號]]、[[迪士尼夢想號|迪士尼夢想號]]和[[迪士尼幻想號|迪士尼幻想號]]游轮所使用，邮轮可以直接靠岸，宾客无需搭乘小船接驳才能上岸。

== 历史 ==

== 简介 ==
漂流岛上的用水是从[[卡纳维拉尔角|卡纳维拉尔港]]所购买，并用船舶运送到岛上一个80000 加仑的淡水储水槽内储存。大约有60名[[迪士尼郵輪公司|迪士尼郵輪公司]]工作人员居住在漂流岛，并在此全时工作。

== 娱乐设施 ==
[[Image:Castaway_Cay_beach.jpg|thumb]]
* “飞翔的荷兰人”（Flying Dutchman）是一艘取材自迪士尼影片《[[加勒比海盗|加勒比海盗]]》的电影道具船。
* “漂流岛邮局”（Castaway Cay Post Office），是一座位于漂流岛上的当地邮局，所有从漂流岛上寄出的信函、明信片和包裹都会盖上专有的漂流岛邮戳。
