'''菅井琴'''（{{jpn|j=菅井 きん|kk=すがい きん|rm=Sugai Kin|f=no}}，{{bd|1926年|2月28日|2018年|8月10日|catIdx=Sugai}}），本名“'''佐藤贵美子'''（{{lang|ja|'''佐藤キミ子'''}}）”，曾用艺名“'''须斋贵美子'''（{{lang|ja|'''須斎キミ子'''}}）”，[[日本|日本]][[女演员|女演员]]。菅井琴因其在电影《{{Link-ja|葬礼 (日本电影)|お葬式|葬礼}}》中的优秀表现，而荣获第9届[[报知电影奖|报知电影奖]]和第8届[[日本电影学院奖|日本电影学院奖]]的“最优秀助演女优赏”，以及第6届[[横滨电影节|横滨电影节]]“助演女优赏”；<ref>{{cite web |title=Kin Sugai - Awards - IMDb |url=https://www.imdb.com/name/nm0837361/awards |publisher=[[IMDb|IMDb]] |accessdate=2018-08-23}}</ref>同时还荣获日本政府所颁发的[[褒章|紫绶褒章]]（1990年）和[[寶冠章|勋四等宝冠章]]（1996年）。<ref name="oricon">{{cite web |title=女優・菅井きんさん死去 92歳 「必殺」シリーズなど |url=https://www.oricon.co.jp/news/2118068/full/ |publisher=[[Oricon|Oricon]] |accessdate=2018-08-23}}</ref> 2008年，菅井琴因主演电影《{{Link-ja|我的奶奶 (日本电影)|ぼくのおばあちゃん|我的奶奶}}》而获得[[吉尼斯世界记录|吉尼斯世界记录]]认证的“世界最高龄电影初次主演女演员”。<ref name="oricon" /><ref name="sponichi" /> 2018年8月10日，菅井琴因[[心力衰竭|心力衰竭]]死于[[东京都|东京都]]，享年92岁。<ref name="oricon" /><ref>{{cite web |title=俳優の菅井きんさん死去　９２歳　「必殺」シリーズ姑役 |url=https://www.asahi.com/articles/ASL8R41RCL8RUCLV009.html |publisher=[[朝日新闻|朝日新闻]] |accessdate=2018-08-23}}</ref><ref name="sponichi" >{{cite web |title=菅井きんさん死去　９２歳　心不全のため都内の自宅で息を引き取る |url=https://www.sponichi.co.jp/entertainment/news/2018/08/23/kiji/20180823s00041000100000c.html |publisher=[[體育日本|體育日本]] |accessdate=2018-08-23}}</ref>
