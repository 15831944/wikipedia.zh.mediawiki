'''菅谷站'''（{{jpn|j=菅谷駅|hg=すがやえき|rm=Sugaya eki|f=y}}）位于[[福岛县|福岛县]][[田村市|田村市]]泷根町菅谷字堂田，是[[东日本旅客铁道|东日本旅客铁道]]（JR東日本）管辖的[[磐越东线|磐越东线]]上的[[鐵路車站|鐵路車站]]<ref name="station"/>。是[[磐越东線|磐越东線]]海拔最高车站<ref name="station"/>。

== 历史 ==
* 1948年10月10日 - 车站投入运营<ref name="station"/>。
* 1986年11月10日 - 无人化改造。但出站口仍由小野新町站派遣员工值守。
* 1987年4月1日 - [[国铁分割民营化|国铁分割民营化]]后成为JR东日本车站。
* 1989年3月11日 - 简易委托化。
* 2016年4月1日 - [[三春站|三春站]]业务委托化，管理站变更为[[郡山車站_(福島縣)|郡山站]]<ref name="station"/>。

== 车站结构 ==
[[侧式站台|侧式站台]]1台1线的[[地上车站|地上车站]]<ref name="station"/>。原[[岛式站台|岛式站台]]1台2线车站，现使用原2站台<ref name="station"/>。

[[郡山站_(福島县)|郡山站]]管理的简易委托站（田村市委托）。出站窗口（营业时间6时30分 - 15时00分 周六日节假日休息）。设置便携POS机，可以办理指定席业务。
