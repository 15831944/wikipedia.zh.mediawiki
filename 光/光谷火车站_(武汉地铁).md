'''光谷火车站'''是[[武汉轨道交通2号线|武汉轨道交通2号线]]和[[武汉轨道交通11号线|11号线]]的换乘车站，位于武汉市[[东湖高新区|东湖高新区]]。<ref>[http://hb.sina.com.cn/news/zwzl/2017-06-13/detail-ifyfzfyz3599527.shtml?from=hb_ydph 张晟. 6条新地铁线 73座车站有了站名<nowiki>[N]</nowiki>. 长江日报，2017年6月13日(4).]</ref>

==概要==	
本站为11号线一期工程西端的起始站，亦是2号线与11号线的换乘车站，两线车站均位于[[国铁|国铁]][[光谷站|光谷站]]的东侧。2号线车站沿国铁光谷火车站东站房设置；11号线车站位于湖口一路与[[国铁|国铁]][[光谷站|光谷站]]东广场地块下。本站为地下三层岛式车站，地下一层为站厅层，地下二层为2号线站台层，地下三层为11号线站台层。2号线车站布置有4个出入口、2个消防出入口，设置有5组风亭；11号线车站共设置10个出入口、7组风亭。两线换乘方式为T型换乘。<ref>武汉地铁集团. [http://www.whrt.gov.cn/public_forward.aspx??url=site_detail.aspx?action=GetTowOne&siteId=3d0598d6-7d3b-4ab1-9cbc-fd04c9a991b5&isSite=1&isopenLine=1 站点信息].</ref><ref>武汉地铁集团. [http://www.whrt.gov.cn/public_forward.aspx??url=site_detail.aspx?action=GetEle_EOne&siteId=902c0bcf-7436-460a-8cf3-f244e512e2a0&isSite=1&isopenLine=0. 工程信息].</ref><ref>{{Cite web |url =http://www.whhkgjt.com/qygl/scjy/2017-06-16/3326.html|title =【城建公司】地铁2号线南延线光谷火车站～佳园路段首台盾构机始发掘进|author =刘丽君|date = 2017-06-16 |publisher = 武汉航空港发展集团有限公司 }}</ref><ref>{{Cite web |url =http://tlgs.baju.com.cn/NewsView.asp?NewsID=1599|title =光谷火车站站主体结构顺利封顶，武汉地铁11号线一标五个车站全部封顶|author =刘忠强，王韦程|date = 2017年1月23日 |publisher = 中国水利水电第八工程局有限公司铁路公司 }}</ref><ref>{{Cite web |url =http://www.whrailway.cn/html/2015/meitikanwutie_0803/42835.html|title =武汉光谷火车站开建 将成国内最佳无缝换乘站|author =作者不详|date = 2015-08-03 |publisher = [[武汉铁路局|武汉铁路局]] }}</ref>
