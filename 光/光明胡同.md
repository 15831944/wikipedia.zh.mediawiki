'''光明胡同'''，是位于[[北京市|北京市]][[西城区|西城区]]中部的胡同。

==简介==
光明胡同原先为南北走向，南到东红门胡同，北到[[西安门大街|西安门大街]]，全长471米，平均宽5米。[[清朝|清朝]]称光明殿胡同，简称光明殿，因为皇家道观[[大光明殿|大光明殿]]在其西侧而得名。1965年改称光明胡同。《京师坊巷志稿》卷上引《[[明世宗|明世宗]]实录》：“大光明殿明嘉靖三十六年（1557年）建，内奉玉皇。”如今大光明殿已无存，其址为[[国家机关事务管理局|国家机关事务管理局]]所在地。1980年代到1990年代，光明胡同两侧[[槐树|槐树]]成荫并有花坛，是北京市花园式街道。过去的光明胡同45号为作家[[刘绍棠|刘绍棠]]故居<ref>{{Cite web |url = http://www.bjdclib.com/subdb/laneculture/lane/200907/t20090703_4902.html |title = 光明胡同 |author =  |date = 2009-07-03 |publisher = 东华流韵 |language =  |accessdate =  |quote =  }}</ref>。

东红门胡同呈“厂”字形，东到[[府右街|府右街]]，西到光明胡同，南端不通行。全长54米，平均宽4米。清朝泛称红门，以官署之门而得名。[[中华民国|中华民国]]时期析为东、西、南三条胡同，该胡同在东，故称东红门（其他两条即后来的[[西红门胡同|西红门胡同]]、[[南红门胡同|南红门胡同]]）。1965年定名东红门胡同<ref>{{Cite web |url = http://www.bjdclib.com/subdb/laneculture/lane/200907/t20090703_4908.html |title =  东红门胡同|author =  |date =2009-07-03 |publisher = 东华流韵 |language =  |accessdate =  |quote =  }}</ref>。

2010年，在北京市西城区进行西城旧城保护和居民住房改善工程项目一区至四区拆迁建设，东红门胡同被拆迁<ref>{{Cite web |url = http://paper.people.com.cn/rmrbhwb/html/2010-11/20/content_677174.htm |title = 公告 |author =  |date =  |publisher =人民日报海外版2010年11月20日，第06版  |language =  |accessdate =  |quote =  }}</ref>。此后光明胡同将东红门胡同剩余部分并入，新的光明胡同北到西安门大街，南到原东红门胡同西口处向东，东到府右街。
