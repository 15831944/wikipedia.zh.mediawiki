'''光明皇后'''（[[大寶_(日本)|大寶]]元年－[[天平寶字|天平寶字]]四年[[六月初七|六月初七]]，即701年－760年7月27日），[[姓|姓]][[藤原氏|藤原氏]]，為[[日本|日本]][[奈良時代|奈良時代]]的[[皇族|皇族]]，[[聖武天皇|聖武天皇]]的[[皇后|皇后]]，又名'''安宿媛'''（{{lang|ja|あすかべひめ}}）、'''光明子'''（{{lang|ja|こうみょうし}}）、'''藤三娘'''（{{lang|ja|とうさんじょう}}），死後追[[諡|諡]]'''天平應真仁正皇太后'''。

==親人==
*父親 [[藤原不比等|藤原不比等]]
*母親 [[縣犬養三千代|橘三千代]]
*姨甥兼夫婿 聖武天皇
*異母姐 [[藤原宮子|藤原宮子]]
*異父兄 [[橘諸兄|橘諸兄]]

== 生平 ==
光明子在[[聖武天皇|聖武天皇]]還是[[皇太子|皇太子]]時與之結婚，並在[[養老_(日本年號)|養老]]二年（718年）生下[[孝謙天皇|阿倍內親王]]。在724年（[[神龜|神龜]]元年）因丈夫即位成為[[聖武天皇|聖武天皇]]，而獲得[[夫人|夫人]]的[[嬪妃|嬪妃]]位號。[[神龜|神龜]]四年（727年）生下[[基皇子|基皇子]]，並將之立為[[皇太子|皇太子]]。次年基皇太子突然夭折（有傳說是[[長屋王|長屋王]]的詛咒），因而發生了[[長屋王之變|長屋王之變]]。叛亂平定後，改元[[天平|天平]]（729年），同年八月將光明子夫人冊封為[[皇后|皇后]]，由於光明子並非皇族出身，她的立后開了「非皇族出身皇后」的先河，也是[[藤原氏|藤原氏]]一族之女子成為皇后的起始。不過因為[[長屋王之變|長屋王之變]]經後世證實，純屬誣告，因而有懷疑長屋王之變是藤原氏一族與光明子一同策劃的陰謀。

在除去對手後，光明皇后所生的女兒[[孝謙天皇|阿倍內親王]]被立為[[皇太女|皇太女]]。749年聖武天皇讓位出家，由[[皇太女|皇太女]]的[[孝謙天皇|阿倍內親王]]即位，而光明皇后也以帝母的身分掌握了實權，並將[[皇后宮職|皇后宮職]]改名為[[紫微中台|紫微中台]]，並由光明皇后的甥兒[[藤原仲麻呂|藤原仲麻呂]]擔任該官職之長官，實行了各式各樣的政策。

[[天平勝寶|天平勝寶]]八年（756年），聖武法皇崩御，兩年後光明皇后被贈與皇太后的名號。[[天平寶字|天平寶字]]四年(760年)，光明皇太后崩御，朝廷追諡為'''天平應真仁正皇太后'''，葬於佐保山東陵。

== 文化上的影響 ==
光明皇后一生都篤信佛教，[[東大寺|東大寺]]及[[國分寺|國分寺]]都是依她的建言而創建的。同時為了廣照天下百姓，因而創設了專照顧貧困人民的「[[悲田院|悲田院]]」，及相關的醫療設施「[[施藥院|施藥院]]」。而在丈夫聖武天皇駕崩后的第四十九日，為了將聖武天皇的遺物有所安置，東大寺的正倉院為此創設，除此之外，[[興福寺_(奈良)|興福寺]]、[[法華寺|法華寺]]、[[新藥師寺|新藥師寺]]等寺院，都是依光明皇后的建言而創設。

同時，光明皇后亦好書道，她曾臨習過[[中國|中國]][[王羲之|王羲之]]的「樂毅論」之帖，而光明皇后所臨之作品也被保存在[[正倉院|正倉院]]之中，另外在[[萬葉集|萬葉集]]中，光明皇后也有四首[[和歌|和歌]]傳世。

== 傳說 ==
[[大阪府|大阪府]][[和泉市|和泉市]]和[[堺市|堺市]][[南區_(堺市)|南區]]中的一座池子名[[光明池|光明池]]，傳說是因為該地乃光明皇后的誕生地，因此該池以“光明”名之。此外還有一個光明皇后為病人吸膿的傳說。

當時光明皇后曾廣設澡堂，以便百姓洗浴和改善衛生習慣，並發願為一千位不方便沐浴的人民服務。
約莫兩年之後，到了第一千位時，在場所有的人都大為震驚，第一千位人民是一位老人，而且全身潰爛惡臭，頭髮鬍鬚也都掉光了，這位老人對光明皇后開口說：「我的病已經很多年了，不論如何醫治都絲毫不見起色，而曾有一位醫生告訴我，只要有一位高貴良善的人願意為我吸出身上的膿血，就可以痊癒了」

皇后聞言，便問：「這是真的嗎？你的病只要這樣做就可以痊癒嗎？」老人回答：「是的，如此一來我就能脫離病痛的折磨。」於是皇后便跪下，開始替這位老人吸出身腳上的膿血，霎時間，這位老人全身大放光明又充滿奇異的香氣，原來這位老人是[[東方琉璃光世界|東方淨琉璃世界]]的[[藥師佛|藥師如來]]之化身（在某些版本中，這位老人是[[觀世音菩薩|觀世音菩薩]]的化身）。而傳說中出現的那間浴室，現存於[[奈良|奈良]][[法華寺|法華寺]]，又據說，光明皇后面貌美麗溫柔，[[法華寺|法華寺]]內本尊[[十一面觀音像|十一面觀音像]]，據說就是以光明皇后為藍本所雕塑而成。
