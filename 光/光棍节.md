'''光棍节'''（11月11日），又稱'''單身節'''，是流行于[[中華人民共和國|中華人民共和國]][[年轻人|年轻人]]的娱乐性节日，以自己仍是[[单身|单身]]一族为傲（「[[光棍|光棍]]」的意思便是「单身」）。它起源于网络文化和校园文化，此前也被称为'''光光节'''。同时，这一天也是众多为单身男女脱离光棍状态而举办交友聚会活动的日子，因此又同时是'''脱光节'''（義為「脫離光棍單身的節日」）。与此同时，也是各大商家（特别是线上购物）以脱光为由打折促销的时期。從2009年11月11日開始，购物网站[[淘寶|淘寶]]及其子品牌[[天貓|天貓]]為首的商家将该日宣傳為「[[双十一|双十一]]狂歡購物節」，随后其他电商也纷纷加入，光棍节逐渐演变成网络购物节日。<ref>[http://finance.chinanews.com/it/2012/11-11/4319008.shtml “光棍节”成电商狂欢节 淘宝天猫销售破百亿] {{Wayback|url=http://finance.chinanews.com/it/2012/11-11/4319008.shtml |date=20121113211423 }}.中新网.2012年11月11日</ref>

==来历==
光棍节是非官方、非传统节日，盛行于网络及民间。它的来历有多种说法，广为认可的是起源于[[南京大学|南京大学]]。[[1993年|1993年]]，南京大学「名草无主」[[寝室|寝室]]四个大四学生每晚举行「卧谈」，一段时间卧谈的主题都是讨论如何摆脱光棍状态，卧谈中创想出了以即将到来的11月11日（数字1象征「单身」）作为「光棍节」来组织活动。从此，光棍节逐渐发展成为南京高校以至各地大学里的一种校园趣味文化<ref>{{Cite news|url = http://news.sina.com.cn/o/2012-11-11/153125556116.shtml|title = 光混节来历说法不一 或起源于南京大学宿舍文化|last = |first = |date = 2012-11-11|work = |accessdate = 2015-11-17|publisher = 龙虎网}}</ref>。随着一批批学子告别校园，这个节日被渐渐带入社会，并随着成年单身男女群体的庞大，以及群体活动和网络媒体的传播，光棍节在社会流行开来，并由光棍节发展出了「脱光节」。

==活動==
光棍节这一天没有固定的庆祝方式和规范的仪式，不同人群会以不同方式庆祝这一节日。但庆祝活动主要集中在聚会、狂欢、购物、[[相亲|相亲]]或结婚等几个方面。<ref>{{Cite news|url = http://www.chinanews.com/cul/2012/11-12/4320128.shtml|title = “光棍节”是怎样炼成的？源于青年自嘲纠结心理|last = |first = |date = 2012-11-12|work = |accessdate = 2015-11-17|publisher = 中国青年报}}</ref>在饮食方面常常以2根油条1个鸡蛋或4根油条1个面包作为节日套餐；购买有“光棍”“1111”象征意义符号的物品收藏或赠送，如以筷子作为节日礼物赠送；在短信及[[微博|微博]]、[[微信|微信]]等社交平台互发以“光棍”“脱单”为主题的笑话、祝福等<ref>林锦凤, 魏卫. [http://d.wanfangdata.com.cn/Periodical/qzsfxyxb201104018 基于社会文化视角的光棍节节日仪式分析][J]. 泉州师范学院学报, 2011, 第4期:72-76.</ref><ref>王璐. [http://www.cnki.com.cn/Article/CJFDTotal-QNYJ201403009.htm 11月11日:从文化建构到商业收编——对“光棍节”和“网购狂欢节”的分析][J]. 青年研究, 2014, 03期:74-83.</ref>；有为脱离单身状态而参加交友或相亲活动的，被称作为“脱光”<ref>{{Cite news|url = http://news.hunantv.com/x/w/20081111/63436.html|title = 光棍节：趣味文化衍生的集体“脱光日”|last = |first = |date = 2008-11-11|work = |accessdate = 2015-11-17|publisher = 金鹰网}}</ref>；已婚人群又以具有两对偶的4個1代表「一夫一妻，一双一家」和「一生一世，一心一意」而以11月11日这一天作为夫妻“忠偶节”来庆祝，而不少新人特意选在这天领证结婚。<ref>{{Cite news|url =http://culture.inewsweek.cn/20130725/detail-66646-all.html|title =大学生“另类”节日 你是如何庆祝的？|last =|first =|date =2013-07-25|work =|accessdate =2015-11-17|publisher =中国新闻周刊}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>有些青年人则将这一天祝贺当作自己可以脱离父母的照顾和资助而能够自食其力自己独立生活的日子。同时，各种企业特别是电子商务企业有许多相关的商业促销及优惠活动。<ref>{{cite web|url=http://www.storm.mg/article/73469|title=1111光棍節「雙11」中國購物狂歡提前上演-風傳媒|first=|last=Storm.mg|date=|publisher=|accessdate=2018-10-29}}</ref>
