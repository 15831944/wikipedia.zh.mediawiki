'''光华教育基金会'''(Kwang-Hua Education Foundation) 是1989年由著名国学大师[[南怀瑾|南怀瑾]]先生發起、[[台湾|台湾]][[润泰集团|润泰集团]]本着「光大[[中华文化|中华文化]]」的宗旨，通过设立[[奖学金|奖学金]]、投资助学项目等形式资助中国教育事业的[[基金会|基金会]]<ref name= "南大"/>。集团设立的其他基金会还包括财团法人纪念尹珣若先生教育基金会、财团法人尹书田纪念医院、光华科技基金会。

== 资助历史 ==
至2010年基金会已先后捐赠10亿元人民币，在中国30多所高校设立了[[奖学金|奖学金]]，资助了超过10万名[[大学生|大学生]]，为中国教育事业做出了巨大贡献<ref name= "南大"/>。

1989年，光华教育基金会在[[北京大学|北京大学]]设立'''光华奖学金'''。基金会自1995年起长期支持[[北京大学光华管理学院|北京大学光华管理学院]]的发展。

2006年9月25日，基金会捐资一亿元人民币予[[浙江大学|浙江大学]]，以支持[[法学院|法学院]]的建设，[[浙江大学光华法学院|浙江大学光华法学院]]因此冠名成立。<ref>{{cite web |url=http://www.zuef.zju.edu.cn/web/show/2852 |language= |title=光华奖学金简介 |publisher=浙江大学教育基金会 |date=2011-01-01 |accessdate= }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2009年4月3日，基金会向[[西安交大|西安交大]]捐赠一亿元人民币<ref>{{cite web|url=http://www.huaxia.com/zjsx/stwl/2009/04/1391284.html | language= | title=台湾光华教育基金会向西安交大捐资亿元  |publisher=华夏经纬网 |date=2009-04-15 |accessdate= }}</ref>，用于[[医学|医学]]学科的建设与发展，[[西安交通大学医学院|西安交通大学医学院]]成为基金会在大陆资助的唯一一所医学院<ref>{{cite web |url=http://xjtunews.xjtu.edu.cn/mtjd/2010-06/1276333763d27492.shtml |language= |title=光明日报：西安交大与光华教育基金会共谱医学教育新篇章 |publisher=光明日报 |date=2010-06-12 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20120710024211/http://xjtunews.xjtu.edu.cn/mtjd/2010-06/1276333763d27492.shtml |archivedate=2012-07-10 }}</ref>。

2010年4月20日，基金会与[[南京大学|南京大学]]签署协议，双方将在5年内共同出资6000万元人民币设立“南京大学人文基金”，以促进人文学科的发展<ref name="南大">{{cite web |url=http://www.js.xinhuanet.com/xin_wen_zhong_xin/2010-04/20/content_19573111.htm |language= |title=台湾光华教育基金会与南大协议设立基金 |publisher=新华网江苏频道 |date=2010年4月20日 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150418071542/http://www.js.xinhuanet.com/xin_wen_zhong_xin/2010-04/20/content_19573111.htm |archivedate=2015年4月18日 }}</ref>。
