'''光熙門站'''位于[[北京市|北京市]][[朝阳区|朝阳区]]，是[[北京地铁|北京地铁]][[北京地铁13号线|13号线]]的一个车站。车站编号是1314。

== 位置 ==
地铁光熙门站位於[[北三环东路_(北京)|北三环东路]]和[[京承高速公路|京承高速公路]]相交的[[太阳宫桥|太阳宫桥]]南面，原[[望和支线|望和支线]][[和平里站|和平里站]]站址北端。

== 结构 ==
本站在采用两层高架车站，一层为站厅，二层为站台

== 站名由来 ==
光熙门站在规划建设阶段曾称“和平里站”，由于接近[[元大都|元大都]]光熙门原址和光熙门北里居住区而更名为“光熙门站”。<ref>{{cite news|url=http://beijing.21dnn.com/3925/2002-6-27/118@304525.htm|title=好找好记不重名 城铁站名正式亮相|date=2002-06-27|accessdate=2012-12-09|author=莫春|deadurl=yes|archiveurl=https://web.archive.org/web/20160304110649/http://beijing.21dnn.com/3925/2002-6-27/118@304525.htm|archivedate=2016-03-04}}</ref>
