《'''光海，成为王的男人'''》（{{kor|k=광해, 왕이 된 남자||hanja=光海, 王이 된 男子}}，{{lang-en|'''''Masquerade'''''}}）是2012年9月13日上映于[[韩国|韩国]]的古装電影，韓國第7部超過1000萬觀影人次的本土電影。在第49屆[[大鐘獎|大鐘獎]]奪得最優秀作品獎、最佳男主角獎、導演獎等等15個獎項。<ref>{{cite web |url= http://news.naver.com/main/read.nhn?mode=LSD&mid=shm&sid1=106&oid=109&aid=0002434915 |title= '광해', 대종상 최우수작품상 등 15관왕 싹쓸이(종합)|accessdate=2012-10-30 |publisher=OSEN |work= |language =ko}}</ref>

== 剧情介紹 ==
古代[[朝鲜|朝鲜]]时期，为了逃避被毒杀的命运，光海君想出一个主意，找与自己相貌相像的平民當盾牌，两个人因故互换身份15天。

== 票房 ==
據影片投資和發行公司CJ娛樂統計，2012年10月20日（上映第38天）下午8時觀影人數突破1000萬人次，成為韓國本土電影史上第7部、2012年繼《[[盜賊門|盜賊門]]》後第2部觀影人數突破千萬的影片。<ref>{{cite web |url= http://chinese.yonhapnews.co.kr/entertain/2012/10/21/0502000000ACK20121021001100881.HTML|title= 《光海》觀影人數破千萬 成韓國第七部最賣座電影|accessdate=2012-10-21 |publisher=韓聯社 |work= |language =zh}}</ref>韓國電影首度在一年內誕生兩部突破千萬觀影人數的影片，創下韓國影史新紀錄。截止12月12日，以1230萬4000人次的觀影總數超越2005年《[[王的男人|王的男人]]》所保持的紀錄。<ref>{{cite web |url= http://news.nate.com/view/20121123n10601 |title= '광해' 1,200만 관객 돌파…'왕의 남자' 바짝 추격|accessdate=2012-11-23 |publisher= |work= |language =ko}}</ref><ref>{{cite web|url=http://www.koreastardaily.com/tc/news/23563 |title=《光海》超越《王的男人》成韓歷史票房第三|work=韓星網 |accessdate=2012-12-12 |language =zh}}</ref>。最終觀影人次為12,319,542人次<ref>{{cite web|url=http://www.kobis.or.kr/kobis/business/stat/offc/findFormerBoxOfficeList.do|title=역대 박스오피스 (공식통계 기준)|publisher=영화진흥위원회|accessdate=2018-08-03}}</ref>。

== 花絮 ==
2012年8月13日，电影的影片制作发布会在[[首尔市|首尔市]]举行。<ref>{{cite web |url=http://ent.163.com/12/0816/08/8913A8HT000300B1.html |title=《光海：成为王的男人》开发布会 韩孝珠秀美腿 |accessdate=2012-08-16  |publisher=网易 |work= |language =zh-cn}}</ref>

电影中融入[[喜剧|喜剧]]诙谐因素。<ref>{{cite web |url= http://ent.nfdaily.cn/content/2012-09/21/content_55218351.htm|title= 李秉宪《光海，成王的男人》演绎另类诙谐风情|accessdate=2012-09-21 |publisher=南方网 |work= |language =zh-cn}}</ref>

2012年10月13日晚間，為了慶祝觀影人數即將突破1000萬，導演[[秋昌民|秋昌民]]和主要演員[[李炳憲|李炳憲]]、[[柳承龍|柳承龍]]和[[韓孝周|韓孝周]]等人穿上片中的戲服，在首爾三成洞COEX會展中心舉辦了影迷答謝會。<ref>{{cite web |url= http://china.joins.com/big5/article.do?method=detail&art_id=93720 |title= 「光海」票房即將突破千萬演員身穿演出服參加粉絲見面會|accessdate=2012-10-15 |publisher=韓國中央日報中文網 |work= |language =zh}}</ref>
