'''衣魚'''（[[學名|學名]]：''{{lang|la|Lepisma saccharina}}''），俗稱'''蠹{{可疑|1=蠹|reason=蠹是蛀蟲}}'''、'''蠹魚'''、'''白魚'''、'''壁魚'''、'''赤木蟲'''、'''書蟲'''或'''衣蟲'''，古名'''蛃'''(音同“丙”)，是一種靈巧、怕光、而且無翅的[[昆虫|昆虫]]，身體呈銀灰色，因此也有'''白魚'''的稱號，嗜食[[食糖|糖]]類及[[淀粉|澱粉]]等[[糖类|碳水化合物]]，牠的體型細長，無翅，身體上佈滿了[[鳞|鱗片]]，[[口器|口器]]為咀嚼式。

衣魚在昆蟲界的分類是：[[衣魚目|衣魚目]]；[[總尾目|總尾目]]，在[[地球|地球]]上已經出現約三億年。

== 外形 ==
不計[[肢體|肢體]]，衣魚的身長約一[[釐米|釐米]]。[[觸角|觸角]]又長又嫩，只有三節體節有足；[[蛻皮動物|脫皮]]三次後，銀灰色的鱗片便會長成，使其身體帶有一種[[金屬|金屬]]般的[[光澤|光澤]]。

頭部有細長30節以上的絲狀觸角，多數有明顯的小型[[複眼|複眼]]，[[腹部|腹部]]有三對能疾走、[[跳躍|跳躍]]的腳，行動敏捷，[[尾部|尾部]]有三根[[尾巴|尾巴]]，因此被稱為「總尾目」或「[[纓尾目|纓尾目]]」。

== 生長 ==
按不同生活環境而定，衣魚從[[幼蟲|幼蟲]]變[[成蟲|成蟲]]需要至少四個月的時間，不過有時候發育期會長達三年。在[[室溫|室溫]]環境下，大概一年就發育為成蟲，壽命約為兩到八年。一條衣魚的一生裡會經歷大約八次[[蛻皮動物|脫皮]]；不過衣魚不斷生長，一年脫皮四次也不足為奇。當溫度在[[攝氏|攝氏]]25至30度，雌蟲就會在物件的裂縫裡產大約一百顆[[卵|卵]]，當溫度過高，衣魚也無法存活；可是在寒冷或乾燥的環境下，衣魚是不會交配的，衣魚較喜好於潮濕的環境下生殖。

== 食物 ==
衣魚愛好的[[食物|食物]]為充滿[[淀粉|澱粉]]質或[[多糖|多糖]]的物品，如：[[黏合劑|膠水]]裡的[[葡聚糖|葡聚糖]]、[[糊精|漿糊]]、[[图书|書籍裝訂物]]、[[相片|照片]]、[[食糖|糖]]、[[體毛|毛髮]]、[[土壤|泥土]]等。可是牠們對[[棉花|棉花]]、[[亚麻|亞麻布]]、[[丝绸|絲]]和[[人造纖維|人造纖維]]等也毫不抗拒，甚至連其他[[昆虫|昆虫]]屍體、自己脫的[[皮膚|皮]]也是照吃如饴。飢餓時甚至連[[皮革|皮革]]製品、[[人造纖維|人造纖維]]布匹等也吃。不過衣魚能夠捱餓數個月，身體機能也不會受任何傷害。

== 生活環境 ==
[[File:Bruce_Chatwin's_Songlines_1987.jpg|thumb]]
[[File:A_silverfish_at_ONE-iAdvantage.jpg|thumb]]
如[[斑衣魚|斑衣魚]]（''Thermobia domestica''）一樣，衣魚出沒於人類聚居地。家居裡的各種地方，如[[冰箱|冰箱]]底部、開[[暖气|暖气]]的[[浴室|浴室]]、[[地板|地磚]]的裂縫裡都可能會有衣魚的蹤影。衣魚喜歡咬破[[图书|書籍]]、[[纖維|纖維]]及[[纺织品|纺织品]]。而斑衣魚則喜歡如麵包店等更溫暖的環境，牠們非常喜歡進食[[麵粉|麵粉]]和[[麵包|麵包]]，不過也會進食[[動物|動物]]製品。

== 繁殖 ==
[[File:Lepisma_saccharina.jpg|thumb]]
交配時，雄蟲跟雌蟲會到處竄動。雄蟲會產下一個用[[膜|薄紗]]包住的[[精囊|精囊]]；由於生理狀態成熟，雌蟲會找到該精囊，拾取作[[受精|受精]]用。

== 天敵 ==
衣魚最有名的天敵是[[蠼螋|蠼螋]]。

衣魚為防止[[蜘蛛|蜘蛛]]、[[蠅虎科|蠅虎]]等天敵捕食，停息時總是不停地擺動著尾梢，誘使天敵將注意力集中到尾梢上來，當尾巴被天敵抓住，分節的尾毛即斷掉，主身體便可乘機逃脫。

== 防治 ==
衣魚雖然對人類生活造成滋擾，但其實是無害的。

在建築物裡，衣魚必須要有潮濕及有空隙的環境才能生存；只要環境乾燥、建築物沒有裂縫，衣魚就會自然消失。以下一些撲滅或消除衣魚的方法，雖然有效，但都是治標不治本的：

* 混合比例為1:1的[[硼砂|硼砂]]和[[食糖|砂糖]]，能有效殺除衣魚。
* [[氯化铵|氯化铵]]水的氣味應該能於24小時內驅趕衣魚。
* 若要捕捉衣魚，將[[石膏|石膏]]粉灑在浸濕的白棉布上，隔夜放在房間一角，放的地方要接近衣魚的藏身處。
* 以下方法同樣有用：可以在衣魚藏身處旁邊放一塊木板，板上再放一顆稍微磨碎的馬鈴薯；衣魚晚上出沒時就會鑽進馬鈴薯裡面大快朵頤，次天早上，你就可以把馬鈴薯連同衣魚一起丟掉。
* 使用[[樟腦丸|樟腦丸]]（或[[萘|萘]]丸）可以讓衣魚不敢靠近。

在中國古代，讀書人家會將一種叫[[芸香属|芸香草]]的植物放在自己的書櫃，能夠驅除書蟲，且會發出一種淡淡的清香，所以一直到今還在延續“書香門第”這一說法。

現代人驅除書蟲用[[樟腦丸|樟腦丸]]，或[[芳香療法|香薰]][[精油|精油]]，也有人用[[辣椒|乾辣椒粉]]和[[蒜|蒜頭]]。書蟲名稱又叫書蠹，書蠹是蠹蟲龐大家族的一種。
