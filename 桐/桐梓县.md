'''桐梓县'''位于[[中華人民共和國|中國]][[贵州省|贵州省]]北部，是[[遵义市|遵义市]]下辖的一个县。县政府驻[[娄山关街道|娄山关街道]]，邮政编码563200。

== 历史沿革 ==
* 唐[[贞观|贞观]]十六年（642年），于桐梓地中部、北部置[[播州|播州]]，遂改称珍州，置[[夜郎|夜郎]]、丽皋、乐源3县。
* 唐[[唐僖宗|僖宗]][[乾符|乾符]]三年（876年），杨端入播，蒙山以南遂属播州杨氏据有，直至北宋末年。
* [[南宋|南宋]][[宋度宗|度宗]][[咸淳|咸淳]]九年（1273年）又改原珍州并属县隶播州。
* [[元代|元代]]，桐梓地先为鼎山县，后改播川县，治设鼎山城，属播州[[宣慰使|宣慰使]]司。元代最后8年曾随播州附[[重庆|重庆]]自立为帝的[[明玉珍|明玉珍]]。
* 明[[万历|万历]]二十九年（1601年），置桐梓县，隶于遵义军民府（后改称遵义府），归四川[[布政使|布政使]]司辖制。
* 清[[雍正|雍正]]六年（1728年），随遵义府改隶于贵州省至今。
* [[民国|民国]]时期，桐梓县曾先后隶属于贵州省黔中道和第五行政督察区，行署两度设桐梓。
* [[中华人民共和国|中华人民共和国]]建立后，隶于贵州省[[遵义专区|遵义专区]]、[[遵义地区|遵义地区]]。1955年，桐梓县第十区全区划为[[重庆市|重庆市]][[南桐礦區|南桐礦區]]。

== 地理 ==
* 桐梓县位于渝黔结合部，[[大娄山|大娄山]]及其支脉[[蒙山|蒙山]]绵延辖区内。
* 全县南北最长处81公里，东西最宽处52公里，总面积3189.8平方公里。
* 桐梓县东邻[[正安|正安]]县、[[绥阳|绥阳]]县；南接[[遵义|遵义]]县、[[仁怀|仁怀]]县；西界[[习水|习水]]县，西北抵重庆市[[綦江|綦江]]县；北与重庆市[[万盛|万盛]]区及[[南川|南川]]市毗连。

== 人口 ==
*全县总人口为64.06万人（2003年），其中城镇非农业人口7.57万人，乡村人口58.15万元。
*全县人口出生率为11.7‰，死亡率5.46‰，人口自然增长率6.24‰。

== 交通 ==
*[[川黔铁路|川黔铁路]]、[[渝貴铁路|渝貴铁路]]、[[210国道|210国道]]及[[渝黔高速公路|渝黔高速公路]]从南至北贯穿全境。

== 经济 ==
*全年完成生产总值（GDP）18.47亿元（2003年）。
*财政总收入1.08亿元，地方财政收入7256万元。
