'''桐谷茉莉'''（{{lang-ja|桐谷まつり}}，1996年8月15日 - ），[[日本|日本]]的[[AV女優|AV女優]]<ref name="C-more"/>，[[秋田縣|秋田縣]]出身。曾為「[[Soft_On_Demand|SOD]]」專屬女優<ref name="SOD">{{Cite web |url=https://ec.sod.co.jp/special/sodstar/kiritani_matsuri.html |title=桐谷まつり SODstar |publisher=[[Soft_On_Demand|ソフト・オン・デマンド]] |accessdate=2018-11-23 |language=ja}}</ref>。現為[[OPPAI|OPPAI]]的專屬女優。所屬於「C-more Entertainment」事務所<ref name="C-more"/>。

== 人物 ==

== 簡歷 ==
2016年12月，在日本[[AV片商|AV片商]]「[[Soft_On_Demand|SOD Create]]」以作品《SODstar マジックミラー号誕生20周年記念作品 桐谷まつり AV Debut》成為專屬女優[[AV|AV]][[出道|出道]]<ref>{{cite web|url=http://www.playno1.com/article-21702-1.html#.W_byEugzaUk |title=桐谷まつり：SOD的超超超級秘密武器現身！ |publisher=PLAYNO.1玩樂達人 |date=2016-11-11}}</ref>。

2018年2月，被提名《[[DMM成人獎|DMM成人獎]]》2018「最優秀女優獎」<ref>{{Cite web |url=http://www.dmm.co.jp/adultaward/archive/2018/nominate_html/=/navi=none/ |title=DMMアダルトアワード2018 ノミネート女優 |publisher=[[DMM.com|FANZAアダルトアワード]] |date=2018-02-17 |language=ja}}</ref>。

2018年6月29日，成為「[[惠比壽麝香葡萄|惠比壽麝香葡萄1.5]]」的[[訓練生|訓練生]]<ref>{{Cite web |url=https://news.dwango.jp/idol/29572-1806 |title=恵比寿マスカッツ1.5、新メンバー21名が決定 |publisher=[[多玩國|ドワンゴ]] |date=2018-06-29 |accessdate=2018-11-23 |language=ja}}</ref>。

2018年8月10日，與[[橋本有菜|橋本有菜]]、{{link-ja|君色華奈|君色花音}}等人宣布成立[[偶像_(日本)|偶像團體]]「un panopticon girl」（アン・パノプティコンガール）<ref>[https://twitter.com/un_pano_girl/status/1027781592322793472 @un_pano_girl 下午1:00 - 2018年8月10日]</ref>。

2018年10月3日，在AV女優專門的[[獨立音樂|獨立]][[唱片公司|唱片公司]]「{{link-ja|MILKY POP GENERATION|Milky Pop Generation}}」發布CD出道<ref>{{Cite news|url=https://news.nifty.com/article/entame/showbizd/12206-163129/ |title=本日10/13(土)アイドルユニット“パノガ”がインストアライブを開催！|work=niftyニュース |date=2018-10-13 |language=ja}}</ref>。

2018年12月19日，在個人[[推特|推特]]上宣布自「SODstar」畢業，同時發表畢業作品<ref>[https://twitter.com/matsuri_kiri/status/1075355606410620929 @matsuri_kiri 下午7:42 - 2018年12月19日]</ref>。
