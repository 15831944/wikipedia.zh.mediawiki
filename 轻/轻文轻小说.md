'''轻文轻小说'''（简称'''轻文'''）是武汉空文网络科技有限公司<ref>{{Cite web|url=https://www.tianyancha.com/company/1483606897|title=武汉空文网络科技有限公司 工商登记注册资料|accessdate=2018-06-24|author=|date=|publisher=天眼查}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>旗下运营的一个定位于正版原创动漫轻小说内容分享互动社区，成立于2015年7月。<ref>{{Cite web|url=https://www.iqing.com/info/#/info|title=关于我们 - 轻文轻小说|accessdate=2018-06-24|author=|date=|publisher=轻文轻小说|language=zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180626094641/https://www.iqing.com/info/#/info|archivedate=2018-06-26|df=}}</ref>

目前主要提供的服务是支持用户创作的[[轻小说|轻小说]]及[[漫画|漫画]]等原创作品发布、引进境外正版轻小说漫画付费内容发布、社区交流互动<ref>{{Cite web|url=https://www.iqing.com/info/#/faq|title=轻文FAQ - 轻文轻小说|accessdate=2018-06-24|author=|date=|work=|publisher=轻文轻小说|language=zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180626094641/https://www.iqing.com/info/#/faq|archivedate=2018-06-26|df=}}</ref>。此外，轻文轻小说开发了名为轻文IQA编辑器的[[视觉小说|视觉小说]]制作引擎<ref>{{cite web |title=简单上手易操作 新一代的视觉小说游戏制作工具——轻文IQA编辑器 |url=http://comic.sina.com.cn/guonei/2018-05-24/doc-ihaysviy0824579.shtml |website=新浪动漫 |accessdate=2018-06-16 |date=2018-05-24 |deadurl=no |archiveurl=https://web.archive.org/web/20180616114833/http://comic.sina.com.cn/guonei/2018-05-24/doc-ihaysviy0824579.shtml |archivedate=2018-06-16 |df= }}</ref>。用户可以通过PC网页版、手机网页版以及客户端app（支持[[Android|Android]]和[[iOS|iOS]]系统）访问轻文轻小说网站。

== 运营历史 ==
轻文轻小说的创建者朱周易曾是中国大陆[[弹幕视频网站|弹幕视频网站]][[AcFun|AcFun]]的[[首席执行官|首席执行官]]，离开AcFun后，他跟AcFun的一些老员工创立了轻文轻小说，以轻小说作为主要内容，并采用了[[用户生成内容|用户生成内容]]的运营模式以提高社区活跃度和促进创作。<ref>{{cite web |title=离开A站的朱周易创办了离开A站的朱周易创办了轻文轻小说，主攻王道系长篇少年向作品 |url=http://money.163.com/16/0423/08/BLATP5H700253B0H.html |website=网易财经 |publisher=钛媒体(北京) |accessdate=2018-06-16 |date=2016-04-23 |deadurl=no |archiveurl=https://web.archive.org/web/20180616114832/http://money.163.com/16/0423/08/BLATP5H700253B0H.html |archivedate=2018-06-16 |df= }}</ref>

2017年6月23日，轻文轻小说与爱奇艺文学联合举办爱奇艺轻小说大赛暨第二届“星耀祭”征文活动。<ref>{{Cite web|url=http://comic.qq.com/a/20170623/021594.htm|title=轻文轻小说携手爱奇艺文学展开耀星祭征文活动，以文字的力量打破次元壁|accessdate=2018-06-24|author=|date=2017-06-23|publisher=腾讯动漫|deadurl=no|archiveurl=https://web.archive.org/web/20180626094641/http://comic.qq.com/a/20170623/021594.htm|archivedate=2018-06-26|df=}}</ref>

== 融资 ==
2015年9月，轻文轻小说完成了由九合创投、掌趣创享领投的，数百万元人民币的天使轮融资。<ref name=":0">{{Cite web|url=http://www.jiemian.com/article/2038599.html|title=轻文获千万元融资，爱奇艺领投|accessdate=2018-06-24|author=三文娱|date=2018-04-06|work=|publisher=界面新闻|language=zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180626094643/http://www.jiemian.com/article/2038599.html|archivedate=2018-06-26|df=}}</ref>

2016年9月，轻文轻小说完成了由高榕资本和[[Bilibili|哔哩哔哩]]领投的，千万元人民币的A轮融资。<ref name=":0" />

2018年4月，轻文轻小说完成了由[[爱奇艺|爱奇艺]]领投、武汉光谷人才基金跟投的，千万元人民币的B轮融资。<ref name=":0" />

== 争议 ==
2018年5月，轻文轻小说被网友指出引进了具有美化[[日本侵华战争|日本侵华战争]]倾向的日本轻小说《[[在异世界开拓第二人生|在异世界开拓第二人生]]》，随后轻文轻小说下架了这部轻小说并对内容进行重新审查，发现作者在提供翻译母本时已经事先删减了具有争议性的内容，而审查员在审查时并未阅读过原版，故未能发现<ref>{{cite web|author1=郭光昊|title=轻小说主人公疑是屠杀三千人侵华日军 日作者道歉|url=http://www.guancha.cn/internation/2018_06_05_459041.shtml|website=观察者网|accessdate=2018-07-05|date=2018-06-05|deadurl=no|archiveurl=https://web.archive.org/web/20180614002259/http://www.guancha.cn/internation/2018_06_05_459041.shtml|archivedate=2018-06-14|df=}}</ref>。随后轻文轻小说也发布公告，称已终止跟作者まいん的合作并要求其道歉<ref>{{cite web|author1=王诗诗|title=美化侵华日军 轻文轻小说全面下架《在异世界开拓第二人生》|url=http://www.donews.com/news/detail/3/3003637.html|website=DoNews|accessdate=2018-07-05|date=2018-05-31|deadurl=no|archiveurl=https://web.archive.org/web/20180712160251/http://www.donews.com/news/detail/3/3003637.html|archivedate=2018-07-12|df=}}</ref>。
