'''轻重量级'''（{{lang-en|light heavyweight}}）是搏击运动的[[体重级别|体重级别]]之一。

== 拳击 ==
在业余拳击中，轻重量级位于[[中量级|中量级]]与[[重量级|重量级]]之间，划分标准为165磅（75公斤）至178磅（81公斤）。而在职业拳击中，轻重量级则位于[[超中量级|超中量级]]与[[巡洋舰级|巡洋舰级]]之间，划分标准为168磅（76公斤）至175磅（79公斤）。{{le|达里乌兹·米切尔泽维斯基|Dariusz Michalczewski}}保持着职业拳击该级别连续卫冕的纪录，他共23次成功卫冕[[世界拳击组织|世界拳击组织]]（WBO）轻重量级拳王。

== 踢拳 ==
在[[踢拳|踢拳]]运动中，不同组织对中量级的划分标准并不相同，如[[国际踢拳联合会|国际踢拳联合会]]的标准为171磅至180磅（77公斤至82公斤），而[[Glory|Glory]]则规定为209磅（95公斤）以下。<ref>[http://www.gloryworldseries.com/en/fighters/rankings/lhw OFFICIAL GLORY RANKING] {{webarchive|url=https://web.archive.org/web/20140110053417/http://www.gloryworldseries.com/en/fighters/rankings/lhw |date=2014-01-10 }}</ref>

== 综合格斗 ==
在[[综合格斗|综合格斗]]中，轻重量级的标准为186磅至205磅（84公斤至93公斤）。
