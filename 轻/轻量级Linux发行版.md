'''輕量級Linux發行版'''是使用相對較少電腦資源的[[Linux发行版|Linux发行版]]。舉例來說，只需要128 MB的[[随机存取存储器|RAM]]及[[奔腾II|奔腾II]]處理器的[[Lubuntu|Lubuntu]]是[[Ubuntu|Ubuntu]]的輕量級變體，Ubuntu需要1 GHz的處理器及1 GB的RAM<ref name="Lubuntu">{{cite web|url = http://lubuntu.net/|title = Lubuntu|accessdate = 14 December 2010|last = Lubuntu Developers|authorlink = |date=December 2010}}</ref><ref name="Lubuntu1010Release">{{cite web|url = http://lubuntu.net/newsletter/lubuntu-1010-released|title = lubuntu 10.10 released|accessdate = 14 December 2010|last = Behling|first = Mario|authorlink = |date = October 2010|deadurl = yes|archiveurl = https://web.archive.org/web/20101016002814/http://lubuntu.net/newsletter/lubuntu-1010-released|archivedate = 16 十月 2010}}</ref><ref name="UbuntuHardware">{{cite web|url = https://help.ubuntu.com/community/Installation/SystemRequirements|title = Recommended Minimum System Requirements|accessdate = 14 December 2010|last = [[Canonical_Ltd|Canonical Ltd]] |date=October 2010}}</ref>。

== 被認為較輕量的發行版 ==

*[[Alpine_Linux|Alpine Linux]] － 注重安全的發行版，使用了[[uClibc|uClibc]]及[[BusyBox|BusyBox]]因而較輕量<ref>{{cite web|url=http://wiki.alpinelinux.org/wiki/Main_Page |title=Alpine Linux Wiki |accessdate=2012-11-07}}</ref>。最小下載大小是60 MB；基本系統（包含可執行核心）在5 MB以下。
*{{link-en|antiX|antiX}} － [[MEPIS|MEPIS]] Linux的輕量級版本，基於[[Debian|Debian]] testing分支。核心安裝：128 MB，基本系統安裝：300 MB。軟體包管理器：[[Synaptic|Synaptic]]<ref>{{cite web|url=http://antix.mepis.org |title=Main Page - antiX |publisher=Antix.mepis.org |date= |accessdate=2012-05-28}}</ref> 。
*[[BasicLinux|BasicLinux]] － 可以在[[Intel_80386|Intel 80386]]及3 MB的RAM下運行<ref name="Nabble">{{cite web|url = http://old.nabble.com/BL-on-CF-IDE-drive-td25683600.html|title = BL on CF IDE drive|accessdate = 16 January 2011|last = Keesan|first = Sindhi|authorlink = |date = October 2009|deadurl = yes|archiveurl = https://web.archive.org/web/20111009120921/http://old.nabble.com/BL-on-CF-IDE-drive-td25683600.html|archivedate = 9 十月 2011}}</ref><ref name="BL">{{cite web|url = http://distro.ibiblio.org/pub/linux/distributions/baslinux/|title = BasicLinux|accessdate = 16 January 2011|last = [[BasicLinux|BasicLinux]]|authorlink = |year = undated}}</ref>。
*[[Bodhi_Linux|Bodhi Linux]] － 基於[[Ubuntu|Ubuntu]]長期支援版本的輕量級發行版<ref>{{cite web|url = http://www.ubuntuvibes.com/2011/01/bodhi-linux-is-lightweight-linux.html|title = Bodhi Linux is a Lightweight Linux Distribution|accessdate = 2011-05-05|author=Nitesh|publisher=Ubuntu Vibes}}</ref>。
*[[CrunchBang_Linux|CrunchBang Linux]] － 相對較大的(771 MB)基於[[Debian|Debian]]的，主要設計來加強速度及易用性的發行版<ref>{{cite web|url = http://www.makeuseof.com/tag/crunchbang-a-lightweight-os-perfect-for-old-and-new-computers-alike/|title = CrunchBang: A Lightweight OS Perfect For Old And New Computers Alike|accessdate = 2013-04-18|author=Justin Pot|publisher=MakeUseOf}}</ref><ref>{{cite web|url = http://desktoplinuxreviews.com/crunchbang-reviews/crunchbang-11-waldorf/|title = CrunchBang 11 Waldorf|accessdate = 2013-05-21|author = Jim Lynch|publisher = Desktop Linux Reviews|deadurl = yes|archiveurl = https://web.archive.org/web/20130522193322/http://desktoplinuxreviews.com/crunchbang-reviews/crunchbang-11-waldorf/|archivedate = 2013-05-22}}</ref>。
*[[Damn_Small_Linux|Damn Small Linux]] － 下載大小約50 MB。額外的軟體在“DSL擴展”中提供。擴展是使用必需額外安裝的Debian的APT工具。“輕到足以在486DX及16MB的RAM上執行”<ref name="SliTaz">{{Cite news|url = http://www.linuxinfusion.com/slitaz-a-light-weight-gnulinux-distribution|title = SliTaz: A light-weight GNU/Linux distribution|accessdate = 11 December 2010|last = Moparx|authorlink = |date=April 2008| work = Linux Infusion}}</ref><ref name="DSL">{{cite web|url = http://www.damnsmalllinux.org/|title = What is DSL?|accessdate = 11 December 2010|last = [[Damn_Small_Linux|Damn Small Linux]]|authorlink = |year = undated}}</ref>。
*{{link-en|Feather Linux|Feather Linux}} － 從[[Knoppix|Knoppix]]衍生的輕量級發行版。現在處於不活躍的狀態<ref>{{cite web |url=http://featherlinux.berlios.de/docs.htm |title=Feather Linux - Docs |publisher=Featherlinux.berlios.de |date= |accessdate=2013-05-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20130307053530/http://featherlinux.berlios.de/docs.htm |archivedate=2013-03-07 }}</ref><ref>{{cite web|url=http://archive09.linux.com/feature/55586 |title=Linux.com :: Feather Linux: Light is right |publisher=Archive09.linux.com |date= |accessdate=2013-05-21}}</ref>。
*Kali Linux mini - [[Kali_Linux|Kali Linux]]的輕量級版本，大小約28 MB。Kail Linux是Backtrack的新代號，此一發行版被用於滲透測試。
*[[Lubuntu|Lubuntu]] － 比[[Ubuntu|Ubuntu]]輕量，其使用[[LXDE|LXDE]]桌面環境<ref name="Lubuntu" />。
*{{link-en|Nanolinux|Nanolinux}} － 基於MicroCore Linux，只有14 MB的發行版<ref>{{cite web|url=http://linux.softpedia.com/get/System/Operating-Systems/Linux-Distributions/Nanolinux-102963.shtml |title = Softpedia|accessdate=25 January 2014}}</ref>。
*[[Peppermint_Linux_OS|Peppermint Linux OS]] － 基於Lubuntu<ref>{{cite web|url=http://peppermintos.com |title=Peppermint |publisher=peppermintos.com |accessdate=13 July 2013}}</ref>。
*{{link-en|Porteus|Porteus (operating system)}} － 它只有不到300 MB的大小<ref name="Porteus">{{cite web|url = http://porteus.org/component/content/article/39-general-info/49-why-choose-porteus.html|title = Why choose Porteus ? - IT IS PORTABLE|accessdate = 30 April 2011|last = Porteus|authorlink = |date = 14 November 2010}}</ref>。可選用[[LXDE|LXDE]]或[[KDE|KDE]]桌面環境。
*[[Puppy_Linux|Puppy Linux]] － 比大多數其他的Linux發行版輕量<ref>{{Cite news|url = http://www.raymond.cc/blog/archives/2010/07/08/puppy-linux-5-0-review-lightweight-fun-fast/|title = Puppy Linux 5.0 Review - Lightweight, Fun, Fast!|accessdate = 11 December 2010|last = Hell-Noire|first = Paul|authorlink = |date = July 2010|work = Raymond|deadurl = yes|archiveurl = https://web.archive.org/web/20100711035158/http://www.raymond.cc/blog/archives/2010/07/08/puppy-linux-5-0-review-lightweight-fun-fast/|archivedate = 2010年7月11日}}</ref>。下載大小：133 MB。軟體包管理器：Synaptic（用於Ubuntu相容版本）。
*{{link-en|Semplice Linux|Semplice Linux}} － 輕量、快速、簡單，基於[[Debian|Debian]] Sid分支<ref name="SempliceLinux">{{cite web|url = http://semplice-linux.org/|title = Semplice Linux Web Site|deadurl = yes|archiveurl = https://web.archive.org/web/20170611052458/http://semplice-linux.org/|archivedate = 2017-06-11}}</ref>。
*[[Slitaz|Slitaz]] － 只有35 MB的發行版。軟體包管理器是Tazpkg<ref name="SliTaz" />。
*[[Tiny_Core_Linux|Tiny Core Linux]] － 只有12 MB的發行版。軟體套件庫：TCZ軟體包<ref name="Lynch01Jul09">{{Cite news|url = http://desktoplinuxreviews.com/2009/07/01/tiny-core-linux-2-1/4/|title = Tiny Core Linux 2.1|accessdate = 11 January 2011|last = Lynch|first = Jim|authorlink = |date = July 2009|work = Desktop Linux Reviews|deadurl = yes|archiveurl = https://archive.is/20130102095918/http://desktoplinuxreviews.com/2009/07/01/tiny-core-linux-2-1/4/|archivedate = 2013年1月2日}}</ref>。
*[[Trisquel#Trisquel_Mini|Trisquel Mini]] － Trisquel使用[[LXDE|LXDE]]的輕量級版本<ref>{{cite web |url=http://trisquel.info/en/trisquel-50-release-announcement |title=Trisquel 5.0 Release announcement |date=September 17, 2011 |publisher=The Trisquel Project |accessdate=September 14, 2012}}</ref>。
