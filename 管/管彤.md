'''管彤'''{{BD|1971年|2月11日|||catIdx= Z张}}出生于[[北京市|北京市]]，毕业于[[中国传媒大学|中国传媒大学]]，为[[中国中央电视台|中国中央电视台]]节目主持人。<ref name=" lion" /><ref name=" wolf" >{{cite web |url=http://www.zhuchijie.com/show_35_z7_t1.html |title= 管彤|publisher=主持界 |accessdate=2011 |work= |language =zh-cn}}</ref>

== 简介 ==
1971年2月11日，管彤出生于[[北京市|北京市]]，她母亲是[[北京人|北京人]]，父亲是[[上海人|上海人]]，父亲都是化工业工作者，她还有一个哥哥，后在[[北京市东直门中学|北京市东直门中学]]毕业，考入[[中国传媒大学|中国传媒大学]]播音系，大三时就参与拍摄电视电影《[[青春作证|青春作证]]》，大学毕业后，管彤被分配到[[北京有线电视台|北京有线电视台]]影视部《[[快乐周末|快乐周末]]》担任主持人。<ref name=" lion" >{{cite web |url= http://ent.scol.com.cn/bbshot/20081127/20081127100531.htm|title=管彤：渴望火炉一样的爱情 |publisher= 四川在线|accessdate= 2008-04-27|work= |language =zh-cn}}</ref><ref name=" wolf" />
1997年，管彤正式调入[[中央电视台戏曲频道|中央电视台戏曲频道]]音乐部担任主持人。<ref name=" wolf" />

== 个人生活 ==
管彤现在仍是单身，她曾向媒体透露自己的择偶标准是：“他可以给我安全感，包容，有责任心。不能小心眼，要理解我的工作。我希望有一天能成为一个操持家务的主妇。”<ref name=" lion" />
