'''管仲'''（{{bd||前725年|前645年|catIdx=G管}}），'''姬'''姓，'''管'''氏，名'''夷吾'''，字'''仲'''，[[諡|諡]]'''敬'''，被称为'''管子'''、'''管夷吾'''、'''管敬仲'''，颍上（今[[安徽省|安徽省]][[颍上县|颍上县]]）人，[[春秋时代|春秋时代]][[法家|法家]]代表人物。[[姜齊|齊國]]的[[政治家|政治家]]，[[哲学家|哲学家]]，與[[樂毅|樂毅]]齊名，[[周穆王|周穆王]]的后代<ref>《春秋左传正义·僖公十二年》：《世族谱》：“管氏出自周穆王。”</ref>。管仲虽然仅是齐国下卿<ref>《左传·僖公十二年》：王以上卿之礼飨管仲，管仲辞曰：“臣，贱有司也，有天子之二守国、高在。若节春秋来承王命，何以礼焉？陪臣敢辞。”王曰：“舅氏，余嘉乃勋，应乃懿德，谓督不忘。往践乃职，无逆朕命。”管仲受下卿之礼而还。</ref>，却被视为中国历史上宰相的典范，任内大兴改革，重视商业。<ref>吳曉波：《歷代經濟變革得失》華品文創出版股份有限公司，2014年，21頁</ref>《[[国语|国语]]·齐语》、{{Wikisource history link|史記|062|管晏列傳}}、《[[管子_(書籍)|管子]]》、《[[左传|左传]]》等都载有他的生活传记，《[[論語|論語]]》中也有幾處關於[[孔子|孔子]]對他的評論，[[北宋|北宋]][[蘇洵|蘇洵]]的《管仲论》也对管仲做出了分析和批判。

== 管鮑之交 ==
年轻时家境贫困，[[鲍叔牙|鲍叔牙]]與管仲往來，发现管仲有才能，勝過自己<ref>《国语·齐语》第一篇：鲍叔对齐桓公说：“臣之所不若夷吾者五：宽惠柔民，弗若也；治国家不失其柄，弗若也；忠信可结于百姓，弗若也；制礼义可法于四方，弗若也；执枹鼓立于军门，使百姓皆加勇焉，弗若也。”</ref>，往來過程中，管仲常佔小便宜，鲍叔牙不以為意，反為他設想，蔚為美談，史稱“[[管鮑之交|管鮑之交]]”。后来，管仲輔佐[[齐国|齐国]][[公子纠|公子纠]]，鲍叔牙輔佐其弟[[齐桓公|公子小白]]。
    
前686年，[[齐襄公|齐襄公]]被[[大夫_(官名)|大夫]][[連稱|連稱]]、[[管至父|管至父]][[兵變|兵變]]殺死，[[公孫無知|公孫無知]]因而[[篡位|篡位]]。前685年春，[[姜齐|齐]][[大夫_(官名)|大夫]][[雍廪|雍廪]]杀了公孙无知，公子小白即位，是为[[齐桓公|齐桓公]]，[[鲁国|鲁国]]却支持其兄公子纠，因此[[齐国|齐国]]和[[鲁国|鲁国]]之间发生战争，管仲射箭中了[[齐桓公|齐桓公]]的带钩，[[齐桓公|齐桓公]]裝死，騙過了管仲。後來齐国战胜，[[鲍叔牙|鲍叔牙]]向[[魯莊公|魯莊公]]要求处决公子纠，并把管仲交给齐国。在[[鲍叔牙|鲍叔牙]]强烈推荐下，[[齐桓公|齐桓公]]不計前嫌，拜管仲为相，甚至尊為「[[仲父|仲父]]」<!--增列仲父名稱-->。管仲在齐国实行了一系列的改革，倡「[[尊王攘夷|尊王攘夷]]」<!--增列尊王攘夷，管仲功不可沒-->，终于帮助[[齐桓公|齐桓公]]成就了霸业。
   
一次，在管仲病危的时候。[[齐桓公|齐桓公]]找管仲商量谁可接相位，提出要将相位传与[[鲍叔牙|鲍叔牙]]。而管仲却坚决反对，认为鲍叔牙虽是[[君子|君子]]，为人近乎完美，但过于清白而容不得一丝丑恶，不适合做丞相；最後管仲推荐了[[隰朋|隰朋]]。<ref>《[[吕氏春秋|吕氏春秋]]·贵公》：“公曰：‘鮑叔牙可乎？’管仲對曰：‘不可。夷吾善鮑叔牙，鮑叔牙之為人也：清廉潔直,視不己若者,不比於人；一聞人之過，終身不忘。’‘勿已，則隰朋其可乎？’‘隰朋之爲人也，上志而下求，醜不若黃帝，而哀不己若者。其於國也，有不聞也；其於物也，有不知也；其於人也，有不見也。勿已乎，則隰朋可也。’”</ref>桓公四十一年，管仲卒。<ref>《[[史记|史记]]·齐太公世家》说齐桓公四十一年，“是岁，管仲、隰朋皆卒。”另《[[史记|史记]]·秦本记》又作：“十二年（前648年），齐管仲、隰朋死。”</ref>

== 管仲变法 ==
{{Expand Section}}
{{wikisource|管子}}
废除[[井田制|井田制]]，建立土地[[税收制度|税收制度]]，允许土地买卖，承认[[土地私有化|土地私有化]]。建立[[常备军|常备军]]。

早在公元前八世紀的管仲已經掌握了鼔勵營商環境的基本法則：虛車勿索，徒負勿入，以來遠人。<ref>《管子·霸言篇》</ref>空車來的和徒步背東西的，都給予免稅優惠，這樣就能吸引遠方的人來做買賣。

== 历史评价 ==
[[File:Guan_zhong_grave.JPG|thumb]]
[[孔子|孔子]]曾言：“管仲之器小哉。”但也说：“微管仲，吾其被髮左衽矣”、“桓公九合[[诸侯|诸侯]]，不以兵车，管仲之力也。如其仁。如其仁”等等。基本上稱讚管仲施政成績優異，使得[[華夏|華夏]]強大，讓[[華夏族|華夏族]]免於遭受[[蠻夷|蠻夷]]的統治，維護了[[華夏文化|華夏文化]]，褒大於貶。

《[[史記|史記]]·管晏列传》中说：“天下不多管仲之賢而多鮑叔能知人也。”“管仲世所謂賢臣，然孔子小之。豈以爲周道衰微，桓公旣賢，而不勉之至王，乃稱霸哉？語曰：‘將順其美，匡救其惡，故上下能相親也。豈管仲之謂乎？’”

[[司馬光|司馬光]]說：“管仲鏤簋朱紘，山楶藻梲，孔子鄙其小器。”<ref>司馬光：〈訓儉示康〉</ref>

根據唐代司馬貞為《史記》做的《索隱》，「王以民為天，民以食為天」是管仲的思想。管仲的原話是「王者以民為天，民以食為天，能知天之天者，斯可矣。」
