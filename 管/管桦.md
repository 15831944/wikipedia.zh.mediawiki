'''管桦'''（{{bd|1922年|1月9日|2002年|8月17日}}）原名'''鲍化普'''，[[河北省|河北省]][[丰润县|丰润县]]人，[[中华人民共和国|中华人民共和国]]作家。

==生平==
管桦生于丰润县三女河乡女过庄村一户普通农家。还乡河的支流泥河流经村边，他的童年便在[[芦苇|芦苇]]荡中度过。8岁进入本村小学学习，后来升初中时因家贫而被迫中断学业。从此，他经常到野外拾柴禾，夏收时拾麦穗，冬季农闲时在家读《[[红楼梦|红楼梦]]》、《[[西游记|西游记]]》、《[[水浒|水浒]]》、《[[七侠五义|七侠五义]]》等古典文学作品<ref name=haijiang>{{Cite web |url = http://www.haijiangzx.com/2017/0306/1700486.shtml |title = 选入语文课本，感动了几代中国人的《小英雄雨来》 |author =  |date = 2017-03-06 |publisher =海疆在线  |language =  |accessdate =  |quote =  }}</ref><ref name=zgwmw>{{Cite web |url =http://ts.wenming.cn/wmbb/201511/t20151120_2971683.shtml  |title =寻找“小英雄雨来”  |author =  |date = 2015-11-20 |publisher = 中国文明网 |language =  |accessdate =  |quote =  }}</ref>。

1938年7月，管桦的父亲[[鲍子菁|鲍子菁]]参加了由中共冀东党组织直接指挥的20万工农参加的[[冀东大暴动|冀东大暴动]]，先后收复[[日军|日军]]控制的9座县城，切断了[[北宁铁路|北宁铁路]]。鲍子菁任[[国民革命军|国民革命军]]第九路军第七师师长，后调入[[八路军|八路军]][[冀东军区|冀东军区]]政治部工作，1944年在与日军作战中牺牲。鲍子菁参加冀东大暴动时，托朋友将家属送到[[天津|天津]]避战火。管桦随母亲在天津居住期间，进入志远中学学习，在校期间阅读了[[鲁迅|鲁迅]]、[[茅盾|茅盾]]、[[巴金|巴金]]、[[田汉|田汉]]、[[丁玲|丁玲]]、[[老舍|老舍]]、[[王统照|王统照]]等不少中国作家的作品，作文成绩提高很大，多次获得语文老师表扬。一位语文老师甚至说：“你将来可能成为一个文学家。”当时管桦酷爱绘画，每日放学回家后总会临摹父亲给他买的《[[芥子园画谱|芥子园画谱]]》<ref name=haijiang/>。

[[冀东抗日根据地|冀东抗日根据地]]成立后，父亲将全家人从天津接回冀东。1940年，管桦参加[[八路军|八路军]]，想进[[抗日军政大学|抗日军政大学]]（简称抗大）学习军事以便到抗日前线杀敌，但抗大不招收无战斗经验的学生。他只得进入[[晋察冀边区|晋察冀边区]]的[[华北联合大学|华北联合大学]]文艺学院文学系学习。为不连累家人，不少学员都把名字改了。管桦起初改名“管华”，但同学们开玩笑说：“管理中华，野心太大了吧！”他改口说：“我喜欢树木，抗战胜利后，我就管理一片树林。”于是他改名“管桦”，后来也成为他的笔名。在校期间他学习革命理论及文学知识，观看了文工团演的《带枪的人》、《母亲》等节目，并且“第一次听到[[福楼拜|福楼拜]]、[[莫泊桑|莫泊桑]]、[[大仲马|大仲马]]等作家的名字和老师对他们作品的分析”<ref name=haijiang/>。

1942年毕业后，管桦被分配到中共冀热边特委主办的救国报社担任随军记者。任职期间1942年他创作出描写抗日战争生活的短篇小说《行军》，发表在《挺进军报》上，自此开始文学创作。1943年，调到冀东军区政治部尖兵剧社工作，历任文艺演出队副队长、队长及[[冀热辽军区|冀热辽军区]]文工团副团长等职。其间创作了大量歌词、剧本、文艺通讯，也写了小说。1945年抗日战争胜利后，随部队挺进[[中国东北|中国东北]]。1946年加入[[中国共产党|中国共产党]]<ref name=haijiang/>。

1948年，管桦因病从部队转业，调任[[东北鲁迅文艺学院|东北鲁迅文艺学院]]研究员。该学院的前身是1938年在[[延安|延安]]创建的鲁迅艺术学院（简称鲁艺），1946年以后，鲁艺先后迁至[[齐齐哈尔|齐齐哈尔]]、[[佳木斯|佳木斯]]、[[哈尔滨|哈尔滨]]、[[沈阳|沈阳]]，并改为东北鲁迅文艺学院。在东北期间，他先后写出歌剧《归队立功》，写出并发表了中篇小说《荆各庄的故事》及短篇小说《雨来没有死》，与别人合著出版了歌剧《[[中华民国国军|国军]]现形记》及话剧《[[蒋介石|蒋]][[大日本帝国|敌]][[汪精卫政权|伪]]合流》。其中歌剧《归队立功》（[[李劫夫|李劫夫]]作曲）获得[[冀察热辽军区|冀察热辽军区]]通令嘉奖，并获[[朱德|朱德]]奖章<ref name=haijiang/><ref name=fenghuo/>。

在抗日战争时期，管桦作为随军记者经常独自活动，为他带路的往往是些年纪很小的[[儿童团|儿童团]]员，他见过许许多多冀东抗日的少年儿童，也听过不少他们的抗日事迹。1945年夏秋之交，冀东军区八路军某部攻打[[玉田县|玉田县]]城，管桦和尖兵剧社其他几人要到前线采访，在城南庄为他们带路的是个十三四岁的孩子，又活泼又自信，一行人走着走着忽然发现远处有座炮楼，孩子被炮楼里射出的子弹击中太阳穴当场牺牲，这给管桦留下了难以忘怀的记忆。后来在随东北鲁迅文艺学院迁到沈阳后，他开始创作短篇小说《[[小英雄雨来|雨来没有死]]》，写成初稿后首先送给同在该学院工作的[[周立波_(作家)|周立波]]审阅，获得周立波称赞和鼓励。很快《雨来没有死》于1948年在《[[晋察冀日报|晋察冀日报]]》副刊上发表，获得读者一致好评。[[中华人民共和国|中华人民共和国]]成立初期，短篇小说《雨来没有死》被选入全国中、小学语文课本（后来课文标题改为《[[小英雄雨来|小英雄雨来]]》），管桦也因此成名。小英雄雨来成为中华人民共和国全国中、小学生心目中的小英雄，他的故事伴随了中华人民共和国几代人的成长<ref name=haijiang/>。

中华人民共和国成立后，管桦先是调到[[天津音乐学院|天津音乐学院]]任职，后任[[中央乐团|中央乐团]]创作员，最终成为北京市作家协会驻会作家。其间，他创作了故事大合唱《飞虎山》（[[张文纲|张文纲]]作曲）、歌曲《听妈妈讲那过去的事情》（[[瞿希贤|瞿希贤]]编曲）等的歌词。在《雨来没有死》的基础上，管桦于1955年写作并发表了中篇小说《[[小英雄雨来|小英雄雨来]]》。1963年，《小英雄雨来》由[[河北人民出版社|河北人民出版社]]出版单行本，后多次再版。后来还出现了多家出版社都印行这部小说的火热局面。该小说后被改编为同名连环画、同名电影、同名儿童评歌剧等等<ref name=haijiang/>。

“[[文革|文革]]”初期，一次冲击北京市文联的中学生[[红卫兵|红卫兵]]要揪斗管桦，拿着皮带准备殴打他，其中有人问了句：“管桦是谁？”旁边有人答：“就是写《小英雄雨来》的那个作家。”这些中学生都很熟悉《小英雄雨来》，因此放过了管桦<ref name=fenghuo/><ref name=haijiang/>。但在“文革”期间，《小英雄雨来》从教材中消失。当时[[天津市|天津市]]中、小学语文教师曾经一致要求将《小英雄雨来》继续编入语文课本。但上级有关领导指这篇课文中“我们是中国人，我们热爱自己的祖国”这句话含有[[大国沙文主义|大国沙文主义]]意味，必须删除，并且要求管桦把“雨来”这个人物改写成革命[[样板戏|样板戏]]《[[红灯记|红灯记]]》里李玉和似的高大形象，还派两位青年教师代表到管桦住处传达上述上级指示。管桦听后严词拒绝，两位青年教师代表乃匆忙返回天津<ref name=haijiang/>。文革期间，管桦被打倒后，突发心肌梗塞。后来他的作品不准发表，他就在家继续写长篇小说《将军河》，并且开始画墨竹<ref name=fenghuo>王志学，风摧不折 火烧不尽——记全国政协委员管桦，北京观察1996(10):32-34</ref>。

[[改革开放|改革开放]]后，管桦是第五、六、七、八届全国政协委员，北京市文联主席，中国作协北京分会主席，一级教授。1949年加入[[中国作家协会|中国作家协会]]<ref name=zgzjw>{{Cite web |url =http://www.chinawriter.com.cn/zxhy/member/8408.shtml  |title = 管桦 (1921～2002) |author =  |date =  |publisher = 中国作家网 |language =  |accessdate = 2017-11-10 |quote =  }}</ref>。

管桦希望在家乡的还乡河公园建一座小英雄雨来纪念园，很快获得中共[[唐山市|唐山市]]委、唐山市人民政府支持。1998年，“小英雄雨来纪念园”在唐山市[[丰润区|丰润区]]还乡河公园落成，雨来纪念碑上有管桦亲笔撰写的碑文<ref name=haijiang/>。

2002年8月17日，管桦因急性[[心肌梗塞|心肌梗塞]]在[[唐山|唐山]]病逝，享年81岁<ref name=shishi>{{Cite web |url = http://ent.sina.com.cn/s/m/2002-08-19/102196223.html |title =著名词作家、北京市文联主席管桦病逝  |author =  |date = 2002-08-19 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。
