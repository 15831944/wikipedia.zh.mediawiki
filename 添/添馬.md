'''添馬'''（英文：'''Tamar'''，中文前稱'''添馬艦'''<ref>英文名稱沒有改變過，一直都是“Tamar”。</ref>）位於[[香港|香港]][[香港島|香港島]][[中西區_(香港)|中西區]]，[[金鐘|金鐘]]北部。曾是英國海軍基地及船塢所在地，1990年代海軍基地遷往[[昂船洲海軍基地|昂船洲]]後空置。2002年提出興建[[添馬艦發展工程|新政府總部大樓計劃]]，於2005年重新進行招標。2007年7月，政府正式挑選承建商金門和協興建築聯營提交的「門常開」設計概念，於2011年5月完成，最後於同年8月至12月逐步啟用。現時是[[香港特別行政區|香港特別行政區]][[政府總部_(香港)|政府總部]]及[[立法會綜合大樓|立法會綜合大樓]]所在地。

傳統上添馬（當時稱「添馬艦」）被認為是金鐘的海旁部分，而[[中環軍營|中環軍營]]則屬於中環區。自從[[添馬艦發展工程|添馬艦發展工程]]完成後，香港政府將「添馬艦」的中文名稱改為「添馬」<ref>僅指地名上的「添馬艦」，原來沉沒該地、名為「[[添馬艦_(軍艦)|添馬艦]]」的戰艦的名稱沒有改變。</ref><ref>「添馬艦」的英文就是“Tamar”，後面並沒有“battleship”之類的標識詞，中文翻譯加上「艦」字是由於中英文的語言習慣並不一致，中文習慣在稱呼戰艦名稱時加上「艦」字。</ref>，並作為地區名稱大量應用於該地建築物的地址及交通道路的路牌上。現今的「添馬」區是指[[愛丁堡廣場|愛丁堡廣場路]]至[[演藝路|演藝路]]之間及其延伸到海邊的空間，南界為[[干諾道|干諾道中]]和[[夏慤道|夏慤道]]。由於中環軍營原本就是添馬艦海軍基地的一部分，故也被認為屬於添馬區的一部分。

== 歷史 ==
「添馬」或「添馬艦」的地名來自[[英國皇家海軍|英國皇家海軍]]的同名軍艦（三桅運兵船）[[添馬艦_(軍艦)|添馬艦]]（HMS Tamar），於1897年至1941年為駐港海軍的主力艦，停在香港的英國海軍船塢。在1941年的[[香港保衛戰|香港保衛戰]]，港府將添馬艦炸沉，以免落入[[日軍|日軍]]手上。英國海軍船塢原位於金鐘，[[第二次世界大戰|第二次世界大戰]]後，金鐘進行[[填海|填海]]工程，海軍船塢遷至位於較北的地點，為紀念添馬艦這艘曾保衛香港的戰艦，政府便把海軍基地命名為添馬艦。政府亦興建了[[夏慤道|夏慤道]]，把原來被海軍船塢分隔開的[[告士打道|告士打道]]與[[干諾道|干諾道]]中連接起來。

昔日的駐港英軍三軍司令部──今[[中國人民解放軍駐香港部隊|中國人民解放軍駐香港部隊]]總部所在的[[威爾斯親王大厦|威爾斯親王大厦]]，設計呈[[漏斗|漏斗]]型。1979年3月4日由到港訪問的[[威爾斯親王|威爾斯親王]]，即皇儲[[查理斯王子|查理斯王子]]舉行揭幕儀式。

1990年代，為配合中環商業區的發展，政府把海軍基地遷往[[昂船洲|昂船洲]]，[[駐港英軍|駐港英軍]]總部則保留，原本船塢所在經填海後成為一片空地。部分土地被用作興建[[直昇機|直昇機]]場及[[中信大廈_(香港)|中信大廈]]。1997年後，解放軍駐港總部仍設於添馬艦。

== 重要活動 ==
添馬艦的空地曾被用作各項活動，包括[[演唱會|演唱會]]、[[工展會|工展會]]、[[馬戲團|馬戲團]]表演、盆菜宴等。部份活動曾創下世界紀錄，並收錄於《[[健力士世界紀錄大全|吉尼斯世界纪录大全]]》。

1997年6月30日，在英國撤出香港，把主權移交給[[中華人民共和國|中國]]前，英方在添馬艦露天場地舉行盛大的告別儀式，主禮嘉賓包括王儲[[查理斯王子|查理斯王子]]、[[英國首相|英國首相]][[貝理雅|貝理雅]]、外相[[郭偉邦|郭偉邦]]、前首相[[戴卓爾夫人|戴卓爾夫人]]及卸任[[港督|港督]][[彭定康|彭定康]]。儀式舉行時下起滂沱大雨，令嘉賓及表演者甚為狼狽，亦令人難忘，亦為英軍的撤走添上悲哀色彩。

1999年11月27日，約4,950人參加「萬人泡茶迎千禧」的[[茶|茶]]會活動，消耗了14,850杯茶和15,000件中式糕餅，打破了最多人同時泡茶的世界紀錄。活動為迎接[[千禧年|千禧年]]，由香港茶人聯合會主辦。

2000年9月24日，為紀念[[世界心臟日|世界心臟日]]，讓市民關注[[心臟|心]]血管病及健康，1060名學生在添馬艦參加「千人跳繩操」，[[跳繩|跳繩]]3分鐘，打破跳繩活動參加人數的世界紀錄，活動由香港心臟學研究學院及東華三院主辦。

2003年8月，香港經歷[[沙士|沙士]]疫症後，政府在添馬艦空地舉辦[[維港巨星匯|維港巨星匯]]，邀請如[[滾石樂隊|滾石樂隊]]等世界著名音樂人來港表演，並以公帑補貼開支，結果入場人數未如理想，惹來社會各界批評政府濫用公帑。

2003年12月19日-2004年2月1日，首次在金鐘添馬艦舉行的[[香港環球嘉年華|香港環球嘉年華]]，為期45天，橫跨聖誕、元旦及春節等重大節日。

2004年4月16日，國際品牌[[路易·威登|路易·威登]]在全球4大城市，包括香港，舉行創立150周年慶典，邀請世界各地共2000名嘉賓出席。在添馬艦搭起巨型帳幕，並以投射效果做成巨型行李箱的模樣。晚會共花費1000萬港元。

2005年3月，[[香港國際電影節|香港國際電影節]]主辦單位在添馬艦架起有5層樓高的[[亞洲|亞洲]]最巨型吹氣[[銀幕|銀幕]]，放映8套不同電影，包括《功夫》及《精武門》等。銀幕長24米、寬12米，面積450平方米。場地可容納4000名觀眾。

2005年12月，香港主辦[[世界貿易組織第六次部長級會議|世界貿易組織第六次部長級會議]]，添馬艦劃為管制區，曾作為各國出席會議者座駕的停車場。

2006年1月8日，添馬艦連續3年舉辦萬人[[盤菜|盤菜]]宴，筵開1,100席，共有14,000人參加，再一次刷新世界紀錄。

2006年2月18日至25日，第九屆[[世界消防競技大賽|世界消防競技大賽]]於香港舉行，這是自1990年開始舉辦以來，首個舉辦該盛事的亞洲城市。主要比賽地點設在添馬艦。共有37個國家和地區的3300名消防員參與。

2006年10月28日，[[瑪麗亞·凱莉|瑪麗亞·凱莉]]到添馬艦作巡迴演出最後一站。

2006年12月15日至2007年4月23日，恢復舉辦闊別兩年的[[香港環球嘉年華|香港環球嘉年華]]，為期135天，橫跨[[聖誕|聖誕]]、[[元旦|元旦]]、[[情人節|情人節]]、[[春節|春節]]、[[元宵|元宵]]、[[清明節|清明節]]及[[復活節|復活節]]等重大節日。

== 添馬艦發展工程 ==
{{Main|添馬艦發展工程}}

[[政府總部_(香港)|新政府總部]]的興建計劃，包括興建4座建築物，其中兩座為29至40層，只較隔鄰的[[中信大廈|中信大廈]]略高，另外兩座則為10層。建築物的高度限制為130至160米。政府總部、行政會議大樓及多用途會議廳預算總建築面積約為11萬平方米，立法會大樓的總建築面積則為26000平方米，總建築面積約136000平方米。此外，添馬艦海旁也是[[中環及灣仔填海計劃|中環填海計劃第三期]]的範圍之一，海旁興建海濱長廊，以及與解放軍駐港總部連接的[[中環解放軍碼頭|中環解放軍碼頭]]等設施。工桯項目2006年6月獲立法會財務委員會通過興建，2008年年初動工，2011年8月落成。

==區議會議席分佈==
由於添馬用地並非用作住宅（由當初的英軍[[軍營|軍營]]至現時的[[辦公室|辦公室]]用途），理論上沒有常住人口（軍隊人員除外）。整個添馬連同[[金鐘|金鐘]]（即東至[[香港演藝學院|香港演藝學院]]外的[[演藝道|演藝道]]、[[香港警察|香港警察]]總部，西至[[紅棉道|紅棉道]]、南至[[堅尼地道|堅尼地道]]地區）都劃入鄰近的[[中環_(選區)|中環]]選區。
