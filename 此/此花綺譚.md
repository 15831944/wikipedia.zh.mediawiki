{{日本作品|此花綺譚|このはな綺譚}}是由[[天乃咲哉|天乃咲哉]]創作的[[日本|日本]][[漫畫|漫畫]]作品，最初名稱為{{日本作品|此花亭奇譚|此花亭奇譚}}，連載于[[一迅社|一迅社]]的漫畫[[期刊|季刊]]雜誌《{{tsl|ja|コミック百合姫S|Comic百合姬S}}》Vol.9（2009年6月18日發售）至Vol.14（2010年9月18日发售）。由于《Comic百合姬S》在Vol.14出版之后废刊合并至当时为[[期刊|双月刊]]的《[[Comic百合姬|Comic百合姬]]》，作者為了兼顧連載於《[[月刊Dragon_Age|月刊Dragon Age]]》的《[[GOSICK|GOSICK]]》漫畫版以及父亲的身体状况，从而选择了休載<ref>{{cite web|script-title=ja:百合姫S休載のおしらせ|url=http://amanosakuya.blog83.fc2.com/blog-entry-549.html|date=2010-08-23|archiveurl=https://web.archive.org/web/20151208163220/http://amanosakuya.blog83.fc2.com/blog-entry-549.html|archivedate=2015-12-08|accessdate=2017-10-11|language=ja}}</ref><ref>{{Cite tweet|author=天乃咲哉|user=amanosakuya|number=915126969905188864|script-title=ja:人を幸せにするような漫画が描ける状態ではありませんでした|date=2017-10-03|accessdate=2017-10-11|language=ja}}</ref>。已出版全2冊[[單行本|單行本]]<ref>{{Cite web|url=https://www.amazon.co.jp/dp/4758071063/|script-title=ja:此花亭奇譚 2|accessdate=2017年8月21日|publisher=[[日本亞馬遜|日本亞馬遜]]|language=ja}}</ref>。已發行《此花亭奇譚 新裝版》上、下2冊<ref>{{Cite web|url=https://www.amazon.co.jp/dp/4344834356/|script-title=ja:此花亭奇譚　新装版 (上)|accessdate=2017年8月21日|publisher=日本亞馬遜|language=ja}}</ref><ref>{{Cite web|url=https://www.amazon.co.jp/dp/4344834577/|script-title=ja:此花亭奇譚　新装版 (下)|accessdate=2017年8月21日|publisher=日本亞馬遜|language=ja}}</ref>。

其後，本作轉到新出版社[[幻冬舍Comics|幻冬舍Comics]]的漫畫雜誌《月刊Comic Birz》（現{{tsl|ja|月刊バーズ|月刊Birz}}）2015年2月號（2014年12月29日發售）恢復連載，標題變更為{{日本作品|此花綺譚|このはな綺譚}}（此花以假名表記，奇改為綺）<ref>{{Cite web|url=http://natalie.mu/comic/news/135083|script-title=ja:天乃咲哉がコミックバーズで新連載、狐の仲居が働く温泉宿を描く|date=2014-12-30|website=漫畫Natalie|accessdate=2017-08-14|language=ja}}</ref>。已出版7冊單行本。

2017年3月宣布製作[[電視動畫|電視動畫]]<ref>{{Cite web|url=https://mantan-web.jp/article/20170330dog00m200009000c.html|script-title=ja:このはな綺譚：アニメ化決定 キツネ少女たちの日常描いたマンガ|date=2017-03-30|website=MANTANWEB|accessdate=2017-08-14|language=ja}}</ref>，以《此花綺譚》為標題，同年10月4日起在[[東京都會電視台|東京都會電視台]]等播出<ref>{{Cite web|url=http://dengekionline.com/elem/000/001/572/1572027/|script-title=ja:TVアニメ『このはな綺譚』が10月より放送開始。柚（声優：大野柚布子）たち6人の魅力が詰まったPV配信|date=2017-08-07|website=電擊Online|accessdate=2017-08-14|language=ja}}</ref>。

== 故事簡介 ==
由[[人魚#八百比丘尼的傳說|八百比丘尼]]撫育長大的野狐「柚」為了學習待人處事而來到高級溫泉旅館此花亭擔任服務員（{{link-ja|仲居|仲居}}），在與服務員前輩們一起生活的同時，也體驗了許多由造訪此花亭的神明所交織出的奇幻事件。

== 登場人物 ==
=== 此花亭 ===
; {{nihongo|柚|柚（ゆず）|聲：[[大野柚布子|大野柚布子]]<ref name="character">{{cite web|url=http://konohanatei.jp/character/|title=登場人物|work=TV動畫「此花綺譚」官方網站|language=ja}}</ref>}}
: 本作主人公。有明亮的茶色長髮，頭頂部短有一束[[呆毛|呆毛]]。
: 小時候被埋在雪堆時，被八百比丘尼撿到並撫養長大。為了學習社會事情而來到溫泉旅館「此花亭」。
: 有些冒失，但有著積極向前的性格。
; {{nihongo|皐|皐（さつき）|聲：[[秦佐和子|秦佐和子]]<ref name="character"/>}}
: 柚新人時代時的教育員，柚非常憧憬的前輩。擁有青紫顏色的長髮。
: 責任感很強，雖然現在正在此花亭工作，但她的夢想是像姊姊一樣成為巫女。
: 實際上無論是舞蹈還是細心程度都超過自己的姊姊，其父母讓其姊柊成為巫女是因為柊連九九乘法都不會僅有跳舞一點高於他人，因此讓聰明又不會惹事的皐去此花亭工作<ref>動畫第10集</ref>。
; {{nihongo|棗|棗（なつめ）|聲：[[諏訪彩花|諏訪彩花]]<ref name="character"/>}}
: 穿着男用和服及短褲，男孩子氣的女服務員。第一人稱為「ボク」
; {{nihongo|蓮|蓮（れん）|聲：[[久保田梨沙|久保田梨沙]]<ref name="character"/>}}
: 愛打扮的優等生。以「保持優雅」為宗旨，但經常在心裡吐出辱罵之言。由於小時候的心理陰影，有着不擅長應對男人的一面。
; {{nihongo|櫻|櫻（さくら）|聲：[[加隈亞衣|加隈亞衣]]<ref name="character"/>}}
: 嬌小而又文靜的少女。讓人很難看出心中所想。
; {{nihongo|桐|桐（きり）|聲：[[沼倉愛美|沼倉愛美]]<ref name="character"/>}}
: 「此花亭」的招待長。非常喜歡煙斗和酒，難以對付的性格，但也經常關心其他的服務員們。
; {{nihongo|椿|椿（つばき）|聲：[[緒方惠美|緒方惠美]]}}
: 「此花亭」的女老闆。
