'''毒理学'''是研究外源性[[化学物|化学物]]及[[物理|物理]]和[[生物|生物]]因素对生物[[有机体|有机体]]的有害作用及其作用机理，进而预测其对人体和生态环境的危害的严重程度，为确定[[安全限值|安全限值]]和采取防治措施提供科学依据的科学，也是对[[毒性|毒性]]作用进行定性和定量评价的一门[[学科|学科]]。由于毒理学的研究目的是为保护生物体的健康或安全提供科学依据的一门学科，因此从学科性质上毒理学属于[[预防医学|预防医学]]，贯穿了预防为主的[[思想|思想]]。

== 联系 ==
毒理学与[[药理学|药理学]]密切相关，目前已发展成为具有一定基础理论和实验手段的独立学科，并逐渐形成了一些新的毒理学分支。毒理学按照其研究目的及所研究的化学物质特性和用途可分为[[工业毒理学|工业毒理学]]、[[军事毒理学|军事毒理学]]、[[环境毒理学|环境毒理学]]、[[药物毒理学|药物毒理学]]、[[法医毒理学|法医毒理学]]和[[放射毒理学|放射毒理学]]等。

== 研究对象 ==
毒理学的研究对象广泛，包括化学因素、物理因素、生物因素，而生物体包括人、动物、植物。因此毒理学与药理学、[[生理学|生理学]]、[[病理学|病理学]]、化学、[[生物化学|生物化学]]、[[生物学|生物学]]有联系；与[[工业|工业]]、[[农业|农业]]、[[经济|经济]]有联系；与[[法医学|法医学]]、[[临床医学|临床医学]]、[[生态学|生态学]]及[[环境保护|环境保护]]有联系；可以说，它与地球上生命的整个未来有联系。因而，毒理学在临床医学、药学、环境保护、动物学、[[优生优育|优生优育]]、职业[[劳动保护|劳动保护]]和食品卫生等领域中均有广泛用途。

== 分类 ==
[[File:Toxicology_Research_at_FDA_(NCTR_1193)_(6009043040).jpg|thumb]] 
因此毒理学的分类非常复杂，可从不同角度分类，并不完全一致。

:从研究内容上可分为：描述性毒理学、机制性毒理学和管理毒理学（也有称为法规毒理学）三部分。

:从依照标准学科划分可分为：[[法医|法医]]毒理学、临床毒理学、管理毒理学或[[法规|法规]]毒理学、研究毒理学等。

:从应用毒理学可分为：[[食品|食品]]毒理学、[[工业|工业]]毒理学、[[农药|农药]]毒理学、[[军事|军事]]毒理学、[[放射|放射]]毒理学、[[环境|环境]]毒理学、[[生态|生态]]毒理学等分支。

:从研究对象可分为：[[昆虫|昆虫]]毒理学、[[兽医|兽医]]毒理学、[[人体|人体]]毒理学和[[植物|植物]]毒理学。

:从研究领域可分为：[[药物|药物]]毒理学、[[环境|环境]]毒理学、食品毒理学、工业毒理学、临床毒理学、法医毒理学、[[分析|分析]]毒理学、军事毒理学、管理毒理学等。

:从研究的靶器官或系统可分为：[[肝脏|肝脏]]毒理学、[[肾脏|肾脏]]毒理学、[[神经|神经]]毒理学、[[生殖|生殖]]毒理学、[[免疫|免疫]]毒理学、[[皮肤|皮肤]]毒理学、[[血液|血液]]毒理学等。

:从机制研究可分为：[[细胞|细胞]]毒理学、[[遗传|遗传]]毒理学、[[膜|膜]]毒理学、[[生化|生化]]毒理学、[[分子|分子]]毒理学。

:从毒物作用时相或过程可分为：[[毒代动力学|毒代动力学]]和[[毒效动力学|毒效动力学]]。
