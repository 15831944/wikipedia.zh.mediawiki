'''毒氣室'''，是一種[[自殺|自殺]]、[[他殺|他殺]]或者[[死刑|死刑]]的工具，方法是將自己或者別人、犯人等上[[鎖|鎖]]在一個封閉的[[空間|空間]]，施放[[毒氣|有毒气體]]，引發[[死亡|死亡]]。

== 簡史 ==
如此的處死方式曾經在[[二次世界大戰|二次世界大戰]]期間被[[納粹德國|納粹德國]]所使用，目的是[[集體謀殺|集體謀殺]]。

== 美國 ==
[[美國|美國]]於過去曾以毒氣室作為處決罪犯的工具，多數是[[殺人犯|殺人犯]]。現時有五個[[州|州]]在條文上保留了這套行刑工具。
毒氣室行刑的成本為美國所有死刑執行方式裡最高的，其次是[[電椅|電椅]]。至今死刑犯仍可以選擇毒氣死刑。{{notetag| Discovery頻道《死囚最後24小時》}}
現時有五個州，[[亞利桑那州|亞利桑那州]]，[[加利福尼亞州|加利福尼亞州]]，[[馬里蘭州|馬里蘭州]]，[[密蘇里州|密蘇里州]]和[[懷俄明州|懷俄明州]]，作為'''次要的'''執行方式。

== 例子 ==
* [[倫常慘劇|倫常慘劇]]：輕生者在[[家|家]]中自製「毒氣室」，[[燒炭自殺|燒炭]]，製造[[一氧化碳|一氧化碳]]、施放[[煤氣|煤氣]][[自殺|自殺]]。
