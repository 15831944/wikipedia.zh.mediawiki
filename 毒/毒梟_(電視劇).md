《'''毒梟'''》（{{lang-en|'''''Narcos'''''}}）是一部美國犯罪網路電視劇，由{{link-en|克里斯·布蘭卡托|Chris Brancato}}、Carlo Bernard和{{link-en|道格·米羅|Doug Miro}}創作和監製。描述毒梟[[巴勃羅·埃斯科瓦爾|巴勃羅·埃斯科瓦爾]]、麥德林集團，以及美國緝毒局探員間的故事。第1季共有10集，在2015年8月28日在[[Netflix|Netflix]]上播放，第二季於2016年9月2日播出，共有10集，Netflix在2016年9月6日已經續訂第3、4季。<ref>https://www.youtube.com/watch?v=1mgXpnfw6Wk</ref>

第一季故事開始於1970-1980年代。第一集場景為1989年哥倫比亞、1973年智利，劇情接續1979年開始。

2018年7月18日，宣布第四季會「重設」為新的Netflix電視劇，名為《{{link-en|毒梟：墨西哥|Narcos: Mexico}}》。新的電視劇安排在2018年11月16日播放，故事會返回1980年代，並且把設定轉移至墨西哥。<ref>{{Cite web |url = https://www.hollywoodreporter.com/live-feed/narcos-first-look-at-mexico-reset-season-1127997 |title = Netflix Releases First Look at Reset 'Narcos: Mexico' |author = Jackie Strause |date = 2018-07-18 |publisher = The Hollywood Reporter |language = en}}</ref>

== 故事 ==
埃斯科瓦尔（Wagner Moura 饰）曾是一个走私商人，机缘巧合之下开始接触毒品生意，野心勃勃的他将无法计数的毒品贩卖到美国，成为了富可敌国的大毒枭。与此同时，他集结了圈内几大巨头，成立了声名显赫的麦德林集团，一时间，埃斯科瓦尔的劲头无人能敌。墨菲（Boyd Holbrook 饰）是美国缉毒局派往哥伦比亚的警探，他和搭档潘那（Pedro Pascal 饰）合作，主要负责埃斯科瓦尔的案件。埃斯科瓦尔当上了议员，甚至决心竞选总统，却在国会当众受辱。之后，哥伦比亚政府试图启用引渡法案对付毒枭们，为了对抗政府，埃斯科瓦尔彻底沦为了反社会的恐怖分子，在不断壮大自己的毒品帝国的同时，用炸弹、暗杀和鲜血提醒着哥伦比亚政府，谁才是真正的老大。
