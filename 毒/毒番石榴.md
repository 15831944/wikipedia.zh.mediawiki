'''毒番石榴'''，学名{{lang|la|'''''Hippomane mancinella'''''}}，又名'''马疯大戟木'''，是一種属於[[大戟科|大戟科]][[烏桕族|烏桕族]]的[[喬木|喬木]]植物，原生於[[美國|美國]][[佛羅里達州|佛羅里達州]]、[[巴哈馬群島|巴哈馬群島]]、[[加勒比海|加勒比海]]、[[中美洲|中美洲]]及[[南美洲|南美洲]]北部等<ref>{{cite web |url=http://www.ars-grin.gov/cgi-bin/npgs/html/taxon.pl?19174 |title=Hippomane mancinella |work=[[Germplasm_Resources_Information_Network|Germplasm Resources Information Network]] |publisher=[[美國農業部|United States Department of Agriculture]] |accessdate=2009-01-27 |language=en }}</ref>[[新热带界|新热带界]]地區<ref>{{cite web |url=http://science.bowenwang.com.cn/plants-manchineel.htm |title=毒番石榴 |publisher=博闻网 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20111121055857/http://science.bowenwang.com.cn/plants-manchineel.htm |archivedate=2011-11-21 }}</ref>，有劇毒。其纹理细密的黄褐色木材常被用来做家具。

== 描述 ==
毒番石榴樹可长至6到15米高，樹皮帶灰色，葉片亮綠，厚密、呈椭圆形，开绿色小花。果实如核桃大小，形似苹果，綠色或黃綠色。

毒番石榴樹通常於海岸的[[沙灘|沙灘]]生長，是非常優良的[[防沙林|防沙林]]作物，其根部能有效把沙抓緊，從而令沙灘免於受海浪或強風[[侵蝕|侵蝕]]。

== 毒性 ==
毒番石榴的白色汁液有剧毒，會令皮膚出現好像得了[[痳瘋|痳瘋]]一樣的病徵，因此又名「马疯大戟木」。亦因為這個原因，毒番石榴亦成為了一些雀鳥躲避掠食者的好地方，例如：[[索哥羅鳩|索哥羅鳩]]就愛在毒番石榴生長的地方來躲避家貓。
