'''沼平車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]，為[[行政院農業委員會林務局|林務局]][[阿里山森林鐵路|阿里山森林鐵路]][[阿里山線|阿里山線]]、眠月線、祝山線、沼平線、水山線之[[鐵路車站|鐵路車站]]。

== 車站構造 ==

== 利用狀況 ==
此站為原阿里山線（主線）的總站，也會是未來新闢支線──水山線──的總站。

== 圖片==
[[File:AFR_Train_Hotel.jpg|thumb]]

== 歷史 ==
*1914年3月14日：設站。
*1981年1月11日：不再以此站為主線總站。
*1999年尾至2000年初曾再次成為主線的總站。
