'''沼津市立长井崎中学校'''（{{jpn|j=沼津市立長井崎中学校|rm=Numazu Shiritsu Nagausakichūgakkō}}），一所位于[[日本|日本]][[静冈县|静冈县]][[沼津市|沼津市]]内浦重须的公立中学<ref>{{cite web |title=長井崎中学（静岡県沼津市）の情報（口コミなど） {{!}} みんなの中学情報 |url=https://www.minkou.jp/junior/school/11792/ |website=我们的中学情报(日语) |language=ja}}</ref>，在動漫[[LoveLive!_Sunshine!!|LoveLive! Sunshine!!]]中是'''浦之星女學院'''（{{Lang-ja|浦の星女学院}}）的原型。<ref>{{cite web |title=《Lovelive！Sunshine！》学校原型曝光_国外新闻_南方网 |url=http://cartoon.southcn.com/c/2015-09/09/content_132502513.htm |website=南方网}}</ref>

== 周围环境 ==
学校位于横跨沼津市内浦重须和西浦木负的长井崎山上，东侧为内浦湾，西侧为骏河湾。被蜜柑林及农田包围，地理位置十分偏僻。可远距离观赏[[富士山|富士山]]。

==沿革==
*[[1981年|1981年]]（昭和56年）4月1日 - 開校
*[[1981年|1981年]]（昭和56年）6月 -舉行開校典禮
*[[1990年|1990年]]（[[平成|平成]]2年）4月 - 舉行創校十周年紀念典禮
*[[2018年|2018年]] （平成30年）- 计划合并内浦小学，西浦小学和长井崎中学<ref>{{cite web |title=長井崎中学校区小中一貫校化推進委員会 |url=http://www.city.numazu.shizuoka.jp/kurashi/kyoiku/kyoiku/numazushi/nagaisaki_ikkan.htm |website=沼津市ホームページ |language=ja}}</ref><ref>{{cite web |title=同步废校？浦之星女子学院原型或将合并废校_动漫_腾讯网 |url=http://comic.qq.com/a/20171225/022227.htm |website=腾讯网 |language=zh-CN}}</ref>

== 设施 ==
学校主教学楼为三层L形结构，并设置有[[棒球场|棒球场]],体育馆、非标准体育场及游泳池等设施。
