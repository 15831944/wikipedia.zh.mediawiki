'''甪直古镇'''（[[汉语拼音|拼音]]：Lù Zhí），古稱“[[甫里|甫里]]”(依本地方言音读 pu li)，是[[蘇州|蘇州]][[吴中区|吴中区]][[甪直镇|甪直镇]]的一个古镇区，是一个国家AAAA级旅游景点。非本地居民入古镇区必须购买60元门票，参观古镇的小桥流水风光和各景点。甪直古镇的民居保留明清风格，禁止拆改，区内禁止设工厂、盖新式高楼，禁止汽车、摩托车通行。甪直古镇有深厚的文化底蕴，[[唐代|唐代]]诗人[[陆龟蒙|陆龟蒙]]，[[明代|明代]]诗人[[高启|高启]]都曾居住在甪直，并留下诗作;[[清代|清代]]改良派思想家、政论家和中国最早的办报人[[王韬|王韬]]出生在此<ref>王韬：''《[[弢园文录外编|弢园文录外编]]·弢园老民自传》'' ISBN 7534817102</ref>；著名教育家、作家[[叶圣陶|叶圣陶]]在甪直开始他的文学创作，他以甪直为第二故乡。镇内的[[保圣寺|保圣寺]]是一座千年历史的禅寺，其中保存唐代塑圣[[杨惠之|杨惠之]]所塑的[[保圣寺罗汉塑像|古罗汉像]]，是国宝级文物，还有被[[梁思成|梁思成]]评为“尤为精美”的[[宋代|宋代]]石[[柱础|柱础]]多件<ref>梁思成著 《建筑设计参考图集·第七集柱础简说》</ref>。甪直还有宋明代古桥数十座，有古桥博物馆之称。[[中华人民共和国建设部|中华人民共和国建设部]]和[[国家文物局|国家文物局]]于2003年10月8日将甪直列入[[中国历史文化名镇|中国历史文化名镇]]。此外，[[甪直|甪直]]还同其他两个江南水乡古镇[[周庄|周庄]]、[[同里|同里]]一齐列入[[联合国教科文组织|联合国教科文组织]]申报世界遗产名单<ref>[http://whc.unesco.org/archive/2004/whc04-28com-14ae.pdf 周庄、同里、甪直 入UNESCO 世界遗产后选名单]。
</ref>。

==概况==
[[File:Luzhi3.jpg|thumb]]
[[File:Luduan.jpg|thumb]]

甪直是[[蘇州|蘇州]]东南25[[公里|公里]]処的的小鎮，原属[[江苏|江苏]][[吴县|吴县]]。2000年12月31日，经[[中国国务院|中国国务院]]批准将吴县并入苏州成为[[吴中|吴中]]区，甪直一并划入。甪直建于[[南朝|南朝]]梁[[天监|天监]]二年，距今有有2500年历史，当时[[保圣寺|保圣寺]]建成，甪直与保圣寺同龄。

甪直的古名“甫里”得名于[[唐代|唐代]][[诗人|诗人]][[陆龟蒙|陆龟蒙]]。陆龟蒙隐居在此镇上，因他又名“甫里先生”，人们就称这个地方为“甫里”。甫里后来又叫甪直是因为古时甫里鎮分为甫里、六直两个区。传说古代独角神兽“[[甪端|甪端]]”守护六直，年年五谷丰登。而[[吴语|吴音]]“六”、“甪”同音，后来“六直”就变成“甪直”了。直到[[清朝|清朝]]末年，甫里正式改为元和县甪直鎮。

根据2004年资料记载甪直镇面积 64平方公里，人口78985人<ref>[http://www.xzqh.org/quhua/32js/0506wz.htm 行政区划网站] {{webarchive|url=https://web.archive.org/web/20090516110047/http://www.xzqh.org/quhua/32js/0506wz.htm |date=2009-05-16 }}</ref>。甪直镇，水陆交通方便，苏沪高速公路、苏昆太高速公路和苏州绕城高速公路从镇中穿过，有公交车直达苏州、[[昆山|昆山]]、[[上海|上海]]。甪直镇北的[[吴淞江|吴淞江]]，可容200吨的船舶航行，直通京杭大运河和上海港。

== 景观 ==

水乡甪直有小桥流水，风景优美，有“东方威尼斯”之称。镇上有各式各样的石板桥、石拱桥40座，其中和丰桥是[[宋代|宋代]]建立的，东美桥和东大桥是[[明代|明代]]建立的。甪直古桥富有建筑学研究价值，因此甪直又有“古桥博物馆”之称。甪直古镇入口处的牌楼抱联柱上刻有当年家居甫里的[[明朝|明朝]]诗人[[高启|高启]]的生动地描绘甪直的小桥流水景象的诗作《甫里景》：右联“''长桥短桥杨柳，前浦后浦荷花”''，左联：''“人看旗出酒市，鸥送船归钓家”''；横额是原全国人大常委会副委员长费孝通题词：''“神州水乡第一镇”''。

===保圣寺罗汉塑像===
[[File:Bao_Sheng_Temple.jpg|thumb]]
[[File:Arhat_sculpture_in_Bao_Sheng_Temple,_Luzhi,_Suzhou.jpg|thumb]]
{{main|保圣寺|保圣寺罗汉塑像}}
[[保圣寺|保圣寺]]是建于[[梁朝|梁朝]]天监二年(公元503年)的千年古刹。极盛在北宋大中祥符六年(公元1013年)。当时叫《保圣禅院》，有五千间殿堂，一千和尚，和[[杭州|杭州]]的[[灵隐寺|灵隐寺]]齐名。
保圣寺博物馆藏有国宝——[[唐代|唐代]][[杨惠之|杨惠之]]手塑的九尊古罗汉像。[[日本|日本]]美术史家[[大村西崖|大村西崖]]教授在考查了甪直保圣寺藏杨惠之塑的古罗汉像，认为是炉火纯青之作。1961年3月4日保圣寺罗汉塑像列入[[中华人民共和国国务院|中华人民共和国国务院]]批准的国家级文物第一批[[全国重点文物保护单位|全国重点文物保护单位]]

===陆龟蒙墓===
[[陆龟蒙|陆龟蒙]]墓近保圣寺。唐代诗人陆龟蒙不愿做官隐居甫里，生时高风亮节。后人为了纪念他，因为他生前喜欢看斗鸭，开了一方水塘，种上荷花，养上几只鸭子，水塘中间有一座清风亭，相传是甫里先生读书的地方。甫里先生墓在清风亭侧，墓碑上刻有“唐贤甫里先生之墓”。在清风亭后有两株参天古银杏，相传为先生手植，已有一千多年了。

===沈伯寒宅===
[[沈伯寒|沈伯寒]]是晚清教育家，早年留学日本早稻田大学，专攻教育学。后来开办甫里小学——今天甪直小学的前身，沈伯寒宅在甪直香花弄，一座石库门大院，青砖照壁镌有《漪韵》二字。

===王韬纪念馆===
[[王韬|王韬]]是中国最早的改良派思想家，政论家和新闻记者，生于清道光八年(1828年)苏州府元和县甫里镇(今甪直镇)，名利宾，字兰卿，号仲弢、天南遁叟、弢园老民。他在[[上海|上海]]和[[英国|英国]][[传教士|传教士]]翻译中国第一部汉语圣经。在[[苏格兰|苏格兰]]帮助[[汉学家|汉学家]][[理雅各|理雅各]]翻译《[[十三经|十三经]]》成英文。返回[[香港|香港]]后创办《[[循环日报|循环日报]]》——世界上第一家中文日报，被尊为中国第一报人。王韬纪念馆在甪直中市街6号，一座清式建筑内。

===叶圣陶纪念馆===

[[File:Ye_Shengtao_Memorial_Park.jpg|thumb]]

著名教育家、作家[[叶圣陶|叶圣陶]]纪念馆，建在叶圣陶1917年至1922年任教过的第五高小的原址上。叶圣陶在甪直开始他的文学创作，他以甪直为第二故乡。

===万盛米行 ===
万盛米行是叶圣陶著名小说《多收了三五斗》里的万盛米行的原形，位于南市下塘街，前有售米柜台，后有大院陈列当时的农具。

===[[萧芳芳|萧芳芳]]演艺馆===
* [[香港|香港]]电影明星，1995年柏林电影节最佳女演员得主[[萧芳芳|萧芳芳]]表演艺术纪念馆，位于萧家老宅。

== 特产 ==
[[File:Pulileg.jpg|thumb]]

* 甫里蹄和甫里鸭：甪直镇的两道名菜为甫里蹄和甫里鸭。甫里蹄和甫里鸭是按照诗人甫里先生的配方制作的，他经常用这两道菜招待客人。几乎每个江南古镇都有自己的蹄髈，实际做法和口味大同小异，不过甪直农村一直就有红烧蹄髈的做法，但凡宴席都喜欢用柴火和大锅烧制红烧蹄髈供宾客享用，味道确实美味可口。
* 甪直萝卜干：根据清代配方酿成腌制的萝卜干。镇上酱品厂之产品。

== 新区 ==
甪直新区有宽阔的马路、崭新的商店、餐馆和住宅。因为离苏州市不远，近年不少苏州市人因市内房价飞涨，迁居甪直，白天到苏州市工作。甪直镇有比较良好的保护古镇的措施，例如设立保护区，新式楼房、新式马路只能在新区开发等等。因此在2004年甪直镇被[[联合国教科文组织|联合国教科文组织]]授予保护古镇杰出成就奖.
