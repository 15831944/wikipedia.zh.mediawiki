'''業處'''（{{lang-pi|kammaṭṭhāna}}；{{lang-sa|karmasthāna}}），佛教術語，是[[南傳佛教|南傳佛教]]常提及的一項修行[[法門|法門]]<ref>《中[[阿含經|阿含經]]·根本分別品·鸚鵡經》：「瞿曇！何因何緣？彼眾生者，俱受人身，而有高下，有妙、不妙。所以者何？瞿曇！我見有短壽，有長壽者，見有多病，有少病者，見不端正，有端正者，見無威德，有威德者，見有卑賤族，有尊貴族者，見無財物，有財物者，見有惡智，有善智者。世尊答曰：彼眾生者，因自行業，因業得報，緣業，依業，業處，眾生隨其高下，處妙、不妙。」</ref>；“[[業_(佛教)|業]]”指造作、行動，“業處”即心行動之處，在[[禪修|禪修]]時心所關注的處所<ref name=szb>[http://wiki.sutta.org/wiki/%E4%B8%9A%E5%A4%84 上座部佛教百科【業處】]</ref>。巴利語系佛教論典《[[清淨道論|清淨道論]]》舉出四十種，稱爲'''四十業處'''<ref name=清淨道論>[[覺音|覺音]]尊者《[[清净道论|清净道论]]·第三·说取业处品》</ref>。

==四十業處==
四十業處，是四十種禪修時心的著落處，即心關注點在何處<ref name=清淨道論/>。

; 十遍
: [[大種|地]]、[[大種|水]]、[[大種|火]]、[[大種|風]]、青、黄、赤、白、[[光明想|光明]]、[[空界|限定虚空]]
; 十不凈（[[墓園九相|墓園九相]]、[[不淨觀|不淨觀]]、[[白骨觀|白骨觀]]）
: 膨張、青瘀、膿爛、断壞、食残、散乱、斬斫離散、血塗、蟲聚、骸骨
; 十[[随念|随念]]
: [[佛隨念|佛]]、法、僧、[[佛教戒律|戒]]、[[佈施|施]]、[[天_(佛教)|天]]、[[死隨念|死]]、[[三十二身分|身至]]、[[安那般那念|安般]]、[[涅槃|寂止]]
; [[四無量心|四無量心]]（[[四梵住|四梵住]]）
: [[慈_(佛教)|慈]]、[[悲_(佛教)|悲]]、[[喜_(佛教)|喜]]、[[捨_(佛教)|捨]]
; [[禅定|四無色界]]
: [[空無邊處|空無邊處]]、[[識無邊處|識無邊處]]、[[無所有處|無所有處]]、[[非想非非想天|非想非非想處]]
; [[食_(佛教)#.E9.A3.9F.E5.8E.AD.E6.83.B3|食厭想]]
; [[大種|四界]]分別（[[界分別觀|四界差別]]）
: 地・水・火・風

==分類==
《清淨道論》將業處分為止业处（{{lang-pi|samatha-kammaṭṭhāna}}）与观业处（{{lang-pi|vipassanā-kammaṭṭhāna}}）两大类。<ref name=清淨道論/>

==安那般那念業處==
[[安那般那念|安那般那念]]（入出息念）是十隨念之一，指通過細緻地關注[[呼吸|呼吸]]的一入一出，澄清蕪雜的妄想，顯現內在{{link-ja|光明 (佛教)|光明|光明}}，培養[[力_(佛教)|定力]]的一種修行法門。[[安那般那念|安那般那念]]（入出息念）业处是非常重要的修行法門，也是[[根機|根機]]普適的禪修入門法，以此為例解釋何謂“禪修時心的著落處”。
#呼吸本身就是造作，是一種业处，故說专注呼吸，细致地观察一入一出，也就是專注业处。
#专注点再具体一点，是指鼻头一带区域，将心落在自己的鼻头处，不落在任何外境，也不随便移动到其他部位，这样可以一直保持专注，止住妄想。<ref name=szb/>
