'''魚乾片'''（{{jpn|j=削り節|hg=けずりぶし|rm=Kezu ri bushi}}），是一種把[[鰹魚乾|鰹魚乾]]或[[鯖魚|鯖魚]]乾、[[沙丁魚|沙丁魚]]乾、[[鮪魚|鮪魚]]乾等，刨成一片一片的[[調味品|調味品]]。

== 鰹魚乾片 ==
'''鰹魚乾片'''，又稱'''柴魚片'''、'''木魚花'''、'''木魚碎'''<ref>[http://hk.apple.nextmedia.com/supplement/food/art/20071224/10566408 木魚碎＝花鰹]，[[蘋果日報_(香港)|蘋果日報]]</ref>，是將鰹魚乾刨成薄片而成，為魚乾片最常見的種類之一。
[[file:Katsuobushi_kezuriki.jpg|thumb]]

== 用途 ==
基本上當作一種調味品使用<ref>{{Cite web |url=https://tw.sushiandsake.net/special/column/detail_1 |title=日本美食百科全書> 專欄> 柴魚片 |access-date=2014-10-01 |archive-url=https://web.archive.org/web/20141006095352/https://tw.sushiandsake.net/special/column/detail_1 |archive-date=2014-10-06 |dead-url=yes }}</ref>。

* 柴魚片常做為日式高湯的底料，也時常加入[[味噌湯|味噌湯]]的湯底，增加味噌湯的鮮味及甜味。
* 柴魚片可用於[[皮蛋|皮蛋]][[豆腐|豆腐]]及涼拌豆腐作為[[調味品|調味品]]。
* 在日本可將柴魚片灑在[[白飯|白飯]]上面，淋上[[醬油|醬油]]後食用，稱為「{{Link-ja|貓飯|猫まんま}}」。
* 灑在[[章魚燒|章魚燒]]做為佐料。
* 市面上的營業餐廳與市售泡麵調味包經常利用營業用[[鰹魚粉|鰹魚粉]]、[[柴魚粉|柴魚粉]]做為湯頭的原料，是甘甜湯頭速成的方案。

== 可能致癌物 ==
由於柴魚是由高溫烘烤煙燻製成，而可能產生[[苯芘|苯芘]]這種致癌物質於柴魚焦黑外皮上，故刨片製成柴魚片食用前，應去除焦黑的外皮。<ref>[https://news.tvbs.com.tw/news/detail/general/36496 追苯芘禍首！直擊柴魚廠 刨皮去癌降險]</ref>。
