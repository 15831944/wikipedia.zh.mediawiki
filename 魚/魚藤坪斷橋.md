'''魚藤坪斷橋'''又名'''龍騰斷橋'''，原名魚籐坪橋，位於[[臺灣|臺灣]][[苗栗縣|苗栗縣]][[三義鄉_(臺灣)|三義鄉]]龍騰村，為[[縱貫線_(鐵路)|縱貫線]]鐵路（後屬[[臺中線|臺中線]]）全線開通前一年（即1907年）所興建完成的磚造橋墩、鋼鈑梁及桁架梁混合型式之橋樑{{r|鐵道史}}。1999年[[921大地震|921大地震]]後，行政院文化建設委員會（今[[中華民國文化部|文化部]]前身）於2000年將魚藤坪斷橋指定為震災紀念物，苗栗縣政府於2003年依文化資產保存法公告指定為縣定古蹟{{r|魚藤}}。

== 沿革 ==
魚藤坪斷橋位於[[縱貫線_(鐵路)|縱貫線]]鐵路（後屬[[臺中線|臺中線]]）十六份（今[[勝興車站|勝興站]]）—大安（今舊[[泰安車站_(舊山線)|泰安站]]）間，跨越小型溪谷，由三叉河出張所所長稻垣兵太郎設計{{r|魚藤}}，因河床無落墩，非屬大型橋梁工程，故[[臺灣總督府交通局鐵道部|臺灣總督府鐵道部]]指定工程技術相對其他包商薄弱的久米組負責興建魚藤坪橋{{r|RN184}}。此橋1906年4月30日開工興建，為拱橋、鋼鈑梁橋及桁架橋混合型式之橋樑{{r|鐵道史}}，中央桁架樑則由美國技師Coopper Schneider設計{{r|魚藤}}{{NoteTag|1=英文版維基記載之技師為{{cite web|language=|url= https://en.wikipedia.org/wiki/Longteng_Bridge
|title= Theodore Cooper 及 Charles Conrad Schneider |author=|publisher=|date=|accessdate=}}。}}，全部工程於1907年6月1日完工{{r|鐵道史}}。1935年4月21日發生[[新竹台中大地震|新竹台中大地震]]，[[新竹州|新竹州]]、[[台中州|台中州]]震災嚴重，此區隧道、橋樑、鐵軌損毀至鉅，此橋亦被震毀，造就此一斷橋之景象。

[[File:1935_Shinchiku-Taichū_earthquake.jpg|250px]]

[[File:The_damaged_Gyo_tō_heida_Bridge_after_the_1935_quake.jpg|250px]]

新竹臺中州大地震震後，魚藤坪橋已不堪使用，且橋體南北兩側均為急彎彎道，因此[[臺灣總督府交通局鐵道部|臺灣總督府交通局鐵道部]]臺中線震災復興事務所放棄改建原橋，而在原橋西側（下游側）80公尺處另行新建第二代魚藤坪橋<ref>蔡宜儒：〈山線巡禮～魚藤坪斷橋〉，《鐵道情報》88期，[[中華民國鐵道文化協會|中華民國鐵道文化協會]]，[[臺北市|臺北市]]，1996年6月號，頁1。</ref>。重建新橋期間為了運送建材，工程單位將舊橋龜裂及有安全顧慮的結構敲除，搭上施工鷹架，方便工程列車運送建材<ref>[[臺灣總督府交通局鐵道部|臺灣總督府交通局鐵道部]]臺中震災復興事務所：《昭和拾年臺灣鐵道震災誌》，[[臺北市|臺北市]]，1939年，「震害の跡」與「臺中線復興の跡」相關照片。</ref>。新橋（第二代魚藤坪橋）於1938年竣工，通車後取代原來的第一代魚藤坪橋（即魚藤坪斷橋）{{r|魚藤}}。

[[File:Gyo_tō_heida_Bridge_after_the_1935_quake_01.jpg|250px]]將舊橋破損斷裂部分敲除後再搭起支架，以利工程列車運行，進行沿線隧道及橋梁重建。]]

1999年之[[921大地震|921大地震]]時，又造成第四號橋墩上半部被震毀崩落{{r|魚藤}}。

[[File:Longteng_bridge.jpg|thumb]]

2010年，三義－泰安舊站之間的舊山線復駛計畫中在此設置[[魚藤坪車站|魚藤坪車站]]<ref>[http://tw.nextmedia.com/subapple/article/art_id/32486904/IssueID/20100505/sec_id/17 停駛13年 苗栗舊山線6月復駛]</ref>。

== 建築設計 ==
魚藤坪斷橋設計之原構想為自北端依序為1孔鋼鈑梁、中央1孔60.96公尺桁架梁、南端2孔鋼鈑梁，並沒有拱橋的設計，後來橋型變更為磚造橋墩之拱橋、鋼鈑梁橋及桁架橋混合型式之橋樑，除南、北端各1座橋台以外，計有8個[[橋墩|橋墩]]、9個橋孔（跨徑9.14公尺半圓拱6孔{{NoteTag|1=拱橋樣式類似於1891年完工的JR[[信越本線|信越本線]]{{cite web|language=|url= http://www.adnet.jp/nikkei/kindai/11/
|title=舊「碓冰第三號橋」|author=|publisher=|date=|accessdate=}}。}}、18.3公尺上承式鋼鈑梁2孔、60.96公尺上承式巴的摩爾式桁架梁1孔{{NoteTag|1=魚藤坪斷橋中央主跨的上承式巴的摩爾式桁梁（Baltimore Truss、ボルチモアトラス），其長度及樣式與1908年完成的JR[[磐越西線|磐越西線]]{{cite web|language=|url= http://www.jasbc.or.jp/routepress21st/rp21st-11-01.php|title=「一之戶川橋」|author=|publisher=|date=|accessdate=}}中央主跨桁梁相同。參閱小野田滋：《鉄道構造物探見》頁130〈図⑤ トラス橋の種類（2）〉與頁131照片（東京，JTB，2003年）。}}），加計橋墩寬度後全長165.8公尺，橋面坡度自南而北為千分之25上坡，橋北端4連拱處為半徑300公尺彎道{{r|鐵道史}}。

斷橋拱橋橋體之紅磚採丁順砌法疊砌，圓拱拱圈以六層紅磚砌拱，拱橋最上端原軌道面兩旁設有類似[[女兒牆|女兒牆]]之矮牆，形成道碴槽，以防止石碴掉落橋下。至於南北端之橋台翼牆，則採用灰色石塊砌築，有別於整座斷橋的紅磚結構意象<ref>蔡宜儒：〈龍騰斷橋—鐵道舊線跡第一名景賞析〉，《鐵道旅行》，人人出版，[[新北市|臺北縣]]，2003年夏季號，頁64～65。</ref>。

== 傳說 ==
據傳當年在鯉魚潭中有鯉魚精作祟，當地居民便到處種植有毒的[[厚果崖豆藤|魚藤]]，並將東面狀似[[青龍偃月刀|青龍偃月刀]]刀背的山命名為[[關刀山_(苗栗縣)|關刀山]]，用意是以關刀斬斷魚藤毒殺鯉魚精之意，即「關刀斬魚藤，魚藤毒鯉魚」。後來當地居民認為魚藤名稱不雅而將地名改為「龍騰」，因此魚藤坪斷橋也被稱為龍騰斷橋{{r|魚藤}}。
