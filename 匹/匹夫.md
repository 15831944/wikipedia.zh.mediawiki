《'''匹夫'''》（英文名：'''An Inaccurate Memoir'''）， 2012年4月23日上映于[[中国大陆|中国大陆]]的动作片<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2012-04-19/23543610197.shtml |title=《匹夫》提档23日18点上映 黄晓明率匪帮干大事 |accessdate= 2012年4月19日|work=|publisher=新浪网 |language = zh-cn}}</ref>。

== 剧情 ==
导演[[杨树鹏|杨树鹏]]在[[北京|北京]]发布会上说，“这部电影讲述乱世之中，一群人如何做出一件有勇气、有担当的事情。”<ref>{{cite web |url=http://ent.qq.com/a/20120216/001306.htm |title= 杨树鹏《匹夫》发先导海报 联手黄晓明热血上阵|accessdate= 2012年|work=|publisher=腾讯网 |language = zh-cn}}</ref>

== 主题 ==
导演杨树鹏解释说：“匹夫二字词性复杂，它可以是褒义的、中性的，甚至是贬义的。电影里的角色也是这样复杂，他们虽是底层人物，但却有勇气，有担当。古人说匹夫之怒，血溅五步，就是这个意思。”<ref>{{cite web |url=http://ent.qq.com/a/20120305/001202.htm |title=《匹夫》帮亮相 杨树鹏联手“大当家”黄晓明 |accessdate= 2012年|work=|publisher=腾讯网 |language = zh-cn}}</ref>

== 特色 ==
片中主角都有“[[三宝|三宝]]”，[[黄晓明|黄晓明]]饰演的大当家的三宝是“步枪、烈酒、大皮袄”；[[张歆艺|张歆艺]]饰演的小姑奶奶的三宝是“辣酱、面膜、爱叨叨”；[[张译|张译]]饰演的高栋梁的三宝是“挨打、吃土、玩命跑”。三人不同的性格搭配，成为影片的一大亮点。<ref>{{cite web |url=http://ent.chinadaily.com.cn/dyneidi/2012-03-06/content_5330271.html |title=张译成《匹夫》线索人物 角色遭遇复杂身份成谜 |accessdate= 2012年|work=|publisher=中国日报网 |language = zh-cn}}</ref>

== 花絮 ==
在拍摄《匹夫》期间，[[黄晓明|黄晓明]]每天坚持吃水煮的鸡胸脯肉和[[蔬菜|蔬菜]]且不加[[盐|盐]]，天天锻炼[[肌肉|肌肉]]，读[[英语|英语]]，拍戏时，光着膀子就露出一副健壮的身材。<ref>{{cite web|url=http://www.yangtse.com/system/2012/03/06/012864355.shtml|title=黄晓明为演《匹夫》吃了两年白水煮鸡胸|accessdate=2012年|work=|publisher=扬子晚报网|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160305011041/http://www.yangtse.com/system/2012/03/06/012864355.shtml|archivedate=2016-03-05}}</ref><ref>{{cite web|url=http://epaper.jinghua.cn/html/2012-03/06/content_767899.htm|title=晓明秀块儿《匹夫》有型|accessdate=2012年|work=|publisher=京华网|language=zh-cn}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
