'''兆康站'''（{{lang-en|'''Siu Hong Station'''}}）位於[[香港|香港]][[新界|新界]][[屯門區|屯門區]][[兆康苑|兆康苑]]的東面，高架於[[屯門河|屯門河]]之上，是[[港鐵|港鐵]][[西鐵綫|西鐵綫]]其中一個車站，連接[[香港輕鐵|輕鐵]][[兆康站_(輕鐵)|兆康站]]，於2003年12月20日啟用。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Siu Hong Station Concourse Shops 201904.jpg | caption1 = 大堂商店
| image2 = MTR SIH (1).JPG | caption2 = 輕鐵轉乘通道
}}
兆康站內設有不同類型的商店供乘客購物，例如有[[便利店|便利店]]、糖果店、[[麵包糕餅店|麵包糕餅店]]、物業顧問等服務，所有商店主要分佈在非收費區內<ref name="shop"/>。

此外，兆康站也有不少自助服務設施供乘客享用，包括[[自動櫃員機|自動櫃員機]]、[[自動販賣機|自動售賣機]]和自動照相機<ref name="shop"/>。大堂內亦設有[[香港郵政|香港郵政]][[郵箱|郵箱]]方便乘客在車站內投寄[[郵件|郵件]]<ref>{{cite web|url=http://www.mtr.com.hk/chi/getting_around/post.html|title=鐵路四處遊＞郵箱服務|work=港鐵網頁|publisher=港鐵公司|accessdate=2011年12月10日|language=zh-hant}}</ref>。由於兆康是輕鐵轉乘車站，所以大堂內設有輕鐵「出站」收費器，方便乘客由輕鐵轉乘西鐵綫時，在入閘前完成[[八達通|八達通]]內的輕鐵乘車紀錄。大堂收費區內亦設有[[廁所|洗手間]]。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Siu Hong Station 2018 06 part1.jpg | caption1 = 1號月台﹙往屯門方向﹚
| image2 = Siu Hong Station 2018 06 part2.jpg | caption2 = 2號月台﹙往紅磡方向﹚
}}
兆康站設有2個月台，並以[[島式月台|島式月台]]的方式排列，所有月台均在興建時裝設[[月台幕門|月台幕門]]。月台位於[[屯門河|屯門河]]上方以高架形式興建，但月台與位於地面的輕鐵月台平排和同一水平。<ref>{{Cite web|publisher=MTR之今昔|title=兆康站車站結構圖|url=http://mtr.hk365day.com/Railway/line/WRL/SIH.html|author=港鐵公司|accessdate=2009-10-19|work=MTR之今昔|date=2009年8月}}</ref>此外，為了避免對附近居民造成滋擾，大部分月台被混凝土圍封，使月台裏面的人以為身處地底。該站內亦設緊急車輛通道。

==接駁交通==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Siu Hong Station (South) Public Transport Interchange 2013.jpg | caption1 = 兆康站（南）公共運輸交匯處
}}
兆康站在[[西鐵綫|九廣西鐵]]興建前，[[九廣鐵路公司|九廣鐵路公司]]早已於當地設立輕鐵兆康站，因此九廣西鐵設計初期，早已計劃將兆康站發展成為西鐵綫及輕鐵的綜合轉乘車站，所以車站設有大量行人通道連接輕鐵月台，方便乘客由西鐵綫轉乘輕鐵路綫往返屯門各區。

此外，兆康站的南北兩端各設有[[公共運輸交匯處|公共運輸交匯處]]；分別為[[兆康站（南）公共運輸交匯處|兆康站（南）公共運輸交匯處]]及[[兆康站（北）公共運輸交匯處|兆康站（北）公共運輸交匯處]]，設有巴士、小巴及的士站，並設有[[港鐵巴士|港鐵巴士]]路綫K51及K58可供乘客轉乘巴士前往[[富泰邨|富泰邨]]、[[大欖|大欖]]或[[青山灣|青山灣]]等地。

==歷史==
