'''堺泉北道路'''（{{lang|ja|堺泉北道路／さかいせんぼくどうろ}}）是從[[大阪府|大阪府]][[高石市|高石市]]至大阪府[[堺市|堺市]][[中區_(堺市)|中區]]的[[收費道路|收費道路]]（{{link-ja|國道26號|国道26号}}）。

全線被指定為[[地域高規格道路|地域高規格道路]]計畫路線上的'''堺泉北道路'''（法定路線名） <ref>{{cite web|date = |title = 地域高規格道路|url = http://www.kkr.mlit.go.jp/road/kansen/tiikikoukikaku_a.html|format = |page = |publisher = 国土交通省近畿地方整備局道路部|accessdate = 2012-6-12}}</ref>。因為通行費採均一100圓收費的關係，有著「100圓高速（{{lang|ja|100円高速}}）」的稱號。2018年4月1日起，由{{link-ja|大阪府道路公社|大阪府道路公社}}交給[[西日本高速道路|西日本高速道路]]（NEXCO西日本）管理，同時收費也改為里程計費制。

高速公路路線編號被編為 {{Ja Exp Route Sign|E90}} <ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/list/index.html|title=高速道路ナンバリング一覧|accessdate=2017年3月3日|publisher=国土交通省}}</ref><ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/map/index.html|title=高速道路ナンバリング路線図|accessdate=2017年3月3日|publisher=国土交通省}}</ref>。

== 概要 ==
全線通過{{link-ja|大阪府道36號泉大津美原線|大阪府道36号泉大津美原線}}新道的中央，從[[1986年|1986年]]（昭和61年）12月開始，花了約4年的時間建設。　

與[[阪和自動車道|阪和自動車道]]和[[阪神高速4号灣岸線|阪神高速4号灣岸線]]直接連接，是往[[關西国際機場|關西国際機場]]的聯絡道路，特別是來自[[大阪府|大阪府]]北部及東部、[[奈良縣|奈良縣]]北部、[[京都府|京都府]]、[[滋賀縣|滋賀縣]]、[[兵庫縣|兵庫縣]]南部内陸部及北部、[[中國地方|中國地方]]、[[東海地方|東海地方]]、[[北陸地方|北陸地方]]等方向，本道路擔綱前述地區前往機場的主要路線的角色。

本道路並非[[快速道路|快速道路]]，但適用與快速道路同等的通行規定。因此，道路標誌的底色為藍色。

=== 路線資料 ===
* {{link-ja|道路構造令|道路構造令}}：第4種第1級
* 車道數：完成4車道（部分除外）
* 設計速度：60 km/h
* 事業費：208億圓<ref>{{cite web|date = |title = 大阪府／堺泉北有料道路|url = http://www.pref.osaka.jp/doroseibi/kansendouro/sakaisenboku.html|format = |page = |publisher = 大阪府都市整備部|accessdate = 2012-6-14}}</ref>
;詳細請參考[[#車道與速限|車道與速限]]。

== 歷史 ==
*[[1991年|1991年]]（平成3年）[[3月15日|3月15日]]：助松出入口 - 平井出入口間通車
*1991年（平成3年）[[12月7日|12月7日]]：平井出入口 - 堺系統交流道間通車
**與[[阪和自動車道|阪和自動車道]]的[[美原北交流道|美原北交流道]] - [[堺交流道|堺交流道]]間一通起車。
*[[1993年|1993年]]（平成5年）[[11月4日|11月4日]]：助松系統交流道 - 助松出入口間通車（'''全線通車'''）
**與[[阪神高速4號灣岸線|阪神高速4號灣岸線]]的[[出島出入口_(大阪府)|出島出入口]] - [[泉大津出入口|泉大津出入口]]間一起通車。
*[[2006年|2006年]]（平成18年）[[2月8日|2月8日]]：菱木收費站（助松方向）廢除並導入[[ETC|ETC]]。大平寺入口及平井入口、平井收費站（全為往助松方向）等設施啟用（'''全設施完成'''）
**與阪和自動車道的{{link-ja|堺收費站|堺料金所}}（下行線，和歌山方向）及[[岸和田收費站|岸和田收費站]]（上行線，大阪方向）一起廢除。<ref>{{cite web|date = 2006-1-19|title = 阪和道本線料金所ノンストップ通行開始|url = http://corp.w-nexco.co.jp/corporate/release/kansai/pdfs/h18/0119.pdf|format = PDF|page = 1 - 7|publisher = 西日本高速道路、大阪府道路公社|accessdate = 2012-6-14}}</ref>
* [[2018年|2018年]][[4月1日|4月1日]]：由大阪府道路公社交給[[西日本高速公路|西日本高速公路]]管裡。名稱由「'''堺泉北收費道路'''」（{{lang|ja|堺泉北有料道路}}）變更為「'''堺泉北道路'''」（{{lang|ja|堺泉北道路}}）。收費方式由均一收費制變更為里程收費制<ref>{{cite web|date = 2018-3-2|title = 平成30年4月1日からの近畿圏の新たな高速道路料金について|url = http://corp.w-nexco.co.jp/corporate/release/hq/h30/0302/ |publisher = 西日本高速道路|accessdate = 2018-3-2}}</ref>。原路線被指定為{{link-ja|大阪府道36號泉大津美原線|大阪府道36号泉大津美原線}}，隨本次管理權移轉更改為{{link-ja|國道26號|国道26号}}<ref>{{Cite journal|url=http://kanpou.npb.go.jp/20180330/20180330g00073/20180330g000730012f.html|2=和書|title=平成30年近畿地方整備局告示第69号|journal=官報|issue=平成30年号外第73号|page=12|publisher=独立行政法人国立印刷局|date=2018-03-30|accessdate=2018-04-01|format=PDF|author=|archive-url=https://web.archive.org/web/20180402035441/http://kanpou.npb.go.jp/20180330/20180330g00073/20180330g000730012f.html|archive-date=2018-04-02|dead-url=yes}}</ref>。
