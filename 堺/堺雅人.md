'''堺雅人'''（{{jpn|j=堺 雅人|hg=さかい まさと|rm=Sakai Masato}}，{{bd|1973年|10月14日|catIdx=Sakai Masato}}），[[日本|日本]]知名男[[演員|演員]]，隸屬於{{link-ja|田邊事務所|田辺エージェンシー}}，妻子為演員[[菅野美穗|菅野美穗]]。[[早稻田大學|早稻田大學]]第一文學部中國文學專修肄業。因演出《[[半澤直樹|半澤直樹]]》與《[[王牌大律師|王牌大律師]]》系列而廣為華人圈觀眾所知悉。[[日劇學院賞|日劇學院賞]]史上唯一一位連續兩屆最佳男主角得主，演技曾被評為「喜怒哀樂全用笑容來表現」<ref name="tbs-yuri">於2012年10月12日播放的《[[大沢悠里のゆうゆうワイド|大沢悠里のゆうゆうワイド]]》</ref>。

== 簡歷 ==
===早年生涯===
於1973年10月14日在[[兵庫縣|兵庫縣]][[神戶市|神戶市]]出生。早年已開始參加與演藝有關的活動。在高中時代曾加入話劇社。之後，入讀[[早稻田大學|早稻田大學]]，並於1992年參與以早稻田大學演劇研究會為主體的「東京ORANGE」劇團，活躍於舞台。他亦成為該劇團的招牌演員，在當時被稱為「早稻田的王子」。<ref>{{cite web|url=http://reymaryya.seesaa.net/article/160724429.html|title=草なぎ剛は酒にも弱いが頭も弱い。堺雅人に馬鹿っぷりをアピール！|publisher=reymaryya|author=REN|date=|accessdate=2013-09-12}}</ref>

===職業生涯===
1994年，堺雅人為了能專注於演藝事業，便中止了自己在早稻田大学的学业，加入了经纪公司{{link-ja|田邊事務所|田辺エージェンシー}}並成为一名演员。<ref>{{cite web|url=http://www.tanabe-agency.co.jp/talent/sakai_masato/|title=堺 雅人|publisher=田辺エージェンシー|date=2013-07-07|author=|accessdate=2013-09-12|deadurl=yes|archiveurl=https://web.archive.org/web/20131020052418/http://www.tanabe-agency.co.jp/talent/sakai_masato/|archivedate=2013-10-20}}</ref>他之後便開始活躍於舞台劇、電影及電視劇。但是，他的父母卻反對他從事演藝業，因此與他断绝关系长达七年。<ref name="履歴書">[http://www.inlifeweb.com/reports/report_3884.html INLIFE 男の履歴書　堺雅人]</ref>父母認可他的演員身份，是從NHK晨間劇「奧黛麗」演出開始。儘管堺雅人為了能專注於演藝事業而決定輟學，但未能獲得知名度。於1994年至2004年期間，仍然沒有機會成名。十年過去，飾演的全是龍套角色。然而即使是這樣，仍然深深熱愛著演戲。<ref name="履歴書"/>

2004年，於[[NHK|NHK]]年度[[大河劇|大河劇]]《[[新選組！|新選組！]]》中飾演主要角色新選組總長「[[山南敬助|山南敬助]]」。在劇中的演出頗受歡迎，NHK因此接獲許多觀眾要求不要讓山南敬助自殺而死。而在山南敬助切腹的第33集，NHK還特例的重播，因此劇開始獲得知名度。<ref>{{cite web|url=http://allabout.co.jp/gm/gc/198987/|title=テレビドラマを支える小劇場系俳優　新選組俳優は早い・うまい・安い|publisher=allabout|date=2004-12-07|author=黒田 昭彦|accessdate=2013-09-12}}</ref>翌年4月，於日本電視雜誌《TV Bros.》中獲選為「最喜歡的男人」第一名。在之後的數年裡，參與了多部電影的演出，包括《[[日式牛仔一品鍋|日式牛仔一品鍋]]》、《[[蜂蜜與四葉草|蜂蜜與四葉草]]》、《[[放課後_(電影)|放课后]]》和《[[超越巔峰_(電影)|超越巔峰]]》。於2008年，橫掃多項演藝獎項的最佳男配角，其中包括[[日本電影金像獎|日本電影金像獎]]、[[每日電影獎|每日電影獎]]、[[藍絲帶獎_(電影)|藍絲帶獎]]、[[日刊體育電影大賞|日刊體育電影大賞]]、[[電影旬報|電影旬報]]獎、[[報知電影獎|報知電影獎]]等<ref name="2008 awards">[http://douga.nikkansports.com/interview/interview_73339.html 堺雅人に助演男優賞／映画大賞 - ニッカン動画] {{webarchive|url=https://web.archive.org/web/20130704162200/http://douga.nikkansports.com/interview/interview_73339.html |date=2013-07-04 }}</ref>。同年，堺雅人於[[NHK大河劇|NHK大河劇]]《[[篤姬_(電視劇)|篤姬]]》一劇中飾演主要角色[[江戶幕府|江戶幕府]]将军[[德川家定|德川家定]]，令他獲得更高的知名度。<ref>{{cite web|url=http://www.ippongi.com/2008/12/18/%E7%AF%A4%E5%A7%AB%E3%81%AE%E8%A6%96%E8%81%B4%E7%8E%87%E3%81%8C%E3%82%88%E3%81%8B%E3%81%A3%E3%81%9F%EF%BC%91%EF%BC%91%E3%81%AE%E7%90%86%E7%94%B1/|title=大河ドラマ「篤姫」の視聴率がよかった１１の理由|publisher=一本気新聞|date=2008-12-18|accessdate=2013-09-12}}</ref><ref>{{cite web|url=http://www9.nhk.or.jp/taiga/special/best10_06/index2.html|title=第5位 篤姫 -エピソードで振り返る! 人気大河「ベスト10」|publisher=大河ドラマ|date=2008|accessdate=2013-09-12}}</ref>自從他在《篤姬》中演出後，開始獲邀飾演其他劇集和電影的主角，其中包括《[[南極料理人|南極料理人]]》、《{{link-ja|庫希歐大佐|クヒオ大佐}}》、《[[Golden_Slumbers|Golden Slumbers]]》、《[[武士的家計簿|武士的家計簿]]》、《[[阿娜答得了憂鬱症|阿娜答得了憂鬱症]]》等電影。<ref name="movie jp">[http://eiga.com/person/32043/ 堺雅人 - 映画.com]</ref><ref name="movie IMDb">{{imdb name|0757007|堺雅人}}</ref>

2009年3月，堺雅人在電視雜誌《TVnavi》內所連載的散文「文·堺雅人」在連載至第50回後正式完結。<ref name="book">[http://www.amazon.co.jp/dp/4167838710 文・堺雅人 (文春文庫) [文庫] - Amazon]</ref>翌年7月，主演電視劇《[[JOKER_不被原諒的搜查官|JOKER 不被原諒的搜查官]]》，是他首次在富士電視台連續劇上擔任主角。<ref>[http://loca.ash.jp/info/2010/d201007_joker.htm ジョーカー許されざる捜査官 ロケ地情報]</ref>

2012年，堺雅人主演富士電視台連續剧《[[Legal_High|Legal High]]》獲得成功及極大的口碑。<ref>{{cite web|url=http://headlines.yahoo.co.jp/hl?a=20130829-00000501-sanspo-ent |title=堺雅人、復活「リーガルハイ」で岡田将生と激突 |publisher=Yahoo! Japan |date=2012-08-29 |author=サンケイスポーツ |accessdate=2013-09-12 |archiveurl=https://archive.is/20130912153040/http://headlines.yahoo.co.jp/hl?a=20130829-00000501-sanspo-ent |archivedate=2013-09-12 |deadurl=yes }}</ref>之後，他亦主演电影《盗钥匙的方法》，並因此获提名為第35回日本奧斯卡最佳男主角。他後來又主演TBS电视台連續剧《[[大奧_～誕生【有功·家光篇】|大奧 ～誕生【有功·家光篇】]]》和电影《[[大奧～永遠～【右衛門·綱吉篇】|大奧～永遠～【右衛門·綱吉篇】]]》。<ref name="asahi">{{Cite journal|author = 林真理子|authorlink = 林真理子|date = 2012-11-09|title = マリコのゲストセレクション 640 磯山晶|journal = 週刊朝日|pages = 112-116|publisher = 朝日新聞出版|accessdate = 2012-11-18}}</ref>翌年，主演电影《[[第七日的奇蹟|第七日的奇蹟]]》。之後，主演TBS電視台連續剧《[[半澤直樹|半澤直樹]]》獲得空前的成功，其收视率最高達到46.7%，成為日本近年來電視劇與堺雅人個人所主演的連續劇中收視最高的一部。<ref name="收視紀錄">{{cite web|url=http://mainichi.jp/mantan/news/20130716dyo00m200004000c.html |title=半沢直樹:第2話視聴率は21．8％　堺雅人主演作が第2話で20％超 |accessdate=2013-07-16 |work= |publisher=每日新聞 |date=2013-07-16 |archiveurl=https://archive.is/20130717063456/http://mainichi.jp/mantan/news/20130716dyo00m200004000c.html |archivedate=2013-07-17 |deadurl=yes }}</ref><ref name="收視紀錄1">{{cite web |url=http://www.sponichi.co.jp/entertainment/news/2013/07/16/kiji/K20130716006228740.html |title=大台突破！堺雅人主演「半沢直樹」第２話で２１・８％ |accessdate=2013-07-16|work= |publisher=[[體育日本|體育日本]] |date=2013-07-16 }}</ref>

2012年12月，堺雅人開始和在拍攝《大奧～永遠～【右衛門·綱吉篇】》時認識的[[菅野美穗|菅野美穗]]交往。<ref>{{Cite web |date=2013-03-22 |url=http://www.sanspo.com/geino/news/20130322/mar13032205070002-n1.html |title=堺雅人＆菅野美穂、超ビッグカップル来月結婚！ |work=sanspo.com（紙面から） |publisher=サンケイスポーツ |accessdate=2013-03-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20130323091223/http://www.sanspo.com/geino/news/20130322/mar13032205070002-n1.html |archivedate=2013-03-23 }}</ref><ref>{{Cite web |date=2013-03-22 |url=http://www.sponichi.co.jp/entertainment/news/2013/03/22/kiji/K20130322005450980.html |title=菅野美穂「結婚したいジタバタ子」だったが…堺雅人と結婚へ |work=スポニチアネックス |publisher=スポーツニッポン |accessdate=2013-03-22}}</ref>於2013年4月2日，二人結為夫妻<ref name="結婚">[http://www.tanabe-agency.co.jp/talent/sakai_masato/houkoku.html ファンのみなさまへ] {{webarchive|url=https://web.archive.org/web/20130927074317/http://www.tanabe-agency.co.jp/talent/sakai_masato/houkoku.html |date=2013-09-27 }}（2013年4月3日）、田辺エージェンシー、2013年4月3日閲覧</ref>。2015年8月，妻子菅野產下一子<ref>[http://www.sanspo.com/geino/news/20150814/mrg15081405010001-n1.html 菅野美穂、第１子の男児出産「家族が増えた喜びでいっぱい」]</ref>。

2018年12月3日，妻子菅野產下長女。<ref>[https://www.oricon.co.jp/news/2124685/full/?utm_source=Twitter&utm_medium=social&ref_cd=jstw003.html 菅野美穂が第2子女児出産「かけがえのない喜びだと感じています」【コメント全文】]</ref><ref>[https://www.sponichi.co.jp/entertainment/news/2018/12/04/kiji/20181204s00041000101000c.html 堺雅人　イクメン宣言！第２子誕生に「できるかぎりの時間と労力を育児に…」]</ref>。

== 人物 ==
{{Refimprovesect|time=2014-11-5}}
* 在學生時代經歷過獨立生活的艱辛，最喜愛吃[[海膽|海膽]]和鮭魚子壽司。曾經剪下[[壽司|壽司]]店傳單的圖標與菜單簡介貼在牆上。又標註：我一定要成為大人物！（喻出人頭地，改善生活）。在2012年被問到流行一時的“世界末日”問題時答道：要把家裡打掃的一塵不染之後吃著鮭魚子壽司和海膽壽司再迎接末日。
*在最貧窮的時候，曾經撿拾路邊蒲公英沾醋吃，被問是什麼味道，他說並沒有特別的味道，就是蔬菜而已。
* 少年時代的愛好是當雜貨店老闆和老書商。
* 由於在早稻田大學學習中國文學，多次在影視劇中背誦詩詞，令旁人驚詫。被大家評價性格儒雅，和善，人緣較好。
* 平日喜歡看書，最喜歡看的書是司馬遼太郎的《街道漫步》散文集和卡拉馬助夫兄弟們。
