'''莖'''是[[植物|植物]]的[[营养器官|营养器官]]之一。是大多数植物可见的主干。当然，例如[[仙人掌|仙人掌]]的[[变态茎|变态茎]]。茎下接根，通过[[木质部|木质部]]将[[根|根]]部吸收到的水分和[[礦物質|礦物質]]往上运输到各营养器官，通过[[韧皮部|韧皮部]]将光合作用的产物往下运输。茎来源于植物[[胚胎|胚胎]]的[[胚芽|胚芽]]。胚轴组成部分的茎，准确地说是[[子叶|子叶]]下的部分。

== 莖的演化 ==
最早擁有莖的植物為現已絕種的[[庫氏裸蕨|庫氏裸蕨]]，現存則是[[松葉蕨|松葉蕨]]，他們沒有真正的[[根|根]]、[[葉|葉]]。因此[[維管束植物|維管束植物]]（[[導管植物|導管植物]]）中，最早出現的器官是莖，[[根|根]][[葉|葉]]則是由莖演化而成。

== 茎的横切面 ==

由外往内可分为表皮（Epidermis），内面的一层皮质的结构因植物而异，可以是[[厚壁组织|厚壁组织]]，[[厚角组织|厚角组织]]和[[薄壁组织|薄壁组织]]。

[[雙子葉|雙子葉]][[木本|木本]]莖的剖析構造中，由外而內為[[周皮|周皮]]（[[木栓層|木栓層]]、[[木栓形成層|木栓形成層]]、[[綠皮層|綠皮層]]）、[[韌皮部|韌皮部]]、[[維管束形成層|維管束形成層]]、[[木質部|木質部]]、[[木髓|木髓]]。其中[[周皮|周皮]]、[[韌皮部|韌皮部]]合稱樹皮。

莖的橫切面也是[[雙子葉|雙子葉]]、[[單子葉|單子葉]]的分辨指標。[[雙子葉|雙子葉]]橫切面中維管束排列為環形；[[單子葉|單子葉]]則為散生，排列不一。

== 茎的變態（[[變態莖|變態莖]]） ==
有些植物的莖，其功用已經特化不只是支持和運輸的功能，其形態也不只是著生枝葉，稱為[[變態莖|變態莖]]。

常見的有[[馬鈴薯|馬鈴薯]]的[[塊莖|塊莖]]、[[仙人掌|仙人掌]]的肉質莖、[[洋蔥|洋蔥]]和百合的[[鱗莖|鱗莖]]、[[荸薺|荸薺]]和甘藍的[[球莖|球莖]]、[[薑|薑]]的[[根莖|根莖]]、[[草莓|草莓]]的[[走莖|走莖]]、[[葡萄|葡萄]]的[[卷鬚|卷鬚]]（莖卷鬚），還有莖（枝條）特化成葉狀的[[蘆筍|蘆筍]]等。
