'''羽庭'''{{BD|1985年|3月21日|||catIdx= L廖}}原名'''廖欣宁'''，出生于[[臺灣|臺灣]][[台北市|台北市]]，毕业于[[臺北市私立華岡藝術學校|华冈艺术学校]]，为[[台湾|台湾]]著名的女[[模特儿|模特儿]]、[[替身演员|替身演员]]和[[电视节目|电视节目]][[主持人|主持人]]，被誉为“替身女王”。<ref name="duck " /><ref>{{cite web |url=http://ent.qq.com/a/20120105/000032.htm |title=“替身女王”羽庭写真书销售破万 包场裸泳庆功 |publisher=[[腾讯网|腾讯网]]|date=2012-01-05 |work= |language =zh-cn}}</ref>

== 简介 ==
羽庭早年在华冈艺校毕业，21岁时因担任[[澳门|澳门]]演员[[梁洛施|梁洛施]]的替身演员[[爆紅|爆紅]]，成为替身演员和模特儿，同时也是电视节目主持人。<ref>{{cite web |url=http://www.hunantv.com/c/20110922/0941004903.html |title= 梁洛施替身大尺度真空诱惑 G奶妹羽庭裸体出镜|publisher=金鹰网 |date=2011-09-22 |accessdate=2011-09-22 |work= |language =zh-cn}}</ref>2011年，羽庭拍攝首本[[全裸|全裸]][[寫真集|寫真集]]《替身》，完全不清場，也未做任何保護措施；她說，[[脫星|脫星]]與[[寫真女星|寫真女星]]是兩回事，她不怕被定型，只希望趁年輕留下作品<ref>{{cite news|author=|work=蘋果即時|publisher=[[台灣蘋果日報|台灣蘋果日報]]|url=https://tw.appledaily.com/new/realtime/20110730/56212|title=不清場免保護 G奶羽庭拍寫真露3點|date=2011-07-30|accessdate=2019-02-28|language=zh-tw}}</ref>。
