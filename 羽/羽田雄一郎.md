'''羽田雄一郎'''（{{lang-ja|'''羽田雄一郎'''}}，{{bd|1967年|7月29日|||catIdx=Hata Yūichirō}}）出生于[[日本|日本]][[东京都|东京都]][[世田谷区|世田谷区]]，毕业于[[玉川大学|玉川大学]]文学部，[[日本民主党|日本民主党]]成员，为日本第十七届[[国土交通大臣|国土交通大臣]]。

== 生平 ==
1967年，羽田雄一郎出生在日本[[东京都|东京都]][[世田谷区|世田谷区]]，后来先后就读于[[成城学园中学校|成城学园中学校]]、[[敬和学园高等学校|敬和学园高等学校]]、[[玉川大学|玉川大学]]文学部，大学毕业后在伊藤忠纪念财团任职。<ref name="goaisatsu">「ご挨拶」『[http://www.y-hata.jp/profile/profile.html 羽田雄一郎（プロフィール）] {{webarchive|url=https://web.archive.org/web/20120903165249/http://www.y-hata.jp/profile/profile.html |date=2012-09-03 }}』羽田雄一郎事務所。</ref>

1999年10月，羽田开始踏入政治舞台，成为日本民主党党员。

2012年6月4日，日本首相[[野田佳彦|野田佳彦]]任命他为[[国土交通大臣|国土交通大臣]]。

== 政治思想 ==
=== 静冈空港 ===
2003年，羽田参与了反对国会议员“[[静冈空港|静冈空港]]”署名活动。<ref>{{Cite web |url=http://kuukouno.hp.infoseek.co.jp/kokaiginsyomei.htm |title=国会議員署名これまでと今後の展望 - 空港はいらない静岡県民の会 |access-date=2012-08-12 |archive-url=https://web.archive.org/web/20090307071308/http://kuukouno.hp.infoseek.co.jp/kokaiginsyomei.htm |archive-date=2009-03-07 |dead-url=yes }}</ref>

=== 靖国神社 ===
羽田一直以来都参拜[[靖国神社|靖国神社]]，他是日本前首相[[羽田孜|羽田孜]]的儿子，同父亲是日本“[[大家一起来参拜靖国神社议员联盟|大家一起来参拜靖国神社议员联盟]]”成员，每年8月15日都会去参拜靖国神社。<ref>{{cite web |url= http://news.qq.com/a/20120810/000767.htm|title=日本交通大臣羽田雄一郎将于15日参拜靖国神社 |accessdate=2012年8月10日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>
