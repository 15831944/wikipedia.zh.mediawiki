《'''羽'''》是日本摇滚组合[[B'z|B'z]]成员[[稻叶浩志|稻叶浩志]]第五张个人单曲，2016年1月13日由[[Being_(公司)|VERMILLION RECORDS]]于日本发行。

== 簡介 ==
除去配信限定發售的單曲外，是距離上一首單曲《[[Okay_(稻葉浩志單曲)|Okay]]》暌違約5年半後發售的單曲。

本單曲除了發售只包含[[CD|CD]]的通常盤以外，還發售了收錄最新LIVE影像的[[DVD|DVD]]/[[Blu-ray|Blu-ray]]的初回限定盤。因其中一首收錄曲「BLEED」被選為[[PlayStation_4|PlayStation 4]]／[[PlayStation_3|PlayStation 3]]遊戲《[[人中之龍_(遊戲)#人中之龍_極|人中之龍 極]]》（[[世嘉|SEGA GAMES]]）的主題歌，所以也發售了附帶了相關影像的「人中之龍盤」特別版本。

稻葉在製作歌曲一開始的時候曾打算做一張專輯。但考慮到去年（2014年）也做了一張專輯，又因為許久沒製作單曲了，於是便收錄了一般單曲碟的容量的作品，以現在的形式發售了。

在2016年1月25日的[[Oricon|Oricon]]週榜以首批發售8.9萬張位列第一，最终累計發售12.3萬張。虽不及前作《Okay》累计13.4万张的成绩，但卻上榜18週，比前作的11週多了7週，也是標題曲《羽》的捆綁動畫《[[名偵探柯南|名偵探柯南]]》的歷代單曲作品中，繼《[[謎_(單曲)|謎]]》的32週、《[[衝動_(B'z單曲)|衝動]]》的19週後，第三首大熱持久作。

== 版本 ==
; 通常盤
: 只包含CD的版本。
; 初回限定盤(CD＋DVD・CD＋Blu-ray)
: 分「CD＋DVD」以及「CD＋Blu-ray」两种版本。附帶影片是在收錄了2014年6月於[[品川|品川]]Stellar Ball進行的10場公演Plemium Live的《[[Koshi_Inaba_LIVE_2014_〜en-ball〜|Koshi Inaba LIVE 2014 〜en-ball〜]]》里，未曾收录的6首现场献唱作品。
; 人中之龍盤(CD+DVD+Blu-ray)
: 「CD+DVD+Blu-ray」版本。使用了Digipak包装。收錄了《人中之龍 極》的片尾曲、未收錄於CD中的《Receive You [Reborn]》的MV，以及《[[人中之龍|人中之龍]]》系列的總監督[[名越稔洋|名越稔洋]]與稻葉浩志的對談影片。此外還附送《人中之龍 極》的專用Product Code，可用於試聽只限在遊戲中使用的限定版本原創片頭影像。
