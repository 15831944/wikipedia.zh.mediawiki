'''羽咲美晴'''（{{lang-ja|羽咲みはる}}，1992年3月3日 - ），[[日本|日本]]的[[AV女優|AV女優]]、[[偶像_(日本)|偶像]]<ref name="T-Powers"/>，[[廣島縣|廣島縣]]出身<ref name="T-Powers"/>。目前為「[[S1_(AV片商)|S1]]」專屬女優<ref name="エスワン">{{cite web|url=https://www.s1s1s1.com/actress/detail/1033976/ |title=公式サイト |publisher=[[S1_(AV片商)|エスワン]]| |accessdate=2016-05-09}}</ref>、三代「[[惠比壽麝香葡萄|惠比壽★麝香葡萄]]」成員<ref>{{cite news|url=https://www.ettoday.net/news/20170215/866927.htm |title=全身性感帶號稱「人間水龍頭」　羽咲美晴一摳就噴 |newspaper=[[ETtoday新聞雲|ETtoday新聞雲]] |date=2017-02-15}}</ref>。所屬於「[[T-POWERS|T-POWERS]]」事務所<ref name="T-Powers"/>。

== 人物 ==
過去曾以「'''西永嶺花'''」的名義在日本[[演藝界|演藝圈]]活動<ref>{{cite web|url=http://www.bs-j.co.jp/gilgamesh/nishinaga.html |title=西永嶺花 - BSジャパン |publisher=[[BS_JAPAN|BSジャパン]] |accessdate=2018-05-19 |language=ja}}</ref>，在當時為「Nicky」雜誌的專屬模特兒，也是[[澀谷|澀谷]]的[[辣妹|辣妹]]偶像團體「GALDOLL」的活躍成員<ref>{{cite web |url=https://www.barks.jp/news/?id=1000068068 |title=渋谷GAL’sからアイドル・ユニット誕生、GAL♥DOLLデビュー |publisher={{link-ja|バークス|BARKS}} |accessdate=2018-05-19 |language=ja}}</ref>，所屬於「V.I.P MODEL AGENCY」（デルタパートナーズ）事務所<ref>{{cite web |url=http://www.vip-model.com/models/ryoka.html |title=たすたすりょおたす～♪西永嶺花 |publisher=デルタパートナーズ |accessdate=2018-05-19}}</ref>。她說雖然自己是前偶像的身分，但是當時的知名度並不高，反而是成為AV女優之後比以前更受歡迎了<ref>{{cite news |url=http://news.dmm.co.jp/article/994102/?page=2 |title=DMMアダルトアワードにノミネートされたMUTEKIデビューの直球アイドルは真面目で研究熱心な頭脳派！今回東風がそんな彼女を泣かせてしまう！？【羽咲みはるインタビュー前編】 (2/3ページ) |publisher=[[DMM.com|DMMニュースR18]] |date=2017-03-20|accessdate=2017-11-06 |language=ja}}</ref>。並在後來也加入AV女優團體「[[惠比壽麝香葡萄|惠比壽★麝香葡萄]]」重新開始了偶像活動。

第一次看[[AV|AV]]是在[[國小|國小]]低年級的時候，從[[中學|中學]]時期慢慢的開始對AV作品感興趣。最終她靠著自己發掘了[[日本色情業|色情行業]]並申請加入了現在的公司<ref name="東スポ">{{cite news |url=https://www.tokyo-sports.co.jp/entame/entertainment/528982/ |title=元渋谷系ユニットのリーダー・羽咲みはるが語ったアイドル時代のＨ事情 |newspaper=[[東京體育|東スポWebトップ]] |date=2016-04-12 |language=ja}}</ref>。

因為胃下垂的關係，在拍攝期間喝水的話會產生體型變化的特徵<ref name="東スポ"/>。

她與同公司的AV女優[[園田美櫻|園田美櫻]]是好朋友，自己在個人[[推特|推特]]中提到兩人在中學二年級的時候就認識，有著至少10年的交情<ref>{{cite web|url=http://www.playno1.com/article-28141-1.html#.W-NMcJMzaUk |title=羽咲みはる不慎說漏嘴！洩露園田みおん秘密？ |publisher=PLAYNO.1玩樂達人 |date=2018-10-06}}</ref>。
