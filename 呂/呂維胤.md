'''呂維胤'''（{{bd|1981年|6月12日|||catIdx=呂}}），出生於[[臺南市|臺南市]]，[[民主進步黨|民主進步黨]][[新潮流系|新潮流系]]成員，現任臺南市南區[[臺南市議員|市議員]]。父親為民進黨台南市黨部創黨黨員<ref name="note1"/>，年幼時雙親在市場販售水果為業，自[[臺南市東區勝利國民小學|臺南市立勝利國小]]、[[長榮中學|長榮中學]]畢業，曾就讀[[國立中正大學|國立中正大學]]政治學系、政治學研究所碩士班。就讀大學時期即至時任立法委員的賴清德的服務處當志工，就讀研究所時成為賴清德的特助，當時父親罹患疾病，一度無法繼續學業，賴清德得知後即額外提供生活費協助才得以完成研究所學業<ref name="note1">{{cite web |url=http://news.e2.com.tw/utf-8/2013-12/5551310.htm |title=栽培新世代、英雄出少年呂維胤辭官參選|date=2013-12-10 |publisher=壹凸新聞 |language=zh-tw}}</ref>。2014年首度參加[[2014年中華民國地方公職人員選舉|市議員選舉]]即以該選區第1高票（得票數：15,588，得票率：23.43%）當選<ref>{{cite web |url=http://election.ltn.com.tw/2014/candidate.php?rtype=2&no=736 |title=臺南市市議員開票結果 |date= |publisher=自由電子報 |language=zh-tw}}</ref>。2018年參加[[2018年中華民國地方公職人員選舉|市議員選舉]]連任成功。

2015年10月19日與洪瑩珍小姐參加臺南市政府集團結婚<ref>{{cite web |url=http://cdns.com.tw/news.php?n_id=1&nc_id=32309 |title=市府集團結婚一二○對已額滿 |date=2015-06-21 |publisher=中華日報 |language=zh-tw}}</ref>。
