'''长葛市实验中学'''是[[长葛市|长葛市]]的一所民办中学，设初中部与高中部，位于长葛市科技工业园区颍川大道北侧。<ref name= "简介"/>

== 简介 ==
学校建于2001年，是一所现代化寄宿制民办公助[[完全中学|完全中学]]，为[[许昌市|许昌市]]示范性高中。投资6000多万元，占地162亩，由河南省教育建筑研究所规划设计，总建筑面积37600平米。<ref name= "简介"/>

建有2座教学大楼、1座实验楼、1座综合办公大楼、1座科技楼、2座学生公寓（装有空调和暖气）、1座教师公寓、1座餐厅（可容纳近千人）和1座礼堂（可容纳近千人）。<ref name="简介">{{cite web |url=http://school.cnycedu.com/xyfm/html/?168.html |language= |title=河南省长葛市民办实验中学简介 |publisher=中国远程教育网 |date= |accessdate=2012年11月 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

现有50多个教学班，在校生3000人；现有教师100多人，其中特级教师5人，高级教师24人，一级教师36人。现任校长是何伟祥，董事长是赵卿茳。<ref>{{cite web |url=http://cgsyzx.30edu.com/news/bb5b647c-03d0-4176-b623-3890c719dc2f/7646f910-6bdc-4558-b8fa-8d98ba80d695.htm |language= |title=长葛市实验中学十年庆典公告 |publisher=长葛市实验中学 |date=2011年9月16日 |accessdate= }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2011年4月，学校与[[河南省实验中学|河南省实验中学]]开始合作办学。<ref>{{cite web |url=http://cgsyzx.30edu.com/news/bb5b647c-03d0-4176-b623-3890c719dc2f/00204c0f-e594-427a-bc50-17b864ef2a3d.htm |language= |title=河南省实验中学与长葛市实验中学合作办学（组图） |publisher=长葛市实验中学 |date=2011年4月26日 |accessdate= }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>目前学校还设有小学部。<ref>{{cite web |url=http://cgsyzx.30edu.com/news/bb5b647c-03d0-4176-b623-3890c719dc2f/efad46bf-cf2e-43e6-b24e-f183744906c6.htm |language= |title=长葛实验中学高初中暨附属小学2011年招生简章 |publisher=长葛实验中学 |date=2011年6月15日 |accessdate= }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
