'''长春轻轨QKZ3型/2000系电动车组'''是[[长春轨道交通|长春轨道交通]]的电车车款之一，在[[长春轨道交通3号线|长春轨道交通3号线]]和[[长春轨道交通4号线|长春轨道交通4号线]]运行。

== 简介 ==
QKZ3型/2000系电动车组由[[长春轨道客车|长春轨道客车]]于2007年生产，3节编组，共20列。车门为[[内崁式门|内崁式门]]，为70%低地板车辆。目前在[[长春轨道交通3号线|长春轨道交通3号线]]和[[长春轨道交通4号线|长春轨道交通4号线]]运行，是长春轻轨的第四款电动车组。其中，编号为201、206-211、218-220的列车外涂装为红色，编号为203-205的列车外涂装为银色，编号为202、212-217的列车外涂装为蓝色。车门由北京博得负责制造，空调为广州中车DLD30型<ref>{{Cite web |url=http://www.zrjc.com/CN/Products/detail.aspx?id=55 |title=存档副本 |access-date=2014-02-14 |archive-url=https://web.archive.org/web/20140222212314/http://www.zrjc.com/CN/Products/detail.aspx?id=55 |archive-date=2014-02-22 |dead-url=yes }}</ref>，(201-219)的[[VVVF|VVVF]]由瑞士[[ABB|ABB]]负责制造，(220)的[[VVVF|VVVF]]由[[中国北车股份有限公司大连电力牵引研发中心|大连电牵]]负责制造<ref>http://www.cnrerd.chinacnr.com/p404.aspx?ProductID=9&CategoryID=119{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，驾驶室内操纵设备为手柄式，为7级牵引、10级制动，网络控制系统和牵引系统为[[中国北车股份有限公司大连电力牵引研发中心|大连电牵]]负责制造<ref>{{cite web |url=http://www.cnrerd.chinacnr.com/p404.aspx?ProductID=138&CategoryID=19 |title=存档副本 |accessdate=2014-02-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20140223131915/http://www.cnrerd.chinacnr.com/p404.aspx?ProductID=138&CategoryID=19 |archivedate=2014-02-23 }}</ref><ref>http://www.cnrerd.chinacnr.com/p404.aspx?ProductID=35&CategoryID=110{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，制动设备为[[克诺尔集团|克诺尔]]制造的液压制动装置<ref>http://www.docin.com/p-70233089.html</ref>。

== 事故 ==
*2011年7月15日11时左右，一列 2000 系列车在[[卫光街站|卫光街]]附近发生与汽车相撞事故<ref>[http://cswbszb.chinajilin.com.cn/html/2011-07/16/content_745803.htm  黑色吉普涉嫌闯红灯 交会路口与轻轨相撞]</ref>，导致停运一小时。事故导致汽车内有一名乘客受伤。
*2012年7月25日17时左右，一列 2000 系列车（编号 201）在[[临河街站|临河街]]站前，主电机及其牵引电机烧毁，事故车辆在八月份送返[[长春轨道客车|长春轨道客车]]大修。目前已于2013年1月18日重新上线。
