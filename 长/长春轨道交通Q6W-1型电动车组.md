'''长春轨道交通Q6W-1型电动车组'''是[[长春轨道交通|长春轨道交通]]的电动车组车款之一。

== 简介 ==
Q6W-1型由[[湘潭电机厂|湘潭电机厂]]在2001年生产，共14列，为70%低地板列车，3节编组，车门为[[内崁式门|内崁式门]]。自2002年开始在[[长春轨道交通3号线|长春轨道交通3号线]]运营，是长春轻轨的第一款电动车组。在2010年，Q6W-1型列车入库封存，存放于停车场内，部分已经改造成固定重联，目前有部分列车开始在高峰期投入服务。

== 列车配置 ==
Q6W-1型分为4种外涂装，分别是红色（01-02、07-08、13-14号）、蓝色（03-04号）、白色（05-06号）和绿色（09-12号）。其中01-04号、07-12号为手动档操作，[[VVVF|VVVF]]由芬兰生产；05-06、13-14号为自动档操作，空调系统为广州中车KLD35B，[[VVVF|VVVF]]由日本{{link-ja|东洋电机|東洋電機製造}}生产。列车空调为广州中车KLD35B型，车门由北京博得负责生产，驾驶室内操纵设备为手柄式，为4级牵引、8级制动，后换成[[中国北车股份有限公司大连电力牵引研发中心|大连电牵]]制造的[[IGBT|IGBT]]-[[VVVF|VVVF]]和网络牵引系统。在车辆大修后车辆涂装编号都有所改变，这14列车将会改造为7列重联车，编号为CCQG1001--CCQG1007。

== 事故 ==
2005年6月24日9时14分左右，一列Q6W-1型列车在[[长春站_(轻轨)|长春站]]与[[辽宁路站|辽宁路]]之间发生脱轨事故<ref>{{Cite web |url=http://www.hangzhou.com.cn/20050101/ca789339.htm# |title=长春轻轨列车发生脱轨事故 |access-date=2012-06-23 |archive-url=https://web.archive.org/web/20160304192428/http://www.hangzhou.com.cn/20050101/ca789339.htm# |archive-date=2016-03-04 |dead-url=yes }}</ref>。不过，并无任何人员发生伤亡。
== 未来计划 ==
由于[[长春轻轨3000系电车|新型列车]]尚未全部交付，长春轻轨正在将此型车翻新改造。目前已经改造出不少于四列的固定重联型，部分列车已经开始载客运行。
