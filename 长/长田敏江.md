'''长田敏江'''（{{lang-ja|長田 敏江}}，{{bd|？||1939年|8月3日}}），日本[[山口县|山口县]][[厚狭郡|厚狭郡]]{{Link-ja|万仓村|万倉村}}人{{r|东京日日新闻1940}}，[[大日本帝國陸軍|大日本帝國陸軍]][[第32师团_(日本陆军)|第32师团]]长田大队的大队长，少佐[[军衔|军衔]]，1939年率军扫荡时在[[山东|山东]]梁山附近遭遇中国[[八路军第115师|八路军第115师]]伏击，战败自杀<ref name=tongji/>。

== 生平 ==
随着[[中国抗日战争|中国抗日战争]]向长期化，持久化发展，[[侵华日军|侵华日军]]为维持占领地的治安和警备为目的，新组建了第32师团等中国治安师团。第32师团于1939年2月7日在东京编成，长田敏江任该师团下辖一个野战步兵大队的大队长。1939年5月，第32师团在青岛登陆，长田大队驻扎在山东省汶上县县城。<ref name=qlwb>{{cite web |author1=邓撰相 |title=今野博,为抗日献身的日本八路 |url=http://epaper.qlwb.com.cn/qlwb/content/20150416/ArticelA21002FM.htm |website=齐鲁晚报 |access-date=2018-08-04 |archive-url=https://web.archive.org/web/20180804153321/http://epaper.qlwb.com.cn/qlwb/content/20150416/ArticelA21002FM.htm |archive-date=2018-08-04 |dead-url=no }}</ref><ref name=tongji/>

1939年初，[[八路军第115师|八路军第115师]]根据中共中央指示转战至鲁西平原。1939年5月，115师代师长[[陈光_(军事家)|陈光]]、政委[[罗荣桓|罗荣桓]]率队东进以威胁[[津浦铁路|津浦铁路]]中段的日军。1939年6月，日军第32师团决定对八路军展开[[扫荡|扫荡]]行动。7月底，第32师团派遣长田大队担任扫荡任务，并加强了部分骑兵和炮兵，组成了一个的步、炮、骑混成大队，由长田敏江指挥，欲将115师部队驱逐出山东。<ref name=tongji>{{cite web |title=1939年陈光将军率部痛歼日本皇族少佐 |url=http://www.chinanews.com/mil/2012/08-17/4115234.shtml |website=中国新闻网-人民网 |access-date=2018-08-05 |archive-url=https://web.archive.org/web/20120821112203/http://www.chinanews.com/mil/2012/08-17/4115234.shtml |archive-date=2012-08-21 |dead-url=no }}</ref>

1939年8月1日上午，陈光和罗荣桓正在[[梁山|梁山]]脚下的独山村（今[[梁山县|梁山县]][[水泊街道|水泊街道]]独山村）东面的孟家林召开庆祝[[八一建军节|八一]]军民联欢会，115师师部及其直属部队均在场，骑兵侦察员送达紧急情报，获知长田大队和部分日军已由汶上县城出发向梁山根据地进犯。115师决定利用高过人顶的庄稼作掩蔽，以少数部队袭扰疲惫敌人，其他部队在梁山南面的前集村和独山村埋伏，对日军进行包围歼灭。8月2日，经过几次袭扰，长田大队在当天夜里被顺利带到八路军预定的围歼地域，8月3日早上战斗结束，歼灭日军300余人{{r|洛}}。战后八路军战士在旧石灰窑外的一处洼地里发现长田的尸体，并通过旁边军装胸前的官徽确认其身份。据中方参战士兵回忆，长田敏江肥头大脑，留有胡子，为负伤后自杀，死时穿着衬衣，一把大洋刀从前胸穿过。<ref>{{cite web |title=抗战老兵口述历史 揭秘八路军对日本战俘改造政策 |url=http://www.takefoto.cn/viewnews-528374.html |website=北晚新视觉 |access-date=2018-08-05 |archive-url=https://web.archive.org/web/20180805132308/http://www.takefoto.cn/viewnews-528374.html |archive-date=2018-08-05 |dead-url=no }}</ref><ref>{{cite web |title=九旬亲兄弟同忆梁山战斗：一一五师歼灭日本“皇族”少佐【2】 |url=http://ah.people.com.cn/n/2014/1214/c358327-23215173-2.html |website=人民网-大众日报 |access-date=2018-08-05 |archive-url=https://web.archive.org/web/20180805132309/http://ah.people.com.cn/n/2014/1214/c358327-23215173-2.html |archive-date=2018-08-05 |dead-url=no }}</ref><ref name=tongji/><ref>{{cite web |title=梁山歼灭战，八路军同等装备下全歼日军一个大队 |url=http://www.haijiangzx.com/2016/1014/1405449.shtml |website=海疆红色网 |access-date=2018-08-05 |archive-url=https://web.archive.org/web/20180606134249/http://www.haijiangzx.com/2016/1014/1405449.shtml |archive-date=2018-06-06 |dead-url=no }}</ref>
