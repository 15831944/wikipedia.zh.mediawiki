'''长白山保护开发区管理委员会'''简称'''长白山管委会'''，是[[吉林省人民政府|吉林省人民政府]]的派出机构，管理[[长白山保护开发区|长白山保护开发区]]。

此机构成立于2005年6月，是[[中共吉林省委|中共吉林省委]]、吉林省人民政府为实现对[[长白山|长白山]]“统一规划、统一保护、统一开发、统一管理”而设立的机构。2006年1月，正式挂牌，为[[正厅级|正厅级]]建制。2006年7月，吉政发[2006]30号文件定义：长白山管委会，具有相当于市（州）政府的行政管理职权<ref>{{Cite web |url=http://changbaishan.gov.cn/zwgk/cbsgwh/201612/t20161228_20541.html |title=长白山管委会简介 |accessdate=2018-06-10 |author= |date=2016-12-28 |work= |publisher=长白山保护开发区管理委员会网站|language=Zh-hans }}</ref>，享有省政府授权、委托的部分经济社会和行政事务管理职能和权限<ref name="a1">{{Cite web |url=http://changbaishan.gov.cn/gk/gltz/ |title=管理体制 |accessdate=2018-06-10 |author= |date= |work= |publisher=长白山保护开发区管理委员会网站|language=Zh-hans }}</ref>。省里将长白山管委会按市（州）对待和管理。

设立的愿景是，“打造东方的[[阿尔卑斯|阿尔卑斯]]”。2012年7月8日，长白山管委会投入1500万元，与某影视集团合拍的电视剧版《[[非诚勿扰|非诚勿扰]]》。

== 相关机构 ==

根据吉林省相关文件，下设三个[[经济管理区|经济管理区]]、17个机关部门、8个中央及省政府直属单位、16个[[事业单位|事业单位]]、2个企业单位<ref name="a2">{{Cite web |url=http://changbaishan.gov.cn/gk/nsjg/ |title=内设机构 |accessdate=2018-06-10 |author= |date= |work= |publisher=长白山保护开发区管理委员会网站|language=Zh-hans }}</ref>。下辖三个经济管理区（[[池北区|池北区]]、[[池西区|池西区]]、[[池南区|池南区]]），由管委会（[[正处级|正处级]]）管理，具有相当于[[县级政府|县级政府]]的行政管理职能和权限<ref name="a1"/>。

=== 机关部门 ===
*党群工作部、办公室、纪工委（监察局）、政法委、机关党委、经济发展局、环境与资源保护局、住房和城乡建设局、交通运输局、旅游管理局、社会管理办公室、财政局、人力资源和社会保障局、国土资源局、招商局、教育局、审计局<ref name="a2"/>。

=== 中央、省政府直属单位 ===

*长白山公安局、长白山工商行政管理局、长白山地方税务局、长白山国家税务局、长白山食品药品监督管理局、长白山质量技术监督局、社会保险事业管理局、长白山司法局<ref name="a2"/>。

*长白山自然保护管理中心、行政服务中心、综合执法支队、机关事务管理中心、长白山科学研究院、文化传播中心、北京办事处、长春办事处、公路管理处、运输管理处、长白山生态环境监测站、医疗保险经办中心、森林防火指挥部办公室、彩票管理中心、接待办公室、信息中心<ref name="a2"/>。

=== 国有企业 ===

*长白山开发建设（集团）有限责任公司、长白山旅游股份有限公司<ref name="a2"/>。
