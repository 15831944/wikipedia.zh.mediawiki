'''长崎县'''（{{jpn|j=長崎県|hg=ながさきけん|rm=Nagasaki ken|f=y}}）是[[日本|日本]][[九州地方|九州地方]]西北部的县，縣廳所在地是[[長崎市|長崎市]]。長崎縣的管轄範圍包括九州本島的部分和九州西北部海域的[[对马岛|对马岛]]、[[壹岐岛|壹岐岛]]和[[五島列島|五島列島]]等離島。「長崎」這一地名來自於「長[[海岬|岬]]」之意（「崎」和「岬」在日語中發音相同）和古代在這裡居住的{{link-ja|長崎氏|長崎氏}}<ref name="dictionary">{{cite book |author={{lang|ja|竹内理三}}|script-title=ja:《角川日本地名大辞典 42 長崎県》 |year=1987 |publisher={{lang|ja|角川書店}} |location=東京都 |isbn=4-04-001420-0|language=ja}}</ref>{{rp|689}}。長崎縣唯一的陸上鄰縣是[[佐賀縣|佐賀縣]]，在[[令制國|令制國]]時代兩縣都屬於[[肥前國|肥前國]]<ref>{{cite web |url=http://www.pref.nagasaki.jp/bunkadb/intro1-1.html |script-title=ja:ひぜん【肥前（国）】 |accessdate=2017-05-21|publisher={{lang|ja|学研}}}}{{ja}}</ref>。長崎縣的海岸線地形極為複雜。雖然長崎縣的面積只排名日本第37位，但是海岸線長度卻達4,195公里，佔日本海岸線總長度的12%，高居日本所有一級行政區首位{{efn|[[北海道|北海道]]的海岸線總長度有3,050公里，但若算入未實際控制的[[南千島群島|南千島群島]]的1,348公里的海岸線的話，其總長度有4,398公里，超過長崎縣<ref name="coastline">{{cite web |url=http://www.pref.nagasaki.jp/bunkadb/intro1-1.html |script-title=ja:世界の中の長崎　～長崎県の海外交流史と文化財～ |accessdate=2017-05-21|publisher={{lang|ja|長崎県}}}}</ref>。}}。在[[鎖國_(日本)|日本鎖國]]時期，長崎是日本唯一獲得幕府承認的與西洋進行貿易的窗口，因此長崎縣的[[天主教|天主教]]信徒比例較高，天主教會在長崎單獨設有[[天主教長崎總教區|天主教長崎總教區]]<ref>{{cite web |url=http://www.city.nagasaki.lg.jp/nagazine/smart/t201204.html　 |script-title=ja:「長崎から世界の宝へ」～長崎の教会群とキリスト教関連遺産の世界遺産登録を目指して～ |accessdate=2017-05-21 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。靠近[[朝鮮半島|朝鮮半島]]和[[中國大陸|中國大陸]]的地理位置使得長崎也受到東亞大陸文化的影響較強。[[長崎新地中華街|長崎新地中華街]]是日本三大中華街之一。對馬則是[[江戶時代|江戶時代]]時獲得幕府承認的和[[李氏朝鮮|李氏朝鮮]]進行貿易的據點。長崎縣地形多山，人口多集中在沿海的狹小平地，加上多離島這一特點，使得海運和空運在長崎縣交通中有重要地位。長崎縣的佐世保市在二戰之前是日本海軍四大[[鎮守府|鎮守府]]之一的[[佐世保鎮守府|佐世保鎮守府]]所在地，現在也是重要的軍港。

== 历史 ==
=== 舊石器時代 ===
[[File:Iki_Harunotsuji_Ruinsi_2009.jpg|thumb]]
長崎縣的[[舊石器時代|舊石器時代]]遺跡有佐世保市的福井洞窟、泉福寺洞窟<ref>{{cite web |url=http://www.pref.nagasaki.jp/jiten/index.php/view/41 |script-title=ja:福井洞窟 |accessdate=2017-05-21 |publisher={{lang|ja|長崎県の遺跡大辞典}} |language=ja}}</ref>；雲仙市的百花台遺跡群<ref>{{cite web |url=http://www.pref.nagasaki.jp/jiten/index.php/view/279 |script-title=ja:百花台遺跡 |accessdate=2017-05-21 |publisher={{lang|ja|長崎県の遺跡大辞典}} |language=ja}}</ref>；平戶市的日之岳遺跡<ref>{{cite web |url=http://www.pref.nagasaki.jp/jiten/index.php/view/110 |script-title=ja:日ノ岳遺跡 |accessdate=2017-05-21 |publisher={{lang|ja|長崎県の遺跡大辞典}} |language=ja}}</ref>等地。長崎縣內的[[繩文時代|繩文時代]]代表性遺跡則有南島原市的原山支石墓群<ref>{{cite web |url=http://www.pref.nagasaki.jp/jiten/index.php/view/298 |script-title=ja:原山支石墓群 |accessdate=2017-05-21 |publisher={{lang|ja|長崎県の遺跡大辞典}} |language=ja}}</ref>、諫早市的有喜貝塚<ref>{{cite web |url=http://www.pref.nagasaki.jp/jiten/index.php/view/75 |script-title=ja:有喜貝塚 |accessdate=2017-05-21 |publisher={{lang|ja|長崎県の遺跡大辞典}} |language=ja}}</ref>、五島列島的白濱貝塚等地<ref>{{cite web |url=http://www.pref.nagasaki.jp/jiten/index.php/view/255 |script-title=ja:白浜貝塚 |accessdate=2017-05-21 |publisher={{lang|ja|長崎県の遺跡大辞典}} |language=ja}}</ref>。
===彌生時代===
[[彌生時代|彌生時代]]之後，這裡開始開始出現稻作農耕文明。長崎縣內的[[壹岐|壹岐]]、[[對馬|對馬]]等地因為位於亞洲大陸文化傳入日本的路徑，而發現了眾多金屬器文物<ref name="dictionary"></ref>{{rp|26}}。甚至在史籍《[[魏志倭人傳|魏志倭人傳]]》中，已經出現「對馬國」、「一支國」的地名，這也是最早的有關長崎縣的文獻記錄<ref name="dictionary"></ref>{{rp|28}}。長崎縣本土的主要彌生時代遺跡有平戶市的里田原遺跡<ref>{{cite web |url=http://www.city.hirado.nagasaki.jp/kurashi/culture/bunka/hiradoisan/hi11.html |script-title=ja:里田原遺跡1 |accessdate=2017-05-21 |publisher={{lang|ja|平戸市}} |language=ja}}</ref>。壹岐島的原之辻遺跡被考古學界認為就是魏志倭人傳中的一支國<ref>{{cite web |url=http://www.iki-haku.jp/harunotsuji/harunotsuji-1.html |script-title=ja:原の辻遺跡概要 |accessdate=2017-05-21 |publisher={{lang|ja|壱岐市立一支国博物館}} |language=ja}}</ref>。

公元4世紀以後，九州島北部開始進入[[大和王權|大和王權]]的勢力範圍之內。今天的長崎縣內最早的{{link-ja|前方後圓墳|前方後円墳}}遺跡出現在5世紀前期<ref name="dictionary"></ref>{{rp|27}}。壹岐島是長崎縣內古墳最多的地區，其中雙六古墳是長崎縣最大的前方後圓墳<ref>{{cite web |url=http://www.ikikankou.com/wp/?p=3811 |script-title=ja:双六古墳 |accessdate=2017-05-22 |publisher={{lang|ja|壱岐市観光連盟}} |language=ja}}</ref>。

===飛鳥時代與大化革新===
進入[[飛鳥時代|飛鳥時代]]後，日本在[[白村江之戰|白村江之戰]]中不敵[[唐朝|唐朝]][[新羅|新羅]]聯軍，為防備來自朝鮮的進攻而在對馬、壹岐修築防禦工事<ref name="dictionary"></ref>{{rp|28}}。[[大化改新|大化改新]]之後，長崎縣本土部分和五島列島是[[肥前國|肥前國]]的一部分、對馬島屬於[[對馬國|對馬國]]、壹岐島屬於[[壹岐國|壹岐國]]。《[[肥前國風土記|肥前國風土記]]》是各令制國風土記中保存狀況較好的五部風土記之一，是貴重的歷史資料<ref name="dictionary"></ref>{{rp|29}}。
===奈良時代與平安時代===
在[[奈良時代|奈良時代]]和[[平安時代|平安時代]]，由於靠近大陸的地理位置，對馬、壹岐多次遭到外敵入侵，其中規模最大的是11世紀初期的[[刀伊入寇|刀伊入寇]]<ref name="dictionary"></ref>{{rp|29}}。但另一方面，長崎縣在當時也是前往東亞大陸的交通要道。[[遣隋使|遣隋使]]和早期的[[遣唐使|遣唐使]]均曾通過對馬、壹岐後登陸朝鮮半島這一路線東渡唐朝。日本和新羅關係惡化之後，遣唐使改為經由平戶等九州西海岸港口渡海前往唐朝、後更從五島列島出發直接渡海。平戶也是[[宋日貿易|宋日貿易]]的重要據點之一<ref name="dictionary"></ref>{{rp|30}}。

平安時代末期，武士階級開始成為各地新的統治者。長崎地區實力最強的武士團{{link-ja|松浦党|松浦党}}在[[壇之浦之戰|壇之浦之戰]]是平家水軍的主力<ref name="dictionary"></ref>{{rp|31}}。[[鎌倉時代|鎌倉時代]]時，武藤氏出任肥前國[[守護|守護]]一職，但並未控制肥前國全境<ref name="dictionary"></ref>{{rp|31}}。1274年（[[文永|文永]]11年）和1281年（[[弘安|弘安]]4年），[[元朝|蒙古]]和[[高麗_(918年－1392年)|高麗]]軍隊曾襲擊對馬、壹岐、鷹島等島嶼<ref name="dictionary"></ref>{{rp|32}}。肥前國在鎌倉時代至[[室町時代|室町時代]]時長期處於群雄割據狀態，加上靠近朝鮮半島和中國的地理位置，因此成為[[倭寇|倭寇]]的溫床<ref name="dictionary"></ref>{{rp|33}}。1419年（[[應永|應永]]26年），李氏朝鮮曾派兵攻打對馬的倭寇根據地（[[應永外寇|應永外寇]]）<ref name="dictionary"></ref>{{rp|34}}。

===日本戰國時代===
[[File:The_Famous_Scenes_of_the_Sixty_States_69_Tsushima.jpg|thumb]]
日本[[戰國時代_(日本)|戰國時代]]時，長崎地區主要[[大名_(稱謂)|大名]]有[[島原半島|島原半島]]的{{link-ja|肥前有馬氏|肥前有馬氏}}、彼杵地方的{{link-ja|大村氏|大村氏}}、松浦地方的{{link-ja|松浦氏|松浦氏|平戶松浦氏}}、五島列島的{{link-ja|宇久氏|宇久氏}}、對馬的[[宗氏|宗氏]]<ref name="dictionary"></ref>{{rp|34}}。1550年（[[天文|天文]]19年），[[葡萄牙|葡萄牙]]船舶停靠平戶。天主教傳教士[[方濟·沙勿略|方濟·沙勿略]]也曾在這一年到訪平戶，開啟了長崎地區[[南蠻貿易|南蠻貿易]]和天主教傳播的歷史。當時進行南蠻貿易的交換條件是允許天主教傳教。有馬氏、大村氏、松浦氏均允許天主教在自己的領地內進行傳教，使得西洋文明開始在長崎地區各地普及，天主教信徒也大幅增加<ref name="dictionary"></ref>{{rp|35}}。[[大村純忠|大村純忠]]和[[有馬晴信|有馬晴信]]更接受[[洗禮|洗禮]]，成為[[吉利支丹大名|吉利支丹大名]]<ref name="dictionary"></ref>{{rp|35}}。

1570年，（[[元龜|元龜]]元年），大村純忠將長崎浦開放給葡萄牙人作為貿易港。開港之後長崎迅速發展，成為長崎地區的最大城市之一<ref name="City">{{cite web |url=http://www.city.nagasaki.lg.jp/syokai/750000/757000/p007003.html　 |script-title=ja:歴史年表　室町時代・安土桃山時代・江戸時代 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1587年（[[天正_(日本)|天正]]15年），[[豐臣秀吉|豐臣秀吉]]平定九州全境，並將長崎劃為直轄地<ref name="dictionary"></ref>{{rp|35}}。豐臣秀吉在平定九州後雖然發布[[伴天連追放令|伴天連追放令]]，但最初對天主教管制並不嚴苛，直到1597年（[[慶長|慶長]]元年）當局下令處死長崎的26名天主教信徒（日本二十六聖人）後才對天主教開始強力鎮壓<ref>{{cite web |url=http://www.catholic-takatsuki.jp/26seijin/　 |script-title=ja:日本二十六聖人 |accessdate=2017-05-08 |publisher={{lang|ja|カトリック高槻教会}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

=== 江戶時代 ===
[[江戶幕府|江戶幕府]]成立之後，長崎地區仍然維持了[[平戶藩|平戶藩]]、平戶新田藩、[[島原藩|島原藩]]、[[對馬藩|對馬藩]]、[[福江藩|福江藩]]、[[大村藩|大村藩]]等諸小藩並存的局面。[[江戶時代|江戶時代]]的長崎縣內分為1幕府領、6藩、5領，局勢極為複雜<ref name="dictionary"></ref>{{rp|39}}。1612年（[[慶長|慶長]]17年）{{link-ja|岡本大八事件|岡本大八事件}}之後，[[德川家康|德川家康]]更發布禁教令，對天主教嚴加鎮壓<ref name="Chishi">{{cite book |author={{lang|ja|野澤秀樹・堂前亮平・手塚章}}|script-title=ja:《日本の地誌10　九州・沖縄》 |year=2008 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-16770-2|language=ja}}</ref>{{rp|276}}。在江戶時代初期，平戶和長崎都是獲得幕府許可的貿易據點。1637年（[[寬永|寬永]]14年），島原藩爆發[[島原之亂|島原之亂]]。幕府將這一事件定性為天主教徒的[[一揆|一揆]]，對天主教的鎮壓更加嚴厲，並開始實施[[鎖國_(日本)|鎖國]]政策<ref name="dictionary"></ref>{{rp|38}}。1641年（[[寬永|寬永]]18年），幕府驅逐居住在長崎的葡萄牙人，並將平戶的荷蘭商館遷到長崎，獲得幕府許可與西洋進行貿易的地點集中到長崎一地<ref name="dictionary"></ref>{{rp|36}}。另一方面，長崎地區在江戶時代仍有眾多天主教信徒在私下信仰天主教<ref>{{cite web |url=https://www.pref.nagasaki.jp/s_isan/outline/02.html　 |script-title=ja:長崎のキリスト教の歴史 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。現在長崎縣的天主教信徒比例仍然大幅高於日本平均水準<ref>{{cite web |url=https://www.cbcj.catholic.jp/wp-content/uploads/2016/05/statistics2015.pdf　 |script-title=ja:カトリック教会現勢2015 |accessdate=2017-05-08 |publisher={{lang|ja|カトリック中央協議会}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

在鎖國時代，宗氏的對馬藩是江戶時代日本唯一獲得幕府許可和朝鮮半島進行貿易的地點，加上長崎是日本對西洋和中國貿易的據點，因此長崎縣內有兩個獲得官方許可的貿易據點<ref name="dictionary"></ref>{{rp|38}}。因長崎地位重要，被幕府列為直轄地。長崎的荷蘭人在當時都被集中居住在[[出島|出島]]，以便於幕府監視並和日本人隔離<ref>{{cite web |url=http://www.city.nagasaki.lg.jp/dejima/history/contents/index001.html　|script-title=ja:出島ヒストリー |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja}}</ref>。並且在元祿時期，原本允許散居的中國人也開始被集中居住在[[唐人屋敷|唐人屋敷]]<ref name="City"></ref>。元祿年間的長崎人口有約6萬人，其中有1萬人是中國人<ref name="dictionary"></ref>{{rp|39}}。江戶時代的長崎不僅是貿易中心，更是[[蘭學|蘭學]]等西洋文化、學術流入日本的窗口<ref name="dictionary"></ref>{{rp|43}}。與長崎相對，平戶、島原等地在江戶時代則是[[城下町|城下町]]，但規模相對較小<ref name="dictionary"></ref>{{rp|41}}。1792年（[[寬政|寬政]]4年），島原火山噴發，當地農業受到嚴重打擊<ref name="dictionary"></ref>{{rp|45}}。

===長崎縣的成立===
[[File:Atomic_cloud_over_Nagasaki_from_Koyagi-jima.jpeg|thumb]]]]
1858年（[[安政|安政]]5年）[[安政條約|安政條約]]簽訂之後，長崎成為開港五港之一<ref name="Chishi"></ref>{{rp|277}}。1868年，長崎奉行河津祐邦在聽聞幕府軍在[[鳥羽伏見之戰|鳥羽伏見之戰]]中戰敗之後出逃。九州各藩藩士和當地的有力人士決定設立長崎會議所，管理長崎政治。江戶幕府崩潰之後，政府成立長崎裁判所，後改為長崎府<ref name="dictionary"></ref>{{rp|46}}。1869年，長崎府改名為長崎縣<ref name="history">{{cite web |url=https://www.pref.nagasaki.jp/nagasakinorekishi/　 |script-title=ja:長崎県の歴史 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1871年8月，日本實施[[廢藩置縣|廢藩置縣]]，各藩均改為縣。同年11月，新成立不久的島原、平戶、大村、福江四縣和長崎縣合併<ref name="sugata">{{cite web |url=https://www.pref.nagasaki.jp/sugata/　 |script-title=ja:長崎県のすがた |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。此後長崎縣轄區範圍還經歷過多次擴大，更在1876年合併佐賀縣全縣，範圍相當於肥前、壹岐、對馬三國。1883年，佐賀縣再次獨立，長崎縣確立了現在的轄區範圍<ref name="sugata"></ref>。

長崎縣是日本最早開始產業近代化的縣之一。長崎造船所是日本重工業的發祥地之一<ref>{{cite web |url=http://www.mhi.co.jp/company/facilities/history/corner_03.html　 |script-title=ja:長崎造船所 史料館 官営期コーナー |accessdate=2017-05-08 |publisher={{lang|ja|三菱重工}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。高島煤礦在官營時期年產量達6.9萬噸，一度佔日本全國產量的三分之一<ref name="dictionary"></ref>{{rp|48}}。長崎市自明治後期開始是日本主要重工業都市和與中國之間的貿易據點之一，在1923年開通至上海的航線<ref name="history"></ref>，有「長崎縣上海市」之稱，可見長崎經濟和中國的密切關係<ref name="dictionary"></ref>{{rp|48}}。佐世保市在1880年代被選為日本海軍的四大軍港之一，從一個漁村迅速發展為軍港都市，在[[甲午戰爭|甲午戰爭]]時扮演了重要角色<ref name="dictionary"></ref>{{rp|49}}。1898年，長崎和佐世保兩地都開通了鐵路<ref name="dictionary"></ref>{{rp|50}}。1930年代後，隨著日本國內戰爭氣氛日益高漲，長崎縣內產業也日益軍事化，重工業地位更加提升<ref name="dictionary"></ref>{{rp|52}}。1942年，[[武藏號戰艦|武藏號戰艦]]在長崎市竣工<ref>{{cite web |url=http://www.mhi.co.jp/company/facilities/history/corner_08.html　 |script-title=ja:長崎造船所 史料館 戦艦武蔵コーナー |accessdate=2017-05-08 |publisher={{lang|ja|三菱重工}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[太平洋戰爭|太平洋戰爭]]後期，長崎縣也開始遭到空襲。1945年6月28日至29日，美軍對佐世保市進行{{link-ja|佐世保大空襲|佐世保大空襲|大空襲}}，超過1,200人遇難<ref>{{cite web |url=http://www.soumu.go.jp/main_sosiki/daijinkanbou/sensai/situation/state/kyushu_03.html　 |script-title=ja:佐世保市における戦災の状況（長崎県） |accessdate=2017-05-20 |publisher={{lang|ja|総務省}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。同年8月9日，美軍在長崎市的浦上地區[[長崎市原子彈爆炸|投下原子彈]]，造成73,884人遇難<ref>{{cite web |url=http://nagasakipeace.jp/japanese/atomic/record/scene/1103.html　 |script-title=ja:原爆の威力 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市 平和・原爆}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

長崎縣在二戰之後一度被盟軍佔領，並進行了實施普選制度、土地改革、學制改革等重大改革<ref name="dictionary"></ref>{{rp|54}}。長崎縣在1950年代初期進行了市町村的大規模合併，縣內市町村數在1956年減少到8市83町村<ref name="dictionary"></ref>{{rp|54}}。長崎縣經濟也在戰後快速復興，{{link-ja|長崎造船所|三菱重工業長崎造船所|長崎造船廠}}一度是世界最大的造船廠<ref>{{cite web |url=http://www.mhi.co.jp/company/facilities/history/corner_12.html　 |script-title=ja:長崎造船所 史料館 戦後の造船コーナー |accessdate=2017-05-08 |publisher={{lang|ja|三菱重工}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。佐世保市經濟同樣以造船業為主，使得造船業在戰後長崎縣工業中佔有中心地位<ref name="dictionary"></ref>{{rp|55}}。多山和多離島的地形特徵使得交通是制約長崎縣發展的原因之一。長崎縣在1960年代開始興建離島機場和離島橋樑，使得離島地區的交通有很大改善<ref name="history"></ref>。長崎縣在20世紀後半發生過兩次大洪水。1957年的{{link-ja|諫早豪雨|諫早豪雨}}的降水量達976毫米，導致683人遇難<ref name="dictionary"></ref>{{rp|24}}。1982年的{{link-ja|長崎大水害|長崎大水害}}的降水量達572毫米，導致294人遇難<ref>{{cite web |url=http://www.jma-net.go.jp/nagasaki-c/gyomu/nagasakisuigai/nagasaki.html　|script-title=ja:昭和57年7月豪雨（長崎大水害） |accessdate=2017-05-22 |publisher={{lang|ja|気象庁}} |language=ja}}</ref>。[[平成|平成]]時代之後，長崎縣先後完成了[[西九州自動車道|西九州自動車道]]、[[長崎自動車道|長崎自動車道]]、[[生月大橋|生月大橋]]、{{link-ja|大島大橋 (長崎縣)|大島大橋 (長崎県)|大島大橋}}、{{link-ja|長崎出島道路|ながさき出島道路}}<ref>{{cite web |url=http://www.nagasaki-dourokousha.or.jp/yuryou/dejima.html |script-title=ja:長崎県の有料道路　ながさき出島道路 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県道路公社}} |language=ja}}</ref>、[[女神大橋|女神大橋]]等大型工程，縣內交通有進一步改善。但另一方面，長崎縣亦面臨重工業蕭條、離島地區人口快速減少等嚴峻課題。

== 地理 ==
[[File:Fugendake_02_Pyroplastic_flow_area.JPG|thumb]]
長崎縣位於九州西北部，東西寬213公里，南北長307公里<ref name="sugata"></ref>。長崎縣按地形可以分為西彼杵、長崎、北松浦半島、平戶；五島、壹岐、對馬；多良岳、島原半島三個地區<ref name="Chishi"></ref>{{rp|260}}。長崎縣本土包括了{{link-ja|北松浦半島|北松浦半島}}、[[西彼杵半島|西彼杵半島]]、[[長崎半島|長崎半島]]、[[島原半島|島原半島]]四個半島，佐世保市、大村市、長崎市、島原市是四個半島的中心城市<ref name="dictionary"></ref>{{rp|21}}。長崎縣地形最大的特徵是海岸地區多是曲折的[[谷灣|谷灣式海岸]]，因此海岸線極為複雜。長崎縣三面環海，只有東面的佐賀縣一個陸上鄰縣。長崎縣的海岸線全長達4,195公里，在日本所有的縣中高居首位<ref name="coastline"></ref>。

長崎縣內地形面積比例是山地63.2%（日本平均值60%）、丘陵10.9%（日本平均值11%）、台地13%（日本平均值16%）、低地12.9%（日本平均值13%）。由於這一比例和日本平均值非常接近，因此有「日本列島的模型」之稱<ref name="dictionary"></ref>{{rp|22}}。長崎縣內主要平原有諫早平原、大村平原、長崎平原等，都是小規模平原<ref name="Chishi"></ref>{{rp|262}}。長崎縣雖然多山，但除了多良山系和雲仙山系之外縣內並無標高超過1,000米的山峰。縣內最高峰是[[雲仙岳|雲仙岳]]的平成新山，標高1,483米<ref>{{cite web |url=http://www.data.jma.go.jp/svd/vois/data/fukuoka/504_Unzendake/504_index.html |script-title=ja:雲仙岳 |accessdate=2017-05-22 |publisher={{lang|ja|気象庁}} |language=ja}}</ref>。長崎縣內火山活動活躍，雲仙岳和{{link-ja|多良岳|多良岳}}都是大型層狀火山<ref name="Chishi"></ref>{{rp|262}}。其中雲仙普賢岳更在1990年代初期至中期長期有噴發活動，是日本戰後最大規模的火山活動之一<ref>{{cite web |url=http://www.data.jma.go.jp/svd/vois/data/fukuoka/504_Unzendake/504_history.html |script-title=ja:雲仙岳　有史以降の火山活動 |accessdate=2017-05-22 |publisher={{lang|ja|気象庁}} |language=ja}}</ref>。長崎縣的河川受地形限制而大多短小湍急，縣內較大河川有本明川<ref>{{cite web |url=http://www.qsr.mlit.go.jp/nagasaki/river_dam/info/abouthonmyo.html |script-title=ja:本明川はこんな川です |accessdate=2017-05-22 |publisher={{lang|ja|国土交通省九州地方整備局}} |language=ja}}</ref>、相浦川等<ref>{{cite web |url=https://www.pref.nagasaki.jp/shared/uploads/2015/02/1423807669.pdf |script-title=ja:相浦川水系河川整備計画 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja}}</ref>。

據海上保安廳的統計，長崎縣島嶼數多達971個，數量高居日本首位，佔長崎縣面積的45%<ref name="nature">{{cite web |url=http://www.pref.nagasaki.jp/shared/uploads/2017/05/1493606153.pdf |script-title=ja:自然環境 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja}}</ref>。長崎縣的島嶼中面積超過1,000平方米的島嶼有594個，有人島有72個<ref>{{cite web |url=https://www.pref.nagasaki.jp/sima/qa/nagsaki-qa.html |script-title=ja:ながさきのしまQ&A |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja}}</ref>。距離長崎縣本土較近的主要離島有[[平戶島|平戶島]]、[[生月島|生月島]]、{{link-ja|的山大島|的山大島}}、{{link-ja|伊王島|伊王島}}、{{link-ja|鷹島 (長崎縣)|鷹島 (長崎県)|鷹島}}等島嶼。長崎縣外海的離島可以分為對馬島、壹岐島和五島列島三個部分。壹岐島本島和屬島面積共有139平方公里，全島是平坦的熔岩台地。自該島出發可乘坐高速船在一小時內到達[[福岡市|福岡市]]<ref name="Chishi"></ref>{{rp|300}}。對馬島面積698平方公里，是長崎縣面積最大的島嶼。對馬也是日本距朝鮮半島最近的領土，距離[[釜山市|釜山市]]只有40公里<ref name="Chishi"></ref>{{rp|300}}。五島列島由[[中通島|中通島]]、[[若松島|若松島]]、[[奈留島|奈留島]]、[[久賀島|久賀島]]、[[福江島|福江島]]五個主島和約200個屬島構成，面積638平方公里<ref name="Chishi"></ref>{{rp|300}}。五島列島在古代是日本距中國最近的地點，因此是遣唐使的出發地和倭寇的根據地。近世時則以基督教文化而著稱<ref name="Chishi"></ref>{{rp|300}}。

[[File:展海峰_-_panoramio.jpg|thumb]]
長崎縣擁有兩個國立公園和兩個國定公園。[[雲仙天草國立公園|雲仙天草國立公園]]以火山景觀為主，是日本首批國立公園之一<ref>{{cite web |url=http://www.env.go.jp/park/unzen/point/index.html |script-title=ja:公園の特長 |accessdate=2017-05-22 |publisher={{lang|ja|環境省}} |language=ja}}</ref>。[[西海國立公園|西海國立公園]]地形變化豐富，以外洋性多島海景觀著稱<ref>{{cite web |url=https://www.env.go.jp/park/saikai/point/index.html |script-title=ja:公園の特長 |accessdate=2017-05-22 |publisher={{lang|ja|環境省}} |language=ja}}</ref>。[[壹岐對馬國定公園|壹岐對馬國定公園]]和[[玄海國定公園|玄海國定公園]]則均擁有谷灣式海岸景觀和豐富的人文歷史特色<ref>{{cite web |url=https://www.pref.nagasaki.jp/bunrui/kurashi-kankyo/shizenkankyo-doshokubutsu/nature/park-nature/69621.html |script-title=ja:長崎県の自然公園の一覧 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja}}</ref>。

=== 氣候 ===
按[[柯本氣候分類法|柯本氣候分類法]]，長崎縣屬於[[副热带湿润气候|副热带湿润气候]]。由於長崎地形縣多山，因此即使相隔不遠的兩地在同一時間也可能天氣有較大差異<ref name="dictionary"></ref>{{rp|24}}。長崎縣氣候受對馬暖流的影響較強，年平均氣溫約有15至16℃，年降水量約有2,000毫米，氣候溫暖濕潤且溫差較小<ref name="dictionary"></ref>{{rp|24}}。2014年，長崎縣的全年降水量有2,134毫米，排名日本第11位。年平均氣溫有17℃，排名日本第五<ref name="nature"></ref>。長崎縣的降水主要分佈在[[梅雨|梅雨]]時期和[[颱風|颱風]]過境時期，梅雨末期時偶爾會有集中豪雨發生。長崎縣在冬季則受季風影響而多陰天，和[[日本海側氣候|日本海側氣候]]有相似之處。偶有降雪但少有積雪<ref name="dictionary"></ref>{{rp|24}}。長崎縣南部地區比北部地區氣候更加高溫多雨。北部地區雖然夏季比南部地區涼爽，但冬季也較為寒冷<ref name="dictionary"></ref>{{rp|24}}。

=== 行政区划 ===
在[[市町村合併|平成大合併]]之前，長崎縣原有79市町村（8市70町1村）。經過平成大合併之後，自2010年3月31日開始，長崎全縣市町村數量已減少至21市町（13市8町）<ref>{{cite web |url=https://www.pref.nagasaki.jp/bunrui/kenseijoho/kennokeikaku-project/sicho_gappei/ |script-title=ja:長崎県市町合併 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja}}</ref>。長崎縣北部地區有[[松浦市|松浦市]]、[[平戶市|平戶市]]、[[佐世保市|佐世保市]]、[[西海市|西海市]]四市，[[北松浦郡|北松浦郡]]的[[佐佐町|佐佐町]]，[[東彼杵郡|東彼杵郡]]的[[東彼杵町|東彼杵町]]、[[川棚町|川棚町]]、[[波佐見町|波佐見町]]。縣央地區有[[諫早市|諫早市]]、[[大村市|大村市]]兩市。島原地區有[[島原市|島原市]]、[[雲仙市|雲仙市]]、[[南島原市|南島原市]]三市。長崎地區有[[長崎市|長崎市]]和[[西彼杵郡|西彼杵郡]]（[[時津町|時津町]]、[[長与町|長与町]]）一市一郡。離島地區中，壹岐和對馬是一島一市（[[壹岐市|壹岐市]]和[[對馬市|對馬市]]），五島列島則分為[[五島市|五島市]]、[[南松浦郡|南松浦郡]]的[[新上五島町|新上五島町]]、北松浦郡的[[小值賀町|小值賀町]]、最北端的{{link-ja|宇久島|宇久島}}屬於佐世保市<ref>{{cite web |url=https://www.pref.nagasaki.jp/area/ |script-title=ja:長崎県市町合併 |accessdate=2017-05-22 |publisher={{lang|ja|地域で探す}} |language=ja}}</ref>。

== 人口 ==
1920年日本首次人口普查時，長崎縣有人口1,136,182人<ref name="history"></ref>。太平洋戰爭爆發前的1940年時，長崎縣有人口1,370,063人<ref>{{cite web |url=https://www.pref.nagasaki.jp/shared/uploads/2017/02/1487144700.pdf　 |script-title=ja:人口動態総覧、長崎県の年次推移 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1960年時，長崎縣人口達到176萬人的峰值，此後開始減少，比日本總人口開始減少的時間早了47年<ref name="population">{{cite web |url=https://www.pref.nagasaki.jp/shared/uploads/2015/12/1449474981.pdf　 |script-title=ja:長崎県長期人口ビジョン |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2017年5月1日時，長崎縣有人口1,357,270日，比峰值減少了約40萬人<ref>{{cite web |url=https://www.pref.nagasaki.jp/bunrui/kenseijoho/toukeijoho/idojinko/291845.html　 |script-title=ja:毎月の推計人口 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。離島地區在2010年時的人口更不到1960年的半數，只佔長崎縣總人口的不到一成<ref name="population"></ref>。長崎縣長期是人口外流縣，特別是以1960年代至70年代的煤礦關閉期的人口外流情況最為嚴重<ref name="population"></ref>。而縣內其他城市流出的人口未流入長崎市而是流入其他地區也是擴大長崎縣人口減少幅度的主因之一<ref name="population"></ref>。長崎縣人口的社會減少率高居日本第三位<ref>{{cite web |url=http://www.pref.nagasaki.jp/shared/uploads/2017/05/1493611644.pdf　 |script-title=ja:人口・世帯 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。老齡化和少子化也是導致長崎縣人口減少的主要原因。長崎縣自2002年開始人口的自然增長率就是負數<ref name="population"></ref>。但長崎縣的[[總和生育率|總和生育率]]長期以來高於日本平均值。2014年，長崎縣的總和生育率有1.66，排名日本第三<ref name="population"></ref>。

== 經濟 ==
2014年代，長崎縣的縣內生產總值有43,103億日元。人均縣民所得有235.2萬日元，相當於日本平均值的82%。就產業構造來看，第一產業佔長崎縣經濟總量的2.6%、第二產業佔16.2%、第三產業佔80.1%。長崎縣第一產業和政府服務在經濟中所佔比重大幅高於日本平均，而製造業和信息通信業的比重則大幅低於日本平均<ref>{{cite web |url=https://www.pref.nagasaki.jp/shared/uploads/2017/03/1488501916.pdf |script-title=ja:平成２６年度県民経済計算の概要 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
=== 第一產業 ===
由於多山的地形限制，長崎縣耕地面積比例較低。2015年，長崎縣耕地面積有49,100公頃，佔縣總面積的11.9%。總農戶數則有33,802戶<ref name="agricultural">{{cite web |url=https://www.pref.nagasaki.jp/shared/uploads/2017/04/1491380485.pdf |script-title=ja:平成29年度ながさきの農林業 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja |title=存档副本 |archive-url=https://web.archive.org/web/20171024154355/https://www.pref.nagasaki.jp/shared/uploads/2017/04/1491380485.pdf |archive-date=2017-10-24 |dead-url=yes }}</ref>。2015年，長崎縣農業生產額有1,553億日元，排名日本全國第22位和九州第五位<ref>{{cite web |url=http://www.maff.go.jp/kyusyu/press/toukei/attach/pdf/161222_13-1.pdf |script-title=ja:平成27年　農業産出額及び生産農業所得（九州）|accessdate=2017-05-22 |publisher={{lang|ja|九州農政局}} |language=ja}}</ref>。就生產金額來看，肉[[牛|牛]]是長崎縣最主要的農產品，其次是[[馬鈴薯|馬鈴薯]]、[[水稻|水稻]]、[[豬|豬]]、[[草莓|草莓]]。2015年時，長崎縣以上農產品的生產金額都超過100億日元<ref name="agricultural"></ref>。長崎縣[[枇杷|枇杷]]的產量是日本第一，馬鈴薯排名日本第二，草莓排名日本第三。[[蘆筍|蘆筍]]、[[洋蔥|洋蔥]]、[[白菜|白菜]]、[[凸頂柑|凸頂柑]]等農產品的產量也是日本前列<ref name="agricultural"></ref>。長崎縣森林面積有242,199公頃，森林覆蓋率有58.6%，林業生產額在近年約有60億日元<ref>{{cite web |url=http://www.pref.nagasaki.jp/shared/uploads/2016/12/1481781552.pdf |script-title=ja:長崎県の森林・林業|accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja}}</ref>。多島的地形和複雜的海岸線使得長崎縣有眾多漁港。2014年，長崎縣的捕撈漁業和養殖漁業的總漁獲量超過26萬噸<ref>{{cite web |url=http://www.pref.nagasaki.jp/shared/uploads/2016/07/1466130779.pdf |script-title=ja:水産業の概況 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja}}</ref>。長崎縣漁業的生產量和生產金額都是日本第二，僅次於北海道。就魚種來看，以[[日本竹筴魚|日本竹筴魚]]、[[白腹鯖|白腹鯖]]、[[鰹魚|鰹魚]]、[[金槍魚|金槍魚]]、[[鰤魚|鰤魚]]、[[管魷目|魷魚]]等魚種最為突出<ref>{{cite web |url=https://www.pref.nagasaki.jp/shared/uploads/2016/03/1456972341.pdf |script-title=ja:漁業生産 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県}} |language=ja}}</ref>。長崎縣的日本竹筴魚、[[真鯛|真鯛]]的生產量是日本第一<ref>{{cite web |url=http://www.joho-nagasaki.or.jp/invest/nagasaki/004/003.html |script-title=ja:観光と食 |accessdate=2017-05-22 |publisher={{lang|ja|公益財団法人長崎県産業振興財団}} |language=ja}}</ref>。

礦業在長崎產業近代化歷程中有重要作用。長崎市的高島煤礦是日本首個使用蒸汽機開採的煤礦。[[端島|端島]]又名軍艦島，是長崎縣的海底煤礦開採中心之一，在鼎盛時期興建了日本第一座鋼筋混凝土公寓，並有超過5.000人在島上生活<ref>{{cite web |url=http://www.at-nagasaki.jp/gunkan/intro/takashima/　 |script-title=ja:高島炭坑・端島炭坑 |accessdate=2017-05-22 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。然而1960年代之後，隨著能源革命的進展，長崎縣的煤礦急劇減少，1974年時已只剩高島煤礦和池島煤礦兩個煤礦。2001年，池島煤礦關閉，長崎縣內已沒有仍在開採的煤礦<ref name="Guide"></ref>{{rp|142}}。

== 文化 ==
=== 方言 ===
長崎縣的方言屬於九州西北部的{{link-ja|肥筑方言|肥筑方言|-{肥筑方言}-}}。長崎縣多山的地形使得古代各地往來不便，加上在江戶時代分為多個小藩，使得縣內各地方言差異較大<ref name="Dialect">{{cite book |author={{lang|ja|坂口至}}|script-title=ja:《長崎県のことば》 |year=1998 |publisher={{lang|ja|明治書院}} |location=東京都 |isbn=4-625-52242-0|language=ja}}</ref>{{rp|2}}。長崎縣本土地區的方言可以分為中南部本土方言（以{{link-ja|長崎方言|長崎弁}}为代表）和北部本土方言（以{{link-ja|佐世保方言|佐世保弁}}为代表）。離島地區則分為南部離島方言（{{link-ja|五島列島方言|五島列島方言}}）和北部離島方言（包括{{link-ja|壹岐方言|壱岐弁}}、{{link-ja|對馬方言|対馬方言}}）<ref name="Dialect"></ref>{{rp|2}}。中南部本土方言使用{{link-ja|二型式重音|二型式アクセント}}。北部本土方言则使用{{link-ja|無重音|無アクセント}}，和佐賀縣西部方言有較多類似之處。中南部本土方言和北部本土方言在文法和詞彙上也有若干不同<ref name="Dialect"></ref>{{rp|2}}。南部離島方言在語音上的特徵之一是語尾[[促音|促音]]、[[撥音|撥音]]较多，在重音上和北部本土方言相同。北部離島方言在重音上和福岡縣西部的-{筑前}-式重音相似，文法上和長崎縣內其他地區差異較大<ref name="Dialect"></ref>{{rp|3}}。

=== 飲食 ===
[[File:Shikairo_Nagasaki_Japan05s.jpg|left]]
長崎縣的飲食文化受到很強的西餐和中餐影響，當地盛行的[[卓袱|卓袱]]就是一種融合了中餐和西餐的宴會飲食<ref>{{cite web |url=http://nagasaki-gourmet.com/nagasakisippoku.html |script-title=ja:長崎の美味しい食べ物 |accessdate=2017-05-08 |publisher={{lang|ja|長崎の美味しい食べ物}} |language=ja}}</ref>。[[什錦麵|什錦麵]]和{{link-ja|皿烏冬|皿うどん}}是長崎市的兩大代表麵食，均由著名中餐廳[[四海樓|四海樓]]初代店主[[陳平順|陳平順]]發明於19世紀末期<ref>{{cite web |url=http://www.tabirai.net/sightseeing/tatsujin/0000263.aspx |script-title=ja:長崎ちゃんぽん 発祥からおすすめ情報まで |accessdate=2017-05-08 |publisher={{lang|ja|たびらい}} |language=ja}}</ref>。[[土耳其飯|土耳其飯]]是一種長崎特色的日式西餐，做法是在米飯上澆醬汁，配以豬排和義大利麵<ref>{{cite web |url=http://nagasaki-gourmet.com/torukorice.html |script-title=ja:長崎トルコライス |accessdate=2017-05-08 |publisher={{lang|ja|長崎の美味しい食べ物}} |language=ja}}</ref>。[[卡斯特拉|卡斯特拉]]則是長崎市具代表性的甜點，起源於古代西班牙的麵包，已有400年以上的歷史，現已成為日式西點的代表<ref>{{cite web |url=http://www.shooken.com/rekishi/ |script-title=ja:長崎カステラの歴史 |accessdate=2017-05-08 |publisher={{lang|ja|松翁軒}} |language=ja}}</ref>。佐世保市因戰後美軍的進駐而形成了獨特的[[漢堡包|漢堡包]]文化。{{link-ja|佐世保漢堡|佐世保バーガー}}是日本最具地域特色的漢堡之一<ref>{{cite web |url=http://nagasaki-gourmet.com/sasebo_burger.html |script-title=ja:佐世保バーガー |accessdate=2017-05-22 |publisher={{lang|ja|長崎の美味しい食べ物}} |language=ja}}</ref>。五島烏冬使用[[飛魚|飛魚]]煮出的湯作為底湯，是日本三大烏冬之一<ref>{{cite web |url=http://www.goto-tenobeudon.jp/history/index.html |script-title=ja:五島うどんの歴史 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県五島手延うどん振興協議会}} |language=ja}}</ref>。此外長崎縣還有具雑煮等獨具鄉土特色的菜餚<ref>{{cite web |url=https://www.nagasaki-tabinet.com/houjin/basic/food.cfm |script-title=ja:長崎の食文化 |accessdate=2017-05-22 |publisher={{lang|ja|長崎県観光連盟}} |language=ja}}</ref>。
