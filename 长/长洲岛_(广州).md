'''长洲岛，'''是位于廣州省城[[广州市|广州市]]东部的一個島嶼，是[[珠江口|珠江口]]内的江心[[岛屿|岛屿]]，共和國後建制歸属[[黄埔区|黄埔区]][[长洲街道|长洲街道]]，轄內常住[[人口|人口]]约2.7万余人。岛内留存有众多的軍事遗址、墓地、名胜景点和纪念胜地，为广州的旅游胜地之一。

== 地理 ==
岛长约4.2[[公里|公里]]，宽约2.1公里，陆地面积7.23[[平方公里|平方公里]]，略呈东北至西南走向、[[哑铃|哑铃]]形状。因形状狭长，故名'''长洲岛'''；又因地处[[黄埔_(广州)|黄埔]]（传统英文名称：{{lang|en|Whampoa}}），又称'''黄埔岛'''。原是珠江口内的岛屿，再由[[珠江|珠江]]的泥沙冲积而成，其主峰大石岗为[[花岗岩|花岗岩]]的山岗，[[海拔|海拔]]89.4[[米|米]]。东、南、西部地势较低，已开垦为农地，而北部山岗较多，为[[工业|工业]]和居民区。

== 历史 ==
[[File:Guangzhou_Haijun_Lieshi_Lingyuan.JPG|thumb]]
[[File:Sun_Yat-sen_memorial_monument.jpg|thumb]]
由于长洲、[[深井|深井]]一带水深、港阔，曾经是中国[[对外贸易|对外贸易]]的重要[[海港|海港]]，[[清朝|清朝]][[粵海关|粵海关]]黄埔分关设在这里，当年曾经商贾云集，盛极一时。岛上的[[柯拜船坞|柯拜船坞]]为中国最早的外商投资企业，也是中国早期产业工人发源地之一。舊日其西北為新洲墟。清[[乾隆|乾隆]]（年）中粵省開港之後，就將[[澳門|澳門]]作為貿易地區，[[黄埔区|黃埔]]這裡就作為卸貨區，外國商船多會在7月到長洲換貨，至[[冬季|冬季]]返回[[澳門|澳門]]<ref name=":0" />。

=== 古迹 ===
岛内[[古迹|古迹]]众多，最著名的有[[孙中山|孙中山]]创办于1924年的[[黄埔军校旧址|黄埔军校]]，为[[全国重点文物保护单位|全国重点文物保护单位]]。还有[[东征阵亡烈士墓|东征阵亡烈士墓]]、孙总理纪念碑、孙中山纪念馆、北伐纪念碑、教思亭、黄埔公园旧址、济深公园旧址、[[袖海亭|袖海亭]]、海军黄埔军事博览中心、深井古民宅、巴斯教墓地、[[竹丝公墓|竹丝公墓]]、凌氏大宗祠、曾氏大宗祠、金花古庙、南海神祠、洪福市、安来市等一批[[军事|军事]][[革命|革命]]类和[[历史|历史]][[文化|文化]]类古迹。

== 军事 ==
[[File:PLA_4801_Factory.jpg|thumb]]
由[[獅子洋|獅子洋]]入省河（[[珠江|珠江]]）必经长洲岛，在珠江前后航道交汇处，為粵省（省城）二重門戶，历来为兵家必争之地。清代时，归属[[广州府|廣州府]][[番禺縣|番禺縣]][[水師提督|水師提標]]中營管轄<ref name=":0">廣東海圖說·黃埔長洲  張之洞 廣雅書局 1889</ref>。

在[[鸦片战争|鸦片战争]]时期，清政府就在此设置了长洲炮台群，包括有：白兔岗炮台、白鹤岗炮台、大坡地炮台、蝴蝶岗炮台、新西岗炮台和旧西岗炮台等多座古[[炮台|炮台]]，形成守卫广州城最后一道海上的防线。

共和国后驻扎有[[中国人民解放军海军|中国人民解放军海军]]基地、海军4801工厂和4307工厂。

== 经济 ==
[[File:Changzhou_Island_way.jpg|thumb]]
岛内有丰富的历史文物和众多名胜古迹，为岛内主要经济来源，未来计划建成长洲岛历史文化保护区<ref>[http://www.chinalawedu.com/news/1200/22598/22617/22843/22853/2006/4/pa896239332946002183-0.htm 关于实施长洲岛历史文化保护区控制性详细规划的通告] {{webarchive|url=https://web.archive.org/web/20070220011540/http://www.chinalawedu.com/news/1200/22598/22617/22843/22853/2006/4/pa896239332946002183-0.htm |date=2007-02-20 }}，法律教育网</ref>。[[农业|农业]]以经济作物为主，盛产[[水果|水果]]，有[[甘蔗|甘蔗]]、[[蔬菜|蔬菜]]、[[香蕉|香蕉]]等，[[特产|特产]]有深井霸王花([[剑花|剑花]])和长洲粉葛。[[工业|工业]]以[[针织|针织]]、[[服装|服装]]为主，还有[[黄埔造船厂|黄埔造船厂]]、[[黄埔水泥厂|黄埔水泥厂]]等[[重工业|重工业]]。

== 公共交通 ==
目前岛内交通并较发达，[[道路|道路]]多沿山坡而建，曲折、坡度大且小路较多，主要有[[金洲大道|金洲大道]]、长洲路、金蝶路等，有赤坎大桥连接[[广州大学城|广州大学城]]。已经建成了[[鱼珠|鱼珠]]至长洲、[[新港路|新港东路]]至长洲的过江隧道和环岛公路。连接广州市区及番禺化龙的[[新化快速路|新化快速路]]亦在此经过。
