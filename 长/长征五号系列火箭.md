'''长征五号系列运载火箭'''是[[中国航天科技集团公司|中国航天科技集团公司]]研制的新一代五米直径低温液体捆绑式[[重型运载火箭|重型运载火箭]]系列。其到达[[近地轨道|近地轨道]]的最大有效载荷能力为25噸<ref name="sd20090304">
{{cite news |title=Long March 5 Will Have World's Second Largest Carrying Capacity |url = http://www.spacedaily.com/reports/Long_March_5_Will_Have_World_Second_Largest_Carrying_Capacity_999.html |accessdate=2012-09-04 |newspaper=Space Daily|date=2009-03-04}}</ref>，而到达[[地球同步轉移軌道|同步转移轨道]]的最大有效载荷能力为14噸。<ref name="sdc20120730">
{{cite web |url=http://www.space.com/16814-china-engine-test-long-march-5-rocket.html |title=China Tests Powerful Rocket Engine for New Booster |website = Space.com |author=Space.com staff |date=30 July 2012 |quote = The more capable Long March 5 rocket is expected to help the country achieve its goal of constructing a space station in orbit by the year 2020, as well as play a key role in China's future space exploration aims beyond low-Earth orbit. The rocket's maiden launch is expected to occur in 2014. }}</ref>

长征五号系列为中国第一个从总体到分系统均采用最新技术的大型液体运载火箭系列<ref name="长征五号">{{cite news |url = http://www.airforceworld.com/a/20150408/490.html |title = 长征五号火箭 |website = AirForceWorld.com |date=2016-06-12 |accessdate = 2016-07-01}}</ref>，是中国目前研制规模和技术跨度最大的航天运输系统工程，新技术比例達95%。长征五号运载火箭载荷能力与[[欧洲空间局|欧洲空间局]]的[[阿丽亚娜5型运载火箭|阿丽亚娜5型运载火箭]]、[[美国|美国]]的[[德尔塔-4运载火箭|德尔塔-4重型运载火箭]]以及[[俄罗斯|俄罗斯]]的[[质子-M运载火箭|质子-M运载火箭]]同级，均屬於大運載的重型火箭；是中國现役起飛質量最大、芯级直径最粗、運載能力最強的火箭。

== 历史 ==
长征五号早於1986年在[[863計劃|863計劃]]支持下开展前期论证和攻关，2001年5月国防科工委开展预先研究。2000年至2001年，120噸級液氧煤油發動機[[YF-100|YF-100]]及50噸級液氫液氧發動機[[YF-77|YF-77]]先後开始立项研制。在20年的可行性研究之后，中国政府在2006年10月立项批准了火箭的开发<ref name="sd20090304" />，由2008年11月成立的[[天津航天长征火箭制造有限公司|天津航天长征火箭制造有限公司]]生产{{坐标|39|06|09.63|N|117|30|05.01|E}}，制造基地设於沿海城市[[天津|天津]][[滨海新区|滨海新区]][[天津经济技术开发区|开发区西区]]。歷十年磨一箭，2016年11月3日20时43分，長征五號于[[海南岛|海南岛]][[中国文昌航天发射场|中国文昌航天发射场]]首次發射并獲圓滿成功。<ref name=":0">{{Cite web|url=http://news.xinhuanet.com/politics/2016-11/03/c_1119846192.htm|title=长五首飞成功！中国“奔月”“探火”靠它了-新华网|accessdate=2016-11-16|work=news.xinhuanet.com}}</ref>

== 研发 ==
[[File:CZ-5_launch_vehicle_model.JPG|缩略图]]

长征五号系列火箭的主要目标是满足在未来20-30年中国到达[[近地轨道|近地轨道]]和[[地球同步轉移軌道|同步转移轨道]]任务的大型有效载荷要求。CZ-5项目最初于2001年2月宣布。然而2007年中国东北展会期间，开发商透露直至當年才最终批准研发預算。<ref>{{Cite web|url=http://mt.sohu.com/20161101/n471945274.shtml|title=长五运载火箭计划11月上旬发射 将承担空间站等重大发射任务-搜狐|accessdate=2016-11-16|work=mt.sohu.com}}</ref>

长征五号运载火箭服役和位于海南岛的文昌航天发射场落成使得[[中国|中国]]具备25吨的[[近地轨道|近地轨道]]运载能力和14吨的[[地球同步轉移轨道|地球同步轉移轨道]]运载能力，可发射20吨级长期有人照料的[[空间站|空间站]]、大型[[空间望远镜|空间望远镜]]、返回式[[月球探测器|月球探测器]]、[[深空探测器|深空探测器]]、超重型[[人造卫星|应用卫星]]等。<ref name=":0" />

其120噸級液氧發動機設計難度大，必須獨力開發上百項新技術或新材料，中国航天推进技术研究院製造初始四台樣機在高台測試時，兩台啟動爆炸，兩台燃油系統燒毀，<ref>[http://tv.cctv.com/2016/04/23/VIDEjLX30mn6AnqqcnLfJ7wW160423.shtml 《焦点访谈》 20160423 长征五号十年磨一“箭”]</ref>第五台樣機才成功，液態燃料槽在2012年6月30日靜力測試時也發現焊接口裂縫，薄壁大直徑燃料槽是國際高端機密技術，焊接工藝和結構設計全靠自力摸索，最終研製成功。

== 设计 ==
长征五号系列火箭現任總設計師是李東，首任總設計師（2001年1月－2006年1月）是徐盛華。[[中國運載火箭技術研究院|中國運載火箭技術研究院]]及[[上海航天技術研究院|上海航天技術研究院]]分別承擔芯级及助推器的研制任務。

长征五号系列火箭的设计思想以通用化、系列化、组合化为重点。可搭载三种专门为其设计的[[火箭发动机|火箭发动机]]，完全采用无毒无污染的[[液氢|液氢]]或[[煤油|煤油]]和[[液氧|液氧]]推进剂，分别为推力为120吨的[[YF-100|YF-100]]液氧煤油[[高压补燃循环|高压补燃循环]]发动机、推力为50吨的[[YF-77|YF-77]][[燃氣發生器循環|燃氣循環]]氢氧发动机和推力为9吨的[[YF-75D|YF-75D]][[膨脹循環|膨脹循環]]氢氧发动机。长征五号系列实行模块化设计，由直径为5米、3.35米和2.25米的三种模块构成。其中5米模块包含两个50吨级的YF-77发动机（芯一级）或两个9吨级的[[YF-75D|YF-75D]]发动机（芯二级），3.35米模块包含两个120吨级的YF-100发动机，2.25米模块包含一个120吨级的YF-100发动机。模块化设计的好处是可以根据需要把不同模块组装成不同推力的火箭，以执行不同的任务。例如系列中最为强大的型号长征五号，以两个5米模块为主推进器，另配有4个3.35米模块为助推器。

120吨级的YF-100[[液氧|液氧]][[煤油|煤油]]发动机已於2012年7月29日測試成功<ref name="我国新一代大推力火箭发动机“极限工况热试车”成功">{{cite web|author =陈钢|title =我国新一代大推力火箭发动机“极限工况热试车”成功|publisher =新华网|date =2012-07-29|language =zh|accessdate =2014-12-08|url =http://news.xinhuanet.com/tech/2012-07/29/c_112562960.htm|deadurl =yes|archiveurl =https://web.archive.org/web/20120801060915/http://news.xinhuanet.com/tech/2012-07/29/c_112562960.htm|archivedate =2012-08-01}}</ref>。但此前研发遭遇一些困难，进度数次推迟<ref name="新一代运载火箭计划2016年首飞">{{cite web |author = |title = 新一代运载火箭计划2016年首飞 |publisher = 中国中央电视台 |work = 新闻联播 |date = 2014-12-08 |language = zh |accessdate = 2014-12-08 |url = }}</ref><ref name = "长征五号火箭拟于2015年上半年首飞 搭载试验卫星">{{cite web |author = 万淑艳 |title = 长征五号火箭拟于2015年上半年首飞 搭载试验卫星 |publisher = 中国新闻网 |date = 2014-03-05 |language = zh |accessdate = 2014-04-07 |url = http://www.chinanews.com/gn/2014/03-05/5914959.shtml}}</ref><ref name="China's Long March 5 Will Not Launch Until 2015: Another delay besets the heavy launcher on which China's space program is betting">{{cite web |author = Bradley Perrett |title = China's Long March 5 Will Not Launch Until 2015: Another delay besets the heavy launcher on which China's space program is betting |publisher = Aviation Week & Space Technology |date = 2013-05-25 |language = en |accessdate = 2014-12-08 |url = http://aviationweek.com/awin/china-s-long-march-5-will-not-launch-until-2015 |quote = "In 2007, the Long March 5 was due to make its first flight in 2013. == 构型 ==
=== 长征五号运载火箭 ===
{{main|长征五号运载火箭}}
[[长征五号运载火箭|长征五号运载火箭]]为二级半构型，芯级采用5米直径，捆绑4个3.35米直径斜头锥助推器，是系列中最为强大的型号。全箭总长56.97米，起飞质量878.556吨，起飞推力约1062吨<!--一說1078吨-->，主要承擔地球同步转移轨道任務，运载能力14吨的，較中国原有运载火箭提升2.5倍。原代号為长征五号E。长征五号运载火箭整流罩長度為12.3公尺<ref name="guokr">{{cite web|url=http://www.guokr.com/article/441822/|title=长征五号首发，中国跻身航天强国|author=Lsquirrel|publisher=果壳网|date=2016-11-03|accessdate=2017-06-27}}</ref>。

=== 长征五号乙运载火箭 ===
[[长征五号乙运载火箭|长征五号乙运载火箭]]為长征五号簡化型，取消第二级，为一级半构型，是中國首款一級直接入軌火箭，全箭总长53.7米，起飞重量837.5吨，近地轨道运载能力25吨，主要承担[[中国空间站|中国空间站]]舱段发射任务等近地轨道任務。长征五号乙运载火箭整流罩長度為20.5公尺<ref name="guokr"/>。
