'''长葛市第一高级中学'''，简称'''长葛一高'''，是[[长葛市|长葛市]]的一所[[高级中学|高级中学]]，老校区位于长葛市八七路164号。

== 概述 ==
学校创建于1907年，由“许长中等蚕桑实业学堂”沿革而来。目前校园占地7.8万平方米。

教师中[[本科|本科]]学历者为80%以上，其中许昌市级拔尖人才2人，省级学科带头人3人(长葛市仅此3人)，省级骨干教师2人，许昌市级骨干教师14人。学校为[[许昌市|许昌市]]重点高中和[[河南省示范性高中|河南省示范性高中]]。学校曾培养出1998年河南省文科状元盛艳慧（考入[[北京大学法学院|北京大学法学院]]）<ref>{{cite web |url=http://www.21xc.com/Article/ShowArticle.asp?ArticleID=103855 |language= |title=昔日状元今何在 |publisher=许昌晨报 |date=2007-05-30 |accessdate= }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、刘峰、邓高等一批[[清华|清华]]、[[北大|北大]]学子<ref>{{cite web|url=http://221.176.208.196/zwgk/readwk.asp?wjid=721&filetype=3|language=|title=长葛市人民政府办公室文件|publisher=长葛市人民政府办公室|date=二○○五年十月二十八日|accessdate=}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2006年5月，一高成为[[济南军区|济南军区]][[国防生|国防生]]生源基地。<ref>{{cite web |url=http://cgyg.30edu.com/news/e788a66f-f3fc-487e-aa1d-76a08139f508/d16a24bf-bbc9-412d-92ec-5e8d3e0f1aab.htm |language= |title=长葛市一高成为济南军区国防生生源基地 |publisher=长葛市一高 |date=2006-05-17 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20120211161856/http://cgyg.30edu.com/news/e788a66f-f3fc-487e-aa1d-76a08139f508/d16a24bf-bbc9-412d-92ec-5e8d3e0f1aab.htm |archivedate=2012-02-11 }}</ref>

2011年7月9日，一高'''新校区'''在市南环路举行工程奠基典礼。新校区总投资1.6亿元，占地面积300亩，总建筑面积13.7万平方米，主要有教学楼、实验楼、艺术楼、图书馆、行政楼、学生宿舍楼、体育馆、食堂、浴室、塑胶跑道及看台等设施<ref>{{cite web |url=http://www.changge.gov.cn/cgcms/jsp/preview.jsp?ColumnID=230&TID=20110711110756449290809 |language= |title=长葛市一高新校区建设项目正式启动 |publisher=电政办 |date=2011-07-11 |accessdate= }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。设计容纳学生6000名，一期工程将于2012年暑期完工。新校区将成为长葛市的标志性建筑群。<ref>{{cite web |url=http://cgyg.30edu.com/news/3d5742a5-dff5-4ef5-9a53-6369a6b1ed44/e5e516c1-5e2b-4407-b084-0419925db435.htm |language= |title=高新校区举行开工奠基仪式 |publisher=长葛市一高 |date=2012年2月27日 |accessdate= }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
