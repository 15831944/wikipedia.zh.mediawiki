'''长江上游'''主干流指[[湖北省|湖北省]][[宜昌市|宜昌市]]以上[[长江|长江]]河段，包括[[川江|川江]]、[[金沙江|金沙江]]、[[通天河|通天河]]和[[长江源|江源]]水系。[[干流|干流]][[上游|上游]]河段长4,504公里，[[流域|流域]][[面积|面积]]约100万平方公里。

== 江源 ==
{{main|长江源}}
江源段自正源[[沱沱河|沱沱河]][[唐古拉山脉|唐古拉山脉]]中段[[格拉丹冬|格拉丹冬]]雪山群[[姜古迪如冰川|姜古迪如冰川]]南侧海拔5,820米[[雪线|雪线]]流出。从雪线算起，包括通天河河段全程，全长1,174公里。
* 自雪线流出算起与南源[[当曲|当曲]]汇合于[[囊极巴陇|囊极巴陇]]，流程346公里。
* [[通天河|通天河]]段全长828公里。自囊极巴陇东流，与北源[[楚玛尔河|楚玛尔河]]汇合为通天河上段，全长278公里。楚玛尔河汇合口以下至[[玉树县|玉树县]][[巴塘河口|巴塘河口]]为通天河下段，全长长约550公里，通天河下段主要支流有主要有[[邦巴涌|邦巴涌]]、[[色吾曲|色吾曲]]、[[登恩涌|登恩涌]]、[[聂恰曲|聂恰曲]]、[[德曲|德曲]]、[[细曲|细曲]]、[[益曲|益曲]]、[[巴塘河|巴塘河]]等。

== 金沙江 ==
{{main|金沙江}}
金沙江为长江上游干流部分，地处[[青藏高原|青藏高原]]和[[滇北高原|滇北高原]]，上自[[青海省|青海省]][[玉树县|玉树县]]巴塘河口，下止四川省宜宾市[[岷江口|岷江口]]，全长约2290公里，约占长江上游干流全长的三分之二。流经青海省、[[西藏自治区|西藏自治区]]、[[四川省|四川省]]和[[云南省|云南省]]，流域面积36.2万平方公里，约占长江上游流域面积的36%。[[明朝|明朝]]以前，一直以[[岷江|岷江]]为长江正源，经过地理学家[[徐霞客|徐霞客]]实地考察，才纠正了“江源于岷”的谬误。金沙江流域面积1200平方公里以上的支流有18条，其中超过1万平方公里一、二级支流9条，[[雅砻江|雅砻江]]流域面积达12.84万平方公里，为最大的支流。金沙江从玉树县巴塘河口至云南省[[丽江市|丽江市]][[石鼓镇|石鼓镇]]为上段，从石鼓镇至四川省[[屏山县|屏山县]][[新市镇|新市镇]]为中段，从新市镇至宜宾市岷江口为下段。

== 川江河段 ==
{{main|川江}}
[[宜宾市|宜宾市]]岷江口以下至[[宜昌市|宜昌市]]以上的长江河段，全长1040公里。跨越[[青藏高原|青藏高原]]和[[横断山脉|横断山脉]]，横贯整个[[四川盆地|四川盆地]]，自西向东流经四川省、[[重庆市|重庆市]]和[[湖北省|湖北省]]。此间汇入的主要一级支流有[[岷江|岷江]]、[[沱江|沱江]]、[[嘉陵江|嘉陵江]]和[[乌江|乌江]]。流域面积约532200平方公里，面积在1000平方公里以上的一级支流有24条。川江干流水能理论蕴藏量达2470万千瓦，规划、在建或已建的水利工程有[[石硼|石硼]]、[[朱杨溪|朱杨溪]]、[[小南海|小南海]]、[[三峡工程|三峡]]、[[葛洲坝水利枢纽|葛洲坝]]等工程，[[长江三峡|长江三峡]]位于川江的[[奉节县|奉节县]]至[[宜昌市|宜昌市]]之间一段。
