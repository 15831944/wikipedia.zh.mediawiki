'''长崎市'''（{{jpn|j=長崎市|hg=ながさきし|rm=Nagasaki shi|f=y}}）是位於[[日本|日本]][[九州_(日本)|九州]]西部的[[都市|都市]]，為[[长崎县|长崎县]][[日本都道府縣廳所在地|縣治]]，也是日本西部的重要港湾城市。其歷史始於[[安土桃山時代|安土桃山時代]]對[[長崎港|長崎港]]的開拓，在[[江戶時代|江戶時代]]的[[鎖國_(日本)|鎖國]]時期，更是日本唯一的國際貿易港口，與[[荷蘭|荷蘭]]、[[中國|中國]]有密切的交流，並有外國人居住區[[出島|出島]]<ref>{{cite web |url=http://www.city.nagasaki.lg.jp/nagazine/hakken0311/　 |script-title=ja:出島回想録 出島が日本と世界にもたらしたもの |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。因此長崎受到西方影響較深，擁有許多[[歐洲|歐洲]]風格的建築，加上市區多坡道，使得長崎的都市景觀和大多數日本其他都市頗為不同。與西方交流的歷史使得早期有許多傳教士隨著商船來到長崎，造成現在長崎擁有較多的[[天主教|天主教]]信徒，天主教會在長崎單獨設有[[天主教長崎總教區|教區]]<ref>{{cite web |url=http://www.nagasaki.catholic.jp/　 |script-title=ja:教区の歴史 |accessdate=2017-05-08 |publisher={{lang|ja|カトリック長崎大司教区}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。也因為與中國的貿易往來，從江戶時期就有許多[[中國人|中國人]]居住於此，[[长崎新地中华街|长崎新地中华街]]是日本三大[[唐人街|唐人街]]之一。在[[第二次世界大戰|第二次世界大戰]]期間，長崎市曾經受到[[長崎市原子彈爆炸|原子彈轟炸]]，成為繼[[广岛市原子弹爆炸|廣島]]之後世界第二個、也是目前最後一個遭到[[核武器|核武器]]攻擊的城市。

長崎市中心三面環山，因大量住宅建於山坡地上，形成許多階梯式或位於斜坡的街道。在長崎市現有建成區面積中，高達七成是[[山坡地|山坡地]]<ref name="Data">{{cite web |url=http://www.city.nagasaki.lg.jp/syokai/730000/731000/p000297_d/fil/34_databook2014.pdf　 |script-title=ja:長崎市データブック2014 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。多山的地形雖然成就了長崎特有的都市景觀，但也造成了交通阻塞和開發用地受限等問題。近年長崎市的人口減少幅度在日本居於前列，已成為深刻的社會問題。但另一方面，長崎憑藉山海毗鄰的獨特景觀和濃郁的異國風情而吸引了眾多遊客，是日本主要觀光都市之一。

== 历史 ==
===十六至十八世紀===
[[File:Plattegrond_van_de_Stad_van_Nakasaki.jpg|thumb]]
長崎的歷史開始於1570年（[[元龜|元龜]]元年）[[吉利支丹大名|吉利支丹大名]][[大村純忠|大村純忠]]將長崎浦開放給[[葡萄牙|葡萄牙]]人作為貿易港<ref name="History1">{{cite web |url=http://www.city.nagasaki.lg.jp/syokai/750000/757000/p007003.html　 |script-title=ja:歴史年表　室町時代・安土桃山時代・江戸時代 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。大村純忠於翌年在長崎興建長崎六町。當時長崎六町的居民多是附近的天主教信徒和來自各地的移民<ref name="Chishi">{{cite book |author={{lang|ja|野澤秀樹・堂前亮平・手塚章}}|script-title=ja:《日本の地誌10　九州・沖縄》 |year=2008 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-16770-2|language=ja}}</ref>{{rp|276}}。1560年（[[天正_(日本)|天正]]8年），大村純忠將長崎奉獻給[[耶穌會|耶穌會]]<ref name="History1"></ref>。然而在1587年（[[天正_(日本)|天正]]15年），[[豐臣秀吉|豐臣秀吉]]在平定九州之後發布[[伴天連追放令|伴天連追放令]]，驅逐天主教傳教士，並在翌年將長崎作為直轄領地<ref name="History1"></ref>。在長崎成為直轄領地初期，當局對天主教的管制並不嚴格，甚至還有新教堂興建<ref name="Chishi"></ref>{{rp|276}}。但在1597年（[[慶長|慶長]]元年），當局下令處死長崎的[[日本二十六聖人|26名天主教信徒]]<ref>{{cite web |url=http://www.catholic-takatsuki.jp/26seijin/　 |script-title=ja:日本二十六聖人 |accessdate=2017-05-08 |publisher={{lang|ja|カトリック高槻教会}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。1612年（[[慶長|慶長]]17年）{{link-ja|岡本大八事件|岡本大八事件}}之後，[[德川家康|德川家康]]更發布禁教令，長崎的教堂全部被毀<ref name="Chishi"></ref>{{rp|276}}。1622年（[[元和_(日本)|元和]]8年）更發生55名天主教徒被公開處死的{{link-ja|元和大殉教|元和の大殉教}}事件<ref>{{cite web |url=http://www.collegium.or.jp/~take/christi/rekisi3.html　 |script-title=ja:キリシタン史　江戸初期の大迫害 |accessdate=2017-05-08 |publisher={{lang|ja|大阪コレギウム}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。

同一時期，長崎的人口持續增加，城市規模也從最初的6町增加到23町<ref name="Chishi"></ref>{{rp|276}}。[[江戶時代|江戶時代]]初期，長崎雖然是幕府設有長崎奉行的直轄領地，但另一方面也是自治色彩較強的城市<ref name="Chishi"></ref>{{rp|276}}。然而在德川幕府開始實施[[鎖國_(日本)|鎖國]]政策之後，長崎開始成為日本唯一的對外貿易據點。1616年，幕府下令外國（中國例外）船隻只能在長崎和[[平戶市|平戶]]進行貿易<ref name="History1"></ref>。1634年（[[寬永|寬永]]11年），幕府開始修築[[出島|出島]]，將散居長崎各地的葡萄牙人集中居住於此<ref>{{cite web |url=http://www.city.nagasaki.lg.jp/dejima/history/contents/index001.html　|script-title=ja:出島ヒストリー |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja}}</ref>。翌年更將外國船隻限制在長崎一地，並禁止日本人前往海外和居住在海外的日本人回國<ref name="History1"></ref>。1637年（寬永14年）發生[[島原之亂|島原之亂]]後，幕府禁止葡萄牙人進入日本，出島一度變為空地。直到1641年（寬永18年）幕府將原本位於平戶的[[荷蘭|荷蘭]]商館搬島出島為主。這一事件也象徵鎖國體制的完成<ref name="History1"></ref>。1688年（[[元祿|元祿]]元年），長崎的中國商船數達到194艘的史上最高紀錄。但在翌年，幕府下令將原本散居長崎各地的中國人集中遷移到[[唐人屋敷|唐人屋敷]]<ref name="History1"></ref>。1663年（[[寬文|寬文]]3年），長崎發生大火，市區大部分被燒毀。此後長崎重新進行都市規劃<ref name="History1"></ref>。

===十九世紀至二戰===
[[File:Atomic_cloud_over_Nagasaki_from_Koyagi-jima.jpeg|thumb]]]]
十九世紀之後，[[美國|美國]]、[[俄羅斯|俄羅斯]]、[[英國|英國]]的艦船都曾到訪長崎並要求開港，但都被幕府拒絕<ref name="History1"></ref>。發生在1828年（[[文政|文政]]11年）的{{link-ja|西博德事件|シーボルト事件}}更使幕府和荷蘭的關係更加緊張<ref>{{cite web |url=http://www.city.nagasaki.lg.jp/kanko/820000/828000/p009222.html　 |script-title=ja:シーボルトの生涯 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。1858年（[[安政|安政]]5年）[[安政條約|安政條約]]簽訂之後，長崎成為開港五港之一，但這也象徵長崎的貿易壟斷地位的終結<ref name="Chishi"></ref>{{rp|277}}。開港之後的長崎在市區南郊興建了外國人居留地，以便隔離外國人和日本人<ref name="Chishi"></ref>{{rp|277}}。外國人居留地聚集了眾多教堂、領事館和西洋式住宅，[[大浦天主堂|大浦天主堂]]和[[哥拉巴園|哥拉巴園]]等長崎代表性的歐式建築均修建於十九世紀後期<ref name="Chishi"></ref>{{rp|277}}。長崎的產業也在這一時期開始近代化。1861年，長崎首個鋼鐵工廠竣工<ref>{{cite web |url=http://www.mhi.co.jp/company/facilities/history/corner_03.html　 |script-title=ja:長崎造船所 史料館 官営期コーナー |accessdate=2017-05-08 |publisher={{lang|ja|三菱重工}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。1870年代初期，長崎不僅鋪設了和[[上海|上海]]之間的海底電纜，還開始了至東京的郵政及電信服務<ref name="History2">{{cite web |url=http://www.city.nagasaki.lg.jp/syokai/750000/757000/p007004.html　 |script-title=ja:歴史年表　明治時代・大正時代 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。1886年，停靠在長崎的[[北洋水師|北洋水師]]艦上軍人未獲許可登陸長崎，並和日本警察發生衝突，是為[[長崎事件|長崎事件]]，最終以雙方各自妥協收場<ref>{{cite web |url=http://www.jikkyo.co.jp/contents/download/9992656956　 |script-title=ja:戦争に酔う国民　─日清戦争と日本人─ |accessdate=2017-05-08 |publisher={{lang|ja|実教出版}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20180327032027/http://www.jikkyo.co.jp/contents/download/9992656956%E3%80%80 |archivedate=2018-03-27 }}</ref>。

1889年，長崎正式設[[市制_(日本制度)|市]]，成為日本首批設市的城市之一。成立之初的長崎市面積有7平方公里，人口有54,502人<ref name="Data"></ref>。十九世紀末期的長崎市集中了海關等重要機構，是中央政府在九州的據點城市之一<ref name="Chishi"></ref>{{rp|278}}。但另一方面，由於[[橫濱港|橫濱港]]、[[神戶港|神戶港]]、[[北九州港|門司港]]的發展，長崎港的貿易地位迅速下降。明治末期時長崎港佔全日本貿易額的比重已只有約1%<ref name="Chishi"></ref>{{rp|278}}。造船產業在這一時期取代貿易，成為長崎經濟的主要產業。1871年，日本政府設立官營{{link-ja|長崎造船所|三菱重工業長崎造船所}}，該造船廠在1887年出售給三菱公司之後成長為日本最大造船廠之一，修建了日本首艘鐵製蒸汽船「夕顏丸」和首艘大型貨客船「常陸丸」<ref>{{cite web |url=http://www.mhi.co.jp/company/facilities/history/corner_04.html　 |script-title=ja:長崎造船所 史料館 三菱創業期コーナー |accessdate=2017-05-08 |publisher={{lang|ja|三菱重工}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。三菱公司還在長崎海域開採海底煤炭<ref>{{cite web |url=http://www.at-nagasaki.jp/gunkan/intro/takashima/　 |script-title=ja:高島炭坑・端島炭坑 |accessdate=2017-05-08 |publisher={{lang|ja|明治日本の産業革命遺産}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>，使得長崎成為三菱公司的[[公司市鎮|公司市鎮]]。1920年日本首次人口普查時，長崎的人口達176,534人，是當時九州人口最多城市<ref name="History2"></ref>。

由於日本太平洋沿岸地區港口的發展，明治之後長崎港的對外貿易主要局限在中國和東南亞。1923年，長崎和上海之間的{{link-ja|日華連絡船|日華連絡船}}「長崎丸」啟航。該船更在1930年實現了和鐵路直接換乘<ref>{{cite web |url=http://tabinaga.jp/history/view.php?category=3&hid=471&offset=1　|script-title=ja:大正・昭和の長崎へ |accessdate=2017-05-08 |publisher={{lang|ja|旅する長崎学}} |language=ja}}</ref>。1934年，長崎市舉辦國際產業觀光博覽會，象徵走出[[大蕭條|大蕭條]]陰影<ref>{{cite web |url=http://www.nagasaki-cci.or.jp/nagasaki/130anniv/step/step1/step1_06.html　 |script-title=ja:国際産業観光博覧会の開幕 |accessdate=2017-05-08 |publisher={{lang|ja|長崎商工会議所}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。然而隨著[[第二次世界大戰|第二次世界大戰]]的開始，長崎市的軍需產業都市色彩日益增強。1942年，[[武藏號戰艦|武藏號戰艦]]在長崎竣工<ref>{{cite web |url=http://www.mhi.co.jp/company/facilities/history/corner_08.html　 |script-title=ja:長崎造船所 史料館 戦艦武蔵コーナー |accessdate=2017-05-08 |publisher={{lang|ja|三菱重工}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。長崎市自1944年開始遭到美軍六次空襲，目標多是工廠<ref>{{cite web |url=http://www.soumu.go.jp/main_sosiki/daijinkanbou/sensai/situation/state/kyushu_02.html　 |script-title=ja:長崎市における戦災の状況（長崎県） |accessdate=2017-05-08 |publisher={{lang|ja|総務省}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。1945年8月9日，美軍在長崎的浦上地區[[長崎市原子彈爆炸|投下原子彈]]，爆炸中心4公里範圍遭到毀滅性打擊，造成73,884人遇難、18,409戶房屋倒塌受損<ref>{{cite web |url=http://nagasakipeace.jp/japanese/atomic/record/scene/1103.html　 |script-title=ja:原爆の威力 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市 平和・原爆}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。長崎市因此和[[廣島市|廣島市]]成為世界迄今僅有的兩個被核武器攻擊的城市。

===戰後長崎===
[[File:Cityscape_of_Nagasaki,_Nagasaki_Prefecture,_island_of_Kyushu,_Japan..jpg|thumb]]
長崎市在戰後進行土地區劃整理，開始復興進程，並在1949年制定了長崎國際文化都市建設法<ref>{{cite web |url=http://law.e-gov.go.jp/htmldata/S24/S24HO220.html　 |script-title=ja:長崎国際文化都市建設法 |accessdate=2017-05-08 |publisher={{lang|ja|法令データ提供システム}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。長崎市復興計劃方針的特點之一是以20萬人口為復興目標。由於戰前長崎市的人口已超過25萬，因此這一人口目標低於戰前人口最高值，可說是極為罕見。這一情況的原因之一是長崎市受到了戰爭的極大毀壞，亦反映出多山的地形極大限制了長崎的發展<ref name="Chishi"></ref>{{rp|278}}。1955年，長崎市在原子彈爆炸地點附近修建和平祈念像，象徵長崎市決意成為和平都市<ref>{{cite web |url=http://nagasakipeace.jp/japanese/map/zone_negai/heiwa_kinenzo.html　 |script-title=ja:平和祈念像 |accessdate=2017-05-08 |publisher={{lang|ja|長崎原爆資料館}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。浦上地區是在原子彈爆炸中受害最嚴重的地區。戰後這一地區的兵工廠用地改為[[長崎大學|長崎大學]]的校園，由工業區變為文教區<ref name="Hyakunen">{{cite book |author={{lang|ja|平岡昭利}}|script-title=ja:《九州 地図で読む百年》 |year=1997 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=4-7722-1665-0|language=ja}}</ref>{{rp|65}}。1977年，兩名犯人劫持了一輛自平戶市開往長崎市的公車，並在長崎市內被發現後和警方對峙，最後一人被射殺另一人被逮捕後判刑，是為[[長崎公車劫持事件|長崎公車劫持事件]]<ref name="Jiken">{{cite web |fix-attempted=yes }}</ref>。1981年，教皇[[若望·保祿二世|若望·保祿二世]]訪問長崎，並倡議廢除核武器<ref>{{cite web |url=http://www.seibonokishi-sha.or.jp/jml/pope-and-mather　 |script-title=ja:ローマ教皇とマザーテレサ |accessdate=2017-05-08 |url=http://www.city.nagasaki.lg.jp/syokai/750000/757000/p007005.html#syouwa　 |script-title=ja:歴史年表　昭和時代 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja |deadurl=no }}</ref>。長崎市在1990年代之後發生過兩次市長被槍擊事件。1990年，時任長崎市長[[本島等|本島等]]被極右團體{{link-ja|長崎市長槍擊事件|長崎市長銃撃事件|槍擊}}，幸而留得一命。但2007年，時任長崎市長[[伊藤一長|伊藤一長]]在被[[山口組|山口組]]{{link-ja|長崎市長槍殺事件|長崎市長射殺事件|槍擊}}之後搶救無效過世。兩起事件皆引發社會轟動<ref == 地理 ==
[[File:Nagasaki_Hashima_01.png|thumb]]
長崎市的轄區範圍包括[[長崎半島|長崎半島]]和[[西彼杵半島|西彼杵半島]]的部分地區，市中心地區位於兩個半島的交界處<ref name="Chishi"></ref>{{rp|274}}。長崎市地形多山，市中心地區更是三面環山。市內最高峰是八郎岳，標高590米<ref name="Geography">{{cite web |url=http://www.city.nagasaki.lg.jp/sumai/660000/667000/p004205_d/fil/1-1.pdf　 |script-title=ja:景観特性と課題 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。多山的地形嚴重限制了長崎的發展。例如長崎國際文化都市建設法原本規定長崎市不得在標高50米以上的地區興建住宅，但由於戰後人口快速增加導致住宅無秩序蔓延，這一法規也形同虛設，後被廢除<ref name="Chishi"></ref>{{rp|283}}。長崎市內的女之都、三原、本原、西山、片淵、小島、稻佐、小江原、西町、滑石等住宅區都修建在山坡上。長崎市的街道寬度亦因地形限制而普遍狹窄，但仍被部分市民認為過寬，甚至有修建之後缩窄的例子。這也導致長崎市交通阻塞問題嚴重<ref name="Chishi"></ref>{{rp|283}}。但多山的地形也使得長崎港內最大水深達45米，利於船舶停靠，是一座天然良港<ref>{{cite web |url=https://www.env.go.jp/water/heisa/heisa_net/waters/nagasakiwan.html　 |script-title=ja:長崎湾 |accessdate=2017-05-08 |publisher={{lang|ja|環境省}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。為獲得平地，長崎市自近代以來多次通過[[填海|填海]]取得建設用地<ref>{{cite web |url=http://library.jsce.or.jp/jsce/open/00044/1992/12-0295.pdf　 |script-title=ja:長崎港の埋立と近代都市の形成 |accessdate=2017-05-08 |publisher={{lang|ja|土木史研究}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。長崎市中心的主要河川有{{link-ja|浦上川|浦上川}}和[[中島川|中島川]]<ref>{{cite web |url=https://www.pref.nagasaki.jp/shared/uploads/2015/02/1423807479.pdf　 |script-title=ja:浦上川水系河川整備計画 |accessdate=2017-05-08 |publisher={{lang|ja|長崎県}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。長崎市中心的主要平地也多位於這兩條河流的沿岸和河口的填海地<ref name="Geography"></ref>。在[[市町村合併|平成大合併]]之後，{{link-ja|伊王島町|伊王島町|伊王島}}、{{link-ja|高島 (長崎縣長崎市)|高島 (長崎県長崎市)|高島}}、[[池島_(長崎縣)|池島]]、[[端島|端島]]等離島被合併入長崎市，現在這些島嶼中不少是長崎重要觀光景點<ref>{{cite web |url=http://www.city.nagasaki.lg.jp/syokai/730000/732000/p028291_d/fil/gappeinokennsyou.pdf　 |script-title=ja:合併の検証 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。

長崎多山的地形使得長崎以成為集約型多核都市為發展目標，在發展市中心地區的同時，亦重視發展市郊的據點的地區，並通過改善市中心和各市郊據點之間的交通來提高都市效率<ref>{{cite web |url=http://www.city.nagasaki.lg.jp/sumai/650000/651000/p024170_d/fil/gaiyou.pdf　 |script-title=ja:長崎市の都市計画の概要 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。

=== 氣候 ===
長崎市屬於海洋性氣候。對馬暖流在長崎市附近海域北上。受洋流的影響，長崎市的溫差變化較小。長崎在7月中旬至9月上旬的最高氣溫超過30℃，但很少超過35℃。另一方面，即使是夜間氣溫也下降幅度較少，7月下旬至8月中旬在夜間氣溫也經常在25℃以上。長崎的冬季較為溫暖，很少降雪。但在東海海上季風吹襲時會有大雪<ref>{{cite web |url=http://www.tabirai.net/sightseeing/nagasaki/info/about/weather.aspx　|script-title=ja:長崎県の気候  |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja}}</ref>。近年長崎有數次大雪記錄。2016年1月24日，長崎市的積雪量達18厘米，創下自有觀測史以來的最深紀錄<ref>{{cite web |url=http://www.jma-net.go.jp/fukuoka/chosa/saigai/20160127_nagasaki.pdf　|script-title=ja:平成28年1月23日から25日にかけての長崎県の大雪と低温について  |accessdate=2017-05-08 |publisher={{lang|ja|長崎地方気象台}} |language=ja}}</ref>。長崎的降水日數也比較多<ref name="Gaiyou">{{cite web|url=http://www.city.nagasaki.lg.jp/shimin/170000/176000/p003973_d/fil/03dai1bu.pdf　|script-title=ja:長崎市の概要|accessdate=2017-05-08|publisher={{lang|ja|長崎市}}|language=ja}}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>，降水量以梅雨季節的6、7月最多，其次是颱風頻繁過境的8、9月<ref>{{cite web|url=http://www.city.nagasaki.lg.jp/syokai/750000/754000/p029691_d/fil/01kisyou.pdf　|script-title=ja:気象|accessdate=2017-05-08|publisher={{lang|ja|長崎市}}|language=ja}}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。

== 人口 ==
1889年長崎設市時有人口54,502人，此後隨著都市發展和市轄區擴大，長崎人口快速增加<ref name="Data"></ref>。1920年日本首次人口普查時，長崎市人口有176,534人，不僅是當時九州人口最多城市，也是當時{{link-ja|六大都市|六大都市}}（[[東京市|東京市]]、[[大阪市|大阪市]]、[[名古屋市|名古屋市]]、[[橫濱市|橫濱市]]、[[京都市|京都市]]、[[神戶市|神戶市]]）之外人口最多的城市<ref name="History2"></ref>。但長崎市中心缺乏可供開發的平地這一特徵導致長崎市人口增長在戰前就比較緩慢。1940年人口普查時，長崎市人口有252,630人，不僅比1937年的276,918人減少，還被[[福岡市|福岡市]]、[[八幡市_(福岡縣)|八幡市]]超過，是九州人口第三多城市<ref name="Population1">{{cite web|url=http://www.city.nagasaki.lg.jp/syokai/750000/754000/p026874_d/fil/02.xlsx　|script-title=ja:人口|accessdate=2017-05-08|publisher={{lang|ja|気象庁}}|language=ja}}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。長崎人口在[[第二次世界大戰|第二次世界大戰]]期間大幅減少，特別是原子彈爆炸導致超過7萬人遇難。1945年時，長崎市只有人口142,748人，和前一年相比幾乎減半，且被[[熊本市|熊本市]]超過，成為九州人口第四多城市<ref name="Population1"></ref>。戰後長崎人口數雖快速恢復，但相比九州其他地區仍較緩慢。1975年時，長崎市人口雖達到505,835人的峰值，但此後開始減少。並被[[鹿兒島市|鹿兒島市]]超過，成為九州人口第五多城市<ref name="Population2">{{cite web|url=http://www.city.nagasaki.lg.jp/syokai/730000/731000/p028263_d/fil/jinkoubijyon.pdf　|script-title=ja:長崎市まち・ひと・しごと創生長期人口ビジョン|accessdate=2017-05-08|publisher={{lang|ja|気象庁}}|language=ja}}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。2000年人口普查時，長崎市人口更被[[大分市|大分市]]超過，變為九州人口第六多城市。

2017年4月1日時，長崎市有人口424,066人<ref>{{cite web |url=http://www.city.nagasaki.lg.jp/syokai/750000/751000/p007001.html　 |script-title=ja:長崎市の推計人口 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。長崎市在1960年代之前是人口流入地區。但自1970年代開始，由於重工業景氣蕭條導致長崎變為人口流出地區。加上長崎市區可供開發的平地有限使得一些大型住宅區選擇在鄰近自治體開發，導致長崎市同時出現市內人口向長崎縣內其他自治體流出的現象<ref name="Population2"></ref>。長崎市人口[[老齡化|老齡化]]、[[少子化|少子化]]問題嚴重。由於生育率持續低於置換水準，2000年代之後長崎市人口的自然增長率也變為負增長<ref name="Population2"></ref>。現在65歲以上人口佔長崎市總人口的約28%，15至64歲人口佔約60%，14歲以下人口只佔約12%<ref name="Population2"></ref>。長崎市的[[總和生育率|總和生育率]]只有1.4，略低於日本平均值1.42，更遠低於長崎縣平均值1.66<ref name="Population2"></ref>。現在長崎市已將人口問題視為市政最重要課題之一，試圖通過提振經濟和促進生育以降低人口減少速度<ref name="Population2"></ref>。

=== 第一產業 ===
第一產業佔長崎市經濟總量的0.7%，這一數字只有長崎縣的四分之一，也低於日本平均值1.1%<ref name="Economy"></ref>。受多山的地形限制，長崎市的耕地面積比例不到日本平均值的一半，耕地面積有2,210公頃，農家數則有2,947戶<ref name="Agriculture">{{cite web |url=http://www.machimura.maff.go.jp/machi/contents/42/201/index.html　 |script-title=ja:グラフと統計でみる農林水産業 |accessdate=2017-05-08 |publisher={{lang|ja|農林水産省}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。長崎市2014年農業生產額有79.7億日元，其中以水果類比例最高，其次是蔬菜類和花卉類<ref name="Agriculture"></ref>。長崎縣是日本最大的[[枇杷|枇杷]]產地，而長崎市的茂木地區又是長崎枇杷生產的中心<ref>{{cite web |url=http://nagasaki-gourmet.com/nagasakibiwa.html　 |script-title=ja:長崎びわ |accessdate=2017-05-08 |publisher={{lang|ja|長崎の美味しい食べ物}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。長崎經濟中水產業的比例是日本平均值的3.7倍<ref name="Economy"></ref>，位於長崎市西北部的{{link-ja|長崎漁港|長崎漁港}}是日本[[東海|東海]]沿岸地區最大漁港之一。2015年，長崎市水產類捕撈量達67,012噸，以[[日本竹筴魚|日本竹筴魚]]數量最多，其次是[[鯖魚|鯖魚]]，兩者合計佔過半比例<ref>{{cite web |url=http://www.city.nagasaki.lg.jp/syokai/750000/754000/p029691_d/fil/05suisangyou.pdf　 |script-title=ja:水産業 |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。{{clearleft}}

== 文化 ==
[[File:Castella_002.jpg|thumb]]
長崎市悠久的與西洋、中國交往的歷史使得長崎的文化比起日本其他地方揉合了更多外來元素。{{link-ja|長崎方言|長崎弁}}屬於{{link-ja|肥筑方言|肥筑方言|肥-{筑}-方言}}，和{{link-ja|博多方言|博多弁}}有較多共同特徵。但在音調上屬於{{link-ja|二型式音調|二型式アクセント}}，和博多方言不同<ref name="Dialect">{{cite book |author={{lang|ja|真田信治}}|script-title=ja:《方言学》 |year=2011 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-51524-4|language=ja}}</ref>{{rp|105}}。長崎市出身的著名作家有以紀實文學著稱的{{link-ja|立花隆|立花隆}}<ref>{{cite web |url=http://www.lib.pref.nagasaki.jp/library/pdf/pro14.pdf |script-title=ja:立花隆 |accessdate=2017-05-08 |publisher={{lang|ja|長崎県立長崎図書館}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20130923220130/http://www.lib.pref.nagasaki.jp/library/pdf/pro14.pdf |archivedate=2013-09-23 }}</ref>
、身兼小說家和歌手等多重身份的{{link-ja|佐田雅志|さだまさし}}<ref>{{cite web |url=http://www.lib.pref.nagasaki.jp/library/pdf/pro29.pdf |script-title=ja:さだまさし |accessdate=2017-05-08 |publisher={{lang|ja|長崎県立長崎図書館}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20160320004303/http://www.lib.pref.nagasaki.jp/library/pdf/pro29.pdf |archivedate=2016-03-20 }}</ref>、日裔英國人作家，2017年[[諾貝爾文學獎|諾貝爾文學獎]]得主[[石黑一雄|石黑一雄]]等人<ref>{{cite web |url=http://www.nagasaki-np.co.jp/news/mizusora/2011/10/12090213.shtml |script-title=ja:カズオ・イシグロ |accessdate=2017-05-22 |publisher={{lang|ja|長崎新聞}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20171007021629/http://www.nagasaki-np.co.jp/news/mizusora/2011/10/12090213.shtml |archivedate=2017-10-07 }}</ref>。文豪[[遠藤周作|遠藤周作]]的鉅作《沉默》以長崎為舞台。現在長崎市內有遠藤周作文學館，收藏展示與他有關的珍貴資料<ref>{{cite web |url=http://www.city.nagasaki.lg.jp/endou/facility/ |script-title=ja:施設紹介 |accessdate=2017-05-08 |publisher={{lang|ja|遠藤周作文学館}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170504170904/http://www.city.nagasaki.lg.jp/endou/facility/ |archivedate=2017-05-04 }}</ref>。長崎市還是著名歌劇[[蝴蝶夫人|蝴蝶夫人]]的故事舞台之一<ref>{{cite web |url=http://www.city.nagasaki.lg.jp/nagazine/hakken0411/ |script-title=ja:オペラ蝶々夫人が知りたい |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170606011313/http://www.city.nagasaki.lg.jp/nagazine/hakken0411/ |archivedate=2017-06-06 }}</ref>。

[[端島|端島]]是長崎市的離島之一，因其外觀酷似軍艦而又名軍艦島。端島曾是海底煤礦，島上在人口最多時居住有多達5,267人，人口密度高居世界之冠<ref>{{cite web |url=http://www.gunkanjima-cruise.jp/about.html　 |script-title=ja:軍艦島の歴史 |accessdate=2017-05-08 |publisher={{lang|ja|軍艦島上陸クルーズ}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。島上還建有日本首個鋼筋混凝土公寓<ref>{{cite web |url=http://www.mlab.ne.jp/columns/news_life_20130716/　 |script-title=ja:日本最古のマンション？ “軍艦島”に学ぶマンションライフ |accessdate=2017-05-08 |publisher={{lang|ja|マンション・ラボ}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。在煤礦關閉之後該島變成無人島，現在則以廢墟觀光而著稱。{{link-ja|長崎帆船祭|長崎帆船まつり}}開始於2000年，集合了世界眾多知名帆船，最初是日本荷蘭交流400週年紀念活動的一部分，現在已成為長崎春季知名的觀光活動<ref>{{cite web |url=http://www.at-nagasaki.jp/festival/hansen/ |script-title=ja:長崎帆船まつり |accessdate=2017-05-08 |publisher={{lang|ja|長崎市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170620014158/http://www.at-nagasaki.jp/festival/hansen/ |archivedate=2017-06-20 }}</ref>。
