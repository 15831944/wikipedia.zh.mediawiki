'''长葛市'''为[[中国|中国]][[河南省|河南省]]下辖[[县级市|县级市]]，由[[许昌市|许昌地级市]]代管。长葛市位于河南省中部，辖境南与[[建安区|建安区]]接壤，西与[[禹州市|禹州市]]毗邻，北与[[新郑市|新郑市]]、[[尉氏县|尉氏县]]交界，东与[[鄢陵县|鄢陵县]]相连。面积650平方公里，人口77.7万人（2018年）。全市辖4个街道、12个镇<ref>{{cite web|url=http://changge.gov.cn/zjzg/016001/20091214/e526b8a7-5817-49f5-9a70-50359fa5e237.html|title=长葛概况|accessdate=2018-04-21|author=|date=|publisher=}}</ref>。

==历史==

相传这里是上古时[[葛天氏|葛天氏]]的故地，后人便命名为长葛。
*[[春秋|春秋]]时属[[鄭國_(諸侯國)|郑国]]，[[郑庄公|郑庄公]]“掘地见母”就在此处。
*[[秦|秦]]置'''长社县'''。
*[[南北朝|南北朝]]时改设'''颍川县'''
*[[隋代|隋代]]再改为'''长葛县'''
*[[唐|唐]]又称长社
*[[宋朝|宋朝]]初，属京西路许州。熙宁五年（1072年），属京西北路许州。元丰三年（1080年），升许州为颍昌府，长葛属颍昌府。
*[[元朝|元朝]]，属汴梁路许州。
*[[明朝|明朝]]，属开封府许州。
*[[清朝|清朝]]，属河南省开归陈许郑道许州直隶州。
*[[中华民国|中华民国]]2年（1913年），废府存道，长葛属开归陈许郑道，3月1日，开归陈许郑道改称豫东道，翌年改称开封道。民国16年（1927年）废道，长葛属中区第二行政区。民国21年（1932年），河南设行政督察区，长葛属河南第一行政督察区。
*[[中华人民共和国|中华人民共和国]]建立后，长葛属许昌专区。1969年改专区为地区，长葛属许昌地区。1986年2月，许昌地区撤消，许昌市升为省辖市，长葛属之。1993年12月14日'''撤县设市'''。

=== 名人 ===
长葛三国之前名人较多，[[东汉|东汉]]末年的名士[[钟繇|钟繇]]、[[钟会|钟会]]、[[徐庶|徐庶]]和《[[诗品|诗品]]》的作者[[钟嵘|钟嵘]]、梁上君子典故的[[陈寔|陈寔]]等人都出生于此。

*[[范守己|范守己]]，名守己，字介儒，今[[古桥乡|古桥乡]]固贤村人。[[明神宗|明神宗]]年间名人，官至[[兵部侍郎|兵部侍郎]]，晚年升任[[太仆卿|太仆卿]]，总理钦天监。他发现了观测打春时辰的方法，推算了打春时辰的变化规律，为华夏历法做出贡献。著有《御龙子集》、《参两通极》、《天官举正》等书，流传于世。
*[[黄杰|黄杰]]（1430年-1496年），字士英，今[[古桥乡|古桥乡]]黄岗村人。1494年升[[户部左待郎|户部左待郎]]，督理太仓粮储。
*[[张蔚蓝|张蔚蓝]]，字霁村，今[[老城镇|老城镇]]岗张村人。1903年中举，清朝[[度支部|度支部]]主事。
*[[杨佩璋|杨佩璋]]（1850-1920），字筱村，今[[后河镇|后河镇]]后河村人。[[内阁学士|内阁学士]]兼[[礼部侍郎|礼部侍郎]]衔，1903年署[[督察院|督察院]]副都御史。<ref>{{Cite web |url=http://www.changge.gov.cn/cgcms/jsp/column/zjcg/zjcg.jsp?columnid=26 |title=历史人物 |access-date=2012-09-29 |archive-url=https://web.archive.org/web/20120315195934/http://www.changge.gov.cn/cgcms/jsp/column/zjcg/zjcg.jsp?columnid=26 |archive-date=2012-03-15 |dead-url=yes }}</ref>

== 地理 ==
[[File:ChinaXuchangChangge.png|thumb]]
=== 区域位置 ===
长葛位于河南省中部，许昌市北部。北纬34°09′~34°20′，东纬113°34′~114°08′。

北靠[[新郑市|新郑市]]，间距24公里；西连[[禹州市|禹州市]]，间距28公里；东北接[[尉氏县|尉氏县]]，间距63公里；东南邻[[鄢陵县|鄢陵县]]，间距44公里。市区北距[[郑州|郑州]]62公里。

市境南北21.4公里，东西51.9公里。

=== 地形 ===
地处豫西丘陵向豫东平原的过渡地带，虽邻近[[黄河|黄河]]，但属于[[淮河|淮河]]流域。

西北高，东南低，呈缓倾斜状。地形大体分为浅山区、岗丘区和平原区三部分：
#浅山区：市西北部，是[[伏牛山|伏牛山]]系[[嵩山|嵩山]]余脉，总面积16.2平方公里，占全市面积的2.5%，包括后河镇的西樊楼、芝芳、陉山、山孔、榆林、洼李、山头高、娄庄8个村；
#岗丘区：分布于和尚桥镇，长社路街道，石固镇，官亭乡和大周镇。总面积66平方公里，占总面积的10.2%；
#平原区：除浅山区和岗丘区外的皆为平原区，总面积566.4平方公里，占市面积的87.3%。[[京广铁路|京广铁路]]以西，为'''山前洪积平原'''地貌，是由山前洪积冲积扇和坡积裙汇合而成的冲积平原；京广铁路以东为'''冲积平原'''地貌，是由[[双洎河|双洎河]]和[[黄河|黄河]]泛滥冲积而成的冲积平原。

=== 土地使用 ===
全市土地总面积为650平方公里，其中市区占25平方公里：
#农用地：49501.7公顷，占总面积的77.9%，其中耕地45024公顷，园地505.3公顷，林地770.2公顷，其它农用地3202.1公顷；
#建设用地：12460.3公顷，占总面积的19.6%，其中居民及工矿用地11667.9公顷，交通用地581.4公顷，水利设施用地211公顷；
#未利用地：1615.6公顷，占总面积2.5%。

=== 气候 ===
全市年平均气温14℃。

年平均降雨量675毫米，一般6-9月份降水量占全年降水量的60%-80%。

市水资源总量为42991.63万平方米，可利用水资源量为21991.53万立方米，人均水资源占有量209立方米。

== 区划人口 ==
全市分为4个街道、10个镇、2个乡，共计360个行政村，包括2609个村民小组。

#4街道：金桥路、长兴路、建设路、长社路街道位于市区；
#12镇：和尚桥、后河、石固、坡胡、老城、大周、董村、南席镇、古桥镇、石象镇、增福庙、官亭；

目前，长葛市政府驻地为[[和尚桥镇|和尚桥镇]]建设路街道八七路中段，市政府正计划将政府驻地迁至'''行政新区'''，新区位于和尚桥镇以'''坡杨村'''为中心的区域，在现驻地东南方。与南邻的许昌市新成立的'''[[许昌新区|许昌新区]]'''衔接，共同打造地区新的增长极，包括'''中原电气谷'''等多个项目。

2008年，市总人口70.61万人，男性36.70万人，占总人口51.9%，女性33.91万人，占总人口48.1%。人口自然增长率4.39‰，人口密度为每平方公里有1034人。市区人口约16.4万人。

汉族占绝大多数，除汉族外，2008年有回、维吾尔等少数民族24个，共9469人。有伊斯兰教、基督教、天主教、佛教和道教等活动场所64处，信教群众近20000人。

== 交通 ==
[[京广铁路|京广铁路]]、[[京珠高速公路|京珠高速公路]]、[[107国道|107国道]]纵贯南北，彭（店）一花（石）公路横穿东西。途经高速有G4和G107。

境内市乡公路四通八达，村村通油路。

==经济==
长葛市为较具实力的[[县级行政区|县（市）]]之一，2009年GDP总量为2,207,533万元（[[人民币汇率|折合323,164万美元]]），[[河南省各县市国内生产总值列表|位居河南各县（市）第十五位]]；人均GDP为32,768元（合4,797美元），[[河南省各县市国内生产总值列表|列第二十位]]。2017年GDP总量为600亿元。

=== 农业 ===
全市耕地面积68万亩，粮食种植面积在55万亩左右，粮食总产过52万吨。农业主产[[小麦|小麦]]、[[玉米|玉米]]、生猪、[[泡桐|泡桐]]、[[烟草|烟草]]、[[蔬菜|蔬菜]]等，特产葛根、[[樱桃|樱桃]]、长葛绒和各种蜂类产品。

=== 副业 ===
农业产业化已形成了四大产业链：
#'''肉食品加工'''产业，拥有各类加工企业30多家，年产值42亿元，年加工生猪达250万头，成为全国生猪调出大县；
#'''粮食加工'''产业，拥有各类加工企业20多家，年加工原粮达30万吨；
#'''蜂产品加工'''产业链，拥有各类企业40家，年产量达1.1万吨，[[蜂蜡|蜂蜡]]出口量和拥有量居全国第一，是长江以北最大的蜂产品集散地和全国三大蜂品加工基地之一；
#'''板材加工'''产业，拥有各类企业55家，年产各类板材1600万标准张，是全国三大[[人造板|人造板]]基地之一<ref>{{Cite web |url=http://www.changge.gov.cn/cgcms/jsp/column/zjcg/zjcg.jsp?columnid=19 |title=长葛概况 |access-date=2011-07-11 |archive-url=https://web.archive.org/web/20110903010714/http://www.changge.gov.cn/cgcms/jsp/column/zjcg/zjcg.jsp?columnid=19 |archive-date=2011-09-03 |dead-url=yes }}</ref>。

=== 工业 ===
境内[[石英|石英]]砂和[[磁铁|磁铁]]矿的储量较大。

工业经济初步形成了[[金刚石|金刚石]]制造及制品(生产规模亚洲第一、世界三强)、机械装备制造(三轮汽车年产销量居全国第三)、电瓷电器(高低压电瓷电器业技术水平居全国同行业领先位置)、建筑卫生陶瓷(年产4000万件，为中国中部卫浴产业基地)、有色金属加工(长江以北规模最大)、农副产品加工、纺织服装和人造板材加工八大产业群。

规划面积14平方公里的市产业集聚区为省级产业集聚区，规划面积11.6平方公里的大周再生金属循环经济产业园被列为国家循环经济试点单位和省级专业园区。但由于金属产业大多以民营企业为主，未能形成合理监管，与之而来的环境问题日益突出。

全市工业企业达2000多家，其中限额以上企业331家，拥有[[黄河旋风|黄河旋风]]、[[众品食业|众品食业]]、[[森源电气|森源电气]]三家上市公司。

=== 商业 ===
各类专业市场总数62个，市钢材交易市场、九鼎美达物流园、金桥农机商贸市场、石固闲置设备调剂市场、大周有色金属市场等在省内外有较高声誉。

== 教育 ==
长葛现有四所普通高中：分别为[[长葛一高|长葛一高]]（许昌市重点高中、河南省示范性高中）、[[长葛二高|长葛二高]]（[[河南省示范性高中|河南省示范性高中]]）、[[长葛三高|长葛三高]]和[[长葛实验中学|长葛实验中学]]（民办中学，设有初中部和高中部）。

== 文化 ==
位于长葛[[石固镇|石固镇]]的[[石固遗址|石固遗址]]是[[新石器时代|新石器时代]]的古遗址，2006年被认定为“全国第六批重点文物保护单位”。石固遗址是继[[新郑|新郑]][[裴李岗遗址|裴李岗遗址]]之后，发现的又一具有重要学术价值的裴李岗文化遗址。遗址包含[[裴李岗文化|裴李岗文化]]和[[仰韶文化|仰韶文化]]两个时期的古文化遗址，距今约4500－7400年<ref>{{cite web |url=http://www.changge.gov.cn/cgcms/sites/cgwhw/preview.jsp?TID=20101001143455266766637 |language= |title=石固遗址 |publisher=葛天文化网 |date=2010-10-01 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20160305060623/http://changge.gov.cn/cgcms/sites/cgwhw/preview.jsp?tid=20101001143455266766637 |archivedate=2016年3月5日 |df= }}</ref>。

2013年5月，“[[十二连城|十二连城]]”古遗址与“[[禅静寺造像碑|禅静寺造像碑]]”石刻被列入第七批[[全国重点文物保护单位|全国重点文物保护单位]]。目前长葛共有各级文物保护单位77处，其中3处国家级，5处省级，13处许昌地市级，56处长葛市级。<ref>{{cite web |url=http://www.changge.so/news/7197.htm |language= |title=长葛市新增2处国家级文物保护单位 |publisher=长葛都市网 |date=2013年5月6日 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20141129054851/http://www.changge.so/news/7197.htm |archivedate=2014年11月29日 }}</ref>
