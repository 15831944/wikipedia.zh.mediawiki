'''长春广播电视台'''（{{lang-en|'''C'''hang'''C'''hun '''M'''edia '''G'''roup，'''CCMG'''}}）是[[中华人民共和国|中华人民共和国]][[吉林省|吉林省]][[长春市|长春市]]的地市级[[电视台|电视台]]与广播电台，创立于2012年11月，由原长春人民广播电台、长春电视台等单位的基础上组建而成，为正局级事业单位。<ref>{{Cite web|url = http://www.chinactv.com/folder14/folder91/2014-02-24/213.html|title = 长春广播电视台简介|publisher = 玉米网|accessdate = 2014-02-24|archive-url = https://web.archive.org/web/20161220082239/http://www.chinactv.com/folder14/folder91/2014-02-24/213.html|archive-date = 2016-12-20|dead-url = yes}}</ref>

== 历史与发展 ==
1946年4月18日，长春人民广播电台开播。

1985年8月，长春电视台开播。<ref>{{Cite web |url = http://www.admaimai.com/tv/tvDetail1_400.htm |title = 长春电视台简介 |publisher = 广告买卖网 |accessdate = 2014-12-04  }}</ref>

1993年，长春有线电视台开播，开播后，播出自办节目。

2001年7月，经国家广播电视总局决定，长春有线电视台与长春电视台进行合并。整合后的长春电视台有两台电视频道播出，进行了重新规划。

2004年，长春电视台已经有五套电视频道播出，每天为电视观众提供100小时电视节目，形成系新化新的播出体系。

2012年11月，原长春人民广播电台、长春电视台等单位合并，组建长春广播电视台。
