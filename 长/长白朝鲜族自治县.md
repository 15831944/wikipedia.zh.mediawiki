'''长白朝鲜族自治县'''（{{朝|'''장백조선족자치현'''|長白朝鮮族自治縣|罗=Changbaek Chosǒnjok Chach'ihyǒn}}），[[中華人民共和國|中華人民共和國]][[吉林省|吉林省]][[白山市|白山市]]下辖的一个[[民族自治县|民族自治县]]，也是除了[[延边朝鲜族自治州|延边朝鲜族自治州]]以外，全国唯一的[[朝鲜族|朝鲜族]][[自治县|自治县]]。面积2497.6平方公里，人口8.5万人， 其中[[朝鲜族|朝鲜族]]人口1.4万。邮政编码134400。自治县人民政府驻长白镇。

== 地理 ==
位于吉林省东南部、[[长白山|长白山]]南麓、[[鸭绿江|鸭绿江]]上游。东南与[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]][[两江道|两江道]][[惠山市|惠山市]]和[[三池渊郡|三池渊郡]]、[[普天郡|普天郡]]、[[三水郡|三水郡]]、[[金正淑郡|金正淑郡]]、[[金亨稷郡|金亨稷郡]]隔江相望，国境线总长260.5公里。西与白山[[临江市|临江市]]接壤，北与白山[[抚松县|抚松县]]交界。

== 历史 ==
*1677年[[清朝|清朝]]封禁此地，不再有居民；1875年清政府解除禁令，实行开发政策。1877年清政府设[[通化县|通化县]]，管辖该地。
*清光绪三十四年（1908年）九月十一日设长白府设治，属奉天省；将临江县以东的长生堡、庆生堡划归长白府。
*1949年长白县划归[[辽东省|辽东省]][[通化|通化]]专署，1964年划归吉林省通化专区。
*1912年属奉天省东边道，1913年改长白府为长白县，属奉天省。
*1958年9月15日成立长白朝鲜族自治县，属吉林省通化行署。1985年4月归[[浑江市|浑江市]]（现白山市）管辖。

==人口==
全縣總人口共8.6萬人，由[[漢族|漢族]]、[[朝鮮族|朝鮮族]]、[[滿族|滿族]]等民族組成，其中朝鮮族共有1.4萬人，佔縣總人口總數的16.9%。<ref>[http://www.seac.gov.cn/art/2012/7/13/art_1717_25979.html 長白朝鮮族自治縣－國家民族事務委員會]</ref><ref>[http://jilin.mofcom.gov.cn/article/sjdixiansw/201303/20130300063960.shtml 長白朝鮮族自治縣投資環境簡介－吉林商務之窗]</ref>

== 经济 ==
[[林业|林业]]、水电业、矿产业、[[人参|人参]]业，医药保健品业比较发达，2004年实现[[国内生产总值|国内生产总值]]10亿元，人均GDP超过1万元。农村人均纯收入2779元，城镇居民人均可支配收入5094元。

== 旅游 ==
素有「长白林海、人参之乡」，[[天池|天池]]南坡景观等风景区较多。干沟子古墓群属于[[春秋战国|春秋战国]]时期。

赴朝边境旅游项目，主要游览朝鲜天池东坡景观、[[金日成|金日成]]革命历史遗址（白头山密营、[[普天堡|普天堡]]战迹地、普天堡战斗胜利纪念塔、两江道事迹馆）和两江道艺术大学专场文艺演出。
