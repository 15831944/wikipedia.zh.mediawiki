'''长城煤矿旧址'''，位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[抚宁县|抚宁县]][[石门寨镇|石门寨镇]]上庄坨村，是[[秦皇岛市文物保护单位|秦皇岛市文物保护单位]]。

==简介==
《抚宁县志》载：1916年当地人刘汝林、齐翰卿等人在上庄坨村创办兴业煤矿公司。1938年1月29日，[[日本|日本]]人占领煤矿，直到1945年[[日本投降|日本投降]]，这8年期间产煤72.8万吨，大部分运往日本<ref name=jiuzhi>{{Cite web |url =http://news.163.com/05/0406/16/1GLU6H070001124T.html  |title = 河北省抚宁县发现日军侵华时期开办的矿山旧址 |author =  |date =2005-04-06  |publisher =网易 |language =  |accessdate =  |quote =  }}</ref>。

2005年3月底，上庄坨村拆旧房子，村民在村北一处旧房后面发现一个已被封堵多年的旧大门，经抚宁县文史研究人员考证，大门及其后面的废墟是1938年[[日军|日军]]占领该地后开办的长城煤矿。大门右侧墙上竖刻有“长城炭矿铁道株式会社长城矿务所”，大门后面是个面积上千平方米的废弃煤矿场区，仍留有多处日本风格建筑<ref name=jiuzhi/>。

2005年，75岁的上庄坨村村民杨保功说，当年日军占领煤矿后雇村民开采，将质量好的煤从[[秦皇岛港|秦皇岛港]]运往日本。他回忆称，当时日本人对矿工有不成文规定：“入井三分险，事故不可免，如果有天灾，出事也不管。”当时常发生坍塌、透水事故，每次都死数十人，杨保功的姐夫就死于煤矿事故。日本人只要见到矿工休息就用鞭子或镐把殴打，有时用豆饼、花生渣代替工钱发给工人。发生矿井事故便将事故井口封死，用挖煤器具将幸存者打死灭口。杨保功说，因为矿工幸存者很少，矿山也被废弃，所以这段历史后来连当地人都已不知道了<ref name=jiuzhi/>。

长城煤矿旧址已被列为秦皇岛市文物保护单位<ref>{{Cite web |url = http://www.xinhuanet.com/local/2015-03/06/c_1114552548.htm |title =河北秦皇岛启动港口近代工业遗产保护利用  |author =  |date = 2015-03-06 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。
