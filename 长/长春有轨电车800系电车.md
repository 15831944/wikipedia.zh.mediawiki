'''长春有轨电车800系电车'''是[[长春轨道交通|长春轨道交通]]的电车车款之一，现已全部退役。

== 简介 ==
800系电车由[[湘潭电机厂|湘潭电机厂]]于2000年生产，单节编组，共10辆。车门为外摆式门和内摆式门，为高地板车辆。曾经于54路运行，是[[长春有轨电车|长春有轨电车]]第八代电车。

== 列车配置 ==
800系分为2种外涂装，分别是绿色（801-804、806-807、809-810）和红色（805 808），IGBT斩波调压器由沈阳新阳光生产。

== 现在情况 ==
由于800系电车在2010年后保养不善，导致使用一段时间后产生了较多问题。在2011年，已开始逐步退役800系电车，现存的此型车已于2014年退役，大部分已被拆毁。<ref>{{cite web |url=http://114.215.137.56:802/files/BulletinThird.aspx?BulletinId=55 |title=存档副本 |accessdate=2017-04-18 |deadurl=yes |archiveurl=https://archive.is/20140730013004/http://114.215.137.56:802/files/BulletinThird.aspx?BulletinId=55 |archivedate=2014-07-30 }}</ref>。
