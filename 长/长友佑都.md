'''长友佑都'''（{{jpn|j=長友 佑都|hg=ながとも ゆうと|rm=Nagatomo Yūto|f=y}}，{{bd|1986年|9月12日||}}），出生于[[爱媛县|爱媛县]][[西条市|西条市]]，[[日本|日本]]足球运动员，司职边后卫，现效力於[[土超|土超]]球会[[加拉塔沙雷體育會|加拉塔沙雷]]，[[日本国家足球队|日本国家足球队]]成员，妻子為女演員[[平愛梨|平愛梨]]。

== 俱乐部生涯 ==
1986年9月12日，長友佑都出生于日本愛媛縣西條市，他是家裏的第二個孩子。進入高中之後，他開始在福岡中學校隊參加比賽。長友高三時和球隊一起參加了全國高中聯賽。長友的學習成績一貫出色，最終20歲的他考取了位于日本東京的[[明治大學|明治大學]]。進入大學後長友堅持參加足球比賽，並在第一年的大學生聯賽結束後入選了最佳11人陣容，得以代表日本大學生隊參加2007年在泰國曼谷舉行的[[2007年夏季世界大學生運動會|曼谷大學生運動會]]。<ref name="asdf">[http://sports.163.com/11/1226/17/7M7H56MP00051CD5.html 人物志:從啦啦隊員到豪門主力 長友佑都奇迹之路],網易體育</ref>

21歲時，長友佑都終于獲得了職業球隊的合同，在[[FC東京|FC東京]]開始職業足球生涯。<ref name="asdf"/>2009年9月，曾有媒體報導稱[[蘇超|蘇超]]球隊[[凱爾特人足球俱樂部|凱爾特人]]在[[中村俊輔|中村俊輔]]的推薦下，考慮引進長友佑都<ref>[http://sports.163.com/09/0915/00/5J7BRMC500051CCO.html 蘇媒曝凱爾特人最新引援 鄭智將再迎東亞隊友？],網易體育</ref>，但長友佑都最終仍然留在FC東京。

2010年7月，長友佑都租借加盟[[意大利足球甲級聯賽|意大利足球甲級聯賽]]升班馬[[切塞納足球俱樂部|切塞納]]。1月31日，[[國際米蘭足球俱樂部|國際米蘭]]宣布租借長友佑都半年，與此同時，[[達維德·桑頓|-{zh-hans:达维德·桑顿;zh-hk:大衛·辛頓;}-]]將會外借至切塞納作為交換<ref>[http://www.inter.it/aas/news/reader?N=51300&L=it Mercato: Yuto Nagatomo è dell'Inter]{{it icon}}</ref>。長友佑都也成為首名加盟國際米蘭的日本球員。2月6日，他在國際米蘭主場5比3擊敗[[羅馬足球俱樂部|羅馬]]的聯賽中替補[[韋斯利·斯內德|-{zh-hans:斯內德;zh-hk:史奈達;zh-tw:史奈德;}-]]出場，完成在國際米蘭的首秀<ref>[http://sports.sohu.com/20110207/n279233871.shtml 百年國米首現日本球員 長友佑都處子秀贏得掌聲],搜狐體育</ref>。3月6日，長友佑都在國際米蘭主場5比2擊敗[[熱那亞足球俱樂部|熱拿亞]]的聯賽中打進他加盟球隊的首個進球。賽季結束後，長友佑都正式轉會加盟國際米蘭，轉會費為650萬歐元。
2013年12月23日，長友佑都在[[米兰德比|米兰德比]]中接替[[坎比亚索|坎比亚索]]的队长袖标，成为[[国际米兰|国际米兰]]百年历史上首位亚洲队长。<ref name="asdf"/>

== 国家队生涯 ==
2008年5月24日，长友佑都在和[[科特迪瓦国家足球队|科特迪瓦]]的友谊赛中第一次代表国家队出场，11月13日，他在和[[叙利亚国家足球队|叙利亚]]的比赛中打进国家队的首个进球。

== 技术特点 ==

作为一名边后卫，长友佑都拥有充沛的体能和强劲的冲刺能力，来回往返奔跑能力极强。同时他的左右脚均为擅长脚，使其可胜任左右边后卫两个位置。长友佑都也拥有较好的技术，譬如2011年9月28日，[[欧洲冠军联赛|欧洲冠军联赛]]小组赛国际米兰客场3:2战胜[[莫斯科中央陆军|莫斯科中央陆军]]的比赛，长友佑都带球到对方右路禁区边缘后，穿裆突破对方的左后卫，并送出妙传帮助队友[[詹保罗·帕齐尼|帕齐尼]]破门得分。
