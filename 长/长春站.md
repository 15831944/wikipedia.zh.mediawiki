'''长春站'''位于[[中国|中国]][[吉林省|吉林省]][[长春市|长春市]][[宽城区|宽城区]]，于1907年建成启用。目前为[[中国铁路沈阳局集团有限公司|中国铁路沈阳局集团有限公司]]直属的客运[[特等站_(中鐵)|特等站]]，位于[[京哈线|京哈铁路]]1003km处。长春站是[[长白铁路|长白铁路]]、[[长图铁路|长图铁路]]、[[长珲城际铁路|长珲城际铁路]]的起点站。

== 车站设施 ==
*北进站口
**12306 “春之约”服务台
*北站房综合换乘中心
*南进站口
*北一售票厅：18个售票窗口（自西向东编号76-92），6台自动售票机（编号7-12），提供蓝色磁卡票。另有数个面向室外的售票窗口，提供蓝色磁卡票。
*南一售票厅：含14个售票窗口（编号61-74），6台自动售票机（编号2-6），提供蓝色磁卡票；4台自动取票机，只办理互联网购票取票业务，提供红色软纸票，编号247-250。
*南二售票厅：由于一楼施工原因整个售票厅暂停使用。已竣工的二楼含14个提供蓝色磁卡票的售票窗口，编号19-32。
*南广场售票厅：提供红色软纸票，现已停用，售票设备移至北一售票厅户外售票窗口。

== 使用情况 ==
长春站是[[京哈铁路|京哈铁路]]、[[哈大铁路|哈大铁路]]、[[长图铁路|长图铁路]]、[[长白铁路|长白铁路]]、[[长珲城际铁路|长珲城际铁路]]、[[哈大客运专线|哈大客运专线]]的客运[[特等站_(中鐵)|特等站]]。每日办理旅客列车332列。<ref>2014年[[2014年5月|5月]]数据</ref>

== 历史 ==
[[File:Hsinking_Station,_Changchun.jpg|thumb]]
[[File:Platform_of_Hsinking_Station,_Hsinking.jpg|thumb]]
[[File:Changchun_Railway_Station_01.jpg|thumb]]
* 1907年：[[日本|日本]]在[[长春市|长春市]]头道沟与二道沟之间设立了「[[满铁附属地|满铁附属地]]」，并在[[宽城子车站|宽城子车站]]南面建成长春站。当时称“长春停车场”，是一座尽头式车站，向西南方引出通往孟家屯、沈阳的铁路，向西北方引出通向宽城子站、哈尔滨的铁路，而向东南的连接长春东站、吉林站的长吉铁路还没有规划建设。
* 1907年11月3日：长春站开设货运业務。
* 1907年12月1日：[[南满铁路|南满铁路]]开通，长春站开办客运业務。
* 1913年3月：由满铁投资32万元，面积4000[[平方米|平方米]]的长春站大楼动工兴建，1914年5月竣工。设计者为[[青木菊治郎|青木菊治郎]]和[[平泽仪平|平泽仪平]]。其前有一个站前圆形广场，直径100间（182米，1间=1.816米）。
* 1992年5月26日：有85年历史的长春站站房以爆破方式拆除。
* 1995年：长春站新站房（即目前的站房）建成启用。
* 2007年，长春站大楼外墙重新装修，瓷砖罩面改为石材罩面
* 2012年9月20日：长春站北出入口改造完毕并投入使用，与长春站长白路候车室共同办理客运业务，同时原南出入口进入施工阶段暂停使用。

==车站组织结构==
长春站设运转、客运、货运、装卸四个车间。车间设有车间主任、党总支书记、车间副主任、管理员、业务指导等。车间下设班，有班主任（正式职名为“值班主任”，九十年代改为“值班站长”）、党支部书记。班下设组。
上述业务部门之外，长春站设办公室、档案室、党办、武装部、教育室、工会、团委、医务室、技术室、安全室、劳资人事室、军代处等科室。
