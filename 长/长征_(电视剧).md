《'''长征'''》，中国大陆于2001年6月播出的以[[长征|长征]]为主题的战争剧，导演[[金韬|金韬]]、[[唐国强|唐国强]]，主演[[唐国强|唐国强]]、[[刘劲|刘劲]]、[[王伍福|王伍福]]、[[陈道明|陈道明]]。<ref>{{cite web |url=http://news.163.com/11/0608/06/760OS5IE00014AED.html |title=长征，贡献给世界的壮丽史诗 |accessdate=2018-11-26|date=2011-06-08 |publisher=网易，来源：新华报业网-新华日报(南京) |work=责任编辑：王晓易_NE0011 |language =zh-cn}}</ref>

== 剧情 ==
{{See also|长征}}
1934年，[[中国共产党|中国共产党]]领导的[[中国工农红军|中国工农红军]]因为[[博古|博古]]、[[李德|李德]]等人的错误领导导致和[[中国国民党|中国国民党]]正面战斗时损失惨重，迫不得已开始向[[陕甘宁边区|陕甘宁边区]]大撤退，在[[贵州|贵州]][[遵义|遵义]]，召开了[[遵义会议|遵义会议]]，确立了毛泽东的领导地位，毛泽东带领中国工农红军历经千辛万苦重重阻碍的长征最终抵达目的地。<ref>{{cite web |url= http://www.hinews.cn/news/system/2011/12/26/013869747.shtml|title=电视剧《长征》 |accessdate=2011 |publisher=南海网 |work= |language =zh-cn}}</ref><ref>{{cite web |url= http://ent.hunantv.com/e/h/20080720/23560.html|title=二十四集电视连续剧《长征》剧情梗概  |accessdate=2008 |publisher=金鹰网 |work= |language =zh-cn}}</ref>

==花絮==

该剧拍摄的原则是“大事不虚、小事不拘”<ref>{{cite web |url=http://www.people.com.cn/h/2011/0611/c25408-3257746949.html |title= 仲呈祥：《长征》具有标志性意义|accessdate=2011年6月11日 |publisher= 人民网|work= |language =zh-cn}}</ref>。

2007年8月31日，该剧在韩国的[[中华电视台|中华电视台]]播出<ref>{{cite web|url=http://www.northnews.cn/2007/0628/14898.shtml|title=中国电视剧《长征》要在韩国播出|accessdate=2007|publisher=正北方网|work=|language=zh-cn}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

剧集播出10年后的2011年，该剧成为[[中国中央电视台|中国中央电视台]]庆祝中国共产党成立九十周年优秀电视剧展播的剧集之一<ref>{{cite web |url= http://tv.people.com.cn/GB/166419/14763451.html|title= 庆祝中国共产党成立九十周年优秀电视剧展播系列报道：《长征》|accessdate=2018-11-26|date=2011-05-28|publisher=人民网，来源：央视网 |work=责任编辑：李佳 |language =zh-cn}}</ref>。
