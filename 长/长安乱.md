《'''长安乱'''》，2004年由中国青年出版社出版发行的小说，作者[[韩寒|韩寒]]，是韩寒用[[解构主义|解构主义]]创作的另类[[武侠小说|武侠小说]]。<ref name=" fish" >{{cite book |author=韩寒 |title=《长安乱》|year=2004年 |publisher=中国青年出版社 |location=北京市 |pages=前言 |isbn= 9787500658917|language = zh-cn}}</ref>

== 内容 ==
古代，武林纷乱不堪，战争频仍，只为争夺“武林盟主”的宝座，朝廷隔山观虎斗，只等各大门派纷纷自相残杀之后收拾残局，一统河山。少林寺5岁的小沙弥释然，聪明伶俐，不喜欢习武，喜欢思考问题，能预知未来。少林寺和武当派一场大战之后，武当派掌门刘云被围住而饿死。

== 角色 ==
*释然，[[少林寺|少林寺]]弟子，聪明伶俐，喜欢思考问题，能预知未来，恋人是喜乐。

*释空，少林寺弟子，释然的师兄，喜欢打斗。

*喜乐，释然的恋人。

*慧竟，少林寺的主持。

*刘云，[[武当派|武当派]]的掌门。

== 风格 ==
小说用[[幽默|幽默]][[讽刺|讽刺]]，尤其是反讽的手法，<ref>{{cite web |url=http://www.people.com.cn/GB/14738/14754/21861/2919629.html |title= 读韩寒的《长安乱》|accessdate= |publisher=人民网 |work= 张斌璐|language =zh-cn}}</ref> 以小沙弥释然从出世的空门到入世的社会，展现了作者对于世界的探究、思考和尝试。<ref name=" fish" />
