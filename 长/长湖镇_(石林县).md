'''长湖镇'''，是[[中华人民共和国|中华人民共和国]][[云南省|云南省]][[昆明市|昆明市]][[石林彝族自治县|石林彝族自治县]]下辖的一个[[乡镇|乡镇]]级行政单位<ref>{{PRC admin/ref sgc|{{PRC admin/parent|53/01/26/106/000}}}}</ref>。面积为134.5平方千米<ref>{{Cite book|title=西部大开发在云南 纪念西部大开发战略实施10周年|last=云南省发展和改革委员会，云南中国西部研究发展促进会编|first=|publisher=昆明：云南人民出版社|year=2011年6月|isbn=978-7-2220-7927-4|location=|pages=717}}</ref>。

== 历史沿革 ==
1988年置维则乡。维则乡名来自驻地维则村。村名来自彝语“维走”的近音译，维为猪，走为交换或抵偿，意为交换猪之地。1949年前光阳洞和燕子山村属陆良县圭山乡，其余各村属路南县圭山乡。1950年至1958年先后属圭山区、圭二区等<ref>{{Cite book|title=路南彝族自治县志|last=昆明市路南彝族自治县志编纂委员会编|first=|publisher=昆明：云南民族出版社|year=1996年11月|isbn=7-5367-1294-4|location=|pages=75}}</ref>。1969年分设维则公社。1984年设维则区。1988年设置维则乡<ref>{{Cite book|title=昆明地名博览辞典|last=吴光范著|first=|publisher=昆明：云南人民出版社|year=2005年1月|isbn=7-222-04154-0|location=|pages=269}}</ref>。2006年3月-4月，原龟山乡的海宜村、乍龙村归入，撤销维则乡设立长湖镇<ref>{{Cite book|title=西南边疆民族研究 第7辑|last=何明主编|first=|publisher=昆明：云南大学出版社|year=2010年6月|isbn=978-7-5482-0151-9|location=|pages=122}}</ref>。

地处石林县境中部，东接[[圭山乡|圭山乡]]、[[亩竹箐乡|亩竹箐乡]]，西连[[鹿阜街道|鹿阜街道]]，南与[[弥勒县|弥勒县]]接壤，北与[[路美邑乡|路美邑乡]]和[[西街口乡|西街口乡]]相邻。政府所在地维则村距石林县城17千米，是全乡政治、经济、文化、交通中心<ref>{{Cite book|title=石林彝族自治县志 1989-2000|last=刘世生主编；石林彝族自治县志编纂委员会编|first=|publisher=昆明：云南民族出版社|year=2006年10月|isbn=7-5367-3448-4|location=|pages=26}}</ref>。

== 行政区划 ==
长湖镇下辖以下地区：维则村、雨胜村、宜政村、舍色村、豆黑村、蓑衣山村、所各邑村、祖莫村、海宜村和乍龙村<ref>{{PRC admin/ref sgc|53/01/26/106/000}}</ref>。

== 产业 ==
最高海拔2071米（窝子山）、最低1880米，地形为典型的[[喀斯特地貌|喀斯特地貌]]，[[石岗岩|石岗岩]]为主等，还有[[长湖|长湖]]、[[圆湖|圆湖]]等盆地。主要种植维小麦、烤烟、柿子等<ref>{{Cite book|title=石林年鉴 2007|last=石林彝族自治县史志办公室编辑|first=|publisher=昆明：云南民族出版社|year=2007年12月|isbn=978-7-5367-3977-2|location=|pages=65}}</ref>。长湖是传说中[[阿诗玛|阿诗玛]]<ref>{{Cite web|url=http://news.china.com.cn/live/2017-06/09/content_38353511.htm|title=少数民族非遗保护：从建章立制到步入“深水区”_新闻中心_中国网|accessdate=2017-07-28|work=news.china.com.cn}}</ref>的故乡<ref>{{Cite book|title=云南 第4版|last=《走遍中国》编辑部编著|first=|publisher=北京：中国旅游出版社|year=2014年1月|isbn=978-7-5032-4875-7|location=|pages=136}}</ref>。

因为当地是[[彝族|彝族]]居住地，每年都有盛况的火把节<ref>{{Cite web|url=http://travel.sina.com.cn/domestic/news/2017-07-17/detail-ifyiakwa4297760.shtml|title=彝乡火苗沸腾 2017年石林火把节开幕_新浪旅游_新浪网|accessdate=2017-07-28|work=travel.sina.com.cn}}</ref>、斗牛节<ref>{{Cite web|url=http://news.163.com/17/0719/11/CPN27BDR00018AOQ.html|title=角力石林深处——云南彝族斗牛速写_网易新闻|accessdate=2017-07-28|last=网易|work=news.163.com}}</ref>。
