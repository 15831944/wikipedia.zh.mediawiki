'''长州藩'''（{{jpn|j=長州藩|hg=ちょうしゅうはん|rm=Chōshū han|f=y}}），歷史可溯至[[日本|日本]][[江户幕府|江户幕府]]时期，位于日本[[本州|本州]]最西（當時的[[周防國|周防國]]和[[長門國|長門國]]）。藩主是[[外樣大名|外樣大名]]・[[毛利氏|毛利氏]]。藩廳設在[[萩城|萩城]]（现[[山口县|山口县]][[萩市|萩市]]），所以又叫'''毛利藩'''或'''萩藩'''。[[幕末|幕末]]，藩廳移轉至周防[[山口市|山口]]的[[山口城|山口城]]（山口政事堂），所以稱'''周防山口藩'''（すおうやまぐちはん）。一般將萩藩・（周防）山口藩時代總稱「長州藩」。

幕末時期，长州藩與[[萨摩藩|萨摩藩]]組成[[萨长同盟|萨长同盟]]，为[[讨幕运动|讨幕运动]]的中心。后来的[[明治维新|明治维新]]时期，长州藩政治家形成了藩阀政治中占主导地位的「長州閥」。時至現代，长州[[山口縣|山口縣]]出身的政治家仍長期把持日本政府。例如[[岸信介|岸信介]]、[[佐藤榮作|佐藤榮作]]、[[安倍晉三|安倍晉三]]等，山口縣出身的[[內閣總理大臣|內閣總理大臣]]達9位，居各[[都道府縣|都道府縣]]之冠。

== 歷史 ==

長州藩距離[[江户|江户]]較遠，[[毛利輝元|毛利輝元]]在[[關原之戰|關原之戰]]中名義上是西軍總大將，幫助[[石田三成|石田三成]]反對[[德川家康|德川家康]]，西軍戰敗后[[毛利輝元|毛利輝元]]被減封，因此歷代藩主與[[德川幕府|德川幕府]]不睦。

=== 幕末 ===

幕末時期，長州藩後來和[[薩摩藩|薩摩藩]]结成[[薩長同盟|薩長同盟]]，共同討伐幕府。長州藩[[奇兵隊|奇兵隊]]是[[明治维新|明治维新]]中新政府軍的主要力量，因此長州藩在此後很長一段時間内掌握日本陸軍，並壓制力挺幕府的[[會津藩|會津藩]]。

幕府末期長州藩的著名思想家[[吉田松陰|吉田松陰]]提出“一君萬民論”，主張天皇之下萬民平等。其門下出現許多[[明治维新|明治维新]]的政治家及軍人。

== 軍隊 ==

長州藩兵是日本最早的西式軍隊，在[[下關戰爭|下關戰爭]]期間曾與[[英國|英]]、[[美國|美]]、[[法國|法]]、[[荷蘭|荷]]等四國交戰。在[[長州征討|長州征討]]之中，長州藩以四千名藩兵擊退十五萬名[[德川幕府|德川幕府]]軍。在[[戊辰戰爭|戊辰戰爭]]，長州藩聯合[[薩摩藩|薩摩藩]]、[[土佐藩|土佐藩]]、[[肥前藩|肥前藩]]組成的新政府軍，對抗德川家。 

[[高杉晉作|高杉晉作]]在1865年在民間建立[[長州藩諸隊|長州藩諸隊]]擊倒俗論派（保守派），成立正義派（倒幕派）。

長州藩諸隊包括[[奇兵隊|奇兵隊]]、報國隊、南園隊等隊伍，成員包括[[武士|武士]]、[[農民|農民]]、[[町人|町人]]等。採用了西式軍制和[[德萊賽針發槍|德萊賽針發槍]]、{{link-en|米尼槍|Minié rifle}}等新式武器，並改革戰術；打破只有[[武士|武士]]才能當兵的規矩，為後來明治維新推行[[徵兵制|徵兵制]]開了先例。

== 著名藩士 ==
由於長州藩在[[討幕運動|討幕運動]]中立下大功，所以[[明治維新|明治維新]]的之中有很多前長州藩士加入新政府，引致形成了日本藩閥政治中佔主導地位的「長州閥」。

[[吉田松陰|吉田松陰]]門下：
* [[伊藤博文|伊藤博文]]：[[日本內閣總理大臣|日本內閣總理大臣]] ，[[朝鮮統監|朝鮮統監]]。日本“明治憲法之父”，任總理大臣期間中日[[甲午战争|甲午战争]]日本勝利，令[[朝鮮|朝鮮]]从清朝獨立，為將朝鮮成為日本的殖民地打好基礎。1909年10月26日於[[哈爾濱|哈爾濱]]車站，被朝鮮[[民族英雄|民族英雄]][[安重根|安重根]]狙殺。日本[[元老_(日本)|元老]]之一。
* [[木户孝允|木户孝允]]（桂小五郎）：[[文部卿|文部卿]]，[[內務卿|內務卿]]。與西鄉隆盛、大久保利通並稱為“維新三傑”。
* [[高杉晉作|高杉晉作]]：[[奇兵隊|奇兵隊]]始創人。
* [[山县有朋|山县有朋]]：三代、九代總理大臣、甲午战争時任[[日本第一軍司令|日本第一軍司令]]，日本元老之一。

其他著名的長州藩出身的名人有：
* [[井上馨|井上馨]]：[[大隈重信|大隈重信]]內閣先後任[[外務大臣|外務大臣]]、[[農商務大臣|農商務大臣]]、第二次伊藤博文內閣時的[[內務大臣|內務大臣]]。從一位大勛位[[侯爵|侯爵]]、元老。
* [[乃木希典|乃木希典]]：日本陸軍上將、第3任[[台灣總督|台灣總督]]。参加过中日甲午战争、[[日俄战争|日俄战争]]。被称为日本陆军的军神。[[明治天皇|明治天皇]]驾崩后切腹殉死。
* [[兒玉源太郎|兒玉源太郎]]：日本陸軍大臣、第4任[[台灣總督|台灣總督]]。参加过中日甲午战争、日俄战争。
* [[寺內正毅|寺內正毅]]：日本陸軍大臣、[[日本內閣總理大臣|日本內閣總理大臣]] ，[[朝鮮總督|朝鮮總督]]。
* [[桂太郎|桂太郎]]：日本陸軍大臣、日本內閣總理大臣、第2任[[台灣總督|台灣總督]]，總理大臣期間日本取得日俄战争勝利。日本元老之一。
* [[佐久間左馬太|佐久間左馬太]]：日本陸軍大將、第5任[[台灣總督|台灣總督]]，任台灣總督期間發動[[五年理蕃計畫|五年理蕃計畫]]消滅反日的台灣原住民。
* [[三浦梧樓|三浦梧樓]]：日本陸軍中將、駐朝鮮全權公使，任駐朝鮮全權公使期間主導[[乙未事變|乙未事變]]，暗殺朝鮮[[明成皇后|明成皇后]]（閔妃）
等等。
