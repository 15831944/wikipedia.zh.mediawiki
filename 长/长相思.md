'''长相思'''，[[词牌|词牌]]名。本為[[唐|唐]][[教坊|教坊]]曲名。[[双调|双调]]三十六字，前后[[阕|阕]]格式相同，各三[[平韵|平韵]]，一[[叠韵|叠韵]]，[[一韵到底|一韵到底]]。

== 異名 ==
[[林逋|林逋]]詞有“吳山青”句，故又名《吳山青》；[[張輯|張輯]]詞有“江南山漸青”句，故又名《山漸青》；[[王行|王行]]詞名《青山相送迎》；《樂府雅詞》名《長相思令》，又名《相思令》。

== 词牌格式 ==
雙調三十六字，前後段各四句三平韻、一疊韻。

{{quote|
{{平仄|[仄][仄]'''平'''，仄[仄]'''平'''（疊）。[仄]仄平平[仄]仄'''平'''。[平]平[仄]仄'''平'''。

仄[仄]'''平'''，仄[仄]'''平'''（疊）。[仄]仄平平[仄]仄'''平'''。[平]平[仄]仄'''平'''。}}
}}

<span style="font-size:smaller;">註：{{平仄|平}}表示平聲，{{平仄|仄}}表示仄聲，{{平仄|[仄]}}表示本仄可平，{{平仄|[平]}}表示本平可仄，粗體表示韻腳。</span>
