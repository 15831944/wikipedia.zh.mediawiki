'''长沙医学院'''位于[[湖南省|湖南省]][[长沙市|长沙市]]和[[衡阳市|衡阳市]]，是湖南省的民办本科大学。

== 历史 ==
1989年，[[湖南省|湖南省]][[衡阳市|衡阳市]]建立了'''湘南医学高等专科学校'''，它是长沙医学院的前身。

2000年，长沙医学院在[[湖南省|湖南省]][[长沙市|长沙市]]建立了分校，标志着长沙医学院开始拥有两所校区。

== 学校结构 ==
长沙医学院拥有十五个院系，另有一所附属医院。
* 第一临床学院、护理学院、医学影像学院、中医学院、体育教研部、思想政治教研部、国际教育学院
* 基础医学系、医学检验系、药学系、公共卫生系、口腔医学系、外语系、管理系、计算机系、中文系

== 校园文化 ==
* 办学理念：质量立校、突出特色、开拓创新、追求卓越
* 教育理念：医学与人文相通，博学与专精兼职，理论与实践并重
* 教育思想：以人为本、学会做人、学会做事；立德、立功、立言和立身
