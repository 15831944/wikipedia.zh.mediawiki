'''长江大桥'''，严格意义上仅指[[宜宾|宜宾]][[岷江|岷江]]口以下[[长江|长江]]段上的[[桥梁|桥梁]]，但通常也泛指长江干流上的所有[[桥梁|橋-{梁}-]]。

== 概述 ==
[[File:Caiyuanba_bridge.jpg|thumb]]

[[长江|长江]]是中国第一大河，全长6380多公里，其中从[[格拉丹冬|格拉丹冬]]源头至[[当曲|当曲]]河口称为[[沱沱河|沱沱河]]，长约358公里，从当曲河口至[[青海|青海]][[玉树|玉树]]县巴塘河口称为[[通天河|通天河]]，长约813公里。从巴塘河口至[[四川|四川]][[宜宾|宜宾]][[岷江|岷江]]口，称为[[金沙江|金沙江]]，长2308公里，岷江口至入海口，称为[[长江|长江]]，长约2884公里。目前长江上已建和在建桥梁共100座，其中沱沱河上2座、通天河上2座、金沙江上41座、长江上55座。宜宾[[岷江|岷江]]口以下55座桥梁全部为大型或特大型桥梁，并且桥名中均含有“长江”二字，宜宾岷江口以上45座桥梁大多为小型或中型桥梁，并且桥名中均不含有“长江”二字。

== 历史 ==

===古代===
有史记载的长江上第一座桥建于[[东汉|东汉]][[建武_(东汉)|建武]]11年（公元35年），当时四川割据势力[[公孙述|公孙述]]为阻止汉军入[[蜀|蜀]]曾在今[[宜昌|宜昌]]附近架设[[江关浮桥|江关浮桥]]（见《[[后汉书|后汉书]].[[岑彭|岑彭]]传》）。公元974年[[北宋|北宋]]大将[[曹彬|曹彬]]在[[南京|南京]]附近的[[采石矶|采石矶]]江面上用3天时间架设一座[[浮桥|浮桥]]，数万宋军得以过江灭掉[[南唐|南唐]]，此外[[晋朝|晋]]、[[南北朝|南北朝]]、[[隋朝|隋]]、[[唐朝|唐]]、[[明朝|明]]等都有在长江上架设浮桥的记载。到了[[清代|清代]]，清军和[[太平天国|太平天国]]的战争中，双方都曾数次在长江上架设浮桥。

===20世纪50年代以前===
[[File:Nanjing_Yangtze_River_Bridge02.jpg|thumb]]
在现今长江上的一百多座桥梁中最古老的桥是[[金沙江|金沙江]]上的[[梓里江桥|梓里江桥]]（又名金龙桥），位于[[云南|云南]][[丽江|丽江]]，始建于清[[光绪|光绪]]六年（1880年），由提督[[蒋宗汉|蒋宗汉]]捐资建造，号称“万里长江第一桥”，是丽江著名[[旅游|旅游]]景点之一，该桥经1934年重修，目前仍可供人和非机动车辆通行。

[[民国|民国]]时期，曾多次有[[武汉|武汉]]修建长江大桥的设想，并于1946年成立了以[[茅以升|茅以升]]为总工程师的“武汉大桥筹备委员会”， 组建了中国桥梁公司，并开始了方案研究和测绘工作，但后因[[第二次国共内战|内战]]爆发未能实现。

===20世纪50、60年代=== 
[[File:Railings_of_Wuhan_Yangtze_River_bridge.JPG|thumb]]
1956年在金沙江上建成[[岗托吊桥|岗托吊桥]]，现该桥被弃用有可能被拆除，[[武汉长江大桥|武汉长江大桥]]是第一座真正意义的长江大桥，1955年开工，1957年建成。1958年开工建设的[[重庆白沙沱长江大桥|重庆白沙沱长江大桥]]是一座[[铁路桥|铁路桥]]。同年建成的[[109国道|109国道]][[沱沱河大桥|沱沱河大桥]]是[[沱沱河|沱沱河]]上的第1座桥，该桥于1987年和2002年进行了两次重建，1963年建的[[214国道|214国道]]通天河大桥是通天河上第一座桥，该桥原为[[拱桥|拱桥]]，2003年重建新桥时改为[[梁桥|梁桥]]。整个50、1960年代共建16座桥，其中泯江口以下长江大桥仅3座，包括1968年的[[南京长江大桥|南京长江大桥]]。

===20世纪70、80年代===
1970、1980年代建成的桥梁也是16座，其中岷江口以下长江大桥4座，分别为1971年的[[枝城长江大桥|枝城长江大桥]]（又名[[宜都长江大桥|宜都长江大桥]]）、1980年的[[重庆石板坡长江大桥|重庆石板坡长江大桥]]、1982年的[[泸州长江大桥|泸州长江大桥]]，另外1979年的三江大桥也被认为是一座长江大桥，虽然它只是连接长江北岸和江中小岛的桥梁，并不跨越长江，但[[葛洲坝|葛洲坝]]建成后，就可以通达南岸了，桥名也改为“葛洲坝三江大桥”。

===20世纪90年代===
1990年代随着中国经济的高速成长，建桥的数量也越来越多、规模也越来越大，10年共建造桥梁17座，其中岷江口以下长江大桥达11座，1999年建成的[[江阴长江大桥|江阴长江大桥]]投资高达36亿元[[人民幣|人民幣]]。此外还有13座长江大桥已开工，处在建设中。

===21世纪===

21世纪长江迎来了建桥高峰，仅2000年至2005年的5年间，长江上就新建桥梁32座，其中岷江口以下长江大桥24座，另有18座大桥（其中岷江口以下长江大桥13座）于2005年至2008年陆续建成通车。此外，未建成的长江大桥还包括：[[南京长江五桥|南京长江五桥]]、[[九江长江二桥|九江长江二桥]]、[[荆岳长江公路大桥|荆岳长江公路大桥]]、[[赤壁长江大桥|赤壁长江大桥]]、[[重庆万州长江三桥|重庆万州长江三桥]]、[[丰都长江二桥|重庆丰都长江二桥]]、[[寸滩长江大桥|重庆寸滩长江大桥]]等。

== 分布 ==

===上海===
[[上海|上海]]境内现有1座隧道和2座大桥。[[上海长江隧桥|上海长江隧桥]]于2009年10月31日建成通车，[[崇启大桥|崇启大桥]]则于2011年12月24日通车。

===江苏===
[[File:NJ2Q.jpg|thumb]]
[[江苏|江苏]]境内现有12座长江大桥、2条长江隧道，至2012年底已建成9座大桥和1条隧道，另有1条隧道在建。已建成的通道中，[[南京|南京]]5座大桥1条隧道、[[镇江|镇江]][[扬州|扬州]]间1座、[[泰州|泰州]][[扬中|扬中]]1座、[[江阴|江阴]][[靖江|靖江]]1座、[[南通|南通]][[苏州|苏州]]间1座，因为地处长江下游，江面宽阔，所以这些桥梁均为特大型桥梁。[[崇启大桥|崇启大桥]]位于最下游，是长江自下而上的第2座桥。

===安徽===
[[安徽|安徽]]境内建成和在建的共10座长江大桥，其中1座位于[[马鞍山|马鞍山]]（2013年通车的[[马鞍山长江大桥|马鞍山长江大桥]]），3座位于[[芜湖|芜湖]]（2000年通车的公铁两用[[芜湖长江大桥|芜湖长江大桥]]，2017年通车的公路桥[[芜湖长江二桥|芜湖长江二桥]]，在建的[[商杭客运专线|商杭客运专线]][[弋矶山长江大桥|弋矶山长江大桥]]），2座位于[[铜陵|铜陵]]（1995年建成的[[铜陵长江大桥|铜陵长江大桥]]，2015年通车的[[铜陵长江公铁大桥|铜陵长江公铁大桥]]），1座位于池州（在建的[[池州长江大桥|池州长江大桥]]），2座位于[[安庆|安庆]]（2004年通车的[[安庆长江大桥|安庆长江大桥]]，2015通车的[[宁安高铁|宁安高铁]][[安庆长江铁路大桥|安庆长江铁路大桥]]），1座连接安庆[[望江县|望江县]]和池州[[东至县|东至县]]（2016年通车的[[望东长江大桥|望东长江大桥]]）。

===江西===
[[江西|江西]]境内有2座长江大桥，[[九江长江大桥|九江长江大桥]]和[[九江长江二桥|九江长江二桥]]（大桥北端均在湖北省境内）。[[九江长江大桥|九江长江大桥]]1993年建成后超过[[南京长江大桥|南京长江大桥]]成为最长的[[公路|公路]][[铁路|铁路]]两用桥，这一记录于1997年被[[芜湖长江大桥|芜湖长江大桥]]打破。九江长江大桥还保持着另一项记录――工期最长记录，1973年开工至1993年建成，工期长达20年。[[九江长江二桥|九江长江二桥]]即九江长江公路大桥，位于九江长江大桥上游10.8公里处。

===湖北===
[[湖北|湖北]]境内有已建成和在建的长江大桥超过20座，主要集中在[[武汉|武汉]]和[[宜昌|宜昌]]。

恩施州区域有[[巴东长江大桥|巴东长江大桥]]。

宜昌区域自上游到下游分别是[[西陵长江大桥|西陵长江大桥]]、[[至喜长江大桥|至喜长江大桥]]、[[葛洲坝三江大桥|葛洲坝三江大桥]]、[[夷陵长江大桥|夷陵长江大桥]]、[[宜万铁路宜昌长江大桥|宜万铁路宜昌长江大桥]]、[[宜昌长江公路大桥|宜昌长江公路大桥]]。

枝江区域自上游到下游分别是[[枝城长江大桥|枝城长江大桥]]、'''[[枝江百里洲长江大桥|枝江百里洲长江大桥]]'''：枝江百里洲规划的长江大桥为公铁长江大桥，超越日本明石海峡大桥，结合了武汉长江大桥，武汉天兴洲长江大桥.沪通长江大桥的优点。将是湖北省一大特色长江大桥。

荆州区域自上游到下游分别是[[荆州长江大桥|荆州长江大桥]]、[[荆岳长江公路大桥|荆岳长江公路大桥]]。

武汉1+8城市圈区域自上游到下游分别是[[武汉军山长江大桥|武汉军山长江大桥]]、[[武汉沌口长江大桥|武汉沌口长江大桥]]、[[武汉白沙洲长江大桥|武汉白沙洲长江大桥]]、[[武汉鹦鹉洲长江大桥|武汉鹦鹉洲长江大桥]]、[[武汉长江大桥|武汉长江大桥]]、[[武汉长江二桥|武汉长江二桥]]、[[武汉二七长江大桥|武汉二七长江大桥]]、[[武汉天兴洲长江大桥|武汉天兴洲长江大桥]]、[[武汉阳逻长江大桥|武汉阳逻长江大桥]]、[[黄冈长江大桥|黄冈长江大桥]]、[[鄂黄长江大桥|鄂黄长江大桥]]、[[鄂东长江大桥|鄂东长江大桥]]、[[黄石长江大桥|黄石长江大桥]]、[[九江长江大桥|九江长江大桥]]（大桥南端在江西省境内）、[[九江长江二桥|九江长江二桥]]（大桥南端在江西省境内）。湖北黃岡長江大橋全長四千零八米，投資二十三億元，2010年二月動工。

===重庆===
共有30座長江大橋。重庆有長江上的“桥梁[[博物馆|博物馆]]”之称，市内建有各种大小桥梁800多座，重庆的长江大桥也是最多的，[[重庆主城區|重庆主城區]]有10座，第一座是1959年建成的[[白沙沱长江大桥|白沙沱长江大桥]]。其中[[鱼洞长江大桥|鱼洞长江大桥]]、[[菜園壩長江大橋|菜園壩長江大橋]]、[[東水門長江大橋|東水門長江大橋]]和[[朝天門長江大橋|朝天門長江大橋]]為[[地鐵|地鐵]]、[[輕軌|輕軌]]公路兩用橋，另还有20座分布在長江沿岸的每个區縣。

===四川===
[[四川|四川]]境内有长江大桥12座（含3座在建），其中9座位于[[泸州|泸州]]，3座位于[[宜宾|宜宾]]。此外金沙江上的桥梁也主要集中在四川，共有超过50座（不含各水电站施工用临时索桥和在建桥梁），其中完全位于四川省内的21座，其余为跨界桥梁连接[[云南|云南]]或[[西藏|西藏]]。

===云南===
[[云南|云南]]境内有金沙江桥14座，包括上述横跨云南和四川。

===西藏===
[[西藏|西藏]]境内有金沙江桥五5座，全部位于西藏和四川的边界上。

===青海===
[[青海|青海]]是长江的发源地，沱沱河和通天河上各建有2座桥，都分布在青海境内。

== 桥梁用途 ==

在宜宾岷江口以下55座长江大桥中[[公路|公路]]桥占了70％以上，有43座。[[铁路|铁路]]桥5座。两用桥有8座，其中公路铁路两用桥6座，[[武汉天兴洲长江大桥|武汉天兴洲长江大桥]]是世界最大的公路铁路两用桥，[[公路|公路]][[地铁|地铁]]两用桥2座，分别为[[重庆鹅公岩大桥|重庆鹅公岩大桥]]和在建的[[重庆朝天门长江大桥|重庆朝天门长江大桥]]，此外在金沙江上还有一座特殊的两用桥——[[宝鼎大桥|宝鼎大桥]]，上层通车下层运[[煤|煤]]，而金沙江[[503电厂输煤桥|503电厂输煤桥]]是一座专用运煤桥，这两座桥均位于[[攀枝花|攀枝花]]。

== 桥梁类型 ==

长江上桥梁类型主要有[[梁桥|梁桥]]、[[拱桥|拱桥]]、[[斜拉桥|斜拉桥]]、[[悬索桥|悬索桥]]、[[吊桥|吊桥]]、[[铁索桥|铁索桥]]等，90年代以前建设的长江大桥以梁桥居多，随着造桥技术的发展，以及对[[航运|航运]]和美观的考虑[[斜拉桥|斜拉桥]]和[[悬索桥|悬索桥]]在90年代开始流行并成为主流，2005年10月建成的[[南京长江三桥|南京长江三桥]]主跨648米取代了[[南京长江二桥|南京长江二桥]]（南叉桥628米）国内第一斜拉桥的位置，在世界上居第三位，仅次于[[日本|日本]][[多多罗大桥|多多罗大桥]]（890米）和[[法国|法国]][[诺曼底大桥|诺曼底大桥]]（856米），建设中的[[苏通长江大桥|苏通长江大桥]]主跨首次超过1000米达到1088米，建成后将超越[[多多罗大桥|多多罗大桥]]，成为世界第一斜拉桥。2005年5月建成的[[润扬长江大桥|润扬长江大桥]]南叉桥主跨达到1490米，取代1999年建成的[[江阴长江大桥|江阴长江大桥]]（1385米）成为国内第一悬索桥，在世界上也仅次于日本[[明石海峡大桥|明石海峡大桥]]（1990米）和[[丹麦|丹麦]][[大带桥|大带桥]]（1624米），居第三位。拱桥多在长江中上游的桥梁中采用，2005年5月建成的[[巫山长江大桥|巫山长江大桥]]主跨492米是世界跨径最大的钢管混凝土拱桥，[[万州长江大桥|万州长江大桥]]（420米）是世界跨径最大钢筋混凝土拱桥，重庆新建的[[重庆朝天门长江大桥|重庆朝天门长江大桥]]主跨超过了500米，达552米，超越上海的[[卢浦大桥|卢浦大桥]]（550米），是世界跨径最大钢桁架拱桥，同时也成为新的拱桥世界冠军。

== 建桥资金 ==

由于建设长江大桥投资巨大，早期的长江大桥如[[武汉长江大桥|武汉长江大桥]]、[[南京长江大桥|南京长江大桥]]等都由国家拨款建造，随着经济发展建桥资金来源也呈现多样化，包括地方政府、[[银行|银行]][[贷款|贷款]]、企业、[[外资|外资]]等等，[[地维长江大桥|地维长江大桥]]就是由重庆腾辉地维水泥有限公司投资建设的，所以该桥没有按惯例以地名命名，而以投资企业命名。[[江津长江大桥|江津长江大桥]]由[[江津|江津]]市政府和[[马来西亚|马来西亚]]南发集团合作修建。[[重庆李家沱长江大桥|重庆李家沱长江大桥]]部分建设资金也利用了[[日本海外协力基金|日本海外协力基金]]贷款。建桥资金多样化也造成部分大桥因资金问题而停工，如[[涪陵长江二桥|涪陵长江二桥]]（又名[[涪陵李渡长江大桥|涪陵李渡长江大桥]]）1994年开工不久就因资金问题停工，后通过[[BOT|BOT]]方式融资得以继续建设。

== 长江大桥的竞争者 ==
2004年12月位于[[武汉长江大桥|武汉长江大桥]]和[[武汉长江二桥|武汉长江二桥]]之间的[[武汉长江隧道|武汉长江隧道]]开工建设，该[[隧道|隧道]]投资20亿元，2008年12月28日通车试运行，这是长江上的第一条隧道。之后[[南京长江隧道|南京长江隧道]]也于2005年10月开挖。规划中[[崇明越江通道工程|崇明越江通道工程]]也是由[[上海长江大桥|上海长江大桥]]和[[上海长江隧道|上海长江隧道]]组成，长江大桥有了竞争者。

== 负面影响 ==

随着大桥越建越多，负面影响也越来越受到关注。其中最主要的是对[[航运|航运]]的影响，目前长江主[[航道|航道]]平均桥距不足50公里。由于桥梁选址、桥墩设计等原因，桥区航道变窄，泥沙淤积严重，[[船舶|船舶]]航行困难。有数据表明2003年到达[[南京|南京]]以上港口的船舶仅占长江上船舶总数的3.6％，吨位仅占总数的1.3％，万吨级的[[安庆|安庆]]、[[武汉|武汉]]港、五千吨级的[[芜湖|芜湖]]、[[铜陵|铜陵]]、[[九江|九江]]等[[港口|港口]]的到港船舶严重不足，[[港口|港口]]大量资源闲置。而船舶撞击桥墩的事故却频频发生，例如[[黄石长江大桥|黄石长江大桥]]1993年4月至9月间就发生撞桥事故16起，沉船9艘。
