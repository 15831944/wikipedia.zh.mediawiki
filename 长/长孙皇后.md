'''长孙皇后'''（{{bd|601年3月15日|||636年7月28日||CatIdx=Z长孙}}），[[河南郡|河南郡]][[洛阳县|洛阳县]]（[[河南省|河南省]][[洛阳市|洛阳市]]东）人，[[中国|中国]][[唐朝|唐朝]][[唐太宗|唐太宗李世民]]皇后。唐朝第一位在位皇后，去世后谥号为“文德皇后”。

== 早年 ==
长孙皇后的父亲[[长孙晟|长孙晟]]是隋[[右骁卫|右骁卫]]将军、著名外交家，平[[突厥|突厥]]之功臣。生母高氏，为[[北齐|北齐]]宗室后裔，是清河王[[高岳|高岳]]之孙女，乐安王[[高劢|高劢]]之女。<ref>《[[s:舊唐書/卷51|舊唐書‧后妃上]]》：太宗文德順聖皇后長孫氏，長安人，隋右驍衛將軍晟之女也。晟妻，隋揚州刺史高敬德女，生-{后}-。</ref>长孙皇后的伯父[[长孙炽|长孙炽]]在北周担任通道馆学士，曾听说过神武公窦毅的女儿[[太穆皇后|窦氏]]年幼时期劝说[[周武帝|周武帝]]宇文邕抚慰突厥皇后，长孙炽记下了此事，长孙炽经常对长孙晟说：“这是个聪颖明智的人，必定有奇特的儿子，不能不设法结为婚姻。”于是长孙晟与窦氏的儿子[[李世民|李世民]]定下婚姻。[[大业_(年号)|大业]]五年（609年），长孙晟去世，[[长孙無忌|长孙無忌]]兄妹与高氏被异母兄长[[长孙安业|长孙安业]]赶出家门，得舅父[[高士廉|高士廉]]收容。大业九年（613年），在高士廉安排下，时年十三的长孙氏嫁给唐国公次子李世民为妻。唐朝建立后，李世民被立为秦王，长孙氏则为秦王妃。武德末年，她竭力争取李渊后宫对李世民的支持，玄武门之变当天，她亲自勉慰诸将士。之后拜太子妃。

== 皇后生涯 ==

公元626年发生[[玄武门之变|玄武门之变]]，李世民登基为帝。其后长孙无垢被立为皇后。

长孙皇后喜图传，即便是梳妆打扮时也手不释卷<ref>后喜图传，视古善恶以自鉴，矜尚礼法......益观书，虽容栉不少废。</ref>。善借古喻今，经常与丈夫一起地共执书卷，谈古论今，与唐太宗感情极好。依据现存史料<ref>唐太宗有二十一女，除长孙皇后和[[韋珪|韋贵妃]]所生之女外，十六女生母不详。</ref>，在唐太宗所有后妃中，长孙皇后所生子女人数是最多的，生下三子四女，共七人。唐太宗的第一个儿子（即-{[[李承乾|太子承乾]]}-）和最后一个女儿（即[[新城公主|新城公主]]）都是她所生。因为长孙皇后的原因，多次对[[长孙安业|长孙安业]]，[[长孙顺德|长孙顺德]]，[[长孙敞|长孙敞]]的违法行为法外开恩。在她死后，李世民多次建庙度人为长孙皇后祈福，不再立皇后，甚至此亲自抚养她所生的[[唐高宗|晋王李治]]和[[李明达|晉陽公主]][[李明達|李明達]]二人，故[[晋阳公主|晋阳公主]]是目前为止唯一能考证的由皇帝亲养的公主。

李世民常与长孙皇后讨论国家大事，起初她以自己是一介女流为由不肯说出自己的看法，而后在李世民的一再请求和命令下终肯为他出谋划策。<ref>《旧唐书·长孙皇后传》：太宗固与之言，竟不之答。</ref><ref>《资治通鉴》：长孙皇后性仁孝俭素，好读书，常与上从容商略古事，因而献替，裨益弘多。</ref>长孙皇后多次让其兄长、位列[[凌烟阁二十四功臣|凌烟阁二十四功臣]]第一的[[长孙无忌|长孙无忌]]向李世民辞官，同时也多次劝李世民不要重用她哥哥，准其辞官。她语重心长地告誡长孙无忌，前朝外戚掌权多危害社稷，切不可重蹈覆辙。在妹妹的告誡下，长孙无忌多次向太宗请辞，为此李世民向皇后解释说他用长孙无忌并非因其为皇亲国戚，而是因为他的才能。然而在长孙皇后的一再请求下，李世民终于同意了她的请求，在长孙皇后在世时她哥哥始终没有执掌大权。

长孙皇后曾著书《[[女则|女则]]》共十篇，已佚。今存诗一首<ref>上苑桃花朝日明，兰闺艳质动春情。井上新桃偷面色，檐边嫩柳学身轻。花中来去看舞蝶，树上长短听啼莺。林下何须远借问，出众风流旧有名。</ref>。她死后与唐太宗合葬于[[唐昭陵|昭陵]]。

长孙皇后在位期间，积极履行皇后职务，曾两次行[[先蚕礼|先蚕礼]]。

== 名字 ==

正史中没有记录长孙皇后的名字，[[后蜀|后蜀]]何光远所著《[[鉴诫录|鉴诫录]]》称长孙皇后[[号|号]]“观音婢”，并[[难产|难产]]而亡<ref>《鉴诫录·卷二》梁武帝为寺家奴，岂禳囚死。长孙-{后}-号观音婢，难忏产亡。</ref>。并无其他史书记载长孙皇后死于难产。

当代影视剧中，编剧使用“长孙无垢”一名。在2008年由[[山西人民出版社|山西人民出版社]]出版、[[柏杨|柏杨]]编著的《中国帝王皇后亲王公主世系录》一书中，称其为“长孙无垢”<ref>{{cite book zh|author=柏杨|coauthors=|title=中国帝王皇后亲王公主世系录|othertitle=|format=|editor=|others=|edition=|chapterauthor=|chapter=|chapterformat=|location=山西|publisher=山西出版集团·山西人民出版社|date=2008|pages=272|accessdate=|id=ISBN 978-7-203-05971-4|doi=|url=http://www.sxskcb.com|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20170622005510/http://www.sxskcb.com/|archivedate=2017-06-22}}</ref>，同时此书的台北版中无“长孙无垢”之名。“长孙无垢”的出处是由山西人民出版社添加，还是柏杨添加皆不得而知。
