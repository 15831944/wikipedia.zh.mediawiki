'''长泰县'''（{{lang-nan|'''Tiông-thòa-koāiⁿ'''}}）是中國[[福建省|福建省]][[漳州市|漳州市]]的一個縣。土地面积912.67平方千米，其中耕地16.7万亩，山地90.3万亩。

== 历史沿革与行政区划 ==
[[隋|隋]][[唐|唐]]时期，现长泰县归[[泉州府|泉州府]][[南安县|南安县]]武德乡。公元876年置场。公元888年，改名武胜场，后改武安场。

[[南唐|南唐]][[保大|保大]]十三年（公元955年），长泰建县，取意“长久安泰”。县治设于今武安镇辖区内，归泉州府。

宋[[太平兴国|太平兴国]]五年（公元980年），改归漳州。

== 地理环境 ==
长泰县北面为泉州市[[安溪县|安溪县]]、南面为[[龙海市|龙海市]]、西面为[[华安县|华安县]]和[[漳州市|漳州市]][[龙文区|龙文区]]，而东南面则为[[厦门市|厦门市]]。

长泰地理可以用“七山一水二分田”来形容，呈向南开口的马蹄形地貌，阻滞了[[台风|台风]]和[[寒流|寒流]]的入侵，气候温暖，雨量充沛，属南亚[[热带海洋性气候|热带海洋性气候]]，年平均气温约21摄氏度，降雨量1460毫米，气候环境有利于农业生产。

== 特产 ==

== 矿产 ==
水能资源蕴藏量大，并已有相当程度的开发，是全国农村初级电气化县。非金属矿藏有[[高岭土|高岭土]]、[[叶腊石|叶腊石]]、[[钾长石|钾长石]]、[[花岗岩|花岗岩]]、[[矿泉水|矿泉水]]、[[温泉|温泉]]、[[九龙璧|九龙璧]]等。

== 旅游景点 ==
长泰森林覆盖率达55%。境内有[[天柱山国家森林公园|天柱山国家森林公园]]、[[百里龙津|百里龙津]]、[[活盘湖|活盘湖]]、[[十里村漂流|十里村漂流]]、[[百丈崖瀑布|百丈崖瀑布]]等风景景点。
