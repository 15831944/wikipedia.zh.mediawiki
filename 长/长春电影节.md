'''长春电影节'''，全称'''中国长春国际电影节'''，创办于1992年，由[[中华人民共和国国家广播电影电视总局|中国广电总局]]、[[吉林省|吉林省]]政府和[[长春市|长春市]]政府共同主办。是经过中华人民共和国国务院批准，国际制片人协会承认的具有国际性的电影赛事。每两年举办一次，与中国电影[[华表奖|华表奖]]隔年举办。与[[金鸡奖|金鸡]][[百花奖|百花]]电影节、[[上海国际电影节|上海国际电影节]]、[[珠海电影节|珠海电影节]]并称“中国四大电影节”。

== 奖项 ==
电影节设下列奖项：
# 最佳华语故事片奖（包括合拍故事片）；
# 最佳外语故事片奖；
# 优秀华语故事片奖；
# 优秀外语故事片奖；
# 最佳编剧奖；
# 最佳导演奖；
# 最佳男主角奖；
# 最佳女主角奖；
# 最佳男配角奖；
# 最佳女配角奖；
对获奖者，电影节组委会分别颁发金鹿奖。

各奖项提名奖1-3个，根据具体情况另设评委特别奖。对获奖影片、编剧、导演、摄影、音乐、演员分别颁发“金鹿奖”。
