'''北京长安俱乐部有限公司'''（简称'''长安俱乐部'''），位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[东长安街|东长安街]]10号，是富华国际集团旗下的高端商务俱乐部。

==简介==
1993年，港商[[陈丽华|陈丽华]]开始在东长安街南侧黄金地段建设[[中国内地|中国内地]]第一家高端商务会所“长安俱乐部”<ref name=qq>{{Cite web |url = https://news.qq.com/a/20081111/001755.htm |title =  政治动脉长安街：中国在汹涌而过|author =  |date = 2008-11-11 |publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref>。1996年10月，长安俱乐部开业<ref name=jj>{{Cite web |url = http://www.changan-club.com/1/index.php?c=article&a=type&tid=34#go |title =俱乐部介绍  |author =  |date =  |publisher =长安俱乐部  |language =  |accessdate = 2018-02-17 |quote =  }}</ref>。长安俱乐部当时是由陈丽华女士的[[香港|香港]]富华国际集团与北京市[[东城区_(1958–2010)|东城区]]体委合资创办，为高档涉外服务设施。长安俱乐部创办时董事长为陈丽华，并聘请[[霍英东|霍英东]]、[[李嘉诚|李嘉诚]]、[[郭炳湘|郭炳湘]]、[[郑裕彤|郑裕彤]]、[[胡慧中|胡慧中]]、[[周怀民|周怀民]]等海内外名士担任名誉理事。当时长安俱乐部是集娱乐、体育、餐饮、商业、写字楼为一体的13层豪华设施<ref>辜坚，爱国恋乡的女委员——访全国政协委员、香港富华国际集团董事长陈丽华，中华女子学院学报1996(3):13-15</ref>。到2000年代，这家入会费1.8万美元的俱乐部吸引了众多巨富，会员包括[[李嘉诚|李嘉诚]]、[[霍英东|霍英东]]、[[杨元庆|杨元庆]]等人<ref name=qq/>。2010年代，该俱乐部董事局主席为陈丽华<ref>{{Cite web |url = http://www.changan-club.com/1/index.php?c=article&a=type&tid=36#go |title = 董事局主席 |author =  |date =  |publisher =  长安俱乐部|language =  |accessdate = 2018-02-17|quote =  }}</ref>，董事长为[[赵勇|赵勇]]<ref>{{Cite web |url = http://www.changan-club.com/1/index.php?c=article&a=type&tid=37#go |title = 董事长 |author =  |date =  |publisher =  长安俱乐部|language =  |accessdate = 2018-02-17 |quote =  }}</ref>。

2001年，北京长安保时捷中心在长安俱乐部首层开业，这是[[保时捷|保时捷]]在[[中国大陆|中国大陆]]开设的首家保时捷中心。2013年6月8日，保时捷中国举办了庆祝该中心重装开幕的典礼，该中心正式晋级为全球首家采用全新保时捷建筑设计标准的保时捷中心<ref>{{Cite web |url =http://auto.sohu.com/20130608/n378442432.shtml  |title = 北京长安保时捷中心 呈现全新保时捷风貌 |author =  |date = 2013-06-08 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
