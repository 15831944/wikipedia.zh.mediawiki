'''蕉岭县'''位于[[中国|中国]][[广东省|广东省]]东北部，韩江上游，属于[[梅州市|梅州市]]管辖。东边和南边与[[梅县区|梅县区]]相邻，西边与[[平远县|平远县]]相邻，北边与福建省龙岩市[[武平县|武平县]]、[[上杭县|上杭县]]相邻。总面积为957平方公里。2002年人口为22万。县政府驻地在蕉城镇。

== 行政区划 ==
2005年5月辖8个镇：蕉城镇、广福镇、文福镇、新铺镇、三圳镇、南礤镇、蓝坊镇、长潭镇。
1个直属地方政府归侨区：[[华侨农场|华侨农场]]

== 历史 ==
蕉岭[[春秋时期|春秋]][[战国时期|战国时期]]属“[[百越|百越]]之地”，公元前214年，[[秦始皇|秦始皇]]平定[[南越|南越]]，蕉岭属[[南海郡|南海郡]][[龙川县|龙川县]]。

明崇祯六年（1633年）建县，名'''镇平县'''。民国3年（1914年）镇平县改名为蕉岭县。

1965年蕉岭县属于梅县地区，1988年蕉岭县属[[梅州市|梅州市]]管辖。

== 民族 ==
蕉岭县以[[汉族|汉族]][[客家人|客家人]]为主，有少量[[少数民族|少数民族]]。

== 语言 ==
蕉岭县大部分居民讲[[客家话|客家话]]。

== 经济 ==
[[File:1963-09_1963年_蕉岭.jpg|thumb]]
[[File:1963-09_1963年_蕉岭水电站.jpg|thumb]]
蕉岭的[[农业|农业]]以[[水稻|水稻]]为主，比较有特色的是[[沙田柚|沙田柚]]、[[龙眼|龙眼]]等[[种植业|种植业]]。[[工业|工业]]比较有特色的是水泥、建材。蕉岭的矿藏资源也非常丰富，[[石灰石|石灰石]]、粘土的储量较大，此外還有[[锰|锰]]、[[铁|铁]]、[[铝|铝]]、[[钨|钨]]、[[铜|铜]]、[[铅|铅]]、[[锌|锌]]、[[锡|锡]]、[[稀土金屬|稀土金屬]]和[[煤|煤]]礦藏。

== 旅游 ==
蕉岭县旅游资源丰富，名胜古蹟有长潭旅游度假区、[[丘逢甲|丘逢甲]]故居、龙潭飞瀑、蓬莱仙境、高台庵等。
