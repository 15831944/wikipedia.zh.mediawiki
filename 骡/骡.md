'''骡子'''（[[学名|学名]]：{{Lang|la|'''''Equus ferus'' × ''asinus'''''}}）是[[马|马]]和[[驴|驴]]的[[杂交种|杂交种]]。严格地说，在中文裡母马和公驴的後代称为[[马骡|马骡]]；母驴和公马的後代称为[[驴骡|驴骡]]。因母马和公驴的基因更容易结合，故而大部分骡都是馬骡。由于[[染色体|染色体]]不成对（63个），[[生殖细胞|生殖细胞]]无法进行正常的分裂（即[[减数分裂|减数分裂]]），故骡子没有生殖能力。

== 主要特征 ==
体型比馬小，比驴大，耳比马长，比驴短。大多数黑色，臀高约1.55米。在其短厚的头，长耳朵，四肢瘦小，蹄窄，短鬃毛，没有板栗（角质生长）内钩。在身高和身体的形状、颈部及哮吼、均匀大衣、牙齿，看来像马。
普遍认为骡子皮肤使它们更有能力抵御阳光和雨水及激烈的撞擊。

== 歷史與產業 ==
[[File:2003mule.png|thumb]]
[[古希臘人|古希臘人]]及[[古羅馬人|古羅馬人]]使用騾子作為[[交通工具|交通工具]]。騾子在20世紀初被用當作[[軍事|軍事]][[運輸|運輸]]工具，牠們被用做載送士兵和沉重的物資。早期的北美[[伊利運河|伊利運河]]還有騾子用的接駁船。中国也曾将騾子作为农业生产和运输工具。現代騾子變的罕見，是因為[[汽車|汽車]]和[[卡車|卡車]]等[[內燃機|內燃機]]車輛的普及。騾子可以穿過陡峭難走的、又黑又深的洞，所以在山区仍然是重要的运输工具，在美國[[科羅拉多大峽谷|科羅拉多大峽谷]]，當地[[觀光業|觀光業]]者使用騾子運送遊客下山。

== 分类 ==
* [[马骡|马骡]]
'''马骡'''，由母[[马|马]]与公[[驴|驴]]雜交所产生的后代。具有[[杂种优势|杂种优势]]，体型若马。身体较大，[[耳朵|耳朵]]较小，尾部的毛又黑又濃。力大且寿命长于马或驴。
* [[驴骡|驴骡]]
'''驴骡'''，正式名稱“'''駃騠'''”。[[雌性|母]][[驴|驴]]和[[雄性|公]][[马|马]]的种间杂种。外貌似驴。耐粗饲，适应性和抗病能力强，精力大而持久，但不及马骡。主要分布于[[中国|中国]][[华北|华北]]地区。

== 习性 ==
性比马倔強，比驴溫顺。骡驹合群性很强，尤其在夜间，总是栖居于马群中间。虽初生骡驹爱独自休息，但很少丢失。骡驹之间喜群居，但不愿与马驹相处。

骡驹胆大、机警、勇敢，勇于与野兽搏斗。好奇心很强，遇有新奇事物，总喜围拢观。

== 食性 ==
以草、树叶為食。骡的食量大于驴小于马，约比马少20%。骡对饲料的消化能力比马高10%，在使役时，骡对饲料干物质的消化率比马高8%。

== 寿命 ==
骡的寿命较长，一般可活到35岁左右，如饲养管理良好，可达50岁，使役可达20年。
