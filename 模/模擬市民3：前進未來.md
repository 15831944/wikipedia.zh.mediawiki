{{vgname|模擬市民3：前進未來|en=The Sims 3: Into the Future|cn=模拟人生3：前进未来|tw=模擬市民3：前進未來}}是《[[模擬市民3|模擬市民3]]》的第十一款[[資料片|資料片]]，也是《模擬市民3》系列最后一部作品，于2013年10月22日推出。<ref>{{cite web|url=http://simsvip.com/2013/05/06/future-themed-ep-last-game-for-sims-3-series/|title="Future Themed" EP, Last Game for Sims 3 Series}}</ref><ref>{{Cite web |url=http://store.steampowered.com/app/249180/?snr=1_7_7_230_150_1 |title=Steam 上的 The Sims 3 - Into the Future |date=2013年10月23日 |publisher=Steam |accessdate=2016-10-18}}</ref>游戏以未来生活为主题，并加入了时光门、未来电车、晶锥[[机器人|机器人]]等众多元素。<ref>{{Cite web|url = http://simsvip.com/2013/07/23/sims-3-into-the-future-fact-sheet-official-screens/|title = Sims 3 Into the Future Fact Sheet + Official Screens|date = July 23, 2013|accessdate = August 13, 2013|publisher = SimsVIP}}</ref><ref>{{Cite web|url = http://gb.thesims3.com/community/blog.html?blogid=513639|title = The Sims 3 Community Blog: Let's go...INTO THE FUTURE!!|date = July 30, 2013|accessdate = August 13, 2013|website = thesims3.com|publisher = Electronic Arts}}</ref>

模擬人生3：進入未來 是電子藝術的模擬人生3系列的第十一和最後的擴展包，並於2013年10月22日在北美髮布<ref>{{Cite web|url=https://simsvip.com/2013/05/06/future-themed-ep-last-game-for-sims-3-series/|title="Future Themed" EP, Last Game for Sims 3 Series|accessdate=2017-03-18|date=2013-05-06|last=SimsVIP|work=SimsVIP}}</ref> 它適用於Windows和OS X.在歐洲，它於2013年10月24日發布，2013年10月25日在英國發布。<ref>{{Cite web|url=https://www.engadget.com/2013/07/23/the-sims-3-goes-back-back-to-the-future/|title=The Sims 3 goes back ... back to the future|accessdate=2017-03-18|work=Engadget}}</ref>

== 遊戲 ==
擴張是未來主題。在安裝時，玩家可以由他們的家庭鄰居中的時間旅行者訪問，他們給他們的模擬他的時間門戶以便在接收來自玩家的幫助時使用。使用這個門戶，模擬人口可以前往未來，進入一個新的附屬社區“綠洲著陸”，一個未知的城鎮建在山區荒原。<ref>{{Cite web|url=https://simsvip.com/2013/07/23/sims-3-into-the-future-140-trailer-screens/|title=Sims 3 Into the Future: 140 Trailer Screens|accessdate=2017-03-18|date=2013-07-23|last=SimsVIP|work=SimsVIP}}</ref>

水鑽（類似於以前的sim遊戲中的simbots和servos）是擴展的主要補充。他們可以建立和完全定制與新的“創建機器人”模式，玩家可以創建各種特徵芯片，確定他們的個性，使用和遊戲玩法。懸浮板，噴氣背包，單軌和懸停車被介紹到遊戲作為新的交通選擇。新職業包括天文學家和bot經銷商。新的對象包括全息電視，食品合成器，建立一個機器人研討會和激光rythimacon，一種新的儀器。<ref>{{Cite web|url=http://gb.thesims3.com/community/blog.html?blogid=513639|title=Let's go.. INTO THE FUTURE!!|accessdate=2017-03-18|work=gb.thesims3.com}}</ref>

模擬人也能夠在時間旅行期間訪問他們的未來後代，並且如果玩家將模擬器發回現在並改變他們的生活方式或命運，則可以改變或消除這些後代。<ref>{{Cite web|url=http://thesims.com/en_GB/the-sims-3-into-the-future|title=Electronic Arts. Retrieved August|accessdate=August 13, 2013.|author=Electronic Arts. Retrieved August|date=August 13, 2013.|publisher=thesims.com|deadurl=yes|archiveurl=https://archive.is/20130814000446/http://thesims.com/en_GB/the-sims-3-into-the-future|archivedate=2013年8月14日}}</ref>

== 發展 ==
在2013年1月8日，電子藝術計劃發布一個未來主義的主題擴展包，仍處於發展的早期階段。<ref>{{Cite web|url=http://www.maxis.com/ |title=Maxis Home Page |accessdate=2017-03-18 |date=1996-11-10 |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/19961110065156/http://www.maxis.com/ |archivedate=1996-11-10 }}</ref>
