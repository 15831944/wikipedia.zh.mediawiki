'''跨境巴士'''，是指往來兩個不同[[國家|國家]]或地區的[[巴士|巴士]]服務，它們運輸乘客跨越[[邊境|邊境]]出入兩地，是站對站客流服務，所以又名'''過境巴士'''、'''直通巴'''、'''跨境快線'''、'''跨境公交车'''等。而在香港及澳門的跨境巴士，由於港澳和大陸的車行方向不同，故跨境巴士需適應兩邊不同的行車方向，且須考取特許駕照方可營運。

== 粤港跨境巴士 ==
粤港跨境巴士主要往来[[香港|香港]]和[[广东省|广东省]]各地。[[香港|香港]]的跨境巴士使用[[皇崗口岸|皇崗口岸]]、[[文錦渡口岸|文錦渡口岸]]、[[沙頭角口岸|沙頭角口岸]]、[[深圳灣口岸|深圳灣口岸]]及[[港珠澳大橋|港珠澳大橋]]往返香港、[[澳門|澳門]]及[[中國内地|中國内地]]。在[[皇崗口岸|皇崗口岸]]除了有往來[[落馬洲|落馬洲]]與皇崗的[[皇巴士|皇巴士]]外，還有直通跨境巴士往來皇崗口岸與[[元朗|元朗]][[錦上路站|錦上路站]]、[[灣仔碼頭|灣仔碼頭]]、[[旺角|旺角]]（[[跨境全日通|跨境全日通]]）、[[荃灣站|荃灣站]]（[[中港直通快線|中港直通快線]]）、[[觀塘|觀塘]][[藍田站|藍田站]]及[[尖沙咀|尖沙咀]][[柯士甸道|柯士甸道]]（[[24小時跨境快線|24小時跨境快線]]）。除此之外，還有一些由香港出發經皇崗、[[深圳灣口岸|深圳灣口岸]]、[[文錦渡|文錦渡]]及[[沙頭角|沙頭角]]前往[[廣州|廣州]]、[[中山市|中山]]、[[佛山|佛山]]、[[新會|新會]]、[[順德|順德]]等地的長途巴士。另外，乘搭跨境巴士到海关後，乘客需要携带行李全部下车并步行通过海关和移民检查，过关后再重新登车前往目的地。

== 澳門的跨境巴士 ==
在[[澳門|澳門]]，亦有提供跨境巴士往來[[澳門|澳門]]、[[香港|香港]]與[[中國內地|中國內地]]，其中[[澳巴|澳巴]]與[[岐關車路|岐關車路]]聯合開辦[[蓮花口岸|蓮花口岸]]專線巴士，由[[路氹邊檢大樓|路氹邊檢大樓]]前往[[珠海|珠海]][[橫琴口岸|橫琴口岸]]，收費澳門幣5元，每日24小時提供服務。另外，中旅也有經營關閘邊檢大樓至[[澳門富豪酒店|富豪酒店]]的路線。另外，乘搭跨境巴士到海关後，乘客需要携带行李全部下车并步行通过海关和移民检查，过关后再重新登车前往目的地。

而澳門主要的跨境旅遊客車服務經營商包括：
*[[香港中旅汽車服務|香港中旅汽車服務]]
*[[岐關車路|岐關車路]]
*[[信德國旅|信德國旅]]

== 新加坡跨境巴士 ==
[[新加坡|新加坡]]的[[新捷运|新捷运]]營辦160及170路巴士線，往來新加坡與[[馬來西亞|馬來西亞]][[新山|新山]]，每日清晨至晚間服務。乘客可以使用[[新加坡元|坡幣]]及[[令吉|馬幣]]繳費。另有跨境的[[星柔快車|星柔快車]]服務。

除此之外，新加坡亦有跨境長途巴士前往[[吉隆坡|吉隆坡]]。

== 中朝跨境巴士 ==
[[中國|中國]][[辽宁省|辽宁省]][[丹東|丹東]]的301路公交車，往來[[丹东|丹东]]口岸与[[新义州|新义州]]火车站，每天8：30及13：30发两班车。

== 中越跨境巴士 ==
[[中國|中國]][[廣西|廣西]]有多條往返[[越南|越南]]的客運班車，途經[[友誼關|友誼關]]、[[龍邦|龍邦]]口岸前[[越南|越南]]。路線：[[南寧|南寧]]至[[越南|越南]][[河內|河內]]，[[南宁|南宁]]至[[越南|越南]][[海防|海防]]、[[桂林|桂林]]至[[越南|越南]][[河内|河内]]、[[崇左|崇左]]至[[越南|越南]][[下龙|下龙]]、[[百色|百色]]至[[越南|越南]][[高平|高平]]。
<br />除此之外，[[中國|中國]][[雲南省|雲南省]][[昆明|昆明]]往返[[越南|越南]][[沙巴|沙巴]]的客運已有計劃開行，途經[[天保口岸|天保口岸]]出境。

==北美洲和墨西哥跨境巴士==
[[灰狗巴士|灰狗巴士]]，Tufesa，TapRoyal提供跨越美墨两国的跨境巴士。另外，乘搭跨境巴士到海关後，乘客需要携带行李全部下车并步行通过海关和移民检查，过关后再重新登车前往目的地。

== 歐洲跨境巴士 ==
由於歐洲大陸大部分地區皆為[[申根區|申根區]]範圍，跨境巴士可在申根區各國內暢行無阻，且乘客與司機無須接受護照檢查。跨欧洲主要是[[欧洲之线巴士|Eurolines]]和[[弗利克斯巴士|FlixBus]]。

巴士車資在[[歐元區|歐元區]]內通常以[[歐元|歐元]]計價，另外在非歐元區可使用[[瑞士法郎|瑞士法郎]]、[[丹麥克朗|丹麥克朗]]、[[捷克克朗|捷克克朗]]、[[瑞典克朗|瑞典克朗]]、和[[英镑|英镑]]等歐元以外貨幣，视巴士駛經的國家而定。
