'''跨巴布亚公路'''（[[印尼语|印尼语]]：Jalan Trans-Papua）是从[[索龙_(印尼)|索龙]]到[[马老奇|马老奇]]共11条互不相连的公路的总称，总长4,330.07公里，其中位于[[西巴布亚省|西巴布亚省]]的有1,070.62公里，位于[[巴布亚省|巴布亚省]]的有3,995公里<ref name="星洲网">{{cite web |url=http://www.sinchew.com.my/node/1642724 |title=佐科威视察跨巴布亚公路 |last= |first= |date=2017-05-13 |website=www.sinchew.com |publisher=星洲网新闻 |access-date=2017-10-16 |quote=}}</ref><ref name=TransPapua>{{cite web |url=http://www.harianpapua.com/jalan-trans-papua-telah-dibangun-tudingan-natalius-pigai-keliru/ |title=Jalan Trans Papua Telah Dibangun, Tudingan Natalius Pigai Keliru |accessdate=2017-03-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20170312045652/http://www.harianpapua.com/jalan-trans-papua-telah-dibangun-tudingan-natalius-pigai-keliru/ |archivedate=2017-03-12 }}</ref><ref name="盛超">{{cite web |url=http://www.ccpit.org/Contents/Channel_4114/2017/0605/818644/content_818644.htm |title=印尼政府压低巴布亚物流运费 |last=盛 |first=超 |date=2017-06-05 |website=www.ccpit.org |publisher=中国国际贸易促进委员会 |access-date=2017-10-16 |quote=}}</ref>。截止2017年5月已完工89%，预计所有道路将于2018年完工<ref name="星洲网"></ref><ref>{{cite web |url=https://finance.detik.com/ekonomi-bisnis/3112503/ini-penampakan-jalan-trans-papua |title=Ini Penampakan Jalan Trans Papua |author=Dana Aditiasari |accessdate=2017-03-10}}</ref>。

到2017年3月，已建起长3,850公里的道路<ref>{{cite web |url=http://www.thejakartapost.com/news/2017/03/07/trans-papua-road-set-to-be-completed-next-year.html |title=Trans-Papua road set to be completed next year |date=2017-03-07}}</ref>，包括884公里在与[[巴布亚新几内亚|巴布亚新几内亚]]的边界公路之中的路段（边界公路长1,068公里）<ref>{{cite web |url=https://kumparan.com/taufik-rahadian/jalan-trans-papua-membentang-di-atas-pegunungan |title=Jalan Trans Papua Membentang di Atas Pegunungan |date=2017-03-07}}</ref>。剩下的跨巴布亚公路将在2017年和2018年间完工，包括总长7,000米的桥梁<ref>{{cite web |url=https://kumparan.com/nurul-hidayati/trans-papua-membelah-gunung-menuruni-lembah |title=Trans Papua, Membelah Gunung Menuruni Lembah |date=2017-05-10}}</ref>，并不是所有的公路都由沥青铺装<ref>{{cite web |url=http://www.guojiribao.com/shtml/gjrb/20170302/308223.shtml |title=佐科维执意建造跨巴布亚公路 |last= |first= |date=2017-03-02 |website=www.guojiribao.com |publisher=国际日报 |access-date=2017-10-16 |quote=}}</ref>。
