'''倍里珠'''是[[日食|日全食]]出現的一種現象。當[[月球|月球]]“掠過”[[太陽|太陽]]時，崎嶇的[[月球邊緣|月球邊緣]]的地形讓陽光在某些地方通過，其他地方卻不行。這個名稱是為了紀念在1836年第一次注意與發現這種現象，並給了正確解釋的[[英國|英國]]天文學家[[弗朗西斯·貝利|弗朗西斯·貝利]]<ref>{{cite web
|url=http://articles.adsabs.harvard.edu/full/1836MNRAS...4...15B

|first=Francis
|work=[[Monthly_Notices_of_the_Royal_Astronomical_Society|Monthly Notices of the Royal Astronomical Society]], Vol. 4, p.15
|bibcode=1836MNRAS...4...15B}}</ref><ref>{{cite book

[[月球地形學|月球地形]]因存在著[[月球山脈列表|山脈]]、[[撞擊坑|隕坑]]、[[月谷列表|峽谷]]和其它的地形特徵，有相當大的[[地勢|地形起伏]]（離隙）。從觀測[[掩星|月掩星]]的掠掩，已經準確知道月球邊緣不規則的輪廓（從遠處看見的月球“邊緣”），因此天文學家已經可以知道在日全食的時候，那些高山峻嶺和山谷會導致倍里珠的出現。雖然，在日全食路徑的中心線倍里珠只會短暫的出現幾秒鐘，但在全食帶的[[本影|本影]]邊緣附近，持續的時間可以長達1-2分鐘。

觀看倍里珠和鑽石環若沒有保護眼睛的適當裝置和措施是不安全的，因為在這兩種現象出現時，[[光球|光球]]仍然是看得見的。

== 大众影响 ==
{{link-de|科斯马斯·达米安·阿萨姆|Cosmas Damian Asam}}可能是最早描繪日全食和鑽石環的寫實畫家<ref>{{cite web
|url = http://apod.nasa.gov/apod/ap080128.html
|title = A Solar Eclipse Painting from the 1700s
|publisher = [[Astronomy_Picture_of_the_Day|Astronomy Picture of the Day]]
|date = 2008-01-28}}</ref>。他的畫作完成於1735年。

2007年，美国[[国家广播公司|NBC]]开播的科幻電視剧《[[英雄_(電視劇)|英雄]]》就是以倍里珠的現象做為開場画面。

1995年，美国[[联合派拉蒙电视网|UPN]]开播的科幻电视剧《[[星艦奇航記：重返地球|星艦奇航記：重返地球]]》以鑽石環的現象做為開場画面。不过那是在一個虛構的太陽系外行星，從太空中看見的。
