'''北京大兴“11·18”火灾'''是2017年11月18日18时许发生在[[北京市|北京市]][[大兴区|大兴区]][[西红门地区|西红门镇]]新建庄二村新康东路8号聚福缘公寓的一场[[重大火灾|重大火灾]]<ref>{{cite news|url=http://bjwb.bjd.com.cn/html/2017-11/20/content_193823.htm|title=市委市政府召开清理整顿重大火灾隐患现场会和专项行动部署电视电话会 蔡奇要求在全市开展安全隐患大排查大清理大整治|publisher=[[北京晚报|北京晚报]]|date=2017-11-20|accessdate=2017-11-27}}</ref>，当时该[[公寓|公寓]]地下[[冷库|冷库]]在制冷设备调试过程中，为压缩冷凝机组供电的铝芯[[電纜|電纜]]发生电气故障造成短路，引燃墙体的[[聚氨酯|聚氨酯]]保温材料，导致19人因[[一氧化碳|一氧化碳]][[一氧化碳中毒|中毒]]死亡，8人受伤。截至2017年11月30日，已有包括起火房屋原始承租人在内的20人因涉嫌重大责任事故罪被[[刑事拘留|刑事拘留]]。同时，北京市大兴区区长亦因此向区人大常委会请辞获准<ref name="20detained">{{cite news|url=http://www.chinanews.com/sh/2017/11-27/8386668.shtml|title=北京大兴11·18火灾排除人为放火嫌疑 20人被刑拘|publisher=[[中国中央电视台|中国中央电视台]]|date=2017-11-27|accessdate=2017-11-27}}</ref><ref>{{cite news|title=北京大兴“11·18”火灾遇难者死因确定|url=http://society.people.com.cn/n1/2017/1123/c1008-29664636.html|accessdate=2017-11-29|agency=新华网|publisher=人民网|date=2017-11-23}}</ref><ref>{{cite news|title=北京大兴“11·18”火灾系电气线路故障所致|url=http://legal.people.com.cn/n1/2017/1127/c42510-29670746.html|accessdate=2017-11-29|agency=新华网|publisher=人民网|date=2017-11-27}}</ref>。

2018年6月，[[北京市人民政府|北京市人民政府]]批复了此次火灾的调查报告。经认定，此次火灾是一起重大生产安全责任事故<ref name=report>{{cite news|title=大兴区“11·18”火灾：重大生产安全责任事故|url=http://newpaper.dahe.cn/jrab/html/2018-06/26/content_257293.htm|accessdate=2018-06-26|publisher=河南法制报|date=2018-06-26}}</ref>。

== 事发背景 ==
本次火灾的发生地聚福缘公寓是一座集储存、生产、居住功能为一体的建筑物，占地面积6080平方米，建筑面积约为20000平方米，地下一层为冷库区，地上一层为餐饮、商店、洗浴、广告制作、服装生产加工储存等商户，地上二层、局部三层为出租房<ref name="18detained">{{cite news|url=http://bjwb.bjd.com.cn/html/2017-11/21/content_194481.htm|title=大兴11·18火灾18名涉案人员被刑拘|publisher=[[北京晚报|北京晚报]]|date=2017-11-21|accessdate=2017-11-27}}</ref>，共设305间房，租住有400余人，楼道呈回字形结构，外侧窗户被铁护栏封死<ref>{{cite news|url=http://www.bbtnews.com.cn/2017/1126/219808.shtml|title=工业大院变住房 火灾背后的聚福缘公寓|publisher=[[北京商报|北京商报]]|date=2017-11-26|accessdate=2017-11-27}}</ref>。该公寓于2002年至2006年分三次建成，未经相关部门审批<ref name="20detained"/>。西红门镇政府在2017年11月初即已启动对新建村的治理工作，整治通知于11月2日下发，原计划于11月18日开始对该区域展开清退行动，但因18日当天正值周六，镇政府临时将行动时间延后两天<ref name="two_days">{{cite news|url=http://bjwb.bjd.com.cn/html/2017-11/26/content_195944.htm|title=悲剧发生在清退前两天|publisher=[[北京晚报|北京晚报]]|date=2017-11-26|accessdate=2017-11-27}}</ref>。

该公寓所在的西红门镇境内曾有27个[[工业大院|工业大院]]<ref>{{cite news|url=http://www.infzm.com/content/130844|title=19人死亡后，北京村镇工业大院，一场火灾加速的腾退|publisher=[[南方周末|南方周末]]|date=2017-11-20|accessdate=2017-11-27}}</ref>，腾退工作早在2010年即已被当地政府提及<ref>{{cite news|author=郝成|url=http://www.cb.com.cn/index.php?m=content&c=index&a=show&catid=73099&id=1210811&all|title=“11•18火灾”背后的北京“工业大院”威胁|publisher=[[中国经营报|中国经营报]]|date=2017-11-21|accessdate=2017-11-27|archive-url=https://web.archive.org/web/20171201081056/http://www.cb.com.cn/index.php?m=content&c=index&a=show&catid=73099&id=1210811&all|archive-date=2017-12-01|dead-url=yes}}</ref>。事发地新建庄注册有82家服装制造企业，服装加工作坊和群租房聚集<ref>{{cite news|url=http://news.sina.com.cn/s/sd/2017-11-21/doc-ifynwxum7937531.shtml|title=大兴火灾过后：新建村撤离|publisher=[[新京报|新京报]]|date=2017-11-21|accessdate=2017-11-28}}</ref>，村内多数道路宽度在2米左右，消防车辆难以进入。<ref>{{cite news|url=http://www.takefoto.cn/viewnews-1334472.html|title=再探新建村：庇护城市劳动者的不应该是这样的地方|publisher=北晚新视觉|date=2017-11-27|accessdate=2017-11-28}}</ref>事发前，该村亦有整治行动，但鲜有租住户响应。<ref name="two_days"/>

== 火灾经过 ==
火灾发生于11月18日18时许。北京市119指挥中心于当天18时15分接到报警，北京市公安局消防局随即调派14个中队、34部消防车、188名消防官兵至火灾现场处置，北京市卫生计生委也派出22辆救护车前往事发现场。消防人员营救搜救出被困人员73人，地下冷库的明火则于当晚21时06分被扑灭<ref name="18detained"/>，但火灾产生的有毒烟气截至11月26日仍未完全散去。<ref name="two_days"/>

北京市公安局等部门的调查显示，公寓的起火部位为地下中部冷库间南墙中部的墙面，该处埋设在聚氨酯保温材料内的电气线路发生故障，导致了此次火灾，而冷库建设过程中铺设接连电线的作业人员均无专业资质。<ref name="20detained"/>

== 伤亡情况 ==
19人在此次火灾中遇难，其中男性11人（18岁以下6人），女性8人（18岁以下2人）。在遇难者中，4人来自山东省、4人来自河南省、4人来自河北省、2人来自陕西省、2人来自北京市大兴区、1人来自黑龙江省、1人来自新疆维吾尔自治区、1人来自吉林省<ref>{{cite news|url=http://www.ce.cn/xwzx/gnsz/gdxw/201711/21/t20171121_26933751.shtml|title=大兴11·18火灾事故通报发布 事故原因进一步调查|publisher=[[北京晨报|北京晨报]]|date=2017-11-21|accessdate=2017-11-28}}</ref>。截至2017年11月20日，8名伤者中7人体征平稳，1人伤情危重。

== 反应 ==
事发当晚，中共北京市委书记[[蔡奇|蔡奇]]、北京市代市长[[陈吉宁|陈吉宁]]赶赴现场指挥抢救处置<ref>{{cite news|url=http://bjrb.bjd.com.cn/html/2017-11/19/content_193564.htm|title=大兴区西红门镇新建村发生火灾已扑灭|publisher=[[北京日报|北京日报]]|date=2017-11-19|accessdate=2017-11-28}}</ref>，北京市委、市政府次日在事发地召开现场会，其后返回市委机关，召开全市安全隐患大排查大清理大整治专项行动部署电视电话会。蔡奇、陈吉宁此后两度再赴西红门镇督导隐患排查整治<ref name=bjrbwx>{{cite news|url=http://www.guancha.cn/society/2017_11_27_436713.shtml|title=大兴火灾后蔡奇三赴西红门：专项整治要坚定有序要有人文关怀|publisher=[[北京日报|北京日报]]微信公号|date=2017-11-27|accessdate=2017-11-28}}</ref>。

事发后，[[北京市安全生产委员会|北京市安全生产委员会]]决定自2017年11月20日起在北京市全市范围内开展为期40天的[[2017年北京安全隐患排查整改行动|安全隐患大排查、大清理、大整治专项行动]]，整治对象包括出租公寓、出租大院、待疏解或正拆除场所、彩钢板建筑、工厂企业、厂房库房、仓储物流、汽配城、批发市场等。<ref>{{cite web|url=http://www.bjsafety.gov.cn/attach/0/ca13b4c472904cca9f5d9ac8c6b13090.pdf|title=北京市安全生产委员会关于开展安全隐患大排查大清理大整治专项行动的通知|publisher=[[北京市安全生产委员会|北京市安全生产委员会]]|date=2017-11-19|accessdate=2017-11-27}}</ref>

11月27日，中共北京市委召开区委书记会，通报了此次火灾事故及[[红黄蓝教育|朝阳区红黄蓝幼儿园]]涉嫌伤害儿童事件处置情况。中共北京市委书记蔡奇表示要“充分认识维护首都安全稳定的极端重要性”，守土尽责，确保一方平安<ref name=bjrbwx/>。

11月29日，北京市纪委监察委公布对此次事件的责任追究结果，大兴区分管安全生产的副区长[[杜志勇|杜志勇]]被停职并立案调查，大兴区[[西红门地区|西红门镇]]党委书记郑亚君、镇长司文韬亦遭到停职和立案调查，同时该镇负责安全生产工作的10名官员也受到处理<ref>{{cite web|url=http://news.sina.com.cn/2017-11-29/doc-ifyphxwa7039712.shtml|title=北京公布大兴西红门重大火灾事故责任调查情况|publisher=[[新华社|新华社]]|date=2017-11-29|accessdate=2017-11-30}}</ref>。同日，大兴区区长[[崔志成|崔志成]]向[[北京市大兴区人民代表大会常务委员会|北京市大兴区人大常委会]]请辞获准，北京市大兴区人大常委会同时任命中共大兴区委副书记、开发区工委副书记[[王有国|王有国]]为大兴区副区长、代理区长<ref>{{cite web|url=http://www.zaobao.com.sg/realtime/china/story20171129-814905|title=北京大兴区区长崔志成辞职|publisher=[[联合早报|联合早报]]|date=2017-11-29|accessdate=2017-11-30}}</ref>。

根据2018年6月发布的调查报告，北京市公安局以涉嫌重大责任事故罪对樊兆田等15名有关责任人员立案侦查，并由检察机关批准逮捕；中共北京市纪委、北京市监察委给予大兴区杜志勇、李强等21名人员党纪和政务处分，对中共大兴区委、大兴区人民政府党组进行通报问责，并责令其作出书面检查；北京市安全监管局给予北京康特木业有限公司（聚福缘公寓出租单位）等3家涉事企业960万元的行政罚款<ref name=report/>。

=== 争议 ===
{{seealso|低端人口}}
由于北京市内的多处[[僭建|违章建筑]]仍然有人租住，大多租户被要求在短时间内迁出，部分租户更被消防、公安等人员砸门告知限期搬离。但大多数租户拒绝搬离，并与当局产生激烈矛盾。大兴火灾发生后，{{Who|有人|}}怀疑火灾事故发生的原因，是有人故意纵火，藉此摧毁贫民居住区，质疑当局借机“清理低端人口”<ref name="langfa">{{cite news|url=http://www.zaobao.com/special/report/politic/cnpol/story20171126-813928|title=两天后断水断电断气 大兴狼垡居民被迫大逃离|publisher=[[联合早报|联合早报]]|date=2017-11-26|accessdate=2017-11-28}}</ref>，除此之外，北京政府在冬季要求租户紧急搬离却不采取补偿措施的行为被指缺乏人道主义<ref>{{cite news|url=http://www.zaobao.com/news/china/story20171125-813698|title=安全隐患清查整治引质疑 北京清理“低端人口”？|publisher=[[联合早报|联合早报]]|date=2017-11-25|accessdate=2017-11-27}}</ref><ref>{{cite web|url=http://www.chinanews.com/gn/2017/11-27/8386647.shtml|title=评论：疏解外来人口，政策应该多些“温度”|publisher=央视网|date=2017-11-27|accessdate=2017-11-28}}</ref>。而《北京日报》发表评论称，当局并未“清理低端人口”<ref>{{cite web|url=http://www.bbc.com/zhongwen/simp/world-42113608|title=北京大兴大火后安全整治 官媒否认借机清理“低端人口”　|publisher=BBC中文网|date=2017-11-24|accessdate=2017-11-28}}</ref>，其指出火灾的发生是因为电路故障，并且此次清退行动波及的除基层与外来人口外，也包括白领阶层与北京当地人。另有评论将把北京市政当局事后的处置行动与“驱逐低端人口”联系起来的言论比作别有用心的“邪恶之火”。评论并称，在[[中华人民共和国|中华人民共和国]]，“每一个公民的政治地位和生命权利都是平等的”，“低端人口”的概念是对部分人群“人格上的污辱”<ref>{{cite web|author1=范丰|title=驱赶“低端人口”之说是邪恶之火|url=http://views.ce.cn/view/ent/201711/27/t20171127_27004497.shtml|website=中国经济网|accessdate=2017-12-15}}</ref>。

而在专项行动开始数日后，[[通州区_(北京市)|通州区]][[台湖镇|台湖镇]]部分公寓并未如期断水断电，但仍被禁止接收新租户。<ref>{{cite news|url=http://www.sohu.com/a/206768450_118622|title=清退中部分租户接到口头通知：不会停水停电，可以住到年底|publisher=经济观察报|date=2017-11-26|accessdate=2017-11-28}}</ref>
