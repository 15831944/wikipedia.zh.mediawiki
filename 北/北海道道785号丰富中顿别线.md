'''北海道道785号丰富中顿别线'''（{{lang-ja|北海道道785号豊富中頓別線}}）是一条连结[[北海道|北海道]][[丰富町|丰富町]]和[[中顿别町|中顿别町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：北海道[[天盐郡|天盐郡]][[丰富町|丰富町]]日曹（＝[[北海道道84号丰富滨顿别线|北海道道84号丰富滨顿别线]]交点）
* 終点：北海道[[枝幸郡|枝幸郡]][[中顿别町|中顿别町]]上駒（＝[[国道275号|国道275号]]交点）
* 总長：29.6千米

== 共线区间 ==
* 天盐郡[[幌延町|幌延町]]上問寒（[[北海道道645号上问寒幌延停车场线|北海道道645号上问寒幌延停车场线]]）
* 幌延町上問寒 - 中問寒（[[北海道道583号上问寒问寒别停车场线|北海道道583号上问寒问寒别停车场线]]）
