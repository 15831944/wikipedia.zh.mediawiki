'''北66線 中港厝－五谷王'''，是位於[[新北市|新北市]]的一條鄉道，西起[[新北市|新北市]][[新莊區|新莊區]]中港路與幸福路口，東迄新北市[[三重區|三重區]]五谷王北街與五谷王北街42巷口，全長2.378公里。{{r|thb}}

== 路線說明 ==
;'''[[新北市|新北市]]'''
*[[新莊區|新莊區]]：'''中港厝'''（起點，[[File:TW_THWtp63-1.png|30px]][[北63線|北63-1線]]岔路）→ 幸福路 → '''思源路口'''（[[File:TW_CHW106a.svg|23px]][[縣道106號|106甲線]]岔路） → 幸福東路 → '''化成路口'''（[[File:TW_THWtp63.png|30px]][[北63線|北63線]]岔路） → '''區界'''

*[[三重區|三重區]]：'''區界''' → 中興北街42巷 → '''中興北街口''' → 五谷王北街42巷 → '''五谷王'''（終點，[[File:TW_THWtp65.png|30px]][[北65線|北65線]]岔路）
