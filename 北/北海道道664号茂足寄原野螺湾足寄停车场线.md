'''北海道道664号茂足寄原野螺湾足寄停车场线'''（{{lang-ja|北海道道664号モアショロ原野螺湾足寄停車場線}}）是一条位于[[北海道|北海道]][[十胜综合振兴局|十胜综合振兴局]]管内[[足寄郡|足寄郡]][[足寄町|足寄町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：北海道足寄郡足寄町上螺湾
* 終点：北海道足寄郡足寄町北1条西1丁目（旧[[北海道池北高原铁道|北海道池北高原铁道]][[足寄站|足寄站]]）
* 总距离：39.2千米

== 经过的自治体  ==
* 十勝综合振興局
** 足寄郡足寄町

== 主要连接道路 ==
* 北海道道143号北見白糠線（足寄町上螺湾）
* 国道241号（足寄町螺湾本町〜終点）：重複
* 国道242号（终点）
* [[北海道道663号植坂足寄停车场线|北海道道663号植坂足寄停车场线]]（终点）

== 历史 ==
* 1969年6月18日 - 路線勘定（北海道告示第1249号）
