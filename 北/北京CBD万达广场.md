'''北京CBD万达广场'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[建国路_(北京)|建国路]]93号，是北京的[[万达广场|万达广场]]之一，也是[[万达集团|万达集团]]总部所在地。

==简介==
北京CBD万达广场（初称“北京万达广场”）地处[[北京CBD|北京CBD]]内，北邻[[中国中央电视台|中国中央电视台]]新楼，规划占地面积超过10公顷，总建筑面积超过50万平方米。集涉外连锁商业中心、国际商务港、商城、酒店、写字楼、商业街区于一体。项目分为南区和北区<ref name=hexin>{{Cite web |url = http://bj.leju.com/2006-12-29/0200167918.html |title = 北京万达确立集团北区发展核心地位 |author =  |date = 2006-12-29 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref name=jj>{{Cite web |url =http://bj.leju.com/scan/2005-06-21/145880323.html  |title = 北京万达国际广场简介 |author =  |date =  2005-06-21|publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

北区（一期）以国际商务港为主，建筑面积30万平方米，由4座中高层商务楼、商业街、商务会所、会议中心、地下车库、城市森林广场组成。2005年6月完工，2005年底投入使用<ref name=hexin/><ref name=jj/>。

南区（二期）建筑面积20万平方米，由北京万达索菲特大酒店、两栋写字楼、城市综合物业组成。2006年12月28日完工并交付使用<ref name=hexin/><ref name=jj/>。同日，[[沃尔玛|沃尔玛]]超市等商家<ref>{{Cite web |url =http://news.sohu.com/20061229/n247325801.shtml  |title =沃尔玛第四家门店低调亮相……  |author =  |date = 2006-12-29 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>以及北京万达国际影城建国路店<ref>{{Cite web |url =http://news.sohu.com/20061229/n247324782.shtml  |title = 北京万达国际电影城盛大开业 |author =  |date = 2006-12-29 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>在南区开业。2007年8月19日，北京万达索菲特大酒店开业<ref>{{Cite web |url = https://www.chinanewswire.com/cn/pr/2007081915542878 |title = 北京万达索菲特大饭店盛大开业|author =  |date = 2007-08-19 |publisher =新闻稿  |language =  |accessdate =  |quote =  }}</ref>。2017年7月19日，万达集团以超过600亿人民币的售价，将旗下13个文旅城及77个酒店分别出售给[[融创中国|融创中国]]及[[富力地产|富力地产]]，签约仪式在北京万达索菲特大酒店举行，这是截至当时中国房地产史上交易金额最大的签约仪式<ref>{{Cite web |url = http://www.guancha.cn/economy/2017_07_21_419386.shtml |title = 一桩创历史交易背后 中国首富的难处与图谋 |author =  |date = 2017-07-21 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>。

2008年9月，万达集团总部从[[大连|大连]]迁至北京CBD万达广场<ref name=hexin/><ref>{{Cite web |url = http://money.163.com/15/0519/08/APVDUVEG00253G87.html |title =万达总部迁址上海冲击波：王健林很喜欢上海  |author =  |date = 2015-05-19 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2018年1月1日起，北京万达索菲特大饭店正式更名为北京万达文华酒店，更名后的北京万达文华酒店也是万达酒店及度假村集团旗下独立自主品牌管理和运营的第19家万达文华酒店<ref>{{Cite web |url =http://www.traveldaily.cn/article/119787  |title = 客源结构变化及本土崛起　外资高端酒店光环渐褪色 |author =  |date =2018-01-16  |publisher =  环球旅讯 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://travel.ce.cn/gdtj/201802/02/t20180202_6137264.shtml |title = 国际酒店品牌“被撤”背后 市场博弈才刚刚开始 |author =  |date = 2018-02-02 |publisher =  中国经济网|language =  |accessdate =  |quote =  }}</ref>。
