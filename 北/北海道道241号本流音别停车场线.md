'''北海道道241号本流音别停车场线'''（{{lang-ja|北海道道241号本流音別停車場線}}）是一条位于[[北海道|北海道]][[釧路市|釧路市]]内的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 路線数据 ==
* 起点：北海道[[钏路市|钏路市]][[音别町|音别町]]茶安别（＝サトンベツ川合流点付近）
* 終点：北海道釧路市音別町本町1丁目（＝[[北海道旅客铁道|JR北海道]][[音别站|音别站]]、[[国道38号|国道38号]]交点）
* 总長：23.7千米

== 经过的自治体  ==
* [[钏路综合振兴局|钏路综合振兴局]]
** 釧路市

== 主要连接道路 ==
* [[北海道道500号音别浦幌线|北海道道500号音别浦幌线]]＝音別町音別原野基線（＝二又簡易郵便局、ムリ川付近）
* [[国道38号|国道38号]]＝音別町本町1丁目（终点）

== 主要橋梁 ==
* 初音橋（[[音別川|音別川]]）＝音別町音別原野基線 - 音別町中音別

== 历史 ==
* 1957年3月30日 路線勘定。（1957年北海道告示第493号）
* 1994年10月1日 路線编号变更为241号。（1994年北海道告示第1468号）

== 沿線 ==
* 憩いの森＝音別町音別原野基線
