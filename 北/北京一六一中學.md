'''北京一六一中学'''（{{lang-en|'''Beijing NO.161 High School'''}}），简称'''一六一中'''，前身是[[京师公立第一女子中学|京师公立第一女子中学]]，1970年代更为现名。学校位于[[北京市|北京市]][[西城区|西城区]][[西长安街街道|西长安街街道]]，在[[故宫|故宫]]、[[北海公園_(北京)|北海]]、[[中南海|中南海]]、[[中山公园|中山公园]]的环抱之中。中校区在西[[长安街|长安街]]1号，是新建的花园式校区；北校区位于北长街南口，由一组仿古建筑构成；南校区在[[前门|前门]]西大街41号。北京一六一中学是[[北京市示范性高级中等学校|北京市示范性高级中等学校]]之一。

== 学校历史 ==
1913年，北京一六一中学前身，[[京師公立第一女子中學|京師公立第一女子中學]]在[[北平|北平]]建立，其校园是现在一六一中学的北校区。建校之初，[[中国共产党|中国共产党]]创始人之一[[李大钊|李大钊]]的诸多学生前来任教，他自己也曾经多次到校授课。在[[一二九抗日救亡运动|一二九抗日救亡运动]]中，北平市学联党团书记[[姚依林|姚依林]]将北平市学联选址于此。1935年，“一二·九”运动时期，学校是北平市学联所在地。1949年[[中华人民共和国|中华人民共和国]]成立后，学校成为北京市最早市属重点中学之一，70年代更名为北京一六一中学。进入二十一世纪来，西城区教育资源进行优化整合，2000年、2004年，北京市第三十中学和北京长安中学（北京市第六中学和北京市第二十八中学于1999年合并而成）相继并入北京一六一中学，成为学校南校区、中校区，学校形成了一校三址的办学格局。2005年，学校被评为北京市示范性普通高中校。

==校园文化==
===校名===
1972年更名后，由中华人民共和国主要领导人[[陈云|陈云]]题写。

===校徽===
诞生于1998年，以火箭飞向太空为基本图案，箭身是“161”的字体变形。

===校歌===
{{Wikisource|北京一六一中學校歌}}
作于中华民国二年（1913年），原为京师公立第一女子中学校歌，有着深厚的历史积淀。由于历史缘故，当时的学校、学府鲜少创作与传唱歌曲，因此{{fact|它是中国最早创作的校歌之一。}}
{{Quote||女中學如一爐火，均平鍛煉文武合。
五金同化共陶融，分析俱平妥。
後日鑄良材，今天先擔荷，何況是神京第一！
聰強慧可，棫-{朴}-菁莪。

女中學如一滴水，增長滋培灌溉遂。
五味同勻善調劑，澆灑養葳蕤。
後退將枯涸，前進莫趑趄，永保此神京第一！
創建開垂，千秋萬歲！}}

===校训===
'''诚  真'''
*诚——做人诚实守信。
*真——做事求真务实。

==校园环境==
北京一六一中学是花园式学校，有大量校园景观。

===校园设施===
学校有形体教室、科学实验室、陶艺教室、生物数码互动教室等，所有教室及理、化、生实验室均装有多媒体设备和实物展示设备。中校区和北校区分别有地下羽毛球和乒乓球场馆。

===名胜古迹===
由于[[文革|文革]]时期的严重破坏，北京民间的古迹已所剩不多。而一六一中学是在老城墙的基础上兴建而成，其中尚存一些文化古迹，其它则是后期建的仿古建筑，这在北京市的中等学校中别具一格。

位于中校区的[[升平署|升平署]]戏楼及其院落，包括四合院、排房、游廊、方亭、花园和升平署戏楼，面积200余平方米，原是清康熙年间建立的由太监掌管的外衙门，称[[南府|南府]]，1827年改为[[升平署戏楼|升平署戏楼]]，现做校图书馆之用<ref>{{cite web |url = http://sdjsb.bjd.com.cn/html/2013-01/28/content_44158.htm |title = 首都建设报：升平署戏楼}}</ref>。

== 所获荣誉 ==
{{unreferenced section}}
学校赢得了多项荣誉，承担了多项教育科研任务，包括：北京市“教育科研先进校”、“金鹏科技团”成员校、北京市“德育先进单位”、全国计算机教育先进单位、世界卫生组织 (WHO)“健康促进学校金牌校”、北京市文明单位等。并被确定为：中央教科所“创新教育研究与实验”课题实验校、联合国科教文组织项目“环境、人口与可持续发展(EPD)”实验校、全国中小学现代信息技术应用实验学校等。
