'''[[北爱尔兰|北爱尔兰]]'''被分為11个區。 因受北爱尔兰政治因素影響，每一個區/市議會（District/City/Borough Council）擁有康樂文化、市政衛生、本地經濟、屋宇管理等事務權利之外，其餘（包括教育、規劃）一律由高層次機關負責。

== 区划變化 ==
現今北愛爾蘭26個本地行政区划模式於1973年成立，取代[[:en:Local_Government_(Ireland)_Act_1898|1898年設立的]]舊行政区划；並預先假設北愛爾蘭政府為北愛爾蘭的管理架構。2015年4月，將26個行政區（含地方議會）減少到11個。<ref>[[:en:Local_Government_(Boundaries)_Act_(Northern_Ireland)_1971|:en:Local Government (Boundaries) Act (Northern Ireland) 1971]]</ref><ref>[[:en:Local_Government_Act_(Northern_Ireland)_1972|:en:Local Government Act (Northern Ireland) 1972]]</ref>

== 六大郡 ==
: 主条目：[[:en:Counties_of_Northern_Ireland|北愛爾蘭六大郡]]

[[File:Mussenden_Temple_mit_Schafen.jpg|thumb]]]]
北愛爾蘭地域由以下[[:en:Counties_of_Ireland|六大郡]]組成：
* [[安特里姆郡|安特里姆郡]]（County Antrim）
* [[阿馬郡|阿馬郡]]（County Armagh）
* [[唐郡|唐郡]]（County Down）
* [[弗馬納郡|弗馬納郡]]（County Fermanagh）
* [[倫敦德里郡|倫敦德里郡]]（County Londonderry）{{Notetag|有[[愛爾蘭民族主義|愛爾蘭民族主義]]者使用英語名稱''County Derry''. {{selfref|Wikipedia policy is to use the official name for the county. The name usage does not indicate an endorsement for either community's opinions.}}}}
* [[蒂龍郡|蒂龍郡]]（County Tyrone）

上述六大郡從1973年起不再為行政区；由26個新本地行政區取代。
