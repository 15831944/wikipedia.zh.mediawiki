'''北70線 營盤－新莊'''，是位於[[台灣|台灣]][[新北市|新北市]]的一條鄉道，西起[[新北市|新北市]][[新莊區|新莊區]]民全里民安路與民安東路路口，東至新北市[[新莊區|新莊區]]文衡里新泰路與中正路路口，全長3.392公里。{{r|thb}}

== 路線說明 ==
;'''[[新北市|新北市]]'''
*[[新莊區|新莊區]]： 營盤（起點，[[File:TW_THWtp71.png|30px]]民安路與民安東路口）→ 民安路 → '''新樹路口'''（[[File:TW_CHW107.svg|23px]][[縣道107號|縣道107號]]岔路） → 瓊林路 → '''瓊林'''（瓊林南路岔路）→ '''豐年街口''' → 新泰路 → '''新莊'''（終點，[[File:TW_PHW1a.svg|23px]][[台1甲線|台1甲線]]岔路）

== 支線 ==
北70線唯一的支線為'''北70-1線'''，是位於[[新北市|新北市]]的一條鄉道，西起新北市[[新莊區|新莊區]]環河路、豐年街與瓊林路口，東迄新北市[[新莊區|新莊區]]中正路與[[三重區|三重區]]重新路五段交界，全長2.583公里。{{r|thb}}

;'''[[新北市|新北市]]'''
