'''北京奥运博物馆'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[国家体育场南路|国家体育场南路]]1号，是由[[北京市人民政府|北京市人民政府]]委托[[北京市文物局|北京市文物局]]筹建的官方唯一以[[奥运|奥运]]为主题的永久性专题博物馆，是北京市文物局直属全额拨款[[事业单位|事业单位]]。<ref name=bjwb>{{Cite web |url = http://www.bjww.gov.cn/zwgk/zsdwDetail.htm |title =  北京奥运博物馆|author =  |date =  |publisher = 北京文博 |language =  |accessdate =  2017-01-04|quote =  }}</ref>

==简介==
北京奥运博物馆地处北京奥林匹克中心区[[国家体育场_(北京)|国家体育场（鸟巢）]]南侧地下零层和负一层，总面积为3.45万平方米。国家体育场是[[2008年北京奥运会|2008年北京奥运会]]主体育场，先后举行了奥运会、[[2008年北京残奥会|残奥会]]开闭幕式、田径比赛、足球比赛决赛。<ref name=bjwb/>

北京奥运博物馆自2008年北京奥运会、残奥会结束后便开始筹备工作，接收了按[[景泰蓝|景泰蓝]]工艺制作的“奥运缶”等北京奥运会、残奥会使用的物品，并接受了各界捐赠物上千件，例如北京奥运会女子体操团体及女子高低杠冠军[[何可欣|何可欣]]捐赠的夺冠时穿的领奖服，北京奥运会男子佩剑冠军[[仲满|仲满]]捐赠的夺冠时使用的手套和练习用剑。<ref>{{Cite web |url = http://news.xinhuanet.com/collection/2009-08/07/content_11839642.htm |title =奥运博物馆筹备工作进展顺利 捐赠物上千件  |author =  |date = 2009-08-07 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>但北京奥运博物馆一直未能开馆。<ref>{{Cite web |url =  http://www.takefoto.cn/viewnews-648016.html|title =  奥运博物馆即将开放 传播奥林匹克文化振奋中华民族精神|author =  |date = 2016-01-12 |publisher = 北晚新视觉 |language =  |accessdate =  |quote =  }}</ref>

按照规划，北京奥运博物馆分成上、下两层。上层是博物馆观众服务层，含票务预约中心、观众服务中心、多媒体查询区、儿童互动区、临时展区、纪念品经营区、办公区，另有1号放映厅和2号放映厅。下层是固定陈列展览层，展览面积1.9万平方米，含固定陈列五个展厅、3号放映厅、3D影院。<ref name=bjwb/>
