'''北92線 埔墘－漳和'''，是位於[[新北市|新北市]]的一條鄉道，北起[[新北市|新北市]][[板橋區_(台灣)|板橋區]]埔墘，南至[[新北市|新北市]][[中和區|中和區]][[漳和|漳和]]，全長2.821公里（已縮短至2.7公里）。{{r|thb}}路線的產生可追溯自日治時代{{r|oldmap}}，自1961年起便被編為鄉道{{r|gazette}}，之後鮮少有改線、改道。台北縣政府興建[[新北環河快速道路|新北環河快速道路]]時連代徵收新店溪畔環河道路土地，北92線部份路段受影響而拆除，部份'''非正式'''的改線至永和路200巷。{{r|lane200}}（請參見右方的路線圖）永和路的路名與[[永和區|永和區]]無關，為日治時期留存的[[永和_(大字)|舊地名]]{{r|oldmap}}，此地名甚至早於永和市（今永和區）的命名，但因中永和地區道路命名紊亂（參見[[雙和|雙和]]），此路又非市區主要幹道，因此常與永和區的永和路混淆。

== 路線說明 ==
;'''[[新北市|新北市]]'''
*[[板橋區_(台灣)|板橋區]]：埔墘0.0k（起點，[[File:TW_THWtp91.png|30px]][[北91線|北91線]]岔路）→ 環中路口0.29k → 區界
*[[中和區|中和區]]：區界 → 中原街口0.6k → 華中橋下1.8k → 橋和路口2.1k → 福美路口2.4k → 福祥路口2.6k → 中和中山路口2.7k（終點，[[File:TW_THWtp88.png|30px]][[北88線|北88線]]岔路，終點里程數原為2.821k）

埔墘至中和環中路口為板橋三民路二段245巷，中和環中路口至中原街為中板路，中原街至華中橋下為立業路，華中橋下至中和中山路口為永和路。
