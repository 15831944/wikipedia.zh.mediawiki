'''北京天主教神哲学院'''，位于[[北京市|北京市]][[海淀区|海淀区]][[双清路|双清路]]后八家村248号，是[[天主教北京教区|天主教北京教区]]的北京市天主教爱国会、北京市天主教教务委员会设立的天主教培育神职人员的大修院。<ref name=tg>[http://www.tianguangbao.org/tangqu/xiudaoyuan.htm 北京天主教神哲学院（修院）简介，天光网，于2012-4-17查阅]</ref>

==简介==
1979年12月21日，[[傅铁山|傅铁山]]神父晋牧[[天主教北京教区|天主教北京教区]]主教。此后，傅铁山主教在1980年初在[[宣武门天主堂|南堂]]西小院开办了已经中断了二十多年的备修院。1981年，该修道院迁到[[北京市|北京市]][[海淀区|海淀区]][[西北旺|西北旺]]，成立'''北京天主教神哲学院'''。1989年6月，该修道院迁至[[西什库教堂|西什库教堂]]后院。1992年9月，该修道院迁至[[朝阳区|朝阳区]][[平房乡|平房乡]][[平房天主堂|平房教堂]]旧址。后来，经过傅铁山主教的努力，[[北京市人民政府|北京市人民政府]]拨款在[[海淀区|海淀区]][[双清路|双清路]]后八家村兴建了一座现代化的修道院。2001年9月，修道院迁入该址。<ref name=tg/><ref>张洪波，神哲学院握手与北京天主教，中国天主教1999年第5期</ref><ref name=ss>张卉，北京天主教神哲学院庆祝建院30周年，中国天主教》2011年第5期</ref>该建筑由[[清华大学建筑学院|清华大学建筑学院]][[沈三陵|沈三陵]]设计。<ref>沈三陵，北京天主教神哲学院，建筑学报1999(3):37-40</ref>

该院的宗旨是“荣主益人，服务人群”。<ref name=tg/>

==后八家天主堂==
为了改变天主教在海淀区没有圣堂的情况，决定在修院开设新的堂区。2003年，成立了后八家圣弥额尔堂区，第一任本堂为培育团的侯天祥神父，这也是海淀区[[改革开放|改革开放]]后第一所正式的天主教堂。<ref name=js>[http://www.catholic-bj.org/content.php?id=5057&fuid=49&tid=49 北京天主教神哲学院圣弥额尔教堂简史，天主教北京教区，2014-05-23] {{webarchive|url=https://web.archive.org/web/20170111044112/http://www.catholic-bj.org/content.php?id=5057&fuid=49&tid=49 |date=2017-01-11 }}</ref><ref>{{Cite web |url = http://www.chinacatholic.org/html/2013/1005/26145.html |title = 北京：后八家弥厄尔堂举行10周年庆典 |author =  |date =  2013-10-05|publisher = 信德网 |language =  |accessdate =  |quote =  }}</ref>

在最初的堂务会管理制度中规定：“北京天主教神哲学院圣弥额尔教堂是北京天主教神哲学院的一部分，神哲学院教堂主要是神哲学院神长，修道生，通过宗教礼仪生活为修生提供牧民实习的场所；同时也是为当地教友提供圣事服务的教堂。”<ref name=js/>
