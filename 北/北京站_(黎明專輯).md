《'''北京站'''》是[[黎明|黎明]]的[[音樂專輯|音樂專輯]]<ref>{{Cite web |url =https://www.weibo.com/ttarticle/p/show?id=2309404182306335121575|title =新力唱片時期《北京站》|author =runriverrun|date = 2017年12月7日 |publisher =微博|language = 中文}}</ref><ref>{{Cite web |url =http://mypaper.pchome.com.tw/ileenho/post/1247516836|title =黎明(Leon Lai)唱片作品年表|date = 2005年5月27日 |publisher =壽司店的肥美亞彌-PChome 個人新聞台|language = 中文}}</ref>，於2000年5月30日推出，由[[黎明|黎明]]及[[雷頌德|雷頌德]]主力製作，專輯收錄了16首歌曲<ref>{{cite web|url= https://mojim.com/twy100171x27x9999.htm|title=北京站專輯歌曲-黎明(Leon)|website=Mojim.com 魔鏡歌詞網|language =中文}}</ref>。有一次黎明回到北京，驚覺原來不知不覺間，北京變化了很大。從前給人感覺落後守舊的城市，已變成一個足以與世界任何先進國家媲美的大都會，促使他產生了這張專輯『北京站』的構思<ref>{{Cite web |url =http://allaboutleon.com/index.cgi?md=read;id=34284|title =如何評價黎明？|date = 2017年4月26日 |publisher =allaboutleon.com|language = 中文}}</ref><ref>{{Cite web |url =https://www.sonymusic.com.tw/album/%E5%8C%97%E4%BA%AC%E7%AB%99-%E9%BB%8E%E6%98%8E-leon-lai-sdd0024/|title =北京站|date = 2000年6月5日 |publisher =台灣索尼音樂娛樂股份有限公司|language = 中文}}</ref><ref>{{cite news |language =中文|author =方晴|title =黎明情繫北京|publisher =大公報|date = 2000年7月6日}}</ref>。
