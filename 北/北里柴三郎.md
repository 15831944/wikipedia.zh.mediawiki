'''北里柴三郎'''（{{jpn|j=北里 柴三郎|hg=きたさとしばさぶろう|rm=Kitasato Shibasaburō}}，[[嘉永|嘉永]]5年12月20日；{{bd|1853年|1月29日|1931年|6月13日|}}），[[大日本帝國|大日本帝國]]的[[醫學家|醫學家]]和[[细菌學家|细菌學家]]。[[從二位|從二位]]・[[勲一等旭日大綬章|勲一等旭日大綬章]]・[[男爵|男爵]]・[[醫學博士|醫學博士]]。

北里是許多重要機構的創始人，包括私立{{link-ja|傳染病研究所|伝染病研究所}}（現・[[東京大學|東京大學]]醫科學研究所）、{{link-ja|土筆岡養生園|土筆ヶ岡養生園}}（現・[[北里大學|北里大學]]{{link-ja|北里研究所醫院|北里研究所病院}}）、私立{{link-ja|北里研究所|北里研究所}}（現・{{link-ja|學校法人北里研究所|学校法人北里研究所}}）。此外也是[[北里大學|北里大學]]校祖、[[慶應義塾大學|慶應義塾大學]]醫學科（現・[[慶應義塾大學醫學部|慶應義塾大學醫學部]]）創始人兼初代科長、{{link-ja|慶應義塾大學醫院|慶應義塾大学病院}}初代院長、[[日本医师会|日本医师会]]創始人兼初代會長。

== 諾貝爾獎提名 ==
北里与[[埃米尔·阿道夫·冯·贝林|埃米尔·阿道夫·冯·贝林]]曾被提名首屆（1901年）[[諾貝爾生理學或醫學獎|諾貝爾生理學或醫學獎]]（最終15人名單內）<ref>[http://www.nobelprize.org/nomination/archive/show_people.php?id=4874 Shibasaburo Kitasako - Nomination Database]</ref>，但只有貝林獲獎<ref>[[讀賣新聞|讀賣新聞]]1988年3月28日</ref>。雖未獲獎，但北里、貝林2人共同開發[[血清|血清]]療法的史實已被確認。日本學界設有貝林・北里獎（ベーリング・北里賞），纪念他的贡献。

2015年，為紀念北里教授而設立的{{link-ja|北里研究所|北里研究所}}、[[北里大學|北里大學]]分別迎接100週年、50週年之後的第一年，該校教授[[大村智|大村智]]獲得日本人第23個[[諾貝爾獎|諾貝爾獎]]、第3個諾貝爾生理學或醫學獎。理事長、校長在祝賀聲明中，視之為「北里的繼承人」：
{{Cquote|北里研究所的創始人北里柴三郎博士，曾在德國成功培養[[破傷風|破傷風]]菌、確立[[血清|血清]]療法，名列首屆諾貝爾生理學或醫學獎候選人，卻留下反常落選的遺憾。如今超過一世紀後，大村教授獲得了諾貝爾獎，可謂繼承了北里博士的學統，令人感慨萬千。<ref>[https://web.archive.org/web/20151006074824/https://www.kitasato-u.ac.jp/ 北里大學2015年10月6日首页]</ref>}}

前[[理化學研究所|理化學研究所]]所長、[[諾貝爾化學獎|諾貝爾化學獎]]得主[[野依良治|野依良治]]評論：「（大村智的獲獎）達成了北里柴三郎教授的悲願」。<ref>[https://www.youtube.com/watch?v=EWUEL0WTRmc 高い志にチャンス 大村智氏　野依氏と対談 - YouTube]</ref>

== 经历 ==
* [[肥后国|肥后国]]北里村（现在的[[熊本县|熊本县]][[阿苏郡|阿苏郡]][[小国町|小国町]]）世代相传的庄园的家里出生。
* 在当地的熊本医学校和[[东京帝国大学|东京帝国大学]][[医学部|医学部]]接受教育，毕业后在[[内务省|内务省]]卫生局供职。
* 1885年-1891年，从师于[[柏林大学|柏林大学]]教授[[罗伯特·科赫|罗伯特·科赫]]。
* 1889年，與[[埃米爾·阿道夫·馮·貝林|埃米爾·阿道夫·馮·貝林]]一起成功世界上最初的[[破傷風|破傷風]]病菌的純粋培養。
