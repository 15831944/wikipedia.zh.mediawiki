'''北京地铁DKZ4型电动车组'''是[[北京地铁|北京地铁]]的[[电动车组|电动车组]]车款之一，现在在[[北京地铁1号线|1号线]]运营。

== 简介 ==
为了应对[[北京地铁复八线|复八线]]的开通，北京地铁于1996年开始订购DKZ4型电动客车。(DKZ4型电动车组由DK28、DK29、DK30、DK31组成)DKZ4型一共186辆，于1998年生产，其中有114辆（编号S401-S419）在[[長春軌道客車股份有限公司|长春轨道客车]]生产，72辆（编号S420-S431）在[[北京地铁车辆厂|北京地铁车辆厂]]生产，已于1999年开始运行于[[北京地铁复八线|复八线]]，后在2000年全程运行于[[北京地铁1号线|1号线]]，本系列列车为北京地铁的第1款[[VVVF|VVVF]]控制的列车。目前，列车配属于[[四惠车辆段|四惠车辆段]]和[[古城车辆段|古城车辆段]]。

== 列车内部 ==
本列车为鼓型车体，车门为气动[[内藏门|内藏门]]，每节车厢共4对车门；另外每节车厢连接处有车门连接，可自动关闭。车内安装有LED显示屏、闪灯图和液晶电视。驾驶室内的主干控制器为手柄式，为牵引4段+制动7段。

== 搭载设备 ==
列车的控制设备为[[可关断晶闸管|GTO]]-[[VVVF|VVVF]]，其中[[VVVF|VVVF]]输出为1200KVA，[[可关断晶闸管|GTO]]为4500V/4000A。逆变器为40KVA的静式变流器，可将直流750V逆变为交流220V和直流DC110V、DC24V。

== 空调改造 ==
为迎接[[2008年北京夏季奥运会|2008年北京夏季奥运会]]，列车于2007年开始在[[北京地铁车辆厂|北京地铁车辆厂]]进行改造<ref>[http://www.bjjsyc.org.cn/template/10002/file.jsp?cid=24&aid=11895 北京市技术总工会英才网:北京地铁DKZ4型电动客车局部技术改造]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }} 2012年8月29日查看</ref>，外部由白色涂装变为了银灰色涂装，机电系统进行了一系列改造。由于早期技术方面所限，列车内部只安装有电风扇，並沒有安裝空調設備<ref>[http://www.gmw.cn/content/2010-06/25/content_1160757.htm 北京地铁不同线路温差大] 光明网 2010-06-25</ref>。为加速列车空调化，因此在2010年，DKZ4型列车开始在[[北京地铁车辆装备有限公司|北京地铁车辆装备有限公司]]加装空调（每节车厢拥有4台制冷量为12kW的松芝KLDL12AAA空调机组）<ref>[http://www.bjsubway.com/node/2342 北京地铁:1号线非空调车改造正在加紧进行] {{webarchive|url=https://web.archive.org/web/20111103145208/http://bjsubway.com/node/2342 |date=2011-11-03 }} 2012年12月20日查看</ref>和内部改造（加装闪灯图和LED显示屏，并对大部分车辆的车窗改为封闭式），最后更名为DKZ4G（最初进行试验的列车为S423和S405）<ref>[http://news.sohu.com/20100807/n274054629.shtml 搜狐网:北京地铁1号线老列车开始装空调 改造陆续开始] 2010年8月7日查看</ref>，直至2012年全部完成。

== 信号改造 ==
为了与1号线新CBTC信号系统兼容调试，原来的DKZ4车组都将进行信号改造，改造内容包括更改操作台，在车头加上CBTC信号接收器，更改控制电路，以及将原有GTO-VVVF更改为東洋電機IGBT-VVVF。初期将原有ATP柜放置于客室，当一号线CBTC系统投入使用时原有ATP柜子全部拆除。第1辆改造车为S402。目前所有车已全部改造完畢。

*四惠车辆段：S401-S429
*古城车辆段：S430-S431

==事故==
2009年6月28日上午9时36分左右，一列DKZ4型列车在[[西单站|西单]]至[[复兴门站_(北京)|复兴门]]之间发生故障<ref>[http://news.sohu.com/20090629/n264823069.shtml 搜狐新闻:北京地铁1号线3次爆响亮光 下行方向停运半小时] 2009年6月29日</ref>，导致在近半小时后线路运营才恢复正常。

==其他==
2013年5月，编号为S402的列车发生故障并退出运营，封存于[[四惠车辆段|四惠车辆段]]中。经信号系统与控制台改造后于2015年2月再次上线运行。
