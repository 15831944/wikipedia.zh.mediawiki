'''北京政变'''，也稱為'''首都革命'''、'''北京事變'''，1924年10月23日发生在[[北洋政府|中华民国]][[北洋政府|北洋政府]]的首都[[北京|北京]]，发动者为[[冯玉祥|冯玉祥]]、[[胡景翼|胡景翼]]、[[孙岳|孙岳]]。

== 背景 ==
1924年9月，[[第二次直奉战争|第二次直奉战争]]爆发。冯玉祥被任命为“讨逆军”第三军总司令，出[[古北口|古北口]]迎战热河方向的奉军。十月份，直系吳佩孚率重兵與奉系張作霖正在山海關激戰不休，陷入僵局。孙中山曾指示[[于右任|于右任]]、[[焦易堂|焦易堂]]、[[徐季龙|徐季龙]]等秘密与[[冯玉祥|冯玉祥]]、[[胡景翼|胡景翼]]、[[孙岳|孙岳]]联系，对其做争取工作。孙中山送给冯玉祥六千本《三民主义》、一千本《[[建国方略|建国方略]]》和《建国大纲》。冯玉祥将这些书发给其官兵阅读学习。

==经过==
10月23日，冯玉祥率部返回北京，包围了总统府，[[劫持|劫持]]了時任[[大總統|大總統]]的直系[[曹錕|曹錕]]，将其囚禁在[[团城|团城]]，推倒了颜惠庆内阁。迫使[[直系|直系]]控制的[[北洋政府|北京政府]]下令停战并解除[[吴佩孚|吴佩孚]]的职务，同屬直系的[[孫傳芳|孫傳芳]]（浙閩分支）早與馮玉祥暗中達成協議，選擇作壁上觀，任憑馮玉祥监禁大总统曹锟，宣布成立“[[西北軍|国民军]]”，史稱「北京事變」或「首都革命」。

此一事件之後，北洋政府聲勢威望大跌，也才讓日後蔣介石為首的北伐軍有可趁之機。

== 后事 ==
政变后，冯玉祥授意摄政内阁通过了《修正清室优待条件》，废除帝号，清室迁出[[紫禁城|紫禁城]]，驱逐[[溥仪|溥仪]]和[[婉容|婉容]]及嫔妃出宫。溥儀於1924年11月29日偕同[[鄭孝胥|鄭孝胥]]、[[陳寶琛|陳寶琛]]逃往日本使館；1925年2月，又赴天津。但冯玉祥无法控制局面，遂邀请南方廣州軍政府的[[孙中山|孙中山]]北上商议。随后又与[[奉系军阀|奉系军阀]]达成协议，请[[段祺瑞|段祺瑞]]入主北京，任中华民国执政。孙中山北上后即病，1925年3月12日病逝于北京。段祺瑞继续主持善后会议，会后成立临时参政院，行使国会职能，其後解散[[中華民國第一屆國會|中華民國第一屆國會]]，並正式廢除[[中華民國臨時約法|中華民國臨時約法]]。

1925年11月冯玉祥策动[[郭松龄|郭松龄]]叛奉，郭失败，奉军入关。1926年4月9日冯玉祥驱逐[[段祺瑞|段祺瑞]]，随即被[[奉系軍閥|奉系]]击败，撤出北京，在[[南口大战|南口大战]]中坚持了3月，后撤回至西北偏僻地区。

== 評價 ==
* [[章太炎|章太炎]]說：“六年溥儀[[張勛復辟|復辟]]，則優待條件自消。”
* [[胡適|胡適]]於信中說：“我對於此次政變，還不曾說過話；今天感於一時的衝動，不敢不說幾句不中聽的話。”又說：“我是不贊成清室保存帝號的，但清室的優待乃是一種國際的信義、條約的關係。條約可以修正，可以廢止，但堂堂的民國，欺人之弱，乘人之喪，以強暴行之，這真是民國史上一件最不名譽的事。”<ref>中華書局版《胡適來往書信選》</ref>
* 中國歷史學者[[余杰|余杰]]曾說：「馮玉祥發動北京事變，有如[[張學良|張學良]]發動[[西安事變|西安事變]]，乃是中國二十世紀上半葉兩大劫難。」<ref>{{Cite book|title=1927：民國之死|last=余|first=杰|publisher=八旗文化|year=2017|isbn=9789869423175|location=新北|pages=30}}</ref>
