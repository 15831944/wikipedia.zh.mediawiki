'''北京金融街丽思卡尔顿酒店'''，位于[[北京市|北京市]][[西城区|西城区]][[金城坊东街|金城坊东街]]1号，是[[北京金融街|北京金融街]]区域内的豪华酒店。

==简介==
北京金融街里兹置业有限公司北京金融街丽思卡尔顿酒店于2006年9月20日成立，是五星级酒店<ref name=wsh>{{Cite web |url = http://news.sina.com.cn/o/2018-01-08/doc-ifyqinzt0473718.shtml |title = 北京金融街丽思卡尔顿酒店因违反《公共场所卫生管理条例》被警告 |author =  |date =2018-01-08  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。这是[[麗思卡爾頓酒店管理集團|丽思卡尔顿酒店集团]]旗下又一五星级酒店，也是其在中国开设的第三家酒店（另两家在[[上海|上海]]和[[香港|香港]]）<ref>{{Cite web |url =  http://finance.sina.com.cn/roll/20060921/0406937769.shtml|title = 丽思卡尔顿酒店跻身北京金融街 |author =  |date =2006-09-21  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

2016年7月，[[中华人民共和国国家发展和改革委员会|中华人民共和国国家发展和改革委员会]]披露消息称，已查实上海新世界淮海物业发展有限公司、沈阳恒隆地产有限公司、北京新捷房地产开发有限公司、中国国际贸易中心股份有限公司、北京新云南皇冠假日酒店、北京金融街丽思卡尔顿酒店6家企业以“[[营改增|营改增]]”为借口，捏造涨价信息、哄抬价格，已于近日责成[[北京市|北京市]]、[[上海市|上海市]]、[[辽宁省|辽宁省]]的价格主管部门依法作出行政处罚，对6家企业各罚款200万元<ref>{{Cite web|url = http://travel.ce.cn/gdtj/201607/29/t20160729_4113192.shtml|title = 北京金融街丽思卡尔顿酒店营改增涨价被罚200万元|author = |date = 2016-07-29|publisher = |language = |accessdate = |quote = }}{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2017年3月，[[沙特阿拉伯国王|沙特阿拉伯国王]][[萨勒曼·本·阿卜杜勒-阿齐兹·阿勒沙特|萨勒曼]]访华，陪同来访的大批沙特阿拉伯王室成员及官员入住该酒店<ref>{{Cite web |url =http://news.163.com/17/0317/14/CFO3PQ1R0001899O.html  |title = 网曝20名沙特王子入住北京5星酒店 妹子机会来了? |author =  |date = 2017-03-17  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2018年1月6日，北京市卫生和计划生育监督所发布的2017年12月《北京卫生监督行政处罚公示》称，北京金融街丽思卡尔顿酒店因为违反《公共场所卫生管理条例》第十四条第一款第（一）项相关规定而被警告<ref name=wsh/>。
