==型号==
===第一代===
在1960、1971年代，北京地铁使用了长春的[[北京地铁DK2型电动车组|DK2]]和[[北京地铁DK3型电动车组|DK3型]]列车。其中的'''DK'''指的是''电动客车diandong keche''.<ref name="autogenenerated1">[http://world.nycsubway.org/world/cn/beijing.html "Robert McConnell's Beijing Subway Webpage"] Accessed Mar. 27,2010{{En}}</ref> 这些型号的列车以及他们的变体，也就是[[北京地铁DK2型电动车组|DK3G]]、[[北京地铁DK20型电动车组|DK20]]、[[北京地铁DK16型电动车组|DK16A]]、[[北京地铁BD1型电动车组|BD1]]和[[北京地铁BD2型电动车组|BD2型]]被统称为北京地铁的第一代列车。<ref>{{zh}} [http://www.bjsubway.com/zjdt/gsyw/dtyy/clsb/cheliang/2008-09-25/3802.shtml 第一代电动客车] {{webarchive|url=https://web.archive.org/web/20091231033650/http://www.bjsubway.com/zjdt/gsyw/dtyy/clsb/cheliang/2008-09-25/3802.shtml |date=2009-12-31 }} Accessed Mar. 27, 2010</ref> 近些年来，[[北京地铁车辆装备有限公司|北京地铁车辆装备有限公司]]翻新了[[北京地铁DK16型电动车组|DK16A]]和[[北京地铁DK20型电动车组|DK20型]]列车，所以他们在21世纪仍然在使用。<ref>{{zh}} [http://www.bjsubway.com/zjdt/gsyw/dtyy/clsb/cheliang/2008-09-26/3806.shtml 厂修车介绍] {{webarchive|url=https://web.archive.org/web/20091231065031/http://www.bjsubway.com/zjdt/gsyw/dtyy/clsb/cheliang/2008-09-26/3806.shtml |date=2009-12-31 }} Accessed Mar. 27, 2010</ref> 翻新过的DK16AG列车2005年投入2号线运营。<ref>{{zh}}[http://www.brsp.cn/Product/index_detail.asp?ArticleID=90 DK16大修改造车] {{webarchive|url=https://web.archive.org/web/20071014131517/http://brsp.cn/Product/index_detail.asp?ArticleID=90 |date=2007-10-14 }} Accessed Mar. 27, 2010</ref>

===第二代===
从1980年代至1990年代早期，北京地铁引入了新的车辆，包括的型号有[[北京地铁DK6型电动车组|DK6]]、[[北京地铁DK9型电动车组|DK9]]以及他们的变体[[北京地铁DK11型电动车组|DK11]]、[[北京地铁DK16型电动车组|DK16]]和[[北京地铁GTO型电动车组|GTO]]。<ref>{{zh}} [http://www.bjsubway.com/zjdt/gsyw/dtyy/clsb/cheliang/2008-09-26/3804.shtml 第二代电动客车] {{webarchive|url=https://web.archive.org/web/20091231033719/http://www.bjsubway.com/zjdt/gsyw/dtyy/clsb/cheliang/2008-09-26/3804.shtml |date=2009-12-31 }} Accessed Mar. 27, 2010</ref>另外还引入了[[M型电动车组|M-1000]]系列列车投入到2号线和13号线运营，制造商是日本的[[东急车辆制造|东急车辆制造]]株式会社.<ref>{{zh}} [http://www.bk670.com/m/frame.php?frameon=yes&referer=http%3A//www.bk670.com/m/redirect.php%3Ftid%3D376%26goto%3Dlastpost "东急"] {{webarchive|url=https://web.archive.org/web/20160417000159/http://www.bk670.com/M/frame.php?frameon=yes |date=2016-04-17 }} Accessed Mar. 28, 2010</ref><ref>{{zh}} [http://www.bjsubway.com/zjdt/gsyw/dtyy/clsb/cheliang/2008-09-26/3810.shtml "M车"] {{webarchive|url=https://web.archive.org/web/20091231064928/http://www.bjsubway.com/zjdt/gsyw/dtyy/clsb/cheliang/2008-09-26/3810.shtml |date=2009-12-31 }} Accessed Mar. 28, 2010</ref><ref name="autogenerated2">[http://rail.kychung.com/?tag=長春 "Nanyue Express"] {{webarchive|url=https://web.archive.org/web/20160304040947/http://rail.kychung.com/?tag=%E9%95%B7%E6%98%A5 |date=2016-03-04 }}</ref>

===第三代===
在1998年，北京地铁开始投入第三代客车，特点为[[VVVF|VVVF]]控制方式. 这些类型有[[北京地铁DKZ4型电动车组|DKZ4]]、[[北京地铁DKZ5型电动车组|DKZ5]]和[[北京地铁DKZ6型电动车组|DKZ6]]。<ref>{{zh}} [http://www.bjsubway.com/zjdt/gsyw/dtyy/clsb/cheliang/2008-09-26/3805.shtml第三代电动客车] Accessed Mar. 27, 2010</ref>  '''DKZ'''代表''电动客车组diandong keche zu'' ，也就是[[动车组|动车组]].<ref name=autogenenerated1 />  长春RVC公司还制造了174辆 DK28-DK31地铁客车，使用[[VVVF|VVVF]]控制设备和交流电动机，投入1号线运行。<ref>[http://www.chinacnr.com/LISTS/product/_MAINPAGE/product/default2.aspx?classid=124 "DK28 Metro Car with VVVF Inverter"] {{webarchive|url=https://web.archive.org/web/20160303201129/http://www.chinacnr.com/LISTS/product/_MAINPAGE/product/default2.aspx?classid=124 |date=2016-03-03 }} Accessed Mar. 27, 2010</ref> 以及136辆 DK32-34 投入13号线。<ref>[http://www.chinacnr.com/LISTS/product/_MAINPAGE/product/default2.aspx?classid=125 "DK32Commuter Train for Beijing"]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }} Accessed Mar. 27, 2010</ref>  在2005年，八通线开始使用青岛四方制造的[[北京地铁SFX01型电动车组|SFX01]]和[[北京地铁SFX02型电动车组|SFX02]]列车。<ref name=autogenenerated1 />
40辆机场快线的列车是长春RVC 和[[庞巴迪运输|庞巴迪运输]]的合资公司生产的，使用了庞巴迪的直线电机技术。.<ref name=autogenerated2 /><ref>[http://www.allbusiness.com/transportation/air-transportation-airports/5382241-1.html "Bombardier Awarded Contract in China for the Beijing Capital International Airport Link With Connection to Summer 2008 Olympic Games Village" ''Business Wire''] Mar. 20, 2006</ref>

==车况==
目前北京地铁1号线、2号线、4号线、5号线、8号线、9号线、10号线、14号线、15号线、大兴线、昌平线、房山线、亦庄线为6节车厢编组，八通线和13号线原为4节车厢编组，现在也已经扩编成6节{{RefTag|{{cite web |url=http://news.xinhuanet.com/society/2008-08/08/content_9041982.htm |title=北京地铁2号线全部更换空调车 八通线扩编完成 |date=2008-08-08 |accessdate=2009-08-02}}}}{{RefTag|{{cite web |url=http://www.chinanews.com.cn/olympic/news/2008/07-09/1307438.shtml |title=北京地铁流量迫近400万 计划新车上线城铁扩编 |date=2008-07-09 |accessdate=2009-02-04}}}}{{RefTag|{{cite web |url=http://beijing.qianlong.com/3825/2009/03/26/4202@4922070.htm |title=13号线列车扩改任务全部完成 增至六节车厢 |date=2009-03-26 |accessdate=2009-08-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20090331120438/http://beijing.qianlong.com/3825/2009/03/26/4202%404922070.htm |archivedate=2009-03-31 }}}}。机场线为4节车厢编组{{RefTag|{{cite web|url=http://www.beijing2008.cn/news/olympiccities/beijing/n214211924.shtml|title=首个无人驾驶轨道交通线 机场线列车开始动车调试|date=2007-02-18|publisher=[[北京奥组委|北京奥组委]]|accessdate=2009-02-04|deadurl=yes|archiveurl=https://web.archive.org/web/20090318052219/http://www.beijing2008.cn/news/olympiccities/beijing/n214211924.shtml|archivedate=2009-03-18}}}}，6号线，7号线，16号线为8节车厢编组。除机场线、14号线、16号线{{RefTag|{{cite web|url=http://www.bjzdb.gov.cn/jsxx/t1092807.htm|title=北京地铁十四号线工程列车编组7B改6A方案专题报告论证会召开|publisher=[[北京市重大项目建设指挥部办公室|北京市重大项目建设指挥部办公室]]|date=2010-12-30|accessdate=2011-02-04|deadurl=yes|archiveurl=https://web.archive.org/web/20131215224728/http://www.bjzdb.gov.cn/jsxx/t1092807.htm|archivedate=2013-12-15}}}}外的线路采用B型列车{{RefTag|name=mtr16atrain|{{cite web|url=http://news.sina.com.cn/c/2010-11-26/012821534425.shtml|title=北京地铁16号线有望用A型车 每趟多运500人|date=2010-11-26|accessdate=2011-02-04}}}}。虽然北京地铁采用缩小间隔的方法提高运力，但是在低票价所带来的大客流量之下，6节编组的B型车辆内仍显十分拥挤{{RefTag|{{cite web|url=http://www.china.com.cn/news/txt/2007-10/26/content_9126848.htm|title=北京地铁成市民出行首选 高峰时挤得像罐头|date=2007-10-26|accessdate=2009-08-12}}}}。{{RefTag|name=mtr16atrain}}

大部分列车内设有[[LED|LED]]显示屏和LED指示灯线路图，可以显示列车运行前方的车站。大部分列车配备[[液晶显示屏|液晶显示屏]]，可以转播电视节目{{RefTag|{{cite web |url=http://sports.sina.com.cn/o/2007-11-07/14193277117.shtml|title=千人试乘北京地铁新车 车载显示屏奥运期间可看比赛 |accessdate=2011-11-30|date=2007-11-07}}}}。

北京地铁列车的空调设备目前存在问题。不同的线路之间空调所设定温度不同，使得乘客在换乘时会感到不适。而1号线旧型列车的电扇在拥挤的车厢内则几乎不能起到通风降温作用{{RefTag|{{cite web |url=http://www.gmw.cn/content/2010-06/25/content_1160757.htm|title=北京地铁不同线路温差大|date=2010-06-25|publisher=光明网|accessdate=2010年7月7日}}}}，不过随着旧型列车的退役，目前的北京地铁列车均拥有空调设备。
