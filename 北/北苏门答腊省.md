'''北苏门答腊省'''（[[印尼语|印尼语]]：Sumatera Utara）是[[印度尼西亚|印度尼西亚]]省份之一。它的省府是[[棉兰|棉兰]]。該省面積共71,680平方公里，西北與[[亞齊省|亞齊省]]接壤，南面與[[西蘇門答臘省|西蘇門答臘省]]相連，東南與[[廖內省|廖內省]]為鄰。北蘇門答臘省的管轄範圍包含遠處於印度洋的多個[[島嶼|島嶼]]，包括尼亞斯群島（Nias）。

== 地理 ==
北蘇門答臘省位處[[蘇門答臘島|蘇門答臘島]]的北部，東西分別面瀶[[馬六甲海峽|馬六甲海峽]]和[[印度洋|印度洋]]。全省東部大部份地區為低地平原，在西岸和南部地勢較高，巴拉神山脈（Barisan Mountains）縱貫其中。[[多巴湖|多巴湖]]是位在該省的一個知名大湖泊，由一個古代死[[火山|火山]]所形成。

== 人口 ==
北蘇門答臘省約有人口11,490,453人（2005年統計數字）。但經歷2004年12月的[[2004年印度洋大地震|南亞地震]]，該省和亞齊特別行政區首當其衝，有近萬人死亡。

== 农业和经济 ==
北苏门答腊省种植曼德海令咖啡豆和林东咖啡豆，大部分出口到美国。该名称是源于居住在当地的曼特宁人名称的近似拼写。这个名称是众多外国购买者中的第一位对其名称误解后而给起的，并且没有一个咖啡品种是产自“曼特宁地区”。而另一方面，林东咖啡，是以林东地区而命名的，该地区也位于北苏门答腊。
