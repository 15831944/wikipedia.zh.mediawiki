'''北海道道625号学园新十津川停车场线'''（{{lang-ja|北海道道625号学園新十津川停車場線}}）是一条位于[[北海道|北海道]][[桦户郡|桦户郡]][[新十津川町|新十津川町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：北海道桦户郡新十津川町字学園（＝[[国道451号|国道451号]]交点）
* 終点：北海道桦户郡新十津川町字中央（[[北海道旅客铁道|JR北海道]][[新十津川站|新十津川站]]）
* 总距离：6.4千米

== 经过的自治体  ==
* [[空知综合振兴局|空知综合振兴局]]
** 桦户郡新十津川町

== 主要连接道路 ==
* [[国道451号|国道451号]]（起点）
* [[国道275号|国道275号]]（新十津川町字中央）

== 主要橋梁 ==
* 新十津川町
** 学总橋（德富川）＝新十津川町字学園-新十津川町字总進
** 总富地橋（总富地川）＝新十津川町字总進-新十津川町字中央

== 历史 ==
* 1969年（[[昭和|昭和]]44年）6月18日 - 路線勘定（北海道告示第1249号）
