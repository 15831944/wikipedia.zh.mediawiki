'''北狩'''，古代[[漢語|漢語]][[委婉|委婉]]詞彙，通常用於描述某[[朝代|朝代]]的皇帝因為某些原因而被外族軍隊俘虜的事件。

==意思==
漢語裡，「[[wikt:北|北]]」為方位<ref name="TWMoE1">{{cite web |title=北 |url=http://dict.mini.moe.edu.tw/cgi-bin/gdic/gsweb.cgi?o=ddictionary&searchid=D0003898&framefile=frame04.htm& |website=國語小字典 |publisher=[[中華民國|中華民國]][[中華民國教育部|教育部]][[國家教育研究院|國家教育研究院]] |accessdate=2018-11-20}}</ref>，「[[wikt:狩|狩]]」有打獵<ref name="TWMoE2">{{cite web |title=狩 |url=http://dict.mini.moe.edu.tw/cgi-bin/gdic/gsweb.cgi?o=ddictionary&searchid=D0004071&framefile=frame04.htm& |website=國語小字典 |publisher=[[中華民國|中華民國]][[中華民國教育部|教育部]][[國家教育研究院|國家教育研究院]] |accessdate=2018-11-20}}</ref>或天子出巡視察<ref name="TWMoE2"/>的意思。

從字面看，「北狩」一詞是「在北方打獵」或「天子在北方出巡視察」的意思，但歷史上，由於中國皇帝成為外族戰俘是一件不光彩的事，所以「北狩」成為了委婉詞，用於描述皇帝被俘虜之事<ref>{{cite web |title=北狩詞語解釋 / 北狩是什麽意思 |url=http://www.chinesewords.org/dict/49546-96.html |website=漢語詞典 |accessdate=2018-11-20}}</ref>。

==使用==
歷史上，[[宋朝|宋朝]][[靖康之變|靖康之變]]和[[明朝|明朝]][[土木之變|土木堡之變]]都有外族軍隊俘虜中國皇帝的事件發生。描述該時代的古書都把這兩次皇帝被俘虜的事件寫成北狩，以便避諱。

===宋朝===
[[南宋|南宋]]學者[[王明清|王明清]]於[[揮麈錄|揮麈錄]]第四卷中以「逮二聖北狩」提及[[宋徽宗|徽宗]]趙佶和[[宋欽宗|欽宗]]趙桓被[[金朝|金朝]]軍隊俘虜之事<ref>{{cite web |author1=[[王明清|王明清]] |title=揮麈後錄/04 |url=https://zh.wikisource.org/wiki/%E6%8F%AE%E9%BA%88%E5%BE%8C%E9%8C%84/04 |website=[[維基文庫|維基文庫]] |accessdate=2018-11-20|quote=逮二聖北狩，彭以無名位，獨得留內庭。}}</ref>。

[[明朝|明代]]作家熊大木的「大宋中興通俗演義」中第五回是描述徽宗和欽宗被金朝俘虜之事。該回名為「宋徽欽北狩沙漠」<ref>{{cite web |author1=熊大木 |title=大宋中興通俗演義/05 |url=https://zh.wikisource.org/wiki/%E6%8F%AE%E9%BA%88%E5%BE%8C%E9%8C%84/04 |website=[[維基文庫|維基文庫]] |accessdate=2018-11-20}}</ref>。

於[[清朝|清朝]]時編寫的[[御定歴代紀事年表|御定歴代紀事年表]]中提及北宋歷史時亦使用了北狩一詞<ref>{{cite journal |title=御定歴代紀事年表 (四庫全書本)/卷081 |journal=御定歴代紀事年表 |volume=81 |url=https://zh.wikisource.org/wiki/%E5%BE%A1%E5%AE%9A%E6%AD%B4%E4%BB%A3%E7%B4%80%E4%BA%8B%E5%B9%B4%E8%A1%A8_(%E5%9B%9B%E5%BA%AB%E5%85%A8%E6%9B%B8%E6%9C%AC)/%E5%8D%B7081 |accessdate=2018-11-20 |quote=...國號宋都於汴傳位太宗至徽欽北狩歴九主凡一百六十八年...}}</ref>。

===明朝===
明代晚期文人[[呂毖|呂毖]]於[[明朝小史|明朝小史]]中提及[[明英宗|明英宗]]朱祁鎮被[[瓦剌|瓦剌]]軍隊俘虜的事件時使用了北狩一詞<ref>{{cite web |author1=[[呂毖|呂毖]] |title=明朝小史/卷07 |url=https://zh.wikisource.org/wiki/%E6%98%8E%E6%9C%9D%E5%B0%8F%E5%8F%B2/%E5%8D%B707 |website=[[維基文庫|維基文庫]] |accessdate=2018-11-20|quote=帝北狩時在虜中，虜中人欲謀害之...}}</ref>。

於[[清朝|清代]]編寫的明朝史書[[明史|明史]]中，明英宗被虜一事僅僅以「帝北狩」三字提及。<ref>{{cite web |author1=[[明史館|明史館]] |title=明史 卷十 本紀第十 英宗前紀 |url=https://zh.wikisource.org/wiki/%E6%98%8E%E5%8F%B2/%E5%8D%B710 |website=[[維基文庫|維基文庫]] |accessdate=2018-11-22 |location=[[清朝|大清國]] |date=1739年 |quote=辛酉，次土木，被圍。壬戌，師潰，死者數十萬。英國公張輔，泰寧侯陳瀛，駙馬都尉井源，平鄉伯陳懷，襄城伯李珍，遂安伯陳塤，修武伯沈榮，都督梁成、王貴，尚書王佐、鄺埜，學士曹鼐、張益，侍郎丁鉉、王永和、副都御史鄧棨等，皆死，帝北狩。}}</ref>

清代文人[[谷應泰|谷應泰]]於[[明史紀事本末|明史紀事本末]]亦以北狩一詞來形容明英宗被虜一事。<ref>{{cite web |author1=[[谷應泰|谷應泰]] |title=明史紀事本末/卷35 |url=https://zh.wikisource.org/wiki/%E6%98%8E%E5%8F%B2%E7%B4%80%E4%BA%8B%E6%9C%AC%E6%9C%AB/%E5%8D%B735 |website=[[維基文庫|維基文庫]] |accessdate=2018-11-20 |date=[[1658年|1658年]] |quote=上皇既北狩，陰遣使諭鎮守太監裴富...}}</ref>

==現代==
今日，「北狩」一詞已經成為歷史詞彙。現代書籍及文章提及宋徽宗、宋欽宗和明英宗被敵軍俘虜時會直接使用「俘虜」<ref>{{cite news |author1=江仲淵 |title=亡國之君的另一面：宋徽宗的極品文青生活 |url=https://www.thenewslens.com/article/102786 |accessdate=2018-11-23 |work=[[關鍵評論網|關鍵評論網]] |quote=許多人對宋徽宗的第一印象很差，因為他在普通教科書上只留下了三項事蹟：      亡國代言人：被金兵俘虜，亡了北宋...}}</ref><ref>{{cite news |author1=程水石 |title=宋徽宗的凄惨下场：被金兵焚尸 嫔妃公主沦为“慰安妇” |url=http://history.people.com.cn/n/2015/0424/c372327-26900179.html |accessdate=2018-11-23 |work=人民網（轉載自[[中國青年報|中國青年報]]） |date=2015-04-24 |language=[[簡化字|簡體中文]] |quote=金人的俘虏包括北宋帝国的两位皇帝宋徽宗和宋钦宗...}}</ref><ref>{{cite book |author1=欒保羣 |editor=夏雪 |title=歷代皇帝軼事 |date=2003年 |publisher=和平圖書有限公司 |location=香港 |isbn=962-238-340-8 |page=379頁 |edition=2003年10月第2版 |accessdate=2018-11-22 |quote=...連自己也成了瓦剌人的俘虜。}}</ref><ref>{{cite news |title=朱祁鎮哥倆確實撕得狠 但「明妃」只是個傳說 |url=http://culture.people.com.cn/n1/2016/0303/c22219-28167123.html |accessdate=2018-11-22 |work=人民網（轉載自[[現代快報＋|現代快報＋]]） |date=2016-03-03 |language=[[簡化字|簡體中文]] |quote=正統十四年（1449），土木之變發生後，朱祁鎮當了瓦剌人的俘虜。}}</ref><ref>{{cite news |title=歷史歷奇 最有情義的皇帝明英宗 |url=http://www.metrodaily.hk/metro_news/%E6%AD%B7%E5%8F%B2%E6%AD%B7%E5%A5%87-%E6%9C%80%E6%9C%89%E6%83%85%E7%BE%A9%E7%9A%84%E7%9A%87%E5%B8%9D%E6%98%8E%E8%8B%B1%E5%AE%97/ |accessdate=2018-11-22 |work=[[都市日報|香港都市日報]] |date=2018-01-23 |quote=明朝正統十四年(1449年)，英宗聽信讒言御駕親征，在「土木堡事變」中被蒙古瓦剌人俘虜。}}</ref>或「戰俘」<ref>{{cite news |title=【歷史長河】高級戰俘明英宗的塞外生活 |url=https://www.bastillepost.com/hongkong/article/1204369-%E3%80%90%E6%AD%B7%E5%8F%B2%E9%95%B7%E6%B2%B3%E3%80%91%E9%AB%98%E7%B4%9A%E6%88%B0%E4%BF%98%E6%98%8E%E8%8B%B1%E5%AE%97%E7%9A%84%E5%A1%9E%E5%A4%96%E7%94%9F%E6%B4%BB |accessdate=2018-11-22 |work=[[巴士的報|巴士的報]] |date=2016-06-20 |quote=...於是明英宗在宦官王振慫恿下，親率五十萬大軍迎敵，結果兵敗土木堡，皇帝成了戰俘。}}</ref>二詞。
