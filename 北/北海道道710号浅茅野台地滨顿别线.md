'''北海道道710号浅茅野台地滨顿别线'''（{{lang-ja|北海道道710号浅茅野台地浜頓別線}}）是一条连结[[北海道|北海道]][[猿拂村|猿拂村]]和[[滨顿别町|滨顿别町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：北海道[[宗谷郡|宗谷郡]][[猿拂村|猿拂村]]浅茅野台地（＝[[国道238号|国道238号]]交点）　
* 終点：北海道[[枝幸郡|枝幸郡]][[滨顿别町|滨顿别町]]仁達内（＝[[北海道道84号丰富滨顿别线|北海道道84号丰富滨顿别线]]交点）
* 总長：10.0千米

== 经过的自治体  ==
* [[宗谷综合振兴局|宗谷综合振兴局]]
** 宗谷郡猿拂村
** 枝幸郡滨頓別町

== 主要连接道路 ==
* 猿拂村
** [[国道238号|国道238号]]
* 滨頓別町
** [[北海道道84号丰富滨顿别线|北海道道84号丰富滨顿别线]]

== 历史 ==
* 1971年3月31日 路線勘定。<!--1971年北海道告示第1000号-->

== 景点 ==
* [[クッチャロ湖|クッチャロ湖]]＝滨頓別町（在秋季和春季，因有许多[[天鹅|天鹅]]的到来而成为十分具有人气的景点）
* 三叶草山公園＝滨頓別町（有可以远望クッチャロ湖的观景台）
