'''北海道道320号旅来丰顷停车场线'''（{{lang-ja|北海道道320号旅来豊頃停車場線}}）是一条位于[[北海道|北海道]][[十胜综合振兴局|十胜综合振兴局]]管内的[[丰顷町|丰顷町]]内的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 指定区間 ==
* 起点：[[北海道|北海道]][[中川郡_(十勝国)|中川郡]][[丰顷町|丰顷町]]旅来（＝[[北海道道911号大津旅来线|北海道道911号大津旅来线]]・旧[[国道336号#渡船国道|国道336号]]交点）
* 終点：北海道中川郡丰顷町丰顷旭町（＝[[北海道道73号带广浦幌线|北海道道73号带广浦幌线]]交点）
* 总長：13.2千米

== 共线区间 ==
* 丰顷町茂岩荣町-丰顷町茂岩本町（[[北海道道210号尾田丰顷停车场线|北海道道210号尾田丰顷停车场线]]）
* 丰顷町育素多-丰顷町丰顷（[[国道38号|国道38号]]）
* 丰顷町丰顷（[[北海道道73号带广浦幌线|北海道道73号带广浦幌线]]）
* 丰顷町育素多-丰顷町丰顷旭町（北海道道210号尾田丰顷停車場線・[[北海道道318号湧洞丰顷停车场线|北海道道318号湧洞丰顷停车场线]]）

== 经过的自治体  ==
* 十胜综合振兴局
** 中川郡丰顷町

== 主要橋梁 ==
* 茂岩橋（[[十勝川|十勝川]]）＝丰顷町茂岩本町-丰顷町育素多
* 丰顷橋（旧利別川<small>＝十勝川支流</small>）＝丰顷町丰顷

* 1994年10月1日 路線编号变更为320号。（1994年北海道告示第1468号）
