'''北京及瀋陽的明清皇家宮殿'''，或簡稱為'''明清故宮'''，是[[聯合國教科文組織|聯合國教科文組織]]所登錄的一項[[中華人民共和國|中國]]的[[世界文化遺產|世界文化遺產]]。登錄的範圍包括[[北京|北京]][[故宮博物院|故宮博物院]]([[故宮|紫禁城]])及2004年追加的[[瀋陽故宮|瀋陽故宮]]等兩個建築群。

== 概要 ==
1987年，北京故宮(紫禁城)被登錄為世界文化遺產。紫禁城是世界现存最大的皇宮，為[[明朝|明]][[清朝|清]]兩朝24個[[皇帝|皇帝]]所居住、辦公的宮殿。由明成祖[[朱棣|朱棣]]於1406年下旨修建，至1420年基本完成。

2004年，瀋陽故宮作為北京故宮的擴展項目被聯合國教科文組織列為世界文化遺產。瀋陽故宮是[[後金|後金]]及滿清初期，[[努爾哈赤|努爾哈赤]]及[[皇太極|皇太極]]兩名皇帝的宮殿，[[顺治皇帝|顺治皇帝]]在此继位，同時也是[[滿清的離宮|滿清的離宮]]之一。

== 登錄過程 ==
* 1987年將北京故宮建築群登錄為世界文化遺產
* 2004年將瀋陽故宮作為北京故宮的擴展項目列入「北京及瀋陽的明清皇家宮殿」此項世界文化遺產

== 登錄標準 ==
因為滿足世界文化遺產下列評定標準而登錄：
: (i) 代表一種獨特的藝術成就，一種創造性的天才傑作。 
: (ii) 能在一定時期內或世界某一文化區域內，對建築藝術、紀念物藝術、城鎮規劃或景觀設計方面的發展產生極大影響。 
: (iii) 能為一種已消逝的文明或文化傳統提供一種獨特的，或至少是特殊的見證。 
: (iv) 人類歷史發展中某—建築風格的傑出範例。
