'''北京市人民政府'''是[[中华人民共和国|中华人民共和国]][[北京市|北京市]]的最高地方行政机构，其前身为1949年1月1日宣告成立的'''北平市人民政府'''，1949年9月27日[[北平|北平]]改称[[北京|北京]]，北平市人民政府遂改称北京市人民政府。1954年8月改组为[[北京市人民委员会|北京市人民委员会]]。1979年12月6日至13日召开的[[北京市第七届人民代表大会|北京市第七届人民代表大会]]第三次会议决定撤销[[北京市革命委员会|北京市革命委员会]]，复设[[北京市人民代表大会常务委员会|北京市人民代表大会常务委员会]]和'''北京市人民政府'''。

==沿革==
北京在[[清朝|清朝]]称[[顺天府|顺天府]]，辖境除今[[北京市|北京市]]外，还包括今[[河北省|河北省]]部分地区，其首长称[[顺天府尹|顺天府尹]]。1912年[[中华民国|中华民国]]建立，顺天府仍维持旧称，[[顺天府衙署|顺天府衙署]]改[[顺天府尹公署|顺天府尹公署]]，民国建立后共有4任顺天府尹。1914年10月4日，[[中華民國大總統|中華民國大總統]][[袁世凱|袁世凱]]頒布《京兆尹官制》（法律十三号）及《京兆地方区域表》（教令第一百三十三号），改顺天府为[[京兆地方|京兆]]，改顺天府尹为[[京兆尹|京兆尹]]，改顺天府尹公署为[[京兆尹公署|京兆尹公署]]。民国时期共有11任京兆尹。

1928年[[国民革命军北伐|国民革命军北伐]]成功后，1928年6月21日[[南京国民政府|南京国民政府]]决定“北京”更名为“北平”，设[[北平特别市|北平特别市]]。1930年6月该市降格为[[河北省|河北省]]省辖市，1930年11月复升为[[院辖市|院辖市]]。该市在1928年10月至1930年10月为河北省[[省会|省会]]。1931年4月改北平特别市为[[北平市|北平市]]，市政府名称随之改为[[北平市政府|北平市政府]]。<ref name=bjsdag>[http://www.da.bj.cn/staticfile/Exhibition/yin/1-1.html 北平市政府，北京市档案馆，于2011-7-23查阅] {{webarchive|url=https://web.archive.org/web/20061130001133/http://www.da.bj.cn/staticfile/Exhibition/yin/1-1.html |date=2006-11-30 }}</ref>

[[日军|日军]]占领北平后，1937年7月30日成立[[北平市地方维持会|北平市地方维持会]]，[[江朝宗|江朝宗]]任会长；8月6日北平市地方维持会推举江朝宗任北平市长；8月8日，日军接管北平防务；8月9日，江朝宗正式就任北平市地方维持会会长；8月19日，江朝宗出任北平市地方维持会市长。10月12日，北平市地方维持会改“北平”为“北京”，该会更名为[[北京地方维持会|北京地方维持会]]。12月14日，[[中华民国临时政府|中华民国临时政府]]成立于北京，以北京为首都，[[王克敏|王克敏]]宣布江朝宗为北京市长。北京地方维持会于12月17日宣布结束，1938年1月20日其所有事务被中华民国临时政府全部接收，[[北京特别市政府|北京特别市政府]]正式成立。1938年1月1日，江朝宗正式就任北京市长；1938年1月10日，[[余晋龢|余晋龢]]为新任北京市长。1938年1月13日，北京特别市政府改称[[北京特别市公署|北京特别市公署]]，直至1945年[[日本投降|日本投降]]。1940年3月，[[汪精卫|汪精卫]]为首的[[汪精卫政权|国民政府]]在[[南京|南京]]成立，中华民国临时政府改为[[华北政务委员会|华北政务委员会]]，北京失去首都地位。<ref name=bjsdag/>

1945年9月日本投降，[[南京国民政府|南京国民政府]]接管了北京市，并宣布恢复该市原名称“北平市”，仍为院辖市。1945年10月，北平市政府重新成立，仍直辖于[[行政院|行政院]]。1949年1月31日[[北平和平解放|北平和平解放]]，2月4日[[北平市人民政府|北平市人民政府]]接管北平市政府。<ref name=bjsdag/>

1949年1月1日，在[[中国人民解放军平津前线司令部|中国人民解放军平津前线司令部]]指挥之下的[[中国人民解放军北平市军事管制委员会|中国人民解放军北平市军事管制委员会]]发布布告宣告成立，[[叶剑英|叶剑英]]任主任；同日，[[华北人民政府|华北人民政府]]下属的[[北平市人民政府|北平市人民政府]]宣告成立，叶剑英任市长。1949年9月27日，[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]决定将北平更名为北京，北平市人民政府遂更名为'''北京市人民政府'''。1949年11月20日至22日召开的[[北京市第二届第一次各界人民代表会议|北京市第二届第一次各界人民代表会议]]选举市长、副市长、市人民政府委员，组成[[北京市人民政府委员会|北京市人民政府委员会]]。12月2日，[[中央人民政府委员会|中央人民政府委员会]]第四次会议通过了任命北京市人民政府委员会名单。12月9日，北京市市长、副市长与北京市人民政府委员宣布就职，北京市人民政府正式成立。1954年8月17日至23日召开的[[北京市人民代表大会|北京市第一届人民代表大会]]第一次会议，决定将北京市人民政府更名为'''[[北京市人民委员会|北京市人民委员会]]'''。1967年4月20日，'''[[北京市革命委员会|北京市革命委员会]]'''成立，北京市人民委员会被取消。1979年12月6日至13日召开的[[北京市人民代表大会|北京市第七届人民代表大会]]第三次会议决定撤销北京市革命委员会，复设[[北京市人民代表大会常务委员会|北京市人民代表大会常务委员会]]和'''北京市人民政府'''。

2015年12月30日，位于[[通州区_(北京市)|通州区]]的北京城市副中心行政办公区举行奠基仪式，2017年1月起各主要建筑主体结构陆续封顶。按照计划，2017年底，北京市市级机关和部分市属委办局将首批启动搬迁<ref>{{Cite web |url =http://news.ifeng.com/a/20170907/51903201_0.shtml  |title =  雄伟！北京市政府新大楼开始揭开“面纱”|author =  |date =2017-09-07  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。2018年10月，[[中共中央|中共中央]]、[[中华人民共和国国务院|国务院]]批复北京市机构改革方案<ref>{{cite news |title=北京机构改革方案获批 今年底市级党政机构调整到位 |url=http://bj.news.163.com/18/1020/12/DUIF1GPB04388CSB.html |accessdate=2018-11-10 |work=[[网易|网易]]北京 |date=2018-10-20}}</ref>。2019年1月10日晚，位于[[东城区|东城区]][[正义路|正义路]]2号的北京市人民政府正式摘牌并移交档案馆<ref>{{cite news |title=北京市人民政府摘牌 城市副中心行政办公区将启用 |url=http://news.sina.com.cn/c/2019-01-10/doc-ihqhqcis5024778.shtml |accessdate=2019-01-11 |work=环球网 |date=2019-01-10}}</ref>，1月11日，北京市人民政府机关与[[中共北京市委|中共北京市委]]、[[北京市人民代表大会常务委员会|北京市人大常委会]]、[[中国人民政治协商会议北京市委员会|政协北京市委员会]]一同搬迁至[[北京城市副中心|北京城市副中心]][[北京城市副中心行政办公区|行政办公区]]新址<ref>{{Cite web |url =https://news.sina.com.cn/c/2019-01-11/doc-ihqhqcis5083453.shtml|title =  北京市政府机关搬迁至通州区 即日起在新址办公|author =  |date =2019-01-11 |publisher = 新浪新闻综合 |language =  |accessdate =  |quote =  }}</ref><ref>{{cite news |title=北京市级行政中心正式迁入城市副中心 蔡奇陈吉宁李伟吉林出席升旗仪式并揭牌 |url=http://shilingdao.beijing.gov.cn/library/191/10/738495/1577140/index.html |accessdate=2019-01-11 |work=北京日报 |date=2019-01-11}}</ref>。

== 职权 ==
根据《{{s|中华人民共和国地方各级人民代表大会和地方各级人民政府组织法}}》第五十九条，县级以上的地方各级人民政府行使下列职权：
{{columns-list|2|
# 执行本级人民代表大会及其常务委员会的决议，以及上级国家行政机关的决定和命令，规定行政措施，发布决定和命令；
# 领导所属各工作部门和下级人民政府的工作；
# 改变或者撤销所属各工作部门的不适当的命令、指示和下级人民政府的不适当的决定、命令；
# 依照法律的规定任免、培训、考核和奖惩国家行政机关工作人员；
# 执行国民经济和社会发展计划、预算，管理本行政区域内的经济、教育、科学、文化、卫生、体育事业、环境和资源保护、城乡建设事业和财政、民政、公安、民族事务、司法行政、监察、计划生育等行政工作；
# 保护社会主义的全民所有的财产和劳动群众集体所有的财产，保护公民私人所有的合法财产，维护社会秩序，保障公民的人身权利、民主权利和其他权利；
# 保护各种经济组织的合法权益；
# 保障少数民族的权利和尊重少数民族的风俗习惯，帮助本行政区域内各少数民族聚居的地方依照宪法和法律实行区域自治，帮助各少数民族发展政治、经济和文化的建设事业；
# 保障宪法和法律赋予妇女的男女平等、同工同酬和婚姻自由等各项权利；
# 办理上级国家行政机关交办的其他事项。
}}
