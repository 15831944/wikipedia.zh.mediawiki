'''北海道道531号小谷石渡岛知内停车场线'''（{{lang-ja|北海道道531号小谷石渡島知内停車場線}}）是一条[[北海道|北海道]][[上矶郡|上矶郡]][[知内町|知内町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：北海道上矶郡知内町小谷石
* 終点：北海道上矶郡知内町重内（旧[[北海道旅客铁道|JR北海道]] [[渡岛知内站|渡岛知内站]]）
* 总距离：10.6千米（内、重複区間0.8千米）

== 经过的自治体  ==
* [[渡岛综合振兴局|渡岛综合振兴局]]
** 上矶郡知内町

== 主要连接道路 ==
* [[国道228号|国道228号]]（重内 - 重内）：重複
* [[北海道道698号汤之里渡岛知内停车场线|北海道道698号汤之里渡岛知内停车场线]]（重内 - 終点）：重複

== 历史 ==
* 1966年（昭和41年）3月31日 - 路線勘定（北海道告示第679号）

== 隧道 ==
* いさりび隧道（延長149米、2002年開通）
* しおさい隧道（延長328.5米、1997年開通）
== 已废除隧道 ==
*1号隧道（延長約17米、開削撤去）
*2号隧道（延長12米、1958年開通､開削撤去）
*3号隧道（延長17.5米、1958年開通､開削撤去）
*4号隧道（延長30米、1961年開通､随着しおさい隧道的开通而关闭）

== 其他 ==
