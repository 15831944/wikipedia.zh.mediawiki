'''北海道大学前札幌农业大学昆虫学和养蚕学教室 '''(日语:北海道大学旧札幌農学校昆虫学及養蚕学教室/ほっかいどうだいがく きゅうさっぽろのうがっこう こんちゅうがく および ようさんがくきょうしつ）是存在于[[札幌市|北海道札幌市]]、[[北海道大學|北海道大学]]校内的设施。由文部大臣官房札幌建筑出张所，[[中條精一郎|中條精一郎]]设计的<ref name="litterapopuli31">{{Cite web|url=http://www.hokudai.ac.jp/bureau/populi/edition31/index.html|title=北海道大学広報誌 季刊【リテラ・ポプリ】31号 建築設計図が語る北大の歴史─〈第12回〉北大交流プラザ「エルムの森」|accessdate=2016-10-01|author=池上 重康}}</ref>。平成12（2000）年4月28日，被收录进了国登录[[有形文化财|有形文化财]]。所在地位于北海道札幌市[[北區_(札幌市)|北区]]北19条西8丁目<ref name="Sapporo1">{{Cite web|url=http://www.city.sapporo.jp/shimin/bunkazai/pdf/documents/p44hokudaikontyukyousitu.pdf|title=北海道大学旧札幌農学校昆虫及養蚕学教室|accessdate=2016-10-01|publisher=札幌市|deadurl=yes|archiveurl=https://web.archive.org/web/20160330061032/http://www.city.sapporo.jp/shimin/bunkazai/pdf/documents/p44hokudaikontyukyousitu.pdf|archivedate=2016-03-30}}</ref>。

== 历史 ==
* [[1901年|在1901年]] ([[明治|明治]] 34岁)12-竣工<ref name="seiyoukannorirekisyo">{{Cite book|title=西洋館の履歴書－北海道－|author=下村仁|publisher=創英社／[[三省堂書店|三省堂書店]]|year=2011|isbn=978-4-88142-509-1|location=|page=152-159|quote=}}</ref>.
* [[1916年|在1916年]] ([[大正|大正]] 5)1月-增建了昆虫飼育室(昆虫学和养蚕学教室西翼的北侧连接)<ref name="seiyoukannorirekisyo" />.
* [[1927年|在1927年]] ([[昭和|昭和]] 2)11月-新建了昆虫学標本室<ref name="seiyoukannorirekisyo" />.
* [[1936年|在1936年]] (昭和11)8月-因为昆虫学关联讲座移动到了农学部本馆，变成了空房间<ref name="seiyoukannorirekisyo" />.
* [[1937年|在1937年]] (昭和12)10月-成立了北方文化实验室<ref name="seiyoukannorirekisyo" />.
* [[1990年|在1990年]] ([[平成|平成]]2)- 成为了放送大学北海道学习中心<ref name="seiyoukannorirekisyo" />.
* [[2000年|在2000年]] (平成12)2月-放送大学北海道学习中心迁移了<ref name="seiyoukannorirekisyo" />.
* [[2000年|在2000年]] (平成12)4月28日-收录进了国登录有形文化财<ref name="seiyoukannorirekisyo" />.
* [[2003年|在2003年]] (平成15)5月-成为了北海道大学交流广场「榆之森」<ref name="seiyoukannorirekisyo" />.
* [[2010年|2010年]] (平成22)6月-榆之森迁移了<ref name="seiyoukannorirekisyo" />.

== 标本室(样本仓库) ==
[[File:Hokkaido_University_-_Storage_of_Entomology_and_Sericulture.jpg|缩略图]]
在昆虫学初代教授[[松村松年|松村松年]]的时代 <ref>{{Cite book|url=http://hdl.handle.net/2115/30006|title=北海道大学の学術標本: 特に昆虫標本について|author=諏訪正明|publisher=北海道大学|year=2003|editor=北海道大学百二十五年史編集室編|pages=307–308|chapter=北大百二十五年史　論文・資料編|chapterurl=http://eprints.lib.hokudai.ac.jp/dspace/handle/2115/28218}}</ref>，1927年增建了標本室(样本仓库)。为了保护昆虫的标本不受火灾侵害，标本室由札幌软石构成的墙壁，钢筋混凝土作成的地板和天花板、附有防火百叶的窗框等各种不可燃构造建成 <ref name="like_hokudai1">{{Cite web|url=http://costep.open-ed.hokudai.ac.jp/like_hokudai/contents/article/463/|title=過山博士の本棚から #1 エルムの森にたたずむ石壁（前編）|accessdate=2016-10-01|author=佐々木学}}</ref>.
