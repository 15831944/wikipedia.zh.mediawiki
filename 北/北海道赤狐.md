'''北海道赤狐'''（日语：{{lang|ja|北狐}}、''Vulpes vulpes schrencki'' ）是广泛分布于北半球的[[赤狐|赤狐]]的[[亚种|亚种]]。主要生息于日本[[北海道|北海道]]、[[库页岛|库页岛]]、[[南千岛群岛|南千岛群岛]]及周边岛屿。

日语中北海道赤狐之正名“'''北狐'''”是[[1924年|1924年]][[岸田久吉|岸田久吉]]<ref>理学博士、農学博士 --『[[キタキツネ#門崎_(2009)|野生動物調査痕跡学図鑑]]』（p397）より。</ref>在研究库页岛生物时所给予的。随后此称呼被广泛运用于北海道和千岛群岛的该种狐狸<ref name=":0">{{Cite book|url=https://www.worldcat.org/oclc/675918751|last=1938-|first=Kadosaki, Masaaki,|last2=1938-|first2=門崎, 允昭,|date=2009|publisher=Hokkaidōshuppankikakusentā|isbn=9784832809147|location=Sapporo|chapter=Yasei dōbutsu chōsa konsekigaku zukan = Illustrated wildlife traces|oclc=675918751}}</ref>。阿伊努语称之为cironnup、sumari、kimotpe 或者hurep<ref>{{Cite book|title=《分类阿伊努语词典》|last=知里|first=真志保|publisher=|year=|isbn=|location=|pages=}}</ref>。

== 描述 ==
北海道赤狐比在日本[[本州|本州岛]]、[[四國|四国岛]]和[[九州_(日本)|九州岛]]的日本赤狐大。耳朵和四肢的外部是黑色的。与大陆赤狐有许多相似之处。

北海道赤狐有42颗牙：包括门齿上下各6颗、犬齿上下各2颗、小臼齿上下各8颗、上颌臼齿4颗以及下颌臼齿6颗。有8个乳房：[[胸部|胸部]]1対、[[腹部|腹部]]2対、[[鼠蹊部|鼠径部]]1対（但同样有7-10个乳房的变异个体存在）<ref name=":0" />。

== 生态 ==
北海道赤狐广布于北海道的草原到高山一带。它们主要吃[[鼠|老鼠]]、野兔、鸟和昆虫。秋天他们也吃水果和坚果。在旅游景点和一些城市地区亦有人在街上喂养它们。

它们挖地道或隧道来作为自己的巢穴。雌性会在早春分娩，幼崽在晚秋时生长独立。雄性独立行动，不干涉抚养后代。

== 疾病和寄生虫 ==
北海道赤狐易感染棘球绦虫和蠕虫。这些都是通过培养现代阿留申群岛的银狐引进到北海道的。棘球蚴病是北海道赤狐的绦虫形感染。感染时间保持于北海道赤狐和老鼠之间。自1999年来，已经使用驱虫药来保证野生北海道赤狐摄入感染率降低。细粒棘球蚴虫卵以粪便排出体外，可被人类吞食并引起棘球蚴病。如果在早期发现可以治疗，但如果治疗延迟或寄生在难以实行手术的部位，则有可能导致死亡。受到感染的北海道赤狐也可以感染家畜和宠物。

== 与人类的关系 ==
北海道赤狐属于野生动物。长期以来人们一直反对人为干预，包括增加喂养。观点认为，把栖息地因素排除在外，这将对北海道赤狐产生不利影响。但是北海道赤狐的棘球蚴病已经被认为是问题。已经有人呼吁采取行动来治理该问题。<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/hf/kth/kak/0000contents/ekino/|title=エキノコックス症の知識と予防  ｜ 保健福祉部健康安全局地域保健課|accessdate=2018-05-20|work=www.pref.hokkaido.lg.jp|language=ja}}</ref>

北海道赤狐通过喂食来熟悉人类。<ref>{{Cite book|url=https://www.worldcat.org/oclc/958869068|last=1944-|first=Hatakeyama, Takemichi,|last2=Nozomu.|first2=Kojima,|last3=1966-|first3=Takahashi, Mitsuhiko,|last4=1944-|first4=畠山, 武道,|last5=望|first5=小島,|last6=1966-|first6=高橋, 満彦,|date=2016.8|publisher=Chijinshokan|isbn=9784805209004|chapter=Yasei dobutsu no ezuke mondai : Zen'i ga hikiokosu seitaikei kakuran chojugai kansensho seikatsu higai.|oclc=958869068}}</ref>比如，经证实，旅游旺季过后，[[知床國立公園|知床国家公园]]的北海道赤狐会在四月离开自己的领地，然后在宇登吕市寻找人们的食物。自上世纪80年代北海道赤狐在一些城市地区有了永久落户的绿地，比如[[札幌市|札幌]]。
