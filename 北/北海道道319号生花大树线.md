'''北海道道319号生花大树线'''（{{lang-ja|北海道道319号生花大樹線}}）是一条连结[[北海道|北海道]][[十胜综合振兴局|十胜综合振兴局]]管内[[大树町|大树町]]和[[幕别町|幕别町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 指定区間 ==
* 起点：北海道[[广尾郡|广尾郡]]大樹町字生花（＝[[国道336号|国道336号]]・[[北海道道881号ホロカヤントー線|北海道道881号ホロカヤントー線]]交点）
* 終点：北海道广尾郡大樹町東本通（＝[[国道236号|国道236号]]・[[北海道道55号清水大树线|北海道道55号清水大树线]]分岐点）
* 总長：18.5千米

== 共线区间 ==
* [[幕别町|幕别町]]忠類錦町-幕別町忠類幸町（[[北海道道657号美成忠类停车场线|北海道道657号美成忠类停车场线]]）
* 幕別町忠類幸町-大樹町東本通（[[国道236号|国道236号]]・[[北海道道15号幕别大树线|北海道道15号幕别大树线]]）
* 大樹町東本通（[[北海道道55号清水大树线|北海道道55号清水大树线]]・[[北海道道773号萌和大树停车场线|北海道道773号萌和大树停车场线]]）

== 经过的自治体  ==
* 十勝综合振兴局
** 广尾郡大樹町 - 中川郡幕別町 - 广尾郡大樹町
