'''北海道道144号登别室兰连络线'''（{{lang-ja|北海道道144号登別室蘭インター線}}）是一条位于[[北海道|北海道]][[登別市|登別市]]内的[[主要地方道|主要]][[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 路線數據 ==
* 起点：[[北海道|北海道]][[登別市|登別市]]幌別町2丁目（＝[[国道36号|国道36号]]交点）
* 終点：北海道登別市緑町3丁目（＝[[道央自動車道|道央自動車道]][[登别室兰IC|登別室蘭IC]]）
* 总長：2.4千米

== 经过的自治体  ==
* [[胆振综合振兴局|胆振综合振兴局]]
** 登別市

== 历史 ==
* 1994年4月1日 路線勘定（路線编号为1107）。
* 1994年10月1日 路線编号变更。
