'''北京地铁昌平线'''是[[北京地铁|北京地铁]]的一条线路。连接[[城六區|北京城区]]及[[昌平区|昌平区]]。线路南起[[西二旗站|西二旗站]]，北至[[昌平西山口站|昌平西山口站]]<ref>{{cite web|url=http://www.bjghw.gov.cn/web/static/articles/catalog_26/article_ff8080814e77f738014e7806e1ca0001/ff8080814e77f738014e7806e1ca0001.html|title=北京市规划委员会关于北京市轨道交通昌平线（二期）工程沿线车站站名命名预案的公示书|publisher=北京市规划委员会|date=2015-07-10|accessdate=2015-07-11|archive-url=https://web.archive.org/web/20150711170604/http://www.bjghw.gov.cn/web/static/articles/catalog_26/article_ff8080814e77f738014e7806e1ca0001/ff8080814e77f738014e7806e1ca0001.html#|archive-date=2015-07-11|dead-url=yes}}</ref>，分为两期建设：一期工程自西二旗站至[[南邵站|南邵站]]，全长21.3[[公里|公里]]，大部分为高架线，设1座地下站，6座高架站及一座[[停车场|停车场]]，于2009年开工，2010年12月30日开通运营<ref>{{ cite web|url=http://www.bjd.com.cn/10zt/201006/t20100625_623379.html |title=本市5条轨道交通线9月完工 |author=北京日报 |publisher=京报网 |date=2010-06-25 |accessdate=2010-07-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20100729220047/http://www.bjd.com.cn/10zt/201006/t20100625_623379.html |archivedate=2010-07-29 }}</ref><ref>{{cite web|url=http://www.bjjtw.gov.cn/gzdt/dtxx/201012/t20101231_33056.htm |title=大兴线、亦庄线、十五号线首开段、昌平线一期、房山线等五条轨道交通新线12月30日开通试运营 |publisher=北京市交通委员会 |date=2010-12-31 |accessdate=2013-09-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20150610203103/http://www.bjjtw.gov.cn/gzdt/dtxx/201012/t20101231_33056.htm |archivedate=2015-06-10 }}</ref>；二期工程自南邵站至[[昌平|昌平]]西山口站，全长10.6千米，正线全部为地下线，设置5座车站及一座车辆段<ref>{{cite news|url=http://bjrb.bjd.com.cn/html/2013-10/08/content_114191.htm|title=北京地铁昌平线二期增设涧头西站|date=2013-10-08|accessdate=2013-10-08|author=马玲|work=北京日报|deadurl=yes|archiveurl=https://web.archive.org/web/20131014113132/http://bjrb.bjd.com.cn/html/2013-10/08/content_114191.htm|archivedate=2013-10-14}}</ref>，2012年开工，2015年12月26日通车试运营<ref>{{cite news|url=http://weibo.com/2778292197/Da4iTsddI|title=@北京地铁 的告知微博|date=2015-12-25|accessdate=2015-12-26|work=新浪微博}}</ref>。

2017年1月3日起，在工作日7时（含）前，持[[市政交通一卡通|市政交通一卡通]]的乘客乘坐昌平线，票价可享5折优惠（需在南邵站至朱辛庄站之间的任意一个车站进站）<ref>{{cite news|url=http://news.ifeng.com/a/20170103/50516197_0.shtml|title=北京24座地铁站今起低峰票价5折 含八通线、昌平线等|date=2017-01-03|accessdate=2017-01-04|author=|work=人民网}}</ref>。在2015年12月22日至2016年12月31日期间的工作日早7点前，持市政交通一卡通的乘客乘坐昌平线，票价优惠为7折<ref>{{cite news|url=http://society.people.com.cn/n1/2015/1222/c136657-27961737.html|title=八通线 昌平线 从这16站上车 7点前7折|date=2015-12-22|accessdate=2017-01-04|author=|work=人民网}}</ref>。

== 南延 ==
[[File:Xi'erqi_Station_End_of_Changping_Line_20160331_121843.jpg|thumb]]
昌平线南延计划已列入北京地铁2020年规划。昌平线南延计划从[[西二旗站|西二旗站]]驶出，设置[[清河站|清河站]]、[[上清桥站|上清桥站]]、[[学清路站|学清路站]]、[[六道口站|六道口站]]、[[学院桥站|学院桥站]]、[[西土城站|西土城站]]和[[蓟门桥站|蓟门桥站]]。最终可能会南延至[[国家图书馆|国家图书馆]]附近，预计2019年年底竣工<ref>{{cite web |url=http://bjrb.bjd.com.cn/html/2016-07/20/content_51537.htm |title=存档副本 |accessdate=2016-08-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20160916022841/http://bjrb.bjd.com.cn/html/2016-07/20/content_51537.htm |archivedate=2016-09-16 }}</ref>。南延綫全长15km，设10座车站。但是由于[[北京地铁9号线|9号线]]将計劃北延至西二旗，並有可能與昌平線貫通運營或換乘，因此9号线即可为13号线分流，達到昌平線南延的作用。

後來南延計劃再度復活，把原來的9號線北端延長綫一分為二，以明光橋西站（13號綫）為分界。國家圖書館站、皂君廟站及明光橋西站屬9號線北端延長綫，明光橋西站至西二旗站屬昌平線南延長綫，因此明光橋西站將成為三綫轉乘點，使西二旗站高峰時間擁擠問題得以緩和。

== 其他 ==
昌平线采用中国完全自主知识产权的信号系统，结束了外国企业对地铁[[CBTC|CBTC]]信号系统的垄断<ref>{{cite web|url=http://news.bjtu.edu.cn/meitijiaoda/2012-06-07/100271.html|title=科技日报头版头条文章：中国地铁轨道交通信号系统自主创新心路历程|publisher=科技日报|date=2012-06-07|accessdate=2012-06-18|deadurl=yes|archiveurl=https://web.archive.org/web/20120621185156/http://news.bjtu.edu.cn/meitijiaoda/2012-06-07/100271.html|archivedate=2012-06-21}}</ref>。与引进外国信号系统相比，国产信号系统可降低成本约20%－30%<ref>{{cite web|url=http://scitech.people.com.cn/GB/13278068.html|title=首套国产轨道交通信号系统调试运行稳定|publisher=科学时报|date=2010-10-22|accessdate=2012-06-18}}</ref>。

昌平线在人性化方面也进行了精心设计。由于很多车站都是高架车站，在道路中间，联通车站的人行天桥上下都设计了扶梯和直梯，每个车站的电梯都超过10條。昌平线的电梯是所有地铁线路中最多的。
