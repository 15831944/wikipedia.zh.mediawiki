'''北京语言大学'''成立于1962年，是[[中华人民共和国|中华人民共和国]]教育部直属高等院校<ref>http://gaokao.chsi.com.cn/gkxx/gxmd/zsgx.shtml 教育部直属高等院校名单
</ref>，中国唯一一所以教外国人汉语为主要特点的高等院校，其前身为'''北京语言学院'''。<ref> http://www.blcu.edu.cn/col/col5682/index.html 学校简介</ref>现在，该校不仅教授外国学生[[中文|中文]]和[[中国文化|中国文化]]，而且也教授中国学生[[外语|外语]]和外国文化。

校园位于[[北京|北京]]市[[海淀区|海淀区]]学院路15号，靠近[[北京大学|北京大学]]和[[清华大学|清华大学]]，传统上被认为是[[中国|中国]]国内[[对外汉语|对外汉语]]教学水平最高的[[大学|大学]]，同时该校对于中国学生进行的[[外语|外语]]教育的水平也在全国名列前茅。目前，北京语言大学的外国学生人数与中国学生人数不相上下，该校不仅拥有中国国内人数最多的[[韩国|韩国]]和[[日本|日本]][[留学生|留学生]]群体，还有大批来自[[西方国家|西方国家]]、[[中东|中东]]地区以及[[非洲|非洲]]的留学生，非常国际化，被誉为“'''小联合国'''”。<ref>http://news.sina.com.cn/c/2008-07-29/100016020303.shtml “小联合国”北京语言大学</ref>以非汉语母语为对象的[[汉语水平考试|汉语水平考试]]（HSK）的考试中心曾设于该校。[[北京语言大学出版社|北京语言大学出版社]]以出版对外汉语类教材而著称。<ref>{{Cite news|url=http://news.xinhuanet.com/newmedia/2014-08/27/c_126924222.htm |title=从细分领域看版权输出惑与解 |quote=以北京语言大学出版社为例，据了解，该社对外汉语教材产品超过3000种。该社副社长王壮在接受《中国新闻出版报》记者采访时表示，北语社语种最多的一套产品《汉语乐园》（儿童汉语教材）有46个语种，覆盖了全世界几乎所有具有汉语学习需求的国家和地区。}}</ref>

== 歷史 ==
1965年1月，'''外国留学生高等预备学校'''正式更名为'''北京语言学院'''，[[毛泽东|毛泽东]]亲笔为该校题写校名。<ref>{{cite web |url=http://www.qianlong.com/3413/2002-9-5/166@407433.htm |title=存档副本 |accessdate=2014-09-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20140911171125/http://www.qianlong.com/3413/2002-9-5/166@407433.htm |archivedate=2014-09-11 }} 北京语言大学的由来</ref>

1978年暑期，接受[[法国|法国]]28名短期学习汉语的留学生，是中国举办短期[[汉语|汉语]]教学的开始。

1981年10月，北京语言学院电教中心摄制完成中国第一部[[对外汉语教学|对外汉语教学]]的电视录像片《中国话》。<ref>{{Cite web |title=新中国对外汉语教学发展 |url=http://www.china.com.cn/market/shihan/420752.htm |quote=我国第一部对外汉语教学电视录像片《中国话》于1981年10月由北京语言学院电教中心摄制完成。}}</ref>

1984年北京语言学院受教育部委托开始研制[[中国汉语水平考试|中国汉语水平考试]]（HSK）考试。<ref>{{cite web |url=http://www.hsk.org.cn/Mainland.aspx |title=存档副本 |accessdate=2014-09-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20130922003707/http://www.hsk.org.cn/Mainland.aspx |archivedate=2013-09-22 }} HSK官网考试中心介绍 </ref>

1989年北京语言学院成立汉语水平考试中心。

1990年中国汉语水平考试通过专家鉴定，并正式实施。

1991年中国汉语水平考试推向海外。

1992年[[国家教委|国家教委]]第21号令确定其为国家级考试。<ref>{{cite web |url=http://bj.edutt.com/book_show_4198/ |title=存档副本 |accessdate=2014-09-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20140911160306/http://bj.edutt.com/book_show_4198/ |archivedate=2014-09-11 }} 中华人民共和国国家教育委员会令第21号</ref>

1996年北京语言学院改名为“'''北京语言文化大学'''”。

2002年北京语言文化大学将名称简化为“'''北京语言大学'''”。
