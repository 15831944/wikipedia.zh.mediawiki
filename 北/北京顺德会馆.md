'''北京顺德会馆'''，是[[广东省|广东省]][[顺德县|顺德县]]人在北京的会馆的总称。

==概述==
北京的顺德会馆有五处，分别位于[[海柏胡同|海柏胡同]]路南、[[永光西街|永光西街]]路东、[[后青厂胡同|后青厂胡同]]、[[丞相胡同|丞相胡同]]、[[天景胡同|天景胡同]]路北。其中，位于永光西街、后青厂胡同、丞相胡同的三座早已遭到拆除<ref name=huiguan/><ref name=foshan>{{Cite web|url = http://www.fsonline.com.cn/p/179580.html|title = 探访北京顺德会馆|author = |date = 2016-11-20|publisher = 佛山在线|language = |accessdate = |quote = }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。仅存位于[[北京市|北京市]][[西城区|西城区]]海柏胡同16号的顺德邑馆，以及位于北京市西城区天景胡同的顺德会馆南馆<ref name=huiguan> {{Cite book | author =胡春焕、白鹤群  | title = 北京的会馆 | location =  | publisher =中国经济出版社  | date = 1994年 | pages = 第43-44页、第67-70页 | ISBN =  | accessdate =  | url =  | language =  | quote =  }} </ref>。

顺德邑馆，又称顺德会馆，位于北京市西城区海柏胡同（原名海波寺街）16号，是北京五座顺德会馆中的主馆，建于19世纪初。其前身是[[清朝|清朝]]学者[[朱彝尊|朱彝尊]]故居。朱彝尊曾入[[翰林院|翰林院]]编修《[[明史|明史]]》，后来遭到弹劾<ref name=foshan/>。清朝[[康熙|康熙]]二十三年三月（1684年），学者朱彝尊从禁苑迁居此处<ref name=huiguan/>。当时他住在一座号称“古藤书屋”的南屋里，因院中种植古[[紫藤|紫藤]]而得名，屋前西南有“曝书亭”<ref name=foshan/>。他写的文集由[[曹寅|曹寅]]捐钱刊刻，被定名为《曝书亭集》。康熙二十五年（1686年）夏，朱彝尊在这里撰写《[[日下旧闻考|日下旧闻]]》一书，康熙二十六年秋（1687年）写成。康熙二十八年三月（1689年），朱彝尊从这里迁居至槐市斜街（今[[下斜街_(北京)|下斜街]]）的[[都土地庙|土地庙]]附近<ref name=huiguan/>。

[[嘉庆|嘉庆]]十九年（1814年），朱彝尊在海柏胡同的旧居由[[兵部|兵部]]右侍郎[[温汝适|温汝适]]（广东顺德人）等人集资购买，改建成顺德邑馆。[[咸丰|咸丰]]七年（1857年），顺德邑馆重修，[[光禄寺|光禄寺]]署正[[郑子卿|郑子卿]]（广东顺德人）找来通晓[[风水|风水]][[堪舆|堪舆]]的通政司[[梁同新|梁同新]]（广东番禺人）调整会馆朝向，形成如今坐南朝北的格局。梁同新当时预言，重修落成次年顺德应考学生将春闱大捷。咸丰九年（1859年）会试，[[李文田|李文田]]中一甲第三名（探花），[[梁思问|梁思问]]中二甲第二十二名。梁同新的预言应验。此事被记录在《重修顺德邑馆碑记》<ref name=foshan/>。

顺德邑馆占地面积7.04亩，房屋136.5间。会馆坐南朝北，分为东、中、西三路院落，其中西院据说是朱彝尊故居“古藤书屋”及“曝书亭”。朱彝尊居住时的“曝书亭”早已无存，现存的“曝书亭”是近代复建，亭顶原是圆顶，后来改建为方顶<ref name=foshan/>。顺德邑馆内原有《签建顺德馆小引》碑，《重修顺德邑馆》碑，馆内的“曝书亭”挂有清朝[[光绪|光绪]]丁未年（1907年）[[龙建章|龙建章]]题写的“曝书亭”匾额，“古藤书屋”挂有清朝[[宣统|宣统]]三年（1911年）[[岑光樾|岑光樾]]题写的“古藤书屋”匾额<ref name=huiguan/>。

顺德人[[罗瘿公|罗瘿公]]生在北京。顺德邑馆所在的海柏胡同靠近[[京剧|京剧]]名伶聚居的椿树地区。罗瘿公来顺德邑馆时，会顺道探访椿树地区。罗瘿公看戏时发现了年轻的[[程砚秋|程砚秋]]，特为其赎身脱籍。民国十一年（1922年）到民国十三年（1924年），罗瘿公为程砚秋编写了十二出新戏，多已成京剧[[程派|程派]]代表剧目<ref name=foshan/>。

[[中华民国|中华民国]]时期，院内的古紫藤被居民砍掉。农历癸巳年秋八月（1953年），[[张次溪|张次溪]]作《朱竹宅先生故居记》一文，记述了从朱彝尊故居到顺德邑馆的变迁。同年秋，张次溪约[[章可|章可]]（字受之）为“曝书亭”绘图<ref name=huiguan/>。1984年，顺德邑馆以“朱彝尊故居”的名义被列为[[北京市文物保护单位|北京市文物保护单位]]<ref name=foshan/><ref name=huiguan/><ref>{{Cite web |url = http://www.bj.chinanews.com/news/2005/2005-07-28/1/5931.html |title = 朱彝尊故居 |author =  |date =2005-07-28  |publisher =北京网  |language =  |accessdate =  |quote =  }}</ref>。

1992年，北京市赴[[香港|香港]]招商团推出[[宣武门|宣武门]]地段旧城改造项目，当时该项目因市政基础设施落后、建设成本高而缺少开发商问津<ref name=daobi>{{Cite web |url = http://mt.sohu.com/20170329/n485494030.shtml |title =  北京最大单体百货“崇光百货”要倒闭了？|author =  |date = 2017-03-29 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。1992年9月，北京庄胜房地产开发有限公司在[[北京市工商行政管理局|北京市工商行政管理局]]登记成立，随后获得了宣武门外一地块的开发权<ref name=dajia/>。这块土地位于[[宣武门外大街|宣武门外大街]]东侧，占地面积25公顷，开发项目取名“庄胜城”，包括“[[庄胜广场|庄胜广场]]”和“庄胜二期住宅区”两个部分。此后该地块西部建成“庄胜广场”（内有庄胜崇光百货及写字楼等），而地块西部则长期未能开发，仍保留着[[胡同|胡同]]和[[四合院|四合院]]，顺德邑馆就在该地块里<ref name=daobi/>。

2009年左右，北京庄胜房地产开发有限公司拖欠[[中国银行|中国银行]]湖南省分行、粤财投资等公司巨额债务无法偿还。2009年10月9日，庄胜公司与信达投资、信达北分签订《合作框架协议书》，信达投资拟受让庄胜公司位于宣武门附近的庄胜二期A-G地块的开发项目并投资建设。2009年10月20日，北京信达置业有限公司在北京市工商行政管理局西城分局登记成立，作为上述项目公司。2010年7月28日，庄胜公司作为甲方，信达投资、信达置业作为乙方，信达北分作为丙方，签订《框架协议书补充协议（三）》。2010年7月30日，庄胜公司和信达北分、信达置业、信达投资签订《执行和解协议书补充协议》<ref name=dajia/>。

2011年，信达投资在北京金融资产交易所挂牌转让其持有的信达置业股权，2012年10月31日由中信国安集团公开摘牌收购信达置业100%股权。2012年10月31日，中信国安投资有限公司在北京金融资产交易所摘牌收购信达置业股权，取得庄胜二期A-G地块危改项目。2012年11月，中信国安集团取得了收购信达置业成交确认书等等交易凭证，并办理完成了工商变更登记<ref name=dajia>{{Cite web |url = http://finance.ifeng.com/a/20170421/15313736_0.shtml |title = “国安府”的前世今生 信达庄胜打架 中信国安遭殃 |author =  |date = 2017-04-21 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

自2012年起，中信国安开始拆迁A-G地块，此后五年内投入约200亿元人民币解决了拖延15年的土地拆迁等问题，在2017年初完成了剩余6家企业、将近500户居民的拆迁，并在2017年3月22日取得土地证<ref name=liushi>{{Cite web |url = http://finance.ifeng.com/a/20170520/15392098_0.shtml |title = 北京二环国安府争夺战 200亿巨额国有资产流失？ |author =  |date = 2017-05-20 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。2013年4月3日，[[北京市发展和改革委员会|北京市发展和改革委员会]]向北京市西城区发展和改革委员会作出《关于西城区庄胜二期A-G地块危改项目核准的批复》（京发改〔2013〕605号），同意北京信达置业有限公司开发建设西城区庄胜二期A-G地块危改项目，项目西到[[永光东街|永光东街]]、东到[[香炉营东巷|香炉营东巷]]、南到[[前青厂胡同|前青厂胡同]]、北到[[香炉营头条|香炉营头条]]，建设内容之一是“文物古迹保护”<ref>{{Cite web |url = http://www.zbwmy.com/xmxx/xmdt/2013/05/17144842074955.html |title =西城区庄胜二期A-G地块危改项目获核准  |author =  |date =2013-05-17  |publisher =中国建设工程招标网  |language =  |accessdate =  |quote =  }}</ref>。2014年，“中信·国安府”一期项目楼盘首次开盘<ref name=liushi/>。顺德邑馆（朱彝尊故居）被保留在“中信·国安府”南院的中心，而顺德邑馆周边全部胡同、四合院均在2016年遭到拆除，被彻底毁灭<ref>{{Cite web |url = http://news.163.com/14/1106/14/AACII8VO00014AED.html |title =  如何看待高端住宅里的“文化”？|author =  |date = 2014-11-06  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。2016年北京市“棚户区改造”名单公布，“信达宣东A-G地块”（即原“庄胜二期A-G地块”）被列入<ref name=xianzhi>{{Cite web |url = http://house.ifeng.com/detail/2016_03_28/50733597_0.shtml |title = 北京西城闲置10年地块易主 拆迁接近尾声 |author =  |date =2016-03-28  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

2013年，北京庄胜起诉信达投资、信达北分、信达置业，要求解除签订的有关协议，返还A-G地块的权益，并支付违约金10亿元人民币。2014年12月18日，[[北京市高级人民法院|北京市高级人民法院]]作出民事判决，驳回庄胜地产的全部诉讼请求，支持了信达置业的反诉。此后北京庄胜上诉到[[中华人民共和国最高人民法院|中华人民共和国最高人民法院]]，2017年3月24日最高人民法院二审判决全盘推翻了一审判决，认为信达投资向中信国安出售信达置业股权是恶意违约行为，北京庄胜有权解除有关协议<ref name=dajia/>。

2016年1月，北京市西城区[[两会|两会]]举行“绿色宜居城区环境建设”、“道路建设和交通环境治理”新闻发布会，宣布西城区将修复朱彝尊故居，称朱彝尊故居主体已修缮完毕，还有一户居民未迁出，西城区文物部门将督促主体单位尽快修缮<ref>{{Cite web |url =http://roll.sohu.com/20160114/n434454062.shtml  |title = 康有为朱彝尊故居将修缮 |author =  |date = 2016-01-14 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.aknews.gov.cn/news/china/253808.html |title = 北京西城以“文道”塑古城风貌 |author =  |date = 2016-02-02|publisher =安康新闻网  |language =  |accessdate =  |quote =  }}</ref>。

2016年3月时，顺德邑馆西跨院已拆除，中跨院尚存正房及倒座房数间，重新改建的东跨院尚存正房两间。2016年，在顺德邑馆周边胡同、四合院的大规模拆除中，顺德邑馆残余建筑大部分已被拆除毁灭。到2016年11月，顺德邑馆仅剩大门、倒座房及东跨院正房两间，另外“曝书亭”也保留下来。但是其中仅有大门和门墩仍是古物，其余建筑都是后来改建的<ref name=foshan/>。

除了海柏胡同的顺德邑馆主馆外，顺德邑馆还在[[后青厂胡同|后青厂胡同]]15号（旧门牌）、椿树上二条6号（旧门牌）也有30多间房屋，占地面积约2亩。现均已无存<ref name=foshan/>。

==顺德会馆南馆==
顺德会馆南馆位于北京市西城区天景胡同中部北侧，旧门牌为4号。天景胡同连通了[[南半截胡同|南半截胡同]]与[[菜市口大街|菜市口胡同]]。顺德会馆南馆约占天景胡同北侧超过四分之一的街面，占地面积2.46亩，四进[[四合院|四合院]]，房屋57间<ref name=foshan/><ref name=huiguan/>。中华民国时期，顺德会馆南馆比海柏胡同的顺德邑馆人气更旺，因为天景胡同周边有其他很多会馆以及在京官员的私宅。顺德会馆南馆是在顺德邑馆之后增建的，建成后吸引了很多在北京逗留时间不长的广东文人、学生以及商人，而海柏胡同的顺德邑馆则以老住户居多。[[李慈铭|李慈铭]]的《郇学斋日记》记载了顺德会馆南馆<ref name=huiguan/>。

顺德会馆南馆保存至今。2016年时，院内居民私搭乱建严重，建筑原貌已难见<ref name=foshan/>。
