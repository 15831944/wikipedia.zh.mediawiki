'''北京大学基础医学院'''是隶属于[[北京大学医学部|北京大学医学部]]的一所[[学院|学院]]，正式成立于2000年。

== 历史 ==
前身是1954年9月14日成立的北京医学院基础医学部，1960年2月改名基础医学系。1985年5月[[北京医科大学|北京医科大学]]成立后，基础医学系更名为基础医学院。2000年4月，[[北京大学|北京大学]]与[[北京医科大学|北京医科大学]]合并后，又更名为北京大学基础医学院。<ref name="简介">{{cite web|url=http://sbms.bjmu.edu.cn/xygk/xyjj/29427.htm | language= | title=学院简介 |publisher=北京大学基础医学院 |date=2013-12-08|accessdate=2017-04-12}}</ref>

基础医学院自1978年恢复招生[[研究生|研究生]]以来，已招收1659名研究生，其中博士生636人、硕士生1023人。目前有15名[[博士后|博士后]]在站人员，以及677名在读博士与硕士研究生。<ref name= "简介"/>

== 现状 ==
学院现有13个系、2个研究所，以及1个医学实验教学中心。有“生物学”和“基础医学”2个[[博士|博士]]学位授权一级学科、8个国家重点学科、2个博士后流动站、4个省（部）级重点实验室，还拥有一些国际先进水平的科研基地和实验技术平台。在2003年全国基础医学一级学科评估中，基础医学院在学术队伍、科学研究、人才培养和学术声誉四个方面综合实力名列第一位。<ref name= "简介" />

现有教职工403人，其中教授71人、副教授102人；183人具有博士学位。拥有4名[[中国科学院院士|中国科学院院士]]、1名[[中国工程院院士|中国工程院院士]]、7名[[长江计划特聘教授|长江计划特聘教授]]，5名国家[[青年千人计划|青年千人计划]]专家；获得[[国务院政府特殊津贴|国务院政府特殊津贴]]13人；以及72位[[博士生导师|博士生导师]]。<ref name= "简介"/>

现任院长为[[尹玉新|尹玉新]]<ref>{{cite web |url = http://sbms.bjmu.edu.cn/xygk/xrld/34593.htm | language= |title = 现任领导 |publisher=北京大学基础医学院 |date= |accessdate=2017-04-12 }}</ref>。

== 荣誉成果 ==
1980年至2005年获得各类科技成果奖164项，其中3项[[国家自然科学奖|国家自然科学奖]]、6项[[国家科技进步奖|国家科技进步奖]]、130项部委和[[北京市|北京市]]科技进步奖、12项国家及国际[[发明专利|发明专利]]、13项其他科技成果奖；2000年至2013年共发表3029篇科研论文，其中有6篇获得全国优秀博士论文，1611篇被[[SCI|SCI]]收录；2000年至2013年承担各类科研课题982项，共获科研经费7.3995亿元。其中承担科技部课题68项，获经费25102万元；获得教育部科学基金项目134项，获经费4264万元；承担341项国家自然科学基金项目，获经费26622万元；获得北京市科技项目74项，经费合计2148万元。<ref name= "简介"/>

基础医学院近年来不断加强[[生物技术|生物技术]]和[[医药|医药]]产品的研究和开发。在基因工程药物、肿瘤的免疫治疗、自身免疫病治疗、心血管疾病的基因治疗、干细胞治疗神经系统退行性疾病，以及角膜疾病、诊断试剂研制等领域，取得了动物实验或临床试用的良好效果。<ref name= "简介" />
