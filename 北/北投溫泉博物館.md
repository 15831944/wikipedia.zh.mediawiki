'''北投溫泉博物館'''（'''{{lang-en|Beitou Hot Spring Museum}}'''）是位於[[台灣|台灣]][[台北市|台北市]][[北投區|北投區]]的史蹟保存類[[博物館|博物館]]，過去是公共的[[溫泉|溫泉]]浴場，因此又名'''北投溫泉浴場'''。

== 歷史 ==
[[File:北投溫泉博物館榻榻米活動大廳.jpg|thumb]]活動大廳]]
[[File:北投溫泉博物館電影院.jpg|thumb]]
[[File:北投溫泉博物館的樓梯.jpg|thumb]]
[[File:北投溫泉博物館大浴池.jpg|thumb]]
[[File:北投溫泉博物館小浴池.jpg|thumb]]
[[File:北投溫泉博物館個人浴池.jpg|thumb]]
[[File:北投溫泉博物館的彩繪玻璃.jpg|thumb]]
北投溫泉浴場建於[[1913年|1913年]][[6月17日|6月17日]]，為[[臺北州|臺北州]]仿照[[日本|日本]][[靜岡縣|靜岡縣]]伊豆山溫泉的方式興建而成，是當時規模最大、最華麗的[[公共浴場|公共浴場]]，由[[臺北廳|臺北廳]]公共衛生單位管理經營。[[1921年|1921年]]日本[[皇太子|皇太子]][[裕仁|裕仁]]來台，乃增建二樓30多[[坪|坪]]御休所；[[1997年|1997年]]二月內政部定為[[三級古蹟|三級古蹟]]，之後由[[台北市政府民政局|台北市政府民政局]]整修為北投溫泉博物館。占地約700坪，是兩層樓仿英式[[磚造|磚造]]建築，一樓為磚造[[浴池|浴池]]，二樓為木造[[休息區|休息區]]，並設有[[娛樂室|娛樂室]]等。

[[1945年|1945年]]，[[國民政府|國民政府]]接收[[台灣|台灣]]之後，此浴場曾一度成為[[國民黨|國民黨]][[北投鎮|北投鎮]][[黨部|黨部]]，及[[民眾服務社|民眾服務社]]，除了辦國家慶典外，也出租供民眾作為辦[[喜宴|喜宴]]的場所。當時該建築一樓後面部分，即現博物館辦公室區及浴場部分，也做為「[[台北縣議會招待所|台北縣議會招待所]]」兼營[[游泳池|游泳池]]。該建築同時還有[[警察局|警察局]][[北投分局|北投分局]][[光明派出所|光明派出所]]，共三單位同時使用該建築。國民黨部遷出後，北投溫泉業也沒落了，最後台北縣議會也不再使用為招待所。而光明派出所亦於[[1988年|1988年]][[2月9日|2月9日]]起遷移至新址[[光明路|光明路]]131之2號。於是該建築變成台北縣人員[[陳師堯|陳師堯]]先生占用之住宅，部分空間出租變成[[電影道具|電影道具]]間，開始急速傾頹。

[[1994年|1994年]][[逢甲大學|逢甲大學]][[建築系|建築系]]的學生[[陳林頌|陳林頌]]由於小時候遊歷北投的美好記憶，以北投溫泉谷地的[[北投溪|北投溪]]、[[北投溫泉公共浴場|北投溫泉公共浴場]]、[[北投公園|北投公園]]等北投歷史空間之研究，為題處理逢甲大學[[畢業論文|畢業論文]]，並測繪了北投公共浴場平面、立面圖、北投溪谷地模型。因陳林頌到處呼籲搶救北投公共浴場，不要被拆除成為[[纜車|纜車]]站。

[[1994年|1994年]]，[[台北市立北投國民小學|北投國小]]的[[呂鴻文|呂鴻文]]、[[黃桂冠|黃桂冠]]、[[謝淑玲|謝淑玲]]、[[許家寶|許家寶]]老師開會規劃八十三年度資源班教學主題，決定以「[[鄉土教學|鄉土教學]]」為重點。呂鴻文、黃桂冠老師在探勘[[北投溪|北投溪]]時，進入「[[台北縣議會招待所|台北縣議會招待所]]」，赫然發現這棟建築即是「[[北投溫泉公共浴場|北投溫泉公共浴場]]」。她們清楚知道這棟建築的歷史價值，幾經[[陳情|陳情]]要求保存這棟建築都沒有下文，最後[[蔡麗美|蔡麗美]]老師發起師生[[聯署|聯署]]了一張[[陳情書|陳情書]]，最後這張陳情書拿給居住[[北投|北投]]，時任[[國大代表|國大代表]]的[[許陽明|許陽明]]。而陳林頌的資料也在這時彙集到了許陽明辦公室，並被採納成為重建北投最重要的理論依據與宣傳資料。許陽明於是利用其與當時[[陳水扁|陳水扁]][[市長|市長]]的緊密關係，親自向陳水扁說明，因此很快就指定成了[[三級古蹟|三級古蹟]]，許陽明開始邀集當地居民組成「北投溫泉博物館」推動小組，並提出主張推動成為「居民親手打造自己社區的博物館」，也積極地向[[台北市政府|台北市政府]]爭取經費，最後獲得市政府採納成為市府的重要專案。為了實踐「居民親手打造自己社區的博物館」這個想法，許陽明徵求全部不支薪的社區[[志工|志工]]，組織「北投溫泉博物館」籌備處，以提出社區主張，協調社區主張，並督促實現主張。同時也協助收集溫泉相關文物無償捐贈「北投溫泉博物館」，並記錄志工籌備工作日誌以昭公信，用以實踐這個台灣史無前例的社區運動。

== 修復 ==
1998年北投溫泉博物館由[[台北市政府|台北市政府]]著手修復成史蹟類保存博物館，並在10月31日對外開放。當時花費總計有1億1181萬餘元；工程分為兩個階段，第一階段是拆除一些岌岌可危的建築體，以及補強管線等，花費336萬元；此階段留下來舊有的溫泉系統包括分水管、[[磺底石|磺底石]]、溫泉陶管，[[口徑磚|口徑磚]]等，這些在營造籌備階段由許陽明、陳林頌等人在工地發現保存，或專程收集的無償文物。

第二階段則是修復與再利用工程，花費9845萬209元。此次主要是為了與鄰近的[[北投溫泉親水公園|北投溫泉親水公園]]一起規劃。這一段時期的修復，參考了許多舊照片和耆老口述，將溫泉博物館內部與外部盡量回復[[日治時期|日治時期]]舊貌。可惜博物館仍有不少珍貴的文物已經遺失，如西側一樓[[拱窗|拱窗]]的[[彩繪玻璃|彩繪玻璃]]，就已在修護前遺失，現在是複製品。

== 現況 ==
該館現已列為台灣第三級古蹟。館內規劃有一樓二樓展區，以北投溫泉發展史為主題，從溫泉的原理、[[北投石|北投石]]、火山（一樓展場）到後期北投溫泉鄉發展史、[[凱達格蘭|凱達格蘭]]族歷史「北投社原住民」、北投產業和台灣好萊塢等規劃（二樓展場）。目前館內參觀人數有限制，同時不得超過一百人在館內參觀，入館也必須更換拖鞋。戶外由於草坪優美，景觀典雅，常可見到即將結婚的新人在此拍攝[[婚紗照|婚紗照]]。溫泉博物館館內最珍貴的眾多北投石，全部由許陽明出面向中華礦石協會，特別是陳嘉林理事長，還有諸多礦石收藏家商量，呼籲「讓北投石回家」時，紛紛捐出讓許陽明一起交由溫泉博物館收藏的珍貴國寶。
