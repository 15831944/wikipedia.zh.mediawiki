'''北京市北海幼儿园'''，位于[[北京市|北京市]][[西城区|西城区]][[地安门西大街|地安门西大街]]22号，隶属北京市西城区西城区教育委员会，是北京市一级一类幼儿园。

==简介==
前身是[[中华民国|中华民国]]时期位于[[文津街|北海三座门]]的[[北京市民政局|北平市社会局]]托儿所。1949年3月，由[[中国人民解放军北平市军事管制委员会|中国人民解放军北平市军事管制委员会]]接收，当时是专门寄养革命干部和革命军人子女的实验托儿所。同年7月，经军管会批准，迁入北海东北角的[[先蚕坛|先蚕坛]]，同年更名为北京市北海幼儿园<ref name=zuji/><ref>北京市北海幼儿园，北京教育：普教版2011年第01期</ref>。第一任园长[[于陆琳|于陆琳]]找各地方老干部要来建筑材料，找[[梁思成|梁思成]]设计了该园<ref name=yull/>。

1999年11月，北京市西城区后海幼儿园并入北海幼儿园，成为北海幼儿园分园（托儿部），专招两岁半左右的学前儿童。北海幼儿园是公办寄宿制幼儿园，隶属于北京市西城区西城区教育委员会，是北京市一级一类幼儿园，2001年被评为首批北京市示范幼儿园<ref name=zuji>{{Cite web |url = http://www.bhbaby.org/page4.aspx?id=24 |title = 发展足迹 |author =  |date =  |publisher =北京市北海幼儿园  |language =  |accessdate =2018-02-18  |quote =  }}</ref>。

2001年，北海幼儿园在全国首创行政教研的干部培养方式，2005年至2007年连续3年在[[中华人民共和国教育部|中华人民共和国教育部]]贯彻落实《纲要》经验交流会上展示成果。北海幼儿园多次承担国家、北京市、西城区科研课题任务，还常接待各国国家元首、政要、社会各界友好人士及幼教界人士来访<ref name=zuji/>。

2014年1月，[[贵阳市|贵阳市]]与北京市西城区签署两地教育共建合作框架协议。同年9月1日，[[北京八中|北京八中]]贵阳分校正式开学。同年10月20日，北海幼儿园在[[贵阳|贵阳]]花果园开办的北京市北海幼儿园贵阳分园开园，并在办园两年后通过[[贵州省|贵州省]]示范幼儿园验收<ref>{{Cite web |url =  http://dsb.gzdsw.com/html/2014-10/21/content_23014.htm|title = 京筑两地7对学校结为“友好姊妹学校” |author =  |date = 2014-10-21 |publisher =贵州都市网  |language =  |accessdate =  |quote =  }}</ref><ref name=zuji/>。
