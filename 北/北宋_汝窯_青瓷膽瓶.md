'''北宋汝窯青瓷膽瓶'''，燒製年代約為北宋哲宗至徽宗（1086-1106）之二十年間。為清宮傳世之[[汝窯|汝窯]]（汝官窯）作品。現存於[[國立故宮博物院|國立故宮博物院]]。

== 尺寸 ==
典藏單位所測為：高17.8公分、口徑3.6公分、足徑8.4公分

== 外觀 ==
# 細頸，腹部形似垂膽，圈足部分略呈外撇。
# 整體罩以溫潤如玉之天青色釉，佈有均勻[[開片|開片]]。
# {{cn|口沿以及足沿鑲有復古鎏金銅扣，為[[乾隆|乾隆]]年間御旨下令修復之痕跡。}}
# {{cn|底部散佈芝麻狀支釘，並以放射狀呈現有[[乾隆|乾隆]]御製詩《詠官窯溫壺》一首。}}

== 形制相似之文物比較 ==
{{section OR}}
* 膽瓶與{{which|玉壺春瓶}}同為裝飾用花器、型制相似，皆為早期佛家淨水瓶的分支。
* 同時期的膽瓶與玉壺春瓶多為喇叭狀的開口，因此有一說此膽瓶最初燒製時之型制為寬口喇叭狀。
