'''北海道道468号清水谷足寄线'''（{{lang-ja|北海道道468号清水谷足寄線}}）是一条连接[[北海道|北海道]][[十胜综合振兴局|十胜综合振兴局]]管内[[河東郡_(日本)|河東郡]][[上士幌町|上士幌町]]和[[足寄郡|足寄郡]][[足寄町|足寄町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。


== 道路概况 ==
* 起点：北海道河東郡上士幌町上士幌清水谷
* 終点：北海道足寄郡足寄町上利別盐幌
* 总距离：47.1千米（内、重複区間5.7千米）

== 经过的自治体  ==
* 十勝综合振兴局
** 河東郡上士幌町
** 足寄郡足寄町

== 主要连接道路 ==
* [[国道273号|国道273号]]（起点）
* [[北海道道88号本别留边蕊线|北海道道88号本别留边蕊线]]（足寄町茅登本町 - 足寄町喜登牛）：重複
* [[北海道道663号植坂足寄停车场线|北海道道663号植坂足寄停车场线]]（足寄町茂喜登牛）
* [[国道242号|国道242号]]（终点）

== 历史 ==
* 1963年（[[昭和|昭和]]38年）10月23日 - 路線勘定（北海道告示第2333号）

== 其他 ==
* 在起点至北海道道88号交点間的1車線区間设置有待避所2个。
