'''北京有轨电车'''的历史始于1899年有[[西门子|西门子]]公司建设的连接[[永定门|永定门]]至[[京奉铁路|京奉铁路]][[马家堡站_(铁路)|马家堡站]]的电车线路，亦是中国第一条[[有轨电车|有轨电车]]线路。北京主要有轨电车路网的发展始于1924年，直至1966年有轨电车完全被[[无轨电车|无轨电车]]取代。21世纪后北京开始陆续引入[[现代有轨电车|现代有轨电车]]系统。

== 历史 ==
===19世纪===
[[File:Tramway_of_Machiapu_Station.jpg|thumb]]
1897年6月，[[津芦铁路|津芦铁路]]延伸到马家堡。马家堡火车站建成后，为了方便京城人员乘车，德国西门子公司与[[中国铁路总公司_(清朝)|铁路总公司]]签订合约，为北京修建连接永定门与马家堡火车站的有轨电车线路。<ref name=xiaoshi>{{Cite web |url = http://www.peoplerail.com/rail/show-468-88317-1.html |title =  马家堡 消失的老站|author =  |date =2010-12-15  |publisher = 人民铁道网 |language =  |accessdate =  |quote =  }}</ref><ref name=sc>{{cite web |title=Across borders Siemens and China – Milestones in a successful  partnership |url=https://new.siemens.com/global/en/company/about/history/news/siemens-in-china.html |accessdate=2019-03-26}}</ref>该线路于1899年5月竣工，6月30日通车，时刻表上称之为“Imperial Chinese Electric Tramway”。线路全长3公里，使用1,435mm[[标准轨|标准轨]]，设永定门与马家堡两站，并于马家堡站与津芦铁路接轨，单程票价为10分。<ref name="gnwtl">{{cite book |author1=Peter Crush |title=Imperial Railways of North China |isbn=978-7516605646 |accessdate=2019-03-26}}</ref>该线路拥有机车、拖车各4辆，每辆车含16个坐席并可站立14人。机车配有{{Tsl|en|Siemens & Halske}} B型电机，最高速度可达20km/h，为当时世界最快。<ref name=sc/>
<ref name="str">{{cite journal |author1=Street Railway Publishing Co. |title=Imperial Railways of North China |date=January to June, 1901 |volume=17 |pages=152 |url=https://books.google.com/books?id=AchJAQAAMAAJ&dq=Machiapu+siemens&source=gbs_navlinks_s |accessdate=2019-03-26}}</ref>马家堡电车站距离马家堡火车站约400米，电车修理车间与发电厂亦位于此站中。发电厂配有两台75马力的复式蒸汽机与两台45千瓦的500V发电机，为整条线路提供电力。<ref name="str"/>

1900年，[[义和团运动|义和团运动]]爆发<ref name="cns2012">{{cite news |title=北京第一条电车线引自德国 义和团进京后被毁 |url=http://www.chinanews.com/cul/2012/02-29/3708247.shtml |work=北京晨报 |date=2012-02-29}}</ref>，同年6月12日，马家堡火车站被义和团焚毁，有轨电车被一同砸毁拆除。<ref name=sanci>{{Cite web |url = http://www.takefoto.cn/viewnews-1243895.html |title = 永定门火车站的三次变迁 铁轨上滚滚而去的历史 |author =  |date =2017-08-20  |publisher =北晚新视觉   |language =  |accessdate =  |quote =  }}</ref><ref>{{cite journal |author1=于音、宋卫忠 |title=京城第一个火车站——马家堡车站考 |journal=首都师范大学学报:社会科学版 |date=2004(z2) |pages=122-126}}</ref>。

===20世纪===
[[File:Peking_1930s.jpg|thumb]]
==== 民国时期 ====
1913年，北京的居民人数已达130余万人，而北京的交通依然十分落后，公共交通事业亟待发展。当时，许多商人先后倡办电车事业，但都以失败告终。北洋政府在意识到兴办电车事业的必要后于1913年10月拟议组织北京电车股份有限公司，1921年6月30日，电车公司正式成立。北洋政府从[[中法实业银行|中法实业银行]]借款作为官股，同时从市民中公开募集商股，官股商股各半，股本共四百万元。其中购股最多的是[[金城银行|金城银行]]与[[盐业银行|盐业银行]]，为电车公司的两家最大股东。<ref name="dasl">{{cite book |author1=北京市档案馆 |title=北京电车公司档案史料 |publisher=北京燕山出版社 |isbn=7540200200 |accessdate=2019-03-26}}</ref>

1922年，电车公司向法国订购一批{{Tsl|en|J. G. Brill Company|Brill}} 21E型车盘，裕信营造厂承办制造电车车身。路轨则购自法国{{Tsl|en|Thomson-Houston Electric Company|CFTH}}公司，由益昌公司负责铺设，全长49380米，轨距为[[米轨|1,000mm]]。铺轨工作于1923年5月开始，1924年8月下旬基本完工。12月17日在[[天安门|天安门]]南侧举行通车典礼，18日第1路正式通车，月底第2路通车，第3、4、5路也相继开通。1929年8月5日，环路电车开行。有轨电车因票价低，速度较快，载客量大，抢走了人力车夫的生意，引发了人力车夫的不满，12月22日发生人力车夫捣毁电车事件，砸坏机车43辆、拖车20辆，造成电车工人8人重伤，电车停驶18天。<ref name=shihua>(Chinese) [http://www.bjqx.org.cn/qxweb/n10473c119.aspx You Long,  "北京有轨电车史话" 《前线》] {{webarchive |url=https://web.archive.org/web/20090109053703/http://www.bjqx.org.cn/qxweb/n10473c119.aspx |date=January 9, 2009 }} 2008-03-06</ref>1930年，第6路通车。1931年1月，第6路延长至和平门。<ref name="dasl"/>

这一时期，电车公司克服重重阻力，使得电车系统竣工通车，事业初具规模。但由于购置设备开支巨大，捐税名目繁多，而股资迟迟无法收齐，逃票现象严重，导致电车公司不敷支出，背上高额债款。而随后的局势动荡也严重地影响了电车公司，营业亏损严重，发展停滞不前。<ref name="dasl"/>

==== 抗战时期 ====
1937年[[七七事变|七七事变]]后北平沦陷，日伪派员接管电车公司。1938年1月，天桥至永定门通车。1939年，电车公司向日本岐阜县{{Tsl|jp|美濃電気軌道|美浓电气轨道}}购得电车10辆，并于次年改造为拖车使用。1940年购得{{Tsl|jp|武蔵中央電気鉄道1形電車|武藏中央电铁1型电车}}5辆，1941年购得{{Tsl|jp|京王電気軌道23形電車|京王电铁23型电车}}1辆，自造机车二辆、拖车六辆。同年日本{{Tsl|jp|洞爺湖電気鉄道|洞爷湖电铁会社}}全线废止，资产全部售予电车公司。1942年购得上海旧机车与拖车共计25辆。1943年，虎坊桥至菜市口之延长线路建成通车，而虎坊桥至和平门线路停驶并于4月拆除。<ref name="dasl"/>

日伪政府统治的这一时期电车公司几次添购车辆，运营略有发展，账面上甚至连续几年出现盈余，但由于购进的电车都是受日寇胁迫购进的淘汰货，且原有车辆已使用多年，无法长期使用，遂于1944年下半年快速由盛转衰，抗战胜利后，电车运营几近瘫痪，只有49辆电车可以行驶，平均每日出车不足十辆。战后，国民党政府接收北平，并对电车事业进行整顿，1946年1月，电车公司自造新车4辆。同年，平均每日出车数量恢复至机车33辆、拖车26辆。但由于国内战事，电车公司再次衰落，北平解放前夕，电车公司已陷入瘫痪状态。<ref name="dasl"/>

==== 新中国时期 ====
1949年2月，[[北平和平解放|北平和平解放]]，人民政府接管电车公司。3月，电车公司开始了“修复百辆车”运动。4月25日，发生北平电车厂纵火案，烧毁机车42辆，拖车17辆。1950年3月16日，电车环行路开通。1951年，电车修造厂成功仿制100型机车与200型拖车。1952年，电车修造厂对武藏中央电铁1型电车与京王电铁23型电车进行分析研究，于年底造出10辆52式有轨电车，直至1954年，共生产55辆52式有轨电车。<ref name="bs">{{cite web |author1=阿古 |title=改革开放四十年，有轨电车大变身 |url=http://www.sohu.com/a/270760269_492547 |accessdate=2019-03-26}}</ref>1955年10月，天桥至红桥电车线路竣工通车。

由于有轨电车轨道磨损、锈蚀严重，无法修补，重置费用庞大；车辆形式不一，零件亦不一致，购置困难，而本地材料易损坏，维护成本甚高<ref name="dasl"/>，且有轨电车噪声大，速度慢，铺设的轨道严重影响城市改造与行车安全，北京市决定另行改建无轨电车。1956年10月17日，京一型无轨电车试车成功，1957年8月，北京首条无轨电车线路全线通车<ref>{{cite web |author1=贾晓燕 |title=“大辫子”电车在北京街头整整跑了60年 |url=http://finance.qianlong.com/2017/0707/1834111.shtml |publisher=北京日报 |accessdate=2019-03-26}}</ref>，1959年3月9日，北京内城有轨电车停驶，1966年5月6日，北京最后一条有轨电车线路停驶。<ref>{{cite journal |author1=张历生 |title=北京最后一条有轨电车停驶 |journal=档案文化 |pages=60}}</ref>

===21世纪===
==== 前门大街 ====
[[File:Qianmen_Dajie_Tram.jpg|thumb]]
前门大街从2004年开始大修，2008年5月28日完工。<ref>[http://beijing.qianlong.com/3825/2008/05/28/3562@4464640.htm 前门大街改造完工，千龙网，2008-05-28]</ref>此次改造中，前门大街重新铺轨，恢复有轨电车。[[北京地铁车辆厂|北京地铁车辆厂]]<ref name="clc">{{cite web |author1=王萌 |title=有轨电车通过严格测试 前门铛铛车最多装百人 |url=http://beijingww.qianlong.com/247/2008/05/14/82@57103.htm |publisher=北京娱乐信报}}</ref>参考52式有轨电车的样式<ref>{{cite web |author1=张楠 |title=前门大街8月7日开街 当当车暂不对游客开放 |url=http://news.sohu.com/20080729/n258450772.shtml |publisher=北京晚报}}</ref>为天街公司生产2辆北京前门超级电容有轨电车。北京前门超级电容有轨电车装有2台牵引电机，使用VVVF调速，最高速度为10km/h<ref name="cjdr">{{cite journal |author1=尚敬，刘良杰，杨大成 |title=北京前门超级电容有轨电车交流传动控制系统 |journal=交流技术与电力牵引 |date=3/2008 |page=45 |accessdate=2019-03-26}}</ref>。为了还原真实，电车的电缸与闸把均为从大连购得的废旧电车部件。<ref name="kt">{{cite news |author1=翟烜 |title=北京铛铛车重现前门大街 票价定为20元 |url=http://news.sina.com.cn/c/2009-01-02/004114969586s.shtml |accessdate=2019-03-26 |work=京华时报 |date=2009年01月02日}}</ref>线路全长840米，使用1,435mm标准轨，设前门与珠市口两站。供电方式为超级电容，无需架设架空线，充满电只需3-5分钟，一次充电可空载运行16公里。<ref name="clc"/>

2009年1月1日，前门大街有轨电车正式运营，单程票价为20元。<ref name="kt"/>

==== 西郊线 ====
{{main|北京地铁西郊线}}
[[File:BJXJ11_entering_West_Gate_of_Summer_Palace_Station_(20171110153956).jpg|thumb]]
西郊线是[[北京地铁|北京地铁]]的一条现代有轨电车线路，由[[北京公交有轨电车有限公司|北京公交有轨电车有限公司]]运营。2011年，西郊线开工建设，2017年12月30日开通试运营。线路全长9.4千米，共6站，在起点[[巴沟站|巴沟站]]可换乘[[北京地铁10号线|北京地铁10号线]]，终点为[[香山|香山]]。全程票价4元，最低3元。列车采用接触网供电，车辆以[[安萨尔多百瑞达|安萨尔多百瑞达]]{{link-it|安萨尔多百瑞达Sirio|AnsaldoBreda Sirio|Sirio型}}为基础，由[[中车大连机车车辆|中车大连机车车辆]]制造。<ref>{{cite news|author=周超|url=http://www.bj.chinanews.com/news/2017/0522/58401.html|title=西郊有轨电车拟年底运营|publisher=中国新闻网|date=2017-05-22|accessdate=2017-06-19}}</ref>

2018年1月1日，西郊线车辆转向架出现故障，随后发生脱轨与放飏事故。<ref>[http://companies.caixin.com/2018-01-22/101201037.html 记者 孙丽朝：《北京西郊线电车脱轨查明 调度失利为直接原因》，财新网 2018年01月22日 ]</ref>

== 未来发展 ==
===亦庄有轨电车===
亦庄有轨电车是位于北京[[亦庄经济技术开发区|亦庄经济技术开发区]]的有轨电车系统，其中T1线已于2018年下半年开工，预计于2019年试运行。<ref>{{cite web |author1=蒲长廷 |title=北京亦庄有轨电车T1线2019年试运行 |url=http://beijing.qianlong.com/2018/0919/2837680.shtml |publisher=千龙网 |accessdate=2019-03-27}}</ref>其他线路仍在规划中。

===顺义区现代有轨电车===
顺义区现代有轨电车是顺义区的首个有轨电车系统，亦是北京地铁公司首次取得的PPP项目。其中T2线已于2017年7月14日开工，全线19.84公里，设22座车站，并与[[首都机场T3航站楼|首都机场T3航站楼]]连接，预计2019年全线贯通试运营。<ref>{{Cite web|url=https://www.bjsubway.com/news/qyxw/yyzd/2017-07-14/128602.html|title=顺义区现代有轨电车T2线正式开工|website=www.bjsubway.com|access-date=2018-01-24}}</ref>

===丰台河西有轨电车===
河西有轨电车是北京市丰台区规划的两条有轨电车线路，T1线从园博园至[[稻田站|稻田站]]，并在稻田站与[[北京地铁房山线|北京地铁房山线]]换乘；T2线从T1线引出，直达青龙湖附近。因涉及周边地块和路网配套等问题，开通时间暂未确定。<ref>{{cite web |author1=刘冕 |title=丰台区将建两条有轨电车线路 |url=http://bj.people.com.cn/n2/2017/0309/c82838-29825341.html |publisher=北京日报 |accessdate=2019-03-27}}</ref>
