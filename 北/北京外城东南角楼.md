'''北京外城东南角楼'''是[[北京外城|北京外城]]东南角的角楼，复建后辟为角楼图书馆，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[龙潭东路|龙潭东路]]9号。

==简介==
该角楼修建于[[明朝|明朝]][[嘉靖|嘉靖]]三十二年（1553年）。[[清朝|清朝]]灭亡后，20世纪初北京外城各门、楼均有较大程度的损坏，外城东南角楼于1920年代至1930年代自然坍塌<ref>{{Cite web |url = http://www.chinanews.com/house/2014/01-09/5717173.shtml |title =北京东南角楼复建工程年内完工  |author =  |date =2014-01-09  |publisher =  中国新闻网|language =  |accessdate =  |quote =  }}</ref>。北京外城东南角楼平面呈方形，通高15米，面阔、进深均6米，两层共辟箭孔12个，没留下实测资料<ref name=wangong>{{Cite web |url =http://www.bjdch.gov.cn/n1683350/n4085736/c4448322/content.html  |title = 北京外城东南角楼修复工程完工 |author =  |date = 	2015-12-08 |publisher = 数字东城 |language =  |accessdate =  |quote =  }}</ref>。该角楼存有部分遗址，位于后来的[[中国医学科学院肿瘤医院|中国医学科学院肿瘤医院]]附近<ref name=ni/>。1920年代，[[瑞典|瑞典]]人[[喜仁龙|喜仁龙]]著的《北京的城墙和城门》一书收录了北京外城东南角楼的一张照片<ref name=xirl>{{Cite web |url =http://city.sina.com.cn/focus/t/2014-01-09/100641244.html  |title = 北京将复建左安门角楼 部分城墙原址修复 |author =  |date =  2014-01-09|publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

2000年代，北京市[[崇文区|崇文区]]提出建设“[[天坛|天坛]]文化圈”，以东北角的[[北京明城墙遗址公园|北京明城墙遗址公园]]及[[北京内城东南角楼|北京内城东南角楼]]，西北角的[[正阳门|正阳门]]箭楼，西南角复建的[[永定门|永定门]]城楼，东南角计划复建的[[左安门|左安门]]城楼，形成“天坛居中，几楼环绕”的城市空间格局<ref>{{Cite web |url =http://news.hexun.com/2007-12-11/102222080.html |title =北京崇文区加快天坛文化圈建设  |author =  |date = 2007-12-11 |publisher =和讯  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.sohu.com/20090918/n266826677.shtml |title = 北京崇文区长牛青山谈“天坛文化圈”发展战略 |author =  |date = 2009-09-18 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://unn.people.com.cn/GB/22220/173431/173432/10402928.html  |title =  第四届文博会崇文区亮出天坛文化圈发展战略|author =  |date = 2009-11-18 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。2010年初，北京市[[崇文区|崇文区]][[两会|两会]]上，北京市崇文区人民政府区长[[牛青山|牛青山]]仍表示有计划今后复建[[左安门|左安门]]<ref name=ni/>。

2010年7月，崇文区撤销，与[[东城区_(1958–2010)|东城区]]合并为新的[[东城区_(北京市)|东城区]]。此后左安门复建计划被搁置，转而开始研究位于左安门以东的北京外城东南角楼的复建。2011年初北京市东城区人民代表大会上，东城区称2011年将重点推进“十项历史文化保护区修缮整治工程”，包括北京外城东南角楼复建<ref name=ni/>。

2011年9月23日，北京市东城区人民政府官方网站公示了东城区拟修复北京外城东南角楼的原因、复建的地理位置和复建后的效果图。该公告称，北京外城东南角楼位于[[左安门桥|左安门桥]]东260米处，长青路东南转角处至护城河，原址现已成为护城河、河道岸坡及绿地，为保护古都风貌，修复北京城垣历史标志性建筑物，经征求专家意见并经考古勘探后，拟在上述位置修复北京外城东南角楼。根据《[[中华人民共和国城乡规划法|中华人民共和国城乡规划法]]》的规定，北京市东城区文化委员会在网上就该拟定项目公示，公示期30天<ref name=ni>{{Cite web |url = http://roll.sohu.com/20110926/n320520851.shtml |title = 京外城东南角楼拟修复 |author =  |date = 2011-09-26 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

2011年，文物部门在对北京外城东南角楼的考古勘探中，在左安门附近找到了老城砖，但仅能大体确定角楼的区域，无法精确定位<ref name=xirl/>。

2012年，在文物古建专家[[王世仁|王世仁]]指导下，完成了北京外城东南角楼的修复初步设计。但受地下管线改移、考古挖掘等因素影响，截至2015年1月东城区东城区文化活动中心、外城东南角楼恢复和[[玉河|玉河]]南区工程未实现预期建设目标。2015年1月，在东城区[[两会|两会]]上，“外城东南角楼景观恢复”再次出现在政府工作报告中，报告提出在2015年内完成该工作<ref>{{Cite web |url = http://www.chinanews.com/sh/2015/01-13/6964218.shtml |title = 北京东城区时隔3年重提东南角楼复建 |author =  |date = 2015-01-13 |publisher =  中国新闻网|language =  |accessdate =  |quote =  }}</ref>。

按照北京市及东城区有关领导的要求，北京市东城区住房和城市建设委员会（北京市东城区历史文化名城保护工作委员会办公室）、北京市东城区文化委员会经与[[北京市规划和国土资源管理委员会|北京市规划和国土资源管理委员会]]、[[北京市水务局|北京市水务局]]、东城区相关部门多次开会研讨和专家考古论证后，形成了角楼修复工程方案，并获[[北京市人民政府|北京市人民政府]]批复，项目被列为东城区历史文化名城重点保护项目和重要政府折子工程项目，于2014年启动实施。该项目位于[[左安门桥|左安门桥]]东、滨河路（[[龙潭东路|龙潭东路]]）东南转角处；北至左安东里小区，东南临护城河，西距[[左安门内大街|左安门内大街]]260米。建设单位是东城区文化委员会，代建单位是正阳恒瑞公司。包括角楼主体工程及配套市政改移工程。总投资估算3975.07万元人民币，其中地上文物主体建筑修复费用由[[北京市文物局|北京市文物局]]安排574万元修缮专项经费，其余投资由东城区历史文化名城保护专项资金安排。角楼主体建筑占地面积约870平方米，总建筑面积约1160平方米；修复长度南北向总长22.43米、东西向总长41.52米。角楼主体建筑最高点高14.4米，城台高6.4米。同时对影响角楼施工的现状约200米市政管线和道路进行改移，对影响角楼工程的河道护堤进行加固<ref name=wangong/>。

2015年12月，外城东南角楼复建工程除外线和绿化外，其他工程已全部完工<ref name=wangong/>。2016年7月，外城东南角楼复建工程的基建部分正式完工<ref name=guanming>{{Cite web |url =  http://www.xdcb.net/html/2016-12/16/content_2_6.htm |title =东南角楼图书馆征集馆名  |author =  |date = 2016-12-16 |publisher = 新东城报 |language =  |accessdate =  |quote =  }}</ref>。复建中参考了老照片和史料，按原比例复建。复建后的外城东南角楼位于[[北京二环路|北京南二环]]护城河内侧、东南转角处河道岸坡，与左安门内路东侧的明城墙遗存左安门值房相映成趣<ref name=piwei/>。复建后的角楼并非原址，而是在原址以北10.9米处。因为南侧是拓宽后的护城河道，并设有人行道，为发挥人行道分卸洪峰的作用，所以只好向北平移10.9米复建角楼。为了复建角楼，[[龙潭东路|龙潭东路]]在角楼以北段进行了道路变线，人行道最窄处已不足1米，同时角楼东南角城墙2米多悬空伸出<ref>{{Cite web |url =http://finance.ifeng.com/a/20160415/14326426_0.shtml  |title = 北京复建左安门角楼 “悬空”城墙引热议 |author =  |date =2016-04-15  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。

复建后的角楼东侧、南侧分别开有两排六个箭窗，合计12个。角楼上下共两层，建筑面积约1160平方米，使用面积约800平方米。复建后的角楼被辟为公益图书馆，主要收藏与老北京历史文化有关的图书，图书馆一层用于读书，二层开办各类文化展览、讲座以及文化活动<ref name=piwei>{{Cite web |url =http://bj.wenming.cn/jj/rdxw/201601/t20160107_3071711.html  |title =外城东南角楼经修复将辟为历史文化图书馆  |author =  |date =2016-01-07  |publisher =  中国文明网|language =  |accessdate =  |quote =  }}</ref>。2016年12月起，角楼图书馆面向公众征集馆名<ref name=guanming/>。2017年10月28日，该馆以“角楼图书馆”之名对公众开放，角楼图书馆是东城区第二图书馆分馆，以政府购买服务的方式委托专业文化公司运营<ref>{{Cite web |url =http://bj.people.com.cn/n2/2017/1028/c82840-30865636.html  |title = 京城左安门角楼变身图书馆 对市民免费开放 |author =  |date =2017-10-28  |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://epaper.ynet.com/html/2017-10/31/content_267623.htm?div=3   |title = 左安门角楼变身图书馆免费开放 |author =  |date = 2017-10-31 |publisher = 北京青年报 |language =  |accessdate =  |quote =  }}</ref>。
