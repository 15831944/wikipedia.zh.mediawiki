'''北京地铁燕房线'''是[[北京地铁|北京地铁]][[北京地铁房山线|房山线]]向[[燕山石化|燕山石化]]地区和[[房山城关|房山城关]]等地的延长线。线路从房山线西延[[阎村北站|阎村北站]]起，向西到[[饶乐府站|饶乐府站]]分岔，主线向西北到达[[燕山石化|燕山石化]]产业区南端的[[燕山站_(地铁)|燕山站]]，支线向西南到达[[周口店镇站|周口店镇站]]。线路均为高架线，全长20.5千米，其中主线14.4千米，支线6.1千米；设计时速100千米。在房山线[[阎村车辆段|阎村车辆段]]北侧设[[阎村北停车场|阎村北停车场]]。

与此同时，房山线西延2.2千米到[[阎村东站|阎村东站]]，线路初期、近期采用4节编组B型列车，同房山线在阎村东站进行[[跨月台轉車站|同台换乘]]；远期燕房线扩编成6节，两线贯通运营<ref>{{cite web|url=http://hps.mep.gov.cn/jsxm/xmsl/201310/W020131028385106132870.pdf|title=北京轨道交通燕房线工程环境影响报告书简本|author=北京市基础设施投资有限公司|publisher=环境保护部|date=2013-10-18|accessdate=2013-12-04}}</ref>。但实际上燕房线在投入运营之时，轨道及道岔便已经与房山线的主线实施了连接。

== 建设 ==

燕房线曾于2011年进行过一次环境影响评价，由于当时《北京市城市轨道交通近期建设规划调整（2007-2016年）》尚未获得批准，以及涉及[[南水北调|南水北调]]的环境影响问题，[[中华人民共和国环境保护部|环境保护部]]退回了环评<ref>{{cite web|url=http://news.sina.com.cn/c/2011-10-18/005423317659.shtml|title=北京轨交燕房线工程因环评未过被叫停|date=2011-10-18|accessdate=2012-01-13}}</ref>。2014年初，燕房线环评通过<ref>{{Cite web |url =http://www.ce.cn/xwzx/gnsz/gdxw/201408/13/t20140813_3342120.shtml  |title =中国大陆首批全自动无人驾驶列车将投入燕房线  |author =  |date = 2014-08-13 |publisher =中国经济网  |language =  |accessdate =  |quote =  }}</ref>。
燕房线先期建设主线，计划投资91亿元<ref>{{Cite web|url=http://finance.huanqiu.com/data/2011-08/1888347.html|title=北京轨交5年规划再增100公里 4条新线今年开工|date=2011-08-08|accessdate=2011-08-25|publisher=环球网|author=[[北京晚报|北京晚报]]|deadurl=yes|archiveurl=https://web.archive.org/web/20120301072442/http://finance.huanqiu.com/data/2011-08/1888347.html|archivedate=2012-03-01}}</ref>，于2013年底开工，2017年12月30日通车<ref>{{cite news|url=http://bjrb.bjd.com.cn/html/2016-01/27/content_348428.htm|title=西郊线明年有望开通|author=曹政|work=北京日报|date=2016-01-27|accessdate=2016-01-28|deadurl=yes|archiveurl=https://web.archive.org/web/20160206010619/http://bjrb.bjd.com.cn/html/2016-01/27/content_348428.htm|archivedate=2016-02-06}}</ref>。

== 运营 ==

燕房线是中国第一条具有完全自主知识产权的轨道交通全自动运行线路<ref name=bjwb>{{Cite web |url = http://www.chinanews.com/gn/2017/12-31/8413342.shtml|title = 北交大技术助力燕房线无人驾驶 四保险确保行车安全 |author =  |date = 2017-12-31 |publisher =中国新闻网  |language =  |accessdate =  |quote =  }}</ref>，是按照世界上列车运行自动化等级的最高级别（GoA4）建设，能够自己完成上电自检，自动发车离站、自动到站开闭车门、自动折返、运营后自动回库休眠、自动洗车等全套操作。燕房线的信号系统和综合监控系统的总集成商是[[北京交通大学|北京交通大学]]的交控科技股份有限公司，北京交通大学研制的基于通信的列车自动控制系统被应用于燕房线<ref name=bjwb/><ref name=xhw/>。

燕房线的运营方为[[北京市轨道交通运营管理有限公司|北京市轨道交通运营管理有限公司]]，简称北京轨道运营，是北京市轨道交通建设管理有限公司设立的全资子公司，是北京市市政府批准的北京第三家轨道交通运营商。公司主营业务涵盖轨道交通客运服务、车辆及设备设施维修服务、技术研发、咨询培训、广告、民用通信等版块。<ref>{{cite web |author1=北京市轨道交通运营管理有限公司 |title=公司介绍 |url=http://www.bjmtroa.com/indishare/sitemaster.nsf/(page)/bjdteSecond?opendocument&database=bjmtroa/gywm.nsf&view=vwWebFlowForShow&path=3}}</ref>
北京轨道运营在燕房线的服务中吸取了[[北京地铁运营公司|北京地铁运营公司]]和[[京港地铁|京港地铁]]公司的优点，为每一座车站增设了带有空调的候车室<ref>{{cite news |author1=北京日报 |title=燕房线早高峰5分钟一趟 全线设站台休息室 |url=http://bj.people.com.cn/n2/2017/1229/c82840-31085497.html |agency=人民网}}</ref>。此外，燕房线的车站和车厢的中英文广播格式也与[[京港地铁|京港地铁]]运营的4号线、14号线和16号线较为相似。

== 列车 ==
燕房线采用两动两拖4节B型列车，列车定员1262人，最高时速80公里，由[[中车长客股份公司|中车长客股份公司]]研制，<ref>[http://news.cyol.com/yuanchuang/2017-12/30/content_16827778.htm     王培莲：“北京燕房线地铁列车投入运行”，《中国青年报·中青在线》2017-12-30]</ref>正常情况下全自动驾驶。是中车长客股份公司2016年12月为香港[[南港岛线|南港岛线]]量身研制的中国首列世界最高等级的全自动驾驶地铁列车。北京燕房线是中国内地首条自动驾驶运营轨道交通线，也是北京市首条采用自主知识产权、全自动运行的地铁线路<ref>{{Cite web |url = http://www.ben.com.cn/zc/sjc/201712/30/t20171230_11077759.html |title = 燕房线、S1线、西郊线齐开 北京轨道交通总里程608公里 |author =  |date = 2017-12-30 |publisher = 京报网 |language =  |accessdate =  |quote =  }}</ref>。每节车厢设2个摄像头，以在控制中心看到车厢内的状况；列车可以自动监视障碍物。每列车仍然设有一名司机，但主要任务是照顾乘客<ref name=xhw>{{Cite web |url = http://www.xinhuanet.com/2017-12/30/c_1122188392.htm |title =  北京地铁燕房线今日开通运营 揭秘“无人驾驶”|author =  |date =2017-12-30  |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。
