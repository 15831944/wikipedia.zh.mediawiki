'''北海道道34号襟裳公园线'''（{{lang-ja|北海道道34号襟裳公園線}}）是一条位于[[北海道|北海道]][[日高振興局|日高振興局]][[幌泉郡|幌泉郡]][[襟裳町|襟裳町]]内的[[主要地方道|主要]][[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 路線数据 ==
* 起点：[[北海道|北海道]][[幌泉郡|幌泉郡]][[襟裳町|襟裳町]]庶野（＝[[国道336号|国道336号]]交点）
* 終点：北海道幌泉郡襟裳町歌別（＝国道336号交点）
* 总長：26.1千米

== 经过的自治体  ==
*日高振興局
**[[幌泉郡|幌泉郡]][[襟裳町|襟裳町]]

== 主要连接道路 ==
* 襟裳町
** [[国道336号|国道336号]]

== 历史 ==
* 1954年3月30日 路線勘定（路線编号为97）。<!--1954年北海道告示第503号-->
* 1994年10月1日 路線编号变更。<!--1994年北海道告示第1468号-->
