'''北海道道508号矢臼场札幌线'''（{{lang-ja|北海道道508号矢臼場札幌線}}）是一条连结[[北海道|北海道]][[石狩市|石狩市]]和[[札幌市|札幌市]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：北海道石狩市親船町
* 終点：北海道札幌市[[北区_(札幌市)|北区]]東茨户1条1丁目
* 总距离：12.3千米（内、札幌市管理区間0.7千米）

== 经过的自治体  ==
* [[石狩振兴局|石狩振兴局]]
** 石狩市
** 札幌市（北区）

== 主要连接道路 ==
* [[北海道道225号小樽石狩线|北海道道225号小樽石狩线]]（起点）
* [[国道231号|国道231号]]（石狩市船場町）
* [[国道337号|国道337号]]（石狩市生振）
* [[北海道道273号花畔札幌線|北海道道273号花畔札幌線]]（札幌市北区東茨户2条1丁目 - 札幌市北区東茨户2条2丁目）：重複
* 国道231号（终点）

== 历史 ==
* 1965年（昭和40年）3月26日 - 路線勘定（北海道告示第553号）
