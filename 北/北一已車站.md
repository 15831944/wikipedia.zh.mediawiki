'''北一已站'''（{{jpn|j=北一已駅|hg=きたいちやんえき|rm=Kita-Ichiyan eki|f=y}}）是位於[[北海道|北海道]][[深川市|深川市]]一已町，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[留萌本線|留萌本線]]的[[鐵路車站|鐵路車站]]。電報略號為'''キイ'''。

車站名稱源自[[阿伊努語|阿伊努語]]「ican」，代表[[大麻哈魚|大麻哈魚]]及[[鱒魚|鱒魚]]產卵的淺灘底。原本是[[石狩川|石狩川]]的地名之一，因為本車站位於其北方而加上「北」字。

部位下行普通列車不會停靠本車站。

== 歷史 ==
* 1955年（昭和31年）7月20日 - [[日本國有鐵道|日本國有鐵道]]留萌本線深川站至秩父別站之間開設'''北一己站'''，開始接待旅客及處理行李業務。車站的建築物由當時已廢站的[[深名線|深名線]][[宇津內臨時乘車場|宇津內臨時乘車場]]搬遷至本車站。
* 1984年（昭和59年）2月1日 - 停止處理行李業務，同時由於閉塞系統改進而廢除列車交會設備，車站無人化。
* 1987年（昭和62年）4月1日 - 由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理，同時升格為'''北秩父別站'''。
* 1997年（平成9年）4月1日 - 車站名稱由北一'''己'''站改稱為北一'''已'''站。

== 車站構造 ==
現時為1面1線的地面車站，側式月台位於路軌的西南方（留萌方向的左方）。沒有設置轉轍器的單線路軌車站。過去曾是設有2面2線，能夠進行[[列車交會|列車交會]]的交換車站。當時位於車站旁的月台北方能夠使用[[平交道|平交道]]前往對面月台的南方。車站的建築物旁（西方）為下行1號線，對面月台為上行2號線。但上述交會設備已於1993年3月被移走。

現時為無人車站。車站的建築物位於西方，與月台中央部份連接。該建築物是車站開業時已存在的，為木造平房。表面的木板已經因為時間而殘舊並且變黑。現在已經沒有使用的辦公室，其窗口現時用木板封鎖起來。設有[[廁所|廁所]]。而過去曾存在的2號線月台，現在變成約70米長的花圃，亦曾於比賽中受表揚。

車站前亦種植了巨大的[[東北紅豆杉|東北紅豆杉]]。

== 載客量 ==
* 1981年度的每天載客量為12人。
* 1992年度的每天載客量為6人。
* 2011年至2015年度平均每天載客量少於1人<ref>{{cite web|url=http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |language=ja |title=平成26年度 線区別の収支状況等について |publisher=JR北海道 |date=2016-02-10 |accessdate=2017-06-21 |archive-url=https://web.archive.org/web/20170407133949/http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |archive-date=2017-04-07 |deadurl=yes }}</ref>。

== 車站周邊 ==
開業時位於[[雨龍郡|雨龍郡]][[一已村|一已村]]（1963年合併至[[深川市|深川市]]）。現時位於深川市的郊外，同時亦是日本少見的適合種植[[稻|稻]]的區域，農地與農家稀疏的分佈在附近。
* {{Link-ja|國道233號|国道233号}}（深川國道）
* [[北海道道281號深川多度志線|北海道道281號深川多度志線]]
* 深川市櫻山公園 - 位於車站以北約1.6km。公園保存著數輛已停用的[[鐵路機車#蒸汽機車|蒸汽機車]]如[[日本國鐵D51型蒸汽機車|日本國鐵D51型蒸汽機車]]及{{Link-ja|日本國鐵C58型蒸汽機車|国鉄C58形蒸気機関車}}。
* 深川市立北新小學
