'''北海道道916号汤内内园线'''（{{lang-ja|北海道道916号湯内内園線}}）是一条位于[[北海道|北海道]][[深川市|深川市]]内的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：[[北海道|北海道]][[深川市|深川市]]中湯内（＝[[北海道道98号旭川多度志线|北海道道98号旭川多度志线]]交点）
* 終点：北海道深川市内園（＝[[国道12号|国道12号]]交点）
* 总長：9.7千米

== 经过的自治体  ==
* [[空知综合振兴局|空知综合振兴局]]
** 深川市

== 主要连接道路 ==
* 深川市
** [[北海道道98号旭川多度志线|北海道道98号旭川多度志线]]
** [[北海道道57号旭川深川线|北海道道57号旭川深川线]]＝納内
** [[国道12号|国道12号]]

== 历史 ==
* 1976年（[[昭和|昭和]]51年）8月31日 路線勘定。<!--1976年北海道告示第3075号-->
该路线的前身为1954年（昭和29年）3月30日所勘定的道道64号（当時）多度志納内停車場線和1957年（昭和32年）7月25日所勘定的道道205号（当時）納内停車場内大部線的部分路段。该两条路线被撤销之后，重组为湯内内園線。
