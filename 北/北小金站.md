'''北小金站'''（{{jpn|j=北小金駅|hg=きたこがねえき|rm=Kita-Kogane eki|f=y}}），是一個位於[[千葉縣|千葉縣]][[松戶市|松戶市]]{{link-ja|小金|小金}}，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[常磐線|常磐線]]的[[鐵路車站|鐵路車站]]。

== 停靠路線 ==
旅客車站只限行走常磐線的{{link-ja|緩行線|緩行線}}（[[千代田線|千代田線]]直通）的[[常磐緩行線|各站停車]]停靠。[[車站編號|車站編號]]是'''JL 26'''。

另外，[[常磐快速線|快速線]]自[[武藏野線|武藏野線]]支線（北小金支線）分岔，分岔地點視作[[馬橋站|馬橋站]]構內。行走武藏野線支線大部分是[[貨物列車|貨物列車]]。每當有臨時旅客列車行走時，車費計算將視作途經[[新松戶站|新松戶站]]。另外，此站的[[日本鐵路車站#所屬線|所屬線]]是常磐線。

== 車站結構 ==
[[島式月台|島式月台]]1面2線的[[地面車站|地面車站]]，設有[[跨站式站房|跨站式站房]]。南側2線的緩行線上有月台。北側2線是{{link-ja|急行線|急行線|快速線}}，沒有設置月台。

== 使用人數 ==
2016年度的1日平均'''上車'''人次為'''24,323人'''。
曾經是松戶市內常磐線車站當中最少人上下車的車站，近年超過了[[北松戶站|北松戶站]]、[[馬橋站|馬橋站]]成為第3位。然而，截至2014年仍居[[松戶站|松戶站]]、[[新松戶站|新松戶站]]、馬橋站之後，排行第4位。
