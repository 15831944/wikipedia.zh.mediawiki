'''北海道道966号十胜岳温泉美瑛线'''（{{lang-ja|北海道道966号十勝岳温泉美瑛線}}）是一条连结[[北海道|北海道]][[上川综合振兴局|上川综合振兴局]]管内[[空知郡|空知郡]][[上富良野町|上富良野町]]和[[上川郡_(石狩国)|上川郡]][[美瑛町|美瑛町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：北海道空知郡上富良野町国有無番地（[[十勝岳温泉|十勝岳温泉]]）
* 終点：北海道上川郡美瑛町扇町
* 总距离：32.9千米

== 经过的自治体  ==
* 上川综合振興局
** 空知郡上富良野町
** 上川郡美瑛町

== 冬季禁止通行路段 ==
*10月下旬至来年的5月上旬期间，望岳台至吹上温泉間路段禁止通行，美瑛町一側起至十勝岳温泉、吹上温泉的路段不可通行。（上富良野一側起的路段在冬季依然可以通行）
