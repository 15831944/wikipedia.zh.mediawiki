'''北陸自動車道'''（{{jpn|j=北陸自動車道|hg=ほくりくじどうしゃどう|rm=Hokuriku jidōshadō|f=y}}）是從[[新潟縣|新潟縣]][[新潟市|新潟市]][[江南區_(新潟市)|江南區]]{{link-ja|新潟中央系統交流道|新潟中央ジャンクション}}至[[滋賀縣|滋賀縣]][[米原市|米原市]][[米原系統交流道|米原系統交流道]]的[[日本高速公路|高速公路]]（[[高速自動車國道|高速自動車國道]]），簡稱'''北陸道'''（{{lang|ja|北陸道／ほくりくどう}}，{{Lang|en|HOKURIKU EXPWY}}）。高速公路線編號則被編為 {{Ja Exp Route Sign|E8}}。

* 設施欄背景色<span style="color:#CCC">■</span>區間為未提供使用之設施，未通車區間名稱全部是暫稱。

* 路線名稱未特別註明者為[[市町村道|市町道]]。
* BS（巴士站）：○為使用中、◆為停用中，未註記者即未設置。
* 停车区：停車場，SA：服務區，IC：交流道，JCT：系統交流道。

== 歷史 ==
* 1961年11月15日 : 新潟市-大津市路線基準制定。
* 1966年7月1日 : 新潟市-滋賀縣坂田郡米原町成為國土開發幹線自動車道預定路線。
* 1972年10月18日 : 金澤西IC-小松IC間通車。
* 1973年10月16日 : 小杉IC-砺波IC間通車。
* 1973年10月17日 : 小松IC-丸岡IC間通車。
* 1974年10月29日 : 砺波IC-金澤東IC間通車。
* 1975年9月9日 : 丸岡IC-福井IC間通車。
* 1975年10月4日 : 富山IC-小杉IC間通車。
* 1976年11月2日 : 福井IC-武生IC間通車。
* 1977年12月8日 : 武生IC-敦賀IC間通車（今庄IC-敦賀IC間暫定為2車道）。
* 1978年9月21日 : 新潟黑埼IC（當時）-長岡IC間通車。
* 1978年10月12日 : 金澤東IC-金澤西IC間通車。
* 1980年4月7日 : 敦賀IC-米原JCT間通車。
* 1980年6月13日 : 今庄IC-敦賀IC間4車道拓寬工程完成。
* 1980年6月19日 : 米原IC通車。
* 1980年9月27日 : 長岡JCT-西山IC間通車。
* 1980年10月14日 : 美川IC通車。
* 1980年12月19日 : 滑川IC-富山IC間通車。
* 1981年10月29日 : 西山IC-柏崎IC間通車。
* 1982年11月17日 : 柏崎IC-米山IC間通車。
* 1983年11月9日 : 米山IC-上越IC間通車。
* 1983年11月15日 : 鯖江IC通車。
* 1983年12月13日 : 朝日IC-滑川IC間通車。
* 1984年9月27日 : 金津IC通車。
* 1987年7月21日 : 上越IC-名立谷濱IC間通車，同時上越IC-新潟黑埼IC（當時）的IC編號變更。
* 1988年7月30日 : 朝日IC-名立谷濱IC間通車，第1次'''全線通車'''，同時米原IC-朝日IC的IC編號變更。
* 1989年6月1日 : 新潟西IC通車的同時新潟黑埼IC-新潟西IC由國道116號新潟西繞道編入，新潟收費站變更為新潟西IC收費站。
* 1992年3月28日 : 小矢部砺波JCT通車與東海北陸自動車道連接。
* 1994年7月28日 : 新潟西IC-新潟龜田IC間通車的同時與磐越自動車道連接，新潟西IC新設第二收費站。
* 1996年3月28日 : 小矢部砺波JCT與能越自動車道連接。
* 1997年11月13日 : 新潟龜田IC-新潟機場IC間通車，'''全線通車'''。
* 1998年10月23日 : 朝日IC-越中境停车区間拓寬為4車道。
* 1999年10月30日 : 上越JCT開通與上信越自動車道連接，同時名立谷濱IC-上越IC間拓寬為4車道。
* 2000年4月25日 : 能生IC-名立谷濱IC間拓寬為4車道。
* 2000年9月5日 : 越中境停车区-親不知IC間拓寬為4車道。
* 2000年9月19日 : 糸魚川IC-能生IC間拓寬為4車道。
* 2000年10月3日 : 親不知IC-糸魚川IC間拓寬為4車道，'''全線均為4車道'''。
* 2002年5月26日 : 新潟中央JCT-新潟機場IC間由北陸自動車道改名日本海東北自動車道。
* 2003年3月29日 : 富山西IC通車。
* 2004年3月20日 : 金澤森本IC通車。
* 2004年10月23日 : [[新潟縣中越地震|新潟縣中越地震]]發生，柿崎IC-三條燕IC間暫時停止通行。
* 2004年12月24日 : 黑埼停车区 智能IC[[社會實驗|社會實驗]]開始（至2006年3月31日）。
* 2005年4月11日 : 德光停车区 智能IC社會實驗開始（至2006年3月31日）。
* 2005年6月1日 : 尼御前SA 智能IC社會實驗開始（至2005年8月31日）。
* 2005年8月1日 : 入善停车区 智能IC社會實驗開始（至2006年3月31日）。
* 2005年12月17日: 南条SA 智能IC社會實驗開始（至2009年3月31日）。
* 2006年10月1日 : 黑埼SIC、入善SIC、德光SIC開始使用。
* 2007年3月25日 : [[能登半島地震|能登半島地震]]發生，加賀IC-富山IC間暫時停止通行。
* 2007年4月1日 : 大潟S IC開始使用。
* 2007年5月31日 : 在米原本路線上的收費站廢除。
* 2007年7月16日 : [[新潟縣中越沖地震|新潟縣中越沖地震]]發生，糸魚川IC-新潟西IC間暫時停止通行。
* 2008年3月23日 : 安宅停车区 智能IC社會實驗開始（至2009年3月31日）。
* 2008年3月29日 : 流杉停车区 智能IC社會實驗開始（至2009年3月31日）。
* 2009年4月1日 : 流杉SIC、安宅SIC、南條SIC開始使用。
