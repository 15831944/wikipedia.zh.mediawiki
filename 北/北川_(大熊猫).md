'''北川'''是一只雌性[[大熊猫|大熊猫]]，2010年4月26日上午于[[四川省|四川省]][[北川县|北川县]]被救起，后栖于[[成都大熊猫繁育研究基地|成都大熊猫繁育研究基地]]。是[[汶川大地震|汶川大地震]]后于北川发现的首只野生大熊猫<ref name=CRI>{{Cite web |url=http://news.cri.cn/gb/27824/2010/05/11/110s2847292.htm |title=野生大熊猫“北川”正逐步恢复 |accessdate=2018-09-28 |author=孙亚萍 |date=2010-05-11 |publisher=国际在线}}</ref>。

==发现及救起==
2010年4月26日，北川被[[四川省|四川省]][[北川县|北川县]][[小坝乡_(北川县)|小坝乡]]江河村的一位农民发现<ref name="人民网" />，[[成都大熊猫繁育研究基地|成都大熊猫繁育研究基地]]随即派出一支救护小组前往救援。北川被发现时，体重只有50千克<ref name= 中国熊猫网 />，内脏受损极为严重，肝、膽、胰都受到了嚴重的傷害<ref name=pandahome>{{Cite web |url=http://www.pandahome.org/big5/dongtai/daxiongmao/2015-11-12/3427.html |title=大熊貓“北川”的幸福成都生活 |accessdate=2018-10-13 |author=鍾婷 |date=2015-11-12 |publisher=成都大熊貓繁育研究基金會}}</ref>。次日被带到成都大熊猫基地接受治疗<ref name="人民网">{{Cite web |url=http://it.people.com.cn/h/2011/1018/c227888-4146203761.html |title=野生大熊猫“北川”落户成都 |accessdate=2018-09-28 |author=李庆 |date=2011-10-18 |publisher=人民网}}</ref>。

到成都之后，北川接受了药物治疗与精心饲养，身体状况得到改观<ref name=pandahome />。

==定居成都==

北川于成都接受了一年治疗，但其血液结果及超声波检查仍未达正常水平。故被留在成都，于2011年正式登陆圈养大熊猫谱系，编号「765」<ref name=登陆谱系>{{cite web|url=https://tech.sina.cn/2011-12-01/detail-iaaaaaaa2501024.d.html?from=wap|title=被救野生大熊猫“北川”登录圈养大熊猫谱系|publisher=四川新闻网-成都商报|accessdate=2018-10-13|date=2011-12-01 }}</ref><ref name=pandahome /><ref name=中国熊猫网 />。

==名字由来==

北川为汶川大地震后在北川发现的首只野生大熊猫，应市民要求而取名北川<ref>{{Cite web |url=https://sichuan.scol.com.cn/myxw/content/2010-05/12/content_740600.htm?node=959 |title=每天输液补营养 被救大熊猫"北川"身体还很弱 |accessdate=2018-10-13 |author=何勇 |date=2010-05-12  |publisher=四川在线-华西都市报}}</ref>。

== 后代 ==
2018年5月20日，北川在成都大熊猫繁育研究基地诞下一雌性幼崽，取名为“川仔”，谱系号为1121<ref>{{Cite web|title=熊猫主页|url=http://www.pandapia.com/Panda/view.html?id=ODEy|work=www.pandapia.com|accessdate=2019-02-23}}</ref>。
