'''北海道道283号砂川新十津川线'''（{{lang-ja|北海道道283号砂川新十津川線}}）是一条连接[[北海道|北海道]][[砂川市|砂川市]]和[[桦户郡|桦户郡]][[新十津川町|新十津川町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。全線均为2[[車線|車線]]。

== 指定区間 ==
* 起点：北海道砂川市西1条北9丁目（＝[[国道12号|国道12号]]・[[北海道道627号文珠砂川线|北海道道627号文珠砂川线]]交点）
* 終点：北海道桦户郡新十津川町字花月（＝[[国道275号|国道275号]]交点）
* 总長：4.0千米

== 经过的自治体  ==
* [[空知综合振兴局|空知综合振兴局]]
** 砂川市
** 桦户郡新十津川町

== 主要连接道路 ==
* 砂川市
** 国道12号＝西1条北9丁目（起点）
** 北海道道627号文珠砂川線＝西1条北9丁目（起点）
* 新十津川町
** 国道275号＝花月（终点）

== 主要橋梁 ==
* [[砂川大橋|砂川大橋]]（[[石狩川|石狩川]]）＝砂川市北光-新十津川町字花月

== 历史 ==
* 1957年7月25日 路線勘定为231号。（昭和32年北海道告示第1487号）
* 1994年10月1日 路線编号变更为283号。（平成6年北海道告示第1468号）

== 沿途施設 ==
* 北光公園＝砂川市西3条北10丁目
