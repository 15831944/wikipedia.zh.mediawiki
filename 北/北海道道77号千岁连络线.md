'''北海道道77号千岁连络线'''（{{lang-ja|北海道道77号千歳インター線}}）是一条位于[[北海道|北海道]][[千岁市|千岁市]]内的[[主要地方道|主要]][[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：[[北海道|北海道]][[千岁市|千岁市]]本町2丁目（＝[[国道36号|国道36号]]交点）
* 終点：北海道千岁市泉泽（＝[[道央自動車道|道央自動車道]][[千岁IC_(北海道)|千岁IC]]）
* 总長：1.5千米

== 经过的自治体  ==
*[[石狩振兴局|石狩振兴局]]
**[[千岁市|千岁市]]

== 主要连接道路 ==
'''千岁市'''
* [[国道36号|国道36号]]＝本町2丁目（起点）
* [[道央自動車道|道央自動車道]][[千岁IC_(北海道)|千岁IC]]＝泉泽（终点）

== 历史 ==
* 1969年6月18日 路线勘定为671号。（昭和44年北海道告示第1249号）
* 1994年10月1日 路線编号变更为77号。（平成6年北海道告示第1468号）
