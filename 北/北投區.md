'''北投區'''是[[臺灣|臺灣]][[臺北市|臺北市]]最北邊、以及面積第二大的行政區，著名的[[北投溫泉|北投溫泉]]及關渡風景區也位在此區。<ref>[http://books.google.com.tw/books?id=VhAfQwAACAAJ&dq=%E5%8C%97%E6%8A%95&hl=zh-TW&sa=X&ei=CYPeUL3TPIKNmQXbnIDACw&redir_esc=y 陳姿安, 北投有愛光明在望：最有故事的溫泉廊道, 2008]</ref><ref>[http://books.google.com.tw/books?id=3URbAAAACAAJ&dq=%E5%8C%97%E6%8A%95&hl=zh-TW&sa=X&ei=QoPeUMuwKZCamQWCm4HYCQ&redir_esc=y 戀戀北投溫泉]</ref>臺灣温泉史源頭在北投，於1896年由[[日本|日本]][[大阪|大阪]]商人平田源吾在此設立臺灣第一家温泉旅館「[[天狗庵|天狗庵]]」。<ref>[http://books.google.com.tw/books?id=jlpQILx3hSwC&pg=PA254&dq=%E5%8C%97%E6%8A%95+%E6%BA%AB%E6%B3%89&hl=zh-TW&sa=X&ei=iYTeUPCyLerOmAXNg4DICA&redir_esc=y#v=onepage&q=%E5%8C%97%E6%8A%95%20%E6%BA%AB%E6%B3%89&f=false 協合力-中衛體系提升企業經營綜效 作者：蘇錦夥]</ref>。其東側以磺溪上接[[中山樓_(陽明山)|中山樓]]右側山脊至[[馬槽|馬槽]]橋沿山溝、南側以[[基隆河|基隆河]]與[[士林區|士林區]]相望，北側鄰[[新北市|新北市]][[三芝區|三芝區]]與[[金山區_(台灣)|金山區]]，西側與新北市[[淡水區|淡水區]]為鄰。

此區[[自然資源|自然資源]]豐富，[[陽明山國家公園|陽明山國家公園]]即位於此區及[[士林區|士林區]]。此區醫療資源亦豐，許多醫療中心、醫護院校均位於本區，是故未來本區欲開發之[[士林北投科技園區|士林北投科技園區]]將以生技醫藥產業為主軸。2010年[[臺北市政府|臺北市政府]]舉辦花卉博覽會而制定區花，北投區為[[櫻花|櫻花]]，臺北市政府民政局也特別製作區花版門牌。

== 歷史 ==
[[File:Scenery_of_Hsin_Beitou.jpg|thumb]]
[[File:臺灣民主&獨立運動先驅蘇東啟(左)與蘇洪月嬌(中)於1956臺北北投Taiwanese_Democracy_Activists_Su,_Tong-ch'i_(left)_&_Hung,_Yüeh-chiao_(center).jpg|thumb]]先驅[[蘇東啟|蘇東啟]]（左）與[[蘇洪月嬌|蘇洪月嬌]]（中）等在1956年合影於北投，兩人所抱者有一人係次女[[蘇治芬|蘇治芬]]]]
*昔日為平埔族[[北投社|北投社]]（[[巴賽語|巴賽語]]：{{Lang|en|Ki-pataw}}，意為「[[巫女|巫女]]」）之地，[[康熙|康熙]]中葉後，漸有[[漢人|漢人]][[屯墾|屯墾]]當地。
*1697年[[郁永河|郁永河]]來北投開採硫磺，紀錄於「[[裨海紀遊|裨海紀遊]]」一書中。
*1875年北投隸屬於淡水縣芝蘭二堡。
*1920年臺灣地方改制，實行街-{庄}-制，北投隸[[臺北州|台北州]][[七星郡|七星郡]]，設「[[北投庄|北投-{庄}-]]」。
*1913年[[臺灣總督府|臺灣總督府]]為迎接皇太子裕仁，臺北廳廳長[[井村大吉|井村大吉]]興建[[北投溫泉公共浴場|北投溫泉公共浴場]]，同時整建成[[北投公園|北投公園]]。
*1946年[[新北市政府|臺北縣政府]]成立，隸屬臺北縣七星區北投鎮，1947年七星區併入淡水區，改隸臺北縣淡水區北投鎮
*1949年時改隸[[陽明山管理局|草山管理局]]北投鎮。
*1967年7月1日臺北市升格院轄市時未併入。直到1968年7月1日隨臨近之景美鎮、南港鎮、木柵鄉、內湖鄉、暨同屬陽明山管理局之士林鎮等鄉鎮劃入臺北市，成為北投區。
*1988年7月16日[[臺灣鐵路管理局|臺鐵]][[臺鐵淡水線|淡水線]]因改建捷運停駛。
*1997年[[臺北捷運|臺北捷運]][[台北捷運淡水線|淡水線]]及[[台北捷運新北投支線|新北投支線]]通車，其中[[唭哩岸站|唭哩岸站]]站名譯音自[[凱達格蘭族|凱達格蘭族]]語之{{Lang|en|Ki-lrigan}}。
*2006年[[北投空中纜車|北投空中纜車]]動工，原預計2007年7月通車，但開工後因為爆發弊案及一些開發爭議，也受到當地居民反對。目前確定停建，[[臺北|臺北]]市政府與廠商協調中。
*2016年日式老車站[[新北投站|新北投站]]火車站預計回歸到[[七星公園|七星公園]]，但是確切位置仍在討論中。

[[File:Beitou_Hotspring_Museum_2015.jpg|thumb]]）]]

'''北投區'''位於[[臺灣|臺灣]][[臺北市|臺北市]]最北邊，與[[士林區|士林區]]用[[淡水河|淡水河]]下游來劃分它是全臺灣面積第十二大的區，著名的[[陽明山國家公園|陽明山國家公園]]及[[臺北榮民總醫院|臺北榮民總醫院]]也在該區。北投區的名字是因為早期[[巴賽族|巴賽族]]認為[[地熱谷|地熱谷]]是女巫施法才會冒煙，所以就用[[巴賽語|巴賽語]]裡的女巫，念起來是「PATAW（巴島）」與北投的[[臺灣閩南語|臺語]]發音很像，故命名為「北投」。

== 行政區域 ==
全區共分為42里，並編列有七個次分區，次分區及各里名稱如下：

關渡次分區：八仙、豐年、稻香、桃源、一德、關渡共六里。

陽明山次分區：泉源、湖山、湖田共三里。

舊北投次分區：奇岩、清江、中央、大同 共四里。

新北投次分區：長安、溫泉、林泉、中心 共四里。

石牌次分區：建民、文林、石牌、福興、榮光、榮華、裕民、振華、永欣、永和、洲美共十一里。

唭哩岸次分區：永明、東華、吉利、吉慶、尊賢、立賢、立農 共七里。

大屯次分區：中庸、開明、中和、大屯、智仁、秀山、文化 共七里。

== 關於北投的更多資訊 ==
*[[臺語|臺語]]歌手[[黃克林|黃克林]]以前有首[[臺語|臺語]]老歌叫做「倒退嚕」，歌詞中的第一段「耶~拜請 拜請拜請 東海岸西海岸 北投西帽山 鶯歌出土炭 草山底勒出溫泉」歌詞中提到的北投西帽山=北投紗帽山。
*[[破壞之王|破壞之王]]：是[[周星馳|周星馳]]1994年的香港喜劇片，電影內容（台灣國語配音版）第一段中，何金銀送外賣回來，對榮記冰室的老闆說：哎，老闆啊，下次再到「北投」那麼遠的地方，千萬不要再叫我去了！那兒有一個斜坡那麼斜，我差點騎不上去啊！老闆對何金銀說：天將降大任於斯人也，必先勞其筋骨，餓其體膚，苦其心志，這一次啊送去「萬華」→電影中有提到「北投」。
