'''北海道道99号和寒鹰栖线'''（{{lang-ja|北海道道99号和寒鷹栖線}}）是一条连结[[北海道|北海道]][[和寒町|和寒町]]和[[鹰栖町|鹰栖町]]的[[主要地方道|主要]][[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）<ref>{{Cite web|url=https://thesaurus.weblio.jp/content/%E5%8C%97%E6%B5%B7%E9%81%93%E9%81%9399%E5%8F%B7%E5%92%8C%E5%AF%92%E9%B7%B9%E6%A0%96%E7%B7%9A|title=北海道道99号和寒鷹栖線の同義語 - 類語辞典(シソーラス)|accessdate=2018-02-22|work=thesaurus.weblio.jp|language=ja}}</ref><ref>{{Cite web|url=https://www.weblio.jp/content/%E5%8C%97%E6%B5%B7%E9%81%93%E9%81%9399%E5%8F%B7%E5%92%8C%E5%AF%92%E9%B7%B9%E6%A0%96%E7%B7%9A|title=北海道道99号和寒鷹栖線とは - Weblio辞書|accessdate=2018-02-22|work=www.weblio.jp|language=ja}}</ref><ref>{{Cite news|url=https://www.dodoshiryo-hokkaido.info/tedium/2008/08/10/takasutown.htm|title=鷹栖町中心部にて｜道道つれづれ線|newspaper=道道資料北海道|accessdate=2018-02-22|language=ja}}</ref>。

== 路線数据 ==
* 起点：[[北海道|北海道]][[上川郡_(天盐国)|上川郡]][[和寒町|和寒町]]西町（＝[[北海道道48号和寒幌加内线|北海道道48号和寒幌加内线]]交点）
* 終点：北海道[[上川郡_(石狩国)|上川郡]][[鹰栖町|鹰栖町]]13線5号（＝[[北海道道72号旭川幌加内线|北海道道72号旭川幌加内线]]交点）
* 总長：27.9千米

== 历史 ==
* 1976年8月31日 路線勘定（路線编号为900）。<!--1976年北海道告示第3075号-->
* 1994年10月1日 路線编号变更。<!--1994年北海道告示第1468号-->
1957年（昭和32年）7月25日撤销已勘定的原道道245号（当時）和寒旭川線，重新进行勘定的路线。

== 事件 ==
* 2003年（平成15年）9月举行的[[环北海道自行车赛|环北海道自行车赛]]曾使用本道作为比赛路线的一部分<ref>{{Cite web|url=http://www.tour-de-hokkaido.or.jp/2015/route/Tour2015-summary.pdf#search=%27%E3%83%84%E3%83%BC%E3%83%AB%E3%83%BB%E3%83%89%E3%83%BB%E5%8C%97%E6%B5%B7%E9%81%93+%E5%92%8C%E5%AF%92%E9%B7%B9%E6%A0%96%E7%B7%9A%27|title=ツール・ド・北海道２０１５大会の概要|accessdate=2018-2-22|author=ツール・ド・北海道|date=|publisher=ツール・ド・北海道}}</ref>。
