'''北海道道339号高根平岸停车场线'''（{{lang-ja|北海道道339号高根平岸停車場線}}）是一条连接[[北海道|北海道]][[芦別市|芦別市]]和[[赤平市|赤平市]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。在芦別市境内沿高根川（[[空知川|空知川]]支流）而走。

== 道路概况 ==
* 起点：[[北海道|北海道]][[芦別市|芦別市]]高根町
* 終点：北海道[[赤平市|赤平市]]平岸仲町4丁目（＝[[北海道旅客铁道|JR]][[根室本線|根室本線]] [[平岸站_(北海道赤平市)|平岸站]]）
* 总長：4.2千米

== 共线区间 ==
* 芦別市高根町－赤平市平岸仲町4丁目（[[国道38号|国道38号]]）

== 经过的自治体  ==

** 芦別市
