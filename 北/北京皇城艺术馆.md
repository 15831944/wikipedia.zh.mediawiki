'''北京皇城艺术馆'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[南池子大街|南池子大街]][[菖蒲河公园|菖蒲河]]沿9号，是展示明清[[北京皇城|北京皇城]]历史文化的博物馆。

==简介==
北京皇城艺术馆位于[[菖蒲河公园|菖蒲河公园]]内。作为[[北京市人民政府|北京市人民政府]]的重点项目及明清[[北京皇城|北京皇城]]保护工作的组成部分，北京皇城艺术馆自2002年初开始筹建，同年9月主体建筑完工。2002年10月，[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军委主席|中央军委主席]][[江泽民|江泽民]]视察了筹备中的北京皇城艺术馆，并为该馆题写馆名。2003年6月20日，北京皇城艺术馆举行开馆典礼，正式对外开放<ref name=cbi>{{Cite web |url =  http://www.cbi360.net/hyjd/20170419/69750.html|title =北京皇城艺术馆  |author =  |date =2017-04-19  |publisher =建筑网  |language =  |accessdate =  |quote =  }}</ref>。

北京皇城艺术馆是北京市唯一的系统研究、挖掘、展示[[北京皇城|北京皇城]]衙暑、仓库、坛庙、园林及拱卫服务、物品管理、祭祀拜祖、戏剧演出等文化的专业博物馆。馆舍是砖木结构仿古[[四合院|四合院]]式建筑，分为地上、地下两层，建筑面积4500平方米，展览面积2500平方米，共有三个展厅、一个多功能厅。“皇城风情展”是该馆的基本陈列，以明清北京皇城历史遗迹为主线，通过文物、图片、模型、文字等展示北京皇城历史文化<ref name=cbi/><ref>{{Cite web |url =http://dy.163.com/v2/article/detail/CI0DGEMM051496PC.html  |title = 不看不知道！北京竟然有这么多有意思的艺术馆 |author =  |date =2017-04-14   |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。
