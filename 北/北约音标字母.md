'''北约音标字母'''（{{lang-en|NATO phonetic alphabet}}），正式稱為'''國際無線電通話拼寫字母'''（{{lang-en|International radiotelephony spelling alphabet}}），是最常使用的[[拼寫字母|拼寫字母]]。雖常稱作「'''語音字母'''」（phonetic alphabets）、'''字母解释法'''或是'''通訊代碼'''，但其字母拼法與[[標音|語音系統]]（比如[[國際音標|國際音標]]）並沒有關連。因[[羅馬字母|羅馬字母]]之發音在各國唸法常有差異（如字母Y在德文發音唸為ypsilon），北約音標字母中的[[A|A]]到[[Z|Z]]（共26个[[拉丁字母|拉丁字母]]）和[[0|0]]到[[9|9]]（共10个[[印度阿拉伯数字|印度阿拉伯数字]]）使用[[截頭表音法|截頭表音]]（如''Alfa''代表 A、''Bravo''代表 B，等等）。由此，無論透過[[無線電|無線電]]或[[電話|電話]]收發語音訊息雙方之[[母語|母語]]是否相同，眾多[[字母|字母]]與[[數字|數字]]組成的關鍵配搭也可較為準確地朗讀及知悉，并確保傳送之語音有一定的可理解性，這在保證航行或涉及人員安全的時候尤為重要。

== 历史 ==
北约音标字母表产生于1950年代，其中A和D的读音取自相应的[[希腊字母|希腊字母]]的读音，其他大多数字母采用当时欧美流行的事物的名词或者经典[[姓名|姓名]]来标记。它在战争时候被[[北约|北约]]军队普遍使用，它取代了其他当时存在的發音字母表。北约也用音标字母单词命名苏俄潜艇，这些称呼被命名为[[北约代号|北约代号]]。

== 應用 ==
除了軍事，北約音標字母表現在也被廣泛應用於[[北美|北美]]和[[歐洲|歐洲]]的商業和通訊領域，並已被[[國際民航組織|國際民航組織]]認可而使用於民用航空。在許多電報操作員使用廣泛使用以後，[[國際電報聯盟|國際電報聯盟]]也對它推廣使用。

== 衍生字母 ==
一些语言中存在英语字母表所没有的[[衍生拉丁字母|衍生拉丁字母]]，其中一部分可用两个北约音标字母表示。

===西班牙语===
在[[西班牙语|西班牙语]]中，字母[[ñ|ñ]]的音标字母为Ñoño。

===德语和瑞典语===
在[[德语|德语]]和[[瑞典語|瑞典語]]中，字母[[å|å]]的音标字母为“Alfa-Alfa”（aa），字母[[ä|ä]]的音标字母为“Alfa-Echo”（ae），字母[[ö|ö]]的音标字母为“Oscar-Echo”（oe），字母[[ß|ß]]的音标字母为“Sierra-Sierra”（ss），字母[[ü|ü]]的音标字母为“Uniform-Echo”（ue）。<ref>"Sambandsregelmente för Försvarsmakten, Telefoni – HKV 12800: 70799" dated 26 June 2006.</ref>

===丹麦语及挪威语===
在[[丹麦语|丹麦语]]和[[挪威语|挪威语]]中，字母[[æ|æ]]、[[ø|ø]]和[[å|å]]也有各自的音标字母。这三个字母在丹麦语中的音标字母分别是“Ægir”、“Ødis”和“Åse”，<ref>{{cite web|url=http://www.glemsom.dk/sprog/fonetisk_alfabet.htm |title=Det fonetiske alfabet |publisher=Glemsom.dk |date= |accessdate=2014-08-11}}</ref>挪威语则分两种情况，民用读法分别是“Ægir”、“Ørnulf”和“Ågot”，军用读法则分别是“Ærlig”、“Østen”和“Åse”。<ref>{{cite web|url=http://www.aktivioslo.no/info/det-fonetiske-alfabetet/ |title=Internasjonalt alfabet for radiokommunikasjon |publisher=Aktivioslo.no |date=2013-01-24 |accessdate=2014-08-11}}</ref>

===捷克语===
[[捷克语|捷克语]]中的“[[ů|ů]]”（由''uo''演化）在音标字母中读作“Uniform-Oscar”（uo）。

=== 芬兰语 ===
[[芬兰语|芬兰语]]字母“å”、“ä”和“ö”的无线电读法分别是“Åke”、“Äiti”和“Öljy”。<ref>{{Cite book|title = Sotilaan käsikirja 2013|last = |first = |publisher = Puolustusvoimat|year = 2013|isbn =  978-951-25-2463-1|location = http://www.puolustusvoimat.fi/wcm/c0acb9804173cfc2934ef78fcb266289/SKK2013_web_small.pdf?MOD=AJPERES|pages = 205}}</ref>

===爱沙尼亚语===
[[爱沙尼亚语|爱沙尼亚语]]有四个特殊字母“õ”、“ä”、“ö”和“ü”，音标字母读法分别为''Õnne''、''Ärni''、''Ööbik''、''Ülle''。
