'''北京地铁平谷线'''，编号为'''22号线'''，是[[北京地铁|北京地铁]]规划中的一条线路，连接北京中心城区与[[平谷区|平谷区]]，中间穿过[[河北省|河北省]]，是北京首条与河北省连通的城市轨道交通线路{{NoteTag|[[北京地铁新机场线|北京地铁新机场线]]草桥至大兴机场段预计2019年9月末与[[北京大兴国际机场|北京大兴国际机场]]同期开通，[[大兴机场站|大兴机场站]]位于[[河北省|河北省]][[廊坊市|廊坊市]][[广阳区|广阳区]]，但大兴机场站主要接驳大兴机场前往北京市区的旅客，可以认为不承担北京与廊坊之间的城市轨道通勤任务。|name=新机场线}}<ref name="bjrb20160923">{{cite news|url=http://bjrb.bjd.com.cn/html/2016-09/23/content_68566.htm|title=轨道交通平谷线将途经燕郊|date=2016-09-23|accessdate=2016-10-31|author=耿诺|work=北京日报|deadurl=yes|archiveurl=https://web.archive.org/web/20161101041438/http://bjrb.bjd.com.cn/html/2016-09/23/content_68566.htm|archivedate=2016-11-01}}</ref>。平谷线线路西起[[朝阳区_(北京市)|朝阳区]]的[[东大桥站_(北京市)|东大桥站]]，向东经过[[通州区_(北京市)|通州区]]后，进入[[河北省|河北省]][[三河市|三河市]]，经过[[燕郊|燕郊]]和三河市区后回到北京市界内，进入平谷区<ref name="bjwb20150615">{{cite news|url=http://bjwb.bjd.com.cn/html/2015-06/15/content_287986.htm|title=平谷线确定为首条区域快线|date=2015-06-15|accessdate=2016-10-31|work=北京晚报|deadurl=yes|archiveurl=https://web.archive.org/web/20161101101824/http://bjwb.bjd.com.cn/html/2015-06/15/content_287986.htm|archivedate=2016-11-01}}</ref>。线路全长78.6千米，其中地面段及高架段约30.1千米，地下段长度48.5千米。<ref name=rails />设站20座，采用D型车。<ref name=rails />线路2016年12月24日开工<ref name="轨道交通22号线今日开工">{{cite news|url=http://m.jinghua.cn/xinwen/guonei/20161224/f265489.shtml|title=轨道交通22号线今日开工|work=京华网|date=2016-12-24|accessdate=2017-01-03|deadurl=yes|archiveurl=https://web.archive.org/web/20170103165308/http://m.jinghua.cn/xinwen/guonei/20161224/f265489.shtml|archivedate=2017-01-03}}</ref>，2017年建设冻结停工开始调整规划，预计2019年规划调整后二次开工，2022年开通运营。

== 路线 ==

== 历史及未来规划 ==
* 在2016年12月24日开工的平谷线线路规划中，平谷线西起[[朝阳区_(北京市)|朝阳区]][[北京四环路|东北四环]]的[[东风北桥站|东风北桥站]]，向东经过[[通州区_(北京市)|通州区]]后，进入[[河北省|河北省]][[三河市|三河市]]，经过[[燕郊|燕郊]]和三河市区后回到北京市界内，进入平谷区<ref name="bjwb20150615">{{cite news|url=http://bjwb.bjd.com.cn/html/2015-06/15/content_287986.htm|title=平谷线确定为首条区域快线|date=2015-06-15|accessdate=2016-10-31|work=北京晚报|deadurl=yes|archiveurl=https://web.archive.org/web/20161101101824/http://bjwb.bjd.com.cn/html/2015-06/15/content_287986.htm|archivedate=2016-11-01}}</ref>。线路全长74千米，设站12座<ref name="bjrb20160923" />，分别为[[东风北桥站|东风北桥站]]、北岗子站、曹各庄北站、徐辛庄站、燕顺路站、燕郊北站、齐心庄站、马坊站、马昌营站、平谷西站、平谷站和泃河湾站，原计划采用国铁制式、[[和谐号CRH6型电力动车组#CRH6F（160km/h）|CRH6F动车组]]<ref>{{cite news|url=http://sdjsb.bjd.com.cn/html/2016-06/22/content_43746.htm|title= 快轨平谷线采用CRH6动车组|date=2016-06-22|accessdate=2016-10-31|author=李博|work=首都建设报}}</ref>，设计速度目标值160km/h，全程运行时间预计在45分钟以内<ref>{{cite news|url=http://jjrb.bjd.com.cn/html/2015-12/31/content_341685.htm|title=地铁平谷线设计时速160公里|date=2015-12-31|accessdate=2016-10-31|author=杨旗|work=京郊日报}}</ref>。线路2016年12月24日开工，2020年投入使用<ref name="轨道交通22号线今日开工"/>。
* 2017年9月，[[中国铁道科学研究院|中国铁道科学研究院]]官网发布《北京轨道交通22号线（徐辛庄至平谷段）工程北京段环境影响报告书》<ref>{{cite web |title=平谷线北京段12月全面开工 预计2020年年底通车|url=http://www.xinhuanet.com//local/2017-09/08/c_1121618791.htm |website=新华网 |accessdate=2019-04-05}}</ref>，工程东起平谷区，经过河北省三河市西至通州区徐辛庄地区，整体呈东西走向。工程线路总长52.9公里，其中北京段30.6公里，除下穿六环路段约2.9公里地下段外，其余均为地面及高架线。北京段设置车站6座，分别为徐辛庄站、马坊站、马昌营站、平谷西站、平谷站和泃河湾站，徐辛庄站为地面站，其余5座车站全部为高架车站。与之前规划相比，徐辛庄站改为地面站。
* 2018年，项目处于冻结停工状态。
* 2018年12月，项目调整为接入[[北京城市副中心|北京城市副中心]]的轨道交通。
* 2019年4月4日，[[中国铁道科学研究院|中国铁道科学研究院]]官网发布《北京市轨道交通第二期建设规划调整环境影响报告书（征求意见稿）》<ref name=railswebsite>{{cite web |title=北京市轨道交通第二期建设规划调整 环境影响评价第二次公示 |url=http://home.rails.com.cn/news.php?id=54325 |website=中国铁道科学研究院集团有限公司 |accessdate=2019-04-05}}</ref><ref name=rails>{{cite web |title=北京市轨道交通第二期建设规划调整环境影响报告书（征求意见稿） |url=http://home.rails.com.cn/download/2019/20190404_2_1_%E9%99%84%E4%BB%B61%EF%BC%9A%E5%8C%97%E4%BA%AC%E5%B8%82%E8%BD%A8%E9%81%93%E4%BA%A4%E9%80%9A%E7%AC%AC%E4%BA%8C%E6%9C%9F%E5%BB%BA%E8%AE%BE%E8%A7%84%E5%88%92%E8%B0%83%E6%95%B4%E7%8E%AF%E5%A2%83%E5%BD%B1%E5%93%8D%E6%8A%A5%E5%91%8A%E4%B9%A6%EF%BC%88%E5%BE%81%E6%B1%82%E6%84%8F%E8%A7%81%E7%A8%BF%EF%BC%89.pdf |website=中国铁道科学研究院集团有限公司 |accessdate=2019-04-05}}</ref>，为支持[[北京城市副中心|北京城市副中心]]建设及交通出行问题，解决中心城区与城市副中心之间的通勤需求，实现主副中心之间的快速联系，增强副中心对于[[北三县|北三县]]地区以及东北部外围新城的辐射带动作用，有序疏解非首都功能，解决[[燕郊|燕郊]]区域大客流出行需求，有效缓解东西部交通走廊客流压力，平谷线调整接入北京中心城区与城市副中心，计划2019年施工，2022年开通。调整后线路全长78.6千米，其中地面段及高架段约30.1千米，地下段长度48.5千米。<ref name=rails />设站20座，采用D型车。<ref name=rails />

=== 换乘站 ===
拟建平谷线有换乘车站6座，在[[东大桥站_(北京市)|东大桥站]]与[[北京地铁6号线|北京地铁6号线]]、[[北京地铁17号线|17号线]]和[[北京地铁28号线|28号线]]换乘，在[[金台夕照站|金台夕照站]]与[[北京地铁10号线|10号线]]和[[北京地铁28号线|28号线]]换乘，在[[红庙站|红庙站]]与[[北京地铁14号线|14号线]]换乘，在[[管庄站|管庄站]]与[[北京地铁八通线|八通线]]换乘，在[[通州北关站|通州北关站]]与[[北京地铁6号线|6号线]]换乘，在[[北京城市副中心站|副中心站]]与[[中国铁路总公司|国铁]]换乘并通过[[北运河东站|北运河东站]]与[[北京地铁6号线|6号线]]换乘。
