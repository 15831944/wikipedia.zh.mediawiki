'''北海北站'''位于[[北京市|北京市]][[西城区|西城区]]，是[[北京地铁|北京地铁]][[北京地铁6号线|6号线]]一期的一座车站。车站因位于[[北海公园|北海公园]]北侧（临近北海公园北门）而得名。北海北车站随着6号线一期工程于2012年12月30日开通投入运营。

== 位置 ==
北海北站地处[[北海公园|北海公园]]北门西侧，北邻[[贤良祠|贤良祠]]，位于[[地安门西大街|地安门西大街]]（[[平安大街|平安大街]]，东西向）地下，呈东西走向。<ref>{{cite web|url=http://www.bjghw.gov.cn/web/static/articles/catalog_14300/article_ff808081262b02de01262c2fa1a90005/ff808081262b02de01262c2fa1a90005.html|title=轨道交通6号线一期工程规划方案公告|date=2010-01-14|accessdate=2012-01-28|archive-url=https://web.archive.org/web/20101227055616/http://www.bjghw.gov.cn/web/static/articles/catalog_14300/article_ff808081262b02de01262c2fa1a90005/ff808081262b02de01262c2fa1a90005.html|archive-date=2010-12-27|dead-url=yes}}</ref>

== 结构 ==
北海北站为地下二层车站，[[岛式站台|岛式站台]]设计。受地理条件限制以及出于地面历史建筑文物保护原因，车站主体为单柱双联拱双层框架结构，主要采用暗挖施工。地下一层为站厅层，地下二层为站台层。车站总长258.5米，有效站台长度为158米，站台宽10.8米。<ref>北京地铁6号线九标北海北站获得北京市基础结构长城杯金质奖，2013-7-8。</ref>

站厅设有包括自助取款机、自助查询机等自助服务设施供乘客使用。

== 出入口 ==
北海北站站厅两端北侧的通道是通往地安门西大街北侧的西北 (A)、东北 (B) 出入口，站厅中部南侧的通道是通往地安门西大街南侧的西南 (D) 出入口。西南 (D) 出入口去往北海公园北门最近。

== 装饰 ==
北海北车站将传统园林中[[回廊|回廊]]的重檐飞椽元素运用在车站顶棚装饰方案中，采用仿[[中国古代建筑|中国古代建筑]]的绿色[[琉璃|琉璃]][[重檐|重檐]][[飞椽|飞椽]]装饰<ref>{{cite news|url=http://roll.sohu.com/20120511/n342903809.shtml|title=地铁6号线一期车站装修全面展开|date=2012-05-11|accessdate=2012-05-11|author=涂露芳}}</ref>，体现北海[[皇家园林|皇家园林]]印象。车站墙面有名为“北海春荫”、“廊下琼华”、“月中静心”运用石材拼贴工艺的壁画装饰，展现北海公园园林景观。<ref>[http://www.bjghw.gov.cn/web/dtggyshz/dtggys_list6.html 2012年度地铁公共艺术品建成展示--6号线]，北京市规划委员会网站。</ref>
