'''北海道道397号丰富停车场线'''（{{lang-ja|北海道道397号豊富停車場線}}）是一条位于[[北海道|北海道]][[天盐郡|天盐郡]][[丰富町|丰富町]]内的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 路線数据 ==
* 起点：[[北海道|北海道]][[天盐郡|天盐郡]][[丰富町|丰富町]]字丰富站前通（=[[宗谷本線|宗谷本線]][[丰富站|丰富站]]）
* 終点：北海道天盐郡丰富町字丰富大通9丁目（=[[国道40号|国道40号]]交点）

== 经过的自治体  ==
* [[宗谷综合振兴局|宗谷综合振兴局]]
** 天盐郡丰富町

== 历史 ==
* 1961年3月31日 路線勘定。（1961年北海道告示第616号）

== 沿線 ==
* 天盐郡丰富町
** [[宗谷本線|宗谷本線]] [[丰富站|丰富站]]=字丰富西3条7丁目
