'''北港武德宮'''，位於[[中華民國|中華民國]][[雲林縣|雲林縣]][[北港鎮|北港鎮]]內、主祀中路武財神[[趙公明|趙公明]]的[[廟宇|廟宇]]。該廟是[[2010年代臺灣|2010年代臺灣]]地區規模最大的財神廟之一。

== 歷史 ==
北港武德宮的[[創辦人|創辦人]]陳茂霖是一位出身雲林縣[[水林鄉|水林鄉]]的[[中醫師|中醫師]]；在遷居北港後，他在[[1955年臺灣|1955年]]購得該廟所在[[地皮|土地]]及[[店舖|店舖]]經營中醫事業。

[[1961年臺灣|1961年]]前後，陳茂霖的[[妻子|妻子]]之身體[[健康|健康]]出現問題；他在遍尋[[醫療|醫療]]方法而不得其解後開始尋求[[信仰|信仰]]及[[宗教|宗教]]上的協助。[[1963年臺灣|1963年]]，陳茂霖在[[新港東興廟|新港東興宮]][[池府千歲|池府千歲]]出巡行經北港時，獲得乩身[[扶乩|扶乩]]指示其[[住宅|住宅]]內有「內神」，並指示其[[虔誠|虔誠]][[敬奉|敬奉]]該[[神|神]]則其妻之問題可獲解決；隨後，陳茂霖依指示在自宅地上設置[[香爐|香爐]]並早晚虔誠敬奉，他的妻子也在不久後恢復健康。

[[1970年臺灣|1970年]]前後，陳茂霖經扶乩得知該「內神」的[[事實|真實]]身分為玄壇元帥[[趙公明|趙公明]]，且其[[神尊|神尊]]乃於[[道光|道光]]年間由一[[陳姓|陳姓]][[信徒|信徒]]自[[中國大陸|中國大陸]][[山東|山東]][[琅琊|琅琊]]迎請攜來，並曾在當地搭有簡易草庵供人[[祭拜|祭拜]]，又因時空變遷而被掩埋於陳茂霖[[所有权|所有]]的[[建築物|建築物]]之下。隨後，陳茂霖遵照扶乩所降指示為其[[雕塑|雕塑]]金身，並於自宅設[[神壇|壇]]供人膜拜，成為日後武德宮的前身。

[[1978年臺灣|1978年]]，陳茂霖考慮到原建物[[腹地|腹地]]狹窄，而[[過年|過年]]或[[神誕|聖誕]]祭典信徒眾多，於是[[捐贈|捐贈]]土地並邀集[[親人|親人]]共同出資興建新廟，並於[[1980年臺灣|1980年]]於現址落成入火安座。

[[1981年臺灣|1981年]]，主祀神武財公的[[扶鸞|鸞文]]開始預示其即將率眾返回其故鄉[[山東|山東]]探視。

[[1988年臺灣|1988年]]，陳茂霖依循鸞文指示率隊前往山東[[齊河縣|齊河縣]][[趙官鎮|趙官鎮]]拜訪武財公[[祖廟|祖廟]]所在地，並在返回臺灣後獲武財公岀筆扶鸞指示以北港武德宮作[[兩岸|兩岸]]及[[華人地區|華人地區]]天官武[[財神開基祖廟|財神開基祖廟]]。<ref name=":0" />

[[2010年代臺灣|2010年代]]，北港武德宮成為[[臺灣地區|臺灣地區]]規模最大的[[財神|財神]]廟之一<ref name=":1">{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20170721/1165899/|title=【壹週刊】武德宮主委摃政府：盲目打壓道教信仰|author=|date=2017-07-21|work=|newspaper=蘋果日報|accessdate=2017-10-20|language=zh-TW}}</ref><ref name=":2">{{Cite news|url=http://www.nextmag.com.tw/realtimenews/news/297784|title=名校碩士很會賺　偏偏老闆不是人|newspaper=Next Magazine TW|accessdate=2017-10-20|language=zh-TW}}</ref>。

[[2013年臺灣|2013年]]甲午年臘月啟建「建成祈安三朝清[[醮|醮]]」。

[[2015年臺灣|2015年]]丙申年臘月啟建「叩謝天恩祈安三朝[[圓醮|圓醮]]」。

== 建築 ==

=== 空間配置 ===
北港武德宮設有正殿、三官殿、聖父母殿、三仙姑殿、福德殿、[[太歲|太歲]]殿、文昌殿等，其後還有一座十數層樓高的廣天大道院。<ref name=":0" />

== 慶典 ==
* [[農曆新年|農曆新年]]：農曆[[正月初一|正月初一]][[卯時|卯時]]廟方會鳴鐘響鼓並打開廟門，信徒們手持[[線香|線香]]搶入廟門並在已封上[[金紙|金紙]]的香爐插下第一注香(又稱「搶頭香」)，以為新的一年奪得好[[彩頭|彩頭]]。農曆過年期間，該廟會發放限量天官賜福小[[紅包|紅包]]；同時，信徒們也會向神明恭請財寶[[米|米]]、[[平安符|平安符]]、求財金及消災解厄補財庫銀等，持之沿[[順時鐘方向|順時鐘方向]]環繞香爐三圈並同時默念親友或自己[[姓名|姓名]]並表達祈願。<ref>{{Cite web|url=http://www.wude.org.tw/celebrate.aspx|title=農曆新年-開春拜財神，好運一整年|accessdate=2017-10-20|author=|date=|work=www.wude.org.tw|publisher=北港武德宮|deadurl=yes|archiveurl=https://web.archive.org/web/20171108122505/http://www.wude.org.tw/celebrate.aspx|archivedate=2017-11-08}}</ref>
* 武財神[[神誕|聖誕]]千秋：[[分靈|分靈]]宮廟、神壇及家戶一同前往北港武德宮為老財神爺[[祝壽|祝壽]]，並按往例由[[歌仔戲|歌仔戲]]及[[布袋戲|布袋戲]]團作大戲三日([[三月十三|三月十三]]、十四、十五)，復於[[三月十四|三月十四]]前備好壽糕、[[壽桃包|壽桃]]及壽麵，於當日晚上[[子時|子時]]開始祭典。同時，廟方會備有福酒(以[[福圓|福圓]]釀造的[[酒|酒]])供信徒飲用。<ref>{{Cite web|url=http://www.wude.org.tw/celebrate01.aspx|title=武財神聖誕千秋-農曆三月十五日拜壽參祝聖誕|accessdate=2017-10-20|author=|date=|work=www.wude.org.tw|publisher=北港武德宮|deadurl=yes|archiveurl=https://web.archive.org/web/20160624133129/http://www.wude.org.tw/celebrate01.aspx|archivedate=2016-06-24}}</ref>
* 平安月慶讚[[中元節與盂蘭盆節|中元]]：該廟在農曆[[七月十五|七月十五]]有提供信眾代為普渡的服務。<ref>{{Cite web|url=http://www.wude.org.tw/celebrate02.aspx|title=平安月慶讚中元-慈悲佈施，普渡眾生以積福|accessdate=2017-10-20|author=|date=|work=www.wude.org.tw|publisher=北港武德宮|deadurl=yes|archiveurl=https://web.archive.org/web/20151221142944/http://www.wude.org.tw/celebrate02.aspx|archivedate=2015-12-21}}</ref>
* 武財神[[開光點眼|開光]]紀念：農曆[[十二月初九|十二月初九]]為武財神[[神尊|金身]]自中國大陸渡海前往臺灣的[[紀念日|紀念日]]。該廟會於[[十二月初七|十二月初七]]、初八、初九舉辦為期三天的[[禮斗|禮斗]]法會，由法僧誦讀經文為信徒消災解厄並祈福求財，信徒亦可參與法會隨同法僧誦經參拜。會後，眾人會焚燒「消災解厄補財庫銀」以化解災厄，以求財運及好運接續而來。<ref>{{Cite web|url=http://www.wude.org.tw/celebrate03.aspx|title=武財神開光紀念-莊嚴禮斗慶週年|accessdate=2017-10-20|author=|date=|work=www.wude.org.tw|publisher=北港武德宮|deadurl=yes|archiveurl=https://web.archive.org/web/20151221143544/http://www.wude.org.tw/celebrate03.aspx|archivedate=2015-12-21}}</ref>

== 特色 ==
* 北港武德宮保有傳統問乩儀式；在儀式中，神明透過[[鸞生|鸞生]]寫下[[嵌字文|藏頭詩]]。每[[農曆|農曆]]月逢初一、初四、初七下午，信徒可在外壇問事，且不收費。<ref name=":2" />內壇問事以廟務為主<ref name=":2" />，廟中所有重要決定(如重大決議與進程、殿堂[[對聯|對聯]]的詩文內容、廟宇建築規劃)皆由神明[[扶鸞|扶鸞]]指示而成，而開基祖廟四字正是由開基老武財公扶鸞指示而成。<ref name=":0">{{Cite web|url=http://www.wude.org.tw/history.aspx|title=傳奇的開始-五路財神發祥源流|accessdate=2017-10-20|author=|date=|work=www.wude.org.tw|publisher=北港武德宮|deadurl=yes|archiveurl=https://web.archive.org/web/20171104131318/http://www.wude.org.tw/history.aspx|archivedate=2017-11-04}}</ref>
* 北港武德宮的[[金紙|金紙]]純以[[竹材|竹材]]製作，觸感比一般金紙粗糙，但不會產生黑煙。<ref name=":1" />
