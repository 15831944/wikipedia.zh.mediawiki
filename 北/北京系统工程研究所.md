'''北京系统工程研究所'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]]，是系统工程研究机构。

==简介==
北京系统工程研究所于1986年成立<ref name=nfw>{{Cite web|url =http://gz.southcn.com/content/2016-01/11/content_140527336.htm?from=singlemessage&isappinstalled=0|title =中央军委装备发展部成立 总装备部成为历史|author =|date =2016-01-11|publisher =南方网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170901074144/http://gz.southcn.com/content/2016-01/11/content_140527336.htm?from=singlemessage&isappinstalled=0|archivedate =2017年9月1日|df =}}</ref>。是以发展国防战略研究、计算机软件技术研究、信息安全技术研究、国防科研项目管理为主的综合性科研机构<ref name=zhaosheng>{{Cite web |url = http://yz.chsi.com.cn/sch/schoolInfo--schId-367835,categoryId-458527,mindex-1.dhtml |title =北京系统工程研究所  |author =  |date =  |publisher =  中国研究生招生信息网|language =  |accessdate =2017-06-16  |quote =  }}</ref>。[[钱学森|钱学森]]、[[朱光亚|朱光亚]]都曾关心该所建设<ref name=nfw/>。截至2014年，该所有[[中国科学院院士|中国科学院院士]]2人，[[中国工程院院士|中国工程院院士]]2人，研究员23人，副研究员52人<ref name=zhaosheng/>。该所原先隶属[[国防科工委|国防科工委]]<ref>黄建新，当今美国军事仿真和模拟的应用情况和发展趋势，军事运筹与系统工程1996(4):38-43</ref>。后转隶[[中国人民解放军总装备部|中国人民解放军总装备部]]<ref name=nfw/>。2016年[[深化国防和军队改革|深化国防和军队改革]]中，撤销中国人民解放军总装备部。
