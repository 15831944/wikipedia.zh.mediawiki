'''北海道道981号上长和萩原线'''（{{lang-ja|北海道道981号上長和萩原線}}）是一条位于[[北海道|北海道]][[伊達市_(北海道)|伊達市]]内的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 概要 ==
* 起点：伊達市上長和町（[[国道453号|国道453号]]交点）
* 終点：伊達市萩原町（[[国道37号|国道37号]]交点）
* 总長：11.2千米
* 路線勘定：1980年（昭和55年）3月31日<ref>1980年3月31日北海道告示第816号</ref>
* 通過的市：北海道伊達市
== 共线区间 ==
* [[北海道道519号泷之町伊达线|北海道道519号泷之町伊达线]]：伊達市上館山町
== 相接道路 ==
* 国道453号（有珠国道）：起点
* [[北海道道703号洞爷湖公园线|北海道道703号洞爷湖公园线]]：伊達市上長和町
* 北海道道519号泷之町伊達線：伊達市上館山町
* 国道37号（胆振国道）：終点
== 交通規制 ==
由于[[冬|冬季]]有[[雪|雪]]，以下路段禁止通行。
* 伊達市乾町263 - 伊達市幌美内町（約1.1千米）
== 主要道路構造物 ==
[[File:Kita-no-Shonan_Bridge_2009.jpg|thumb]]
;北之湘南橋
:在伊達市上長和町，橋長291[[米_(单位)|米]]，宽9.5米（含[[人行道|人行道]]2.0米）的連續鈑桁・單純合成鈑桁[[橋梁|橋梁]]。横跨在[[長流川|長流川]]上。1990年（[[平成|平成]]2年）架橋。橋名中的「北の湘南」是伊達市的[[愛称|愛称]]。橋的西面有[[停车场|停车场]]，可以远眺[[昭和新山|昭和新山]]。
;伊達隧道
:在伊達市上館山町，是一条总長637米、宽9.75米的[[隧道|隧道]]。1992年（平成4年）4月貫通、1995年（平成7年）12月竣工。
