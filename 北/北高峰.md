'''北高峰'''是[[杭州|杭州]][[西湖|西湖]]西北方的一座[[山峰|山峰]]，高314米，是杭州较高的山峰<ref name="登山">{{cite web |url=http://www.hangzhou.com.cn/20040101/ca535302.htm |language= |title=登北高峰 |publisher=杭州网 |date=2004年9月20日 |accessdate= |archive-url=https://web.archive.org/web/20120107030629/http://www.hangzhou.com.cn/20040101/ca535302.htm |archive-date=2012-01-07 |dead-url=yes }}</ref>。著名的[[灵隐寺|灵隐寺]]便位于其山脚下，从寺西侧上山，石磴多达数千级，沿途山溪清流回转、林木重叠，古人有诗赞：‘一路松声长带雨， 半空岚气总成云’。山东侧现建有载人[[索道|索道]]，上下山往返仅需六、七分钟。

山巅有著名的[[灵顺寺|灵顺寺]]，它始建于[[东晋|东晋]][[咸和|咸和]]年间，距今有1600多年的历史，该寺从[[宋代|宋代]]开始就供奉“五显[[财神|财神]]”，[[香火|香火]]旺了上千年，有“'''天下第一[[财神庙|财神庙]]'''”的美誉。灵顺寺于2005年进行了大规模整修后对外开放，香火一直十分旺盛<ref>{{cite web |url=http://www.fjsy.net/syShow.aspx?syID=4575&kind=5&nID=27268 |language= |title=北高峰财神庙——关于印旭法师的佛学理念 |publisher=杭州北高峰灵顺寺 |date= |accessdate=2012年12月 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

北高峰与[[南高峰|南高峰]]形成的'''[[双峰插云|双峰插云]]'''是[[西湖十景|西湖十景]]之一<ref name="介绍">{{cite web |url=http://www.aoohz.com/mingsheng/old10-sfcy.htm |language= |title=双峰插云 |publisher=杭州旅游 |date= |accessdate=2012年12月 |deadurl=yes |archiveurl=https://web.archive.org/web/20120611071038/http://www.aoohz.com/mingsheng/old10-sfcy.htm |archivedate=2012-06-11 }}</ref>。

历史上，[[苏东坡|苏东坡]]、[[乾隆帝|乾隆帝]]、[[毛泽东|毛泽东]]等名人曾多次攀登此山。<ref name= "登山"/>
