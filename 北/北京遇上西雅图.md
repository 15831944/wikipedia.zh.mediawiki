{{engname|《北京遇上西雅图》|''Finding Mr. Right''  /  ''Anchoring in Seattle''}}是2013年由[[薛晓路|薛晓路]]编剧并执导的一部爱情喜剧电影。2013年3月21日在中国大陆上映。<ref>{{Cite web |url=http://www.51oscar.com/index.php/topic/topicView/id/232 |title=浪漫喜剧《北京遇上西雅图》21号上映 |access-date=2013-03-23 |archive-url=https://web.archive.org/web/20130321173718/http://www.51oscar.com/index.php/topic/topicView/id/232 |archive-date=2013-03-21 |dead-url=no }}</ref>2013年3月28日在香港上映。<ref>{{Cite web |url=http://hk.movies.yahoo.net/movie/details/19856 |title=2013年3月28日在香港上映 |access-date=2013-03-28 |archive-url=https://web.archive.org/web/20130213130503/http://hk.movies.yahoo.net/movie/details/19856 |archive-date=2013-02-13 |dead-url=no }}</ref>而導演薛曉路以本片拿下第10屆中國[[華鼎獎|華鼎獎]]最佳新銳導演的殊榮。

==剧情==
「拜金女」文佳佳（[[湯唯|湯唯]]飾）曾為美食雜誌編輯，對愛情充滿了像電影《[[西雅圖夜未眠|西雅圖夜未眠]]》一樣的浪漫幻想。而在現實中，她不遠萬里隻身來到西雅圖的月子中心待產生子。

在月子中心，文佳佳炫富的作風引發了房東黃太（[[金燕玲|金燕玲]]飾）和其他孕婦周逸（[[海清|海清]]飾）、陳悅（[[買紅妹|買紅妹]]飾）的反感，倍感孤獨的她只能向司機郝志Frank（[[吳秀波|吳秀波]]飾）傾訴心聲。看上去木訥老實的「落魄大叔」Frank並不是一個平庸的男子，他在中國曾是一流的心血管疾病方面的名醫。在相處中，Frank的體貼包容漸漸融化了文佳佳的刁蠻任性。當文佳佳的富豪男友突然失蹤後，一夜之間變成窮人的文佳佳得到了Frank無微不至的照顧，跟Frank和他的女兒Julie（[[宋美曼|宋美曼]]、[[宋美慧|宋美慧]]飾）一起生活的這段日子，讓文佳佳找到了家的溫暖，亦與月子中心其他孕婦成為好友。

經歷許多變故的文佳佳終於生下了孩子，就要結束她在西雅圖的顛沛之旅時，她與Frank之間已經產生了微妙的變化，這時的她終於意識到自己真正追求的是什麼。離開西雅圖並不代表結束，而是下一次相遇的開始……

== 票房 ==

=== 中国大陆 ===
首周四天票房达到7541万人民币，挤下上周票房冠军《[[虎胆龙威5|虎胆龙威5]]》的7414万人民币。截至3月31日，影片累计票房已达到2.3亿成为历年来三月档期票房最高的华语片<ref>{{cite web|url=http://ent.qq.com/a/20130331/000159.htm | language= | title="《北京遇上西雅图》票房破两亿 创三月最高纪录 |publisher=网易 |date=2013-04-08 |accessdate= }}</ref>。

根據中国电影报官方微博透露《北京遇上西雅图》已蟬連四週票房冠軍<ref>[http://e.weibo.com/2304129841/zpdKvtWBX 2013年第12周（3.18-3.24）]3月26日 </ref><ref>[http://e.weibo.com/2304129841/zqhSCgHtZ 2013年第13周（3.24-3.31）]4月2日</ref><ref>[http://e.weibo.com/2304129841/zrl8EFA9K?ref=http%3A%2F%2Fe.weibo.com%2Fu%2F2304129841%3Ftype%3D0 2013年第14周（4.1-4.7）]4月9日</ref><ref>[http://e.weibo.com/2304129841/zspnwt7cV?ref=http%3A%2F%2Fweibo.com%2Fu%2F1670003214 2013年第15周（4.8-4.14）] {{Wayback|url=http://e.weibo.com/2304129841/zspnwt7cV?ref=http%3A%2F%2Fweibo.com%2Fu%2F1670003214 |date=20140110090941 }}4月16日</ref>。5月5日，《北京遇上西雅图》公映46天正式下档，根據新聞出版社廣電總局電影局消息，《北京遇上西雅图》最终票房為5.1967亿元，是2013年度中國大陸電影票房第8名<ref>{{cite web |url=http://news.xinhuanet.com/yzyd/ent/20140109/c_118898042.htm |language= |title=2013年电影票房十强影片出炉：《西游》登榜首 |publisher=新华网 |date=2014-01-09 |accessdate= |archive-url=https://web.archive.org/web/20140112210233/http://news.xinhuanet.com/yzyd/ent/20140109/c_118898042.htm |archive-date=2014-01-12 |dead-url=no }}</ref>，成功打破了《[[非诚勿扰2|非诚勿扰2]]》所创下的4.72亿元的国产爱情片票房纪录，成为内地影史上最卖座的爱情电影<ref>{{cite web |url=http://www.chinanews.com/yl/2013/04-17/4739743.shtml |language= |title="《北京遇上西雅图》卖座超"非诚" 票房破4.74亿 |publisher=中国新闻网 |date=2013-04-17 |accessdate= |archive-url=https://web.archive.org/web/20130418054842/http://www.chinanews.com/yl/2013/04-17/4739743.shtml |archive-date=2013-04-18 |dead-url=no }}</ref>。
