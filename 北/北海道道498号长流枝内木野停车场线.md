'''北海道道498号長流枝内木野停车场线'''（{{lang-ja|北海道道498号長流枝内木野停車場線}}）是一条位于[[北海道|北海道]][[十胜综合振兴局|十胜综合振兴局]]管内[[河東郡_(日本)|河東郡]][[音更町|音更町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：北海道河東郡音更町長流枝幹線
* 終点：北海道河東郡音更町木野大通西（旧[[日本国有铁道|国铁]][[木野站_(北海道)|木野站]]）
* 总距离：21.2千米

== 经过的自治体  ==
* 十勝综合振兴局
** 河東郡音更町

== 主要连接道路 ==
* 北海道道31号音更池田线（起点）
* [[北海道道316号上士幌音更线|北海道道316号上士幌音更线]]（下士幌北 - 下士幌）：重複
* 北海道道73号带广浦幌线（下士幌 - 宝来南一条、木野大通東 - 木野大通西）：重複
* 国道241号（木野大通東 - 木野大通西）：重複 

== 隧道 ==
* 宝来隧道（105米）

== 历史 ==
* 1965年（[[昭和|昭和]]40年）3月26日 - 路線勘定（北海道告示第553号）

== 沿線 ==
* 筒井温泉 - 包括[[十胜川温泉|十胜川温泉]]。
