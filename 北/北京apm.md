'''北京apm'''，原名'''新东安市场'''、'''新东安广场'''，是位于[[北京市|北京市]][[王府井大街|王府井大街]]北段的一座商业[[大厦|大厦]]。

==简介==
[[File:Beijing_apm,_2008-10-19.jpg|thumb]]

1993年开始由北京[[东安集团|东安集团]]和[[香港|香港]][[新鸿基地产|新鸿基地产]]有限公司合资兴建，在[[东安市场|东安市场]]原址建设，1998年竣工开业，名为新东安市场，<ref>{{cite web |url = http://www.people.com.cn/GB/tupian/31655/2895711.html |title = 新东安市场 |publisher=[[人民网|人民网]] |date=2004-09-30 |accessdate=2011-05-25}}</ref> 获得1999年度建筑工程[[鲁班奖|鲁班奖]]。

该建筑地上11层，地下3层，具有购物、餐饮、娱乐和办公多种功能。[[新东安影城|新东安影城]]、[[东来顺|东来顺]]新东安总店均位于此处。2005年11月开始，原新东安市场进行升级改造，改建成高档购物场所<ref>{{cite web |url = http://www.ynet.com/view.jsp?oid=6823082 |title = 百年新东安将“改名换面” |publisher = 《[[北京青年报|北京青年报]]》 |date = 2005-11-17 |accessdate = 2011-05-25 |deadurl = yes |archiveurl = https://web.archive.org/web/20070519184727/http://www.ynet.com/view.jsp?oid=6823082 |archivedate = 2007-05-19 }}</ref>。2008年4月20日正式更名为'''北京apm'''。<ref>{{cite web |url = http://news.qq.com/a/20080421/000180.htm |title=新东安正式更名APM |publisher=[[京华时报|京华时报]] |date = 2008年4月21日 |accessdate=2012年11月5日}}</ref> 名字乃參考位於香港[[觀塘|觀塘]]，同屬新鴻基地產的[[創紀之城|創紀之城]]購物商場 [[apm_(香港)|apm]]<ref>{{cite news|title=新东安广场变身北京APM 老字号的品牌逆转故事|url=http://house.ifeng.com/detail/2014_07_28/47552134_0.shtml|publisher=凤凰网|date=2014-07-28}}</ref>。
