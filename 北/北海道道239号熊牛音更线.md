'''北海道道239号熊牛音更线'''（{{lang-ja|北海道道239号熊牛音更線}}）是一条连结[[北海道|北海道]][[十胜综合振兴局|十胜综合振兴局]]管内[[上川郡_(十勝国)|上川郡]][[清水町_(北海道)|清水町]]和[[河東郡_(日本)|河東郡]][[音更町|音更町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 路線数据 ==
* 起点：北海道上川郡清水町字熊牛（＝[[北海道道75号带广新得线|北海道道75号带广新得线]]交点）
* 終点：北海道河東郡音更町字音更西（＝[[北海道道133号音更新得线|北海道道133号音更新得线]]交点）
* 总長：24.3千米

== 共线区间 ==
* [[芽室町|芽室町]]平和西（[[北海道道54号东瓜幕芽室线|北海道道54号东瓜幕芽室线]]）

== 经过的自治体  ==
* 十勝综合振興局
** 上川郡清水町 - 河西郡芽室町 - 河東郡音更町

== 主要连接道路 ==
* 清水町
** 北海道道75号带广新得線＝熊牛（起点）
* 芽室町
* 北海道道54号東瓜幕芽室線＝平和西（重複）
* 音更町
** 北海道道133号音更新得線＝音更西（终点）
** [[北海道道214号川西芽室音更线|北海道道214号川西芽室音更线]]＝東士狩西

== 主要橋梁 ==
* 音更町
** 万年橋（[[然別川|然別川]]）＝字万年東 - 字東士狩西

== 历史 ==
* 1957年3月30日 路線勘定。（1957年北海道告示第493号）
* 1994年10月1日 路線编号变更为239号。（1994年北海道告示第1468号）
