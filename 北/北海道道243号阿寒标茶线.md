'''北海道道243号阿寒标茶线'''（{{lang-ja|北海道道243号阿寒標茶線}}）是一条连接[[北海道|北海道]][[钏路市|钏路市]]和[[川上郡_(北海道)|川上郡]][[標茶町|標茶町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 指定区間 ==
* 起点：北海道釧路市阿寒町舌辛原野（[[国道240号|国道240号]]交点）
* 終点：北海道川上郡標茶町字オソツベツ原野（＝[[国道391号|国道391号]]交点）
* 总長：52.3千米

== 共线区间 ==
* [[阿寒郡|阿寒郡]][[鶴居村|鶴居村]]字幌呂 - 同村字雪裡（[[北海道道53号钏路鹤居弟子屈线|北海道道53号钏路鹤居弟子屈线]]）

== 经过的自治体  ==
* [[钏路综合振兴局|钏路综合振兴局]]
** 釧路市 - 阿寒郡鶴居村 - 川上郡標茶町

== 主要连接道路 ==
* 釧路市
** 国道240号＝阿寒町舌辛原野（起点）
** [[北海道道666号徹别原野釧路线|北海道道666号徹别原野釧路线]]＝阿寒町仁仁志别原野
* 鶴居村
** 北海道道53号釧路鶴居弟子屈线＝幌呂、雪裡（重複）
* 標茶町
** 国道391号＝オソツベツ原野（终点）
** [[北海道道1052号久著吕原野线|北海道道1052号久著吕原野线]]＝クチョロ原野
** [[北海道道1060号久著吕原野塘路线|北海道道1060号久著吕原野塘路线]]＝久著呂原野

== 主要橋梁 ==
* 釧路市
** 阿寒川橋（[[阿寒川|阿寒川]]）＝阿寒町舌辛原野
* 標茶町
** 沼幌橋（ヌマオロ川<small>＝[[釧路川|釧路川]]支流</small>）＝オソツベツ原野

== 历史 ==
* 1957年3月30日 路线勘定。（1957年北海道告示第493号）
* 1994年10月1日 路线编号变更为243号。（1994年北海道告示第1468号）
