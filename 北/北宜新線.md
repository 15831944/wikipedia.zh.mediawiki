'''北宜新線'''，亦稱為'''北宜直線鐵路'''，是針對[[臺灣鐵路管理局|台鐵]][[東部幹線|東部幹線]][[臺北市|臺北]]—[[宜蘭縣|宜蘭]]間現有的95.9公里路線過於冗繞而提出的截彎取直方案，並以「臺鐵南港至花蓮提速改善計畫」的名義進行規劃<ref name="提速改善計畫">[http://www.rrb.gov.tw/04100-591.aspx?id=59 北宜新線鐵路 臺鐵南港至花蓮提速改善計畫]，交通部鐵路改建工程局</ref>。目前全線規劃起自[[南港車站|南港車站]]，經[[大溪車站|大溪車站]]至[[頭城車站|頭城車站]]<ref name="環境影響說明書">[http://www.rrb.gov.tw/04100-592.aspx?id=59 北宜新線鐵路 臺鐵南港至花蓮提速改善計畫 (北宜新線鐵路) 環境影響說明書]，交通部鐵路改建工程局</ref>，尚待第二階段環境影響評估<ref name="提速改善計畫" />。

==簡介==
北宜新線興建的目的，是為了解決東部幹線因線路迂迴，所導致行車時間過長的問題。建設北宜新線可以縮短台北至宜蘭的鐵路客運時間，取代公路運輸的部分功能，以期減輕[[北宜高速公路|北宜高速公路]]的[[塞車|塞車]]問題。

2011年[[交通部鐵路改建工程局|交通部鐵路改建工程局]]的「南港至花蓮提速改善計畫可行性研究期末審查會」確定將北宜新線鐵路被評為最優先方案，同時該案也已經報[[中華民國交通部|中華民國交通部]]審查<ref>{{cite web|url=http://www.cna.com.tw/news/FirstNews/201111190027-1.aspx| title=北宜直鐵有望 最快1年半動工|publisher=中央通訊社|date=2011-11-19|accessdate=2014-04-30 |language=中文}}</ref>

本線台北端起自[[南港車站|南港車站]]（已經於[[臺北鐵路地下化專案|臺北鐵路地下化專案]]施工時預留[[月台|月台]]與[[隧道|隧道]]銜接介面），宜蘭端則選定在[[頭城鎮|頭城鎮]]武營橋北岸，沿線計畫高架路段長10.09公里，原宜蘭線[[礁溪車站|礁溪車站]]配合路線遷址，其中（方案1B）為穿越[[翡翠水庫|翡翠水庫]]集水區，但在[[雪山隧道|雪山隧道]]開工後發現問題，為了避開水源地問題，路線東移計劃：

* 南港＝大溪間行經雙溪新站

2014年4月22日鐵工局決定採用新的方案二<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20140422/383989/ 北宜直鐵路廊拍板　南港通宜蘭47分鐘]</ref>，工期預計為十年，完工後移轉給台鐵經營。預期可縮短北部與東部往來的時間，台北＝宜蘭行車時間將縮為47分鐘。

但[[臺北市市長|臺北市市長]][[柯文哲|柯文哲]]當選後，重新提出最短路線方案，如果翡翠水庫的環保問題是可以解決，仍不放棄爭取<ref>[http://www.ettoday.net/news/20141210/436923.htm#ixzz3LfM1rb2l 北宜直鐵爭議　鄉民還原柯政見：「最短路線」非穿越 | ETtoday政治新聞 | ETtoday 新聞雲 ]</ref>，其方案南港－頭城間36.3公里，以3座長隧道連接，台北－宜蘭間行車時間可望縮短至39分鐘。<ref>[http://www.cna.com.tw/news/firstnews/201412070059-1.aspx 柯林會：不放棄北宜直鐵最短案]</ref>（目前台鐵傾斜式列車行車時間約1小時5分）

==路線資料==
*經營管轄：[[臺灣鐵路管理局|臺灣鐵路管理局]]
*路線距離：臺北市南港＝新北市汐止 = 宜蘭縣大溪＝宜蘭縣頭城間共53[[公里|公里]]（2014年4月選定方案）
**臺北市南港＝宜蘭縣頭城間共36[[公里|公里]]（最短路線方案）
*[[軌距|軌距]]：1,067毫米
*車站數：6（含起訖車站）（2014年4月選定方案）
*雙線區間：全線
*[[電氣化鐵路|電化]]區間：全線（[[交流電|交流]]25KV）
*五分之一為高架路段，其他路段主要由兩個長-{山岳}-隧道組成
*工期約10年

==計劃車站一覽==
此路線最初設計與[[北宜高速公路|北宜高速公路]]類似，計畫以長隧道方式穿越[[坪林區|坪林]]、[[石碇區|石碇]]山區（[[雪山山脈|雪山山脈]]）。但由於北宜高速公路施工後地質災難不斷，且顧慮水源保護區問題，全線東移為經由雙溪，以避開地質敏感區（北宜新線、改善北迴舊線安全設施、[[花東線鐵路瓶頸路段雙軌化暨全線電氣化計畫|花東線電氣化及部份區間雙線工程]]）<ref>{{cite web|url=http://www.rrb.gov.tw/04100.aspx?id=59&lan=ch| title=北宜新線鐵路|publisher=交通部鐵路改建工程局|date=2014-06-13|accessdate=2014-06-16 |language=中文}}</ref>。

== 環境評估問題 ==
北宜新線興建案，於2006年5月19日送至[[行政院環境保護署|行政院環境保護署]][[環境影響評估|環境影響評估]]時，環保署環境影響評估審查委員會作出「不應開發」決議，建議另尋替代方案，退回北宜新線鐵路開闢計畫<ref>{{cite web|url=http://www.epa.gov.tw/FileDownload/FileHandler.ashx?FLID=1786|title=東部鐵路快捷化計畫－北宜直線鐵路（含礁溪車站東移段）初步綜合規劃環境影響說明書|publisher=行政院環境保護署|date=2007-12-27|accessdate=2014-04-30|language=中文|deadurl=yes|archiveurl=https://web.archive.org/web/20140502032435/http://www.epa.gov.tw/FileDownload/FileHandler.ashx?FLID=1786|archivedate=2014-05-02}}</ref>。

環評審議不應開發的四點理由：
# 經過[[新店區|新店]]青潭、基隆河及雙溪自來水、雙溪、貢寮飲用水等水質水量保護區，以及[[汐止區|汐止區]]白雲里康誥坑崩塌地等環境敏感區位，對環境具有重大不利影響。
# 行經[[台北斷層|臺北斷層]]等九處[[斷層|斷層]]，其對環境影響範圍、程度及其因應措施未能確實評估說明。
# 以沿線之地質構造、地下含水層之分佈、含水層面及其水量雖有初步說明，惟未予以深入評估，因此對環境影響亦未能掌握及因應。
# 宜蘭端考古遺址有頭城、頂埔、福成橋、福德坑橋、下埔、二城、大竹圍、奇立丹、份尾等，稍遠的有沙港橋、淇武蘭、瑪璘、武暖…等，如此多之考古遺址均未予以深入評估，因此對環境影響亦未能掌握及因應。

然而宜蘭地方人士仍積極爭取中，前宜蘭縣長[[呂國華|呂國華]]表示：「交通部應未雨綢繆，提早做好興建規劃。」。<ref>[http://www.epochtimes.com/b5/6/10/30/n1503254.htm 北宜直線鐵路無望？呂國華：中央應未雨綢繆]</ref>

2010年12月7日，[[宜蘭縣|宜蘭縣]][[國民黨|國民黨]]籍[[立法委員|立法委員]][[林建榮|林建榮]]安排[[宜蘭縣縣長|宜蘭縣縣長]][[林聰賢|林聰賢]]等30名地方首長及政商代表，拜會[[交通部|交通部]]長[[毛治國|毛治國]]，得到2011年上半年就會完成北宜新線規劃評估的回覆。

交通部同意投入新台幣近千萬元，重新規劃替代路線（分三段整建方案）進行可行性評估。<ref>[http://www2.cna.com.tw/ShowNews/Detail.aspx?pNewsID=201111190116&pType1=TD&pType0=aHEL&pTypeSel=0&pPNo=2 北宜直鐵案評估 政院准了 沈如峰/中央社/宜蘭 2011/11/19 15:51] {{webarchive|url=https://web.archive.org/web/20160304110803/http://www2.cna.com.tw/ShowNews/Detail.aspx?pNewsID=201111190116&pType1=TD&pType0=aHEL&pTypeSel=0&pPNo=2 |date=2016-03-04 }}</ref>鐵路改建工程局於2011年8月曾在宜蘭一場會議中評述此三方案<ref>[http://www.e-land.gov.tw/public/Data/111214342829.pdf 幸福宜蘭－2011年10-11月特刊 - 宜蘭縣政府] {{webarchive|url=https://web.archive.org/web/20120425113709/http://www.e-land.gov.tw/public/Data/111214342829.pdf |date=2012-04-25 }}</ref>之優劣。
*第1方案：選擇含總長37.3公里的[[南港車站|南港]]至[[頭城車站|頭城]]路段，行車時間33分鐘。（路線、行車時間短，最容易轉移國道5號假日車流）
*第2方案：全長49公里的南港至[[大溪車站|大溪]]路段，行車時間39分鐘。（減少環境衝擊，工程經費較方案1節省100億）
*第3方案：全長59.9公里的南港至[[雙溪車站|雙溪]]改善路段與雙溪到大溪的直線鐵路行車時間 47分鐘。（效益最低）
經專家學者對此三方案的「旅遊週期」、「建設成本」、「工期」、「施工風險」、「環境影響」與「經濟效益」等各方權重評析後，第2方案以略優第1方案（差距0.02分）獲選。時任交通部長毛治國於2011年11月21日於[[宜蘭縣|宜蘭]]視察時表示，行政院已經於11月14日核定通過此案之可行性評估，交通部亦逐步執行綜合規劃與相關環評資料，倘作業順利計1年半內可動工興建，屆時台北到宜蘭可縮短至少40分鐘，台北至台東也可縮短至3個小時內完成。

== 礁溪、頭城鐵路高架 ==

2019年3月11日宜蘭縣政府與行政院公共工程委員會達成共識，決議先行施作宜蘭及羅東段，頭城及礁溪段則與北宜直鐵合併辦理。

==暫代措施==
[[File:TRA_TED1002_at_Hualien_Station_20080523.jpg|right]]

由於北宜新線尚未通過[[環境評估|環境評估]]，台鐵為了減低北宜高速公路通車後，國道客運對運量所帶來的衝擊，同時縮短宜蘭線東北角段的路線彎曲所拉長的行車時間，自2006年起導入能高速過彎的[[傾斜式列車|傾斜式列車]]，即[[太魯閣列車|太魯閣號]]與[[普悠瑪列車|普悠瑪號]]，將台北＝宜蘭的行車時間從80分鐘縮短至67分鐘，加強與國道客運的競爭力。但傾斜式列車也有缺點，在於其傾斜機制會比一般列車對鐵軌造成更多的磨損。
