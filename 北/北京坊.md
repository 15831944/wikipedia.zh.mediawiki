'''北京坊'''，位于[[北京市|北京市]][[西城区|西城区]][[大栅栏|大栅栏]]历史文化保护区东北角，是一个以文创等生活体验为主的商业建筑集群。

==简介==
北京坊东到[[珠宝市街|珠宝市街]]，西到[[煤市街|煤市街]]，南到[[廊房二条|廊房二条]]，北到[[前门西河沿街|前门西河沿街]]，不包括[[前门大街|前门大街]]上的[[谦祥益|谦祥益]]。该项目自2006年开始规划，作为大栅栏历史文化保护区的建筑集群设计，西城区邀请[[吴良镛|吴良镛]]任总顾问，[[吴晨|吴晨]]任北京坊项目总建筑师，按照“和而不同”的原则，邀请[[王世仁|王世仁]]、[[朱小地|朱小地]]、吴晨、[[崔恺|崔恺]]、[[朱文一|朱文一]]、[[边兰春|边兰春]]、[[齐欣|齐欣]]7位建筑师参与北京坊沿街8栋单体建筑的设计。整个“北京坊”项目计划建设26栋新建筑，包括上述沿街8栋单体建筑。北京坊整体建筑风格延续[[中华民国|中华民国]]时期建筑特色，恢复了[[排子胡同|排子胡同]]等原有胡同肌理，拆除了原有的绝大部分老建筑，仅保留了[[北京劝业场|北京劝业场]]及其东侧的宝恒祥金店、谦祥益老号旧址<ref>{{Cite web |url =http://china.huanqiu.com/hot/2016-12/9806514.html  |title = 北京文化新地标“北京坊”2017年亮相 |author =  |date =  2016-12-14 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref><ref name=xin>{{Cite web |url = http://www.bj.xinhuanet.com/bjyw/2017-01/17/c_1120325244.htm |title = 北京坊：打造北京文化新地标 |author =  |date = 2017-01-17 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。

2017年1月16日，北京坊建筑集群落成，同时举行北京坊建筑集群设计颁牌仪式。刻有上述7位建筑师姓名并记录设计与建筑特色的纪念铜牌，被悬挂到每位建筑师设计的建筑上，同时在街区内公共区域树立建筑集群设计总牌<ref name=xin/>。2017年1月21日至2月11日，北京坊2017新春文化坊会举行，开幕仪式1月21日在北京劝业场举行，这是北京坊街区与建筑亮相后举办的第一个大型公共文化活动<ref>{{Cite web |url = http://collection.sina.com.cn/scgd/2017-01-22/doc-ifxzusws0051444.shtml |title =中国式新生活：北京坊2017新春文化坊会开幕  |author =  |date = 2017-01-22 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。北京坊引进的主要是展馆、书店、文创等商业形态，不做纯商业楼宇，定位为“中国式生活体验区”<ref name=xin/>。

北京坊建筑集群落成时，廊房头条南侧仍在施工，在工地北墙外临时沿街布置了一个长约300米的“北京坊露天美术馆”。这一理念由艺术家[[王永刚|王永刚]]提出，由北京广安控股有限公司制作。2017年1月21日至3月16日，北京坊露天美术馆的首展“街道的面孔”开展<ref>{{Cite web |url = http://artist.artron.net/yishujia0268308/3-8637.html |title = 北京坊露天美术馆首展：街道的面孔 |author =  |date =2017-01-22  |publisher =  雅昌艺术网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.ifeng.com/a/20170714/51433226_0.shtml |title = 来北京坊露天美术馆看“城市DADA” |author =  |date = 2017-07-14 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

==组成==
整个北京坊以北京劝业场为中心，分为西区（W）和东区（E）。
*[[北京劝业场|北京劝业场]]：位于廊房头条17号，是[[全国重点文物保护单位|全国重点文物保护单位]]。作为当代文化艺术的交流中心“劝业场文化艺术中心”开放。是西城区政府举办文化活动的重要场所，也是每年秋季北京国际设计周的主要展区<ref name=jiachuan>{{Cite web |url =http://www.sohu.com/a/159058278_214269  |title =家 · 传：北京坊老邻居们的家常  |author =  |date = 2017-07-21 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
*W1：广安城市馆，位于北京劝业场西北侧，前门西河沿街南侧。
*W2：位于北京坊西北角，煤市街东侧，前门西河沿街南侧。
*W3：位于W2和W5之间。2017年万仟堂、日用之道开业。
*W4：位于W3西侧，煤市街东侧。2017年英园、祖母的厨房开业。
*W5：位于W3和W7之间。2017年inWe、睛姿眼镜、韩时烤肉、北平咖啡开业。
*W6：位于W5西侧，煤市街东侧。2017年北京天竺综合保税区德国商品展示店、兰芳园开业。
*W7：位于廊房头条北侧，三阳号西侧。2017年巧虎开业。
*W8：位于廊房头条北侧，W7西侧。
*W9：位于北京坊西南角，廊房头条北侧，W8西侧，煤市街东侧。2017年莲香斋、MS CHLOE、红鼎海鲜捞开业。
*W10：三阳号，位于廊房头条19号、21号，北京劝业场西侧。原先作为“廊房头条19号商店”被列为[[北京市西城区普查登记文物|北京市西城区普查登记文物]]。2011年在北京坊建设过程中被违法拆除。2014年西城区文化委员会公布的北京市西城区普查登记文物名单中已不再收录。该店原先名字不详。2017年北京坊建筑集群落成时，该店已仿照原样式重建，西侧二层外墙上嵌有砖雕“三阳号”三字。故该新建筑称“三阳号”或三阳金店。2017年周氏兄弟国际艺术中心开业。
*E11：位于北京坊东北角，前门西河沿街南侧。2017年11月[[葉壹堂|PAGE ONE]]试营业。
**保利国际影城天安门店：2016年12月在北京坊东区地下开业，入口位于E11东部，地处[[前门大街|前门大街]]西侧，[[谦祥益|谦祥益]]北侧<ref>{{Cite web |url = http://theater.mtime.com/China_Beijing_Chaoyang/9389/info.html   |title =保利国际影城天安门店  |author =  |date =  |publisher = 时光网 |language =  |accessdate = 2017-11-01 |quote =  }}</ref>。
*E12：位于北京坊东区，北京劝业场东北侧，前门西河沿街南侧。2017年初，家传文化体验中心开业，是该体验中心在中国开设的首店<ref name=jiachuan/>。
*E13：位于E11和E15之间，[[谦祥益|谦祥益]]西侧。其西侧是个小广场。
*E14：位于廊房头条北侧，[[谦祥益|谦祥益]]南侧。2017年4月27日下午，INTERSPORT中国北区茵特体育战略发展新闻发布会暨北京坊店开业典礼举行<ref>{{Cite web |url =http://www.sohu.com/a/137036156_498578  |title = INTERSPORT中国北区战略发布，茵特体育前门北京坊店隆重开业 |author =  |date = 2017-04-28 |publisher =  搜狐|language =  |accessdate =  |quote =  }}</ref>。
*E15：[[谦祥益|谦祥益]]老号旧址，位于廊房头条11号，“谦祥益老号旧址门面”被列为[[北京市西城区普查登记文物|北京市西城区普查登记文物]]。2017年PINVITA开业。2017年局气在E14、E15开业。
*E16：宝恒祥金店，位于廊房头条15号，北京劝业场东侧。作为“宝恒祥商店”被列为[[北京市西城区普查登记文物|北京市西城区普查登记文物]]。建造年代和设计师不详。三开间，两层，砖木结构。西侧女儿墙上有一组文字砖雕，最上面是“宝恒祥”三字，下方两侧是“金珠店”、“银器行”，中央是一段[[英文|英文]]：“PAO HENG HSIANG GOLD & SILVER WARE OF EVERY DESCRIPTION CHOICE PEARLS FINE WORKMANSHIP ONE PRICE ONLY”。
