'''北京航空'''是一家位于[[中国|中国]][[北京|北京]]的航空公司，其基地为[[北京首都国际机场|北京首都国际机场]]。[[2018年|2018年]][[4月24日|4月24日]]，北京航空获得中国民用航空局批准筹建。<ref>{{Cite web|url=http://www.caac.gov.cn/XXGK/XXGK/TZTG/201804/t20180424_187583.html|title=关于拟批准北京航空有限责任公司筹建的公示|accessdate=2018-04-24|work=www.caac.gov.cn}}</ref>

== 历史 ==
[[File:B-3999@PEK_(20180131165908).jpg|thumb]]]]
原北京航空成立于2011年，由[[中国国际航空|中国航空集团有限公司]]、北京控股集团有限公司、北京市国有资产经营有限责任公司、中达瑞银投资有限公司共同出资组建，经营公务机业务。<ref>{{Cite news|url=http://companies.caixin.com/2018-04-24/101238661.html|title=国航拟重组公务机子公司 或进军北京新机场|accessdate=2018-04-24}}</ref>

有业内人士推测，随着[[北京大兴国际机场|北京新机场]]落成日期日益临近且被赋予服务[[雄安新区|雄安新区]]的重要任务，[[中国南方航空|中国南方航空]]、[[中国东方航空|中国东方航空]]纷纷表示将会在新机场投入大量资源，原北京航空的大股东[[中国国际航空|中国国际航空]]在北京新机场却没有布局。国航重整北京航空，意在北京新机场分一杯羹。<ref>{{Cite web|url=http://finance.china.com/news/11173316/20180424/32345556.html|title=国航筹建新“马甲” ，剑指北京新机场_经济频道_中华网|accessdate=2018-04-24|last=13465,18063|first=ck|work=finance.china.com}}</ref>

== 运营 ==
北京航空与[[中国国际航空|中国国际航空]]统一运营，使用中国国际航空二字代码CA，航线航班、时刻和停机位等资源在中国国际航空资源内统一调配，不新增资源需求。

飞行、机务、签派、乘务、运输、安保等专业技术人员从中国国际航空划拨。北京航空总经理将由曾担任中国国际航空培训部总经理的戴汉勇出任。<ref>{{Cite web|url=https://finance.qq.com/a/20180424/030199.htm|title=北京航空筹建初审获批 将与大股东国航统一运营_财经_腾讯网|accessdate=2018-04-24|work=finance.qq.com|language=zh-CN}}</ref>

== 机队 ==
[[File:Boeing_737-89L_‘B-5443’_Beijing_Airlines_(32582101587).jpg|thumb]]
北京航空目前主要为[[波音737新世代|波音737-800]]飞机，而其公务机机队则主要为[[空中客车A320系列#A318|空中客车A318]]，拟引进飞机在中国国际航空股份有限公司飞机引进指标中统一调配。

== 股权 ==

* [[中国国际航空股份有限公司|中国国际航空股份有限公司]]持有51%股权
* 北京控股集团有限公司持有18%股权
* 北京市国有资产经营有限责任公司持有18%股权
* 中达瑞银投资有限公司持有13%股权
