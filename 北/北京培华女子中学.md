'''培华女子中学'''曾是一所位于[[北京|北京]]的[[教会|教会]]学校，由[[英国|英国]]在华[[传教士|传教士]][[苏慧廉|苏慧廉]]（William Edward Soothill）之女谢福芸于1914年创办。
==创办过程==
[[File:Lin_Huiyin_11.jpg|thumb]]
[[民国|民国]]初年，苏慧廉返回英国筹得300万[[英镑|英镑]]之后，准备在[[中国|中国]]创办“华中联合大学”，但是计划却因为[[第一次世界大战|第一次世界大战]]的爆发而落空。之后，苏慧廉毕业于[[牛津大学|牛津大学]]的女儿谢福芸在北京创办了“培华女子中学”。中国有名的女建筑师[[林徽因|林徽因]]曾于1916年就读于该校。<ref>{{Cite web |url = http://www.timetimetime.net/renwu/16184.html |title = 苏慧廉：他曾影响中国 |author =  |date = 2013-08-28 |publisher = 阅读时间网 |language = zh-hans |accessdate = 2014-12-10 |quote =}}</ref><ref>{{Cite web |url = http://www.christiantimes.cn/news/14166/【独家特稿】沈迦藉寻找苏慧廉还原百多年前来华传教士真实模样 |title = 【独家特稿】沈迦藉寻找苏慧廉还原百多年前来华传教士真实模样 |author = 王新毅 |date = 2014-03-08 |publisher = 基督时报|language = zh-hans |accessdate = 2014-12-10 |quote =  }}</ref>
