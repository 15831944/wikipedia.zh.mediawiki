'''北京外交人员服务局'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[朝阳门内大街|朝阳门内大街]]223号，是隶属[[中华人民共和国外交部|中华人民共和国外交部]]的[[事业单位|事业单位]]<ref>{{cite news|title=北京外交人员服务局成立50周年庆典晚会举行|url=http://news.163.com/12/0121/13/7OA0826000014JB6.html|accessdate=2014-05-08|newspaper=网易}}</ref>。

==简介==
1953年6月，经[[中央人民政府政务院|中央人民政府政务院]]总理[[周恩来|周恩来]]批准，组建'''外交人员服务社'''。1955年初，在外交人员服务社的基础上组建'''外交人员服务处'''。1962年1月2日，经[[中华人民共和国国务院总理|中华人民共和国国务院总理]][[周恩来|周恩来]]、副总理[[陈毅|陈毅]]、[[习仲勋|习仲勋]]审定批准，正式成立'''北京市外交人员服务局'''。该局由外交部和北京市双重领导。1968年3月，改由外交部领导。1970年1月，又改回由外交部和北京市双重领导。1984年7月，北京市外交人员服务局更名为'''北京外交人员服务局'''，隶属外交部<ref>{{Cite web |url = http://www.bds-cn.com/html/home/fazhanlicheng/ |title = 历史沿革 |author =  |date =  |publisher =  北京外交人员服务局|language =  |accessdate = 2016-12-31 |quote =  }}</ref>。
