'''北海道道24号钏路停车场线'''（{{lang-ja|北海道道24号釧路停車場線}}）是一条位于[[北海道|北海道]][[钏路综合振兴局|钏路综合振兴局]][[釧路市|釧路市]]内的[[主要地方道|主要]][[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。通称北大通（きたおおどおり）。

== 道路概况 ==
* 起点：[[北海道|北海道]][[釧路市|釧路市]]北大通13丁目（＝[[北海道旅客铁道|JR]][[根室本线|根室本线]][[钏路站|钏路站]]）
* 終点：北海道釧路市北大通5丁目（＝[[国道38号|国道38号]]・[[国道44号|国道44号]]交点）
* 总長：0.7千米

== 经过的自治体  ==
*[[钏路综合振兴局|钏路综合振兴局]]
**[[釧路市|釧路市]]

== 主要连接道路 ==
'''釧路市'''
* [[北海道道53号钏路鹤居弟子屈线|北海道道53号钏路鹤居弟子屈线]]＝北大通8丁目
* [[国道38号|国道38号]]＝北大通5丁目（终点）
* [[国道44号|国道44号]]＝北大通5丁目（终点）

== 历史 ==
* 1954年3月30日 路線勘定为45号。（1954年北海道告示第503号）
* 1994年10月1日 路線编号变更为24号。（1994年北海道告示第1468号）
