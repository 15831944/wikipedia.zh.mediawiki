'''北上门'''，原址位于今[[北京市|北京市]][[东城区_(北京市)|东城区]][[故宫博物院|故宫博物院]][[神武门|神武门]]以北、[[景山公园|景山公园]]景山门以南，现已无存。

==简介==
北上门位于[[故宫博物院|故宫博物院]][[神武门|神武门]]以北、[[景山公园|景山公园]]景山门以南，坐北朝南。[[元朝|元朝]]史料记载，[[元大都|元大都]]宫城外无护城河，而有“大内夹垣”（即宫城卫城），“大内夹垣”的门是“上门”。可知北上门的命名应在元朝[[元世祖|元世祖]][[至元|至元]]初年修建大都皇宫和大内夹垣时。有学者认为，北上门的前身可能是[[金朝|金朝]][[太宁宫|太宁宫]]内廷的紫宸门，不过这种说法尚未获得公认。[[明朝|明朝]][[永乐|永乐]]年间，按明里制在元皇宫与元大内夹垣之间规划了皇宫护城河（即今[[筒子河|筒子河]]），并拆除了皇宫外侧元代的“大内夹垣”，但保留了“大内夹垣”的北上门、东上门、西上门及阙左门、阙右门<ref name=zgw>{{Cite web |url =http://www.china.com.cn/culture/txt/2008-01/21/content_9562772.htm  |title = 老北京“北上门”南向之谜 |author = 郭文若 |date = 2008-01-21　 |publisher = 中国网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://news.163.com/11/0610/14/766PHVJQ00014AED.html|title = 专家建议恢复“北上门” |author =  |date = 2011-06-10 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

《[[明宫史|明宫史]]》记载，[[崇祯帝|崇祯帝]]游禁苑的路线为：出[[东华门|东华门]]和东上门，沿东卫城与东禁城之间的驰道向北，过东上北门，到东长街（今[[北池子大街|北池子大街]]）北口，折向西至北上东门外，再折向北，进入山左里门。游毕，皇帝过北上门，进入[[神武门|玄武门]]回宫；其他人则按来时原路回宫。可见当时北上门仅供皇帝通行<ref name=zgw/>。

[[清朝|清朝]]定都北京后，继承了明朝的规划并继续沿用北上门<ref name=zgw/>。

[[中华人民共和国|中华人民共和国]]成立后，1956年5月底至6月上旬，北上门被拆除。这是[[北京中轴线|北京中轴线]]上，继北中门、[[地安门|地安门]]之后被拆除的六座城门中的第三座，此后又有[[永定门|永定门]]城楼及箭楼、[[中华门_(北京)|中华门]]被拆除<ref name=zgw/>。
