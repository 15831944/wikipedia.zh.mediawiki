'''台灣基督長老教會北斗教會'''位於[[臺灣|臺灣]][[彰化縣|彰化縣]][[北斗鎮_(台灣)|北斗鎮]]，地址為彰化縣北斗鎮元市街9號，是[[台灣基督長老教會|台灣基督長老教會]]彰化中會員林區所轄的教會之一。

== 沿革 ==
1918年，[[英國|英國]][[蘇格蘭|蘇格蘭]]籍[[梅鑑霧|梅鑑霧]]（[[英文|英文]]：Campbell N. Moody）牧師和許定田長老召集[[溪州鄉|溪州鄉]]東州村信徒於當時[[日治臺灣|日治臺灣]][[臺中廳|臺中廳]][[北斗支廳|北斗支廳]][[舊眉|舊眉區舊眉庄]]東州（今[[彰化縣|彰化縣]][[溪州鄉|溪州鄉]]東州村）成立「溪州基督教佈道所」；1923年，由於當時「溪州基督教佈道所」人數增加引起當時[[日本|日本]]籍[[北斗郡|北斗郡]]郡守石渡英吉長老的關心，所以他和當時北斗街有志林伯餘、陳宇宙等人商議將佈道所遷至北斗，也因此當年的7月12日在北斗街宮前路租一店鋪開設「北斗基督教佈道所」，是北斗教會正式設教之開始，也是現今北斗地區[[基督教|基督教]]的信仰中心之一。

== 建築 ==
「溪州基督教佈道所」時期使用竹材造三間上蓋蔗葉的平房做為禮拜堂使用。1923年，佈道所遷於北斗後在1926年時於現址購地買屋也將原屋改建竹造房屋為禮拜堂用，是為'''第一代建築'''。1935年6月7日受到[[1935年新竹－台中地震|台中墩仔腳大地震]]餘震影響震毀而重建，並於1937年3月25日重新落成，是為'''第二代建築'''。現今'''第三代建築'''於1972年時因木造禮拜堂使用快40年，雖尚穩固，但已感老舊狹小；經會友同意後開始重建禮拜堂，建造二樓之鐵筋磚造建築物，一樓為主日學教室、二樓為禮拜堂，於1974年3月16日落成。

== 歷史 ==
* 1918年：由[[英國|英國]][[蘇格蘭|蘇格蘭]]籍[[梅鑑霧|梅鑑霧]]牧師和許定田長老召集[[溪州鄉|溪州鄉]]東州村信徒於東州村中通巷成立「溪州基督教佈道所」。
* 1923年：[[日本|日本]]籍[[北斗郡|北斗郡]]郡守石渡英吉長老和當時北斗仕紳林伯餘、陳宇宙等人和教會信徒商議，於7月12日時將佈道所遷至北斗街宮前路租借店鋪，由王倚牧師帶領，開設「北斗基督教佈道所」，故現今以此為北斗教會正式設教之開始。
* 1926年：於元市街現址購地買屋，遷移。
* 1928年：改建竹造房屋為禮拜堂。
* 1933年12月24日：舉行教會設教十周年禮拜。
* 1935年6月7日：禮拜堂受到[[1935年新竹－台中地震|台中墩仔腳大地震]]餘震影響震毀而重建。
* 1937年3月25日：禮拜堂重新落成。
* 1939年3月：升格為堂會。
* 1952年12月：建築木造牧師館。
* 1955年4月17日：成立女宣道會。
* 1958年：設立「溪州長老教會」，並於10月10日動土。
* 1959年1月2日：溪州長老教會落成啟用，10月將北斗長老教會溪州堂會會員移屬[[溪州長老教會|溪州長老教會]]。
* 1969年3月：與合興長老教會結為兄弟教會。
* 1970年1月：成立「北斗主愛儲蓄互助社」。
* 1973年6月：教會重建。
* 1974年3月16日：禮拜堂新堂落成。
* 1974年8月：開辦「主愛幼稚園」
* 1982年：牧師館新館動工。
* 1984年1月：牧師館新館落成。
* 1997年4月：本會加入彰化縣生命線團體會員。
* 1997年12月：主愛幼稚園停辦。
* 2008年7月：與彰化YMCA合辦第一屆社區快樂兒童營。
* 2010年7月：停辦社區快樂兒童營。
* 2011年1月：與[[韓國|韓國]]{{link-ko|大韓耶穌教長老會|대한예수교장로회(합동)}}[[三一教會|三一教會]]合辦第一屆[[韓國|韓國]]三一教會北斗短宣<ref>{{Cite web|url=http://tcnn.org.tw/archives/10468|title=來台20年　韓國三一教會千人歡聚|accessdate=2017-11-25|author=林宜瑩|date=2014-08-13|last=|work=|publisher=台灣教會公報新聞網|language=}}</ref>。
* 2011年11月：與[[世界展望會|世界展望會]]合辦品格教育才藝班。
* 2012年7月：與彰化[[基督教青年会|YMCA]]復辦社區快樂兒童營。
* 2013年7月：[[烏干達|烏干達]]籍Nkambwe Steven牧師信息分享。
* 2013年7月12日：舉行設教90週年禮拜。
* 2013年11月10日：舉行90週年感恩禮拜。
* 2013年11月24日：舉辦90週年園遊會<ref>{{Cite web|url=http://tcnn.org.tw/archives/8510|title=彰化中會北斗教會　園遊會逛展覽慶週年|accessdate=2017-11-25|author=黃以勒|date=2013-11-27|last=|work=|publisher=台灣教會公報新聞網|language=}}</ref>
* 2013年12月22日：與[[天主教|天主教]][[北斗聖神堂|北斗聖神堂]]、[[貴格會|貴格會]]北斗教會、北斗平安台福教會舉辦北斗鎮聯合報佳音活動。
* 2014年8月3日：台裔美籍[[NBA|NBA]]球星[[林書豪|林書豪]]之父親[[林繼明|林繼明]]博士分享見證<ref>{{Cite web|url=http://tcnn.org.tw/archives/14198|title=林書豪為主耶穌打球 父林繼明回鄉做見證|accessdate=2017-11-25|author=黃以勒|date=2014-08-13|last=|work=|publisher=台灣教會公報新聞網|language=}}</ref><ref>{{Cite web|url=https://www.ct.org.tw/1259527|title=林書豪父親林繼明 回鄉分享豪爸的教養課|accessdate=2017-11-25|author=何毓芬|date=2015-07-16|work=|publisher=基督教論壇報|language=}}</ref>。
