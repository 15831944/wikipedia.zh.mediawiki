== 劇情 ==
故事背景為1990年代中期的[[朝鮮半島|朝鮮半島]]，改編自韓國[[大韓民國國家情報院|國家安全企劃部]]的情報員「黑金星」的真實故事<ref>{{Cite web |url = https://stars.udn.com/star/story/10090/3138696 |title = 南北韓危機未解？「北風」坎城首映震撼全球 |author = 陳建嘉 |date = 2018-05-12 |publisher = 聯合報 |language = zh-hant |accessdate = 2018-05-13 }}</ref>。劇中主角偽裝成商人與朝鮮進行商業交易，藉此滲透至各軍事單位打探情報，卻陷入兩國權力高層之間的恐怖鬥爭<ref name = C260404/>。

== 製作 ==
2017年1月24日開始拍攝，並於同年7月25日殺青<ref>{{cite web|url=http://koreanfilm.or.kr/jsp/news/news.jsp?pageIndex=2&blbdComCd=601006&seq=4407&mode=VIEW|title=HWANG Jung-min Wraps NK Action-Drama BLACK VENUS|website=Koreanfilm.or.kr|accessdate=11 January 2018}}</ref>。因劇組無法於北京實地取景，而改在臺灣的台北、基隆、高雄等地為期1個半月拍攝<ref>{{Cite web |url = https://www.mirrormedia.mg/story/20180512insight001/ |title = 黃晸玟祕密來台拍攝　《北風》坎城首映獲好評 |author = 廖佩玲 |date = 2018-05-12 |publisher = 鏡週刊 |language = zh-hant |accessdate = 2018-05-13 }}</ref>。將台北市的[[華西街夜市|華西街夜市]]打造成[[王府井大街|王府井大街]]，以及[[中山樓_(臺北市)|中山樓]]改成[[金正日|金正日]]接見黑金星的[[萬壽臺議事堂|萬壽臺議事堂]]<ref>{{Cite web |url = https://stars.udn.com/star/story/10090/3138816 |title = 韓片「北風」把台北變北京與北韓 臨演台灣國語露餡？ |author = 唐在揚 |date = 2018-05-12 |publisher = 聯合報 |language = zh-hant |accessdate = 2018-05-13 }}</ref><ref>[https://www.mirrormedia.mg/story/20180906ent007/ 韓片《北風》來台拍戲巧奪天功　華西街中山樓通通被改造]. 鏡週刊. 2018-09-06.</ref>。
