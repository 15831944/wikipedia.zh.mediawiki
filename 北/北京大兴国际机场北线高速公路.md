'''北京大兴国际机场北线高速公路'''（[[京津冀跨省市省级高速公路|京津冀高速]]'''S3300'''，即原'''新机场北线高速公路'''<ref>{{cite web |author1=北京市质量技术监督局 |title=京津冀跨省市省级高速公路命名和编号规则（北京市地方标准 DB11/T 3003—2016） |url=http://202.106.162.203/DB11!T~3003-2016.pdf?id=U3RhbmRhcmQsMjA5OTQ2Mg== |accessdate=2019-02-08 |archive-url=https://web.archive.org/web/20190102143129/http://202.106.162.203/DB11!T~3003-2016.pdf?id=U3RhbmRhcmQsMjA5OTQ2Mg== |archive-date=2019-01-02 |dead-url=no }}</ref>）是一条位于[[北京大兴国际机场|北京大兴国际机场]]北侧的横向高速，是大兴国际机场外围“五纵两横”综合交通主干路网{{notetag|其中，“五纵”分别是{{China Expwy Name|G45|[[京开高速公路|京开高速]]}}、[[京雄城际铁路|京雄城际铁路]]、{{China Expwy Name|S3501|[[北京大兴国际机场高速公路|大兴机场高速]]|P=Beijing}}、[[北京地铁|地铁]][[北京地铁新机场线|新机场线]]、{{China Expwy Name|G3|[[京台高速|京台高速]]}}，“两横”分别是{{China Expwy Name|S3300|[[北京大兴国际机场北线高速公路|大兴机场北线]]|P=Beijing}}、[[城际铁路联络线|城际铁路联络线]]。}}的重要组成部分，满足[[石家庄|石家庄]]、[[保定|保定]]、[[大兴区|大兴]]、[[天津市|天津]]、[[沧州|沧州]]、[[唐山|唐山]]等地进入大兴国际机场的通道需求。其中，北京段中段计划于2019年6月开通。

==概況==
大兴机场北线高速西至[[涿州|涿州]]，东至[[廊坊|廊坊]]，与[[京开高速公路|京开高速公路]]、[[北京大兴国际机场高速公路|大兴机场高速公路]]、[[京台高速公路|京台高速公路]]、[[首都环线高速公路|首都环线高速公路]]、[[京雄高速公路|京雄高速公路]]等公路连接，为双向八车道高速公路。
===北京段===
大兴机场北线高速（北京段）全长约25公里，西至京冀界（涿州），东至京冀界（廊坊），设计速度120公里/小时，设有主线桥22座、互通立交4座，分为中段、东延段、西延段三段。其中中段桥梁于2019年3月由[[北京市规划和国土资源管理委员会|北京市规划和国土资源管理委员会]]公示命名<ref>{{cite web |title=关于新机场北线（京开高速-京台高速）高速公路工程桥梁名称命名预案的公示书（公示期限7天） |url=http://ghgtw.beijing.gov.cn/cxgh/21/61/413226/280745/index.html |publisher=[[北京市规划和自然资源委员会|北京市规划和自然资源委员会]] |accessdate=2019-03-12 |date=2019-03-07}}</ref>。

大兴机场北线高速（北京段）中段西起[[京开高速公路|京开高速]]黄垡桥北980米处，东至[[京台高速公路|京台高速]]大渠桥南侧，上跨[[京九铁路|京九铁路]]、[[北京大兴国际机场南北航站楼联络线|南北航站楼联络线]]，下穿[[北京大兴国际机场高速公路|大兴机场高速]]、[[京九客运专线|京九客运专线]]、[[北京地铁|地铁]][[北京地铁新机场线|新机场线]]，长约14.66公里，工程总造价约83.6亿元<ref>{{cite web |author1=北京市首都公路发展集团有限公司 |title=新机场北线（京开高速公路-京台高速公路）高速公路工程环境影响报告书 |url=http://app2.bjepb.gov.cn/cpout/showout/ProjViewManageTypeAccept.aspx?county=1&ProjId=60e3e2af-bb9c-4a69-bbed-dc0e655a0d3d283&title=0 |publisher=大兴区环保局 |accessdate=2019-02-15 |date=2016-11-18}}</ref>，已于2017年12月开工<ref>{{cite news |title=北京新机场北线高速公路今开工 2019年将全线贯通 |url=http://www.chinahighway.com/news/2017/1149638.php |accessdate=2019-02-08 |work=中国公路网 |date=2017-12-14 |archive-url=https://web.archive.org/web/20190209124053/http://www.chinahighway.com/news/2017/1149638.php |archive-date=2019-02-09 |dead-url=no }}</ref>。全线计划于2019年6月与[[北京大兴国际机场高速公路|大兴机场高速]]同步开通<ref>{{cite news |title=新机场高速、新机场北线高速2019年贯通 |url=http://www.beijing.gov.cn/bmfw/jtcx/ggts/t1567993.htm?from=groupmessage&isappinstalled=0 |accessdate=2019-02-08 |work=北京市政务门户网站 |date=2018-11-15 |archive-url=https://web.archive.org/web/20190209125846/http://www.beijing.gov.cn/bmfw/jtcx/ggts/t1567993.htm?from=groupmessage&isappinstalled=0 |archive-date=2019-02-09 |dead-url=no }}</ref>。

大兴机场北线高速（北京段）东延段西起[[京台高速公路|京台]]互通立交，东至京冀界{{notetag|name="lf"}}，长约2.9公里，与大兴机场北线高速（廊坊段）相接；西延段东起[[京开高速公路|京开]]互通立交，连续上跨[[永定河|永定河]]灌渠、永定河左堤路及[[永定河|永定河]]，西至京冀界{{notetag|name="zz"}}，长约8.4公里，与大兴机场北线高速（涿州段）相接。东西延工程计划于2017年年底前开工，2019年7月前开通<ref>{{cite news |title=北京新机场北线高速东西延计划年底前开工，2019年7月建成通车 |url=http://www.china-highway.com/Home/News/bencandy/id/132509.html |accessdate=2019-02-14 |work=中国高速网 |date=2017-10-16}}</ref>。

===廊坊段===
大兴机场北线高速（廊坊段）西至大兴机场北线高速（北京段）京冀界，与[[首都环线高速公路|首都环线高速公路]]相交，东至[[廊坊市|廊坊市]][[广阳区|广阳区]]艾各庄村西南西外环，与规划的艺术大道西延顺接，全长9.97公里，于2018年2月开始前期工作<ref>{{cite web |author1=河北省发展改革委 |title=关于同意北京新机场北线高速公路廊坊段建设项目开展前期工作的函（冀发改基础〔2018〕235号） |url=http://info.hebei.gov.cn/eportal/ui?pageId=1966210&articleKey=6778946&columnId=330035 |publisher=[[河北省人民政府|河北省人民政府]] |accessdate=2019-02-14 |date=2018-02-13}}</ref>，同年7月批准用地<ref>{{cite web |author1=河北省国土资源厅 |title=河北省国土资源厅关于新机场北线高速公路廊坊段项目用地的预审意见（冀国土资函〔2018〕409号） |url=http://www.hebgt.gov.cn/heb/zxbs/spjggb/ghjh/201532567581889.html |publisher=河北省人民政府 |accessdate=2019-02-14 |date=2018-07-26 |archive-url=https://web.archive.org/web/20181001230310/http://www.hebgt.gov.cn/heb/zxbs/spjggb/ghjh/201532567581889.html |archive-date=2018-10-01 |dead-url=no }}</ref>，同年9月开工，其中西环至廊坊主线收费站段设计速度100公里/小时，廊坊主线收费站至京冀界段设计速度120公里/小时<ref name="jj">{{cite web |author1=廊坊市交通运输局 |title=新机场北线高速公路廊坊段PPP项目市场测试公告 - 项目简介 |url=http://www.lf.gov.cn/Item/77369.aspx |publisher=廊坊市人民政府 |accessdate=2019-02-14 |date=2018-04-02}}</ref>。该路段与全长6.2公里的[[105国道|105国道]]空港新区连接线同步建设，总投资约42.92亿元<ref>{{cite news |author1=齐雷杰 |title=北京新机场北线高速廊坊段全线开工建设 |url=http://www.xinhuanet.com/local/2018-09/26/c_1123484161.htm |accessdate=2019-02-08 |work=[[新华网|新华网]] |date=2018-09-26 |archive-url=https://web.archive.org/web/20190209123747/http://www.xinhuanet.com/local/2018-09/26/c_1123484161.htm |archive-date=2019-02-09 |dead-url=no }}</ref><ref>{{cite news |author1=陈正 |author2=孟宪峰 |title=北京新机场北线高速廊坊段全线开工 |url=http://hebei.hebnews.cn/2018-10/07/content_7053791.htm |accessdate=2019-02-08 |work=河北新闻网 |date=2018-10-07 |archive-url=https://web.archive.org/web/20190209191454/http://hebei.hebnews.cn/2018-10/07/content_7053791.htm |archive-date=2019-02-09 |dead-url=no }}</ref>。
===涿州段===
大兴机场北线高速（廊坊段）东至大兴机场北线高速（北京段）京冀界，向西连接[[京雄高速公路|京雄高速公路]]后连接[[京港澳高速公路|京港澳高速公路]]，远期规划延伸至[[京昆高速公路|京昆高速公路]]<ref name="jj"/>。
