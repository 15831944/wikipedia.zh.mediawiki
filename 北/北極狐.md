'''北极狐'''（[[學名|學名]]：{{lang|la|''Alopex lagopus''}}）别名'''雪狐'''或'''白狐'''，在寒冷的[[北極|北極]][[凍原|凍原]]地區是常見的小型[[犬科|犬科]]動物，但由於其毛皮是市场上的高档貨，因此成了人们竞相猎捕的目标。

== 特征 ==
北极狐额面狭，[[吻|吻]]部很尖，耳短而圆，颊后部生长毛，脚底部也密生长毛，所以适于在冰雪地上行走，尾毛蓬松，尖端白色，身体略小于[[赤狐|赤狐]]。北极狐毛皮既长又软且厚，所以北极狐可忍受严寒。冬天毛色为纯雪白色，仅无毛的鼻尖和尾端黑色，自春天至夏天逐渐转变为青灰色，特称“青狐”。

雄性北极狐体长约55厘米，尾长31厘米，体重3.8公斤；雌性53厘米，30厘米，3.1公斤。肩高在25-30厘米之间。

== 食物 ==
北极狐的食物包括[[旅鼠|旅鼠]]、[[鱼|鱼]]、[[鸟类|鸟类]]与鸟[[蛋|蛋]]、[[浆果|浆果]]和[[北极兔|北极兔]]，有时也会漫游[[海岸|海岸]]捕捉贝类，但北極狐主食还是旅鼠。当北极狐闻到在旅鼠窝的气味或听到旅鼠窝里旅鼠的尖叫声时，它会迅速地挖掘位于雪下面的旅鼠窝，当扒得差不多时，北极狐会高高跳起，借着跃起的力量，用腿将旅鼠窝压塌，然后将窝里的旅鼠一网打尽。在极度饥饿的情况下，北极狐会自相攻击。

== 分布和习性 ==
北极狐分布于[[北冰洋|北冰洋]]的沿岸地带及一些岛屿上的[[苔原地带|苔原地带]]，能在零下70°環境下，北极狐喜欢在丘陵地带筑巢，而北极狐的巢有几个出入口，当遇到暴风雪时，北极狐可以呆在窝里一连几天不出来。北极狐年年都为它的巢穴进行一些维修和扩展，以便能长期居住。[[夏天|夏天]]，当食物丰富时，北极狐会把部分食物储存在它的巢穴中。[[冬天|冬天]]，当巢穴中所储存的食物被消耗殆尽时，白狐会跟踪北极熊，拣食[[北极熊|北极熊]]所吃剩的残羹剩饭。所以冬天里北极熊身后总会有2到3只白狐在悄悄地跟踪。但当北极熊非常饥饿时，它也会攻击北极狐。

== 家庭 ==
成年的北極狐交配後，會組成一個家庭，由公北極狐去找食物，而北極狐被飢餓的北極熊攻擊時，失去公狐狸的家庭會另尋居住地或投靠別的北極狐，不少北極狐幼仔往往在這時候凍死。
