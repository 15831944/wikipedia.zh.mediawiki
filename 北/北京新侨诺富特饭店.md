'''北京新侨诺富特饭店'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[崇文门西大街|崇文门西大街]]1号。

==简介==
1954年，'''新侨饭店'''建成开业，成为当时北京屈指可数的高级饭店之一。1998年，饭店重新装修<ref name=sohu>{{Cite web |url = http://travel.sohu.com/20080629/n257814396.shtml |title =北京新侨诺富特饭店  |author =  |date = 2008-06-29 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。2001年，[[法国|法国]][[雅高集团|雅高集团]]参与管理北京首旅集团旗下的北京新侨饭店，饭店更名为'''北京新侨诺富特饭店'''<ref>{{Cite web |url =http://finance.sina.com.cn/o/20011202/150120.html  |title = 首旅集团引进法国品牌 2008年前发展50家酒店 |author =  |date = 2001-12-02 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://finance.sina.com.cn/b/20040620/1851823980.shtml |title = 首家全跃层酒店诺富特海润将落户中关村 |author =  |date =2004-06-20  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。2004年，位于北京新侨诺富特饭店2层的凯涛法餐厅在装修后重张开业<ref>{{Cite web |url = http://www.cnr.cn/metro/chi/200407/t20040731_504043524.shtml |title =凯涛法餐厅 新装亮相京城——北京新侨诺富特饭店  |author =  |date =2004-07-31  |publisher =中国广播网  |language =  |accessdate =  |quote =  }}</ref>。

北京新侨诺富特饭店位于[[崇文门|崇文门]]路口西北侧，分为三星（B楼）6层、四星（A楼）14层。裙楼2到5层是餐厅，饭店内设有各种娱乐设施，以及温泉水疗中心<ref name=sohu/><ref>{{Cite web |url = http://www.chinatoday.com.cn/chinese/society/zx/201707/t20170728_800101173.html |title =北京新侨诺富特饭店推出公益募捐活动  |author =  |date = 2017-07-28 |publisher = 今日中国 |language =  |accessdate =  |quote =  }}</ref>。
