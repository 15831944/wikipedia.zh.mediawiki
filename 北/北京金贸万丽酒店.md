==简介==
北京金贸万丽酒店的前身是王府井大飯店，于1994年开业<ref>[http://cltzz.bjfu.edu.cn/xsxx/75591.htm 西卡国际设计学校免费公开课开课通知，北京林业大学，于2013-04-22查阅] {{webarchive|url=https://web.archive.org/web/20140311080641/http://cltzz.bjfu.edu.cn/xsxx/75591.htm |date=2014-03-11 }}</ref>。2003年重新进行了装修。王府井大饭店楼高14层，共有客房399间，其中行政楼层可俯瞰[[紫禁城|紫禁城]]的全貌。王府井大饭店还设有室内泳池、阳光娱乐俱乐部、会议厅等等设施<ref>[http://travel.qianlong.com/33916/2011/07/29/3322@7212539.htm 北京王府井大饭店，千龙网，2011-07-29] {{webarchive|url=https://web.archive.org/web/20130424083934/http://travel.qianlong.com/33916/2011/07/29/3322@7212539.htm |date=2013-04-24 }}</ref>。

经过装修改造，2014年9月1日，该酒店作为“北京金贸万丽酒店”开业，这是“万丽”在北京的第二家酒店，由由中国金茂（集团）有限公司投资兴建，由万豪国际管理。酒店设有金茂宴会厅、五间会议室、室内游泳池、水疗中心、健身中心、“万丽轩”中餐厅、“燃”全天候餐厅、“R Bar”、“福盈阁”四合院餐厅等设施<ref>{{Cite web |url =http://news.163.com/14/0902/14/A554CM1R00014Q4P.html  |title = 金茂万丽酒店落户王府井 |author =  |date = 2014-09-02 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url =http://epaper.jinghua.cn/html/2014-09/03/content_122839.htm|title =金茂万丽酒店正式开业|author =|date =2014-09-03|publisher =京华网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170227234947/http://epaper.jinghua.cn/html/2014-09/03/content_122839.htm|archivedate =2017-02-27}}</ref><ref>{{Cite web |url =  http://travel.163.com/15/0902/16/B2H6OJRH00064M2L.html|title = 万“丽”挑一 北京金茂万丽酒店荣庆开业一周年 |author =  |date = 2015-09-02 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.163.com/14/0818/11/A3U5O03E00014JB6.html  |title =  北京金茂万丽酒店落户京城 今年9月正式开业迎宾|author =  |date = 2014-08-18 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。
