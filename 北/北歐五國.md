'''北歐五國'''是位於[[北歐|北歐]]的[[丹麥|丹麥]]、[[芬蘭|芬蘭]]、[[冰島|冰島]]、[[挪威|挪威]]和[[瑞典|瑞典]]及其附屬領土如[[法羅群島|法羅群島]]、[[格陵蘭|格陵蘭]]和[[奧蘭群島|奧蘭群島]]的統稱。

該五國及三自治地區的歷史背景緊密連繫，社會和政治制度也相近。政治上雖然不是共同體，但都參與[[北歐理事會|北歐理事會]]；語言上有三種語系，分別為[[印歐語系|印歐語系]]的[[斯堪的納維亞語支|斯堪的納維亞語支]]、[[烏拉爾語系|烏拉爾語系]]的[[芬蘭-烏戈爾語族|芬蘭-烏戈爾語族]]和[[薩米語|薩米語]]，以及[[愛斯基摩-阿留申語系|愛斯基摩-阿留申語系]]的[[格陵蘭語|格陵蘭語]]。北歐五國共佔地350萬平方公里（格陵蘭佔其中60%的土地），人口約2,500萬。

近年，普遍被歸類為[[波羅的海三國|波羅的海三國]]的[[愛沙尼亞|愛沙尼亞]]爭取躋身成為北歐國家一員。北歐五國是愛沙尼亞重要的投資和貿易夥伴，而愛沙尼亞在語言、種族和文化上與芬蘭有關聯，文化上亦與瑞典和丹麥有相似的地方。

== 與斯堪地納維亞的比較 ==
[[File:Scandinavia location map definitions.PNG|thumb|250px

'''橙色部分：'''可被視為屬於斯堪地納維亞的國家；<br>
'''黃色部分：'''最寬鬆的定義，與紅色和橙色部分統稱北歐五國。]]
{{see also|斯堪地納維亞}}
通常'''斯堪地納維亞'''指的是丹麥、挪威與瑞典，而'''北歐五國'''則清楚說明為丹麥、芬蘭、冰島、挪威和瑞典這五個國家，再加上各自的海外屬地（格陵蘭、法羅群島與奧蘭群島）<ref name="Nordic Council">{{cite web|url=http://www.norden.org/en/about-nordic-co-operation |title=關於北歐五國之合作論壇 |publisher=[[北歐理事會|北歐理事會與北歐部長理事會]] |date=2007年10月1日 |accessdate=2008年1月9日 |first=Elvind |last=Saetre |quote=北歐五國之組成為丹麥與其屬地（法羅群島和格陵蘭）、芬蘭與其屬地（奧蘭群島）、冰島、挪威和瑞典。 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20140326122905/http://www.norden.org/en/about-nordic-co-operation |archivedate=2014年3月26日}}</ref>，因此斯堪地納維亞即是北歐五國的一部分。另外尚有一種分類是[[芬諾斯堪底亞|芬諾斯堪底亞]]，其指的是斯堪地納維亞、芬蘭和[[卡累利阿|卡累利阿]]，但不包括丹麥與其他海外屬地。不過此種劃分方式僅是[[地質學|地質學]]的專有名詞，亦指[[波羅的地盾|芬挪斯堪地盾]]（或稱作波羅的地盾）。

斯堪地那維亞國家：
* {{flagcountry|丹麥}}（[[議會制|內閣]][[君主立憲制|君主立憲制]]。)
* {{flagcountry|挪威}}（內閣君主立憲制，並於1905年獨立成為[[君主國|君主國]]。）
* {{flagcountry|瑞典}}（內閣君主立憲制。）

北歐五國：
* 斯堪地那維亞三國
* {{flagcountry|芬蘭}}（[[議會共和制|議會共和制]]，於1917年脫離[[俄羅斯帝國|俄羅斯帝國]]獨立。）
* {{flagcountry|冰島}}（議會共和制，雖於1918年成立冰島共和國，但一直到1944年正式獨立前仍與丹麥組[[共主邦聯|共主邦聯]]。）

與其海外屬地：
* {{flagcountry|法羅群島}}（丹麥領土，於1948年獲丹麥授與[[高度自治|高度自治]]。）
* {{flagcountry|格陵蘭}}（丹麥領土，於1979年獲自治權。）
* {{flagcountry|奧蘭群島}}（芬蘭領土，自1920年起成為芬蘭的自治省。）

== 歷史 ==

北歐五國有共同的歷史，在社會和文化上關係密切。在[[歐洲黑暗時代|黑暗時代]]，現時挪威、瑞典、丹麥和冰島一帶有着相似的文化、語言（[[古諾爾斯語|古諾爾斯語]]）和[[宗教|宗教]]（[[北歐神話|北歐神話]]）。基督教於1000年傳入後，本土化使丹麥、瑞典、挪威形成三個獨立王國。由12世紀起，現時是芬蘭的地區（語言上屬[[芬蘭-烏戈爾語族|芬蘭-烏戈爾語族]]）開始融入瑞典，形成了瑞典王國；而冰島、法羅群島、[[設德蘭群島|設德蘭群島]]、[[奧克尼群島|奧克尼群島]]、格陵蘭和大部分[[蘇格蘭|蘇格蘭]]和[[愛爾蘭|愛爾蘭]]則屬於挪威。所有北歐五國跟隨[[宗教改革|宗教改革]]並接納[[路德宗|路德宗]]。

14世紀，丹麥、挪威（連同冰島）和瑞典（連同芬蘭）組成[[卡爾馬聯盟|卡爾馬聯盟]]，由同一[[君主|君主]]統治。丹麥很快主導了聯盟，但在16世紀初期，瑞典重新成立獨立王國；而丹麥對挪威的支配，直至1814年被迫將挪威割讓予瑞典國王才完結；冰島、格陵蘭和法羅群島仍屬丹麥。17世紀，瑞典躋身成為歐洲[[大國|大國]]之一，但其後逐一失去領土，甚至到1809年失去芬蘭；芬蘭成為附庸[[俄羅斯|俄羅斯]][[沙皇|沙皇]]的自治[[芬蘭大公國|芬蘭大公國]]。

1905年，瑞典與挪威自19世紀開始的聯盟隨着挪威不滿而瓦解。與此同時，[[斯堪地那維亞主義|斯堪地那維亞主義]]在斯堪地那維亞冒起，旨在統一瑞、挪、丹三國，但成效不大。[[一戰|一戰]]及俄國[[十月革命|十月革命]]期間，芬蘭成為獨立國家，組成北歐國家共同體的想法開始出現。[[二戰|二戰]]期間的1944年，冰島由丹麥獨立出來，至此，其後在1952年組成的北歐理事會的成員才全部成立。

二戰後，北歐五國的政策有共同的特色，如所有北歐國家都有由稅收支付的龐大公共福利體系，以及[[社會民主主義|社會民主主義]]的立法政體，主要由在兩戰期間上台的社會民主主義者主導。
