'''北京贵宾楼饭店'''（{{lang|en|Grand Hotel Beijing}}）位于[[中國|中國]][[北京市|北京市]][[东城区_(北京市)|东城区]][[东长安街|东长安街]]35号，是一家五星級[[酒店|酒店]]<ref name=yel/>。

==简介==
1973年，[[北京饭店|北京饭店]]建东楼时， [[中华人民共和国国务院总理|中华人民共和国国务院总理]][[周恩来|周恩来]]亲自参与提出意见，建议以旅游饭店为主的格局兴建东楼。后来周恩来的这个愿望在十多年后建设贵宾楼时被实现<ref name=ky>[http://www.gotoread.com/mag/51/sarticle_35651.html 北京贵宾楼饭店迎来开业20年，商业文化，2010-11-23] {{webarchive|url=https://web.archive.org/web/20160314064033/http://gotoread.com/mag/51/sarticle_35651.html |date=2016-03-14 }}</ref>。

1990年9月22日，北京贵宾楼饭店開業。该饭店是北京首都旅游集团有限责任公司、[[香港|香港]][[霍英东|霍英东]]投资有限公司合资建造的豪华五星级饭店，并由霍英东定名“贵宾楼”<ref name=yel>[http://www.126hotel.cn/hotel/theme/info.php?hotelid=6&columnid=1 酒店简介，北京贵宾楼饭店，于2012-9-28查阅] {{webarchive|url=https://web.archive.org/web/20140310141849/http://www.126hotel.cn/hotel/theme/info.php?hotelid=6&columnid=1 |date=2014-03-10 }}</ref><ref name=qinmin>{{Cite web |url = http://finance.ifeng.com/a/20130727/10281032_0.shtml |title =贵宾楼改走亲民路线 外卖熟食窗口开张  |author =  |date =2013-07-27  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。该饭店按国际五星级饭店的标准进行设计，重点接待国内外重要政治任务<ref name=ky/>。

北京贵宾楼饭店的所在地原先是[[清朝|清朝]]末年的神庙[[堂子|堂子]]。该饭店南侧为[[北京皇城|北京皇城]]牆的东南角，东侧连接[[北京饭店|北京饭店]]且中间连接部分横跨[[晨光街|晨光街]]（现晨光街该段已改成该饭店内部道路），西临[[南河沿大街|南河沿大街]]，北隔[[霞公府街|霞公府街]]同[[天安大厦|天安大厦]]相对，十分靠近[[紫禁城|紫禁城]]、[[天安門廣場|天安門廣場]]及[[王府井大街|王府井大街]]等[[旅遊|旅遊]]名勝<ref name=yel/>。

北京贵宾楼饭店有限公司成立于1986年，是中外合资企业。2010年代，该公司的中方投资人是北京市北京饭店，外方投资人是德凯投资有限公司<ref>{{Cite web |url =http://qyxy.baic.gov.cn/xycx/queryCreditAction!qyxq_view.dhtml?reg_bus_ent_id=6C16A981BD2D47C6B7B4E2AABDA42494&credit_ticket=CA58A8BB34217B4E554260AD801A9B84  |title =北京贵宾楼饭店有限公司  |author =  |date =  |publisher =北京市企业信用信息网  |language =  |accessdate =2018-02-27  |quote =  }}</ref>。

2010年，北京贵宾楼饭店推出红墙熟食礼盒，成为该饭店外卖畅销产品之一。后来还在饭店大楼西北侧开设了熟食外卖窗口。同时，饭店二层的御福会餐厅也改走亲民路线，新推出了整套平价百姓菜单<ref name=qinmin/>。
