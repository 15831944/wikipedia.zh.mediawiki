'''北海道道1038号直别共荣线'''（{{lang-ja|北海道道1038号直別共栄線}}）是一条连结[[北海道|北海道]][[十胜综合振兴局|十胜综合振兴局]]管内的[[浦幌町|浦幌町]]直别和吉野共荣的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：北海道[[十勝郡|十勝郡]][[浦幌町|浦幌町]]直別（＝[[国道38号|国道38号]]交点）
* 終点：北海道十勝郡浦幌町共荣（＝国道38号交点）
* 总長：30.1千米

== 经过的自治体  ==
* 十胜综合振兴局
** 十勝郡浦幌町

== 主要连结道路・隧道 ==
* 浦幌町
** [[国道38号|国道38号]]
** 厚内隧道(218.8米)
** [[国道336号|国道336号]]＝共荣

== 历史 ==
* 1983年3月31日 路線勘定。<!--1983年北海道告示第674号-->

== 其他 ==
