'''北海道醫療大學站'''（{{jpn|j=北海道医療大学駅|hg=ほっかいどういりょうだいがく|rm=Hokkaidō-Iryōdaigaku eki|f=y}}）是位於[[北海道|北海道]][[石狩郡|石狩郡]][[當別町|當別町]]字金澤，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[札沼線|札沼線]]（學園都市線）的[[鐵路車站|鐵路車站]]。車站編號為'''G14'''。電報略號為'''{{lang|ja|イリ}}'''。

== 概要 ==
本車站為[[北海道醫療大學|北海道醫療大學]]當別校園最近的車站。同時是札沼線（學園都市線）中使用[[Kitaca|Kitaca]]最北的車站，以及此路線進行[[電氣化鐵路|交流電氣化]]的終點站。本車站以北的車站並沒有設置車站編號，列車數量亦大幅減少。而往[[新十津川車站|新十津川車站]]方向的列車，由鄰近的[[石狩當別車站|石狩當別車站]]開始改為[[一人控制|一人控制]]。

由2008年（平成20年）10月25日開始直至現在，本車站仍是使用交通系[[智能卡|智能卡]]的最北面的車站。

2016年11月18日，JR北海道列出數條難以單獨維持營運的路線，札沼線的北海道醫療大學站至新十津川站路段被名列其中。在獲得沿線自治體同意後，JR北海道已向國土交通省申請廢止此路段，預定在2020年5月7日起停駛廢線<ref name="jrhokkaido/press/2016/161118-3">{{Cite press release|url=http://www.jrhokkaido.co.jp/press/2016/161118-3.pdf|format=PDF|language=ja|title=当社単独では維持することが困難な線区について|publisher=北海道旅客鉄道|date=2016-11-18|accessdate=2016-11-18|archiveurl=https://web.archive.org/web/20161118050351/http://www.jrhokkaido.co.jp/press/2016/161118-3.pdf|archivedate=2016年11月18日|deadlinkdate=2017年10月}}</ref>。

== 歷史 ==
* 1981年（昭和56年）12月1日：[[日本國有鐵道|日本國有鐵道]]（國鐵）[[札沼線|札沼線]]大學前臨時乘降場開始營運。當時的大學名稱為東日本學園大學。開始處理旅客業務。
* 1982年（昭和57年）4月1日：升格為'''大學前站'''<ref name=school>{{cite web|url=http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |language=ja |title=札沼線（学園都市線）の電化について |publisher=JR北海道 | date=2009-09-09 |accessdate=2017-08-16 |archive-url=https://web.archive.org/web/20091006234745/http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |archive-date=2009-10-06 }}</ref><ref name=school2>{{cite web |url=http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |language=ja |title=札沼線（学園都市線）の電化開業時期について |publisher=JR北海道 |date=2011-10-03 |accessdate=2017-08-16 |archive-url=https://web.archive.org/web/20111205160350/http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |archive-date=2011-12-05 |deadurl=yes }}</ref>。
* 1987年（昭和62年）4月1日：由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理。
* 1991年（平成3年）3月16日：札沼線的暱稱設定為「'''學園都市線'''」<ref name=school/><ref name=school2/>。
* 1995年（平成7年）3月16日：由於大學進行改名，改稱為'''北海道醫療大學站'''。同時設置折返用月台（現時的2號線）。
* 1996年（平成8年）3月16日：札沼線石狩當別站至新十津川站的列車開始改為[[一人控制|一人控制]]。
* 2000年（平成12年）：札沼線[[桑園車站|桑園車站]]至[[石狩月形車站|石狩月形車站]]引入自動進路制御裝置（PRC）。
* 2007年（平成19年）10月1日：設定車站編號（'''G14'''）<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |language=ja |title=駅番号表示（駅ナンバリング）を実施します |publisher=JR北海道 |date=2007-09-12 |accessdate=2017-08-16 |archive-url=https://web.archive.org/web/20070930015220/http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |archive-date=2007-09-30 |deadurl=yes }}</ref>。
* 2008年（平成20年）10月25日：開始使用[[儲值車票|智能車票]]「[[Kitaca|Kitaca]]」<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |language=ja |title=Kitacaサービス開始日決定について |publisher=JR北海道 |date=2008-09-10 |accessdate=2017-08-16 |archive-url=https://web.archive.org/web/20080913103748/http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |archive-date=2008-09-13 |deadurl=yes }}</ref>。
* 2012年（平成24年）6月1日：札沼線桑園站至本車站正式[[電氣化鐵路|電氣化]]（[[交流電|交流電]]20,000[[伏特|V]]、50[[赫兹|Hz]]）<ref>{{cite web|url=http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |language=ja |title=学園都市線電化開業に伴う電車の投入（第一次）について |publisher=JR北海道 | date=2012-03-14 |accessdate=2017-08-15 |archive-url=https://web.archive.org/web/20120925203437/http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |archive-date=2012-09-25 }}</ref><ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |language=ja |title=平成24年10月ダイヤ改正について |publisher=JR北海道 |date=2012-08-03 |accessdate=2017-08-16 |archive-url=https://web.archive.org/web/20120925203427/http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |archive-date=2012-09-25 |deadurl=yes }}</ref>。
* 2020年（[[令和|令和]]2年）5月7日：北海道醫療大學站至新十津川站路段停駛廢線，本站成為札沼線的終點站<ref name="jrhokkaido/press/2016/161118-3" />。

== 車站構造 ==
現時為側式月台2面2線的地面車站。其中2號線為缺角式月台，主要作為終點站的軌道。前往新十津川方向的列車都會在1號線出發。而大部份以本車站為終點站及折返的列車（由札幌出發）都會於2號線進行，只有少部份因為需要進行交換列車而在1號線進行。此外，1號線的月台能夠容納6輛列車的長度。

無人車站。設置簡單的[[自動售票機|自動售票機]]及[[Kitaca|Kitaca閘機]]。過去曾設置站房但已被移走。Kitaca只能夠於札幌方向使用，無法於新十津川方向使用。此外，亦沒有設置販賣車票及Kitaca的服務。
