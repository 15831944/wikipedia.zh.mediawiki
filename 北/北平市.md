'''北平市'''，通稱'''北平'''（[[漢語拼音|漢語拼音]]：Beiping；[[郵政式拼音|郵政式拼音]]：Peiping），簡稱「'''平'''」，在[[民國紀年|民國]]十七年（1928年）前為[[中華民國首都|中華民國首都]]<ref>{{cite web|title=國家符號：國都的選定|publisher=中華民國國史館|quote=國都必須因應事實需要不斷搬遷：元年3月，遷都北京；16年4月，奠都南京；26年12月，遷都重慶；35年5月，還都南京；38年4月，遷都廣州；38年10月，遷都重慶；38年12月，遷都臺北。|accessdate=2011-01-05|url=http://book.drnh.gov.tw/Content_Display.aspx?MenuKey=46|deadurl=no|archiveurl=https://web.archive.org/web/20120530122901/http://book.drnh.gov.tw/Content_Display.aspx?MenuKey=46|archivedate=2012-05-30}}</ref>，是[[中華民國|中華民國]]在[[兩岸分治|兩岸分治]]前設置的12個[[直轄市_(中華民國)|直轄市]]之一。轄區約同今[[北京市|北京市]]的[[城六區|城六區]]。

== 歷史沿革 ==
{{Main|北京歷史}}

用來指代燕京的「北平」是明朝初年爲貶低[[元大都|元大都]]所創造的名字（'''[[北平府|北平府]]'''）。西晉更名爲北平郡的、[[戰國時代|戰國時代]][[燕國|燕國]]設置[[右北平郡|右北平郡]]，轄區在[[唐山市|唐山地級市]]、[[薊縣|薊縣]]及[[寧城縣|寧城縣]]一帶；北魏時期的[[北平郡|北平郡]]及其郡治[[北平縣_(西漢)|北平縣]]，轄區在[[保定市|保定地級市]]一帶。其餘歷史地名「北平」則根本不在京津冀地區。至永樂還都北京，[[北平府|北平府]]這一名稱僅使用了三十四年零五個月。

[[清朝|清朝]][[宣統|宣統]]三年（1911年）[[辛亥革命|辛亥革命]]後，民國元年（1912年）1月1日，[[中華民國臨時政府_(南京)|中華民國臨時政府]]定都[[金陵|金陵]]，稱[[南京|南京]]。同年3月，[[袁世凱|袁世凱]]將臨時政府遷至北京，成為首都。[[民國|民國]]伊始，北京的地方體制仍依清制，稱[[順天府|順天府]]。直至民國三年（1914年），改順天府為'''[[京兆地方|京兆地方]]'''，直轄於[[北洋政府|北洋政府]]。這一時期，北京新建了[[有軌電車|有軌電車]]系統，和一批現代的文化教育機構，如[[北京大學|北京大學]]、[[燕京大學|燕京大學]]、[[清華大學|清華大學]]、[[輔仁大學|輔仁大學]]、[[北京協和醫學院|協和醫學院]]等。

民國十七年（1928年）[[國民革命軍|國民革命軍]][[北伐|北伐]]，攻佔[[北京|北京]]，[[張作霖|張作霖]]退回[[關外|關外]]。後被日本[[關東軍|關東軍]]謀刺於[[皇姑屯事件|皇姑屯]]，[[東北易幟|東北易幟]]，[[北洋政府|北洋政府]]結束運作。

[[北伐戰爭|北伐戰爭]]後，[[中華民國|中國]]以[[南京|南京]]為[[首都|首都]]，撤銷原京兆地方，北京遂改名為'''北平特別市'''。民國十九年（1930年）6月，北平市降格為[[河北省|河北省]]省轄市，同年12月復升為[[直轄市|直轄市]]。這一時期，北平儘管不具首都的地位，但在[[教育|教育]]方面仍有關鍵的優勢，被國際人士稱為「中國的[[波士頓|波士頓]]」。

民國二十六年（1937年）[[七七事變|七七事變]]後，北平被[[日軍|日軍]]佔領，將北平改名為北京。民國三十四年（1945年）[[大日本帝國|日本]]宣佈向[[同盟國_(第二次世界大戰)|同盟國]]投降，[[中國抗日戰爭|中國抗日戰爭]]結束。同年8月21日第十一戰區[[孫連仲|孫連仲]]部接收北京，並恢復原名'''北平'''。

為與中國共產黨對決，於民國三十六年（1947年）11月在[[蔣中正|蔣中正]]的命令下於北平成立'''華北剿匪總司令部'''。民國三十八年（1949年）1月，[[傅作義|傅作義]]與[[中國共產黨|中國共產黨]]達成和平協定，率[[中華民國國軍|中華民國國軍]]25萬守軍投降，1月31日，[[中國人民解放軍|中國人民解放軍]]和平進入北平市，宣告“[[北平和平解放|北平和平解放]]”。1949年9月27日，北平市改稱'''[[北京市|北京市]]'''，北平市走入歷史。

== 行政區劃 ==
[[File:Beijing_1948_JP.jpg|thumb]]
民國初，北京城內劃分為10區。民國十七年（1928年）12月19日北平市政府下令推行「地方自治」，設立市轄區（自治區）。將全市分為15個區，各區以序號命名。民國三十四年（1945年）抗戰勝利後，國民政府接收後，全市分為16個區，各區以序號命名。民國三十六年（1947年）3月，原郊區的第十三區至第十六區調整分界，由原來4個區改為8個區，即第十三區至第二十區。至民國三十八年（1949年）1月底前不變。

== 管轄範圍 ==
民國時代的北平市轄區範圍，大致相當於今日[[北京|北京]]的城六區略小。

== 人口 ==
民國三十六年（1947年）國民政府主計處統計局公佈的統計資料，北平市人口為167萬2438人<ref>[http://www.xzqh.org/bbs/read.php?tid=1014&fpage=18 民國直轄市、省轄市一覽表（出自《行政區劃論壇》）]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

隨中華民國政府來臺人數，據中華民國內政部戶政司於民國七十九年（1990年）普查外省籍人口按本省籍分的資料，原北平市籍在台人數為1萬3949人，佔外省籍269萬4917人口當中的0.52%。

== 政府體制 ==

=== 市行政機構 ===
{{See also|北京市政府}}

==== 從順天府到京兆地方 ====
[[清朝|清朝]]為[[順天府|順天府]]'''[[順天府衙署|順天府衙署]]'''，其首長稱[[順天府尹|順天府尹]]。民國元年（1912年）[[中華民國|中華民國]]建立，[[順天府|順天府]]仍維持舊制，[[順天府衙署|順天府衙署]]改'''[[順天府尹公署|順天府尹公署]]'''，首長為[[順天府尹|順天府尹]]。民國三年（1914年）10月4日，[[中華民國大總統|中華民國大總統]]袁世凱頒布《[[京兆尹官制|京兆尹官制]]》（法律13號）及《[[京兆地方區域表|京兆地方區域表]]》（教令第133號），改[[順天府|順天府]]為[[京兆地方|京兆]]，順天府尹公署改為'''[[京兆尹公署|京兆尹公署]]'''，首長為[[京兆尹|京兆尹]]。

北京作為新成立的中華民國的首都，市政管理體系仍然承襲清代舊制。民國三年（1914年）4月，內務總長[[朱啟鈐|朱啟鈐]]鑑於京師市政的重要，劃定市區，於順天府下設立'''[[京都市政公所|京都市政公所]]'''，置督辦1人，由內務總長兼任。督辦之下設總務處，綜理全所事務。京都市政公所內部分設文書、登記、捐務等科。民國五年（1916年）9月，京都市政公所改組，裁撤總務處，仍設提調一職。督辦之下分設文書、調查、經理、測繪、工程、交際、出納7科，稍具市政組織規模。民國七年（1918年）1月，根據《京都市政公所暫行編制》，將科擴展為處，將營造局改為第四處。處設處長、副處長各1人。民國十六年（1927年）11月，頒行《修正京都市政公所暫行編制》，仍置督辦、會辦、坐辦，會辦改由京師警察廳總監兼任，以收合作之效；並為聽取市民意見，增設評議會，評議長由督辦兼任。

==== 北平市 ====
民國十七年（1928年）[[國民革命軍北伐|國民革命軍北伐]]攻克京都後，1928年6月21日國民黨[[中央政治會議|中央政治會議]]第145次會議決定，「京都」於6月28日更名為「北平」，並設[[北平特別市|北平特別市]]，直隸國民政府<ref name="bjsdag71">《[[國民政府公報|國民政府公報]]》第71期，民國17年6月，第5頁。</ref>。此前的26日，任命北平特別市市長<ref name="bjsdag70">《國民政府公報》第70期，民國17年6月，第5頁。</ref>。8月21日，'''[[北平特別市政府|北平特別市政府]]'''正式成立。依據《特別市組織法》，設市長1人、參事若干人，下置財政、土地等局。民國十七年（1928年）10月起，河北省政府駐此。民國十九年（1930年）6月，根據新頒佈的《[[市組織法|市組織法]]》，省會所在地必須為[[省轄市|省轄市]]。由此，該市降格為[[河北省|河北省]]省轄市，直隸河北省政府。同年10月，河北省政府遷駐天津市。11月再次復升為[[院轄市|院轄市]]，稱'''[[北平市政府|北平市政府]]'''，直隸於行政院<ref name="bjsdag3">[http://www.da.bj.cn/staticfile/Exhibition/yin/1-1.html 北平市政府] {{webarchive|url=https://web.archive.org/web/20061130001133/http://www.da.bj.cn/staticfile/Exhibition/yin/1-1.html |date=2006-11-30 }}，北京市檔案館，於2011-7-23查閱</ref>。

民國二十六年（1937年）7月，[[日軍|日軍]]佔領北平，隨後改稱「'''北京市'''」。民國三十四年（1945年）8月[[日本|日本]]投降，北平光復，[[南京國民政府|南京國民政府]]重組北平市政府。9月接管北京市，並宣佈恢復該市原名稱「北平市」，仍為[[院轄市|院轄市]]；10月，[[北平市政府|北平市政府]]重新成立，仍直轄於[[行政院|行政院]]。民國三十六年（1947年）1月13日，行政院公佈《北平市政府組織規程》<ref name=bjsdag2726>《國民政府公報》第2726號，民國36年1月17日，第3頁。</ref>，市政府直隸於行政院，設市長1人、參事2人，下置民政、財政等局。民國三十八年（1949年）1月31日，「[[北平和平解放|北平和平解放]]」，由中共[[華北人民政府|華北人民政府]]下屬的[[北平市人民政府|北平市人民政府]]接管。
