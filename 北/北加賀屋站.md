'''北加賀屋站'''（{{jpn|j=北加賀屋駅|hg=きたかがやえき|rm=Kita-Kagaya eki|f=y}}）是一個位於[[日本|日本]][[大阪府|大阪府]][[大阪市|大阪市]][[住之江區|住之江區]]北加賀屋二丁目，屬於[[大阪市高速電氣軌道|大阪市高速電氣軌道]]（大阪地下鐵）[[四橋線|四橋線]]的[[鐵路車站|鐵路車站]]。車站編號是'''Y20'''。

== 歷史 ==
* 1972年（[[昭和|昭和]]47年）11月9日 - 伴隨四橋線玉出－住之江公園間開通而開業。
* 2018年（平成30年）4月1日 - [[大阪市交通局|大阪市交通局]]的民營化，成為[[大阪市高速電氣軌道|大阪市高速電氣軌道]]（大阪地下鐵）的車站。

== 車站結構 ==
單式與島式合成2面3線月台的[[地下車站|地下車站]]（地下2樓）。
