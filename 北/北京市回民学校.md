'''北京市回民学校'''，位于[[北京市|北京市]][[西城区|西城区]][[广安门内大街|广安门内大街]]225号，是专门招收[[回族|回族]]等少数民族学生的普通中学。

==简介==
北京市回民学校的前身是成达师范学校、西北中学、燕山中学。[[清朝|清朝]][[光绪|光绪]]三十三年（1907年）[[牛街礼拜寺|牛街礼拜寺]][[阿訇|阿訇]]王宽创办的“回教师范学堂”；1925年4月24日，在此基础上创办成达师范学校，这是一所[[汉语|汉语]]、[[阿拉伯语|阿拉伯语]]兼授的新型回民师范学校<ref name=sohu/>。1929年成达师范学校迁到北平，[[抗日战争|抗日战争]]时期辗转迁徙，1941年在[[广西|广西]][[桂林|桂林]]归为国立，1946年从[[重庆|重庆]]迁回北平<ref name=jj/>。1928年，北平的清真中学“西北中学”创办，靠募捐筹得经费。1948年，私立初级中学“燕山学校”创办<ref name=sohu>{{Cite web |url = http://news.sohu.com/20050518/n225609571.shtml |title =校园记忆·北京市回民学校  |author =  |date = 2005-05-18 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。

1949年[[北平和平解放|北平和平解放]]后，成达师范学校、西北中学、燕山中学师生对旧教育制度不满，要求人民政府接管。经[[华北人民政府|华北人民政府]]高等教育委员会同意，1949年4月7日正式接管成达师范学校，由[[马玉槐|马玉槐]]任校长，[[杨明德|杨明德]]任副校长，随即对学校进行改造。此时，西北中学、燕山中学仍处在挣扎状态，回民群众强烈要求人民政府将两校改成公立，与成达师范学校合并，成立回民学院。[[中共北京市委|中共北平市委]]统战部部长[[崔月犁|崔月犁]]对此很重视，与马玉槐一同到华北人民政府高等教育委员会反映了回民师生愿望以及中共北平市委的想法，获得批准<ref name=ma>{{Cite web |url = http://dangshi.people.com.cn/GB/144956/10907363.html |title = 回忆解放初期北京的民族工作 |author =马玉槐  |date =2010-02-02  |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。

1949年10月6日，成达师范学校、西北中学、燕山中学合并组建“'''国立回民学院'''”，这是[[中华人民共和国|中华人民共和国]]成立后国家建立的第一所民族院校<ref name=sohu/>。校址位于[[广安门内大街|广安门内大街]]的工艺官局陈列所旧址<ref name=gu>{{Cite web |url = http://www.huaxia.com/zhwh/whxx/2015/05/4413073.html |title =北京早期专业性博物馆：古观象台变身天文陈列馆  |author =  |date = 2015-05-21 |publisher =华夏经纬网  |language =  |accessdate =  |quote =  }}</ref>。1950年，回民学院中学班招收第一个女生班，1953年该班成为优秀班集体，并被命名为“[[卓娅·科斯莫杰米扬斯卡娅|卓娅]]班”<ref name=sohu/>。回民学院成立后，规模不断扩大，面向全国招生，在校生最多时有2400多人。设中学部、师范部、阿拉伯文专修班（后发展为[[中国伊斯兰教经学院|中国伊斯兰教经学院]]），成为回民的最高学府<ref name=ma/>。

1958年起，回民学院的干部、教师陆续调离，校园缩小。另外还抽调部分骨干教师分别参与筹建[[宁夏回族自治区|宁夏回族自治区]]、[[北京宣武红旗业余大学|宣武红旗夜大]]、宣武师范、[[中国伊斯兰教经学院|中国伊斯兰教经学院]]等。回民学院本部在1963年分为'''北京市回民中学'''、'''北京市第六十八中学'''。[[文革|文革]]时期，北京市回民中学更名为'''北京市第一三五中学'''，1979年6月复名'''北京市回民中学'''。1985年，北京市回民中学、北京市第六十八中学重新合并，组建'''北京市回民学校'''，北京市教育委员会将北京市回民学校确定为首批办好的市级重点中学<ref name=jj>{{Cite web |url =http://www.bjhmxx.com/spage.aspx?id=15  |title =北京市回民学校简介 |author =  |date =  |publisher = 北京市回民学校 |language =  |accessdate =  |quote =  }}</ref><ref name=gu/>。1997年被评为北京市[[宣武区|宣武区]]“全面育人，办有特色”的学校<ref>{{Cite web |url =http://www.gmw.cn/01gmrb/1998-10/28/GB/17859%5EGM6-2810.HTM  |title =北京市回民学校硕果累累  |author =  |date = 1998-10-28 |publisher =光明日报  |language =  |accessdate =  |quote =  }}</ref>。2002年，被北京市教育委员会评定为北京市高中示范校。2003年，北京市第一八一中学并入北京市回民学校。北京市回民学校现设有初中、高中<ref name=jj/>。
