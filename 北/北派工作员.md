'''北派工作员'''（朝鲜语：북파공작원）是指[[韓國|韓國]]為对付[[朝鲜|朝鲜]]而成立的[[特工|特工]]人員。

==概要==
北派工作员是韩国政府针对朝鲜成立的特工组织。[[朝鲜战争|朝鲜战争]]中的1952年8月，[[大韓民國國軍|大韓民國國軍]]发动的以抓捕[[金日成|金日成]]为目的而入侵朝鮮[[江原道|江原道]][[元山|元山]]的作战宣告失败<ref>[http://japanese.joins.com/article/899/68899.html?servcode=400&sectcode=420 ＜ピープル＞北派工作員キム・ドンソック氏…「金日成を４分差で逃した」] [[中央日報|中央日報]] 2005年10月24日</ref>。同时，入侵朝鮮过程中，派往北方的女性特工均遭[[拘捕|拘捕]]甚至[[强奸|强奸]]，沦为[[慰安婦|慰安婦]]<ref>{{cite web

韩战结束后，韩国政府继续向北派遣特工进行渗透工作。至1972年，韩国逾13,835人被派赴朝鲜<ref name="libertytimes">[http://www.libertytimes.com.tw/2004/new/feb/18/today-int4.htm 《國際索隱》南韓影片 喚起特遣隊員悲歌] {{webarchive|url=https://web.archive.org/web/20100325233514/http://www.libertytimes.com.tw/2004/new/feb/18/today-int4.htm |date=2010-03-25 }}－[[自由時報|自由時報]]電子新聞網-國際新聞</ref>，其中死亡的7,726人最终以失踪处理<ref>{{cite web

政府通常以高额报酬劝诱候选者参加北派工作，工作員任务成功则对其支付全额报酬，对于机密任务，执行者不得有任何抗议或请求。这些人员一旦加入其中，则终身受到监视，无法从事许多常人能够从事的职业，甚至饱受社会上歧视的眼光。北派工作员属于正规军，工作员必须尽守兵役义务，随时听从军队召集。[[金大中|金大中]]政权以来，原工作员不断发起活动，要求[[国家赔偿|国家赔偿]]<ref>[[青木理|青木理]]『北朝鮮に潜入せよ』</ref>。
