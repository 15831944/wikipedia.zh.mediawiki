'''北海道道699号室兰港线'''（{{lang-ja|北海道道699号室蘭港線}}）是一条位于[[北海道|北海道]][[室蘭市|室蘭市]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：北海道室蘭市祝津町1丁目（[[室蘭港|室蘭港]]）
* 終点：北海道室蘭市山手町1丁目
* 总距离：4.0千米

== 经过的自治体  ==
* [[胆振综合振兴局|胆振综合振兴局]]
** 室蘭市

== 主要连接道路 ==
* [[国道37号|国道37号]][[白鸟新道|白鸟新道]]（起点）
* [[北海道道844号祝津西小路中央线|北海道道844号祝津西小路中央线]]（祝津町2、中央町4）
* [[北海道道919号中央东线|北海道道919号中央东线]]（中央町3）
* [[国道36号|国道36号]][[室蘭新道|室蘭新道]]（终点）
