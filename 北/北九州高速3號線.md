'''北九州高速3號線'''（{{lang-ja|'''北九州高速3号線'''}}，假名：きたきゅうしゅうこうそく3ごうせん）是一條連接[[福岡縣|福岡縣]][[北九州市|北九州市]][[小倉北區|小倉北區]][[愛宕JCT|愛宕JCT]]與同縣同市同區[[東港JCT|東港JCT]]的[[北九州高速道路|北九州高速道路]]路線。

== 連接高速道路 ==
* [[北九州高速1號線|北九州高速1號線]]（於[[愛宕JCT|愛宕JCT]]連接）
* [[北九州高速2號線|北九州高速2號線]]（於[[東港JCT|東港JCT]]連接）

== 設施 ==
* 全線都在[[福岡縣|福岡縣]][[北九州市|北九州市]]內。
* 如出入口的連接路線中沒有標記，即是連接市道。
