'''北谷灣線'''（{{jpn|j=北リアス線|hg=きたリアスせん|rm=Kita-riasu sen|f=y}}）是一條連結[[岩手縣|岩手縣]][[宮古市|宮古市]]的[[宮古站|宮古站]]與[[久慈市|久慈市]]的[[久慈站|久慈站]]，屬於[[三陸鐵道|三陸鐵道]]的鐵路線。

沿線是[[三陸海岸|三陸海岸]]連續的{{link-ja|海岸段丘|海岸段丘}}，沿線的[[溺灣|谷灣式海岸]]與[[南谷灣線|南谷灣線]]的風景互相對照。

== 路線資料 ==
* 管轄（事業類別）：三陸鐵道（[[鐵路公司|第一種鐵道事業者]]）
* 路段（{{link-ja|營業距離|営業キロ}}）：宮古－久慈 71.0公里
* [[軌距|軌距]]：1067毫米
* 站數：17個（包括起終點站）
* 複線路段：沒有（全線[[單線鐵路|單線]]）
* 電氣化路段：沒有（全線{{link-ja|非電氣化|非電化}}）
* [[閉塞_(鐵路)|閉塞方式]]：特殊自動閉塞式（軌道回路檢知式）
* [[車輛基地|車輛基地]]所在站：[[久慈站|久慈站]]

== 運行形態 ==
在2011年3月11日發生[[東日本大地震|東日本大地震]]後，全線一度不能通行，同年3月中，[[宮古站|宮古站]]至[[小本站|小本站]]、[[陸中野田站|陸中野田站]]至[[久慈站|久慈站]]路段重新通車，2012年4月1日，[[田野畑站|田野畑站]]至陸中野田站亦重新通車，全線已於2014年4月6日起完全復業。<ref>{{cite web |url=http://www3.nhk.or.jp/news/html/20140406/t10013535221000.html |title=存档副本 |accessdate=2014-04-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20140407103702/http://www3.nhk.or.jp/news/html/20140406/t10013535221000.html |archivedate=2014-04-07 }}</ref>

2014年全線恢復前，路線分成兩個區間各自運輸，北面久慈站至田野畑站每天開出10班上行、9班下行，另外1個短程來回[[陸中野田站|陸中野田站]]；南面宮古站至[[小本站|小本站]]則上下行各開出7班；至於中斷路段，則每天來回開出7班接駁巴士，假日時尾班接駁巴士會停開。<ref>[http://www.sanrikutetsudou.com/service_info/jikoku/H240401_jikoku_shinsai.pdf 2012年4月1日起北谷灣線及接駁巴士時間表。（日文）] {{webarchive|url=https://web.archive.org/web/20120710231109/http://www.sanrikutetsudou.com/service_info/jikoku/H240401_jikoku_shinsai.pdf |date=2012年7月10日 }}</ref>

至於在東日本大地震前，大部分列車都是行走全線，各站停車的普通列車，另外每日有1班來回班次會延長至[[山田線|山田線]]、[[南谷灣線|南谷灣線]]直通至[[盛站|盛站]]。另外因應不同的季節，會開設臨時列車，例如復古座敷車輛「三陸海風」、和[[JR東日本|JR東日本]]共同營運的「RIAS、Sea Liner」與「[[北山崎|北山崎]]三陸火車」。

== 歷史 ==
北谷灣線最初的構思始於[[鐵道敷設法|改正鐵道敷設法]]別表第6號規定的預定線－「岩手縣久慈經小本至宮古鐵道」。其後在1972年，宮古至田老之間的'''宮古線'''開通；隨後由普代至久慈之間的'''久慈線'''也於1975年開業。1980年日本國會通過[[日本國有鐵道經營再建促進特別措置法|國鐵再建法]]，要求廢止一些客量不足的鐵路線，並且停止一切興建中的路段。久慈線和宮古線被納入要廢止的名單中，而建設中的[[田老站|田老站]]至[[普代站|普代站]]路段間工程亦被中斷。後來經以岩手縣政府為首，成立[[日本第三部門鐵道|第三部門鐵道]][[三陸鐵道|三陸鐵道]]，要求接手兩線的營運權及其繼續伸延工程，獲得了批准。工程因而得以繼續，並在1984年4月1日完成貫通工程後，一次過將久慈線和宮古線轉承，並改名為'''北谷灣線'''。

JR東日本曾經計畫把[[岩泉線|岩泉線]]與小本站連接，並擬改稱為小本線，但由於已經有由[[岩泉自動車運輸|岩泉自動車運輸]]與[[岩手縣北自動車|岩手縣北自動車]]營運的接岩泉與小本，所以便決定取消此計畫。

=== 宮古線 ===
*1972年2月27日：'''宮古線'''宮古至田老之間 (12.8km) 開業，開始客運服務。新設一之渡、佐羽根、田老站。
*1981年9月18日：被指定為第1次特定地方交通線，列入停運路線列表中。
*1984年4月1日：由三陸鐵道繼承此線。

=== 久慈線 ===
*1975年7月20日：'''久慈線'''久慈至普代之間 (26.0km) 開業，開始客運服務。新設陸中宇部、陸中野田、野田玉川、堀内、普代站。
*1981年9月18日：被指定為第1次特定地方交通線，列入停運路線列表中。
*1984年4月1日：由三陸鐵道繼承此線。

=== 北谷灣線 ===
*1984年4月1日：三陸鐵道北谷灣線宮古至久慈之間（71.0km）開業。其中田老至普代之間（32.2km）為新建成區間，其餘則由國鐵繼承而來。新設攝待、小本、島越、田野畑站。
*1984年12月22日：新設白井海岸站。
*1986年7月26日：於野田玉川與陸中野田之間新設十府浦臨時站，在1994年（平成6年）夏季後停用<ref>{{cite web |url=http://www.nnet.ne.jp/~kunohe/noda/1108/N1108a.html |title=存档副本 |accessdate=2012-07-01 |0=2011-07-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20110722114953/http://www.nnet.ne.jp/~kunohe/noda/1108/N1108a.html |archivedate=2011-07-22 }} </ref>
*2010年10月16日：新設山口團地站<ref name="yamaguchidanchi">[http://www.sanrikutetsudou.com/2010/08/%e5%b1%b1%e5%8f%a3%e5%9b%a3%e5%9c%b0%e9%a7%85%e3%81%ae%e6%84%9b%e7%a7%b0%e5%8b%9f%e9%9b%86.html 三陸鐵道新站命名召募] {{webarchive|url=https://web.archive.org/web/20120315170206/http://www.sanrikutetsudou.com/2010/08/%e5%b1%b1%e5%8f%a3%e5%9b%a3%e5%9c%b0%e9%a7%85%e3%81%ae%e6%84%9b%e7%a7%b0%e5%8b%9f%e9%9b%86.html |date=2012-03-15 }}</ref><ref>[http://www.sanrikutetsudou.com/2010/09/%e5%b1%b1%e5%8f%a3%e5%9b%a3%e5%9c%b0%e9%a7%85%e6%84%9b%e7%a7%b0%e6%b1%ba%e5%ae%9a%ef%bc%81%ef%bc%81.html 決定稱為山口團地站] {{webarchive|url=https://web.archive.org/web/20110716004726/http://www.sanrikutetsudou.com/2010/09/%E5%B1%B1%E5%8F%A3%E5%9B%A3%E5%9C%B0%E9%A7%85%E6%84%9B%E7%A7%B0%E6%B1%BA%E5%AE%9A%EF%BC%81%EF%BC%81.html |date=2011-07-16 }}</ref>。
*2011年3月11日：發生[[東日本大地震|東日本大地震]]，全線停運，海嘯令島越站及其架空橋樑沖毀。
*2011年3月16日：陸中野田至久慈之間重新營運，並設立復興支援列車，直至3月31日前免費乘坐。
*2011年3月20日：宮古至田老之間再次營運。
*2011年3月29日：田老至小本之間再次營運。
*2012年4月1日：田野畑至陸中野田之間再次營運<ref>[https://archive.is/20120711173651/mainichi.jp/norimai/train/trainflash/news/20120401mog00m040002000c.html 三陸鐵道：陸中野田至田野畑重新營運] - 每日新聞、2012年4月1日</ref>。
*2014年4月6日：小本至田野畑之間再次營運<ref>[http://www.tohkaishimpo.com/scripts/index_main.cgi?mode=kiji_zoom&cd=nws7261 三陸鐵道南谷灣線　預計26年4月全線再通] {{webarchive|url=https://web.archive.org/web/20120104080617/http://www.tohkaishimpo.com/scripts/index_main.cgi?mode=kiji_zoom&cd=nws7261 |date=2012-01-04 }} - 東海新報、2012年1月1日</ref>，至此全線恢復。
*2018年3月28日，三陸鐵道召開董事會，會中宣佈JR東日本移轉的宮古釜石間山田線將於2019年3月23日起恢復通車營運，同時會與現有的北谷灣線與南谷灣線整合成「[[谷灣線|谷灣線]]」（{{lang|ja|リアス線}}），總計全長163公里，一舉成為[[日本第三部門鐵道|日本第三部門鐵道]]中里程最長的路線。<ref>[https://www3.nhk.or.jp/lnews/morioka/20180328/6040000473.html 三鉄に移管区間の運行開始日は] {{Wayback|url=https://www3.nhk.or.jp/lnews/morioka/20180328/6040000473.html |date=20180330211358 }}</ref>

為了活化鐵路公司的財務狀況，三陸鐵道曾在2009年時嘗試以出租旗下車站的暱稱[[命名|命名]]權爭取廠商的廣告贊助，公開招募暱稱的對象包括了北谷灣線上的久慈、宮古與南谷灣線上的釜石、盛等四個車站。因此，在同年4月27日之後的1年間宮古站曾被名為「Eyeful Home宮谷」（{{lang|ja|アイフルホーム宮古}}{{NoteTag|「EyefulHome」是總公司位在[[盛岡市|盛岡市]]的天狼星住宅建設公司（{{lang|ja|株式会社シリウス}}）旗下所擁有之住宅建案品牌。}}）{{R|IwateNP_20090424}}，同年6月1日之後的一年間久慈站也曾被名為「[[琥珀|琥珀]]王國久慈」{{NoteTag|「琥珀王國」是由久慈市觀光物產協會配合當地的兩家琥珀工藝品廠商所共同出資贊助的命名。}}{{R|IwateNP_20090619}}。

== 文化作品中的曝光 ==
日本[[NHK|NHK]]電視台在2013年4月至9月期間播放的晨間[[連續電視小說|連續電視小說]]系列作品《[[小海女|小海女]]》（{{lang|ja|あまちゃん}}），是一齣以位在岩手縣的虛構城市、以久慈市為藍本的「北三陸市」為故事場景的作品。在該劇中作為故事主軸之一的虛構地方第三部門鐵路業者、常簡稱「北鐵」的北三陸鐵道就是以三陸鐵道作為藍本，並以北谷灣線（在劇中易名為「谷灣線」）及沿線各車站作為場景，包括易名為「北三陸車站」的久慈，易名為「袖濱車站」（{{lang|ja|袖が浜駅}}）的堀內，與譯名為「畑野車站」的田野畑。除了車站與沿線設施外，劇中登場的鐵路列車也是直接由三陸鐵道的車輛演出。由於電視劇的播出，三陸鐵道與北谷灣線的知名度大增，配合大量的觀光人潮三陸鐵道也順勢推出相關的旅遊行程，尋訪電視劇的拍攝景點{{R|Sanriku_tour}}。
