'''北京市西城区洁民幼儿园'''，位于[[北京市|北京市]][[西城区|西城区]][[马甸|马甸]]裕中西里小区36号楼，是隶属北京市西城区教育委员会的[[事业单位|事业单位]]。

==简介==
光绪二十九年（1903年）秋，湖北幼稚园在[[武昌|武昌]]成立，是中国第一个官办学前教育机构。同年，[[章宗祥|章宗祥]]、[[章继诗|章继诗]]、[[陈彦庵|陈彦庵]]（章宗祥夫人）在北京西城创建了北京历史上首座蒙养院——'''京师第一蒙养院'''<ref>{{Cite web|url =http://szb.bjedu.cn/xlxz/2013-03-15/10696.html|title =从毕业证书看北京早期女子教育|author =毕树政|date =2013-03-15|publisher =京教纵横|language =|accessdate =|quote =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name=gk/><ref name=ifeng>{{Cite web|url = http://edu.ifeng.com/gundong/detail_2013_04/30/24822066_0.shtml|title = 北京第一所国人创办的幼儿园|author = |date = 2013-04-30|publisher = 凤凰网|language = |accessdate = |quote = }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[清朝|清朝]]光绪二十九年十一月二十六日（1904年1月13日）呈报并着实行的《[[奏定学堂章程|奏定学堂章程]]》中包括有《奏定蒙养院章程及家庭教育法章程》，规定设蒙养院，收3至7岁幼儿。1907年2月[[学部_(清朝)|学部]]颁布《奏定女子小学堂章程》后，京师第一蒙养院附设女子两等小学堂成立，是北京最早设立的女子小学堂<ref name=gk/><ref name=ifeng/>。

1922年，[[北洋政府|北洋政府]]教育部颁布《学校系统改革令》，“京师第一蒙养院”更名为“'''洁民幼稚园'''”，后来又更名为“'''洁民幼儿园'''”，以秉持“育人以著其洁”的教育观<ref name=gk/><ref name=ifeng/>。

1949年[[中华人民共和国|中华人民共和国]]成立后，洁民幼儿园响应[[新民主主义|新民主主义]]教育方针，优先录取工人、军人、干部、烈属、职业妇女的子女。教育内容增加了“爱祖国、爱人民、爱科学、爱劳动、爱护公共财物”的“五爱教育”，推行教养并重、全面发展的教育方针<ref name=gk/>。 

1964年，洁民幼儿园被北京市西城区教育局接收，由私立改为公立幼儿园。1970年代末至1990年代初，洁民幼儿园开办四个教学班，共18名教职工。1995年，洁民幼儿园由[[南榆钱胡同|南榆钱胡同]]迁址裕中西里小区，1995年11月正式开班。洁民幼儿园成为第一批早教基地，1998年又被评为一级一类幼儿园。1997年洁民幼儿园提出“自然化教育”理念，并于同年10月18日向社区开放。2004年获评为北京市西城区示范幼儿园，2005年获评为北京市市级示范幼儿园。幼儿园逐步形成了“以人为本，以自然为本”的教育特色。2011年，洁民幼儿园园标诞生。2013年11月29日至12月6日，洁民幼儿园举办了建园110周年庆典活动<ref name=gk>{{Cite web |url =http://www.bjjmyey.org/page4.aspx?id=30  |title =园所概况  |author =  |date =  |publisher =  北京市西城区洁民幼儿园|language =  |accessdate =  2017-06-26|quote =  }}</ref><ref name=ifeng/>。
