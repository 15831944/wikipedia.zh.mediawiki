'''北欧神话'''是[[斯堪的纳维亚地区|斯堪的纳维亚地区]]所特有的一个[[神话|神话]]体系，其形成时间相对的较晚于世界上其他几大[[神话|神话]]体系。[[地理學|地理學]]上的[[北歐|北歐]]，包括今日的[[瑞典|瑞典]]、[[挪威|挪威]]、[[丹麥|丹麥]]、[[冰島|冰島]]和[[芬蘭|芬蘭]]等國，但一般所稱的北歐神話並不包括芬蘭，因為芬蘭语并不属于斯堪的纳维亚语言，芬兰人有自己的神話，见[[芬兰神话|芬兰神话]]。

北欧神话中有许多部分十分特别，大部分的神話都會描寫創世的榮光，但北歐神話卻著力描述于毀滅。神話中的所有事物，包括神都不拥有永恒。因为他们的想法接近于事实，所以对他们而言，所有东西都是不完美的，這是北歐神話和世界其他神話最不一樣的地方。而另一方面，北欧神话相信当万物消亡，新的生命将再次形成，世界上的一切都是循环的。

== 歷史 ==
北欧神话的口头传播[[历史|历史]]可追溯到1世纪－2世纪。北歐神話中[[英雄|英雄]]的後裔，即今[[斯堪的那維亞半島|斯堪的那維亞半島]]及[[德意志地區|德意志]]東北低地的[[日耳曼民族|日耳曼民族]]。西元[[五世紀|五世紀]]左右，[[日耳曼人|日耳曼人]]大致分佈在[[萊茵河|萊茵河]]、[[多瑙河|多瑙河]]地帶，與[[羅馬帝國|羅馬帝國]]相鄰。

7世纪左右，随着一批北上的[[移民|移民]]移至[[冰岛|冰岛]]等处，日耳曼人的神話在[[冰島|冰島]]地區反而保存下來了。至今保留的较为完整的有[[冰岛史诗|冰岛史诗]]《[[埃达|埃达]]》以及[[日耳曼史诗|日耳曼史诗]]《[[尼伯龙根之歌|尼伯龙根之歌]]》。

== 世界觀 ==
=== 神族與奇幻生物 ===
北欧神话是个多神系统，大致上可分成5个体系：[[巨人|巨人]]（{{lang|en|Giants}}）、[[阿萨神族|阿萨神族]]（{{lang|en|Aesir}}）、[[华纳神族|华纳神族]]（{{lang|en|Vanir}}）、[[精灵|精灵]]（{{lang|en|Elves}}）以及[[侏儒_(北歐神話)|侏儒]]（{{lang|en|Dwarves}}）。

巨人是最早的生命，生出了諸神，但同时也是众神最大的敌人，可以将之理解为人格化的自然力量。神分为两个部族，以主神[[奥丁|奥丁]]（{{lang|en|Odin}}）为首领的阿萨神族以及以大海之神[[尼約德|尼約德]]（{{lang|en|Njord}}）为首领的华纳神族，其中主要神有十二个。[[精灵|精灵]]及[[侏儒|侏儒]]属于[[半神|半神]]，他们为神服务，属于[[日耳曼|日耳曼]]地区的特殊创造。

=== 創世神話 ===
世界初開之際，沒有天也沒有地，在濃霧中，只有一道[[金倫加|金倫加]]（{{lang|en|Ginunaga}}）鴻溝，鴻溝之北為「冰之國」[[尼福爾海姆|尼福爾海姆]]（Niflheim），其中有泉名[[赫瓦格密爾|赫瓦格密爾]]（{{lang|en|Hvergelmir}}），是所有河水的源頭，據說其中一條河的水帶有劇毒。鴻溝之南是「火之國」[[穆斯貝爾海姆|穆斯貝爾海姆]]（Muspelheim）。在熱氣與寒冰的交錯中，誕生了[[霜巨人|霜巨人]]之祖[[尤彌爾|尤彌爾]]（{{lang|en|Ymir}}）和一頭名為[[歐德姆布拉|歐德姆布拉]]（{{lang|en|Audhumbla}}）的巨大母牛。尤彌爾吃著歐德姆布拉分泌的奶水維生，而歐德姆布拉則舔食寒冰上的鹽粒。

之後，[[尤彌爾|尤彌爾]]的身上變出了智慧巨人[[密米爾|密米爾]]（{{lang|en|Mimir}}）和女巨人[[貝斯特拉|貝斯特拉]]（{{lang|en|Bestla}}），這是霜巨人的家族。另外，在母牛的舔舐下，冰中則出現諸神之祖[[布利|布利]]（{{lang|en|Buri}}）。布利生下[[包爾|包爾]]（{{lang|en|Borr}}），包爾和女巨人[[貝斯特拉|貝斯特拉]]（Bestla）生下了[[奧丁|奧丁]]（{{lang|en|Odin}}）、[[威利|威利]]（{{lang|en|Vili}}）、[[菲_(北歐神)|菲]]（{{lang|en|Ve}}）。巨人族和諸神相當不合，雙方挑起戰火。後來尤彌爾被諸神所殺，巨人族戰敗，剩下的巨人逃往[[約頓海姆|約頓海姆]]（{{lang|en|Jothuheim}}），並繼續對諸神抱持著恨意。

諸神拿尤彌爾的身體來佈置出大地和天空。尤彌爾的[[頭顱|頭顱]]化為[[天空|天空]]，腦髓為[[雲|雲]]，[[身體|身體]]成了大地，[[血液|血液]]成為[[海洋|海洋]]，[[骨骼|骨骼]]變成[[山脈|山脈]]，[[毛髮|毛髮]]變成[[樹木|樹木]]。當尤彌爾屍身腐爛長出[[蛆|蛆]]，這些蛆就變成了[[精灵|精灵]]及[[侏儒_(北歐神話)|侏儒]]。[[奧丁|奧丁]]命四個具有怪力的侏儒支撐著天空的四角。尤彌爾的眉毛則被用來造成牆壁用來圍住「中間世界/中庭」（{{lang|en|Midgard}}，米德加爾特）。然後奧丁又捕捉[[穆斯貝爾海姆|穆斯貝爾海姆]]的火焰，將他們化為[[星星|星星]]、[[月亮|月亮]]、[[太陽|太陽]]，並訂定了[[四季|四季]]的運行。諸神又取來[[梣樹|梣木]]枝造成[[男人|男人]]，用[[榆樹|榆樹]]枝造了[[女人|女人]]。男的取名「阿斯克」（{{lang|en|Ask}}，即「[[梣樹|梣樹]]」），女的叫做「恩布拉」（{{lang|en|Embla}}，即「榆樹」）。奧丁給他們生命和靈魂，[[威利|威利]]給他們理性與動作，[[菲_(北歐神)|菲]]給他們感情、儀表和語言。這就是人類的始祖。

=== 末世神話 ===
北歐神話的一大特色就是世界最終是會毀滅的，這就是[[諸神的黃昏|諸神的黃昏]]。這場戰役無可避免，也註定眾神必定失敗，但即使如此，北歐的眾神依然坦率的面對這最終的結局。而在世界滅亡之後，殘存的神會再次建立起新的世界。

== 九個世界 ==
[[世界之樹|世界之樹]]（{{lang|en|Yggdrasil}}）支撐的世界分為三層九個國度，以下列出大致的分類，但事實上由於正確的資料已經失傳，所以對於九個世界的說法不只一種。
* [[阿斯嘉特|阿斯嘉特]]（{{lang|en|Asgard}}）：神國。[[阿薩神族|阿薩神族]]居住的地方。其中最大的城堡就是奧丁的金宮（{{lang|en|Gladsheim}}）。[[英靈殿|英靈殿]]（{{lang|en|Valhalla}}）也是位在此。
* [[華納海姆|華納海姆]]（{{lang|en|Vanaheim}}）：[[華納神族|華納神族]]居住的地方。天地之間萬物的生養繁殖、海洋和風，都歸華納神族控制。此神族通曉許多連奧丁都不知道的神祕咒法。
* [[亞爾夫海姆|亞爾夫海姆]]（{{lang|en|Alfheim}}）：[[精靈|精靈]]之國。精靈居住的地方。精靈生得特別美麗，擁有很大的法力。他們照料花草，是最愛光亮，是善良親切的小妖精。
* [[米德加爾特|米德加爾特]]（{{lang|en|Midgard}}）：人類居住的地方。這裡有由冰、火、空氣構成的三色[[彩虹橋_(北歐神話)|彩虹橋]]（{{lang|en|Bifrost}}），可以通往神國。
* [[約頓海姆|約頓海姆]]（{{lang|en|Jothuheim}}）：[[巨人|巨人]]居住的地方。人類世界到「巨人國度」的路標是一個恐怖的「鐵森林」（{{lang|en|Jarnvid}}）；也有傳說海的那一邊就是「巨人國度」。
* [[斯瓦塔爾法海姆|斯瓦塔爾法海姆]]（{{lang|en|Svartalfheim}}）：[[侏儒|侏儒]]居住的地方。侏儒是厲害的巧匠，擁有種種神祕的力量和深遂的知識，他們打造出很多寶物。
* [[赫爾海姆|赫爾海姆]]（{{lang|en|Helheim}}）：和冥國女王[[赫尔_(北欧神话)|赫爾]]（{{lang|en|Hel}}）同名的國度，也翻作「地獄」。這是一個冰冷多霧的地方，也是個永夜的場所，只有亡者才能到達。
* [[尼福爾海姆|尼福爾海姆]]（{{lang|en|Niflheim}}）：冰之國。和死亡國沒有明顯分別的冰雪世界。
* [[穆斯貝爾海姆|穆斯貝爾海姆]]（{{lang|en|Muspelheim}}）：火之國。金倫加鴻溝之南，由巨人史爾特爾守護的酷熱國度。

== 主要神 ==
* [[奧丁|奧丁]]（{{lang|en|Odin}}）：[[阿薩神族|阿薩神族]]（{{lang|en|Aesir}}）的主神；掌管戰爭、死亡、智慧、詩歌等。
* [[弗麗嘉|弗麗嘉]]（{{lang|en|Frigga}}）：-{zh-hans:神后; zh-hant:神后;}-，奧丁的妻子；掌管婚姻和家庭；負責紡織雲彩。
* [[索爾|索爾]]（{{lang|en|Thor}}）：奧丁的兒子，擁有雷神的稱號；負責掌管戰爭與農業，為巨人的主要敵人。
* [[希芙|希芙]]（{{lang|en|Sif}}）：索爾的妻子，土地和收穫女神。
* [[洛基|洛基]]（{{lang|en|Loki}}）：淘氣和惡作劇之神；屬巨人族；奧丁的結拜兄弟。據說是名才華洋溢的美男子。北歐神話中沒有太明顯的善惡定位。
* [[巴德爾|巴德爾]]（{{lang|en|Baldur}}）：光明之神；奧丁和弗麗嘉之子。
* [[霍德爾|霍德爾]]（{{lang|en|Hodr}}）：黑暗之神；巴德爾的盲眼[[孿生|孿生]]兄弟。
* [[凡賽堤|凡賽堤]]（{{lang|en|Forseti}}）：真理與正義之神；巴德爾的兒子。
* [[布拉基|布拉基]]（{{lang|en|Bragi}}）：詩詞、智慧、雄辯之神；奧丁的兒子。
* [[伊登|伊登]]（{{lang|en|Iðunn}}）：青春女神；布拉基的妻子；[[阿斯嘉特|阿斯嘉特]]萬年花園的主人，掌管恢復青春的蘋果。
* [[尼約德|尼約德]]（{{lang|en|Njord}}）：海洋之神，华纳神族的领袖，后来归顺了阿萨神族。
* [[弗雷|弗雷]]（{{lang|en|Freyr}}）：豐饒之神；主司收成和天氣；海洋之神[[尼約德|尼約德]]（{{lang|en|Njord}}）之子，他出身於[[華納神族|華納神族]]，和弗蕾亞一起作為人質交換來到阿薩族。
* [[弗蕾亞|弗蕾亞]]（{{lang|en|Freyja}}）：繁育及戰爭女神；弗雷的妹妹；她和奧丁平分戰爭中的死者。
* [[提爾|提爾]]（{{lang|en|Tyr}}）：獨臂戰神；被[[芬里爾|芬里爾]]狼（{{lang|en|Fenrir}}）咬斷了一隻手。
* [[海姆達爾|海姆達爾]]（{{lang|en|Heimdall}}）：[[彩虹橋|彩虹橋]]（{{lang|en|Bifrost}}）的看守者；[[諸神的黃昏|諸神的黃昏]]時會吹響他的號角。

== 與星期的關係 ==
北歐神話與[[英語|英語]]中[[星期|星期]]的名稱有很大的關聯。

* 星期日（{{lang|en|Sunday}}）：巨人蒙迪爾法利（Mundilfari）之女[[蘇爾_(北歐神祇)|蘇爾 (北歐神祇)]]（{{lang|en|Sol}}）之日。是古[[日耳曼|日耳曼]]（{{lang|en|Teutonic peoples}}）民族祭祀太陽的日子。
* 星期一（{{lang|en|Monday}}）：巨人蒙迪爾法利（Mundilfari）之子[[瑪尼|瑪尼]]（{{lang|en|Mani}}）。是[[盎格魯-薩克遜|盎格魯-薩克遜]]（{{lang|en|Anglo-Saxon}}，日耳曼人一支）人的月亮之日。
* 星期二（{{lang|en|Tuesday}}）：是以戰神[[提爾|提爾]]（{{lang|en|Tiu}}或{{lang|en|Tiw}}，北歐戰神Tyr的古英語名字）之名而定。
* 星期三（{{lang|en|Wednesday}}）：這個名字來自古代祭祀主神[[奧丁|奧丁]]的日子（{{lang|en|Woden's day}}，{{lang|en|Woden}}是古英語對奧丁的稱呼）。
* 星期四（{{lang|en|Thursday}}）：取自北歐神話中的雷神[[索爾|索爾]]（{{lang|en|Thor}}）。這是古[[日耳曼|日耳曼]]人一星期中最神聖的一天，會議通常在這天舉行，且議員中午前未出現，就會被取消資格，所以[[索爾|索爾]]也是會議的守護神。
* 星期五（{{lang|en|Friday }}）：來自古英語（{{lang|en|Frigedaeg}}），指的是女神[[弗蕾亚|弗蕾雅]]（{{lang|en|Freyja}}）之日，或說是奧丁之妻[[弗丽嘉|弗丽嘉]]（{{lang|en|Frigga}}）。不過也有人認為兩位女神有可能是同一人。
* 星期六（{{lang|en|Saturday}}）：取自[[羅馬神話|羅馬神話]]的[[農神|農神]]──[[萨图尔努斯|萨图尔努斯]]。這是唯一和北歐神話無關的。

== 北歐神話解密 ==
* 北歐神話的流傳：北歐神話最早流傳在[[芬蘭|芬蘭]]、[[瑞典|瑞典]]、[[挪威|挪威]]及[[冰島|冰島]]一帶，甚至在北美及[[格陵蘭島|格陵蘭島]]，是[[日耳曼|日耳曼]]民族之一的[[斯堪的那維亞|斯堪的那維亞]]民族（[[維京人|維京人]]的祖先）所創。八世紀到十一世紀，維京人威震全歐洲，是歐洲海上最可怕的海盜，這也是北歐神話的全盛期。十三世紀以後，維京人式微，歐洲教會的勢力大增，北歐神話開始失傳，維京人也開始改信基督教、天主教，只剩冰天雪地的[[冰島|冰島]]以詩歌及散文的型式，流傳[[北歐吟唱詩人|北歐吟唱詩人]]（{{lang|en|Skald}}）之間。
* [[諸神的黃昏|諸神的黃昏]]：在諸神黃昏最後，指出世界將被熊熊火焰所滅，而在熊熊火焰消退之後將誕生出新的世界。其實這裡的火焰有可能指的就是「火山爆發」，北歐由於地殼不穩，因此地震與火山活動頻繁。在古代北歐人看來，火山爆發宛如世界末日一般，但火山爆發之下誕生出的新大陸又充滿無限生機。因此將恐怖毀滅但又能誕生生命的自然現象給套用到神話中。
