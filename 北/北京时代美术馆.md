'''北京时代美术馆'''，位于[[北京市|北京市]][[海淀区|海淀区]][[复兴路_(北京)|复兴路]]69号[[华熙LIVE·五棵松|华熙LIVE·五棵松]]。

==简介==
北京时代美术馆成立于2008年5月，是民间非营利性美术馆，由华熙国际投资集团有限公司投资创办。起初位于北京市[[朝阳区|朝阳区]][[建国门外大街|建国门外大街]]6号中环世贸中心D座35-37层<ref>{{Cite web |url = http://shuhua.gmw.cn/2014-11/04/content_13758563.htm |title = 北京时代美术馆（一） |author =  |date = 2014-11-04 |publisher = 光明网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://finance.ifeng.com/collection/ysjg/20120314/5748849.shtml|title = 北京时代美术馆简介 |author =  |date =  2012-03-14|publisher =  凤凰网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url = http://www.bloomage.com/industry/cultural|title = 文化体育|author = |date = |publisher = 华熙国际投资集团有限公司|language = |accessdate = 2017-05-17|quote = |archive-url = https://web.archive.org/web/20170703163614/http://www.bloomage.com/industry/cultural|archive-date = 2017-07-03|dead-url = yes}}</ref>。首任馆长是王艺。首展为《热血5月·2008》，邀请中国国内写实派画家为“[[汶川大地震|汶川大地震]]”集体创作了一幅长卷，拍卖所得全部3600多万元人民币善款用于震区灾后文化重建。除展览外，该馆自2013年起还引入[[古琴|古琴]]、[[现代舞|现代舞]]、禅乐、戏剧等<ref name=liu>{{Cite web |url = http://art.china.cn/talk/2015-09/01/content_8203507.htm |title =刘燕：跨视野——助力艺术生态 共建学术平台  |author =  |date =2015-09-01  |publisher = 中国网 |language =  |accessdate =  |quote =  }}</ref>。

2016年，北京时代美术馆迁至五棵松新馆（属于五棵松文化体育中心二期，位于华熙LIVE·HI-UP东北角）。2016年10月29日，由[[中国美术家协会|中国美术家协会]]、[[中国国家画院|中国国家画院]]主办、华熙国际投资集团有限公司协办，北京时代美术馆承办的“王艺2016研究展”在北京时代美术馆开幕，这也是新馆举办的首个展览<ref>{{Cite web |url =  http://cul.qq.com/a/20161031/036323.htm?winzoom=1|title = 王艺2016研究展暨北京时代美术馆五棵松新馆开幕展隆重开幕 |author =  |date = 2016-10-31 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref><ref name=liu/>。
