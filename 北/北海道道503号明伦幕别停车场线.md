'''北海道道503号明伦幕别停车场线'''（{{lang-ja|北海道道503号明倫幕別停車場線}}）是一条位于[[北海道|北海道]][[十胜综合振兴局|十胜综合振兴局]]管内[[幕別町|幕別町]]内的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：北海道[[中川郡_(十勝国)|中川郡]][[幕別町|幕別町]]明倫（＝[[北海道道62号丰顷糠内芽室线|北海道道62号丰顷糠内芽室线]]交点）
* 終点：北海道中川郡幕別町本町（＝[[北海道旅客铁道|JR北海道]][[幕别车站|幕别车站]]）
* 总長：19.4千米

== 共线区间 ==
* 幕別町寿町 - 幕別町本町（[[北海道道15号幕别大树线|北海道道15号幕别大树线]]）
* 幕別町本町（[[北海道道260号幕别停车场线|北海道道260号幕别停车场线]]）

== 经过的自治体  ==

** 中川郡幕別町
