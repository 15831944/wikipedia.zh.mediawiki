'''北海道道327号辨景幌别线'''（{{jpn|j=北海道道327号弁景幌別線}}）是一条连结[[北海道|北海道]][[壮瞥町|壮瞥町]]和[[登别市|登别市]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 指定区間 ==
* 起点：北海道[[有珠郡|有珠郡]]壮瞥町字黄溪（＝[[北海道道2号洞爷湖登别线|北海道道2号洞爷湖登别线]]交点）
* 終点：北海道登別市幸町2丁目（＝[[国道36号|国道36号]]交点）
* 总長：16.8千米

== 共线区间 ==
* 登別市中央町4丁目-登別市幌別町5丁目（[[北海道道387号幌別停车场线|北海道道387号幌別停车场线]]）

== 未開通区間 ==
* 壮瞥町字黄渓-登別市矿山町（＝矿山橋附近）

== 经过的自治体  ==
* [[胆振综合振兴局|胆振综合振兴局]]
** 有珠郡壮瞥町
** 登別市
