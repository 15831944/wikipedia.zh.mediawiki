'''北美殖民地戰爭'''是17世紀至18世紀發生在[[殖民地|殖民地]]時代的[[北美大陸|北美大陸]]的一連串戰爭之總稱。各殖民地的戰爭和發生在[[歐洲|歐洲]]的[[戰爭|戰爭]]呼應。這一連串戰爭在[[美國|美國]]總稱'''{{lang|en|French and Indian Wars}}'''，在[[魁北克|魁北克]]總稱'''{{lang|en|Intercolonial Wars}}'''。另外，[[威廉三世_(英格蘭)|威廉三世]]時代開始持續的英法相爭到[[拿破崙戰爭|拿破崙戰爭]]終結為止也被稱作[[第二次百年戰爭|第二次百年戰爭]]。

== 概要 ==
北美的戰爭雖然有[[西班牙|西班牙]]及[[新西班牙|新西班牙]]（1519年–1821年）、[[荷蘭|荷蘭]]及[[新尼德蘭|新尼德蘭]]（1614年–1674年）参加，但是，基本上是「[[英國|英國]]、[[英屬美洲|英屬美洲]]（1607年-1783年）、同盟的[[印地安|印地安]]部族軍（[[易洛魁聯盟|易洛魁聯盟]]）」對「[[法國|法國]]、[[新法蘭西|新法蘭西]]（1534年–1763年）、同盟的印地安部族軍（[[瓦巴納基聯盟|瓦巴納基聯盟]]）」的戰爭。主因是擴張中的英法北美殖民地因西部或内陸地域支配權的競合，另外，歐洲爆發戰爭也必定波及北美大陸。以下是一連串的北美殖民地戰爭。

== 威廉王戰爭 ==
{{main|威廉王之戰}}
1689年，因[[光榮革命|光榮革命]]成為英王的[[威廉三世_(英格蘭)|威廉三世]]参加歐洲的[[奧格斯堡同盟|奧格斯堡同盟]]，英法的對立開始。因此，在北美的英屬美洲和法屬[[加拿大|加拿大]]也發生戰爭，但是，未獲得決定性的結果，因1697年的[[雷斯威克條約|雷斯威克條約]]而停戰。

== 安妮女王战争 ==
{{main|安妮女王战争}}
呼應歐洲的[[西班牙王位繼承戰爭|西班牙王位繼承戰爭]]而開始的北美殖民地戰爭（1702年～1713年），英軍占領法屬[[阿卡迪亞_(加拿大)|阿卡迪亞]]的皇家港，根據1713年的[[烏特勒支條嗯約|烏特勒支條嗯約]]，獲得加拿大東部的[[紐芬蘭島|紐芬蘭島]]和[[哈得遜灣|哈得遜灣]]地域。

== 喬治王戰爭 ==
{{main|喬治王戰爭}}
呼應歐洲的[[奧地利王位繼承戰爭|奧地利王位繼承戰爭]]的北美殖民地戰爭（1744年～1748年），新英格蘭殖民地軍攻陷加拿大東部的要衝路易斯堡要塞，但是，根據[[第二亞琛和約|亞琛條約]]，要塞返還法國。

== 法國－印第安人戰爭 ==
{{main|法國－印第安人戰爭}}
法國－印第安人戰爭是歐洲的[[七年戰爭|七年戰爭]]（1756年 - 1763年）的2年前在北美開始的英法殖民地戰爭（1754年 - 1763年），最初是和印地安同盟的法軍占優勢，但是，最終以英軍占領[[魁北克市|魁北克市]]和[[蒙特婁|蒙特婁]]，壓制加拿大。

1763年，[[巴黎和約_(1763年)|巴黎條約]]規定法國割讓密西西比河東側和加拿大全域給英國。法國實質上喪失北美的殖民地，英國則維持殖民地。

== 結果 ==
戰爭進行中，英國漸次掌握優勢。但是，英國獲得壓倒性的勝利反而導致新英格蘭殖民地的喪失。法國的威脅消失後，新英格蘭殖民地不再需要英國的軍事保護。巴黎條約簽訂的12年後，爆發[[美國獨立戰爭|美國獨立戰爭]]（1775年-1783年）。
