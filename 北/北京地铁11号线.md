'''北京地铁11号线'''是[[北京地铁|北京地铁]]的一条拟建线路，又名'''[[2022年冬季奥林匹克运动会|冬奥]]支线'''。11号线西段全线位于[[北京市|北京市]][[石景山区|石景山区]]，北起[[金顶街站|金顶街站]]，南至[[首钢站|首钢站]]。西段线路全长3.6千米，设4座车站，为[[2022年冬季奥林匹克运动会|2022年冬季奥林匹克运动会]]配套建设项目之一，预计2022年开通试运营。<ref>{{cite web |title=北京石景山：2019年力争开建M11线冬奥支线 |url=http://www.bj.xinhuanet.com/bjyw/2019-01/11/c_1123977853.htm |website=新华网北京频道 |accessdate=2019-04-05}}</ref>

== 路线 ==

== 历史及未来规划 ==
* 2006年，11号线最初设计为从[[万柳地区|万柳]]到[[大红门|大红门]]的L形线，与L形的[[北京地铁10号线|北京地铁10号线]]一起构成地铁环线。在10号线二期通车形成环线后，11号线规划被取消。

* 2010年1月，[[石景山区|石景山区]]人民政府披露了北京地铁11号线的计划，该线路将穿越[[首钢集团|首钢集团]]，并将与[[北京地铁4号线|4号线]]和[[北京地铁16号线|16号线]]换乘。

* 2019年4月4日，[[中国铁道科学研究院|中国铁道科学研究院]]官网发布《北京市轨道交通第二期建设规划调整环境影响报告书（征求意见稿）》<ref>{{cite web |title=北京市轨道交通第二期建设规划调整 环境影响评价第二次公示 |url=http://home.rails.com.cn/news.php?id=54325 |website=中国铁道科学研究院集团有限公司 |accessdate=2019-04-05}}</ref><ref>{{cite web |title=北京市轨道交通第二期建设规划调整环境影响报告书（征求意见稿） |url=http://home.rails.com.cn/download/2019/20190404_2_1_%E9%99%84%E4%BB%B61%EF%BC%9A%E5%8C%97%E4%BA%AC%E5%B8%82%E8%BD%A8%E9%81%93%E4%BA%A4%E9%80%9A%E7%AC%AC%E4%BA%8C%E6%9C%9F%E5%BB%BA%E8%AE%BE%E8%A7%84%E5%88%92%E8%B0%83%E6%95%B4%E7%8E%AF%E5%A2%83%E5%BD%B1%E5%93%8D%E6%8A%A5%E5%91%8A%E4%B9%A6%EF%BC%88%E5%BE%81%E6%B1%82%E6%84%8F%E8%A7%81%E7%A8%BF%EF%BC%89.pdf |website=中国铁道科学研究院集团有限公司 |accessdate=2019-04-05}}</ref>，为支撑首钢北区建设及交通出行问题，保障2022年冬奥会及赛后大型活动的大客流疏散，提出新增冬奥支线项目，构建区域内南北骨干线，计划2019年施工，2022年开通。

=== 换乘站 ===
拟建11号线西段有换乘车站1座，在[[金安桥站|金安桥站]]与[[北京地铁6号线|6号线]]和[[北京地铁S1线|S1线]]换乘。
