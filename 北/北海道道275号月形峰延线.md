'''北海道道275号月形峰延线'''（{{lang-ja|北海道道275号月形峰延線}}）是一条连结[[北海道|北海道]][[桦户郡|桦户郡]][[月形町|月形町]]和[[美唄市|美唄市]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。通称「樺戸道路」。

== 指定区間 ==
* 起点：北海道樺户郡月形町字階乐町（＝[[国道275号|国道275号]]交点）
* 終点：北海道美唄市峰延町本町（＝[[国道12号|国道12号]]交点）
* 总長：11.0千米

== 共线区间 ==
* 月形町字階乐町-美唄市西美唄町大曲（[[北海道道6号岩见泽月形线|北海道道6号岩见泽月形线]]）
* 美唄市上美唄町南（[[北海道道139号江别奈井江线|北海道道139号江别奈井江线]]）

== 历史 ==
* 1957年7月25日 路線勘定为218号（1957年北海道告示第1487号）
* 1994年10月1日 路線番号を275号に変更。（1994年北海道告示第1468号）
该线路的前身为1944年11月25日所勘定的準地方費道159号月形峰延停車場線的部分路段。
