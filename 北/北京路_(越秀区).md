'''北京路'''是[[广州市|广州市]][[越秀区|越秀区]]一條集[[文化|文化]]、[[娱乐|娱乐]]、[[商業|商業]]于一体的[[街道|街道]]，是廣州市有史以來最繁华的商業中心<ref>{{zh}}[http://www.ch1188.com/wadq/ShowArticle.asp?ArticleID=6566 廣州步行街] {{webarchive|url=https://web.archive.org/web/20071013182437/http://ch1188.com/wadq/ShowArticle.asp?ArticleID=6566 |date=2007-10-13 }}，於2009年4月16日查閱</ref><ref>{{zh}}[http://www.dragonstrail.com.cn/guangzhou/bjroad/aboutbj.html 广州北京路步行街简介] {{webarchive|url=https://web.archive.org/web/20070217052600/http://www.dragonstrail.com.cn/guangzhou/bjroad/aboutbj.html |date=2007-02-17 }}，於2009年4月16日查閱</ref>，日均人流量约35万人次<ref name="古1">{{zh}}[http://gb.cri.cn/1321/2008/08/28/542s2215805.htm 千年古道，繁华盛世--广州北京路今貌]，於2009年4月16日查閱</ref>。北京路北起[[广卫路|广卫路]]，南到[[沿江路_(广州)|沿江中路]]，全长1500多米。

== 歷史 ==
[[File:Wing_Hon_Road1920.jpg|left]]
[[File:Wing_Hon_Road_Canton.jpg|thumb]]
[[File:Wing_Hon_Road.jpg|right]]交汇处，尽头为现[[广东财政厅旧址|广东财政厅旧址]]]]
旧时的北京路，是广州[[大南门|大南门]]（今北京路、大南路口）内外的一条南北向主要街道。[[宋代|宋代]]灭[[南汉|南汉]]后，拆南城墙，将南城沿至江边，建“双阙”。1244年，建双门楼，从此北京路中段被称作'''双门底'''，渐成为商业区。[[清代|清代]]时，今北京路由北至南依次为'''承宣直街'''、'''双门底'''、'''雄镇直街'''、'''永清街'''，以卖书坊、古董市、花市为著名<ref>[http://www.southcn.com/news/gdnews/nanyuetuijian/200208070745.htm 揭开北京路的历史谜团]</ref>。清雍正年间，现北京路与沿江路交汇的[[珠江|珠江]]边，是[[天字码头|天字码头]]的所在地，为清代時是專為迎送官員而設，附近的接官亭正是當時迎接官員的地點。

[[中华民国|中华民国]]九年（1920年），广州拆城墙开路，将此路改名为“'''永汉路'''”（国民政府使用粤音英文名“{{lang|en|Wing Hon Road}}”），取的是清朝已灭，把南街名“永清”变为“永汉”之意<ref>{{zh}}[http://chinareviewagency.net/doc/1004/4/6/3/100446372.html?coluid=55&kindid=1153&docid=100446372 道不尽的北京路]，於2009年4月15日查閱</ref>。1936年，曾任[[國民政府|國民政府]][[中華民國立法院|首任立法院長]]的[[胡漢民|胡漢民]]去世後，又胡本身乃廣州人，為紀念之，永汉路更名為“'''漢民路'''”。1949年10月[[中国共产党|中国共产党]]夺取广州後恢復原稱。1966年[[文化大革命|文化大革命]]期间被当局改名为“北京路”，寓意广州人民拥护党中央，向往首都北京之意。至今沒有恢復舊名。

=== 文化 ===
[[File:Beijing_Lu_Pedestrian_Area_2019.jpg|right]]
北京路从清代起已有壁鱼堂、汲古堂等书店和曾宗周朱墨店、刘中山笔店，這一带还聚有大量刻书坊<ref>{{zh}}书坊街“城市书房”的书香与鱼腥，南方都市报，2004年8月12日</ref>。
[[File:商務印書館廣州分館.jpg|thumb]]
从1907年1月上海的[[商务印书馆|商务印书馆]]在永汉北路创办分馆开始，就不断有新的书局、书店在这里设立。1912年冬，上海的[[中华书局|中华书局]]在永汉北路设立分局（今[[聯合書店|聯合書店]]）。1921年，以出版教科书和文化教育用书为主的上海世界书局在惠爱路昌兴街设分局，后又在永汉北路设立分局。1924年，上海民智书局在永汉北路设立分局，广益书局，会文堂书局，锦章书局等大型出版机构也先后在永汉北路设立分局<ref>廣東省志·出版。</ref>。在[[陳濟棠|陈济棠]]主粤的1920-1930年代，广州市区基本形成以永汉北路、文德北路、光复中路、十八甫等为中心的出版发行集中地<ref>{{Cite web|url=http://www.china.com.cn/book/txt/2009-12/27/content_19139388_2.htm|title=广州北京路:民国书业曾风流 好像一个传说|accessdate=|author=|date=2009-12-27|publisher=中国网}}</ref>。

现在北京路上有两间琴行、六间书店，分别是有着建于1952年的[[新华书店|新华书店]]、建于1954年的[[广州市儿童书店|儿童书店]]、[[广州市科技书店|科技书店]]、[[聯合書店|聯合書店]]、古籍书店、教材书店和外文书店。北京路附近还有文具用品店三多轩和古玩店集雅斋。

在北京路範圍同時有其他文物古迹，包括：[[南越国木构水闸遗址|南越国木构水闸遗址]]、明朝[[大佛寺_(广州)|大佛寺]]、明朝[[广州都城隍庙|城隍庙]]、明清大南门遗址、清代庐江书院、[[广州起义纪念馆|广州起义纪念馆]]等，正在发掘的古迹有：秦番禺城遗址、[[秦汉造船工场遗址|秦汉造船工场遗址]]、西汉[[南越国宫署遗址|南越国宫署遗址]]、唐清海军楼遗址等<ref>{{zh}}[http://gz.bendibao.com/live/2007911/live13116.asp 北京路商业街]，於2009年4月15日查閱</ref>。

2002年7月，北京路的北段出土了自[[唐代|唐代]]直到民国时期共五朝的十一層路面，而南段则发掘出宋代至明清时期共5层的拱北楼建筑基址。出土的文物包括大量石條、牆磚，以及[[宋代|宋代]]拱北樓基址、[[明代|明代]]拱北樓抱鼓石。遺跡現用玻璃覆蓋，原地保留<ref>{{zh}}千年古道现繁华北京路，《人民日报》华南新闻 ，2002年09月12日第一版</ref>。

== 商业步行街 ==
1997年2月，北京路正式开通为双休日步行街，逢双休日实施只允许公共汽车行驶的准步行。从1998年到2001年期间，北京路逢节假日实施所有车辆均不能行驶的全步行；2002年元旦起，北京路正式实施全日制步行<ref name="古1"></ref>。北京路是百年老店和广州老字号商铺的集中地，店铺众多，规模大，经营样式齐全，各类商铺卖场，以及酒楼、餐饮食肆、文化娱乐场所和银行等总计超过五百家<ref name="商1">{{zh}}
[http://www.guangzhou.gov.cn/node_646/node_653/2005-06/111874115952659.shtml 广州步行街指南] {{webarchive|url=https://web.archive.org/web/20080512023452/http://www.guangzhou.gov.cn/node_646/node_653/2005-06/111874115952659.shtml |date=2008-05-12 }}，於2009年4月15日查閱</ref>。沿街商铺以经营服装、百货、鞋类、餐饮、珠宝为主，现有时装、皮具专卖店近300多家，服装、鞋业、精品、百货等综合商场20多家<ref name="商1"></ref>。广州市最早的肉菜市场——禺山市场亦位于北京路上。现代化的大[[百货公司|百货公司]]则有建于20世纪40年代的[[新大新公司|新大新百货公司]]，[[广百股份|广州百货公司]]、[[银座|银座]]、[[五月花商业广场|五月花商业广场]]等大商店。2008年，[[天河城百货|天河城百货]]还进驻了早前新的[[名盛广场|名盛广场]]，令附近的商业变得更加兴旺。

== 饮食 ==
北京路的饮食消费层次非常之多，从低价的小吃店、快餐店到高档的餐厅，应有尽有，餐饮种类亦丰富多样，有古色古香的广州地道[[西关|西关]]美食，有全国各地风味菜品，有越南、意大利、法国等特色西餐料理，有回转寿司、铁板烧等[[日本料理|日本料理]]，此外还有韩国菜、[[印度菜|印度菜]]以及东南亚美食等等<ref>[http://eat.gd.sina.com.cn/food/070706/index.html 北京路美食新体验] {{Wayback|url=http://eat.gd.sina.com.cn/food/070706/index.html |date=20160304192927 }}，於2009年4月16日查閱</ref>。新式食肆主要集中在光明广场，临近的[[惠福路|惠福东路]]在2010年起也大力发展饮食，临街食肆林立。

北京路的老字号餐厅有[[聚宝乳鸽|聚宝乳鸽]]、[[太平館餐廳|太平館餐廳]]、[[银记肠粉|银记肠粉]]等。北京南还有很多消费水平比较低的小食店，里面的[[雲吞麵|雲吞麵]]、[[牛腩|牛腩]]粉、[[鱼蛋|鱼蛋]]粉、[[拉肠|拉肠]]也非常有特色，街边更有烧烤店营业。

== 娱乐 ==
北京路拥有两间[[电影院|电影院]]，分别是[[青年文化宫|青年文化宫]]电影院和[[永汉电影院|永汉电影院]]，也邻近市一宫电影院和星汇电影院以及五月花电影院。都处于北京路辐射地段。
