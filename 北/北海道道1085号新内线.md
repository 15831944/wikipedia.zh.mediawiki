'''北海道道1085号新内線'''（{{lang-ja|北海道道1085号新内線}}）是一条位于[[北海道|北海道]][[十胜综合振兴局|十胜综合振兴局]]管内[[上川郡_(十勝国)|上川郡]][[新得町|新得町]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：北海道上川郡新得町字新内
* 終点：北海道上川郡新得町字新内
* 总距离：2.2千米

== 通過综合振兴局 ==
* 十胜综合振兴局
** 上川郡新得町

== 主要连接道路 ==
* [[国道38号|国道38号]]（终点）

== 历史 ==
* 1988年（昭和63年）3月31日 - 路線勘定（1988年北海道[[告示|告示]]第548号）

== 冬季封闭 ==
以下区間(400米)实行冬季封闭。
* 新得町字新内西6線159番3地先 - 157番1地先（新内门）
