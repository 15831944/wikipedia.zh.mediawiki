'''北海道道26号带广停车场线'''（{{lang-ja|北海道道26号帯広停車場線}}）是一条位于[[北海道|北海道]][[十胜综合振兴局|十胜综合振兴局]][[带广市|带广市]]内的[[主要地方道|主要]][[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。全路段与[[国道236号|国道236号]]并行。

== 道路概况 ==
* 起点：[[北海道|北海道]][[带广市|带广市]]西2条南11丁目（＝[[北海道旅客铁道|JR]][[根室本线|根室本线]] [[带广站|带广站]]）
* 終点：北海道带广市西2条北1丁目（＝[[国道38号|国道38号]]交点）
* 总長：1.4千米

== 经过的自治体  ==
*[[十胜综合振兴局|十胜综合振兴局]]
**[[带广市|带广市]]

* [[国道38号|国道38号]]＝西2条北1丁目（终点）

== 历史 ==
* 1954年3月30日 路线勘定为47号。（1954年北海道告示第503号）
* 1994年10月1日 路線编号变更为26号。（1994年北海道告示第1468号）
