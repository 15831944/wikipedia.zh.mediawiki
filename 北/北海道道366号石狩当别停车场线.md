'''北海道道366号石狩当别停车场线'''（{{lang-ja|北海道道366号石狩当別停車場線}}）是一条位于[[北海道|北海道]][[当别町|当别町]]内的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 道路概况 ==
* 起点：[[北海道|北海道]][[石狩郡|石狩郡]][[当別町|当別町]]園生（＝[[北海道旅客铁道|JR]][[札沼线|札沼线]] [[石狩当别站|石狩当别站]]）
* 終点：北海道石狩郡当別町弥生（＝[[北海道道81号岩见泽石狩线|北海道道81号岩见泽石狩线]]・[[北海道道112号札幌当别线|北海道道112号札幌当别线]]交点）
* 总長：0.6千米

== 经过的自治体  ==
* [[石狩振興局|石狩振興局]]
** 石狩郡当別町

== 主要连接道路 ==
* 当別町
** [[北海道道81号岩见泽石狩线|北海道道81号岩见泽石狩线]]＝弥生（終点、与[[国道275号|国道275号]]相接）
** [[北海道道112号札幌当别线|北海道道112号札幌当别线]]＝弥生（终点）

== 历史 ==
* 1961年3月31日 路線勘定。（1961年北海道告示第616号）
