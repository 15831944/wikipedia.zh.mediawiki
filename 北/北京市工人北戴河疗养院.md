'''北京市工人北戴河疗养院'''，位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[北戴河区|北戴河区]]草厂西路26号，是[[北京市总工会|北京市总工会]]直属[[事业单位|事业单位]]<ref>{{Cite web |url = http://acftu.people.com.cn/n/2013/0424/c67502-21257244.html |title = 组织全覆盖 服务零距离 |author =  |date = 2013-04-24 |publisher = 人民网 |language =  |accessdate =  |quote =  |archive-url = https://web.archive.org/web/20130508130916/http://acftu.people.com.cn/n/2013/0424/c67502-21257244.html |archive-date = 2013-05-08 |dead-url = yes }}</ref>，也是北戴河历史最悠久的疗养院之一。

==简介==
北京市工人北戴河疗养院坐落在北戴河海滨中部，西望[[联峰山|联峰山]]，西侧紧邻[[中共中央直属机关北戴河服务局|中共中央直属机关北戴河服务局]]，南邻[[碣石园|碣石园]]，再向南便是海滨沙滩。1952年，北京市工人北戴河疗养院经[[中央人民政府政务院|中央人民政府政务院]]总理[[周恩来|周恩来]]批准建立<ref name=lym/>。

1950年代到1960年代，该疗养院仅有床位一百多张，每年能接待一、两千位劳动模范。当时，著名劳动模范[[李瑞环|李瑞环]]、[[倪志福|倪志福]]、[[张百发|张百发]]、[[张秉贵|张秉贵]]等都曾来此休养。[[文革|文革]]时期，疗养院被指责为“越养越[[修正主义|修]]”，被迫关闭，休养楼年久失修，院内杂草丛生，不少地块还种了庄稼或被人侵占<ref name=lym/>。

[[中共十一届三中全会|中共十一届三中全会]]后，疗养院重现生机。1980年，[[北京市人民政府|北京市人民政府]]拨款新建了七号、八号、九号三栋新楼。1991年又对此楼装修改造，添置高档家具。1997年，北京市总工会投资在疗养院东院新建了二星级宾馆“京工宾馆”，疗养院床位增至480多张，每年可接待近万人<ref name=lym/>。2012年8月，来自北京市各行各业的80名劳动模范在北京市总工会安排下，在2012年重新装修一新的北京工人北戴河疗养院度过4天休养假期<ref>{{Cite web |url = https://news.qq.com/a/20120814/000165.htm |title =北京80名劳模前往北戴河休养 最小者24岁  |author =  |date = 2012-08-14 |publisher =  腾讯新闻|language =  |accessdate =  |quote =  }}</ref>。

北京市工人北戴河疗养院是北京市总工会在北戴河地区唯一的疗养事业单位。总占地面积近80亩。分为东院、西院，东院比西院稍大。另有一个小庭院占地2亩多，因大门为[[月亮门|月亮门]]，故称“月亮门小院”。整个疗养院内有别墅楼、休养楼、宾馆十多栋。主要建筑有<ref name=lym/>：

*[[北戴河近代建筑群|五凤楼]]：西院内中央有相邻的五栋小楼，是[[中华民国|中华民国]]时期建筑，人称“五凤楼”，2006年列入[[全国重点文物保护单位|全国重点文物保护单位]]“[[北戴河近代建筑群|北戴河近代建筑群]]”<ref name=lym>李玉梅，风景这边独好——记北京市工人北戴河疗养院，工会博览2000(11):23-23</ref>。2010年代，五凤楼已外包出去，不对散客提供入住服务<ref name=sohu>{{Cite web |url =http://www.sohu.com/a/193683685_148923  |title =北京市工人北戴河疗养院  |author =  |date = 2017-09-22 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。
*鸿雁楼：位于西院的北区，五凤楼东北，三层米黄色楼房，是对外开放的客房楼<ref name=sohu/>。
*京工宾馆：即东院，主要建筑有<ref name=sohu/>：
**五月楼：位于北部
**贤勤楼：位于东北，五月楼以东
**明月楼：位于东部，贤勤楼以南
**满江楼：位于西北，五月楼西南
**云帆楼：位于西部，满江楼东南
**幽燕楼：位于中央，明月楼和云帆楼之间
**观海楼：位于南部，幽燕楼以南<ref name=sohu/>
