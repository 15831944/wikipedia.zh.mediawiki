== 背景 ==
{{see also|北漂}}
根据人口普查和抽查，北京市1995年以来一直是全国[[总和生育率|总和生育率]]最低的一級行政區。2000年以来，北京市总和生育率处于0.66和0.71之间，不到世代更替水平的三分之一，不仅低于[[中国大陆生育率|中国其他省份]]，也低于[[各国生育率列表|所有其他国家]]<ref>{{cite web|title=全国人口普查公报|url=http://www.stats.gov.cn/tjsj/tjgb/rkpcgb/qgrkpcgb/|website=中华人民共和国国家统计局|accessdate=2017-12-09}}</ref>。

据统计，2003年北京市外来人口总计为409.5万人，其中农村从业人员159.6万，从事第二、第三产业的约为100万人。其特点是年龄偏大，文化水平和技术水平偏低，存在就业稳定性差、企业用工不够规范等问题，就业集中在生产运输和商业、服务业等领域<ref>{{cite book|author1=周桂元|editor=农业部软科学委员会办公室|title=农村劳动力转移与农民收入|date=2010|publisher=中国财政经济出版社|isbn=9787509524312}}</ref>。目前，在北京2000万人口中，有三分之一，即700万外来务工人员<ref>{{cite web|title=“北漂”族住房调查：房子比工作更难找|url=http://news.sohu.com/20170408/n487249643.shtml|website=搜狐新闻|accessdate=2017-12-08|date=2017-04-08}}</ref>，主要来自河北、河南、山东<ref>{{cite web|title=北京驅逐低端人口，恐掉「中等收入陷阱」|url=https://technews.tw/2017/12/09/socially-vulnerable-groups-china/|website=科技新报|publisher=商業周刊|accessdate=2017-12-09|date=2017-12-09}}</ref>。

在北京市的管理者眼中，低端人口不是“北京人”，不应该来挤占北京市日益紧张的公共资源；低端人口多数从事体力工作，不好看、不体面，不符合北京国际大都市的形象和定位。因此，北京市人大常委会建议，采取措施引进高端人才，减少低端劳动力，以应对北京市人口持续增加的压力。还有北京市政协委员提出，有必要对进京人口设门槛。<ref name=编码008/>

北京外来人口也称为'''北漂族'''，是指来自非北京地区的、非北京户口、在北京生活和工作的外地人<ref>{{cite web|title=【直击中国】 北漂、底族成低端人口；北京这是咋了？|url=http://www.rfa.org/cantonese/commentaries/on-china/on-china-11272017083250.html|website=自由亚洲电台|accessdate=2017-12-08|date=2017-11-27}}</ref>。北漂族在来京初期都很少能有固定的住所，其自身也因诸多原因而不能对于北京有更多的认同感，故此得名。90年代末，政府开始把一些防空洞、地下隧道改造后作为出租房租赁。由于北京市一般住宅平均月租金超过3000元，因此[[地下室|地下室]]往往是低收入北漂族的唯一选择。目前北京有5500个地下室，据估计住了有100万人。<ref name=chinadaily>{{cite web|title=美女画家住北京地下室 北漂族心酸生活谁知|url=http://www.chinadaily.com.cn/interface/toutiaonew/53002523/2016-05-09/cd_25155879.html|website=中国日报网|accessdate=2017-12-08|date=2016-05-09}}</ref>

[[北京市委|北京市委]]书记[[蔡奇|蔡奇]]要求到2020年，常住人口规模要控制在2300万人以内<ref>{{cite web |url= http://news.sina.com.cn/o/2017-09-30/doc-ifymksyw4999436.shtml|title=北京控制人口规模是首当其冲的硬任务|first= |last= |work= 新浪新闻|accessdate=2017-09-30}}</ref>。有香港媒体分析称驱离低端人口事件可能与此有关<ref>{{cite web |url= https://www.hk01.com/%E5%85%A9%E5%B2%B8/138906/-%E9%A9%85%E4%BD%8E%E7%AB%AF%E4%BA%BA%E5%8F%A3-%E5%8C%97%E4%BA%AC%E6%9B%B8%E8%A8%98%E8%94%A1%E5%A5%87-%E7%8B%A0%E8%A9%B1-%E6%9B%9D%E5%85%89-%E8%A6%81%E6%B1%82%E5%AE%98%E5%93%A1%E6%95%A2%E6%96%BC%E7%A1%AC%E7%A2%B0%E7%A1%AC|title=北京書記蔡奇「狠話」曝光 要求官員敢於硬碰硬|first= |last= |work=香港01|accessdate=2017-12-05}}</ref>。

北京市限制“低端人口”的做法也遭到了舆论和学者的批评<ref>{{Cite web|url=http://focus.news.163.com/11/0603/15/75KR92SD00011SM9.html|title=清理外来人口难解北京困局|accessdate=2017-12-06|date=2011-06-03|work=京华时报(北京)|language=zh-CN|author=京华时报}}</ref><ref>{{Cite web|url=http://star.news.sohu.com/s2012/renkouguimo/index.shtml|title=论座：控制大城市人口规模是个伪命题|accessdate=2017-12-06|date=2012-02|work=搜狐网|language=zh-CN|author=北民}}</ref>。[[国务院发展研究中心|国务院发展研究中心]]社会发展研究部副部长贡森认为，虽然过多的“低端人口”可能影响一个城市的外在形象，但城市的包容性更重要，是“内在美”<ref>{{cite book|author1=高明勇|title=坐下来的中国|date=2014-03-01|publisher=浙江人民出版社|isbn=9787213059995|chapter=为何地方不“爱”普通劳动者}}</ref>。他主张同等数量的“高端人口”消耗的资源远高于“低端人口”。用“高端”取代“低端”，必然加剧交通、住房和用水用地问题<ref>{{Cite book|title=坐下来的中国|last=高明勇著|first=|publisher=杭州：浙江人民出版社|year=2014年3月|isbn=978-7-213-05999-5|location=|pages=132}}</ref><ref>{{Cite book|title=中国大城市化共识 The consensus on China's urbanization|last=童大焕主编|first=|publisher=北京：东方出版社|year=2014年10月|isbn=978-7-5060-7749-1|location=|pages=57}}</ref>。《人民日报》指出，如果真的对所谓的“低端”劳动力强制退出，其结果必然是百姓生活变得极为不便，生活成本大幅提高<ref>{{Cite web|url=http://finance.people.com.cn/GB/12860274.html|title=“低端”劳动力强制退出 谁是城市多余的人?|accessdate=2017-12-06|date=2010-09-30|work=人民网-人民日报|language=zh-CN|author=白天亮}}</ref>。

2017年11月26日，北京市安全生产委员会针对“安全隐患大排查大清理大整治专项行动”是在驱赶“低端人口”的质疑，回应称“不存在驱赶‘低端人口’说法”、“这是不负责任，也是毫无根据的，没有‘低端人口’一说。”<ref>{{Cite web|url=http://news.youth.cn/sh/201711/t20171126_11065186.htm|title=北京安委会回应“驱逐低端人口”传言：从来没有“低端人口”一说_新闻频道_中国青年网|accessdate=2017-11-26|author=青网新闻邵志凯|work=青网新闻}}</ref><ref>{{cite news|title=安全隐患清查整治引质疑 北京清理“低端人口”？|url=https://www.zaobao.com.sg/special/report/politic/cnpol/story20171125-813698|accessdate=2017-11-26|agency=[[联合早报|联合早报]]|date=2017-11-25}}</ref>但承认在疏解外来务工人员的过程中，北京市对合法缴纳社保但被迫离职的外来务工人员发放一次性生活补贴<ref>{{cite news|title=北京：外来务工人员因疏解失业可获补贴 最多可领13536元|url=http://www.guancha.cn/society/2017_12_07_438212.shtml|accessdate=2017-12-09|agency=中新网|publisher=观察者|date=2017-12-07}}</ref>。
