'''北岛'''（{{bd|1949年|8月2日|catIdx=Bei北島}}），原名'''赵振开'''，[[中国|中国]]当代[[诗人|诗人]]，为[[朦胧诗|朦胧诗]]代表人物之一。他用北岛这个笔名是因为他来自北方，而且偏爱孤独。他先后获瑞典笔会文学奖、美国西部笔会中心自由写作奖、古根海姆奖学金、金花环奖等多个奖项，并被选为美国艺术文学院终身荣誉院士。之前擔任[[香港中文大學|香港中文大學]]東亞研究中心人文學科講座教授，现为中大文學院榮譽教授<ref>{{Cite web|url=https://www.arts.cuhk.edu.hk/web/zh-tw/about-us/people|title=中大文學院 - 學院人員|accessdate=2019年2月12日|author=中大文學院行政組|date=2018-19年|publisher=中大文學院}}</ref>。

==生平==
===早年===
北岛祖籍[[浙江|浙江]][[湖州|湖州]]，1949年生于当时的[[北平|北平]]（现[[北京|北京]]）。在少年时期，他是一名红卫兵。他对文化大革命有疑虑，在1969年到1980作为一名工人接受了“再教育”。他毕业于[[北京四中|北京四中]]。1969年当建筑工人，后作过翻译，并短期在《新观察》杂志作过编辑。1970年开始写作，1978年与[[芒克|芒克]]等人创办《[[今天_(杂志)|今天]]》杂志。

===六四运动===
1989年4月15日[[胡耀邦|胡耀邦]]去世后中国[[北京|北京]]爆发大规模学潮悼念胡耀邦，4月26日《[[人民日报|人民日报]]》发表社论将学潮定性为动乱进一步加剧局势紧张，20万学生进入[[天安门广场|天安门广场]]以示抗议。当日晚，北岛与150名学者连署請願信送交[[全国人大|全国人大]]，聲援學生，但未获接收<ref>[http://www.64memo.com/disp.aspx?Id=8164 致中共中央、人大常委会、国务院的公开信（摘自《天安门一九八九》台北联经出版社）]</ref>。由于北岛因在[[六四运动|六四运动]]中的特殊作用，不见容于中国政府。

===移居海外===
1989年移居国外，曾一度旅居[[瑞典|瑞典]]等七个国家，他在世界上多个国家进行创作，寻找机会朗读自己的诗歌。

1990年在北岛的主持下《今天》文学杂志在[[挪威|挪威]]复刊，至今仍在世界各地发行，其网络版和论坛（www.jintian.net）也享誉世界各地汉语文学圈。

1994年曾經返回中國，在北京入境時被扣留，遣送回美國，曾任教于[[加利福尼亞大學戴維斯分校|加利福尼亞大學戴維斯分校]]，还曾是[[斯坦福大学|斯坦福大学]]、[[伯克萊加州大學|加利福尼亞大学伯克莱分校]]、[[香港中文大学|香港中文大学]]客座教授。另外，他还曾以访问学者的身份在杜伦大学中文系担任讲师。在纽约居住期间，曾在[[纽约州立大学石溪分校|纽约州立大学石溪分校]]英文系教授诗歌创作。

===回国===
根据北島在《城门开》一书描写，2003年1月11日，他还去304医院看望瘫痪在床的父亲，父子俩都知道是最后一次见面，北島还说他们最后还说道：我爱你。第二天，北岛乘机再次离开中国。

2007年，北島收到[[香港中文大學|香港中文大學]]的聘書。8月，北島正式搬到[[香港|香港]]，與其家人團聚，結束其近20年的歐美各國漂泊式生活<ref>[http://www.chinesepen.org/Article/srsh/200803/Article_20080317003853.shtml 对话北岛：诗人是独自挖煤的矿工，读者是在大地上行走的人] {{webarchive|url=https://web.archive.org/web/20080322215916/http://www.chinesepen.org/Article/srsh/200803/Article_20080317003853.shtml |date=2008-03-22 }} [[田志凌|田志凌]] 南方都市報</ref>。

2011年8月8日，北岛获准入境参加[[青海湖国际诗歌节|青海湖国际诗歌节]]并接受了[[新华社|新华社]]的专访<ref>{{cite web|url=http://news.xinhuanet.com/society/2011-08/10/c_121838619.htm|title=北岛现身青海湖诗歌节 20多年来首次参加国内官方活动 |accessdate=2011-08-10 |publisher=新华网 |language=zh-hans}}</ref>。

「時間的玫瑰——北島詩歌之夜」周三（2013年10月30日）在中文大學的邵逸夫堂舉行，600多人入場，創下主辦方同系列活動的最高紀錄。

2014年10月17日，北岛阔别45年重回北京四中，并表明希望能回北京四中开课<ref>{{cite web|url=http://culture.people.com.cn/n/2014/1017/c22219-25855992.html|title=北岛阔别45年重回母校：希望能回北京四中开课(图) |accessdate=2014-10-19 |publisher=人民网 |language=zh-hans}}</ref>。

2015年3月，北岛获马其顿—斯特鲁加国际诗歌节授予的最高荣誉金花环奖（Golden Wreath Award）桂冠，成为继[[绿原|绿原]]之后第二位获此殊荣的中国诗人<ref>{{cite web|url=http://cn.rfi.fr/%E4%B8%AD%E5%9B%BD/20150328-%E5%B1%85%E6%B8%AF%E5%A4%A7%E9%99%86%E8%AF%97%E4%BA%BA%E5%8C%97%E5%B2%9B%E5%A4%BA%E9%87%91%E8%8A%B1%E7%8E%AF%E5%A5%96%E6%A1%82%E5%86%A0%E6%88%90%E4%B8%8B%E5%B1%8A%E8%AF%BA%E8%B4%9D%E5%B0%94%E5%A5%96%E7%83%AD%E9%97%A8/|title=居港大陆诗人北岛夺金花环奖桂冠成下届诺贝尔奖热门|accessdate=2015-04-01|publisher=法国国际广播电台|language=zh-hans}}</ref>。

北岛的作品已被译成二十多种文字出版。代表作包括作于1976年文革结束后的《回答》，其中的“卑鄙是卑鄙者的通行证，高尚是高尚者的墓志铭”已经成为中国新诗名句。在美国，其作品由 Zephyr Press 出版。曾多次获[[诺贝尔文学奖|诺贝尔文学奖]]提名<ref>{{cite web |url=http://book.ifeng.com/special/beidaonuobeier/200910/1007_8291_1378059.shtml|title=《北岛2009年诺文学奖赔率50/1 排第28位》|accessdate=2009年10月7日|work=|publisher=凤凰网|language= zh-hans}}</ref><ref>{{cite web |url=http://book.ifeng.com/special/beidaonuobeier/200910/1008_8291_1378574.shtml|title=《北岛多次进入诺文学奖终审名单 有时仅差一票惜败》|accessdate=2009年10月8日|work=|publisher=凤凰网|language= zh-hans}}</ref>，是当今影响最大，也最受国际承认的中国诗人。

==荣誉==
先后获[[瑞典笔会|瑞典笔会]]文学奖、美国西部笔会中心自由写作奖、[[古根海姆奖学金|古根海姆奖学金]]、金花环奖等，并被选为[[美国艺术文学院|美国艺术文学院]]终身荣誉院士。曾經獲得美國布朗大學頒授榮譽文學博士學位，並由布朗大學校長路斯．西蒙斯教授於典禮上宣讀讚詞：「''你用深邃的、充滿力量、讓人難以忘懷的詩句，向你的祖國和世界發出了聲音，譜寫自由和表達的樂章。你曾經寫道：『我不相信天是藍的，我不相信雷的回聲，我不相信夢是假的，我不相信死無報應。''』......''為了你對於人性根本之美德堅持不懈的信仰、為了你對於書寫文字之力量的信心，為了你對於人類之自由的深切信念，我們授予你文學博士學位以致敬意。''」頒授儀式於第243屆布朗大學畢業典禮中舉行<ref>{{Cite web|url=https://www.cpr.cuhk.edu.hk/tc/press_detail.php?id=1046&t=%E4%B8%AD%E5%A4%A7%E5%8C%97%E5%B3%B6%E6%95%99%E6%8E%88%E6%A6%AE%E7%8D%B2%E7%BE%8E%E5%9C%8B%E5%B8%83%E6%9C%97%E5%A4%A7%E5%AD%B8%E6%A6%AE%E8%AD%BD%E6%96%87%E5%AD%B8%E5%8D%9A%E5%A3%AB%E5%AD%B8%E4%BD%8D&id=1046&t=%E4%B8%AD%E5%A4%A7%E5%8C%97%E5%B3%B6%E6%95%99%E6%8E%88%E6%A6%AE%E7%8D%B2%E7%BE%8E%E5%9C%8B%E5%B8%83%E6%9C%97%E5%A4%A7%E5%AD%B8%E6%A6%AE%E8%AD%BD%E6%96%87%E5%AD%B8%E5%8D%9A%E5%A3%AB%E5%AD%B8%E4%BD%8D|title=中大北島教授榮獲美國布朗大學榮譽文學博士學位|accessdate=2019.02.12|author=中大傳訊及公共關係處|date=2011.06.12|publisher=中大傳訊及公共關係處}}</ref>。其他獲授銜人士包括參與Gemini及Appollo太空計劃的宇航員David Scott，他是第十二位登陸月球及首位於月球上駕車者；曾兩次獲普立茲獎的《紐約時報》專欄作家Nicholas Kristof；以及三度奪奧斯卡金像獎的著名影星、導演及製作人[[杰克·尼科尔森|Jack Nicholson]]。

==文學意見==
[[德國|德國]]漢學家[[魯道夫·瓦格納|魯道夫·瓦格納]]認為北島在1980年代後期出國後所寫的詩沒有長進，「基本上重複原來的意象，新的發展很少。北島沒有前進，許多中國作家也都是這樣重復自己<ref>茉莉，(1999年)，《茉莉訪瓦格納》，《開放》雜誌1999年3月號，香港。</ref>。」

而德国另一汉学家[[顾彬|顾彬]]对北岛却极为推崇，在其文章和多次访谈中对北岛的人格道义，诗歌和散文都赞誉有加。

瑞典诺贝尔文学奖终身评委[[马悦然|马悦然]]一直对[[沈从文|沈从文]]、北岛、[[李锐_(小说家)|李锐]]、[[高行健|高行健]]非常喜欢，并且将他们的作品翻译成为[[瑞典文|瑞典文]]<ref>{{cite web|url=http://www.chinawriter.com.cn/yc/2000/2000-10-18/27668.shtml|title=《走近诺贝尔文学奖》|accessdate=2000年10月18日|work=|publisher=中国作家网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20141020160217/http://www.chinawriter.com.cn/yc/2000/2000-10-18/27668.shtml|archivedate=2014年10月20日}}</ref>。
