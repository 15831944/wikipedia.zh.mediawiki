'''北京地铁2号线'''，前称'''北京地铁环线'''<ref name="地铁大事记3" />，是大致沿[[北京城池|原北京城池内城]]而建的呈环线运行的一条[[北京地铁|北京地铁]]线路，设有18个站和和1个车辆段，线路全长23.1千米，全部为地下线路。标识色是<span style="color:{{北京地铁颜色|2}}">蓝色</span>。

== 概述 ==
全线线路东段、北段、西段的走向与[[北京二环路|北京二环路]]重叠，线路南段沿[[长椿街|长椿街]]－[[宣武门|宣武门]]－[[前门|前门]]－[[崇文门|崇文门]]－[[北京站|北京站]]－[[建国门|建国门]]行驶，全线线路大部分延原北京内城原址地下（除了往北京站的线路），全线共18座车站。[[西直门站|西直门站]]和[[积水潭站|积水潭站]]是内、外环线进行司机换班、始发以及收车的车站。位于积水潭站与西直门站之间北侧的[[太平湖车辆段|太平湖车辆段]]，是北京地铁2号线列车停车库，车辆段在地面，两侧有线路连接地下的2号线。

自长椿街站至北京站区段这6个车站原属[[北京地下铁道一期工程|北京地下铁道一期工程]]线路（长椿街站与现[[北京地铁1号线|1号线]][[南礼士路站|南礼士路站]]相接，向西延伸往苹果园站方向）的一部分，于1969年10月1日开通；其余车站，自[[复兴门站|复兴门站]]，沿[[西直门|西直门]]、[[东直门|东直门]]至[[建国门站|建国门站]]为二期工程，于1984年9月20日开通；在1987年12月28日，两部分线路成环线运营，而地铁一线运营区段由南礼士路站延伸至到复兴门站后折返运行，不再向南转向连接[[长椿街站|长椿街站]]。

== 历史 ==
{{seealso|北京地铁一期工程}}
1970年北京地下铁道二期工程开始进行勘测设计，于1971年3月开工。二期工程是由一期工程北京站站东折返线起，向东北绕过[[北京古观象台|北京古观象台]]，延原北京内城旧址地下经建国门、东直门、西直门、复兴门与一期工程的[[南礼士路|南礼士路]]至长椿街区间相接。<ref name="地铁二期建设">[http://www.chinametro.net/Content/DisplayNews.aspx?id=18691 北京地铁二期工程十年建设历程] {{webarchive|url=https://web.archive.org/web/20131102120517/http://www.chinametro.net/Content/DisplayNews.aspx?id=18691 |date=2013-11-02 }}，中国城市轨道交通网。</ref>
至1977年主体结构基本完成。1981年底才基本竣工。由于当时地铁二期工程是边设计、边施工的情况下进行的，加上正处于“[[文化大革命|文化大革命]]”动乱期间，工程进展受到影响。由于存在问题较多，在1982年到1984年期间对地铁设备进行了大量整改。<ref name="地铁二期建设" /> 
在1983年4月1日北京地铁二期工程的[[西直门站|西直门站]]、[[车公庄站|车公庄站]]、[[阜成门站|阜成门站]]至[[复兴门站|复兴门站]]四个站区段开始试运行。

1984年9月20日北京地铁二期工程通车运营，沿西、北、东二环路按“马蹄形”运营，运行区段由复兴门车站至建国门车站之间往返，共12座车站，线路长度16.1公里。<ref name="地铁大事记3">[http://www.bjsubway.com/corporate/dtdsj/# 地铁公司1981--1990年地铁大事记]，北京市地铁运营有限公司官网。</ref> 北京地铁二期工程至全部通车历时13年。

1987年12月28日，北京地铁一线、环线开始独立运行，北京地铁一线从苹果园站到复兴门站折返运行，原北京地铁一线从长椿街站至北京站区段两端分别与二期工程的复兴门站、建国门站相连形成环线运行，至此成为一条完整的环线，共18座车站，线路长度23.03公里，始称'''北京地铁环线'''。一线与环线在复兴门站实现换乘（北京地铁史上首个换乘车站）。1989年12月29日，北京地铁二期工程正式验收交付。<ref name="地铁大事记3" />

从1950年代到1980年代，[[邓小平|邓小平]]曾三次亲临北京地铁视察，并多次作指示和批示。1984年，北京地铁二期工程——环城线通车试运营后，身着灰色[[中山装|中山装]]的邓小平偕女儿[[邓林|邓林]]、[[邓榕|邓榕]]和外孙女，在[[中共中央办公厅|中共中央办公厅]]主任[[王兆国|王兆国]]、[[中共北京市委|中共北京市委]]书记[[李锡铭|李锡铭]]等人陪同下考察北京地铁二期工程。在[[复兴门站|复兴门站]]，当北京市的同志介绍说在站台两侧分别停有中国[[长春客车厂|长春客车厂]]制造的“401”地铁列车和新购的[[东急车辆制造|日本“东急”]]地铁列车，问先乘哪一列时，邓小平连声说：“坐中国车好，先坐中国车。”<ref>{{Cite web |url = http://cpc.people.com.cn/n1/2016/0405/c69113-28251075.html |title = 最后一次视察北京 |author =  |date = 2016-04-05 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>

1997年6月2日，环线地铁[[移动电话|移动电话]]系统开通，是中华人民共和国首个开通的地下铁道[[移动通信|移动通信]]系统。<ref name="地铁大事记3" /> 

2002年1月开始，随着北京地铁更换车站标志牌，该线的导向标志由“环线”更名为“2号线”，始称“'''北京地铁2号线'''”。<ref>{{cite news|url=http://news.xinhuanet.com/newscenter/2002-07/25/content_496610.htm|title=“环线”改称“2号线” 北京地铁“叫号”取名|publisher=[[北京青年报|北京青年报]]|date=2002-07-25|accessdate=2015-04-14}}</ref>

2017年8月20日，2号线全线加装的安全门正式投入使用<ref>{{cite web |url=http://www.beijing.gov.cn/bmfw/zxts/t1488894.htm|title=8月20日地铁2号线全线启用站台门  |accessdate=2017-09-10 |publisher=北京日报 |date=2017-09-10 }}</ref>。

=== 事故 ===
*1989年5月7日4时45分，北京地铁由于鼓楼变电站值班员违章，一人操作，导致积水潭变电站825V一号馈出开关起火烧损，影响积水潭站首车内、外环不能按时开出。积水潭站外环中断行车2小时20分，内环中断行车1小时51分，酿成行车大事故，中断安全运营1257天的纪录。

*1995年11月5日18时13分，北京地铁环线内环561次438车组，在长椿街站至复兴门区间与313次434车组发生追尾事故，有32名乘客受轻伤，1名乘客留观无危险。列车部分车钩损坏，内环中断运营302分钟。 <ref>{{cite web |title=北京地铁大事记  - 行业数据 - 中国交通技术网{{!}}一个有观点的行业网站 |url=http://www.tranbbs.com/MResearch/Daten/MResearch_126695.shtml |website=中国交通技术网}}</ref>

== 轶事 ==
北京地铁2号线因为是分为一、二期工程分别建设，长椿街站至北京站区间的车站（一期工程建设，与1号线西段的车站相仿）与此线的其他车站（二期工程建设）站台、出入口通道、车站装饰风格等方面可发现一些明显的不同。部分二期工程建设的车站为未来规划建设线路预留了可换乘的站台空间。

== 使用车辆 ==
北京地铁二期的列车编组最初使用4辆编组，1995年4月26日开始北京地铁环线列车编组改为6辆编组运营。
