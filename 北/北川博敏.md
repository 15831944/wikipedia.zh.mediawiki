{{Japanese|japanese=北川 博敏|kana=きたがわ ひろとし|romaji=Kitagawa Hirotoshi}}
'''北川博敏'''（{{bd|1972年|5月27日}}），曾[[歐力士野牛|歐力士野牛]]所屬的[[職業棒球|職業棒球]]選手（[[内野手|内野手]]）。為2007年度、2008年度的選手會長。

臉上總是帶著笑容，從[[阪神虎|阪神虎]]時代就擁有'''スマイリー（微笑者）'''的綽號，另一個綽號為'''[[麵包超人|麵包超人]]'''（因為臉長的像麵包超人）。因為總是能在關鍵時刻有好的表現，所以又被稱為「招來奇跡的男人」。

== 球風 ==
[[三振|三振]]少，擁有纏鬥力高的打擊技巧。雖然是屬於中距離[[打者|打者]]，但在2003年、2004年時[[長打_(棒球)|長打]]有減少的傾向，而打出了稍多的[[雙殺|雙殺]]打。

雖然在守備範圍方面有些狹窄，但是在使用[[棒球手套|棒球手套]]的方法上非常漂亮，比起2007年被公認守備優異的[[福浦和也|福浦和也]]選手0.995的[[守備率|守備率]]還要高，達到了聯盟最高的0.998（但是[[金手套獎|金手套獎]]是福浦選手）。

== 來歷 ==
[[兵庫縣|兵庫縣]][[伊丹市|伊丹市]]出身。國中1年級的時候搬到[[埼玉縣|埼玉縣]][[志木市|志木市]]。在埼玉縣立大宮東高中3年級的時候擔任主將，在夏天的全國高中棒球大賽埼玉大賽中，以[[全壘打|全壘打]]帶領學校第一次參加[[甲子園|甲子園]]。後來進入了屬於在東都大學棒球連盟的[[日本大學|日本大學]]，與同年級的[[高根澤力|高根澤力]]爭取正[[捕手|捕手]]的位置。以捕手作為先發球員，3年級的春季聯盟戰時，獲得了聯盟最高打擊率，也被選為最佳九人。也被選為同年的日美大學棒球選手權大賽日本代表。4年級時擔任主將，再度被選為日美大學棒球賽的日本代表。聯盟通算出塞75場，244個[[打數|打數]]有66隻[[安打|安打]]，[[打擊率|打擊率]]為.270，4支全壘打、33[[打點|打點]]。在1994年的選秀會上，被阪神隊以第二順位指名入團。當時的[[背號|背號]]是'''9'''號。同年球季結束時，與[[湯舟敏郎|湯舟敏郎]]、[[山崎一玄|山崎一玄]]3人，和[[酒井弘樹|酒井弘樹]]、[[面出哲志|面出哲志]]、[[平下晃司|平下晃司]]交換，被交易到[[大阪近鐵野牛|大阪近鐵野牛]]隊。背號也改成'''46'''號。

2001年，受到梨田監督的認可，成為一軍先發，在4月28日打出了生涯第一支全壘打。在5月27日，29歳的生日當天打出了這輩子第一隻的[[再見安打|再見安打]]，在訪問台上流下了眼淚。9月26日對[[歐力士野牛|歐力士藍浪]]戰時，在9局下半無人出局[[滿壘|滿壘]]，落後3分的情況下以[[代打|代打]]身分上場，從新人[[大久保勝信|大久保勝信]]的手中打出了職業棒球史上第一隻'''「代打逆轉再見滿壘封王全壘打」'''，被稱為「'''奇蹟男'''」。2001年的日本大賽中，雖然球隊以1勝4敗的成績，輸給了[[東京養樂多燕子|東京養樂多燕子]]隊，但是也創下了全比賽出賽，14打數7安打、打擊率五成的成績。關於決定冠軍的那一支再見全壘打，球團在同年簽約時開了個玩笑，「'''那隻全壘打只有差不多10萬円的價值'''」。但是比起之前的年薪，幾乎是加倍地大幅調薪。另外，因為這支全壘打實在是太戲劇性了，於是在媒體上不停的出現，在球落點附近的廣告也因此不停的在全國放送，廣告商還因此贈送了10萬円給北川。北川本人也說過這支全壘打是「這一生中最高興的事情」。

2002年，在[[開幕戰|開幕戰]]時負傷，以43場比賽、打擊率.266、1隻全壘打、8打點的成績結束該球季。

2003年，在5月25日時代替打擊狀況不好的[[中村紀洋|中村紀洋]]，第一次打第四棒，結果打出了4安打3打點的好成績，當年也留下了打擊率.309、13隻全壘打、50打點的成績。

在2004年時為了發揮打撃能力而正式轉為内野手。開幕前因[[吉岡雄二|吉岡雄二]]受傷，被選為先發選手，除了球團罷工的2場比賽之外剩餘的133場比賽都有出場，留下了打擊率.303、20隻全壘打、88打點的自我最高成績。同年[[太平洋聯盟|太平洋聯盟]]全比賽出場的人只有北川、[[中島裕之|中島裕之]]與[[川崎宗則|川崎宗則]]3人。球季終了後，背號變更為'''23'''號。

2005年，以作為隊上的主力與靈魂人物而被期待著。雖然成績在打擊率方面有所降低，但是因打點為隊上最多而被對方投手提防著，可看見最為隊上主力的活躍。

2006年仍舊以隊上的主力活躍著但在球季中盤因飛撲接球而損傷了右肩。之後被確定為是可能影響運動生涯的右肩關節唇損傷。但是北川仍舊在8月13日對[[福岡軟體銀行鷹|福岡軟體銀行鷹]]時，強行出場。（手術在17日實行，以後的比賽都沒有出場）。

2007年，就任選手會長，受傷痊癒後仍然擔任隊上主力，棒次為第五棒。與[[Greg_LaRocca|Greg LaRocca]]、[[Tuffy_Rhodes|Tuffy Rhodes]]三人總共打了78隻全壘打（但是北川只有9隻），是12個球團中擁有屈指可數的破壊力的中心打者。但雖然是右肩傷癒回歸的第一個球季，球隊的戰績卻是最後一名。

2008年，在春季訓練中[[脫臼|脫臼]]，開幕的時候在二軍。4月29日對[[東北樂天金鷹隊|東北樂天金鷹隊]]時，回到一軍，隨著LaRocca的脫離，被任命為[[三壘手|三壘手]]。在本人[[部落格|部落格]]上的文章說，已經約12年沒有守三壘了。在5月16日時對[[千葉羅德海洋|千葉羅德海洋]]戰，從[[渡邊俊介|渡邊俊介]]手中敲出2打席連續全壘打，並創下自己最佳的1場比賽6打點紀錄。但是，在夏天之後打擊狀況不佳，多被調到六、七棒。是一個得點圏打擊率.244，未達到[[規定打席|規定打席]]等，不太順遂的球季。另一方面，也是2005年以來，全壘打的隻數達到二位數，[[犧牲打|犧牲高飛球]]也是2年連續聯盟的第一名。

2009年隨著[[José_Fernandez|José Fernandez]]的加入與LaRocca的打擊狀況的恢復，比起去年，想要成為先發選手的困難度更高了。主要是以代打為主，但因為4個洋將一個接著一個脫離戰線，一直到球季結束都是先發選手。但是，除了打擊率以外的成績幾乎都比去年要低，全壘打也只有2隻。

2010年，開幕戰以先發第4棒[[指定打擊|指定打擊]]出場。本職的一壘手位置被[[亞力士·卡布瑞拉|亞力士·卡布瑞拉]]與[[T-岡田|T-岡田]]搶走，多以指定打擊身分上場，在打撃方面有著高水準的記錄。6月2日對中日戰，本來是落後7分，球隊在8局下半無人出局時追回了3分，之後的打席揮出了滿壘全壘打，也對於後來的球隊的再見比賽作出了貢獻。之後也以主力打者的身分持續活躍著，但在8月13日時抹銷一軍登錄，而缺席了10場比賽。原因為，在7月14日對羅德戰時，受到打到了左小腿自打球的影響，而變成蜂窩性組織炎。復歸後，雖然沒辦法像戰線離脫前那樣發揮打擊能力，但是在9月的比賽中野接連完成了100支全壘打與1000隻安打的記錄。最終以出賽119場比賽，打擊率.306、12支全壘打與61打點的成績結束球季。

== 人物 ==
以愛開玩笑的性格被大家所知，微笑是他的註冊商標。因為總是一直微笑，所以分辨喜怒哀樂的方法是「請觀察鼻子部份」<ref>球季結束時，脫口秀上本人自己講的。</ref>。但是在阪神時代對[[中日龍|中日龍]]戰，分數領先對方的時候，打擊的[[山崎武司|山崎武司]]看到當時身為捕手的北川在笑，加上又被投內角球被激怒，進而發展成打群架的結果。在[[大阪巨蛋|大阪巨蛋]]比賽的時候，能力似乎會提升。

興趣是釣魚，在球季結束或空閒的時候會常常更新自己部落格的相片，其中也有飼養[[龍蝦|龍蝦]]的照片。
