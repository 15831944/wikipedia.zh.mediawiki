'''北九州市'''（{{jpn|j=北九州市|hg=きたきゅうしゅうし|rm=Kitakyūshū shi|f=y}}）是[[日本|日本]][[福岡縣|福岡縣]]北部的一個[[政令指定都市|政令指定都市]]，位於[[九州_(日本)|九州島]]最北端，隔[[關門海峽|關門海峽]]與[[本州|本州]]相望。北九州是[[北九州工業地帶|北九州工業地帶]]的中心都市，在1963年由[[門司市|門司市]]、[[小倉市|小倉市]]、[[戶畑市|戶畑市]]、[[八幡市_(福岡縣)|八幡市]]、[[若松市|若松市]][[市町村合併|合併]]誕生，是[[日本三大都市圈|日本三大都市圈]]之外的首個政令指定都市，也是首個非都道府縣廳所在地的政令指定都市，現在北九州市仍是西日本非都道府縣廳所在地城市中人口最多的城市<ref name="Saikaihatsu">{{cite web|url=http://www.reinet.or.jp/?page_id=5109　|script-title=ja:再開発で施設整備進む　～人の流れや運営に課題も～|accessdate=2017-04-14|publisher={{lang|ja|日本不動産研究所}}|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20180327031853/http://www.reinet.or.jp/?page_id=5109%E3%80%80|archivedate=2018-03-27|title=存档副本}}</ref>。

北九州市是九州地方僅次於[[福岡市|福岡市]]的第二大城市，和關門海峽對岸的[[山口縣|山口縣]][[下關市|下關市]]共同組成了{{link-ja|關門都市圈|関門都市圏}}<ref name="Kanmon">{{cite web |url=https://www.kitakyu-u.ac.jp/iurps/pdf/kan17_3.pdf　 |script-title=ja:第３章 関門大都市圏における最近の地域構造の変化と行政境界 |accessdate=2017-04-20 |publisher={{lang|ja|北九州市立大学}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。地處九州玄關口的北九州市是九州主要的公路和鐵路起點，堪稱陸上交通的樞紐。臨關門海峽的地理位置也使得北九州市成為海上交通的要衝、重要的物流及港灣都市。北九州憑藉鋼鐵工業而確立了工業都市的地位，但由於公害問題和產業構造的變化，北九州市的人口流出問題嚴重，自1980年代之後長期陷入人口減少的局面<ref name="Saikaihatsu"></ref>。現在北九州市的環境狀況已經得到改善，並且被[[經濟合作與發展組織|經濟合作與發展組織]]認定為亞洲首個綠色成長模範城市<ref>{{cite web |url=http://www.city.kitakyushu.lg.jp/kankyou/00500002.html　 |script-title=ja:OECDグリーンシティプログラム |accessdate=2017-04-20 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 歷史 ==
=== 古代史 ===

name="UImy">{{cite web |url=http://www.ui-my.com/kita02.html　 |script-title=ja:北九州市の歴史 |accessdate=2017-04-21 |publisher={{lang|ja|UImy北九州倶楽部}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。646年時（[[大化_(日本)|大化]]2年），政府在現在的和布刈神社附近設立門司關，是現在門司區地名的由來<ref>{{cite web |url=http://www.crossroadfukuoka.jp/event/?mode=detail&id=4000000001241&isSpot=1　 |script-title=ja:門司ヶ関址碑 |accessdate=2017-04-21 |publisher={{lang|ja|福岡県観光連盟}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20180327031853/http://www.crossroadfukuoka.jp/event/?mode=detail&id=4000000001241&isSpot=1%E3%80%80 |archivedate=2018-03-27 }}</ref>。1185年（[[元曆|元曆]]2年/[[壽永|壽永]]4年），[[源平合戰|源平合戰]]的最後一役[[壇之浦之戰|壇之浦之戰]]就發生在北九州市和下關市之間的關門海峽<ref>{{cite web |url=https://shimonoseki.travel/story/dannoura/index.html　 |script-title=ja:壇ノ浦の合戦 1403年（[[應永|應永]]10年），{{link-ja|大內盛見|大内盛見}}被任命為豐前守護。自此至16世紀中期，北九州一帶被劃入[[大內氏|大內氏]]的勢力範圍<ref>{{cite web |url=http://oouchibunka.jp/ouchishi/outline/06.html　 |script-title=ja:義興・義隆の時世 |accessdate=2017-04-21 |publisher={{lang|ja|歴史の町山口を甦らせる会}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1551年（[[天文|天文]]20年），大內氏遭到家臣[[陶晴賢|陶晴賢]]謀反，實權被陶晴賢所奪。此後[[大友義鎮|大友義鎮]]趁機攻入曾是大內氏領地的豐前和筑前地區，成為九州最大的勢力<ref>{{cite web |url=http://www.city.usuki.oita.jp/docs/2014020600551/　 |script-title=ja:大友宗麟 |accessdate=2017-04-21 |publisher={{lang|ja|臼杵市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1587年（[[天正_(日本)|天正]]15年），[[豐臣秀吉|豐臣秀吉]]在平定九州之後將小倉六万石封給[[毛利勝信|毛利勝信]]<ref>{{cite web |url=http://senjp.com/kathunaga/　 |script-title=ja:毛利勝永と毛利勝信【大阪の陣で義を果たした名将】 |accessdate=2017-04-21 |publisher={{lang|ja|戦国武将1000記事 }} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[關原之戰|關原之戰]]之後，[[細川忠興|細川忠興]]成為新的北九州一帶的統治者。他在1602年（[[慶長|慶長]]7年）修築[[小倉城|小倉城]]，開啟了小倉城下町的歷史<ref name="Kokurahan">{{cite web |url=http://www.city.kitakyushu.lg.jp/kokurakita/file_0082.html　 |script-title=ja:小倉城と小倉藩 |accessdate=2017-04-21 |publisher={{lang|ja|歴史の町山口を甦らせる会}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1632年（[[寛永|寛永]]9年），[[小倉藩|小倉藩]]第二代藩主[[細川忠利|細川忠利]]被轉封到[[熊本藩|熊本]]，[[小笠原忠真|小笠原忠真]]成為新的小倉藩主，此後直到幕末為止，{{link-ja|小笠原氏|小笠原氏}}都是北九州地區的統治者<ref name="Kokurahan"></ref>。

=== 明治維新到五市合併 ===
[[File:Governmental_Yawata_Iron_&_Steel_Works.JPG|thumb]]]]
[[明治維新|明治維新]]之後，日本在1871年實施[[廢藩置縣|廢藩置縣]]，北九州地區被劃入[[小倉縣|小倉縣]]。不過僅在5年之後小倉縣的大部分地區就被併入[[福岡縣|福岡縣]]，此後北九州一帶一直屬於福岡縣<ref name="UImy"></ref>。地處交通和軍事要地的北九州得到了中央政府的高度重視。1889年門司港被指定為特定輸出港<ref name="Moji">{{cite web |url=http://www.city.kitakyushu.lg.jp/files/000063488.pdf　 |script-title=ja:明治時代 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>和1891年門司車站（現[[門司港車站|門司港車站]]）至高瀨車站（現[[玉名車站|玉名車站]]）之間的鐵路開通<ref>{{cite web |url=http://www.k-rhm.jp/history.html　 |script-title=ja:九州鉄道の歴史 |accessdate=2017-04-21 |publisher={{lang|ja|九州鉄道記念館}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>意味著北九州成為九州的玄關口。1898年[[日本皇軍|日本皇軍]]將[[第12師團|第12師團]]總部設在小倉<ref>{{cite web |url=http://www.city.kitakyushu.lg.jp/kokurakita/file_0081.html　 |script-title=ja:小倉と日露戦争 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>和1901年[[八幡製鐵所|八幡製鐵所]]開業<ref>{{cite web |url=http://www.japansmeijiindustrialrevolution.com/site/yawata/component01.html　 |script-title=ja:官営八幡製鐵所 |accessdate=2017-04-21 |publisher={{lang|ja|明治日本の産業革命遺産}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>則象徵北九州開始成為「軍都」和「重工業都市」。西日本最大的要塞{{link-ja|下關要塞|下関要塞}}的部分區域位於北九州市沿海地區，小倉城和現在若松區的沿海地區曾長期禁止普通民眾進入，可見北九州的軍事地位<ref>{{cite web|url=http://www.fsg.pref.fukuoka.jp/kyureki/_common/pdf/ondemand/0201.pdf　|script-title=ja:下関要塞地区域標|accessdate=2017-04-21|publisher={{lang|ja|福岡県}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1899年，[[門司市|門司市]]成為北九州地區最早設[[市制_(日本制度)|市]]的城市<ref name="Moji"></ref>。翌年小倉也順利設市<ref>{{cite web |url=http://www.city.kitakyushu.lg.jp/kokurakita/file_0005.html　 |script-title=ja:小倉市制施行からの歴史 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。進入大正時代之後，若松、八幡、戶畑亦相繼設市<ref name="Moji"></ref>。

自明治到[[第二次世界大戰|第二次世界大戰]]的時期是北九州地區人口快速增加和工業化的時代。在明治末期到大正時期，貿易據點的門司和煉鋼廠所在的八幡是北九州五市中發展最快的地區<ref name="Chishi">{{cite book |author={{lang|ja|野澤秀樹・堂前亮平・手塚章}}|script-title=ja:《日本の地誌10　九州・沖縄》 |year=2008 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-16770-2|language=ja}}</ref>{{rp|165}}。自明治後期至大正初期，八幡是日本人口增長最快的城市。1891年時，尚未興建煉鋼廠的八幡僅有人口2118人，但在1919年時已增加到84682人。八幡在不到30年的時間內人口增加了約40倍，成長為當時九州的第二大城市<ref name="History">{{cite web |url=https://www.kitakyu-u.ac.jp/iurps/pdf/kan20_2-1.pdf　 |script-title=ja:下関市・北九州市における斜面地居住の形成と展開 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市立大学}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。門司在1868年時人口僅有2719人<ref name="Moji"></ref>。但到[[昭和|昭和]]初期，門司已開通了前往中國[[大連|大連]]、[[天津|天津]]等地的航路<ref>{{cite web |url=http://www.city.kitakyushu.lg.jp/files/000063491.pdf　 |script-title=ja:昭和時代（2）戦後 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是連接日本本土和朝鮮半島及中國的結點，一躍成為西日本的貿易和金融中心<ref name="Chishi"></ref>{{rp|165}}。1942年時，門司已成為面積66.7平方公里，人口約有15.2萬人的重要城市<ref name="Moji"></ref>。戶畑在明治中期時只是一個人口約3000人的漁村，但隨著鄰近的八幡地區的工業發展，戶畑的人口也快速增加。特別是1914年[[旭硝子|旭硝子]]和1921年八幡製鐵所在戶畑設廠使得戶畑也發展成為工業城市<ref>{{cite web |url=http://www.city.kitakyushu.lg.jp/tobata/file_0010.html　 |script-title=ja:戸畑区のあゆみ |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1922年時，戶畑的人口已增加到35138人<ref name="History"></ref>。若松在北九州的工業格局中則扮演了煤炭輸出港的角色。1889年時若松僅有人口2764人，但隨著港口的發展和鐵路的開通，若松同樣迎來了人口的急劇增加。1908年時，若松已有人口24742人<ref name="History"></ref>。相對於其他四市，城下町小倉雖然發展速度相對較慢，但同樣得到了鄰近地區飛速發展的波及效果。特別是在進入20世紀之後，隨著第12師團的進駐，小倉的人口也開始急速增加。成立於1933年的{{link-ja|小倉陸軍造兵廠|小倉陸軍造兵廠}}是當時日本最大的兵工廠之一，也是[[氣球炸彈|氣球炸彈]]的主要產地<ref>{{cite web |url=http://www.city.kitakyushu.lg.jp/kokurakita/file_0083.html　 |script-title=ja:小倉陸軍造兵廠 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[File:West_Japan_Industrial_Club.jpg|thumb]]
1942年，連接門司和下關的{{link-ja|關門隧道 (山陽本線)|関門トンネル (山陽本線)|關門隧道}}開通，是世界首條海底隧道，顯示北九州地區在二戰爆發之後仍取得發展<ref>{{cite web |url=http://www.jcca.or.jp/kaishi/236/236_doboku.pdf　 |script-title=ja:世界初の海底トンネル「関門鉄道トンネル」 |accessdate=2017-04-21 |publisher={{lang|ja|建設コンサルタンツ協会}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。然而北九州地區重要的軍事和工業地位亦使其成為美軍空襲的目標。北九州市在二戰期間經歷了三次大規模空襲。1944年6月16日的空襲是美軍首次使用[[B-29超級堡壘轟炸機|B-29超級堡壘轟炸機]]轟炸日本本土，範圍覆蓋整個北九州地區，322人在空襲中遇難<ref name="Attack">{{cite web |url=http://www.city.kitakyushu.lg.jp/files/000763627.pdf　 |script-title=ja:北九州市の空襲被害 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1945年6月29日的空襲則以門司市為主要目標，55人遇難，3616戶住宅被燒毀<ref name="Attack"></ref>。1945年8月8日的空襲以八幡、若松、戶畑三地為目標<ref name="Attack"></ref>。1945年8月9日，美軍原本計劃在小倉投下[[原子彈|原子彈]]，但由於當天小倉天氣陰天加上前一天八幡地區空襲之後的煙霧尚未散去，小倉上空視界不良，因此原子彈投彈地點改為[[長崎市原子彈爆炸|長崎]]，小倉幸運的逃過一劫<ref name="Moji"></ref>。

由於北九州地區五市之間關係密切，因此自20世紀初期開始就有合併構想。1903年時，已有門司、小倉合併的構想和若松、戶畑合併的構想<ref name="Combination">{{cite web |url=http://www.ncbank.co.jp/corporate/chiiki_shakaikoken/furusato_rekishi/kitakyushu/012/kitakyushu_nenpu.html　 |script-title=ja:北九州五市合併年表 |accessdate=2017-04-21 |publisher={{lang|ja|西日本シティ銀行}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1922年時，福岡縣議會提議北九州地區一體化<ref name="Combination"></ref>。1928年，曾有合併若松、戶畑兩市的構想，但因新市政府的所在地而最終擱淺<ref name="Chishi"></ref>{{rp|167}}。1934年，曾有若松、小倉、戶畑三市合併的運動，但因戶畑的消極態度而不了了之<ref name="Combination"></ref>。1943年和1947年亦曾有兩次合併運動，但均因門司的反對而未能實現<ref name="Combination"></ref>。直到1960年，因北九州工業地帶經濟蕭條，合併主張再次升溫。1963年2月10日，五市正式合併，北九州市誕生，並在同年4月1日成為[[政令指定都市|政令指定都市]]<ref name="Combination"></ref>。而在討論合併後新市名時，曾進行了票選，原本票選第一名是「西京市」，第二名才是「北九州市」。但因為有人提出必須只有[[日本史|歷史]]上[[天皇|天皇]]曾經居住過的城市才可以使用「[[首都|京]]」的名稱，因此最後選擇了第二名的「北九州市」<ref>{{cite web |url=http://www.ncbank.co.jp/corporate/chiiki_shakaikoken/furusato_rekishi/kitakyushu/012/01.html　 |script-title=ja:五市合併 |accessdate=2017-04-21 |publisher={{lang|ja|西日本シティ銀行}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

=== 北九州市的誕生 ===
[[File:Riverwalk_Kitakyushu_20130721.jpg|thumb]]
在實現合併之後，1970年代的北九州市迎來了眾多重大建設。1973年[[關門橋|關門橋]]的開通使得關門海峽兩岸的交通更加便捷<ref>{{cite web |url=http://showa.mainichi.jp/comeback/2008/11/post-5860.html　 |script-title=ja:昭和４８年１１月１４日　関門橋、開通 |accessdate=2017-04-21 |publisher={{lang|ja|每日新聞}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，翌年[[山陽新幹線|山陽新幹線]]的通車使得北九州前往[[大阪市|大阪市]]和[[東京|東京]]的陸上交通大幅改善<ref>{{cite web |url=http://www.ikkyo-tekken.org/studies/1993/1993_22.pdf　 |script-title=ja:第2章 山陽新幹線の歴史 |accessdate=2017-04-21 |publisher={{lang|ja|一橋大学鉄道研究会}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1972年新市政府大樓的竣工和1974行政區劃的調整象徵北九州市的市政也在1970年代中期實現了整合<ref>{{cite web |url=http://www.city.kitakyushu.lg.jp/kouhou/file_0007.html　 |script-title=ja:市の歴史1970年～1979年 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在經濟高速成長時期，北九州雖然迅速工業化，但也帶來了嚴重的環境污染問題<ref>{{cite web |url=http://www.city.kitakyushu.lg.jp/kankyou/file_0268.html　 |script-title=ja:産業の隆興と公害の発生 |accessdate=2017-04-23 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[洞海灣|洞海灣]]有「死海」之稱<ref>{{cite web |url=http://www.city.kitakyushu.lg.jp/kankyou/file_0264.html　 |script-title=ja:ばい煙の空、死の海から奇跡の復活 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1970年代之後，在民眾和政府的努力下，北九州的自然環境有了大幅改善<ref>{{cite web |url=http://www.city.kitakyushu.lg.jp/kankyou/file_0269.html　 |script-title=ja:公害克服への取り組み |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。然而另一方面，由於重工業在經濟中的地位不斷下降，以重工業為主的北九州市也面臨嚴重的產業轉型和人口外流的問題。北九州市的人口自1979年開始就不斷減少，企業的人力需求也低於日本平均水準<ref name="Genjo">{{cite web |url=http://www.city.kitakyushu.lg.jp/page/kitakyushulife/topics/2016/docs/072701-02.pdf　 |script-title=ja:北九州市の現状 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1988年，北九州市在合併25週年之際提出了{{link-ja|北九州市文藝復興構想|北九州市ルネッサンス構想}}，將都市發展方針由原本的均衡發展（多核都市論）改為以小倉都心和[[黑崎車站|黑崎站]]副都心為中心的集中型都市，若松、戶畑、{{link-ja|折尾|折尾}}等地則改為地域中心核，施政方針大幅轉換<ref>{{cite web |url=https://ssl.city.kitakyushu.lg.jp/files/000028029.pdf　 |script-title=ja:第2章　都市計画の理念と目標 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。現在北九州市以科技、環保、旅遊等產業為發展重點<ref>{{cite web |url=https://www.kitakyu-u.ac.jp/iurps/pdf/02_1.pdf　 |script-title=ja:産業都市「北九州」再生の軌跡 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市立大学}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，並在2011年被選入綠色亞洲國際戰略綜合特區<ref>{{cite web |url=http://www.city.kitakyushu.lg.jp/files/000108450.pdf　 |script-title=ja:グリーンアジア国際戦略総合特区 |accessdate=2017-04-21 |publisher={{lang|ja|北九州市}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，試圖提升都市競爭力。

== 地理 ==
[[File:Kanmon_Straits_from_space_cropped_rotated_90_degrees_CCW.jpg|thumb]]的衛星地圖。下側是北九州市，上側是下關市。]]
北九州市位於[[九州_(日本)|九州島]]的東北端，市區北側是[[日本海|日本海]]（{{link-ja|響灘|響灘}}），東側是[[瀨戶內海|瀨戶內海]]（{{link-ja|周防灘|周防灘}}）<ref>{{cite web|url=http://www.kitaqport.or.jp/jap/topics/01_p4-27.pdf　|script-title=ja:第１章　北九州市における水際線の現状|accessdate=2017-04-21|publisher={{lang|ja|北九州港}}|language=ja}}{{dead link|date=2017年11月 web|url=http://www.city.kitakyushu.lg.jp/san-kei/file_0047.html　|script-title=ja:洞海湾の歴史|accessdate=2017-04-21|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot 北九州市內有大小260條河川，但除了市區西南界的{{link-ja|遠賀川|遠賀川}}之外沒有較大的河川<ref>{{cite web|url=http://www.city.kitakyushu.lg.jp/kensetu/05100102.html　|script-title=ja:北九州市の河川|accessdate=2017-04-21|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，且因地形多山使得轄區內有眾多[[水庫|水庫]]<ref name="Chikei"></ref>。北九州市的主要島嶼有{{link-ja|藍島|藍島}}、{{link-ja|馬島 (福岡縣)|馬島 (福岡県)|馬島}}<ref>{{cite web|url=http://ijuu-teijuu.pref.fukuoka.lg.jp/municipalities/island/01.html　|script-title=ja:馬島|accessdate=2017-04-21|publisher={{lang|ja|福岡県庁 広域地域振興課}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20180327031853/https://ijuu-teijuu.pref.fukuoka.lg.jp/municipalities/island/01.html%E3%80%80|archivedate=2018-03-27}}</ref>、{{link-ja|白島 (福岡縣)|白島 (福岡県)|白島}}，[[北九州機場|北九州機場]]也興建在一個人工島之上<ref>{{cite web|url=http://www.kitaqport.or.jp/jap/umibe/area/airport.html　|script-title=ja:北九州空港ゾーン （観光・交流拠点エリア）|accessdate=2017-04-21|publisher={{lang|ja|北九州港}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。北九州市雖然是福岡縣面積最大的市町村，但由於地形多山，住宅和工廠多集中在狹小的平原地區。缺乏平地的這一地形特徵也使得北九州市在開發歷史上多次通過填海獲得土地。現在若松區、戶畑區和小倉北區的大部分港灣地區土地都是通過填海所得<ref name="Coastline"></ref>。

由於北九州市是由五市對等合併成立的都市，因此在合併之初，北九州市以「多核都市論」為城市規劃的指導方針，實施均衡發展的策略<ref>{{cite web|url=https://www.kitakyu-u.ac.jp/iurps/pdf/jisedai_3.pdf　|script-title=ja:次世代の都市像|accessdate=2017-04-22|publisher={{lang|ja|北九州市立大学}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。然而由於這一政策成本高昂，加上經濟蕭條，使得均衡發展的方針難以為繼<ref>{{cite web|url=https://www.kitakyu-u.ac.jp/iurps/pdf/2015region_4.pdf　|script-title=ja:北九州の旧五市合併が財政の効率性に与えた影響|accessdate=2017-04-22|publisher={{lang|ja|北九州市立大学}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1988年北九州市提出{{link-ja|北九州市文藝復興構想|北九州市ルネッサンス構想}}之後，將以[[小倉站_(福岡縣)|小倉車站]]為中心的小倉都心和以[[黑崎車站|黑崎車站]]為中心的黑崎副都心為發展重點，由均衡發展路線轉換為集中型都市<ref>{{cite web|url=https://www.mlit.go.jp/crd/index/case/pdf/0905kitakyusyucity.pdf　|script-title=ja:北九州市中心市街地活性化基本計画 （小倉地区・黒崎地区）について|accessdate=2017-04-22|publisher={{lang|ja|国土交通省}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。小倉都心位於[[小倉北區|小倉北區]]，是北九州市的商業、行政中心地區。由於美軍在二戰期間將小倉列入原子彈爆炸目標，因此並未對小倉進行大規模空襲，使得小倉仍然部分保留了江戶時代的[[城下町|城下町]]都市格局<ref name="Hyakunen">{{cite book |author={{lang|ja|平岡昭利}}|script-title=ja:《九州 地図で読む百年》 |year=1997 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=4-7722-1665-0|language=ja}}</ref>{{rp|15}}。黑崎副都心位於[[八幡西區|八幡西區]]，是北九州西部的商業中心。黑崎是日本少數擁有放射型路網格局的市區<ref name="Hyakunen"></ref>{{rp|14}}，城市功能則以住宅和商業區為主<ref name="Plan">{{cite web|url=http://www.pref.fukuoka.lg.jp/uploaded/life/87267_16898382_misc.pdf　|script-title=ja:北九州都市計画|accessdate=2017-04-22|publisher={{lang|ja|福岡県}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。除了兩大中心地區之外，北九州市的主要繁華地區還有門司區的中心街區{{link-ja|門司港|門司港 (地区名)}}<ref>{{cite web|url=http://www.city.kitakyushu.lg.jp/files/000735444.pdf　|script-title=ja:社会資本総合整備計画　北九州市門司港地区都市再生整備計画（第３期）|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；小倉南區西北部的{{link-ja|德力|徳力 (北九州市)}}；地處小倉和黑崎之間，以住宅區為主的戶畑<ref>{{cite web|url=http://www.city.kitakyushu.lg.jp/files/000028208.pdf　|script-title=ja:北九州市都市計画マスタープラン戸畑区構想「戸畑区のまちづくり方針」|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；主要工業區的八幡<ref>{{cite web|url=http://www.city.kitakyushu.lg.jp/files/000028153.pdf　|script-title=ja:北九州市都市計画マスタープラン八幡東区構想「八幡東区のまちづくり方針」|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；正在轉型為住宅區和文教地區的若松<ref>{{cite web|url=http://www.city.kitakyushu.lg.jp/files/000028137.pdf　|script-title=ja:北九州市都市計画マスタープラン若松区構想「若松区のまちづくり方針」|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>等地區。
 
=== 氣候 ===
北九州市的關門海峡和響灘沿岸地區屬於[[太平洋側氣候|太平洋側氣候]]<ref>{{cite web |url=http://www.jma-net.go.jp/fukuoka/kaiyo/tenkou_column1.html　|script-title=ja:【コラム１】九州北部地方（福岡）の気候は日本海型？ |accessdate=2017-04-21|publisher={{lang|ja|福岡管区気象台}} |language=ja}}</ref>，周防灘沿岸地區氣候屬於[[瀨戶内海式氣候|瀨戶内海式氣候]]。冬季的北九州市受到[[日本海|日本海]]的對流雲影響而多陰天，但降水量遠少於日本海沿岸地區<ref name="Weather">{{cite web |url=http://www.jma-net.go.jp/fukuoka/kaiyo/tenkou_main.html　|script-title=ja:九州北部地方の天候の特徴（解説） |accessdate=2017-04-21|publisher={{lang|ja|福岡管区気象台}} |language=ja}}</ref>。春季是北九州一年之中氣溫變動較大的時期。在春季後期日照時間逐漸變長<ref name="Weather"></ref>。關門海峽在早春容易發生濃霧<ref name="Fog">{{cite web|url=http://www.pa.qsr.mlit.go.jp/kanmon/2kanmon/index2-3.html　|script-title=ja:関門海峡の気候の特徴|accessdate=2017-04-21|publisher={{lang|ja|九州地方整備局 関門航路事務所}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。北九州市雖然夏季炎熱，但因位置偏北且臨近海岸，平均氣溫略低於福岡市<ref>{{cite web|url=http://www.tabirai.net/sightseeing/fukuoka/info/about/weather.aspx　|script-title=ja:福岡県の気候|accessdate=2017-04-21|publisher={{lang|ja|たびらい福岡}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20180327031853/http://www.tabirai.net/sightseeing/fukuoka/info/about/weather.aspx%E3%80%80|archivedate=2018-03-27}}</ref>。全年降水亦多集中在夏季。9月時北九州易受颱風影響，同樣是降水較多的時期。進入10月之後隨著高氣壓勢力的增強，北九州進入秋季，多是晴朗的天氣<ref name="Weather"></ref>。海上氣象方面，響灘屬於日本海一部分的，在冬季風浪較大；周防灘則呈現典型的瀨戶內海式特徵，全年都較為平穩<ref name="Fog"></ref>。

== 人口 ==
1920年日本首次人口普查時，北九州地區人口合計有433,325人<ref name="Population1">{{cite web|url=http://www.city.kitakyushu.lg.jp/page/toukei/sougou/nenkan/03/0314.xls　|script-title=ja:国勢調査人口の推移|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1940年時，北九州地區人口大幅增加到818,950人<ref name="Population1"></ref>。雖然二戰導致北九州市人口一度大幅減少，1947年時只有622,417人<ref name="Population1"></ref>。戰後北九州市人口再次開始增長，1950年代初期時已恢復到戰前最高水準<ref name="Population1"></ref>。1963年五市合併時，北九州市有人口1,032,648人，是當時福岡縣和九州地方人口最多的城市，也是日本人口第七多的城市和[[日本三大都市圈|三大都市圈]]以外人口最多的城市<ref>{{cite web|url=http://www.agi.or.jp/reports/report2015-06.pdf　|script-title=ja:北九州の人口動態と都市構造に関する研究|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。然而重工業為主的經濟構造導致北九州長期經濟增長緩慢，使得北九州陷入嚴峻的人口流出問題，人口增長速度也相對緩慢。1979年，北九州市人口達到1,068,415人的峰值之後開始減少，人口數也在這一時期被[[福岡市|福岡市]]超過<ref name="Genjo"></ref>。北九州市的人口減少速度在[[日本泡沫經濟|泡沫經濟]]時期進一步加速，此後雖然在1990年代趨於平緩，但在2000年代之後再次開始加速，並在2005年跌破100萬<ref name="Population2">{{cite web|url=http://www.dbj.jp/pdf/investigate/area/kyusyu/pdf_all/kyusyu0810_01.pdf　|script-title=ja:北九州都市圏構造を考える|accessdate=2017-04-22|publisher={{lang|ja|日本政策投資銀行}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在1965年至2010年期間，北九州市人口減少了7萬人<ref>{{cite web|url=http://www.agi.or.jp/wp-content/uploads/2014/08/icme12-20140723.pdf　|script-title=ja:全国から見た北九州市|accessdate=2017-04-22|publisher={{lang|ja|国際東アジア研究センター}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2010年至2015年期間，北九州市人口減少了1.5萬人，是這五年期間中日本人口減少數最多的自治體<ref>{{cite web|url=http://www.sankei.com/region/news/160227/rgn1602270044-n1.html　|script-title=ja:２７年国勢調査　福岡都市圏“独り勝ち”　北九州市と長崎市は大幅減|accessdate=2017-04-22|publisher={{lang|ja|産経ニュース}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2017年4月1日時，北九州市有人口950,429人<ref name="Population5">{{cite web|url=http://www.city.kitakyushu.lg.jp/files/000766020.xls　|script-title=ja:平成29年4月1日現在の推計人口|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。社會減少是北九州市人口減少的主因。北九州市自1965年開始就是人口流出城市，一度出現過一年內人口流出過萬人的情況，近年幅度雖有所減少但仍呈現人口外流情況<ref name="Population3">{{cite web|url=http://www.city.kitakyushu.lg.jp/files/000733607.pdf　|script-title=ja:北九州市まち・ひと・しごと創生総合戦略|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。北九州市的[[總和生育率|總和生育率]]為1.55，高於日本平均水準，並在政令指定都市中是最高水準<ref>{{cite web|url=http://www.city.kitakyushu.lg.jp/files/000715185.pdf　|script-title=ja:１－２．北九州市の都市構造の現状等|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。但在2003年之後，由於老齡化和少子化的進展，北九州市人口的自然增長率也轉入負成長<ref name="Population2"></ref>。現在北九州市已將增加工作崗位，降低人口減少速度作為市政工作的最重要課題之一<ref name="Population3"></ref>。

北九州市在合併之前，舊五市中以八幡和小倉兩市人口最多。1960年時，八幡市佔北九州地區五市人口的33.8%、小倉市佔29.0%、門司市佔15.4%、戶畑市佔11.0%、若松市佔10.8%<ref name="Population4">{{cite web|url=http://www.city.kitakyushu.lg.jp/page/toukei/sougou/nenkan/03/0301.xls　|script-title=ja:人口の推移|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。小倉地區因平地較多且服務業較為發達，在1980年代中期之後成為北九州市發展的中心，人口比重也明顯升高。2015年時，小倉地區佔北九州總人口的41%、八幡地區佔33.8%、門司區佔10.4%、若松區佔8.6%、戶畑區佔6.2%<ref name="Population4"></ref>。2015年時，北九州市總人口中有外國人11,618人，其中以韓國、朝鮮人最多，中國人次之<ref>{{cite web|url=http://www.city.kitakyushu.lg.jp/page/toukei/sougou/nenkan/03/0306.xls　|script-title=ja:国籍別外国人住民数|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 政治 ==
在日本[[眾議院_(日本)|眾議院]]選舉中，北九州市[[若松區|若松區]]、[[八幡西區|八幡西區]]、[[八幡東區|八幡東區]]、[[戶畑區|戶畑區]]屬於[[福岡縣第9區|福岡縣第9區]]；[[門司區|門司區]]、[[小倉南區|小倉南區]]、[[小倉北區|小倉北區]]屬於[[福岡縣第10區|福岡縣第10區]]<ref>{{cite web|url=http://www.pref.fukuoka.lg.jp/uploaded/life/189617_51482721_misc.pdf　|script-title=ja:福岡県の衆議院小選挙区の区割|accessdate=2017-04-22|publisher={{lang|ja|福岡県}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。就過去的選舉結果來看，福岡9區在過去是[[民主黨_(1998年)|民主黨]]較強的選舉區，但在自2012年開始的三次眾議院選舉中，[[自由民主黨_(日本)|自由民主黨]]{{link-ja|三原朝彥|三原朝彦}}取得勝利<ref>{{cite web|url=http://www.yomiuri.co.jp/election/shugiin/2017/kaihyou/ya40.html　|script-title=ja:【小選挙区】福岡|accessdate=2017-10-24|publisher={{lang|ja|読売新聞}}|language=ja}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。福岡10區則是自由民主黨較強的選舉區。自小選舉區導入以來，除了2009年[[第45屆日本眾議院議員總選舉|第45屆日本眾議院議員總選舉]]之外，自民黨一直在該選區取得勝利。現任福岡10區的眾議員是[[山本幸三|山本幸三]]<ref>{{cite web|url=http://www.pref.fukuoka.lg.jp/uploaded/life/137171_50597419_misc.pdf　|script-title=ja:候補者得票数（選挙区別）|accessdate=2017-04-22|publisher={{lang|ja|平成26年12月14日執行　第47回衆議院議員総選挙及び第23回最高裁判所裁判官国民審査}}|language=ja}}{{dead |url=http://www2.dpj.or.jp/news/?num=9557　|script-title=ja:北九州市長選挙で、北橋健治さんが自公推薦候補らを破り初当選 |accessdate=2017-04-22|publisher={{lang|ja|民主党}} |language=ja}}</ref>。北橋健治在2011年、2015年<ref>{{cite |fix-attempted=yes }}</ref>、2019年的市長選舉中亦順利連任<ref>{{cite 北九州市由於和位於關門海峽對岸的[[下關市|下關市]]往來密切，兩地被合稱為{{link-ja|關門都市圈|関門都市圏}}，現在[[日本道州制議論|日本在探討實施道州制]]的同時，也有人提出將下關市與北九州市合併為獨立於道州之外的關門特別市<ref>{{cite web|url=https://www.kitakyu-u.ac.jp/iurps/pdf/03_4.pdf　|script-title=ja:関門特別市構想の課題と展望に関する予備的考察|accessdate=2017-04-22|publisher={{lang|ja|北九州市立大学}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。由於北九州市以舊有的五個城市為單位興建公共設施，因此在日本所有政令指定都市中，北九州市的人均公共設施面積高居首位<ref>{{cite web |url=http://www.sankei.com/politics/news/140809/plt1408090018-n3.html　 |script-title=ja:市民派市長の限界　大型事業・行革に乏しいスピード感… |accessdate=2017-04-20 |publisher={{lang|ja|産経ニュース}} |language=ja }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是政令指定都市平均水準的1.5倍<ref name="Intro">{{cite web|url=http://www.city.kitakyushu.lg.jp/files/000734155.pdf　|script-title=ja:３ 本市の現状と将来動向|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。但這也帶來財政負擔沉重的問題，北九州市的行政費用也在政令指定都市中高居次席<ref name="Intro"></ref>。

== 經濟 ==
=== 農業 ===
農業在北九州市經濟中比重很小。2010年時，北九州市耕地面積有1,770公頃，農家3,003戶，農業就業人口有2,495人<ref name="Agriculture">{{cite web|url=http://www.city.kitakyushu.lg.jp/san-kei/file_0460.html　|script-title=ja:農業統計|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。北九州市農業呈現都市近郊農業的特點。農產品中生產金額最高的是蔬菜，其次是稻米<ref name="Agriculture"></ref>。北九州市是福岡縣最大的[[西瓜|西瓜]]產地，[[竹筍|竹筍]]、[[茼蒿|茼蒿]]、[[番茄|番茄]]的產量也居福岡縣前列<ref>{{cite web|url=http://www.city.kitakyushu.lg.jp/san-kei/file_0328.html　|script-title=ja:現状、課題、特徴|accessdate=2017-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。 
=== 工業 ===
[[File:140721_Yawata_Steel_Works_from_RRH_Kitakyushu_Japan01s3.jpg|thumb]]
北九州市是[[北九州工業地帶|北九州工業地帶]]的中心都市，也是日本最重要的[[重工業|重工業]]都市之一。2014年，北九州市的市內生產總值有35358億日元，其中第二產業佔約四分之一，而在工業之中又以鋼鐵產業和化工業的比重最高<ref name="Data">{{cite |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。然而由於過於偏重重工業，北九州工業地帶在日本工業領域的地位和過去相比明顯下降。1963年時，北九州市的工業品生產額佔日本的1.9%，2009年時已下降到0.7%<ref == 文化 ==
[[File:The_University_of_Kitakyushu_kitagata_campus.jpg|thumb]]
北九州市雖然是一座自近代才開始迅速興起的城市，但因其地處樞紐的地理位置加上來自各地的移民流入的歷史，使得北九州市文化十分多樣化。這一特徵也體現在北九州市的方言。{{link-ja|北九州方言|北九州弁}}屬於{{link-ja|豐日方言|豊日方言}}。由於北九州市人口大多為來自各地的移民，因此北九州方言是[[九州方言|九州方言]]中音調最為接近標準語的方言，在文法上則受到瀨戶內地方的方言的影響<ref>{{cite web |url=http://www.lib.kobe-u.ac.jp/repository/81001519.pdf |script-title=ja:言語接触の遍在性 |accessdate=2017-04-22 |publisher={{lang|ja|神戸大学}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170422212913/http://www.lib.kobe-u.ac.jp/repository/81001519.pdf |archivedate=2017-04-22 }}</ref>，和福岡市的方言有較大不同。在曾在北九州居住過的作家中，以[[松本清張|松本清張]]和[[森鷗外|森鷗外]]最為著名<ref>{{cite web |url=http://www.city.kitakyushu.lg.jp/kokurakita/file_0096.html |script-title=ja:森鴎外・松本清張 |accessdate=2017-04-22 |publisher={{lang|ja|北九州市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170423062506/http://www.city.kitakyushu.lg.jp/kokurakita/file_0096.html |archivedate=2017-04-23 }}</ref>。森鷗外曾以軍醫身份在小倉工作過，其舊居現在是博物館<ref>{{cite web |url=http://www.gururich-kitaq.com/search/category/detail.php?id=147 |script-title=ja:森鴎外旧居 |accessdate=2017-04-22 |publisher={{lang|ja|北九州市情報発信強化委員会}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170422213224/http://www.gururich-kitaq.com/search/category/detail.php?id=147 |archivedate=2017-04-22 }}</ref>。松本清張的人生中有約一半時間是在北九州市度過，留下了眾多以北九州為舞台的作品。現在小倉城附近設有[[北九州市立松本清張紀念館|北九州市立松本清張紀念館]]，收藏展示有關松本清張創作生涯的展品<ref>{{cite web |url=http://www.gururich-kitaq.com/search/category/detail.php?id=57 |script-title=ja:北九州市立松本清張記念館 |accessdate=2017-04-22 |publisher={{lang|ja|北九州市情報発信強化委員会}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170422212459/http://www.gururich-kitaq.com/search/category/detail.php?id=57 |archivedate=2017-04-22 }}</ref>。

== 觀光 ==
[[File:140721_Former_Moji_Customs_Kitakyushu_Japan03bs5.jpg|right]]
北九州市是一座重工業都市，在過去觀光產業並不發達。但自北九州文藝復興構想提出以來，北九州市已將觀光產業作為重點產業發展，並且積極利用近現代的產業遺產，將其改造為觀光資源<ref name="Chishi"></ref>{{rp|171}}。2015年，有2571.4萬人觀光客造訪北九州，比2014年增加了1.1%<ref name="Tourism">{{cite web|url=http://www.city.kitakyushu.lg.jp/files/000715363.pdf　|script-title=ja:北九州市観光動態調査|accessdate=2016-04-22|publisher={{lang|ja|北九州市}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。但北九州市的觀光產業也面臨諸多嚴峻課題，如熱門景點之一的{{link-ja|宇宙世界|スペースワールド}}在2017年宣告關閉<ref>{{cite web |url=http://www.nikkei.com/article/DGXLASDZ16I76_W6A211C1TI5000/　|script-title=ja:スペースワールド17年末閉園「経営難が理由でない」|accessdate=2016-04-22|publisher={{lang|ja|日本経済新聞}} |language=ja}}</ref>。[[小倉城|小倉城]]和小倉城庭園是北九州市最具代表性的古蹟之一。現在的小倉城天守重建於1959年，是小倉的地標之一<ref>{{cite web|url=http://www.kokura-castle.jp/about/　|script-title=ja:小倉城について|accessdate=2016-04-22|publisher={{lang|ja|小倉城}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。小倉城附近的{{link-ja|RiverWalk北九州|リバーウォーク北九州}}則是北九州規模最大的購物中心之一，並集合了北九州藝術劇場和北九州市立美術館分館，是小倉地區最大的商業和觀光設施之一<ref>{{cite web|url=http://riverwalk.co.jp/information/about1/　|script-title=ja:リバーウォーク北九州について|accessdate=2016-04-22|publisher={{lang|ja|リバーウォーク北九州}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。同樣鄰近小倉城的{{link-ja|旦過市場|旦過市場}}的歷史開始於大正時代，代表了北九州商業和飲食文化的傳統一面，有「北九州的廚房」之稱<ref>{{cite web|url=http://www.tangaichiba.jp/about/about01/　|script-title=ja:旦過市場の生い立ち|accessdate=2016-04-22|publisher={{lang|ja|旦過市場}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

北九州的主要博物館有展示有關北九州自然史和人文史的{{link-ja|北九州市立生命之旅博物館|北九州市立いのちのたび博物館}}<ref>{{cite web|url=http://www.kmnh.jp/guide/　|script-title=ja:展示案内|accessdate=2017-04-22|publisher={{lang|ja|北九州市立いのちのたび博物館}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、收藏展示眾多[[蒸汽機車|蒸汽機車]]的{{link-ja|九州鐵道紀念館|九州鉄道記念館}}<ref>{{cite web|url=http://www.kmnh.jp/guide/　|script-title=ja:鉄道記念館とは|accessdate=2017-04-22|publisher={{lang|ja|九州鉄道記念館}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、以教堂為設計概念的{{link-ja|北九州市立美術館|北九州市立美術館}}<ref>{{cite web|url=http://kmma.jp/about/　|script-title=ja:美術館について|accessdate=2017-04-22|publisher={{lang|ja|北九州市立美術館}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。地處九州玄關的門司港保留有眾多明治至昭和前期的西洋式建築。當地政府自1988年開始耗資300億日元對門司的西洋建築進行整修，並將這一地區統一規劃為[[門司港懷舊|門司港懷舊]]，吸引了眾多遊客<ref>{{cite web|url=http://www.retro-mojiko.jp/about/mojiko-retro.html　|script-title=ja:門司港レトロについて|accessdate=2017-04-22|publisher={{lang|ja|門司港レトロ倶楽部}}|language=ja}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。北九州多山的地形使得市內有眾多眺望勝地，其中自[[皿倉山|皿倉山]]看到的夜景有「百億美元的夜景」之稱，被選入新日本三大夜景<ref>{{cite web |url=http://yakei.jp/japan/spot.php?i=sarakura　|script-title=ja:皿倉山の夜景 |accessdate=2017-04-21|publisher={{lang|ja|yakei.jp}} |language=ja}}</ref>。
