'''北大曲站'''（{{jpn|j=北大曲駅|hg=きたおおまがりえき|rm=Kita-ōmagari eki|f=y}}）是一座位於[[秋田縣|秋田縣]][[大仙市|大仙市]]的[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[田澤湖線|田澤湖線]]的鐵路車站。

== 車站構造 ==
一個[[單式月台|單式月台]]，1面1線。設有候車室。為一個[[無人站|無人站]]。

== 历史 ==
* 1965年（昭和40年）11月21日：車站啟用。
* 1987年（昭和62年）4月1日：因[[國鐵分割民營化|國鐵分割民營化]]，本站成為[[東日本旅客鐵道|東日本旅客鐵道]]的車站。

== 車站周邊 ==
* 大仙市政廳四屋出張所
* 農業・食品産業技術綜合研究機構 東北農業研究中心 水田利用部
* [[國道105號|國道105號]]
