'''北九州高速4號線'''（{{lang-ja|'''北九州高速4号線'''}}，假名：きたきゅうしゅうこうそく4ごうせん）是一條連接[[福岡縣|福岡縣]][[北九州市|北九州市]][[八幡東區|八幡東區]][[大谷JCT|大谷JCT]]與同縣同市同區[[枝光出入口|枝光出入口]]的[[北九州高速道路|北九州高速道路]]路線。

== 連接高速道路 ==
* [[北九州高速4號線|北九州高速4號線]]（於[[大谷JCT|大谷JCT]]連接）
* [[黑崎繞道|黑崎繞道]]（於[[東田出入口|東田出入口]]連接）

== 历史 ==
* 2001年7月2日 : 枝光出入口至大谷出入口之間開通。
* 2010年12月15日 : 東田出入口啟用。
* 2012年9月30日 : 東田出入口可以連接[[國道3號|國道3號]][[黑崎繞道|黑崎繞道]]<ref>{{PDFlink|[http://www.qsr.mlit.go.jp/kitakyu/topics/data_file/1344388185_0.pdf 國道3號黑崎繞道與北九州都市高速道路連接]}}（國土交通省九州地方整備局北九州國道事務所，2012年8月8日參閲）</ref>。
