'''北81線 中埔－山員潭子'''，是位於[[新北市|新北市]]的一條鄉道，東起[[新北市|新北市]][[三峽區|三峽區]]中埔里大同路與民權街、中園街口，西至新北市[[三峽區|三峽區]]弘道路末端，全長6公里。{{r|thb}}

== 路線說明 ==
;'''[[新北市|新北市]]'''
*[[三峽區|三峽區]]：'''中埔'''（起點，[[File:TW_THWtp85.png|30px]][[北85線|北85線]]民權路與中園街岔路）→ 大同路 → '''麻園'''（[[File:TW_THWtp81-1.png|30px]]北81-1線岔路） → 弘道路 → '''福德坑''' → '''福德''' → '''山員潭子'''（終點）

== 支線 ==
北81線唯一的支線為'''北81-1線'''，是位於[[新北市|新北市]]的一條鄉道，即三峽區麻園路和二鬮路。北起新北市三峽區大同路口，南至新北市三峽區中正路口，全長2.58公里。{{r|thb}}
