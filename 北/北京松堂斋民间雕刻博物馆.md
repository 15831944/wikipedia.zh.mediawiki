'''北京松堂斋民间雕刻博物馆'''，也称'''北京松堂斋博物馆'''或'''北京松堂博物馆'''，是一家位于[[中华人民共和国|中华人民共和国]][[北京市|北京市]][[东城区|东城区]][[国子监街|国子监街]]3号院的民营博物馆，成立于2001年10月<ref>{{cite web|title=北京松堂斋民间雕刻博物馆|url=http://culture.bjdch.gov.cn/n11892200/n12287670/12358947.html|work=文化东城|publisher=北京市东城区人民政府|accessdate=2013-11-30|deadurl=yes|archiveurl=https://web.archive.org/web/20131203023153/http://culture.bjdch.gov.cn/n11892200/n12287670/12358947.html|archivedate=2013-12-03}}</ref>，2001年11月正式开馆<ref name=grrb/>（一说2002年2月2日正式开放<ref>{{cite news|title=北京松堂斋博物馆开放|url=http://www.gmw.cn/01gmrb/2002-02/06/05-59518ED68846AD9948256B5700838C35.htm|accessdate=2013-11-30|newspaper=光明日报|date=2002-02-06}}</ref>）。“北京松堂斋民间雕刻博物馆”是博物馆在中华人民共和国[[国家文物局|国家文物局]]注册的名字。按照博物馆的介绍，“北京松堂斋博物馆”包括“北京最具特色四合院”、“三雕艺术博物馆”、“元青花博物馆”三部分。

== 历史 ==
北京松堂斋民间雕刻博物馆建馆之初，馆址设在北京市[[宣武区|宣武区]][[琉璃厂东街|琉璃厂东街]]14号，[[琉璃厂|琉璃厂]]东街14号原是[[明|明]][[清|清]]两代时的[[中庸书局|中庸书局]]，[[中华民国|民国]]时改名为[[谊民书店|谊民书店]]<ref>{{cite news|title=北京松堂斋民间雕刻博物馆中国网/博睿中天|url=http://www.china.com.cn/chinese/2005/wybwgzl/866442.htm|accessdate=2013-11-30|newspaper=中国网/博睿中天|date=2005-05-19}}</ref>，两层，面积254平方米。据李松堂自述，此处是他家的私宅，他花了520万元[[人民币|人民币]]将其翻修为博物馆<ref name=bjwb/>。博物馆票价10元<ref name=grrb>{{cite news|title=民营博物馆何时告别尴尬？|url=http://news.sina.com.cn/c/2003-06-08/1014199206s.shtml|accessdate=2013-11-30|newspaper=工人日报天讯在线|date=2003-06-08}}</ref>。后来，一段时间内，李松堂的博物馆实行“观众自定票价”，即观众先参观，再付博物馆票款，票价由参观者自行决定<ref name=bjwb>{{cite news|title=松堂斋博物馆典藏千年建筑|url=http://www.bj.xinhuanet.com/bjpd_sdwm/2005-12/26/content_2778022.htm|accessdate=2013-11-30|newspaper=北京晚报|date=2005-12-26|deadurl=yes|archiveurl=https://web.archive.org/web/20131204023009/http://www.bj.xinhuanet.com/bjpd_sdwm/2005-12/26/content_2778022.htm|archivedate=2013-12-04}}</ref>。2008年，博物馆搬迁至国子监街3号院。

== 展品 ==
北京松堂斋民间雕刻博物馆以石雕、木雕、砖雕等收藏为主。李松堂爱好石雕等建筑构件，馆藏的石雕等物是他自己四处收集而得，来自中国多个省区，好多砖雕来自北京拆掉的[[四合院|四合院]]。<ref>{{cite news|title=对古民间雕刻的“临终关怀”|url=http://news.sina.com.cn/s/2004-11-30/15485079913.shtml|accessdate=2013-11-30|newspaper=时代人物周报|date=2004-11-30}}</ref>主要展品有[[宋|宋]]墓砖雕“行孝图”、[[元|元]]代“[[胡人|胡人]]饮兽图”门墩、清代石雕梁“皇帝巡游图”、清[[道光|道光]]年间制作的伦理透雕屏风、[[山西|山西]]砖雕门楼、北京木雕门楼、[[江西|江西]]石雕门楼等<ref>{{cite news|title=李松堂和他的松堂斋(组图)|url=http://news.sina.com.cn/o/2005-04-12/04075621601s.shtml|accessdate=2013-11-30|newspaper=人民网-人民日报海外版|date=2005-04-12}}</ref>。

== 争议 ==
松堂斋博物馆还设有[[元青花|元青花]]馆。馆长李松堂称，他收藏了89件元[[青花瓷|青花瓷]]真品，这些瓷器均购自国外（包括[[欧洲|欧洲]]、[[北美|北美]]、[[西亚|西亚]]、[[日本|日本]]等地），曾在[[加拿大|加拿大]][[温哥华|温哥华]]展出，馆藏数量已超过[[土耳其|土耳其]]的[[托卡比皇宫|托卡比皇宫]]与[[伊朗|伊朗]][[德黑兰|德黑兰]]的[[国家博物馆|国家博物馆]]<ref>{{cite news|title=面对元青花的思考 ——北京松堂博物馆北美元青花藏品展后记|url=http://epaper.idoican.com.cn/bjtjb/html/2011-11/17/content_3123656.htm|accessdate=2013-11-30|newspaper=团结报|date=2011-11-17|deadurl=yes|archiveurl=https://web.archive.org/web/20131203012843/http://epaper.idoican.com.cn/bjtjb/html/2011-11/17/content_3123656.htm|archivedate=2013-12-03}}</ref><ref>{{cite news|title=元青花北美藏品展在国子监北京松堂斋博物馆(图)|url=http://money.cnfol.com/110704/160,1549,10192622,00.shtml|accessdate=2013-11-30|newspaper=京华时报|date=2011-07-04}}</ref>。然而，通常的说法是：全球元青花存世量不足300件；土耳其托普卡比博物馆藏有40件青花瓷，位居世界第一；伊朗德黑兰国家博物馆收藏32件，排在世界第二；中国[[江西省|江西省]][[高安市博物馆|高安市博物馆]]藏19件，列世界第三、中国第一<ref>{{cite news|title=高安：中国元青花的首藏之地|url=http://culture.gmw.cn/2012-10/12/content_5349656.htm|accessdate=2013-11-30|newspaper=光明网|date=2012-10-12|deadurl=yes|archiveurl=https://web.archive.org/web/20131203130509/http://culture.gmw.cn/2012-10/12/content_5349656.htm|archivedate=2013-12-03}}</ref>。
