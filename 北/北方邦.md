'''北方邦'''（{{lang-hi|उत्तर प्रदेश}}；[[拉丁字母|拉丁字母]]转写：{{Unicode|Uttar Pradesh}}）处于[[印度|印度]]北部，和[[尼泊爾|尼泊爾]]接壤，毗鄰國內的[[北阿坎德邦|北阿坎德邦]]、[[喜马偕尔邦|喜马偕尔邦]]、[[哈里亚纳邦|哈里亚纳邦]]、[[拉贾斯坦邦|拉贾斯坦邦]]、[[中央邦|中央邦]]以及[[比哈尔邦|比哈尔邦]]。人口逾两亿，是印度[[人口|人口]]最多的[[印度的邦和中央直辖区|邦]]，亦是世界人口最多的[[一級行政區|一級行政區]]。

== 地理 ==
北方邦境內有數條河流和[[運河|運河]]，南部邊界多山，但大部分土地為低窪的[[沖積平原|沖積平原]]。土地的灌溉主要依靠雨水、地下水、水井及河流的堰堤。氣候溫和，一年兩收，主要收成有[[小麥|小麥]]、[[蔗糖|蔗糖]]、[[大米|大米]]、[[小米|小米]]、[[黃麻|黃麻]]、[[煙草|煙草]]及[[棉花|棉花]]等。建築用石及玻璃工業用的高質量沙石產量充足。鐵路公路四通八達，亦有多所大學。官方語言為[[印地語|印地語]]及[[烏爾都語|烏爾都語]]。 
== 歷史 ==
北方邦於1833年稱[[孟加拉管區|孟加拉管區]]，不久分為[[亞格拉省|亞格拉]]及[[阿瓦德省|阿瓦德]]兩省。1877年處於統一治理之下。1902年稱亞格拉與阿瓦德省，1935年時改名聯合省，1950年改現名。

== 經濟 ==
北方邦經濟以農業為主，北方邦為印度最大的糧食產地。工業集中於蔗糖和棉花加工業，產有紙張、化學品、紡織品、食用油、皮革、手工針織品及糖。礦產有[[煤|煤]]、[[葉臘石|葉臘石]]、[[磷灰鹽|磷灰鹽]]、[[石灰石|石灰石]]、[[銅|銅]]、[[矽石|矽石]]及[[鋁礬土|鋁礬土]]。

== 名勝古蹟 ==
北方邦有许多名胜[[古迹|古迹]]，如[[瓦臘納西|瓦臘納西]]、[[泰姬陵|泰姬陵]]等。[[桑伽施|桑伽施]]為[[佛教八大聖地|佛教八大聖地]]之一，相傳為佛陀由[[忉利天|忉利天]]返回之處，西部法魯卡巴德地區的[[卡瑙季|卡瑙季]]（Kanauj），《[[大唐西域記|大唐西域記]]》稱之為[[曲女城|曲女城]]，[[戒日王|戒日王]]曾於曲女城為[[玄奘|玄奘]]舉辦無遮大會確立了“真唯識量”，還記載了與[[無著|無著]]和[[世親|世親]]論師有關的古城[[阿约提亚|阿约提亚]]。
