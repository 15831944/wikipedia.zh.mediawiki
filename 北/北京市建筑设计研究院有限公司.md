'''北京市建筑设计研究院有限公司'''（简称BIAD），位于[[北京市|北京市]][[西城区|西城区]][[南礼士路|南礼士路]]62号，是[[北京市人民政府国有资产监督管理委员会|北京市人民政府国有资产监督管理委员会]]监管企业。

==简介==
北京市建筑设计研究院有限公司是北京市人民政府出资并按照《[[中华人民共和国公司法|中华人民共和国公司法]]》设立的国有独资公司，是[[北京市人民政府国有资产监督管理委员会|北京市人民政府国有资产监督管理委员会]]监督管理的一级企业'<ref name=jieshao/>。

1949年，在[[中华人民共和国|中华人民共和国]]成立前夕，'''公营永茂建筑公司设计部'''成立。1951年6月，改为'''公营永茂设计公司'''。1952年4月，改为'''北京市建筑公司设计部'''，隶属北京市建筑公司。1953年3月，改为'''北京市建筑工程局设计院'''，隶属北京市建筑工程局。1954年11月，改为'''北京市设计院'''。1955年6月，改为'''北京市规划管理局设计院'''，隶属北京市规划管理局。1960年4月，改为'''北京市建筑设计院'''。1989年7月，改为'''北京市建筑设计研究院'''<ref name=jieshao/>。1992年，成为[[改革开放|改革开放]]后最早进入国际建筑市场的[[中国内地|中国内地]]设计单位之一。2003年，成为北京市人民政府国有资产监督管理委员会监督管理的企事业单位之一<ref name=wy/>。2012年6月，经过转企改制，改为'''北京市建筑设计研究院有限公司'''<ref name=jieshao>{{Cite web |url = http://www.biad.com.cn/page.php?id=13 |title = 企业介绍 |author =  |date =  |publisher = 北京市建筑设计研究院有限公司 |language =  |accessdate = 2017-06-01 |quote =  }}</ref>。

1950年代，设计了[[人民大会堂|人民大会堂]]、[[中国国家博物馆|中国革命历史博物馆]]、[[民族文化宫|民族文化宫]]、[[中国人民革命军事博物馆|中国人民革命军事博物馆]]、[[工人体育场|工人体育场]]。1960年代至1970年代，设计了[[工人体育馆|工人体育馆]]、[[首都体育馆|首都体育馆]]、[[北京饭店|北京饭店]]、[[毛主席纪念堂|毛主席纪念堂]]。1980年代，设计了[[中国国际展览中心|中国国际展览中心]]、[[昆仑饭店|昆仑饭店]]、[[1990年亚运会|第11届亚运会]]场馆。1990年代，设计了[[首都国际机场|首都国际机场]]2号航站楼、[[北京西站|北京西站]]、[[恒基中心|恒基中心]]、[[东方广场|东方广场]]、国际金融大厦、[[北京植物园|北京植物园]]温室、[[西单文化广场|西单文化广场]]。2000年代，设计了[[国家大剧院|国家大剧院]]、首都国际机场3号航站楼、[[中国电影博物馆|中国电影博物馆]]、[[中国科技馆|中国科技馆]]、[[中国石油|中国石油]]大厦、[[中国工商银行|中国工商银行]]总行办公楼、[[凯晨世贸中心|凯晨广场]]、[[联想|联想]]研发基地、[[北京电视台|北京电视中心]]、[[全国人大机关办公楼|全国人大机关办公楼]]、[[中华人民共和国商务部|商务部]]办公楼改造、[[清华大学美术学院|清华大学美术学院]]，以及[[国家体育馆|国家体育馆]]、[[华熙LIVE·五棵松|五棵松文化体育中心]]、[[国家会议中心_(中国)|奥林匹克公园国家会议中心]]、奥林匹克公园中心景观及下沉广场等[[2008年北京奥运会|2008年北京奥运会]]工程，另外在中国其他城市设计了[[深圳文化中心|深圳文化中心]]、[[海南|海南]][[博鳌亚洲论坛|博鳌亚洲论坛]]、[[三亚喜来登酒店|三亚喜来登酒店]]、[[浙江|浙江]][[中国美术学院|中国美术学院]]、[[新疆体育中心|新疆体育中心]]、[[河南省体育场|河南省体育场]]、[[青岛国际帆船中心|青岛国际帆船中心]]、[[广州站|广州站]]、[[南京南站|南京南站]]、[[深圳宝安国际机场|深圳宝安国际机场]]航站楼、[[昆明长水国际机场|昆明长水国际机场]]航站楼、[[福州|福州]][[海峡国际会展中心|海峡国际会展中心]]、[[上海世博会|上海世博会]]项目等<ref name=wy>{{Cite web |url = http://news.163.com/12/0613/02/83RL0PQV00014AED.html |title =市建筑设计研究院转企改制  |author =  |date = 2012-06-13 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://cul.sohu.com/20091020/n267567321.shtml |title = 北京市建筑设计研究院简介 |author =  |date = 2009-10-20 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。
