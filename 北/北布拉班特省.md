'''北布拉班特省'''（{{lang-nl|Noord-Brabant}}）是荷兰的一个省，地处该国南部，南面与[[比利时|比利时]]交接，[[马斯河|马斯河]]位其北面，东靠[[林堡省_(荷兰)|林堡省]]，西连[[西兰省|西兰省]]。省会城市是[[斯海尔托亨博斯|斯海尔托亨博斯]]，最大城市是[[埃因霍溫|埃因霍溫]]。(可以查看 [http://maps.google.com/?ie=UTF8&om=1&z=9&ll=51.505323,5.045471&spn=0.820606,2.570801&t=h Google maps].)

== 历史 ==
在17世紀之前，現今的北布拉班特省的大部分區域都是[[布拉班特公國|布拉班特公國]]的一部分，布拉班特公國的南部亦即是現今的[[比利時|比利時]]。14及15世紀，這地區經歷了黃金時期，特別是[[魯汶|魯汶]]、[[安特衛普|安特衛普]]（這兩座城市都位於現今的比利時）、[[布雷達|布雷達]]及[[斯海爾托亨博斯|斯海爾托亨博斯]]。

[[烏特勒支聯盟|烏特勒支聯盟]]在1579年成立，[[荷蘭共和國|荷蘭共和國]]與[[西班牙|西班牙]]在荷蘭南部爆發戰爭，布拉班特公國成為戰場。根據[[西發里亞和約|西發里亞和約]]的協議，布拉班特公國的北部劃歸荷蘭所有，屬布拉班特公國的一部分，歸聯邦統治（荷蘭共和國的省份都是自治的）。

這地區作為一個軍事緩衝區，1796年，法國在荷蘭建立[[巴達維亞共和國|巴達維亞共和國]]，布拉班特公國成為該國的一個省份。法國的重組打破了這一狀況，這裡被分為多個區域。

1815年，比利時及荷蘭組成[[荷蘭聯合王國|荷蘭聯合王國]]，建立北布拉班特省，以區分比利時的南布拉班特省，比利時在1830年脫離荷蘭聯合王國。比利時及荷蘭的邊界很特別，兩國的邊界存在著一些飛地，如[[巴勒海托赫|巴勒海托赫]]。北布拉班特省的邊界擴大到[[荷蘭_(地區)|荷蘭省]]的部分地區及[[拉文斯坦|拉文斯坦]]，還有多個有自治權的地區。拉文斯坦以往是隸屬[[克里維斯公國|克里維斯公國]]的。

19世紀末，北布拉班特省逐漸工業化。[[飛利浦公司|飛利浦公司]]及[[DAF|DAF]]都在[[埃因霍温|埃因霍温]]設有廠房，使埃因霍温成為國家第五大城市，[[蒂爾堡|蒂爾堡]]和[[黑爾蒙|黑爾蒙]]是紡織業的中心。

== 地理 ==
與大部分荷蘭的地區一樣，北布拉班特省的地勢平坦。大部分的人口都居住在城區，其他土地作農業用途，大部分地區都無人居住。森林、丘地都可見於北布拉班特省。

北布拉班特省以[[默茲河|默茲河]]為北界，默茲河流經國家公園[[比斯博斯|比斯博斯]]。

== 经济 ==
北布拉班特省以農業、工業及服務行業為主。主要的農產品是[[小麥|小麥]]及[[甜菜|甜菜]]，畜養牛隻及豬隻。工業以汽車製造、電子、紡織及製鞋為支柱。

踏入二十世紀，旅遊業成為北布拉班特省其中一項重要的經濟活動，北布拉班特省的森林及恬靜美麗的環境致使旅遊業得以發展。另外，位於[[卡茨赫佛爾|卡茨赫佛爾]]（Kaatsheuvel）的比荷盧關稅聯盟最大的主題公園[[艾夫特琳|艾夫特琳]]（Efteling）亦是北布拉班特省的旅遊熱點。

== 政治 ==
北布拉班特省的地方議會共有七十九席，以女王專員為首，現時北布拉班特省的女王專員是[[漢伊·邁韋亨|漢伊·邁韋亨]]。地方議會是由居民選舉產生的，女王專員由女王及荷蘭內閣指派。基督教民主黨現時在北布拉班特省地方議會得二十九席，是議會的最大政黨。

北布拉班特省的日常事務由''Gedeputeerde Staten''負責，向女王專員負責，其成員等同於大臣（部長）。
