'''北戴河办公制度'''，是[[中国共产党中央委员会|中共中央]][[1954年|1954年]]至[[1965年|1965年]]以及[[1984年|1984年]]至[[2002年|2002年]]间每年[[夏季|夏季]]在[[秦皇岛市|秦皇岛]][[北戴河区|北戴河]]办公的制度。2003年[[胡锦涛|胡锦涛]]出任[[中国共产党中央委员会总书记|总书记]]后，中共中央废除该制度。[[习近平|习近平]]出任总书记后，再次开始召开北戴河会议<ref>{{cite news |author1=方冰 |title=习近平北戴河会议遭遇什么挑战？ |url=https://www.voachinese.com/a/wu-guoguang-bei-daihe-20170809/3979082.html |accessdate=2018-08-07 |agency=美国之音 |date=2017年8月10日}}</ref>。

==简介==
1953年秋，中央决定每年夏季在北戴河办公，北戴河办公制度正式形成<ref name=nfzm/>。中共中央确定在北戴河夏季办公，实际带有半休养半工作性质<ref name="gmw">{{Cite web|url=http://history.gmw.cn/2011-09/07/content_2601777.htm|title=中央领导人的北戴河暑期办公制度|date=2011-09-07|publisher=光明网|access-date=2018-02-05|archive-url=https://web.archive.org/web/20180206131545/http://history.gmw.cn/2011-09/07/content_2601777.htm#|archive-date=2018-02-06|dead-url=yes}}</ref>。同年冬，[[中共中央办公厅|中共中央办公厅]]决定筹建[[中共中央办公厅警卫局|中共中央办公厅警卫局]]北戴河管理处，为中央暑期在北戴河办公服务<ref name="ddzg">{{Cite book|title=红墙记忆|author=王凡、东平|publisher=当代中国出版社|year=2009年|isbn=7801708652|quote=第五章 神秘的北戴河中央别墅区}} </ref>。1954年11月，[[中华人民共和国国务院|国务院]]确定北戴河休养区的使用方针是：“为中央暑期办公服务。”为保障中央领导办公，1954年成立了北戴河暑期工作委员会，下设暑期工作办公室<ref name="nfzm">{{Cite web|url=http://www.southcn.com/news/china/zgkx/200307240926.htm|title=南方周末:中央取消北戴河办公制度对当地有何影响|date=2003-07-24|publisher=南方网}}</ref>。1957年新组建的[[中共中央直属机关北戴河服务局|中共中央直属机关事务管理局北戴河管理处]]管理整个别墅区。当时别墅区有新旧楼房183处，总建筑面积超过12.5万平方米，占地面积超过3平方公里<ref name=ddzg/>。

从1953年到1965年，夏季的中央重要会议几乎都在北戴河召开<ref name="liren">{{Cite web|url=http://politics.people.com.cn/n/2014/0812/c1001-25448429.html|title=历任领导人在北戴河如何疗养：工作休息两不误|date=2014-08-12|publisher=人民网}}</ref>。1966年[[文化大革命|文革]]爆发，文革期间北戴河办公制度中断，暑期工作委员会也随着文革爆发而被取消<ref name=nfzm/>。

1983年，国务院领导到[[秦皇岛市|秦皇岛]]视察，表明暑期中共中央、国务院要在北戴河办公。1984年，北戴河办公制度恢复。该年，来北戴河办公休息的中央领导47人，部、省、军级以上领导379人。同年，[[中国人民政治协商会议全国委员会|全国政协]]北戴河休养所和[[全国人民代表大会|全国人大]]北戴河休养所先后建成，1985年开始各自接待本部门领导办公休养。1985年，中央五大班子（[[中国共产党中央委员会|中共中央]]、[[中华人民共和国国务院|国务院]]、[[全国人民代表大会|全国人大]]、[[中国人民政治协商会议全国委员会|全国政协]]、[[中央军事委员会|中央军委]]）包括[[中国共产党中央顾问委员会|中顾委]]的领导都来北戴河办公休养<ref name=nfzm/>。此后直到2002年，北戴河每年夏季都是中共中央、国务院、中央军委、全国人大常委会、全国政协等单位处理内政外交的重要地点<ref name=ruhe/>。

2003年以[[胡锦涛|胡锦涛]]为首的新一届国家领导班子上任不久，2003年7月19日[[人民网|人民网]]发布消息称：“中央已经决定，今年夏季，中共中央、国务院、全国人大、全国政协、中央军委五大领导班子将不到北戴河办公。中央各部委负责人在此期间外出和休假，均须严格按照有关规定执行，不得擅自去北戴河等避暑胜地。”北戴河办公制度自此取消<ref>{{Cite web |url =http://www.people.com.cn/GB/shizheng/1026/1975347.html  |title = 中央决定五大领导班子今夏不到北戴河办公 |author =  |date =2003-07-09  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。

== 部分重要会议和决策 ==
在北戴河办公制度实行期间，每年暑期中央会议一般都在北戴河召开，几乎每次[[中国共产党全国代表大会|中国共产党全国代表大会]]之前的酝酿会议也都在北戴河举行。部分重要会议和决策如下<ref name=nfzm/>：

*1958年8月17日至30日，[[中国共产党中央委员会主席|中共中央主席]][[毛泽东|毛泽东]]在北戴河主持召开[[中国共产党中央政治局|中共中央政治局]]扩大会议（简称北戴河会议），这次会议是中共发动、领导“[[大跃进|大跃进]]”运动的重要会议，全面制定了“大跃进”运动的各项主要计划。参加会议的有中央政治局委员，各省、自治区、直辖市党委[[第一书记_(中国)|第一书记]]，以及国家机关各部门[[党组|党组]]负责人等。会议讨论了1959年的国民经济计划、[[第二个五年计划_(中国)|第二个五年计划]]、当前的工业生产、农业生产和农村工作问题、商业工作问题、教育方针问题及加强民兵工作等问题。制定和通过了《中共中央关于1959年计划和第二个五年计划问题的决定》、《1959年度国民经济计划主要指标》、《关于第二个五年计划的意见》、《中共中央关于改进计划管理制度的规定》、《中共中央关于1959年农业生产安排的决议》、《中共中央关于今冬明春在农村普通展开社会主义和共产主义教育运动的指示》、《中共中央、国务院关于教育工作的指示》、《中共中央、国务院关于各级干部参加体力劳动的决定》、《中共中央关于民兵问题的决定》和《中共中央关于继续展开[[除四害|除四害]]运动的决定》等40个文件，并且公开发表了《中共中央政治局扩大会议号召全党全民为生产1070万吨钢而奋斗》的会议公报以及《中共中央关于在农村建立[[人民公社|人民公社]]问题的决议》。会后，以全民[[大跃进|大炼钢铁]]及大办人民公社为标志，“大跃进”运动达到高潮<ref>{{Cite web|url=http://history.people.com.cn/n/2014/0815/c387668-25474942.html|title=1958年北戴河会议将“大跃进”推向高潮|date=2014-08-15|publisher=人民网}}</ref>。会议期间，8月17日毛泽东作出[[金门炮战|炮击金门]]的最后决定。18日他致信[[彭德怀|彭德怀]]，亲自部署炮击金门，并提出“直接对[[蔣中正|蒋]]，间接对[[美国|美]]”<ref>{{Cite web|url=http://history.people.com.cn/n/2014/0815/c387668-25474951.html|title=1958年，毛泽东在北戴河作出炮击金门的决策|date=2014-08-15|publisher=人民网}}</ref>。
*1960年7月5日至8月10日，中共中央在北戴河召开工作会议（简称北戴河工作会议）。因会前有[[布加勒斯特会议|布加勒斯特会议]]事件，会议进行中有[[苏联|苏联]]政府突然撤回专家、撕毁合同事件，同时鉴于上半年国家经济状况不佳及一些大城市粮食供应日益紧张的状况，会议讨论了国际问题和国内经济问题。会议通过了《中央关于全党动手，大办农业、大办粮食的指示》和《中央关于开展以保粮、保钢为中心的增产节约运动的指示》两个文件，并批准[[李富春|李富春]]、[[薄一波|薄一波]]提出的《1960年第三季度工业交通生产中的主要措施》，决定今后计划不再搞两本账，只搞一本账，不搞计划外的东西。会上也通过了《关于向党员干部介绍布加勒斯特会议情况和中苏关系问题的通知》。会议还决定成立中央局<ref>{{Cite web|url=http://qh.people.com.cn/GB/182782/225236/225247/15080321.html|title=1960年7月5日至8月10日 中央举行工作会议,研究国际问题和国内经济调整问题|date=2011-07-05|publisher=人民网}}</ref><ref>{{Cite web|url=http://cpc.people.com.cn/GB/69112/70190/70192/4765493.html|title=毛泽东传（1949―1976）：二十六、庐山会议后的一年四个月（下） (5)|accessdate=2018-02-27|publisher=人民网}}</ref>。
*1962年8月6日至8月下旬，中共中央在北戴河举行中央工作会议。这次会议是在[[三年困难时期|三年困难时期]]后期举行，会议起初讨论的重点是怎样把粮食生产搞上去。但毛泽东发表关于阶级、形势、矛盾三个问题的讲话，集中讲了[[阶级斗争|阶级斗争]]形势问题，号召阶级斗争要“年年讲，月月讲，日日讲”。该讲话是针对会议各组只讲经济问题，不讲阶级斗争问题作出的批评，也是对[[邓子恢|邓子恢]]的[[家庭联产承包责任制|包产到户]]责任制的批评，邓子恢背后是[[刘少奇|刘少奇]]。毛泽东讲话后，会议立即转向，[[柯庆施|柯庆施]]点名批评[[陶铸|陶铸]]的田间管理包工到户。[[周恩来|周恩来]]则说，陶铸的田间管理包工到户与邓子恢的包产到户不同，这为陶铸解了围。会议用部分时间讨论了毛泽东的讲话，并以其讲话精神为指导，为即将召开的[[中国共产党第八届中央委员会第十次全体会议|中共八届十中全会]]准备文件。会议期间，因[[中国共产党云南省委员会|中共云南省委]]书记[[阎红彦|阎红彦]]反映《[[中国青年报|中国青年报]]》连载的小说《[[反黨小說《劉志丹》案|刘志丹]]》存在严重政治问题，会议乃将连载《刘志丹》小说的《中国青年报》分发到各大组讨论。天气转凉后，中央工作会议宣布结束，没作会议总结便散会。会后随即在北京召开中共八届十中全会<ref>{{Cite web|url=http://history.people.com.cn/n/2014/0815/c387668-25475074.html|title=1962年北戴河会议：阶级斗争升温 走上通向文革之路|date=2014-08-15|publisher=人民网}}</ref>。
*1983年7月16日，鉴于全国治安形势严峻，[[中华人民共和国公安部|公安部]]向中央报送了《关于发挥专政职能改善公安装备的报告》。17日，公安部部长[[刘复之|刘复之]]抵达北戴河海滨。19日上午[[邓小平|邓小平]]召集刘复之、[[彭真|彭真]]来自己在北戴河的疗养住所谈话，邓小平拿着该报告对刘复之说，“你们这个报告不解决问题。”邓小平提出三年内组织三次战役，“严就能治住。”由此拉开了“[[中华人民共和国严打行动|严打]]”的序幕<ref>{{Cite web|url=http://history.people.com.cn/n/2014/0815/c387668-25473734.html|title=1983年“严打”：非常时期的非常手段|date=2014-08-15|publisher=人民网}}</ref>。
*1980年代，在北戴河数次会见[[日本|日本]]客人时，邓小平肯定了农村改革，并向外国友人展现了实行[[改革开放|改革开放]]的坚定决心。当时有部分人对改革开放，特别是兴建[[经济特区|经济特区]]表示担心，邓小平对此亮明态度。1985年8月1日，邓小平会见日本[[公明黨|公明党]]访华团，强调改革开放坚定不移<ref name=ruhe>{{Cite web |url =http://history.people.com.cn/n/2014/0815/c387668-25473503.html  |title =“北戴河会议”如何影响中国  |author =  |date = 2014-08-15 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
*1988年8月15日到17日，中央工作会议在北戴河召开，批准了物价和工资改革方案，同时决定用最严厉手段压缩基本建设以及集团购买力，为改革创造外部条件。“[[价格闯关|价格闯关]]”由此开始。8月19日，新闻公布了中央政治局讨论并通过了《关于价格、工资改革初步方案》的消息，部分省市随即发生大规模抢购风。价格闯关不久即终止<ref>{{Cite web|url=http://business.sohu.com/20140822/n403668468.shtml|title=邓小平价格闯关与1988年的大通胀|date=2014-08-22|publisher=搜狐}}</ref>。
*1997年暑期，[[中国共产党第十五次全国代表大会|中共十五大]]准备工作在北戴河进行。十五大报告的起草班子在北戴河按照中共中央政治局的意见和各方面的建议，对报告进行了修改<ref name=ruhe/>。
