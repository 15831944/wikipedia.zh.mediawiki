《'''北京日報'''》是一份[[中华人民共和国|中华人民共和国]][[北京市|北京市]]的地區[[日報|日報]]，於1952年10月1日創刊，現為[[北京日報報業集团|北京日報報業集团]]九份報章之一。該報為當地發行量最大的[[報章|報章]]，屬於[[中國共產黨|中國共產黨]][[中国共产党北京市委员会|北京市委员会]]的[[機關報|機關報]]。[[毛澤東|毛澤東]]於1952年和1964年兩次為《北京日報》題寫報頭。现任社长为[[梅宁华|梅宁华]]。

== 要求骆家辉公布财产事件 ==
2012年5月14日下午，《北京日报》在[[新浪微博|新浪微博]]官方账号转发文章“[[美国驻华大使|美国驻华大使]][[骆家辉|骆家辉]]喝咖啡、坐经济舱是包装成‘平民’的作秀行为”时，提出“请骆家辉公布财产”的要求，此微博瞬时引起中国网民围观和热议，将骆家辉同中国时下最敏感的辩论相联系，即公布党政官员私人财产的问题<ref>{{cite web |url = http://news.163.com/12/0516/09/81K9T85300014AED.html |title = 逼骆家辉晒家产让谁尴尬 |work = 金陵晚报 |publisher=网易 |date=2012-05-16 |accessdate=2012-05-21}}</ref>。其实，在美国[[:en:Center_for_Responsive_Politics|Opensecrets.org]]网站，一直可以清晰地查阅骆家辉2008年至2010年的财产状况，还可查到[[奥巴马|奥巴马]]总统、所有内阁成员及国会议员的财产。[[美国驻华大使馆|美国驻华大使馆]]证明：骆家辉在[[美国联邦政府|美国联邦政府]]行政部门的官员当中是第六富人；他的工资在当[[美国商务部长|美国商务部长]]的时候，年薪为191300[[美元|美元]]；现在作为美国驻华大使，年薪为179700美元<ref>{{cite web |url = http://www.zaobao.com/wencui/2012/05/chosun120520.shtml |title = 被指“平民秀”骆家辉微博公开财产 |work= （韩国）朝鲜日报 |publisher=联合早报 |date=2012-05-20 |accessdate=2012-05-21}}</ref>。不少网民将此贴转给《北京日报》官博，要求该报社长梅宁华公布个人财产；但《北京日报》并未做出回应，其官博头一天要求骆家辉公布财产的帖子已经删除<ref>{{cite web|url=http://www.nbweekly.com/news/china/201205/29970.aspx|title=洋微博混江湖|publisher=南都周刊|date=2012-05-21|accessdate=2012-05-21|deadurl=yes|archiveurl=https://web.archive.org/web/20120524042444/http://nbweekly.com/news/china/201205/29970.aspx|archivedate=2012-05-24}}</ref><ref>{{cite web|title=《北京日报》要骆家辉公布财产 网民查出反求诸京官无回应（图）|url = http://www.rfa.org/mandarin/yataibaodao/bj-05152012101602.html |date=2012-05-15|accessdate=2012-08-06 |publisher=RFA}}</ref>。
