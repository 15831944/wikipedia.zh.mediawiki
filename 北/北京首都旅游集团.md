'''北京首都旅游集团有限责任公司'''（简称'''北京首旅集团'''），是[[北京市人民政府国有资产监督管理委员会|北京市人民政府国有资产监督管理委员会]]监管企业，也是中国最大的旅游企业集团之一<ref name=lhh/>。

==简介==
首旅集团以经营旅游业为主，业务范围包括酒店、旅行社、汽车、餐饮、购物、娱乐、会展、景区等<ref name=lhh/>。1998年2月28日，根据[[北京市人民政府|北京市人民政府]]京政函〔1998〕3号《关于北京旅游集团有限责任公司组建方案和北京旅游集团有限责任公司章程的批复》精神, '''北京旅游集团有限责任公司'''在原北京市旅游局所属的全资、控股的33家企业基础上组建成立<ref>{{Cite web |url =http://bond.hexun.com/2004-11-16/101232408.html  |title = 04年北京首都旅游集团有限公司公司债上市公告 |author =  |date =2004-11-16 |publisher =  和讯|language =  |accessdate =  |quote =  }}</ref>。2000年6月，首旅股份在[[上海证券交易所|上海证券交易所]]上市<ref name=lhh>{{Cite web |url = http://www.cec-ceda.org.cn/famousdb/qiye1410/intro.html |title =  北京首都旅游集团有限责任公司 |author =  |date =  |publisher = 中国企业联合会 |language =  |accessdate =2018-03-11  |quote =  }}</ref>。2000年以来，北京旅游集团有限责任公司先后接收了自中央脱钩的华龙旅游实业发展总公司、康辉旅行社集团、[[北京国际饭店|国际饭店]]、[[兆龙饭店|兆龙饭店]]等企业。2002年12月，根据北京市人民政府京政办函〔2000〕157号，北京旅游集团有限责任公司更名“'''北京首都旅游集团有限责任公司'''”<ref>{{Cite web |url = http://money.163.com/10/0901/01/6FF5TJQ000253B0H.html |title =北京市西单商场股份有限公司发行股份购买资产暨关联交易预案  |author =  |date = 2010-09-01 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2004年，在[[中共北京市委|中共北京市委]]、[[北京市人民政府|北京市人民政府]]、[[北京市人民政府国有资产监督管理委员会|北京市国资委]]推进国有企业布局调整、深化国企改革的背景下，首旅集团、新燕莎集团、[[全聚德|全聚德]]集团、[[东来顺|东来顺]]集团、[[北京古玩城|古玩城]]集团合并重组<ref name=lhh/>。2007年11月，全聚德集团进入[[A股_(中國)|A股]]市场<ref name=lhh/>。2010年，北京农业集团、西单友谊集团先后并入首旅集团。2014年，首旅收购了雅客驿怡家、宁波南苑股份等民营企业。2016年，首旅集团旗下的北京首旅酒店（集团）股份有限公司（简称首旅酒店）收购[[如家酒店|如家酒店]]<ref>{{Cite web |url =http://www.sohu.com/a/220229934_476087  |title = 整合王府井东安后 千亿资产的首旅集团兜里都有啥？ |author =  |date =2018-02-01  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。此前2006年10月“如家酒店连锁”登陆[[美国|美国]][[纳斯达克|纳斯达克]]市场<ref name=lhh/>，被首旅酒店收购后，如家从美国纳斯达克退市。2018年，经中共北京市委、北京市人民政府批准，北京市国资委决定对首旅集团与北京王府井东安集团实施合并重组，将北京王府井东安集团的国有资产无偿划转给首旅集团，北京王府井东安集团保留独立法人地位<ref>{{Cite web |url =  http://news.163.com/18/0131/09/D9FH7V88000187VE.html|title =  北京国资重组2018年头炮：王府井母公司并入首旅集团|author =  |date = 2018-01-31 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。 

首旅集团还在中国沿海城市、重点旅游城市布局，先后收购[[三亚|三亚]]南山文化苑、[[广州|广州]]天伦万怡大酒店、[[深圳|深圳]]凯宾斯基酒店、[[西安|西安]]建国饭店、[[郑州|郑州]]兴亚饭店等，投资建设海南三亚国宾馆<ref name=lhh/>。

首旅集团多次承担国家和北京市重大活动的服务任务，先后承担1999年[[首都各界庆祝中华人民共和国成立50周年大会|国庆50周年大会]]，2005年北京财富论坛，2006年11月[[中非合作论坛|中非合作论坛]]北京峰会，[[诺贝尔奖|诺贝尔奖]]获得者北京论坛，[[中共十六大|中共十六大]]、[[中共十七大|中共十七大]]、全国[[两会|两会]]等的接待任务。另外还先后参与2003年抗击“[[非典|非典]]”、[[2008年北京奥运会|2008年北京奥运会]]等的保障任务<ref name=lhh/>。

首旅集团旗下有众多企业。北京首旅建国酒店管理有限公司是该集团的酒店管理品牌之一。首旅集团旗下上市公司有：
*北京首旅酒店（集团）股份有限公司（简称首旅酒店）：北京首都旅游股份有限公司（简称首旅股份）是经北京市人民政府京政办函〔1999〕14号批复批准，由北京首都旅游集团有限责任公司、北京城乡贸易中心股份有限公司、[[清华同方|清华同方]]股份有限公司、中国北京[[全聚德|全聚德]]集团有限责任公司、北京市昌平区十三陵特区旅游服务开发总公司作为发起人设立的股份有限公司。1999年2月12日在北京市工商行政管理局注册登记。2000年在上海证券交易所上市<ref>{{Cite web |url =  http://finance.sina.com.cn/stock/company/sh/600258/10_8.shtml|title = 北京首都旅游股份有限公司2006年半年度报告 |author =  |date = 2006-08-26 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。2013年更名北京首旅酒店（集团）股份有限公司（简称首旅酒店）<ref>{{Cite web |url =http://epaper.bjnews.com.cn/html/2013-08/08/content_455807.htm?div=-1  |title = 首旅股份更名“首旅酒店” |author =  |date =  2013-08-08|publisher = 新京报 |language =  |accessdate =  |quote =  }}</ref>。2017年和如家酒店集团合并后，宣布更名北京首旅如家酒店（集团）股份有限公司，但工商登记信息中未更名<ref>{{Cite web |url = http://www.bbtnews.com.cn/2017/0328/186956.shtml |title =首旅酒店正式更名首旅如家 三大事业部架构浮出水面  |author =  |date = 2017-03-28 |publisher =  北京商报网|language =  |accessdate =  |quote =  }}</ref>。
*北京首商集团股份有限公司（简称首商股份）：北京市西单商场股份有限公司（简称[[西单商场|西单商场]]）是于1993年5月经北京市经济体制改革委员会京体改委字〔1993〕第49号文批准设立，由北京西单商场集团等五家单位为共同发起人组成的定向募集股份有限公司。1996年在上海证券交易所上市。2011年7月20日公司名称变更为北京首商集团股份有限公司（简称首商股份）<ref>{{Cite web |url =http://vip.stock.finance.sina.com.cn/corp/go.php/vCI_CorpInfo/stockid/600723.phtml  |title =公司简介—首商股份(600723)  |author =  |date =  |publisher =新浪  |language =  |accessdate =2018-03-11  |quote =  }}</ref>。
*中国全聚德（集团）股份有限公司（简称[[全聚德|全聚德]]）：1993年5月中国北京全聚德集团成立。1994年6月，由全聚德集团等6家企业发起设立北京全聚德烤鸭股份有限公司。2004年4月，首旅集团、全聚德集团、新燕莎集团战略重组，首旅集团成为北京全聚德烤鸭股份有限公司第一大股东。2005年1月，北京全聚德烤鸭股份有限公司更名中国全聚德（集团）股份有限公司。随即该公司进一步收购聚德华天控股有限公司30.91%股权，与北京华天饮食集团并列聚德华天控股有限公司第一大股东。2007年4月，[[仿膳饭庄|仿膳饭庄]]、[[丰泽园饭店|丰泽园饭店]]、[[四川饭店|四川饭店]]进入全聚德股份公司。2007年公司在[[深圳证券交易所|深圳证券交易所]]上市<ref>{{Cite web |url = http://gongsi.hexun.com/introduction.aspx?cid=283 |title =全聚德公司简介  |author =  |date =  |publisher =和讯  |language =  |accessdate =2018-03-11  |quote =  }}</ref>。

首旅集团的各项业务有：
*酒店业：首旅集团投资及管理的酒店总数超过100家。包括由[[北京饭店|北京饭店]]、[[北京贵宾楼饭店|北京贵宾楼饭店]]、[[凯宾斯基|凯宾斯基]]等品牌管理的近十家五星级酒店，以首旅建国为品牌管理的数十家三、四星级酒店，以如家、欣燕都为品牌管理的近百家经济型酒店<ref name=lhh/>。
*旅行社业：首旅集团控股四家大型全资质旅行社——中国康辉旅行社、神舟国旅集团、中国海洋国际旅行社、中国民族旅行社。中国康辉旅行社在全国有近百家分支机构，神舟旅行社在北京地区有100余个营销网点<ref name=lhh/>。
*餐饮业：汇集[[全聚德|全聚德]]、[[东来顺|东来顺]]、仿膳饭庄、丰泽园饭店、四川饭店和[[聚德华天|聚德华天]]等品牌，拥有200多家连锁经营店<ref name=lhh/>。
*汽车服务业：拥有8000余辆运营汽车，以及汽车销售、修理、租赁、油品供应的服务体系。首旅集团控股的首汽集团是中国最大的旅游汽车公司之一<ref name=lhh/>。
*旅游商业：拥有燕莎品牌，整合了[[燕莎友谊商城|燕莎友谊商城]]、[[贵友大厦|贵友大厦]]、金源新燕莎MALL、燕莎奥特莱斯购物中心、古玩城等<ref name=lhh/>。
*景区景点业：以资源开发为重点<ref name=lhh/>。
*会展业：拥有[[北京展览馆|北京展览馆]]及酒店会展设施<ref name=lhh/>。
*旅游培训：依托首旅集团培训中心及[[博士后|博士后]]科研工作站建立<ref name=lhh/>。
