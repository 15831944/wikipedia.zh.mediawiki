《'''北極特快車'''》（{{lang-en|''The Polar Express''}}）是[[克里斯·凡·艾斯伯格|克里斯·凡·艾斯伯格]]創作的兒童[[繪本|繪本]]<ref name="彭懿">{{Cite book|title=圖畫書：閱讀與經典|last=彭懿|publisher=二十一世紀出版社|year=2006年9月|isbn=7-5391-3329-5|location=南昌}}</ref>，於1985年出版。這本書在1986年獲得[[凱迪克獎|凱迪克獎]]金獎，並在2004年被改編為[[北極特快車|同名電影]]。

凡·艾斯伯格曾表示創作《北極特快車》的靈感是來自他童年時在家鄉[[大急流城_(密西根州)|大急流城]]過[[聖誕節|聖誕節]]的經歷<ref>{{Cite web|title='Polar Express' author Chris Van Allsburg on how Christmas in Grand Rapids inspired beloved book|url=https://www.mlive.com/entertainment/grand-rapids/2015/11/polar_express_author_chris_van.html|work=mlive.com|date=2015-11-17|accessdate=2019-04-17|language=en-US|first=Jeffrey|last=Kaczmarczyk}}</ref>。

== 故事簡介 ==
一位十歲的小男孩在[[聖誕夜|聖誕夜]]搭上了前往[[北極|北極]]的列車，來到了位於北極的一座大城市內，城內有著成千上萬的{{Le|聖誕精靈|Christmas elf}}。當[[聖誕老人|聖誕老人]]問小男孩想要什麼樣的{{Le|聖誕禮物 (聖誕節)|Christmas gift|聖誕禮物}}時，小男孩說只想要[[雪橇|雪橇]]上的一枚銀鈴，於是聖誕老人便將一枚銀鈴送給小男孩。當小男孩乘坐列車回家，卻發現放在口袋裡的銀鈴已經不慎丟失了。

次日早晨，小男孩與妹妹莎拉在拆禮物時，發現了聖誕老人寄來的禮盒，盒中就裝著那枚銀鈴。當小男孩搖動銀鈴時，它能發出小男孩與妹妹有史以來聽過最美妙的聲音，可是他們的父母親卻聽不見。

== 榮譽 ==
《北極特快車》獲得了凱迪克獎金獎（1986年）、美國「家長的選擇」基金會繪本獎（1985年）、美國《[[波士頓環球報|波士頓環球報]]》／《號角書》雜誌獎繪本銀獎（1986年），還入選[[美國圖書館協會|美國圖書館協會]]年度好書推薦、《[[紐約時報|紐約時報]]》年度最佳繪本、[[紐約公共圖書館|紐約公共圖書館]]「每個人都應該知道的一百種繪本」、{{le|美國全國教育協會|National Education Association}}推薦一百本最佳童書<ref name="彭懿" />。

== 改編作品 ==
在2004年，被改編為同名[[3D動畫|3D動畫]]電影《[[北極特快車|北極特快車]]》，由[[勞勃·辛密克斯|勞勃·辛密克斯]]執導，[[湯姆·漢克斯|湯姆·漢克斯]]等人配音<ref>{{Cite web|title=湯姆漢克：還好不是人演的|url=https://tw.entertainment.appledaily.com/daily/20041023/1329189/|accessdate=2019-04-17|language=zh-TW|author=塗翔文|date=2004-10-23|publisher=臺灣蘋果日報}}</ref>。
