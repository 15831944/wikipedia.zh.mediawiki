'''北回歸線車站'''位於[[臺灣|臺灣]][[嘉義縣|嘉義縣]][[水上鄉|水上鄉]]，曾為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]的[[鐵路車站|鐵路車站]]<!-- [[國音電碼|國音電碼]]：ㄅㄏ-->。未來配合[[臺鐵捷運化|臺鐵捷運化]]政策及[[嘉義市區鐵路高架化計畫|嘉義市區鐵路高架化計畫]]，計畫恢復本站作為通勤車站、辦理客運業務，車站名稱將改為'''北回'''。

== 車站構造 ==
*站房一棟。

== 利用狀況 ==
*以前由此站向北分歧出[[中油嘉義溶劑廠支線|中油嘉義溶劑廠支線]]，車站南方有[[台灣水泥|台灣水泥]]儲存槽。

== 車站周邊 ==
*下寮村
*回歸村
*[[北回歸線|北回歸線]]標誌
*[[嘉義機場|嘉義機場]]

== 歷史 ==
*1937年2月1日：設立「信號場」。
*1961年5月1日：由三等站降為甲種簡易站，且改為專辦貨運站，並指定由[[嘉義車站|嘉義站]]管理。
*2002年8月13日：裁撤。
*未來配合[[臺鐵捷運化|臺鐵捷運化]]政策，計畫恢復本站作為客運，車站名稱將改為北回。
