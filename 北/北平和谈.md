{{国共谈判史}}

'''北平和谈'''是[[第二次國共內戰|第二次國共內戰]]期間，[[中国共产党|中国共产党]]代表团与[[中華民國政府|中華民國政府]]代表團（以[[國民政府|國民政府]]代表团名義{{NoteTag|[[國民政府|國民政府]]在1947年[[行憲|行憲]]後已被[[中華民國政府|中華民國政府]]取代，然因中國共産黨、[[中國民主同盟|中國民主同盟]]等黨派及人民團體1946年反對[[中國國民黨|中國國民黨]]召開的制憲[[國民大會|國民大會]]，拒不承認該《憲法》及依該憲法成立的中華民國政府的合法性。1949年南京方面代表團遂以國民政府名義展開和談。}}）於1949年4月在[[北平|北平]]进行的[[停火|停火]][[谈判|谈判]]。該年4月1日，政府代表团自[[南京|南京]]飞抵北平。4月13日至4月15日，双方代表团进行实质磋商，拟定《[[s:国内和平协定|国内和平协定]]（最后修正案）》，并商定于4月20日签字。4月20日，政府电告南京代表团拒绝签字。4月21日，[[中国人民革命军事委员会|中国人民革命军事委员会]]主席[[毛泽东|毛泽东]]、[[中国人民解放军|中国人民解放军]]总司令[[朱德|朱德]]发布《[[s:向全国进军的命令|向全国进军的命令]]》，[[中国人民解放军|中国人民解放军]]随即发起[[渡江战役|渡江战役]]，談判終告失敗。

== 背景 ==
{{Quote box| width = 300px | align = right | quoted = true | bgcolor = #FFFFF0 | salign = right | quote = （蒋介石所提出的）“和平”可以，“和平”而有害于四大家族和买办地主阶级的国家的“独立完整”，那就万万不可以。“和平”而有害于中美友好通商航海条约、中美空中运输协定、中美双边协定等项条约，有害于美国在华驻扎海陆空军，建立军事基地，开发矿产和独占贸易等项特权，有害于将中国作为美国殖民地的地位，一句话，“和平”而有害于这一切保护蒋介石反动国家的“独立完整”的办法，那就一概不可以。
 | source= [[毛泽东|毛泽东]]：《{{s|评战犯求和}}》（1949年1月4日）}}

1949年初，中國共產黨在[[三大戰役|三大戰役]]；[[遼瀋戰役|遼瀋戰役]]、[[淮海战役|淮海战役]]、[[平津战役|平津战役]]大獲全勝，[[中國國民黨|中國國民黨]]在[[中国东北地区|中国东北地区]]和[[華北|華北]]全面失去控制。1月16日，中國國民黨一部份[[民意代表|民意代表]]，發表主張：「促請國共迅取[[和平|和平]]，即日[[停戰|停戰]]，開始商談。」同時，「華北人民和平促進會」亦電籲雙方停戰為之呼應<ref name="寧靜">[[蔣經國|蔣經國]]：〈危急存亡之秋〉，刊《風雨中的寧靜》，台北：[[正中書局|正中書局]]，1988年</ref>{{rp|134}}。當天晚上，[[蔣介石|蔣介石]]為著「和談」問題，邀約[[中國民主社會黨|中國民主社會黨]]及[[中國青年黨|中國青年黨]]代表及有關人員討論時局，並對[[毛澤東|毛澤東]][[廣播|廣播]]及[[政策|政策]]交換意見<ref name="寧靜" />{{rp|134}}。會中，[[邵力子|邵力子]]主張「[[無條件投降|無條件投降]]」<ref name="寧靜"/>{{rp|134}}。1月19日，[[行政院|行政院]]舉行[[政務會議|政務會議]]。決議：「派代表赴[[延安|延安]]與中共談判和平。」<ref name="寧靜" />{{rp|135}}。1月21日，[[總統|總統]]蔣介石宣布下野，由[[副總統|副總統]][[李宗仁|李宗仁]]代理總統職務。李宗仁代理總統職務後，設法與中共談判停戰。1月22日，李宗仁電邀[[李濟深|李濟深]]、[[章伯鈞|章伯鈞]]、[[張東蓀|張東蓀]]等，共同策進「和平運動」<ref name="寧靜"/>{{rp|141}}。另由[[行政院會議|行政院會議]]決定：派邵力子、[[張治中|張治中]]、[[黃紹竑|黃紹竑]]、[[彭昭賢|彭昭賢]]、[[鍾天心|鍾天心]]等為代表，並指定邵力子為首席，等候中國共產黨代表，於雙方同意之地點，進行「和談」<ref name="寧靜"/>{{rp|141}}。

1949年2月1日，李宗仁決定派「人民代表」[[顏惠慶|顏惠慶]]等「前往北平求和」<ref name="寧靜"/>{{rp|148}}。中國共產黨表示拒絕「南京的和平代表團」前往<ref name="寧靜"/>{{rp|148}}。2月5日，行政院遷移[[廣州|廣州]]辦公，南京只保留代總統辦公室<ref name="現代史">李守孔著，《中國現代史》，台北，三民書局，1973年9月，ISBN 9571406635</ref>{{rp|169}}。同日，李宗仁私人代表[[甘介侯|甘介侯]]，「組織李氏之私人代表團，由顏惠慶、[[章士釗|章士釗]]、[[江庸|江庸]]、[[凌憲揚|凌憲揚]]、[[歐元懷|歐元懷]]、[[侯德榜|侯德榜]]六人為代表」<ref name="寧靜"/>{{rp|150}}，準備赴北平，「試探和談」<ref name="寧靜"/>{{rp|150}}。惟中國共產黨廣播：「不承認李所代表之南京政府」<ref name="寧靜"/>{{rp|150}}。2月8日，李宗仁私人代表顏惠慶、章士釗、江庸等，「原定今日飛往北平」<ref name="寧靜"/>{{rp|152}}，臨行突又為中國人民解放軍所阻。2月13日，[[上海|上海]]和平代表團顏惠慶、邵力子、章士釗等一行飛赴北平，準備與中國共產黨「談判」<ref name="寧靜"/>{{rp|154}}。

中国共产党方面，[[中国人民解放军第三野战军|中国人民解放军第三野战军]]在2月20日向所属各部发布“[[渡江戰役|京沪杭战役]]预备命令”。2月25日，[[中国共产党中央军事委员会|中国共产党中央军事委员会]]电示[[中国人民解放军第二野战军|中国人民解放军第二野战军]]、第三野战军：3月中旬或下旬，应准备占领[[浦口|浦口]]及炮击南京<ref>{{cite book zh |editor = 南京市档案馆 |title = 南京解放 |location=北京 |publisher=[[中国档案出版社|中国档案出版社]] |date = 2009年4月 |page  = 689 |accessdate= |ISBN = 9787510500169 }}</ref>。

3月3日，李宗仁指定：「[[吳鐵城|吳鐵城]]先生、邵力子等十人，研討與」中國共產黨「和談方案」<ref name="寧靜"/>{{rp|164}}。3月25日，李宗仁派邵力子、張治中、黃紹竑、章士釗、[[李蒸|李蒸]]等為和談代表，正式成立代表團；「並指定邵力子為首席代表」<ref name="寧靜"/>{{rp|172}}。3月26日，中國共產黨廣播稱：「中共中央決定：(一)談判定4月1日開始；(二)談判地點在北平；(三)中共中央派[[周恩來|周恩來]]、[[林伯渠|林伯渠]]、[[林彪|林彪]]、[[葉劍英|葉劍英]]、[[李維漢|李維漢]]等為代表，以周恩來為首席代表」<ref name="寧靜"/>{{rp|172－173}}，正式通知李宗仁與南京中國國民黨「所派代表團」<ref name="寧靜" />{{rp|173}}恢復會談<ref>[[寒山碧|寒山碧]]：《[[鄧小平|鄧小平]]從南京打到[[成都|成都]]》</ref>。

== 過程 ==
4月1日，中華民國政府以國民政府為名義的和談代表團由[[張治中|張治中]]任團長，成員有[[邵力子|邵力子]]、[[黃紹竑|黃紹竑]]、[[章士釗|章士釗]]、[[李蒸|李蒸]]、[[劉斐|劉斐]]等，由南京飛抵北平[[南苑機場|南苑機場]]<ref name="寧靜" />。{{rp|176}}顧問[[屈武|屈武]]以及[[李俊龍|李俊龍]]、[[金山_(官員)|金山]]一行9人，於上午乘[[中国航空公司|中国航空公司]]專機，在機場發表書面談話，表示「謹慎地秉承政府的意旨，幾最大的誠意，和中共方面進行商談，希望能夠獲得協議，使真正永久的和平，得以早日實現」<ref name="民國史大事記">李新 總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室 編，韓信夫、姜克夫 主編：《中華民國史大事記》（全十二卷，共十二冊），北京：[[中華書局|中華書局]]，2011年7月</ref>{{rp|8859－8860}}。下午專機抵北平<ref name="民國史大事記" />{{rp|8860}}。中國共產黨代表團秘書長[[齊燕銘|齊燕銘]]到機場接機，隨後國民政府和談代表團抵達。晚上，[[周恩來|周恩來]]、[[林伯渠|林伯渠]]、[[林彪|林彪]]、[[葉劍英|葉劍英]]、[[李維漢|李維漢]]、[[聶榮臻|聶榮臻]]等在[[华风宾馆|六國飯店]]公宴南京代表團一行<ref name="民國史大事記" />{{rp|8860}}。下午6時，宴後周恩來、林伯渠與張治中、邵力子交談<ref name="民國史大事記" />{{rp|8860}}。

4月2日，和談代表團向周恩來建議：「和談之前雙方軍隊固守原防。」中國共產黨不允，要求国民政府[[江南|江南]]憲警必須全部撤退，而政府則可暫維現狀<ref name="寧靜"/>{{rp|176}}；並着[[李宗仁|李宗仁]]、[[白崇禧|白崇禧]]、[[何應欽|何應欽]]、[[顧祝同|顧祝同]]、[[于右任|于右任]]、[[居正|居正]]等「皆去北平」<ref name="寧靜"/>{{rp|176}}。4月5日上午9時，和談預備會議開始<ref name="現代史"/>{{rp|169}}。中國共產黨經由[[民主党派|第三方面]]代表，「對李宗仁發出最後通牒，要求其在十二日限期內投降」<ref name="寧靜"/>{{rp|178}}。提出“聯合委員會”之組織，毛澤東任「主席」，而李宗仁副之；李宗仁須前往北平，「共同管理國共軍隊之移交」<ref name="寧靜"/>{{rp|178}}。4月7日，李宗仁向中國共產黨建議：「隔長江而分治」<ref name="寧靜"/>{{rp|178}}。中國共產黨則堅持「無論和、戰，均須過江」<ref name="寧靜"/>{{rp|178}}。4月8日，毛澤東對張治中等和談代表，「發表談話」<ref name="寧靜"/>{{rp|179}}。4月13日，和談第一次正式會議，在北平舉行<ref name="寧靜"/>{{rp|182}}。中國共產黨闡述其立場與協定草案之理由<ref name="寧靜" />{{rp|182}}。周恩來復提出《國內和平協定》，除前述之要求外，另列舉8條24款，限5日內接受，並聲稱解放軍須在協定簽字後3日內渡江<ref name="現代史"/>{{rp|169}}。

== 破裂 ==
4月15日，和談第二次正式會議，仍在北平舉行<ref name="寧靜" />{{rp|183}}。周恩來將修正後之《[[s:國內和平協定|國內和平協定（最後修正案）]]》，限張治中等4月20日簽字，並表示不論戰爭或和平，屆時解放軍一定渡過長江<ref name="現代史" />{{rp|169}}。4月17日，中國共產黨堅持渡江要求<ref name="寧靜" />{{rp|185}}。李宗仁向中國共產黨「要求展緩簽訂協日期」<ref name="寧靜" />{{rp|185}}。4月19日，南京政府發表廣播，表示對和平協定未能全部同意，[[共軍|共軍]]渡江難以接受。「行政院予以拒絕」<ref>陳布雷等：《蔣介石先生年表》，台北，傳記文學出版社，1978年6月1日，第59頁</ref>簽字，因協定比「八項和平條件」更苛刻。4月20日，中國國民黨中央常務委員會發表聲明，指斥中國共產黨之《國內和平協定（最後修正案）》歪曲事實。<ref name="現代史" />{{rp|170}}国民政府與中國共產黨之最後「和談」因之破裂，張治中、邵力子等，則於4月23日投向中國共產黨。<ref name="現代史" />{{rp|170}}解放軍五十萬人全面向[[江南|江南]]進攻，欲以武力壓迫「政府簽訂投降式之協定」<ref name="寧靜" />{{rp|186}}，偷渡[[長江|長江]]。<ref name="現代史"/>{{rp|170}}4月21日，[[毛澤東|毛澤東]]、[[朱德|朱德]]發佈總攻擊令，於是解放軍[[林彪|林彪]]、[[彭德懷|彭德懷]]等部，亦分別對[[武漢|武漢]]及[[西安|西安]]進攻。<ref name="現代史"/>{{rp|170}}[[渡江战役|渡江战役]]開始<ref>寒山碧：《[[鄧小平|鄧小平]]從[[南京|南京]]打到[[成都|成都]]》</ref>。
