'''北海道道287号厚真滨厚真停车场线'''（{{lang-ja|北海道道287号厚真浜厚真停車場線}}）是一条位于[[北海道|北海道]][[厚真町|厚真町]]内的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。

== 指定区間 ==
* 起点：北海道[[勇拂郡|勇拂郡]]厚真町新町（＝[[北海道道10号千岁鹉川线|北海道道10号千岁鹉川线]]交点）
* 終点：北海道勇拂郡厚真町字滨厚真（＝[[北海道旅客铁道|JR]][[日高本線|日高本線]] [[滨厚真站|滨厚真站]]）
* 总長：13.7千米

== 共线区间 ==
* 厚真町字上厚真-厚真町字滨厚真（[[国道235号|国道235号]]）

== 经过的自治体  ==

== 历史 ==
* 1957年7月25日 路線勘定。（238号）（1957年北海道告示第1487号）
* 1994年10月1日 路線编号变更为287号。（1994年北海道告示第1468号）
该路线的前身为1944年11月25日所勘定的準地方費道151号厚真滨厚真停車場線。
