'''北89線 板橋－內藤寮坑'''，是位於中華民國[[新北市|新北市]]的一條鄉道，北起[[新北市|新北市]][[板橋區_(台灣)|板橋區]]信義里信義路、四川路口，南至新北市[[土城區|土城區]]廷寮里永豐路207巷口，全長3.1公里。{{r|thb}}

== 路線說明 ==
;'''[[新北市|新北市]]'''
*[[板橋區_(台灣)|板橋區]]：'''板橋'''（起點，[[File:TW_PHW3.svg|23px]]四川路一段、二段路口）→ 信義路 → '''和平路口''' → '''四汴頭'''（廣權路岔路）→ '''區界'''

*[[土城區|土城區]]：'''區界''' → 永豐路 → '''立德路口''' →  '''金城路口'''（[[File:TW_THWtp88.png|30px]][[北88線|北88線]]岔路） → '''明德路口''' → '''內藤寮坑'''（終點，永豐路207巷口）

== 支線 ==
北89線唯一的支線為'''北89-1線'''，是位於[[新北市|新北市]]的一條鄉道，即土城區青雲路。北起新北市土城區青雲路與永豐路口，即板橋土城兩區交界，南至新北市土城區青雲路與石門路口，全長2.25公里。{{r|thb}}

;'''[[新北市|新北市]]'''
*[[土城區|土城區]]：'''外藤寮坑'''（起點，[[File:TW_THWtp89.png|30px]]青雲路與永豐路口，板橋土城交界）→ 青雲路 → '''學府路口''' → '''金城路口''' → '''清水路口'''（[[File:TW_THWtp88.png|30px]][[北88線|北88線]]岔路） → '''明德路口''' → '''清水坑'''（終點，青雲路與石門路口）
