'''亦庄线'''开通于2010年12月30日，是[[北京地铁|北京地铁]]的一条线路，途经[[丰台|丰台]]、[[朝阳区_(北京)|朝阳]]、[[大兴|大兴]]、[[通州区|通州]]四个区，连接北京市区和[[亦庄|亦庄]]新城[[北京市经济技术开发区|经济技术开发区]]。这条线路始于[[宋家庄站_(北京)|宋家庄站]]，终于[[亦庄火车站_(地铁)|亦庄火车站]]，全长23.3km，共设车站14座，其中有8座地面站，6座地下站。亦庄线可在[[宋家庄站_(北京)|宋家庄站]]与{{北京地铁线路名|5}}和{{北京地铁线路名|10}}换乘，未来可在[[亦庄火车站_(地铁)|亦庄火车站]]则可以换乘[[京津城际铁路|京津城际铁路]]。线路的标志色为桃红色<ref>{{cite web|url=http://news.sohu.com/20100426/n271740866.shtml|title=北京大兴线亦庄轨道交通线7月试车 年底试运营|date=2010-04-26|accessdate=2010-04-26}}</ref>。

== 历史 ==
=== 早期规划 ===
早在2004年，亦庄线的规划就已经被原则性同意，线路走向与现在线路主要在亦庄开发区核心区内有所区别，长度略短。

2008年初亦庄线的规划被完全确定，原计划接驳5号线途径旧宫，亦庄桥南后沿博大路（北京经济技术开发区核心区住宅区与工业区的分界大街）的规划改变为了跨越博大路，沿中和街行进的线路。
=== 最终规划以及施工 ===
2007年12月8日，亦庄线正式开工<ref>{{cite news|url=http://news.xinhuanet.com/politics/2007-12/09/content_7219513.htm|title=北京5条地铁8日上午开工 2015年全部完工|date=2007-12-09|accessdate=2012-01-26|author=陈斯|coauthors=白冰}}</ref>。亦庄线采用了BT（建设－移交）模式，这种建设方式是指先由建设方建设，再转交给运营公司。这种方式占亦庄线融资额的三分之一。<ref name=bt>{{cite web|url=http://cx.bjjtw.gov.cn/page/20124043/124719414085246.html|title=地铁亦庄线明年底通车 BT项目投资量完成30%|date=2009-07-10|accessdate=2009-07-27|deadurl=yes|archiveurl=https://web.archive.org/web/20110305150332/http://cx.bjjtw.gov.cn/page/20124043/124719414085246.html|archivedate=2011-03-05}}</ref><ref>{{cite web|url=http://news.sina.com.cn/c/2008-10-17/094416472911.shtml|title=城铁亦庄线“BT工程”正式开工|date=2008-10-17|accessdate=2009-07-27}}</ref> 而不久之后的2008年12月2日，在中国政府推出[[扩大内需十项措施|扩大内需十项措施]]之后，北京市政府要求在建地铁加快工期，亦庄线的工期进行了重排，通车日期从2012年5月提前到2010年底<ref>{{cite web|url=http://bj.people.com.cn/GB/8451346.html|title=京地铁亦庄线2010年建成 比原计划提前两年通车|date=2008-12-03|accessdate=2009-07-27}}</ref>。

2010年9月，此线路主体施工全部完工，并开始3个月的空载试运行。2010年12月30日，除亦庄火车站外其余全部线路开通试运营。<ref>{{ cite web|url=http://www.bjd.com.cn/10zt/201006/t20100625_623379.html |title=本市5条轨道交通线9月完工 |author=北京日报 |publisher=京报网 |date=2010-06-25 |accessdate=2010-07-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20100729220047/http://www.bjd.com.cn/10zt/201006/t20100625_623379.html |archivedate=2010-07-29 }}</ref>2018年12月30日，亦庄火车站开通运营。

== 车站 ==
[[File:BJ_Subway_Songjiazhuang_transfer.jpg|thumb]]
亦庄线的14座车站中，宋家庄站至小红门站，次渠南站至亦庄火车站为地下站，旧宫站至经海路站为地上高架车站。地下站除宋家庄站外其余车站结构设计与五号线的双层地下[[岛式站台|岛式站台]]相似，在内饰上有所不同，而地上高架站台结构为[[侧式站台|侧式站台]]，外形设计又分为两种：“飘板”式和“木盒子”式。<ref name=xinbao>{{cite web|url=http://news.163.com/09/0723/01/5ESAT8OK000120GR.html|title=亦庄线8个地面站外观揭晓|publisher=[[北京娱乐信报|北京娱乐信报]]|date=2009-07-23|accessdate=2009-07-26}}</ref><ref name=bt />旧宫站，亦庄文化园站，同济南路站，荣京东街站分别设有可容纳200个停车位的P+R停车场以方便亦庄开发区以及马驹桥的居民停车搭乘地铁出行。<ref>

== 技术 ==
[[File:BJ_Subway_YZ_Seat.JPG|thumb]]
亦庄线将采用中国国产信号系统，以结束外国企业对北京地铁信号系统的垄断，可以降低购买费用和运营成本。<ref>{{cite web|url=http://beijing.qianlong.com/3825/2009/06/11/4202@5031136.htm |title=轨道交通亦庄线将采用国产信号系统 |publisher=[[千龙网|千龙网]] |date=2009-06-11 |accessdate=2009-07-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20110305152022/http://beijing.qianlong.com/3825/2009/06/11/4202%405031136.htm |archivedate=2011-03-05 }}</ref>
