'''[[北海道|北海道]]旗'''（{{jpn|j=北海道旗|hg=ほっかいどうき|rm=Hokkaidōkii|f=}}）是日本的47面[[日本都道府县旗帜|都道府县旗]]之一。该条目是对'''北海道旗'''以及'''北海道章'''（{{jpn|j=北海道章|hg=ほっかいどうしょう|rm=Hokkaidōshō|f=}}）的解说。

== 制定经过 ==
道旗、道章是1967年正式制定的。这一节介绍[[开拓使旗|开拓使旗]]（即'''北辰旗'''（{{jpn|j='''北辰旗'''|hg=ほくしんき|rm=Hokushinki|f=}}））与其由来，以及有关''北辰旗''的历史。

=== 开拓使旗（北辰旗）===
'''[[北海道|北海道]]开拓使旗'''使用的是[[青色|青色]]底色红[[五芒星|五芒星]]的，这面'''北辰旗'''是道民所喜爱的。历史上，开拓使公用船船长任命了专员[[蛯子末次郎|蛯子末次郎]]，并使用了他所设计的公用船船章。这面旗帜于1872年2月正式确立，使用的就是[[蛯子末次郎|蛯子末次郎]]所设计的船章。<ref>{{cite web |url =http://www.hokkaido-jin.jp/issue/sp/200603/sp_01.html|title =北辰旗的设计初始，蛯子末次郎设计的船章 |date = |language =ja |publisher =北海道人·北海道之星专题|accessdate = }}</ref>
同年9月，开拓使次官[[黑田清隆|黑田清隆]]向政府提出要求：把'''北辰旗'''的设计由红的[[五芒星|五芒星]]变更为红的[[七芒星|七芒星]]。但遭到了政府部门的拒绝，使黑田的设计变更案成了幻想，在那之后，1878年'''开拓使旗'''（即'''北辰旗'''）被取消悬挂，直到1882年三县（[[根室县|根室县]]、[[札幌县|札幌县]]、[[函馆县|函馆县]]）一局（[[北海道事业管理局|北海道事业管理局]]）再次统合，却未产生旧[[北海道厅|北海道厅]]时代的标志与旗帜。<ref>{{cite web |url =http://www.hokkaido-jin.jp/issue/sp/200603/sp_02.html|title =北辰旗的星星标志 |date = |language =ja |publisher =北海道人·北海道之星专题|accessdate = }}</ref>

=== 道旗、道章的制定 ===
1960年代，其他[[都|都]]、[[府|府]]、[[县|县]]相继制定了他们的[[都|都]]、[[府|府]]、[[县|县]]旗，[[北海道|北海道]]在1969年开道100周年的活动上制定了目前的县旗来彰显它的新动力。在道内开展活动时，依靠众多的美术团体的支持，总共获得了超过7,500份的道旗道章设计方案。最终经过'''北海道旗，北海道章委员会'''的评选，于1967年做出了采用[[栗谷川建一|栗谷川建一]]所设计方案的决定。5月1日，第775号道告示《北海道章与北海道旗的有关事项》上正式制定了[[北海道|北海道]]的道旗道章。<ref>{{cite web|url =http://www.reiki.pref.hokkaido.jp/d1w_savvy/HTML_TMP/svhtml-562732482.0.Mokuji.0.0.DATA.html|title =北海道章和北海道旗|date =|language =ja|publisher =北海道|accessdate =}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 设计含义 ==
'''北辰旗'''的[[五芒星|五芒星]]以及只处于想象阶段的[[七芒星|七芒星]]所体现的那种[[开拓者|开拓者]]、[[先驱者|先驱者]]雄壮的精神，以及对[[北海道|北海道]]未来能够继续传承这种精神，都在道章上得到了很好的诠释。另外，道章的有关设色并没有特定的说明。
道旗的配色基于原来黑田的设计，绀色象征的是[[北海|北海]]（即北[[日本海|日本海]]与南[[鄂霍次克海|鄂霍次克海]]）与天空，中央的是县章，最中央是象征道民不屈的精神的红色[[七芒星|七芒星]]，在它周围的是代表荣耀与风雪历程的白色光芒。
