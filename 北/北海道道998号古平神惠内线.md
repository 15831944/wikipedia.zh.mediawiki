'''北海道道998号古平神惠内线'''（{{lang-ja|北海道道998号古平神恵内線}}）是一条连结[[北海道|北海道]][[古平町|古平町]]和[[神惠内村|神惠内村]]的普通[[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。将[[積丹半島|積丹半島]]横向分段开。

== 路線概要 ==
* 起点：北海道[[古平郡|古平郡]][[古平町|古平町]]大字滨町（＝[[国道229号|国道229号]]交点）
* 終点：北海道[[古宇郡|古宇郡]][[神惠内村|神惠内村]]大字神惠内村川向（＝国道229号交点）
* 总長：32.0千米

== 通過的自治体 ==
* [[后志综合振兴局|后志综合振兴局]]
** 古平郡古平町
** 古宇郡神惠内村

== 主要接续道路 ==

** [[国道229号|国道229号]]

* 神惠内村
** 国道229号

== 歴史 ==
* 1982年3月29日 路線勘定。<!--1982年北海道告示第574号-->
该线路过去曾是国道229号。经过[[積丹町|積丹町]]路线的变更过后降格为道道（道级道路）。

== 隧道 ==
* 六志内隧道（215米）
* 清川隧道（455米）
* 神惠内隧道（283米）
