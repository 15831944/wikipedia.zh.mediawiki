'''北京市人民政府天安门地区管理委员会'''，简称'''天安门地区管委会'''，是负责组织协调[[天安门|天安门]]地区管理工作的[[北京市人民政府|北京市人民政府]]派出机构<ref name="elyw" />。

==沿革==
1990年2月，北京市人民政府天安门地区管理委员会设立<ref>{{Cite web|url =http://zhengwu.beijing.gov.cn/gzdt/bmdt/t1103172.htm|title =天安门管委会成立20周年|author =|date =2010-02-06|publisher =首都之窗|language =|accessdate =|quote =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；是年7月，挂牌成立。

2004年3月11日，北京市人民政府第20次常务会议审议通过《北京市天安门地区管理规定》，2004年3月18日北京市人民政府令第144号公布，自2004年4月20日起施行<ref>北京市天安门地区管理规定，北京市人民政府公报2004(7)</ref>。

2008年2月26日，《北京市人民政府办公厅关于印发〈北京市人民政府天安门地区管理委员会主要职责内设机构和人员编制规定〉的通知》（京政办发〔2008〕12号）印发了《北京市人民政府天安门地区管理委员会主要职责、内设机构和人员编制规定》<ref>{{Cite web|url =http://govfile.beijing.gov.cn/Govfile/front/content/22008012_0.html|title =北京市人民政府办公厅关于印发《北京市人民政府天安门地区管理委员会主要职责内设机构和人员编制规定》的通知|author =|date =2008-02-26|publisher =首都之窗|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20090106025424/http://govfile.beijing.gov.cn/Govfile/front/content/22008012_0.html|archivedate =2009年1月6日|df =}}</ref>。

2008年3月24日，北京市人民政府第2次常务会议审议通过《北京市人民政府关于修改〈北京市天安门地区管理规定〉的决定》，2008年3月31日北京市人民政府令第203号公布，自公布之日起施行<ref>{{Cite web |url =http://news.xinhuanet.com/local/2008-05/05/content_8106921.htm |title =北京修改《北京市天安门地区管理规定》(全文)  |author =  |date = 2008-05-05 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。

==职责==
根据《北京市天安门地区管理规定》（市政府第203号令），北京市人民政府天安门地区管理委员会是负责组织协调天安门地区管理工作的北京市人民政府派出机构。主要职责：

#贯彻执行国家及本市有关法律、法规和政策。
#组织协调天安门地区升挂国旗活动；会同有关部门做好在天安门地区举行的外事迎宾、节日庆典等活动。
#负责天安门地区的市容和环境卫生；组织协调天安门地区社会秩序、市政公用设施、城市绿化等工作；会同有关部门做好天安门地区文物保护工作。
#负责天安门地区应急管理工作。
#负责对有关部门在天安门地区的日常管理工作进行监督检查。
#负责天安门地区服务设施的统一规划工作。
#负责天安门城楼的开放与管理工作。
#承办市委、市政府交办的其他事项<ref name=elyw/>。

==机构设置==
根据《北京市人民政府办公厅关于印发〈北京市人民政府天安门地区管理委员会主要职责内设机构和人员编制规定〉的通知》（京政办发〔2008〕12号）等文件精神，北京市人民政府天安门地区管理委员会内设机构为<ref name=elyw>{{Cite web |url = http://zhengwu.beijing.gov.cn/zjgl/ys/2015/szfpcjg/t1382633.htm |title = 北京市人民政府天安门地区管理委员会 |author =  |date = 2015-03-03 |publisher = 首都之窗 |language =  |accessdate =  |quote =  |archive-url = https://web.archive.org/web/20150317074620/http://zhengwu.beijing.gov.cn/zjgl/ys/2015/szfpcjg/t1382633.htm |archive-date = 2015-03-17 |dead-url = yes }}</ref>：
*天安门地区运行调度中心
*办公室
*[[天安门城楼|天安门城楼]]管理处
*[[天安门广场|天安门广场]]管理处
*旅游服务处
*基本建设处
*财务处
*人事处
*机关党委

北京市人民政府天安门地区管理委员会下属5个预算单位：
*北京市城市管理综合行政执法局天安门分局（行政单位）
*天安门地区综合服务中心（全额拨款事业单位）
*天安门地区市容环境卫生管理事务中心（全额拨款事业单位）
*[[北京会议中心|北京会议中心]]（差额拨款事业单位）
*北戴河干休所（差额拨款事业单位）
