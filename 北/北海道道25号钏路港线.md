'''北海道道25号钏路港线'''（{{lang-ja|北海道道25号釧路港線}}）是一条位于[[北海道|北海道]][[钏路综合振兴局|钏路综合振兴局]][[钏路市|钏路市]]内的[[主要地方道|主要]][[道_(行政区)|道级]][[都道府县道|道路]]（[[北海道道道列表|北海道道]]）。通称南大通（みなみおおどおり）。

== 道路概况 ==
* 起点：[[北海道|北海道]][[钏路市|钏路市]]南大通8丁目（＝[[重要港湾|重要港湾]][[钏路港|钏路港]]）
* 終点：北海道钏路市南大通1丁目（＝[[国道38号|国道38号]]交点）
* 总長：1.2千米

== 经过的自治体  ==
*[[钏路综合振兴局|钏路综合振兴局]]
**[[钏路市|钏路市]]

== 主要连接道路 ==
'''钏路市'''
* [[国道38号|国道38号]]＝南大通1丁目（终点）
* [[北海道道113号钏路环状线|北海道道113号钏路环状线]]＝南大通1丁目（終点与[[ロータリー|ロータリー]]之间相接）

== 历史 ==
* 1954年3月30日 路線勘定为46号。（1954年北海道告示第503号）
* 1994年10月1日 路線编号変更。（1994年北海道告示第1468号）
