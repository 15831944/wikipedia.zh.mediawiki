'''整體浴室'''（{{lang-en|Unit Bath}}）<ref>{{Cite journal|title=ユニットバス|url=https://ja.wikipedia.org/w/index.php?title=%E3%83%A6%E3%83%8B%E3%83%83%E3%83%88%E3%83%90%E3%82%B9&oldid=70226188|date=2018-10-10|journal=Wikipedia|language=ja}}</ref>又稱為系統式衛浴，整體浴室是在工廠內使用高防水材料預先模製天花板，浴缸，地板，牆壁等的浴室，並且在運送到現場後組裝。 平面圖等使用的符號是UB<ref>{{Cite web|title=間取りに関する用語｜不動産用語辞典｜渋谷の賃貸,不動産物件ならビッグ|url=http://www.big-tokyo.com/beginner/terms2.html|work=www.big-tokyo.com|accessdate=2019-03-13}}</ref>。

== 概要 ==
整體浴室被大量使用在日本居家與飯店，不同於傳統浴室施工方法，整體浴室模具成形後，再到現場安裝，所以整體浴室大多有固定規格尺寸；不像傳統施工方法，要一片一片的黏貼牆面及地面磁磚，其優點是一體式底盤減少了漏水的可能，而且施工期間短，適用於各種建築物，常用於酒店，公寓，醫院等。近年來，整體浴室尺寸和設備選項方面有越來越多的選擇，因此它通常被稱為“系統衛浴”。英語又稱為Bath module。截至2014年，日本浴室的市佔率約95％。

== 歷史 ==
類似的整體浴室概念於1938年，由[[巴克敏斯特·富勒|巴克敏斯特·富勒]]提出，他是美國哲學家、建築師和發明家，由他提出的整體浴室，是以金屬作為結構，並在美國提出專利申請，但整體浴室當時並沒有在美國市場上普及。

現在常見的整體浴室是以[[玻璃钢|FRP]]材料製造，由日本開發設計的，為因應[[1964年夏季奥林匹克运动会|1964年的的東京奧運會]]，以容納大量的外國遊客，而興建[[新大谷飯店|新大谷飯店]]<ref>{{Cite web|title=About Us|url=https://www.newotani.co.jp/en/tokyo/about-us/|work=Hotel New Otani Tokyo|accessdate=2019-03-13|language=en}}</ref>，為了節省內部裝潢工程耗費之勞動力，而開發製造以模具生產的一體化衛浴設備。主要參與開發的兩家公司是日立化學（現為：[[Housetec|Housetec]]<ref>{{Cite web|title=Housetec|url=http://www.housetec.tw/|work=www.housetec.tw|accessdate=2019-03-13}}</ref>）和東洋陶瓷（現為：[[TOTO|TOTO]]）。透過方便搬運和安裝的整體浴室，浴室工程由一個月縮短到幾個小時。事實上，在新大谷，所有房間（超過1000個房間）安裝整體浴室據說只花了三個半月內完成。

二次大戰後新建的日本住宅往往以提供許多住宅為優先，品質反倒是其次。因此，這個時期的日本被迫必須以短的工期提供許多住宅。因此，防水性佳、能夠縮短工期的整體浴室正式普及<ref>{{Cite web|title=系統浴室｜商品｜Housetec|url=http://www.housetec.tw/products/bathroom.html|work=www.housetec.tw|accessdate=2019-03-13}}</ref>。

== 整體浴室規格尺寸 ==
在日本住房的整體浴室的規模幾乎是標準化的。 有“1216”，“1317”，“1416”，“1418”，“1616”，“1818”，“1620”等，它們表示垂直方向和水平方向的尺寸為10cm的數字。
