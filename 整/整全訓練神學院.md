'''整全訓練神學院（Full Training of Christian Workers Seminary）'''於2001年9月1日在[[香港|香港]]成立，接辦前信心神學院<ref name="build_up">{{cite web |title=整全訓練神學院院訊 |url=http://www.ftcws.org/ftmagazine/nl/200109nl.htm |publisher=《整全報》，2001年9月 |accessdate=2009年5月19日 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20080429223927/http://www.ftcws.org/ftmagazine/nl/200109nl.htm |archivedate=2008年4月29日 }}</ref>。'''整全訓練神學院'''乃一非宗派性的獨立機構，因此全無母會或所屬教會固定支持<ref name="introduction">{{cite web |title=學院簡介 |url=http://www.ftcws.org/introduction.htm |publisher=整全訓練神學院網頁 |accessdate=2009年5月19日 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20080804020316/http://www.ftcws.org/introduction.htm |archivedate=2008年8月4日 }}</ref>。學院位於[[九龍|九龍]][[深水埗|深水埗]]青山道64號名人商業中心1106、1301-03室<ref name="contact_us">{{cite web |title=聯絡我們 |url=http://www.ftcws.org/contactus.htm |publisher=整全訓練神學院網頁 |accessdate=2009年5月19日 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20080207222133/http://www.ftcws.org/contactus.htm |archivedate=2008年2月7日 }}</ref>。學院篤信[[聖經|聖經]]每字無誤，視之為一切信仰、生活、事奉及禮儀的最高權威，也深信舊約的[[先知|先知]][[門徒訓練|門徒訓練]]，和新約主[[耶穌|耶穌]]的[[門徒訓練|門徒訓練]]，就是最理想的[[神學|神學]]訓練模式，這才是以「生命影響生命」。學院不但注重教授[[聖經|聖經]]及[[神學|神學]]知識，也注重訓練學生將真理實踐在生活之中，務求在生命、品格、靈性、生活、[[靈修|靈修]]、事奉……達到「整全訓練」。<ref name="mission">{{cite web |title=異象使命 |url=http://www.ftcws.org/mission.htm |publisher=整全訓練神學院網頁 |accessdate=2009年5月26日 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20071012094247/http://www.ftcws.org/mission.htm |archivedate=2007年10月12日 }}</ref>

==異象與使命<ref name="mission">{{cite web |title=異象使命 |url=http://www.ftcws.org/mission.htm |publisher=整全訓練神學院網頁 |accessdate=2009年5月26日 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20071012094247/http://www.ftcws.org/mission.htm |archivedate=2007年10月12日 }}</ref>==
學院確認現今就是『末後的日子』。為針對『末世危機』，忠心等候主再來，學院認為[[傳道人|傳道人]]應有『整全裝備』。例如：
* [[講道|講道]]有『屬靈而有深度的聖經亮光』；
* [[禱告|禱告]]生活常達到『與神同行，享受靈裡的安息』，生活能發揮『屬靈人的感染力』；
* 在靈性品格上能『效法主的柔和謙卑』，言語行為能『作眾信徒的榜樣』；
* 在佈道事工上能做到『傳福音生活化，直到世界的末了』；
* 在純正[[信仰|信仰]]上有『明確立場』，能分辨真偽，『不與任何異端妥協』；
* 在逼迫的日子中能『忍耐到底，至死忠心』；
* 養成『自修研究精神』，畢業後仍能不斷在[[神學|神學]]上『閱讀自修』，成為時代的工人。
