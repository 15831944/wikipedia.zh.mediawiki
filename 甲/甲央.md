'''甲央'''（{{bd|1943年|10月||}}）[[藏族|藏族]]，[[西藏|西藏]][[亚东|亚东]]人，[[中华人民共和国|中华人民共和国]]官员，[[西藏自治区文化厅|西藏自治区文化厅]]党组成员、副厅长兼文物局局长、文化厅巡视员<ref name=shsh>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1819341 |title = 原西藏文物局局长甲央病逝，曾对布达拉宫维修投入大量心血 |author =  |date = 2017-10-10 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1956年4月参加工作，1962年加入[[中国共产党|中国共产党]]。长期从事文化文物工作，先后在[[西藏革命展览馆|西藏革命展览馆]]、西藏自治区文教局、西藏自治区文物管理委员会、[[西藏自治区文化厅|西藏自治区文化厅]]、[[西藏自治区文物局|西藏自治区文物局]]任职。1986年，任西藏自治区文化厅副厅长，主要分管文物工作。他是西藏自治区文化厅[[党组|党组]]成员、副厅长兼文物局局长、文化厅巡视员。1989年，兼任[[布达拉宫|布达拉宫]]维修工程施工办公室主任。后兼任西藏重点文物保护工程领导小组副组长。他在在布达拉宫一、二期保护维修工程等的文物保护利用工作中作出了突出贡献。2002年退休。2003年，被聘为西藏三大重点文物保护维修领导小组办公室副主任，重点负责专家咨询组的工作并且兼任专家咨询组组长<ref name=shsh/><ref>{{Cite web |url = http://news.enorth.com.cn/system/2001/08/03/000107981.shtml |title = 布达拉宫二期维修工程举行开工前佛事活动 |author =  |date =2001-08-03  |publisher =北方网  |language =  |accessdate =  |quote =  }}</ref>。

2017年10月3日，因[[心脏病|心脏病]]在[[拉萨|拉萨]]逝世，享年74岁<ref name=shsh/>。
