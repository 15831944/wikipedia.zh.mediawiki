'''鵝掌藤'''，又稱七葉蓮、七葉藤、狗腳蹄。屬於著生攀緣灌木<ref name="tw-tree">{{cite web
|url       = http://subject.forest.gov.tw/species/twtrees/book2/index-1.htm
|title     = 鵝掌藤
|work      = 臺灣樹木  解說（二）
|editor1   = 呂福遠
|editor2   = 歐辰雄
|editor3   = 呂金誠 
|publisher = 行政院農業委員會
|year      = 1997
|page      = p182}}</ref>。原產於[[海南島|海南島]]、[[廣東|廣東]]、[[廣西|廣西]]、[[臺灣|臺灣]]、[[印度|印度]]、[[澳洲|澳洲]]……等，在台灣，主要分布於山麓至高地<ref name="tw-tree"/>。可以作為[[中藥|中藥]]，有祛風除濕、活血、止痛、壯筋骨、消腫之效。治[[胃痛|胃痛]]、[[風濕|風濕]]、[[痺痛|痺痛]]、[[關節炎|關節炎]]、[[跌打|跌打]]損傷。也是優良的[[觀葉植物|觀葉植物]]，由於其不喜歡陽光，適合較高的樓屋住宅[[太阳光|日照]]不足之處種植[[綠化|綠化]]。
[[File:貓鼻頭公園鵝掌藤.jpg|thumb]]的鵝掌藤。]]
[[早田文藏|早田文藏]]於台灣採集發現，具有附生性不同於[[鵝掌柴|鵝掌柴]]<ref name="IconPlFormos"> [http://tai2.ntu.edu.tw/ebook/ebookpage.php?volume=6&book=Icon.%20Pl.%20Formos.%20&page=23 臺灣植物圖譜 Icon. Pl. Formos. vol.6 p.23-24], ...Near Heptapleurum octophyllum HANCE ; but distinguishable from it by the much smaller, thicker, coriaceous leaves and in the very depressed rounded stigmata. H. octophyllum is terrestrial, but this new species is always found to be epiphytic.</ref>。
