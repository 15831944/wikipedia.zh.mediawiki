《'''笔仙惊魂'''》（{{lang-en|'''BiXian panic'''}}）2012年6月8日上映于[[中国大陆|中国大陆]]的青春校园恐怖片。<ref>{{cite web |url=http://ent.qq.com/a/20120516/000175.htm |title= 《笔仙惊魂》曝剧照 中国式笔仙6月8日正式现身|accessdate=2012年5月16日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

== 剧情 ==
电影讲述2对艺术大学的大学生情侣小武、慕凡、柳丝丝、凌菲儿在合租的四合院里玩中国古代最为禁忌的一种道术“请笔仙”，结果请来了[[地狱|地狱]]中的阴魂，阴魂纠缠着他们索命。

== 花絮 ==
电影中，柴碧云出演一个外表清纯可爱内心却是黑暗阴毒的女生。<ref>{{cite web |url=http://et.21cn.com/gundong/etscroll/2012/05/28/11996183.shtml |title=柴碧云《笔仙惊魂》惊悚亮相 外表清纯内心可怕 |accessdate=2012-05-28 |publisher=21CN |work= |language=zh-cn }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

演员何杜娟因为胆小，拍摄期间，她一个人晚上吓得不敢去上厕所。<ref>{{cite web |url=http://ent.huanqiu.com/movie/yingshi-neidi/2012-05/2763309.html |title=《笔仙惊魂》剧情泄密 何杜娟遇导演潜规则 |accessdate=2012-05-28  |publisher=环球网 |work= |language =zh-cn}}</ref>

男主演王一该片中出演一个才华横溢、帅气斯文的[[屌丝|屌丝]]男。<ref>{{cite web |url= http://ent.qq.com/a/20120523/000053.htm|title=王一《笔仙惊魂》成精英男 精湛的演技折服粉丝 |accessdate= 2012年5月23日|publisher=腾讯网 |work= |language =zh-cn}}</ref>

电影的配乐是日本恐怖片大师[[室田宪一|室田宪一]]，他为该片的音乐配置音乐背景。<ref>{{cite web |url= http://news.xinhuanet.com/ent/2012-05/25/c_123188259.htm|title=《笔仙惊魂》惊悚看点揭秘 情节渗人悬念迭出 |accessdate=2012年5月25日 |publisher=新华网 |work= |language =zh-cn}}</ref>

电影摒弃了[[泰国|泰国]]和[[日本|日本]]式样的神秘阴森、血腥淋淋的镜头，开启了一种“新恐怖主义”。<ref>{{cite web |url=http://ent.qq.com/a/20120518/000065.htm |title=《笔仙惊魂》打造另类题材“新恐怖主义”电影 |accessdate=2012年5月18日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

为了获得收视率和票房，电影也加入了[[床戏|床戏]]的部分，以满足观众的感情需要。<ref>{{cite web |url= http://ent.163.com/12/0522/07/823H67U3000300B1.html|title= 《笔仙惊魂》曝剧照 导演称真诚创作望观众认同|accessdate= 2012-05-22|publisher= 网易|work= |language =zh-cn}}</ref>
