'''{{PAGENAME}}'''或称'''笔架山林场'''<ref name="a1"/>，是[[中华人民共和国|中国]][[广东省|广东省]][[清远市|清远市]]林业局下辖的一个[[林场|林场]]，辖区以'''笔架林场'''之名列为[[清新区|清新区]]的[[类似乡级单位|类似乡级单位]]<ref>{{Cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2017/44/18/441803.html|title=笔架林场 |accessdate=2018-09-17 |author= |date=|work= |publisher=国家统计局网站|language=Zh-hans }}</ref>。2018年6月，笔架林场的社会管理职能由清远市林业局移交给清新区人民政府<ref name="a1"/>。

林场成立于1958年，土地总面积96.7平方公里，其中国营林场5.8万亩，集体山林8.3万亩。内有一个市级自然保护区和一个广东省级森林公园<ref name="a1">{{Cite web |url=http://www.qingxin.gov.cn/info/3000551816|title=笔架山林场社会管理职能全面移交清新区 |accessdate=2018-09-17 |author= |date=2018-06-20 |work= |publisher=清新区人民政府网站|language=Zh-hans }}</ref>。

== 行政区划 ==

2017年时，代管七个[[行政村|行政村]]<ref name="a1"/>，分别是：坑口村、坑尾村、三坑滩村、中山村、根竹园村、车头村、田边村<ref>{{Cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2017/44/18/03/441803450.html|title=笔架林场 |accessdate=2018-09-17 |author= |date= |work= |publisher=国家统计局网站|language=Zh-hans }}</ref>。2018年时，相关资产已移交清新区人民政府<ref name="a1"/>。
