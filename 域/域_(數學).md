在[[抽象代数|抽象代数]]中，'''域'''（{{Lang-en|Field}}）是一种可進行[[加|加]]、[[減|減]]、[[乘|乘]]和[[除|除]]（除了[[除以零|除以零]]之外，「零」即加法單位元素）運算的[[代數結構|代數結構]]。'''域'''的概念是'''[[数域|数域]]'''以及[[四则运算|四则运算]]的推广。

域是[[环_(代数)|环]]的一种。域和一般的环的区别在于域要求它的元素（除'''零元素'''之外）可以进行[[除法|除法]]运算，这等于说每个非零的元素都要有[[逆元|乘法逆元]]。體中的運算关于乘法是可交换的。若乘法運算沒有要求可交換則稱為'''除環'''（{{Lang|en|division ring}}）或{{Lang|en|skew field}}。

==定义==
非正式的講，體是種集合，集合中的元素可以做兩種運算，"加法"：<math>a+b</math> 和"乘法"： <math>a\cdot b</math>, 且要求集合中任意元素 <math>a</math> 有加法反元素 <math>-a</math>，對所有非零元素 <math>b</math> 有乘法反元素 <math>b^{-1}</math>，這種性質讓我們可以用以下方法來定義加法和乘法的"反運算"，減法：<math>a-b</math> 和除法 <math>a/b</math>
:<math>a-b=a+(-b)</math>
:<math>a/b=a\cdot b^{-1}</math>

===定义1===
域是[[交换律|交换性]][[除环|除环]]。 

===定义2===
域是一種交換環(''F'', +, *)，當中加法單位元（0）不等於乘法單位元（1），且所有非零元素有乘法逆元。更簡單講就是：域是可交換[[除環|除環]]。

===定义3===
域是個集合 <math>F</math> 且帶有加法和乘法兩種運算，這裡“運算”可以想成是種映射，對任意兩元素
<math>a,b\in F</math>，這映射將此兩元素對應到某元素，且這些運算满足如下性质：
;在加法和乘法兩種運算上封閉
:對所有屬於F的<math>a, b</math>，<math>a+b</math>和<math>a*b</math>屬於F（另一種說法：加法和乘法是F上的二元運算）。
;加法和乘法符合結合律
:對所有屬於F的<math>a, b, c</math>，<math>(a+b)+c=a+(b+c)</math>，<math>(a*b)*c=a*(b*c)</math>
;加法和乘法符合交換律
:對所有屬於F的<math>a, b</math>，<math>a+b=b+a</math>，<math>a*b=b*a</math>
;符合乘法對加法的分配律
:對所有屬於F的<math>a, b, c</math>，<math>a*(b+c)=(a*b)+(a*c)</math>
;存在加法單位
:在F中有元素0，使得所有<math>a\in F</math>，<math>a+0=a</math>
;存在乘法單位
:在F中有不同于0的元素1，使得所有屬於F的<math>a</math>，<math>a*1=a</math>
;存在加法逆元
:對所有屬於F的<math>a</math>，存在<math>-a</math>使得<math>a+(-a)=0</math>
;非零元素存在乘法逆元
:對所有<math>a \ne 0</math>，存在元素<math>a^{-\!1}</math>使得<math>a*a^{-1}=1</math>

其中0 ≠ 1的要求排除了平凡的只由一个元素组成的域。

由以上性质可以得出一些最基本的推论：
:−(''a * b'') =(−''a'')* b = ''a'' *(−''b'')
:''a'' * 0 = 0
:如果''a * b'' = 0，则要么''a'' = 0，要么''b'' = 0

==例子==
* 許多常见的数域都是域。比如说，全体[[复数|复数]]的集合<math>\mathbb{C} </math>对于加法和乘法构成一个域。全体有理数的集合<math>\mathbb{Q}</math>也是一个域，它是<math>\mathbb{C} </math>的'''子域'''，并且不包含更小的子域了。
* [[代数数域|代数数域]]：代数数域是有理数域<math>\mathbb{Q}</math>的[[有限扩张|有限扩域]]，也就是说代数数域是<math>\mathbb{Q}</math>上的有限维[[向量空间|向量空间]]。代数数域都同构于<math>\mathbb{C} </math>的子域，并且这个同构保持<math>\mathbb{Q}</math>不变，即这个同构把每个有理数都映射到它自身。代数数域是[[代数数论|代数数论]]研究的对象。
* [[代数数|代数数]]构成的域：所有的[[代数数|代数数]]的集合对于加法和乘法构成一个域，记作<math>\overline{\mathbb{Q}}</math>。<math>\overline{\mathbb{Q}}</math>是有理数域<math>\mathbb{Q}</math>的代数闭包（见下）。<math>\overline{\mathbb{Q}}</math>是特征为零的[[代数封闭域|代数封闭]]的域的一个例子。
* 全体[[实数|实数]]的集合<math>\mathbb{R} </math>对于加法和乘法构成一个域。实数域是复数域<math>\mathbb{C} </math>的子域，也是一个[[有序域|有序域]]。后者使得[[实数|实数]]域上能够建立起[[微积分|微积分]]理论。
* 所有的实[[代数数|代数数]]的集合也构成一个域，它是<math>\mathbb{R} </math>的一个子域
* 任意一个[[有限域|有限域]]的元素个数是一个素数q的乘方，一般记作'''F'''<sub>q</sub>，就是所谓的[[伽罗瓦域|伽罗瓦域]]。任意一个元素个数是素数q的域都同构于'''Z'''/''p'''''Z''' = {0, 1, ..., ''p'' − 1}。令''p'' = 2,就得到最小的域：'''F'''<sub>2</sub>。'''F'''<sub>2</sub>只含有两个元素0和1运算法则如下：

* 设''E''和''F''是两个域，''E''是''F''的子域，则''F''是''E''的 '''扩域'''。设''x''是''F''中的一个元素，则存在着一个最小的同时包含''E''和''x''的''F''的子域，记作''E (x)''，''E (x)''称作''E''在''F''中关于  ''x''的'''[[单扩张|单扩张]]'''。比如说，复数域<math>\mathbb{C} </math>就是实数域<math>\mathbb{R} </math>在<math>\mathbb{C} </math>中关于[[虚数单位|虚数单位]]''i''的单扩张
* 每一个有乘法么元的环''R''都对应着一个包含它的域，称为它的[[分式域|分式域]]，记作''K''(''R'')。分式域的具体构造方法是定义类似于最简分数的等价类，再将环“嵌入”其中（详见[[分式域|分式域]]）。可以证明，''K''(''R'')是包含''R''的“最小”的域。
* 设''F''是一个域，定义''F (X)''是所有以''F''中元素为系数的[[分式|分式]]的集合，则''F (X)''是''F''的一个扩域。''F (X)''是''F''上的一个无穷维的[[向量空间|向量空间]]，这是域的[[超越扩张|超越扩张]]的一个例子。
* 设''F''是一个域，''p''(''X'')是[[多项式环|多项式环]]''F''[''X'']上的一个[[不可约多项式|不可约多项式]]，则[[剩余类环|商环]]''F''[''X'']/<''p''(''X'')>是一个域。其中的<''p''(''X'')>表示由''p''(''X'')生成的[[理想子环|理想]]。举例来说，'''R'''[''X'']/<''X''<sup>2</sup> + 1>是一个域（同构于复数域<math>\mathbb{C} </math>）。可以证明，''F''的所有单扩张都同构于此类形式的域。
* 若''V''是域''F''上的一个代數簇，则所有''V → F ''的有理[[函数|函数]]构成一个域，称为''V''的'''函数域'''。
* 若''S''是一个[[黎曼曲面|黎曼曲面]]，则全体''S → C ''的[[亚纯函数|亚纯函数]]构成一个域。
* 由于[[序数|序数]]的[[类_(数学)|类]]不是集合，因此在其上定义的[[尼姆数|尼姆数]]不能构成真正的域。但它满足域的所有条件，且其任意封闭子集（如小于<math>2^{2^n}</math>的所有[[自然数|自然数]]构成的子集）都是域。

== 基本性质 ==
* 域''F''中的所有非零元素的集合（一般记作''F''<sup>×</sup>）是一个关于乘法的[[阿贝尔群|阿贝尔群]]。''F''<sup>×</sup>的每个有限子群都是[[循环群|循环群]]。
* 若存在正整数''n''使得0 = 1 + 1 + ... + 1（''n''个1），那么这样的''n''中最小的一个称为这个域的'''特征'''，特征要么是一个素数''p''，要么是0（表示这样的''n''不存在）。此时<math>F</math>中最小的子域分别是<math>\mathbb{Q}</math>或有限域<math>\mathbb{F}_p</math>，称之为<math>F</math>的'''素域'''。
* 一个交换环是域当且仅当它的[[理想子环|理想]]只有自身和零理想。
* 在[[选择公理|选择公理]]成立的假设下，对每个域''F''都存在着唯一的一个域''G''（在同构意义上），''G''包含''F''，''G''是''F''的[[代数扩张|代数扩张]]，并且''G''[[代数封闭域|代数封闭]]。''G''称作由''F''确定的[[代数闭包|代数闭包]]。在很多情况下上述的同构并不是唯一的，因此又说''G''是''F''的一个代数闭包。
