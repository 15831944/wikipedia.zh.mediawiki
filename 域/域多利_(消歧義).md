'''域多利'''（{{lang-en|Victoria}}），是維多利亞的別譯。域多利可以指：

== 香港 ==

* '''[[域多利港|域多利港]]'''：

:: 又稱'''維多利亞港'''，是位於[[香港|香港]]的[[港口|海港]]，位處[[香港島|香港島]]與[[九龍|九龍半島]]之間，為香港[[市中心|市中心]]重要的地理區塊。由於港闊水深，又屬天然良港，香港亦因此有「[[東方之珠|東方之珠]]」的美譽。

* '''[[舊域多利醫院|舊域多利醫院]]'''：

:: 位於[[香港島|香港島]][[山頂|山頂]][[白加道|白加道]]15至17號，是[[香港|香港]]昔日一所公營[[醫院|醫院]]，在1897年至1947年間運作。醫院關閉後，主樓被拆卸並改建為[[政務司司長官邸|政務司司長官邸]]，建於1921年的產科翼樓保留，現被列為[[香港三級歷史建築|香港三級歷史建築]]。

* '''[[域多利軍營|域多利軍營]]'''：

:: 又稱'''域多利兵房'''，是[[香港|香港]]昔日[[駐港英軍|駐港英軍]]一個主要[[軍營|軍營]]，位於[[香港島|香港島]][[金鐘|金鐘]]，約於[[紅棉路|紅棉路]]、[[堅尼地道|堅尼地道]]及[[金鐘道|金鐘道]]交界一帶，鄰接當時的威靈頓軍營及[[美利軍營|美利軍營]]。現址為[[香港公園|香港公園]]的一部分，軍營內的部份建築物得以保留，舊三軍司令官邸被列為[[法定古蹟|法定古蹟]]（今[[茶具文物館|茶具文物館]]），另外數座建築物被列為[[香港一級歷史建築|香港一級歷史建築]]。

* '''[[域多利道|域多利道]]'''：

:: 是[[香港|香港]]主要道路之一，位於[[香港島|香港島]]區，是除了[[薄扶林道|薄扶林道]]之外唯一連接香港島[[堅尼地城|堅尼地城]]及[[薄扶林|薄扶林]]的道路。

* '''[[舊域多利軍營卡素樓|舊域多利軍營卡素樓]]'''：

:: 即'''香港視覺藝術中心'''，是[[康樂及文化事務署|康樂及文化事務署]]的[[藝術推廣辦事處|藝術推廣辦事處]]轄下的藝術發展中心，位於[[香港島|香港島]][[金鐘|金鐘]][[香港公園|香港公園]]內的西面，為一所讓[[視覺藝術|視覺藝術]]家互相觀摩學習交流創作發展的地方。

* '''[[域多利亞公眾殮房|域多利亞公眾殮房]]'''：

:: 是[[香港|香港]]一個公眾[[殮房|殮房]]，於1972年建成。殮房處於[[香港島|香港島]][[堅尼地城|堅尼地城]]西部（西環尾）的[[西寧街|西寧街]]（不過地址為[[域多利道|域多利道]]34號），樓高兩層，由[[衞生署_(香港)|衞生署]]管理。

* '''[[舊域多利軍營軍火庫|舊域多利軍營軍火庫]]'''：

:: 位於[[香港島|香港島]][[金鐘|金鐘]][[正義道|正義道]]9號。軍火庫庫附屬於[[域多利軍營|域多利軍營]]，四座建築物在1843年至1930年代落成，作為研製與儲存炸藥之用。[[駐港英軍|駐港英軍]]於1979年遷出軍營後，舊軍火庫遂成為政府工作間及倉庫，其後被[[亞洲協會|亞洲協會]]香港分會活化保護，於2012年改為亞洲協會香港中心的新會址。

* '''[[域多利城|域多利城]]'''：

:: 位於[[香港|香港]][[香港島|香港島]]西北岸，即現時香港[[中西區_(香港)|中西區]]及[[灣仔區|灣仔區]]一帶，為[[英國人|英國人]]於1841年佔領香港島後所建立的城市，中心為[[香港聖公會|香港聖公會]]的[[聖約翰座堂_(香港)|聖約翰座堂]]。故址位於今天[[香港島|香港島]]的政治與金融商業中心地帶，即[[中西區_(香港)|中西區]]及[[灣仔區|灣仔區]]。

* '''[[域多利精神病院|域多利精神病院]]'''：

:: 是[[香港|香港]]早期的[[精神病院|精神病院]]，設於[[香港島|香港島]][[西營盤|西營盤]][[東邊街|東邊街]]盡頭近[[般咸道|般咸道]]，在1884年啟用，其後歷經擴建，至1961年才被全新的[[青山醫院|青山醫院]]取代。

* '''[[域多利遊樂會|域多利遊樂會]]'''：

:: 是一個位於[[香港|香港]]西貢大網仔路的體育會。該機構成立於1849年，用以推動如[[划艇|划艇]]、[[风帆|風帆]]和渡海泳等水上活動，是香港歷史最悠久的體育會。

* '''[[域多利監獄|域多利監獄]]'''：

:: 前稱'''中央監獄'''，是[[香港|香港]]的首所[[監獄|監獄]]，位於[[香港島|香港島]][[中環|中環]][[奧卑利街|奧卑利街]]16號，1995年列為[[香港法定古蹟|香港法定古蹟]]，與[[舊中區警署|舊中區警署]]、[[前香港中央裁判司署|前香港中央裁判司署]]組成[[中區警署建築群|中區警署建築群]]。域多利監獄於2006年結役後，域多利監獄在內的[[中區警署建築群|中區警署建築群]]，復修為旅遊用途[[大館|大館]]。

* '''[[域多利道扣押中心|域多利道扣押中心]]'''：

:: 俗稱'''域多利拘留所'''、'''白屋'''，位於[[香港|香港]][[香港島|香港島]][[摩星嶺_(香港)|摩星嶺]][[域多利道|域多利道]]及摩星嶺徑交匯處，曾經是[[皇家香港警務處|皇家香港警務處]][[政治部_(香港警察)|政治部]]的[[拘留所|拘留所]]。

== 加拿大 ==

* [[維多利亞_(不列顛哥倫比亞)|'''域多利''']]：

:: 是[[加拿大|加拿大]][[卑詩省|卑詩省]]的一個[[城市|城市]]，為該省[[首府|首府]]以及[[首府地區_(卑詩省)|首府地區]]的行政中心。位於[[溫哥華島|溫哥華島]]最南端，面積19.47平方公里，南面瀕臨[[胡安·德·富卡海峽|胡安·德·富卡海峽]]，西接[[埃斯奎莫爾特|埃斯奎莫爾特]]，北臨[[沙尼治|沙尼治]]，東鄰[[橡樹灣|橡樹灣]]。

* '''[[維多利亞國際機場|域多利國際機場]]'''：
:: 是一座位於[[加拿大|加拿大]][[卑詩省|卑詩省]]首府地區[[北沙尼治|北沙尼治]]的國際機場，離該省省會[[維多利亞_(不列顛哥倫比亞)|維多利亞]]以北約22公里。

* '''{{Tsl|en|Victoria Inner Harbour Airport|域多利内港機場}}'''：
:: 位於加拿大的維多利亞港口，毗鄰加拿大[[不列顛哥倫比亞省|不列顛哥倫比亞省]]的維多利亞。該機場被[[加拿大運輸部|加拿大運輸部]]列為入境機場，並由{{tsl|en|Canada Border Services Agency|加拿大邊境服務局}}提供服務。
