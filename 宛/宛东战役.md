'''宛東戰役'''是[[中國人民解放軍|中國人民解放軍]]與[[國民革命軍|國民革命軍]]之間的一場戰役。此役是[[第二次國共內戰|第二次國共內戰]]攻守形势轉換階段南线解放軍转出外线后，第一次以国军一个机动[[兵团|兵团]]为目标的作战<ref>{{Cite web|url =http://www.hada.gov.cn/w_NewsShow.asp?ID=305:1098|title =档案里的故事----宛东战役|author =王亚丽|date =2011-03-04|publisher =南阳市宛城区档案馆|language =简体|accessdate =2014年2月12日|deadurl =yes|archiveurl =https://web.archive.org/web/20150402135449/http://www.hada.gov.cn/w_NewsShow.asp?ID=305%3A1098|archivedate =2015年4月2日}}</ref>。

== 背景 ==
1948年5月，中華人民共和國中央軍委指示[[中原野戰軍|中原野戰軍]]牵制[[國民革命軍|國民革命軍]][[胡琏|胡琏]]兵团于[[平漢路|平漢路]]以西，以利解放軍[[粟裕|粟裕]]兵团渡黄河南下。為此时任中原野战軍司令员[[劉伯承|劉伯承]]、政治委员[[鄧小平|鄧小平]]根据中共中央军委的指示，发起了宛东战役。
== 戰役準備==
解放軍方面：
* 以[[陈唐|陈唐]]兵团附秦纵一个旅在[[郾城|郾城]]附近箝制整十八军。
* 以[[陈锡|陈锡]]联指挥一、三、六纵共三个纵队作为东兵团，于五月二十七日佯攻[[确山|确山]]，适时西进歼击[[张轸|张轸]]兵团。
* 以[[陈赓|陈赓]]指挥四、二、十共三个纵队及[[桐柏|桐柏]]部队作为西兵团，由[[南阳|南阳]]地区尾张轸兵团东进，在[[赊旗镇|赊旗镇]]、[[唐河|唐河]]以东地区开始攻擊，以利东兵团西进围歼之。
== 經過 ==
* [[陈唐|陈唐]]兵团自5月31日至6月3日，阻击由[[临颖|临颖]]南下整十八军于[[漯河|漯河]]地区，毙伤國軍二千二百余人，解放軍伤亡五百七十人，伤亡为四比一，完全达成其箝制任务。
* 西兵团以四纵之十、十一两旅于5月29日在[[赊旗镇|赊旗镇]]以南之埠口扭击[[张轸|张轸]]兵团。是夜，由于该两旅向赊旗镇以北撤退，西兵团又迷于表面的情况，30日将四纵主力东调羊册、郭集地区，将二、十两纵东调羊册西南地区（实际该两纵因执行解放軍28日四时基本命令，只到了[[高庙集|高庙集]]、[[兴隆镇|兴隆镇]]、[[苗庙|苗庙]]地区）而夹击之，以致张轸于31日晨由埠口经桥头镇向[[南阳|南阳]]回撤。西兵团在就势进击之中，仅在马刘营、连庄殺傷[[五十八师|五十八师]]六千余名，另在[[娄子庄|娄子庄]]、[[小司庄|小司庄]]之线阻击由[[南阳|南阳]]地区回援之國軍，共截击与歼灭张轸万余人，其中俘虏五十八师副师长肖本元以下五千三百余人。
* 东兵团于5月27日佯攻确山，吸引敌整十八军由临颖向[[漯河|漯河]]南进，也吸引了张轸兵团由[[南阳|南阳]]东援，正阳之敌整二十八师不敢单独先援确山。28日，该兵团除留六纵之十八旅一部仍围确山佯动，一纵之二十旅监视正阳之敌外，主力即由确山地域先向[[春水|春水]]、[[泌阳|泌阳]]之线西进，适机兜歼张轸兵团。
