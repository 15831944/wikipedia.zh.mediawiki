'''迪里拜尔·尤努斯'''（{{lang-ug|دىلبەر يۇنۇس}}，{{lang-fi|Dilber Yunus}}，[[艺名|艺名]]：Dilbèr，{{bd|1958年|10月2日|||catIdx=D迪}}），[[中国|中国]][[维吾尔族|维吾尔族]]抒情[[花腔|花腔]][[女高音|女高音]]歌唱家，被誉为中国的[[菲洛墨拉|菲洛墨拉]]。她具有[[芬兰|芬兰]]国籍<ref name="hs"/><ref name="110.com"/>。

== 生平 ==
迪里拜尔1958年出生于中国[[新疆|新疆]][[喀什|喀什]]。1976年迪里拜尔考入[[新疆歌舞团|新疆歌舞团]]，師從[[郭凌弼|郭凌弼]]。1978年畢業時舉辦獨唱音樂會。1980年考入[[中央音乐学院|中央音乐学院]]声乐歌剧系，1987年获[[硕士|硕士]]学位。在此期間她主要師從於[[沈湘|沈湘]]、[[李晉瑋|李晉瑋]]。1984年，迪里拜尔在第一届{{tsl|fi|Mirjam Helin|米利埃姆·海林}}（Mirjam Helin）国际歌唱比赛中获奖。1988年应聘到[[芬兰|芬兰]]国家歌剧院担任独唱，随后成为[[芬兰国家歌剧院|芬兰国家歌剧院]]及[[瑞典|瑞典]]{{tsl|en|Malmö Opera|马尔默歌剧院}}终身独唱艺术家。自2008年至2018年，迪里拜尔任职中国音乐学院声乐歌剧系特聘教授，硕士生导师<ref>{{Cite web|url=http://www.ccmusic.edu.cn/subsite/sgx/2015/12/06/%E8%BF%AA%E9%87%8C%E6%8B%9C%E5%B0%94/|title=迪里拜尔（特聘教授） {{!}} 中国音乐学院声歌系|accessdate=2018-09-28|work=www.ccmusic.edu.cn}}</ref>。2018年9月，迪里拜尔重返中央音乐学院，担任声乐歌剧系教授<ref>{{Cite web|url=http://www.ccom.edu.cn/xwyhd/xsjd/2018f/201809/t20180912_50120.html|title=中央音乐学院-中央音乐学院2018-2019学年度开学典礼暨教师节庆祝大会隆重举行|accessdate=2018-09-28|work=www.ccom.edu.cn}}</ref><ref>{{Cite web|url=http://culture.people.com.cn/n1/2018/0821/c1013-30241151.html|title=歌剧院很多，迪里拜尔只有一个|author=韩轩|publisher=人民网|accessdate=2018-09-28}}</ref>。

迪里拜尔已在几十个国家举行了数百场独唱音乐会，用[[意大利语|意大利语]]、[[德语|德语]]、[[英语|英语]]、[[法语|法语]]、[[西班牙语|西班牙语]]等多种语言演出过几十部世界著名歌剧，并同许多世界著名指挥家及交响乐团合作。1993年到1996年在[[德國|德國]][[波恩|波恩]]國家歌劇院任獨唱演員。1997年、1998年连续获世界著名女高音歌唱家[[尼尔松大奖|尼尔松大奖]]（Brigit Nilsson Prize）——瑞典最佳歌剧演员奖。2004年11月，获中国金唱片专辑奖和最佳演员奖。

作为[[芬兰独立宣言|芬兰独立100周年]]庆祝活动的一部分，迪里拜尔于2017年4月8日在北京[[中山音乐堂|中山音乐堂]]举办独唱音乐会。音乐会的节目中包括芬兰著名作曲家[[西贝柳斯|西贝柳斯]]和[[奥斯卡·梅利康托|梅利康托]]的作品。<ref name="finland.cn"/>

== 主要歌劇角色 ==
迪里拜爾演出的主要歌劇角色有：
*[[威尔第|威尔第]]的《[[弄臣|弄臣]]》中的吉爾達
*[[多尼采蒂|多尼采蒂]]的《[[拉美莫尔的露琪亚|拉美莫尔的露琪亚]]》中的路其亞
*[[莫扎特|莫扎特]]的《[[魔笛|魔笛]]》中的夜女王
*[[贝里尼|贝里尼]]的《{{tsl|en|La sonnambula|夢遊女}}》中的莉薩
*[[斯特拉文斯基|斯特拉文斯基]]的《[[夜鶯|夜鶯]]》中的夜鶯
*[[多尼采蒂|多尼采蒂]]的《[[爱情灵药|爱情灵药]]》中的阿迪娜
*[[多尼采蒂|多尼采蒂]]的《{{tsl|en|La fille du régiment|团隊的女兒}}》中的玛丽
*[[普契尼|普契尼]]的《[[贾尼·斯基基|贾尼·斯基基]]》中的劳莱塔
*[[马斯涅|马斯涅]]的《[[维特|维特]]》中的苏菲
*[[雅克·奥芬巴赫|奥芬巴赫]]的《[[霍夫曼的故事|霍夫曼的故事]]》中的奥林匹亚
*[[理查德·施特劳斯|理查德·施特劳斯]]的《{{tsl|en|Ariadne auf Naxos|阿里阿德妮在纳克索斯}}》中的泽比内塔
*[[理查德·瓦格纳|理查德·瓦格纳]]的《[[帕西法尔|帕西法尔]]》中的花仙
*[[罗西尼|罗西尼]]的《[[塞维利亚的理发师|塞维利亚的理发师]]》中的罗西娜
*[[威尔第|威尔第]]的《[[化妝舞會_(歌劇)|化妝舞會]]》中的奥斯卡

== 家庭 ==
迪里拜尔于2004年10月与中国电影艺术家[[于洋_(中国电影演员)|于洋]]之子、电影导演[[于晓阳|于晓阳]]结婚。2005年1月，于晓阳在拍片旅途中突然病故，迪里拜尔由于悲痛，一度失声。2006年2月，迪里拜尔在经历个人不幸一年之后，首次在[[北京|北京]][[保利剧院|保利剧院]]召开独唱音乐会。
