'''赫尔曼·菲舍尔'''（{{lang-de|Hermann Fischer}}，{{bd|1884年|4月6日|1962年|7月18日}}），[[德国|德国]][[建筑师|建筑师]]，[[津浦铁路济南车站旧址|津浦铁路济南车站]]的设计者。

== 生平 ==
1884年4月6日，赫尔曼·菲舍尔出生于德国[[南哈尔茨|南哈尔茨]]的{{tsl|de|Roßla|罗斯拉}}，曾就读于希尔德堡豪森大学，毕业后成为一名建筑师。1908年，赫尔曼·费舍尔通过[[西伯利亚铁路|西伯利亚铁路]]抵達中国，并在山东济南任职<ref name="生活日报">{{zh-hans}}{{cite web |author1=雍坚 |author2=王倩 |title=设计老火车站时,菲舍尔只有24岁他在济南恋爱生子,待了七个年头 |url=http://paper.dzwww.com/shrb/PDF/20121206/A14.pdf |page=A14 |website=大众网-生活日报 |accessdate=2019-02-21 |date=2012-12-06 |archiveurl=https://web.archive.org/web/20190301061701/http://paper.dzwww.com/shrb/PDF/20121206/A14.pdf |archivedate=2019-03-01 |deadurl=no}}</ref>。1909年至1913年期间，他受中德两国政府委托参与了[[膠濟鐵路|胶济铁路]]的建设，1908年至1914年間則投入[[津浦鐵路|津浦鐵路]]的興建工程，并主持设计了[[津浦铁路济南车站旧址|津浦铁路济南车站]]，於1912年使該站完工<ref name="法蘭克福匯報">{{de}}{{Cite news |script-title=de:Deutscher Bahnhof in China : Nostalgie plus Shopping-Mall |url=https://www.faz.net/aktuell/gesellschaft/menschen/deutscher-bahnhof-in-china-nostalgie-plus-shopping-mall-12599551.html |publisher={{lang|de|[[法蘭克福匯報|Frankfurter Allgemeine]]}} |author={{lang|de|Petra Kolonko}} |date= 2013-10-01|archiveurl=http://archive.is/JAya6 |archivedate=2019-03-01}}</ref>。因此，中国政府在1914年授予他七等[[嘉禾獎章|嘉禾奖章]]<ref name="舜网" />。此外，[[津浦铁路局济南机器厂旧址|津浦铁路局济南机器厂]]的道格米里办公大楼也是出自他的设计<ref>{{zh-hans}}{{cite web |author1=张汝峰 |title=人文齐鲁 济南铁路大厂的百年记忆 |url=http://news.ifeng.com/a/20170831/51826827_0.shtml |website=凤凰网资讯-齐鲁晚报社 |accessdate=2019-02-21 |date=2017-08-31 |archiveurl=http://archive.is/rX1jQ |archivedate=2019-02-22|deadurl=no}}</ref>。1910年，菲舍尔夫妻在济南修建住宅。1914年，菲舍尔离开济南前往[[法国|法国]]。1926年，他移民[[菲律宾|菲律宾]][[马尼拉|马尼拉]]，1962年7月18日在马尼拉去世<ref name="生活日报" />。

== 家庭 ==
赫尔曼·菲舍尔的妻子阿斯塔（{{lang|de|Asta}}）是其駐华工作期间的同事阿图尔·加布勒-贡波特（{{lang|de|Arthur Gabler-Gumbert}}）的妹妹，比赫尔曼小7岁<ref name="生活日报" />，两人在阿斯塔到[[中國|中国]]探望哥哥时相识。赫尔曼·费舍尔在[[长城|长城]]上向阿斯塔求婚，两人于1910年11月1日在[[青岛|青岛]]的一家教堂结婚，婚后曾在[[北京|北京]]生活过<ref name="齐鲁晚报">{{zh-hans}}{{cite web|author1=董从哲|author2=马云云|title=济南老火车站设计者费舍尔：在中国收获浪漫爱情|url=http://www.chinanews.com/cul/2012/05-22/3906409.shtml|website=中国新闻网-齐鲁晚报|accessdate=2019-02-20|date=2012-05-22|archive-url=https://web.archive.org/web/20120524093521/http://www.chinanews.com/cul/2012/05-22/3906409.shtml|archive-date=2012-05-24|dead-url=no}}</ref>。1928年阿斯塔随丈夫移居菲律宾，1971年回到德国，1977年在德国去世。

夫妻二人育有三子：

* 长子赫伯特·菲舍尔于1911年2月16日诞生在济南；
* 二儿子维尔纳·费舍尔与儿媳乌苏拉·韦伯（{{lang|de|Ursula Weber}}）在[[上海市|上海]]相识结婚，乌苏拉出生于中国，随费舍尔一家迁往[[菲律宾|菲律宾]][[马尼拉|马尼拉]]之前一直在上海生活<ref name="齐鲁晚报" />。
** 孙女西维亚出生在菲律宾，15岁时回到德国<ref>{{zh-hans}}{{cite web|author1=王磊|author2=赵洪栋|author3=隋乔|author4=段海丽|title=独家对话西维亚：济南找到“心灵之根”|url=http://www.dzwww.com/2012/jnlhcz/zxbd/201212/t20121212_7785370.htm|website=大众网|access-date=2019-02-20|archive-url=https://web.archive.org/web/20131026063659/http://www.dzwww.com/2012/jnlhcz/zxbd/201212/t20121212_7785370.htm|archive-date=2013-10-26|dead-url=no}}</ref>。
* 小儿子生于1918年8月27日，二战期间在[[菲律宾|菲律宾]][[巴丹省|巴丹]]被日本人杀害<ref name="生活日报" />。

== 纪念 ==
