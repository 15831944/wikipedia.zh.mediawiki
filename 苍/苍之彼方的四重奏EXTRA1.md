{{Vgname|蒼之彼方的四重奏 EXTRA1|ja=蒼の彼方のフォーリズム EXTRA1}}是由[[sprite_(遊戲品牌)|sprite]]於2017年6月30日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]
<ref>{{cite web |title=有坂真白のアフターストーリー！sprite『蒼の彼方のフォーリズム EXTRA1』が本日発売 |url=http://moedigi.com/game/2017/06/aokana-extra1-release/ |website=moedigi|language=ja |date=2017年6月30日}}</ref>，《[[蒼之彼方的四重奏|蒼之彼方的四重奏]]》的衍生作品。本作沿用前作的世界观和设定，以前作有坂真白的路线来展开，讲述了有坂真白和日向晶也的恋爱日常。

==故事介绍==
憧憬着的前辈，我最喜欢的前辈。可以相互倾诉烦恼，一起努力的伙伴们。鼓起勇气伸出双手，紧紧抓住重要的东西不放，最终抵达幸福的终点站。思虑一闪而过。成绩接近红线；和莉佳、有梨华进行了甜点比赛，然而惨败；美咲和明日香宣称“晶也的情人”!?这样的发展继续下去，前辈会被抢走，我也会留级，“真白乌冬面”将面临倒闭!?就在此时，母亲牡丹突然提议让前辈和我约会一晚。”我要做个可口的便当！再打扮得漂漂亮亮！”我对晶也这样说。在约会日到来之前，在大家的帮助下，展开突击特训，来提升的我女子力！！到底，晶也和真白是否真的能够迎来HappyEnd呢？

== 登场角色==
=== 主要角色 ===
; {{nihongo|日向晶也|日向 晶也 （ひなた まさや） }}
:本作男主角，真白的恋人，一直支持着她。
; {{nihongo|有坂真白|有坂 真白 （ありさか ましろ） |聲：瀬良みと}}
:家里开的乌冬面店是四岛列岛的地方特产。久奈浜FC部唯一的1年级学生。把失去干劲的美咲带回了FC部，体会到了努力的乐趣.。特别喜欢总是支持着自己的晶也，交往以后一直向他撒娇。

=== 其他角色 ===
; {{nihongo|仓科明日香| 倉科 明日香（くらしな あすか）|聲：泽田なつ}}
:对谁都是礼貌地讲敬语。坦率的性格，是一心一意努力的人。将真白视为可爱的学妹爱护着，也教导了她要保持放松的心态。
; {{nihongo|鸢泽美咲| 鳶沢みさき（とびさわ みさき）|聲：绀野由梨}}
一时心血来潮的心肠。特别喜欢乌冬面。即便是真白和晶也成为了恋人的关系，她还是真白爱慕的前辈。
; {{nihongo|市之濑莉佳| 市ノ瀬 莉佳 （いちのせ りか）|聲：三代眞子}}
:在高藤上学的日向家的邻居。通过与真白组建'''速度同盟'''，成为了可以拜访对方的家的朋友。作为真白挚友应援着她的恋情。
; {{nihongo|青柳窗果| 青柳 窓果 （あおやぎ まどか）|聲：铃田美夜子}}
:晶也的同班同学，久奈浜FC部的经理兼部长。一边欺负着真白和晶也，一边温柔地守护着他们。
; {{nihongo|保坂实里| 保坂実里（ほさか みのり）|聲：桃井穂美}}
:真白的同班同学兼广播部部员。间接担当着真白的恋爱顾问，总是被她牵着鼻子走。
; {{nihongo|虎鱼有梨华| 虎魚有梨華（おこぜ ありか）|聲：阳月ひおり}}
:真白的幼驯染。四岛水产1年级生。只要和真白见面就会发生争吵，实际上非常关心她。
; {{nihongo|有坂牡丹| 有坂牡丹（ありさか ぼたん）|聲：大和えみ}}
:真白的母亲，和真白父亲一起把“'''真白乌冬店'''”搞得红红火火。溺爱独生女的真白，也给她的男友晶也提供帮助。
; {{nihongo|真藤一成| 真藤一成（しんどう かずなり）|聲：梅咲チャーリー}}
:高藤FC部三年级原部长。引退后也曾作为教练留在社团活动中。憧憬幼年时期的日向晶也而开始了FC，对晶也抱着不寻常的感情。
