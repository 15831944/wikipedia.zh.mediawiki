《'''她愛上了我的謊'''》（{{lang-ja|カノジョは嘘を愛しすぎてる}}）是[[日本|日本]]漫畫家[[青木琴美|青木琴美]]的[[少女漫畫|少女漫畫]]作品，由《[[Cheese!|Cheese!]]》的2009年5月號起於該雜誌開始連載。曾獲得第59屆[[小學館漫畫賞|小學館漫畫賞]]（少女向部門）<ref>{{cite web|url = http://natalie.mu/comic/news/108250 |language = ja |title = <span lang = "ja">第59回小学館漫画賞に土竜の唄、マギ、カノ嘘など輝く</span> |publisher = [[Natalie_(網站)|Natalie]] |date = 2014-01-22 |accessdate = 2014-01-30}}</ref>。改編[[電影|電影]]於2013年12月在日本上映，由[[佐藤健|佐藤健]]主演，女主角為[[大原櫻子|大原櫻子]]<ref>{{cite web|url = http://eiga.com/news/20130307/1/ |language = ja |title = <span lang = "ja">佐藤健「カノジョは嘘を愛しすぎてる」に主演！ヒロインは17歳・現役女子高生が大抜てき</span> |publisher = eiga.com |date = 2013-03-07 |accessdate = 2014-01-30}}</ref>。2017年3月韓國[[tvN|tvN]]播出改編成電視劇的《[[她愛上了我的謊_(韓國電視劇)|她愛上了我的謊]]》。

== 電視劇 ==
《她愛上了我的謊》外傳，標題為《'''她愛上了我的謊 另一個故事 〜我和她相會的故事〜'''》<ref>[http://news.mynavi.jp/news/2013/11/12/264/佐藤健主演『カノ嘘』、オムニバスドラマ全10話の地上波放送が決定!]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }} マイナビニュース</ref>。

2017年3月韓國[[tvN|tvN]]播出改編成電視劇的《[[她愛上了我的謊_(韓國電視劇)|她愛上了我的謊]]》，由[[李玹雨|李玹雨]]、[[Joy_(Red_Velvet)|Joy]]主演<ref>[http://www.koreastardaily.com/tc/news/89903 tvN新月火劇《她愛上我的謊》主演陣容確定！李玹雨、Joy、李廷鎮、李瑞元]</ref>。

同時，中國方面，宣布翻拍同名劇集，計劃於2018年第二季7月開拍，於北京衛視和江蘇衛視首播，集數為50，男女主角未定。
