《'''她和我和戀人。'''》（{{lang|ja|彼女と俺と恋人と。}}）是日本[[WillPlus|WillPlus]]公司旗下品牌[[PULLTOP|PULLTOP LATTE]]在2012年12月14日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]。<ref>{{cite web|url=http://www.getchu.com/soft.phtml?id=749896|script-title=ja:彼女と俺と恋人と。 初回版|publisher=Getchu.com|accessdate=2015年10月29日|language=ja}}</ref>遊戲簡稱「{{lang|ja|ととと}}」。<ref>{{cite web|url=https://vndb.org/v10972|script-title=en:Kanojo to Ore to Koibito to.|trans_title=她和我和戀人。|publisher=The Visual Novel Database|accessdate=2015年10月29日|language=en}}</ref>2014年10月31日發售與次作《{{lang|ja|恋する夏のラストリゾート}}》合製[[Fan_disc|Fan disc]]《{{lang|ja|彼女と俺と、恋するリゾート ～彼女と俺と恋人と。＆恋する夏のラストリゾートWファンディスク～}}》<ref>{{cite web|url=http://www.getchu.com/soft.phtml?id=820679|script-title=ja:彼女と俺と、恋するリゾート ～彼女と俺と恋人と。＆恋する夏のラストリゾートWファンディスク～|publisher=Getchu.com|accessdate=2015年10月29日|language=ja}}</ref>和3作合集《{{lang|ja|PULLTOP LATTE 彼女と恋する3ピースパック}}》。<ref>{{cite web|url=http://www.getchu.com/soft.phtml?id=820470|script-title=ja:PULLTOP LATTE 彼女と恋する3ピースパック|publisher=Getchu.com|accessdate=2015年10月29日|language=ja}}</ref>

== 故事 ==
松上遥人為了救遭遇交通事故的美萩野綾乃而受傷昏迷住院。在住院的這段時間綾乃每天探望，出院時也要求繼續每天到陣富莊照顧遥人。遥人的傷勢復原後在準備離別的時候，遥人接受綾乃的告白而成為一對戀人並且開始兩人的同居生活。然而隨著其他女性陸續的來訪，兩人的甜蜜生活會有什麼變化。

== 角色 ==
=== 主要角色 ===
; {{nihongo|松上遙人|松上 遥人（まつがみ はると）}}
: 本作男主角，大學1年級學生。獨自住在陣富莊。將優子視作妹妹，很寵愛妹妹すすき。
; {{nihongo|美荻野綾乃|美萩野 綾乃（みはぎの あやの）||配音：奏雨<ref>{{cite web|url=http://www.pulltop.com/PL01/character.html|script-title=ja:美萩野 綾乃|publisher=[[PULLTOP_LATTE|PULLTOP LATTE]]|accessdate=2017年1月22日|language=ja}}</ref>}}
: 知德學院3年級學生，溫柔美麗的大小姐。在車禍發生時被遙人所救，之後便時常探望遙人，遙人出院後仍每天去他家中照顧他。在遙人手康復後，因為沒有理由可以在來照顧他，於是在送別前主動吻了遙人，經一番波折後兩人成為了戀人。
; {{nihongo|德吉優子|徳吉 優子（とくよし ゆうこ）||配音：美月<ref>{{cite web|url=http://www.pulltop.com/PL01/character2.html|script-title=ja:徳吉 優子|publisher=PULLTOP LATTE|accessdate=2017年1月22日|language=ja}}</ref>}}
: 知德學院3年級學生，遥人的青梅竹馬。喜歡遥人但沒有告白。成績很好，在學校受許多人敬愛。
; {{nihongo|白兎筑紫|白兎 つくし（はくと つくし）||配音：{{lang|ja|百瀬ぽこ}}<ref>{{cite web|url=http://www.pulltop.com/PL01/character3.html|script-title=ja:白兎 つくし|publisher=PULLTOP LATTE|accessdate=2017年1月22日|language=ja}}</ref>}}
: 知德學院3年級學生，綾乃的朋友。因為很重視綾乃而敵視遥人，為了保護綾乃而來到陣富莊。綾乃生她的氣時會變得很老實。
; {{nihongo|小澤見千景|小沢見 千景（こぞみ ちかげ）||配音：[[遠野Soyogi|遠野Soyogi]]<ref>{{cite web|url=http://www.pulltop.com/PL01/character4.html|script-title=ja:小沢見 千景|publisher=PULLTOP LATTE|accessdate=2017年1月22日|language=ja}}</ref>}}
: 陣富莊的管理員。身材很好的美人。不會干涉遥人和綾乃的同居生活。酒量不好，喝醉後個性會變得撫媚，事後沒有記憶。
; {{nihongo|美荻野小乃花|美萩野 小乃花（みはぎの このか）||配音：[[小倉結衣|小倉結衣]]<ref>{{cite web|url=http://www.pulltop.com/PL01/character5.html|script-title=ja:美萩野 小乃花|publisher=PULLTOP LATTE|accessdate=2017年1月22日|language=ja}}</ref>}}
: 綾乃的妹妹。因為好奇而偷偷跟蹤姊姊而來到陣富莊。

=== 其他角色 ===
; {{nihongo|松上芒|松上 すすき（まつがみ すすき）|配音：{{lang|ja|藤咲ウサ}}<ref>{{cite web|url=http://www.pulltop.com/PL01/character6.html|script-title=ja:松上 すすき|publisher=PULLTOP LATTE|accessdate=2017年1月22日|language=ja}}</ref>}}
