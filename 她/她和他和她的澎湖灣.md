《'''她和他和她的澎湖灣'''》是[[台湾|台湾]]艾蘿塔斯有限公司即Erotes Studio制作组继《[[雨港基隆|雨港基隆]]》和《[[五月茉莉|五月茉莉]]》后所制作的一部[[视觉小说|视觉小说]]兼[[美少女游戏|美少女游戏]]。游戏故事舞台设定在1949年的[[澎湖縣|澎湖縣]][[馬公市|馬公市]]，改编自[[澎湖七一三事件|澎湖七一三事件]]<ref>{{cite web |author1=洪嘉蓮 |title=二二八事件、澎湖七一三事件⋯⋯《返校》之外，這些遊戲帶你認識台灣史 |url=https://theinitium.com/article/20170126-city-games-taiwan-indie-play/ |website=[[端傳媒|端傳媒]] |accessdate=2018-08-10 |date=2017-01-26}}</ref>，并于2017年7月13日正式发售<ref name="正式發售">{{cite web |author1=Jessica |title=經數度延期後《她和他和她的澎湖灣》正式發售 |url=https://gnn.gamer.com.tw/1/149641.html |website=GNN新聞 |accessdate=2018-08-10 |date=2017-07-13}}</ref>。

== 剧情 ==
{{Plot style|time=2018-08-10}}
1949年，身为[[山東省_(中華民國)|山东]]学生的游戏男主角尹曉風因为[[国共内战|国共内战]]而跟随其他学生逃亡到澎湖，后与另外两名来自山东的女生孫儀芳和鄧琳相识并产生恋爱感情<ref>{{cite web |author1=Joel Couture |title=Blue Blood Lagoon Is A Romance Story Set In The Chinese Civil War |url=http://www.siliconera.com/2016/07/30/blue-blood-lagoon-romance-story-set-chinese-civil-war/ |website=siliconera |accessdate=2018-08-10 |date=2016-07-30 |archive-url=https://web.archive.org/web/20170520081914/http://www.siliconera.com/2016/07/30/blue-blood-lagoon-romance-story-set-chinese-civil-war/ |archive-date=2017-05-20 |dead-url=no }}</ref>。

== 制作 ==
Erotes Studio制作组的[[首席执行官|首席执行官]]李處守称希望通过制作《她和他和她的澎湖灣》，使玩家在玩游戏的过程中把自己代入到当时的历史事件中，去了解当时的历史，以达到比[[教科书|教科书]]和历史书藉更好的教育效果。游戏的制作耗费了两年的时间，期间Erotes Studio制作组曾实地考察澎湖并采访澎湖七一三事件的经历者，并查阅了不少文献。<ref>{{cite web |author1=吳柏緯 |title=713、228與鄭南榕…玩遊戲也能了解台灣歷史 |url=http://news.ltn.com.tw/news/life/paper/1091671 |website=[[自由時報|自由時報]] |accessdate=2018-08-10 |date=2017-04-05 |archive-url=https://web.archive.org/web/20170611025921/http://news.ltn.com.tw/news/life/paper/1091671 |archive-date=2017-06-11 |dead-url=no }}</ref>
