'''孛兒只斤氏'''{{NoteTag|{{标音|繁=博爾濟吉特|简=孛儿只斤|拼音=Bóérjìjǐn}}}}（{{mong|m=ᠪᠣᠷᠵᠢᠭᠢᠨ|Ppp=Borǰigin|k=Боржигин}}），[[清代|清代]]譯為'''博爾濟吉特氏'''（{{lang-mnc|ᠪᠣᡵᠵᡳᡤᡳᡨ|v=Borjigit|a=Borjigit|p=Borzhigit}}）或'''博尔济锦氏'''。稱“特”者，為其複數形式，表示人數多的意思<ref>{{cite web |url = http://ctext.org/library.pl?if=gb&file=25374&page=111 |title = 《天咫偶聞》影印本 }}</ref>。[[孛端察兒|孛端察兒]]的後裔為孛兒只斤氏，後[[也速該|也速該]]恢復使用此氏族名<ref>《蒙古秘史》，余大鈞譯，河北人民出版社，2001年5月出版，30頁、41頁</ref>。

== 元代 ==
[[成吉思汗|成吉思汗]]統一蒙古後規定，只有本家的嫡系後裔，才有繼承蒙古大汗及留在蒙古本部的資格；後世便稱作[[黄金家族|「黃金血胤」或「黃金氏族」]]。'''博爾濟吉特氏'''於[[科尔沁部|蒙古科爾沁]]一脈，源於成吉思汗的弟弟[[哈布圖哈薩爾|哈布圖哈薩爾]]。

== 清代 ==
在清朝時期，因後宮中有許多后妃皆出此姓氏，使得博爾濟吉特氏成為一個很顯赫的姓氏。来自[[科尔沁部|科尔沁部]]的人数最多，尤其是在清朝初年的[[清朝後宮#關外五宮制|崇德五宮]]。

* [[清太宗|清太宗]]：[[孝端文皇后|孝端文皇后]]、[[孝莊文皇后|孝莊文皇后]]、[[敏惠恭和元妃|敏惠恭和元妃]]、[[懿靖大貴妃|懿靖大貴妃]]（出于[[阿巴噶部|阿巴噶部]]）、[[康惠淑妃|康惠淑妃]]、[[扎鲁特博尔济吉特氏|东宫福晋]]（出于[[扎鲁特部|扎鲁特部]]）
* [[清世祖|清世祖]]：[[世祖廢后|世祖廢后]]、[[孝惠章皇后|孝惠章皇后]]、[[淑惠妃|淑惠妃]]、[[恭靖妃|恭靖妃]]、[[端順妃|端順妃]]、[[悼妃|悼妃]]

== 现代 ==
在[[外蒙古|外蒙古]]，1920年代[[蒙古人民革命党|蒙古人民革命党]]為改进税收（当时姓可以被用来判断一个人往日地位的高低），遂下令取消[[姓氏|姓氏]]，導致同名字的人太多。儘管1997年[[蒙古国|蒙古国]]政府通过法律要求国民重冠姓氏，但是此法规的影响并不大。至2004年蒙古国政府強制國民[[身份證|身份證]]上冠姓，但又衍生許多國民以'''博爾濟吉特'''為姓氏的問題。至2004年10月，约有90%的公民为自己冠姓，但是姓大多數-{只}-在正式场合使用，一些人還是不记得自己的姓<ref>http://seattletimes.nwsource.com/html/nationworld/2002071306_mongolia24.html</ref>。
自古以來，蒙古族命名傳統並無類似漢人的「姓」(或Family name)的概念，姓名結構是「父名」+「子名」(親子聯名制)。

蒙古籍日本相撲69代橫綱白鵬，其蒙古名為「芒可巴特·達瓦扎爾格勒」(其前名「芒可巴特」為父名，後名「達瓦扎爾格勒」為自己的名，仍無所謂的姓或家族名Family Name)，蒙古籍日本相撲人物[[朝青龍明德|道勒格爾蘇倫·達格瓦道爾吉]]、[[日馬富士公平|達瓦尼馬·拜馬道爾吉]]、[[鶴龍力三郎|馬納恩加加拉文·阿納德]]（日本第68、70、71代橫綱）也是一樣。
