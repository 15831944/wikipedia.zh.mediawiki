'''粵語[[懶音|懶音]]'''指的是[[粵語|粵語]]語音的內部規律中的流變。這一現象可以增加同音字而且導致意義上的歧義，從而得名懶音。但懶音一稱也存有各種爭議，因為某些音變中的合併及變異并不起因在懶惰。懶音或許起因在沒有正式地接觸標準發音的教育，或者甚至是受到鄉音的影響。粵語懶音普遍出現在[[韻母|韻母]]上，而也有部分的懶音出現在複雜的[[輔音|輔音]]、聲調當中。

==[[聲母|聲母]]==
===n、l合流===
====因素====
* 客家口音的遗留（[[香港客家話|香港客家話]]n-、l-不分。）
* 语流音变
:20世紀50年代，在[[香港|香港]]開始出現n、l合一的現象。<br>
* [[語音同化|語音同化]]
* 方言口音（[[南海|南海]][[口音|口音]]、[[西關口音|西關口音]]、番禺口音、順德口音）
* 其他[[汉语方言|汉语方言]]的影响（20世纪70年代香港鼓励粤语取代其他汉语方言在公众场合使用，原先以客家话、[[闽南话|闽南话]]等方言为母语的使用者在学习粤语时，带入母语n/l不分的发音习惯。而[[香港客家話|香港客家話]]是　n-、l-不分。）

====現狀====
粵語[[廣府話|廣府方言]]的使用者，尤其是部分在[[廣東|廣東]]和[[香港|香港]]地區的年輕人{{Citation needed}}，鼻音n和舌邊音l在日常生活当中一般是自由变体，以舌邊音l为代表。n/l合流現象，除了南番順地區，较少见于广东，某些粵語方言仍然清晰地區分这些声母。有香港播音員相反地將「藍色」讀成「男色」。

====例子====
男女（naam4 neoi5/nam4 nêu5）<ref>本頁面中之粵語拼音，前者爲香港粵語拼音，後者爲廣東教育廳發佈的《廣州話拼音方案》中的粵語拼音。</ref> -> 藍呂（laam4 leoi5/lam4 lêu5）

===ng鼻音脫落===
====因素====
* 语流音变
* 語音同化
* 方言口音（[[番禺|番禺]]口音、[[順德|順德]][[口音|口音]]、蜑家口音）

====現狀====
聲母ng的脫落在香港地區较为普遍。相反地，[[廣州|廣州]][[西關口音|西關口音]]{{Citation needed}}一般把零聲母读一律加上ng聲母。廣州有一道菜的名字非常經典，可以練習ng聲母：“芽菜炒牛肉”（ngaa4 coi3 chaau2 ngau4 yuk6）。

====例子====
我（ngo5） -> o5

===複合聲母w（u）<ref>本頁面中之粵語拼音，於括號外者爲香港粵語拼音，於括號內者爲廣東教育廳發佈的《廣州話拼音方案》中的粵語拼音。</ref>脫落（韻母o之前）===
====因素====
* 语流音变
* 語音同化
* 方言口音
* 其他漢語語言的影響（客家話）

====現狀====
{{When|近年}}來，[[廣東|廣東]]地區[[粵語|粵語]]使用者念以o為韻母的字時往往把gw（gu）發成g，把kw（ku）發成k。此現象於港澳亦常見。

====例子====
'''廣'''（gwong2/guong2）府 -> '''港'''（gong2）府<ref>[http://big5.citygf.com/cul/005017/201103/t20110307_1303359.html 多少先生名叫“吳廣德”-文化評論-文化名城-廣佛都市網] {{Wayback|url=http://big5.citygf.com/cul/005017/201103/t20110307_1303359.html |date=20140227081512 }}</ref><br>
水'''果'''（gwo2/guo2） -> 水'''嗰'''（go2）<br>
轮'''廓'''（kwok3/kuog3） -> 轮'''确'''（kog3）

由於客家話中並沒有gw-及kw-聲母，故此，部分人可能受客家話的影響。就以上例子，「廣」與「港」在客家話裏是同音（gong）；「果」與「嗰」在客家話裏都是同音（go）；「廓」與「確」在客家話裏也是同音（kok）<ref> www.hkilang.org 香港本土語言保育協會 發音字典</ref>。

==[[韻母|韻母]]==
===韻尾-ng與-n不分===
====因素====
* 音变（语流音变、同化、异化）
* 方言口音(部份鄉鎮番禺口音、蜑家口音)

====現狀====
大部分人能辨別並準確發出這兩個韻尾。{{Citation needed}}

====例子====
* 冷（laang5/lang5） -> 懶（laan5/lan5）
* 猛（maang5/mang5） -> 晚（maan5/man5）
* [[恒生銀行|恒生銀行]] -> 「痕身」{{IPA|ɐn}}「寒」

===韻尾–k（-g）與-t（-d）不分===
====起源====
和韻尾-ng與-n不分相若（因韻尾-ng與-n的對應入聲剛好分別為韻尾–k（-g）與-t（-d））。{{Citation needed}}

====現狀====
和韻尾-ng與-n不分相若。
====例子====
策（tsaak8/tsag3） -> 察（tsaat8/tsad3）<br>
責（zaak8/zag3） -> 札（zaat8/zad3）

===韻母ng變成m===
====因素====
* 语流音变
* 語音同化

====現狀====
粵語中发m音的汉字仅有『唔』一字，其他字如五、吳、誤等一并从ng音。现在有不少的人把ng韻母音和m音混同。

====例子====
'''吳'''（ng4） ->'''唔'''（m4）<ref>[http://big5.citygf.com/cul/005017/201103/t20110307_1303359.html 多少先生名叫“吳廣德”-文化評論-文化名城-廣佛都市網] {{Wayback|url=http://big5.citygf.com/cul/005017/201103/t20110307_1303359.html |date=20140227081512 }}</ref><br>
'''誤'''（ng6）會 -> m6會

==[[聲調|聲調]]==
===第一聲（[[陰平|陰平]]55/53）中[[高平|高平]]調55與[[高降|高降]]調53不分===
====現狀====
粵語第一聲（陰平55/53）分為[[高平|高平]]調55（與普通話第一聲[阴平55]相同）和[[高降|高降]]調53（與普通話第四聲[去聲51]相似）。一般來説，除了老年層和吟詩場合，現在人們讀[[陰平|陰平]]調時一律讀成[[高平|高平]]，所以[[聲調系統|聲調系統]]只剩下九調。<ref>中山大學中文系[[譚步雲|譚步雲]]博士認為廣州話有高平變調和兩個陰平調，香港話一般只有高平變調。</ref>

====例子====
好孫（高平）-> 好酸（高降）<br>
方（高平）糖 -> 荒（高降）唐

===第二聲（[[陰上|陰上]]35）與第五聲（[[陽上|陽上]]13）不分===
====起源====
受某些[[方言|方言]]的影響，如清遠話、中山石岐口音。{{Citation needed}}

====現狀====
總括來説，大部分人都能分清這兩個聲調，只有極少部分人區分不清。<br>{{Citation needed}}

====例子====
枉wong2 -> 往wong5<br>
想soeng2/sêng2 -> 上soeng5/sêng5<br>
鄙pei2/péi2 -> 婢pei5/péi5<br>
寫se2/sé2 -> 社se5/sé5

===第三聲、第八聲（[[陰去|陰去]]33、[[中入|中入]]3）與第六聲、第九聲（[[陽去|陽去]]22、[[陽入|陽入]]2）不分===
====例子====
=====陽去、陽入讀作陰去、中入=====
協hip9/hib6 -> hip8/hib3<br>
昨zok9/zog6 -> zok8/zog3
=====陰去、中入讀作陽去、陽入=====
乙jyut8/yud3 -> jyut9/yud6<br>
錫sek8/ség3 -> sek9/ség6

===[[入聲|入聲]]的[[變調|變調]]與不變調不分===
====起源====
[[粵語|粵語]][[入聲|入聲]]可以變調，有的更有辨義作用。但由於其中一些變調比較少出現，所以造成一些年輕人區分不清。
====現狀====
大部分人能勉強區分。
====例子====
去買嚿玉juk9/yug2<br>
去買嚿肉juk9/yug6<br>
肉玉本同音，是第九聲（陽入22），玉可以變調讀第二聲（陰上35），但肉不可以。<br><br>
半個月jyut9/yud6（十五天）<br>
半個月jyut2/yud2（半個月亮）

==其它==
===[[文白二讀|文白二讀]]區分不清===
====起源====
粤语与各现有汉语分支一样都存有文白二读现象，粤语中的文读白读之间发音差异明显较小，出现频率也不高，因此有部分人區分不清。
====現狀====
大部分人能區分清楚。
====例子====
=====韻母=====
最常见的形态是以白读-eng韵母取代文读-ing韵母，如病、命、钉、听、岭白读-eng，文读-ing。
=====聲母=====
文讀：'''近'''（gan6/gen6）代，白讀：远'''近'''（kan5/ken5）。
=====其他=====
'''逆'''白讀ngaak9/ngag6，文讀jik9/yig6。

===[[普通話|普通話]]的影響===
====起源====
甲、乙二字粵音相異，但國音相同，導致講者誤會兩字粵音亦相同，因此把甲之粵音張冠李戴，套用於乙字。<br>
相反現象（甲、乙二字國音相異，但粵音相同而導致誤讀）亦見於初學國語之母語為粵語之人士。<br>
[[中國大陸|中國大陸]][[推廣普通話|推廣普通話]]教育，某些學生受普通話影響嚴重。
====現狀====
[[廣東|廣東]]學生（特別是非粵語為母語者）的此种現象較嚴重。
====例子====
橫（waang4/wang4） -> 衡（hang4/heng4）<br>
榮（wing4） -> 容（jung4/yung4）<br>
函（haam4/ham4） ->含（ham4/hem4）<br>
熠（jap9/yep6） -> 藝（ngai6/ngei6）<br>
坡（bo1） -> po1 <br>
凝（jing4/ying4） -> 寧（ning4）<br>
恩（jan1/yen1） -> 奀（an1/en1）<br>
宏（wang4/weng4） ->  红（hung4/hung4）
