'''异地高考'''是2011年在[[中华人民共和国|中华人民共和国]]开始出台和逐步落实的教育政策和方案，政策出台的主要原因是[[户籍|户籍]]问题和教育资源分配不公，2012年，异地高考正式在全中国推行。

== 背景 ==
{{See also|高考移民}}

由于中国各省份地区之间发展不平衡，部分地区如[[北京市|北京市]]、[[上海市|上海市]]等教育资源得天独厚，北京[[清华大学|清华大学]]在北京市的招生数是苏、皖、鄂、川四省招生数的总和，而此四省是北京市人口的20倍；2011年，[[北京大学|北京大学]]录取率，按万分比比较：[[安徽省|安徽省]]为1.27，[[广东省|广东省]]为1.4，[[贵州省|贵州省]]为1.48，[[河南|河南]]为1.87，而北京高达52.5。<ref name=" souhu" >{{cite web |url= http://finance.ifeng.com/money/roll/20120913/7026324.shtml|title=面对“北京人大学”，北大的风度呢？ |accessdate=2012年9月13日 |publisher=凤凰山 |work= |language = zh-cn}}</ref>2012年，各地高考人数，安徽省50.6万，江苏省47.4万，[[湖北省|湖北省]]46.2万，河南省82.5万，北京市7.3万，而招生人数北京市619人，河南省108人，安徽省45人，江苏省60人。<ref name=" wyi" />

由于教育不公、户籍歧视和大学属地化，导致中国出现一个特殊的群体“[[高考移民|高考移民]]”，特点是“西进、南下、东闯”，因为中国西部、东部、南部高考录取占有绝对优势，移民地以河南省、[[湖南省|湖南省]]、湖北省、[[江西省|江西省]]、安徽省外移者最多。

== 沿革 ==
2011年3月，十一届全国人大四次会议上，[[中华人民共和国教育部部长|中华人民共和国教育部部长]][[袁贵仁|袁贵仁]]说，“教育部目前正在和上海、北京研究，逐步推进异地高考”。

2012年3月下旬，十一届全国政协五次会议上，袁贵仁指出，“异地高考改革方案将在10个月内出台”<ref>{{cite web |url = http://news.xinhuanet.com/politics/2012lh/2012-03/03/c_111599336.htm |title=袁贵仁：异地高考改革方案进入“最后冲刺阶段” |accessdate=2012年3月3日 |publisher=新华网 |work= |language =zh-cn}}</ref>。

2012年7月，[[中华人民共和国国务院|中华人民共和国国务院]]同意异地高考方案。

2012年8月31日，[[国务院办公厅|国务院办公厅]]发布文件，要求各地在2012年年底前出台异地高考具体办法<ref>{{cite web |url = http://edu.qq.com/a/20120901/000077.htm |title=国务院要求各地年底前出台异地高考具体办法 |accessdate=2012年9月1日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>。

== 争议 ==
异地高考政策出台后，教育资源分配成为焦点，在京、沪外籍和{{which|其他各省人}}大多对政策表示肯定和支持，但是此政策招来大部分北京人和上海人的抗议。<ref>{{cite web |url= http://news.sciencenet.cn/htmlnews/2012/10/270606.shtm|title=中国科学报：异地高考的“攻守”博弈 |accessdate=2012-10-18  |publisher=科学网 |work= |language =zh-cn}}</ref>

2012年9月，河南省新蔡人程帅帅曾赠送“北京人大学”牌匾给北京大学，抗议教育不公、户籍歧视和大学属地化，但遭到当局抓捕，一度关押在[[海淀区|海淀区]]派出所8小时。<ref name=" souhu" /><ref>{{cite web |url= http://news.sohu.com/20120912/n352899386.shtml|title=男子给北大送“北京人大学”校牌 被扣留8小时 |accessdate= 2012年9月12日|publisher=搜狐网 |work= |language = zh-cn}}</ref><ref name="wyi">{{cite web |url = http://news.163.com/12/0913/15/8B9TUUK700014AED.html |title = 小伙送北大“北京人大学”牌匾被遣送回乡|accessdate=2012-09-13  |publisher=网易 |work= |language =zh-cn}}</ref><ref>{{cite web |url = http://news.qq.com/a/20120912/000081.htm |title=河南网友送“北京人大学”匾给北大被警方扣留 |accessdate=2012年9月12日 |publisher=腾讯网 |work= |language = zh-cn}}</ref>

2012年10月18日，20多名北京人到[[北京市教育委员会|北京市教育委员会]]抗议异地高考政策，他们拿出材料，指出外籍人的缺点和毛病：外籍孩子素质极差，打架斗殴带坏了本地小孩；外籍人占用了过多的本地教育资源；外籍人严重影响了本地居民的正当权益等等<ref>{{cite web |url = http://news.qq.com/a/20121019/001392.htm |title= 京籍人士到教委抗议放开异地高考 称权益受损大|accessdate=2012年10月19日 |publisher=腾讯网 |work= |language =zh-cn}}</ref><ref>{{cite web |url = http://news.ifeng.com/gundong/detail_2012_10/20/18400165_0.shtml |title=京籍家长反对开放异地高考，有没有道理？ |accessdate=2012年10月20日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>。

2014年7月30日，在《关于进一步推进户籍制度改革的意见》发布会中，教育部副部长[[刘利民|刘利民]]表示教育部将会同相关部门，从规划、经费、分类指导、加强管理四个方面积极推动随迁子女在流入地接受教育并且参加升学考试。

据其介绍，到2014年7月30日为止，有27个省份明确了随迁子女在当地参加中考的政策，有30个省份明确了随迁子女在当地参加高考的政策。2014年有28个省市开始解决随迁子女在当地参加高考的问题，就是大家所说的“异地高考”问题。2014年有5.6万名符合条件的随迁子女办理了高考的报名手续。<ref>[http://www.ce.cn/xwzx/gnsz/gdxw/201407/30/t20140730_3258249.shtml “异地高考”不再难 教育部四大措施保障随迁子女教育权利] {{Wayback|url=http://www.ce.cn/xwzx/gnsz/gdxw/201407/30/t20140730_3258249.shtml |date=20140808042726 }}，中国经济网，2014年7月30日</ref>
