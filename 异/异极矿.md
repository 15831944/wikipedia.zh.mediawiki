'''异极矿'''（Hemimorphite，化学式：Zn<sub>4</sub>(Si<sub>2</sub>O<sub>7</sub>)(OH)<sub>2</sub>·H<sub>2</sub>O），是[[炉甘石洗剂|炉甘石洗剂]]中的一种成分。作为一种[[硅酸盐矿物|硅酸盐矿物]]，异极矿在历史上开采自锌矿和铅矿的上半部，与菱锌矿（smithsonite）（ZnCO<sub>3</sub>）相关联。异极矿和菱锌矿曾被认为是同一种矿石，被归为[[炉甘石|炉甘石]]一类；直到18世纪下半叶才发现出现在炉甘石中的这两种矿石归属不同种类。

异极矿是这两者中较为稀有的一种，得名于其晶体的异极象（hemimorph）生长，即晶体终止于不一样的表面（dissimilar faces）。异极矿常常形成晶状外壳层，以及大块颗粒状的、球形或肾形的聚合物（aggregates），具有从中心放射的条纹；或是针状的、纤维状的、钟乳石状的，极少数是风扇状的晶簇。

一些种类的异极矿在短[[紫外|紫外]]波段（253.7 nm）表现出强烈的绿[[荧光|荧光]]，在长紫外波段则表现出较弱的粉红荧光。

== 产地 ==
[[File:Hemimorphite-227534.jpg|左]][[杜兰戈州|杜兰戈州]]的异极矿（尺寸：2.9 x 2.1 x 2.0 cm）]]
异极矿多见于[[闪锌矿|闪锌矿]]上半部分的[[氧化|氧化]]，以及来自于俗称“{{le|铁帽|gossan}}”的二级矿物。异极矿是一种重要的锌矿，其中含多达54.2%的金属，以及硅、氧和氢元素。
[[File:Hemimorphite-278499.jpg|左]][[文山市|文山市]]的蓝色异极矿（尺寸：9.2 x 4.8 x 3.1 cm）]]

位于[[比利时|比利时]]和[[德国|德国]]边境交界处的换质（metasomatic）矿床是比较有名的异极矿，其中包括位于比利时的{{le|维埃耶山|Vieille Montagne}}和德国的[[亚琛|亚琛]]的矿床。其他的异极矿产地还有[[波兰|波兰]]（[[上西里西亞|上西里西亞]]的[[塔爾諾夫斯凱古雷|塔爾諾夫斯凱古雷]]区域）；[[美国|美国]]（[[宾夕法尼亚州|宾夕法尼亚州]]的[[菲尼克斯維爾|菲尼克斯維爾]]附近、[[密蘇里州|密蘇里州]]的铅-锌区（lead-zinc district）、[[蒙大拿州|蒙大拿州]]的{{le|埃尔克霍恩|Elkhorn, Montana}}、[[科羅拉多州|科羅拉多州]]的[[莱德维尔_(科罗拉多州)|莱德维尔]]和[[新墨西哥州|新墨西哥州]]的{{le|欧根山脉|Organ Mountains (New Mexico)}}）；以及[[北非|北非]]。[[泰国|泰国]]西部的Padaeng矿床、[[撒丁岛|撒丁岛]]、[[西伯利亚|西伯利亚]]的[[尼布楚|尼布楚]]、[[意大利|意大利]]的{{le|卡韦－德尔普雷迪尔|Cave del Predil}}、[[奥地利|奥地利]][[克恩顿州|克恩顿州]]的[[普隆比埃|普隆比埃]]和[[英格兰|英格兰]][[德比郡|德比郡]]的[[馬特洛克|馬特洛克]]也是异极矿的产地。
