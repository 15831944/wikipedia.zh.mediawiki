'''器官'''是[[动物|动物]]体或[[植物|植物]]体的由不同的[[细胞|细胞]]和[[组织_(生物学)|组织]]构成的结构，用来完成某些特定功能，并与其他分担共同功能的器官，一起组成各个[[系统|系统]]（动物体）或整个[[个体|个体]]（植物体）。<ref>{{cite web |url=http://www.merriam-webster.com/dictionary/Viscus |title=Viscus - Definition |format= |work=Merriam-Webster Online Dictionary |accessdate=14 December 2009}}</ref><ref>{{cite web |url=http://www.ncbi.nlm.nih.gov/mesh?term=viscera |title=Viscera |format= |work=MeSH |accessdate=14 December 2009}}</ref>

== 植物体的器官 ==
植物体的器官分为[[生殖|生殖]]器官和[[营养|营养]]器官两类：[[营养器官|营养器官]]包括[[根|根]]、[[茎|茎]]和[[叶|叶]]；[[生殖器官|生殖器官]]包括[[花|花]]、[[果实|果实]]和[[种子|种子]]。
前者之主要功用為吸收、運輸養分為主，以利生長，稱為營養器官。後者以透過有性生殖，並以傳播後代為主要功能，稱為「生殖器官」。

== 动物体的器官 ==
动物体的器官类型较为复杂，一个器官可以属于多个系统。动物体的主要器官有：[[心臟|心臟]]、[[肺|肺]]、[[腦|腦]]、[[眼|眼]]、[[脾臟|脾臟]]、[[胰臟|胰臟]]、[[腎臟|腎臟]]、[[肝臟|肝臟]]、[[腸臟|腸臟]]、[[皮膚|皮膚]]、[[子宮|子宮]]、[[膀胱|膀胱]]和[[骨骼|骨骼]]等。

=== 維生器官 ===
維生器官是[[人體|人體]]內維持[[生命|生命]]的器官。如果身體內的維生器官不能完全運行正常的話，一個人可能很快[[死亡|死亡]]。主要的維生器官有：

* 腦部，負責控制和協調[[呼吸|呼吸]]、[[心跳|心跳]]、[[激素|荷爾蒙]]生產、[[感覺|感覺]]接收、[[肌肉|肌肉]]運動等
* 心臟，將含有充分[[氧氣|氧氣]]及養分的[[血液|血液]]送至全身，供應各組織器官
* 肺部，負責呼吸及使血液帶氧
* 腎臟，將[[血液|血液]]內的[[廢物|廢物]]移除帶到膀胱之內

其他負責[[消化|消化]]和[[排泄|排泄]]的器官，則對於長期維生有必要性。但有很多人雖然缺少腎臟、脾臟和腸臟等器官的情況下依然生存，不過當然需要機器的幫助。
