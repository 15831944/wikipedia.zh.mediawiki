'''石狩橋本站'''（{{jpn|j=石狩橋本駅|hg=いしかりはしもとえき|rm=Ishikari-Hashimoto eki|f=y}}）是位於[[北海道|北海道]][[樺戶郡|樺戶郡]][[新十津川町|新十津川町]]上德富，隸屬於[[日本國有鐵道|日本國有鐵道]]（國鐵）[[札沼線|札沼線]]，已廢除的[[鐵路車站|鐵路車站]]。

原本車站名稱預定是其所在地「新十津川」，但由於附近村落反對因此將車站名稱改為附近的町名「橋本」，再加上舊國名「石狩」。而「新十津川」則被鄰近的車站使用。

== 歷史 ==
* 1931年（昭和6年）10月10日 - [[札沼線|札沼北線]][[石狩沼田車站|石狩沼田]]至中德富（初代，現在的[[新十津川車站|新十津川]]）開始營運。
* 1943年（昭和18年）10月3日 - 由於[[太平洋戰爭|太平洋戰爭]]擴大，本車站作為[[不要不急線|不要不急線]]而停止營運。
* 1953年（昭和28年）11月3日 - 札沼線[[浦臼車站|浦臼]]至[[雨龍車站|雨龍]]路段恢復營運。
* 1972年（昭和47年）6月19日 - 札沼線新十津川至石狩沼田路段廢除，本車時同時廢站。

== 車站構造 ==
廢站時為有職員駐守的地面車站。

站房位於札幌方向的左方東側，為[[側式月台|側式月台]]1面1線的車站。而站房旁往札幌方向設有貨物月台，並設有1條引入線。

== 車站周邊 ==
只餘下營運時已存在的農業倉庫，而車站的土地已變成住宅地。
