'''石刚'''（{{bd|1958年|9月|catIdx=S石}}），[[安徽|安徽]][[桐城|桐城]]人；[[中国科技大学|中国科技大学]]毕业，[[上海交通大学|上海交通大学]]管理学院工学硕士。1986年开始，在[[中华人民共和国国务院|国务院]]多个宏观经济管理部门，从事经济预测与分析工作。现任[[国务院研究室|国务院研究室]]副主任、兼[[李克强|李克强]]总理办公室主任。<ref>{{cite web |url=http://renshi.people.com.cn/n/2013/0702/c139617-22044135.html|title=国务院研究室副主任石刚兼任总理办公室主任|accessdate=2013年7月2日|publisher=人民网－中国共产党新闻网|work= |language=zh-cn}}</ref>

==经历==
1982年，石刚从[[中国科技大学|中国科技大学]]毕业，被分配到合肥市无线电三厂工作。1983年9月，考入[[上海交通大学|上海交通大学]]管理学院；1985年底，获工学硕士学位，并进入[[国家信息中心|国家信息中心]]经济预测部工作，成为技术官僚。

1997年，调入[[国家计委|原国家计委]]国民经济综合司工作，从事宏观经济政策研究、经济形势分析和经济预测、组织编制国民经济和社会发展年度计划等；2006年6月起，担任[[国家发展和改革委员会|国家发展和改革委员会]]综合司司长。<ref>{{cite web |url=http://www.guancha.cn/politics/2013_07_02_155269.shtml|title=国务院研究室副主任石刚兼任总理办公室主任（简历）|accessdate=2013年7月2日|publisher=观察者|work= |language=zh-cn}}</ref>在李克强担任[[国务院副总理|国务院副总理]]期间，曾牵头负责制定“[[十二五规划|十二五规划]]”；而石刚，作为发改委负责编制经济计划的综合司司长，正是参与这项工作的核心人物之一，两人又系桐城老乡。2013年，李克强出任[[国务院总理|国务院总理]]后，石刚先是于当年6月被提拔为[[国务院研究室|国务院研究室]]副主任<ref>{{cite web |url=http://politics.people.com.cn/n/2013/0620/c1001-21908834.html|title=国务院任免一批工作人员|accessdate=2016年4月5日|publisher=人民网|work= |language=zh-cn}}</ref>，后又兼任李克强总理办公室主任<ref>{{cite web |url=http://news.hexun.com/2013-07-03/155724498.html|title=石刚出任李克强大秘 此前在发改委研究经济政策|accessdate=2016年4月5日|publisher=原载《21世纪经济报道》|work= |language=zh-cn}}</ref>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。
