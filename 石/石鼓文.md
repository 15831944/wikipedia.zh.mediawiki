'''石鼓文'''处于承前启后的时期，承[[秦国|秦国]]书风，为[[小篆|小篆]]先声。石鼓文刻于十座[[花岗岩|花岗岩]]石上，因石墩形似鼓，故称为“[[石鼓_(先秦)|石鼓]]”。石鼓文与[[金文|金文]]有较大差别，具有明显的动感，是中国现存最早的刻石文字。

现存的石鼓文来自于[[唐朝|唐朝]]收集的十石鼓<ref name="gugong">故宫博物院珍宝馆馆藏说明</ref>，上面刻有文字。现存最早的拓片是[[宋朝|宋朝]]流传下来的<ref name="gugong"/>，当时认为是描述[[周穆王|周穆王]]出猎的场面，后来的考古考证认为是[[秦穆公|秦穆公]]时代的作品，有的字已经残缺不全。当时由于尚没有发现[[甲骨文|甲骨文]]，所以被认为是[[中国|中国]]最古老的文字。[[康熙|康熙]]曾经复制了十面石鼓，补刻文字，放在[[承德|承德]][[避暑山庄|避暑山庄]]的宫殿后面。

== 溯源 ==
石鼓文亦称“猎碣”或“雍邑刻石”，[[唐代|唐代]]初出土于天兴三畴原（今陕西省宝鸡市凤翔三畴原），唐代诗人[[韩愈|韩愈]]曾作《石鼓歌》，其中有“周纲凌迟四海沸，宣王愤起挥天戈”的诗句。可见，在唐代普遍认为石鼓文出于周代。至宋代，[[欧阳修|欧阳修]]认为石鼓文为[[周宣王|周宣王]]时期[[史籀|史籀]]所作。近代[[罗振玉|罗振玉]]《石鼓文考释》和[[马叙伦|马叙伦]]《石鼓文疏记》将石鼓文的历史缩短到了[[战国|战国]][[秦|秦]]，认为是[[秦文公|秦文公]]时期出现的。[[郭沫若|郭沫若]]又考证石鼓文的制作年代为[[秦襄公|秦襄公]]八年。

== 流转 ==
石鼓文历史上曾辗转流离。唐末至[[五代|五代]]，战乱频繁，十面石鼓散于民间，到宋代时又集齐，为[[宋徽宗|宋徽宗]]于大观二年（1108年）迁至汴京国学。后又为[[金朝|金]]人掳走。元代时再次记起收于北京文庙。至近代[[抗日战争|抗日战争]]时期，[[故宫博物院|故宫博物院]]院长[[马衡|马衡]]等将石鼓迁到江南躲避战乱，抗战胜利后又运回北京。

另外[[清|清]][[乾隆|乾隆]]五十五年（1790年），乾隆帝曾令人仿刻十鼓，置于太学。

== 书体特征 ==
石鼓文上承西周金文，下启秦代小篆，与先秦时期的大篆又有所不同，可以说是大篆与小篆之间的一种书体。其笔法方正、均衡，布局紧凑，笔法圆阔，极为周致。为历代习篆书家所喜爱。

== 石鼓文的传承 ==
石鼓文对后世的书法与绘画艺术有着非常重大的影响，不少杰出的书画家如：杨沂孙、吴大澄、吴昌硕、朱宣咸、熊国英、曹宇、王福庵等都长期研究石鼓文艺术，并将其作为自己书法艺术的重要养分，进而融入进自己的《石鼓文》书法，朱宣咸(作)绘画艺术之中。
