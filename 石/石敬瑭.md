'''晉高祖石敬瑭'''（{{bd|892年|3月30日|942年|7月28日|}}<ref name=HFD75/><ref name=ZZTJ283>《[[资治通鉴|资治通鉴]]》[[:s:資治通鑑/卷283|卷二百八十三]]</ref>）是[[五代十国|五代十国]]時期的[[后晋|后晋]]开国皇帝（936年11月28日–942年7月28日在位）。[[庙号|庙号]]'''高祖'''，[[谥号|谥号]]'''圣文章武明德孝皇帝'''。他把[[燕雲十六州|燕雲十六州]]割讓给契丹，使[[中原|中原]]地區丧失了北方屏障，並向[[辽太宗|辽太宗]]自称'''儿皇帝'''。

== 早期 ==
《[[新五代史|新五代史]]》指石敬瑭的祖先为[[中亚|中亚]]人，从[[沙陀|沙陀]]移居[[太原市|太原]]，但發挖出土的石重貴墓誌銘則指他是[[后赵|後趙]][[石勒|石勒]]之後裔。

父[[石紹雍|石紹雍]]，母[[孝元懿皇后|何氏]]。石紹雍从[[李克用|李克用]]父子征战，官至洺州[[刺史|刺史]]。

石敬瑭自少为[[唐明宗|李嗣源]]（日後的唐明宗）赏识，为其亲兵将领，被招为女婿。[[唐庄宗|後唐莊宗]][[同光|同光]]四年（926年），[[邺都之变|邺都之变]]，石敬瑭力劝李嗣源入[[汴京|汴京]]，转攻[[洛阳市|洛阳]]。李嗣源即位后，石敬瑭历任[[保义镇<!--?-->|保义]]、[[宣武区|宣武]]、[[河东郡_(中国)|河東]]诸镇[[节度使|节度使]]。

934年，閔帝[[唐閔帝|李從厚]]徙石敬瑭為[[成德節度使|成德節度使]]。閔帝討伐潞王[[李从珂|李从珂]]失敗，逃到[[衛州|衛州]]向石敬瑭求援，可是石敬瑭的部下把閔帝隨從殺盡，石敬瑭把閔帝安置在[[衛州|衛州]]，最後閔帝被李从珂派人殺死。

== 建國 ==
末帝李从珂继位后，任石敬瑭為河東節度使，後來開始對石敬瑭起疑，石敬瑭也暗中謀自保。石敬瑭以多病為理由，上表請求朝廷調他往其它[[藩鎮|藩鎮]]，借此試探朝廷對他的態度。末帝在[[清泰|清泰]]三年（936年）五月改授石敬瑭為[[天平節度使|天平節度使]]，並降旨催促赴任。石敬瑭懷疑末帝對他起疑心，便举兵叛变。後唐派兵討伐，石敬瑭被圍，向[[契丹|契丹]]求援，九月契丹軍南下，擊敗唐軍。石敬瑭在十一月受契丹冊封為大晉皇帝，認契丹可汗为父，自稱兒皇帝，然後向洛陽進軍，末帝在閏十一月（937年1月）自焚，後唐遂亡。

石敬瑭滅後唐後，按约定将[[燕雲十六州|燕雲十六州]]献给契丹，使[[中原|中原]]地區丧失了北方屏障，另外每岁奉绢三十万匹，並向比他小10歲的辽太宗[[耶律德光|耶律德光]]稱自己為“兒皇帝”，耶律德光为“父皇帝”。

== 後期 ==
石敬瑭在位期間，各地將領魏博节度使[[范延光|范延光]]、西京留守[[张从宾|张从宾]]、[[成德节度使|成德节度使]][[安重荣|安重荣]]、[[山南东道节度使|山南东道节度使]][[安从进|安从进]]等引发的叛變事件不斷，他的兒子[[石重信|石重信]]和[[石重乂|石重乂]]亦遭叛軍殺害。后因成德節度使安重榮及河东节度使[[劉知遠|劉知遠]]先後接受[[吐谷浑|吐谷浑]]部族投降，石敬瑭屡遭契丹责问，乃忧愤而死。

== 後世評價 ==
《[[旧五代史|舊五代史]]》稱讚石敬瑭的謙虛、節儉；「旰食宵衣，禮賢從諫」、「以絁為衣，以麻為履」，後又責怪他向契丹乞兵，反而使得百姓陷入連年戰火；「強鄰來援，契丹自茲而孔熾，黔黎由是以罹殃。」「兵連禍結、舉族為俘」，這無疑是決鯨海以救焚，結果自己溺死了、飲鴆漿而止渴，結果毒死自己。《舊五代史》最後為他惋惜，如果他是靠自己的力量取得帝位，以他的節儉、謙卑、公正的態度，即使功德不超過前人，亦可謂仁慈恭儉之主。
