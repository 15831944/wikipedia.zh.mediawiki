'''石碇服務區'''位於[[新北市|新北市]][[石碇區|石碇區]]，是[[蔣渭水高速公路|蔣渭水高速公路]]自[[南港系統交流道|南港系統交流道]]（起點）以來的第一個服務區，里程為4公里，必須透過[[石碇交流道|石碇交流道]]接[[文山路|文山路]]往[[深坑|深坑]]方向行駛，才能進入。

== 簡介 ==
此服務區是蔣渭水高速公路沿線第一個營運的服務區，也是台灣第一個沒有設專用匝道與國道銜接、需從交流道離開國道才能前往的服務區，經營商為[[全家便利商店股份有限公司|全家便利商店股份有限公司]]，服務區內有全家便利商店石碇國道店、熟食區<ref>[http://www.freeway.gov.tw/NorthService/Shiding 石碇服務區]</ref>。服務區旁有打造歐式和日式風格的景觀公園。

== 加油站 ==
石碇服務區本身並未附設加油站，但可透過[[市道106乙線|市道106乙線]]前往下列鄰近的加油站：
*[[台灣中油|台灣中油]]石碇站：文山路一段8號
*台灣中油建德站：文山路一段9號

== 對外交通 ==
由於本服務區坐落在一般道路上，因此除了從石碇交流道進出以外，非使用高速公路的民眾亦可經由市道106乙線（文山路一段）抵達石碇服務區。而服務區也劃有機踏車停車格供[[自行車|自行車]]與[[摩托車|摩托車]]停放。

=== 大眾運輸 ===
*[[新北市區公車923路線|923]]、[[葛瑪蘭客運|葛瑪蘭客運]]1915至「楓子林路口」站下車後再步行5分鐘。
*[[台北聯營公車660路線|660]]、[[台北聯營公車660路線|660（區）]]、[[新北市區公車795路線|795]]、[[新北市區公車666路線|666]]、[[台北聯營公車912路線|912]]、[[台北聯營公車949路線|949]]至「八分寮」站下車後再走楓子林路接文山路一段，約10分鐘即可到達。

== 周邊規劃 ==
[[新北市政府捷運工程局|新北市政府捷運工程局]]所規劃的[[深坑輕軌|深坑輕軌]]其終點站與停放列車的石碇機廠就設立於附近，未來將成為[[大臺北地區|大臺北地區]]民眾搭乘[[捷運|捷運]]轉乘[[客運|客運]]至[[宜蘭縣|宜蘭縣]]的轉運站。
