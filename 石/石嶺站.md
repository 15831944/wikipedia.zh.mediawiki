'''石嶺站'''（{{jpn|j=石嶺駅|hg=いしみねえき|rm=Ishimine eki|f=y}}）是位於[[日本|日本]][[沖繩縣|沖繩縣]][[那覇市|那覇市]]的一個興建中的[[鐵路車站|鐵路車站]]，為日後[[沖繩都市單軌鐵路|沖繩都市單軌鐵路]]伸延時所設置的新站，並且是伸延段內唯一的車站名，是由申請到最後定案上均沒有更改過<ref>{{Cite web|url=http://www.mlit.go.jp/common/000189395.pdf|format=PDF|title=沖縄都市モノレール株式会社に係る軌道運輸事業の特許について（別紙）|publisher=[[国土交通省|国土交通省]]|date=2012-01-25|accessdate=2012-12-11}}</ref>。

本站是伸延區間內唯一座落於那覇市的車站，那覇市政府曾經希望讓首里～石嶺一段首先提早開業<ref>{{Cite news|url=http://ryukyushimpo.jp/news/storyid-208911-storytopic-3.html|title=石嶺駅、先行開業も　沖縄都市モノレール|publisher=[[琉球新報|琉球新報]]|date=2013-07-03|accessdate=2013-10-14}}</ref><ref>{{Cite news|url=http://article.okinawatimes.co.jp/article/2013-10-10_55162|title=モノレール延長 先行開業、16年にも石嶺駅|publisher=沖縄タイムス|date=2013-10-10|accessdate=2013-10-14|deadurl=yes|archiveurl=https://web.archive.org/web/20131012055832/http://article.okinawatimes.co.jp/article/2013-10-10_55162|archivedate=2013-10-12}}</ref>，可是因收地問題，最終未能如願，以致最終要與其他新車站一併於2019年通車。

== 車站結構 ==
預定採用[[島式月台|島式月台]]設計，共設有1面2線的月台配置，與現時[[儀保站|儀保站]]的設計相同<ref>[http://www.pref.okinawa.jp/site/doboku/toshimono/enchou.html 延長事業について] - 沖縄県、2014年5月2日，2015年7月5日閲覧</ref>。

== 歷史 ==
* 2012年1月25日：[[國土交通省|國土交通省]]批准[[沖繩都市單軌鐵路|沖繩都市單軌鐵路]]經營[[首里站|首里]]～[[日子浦西站|浦西]]共4.1公里的軌道事業特許認可<ref>{{Cite web |date=2012-01-25 |url=http://news.mynavi.jp/news/2012/01/26/005/index.html |title=ゆいレール首里駅から延伸へ - 2019年春開業予定、バスとの乗り継ぎも考慮 |publisher=マイナビニュース|accessdate=2013-11-12}}</ref>。
* 2013年11月2日：進行車站興建起工式<ref name="times131103">{{Cite web |date=2013-11-03 |url=http://article.okinawatimes.co.jp/article/2013-11-03_56124 |title=モノレール延長着工　首里から4駅新設 |publisher=沖縄タイムス |accessdate=2013-11-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20131111212633/http://article.okinawatimes.co.jp/article/2013-11-03_56124 |archivedate=2013-11-11 }}</ref>。
* 2014年12月26日：站名決定繼續採用「'''石嶺站'''」<ref name="ryukyu1">[http://ryukyushimpo.jp/news/storyid-236496-storytopic-82.html てだこ浦西、浦添前田など　ゆいレール浦添延伸、４駅名決まる]，琉球新報，2014年12月26日。{{ja}}</ref>。
* 2019年：預計開業<ref name="times131103"/>。
