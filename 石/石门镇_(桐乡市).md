'''石门镇'''，原石门乡和羔羊乡合并而成，是中国[[浙江省|浙江省]][[嘉兴市|嘉兴市]][[桐乡市|桐乡市]]下辖的一个[[镇|镇]]。

== 沿革 ==
石门镇是由原石门乡和羔羊乡合并而成。石门乡以环绕石门镇而得名，并无建制。1950年4月，石湾镇、石门镇合并。1958年，设立石门公社，改隶属于桐乡，管辖立新、叶家浜、石门、民联、长丰、洪泾桥、天星、利星、新星和颜井桥村。1961年，合建为石门公社。1983年，改为石门乡。1987年2月，撤石门乡并石门镇<ref>{{Cite book|title=浙江区划地名溯源|last=《浙江区划地名溯源》编委会编|first=|publisher=杭州：浙江大学出版社|year=2007年5月|isbn=7-308-05332-6|location=|pages=328}}</ref>。2001年10月，桐乡市乡镇行政调整，撤销[[羔羊乡|羔羊乡]]并入石门镇<ref>{{Cite book|title=中国共产党浙江省嘉兴市组织史资料 第4卷|last=中共嘉兴市委组织部编|first=|publisher=杭州：浙江人民出版社|year=2009年12月|isbn=978-7-213-04176-1|location=|pages=300-301}}</ref>。

==辖区==
该镇辖有<ref>{{Cite book|title=浙江建置区划沿革|last=俞志壮主编|first=|publisher=杭州：浙江大学出版社|year=2009年4月|isbn=978-7-308-06705-8|location=|pages=387}}</ref>：玉溪社区、叶新村、同星村、颜井桥村、白马塘村、石门村、墅丰村、民丰村、民联村、殷家漾村、陆家庄村、春丽桥村、周墅塘村、郜墩村、东池村、安全村、羔羊村、二大埭村、崇安村。

== 产业 ==
石门镇地处京杭大运河两侧，境内地势东南高，西北低。田地交叉，高低悬殊，形成桑基圩田。镇区河港密布，特具水路交通优势。现在内河航班直达乌镇、平湖、嘉兴、苏州、杭州、上海等地。1971年，运河市区段疏拓河面宽50米，通航能力100吨级以上，成为重要的水运码头<ref name=a />。

石门有已经规模化的酿酒业与榨油业，并形成丝绸、纺织、化纤、皮塑，机械，五金、家电、建材、医药、食品等工业门类。1988年列入浙江省对外开放重点工业卫星镇。农业以水稻为主，经济作物有[[蚕桑|蚕桑]]、[[杭白菊|杭白菊]]、烟叶、油菜籽、蔬菜等。镇上设有烟叶加工厂和药材加工场，是桐乡晒红烟、[[杭白菊|杭白菊]]的主要加工基地<ref name=a>{{Cite book|title=中国城镇 浙江卷|last=李家鸿主编；《中国城镇·浙江卷》编辑委员会编|first=|publisher=北京：中国城市出版社|year=1991年12月|isbn=7-5074-0607-5|location=|pages=710-711}}</ref>。当地民众将杭白菊开发为旅行景点<ref>{{Cite book|title=浙江 第4版|last=《走遍中国》编辑部编著|first=|publisher=北京：中国旅游出版社|year=2014年1月|isbn=978-7-5032-4890-0|location=|pages=218}}</ref>。

==文化 ==
石门也是著名漫画家[[丰子恺|丰子恺]]的故乡。1938年其画房“缘缘堂”毁于侵华日军炮火。1984年，桐乡县人民政府重建“缘缘堂”，并被列为县级重点文物保护单位<ref>{{Cite book|title=中国古城古镇古村旅游发展研究|last=魏小安著|first=|publisher=北京：中国旅游出版社|year=2009年4月|isbn=978-7-5032-3771-3|location=|pages=310-313}}</ref>。

石门镇在宋朝庆元间有[[黄勉斋|黄勉斋]]创办的崇义书院，元朝末年四明山迁居该镇的[[周致尧|周致尧]]创办宣公书院；清朝有开文书院。民国时期办有[[振华女校|振华女校]]。中华人民共和国成立后，建立石门完全中学<ref>{{Cite book|title=浙江省名镇志|last=浙江省名镇志编纂委员会编|first=|publisher=上海：上海书店出版社|year=1991年5月|isbn=7-80569-398-6|location=|pages=213-216}}</ref>。石门镇的文物有吴疆越界的古垒石处、张氏东西园、[[清乾隆南巡大营|清乾隆南巡大营]]等<ref name=a />。在其颜井桥村罗家角挖掘出的[[罗家角遗址|罗家角遗址]]，属[[马家浜文化|马家浜文化]]，2001年被列为[[第五批全国重点文物保护单位|第五批全国重点文物保护单位]]<ref>{{Cite book|title=江南胜迹|last=萧梦龙主编|first=|publisher=南京：江苏科学技术出版社|year=1993年12月|isbn=7-5345-1709-5|location=|pages=352}}</ref>。
