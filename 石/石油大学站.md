== 车站概要 ==
本站位于新都大道与育英路交叉口，沿新都大道呈东西向布置，是[[成都地铁3号线|成都地铁3号线]]的一座车站，于2018年12月26日开通<ref>{{Cite web|url=http://scnews.newssc.org/system/20181220/000931114.html|title=从北走到南！成都地铁3号线二三期车站正式亮相 -四川新闻-四川新闻网|accessdate=2019-01-20|work=四川新闻网}}</ref>。

== 车站结构 ==
