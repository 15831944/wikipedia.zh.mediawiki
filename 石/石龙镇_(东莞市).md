'''石龍鎮'''位于[[中国|中国]][[广东省|广东省]][[東莞市|東莞市]]北部，是东莞市经济较为发达、人口密度较大的[[镇|镇]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|44/19/00/109/000}}}}</ref>。东北为[[惠州市|惠州市]][[博羅縣|博羅縣]][[石湾镇|石湾镇]]，西北为[[广州市|广州市]][[增城区|增城区]][[石滩镇|石滩镇]]，南面是東莞[[茶山鎮|茶山鎮]]，西面是[[石碣鎮|石碣鎮]]，在[[清朝|清朝]]，石龙就以活跃的商业贸易而驰名省港，曾与廣州、[[佛山|佛山]]、[[顺德|顺德]][[陈村镇_(佛山市)|陳村]]（简称"省佛陈龙"）并称“廣東四大名鎮”，是东莞地区最早设镇的地方。而石龍人[[陳鏡開|陳鏡開]]在1956年打破[[舉重|舉重]]輕量級[[挺舉|挺舉]]項目，而以後本鎮又多度出現舉重選手，故有「舉重之鄉」稱號。

== 交通 ==
[[廣深鐵路|廣深鐵路]]纵贯石龍鎮，並設有[[石龍站|石龍站]]，是東莞繼東莞火車站之後於東莞另一鐵路大站，每天有多班[[列車|列車]]行經。火車行車時間深圳到石龍40分鐘，廣州到石龍22分鐘，來往廣深非常方便。

2014年1月8日開始[[石龍站|石龍站]]停止使用，由新[[東莞站|東莞站]]取代，而原位於常平的旧「[[東莞站|東莞站]]」站名則改回「[[常平站|常平站]]」，「常平站」的长途列车服务亦同时迁往石龙镇新東莞站。原有[[石龍站|石龍站]]在搬遷後改建為鐵路博物館及貨運場及客技編組站之用。

石龙镇新[[東莞站|東莞站]]將為[[东莞轨道交通2号线|東莞軌道交通R2線]]的其中一個起點，{{fact|time=2016-07-24|预计2016年通车使用，並可辦理廣九城際列車服務}}。

== 经济 ==
[[File:石龙火柴_2.jpg|thumb]]
石龙镇的经济主要以引进外资特别是外资为主。目前石龙镇的主要外资企业包括日本[[京瓷|京瓷光学]]及京瓷美达、[[柯尼卡美能达|柯尼卡美能达]]、[[太阳光学|太阳光学]]、[[津威饮料|津威饮料]]、[[三协精机|三协精机]]等。

== 电视广播 ==
长年以来，石龙镇当局会每天晚上10:45在[[翡翠台|翡翠台]]插播由石龙镇文化广电服务中心制作的《石龙新闻》，时间通常为10至15分钟，令民众无法完整收看正常的节目。广东省广播电影电视局声称，因石龙镇没有自己的电视频道，根据“有关协议”（无法得知协议名称及内容），故需要在翡翠台插播，如果个别时期镇的“宣传任务”比较重，《石龙新闻》的播出时间可能更为偏长。广电当局亦没有解释为何《石龙新闻》一定要在翡翠台，而不在广东省或东莞市电视频道，如[[广东新闻频道|广东新闻频道]]或[[东莞电视台|东莞电视台]]中插播。<ref>{{cite web|title=石龙新闻播放时间 |url=http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=17443 |publisher=广东广播影视网 - 咨询投诉 |date=2012-02-04 |accessdate=2012-03-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20140222205739/http://www.rftgd.gov.cn/wsbs/note_view.dy?mid=17443 |archivedate=2014-02-22 }}</ref>
