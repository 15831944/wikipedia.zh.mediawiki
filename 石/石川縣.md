'''石川縣'''（{{jpn|j=石川県|hg=いしかわけん|hgt=いしかはけん|rm=Ishikawa ken|f=y}}）是[[日本|日本]]的一個[[日本行政區劃|縣]]，為[[北陸地方|北陸]]三縣之一，縣廳所在地是[[金澤市|金澤市]]。轄區相當於過去令制国中的[[加贺国|加贺国]]与[[能登国|能登国]]。石川縣東西寬約100公里，南北長約200公里，形狀狹長。西南部的加賀地方的西側日本海海岸線呈直線形狀，東側是[[兩白山地|兩白山地]]諸峰，東南部有縣內最高峰白山，高2,702米。北部的能登地方則大致相當於[[能登半島|能登半島]]。石川縣的氣候屬於[[日本海側氣候|日本海側氣候]]。自[[西伯利亞|西伯利亞]]吹來的乾冷季風在日本海上空受對馬暖流的影響使得水汽含量增加，被兩白山地擋住後給石川縣帶來充沛降水<ref name="weather">{{cite web |url=http://www.jma-net.go.jp/kanazawa/mame/tokusei/tokusei.html |script-title=ja:石川県の気象特性 |accessdate=2017-05-11 |publisher={{lang|ja|金沢地方気象台}} |language=ja}}</ref>。石川縣的全年降水量在所有都道府縣中高居第五<ref name="Nature">{{cite web |url=http://toukei.pref.ishikawa.jp/dl/3264/H29-1.pdf |script-title=ja:自然と人口 |accessdate=2017-05-11 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。特別在冬季，由於西北季風的影響，石川縣降水較多，山區是豪雪地帶<ref name="tikei">{{cite web |url=http://www.pref.ishikawa.lg.jp/sizen/kankyo/2.html |script-title=ja:石川の地形、地質、気象 |accessdate=2017-05-12 |publisher={{lang|ja|石川県}} |language=ja}}</ref>，且在降雪時常有打雷現象。石川縣人口約有115萬人，其中以金澤市人口最多，約有46萬人，佔石川總人口的約四成。其次是[[小松市|小松市]]和[[白山市|白山市]]。石川縣人口高度集中在南部的加賀地方<ref name="Population">{{cite web |url=http://toukei.pref.ishikawa.jp/dl/3187/H27census_jinkou.pdf |script-title=ja:平成 27 年国勢調査 人口等基本集計結果（石川県関係分）の概要  |accessdate=2017-05-12 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。金澤市是北陸三縣人口最多的城市，也是北陸經濟的中心之一。

石川縣經濟的特徵之一是過半數製造業從業者從事機械相關產業的工作<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/keikaku/shigoto/documents/honbun.pdf |script-title=ja:石川県産業人材プラン |accessdate=2017-05-12 |publisher={{lang|ja|石川県}} |language=ja}}</ref>，人均機械製造業生產額也是日本第一<ref>{{cite web |url=http://toukei.pref.ishikawa.jp/dl/2344/syuyou2312.pdf |script-title=ja:主要データ集 平成23年12月号  |accessdate=2017-05-12 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。在[[江戶時代|江戶時代]]，以[[加賀國|加賀國]]、[[能登國|能登國]]、[[越中國|越中國]]為領地的[[加賀藩|加賀藩]]注重學術與文藝發展，使得[[城下町|城下町]]金澤傳統文化興隆並維持至今。石川縣各地的傳統工藝也十分發達，人均日本美術展覽會及日本傳統工藝展的入選者數是日本第二<ref name="Culture">{{cite web |url=http://toukei.pref.ishikawa.jp/dl/3264/H29-6.pdf |script-title=ja:教育と文化 |accessdate=2017-05-12 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

石川縣這一地名來源於縣內的[[石川郡_(石川縣)|石川郡]]，而石川郡的名稱則來自於{{link-ja|手取川|手取川}}。手取川河床上大小岩石眾多，有「石川」的別稱<ref name="Chishi">{{cite book |author={{lang|ja|藤田佳久・田林明}}|script-title=ja:《日本の地誌7　中部圏》 |year=2007 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-16767-2|language=ja}}</ref>{{rp|559}}。

石川縣位於[[本州|本州]]島中部的日本海沿岸，若按[[日本地理分區|八分區]]屬於[[中部地方|中部地方]]，細分的話則屬於日本海沿岸的北陸地方。石川縣東西長100.9公里，南北長198.4公里，海岸線全長580.9公里<ref name="gaiyou">{{cite web |url=http://www.pref.ishikawa.lg.jp/kensei/koho/gaiyo/p0.html |script-title=ja:石川県の概要（位置、魅力、自然） |accessdate=2017-05-12 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。石川縣面積4186.09平方公里，是日本面積第35大縣<ref>{{cite web |url=http://www.gsi.go.jp/KOKUJYOHO/MENCHO/201610/area_todofuken.pdf |script-title=ja:都道府県別面積 |accessdate=2017-05-12 |publisher={{lang|ja|国土地理院}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170608055310/http://www.gsi.go.jp/KOKUJYOHO/MENCHO/201610/area_todofuken.pdf |archivedate=2017-06-08 }}</ref>。石川縣西側、北側、北東側臨日本海，西南接[[福井縣|福井縣]]，東接[[富山縣|富山縣]]，東南接[[岐阜縣|岐阜縣]]。日本的領海基線自石川縣[[輪島市|輪島市]]{{link-ja|舳倉島|舳倉島}}直接到[[新潟縣|新潟縣]][[佐渡市|佐渡市]]，未經富山縣海域<ref>{{cite web |url=http://www1.kaiho.mlit.go.jp/JODC/ryokai/kakudai/nih_idx.html |script-title=ja:直線基線十二区域/本州北西岸北部 |accessdate=2017-05-12 |publisher={{lang|ja|海上保安庁}} |language=ja}}</ref>。

石川縣北部的能登地方和南部的加賀地方地形截然不同<ref name="topography">{{cite web |url=http://www.hrr.mlit.go.jp/ekijoka/ishikawa/pamphlet/ishikawa_map5.pdf |script-title=ja:石川県の地形・地質 |accessdate=2017-05-14 |publisher={{lang|ja|国土交通省北陸地方整備局}} |language=ja}}</ref>。能登地方由朝東北方向突入日本海的[[能登半島|能登半島]]構成，地形多是[[第三紀|第三紀]]形成的[[火山岩|火山岩]]及[[沉積岩|沉積岩]]形成的丘陵狀山地<ref name="topography"></ref>。能登半島沒有[[一級水系|一級水系]]，但有町野川、大海川等二級水系<ref name="noto">{{cite web |url=http://www.pref.ishikawa.lg.jp/shinkou/hantou/documents/keikaku2015.pdf |script-title=ja:能登地域半島振興計画 |accessdate=2017-05-14 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。能登半島海域海底至舳倉島附近是[[大陸架|大陸架]]地形，更北端則是白山瀨、大和堆等中深度海域<ref>{{cite web |url=http://www.jma-net.go.jp/jsmarine/japansea.html |script-title=ja:日本海の基礎知識 |accessdate=2017-05-14 |publisher={{lang|ja|気象庁}} |language=ja}}</ref>。能登半島北部多是300米以下的低山和丘陵地帶，西北側海岸是險峻的海蝕崖，東南側則屬於沉降性海岸，海岸線較為複雜。東南部七尾灣內有[[能登島|能登島]]<ref name="noto"></ref>。半島中部是眉丈山南麓断層形成的帶狀低地（邑知潟地溝帶）<ref name="topography"></ref>。半島南部是以寶達山為中心的低山（寶達丘陵），西側海岸線是沙灘海岸（千里濱）<ref name="keikan">{{cite web |url=http://www.pref.ishikawa.lg.jp/toshi/keikan/documents/02.pdf |script-title=ja:石川県の景観 |accessdate=2017-05-14 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

加賀地方東南部以山地（兩白地形）為主，石川縣最高峰白山（2,702米）位於這一地區。西北部則是由河流沖積形成{{link-ja|金澤平原|金沢平野}}<ref name="keikan"></ref>。白山是自白山經[[中國山地|中國山地]]延伸至九州北部的白山火山帶的一部分，部分地區分佈有火山岩和[[凝灰岩|凝灰岩]]<ref name="topography"></ref>。流經加賀地方中部的{{link-ja|手取川|手取川}}發源於白山，注入日本海，全長72公里，流域面積809平方公里，是石川縣最大的河川<ref>{{cite web |url=http://www.hrr.mlit.go.jp/kanazawa/river/t_pamph/p1.html |script-title=ja:手取川流域の概要 |accessdate=2017-05-14 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。手取川流域面積90%是山地，是日本屈指的急流河川<ref>{{cite web |url=http://www.hrr.mlit.go.jp/kanazawa/mb2_jigyo/river/tedoconf/cnf02/mat02_2.pdf |script-title=ja:平成19年度手取川事業概要 |accessdate=2017-05-14 |publisher={{lang|ja|国土交通省北陸地方整備局金沢河川国道事務所}} |language=ja}}</ref>。上游是受河流侵蝕作用形成的河谷地形，中游則[[河階|河階]]地形發達。下游是以白山市鶴來地區為頂部的[[沖積扇|沖積扇]]<ref name="topography"></ref><ref name="keikan"></ref>。[[梯川|梯川]]是發源於加賀地方西南部白山山系{{link-ja|大日山|大日山}}連峰鈴岳的一級水系，全長42公里，流域面積271.2平方公里<ref>{{cite web |url=http://www.hrr.mlit.go.jp/kanazawa/mb2_jigyo/river/jigyo/kake1.html |script-title=ja:梯川のあらまし|accessdate=2017-05-14 |publisher={{lang|ja|国土交通省北陸地方整備局金沢河川国道事務所}} |language=ja}}</ref>。加賀地方北部，流經金澤市內的{{link-ja|犀川 (石川縣)|犀川 (石川県)|犀川}}和{{link-ja|淺野川|浅野川}}均是二級水系，流域河階地形發達<ref name="tikei"></ref>。加賀地方的海岸線是單調的沙灘海岸，沿海地區的平原有木場潟、柴山潟、河北潟等[[潟湖|潟湖]]<ref name="tikei"/>。

=== 自然公園 ===
石川縣境設有[[白山國立公園|白山國立公園]]一處國立公園；[[能登半島國定公園|能登半島國定公園]]、[[越前加賀海岸國定公園|越前加賀海岸國定公園]]兩處國定公園；山中・大日山、獅子吼・手取、碁石峰、白山一里野、醫王山五處縣立自然公園<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/sizen/kankyo/8.html |script-title=ja:県立自然公園 |accessdate=2017-05-14 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。白山國立公園的範圍包括以白山為中心的兩白山地，橫跨石川、富山、福井、岐阜四縣<ref>{{cite web |url=http://www.env.go.jp/park/hakusan/intro/index.html  |script-title=ja:白山国立公園 |accessdate=2017-05-14 |publisher={{lang|ja|環境省}} |language=ja}}</ref>。其中原生地區佔80%以上的面積，其中尤以日本山毛欅的原生林面積較大。另外白山國立公園還是日本最西端的越過越過[[林木線|林木線]]的高山地帶，是許多珍貴動植物的生存西限或南限<ref name="kokuritukokutei">{{cite web |url=http://www.pref.ishikawa.lg.jp/sizen/kankyo/7.html  |script-title=ja:石川県の国立・国定公園 |accessdate=2017-05-14 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

能登半島國定公園是以能登半島海岸一帶為主體的國定公園，橫跨石川縣和富山縣。公園西北部{{link-ja|能登金剛|能登金剛}}和{{link-ja|禄剛崎|禄剛崎}}等險峻的海蝕景觀和東南部七尾灣及能登島等柔和的沉降景觀形成鮮明對比<ref name="kokuritukokutei"></ref>。越前加賀海岸國定公園包括自加賀市至福井縣[[敦賀市|敦賀市]]的海岸線。石川縣內的柴山潟內的片野鴨池是[[雁|雁]]、[[鴨|鴨]]的飛来地，被列入[[濕地公約|濕地公約]]加以保護<ref name="kokuritukokutei"></ref>。

=== 氣候 ===
[[File:Kenrokuen_in_winter_01.jpg|thumb]]
石川縣屬於[[日本海側氣候|日本海側氣候]]，日照時間較短。冬季時這一特徵尤為顯著，受西北季風影響，氣溫較低且多降雪<ref name="weather"></ref>。石川縣在冬季多雷電天氣，冬季的打雷發生次數是日本第一<ref name="weather"></ref>。低氣壓在通過日本海時，自東南方向吹來的強風在越過兩白山地時會發生[[焚風|焚風]]現象。和[[中部地方|中部地方]]太平洋沿岸的[[東海地方|東海地方]]相比，石川縣的[[梅雨|梅雨]]現象較不明顯<ref name="weather"></ref>。能登地方中部至加賀地方的平原地區氣候較為溫和，能登地方北部的年平均氣溫較低，加賀地方山區則低溫且多雨豪雪<ref name="tikei"></ref>。

能登地方由於突入日本海，容易受到寒暖季風的影響。和北陸地方其他都市相比，能登的夏季較為涼爽，冬季的降雪量也較少。年平均氣溫約13至14°C，但能登地方北部氣溫較低。年降水量約1,700至2,100毫米，年日照時間約1,500至1,700小時，最深積雪約有20至60厘米<ref name="weather"></ref>。

加賀地方的平原地區氣候比較溫和，年平均氣溫約13至15°C。年降水量約2,100至3,100毫米，年日照時間約1,400至1,700小時，最深積雪約40至50c厘米<ref name="weather"></ref>。金澤市全年降雪量超過200厘米的年份在1960年代有8次、1970年代有6次、1980年代有9次、1990年代未有記錄、2000年代有3次。自1990年代以來明顯減少<ref>{{cite web |url=http://www.data.jma.go.jp/obd/stats/etrn/view/annually_s.php?prec_no=56&prec_ch=%90%CE%90%EC%8C%A7&block_no=47605&block_ch=%8B%E0%91%F2&year=&month=&day=&elm=annually&view= |script-title=ja:気象統計情報 金沢 年ごとの値 主な要素 |accessdate=2017-05-14 |publisher={{lang|ja|気象庁}} |language=ja}}</ref>。加賀地方冬季日照時間極少，夏季月平均日照時間約有180小時，冬季只有70小時。加賀地方山區的最深積雪平均值在220厘米以上，是平原地區的4倍以上。[[白山市|白山市]]白峰地區的最深積雪記錄達480厘米。這一地區大雪的原因是因西伯利亞的乾冷季風在通過日本海的對馬暖流上空時水蒸汽含量增加，碰到兩白山地之後上冷卻並產生降雪<ref name="weather"></ref>。

=== 行政區劃 ===
石川縣可以分為北部的能登地方和南部的加賀地方，相當於[[令制國|令制國]]中的[[能登國|能登國]]、[[加賀國|加賀國]]。加賀地方包括[[金澤市|金澤市]]、[[白山市|白山市]]、[[小松市|小松市]]、[[加賀市|加賀市]]、[[野野市市|野野市市]]、[[能美市|能美市]]、[[河北市|河北市]]七市和[[津幡町|津幡町]]、[[内灘町|内灘町]]、[[川北町|川北町]]三町，人口約有95萬，佔石川縣的83.4%。能登地方包括[[七尾市|七尾市]]、[[輪島市|輪島市]]、[[羽咋市|羽咋市]]、[[珠洲市|珠洲市]]、[[志賀町|志賀町]]、[[宝達志水町|宝達志水町]]、[[能登町|能登町]]、[[穴水町|穴水町]]、[[中能登町|中能登町]]。人口約有19萬人，佔16.6%<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/sichousien/koiki/torikumijyoukyou.html |script-title=ja:合併市町一覧表 |accessdate=2017-05-14 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

=== 原始時期 ===
[[File:Okyozuka.jpg|thumb]]）]]
石川縣內的舊石器時代和[[繩文時代|繩文時代]]早期的遺跡很少，直到繩文時代中晚期時才開始出現大量遺跡。[[能登町|能登町]]的真脇遺跡橫跨繩文時代前期至晩期，有約4,000年的歷史<ref>{{cite web |url=http://www.ishikawa-maibun.or.jp/sanpo/mawaki.html |script-title=ja:真脇遺跡 |accessdate=2017-05-14 |publisher={{lang|ja|財団法人石川県埋蔵文化センター}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170823041145/http://www.ishikawa-maibun.or.jp/sanpo/mawaki.html |archivedate=2017年8月23日 |df= }}</ref>。繩文時代後期的遺跡則有金澤市的近森遺跡<ref>{{cite web |url=http://www4.city.kanazawa.lg.jp/11104/bunkazaimain/shiteibunkazai/kinenbutsu/tikamori.html |script-title=ja:チカモリ遺跡 |accessdate=2017-05-14 |publisher={{lang|ja|金沢市}} |language=ja}}</ref>、野野市市的御經塚遺跡<ref>{{cite web |url=http://www.ishikawa-maibun.or.jp/sanpo/okyouzuka.html |script-title=ja:御経塚遺跡 |accessdate=2017-05-14 |publisher={{lang|ja|財団法人石川県埋蔵文化センター}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20130706031843/http://www.ishikawa-maibun.or.jp/sanpo/okyouzuka.html |archivedate=2013年7月6日 |df= }}</ref>。1980年，在近森遺跡發現了將栗樹巨木縱切排成圓形的環狀木柱列，之後在真脇遺跡也有發現，但用途至今不明。羽咋市的吉崎·次場遺跡是北陸地方最大的[[彌生時代|彌生時代]]遺跡，並發現了這一時期北陸地方和[[近畿地方|近畿]]、[[東北地方|東北]]、[[山陰地方|山陰]]等地交流的證據<ref>{{cite web |url=http://www.ishikawa-maibun.or.jp/sanpo/yossakisuba.html |script-title=ja:吉崎・次場遺跡 |accessdate=2017-05-14 |publisher={{lang|ja|財団法人石川県埋蔵文化センター}} |language=ja |title=存档副本 |archiveurl=https://web.archive.org/web/20161012094425/http://www.ishikawa-maibun.or.jp/sanpo/yossakisuba.html |archivedate=2016-10-12 |deadurl=yes }}</ref>。

在能美市發現了有50多處古墳的能美古墳群，並在位於其中心的和田山・末寺山古墳群發現了大量兵器和武具<ref>{{cite web |url=http://www.ishikawa-maibun.or.jp/sanpo/wadayama.html |script-title=ja:和田山・末寺山古墳群 |accessdate=2017-05-14 |publisher={{lang|ja|財団法人石川県埋蔵文化センター}} |language=ja |title=存档副本 |archiveurl=https://web.archive.org/web/20170827234358/http://www.ishikawa-maibun.or.jp/sanpo/wadayama.html |archivedate=2017-08-27 |deadurl=yes }}</ref>。能美古墳群中的秋常山1号墳是一座全長約140米的前方後圓墳。中能登町的雨宮古墳群則發現了北陸最大的前方後方墳雨宮1號墳。七尾市能登島的須曾蝦夷穴古墳的墓室是穹頂型，和[[朝鮮半島|朝鮮半島]]的古墳相通。

=== 飛鳥至平安時代 ===
[[File:Keta-taisha_keidai.JPG|thumb]]參拜過的氣多大社]]
石川縣在[[飛鳥時代|飛鳥時代]]屬於[[越國_(日本)|越國]]，三越分割之後屬於[[越前國|越前國]]。進入[[奈良時代|奈良時代]]之後，718年（[[養老_(日本年號)|養老]]2年），越前國的羽咋、能登、鳳至、珠洲4郡另立[[能登國|能登國]]。741年（[[天平|天平]]13年），能登國被併入越中國。這時[[大伴家持|大伴家持]]被任命為越中國的國司。757年（[[天平勝寶|天平勝寶]]9年），能登國再次被分出越中國。[[平安時代|平安時代]]初期的823年（[[弘仁|弘仁]]14年），[[越前國|越前國]]的加賀、江沼2郡被分出，另立[[加賀國|加賀國]]。

[[七尾市|七尾市]]的{{link-ja|能登國分寺|能登国分寺}}原本是由能登地方的支配者能登臣一族修建於[[飛鳥時代|飛鳥時代]]的寺院，在843年（[[承和_(日本)|承和]]10年）成為[[國分寺|國分寺]]，有[[法起寺|法起寺]]式的[[伽藍|伽藍]]。在超過400年的時間內都是能登的佛教中心<ref>{{cite web |url=http://bunka.nii.ac.jp/heritages/detail/210060 |script-title=ja:能登国分寺跡附建物群跡 |accessdate=2017-05-14 |publisher={{lang|ja|文化遺産オンライン}} |language=ja}}</ref>。自奈良時代至平安時代，[[渤海國|渤海國]]時節頻繁到達能登半島進行貿易。志賀町的福浦港是[[渤海使|渤海使]]船舶的修理和住宿地，能登國在平安時代初期接待渤海使的能登客院位於這裡<ref>{{cite web |url=http://www.ishikawa-c.ed.jp/content/m-media/rekishi/rekishiSOZAI/fukuura/fukuura.html |script-title=ja:大陸との貿易の窓口 富来福浦港 |accessdate=2017-05-14 |publisher={{lang|ja|石川県教育センター}} |language=ja}}</ref>。

[[野野市市|野野市市]]的末松廢寺遺跡是加賀地方北部的有力氏族道君在7世紀後期設立的寺院，為法起寺式伽藍，屋頂部分磚瓦燒製於[[能美市|能美市]]辰口地區<ref>{{cite web |url=http://www.ishikawa-maibun.or.jp/sanpo/suematu.html |script-title=ja:大陸との貿易の窓口 富来福浦港 |accessdate=2017-05-14 |publisher={{lang|ja|石川県教育センター}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20150707235448/http://www.ishikawa-maibun.or.jp/sanpo/suematu.html |archivedate=2015年7月7日 |df= }}</ref>。在奈良和平安時代，北陸地方有眾多[[東大寺|東大寺]]、[[西大寺_(奈良市)|西大寺]]等寺院的莊園。橫跨白山市至金澤市的東大寺領橫江莊也是其中之一<ref>{{cite web |url=http://www.ishikawa-maibun.or.jp/sanpo/kamiaraya.html |script-title=ja:東大寺領横江荘遺跡　上荒屋遺跡 |accessdate=2017-05-14 |publisher={{lang|ja|石川県教育センター}} |language=ja |title=存档副本 |archiveurl=https://web.archive.org/web/20170824230835/http://www.ishikawa-maibun.or.jp/sanpo/kamiaraya.html |archivedate=2017-08-24 |deadurl=yes }}</ref>。平安時代也是以白山為信仰對象的{{link-ja|白山信仰|白山信仰}}的起源期。登山道起點之一的[[白山比咩神社|白山比咩神社]]是信仰的據點<ref>{{cite web |url=https://www.pref.ishikawa.lg.jp/hakusan/publish/sizen/documents/sizen21.pdf |script-title=ja:白山の禅定道 |accessdate=2017-05-14 |publisher={{lang|ja|石川県白山自然保護センター}} |language=ja}}</ref>。

在平安時代末期的源平和戰中，[[源義仲|源義仲]]（木曾義仲）在倶利伽羅峠之戰中戰勝平家軍隊<ref>{{cite web |url=http://kankou.town.tsubata.ishikawa.jp/content/detail.php?id=40 |script-title=ja:倶利伽羅古戦場 |accessdate=2017-05-14 |publisher={{lang|ja|津幡町}} |language=ja}}</ref>，之後又在篠原之戰中乘勝追擊，攻入京都。

=== 鎌倉至戰國時代 ===
[[File:Torigoe01.jpg|thumb]]
在[[鎌倉時代|鎌倉時代]]，幕府新設[[守護|守護]]一職來管理令制國軍政事務。比企朝宗和北條朝時先後擔任加賀國和能登國的守護。[[室町時代|室町時代]]之後，加賀國由[[斯波氏|斯波氏]]、[[富樫氏|冨樫氏]]控制，能登國由{{link-ja|吉見氏|吉見氏}}、[[畠山氏#七尾畠山氏|畠山氏]]控制。[[應仁之亂|應仁之亂]]時期，[[浄土真宗|浄土真宗]]開始在加賀國普及。農民發動的[[加賀一向一揆|加賀一向一揆]]顛覆了守護富樫政親的統治，加賀國因此脫離武士支配長達約百年。[[本願寺|本願寺]]在金澤的台地上興建尾山御坊，並以此為支配據點<ref>{{cite web |url=http://www.ishikawa-c.ed.jp/content/m-media/rekishi/rekishiSOZAI/ikkoikki/ikkoikki.html |script-title=ja:百姓ノ持チタル国〜加賀一向一揆〜 |accessdate=2017-05-14 |publisher={{lang|ja|石川県教育センター}} |language=ja}}</ref>。和本願寺敵對的[[織田信長|織田信長]]派遣[[柴田勝家|柴田勝家]]平定石川地區，並將能登國封給[[前田利家|前田利家]]，加賀國封給[[佐久間盛政|佐久間盛政]]。織田信長死後，[[豐臣秀吉|豐臣秀吉]]掌握實權，加賀國改由[[前田利家|前田利家]]所領。他在尾山御坊的遺跡處修建尾山城（金澤城），並開始建設城下町。

在能登國，1420年代後期，畠山滿慶開始修築七尾城，並以此為統治據點。畠山氏的統治在第七代當主畠山義總的時代達到鼎盛。義總死後大名則淪為傀儡。1560年（[[永禄|永禄]]3年），第九代當主畠山義綱雖奪回實權，卻在1566年（永禄9年）的政變中被逐出能登國。1577年（[[天正_(日本)|天正]]5年），[[上杉謙信|上杉謙信]]攻入能登國，畠山氏滅亡。

=== 江戶時代 ===
[[File:Kanazawa_Castle_Gate.JPG|thumb]]
[[前田利家|前田利家]]的長子[[前田利長|前田利長]]在關原之戰中支持[[德川家康|德川家康]]的東軍，戰後被賜予[[越中國|越中國]]。利長在[[江戶幕府|江戶幕府]]的幕藩體制下是統有加賀國、能登國、越中國三國的[[加賀藩|加賀藩]]藩主。加賀藩前田家雖是[[外樣大名|外樣大名]]，卻是[[大名_(稱謂)|大名]]中[[石高|石高]]之首，領有約120萬石。1639年（[[寬永|寬永]]16年），[[前田利常|前田利常]]將家督讓給長子[[前田光高|前田光高]]，並將次子前田利次分封至[[富山藩|富山藩]]，三子前田利治分封至{{link-ja|大聖寺藩|大聖寺藩}}。然而在1645年（[[正保|正保]]元年），前田光高突入過世，第五代藩主光高長子前田綱紀年幼，由利常輔佐藩政。在利常輔佐綱紀的時期，實施以救濟貧農，徹查稅收的改作法農政改革，並取得效果使得藩政安定。自這一時期，加賀藩開始將儲存的大米經日本海運至[[關門海峽|關門海峽]]，再經[[瀨戶內海|瀨戶內海]]運到大坂，是之後西回海運的前身<ref>{{cite web |url=https://www.kaijipr.or.jp/mamejiten/fune/fune_3.html |script-title=ja:北前船（きたまえぶね）ってどんな船|accessdate=2017-05-14 |publisher={{lang|ja|日本海事広報協会}} |language=ja}}</ref>。不過在16世紀中期，發生了白山噴發和手取山洪水等災害，造成多人遇難<ref name="sabou">{{cite web |url=http://www.hrr.mlit.go.jp/kanazawa/hakusansabo/03history/saigai01.html  |script-title=ja:手取川災害略史|accessdate=2017-05-14 |publisher={{lang|ja|金沢河川国道事務所}} |language=ja}}</ref>。

[[File:KUTANI_Ewer.JPG|thumb]]
加賀藩注重發展產業，獎勵學術和文藝。使得以城下町金澤為中心的傳統文化興隆至今。金澤城內的御細工所原本是修理兵器武具的組織。前田利常使其開始製作修理茶具和美術工藝品，前田綱紀時期更擁有超過20種職種。綱紀還鼓勵學術，招聘木下順庵、室鳩巢、稻生若水等學者，並收集古今東西圖書，創建尊經閣文庫<ref>{{cite web |url=http://www.library.pref.ishikawa.jp/booklist/2008/168.html |script-title=ja:第168回企画展示　「尊経閣叢刊」にみる古典文学の世界|accessdate=2017-05-14 |publisher={{lang|ja|石川県立図書館}} |language=ja}}</ref>。[[兼六園|兼六園]]也始建於綱紀時期，但現貌則是完成於[[江戶時代|江戶時代]]後期<ref>{{cite web |url=http://www.pref.ishikawa.jp/siro-niwa/kenrokuen/history.html |script-title=ja:兼六園の歴史|accessdate=2017-05-14 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

江戶時代因在輪島發現[[矽藻土|矽藻土]]，當地開始生產名為[[輪島塗|輪島塗]]的堅固漆器，廣泛用於日用食器。北前船停靠地的輪島港利用海運之利將其行銷日本全國。在江戶時代後期更加入[[戧金|戧金]]和蒔絵技法，發展成為美術工藝品<ref>{{cite web |url=http://www.wajimanuri.co.jp/history.html |script-title=ja:輪島塗の歴史 |accessdate=2017-05-14 |publisher={{lang|ja|田谷漆器店}} |language=ja}}</ref>。大聖寺藩早江戶初期開發礦山，並因在九谷村發現[[高嶺石|高嶺石]]而開始生產彩繪瓷器（九谷燒）。九谷燒的生產雖然曾一度中止，但之後在加賀藩的努力下復興，在明治時期更遠銷海外<ref>{{cite web |url=http://www.kutaniyaki.or.jp/kutaniyaki/index.html |script-title=ja:九谷焼とは |accessdate=2017-05-14 |publisher={{lang|ja|能美市九谷焼資料館}} |language=ja}}</ref>。

江戶時代後期，加賀藩還在藩校中分設文學校明倫堂和武學校經武館。明倫堂除了[[儒學|儒學]]之外，還教授[[易學|易學]]、醫學、本草學、曆法學、[[和算|算學]]。經武館則教授[[馬術|馬術]]、[[劍術|劍術]]。幕末時期還設立洋式兵學校壯猶館，為進行航海、測量實習還設立軍艦所，自歐洲購入西式艦船，注重海防<ref>{{cite web |url=http://www.lib.kanazawa.ishikawa.jp/kinsei/hokuetsu.pdf |script-title=ja:加賀藩と北越戦争 |accessdate=2017-05-14 |publisher={{lang|ja|金沢市図書館}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20140905023626/http://www.lib.kanazawa.ishikawa.jp/kinsei/hokuetsu.pdf |archivedate=2014年9月5日 |df= }}</ref>。

=== 近代 ===
[[File:Shiinoki_Geihinkan.jpg|thumb]]
1869年[[版籍奉還|版籍奉還]]之後，加賀藩改為金澤藩。第14代藩主前田慶寧被任命為金澤藩知事。1871年8月，日本實施[[廢藩置縣|廢藩置縣]]，金澤藩域改為金澤縣（第1次），大聖寺藩則改為大聖寺縣。同年11月，兩縣被廢。旧金澤縣中[[越中國|越中國]]部分（不含射水郡）新設{{link-ja|新川縣|新川県}}，能登國和越中國射水郡新設{{link-ja|七尾縣|七尾県}}，加賀地方設立金澤縣（第2次）。1872年，金澤縣廳搬到石川郡美川町（現[[白山市|白山市]]美川南町），並改名石川縣。同年10月，七尾縣（不含射水郡）和石川縣合併（射水郡和新川縣合併），11月時再合併{{link-ja|足羽縣|足羽県}}白山山麓18個村莊，領域相當於現在的石川縣。1873年，石川縣廳搬回金澤，但縣名並未變更。1876年，新川縣（大致相當於現在的[[富山縣|富山縣]]）和{{link-ja|敦賀縣|敦賀県}}（大致相當於現在的[[福井縣|福井縣]]）的{{link-ja|嶺北|嶺北}}劃入石川縣，這一時期被稱為大石川縣。不過在1881年和1883年，福井縣、富山縣相繼劃出石川，也確定了石川縣現在的縣域<ref>{{cite web |url=http://toukei.pref.ishikawa.jp/annual/kaiseki/iis/iis_h14/iis14-j/iis14-j1.pdf |script-title=ja:石川県の沿革 |accessdate=2017-05-14 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

1874年，舊金澤藩士長谷川準也（之後出任金澤市長）仿效官營模範工廠[[富岡紡紗廠|富岡紡紗廠]]，創建金澤紡紗廠，是石川縣內殖產興業的先驅。金澤紡紗廠工人津田吉之助之子津田米次郎致力於織機的機械化，在1900年發明了日本首個力織機，金澤紡紗廠也因此產量大幅增加<ref name="Name">{{cite book |author={{lang|ja|竹内理三}}|script-title=ja:《角川日本地名大辞典 17 石川県》 |year=1981 |publisher={{lang|ja|角川書店}} |location=東京都 |isbn=4-04-001170-8|language=ja}}</ref>{{rp|43}}。1887年4月，金澤設立舊制第四高等中學校，是現在[[金澤大學|金澤大學]]的前身<ref>{{cite web |url=http://muse.w3.kanazawa-u.ac.jp/50th/genryu/shikou_history.html |script-title=ja:金沢大学50年史 |accessdate=2017-05-15 |publisher={{lang|ja|金沢大学}} |language=ja}}</ref>。1898年10月，在金澤城內還設立了[[大日本帝國陸軍|就陸軍]][[第9師團|第9師團]]司令部。金澤因此發展為北陸地區的學術和軍事中心。

[[File:Kanazawa_Station_in_Meiji_era.JPG|thumb]]
1897年9月，[[北陸本線|北陸線]]自[[福井站_(福井縣)|福井站]]延伸至{{link-ja|小松站|小松駅}}。1898年（明治31年）4月又延伸到[[金澤站|金澤站]]，並在同年11月延伸至[[高岡站|高岡站]]。在同年4月，[[七尾線|七尾鐵道]]也開通了自津幡臨時站至矢田新站的路段，並在1907年被國有化，1925年延伸至和倉溫泉站，1935年延伸至輪島站。1900年，金澤開始電力送電，並在翌年開通市內電話，1908年開始供給瓦斯<ref name="tuusi">{{cite web |url=http://www4.city.kanazawa.lg.jp/13003/rekishi/tsuushi.html |script-title=ja:金沢の歴史（通史） |accessdate=2017-05-15 |publisher={{lang|ja|金沢市}} |language=ja}}</ref>。

1918年，富山縣魚津發生[[米騷動|米騷動]]，石川縣也受到波及<ref>{{cite web |url=http://dspace.lib.kanazawa-u.ac.jp/dspace/bitstream/2297/10828/1/hashimoto-5.pdf |script-title=ja:米騒動 |accessdate=2017-05-15 |publisher={{lang|ja|金沢大学学術情報リポジトリ}} |language=ja |title=存档副本 |archive-url=https://web.archive.org/web/20170202022949/http://dspace.lib.kanazawa-u.ac.jp/dspace/bitstream/2297/10828/1/hashimoto-5.pdf |archive-date=2017-02-02 |dead-url=yes }}</ref>。1919年，金澤市內的路面電車開始運行<ref name="tuusi"></ref>。金澤的城市化以此為契機有明顯發展。咖啡廳、電影等大眾文化開始普及。1934年，手取川流域發生大洪水，造成97人死亡，15人失踪<ref>{{cite web |url=http://www.hrr.mlit.go.jp/kanazawa/hakusansabo/03history/saigai02.html |script-title=ja:昭和9年災害概要 |accessdate=2017-05-15 |publisher={{lang|ja|金沢河川国道事務所}} |language=ja}}</ref>。

=== 現代 ===
[[File:Komatsu_airport_terminal_building.jpg|thumb]]
石川縣是日本二戰中受空襲次數最少的縣之一，使得眾多基礎設施得以逃過戰火摧殘。戰後美軍接收內灘村的砲彈試射場，引發當地民眾的反對運動，被稱為{{link-ja|內灘闘争|内灘闘争}}<ref>{{cite web |url=http://www.town.uchinada.lg.jp/webapps/www/info/detail.jsp?id=115 |script-title=ja:内灘闘争 |accessdate=2017-05-15 |publisher={{lang|ja|内灘町}} |language=ja}}</ref>。美軍還接收了[[大日本帝國海軍|舊海軍]]的[[小松飛行場|小松飛行場]]，並在1955年開通了至大阪的不定期航線。1958年美軍歸還機場之後，[[航空自衛隊|航空自衛隊]]駐紮小松飛行場並在1961年開設航空自衛隊小松基地，正式成為自衛隊和民航共用的機場<ref>{{cite web |url=http://www.komatsuairport.jp/about/history.html |script-title=ja:沿革 |accessdate=2017-05-15 |publisher={{lang|ja|小松空港}} |language=ja}}</ref>。

1957年，[[日本放送協會|NHK]]開始在石川縣播出電視節目。翌年，商業廣播[[北陸放送|北陸放送]]也開始播出電視。1962年更開始播出彩色電視。同年12月末至翌年2月初，北陸地方發生連續大範圍降雪記錄。金澤市的最深積雪達181厘米，造成交通通信障礙、停電、住宅倒塌等影響<ref>{{cite web |url=http://www.data.jma.go.jp/obd/stats/data/bosai/report/1963/196301/196301.html |script-title=ja:昭和38年1月豪雪 |accessdate=2017-05-15 |publisher={{lang|ja|気象庁}} |language=ja}}</ref>。

1963年，北陸本線福井站至金澤站區間實現電氣化<ref name="kensei3839">{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/38-39.html |script-title=ja:県政の主なあゆみ昭和38年〜昭和39年 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。翌年，金澤站至[[富山站_(日本)|富山站]]區間也實現電氣化。[[日本國有鐵道|國鐵]]能登線也在1959年開通穴水站至鵜川站區間，1964年實現全線通車<ref name="kensei3839"></ref>。另一方面，1967年，金澤市內的路面電車被廢除<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/42-43.html |script-title=ja:県政の主なあゆみ昭和42年〜昭和43年 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。1972年，石川縣首條高速公路[[北陸自動車道|北陸自動車道]]的金澤西至小松區間開通<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/46-47.html |script-title=ja:県政の主なあゆみ昭和46年〜昭和47年 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。1978年，石川縣內的北陸自動車道全線通車<ref name="kensei5253">{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/52-53.html |script-title=ja:県政の主なあゆみ昭和52年〜昭和53年 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。能登也在1973年開通了能登海濱公路<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/48-49.html |script-title=ja:県政の主なあゆみ昭和48年〜昭和49年 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>，1982年實現了能登收費公路全線通車<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/57.html |script-title=ja:県政の主なあゆみ昭和57年 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。1977年，尾口村（現白山市）和岐阜縣[[白川村|白川村]]之間的白川超級林道通車<ref name="kensei5253"></ref>。

[[File:Senmaida.jpg|thumb]]]]
1980年，石川縣完成了以治水、提供都市用水、發電為目的的手取川水庫。該水庫提供了北起七尾市能登島，南至加賀市縣內七成人口的用水<ref>{{cite web |url=http://www.hrr.mlit.go.jp/kanazawa/mb2_jigyo/river/jigyo/dam.html |script-title=ja:手取川ダムの概要 |accessdate=2017-05-15 |publisher={{lang|ja|国土交通省北陸地方整備局}} |language=ja}}</ref>。1993年，志賀町的志賀核能發電廠開始運行<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/h5-2.html |script-title=ja:県政の主なあゆみ平成5年後期 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。但在2003年，原本計劃興建在珠洲市的珠洲核能發電廠被凍結<ref>{{cite web |url=http://www.rikuden.co.jp/press/attach/03120501.pdf |script-title=ja:珠洲原子力発電所計画の凍結について |accessdate=2017-05-15 |publisher={{lang|ja|北陸電力}} |language=ja}}</ref>。

昭和末期之後，能登地方從以鐵路為中心的交通體系變為以公路和航空為中心的交通體系。1988年，JR能登線被廢除，交由第三部門能登鐵道運行<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/63.html |script-title=ja:県政の主なあゆみ昭和63年 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。1991年，JR七尾線的津幡站至和倉溫泉站區間雖實現電氣化，但和倉温泉站至輪島站區間則交由能登鐵道運行<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/h3-2.html |script-title=ja:県政の主なあゆみ平成3年後期 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。但在2001年，能登鐵道七尾線的穴水站至輪島站區間被廢止<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/chiji/gikai/h12_9_18/11_noto.html |script-title=ja:県議会の議案説明要旨 9月補正予算の概要について |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。2005年，能登線也被廢除<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/h17-1.html |script-title=ja:県政の主なあゆみ平成17年前期 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。與之相對的是，2003年，石川縣第二個機場[[能登機場|能登機場]]開業<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/h15-1.html |script-title=ja:県政の主なあゆみ平成15年前期 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。[[能越自動車道|能越自動車道]]也在1998年<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/h10-1.html |script-title=ja:県政の主なあゆみ平成10年前期 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>和2006年兩度延長<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kenmin/kouhou/nenpyo/h18-1.html |script-title=ja:県政の主なあゆみ平成18年前期 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

2007年3月25日，能登半島海域發生6.9級地震，部分地區受災嚴重<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/bousai/bousai_g/notohanto_eq/kirokushi/index.html |script-title=ja:平成19年能登半島地震災害記録誌 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。2008年7月28日，淺野川時隔55年再次發生泛濫，創下200年一遇的豪雨記錄<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kasen/sansha-i/documents/hokoku.pdf |script-title=ja:局所的豪雨に対応した新たな河川管理検討委員会報告書 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

2009年，金澤市被聯合國教科文組織列入創造都市<ref>{{cite web |url=http://www4.city.kanazawa.lg.jp/11001/souzoutoshi/torikumi/index.html |script-title=ja:ユネスコ・クラフト創造都市 |accessdate=2017-05-15 |publisher={{lang|ja|金沢市}} |language=ja}}</ref>。同年，能登地方北部的傳統祭禮{{link-ja|饗祭|アエノコト}}被列入世界無形文化遺產<ref name="aenokoto">{{cite web |url=http://www.pref.ishikawa.lg.jp/kyoiku/bunkazai/mukeibunkaisan/index.html |script-title=ja:「奥能登のあえのこと」ユネスコ無形文化遺産の「代表一覧表」に記載 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。2011年，能登的里山里海被列入世界農業遺產<ref>{{cite web |url=http://www.fao.org/fileadmin/templates/giahs/PDF/GIAHS_Noto_proporsal_17_Dec.pdf |script-title=en:Noto's Satoyama and Satoumi Globally Important Agricultural Heritage Systems(GIAHS) |accessdate=2017-05-15 |publisher={{lang|ja|FAO}} |language=en}}</ref><ref>{{cite web |url=http://www.maff.go.jp/j/pr/aff/1109/spe1_03.html |script-title=ja:特集1 世界農業遺産(3) 能登の里山里海 |accessdate=2017-05-15 |publisher={{lang|ja|農林水産省}} |language=ja}}</ref>。

== 人口 ==
1920年日本首次人口普查時，石川縣有人口747,360人<ref name="population1">{{cite web |url=http://toukei.pref.ishikawa.jp/dl/3257/27-03.pdf |script-title=ja:人口 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。由於包括石川縣在內的日本海沿岸地區在戰前是工業化後進地區，長期扮演勞動力輸出地區的角色，石川縣人口增加緩慢。直到1970年時，石川縣人口才突破百萬大關<ref name="population1"></ref>。此後石川縣的人口增長速度有所加速。2000年人口普查時，石川縣達到人口峰值1,180,977人。此後由於老齡化和少子化問題，石川縣人口開始減少。2017年3月31日時，石川縣有人口1,149,894人<ref>{{cite web |url=https://www.pref.ishikawa.lg.jp/sichousien/juki/h29_3.html |script-title=ja:住民基本台帳月報（平成29年3月） |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。石川縣人口自2005年開始自然增長率為負數，且自1980年代以來長期出現人口外流問題<ref name="population2">{{cite web |url=https://www.pref.ishikawa.lg.jp/kikaku/keikaku/documents/150828shiryo2.pdf |script-title=ja:いしかわ創生人口ビジョン |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。另一方面，石川縣的總和生育率有1.45，略高於日本平均值<ref name="population2"></ref>。

== 政治 ==
[[File:Ishikawa_Prefectural_Office.jpg|right]]
1996年日本導入[[小選舉區制|小選舉區制]]之後，在[[眾議院_(日本)|眾議院]]選舉中，石川縣被劃分為3個選舉區<ref>{{cite web |url=http://www.soumu.go.jp/main_content/000240057.pdf　 |script-title=ja:石川県 |accessdate=2017-05-15 |publisher={{lang|ja|総務省}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。這三個選舉區均是[[自由民主黨_(日本)|自民黨]]佔據優勢的選區，其中又以日本前首相[[森喜朗|森喜朗]]所在的[[石川縣第2區|石川縣第2區]]最為強勢，其他政黨候選人從未當選。2000年之後，[[民主黨_(1998年)|民主黨]]在[[石川縣第1區|石川縣第1區]]有過兩次當選記錄，在[[石川縣第3區|石川縣第3區]]有過三次當選記錄<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/senkan/kako_data/syugi_kako.html　 |script-title=ja:過去の選挙結果―衆議院議員総選挙・最高裁判所裁判官国民審査― |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在[[參議院_(日本)|參議院]]選舉中，石川縣屬於[[石川縣選舉區|石川縣選舉區]]，共有2名議員。目前自民黨獨占這兩個議席<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/senkan/kako_data/sangi_kako.html　 |script-title=ja:過去の選挙結果―参議院議員通常選挙― |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

石川縣在戰後只有經歷了四位[[石川縣知事列表|知事]]。{{link-ja|中西陽一|中西陽一}}在1963年首次當選石川縣知事之後連續八次當選，於1994年在任期中去世，執政石川縣長達31年，是日本執政時間最長的知事。現任石川縣知事是{{link-ja|谷本正憲|谷本正憲}}。他在1994年首次當選以來也已連續六次當選<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/chiji/profile.html　 |script-title=ja:知事のプロフィール |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。{{link-ja|石川縣議會|石川県議会}}共有43名議員，其中自民黨是最大黨派，有30名議員<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/gikai/meibo/kaiha20150430.html　 |script-title=ja:会派別名簿 |accessdate=2017-05-12 |publisher={{lang|ja|石川県}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2015年度，石川縣財政收入約5601億日元，支出約5517億日元，略有實現財政盈餘<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/zaisei/zaisei/aramasi2016-12/documents/h28aramasi2-2.pdf　 |script-title=ja:平成 27 年度決算の状況 |accessdate=2017-05-12 |publisher={{lang|ja|石川県}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

=== 第一產業 ===
[[File:OMC01.jpg|thumb]]
石川縣耕地面積有42,400公頃，佔石川縣面積的一成<ref name="agriculture">{{cite web |url=http://www.pref.ishikawa.lg.jp/nousei/vision/documents/03genkyou.pdf　 |script-title=ja:いしかわの食と農業・農村の現況 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2014年度，石川縣的農業生產額有475億日元，其中[[大米|大米]]佔一半比例，其次是蔬菜和畜產和<ref name="agriculture"></ref>。石川縣注重農產品的品牌化，近年來推出了「加賀蔬菜」<ref>{{cite web |url=http://www.kanazawa-kagayasai.com/kagayasai/　 |script-title=ja:加賀野菜について |accessdate=2017-05-15 |publisher={{lang|ja|加賀野菜}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、「能登蔬菜」<ref>{{cite web |url=http://www.noto-yasai.jp/about/　 |script-title=ja:能登野菜とは？ |accessdate=2017-05-15 |publisher={{lang|ja|能登野菜振興協議会}} |language=ja }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>等品牌。石川縣農業綜合研究中心還開發了名為「紅色浪漫」的新品種高級[[葡萄|葡萄]]<ref>{{cite web |url=http://www.jeinou.com/technology/2008/02/12/093901.html|script-title=ja:赤色の大粒ブドウ新品種'ルビーロマン |accessdate=2017-05-15 |publisher={{lang|ja|みんなの農業広場}} |language=ja}}</ref>。石川縣廳還對在縣內飼養的黑毛和牛中高品質的能登牛進行認證<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/nousan/chikusan/chikusann/notoushi.html|script-title=ja:能登牛について |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

2008年，石川縣林業生產額有28.6億日元，只有1985年時69.2億日元的41%。另一方面，木材供給量中縣產材的比例從1975年的18%增加到2009年的39%。樹種別原木生產量中以[[日本柳杉|日本柳杉]]最多<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/shinrin/kikaku/documents/forestvision2011_p59-p83.pdf |script-title=ja:いしかわ森林・林業・木材産業振興ビジョン2011 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。能登半島出產的[[羅漢柏|羅漢柏]]在品質上享有盛譽<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/ringyo/ate/documents/notonoate.pdf |script-title=ja:能登のあて |accessdate=2017-05-15 |publisher={{lang|ja|石川県農林水産部}} |language=ja}}</ref>。

石川縣2005年的海綿漁業、養殖業生產量有60,306噸，換算成金額有219億日元，是本州日本海沿岸12府縣中生產量第三大縣和生產金額最多的縣<ref name="suisan">{{cite web |url=http://www.pref.ishikawa.lg.jp/suisanka/newvision/documents/virsion1.pdf |script-title=ja:石川県新水産振興ビジョン2007 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。石川縣捕獲的主要海魚有[[斑點莎瑙魚|斑點莎瑙魚]]、[[日本竹筴魚|日本竹筴魚]]、[[白腹鯖|白腹鯖]]、[[鰤魚|鰤魚]]、[[河魨|河魨]]、[[太平洋褶柔鱼|北魷]]等<ref>{{cite web |url=http://www.pref.ishikawa.jp/suisan/center/sigenbu.files/opendata/top.html |script-title=ja:主要魚種の年動向（石川県主要10港の水揚量と魚種概要） |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。石川縣的內水漁業則以[[香魚|香魚]]為主<ref name="suisan"></ref>。

=== 第二產業 ===
[[File:Kohrinbo_ATRIO.JPG|thumb]]
2014年，石川縣的工業品生產金額有26019億日元，其中機械產業就佔了約三分之二的比例，且加賀地方集中了八成以上的生產金額<ref>{{cite web |url=http://toukei.pref.ishikawa.jp/dl/3105/H26Kogyo_2.pdf |script-title=ja:調査結果の概要 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。世界第二大建設機械企業[[小松製作所|小松製作所]]創業於小松市，市內有眾多小松製作所的工廠和關係企業，形成了[[公司市鎮|公司市鎮]]<ref>{{cite web |url=http://www.city.komatsu.lg.jp/5127.htm |script-title=ja:乗りもののまち |accessdate=2017-05-15 |publisher={{lang|ja|小松市}} |language=ja}}</ref>。世界著名[[顯示器|顯示器]]生產企業[[藝卓|藝卓]]的總部位於白山市<ref>{{cite web |url=http://www.eizo.co.jp/company/information/outline/index.html |script-title=ja:基本情報 |accessdate=2017-05-15 |publisher={{lang|ja|EIZO}} |language=ja}}</ref>。金澤市的[[I-O_數據設備|I-O 數據設備]]是日本重要的生產電腦相關機械的企業<ref>{{cite web |url=http://www.iodata.jp/company/guidance/outline/index.htm |script-title=ja:会社概要・所在地 |accessdate=2017-05-15 |publisher={{lang|ja|アイ・オー・データ機器}} |language=ja}}</ref>。總部位於河北市的{{link-ja|PFU|PFU}}是[[富士通|富士通]]的全資子公司，也是世界最大的[[影像掃描器|影像掃描器]]生產企業<ref>{{cite web |url=https://www.pfu.fujitsu.com/imaging/history.html |script-title=ja:スキャナの歩み |accessdate=2017-05-15 |publisher={{lang|ja|PFU}} |language=ja}}</ref>。此外石川縣還有眾多[[村田製作所|村田製作所]]的相關企業。石川縣企業的特徵之一是有眾多活躍於[[利基市場|利基市場]]的企業。如總部位於金澤市的津田駒工業是世界最大的織機企業，石野製作所是日本最大的迴轉壽司傳送帶生產企業<ref>{{cite web |url=https://www.pref.ishikawa.lg.jp/syoko/sangyouisan/documents/05sharetop.pdf |script-title=ja:本県の主なモノづくり企業・シェアトップ企業 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

石川縣工業的另一特徵是傳統工藝產業發達。金澤市是日本最大[[金箔|金箔]]產地，佔日本國內生產量的98%以上<ref>{{cite web |url=https://www.hakuichi.co.jp/kanazawahaku/ |script-title=ja:金沢箔について |accessdate=2017-05-15 |publisher={{lang|ja|箔一}} |language=ja}}</ref>。輪島塗、九谷燒、山中塗、金澤漆器、金澤佛壇、加賀友禪、牛首紬、美川佛壇、七尾佛壇、能州紬、珠洲燒等都是石川著名的工藝品。
=== 第三產業 ===
{{link-ja|北國銀行|北國銀行}}創建於1943年，由加能合同銀行、加州銀行、能和銀行三家銀行合併誕生，是石川縣唯一的一家地方銀行<ref>{{cite web |url=http://www.hokkokubank.co.jp/company/about/outline/profile.html |script-title=ja:北國銀行って、どんな銀行？ |accessdate=2017-05-15 |publisher={{lang|ja|北國銀行}} |language=ja}}</ref>。2014年度，石川縣的商品銷售額達34,694億日元，其中批發業有22,948億日元，零售業有11,746億日元<ref name="syougyou">{{cite web |url=http://toukei.pref.ishikawa.jp/dl/3116/26s.pdf |script-title=ja:商業統計調査結果概要 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。金澤市是石川縣的商業中心，集中了石川全縣約三分之二的商品銷售額<ref name="syougyou"></ref>。

石川縣的觀光業在近年因[[北陸新幹線|北陸新幹線]]的開通而取得顯著發展。2015年，石川縣的觀光客人數達2501.8萬人，創下統計開始以來的最高記錄。在石川縣各地區中以金澤地區的觀光客最多，其次是能登地區、加賀地區、白山地區<ref name="tourism">{{cite web |url=http://toukei.pref.ishikawa.jp/dl/3229/kankoutoukeiH27.pdf |script-title=ja:平成27年　統計からみた石川県の観光 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

== 文化 ==
=== 方言 ===
[[File:The_Forth_High_School_Memorial_Museum_of_Cultural_Exchange_Ishikawa01s5s4110.jpg|left]]
石川縣的方言可以分為能登地方的{{link-ja|能登方言|能登弁}}和加賀地方的{{link-ja|加賀方言|加賀弁}}，兩者均屬於{{link-ja|北陸方言|北陸方言}}。石川縣的方言呈現出東日本和西日本方言過渡地區的特徵，重音介於{{link-ja|京阪式重音|京阪式アクセント}}和{{link-ja|東京式重音|東京式アクセント}}之間，在文法和詞彙上也同時受到了兩方的影響<ref name="Dialect">{{cite book |author={{lang|ja|真田信治}}|script-title=ja:《方言学》 |year=2011 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-51524-4|language=ja}}</ref>{{rp|58}}。白山山麓地區的{{link-ja|白峰方言|白峰弁}}頗為獨特，被認為是[[語言島|語言島]]<ref>{{cite web |url=http://dspace.lib.kanazawa-u.ac.jp/dspace/bitstream/2297/23821/1/AA12392672-2-61.pdf |script-title=ja:白山麓白峰の方言特徴と昔話に見られる方言の語法 |accessdate=2017-05-15 |publisher={{lang|ja|金沢大学}} |language=ja }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
=== 文學 ===
[[泉鏡花|泉鏡花]]、{{link-ja|德田秋聲|徳田秋声}}、[[室生犀星|室生犀星]]是石川縣三大文豪<ref name="novelist">{{cite web |url=http://www.pref.ishikawa.jp/shiko-kinbun/info/novelist-01.html |script-title=ja:三文豪 |accessdate=2017-05-15 |publisher={{lang|ja|石川近代文学館}} |language=ja}}</ref>。泉鏡花師從[[尾崎紅葉|尾崎紅葉]]，是日本浪漫主義文學的領導者，著有《夜行巡査》、《外科室》、《高野聖》等作品<ref name="novelist"></ref>。德田秋聲同樣師從尾崎紅葉，是著名自然主義作家，獲得過菊池寬獎<ref name="novelist"></ref>。室生犀星則身兼詩人和小說家，著有《愛的詩集》、《幼年時代》等作品<ref name="novelist"></ref>。
=== 飲食 ===
石川縣的飲食文化也可以分為加賀地方和能登地方兩個部分。能登地方內陸降雪較多，環境嚴峻，主要栽培小麥和豆類，主食也以麥飯和麵食為主。沿海地區水產資源豐富，飲食也以魚類、藻類為主。加賀地方自江戶時代開始就物產發達，飲食文化也十分多樣。因魚類最易入手，使得加賀地方的飲食也以魚類為主。另外由於淨土真宗影響力強大，因此[[素齋|精進料理]]傳統濃厚。加賀地方的沿海地區雖以稻米為主食，但白山山麓一帶則多以雜糧為主食，飲食文化較為簡樸<ref>{{cite web |url=http://www.maff.go.jp/j/study/syoku_vision/manual/pdf/txt11.pdf |script-title=ja:石川県 |accessdate=2017-05-15 |publisher={{lang|ja|農林水産省}} |language=ja}}</ref>。
=== 節慶活動 ===
石川縣主要的節慶活動有金澤市的{{link-ja|金澤百萬石祭|金沢百万石まつり}}、白山市的{{link-ja|寶來祭|ほうらい祭り}}、小松市的{{link-ja|旅祭|お旅まつり}}等。金澤百萬石祭開始於1952年，在每年六月舉行，是宣傳金澤市文化重要的旅遊活動之一，以百萬石遊行而著稱<ref>{{cite web |url=http://100mangoku.net/about.html |script-title=ja:金沢百万石まつりとは |accessdate=2017-05-15 |publisher={{lang|ja|金沢百万石まつり}} |language=ja}}</ref>。寶來祭在每年十月舉行，正式名稱是金剱宮秋季例大祭，現已演變為觀光活動<ref>{{cite web |url=http://www.urara-hakusanbito.com/festival/view/9 |script-title=ja:ほうらい祭り |accessdate=2017-05-15 |publisher={{lang|ja|うらら白山人}} |language=ja}}</ref>。旅祭在每年五月中旬舉行，是菟橋神社和本折日吉神社的春季祭礼，以曳山兒童歌舞伎著稱<ref>{{cite web |url=http://www.komatsuguide.jp/index.php/otabimatsuri/detail/otabimaturitoha |script-title=ja:お旅まつりとは |accessdate=2017-05-15 |publisher={{lang|ja|こまつ観光物産ネットワーク}} |language=ja}}</ref>。
=== 媒體 ===
石川縣有{{link-ja|北國新聞|北國新聞}}和{{link-ja|北陸中日新聞|北陸中日新聞}}兩份地方報紙。北國新聞創刊於1893年，是石川縣內市場佔有率最高的報紙<ref>{{cite web |url=https://sec.hokkoku.co.jp/saiyou/info.html |script-title=ja:北國新聞社とは |accessdate=2017-05-15 |publisher={{lang|ja|北國新聞}} |language=ja}}</ref>。{{link-ja|北陸中日新聞|北陸中日新聞}}是[[中日新聞|中日新聞]]的北陸版，發行範圍包括石川、富山兩縣<ref>{{cite web |url=http://www.chunichi.co.jp/info/enkaku.html |script-title=ja:社史・沿革 |accessdate=2017-05-15 |publisher={{lang|ja|中日新聞}} |language=ja}}</ref>。石川縣內的電視台有[[NHK金澤放送局|NHK金澤放送局]]、[[北陸放送|北陸放送]]<ref>{{cite web |url=http://www.mro.co.jp/corporate/profile/ |script-title=ja:会社概要 |accessdate=2017-05-15 |publisher={{lang|ja|北陸放送}} |language=ja}}</ref>、[[石川電視台|石川電視台]]<ref>{{cite web |url=http://www.ishikawa-tv.com/outline/ |script-title=ja:会社情報 |accessdate=2017-05-15 |publisher={{lang|ja|石川テレビ}} |language=ja}}</ref>、[[金澤電視台|金澤電視台]]<ref>{{cite web |url=http://www.tvkanazawa.co.jp/company/index.html |script-title=ja:会社情報 |accessdate=2017-05-15 |publisher={{lang|ja|テレビ金沢}} |language=ja}}</ref>、[[北陸朝日放送|北陸朝日放送]]<ref>{{cite web |url=http://www.hab.co.jp/company/about/ |script-title=ja:会社案内 |accessdate=2017-05-15 |publisher={{lang|ja|北陸朝日放送}} |language=ja}}</ref>。其中NHK金澤放送局和北陸放送也有製作廣播節目。
=== 體育 ===
石川縣主要的職業體育俱樂部有參加[[棒球挑戰聯盟|棒球挑戰聯盟]]的{{link-ja|石川百萬星|石川ミリオンスターズ}}和[[日本職業足球乙級聯賽|日本職業足球乙級聯賽]]的[[金澤薩維根|金澤薩維根]]。石川百萬星創建於2007年，主要使用{{link-ja|石川縣立棒球場|石川県立野球場}}、{{link-ja|金澤市民棒球場|金沢市民野球場}}作為主場<ref>{{cite web |url=http://www.m-stars.jp/team.html |script-title=ja:チーム情報 |accessdate=2017-05-15 |publisher={{lang|ja|石川ミリオンスターズ}} |language=ja}}</ref>。金澤薩維根的前身是創建於1956年的金澤之前俱樂部，在2006年改為現名，並在2014年加入[[J聯賽|J聯賽]]<ref>{{cite web |url=http://www.zweigen-kanazawa.jp/htm/clubteam.html |script-title=ja:クラブ紹介 |accessdate=2017-05-15 |publisher={{lang|ja|ツエーゲン金沢}} |language=ja}}</ref>，以石川縣西部緑地公園田徑場為主場球場。
=== 教育 ===
石川縣有[[金澤大學|金澤大學]]和[[北陸先端科學技術大學院大學|北陸先端科學技術大學院大學]]兩所國立大學。金澤大學成立於1949年，由舊制金澤醫科大學、舊制第四高等學校、金澤高等師範學校、金責高等工業學校合併誕生，是舊六醫科大學之一，也是本州日本海沿岸地區代表性的綜合性大學<ref>{{cite web |url=http://www.kanazawa-u.ac.jp/university/history_song |script-title=ja:沿革・校歌 |accessdate=2017-05-15 |publisher={{lang|ja|金沢大学}} |language=ja}}</ref>。北陸先端科學技術大學院大學成立於1990年，是一所只提供大學院教育的研究型大學<ref>{{cite web |url=https://www.jaist.ac.jp/about/history/ |script-title=ja:沿革 |accessdate=2017-05-15 |publisher={{lang|ja|北陸先端科学技術大学院大学}} |language=ja}}</ref>。石川縣還有石川縣立看護大學、[[石川縣立大學|石川縣立大學]]、金澤美術工藝大學三所公立大學，金澤醫科大學、金澤學院大學、金澤工業大學、金澤星稜大學、金城大學、北陸大學、北陸學院大學七所私立大學<ref>{{cite web |url=http://www.gakkou.net/daigaku/src/?srcmode=pref&p=17 |script-title=ja:石川県 |accessdate=2017-05-15 |publisher={{lang|ja|ナレッジステーション}} |language=ja}}</ref>。

==交通==
[[File:Kanazawa_eki.jpg|thumb]]
2015年3月14日[[北陸新幹線|北陸新幹線]][[長野站|長野站]]至[[金澤站|金澤站]]開業之後<ref>{{cite web |url=http://www.pref.ishikawa.jp/shink/hokuriku-shinkansen/about.html |script-title=ja:北陸新幹線って？ |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>，石川縣內鐵路交通發生很大變化。[[北陸本線|北陸本線]]金澤以西的部分仍由[[西日本旅客鐵道|JR西日本]]運營，金澤以東的部分則改由[[IR石川鐵道|IR石川鐵道]]運營<ref>{{cite web |url=http://www.ishikawa-railway.jp/route/ |script-title=ja:路線・駅 |accessdate=2017-05-15 |publisher={{lang|ja|IRいしかわ鉄道}} |language=ja}}</ref>。北陸本線不僅是石川縣南部各城市之間的主要交通手段，也有眾多自石川縣外駛來的[[特別急行列車|特急列車]]和貨物列車，是石川縣鐵路交通的幹線。[[七尾線|七尾線]]同樣由JR西日本運營，是連接加賀地方北部和能登地方的重要交通路線。此外石川縣內還有[[能登鐵道|能登鐵道]]運營的{{link-ja|能登鐵道七尾線|のと鉄道七尾線}}<ref>{{cite web |url=http://www.nototetsu.co.jp/html/ensen.html |script-title=ja:沿線ガイド |accessdate=2017-05-15 |publisher={{lang|ja|のと鉄道}} |language=ja}}</ref>，{{link-ja|北陸鐵道|北陸鉄道}}運營的{{link-ja|淺野川線|北陸鉄道浅野川線}}、{{link-ja|石川線|北陸鉄道石川線}}三條鐵路路線<ref>{{cite web |url=http://www.hokutetsu.co.jp/tetsudo/rw-rosen.pdf |script-title=ja:鉄道路線図 |accessdate=2017-05-15 |publisher={{lang|ja|北陸鉄道}} |language=ja}}</ref>。

[[File:Sotokanjo-daimonzan.jpg|thumb]]
石川縣內主要的幹線公路有沿加賀地方海岸線的[[北陸自動車道|北陸自動車道]]和[[國道8號_(日本)|國道8號]]，以及連接加賀地方北部和能登地方的[[能登里山海道|能登里山海道]]及{{link-ja|國道159號|国道159号}}<ref>{{cite web |url=http://www.jaf.or.jp/member/dguide/exway/dm_highway/pageindices/index4.html |script-title=ja:高速道路案内図 |accessdate=2017-05-15 |publisher={{lang|ja|日本自動車連盟}} |language=ja}}</ref>。金澤市在郊區修建了{{link-ja|金澤外環狀公路|金沢外環状道路}}以緩解流量<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kenouciv/jigyou/douro1.html|script-title=ja:道路事業（外環状道路） |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。加賀地方和能登地方的交通節點津幡町亦是交通堵塞多發區，政府也在這裡增建公路以緩解堵塞問題<ref>{{cite web |url=http://www.hrr.mlit.go.jp/kanazawa/mb5_kouhou/press/h19/p0207.pdf|script-title=ja:一般国道８号 津幡北バイパス全線開通！ |accessdate=2017-05-15 |publisher={{lang|ja|国土交通省}} |language=ja}}</ref><ref>{{cite web |url=http://www.hrr.mlit.go.jp/kanazawa/mb5_kouhou/press/h20/p0331_01.pdf|script-title=ja:国道１５９号津幡バイパスの事業推進 |accessdate=2017-05-15 |publisher={{lang|ja|国土交通省}} |language=ja}}</ref>。

石川縣內的高速巴士多以金澤站為起點，出發前往名古屋、關西、關東、東北等地。中短途巴士則各地區有不同公司運行。金澤市近郊由北陸鐵道、北鐵金澤巴士、西日本JR巴士運行；小松地區由小松巴士運行；白山地區由加賀白山巴士運行；加賀地區由加賀溫泉巴士運行；中能登地區由北鐵能登巴士運行；能登島地區由能登島交通運行；奧能登地區由北鐵奧能登地區巴士運行<ref>{{cite web |url=http://www.ishikawabus.jp/use/rosen.html|script-title=ja:路線バスの利用 |accessdate=2017-05-15 |publisher={{lang|ja|石川県バス協会}} |language=ja}}</ref>。石川縣導入了{{link-ja|當地車牌號|ご当地ナンバー}}制度，縣內有「石川」和「金澤」兩種車牌號<ref>{{cite web |url=http://wwwtb.mlit.go.jp/hokushin/ishikawa/kanazawa_num.html |script-title=ja:金沢ナンバーが登場 |accessdate=2017-05-15 |publisher={{lang|ja|北陸信越運輸局石川運輸支局}} |language=ja}}</ref>。

石川縣內有{{link-ja|金澤港|金沢港}}和{{link-ja|七尾港|七尾港}}兩座重要港灣，此外還有10座地方港灣。金澤港於1930年代開始開發，並在1970年正式開港，現在有前往中國及韓國的定期貨櫃航線<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/kanazawakouwan/p01a.html |script-title=ja:施設紹介 - 金沢港の沿革・組織 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。七尾港是一座天然良港，其通商歷史可以追溯至室町時代，是石川縣最大的貨物港<ref>{{cite web |url=http://www.pref.ishikawa.lg.jp/nanaokouwan/minatonogaiyou.html |script-title=ja:七尾港の概要 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

石川縣有[[小松飛行場|小松機場]]和[[能登機場|能登機場]]兩個機場。小松機場的跑道長度達3000米，可起降大型噴氣機。現在小松機場有前往[[東京國際機場|羽田]]、[[成田國際機場|成田]]、[[新千歲機場|札幌]]、[[仙台機場|仙台]]、[[福岡機場|福岡]]、[[那霸機場|那霸]]、[[仁川國際機場|首爾]]、[[浦東國際機場|上海]]、[[桃園國際機場|台北]]的定期航線<ref>{{cite web |url=http://www.komatsuairport.jp/flight/ |script-title=ja:フライト情報 |accessdate=2017-05-15 |publisher={{lang|ja|小松空港}} |language=ja}}</ref>。2014年度，小松機場的乘客數超過212萬人<ref>{{cite web |url=http://toukei.pref.ishikawa.jp/dl/2997/k_air_h26.pdf |script-title=ja:平成26年度小松空港利用状況 |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。能登機場則運行有前往東京的定期航線<ref>{{cite web |url=https://www.noto-airport.jp/flight/ |script-title=ja:フライト情報 |accessdate=2017-05-15 |publisher={{lang|ja|のと里山空港}} |language=ja}}</ref>。在開始運營12年後，能登和東京之間的航線乘客數已達15萬人<ref>{{cite web |url=http://toukei.pref.ishikawa.jp/dl/2999/noto12th.pdf |script-title=ja:能登－羽田便利用状況（１２年目） |accessdate=2017-05-15 |publisher={{lang|ja|石川県}} |language=ja}}</ref>。

==觀光==
[[File:Koiji_Coast.jpg|thumb]]
石川縣無論是人文觀光資源還是自然觀光資源均十分豐富。金澤在江戶時代是三都（[[江戶|江戶]]、[[京都|京都]]、大坂）之外人口最多的城市，擁有濃厚的城下町文化底蘊。[[金澤城|金澤城]]是有「加賀百萬石」之稱的前田家的居城，被日本政府指定為史跡。城內的石川門等建築保留有江戶時代原貌，是[[重要文化財|重要文化財]]。2001年以來，金澤市完成了菱櫓、橋爪門、河北門、玉泉院丸庭園等重要建築的重建工作，使得金澤城更接近江戶時代的原貌<ref>{{cite web |url=http://www.pref.ishikawa.jp/siro-niwa/kanazawajou/facility/index.html |script-title=ja:菱櫓・五十間長屋・橋爪門続櫓 |accessdate=2017-05-15 |publisher={{lang|ja|金沢城公園}} |language=ja}}</ref>。金澤城附近的[[兼六園|兼六園]]是日本三名園之一，是加賀歷代藩主的大名庭園，被日本政府認定為特別名勝，也是日本最著名的廻遊式庭園之一<ref>{{cite web |url=http://www.pref.ishikawa.jp/siro-niwa/kenrokuen/about.html |script-title=ja:兼六園とは |accessdate=2017-05-15 |publisher={{lang|ja|兼六園}} |language=ja}}</ref>。金澤市的[[東山東|東山東]]、[[主計町|主計町]]等地因擁有眾多江戶時代後期至明治時期的傳統建築而保留有城下町的獨特氣氛，被列入[[重要傳統的建造物群保存地區|重要傳統的建造物群保存地區]]<ref>{{cite web |url=http://www4.city.kanazawa.lg.jp/11107/rekisimatizukuri/jyuudenkentiku2.html |script-title=ja:伝統的建造物群保存地区 |accessdate=2017-05-15 |publisher={{lang|ja|金沢市}} |language=ja}}</ref>。[[金澤21世紀美術館|金澤21世紀美術館]]是日本最具特色的現代藝術的美術館之一，建築本身也獲得威尼斯金獅獎肯定<ref>{{cite web |url=http://www.kanazawa21.jp/data_list.php?g=11&d=1 |script-title=ja:美術館のコンセプトについて |accessdate=2017-05-15 |publisher={{lang|ja|金沢21世紀美術館}} |language=ja}}</ref>。

石川縣內溫泉度假地眾多，最為著名的有以「加賀四湯」（粟津溫泉、片山津溫泉、山代溫泉、山中溫泉）而著稱，位於加賀市、小松市的{{link-ja|加賀溫泉鄉|加賀温泉郷}}<ref>{{cite web |url=http://www.kaga410.com/about/ |script-title=ja:加賀四湯とは |accessdate=2017-05-15 |publisher={{lang|ja|加賀四湯どっとこむ}} |language=ja}}</ref>；日本前列的高級溫泉度假區[[和倉溫泉|和倉溫泉]]<ref>{{cite web |url=http://www.wakura.or.jp/about/ |script-title=ja:和倉温泉について |accessdate=2017-05-15 |publisher={{lang|ja|和倉温泉観光協会・和倉温泉旅館協同組合}} |language=ja}}</ref>等地。能登半島的沿海地區地形獨特，擁有{{link-ja|能登金剛|能登金剛}}、[[見附島_(石川縣)|見附島]]、{{link-ja|戀路海岸|恋路海岸}}、{{link-ja|哥吉拉岩|ゴジラ岩 (石川県)}}等景點<ref>{{cite web |url=http://www.hot-ishikawa.jp/sys/data?page-id=6133 |script-title=ja:能登半島国定公園 |accessdate=2017-05-15 |publisher={{lang|ja|ほっと石川旅ネット}} |language=ja}}</ref>。石川縣東南部的[[白山_(日本)|白山]]擁有[[白山比咩神社|白山比咩神社]]等神社，和[[立山_(日本)|立山]]、[[富士山|富士山]]並列為日本三名山之一，是宗教信仰上的聖地<ref>{{cite web |url=http://www.shirayama.or.jp/hakusan/ |script-title=ja:白山信仰と白山比咩神社 |accessdate=2017-05-15 |publisher={{lang|ja|白山比咩神社}} |language=ja}}</ref>。白山亦因保存有良好的生態環境而被列入國立公園加以保護，登山活動也十分興盛<ref>{{cite web |url=http://www.bes.or.jp/invitation/np/hakusan.html |script-title=ja:白山 - 自然公園への招待 |accessdate=2017-05-15 |publisher={{lang|ja|自然公園財団}} |language=ja}}</ref>。
