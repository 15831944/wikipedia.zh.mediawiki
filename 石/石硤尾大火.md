'''石硤尾大火'''是指於1953年12月25日[[聖誕節|聖誕節]]，在[[香港|香港]][[九龍|九龍]][[石硤尾|石硤尾]][[寮屋|木屋區]]發生的[[火災|火災]]，受災面積達45畝，有50,000多名災民無家可歸。這場大火最終導致了[[香港公共房屋|香港公共房屋]]的誕生。

== 背景 ==
[[第二次國共內戰|國共內戰]]前後，大批中國大陸難民湧港，於九龍北部山邊多處搭建木屋區，一度有近25%人口居於寮屋。木屋區火警時有發生，例如1951年11月21日的[[九龍城|九龍城]][[東頭村|東頭村]]大火，故居民對火警逃生多有準備。當時社會亦不多聖誕節活動。

== 事件經過 ==
1953年12月25日晚上9時25分，[[白田村|白田村]]眾安道124號一間木屋的2樓一單位，有一名製鞋住戶在燃點[[火水燈|火水燈]]時，火種不慎燒著[[被子|棉胎]]及製鞋[[膠水|膠水]]，引起火警。火警發生後最初時火勢並不猛烈，僅波及旁邊30多間房屋。過了很久，一陣強烈的北風令火勢迅速四處蔓延，不足10分鐘，大火已波及數百戶。晚上10時，火頭由白田村向東灣及正街蔓延，火勢一發不可收拾。

火警發生時，當時的消防局長哥民以災情嚴重為理由，立刻下令香港區兩架滅火車渡海，並召集所有後備消防人員前往災場協助灌救。雖然當時有很多消防員到場救火，但是到了晚上11時10分，整個白田村已被焚毀，不論是木造或石造的房屋，都一律化為灰燼。火場四周一片頹垣敗瓦，但大火餘勢未了，直至12月26日凌晨2時30分火勢才受到控制。

當時的[[香港總督|香港總督]][[葛量洪|葛量洪]]向[[英國|英國]]政府報告，指這場[[石硤尾|石硤尾]]大火，涉及多個木屋區，包括白田上村、白田下村、石硤尾村、窩仔上村、窩仔下村及大埔村，災場廣及41英畝，即相當於164,000平方公尺。大火歷時6個小時才受控，造成3死51傷，燒毀木屋2,580房，約12,000多個家庭合共58,203人頓成災民<ref>{{zh-hant}}{{cite news | title=災民登記辦理完竣 |publisher=[[工商日報|工商日報]]第五頁 |date=1953年12月31日 }}</ref>，無家可歸。

== 籌款賑災 ==
1月17日，[[太極拳|太極拳]]手[[吳公儀|吳公儀]]與[[白鶴拳|白鶴拳]]手[[陳克夫|陳克夫]]在澳門新花園泳池比武，[https://www.youtube.com/watch?v=2FsZyPjsjTA 吳陳比武1954 - YouTube]，兩回合後裁判[[何賢|何賢]]宣判「不勝、不和、不負」，籌得27萬港元善款<ref>{{Cite web |url=https://mmis.hkpl.gov.hk/shek-kip-mei-fire |title=石硤尾大火 - 多媒體資訊系統 |accessdate=2017-01-23 |work= |publisher=香港公共圖書館 |date= }}</ref><ref>{{Cite web |url=https://www.e123.hk/ElderlyPro/details/361061/50/ |title=無障礙好去處 集體回憶美荷樓 |accessdate=2017-01-23 |work= |publisher=長青網 |date=2014年11月7日 }}</ref>。

== 影響：公共房屋 ==
大火之後，廣大的木屋區變為廢墟，災民流離失所，[[香港殖民地時期#香港政府|香港政府]]為安置災民，於災場附近興建兩層高的[[平房|平房]]。平房以當時的[[工務局|工務局]]局長包寧命名，稱為「包寧平房」<ref>[http://www.lcsd.gov.hk/b5/ppr_release_det.php?pd=20040601&ps=02 文化博物館展覽細訴香港公共房屋的故事] {{webarchive|url=https://web.archive.org/web/20041209014052/http://www.lcsd.gov.hk/b5/ppr_release_det.php?pd=20040601&ps=02 |date=2004-12-09 }}</ref>。之後在火災原址再興建29棟六至七層高H型的[[徙置大廈|徙置大廈]]，供災民入住，就是石硤尾徙置區（今[[石硤尾邨|石硤尾邨]]），其中第9-12座是由[[聯合國|聯合國]]捐款建成。自此之後，政府開始興建[[香港公共屋邨|公共房屋]]，為低下層市民提供住屋[[福利|福利]]，而現時負責興建公共房屋的[[香港房屋委員會|香港房屋委員會]]前身[[屋宇建設委員會|屋宇建設委員會]]亦在這個背景下成立。
