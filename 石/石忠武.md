'''石忠武'''（{{bd|1958年|||}}）籍贯不详，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
石忠武曾任原[[中国人民解放军副总参谋长|中国人民解放军副总参谋长]][[吴铨叙|吴铨叙]]上将的秘书。后来任[[中国人民解放军总参谋部军训和兵种部|中国人民解放军总参谋部军训和兵种部]]合同战术训练局局长，任内曾担任“确山—2006”演习副总导演，因频繁给参演官兵“使绊”、出难题，被参演官兵们送了个绰号“石黑手”<ref name=ifeng>{{Cite web |url =  http://news.ifeng.com/a/20171009/52516824_0.shtml|title = 石忠武少将任国防大学副校长 曾被称“石黑手” |author =  |date =2017-10-09  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref name=caixin>{{Cite web |url = http://china.caixin.com/2017-10-09/101143539.html |title =曾被称“石黑手” 石忠武少将任国防大学副校长  |author =  |date = 2017-10-09 |publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref>。

2009年5月，石忠武任[[中国人民解放军工程兵指挥学院|中国人民解放军工程兵指挥学院]]院长。2011年7月，任[[中国人民解放军石家庄陆军指挥学院|中国人民解放军石家庄陆军指挥学院]]院长，任内担任“联教—2012·确山”联合演练总导演。2016年5月，石忠武佩戴陆军臂章亮相[[中国中央电视台|中国中央电视台]]《军事报道》栏目，显示此前隶属[[中国人民解放军总参谋部|中国人民解放军总参谋部]]的石家庄陆军指挥学院已转隶[[中国人民解放军陆军|中国人民解放军陆军]]。2016年8月，石忠武在《[[解放军报|解放军报]]》发表文章《贴近实战关键要贴近对手》。2017年，石忠武任新调整组建的[[中国人民解放军国防大学|中国人民解放军国防大学]]副校长<ref name=ifeng/><ref name=caixin/>。

2010年7月晋升少将军衔<ref name=caixin/>。
