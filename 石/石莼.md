'''石莼'''（[[学名|学名]]：''{{lang|la|Ulva lactuca}}''），又名'''海莴苣'''（{{lang|en|sea lettuce}}），为[[石莼属|石莼属]]下的一种[[绿藻|绿藻]]。  

==描述==
石莼是一种薄平、长有固着器的绿藻植物。其边缘有时为褶边开裂。其叶通常长至18厘米或更长，横截面长至30厘米<ref>{{cite web
 | url  = http://www3.gettysburg.edu/~rcavalie/bda_i/078.html 
 | work = Gettysburg College
 | title= Ulva lactuca
 | accessdate=2007-12-28
}}</ref>。其为有两个细胞厚度的柔软、半透明的叶状体<ref>{{cite web
 |url         = http://omp.gso.uri.edu/doee/biota/algae/chloro/ulvaxf.htm
 |work        = University of Rhode Island
 |title       = Ulva lactuca
 |accessdate  = 2007-12-28
 |deadurl     = yes
 |archiveurl  = https://web.archive.org/web/20080210015236/http://omp.gso.uri.edu/doee/biota/algae/chloro/ulvaxf.htm
 |archivedate = 2008-02-10
}}</ref>。没有[[叶柄|叶柄]]，通常依靠圆盘状固着器固定<ref name="Burrows 91">Burrows, E.M. (1991) ''Seaweeds of the British Isles. Volume 2 Chlorophyta.'' Natural History Museum, London. ISBN 0-565-00981-8</ref>。颜色呈绿色至深绿色，[[叶序|叶序]]为不规则排列。其[[叶绿体|叶绿体]]为1-3个蛋白核呈杯状排列。其他石莼属的植物也有相似的特征。

==分布==
石莼的分布非常广泛：包括[[欧洲|欧洲]]、[[北美洲|北美洲]]（东、西海岸）、[[中美洲|中美洲]]、[[加勒比|加勒比]]、[[南美洲|南美洲]]、[[非洲|非洲]]、印度洋群岛、东南亚、[[中国|中国]]、太平洋群岛、[[澳大利亚|澳大利亚]]和[[新西兰|新西兰]]等<ref>{{cite web
