'''石水站'''（{{kor|諺=석수역|漢=石水驛|羅=Seoksu yeok|f=y}}）是一個位於[[京畿道|京畿道]][[安養市|安養市]]{{link-ko|萬安區|만안구}}{{link-ko|石水洞|석수동}}，屬於[[京釜線|京釜線]]的[[電鐵站|電鐵站]]。車站鄰近[[首爾特別市|首爾特別市]][[衿川區|衿川區]]始興洞，但車站官方位置是[[京畿道|京畿道]][[安養市|安養市]]，因此不可使用首爾市內的{{link-ko|首都圈電鐵定期券|수도권 전철 정기권}}。
== 車站結構 ==
2面4線雙[[島式月台|島式月台]]的[[地面車站|地面車站]]。當中有2條是通過線。出口共2個。現在設有[[月台幕門|月台幕門]]。

== 历史 ==
* 1982年8月2日：以配置簡易站開始營業<ref>철도청고시 제46호, 제47호, 1982년 7월 28일.</ref>。
* 1992年3月2日：改為運轉簡易站<ref>철도청고시 제1992-4호, 1992년 2월 17일.</ref><ref>但是，1992年鐵道統計年報主要運輸營業日誌記載為2月12日。</ref>。
* 1998年1月1日：降格為無配置簡易站<ref>[http://theme.archives.go.kr/viewer/common/archWebViewer.do?singleData=N&archiveEventId=0028409627#172 철도청고시 제1997-70호], 1997년 12월 30일.</ref>。
