'''石城县'''在中国[[江西省|江西省]]东南部，是[[赣州市|赣州市]]下辖的一个县。

== 历史沿革 ==
早在[[新石器时期|新石器时期]]，石城境内就有人类活动。

[[三国|三国]]·[[吴|吴]][[嘉禾|嘉禾]]五年（公元236年），析[[雩都|雩都]]地置'''揭阳县'''，为石城建县之始，先后属[[庐陵郡|庐陵郡]]、[[南康郡|南康郡]]。[[晋|晋]][[太康|太康]]五年（公元284年），改名'''陂阳县'''。

[[隋|隋]][[开皇|开皇]]十三年（公元593年）设'''石城场'''，以境内“四面环山，耸峙如城”得名。

[[南唐|南唐]][[保大|保大]]十一年（公元953年），升石城场为'''石城县'''，并沿用至今。<ref>{{cite web|url=http://www.shicheng.gov.cn/scgk/lsyg/200810/t20081017_5248.htm|title=历史沿革|accessdate=2009-12-14|author=|publisher=石城县人民政府网站|language=zh-hans}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 地理 ==
石城县位于江西省赣州市东北部，东邻福建宁化县，南抵[[瑞金市|瑞金市]]及福建[[长汀县|长汀县]]，西毗[[宁都县|宁都县]]，北靠[[广昌县|广昌县]]。

南北长71.8公里，东西宽53.7公里。全县总面积1581.53平方公里，其中山地占89％，耕地占10％，水面占3％，是典型的东南丘陵低山地区。<ref>{{cite web|url=http://www.shicheng.gov.cn/scgk/scgk_dlwz/200810/t20081018_5918.htm|title=地理位置|accessdate=2009-12-14|author=|publisher=石城县人民政府网站|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20111231003442/http://www.shicheng.gov.cn/scgk/scgk_dlwz/200810/t20081018_5918.htm|archivedate=2011-12-31}}</ref>

== 经济 ==
2008年全县实现生产总值177894万元，同比增长11.0%。<ref>{{cite web |url=http://www.shicheng.gov.cn/scgk/scgk_zhsl/200903/t20090323_11879.htm |title=石城县2008年经济运行分析及2009年展望 |accessdate=2009-12-14 |author= |publisher=石城县人民政府网站 |language=zh-hans }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 人口 ==
2006年末总户数80872户，总人口302545人，其中男性158519人，女性144078人；60岁及以上老年人口占10.3%。<ref>{{cite web|url=http://www.shicheng.gov.cn/scgk/scgk_rkjy/200811/t20081108_7017.htm|title=人口就业|accessdate=2009-12-14|author=|publisher=石城县人民政府网站|language=zh-hans}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

石城是纯客家县，使用[[客家语|客家语]][[石城话|石城话]]。<ref>{{cite web|url=http://www.shicheng.gov.cn/scgk/scgk_kjfq/200811/t20081127_7963.htm|title=石城客家风情|accessdate=2009-12-14|author=|publisher=石城县人民政府网站|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20111231004212/http://www.shicheng.gov.cn/scgk/scgk_kjfq/200811/t20081127_7963.htm|archivedate=2011-12-31}}</ref>

== 文化教育 ==
石城东临[[福建省|福建省]][[宁化县|宁化]]的“客家祖地”[[石壁村|石壁]]，是客家语言、文化形成的重要地点之一。
