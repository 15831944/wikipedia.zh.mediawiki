'''石錫勳'''（{{bd|1900年|8月2日|1985年|9月22日}}），台灣[[醫師|醫師]]、政治人物。[[彰化市|彰化市]]人。曾出任[[彰化市|彰化市]]首任官派市長。[[台灣日治時期|日治時期]]曾加入[[台灣文化協會|台灣文化協會]]從事反對運動，戰後反對[[國民黨|國民黨]]一黨獨大，多次參與選舉，屢敗屢戰。

== 早年生平 ==
石錫勳出生於彰化街北門外（今彰化市），父親石汝鏘是著名中醫師。[[大正|大正]]六年（1917年）考入[[臺灣總督府醫學校|臺灣總督府醫學校]]就讀。

石錫勳曾表示在醫專時期，出現臺灣學生與日本學生的對立，然而校方往往坦護日本學生。{{r|臺|page=285}}石錫勳與醫專同班同學[[吳海水|吳海水]]、丁瑞魚、張梗、甘文芳等人則時常出面打抱不平，遂形成領導集團。而已經在[[大稻埕|大稻埕]]大安醫院開業的學長[[蔣渭水|蔣渭水]]時常來找他們聊天，並開始交換意見討論。{{r|臺|page=285}}大正十一年（1922年）4月畢業。<ref>[http://ds3.th.gov.tw/ds3/app007/list3.php?ID1=0071022655a005 卒業證書授與（醫學專門學校）]，臺灣總督府府報資料庫查詢。</ref>

大正十年（1921年）[[蔣渭水|蔣渭水]]成立[[臺灣文化協會|臺灣文化協會]]時，當時就讀醫科五年級的學弟石錫勳，就成為蔣渭水的吸收對象。{{r|自|page=139}}最後石錫勳參加文化協會的創立，成為創會理事。畢業後石錫勳到[[高雄|高雄]][[旗津|旗津]]開業。

大正十二年（1923年）「臺灣議會設置請願期成同盟會」組織時，石錫勳曾擔任[[高雄州|高雄州]]總幹事。{{r|自|page=139}}在2月21日成立典禮上被選為專務理事。{{r|臺|page=205}}同年底因[[治警事件|治警事件]]被捕。一審宣判無罪，二審被判罰金百圓。大正十四年（1925年）2月20日三審定讞，維持二審宣判。石錫勳成為治警事件被判罪者最年輕的一位。

== 戰後 ==
日本投降後。1945年，石錫勳出任彰化地區「歡迎國民政府委員會」主席，並備委員會推選為[[彰化市|彰化市]]長。發起成立[[國民黨|國民黨]]彰化市黨部及[[三民主義青年團|三民主義青年團]]。然而據[[楊克煌|楊克煌]]回憶，石錫勳上任後一個多星期，就遭國民黨政府派人取代。國民黨統治臺灣後一直特意疏離具社會基礎、受民間尊敬的反殖民運動精英。石錫勳只是其中一例。{{r|自|page=139}}石錫勳後來於1946年當選彰化市參議員，然而最後因看清國民黨而退黨。<ref>[http://www.peoplenews.tw/news/55e5b9c5-273f-46a5-b07b-527f1ecef927 磺溪傲骨石錫勳 從反日一路到反國民黨]，[[陳婉真|陳婉真]]，2016年5月11日。</ref>

1949年，石錫勳一度遭[[警備總部|警備總部]]以與[[廖文毅|廖文毅]]聯絡為由，將其逮捕、拘禁了三個月。<ref>[http://www.peoplenews.tw/news/55e5b9c5-273f-46a5-b07b-527f1ecef927 磺溪傲骨石錫勳 從反日一路到反國民黨]，[[陳婉真|陳婉真]]，2016年5月11日。</ref>{{r|自|page=139}}

1954年、1957年、1960年石錫勳曾多次參選[[彰化縣縣長|彰化縣縣長]]。然而均因不敵國民黨的買票和作票而落選。<ref>[http://www.peoplenews.tw/news/55e5b9c5-273f-46a5-b07b-527f1ecef927 磺溪傲骨石錫勳 從反日一路到反國民黨]，[[陳婉真|陳婉真]]，2016年5月11日。</ref>在[[雷震|雷震]]籌備組新黨時，石錫勳曾參與組黨，成為彰化地區唯一參與者。<ref>[http://www.peoplenews.tw/news/b57ea54a-8f5b-404c-8dd4-7ebf2e587866 【專文】文協鬥士+國民黨政治犯 石錫勳的子女談老爸]，[[陳婉真|陳婉真]]，2016年11月25日。</ref>被推為十七名召集委員之一。

1968年，石錫勳獲[[青年黨|青年黨]]提名為彰化縣長候選人，然而隨即遭到逮捕，判刑7年。並罹患心臟病，高血壓、氣喘等多種疾病。其夫人曾四處請託，甚至去找文協時期和他對立的[[蔡培火|蔡培火]]，一度被蔡培火奚落說：「你以為我不知道他們以前都罵我『臭培火』。」不過蔡培火最後仍基於[[基督徒|基督徒]]的立場，還是幫他爭取到保外就醫。<ref>[http://www.peoplenews.tw/news/b57ea54a-8f5b-404c-8dd4-7ebf2e587866 【專文】文協鬥士+國民黨政治犯 石錫勳的子女談老爸]，[[陳婉真|陳婉真]]，2016年11月25日。</ref><ref>[http://www.laijohn.com/archives/pc/Chio/Chio,Shun/wife/life/Chio,Bsa.htm 母親的一生與她的信仰力量]，石美莎撰。</ref>

保外就醫後，石錫勳從此遠離故鄉，於1985年病逝於高雄。

== 家庭 ==
* 石錫勳二哥石錫烈曾任幸生醫院院長、庄協議會員。五弟[[石錫純|石錫純]]曾任南亞製粉會社專務，日治時期與[[鹽分地帶|鹽分地帶]]文學家有多次往來。
* 石錫勳夫人張慶美畢業於[[長榮女中|長榮女中]]，曾就讀於東京帝國女子醫專。回台後曾任台中醫院醫師。<ref>[http://www.laijohn.com/archives/pc/Chio/Chio,Shun/wife/life/Chio,Bsa.htm 母親的一生與她的信仰力量]，石美莎撰。</ref>
