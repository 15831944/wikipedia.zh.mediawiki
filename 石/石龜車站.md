'''石龜車站'''位於[[台灣|台灣]][[雲林縣|雲林縣]][[斗南鎮|斗南鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]的[[鐵路車站|鐵路車站]]，在[[中華民國省道|省道]][[台1線|台1線]]旁邊。須經由農間小路進去。

== 車站構造 ==
*[[岸式月台|岸式月台]]兩座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，僅有[[臺鐵區間車|區間車]]停靠。
**指定由斗南站管理。
*本站是縱貫線南段停靠車次及使用人次最少的車站。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1958年10月24日：設立「278K號誌站」。
*1971年7月20日：改站名為「石龜站」並升級成簡易站，指定由[[斗南車站|斗南站]]管理。
*1977年5月1日：由簡易站降級為招呼站。
*2012年10月25日：一名身份不明的男子在南下第171次自強號列車經過時突然自月台躍下，遭列車撞擊死亡。{{R|NowNews20121026}}。
*2013年9月30日：啟用多卡通刷卡機。
