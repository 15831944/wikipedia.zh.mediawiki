'''石榴車站'''位於[[台灣|台灣]][[雲林縣|雲林縣]][[斗六市|斗六市]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*木造車站。主要由事務室（辦公室）、信號室（行車運轉室）、宿值室（休息室）、廊下（走廊）及儲藏室等空間組成。
*[[岸式月台|岸式月台]]兩座。
*本站山側月台原為[[島式月台|島式月台]]，後來降為招呼站之後，站內之側線已全部拆除，僅留下兩條通過線。
*已翻新整修。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，僅有[[臺鐵區間車|區間車]]停靠。
*由斗六站管理。
*本站是縱貫線南段停靠車次及使用人次第二少的車站，僅次於[[石龜車站|石龜車站]]。

== 歷史 ==
*1905年10月15日：設「石榴班驛」<ref name=":0">{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>，石榴非植物[[石榴|石榴]]，而是鐵道上的石碴。
*1908年9月19日：因業務冷清，停止營業。
*1941年：復設「石榴班信號場」。
*1945年：後改稱「石榴班號誌站」。
*1950年6月1日：由「石榴班號誌站」改名「石榴班車站」為三等站。
*1955年3月1日：更名為「石榴站」。<ref name=":0" />
*1990年8月15日：由甲種簡易站降為招呼站，由[[斗六車站|斗六站]]管理。<ref name=":0" />
*2006年12月22日：石榴車站站房和周圍的職員宿舍以「石榴站車站及職員宿舍」之名登錄為歷史建築<ref>[http://www.boch.gov.tw/boch/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&caseId=IA09603000001&version=1&assetsClassifyId=1.2&menuId=310&siteId=101 文化部文化資產局－石榴站車站及職員宿舍] {{webarchive|url=https://web.archive.org/web/20160304140655/http://www.boch.gov.tw/boch/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&caseId=IA09603000001&version=1&assetsClassifyId=1.2&menuId=310&siteId=101 |date=2016-03-04 }}</ref><ref>許素惠，[http://www.chinatimes.com/newspapers/20161102000551-260107  最美木造驛站 苦伕見證演進史]，中國時報，2016年11月02日</ref>。
*2012年12月：站房整修完工<ref>[http://n.yam.com/greatnews/politics/20130701/20130701587777.html 立委劉建國爭取石榴車站宿舍再利用工程費180萬元]</ref>。
*2013年9月30日：啟用多卡通刷卡機。
