'''石門站'''（{{lang-en|'''Shek Mun Station'''}}）是[[港鐵|港鐵]][[馬鞍山綫|馬鞍山綫]]的一個架空車站，位於[[沙田區|沙田區]][[石門_(香港)|石門]]東南部安明街旁，並於2004年12月21日啟用。

===大堂===
{{ multiple image | direction = vertical | align = right | width = 220 
| image1 = Shek Mun Station Concourse 201412.jpg | caption1 = 大堂（2014年）
}}
石門站設有7間商店，包括[[便利店|便利店]]、[[麵包糕餅店|麵包糕餅店]]、零食專賣店等。站內亦設有不同銀行所提供的[[自動櫃員機|自動櫃員機]]<ref name="shops"/>。

===月台===
{{ multiple image | direction = vertical | align = right | width = 220
| image1 = Shek Mun Station 2017 11 part4.jpg | caption1 = 1號月台
| image2 = Shek Mun Station 2017 11 part3.jpg | caption2 = 2號月台
| image3 = Shek Mun Station 2018 06 part2.jpg | caption3 = 月台佈置
}}
車站設有兩個月台，並以1個島式月台設計排列，月台最初長4至5卡，南端位置靠近[[小瀝源渠|小瀝源渠]]有預留空間，預留空間會於沙中綫興建時擴建，可額外增至8卡列車長度，以應付未來發展，現時擴建工程已經完成。此外，車站南面設有'''石門[[側線_(鐵路)|側線]]'''，方便列車調動或信號故障時使用。

[[File:Shek_Mun_Station_2017_11_part1.jpg|thumb]]

====月台閘門====
<gallery>
Shek Mun Station platform extension.jpg|月台擴建部份及閘門工程
</gallery>
本站於2016年進行月台閘門安裝工程，首先在2號月台車尾開始，其後1號月台車頭及車尾亦展開工程，現階段2號月台已完成安裝閘門，全部月台閘門已開始進入營運狀態。

==鄰接車站==
石門站是馬鞍山綫的中途站，乘客可在此乘坐往烏溪沙或大圍列車，往返[[九龍|九龍]]及[[新界|新界]]等地。

==使用狀況==
{{original research|章節}}
{{ multiple image | direction = vertical | align = right | width = 220
| image1 = HK KCR MOS RailwayShekMun Platform.jpg | caption1 = 兩鐵合併前的月台（2004年）
}}
此站啟用初期由於距石門工業區較遠，基本上只靠往返石門工業區上班之乘客，因此令使用率很低，尤其非繁忙時間及晚上為甚。直至[[碩門邨|碩門邨]]入伙及鄰近[[香港浸會大學國際學院|香港浸會大學國際學院]]和中小學啟用後，人流才略為上升，但晚間的使用率仍然偏低。本站雖然接近[[濱景花園|濱景花園]]等屋苑，但由濱景花園前往此站需要步行約10至15分鐘，加上鄰近的巴士站已有不少巴士路線可前往市區，鄰近的酒店旅客亦較多會乘搭[[龍運巴士A41P線|龍運巴士A41P線]]前往機場，因此較少使用本站。

2010年一幅位於安耀街、安群街與安麗街交界、沙田市地段第433號的商貿用地，以8.16億元批予[[滙豐銀行|滙豐銀行]]，年期為50年。滙豐表示將於該地建後勤中心。同年政府有意在安興里興建[[骨灰龕|骨灰龕]]場，遭到當地居民強烈反對，計劃最後不了了之。
