'''石岡車站'''位於[[臺灣|臺灣]][[臺中市|臺中市]][[石岡區|石岡區]]，曾為[[臺灣鐵路管理局|臺灣鐵路管理局]][[東勢線|東勢線]]（現已廢止）的鐵路車站。

== 車站構造 ==
* 島式月台一座。

== 現況 ==
已於1991年時廢站停止營運的石岡車站，站內的鐵路線早已拆除，原路線被改建成為[[單車徑|自行車道]]（[[東豐自行車綠廊|東豐自行車綠廊]]）。車站廢站之後月台並未拆除，而是原地保留並於旁邊留置兩節[[普通車|普通車]]車廂，作為旅客參觀留念用途。除了觀光用途外，此殘留月台也成為附近攤商的小集散地，而被稱為「零擔月台」（「零擔仔」在[[臺語|臺語]]中意指「零售攤子」），之後因為諧音之故，又常被暱稱為「零蛋月台」。

石岡車站在廢站前，因站內設有兩線路軌可作為東勢線上列車的會車車站。其在廢站前原為招呼站等級的無人車站。

== 車站週邊 ==
* [[石岡水壩|石岡水壩]]
* [[臺中市石岡區石岡國民小學|臺中市石岡區石岡國民小學]]

== 歷史 ==
* 1959年1月12日：隨著東勢線正式通車而啟用，站等為三等站，設有三股軌道及島式月台一座。
* 1971年5月1日：降為簡易站，由[[豐原車站|豐原站]]管理。
* 1983年3月1日：降為招呼站
* 1991年9月1日：隨著東勢線廢線而廢站。
