'''石狩追分站'''（{{jpn|j=石狩追分駅|hg=いしかりおいわけえき|rm=Ishikari Oiwake eki|f=y}}）是位於[[北海道|北海道]][[雨龍郡|雨龍郡]][[雨龍町|雨龍町]]滿壽，隸屬於[[日本國有鐵道|日本國有鐵道]]（國鐵）[[札沼線|札沼線]]，已廢除的[[鐵路車站|鐵路車站]]。

由於本車站鄰近過去[[石狩國|石狩國]]及[[天鹽國|天鹽國]]在交通上的重要分歧點，因此被稱為「追分」。而加上舊國名「石狩」是因為開業時已存在[[追分車站_(北海道)|相同名字的車站]]。

== 歷史 ==
* 1931年（昭和6年）10月10日 - [[札沼線|札沼北線]][[石狩沼田車站|石狩沼田]]至中德富（初代，現在的[[新十津川車站|新十津川]]）開始營運。
* 1943年（昭和18年）10月1日 - 由於[[太平洋戰爭|太平洋戰爭]]擴大，[[石狩月形車站|石狩月形]]至本車站作為[[不要不急線|不要不急線]]而停止營運。本車站成為終點站。
* 1944年（昭和19年）7月21日 - [[不要不急線|不要不急線]]延長，本車站至石狩沼田停止營運。
* 1956年（昭和31年）11月16日 - [[雨龍車站|雨龍]]至石狩沼田路段恢復營運，全線回復正常服務。
* 1972年（昭和47年）6月19日 - 札沼線新十津川至石狩沼田路段廢除，本車站同時廢站。

== 車站構造 ==
站房位於札幌方向的左方東側，為類似島式的[[側式月台|側式月台]]1面1線的地面車站。月台與站房之間設有1條貨物裝卸線，沒有設置貨物月台。此外，貨物裝卸線與列車路軌為一線貫通的狀態。

== 車站周邊 ==
車站設置已經全部移走，並豎立車站的紀念石碑。
